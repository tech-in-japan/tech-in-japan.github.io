Hackathon "Budget-Pro"：競争での勝利への第一歩





ロシアの財務省は、連邦執行機関のオープン性の概念に従い、オープン状態データの方向性を開発し続け、第3回コンテスト 「Open State Financial Data BudgetApps 2017」への参加申請を受け入れ始めました。



8月12〜13日に、Infocultureはロシア財務省とともにBudget-Proハッカソンを開催し、 「ラピッドプロトタイピング：テクノロジーとツール」をテーマに、設計とプロトタイピングの分野のマスタークラスの専門家を募集します。 その後、主要部分が始まります。



Hackathonの参加者には、プロトタイプを作成するのに役立つオープンな公開財務データに基づいた一連のプロジェクトコンセプトが提供されます。

独自のコンセプトの策定と開発も歓迎します。 この場所で、私はより詳細に立ち止まって、いくつかの神話を払拭しようとします。 潜在的な参加者は「財務データ」という用語に焦点を合わせていることが多く、ロシア財務省の公開データには、税関および税務サービスなどの下位サービスのデータや地域執行機関のデータも含まれているという事実を見失っています。そして、市当局、すなわち：



  • ロシア連邦財務省 。
  • 連邦財務省 ;
  • 連邦税務局 ;
  • 連邦税関 ;
  • アルコール市場の規制のための連邦サービス 。
  • ロシア連邦の予算システムの統一ポータル「電子予算」 。
  • 州（市）機関に関する情報を掲載するための公式ウェブサイト 。
  • 連邦情報アドレスシステム ;
  • 公共調達の分野における統一情報システムの公式サイト 。
  • プロジェクトKGI 州の支出 。
  • 連邦予算補助契約の登録 ;
  • 予算プロセスの参加者 ;
  • 地域の執行機関の開かれた州の財務データは地域の財務省です。


ハッカソンの主なタスクは、ラピッドプロトタイピング技術を使用して、提案または独自のコンセプトのアイデアをモバイルアプリケーション、Webサービスまたはインターネットポータル、Telegramボットのプロトタイプの形で提示することです。 概念、タスク、およびアイデアの更新されたリストがリポジトリに公開されています 。



効果的な作業のために、ハッカソンのすべての参加者にAxure.comのアカウントが提供されます。



公式文書からのデータであるが、機械可読形式ではないデータを使用できることを別に強調する必要があります。 そして、そのようなデータを自動化された形式に変換する努力をする特に意欲的な参加者は、オープン性に対する貴重な貢献に対して個別に授与されます。



このイベントは主にウェブ開発者とデザイナーを対象としていますが、ラピッドプロトタイピングとモデリングのスキル、およびオープンな政府の財務データに基づくプロトタイプの作成に興味があるすべての人を招待しています。



特に地域からの参加者のために、私たちは専門家によるプレゼンテーションのビデオ放送と、プロジェクトのプレゼンテーションでビデオ会議形式でリモートで話す機会を組織します。



ハッカソンの賞金は2万ルーブルとパートナーからのその他の賞金です。



最高のハッカソンプロトタイプはInfocultureのサポートを受け、プロジェクトを作業状態に仕上げることができます。 彼らの著者はまた、9月8日にモスクワ金融フォーラムでロシア連邦財務大臣のおかげでポートフォリオを補充する機会を持つでしょう。 詳細はハッカソンで公開されます。



ここでハッカソンに登録できます 。



ハッカソンで作成されたプロジェクトはプロジェクトアプリケーションとしてコンテストに参加し、その作成者はその結果に基づいて賞金を申請することを期待しています。 コンテストの賞金総額は495千ルーブルです。 コンテストの受賞プロジェクトの選択と賞金の授与は、2017年11月に開催されます。



既に、 コンテストへの参加を申請し、メールopendata@minfin.ruまたは電話8-800-500-10-92でコンテストのコンタクトセンターからサポートを受け、またBudgetAppsコンテストの他のイベントへの参加を計画することもできます。



ロシア財務省の公式ウェブサイトおよびソーシャルネットワーク（ FacebookおよびVkontakte）で競争のニュースをフォローしてください。
