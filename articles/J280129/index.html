テキストと行の間隔が通常のリンクには点線の下線

残念ながら、テキスト装飾のスタイル：ドットはほとんどのブラウザでまだサポートされておらず、そのようなことをしたいという要望があります。

境界線下：点線が助けになります。

ただし、text-decoration：underlineで実装されたテキストと下線の間に1pxの距離がある場合、border-bottomは2を与えます。 その効果は、下線がテキストからわずかに引き裂かれていることであり、あまり良くありません。

長い実験の間に、すべてのブラウザーで必要なものを提供するパラメーターが見つかりましたが、テキストが1行にある場合にのみ機能します。 



a { border-bottom: 1px dotted; text-decoration: none; display: inline-block; height: 1em; line-height: 1.05em; }
