大学生に開発用の有料ツールを提供する5つの組織





こんにちはHabr！



今日、多くの組織は学生に一種の「アクション」を用意して、フリーソフトウェア、またはボーナスと特権（および組織自体からではなくパートナーから）を受け取ることを許可しています。



したがって、この投稿を読んだ場合、あなたは大学生であり、大規模な組織のさまざまなソフトウェアやその他のグッズに無料でアクセスするプログラム（1つだけではないかもしれません）に参加したいと思っています。



1. GitHubによるStudent Developer Pack

「ほとんどの学生にとって、開発ツールの価格は高すぎる可能性があります。 そのため、パートナーと共にGitHub Student Developer Packを作成しました。多くの開発ツールを1か所にまとめ、実際に学習する機会を提供します。

詳細
パッケージに含まれるもの：

  • AWSクラウドへのアクセス、無料のトレーニング、リソースへのアクセス。 合計100〜115ドルのAWSローン15件のサブスクリプションが含まれます。
  • Bitnamiからワンクリックでクラウドアプリケーションをインストールします。 年間「ビジネス3」プランへのサブスクリプションが含まれます。
  • クラウドソーシングおよびデータマイニングプラットフォームCrowdFlowerへのアクセス。
  • DigitalOceanクラウドホスティングの50ドルのメンバーシップ。
  • DNSimpleのサービスと堅牢なワンクリックDNS APIを管理します。 2年間の個人DNSプランのサブスクリプション。
  • サービス「プログラマーのライブサポート」HackHandsの25ドルのサブスクリプション。
  • トレーニング期間中、GitHubにある無制限のプライベートリポジトリ。
  • Visual Studio IDEを含むMicrosoft Azureクラウドサービスと開発者ツールのセット。トレーニング期間中も同様です。
  • NameCheapから1年間「.me」ゾーンにある1つの無料ドメインと、そのための無料の年間SSL証明書。
  • 高速アプリケーション用のデータベース開発サービスであるOrchestrateでのトレーニング中の開発者アカウント。
  • 学生は、SendGridで毎月15Kの無料メールを使用して計画します。これは、メールを介したトランザクションの追跡とマーケティングのための信頼できるサービスです。 この計画は、研究期間中利用できます。
  • Stripe Webおよびモバイル決済サービスからの最大1000ドルの無料トランザクション処理。
  • トレーニング期間中、「継続的開発」Travis CIのホスティングでのプライベート開発への無料アクセス。
  • ナノディグリートレーニングプログラムに参加するためのUdacity Student Developer Pack。 毎月、トレーニングプログラムのいずれかを選択できます。
  • UE4-PC、コンソール、スマートフォン、VRデバイス、およびブラウザーでゲームを開発するための完全なツールセット。 これらはすべて、トレーニング期間中です。




2.マイクロソフトのDreamSpark

「マイクロソフトのプロフェッショナルな開発および設計ツールに無料でアクセスできます。」

詳細
パッケージに含まれるもの：

  • Microsoft Azureへのアクセス-クラウドコンピューティングのプラットフォーム。
  • Windows Server-サーバーを保守するためのオペレーティングシステム。
  • Visual Studio Professional-IDEおよびその他のソフトウェア開発ツール。
  • Windowsストアおよびマーケットプレイスへの無料登録。
  • Microsoft Expression Studio-開発者とデザイナー向けのパッケージ。
  • Windows Embedded-ハードウェア要件の低いOS。
  • SQL Server-リレーショナルデータベース管理システム




3. Adob​​e Creative Cloud

「Photoshop CCやIllustrator CCなどの20以上のプログラム、および独自のポートフォリオサイト、有料フォント、20 GBのクラウドストレージを含む、PCおよびモバイルデバイス用のクリエイティブアプリケーションの完全なコレクション。」

PSこれは無料ではなく、最大60％の割引しか受けられないことに注意してください。



4. JetBrainsのプロフェッショナル開発者ツール

「Java、PHP、Ruby、Python、JavaScript、Objective-C、または.NETを学習していますか？」

それとも、単に研究を計画しているのですか？ JetBrainsのプロの開発者ツールを今すぐ始めましょう。 そして最も重要なことは、無料です。」

詳細
学生として登録すると、次のツールにアクセスできます。

  • ReSharper- MVSの生産性を向上させるプラグイン。
  • IDEA-Java、JS、Pythonなど、多くのプログラミング言語で開発するためのIDE。
  • AppCode-iOS、OSXの開発者向けのスマートIDE。
  • CLion-C / C ++開発者向けのクロスプラットフォームIDE。
  • RubyMine-Ruby on Railsの開発者向けIDE。
  • DataGrip-データベースおよびSQLコードを操作するためのIDE。
  • PHPStorm-PHPで開発するためのIDE。
  • PyCharmは、PythonおよびDjangoでの開発に最適なIDEです。
  • WebStorm-JavaScript開発用のスマートIDE
