I2P：任意のブラウザーからの透過的なアクセス









最近 、 ロックに 関連するニュースの量は 屋根を通過して います。



しかし、人々はどこへ行くのでしょうか？



情報へのアクセス、言論の自由、および少なくとも憲法にあるそれらの規則は尊重されますか？



メッシュネットワークは適合せず、 Tor-必要なものを提供できず、 I2Pは残ります。



今日は、 すべての主婦がこのネットワーク上の任意のサイトを開くことができるように、I2Pで透過的なアクセスを設定する方法についてお話します。



最初のステップ



明らかに、まだインストールしていない場合はI2Pが必要になります。今すぐインストールするか、明日、おそらく公式のI2P Webサイトもブロックされます。



www.i2p2.de/download

インストール中に、I2Pサービスをインストールすることに同意します。これにより、コンピューターの電源を入れたときにバックグラウンドで機能することができます。



カスタマイズする



ページhttp://127.0.0.1:7657/configで 、速度制限をチャンネルの60％に設定します（気にしない場合はもっとできます）



システム内のすべてのブラウザーから簡単にアクセスできるようにする



I2Pには、ブラウザのプロキシ自動構成ファイルが含まれています。Windowsでは、次の場所にあります。

C：\ Program Files \ i2p \ scripts \ i2pProxy.pac



次に、IEからフィードする必要があります。











ブラウザー設定->接続->ネットワーク設定に移動します











自動構成スクリプトを使用->このファイルの前にアドレスを貼り付けますC：\ Program Files \ i2p \ scripts \ i2pProxy.pac （任意のホスティングにアップロードしてリンクを挿入できます）



それだけです



お茶を注いでゆっくり飲むことができます。i2pネットワークへの新しいノードの統合はすぐには行われません。カップが空になるまでに、ネットワーク上の多くのサイトのいずれかにアクセスできます。



すべてのブラウザがI2Pネットワークにアクセスできるようになりました

この設定により、誰かがダブルコールでそれを必要とするかどうかを判断できます。



したがって、蜂起の試みを準備したり、自殺のさまざまな方法のさまざまな利点について話したりする場合は、I2PネットワークにのみアクセスできるI2P用の別のブラウザを使用することをお勧めします
