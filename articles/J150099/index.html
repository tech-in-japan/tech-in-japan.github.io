ミントとは？ これはMegaFon + Intelです。 数ヶ月間、IntelはOrangeオペレーターと協力してこのモデルをヨーロッパで販売してきました。今が私たちの番です。 オペレータからオペレータに変更するときの外観とハードウェアの詰め物の両方は、何も変わらないので、わずかに変わります-裏表紙にネームプレートが必要です。





デバイスの特性は次のとおりです。

  • オペレーティングシステム：Android 2.3
  • 2G標準：GSM / GPRS / EDGE 1800/1900/850/900 MHz
  • 3G標準：UMTS 1900/2100/850/900 MHz、HSPA + 21.1 Mbps、HSUPA 5.76 Mbps
  • ディスプレイ：解像度1024x600ピクセルの4.03インチ容量性TFT
  • プロセッサー：Intel Hyper-Threading TechnologyをサポートするIntel Atom Z2460、1600 MHz
  • RAM容量：1024 MB
  • 内部メモリの量：16 GB
  • メインカメラ：8メガピクセル、ビデオ1920x1080、オートフォーカス、LEDフラッシュ、4xデジタルズーム
  • フロントカメラ：1.3 MP、ビデオ640x480
  • RDSを使用したFMラジオ
  • 通信およびインターフェース：Wi-Fi（802.11b / g / n）、Bluetooth、NFC
  • コネクタ：3.5 mmオーディオ、Micro-USB、microHDMI
  • センサー：加速度計/照度/近接
  • GPS衛星ナビゲーション
  • バッテリー：1460 mAhの取り外しできないリチウムイオンバッテリー
  • 時間：336時間/ 14時間
  • SIMカードのフォームファクター：Micro-SIM
  • 寸法：123x63x11 mm
  • 重量：124グラム
