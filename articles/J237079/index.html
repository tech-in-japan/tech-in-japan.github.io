Makeblock Starter Robot Kit V2.0のレビュー。 パート1.開梱

今日、私は高度なMakeblock Starter Robot Kit V2.0のデザイナーの所有者になりました。 RuNetでこの珍しいレビューを1つも見つけられなかったので、私の意見では興味深いデザイナーであるため、この製品の印象を共有することにしました。 今日は、 Makeblock Starter Robot Kit V2.0が何であるか、パッケージに何が含まれているか、そして写真付きの簡潔なストーリーを添付します。



Makeblockとは何ですか？



Makeblockは、中国の深Shenzhenで開発されたオープンデザインプラットフォームです。 同じ名前のMakeblock社が生産および販売しています。 Makeblockは 、開いていることに加えて、 次の特徴もあります。



  • 2色（青と金）の耐久性のある陽極酸化アルミニウム部品。
  • Arduinoと互換性のあるコントローラーに基づく電子機器。
  • モーターを含む レゴマインドストーム部品を使用する機能。




なぜメイクブロック



大学の同僚の1人（現在はYandexにいる）のアドバイスで、LEGO Mindstorms EV3に出会った1年足らず前にロボット工学に夢中になりました。 1つの産業用ロボットのモデルを開発する必要がありましたが、私の同僚はすでに間接参加でゼロから飛行ロボットを構築した経験があるため、既製のプラットフォームを使用することにしました。 選択肢はすぐに新しいLEGO Mindstorms EV3に落ちました。 LEGOの開発により、この選択には重大な欠点があることが明らかになりました。



  • LEGOから派生したデザインは非常に脆弱です。
  • キット自体だけでなく、不足している部品も非常に高価です。
  • コンピューターの機能はかなり制限されています。
  • EV3の周辺機器の範囲も限られています。
  • さらに、EV3はグラフィカルにのみプログラムされます。




したがって、私はそのような欠点を持たないロボットプラットフォームを探し始めましたが、それらが詳細にレゴと互換性があることが望ましいです。 途中で、Habrahabrに関する1つの投稿のおかげで、Arduinoに興味を持ち、買収して習得し始めました。 「lego arduino」という単語をウェブで検索すると、Makeblockにアクセスできました。 中国の人たちはすでに私のことを考えていて、市場に非常に興味深い製品を提供していたことがわかりました。 しかし、3つの事実は特に印象的でした。



  • Makeblockに基づく既製セットの範囲には、2つの「標準」ロボットを構築するためのセットがありました。
  • プロッタプロット用のキット。
  • さらには3Dプリンターも！




Makeblock Starter Robot Kitとは何ですか？



その結果、色域全体に慣れてきたので 、 Makeblock Starter Robot Kit V2.0を選択しました 。 このようなキットを入手した後、私はArduino互換コントローラーに基づいて、超音波センサーと赤外線信号受信機を備えた2つの走行ロボットの1つを構築する機会を得ました。 しかし、電子部品とメカニクスを別々のセットで、また各部品を別々に入手する可能性もありますが、より高価になります。 製品ラインには、他のセットに加えて、多数のセンサー、モーター、あらゆる種類のメカニックがあります。



Makeblock Starter Robot Kit V2.0は、ロボットを構築するためのポストキットを作成した時点で最新のものです。 従来のArduino互換コントローラーとセンサーとモーターをRJ11コネクターで接続するためのシールドを組み合わせた新しいコントローラーを使用している点で、最初のバージョンとは異なります。



新しいバージョンのコントローラーに加えて、セットには以下が含まれます。



  • 超音波距離計タイプHC-SR04;
  • 赤外線受信機;
  • 赤外線リモコン;
  • 2つの中型モーター;
  • プラグコネクタ付きバッテリーパック。
  • 2つの透明なブラケット。
  • トラックのトラック。
  • 2つのゴム製タイヤ;
  • 4つの大きな滑車、それらは車輪です。
  • いくつかのビーム;
  • ジョッキーホイール;
  • 最低限必要なファスナー;
  • RJ11ケーブル
  • USBケーブル
  • メカニックを組み立てるためのツール。
  • 指示。




これらはすべて、重量約1.5 kgの小さな26 x 20 x 10 cmのスタイリッシュな箱にきれいに詰め込まれています。
