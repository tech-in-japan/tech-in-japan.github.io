PVS-Studio for Microsoft Visual Studio Community 2013









マイクロソフトは、次のソフトウェア製品であるVisual Studioの新しい無料バージョンを発表しました。唯一の制限は、エンタープライズアプリケーションの開発の禁止です。



このイベントは、 PVS-Studio製品にとってどのような意味がありますか？



Visual Studio Community 2013は、Web、クラウド、デスクトップ、モバイルデバイスなど、多くのプラットフォームおよびシナリオ向けのソリューションを開発者が作成できる新しい無料版です。



Visual Studio Community EditionとExpress Editionには、主に2つの違いがあります。

  1. Visual Studio Expressエディションでは、ユーザーは拡張機能を使用できません。 Visual Studioギャラリーには、開発者の生産性を向上させることができる5,000を超えるVisual Studio用の優れたプラグインがありますが、Visual Studio Expressユーザーは利用できません。
  2. Visual Studio Expressエディションはプラットフォーム固有です。 Express for Web; Express for Windows; Express for Windows Desktop ...しかし、Visual Studio Community Editionでは、サポートされているすべてのプラットフォーム用のプロジェクトを開発できます。


もちろん、拡張機能とアドオンをインストールする機能はPVS-Studioにとって重要です。 Visual Studio 2013 Professionalの有料版に迫る機能的な機能を備えた本格的な製品であるVisual Studioの無料版は、静的分析を使用してソースコードの品質を制御することが重要な大規模アプリケーション向けの魅力的な開発環境になりつつあります。



このステップはおそらく、Microsoftプラットフォームのアプリケーション開発者の輪を広げ、スタートアップ、中小企業、個人開発者、オープンソースアプリケーション開発者コミュニティ、および学生に完全に機能するアプリケーション開発環境を提供することを目的としています。



Visual Studio Community 2013でのPVS-StudioとVisual Studio 2013のプラグインの機能の統合をテストするために、いくつかのプロジェクトがテストされました。
  • Media Player Classic Home Cinemaは、Windows用の無料のオーディオおよびビデオプレーヤーです。 プロジェクトは記事の資料を収集しませんでしたが、Coverityを使用してより多くの資料を確認しました。
  • LibreOfficeは、無料の独立したオープンソースのオフィススイートです。 検証の結果は、近日中にリリースされる記事で入手できます。
  • 検証の候補は、ガベージコレクションやマシンコードへのコンパイルなど、さまざまな機能を実装する.NET Coreの重要な要素であるCoreCLRです。
