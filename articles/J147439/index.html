マイクロソフトがPerceptive Pixel Inc.を買収



知覚的ピクセルの大規模なマルチタッチハードウェアおよびソフトウェアテクノロジーは、新しいコラボレーションと生産性のシナリオを解き放ちます。



レドモンド、ワシントン、ニューヨーク-2012年7月9日-Microsoft Corp. およびPerceptive Pixel Inc. （PPI）は本日、Microsoftが大規模なマルチタッチディスプレイソリューションの研究、開発、および生産で有名なリーダーであるPPIを買収することで決定的な契約を締結したことを発表しました。



「PPIの買収により、補完的な強みを活用できるようになり、この市場の進化を加速できることを嬉しく思います」と、MicrosoftのOffice Division社長、Kurt DelBene氏は述べています。 「PPIの大型タッチディスプレイは、OEMのハードウェアと組み合わせると、強力なWindows 8ベースのPCとなり、生産性とコラボレーションの新たな可能性を開きます。」



2006年にマルチタッチテクノロジーの有名な先駆者であるジェフハンによって設立されたPPIは、2007年初頭に最初のマルチタッチワークステーションと大型壁ソリューションを出荷しました。米国大統領選挙。 2009年、スミソニアンは、インタラクションデザインの最初のカテゴリでナショナルデザイン賞を受賞しました。 PPIの特許技術は、政府、防衛、放送、エネルギー探査、エンジニアリング、高等教育などのさまざまな業界で使用されており、ソフトウェアとハ​​ードウェアの両方の専門知識は、コラボレーション、会議、プレゼンテーションなどの幅広いシナリオの成功に貢献します。



「私たちは、人々が協力してコミュニケーションをとることを可能にするテクノロジーを構築するという相互の情熱に協力することに非常に興奮しています」とハン氏は言います。 「Microsoftに参加することで、Microsoft Office部門の途方もない勢いを活用し、その製品と緊密に相互運用し、この技術を非常に幅広い顧客に提供できるようになります。」



取引条件は明らかにされていません。 買収は、規制当局の承認を含む慣習的な締結条件の対象となります。



Perceptive Pixel Inc.について



Perceptive Pixel Inc. （PPI）はインタラクションデザインのリーダーとして認知されており、2008年の歴史的な米国大統領選挙で際立った大規模で無制限のマルチタッチデバイスの開発を先駆けてきました。 PPIの高度なマルチタッチテクノロジープラットフォームにより、専門家はコンテンツに近づき、理解、コミュニケーション、コラボレーションの障壁を取り除くことができます。 PPIは非公開で、本社はニューヨークにあります。 詳細については、 www.perceptivepixel.comをご覧ください 。
