Python Meetup：1月の会議





素晴らしく力強いパイソン、非公式な雰囲気、興味深いレポート、そして志を同じくする多くの人々-これらはすべて、毎月最終金曜日に伝統的にミンスクで行われます。 この出版物では、1月の会議のビデオレポートを見ることができます。RomanImankulovの「 データ解析用Python 」とStas Rudakovの「 ロギングに関するもう1つの退屈なレポート 」です。 ビデオと猫の下のスライド共有へのリンクをご覧ください。



1. データ分析用のPython 、Roman Imankulov、 Doist.io社 。





データ分析レポートは、主に試してみたいが、どこから始めればよいのか分からない人向けです。 誰がこれを必要としているのか、どこでデータを取得するのか、初心者のPythonアナリストが自由に使えるツール、最初からすべてから得られるメリットについて。



2. 別の退屈なロギングレポート 、Stas Rudakov、 Game Stream LLC / Wargaming.net 。





ログの価値はしばしば過小評価されますが、無駄になります。 ミーティングのすべての参加者と活発に議論し、理解するためにレポートを作成します。ログを書き込む方法、場所、理由 さらに、ログからより多くの情報を取得する方法の問題を提起しました。



PS：春の最初の月は、 Python Meetupの例外ではありません。 したがって、 3月28日の夜を安全に予約できます。いつものように、会議はミンスクの中心で開催されます。 Facebookページの詳細に従ってください： facebook.com/MinskPythonMeetup
