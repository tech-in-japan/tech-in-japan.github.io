Windowsでの「ランチャー」の概要

Launchyに関する投稿は、このタイプのいくつかのプログラムについて書かれたコメントに新しい年の前にハブに登場しましたが、少なくともそれらの概要を簡単に把握できなかったため、何を選択するかを決めるのはかなり難しく、一部の人々はそのようなものを初めて見ました、私はそのようなプログラムのミニレビューを書くことにしました：

Launchy 、 Type and Run 、 SlickRun、およびExecutor 。

Launchy 2.0

公式サイト

ダイレクトリンク （4.4mb）



プログラムは最近言及され、そのすべての機能が説明されています。

繰り返し、したがって、印象について簡潔に。

それ自体ではなく、10MBのハードドライブを要求します。 起動後、約14〜16MBかかります。 キーを離すとすぐに呼び出されることに注意してください。 デフォルトでは、 「Alt」+「Space」を押すと起動しますが、これはあまり便利ではありませんが、設定を変更できるため、怖くありません。

検索は「pe」と入力された単語の一部で行われます。この音節を含む他のプログラムは「Opera」によって発行されました。

Launchyがインデックス付きフォルダーでのみ機能することは少し不便です。ただし、独自の機能を追加することはできますが、拡張機能にマスクを使用しても、さまざまな追加設定やアンインストールファイルを追加できます。 一部の人にとってのこの仕事の原則はプラスにすぎませんが。

基本的な構成には選択肢がありますが、サイトにはスキンのセットがあり、誰もが自分で見つけることができます。 さまざまなプラグインを接続することもできます。



多くの設定、外観、スキャン用のディレクトリ、プラグイン、情報はありません。

一般的には良い印象しかありませんが、私が言ったように、起動するファイルの構成を作成する方が便利です。

v4b7を入力して実行する

ダイレクトリンク （2.8mb）



ディスクは約4メガバイト、RAMでは約6メガバイトかかります。 打ち上げもすぐです。 デフォルトでは、 「Scroll Lock」を押すと起動しますが、設定で変更できます。

オートコンプリート、最近の起動の履歴があります。いくつかの文字を入力してTabを押すと、起動されたすべてのファイルをクリックします。

構成を簡単に作成するために、右クリックメニューで「タイプに追加して実行...」という項目を追加できます。プログラムのショートカットをクリックすると、構成エディターが呼び出されます。ここで、実行するエイリアスを入力するだけです。 開始するには、追加のパラメーターを指定できます。



追加のプラグインもあります。 ロシア語がサポートされています。

直接リンク （967kb）



ディスク上では3MB、RAMでは約7MBかかりました。 他の人と同じように始めるのはとても速いです。

Launchyの場合と同じように検索すると、アプリケーションの画像が表示されます。 TaRのように「Tab」を押すと、使用可能なオプションが変わります。
