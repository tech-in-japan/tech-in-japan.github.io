Parrot Security OS-Kali Linuxの代替





Parrot Security OSは、Debian-linuxをベースにしたセキュリティディストリビューションです。 初心者にもプロにも適した学習しやすい。 この記事では、このディストリビューションと、私が話をした何人かの貢献者M. EmrahÜNSÜRのプロジェクトの開発についてお話します。



Parrot Security OS



Debian-linuxに基づいて成長しているセキュリティディストリビューションの人気が高まっています。 初心者にも専門家にも適した非常に簡単な学習方法。 この配布は、侵入テストとインターネット上の匿名作業の両方を目的としています。



非常に簡単で効果的なツールであり、多くのセキュリティ専門家は、特にParrotが更新にKaliリポジトリを使用しているため、ますます「食いしん坊」のKaliの代わりになることを発見しました。 MATEシェルとLightDMディスプレイマネージャーを使用します。







機能はKali Linuxに似ていますが、ここでも、セキュリティテスト用の特別なソフトウェアがシステムに大量に提供されています。



特徴的な機能のうち、Kaliの匿名性よりも大きなバイアスと呼ばれるものがあります。それは、I2P（見えないインターネットプロジェクト）とプリインストールされたTORサービスの統合です。



Parrot Security OS（3.8-JollyRoger）には、いくつかのバージョン（32および64ビットプラットフォームを含む）があります。



  • Parrot Security 3.8 Full Edition-フルバージョン。
  • Parrot Lite 3.8 Home Edition-アセンブリの基礎としてのユーティリティなし。
  • Parrot AIR 3.8-ワイヤレスネットワークのテスト用。
  • Parrot Cloud Edition-クラウドサービスで展開するためのバージョン。
  • 組み込みデバイスとIoT-Raspberry Pi、Orange Pi、Pine64。


プロジェクト開発



このディストリビューションはかなり有望に見えるので、もう少し知りたいと思い、貢献者の一人とプロジェクトの積極的な参加者であるM. EmrahÜNSÜRと話しました。



M. EmrahÜNSÜR（meu @ Parrot Project）は、民間企業のセキュリティ研究者です。 彼は、Parrotプロジェクトフォーラムの寄稿者およびモデレーターです。

独自の配布キットを作成する理由 このようなことをする理由はいくつかあります。 主な理由は、最も便利なパッケージに使用されているソフトウェア製品のほとんどが常に手元にあるということです。 Parrot Security OSの場合、開発者は何らかの理由で、継続的な更新、最新のテクノロジー、実際にペンテスト中に使用されるソフトウェアの最新リストを備えた、シンプルですぐに使用できるアセンブリという形で、現在のディストリビューションに満足していませんでした。 （BlackArch c> 1500ユーティリティを理解する時が来ました）。 また、この分布は日常の作業に適している必要があります。



プロジェクトの断片化について文句を言う人のために、オープンソースコミュニティがどのように機能するかについての情報があります。断片化は、同様のプロジェクトの他のチームからの多様化とアイデアの自由な表現を引き起こします。 また、プロジェクトの枠組みの中で、協力が行われています

Debian pkg-security。



開発計画には、開発、システム管理、ネットワークインタラクション、暗号化、マルウェア分析、ペンテストなどに関するドキュメントとテキストの完全なプールの作成が含まれます。別の開発ベクトルは、新しい組み込みデバイスのサポートです。 また、主な問題-ハードウェアの互換性-は多大な労力を必要とし、開発計画はPython、C、Bashを知っているコミュニティの新しいメンバーを引き付けることです。
