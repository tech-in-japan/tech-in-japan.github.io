管理されているITインフラストラクチャ：Web会議がリリースされました



2か月以上前、プロジェクト「ITインフラストラクチャの管理」を開始しました。このプロジェクトでは、ITプロフェッショナルがNetWrixソリューションを使用してITインフラストラクチャの管理を簡素化する方法についてお話しすることを約束しました。 このために、Web会議の形式が選択されました。 毎週、ITインフラストラクチャの1つ以上の関連コンポーネントについて話し、それらを監視するように設計された作業プログラムで示しました。

300人以上がWeb会議に直接参加しました。

しかし、我々のソリューションについてもっと知りたいと思う人々に機会を否定するのは間違っていると思い、Youtubeにすべてのエントリを投稿しました。

合計で、6つの会議を開催しました。以下はそれらのリストです。



Active Directoryおよびグループポリシー：組織の変更に遅れないようにする



Webキャストでは、管理者が2つのNetWrixソリューション（Active Directory Change ReporterとGroup Policy Change Reporter）を使用してActive Directoryとグループポリシーの変更を制御する方法について説明しました。 プログラムは、シンプルでわかりやすい変更レポートを送信し、重要なセキュリティイベントのアラートを生成し、ADオブジェクトとグループポリシーを復元できます。 NetWrix Inactive Users Trackerもレビューされ、非アクティブなユーザーアカウントが特定されました。

ファイルサーバーで発生していることの制御：NetWrix File Server Change Reporter



ファイルへのアクセスを監査するための標準システム（およびより広く採用されている場合はファイルサーバー）には、多くの要望が残されています。イベントログに生成される情報が多すぎるため、管理者はファイルサーバーで何が起こっているかについて迅速に答えを得ることができます。 Web会議で、NetWrix File Server Change Reporterソリューションがレビューされ、ファイルサーバーの監査プロセスが簡素化されました。

Exchange管理：変更とメールボックスアクセスの追跡

NetWrix SQL Server Change Reporterを使用してSQL Serverの変更を制御する
