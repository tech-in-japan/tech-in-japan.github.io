Dental Cloud Dental Automation Cloud Serviceがリリースされました





Dental Cloudは 、同名の歯科医院と診療所を管理するためのオンラインサービスの更新を発表しました。これは、今年3月に開始されました。



開発者によると、サービスには「医師の職場」が補充されたという。 アップグレードの前に、Dental Cloudサービスは患者との作業を実装し、医師の人員配置表と予定表を維持し、アカウントを操作し、クライアントへの請求を行いました。 更新後、各医師は、患者の「紙」医療記録で治療プロセスを実施するか、電子形式で病歴を維持することに切り替えるかを選択できるようになります。



Dental Cloudに追加された他の機能には、グラフィック歯ブラシがあります。 患者カード; 訪問カード; 治療カード; アポイントメントアポイントメントウィジェット-クリニックのWebサイトに投稿され、サービス自体と統合されています（アポイントメント情報はシステムに自動的に投稿されます）。



同社の独立取締役であるアレクセイ・カラチニコフは、次のように述べています。 さらに、更新されたバージョンのDental Cloud SaaSサービスでは、インターフェイスが完全に変更されました。現在は、単一ページアプリケーションの概念で作成されており、ブラウザでモバイルデバイス上のサービスを操作できます。



「近い将来、Dental Cloudサービスは、医師と患者間の多面的なコミュニケーションが実装される本格的なコミュニケーションプラットフォームに変わります。 一方で、歯科医院と診療所を管理するための当社のクラウドサービスは、医師や小規模歯科医院の実践のニーズを完全に満たしています」と、同社のCEOであるヤロスラフオセトロフは述べています。
