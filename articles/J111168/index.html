Psx4droid v2 1.0.9-Sony PlayStation Oneエミュレーター







そして最後に、おそらく最も興味深いコレクションエミュレーターです。 悲しいかな-今回はすべてがそれほどスムーズではありません。 互換性はそれほど高くありません-エミュレーターはゲームの半分以上を起動しますが、それでも-互換性のあるものが十分ではありません。 また重要なことは、エミュレーション速度はトップエンドデバイスでも高くありません。 これは、エミュレーション中にグラフィックアクセラレータがサポートされていないためです。 それにもかかわらず、エミュレーターは最近活発に開発されていますが、私たちが望むほど速くはありません（たとえば、最新バージョンではSamsung Galaxy Spicaのサポートが追加されました）。 OpenGL ESのサポートはまだ約束されていますが、残念ながら、開発者がこれによって何を理解しているかは明らかではありません。 それにもかかわらず、現在の機能：

  • トップエンドのスマートフォンのサポート（600MHz +プロセッサ周波数を推奨）
  • サポートを保存
  • マルチタッチのサポート
  • 仮想キーボード設定を構成する
  • 物理的な電話ボタンにアクションを割り当てる機能
  • 自動フレームスキップのサポート
  • 一般的なラムフォーマットのサポート（BIN / ISO / IMG / PBP / Z / ZNX / Eboot）


重要：エミュレーターを機能させるには、オリジナルのbios PSOneが必要です。
