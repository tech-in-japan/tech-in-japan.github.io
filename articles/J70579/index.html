Windows 7を使用する準備はできていますか？ いくつかのホットキー

私はすぐに謝罪したい、記事はあまり技術的ではありません。 作業を容易にするすべてのWindows 7ホットキーを既に知っていると思われる場合は、このトピックをスキップしてください。



偶然、新しいWindows 7ツールバーに関連付けられた「ホット」キーを備えたタブレットに出会いました。できるだけ多くの便利な組み合わせを収集した後、私は自分自身のために多くの新しくて便利なことを学びました。





1. アプリケーションの新しいインスタンスを開きます。 （必ずしもではないが）タスクバーに何らかの種類のアイコン（たとえば、Word）を添付し、新しいプログラムインスタンスをすばやく起動したい場合-前に、これを行いました：アイコンを右クリックして、新しいプログラムインスタンスを既に選択しました。 Shiftキーを押したままにすることですべてが軽減できることがわかりました。Shiftキーを押しながらアイコンをクリックすると、アプリケーションの新しいインスタンスが起動します。 アイコン上でマウスホイール（マウスの中央ボタン）をクリックして、プログラムの新しいインスタンスを開くこともできます（ フラッシュフェイスで ）



2. 管理者に代わってプログラムを開きます。 最初の場合と同様に、私は以前にこのようにしました。アイコン上でマウスの右ボタンを使用し、次にプログラムのショートカット上でマウスの右ボタンを使用して、[管理者として実行]を選択しました CtrlキーとShiftキーを押しながらアイコンをクリックすると、問題が解決します 。プログラムを管理者として実行してもかまわないというお気に入りのウィンドウが表示されます。 管理者としてプログラムを絶えず実行する必要がある場合-これはプログラムショートカットの[互換性]タブで構成できます-[管理者としてこのプログラムを実行する]オプションがあります。 Visual Studioでこれを行いましたが、インターネットインフォメーションサービスを操作するのに十分な管理者権限を持っていないことが時々ありました。



3. 古い忘れられたウィンドウのメインメニューを参照してください 。 それが何であるかを言葉で説明することは困難です、私は写真を見せます：







これは、実行中のアプリケーションのアイコンを右クリックしたときに、以前のバージョンのWindowsで見た標準メニューです（実行していない場合は、ショートカットファイルメニューになります）。 開始するには、もう一度Shiftキーを押しながら右クリックする必要があります 。 実行中のウィンドウ（インスタンス）のグループの場合、このグループのウィンドウを管理するためのコンテキストメニューがあります。



4.グループの次のウィンドウをアクティブにします。 プログラムの複数のインスタンスが実行されており、グループアイコンが表示されている場合 、 Ctrlキーを押しながら左クリックすると、グループの次のウィンドウが順番にアクティブになります。



5.アイコンの番号でタスクバーからプログラムを実行します。 Windowsロゴキーのクランプ タスクバーのアイコンの番号を押す -プログラムインスタンスが起動するか、実行中のウィンドウをアクティブにして（1 :)からカウントダウンする）、まったく新しいプログラムインスタンスを起動する-Shiftキーを押したままにします 。 最後のアクティブウィンドウをアクティブにするには、 Ctrl キーを押しながらジャンプリストを開き、 Altキーを押します。



6.タスクバーのアイコンを確認します。 マウスを使用しない別の方法。 Windows ロゴキーのクランプ Tキーを押します-タスクバーアイコンに従って順番にフォーカスを変換します。



また、Windowsおよびデスクトップを操作するための既知のキーを推奨することもできます。





Windowsロゴキー + D-デスクトップを表示



Windowsロゴキー + E-エクスプローラーを開く



Windowsロゴキー + F-検索ウィンドウを開く



Ctrl + Windowsロゴキー + F-コンピューター検索ウィンドウを開く



Windowsロゴキー + L-コンピューターを「ロックアップ」する



Windowsロゴキー + R- [ファイル名を指定して実行]ウィンドウを開く



Windowsロゴキー + Tab-誰もがFlip 3Dを知っている



Windowsロゴキー +スペースバー -エアロピック-デスクトッププレビュー（すべてのウィンドウが透明になるとき）



Windowsロゴキー +矢印キー（上|下|左|右） -アクティブウィンドウの位置を制御します（最大化|最小化|左端に添付|右端に添付）



Shift + Windowsロゴキー +（左|右）キー -ウィンドウを隣接するモニターにリダイレクトします



Shift + Windowsロゴキー +上キー -ウィンドウを垂直方向に最大化します



Windowsロゴキー +ホーム -アクティブなウィンドウを除くすべてのウィンドウを最小化します



Windowsロゴキー + P-プレゼンテーションのデスクトップの表示方法のモードを選択します
