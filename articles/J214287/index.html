Colobot Gold-Colobotのオープンソースバージョン





こんにちはHabr。 この素晴らしいおもちゃのソースコードをPPCコミュニティ（Colobotのポーランドポータル）に転送してからすでに多くの時間が経過しているという事実にもかかわらず、愛好家の手に落ちた後、Epsitecの作成の運命がどのように生まれたかについてのハブに関する記事はまだありませんでした。



何らかの理由でこのゲームについて聞いたことがない場合は、Habréに関するすばらしい記事があります。 さて、Colobot PPCの一部として何が行われ、何が行われる予定かについてお話します。



PPC






colobot.info開発者の公式サイト。 フォーラムの大部分はポーランド語ですが、英語のセクションにある情報から、作業はゆっくりですが確実に進行していることが明らかになります。 PPCはまた、olobot2およびolobotGoldの商業プロジェクトを計画していないとも述べています。



完了した作業






2013年6月24日に、Colobot Goldプロジェクトはアルファテスト段階に入りました。 プロジェクトの目標は、元のゲームを更新して改善することです。 多くの変更があり、作業は進行中です。 GitHubでこのすべてのビジネスに精通することができます 。 メインから-ゲームは現在、OpenGLを搭載したエンジンで実行され、マルチプラットフォームになっています。 多くのテクスチャがより高い解像度のテクスチャに変換されました。 CBOT言語（ゲーム内プログラミング言語）、新しいゲームレベル、ロシア語の翻訳に新しい機能が追加されました。 これは、Goldバージョンがグラフィックスが改善された前身のバージョンであるだけでなく、高度な機能も備えていることを示唆しています。これは朗報です。



計画






ゲームには多くの計画があります-Colobot 2のリリース、新しい機能とゲームのMOD、新しい植民地化ボットのゲームへの追加、プレイヤーの能力の拡張。 Colobot 2の開発は、Colobot Goldの作業完了後に開始する予定です。 新しいゲームのアイデアはまだ議論されています。 開発は別のリポジトリで行われる可能性が高く、完全なコードのリファクタリングが計画されています。



さらに、Android用のバージョンをリリースする予定です。 テクスチャパックのサポートおよび改造のための他の機能についてのアイデアがあります。 （この記事の著者のほとんどはマルチプレイヤーの夢ですが、彼についての言葉ではありませんが、Colobot 2がどうなるかは誰が知っていますか...）



また、Steam Greenlightにプロジェクトをドロップしたいということも言及する価値があります。



ゲームを開始するには？


すでにゲームを試してみたい人のために、このサイトはWindowsおよびGNU / Linux向けの安定した開発バージョンのゲームを定期的にアップロードしています。 Windows、BSDライクなシステム、およびMacOS向けのゲームの開発バージョンをビルドする手順は、GitHubにあります。 理論的には、ゲームは任意のプラットフォーム用にコンパイルできます。



Linuxで最新の変更を加えたゲームの開発バージョンを実行したい人向けの説明を提供します。 依存関係のリストはここにあります 。 



git clone --recursive --branch=dev https://github.com/colobot/colobot.git mkdir colobot/build && cd colobot/build cmake .. -DCMAKE_INSTALL_PREFIX=/some/install/path make install




ゲームを開始するには、次のコマンドを使用します。 



 cd /some/install/path/games ./colobot -language ru -datadir share/games/colobot/








PS


現在、無料でプレイできるゲームは、脳、アプリ内購入、DLC、その他の悪を使わずに、その本質が画面上で単調な突きに要約されています-このようなプロジェクトはこれまで以上に助けが必要です。 コードの支援、リソースの操作、翻訳の改善、バグの検索を行うことができます。また、資金でプロジェクトをサポートすることもできます（同じGreenlightにお金が必要です）。 これは、公式サイトcolobot.infoで実行できます。
