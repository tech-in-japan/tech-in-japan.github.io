Operaアプリストアは、Nokiaスマートフォンおよび携帯電話のNokia Storeに置き換わる

WindowsとWindows Phoneの開発者のコ​​ミュニティの開発に焦点を当て、Microsoftは、ノキアの古典的な携帯電話を愛する多くの人々がまだいることを忘れていません。 まだこれらのデバイスのいずれかを使用している、および/またはそれらのアプリケーションを開発している場合は、良いニュースがあります。







Nokia Storeは間もなく閉鎖されるという事実にもかかわらず、これは開発者がアプリケーションのためのスペースを持たず、ユーザーがそれらをダウンロードする必要がないという意味ではありません。 2015年の第1四半期から、 Opera Mobile Storeは、Nokiaの携帯電話やスマートフォンのデフォルトアプリストアとして、多くのプラットフォームで後者に置き換わります。



つまり、来年の初めから、シリーズ40、シリーズ60、Symbian、Asha、Nokia Xデバイスを使用してNokia Storeにアクセスしたユーザーは、Opera Mobile Storeアプリストアに自動的にリダイレクトされ、そこで数万の電話アプリケーションへのアクセスが提供されますノキア



この発表は2014年8月に発表された合意に続き、Operaは上記のプラットフォームで実行されるスマートフォンおよびNokia製の携帯電話のデフォルトのブラウザーになります。



マイクロソフトは、Opera Mobile Storeへの移行により、従来のNokiaデバイスを使用している何百万人もの人々が、新しいアプリケーション、ゲーム、およびコンテンツを確実に入手できる方法を継続できることを期待しています。



Opera Mobile Storeは、世界で最も有名で人気のあるアプリストアの1つで、13のカテゴリで300,000以上のアプリとゲームを提供できます。 このサービスの人気は、トラフィックの高さによって証明されています。これは、世界の196地域から毎月約2億人のユニークユーザーがいます。



上記のことは、Operaアプリストアが開発者にアプリケーションを配布するための適切な範囲を提供することを意味します。 Nokia Storeと同様に、新しいストアでは、開発者がローカルおよびグローバル市場の両方で有料および無料のアプリケーションを公開できます。







Nokia StoreからOpera Mobile Storeへのユーザーの移行により、後者はダウンロードの点で世界のトップ3のアプリストアに入ると予想されます。



Opera Softwareは、開発者がアプリケーションをOpera Mobile Storeで公開するように今すぐ招待します。Microsoftは、この電話でオスロ企業を完全にサポートします。 近い将来、これを行う方法に関するマイクロソフトからの詳細情報を期待できます。



それまでの間、詳細なQ＆Aに精通し、 開発者アカウントを作成することをお勧めします。



2015年前半にOpera Mobile Storeが完全に稼働すると、Nokia Storeは完全に閉鎖されます。



この投稿の最後に、素晴らしいアプリケーションを作成するための努力に感謝し、新しいサイトでの成功を願っています。 一緒にいてください。
