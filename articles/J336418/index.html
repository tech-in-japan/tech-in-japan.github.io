Hackathon Openfights Codility





こんにちは、Habr！



9月16日と17日に、モスクワはOpen Fights Codility hackathonを開催します。 これはディスカバリーの友人によって組織されており、Redmadrobotはモバイルの方向を監視し、チームを支援し、審査員の1人として評価します。



最初のオンライン段階では、銀行の小売顧客向けのクールなソリューションを考え出す必要があります。 それは、銀行のアシスタント、自動請求ツール、またはまったく別のものである可能性があります。 主な条件は、アプリケーションがクライアントの問題を解決し、銀行の製品との相互作用を改善することです。



GitHubで最高のアイデアとプロファイルを持つチームは、ワークステーションに2日間の集中的なハッカソンに参加し、提案された言語の1つとシークレットバンクAPIを使用して、実際に動作するモバイルアプリケーションまたはWebアプリケーションアプリケーションを独自の方法で作成します。



審査員は、アイデアの品質と拡張性、コードの品質、選択したアーキテクチャについてプロジェクトを評価します。



賞品



すべての参加者は、Otkrytieの開発者とモバイルバンキングおよびインターネットバンキング部門の責任者と、業界の専門家との非公式なコミュニケーションを行います（ グローバルファイナンス2017で最初の3位を獲得し、他のさまざまな賞を受賞しました）。 Redmadrobotの開発者と幹部（このような人のことを聞いたことがないのなら、これらは今年4つの評価を獲得している人たちです）と業界の他の価値ある代表者です。



受賞者には、当社、Netologia、Workstation、Electronic Cloud、JetBrains、およびその他のハッカソンパートナーから賞品が贈られます。 そして、もちろん、Otkrytieからの賞金は最高を待っています-150,000ルーブル、2番目に150,000ルーブル、3番目に100,000ルーブル。



ツール



参加者は幅広い言語のスタックで作業し、銀行のシークレットAPIにアクセスします。



次の言語から選択できます。



  • スイフト
  • コトリン
  • Python
  • Php
  • Java
  • Node.js
  • Javascript
  • .NET
  • Ruby on Rails


API：



  • IB / MB請求書（電子請求書発行）
  • ジオコーディング（最も近いATMおよびブランチの決定）
  • 為替レートを取得する
  • カード製品のデータの受信（トランザクションのリスト、現在の残高、レート）
  • すべての預金に関する情報
  • 現金ローンの利用規約


ハッカソンのメンバーになる方法



単独またはペアで参加できます。 必要なのは：



  1. 優れた開発者である（たとえば、GitHubのプロファイルで証明する）
  2. 興味深いアイデアを思いつきます（アプリケーションで簡単に説明してください）
  3. サイトに登録（締め切り-9月8日）
