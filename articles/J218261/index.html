Big Premiere Time：Microsoft BUILD Conferenceでのノキア

こんにちは、Habr！

ご存知かもしれませんが、最近、大きなMicrosoft BUILD Developer Conference 2014が開催されています。 2013年には、6,000人以上の開発者がこのイベントに参加し、最も印象的なイベントは、発表とデモンストレーションに加えて、印象的なインタラクティブライトショーNokia Lumiaでした。







ただし、今年はMicrosoftとNokiaの首位の規模が過去を大きく上回る可能性があります。WindowsPhone 8.1、主力スマートフォンであるNokia Lumia 930、および今後のベストセラー製品であるNokia Lumia 630/635がBUILDで紹介されます。



Windows Phone 8.1



新しいスマートフォンについて話す前に、そのソフトウェアコンポーネントであるWindows Phone 8.1に注目します。 モバイルプラットフォームの更新の詳細なレビューは、近い将来、Habrahabrの公式Microsoftブログで公開される可能性が最も高いため、重要な変更に焦点を当てています。







もちろん、主なイノベーションはCortana音声アシスタントです。これはWindows Phoneの標準検索を完全に置き換えます。 さらに、人気のあるHALOゲームシリーズの仮想アシスタントにちなんで名付けられたCortanaは、単なる音声検索ではなく、一種の人工知能を備えた本物のアシスタントです。 開発中に、マイクロソフトの同僚は、優れたアシスタントを最適にするものを見つけるために、最高のパーソナルアシスタントと数十のインタビューを行いました。 自然な音声クエリのサポートのおかげで、彼女はスマートフォン内およびグローバルネットワーク上で情報を見つけることができるだけでなく、1日を完全に計画し、アプリケーションなどを起動できます。







たとえば、Cortanaの機能は、予定の別の予定が以前に通信またはカレンダーに表示されていた場合、予定の要求の交差に応じて、予定の交差点を即座に識別して報告する機能によって示されます。 Cortanaは、指定されたパラメーター（「モスクワの良い寿司バーを見つける」）に従ってレストランを検索し、テーブルの予約、街の天気に関する質問への回答、スポーツ試合の結果の報告を支援することもできます。 さらに、アシスタントに「次回妹に電話するとき、新しい子犬について尋ねるよう思い出させる」という形式を思い出させるように指示することができます。







Cortanaはシステムに密接に統合されていますが、サードパーティのアプリケーションを使用して、たとえばFacebook、Skype、またはTwitterと対話することにより、その機能を拡張できます。 さらに、ユーザーはCortanaで処理できるデータを制御できます：興味、リマインダー、音楽、場所など。







Cortanaは、最初は米国市場でベータ版として表示され、その後、他の地域で徐々にサポートを受ける予定です。



待望のWindows Phone 8.1の革新は、通知センターです。 そのメニューは、上から下へ「従来の」スワイプで呼び出すことができます。 通知センターは、ソーシャルネットワーク、インスタントメッセンジャー、その他のアプリケーションからの最新のイベントを表示するだけでなく、設定や切り替えモードにすばやくアクセスできます。 同時に、各ユーザーは個人のニーズに合わせてクイックアクセスリストをカスタマイズできます（たとえば、Wi-Fi、Bluetooth、GPSスイッチを強調表示します）。







いくつかの素晴らしいオプションがプラットフォームインターフェースを取得しました。 まず、ユーザーは「タイル」の単色の代わりに単一の背景画像を設定する機会を得ました。 この場合、背景画像は静的で、上下にスクロールすると透明な「タイル」がその上を移動します。







次に、Windows Phoneのほとんどのデバイスに3列の「タイル」をインストールできるようになりました。 以前の3行の「初期画面」は、大画面のNokia Lumia 1520および1320の特権であったことを思い出してください。







第三に、組み込みのテーマのセット全体のおかげで、「ロック画面」をカスタマイズする機会です。 ユーザーがスマートフォンのロックを解除して通知を表示する方法に多様性を追加する必要があります。







アプリストアのインターフェースも、アプリの評価表示ページ、スタートページ、その他多くのメニュー項目の新しい外観など、多くの変更が加えられています。



Windows Phoneのキーボードは、かつてモバイルデバイスに最適なキーボードとして認識されていましたが、大幅に更新されました。 独自の名前であるWord Flowを取得したキーボードは、「スワイプジェスチャ」およびより高度なインテリジェンスのサポートを受けました。 彼女は現在、コンテキストによって単語を予測できるだけでなく、たとえば、テキストに絵文字を自動的に配置することもできます。







ちなみに、ボタン自体はタッチセンシティブ（物理的に読み取る）であるだけでなく、仮想であることができるため、デザイナーはWindows Phone 8.1で新しいスマートフォンを自由に開発できます。 これの鮮明な例は、主要な機能を管理するための同様の方法を受けた新しいラインアップのスマートフォンLumia 630/635でした。







これらのスマートフォンは、WP8.1の別の新機能、つまり2枚のSIMカードを使用する機能もサポートしています。 しかし、これに少し戻って。



Nokia LumiaのユーザーにアピールするWindows Phone 8.1の便利な機能には、さまざまなアプリケーションやシステムメニュー用の待望の個別ボリュームがあります。 これは、たとえば、スマートフォンの全体の音量とは別に音楽の音量を調整できることを意味します。 ちなみに、システム更新を伴う音楽とビデオアプリケーションは、Xbox MusicとXbox Videoの2つの独立したサービスに分割されます。



そして、開発者（そしてその結果として最終消費者）にとって最も重要な革新の1つは、Windows Phone 8.1とWindows 8.1の両方で実行できるユニバーサルアプリケーションのサポートになりました。 さらに、アプリケーションがそのようなアプリケーションである場合、ユーザーはWPおよびWindowsデバイスのいずれか（！）で実行するために1回購入するだけで済みます。



Windows Phone 8.1の革新的なリストには、次のものも含まれます。

•VPNおよびS / MIMEのサポート。

•DLNAのサポート。

•ハブ「People」でのソーシャルネットワークTwitterおよびFacebookの機能の統合の改善。

•WebGLをサポートし、HTLM5のサポートを改善したİnternetExplorer 11。

•バッテリー電力センスは、バッテリー電力を最も消費するプログラムを識別します。

•Wi-Fi Sense-パスワードを入力せずにSkype、Facebook、Outlookの連絡先と安全な接続を共有できる機能。

•静かな時間。指定した期間内にユーザーに連絡できる連絡先の輪を制限できる機能。

•無線によるアプリケーションの自動更新（Windows 8.1と同様）。

•NFCの作業の変更。

•メモリカードにアプリケーションとゲームをインストールします。

•WiFi Directをサポート。

•カレンダーの改善。







ノキアシアン



そして今、最も重要なことは、Windows Phone 8.1がWindows Phone 8を搭載した既存のすべてのLumiaモデルに、プロプライエタリなアップデートLumia Cyanとともに「無線で」広まることです。



Windows Phone 8.1の上記の革新に加えて、Nokia Lumiaユーザーは新しい画像操作機能にアクセスできます。 主な改善点はフォトギャラリーの表示に関係し、さらに、Creative Studio画像を操作するための高度なアプリケーションが表示されます。







Nokia Storiesアプリケーションも更新され、ビデオスライドショー形式で撮影した写真を共有できるようになります。







アップグレード後、Windows Phone 8.1ビジネスパッケージはすべてのLumiaデバイスでも利用できるようになります。これには、企業で使用する多くのアプリケーションや追加のデータ保護システムが含まれます。 このビジネスパッケージにより、新しいデバイスはビジネスデバイスのセグメントでの地位を強化することができます。



Lumia Cyanの更新により、Lumiaラインの新しいデバイスで導入されたSensorCoreテクノロジーのサポートが提供されます。 Bing HealthやFitnessなどのフィットネスソリューションで、スマートフォンセンサーからデータを受信するための根本的に新しいエネルギー効率の高い機会を提供します。







更新とともに、2つの開発者ツールが発表されました。



-いくつかのパートナー企業で既にテストされているSensorCore SDKは、SensorCoreテクノロジーを使用してアプリケーションを作成する際に開発者に役立ちます。



-Nokia Lumiaで画像を操作するための更新された開発パッケージImaging SDK 1.2 BETAは、画像およびソーシャルネットワークを操作するための次世代アプリケーションを作成する新しい機会を提供します。

本日より、パッケージはこちらから入手できます 。 新しいイメージングSDK1.2では、高度なイメージング機能とフィルター効果が導入されています。



今後の投稿で、新しいツールについて確実にお知らせします。



ノキアLumia 930









BUILDカンファレンスの初日のスターは、もちろん、新しいフラッグシップスマートフォンNokia Lumia 930でした。







スマートフォンには、解像度1920 x 1080ピクセル（ピクセル密度は441ピクセル/インチ）のClearBlackテクノロジーを備えた5インチOLEDスクリーンが搭載されています。 Gorilla Glass 3で覆われたタッチスクリーンは、伝統的に高感度という特徴があり、手袋をしたままスマートフォンを制御できます。







Nokia Lumia 930ケースは、シルバーまたはブラックのモノリシック金属フレームと、ブラック、ホワイト、オレンジ、またはグリーンのポリカーボネート製バックカバーの2つの部分で構成され、内部に含まれるコンポーネントを保護します。 このような素材とさまざまなカラーソリューションの組み合わせのおかげで、スマートフォンは明るいデザインの愛好家と厳格なソリューションのファンの両方にアピールできます。 ちなみに、スマートフォンのすべてのコンポーネントを含む金属フレームは、美観だけでなく、デバイスの全体的な強度を高めるためにも使用されました。 スマートフォンの寸法は137 x 71 x 9.8 mm、質量は167 gです。







スマートフォンは、クアッドコアQualcomm Snapdragon 800プロセッサと2.2 GHzのクロック周波数、Adreno 330グラフィックチップ、RAM 2 GB、内部メモリ32 GBに基づいて構築されています。



Nokia Lumia 930のメインカメラはPureViewテクノロジーを使用して作られており、解像度は20メガピクセル（1 / 2.5インチフォーマットセンサー）です。 カメラはZEISS光学部品とデュアルLEDフラッシュを受け取りました。 F / 2.4の最大絞り値のおかげで、スマートフォンは低照度で高品質の写真を撮ることができます。







さらに、高品質のカメラと4つのマイクのシステムを含むリッチレコーディングテクノロジーのおかげで、スマートフォンは完璧な音質でビデオを1080p形式で記録できます。







スマートフォンは、LTEネットワーク、Wi-Fi WLAN IEEE 802.11 a / b / g / n / ac、Bluetooth 4.0およびNFCなど、すべての最新の通信規格をサポートしています。



バッテリー容量は2420 mAhで、16時間の通話時間と9時間のビデオ視聴を提供します。 Nokiaの主力デバイスに適しているため、スマートフォンはQiワイヤレス充電をサポートしています。







Nokia Lumia 930の販売は、ヨーロッパ、アジア、中東、およびインドで6月に開始されます。

Nokia Lumia 630 / Lumia 635









将来のベストセラーであるエントリーレベルのスマートフォンであるNokia Lumia 630とLumia 635の特性とデザインは似ていますが、基本的な違いがいくつかあります。 Lumia 630には1つと2つのSIMカードをサポートする2つのバージョンがあり、Lumia 635はLTEネットワークをサポートします。 Lumia 630にはマットプラスチックケースもあり、Lumia 635には、「前の製品Lumia 625のように」「奥行き効果」のある2層の光沢プラスチックの交換可能なパネルがあります。







Windows Phone 8.1の2枚のSIMカードのサポートの実装は興味深いものです。各電話番号には、通話とメッセージ用の独自の「タイル」があります。 この場合、スマートフォンの設定で、ユーザーはデフォルトで連絡先から特定の番号への通話用のSIMカードの選択を設定できます。







両方のスマートフォンには、854 x 480ピクセルの解像度のClearBlackテクノロジーを備えた4.5インチIPSディスプレイが搭載されています。 これらのモデルは、クロック周波数1.2 GHzのクアッドコアQualcomm Snapdragon 400プロセッサーに基づいて構築されており、512 MBのRAMと8 GBの内部メモリがあり、microSDを介して（！）128 GBまで拡張できます。 Windows Phoneは、1 GB未満のRAMを搭載したデバイスで動作するように最適化されているため、ユーザーは制限なくアプリケーションやゲームを実行できます。







各デバイスのカメラを使用すると、5メガピクセルの解像度で写真を撮ることができます。これは、初期価格セグメントのモデルに適した指標です。 バッテリーの容量は1830 mAhです。







デバイスは、高価なLumiaスマートフォンに匹敵する機能を提供します。 デフォルトでは、Bing HealthとFitness、Microsoft Officeアプリケーションスイート、Nokia Camera、Nokia MixRadioの無料ストリーミング音楽、およびここから無料の世界地図とナビゲーションを備えたSensorCoreテクノロジーをサポートしています。







ロシアでは、Nokia Lumia 630が5月に1枚のSIMカードを搭載したデバイスで7,990ルーブル、2枚のSIMカードを搭載したデバイスで8,490ルーブルの価格で販売される予定です。



Nokia Lumia 635は、この夏から米国を含む多くの国で利用可能になります。



Lumia 630およびLumia 635には、白、オレンジ、黄色があります。



また、ロシア、ウクライナ、カザフスタンでは、各デバイスに黒い取り外し可能なケースが用意されています。
