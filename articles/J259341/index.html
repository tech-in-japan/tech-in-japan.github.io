MicrosoftはユーザーにWindows 10のアップグレードを強制する

マイクロソフトは昨日、Windows 10のリリース日を2015年7月29日に発表しました 。また、昨日、Windows 7および8/8 8.1ユーザーは、実行中のGWX（Get Windows X）プロセスのアイコンを更新エリアに表示できました。 GWXファイルはC：\ Windows \ System32 \ GWXディレクトリにあり、 KB3035583がインストールされている場合にのみ存在します。 このプロセスは、ユーザーがタイムリーな更新のためにWindows 10のコピーを「予約」することを示唆しています。 将来、GWXはWindows 10へのアップグレードに使用されます。







以前に、マイクロソフトはWindows 10のホームユーザー向けに、更新プログラムが利用可能になり次第、更新プログラムを配信するため、リリースされたすべての更新プログラムが1日待たなければならない、安全性の低いパッチ火曜日モデルを放棄すると書きました。 これで 、ホームユーザーの場合、ダウンロード後に更新プログラムが自動的にインストールされ、ユーザーがインストールを禁止または遅延できないことがわかりました。



上記の更新KB3035583（ 更新により、Windows 8.1およびWindows 7 SP1のWindows Update通知の追加機能が有効になります ）は、Windows 10にアップグレードするために必要です。GWXディレクトリには、config.xml構成ファイルが含まれます。 更新プログラムの存在は、インストールされた更新プログラムの一覧（[コントロールパネル]-> [プログラムと機能]-> [インストールされた更新プログラムの表示]）で確認し、KB3035583を検索することもできます。
