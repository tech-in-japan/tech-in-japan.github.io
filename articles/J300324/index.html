Baidu Videoは、ファンドとBaiduから1億5500万ドルの投資を集めました





中国の会社BaiduはすでにUberなどの企業に投資しており、事業の有望な分野を開発し続けています。BaiduVideoを別のサービスとしてリリースした後、サービスは1億5500万ドルの投資を受けました。 投資家には、Baidu、Shanghai New Culture Media Group、Tianshen Yule、Softbank Asia Infrastructure Fund、Redpoint Venturesが含まれます。



「Baidu Videoは、中国のプロフェッショナルビデオコンテンツのサービスとしてリーダーになることを目指しています。 現在、Baidu Videoはプラットフォームの開発に注力し、視聴者にサービスを提供し続けています」と、Baidu VideoのTechCrunch広報担当者は述べています。


アリババによるYouku Tudouの買収が完了した後、Baidu Videoが別の会社としてスピンオフしました。YoukuTudouは、最も人気のあるビデオサービスであるYoukuとTudouの背後にあります。 これらの企業は以前は合併していましたが、どちらのサービスもまだ別々のサイトとして機能しています。







Baidu Videoに加えて、BaiduポートフォリオにはiQiyiサービスも含まれています。これはBaiduが28億ドルで閉鎖会社を作ることができます-この噂は最近現れました。 Baidu Video事業の分離は正しい動きと考えることができます。したがって、Baiduはサービスに対する直接的な圧力を中和し、外部投資家を引き付けて、追加の経験と資金を得ることができます。 このため、理論上、Baidu Videoは独立したユニット内よりも依存度が低く、Baiduと対話します。これにより、プロジェクトをより迅速に開発できます。



潜在的な視聴者を考えると、Baidu Videoは将来、広告主にとって魅力的な分野でYouTubeと競合できる国際的なサービスになる可能性があります。 世界中の国々の市場のユーザーに関しては、検索エンジンや有名なブランドへの統合により、これまでのところGoogleのソリューションは先を行っています。
