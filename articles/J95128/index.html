新しいバージョンのコードのリリース:: Blocks IDE 10.5







数日前、C ++およびwxWidgets- Code :: Blocksで書かれたかなり人気のあるオープンIDEの新しいバージョンが（2年間の開発後）リリースされました。 主な開発言語はC ++です。 環境の機能を拡張するプラグインのシステムがあります。



多数のコンパイラをサポートしています。

  • ミンウ
  • デジタルマーズC / C ++
  • Microsoft Visual C ++
  • Borland C ++
  • ワトコム
  • Intel C ++コンパイラ


インターフェースのいくつかの機能の中で：

  • コードの強調表示
  • コード補完
  • 折り畳み
  • プラグインのサポート
  • ツールバーシステム（外観を自分用にカスタマイズできます）


必要なプラットフォームのダウンロードセクションでダウンロードできます。

ダウンロードリスト：

  • 最新バージョンのソース
  • SVNリポジトリ
  • WindowsXP 32ビット（23.3 MB）
  • Linux Debian 32ビット（18.6 MB）
  • MacOS（28.9 MB）


スクリーンショットセクションの古いバージョンのスクリーンショットをご覧ください。



コード::ブロック10.5変更ログ

最新バージョンでは、インターフェースが大幅に改善されました（現在、環境はクラッシュせず、アクションを実行しています）。 多くのマイナーなバグ修正。 コードの自動置換とデバッグモードの大幅な改善（私の目を引いた最初のことは、ベクターの内容を表示できることです）。



ソースからいくつかのスクリーンショットを収集しました。
