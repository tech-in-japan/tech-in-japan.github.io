New controllers



What is new and interesting in terms of iron? For example - a two-channel wireless switch of everything your heart desires. It is installed in the standard podozetnik, Wi-Fi, two outputs, two inputs, so that in the absence of a network, something can be turned on or off with the most ordinary switch.







What else? Thermostat Two outputs - one for switching the load on and off, the other for controlling either the warm floor, or a valve on the battery, or a conventional heater. It's harder with him, the temperature can only be set via the network.







Included - several weekly timers, several weekly profiles for temperature.



Or, for example, the further development of the decorative lighting controller.



4 outputs with PWM, output for address lamps, output for controlling external power supply or any load. Local control is an encoder, so it can also be turned on and off without a network. My ideology is such that the basic functions are available without any Internet and Wi-Fi.



Inside, there are also sleep and wake-up timers, various effects, regular timers with the inclusion of both light and effects in time, user presets and all that is different.

New software



And now about moving in a spiral.



At the very beginning, I wrote that I had come a long way from a PC application, a mobile application, and eventually settled on the Web version of the interface. But nothing can be done, life also did not stand still, and it turned out that universality is good, and specialization is convenient. Therefore, an application for the Android platform was developed. All settings, icons, pictures, and so on - the same as in the Web version, completely rework the interface - this, at this stage, is beyond my strength.







One of the important innovations is the ability to work locally, inside the home subnet, so now the absence of the Internet is not terrible. Just in the settings you can specify the name of the subnet and the local address of the devices, and if the phone is on this subnet, then the entire exchange goes to the local addresses.



All this in the minimum configuration works without any gateways, servers and other things. Enabled, connected to the router via the internal web interface, installed the application, downloaded the configuration and that's it. Fast, easy, convenient. If you need timers, scripts, access from afar - there is no way to do without the Internet.



Two types of widgets have also been developed. The first widget is a five-button, the second is one-button. Each button can be assigned any number of actions. Some sort of script server.
