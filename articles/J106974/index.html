Windows 7 SP1およびWindows Server 2008 R2 SP1リリース候補が正式に入手可能に

Windows 7およびWindows Server 2008 R2の最初のサービスパックの公式リリース候補は、公式にダウンロードできます。



マイクロソフトは、Windows 7の最初のサービスパックが目立った革新をもたらすことはなく、いくつかの新しいシステム機能をサポートする累積的な更新に過ぎないことを長い間繰り返しました。



Windows 7のサービスパックで最も重要な革新：
  • 外部サービスとリンクするための追加オプション。 Windows 7では 、組織間のプラットフォーム互換性と認証機能の両方を改善するために、 WS-Federationプロトコルをサポートするサードパーティサービスと対話するための追加サポートが追加されます。
  • HDMIオーディオデバイスの改善された作業。 コンピューターをHDMLオーディオデバイスに接続しているときに、ごくわずかな割合のWindows 7ユーザーで問題が発生しました。 システムの再起動後に、デバイスへの接続が切断された可能性があります。 この問題は、フィックスパックで解決されます。
  • XPSドキュメントで混合ページを印刷するときの問題を解決しました 。 Windows 7では、 XPSビューアーツールからXPSドキュメントを印刷するときに、ポートレートページとランドスケープページに問題があります。 印刷は1つのモードでのみ行われます。 この問題はSP1で解決されています。
Windows Server 2008 R2に関連する変更：



  • ダイナミックメモリテクノロジー
  • Microsoft RemoteFXテクノロジー。
  • DirectAccessを使用する場合の追加のスケーラビリティと高可用性機能。
  • エクストラネット上の管理されたサービスアカウント（MSA）のサポート
  • 待ち時間の長いネットワーク（インターネット）に接続されているドメインコントローラーへの認証トラフィックの増加した量を管理するためのサポート。
  • フェールオーバークラスターのディスクサブシステムの改善。
Windows 7とWindows Server 2008 R2の両方に影響する変更：
  • 「起動後に以前のフォルダを元に戻す」設定の動作が変更されました。 以前は、「起動後に以前のフォルダーを元に戻す」設定は、カスケードモードで開いているフォルダーを返していました。 最初のサービスパックをインストールすると、フォルダーは元の位置で開きます。
  • RRASおよびIPsecの認証タイプのサポートの拡張。 IKEv2プロトコルに新しいタイプの識別子のサポートが追加されました。これにより、IKEv2の追加の識別形式（電子メールIDや証明書サブジェクトなど）を使用した認証が可能になります。
  • プロセッサの高度なベクター拡張機能（Advanced Vector Extensions、AVX）のサポート。 Windows 7 SP1は、新しいAVXプロセッサ命令セットをサポートします。 このキットは2008年にIntelとAMDによって承認されました。 これらの命令セットをサポートする最初のプロセッサは2011年にのみ登場します。
これらおよびその他の変更の詳細については、 公式ドキュメントを参照してください。



このリンクを使用して、 Windows 7およびWindows Server 2008 R2 Service Pack 1リリース候補（KB976932）をダウンロードします。
