初心者向けのマイニングを開始する方法

記事のアイデアとタイトルは、最近の投稿に対するコメントの中でXaoによって提案されました。 この手順は、暗号通貨共同生成プールの一部としてWindowsまたはLinuxを実行している通常のコンピューターとラップトップでマイニングを開始するのに役立ちます。



dzzhが正しく述べているように

現在、ビットコインをマイニングすることは意味がありません。 いくつかの小さなコインを生成し、ビットコインを介して取引所に引き出します-はい。


したがって、暗号通貨を選択してマイニングを開始する必要があります。 この手順は、5つの簡単な手順で構成されています。

  1. フォーク選択
  2. プールの選択
  3. マイナーの選択
  4. マイニングローンチ
  5. ウォレットのまとめ


1.フォークの選択



独自のソースコードに基づく暗号通貨と同様に、ビットコインのかなりの数のフォークがすでに登場しています。 選択の主な基準は、 暗号通貨マイニングの収益性です 。 たとえば、私にとっての2番目の基準は、その流動性です。 いわゆる暗号通貨の使いやすさ 「リアルマネー」（ルーブル、ドル、銀行口座のユーロ）。

使用される暗号化アルゴリズムも重要です。 ビットコインはSHA-256に基づいており、ビットコインのマイニングは高価な農場の所有者にとって意味があります。 ただし、たとえば、 scryptはメモリを集中的に使用します。 これにより、暗号化暗号通貨用の特別なASICプロセッサの使用と作成が難しくなり、マイニング用にビデオカードを備えた従来のコンピュータを効率的に使用できるようになります。



採掘収益性


言い換えれば、計算の複雑さの単位ごとに獲得できる暗号通貨の量（ハッシュレートの単位として取得できるため、hash / sが測定されます）。

たとえば、1日または1か月あたりに受け取るコインの数など、自分の収益性を評価することは理にかなっています。

たとえば、私のNvidia GTX 770グラフィックカードを使用すると、フェザーコインを210 khash / sの速度でマイニングでき、プールから1日あたり約6 FTCを受け取ることができます。

次のリソースは、収益性の評価に役立ちます。

  • エネルギーおよび収益性計算機www.coinwarz.com
  • 一般的なフォークの比較表を備えた電卓（速度を下に入力してください） dustcoin.com




暗号通貨の流動性


フォークコースのタイムラインに関する以下のリソースは、フォークの評価に役立ちます。

  • coinmarketcap.com
  • www.coinchoose.com
  • bitcoinwisdom.com


または、ロシア語の取引所btc-e.comで取引されているフォークから選択するだけで、マイニング収益を「キャッシュアウト」します。 これにより、マイニングから利益を得るためのパス上の追加ステップが回避されます。 それでも取引所で取引されていないフォークを選択する場合は、 cryptsy.com交換機へようこそ。



2.プールの選択



マイニングから得られる利益の重要な値は、プールコミッションです。マイニング中にプールが獲得するブロックの量の割合で、残りのブロック報酬はプールのマイナー間で共有されます。 また、プールから財布へのコインの引き出しに関する取引からの手数料に注意を払う必要があります。 ワーカーを監視し、統計を表示するためにプールが提供する追加機能は重要です。

選択したプールで、ワーカーの登録とログインとパスワードの作成が必要です。 暗号通貨をマイニングするコンピューターと同数のワーカーがいることをお勧めします。 ただし、1人のワーカーで複数のコンピューターでマイニングを実行すると、その計算能力と結果が要約されます。



3.マイナーの選択



SHA-256、暗号化アルゴリズムを使用して暗号通貨をマイニングするには、一般的なマイナーのいずれかを使用できます。

  • cudaminer -Nvidiaグラフィックカード（GPU）のマイニング用
  • cgminer -ATIビデオカード（GPU）のマイニング用
  • pooler cpu miner （minerd）-プロセッサー（CPU）のマイニング用




4.マイニングの開始



たとえば、グラフィカルインターフェイスのないLinuxで cgminerマイナーを使用している場合は、コマンドラインに次のように入力します。 

./cgminer --scrypt -o stratum+tcp://_: -u Weblogin.Worker -p Worker_password
Weblogin.Workerはワーカーの名前で、Worker_passwordはパスワードです

この例では、プールがストラタムプロトコルをサポートしていることを前提としています。これは、すべての最新のプールの事実上の標準です。



CPUマイニング


ビデオカードがない場合は、中央処理装置でマイニングできます。

  1. Windows用のPooler CPU Miner 32ビットまたは64ビットをダウンロードします 。
  2. アーカイブを解凍し、作成したフォルダーに移動します。
  3. 次の内容のテキストドキュメントを作成します。 

      minerd.exe --url stratum+tcp://_: --userpass Weblogin.Worker:Worker_password


    Weblogin.Workerはワーカーの名前で、Worker_passwordはパスワードです
  4. 文書を保存してから、例えばminer.batに名前を変更して実行します


承認パラメーターとプールURLが正しく構成されている場合、起動後にプール内のマイニングに関する情報を示すウィンドウが表示されます。



5.ウォレットのまとめ



選択したフォークの公式サイトからウォレットをダウンロードします。

ウォレットにアドレスを作成して、プールで獲得したコインを受け取り、プールのアカウント-支払いセクションに入力します。

btc-e.comなどの暗号通貨取引所への支払いは直接許可されているため、まず選択して、選択した暗号通貨を補充するためのアドレスを登録および作成する必要があります。



取引から引き出しまでの手数料を忘れないでください-プールの残高から暗号通貨を引き出す頻度が高いほど、この手数料を支払う頻度が高くなります。
