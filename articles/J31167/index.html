改造：スチームパンクなスタイルのモニター。

前の記事で、 Jake von Slattが古いクラシックキーボードを芸術作品に変えた方法について話しました。



今回は、この人がどのようにスチームパンクスタイルのLCDモニターを作ったかを説明します。



読者の要望に応じて、私は彼の話を一人称で翻訳します。



私の新しいキーボードは、最新のモニターの前では正しく見えませんでした。 そこで、私はプロジェクトにDell 1907FPを使用することにしました。





分解すると、すべての詰め物がスチールケースに入っていることがわかりました。 新しい真鍮製フレームを作成するのではなく、目的の効果を得るために古いケースをペイントするだけにしました。 Rustoleumで金色の塗料を購入し、モニターを古風に見せるために薄い層を適用しました。





悪くない！ 結果は19世紀のスタイルに非常に似ています。 Sequential Glassから購入したガスランプホルダー。 実際、最初はこれらのホルダーをプロジェクトで使用するつもりはありませんでしたが、箱を開けるとすぐに、それらが便利になる場所をすぐに実感しました。





フレームに真鍮フレームが見つからなかったため、 True Valueで購入しました。 モニターフレームを作成し、ホルダーをネジで固定しました。 すべての準備が整った後、ネジの棚をドレメルで切り取りました 。





ラマはあまりにも「裸」に見えました。 それで、私は真鍮シートからいくつかのブラケットを作りました。





次に、ステープルの端を滑らかにし、直角に曲げました。





最初はボタン用の小さな金属製のセルを作りたかったのですが、それから古い時計のチャイムをレバーとして使用することにしました。





モニタースタンドを大理石にしたかった。 そこで、 Google Imagesを使用して、 大理石のキーワードで大きな写真を検索しました。そこで、この素晴らしいテクスチャを見つけました。 レーザーカラープリンターで写真を印刷し、4本の脚にある木製のベースに接着しました。 モニターの金属ケースと同じように塗装されたフレームでベース全体を覆った。





2本の0.5インチねじでホルダーをベースに取り付けました。









モニターをホルダーに取り付けるハンドルは、 True Value Hardwareのキッチンハンドルです。





背面のボタンで何かをする必要があります。おそらくニスです。
