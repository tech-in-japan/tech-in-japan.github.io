iOSおよびiOS開発センター向けFacebook SDK 3.0がリリースされました

FacebookはiOS向けのFacebook SDKの新バージョンを発表し、開発者にiOS 6の多くの新機能とサポートを提供しました。また、Facebookはすべてのツールとリソースを集めたiOS Dev Centerをコミュニティに導入しました。







Developers BlogのFacebook エントリには、iOS Dev Centerが、Getting Startedチュートリアル、Facebook統合でiOSアプリを作成する方法を説明するチュートリアル、およびに関するすべての質問を明確にすることを目的とする他​​の多くの資料を含む、新しい開発者向けの多くの資料を提供していると書かれていますFacebookのアプリケーションへの統合。 さらに、新しいデベロッパーセンターには、開発者がコーディングに必要とする可能性のあるすべてのリファレンスドキュメントがあります。



これに加えて、iOS向けFacebookでのログイン、iOS向けのフィードダイアログ、ネイティブアプリへのリンク、複数のアプリでのアプリIDの共有などのウォークスルーがあります。



SDKに関して、最も興味深いアップデートは次のとおりです。



  • セッション管理の効率を改善します。つまり、開発者がFacebookで認証を使用して操作しやすくなります。
  • プロフィール、Facebookプレイスなどからユーザーの写真を表示する既製のツール。
  • メモリ管理の改善を含む、ネイティブiOSプログラミング言語でのObjective-Cの機能サポートの改善。
  • Facebookの独自のアプリケーションプログラミングインターフェイス（API）に対する、すぐに使用できる改善されたサポート。




Facebookは、iOS6がリリースされるとすぐに、SDKが以前に発表されたネイティブのFacebookログイン機能の使用を開始することも約束しています。
