ウォームチューブゲーム：Warlords Classic for iOS

幼い頃、私は寒いノリリスクに住んでいました。 冬、厳しい霜の中で、公教育学部は「第1シフトの学生のクラスに出席しないように...」（同胞は理解します...）-これは「作動」と呼ばれました。 一般的に、私たちはほとんど勉強しませんでした。 しかし、私は母に行って設計研究所で働く機会がありました。そこで、私はIBM PC互換のコンピューターで1〜2時間プレイすることができました（ここでは-40度の風は邪魔ではありません）。 そして、私の好きなゲームはWarlordsでした...







ギフトコードについては、PMにお問い合わせください。 :(以上。



年が経ち（数十年）、すでに熟練したプログラマーである私は、iPhone用にWarlordsを書くことにしました（iPadはまだありませんでした... 2009年、秋）。 すべては問題ありませんが、iOSの開発経験はほんの数か月で、まだalloc / init / retain / release / autoreleaseの魔法を理解していませんでした。 しかし、これで私を止めることはできませんでした-2か月の作業（実際、トレーニングであることがわかりました）、ゲームの準備ができました（元の機能の80％が実現しました）。 100％が無作法ではないと決定し、ゲーム内のマップとグラフィックリソースをDOSバージョンから完全に切り取ったままにして、Knightsという名前でAppStoreにリリースしました（つまり、90％の無作法なコマーシャルを入手しました）。



2週間が経ちました...そして、メールボックスにはSteve Foknerからの手紙がありました。1998年にオーストラリアで生まれ、1968年に生まれたオリジナルゲームの作者です。 手紙では、すべてのインスタンスの裁判所は私に約束し、ペナルティを支払うことを要求し、AppStoreからすぐにゲームを削除しました。 Warlordsファンが私のゲームを見つけて、不足している機能を実装したり、マルチプレイヤーゲームを固定したりするようスティーブに書き始めたことがわかりました。 彼は後で彼が非常に驚いたと言った。



私はゲームをセールから削除する以外に選択肢がありませんでした。 しかし、私は勇気を出し、利益の一部を差し引くために、元の名前で同じゲームをリリースすることを申し出ました。 そして見よ、スティーブは喜んで同意した。 現在、AppStoreには最も公式なポートがあります。 誰もがその没収を忘れてしまいました。そして今、私は今後5年間、ゲームWarlords for iOSの独占権の所有者です。



数ヶ月が経ち、ゲームは低迷し、私はそれを忘れていました。 さらに、「二次コード麺」に変更を加えることはできませんでした。 私は他のプロジェクトを取り上げ、iOS開発者のスキルを高めました。



そして今、去年の春、それは私に現れました！ しかし、iPadは大きくて便利なiPhoneで、Warlordsをプレイするのに非常に適しています。 はい、AppleはGame CenterでTurn Based Games（タイムアウト付き）をサポートするiOS 6をリリースしました。 だから、あなたは少し血でマルチプレイヤーゲームを実装することができます。



ゼロから書き直すことにしましたが、いくつかのカードとGame Centerを介したゲームに注目しました。 奇妙なことですが、すでに多くの経験を積んでゼロから書くのにも同じ2か月かかりました。 そして6か月後、iPadゲーム（iPhoneとiPadの両方で実行）からユニバーサルアプリケーションを作成しました。その後、売り上げはほぼ2倍になりました。







ゲームには3枚のカードがあります-古典的なIlluria、Warlords 2のHadesh、および私の想像力-Krylantis。 ゲームはその「ペニー」をもたらし、ファンを喜ばせ、ベッドで私が既に知っているライバルに対して次の動きをするときに妻を怒らせます。



書かれたエディターの助けを借りて、私はWarlords 2とWarlords 2 Deluxeからマップを作成し続けます（まあ...移植）。 人々はそれが好きです。 すべての新しいカードは完全に無料ですが、ユーザーは何度も支払いたいと繰り返し書いています。 現在の5ドルの価格は私にとっては十分であり、追加料金を請求することなく作業を続ける価値があると思います。 秋にはエリテアとイスラディアが期待されており、すでにグラフィックスはWL2 Deluxeのものです。



肯定的なフィードバックとAppStoreの4.5つ星の正直な評価は、本当に心を温めます。 特に、新しいマップごとに盾を描くのが好きです。

国別のダウンロードは次のとおりです（ゲームは多数の言語に完全に翻訳されています）。





私は最近、iOSアプリケーションからの利益を3倍に増やす記事を公開しました。 進歩ではなく、単なるテクニックです。彼は、無料のアプリケーションのダウンロード数（したがって、コンバージョン収入）のアプリケーションのサイズへの依存を「証明」しました。 自信を持って、有料アプリケーションの場合、「このエンジンは動作しません」と言えます。 Warlords Classicのサイズを32 MBから9 MBに縮小しても、販売には影響しませんでした。



iPhoneの実装はあまり成功していないことを認めています-すべてが小さくて不便です、つまり ほとんど「単純な削減」に移植されました。 ただし、親族が数日間iPadを使用した場合でも、割り当てられた24時間以内に移動する機会があります。 おそらく私は状況を修正します。 いつか。
