Linuxコーダーライフ

完全なPhotoshopがないため、ほとんどの開発者はWindowsまたはOS Xを使用します。 しかし、Nixシステムに関心がある人はどうでしょうか？ 次に、興味があるかどうかを確認します。猫をお願いします。



私はFedora 22を使用するシステムとして、他のシステムで同じことをすることを誰も気にしません。



開発環境

彼らは味について議論しません、誰もがより便利な環境を使用します。 Windowsではメモ帳++を使用し、OS X Coda2ではFedoraでGeanyを使用しました。



インストールする： 



sudo dnf install geany geany-plugins-common geany-themes


多くの場合、サーバー上のファイルを直接操作する必要があります。サードパーティのftpクライアントを使用するか、nautilusを介して直接接続できます。



メモ帳++またはcoda2では、組み込みのftpマネージャーが好きで、少しグーグルで検索しましたが、geanyには同じプラグインが見つかりました。 プラグインは、メモ帳++で機能をコピーします



1）必要なパッケージをインストールしてプラグインをビルドします。 



 sudo dnf install gcc geany-devel libxml2-devel libcurl-devel openssl-devel


2） ダウンロードして、便利なフォルダーに解凍します 。

3）cdフォルダーに移動します

4）収集する 



 make && make install && make clean


その後、Geany設定でプラグインを有効にする必要があります。

このプラグインは、プロファイルの操作をサポートするという点で便利です。 ブラケットまたは同じSublimeのプラグインは、その方法を知りません。



残念ながら、プラグインはUbuntu 14.04、15.04のGeanyでは機能しません（他のシステムでは試していません）、何に接続されているのかわかりません。 プラグインの作成者に連絡しましたが、プラグインの開発は凍結されており、作成者が新しいシステム用にプラグインを書き換える可能性は低いと彼は言いました。 habrasocietyの誰かがプラグインを修正してくれたら、とてもうれしいです。

ここからプログラムをダウンロードしてください 。



このプログラムを使用すると、レイアウトを削減できるだけでなく、開発時間も短縮できます 。詳細については、 こちらをご覧ください 。 このプログラムには14日間の試用期間があります。 さらに、9ドルからのサブスクリプションが必要です。このプログラムがどれだけの時間を節約できるかを考えるとそれほどではありません。 しかし、2週間ごとに1回限りのアカウントを登録する必要はありませんか？

映像




さらに、きめ細かい画像処理を行うには、gimpとpintaをインストールします。 



 sudo dnf install gimp gimp-elsamuko gimp-data-extras pinta


これで十分でない場合は、 Photoshopを仮想マシンにインストールできます。 私の場合、情報はあなた自身の研究のためであるため、これは必要ありません。



FTPクライアント

FTPを操作するための2つのオプションがあります。

1）nautilusでftp接続を作成することにより、接続方法に関する情報がネットワーク上で簡単に見つかります。

2）サードパーティのftpクライアントを使用します。



FileZillaが好きです： 



 sudo dnf install filezilla


フォント



ほとんどのサイトは、Windowsでデフォルトのフォントを使用します。 インストールするには、 こちらからrpmをダウンロードしてください



このパッケージにはタホマはありません。次のように配信できます。 



 cd ~w get http://www.stchman.com/tools/MS_fonts/tahoma.zip sudo unzip -d /usr/share/fonts/truetype/msttcorefonts ~/tahoma.zip sudo fc-cache -f -vrm -f ~/tahoma.zip
