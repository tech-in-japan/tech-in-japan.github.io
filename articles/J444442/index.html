Jetson Nano：Nvidia Machine Learning Single Board





昨日、NvidiaはJetson Nano ：シングルボードAIコンピューティングコンピューターを発表しました。 CUDA-X AIライブラリをサポートする小型コンピューターは、472ギガフロップスを提供し、わずか5ワットで最新のAIワークロードを実行します。



シングルボードプレーヤーがGPUテクノロジーコンファレンスで紹介され、プレゼンテーションはNvidiaの創設者兼CEOであるJensen Huangによって行われました。



技術仕様：



  • GPU：128 CUDAコアを備えたNvidia Maxwellアーキテクチャ

  • プロセッサー：クアッドコアARM Cortex-A57 MPCore

  • ビデオ：エンコードおよびデコード用に、毎秒30フレームで4K（H.264 / H.265形式）および毎秒60フレームで4K（H.264 / H.265形式）

  • ビデオ出力：HDMI 2.0またはDP1.2 | eDP 1.4 | DSI（1×2）、同時に2

  • RAM：4 GB LPDDR4 64ビット; 25.6 GB / s

  • フラッシュメモリ：16 GB eMMC

  • カメラ：12ライン（3×4または4×2）MIPI CSI-2 DPHY 1.1（1.5 Gbit / s）、12x（モジュール）および1x（開発者キット）

  • コネクタ：1×1/2/4 PCIE、1×USB 3.0、3×USB 2.0

  • I / O：1×SDIO / 2×SPI / 6×I2C / 2×I2S / GPIOピン

  • ネットワーク：ギガビットイーサネット

  • OSサポート：Linux for Tegra

  • モジュールサイズ：69.5×45 mm

  • 開発キットのサイズ：100×80 mm

  • 接続：260ピンコネクタ


Jetson Nanoには2つのバージョンがあります。



  1. 開発者、メーカー、愛好家向けの開発キットは99ドル。
  2. 129ドルで大衆市場向けのシステムを作成したい企業向けの既製のモジュール。


Jetson Nanoは高解像度センサーをサポートし、複数のセンサーからの情報を同時に処理し、複数のニューラルネットワークを同時に実行できます。 また、多くの一般的なAIフレームワークもサポートしているため、開発者はお気に入りのモデルとフレームワークを統合できます。



Nvidiaによると、安価なシングルボードカードは「以前はもっと高価なボードを買う余裕がなかった企業、スタートアップ、研究者に最適」です。 したがって、Jetsonプラットフォームは視聴者を大幅に拡大し、AIアクセラレータは実際にはほとんど消費者向け製品になります。 少なくともこのメーカーはこのメーカーを購入できます。 Nvidiaによると、このボードは「最新のAIの力を安価なプラットフォームにもたらし、メーカー、発明者、開発者、学生からの革新の新しい波を刺激します。 以前は不可能だったAIプロジェクトを作成し、既存のプロジェクトを新しいレベルに引き上げることができます-モバイルロボットとドローン、デジタルアシスタント、自動化されたデバイスなど。



このキットにはフルデスクトップLinuxのサポートが付属しており、多くの一般的な周辺機器およびアクセサリと互換性があります。 これを理解するのに役立つ参考書も用意されています。 最後の手段として、同僚が技術的な質問に答えるJetson開発者フォーラムで質問をすることができます。



これは、Jetsonファミリーの最初のNvidia製品ではありません。これには、スタンドアロンマシン用の強力なJetson AGX Xavierシステムと、組み込みアプリケーション用のJetson TX2 （AI at the Edge）も含まれています。









Jetson AGX Xavier



比較のために、Jetson AGX Xavierは、テンソルコアを備えた512コアVolta GPUで動作し、ディープラーニングアクセラレータ、16 GBのメモリ、特殊なコンピュータービジョンアクセラレーター（7ウェイVLIWビジョンプロセッサー）、最大8 Kの解像度でビデオをエンコードし、同時にデコードする機能があります最大12ストリーム4K。 この多くは、Jetson Nanoシングルボードにはありません。 しかし、それはJetson AGX Xavierよりもはるかに小さくて安価です：87×100 mmに対してわずか69.6×45 mmで、価格は1299ドルに対して99ドルです。 ことわざにあるように、違いを感じてください。



Jetson TX2ファミリーの3番目のメンバーには、TX2（8GB）、TX2 4GB、およびTX2iの3つのバージョンがあります。 また、フラッシュメモリの容量（16〜32 GB）と、内蔵Wi-Fiの存在も異なります。Wi-FiはTX2（8GB）のみに搭載されています。









Jetson TX2



Jetson TX2は、256のNvidia CUDAコアでPascal GPUを実行します。 他の特性とサイズ（87×50 mm）によると、Jetson TX2ボードはJetson Nanoに似ていますが、コストが大幅に高くなります：約600ドル（開発キット）。 そのため、Jetson Nanoは数百ドルよりも安価です。これはNvidiaにとって特別なものです。
