コンソールゲーム：ビデオゲームへの愛がどのように面白いゲーム開発者に成長したか

こんにちは、Habr！

今日、私たちはブログのためにやや非公式の物語を共有することにしました。 それは、子供時代のコンソールビデオゲームへの愛が、かなり珍しいゲームプロジェクトを作成するための原動力となった1人の開発者についてのものです。







Christer KaitilaまたはMcFunkypantsは、彼の友人が彼に電話するのを好むように、AtariとTronのコンソールが支配したときに成長しました。 そして、もちろん、ビデオゲームの流行は彼を追い越しませんでした。 しかし、大半の8ビットの戦いでは楽しい思い出しか残されていませんでしたが、クリスにとっては、彼のキャリア開発は前もって決定されていました。







今、ゲームの経験豊富な開発者として、古き良きビデオゲームとgamedevコミュニティに敬意を表することにしました...少し前、Chrisはタワーディフェンスゲームを作成するためのオープンソースツールを導入しました。 さらに、彼は毎年恒例のチャリティーゲームジャム開発イニシアチブの作成者でもあり、開発者がゲーム内購入を通じてチャリティー子供の資金に寄付する可能性のあるゲームを作成することを奨励しています。



夢がかなう



「 そのような人がいますが、私は常にビデオゲームを作成することを夢見ていました。 この願望は、アタリ2600年の時代から生まれました。私が子供の頃に幸運だった時代です。 そして一般的に、ゲームの成長と発展と同時に成長すること-それは素晴らしいことです。あなたはこの夢を見ることができるだけです。 そして、それはgamedevへの私の情熱をより強く燃やしただけです」とクリスは言います。



彼の最初の開発経験は「蛇」でした。これは、TRS-80ココセットトップボックスの最初の映画「トロン」のオートバイに触発されました。



「 これは、カスタムハードドライブがまだなかった時代であり、オーディオファイルにソースファイルを記録していました。 「。



ティーンエイジャーがビデオゲームにどれだけのお金を費やしているかを見て、クリスはその上で良いキャリアを作ることができることに気付き始めました。



それが彼の主な目標になりました。



開発者としての彼の自己啓発の始まりは、彼がフリーランスに従事していたゲームプロジェクトとともに、小学校でのんびりしました。 ゲーム開発者をさらに深く掘り下げて、彼は新しいプログラミング言語、多くのフェスティバルやハッカソン、そして何十ものアプリケーションを学びました。 しかし、Chris自身が観察しているように、サードパーティ企業が彼を開発者として雇い始めるためにゲームのポートフォリオを作成するには10年かかりました。



タワーゲームスターターキット



昨年末、クリスは開発者の伝記に新しいページを開きました。 Woot Studioと協力して、彼はやや非標準のプロジェクトであるTower Game Starter Kitをリリースしました。 これは、Windows Phone、Windows 8、およびHTML5対応ブラウザー用に独自のタワーディフェンススタイルのゲームを作成するための無料のクロスプラットフォームオープンソース開発ツールキットです。







「 タブレットとスマートフォンの両方で同様に機能し、最も重要なこととして、レンダリングに高いパフォーマンスを必要としないシンプルなものを作成したかったのです。 そのため、この開発キットは判明しました 」とChrisは言います。 「 Windows Phone向けのHTML5ゲームの開発方法について、あらゆるバックグラウンドを持つ開発者に学んでもらいたい 」



「 Woot Studioと一緒にTower Game Starter Kitを作成するとき、「プロトタイププロジェクト」、つまりVisualの完全に機能し認定されたプロジェクトであるMVP（最小限の機能を備えた製品）を含めるとよいことに同意しました。 Studio。コンパイルしてApp Storeに投稿できます。







このツールキットを使用すると、誰でも、特別なプログラミングスキルを持っていない人でも、ゲームの開発を開始できます。



「 ブラシを持っている人は誰でも描画を開始できます。 また、Tower Game Starter Kitを使用すると、初心者でも誰でもゲームを作成できます。 当然、アプリケーションを作成するスキルがあればあるほど、ゲームを作成するのが簡単になり、すべてがより明確になります 」とクリスは言います。



「 実際、新しいレベルを変更して作成し、新しいグラフィックやテクスチャを追加し、コードに触れることなくサウンドトラックを編集できます 」と彼は付け加えます。





タワーゲームスターターキットのレビュー



では、なぜこのプロジェクトは無料で配布されているのですか？



「 このプロジェクトの主な目標は、HTML5を使用してWindows Phone用のゲームを作成するために、できるだけ多くの人々を引き付けることです」とChrister氏は説明します。 「 さらに、Tower Game Starter Kitのすべての機能にもかかわらず、これはまだトレーニングプロジェクトであり、ほとんどのゲーム開発者にとって一種の「プローブ」です 。」







ツールキットの使用を開始する前に、ChrisはTower Game Starter Kitを使用して作成された多数のゲームをよく理解することをお勧めします。 たとえば、Windows 8用のTiny Tidal Tower Defense 。



コンペティション「1か月、1ゲーム」



クリスはすでに人生で何かをしているように見えますが、ゲームに関連するすべてのことに対する彼の飽くなき欲求は、彼が別のプロジェクトを作成することになりました-競争「 One Month、One Game 」。 これは、開発者が年に12回ゲームを作成したり、ゲームの作成に参加したりするための動機付けとなる興味深いアクティビティです。



現在、プロジェクトサイトには約8,000のゲーム開発者のプロファイルがあります。 同時に、サイトへの登録は非常に簡単です。必要なのはTwitterアカウントだけです。



「 このコンペは最高のゲームを競うというよりも自分とのコンペであるため、参加者をルールに限定しませんでした 」とクリスは言います。 「 プロジェクトの重要なアイデアは、開発者がアプリケーションを作成するよう動機付け、それによってスキルを向上させることです 。 朝の定期的な運動があなたを強くするように 。」



もちろん、すべての人が月に1つのゲームを作成するという課題に対処できるわけではありませんが、クリスはいずれにしても、プログラムに参加する価値があると考えています。 「 1つのゲームを作成したとしても、競争への参加は依然として意味をなします。 あなたの分野で成功する唯一の方法があります-練習すること。 それが「1か月、1ゲーム」の要点です。



コミュニティクレジットの返却



「 私は、それが私のためにしてくれたことに感謝して、コミュニティに何かを返す機会を持つ数少ない幸せな人々の一人です。 それを私の趣味と考えてください、とクリスは言います。 「 私にとって、ゲーム開発者であることはロックスターのようなものです。」 私は夢を生きています。 そして、あなたもアイデアが人々を助ける開発者の一人になることができます 。」



冒頭で述べたように、クリスは年に一度のチャリティーゲームジャムイニシアチブを作成しました。



次回は、今年の11月にイベントが開催されます。 彼のTwitterアカウントからイベントに近いすべての情報を見つけることができます 。



さらに、クリスのプロジェクトはすべて、彼の個人ウェブサイトで見つけることができます。



ところで、Tower Game Starter Kitに興味がある場合は、コンテストの1つに参加して、DVLUPプロジェクトで追加ポイントを獲得できます。
