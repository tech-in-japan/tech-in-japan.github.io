IFA 2013での最初の日、私は自然にSamsungブースに急いで行きました。そこではヨーロッパ人が最初に展示され、Galaxy Note 3スマートフォンを手に持っていました。同じ日に東芝とパナソニックのタブレット、TCL / AlcatelとArchosスマートフォンに精通しました。 多少の遅れはありますが、個人的な印象や写真を共有しています。









サムスン


サムスンパビリオンは、IFA 2013で最大規模であり、独自のステージと多くのテーマスタンドを備えていると予想されていました。 私の計算によると、少なくとも200個のSamsung Galaxy Note 3が展示されていました。







これらのスマートフォンには多くの人がいましたが、展示されているコピーの数が多いため、誰もが文字通り目新しさを手にすることができました。



Galaxy Note 3


そして私は言わなければならない、私はデバイスが好きだった。 出来栄えは標準です。







周囲の金属ストリップは波形であり、明らかに、これはデバイスが手のひらにさらに密着するように行われています。 ちなみに、スマートフォンのサイズに違和感はありませんでした。おそらく手の大きさのせいでしょう。 しかし、Sony Xperia ZUの管理は本当に難しいです。































Galaxy Note 3にはいくつかのカラーオプションがあり、フロントパネルの装飾が大きく異なります。 たとえば、ダークグレーバージョンのフロントパネルのデザインは、つや消しアルミニウムとの関連性をもたらし、白とピンクのバージョンでは、パターンはすでに幾何学的です-ストライプの菱形。































以前、Samsungのスマートフォンを扱う必要がありましたが、私の意見では、Galaxy Note 3が最初であり、その金属デザイン要素は実際に金属のように見えます。 それらは非常に注意深く作られており、特に底部のスピーカーの金属メッシュ、聴覚スピーカーを覆うわずかに突出した金属ストリップ、および一体型のボリュームコントロールスイングが気に入っています。 最後に、サムスンは安く見えません！



部分的に「高価な」外観は、実際に革でできている裏表紙により実現されています。 非常に興味深いソリューションですが、実用的ですか？ 事実、白いタブレットのGalaxy Note 10.1に行くと、その背面パネルも革でトリミングされているので、黒い汚れに不愉快な驚きを覚えました。 展示サンプルは革（！）スタンドの上に置かれ、しっかりと固定されていることに注意してください。 絶対に落ちませんでした。 手が汚れただけで肌がとても汚れたことがわかりますか？ そして、バッグやバックパックの底にスマートフォンやタブレットを投げるとどうなりますか？ そして、あなたがいつもそれを着るなら？ すでに自宅や車の中で、展示会のように、デバイスは明らかにきれいな白い革のカウンターの上に横たわることはありません...







当然、蓋を慎重に扱うことができ、すべての汚れがはがれますが、私の意見はこうです：常に持ち歩くモバイルデバイスでは、皮膚は実用性においてプラスチックより劣っています。 白とピンクのバージョンでどれくらい「伸びる」かは大きな疑問です。 この場合、暗い灰色のGalaxy Note 3を購入することをお勧めします。暗い肌の汚れは、非常に長い間目立ちません。



Galaxy Note 3にはLEDイベントインジケーターがあります。 フロントパネルの下に隠れており、拡散光で燃えます。 おもしろいことがわかります。最初は「焦点を合わせたい」のですが、目を細めると、最初は「シャープネス」が提供されていなかったことがわかります。 同様に、「トリック」はこれです。







サムスンギャラクシーノート3について話すと、彼のために特別にリリースされたカバーについて誰も言うことができます。 私もそれらがとても好きで、ここにあるのは、スマートフォンの背面壁がケースの壁に非常に近いため、スマートフォンとケースが一体となっているように見えることです。 数秒間、ケースとスマートフォンを接続する場所を見つけようとしました！ そして、結局のところ、ケースの裏壁-スマートフォンのカバーのように、区別することはできません！ このような「密接な関係」のGalaxy Note 3と保護デバイスの開発者-5つのポイント。

ギャラクシーギア




Galaxy Note 3の後、私はすぐにGalaxy Gearウォッチに切り替えました。これはSamsungが非常に誇りに思っているものです。







私の意見では、誇りに思うものは何もありません。 デバイスは重く、かなりかさばったことが判明しました。 私の近くの時計を長い間見ている数人の女の子が同じことを話しました-サムスンは非常に大きな時計を持っていました。 これは、Galaxy Gearが男性よりも女性の視聴者に焦点を合わせているという事実にもかかわらずです。 ターゲットオーディエンスからのそのようなフィードバックは次のとおりです...























ちなみに、Galaxy Gearは私の手にかろうじて固定されていました。 彼らは男性の手のためではないという別のヒント？











タッチスクリーンを制御するのも不便でした。 画面が小さく、指が大きい場合、たとえば設定で、3つのサブメニュー項目のストリップが画面に表示されている場合、各サブ項目の真ん中を正確に狙う必要があります。そうしないと、計画した間違ったことを完了する可能性が非常に高くなります。 時計のこのような機能に慣れるだけの時間がなかったのかもしれませんが、一方で、Galaxy Gear 3の背後にあるGalaxy Note 3の半分のコストを「ロールオフ」し、他の何かに慣れなければならないのはなぜですか。 iPod nanoの管理方法をほとんどすぐに見つけたのに、10〜15分でGalaxy Gearをマスターできなかったのはなぜですか。 確かに私はしたくなかったからではありません。 一言で言えば、私は韓国人にインターフェースで作業することを勧めます。 私の意見では、独自のメニューを備えたテンプレートAndroidはこのようなデバイスには適していないため、システムを大幅に変更する必要があります。



































ちなみに、時計のボタンは1つだけであり、これもGalaxy Gearであるポータブルコンピューターでの作業を容易にしません。



特異な潮の領域の時計の片側にカメラの目があります。



















この配置により、Galaxy Gearは「スパイ」撮影を行うための優れたデバイスになります。下のビデオをご覧ください。近くに立っている女の子は、私が彼らを撮影していることにさえ気づきませんでした！







ただし、通信の場合、時計を身に着けている人はストラップではなく画面を見るため、カメラをフロントパネルに配置する方が論理的です。 なぜサムスンがこのストラップで庭をフェンスで囲うことを決めたのかは、私にはまったくわかりません。



そして最後に、ソフトウェア。 以下のビデオを見ると、Galaxy Gearのソフトウェアコンポーネントが何であるかを完全に把握できます。







私は、その性能または外観のいずれかによって、サムスンの時計がまったく好きではなかったと言わなければなりません。 私は彼らの中に、韓国人が誇りに思ったり、自慢することさえできなかった。 追加の機能（歩数計など）は二次的であり、主な機能のいくつか（ここでは主にカメラの魚眼レンズ）はかなり不器用に実装され、メニューは完全に直感的です。 2009年にLG GD910ウォッチを見たとき、よく覚えています。 しかし、今、4年後（！）、サムスンは実際に同じことを提供しています。 はい、申し出だけでなく、素晴らしいファンファーレを提供します！ 同様に、私たちは開発し、働きました、すべてがあなたのためです。 Galaxy Gearは、スマートフォンなしでは無力であるとさりげなく言及していますが、2つ目は、Galaxy Note 3およびGalaxy S 4とのみ互換性があることです。 ソニー自体は、その時計から誇大広告を作成していませんが、その間、第二世代に成長しており、このアプローチがはるかに好きです。 ペブルは言うまでもありません。







一言で言えば、私はギャラクシーギアをサムスンの大きな間違いと呼ぶ傾向があります。 おそらくこれは会社にも理解されているでしょう。さもなければ、このデバイスは販売されないという噂がどこから来たのでしょうか。代わりに、6か月後にGalaxy Gear 2の修正されたコピーが表示されますか？

東芝




IFA 2013で、東芝のタブレットを発見しました。これまでにタブレットに出会ったことはありませんでした。 私は認めなければならない、私は日本の会社のタブレットのライン全体が好きだったが、その理由を説明するのは難しい。 それらはデザインの面で傑出したものを何も表しておらず、ソフトウェアも同じタイプのものです。例として、ニッチと価格グループに明確な区分のみを置くことができます（フラッグシップモデルはExcite Pro、予算モデルはExcite Pure、いくつかの中間オプションはペン入力を提供するExcite Writeです）仕様（Excite ProにはRetinaディスプレイとHDMI出力があります）。



もちろん、最も興味深いのは、NVIDIA Tegra 4プラットフォーム上に構築されたExcite Proで、その特性といくつかのレイアウト機能を下の写真で評価できます。



















そして今、パフォーマンスについて。 AnTuTu、Quadrant、Vellamoの3つのベンチマークでデバイスを駆動することができました。

パナソニック




パナソニックのモバイルハイライトは、今年初めに発表された4K画面の20インチタブレットでした。 デバイスは4Kタブレットと呼ばれます。







このデバイスは、アスペクト比15:10、解像度3840 x 2560ピクセルのIPS Alphaパネルを使用しています。 対角サイズが20インチの場合、ピクセル密度はまったく印象的ではなく、わずか230 ppiです。







タブレットの仕様リストには、1.9 GHz Intel Core i5-3437Uプロセッサー、NVIDIA GeForce 745Mグラフィックスカード、4または8 GBのDDR3 RAM、128または256 GB SSD、Wi-Fi 802.11a / b / g / nおよびBluetoothアダプターが含まれます4.0、1つのUSB 3.0ポート、1つのSDXCスロット。 ドックコネクタも提供されます。 追加料金で、4KタブレットにAnoto Penデジタルペンとスマートカードサポートを装備できます。







タブレットを使用することは不可能であり、システムへのアクセスがブロックされました。 実際、美しい写真を検討するか、ペンの仕事を評価するかのいずれかでした。 ただし、注意すべきことがあります。 まず、デバイスの画面は本当に良いです。 しかし、これは明らかですが、さらに...







4Kタブレットは一種のレイアウトの奇跡です。 あなた自身の判断：ケースはタブレット、アクセサリーのように薄く、伝統的なラップトップのように、画面は一般的に（今のところ）この世界のものではありません。 その結果、多くのニュアンスがあります。 デバイスはかなり重いことが判明し、自宅から職場まで毎日持ち運べるほどポータブルではありませんでした。 さて、彼はいつもテーブルに横たわっていると思いますが、その後、側面にあるコネクタへのアクセスが著しく悪化します。 また、USBフラッシュドライブを挿入すると動作する可能性があり（その後、厚くない場合のみ）、メモリカードの貼り付けが問題になります。 「フラット」な位置では、通気孔とスピーカースロットが重なるという事実については話していない。



















私の意見では、別の疑わしい解決策は、左側パネルの斜角部分にLEDインジケータを配置することです。 繰り返しますが、タブレットが水平面に置かれている場合、これらのLEDはテーブル上でちらつきます... そして、ライトだけでドライブ、メモリカード、電源回路の状態を判断することが可能になります。 それらをデバイスの前面に持ってこなかった理由は明らかではありません。







また、このような大きなデバイスの場合、宣言されたバッテリー寿命（約2時間）では十分ではないと思います。 しかし、これはささいなことです。



一般的に、4Kタブレットの幅広い商業的見通しは見ていませんが。 少なくとも現在のバージョンでは。 はい、おそらくデバイスは著名なデザインスタジオや建築スタジオによって注文されますが、これは需要を満たすでしょう。 一方、誰かが最初であり、最初に気になるはずです...まあ、あなたは知っています。



TCL /アルカテル




中国の会社TCLは、IFAのフラッグシップスマートフォンであるアルカテルワンタッチヒーローで「撃たれ」ました。











一方で、デバイスの特性は非常に良好です-疎油性コーティングを施した6インチFull HD OGS画面、13メガピクセルのカメラ、2 GBのRAM、一方で、マルチタッチは5回のタッチのみで、内蔵フラッシュメモリは最大16 GBに達します。 一言で言えば、旗艦は旗艦のままですが、その予算カテゴリにあります。







デバイスのアセンブリは優れており、パネルは互いにしっかりと取り付けられています。 展示サンプルが非常によくできていれば、コモディティデバイスの製造レベルは少なくとも悪くはないと予想できます。 デバイスは手にしっかりと収まりますが、サイズが大きいため片手で操作することができません。 公平を期すため、ジェスチャを使用してデバイスの一部の機能を制御する機能を開発者が管理していることに注意してください。



























Alcatel One Touch Heroの背面パネルは、高品質のマットプラスチックソフトタッチで作られており、実質的に指紋が残りません。







残念ながら、スマートフォンはテストできませんでした; TCLはTCLスタンドで動作しなかったか、インターネットが切断されました。 ただし、このスマートフォンがMediaTek MT6589Tプラットフォームを使用していることは間違いありません。その機能はよく知られています。



フラッグシップモデル専用に設計されたMagicFlipアクセサリーは、Alcatel One Touch Heroの隣に展示されました。LEDスクリーンが組み込まれたカバー、E Inkスクリーンが付いたカバー、ワイヤレスバッテリー充電を提供するカバーです。











IFAでのもう1つのTCLプレミアは、低価格のスマートフォンであるAlcatel Pop Cシリーズです。C1、C3、C5、C7です。 これは、モデルL1、L3、L5、L7、L9がある低価格のスマートフォンLG Optimus Lのラインに対する一種の反応と言えます。 同意します。名前のこのような一致がランダムと見なされる可能性は低いです。 特に、LG Optimus Lを発売するというアイデアが、高い売り上げによってその有効性を証明した後です。



実際、Pop Cについて語るのに特別なことはありません。マルチカラーデバイスはプラスですが、ケースの素材は非常に安価であり、デザインは表現力に欠けています-これらは欠点です。 機能は予算クラスに対応しています。















ディスプレイ上の唯一のアルカテルタブレット-One Touch Evo 8 HD-は、外観がまったく気に入らなかった。 まず、カバーの色：それらはある種の派手で「緊張」しています。 第二に、素材で：汚れやすいフロントパネル「優れた」は、ブランドと滑りやすいバックパネルによって補完されます。 画面の明るさは私には不十分なようでした。















Alcatel One Touch Evo 8 HDのソフトウェア部分-興味深いことはありませんが、タブレットはAndroid 4.2で動作します。 技術仕様も際立ったものではありません：デュアルコアRockchip RK3066 1.6 GHzプロセッサー、1024 x 768ピクセルの解像度を持つ8インチIPSスクリーン、1 GBのRAM、4 GBのフラッシュメモリ、2および3 MPの解像度を持つカメラ。 唯一の良いニュースは、バッテリー容量が印象的な4150 mAhであることです。 このデバイスの自律性が完全な順序であると仮定する必要があります。











アルコス




むかしむかし、タブレットがまだすべてを実行していなかったとき、Archos製品を実際に見て、それらは宇宙のように見えました。 時間が経ち、業界が変わり、Archosはスマートフォンの生産を開始しました...そして今、私は本当に時計を戻したいと思っています。IFAで会社はスマートフォンを製造しない方法を示したからです。







スマートフォンのArchosシリーズは、チタン、プラチナ、および酸素の3つのシリーズで構成され、8つのモデルが消費者に提供されます。 バジェットデバイスは第1シリーズに属し、第2シリーズの代表は「エレガントなデザインと最高のパフォーマンス」を特徴としていますが、開発者の定義によると、「酸素」スマートフォン（これまではラインが1つのモデルのみで表されていました）は、高速で効率的で手頃な価格である必要があります。



Platinumシリーズのデバイスのことを知り始めたのですが、結局それらになりました。 なんで？ すでに2番目のデバイスの後、Archosはスマートフォンを拾いたくなかったからです。 私は、ヨーロッパでかなり発達したブランドを使用している会社が、最も安い中国のOEMを旧世界市場に供給することを決めたと感じました。 明らかに、彼らはすべてを節約しました：機器（下の写真の特性を参照）、および仕上げ材。 後者は、フロントパネルとリアの両方のデザインで使用されている汚れやすい光沢を率直に混乱させました。 そして、簡単に汚れます-それを穏やかに置くために！ ケースの「半分」の適合は重要ではなく、エレベーターがあります。 最後に、Archosスマートフォンの設計が欠落しています。
