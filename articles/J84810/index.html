Weborama + Java



habrahabr .ruのほとんどのユーザーは、音楽を聴くことに時間を割いていないと確信しています。

古い曲がすっかりうんざりしていて、新しい何かを聞きたいとき、役に立つWebサービスが役立ちます。

サイトからポピュラーな曲を直接聴くことができる多くのWebサービスがあり、その中からwebrama （サイトプレーヤー）を選択できます。



ただし、この記事はWebフレームで音楽を聴くのがどれほど便利かについてではなく、JavaとWebフレームを使用してポピュラー音楽のコレクションを整理するのに役立ちます。





そして、すべてが非常に興味深いものです。 ウェブパノラマは音楽を聴くのに非常に便利なサービスですが、音楽をダウンロードすることはできません。

音楽をダウンロードする方法は？



幸いなことに、私たちのサイトから音楽をダウンロードすることは提供されていません。 しかし、あなたはそれを自分で聞いて他の人を喜ばせることができます。





だから 、私たちのプログラムが何をすることができるかを決めることは残っています。

以下はプログラムの主な機能です。

  • ジャンル別の人気曲をダウンロードします。
  • アーティストによる曲のダウンロード。
  • このリクエストの曲を検索
  • リクエストが正しくない場合でも、プログラムはアーティストまたは歌を見つけようとします
  • すべての「重い」アルゴリズムは、個別のスレッドで実行する必要があります


基本的な機能により、「プラス」が残ることは明らかです

  • ID3v1（ID3v2_2もサポート）タグ
  • タグのないすべてのmp3は、プログラムの起動時に削除されます
  • プログラムは、同じアルバムに同様のトラックがあるかどうかを判断します
  • プログラムは同時ダウンロードの数を制限できます
  • プログラムはインストーラーとしてユーザーに提供される必要があります


これをすべて実装するために必要なものについて:)

  • Java開発キット-JDK
  • mp3タグファイルに追加するライブラリ-jid3lib
  • 開発環境-NetBeans IDE
  • インストーラーとランチャーを作成するプログラム-install4j


メインプログラムウィンドウの表示



ユーザーへの製品の提供

通常のユーザーにむき出しのjarまたはjar + batを提供するのは正しくありません。多くの場合、通常のユーザーがjava（はい、超人）で抱えている問題の数を知っています。 問題に対する答えは非常に迅速に見つかりました-install4j 。 install4jは、MacOS、Windows、Linuxのインストーラーとランチャーを作成できるという事実に加えて、Windows、linuxのjreの最新バージョンをインストーラーに追加することもできます（残念ながら、macにはそのようなオプションはありません）。



install4jのビルドプロセス



したがって、Javaで記述された本格的なプログラムが得られますが、ユーザーは推測できなかったかもしれません。

しかし、彼らが蜂蜜の樽で言うように、軟膏には常にハエがあります.install4jの価格は1119ドルと399ドルで、窓の下にのみ構築するオプションがあります。
