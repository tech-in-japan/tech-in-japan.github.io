オラクル社、新しい4および8ソケットシステムを発売

オラクルは、IntelのXeon E7-8895 v2プロセッサをベースにした新しい4および8ソケットシステムを発売しました 新しいSun Server X4-4およびSun Server X4-8は、最適なパフォーマンスを実現するために、使用されるコア数またはカーネル負荷に基づいて動的な最適化を構成できる、独自のハードウェアグレードのコンピューティングエラスティック機能を提供します。







Intelとの緊密なコラボレーションにより、オラクルはシステム用の新しいXeon E7-8895 v2プロセッサーを最初に入手し、Oracle SolarisおよびOracle Linuxオペレーティングシステムとの最大の互換性を確保することができました。 Sun Server X4-4は、主にビジネス分析と分析タスク、およびサーバー統合のために設計されたため、これらのシステムはVDS / VPSに理想的なプラットフォームとなり、分析タスクをオンラインで実行するアプリケーションに適しています。 Sun Server X4-8は、Oracle Database In-Memoryおよび垂直スケーリングの最適なオプションとして評価されています。 Sun X4-8は、最大6TVまでのPCIフラッシュメモリをサポートします。これにより、キャッシュによりデータベースのパフォーマンスが大幅に向上します。 最上位のデータベースで動作するほとんどすべてのアプリケーションは、Database In-Memoryオプションを有効にすると、より高速に実行できます。



Xeon E7-8895 v2プロセッサは偶然選択されませんでした。 Intelが開発したグルーレスアーキテクチャを採用しています。

I / Oデバイス、ネットワークインターフェイス、およびプロセッサ間の通信には、特別に設計されたQPIバスが使用されます。 4つのプロセッサを搭載したサーバーでは、これらはすべてこのバスを介して直接相互に接続します。







8プロセッササーバーでは、各プロセッサは3つの隣接するプロセッサに直接接続され、そのうちの1つを介して他の4つに接続されます。







このアーキテクチャの主な利点は、絶対にすべてのメーカーが8プロセッササーバーを製造できることです。 メーカーによる特別な開発の必要がないため、4ソケットおよび8ソケットサーバーのコストが削減されます。
