Safari 4はChromeよりも高速です



Safari 4ブラウザーは、JavaScriptブラウザーでテストされた最速のCnet UKであることが判明しました。 ジャーナリストを確認するには、 SunSpiderテストスイートを使用しました。

Safari 4のWindowsバージョンは、910ミリ秒でテストスイートを完了しました。 したがって、Minefield 3.2a1（Firefox 3.2、1136ミリ秒）、Google Chrome（1177ミリ秒）、Firefox 3（3250ミリ秒）をバイパスしました。 Internet Explorer 7およびInternet Explorer 8はパフォーマンスが最も低く、テストはそれぞれ39026ミリ秒と5839ミリ秒で完了しました。

MacOS Xでは、競合他社に対するSafari 4のリードは、Firefox 3の4倍以上、Opera 9.6の7.5倍先でした。 MacOS XのSafariの新しいバージョンでのほぼ同じ結果は、969ミリ秒でテストを完了したMinefield 3.2a1によってのみ示されました。 Safariは967ミリ秒で同じテストを実行しました。 （ C ）
