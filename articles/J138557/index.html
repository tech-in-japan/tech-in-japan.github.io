MSDNロシア語Web開発セクションの更新：新しいデザイン、新しいトレーニングコース、および記事

先日、MSDN WebサイトでWeb開発に関するロシア語の大きなセクションの更新と再設計を受け取りました。





更新されたセクションは、 ASP.NET MVC 、 ASP.NET Web Forms 、およびWebMatrixの3つのメインセクションで構成され、開発者向けのMicrosoft Webプラットフォーム開発の3つの主要な方向を明確に定義しています。



サイトの更新に伴い、主要なセクションで新しい資料が公開されました。 新しい 16部構成のWebMatrixチュートリアルのリリースについてはすでに書きました。 それに加えて、次の資料が更新されたセクションに表示されます。
  • ASP.NET MVC 3 の24のチュートリアルと、 MVC 4のイノベーションの説明。
  • ASP.NET Web Formsのロシア語の資料のまったく新しい大規模なセット（66記事）：
    • パート1. ASP.NET Webフォームの概要（4つの記事）;
    • パート2.プレゼンテーションとデータ管理（4つの記事）;
    • パート3. ASP.NETのデータ管理の要素（11記事）;
    • パート4.データの操作（13の記事）。
    • パート5. ASP.NETでのルーティング（5件の記事）;
    • パート6. ASP.NETのセキュリティ問題（3件の記事）;
    • パート7.キャッシュとパフォーマンスの最適化（9記事）;
    • パート8.国際化（3件の記事）;
    • パート9.サーバーへのASP.NET Webプロジェクトの公開（14記事）。
