Microsoft Search Server 2010 Express

多くの企業にはファイルサーバーがあり、多くのユーザー情報が保存されています。 このようなガベージダンプを検索するには（だけでなく）、優れたMicrosoft Search Server製品があります。







Microsoft Search Server 2010の現在のバージョンは現在入手可能で、Sharepoint 2010に基づいています。

フル版とエクスプレス版の2つのエディションがあります。 エディションの比較 。



Microsoft Search Serverを使用すると、企業ネットワーク上で関心のある情報を検索できます。

-31の形式をサポート：Microsoft Office、HTML、SharePointサイト、Open Document、PDFなどの形式。

-シーカーがアクセスできるドキュメントのみを発行に表示します

-異なる語尾での検索（ロシア語の場合、これは重要です）

-特定のディレクトリ、ファイルタイプで検索を指定できます

-Windows Explorer（Explorer）に統合

-モバイル版があります

-検索クエリを分析できます

などなど



これは興味深いプレゼンテーションです。
