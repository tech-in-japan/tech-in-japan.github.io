タブレットBmorn V99のレビュー





購入の目的


私たちは祖母にタブレットを与え、彼らが常にスカイプに巻き込まれ、私たちのプライベートブログを簡単かつ自然に読めるようにすることにしました。



もちろん、祖父は長い間コンピューターを使いこなしてきましたが、祖母はそのような複雑なデバイスを少し怖がっており、タブレットはまだシンプルです。



私たちはこの目的のために、いくつかの中国製デバイスが非常に適していると判断しました。 さらに、それらのレビューは、一般的に、悪くはありません。





選択と価格


この選択は、aliexpress.comですぐに注文されたMOMO11 Birdで決まりました。 購入費用は237USDで、そのうち：

198,99USD-タブレット自体、

20USD-充電器、保護フィルム、およびシンプルなケース（何らかの理由で、充電器はデバイスの基本コストに含まれていません）、

23.08USD-パタヤへの配送（当時タイにいた）、

-5USD-AliExpressへの登録時に受け取ったクーポン。



しかし、すぐに売り手は、メーカーがこれらのデバイスの画面に問題を抱えていたことを解約し、今のところ販売を停止しています。 代わりに、提案されたBmorn V99。 特性に応じて、彼は私たちのために手配し、価格はMOMOと同じであり、私たちは交換に同意しました。



タブレットの機能：

画面：9.7インチHD、4：3 IPS、10ポイントタッチ

解像度：1024×768ピクセル

ケース素材：プラスチック

チップセット：Boxchip A10

OS：Android 4.0

RAM：DDR3 512MB / 1GB

カメラ：前面と背面-両方とも2MP

WiFi：802.11 b / g / n

3G：3G外部モジュールのサポート（EVDO / TD-SCDMA / WCDMA）

TV-OUT：1080p HDMI出力

ビデオ：H.264、H.263、VC-1、MPEG-1、2、4、DIVX-3、4、5、6、XVID、WMV7 / 8、VP6、AVSなどをサポート

オーディオ：MP3、WMA、OGG、FLAC、APE、AAC、AC3、ATRA、DTSなどをサポート

写真形式：JPG、JPEG、BMP、GIF、PNG、JFIFをサポート

電子書籍：EPUB、PDF、FB2、PDB、CHM、HTML、TXT、UMD、WORD、EXCELをサポート

ゲーム：Angry Birds、Fruit Ninja、3Dゲームなど

その他の機能：辞書、電卓、カレンダー、その他のAndroidアプリケーション。

言語：多言語サポート

USB接続：USB2.0高速伝送

バッテリー：3.7V 6000MA

フラッシュメモリ：4GB / 8GB / 16GB

メモリカードのサポート：最大32GB。



配送


ウェブサイトの注文は2月28日に発行され、3月4日にEMSに転送され、8日にはすでにバンコクにあり、3月15日にのみ、彼らはパタヤのタイポスト中央オフィスでそれを受け取ることができるという通知を出しました。



宅配ボックスの空所に発泡スチロールを充填し、デバイス自体が入ったボックスを1層のバブルラップで包みました。 すべてが安全かつ健全に到着し、箱の一角だけが少し入った。
