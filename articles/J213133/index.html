.NET開発者向けのオンライン会議。 3月1日

今年3月1日土曜日に開催され、.NETプラットフォームでの開発に専念する、今年の最初のDEV Labsシリーズ会議にご参加ください。 シリーズの以前のイベントと同様に、アクセス可能な形式で短時間で視聴者に有益な情報を最大限に伝えるようにします。 プログラムの形成に重点を置いているのは、自明ではないタスクを解決するために習得した実践的なスキルです。



会議承認済みレポート：



  • .NETのガベージコレクション。 機能、隠れた問題、ベストプラクティス
  • パブリックAPIの作成方法。 アプリケーションAPIの開発におけるエラーと困難
  • ASP.NET MVC で45分でコンテンツ管理システムを作成
  • Entity Framework Code First＆Migrations。 EF Code-First Data Access Technology
  • C＃ヒントとコツ。 C＃言語の「高度な」機能の動作の微妙さ




会議のウェブサイトでレポートと講演者に関する詳細情報を見つけることができます。



イベントの形式は1日、各プレゼンテーションで45分から1.5時間までの5つのレポートです。 各レポートの後、セッションは質問のためにスケジュールされます。 会議の参加者になるには、 登録する必要があります 。 参加は無料です。



UPD：会議はモスクワ時間の11:00に開始され、参加者用の追加のテストセッションが行われます。 最初のレポートは12:00に始まります。
