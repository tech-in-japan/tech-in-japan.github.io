WiMax受信用のWi-Fiアンテナのテスト

この記事では、miniPCIe WiMaxモジュールをSamsung Q1 Ultraにインストールすることについて説明しました。

コントローラの配送を待つ過程で、アンテナをどうするかという質問に悩まされました。

Q2Wは 私の答えを少し安心させてくれたので 、アンテナで運試しをすることにしました。



この記事では、UMPCに統合されたWi-Fiコントローラーへの外部アンテナの追加について説明しました。



そして、以下にWiMaxの使用結果を示します。







アップ。 この記事には3つの更新があります。

1.受信状態が悪い場合のHTC MAX 4Gとの比較。

2.新しいアンテナのテスト-勝者

3. google.comトレースの結果とHouse MDの第7シーズンのトレントテスト:)





ここに私が持っていたものがあります：



写真の左から右：リモートマウント、Planet 5dbiアンテナ、D-Link DWL-R60AT 6dbi、「通常のWi-Fiアンテナ」



アンテナなしの速度測定から始めました。 はい、アンテナがなくても、デバイスは信号をキャッチしました。 インテル5150には何かが組み込まれているのかもしれません。アンテナコネクタに接続するミニ同軸ケーブルに何かが引っかかっているのかもしれません。

インテル5150を手に入れるとすぐに、WiMaxアンテナとWi-Fiアンテナの互換性に関する疑問が大幅に減少したことに注意してください。

WiMaxとWi-Fiをサポートするアダプターには、個別のアンテナ出力はありませんが、内部アンテナと外部アンテナのみがあります。







そこで、屋内でテストを行いました。

1.アンテナがないと、写真は暗いです。





速度1Mビット; CINR / RSSI：14dB / -81dBm



2. Dリンクルーターの「通常のWi-Fiアンテナ」が残っている状態





速度7.21 Mbit; CINR / RSSI：32dB / -58dBm



3. Planet 5dbiアンテナを使用





速度6.61 Mbit; CINR / RSSI：30dB / -57dBm



4. D-LinkアンテナDWL-R60AT 6dbiを使用





速度7.72 Mbit; CINR / RSSI：30dB / -64dBm

これは指向性アンテナであり、上記はカーディナルポイントの8つの位置の最良の結果です。



最悪のシナリオを紹介します。





速度5.19 Mbit; CINR / RSSI：26dB / -68dBm



WiFiアンテナのテストで高い指向性が示されている間に、このような小さな散乱を観察するのは十分奇妙です。 たぶん私は基地局に囲まれていますか？



待望の結論：

-Wi-Fiアンテナの大部分はWiMaxと互換性があります。 （一般的に、3つのうち3つがテスト済み）

-外部アンテナは優れた信号品質を提供します。
