サンクトペテルブルクのSpbデー





あなたがPocket PCとSmartphoneの所有者であり、それを積極的に使用している場合、 Spb Software Houseの後援の下で 2008年8月9日に開催される無料の円卓会議に興味があるかもしれません。



これは、志を同じくする人々と出会ったり、パートナーを見つけたり、「Spb ...」の開発者と話し合ったりして、新しいプログラムのアイデアを議論し、古いアプリケーションに興味深い未実現の機能を提供する絶好の機会だと思います。 誰が知っていますか？..



カンファレンスは、メトロキーロフスキーザヴォード、セントの中心部近くで開催されます。 ルネサンスd 4。

このラウンドテーブルでは、Pocket PCおよびSmartphone（モバイルテレビ、ラジオ、メールの閲覧、サイトの閲覧、オンラインゲーム、IMなど）でのインターネットの使用について説明します。 このラウンドテーブルでは、Spb Software Houseの新製品が発表されます。

4.モバイル決済

このラウンドテーブルでは、Pocket PCとSmartphoneを使用して購入する可能性について説明します。 Spb Software Houseの代表者は、この方向での会社の現在の活動について話します。
