Nautilusの軽快なピーク

グラフィックファイルをすばやく確認するという終わりのない探求で、すばらしいGloobusプログラムに出会いました。これは、Linuxを搭載したネットブックの幸運な所有者を喜ばせます。 すぐに予約すると、Nautilusファイルマネージャーの下ですべてが研ぎ澄まされますが、誰もこの奇跡を何かに悩ませることはありません。







注意：GloobusとNautilusを正しく接続するには、パッチを適用したNautilus（以下の手順に従って自動的にインストールされます）が必要ですが、1つあります！ ネットブックの画面上のスペースを最小限に抑えるために、インターフェイスが若干変更されています。

記事の最後の写真をご覧ください。



仕事のスピード（ネットブックに関連）と雑食性の両方に嬉しく驚きました-開発者はグラフィックファイルだけに限定されませんでした。 Gloobusは、XCF、PSD、MPG、AVI、MP3、PDF、DOC、ODF、XLS、PPT、ODS、CBR、CBZ、ICNS、TXT、C ++、C、JPG、PNG、SVG、フォルダー、およびその他の形式を「ダイジェスト」します。







フォルダーのプロパティを確認します







スケジュールを見る







音声を聞く







ビデオを見る







および文書







表示には、通常（高速）とカバーフロー（Appleアプリケーションのようにカバー）の2種類があり、ネイティブリポジトリからは通常の表示のみが利用できます。







設置



キーを追加



sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 0624A220

sudo gedit /etc/apt/sources.list



テキストエディタで、行の最後に挿入します。



deb http://ppa.launchpad.net/tualatrix/gloobus/ubuntu jaunty main

deb-src http://ppa.launchpad.net/tualatrix/gloobus/ubuntu jaunty main



保存して閉じます。 ソースのリストを更新しています。



sudo apt-get update



Gloobusをインストールする



sudo apt-getインストールgloobus-preview



パッチを適用したNautilusが起動するようにシステムを更新します



sudo apt-get upgrade



Nautilusを再起動します
