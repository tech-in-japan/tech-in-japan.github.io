「信頼できるワイヤレスネットワークが必要」



最近、「N個のアクセスポイントとコントローラーを数える」というリクエストだけでなく、「ユーザーが自分のガジェットを仕事で使用できる信頼性の高いワイヤレスネットワークが必要」というリクエストを頻繁に受け取るようになりました。 今日は、そのような場合に何をするか、どのツールを使用するかを説明します。



そのようなリクエストを受信すると、私たちはクライアントに彼の仕事を理解するのに役立つ質問をし始めます。 結果を要約して、「優れた」信頼性の高いワイヤレスネットワークが満たすべき主要な基準のリストを作成しました。

  1. 無線チャネルで高速を提供します。
  2. 特定のスペースのカバレッジ。
  3. N番目の顧客の同時作業の可能性。
  4. コントローラーの機能のみを使用して、さまざまなレベルのセキュリティを備えたゲストアクセスと稼働中のネットワークを作成する機能。
  5. コントローラを使用して、トラフィックをフィルタリングし、ポリシーを割り当て、ワイヤレスセキュリティを確保する機能。
  6. このソリューションのセットアップのシンプルさ、すなわち サードパーティの専門家の関与なしで自分で。
  7. パフォーマンスの保証を取得する可能性-テスト、またはデモ、そして好ましくはフィールドで;
  8. もちろん、興味深い価格です。


この投稿の一環として、最初の3つのポイントについて説明します。 機器の選択、機器の量、およびその場所は、物理レベルでのワイヤレスネットワークの信頼性に直接影響します。 そのため、プロジェクトを実装する前にネットワーク設計フェーズを実行することをお勧めします。



残りのポイントについては、 以前の投稿で部分的に書きましたが、おそらく次のように書きます。



実際の顧客タスク



目的：建物の片側と同じ階にある6つの部屋で構成されるオフィスでワイヤレスネットワークを編成する。



在庫あり：部屋のレイアウト。ほとんどの場合、火災避難計画から取られています。 赤色の目盛りはターゲット領域をマークします。







カバレッジエリアを計算するソフトウェアはたくさんありますが、Aruba Networksが開発したオリジナルプログラムVisualRF Planを使用することをお勧めします。 現在、アルバはWi-Fiソリューションの技術的に最も先進的な開発者です（私たちの意見とGartnerの意見）。



アカウントをお持ちの場合、このプログラムはベンダーのWebサイトからダウンロードできます。 また、Googleマップに基づいて、ストリート/外部領域のネットワークカバレッジを設計するためのオンラインバージョンもあります 。



この計画をスケールの表示とともにプログラムにロードし、マウスを使用して壁の位置とタイプをマークします。 このプログラムでは、4つの壁オプションが利用できます。

  • コンクリート-コンクリートから;
  • キュービクル-「キュービクル」、オフィスパーティション。
  • 乾式壁-乾式壁から。
  • ガラス-ガラス製。






私たちの場合、コンクリートの壁は赤、黄色の石膏ボードであり、私たちの敷地の現実に対応しています。 ワイヤレスネットワークでカバーする必要がある領域をマークします（図の灰色）。







計画する領域を選択し、アクセスポイントのパラメーターとそれらの間のサブスクライバーの分布（最大ユーザー数、速度、チャネルごとのユーザー数）を設定する必要があるダイアログボックスを起動します。



私たちの場合、割り当てます：

  • ユーザーごとの最低速度90Mbps。
  • ユーザーは40人のみですが、チャネルごとに5人以下です。
  • 隣接するオフィスを妨害しないように、送信機のパワーを平均に設定します。
  • スペース（環境）のパラメーターを決定します：オフィスジャイロを公開します。
  • ポイントのタイプは、5 GHzの周波数で動作するクライアントをサポートする機能のためにAP-105として定義されます（ところで、無線の負荷が少なくなります）。






アクセスポイントAruba Networks



ここでは、 アルバアクセスポイントについて少し話をする必要があります。 現在、アルバのラインナップには多数のアクセスポイントがあります。



「標準」アクセスポイント：


AP-68-1つの無線モジュール、周波数範囲は2.4 GHzのみで、総速度は最大150 Mbit / sです。これは、加入者の密度が低い部屋向けです。

AR-93H -1つの無線モジュール、2x2 MIMOテクノロジーは、2.4 GHzまたは5 GHz帯域で動作でき、最大300 Mbit / sの合計速度、4つのイーサネットポートを備え、ケーブルインフラストラクチャを開発できます。

AR-93-1つの無線モジュール、2x2 MIMOテクノロジーは、2.4 GHzまたは5 GHz帯域で動作でき、最大300 Mbit / sの総速度で、低密度のオフィスや施設に最適です。

AP- 105-2つの無線モジュール、2x2 MIMOテクノロジーは、2.4 GHzおよび5 GHz帯域での同時動作をサポートし、さまざまなタイプの無線モジュールを使用する顧客が集まる部屋で無線ネットワークを編成できます。

AP-135-2つの無線モジュール、3x3 MIMOテクノロジーには、2つの10/100 / 1000BASE-Tポートが含まれ、総速度は最大450 Mbpsで、劣悪な無線条件でも高いパフォーマンスと速度を提供します。



リモートワークステーションを整理するためのアクセスポイント


たとえば、従業員が出張中の場合に非常に役立ちます。



RAP-2WG -2.4 GHzでのみ動作します。最大速度は54 Mbpsで、タバコのパックのサイズです-外出先で持ち歩くのに最適です。

RAP-3WN -1つの無線モジュール、2x2 MIMOテクノロジーは、2.4 GHz帯域で動作し、インスタントモードをサポートし、USBポートを備えています-リモートオフィスの編成に適した3G / 4Gモデムを接続できます。

RAP-5WN -1つの無線モジュール、3x3 MIMOテクノロジーは、2.4 GHzまたは5 GHz帯域で動作でき、最大300 Mbpsの集約速度、4つの銅ポートとUSBポートを備え、3G / 4Gモデムを接続できます。

RAP-100シリーズ -2つの無線モジュール、2x2 MIMOテクノロジーは、2.4 GHzおよび5 GHz帯域での同時動作をサポートし、外部アンテナを接続する機能があり、USBポートがあります-3G / 4Gモデムを接続できます。



次のアルゴリズムは、アクセスポイントのタイプを判別するのに役立ちます。







統計情報によると、2.4 GHzでの動作をサポートするデバイスの割合は、5 GHzで動作するデバイスの割合よりもはるかに大きいことに注意してください。 一部のレポートによると、比率は約67％から33％です。 互換性のあるデバイスがある場合、または表示する予定がある場合は、デュアルチャネルアクセスポイントを選択するのが理にかなっているのは論理的です。



外部アンテナの必要性は、たとえば信号を狭い長い廊下に向ける必要がある場合に、部屋の特定のジオメトリによって決定できます。 場合によっては、内部アンテナと外部アンテナのどちらを選択するかは、クライアントの好みの問題かもしれません。



オプション番号1：中出力のAP-105



この写真に戻りましょう。







ポイントAP-105のタイプを選択し、[OK]ボタンをクリックします。 指定されたパラメーターに基づいて、次の図を取得します。







結果のスキームは、色付きの領域が次の値に対応する特定の領域の無線周波数カバレッジを反映します。

  • 赤-信号レベル：-45db;
  • オレンジ-信号レベル：-55db;
  • 薄緑：-65dbなど


上記の3つのゾーンにいるときは、快適に作業できます。 これを実装するために、プログラムは4つのアクセスポイントのインストールを提案しました。



オプション番号2：AP-135を最大



より強力なトップエンドAP-135アクセスポイントと最大電力（30 dbm）の代替案を検討してください。

  • 40ユーザー、チャネルごとに最大10。
  • 最低速度90Mbps;
  • 各送信機の電力は最大に設定されています。
  • アクセスポイントはAP-135（高ユーザー密度）を選択します。


取得するもの：







最大電力での2つのAP-135アクセスポイントは、平均電力で4つのAP-105よりも悪くないタスクに対処する必要があることがわかります。 ただし、微妙な違いがあります。



結論と結果



異なるメーカー、異なるモデル、異なるポイント数など、将来のワイヤレスネットワークを設計する方法はさまざまです。 ソフトウェアツールは、さまざまなオプションを評価し、それらのいずれかを優先して議論された選択を行うための便利な方法で役立ちます。 ただし、考慮する必要がある微妙な違いもあります。



まず、AP-135の価格はAP-105と比較して1.5倍高く、4つの新しいアクセスポイントではなく、2つの上位アクセスポイントを取得すると、値のゲインは半分未満になります：1.5x * 2 = 3x 4倍。



第二に、場合によってはアクセスポイントの最大送信機電力を設定しても、減衰の問題が解決しない場合があります。 エンドデバイスにはまだトランスミッタがあり、すべての障害を「突破」する必要があります。



部屋が異なる階にある場合、最大に設定されたポイントは互いに詰まります。2.4の範囲ではご存じのように、ばらばらのチャネルは3つしかありません。 オフィスが1フロアのみにある場合でも、隣人を「妨害」し始めます。



この記事では、オフィスにワイヤレスネットワークを提供するプロジェクトを計算する実際の例を示しました。 私たちが通常提供する次のステップは、顧客が同意し、その地域で選択された機器のフィールドテストを実施します。 これらの目的のために、Aruba Networksデモ基金がコンピテンスセンターに形成されました。
