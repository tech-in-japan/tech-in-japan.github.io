新製品のリリース-SQL Management Studio for Oracle！

EMSデータベース管理ソリューションは、新製品のリリースをお知らせします。 SQL Management Studio for Oracleは、完全なデータベース管理および開発ソリューションです。 最も重要なデータベース管理タスクを解決するために設計されたコンポーネントセット全体により、SQL Studioは、データベースの管理とオブジェクトの管理、データベースの移行、比較、抽出、データのインポート、エクスポート、比較に不可欠なツールを提供します。 SQL Studioは、これらすべてのツールを組み合わせて、24時間体制で動作できる強力で便利な単一の作業環境にします。 アプリケーションまたはデータベースの開発者、管理者、アナリストのいずれであっても、SQL Studio for Oracleは、作業をこれまで以上に生産的にするために必要なすべてのツールを提供します！



EMS SQL Management Studio for Oracleは、一連のスタンドアロンSQLツールとどのように違いますか？

  1. 新しいEMS SQL Management Studioは、ユーザーに多くの非常に便利な追加機能を提供します。

    • ユーティリティとサービスの便利な起動
    • すべてのテンプレートを1つのリポジトリに保存する
    • すべてのサービスおよびユーティリティのデータソースの単一リスト
    • 複雑なタスクの作成とその自動実行
    • タスクで外部アプリケーションを実行する機能
    • すべてのタスクのログを単一のデータベースに保存する
    • カスタム通知を作成します。 さまざまな方法でそれらを送ります
    • すべてのアプリケーションに単一の視覚スタイルを設定する機能
    • テクニカルサポートへの迅速なアクセス
  2. スイートによって販売されているユーティリティのセットにすぎないパッケージとは異なり、EMS SQL Management Studioは、単一のインストール配布、すべてのコンポーネントを登録するための単一のインターフェイス、およびスタジオに含まれる機能モジュールを更新するための単一のシステムを備えた完全で独立した製品です。
  3. 最後に、SQL Studioの所有コストは、一連の製品の所有コストよりも大幅に低くなっています。 SQL Studioのコストには、ソフトウェアメンテナンスへの1年間のサブスクリプションが既に含まれています。これにより、SQL Studioの所有者は1年間無料で更新を受信できます。 その後のサブスクリプションの更新は、SQL Studioの年間コストの20〜30％のみで、キットの所有者は、それに含まれる製品の各アップデートを小売価格の50％で個別に購入する必要があります。




SQL Studioの主な利点：

  • クイックタスク起動により、結果のみに集中できます。
  • テンプレートを格納する単一のリポジトリにより、ユーティリティの操作がはるかに簡単になります
  • カスタマイズ可能なタスクにより時間を大幅に節約できます
  • スケジューラを使用すると、日常のタスクを自動化できます
  • さまざまな通知により、タスクの進行を制御できます
  • すべてのログは1か所に保存されるため、操作を簡単に監視できます
  • すべてのアプリケーションに単一の視覚スタイルを設定する機能




Oracle用のEMS SQL Management Studioは、 http ：//www.sqlmanager.net/products/studio/oracle/downloadからダウンロードできます。

評価版は完全に機能し、30日間自由にテストできます。
