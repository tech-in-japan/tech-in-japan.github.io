なぜ古典的な自動運転車は不可能であり、商業的見通しがないのか







自動車の技術革新に少なくとも従う人は、本格的な自動運転車が非常に長い時間道路を走る必要があることをよく知っています。 しかし、彼の外見は5年ごとに、さらに5年間絶えず移されています。 明らかに、何かがうまくいかず、最も頑固な楽観主義者でさえあいまいな疑いに苦しめられ、「人工知能の冬」を覚えている人たちは暗闇に陥ります人工知能は理解の障壁に陥りました 。 無制限の機能を備えたGoogle（Weimo）が10年で高価なおもちゃ以外に何もしなかった場合、現在の人工知能技術を備えた本格的な商用の第5レベルの自動車両（AA）不可能です。 間接的に、これはWeimo自体でも認識されており、2018年12月の最後のテスト（Waymoは米国で有料の無人タクシーサービスを開始しました）で、ドライバーはホイールの後ろに戻されました（万が一の場合、ドライバーは力に反応できるようにホイールの後ろに配置されます-重大な状況または無人システムの障害）。 そして、2019年に本格的なAAが私たちから遠くなり、近づかないという事実に関する記事は、多くのマイナスを獲得しなくなりました（5番目のレベルは近づかず、遠くなります） 。



この記事では、古典的なAAを作成するための多数の長い試みの間に生じた克服できない障害を分析します。 つまり、私たちは、N年後の明るい未来の広告の約束についてではなく、今日の実情について話します。



そして、一般的にオートマチックカー（AA）とは何ですか？



素人の視点では、これはコンピューター、つまり特定の人工知能（AI）によって制御される車です。 したがって、明らかな解決策： 通常の AIを作成し、それを1つのロータリーステレオカメラに接続し、任意の車に乗せ、通常の運転トレーニングコースに送り、資格のあるドライバーを取得します。 ただし、この決定に近いものは見当たりません。 つまり、既存のAIは人間のドライバーと競合できません。 そのため、AAの開発者は不正行為を強いられ、一種の代理（高価なセンサーの束、何をどのように行うかを理解していないニューラルネットワークなど）を提供します。 これはすべて、完全な（強力な）AIが表示されるまでの一時的なソリューションであることに注意してください。 その出現により、これらの高価なレーダーとライダーはすべて不要なものとして捨てられます（人はそれらなしで行います）。 つまり、AA自体を作成するための古典的なアプローチは、ある種の欺ceptionのスマックです。 理由を理解しましょう。



定義から始めましょう。第5レベルの本格的なAAとはどういう意味ですか？ これは、平均的な人間のドライバーの特性を備えた実際の道路状況で車を運転できる特定の自動化された複合施設（ACS）を意味します。 問題はすぐに発生します-そのような「平均的なドライバー」とはどのような獣ですか？ これは病院の平均気温ですか？ しかし、私はこの問題の解決策をPRマネージャーに任せ、タスクを大幅に簡素化することを提案します。AAを運転免許証のみを受け取った初心者ドライバーのレベルと比較してください。



つまり、初心者ドライバーと最も高度なオートパイロットを比較します。特定のドライバーではなく、現在の技術開発レベルで最も可能性の高いオートパイロットです。 これを行うには、初心者の運転手と最も完璧な自動操縦のすべての特性の比較表を作成する必要があります。

中間的な結論：古典的なAAの経済的収益性の概念は、一般に信じられているほど明白ではありません 。 キャラバンによるトラックの動きのテストが広く普及してから3年が経過し、キャラバンの動きの技術には技術的または法的問題はありませんが、技術は商業化されていないことに注意してください。



古典的な自動運転車は渋滞を増やし、規制を複雑にします。



しかし、上記のすべての問題が何らかの形で奇跡的に解決され、ステアリングホイールとペダルなしで、第5レベルの自律性の本格的なAAが得られたと仮定します。 わあ！ わあ！ しかし、誰が彼を必要とし、誰が彼にお金を払うのでしょうか？ 当然のことながら、このプロセスは即座に行われることはなく、誰かがこれらの内気さを行き来しますが、古典的なAAを導入するロジックは次のとおりです。



  1. この技術は自動タクシーで最も効果的です。
  2. ドライバーへの給与の不足は、旅行の価格を大幅に引き下げます。
  3. これにより、公共交通機関の乗客に対する需要が劇的に増加します。 安いタクシーが自家用車からではなく、公共交通機関から運ばれることがUberによって証明されています。
  4. Uberモデルによると、需要、ビジネス、個人の所有者を満足させると、自動タクシーの数が急激に増加します。
  5. 車間の距離を短くする効果は、同時に移動する車の数を急激に複数増加させる効果よりも大幅に少ないため、動きが上がります（現在、車の80％はドライバーなしですが、自動タクシーモードに移行できます）。
  6. 明らかに、AAを使用したタクシーパークの従来のビジネスは、プライベートAAと競合することはできません。これは、個人所有者が車の費用を償却する必要がないためです（週末にカントリーハウスへの旅行のために購入しました）。 したがって、移動する自動タクシーの数は迅速かつ予測不能に変化する可能性があるため 、 交通渋滞を規制することははるかに困難になります （個人商人の制御は非常に困難です、Uberが証明）

  8. その結果、輸送サービスの価格はすべて （解雇された運転手、ビジネス、タクシーの乗客、公共交通機関、特に普通の自動車の所有者）に対して高くなります。 ジャンプ後の価格がめったに初期レベルまで下がらないため、価格は上昇します。


したがって、結果として、古典的なAAは誰にとっても有益ではないことがわかります。



  1. AAは、給与の代わりに、「AAの税金」とほぼ同じ金額を支払う必要があるため、タクシー事業にとっては有益ではありません。 これはAA税です。なぜなら、人間の運転手によるタクシーの数はタクシー運転手の数によって厳密に制限されており、これはかなり自然な制限だからです。 そして、自動タクシーの数は信用率によってのみ制限されます。つまり、事実上何もありません。 ただし、AA自体のコストは、同じ通常よりも大幅に高くなっています。 その結果、ビジネスの観点から、完全な損失。
  2. プライベートトレーダーのAAは、「AAにかかる税金」も支払わなければならないため、収益性がありません。 州は「怠A」で、だれが自分だけにAAを使用し、誰に課税するのかを決定するからです。
  3. タクシーの乗客は、この状況も有益ではありません。 少なくとも価格は下がらないので、交通渋滞とその予測不能性は増加します。
  4. 公共交通機関の乗客、交通渋滞の増加も役に立たない。


そして、これらすべては、「ドローンの悪夢」を考慮せずに、駐車料金を支払わないために、ドローンが単に市内中心部を一周するときです。 つまり、有料駐車場は消えます。 国境への乗り換えがあるだけなので、センターへの入場も有料です。 真剣に訓練されたVision Zeroの利点であるAAの完全な差別のみが残っています。
