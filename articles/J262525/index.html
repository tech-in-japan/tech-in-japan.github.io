イベントでは、4G、3G、2Gテクノロジーの途切れないモバイル通信を確保する必要がありました。 蓄積された経験を使用し、さまざまな技術ソリューションを適用して、すべての新しいサミット施設でモバイル通信カバレッジを展開する必要がありました。 この集中的なプロジェクトの組織的および技術的な部分を担当しました。

ウファは、ヨーロッパとアジアの国境にあるヴォルガ地域の主要地域の1つであるバシコルトスタン共和国の首都です。 この多面的で多国籍な都市は、西部、東部、ソビエト時代の時代のすべての伝統を有機的に吸収し、これらすべての異なる方向を単一の全体に結合しました。 ウファは、歴史的にも地理的にも、SCOの国家元首および政府の国際首脳会議（中国、ロシア、カザフスタン、タジキスタン、キルギスタン、ウズベキスタン、およびインドとパキスタン）やBRICS（BRICS（ブラジル、ロシア、インド、中国、南アフリカ）。

イベントのウファの集中準備は、組織委員会が主導し、関係するすべての産業部門の方針に沿って行われました（より正確には、連邦と共和党の2つの組織委員会さえありました）。 同時に、セキュリティの問題により、モバイルネットワークの対象範囲を対象に計画するための情報はほとんどありませんでした。 市内の外国人代表団の移動ルートに関する予備的なデータは広められておらず、最終的な滞在地点しかありませんでした。 そのため、私たちはすぐに、100万人が住む都市全体で4G / 3G / 2Gネットワ​​ークをウファで持続的にカバーするイベントの準備をすることにしました。



タスク完了



ワーキンググループの同僚と集まって、ウファとモスクワの両方で一度に2つの方向からプロジェクトを開始したので、モバイルネットワークの計画、構築、最適化に対するすべてのアプローチを同期しようとしました。 私たちは責任のあるゾーンに分割され、「何からの迅速な対応」の方法と「将来のネットワークの長期的な開発」の方法の両方によって、ウファネットワークを準備するためのすべての可能なオプションを分析しました。 最終的に、両方の方法が使用されました。





街の南の「門」を通るウファへの入り口



SCOおよびBRICSサミットの対象物で現在の4G / 3G / 2Gカバレッジを描くために、測定機器のある場所に行き、加入者の目を通してモバイルネットワークに関する情報を収集しました。 第一に、サミットの既存および建設中のオブジェクトを測定するために「生きる」必要がありました。 残念ながら、いくつかの施設はまだ準備ができておらず、重要なイベントの先月まで、特に新しいヨーロッパのホテルについては建設中であったため、4G / 3G / 2Gのカバレッジの図を描くには、典型的な建物の同様の建物の経験的知識に頼らなければなりませんでした無線伝搬モデル。 多くの場合、新しいオブジェクトの信号機は、ほとんど常に最後に「入り」、展開し、ビルダーが仕上げ作業を実行するのとほぼ同時に行われます。 したがって、システムを最適化および調整する時間はほとんど残りません。 このようなタスクでのモバイル通信の利点は、ワイヤレスであるということです。したがって、長期の施設では、建物の隣接する屋根からカバレッジを展開するための予備のオプションがあります。

空港での新しい国際ターミナルの建設



事実、ウファ空港では、国内の航空会社のターミナル用の安定した外部4G / 3G / 2Gカバレッジがすでにあり、それで十分でした。 最初は、ストリートベースステーションのキャパシティを増やすことで、既存のカバレッジは国際航空会社の新しいターミナルに十分であると想定されていました。 唯一の恥ずかしさは、このターミナルに多くの金属構造があったことです。 建築業者がターミナルを最新のパネルで覆ったときにすべてが変わり、建物が完全に見えるようになりました。 建設中のターミナル内の4G / 3G / 2Gの測定では、別個の内部コーティングを展開しないと実行できないことが示されました。 外部の道路信号で遠方から「突破」することは非常に困難でした。 この空港の位置のレイアウトを内側に変更し始め、4G / 3G / 2G基地局の位置を建物に「移動」し、建設中に請負業者と一緒にすべての建設図面を改造しました。



国際端末に内部カバレッジを展開するために、パッシブ分散アンテナシステムを選択しました。 3人のモバイルオペレータはすべて同じ条件下で端末の立ち上げに直面したため、施設での合同会議の後、3人用の内部アンテナシステムを設計することにしました。 そのため、特別なプロジェクトでは、このオブジェクトだけで問題の複雑さが一気に増大し、均一なカバレッジを作成するために、多くの低電力アンテナをすばやく計画し、到着ゾーンと出発ゾーンの建物のすべてのフロアに均等に「吊るす」必要がありました。



さらに、ネットワークの準備中に、サミットの都市オブジェクトにカバレッジを取得する方法について考えられるすべてのオプションを分析しました。 SCOとBRICSの主要なイベントはすべて、割り当てられたゾーンの都市の外ではなく、南部の百万番目の都市の中心で行われたため、組織委員会はウファのサミットの開催地を選択するのにかなり成功したと言わなければなりません。 ウファと歴史的名所のビジネス活動の中心は徒歩圏内にありました。 また、この街の一部には、独自の通信インフラストラクチャもありました。 ここに追加の基地局を構築し、既存のモバイルネットワークに有機的に適合させる必要がありました。

4G / 3G / 2Gネットワ​​ークを構成する



Ufaでのモバイルネットワークのその後のセットアップについては、前回の加入者の好みについて一般的にお話しする価値があります。 現在では、大都市では、構築された基地局の数がモバイルネットワークの認識を解決できなくなっています。 かつて「幸福」と認識されていた4G / 3G / 2Gカバレッジの可用性の問題は、サービスを使用する感覚の主観的な要因に変わりました。 修辞的な質問は、1日あたり広い領域で活発に移動し、建物に留まり、次に路上で、移動車両で、モバイルネットワークですべてのサービスを同時に使用しようとする加入者の「幸福」係数を設定する方法です。
