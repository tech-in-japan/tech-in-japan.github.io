印刷解像度の背後にあるものは何ですか？ マルチビット技術はどのように機能しますか？





新しいレーザープリンターまたはMFPを選択するときは、「ドット/インチ」（dpi）などの概念に必ず出くわします。 この概念は、デバイスの印刷解像度、つまり 印刷画像の1インチあたりのドット数。 1インチあたりのドット数が多いほど、画像が鮮明になります。 ただし、このパターンは、マルチビットカラー印刷テクノロジーを搭載した京セラレーザープリンターおよびMFPには適用されません。このテクノロジーにより、競合デバイスと同等の品質の印刷ジョブをはるかに高い解像度で実行できるためです。 そして、ここに理由があります。









オンまたはオフのポイント



画像はピクセルで形成され、ピクセルは原則としてドットで形成されます：オンまたはオフ-またはポイントなし、またはこれらは4つのカラーチャネルの各ポイントです：シアン、レッド、マゼンタ、イエロー、またはブラック（黒）。 同じ色の色付きのドットはまったく同じです。



京セラマルチビットテクノロジー



マルチビットテクノロジーのおかげで、京セラのカラープリンターとMFPは、ドットを交互にオンまたはオフするだけではありません。各カラードットは、トナー濃度の変化により最大16の濃淡を持つことができます。 したがって、4つの場合すべての印刷の色は、特定の解像度の従来の印刷よりもきれいになります。 京セラでは、「9,600 dpiに相当」という用語（16 x 600 = 9,600）を使用して、京セラマルチビットテクノロジを搭載した600 x 600 dpiデバイスの印刷品質を説明しています。 この技術の利点は、1インチあたりの垂直ドット数を増やすことなく、印刷速度を落とすことなく、高品質のカラー印刷を作成できることです。



1インチあたりのドット数の測定方法



通常、印刷解像度に関しては、600 x 600または1,200 x 1,200 dpiについて話します。 最初の数は、1インチあたりの水平ドット数を意味し、主にデバイスのレーザーコンポーネントによって決まります。2番目の数字は、1インチあたりの垂直ドット数を意味し、ページが印刷プロセスを通過する速度によって決まります。 したがって、1,200 x 600 dpiの解像度のデバイスは、1,200 x 1,200 dpiの解像度のデバイスよりも高速に印刷されます。 一般的なルールは、「インチあたりのドット数が多いほど、画質が高くなり、印刷速度が低下する」です。



京セラのマルチビット印刷精度



以下の画像を比較してください：京セラマルチビットテクノロジーを搭載したデバイスで生成された画像は、600 x 600 dpiの印刷解像度を備えた競合デバイスの画像よりも鮮明であることが一目でわかります。 さらに、解像度が1,200 x 1,200 dpiのデバイスによって生成される画像よりも悪くありません。







また、1,200 x 1,200 dpiの解像度のデバイスと同じ競合メーカーの解像度600 x 600 dpiのデバイスとの間の印刷鮮明度の違いが重要でないことも重要です。 その理由は、人間の目の構造的な特徴にあります。25cmの通常の距離で読み取る場合、人間の目は0.04 mmの距離で隔てられたポイントを区別できません。 1インチあたり600 x 600ドットの解像度で印刷するときの2点間の距離は0.04 mmをわずかに超え、1インチあたり1,200 x 1,200ドットの解像度で印刷するときの2点間の距離はこの知覚範囲にあるため、人間の目は捉えません違い。



したがって、覚えておくことが重要です。1インチあたりのドット数が多くても、必ずしも画質が大幅に向上するわけではありません。



だから...



...優れた印刷品質のカラーデバイスを探しているときは、1インチあたりのドット数だけでなく、注意を払ってください。 結局、マルチビット印刷技術を搭載した600 x 600 dpiの解像度のデバイスは、1,200 x 1,200 dpiの解像度のデバイスの印刷品質と同等の印刷品質を実現できますが、はるかに高速です。 さらに、1,200 x 1,200 dpiの解像度を持つ多くの競合他社のデバイスは、デフォルトで600 x 600 dpiの印刷モードに設定されており、指定された最大印刷速度に対応しています。 つまり、1,200 x 1,200 dpiの印刷解像度でデバイスを購入した場合、デフォルト設定を変更して印刷速度を落とさなければ、期待した品質が得られない可能性があります。
