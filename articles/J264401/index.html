IBM、オープンソース開発者コミュニティ向けに50のプロジェクトを開始





IBMは、最先端のオープンソーステクノロジーを備えた新しい開発者プラットフォームの立ち上げを発表しました。 さらに、IBMはオープンソースコミュニティ向けに50のプロジェクトを立ち上げます。その目標は、企業へのサービスの導入を加速し、モビリティと分析の分野、およびその他の急速に成長している分野で新しいクラスのイノベーションを促進することです。



developerWorks Openは、開発者が最新のIBMソリューション、会社の技術知識、および経験にアクセスできるクラウドベースのプラットフォームです。 さらに、このプラットフォームを使用すると、プログラマーのグローバルネットワークと対話して、プロジェクトの作成と実装を加速できます。 開発者は、コードをダウンロードするだけでなく、ブログにアクセスしたり、ビデオを視聴したり、作業に役立つツールやテクニックにアクセスしたりできます。 このプラットフォームのおかげで、プログラマは、顧客のさまざまなビジネス要件を満たすオープンソースアプリケーションをより迅速に作成および実装できるようになります。



今日のオープンソースコミュニティ開発モデルでは、重要な戦略的ビジネス要件は考慮されていません。 この問題を解決するために、IBMは、医療、モバイルテクノロジー、小売、保険、銀行の分野で、実際のビジネスタスクを遂行することを目的とした多くの業界プロジェクトを立ち上げています。



この発表は、オープンソースソフトウェアのサポート戦略に対するIBMのコミットメントを再確認します。 20年以上にわたり、同社はオープンソースソフトウェアの動きをリードしており、何千人ものプログラマーをオープンソースソフトウェアの開発に引き付けてきました。 このイニシアチブの目標は、開発者が実際のビジネス上の問題を解決できるオープンソース対応のアプリケーションを作成できないようにする障害を取り除くことです。



developerworks Openポータルで既に紹介されている膨大なテクノロジーの中で、IBMは主要分野のプロジェクトへのアクセスを開放しているため、開発者は最も高度なソリューションを使用できます。 さらに、IBMは、ソフトウェア開発のためにBluemixクラウドプラットフォームでこれらのサービスを開始します。



たとえば、IBMは、モバイルソリューションのMobileFirstポートフォリオの多くのアプリケーションのソースをオープンにします。 これらは、ヘルスケア、小売保険、モバイルバンキングの市場への参入を目指すサービス開発者を支援するように設計されています。



  • IBM Ready App for Healthcareは、モバイルデバイスを使用して、自宅で理学療法コースを受講しながら患者のケアの進捗状況を監視するのに役立つアプリケーションです。
  • IBM Ready App for Retailは、直接通信チャネルを使用することにより、顧客サービスの品質を向上させ、個別のアプローチを提供し、店舗での買い物に対する消費者の認識を変えるのに役立ちます。
  • IBM Ready App for Insuranceは、住宅所有者と保険会社の関係を改善するのに役立ち、さまざまなIoTサービスを使用して住宅を公共料金メーターと同期します。
  • IBM Ready App for Bankingは、金融機関がモバイルソリューションのビジネスオーナーのニーズを満たすと同時に、新規顧客を獲得するのに役立ちます。




さらに、IBMは以下を含む、分析分野のいくつかのテクノロジーのソースコードを公開しています。



  • Activity Streamsは、開発者にサンプルのコーディングモデルと形式を提供し、ユーザーがアプリケーションや相互にどのように対話するかを説明します。
  • Agentless System Crawlerは、すべての種類のクラウドプラットフォームとランタイムを監視できる単一のクラウド監視および分析システムを提供します。
  • IBM Spark for Apache Sparkは、誰でもSparkプラットフォームの分析機能にアクセスできるソリューションです。 ベータ版はBluemixで入手できます。




IBMは、以下を含むオープンソースのクラウドデータサービスも継続していきます。



  • Bluemix Service Broker上のIBM Object Storageを使用すると、OpenStack SwiftとCloud Foundryを統合して、クラウドの場所に関係なく、クラウド内のデータにすばやくアクセスできます。




DeveloperWorks Openは、クラウドサービス開発者向けの重要な期間に開始されました。 組織は、複数のクラウド環境を使用する利点を最大限に活用しようとしています。 たとえば、クラウドアプリケーションを作成する場合、開発者は、申し立てに対して保証するために、ビデオを保存し、写真を編集するサービスを必要とします。 プログラムを見つけて実装するのは難しいかもしれません。 developerWorks Openは、業界で最もサポートされているソリューションの選択を支援します。



IBMのクラウドアーキテクチャおよびテクノロジー担当副社長であるエンジェルディアス博士は、次のように述べています。「IBMは、オープンソースがクラウド内のアプリケーションの革新的な開発における重要な要素であると確信しています。 「developerWorks Openにより、IBMの最先端ソリューションのソースコードを公開し、コミュニティの成長と既知のテクノロジーの実現に役立つと考えています。」



現在、IBMはSpark、OpenStack、Cloud Foundry、Open Contain Project、Node.js、CouchDb、Linux、Eclipse、Apacheなど、150以上のオープンソースプロジェクトに積極的に関与しています。 同社はすでに後者とのパートナーシップを確立しています。 このようなプロジェクトへの参加は、IBMソフトウェア製品を使用する際の知識と実践的スキルの拡大に役立ちます。 developerWorks Openプログラムは、IBMの戦略を実装する際の注目すべきステップであり、同社は革新的なソリューションの提供とクラウドコンピューティングによる革新においてビジネスをサポートしています。
