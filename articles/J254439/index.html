ウィークリービルドVivaldi 1.0.141.2

みなさんこんにちは！



本日、1.0.141.2という番号でVivaldiブラウザの新しい週刊ビルドをリリースしました。 その中で、タブの配置に取り組み続けました。 特に、今では垂直方向と水平方向の両方で1つのウィンドウでタブを開くことができます。 また、タイルを使用してタブを配置する機能も追加しました。



空間ナビゲーションは、この機能を最適化することに加えて、より良く、より速く動作するようになり、インデックスを使用することにしました。



また、ノートを操作し、1つのノートに複数の添付ファイルを追加する機能を追加しました。



ただし、インターネットにとどまらないようにしています。 このアセンブリでは、コンピューターのローカルフォルダー内の画像を表示する機能を追加しました。 また、写真を表示するときに「移行」ボタンを使用することもできます。 さらに、アドレスバーのシステムパスを使用してローカルディスクを操作できるようになりました。



変更は、Windowsプラットフォームの自動更新システムに影響しました。 シンプルになり、インストールパッケージをダウンロードする必要がなくなりました。ブラウザコードのみが更新されます。 確かに、これらの変更は次のビルド後にのみ評価できます:)



そしてもう1つの重要なニュースです。 水曜日に、私たちは次のTechnical Previewを3番でリリースすることを計画しました。そのため、今日のアセンブリは、TP3のリリースで修正できるバグや欠陥のテストの場として機能します。 この興味深い問題であなたの助けを待っています:)



ブラウザをダウンロードするためのリンク：



  • Windows 32ビット
  • Windows 64ビット
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット

  • VB-4637自動更新では、インストールディレクトリをインストーラーに渡す必要があります。
  • VB-4468は、コンテキストメニューから削除をメモします
  • VB-2171はURLのHTTPインジェクションを修正しました
  • VB-4104複数の添付ファイルをメモに追加する
  • VB-70閉じたページのタイルビューを更新
  • VB-1376アドレスバーをウェブページスタックから移動する

  • VB-4525デフォルトのキーボードショートカットアップデートの混乱
