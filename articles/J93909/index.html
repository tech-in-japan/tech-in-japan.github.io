GoogleがGIPS VoIPエンジンを購入するのはなぜですか？

数時間前、Googleによる新しい買収を発表する公式プレスリリースが発表されました。ノルウェーの企業Global IP Solutions （GIPS）は、ビデオ会議とIPテレフォニー用のソフトウェアを開発しており、この分野で20以上の特許を保有しています 。



GIPSエンジンは、Yahoo Messenger、Cisco WebEx、およびLotus Sametimeに基づいています。 GIPSテクノロジーは、携帯電話メーカーのLG、Samsungなどで音声品質を改善するために使用されています。 Global IP Solutionsが作成したさまざまな開発のデモを以下に示します 。



Googleが新しいアセットを使用するオプションの中には、Skypeの通常の代替品（ 6か月前に Gizmo5を購入したことを思い出してください）、市場で最高のビデオ会議システムを作成し、Android向けのビデオチャットを改善する（GIPSがAndroid向けの最初のビデオチャットをリリースしたことを思い出してください） 4月）。



グローバルIPソリューションはかなり大規模な会社であり、その株式は地元の取引所で取引されており、GoogleのオファーはグローバルIPソリューションの現在の市場価値より27.5％高くなっています。 一方、Googleのオファーは、今年1月に購入交渉が開始された時点のGIPSのコストよりも142％高くなっています。



公平を期して、Global IP Solutionsの購入はまだ未成年者にはなっていないと付け加えなければなりません。 取引を完了するには、株主の90％の承認を得る必要があります。現在、購入は会社の経営陣と証券保有者の50％によって承認されています。 オファーの有効期限は2010年7月4日です。
