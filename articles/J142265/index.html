Windows 8エディションが知られるようになりました

Windows ブログの Microsoftは、新しいオペレーティングシステムのエディションのリストを正式に命名しました： Windows 8 、 Windows 8 Pro 、 Windows RT 、およびWindows 8 Enterprise 。



以前のバージョンのWindowsと比較して、エディションの数は大幅に削減されています。



x86プロセッサをベースにした通常のユーザーシステム、デスクトップ、ラップトップ、およびタブレットの場合、 Windows 8の通常バージョンがリリースされます。



Proバージョンには、暗号化、仮想化テクノロジ、ドメインにPCを含め、グループポリシーを使用してPCを管理するエンタープライズレベルの機能が提供されますWindows 8 Proは、Windows Media Centerマルチメディアパッケージを備えた唯一のエディションです。



Windows 8 Enterpriseには、Windows 8 Proのすべての機能に加えて、PCの管理と展開、高度なセキュリティポリシー、仮想化、新しいモビリティシナリオ（？）などを含むIT組織機能が含まれます。



Windows RT （以前はWindows 8 ARMと呼ばれていました）は、タブレットおよびARMプロセッサベースのその他のコンピューター向けのWindowsエディションです。 タッチスクリーン（Office 15として知られるパッケージ）用に最適化されたWord、Excel、PowerPoint、OneNoteのバージョンが含まれます。 オペレーティングシステムにはInternet Explorer 10の特別なバージョンも含まれますが、使い慣れたWindows Media Playerはありません。 Windows RTは既存のWindowsプログラムと互換性がないことに注意してください。 Windows RTを単独でインストールすることはできません。工場出荷時にプレインストールされたシステムとして新しいコンピューターでのみ入手できます。



すべてのエディションにはMetroインターフェースがあり、Windowsストアアプリストアをサポートします。
