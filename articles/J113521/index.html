Facebookのバックグラウンドではない

INQは本日、Facebookとのコラボレーションで作成された2つのAndroidスマートフォンモデルを公開しました。

INQ Cloud TouchとINQ Cloud Qの携帯電話はどちらも、Facebookがコミュニケーションの主要な方法の1つになっている人向けです。 これは、Facebookの機能が携帯電話のシェルに深く統合されていることで表されます。Facebookアカウントのニュースフィードはホーム画面として機能し、すべての主要なFacebookアプリケーション（チャットとメッセージ、場所、友達リスト、通知）へのアクセスも行われます。 携帯電話は、FacebookシングルサインオンとSocial Graph APIも使用します。 後者は、たとえば、あなたが最もコミュニケーションをとる友人たちの正確なニュースを追跡するのに役立ちます。 Androidの「ネイティブ」サービスは忘れられません。たとえば、FacebookのイベントはGoogleカレンダーと同期されます。





携帯電話のオーディオプレーヤーも「曇り」です-Spotify。 それを呼び出す特別なボタンもあります。

INQは、ソーシャルモバイルカテゴリの先駆者であると考えています。 3年前に英国の「3」オペレーター向けにSkypephoneおよびINQ1モデルをリリースしたのは彼らであり、人気のあるオンラインサービス（Facebookを含む）のサポートも組み込まれていました。 それ以来、Facebookはソーシャルネットワークの1つ以上のものになり、比較的単純なデバイス（フィーチャーフォンクラス）の電話は、Androidプラットフォームに基づく強力なスマートフォンになりました。 電話の作成者は、これはFacebookphoneではなく、INQの電話であり、Facebookを介したコミュニケーション、Spotifyを介した音楽の聴取などに役立つことを強調しています。



どちらの携帯電話にもAndroid 2.2が搭載されます。 INQ Cloud Touch（写真では彼だけです）はタッチデバイスですが、Cloud Qは2.6インチの画面を備えたQWERTYスマートフォンになります。英国では、今年の第2四半期と第3四半期にそれぞれ、Carphone WarehouseとBest小売チェーンで販売されます買う。
