Microsoft Remote Desktopはすでに新しいLumiaにあります

こんにちは、Habr。



Windowsをデスクトップまたは作業用OSとして使用し、Lumiaスマートフォン（またはWP上の他のスマートフォン）を使用している場合、おそらく朗報があります。リモートデスクトップ管理用のMicrosoftリモートデスクトップアプリケーションは、「アプリケーションストア」で既に利用可能です。







アプリケーションがWindows Phoneバージョン8.1で動作する間、すぐに言及する価値があります。 したがって、 Nokia Lumia 630/930の所有者と、開発者向けにWP8.1バージョンを個別にインストールしたスマートフォンユーザーは、リモートデスクトップを使用できるようになります。 残りは、公式アップデートの「ローリング」を待つ必要があります（WP8を搭載したすべてのLumiaで利用可能になる予定です）







名前が示すように、Microsoft Remote Desktop（「試用版」のステータスのまま）を使用すると、リモートPCに接続し、すべての作業リソースに接続できます。



Microsoftリモートデスクトップは、Windowsの次のバージョンのいずれかで動作できます。

•Windowsクライアントを備えたOS（ProまたはEnterprise）：Windows 8.1、Windows 8、Windows 7、Windows Vista。

•Windows Server OS：Windows Server 2012 R2、Windows Server 2012、Windows Server 2008 R2、Windows Server 2008、Windows Server 2003 R2、Windows Server 2003。



アプリケーションは、リモートデスクトッププロトコル（RDP）8.1を使用し、RemoteFXテクノロジーとネットワークレイヤー認証セキュリティプロトコルをサポートします。



最初の2つは、使い慣れたジェスチャと高品質のオーディオ/ビデオ送信をサポートする便利なマルチタッチコントロールを担当します。 さらに、Windows 8 / 8.1の場合、アプリケーションはマルチタッチサポートをサポートします（最大10ポイント）。 リモートコンピューターがWindows 7以前のバージョンを実行している場合、この場合、ユーザーはカーソルとタッチパッドスタイルのコントロールを使用できます。







ネットワーク層認証セキュリティプロトコルは、ユーザーのプライバシーを保護します。 ただし、アプリケーションは新しい暗号化をサポートしていない古いサーバーで動作でき、安全でないサーバーへの接続についてユーザーに警告します。



楽しいことから：ユーザーはリモートコンピューターに自分の名前を設定できます。 このオプションは、リストに複数のデバイスがある場合に混乱しないようにするのに役立ちます。







デスクトップのスタートメニューまたはスタート画面に戻る必要がある場合は、スマートフォンのハードウェアカメラボタンを押します。



おそらく、Windows Phoneのリモートデスクトップのもう1つの際立った機能は「継続性」です。Lumiaがスタンバイモードになった場合でも、スマートフォンのロックを解除すると、デスクトップはすぐに使用できます（リモートPCに再接続する必要はありません）。



PCのセットアップについてご質問がある場合は、ヘルプをご覧ください 。



PS：Microsoft Remote Desktopは、他のモバイルプラットフォーム（iOSおよびAndroid）でも使用できます。 アプリケーションの機能は同じであり、 App StoreとGoogle Playで無料で入手できます。
