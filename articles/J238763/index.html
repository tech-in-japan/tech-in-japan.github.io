MicrosoftがWindows 10を導入





数時間前、MicrosoftはWindows 8.1に代わる新しいWindowsオペレーティングシステムを導入しました。 オペレーティングシステムはWindows 10と呼ばれます。



新しいOCは、膨大な種類のさまざまなデバイスに共通のオペレーティングシステムとして位置付けられています。



すべてのデバイス用の統合Windows



Windows 10は、以前のバージョンのOCよりも幅広いデバイスにインストールされます。 これらのデバイスには、パーソナルコンピューター、ラップトップ、タブレットだけでなく、スマートフォン、テレビ、さまざまなボード、モノのインターネットシナリオに使用される産業用デバイスも含まれます。



新しいOCは、アプリケーション開発用の単一プラットフォームであり 、 ストアを通じてアプリケーションを配信するための単一チャネルにもなります。



ユーザーインターフェースの改善



オペレーティングシステムインターフェイスは、一方では馴染みのあるままで、他方ではいくつかの新しい便利な機能を取得しました。



スタートメニュー



デスクトップの左下隅にあるロゴをクリックして開きます。 メニューの左側の列には、最近起動したアプリケーションへのショートカットが含まれ、右側にはユニバーサルアプリケーションのタイルが含まれています 。 メニューには、ローカルシステムだけでなくインターネットでも検索するための電源管理ボタンと更新された検索機能も含まれています。







ユニバーサルアプリケーションウィンドウ



現在、クラシックデスクトップアプリケーションと新しいユニバーサルアプリケーションの両方が、デスクトップ上の独自のウィンドウで起動されます。







スナップ機能の改善



一度に複数のアプリケーションで作業することがより簡単で便利になりました。 スナップ機能により、画面上で最大4つのアプリケーションを同時に開くことができ、Windowsは実行中の他のアプリケーションとその配置方法を通知します。 Windows 10は、未割り当ての画面スペースを他のアプリケーションでより有効に活用するのに役立ちます。







たくさんのデスクトップ



ユーザーは複数のデスクトップを作成し、それらにアプリケーションウィンドウを配置できます。 作成されたすべてのテーブルを画面に表示して、それらを切り替えることができます。







マルチタスク



新しいタスクビューボタンがタスクバーに表示されました。 クリックすると、実行中のすべてのアプリケーションのウィンドウが縮小された形で画面に表示されます。



企業ユーザーとデータへの特別な注意。



Windows 10の作成時には、企業ユーザーに特別な注意が払われました。

新しいOSは、組織が使用するすべての従来の管理システムと互換性があります。



オペレーティングシステムには、あらゆる種類のデバイス上の仕事用データと個人用データ​​を分離するための新機能が含まれており、BYODの概念を組織に変換します。



新しいWindowsをお試しください



10月1日のWindows 10の公式ページでは、 Windows 10 Technical Previewをダウンロードできます 。

新しいOCを自分の目で見て、自分で試してみる機会をお見逃しなく。

オペレーティングシステムのリリース日とWindows 10へのアップグレード条件に関する情報は、後日発表されます。



UPD： Windows 10の暫定版が既にダウンロード可能です。
