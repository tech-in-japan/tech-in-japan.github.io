Tair-3C +コンバーターK-1。 ニコンインフィニティ

この記事では、方法よりもTair-3Cレンズに焦点を当てます。

Nikonシステムに無限大が存在するように、K-1テレコンバーターを作り直します。





Tair-3C、K-1 MS、KP-42 \ N



私は、M42スレッドのソビエトレンズと、KP-42 \ Nアダプターと組み合わせて、NikonカメラのM42ネジ接続を備えたK-1 MSテレコンバーターを使用することを考えていました。



問題は、無限が存在するようにテレコンバーターをリメイクする方法ですか？



内部のレンズは、3つのレンズ、コントロール、およびゼニスカメラの露出計に絞り値を送信するための電気回路を備えたモノリシックガラスのように見えます。 結果（無限大）を達成するには、アダプターKP-A \ 42およびKP-42 \ Nを使用してレンズを3 mm近づける必要があります。これには、タイプAの3 mmメガネと3 mmアダプターKP-A \ 42を研磨する必要があります。 これは間違った動きです。 ソーイングスリーブAは次の2つの理由で機能しません：（一度）アダプターがレンズに取り付けられない、（2）アダプターKP-A \ 42をネジで締めることができません、これらのネジの穴の中央まで3mmを削ると抜けてしまうからです





Tair-3C。 マウントタイプA



彼はアダプターの幅を測定し始め、2mmの差を見つけました。





KP-A \ N





KP-A \ 42、KP-42N



つまり、使用するアダプターの厚さに応じて、コンバーターの幅を2〜3 mm減らす必要があります。



参考：最も薄いアダプターは中国製で、厚さの平均はソビエトKP-42 \ Nで、最も厚いアダプターはステンレス製です（自家製または中国製）。



コンバーターを3 mmに近づけるために、コンバーターでどのように削減できるかを検討し始めました。 M42スレッドのスリーブが挿入されている側の3 mmを研削できることが判明しました。





コンバーターとスリーブM42





カットあたり3mm





カットあたり3mm



マークは、以前に3mmに設定されたキャリパーの鋭い端で配置できます。 作業中に部品が万力から飛び出さないようにしっかりと固定する必要があります。また、コンバーターレンズを金属粉塵から隠す必要があります。







彼らは何とか問題なくコンバーターボードを自分でカットすることができましたが、M42スリーブを使用して、3mmケースをカットするように頼み、ターナーに頼りました。







コンバーターの詳細（M42ブッシングとビード）を減らした後、幅がすべての側面で同じであることを確認します。 これは、キャリパーを使用して実行できます。







いずれかのセクションの幅が小さい場合は、ガスケットを配置することをお勧めします。 接着剤で修正できます。







ガスケットを取り付けたら、スリーブをネジで固定します。 必要に応じて、部品をほこりや接着剤できれいにします。



美しい外観を与え、作業の痕跡を隠すために、開いた金属部分をペイントできます。 これにはマットスプレーペイントが適しています。



警告：手（手袋）、体の開いた部分（ドレッシングガウン）、気道（マスク）、目（メガネ）を保護した後、換気の良い非居住施設で塗料を使用する必要があります。 スプレーペイントは、部品から少なくとも30 cmの距離を置いて重ねて塗布する必要があります。 塗料の乾燥時間は10分から24時間です。 非居住施設で部品を乾燥させます。





塗装用プロット





塗装後



塗装後、塗料を乾かします。



塗料が乾くと、作業の最終段階である小さなつまらないものと部品のテストを開始できます。



部品の作業位置を検索する時間を短縮するために、ドットでマークを付けてから（浅い、0.5-07mm）穴を開け、それらに赤いアクリル塗料を塗ります。





タグの準備





完成したラベル



ゴム手袋で穴をペイントします。 布で塗料の残留物を取り除き、水で湿らせた布で汚れを取ります。



テストは最後のステップです。 たとえば、Nikkor 50mmレンズ付きカメラで撮影したフレームと、Tair-3Cレンズ、K-1 MSテレコンバーター、Photosniper-12バットで撮影したフレーム。
