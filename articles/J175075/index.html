キーボードから最大限のパフォーマンスを引き出す

仕事のほとんどの時間は何をしていますか？ ほとんどの場合、情報を作成または編集します。 そして、あなたの多くはこの情報を持っています-テキスト：プログラムコード、設定ファイル、レター、チャットラインなど。

より速く作成し、同時にタイピングにあまり注意を払わないようにしたいですか？ この問題のメインツールであるQWERTYキーボードを使用する場合、生産性を高める効果的な方法をいくつか説明します 。



これに時間を費やす価値があるかどうか疑っていますか？ 覚えておいてください-PCの世界では30年ですべてが変わりましたが、キーボードは同じままです。 代替テキスト入力方法-音声認識、文章、ジェスチャー、「鳥」言語（ハロー、グラフィティ！）、コードキーボード-は機能せず、普及しませんでした-そして、おそらく非常に長い間、まだキーボードからテキストを入力するでしょう。





1.ブラインド印刷



もちろん、これはポイント1です。



なんで？


  • 印刷速度が向上しています。
  • 入力プロセスに集中する必要がなくなりました 。 あなたは頭の中でテキストを発音し、あなたの手はそれを入力します。 次のウィンドウでテキストを入力して記事を読んだり、次の行を熟考したり、電話で話したりすることができます。
  • ボタンに書かれた文字に依存することはありません-ラップトップ上でロシア語の文字なしで、またはステッカーや彫刻なしで消去された文字で作業できます。 バックライトなしで夜間に問題なく印刷できます。




学習方法


  • 従来の方法は、「キーボードシミュレータ」を使用することです。 トレーニングだけが必要な場合は、 klava.orgまたは同様のオンラインシミュレーターに満足できます 。 よりアカデミックなアプローチが必要な場合は、長い導入テキスト、評価、感情的な態度、週に数回のスパムレターの動機付け、その他の社会的問題を含むオンライン「 キーボード上のソロ 」に移行してみてください。
  • しかし、私たちは反対に行きます。 キーボードの文字を接着するだけです。 その後、最初の1週間は非常に不便になり、タイピング速度が低下し、信じられないほどのタイピングエラーが増加しますが、1週間後にタイピング速度は「視覚」速度を超え、さらに増加し​​ます。 自分自身を助けるために、画面の横にキーボードレイアウトの印刷物を置くことができます。 画面、別の（無効な）キーボードを見ることができますが、入力するキーボードは見ないでください。




この後、同じklava.orgのクラスは単に天国からのマナになり、 バタン （スマイリーフェイス）で開催されます。

3.新しいキーボードを購入する



ロシア文字のないキーボードを購入し、より便利なキーボードを購入します。 より便利になります：

  • ショートストロークキー（ラップトップなど）。 古いキーボードのボタンを最後まで押すと、どれだけ余分な時間と労力が費やされたかに驚くでしょう。
  • 2つに分かれた半分の人間工学に基づいたデザイン 。 これが必要な場合-Mlcrosoft Natural
  • アルファベット以外のキーの標準レイアウト 。 追加のキー（ホーム、エンド、矢印など）が他のキーボードと異なる場所にある場合、それらのキーの場所を知ることは意味がありません。




ロシア語の文字なしでキーボードを手に入れるには- 中国語かeBayで入手してください。 興味深いオプションのうち、より高価なものはApple MB110です;安価なオプションのうち、プラスチック製の偽物はA4Tech KV-300です



4. Caps Lockによるレイアウトの切り替え



Alt + ShiftとControl + Shiftがどれほどひどいものかは思い出せません。 また、Command + Space（MacだけでなくWindowsでもレイアウトを切り替える主な方法です）もあまり便利ではありません。

長年使用していた方法はLShift + RShift（入力時に小指で押すのが便利）でしたが、結局はCaps Lockをお勧めします。

  • Caps Lockは不要で 、基本的に干渉するだけです。 （前回テキストを入力したときを覚えていますか？）
  • 大きく、見つけやすく、 1本の指でクリック
  • 多くのキーボードには 、現在のレイアウトを表示できるLEDがあり、非常に直感的な切り替えが可能です。




作り方


窓
haali.su/winutilsプログラム（「lswitch 20」パラメーターで実行）またはwww.gooli.org/blog/recapsを使用します。

また、 habrahabr.ru / post / 41726にはコメントがいくつかあります。

LEDを使用したディスプレイレイアウトは機能しません。 私はWindowsでの作業経験があまりありません。コメントでもっと興味深い解決策を見つけてうれしいです。


Linux
最も幸運です-仕事に必要なものはすべてOSに組み込まれています。

UbuntuおよびGnomeベースのシステムで、次を開きます。

システム設定 → キーボード → レイアウト設定 → 詳細設定...

そして、アイテムを確認します。

  • レイアウトを変更するためのキー → Caps Lock
  • キーボードインジケーターを使用... → Caps Lock


Gnomeのないシステムがある場合、これを行うパラメーターをxorg.confで見つけて入力できます



Mac OS
1996年、Tyoma Lebedevがこの問題に対処しました。 そのレイアウト「 Methododica 」はロシア語と英語の文字をCaps Lockの異なる状態に割り当て、原則としてレイアウトを切り替えるメカニズムを使用しません。

5.ロシアの活字体



「 ロシアのウィンドウ 」のレイアウトにはいくつかの不便さがあります-例えば、カンマは、最も不便な方法で入力されます-これは最も人気のある句読点であるにもかかわらず、文字「e」は誰もそれに従わないような場所にあるという事実にもかかわらず さらに、ロシア語では句読点が数字よりも頻繁に使用されます。 これが、GOST 6431-90に対応する便利なレイアウトを「 ロシアタイプライティング」という名前で作成できるようにしたものです。





（はい、句読点はShiftを使用せずに入力され、数字は逆にShiftを使用して入力されます。数字キーパッドを備えたキーボードを使用している場合は、Shiftは不要です）



作り方


このレイアウトは、すべての一般的なデスクトップOSに含まれています。 システムレイアウトのリストで見つけて、標準のロシアレイアウトの代わりに追加します。 これで、1つのキーを押すだけですべての基本的な句読点を入力できるようになり、「」が小指の下に便利に配置されます。



Mac OS X 10.9 Mavericksの一般的な簡素化の過程で、ロシアのタイプライターのレイアウトは削除されました。 元に戻すには、 ここから「BG46」レイアウトを配置するか、 ここで Caps Lockに切り替えて結合レイアウトをダウンロードします 。

7.ドヴォルザークとコレマック



Dvorakレイアウトについて聞いたことがあります -もともと1932年からのラテン語キーボードの高性能レイアウトです。それは悪くなかったというわけではありません-しかし、それはあまりにもクールで珍しく、その有効性に対して抽象的です-レイアウトの世界のYfquilのようなものです。 代わりに、モダンで便利であると同時により効果的なオプションを提案します-Colemacレイアウト







それは何を与えますか？


ウィキペディアを引用します：

  • スピード 。 QWERTYよりも高速で、Dvorakよりもわずかに高速
  • QWERTYとの互換性。 QWERTYとColemakの部分的な類似性により、ユーザーは、QWERTYとColemakの両方を使用できます。1つのレイアウトから別のレイアウトへの移行に大きな困難を感じることはありません
  • 人間工学 英語の最も一般的な10文字は、キーボードの2番目（ホーム）の行にあります。 ColemakはQWERTYよりも122％頻繁にホーム行を使用します
  • 現代のコンピューターの現実への適応。 すべての特殊文字（セミコロンを除く）は、古い場所にあります。 標準のキーの組み合わせ（Ctrl + Z、Ctrl + Sなど）は、片手で押すのに適しています。 （他のホットキーは単に移動することに注意してください。）




キーボードをやり直す必要がありますか？


できたらいいのに！ ただし、多くのキーボード（主にラップトップとApple）は簡単に再作成できます。



古い場所からキーを削除する必要があります-原則として、このためにキーを引き上げるだけで-スキームに従って新しい場所に配置します。 注意：これを行うと、ボタンが破損する危険があります。 キーボードの解析方法に関する指示を探し、最も不必要なボタンを試してください。 変更についての良い話はすでにHabréにありました 。



変更後に最初に気付くのは、人差し指の下にその場所を残したリスクです。 接着剤銃で新しいリスクをかけるのが最も簡単です。 そして、古いもの-絶対に必要ない場合-は刃で切ることができます。

その結果、何が得られましたか？



すべての可能性を最大限に活用した場合、次のことができます。

  • 小指のすぐ下にある1つのボタンを押すと、レイアウトを切り替えることができます 。
  • あなたは数週間苦労し、その後あなたは本物の文字「」を含むロシア語を素早く盲目的にタイプします。
  • 基本的な句読点を入力するには、2つのキー（他のすべての一般的な活版印刷文字）を組み合わせて1つのキーを押します。
  • ほとんど適応なしで、あなたは英語でより速く（盲目的ではないにせよ）タイプし始めます。 Colelmakは非常にクール（スマイリー）なので、ダイヤル速度は増加し続けます。
