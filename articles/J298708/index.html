ビジネスパーソン：ラリー・エリソン、440億ドルの財産を持つOracleの共同設立者





新しいコラム「ビジネスパーソン」は、メガマインドの読者に、世界中の最も著名なIT起業家を取り巻くストーリーや事実、経営陣が合理的な限界を超えている企業の価値について話しています。 私たちはビジネスマンの「ニューウェーブ」だけに限定されず、通常「古い学校」と呼ばれる人々について話します。



今日、クローズアップは、オラクルの永続的な共同設立者であるラリー・エリソンの人生と歴史です。



人格形成







1944年8月17日にニューヨークで生まれ、19歳の未婚のユダヤ人フローレンススペルマンと、イタリア出身のアメリカ空軍のパイロットから。 幼児期の母親は、ラリーにシカゴの叔母リリア・スペルマン・アリソンと叔父ルイス・エリソン（ロシア移民）の教育を与え、9ヶ月で彼を養子にしました。 48年以上にわたり、ラリーは母親の名前すら知らず、彼女に会ったこともありませんでした。



シカゴで幼少期を過ごした後、彼はイリノイ大学アーバンアンドシャンペーン校に入学しましたが、リリアスペルマンエリソンの死後2年目には勉強を拒否しました。 しかし、これはすべて先史時代に過ぎませんでした-1年後、彼がカリフォルニアで過ごした夏、ラリーはシカゴ大学に入学し、そこで最初のコンピューターとコンピューター技術の最初のモデルに精通しました。



キャリアと他のプロジェクト







1966年にカリフォルニアに移り、その瞬間から、彼はウェルズファーゴ、アムダールなどの企業の仕事に飛びつきました。 後者では、彼はプログラミングを学び、その後Ampexで仕事を得ました。



Ampexでの作業の開始は、Larryの人生のターニングポイントでした。当時、同社はデータベース管理システム（DBMS）を作成するためにCIAと大きな契約を結んでいました。 このDBMSはその後、コード名Oracleを受け取りました。



1977年、ラリーはパートナーのボブマイナーとエドオーツと共に、新しい会社Software Development Laboratories（SDL）を設立しました。 会社の初期資本はわずか2,000ドルで、そのうちエリソンの個人ポケットからの1,200ドルです。







当時、彼らはリレーショナルデータベースに関するIBMの科学者エドワードコッドの理論に触発されました。これは、情報への後続のアクセスを格納および取得する特別な方法です。 当時、このようなデータベースのアイデアは革命的でした-Oracleデータベースストレージシステムを設計したのはこの理論でした。 1979年、創業者はSDLをRelational Software Inc.に名前変更し、同年にOracle v2の2番目のバージョンがリリースされました。 そして1982年に、名前はOracleに変更されました。 Oracle Databaseの主な製品の中心は、リレーショナルデータベース理論でした。







Oracle Databaseを使用していたとき、エリソンと同僚はIBM System Rの互換バージョンの開発に苦労していました。 IBMはソースコードの開示に消極的であり、重要な問題になっており、2つのシステム間の互換性は達成されていません。







高い成長率により、オラクルは1986年に5500万ドルの利益でIPOに上場し、わずか3年で利益を5億8400万ドルに増やしました。しかし危機がありました。1990年に深刻な問題に直面し、大きな損失を報告しました。 当時、同社はコスト削減のために400人以上の従業員をレイオフしました。 会計文書でまだ完了していないトランザクションの販売と会計の登録のエラーにより、オラクルは現地市場の規制当局との困難に直面しています。







その結果、Oracleは倒産に近づき、Sybaseなどの競合他社が最初の市場シェアを徐々に獲得し始めました。 エリソンは開発の正しい道を選択するのに数年を費やしました-1992年、Oracle7のリリースは状況を改善しました。







当時、主要市場のプレーヤー間の競争は最高潮に達していました。OracleやInformixの広告戦争の時代のように、90年代は多くの人に記憶されていました。 そこで、オラクルのオフィスの近くに看板を買って、「注意、恐竜が道を渡る」という碑文を掲示し、時代遅れのオラクルのテクノロジーをほのめかしています。







あらゆる状況で利益を見つける能力は、エリソンの利益のために繰り返しプレイすることができました-スタートアップSalesforce.comを設立した後継者Mark Benioffの退任中であっても、彼は関係を壊しませんでした。 Oracleの共同設立者は、友人の新しいプロジェクトに約200万ドルを投資した最初の人物の1人でした。



その後、Mark Benioffは、OracleがSalesforceの直接のライバルを開発していることを知り、エリソンに取締役会を辞任して株式を売却するよう要求しましたが、拒否のために彼を解雇しました。 その結果、エリソンはSalesforceの株式ブロックを保持し、現在400億ドル相当の会社の株式を取得することができたため、実際には直接のライバルが成長しても利益を上げることができました。



これと同様に、エリソンはNetSuiteプロジェクトの元OracleのトップマネージャーであるEvan Goldbergに1億2500万ドルを投資しました。 そして、そのようなベースを販売し、競合他社から直接利益を得て、その株式の価値を成長させようとしたのはオラクルでした。



スティーブ・ジョブズでさえ、1997年にアップルに戻ったとき、取締役会の席に着くという要請でアリソンに頼りました。 しかし、アリソンは時間不足のためgiveめざるを得なかった。



珍しい趣味







アリソンは、常にジェット、ヨット、ゴルフに特別な情熱を注いできました。オラクルのリーダーシップの間に、この情熱は強まりました。 その後、オラクルのヘッドは彼自身のゴルフクラブを買収しました。







しかし、エリソンの投資の一部は、見通しとその後の収益の合理的な評価とは関係ありません-2012年に有名なロストシリーズが3億ドルで撮影されたハワイのラナイ島を買収しました。







島の人口は3000人であり、アリソン自身がマーケティング担当者を島に派遣して、島の潜在的な可能性を検討したことは注目に値します。



プレゼント







2014年、アリソンはオラクルのCEOを正式に辞任し、会社の経営をMark HurdとSafra Katzに移しました。
