KDE 20年



KDEはまた、記念すべき年であり、プロジェクトは20年目を迎えました。 Habréでは、このイベントが気付かれずに通過したため、このギャップを埋める必要があります。







1996年10月14日、Matthias Ettrich は 、システム管理者ではなく、すべてのアプリケーションとインターフェイス要素が同じデザインと共通の動作スタイルを持つ、一般ユーザー向けのモダンで本格的な無料のグラフィカル環境の作成を発表しました。







  • KDE 1.0は1998年7月12日にリリースされました-最初の安定版リリースです。
  • KDE 2.0 は 2000年10月23日にリリースされました。
  • KDE 3.0-2002年4月3日リリース 。
  • KDE 4.0-2008年1月11日リリース 。
  • KDEフレームワーク5-2014年7月7日
  • KDE Plasma 5-2014年7月7日と同様 。

Git 10年



2016年4月7日Gitは10歳になりました。これはLinusの2番目の主要なイニシアチブであり、最初のイニシアチブと密接に関連しています。 Gitは、プロプライエタリなBitKeeperの代替品を見つける必要があるために誕生しました。BitKeeperは誰もが既に慣れていましたが、もはや使用できませんでした。







しばらくの間、すべてが順調に進んでいるように見えたが、BitKeeperは開発者の作業を楽にしました。 誰が変更の権利を持っているかを心配する必要がなくなり、それぞれがソースツリーの独自のブランチで動作するようになりました。ソースコードの分散マージの可能性により、すべての人の労力が大幅に節約されました。 それを超えて、危機は醸造され、それがGitの創造につながりました 。
