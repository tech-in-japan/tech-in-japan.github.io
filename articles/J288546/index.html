新年が近づくと、冬は暖かくなります。 そして、この惑星はさらに素晴らしい、素晴らしいです。 そして、その上にいる人々は、さまざまな形の憎しみとen望でお互いを驚かせることは決してありません。 インターネットが何千ものサルがまだ戦争と平和を書くことができないことを証明したという古いジョークを覚えていますか？ しかし、彼らは最もアクセスしやすい場所で最も簡単な言葉を書くことができます。 なぜこれをしているのですか？ ドク、あなたを前進させる！



性別の週



何かが起こり、先週、性別の面で悪化がありました。 戦争当事者のそれぞれは、挑発的なメッセージと射撃で注目されました。 実際、この対立、男性対。 女性は、すでにかなりうんざりしています。 それでも、だれもあきらめませんが、その反対は常に述べられています。 私は女性が好きですが、私はすぐに農民のためです。 それで、最初にマニフェストが普通の男性を支持して現れました 。 「 私は反対します、紳士です！今、男性が変わったものに反対します。私たちの周りで発展している情勢に反対します。しかし、今、人間のような生物種の完全な劣化があります。 「普通の異性愛者は、私たちの社会で最も権利を奪われた生き物です。女性はとても残忍で、マリアナ海溝のどこかに押し込まれ、どこかに殺到するのが正しいのです 。」 このマニフェストは私の後半でテストされましたが、最後まで読んでいませんでしたが、敗者によって書かれ、人生と女性に怒られ、それはbeatられるべきだと言いました。 私はそれを理解しているように、私は静かにした方が良い。 死んだり、ただ不自由になったりするよりも、身をかがめる方が良い。 約1日半が経過しました。前回の投稿は、それほど記念碑的な答えではありませんでした。 「 この最も男性的な」は「ゼロから現れるのではなく、羽毛関係「少年少女」が鉄筋コンクリートの家族に発展するその時から」。そして、これは「男性のあるべきではない」 「私たちは同じ選択をしたので」という義務が現れました。さらに、毎日、あなたはますます多くを持っているでしょう。私と同じように。家族は精神的な一致だけではありません 。 現在、ブロガーを止めることはできません。痛みを伴うトピックが見つかったため、誰もがこの問題について自分の考えを誇示する必要があります。 しかし、その後、男性だけが手に落ちます。 まず、「 ハニー、私たちは夕食に何を食べていますか？ 」という質問を燃やすために熱い鉄で呼ばれます。それを「 ハニー、夕食を食べますか？ 」に置き換えます。 これらの2つの質問のうち、普通の人が尋ねるべき質問はどれですか。 第二に、 適切な求愛についての頭脳があります 。 さらに、この適切な求愛は、私、あなたがため息のオブジェクトに与える花だけに依存していることが判明しました！ 色覚異常の人は厳密に禁忌です-動揺。 「 赤い花は、中年の女性の記念日、または女優、歌手、愛人のいずれかに贈られます。白い花は、葬儀や花嫁に贈られます（はい、私も選択肢が好きですが、そうです） 。 などなど。 一般に、本物の男性が従わなければならない多くのポイントがあります。 第三に、特に本物の男になることをまだ決めておらず、一般的な男にすぎない場合は 、女の子のもろさを理解し、評価するように訓練されます 。 「 彼女は、火星人とニジニ・ノヴゴロドの混合物を話しても、必死に理解されることを望んでいます。たとえロンドンが彼女のフレーズをうらやましくても 。」 「ハニー、夕食はいかがですか？」とだけ尋ねている人がいるようです-適切な量の適切な色のチューリップを与えている間、彼は「あなたは私を愛していないようです！」



「ベラルーシのために！」週間



ベラルーシ人が投票を好むという事実は、長い間知られています。 今週、つまり私たちのものは、この自然な情熱を完全に示す別の理由があります。 当然のことながら、インターネット上のすべてのものと同様に、このベンチャーは現実のグロテスクなパロディに変わりました。 彼は有名なLiveJournalユーザー自身の名前の賞を1つ編成し、各ノミネートで勝者に常緑樹1000株を約束しました。 最初に、候補者はすべての来訪者によって提案され、次に投票リストが作成されました。 ほとんどオスカー、そのような小さな町であり、応募者の間で絶えず争いをしています。 そして、どういうわけか偶然にも、同胞のヴィナが「 LiveJournalの年間最優秀作家 」にノミネートされた応募者の1人になったのです。 これが楽しみの始まりです。 そして、私は同胞の行動を恥じています。 たとえば、何らかの理由で、誰かが492年前の戦いを思い出すことにしました 。 さらに、投票のスローガンは、 「 ズワッツァの人々の権利のための人々ўバラバへのベラルーシのペラモグのために 」という不可能性に対してばかげていると考えられました。 AAA！ 私の脳ではありません！ この投稿に対するコメントの1つが正しく述べているように：私のフォイルハットはどこにありますか？ 理論的には、この投稿だけで私の脳が私の耳を流れるのに十分でしたが、ほとんどすぐに次の奇跡に出会いました。 もちろん、これは非常に奇妙なユーモアと冗談であることは理解していますが、既に「KG / AM」の大きなラウンドスタンプを取得し、「著者」をキャッチして、ブランドのように額に付けたいと思っています。 水文気象センターの報告によると、全国の退化のレベルは自然の背景を超えていません。 このストーリー全体では、常識の島として、私はビナ自身の行動に満足しています。ビナは一般的にこれについてまったくコメントしていませんが、地元のユーロビジョンを静かに見ています。 来年もよろしくお願いします！



死んだ週



私は恐怖よりも好奇心で死ぬ可能性が高いです。 いいえ、もちろん、私自身も死ぬことを恐れており、親relativeや友人を心配していますが、死自体​​は私にとって興味深いものです。 おそらくこの理由で、私は死んだ人々の日記に興味がありました。 むかしむかし、私はすでにこのメッセージについて書きましたが、それ以来リストは更新され、補足されました。 「 記憶は生きている限り、人は生きていると言います。インターネットでは、記憶は永遠に生き続けることができます。人はずっといなくなっていますが、フォーラムでの彼からのメッセージは生き続け、答えることができます。著者自身だけが答えることはありません。一年前、初めて死者の日記に出会った。アニメに関する情報を探して、人々のサイトに行った。少女は素朴に素朴に書いたので、興味を持って読み始め、読んでいるうちにこの少女に突然気付いた私は死んだ人の手紙を読むことはもはや生活の世界にはありません 「私は客室に登って、自分の思い込みを確信しました。サイトは更新されていません。しかし、ゲストは生きています 。」



前の段落の苦痛な印象を払拭するために、死者に関する別の話。 人が死んだという事実にもかかわらず、しかし、物語は非常に面白いと肯定的です。 さらに、正統派の父親は彼女に伝え、彼女は80年代半ばに彼に偶然に会いました。 地域委員会で亡くなった人の話。



今週の興奮



そして、私は将来の世代が心配です。 若い人に似ていますが、私をフォローしている仲間は違うと感じています。 そして、もちろん、もっと悪いことです！ 私たちの時代には、木が高く、草が緑で、アイスクリームが美味しく、女の子が暖かかっていたので、ご存知でしょう。 引用：「 子どもの頭の中で学校の歴史がどのようなものになるかは誰にもわかりません。事実、日付、トフトミーシュやメトディウスのような奇妙な名前は、テレビ、書籍、コンピューターゲームからの情報の流れと混ざり合い、時には真の代替物語に変わります。」 代替ストーリーはすでにここにあります 。



そして、これで、私が完全だと思うブログの最も邪悪なレビューのリリース。 カトリック教徒、来たる！
