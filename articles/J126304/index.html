Firefox 8には、サードパーティのアドオンをブロックするシステムがあります





Firefoxの現在のバージョンはバージョン5.0であるという事実にもかかわらず、開発チームのブログでは、 Firefox 7.0および8.0のバージョンに関する将来の計画を読むことができます。 8番目のバージョンは早くも11月8日に表示されるので、待機時間はそれほど長くありません。 そのため、バージョン8.0では、サードパーティのプログラム/サイトによってインストールされたアドオンを自動的にブロックする機能があります。 ブラウザーを開くと、組み込みのセキュリティシステムがサードパーティアドオンの存在を確認し、ユーザーに警告します。



疑わしいものがインストールされている場合、ブラウザはこのインストールを確認するようユーザーに要求します。チェックボックスをオンにする必要があります。 つまり そのような追加によって引き起こされる損害の場合、ユーザーは自分を責めるだけでよく、他の誰も責めません。







セキュリティシステムが新しい追加だけでなく、以前にインストールされたすべてを分析することも興味深いことです。 そのため、ブラウザを8番目のバージョンに更新すると、サードパーティ企業のすべてのアドオンが無効になります。 それらを有効にするには、検出されたアドオンのリストを見て、対応するボックスをチェックする必要があります。 そうしないと、そのようなアドオンはアクティブになりません。







さて、今のところ、Mozilla Firefox 6.0がリリースされるのを待っています。このバージョンは明日、Mozilla Firefox 7.0は9月27日にリリースされます。上記のように、8番は11月8日にリリースされます。
