第2回Apple Developers Community Spbミーティング





2010年12月16日、 Ingriaビジネスインキュベーターで、Apple開発者の2回目の会議で、コード名は「 Road to the Young 」になりました。



最初の会議の後のフィードバックに感謝します。私たちの主なアイデアをサポートする志を同じくする多くの人々がいたことを非常にうれしく思います。



今回は、モバイルアプリケーションで収益を上げたいが、どこから始めたらよいかわからない初心者にいくつかの考えを与えることにしました。 彼らは彼らが基本を理解するのを助け、マスタークラスでの仕事において専門家によって正しい方向に導かれます：

  • アレクサンダー・チェルニー（ ミハイル・ケチノフのスタジオ ）「 ダミーのためのiPhoneの開発 」;
  • Ivan Chashkin（ Zen Mani ）「 ダミーのためのHTML5 iPhone開発 」;
  • アントンマキシモフ（ e-Legion ）「 ダミー用Windows Phone 7の開発 」
こちらからイベントに登録できます 。 猫の会議とワークショップの詳細。



モバイルデバイスのアプリケーション市場は急速に成長しており、急速に出発する列車に飛び乗って今すぐお金を稼ぐことができますが、それは非常に重要です。





マイクロソフトの新しいモバイルOS、たとえば、現在のところ、そのためのプログラムは非常に少なく、巨大な市場シェアを「食い止める」ユニークな機会があるとしましょう。 なぜ待つの？ アントンマキシモフによるワークショップにサインアップし、志を同じくする人々の会社で開発するための最初の一歩を踏み出します。



マイクロソフトが好きではありませんか？ 問題ありません。 最大のソフトウェア市場であるAppStoreに注目する価値があります。 Alexander Chernyは、特に開発の原則、ツール、言語など、iPhone開発のアイデアを提供します。



Macbookをお持ちでなく、目標cを学習したい場合でも、手ぶらで放置することはできません。 Ivan Chashkinが、iPhoneおよびAndroid用のJavaScriptの記述方法を説明します。



マスタークラスに完全に参加するには、ソフトウェアがプリインストールされたラップトップが必要です。 つまり、 iPhone開発について聞きたい場合は、XcodeとiOS SDKがインストールされたMacが必要です。 Windows 7 Phone向けの開発には、使い慣れたVisual Studio 2010と、Microsoft Silverlight for Windows Phoneツールキット、Expression Blend for Windows Phone、およびWindows Phone Developer Toolsが必要です。 iPhone HTML5用に開発したIvan Chashkinのリスナーは、WebKit-a ChromeまたはSafari（最新バージョン）なしではできません。



マスタークラスは並行して開催され、記録されませんが、投票の結果によると、最も興味深いクラスは繰り返し、より広範囲に開催されます。
