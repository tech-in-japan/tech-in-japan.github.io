名前にアンダースコアがあるユーザー

そのようなhabruiser @_meowth_がありますが、彼のプロファイルにアクセスしようとすると、FFはFirefoxが_meowth_.habrahabr.ruでサーバーを見つけられないと報告します 。 RSSフィードからトピックにアクセスしようとすると、同じことが起こります。 問題は非常に簡単に解決されます-meowth.habrahabr.ruへのリンクを少し修正するだけですが、そのトピックは表示されません（別のユーザーのように感じます）。 ただし、RCCで受け取ったトピックへのリンクを同じ方法で変更した場合、たとえば_meowth_.habrahabr.ru/blog/57083からmeowth.habrahabr.ru/blog/57083に変更すると、RCCに届いたそのトピックに到達します。

  • Mozilla / 5.0（X11; U; Linux i686; en-US; rv：1.9.0.4pre）Gecko / 2008101311 Firefox / 3.0.4pre（Swiftfox）
  • Mozilla / 5.0（X11; U; Linux i686; en-US; rv：1.9.0.8）Gecko / 2009032711 Ubuntu / 8.10（intrepid）Firefox / 3.0.8
  • アローラ0.6
  • Opera 9.64
  • Mozilla / 5.0（互換性、Konqueror / 4.2、Linux）KHTML / 4.2.2（Geckoなど）
