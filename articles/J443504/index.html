World Wide Webは30歳です。次に何が起こるでしょうか？





WWWの30歳の誕生日に、World Wide Web Foundationの創設者[World Wide Webの開発とアクセシビリティに特化した国際的な非営利組織/ trans。] Tim Berners-Leeは、ウェブが私たちの世界をどのように変えたかを思い出し、すべての人類に利益をもたらすより良いウェブを作成するために何をする必要があるかについて話し合います。



今日、情報管理システムを作成するという最初の提案から30年後の2019年3月12日、世界の人口の半分はオンラインに座っています。 私たちがどこまで来たかを祝う時であり、これからもっともっとやるべきことがあるということに気づく時です。



ウェブは、都市の広場、図書館、診療所、店舗、学校、デザインスタジオ、オフィス、映画館、銀行などに変わりました。 もちろん、それぞれの新機能と新しいサイトの出現により、オンラインの人とオフラインの人とのギャップが広がり、誰でもウェブにアクセスできるようにする必要性がさらに緊急になります。



そして、ウェブは孤立したグループに投票する権利を与えられ、私たちの日常生活を容易にする多くの機会を生み出しましたが、それは詐欺師の機会も作り出し、憎しみを広める人々が発言することを可能にし、さまざまな犯罪の委託を促進しました。



Webの誤用のニュースの中で、多くの人々がWebを恐れており、それが良いものを運んでいるかどうかがわからない理由は明らかです。 しかし、過去30年間でWebがどれだけ変化したかを考えると、次の30年間でWebをより良い状態に変えることはできないと思い込んで、giveめずに想像してください。 今日、giveめて改善されたWebを作成しない場合、これは私たちを失望させるWebではなく、失望させます。



問題を解決するには、まず問題を明確に特定して理解する必要があります。 今日のWebの問題の3つの原因は、私にとって印象的です。



  1. 州が後援する攻撃やハッキング、違法行為、オンラインでの嫌がらせなど、故意の悪意のある行為。
  2. ユーザーの価値を無視するひねくれたインセンティブを作成するシステムの機能原理-たとえば、 クリックベイトや誤報のバイラル拡散を促進する広告ベースの収益性モデル。
  3. 善意の不注意による否定的な結果、例えば、怒りの論争、意見の非妥協、オンラインディスカッションの質。


最初のカテゴリを完全に排除することはできませんが、オフラインで行われたように、このような動作を最小限にする動作の法則とルールを作成できます。 2番目のカテゴリでは、インセンティブを変更するためにシステムを再設計する必要があります。 3番目のカテゴリでは、既存のシステムを理解し、新しいシステムをモデル化するか、既存のシステムを微調整するための調査が必要です。



1つの政府、1つのソーシャルネットワーク、または人間の精神を非難することはできません。 問題の原因に焦点を当てるのではなく、問題の症状を追跡することで、不必要な単純化にエネルギーを浪費するリスクを負います。 すべてを正しく行うために、私たちは集まってグローバルなウェブコミュニティになる必要があります。



世代の歴史の重要な瞬間に、以前に住んでいた人々は、より良い未来を創造するという目標に協力しました。 世界人権宣言は、さまざまな人々が基本原則に同意することを許可しています。 国際海事法と宇宙条約により 、公共の利益のために新しいフロンティアを維持することが可能になりました。 そして今、ウェブが世界全体を変えているとき、ウェブへのアクセスが人権とみなされ、共通の利益のために作成されることを保証する責任があります。 そのため、Web Foundationは政府、企業、市民と協力して、新しいWorld Wide Web Contractを作成しています。



この契約はリスボンのWorld Wide Web Summitで開始され、Webの明確なガイドライン、法律、標準を確立する必要性に同意した人々が集まりました。 このプロジェクトを支援する人々は、その最初の原則を支持し、責任を共有するために協力します。 どのグループも一緒にこれに取り組むべきではなく、どんな入力でも大歓迎です。 政府、企業、市民が貢献しており、今年は成果を上げることに尽力しています。



政府は法律や規制をデジタル時代に適応させる必要があります。 彼らは、市場が競争力があり、革新的で、オープンであることを保証しなければなりません。 彼らは、人々の権利と自由をオンラインで保護する責任があります。 私たちは政府のオープンウェブの支持者を必要としています-公務員と、民間部門の利益が公益を脅かし、オープンウェブを守る場合に行動する選出された代表者です。



企業は、迅速な利益を追求する際に、人権、民主主義、科学的事実、または公共の安全を忘れないようにするために、より多くのことを行う必要があります。 プラットフォームと製品は、プライバシー、多様性、セキュリティを考慮して開発する必要があります。 今年、テクノロジー企業の複数の従業員がビジネス慣行の改善をどのように要求したかを見ました。 この種の活動を奨励する必要があります。



そして最も重要なことは、市民は企業や政府が約束を果たすことを要求し、人々を中心としたグローバルなコミュニティとしてウェブを尊重することを要求しなければなりません。 自由で開かれたウェブを擁護する政治家を選出しない場合、建設的で健全なオンラインでの議論を奨励しない場合、データの保護を必要とせずにすべてに同意し続ける場合、これらを作成する責任を放棄します問題は政府にとって優先事項です。



Webの戦いは、私たちの時代の最も重要な目標の1つです。 今日、世界の半分はオンラインです。 これまで以上に、残りの半分が前半に遅れないようにし、全員がウェブに貢献し、平等、機会、創造性を促進することを保証することが重要です。



World Wide Webコントラクトは、編集の短いリストではなく、オンラインコミュニティとの関係の理解に変化をもたらすプロセスでなければなりません。 それは、誘導灯として機能するのに十分明確であるが、急速な技術変化に適応するのに十分柔軟でなければなりません。 これは、デジタルユースから、より成熟した、責任感のある、包括的な未来への私たちの旅です。



ウェブはすべての人のためのものであり、一緒に私たちはそれを変える能力を持っています。 簡単ではありません。 しかし、あなたが少し夢を見て一生懸命働くなら、私たちが望むウェブを手に入れることができます。
