シンプルで静か、AMD Geodeを搭載

ALIXマザーボードは、シンクライアントだけでなく多機能ネットワークデバイス用に設計された小型で低電力のマザーボードです。



このトピックでは、この鉄をCISCOと比較することを敢えてしませんが、それだけで注目に値します。



ALIXボードは（もちろん、台湾の工場で）スイスのPC Enginesによって製造されています。

今日は、ALIX.2d13プラットフォームとそれに基づいたルーターについてお話します 。

前の投稿のように、ネットワークプラットフォームに焦点を当てます。 思い出させて
ネットワークプラットフォームは、さまざまなネットワークタスク（ルーティング、NAT、VPN、プロキシ、ファイアウォールなど）用に調整されたソフトウェアがインストールされた通信ネットワークで動作するように設計されたハードウェアです。






  • ボードは低消費電力のAMD Geodeチップを使用しているため、デバイスは負荷で6〜10 Wの電力しか消費しません。
  • x86アーキテクチャにより、FreeBSD、Linux、またはそれらに基づいた特殊なディストリビューションなど、分散システムをOSとして使用できます。 テストモードでは、Windows XP Embeddedがボード（VGA搭載モデル ）で起動されました。
  • 補助暗号化ブロックAMD Geode LXセキュリティブロック。AES-128ハードウェア暗号化アクセラレーションとハードウェア乱数ジェネレーターをサポートします。


  • 各イーサネットポートには独自のコントローラーがあり、イーサネットサブシステムの最大のパフォーマンスを保証します。
  • MiniPCI拡張スロットを使用すると、POST（診断ボード）から始まり、ギガビットイーサネット、RS-232、CAN（コントローラー）で終わる周辺機器を接続できます。
  • Alix.6f2ボードには、内部3Gモデムを接続するためのMiniPCI-Eスロットもありますが、負荷分散によるハイブリッドMultiWAN構成のサポートが表示されます：3 WAN + WiFi LAN /イーサネット+ 3G /イーサネット+ Wi-Fi / Wi-Fi + 3G
