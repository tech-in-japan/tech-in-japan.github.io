Appleは、PCクローンにインストールするためにOS X Leopardを開く場合があります

しばらくの間、MACコミュニティが頑固に気づきたくないという噂がありました。

2009年1月にリリースされたApple OS X 10.6 Snow Leopardは、すべてのPCクローンにインストールできるようになります:)

そして、WWDC`2008の前に、PC用のApple OS X Leopardのオープンに関する噂は激化し続けています、次のニュース記事はそれを示唆します：

• 「AppleはPC用にOS Xを準備していますか？」 信じられないほどですか？」

• 「今日、Appleは開発者向けにMacを導入しますか？」

（たとえば、前回の記事では、新しいコンピューターが、Windows製品をApple OS X Leopard OSに転送するための便利で安価なツールを提供するWindows-PC開発者向けであることを明確に示唆しています！）



このトピックに関する議論の理由は、記事（2007年秋に公開された記事）にあります。

• 「Apple MacOS Xの未来は何ですか？」



まあ、もちろん、これらは信じられないほどの噂です。 しかし、結局のところ、すべては通常うわさで始まります。たとえば、すでに2005年1月に、Apple MacからIntelプロセッサへの移行についてうわさが広まり始めました（その後記事を書きました）

• 「ソビエトの国からApple Computer IncのCEO Steve Jobsへの手紙」

そして、誰もそれを信じることができませんでした、2005年半ばにWWDCの2005年にアップルはこれを公式に発表しました！

おそらく今日のWWDC`2008で、Windows開発者向けのコンピューターであるMac Fusionを紹介して、AppleがApple PC XにインストールするためのApple OS X 10.6 Snow Leopardのオープンを正式に発表することを誰が知っていますか？
