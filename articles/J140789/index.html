ISPmanagerでのWindowsのメール管理サポート

ISPsystemは、MS Windows Server 2008 R2オペレーティングシステム用のISPmanager 4.4.3をリリースしました 。 新しいリリースでは、メール管理のサポートを実装できるようになりました。これらの革新に関連して、もちろん、お客様によるこの製品の使用に関していくつかの疑問が生じました。 この記事で検討したいこれらの質問の中で最も一般的なものです。





そのため、MS Windows Server 2008 R2オペレーティングシステム用のISPmanager 4.4.3の新しいリリースでは、メール管理サポートが実装されました。 これはWindowsファンにとって朗報です。パネルからメールサーバーを制御できるようになりました。 HMailServerがメールサーバーとして選択されました。



このメールサーバーの選択に影響を与えたものは何ですか？ さまざまな基準に従って、Windowsオペレーティングシステム用のソフトウェア製品の市場で提示されているいくつかの異なるメールサーバーを分析しました。



  • 機能性
  • サーバーデータをさまざまなデータベースに保存するためのサポート、
  • ライセンス機能、
  • COMテクノロジーを使用したメールサーバー管理


そして、私たちの選択はhMailServerバージョン5.3でした。 さらに、hMailServerはWindowsプラットフォーム用の無料のメールサーバーです。



このメールサーバーは、メールサーバーに必要なすべての機能を提供し、GPLライセンスの下で配布されます。 hMailServerには、SMTP、POP3、IMAP4プロトコル、仮想メールドメイン、SSL暗号化、ルーティングのサポートが含まれています。 メールをチェックするための外部アンチウイルスであるアンチスパムモジュールを使用することもできます。ClamWinおよびSpamAssassinと統合することもできます。 もう1つの重要な機能は、MS SQL、PostgreSQL、MySQL、またはMS SQL Server Compactを実行する統合データベースなど、さまざまなデータベース形式でデータを保存するためのサポートです。 さらに、サーバーはhMailServerと通信するためにISPmanagerで使用されたCOMテクノロジーを使用して制御する機能をサポートします。



hMailServerの使用を開始する方法



現在、ISPmanagerは、hMailServer構成の基本的な側面と同様に、メールドメインとメールボックスでの作業を実装しています。 メールサーバーの使用を開始するための小さなガイドを提供します。



ISPmanagerのメインメニューの[サーバー設定]の下に、[メール設定]の項目が表示されました。 このメニュー項目を使用して、hMailServer構成フォームが呼び出されます。



hMailServerがまだインストールされていない場合、または構成がまだ行われていない場合、フォームは次のようになります。





指定されたリンクをクリックすると、hMailServerがダウンロードを開始します。 メールサーバーが既にインストールされている場合は、この形式でhMailServerのインストール中に指定された管理者パスワードを入力する必要があります。



パスワードを入力すると、ISPmanagerはhMailServerに接続します。 将来、サーバー構成フォームは次のようになります。





このフォームを使用して、使用するプロトコルの基本設定を行うことができます。 自動禁止設定は、ISPmanager 4.4.4のリリースで表示されます。



プロトコル設定



プロトコル構成は、次のように実行されます（SMTPプロトコル構成例を使用）：

1.プロトコルの使用を有効または無効にできます。

2.このプロトコルを使用して同時接続の数を指定することもできます。







SMTPプロトコルの設定には、「ローカルサーバー名」フィールドもあります。 このサーバー名は、メッセージヘッダーに示されています。 メールを受信するサーバーは、メールの送信元IPアドレスと指定されたサーバー名の名前の対応を確認できます。名前が一致しないか、記入されていない場合、メッセージは無視されるか、スパムとしてマークされます。 したがって、このフィールドが正しく入力されていることを確認してください。 サーバーに複数の名前がある場合は、それらのいずれかを指定します。 他のプロトコルも同様に構成されます。



それだけです 情報があなたにとって完全に役立つことを願っています。
