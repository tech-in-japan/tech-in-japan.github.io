TechEd Russia 2011のHP

私はあなたについては知りませんが、あなたは楽観主義を決して失うべきではないと私は心から信じています。 そして、今、驚くべきMicrosoft TechEd Russia 2011の開催のちょうど1日前に、HPブースとその周辺で開催されるカンファレンスの2日間すべてで開催される、それほど劣らない驚くべきコンテストを発表する準備ができました。







そのため、HPとMicrosoftの技術に関する知識をテストし、上の図から賞品の1つを獲得したい場合は、 必ずラップトップを持参してください 。 TechEd展示エリアでWi-Fiネットワークをキャッチしたら、Internet Explorer *アドレスを入力してコンテストWebサイトにアクセスできます。



//hp-konkurs.teched



サインアップして、必ず実際のメールと電話番号を含めてください。 賞品をすぐに見つけるために電話が必要になるので、メインのログインにした-コンテストの進行状況を追跡します。



登録後、コンテストのタスクのリストが表示されます。 タスクは、HPおよびMicrosoftテクノロジーの実際の実践に基づいています。 すぐに解決できるものもあれば、前のものを解決した後に解決できるものもあります。 各問題を解決する方法は異なる場合がありますが、それぞれについて、実装を成功させるための独自の基準があります。



あなたの目標は、利用可能なすべてのタスクを他の全員よりも先に解決し、各タスクにできるだけ時間を費やすことではありません。 HPブースでは、すべての参加者と開いているすべての仮想マシンのデスクトップの現在の進行状況を表示します。 ところで、すべての仮想マシンは、HPのブースに直接インストールされているHPのクールなハードウェアで実行されます。



現時点で利用可能なすべてのタスクを解決したので（あなたを疑わない！）、急いでリラックスしないでください！ 私たちは常にタスクのリストを補充し、あなた自身を証明し、リーダーシップを強化し、ライバルに追いつくための追加の機会を与えます。 TechEdの2日目には、1日目のタスクの一部が使用できなくなるため、「後で」決定を先送りしないでください。



まあ、私は何も忘れていないようです。 そして、はい、パスポートを取得することをお勧めします。11月10日のTechEd閉会式で賞品を受け取り、彼と一緒に家に帰ることができます。



HPブースでお会いしましょう！



*クライアントコンピューターの技術要件は次のとおりです。



-Microsoft Windows XP 3以降（32ビットまたは64ビット）

-Internet Explorer 7以降（32ビットのみ ）

-ActiveX対応、1回限りのアプレットインストールが必要

-スタンドの周りの快適なオットマンに座っている間に膝の上にコンピューターを置く機能。 パフは他の人のスタンドに沿って引っ張らないでください！ :)
