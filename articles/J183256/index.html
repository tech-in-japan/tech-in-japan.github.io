Meizu MX2をFlyme 2.2.3に更新します

ハブラリュディ、おめでとうございます！ 6月上旬に、Meizu MX2のバージョンFlyme 2.2.3への別の更新がリリースされました。 以下に、主要な変更点のリスト、手動インストールの手順、およびファームウェアの公式バージョンとダブルステータスバー付きのカスタムへのリンクを示します。







何らかの理由でスマートフォンが自動的に更新されていない場合は、手動でフラッシュできます。

Meizu MX2用ファームウェアFlyme 2.2.3 [ mega ] [ Yandex.disk ]



update.zipアーカイブをダウンロードしてから、解凍せずにスマートフォンのルートフォルダーにドロップする必要があります。 次に、 音量を上げるボタンと電源ボタンを同時に押しながら、デバイスの電源をオフにしてから再びオンにします。 ファームウェアの更新ウィンドウが表示されます。 [データを同時にクリア]行の横にあるチェックボックスをオンにすることをお勧めします。デバイスは更新時にメモリをクリアします（したがって、バックアップした方がよい）が、更新後はファームウェアが不安定になる可能性が低くなります。 次に、[ アップグレード ]ボタンをクリックして約1分間待機する必要があります。アップグレードプロセスには約時間がかかります。 スマートフォンが再起動し、少し「考え」て、ファームウェアの更新プロセスがすべて完了します。



新しいファームウェアは基本的に、以前のバージョンのいくつかの欠点を修正します。

Flyme 2.2.3の変更点の完全なリストは次のとおりです。



ディスプレイ

  • [設定]> [画面]> [ボタンの明るさ]オプションを使用したときに、画面の明るさを誤って変更する問題を修正しました。


プリインストールされたアプリケーション

  • 削除されたGoogleアプリ：Google +、Google翻訳、Gtalk


電話番号

  • 通話中に誤って音量を下げる問題を修正
  • オーディオヘッドセットを使用して着信コールを拒否した場合に音楽の音量をゼロにリセットする問題を修正
  • ヘブライ語のシステム言語を使用しているときに「設定>モバイルネットワーク>ネットワークモード」を入力する際のバグを修正しました
  • 音楽とビデオ
  • 一部のFLVファイルが正しく再生されない問題を修正
  • 「音楽」>「最近」>「アルバム」と入力したときにアプリケーションを閉じる際のエラーを修正しました


カメラ

  • 「パノラマ撮影」モードの電源ボタンで電話機がロックされている場合のフラッシュのバグを修正しました
  • 「連続撮影」モードの電源ボタンで電話がロックされているときのフラッシュのバグを修正


時計/アラーム

  • アラーム設定の不完全なサウンドスケールの問題を修正
  • カスタム着信音を選択して保存するときにアプリケーションを閉じる際のエラーを修正


ギャラリー

  • 通知バーの電子メール通知に関する誤った作業を修正しました


その他

  • 「Phone Finder」サービスで電話をブロックした場合に、FlymeブラウザーでWebページを開く際のエラーを修正
  • 壁紙設定で「ホーム」ボタンをクリックしたときにプログラムを閉じるエラーを修正


Flyme 2.2.3を搭載したMeizu MX2のデスクトップは、以前のバージョンと比較して変更されていません。1つのステータスバーと、下端からアイコンブロックへの顕著なインデントが残っています。 ただし、これはカスタムファームウェアまたはパッチで修正できます。左側には公式のFlyme 2.2.3のスマートフォン画面が表示され、右側にはこのファームウェアに基づくカスタムが表示されます。







カスタムファームウェアはここで言うことができます： [ mega ] [ Yandex.disk ]

インストールプロセスは、公式ファームウェアの場合と同様です。



PS Flymeのディスカッションスレッドの1つで、Meizuのイデオロギー的かつ首謀者でありリーダーであるJack Wongが、奇妙なスクリーンショットを投稿しました。







Flyme 2.xの現在のバージョンのスタイルは維持されていますが、アイコンはよりフラットになっていることがわかります（iOS7のトレンド？）。 おそらく、これがFlyme 3の外観です。
