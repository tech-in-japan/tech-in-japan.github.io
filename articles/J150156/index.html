CSS Summit 2012、私の最初のオンライン会議





8月上旬に、私は4回目の年次CSS Summitオンライン会議に参加しました。 会議は3日間連続で開催され、CSS3トレーニング、高度なCSS、プリプロセッサの分野でレポートが分類されました。 すべてのレポートは英語です。



私たちにとって、これはオンライン会議に参加した最初の経験であり、クライアントの開発に特に興味がなくても、そのような会議を開催するための形式について読むことは興味深いかもしれません。



会議全体および個々の日の両方のレポートの記録は、公式ウェブサイトで購入できます。 20％割引のプロモーションコード-「20SNOOK」。



カットの下で、イベントの組織についてのいくつかの言葉、プレゼンテーションへのリンク、および興味深い論文の簡単な要約とウェブ開発についての考え。



どうでしたか



これらの日付に、チーム全体としてリガオフィスに特別に集まり、アメリカのタイムゾーンに合わせて調整しました（リガの時間によると、会議は17:00から24:00に開催されました）。 すべてのスピーカーが1か所から放送され、一般にイベントは通常のオフライン会議を放送するようなものであると予想されていましたが、残念ながら...スピーカーは自宅やオフィスから、彼らが言った（そして誰かがシートから読んだ）半眠りの声でかなり準備ができていませんでしたレポート。 すべてのセッションを通して、スピーカーは一般の人々に話しているのではなく、同僚にコーヒーを飲みながら、あらゆる種類の「ええと...」（寄生的な言葉）、さまざまなためらい、スライド付きのジャムをしているという印象でした。



それにもかかわらず、主催者自身が良い仕事をし、コミュニケーションの中断はなく、ビデオ録画は時間通りに配信され、報告書はプログラムに厳密に従っていました。 放送はAdobe Connectを経由しました 。
