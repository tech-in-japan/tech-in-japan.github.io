あるかどうか：職場でのWeb 2.0（事実の小さな束）





自分で判断する：





Web 2.0の使用は参加を意味します。 閲覧と読書が少ない-より多くのアクティビティと評価



web 2.0誰でも（まあ、またはほぼ全員）使用方法を知っている



Web 2.0の使用が効果的な分野は、次のものに関連しています。

-知識の普及

-問題の解決

-アイデアの評価を含むイノベーション計画

-インタラクション（顧客との対話を含む）/チームワーク



Web 2.0に対する企業の障壁

-セキュリティ（それでも！）

-機密性

-有効性（たとえば、ブログで評価する方法はまだ明確ではありません）

-企業文化（まあ、いつものように）

-弁護士（どうして彼らなしで？）

-企業製品が関連付けられている品質サポートと実装の欠如（！）

-投資収益率に関する一般に認められた財務モデルの欠如



ほとんどの大規模プロジェクトは小規模パイロットとして開始されました。たとえば、部門wiki



実装におけるブレーキの主な理由は、これがすべて作業効率を向上させることができる方法の大ボスの誤解です



ただし：



イングランドでは、Web 2.0テクノロジーに1ペニーに投資した企業はわずか32％でした。



McKinseyの調査によると、企業はWeb 2.0テクノロジー（ブログ32％、ポッドキャスト35％、wiki 33％）よりも自動化（企業のWebサービス80％、コラボレーションシステム47％）に投資する可能性がはるかに高いことが示されています



次に、この調査について説明します。




Web 2.0を実装した企業の56％は、内部相互作用の改善、53％-チームビルディング、50％-組織のオープン性の向上について述べています。



Web 2.0の使用方法：

70％-顧客との対話

51％-サプライヤーおよびパートナーとの相互作用

75％-内部従業員の対話管理

20％-ブログを使用して顧客サポートを改善する

50％強-組織内のナレッジマネジメント

50％弱-新製品開発用



Web 2.0を実装した企業の半数以上が投資に非常に満足しており、失望しているのはわずか13％です。



ここに事実がありますが、おそらく役に立つでしょう。 そうでない場合は、いいえ）
