Intelでゲームを開発する：GPAと5月1日のウェビナー



Civilization、STALKER、またはRise of Flightなどのゲーム業界の傑作で共有できるものは何ですか？ これが驚くほどリアルなグラフィックとユニークな中毒性のゲームプレイであると言うなら、間違いなく正しいでしょう。 しかし、コンピューティングリソースの要件が高いにもかかわらず、これらのゲームはすべて予算のハードウェアでうまく機能することを忘れないでください。 そして、何時間ものゲームプレイを楽しむために、強力な個別のグラフィックカードを備えたゲームシステムを用意する必要はありません。 はい、開発者はさまざまなゲームプラットフォームの機能を考慮しました。 そして、おそらく、インテルグラフィックパフォーマンスアナライザー（インテルGPA）が困難な最適化ビジネスにおいて非常に貴重な支援を提供したことを皆さんが知っているわけではありません。



このツールを使用すると、ゲームのパフォーマンスが制限されている理由を簡単に克服できます。 それでもIntel GPAに慣れていない場合は、実際に試してみましょう。 特に、7月1日金曜日に、 Intelグラフィックツールに関するワークショップを開催し、アプリケーションのパフォーマンスを最大化するために作業中にそれらを使用する方法を説明します。





7月1日11:00-Adobe Connectプラットフォームでの会議



7月1日、モスクワ時間の午前11時に、ウェビナーの放送を開始します。 参加するには、 IntelCorp.regsvc.com / E5928で事前登録が必要です。 登録フォームに記入して送信すると、「仮想オーディエンス」に入るためのリンクが記載されたメールが届きます。 参加するには、追加のソフトウェアは必要ありません。WebブラウザーとAdobe Flash Playerのみが必要です。これは、99.9％の確率でPCに既にインストールされています。



スピーカーは誰ですか？



このウェビナーは、Philipp Gerasimovが主催します。PhilipGerasimov氏は、「 5週間のIntel 」プロジェクトに精通しているかもしれません。 KRIで彼に会うこともできます。彼はIntel GPAを代表し、著名な企業がどのようにゲームの印象的な最適化結果を達成したかについて彼の助けを借りて話しました。 それでもCWIを訪問する機会がなかった場合は、今回のフィリップの報告を見逃さないように、事前に登録することをお勧めします。



それはどうなるのでしょうか？



このイベントは、コンピューターゲームの最適化の問題に特化しています。 パフォーマンスを分析し、グラフィックプログラムをデバッグするためのIntelのツールについて学習します。これらのツールは、すべてのゲーム開発者の武器になります。 もちろん、プログラムのハイライトはIntel Graphics Performance Analyzers（Intel GPA）です。 この製品は2009年にCWIで初めて導入され、グラフィックスを最適化する独自のアプローチにより、開発者の間で非常に人気がありました。 少し前に、 Intel GPA 4.0ツールの新しいバージョンがリリースされました。これには3つの主要なモジュールが含まれています。

  • System Analyzer HUD（ヘッドアップディスプレイ）は、パフォーマンスを監視および分析するためのダッシュボードです。 CPU / GPU / Direct3Dメトリックは、ゲームフレームの上にリアルタイムで表示されます。 このツールは、最適化に向けた最初のステップです。 アプリケーションのパフォーマンスが制限されているかどうかを判断できます-CPUまたはGPU。
  • フレームアナライザーは 、キャプチャされたフレームの詳細な分析とデバッグのために設計されており、最も困難なD3D API呼び出しを見つけてボトルネックを解消できます。
  • Platform Analyzerは、すべてのCPUコアのスレッドパフォーマンスを視覚化するツールです。


ウェビナーでは、 Intel Graphics Checkerについても説明します。 このツールは、Intelパートナープログラムの一部として提供されており、アプリケーションのパフォーマンスをすばやく評価したり、コードの最適化に関する推奨事項を含む詳細なレポートを受け取ったりすることができます。 新しいバージョン3.0には、Intel GPAとの統合が含まれています。 したがって、FPSアプリケーションに加えて、このツールを使用すると、中央およびグラフィックプロセッサの使用効率に関する情報を取得できます。



Intel GPAおよびIntel Graphics Checkerツールは、グラフィックアプリケーションの構築と最適化に本当に役立ちます。 7月の朝にウェビナーをご覧ください。実際の例では、これらのツールを使用してボトルネックを迅速に特定し、コードの最適化を実行する方法がわかります。 また、 Intelパートナープログラムのメンバーにとっての新しい利点と、ツールを無料で入手する方法についても学びます。
