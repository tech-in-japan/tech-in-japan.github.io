Visual StudioでCtrlキーを使用できるもの

1つのシンプルなCtrlキーで、Visual Studioでの作業を大幅に簡素化できます。 私たちによく知られている典型的な組み合わせに加えて、あまり知られていないがより価値のあるものも数多くあります。



以下は、Ctrlキーでできることのリストです。





編集と印刷：



Ctrl + C、Ctrl + V、Ctrl + X、Ctrl + S-最もわかりやすい

Ctrl + P-あなたは決して知らない、突然誰かがコードを印刷したい

Ctrl + Del、Ctrl + Backspace-カーソルの右側、カーソルの左側の単語を削除します

Ctrl + Enter、Ctrl + Shift + Enter-現在の上、現在の下に線を追加

Ctrl + J-完全な式入力

Ctrl + U-カーソルの前の文字を小文字に変換します

Ctrl + Shift + U-カーソルの前の文字を大文字に変換します

Ctrl + =-前のカーソル位置へのコードの選択

Ctrl +左、Ctrl +右-前の単語に移動、次の単語

Ctrl + Shift +左、Ctrl + Shift +右-カーソルの左側、カーソルの右側の単語を選択

Ctrl + K、Ctrl + C（組み合わせ）-行をコメント化する

Ctrl + K、Ctrl + U（組み合わせ）-行のコメントを外します



検索とナビゲーション：



Ctrl + F、Ctrl + Shift + F-ドキュメント内の検索、ファイル内の検索

Ctrl + H、Ctrl + Shift + H-文書内の置換で検索、ファイル内の置換で検索

Ctrl + F3、Ctrl + Shift + F3-選択したテキストを順方向および逆方向に検索

Ctrl + F4-現在のウ​​ィンドウを閉じます

Ctrl + F6、Ctrl + Shift + F6-次のウィンドウ、前のウィンドウに移動します（最後に使用した原則に従って）

Ctrl + Tab、Ctrl + Shift + Tab-同じ効果、ウィンドウの表示のみ

Ctrl + Alt +上、Ctrl + Alt +下-開いているウィンドウのドロップダウンリスト

Ctrl +--前のカーソル位置に移動



-ツールバー上

Ctrl +上、Ctrl +下-次のツールグループ、前のツールに移動

Ctrl + PageUp、Ctrl + PageDown-次のウィンドウに移動する、前の



コンパイルとデバッグ：



Ctrl + F5-デバッグなしで開始

Ctrl + Shift + B-ソリューションを構築する

Ctrl + D、Ctrl + E（組み合わせ）-デバッグ時の例外のリスト

Ctrl + Shift + F9-すべてのブレークポイントを削除



リファクタリング：



Ctrl + R、Ctrl + M（組み合わせ）-メソッド選択

Ctrl + R、Ctrl + E（組み合わせ）-メソッドのカプセル化

Ctrl + R、Ctrl + I（組み合わせ）-インターフェースの選択

Ctrl + R、Ctrl + V（組み合わせ）-パラメーターを削除

Ctrl + R、Ctrl + O（組み合わせ）-パラメーターの順序を変更する



その他：



Ctrl + I-カーソルを双眼鏡に変えます

Ctrl +スペース-IntelliSenseプロンプトを表示する

Ctrl + Alt + A-コマンドラインを開く

Ctrl + K、Ctrl + I（組み合わせ）-ツールチップ

Ctrl + K、Ctrl + X（組み合わせ）-テンプレートの挿入

Ctrl + K、Ctrl + F（組み合わせ）-選択したテキストのフォーマット

Ctrl + Alt + Space-入力時に関数パラメーターを出力

Ctrl + Shift + A-新しいアイテムを追加



-ブックマーク（すべて組み合わせて）

Ctrl + K、Ctrl + K-ブックマークを設定する

Ctrl + K、Ctrl + N-次のブックマークに移動

Ctrl + K、Ctrl + P-前のブックマークに戻る

Ctrl + K、Ctrl + L-すべてのブックマークを削除



そして最後に、私のお気に入りはCtrl + '。'（ラテン語にドットを追加）です

最も一般的な不正確さは、まだ作成されていない、または接続されていないクラスを使用しようとすることです。 このために、Ctrl +の組み合わせがあります。 クラス名の後にクリックするだけで十分です。ポップアップウィンドウが表示され、必要なライブラリを含めるか、新しいクラスを作成できます。



これで、おそらく、リストは完全です。 重要な何かが見落とされており、重要でもない場合はお知らせください。
