ビデオをGoToMeeting（g2m）からwmv3に変換する方法

私の仕事の責任には、社内従業員向けの社内トレーニングとウェビナーの実施が含まれます。



GoToMeetingを通じてウェビナーを実施しています。



ただし、GoToMeetingからのビデオは、開くことができる完全に珍しいwmvファイルとして保存されます

  • 特別なコーデック（g2m）をインストールした後にのみ、
  • Windows（またはポピー）でのみ。
g2mコーデックがないと、ファイルは次のように開きます。音声があり、画像がありません。



そして、従来の方法を使用して同様のファイルを他の形式に変換することは、完全な失望です。 すべての編集者がこれに対処するわけではなく、結果は次のようになります。ビデオは高速で再生されますが、サウンドは急いでいます...



問題は解決しました



アクションの一般的なロジック

  1. 最初に、ネイティブのGoToMeetingツールでウェビナーの記録を変換する必要があります。

    その結果、同じwmvファイルを取得しますが、g2mではなくwmv3コーデックでエンコードされます。 このようなファイルは、Ubuntu（標準のコーデックがすべてインストールされている場合）でも、通常の方法で既に通常のプレーヤーで開くことができます。
  2. 次に、必要に応じて、ファイルを別の形式に変換します。 たとえば、aviで。
  3. Aviファイルはすでに簡単に編集できます...


環境の準備



1. g2mコーデックをインストールします。

* Windowsおよび/ dev / handsを起動します/

* コーデックをダウンロードします。

*インストール。

2.インストール結果を確認します。C：\ Program Files \ Citrix \ GoToMeeting \ 799 \



ディレクトリパスの番号は、インストールされているコーデックのバージョンによって異なります。 この場合、G2mバージョン5.0ビルド799がインストールされているため、結果のディレクトリは「799」と呼ばれます。



ブロードキャストを開始する前に、ビデオの録画方法を基本的に設定する必要があります。



1.インターネットに接続していることを確認してください。 これがないと、g2mは起動しません。

2. G2mを起動します。 「オレンジ色の雪片」g2mがトレイに表示されます。

3.スノーフレークのプロパティを呼び出します（右クリック> [設定]）

4.次のパスに従います。[カテゴリ]> [録画]> [ビデオ]> [Windows Media Playerファイルに変換]> [OK]。



ウェビナーを実施する



モニターでシャーマニズムの記録を有効にします。



変換用のビデオファイルを準備する



結果のビデオファイルをハードドライブの別のディレクトリにコピーします。



お勧め



1.提案されたファイルに後続の変更が直接適用され、元のファイルが危険にさらされることはないため、実際にファイルをコピーします。

2.ディレクトリに「C：\ 123」のような単純で短い名前を付けます。

3.ファイルに「456.wmv」のような単純で短い名前を付けます。



命名の単純さと短さにより、引用符の使用を避けることができます。 引用符が不当に「トランスコーディング初期化エラー」エラーにつながる場合がありますが、必要ありません。



g2mからwmv3へのファイルのトランスコード



1.コンソールを起動します（またはFarを使用します）。

2.コンソールで、g2mコーデックがインストールされているディレクトリに移動します

> cd "c:\Program Files\Citrix\GoToMeeting\799"





3.コーデックを含むディレクトリから次のコマンドを使用して、「ソース」パラメーターとしてトランスコードするファイルを指定してg2mtranscoder.exeプログラムを実行します。

> g2mtranscoder source=EXPECTED_FILE





4.例：

> g2mtranscoder source=C:\123\456.wmv





5.変換プロセスを示す別のg2mウィンドウが表示されます。



物事はゆっくりですが、確実に待っています。



結果



外側では、何も変わりません。



ディスク上の同じファイル、ウィンドウ外の同じ風景。



しかし、突然、C：\ 123 \ 456.wmvがUbuntuを含むすべてのもので開くことができるようになりました。



したがって、他の形式に簡単に変換できます...
