Ubuntu 「頭に思い浮かぶ」または論争する理由

著者のKottenatorの 保守主義に対する明るい未来であるUbuntu Unityの最近のトピックを読んだ後、私はこの記事の議論が好きでした。 ロシアでは慣習的であるように、すべてのユーザーは自分自身の上に毛布を引っ張ります。 それでは、Ubuntuを思い浮かべた理由をお話ししましょう。



新年のセール（2007年が近づいていました）のある日、Acer Aspire One ZG5ネットブックを購入しました。 内部には、面白いものはほとんどありませんが、ノートブックは便利なものです。 彼の工場のOS Linpusをしばらくの間、私は何かを変える必要があることに気づきました。 選択はubuntuに落ちました。



ubuntaのインストールはとても簡単で、強制されませんでした。 しかし、その後、「はい、hemo核が発生し始めました。」 セットアップは長く、多くの質問、紛争、誤解などを引き起こしました。 最近、新しくリリースされたUbuntu 10.04ディストリビューションが入ったボックスを手に入れました。 そして私にとってはすごかった。 以前の配布と比較して、多くのアクションが単に必要なくなっただけです。 しかし、会話はそれについてではありません...





Ubuntu 10.04をインストールした後、数日苦しみ、変更を始めました。



不要なローカライズを削除する


「ノートブック」にかなり小さな場所（すべてについてのすべてについて8 GB）があるため、スペースの可用性を監視する必要があります。 私の喜びには、マシン上ですべてを実行できるプログラムが見つかりました-localepurge。

sudo apt-get install localepurge





インストール中に、どのロケールをそのままにしておくかを指定するように求められ、最初の「クリーニング」が開始されます。 作業に必要なロケールアプリケーションを保存するために、次のものを残しました。

en

en_US

en_US.ISO-8859-15

en_US.UTF-8

ru

ru_RU

ru_RU.CP1251

ru_RU.KOI8-R

ru_RU.UTF-8





localepurgeを実行した後、約300 MBのガベージが「クリーニング」されました。 別のプラス-アプリケーションをインストールするたびに-設定に従って自動的に実行され、新しいアプリケーションがクリーンアップされます。



アーカイバ


ビルトインで既に非常に快適に構成されたアーカイブマネージャーを使用すると、グラフィカルインターフェイスですべての作業を実行できます。 しかし、これには1つの欠点があります-rarアーカイバの欠如です。 火星からの信号の影響、偏見、多分わかりませんが、rarをデータアーカイバとして使用するのは不適切なようです。 私の意見では、それは90年代のさまざまなプログラムに属します-グラフィカルインターフェイスを備えたWindowsマシンの唯一のアーカイバは「彼」でした。

この問題は、1つのコマンド（おそらく2つ-設定されているリポジトリに応じて）で処理できます。

sudo apt-get install rar

sudo apt-get install unrar







ネットワークの構築


Windowsを実行しているホームネットワークにさらにいくつかのコンピューターがあるため、それらのリソースにアクセスするための設定を行います。

理由はわかりませんが、Ubuntuのインストール時にsambaが自動的にインストールされませんでした。 しかし、何も-あなたの手でそれを行いません。

sudo apt-get install samba

sudo apt-get install cifs

sudo apt-get install cifs-utils





その後、すべてのリソースを表示および編集できるようになります。

また、すべての音楽とビデオをいずれかのコンピューターに保存し、システムの起動時に自動的に必要なフォルダーにパーティションをマウントしたいという要望があります。

/ etc / fstabに必要な行を追加します（このようになりました）

//mousetrap/music /home/mouse495/music cifs rw,nosuid,nodev,allow_other,username=user,password=Rah7chis,file_mode=0777,dir_mode=0777 0 0





ここで修正する必要があるのは

// mousetrap / music-一般的なWindowsリソースの名前で、

/ home / mouse495 / music-マウントするアドレスへ、

ユーザー名=ユーザー -それぞれ、Windowsおよび

password = Rah7chisは、それぞれこのユーザーのパスワードです。



Nautilusをカスタマイズする


多くの初心者は、「アドレスバーはどこですか？」と疑問に思います。その後、 Ctrl + Lの組み合わせがあり、あまり満足していないことがわかります。

この問題の解決策は非常に簡単です-コンソールで、次のように入力します。

gconftool -s --type bool /apps/nautilus/preferences/always_use_location_entry true





バウル！ アドレスバーは常に表示されます！



更新しています...


ある程度の一貫性を持ってUbuntuを使用している人は、更新マネージャーが常に「何かを更新する」独自の方法を取得し、提案を絶えずクロールしていることを知っています。 私も好きではありません。 したがって、次のことを行います。

gconftool -s --type bool /apps/update-notifier/auto_launch false





それだけです！ 彼はもう私たちを気にしません！



Skype通話


私にとって最大の失望は、Skypeから誰かを呼び出す問題でした。 ビデオでは、すべてが問題なく、音声でも問題ありませんが、マイクには問題があります。 しかし、何も解決されません！ サービスでの過去の仕事を思い出して、ラップトップやネットブックのマイクがモノラルであることをよく知っていました。 ここから掘ります。

最初に必要なことは、パルスオーディオからボリュームコントロールを設定し、標準のalsaに置き換えることです。

これを行うには、単純にpulse-audioをインストールします：

sudo apt-get install pulse-audio





インストール後、念のため、再起動します。

設定を調整するには、パルスオーディオからボリュームコントロールを実行し、[入力デバイス]タブを選択して、[フロント左]を80％に、[フロント右]を10％に設定します。 Skypeを起動し、今聞いたことを喜んでいます。
