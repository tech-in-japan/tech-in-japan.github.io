Data Fest 2018：発表と登録





4月28日にFLACONデザインファクトリーの領域で開催される第5回モスクワデータフェストにご招待します。 Data Festは、機械学習、データ分析、処理、およびプレスがAIと呼んでいるものに関連する研究者、エンジニア、開発者向けの最大の無料カンファレンスです。



Mail.Ru Group製品のAIとMail.Ru Mailの「スマート」な回答、VKontakteとOdnoklassnikiでの推奨事項とコンピュータービジョンの機能、Alibabaでの機械翻訳、Quantum Machine Learningの詳細などを学びます。別の！



会議への入場は無料ですが、事前の登録が必要です。



Data Fest Moscow 2018について



Data Fest 2018-7つの部屋に30を超えるセクションがあり、ここでデータサイエンスの世界に真っ向から飛び込み、急速に発展しているこの分野の専門家のコミュニティがどのように生活しているかを調べます。



ロシアの主要研究グループの特別ワークショップ：



  1. MIPTのコンスタンチン・ヴォロンツォフグループによるBIGARTMマスタークラスでテキスト分析とテーマモデリングを教えます。
  2. Evgeny Sokolovのグループ、HSEによるRecSysマスタークラスのレコメンダーシステムのベストプラクティスを示します。
  3. Skoltech、Ivan OseledtsのグループのマスタークラスDeepTensorでテンソルメソッドを理解するのに役立ちます。
  4. Dmitry VetrovのグループHSEによるDeepBayesワークショップでベイジアンアプローチの深さを示します。
  5. SkoltechのVictor LempitskyによるDeep Computer Visionグループのマスタークラスでコンピュータービジョンがどのように機能するかを説明します。
  6. 会話インテリジェンスと対話システムは、iPavlovマスタークラスで、ミハイルバーツエフのグループであるモスクワ物理学技術研究所によって指導されます。
