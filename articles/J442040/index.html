事業開発の効果的なツールとしてのパートナーシップ

Khabravchaneの紳士、私は今日、「戦略的パートナーシップ」とは何かを検討することを提案します。

厳密に判断しないでください、これは私の最初の公開記事です。 猫の下にようこそ。













今日は、次の問題を検討することを提案します。







  1. パートナーシップとは何ですか？
  2. パートナーシップの種類と目標は？
  3. パートナーシップの開発をどこから始めますか？
  4. 次はどこへ行きますか？


まず、生き生きとした非常に成功した例に基づいて、戦略的パートナーシップが何であるかを見てみましょう。







1.戦略的パートナーシップとは何ですか？

80年代初頭、当時ほとんど知られていなかったMicrosoftという名前で、当時5年以上にわたって市場に存在していた若くて野心的な会社がIBMと契約を結びました。 契約では、新世代のIBM PCのすべてのコンピューターに、Microsoftが開発したDOSオペレーティングシステムが搭載されると述べています。

このようなパートナーシップの結果、Microsoftは販売された各パーソナルコンピューターに対して報酬を受け取るようになりました。 83年目には、同社の収益は5500万ドルに達し、それに加えて、同社は市場で最も影響力のある開発者となり、1990年には、コンピューター市場におけるマイクロソフトのシェアは90％を超えました。 今日、マイクロソフトの資本金は807.66億ドルです。







もちろん、マイクロソフトはこのビジネス開発の方法論の先駆者ではありませんでした。

戦略的パートナーシップのメリットを最初に目にしたのは航空会社でした。 最初の航空会社が航空会社やホテルと一連の契約を締結した後、ビジネスはパートナーシップチェーン全体で積極的に発展し始めました。

このような効果的なツールが市場に登場した後、いわゆる「アフィリエイトプログラム」がビジネスの他の分野に押し上げられました。

今日、アフィリエイトネットワークは、さまざまなキャッシュバック、パートナーストアでの購入の割引クーポンなどの形で毎日見つかります。

上記のすべてを要約すると、戦略的パートナーシップは、ブランド、製品、またはサービスを促進するための効果的なツールと言えます。







2.パートナーシップと目標の種類は？







そもそも、パートナーシップは2種類しかありません。1つ目は登録されたパートナーシップで、2人以上が参加します。2つ目は、合法的に記録されていないが、すべての参加者が存在するサポートです。

組織の形態については、次の分野を区別できます。







•最も一般的な形式は、商業パートナーシップです。

あるビジネスのターゲットオーディエンスが別のビジネスのターゲットオーディエンスと同じ場合。

例：

カーディーラー。

ディーラーほど、ビジネスパートナーシップに関心はありません。

パートナーシップにおける彼らの主な目標と利益は、自動車を転売する機会を得ることではなく、彼らが販売した自動車を修理できるようにすることです。

従来のディーラーセンターには、ビジネスの純利益の大部分を稼ぐサービスステーションエリアが常に含まれています。

これがどんなに馬鹿げているように聞こえるかもしれませんが、ほとんどの場合、販売センターは実質的にお金を稼ぐことなく大きな割引であなたに車を与えることができますが、この方法で彼は年に一度のメンテナンスと修理のために彼に来る忠実な顧客を得ます。

同様に、自動車メーカーは、このようなパートナーシップから、製品の別の販売チャネルを受け取りますが、それ自体を販売することはなく、生産に集中することができます。







•パートナーシップの次の形態は商業形態ではありません。

この形式は、製品またはサービスの促進における自発的な協力と支援を意味します。

非営利的なパートナーシップの例として、私はソーシャルネットワークとしての役割を果たすことができます。

グループの2人の所有者は、聴衆の相互交換に同意し、リンクグループでの公開によって読者をパートナーグループに招待し始めます。 または、有名なブロガーが別の有名なブロガーを彼のチャンネルに招待して、エピソードの共同録音をするとき。 視聴者の特定の「リダイレクト」が判明します。







•パートナーシップの3番目の形式は、有限パートナーシップです。

このようなパートナーシップでは、各当事者は以前に合意したフレームワーク内で単独で責任を負います。 多くの場合、このタイプのパートナーシップは、クライアントとのコンタクトのサイクル全体が異なるパフォーマーに分割される「パートナーシップ連合」で使用されます。

例として、ターンキーベースで運営している建設会社を選び出すことができます。

もちろん、そのようなスキームでは、そのフレームワークにプロジェクトの制御と管理が含まれるパートナーが存在します。 法律文書や技術文書に関するすべての問題を解決するパートナーがいます。 3番目のパートナーが建築材料の供給を担当し、4番目のパートナーが建築します。 ご覧のとおり、ここでは、各パートナーが作業の各部分に対して個人的な責任を負っています。

多くの場合、このタイプのパートナーシップがLLCやJSCなどの本格的な協会に発展するには、いくつかの完了したプロジェクトで十分です。







•無視できないパートナーシップの次の形式は、完全なパートナーシップです。

このようなパートナーシップは、この協力のすべての当事者の責任を意味します。

このタイプのパートナーシップは、日常生活のどこで見ることができますか？

このような経験は、複数の企業が共同開発した製品で確認できます。 良い例は、Apple Payの積極的なプロモーションと使用です。 世界最大の顧客ベースの1つを所有するマスターカードとのパートナーシップのおかげで、Appleはキャッシュレス決済市場を非常に迅速かつ効率的に拡大しています。

2014年末から現在まで、約1億2,700万人が非接触型決済にApple Payを使用しています。 大規模でアクティブな支払いシステムのいずれも、このような成長のダイナミクスをまだ示していません。







•そして最後の、しかしすべての可能な形態の最後からは程遠い-戦略的パートナーシップ。

この形式には、長期的な協力が含まれます。 同様のタイプのパートナーシップは既成のビジネスに固有のものであり、市場で強化されており、ニッチを拡大し始めています。 このタイプの協力では、それほど単純ではありません。 ビジネスパートナーがあなたの会社よりも市場で大きなプレーヤーであり、ビジネスセクターがこのパートナーの発展のために戦略的である場合、会社が合併または吸収される可能性が高くなります。 この提携が相互に有益である場合は良いことですが、あなたのビジネスが選択されるか、その統合が別のビジネスと統合される必要があるようなフレームワークに追い込まれる瞬間を除外することはできません。 したがって、戦略的パートナーシップと言えば、得られる見込みと結果を常に予測する必要があります。







3.パートナーシップの開発をどこから始めますか？







常に、ビジネスの促進または開発を目的としたプロセスを実行する前に、自分で始める必要があります。

あなたの会社が何をしているのかを決めた場合; 会社が市場でどのように位置付けられているか。 あなたのビジネスはクライアントにどのような価値をもたらしますか？ あなたのビジネスのクライアントは誰ですか。 このクライアントの場所-あなたはすでに巨人の仕事をしているので、このステップを安全にスキップして次の段階に進むことができます。そうでない場合は、これらの点を調べてすべての質問に答えることを強くお勧めします。 また、これらの質問すべてに正直に答えることなくして、効果的なアフィリエイトチャンネルを構築することはできません。

ビジネスの主要なパラメータを自分で決定した後、パートナーの検索に進みます。

検索時に注意する必要があるもの：







•視聴者が交差するビジネスの種類。

ここではすべてがシンプルで、同じオーディエンスが異なる市場セグメントでビジネスとやり取りすることができます。購入後または購入前にクライアントがどこに行くかを決めるだけです。







•クライアントとのタッチレベルを決定します。

パートナーシップの非常に重要な要因は、すべてのパートナーとクライアント間の相互作用の瞬間です。 この点で、クライアントとの仕事のどの段階でビジネスが含まれるかを理解することが重要になります。

メイン製品をクライアント（スマートフォンなど）に販売してから、クライアントがパートナーのWebサイトにアクセスしてSIMカードを登録し、その後、クライアントが別のパートナーの店に行って新しいスマートフォン用のアクセサリーを購入すると便利です。 この場合、「resする」パートナーがあなたのところに来て、あなたから何も買わないという事実を心配することはできません。

しかし、このチェーン全体で、あなたが最初のタッチではない場合はどうでしょうか？

ここでは、パートナーとのコミュニケーション、クライアントとの彼の仕事の瞬間の研究のみが役立ちます。 パートナーを見つける段階でこれを行わないと、時間を失う可能性があります。 視聴者の忠誠心を高めようとしないパートナーと一緒にいると、ビジネスを効果的に構築できる可能性は低くなります。







•コミュニケーション。

したがって、これまでのすべての手順を完了し、すべてを調査し、潜在的に協力関係を確立できる企業の膨大なリストを作成しました。

質問が発生します：話し始めるには？ 共通の興味を見つける方法

パートナーを引き付ける方法はたくさんありますが、実際には、顧客を引き付ける方法とそれほど違いはありません。 それでも、何年もテストされてきたものがあります。「文字+電話」です。

電話で潜在的なパートナーと通信するには、情報の理由が必要です。

手紙は、有益で簡潔で、特定のパートナーシップの申し出を含む必要があります。 手紙を書くとき、その目的はパートナーをあなたに変えることではなく、まずそれを会議に結びつけることだということを理解する必要があります。 これが対面の会議であるか、たとえばSkypeであるかは関係ありません。 電話は常に手紙への関心を示す機会として機能します。例えば：「こんにちは！ 私の名前はイヴァン・イワノフです。メールで手紙を送りました。 オリエント、受け取った？」 そのような電話の後、潜在的なパートナーの99％があなたが送った手紙を読むために間違いなく入ります。







•ミーティング。

アフィリエイトチャネルを形成するとき、パートナーと会うことが非常に重要です。 そして、最初の会議からパートナーにこれを浸透させることも同様に重要です。 メールによるやり取りを通じて、または一般的に、専任のマネージャーが会社間でやり取りしている間は、アフィリエイトチャネルの開発について話すことはできません。 結果はビジネスリーダーと共同開発計画の間の直接的なやり取りによってのみ得られますが、日常的な作業と、継続的な監視が不要なプロセスの一部をマネージャーに安全に委任できます。







4.次に進むべき場所は？

それで、あなたはすでにあなたのパートナー企業を決定し、彼らと話し、あなたの協力のタイプに応じて多くの合意をしましたが、次は何ですか？ インタラクションのオプションは何ですか？

実際、すべての手続きが完了し、製品とサービスを共同で販売できるようになると、計画が非常に重要なポイントになります。 共同活動のロードマップについて話し合うためにパートナーと合意します。

この段階では、アクティビティ自体を整理するだけで問題が発生する可能性があります。 パートナーには常に十分な能力や有能な従業員がいるとは限りません。

アフィリエイトプログラムの一環として実施できる膨大な数の活動にもかかわらず、私は次のことを強調しています。







•共同プロモーションとボーナス、いわゆるクロスマーケティング。

クロスマーケティングには明らかな利点があります。







  1. パートナーのオーディエンスによる顧客フローの増加。
  2. 市場でのユニークなオファーの形成。
  3. マーケティング予算を削減します。
  4. 忠誠心を高めます。
  5. ブランド認知度をさらに高めることができます。


「クロスマーケティング」の例は、Duracellバッテリー専用のOral-B歯ブラシです。MacDonaldのレストランチェーンでHappy Mealを購入すると、おそらく映画に登場する新しい漫画のおもちゃが見つかるでしょう。







•次に強調するツールは、内部割引プログラムまたはいわゆる「共同ブランドカード」です。

確かに、あなたのそれぞれの財布には、店舗が何であれ、少なくとも1枚のボーナスカードがあります。 いわゆる「ロイヤルティの割引」を提供するか、ポイントを累積してさらに計算します。

提携ブランドのカードでも同様のことが行われますが、単一の店舗または店舗チェーン内ではなく、アフィリエイトネットワーク内、および必ずしも相互接続されていないニッチ内でもボーナスを使用できます。

ボーナス、キャッシュバック、ポイントを計算することでクライアントを巻き込んで保持するこのプログラムの助けを借りて、パートナーはアフィリエイトネットワークのオペレーション数の増加を刺激します。 これにより、販売量とこれらすべてが増加します。







•共同製品。

このツールは当然「共同ブランディング」に起因しますが、それでも開発の別の方向としてそれを選び出します。

共同製品は、その生産の技術的可能性、ある方向または別の方向の能力の欠如、市場への迅速な参入能力の欠如、および単に経済制裁の回避策に関する多くの問題をしばしば解決できます。

今日、私たちは、反ロシア制裁を回避するために、ロシア連邦の既存の提案に基づいて共同製品を単純に採用および開発する西洋の投資家を観察しています。
