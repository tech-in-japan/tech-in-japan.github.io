FSFがマイクロソフトに転向：違法化することはできません

GPLv3ライセンスのアップグレードを無視するというマイクロソフトの意図は違法です。 レドモンドの巨人に対するこのような警告は、Free Software Foundationによって公開されました。



これは、Novell、Xandros、Linspire（これら3社は公式のMicrosoftパートナーです）との契約の下でサポートするGPLソフトウェアライセンスの3番目のバージョンを受け入れる義務はないと思われるというMicrosoftのスキャンダルな声明です。 たとえば、SUSE Linux Enterprise Serverシステムの場合、ライセンスをアップグレードするつもりはありません。



そして2か月後、Free Software Foundationの公式の対応が続きました。 FSFの弁護士は、Microsoftの操作を分析し、それを小切手と宣言しました。つまり、法を破ったと直接非難しました。



FSFステートメントのテキストから引用します 。



GPLv3をめぐる状況に関するマイクロソフトの解釈に同意しません。 マイクロソフトは、他の人の著作権の下で引き受けられた義務の暫定的な拒否の行為をコミットできません。 マイクロソフトがGPLv3に基づいてライセンスされた作業を配布する場合、または他の人にそれを代わって配布する場合は、ライセンスの条件に従って配布する必要があります。 彼女は... GPLv3要件の例外を宣言できません。



マイクロソフトは、いわゆる「知的財産」の尊重を期待していると述べています。これは、特許法と著作権法およびその他の無関係な法律を混同し、それらが行う他のことを隠すために設計されたプロパガンダ用語です。 マイクロソフトは、リソースが十分であり、ライセンサーを保証していることを保証します。マイクロソフトが著作権を尊重し、ライセンスの条件を順守することを保証します。
