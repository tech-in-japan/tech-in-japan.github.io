ブートメニューGRUB2を編集するためのGrub Customizerプログラムマネージャー





Grub Customizerは、gui上のGRUB2の新しい設定マネージャーです。 現時点では、ブートローダーの選択メニュー項目の名前変更、並べ替え、削除/追加、非表示が可能です。



もう1つの興味深い事実は、GRUB自体を再インストールしても、変更が消えないということです。



Ubuntu PPA（LucidおよびMaverick）からGrub Customizerを入手可能



sudo add-apt-repository ppa：danielrichter2007 / grub-customizer

sudo apt-get update

sudo apt-get install grub-customizer



ちなみに、BURGもサポートしています。



私は個人的にUbuntu 10.10を確認しました。brick812、GamePad64はそれらのおかげで、ubuntu 10.04および10.10でBURGを使用できることを確認しました。
