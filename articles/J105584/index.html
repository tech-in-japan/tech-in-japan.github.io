エントリー



私は、非常に多くの関心を持っている多くの人と同じように、Play Station 3をハッキングしようとして失敗するのを長い間見ていました。しかし、ある日、PS Jailbreakについてのニュースがありました。 その後、この奇跡のデバイスまたは偽物ではないことが明らかでした。 はい、そしてそこにシャーマニスティックなデバイスを挿入するだけでプレフィックスに侵入することは面白くありません。 しかし、私はPS3のシーンをもっと詳しく見始めました...



PS Groove-オープンソースバージョンのジェイルブレイクが登場しました。 そのため、特定のUSBコントローラーを使用してフラッシュし、 PS3脱獄のアナログを取得することができました（登録が必要です）。 しかし、私の手は、このガイドで説明されているアクションを実行するために間違った場所から成長しています。 さらに、特別なUSBコントローラーが必要でしたが、これは持っていませんでした。



その後、 PS Grooveをさまざまな興味深いスマートフォンに移植し始めました。 最初はN900でした。 Play Station 3を購入したのは、このデバイスで脱獄がリリースされた日でした。



N900の幸運な所有者である私の友人は、すぐにマニュアル全体を完成させ、ハッキングされたシステムを手に入れました。



ゲームのバックアップコピー（ジェイルブレイクの主なポイント）をプレイする場合は、多くのゲームが外部USB-HDDでの動作を拒否するため、可能な限り多くの内部ハードドライブを備えたセットトップボックスを購入した方がいいと気が散ります。



しかし、N900 PS3脱獄の幸せは長続きしませんでした。 すべての自作（自作アプリケーション）の優れた作業と一定数の試行錯誤のおかげで、私は1日1回安定してハングシステムを受け取りました。 また、再起動後、システムは中断されていない位置に戻ります。 （もちろん、ここで議論することができます：一方ではプラス、その場合はハッキングされていないシステムを即座に取得できますが、他方ではマイナスも明らかです）それで、私はしばしばN900の友人に電話しなければなりませんでした。



私はそれがそれほど長く続かないことを知っていました。 そのため、ビールを飲むことができないことが多く、男性の強い友情はそのようなテストに耐えられなかったため、他のシステムのPS Grooveポートは単純に行かなければなりませんでした。 私が覚えている限りでは、最初はPalm Preがあり、それからiPhone / iPodに関する半偽のニュースが現れ始めました。 そして、その非常に友人の助けがなければ、 行動のガイドに出くわしました。



ところで、ここにPSジェイルブレイクを作成できるデバイスのリストがあります。



iPhone 3G PS3脱獄



次は、マニュアルの無料翻訳で、上記へのリンクがあります。 ただし、個人的な経験のみを説明します。したがって、iPod、iPhone 4などについては何もチェックしなかったため、マニュアルの一部は省略します。



必要なのは：

  • iPhone 3G
  • すでにRedsn0wまたはblackra1nを使用してジェイルブレイクが行われているファームウェア3.1.2（上記では試していませんが、動作するはずです）
  • PS3、もちろん、ファームウェア3.41（3.42より前のバージョンのように、3.42、3.50はハッキングできません）。
  • ハッキングに必要なダウンロード済みファイル（ Windows 、 Mac OS X ）を備えたボロボロのAndroidのすぐに使えるインストーラー。 私はMac版のみを試しました。


当然、今ではすべてのPS3が3.41より前のファームウェアで販売されています。 それにもかかわらず、後のファームウェアでは、必要のないものが出てきました。そのため、インターネットを介したコンソールの定期的な更新はオプションではありません。 必要なのは、たとえばこちらからファームウェア3.41をダウンロードすることだけです。 FAT32ファイルシステムを搭載したUSBフラッシュドライブを使用して、ルートにPS3フォルダーを作成し、その中に-更新し、アーカイブからPS3UPDAT.PUPを入れます。 その後、USBフラッシュドライブをPlay Station 3に挿入し、メニュー項目[システムの更新]-> [記憶媒体からの更新]を選択する必要があります。 すべての人々。





残りは簡単です。 データケーブルを介してiPhoneをコンピューターに接続し、PS3をハッキングするためのファイルを含むアーカイブを解凍し、iPhoDroidを起動して、撮影ボタンを押します！.. 。 ブートローダーは永久に電話機にインストールされます。 その後、Androidアイテムを安全に選択し、Linuxのダウンロードを確認できます。



電話画面にINITという単語が表示されたら、到達する必要があります。PS3ジェイルブレイクの準備は完了です。

さらに、すべてのアクションはどのデバイスでも同じです。 ジェイルブレイクPS3を実行するには、次のようにします。

  1. コンソールからすべてのドライブとUSBデバイスを削除します。
  2. 電源を切り、電源コードを引き出します。
  3. iPhoneをデータケーブル経由でPS3に接続します。
  4. 電源コードを挿入します。
  5. コンソールの電源ボタンを押し、1秒後に離さずに、イジェクトボタンを押し続けます。 3つの「ピーク」を聞いた場合、すべてが正常であり、電源ボタンを離す必要があります。
  6. iPhoneの画面に刻まれた文字が停止するか、TV画面にホワイトノイズが消えるまで（PS3がオンになり始める）、イジェクトボタンを押し続けます
  7. ボタンを放した後、「ゲーム」メニューで「アスタリスク」記号で始まるアイテムの存在を確認します-/ app_home / PS3_GAME /およびパッケージファイルのインストール。
  8. すべてがうまくいけば、iPhoneの電源を切ることができます。 他のすべてが失敗した場合は、ボタンを押したままの間隔で踊ってみてください-これは役立つはずです。
  9. 電源ボタンとホームボタンを5秒間押し続けると、電話機を通常の位置に戻すことができます。




その後、ジェイルブレイク手順全体が5分以内に完了します。 もちろん、頻繁に行う必要はないことを書きたいと思いますが、悲しいかな、これは真実ではありません。 ジェイルブレイクがなくてもPS3はランダムな場所にハングアップするのが大好きで、ハードドライブでプレイするとこのチャンスが増えるようです。



自作



homebrewをPS3にインストールするには、フラッシュドライブのルートにある.pkgインストーラーにFAT32を入力し、[パッケージファイルのインストール]を選択します。



バックアップマネージャー


当然、ゲームのバックアップコピーを作成するには、いわゆるバックアップマネージャーが必要です。これは、これらの同じコピーを管理できるプログラムです。



マネージャーのいくつかのバージョンを試した後、 Open Manager Jurai Modを選択しました（登録が必要です）。 より快適に見え、クラッシュが著しく少なくなります。







使い方はとても簡単です。 ゲームのコピーを作成するには、「選択」をクリックします。 次に、作成するプレフィックスメディアで使用可能なすべての中から順番に選択するように求められます。 もちろん、内部ハードドライブにコピーを作成することをお勧めします。 これは、外部デバイスがFAT32ファイルシステムでのみサポートされており、ゲームのバックアップコピーに含まれていることが多い4 GBを超えるファイルをサポートしていないためです。 サポートされているゲームのリストもありますが、私が出会ったすべてのゲームが内蔵ハードドライブで動作し、アップデートが動作しないため、私はそれを必要としませんでした。



また、マネージャーはもちろん、ゲームのコピーを実行し、それらを他のメディアにコピーできます。 4GBを超えるファイルを部分に分割する機能が宣言されているようです。その後、ゲームを外部ネジにコピーできます。とにかく壊れたファイルで開始されないため、既に別の場所にあります。 しかし、彼女は私には役に立たないようでした。 なんで？



FTPサーバー


PS3にFTPサーバーをインストールしたため。 選択肢は2つしかありませんでしたが、バグが少なく、少なくともシステムの不具合のためにシステムの再起動に至らなかったもので停止する必要がありました。 ここにあります （登録が必要です）。



インストール後、サーバーを起動し、FTPクライアントでローカルネットワークのポートPS 22、匿名ログインでIP PS3を入力します。 クライアント設定では、同時接続の数を1つ以下に設定することをお勧めします。 上記のすべてがFilezillaで正常に機能します。



この接続を使用すると、PS3を選択せず​​に削ることができます。システムファイルを調べてみると、おそらくもっと面白いことがあります。 しかし、私たちは主にゲームに興味があります。 それらは/ dev_hdd0 / game / OMAN46756 / GAMEZにあります。 バックアップマネージャーを使用してコピーを作成した場合、ゲームの名前の代わりにBCAS20078のようなものがあります。 ただし、これらのフォルダーは簡単に名前を変更できます（主なことはキリル文字と制御文字を使用しないことです）。また、その中には画像ファイル（通常はPIC1.PNG）があり、どのようなゲームかを理解することができます。



FTPサーバーを使用すると、ゲームをコンピューターにコピーして、コンソールの内部ハードドライブの空き容量を増やすことができます。



実際、それは私が使用するすべての自作です。 また、Vuzeを使用するメディアサーバーとして追加することもできます。これにより、トランスコードせずにPS3上のほぼすべてのaviファイルを表示できます。 このソリューションは完全に私に合っています。
