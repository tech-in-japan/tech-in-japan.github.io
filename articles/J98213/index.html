Compiz 0.9.0リリース







X Window Systemの複合ウィンドウマネージャーであるCompiz 0.9.0が本日リリースされました。 ウィンドウの外観とその効果をカスタマイズできます。

前回の安定版からほぼ1年が経過しました。



最新バージョンの主な変更：

  • c ++でコードを完全に書き直しました（siでした）。
  • レンダリングはOpenGLとCompositeの2つのプラグインに分割されました
  • アニメーションを制御する再設計されたプラグイン
  • 新しい開発者API（すべてのプラグインが書き直されました）


主なイノベーションから：

  • デスクトップの壁紙を自動的に変更するプラグイン
  • Aero Snapと機能的に類似したプラグイン（これはKWinに既に存在します）
  • automake cmakeの置き換え
