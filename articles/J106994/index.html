Evernote 4 for Windowsがここにあります！



Evernote for Windowsは私たちのアプリケーションの中で最も人気があり、今日、Windowsクライアントの大規模な進化を発表できることを嬉しく思います。 Evernote 4をご覧ください！



Evernote 4をインストール»



以下では、なぜこのバージョンが史上最高と言えるのかを説明します。





最初からやり直す



Evernote 4は、すべての点でEvernote 3.5と根本的に異なります。 バージョン3.5には多くの素晴らしい新機能が追加されましたが、フォントがぼやけていたり、起動時間が長い、メモリ消費量が多い、グラフィックカードのサポートが不十分など、簡単に修正できない多くの問題に遭遇しました。 これはすべて、3.5（Windows .NETおよびWPF）の基礎となるテクノロジの仕様によるものであり、どのような方法でも影響を与えることはできませんでした。 最終的には、ユーザーが私たちに求めた新機能に取り組む代わりに、プラットフォームのバグとの戦いに滑り込んだ。



最終的に、C ++のみを使用してゼロから始めることにしました。 あなた自身が見るように、結果は単に驚くべきものです。 この新しいバージョンは、Windowsクライアントの開発を高速化するための基盤となります。



スピード



Evernote 4は、ネットブック、Windows XPを搭載した古いコンピューター、Windows 7を実行するトップモデルなど、あらゆるコンピューターで便利な作業を提供する必要があります。



ハードウェアを使用したテストで、Evernote 4は5倍速く実行され、Evernote 3.5の半分のメモリを使用することがわかりました。



外観



ユーザーインターフェイスのすべての要素を再設計しました。 新しいデザインでは、ボタンやスクロールバーではなくメモに集中できます。 新しいバージョンは、あらゆるアスペクト比とさまざまな入力デバイスを備えたさまざまなタイプのディスプレイで非常によく機能します。 したがって、マウス、キーボード、またはタッチスクリーンを使用してコンピューターとやり取りする場合、Evernote 4を使用すると便利です。







私たちのお気に入りのインターフェースの強化の一部を以下に示します。

  • スペースの節約：ツールバーとシステムメニューが統合され、画面上のスペースが少なくなりました。 ただし、設定ダイアログでいつでもクラシックメニューに切り替えることができます。
  • 応答速度：メモの切り替え、検索結果の表示、Evernoteウィンドウのオープンなど、ほとんどすべてのアクションがはるかに高速になりました。
  • メモ作成の簡素化：作成するメモの種類（テキスト、手書き、またはWebカメラ画像）を選択できるように、結合された「新しいメモ」を追加しました。 次回、プログラムはすでにあなたの選択を記憶しています。
  • 検索行の新しい位置：メモによる検索と現在の検索条件の説明は、より論理的なメモのリストのすぐ上に配置されました。 検索行の最後にある小さな矢印をクリックすると、現在の検索の説明と見つかった結果の数を表示できます。 この説明を非表示にする機能により、メモ自体を表示するためのスペースが解放されます。
  • メモに関するより明確な情報：メモ情報パネルがより明確でわかりやすくなり、メモの名前、メモ帳、およびソースへのリンクがパネルの上部に表示されます。 検索バーと同様に、メモに関する詳細情報を開くことができます。 情報パネルをナビゲートし、キーボードを使用してラベルを付けることが、はるかに高速で効率的になりました。
  • その他の対話オプション： Evernote 4はマルチタッチと多くのキーボードショートカットをサポートしています 。


それだけではありません...



何よりもまず、Evernote 4は、サービスと連携する新しい便利な方法を提供し、今後数か月で実装される膨大な数の新機能の強力なプラットフォームになるために作成されました。 しかし、最初からいくつかの興味深い機能を追加することにしました。



改善されたコピー

Webページのコピーされたスニペットは、Evernoteで最も人気のあるタイプのメモです。 そのため、Evernote 4では、コピープロセスを大幅に更新することにしました。 これで、Internet Explorer、Outlook、Firefoxからコンテンツを保存すると、現在のアプリケーションのウィンドウを離れることなく、タグを追加してノートブックを選択できるポップアップノート作成ウィンドウが表示されます。 Shiftキーを押しながらこのダイアログをスキップできます。



Microsoft Outlookユーザーは、メッセージ、添付ファイル、さらには数通の手紙をEvernoteアカウントにコピーする機会をお楽しみください。 Internet ExplorerおよびFirefoxからコピーする場合、より多くのスタイルおよびフォーマット要素が保存されるようになり、クリッパーはより確実に動作します。



強化された編集

過去9か月間、ノートの編集機能の改善に費やしてきましたが、次のような多くの新しい素晴らしい追加機能を誇っています。

  • テーブルの編集（列と行の追加を含む）;
  • 箇条書きおよび番号付きリストを使用したより便利な作業。
  • Internet Explorer、Office、およびその他のアプリケーションからのリッチテキストのコピーと貼り付けのサポートが改善されました。
  • また、ノートエディターのツールバーを使用して、より多くの色を選択できるようになり、見栄えがよくなりました。


スマートヘッダー

Evernoteは、保存されるコンテンツのタイプに基づいてノートヘッダーを自動的に生成する機能がよりインテリジェントになりました。 たとえば、これはコピーされたWebフラグメントのテキストの最初の行またはWebページのタイトルです。 もちろん、ノートのタイトルはいつでも編集できます。



柔軟な印刷設定







1つまたは複数のメモを印刷する場合、印刷ダイアログの[設定]ボタンをクリックして、印刷に含めるメモに関する情報を選択できます。



Windows 7：ジャンプリスト

Windows 7のルックアンドフィールのカスタマイズに加えて、Evernoteをタスクバーに固定し、Evernoteの基本機能（同期、検索、作成）にすばやくアクセスできる遷移リストなど、Windows 7の多くの新機能のサポートも含まれています。注）Evernoteが実行されていない場合でも。



Windows 7：自動ロケーション

Evernote 4は、Windows 7の新しいジオロケーション機能のほとんどもサポートしています。ノートにロケーションデータを簡単に割り当てることができるようになりました。 メモ情報パネルの「クリックして場所を設定してください...」というテキストをクリックすると、コンピューターが場所を見つけてメモに追加します。



不要なデータの隠蔽

画面スペースをより有効に利用するために、必要になるまでいくつかのインターフェース要素を非表示にしました。

  • メモに関する情報：メモのタイトルの近くにある山形をクリックすると、メモに関する情報が表示されます。
  • 検索語の説明：検索語を表示および編集するには、検索バーの横にある山形をクリックします。
  • ズーム：対応するビューのサムネイルのサイズを変更するには、マウスカーソルをノートリストの下部に保持します。 ズームコントロールスライダーがそこに表示されます。
  • 同期ステータス： Evernote 4が変更を同期するたびに、対応するステータスバーが画面の右下に一時的に表示されます。不要なスペースはあまり必要ありません。


Evernote 4をインストールする



Evernote 4は今すぐダウンロードできます！



Evernote 4をインストールすると、Evernote 3.5の既存のバージョンが削除されることに注意してください。



今後数週間で、Evernote 4ディストリビューションを手動でダウンロードしてインストールしたくない人のために、クライアントを自動的に更新する機能を追加します。



Firefoxを使用している場合は 、アドオンの更新を確認して、ブラウザに対応する最新の互換性のあるEvernote 4バージョンの拡張機能を入手してください。
