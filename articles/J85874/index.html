Intel Core i7 980X（ガルフタウン）の最初のテスト



OCClub.ruチームメンバーは、ロシアで初めて、新しい6コアIntel Core i7 980Xプロセッサのサンプルを自由に入手できました。 エンジニアリングバージョンは、ガルフタウンデスクトップソリューションの完全なアナログであるサーバーマシンXeon DP（Westmereコア）のソリューションです。 最初のテストは次のとおりです。







4 GHzの周波数で安定させるには、プロセッサに1.28 Vの電圧が必要でした。すべてのマザーボードがこれらのプロセッサをサポートしているわけではありません。この問題はBIOSを更新することで解決します。 BIOSバージョン1802でAsus Rampage II Extremeマザーボードを使用しました。マルチスレッドwPime 32Mテストの結果は4.234秒でした。 同様の結果を得るには、クアッドコアIntel Core i7プロセッサー920、950、960、975で5500 MHzを超えるクロック周波数が必要でした。







3DMark05では、単一のRadeon HD5870グラフィックスカードでの結果は31857 3DMarksでした。 Uncoreの頻度が低いため、結果はBloomfieldプロセッサよりもわずかに弱いです。 ブルームフィールドプロセッサのUncore周波数はRAM周波数の2倍に設定されており、Gulftownの場合、周波数は1.5倍高く設定されていることを思い出してください。 メモリコントローラーの周波数が低いため、特定のアプリケーションの新しいプロセッサはブルームフィールドプロセッサよりも劣る場合があります。







3DMark Vantageでは、結果は23192 3DMarksです。これは、このような周波数に対して非常に良好で高い結果です。 まあ、Vantageはすべてのコアで正常に動作するため、CPUテストの結果はおかしくなりました。



Intel Core i7 930が利用可能になりました



本日、新しいIntelプロセッサCore i7 930の販売が開始されました。2010年2月28日は、Intel Core i7-930プロセッサの公式発表の日です。 目新しさはまだロシアに届いていませんが、目新しさを楽しみにしていたヨーロッパ人は、今日の夢を実現できます。 実際、加工業者はすでにヨーロッパの店舗の棚に登場しています。



Core i7-930は、45 nmプロセステクノロジーの標準に従って作成され、最大3.06 GHz（133 MHz x 23）のターボブーストで、2.8 GHz（133 MHz x 21）の周波数で動作する4つの物理コアを備えています。 また、前身のCore i7-920と同様に、新参者は第3レベルに8 MBのキャッシュを持ち、ハイパースレッディングテクノロジーをサポートしています。 TDP Core i7-930は130ワットです。



このCPUのコストは254.86ユーロで、これは以前のCore i7-920よりも約25ユーロ高価です。



AsusがRadeon HD5970のコードネームAresを発表



過去数か月で、ビデオカードの非参照バージョンのリリースが非常に人気になりました。 ビデオカードメーカーは、バンドルのみが異なる製品はユーザーの関心を引くことができないことに気付きました。 同社のエンジニアはリファレンスデザインの再設計に着手し、MSI LightingのGigabyte Super OverClockシリーズを見ました。 Asusは、Republic of Gamersシリーズのカード、Asus Radeon HD5970 Aresを発表しました。







プロトタイプビデオカードの画像は、非常に魅力的で生産的に見えます。 開発者によると、このカードは標準よりも25％高速であることが判明しました。 結局、GPUとメモリの周波数はそれぞれ725 MHzと1000 MHzから850 MHzと1200 MHzに増加しました。



CeBIT 2010での3Dスクリーンを備えたMSIからのモノブロック



CeBIT 2010は、ITテクノロジーと少しでも関係があるすべての人を楽しみにしています。 結局のところ、最も興味深いニュースが近い将来に私たちを待っているのはハノーバーからです。 そのため、MSIは、「キャンディバー」フォームファクターに、さらには3Dスクリーンを搭載したPCを導入する予定であることがわかりました。







Monoblocks MSIは、ホームメディアセンターとしての使用を提案しています。 3Dスクリーンを備えたキャンディーバーは、この種の最初のものです。 120 Hzの周波数のパネルとシャッターグラスによって、音量効果を確保する必要があります。



真の愛好家のためのEVGA W555ボード



EVGAは、熱心な市場に明確に照準を合わせています。 人気の分類されたラインナップに従って、開発者は2つのLGA1366ソケットを備えたマザーボードを導入しました。 このようなボードは、かなり強力なサーバーの基礎になります。 結局のところ、新しい6コアプロセッサであるGulftownとWestmereとの連携に重点が置かれています。







RAM用の12スロット、7つのPCI-Expressスロット、デジタル電源回路-このラインアップの高品質と独自性の組み合わせは、一般の人々にとって大きな関心事です。







新製品の価格は明らかにされていないが、予備情報によれば、1000ドルに達する可能性がある。
