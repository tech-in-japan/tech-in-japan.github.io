GDG DevFest Gorky 2017







こんにちは、Habrとhabrozhitel！ ニジニ・ノヴゴロドに行ったことはありますか？ 古代ニジニ・ノヴゴロドのクレムリンの壁からの夕日を背景に、岡とヴォルガの合併を観察しましたか？ フェドロフスキー堤防のパノラマを楽しみましたか？ ヴォルガ川を越えてヨーロッパで最も長いケーブルカーの1つに乗っていますか？ 今がその時です！













2017年11月18日、ニジニノヴゴロドで、Googleテクノロジーに特化した第3回国際会議GDG DevFest Gorky 2017が開催されます。







GDGとは何ですか？



これは、Android、Maps、App Engine、Chrome、Web Toolkit、Google Plusなどの開発者向けのGoogleテクノロジーに基づいた非営利のITコミュニティです。







GDG DevFestとは??



これは、開発者向けの一連のIT会議です。 世界中で開催され、Google Developer Groupsコミュニティによってホストされています。







GDG DevFest Gorkyとは??



これは開発者向けの国際会議で、ロシアで最も美しい都市の1つであるニジニノヴゴロドのGDGニジニノヴゴロドコミュニティによって毎年開催されます。







この壮大なイベントは、 ニジニノヴゴロド大学の壁の中で開催されます。 N.I. ロバチェフスキーとその壁に膨大な数の開発者や他のIT専門家を集めましょう！







今年の会議は真に国際的なものとなり、 ロシア、イギリス、インド、ポーランド、トルコ、セルビア、ウクライナなど世界中のGoogle Developers Groupコミュニティの講演者によるプレゼンテーションが行われます。







  • Alexey Korovyansky（モビラトリアム、オムスク、ロシア）-Google Developer Expert。モバイル開発者の間でVR / ARの機能を積極的に推進しています。 オムスクのGDGコミュニティの創設者。 現在、AlexeyはMobilatorium（創造的なVR / AR研究室）を組織しており、効果的なチームとともに革新的なVR / ARプロジェクトを開発しています。
  • Alexey Rybakov（DataArt、ハリコフ、ウクライナ） -シニアソフトウェアエンジニア、テクニカルエバンジェリストDataArt、GDG Kharkiv-Centerの主催者、GDGカンファレンスDevFest、Droidcon、Mobius、IT NonStop、IT Talkのスピーカー。 彼は、12年以上（Palm OS、Windows CEからiOSまで）さまざまなモバイルシステムのゲームをプログラミングしてきました。
  • ミロスマリンコビッチ（カウンターパス、ノヴィサド、セルビア） -開発者。 主催者GDG Novi Sad。 関心分野：Android、最新のWeb、IoT。 イノベーションとプログラミングについて新しいことを学びたい。 空き時間には、さまざまなオープンソースプロジェクトに取り組んでいます。
  • Resul Kaner Yildirim（トルコ、イスタンブールのInnovationBox）-Google Developer Expert、UXデザイナー、テクノロジーオタク。 UX、IoT、Androidのプロフェッショナル。 2013年からGDGイスタンブールを率いて、たとえばDroidcon、GDG DevFestなどのさまざまなイベントを開催、参加、出演しています。


スピーカーは、Android、Web、Firebase、Google Cloud、VR / AR、IoT、Kotlin（Android Auto、Instant Apps、Android Things、ARCore、Google Assistantなど）、およびニューラルネットワークと機械学習に関するプレゼンテーションを行います。

そして今年は、Googleテクノロジーを使用した開発分野の技術専門家に加えて、プロジェクトの推進と開発の分野の専門家が会議で講演するほか、ニジニノヴゴロドのDevFestで初めてiOSの専門家がすべての経験を共有し、ベストプラクティス。 例：







  • Adam Wesolowski（GrowthEngine.xyz、ヴロツワフ、ポーランド）は経験豊富なマーケティング担当者であり、スタートアップの成長とオンラインマーケティングイベントの拡大を支援しています。 GrowthEngine.xyzマーケティング代理店の創設者兼CEO、Google Developer Expert、およびサンフランシスコとワルシャワのGoogle Launchpadメンター。 彼は、Google、PizzaPortal、SecurionPay、Nasza-klasaなどの企業で働いていました。 彼はスプリントとA / Bテストで働くのが好きです。
  • Marat Ishmeev（モスクワ、ロシア、ティンコフ） -応用数学およびコンピューターサイエンスのマスター。 ロシアおよび外国の雑誌に微分方程式に関する多くの論文を発表しました。 2011年以降、iOS向けのモバイル開発に従事しています。


講演者とイベントのプログラムの 完全なリストは 、当社のウェブサイトで見つけることができます。







私たちのフェスへの訪問は、貴重な経験を獲得し、新しい技術について学び、現在習得している技術のスキルを引き出し、同僚と新しい製品について話し合い、大きなITコミュニティの一員になる機会です。 非公式の雰囲気の中で、さまざまな分野の最新の技術動向について、その分野の専門家と話すことができます。その専門家は、講演者だけでなく、会議参加者の間でも集中します。
