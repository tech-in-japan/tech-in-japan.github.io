Synology®がDiskStation Manager 4.1ベータ版を発表





Synologyは、新しいDiskStation Manager（DSM）4.1オペレーティングシステムのベータリリースを発表しました。



新しいOSはユーザーに以下を提供します。

  • 仮想インフラストラクチャ向けの高度で信頼性の高いストレージソリューション
  • Cloud Station、Syslog Server、Mail Serverなどのサービスを展開して、ビジネスおよび管理のパフォーマンスを向上させる
  • 効果的なリソース割り当てシステムの作成と、企業標準を満たすデータを共有するための高性能ソリューションの使用
  • 新しいVideo Stationサービス、強化されたPhoto Station、Audio Station、モバイルアプリで強化されたマルチメディアエクスペリエンス
  • 更新されたSurveillance Station 6ソフトウェアパッケージでビデオ監視システムのセキュリティを向上させる


DSM 4.1は、Array Integration（VAAI）テクノロジー向けのVMware vStorage APIをサポートする最初のオペレーティングシステムの1つであり、DiskStationネットワークドライブでフルコピー、ゼロブロック、アトミックテスト＆セット（ATS）およびストレージ再生機能を実行してハイパーバイザーをアンロードできますとRackStation。 VAAIテクノロジーを使用すると、vSphere 5.0のパフォーマンスが向上し、ディスク容量が最適化されます。







DSM 4.1は、生産性と高度な機能の大幅な向上をユーザーに提供します。 Cloud Stationサービスは、2つの共有フォルダーのサポート、およびモバイルデバイスからファイルを受信および転送する機能を追加しました。 Cloud Stationは安全なHTTPSおよびプロキシサーバーをサポートするようになりました。 Syslog Serverサービスは、クイックブラウジング、キーワード制御、データアーカイブポリシーの設定などの機能の登場によりさらに便利になりました。 メールサーバーアプリケーションは、SMTPプロトコル、追加の電子メールボックス、複数のドメイン、スパムフィルター、留守番電話、およびメッセージ数と電子メールトラフィックをリアルタイムで監視する機能をサポートする完全に機能するメールサービスになりました。







DSM 4.1は、強力なデータ共有ソリューションを提供します。 トラフィック制御および速度制限機能により、さまざまなアプリケーション、ユーザー、およびグループのリソースの優先順位付け、予約、および使用の制限が可能になります。 リソースモニターを使用すると、ヒット数とデータ転送率に関するライブレポートと統計を受信できます。 SFTPおよびFXPプロトコルのサポート、およびプロキシサーバーを介したインターネット接続の確立機能により、企業標準に従ってより効率的で安全なデータ転送が提供されます。 さらに、複数のDiskStationおよびRackStationデバイスに保存されている共有フォルダーの同期を提供し、それらを単一のデータ配列に結合します。







DSM 4.1には、高度なマルチメディア機能が含まれています。 新しいVideo Stationサービスのページには、インターネットから自動的にダウンロードされたメディアコンテンツに関する映画のポスターと詳細情報が表示されます。 TV Recorderモジュールを使用すると、テレビ番組をオンラインで視聴したり、スケジュールに従って録画したりできます。 DSビデオモバイルアプリケーションを使用すると、iOSデバイスでビデオを再生したり、プログラムの録画スケジュールを変更したりできます。 Photo Stationを使用すると、カスタムアルバムを時系列で配置したり、Googleマップにリンクしたりできます。 場所と名前によるフィルター、および自動顔認識を使用すると、写真を並べ替えるプロセスが非常に容易になります。 更新されたAudio Stationアプリケーションを使用して、歌詞を表示し、各ユーザーの個人ライブラリを作成できるようになりました。 インターネットに接続していなくても、所有するiOSデバイスがキャッシュに保存された音楽を再生できるようになりました。 モバイルアプリケーションDS audioおよびDS photo +は、Windows Phoneベースのデバイスをさらにサポートしています。 同様に、更新されたDSダウンロードアプリケーションにより、ユーザーはダウンロードタスクを管理できます。







Surveillance Station 6の高度なビデオ監視アプリケーションインターフェイスには、さまざまなビデオ再生チャンネルに対応する4つのスライダーが追加され、ビデオ録画の特定の期間に簡単に戻ることができます。 タリーカウンター、仮想フェンス、非アイドルゾーンなど、オブジェクトを追跡するための新機能により、監視ゾーンに侵入する際のビデオ監視システムの信頼性が大幅に向上します。 Web APIを使用することにより、開発者はSurveillance Stationを独自のアプリケーションと統合することが容易になりました。
