エイプリルフールの記事を書く方法

多くの人々はその場しのぎの原子炉に関する最近の記事を覚えています。 この記事はエイプリルフールのジョークであり、非常に成功したことを認めなければなりません。



この短い記事では、これが冗談であることと、エイプリルフールの成功記事の重要な要素と思われることをすぐに理解できる方法を説明したいと思います。これは、1年で記事を書くのに役立ちます（または読んでいるときに警告します）。 同様の詐欺が通常のメディアで頻繁に見られることも注目に値します。



それでは、始めましょう（最初の記事を見る人にも反論が見えるように、DIYで公開しています）。



これは実際にはそうではないことを理解する方法は？


reddit（翻訳リンクがそこに公開された）で、1人のコンパニオンが正しい計算を行いました：7ミリワットの電力で、10年間で0.007J / 24h / 3600s / 365d / 10y = 2212000ジュールのエネルギー、または1e25 eV（7e20ベータ崩壊）トリチウム（それぞれ18keV）。 1モルの物質は6e23原子です。ここでは0.001モルになります。つまり、3ミリグラムは100％の効率で10年で減衰します。



バッテリーの効率が10％、全世界の33％がバッテリーに入り、蛍光体の効率が50％の場合、200 mgのトリチウムがすでに必要です。 そして、これは2200立方センチのガスです。



実際、マイクログラムがある場合、トリチウムは良好であり、放出されるエネルギーは非常に小さいため、自分の機器で測定することさえできません。 何かがあることがわかるだけなので、これらは少なくともナノワットです（まあ、少なくとも私はナノテクノロジーの発展に貢献しました ）



過度に複雑なスキームで保護されているものもあります。レンズは役に立たず、ただのホイルでできたピラミッドです。 実際、「真剣に」デバイスを組み立てたとき、まさにピラミッドがありました。



写真のジャグリングについて


FoxNewsのレポーターにこんにちは。 キーチェーンは非常に薄暗いため、日中は見えません。 夜には、緑色のダッシュが見えるだけです。 記事の写真のように、輝きだけを見えるようにする方法は？



暗い部屋、ISO 1600、30秒のシャッター速度。 最も困難なことは、最初の写真でキーチェーンを腕に写すことでした-20回の試みがなくなったため、腕を動かさないことは困難です

もちろん、LEDには小さなバッテリーが搭載されており、電子回路で回路基板の裏側に目立たないように接着されています（ただし、LEDはすべて点灯しているため、何も見えません）。 イオニスタははんだ付けさえされていませんでした。



主な成功要因


  1. 過去の成功した記事をすぐに参照してください
  2. 私たちは理論への短い遠足から始めます、すべては可能な限り深刻です
  3. 記事の本文には、可能な限り多くの関連リンクが含まれています
  4. 真実に似た数字を示しますが、検証は困難です
  5. 写真を優しくリグ-フォトショップ、博覧会...
  6. 必要なコンポーネントを購入できるリンクを提供します
  7. 私たちは挑発に屈しません;-)


記事をお楽しみください。 すでに装身具を注文している場合-気を悪くしないでください:-)



PS。 そして最後に、教えることは軽いです！
