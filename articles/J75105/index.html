プラットフォーム-2010。初日

こんばんは



モスクワで開催される技術会議「プラットフォーム-2010」の初日は終了しました。 この日はとても忙しくて面白かったので、今しか呼吸ができません。



それでは、今日のMicrosoft ITコミュニティに満足したことは何ですか？



まず第一に、私は組織の高レベルに注目したい-彼らはそれを熟考し、十分な場所があり、インターネットが機能し、ディスクが配布され、彼らが飲んだお茶を飲んだ。 そして、これは約1500人の参加者が来たという事実にもかかわらずです。



さまざまなスタンド、多数の資料、認定試験を無料で受ける能力、パートナーからのコンテストと情報、サッカー競技、xbox-これは、一日中自分でできることの完全なリストではありません。



私が最初に行ったのは、明日の朝に登場するマイクロソフトの非公式ポッドキャストを録画することでした。 ポッドキャストの記録プロセスを見て、そしてもちろん、伝説的な逆CLRの本を自分で見るのは非常に面白かったです:-)ポッドキャストは会議とその参加者に完全に捧げられています。



次に、レポートについて。 それらの多くがあり、誰もが自分の興味のあるトピックを選択することができました。



Exchange 2010のテクノロジーと利点の概要について説明しました。SharePoint2010に精通しました。これらの製品に精通していませんが、聞いたことは印象的です。



何よりも、Visual Studio 2010とMicrosoft StreamInsightのレポートが気に入りました。 最初のケースでは、TDDの使用、C＃でのスタジオ用の独自のプラグインおよび拡張機能の作成の可能性、およびC＃およびVB.NET言語の開発における相関関係について詳しく学びました（スピーカーはVBのIDEの開発チームで働いています。明確化））。



StreamInsightについて説明する場合、これはリレーショナルデータストリームからリアルタイムで知識を抽出できるテクノロジーです。 それはまだ非常に湿っていますが、実際には、原則として何も驚くべきことではありません。 ツールボックスから必要なのは、スタジオ、LINQ、およびStreamInsightだけで、約10 MBかかります。 SQL Serverに同梱されますが、実際には何の関係もありません。 レポートの記録をご覧になることをお勧めします。記録はまもなく会議のウェブサイトで利用可能になります。



その日は大量のビールを飲むことで終わり、科学的には「ビュッフェテーブル」と呼ばれました。 ところで、ゴブリン自身は映画の前にいくつかの言葉を言った:-)。



今のところ、雑誌とディスクを整理し、明日の続編の準備をします。



PS入り口でラップトップに何らかのラベルを貼り付けましたが、出口には表示されませんでした。 どこかに落ちた。 しかし、セキュリティは気にしなかったので、デスクトップの壁紙の色、管理者パスワード、ショートカットの存在に関する私の知識をテストすることにしました。 パスワードがなく、ラベルが標準であり、写真のみを記述すればよいのは良いことです。一般的には楽しいです。 明日はラップトップを持っていません、突然何かを忘れます...



更新しました。 オンラインでレポートを見ることができます。 これを行うには、会議Webサイトの適切なセクションに移動し、ブラウザでSilverlightプレーヤーを介してレポートを見ることができます。 または、mms：//80.247.33.46/t1でメディアプレーヤーを使用します。ここで、t1はホール番号です（合計で8つあります）。
