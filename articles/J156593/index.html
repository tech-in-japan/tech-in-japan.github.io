Cの整数型のサイズ



Cは移植性があるため、基本的な整数型（ char

、 int

それらのサイズは完全に任意でした。標準は、すべての基本的な整数型に対して受け入れられる値の最小範囲を確立します。 すなわち



  • signed char:



    -127 ... 127（-128 ... 127ではなく、同様に他のタイプ）
  • unsigned char



    ：0 ... 255（= 2 8 -1）
  • signed short

  • unsigned short



    ：0 ... 65535（= 2 16 -1）
  • signed int



    ：-32767 ... 32767
  • unsigned int



    ：0 ... 65535（= 2 16 -1）
  • signed long



    ：-2147483647 ... 2147483647
  • unsigned long



    ：0 ... 4294967295（= 2 32 −1）
  • signed long long



    ：-9223372036854775807 ... 9223372036854775807
  • unsigned long long
