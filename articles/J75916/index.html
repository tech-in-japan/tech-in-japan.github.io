家計簿、またはあなたのお金がどこに行くか知っている

多くの人々が次のような問題を抱えています。お金があるように見えますが、ここではホップがあります-そして何もありません（もちろん誇張）。 すべてが酔っ払っているか、ビジネスに費やされているか、しゃっくりしている。

彼自身も同様の問題に直面した。 3万ルーブルの給料で。 （そして、私たちの都市にとってはまったく悪いお金ではありません）7000ルーブルとの違い。 まったく気づかなかった。 彼は車が欲しかったようで、HTCファッショナブルが欲しかったようですが、ここではすべてが同じ食べ物と食べ物、ビールとビールです。



まあ、叙情的な部分から離れて。 一度、 ビールで、愚か者は簿記の方法によって彼の財政のすべてを制御しようとすることに決めました。 当時、私が知っていた唯一のソフトウェアは（伝聞ではなく）1C 8アカウンティングでした。 見つけて、セットして、開けて、怖がって、閉じて。 私は簿記とは何の関係もない男なので、1Cに煩わされることなく、他の人向けの製品の方向に目を向けることが決定されました。 そして、あなたはどう思いますか？ 存在！



以下では、自宅での簿記に焦点を当てた製品の簡単な概要を整理します。



マネートラッカー



MoneyTrackerは、ホームアカウンティングを実行するためのシンプルで便利なプログラムです。 最初に注目されたのは、 素晴らしいインターフェイスです。



すべてのメインモジュールは、トップメニューの大きなボタンでレンダリングされます。 主なモジュールには次のものが含まれます：記事（命名規則など）、アカウント、収入/費用の統合リスト、収益、費用、振替、カレンダー（スケジューラーのようなもの）、予算、およびレポート（表形式またはグラフィック表示）。 一般的に、トップメニューの大きなボタンにはすべての基本が含まれていると言えます。 プログラムの残りの部分は、通貨のさまざまなディレクトリ、お金の可能な場所、製品、家族、さまざまなダイナミクスなどです。

何が好きでしたか：

  1. インターフェイスは素晴らしく、使用可能です。
  2. 在庫ガイドには、頻繁に見られる商品やサービスがすでにいっぱいになっています。
  3. 私は最大1時間でソフトウェアを理解しました。すべてのボタンが手元にあり、ヘルプはただのああではなく、有益なものです。 レポートも私の目的に完全に適しています。
  4. カレンダーには、1回限りの計画と日次/月次計画の両方の可能性があります。


気に入らなかったもの：おそらくカレンダーだけ。 もっと柔軟な計画が見たいです。 たとえば、お金があればすぐに何かを購入したい+購入後、2000ルーブルが私のポケットに残ります。

ライセンス費用： 400摩擦。



Keepsoft



KeepSoftのインターフェースはMoneyTrackerほど美しいものではなく、その機能が...



上記のすべてがあります（ただし、命名参照は空であり、計画はカレンダーの形式ではなく、テーブルにあります）。 しかし、次に掘り下げる-PDAとの同期！ PDAが不足しているため、PDAのバージョンは表示されませんでしたが、チェックが失われると問題は急激になくなります。 私は自分でMoneyTrackerを使用していますが、店で過ごしたものを自宅で録音することを「忘れる」という絶え間ない問​​題があります。 ここではすべてが簡単です-私はそれを使い、PDAに書き、帰宅し、同期しました。

その他の利点：組み込みのバックアップ、インポート/エクスポート/データ同期、単一の共通検索。

プライベートライセンス： 400ルーブル。

ファミリーライセンス： 650ルーブル。
