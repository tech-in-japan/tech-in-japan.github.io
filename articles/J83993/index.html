Windows 7アップデートは、インストールされたコピーの合法性を検証します

1〜2週間以内に、Windows Updateに新しい「重要な」更新が表示されます。 （無料翻訳）「Windows 7のWindows Activation Technologies Update」（「Windows 7のWindows Activation Technologies Update」）と呼ばれます。Windows7の違法コピーがある場合は注意してください。

このオプションの更新プログラムは、違法コピーの兆候と見なされる70の「既知の潜在的に危険なアクティベーションエクスプロイト」を検出します。

ただし、熱くなっている場合は、パニックに陥らないでください。 マイクロソフトは、個人情報が送信されないことを約束します。 代わりに：

アクティベーションエクスプロイトが見つかると、Windowsはユーザーに通知し、この問題を解決する方法を提案します。ほとんどの場合、これには数回のクリックしか必要ありません。 本物の「Windows 7」がインストールされているマシンは、気にしたり怖がったりすることはありません。メッセージなしでバックグラウンドで更新されます。 Windows 7が正規品でない場合、ユーザーには2つの主なオプションを備えた通知ウィンドウが表示されます：「詳細」と「正規品の購入」。 デスクトップの壁紙は色で塗りつぶされます（すべてのユーザーアイコン、ガジェット、固定されたアプリケーションはそのまま残ります）。 定期的なリマインダーとデスクトップ上の一定のウォーターマーク-これはユーザーをプッシュするものです。

ユーザーがWindowsのコピーの機能の低下に気付かないことを知っておくことが重要です-ユーザーアプリケーションは期待どおりに動作し、個人情報へのアクセスは変更されません。




もちろん、Windows 7のコピーが完全に正規品であっても、この更新プログラムをダウンロードしない正当な理由があります。 Windows Checkは、海賊版として合法コピーを受け入れる傾向があることで知られています。 これらはまれなケースですが、このhemoを危険にさらしたい人はほとんどいません。

翻訳者から：無料のウイルス対策/マルウェア対策のMicrosoft Security Essentialsなど、正規のWindowsを必要とするプログラムはインストールされなくなります（既にインストールされている可能性があります）。



また、ソースリンクには、約90日ごとにさらに検証が行われることが記載されています。 もちろん、エクスプロイトデータベースは補充されます。 何度も約束され強調されているように、アップデートはダウンロードできず、インストールされているものでも削除できます。
