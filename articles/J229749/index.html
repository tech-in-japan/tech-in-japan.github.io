Raspberry開発者はRaspberry Pi Model B +をリリースしました

最初のRaspberry PiミニPCが出荷されてから2年以上が経過しました。 開発者がこのデバイスに新しい追加を作成し、新しいソフトウェアを作成しているという事実にもかかわらず、モデルは変更されず、すべてが2年前と同じままです。







しかし、今ではすべてが変わっています。 Raspberry Piの開発者は、Raspberry Pi Model B +と呼ばれる新しいデバイスを作成しました。 公式発表はすでに同社のブログに投稿されており、一部のサイトではデバイスを注文できます。 カットの下-アップデートの詳細。







新しいデバイスの図を次に示します。







新機能（以前のモデルのように、26ピンではなく40ピン）：







新しいデバイスの機能は次のとおりです。



  • GPIO 前述のように、前のモデルのように、26個ではなく40個のピンがあります。
  • USBポート モデルBの2つのポートではなく、4つのUSB 2.0ポート。
  • MicroSD。 以前はSDカードスロットがありましたが、このスロットはmicroSDバージョンに置き換えられました。
  • 消費電力の削減。 開発者は、消費電力を0.5〜1Wに削減することができました。
  • 音。 ノイズリダクション、音質改善。
  • 改善されたフォームファクター。 開発者は、ボードをより合理的で作業に便利なものにしました。








プロセッサー（BCM2835）、RAMの量（512 MB）、およびソフトウェアは同じままでした。



ロシアでは、 Raspberry Pi Model B +も注文可能です。
