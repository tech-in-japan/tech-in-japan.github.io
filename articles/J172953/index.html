Mozillaは、オンライン学習用のオープンな証明書標準であるOpen Badges 1.0を正式にリリースしました





オープンバッジの背景にある考え方は、大規模なものと同じくらい簡単です。コースの最後に参加者に与えられるバッジのオープンスタンダードを使用して、インターネットで利用可能なカリキュラム、コース、レッスンを組み合わせます。 つまり、オープンバッジは、ソーシャルネットワークや専門家のコミュニティ、個人のWebサイト、履歴書に配置できる標準のミニ卒業証書です。



すべての組織はバッジを発行できますが、各バッジの重要性と「急峻さ」は、バッジを発行した人の権限によってのみ決定されます。 小さな会社でも巨大な大学でも構いません。 現在、参加者のリストにはすでにスミソニアン美術館、ニューヨーク教育省、イリノイ大学を含む600の組織があります。 Microsoft、NASA、Pixarなどは、すでにOpen Badgesインフラストラクチャに参加する計画を発表しています。



今日、高等教育には多額の費用がかかります。また、多くの場合、最高の大学の教師が教えるオンラインコースで、大学の学生が受けるものに匹敵する知識を学生に提供できれば、大学は卒業証書や証明書を急ぐことはできません。 多くの人々は、伝統的な閉鎖的で保守的な高等教育システムには脱獄が必要だと考えています。 Open Bagesは、手頃な価格の独立した教育と知識検証のこの世界的なシステムに必要な基盤となります。









技術的には、各アイコンは、受信者のメールアドレスのハッシュ、組織を発行したアイコンの名前に関する情報、その画像へのリンク、およびアイコンを発行する基準の詳細な説明を含むJSONデータ構造です。不正に受け取った。 そのような構造の例を次に示します。 



{ "name": "Awesome Robotics Badge", "description": "For doing awesome things with robots that people think is pretty great.", "image": "https://example.org/robotics-badge.png", "criteria": "https://example.org/robotics-badge.html", "tags": ["robots", "awesome"], "issuer": "https://example.org/organization.json", "alignment": [ { "name": "CCSS.ELA-Literacy.RST.11-12.3", "url": "http://www.corestandards.org/ELA-Literacy/RST/11-12/3", "description": "Follow precisely a complex multistep procedure when carrying out experiments, taking measurements, or performing technical tasks; analyze the specific results based on explanations in the text." }, { "name": "CCSS.ELA-Literacy.RST.11-12.9", "url": "http://www.corestandards.org/ELA-Literacy/RST/11-12/9", "description": " Synthesize information from a range of sources (eg, texts, experiments, simulations) into a coherent understanding of a process, phenomenon, or concept, resolving conflicting information when possible." } ] }




アイコンを認証するには、一意のURLとデジタル署名の2つのメカニズムがありますが、これまでのところ最初の1つだけが機能します。 仕様、バッジの発行と確認のプロセスの詳細な説明、および必要なすべてのツールとソースがGithubで公開されています。
