週末の前にウォームアップ：Raspberry Pi、Arduino、無料の夜があります







週は週末にスムーズに近づくので、長い休暇のために何かを考え出す時が来ました。 たとえば、 Raspberry Piからサーマルイメージャーを組み立てて、「エンジンがどれだけ暖かくなるか」と「プログラマーの頭の温度はいくらになるか」という永遠の質問に答えることができます。 または、 Arduinoに基づいた独自のバージョンのGoogle Glassですが、非常に特殊なバージョンです。







ArduinoとRaspberryについて耳の隅から聞いたばかりの場合、違いのある小さなネタバレがあります。

ArduinoとRaspberryは、「ホーム」プロジェクトと小規模な自動化のために設計された1つのプリント基板上のミニコンピューターです。 ボードにはいくつかの基本的な違いがあります。







  • クロック周波数によると、Raspberry PiはArduinoよりもはるかに高速で、Piにはより多くのRAMがあります。







  • オペレーティングシステムがRaspberry Piで実行されている可能性があります。







  • Arduinoとは異なり、周辺機器をRaspberry Pi USBポートに接続できます。たとえば、ネットワークアダプターやモデムなどです。







  • Arduinoは、さまざまなセンサー信号と手動入力に応答するだけのハードウェアプロジェクトに最適です。


Arduinoはセンサー管理を任せられ、Raspberry Piはより複雑なコンピューティングを行うことができます。







ArduinoやRaspberryはAliExpressのようなサイトで購入できますので、興味深い部分に移りましょう。 以下に、時間がかかるだけでなく、真のメリットをもたらすTOP 5プロジェクトを示します。 または、認知の観点から少なくとも興味深い。







マニピュレーターを使用した磁場測定



3つの自由度を持つ「アーム」上の可動磁力計は、蓋を開けた状態でシステムユニットの磁気放射の大きさについての好奇心を最終的に解消します。 または、負荷がかかっているハードドライブの磁気的背景に関する興味深い情報を取得するのに役立ちます。 ロボットは、選択したポイントで値を正確に測定し、磁場のマップを作成します。













マニピュレーターを機能させるには、 ATmega32u4マイクロコントローラーにArduino Pro Microが必要です。 情報を取得する際、センサーは大量のデータを生成するため、迅速に処理する必要があるため、ラップトップとLUFA USBライブラリがさらに必要になります。 センサーの結果は、MATLABで作成するか、無料の代替ツールであるOctaveを使用できます。







実際の「手」には、サーボといくつかの接続部品が必要です。 マニピュレータを「復活」させると、この興味深いプロジェクトの作者によって親切にレイアウトされたコードが許可されます。 このような自動化のおかげで、著者は以前に測定された一般的な磁気バックグラウンドを考慮に入れて、研究対象の磁場の3次元マップを作成することができました。









カードの解像度は51 x 51 = 2601のデータポイントに達するため、測定されたフィールドの「手」にあるスチールボルトの影響も追​​跡できます。







赤外線カメラFLIR LeptonおよびRaspberry Pi









サーマルイメージャーは、オフィスや猫の熱漏れを見つけるのに役立ちます。 または、セキュリティシステムの要素として適合させることができます。 極端な場合、ディスクシェルフの熱特性を測定するのに刺激的な30分を費やすことができます。







サーマルイメージャーは、解像度80×60ピクセルのマイクロボロメトリックマトリックスを備えたFLIR Leptonサーマルカメラに基づいています。













カメラはRaspberry Piコントローラーに接続します。 ボードのSPIおよびI2Cインターフェイスをアクティブにし、ビデオストリームを表示するためのソフトウェアをインストールする必要があります。 1組の18650バッテリー、UBECなどの電圧安定装置を追加し、すべてのアクセサリをケースに入れます。













今日、サーマルイメージャー自体は手頃な価格ですが、店舗オプションのコストは自家製のものよりも数倍高く、自分の目的に合わせて調整できるとは限りません。 そして、例えば、 プロジェクトの著者として、ハイブリッドロボットビジョンシステムなど、目標は最も広くなる可能性があります。







電圧計用のArduino拡張現実メガネ



拡張現実はポケモンを捕まえるだけでなく、今日では軍隊、そして今ではエンジニアによって積極的に使用されています。 プロジェクトの著者は、シールド内のオープン接点の電圧を測定することを決定し、電圧計を見るために首を回すのに疲れたようです。 その結果、彼はプロセスを簡素化し、短絡の可能性を減らす方法のアイデアを思いつきました。







しかし、アイデアは単純です。電圧計からの情報は、Bluetoothモジュールを介してデバイスに送られるため、 単純なメガネが「スマート」になります。













このデバイスを実装するには、 Arduino Proマイクロマイクロコントローラー、BluetoothモジュールV4.0 HM-11 BLEモジュール、バッテリー-280mA Lipo、ディスプレイ-Micro OLED 0.66 64x48ピクセル、レンズ、およびこれらすべてを配置できるプラスチックボックスが必要です。 このプロジェクトの著者は、このような「スマート」グラスの合計コスト（マルチメーターを含まない）は70ユーロを超えないと主張しています。









このプロジェクトは、注意が必要なさまざまな状況で使用し、プロセスに集中することを希望する他のオタクにすでに評価されていました。 たとえば、スポーツ飛行機で飛行中にテレメトリーデータを読み取ることができます。







データセンター監視ロボット



インドのデータセンターの1つでは、サーバーの温度を監視するためにロボット全体が組み立てられました。 センターの従業員は当初、多くのセンサーを配置することを計画していたため、後で部屋全体のヒートマップを作成することができました。 しかし、実装は地元の予算には高すぎました。 妥協案として、サーバーラックを囲み、Wi-Fi経由で温度データを送信するロボットが登場しました。













ロボットは、ファッショナブルなiRobot Create掃除機、ネットブック、 Arduino UNOマイクロコントローラー、3つの温度センサーが配置されたPVCマスト、およびWebカメラで構成されています。 センサーから送信されたデータは、ヒートマップに変換されます。 ソースコードと詳細な手順は公開されていませんが、プロジェクトの作成者からのリクエストに応じて入手できます。









「宿題」として、サーバーをより完全に監視するために他のセンサーを接続できます。たとえば、湿度センサーは、予定外の洪水から鉄を節約する可能性を高めます。







Arduinoコンビネーションロック



サーバールームに飲み物やその他の貴重品を安全に置いておくには、電子ロックが便利です。これは、現代の鍵ではなく不便な鍵に大きく追加するものです。 このような城のオプションの1つを実装すると、ユーザーnikitoskが提供されます 。







ドアの外側にはパスワードを入力するためのキーボードがあり、内側にはメイン構造が固定されています。 4桁のパスワードを入力して「ポンド」を押すと、ドアが開きます。













城に必要なもの：







  1. Arduino UNO ;
  2. Arduino protoshield +ブレッドボード;
  3. ドライバーL293D;
  4. D-LINKハブからの電源とハウジング。
  5. アラームからのリモート制御。




Arduino のコードと追加コンポーネントのリストは、プロジェクトの記事に記載されています 。
