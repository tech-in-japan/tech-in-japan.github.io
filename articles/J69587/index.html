IPアドレスを記録する別の方法

おそらく、一部の人々はそのようなことを知っていますが、私は、すべてからは程遠いと思います。



ご存じのとおり、IPv4アドレスは32ビットの数値です。 従来、IPアドレスは、10進数でドットで区切られた4オクテットの形式で書き込まれます。 例： 77.88.21.8

ただし、あまり一般的ではない他の記録オプションがあります。



まず、4つのオクテットはそれぞれ8進表記または16進表記で記述できます。

77.88.21.8 = 0x4d.0x58.0x15.0x8 = 0115.0130.025.010または、 77.88.0x15.010を混在させることもできます。



第二に、アドレスは数字として書くことができます：

77 * 256 3 + 88 * 256 2 + 21 * 256 1 + 8 * 256 O = 1297618184

クレーム1と同様に、この番号は8進表記および16進表記で書くことができます： 1297618184 = 011526012410 = 0x4D581508 。



正直なところ、これらの形式を説明するRFCは見つかりませんでしたが、ブラウザ、pingコマンドなど、どこでも機能します。 自分で試してみてください。
