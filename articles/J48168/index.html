RoundCube-Gmailの拒否

こんにちは、ハブラフチャン。

私たちは皆、強力な企業であるグーグルについて知っており、私たちのほとんどはおそらくこの会社のプログラムを使用しています。



永遠のステータスが「ベータ」であるこれらのプログラムの魂を好まない人もいますが、誰かがGoogleを「邪悪な企業」と見なし、疑わしいサーバーからのCookieの受信を禁止しています。



炎に火をつけてホリバーを始めたくないのですが、もしあなたがグーグルと呼ばれる魔法のベールを振り払いたいなら、私はあなたを助けようとします。



いくつかの理由で、この投稿を大、日に書いています。



昨日、おそらく最高の無料の「ウェブベース」ラウンドキューブ電子メールクライアントがリリースされ、そのリリースは年末年始に捧げられました。 このようにして、開発者はユーザーを満足させたいと考えていました。







この記事の読者の多くはWeb開発に直接関与しており、この記事の鍵となる独自のホスティングを持っています。



GMailを置き換えるには、 roundcubeを試すことをお勧めします。 これは、「マジックトリック」にajaxテクノロジーを使用するWebクライアントと同じで、驚くほどの外観を持っています。 :)



roundcubeをインストールするための基本要件：



  • IMAP
  • mysqlまたはpostgres
  • php5




プログラムの主な機能：





  • 多言語サポート
  • MIMEおよびHTMLメッセージの完全サポート
  • 添付ファイル付きのメッセージを作成する
  • マルチセンダーのサポート
  • アドレス帳
  • ダイヤルのアドレス帳検索サポート
  • メッセージを作成するための書式付きテキスト/ HTMLサポート
  • 添付ファイル付きのメッセージの転送
  • IMAPディレクトリ管理
  • メッセージにすばやくアクセスするためのキャッシュ
  • 外部SMTPサーバーのサポート
  • メッセージと連絡先を検索する
  • スペルチェッカー
  • 高度なユーザーインターフェイス
  • 無制限のユーザーとメッセージ
  • 「スキン」使用時のインターフェースの完全な変更




このプログラムのスクリーンショットは次のとおりです。



  • 文字のリストとドラッグアンドドロップ機能の実装



  • プライバシーメッセージを表示



  • 選択したメッセージを表示します。 メッセージおよびMULTIPART添付ファイルで有効なHTML



  • 統合アドレス帳



  • 添付ファイル付きの新しいメッセージを作成し、組み込みのアドレス帳で受信者の検索を使用する



  • 複数のテキスト署名された送信者を構成する



  • 簡単なIMAPディレクトリ管理





このプログラムに興味がある場合は、インストールを開始します。



プログラムのインストール：



  • 新しいバージョンをダウンロードする
  • アーカイブを目的のtar xvfz roundcubemail-0.2-stable.tar.gz



  • chmod 777 temp/ logs/



    ディレクトリの権限を変更します
  • roundcube用の新しいデータベースを作成しますmysqladmin create roundcube



  • ブラウザにhttpを入力します：// address-where-available-our-roundcube / installer /
  • インストーラーの指示に従って、roundcubeプログラムを構成します
  • 作成してテストした後、インストーラーディレクトリ rm -rf installer/



    削除します。
  • そして今、あなたは自分自身を祝福することができます、あなたは正常にプログラムをインストールしました




ただし、常に最新のアップデートを入手したい場合は、他の方法を使用できます（サーバーにsshとsvnがある場合）。

  • トランクから最新の変更を（サービス.svnファイルと共に）ダウンロードします svn co svn.roundcube.net/trunk/roundcubemail email/



  • 前のケースで行ったように、権利を変更し、同じように設定します
  • これで、最新の更新を取得する必要がある場合は、email /ディレクトリ（プログラムを展開した場所）でsvn up



    を実行するだけで済みます。




この記事が、別の新しいオープンソース製品について知るのに役立つことを願っています。 インストールについて質問がある場合は、コメントを記入してください。
