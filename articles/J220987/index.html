Java User Group EKBの最初の会議のお知らせ

こんにちは、Habr！



複数の会社がエカテリンブルクでJavaを開発しているという事実にもかかわらず、Javaコミュニティはありません。 しかし、何かが足りないときは、それを受け取ってそれを行うだけです。 だから、JUG.ruの次の会議を見て、エカテリンブルクにはJava開発者が集まり、Javaテクノロジーのニュースについて話し合い、Javaの内部について話し合い、話し合うことができる場所が必要であることが最終的に明らかになりました！



そこで、JUG.EKBが生まれました-エカテリンブルクのJava開発者のコ​​ミュニティ。



JUG-Javaユーザーグループ、会議JUG.EKB-Javaの世界からのさまざまな技術の議論、経験の交換、興味深いコミュニケーションだけ！

JUG.EKBの定期的な会議を開催して、最も興味深いトピックに関するJavaの専門家による講義を開催する予定です。

私たちのサイトはhttp://jugekb.ruで、VKのコミュニティはhttp://vk.com/jugekbです。



すべてのJava開発者をJUG.EKBの最初の会議に招待します。





最初の会議のレポートは次のとおりです。

  • JetBrainsのAndrey Starovoit- 「コードテストの実践」

    アプリケーションの起動段階をバイパスして、IDEから直接エラーに対処する方法について説明します。



  • NaumenのSergey Tselovalnikov- 「Javaアプリケーションにスクリプトを実装して熊手を踏まない方法」

    アプリケーションにダイナミクスを追加すると、Groovyまたはその他のスクリプト言語がJavaコードに表示されます。 そして、Groovyは彼の魔法をもたらします。 最初はそれで十分ではありませんが、それからますます多くなります！ スクリプトとともに、多くの質問が発生します-そして、もし？、しかし、どうですか？、そしてなぜ？ そして、このクラスの名前を変更するとどうなりますか？ スクリプトを書くのはどれほど便利ですか？ スクリプトが壊れたのはなぜですか？ 等

    スクリプトの作成を楽しくする方法を説明します。彼らのサポートは苦痛ではなく、Groovyはあなたの親友になりました！



5月15日19時30分、8月3日8日、フリーカフェ（ショッピングセンターMytniy Dvor）での会議でお待ちしています。
