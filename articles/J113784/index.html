Windows Phoneプラットフォームの将来が明らかになります：マルチタスク、IE9およびGPU Webページアクセラレーション、Kinectとの統合など





つい最近、 スティーブバルマーの映画の中で 、マイクロソフトは2011年にWindows Phoneプラットフォームを開発する計画を明らかにしました。 Windows Phoneに関連するマイクロソフトのサービスに関していくつかの発表が行われ、プラットフォーム自体、オペレーティングシステム、およびそのアプリケーションの開発に関連するいくつかの発表が行われました。



アナウンスを順番に見ていきましょう。



WP7以降のまとめ


第一に、MicrosoftはWindows Phone 7の顧客満足度が高いことを指摘しています。同社によると、製品満足率は93％であり、9人に1人がWindows Phone 7の購入を推奨しています。



同社は、販売開始から最初の2か月で、Windows Phone 7の200万ライセンスの電話メーカーへの販売を発表しました。



Microsoftは、Windows Phone Marketplaceのアプリケーションの数が8,000を超えたと報告しました。 Windows Phone 7の28,000人の公式開発者が登録され、モバイルプラットフォーム用のアプリケーションを開発するために設計されたWindows Phone 7用の無料のVisual Studioパッケージが100万回以上ダウンロードされました。



Windows Phoneの将来の革新


キンアウト中に、2011年後半にWindows Phone 7ベースの電話を受信する多くの革新が発表されました。







Internet Explorer 9は 、デスクトップと同じ形式でWindows Phone 7に移行されます。 これは、FullHDのHTML5ビデオを含む、すべてのグラフィックサブシステムの最新のWeb標準およびGPUアクセラレーションに対する幅広いサポートを意味します。 Windows Phone 7のIE9は、オペレーティングシステムのテスト内部ビルドに既に存在します。 デモでは、 FishIEタンクテストでIE9ブラウザーを使用したWP7デバイスの動作が示されました。GPUを使用して高速化されたブラウザーは優れた結果を示しました。 Safariブラウザーを搭載した近くのiPhone 4では、グラフィックスのハードウェアアクセラレーションが不足しているため、速度が著しく遅れています。



サードパーティアプリケーションのマルチタスクサポートが表示されます。 多くのIT愛好家がこれまで長い間話してきたことは、2011年後半に起こります。サードパーティのWindows Phone 7アプリケーションはマルチタスクが可能になります。 プレゼンテーションでは、マルチタスクの実装方法が示されました。



Returnボタンを押すと、ユーザーは前のウィンドウに移動でき、この時点で動作しているアプリケーションは今日のように終了しませんが、メモリに残り、コンテキスト切り替えイベントを処理できます。たとえば、ゲームが一時停止する場合があります。 飛行機シミュレーターのゲーム中に、スピーカーがシステムのメインメニューに移動し、アプリケーションを起動したり、SMSを書いたり、メールに返信したり、自動的に一時停止したゲームに戻ったりする方法を示しました。 一時停止をオフにして、ゲームは続行しました。







上記のスクリーンショットをご覧ください。 これは、WindowsのAlt-Tabに類似した、実行中のアプリケーションのパネルです。 ユーザーはReturnボタンを長押しして、実行中のアプリケーションのサムネイルのアニメーションリストを取得できます。 リストの要素間を移動して、ユーザーは必要なアプリケーションに切り替えることができます。 すべてが通常のデスクトップオペレーティングシステムのようなものです。 デモでは、この切り替えモードは非常に迅速に機能し、遅延なしにリストに切り替え、タスクへの切り替えは即座に実行されました。



別のマルチタスクシナリオが示されました。ユーザーが別のアプリケーションまたはメインメニューに切り替えた後、バックグラウンドでサードパーティのアプリケーションでインターネットラジオを再生します。 さらに、このようなアプリケーションは、ユーザーの音楽制御（ポップアップメニューから一時停止、停止...）にすばやくアクセスするためのAPIを受け取ります。







すべてのWindows Phone 7所有者は、25ギガバイトの無料のSkyDriveクラウドストレージを使用して、オフィスのドキュメントを保存できます。 したがって、OfficeサブシステムはSkyDriveとの統合を受け取り、ドキュメントだけでなく、友人があなたのために共有している場合は、友人のドキュメントも判断してダウンロードできるようになります。



大きな驚きは、 Windows Phone 7デバイスと革新的なXbox 360 Kinectセンサーの統合の発表でした。 Kinectはわずか60日間で800万台の売り上げで売り切れたことを思い出させてください。 史上最速のガジェットとなった理由。 WP7とKinectの統合がどのように見えるかは、次のビデオで見ることができます（実際のハードウェアでの実際のゲームが示されています）。







Facebookとの既存の統合レベルで、ソーシャルネットワークTwitterとの組み込み統合が行われます。







すぐに、米国のオペレーターSprintとVerizonのネットワークのユーザーは、CDMAネットワークをサポートするWindows Phone 7に基づいた電話の対応するバージョンを受け取ります 。



Windows Phone 7ユーザーは、 3月上旬に最初のシステムアップデートを受け取ることが確認されました。これにより、コピー/貼り付けが行われ、Silverlightアプリケーションの読み込みが大幅に加速されます。



ノキアとの契約


キンクアウトの際、ノキアとの契約に関する情報が再び提供されました。ノキアは、スマートフォンの主要なモバイルプラットフォームとしてWindows Phone 7を使用します。 Microsoftは、この契約に基づいて、すべてのWP7ベンダーがNokiaとのプラットフォームの共同開発の利点を評価すると報告しています。OviMaps（NAVTEQ）を備えた更新されたBingカードなどの革新がすべての人に利用可能になるためです。 そして、モバイルプラットフォームだけではありません。 Steve Ballmer氏によると、BingとOvi Mapsを統合することの利点は、サービスを使用するすべてのBingユーザーに感じられます。



おわりに


Windows Phone 7の更新プログラムのリストは承認のみ可能です：アプリケーション間の便利な切り替えを伴うマルチタスク、最新のWeb標準、GPUアクセラレーショングラフィックス、ブラウザーでのFullHDビデオの再生機能、Kinectとの統合、Twitterサービスとの統合、専用クラウドストレージへのアクセス大音量。



これらすべての革新により、モバイルデバイス市場での競争力を高めることができます。 ノキアとの提携と、更新プログラムのリリース時点でのこの会社からの最初のデバイスのリリースの可能性により、この2011年にはすでにプラットフォームにかなりの人気があります。
