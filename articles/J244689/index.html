#TcehとMicrosoft Cloud Work Weekend

スタートアップは単なるアイデアではなく、スケーラブルなビジネスモデルを模索するプロジェクトです。 彼にはチームが必要で、数時間でゼロから組み合わせることができます。 彼は、市場での最大限の準備、サポート、主要な人々の知識、および強力な技術基盤を必要としています。



#tceh と Microsoftの 週末である場合は、このすべてを1つの週末に見つけることができます 。



12月12〜14日に、クラウドでホストする必要があるプロジェクトに重点を置いて、モスクワの中心部で収穫期を過ごします。 ここでは 、初期段階のアイデアやプロジェクトを盛り上げ、チームを強化し、仕事をしたいスタートアップを見つけ、15〜20の業界専門家や投資家に会うことができます。







週末は何ですか？



収穫の 週末 は、一部は無料の週末の週末に相当し、一部はハッカソンです-開発者だけではありません。 そして、睡眠の中断;）



  • すべてのプロジェクトは事前に選択されており、主催者は参加者にプロジェクトについて伝えます。
  • 金曜日の夜には、直接会ってチームを結成します。
  • 土曜日に、作業セッション（ 市場分析、着陸、パッケージング + 開発者向けの個別のストリーム ）を行います。
  • 日曜日に、専門家や投資家からフィードバックを受け取ります。


ハーベストは4年間、週末に興味深いアイデアの著者と若い専門家をチームにまとめ、プロジェクトを作成するための第一歩を踏み出し、教育プログラム内の知識とツールを提供し、さらに15から20の起業家、投資家、業界の専門家にアクセスします。



プロジェクトの65％は、Harvestの後に急速に成長し始めます。なぜなら、彼らはここで共同創業者を見つけ、欠員を閉じ、有名な専門家からの推奨事項と小さな業界PRを受け取るからです。



Lonersは、大規模および中規模の企業の連絡先を取得して、キャリアを加速します（そのような場合があります）。知識と最初の経験を使って将来自分のプロジェクトを作成するか、その場で好きなプロジェクトを見つけます







開発者向けの特別なストリーム



仲間のヒューマニストが着陸、CA、ピッチについて耳を傾ける一方で、 マイクロソフトの 専門家 がクラウドと開発について話すように 誘い ます。



土曜日の14:00に特別なストリームが開始されますが、顧客開発（潜在的な市場での適切な作業）に関する最初の作業セッションはチーム全体で開催されます。



週末の勤務の利点は必ずしも瞬時ではありませんが、 Afisha開発者に関する話があります。開発者はここでジオロケーションプロジェクトを見つけ、プロトタイプの作成を助け、CTOになりました。 現在、彼らは積極的に開発しています。 ドローンをピザの配達に使用した男もここから始まりました（その後、彼は会議アプリケーションを作成していました）。 Gistrollを書いたTolya Makarevichは、Harvestでプロジェクトを作成した最初の経験も受けました。



そして、ここで多くの有用な連絡先を作ることができます -専門家の間でハーベストはボブクと他の尊敬された人々でした。



開始の数日前に、主催者から選択したプロジェクトのリストが送信されます。見逃さないように登録してください。
