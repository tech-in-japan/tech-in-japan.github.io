ホスティングプロバイダーServer4You

適切なVPSホスティングの検索の問題が繰り返し発生しました。 他のプロバイダーの中で、 server4you.comが言及されました。 私たちには価格と品質の点で最高の1つであるように思われ、プロジェクトをこのホスティングに移行することが決定されました。 次に転送結果について説明します。







簡単な説明


会社はドイツにあります。 VPSとDedicatedの2種類のホスティングを提供します。 この会社に関するウェブサイトwww.webhostingtalk.comには、肯定的なレビューと否定的なレビューの両方があります。 誰も気にせず、読むことができます。 私たちの意見では、より肯定的なレビューがあります。



このプロジェクトでは、最も単純なvSERVER PRO X4料金を選択しました。

  • 25,000 MBウェブスペース（SCSI）
  • 1,024 MBのRAM
  • 2.048 MB FlexRAM
  • 5,000 GB帯域幅/月
  • 1 IPアドレスを含む
  • サーバーの場所：ヨーロッパ


この料金の費用は月額15ドルです。

サーバーをレンタルできる日付：1、12、および24か月。



1か月の期間があるという事実にもかかわらず、半年ごとまたは年ごとの請求書を発行することを検討する価値があります。 1か月注文しましたが、法案は90ドルでした。 その後、期間を12か月に変更しました。



登録後、彼らは私たちに電話し、登録を確認し、数時間後にサーバーが提供されました。 したがって、支払いなしでサーバーを取得できます。 たとえば、テストします。 請求書は4日後に発行されました。 請求書が支払われていなかった場合、サーバーは切断され、このサーバーの再登録には25ドルかかります。



また、12か月または24か月の期間を選択する場合、最初の6か月が無料で提供されることに注意してください。



支払いは、Alfa BankのMasterCardを使用して行われました。

アプリケーションのインストール


サーバーにインストールされたプログラムについて簡単に説明します。 たぶん誰かが役に立つでしょう。



  • サーバーを注文するときに、CentosOS 5.2 + LAMPが選択されました
  • パフォーマンスをテストするには、手動でphoronix-test-suiteをインストールしました 。 残念ながら、このパッケージはまだCentOSリポジトリで利用できません。
  • このプロジェクトでは、HostCMSを使用します。 このCMSには、LAMPに加えて、次のモジュールが必要です。



    yum install php-devel php-zend-optimizer php-xml php-mbstring







    zend-optimizerをインストールするには、最初に対応するatomicorpリポジトリを接続する必要があります。



    wget -q -O - www.atomicorp.com/installers/atomic.sh | sh







    php-xmlパッケージは、PHP 5のLibXSLTサポートを追加します。



  • Muninプログラムがモニタリング用に選択されました：



    yum install perl-Cache perl-Cache-Cache munin munin-node
