2016年6月4日-曇りの日2.0。 クラウドの人工知能

4月初旬に、参加者が1日でMicrosoft Azureクラウドを知るのに役立つレポートとラボで構成される仮想クラウドの日がありました。 このイベントは非常に成功したため、良い伝統を守り、 クラウドの人工知能のトピックに特化したCloud Day 2.0を過ごします。







なぜ人工知能のトピックを選択したのですか？ 現在、マイクロソフトは「 ボットパーソナルコンピューティング」の方向を急速に発展させています。このフレームワークの範囲内で、 Microsoft Bot FrameworkとCognitive Services （以前のProject Oxford）が発表されました。 これらすべてにより、アプリケーションにインテリジェンスを追加し、新しい方法でユーザーと対話することができます。これらすべては、既成のクラウドサービスを使用して、非常にシンプルなコードの助けを借りて行われます。 曇りの日のコンテンツを特別に開発し、プログラムができる小学生や学童でも理解できるようにしました。



曇りの日にサインアップした最初の800人は、Microsoft Azureクラウドリソースを使用する月に 100ドルのプロモーションコードを受け取ります。 誰かが足りない場合、演習の80％ を学生向けの無料クラウドオファーを使用して行うことができます。



曇りの日の一部として、いくつかの大きなトピックが提示されます。

  • AI の簡単な紹介。AIのさまざまな分野、現代世界でのAIの役割、およびよりパーソナルコンピューティングの方向性について一般的に説明します。 モスクワ航空研究所の応用数学部で「人工知能」コースを教えているので、この講義では、最初の講義で学生に話すことのほんの一部を共有します。
  • Microsoft Bot Frameworkを使用してチャットボットを作成します 。 Haberの記事で既にこのトピックに触れましたが、今度は、フレームワークのさまざまな要素がどのように使用されるかについて、ダイアログ、会話チェーン、FormFlowテクノロジー、認知LUISサービスを使用した自然言語認識について詳しく説明します。
  • 認知サービスの利用 、特に、人、性別、年齢、感情の認識サービス。 インターネット上で最も幸せな人を画像で判断する興味深い計算実験を実施します（利用可能なクラウドリソースを使用して）自宅でこの実験を続行できます（たとえば、VKontakteの友人の中で最も幸せな人を見つけます）。
  • Machine Learning ：Azure Machine LearningとMicrosoft R Serverを使用してデータから知識を抽出します。 私たちの最高の機械学習の専門家であるアレクサンダー・ベロセルコフスキーは、魅力的な例を示します。 過去の曇りの日にこのトピックについて少し触れましたが、エキサイティングな続きがあなたを待っています！


前回と同様に、便利なペースで視聴できるビデオ講義と、一連の簡単な実験室作業があります。 曇りの日には、当社の専門家がお客様の質問にリアルタイムでお答えします。 トレーニングを効果的に完了するには、課題を行うことをお勧めします。そのためには、次のものが必要です。

  • Microsoft Studio SDKがインストールされたVisual Studio 2015 （できればUpdate 2）。
  • Microsoft Azure Cloudへのアクセス 。 登録後、イベントの数日前に、クラウドを使用するためのプロモーションコードを受け取ります。これは、イベントで時間を無駄にしないために事前にアクティブ化する必要があります。 Cloud Day 1.0の 最初の講義で 、コードをアクティブにする手順を見つけることができます。


曇りの日2.0でお待ちしています！ これが夏の良いスタートになることを願っています！
