ウランウデ出身のプログラマーのデルフィックサクセスストーリー

世界は厳しいですが、小さく、誰もがそれを変える機会があります。 1996年、新しく登場したDelphiの研究を始めて、この言語の運命がどうなるか想像できませんでした。JavaとC＃に大きな影響を与え、ボーランドの会社自体の運命は、すべての古い学校のエンコーダーがガチョウの隆起を持っています肌に。



私は歴史的な出来事の小さな断片を作って、ウランウデ出身のあまり知られていないロシア人プログラマーについて話そうとしました。 ロシアの中心にいる一人が歴史に没頭し、大企業の競争に影響を与えることができるのは非常にクールです。 これにより、努力と希望を正当化できると考えるすべての理由が得られます。







写真では、古代ギリシャの都市デルファイのタイプの1つであり、その後、開発環境がデルファイと命名されました。 ソース-en.wikipedia.org



ボーランドの運命は困難であることが判明しました。 DelphiのチーフディベロッパーであるAnders HalesbergをMicrosoftに誘い、その後、象徴的なBorlandを不明瞭なInpriseに奇妙に改名しました。 売上高の減少、およびその結果としての部品自体の会社自体の一連の再販。



2005年にGoogleがどのようにモバイル市場の重要性を認識したかについては詳しく説明しません。 しかし、2005年には、Googleは1億3,000万ドルでAndroidを購入しています。



2007年、iPhoneのリリースにより、多くの人が新しい時代が到来したことを理解しており、Appleはモバイル市場におけるリーダーシップへの道のすべてを一掃するでしょう。 GoogleでさえAndroidを使用する時間がありませんでした。Androidの最初のバージョンはSDKとともに2008年にのみ登場しました。



2007年、新しく設立されたトーマクレッシーブラボー基金は、エンバカデロの株式の100％を償還します。 翌年、2008年、EmbarcaderoはCodeGearを23百万ドルで購入し、Delphiの特許が譲渡されました。 2006年以来、CodePearはPalmPilotハンドヘルド用のC ++コンパイラのサプライヤとして覚えています。これは、今日の大画面を備えた強力なスマートフォンの最初のプロトタイプになりました。



アナリストによって予測された可能性について明らかに頭を失った2010年に、HPは、Appleよりもクールになることを期待して、その価値のピーク時にPalmを10億ドルで買収しました。 次の2011年にはすでに、HPはPalm Preモバイルデバイスの製造を中止しました。



その後、モバイル市場でのこの不平等な戦いで、小さいながらも非常にクールな瞬間が起こります。 新製品の最初の購入者はオタクであり、その中には多くのプログラマーがいると信じています。そして何よりも、彼らはデバイスのクールさと視点を決定します。 成長するモバイル市場の主な見通しの1つは、クロスプラットフォームアプリケーション開発の可能性です。



2011年、上記のDelphiの遺産を継承したEmbarcaderoは、VG-Sceneの権利を購入しました。これは、Ulan-UdeのEvgeny Kryukovが次世代のベクトルベースのGUIライブラリとして開発したGUIフレームワークです。 エフゲニー・クリコフは後に、トロント、メルボルン、ミュンヘン、メイデンヘッド（イギリス）、スペイン、サンクトペテルブルク（ロシア）に事務所があるエンバーカデロで働くことを申し出ました。



2012年、FireMonkey（FMX）と呼ばれるこのライブラリは、VCLの主要パッケージであるDelphi XE3、Delphi XE2、C ++ Builder XE2、およびRadPHPおよびEmbarcadero Prismを含むRAD Studio XE2に含まれています。 DelphiがWindows、Mac OS X、Linux、Android、およびiOSで同じコードを実行できるようにするのは、このライブラリです。

FireMonkeyでは、ユーザーインターフェースをビジネスロジックおよびデータアクセスメカニズム（model-view-controllerを参照）から分離することができます。特に、アプリケーションの非視覚的な部分をクラウドサービス（AmazonまたはAzure）に配置できます。
Evgeny Kryukovの開発により、DelphiとEmbarcaderoの両方が持続可能な競争力に戻りました。 Delphi / Object Pascal言語の人気は低下しているにもかかわらず、2019年にはTIOBEプログラミング言語のランキングで14位になりました。
