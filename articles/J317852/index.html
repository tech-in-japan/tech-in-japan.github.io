顧客に明けましておめでとうございます。







新年の休暇が来ています-それは積極的な販売とお祭りの時間です。 新年のあいさつは、顧客に自分自身を思い出させる良い方法であり、積極的に認識されています。



大量の情報スパムの中で目立つ方法は？ おもしろいオプションは電話での挨拶です。 特に、このプロセスを自動化し、このタスクに追加のオペレーターを関与させない場合。



コストを最小限に抑え、プロセスを可能な限り自動化する必要があります。 これを行うには、自動モードで電話をかけることができるサービスが必要です。 機能が十分に開発されたサービスが適しています。 この記事では、このAPIの例を示します 。



自動化する理由 結局のところ、生きている人とのコミュニケーションはより大きな結果をもたらします！ 実際のところ、これはお祝いであり、直接販売ではありません。 そして、このようなお祝いの費用は、顧客ベースに手動で電話をかけるよりも桁違いに低く、時間がかかりません。



まず、メッセージを準備する必要があります。

Hello％clientname％！ 会社会社名はあなたに明けましておめでとうございます！ 新しい年に、成功と繁栄をお祈りします！ ギフトとして無料のお土産を提供します！ 数字の1を押すと、ギフトをどこで受け取れるかがわかります。


このメッセージは、各メッセージ-％clientname％の変更部分が含まれていることを示しています。 一般に、クライアントとの名前によるコミュニケーションは、彼をより積極的に設定します。



実験中、フレーズ全体の再生品質は平均的であることが判明しました。 「Hello％clientname％！」とその他すべての2つの部分に分けることにしました。 フレーズの2番目の部分は変更されません。最初の部分を発音するものにできるだけ近い音声でレコーダーに録音します。



次に、IVRメニューを作成し、フレーズの2番目の部分を追加します。 数字の1をクリックすると、オペレーターと接続します。





テストにはPowerShellを使用します。 システムにインストールされていない場合は、 ここから入手できます 。



PowerShellコンソールでのテスト： 



$web = New-Object System.Net.WebClient $url = 'https://digitoffice.ru/api2/dsk3z6r3drfxp4ghyua9kgsedk/initCall?to=79xxxxxxxxx&cid=7yyyyyyyyyy&ivr=menu1&phrase_params=5,1,0.9&phrase= !' $web.DownloadString($url)


またはBASHコンソールで 
 wget -O - "https://digitoffice.ru/api2/dsk3z6r3drfxp4ghyua9kgsedk/initCall?to=$number&cid=70000000002&ivr=menu1&phrase= $name!" 2>/dev/null




ここで、79xxxxxxxxx-呼び出し先、7yyyyyyyyyyy-コールセンターの電話番号。 この番号からコールが発信され、不在着信時にコールバックされます。 この番号への着信は、音声メニューmenu1にも転送できます。



次に、自動呼び出しのベースを準備します。 たとえば、Excelで準備できます。 最初の列には電話番号、2番目の列にはクライアントの名前を書きます。 次に、データベースをcsv形式でエクスポートします。 ベースは次のようになります。 



 79000000001; 79000000002; 79000000003;  ...


次に、PowerShellの小さなスクリプトで自動呼び出しを開始します。 このスクリプトは、call.ps1などのファイルに保存するか、単にPowerShellウィンドウにコピーして貼り付けることができます。 主なことは、オフィス番号、音声メニューの名前、APIkeyを指定し、phones.csvファイルが同じディレクトリにあることを確認することです。 



 $web = New-Object System.Net.WebClient Import-Csv -Delimiter ';' -Header phone, name “phones.csv” | ForEach-Object { # process line $url = 'https://digitoffice.ru/api2/dsk3z6r3drfxp4ghyua9kgsedk/initCall?cid=70000000002&ivr=menu1&phrase_params=5,1,0.9&to=' + $_.phone + '&phrase= ' + $_.name + '!' $web.DownloadString($url) echo $_ >> log.txt Start-Sleep -Seconds 3 }


またはBASHスクリプト 
 #!/bin/sh cat $1 | while IFS=';' read number name ; do echo $name $number wget -O - "https://digitoffice.ru/api2/dsk3z6r3drfxp4ghyua9kgsedk/initCall?to=$number&cid=70000000002&ivr=menu1&phrase_params=5,1,0.9&phrase= $name!" 2>/dev/null echo $number » dial.txt sleep 3 done


スクリプトをcall.shに保存し、実行可能ファイルをchmod + x call.shにします

コマンド./call.shphones.csvを実行します



Start-Sleep / sleepコマンドは、呼び出し間の一時停止サイズを調整します。 一時停止の長さは、顧客ベースのサイズとコールを受信するオペレーターの数によって異なります。 顧客ベースをいくつかの部分に分割し、それらを手動で起動することをお勧めします-プロセスを制御しやすくなります。



これは、顧客へのお祝いを整理するのに難しい方法ではありません。
