Windows 8用のMozilla Firefox Metroデザインコンセプト

Windows RTでのサードパーティブラウザの使用を制限するというMicrosoftの意図と、Firefox Metroプロジェクトに取り組んでいる財団のUXデザイナーの1人であるYuan WangがMozillaとGoogleからのアピールを知ろうとする前に、彼の作品のビデオを公開しました新しいブラウザの概念設計の上。



以下のビデオは、Firefoxのメインウィンドウ、クイック選択パネル、ブックマークの操作、その他のポイントを示しています。
