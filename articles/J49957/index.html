自宅でピジンにパッチを当てて収集する

このトピックは、Pidgin IMクライアント用のパッチのリリースから着想を得ています。 本質は他のクライアントの本質と同じです-ランダムに生成された値の代わりに、既知の「正しい」オプションの番号がシーケンス番号に置き換えられます。



どうする 迅速にアップグレードする方法は？



遅延-liboscar.so.0.0.0をダウンロードし、事前に保存して、/ usr / lib / purple-2に配置します。 バージョン2.5.2でチェックすると、2.5.4で動作するはずです。 32ビットアーキテクチャ用に構築。



怠Notではありません-読んでください！



1）ターミナルを開き、eveginからpidginソースとパッチをダウンロードします

wget downloads.sourceforge.net/pidgin/pidgin-2.5.4.tar.bz2

wget developer.pidgin.im/raw-attachment/ticket/8198/pidgin-NOT-UPSTREAM-2.5.4-icq-russia.patch

2）アセンブリに必要なパッケージをインストールします（すべてが必要というわけではありません。起動時に./configureは対応する機能を無効にするキーを提供します）

sudo aptitude install libglib2.0-dev libgtk2.0-dev libxss-dev libstartup-notification0-dev libgtkspell-dev libxml2-dev libgstreamer0.10-dev libmeanwhile-dev libavahi-ui-dev libdbus-glib-1-dev network-manager -dev libperl-dev libgnutls-dev tcl-dev tk-dev gcc make

3）pidgin-2.5.4.tar.bz2を解凍します

tar xvfj pidgin-2.5.4.tar.bz2

4）パッチ！

パッチ<pidgin-NOT-UPSTREAM-2.5.4-icq-russia.patch

5）構成、組み立て、インストール

cd pidgin-2.5.4

./configure --prefix = / opt / && make

sudo make install



私たちは/ opt / bin / pidginを開始し、喜んでいます:)
