動的なコード検証-オンラインショッピングの利便性を維持しながら詐欺のリスクを最小限に抑えます

動的コード検証（DCV）テクノロジーは、カード詐欺に対する国際的な戦いの突破口です。 CNPトランザクションに最高レベルのセキュリティを提供します。 統計によると、EMVカード市場での詐欺の70％はCNP詐欺です。 DCVソリューションはCPN操作をどの程度正確に保護しますか？





Gemalto Dynamic Code Verificationは、カードを使用しないトランザクション保護（CNP）ソリューションであり、パラメーターは発行銀行によって設定され、ディスプレイ付きの支払いカード、モバイルデバイス用のソリューション、確認サーバーの3つの主要な要素が含まれます。 この構成により、銀行はインターネットでの購入の利便性を損なうことなく、詐欺の可能性を最小限に抑えることができます。



2013年、全世界のインターネットユーザーの41.3％1がオンラインで商品を購入しました。 予想どおり、2017年にはこの数値は46.4％に増加します。 オンライン購入の増加と物理的なカードトランザクションの保護の強化により、支払いの不正は、次の（攻撃者にとって最も魅力的であると同時に、最も脆弱な）チェーンリンクであるカードレストランザクション（CNPオペレーション）に予想通りシフトします。 単一欧州支払スペース（SEPA）で発行され、世界中で取得されたカードを使用した不正取引の合計額は、2013年に14億4,000万ユーロ2であり、2012年より8％増加しています。



この成長の背後にある主な要因は、CNP詐欺であり、2013年の損失は958百万ユーロでした。



そのため、CNPオペレーションでの詐欺を防止し、そのような犯罪と闘うことが、オンラインショッピングの汎用性と利便性を維持するための前提条件になります。



消費者はオンラインショッピングを便利にしたい



ほとんどの消費者はセキュリティの必要性を理解していますが、オンラインショッピングエクスペリエンスを可能な限り高くしたいと考えています。

快適。 統計によると、平均して、トランザクションの68.53％が不完全であることが判明しています。買い手は、複雑すぎるか、時間がかかりすぎる場合、チェックアウトプロセスを停止します。



発行者は、既存のインフラストラクチャの変更を最小限に抑えたい



詐欺の発生率の高まりは、銀行や店舗が3Dセキュアテクノロジーを使用して認証ソリューションを実装するよう促しています。 ただし、そのようなソリューションは支払いインフラストラクチャにうまく適合せず、消費者の間で常に人気があるわけではありません。



Gemalto Dynamic Code Verification（DCV）スイート



GemaltoのDynamic Code Verificationツールキットは、ディスプレイ、モバイルデバイス用アプリケーション、および確認サーバーを備えた支払いカードの使用に基づく、CNPトランザクションを保護するための完全なソリューションです。 このソリューションにより、カード発行会社は、従来はオンラインでの購入に使用されていた静的で安全な暗号を、顧客のEMVカードまたはモバイルアプリケーションに表示される動的コードで置き換えることができます。 限られた期間だけ有効なこのコードは、不正取引のために詐欺師による盗まれた静的カードデータの使用を除外します。



さらに、Gemalto DCVソリューションにより、カード発行会社は、オンラインストア、支払いシステムのオンライン支払いのインフラストラクチャを変更することなく、不正リスクを減らすことができます。



Optelio DCVは、EMV標準の古典的な連絡先または非接触型決済カードであり、その背面には動的コードが表示されます。 このコードは、カード所有者の参加なしに、時間の経過とともに自動的に変更されます。



Ezio DCVモバイルソリューションにより、顧客は携帯電話でカードを保存し、動的コードを読み取ることができます。 ユーザーの要求に応じて、アプリケーションをPINコードで保護できます。これにより、追加レベルの認証保護が提供されます。 このソリューションにより、安全に買い物をすることができます。

オンラインで保存されたカードを使用し、DCV機能を使用します。



Ezio DCVサーバーは、ユーザー、カード、およびデバイスに関連するすべての情報を保存します。 このサーバーは、カードまたはモバイルソリューションによって生成された動的コードを確認し、結果を認証ノードに送信します。

セキュリティの向上。 ジェムアルトのDCVテクノロジーは、CNP詐欺との国際的な戦いにおける真のブレークスルーであり、カード決済ビジネスの弱点ではなくなったCNPトランザクションの保護に役立ちます。



詐欺事件の数と詐欺による損失額を減らす。 提案されたソリューションは、発行者が詐欺を即座に終わらせ、CNP詐欺の対策に関連するコストを削減するのに役立ちます（特に、コールセンターの作業、請求の解決、カードの再発行に関して）。



実装の容易さ。 DCVソリューションは発行者に焦点を当てています。発行者は実装を完全に制御できます。 統合プロセスにオンラインストアやアクワイアラーバンクを関与させる必要はありません。



すべてのコンポーネント-マップ、モバイルアプリケーション、およびサーバーは完全に統合されています。 確認サーバーは標準コンポーネントに従って実装されます

ITインフラストラクチャ。 これにより、Gemalto DCVソリューションを既存の支払いアーキテクチャに簡単に統合できます。



選択の自由。 発行者は、ソリューションをマップ上にのみ、モバイルアプリケーションとしてのみ展開するか、両方を組み合わせて使用​​することができます。 これにより、ユーザーの特定のセグメントのニーズに適応し、後者の最適な条件を提供できます。
