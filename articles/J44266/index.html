Windows Mobile 6.5はまもなく登場します...

前述のように、Windows Mobileオペレーティングシステムの次のバージョンが、7ではなく、インデックス6.5でリリースされることは間違いありません。



モトローラのモバイルデバイスセクターの最高執行責任者であるSanjay Jhaは、10月末に偶然または意図的に、2009年後半にWindows Mobile 6.5ハンドヘルドを発売する計画について話しました。



このフレーズは、MicrosoftのCEOであるスティーブバルマー（スティーブバルマー）がいなければ、ランダムな予約のままでした。 オーストラリアのTelstra Investor Briefing 2008カンファレンスで、BallmerはWindows Mobile 6.5の存在を確認しました。WindowsMo​​bile 6.5の発表は、来年上半期に予定されていると彼は言いました。







UPD：既にあった場合、検索は何ももたらしませんでした。





残念ながら、Windows Mobile 6.5について話すとき、Ballmerは詳細には触れませんでしたが、インターフェイスの改善とソフトウェアアプリケーションのより正確な作業を待つことを約束しました。 Steve Ballmerは、Zuneプレーヤーソフトウェアアプリケーションを新しいオペレーティングシステムに統合するという同社の意図も発表しました。 この革新は、モバイル版とPC版の両方に適用されます。



Softpedia.comは、Microsoftの情報源を引用して、Windows Mobile 6.5が来月のテストのためにMicrosoftパートナー企業に送られると主張しています。 テストがうまくいけば、Windows Mobile 6.5は2009年の第1四半期に本当に光を見るでしょう。



Windows Mobile 7オペレーティングシステムに関しては、すべてのアナリストは、2009年の第3四半期より早く登場することを期待しています。 Ballmer自身は、Windows Mobile 7について言えば、期限に言及しないようにしました。
