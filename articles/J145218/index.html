Firefox 13リリース-SPDY、新しいホームページ、タブ付き









Firefox 13のベータリリースから1か月以上が経過し、今日、最終的に最終バージョンが登場しました。 これはTechCrunch によって報告されています。 新しいバージョンで最も重要なのは、長く期待されている新しいホームページと新しいタブのサムネイルです。 さらに、FirefoxはGoogleのSPDYプロトコルをサポートするようになりました。これにより、それをサポートするサイトのダウンロード時間を大幅に改善することができます。



ブラウザの起動速度を改善するために、Mozillaは新しい起動時の自動タブ回復を無効にしました。 ただし、この関数は手動で返すことができます。











Firefox 13が起動すると、新しいホームページが読み込まれ、ダウンロード、ブックマーク、閲覧履歴、アドオン、設定にすばやくアクセスできます。 以前のバージョンと同様に、ページの中央はGoogleからの検索で占められています。 新しいタブを開くと、ChromeなどのFirefox 13には、最もアクセスの多いサイトのサムネイルが表示されます。



全体として、これはここ数か月で最も興味深いFirefoxのリリースの1つです。 ここですべての変更の詳細な説明を見つけることができます。 公式のダウンロードリンクは明日表示されますが 、 Windows 、 Linux、およびMac用のFirefox 13を既にダウンロードできます。
