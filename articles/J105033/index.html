USBフラッシュドライブ

フラッシュドライブがユーザーの「進歩」の指標と見なされていた時代は忘れられません-ボリュームが大きいほど、より涼しいと思われます。 128メガ、256、512 ...これらのボリュームの価格は大きく異なり、選択は非常に少なかった。







今ではすべてが少し異なります-ボリューム自体に苦労はありません。代わりに、「最も必要な」だけでなく、「必要なものすべて+少なくとも年に1回役立つすべてのもの」を持ち運ぶという単純な（完全に自然な）欲求があります。



多くのドライブが使用されただけでなく、非常に多くのドライブがありました-あらゆる味と色のボリューム。 一方、このノイズの下で、データが交換される速度という別の質問が前面に出ました。 最近、データ転送速度が「ボトルネック」になっています-ドライブが大きいほど時間がかかるためです。



しかし、すべてが私が悪化したほど悪いわけではありません-誰もが本当に高速フラッシュドライブを必要とすると想像しても、進歩はありません-新しい高速インターフェースの広範囲な出現、メモリ技術の開発、「ギガバイト」の論理的な低下など...私自身はこれを待っていますが、今のところはすでに持っているものを使用しています。 たとえば、今日、32GB Verbatim eSATA / USB Combo SSDフラッシュドライブのレビューがあります。その主な特徴の1つは、コンピューターに接続するための2つのインターフェースの存在です。 異常であり、説明から判断すると、非常に興味深い-だから私はこれらの行を書いています。



外観



フラッシュドライブは黒で、耐摩耗性のあるマットプラスチック製です。 サイズは非常に大きく、およそ、オペレータの3Gモデムのようです。







しかし、ここでは少なくともそのような寸法がどこから来るのかが明らかです-ドライブの両側はコネクタにあります。







一方で、USB（2.0）、もう一方では、ご想像のとおり、eSATA。 何でも使用できます-切り替えは、信頼性の高いロック付きの可動スイッチにより行われます。







ドライブに加えて、このキットには小さな紙のマニュアルとY字型のワイヤも含まれています。これは、初めてeSATAのマザー、もう1つはUSBとSATAのファザーです。 賢い人たちは次のように提案しました。 つまり、1つのコネクタを介してデータ交換と電力供給が行われます。 そのためには、同じPowered eSATAまたはeSATA Comboコネクタを備えたボードが必要です。SA​​TAはUSBと組み合わされ、USBとして使用できますが、この場合は電力のみが供給されます。 しかし、ボードに通常のeSATAコネクタがある場合は、スプリッターを使用してフラッシュドライブに電力を供給する必要があります 。







特徴



最も興味深いのは、もちろん、速度インジケータです-接続インターフェースに関係なく印象的です。







根拠がないように、 HD Tuneプログラムの速度インジケーターのすべての種類のグラフ（読み取りおよび書き込み）を以下に示します。

あいまいな録音スケジュールにまだ会っていないので、正直なところ、少し混乱しました。 わかりました、グラフ-事実は、ドライブへのデータの書き込みは実際にやや独特です。 そのため、たとえば、非常に大きなファイル（たとえば、映画）をコピーする場合、コンダクターは非常に高速（最大150Mb / s）を表示しますが、時間の経過とともに急速に低下します。

このため、次の画像が得られます-最大500 MBのサイズのファイルがほぼ瞬時にコピーされます（たとえば、453MBのクリップが6秒でドライブに表示されます。つまり、書き込み速度は75MB / sをわずかに超えます）。さらに時間がかかります（1.47GBの映画は40秒でコピーされます。つまり、ほぼ37Mb / sです）。 さらに、多数の小さなファイルをコピーすると、速度も「小さく」なります-1.92GBの音楽フォルダがUSBフラッシュドライブに65秒間コピーされました。つまり、速度は29.5Mb / sでした。 どうやら、すべてがキャッシュ内にありますが、いずれにしても、ドライブは非常に高速です。



良い例です-468メガバイトのビデオがUSBフラッシュドライブにほとんどコピーされ、ファイルマネージャーはこのアクションの時間を計算する時間すらありませんでした。







上記の図はすべて、最も一般的なUSBインターフェイスバージョン2.0で得られたものです。eSATAの場合、結果はさらに素晴らしいものになると思います。



だから何よりも、高速は「小さな」ボリュームの断片ファイルで顕著です（最大300-500MB。ところで、ほとんどの場合、そのようなファイルで作業する必要があります）が、まだ大量のデータを待たなければなりません。 いずれにしても、このドライブの速度は、他のほとんどの「通常の」フラッシュドライブの速度よりも高速です。

ドライブに、最初のフォーマットの犠牲になった何らかの中程度のユーティリティソフトウェアがあったことを除外しません。

* UPD * USBフラッシュドライブには、 EasyLockユーティリティの形式のソフトウェアが実際にありました。 コンピューターに追加のソフトウェアをインストールしなくても、コンボSSDへの転送時に既にデータを暗号化できます（256ビットAESアルゴリズムを使用）。

終わり



部分的に、USBフラッシュドライブを介して情報を交換する際の「システムボトルネック」の問題により、USB 3.0インターフェイスの外観が解決されますが、システムのアップグレードとドライブの変更に全員が同意するわけではありません。 私のレビューからのフラッシュドライブは、まったく待つ必要がないことの明確な例です。また、ユビキタスUSB 2.0でも珍しいeSATAでも、既存のインターフェイスで十分な速度を得ることができます。これは、インターフェイスそのものではなく、ドライブ自体をいっぱいにすることです。

価格？ 私は同意します、このお金で2TBのドライブを取り、1週間それを食べることができます...しかし、これは、毎日（ドライブに比べて）小さくて速い強力なフラッシュドライブが必要な場合は何でもありません。 この場合、高価格は少なくとも何らかの形で正当化することができます-少なくとも、USB 3.0（マザーボードやラップトップは言うまでもありません）のコントローラーと、新しいインターフェイス用の同じサイズのフラッシュドライブを購入する量よりも少なくなります。 したがって、再び、あなたはニーズから先に進む必要があります。



* UPD * 「フォーク」サービスusbflashspeed.comのバージョンによると、フラッシュドライブの読み取りおよび書き込み速度が最高であることが判明しました 。 どうやら、結果はUSB 2.0ポートの仕様に基づいていたようです。
