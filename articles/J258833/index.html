iOSアプリのコンテンツをGoogle検索結果やその他の良いニュースに表示する

こんにちは、Habr！ ほんの数日前、Googleの検索結果にAndroidアプリケーションのコンテンツを表示することについて話しました。 しかし、私たちはそこで終わりではありませんでしたが、今日はiOSアプリケーションのインデックス作成を紹介します。 iPhone、iPad、iPod touchのユーザーは、検索結果からモバイルアプリケーションページを直接開くこともできます。







この機能は、今後数週間でiDeviceの所有者が利用できるようになります。 動作するには、公式アプリケーションまたはGoogle Chromeブラウザーを使用してGoogleアカウントにログインする必要があります。その後、最初のインデックス付きアプリケーションからコンテンツを受信できます。



iOSアプリケーションのインデックス作成をリクエストする方法



残念ながら、この機能へのアクセスを全員に提供することはできません。 これまで、この機会は、iOS開発に精通しており、イノベーションを成功裏にテストするのに十分な視聴者を抱えている、最も人気のある責任ある開発者の限られた数のアプリケーションに実装されています。 しかし、これは参加できないという意味ではありません。 近い将来、すべてのソフトウェア会社がアプリケーションのインデックスを作成できるようになります。 それまでの間、準備することをお勧めします。



4つの簡単な手順で、検索結果からアプリケーションコンテンツに直接アクセスすることを最初に活用できます。

  1. iOSアプリケーションのコンテンツへのリンクのサポートを実装します。
  2. アプリケーションから、ワンクリックで検索結果に戻ることができることを確認してください。
  3. サイトのアプリケーションのコンテンツへのリンクの説明を作成します。
  4. インデックス作成リクエストを送信します。 リクエストを送信しても、iOSアプリケーションのコンテンツへのリンクがGoogle検索結果に表示されるとは限らないことに注意してください。


goo.glを使用するアプリケーションのコンテンツへのリンク



2番目の良いニュースは、最新のWebでのアプリケーションの統合にも関係しています。 アプリケーションコンテンツのインデックス作成の新機能により、goo.glは今日からWebページを指すだけでなく、モバイルアプリケーション内のコンテンツへの短いリンクを作成するためにも使用できます。



この機能を機能させるには、AndroidまたはiOSアプリケーションのインデックスを作成する必要があります 。 コンテンツのインデックス作成後、特別なgoo.gl URLを作成して、サイトの特定のページに移動するだけでなく、モバイルアプリケーション（インストールされている場合）内で選択したコンテンツを開き、モバイルデバイスの所有者の生活を楽にします。



ユーザーにアプリケーションがインストールされていない場合、リンクは「昔ながらの方法」で機能し、サイトに送信します。 この機能は、新規および以前に作成された短いgoo.gl URLの両方をサポートします。





Zillowは、Twitterページに短いURLを公開しており、アプリケーション内の特定のページに移動できます。



みんなのためのユニバーサルリンク



アプリケーションの「共有」機能でURL Shortener APIを使用すると、任意のデバイスのユーザーがgoo.glユニバーサルショートリンクをアプリケーション内のコンテンツに共有したり、リソースに投稿したりできます。



仕組みは次のとおりです。Googleマップアプリケーションを使用して、お気に入りのレストランの住所を共有するとします。 さまざまなプラットフォームの新しいgoo.glリンクとの共有をサポートしているため、[共有]ボタンをクリックすると、共有に便利な1つの短いURLが作成されます。 このようなリンクをクリックすると、システムはユーザーのプラットフォームとGoogleマップアプリケーションの可用性を決定します。 インストールされている場合、コンテンツはその中に直接表示されます。 それ以外の場合、ユーザーはマップ上のこのコンテンツを含むページにリダイレクトされます。



自分で試してみてください！ これを行うには、Googleマップアプリケーションを使用してスマートフォンで次のリンクを開きます： http : //goo.gl/maps/xlWFj



goo.glのコンテンツへのリンクを設定する



リンク短縮サービスの操作は簡単です。

  1. 開始するには、AndroidアプリまたはiOSアプリのインデックスを作成します。 これを行うには、 g.co / AppIndexingページの指示に従って、既存のコンテンツに関するデータを収集します。 Google検索のコンテンツリンクとは異なり、goo.glリンクはすべてのAndroidアプリまたはiOSアプリの開発者が利用できます。 インデックス作成が完了すると、goo.glの短いリンクがアプリケーションにリダイレクトされます。

  2. アプリケーションまたはメールニュースレターの「共有」機能でShortener API URLを実装することもできます。

    この場合、ユーザーがアプリケーションに直接アクセスできるリンクが自動的に作成されます。
