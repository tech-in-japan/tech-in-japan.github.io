スマートQ7 MIDレビュー

新しいおもちゃ-Smart Q7 MIDを入手しましたが、ハブでレビューを1つも見つけられなかったので、教えて表示します





特徴



プロセッサー： Samsung ARM S3C6410 667 MHz

メモリ： 128 MB RAM、1GB ROM

オペレーティングシステム： Ubuntu Linux（デフォルト）、WindowsCE（ライセンス購入が必要です〜15 $）、Android、Mer

画面： 7 "800 x 480 TFTタッチ（指、爪、スタイラス、および一般的にできるすべてのものを突くことができます）

バッテリー：リチウムポリマー4500mAH（約6〜10時間のアクティブな作業と約2日間の音楽鑑賞に十分）

インターフェース： USBホスト、USB OTG、SD / SDHCカード用スロット

ワイヤレス： WiFi、Bluetooth（USBホストのアダプター）

オーディオ：通常の3.5 mmヘッドフォンジャック、内蔵ステレオスピーカー

重量：約0.5 kg



外観





デバイス自体の正面図：左、スクロールアップ/ダウンボタン、機能ボタン





側面図：ヘッドフォン入力、USBホスト、USB OTG、電源





接続されたキーボードとサイズ比較のためのnokia 3250



含まれるソフトウェア



デフォルトでインストールされた各プログラムの詳細なレビューを行うことは意味がないと思います。オペレーティングシステムは通常のUbuntu（4月9日または10月9日）であるため、メインについて簡単に説明します。

インターネット： Claws Mail、Torrentダウンロード、RSSリーダー、Pidgin、Midori（WebKitエンジン（Safariのブラウザ）の非常に気の利いたブラウザ、唯一の欠点は、Flashのサポートがないことです。つまり、YouTubeも視聴できません）

ビデオプレーヤー： SMPlayerはMPlayerベースのプレーヤーで、通常のMPlayerが再生するすべてを再生します。 mpeg4形式（xvid、divx、h264など）のフルレングスの映画と約500〜1000 kbpsのビットレートを落ち着いて見るのに十分なプロセッサ能力があります。

オーディオプレーヤー： Sonata + MPD

ブックリーダー： FBReader-ハーフシートA4のサイズのデバイスはほとんど本のようなものなので、本を読むのは非常に便利です。

オフィス： AbiWord、Gnumericおよびevince（pdfリーダー、djvu）

その他：メモ帳（リーフパッド）、オーガナイザー（osmo）、ターミナル（ない場合）、アーカイバー、辞書、およびさまざまなシステムユーティリティ



インターネット接続



売り手が約束したように、USBモデムをサポートしています 。 メガホンモデムキットから簡単に接続できるHuawei E160G（ファームウェアバージョン5については以下で説明）：

  • モデムを挿入し、画面上の3Gアイコンをクリックします
  • 表示されるリストで、GPRS / EDGEを選択し、Gnome-pppが開き、セットアップをクリックします
  • 設定で「検出」をクリックします-モデムは/ dev / ttyUSB0になり、最大速度を設定します
  • 「Init String」ボタンを押して、2行目をAT + CGDCONT = 1、「IP」、「internet」に固定します（メガホンの場合）
  • ウィンドウを一番下までスクロールし（中央の左ボタンを押してウィンドウをドラッグします（Alt +マウスでドラッグアンドドロップなど）、[閉じる]をクリックします
  • ユーザー名とパスワードのフィールドでは、インターネットをハンマーで打ちます
  • 「接続」をクリックして、接続するのを待ちます


WiFi接続はさらに簡単です。

  • 画面またはトレイのWiFiアイコンのWiFiボタンを押します
  • 必要なWiFiアクセスポイントを選択し、[接続]をクリックします
  • ポイントがパスワードで保護されている場合など、必要なデータを入力してお楽しみください
