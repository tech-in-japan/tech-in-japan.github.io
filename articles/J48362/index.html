USB付きmicroSD



このように始めましょう 。SDカードリーダーを備えたラップトップを使用し、そこに通常のUSBスティックを挿入して、突き出ないようにします。



いいえ、この方法の方が良いです。SDを備えたカメラがありますが、何かを捨てたいときはいつでも、家の中を走り回って（またはさらに悪いことにオフィスの周りを）ケーブルを見つけようとして失敗します（せいぜい-miniUSB） 、またはカードリーダー、そしてあなたは思う：「まあ、それはなんて不便だ、21世紀はすべて同じだ！」



または 、お気に入りの携帯電話でパーティーでたくさんのクールな写真を撮ったので、もちろん、今ではVKontakteにアップロードしたいと思います。 「サムスン用のレースはありますか？」 まあ、少なくともカードリーダー？ microSD-SDアダプターは一緒に使用しませんか？」



一般的に、今、これは過去の紳士です。 A-DATAリーダーシリーズmicroSD Trioを紹介します。



SDカードを作成して、カードリーダーなしで任意のコンピューターで読み取れるようにすることをお勧めします。 A-DATAはそれを具体化しました。 これは、内蔵USBコネクタを備えたmicroSDからSDフォーマットのアダプターです。 画像にすべてが表示されます-microSDカードがこのミニチュアカードリーダーの側面に挿入され、USBコネクタが検出される取り外し可能なキャップが下（SDコネクタの反対側）に挿入されます。



もちろん、USBコネクターはSD自体よりも少し薄く、本格的なコネクターのようにケーシングがありません。そのため、すべてのUSBコネクターと一部の埋め込み式ソケットにうまく収まりません。 、コンピューターのバネ接点によって上昇します。これにより、非常に良好な接点が存在しない可能性があります。



USBコネクタのカバーは非常にしっかりと固定されています。 microSDがカードリーダー内で脱落することを恐れる必要はありません。 一般的に、私はこの購入に非常に満足しています。 私にとっての主な利点は、書き込み中にUSBフラッシュドライブをラップトップに直接保存できることです。



EeePC 1000上のSteelBytes HD_Speed 1.5.3.64プログラムを使用して、読み取り速度と書き込み速度について、付属のmicroSDカード（4GB）でカードリーダーをテストしました。結果は次のとおりです。

SDカードとして動作する場合： 読み取り速度17Mb \ s 、 書き込み速度15Mb \ s

USBフラッシュドライブとして動作する場合：読み取り速度17Mb \ s 、書き込み速度12Mb \ s



キットの内容：

-カードリーダー自体

-USBコネクタのキャップ（すり切れている、キャップを付けたままSDコネクタに収まり、突き出ない）

-microSDカード



また、USB経由で作業しているときに青色に点灯するLEDもあります。 SDとして作業しているとき、それがオンになっているかどうかを確認することはできませんでした。 LEDは薄い白いプラスチックの下に隠れているため、目を傷つけることはほとんどありません。



広告ではない：

リバーステーションのNeoTorgで1290ルーブルのセットで4GBカードを使って、こんな小さなものを買った。 また、330（と思われる）ルーブル用の1GBカードで同じセットが見つかりました。 8GBカードのキットはありませんでした。



また、A-Data（「A-Data SD Duo」）から4GBのUSBインターフェイス（カードリーダーではない）を備えたSDカードが簡単に発見されました。 残念ながら、私は気づかなかった- スピーディまたはターボ 。 容量も4GBでしたが、8GBはそうではありませんでした。



PS>私はEeePC 1000にUSBがあるので、左側のコネクタにのみ「装着」します。 おそらく、ビルド品質、またはおそらく正しい品質が揺らいだでしょう。 また、USB経由で接続するときにドライバーをインストールし（リムーバブルディスクではなく、カードリーダーを見つけました）、再起動を要求しました。 しかし、それがなくても、すべてがうまくいきました。
