マイクロソフトは、アップグレードされたWindows 10をダウンロードポータルからリコールしました

マイクロソフトは、Windows 10ダウンロードポータルからWindows 10バージョン1511をリコールしました。メディア作成ツール（MTC）をダウンロードするためのリンクは、 こちらから入手できます 。 これにより、ユーザーは指定されたバージョンのWindows 10をインストールできます。このバージョンは、このOSのすべてのユーザーの更新プログラムとして利用できます。 これで、リンクは10.0.10240.16480という番号のWindows 10の以前のバージョンのMTCツールで利用できます。 したがって、Windows 10の新しいコピーをインストールするユーザーは、ビルド10240をダウンロードしてからOSを更新する必要があります。







このようなマイクロソフトの決定の理由は完全には明らかではありません。おそらくこれはOSの重大なバグです。 Windows IT Proのエディションによると、新しいWindows 10のユーザーは、Bitlockerドライブ暗号化機能のアクティブ化で問題が発生する可能性があります。 Windows 10バージョン1511の新しいバージョンは、既にインストールされているOSのアップデートとしてダウンロードできます。



Ars Technicaの解説では、Microsoftは、OSのこの種の重要な更新は、既にインストールされているOSの更新としてのみ利用可能であり、MCTを使用したインストールの更新ディストリビューションとしては配布しないと述べました。



11月の更新プログラムはもともとMCT（Media Creation Tool）を介して利用可能でしたが、同社は今後のインストールはWindows Updateを介して行うことを決定しました 。 必要に応じて、MCTツールを使用してWindows 10 [ビルド10240]をダウンロードできます。 11月の更新プログラムは、Windows Updateを介して配信されます。



マイクロソフトは、Windows 10 11月10日更新プログラムを撤回していません。 同社は11月の更新プログラムを徐々にリリースしています。WindowsUpdateで表示されない場合は、すぐに表示されます。


Windows 10の11月の更新は、このOSの最初の主要な更新であり、ほぼ2週間前に公開されました 。 Microsoftによると、このアップデートにより、Windows 10のロードが大幅に高速化され、特定の種類のデバイスとの互換性の問題が解消され、Edge Webブラウザーのセキュリティが向上し、OSにも多くの改善が加えられました。
