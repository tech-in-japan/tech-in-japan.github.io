ドバイでの雇用、雇用主からの眺め









以前、ドバイで仕事を探している方法について書きましたが、今では私自身が会社の従業員を探し始めました。 私の経験により、UAEの求職者がここで説明するレーキを踏まないことを願っています。 この記事はある種の啓示ではなく、応募者を見つける際の私たちの問題は一般に広まっていることをすぐに指摘したいと思います。



そのため、当社では2つの空席が開かれました.1つはPHP開発者用で、2つ目はHTML / CSSおよびPHPの知識を持つコンテンツマネージャー用です（実際、コンテンツを処理し、コードやHTMLにわずかな変更を加えるだけです）。 標準的なスキームに従って行動を開始しました。さまざまなサイトで2つの空席のアナウンスを行い、潜在的な従業員の再開を待ち始めました。



CVコレクション



1週間足らずで、両方の空席について約300人の履歴書が届きましたが、そのうち3人の履歴書はchatru.com （ドバイで最も人気のあるロシア語ポータル）からのもので、ほとんどはwww.dubizzle.comからのものでした 。

まず、応募者に関する一般データ：











ご覧のとおり、仕事に就きたいと思っている人の4分の1は女の子です。個人的にはロシアに比べて少し変わっています。











この図は、候補者の大半がインドとパキスタンから来たことを示しています。 同時に、これらの候補者の大多数の給与期待は6000ディルハム（約1,500ドル）内で変動します。



将来的には、履歴書を使用した作業は欠員によって分割されました。



PHP開発者



約120人がPHP開発者の欠員の発表に応えました。



現在、ドバイにいない従業員を雇用しないという厳しい制限がありました。そのため、送られたすべての履歴書のうち、60人しか残っていませんでした。



追加の質問とともに、各候補者に手紙が送られました。 質問の本質は重要ではありません。主なことは、どの履歴書にも雇用主にとって関心のあるすべての質問に対する答えがなかったという事実です。



  1. 完成したプロジェクト（ポートフォリオ）へのリンクのリストと、申請者がこのプロジェクトで具体的に行ったことに関するコメント。
  2. 応募者がレイアウトデザイナーまたは開発者として自分を専門的に評価する度合い。
  3. 人気のあるCMSの経験;
  4. 給与の期待と労働条件。


60人のうち、15人が追加の質問に回答し、そのうち9人のみが同じ日に回答し、4人が翌日に回答し、さらに2人が2日後に回答しました。 。

回答の研究の終わりに、6人がインタビューに招待されました。そのうちの1人は一般的な知識を経験せず、誰かが履歴書と質問への回答を書いたようです。 パキスタンからの別の候補者はひどい会話の英語であったので、彼は翻訳者と一緒に来ました、そして、彼が十分な知識を持っていたという事実にもかかわらず、我々は彼の立候補を拒否しました。 別の2人の候補者はテストの問題に対処せず、残りの2人の候補者（ロシア人とインド出身）が同じポイントを獲得し、最終会話でロシア語が選ばれました。



電子メールでの作業は作業を大幅に簡素化し、時間のコストはそれほど大きくありませんが、ご覧のとおり、「戻り」も小さいです。



HTML / CSSおよびPHPの知識を持つコンテンツマネージャー



この欠員の方法論はすでに異なっていたので、私たちは私たちに興味のある履歴書を候補者に電話し、電話で話をし、最後に、潜在的な従業員が私たちに興味を持っている場合、面接に招待されました。 約40人の候補者のうち、12人が招待され、その結果、適切な従業員が見つかりました。



求職者への電話と面接の実施には約2週間かかりましたが、欠員の要件はより少なくなりました。 将来的には、候補者と連携する同様の方法を計画していません。



応募者への簡単な推奨事項



  1. 履歴書には、雇用主に必要なすべての情報、つまり、あなたに関する簡単な情報、あなたの仕事のリスト（コメント付き）、給与の期待値を含める必要があります。 誰もあなたの趣味や豊かな内なる世界の説明に注意を払わないでしょう。
  2. 写真を履歴書に入れることをお勧めします。そうすれば、雇用主は会議であなたを認識しやすくなります。
  3. 履歴書自体に加えて、いわゆるカバーレターとも呼ばれる手紙に添付してください。これにより、雇用主は最初の選択を行いやすくなります。
  4. 雇用主に送信されるファイルの名前には、「Resume_job_28sep2014_Dubai.pdf」や「CURRICULUM_VITAE_2.pdf」ではなく、「vasya_pupkin_resume.pdf」などの名前を含める必要があります
  5. メールで連絡する場合は、すぐに返信する必要があります。そのため、翌日に面接に招待される可能性が高くなります。
  6. 雇用主の都市で直接仕事を探す方が良いでしょう。あなたは他の国からの仕事に惹かれる非常に印象的な候補者でなければなりません。
  7. 本当のスキルと知識を書いてください。現実はインタビューで明らかになりますが、あなたに対する態度は悪化します。
  8. あなたが本当に仕事をする準備ができている給与を示してください、あなたの要件を過小評価する意味はありません、ただ面接に入るだけです。 たとえば、私たちが選んだ候補者は、少なくとも彼の履歴書をより詳細に検討することを余儀なくされるよりも、平均よりも高い給与期待を示しました。
