ドキュメントと設定を別のドライブに自動的に転送します

最後のステップのみの自動ソリューションがあり、コンソールユーティリティがこれに役立ちます：Regfind.exe。 Windows Server Resource Kit Toolsに含まれています。

バッハスクリプトを使用して作業します。 次のコンテンツ：



REM

SET from_drive=c

SET from_path=Documents and Settings



REM

SET to_drive=d

SET to_path=Users



REM , %SystemDrive%

regfind.exe "%%SystemDrive%%\%from_path%" -r %to_drive%:\%to_path%



REM ,

regfind.exe "%from_drive%:\%from_path%" -r %to_drive%:\%to_path%



REM ,

regfind.exe %from_drive%:\DOCUME~1 -r %to_drive%:\%to_path%
