AppClub Kyiv {build、monetize}-モバイル愛好家や専門家向けのクラブ

最近、キエフでAppClub Kyivと呼ばれるモバイル愛好家や専門家のためのクラブを組織しました。 最初の会議は、世界でのWindows 8の世界的な発売に捧げられ、約25人の参加者が集まりました。 それ以前は、数年にわたってユーザーグループ会議を開催していたため、このアイデアはどこからともなく生まれました。







クラブの目標：モバイルの開発、プロモーション、収益化のトピックに関する専門的なコミュニケーション、およびモバイルスタートアップに自分自身について話し、ライブフィードバックを得る機会を提供する。 プラットフォームは重要ではありません。



次のようなトピックについて話し合う予定です。

  • クラウドファンディングとクラウドソーシング


Windows PhoneおよびWindows 8開発の詳細



12月20日に、2回目のAppClub Kyiv会議を開催し、Windows Phone 8およびWindows 8の開発の実際的な側面を検討し、成功したプロジェクトの開発における実際的な経験のみを共有します。



計画レポート：



  1. WP7アプリケーションをWindows Phone 8に移行しています。
  2. Windows Phoneアプリケーションのパフォーマンスを改善する。
  3. Windows 8アプリケーション開発：実際の体験。


スピーカー：



  1. Krakovetsky Alexander（DevRain Solutions）。 マイクロソフトリージョナルディレクター、マイクロソフトMVP、wp7rocks.comの著者。
  2. Galushko Mikhail（DevRain Solutions）。 MCPD by wp7rocks.com。


プログラムは少し変更できますが、それは良いことだけです。



参加は支払われますが、合理的な制限内です。 appclubkyiv.eventbrite.comで登録できます。



AppClub Kyivのイベントは、 Facebookグループ 、 Twitter 、およびサイトで監視できます。
