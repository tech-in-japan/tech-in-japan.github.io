VirtualTreeView 5.01アップデート

VirtualTreeViewは、ツリー、リスト、およびテーブルを表示するための優れた無料コンポーネントです。

このコンポーネントを使用すると、ほぼ任意にデータの出力をカスタマイズできます。タスクと想像力によってのみ制限されます。



Windows開発者向けのMySQLクエリブラウザ 、 The Bat！ 、 Smart Inspectなどの多くの製品で使用されています。

このコンポーネントは、ノードでの作業の高速性と設計のセットアップの幅広い可能性によって区別され、ユニコードをサポートします。



アップデート5.01は3日前にリリースされ、 VirtualTreeViewは以下をサポートします。

  • Delphi XE3
  • C ++ Builder XE2！


カットの下の変更の完全なリストを参照してください。



変更点のリスト：

  • RAD Studio XE3のサポートが追加されました
  • OnAfterHeaderExportイベントのアナウンスが修正され、OnBeforeHeaderExportと同じクラスのフィールドが使用されます。
  • Delphi 2007パッケージの問題を修正
  • Delphi 2009および2010パッケージは、XEおよびXE2（カード＃279）と同様に、暗黙的なビルドパッケージにもなりました。
  • 作成＃251：C ++ Builder XE2のサポートを追加
  • 修正＃274：TBaseVirtualTree.GetLastVisibleの無効な停止条件
  • Delphi 7のランタイムパッケージの修正
  • 修正＃273：国際文字のステップバイステップ検索がDelphi 2009-XE2で機能しなかった
  • いくつかのマイナーなバグ修正が含まれています。
