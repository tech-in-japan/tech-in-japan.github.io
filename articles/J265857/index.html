素晴らしい休暇が終わりに近づいています。 あなたがすべてのあなたの目標を実現し始めるために強さを獲得することができたことを願っています！ 私たちのチームも素晴らしい休息をとっており、あなたのためにさらに面白いことをする準備ができています！ しかし、来年の話に移る前に、この中にいくつの興味深いことがあったか思い出してください。



私たちはオンラインでだけでなく、個人的にも会議で多くのことを話しました。 間違いなく、過去1年間で最も印象的なイベントは、 Microsoft Developer Tourです！ ベラルーシやカザフスタンだけでなく、大国の多くの都市を訪れ、何千人もの開発者と個人的に会えたことを非常に嬉しく思います。 実際、これには非常に重要な理由がありました。 2014年10月、Microsoftは新しいオペレーティングシステムであるWindows 10と強力な開発環境Visual Studio 2015の新しいバージョンを発表しました。 #msdevtour Webサイトにアクセスして、その様子を思い出し、レポートの記録を確認してください。



定期的かつ伝統的になった今年のもう1つの重要なイベントはDevConです。 今年のカンファレンスは記念日を祝いました-5年間、5月末に美しい郊外の複合施設で開発者を集め、2日間、Microsoftプラットフォームの下で開発環境に没頭し、最新のニュースを見つけ、世界の専門家や志を同じくする人々とチャットできるようにしました楽しい時間を！ 面白い？ DevCon 2015カンファレンスのレポートの記録はこちらから入手できます 。

ハッカソンでは、耳を傾けるだけでなく、頭と手を動かすこともできます。開発者は48時間で独自のプロジェクトを作成します。 伝統によれば、最も人気があるのはゲームハッカソンです！ ロシアのゲームのテーマは特に急速に発展しており、すべての新しいプロジェクトをサポートし、ハッカソンでのエキサイティングなゲームの出現を支援しています。 CWI （Game Developers Conference）の一環としてハッカソンを開催し、友人やパートナーであるGamesJamプラットフォームと共に素晴らしいハッカソンを作りました！ ゲーム業界に特別な注意を払った結果の1つは、国際大会Imagine Cupのゲームカテゴリでロシアが2年連続で勝利したことです。 IC 2015の勝者となったOVIVOゲームを備えたIzHardチームは、ハッカソンの 1つにチームとして登場しました！

  • FB 、 VK 、 Twitter 、 Instagramの開発グループで積極的に連絡を取ります。 すべての最新ニュースをすばやく知りたい場合は、コミュニティに参加してください！ グループに質問やフィードバックを残すこともできます。
