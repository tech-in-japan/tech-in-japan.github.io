Windows XPは新しい英国の航空母艦で稼働します



空母「エリザベス女王」



全世界が唯一のロシア空母クズネツォフ提督の冒険を思いやりをもって見た後、英国艦隊も批判の原因を与えました。 財務省に35億ポンド（現在の為替レートで44億8000万ドル）の費用がかかった新しい空母クイーンエリザベス（HMSクイーンエリザベス）で、ジャーナリストは古いWindows XPオペレーティングシステムを実行しているコンピューターに気付きました 。



2017年6月26日、最も近代的なイギリスの空母が最初に外洋に出航しました。 イギリス海軍のために建造された最大かつ最も強力な船です。 空母の建設には10,000人が関与しました。 そのセクションは、英国の6つの造船所で製造され、国内最大の造船所があるRosythに納入されました。



船に向けて出発する前に、ジャーナリストをツアーに招待しました。 すべての秘密にもかかわらず、彼らは制御室に入れられさえしました。 そして、ここで、 タイムズとガーディアンの記者は、セキュリティサポートがマイクロソフトのサポートから削除されたため、セキュリティ更新プログラムが利用できなくなった古代のOSを撮影することができました（ただし、公共部門の一部のクライアントは、システムの延長サポートに対してMicrosoftに多額の費用を支払いましたが、公式のサポートは既に終了）。







事実は、船の建設がほぼ20年の間続いたということです。 どうやら、コンピューター機器とソフトウェアのインストールは2000年代初期に行われたようです。 当時、Windows XPは最も先進的なオペレーティングシステムでした。 さて、船を装備した後、彼らはすべてをそのままにすることにしました。



英国国防総省の情報筋は、 タイムズ紙に、ソフトウェアの更新が速すぎ、2004年に新しくなったように見えるものは時代遅れになったと語った。エリザベス女王のコンピューターの再装備は、2026年に打ち上げる予定の「ツイン」航空母艦「プリンスオブウェールズ」の装備と同時に行われる可能性が高いことを



Windows XPは、数年前にMicrosoftによって廃止されました。 セキュリティ更新プログラムはリリースされていませんが、最近、WannaCryの流行中に、Microsoftは例外として、暗号化ワームが拡散するSMBの脆弱性をカバーするパッチをリリースしました（ただし、後にWindows XPコンピューターは感染に対して無防備であることが判明しましたが、 「死のブルースクリーン」に該当します ）。



それでも、専門家はWindows XPを軍艦にインストールすることは非常に危険なイベントであると考えています。「XPが運用用に設計されている場合、非常に危険です」とサリー大学のコンピューターサイエンス教授であるAlan Woodward氏は言います。 「何十年もの寿命を持つ新しい船に古いシステムを設置したのはなぜですか？」



しかし、船長のマーク・デラーは心配する必要はないと考えています。 彼は、船にはサイバー攻撃のリスクをゼロに減らす非常に厳しい手順があると言います：「中国が特定の物で市場をあふれさせたいと思って、彼らがこのビジネスに3000万を投資するなら、デバイスの1つは最終的に私たちの供給システム保護に浸透できます。 私たちには、これを常にフォローしている人々がいます。」



マーク・デラーは、船上のコンピューターの分離はNASAのコンピューターの分離と比較できることを明らかにしました。 「私たちの無線周波数をdrれさせたい人については、私たちに影響を与えようとする人を追い払う護衛と駆逐艦が近くにあります。 これは海での通常のルーチンです」と船長は言った。





船長マーク・デラー



ニック・リーソン中佐は、船がハッカーの攻撃から身を守ることができると付け加えました。これは、コンピューターの専門家グループが船で作業するためです。



国防総省のマイケルファロン長官も、Windows XPを軍艦で使用することを脆弱性とは考えていません。「重要なのはシステムそのものではなく、その周辺のセキュリティ対策です」この点に関して脆弱性はありません。」



65,000トンの航空母艦は、2017年6月26日正午にドックを出て、数百メートル近くに停泊しました。 ドックだけを残すことは、この巨人にとって難しい操作でした。ドックを離れるとき、空母の底の下の深さは50 cmで、ドックの壁までの距離は約10〜20 cmでした。



6月26〜27日の夜、彼は北海への6週間の試験飛行に行きました。 船の航空機の翼には、最大24台のF-35BライトニングII戦闘爆撃機とマーリンヘリコプターが含まれます。 空母の甲板からのヘリコプター飛行は2017年に開始され、F-35B飛行は2018年末に開始されます。 2020年には、艦艇の完全な戦闘準備が期待されます。 イギリス海軍の代表は、2020年までにWindows XPが確実にアップグレードされることを約束しました。



ジャーナリストは、すべてがそれほど悪いわけではないと言う。 最終的に、空母はパリ国際空港のようにWindows 3.1に耐えることができました 。







おそらく、Windows 3.1は現代のウイルスを恐れておらず、WannaCryはそれを受け入れません。
