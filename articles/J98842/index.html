オープンソースの電子日記システム：1年後

1年前、私のシステム の道筋はHabréで始まりました。学校でのコンピュータ化を簡単に開始できるように設計されています。 リンクをたどって読むのが面倒な人のために：学校向けの電子日記の禁欲的なオープンソースシステムであるDjangoで書かれています。 この投稿以降、多くの変更が行われました。

私は、この時期の楽しいイベントやそれほど多くないイベントをカテゴリに分類しようとしました。



技術





この間に発生した最も重要な変更は、ライセンスの変更です。 私はGNU GPL v3を放棄して、新しいBSDライセンスを支持しました。これは開発のレッドフラッグになりました。 同じ夜、 IKA GROUPから連絡があり、 E-Schoolプロジェクトとの統合を提案しました。 この協力のおかげで、素敵なインターフェースと多くの新機能がシステムに登場しました（すべてIKA GROUPSによるものです）。 それ以来、もしあなたの学校が私のシステムで動作していれば、いつでもターンスタイル、電子図書館、ビュッフェなど、すぐに開発される他のシステムと統合することができます。

IKA GROUPは、他のソフトウェアおよびハードウェアソリューションとの無料統合を可能にするREST API （JSONはデータ転送の標準として使用されます）の作成を奨励​​しました。 保護者への配布用の成績、サードパーティアプリケーションの設定、管理者には成績を設定する機会、学生にはプロフィール写真「写真」、ウクライナでのシステムを使用する機会（12ポイントシステム）、システム、課金の実装のためのベースが作成され、SMS通知の可能性が現れ、テストシステムの開発が開始されました。

学校の保護者、校長、教師とコミュニケーションをとるとき、評価付きのSMS通知について質問がありました。 この問題は、特別なゲートウェイの開発によって最近解決されました。 実装スキームは次のとおりです。

  • 電子日記システムのコピーからメッセージを受信し、それを保護者にブロードキャストする特別なSMSゲートがあります。 デフォルトでは、ゲートウェイを使用できますが、サードパーティに変更できるのは、それが仕様を満たしている場合のみです。 （最初にゲートにシステムのコピーを登録する必要があります）
  • 生徒のプロフィールには、保護者の電話番号が表示されます。これは、ゲートにも送信されます。
  • 親は便利な方法でSMSを支払います（公式のゲートウェイでは、SMS経由の支払いのみが実装されています）
  • それ以降、親はSMSで子供の冒険に関する通知を受け取ります。


読んでいる時に、SMSマーケティングのために非常に有利なオファーを私に提供したい人は、心配しないでください。 公式ゲートウェイはAvisoSMSを介してSMSを送受信しますが、これは非常に満足しており、拒否するつもりはありません。



社会



このプロジェクトは全ロシアのオリンピック「 コンステレーション 」に参加し、指名「教育システム」で3位になりました。対戦相手の不足。 まあ、彼らと地獄に。

このシステムやその他のメリットについては、優雅な州から私が「 Breakthrough 2010 Winners Forum 」に招待されました。 そこで、システムの作成者として、Rosmolodezhとの情報サポートに関する契約に署名しました。 それから6か月が経ち、この本当に役立つ組織から受け取ったのは、私と私の家族の財務状況についてのアンケートに記入するよう求める2つの手紙だけです。



使用場所



このシステムは、A.S。プーシキンにちなんで名付けられたブリャンスク市のリセウムNo. 1で部分的に使用されています。 残念ながら、管理の慣性により、システムを完全に実装することはできません。

最初に呼び出された聖アンドリューにちなんで名付けられた士官候補生海軍隊では、私のシステムは回転式改札口と共に使用されます。 ニュースレポートをお楽しみください。



ルハンスク地方のクラスノドン市のすべての学校で、残念ながら、私の電子日記システムは何とも統合されずに導入されています。 残念ながら、このプロジェクトにはウェブサイトがありませんが、近代化の責任者向けのブログがあります。

また、このシステムは、ブリャンスク市の62の学校でゆっくりと重く実装されています。



まとめ



学生のテストプロジェクトの需要が非常に高いことに驚きました。 学校長が提供されたときに決定を下すことができることの遅さと遅さに非常にd然としている、悪いマナー、景品を申し訳ありません。

システムにはまだロゴとまともなサイトがありません-残念ながら、1年間、利他主義のデザイナーはいませんでした（ええ、下品なヒント）。

ご清聴ありがとうございました。 協力を提供したい場合、私は非常にうれしいです。
