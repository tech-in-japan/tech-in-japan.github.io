HTML5 File APIに感謝するか、ID3タグを読み取り、MP3ファイルをアップロードせずにフォームに入力します



HTML5の登場により、多くの新しい興味深い機能が追加されました。 さらに優れたアプリケーションを作成できます。

  • タグから音声ファイルの形式を入力します
  • EXIF写真フォームに記入します
