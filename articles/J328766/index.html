開発者向けのIOS、Android、およびMacアプリ

この記事では、iOS、Android、およびMacの公式市場で利用可能な開発者の多様な機能ツールを紹介します。これにより、初心者のコーダーと経験豊富な開発者の両方が楽になります。 おそらく、私たちのリストには、何らかの理由で聞いたことのない興味深いユーティリティがあり、それを使用するのに役立つでしょう。









iOS









CodeHubは、GitHubでプロジェクトを監視し、プロジェクトに関する情報を表示し、リポジトリを作成し、コードエディターで変更を加えるための便利な方法です。 このアプリケーションはGitHub.comおよびGitHub Enterprise（PROバージョン）のサービスをサポートしており、プロファイルの切り替え、携帯電話からの画像のアップロード、コメントの投稿などができます。 視覚認識の構文を改善するために、強調表示が実装されています。



これはオープンソースプロジェクトであり、そのソースはこちらにあります 。









Googleのプロジェクト-Fabric-を使用すると、電話で直接アプリケーションを監視できます。クラッシュに関するアラートとアクティブユーザーに関する詳細な分析データをリアルタイムで受信し、常に把握することができます。 問題が発生すると、問題を報告するプッシュ通知が送信されます。



この製品を使用すると、考えられる障害の原因に関する情報、OSバージョンの使用に関する統計、およびその他の多くの有用なデータも受信できます。 もちろん、FabricはGoogle Playマーケットでダウンロードできます。









CppCodeは、組み込みのlibcおよびlibc ++を備えた安全なCおよびC ++コンパイラであり、サードパーティのライブラリを必要としません。 アプリケーションには、インテリジェントな構文強調表示、組み込みのファイルナビゲーター、C / C ++コード例もあります。 コードの作成中にエラーが発生した場合、アプリケーションはこれをユーザーに通知します。これは、トレーニングに製品を使用するユーザーにとって非常に便利です。 CppCodeがオフラインで動作することは注目に値します。









Appleが推奨するCode Viewerは、最高のコード読み取りアプリの1つです。 このアプリケーションは、Swift、C、C ++、JS、ObjC、C＃、HTML、XML、Java、Python、Rubyなどの60以上のプログラミング言語をサポートしています。 また、コードビューアーを使用すると、Safariブラウザーまたは別のアプリケーションから直接、新しいコードまたは新しいファイルを簡単に追加できます。









InVisionアプリケーションは、設計プロジェクトでのコラボレーションの整理に適しています。これにより、ユーザーはプロジェクトのステータスを追跡し、同僚と作業上の問題について話し合うことができます。 アプリケーション開発者は、すべてのプロトタイプのオフラインサポート、プロジェクトにコメントするための便利なダイアログ、クイックユーザーテスト、およびその他の同等に役立つ機能を提供します。



Android









CppDroid -C / C ++ IDE-Android開発環境 。その内部で、詳細なガイドを使用して、CまたはC ++コードをコンパイルし、リアルタイム診断を行い、これらの言語でプログラミングを学習できます。 アプリケーションには、構文の強調表示がカスタム化されており、関数の使用と言語の学習が可能な限り効率的になります。









モバイルC（C / C ++コンパイラ）は、初心者のコーダーに特に役立つコードエディターです。 アプリケーションは、C、C ++、Python、Javascript、Luaをサポートし、標準Cライブラリを含み、C / C ++構文診断も提供します。









Anacode IDE Android / C / C ++ / JAVAアプリケーションを使用すると、Android、C / C ++およびJavaプロジェクトをオフラインでコンピューターなしで作成できます。 これは、JAVA、HTML、CSS、PHP、JavaScript、およびC / C ++ファイル用の統合開発環境およびソースコードエディターであり、モバイルデバイスで迅速かつ簡単にコードを記述および編集できます。









モバイルコンパイラIDEであるDcoderは、 25を超えるプログラミング言語で動作する、非常に活気のある構文強調コードコンパイラです。 それを使用して、プログラムを作成し、コードをデバッグし、また、自分に都合の良い場所でコーディングを行うことにより、特定の言語の知識を補充することができます。









Show Javaは、Androidアプリケーション（XMLファイルと画像を含む）からソースコードを抽出し、逆コンパイルされた各ソースファイルにコメントを提供する逆コンパイラーです。 このアプリケーションの特徴は、すべてのAndroidデバイス（シングルコアプロセッサを搭載したデバイスを除く）での使いやすさと高速操作です。 バックグラウンドで動作することができます。



Mac OS X









開発者および設計者向けの直感的なユーザーインターフェイスを備えた汎用性の高いCodeBoxコードスニペットマネージャー 。 このアプリケーションは、タグ、フォルダー、グループ、スマートグループを使用したフラグメントライブラリの管理を提供し、メニューバーからアクティブなアプリケーションに直接フラグメントを編集、コピー、または貼り付けできます。









SnippetsLabは、より効率的で高速なコード処理と簡単なアクセスを実現する別のMacフラグメントマネージャーです。 アプリケーションはiCloudによってサポートされており、これにより、複数のデバイスでデータベースを同期できます。 また、CodeBoxライブラリファイルからインポートします。









Sketchode 2は、設計プロジェクトの要素にすばやくアクセスするために特別に作成された、デザイナーと開発者の間のコラボレーションツールです。 要素の検索は、色、テキストスタイル、画像で可能です。 開発者はそれぞれの詳細情報を受け取ります。 プロジェクトのアンロードは、とりわけJiraサービスを使用して実行されます。









iHexeは、一般的なファイル情報を表示するインテリジェントなHexエディターです。 その大きなプラスは、サイズが最大118 GBのかさばるファイルを操作できる一方で、速度が低下せず、デバイスのメモリが詰まらないことです。









Toadツールキットは、OS X開発者に役立つデータベース管理を簡素化し、Oracle、MySQL、PostgreSQL、MongoDBデータベースに接続して、ユーザーにプラットフォームサポートを提供します。
