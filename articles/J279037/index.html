Tasktopがロシアで正式に発表されました

私たちの新しいパートナーであるカナダ企業Tasktopを紹介させてください。そのソフトウェア製品は現在、ロシアで正式に代表、実装、サービスしています。 Tasktopは、開発環境、バージョン管理システム、自動テストシステム、バグトラッカー、サービスデスクオートメーションシステムなど、何らかの形でソフトウェア開発に関連するシステムの統合を専門としています。これは非常にニッチなビジネスですが、より興味深いものです。 さまざまな絶対に必要なIT製品の多彩なモザイクが時間とともに発展し、それらを完全なソリューションに結合する必要がある場合はいつでも、Tasktopソリューションが役立ちます。



厳密に言えば、Tasktop Sync、Tasktop Data、およびTasktop Devの3つの製品があります。







タスクトップ同期


Tasktop Syncは、個別のサーバー同期サービス、さまざまなシステムへの一連のインターフェイス、これらのインターフェイスの構成環境、およびレポートモジュールで構成されるプロセスおよびデータ同期ソリューションです。 たとえば、HP QCからJiraおよびSBMまでのテスターに​​よって検出および記録された欠陥を開発者が見つけてから、パッチステータスを自動的に更新するようにします。 実際には、アーキテクチャ、ストレージ構造、API、およびさまざまなタイプのデータ、ユーザープロファイル、ディレクトリ、プロジェクトのマッピングなど、さまざまなメーカーのさまざまな製品の目的を考慮して、これらのインターフェイスの構成環境の実装にハイライト全体が隠されています。



サポートされている統合のリストには、多くの人気のある有名な製品があります。
Atlassian JIRA、Atlassian Service Desk、ブループリント、BMC Remedy、Borland StarTeam、Bugzilla、CA、Agile Central、CA Agile Planning、CA Harvest、CA PPM、GitHub Issues、HPE ALM、HPE PPM、HPE Quality Center、IBM Bluemix、IBM Rational ClearQuest、IBM Rational DOORS、IBM Rational DOORS Next Generation、IBM Rational Quality Manager、IBM Rational Requisite Pro、IBM、Rational Team Concert、iRise、Jama、Microsoft Project Server、Microsoft Team Foundation Server、Microsoft Test Manager、Microsoft Visual Studio Team Services （VSO）、Planview Enterprise、Polarion ALM、Rally、SD Elements、Serena Business Manager、Serena Dimensions Requirements Management、ServiceNow、ServiceNow PPM、ServiceNow SDLC、ThoughtWorks Mingle、Tricentis Tosca、VersionOne、WhiteHat、Zendesk



このリストには、多くのサービス自動化システムと開発管理システムが含まれているという事実に注目してください。 これは偶然ではないように思われますが、アプリケーションシステム開発者が修正を行うインシデントや問題を明確に管理するというビジネスの特定のニーズへの答えです。 2つの世界の相互作用で考えられるもう1つのシナリオは、構成管理とリリース管理、またはITIL変更管理プロセスによる新しいソフトウェアバージョンのインストールです。



このような統合の問題を解決するとき、Tasktop Syncなしで実行できますか？ もちろん、はい。 Tasktopの提案の興味深いところは何ですか？



  1. 汎用性と均一性。 1つのソリューションで、開発ツールと管理ツールを統合するすべてのニーズを迅速にカバーできます。
  2. 決定のスピード。 サポートされている統合のリストで製品を見つけ、レビューとテストのためにバージョンをダウンロードし、同期ブリッジをセットアップして結果を評価します。
  3. 製品のさまざまなバージョンのサポート。 使用する開発ツールの新しいバージョンは常にリリースされています。 情報の同期が機能し続けるためには、データ交換プロセスに影響を与える可能性のある変更を常に監視する必要があります。 そして、これはTasktopが解決するもう1つの不幸です。
  4. 製品を時間をかけて同期する必要がなくなることがあります。 Tasktopソリューションはサブスクリプションで販売されており、急速に変化するITビジネスの状況に非常に便利です。


タスクトップデータ


Tasktop Dataは、ソフトウェアの開発、構築、インストール、およびメンテナンスに使用されるさまざまなツールから情報を収集するためのソリューションです。 Tasktop Syncと同様に、管理者はデータ収集ブリッジをセットアップしますが、データは別のシステムに転送されず、中間ストレージ構造に集約および保存されるだけなので、サードパーティのレポート分析ツールを使用して、必要なメトリックを計算し、構築できます必要なレポート、さまざまなレベルのマネージャー向けの情報インジケーターのパネルに情報を公開します。



Tasktop dev


Tasktop Devは、かなり人気のあるオープンソースプロジェクトEclipse Mylynの商用バージョンです。



Eclipse Mylynは、タスクを管理し、この情報とアクションを組み合わせてリビジョンをバージョン管理システムに保存するためのコンテキストを提供することにより、Eclipse環境の開発者の間で人気を博しています。



Tasktop Devには、一般的な商用バージョン管理システムと、タスクの実行時間を制御するための追加機能などのサポートが含まれています。 Dev Proの改訂には、Outlook、Zendesk、Google Apps、VersionOne（完全なリストはwww.tasktop.com/tasktop-dev ）、デスクトップにインストールするための別のアプリケーション、ブラウザーでドキュメントを表示するためのサービスとの統合パッケージも含まれます。



Proに加えてDev Enterpriseエディションには、Microsoft VisualStudio IDEのサポートが含まれていますが、さらに重要なのは、Microsoft TFSリポジトリです。 JiraまたはSBMの開発者のタスクをMicrosoft TFSタスク管理オブジェクト（バグ、タスク、アイテム）に同期する必要がある場合、TaskTop Devはこれを簡単に行うことができます。



ちなみに、Tasktop Dev ProはSBMパッケージに無料で含まれています。
