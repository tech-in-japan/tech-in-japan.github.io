Yandex Data Analysis Schoolの10年

今日は、 Yandex Data Analysis School の 10周年を迎えます。 9年前に入社し、7年前に卒業し、同じ2010年の7月21日にYandexの従業員になりました。



それ以来、私、Yandex、ShADのすべてが大きく変わりました。 しかし、私は学校の壁から学んだいくつかの教訓がありますが、それは今でも自分に関連していることが判明しており、そうなることはほとんどありません。





当初、私はこの投稿を個人的なメディアチャンネルで作成しましたが、その後、同僚はHabrと共有することが理にかなっていると確信しました。 おそらく、このテキストはあまりにも大げさであることが判明しましたが、ちょっとした大げさなしでは機能しないという非常に奇妙なことです。



限られた知識



私は、応用数学科MPEIの3年目を終えたばかりで、学校に入りました。 実際、学校の創作者は卒業生-学士、つまり4年生を期待していると言われていました。 だから私はそれをかなり危険にさらした。



口頭試験でのインタビューの委員会は印象的でした：イリヤ・ボリソヴィッチ・ムニク、 maxim_babenko 、レナ・ブニーナ。 イリヤ・ボリソヴィッチは、確率論について、そして数学的統計について質問をしました。 そして、統計は4年目に開催されたため、まったく知りませんでした。



それから私はそれについて何も知らず、確かに単一の質問に答えないだろうと認めました。 最終的にイリヤ・ボリソヴィッチが彼の学問分野で私のパフォーマンスを非常に高く評価したのは予想外でした。 これらの分野のいずれかをまったく知らなかったので、それは非常に珍しいことでした。 少なくとも、このような評価は、学校や大学で私がよく知っている通常の基準に適合しませんでした。



それから、これが非常に論理的であることに気付きました。 委員会は私の知識の限界をすぐに理解することができました。 また、この制限を克服するための理解可能な方法があったことも重要です（4番目のコースに進み、すべてを調べてください）。 あなたの知識の境界を知らないことは、単に何かを知らないことよりもはるかに悪いです。





マキシム・バベンコ、エレナ・ブニーナ、アルカディ・ヴォロズ、イリヤ・ムニク、グリゴリー・コンダコフ


人は自分の知識の限界に気づくと、自分が無能であるということと決定すべきでないことを理解します。 自分の意識の限界を押し広げるために、どの方向に動くべきかを知っている。 不当な自信によって混乱する可能性のある人々からの時間はかかりません。



知的誠実さが主なものです。 そして、イリヤ・ボリソヴィッチは私にこれを教えた最初の人でした。



人



C ++クラスの最初の1時間で10個の問題のうち5個を解決したとき、私は自分自身を非常に誇りに思っていたことを覚えています。 私の友人の何人かは、私と同時に観客を去り、彼らの能力について非常に懐疑的でした。 その後、私は彼らが5人、7人を決定したことを知りました。 そして、例えば、ジェンヤ・レミゾフは、10人全員を決定しました。



自分を最高で、賢く、クールであると考えるのは悪い習慣です。 自分が成功したと考えてください。 特に、そのような考えは、20歳の若者に害を及ぼします。若者は、方法を知らないだけでなく、これを検証する機会もありません。 そのような錯覚は、それが何かにつながる可能性がある場合にのみ、劣化につながります。



高校では、自分を非常に賢い人と考え、地区大会での失敗したパフォーマンスを時折（1年に1度）偶然の出来事に帰すのは簡単です。 研究所のクラスも、この世界観の何かを常に変えることはできません。





2016年のシャッド学生


そのような快適な感覚から出て、あなたと同じことをすることができますが、自分自身をスーパープロフェッショナルとは考えない人々に会うことはさらに驚くべきことです。 さらに多くのことができると同時に、非常に批判的に自分自身を取ることができる人々に会うことはさらに驚くべきことです。 それは、まったく新しい、なじみのないタイプの思考でした。



また、同校の同級生の多くが機械工学の卒業生、または例えば科学の候補者だったことにも興味がありました。 もちろん、このような素晴らしい人々と研究に参加することは無意味でした。彼らは単に私よりも優れていて、これはトレーニングの全期間を通して続きました。 これはすでに完全に新しい感覚であることが判明しました。文字通りすべての面であなたより優れている人々の隣で勉強し、同時にすべてが正常であり、さらに素晴らしいことを理解することです。



人は、自分よりも多くのことを知っている人の隣にいるときだけ、本当に学ぶことがわかった。 近くのより上品な人がenましいまたは悲しむ理由ではなく、開発に資する唯一の可能な状況であることが判明しました。 間違いを犯すことは正常です。 知らないというのは普通のことです。なぜなら、世界には、すべてを認識することは人間にとって不可能なほどのさまざまな知識があるからです。 学校には、生徒と教師の両方から学ぶことができる多くの人々がいることは素晴らしいことです。



このような近隣地域のもう1つの大きな効果は、タスクを処理できるという理解です。 同じことをしているが管理している人々を直接観察します。 その結果、いつかあなたが管理できるという希望があります。



雰囲気



すでに大学での勉強と仕事を組み合わせている若者がShADに来て、週に3〜4晩（9〜12時間！）を別のクラスに割くと、さらに、宿題にどれだけの時間を費やすかわからない-これはそれ自体で戸惑いを引き起こします。 なぜ彼らはSHADに行くべきなのでしょうか？ 彼らはすでに中等度の名門大学で勉強しており、もちろん、将来の生活と職業活動を成功させるために必要な知識とスキルをすべて与えていますよね？ そして追加の知識-なぜ必要なのでしょうか？ 結局のところ、これは、実践するプログラマーがまったく使用しない、理解できない不必要な分野の余分なエネルギーの無駄遣いです。



驚くべきことに、SHADに来る人々は、そのような考えから可能な限り離れています。 習得した知識の直接的な利益に惹かれません。 彼らは未知に惹かれます。

一定の知的調子で自分を維持する唯一の可能な方法は、常に知識を疑うことです。 新しいことを学び、新しい分野で自分自身を試して、あなたよりも良いことを理解している人と知り合うことができます。 人がまだ強くなく、彼の人生と世界観を突然変えることができる新しい挑戦と新しい領域を探すために常に、そして彼と他の人々を通して-これはすべての偉大な人々が行く方法です。

たとえば、まだ学ぶべきことがたくさんあるというフレーズを期待する可能性が高かった。 しかし、それはまさに聞こえたとおりに聞こえました。 学校を卒業したばかりの人が、すでに会社の壁にいる長老たちに何かを教えることができるという考えは、私にとって非常に革命的でした。
