米国議会はパテントトロール法を可決





「2013年イノベーション法」法案は、米国議会の下院である下院を通過しました。 両方の党が彼に投票し、合計得点は325-91でした 。 このような全会一致は理解できます。最近、中小企業の所有者、ホテル、レストランから取締役会を打ち負かそうとするパテントトロールに関する話は、大衆の激しい抗議を引き起こしました。



重要なことは、議会は法案の修正を首尾よく阻止し、法案から重要な点を排除しようとしたことです。



最も白熱した議論は、「敗者の支払い」の原則に関する法廷手数料の移転に関する条項を法案から除外する改正を中心に展開されました。 事実は、米国民主党は伝統的にそのような原則に反対しており、非常に小さなケースでその不正を説明している。 実際、100ドルの罰金と5,000ドルの訴訟費用を支払うことは、どういうわけか奇妙です。 したがって、民主党はこの修正案174-18に賛成票を投じました。



パテントトロールの場合、不正の問題はなく、通常、ここでのプロセスは100ドルに限定されません。 ですから、25-195年の修正をブロックした共和党員に感謝する必要があります。 この修正の採択の全体的な結果は199-213です。



別の修正案は、法案の別の重要な点-技術ユーザーに対する訴訟を遅らせる原則-を弱めようとしました。 共和党員は、この原則の効果を、収入が最大2,500万ドルの企業にのみ制限しようとしました。



結局、議会は民主党のジョン・コンヤーズによる別の修正案を阻止しました。民主党のジョン・コンヤーズは法案を削減し、透明性に関する条項のみを残そうとしました。 知的財産の所有者によるロビー活動は失敗した可能性があります：157-258。



その結果、議会は1件の修正案のみを可決しました。これにより、司法実務ではめったに使用されなかった古い規則が法案から削除されます。



法律「2013年イノベーション法」は、いくつかの方法で最も悪質なトローリングを停止することを目的としています。



1. 詳細情報 。 訴訟を提起する場合、必須の要件は、どの特許が侵害されており、どの特定の製品がこれらの特許をどのように侵害しているかに関する情報を提供することです。 この規則により、トロールは大量の請求を提出することが難しくなります。被告ごとに個別の文書を準備する必要があります。



2. 訴訟費用の換算 。 裁判所は、勝者が被った法的費用を補償するために敗者から要求する権利を受け取ります。 これは、トロールが被害者を脅迫することに成功した重要なポイントです。彼らは、法廷で勝ったとしても、トロールが要求する以上の金額を支払うことを理解しています。



3. 透明性 。 法律は、原告が自らの利益のために漏らすことを厳しく要求している。 今日、ほとんどのトロールは特定の特許の下で作成された1日のアフィリエイトです。 会社が子会社である場合、裁判所の敗北後の1日の会社が破産を宣言し、費用の補償を拒否した場合、被告は法的費用について被告に補償するために親会社を裁判所に連れて行く権利を有します。



4. テクノロジーユーザーに対する遅延事例 。 法律は、この製品の製造元が試用版を自社に譲渡することに同意した場合、シリアル製品を使用する企業に対する訴訟を一時停止することを裁判所に義務付けています。 たとえば、2011年には、「公共の場所でWiFiルーターを使用する」ために、数百のホテル、カフェ、レストランがパテントトロールInnovatio IP Ventures、LLCに2,300ドルから5,000ドルを支払いました。 これは、トロールが製造業者に対する大規模な訴訟ではなく、テクノロジーユーザーに対して数百件の小さな請求を提出するための新しい戦術を適用した最初のケ​​ースの1つでした（チャンスはありませんが、ここでは少なくとも10万ペニーを回収できます）。



5. 裁判所への文書の義務的開示の改革 。 現行法の下では、原告は、被告がプロセスへの参加に同意し、裁判所手数料およびその他の費用（多くはリスクを冒さない）を支払うまで、特許番号を指定せず、他の情報を提供しない権利を有します。 特許に関する情報が最初から発見された場合、トロールのばかげた主張がすぐに明らかになる可能性があります。



6. 発行後の検査 。 法律は、すでに付与された米国特許に異議を唱える能力を拡大しています。



電子フロンティア基金は法案を承認し、これまでにリリースされたトロルに対する最高の法案と呼びました 。
