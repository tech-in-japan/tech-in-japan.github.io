FBIはロシアのチップオフプログラマーMicrosoftを拘束した

シアトルで昨日、FBIのエージェントは、Microsoftの企業秘密を漏らした疑いでロシア人プログラマーのアレクセイ・キブカロを拘束した 。



モスクワ州立大学の卒業生で、科学の候補者であるAlexey Kibkalo（ akibkalo ）は、マイクロソフトで7年以上（ 2005年5月から2012年9月まで ）勤務し、最近、MEA地域（80か国以上）で仮想化テクノロジーとプロジェクトを担当する上級アーキテクトを務めました。 彼は技術会議で繰り返し話をしており、ITコミュニティでよく知られています。 彼は、世界で最も認定された10人のマイクロソフトスペシャリストの1人です。 彼は、Microsoft Learningとともに多くのコースと試験の開発に参加しました。 Microsoft Pressの書籍の共著者。 TechNetで最も人気のあるロシア語のブログの1つの創設者および共著者。



2013年、Alexey KibkaloはMicrosoftを5nineに去り、Microsoft Hyper-Vの管理およびセキュリティ製品の開発を調整しました。



Kibkaloは、このOSの公式リリースのずっと前に、独自のActivation Server Software Development Kitや内部Windows 8ビルドなど、名前のないフランスのブロガーに公開するために「Microsoft企業秘密」を譲渡したと非難されています。



Microsoftの関係者はFBIに、Alexeyの活動は「コードリバースエンジニアリングに貢献できたはず」であり、Windows認証メカニズムを迂回したと述べました。



彼らはブロガーに対して訴訟を起こすことはありません。なぜなら彼はパブリックドメインで実質的にスクリーンショットしか公開しておらず、アレクセイを「違法」行動から落胆させたようです。 2012年9月、何らかの理由でブロガーが別のマイクロソフトの従業員に、真正性を確認するためのリクエストとともにActivation Server SDKコードを送信しましたが、ハッシュを送信するだけで問題が発生しました。 彼はすぐにマイクロソフトセキュリティサービスに連絡し、調査を開始しました。 Hotmailでブロガーのメールボックスを調べたところ、Windows 8のホットフィックスへのリンクが記載されたAlexei Kibkaloからの手紙を見つけました。このOSのリリースのずっと前に、噂だけがありました。



サーバーからログを取得することにより、MicrosoftはKibkaloとブロガー間のIMメッセージの履歴を復元することができました。 通信には、プログラマーを起訴するのに十分な妥協的な証拠がありました。 さらに、Aleksey自身は、Microsoftの調査官との会話の中で、いくつかの企業プログラムとドキュメントを「側に」転送していることを認めたと言われています。 2012年9月、彼は辞職するか、Microsoftから解雇されました。 2013年半ばに、この事件はFBIに移管され、現在、歴史は完全に不愉快な方向に進んでいます。



アレクセイは、企業秘密を漏らすことに加えて、MicrosoftキャンパスのBuilding 9の領土に違法に侵入し、サーバーのコンテンツをコピーしようとする罪に問われています。



キブカロに対する調査はおそらく秘密に行われた。 彼の最新のTechNet公式ブログ投稿はほんの1週間前でした。 アレクシーは仕事に真剣に取り組んでおり、調査が進行中であるとは思わなかったようです。
