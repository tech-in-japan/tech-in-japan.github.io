外国人同僚とのコミュニケーションのための5人のスモールトーク技術者

すべての英語学習者は、天気について話すことについて教科書のジョークを聞いたことがあるはずです。 私たちの文化では、本質的にはささいな話である空っぽの話はあまり受け入れられません。 ちょっとした話-もう一つ。 しかし、完全な会話になる前に、会話を打ち切る必要があります。 また、海外旅行をする場合は、店への旅行でさえ、ささいな話なしにはできないという事実に、心の準備をする必要があります。 何かについて、または抽象的なトピックについて話すことが難しい場合は、この記事が役立ちます。







無についての対話、拘束力のない会話、抽象的なトピックに関するリラックスした会話：これらの小さな話の定義はすべてある程度真実です。 しかし、メンタリティによって、突然、見知らぬ人に挨拶する理由を突然与えられず、今日の天気がひどいことに気付かないことがあります。 本当にそこにあるのは、見知らぬ人とのコミュニケーションが疑わしいものであると考えられるとき、私たちは丁寧な笑顔を持っていることです。 これは、私たちがすべて悲観的で社交的でないことを意味するのではなく、不誠実な笑顔や空っぽの会話が好きではありません。 しかし、海外旅行や職場での外国人との密接なコミュニケーションがある場合にのみ、何かについて話すことを学ぶ必要があります。



ところで、興味を示し、正しい方向に人を向ける能力は、あなたがたくさん自分で話すことからあなたを救うことができます、そしてあなたがまだあなたの英語について非常に恥ずかしければ、小さな話はあなたがいくつかのフレーズを言うのに役立ちます（少しだけ！）そしてそれを聞く（常に！）有用）。



テクニック1：エコーの質問



ここではすべてが非常に簡単です。 あなたは彼らがあなたに言っていることを聞いてから、助動詞/法動詞または動詞の助けを借りて質問します 。 次の例を使用してこの手法を検討してください。

-家族と私はカンボジアでの休暇から戻ってきました。

- あなたは？


この場合、最初のフレーズで補助動詞が聞こえるため、問題はありません。 あなたの代名詞youでそれを繰り返すだけで、ロシア語の「Really？」、「Really？」に似た質問ができます。

-彼らは先週出張に行きました。

-彼らはしましたか？


ここでは、明示的な補助動詞はありませんが、現在シンプルまたは過去シンプルに文がある場合、補助動詞はdoであることがわかります 。 正しい形状に配置するだけです。

-彼女はわずか5歳ですが、すでに2つの言語を話すことができます。

-彼女はできますか？


モーダル動詞は補助を必要とせず、それ自体が補助であることを忘れないでください。



イントネーションも覚えておいてください。 すべてのエコー質問には、上方へのイントネーションが必要です。これは、詳細を聞きたいことを対話者に示しています。



テクニック2：エコーワード



エコー質問を使用して対話者の言葉に興味を示すことを学んだら、2番目のテクニックであるエコー言葉を習得し始めることができます。 この手法には同じ目標がありますが、ポイントです。 エコー質問が文全体に対して行われた場合、ここでキーワードに耳を傾け、対話者が詳細に伝えることができる瞬間を選択する必要があります。



エコーワードは、昇順のイントネーションを含む質問の形式と、降順の感嘆符の形式の両方で繰り返すことができます。 前者の場合は追加情報を受け取り、後者の場合は興味や共感を示し、驚きや喜びを分かち合います。

-私は1年間ラテンアメリカを旅しました 。

- 一年？


または

-私は一年中南米を旅行しました。

- ラテンアメリカ？


この例では、場所と期間の両方を選択できます。 申し出を聞いて、上向きのイントネーションでキーワードを繰り返してください。対談者はおそらく、どの国に行ったのか、なぜ長い間旅行したのかを教えてくれるでしょう。

-ここに着くまでに2時間かかりました。

-2時間！


また、この例では、対話者は共感することしかできないため、時間間隔を繰り返すことで理解を示すことができます。



テクニック3：興味を表すための間投詞と表現



単一の言語で間投詞なしにできることはありません。また、各言語には、会話をサポートするために設計された独自の表現があります。 確かに、それはすべて会話の形式に依存します。



英語の古典的なロシア語の「はい」はあまり使用されず、「本当に」という言葉は何でも表現できます。好奇心から脅威まで。 アドバイスは1つだけです。もっと聞く。 テレビ番組、映画、ポッドキャスト、YouTube番組は、イントネーションに慣れ、どの場合に1つまたは別の表現を使用するのが適切かを理解するのに役立ちます。 そして、ダイアログは次のようになります。

-それで、昨日、救急車についてそのショーを見ていました。

-そうです。

-3つの非常に興味深いケースと1つのデマ電話がありました。

- そうそう？

-ええ、ケースはすべて異なっていました。 自動車事故とその犠牲者2人、脳卒中の患者と発熱のある少女がいました。

- なんてひどい！

-そう思われるかもしれませんが、これらの機関の仕事を内側から見るのは非常に興味深いです。

- ほんと？

-はい、コントローラーが実際にどのように呼び出しに応答し、それを乗組員に分配するのか決してわかりません。

- まあ、それは奇妙に聞こえます。


ところで、このダイアログの最後のフレーズに注意してください。 通常、私たちは、助動詞が否定と疑問でのみ生じると教えられますが、これは完全に真実ではありません。 動詞の意味を高めるために肯定文に表示される場合があります。 この場合、「本当に」、「そして実際に」としてロシア語に翻訳されます。 そのような場合、意味動詞の末尾の-sが消えることを覚えておいてください。



テクニック4：特別な問題



質問を正しく行うことは、会話を続けるための優れたトリックです。 この段階で、既に学習したすべてのテクニックに干渉し、質問を追加して会話がより生き生きと聞こえるようにすることができます。 念のため、特別な質問とは何か、質問する方法を覚えておいてください。



特別な質問はしばしばWh-questionsと呼ばれます。これは、これらの質問の大部分がwhat、when、where、why、whoの言葉で始まるためです。 同じカテゴリには、どのように頻度、数、量、距離など、単語howを含むすべての質問が含まれます。 これらの質問は、追加や状況について尋ねられます。 特別な問題のスキームは同じです：



Wh-word / How->助動詞->主語->意味動詞->残りの文のメンバー



例：

今夜どこで彼に会いますか？ 助けてくれませんか？


これとは別に、質問語whoを使用して、主題に質問を表示できます。 同じwh-wordのように見えますが、文法を覚えておいてください。現在のシンプルと過去のシンプルには、主題に対する質問に補助動詞はありません。



例：

誰があなたと一緒に行きましたか？ 彼の代わりに誰が任命されますか？


ちょっとした話で特別な質問がどのように役立つかを見るために、最初の例に戻って、一度にいくつかのテクニックを混ぜて少し変更することができます。

-彼女は5歳ですが、すでに2つの言語を話すことができます。

-本当に？ 彼女は何語を話しますか？


-私は一年中南米を旅行しました。

-一年？ なぜそんなに長いの？


テクニック5：追加の質問



英語では、追加の質問はフォローアップ質問と呼ばれます 。 対話者の言葉ではなくこのような質問をするので、このテクニックは最近英語を学び始めた人にとって最も難しいものです。 原則として、追加の質問は会話のトピックに関連していますが、興味に基づいて質問します。 前の例の1つを拡張し、そのような質問が実際にどのように機能するかを確認します。

-家族と私はカンボジアでの休暇から戻ってきました。

-あなたは？ 気に入りましたか？

-はい！ 人気の観光スポットをいくつか訪れました。

- 私は彼らが常に混雑していることを読みました、それは本当ですか？

-部分的にはい。 しかし、彼らは間違いなく訪れる価値があります！

- どのくらい滞在しましたか？

-2週間。

- それで十分でしたか？

-はい、私たちはすべての時間を遠足に捧げたいとは思わなかったので、リラックスするのに十分な時間がありました。


したがって、いくつかの単語を繰り返すことで、完全な会話を始めることができます。 主なことは、（状況が非常に形式的な場合）議論に興味深く受け入れられるトピックを見つけることです。 世間話は長く続くことはありませんが、あなたとあなたの対談者の両方の気分を著しく高めることができます。
