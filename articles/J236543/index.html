ASUS G550JKゲーミングラップトップを確認する

ゲーマーズ共和国のノートブックシリーズは、最高のパフォーマンスを提供します。 ターゲットオーディエンス-ゲーマーを考えると、これは驚くことではありません。 このシリーズの一部として、多数のデバイスがリリースされています。 RoGブランドのラップトップに加えて、マザーボード、ビデオカード、サウンドカード、マウス、その他のデバイスもあります。 今日は、16 GBのRAMとFullHDスクリーンを備えた4コアCore i7プロセッサを搭載したASUS G550JKについてお話します。





技術仕様



プロセッサー： Intel Core™i7 4700HQ 2.4 GHz Quad Core。

オペレーティングシステム： Windows 8.1。

チップセット： Intel HM86 Express。

RAM： 16GB DDR3L 1600MHz SDRAM。

画面： 15.6 "16：9 IPS FHD（1920x1080）LEDバックライト、マット。

ビデオ埋め込み： Intel HD Graphics 4600。

ビデオディスクリート： NVIDIA GeForce GTX850M 4GB DDR3 VRAM。

内蔵メモリ： 1TB HDD。

光学ドライブ：スーパーマルチDVD。

ネットワーク： 802.11 b / g / n、Bluetooth™V4.0 + HS、10/100/1000 BaseT。

インターフェイス：ヘッドセット用3.5 mmオーディオジャック、3 x USB 3.0、RJ45 LAN、HDMI、ミニディスプレイポート、カードリーダー（SD / SDXC / MMC）。

サウンド： 4つのスピーカー+外部サブウーファー、ノイズリダクション機能付きの2つのマイク。

バッテリー： 59 Wh

寸法： 383 x 255 x 27.7 mm。

重量： 2.4 kg。



パッケージバンドル



ラップトップは、マット仕上げのかなり大きな箱に入っています。 黒と赤のパッケージデザインは、Republic of Gamersシリーズデバイスの典型的なデザインです。





箱を開けてラップトップを取り出し、布製の封筒に入れます。







ラップトップの下には、外部サブウーファー、ドキュメント、および充電器があります。





充電器は、光沢のある両端を持つつや消しプラスチックでできています。 一般に、充電は165x76x30 mmの非常に大きいことが判明しましたが、そのようなラップトップには1.5倍以上の充電器がありました。





電圧19 V、電流6.32 A、最終的には120ワットになります。





外観



通常、ゲーム用ラップトップは、印象的な要素を備えた過酷で残酷なデザインになります。 ここでは、2.4 kgの小さな重量のかなり薄い厳格なラップトップです。 赤い細い線とRoGロゴのみが、ゲーム用ラップトップを持っていることを示しています。





トップカバーは、珍しい質感のブラシ仕上げのアルミニウムでできています。 表面はマットですが、指紋を収集します。 確かに、それらは特定の角度でのみ見ることができ、簡単に消去されます。





ASUSのロゴの下には、暗闇の中で輝く共和国のゲーマーズネームプレートがあります。





蓋は1つの大きなヒンジに取り付けられています。 その下には冷却システムのブロワー、つまり 熱風が吹き返します。





120度の開口角度で、スクリーンはほぼすべての位置に固定されています。

ラップトップの底面は、硬い金属シートで覆われています。





下はサウンドシステムの穴、中央は冷却システムの換気グリルです。





インターフェース



ASUS G550JKの外部インターフェイスを検討してください。

左側には充電器のコネクタがあり、次にサブウーファーの黄色の出力があります。 ラップトップの厚さが薄くなっているため、RJ-45コネクタにはヒンジ付きの蓋が付いています。 モニターとテレビを接続するための2つのコネクタがあります。これらはHDMI 1.4とミニDisplayPort 1.2です。 同時に、3つのモニターを接続できます。1つはHDMI経由で、2つはミニDisplayPort経由で接続できます。 周辺機器には、2つのUSB 3.0コネクタと、ヘッドセットを接続するための3.5 mmオーディオジャックがあります。





前面には、左から右へのインジケータがあります：電源、バッテリーステータス、ディスクアクティビティ、Wi-Fi、Caps Lock。





右側のほぼ全体がDVDドライブ、別のUSB 3.0、SD / XCカードリーダー、およびKensington Castleの穴で占められています。





音



強力なハードウェアの充填に加えて、開発者は優れたサウンドの世話もしました。





マルチメディアNシリーズと同様に、ラップトップには低周波再生を改善するためのリモートサブウーファーがあります。







サウンドシステムは、ラップトップの前のミシン目にある4つのスピーカーで構成されています。





サウンドを微調整し、既製のプリセットを選択するために、特別なソフトウェア「ROG Audio Wizard」がプリインストールされています。





カメラの横に2つのマイクが存在することにより、ノイズが低減されます。





これらすべてのおかげで、私たちはラップトップの中でも最高のサウンドシステムを手に入れました。



スクリーン



画面サイズは15.6インチです。 私たちの前にあるのは、1920 x 1080ピクセルのFullHD解像度を持つ高品質のIPSマトリックスです。 大きな解像度と小さな対角線の組み合わせにより、非常に鮮明な画像が得られます。





画像はシャープに見え、画面は色を完全に再現し、コントラストが高くなっています。





従来、IPSマトリックスには大きな視野角があり、色が消えたり歪んだりしません。









また、画面の大きなプラスは、光沢がなく、指紋で覆われていないマット仕上げです。





入力機器



良好なリターンと大きなキーストロークを備えたキーボードアイランドタイプ。 大きなケースのおかげで、キーの追加のデジタルブロックが右側にあります。





全体的なROGデザインを満足させるために、キーの記号は赤で作られています。





左側には、特定のアプリケーションの起動を割り当てることができる個別のボタンがあります。





キーボードは使いやすく、下部には手首用のスペースがたくさんあり、手は体重をかける必要はありません。









暗闇で遊ぶファンは、キーボードのバックライトが赤く光るのが好きです。





このため、キーの記号はレーザー彫刻されています。これは、キーのライトではなく、記号自体です。





バックライトの明るさはホットキーによって変化し、合計3つのレベルが利用可能です。 キーストロークがない場合は、しばらくしてバックライトが消灯します。

寸法が10.5 x 7.5 cmの大きなタッチパッドは、触り心地がよく、感度が良好です。 彼はすべてのプレッシャーをうまく満たし、「誤った」タッチを除外します。





Windows 8では、すべてのシステムジェスチャが機能し、マルチタッチがサポートされています。 プレインストールされたASUS Smart Gestureソフトウェアは、タッチパッドでの作業を微調整するのに役立ちます。





性能



もちろん、ゲーム用ラップトップで最も重要なことは、そのハードウェアの充填です。 ASUS G550JKの心臓部は、最新世代の高性能HaswellプロセッサであるIntel Core i7-4700HQです。 ハイパースレッディングテクノロジーにより、8スレッドで同時に動作します。 4つのコアはそれぞれ2.4 GHzの周波数で動作し、ターボブーストモードでは3.4 GHzに増やすことができます。 このプロセッサは、市場で最も生産性の高いモバイルソリューションの1つです。





RAMの量は16GB、DDR3 1600MHzです。 このような大容量は、すべてのゲームに十分であり、すべてのメモリを獲得するには、一生懸命努力する必要があります。





ASUS G550JKには、2つのビデオアダプタが搭載されています。 統合されたIntel HD 4600は、パフォーマンスグラフィックが不要な場合、通常どおり実行されます。 したがって、内蔵ビデオを使用すると、ラップトップはバッテリーで長持ちします。





ゲームでは、メインのNVIDIA GeForce GTX 850Mグラフィックスカードが接続されます。 DDR3ビデオメモリの容量は4GBです。 パフォーマンスを向上させるために、グラフィックカードはわずかにオーバークロックされます。

FullHD解像度でいくつかのゲームをテストした後、次の結果が得られました。

高い設定のBattlefield 4は、非常に快適な30〜50 FPSを生成します。 設定がウルトラに設定されている場合、フレームレートは20に低下します。

理由は定かではありませんが、 World of Tanksは高い設定で減速し、約35〜40 FPSを生成します。

アサシンクリードブラックフラッグは、高い設定で非常に快適にプレイでき、陸上および海上で30-50 FPSを提供します。

ゲーム中、ラップトップの上部とキーボードの右側が加熱されます。 通常、ゲームでは、左手はタッチパッドの左側の手首であるWASDキーに置かれます。 これらの場所では、暖房はマークされておらず、プレーするのは非常に快適です。

冷却システムは2つのファンで構成されています。 彼らはかなり静かに動作します。 適度な音量では聞こえません。



バッテリーとランタイム



内蔵バッテリーの容量は59W * hです。 このような食いしん坊の鉄では、記録的な数字を示すことは困難です。 そして、このようなコンパクトなケースでは、かなり容量の大きいバッテリーを取り付けるのは簡単ではありません。

中程度の明るさでHDビデオを視聴すると、ラップトップは3時間続きました。

平均的な画面の明るさとインターネットサーフィンの実行時間は3.5時間でした。

ネットワークをオフにし、最小輝度を設定し、省エネプロファイルをオンにして、オフィスアプリケーションでの作業を制限すると、最大4時間に達する可能性があります。

中程度の明るさの生産的なゲームでは、バッテリーは1〜1.5時間しか持続しません。

フル充電には2時間かかります。





おわりに



ASUSの新しいゲーム用ラップトップは、最高のパフォーマンスでケースのサイズを縮小するという方向で、異なる設計コンセプトを示しています。 高性能なコンピューターは、家から持ち帰りたくない重いモンスターではなく、非常に普通の軽量のラップトップとスタイリッシュでシンプルなデザインです。 もちろん、このような電力消費では、長いバッテリ寿命を維持するのは難しく、壁のコンセントに縛られます。 Intel Core i7-4700HQは、最速のラップトッププロセッサの1つです。 ASUS G550JKは、ゲーマーだけでなく、モバイルコンピューターからの完全なフィードバックを必要とするユーザーにもユーザーを見つけます。
