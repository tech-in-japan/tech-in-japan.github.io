RUVDSは、毎年のAlgorithmus Exchangeコンテストの参加者にVDSサーバーを提供します





Algorithmus 2016コンペティションはすでに6回開催されています。 EasyMani 自動フォロープラットフォームでモスクワ取引所の支援を受けて、 MFD-InfoCenter会社によって実行されます 。 トレーダーと資産運用会社は、モスクワ市場の株式市場と先物のリターンで競います。



このコンペティションは、資産運用会社が取引を公表し、その後入札の結果を表示し、投資家と契約者と一緒にビジネスを組織する機会を提供します。 したがって、成功したトレーダーは、取引所での自身のオペレーションを通じてだけでなく、EasyMANiを使用して戦略に加入している投資家から報酬を受け取ることによって、追加の収入を受け取ることができます。



同時に、トーナメントへの高周波トレーダーの参加は除外されます。これにより、参加者がお互いの前にいる可能性が等しくなります。 プライベートトレーダー（アセットマネージャー）とプロの株式市場参加者の両方が競争に参加できます。



RUVDSは Algorithmusコンテストのパートナーに加わりました。 トーナメントの参加者にクラウドインフラストラクチャをユニークな価格で使用する機会を提供します。設定に関係なく、Windowsの仮想VPSサーバーで最大30％の割引を受けます。



アルゴリズム取引と自動フォローで仮想サーバーを使用する利点は、通信と電力が途切れることのない信頼性の高いサーバープロビジョニング、世界中のどこからでも24時間利用可能、高い応答率、取引所への地理的近接性です。 サーバー自体には、最新かつ強力な機器が装備されており、ライセンスを取得したWindows Server 2012 R2 OSを実行しているため、高品質と信頼性が保証されます。



RUVDSチームにとって、これは金融機関やトレーダーとの協力の最初の経験ではありません。 私たちのクライアントの中には、最大の市場プレーヤーと、自身のデータセンターとモスクワ取引所のコロケーションゾーンの両方で仮想サーバーが提供される何百人ものプライベートトレーダーがいます 。 プロバイダーのチームには、Algeofondや証券会社からの移民も含まれており、トレーダーのタスクに関連するインフラストラクチャを提供できます。



MFDを知らない人は、金融情報のロシアのプロバイダーです。 会社の主な製品：NetInvestor、DIXI +、EasyMANi。 EasyMANiは、ロボットの投資家、マネージャー、クリエイター向けのオンラインコミュニティです。 投資家向けEasyMANiは、戦略の管理と取引の選択、実行シグナルの購読、取引の公開、ブログの管理などのサービスを実装します



現在、3,500人を超える加入者がEasyMANiシステムに登録されています。 EasyMANiシステムは、自動フォローアップサービスを編成するためのブローカーに依存しないプラットフォームです。 つまり、アセットマネージャーは1つのブローカーを介して、投資家とサブスクライバーは他のブローカーを介して作業できます。 同時に、誰でもさまざまな取引端末（NetInvestor、QUIK、Smart X、ALOR-Trade、Transaq）を使用できます。



*プロモーションコードの基本割引10％。 3か月、6か月、または12か月の期間を支払う場合、割引は最大30％になります。
