LTEのキャリアアグリゲーションまたは次のスマートフォンを購入する際に何を探すべきか

モバイルインターネットの速度は、キャリアとデバイスの両方に依存します。 4G +またはLTE-Advanceという用語が事業者のネットワークを表すために使用される場合、それはキャリアアグリゲーションテクノロジーがネットワークでサポートされていることを意味し、モバイルインターネットの高速化を保証します。 スマートフォンがサポートしているロシアの携帯電話事業者のネットワークにどのように実装されているのか、この記事はそれについてです。



この技術がどのように機能するかを理解するために、道を想像してみましょう。 明らかに、2車線道路のスループットは、1車線道路のスループットよりも高くなっています。 3車線または4車線のトラックでは、異なる速度で移動するさらに多くの異なる車を見逃すことができます。



道路と同様に、セルラー通信にはキャリアがあります-有用な情報が送信される無線周波数。 データ転送のためにキャリアを集約（結合）すると、多くのネットワーク帯域幅が得られるため、特定の加入者のモバイルインターネットの速度が向上します。



ロシアでは、現在、第4世代ネットワークに4つの周波数範囲が使用されています。

周波数範囲 1800 MHz 2600 MHz 800 MHz 2600 MHz

各範囲の帯域幅とカバレッジエリアは、オペレータによって異なります。 たとえば、一般的に優れたXiaomi mi5スマートフォンの国際バージョンは、20番目の範囲をサポートしていません。 ロシアの加入者にとって、これは非常に重要です。 この範囲の小さな5 MHz帯域、電力制限、さまざまなオペレーターのさまざまなカバレッジにもかかわらず、範囲自体は高い屋内浸透率を持ちます。 スマートフォンは屋内でデータを送信する可能性が高くなります。



別の例では、MegaFonには7番目の帯域の40 MHz連続スペクトルがありますが、他のオペレーターには20 MHzしかありません。 これは、完全に実装されていませんが、オペレータに大きな利点をもたらします（高速および大容量の場合）。



MTS加入者にとって、38番目の帯域がスマートフォンでサポートされているかどうかを確認することは確かに重要です-MTSはモスクワでこの範囲の高いカバレッジを持っています。 たとえば、iPhone 6Sの7人のユーザーは勝利し、スマートフォンは38番目の範囲で動作しました。



道路から類推すると、新しいスマートフォンは4つの道路（車線）すべてに沿って走行できるはずです。



したがって、新しいスマートフォンを選択する際、モバイルインターネットの速度が重要な場合、このデバイスが3、7、20、38の4つの範囲すべてを理想的にサポートするかどうか、仕様を読むときに注意する必要があります。範囲3および7、中央のオプション-3、7および20の範囲の存在。



次に、キャリアの集約について説明します。



現在、ロシアの事業者は次のキャリアアグリゲーションの組み合わせをサポートしています。

オペレーター メガフォン MTS ビーライン Tele2
組み合わせ 7 + 7、3 + 7、3 + 7 + 7 3 + 38、3 + 7 3 + 7 -


したがって、MegaFonはモスクワとサンクトペテルブルクで最大の構成になっています。これは、3番目の帯域から20 MHz、7番目の帯域から20 + 20 MHzの3つのコンポーネントの集合です。



加入者デバイスはカテゴリに分類されます。 現在最も一般的なのは、第4カテゴリーCAT4のデバイスです。 これは、受信（ダウンリンクまたはDL）でのモバイルインターネットの最大達成可能速度は、送信（アップリンクまたはUL）-50メガビット/秒で150メガビット/秒になることを意味します。 これは理想的な条件で達成可能な最大速度であることに注意することが重要です-主なものは塔から遠くない、セル内に加入者がいない、光トランスポートが基地局に運ばれているなどです

カテゴリー4のスマートフォンは4つのロシアのオペレーターすべてで機能し、6番目のカテゴリーのデバイスはMegaFon、MTSおよびBeelineネットワークで最大の結果を表示できますが、これまでの9番目のカテゴリーのデバイスはMegaFonネットワークで最大までしか表示できません。



キャリアアグリゲーションは、カテゴリ6から始まるデバイスに表示されます。 CAT6デバイスは2つのキャリアアグリゲーションをサポートし、すでに市場に出回っています。 これは、たとえば、Lenovo Moto P2、Lenovo Moto Z Play、Lenovo Moto Z、Sony Xperia X、Xiaomi Mi Note 2、Apple 6Sなどです。



カテゴリ9デバイスが登場し始めたばかりです。 これはHTC M10、Apple iPhone 7、Sony Xperia XZです。 2017年には、12番目と16番目のカテゴリのスマートフォンの登場が予想されます。まず、ASUSとSonyのフラッグシップモデルに注目してください。 オペレーターが追加の4x4 MIMOおよび256 QAMテクノロジーを実装すると、ピーク速度が可能になります。 オペレータインフラストラクチャソフトウェアを更新した結果、256 QAM変調が有効になっている場合、4x4 MIMOテクノロジー（受信用に4アンテナ、送信用に4アンテナ）にアンテナ管理への投資が必要になります。 この点で、最大ピーク速度は、ビジネス/ショッピングセンター、駅などのホットスポットでのみ最初に可能になります。



スマートフォンのカテゴリーを知る方法は？ これは簡単な作業ではありません。 良い兆候は、使用されているチップセット（プロセッサ/モデム）です。 たとえば、 こちらで見つけることができます。 6番目のカテゴリのデバイスでは、Qualcomm Snapdragon 625、626、653、435、および9番目のカテゴリ-Qualcomm Snapdragon 820および821に依存します。これは、残念ながら、最大の結果を保証することはできません。たとえば、ZTE Z11スマートフォンSnapdragon 820チップセットですが、グローバルバージョンは4番目のカテゴリのデバイスとしてのみ機能します。
