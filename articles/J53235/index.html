理論上の二足ロボット

皆さん、私は、邪悪な科学者が恐怖と破壊をもたらす巨大な殺人ロボットの軍隊を作成する余裕があるまで生きてきました。 私は後で人類の破壊に対処します、そして今、私はロボット二足歩行のためにかなり陽気な2足のシャーシを作成する可能性を理論的に証明しようとします。







何を受け取りたいですか？ 重量が約1 kgで、手足の長さがそれぞれ5 cmの2つの「骨」で構成されるシャーシが欲しいのですが。 ランニングモンスターはより多くの恐怖を引き起こすため、2 r / sの速度で脚を回転させるとよいでしょう。 歩行中の足のバランスが崩れるため、足をできるだけ明るくすることが重要です。 手足では、エンジンの重量が最も大きくなります。つまり、エンジンは脚の外側に配置されます。



ダチョウのようなものになりました。 それほどひどくないので、後でこれに取り組む必要があります。





今では、たとえおよそであっても、少し数える必要があります。

長さ5 cmの脚の各部分は、シャーシ全体の重量-10 N（惑星地球上で1 kgの質量）を支える必要があります。 必要な力のモーメントは、レバーの長さとその端に加えられる力の積になります。



T = L・F;

T = 0.05 m10 N = 0.5 Nm、

ここで、Tは力のモーメント、Lはレバーの長さ、Fは加えられた力です。



エンジンをピックアップする必要があります。 電気航空機モデルで使用されるブラシレスモーターは理想的です。 彼らは少し重く、大きな力を開発し、高効率を持っています。 最小エンジン出力Pは、レバーの力のモーメントTとその回転角速度Vの積に等しくなります。



P = T・2pi・V;

P = 0.5 Nm・6.2832・2 rpm = 6.2832 W、

ここで、Pはエンジン出力、Vは角速度です。



Eeee ...さて、ささいなことは必要ありません。12ワットの電力で十分です。 ハッカーA10-15Sのようなエンジンはまともに見えます（広告がない、私の目を引いた最初のエンジン）。







特徴

エンジンの瞬間を知るといいのですが...



T = P /（2piV）;

T = 33 W /（6.2832187.8 rpm）= 0.028 Nm、

ここで、Tは力のモーメント、Pはエンジン出力、Vは角速度です。



...また、ギアボックスのパラメーター。脚の角速度Voutが2 rpmで、トルクToutが0.5 Nmである必要があり、エンジンが11270を与えることに注意してください。



Tin・Vin = Tout・Vout;

Tin =（Tout・Vout）/ Vin;

Tin =（0.5 Nm2 rpm）/ 166 rpm = 0.006 Nm、

ここで、TinとToutはギアボックスの入力と出力の瞬間であり、VinとVoutはそれぞれ回転です。



ギア比N：



N = Vin：Vout;

N = 166：2 = 83：1、

ここで、Nはギア比、VinとVoutは入力と出力の回転数です。

脚のメカニズムを紙の上で理解する時が来ました。 古い本を調べてみると、必要なものが見つかりました-ジョイントのベベルギアとチューブラーシャフト。 この場合、次のようになります。
