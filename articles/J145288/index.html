AntiSMS-ブロッカーの迅速かつ効果的な治療

友人の皆さん、ご挨拶！



私たちは皆、原則としてSMS経由または端末経由の支払いを介して、作成者への送金を受け取る前にコンピューターを停止するトロイの木馬ブロッカーについて聞いています。 より経験豊富なユーザーは決して彼らを捕まえませんが、彼らの親relative、知人、顧客-しばしばそして定期的に。



ブロッカーの問題を解決するように設計された小さなプログラムを紹介します。さらに、これを完全自動モードで実行します。 私の意見では、インターネット居住者は常にバナーブロッカーを拾う可能性が高く、経験豊富なユーザーはわずか5分で隣人がコンピューターを治すのを助けることができるので、絶対に誰もがそのようなプログラムを備えたディスクまたはブート可能なフラッシュドライブを持っている必要があります。



特に、このユーティリティは、クライアントコンピューターのセットアップに関与するシステム管理者およびエンジニアを支援します。 ブロックされたコンピューターがパックで運ばれた場合、迅速な処置により時間と労力を大幅に節約できます。 簡単な指示に従って、ブロッカーを効果的に削除すると同時に、オペレーティングシステムの多くの誤動作を修正できます。



プログラムについて



このプログラムはAntiSMSと呼ばれ、このトピックはバージョン2.1のリリースを記念して作成されました。 現時点では、すべての基本的なアイデアが実装されているため、戦闘での使用には安全にお勧めします。



ユーティリティ作成の基本原則



  • WinPEからのみ機能します。 このモードでは、トロイの木馬はシステムの修復に干渉できません。
  • あらゆる目的のために完全に無料。 使用上の制限はありません。
  • 自動作業。 最初にコンピューターを見た人が使用できます。
  • 安全性 多くのチェックにより、ユーティリティが鉄の友人を傷つけないことが保証されます。
  • 速度とサイズ。 彼女は本当に小さくて速いです！

ご注意


WinPEでプログラムを使用する場合は、便利なアドオンをコピーしてシステムファイルを復元し、一部のチェック済みファイルを除外します。



使用説明書



経験豊富なユーザーの場合：

  1. AntiSMSを起動し、動作中のシステムを再起動した後、 ウイルス対策ソフトウェアでクイックスキャンを実行します。
  2. msconfigを起動し、必要に応じてスタートアップ項目とサービスを視覚的に確認し、安全であるがデジタル署名がないため無効になっているものを有効にします。


あなたが経験の浅いユーザーの場合：

  1. ブートディスクイメージをダウンロードし、ディスクまたはフラッシュドライブに書き込みます。
  2. このディスクから起動し、デスクトップでAntiSMSアイコンを起動します。
  3. 動作中のシステムを再起動し、 ウイルス対策ソフトウェアでクイックスキャンを実行します。
  4. スタート->実行-> msconfig->通常の実行-> OKをクリックします。 これにより、すべてのスタートアップが有効になりますが、トロイの木馬はありません。 再起動後に問題が再び発生した場合、アンチウイルスはこのトロイの木馬をまだ検出していません。 この場合、AntiSMSを再度実行し、この項目を実行しないでください。
  5. ウイルスの後にインターネットが機能しない場合は、動作中のシステムでAntiSMSを起動し、ネットワーク設定をリセットします。


プログラムの特徴



  • このプログラムは、WinXP以降のx86およびx64の任意の数のハードドライブ、システム、およびユーザーで動作します。
  • 各論理ドライブのルートにあるautorun.infファイルは、存在する場合は削除されます。
  • ユーザープロファイルでは、削除すべきではないフォルダーから実行可能ファイルが削除されます。
  • システムおよびユーザーの一時フォルダーは完全に消去されます。
  • hostsファイル内のすべての非標準エントリはコメント化されます。
  • ドライブを除くすべてのデバイスの自動起動は無効になります。
  • 重要なレジストリの場所（ShellやUserinitなど）が復元されます。
  • すべての一時的なスタートアップキー（RunOnceやRunOnceExなど）はクリアされます。
  • イメージファイル実行オプションのすべてのシステムプロセスデバッガーが削除されます。
  • ユーザーとシステムのすべてのポリシーが削除されます。
  • プログラムの使用制限のポリシーでは、無制限のレベルが設定されます。
  • 署名されていないサービスはすべて無効になり、システムの起動後にmsconfigで復元できます。
  • レジストリの起動時に署名されていないファイルはすべて無効になります。msconfigを使用して回復できます。
  • スタートアップフォルダー内の署名されていないファイルは無効になり、msconfigを使用して復元できます。
  • 署名されていないスケジュールされたタスクは、回復できるように* .bakに名前が変更されます。
  • トロイの木馬による違反の後、稼働中のシステムからネットワークパラメータを復元することが可能です。
  • 実行可能ファイルを起動するためのパラメーターが復元されます。
  • スクリプトはスタートアップレジストリおよびスタートアップフォルダから削除されます。msconfigを使用して復元できます。
  • Windows 7では、msconfigはスタートアップ項目とサービスを無効にする時間も追加します。
  • Windows XP x86の場合、ブート設定はセーフモードで復元されます。
  • WinXP x86、Vista x86-x64およびWin7 x86-x64の場合、メインシステムファイルは署名されていなければ復元されます。
  • 既知のすべてのMBRブロッカーが修復され、感染したセクターのバックアップコピーがバックアップフォルダーに保存されます。
  • 非標準のMBRの保存を実装しました。これにより、未知のトロイの木馬の処理が高速化されます。
  • トロイの木馬の悪意のあるアクションからシステムをより深くクリーニングするように実装しました。
  • ショートカットは自動実行フォルダーでも処理され、未署名はmsconfigを使用して復元できます。
  • WinPEで文字がどのように混在しているかに関係なく、すべてのパーティションが正しく認識されます。
  • ブートディスクはexFATをサポートし、最新のコントローラードライバーが含まれています。
  • パラメーターAppInit_DLLsは正しく処理され、署名されていないライブラリーのみが削除されます。
  • ファイルのバックアップとプログラムログは、％Temp％\ AntiSMSフォルダーに保存されます。
  • Universal Virus Snifferプログラムのチェック済みファイルのデータベースがサポートされています。


おわりに



このプログラムは、3か月間、多くのユーザーによって積極的に使用されていますが、完璧に制限はなく、建設的な批判は大歓迎です。 私はユーティリティの開発についての私自身のビジョンを持っているので、すべての願いを考慮に入れることができるわけではなく、私の自由時間には多くの休息があります。 それにもかかわらず、プログラムは多くの革新的なアイデアが実装されているレベルまで成長し、ブロッカーを破壊するための組み合わせに変わりました。 もちろん、トロイの木馬を見逃さない方が良いですが、トロイの木馬を捕まえたので、その治療を迅速で楽しいものにしてください。 皆さんに幸運を！
