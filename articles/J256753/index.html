Vivaldi 1.0.162.2 TP3候補

みなさんこんにちは！



今日は普通の集会ではありません。 第一に、それは並外れたものであり、第二に、次のテクニカルプレビューの3番の予備リリースです。言い換えると、TP3リリースは数日前に約束され、技術的な理由により延期されて最終的に終了しました。

  • Linux RPM 32ビット
  • Linux RPM 64ビット

  • Linux DEB 64ビット
