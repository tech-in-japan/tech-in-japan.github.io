IE 6がユーザーの頭の中にあるので、なぜこれがしっかりとお尻（この言葉は私を悩ませます）ですか？ IE 6を使用している人の肖像は何ですか？ IE 6はひどい音を立てますか？ そして、これをまだ理解していない人たちに何をすべきか？



ほとんどのインターネットユーザーの20％近くが、習慣を信頼するという特性を持っています。 私も認めます、私も固執し、IEから変更したくありませんでした。 今、私はIE 6を支持する単一の引数を見つけませんが、4つの反対があります：



  1. タブなし。
  2. 確率の高いサイトは不器用に表示されます。
  3. ブラウザが誤って閉じた後、最後に表示したページを再度検索する必要があります。
  4. ページの保存は、キャッシュからではなく、新しいリクエストで行われます。




私はIE 5.5のリリースを覚えています。IE5と比較すると、写真付きのページを保存できました。 かっこよかった。 しかし、それは2000年でした。 IEが勝者となったブラウザ戦争を覚えています。 また、Netscapeは無料のMPL（Mozilla Public License）の下でエンジンの形でのみ灰を残したため、 IE 6は独占者になりました。 IE 6と代替ブラウザがあるという考えは、ユーザーの心に浸透しました。



ユーザーの問題は習慣にあります。 アップデートまたは別のブラウザをダウンロードする方法がないというわけではありません。 変更を受け入れるのが面倒です... IE 6ユーザーのポートレートに関しては、考慮に値するカテゴリが1つだけあります。「非上級」ユーザーです。 これらは、バージョンに関する大騒ぎに興味がなく、インターネット、メールなどを使用したいだけです。彼らは簡単に新しいバージョンのブラウザに転送することができます。







開発者の問題は、組版に時間を浪費し、 IE 6の修正を見つけることです。 商取引に関しては、他の人のお金のために時間を費やすことができます。 私のプロジェクトに関しては、すでに意見があり、 IE 6のサポートは行っていません。 私のサイトは、白髪のユーザーをユーモラスに送信してブラウザを更新します。 Andrei Sabinin （ s_a_p ）も行いました。 共同プロジェクトのサイトでは、よりソフトになりました。警告が表示されます。 同時に、サイトはなんとなく正直に開いて動作します。



DivX （ビデオコーデック）の出力を思い出して、みんながお互いに尋ねると、私の目の前に写真がポップアップ表示されます。「映画が表示されない理由を知っていますか？ 音があり、絵はありません！「彼らは質問と回答を受け取りました：「コーデックをインストールしましたか？」 IE 6が終わりました。 「 IE 6の下」や他のブラウザーを個別に使用するのではなく、通常のレイアウトを実行したいので、すべてが揃っているので前進したいと思います。 そして、これは不可能だと思いますが、同じ車はすべて改良なしでVAZの組立ラインから出て行き、Webサイト開発者は依然として固定レイアウトを作成します。 またはしないでしょう:-)？
