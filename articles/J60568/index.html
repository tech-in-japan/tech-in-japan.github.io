Opera Miniのロシアのユーザーの活動は1年で3倍以上になりました





平均的なロシアのユーザーOpera Miniは、1か月あたり400を超えるWebページを閲覧し、7.47 MB​​のデータをダウンロードします



オスロ、ノルウェー-2009年5月27日。



Opera Softwareは、Opera Miniの使用に関するデータに基づいて、2009年4月にモバイルインターネットの状況と傾向に関する最新のレポートをリリースしました。 このレポートには、ダイナミクスに関する一般情報とユーザーのローカル設定に関する情報の両方が含まれています。 今回、Opera Softwareは、世界のさまざまな地域のモバイルオペレーターのトラフィックに関する集約データを発表しました。



Opera Softwareの最高経営責任者であるJon von Tetzchnerは次のように述べています。 -同時に、当社のブラウザはモバイルインターネットサービスの加入者の注目を集めています。 Opera Miniユーザーと通信事業者の両方の双方が勝つという、Win-Winのコンセプトを信じています。」



完全なレポートは、 www.opera.com / smwにあります。



主な調査結果



•2009年4月、2,340万人以上がOpera Miniブラウザーを使用しました-1か月前の1.5％以上、2008年4月の140％以上。

•過去1か月間で、ユーザーは87億ページ以上を閲覧していたため、1年前の結果は249％を超えています。 Opera Miniを使用すると、合計1億5100万MBの情報がモバイルデバイスにダウンロードされました-前年より295％増加しました。 Opera Miniはデータを90％圧縮できます。 圧縮されていない場合、ユーザーは約1.4ペタバイトの情報をダウンロードする必要があります。

•2008年4月と比較して、ロシアのOpera Miniユーザーの数は82.9％増加し、ユーザーがロードするWebページの数は205.5％増加しました。 ロシア人は月平均416のWebページを閲覧しました。

•4月、ロシア人は平均7.47 MB​​のデータをダウンロードし、平均Webページサイズは18.39 KBでした。 Opera Miniがデータを圧縮しなかった場合、この数値はほぼ184 KBに増加します。

Opera Software ASAについて

Opera Software ASAは、PC、携帯電話、その他のインターネットデバイスでWebページを表示するための新しい可能性を切り開きました。 Operaによって開発されたクロスプラットフォームブラウザーテクノロジーは、その機能、標準への準拠、小さなサイズで知られています。 同時に、高速で安全かつ動的なインターネットアクセスをユーザーに提供します。 Opera Softwareのメインオフィスはノルウェーのオスロにあり、世界中にオフィスがあります。 同社は、OPERA証券コードでオスロ証券取引所に上場しています。 詳細については、 www.opera.comをご覧ください。
