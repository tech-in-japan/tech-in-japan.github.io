マルチプラットフォームのポータブルGP2X Wizエミュレーターの概要

デバイスとそのコミュニティの機能を十分に評価するために、購入日から8か月後にこのレビューを公開することにしました。



多数の移転（2008年10月末に初期リリースが計画されていた）、ファームウェアの改良、および製品の通常の起動に必要な数のネイティブアプリケーションの作成後、GP2X Wizは2009年4月末にリリースされました。







このデバイスの主な目的は、すでに廃止されているスロットマシンなどのさまざまなプラットフォームをエミュレートすることですが、これによりマシンが終了することはなく、DOSエミュレーションとさまざまなプラットフォームのヒープもサポートします（Amiga、 Amstrad CPC、Atari ST、Commodore 64、ZX Spectrum 、 TI99 、 MSX ）。 他のポータブルデバイスのアナログやエミュレータとは異なり、このデバイスは主に高品質のエミュレーション用に設計されており、オタクの外観を見た人に2番目の若者を簡単に呼吸させます。

その主な目的に加えて、製造業者はデバイスに組み込みのゲーム、ビデオおよび音楽プレーヤー、画像および漫画本ビューアー、電子書籍リーダー、マルチゾーン時計、電卓およびボイスレコーダーを提供しました。











技術仕様



画面：AMOLED、320x240ピクセル、2.8インチ、タッチ;

プロセッサー：533 MHz ARM9;

RAM：64 Mb;

メモリ：1 GBの内部フラッシュメモリ、SDカードスロット（SDHCカードがサポートされています）;

オペレーティングシステム：LinuxベースのOS

電源：取り外し可能な2000 mAhリチウム電池。

出力：3.5 mmヘッドフォンジャック、内蔵マイク、スピーカー。

寸法：121x61x18 mm;

重量：136 g（バッテリー付き）、97 g（バッテリーなし）



納入範囲



パッケージに含まれるもの：

-スタイラス付きデバイス

-充電およびPCとの同期用のUSBケーブル

-英語、ドイツ語、フランス語、スペイン語、イタリア語、韓国語、中国語、日本語の最初のガイドドキュメント。



ビルドの品質は非常に優れており、バックラッシュはありません。韓国語（中国語と混同しないでください）アセンブリは、それ自体が感じられます。 ケースは、小さなメタリックゴールドの斑点のある濃い灰色の光沢のあるプラスチックでできています。 指紋はデバイスの本体に残りますが、プラスチックの異常な色で十分にマスクされます。



納入時のデバイスのタッチスクリーンは、透明なセロハンフィルムで覆われています。 保護するために、何らかの高密度の反射防止保護フィルムで画面を覆うことをお勧めします。 スタイラスへの応答の精度はあまり良くありません。画面の端に沿った「デッド」ゾーンの存在に関して小さな欠陥がありますが、これは主な機能にはあまり影響しません。 太陽の下では、PocketPC画面と比較すると、画面がはっきりと見えます。 視力に負担をかけずに日陰で遊ぶことができます。







ヘッドフォンジャックは標準（3.5 mm）で、ヘッドフォンはきつく、音は高品質（Hi-Fiクラスのヘッドフォンでテスト済み）で、外部スピーカーは非常に大きく、十分な音質（サイズ）を持ち、デバイスにうまく収まります。

コントロールボタンは非常に反応がよく、ストロークはソフトです。 右側のキーは不快に思えるかもしれませんが、操作中に慣れます。

バッテリーはリチウムイオンで、交換可能な2000 mAhのバッテリー容量で、約7〜8時間の連続再生とビデオ視聴に十分です。

コンピューターに接続すると、通常のフラッシュドライブとして定義され、1ギガバイトのユーザーフラッシュ領域または挿入されたSDカードのいずれかを共有しながら、SDHCカードがサポートされます。16ギガバイトのカードは問題なく機能しました。 ケーブルを介してデバイスにファイルを転送する速度は、USB 1.1プロトコルの最大帯域幅によって制限されます。

充電器とイヤホンは含まれていません。 デバイスを充電するには、「マザー」タイプのUSBコネクタを備えた電源が適しています。車のUSB充電で問題なく充電されました。

主な機能について詳しく説明します。



内蔵ゲーム



デバイスのネイティブソフトウェア環境で実行される6つの楽しい小さなアーケードゲーム フラッシュゲームは、デバイスの画面に合わせて考案され、完全に時間を浪費し、プレイアビリティは非常に良好で、グラフィックと音楽は快適です。

1.アニマッチ

2. Boomshine2x

3.無数

4.スクエアタワーディフェンス

5.テールテイル

6.ウィズテルン







フラッシュプレーヤー



スタンドアロンアプリケーションとして、およびこのデバイスの画面用に最適化されたフラッシュゲームランサーとして実装されています。 解像度が画面解像度を超える場合、フラッシュドライブは片側でスケーリングされます。 フォントとベクターグラフィックスは滑らかになり、ビットマップ要素は少し悪くなります。 スタイラスがサポートされています。

管理は簡単です-開いて、閉じて、終了し、90、180度の回転。 より複雑なゲームの場合、自由裁量でキーをバインドできます。 前述のとおり、すべてのAction Script 2クラスと3番目のバージョンの一部がサポートされています（正確なリストはありません）。 デバイスの動作を確認するためにいくつかのファイルをコンパイルしました-結果はビデオで確認できます。





画像/コミックを見る



基本的に2つの異なるプログラムですが、インターフェイスはほぼ同じで、違いは管理のみです。 メニューを使用して、ファイル間を移動したり、画像を回転させたり、画像を拡大または縮小したりできます。 上から見ていると、迷惑なステータス用語が出てきて、漫画を読むのが少し難しくなります。 次のファイルタイプがサポートされています：JPG、GIF、BMP、PNG（アニメーションPNGはサポートされていません）。 漫画本の表示は、画像ビューアーからほぼ完全にインターフェイスを借用していますが、画像は画面上で（またはズームでジョイスティックを操作することにより）ナビゲートされ、画像が終了したら左から右にジャンプしてから、左から右にジャンプします。



音楽プレーヤー



通常、キリル文字を認識し、コントロールインターフェイスは単純です（停止、一時停止、次、前、両方向への巻き戻し、現在のトラックのナビゲーションバー）、プレーヤーはディレクトリで音楽を再生します。ランダム再生モードがあり、ディレクトリの再生を繰り返し、1つのトラックの再生を繰り返します。 再生中に、取り外し不可の視覚化がスピンします。 再生品質は優れています。 サポートされている形式は、WAV、MP3、OGGです。







ボイスレコーダー



インターフェースはいシンプルで、内蔵マイクからメモリーカードにステレオサウンドをwav形式で記録します。ストリームは16ビットでエンコードされ、信用を落とす周波数は48000Hzです。 録音品質は非常に良好で、実質的に外部ノイズはありません。







ビデオプレーヤー



インターフェイスは完全に音楽プレーヤーから借用されています。 サポートされているコーデックMPEG4、DIVX、XVID。 320 x 240 25 fpsのビデオ再生は問題なく行われ、メーカーはビデオビットレートが2048 kb / s以下、25 fpsで640 x 480の解像度のサポートを約束しています。 字幕はサポートされていませんが、パーソナルコンピューターの最新機能を備えたアニメや新鮮な映画業界の製品のファンは、ビデオシーケンス自体にビデオをマウントすることで、常にビデオを小さな画面にトランスコードできます。 サードパーティのより機能的なプレーヤーが存在することは間違いありませんが、これは時間の問題です。







読者



シンプルなコントロール、ブックマークがサポートされ、私に関連するすべてのエンコーディング（UTF-8、ANSI、Unicode、KOI-8R）のキリル文字は理解できませんでした。 原則として、キリル文字が使用されているテキストを読むことは可能ですが、そのためには、コンソール上のフォントをキリル文字セット（標準Linuxフォント）を含む類似のフォントに置き換える必要があります。 代替の、より機能的なリーダーがありますが、私はそれらを使用しません（デバイスの画面は読むには小さすぎます。私はそのようなもののための電子リーダーを持っています）。



主催者



カレンダー、世界時間、計算機、タイマーで構成されています。 シンプルなインターフェイス、指のコントロール。

エミュレーター



このセクションを作成するにあたり、私はすべてをカバーしようとはしませんでした（これはWindowsまたはLinuxで可能なすべてのエミュレータの比較であるため）が、最も興味深いものに触れました。 私は資料を写真で説明しませんが、ビデオを添付します。



Nintendo Entertainment System （ロシアでは「Dendy」として知られています）



低いプロセッサ周波数（200メガヘルツ）でも問題のないエミュレーション、エミュレーターのオプションメニューでは、グローバル構成と個々のラムの構成の両方を保存できます。オプションの数は驚くべきもので、その品質はデスクトップコンピューターのアナログに劣りません。

次のROMがテストされています。

-スーパーマリオ

-ダークウィングダック

-チップ＆デールレスキューレンジャー

すべての画像は問題なく開始され、ゲームの状態を保存して読み込むことができます。 エミュレータメニューはあまり便利に呼び出されません（キーを同時に押してサウンドを追加および減少します）

グラフィック、コントロール、サウンドはかなり良いです。



スーパーファミコン （SNES）



エミュレーションは良好で、最悪の事態が予想されました。 デフォルト設定の品質に満足しているため、エミュレータのオプションについては詳しく説明しませんでした。 かなりまともなグラフィックスを備えたゲームでは、描画が遅くなり、画面全体に対角線が形成されます（最初はイライラしますが、時間が経つと慣れてきます）。 しかし、この欠点は、品質の低いデバイスファームウェアが原因で発生します。 これは、アプリケーション構成で120ヘルツのスイープを設定することで処理されます。このモードでは、バッテリーの速度が多少速くなることに注意してください。



ジェネシス （ロシアでは「セガメガドライブ」として知られています）



おそらく、制御、ゲーム中のセーブ、セーブの面で最も優れたエミュレーターです。 斜めのストリップはありません。 グラフィックはフルスクリーンに対応し、サウンドもエミュレートされ、すべてのROM（私は約20を試しました）は問題なく完全に起動しました。 エミュレーターは当然の5ポイントを獲得します。



ZX Spectrum



はい、そのようなエミュレーターがあり、エミュレーションの品質も最高です。このプラットフォームのすべてのハードウェア実装がサポートされています。 制御キーを自由にバインドして、各画像の個人設定を保存できます。 オーディオおよびビデオエミュレーションの品質は優れています。







アプリケーション開発



悲しいかな、良い製品であっても軟膏にハエがあります。 アプリケーション開発のための環境がないとは言いたくありませんが、それはそこにありますが、それは職人によって「ひざまずき」組み立てられ、ライブラリは少し曲がっていますが、一般的にはアプリケーションを書いて、移植し、コンパイルします。

開発は、プログラミング言語に多少なりとも流anyoneな人にとっては簡単です。

製品コミュニティに関しては、それらは全国的に断片化されており、開発者はやや怠け者です。 内部に本格的なLinuxカーネルがあるので、すでにC / C ++用のLinuxを開発した経験がある人にとっては簡単です。



GamePark Holdings （製品の製造元、以下GPH）



メーカーについて簡単に説明したいと思います。

-非常に奇妙なステップは、公式SDKなしの製品のリリースでした。夏（2010年）に約束されています。

-私は、その部門の数が少ないため、GPHはそれに割り当てられたタスクに対処できないように思われます。

-GPHはFunGPポータルの作成を計画しています（開発はすでに進行中で、ライセンスの問題は解決中です）。これは、基本的に、ライセンス済みのクリーンROMおよびネイティブアプリケーションを購入できるアプリケーションストアです。

-開発者とユーザーのコミュニティは、製造業者とは別に住んでおり（非常に非論理的です）、ファームウェアの開発には参加していません。

-ファームウェアのバグは長い間修正されており、新しい機能を備えたファームウェアのリリースは遅れています。

-すべてのエミュレーターはコミュニティによって排他的に生まれましたが、公式のGPH Webサイトでデバイスの機能の1つとして言及されています。



ハブのレビューから、あなたの多くがこのデバイスの中国のアナログであるDingoo A320について知っていることを知っています。

はい、一部の場所ではこのデバイスがWizより優れていることに同意しますが、別の観点があります。

-ディンはもはや生産されていません、これは一度限りの製品です。

-Dingの存続期間中、Linuxは移植されましたが、それを超えていませんでした（開発者コミュニティが小さすぎます）。

-Wizのユーザーおよび開発者のコ​​ミュニティはdingコミュニティよりも「遺伝的に」優れています。これは最初のGP2X製品ではなく、最後ではないと考えなければなりませんが、dingはすでにリリース段階でメーカーによって慎重に「滅菌」された製品です;

-VISAにはタッチスクリーンがあり、アプリケーションの機会が増えます。



鉄



このセクションでは、デバイスの購入とそのプロパティに関する2つのことに触れたいと思います。



このデバイスはドイツ語と英語のオンラインストア（ちなみに単なる再販業者）から注文することで購入できますが、 実績のある直接サプライヤーであるplay-asia.comを信頼することをお勧めします。私は知り合いがいません。彼らは多額のお金を買うことに加えて、小さな日本の驚きをパッケージに入れています。



デバイスのコネクタとケーブルについては、会社が韓国人であるため、デバイスのコンポーネントの一部は韓国のメーカーSamsungから借りました。 デバイスの出力ピン配列はユーザ​​ーコミュニティに知られており、ユーザーはアナログTVおよびオーディオ出力を使用し、イーサネットを介してコンソールをコンピューターに接続し、デバイスを充電し、そこからデータを受信できます。 同期には、Samsungプレーヤーの一部のモデルからのケーブルが適しています。Samsung電話機の一部のモデルからの充電に適しています。 クレードルが近い将来にデバイス用にリリースされることを予想するのは非常に論理的です（前世代のコンソールの場合と同様）。クレードルにもRJ-45出力があり、一部のブラウザは既にVisaに移植されているという噂があります。
