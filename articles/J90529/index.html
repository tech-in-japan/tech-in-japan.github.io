ヨーロッパ（特にチェコ共和国）でコンピューターサイエンスを教える方法

はじめに



MSTUでの教育プロセスに関するトピックを読んだ後。 バウマン 、私はこのトピックについても伝えるべきことがあると決めました。 私は物理学と数学の学部（情報学科）のプラハのチャールズ大学で（実際、まだ正式に勉強しています）留学を考えている人だけでなく、多くの人が外国の教育がどのようなものかを知りたいと思います将来のプログラマ向け。



私は1つの大学でしか勉強していませんでしたが（異なる国の異なる大学の学生と話をしましたが）、私はそれについてしか語ることができません。 しかし、ほとんどすべてのヨーロッパの高等教育機関のシステムはほぼ同じです。 ですから、このトピックが、それがすべてそこでどのように機能するかの一般的なアイデアを作成することに興味のある人を助けることを願っています。





入場料



免責事項：私は長い間行動しており、それ以来何かが変わった可能性があります。



コンピューターサイエンスのためにチェコの大学（チャールズ大学のようなポンツーンでさえ）に入学するのは非常に簡単です。 ここでの競争は1箇所あたり約2人であり、一度に多くの大学に応募できます。したがって、一部の学生は「人を見る、自分を見せるため」に試験に参加します。



入学試験自体は、物理学、数学、コンピューターサイエンスの3つの科目で行われ、入学する学部（物理学、数学、コンピューターサイエンス）に関係なく行われます。 3つの最悪の結果はカウントされません。つまり、1つの試験が完全に圧倒されたり、まったく受験されなかったりする可能性があり（物理学には行きませんでした）、これは結果にまったく影響しません。



試験は非常にシンプルで典型的です。 たとえば、数学の試験では、単純な三角方程式、単純な平面問題、「マーシャとダーシャはリンゴを5個持っているので、6番目のリンゴを盗んだ」などの論理問題を解きます。 コンピュータサイエンスの場合-たとえば、「このプログラムは何個の星を表示しますか？」という質問に答えます（添えものなし）。 各パートに4つのタスクがあり、試験を解くために90分が与えられます。



まだ入場に障害はありません。 また、チューターは受け入れられません。 そして、後者が低い競争で説明されている場合、競争が場所ごとに約7-8人である部門でさえ非難はありません（またはほとんどありません）（チェコ共和国では、これは、奇妙なことに、あらゆる種類の言語と政治学です）。



トレーニング



大学への入学が非常に簡単な場合、ここでの学習は容易ではなく、学生の約60〜70％が卒業証書を修了しません。



ここのシステムは非常に快適です。各アイテムを渡すために3回試行されます。 3回の試行後、学生が科目に合格しなかった場合、彼は翌年に自分のためにそれを書くことができ、彼は再び3回の試行を行います。 合計6回、各アイテムを放棄しようとします。



一方、6回の試行に必須のオブジェクトを渡さなかった場合、その人は大学から追放されます。 または、十分な数のローンを獲得していない場合-それらも除外されます。 花崗岩をかじるためのすべてのリベラルな要件により、科学はここで迅速かつ明確に、問題なく除外されます。 学部長を除外しないように説得することはおそらく可能ですが、非常に（非常に）正当な理由が必要です。



しかし、除外されるためには、研究のために本当に非常に必死に採点されなければなりません。 まあ、追放された人でさえ、それをもう一度やり、「優秀」と「良い」に合格したアイテムを自動的にカウントする機会があります。



アイテム、スケジュール



コンピューターサイエンスに関するいくつかのプログラムがあります。ソビエト後の大学の学科のようなものです。 「理論的なコンピューターサイエンス」、「プログラミング」、「データベース」。 各プログラムには、必須科目のリスト（マタン、代数、プログラミング、アルゴリズム、その他すべてを含む）、半必須科目の大規模なリスト（すべてを渡す必要はなく、必要なクレジット数を集めるだけで、通常12から3-4科目を選択する必要があります）あなたの好みに合わせて）および無料の科目（学部で研究されている科目であればどれでも構いません。たとえば、これらの科目の1つは「物理学の歴史」でした）。 必須科目はすべて合格する必要があります。 また、半必須の一部を渡す必要があります。



生徒は自分の好みに応じて自分で選択できるので、それに応じて、好きなように自分でスケジュールを設定できます。 そして、彼に興味のある科目のみを（必須を除いて）研究する。 ここでの自由度は非常に大きいです。 一部の人々はそれを使用します（彼らは自分たちで最も「無料」のアイテムを選択します）が、私の観察によると、かなりの数があります。



クレジットトレーニングシステム



私たちの大学では、すべてがローンのシステム上に構築されています。 つまり、各学生は次の学期に編入するために、学期ごとにある程度の単位を取得しなければなりません。 通常、この数は1週間あたり約40時間の学習負荷に相当します-1週間あたり1時間の授業= 1クレジット。 クレジットは、試験またはクレジット（対象者に試験がない場合）に対して発生します。 クレジットは「予備」で蓄積できます-最初の数学期に一連の試験に合格します。 これは、例えば、勉強を離れずに6か月または1年ヒマラヤに行きたい人にとって非常に便利です（ここにいくつかあります）。



トレーニングレベル



私が注意できることから（主観的に）：

  1. 教師レベル：チェコの大学教師の90％は非常に良いレベルです。 第一に、ここの教師のほとんどは40-45歳以下であり、科学分野で非常に積極的です。第二に、彼らは非常に率直かつ喜んで教材の理解を助けます。第三に、私は試験で問題を経験したことがありません。なぜ証拠は講義のようになかったのですか？」 講義ではなく、お気に入りの本に書かれている通りに何かをしたとき、先生は座って私の証拠を理解し、完全に理解して初めて評価を行いました。 そして、私の答えが正しかったが、彼が持っていたものと同じではない場合、これは評価に影響しませんでした（反対に、多くの人が誰かが何か新しいことを書いたことに非常に満足していました）。



    また、応用分野では、彼らが本当に彼らを理解している人々によって教えられていることにも注意すべきです。 たとえば、件名「F＃」は、Microsoftでこの言語を開発し、それについて本を書いた人が教えています。 C ++ / Java / C＃でのプログラミングは、これらの言語で非常に長い間書いて、実際のお金を稼ぐ実際のプロジェクトを書く人々によって教えられます。 「Linux Administration」または「Computer Security」という科目の教師は、週に数時間しか教えません。残りの時間は、管理またはセキュリティの専門家として働いています。



  2. 素材の関連性：プログラミングは非常に動的な領域であるため、ほとんどのアイテムは常にプログラムを更新する必要があります。 私たちの大学は成功します。 彼らはすぐに私たちと一緒にC＃言語でプログラミングすることを学びます（C ++の主題もあり、必須であり、asmは半必須です）。 プロセッサーのアーキテクチャーを研究する場合、Pentium 4の例を使用して研究し、マイクロコマンドのレベルでCore 2のような現代のアーキテクチャーとの違いについていくつかの講義を行います。 「Android向けプログラミング」、「iPhone向けプログラミング」というテーマがあります。 一般に、関連性に問題はありません。



  3. トレーニング資料：トレーニング資料に特に問題はありません。 まず、非常に優れたコンピューター文学のライブラリーがあります（多くの言語で）。 非常に必要な本がない場合は、教員の費用で注文できます（ただし、これを行うことができるのは教師のみです）。 第二に、学部には設備の整ったコンピューターラボがいくつかあります。 それらにリモートで接続し、コンピューティングパワーを使用できます（たとえば、夜間にニューラルネットワークなどのさまざまな悪意のあるものを残します）。



  4. 卒業証書とコースワーク：少し前に、私はBSU（ベラルーシ州立大学）の元クラスメートと会い、1人の女の子が2週間で非常に緊張して卒業証書を書いたと私に話しました。 これが実際の状況をどれだけ反映しているかはわかりません。 学部では、昨年の調査で卒業生が卒業証書を書く平均時間を500時間と見積もっていました。 もちろん、「あなたが尋ねたが、それを測定した」とき、それはインドのコンドームの大きさの冗談のようになりえます。 しかし、私の考えでは、卒業証書がここで受け入れられるには、10冊の本を再編するだけでは不十分です。



    そして、ここには学期論文はありません。 しかし、多くの必須テストがあります-C ++のプロジェクト、Javaのプロジェクト、HaskellおよびPrologのプロジェクト（これは必要な主題でした）、データベースを備えたPHPのサイト、Flashのプロジェクト、CのUnix用ユーティリティ（ソケットまたはrloginを使用したサーバーレベルで）。 また、MIPSのオペレーティングシステムの一部（ホラーホラーと2週間の眠れない夜）を書く必須の主題「オペレーティングシステム」もあります。 一般に、すべてのテストを正直に実行した学生は、すべての非常に大きな束を少しずつできるようになり、プログラマの実際の生活のために非常によく準備されます。







訪問、グループ、その他の管理



大学には「必須スケジュール」などの概念がないため、グループの概念も非常にarbitrary意的です。 私はグループ31の1年生でしたが、それは絶対に何の意味もありませんでした-誰もが異なる科目にのみ行って、私の「クラスメート」がどのように見えるかさえ知りません。



この場所では恥ずかしい思いをするはずですが、大学を定期的に訪れたわけではありません（本で同じことが読めるのにマタンに行くのはなぜですか？）。 筆記試験問題を誤って別の教師に渡そうとしたこともあります（試験で初めて教師を見たのですが、その前は彼の姿がわかりませんでした）。 そして、これは試験で問題になったことはありません（試験では、「なぜ講義であなたに会えなかったのか？」という質問に対するヒントすらありませんでした。しかし、講義についてはどうですか-教師は実際に私を見さえしませんでした）。 学生が試験に合格した/テストを書いても、違いはありません。



オプションの訪問では、ほとんどの学生が2番目または3番目にコースを開始します。 個人的には、3年目からフルタイムで働いています。 これは私の平均スコア（5段階で4.60から4.72）にいくらか反映され、標準5の代わりに7年間勉強しています（ここでは仕事の裏に隠れていますが、大学以外の生活の中で一定量のガウジングと多くの興味深いことも役割を果たしています希望する場合は、フルタイムでも、標準時間で完全に終了できます）。 これは完全に正常な状況です。



また、成績を与えられたと信じている学生は不公平であり、教師は彼が何かを嫌っているという理由だけでそれをやったので、声明を書いて委員会による再審査を求めることができます（通常2-3人の教師） 「バイアス」は含まれません）。 私はこれを一度しましたが、このプロセスは非常に正常で正気です-これはスキャンダルとは見なされません。「彼は同僚を怒らせます-ゆりかごをあげましょう！」 生徒と教師がキャラクターに同意しなかったことがたまたま起こります-生徒の攻撃を避けるために、外部の教師が再検査します。



結論



地元の学生は数学や理論の分野で一般的にやや弱い基盤を持っていると思います-私たちは表面積分を経ずに、多くのソビエト後の大学で教えられているすべてのことをしませんでした）



一方、ここの学生は、不必要な資料や時代遅れの資料を大量に詰め込んでおらず、自分が主題を理解しておらず、「講義でのやり方ではない」という理由で「失敗」をしている教師には意的ではありません。 ここの教師のほとんどは、人間としても教師としても非常に健康的です。 あなたは彼らと一緒にビールを飲みに行き、一般的に対等な立場でコミュニケーションを取ることができます。 そして、最も重要なことは、彼らの多くが彼らが教える分野で実際のプロジェクトに取り組んでいるということです。



まあ、主なポイントは自由と機会です。 ここでの機会は海です-欲望がある場合。 記録できる分野とエリアのリストは非常に印象的です。 しかし、誰も学生をスティックの下から強制することはありません-勉強もこれらの機会を使用することもしません。 誰も彼を引っ張ったり、「悲しい表情のために」トップ3を入れたりしません。 そして、卒業証書を取得するには、まだ一生懸命働く必要があります-しかし、そのような卒業証書は、少なくとも、多くを言います。



よくある質問

  1. どの言語が教えられていますか？

    トレーニングはチェコ語です。 書かれた作品とメールはチェコ語の英語で書くことができます。 私の最初の年には、ロシア語でマタンと代数（チェコ語の噴水なし）を通過しました。 「ここでこの公式が出て、解決策が見つかったと言った」というレベルで、入学試験に多くの誤りを書いた。



  2. 外国人は学費を払わなければなりませんか？

    必要ありません。 外国人はチェコ共和国で無料で勉強できます。 一方、学生は、住宅、食事、保険などのサードパーティの費用を自分で負担します。 生活費は月に約500ドルです。 最適な値は約700〜800です。



  3. チェコ人は外国人を怒らせますか？ 彼らはロシア人が嫌いだと言います。

    怒らないでください。 私にはたくさんのチェコ人の友人、チェコ人の夫がいます。そして、チェコ共和国で7年間、チェコ人はまだ私を怒らせていません。



  4. 外国人は寮を受け取りますか？ そして、そこに住む方法は？

    ホステルは与えます。 それは月に約180ドルかかり、部屋は通常2人用に設計されています。 ホステルの条件は非常に良好です。



  5. 奨学金はもらえますか？

    支払わないでください。 研究の結果に基づいた奨学金があります-しかし、彼らは2年目から優秀な学生にのみそれを与え始め、それは約50ドルです。 社会的な奨学金や、人々の友情の奨学金、またはその他の奨学金を得ることができるとは考えられません-大学はそのようなものを与えません。
