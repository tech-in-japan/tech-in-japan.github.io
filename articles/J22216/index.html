「タグライン'08」の評価。 投票。

多くの人はすでに、タグラインの評価が第2段階である投票に移行していることを認識しています。 動揺していなくても、私は正直非常に驚いています。 しかし、最初に、最近の過去に戻りましょう。





私の知る限り、すべては2006年6月に始まりました。 顧客になりすましている1人の明るい頭が、すべての市場リーダーに電話をかけ、標準的なサイトの作成を依頼しました。 その結果、受け取った情報は「 Webサイト開発コストによるTOP-30 Runet Web Studiosの評価 」として公開されました。 たぶん、このアプローチは完全に倫理的ではありません...それにもかかわらず、それは本当のブレークスルーでした、誰もが最終的にLebedev Studioでサイトを開発するのにどれくらいの費用がかかるかを知りました:)。



翌年、最初のTagLineレーティングが開始されました。 事前登録は310のスタジオを通過しました。 その後、レーティングの形成に関するすべての作業は、独立した専門家とレーティングオーガナイザーの肩の上に落ちました...スタジオの代表者は、簡単な登録を経なければなりませんでした。 検証後の最も興味深い意見やコメントがサイトで公開されました。 トップがどれほど適切であるかについてはコメントしませんが、結局、タスク自体は非常に困難であり、私が理解している限り、まったく経験がありませんでした。



私が理解しているように、今年、主催者と専門家は、評価のすべての参加者を「完全なプログラム」「ワークアウト」にすることを決定しました。 一方、「マスター-マスター」、トップになりたい場合は、曲がりますが、そうではありません...



現在、投票は6つのステップで構成されており、各ステップは必須です。 まず、すべての参加者（予備登録に合格した610人）に、市場の一般的な状況について質問します。 これらの質問（フォームのすべてのフィールドは必須です）：

  1. 現在の市場状況を簡単に説明してください。 プレイヤーと顧客の両方の市場に存在する主な問題を説明してください。
  2. 市場規模。 過去1年間にRunetで市場で活動している専門企業（フリーランサーはカウントされません）によって、サイトの作成、サポート、開発のための市場のボリュームが考慮されます。
  3. プレイヤーの数。 現在市場で営業しているプロのウェブスタジオの数を示してください。
  4. 今後の主な傾向/市場開発の傾向を説明してください。


市場の動向や問題についてはある程度意見があるかもしれませんが、それを表明すべきかどうかを自分で決めたいと思います。



または、 小さな地域のスタジオのオーナーである私が、ロシアのウェブデザイン市場の規模をどのように知っているのか教えてください。 または、それはそのような宝くじです、推測する人々のために-特別賞？ 結局のところ、私は税金で働いていないか、会計室があります...



私はあなたのことは知りませんが、私は非常識です。すべてのプロのチームを数えるという考えはないでしょうから、おそらくブルドーザーから番号を選ぶでしょう。 アナリティクスを理解することは私の職業ではありません。ウェブサイトを運営しています！ もっと面白い。



第二段階では、売上高でトップ10のスタジオを作ることを提案します！ オーガナイザーに敬意を払って：誰がいくら稼いでいるのかわかりませんが、そうではありません-私は嘘をついています、ロストフのいくつかのスタジオのスピードを想像できます。 ステージは必須であるため、 Lebedev Studioを最初に配置し、残りをランダムに配置します。人々に喜んでもらいましょう。



3番目のステップに進みます。 名声によるトップ10のスタジオ。 ここには大きな疑問があります...多分それはブランドの名声に基づいて評価を構築するのが正しいかもしれませんが、私の意見では、それは光沢のある雑誌にとってはどういうわけかです...



最初に10のスタジオのリストをコンパイルし、次に1から10の星を付ける必要がある理由は完全には不明です...ちなみに、インターフェイスは不気味で、すべてが非常に遅いです...しかし、これはまったく異なるトピックです...



会社、仕事、技術に関する「ステップ4、5、6」の評価に戻りましょう。これはおそらく、残しておく必要がある唯一のものです。



一般的に、なぜ私はこれをすべてやっているのですか...正直、私は誰かを怒らせたくありませんでした...それは、評価の信頼性と普及を追求する中で、彼らは再び意味を忘れていたようです:(



PSどういうわけか、それは感情的すぎることが判明しました、申し訳ありません...おそらく長すぎます:(。
