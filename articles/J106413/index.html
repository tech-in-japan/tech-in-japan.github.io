ヒュンダイITは、タッチスクリーン付きの70インチコンバーチブルテーブルを導入しました



2007年5月、マイクロソフトは世界に珍しい「コーヒーテーブル」を紹介しました。 家具の断片で、フォームだけが彼に関連していました。 Microsoft Surfaceと呼ばれるノベルティは、インタラクティブなマルチタッチテーブルでした。

Microsoft Surfaceは高価な30インチのマルチタッチテーブルであり、その機能は現在販売されているマルチタッチネットブック、ラップトップ、パネルの機能と同じであると言っても過言ではありません。



Microsoft Surfaceは、デジタルリアリティと対話するための新しい原則と方法を人々に示しました。 開発者がこれに基づいて作成するもの、明日のコンピューターがどのようになるかを想像するのはさらに困難です。 Microsoft Surfaceが見つけるアプリケーションと、デバイスで使用されるテクノロジーの開発方法を想像することしかできません。 しかし、ユーザーと批評家の両方にノベルティが好まれたという事実にもかかわらず、さまざまな理由で（もちろん、その主な理由は高価格でした）、Surfaceは部分的な製品であることが判明しました。



今年、日本企業のソニーはatracTableと呼ばれるMicrosoft Surfaceの代替を導入しました。 この製品は、2008年10月に初めてデモンストレーションされたAtracsysの元の開発に基づいて作成されました。 ソニーは適切な技術を取得し、35インチのatracTableインタラクティブデスクトップを導入しました。これは、多くの点でマイクロソフトを凌interactiveしています。



現在、インタラクティブテーブルには、ヒュンダイITによって開発された別の競合他社があり、Microsoft Surfaceアナログのニッチで足場を築く可能性があります。 同社は、韓国の情報技術展示会Kinexで、一見、巨大なiPadに似たタッチスクリーンの対角70インチのインタラクティブテーブルのプロトタイプを発表しました。





残念ながら、ヒュンダイはまだ技術的な詳細を明らかにしていません。 利用可能な情報によると、テーブルのベースは、必要に応じて画面を垂直面に変換できる変換可能なデザインです。 このポジションでは、彼はフラットパネルテレビの機能に対処することができます。 パネル解像度-atracTableと同様、FullHDはmutltitouchテクノロジーをサポートし、iPadと同期できます。 インタラクティブパネルの適用は、教育およびビジネスの分野だけでなく、エンターテイメント産業の分野でも考慮されています。



おそらく、Surfaceのユーザーの1人は、ダブルスクリーンを使用して同様のテーブルに変更したいと思うでしょうが、これまでのところ新製品の見通しはかなり曖昧です。 同社がプロトタイプを適切に評価し、明るい未来を持つことを願っています。
