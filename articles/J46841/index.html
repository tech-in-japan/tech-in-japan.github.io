Habrの組み込み検索をGoogleに置き換えます

最近、HabréのVoIPに関連するメモを積極的に探しています。 「ネイティブ」の検索エンジンは長い間私を失望させ、私が怠け者になるためにGoogleに頼るたびに。 なぜなら 不要なコンテンツをカットするために長い間Proxomitronを使用していますが、別のフィルターを追加することにしました（default.cfgからの抜粋）：



Name = "Habrahabr.ru - search hack"

Active = TRUE

URL = "*habrahabr.ru*"

Limit = 512

Match = "<input type="text" name="q" value=" site:habrahabr.ru" /><input type=hidden name="hl" value="ru">"

Replace = "<input type="text" name="q" value=" site:habrahabr.ru" />"

"<input type=hidden name="hl" value="ru">"



Name = "Habrahabr.ru - search hack"

Active = TRUE

URL = "*habrahabr.ru*"

Limit = 512

Match = "<form method="get" action="http://www.google.ru/search">"

Replace = "<form method="get" action="http://www.google.ru/search">"
