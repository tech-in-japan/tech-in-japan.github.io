ロシアでのインテルソフトウェア会議





9月22日と27日に、この種の最初のイベントが2つのロシアの首都モスクワとサンクトペテルブルクで開催されます-Intel Software Conference 。 近年、インテルはプログラミングツールの開発に多大な投資を行い、ベクター命令、マルチコアプロセッサ、クラスターなどのさまざまなレベルで並列処理を効果的に使用できるようにしています。 会議では、業界のリーダーと専門家がこれらのソフトウェア製品と開発者によるアプリケーションの詳細について話します。 事前登録により、会議への参加は無料です。 ただし、座席数は限られているため、このイベントに参加したい場合は、 今すぐ登録してください！



IntelのリードエンジニアであるRobert Gevaが率いる同社のスペシャリストは、高性能コンピューティング、クライアントおよび統合ソリューションを実装するための並列プログラミングモデル、およびこの分野の将来の方向性について話します。 インテルの革新的な並列プログラミングモデルの組み合わせの使用方法を学びます。 Intel Parallel Building Blocks チルク さらに、Intel アレイ構成要素とインテル スレッド化ビルディングブロック。



他のトピックの中でも、両方の会議でツールについて詳しく説明します。このツールには、C / C ++およびFortranの開発者向けのツール一式が含まれています-Intel Parallel Studio XE。 Parallel Studio XE 2011製品に含まれているStatic Security Analysisツールを使用して、プログラムの正確性をさらに検証する方法について説明します。



したがって、開発と最適化があなたにとって空の言葉でない場合は、 この機会を利用してください 。
