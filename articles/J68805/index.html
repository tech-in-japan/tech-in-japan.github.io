ホームサーバー：Zotac IONITX-AおよびCFI-A8989シャーシ





Zotac IONITX-Aマザーボードは、特に紹介する必要はありません。ここでは多くの人が既に聞いて知っています。 それで、最後に、私たちの村でそれを獲得する機会がありました。



簡単に：

-miniITXフォーマット。

-Intel Atom N330プロセッサー、GeForce 8400M GS;

-最大4GBのRAM（デュアルモード）;

-見積価格-230ドル。



パッケージ内のパッケージ：

-大規模なラジエーターを備えたボードZotac IONITX-A。

-3（貪欲ではない!!!）SATAループ。

-Wi-Fiアンテナ。

-プロセッサーのファン。

-4本の黒いボルト（後で詳しく説明します:)）;

-別の電源コードを備えた電源（Delta APD-90CD、中国製）;

-少しの古紙。

-ドライバ付きのCD（注意-Windowsの場合）。





このボードで最初に出会ったのは、プロペラをプロセッサに接続する方法が完全に不明確であることです。 クリロフのf話「猿と眼鏡」のヒーローを自分から彫ろうとする試みについての話には飽きません。すぐに決定に進みます。 ボルトはラジエーターの長穴にまっすぐねじ込まれています。







2GB RAM + SATA 200GB WD + 22 ''モニターはすぐにボードに接続されました。 コンピューターの電源をオン/オフするにはドライバーを使用し、マザーボードの接点を短絡する必要があることに気づいたとき、残念なことに私を待っていました。 しかし、しばらく実行するのに十分でした。 ファンのノイズが、同じビデオカードを搭載したネットブックやラップトップとまったく同じであることを確認しました。 彼は本当に大したことではないが、事件をかき消すことを望んでいる。



では、建物に移りましょう。 このおもちゃのケースを選ぶのは非常に難しい仕事であることが判明しました。 eBayとウクライナにあるほとんどすべてを表示しました。 しかし、実際にはminiITXのケースに関する情報はほとんどないため、120ドルから300ドルの価格を支払うことに同意しませんでした。 議論の中で最も「人気のある」ものの1つは、CFI-A8989の事例でした。 私を含む多くの人の選択は明らかです-50ドルの価格はまだ許容範囲です。 事件は非常に人気があり、私たちの街ではオンラインストアにあることが判明しました。 注文は配達、120分で、ケースは私の手にあります。 開いて、ケースから150Wの内部電源を捨てます（まだマザーボードに接続する場所がありません）。 20分以内に、ケース内のすべての鉄の組み立てが完了しました。 起動して...しかし、これは次のシリーズにありますが、今のところは写真と鉄のいくつかのマイナスについて見てみましょう。 プラスについては説明しません-このシステム自体は大きなプラスです:)。



カバーの下にあるものを見てみましょう。 左側のマザーボードには4本のボルトがあり、右側にはネイティブの150W電源を捨てた後の空きスペースがたくさんあります。







トップビュー。 左側には、推定熱を除去するために働くネイティブファンがあります。 今後の見通し-私の設定（ホームWebサーバー、svn、rocking）では、熱は検出されませんでした。 しかし、このファンの騒音は非常に明白であり、非常に迷惑です。 3.5インチハードドライブの位置は上から見ることができ、5.25インチデバイスはその上に自由に配置できます。







これは右側の空の場所です。ケースの底部にネジ止めされたネイティブ電源ユニットから下に穴が見え、後ろにコードがぶら下がります。







私が予想していなかったのは230ドルのマザーボードで、wi-fiアンテナとコネクタに挿入された電源の不一致でした。 2つの解決策-力ずくで固執するか、電源またはwi-fiアンテナからプラスチック片を切断します。 これまでのところ、ソリューションNo. 1が適用されています。







そして、マザーボードを少し引いた。 電源とwi-fiの写真がないため、メーカーは背面パネルに明確に保存しました。それらの代わりにLine-In、Line-Outの写真があります。 事実はただ不快に動揺し、豊富な機器とささいなことのそのようながらくた。 部品の非中央配置も、不快な後味を残しました。 一般に、背面パネルは完全に失望し、モニターコードを取り外すときのパネルの非常に不快な金属スプラッタです。 しがみついているために再び起こります。







しかし、これはすべてささいなことです。 主なものはもちろんノイズです。 このおもちゃを購入した結果は、サイレントホームサーバーであるはずでした。 あなたがそれを購入してそれがそうであると思うならば、忘れてください、ケースを閉じたときの雑音はまだそこにあります。 そのため、先日、車のオブシュムコイで住宅カバーの一部を貼り付けようとするはずです。 ここで結果を更新に追加します。 それがうまくいかない場合は、フレームをそのままにして、残りの部分を木材から行いますが、とにかく後でそれを行います:)。



一般に、それは非常に価値があり、最も重要なことには安価なコンパクトマシンであることが判明しました。 いくつかの理由で選択が彼女にかかった。

-料金はIntelに比べて高価で、N230に基づいて安くなることは可能でしたが、50〜70ドル（若いモデルのZotac IONITX）を節約することはできませんでした。

-ケースは安価で、手頃な価格で、「非常に」うるさいことが判明した場合でも、私たちの手は正常に成長しているように見え、ソ連で生まれ、何かを思いつき、フレームがあります-残りをリベットします。 :)。
