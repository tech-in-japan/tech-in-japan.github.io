Wiredは勝者と敗者のリストをWeb 2.0で公開しました

Wired Magazineは人気投票を実施し、Web 2.0世代の最高および最悪のサイトを特定しました。



このリストを分析すると、Web 2.0には本当に多くの新しい興味深いアイデアがあることがはっきりとわかります。 しかし、さらに悪いアイデアがあります。 Wired Magazineは、最高の5つのサイトと最悪の5つのサイトを含む10のWeb 2.0サイトのリストを公開しています 。 もちろん、これは氷山の一角にすぎず、Web 2.0の勝者と敗者の「本当の」リストははるかに長いものです。

例として最も特徴的なサイトが選択されます。



勝者



写真サービスFlickr 。 最高の世代2.0 Webサイトの1つ。 最新のインターネットテクノロジーにあまり詳しくない新人が、タグとは何か、なぜタグについての話題が多いのかと尋ねたら、Flickrを見せればすぐにすべてが明らかになります。 このサイトは、ウェブのセマンティック分類の最良の例の1つです。 Flickrのその他の利点：オープンソフトウェアインターフェイス、Creative Commonsの下で写真のライセンスを取得する機能、ユーザーコミュニティ内でソーシャルネットワークを構築するための多数の機能。



ポッドキャスティング端末Odeo 。 数年前に最初のポッドキャストが登場したとき、ポッドキャストが非常に困難であるとすぐに誤解を招き、ポッドキャストを記録するには高価な機器と複雑なプログラムが必要です。 そして、Odeoサービスが登場し、これらすべての神話はすぐに消えました。 このサイトでは、ブラウザを閉じずにポッドキャストを記録および公開できます。 すべての操作はフラッシュモジュールを介して実行され、さらに、会社は<a

href = " studio.odeo.com/create/home ">携帯電話から直接ポッドキャストを記録および公開できるプログラム。つまり、ポッドキャスト用のマイクさえ必要ありません。



書き込み可能なテキストエディター：MS Wordの優れた代替手段。 まもなく、Web 2.0サービスは、PC用の従来のオフィスプログラムに完全に取って代わることができます。 メールプログラムとテキストエディターは、ブラウザーを介して動作し、オフラインの「競合他社」のほぼすべての基本機能をサポートするWebアプリケーションの役割にすでに優れていると感じています。



多くのユーザーを対象としたDel.icio.usオンラインブックマークサービスは、日常業務に不可欠なツールとなっています。 この公開サイトのページには、Webで見つけた貴重なコンテンツへのリンクがあります。 リンクの公開は、専用のツールバーをクリックするだけです。 友達がウェブで見つけたものをフォローし、選択したタグのRSSフィードを読むことができます。



NetVibes Personal Aggregatorは、過去のモンスター型のポータルの子孫です。 今日まで残っている巨大なサイトでは、ニュースを読んだり、天気を調べたり、星占いを知ったり、最新のスポーツ結果を調べたりすることができます。 フランスのWebサービスNetVibesを使用すると、各ユーザーは独自のパーソナルポータルを作成し、自分が見たい情報だけをそこに配置できます。



敗者



Web 2.0の主な「敗者」はソーシャルサービスMySpaceでした。 彼がそのように認識されたのはこれが初めてではありません。 100万人以上のMySpaceユーザーに感染したバナーで最近の話を思い出すだけで十分です 。



視聴者が1億人の失敗したサービスとは何ですか？ それはあなたができることが判明しました。 客観的には、このサイトは非常に不便なインターフェースを備えており、自動的に開始されるアニメーションとサウンドは言うまでもなく、広告はあまりにも迷惑です。 しかし、残念なことに、これは、いわゆる「レミングス」の百万人強の軍隊、つまり主にティーンエイジャーの間の経験の浅いインターネットユーザーがまさに好むものです。



専門のWebサイトであるSquidooは、あらゆるトピックに関するコンテンツを公開するための「レンズ」という独自のコンセプトを考案しました。 ある分野の専門家は「レンズ」を設定し、ある分野の貴重なコンテンツへのリンクを収集し、ページから広告収入の一部を受け取りました。 しかし、この概念は理論的にのみ良いことがすぐに明らかになりました。 実際には、サイトの多くのセクションが広告ゴミに変わりました。 他の「レンズ」のほとんどは、通常のGoogle検索よりも有用ではありませんでした。 結局のところ、最高の専門家は30ドル以下しか稼ぐことができません。つまり、自分の専門ブログから得られるよりもはるかに少ない収入です。



「敗者」のタイトルのもう1つの候補は、非常に最近の目新しさ、おそらく匿名のブラウザBrowzarで 、これはクッキーとサーフィンの履歴を自動的に消去します。 繰り返しますが、理論的には、アイデアは良いように見えるかもしれません（ユーザーのプライバシーを保証します）が、実際にはブラウザはキャッシュと閲覧履歴を完全に削除することはできません。 さらに悪いことに、ユーザーをコンテキスト広告でいっぱいのブランド検索ページに送ります。



サイトFo.rtuito.usは、ソーシャルネットワークの非常に奇妙な実装です。 このサービスは、人々を集める代わりに、完全にランダムな方法で友人を選択します。



Friendsterは、インターネット上の最初のソーシャルネットワークの1つです。 残念ながら、このサービスはその人気に耐えることができませんでした。 サーバーは負荷に対処できず、問題は数か月連続で続きました。 結局、ユーザーはうんざりし、同じ機能を備えたMySpaceが登場するとすぐに、ほとんどがそこに行きました。
