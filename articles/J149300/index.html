Steamはゲーム市場を超えています





Valveの最近の声明によると、同社のソフトウェア配布サービスはゲームアプリケーションの市場を超えて、はるかに幅広いソフトウェアの選択肢を提供します。



新しいSteamは、Steamストアに存在するゲームと同じ利点をすべて提供します：



  • 簡単なアプリケーションのインストール
  • 自動更新
  • Steamクラウドで行われた作業を保存する機能。




更新されたサービスの開始は9月5日に予定されています。



SteamをApple App StoreやCanonical Ubuntu Software Centerなどの製品に合わせようとするValveの動きは、Microsoft Windows 8 Storeと十分に競合します。



LinuxユーザーがLinux用Steamクライアントでもこのような機能を利用できることを心から願っています。
