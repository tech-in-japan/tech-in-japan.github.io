Ubuntu 10.04 Alpha 1リリース

昨日、Ubuntu 10.04 Lucid LynxのLTSリリースの最初のアルファリリースがメーリングリストで発表されました。

Ubuntu 10.04 Lucid Lynxの重要な革新の1つは、HALシステムからの削除であり、udisksおよびupowerと組み合わせてDeviceKitに置き換えました。 また、リリースはDebian Unstableに基づいていないことが決定されました。 Ubuntu 10.04 Lucid Lynxのカーネルバージョンは2.6.32です

* Ubuntu

* UECおよびEC2用のUbuntuサーバー

* Ubuntu ARM

* Kubuntu

* Xubuntu
