プロジェクト経済評価（パート1）

親愛なる読者の皆さん、約束された記事はどのように同意しましたか。

この記事は、スタートアップの資金調達に関する最初の記事の続きで考案されました。 当初、私は投資家がプロジェクトにお金を投資する前に注意を払う一連の指標として考えていましたが、これらの指標は他の参加者にとって興味深いと思われます。

以下に示すインジケーターを使用して、ほぼすべてのプロジェクトを分析できます。これにより、スタートアッププロジェクトの導入の有効性を賢明に評価できます。

しかし、大量の資料があるため、投資プロジェクトを評価するための指標の問題を一連の記事に分割することにしました。





計算中によく使用されるのは、売掛金と買掛金の指標ではなく、特定の現金受領額とその流出額-キャッシュフローです。

より完全に理解するには、キャッシュフローの概念に注意を払う必要があります。



キャッシュフローは、その経済的内容から抽象化された数値シリーズであり、時間の経過とともに分配される一連の支払いで構成されます。 これは、投資の経済効率の指標を計算したり、経済主体のキャッシュフローを経時的に分析したりするために使用されます。



この定義は、特定の期間における一連の移動オブジェクトとしてのフローの一般的な理解に対応しています。



正味現在価値（NPV）





NPVは、今日まで削減された支払フローの割引値の合計です。 NPVは、現在の時点（投資プロジェクトが評価される瞬間）まで減少したすべてのキャッシュインフローとアウトフローの差です。 キャッシュインフローがプロジェクトの実施に関連する初期投資コストと定期的なキャッシュアウトフローを返済した後、投資家がプロジェクトから受け取ると予想されるキャッシュの量を示します。 現金支払いは時間価値とリスクに基づいて評価されるため、NPVはプロジェクトによって付加価値として解釈されます。 また、投資家の総利益として解釈することもできます。



代替プロジェクトを比較する場合、NPVが高いプロジェクトを優先する必要があります。



NPV = ∑（（NPV）/（1 + r）^ t）-∑ K /（1 + r）^ t



CHDP-純キャッシュフロー

K-投資投資;

rは割引率です。



結果の解釈：

  1. NPV> 0-プロジェクトは有効であり、その収益は代替投資の収益を上回っています
  2. NPV = 0-投資家は、分析された投資プロジェクトに投資するか、別の方向に投資するかを気にするべきではありません
  3. NPV <0の場合、プロジェクトは無効です。




重要です。 負のNPVは、正の推定利益で取得できます。 したがって、非効率！=損失を生む。



純割引自己資本利益率（CHDsk）





この指標の特徴は、借りた資金の移動とその使用料の会計処理です。



NPV = ∑（（NPV-D-PP）/（1 + r）^ t）-∑（K-KK）/（1 + r）^ t



D-ローンの返済可能な債務;

PP-ローンの返済利息。

KK-クレジットを集めました。



収益性指数





ID = NPV / ∑ K /（1 + r）^ t



指標は相対的であるという事実により、投資の観点と量において比類のないスタートアッププロジェクトを比較する機会を得ます。



内部収益率（GNI / IRR）





GNIは、正味現在価値が0になる金利です。NPVは、これまでに割引かれたキャッシュフローに基づいて計算されます。



GNI = r1 + f（r1）*（r1-r2）/ f（r1）-f（r2）



r1は、f（r1）> 0（f（r1）<0）になる割引係数の値です。

r2は、f（r2）<0（f（r2）> 0）になる割引係数の値です。



プロジェクトマネージャーは、次の理由からこの特定のインジケーターを好みます。

  1. GNIは、プロジェクト情報の機密性を大幅に保証します
  2. GNIインジケータを使用すると、「安全マージン」を判断できます。


多くの場合、GNIは、プロジェクトを無効にせずにプロジェクトを実施するために融資を受けることができる最大率であると考えられています。 この声明は完全に真実ではありませんが、そのような詳細には立ち入りません。
