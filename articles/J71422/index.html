Windows 7でのWinamp-win7shell

Windows 7のすべてのユーザーは、新しいタスクバー機能-Aero Peek、組み込みの進行状況バー、オーバーレイステータスアイコン、ジャンプリストを知っています。 Habrでは、アプリケーションでこれらの関数をプログラミングする技術についても説明されました。 Windows Media Player 12など、多くの標準Windows 7アプリケーションで使用されています。



WMPを好まないが、古き良きWinampを好む人のために、新しいタスクバーを利用する機会もあります。 これを行うには、code.google.com / p / win7shellから入手できるwin7shellプラグインを使用します。



このプラグインは、Winampユーザーに次の機能を提供します。

  • 現在のトラックの再生を示す進行状況バー。 一時停止すると黄色に変わり、停止すると赤に変わります。



  • ミニアイコンは、単独で、または進行状況バーと組み合わせて、再生、一時停止、停止します：



  • Aero Peek-現在のトラック、アルバムアート、およびプレーヤーコントロールに関する情報を表示するためのさまざまなオプション：



  • ジャンプリストは、最近再生され、頻繁に再生されるトラックの右クリックリスト、および頻繁に呼び出されるWinampオプションのリストです。





プラグインの作成者はプロジェクトに積極的に取り組んでおり、Google Codeを介して一般の人々とコミュニケーションを取り、更新を頻繁にリリースしています。 昨日このプラグインに出会い、Windows 7のWinampユーザーに強くお勧めします。
