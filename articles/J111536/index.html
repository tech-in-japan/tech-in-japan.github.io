一般的なホットキー





  • alt + shift + G-スクリーンショットを撮る
  • alt + G-画面を更新します（デバイス画面を部分的に再描画する機能）
  • shift + alt + m（スタート画面上） -マイナーグッズを起動します。 ミネラルスクリーンでGを押すと、別のゲームGoMokuが使用可能になります。
  • alt + home -Kindleストアを開きます（インターネット接続が必要です）
  • キーボードの一番上の行のalt +ボタン-1〜9および0の番号をダイヤルします
  • メニューキー - メニューを呼び出して時間を表示できます。 開始画面には、使用可能な空き領域に関する情報も表示されます。
  • 下からスライダー（スライド） -格納位置で7秒間保持すると、Kindleをオフにできます。 15秒間保持すると、リーダーを再起動できます。 再起動はメニューからも利用できます（[ホーム]-> [メニュー]-> [設定]-> [メニュー]）。




読み取りモードのホットキー



  • alt + B-ブックマークの追加/削除（ブックマークページは曲がった角で示され、ブックマークは[メモとマークの表示]の下のメニューから使用できます。.mobiをチェックしました。すべてのテキスト形式（.pdfを除く）で同じ操作が疑われます）
  • ジョイスティック。 押し上げます。 -カーソルを画面の下に表示します。
  • ジョイスティック。 押し下げます。 -画面のカーソルを上に表示します。
  • ジョイスティック。 右を押す。 -次の章。
  • ジョイスティック。 左を押します。 -前の章。




テキスト読み上げ



  • shift + Sym-この機能をオンまたはオフにします。
  • 戻る -ピースを切断します。
  • スペースバー - 一時停止します。
  • カーソルを制御することにより、テキストを大声で再生する順序を制御できます。




音楽コントロール

より複雑なもの





ファームウェアの更新



ファームウェアの更新は、リリース後しばらくして（Wi-Fiがオンまたは3Gが使用可能になったとき）自動的に利用可能になりますが、短気なユーザーやデバイスでインターネットにアクセスできないユーザーには、次の方法があります。



  1. アドレスwww.amazon.com/gp/help/customer/display.html?nodeId=200324680にアクセスします
  2. 設定ページでKindleのバージョンを確認してください
  3. サイトからの簡単な指示に従います。
    1. そのバージョンのファームウェアをダウンロードする
    2. リーダーを接続する
    3. ダウンロードしたファイルをKindleのルートに書き込みます
    4. ケーブルを外します
    5. ホーム->メニュー->設定->メニュー-> Kindleの更新
    6. 待って、新しいファームウェアをお楽しみください




画像を見る



Kindleには画像ビューアーが組み込まれています。 どんな目的で使用できるかはわかりませんが、怠け者のコミック愛好家は気に入ってくれるかもしれません。 残りは.pdfに変換することをお勧めします。 .pngおよび.jpg画像をチェックしました。 この不気味な機会を楽しむには、次のことを行う必要があります。





  1. デバイスを接続する
  2. ルートにpicturesというディレクトリを作成します
  3. ディレクトリ内に、別のディレクトリを作成します（任意の名前で（理由内！）、画像に直接配置された画像は表示されません）。 写真内に任意の数のディレクトリを作成できます。
  4. 必要な数の画像をダウンロードします。

難しいこと



これらは、Kindleをより高度なデバイスにするための非公式の方法です。 自己責任で使用してください。



脱獄（必要な部分、とにかく完了！）



まず最初にすべきことは、この素晴らしいハックをインストールすることです。 タイトルにも関わらず、それ（およびそれ以降のすべて）のインストールは非常に簡単です。



  1. mobileread.comのリンクをたどる
  2. kindle-jailbreak-XXX.zipアーカイブをダウンロードします。これを書いている時点では、このメモはkindle-jailbreak-0.4.N.zipです。
  3. 解凍し、update_XXX_install.binファイルを見つけます、c XXX？ Kindleのバージョンに適した
  4. Kindleをコンピューターに接続し、ファイルをデバイスのルートにアップロードします
  5. USBケーブルを取り外します
  6. [ホーム]-> [メニュー]>設定-> [メニュー]> Kindleを更新します。

自分のスクリーンセーバー







このハックを使用すると、公式のスクリーンセーバーを自分のものに置き換える機会が得られます。

インストール手順は、ジェイルブレイクのインストールに似ています。 フォーラムmobileread.comからファイルkindle-ss-XXX.zip（メモを書いている時点でkindle-ss-0.18.N.zip）をダウンロードし、前の段落のすべてのステップを繰り返すだけで十分です。



ハックをインストールすると、 links / screensaversディレクトリがデバイスのルートに表示されます。このディレクトリに 、解像度600X800のモノクロの.png / .jpg画像をアップロードできます。 このタイプの画像を見つけることは難しくありません。怠け者のために、いくつかのリンクを公開しています。

少数の人には、たとえば、写真のGoogle検索を使用する簡単なアドバイスを与えることができます。



Kindleをプレゼントとして準備している場合、この人物の写真をいくつか記入してください。彼は喜んでいると思います。



効果を得るには、デバイスを再起動する必要があります。 ただし、 rebootsという空のファイルをlinksディレクトリに置くだけで（または、この目的のために特別に存在する既存の自動再起動ファイルの名前を変更できます）、KindleはPCから切断して10秒後に再起動します。



スクリーンセーバーは、ファイル名に応じてアルファベット順に周期的に適用されます。 再起動のたびに画像を混在させるには、同じlinksディレクトリに空のランダムファイルを作成するだけです。 その後、再起動するたびに、スクリーンセーバーを含むディレクトリ内の画像はNUMBER_NAMEの形式になります。
