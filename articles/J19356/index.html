カーネル2.6.24でVMware Workstationを起動します

Linux用VMware Workstation 6.0.2の現在のバージョン（ビルド：59824）は、2.6.24カーネルをサポートしていません。 この状況を修正するには、以下で説明するいくつかの手順に従うだけで十分です。



1. vmwareの動作を停止する

Ubuntuの場合：sudo /etc/init.rc/vmware stop

他では：

$ lsmod | grep vm

sudo rmmod vmmon

sudo ifdown vmnet2。 サービスを削除する

sudo /usr/bin/vmware-uninstall-vix.pl

sudo /usr/bin/vmware-uninstall.pl



3. VMware Workstation 6.0.2の最新バージョンをダウンロードして解凍します。 次にインストールします：

sudo vmware-install.pl

インストーラーのすべての質問に答え、モジュールの作成時に表示されるエラーを無視します。



4. vmware-any-any-update-116パッチをダウンロードして解凍します

打ち上げる

sudo runme.pl



残念ながら、パッチの作成者は不明であり、パッチはvmwareフォーラムにあります。



PS：この方法は、カーネル2.6.24のすべてのバージョンで正常に機能します。
