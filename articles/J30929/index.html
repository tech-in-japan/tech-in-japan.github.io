NWCインタビュー：Jim Foundation、Linux Foundationエグゼクティブディレクター

最近設立されたThe Linux Foundationコンソーシアムのエグゼクティブディレクターであるジムゼムリンは、ネットワークコンピューティングからインタビューを受けました。



Linux Foundationは、2つの組織（Open Source Development LabsとFree Stadarts Group）を一度に置き換えました。 コンソーシアムは何をしなかったのでしょうか？



新しい組織は、ビジネスプラットフォームとしてのオープンソースまたはLinuxのアイデアを促進するために多くの時間を費やすことを計画していません。 代わりに、プロプライエタリな競合他社からLinuxを保護します。



たとえば、 SCOで現在進行中の訴訟と同様の訴訟が発生した場合、財政的および法的サポートを提供します。 また、たとえばLinus Torvaldsのような人々が平和に働くことができるように、開発者に対する法的サポートも組織しています。



コンソーシアムの多くの企業が実際に競争しています（Red HatとNovell、IBMとHP）。 共通の基準を採用するように説得するのは現実的ですか？



これには注意が必要です。 しかし、歴史は、利害関係が高いときに人々が戦略的提携に参加できることを教えてくれます。 Linuxエコシステムに参加する競合他社は、Microsoft Windowsの形で独占的な独占企業と協力するための大きなインセンティブを持っています。



あなたはそのような敵をマスターしていますか？ マイクロソフトには、数千の自社開発者、数千のパートナー企業、そして最終的には290億ドルの銀行口座があります。



Microsoftでの立場を明確にしましょう。彼らは間違いなく成功した会社だと思います。 彼らは、巨大な法務部門の助けを借りてプラットフォームを守る素晴らしい仕事をしています。 さまざまなマーケティングの動きを通じて、市場の恐怖を呼び起こします。 そして、それらの標準は普遍的です。



Linuxコミュニティのタスクは、MSが正確に何をするのかを自分自身で理解し、すべて同じように効果的に行うことです。



あなたは敵を賞賛します。 マイクロソフトに参加を依頼しませんでしたか？



いいえ、彼らはしませんでした。



LinuxとWindows間のアプリケーションの不均衡はどのくらいで解消されますか？



とても速い。 オープンソース開発環境の大きな利点は、コードとイノベーションの共有による知識の迅速な共有です。 Linux OSのリリースには7年の間隔はありません。 アプリケーション開発者は、消費者が必要とする機能を追加するのを待つ必要はありません。



これで、Linuxエコシステムの強力なサーバー側を観察できます。 すぐに「デスクトップ」の分野で進歩が見られます。



ほんと？ この分野でのLinuxのシェアはごくわずかです。



マイクロソフトは巨大な独占企業であると認めます。 しかし、国内のソフトウェアエコシステムを構築するために、MS Windowsの代替としてLinuxを推進する理由がある世界には十分な地域があります。 オープンソースは、ベストプラクティスを自由に使用し、それに基づいてイノベーションを実装する絶好の機会を提供します。 効果的で経済的な「デスクトップ」システムを構築する際のラグの非常に速い削減がすぐにわかると思います。



先ほど、Linus Torvaldsがあなたの従業員になると言っていました。 この情報公開の時代に、彼の給料に名前を付けることができますか？



いいえ、できません。これは秘密です。 しかし、確かにあなたに言います。
