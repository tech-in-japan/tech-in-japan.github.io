Nokia Developer Days：初日

こんにちは、Habr！



今日と明日、Nokiaデバイスで使用されるプラットフォーム上のモバイルアプリケーションの開発者向けに毎年恒例のNokia Developer Daysカンファレンスを開催しています。



会議の参加者は、ノキアのモバイル開発者および戦略的パートナーです。 このイベントは、ノキアが開発者に提供する戦略と機会の問題、およびアプリケーションの作成プロセスを加速するテクノロジーに対処します。



イベントの初日は終わりました。イベントに参加できず、何らかの理由でオンライン放送を見なかった人のために、すべてのスピーチの短いダイジェストを準備しました。 少し後に、すべてのプレゼンテーションとイベントのビデオを受け取り、その後完全なレポートを作成します。



明日、 2日目のオンライン放送を視聴できることを忘れないでください。



オープニングスピーチ

Nokia Developer Daysは、ノキアのユーラシアの開発者と交流する部門の長であるセルゲイ・ルデンコと、 Habréで有名なマイクロソフトの戦略技術の専門家部門の長であるミハイル・ミクセン・チェルノモルディコフによって開かれました。 スピーカーは、Windows Phoneの過去、現在、未来について、1週間前にWindows Phone Summitで発表された新しいWindows Phone 8オペレーティングシステムのイノベーションの説明に特に注意を払って話しました。 そこで、Nokia Developer Daysカンファレンスが開かれました。



Nokia開発者向け機能

オープニングスピーチの後、Nokiaの開発者向け戦略的マーケティング責任者であるNaresh Chouhanが登場しました。 彼は、Nokia AshaやNokia Lumiaなど、さまざまなスマートフォン向けのアプリケーションを作成するNokia開発者の能力に関するレポートを読みました。 Nareshは、プレゼンテーションのスライドで、各プラットフォームについて、アプリケーションのダウンロードと購入の数が急速に増加していることを指摘しました。これは朗報です;-)







Windows Phoneのアプリケーションの収益化とプロモーション

その後、Mikhail Chernomordikovが再びステージに登場し、Windows Phone用に作成されたアプリケーションの収益化とプロモーションに関するプレゼンテーションを行いました。 彼は、プロモーション活動の手段とそうでない手段を詳細に説明しました。 彼のレポートでは、Windows Phoneアプリケーションストアの構造と機能が調査され、最近10万のアプリケーションのマークを通過しました。 マイケルは、アプリケーションのプロモーションでさまざまなサポートプログラム（Habréの「The Magnificent Seven of Applications 」）を紹介し、Nokiaの例を使用してデバイスメーカーとの対話について話しました。







最高のアフィリエイトプロジェクト

Digital Cloud Technologiesのモバイルアプリケーション開発責任者であるSergey Lutai Lutaiは、Webプロジェクトの開発からモバイルアプリケーションの作成まで、ビジネスの方向性をどのように変えたかを語りました。 Chernomordikovが始めた収益化のトピックを続けて、スピーカーはWindows Phone Marketplaceで公開された後のアプリケーションの開発と収益化について話しました：通知を使用する必要性、ライブタイル、更新と試用モードの必要性。







位置情報サービス

Sergey Rudenkoは、Nokia Developer Daysカンファレンスの初日を、ジオロケーションサービスについての講演で締めくくった。 セルゲイはスピーチの中で、Nokiaカードの現状（200か国、150万都市、2500万通りの範囲）について話し、Windows Phone 8のアプリケーションの開発者が近い将来に得るマップの開発計画と機会を共有しました（たとえば、独立した開発者が使用できるようになります） Nokiaのアプリケーションのオフラインマップ）。 Sergeyは、 Nokiaマップ 、 Nokia Maps API 、およびWebプロジェクト（Maps API for JS）およびNokiaスマートフォンで使用されるさまざまなプラットフォームでのマップの作成のための交通渋滞計算の方法論と技術について話しました-Nokia Asha（Maps API for Java ME）からNokia Belle（QT Maps API）。



明日はNokia Developer Daysカンファレンスの2日目であり、Nokia Developer Webサイトでも放送されます。 6月29日の10:00から放送を見ることができます。
