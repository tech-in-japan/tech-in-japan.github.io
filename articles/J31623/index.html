Bill Gates Epic Windowsユーザビリティレター

Windows XPの使いやすさに悩まされていますか？ ビル・ゲイツはまだ聞いていません。 マイクロソフトが発行した社内通信には、無視できない素晴らしいビルゲイツの手紙が1つあります。 2003年1月15日付けですが、まだいくつかの関連性が失われていません。 一般的に、この手紙は、Microsoftの創始者からのメッセージというよりは、Windows嫌いのブログの悪意のある投稿に似ています。



From：ビル・ゲイツ

送信： 2003年1月15日水曜日10:05 AM

宛先：ジム・オールチン

Cc：クリス・ジョーンズ（WINDOWS）; バーラト・シ​​ャー（NT）; ジョーピーターソン; ウィルプール; ブライアンバレンタイン; アヌープ・グプタ（研究）

件名： Windowsユーザビリティ系統的劣化フレーム



Windowsのユーザビリティがどのように悪化しているか、グループマネージャーがこれらの問題に対処していないことに少しがっかりしています。



昨日、私自身の経験についてお話しましょう。 MoviemakerをダウンロードしてDigital Plusを購入することにしました...それで、Microsoft.comに行きました。 ソフトウェアのダウンロードセクションがあり、私はそこに向かいました。



5回以来、タイムアウトのためにダウンロードページをまったく開けませんでした。 6回目から、8秒の遅延の後、まだブートしました。 このサイトは非常に遅いため、使用できません。



必要なソフトウェアは、最も人気のある5つのダウンロードのリストに含まれていなかったため、リストを50に拡張しました。



このリストは完全に紛らわしいです。 一部の名前と比較すると、フォルダーCの名前でさえ、\ Documents and Settings \ billg \ My Documents \ My Picturesは明確で理解しやすいようです。 システムによってフィルタリングされないものや、その他の多くの奇妙なものがあります。



リストを「メディア」のみに制限しようとしました。 まだ映画製作者はいません。 検索で「映画」と入力しました。 なし。 「映画メーカー」と入力しました。 なし。



その後、私はあきらめて、アミールに手紙を送りました。Moviemakerをダウンロードするためのページはどこですか？ 彼女も存在しますか？



彼らは、ダウンロードセクションを使用して何かをダウンロードすることは、ユーザーが期待するものとはまったく異なると私に言った。 同様に、サイトのメインページに移動して、キーワード「movie maker」（「moviemaker」ではありません！）を使用してサイトのメイン検索を開始する必要があります。



試しました。 サイトは感動的に移動しましたが、6秒後にページが読み込まれました。 必要なものをダウンロードするボタンが表示されるようになりました。



実際には、結果はよりパズルのようでした。 Windows Update Webサイトにアクセスして、そこで一連の呪文を作成するように言われました。



それは絶対に不安でした。 なぜ他の場所に行き、システムスキャンを実行してMoviemakerをダウンロードする必要があるのですか？



それでも、Windows Updateセクションに移動しました。 システムは、多数のシステム要素をダウンロードすることを決定しました。 一度だけではなく、何度か奇妙なダイアログボックスが表示されました。 Windows Updateは、Windowsで共通言語を見つけられませんか？



その後、スキャンを開始しました。 しばらく時間がかかり、17 MBのごみをダウンロードすることが非常に重要であることが通知されました。



そして、これはデルタパッチを作成することを知らされた後ですが、SCARESTの可能な方法で説明された6つの要素を提供する代わりに、17 MBをダウンロードする必要がありました。



私はまだダウンロードを開始しましたが、それは高速でした。 次に、インストールを実行したかった。 このプロセスには6分かかり、この時点ではコンピューターが非常に低速であったため、現時点では他のタスクに使用できませんでした。



この6分間、彼は一体何をしていましたか？ ある種の狂気。 これは、ダウンロードが完了した後です。



その後、システムはコンピューターを再起動するように指示しました。 なぜこれを行う必要がありますか？ 毎晩再起動します-今すぐ再起動する必要があるのはなぜですか？



しかし、彼女はそれを主張したので、私はまだ再起動しました。 もちろん、これはOutlookで開いているウィンドウの現在の状態を完全に失うことを意味しました。



そこで、システムに戻り、Windows Updateに再度送信しました。 Moviemakerだけが必要だったので、私はすでにそこに行った理由を完全に忘れていました。



そこで、Microsoft.comに戻って指示を調べました。 「WindowsXP」というリンクをクリックする必要がありました。 これはなぜですか？ Windows Updateは、Windows XPを実行していることを本当に知りませんか？



このボタンをクリックすると何が起こりましたか？ たくさんの不必要なものを手に入れたので、Moviemakerは間違いなくその中にありました。



だから、私はダウンロードし始めました。 すぐに実行されましたが、インストールには数分かかりました。 動作が遅いことは驚くべきことです。



ある時点で、Windows Media Series 9をダウンロードする必要があるというメッセージが表示されました。



それで、私はそれをすることにしました。 今回は、「開く」や「保存」などのボタンが付いたダイアログが目の前に現れました。 何をすべきかについての指示やヘルプはありません。 そして、私は何をすべきか分からなかった。



ダウンロードは迅速で、このプログラムのみのインストールには7分かかりました。



さて、私は今、Moviemakerを持っていると思いました。 これを確認するために、「プログラムの追加と削除」メニューに行きました。



彼はそこにいませんでした。



何があったの？ Microsoft Autoupdate Exclusiveテストパッケージ、Microsoft Autoupdate Rebootテストパッケージ、Microsoft Autoupdateテストパッケージ1、Microsoft Autoupdateテストパッケージ2、Microsoft Autoupdateテストパッケージ3を含むすべてのゴミ。



誰かがWindowsの使用可能な部分だけを捨てることに決めましたか？ ファイルシステムは使用できなくなりました。 レジストリは使用できません。 プログラムのリストは唯一の通常の場所のままでしたが、今では台無しになっています。



しかし、それはほんの始まりに過ぎませんでした。 後で、「Windows XP修正プログラム、詳細についてはQ329048を参照してください」のようなものを見ました。 Q329048とは何ですか？ ここでこれらのパッチは何をしていますか？ 一部のパッチは、「参照」ではなく「Q810655」と呼ばれます。 詳細については、Q329048を参照してください。



これは絶対的な混乱です。



映画製作者はまったくいませんでした。



だから、私はMoviemakerを手に入れる希望をあきらめ、Digital Plus Packageをダウンロードすることにしました。



自分に関する多くの情報を入力する必要があると通知されました。



私はこれをすべて入力しましたが、システムはどこかに封印されていると判断し、すべてを再入力する必要がありました。 もちろん、私が前に入力したほとんどすべてが消えました。



私は5回正しくダイヤルしようとしましたが、毎回すべてを消去し、もう一度ダイヤルしました。



だから、1時間以上の狂気、プログラムのリストの詰まり、脅迫、そしてMicrosoft.comがひどいサイトだという信念の後に、私はPlus PackageではなくMoviemakerを手に入れなかった。



私が見たユーザビリティの問題に対する注意の欠如は、私に衝撃を与えました。 以前は、Windowsネットワークの使いやすさや、802.11を使用しようとしたときに受け取ったメッセージで底を打ったと思っていました（ルート証明書に関するメッセージがまだ気に入りませんでしたか？）



本当にこれらすべてを使用できるようになったら、きっと追加のレビューを書くでしょう。
