非常に低い設定での半減期2

こんにちは。

私は人々が可能な限り低い設定でHL2を立ち上げようとするフォーラムに出くわしました;そこにリストされているチームは部分的に機能し、部分的に機能しませんでした。 私はすでにNGで飲むことにうんざりしているので、ライセンスを棚から取り出し、最新バージョンに更新して、自分で実験を行うことにしました。

そのような設定で品質を過小評価することは不可能です。











カットの下のスクリーンショット、ビデオ、設定：









そのため、コンソールでコマンドを入力すると、画像の品質が低下する段階のスクリーンショットを示します。











































次のように実行する必要があります。

D：\ゲーム\ Steam \ Steam.exe -applaunch 220 -console



autoexec.cfgファイルの内容、

パスD沿いにあります：\ゲーム\ Steam \ SteamApps \ jack7277 \ half-life 2 \ hl2 \ cfg ：

con_enable 1

sv_cheats 1

mat_dxlevel 95 //ここではDX 6 = 60を設定できますが、品質は95、オプション60、70、80、90、95よりも美しくなります

mat_picmip 4

r_WaterDrawReflection 0

r_WaterDrawRefraction 0

mat_wateroverlaysize 8

r_lod 3

mat_diffuse 1

mat_bumpmap 0

r_shadows 0

mat_fullbright 1

mat_forceaniso 0

mat_forcehardwaresync 0

mat_reducefillrate 0

mat_mipmaptextures 0

mat_filtertextures 0

mat_envmapsize 32

mat_envmaptgasize 8

mat_antialias 0



みなさん、明けましておめでとうございます、私たちはこれまで休みがあります！
