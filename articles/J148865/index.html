Nokia Developer Days：ビデオ録画

1か月ほど前に、Nokiaプラットフォーム（Windows Phone、Series 40、Symbian ^ 3）用のアプリケーションを作成し、Nokia製品を使用して独自のアプリケーション（Nokia Mapsなど）を作成する開発者向けの2日間のイベント、Nokia Developer Daysを開催しました。 本日、このイベントのすべてのスピーチのビデオを公開します。 便宜上、各ビデオの下に対応するプレゼンテーションを複製しました。







ノキアのセルゲイ・ルデンコによるオープニングスピーチ



ノキアの開発者エンゲージメント部門の責任者であるセルゲイ・ルデンコは、入門レポートでイベントを開きました。そこでは、会社の開発計画と開発者向けのインセンティブプログラムを共有しました。 興味深い事実：セルゲイは、ノキアLumia 800でプレゼンテーションを行いました。







マイクロソフトのミハイル・チェルノモルディコフによるオープニングスピーチ



2番目は、マイクロソフトの戦略的技術の専門家であるMikhail Chernomordikovでした。 彼のスピーチで、彼はセルゲイ・ルデンコのスピーチを続けましたが、すでにマイクロソフトの観点からです。 Michaelは、Windows Phone 7プラットフォームの現在の成長率を共有し、Windows Phone 8がもたらす革新についても話しました。

Nokia開発者向け機能



3番目の講演は、ノキアの開発者の能力についてでした。 Nokia Asha（シリーズ40）とWindows Phoneの開発の見通しは、Nokiaの開発者向け戦略的マーケティング責任者であるNaresh Chouhanによって語られました。 そして誰もが彼の言葉を信じるように-彼は非常に美しいグラフと数字で彼の言葉を確認しました。 注意、レポートは英語です。

Windows Phoneのアプリケーションの収益化とプロモーション



アプリケーションの収益化については、MicrosoftのMichael Chernomordikov氏は述べています。 彼は遠くからスピーチを始め、平均的なユーザーとその好みに関するデータを共有しましたが、これはアプリケーションを開発する際に留意すべき非常に重要な情報です。 その後、Mikhailは、Sergey Rudenkoがオープニングスピーチで少し語ったように、アプリケーションを収益化するためのさまざまなトリックとツール、およびNokiaとMicrosoftのチャネルを含むアプリケーションを促進するための主要なチャネルについて話しました。

収益化と統合請求



inneractiveのアカウントマネージャーであるOrne Cesisは、Nokia Store（およびMarketplace、Google Play、App Storeなど）の無料アプリと組み込み課金アプリの収益化について、彼女が働いている会社の発展を利用して語りました。

最高のアフィリエイトプロジェクト



レポート「Best Partnership Projects」で、デジタルクラウドテクノロジーのモバイルアプリケーション開発部門の責任者であるSergey Lutaiは、Webプロジェクトの開発からモバイルアプリケーションの作成まで会社のビジネスの方向性を変えた経験を共有し、Windowsで公開された後のアプリケーションの開発と収益化についても話しましたPhone Marketplace-経験豊富な開発者からのこれらのヒントは、Windows Phoneの多くの初心者開発者に役立ちます。

位置情報サービス



Sergey Rudenkoは、Nokia Developer Daysカンファレンスの初日を、ジオロケーションサービスについての講演で締めくくった。 Sergeyは講演で、Nokiaカードの現状（200か国、150万都市、2500万通りの範囲）と、マップの開発計画とWindows Phone 8アプリケーション開発者が受け取る印象的な機能について共有しました。Sergeyは、交通渋滞の計算方法と作成技術について話しました。 Nokia Maps、Nokia Maps API、およびWebプロジェクト（Maps API for JS）およびNokiaスマートフォンで使用されるさまざまなプラットフォームでのマップの使用について-Nokia Asha（Maps API for Java ME）からNokia Belle（Qt Maps API）まで。
