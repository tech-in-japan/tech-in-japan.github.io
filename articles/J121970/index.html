VS2010 SP1のWeb標準アップデート

本日、Web Platform and Toolsチームは、Web標準用の最初のVisual Studio更新プログラムのリリースを発表し、Visual Studio 2010のすべてのバージョンでHTML5、CSS3、およびJavaScript APIのサポートを拡大しました。







この更新プログラムは、Visual Studio Galleryからダウンロードできます 。



この更新では、次のヒント（インテリセンス）と検証が追加されます。

  • HTML5



    • オーディオとビデオおよび関連するタグ、
    • 新しいタイプの入力フォーム、
    • ドラッグアンドドロップ
    • ワイアリア、
    • マイクロデータ
    • Schema.orgのサポート


  • CSS3



    • 2D変換
    • 3D変換
    • アニメーション
    • 背景と境界線、
    • 基本的なボックスモデル、
    • 基本的なUI
    • 振る舞い
    • 色
    • 柔軟なボックスレイアウト、
    • フォント
    • ページメディア用に生成されたコンテンツ、
    • ハイパーリンクプレゼンテーション、
    • 行
    • リスト、
    • マーキー
    • メディアクエリ、
    • マルチコラム
    • 名前空間、
    • ページングされたメディア
    • プレゼンテーションレベル、
    • ルビー
    • セレクター
    • スピーチ、
    • 構文
    • テンプレートのレイアウト、
    • テキスト
    • トランジション、
    • さまざまなブラウザの接頭辞サポート
  • JavaScript API



    • Geolocation API
    • ローカルストレージ
