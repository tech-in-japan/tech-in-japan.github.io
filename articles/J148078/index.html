Windows 8は10月26日にリリースされます





Microsoftパートナー向けのMicrosoftのGlobal eXchange（MGX）販売イベントで、MicrosoftのWindows社長Stephen Sinofskyは、2012年10月26日金曜日にWindows 8クライアントオペレーティングシステムの新しいバージョンがリリースされることを正式に発表しました。 「これは少なくとも過去17年間で同社にとって最大のイベントです」とシノフスキーは語り、Windows 95のリリースを示唆しました。



マイクロソフトは、既存のシステムを更新するためにWindows 8を「箱に入れて」販売することを拒否するかもしれないという意見が以前にあったという事実にもかかわらず、これは明らかに裕福ではないことが判明しました。 Windows 8のRTMバージョンのリリースは、8月の初めにそれほど具体的には計画されていません。 このバージョンのWindows 8へのアクセスは、有効なソフトウェアアシュアランス契約を結んでいる企業のパートナーと顧客、およびMSDN / TechNetサブスクリプションの開発者と専門家が利用できます。



Windows XP、Vista、7システムではない古いシステムのユーザーには、39.99ドルでWindows 8 Proにアップグレードするための特別なシステムが利用できることを思い出してください（ロシアとウクライナを含む）。



また、タブレット用のWindows RTの特別なバージョンが初めてリリースされる予定です。 Windows 8のリリース日の公式発表の前日、IntelとMicrosoftがWindows 8を搭載した20種類のタブレットを準備していることがわかりました。
