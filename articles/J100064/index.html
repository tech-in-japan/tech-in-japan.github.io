Youtubeは新しいHTML5フレンドリーな埋め込みコードを提供します

YouTubeに、iframeに基づいて外部サイトにビデオを埋め込む新しい方法についてのブログ投稿が掲載されました。

この方法の主な利点は、FlashとHTML5を選択できることです。 選択は、Youtube Webサイトの特定のユーザーの設定によって異なります。 新しい埋め込み可能なコードはまだ最終段階にありますが、開発者は改善プロセスがより速くなるように、サイトがそれらの実装を開始することを望んでいます。

これまで、Youtubeの動画はFlashプレーヤーとしてのみページに埋め込むことができました。 ご存知のように、すべてのモバイルデバイスがFlashをサポートしているわけではありません。 新しい埋め込み方法の導入により、Youtubeのビデオはさらにアクセスしやすくなります。



新しいコードは次のようになります。



< iframe class ="youtube-player" type ="text/html" width ="640" height ="385" src ="http://www.youtube.com/embed/video_id" frameborder ="0" >
