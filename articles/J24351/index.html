モスクワがスティーブバルマーとReMIX会議を開催

ReMIXカンファレンスは、Webテクノロジーとマイクロソフト製品と他社のプラットフォームおよびテクノロジー、およびオープンスタンダードとの相互作用の問題に特化した独自の技術カンファレンスです。 カンファレンスでは、Microsoft CEOのSteve BallmerがロシアのWeb開発者とMicrosoftのインターネット戦略のビジョンを共有します。



2008年5月23日に、ReMIX会議（www.remix.ru）がモスクワで初めて開催されます。これは、マイクロソフトがWeb開発者、Webデザイナー、およびアクティブなインターネットユーザーのために3年目（www.visitmix com）。



マイクロソフトのCEOであるスティーブバルマーは、ロシアへの1日の訪問の一環として、この会議について講演します。 彼は有名なインターネット活動家のアントン・ノシクとの公開討論に参加し、ロシアのインターネットコミュニティの代表者からの質問にも答えます。



ウェブサイトwww.remix.ruで今日から最も差し迫った問題を特定するために、コンテスト「Steve Ballmerに質問する」が開始されます。 受賞者は投票により決定され、会議への参加が招待されます。



www.remix.ruでオンライン登録すると、イベントのウェブキャストに参加できます。



ReMIX会議の枠組みの中で、ラウンドテーブルはRunetの有名な人物とともに開催され、Microsoftの専門家はInternet Explorer 8、SilverlightTM 2.0、Windows Server 2008、IIS7などに関する5つのレポートを実施します。 さらに、参加者はパートナーの展示会とXboxゲームエリアを訪れることができます。



「インターネットとそのロシア語部分の急速な発展には、専門家による高度な技術の使用が必要です」とロシアのマイクロソフト駐在員事務所の技術専門家、ペトルディデンコは述べています。 -ReMIX会議では、参加者は最新のMicrosoftインターネットソリューションの全体像を把握できます。 それらをどのように使用し、どのように他社の製品やプラットフォームとオープンスタンダードで相互作用するかについて活発な議論が行われると確信しています。」
