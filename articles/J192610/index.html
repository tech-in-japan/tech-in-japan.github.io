Jedi-Python用のオートコンプリートライブラリ

そのため、興味深いライブラリ（ GitHub ）で広大なgithubに出会いました。 以下は著者からの言葉です。

Jediは、IDE /エディターでPythonコードを自動補完するために使用できるツールです。 ジェダイは働いています。 ジェダイは速い。 組み込み関数を含むPython構文のすべての基本要素を理解します。

さらに、Jediは2つの異なるgoto機能をサポートし、名前の変更をサポートしています。また、Pydocおよびその他のIDEチップもサポートしています。

Jediは非常にシンプルなAPIを使用してIDEに接続します。 主な実装は、Jediオートコンプリートを使用したVIMプラグインです。 IDEでのJediの使用をお勧めします。 とても簡単です。 問題（ライセンスを含む）がある場合-私に連絡してください。

Jediはそのようなエディターで使用できます。

  • Vim（ jedi-vim 、 YouCompleteMe ）
  • Emacs（ Jedi.el ）
  • SynWrite（ SynJedi ）
  • Sublime Text（ Sublime JEDI [ST2 + ST3]、 anaconda [ST3のみ] ）
