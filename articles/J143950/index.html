インターネット業界の成功は明らかです。 ただし、この記事の目的は、インターネットのすべての成果と賞賛の歌のドライリストではなく、将来の発展のためのベクトルを策定する試みです。 2007年頃から、このような記事は珍しくなく、インターネットの著名な人物の多くはすでにWeb 3.0についての考えを表明しています。 この数字はどこから来たの？



空想するのは意味がないので、一般に受け入れられていると一般に考えられているシステムを使用します。 その意味は、World Wide Webが段階的に進化しており、特定の時点で3番目の、しかし2番目の「インターネット」バージョンであるWeb 2.0であるという事実に要約されます 。 非常に技術的ではありませんか？ この用語の出現の歴史を語りたくはありません-誰でもウィキペディアで必要な情報をすべて簡単に見つけることができます。 ただし、この概念はWorld Wide Webの開発の一般的な傾向を指している可能性が高く、概してテクノロジー自体の開発とはほとんど関係がないことに注意してください。 はい、はい！ 今日は技術についてではなく、彼らの助けを借りて世界に導入されている新しい意味についてお話します。



まず、ネットワーク開発の前の各ステップを簡単に説明してみましょう。

  1. Web 0.0（pre-web）-情報交換のためのコンピューターネットワークを作成する最初の試み。
  2. Web 1.0-ネットワークの配布、コンテンツが管理者によって作成される多くの情報サイト、静的データ、訪問者、ユーザーは情報を「吸収」することしかできません。 最初の企業は、ネットワーク上に代表オフィスを開きます。
  3. Web 2.0-ネットワークの社会化、ユーザーがコンテンツの作成に直接参加する機会を得た、誰もが情報源になることができます。 動的データ。 インターネットは単なる情報スペースではなく、World Wide Webを介して接続されていることを私たちは皆理解していました。 グローバルブランドはネットワークに「登り」つつあります。
  4. Web 3.0-？






それを取得しましょう。 Web 2.0の主な成果は、それほど多くのテクノロジではなく、新しいレベルの対話へのアクセスとユーザーのネットワークへの統合、および新しいタイプのコンテンツとその作成方法でした。 インターネットはついに安定して急成長している業界に変わり、インターネット企業の売上高と巨額の資本は数十億に上りました。 現在、誰もがFacebookに座って、 TwitterとYouTubeにグーグル情報を投稿しています。 いいでしょう 次は？ 実際、瞬時の情報交換、「クラウド内」のデータ、およびその他の最新テクノロジーの可能性は、それ自体が目的よりもグローバルな目標を達成する可能性が高いです。



現代のインターネットの主な傾向を最大限に一般化すると、 「現実の世界から仮想世界への移行」というフレーズで表現できます。 現実の世界では、常に友人がいましたが、今では仮想の「友人」がいます。 常に、1つのアイデアや関心によって結ばれたコミュニティ、パーティー、人々のグループがありました。 これはすべてオンラインです。 新聞、雑誌、ラジオ、テレビなど、これらすべての古典的なメディアは仮想化に移行しています。 pdf形式または音声形式であるため、人々は本をあまり読み始めませんでした。 人々は実際よりもオンラインです。 そこでは、彼らはもう少し美しく、自信があり、もう少し成功し、教育を受けています。 これはすべて素晴らしいことですが、それでも「マトリックス」の特定の形を思い起こさせますよね？ 私たちは常にリアルタイムを節約し、仮想化に費やそうとします。





最も純粋な形でのこの進歩は、速く、技術的で、無責任です。 数年前の世界的な傾向は今や時代遅れになっています。 人類は進歩のレールに乗り出し、トロリーを解散させ、猛烈なスピードで駆け回り、同時に前の線路を建設しました。 さらに、これらのレールが私たちをどこへ導くのか、誰も本当に知りません。 インターネットはスポンジのように数百万テラバイトの情報を吸収し、最も成功したサービスはその構造化と読み取り可能な形式への変換に取り組んでいます。 何のために？ それとも、彼らが誰のためにそれをするのかを言ったほうがいいですか？ ここではすべてが非常に平凡です-このすべての楽しみのためにお金を払う人のために。
