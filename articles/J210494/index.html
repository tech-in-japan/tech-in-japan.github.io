世界で「環境に優しい」データセンター：簡単な概要





現在、空気のような大規模なテクノロジー企業のデータセンターが必要です。 もちろん、サーバー、冷却システム、照明システムを備えたデータセンターは多くのエネルギーを消費します。 もちろん、状況は10〜15年前に機能していたデータセンターに比べて改善されましたが、省エネテクノロジーの完全な勝利について語ることができるほどではありません。



しかし、DCの中間よりも省エネの状況がはるかに良いデータセンターがあります。 そのようなデータセンターはあまりありませんが、そうあります。この短いレビューでは、最も興味深い「グリーン」データセンターのいくつかを評価することをお勧めします。



フィンランドのGoogleデータセンター（ハミナ、フィンランド）









このデータセンターは、他の「グリーン」DCと同様に、その場所から恩恵を受けています。



問題は、この地域では沿岸水域（およびDCは海岸のすぐ隣にある）の温度がゼロよりわずか数度高いことです。 そして、この状況は一年中見られます。



さらに、DC自体は、既製の冷水取水システムをすでに備えている旧製紙工場の敷地内にあります。 もちろん、システムをデータセンターのニーズに適合させる必要がありましたが、すでにそうでした。 統合されたクリーニングシステムもあり、リターンシステムは、環境に実質的に害が及ばないように設計されています。







これらのすべての要因により、この特定のデータセンターのPUE（電力使用効率）などの指標はわずか1.1です。 通常のデータセンターでは、PUEは1.6〜1.8です。



Equinix AM3データセンター









アムステルダムの「サイエンスパーク」にあるこのデータセンターは、世界で最も環境に優しいものの1つです。 これは特に、2012年のDatacenterDynamics Green Data Center Awardで確認されました。 このデータセンターでは、地上技術と現在宇宙飛行士の利益に役立つ技術の両方を使用しています。



たとえば、燃料電池が使用されます。 また、最新の変更をすべて加えた「グリーン」冷却システムも使用しています。



個人のニーズに合わせて、このデータセンターは年間約80万キロワット時を生成します。





ここで約1分間巻き戻すことができます。最初は広告だけです。



Microsoft Standalone DC（米国ワイオミング州）









マイクロソフトはまた、サーバーラックへの燃料電池（メタン）の直接導入という新しい技術を提供し、他の企業に遅れをとっていません。



データセンターの建物は、水処理プラントの隣にあります。 なんで？ そして、このガスをどこかから輸送する必要なく、駅で放出されたメタンを使用するために。 実際、バイオガスについては、メタンが優勢な有機物の生化学プロセス中に放出されるいくつかのガスの混合物について詳しく説明しています。



「メタン」データセンターには、従来のDCと比較して次の利点があります。

  1. 停電による機器の故障の可能性の減少。 この場合、データセンターは独自のエネルギー会社であるため、地域の一般的なエネルギーシステムの負荷/障害が自律機器の障害につながることはありません。
  2. インフラストラクチャコストの削減。 これについては既に説明しました-無停電電源装置、変圧器、およびその他の電力機器がないため、データセンターのエネルギーインフラストラクチャを作成するコストは大幅に削減されます。
  3. 機器のエネルギー効率の向上。 これはPUEだけではありません。 プロジェクトの著者によると、システムの全体的なエネルギー効率は2倍になります。
  4. データセンターは、都市/地域の広範なエネルギーインフラストラクチャから離れて、メタンがあるほぼどこにでも構築できます。 メタンに加えて、「ソーラー」エネルギー、風力エネルギー、その他の代替エネルギー源を使用できます。
。



Google Data Center（張華、台湾）









Dobra Corporationの新しいデータセンターには、約6億米ドルが投資される予定です。 このデータセンターは風力エネルギーを使用し、独自のウィンドファームがデータセンターに提供するために構築されました。







さらに、ここでは珍しい冷却システムが使用されています。 そのため、ここでクーラーとして使用される水は夜に冷やされ（トートロジーは申し訳ありません）、その後、必要に応じて日中に使用して、断熱貯蔵庫に集められます。



1日が始まると、冷却システムで冷水が使用され、夜間に蓄積された水は1日の明るい（したがって暖かい）時間に十分です。 したがって、台湾のデータセンターは、従来のDCよりもはるかに少ない電力しか使用しません。



NSA Data Center（米国メリーランド州）









これについてはすでに DCで書いていますが、ここで興味深いのは、インフラストラクチャを冷却するために、ここで廃水（処理済み）が使用されることです。



この施設では、データセンターの機器を冷却するために、1日あたり約1,900万リットルの処理済み排水を使用する予定です。 ハワード郡当局とは既に署名済み（まだ署名されていない場合はNSA）で、NSAデータセンターを冷却するための廃水の使用を規定しています。 この合意がなければ、処理された廃水は処理後すぐに最も近い川に排出されます。



機関のデータセンターを冷却するために水を取る新しいポンプ場を作成するために必要なのは、わずか4,000万米ドルです。 フォートミードに建設中のデータセンター自体は、2016年にのみ立ち上げられます。 しかし、このデータセンターの全費用は8億6,000万米ドルです。



LuleaのFacebook Arctic Data Center









このデータセンターについても書きました 。FacebookのDCは、PUEが1.07で、世界で最も（最もではないにしても）グリーンデータセンターの1つです。 これは、フィンランドのGoogleのデータセンター（PUE = 1.1）よりも優れています。



ちなみに、ハバソコミュニティから聞くのは面白いでしょう。おそらく、さまざまな種類のエネルギー効率の高い技術を使用する他の「グリーン」DCを知っていますか？ それらについて読むことは興味深いでしょう、私たちはコメントで議論を手配することを提案します。
