Microsoftの無料のMorro Antivirus



マイクロソフトは、コード名「Morro」と呼ばれる無料のウイルス対策ソフトウェアの開発を数か月以内に完了します。 新しいウイルス対策は、 Windows Live OneCareサービスを置き換えることが期待されています 。 6月30日、マイクロソフトはそのライセンスの販売を停止する予定です。 Windows Live OneCareには、自動更新されるデータベースを備えたウイルス対策、スパイウェア検出ツール、ファイアウォール、情報バックアップツール、およびWindowsオペレーティングシステムを微調整するためのユーティリティが含まれることを思い出してください。



「モロ」の開発について少し



マイクロソフトは、2008年11月にアンチウイルス/スパイウェア対策を無料にするという最初の声明を発表し始めました。その後、企業はセキュリティソリューションのニーズの高まりに応えるために独自の製品を開発することを決定しました。



新しいマイクロソフトソフトウェアの名前は、ブラジルのサンパウロの村にちなんで付けられました。




Morroは2009年後半に無料で利用可能になる予定でした。 開発者は、この製品は低帯域幅ネットワークとWindowsベースのモバイルコンピューター向けに最適化されると主張しました。 既存のWindows Live OneCareサービスを利用しないために、同社は2009年6月30日にサブスクリプションの終了を発表しました。



また、新しいソリューションが企業のタスクに対処できることにも留意されました。 ただし、Morroには（OneCareの場合のように）追加のツールは含まれません-システム用の非セキュリティバックアップおよび微調整ツール。



ユーザーにアンチウイルスを課さないために、それは独立したパッケージとして配布され、Microsoft Webサイトから自由にダウンロードできます。 Windows XP、Windows Vista、およびWindows 7のサポートが発表されました。



途中のモロ



現時点では、企業によると、Morroは今年の6月1日からMicrosoftによる内部テストを受けています。 ただし、ベータ版（またはそのようなもの）が会社の公式Webサイトでも利用可能になることが約束されています。 マイクロソフトは、シマンテックとマカフィーのソフトウェアで新製品の主要な競合他社を見ています。 しかし、それらの年間サブスクリプションが年間約40ドルであり、Morroが無料で配布されることを考えると、Microsoftにはウイルス対策ソフトウェア市場で一定のシェアを獲得するチャンスがあります。 当然、将来のウイルス対策製品は、あらゆる種類のウイルス、スパイウェア、ルートキット、トロイの木馬に対する基本的な保護を提供します。



最初のスクリーンショット？



最近、モロのスクリーンショットがウェブに登場しました。 しかし、これまでのところ、彼らは特に信頼されるべきではないと思います。 スクリーンショットは製品開発の初期段階のみを反映していると考えられています。 以下は、ディスクをスキャンするためのオプション（高速、完全、および任意）、ウイルスデータベースの関連性のステータス、および自動スキャンのスケジュールです。



























Morroインターフェイスは、現在のWindows DefenderであるWindows Vistaファイアウォールに似ていることに気付くかもしれません。 マイクロソフトはこれを否定しません。 Simon Bisson氏によると、MorroはDefenderコードに基づいているため、インターフェイスに類似点があります。
