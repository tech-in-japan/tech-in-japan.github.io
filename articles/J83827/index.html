  • Microsoft .NET Framework 4クライアントプロファイルRC （69.7MB）

  • Microsoft Visual Studio Test Professional 2010 RC-ISO （408.4MB）

  • Visual Studio Agents 2010 RC-ISO （514.7MB）
  • Microsoft Visual Studio Team Foundation Server 2010 RC-ISO （1568.7MB）
  • Visual Studio 2010および.NET Framework 4トレーニングキット-2月リリース （199.4MB）
