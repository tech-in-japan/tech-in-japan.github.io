スマートウェポン-ブロックチェーン、クラウド、IoTの子

過去20年にわたり、技術者は武器の安全性の問題を解決するために多くの努力をしてきましたが、理由もなく：2015年に米国だけで約300人が銃器で死亡しました。 彼らの努力は、BluetoothおよびWi-Fiテクノロジー、無線周波数センサー、モノのインターネット （IoT）の開発に歩調を合わせました。 それにもかかわらず、物事はまだそこにあります。







モノのインターネットとブロックチェーンが武器にどのように関係しているかを説明する前に、武器自体のトピックに飛び込みましょう。 今年の4月に、オバマはスマート兵器の研究に資金を割り当てました。 スマート武器は、そのような武器を持つべきではない人々、つまり泥棒、子供、または不正ユーザーの手による暴力を防ぐための技術的解決策です。 スマートガンは、許可されたユーザーによってアクティブにされた場合にのみ発射できるようにするセキュリティ機能を備えた銃器です。 これらのセキュリティ機能は、誤用、事故、武器の盗難、所有者に対する武器の使用、自傷行為を防ぐことができます。 スマートウェポンは、 RFIDチップやその他の近接マーカーの使用、指紋認識、機械的ロックなど、さまざまな方法で許可ユーザーと許可ユーザーを区別できます。







ブロックチェーン-セキュリティの質問への答え





インテリジェントな武器が出現する前は、IoTデバイスの分散型インフラストラクチャがなかったため、武器などのスマートデバイスにはセキュリティリスクが多すぎました。 知的に、武器はこれらの障壁を破ります。



ブロックチェーンシステムは、一元化された制御とレポートなしで銃器を有効化、無効化、追跡、無効化する機能を提供します。 これは主に、ソフトウェアを作成し、メーカーや3D印刷プロジェクトが銃器のセキュリティと防衛能力を高めるために使用するハードウェアスマートデバイスをサポートするインフラストラクチャを開発するプロジェクトです。 これは、Lisk、BitTorrent、Telehashの組み合わせを使用するピアツーピアネットワークです。



簡単に言えば、武器が発砲すると、「ショットスポット」テクノロジーがデータを検知して記録します。 緊急要員も通知を受け取ります。 いくつかのショットスポットが連携して、武器の検出と拡散を防ぎます。



スマートデバイスが特別なアプリケーションを介して最初に設定されると、所有者のBluetooth認証が設定され、承認されたすべてのユーザーがデバイスの所有者によって追加されます。 それらに関するデータはローカルに保存されます。 認証方法は、デバイスとメーカーによって異なる場合があります。 銃に残っているチャージ数の計算と認証パラメーターは、デバイスの所有者が記録したい他のデータとともにリアルタイムで保存されます。 たとえば、デバイスのローカルレジスタで暗号化を使用して固定された日付、時刻、場所を指定できます。



デバイスがインターネットにアクセスすると、データはネットワークと同期され（デューコンピューティングからのものがありますか？:)、デバイスの所有者は、選択した設定に従ってプッシュ、SMS、電子メールなどの通知を受け取ります。 データは、ネットワーク上のデータベースキーを使用して暗号化されます。 そのため、所有者から受け取ったデバイスのデータを持っている人だけが、そのデバイスの情報を表示できます。



このような武器ネットワークを作成する目的は、イベントロギングやリモートコントロールなど、デバイス所有者のみが実行できるスマートデバイスの機能をサポートすることです。 たとえば、武器に対してスマートロックを使用できます。これにより、所有者に武器に対する不正な操作を警告します。 これは、デバイスの所有者がその使用と配置を制御するのに役立ちます。



保護するが、「重量」ではない



この技術には、銃に取り付けられたロックが含まれます。 スマートデバイスのカスタムプロトタイプが開発されていますが、ネットワークとインフラストラクチャの開発と保守に重点が置かれています。 ネットワークは、SDK、 API 、および他のメーカーのサポートを提供します 。



したがって、インテリジェント兵器開発者の目標は、安全な分散インフラストラクチャを作成して、銃器の違法使用に関する活動を減らすことです。 スマート武器は、武器の盗難や所有者に対する不可抗力状況での使用を防ぐことができます。 残念なことに、そのような武器の開発は、あらゆる点で米国議会によって妨げられています。米国議会は、スマート武器がその時点で、そして自らの意思で武器を取得、携帯、使用する人権を侵害していると考えています。



前任者



ちなみに、最新の開発のプロトタイプはO'Dwyer VLe電子兵器であり、これには生体認証技術が含まれています。 生体認証技術は、NJITが特許を取得しており、子供に安全で、1人の所有者に適合した「非個人化」銃を作成できるため、他の誰も使用できません。 これは、オーストラリアで開発された新しい弾道技術が使用された世界で最初の100％電子ピストルです。ピストルには可動部品がなく、別個の店舗がなく、弾薬がありません。



7発のピストルは、1トリガープルで1分間に最大60,000ラウンドの速度で複数のラウンドを撮影できます。 ピストルの設計には、電子キーに基づく生体認証システムが含まれます。これは、特別なトランスポンダーリングを着用する許可ユーザーのみが認識します。 電子銃は、追加要素としてではなく、武器のオリジナルデザインでのパーソナライゼーションテクノロジーの使用開始の先例となりました。
