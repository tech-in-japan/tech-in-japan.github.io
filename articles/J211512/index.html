IBMはX-Architectureの第6世代を導入しました





1月16日、IBMはSystem xおよびPureSystemsサーバー用の新世代のアーキテクチャを発表しました。 X6のアーキテクチャはパフォーマンスを大幅に最適化できるため、クラウドコンピューティングや分析などの使用分野を含め、x86ファミリサーバーの収益性を最適化できます。



ビジネスインテリジェンスについては、企業はますます分析データを使用して、自分の仕事に役立つ情報を取得しています。 さらに、過去数年間で、多くのビジネスタスクがクラウドに移行しました。 まず第一に、これは効率を高め、コストを削減するために行われます。 最適なプラットフォームとして、ビジネスの代表者はしばしばx86サーバーを使用します。



新しいX6アーキテクチャは、IBMが新しい研究開発に投資した結果として生まれました。 新機能に関しては、新しいアーキテクチャには多くの機能があります。 これは、サードパーティのメーカーよりも大量のRAM（3倍）を使用する能力、および作業速度などです。





IBM System x3950-X6で3倍のRAM



X6アーキテクチャの主な機能は次のとおりです。





• 速度 。 EXFlash統合ストレージDIMMは、業界初のDIMMベースのサーバーストレージサブシステムであり、フラッシュメモリ用に最大12.8テラバイトの超高速ディスクサブシステムを提供し、アプリケーションパフォーマンスを向上させ、ビジネスインテリジェンスアプリケーションに必要な最速の書き込み速度を提供します。 X6は、データベース操作の処理におけるレイテンシを大幅に削減し、必要性を減らすか、SAN / NASストレージの使用を完全に排除することにより、ライセンスコストとデータストレージコストを削減します。 （2）



• 柔軟性 。 数世代のプロセッサをサポートするスケーラブルなモジュール設計により、競合他社と比較して取得コストが最大28％削減されます（3）。 X6アーキテクチャは、革新的な技術開発により安定性と柔軟性を提供し、ユーザーがシステムをリアルタイムで拡張し、将来的に機器を効果的にアップグレードできるようにします。 セットアップと構成のクイックダウンロードにより、インストールとシステムライフサイクル管理が簡素化されます。



• フォールトトレランス 。 X6アーキテクチャの機能により、クラウドベースのモデルを使用して、最も重要なアプリケーションで動作するITを構築できます。 プロセッサおよびメモリシステムの復元機能は、潜在的な問題を事前に特定し、それらに対処するための決定を独自に行うことにより、アプリケーションの稼働時間を最大化します。 さらに、Upward Integration Modulesはコストを削減し、管理プロセスを大幅に簡素化します。







新しいアーキテクチャで設計されたサーバーソリューションのポートフォリオには、4プロセッサのSystem x3850 X6サーバー、8プロセッサのSystem x3950 X6 、およびスケーラブルなコンピューティングノードIBM Flex System x880が含まれます。





IBM System x3850 X6 with SSD



さらに、IBMはSystem x3650 M4 BDストレージサーバーを導入しました。これは、最大14の内部ディスクをサポートするデュアルプロセッサラックマウントサーバーで、合計ストレージ容量は最大56テラバイトです。 今日の業界では、これが単一サーバーでのデータストレージの最大量です。 System x3650 M4 BDは、以前のIBM System xサーバーモデルよりも最大46％生産性が高く、ビッグデータ処理の分散ワークロードに最適です。 （4）



X6の新しいソリューション





エンタープライズアプリケーションをクラウドに移行し、分析を使用してビジネスパフォーマンスを評価するお客様には、迅速な展開、効率性、生産性の向上のための統合ソリューションが必要です。 これらの目標を達成するために、IBMは、分析、データベース、クラウドコンピューティング向けのX6アーキテクチャに基づく新しいソリューションを作成しました。 ソリューションポートフォリオには、X6でのBLUアクセラレーションを備えたDB2用のIBM System x Solutions、X6でのSAP HANA向けのIBM System x Solutions、およびIaaS向けのX6でのVMware vCloud Suite用のSystem x Solutionsの強力なシステム分析システムが含まれます。
