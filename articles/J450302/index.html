アマチュア無線技術：中国の工場でのプロトタイプPCB生産







出版の目的 ：最小限の時間とお金で簡単なプリント基板を作る方法を教えます。







すでに書かれているように、アマチュア無線の創造性には家族の予算が限られており、家族からの時間に制限があります。 CAD / CAMの使用は、ラジオ愛好家がこれらの制限に対処するのに役立ちます。 さらに、CADはホームコンピューターにインストールでき、CAMは世界の別の場所に配置できます。







この記事では 、中国のCAM用にGerber RS274X形式のジョブをCAD Eagle 7.7.0から準備およびアップロードする方法について説明します。 ロシア連邦でのプリント回路基板の製造とその出荷の注文方法。







問題の声明



国内企業での1つのプロトタイププリント基板の緊急生産では、緑色のはんだマスク、コーティングされたPOS-63（HASL）コンタクトパッド、ホワイトシルクスクリーン印刷を備えた100 * 80および厚さ1 mmのボードの場合、4〜5営業日と2823.14ルーブルがかかります片側。 送料を除く。







それは私には少し高価に思えました。 そのような場合に通常起こるように、私は同僚に良いアドバイスを求めましたが、同僚は中国のアマチュアプロジェクト用に同様のプリント回路基板を注文し、そのような基板の製造コストはLUTに匹敵するほど優れた品質であると言いました。







これらの紹介文により、問題文は次のようになり始めました。







  1. プロトタイプの優先価格で中国のPCBメーカーを選択します。
  2. 開発したボードがこのメーカーの技術要件に準拠しているかどうかを確認してください。
  3. プリント回路基板を製造するためのジョブを希望の形式でCADからアンロードし、ダウンロードしたジョブを確認します。
  4. 注文し、支払い、配達を手配します。


技術的要件を考慮したメーカーの選択



中国には、同様のサイトと同様のサービスの同様の価格を持つプリント回路基板のメーカーが多数あります。 たとえば、 pcbgogo.comを選択しました 。













上記のスクリーンショットから、5米ドルで、寸法が100 * 100以下で、厚さ0.6〜1.6 mmの5枚（10枚）の両面印刷回路基板を、緑の（だけでなく）はんだマスクとシルクスクリーン印刷で製造する準備ができていることがわかります両面。 パッドはHASLでコーティングされており、最小トラック幅とトラック間隔は6ミル（0.1524 mm）、最小穴径は0.3 mmです。







スクリーンショットのプリント基板の寸法は100 * 80 mmです。 これは、長年にわたってホームプロジェクトで使用してきたCAD Eagle 7.7.0の無料バージョンの制限によるものです。 メーカーの上記の技術要件は、Eagle 7.7.0のデフォルト設定で満たされています。







チェックアウト、送料の節約



サイトに登録します。メールアドレスを入力してください。 電話番号 音訳での配達先住所と氏名（ロシア連邦の居住者にはミドルネームが必須）。







PCB Instant Quote Webサイトタブで、ボードの寸法（100 * 100 mm以下）を入力し、5または10の順序でボードの数を選択します。ボードが2層、ボードの厚さが0.6 ... 1.6、はんだマスクの色であることを示します紫やマットではありません。 HASLをENIGに置き換えることに同意しません。 残りはデフォルトです：













「計算」ボタンを押します。 ロシアでの配送は、Eパケットサービスで選択します。 ボードの数とその厚さを試してみると、たとえば、ロシアに出荷する場合、100 x 80 mmのボードを5枚、厚さ1 mmで8ドル、10-10ドルで出荷することがわかります。 この事実を覚えておいてください。







注文はバスケットに入れます。 ガーバーファイルを含むzipアーカイブをオーダーにアップロードします。













ガーバーファイルを確認した後にのみ、注文の支払いに進むことができます。 それまでの間、注文の詳細を確認します。













pcbgogo.comでの注文の支払いは、PayPal、Western Union、および銀行振込で可能です。 どこでも手数料。 考慮された例のPayPal支払い構造：5 USD（36％）で5つの手数料を支払う、8 USD（57％）でEパケットサービスによってロシア連邦に送信する、PayPal手数料1 USD（7％）。 ルーブルでは、これは1000未満ですが、オーバーヘッドの178％を取得します。







送料を節約しようとします。 10枚のボードを出荷すると10米ドルの費用がかかり、この順序で別のプロジェクトからさらに5枚のボードの製造が含まれることを思い出します。













現在、支払いの構造は次のとおりです。手数料2 * 5 USD（48％）の生産、Eパケットサービスによるロシアへの配送10 USD（48％）、PayPal手数料1 USD（4％）。 すでにオーバーヘッドの108％を取得しています。 これは178％を大幅に上回っています。







仮説を確認し、注文から2番目のボードを削除しました。







注文の支払い後、回路基板の製造には1日半かかります。 サイト上の特別な形式で、プロセスチェーンに沿ったボードの通過を追跡できます。







EパケットサービスによるPermへの注文の配送には、6〜15日かかります。 納期は、税関検査の速度に大きく影響されます。







これで、中国でのプリント基板の製造に関する注文についての私の話は完了できますが、...







トピックGERBERは非公開



ガーバーファイルをオーダーにロードする前に、これらが正確にファイルであり、PCBプロジェクトからのガーバーファイルのアンロードが成功したことを確認することは不要です。 これを行うには、製造元のWebサイトでOnline Gerber Viewerを使用できます。













スクリーンショットは、レイヤーファイルの名前を示しています。 これらのファイルにそのような名前と拡張子がある理由は、「kat」の下で読むことができます。







Eagle 7.7.0での生産のためのPCB設計の準備

CAMプロセッサの構成



例からlpf.brd回路基板プロジェクトを開き、[ファイル]メニューから[CAMプロセッサ]を選択し、プロセッサの[CAM]ウィンドウに移動します。 タスクrs274x.camを開き、以下を取得します。













タブジョブの設定は次のようになります。







セクション 装置 ファイル名 レイヤー
copper_top GERBER_RS274X ％P / RS274X /％N.copper_top.cmp トップ、パッド、ビア
はんだマスク_トップ GERBER_RS274X ％P / RS274X /％N.mask_top.stc tStop
silkscreen_top GERBER_RS274X ％P / RS274X /％N.silk_top.plc tPlace、tNames
copper_bottom GERBER_RS274X ％P / RS274X /％N.copper_bot.sol ボトム、パッド、ビア
はんだマスク底 GERBER_RS274X ％P / RS274X /％N.mask_bot.sts bStop
silkscreen_bottom GERBER_RS274X ％P / RS274X /％N.silk_bot.pls bPlace、bNames
ドリル EXCELLON_24 ％P / RS274X /％N.drill.drd ドリル、穴
アウトライン GERBER_RS274X ％P / RS274X /％N.outline.gml 寸法


ガーバーファイルをアップロードするためのRS274Xフォルダーは、プロジェクトフォルダー内に手動で作成されます。

％Pマクロは、ガーバーファイルの名前のプロジェクトフォルダーへのフルパスを置き換えます。

macro％Nは、ダウンロードが実行される元のガーバーファイルの名前でPCBプロジェクトファイルの名前を置き換えます。

次に、ガーバーファイルの名前が、理解しているレイヤーの説明に置き換えられます。

ガーバーファイル拡張子は、CAD Eagle 7.7.0のこれらのレイヤーのファイル用に選択された標準です。







タスクはProcess_Jobボタンによって起動されます。 16個のファイルがRS274Xフォルダーにダウンロードされ、そのうち8個の拡張子が.dpiと.driのファイルは安全に削除できます。







残りの8つのファイルには、Gerber RS274X形式のPCBレイヤーデータとExcellon形式の穴データが含まれています。 製造元のWebサイトのオンラインGerber Viewerは、これらのファイルを受け入れ、追加設定なしでレイヤーに配布します。







ガーバーファイルが正しく読み取られることを確認し、これら8つのファイルをzipアーカイブにパックします。 これで、生産のためのPCB設計の準備が完了しました。







そしてもう1つ、PCBプロジェクトからファイルをアップロードする前に、ガーバーファイルは、このプロジェクトがメーカーの技術的要件に適合しているかどうかを確認することを妨げません。







プロジェクトの技術要件を設定する



製造を成功させるには、プリント回路基板の設計がメーカーの技術要件を満たしている必要があります。 技術要件Eagle 7.7.0には、PCBデザインに添付された* .druファイルが含まれています。

簡単な要約



この出版物では、個人的な例が検討されています。生産のためにプリント回路基板プロジェクトを準備するプロセス。 中国でのプリント基板製造の注文プロセス。







中国の工場で最小限の時間とお金で簡単なプロトタイプのプリント回路基板の製造を注文するには：







  1. メーカーの技術要件に合わせてCADを構成したら、これらの設計要件（設計ルール）を将来の使用に備えてファイルに保存します。
  2. PCBプロジェクトをGerber RS274X + Excellon形式にアップロードするプロセスをCADで一度構成し、将来の使用のためにタスクをファイルに保存します。
  3. PCBデザインをガーバーにアップロードする前に、メーカーの技術的要件に準拠しているかどうかデザインを確認してください。
  4. PCBデザインをガーバーにアップロードした後、ビューアでアンロードの結果を確認してください。
  5. 1つのEパケットで複数の注文を行います。
  6. PayPalアカウントを使用して、注文の支払いを行います。
