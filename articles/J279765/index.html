無料の開発者機能パック：Visual Studio Dev Essentials





みなさんこんにちは！



一般的な考えに反して、Microsoftの無料の開発ツールには、豊富で強力なIDE Visual Studio 2015 Communityだけでなく、他の便利なコンポーネントや製品も数多く含まれています。 たとえば、 Visual Studio Team Servicesには、5人以下のチーム向けの独自の無料オファーがあります。 Visual Studio Dev Essentialsを使用するすべての開発者は、これらおよびその他の多くのオファーを利用できます。



2015年11月に発表された新しいVisual Studio Dev Essentialsプログラムは、 Visual Studioの開発者がメインの無料コンポーネントに集中的にアクセスできるようにし、Parallels、Xamarinなどのサードパーティ企業のツールや製品など、完全なソフトウェア開発サイクルを提供するさまざまなツールも提供しますなど

コンポーネントの概要



プログラムにアクセスするには、Microsoftアカウント（以前のLive ID）が必要です。 ここで登録できます 。



Visual Studio Dev Essentialsポータルに入った後、ページの最初に、最も興味のあるオファーを記録できます。 デフォルトでは、Visual Studioファミリーツールがここで提供され、Microsoft Azureクラウドプラットフォームの無料提供などが提供されます。



各プログラムコンポーネントについて簡単に説明します。



Visual Studio 2015 CommunityおよびVisual Studio Team Servicesに加えて、プログラムポータルで、 Visual Studio Codeをダウンロードできます。これは、Windows、OS X、およびLinuxオペレーティングシステムで利用可能なクロスプラットフォームコードエディターです。 40以上の言語、MonoおよびNode.jsのデバッグ、および他の多くの追加機能をサポートしています。 Visual Studio Marketplaceのコンポーネントを使用した機能拡張を含みます。



このオファーで利用可能なMicrosoft R Serverコンポーネントの開発者バージョンを使用すると、Windows、Hadoop、Terradata、およびLinux上でR言語で強力な分析ソリューションを作成できます。



Microsoft Azureクラウドプラットフォームの試用期間は1か月であり、Dev Essentialsのメンバーへのオファーは、ポータルで提供されるサービスの使用に対して年間25ドルの月額の提供を保証します。 Azure App Service -Azureサブスクリプションで利用可能なサービスには、Webアプリケーション、モバイルバックエンドなどの作業が含まれます。 機能の一部を無料で提供します。



Dev Essentialsメンバー向けの限定Pluralsightオファーには、さまざまなテクノロジーのオンライン学習を提供するグローバルプラットフォームから6か月間のトレーニングコースへのアクセスが含まれます。 Xamarin大学のモバイルテスト -Xamarinの専門家によるC＃でのiOSおよびAndroidアプリケーションの開発に関するトレーニングビデオコースと手順-Xamarin University Mobile Testingをご覧ください。



ツール







Application Insightsは、Microsoft Azureクラウドプラットフォームのクロスプラットフォームコンポーネントであり、開発者は開発されたサービスとアプリケーションの実行可能性とパフォーマンスに関する最新情報を無料で受け取ることができます。



Hockey Appは、ベータテスト中のモバイルおよびデスクトップアプリケーションの動作、アプリケーションのテストビルドの配布、クラッシュレポートおよびフィードバックの収集中のタイムリーな情報を取得するために、マイクロソフトのメインサービスとしてモバイル開発者に役立ちます。



Windows 10（UWP）のユニバーサルプラットフォームの開発に精通したい人のために、Dev Essentialsプログラムには、Hyper-V、VMWare、VirtualBox、Parallels用の使い慣れた仮想マシンの形式でUWPアプリケーションを開発するために必要な既製の環境が含まれています。



Macユーザーは、Dev Essentialsパーソナルポータルページでコンピューター上の任意のプラットフォームのアプリケーションを開発およびテストできます。ParallelsDesktop for Mac Proの 3か月のサブスクリプションと、 Parallels Accessの同様のオファーを取得できます。



Power BIは 、開発者が便利な場所やあらゆる種類のデバイスでデータを便利かつ明確に提示するのに役立ちます。



Dev Essentialsプログラムの以下の提案は、コンピューターにソフトウェアをインストールすることなく、同時に完全に無料で、新しいOffice Onlineの機能を使用して、ブラウザーで直接プロジェクトドキュメントでチームと共同で作成、編集、作業できることを思い出させます。 そして、たとえば、OneDriveクラウドストレージに配置して保存します。

たとえば、プログラムの参加者は、Microsoft開発者を含むWintellectNOWの専門家からのコースへの3か月のサブスクリプションにアクセスできます。 マイクロソフトの専門家やパートナーからの膨大な数のトレーニングコースを含む、 マイクロソフトのお気に入りの教育プラットフォームであるMicrosoft Virtual Academyもポータルに表示されます。
