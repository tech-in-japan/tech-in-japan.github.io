モバイル開発者向けの5つの便利なツール





投稿者：Grigory Kharchenko、モバイル開発責任者、Kokoc Mobile（Kokoc Group）。



アプリアニー



彼は、モバイルアプリケーションのマーケティング分析を専門としています。 このサービスは、最大の世界市場のすべての製品をスキャンし、国ごとのカテゴリの評価を表示します。 そのため、アプリケーションの公開から翌日までに開発者は、特定の評価でどのような場所を取るのかを知ることができます。 重要なのは、App Annieは、あなた自身だけでなく、他の人のアプリケーションも追跡することを可能にします：リリースがいつ行われたのか、トップのどこを占めるのか、それが更新される頻度、どれくらいのお金をもたらすのか、どの国で人気があるのか​​。







Git



コードバージョンコントローラー。 このツールのおかげで、開発者はコードに加えられたすべての変更を確認し、必要に応じてそれらの1つに戻ることができます。 また、すべての情報はサーバーに保存されるため、世界のどこからでもアクセスできます。







テストライト



iOSアプリ開発者の間で人気のあるサービス。 独立したチームによって作成され、しばらくの間、独立したプログラムとして機能していました。 Appleは2014年の初めにそれを購入し、現在ではiTunesに統合されたネイティブツールです。 TestFlightは、iOSアプリケーションをApp Storeに公開する前にテストするのに役立ちます。使用するには、製品を登録し、「テスターを招待」ボタンをクリックして、メールアドレスを追加します。 その結果、アプリケーションのベータ版では、開発者だけでなく顧客もリモートで確認およびテストできます。







レッドマイン



タスク管理用に作成されたアプリケーション。 複数のプロジェクトを実行し、RSSフィードと電子メールを使用してそれらの変更を通知し、エラーを追跡し、時間コストを考慮することができます。 重要なのは、Redmineには柔軟な役割ベースのアクセスシステムがあり、新しいユーザーが自分で登録できることです。







解析



開発者がiOSとAndroidの両方にプッシュ通知を送信するために使用するサービス。ユーザーに製品ニュースを通知し、それらとの対話を維持します。 このツールを使用すると、プッシュ通知のA / Bテストを実行し、コンテンツが最も効率的に機能する通知を選択し、プッシュキャンペーンの成功を分析し、クラウドストレージサービスを使用できます。
