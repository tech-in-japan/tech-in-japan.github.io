Windows Phoneでの新しい2GISのベータ版（オフライン！）





ユビキタスなWi-Fi、LTE、クラウドテクノロジーの時代には、オフラインアプリケーションを放棄できると考える人もいます。 インターネットがどこにでもあるとは限らず、常に同意するとは限らないことを知っている実際に存在する人々。 Windows Phoneストアでの3千件を超えるレビューと、これのオフライン2GIS確認のリクエスト。



インターネットがないと思われるコンピューターユーザーでさえ、MacおよびLinux用のオフラインバージョンを作成するように求められました。 ただテストしています。 モバイルデバイスでは、接続の問題がはるかに一般的です。



Windows Phoneは、2GISがインターネットなしで機能しなかった唯一のプラットフォームでした。 はい、アプリケーションはありますが、2GIS APIで動作し、ネットワーク接続が必要です。 なぜそれが起こったのか-長い話であり、一部はすでにHabréで語られている 。

新しいデザイン

もう一度、モバイルデバイスでディレクトリとマップを操作するシナリオを再考しました。 彼らは、コンピューター用の新しい2GISの開発で得た経験を考慮に入れました。 その後、彼らはプラットフォームのガイドラインに従って、UXを便利で快適にするインターフェースを作成しました。



詳細な三次元地図

新しい3次元エンジン-マップを回転、接近、移動する際の滑らかさ、およびマップへのテーピング時の遅延がありません。 フェンス、門、モニュメントに至るまでの包括的な詳細と、有名な建物の3Dモデルとの組み合わせで、これは直火です！



組織の実際の詳細なディレクトリ

住所、勤務時間、連絡先、一般に、組織の選択に必要なすべてのもの。 支払い方法と料理の種類から、場所の数とWi-Fiの可用性まで。 フィルタ、レビュー、写真に加えて、適切な会社を選択するのに役立ちます。 企業に関するこの知識のおかげで、「グルジア料理のカフェ」を見つける方法を知っています。



バックグラウンドの更新

最初、2GISには2種類の更新が必要でした。アプリケーション自体の場合と、ディレクトリを含むマップの場合です。 ご存知のように、これは不便です。 最初の問題がアプリケーションによって自動的に解決された場合、2番目の問題は頭痛の種でした。 新しいベースをセットアップするのに数分かかり、失敗した状況では数十分かかりました。 現在、2GISは特別な注意を必要とせずにバックグラウンドで更新されています。
