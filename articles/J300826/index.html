ロシアのオンラインストアはどのようなパーソナライズサービスを使用していますか？ 事例と例





売上とコンバージョンを増加させようとすると、おそらくRunetの広大な地域から多数のメソッドとサービスを試してみました。



  • あらゆる種類の「コールバックハンター」と「ソープコレクター」をテストしましたが、迷惑なポップアップウィンドウと無関係なオファーは顧客をはじき、サイトを常駐の悪に変えました。
  • ロイヤルティおよび販売プログラムを開始しました-売上高は増加しましたが、利益（明らかな理由により）は大幅に増加しませんでした。
  • 彼らは顧客志向になり、ソーシャルメディアで個人的なアプローチと顧客とのオープンなコミュニケーションを組織しましたが、顧客からの熱烈な愛の代わりに、彼らはコミュニティで多くの否定を受け取りました。
  • 私たちはHabréでコンバージョンを高めるためのステップバイステップガイドを読み、多数のA / Bテストを実施し、すでに最大2％のコンバージョンに達しているようです。


しかし、今日の2％のコンバージョンはロシアのeコマースにとって十分ではなく、この数字を1.5倍、さらには2倍に増やすことができると言うとどうなりますか？ この記事では、ロシア市場で最近積極的に開発され始めた、比較的新しいツールであるパーソナライゼーションサービスについてお話したいと思います。 パーソナライゼーションサービスは、サイトの各訪問者に個別のオファーを表示するように設計されています。彼が本当に必要としているもの、今ここで購入するものです。 これらのサービスを使用する主な利点は、比較的簡単な実装（場合によっては無料の試用期間）により、オンラインストアのコンバージョンと利益が迅速かつ大幅に増加することです。



オンラインストアのパーソナライズは、行動やその他の情報に応じて、さまざまなコンテンツまたは機能、個々のオファーおよび機会をさまざまな訪問者（訪問者のセグメント）に示すプロセスです。



＃1 HucksterBotは、通常は購入しないユーザーに販売する割引パーソナライズサービスです。



動作原理：

システムは、オンラインストアへの訪問者に関するデータを収集し、行動を分析し、販売にあまり変換されていないユーザーのさまざまなセグメント（以前の広告チャネルを通じて、以前に閲覧した商品の割引、無料配送など）を提供します。キャンペーンなど）。 商品とカテゴリの割引のセグメントとサイズは、アカウントで構成されます。







費用：

製品の有効性を評価できる無料期間（30日間）があります。 有料関税は、サービスが提供する注文数に応じて計算されます。 小規模な店舗（1か月あたり最大10件のHucksterbotによる注文）の場合、このサービスは無料です。 サービスが1か月あたり最大100件の注文をもたらす場合、そのコストは1900ルーブル/月になります。



成功事例：

Garden Machinesは、サンクトペテルブルクのマーケットリーダーである園芸機器を販売する小売チェーンです。 Googleアナリティクスによると、ネットワークにオフラインストアがない地域でのコンバージョンは、ネイティブ地域の6倍でした。 会社は新しいアプローチを適用して売上を増やし、仮説を検証することを決定しました。コンバージョン率の低い特定の訪問者セグメント（地域、広告チャネル、モバイルユーザーを通じてサイトにアクセス）にパーソナライズされた割引を提供すると、コンバージョンが増加し、これらのチャネルが拡大します。 HucksterBotの導入により、地域販売による粗利益を2.7倍、売上を3倍に増やすことができました。 オンラインストアの全体的なコンバージョンは60％増加しました。 詳細なケースをお読みください 。



＃2 Personaliは、購入者がオンラインストアのWebサイトで直接交渉できる割引パーソナライゼーションサービスです。



動作原理：

このシステムは、サイト訪問者の行動を分析し、購入プロセスに関与させます。たとえば、潜在的な顧客に商品の代金を支払おうとする価格を入力してもらいます。 示された価格が店舗に適している場合、買い手は示された分だけ商品の代金を支払います。







費用：

Personaliは最近ロシア市場に参入し、ここで大規模なオンラインストア向けのエンタープライズソリューションとして位置付けています。 したがって、統合のコストは比較的高く、個別に計算されます。 製品のパイロットローンチが可能です。



「 私たちはサービスを6か月間テストしています-統合は2015年10月に完了しました 。」220ボルトの第一副部長Leonid DovladbegyanはKommersantに語りました。 コンバージョンと売り上げにプラスの効果があります。 」




＃3 RetailRocket-個人向け製品の推奨事項とメールマーケティングパーソナライズサービス



製品の推奨事項の動作原理：

システムはユーザーの行動に関するデータを収集し、データ分析と製品マトリックスに基づいて推奨事項を生成します。 さらに、特定の時点で、サービスは製品の推奨（ストアで最も人気のある製品、検索の推奨、類似製品など）を表示します。これにより、コンバージョンの増加、平均チェック、および繰り返し購入の頻度によりオンラインストアの収益が増加します。







費用：

RetailRocketは、小規模なオンラインストアの無料からエンタープライズソリューションまでの月払いでいくつかの料金を提供します。エンタープライズソリューションのコストは個別に計算されます。 さらに、CPOモデルによる支払いも可能です（推奨システムを介して行われた各注文に対する支払い）。



成功事例：

Allsoft.ruオンラインストアによるシステムの実装の有効性は、A / Bテストを実施することにより決定されました：サイトの全聴衆は等しいセグメントに分割されました。 1つはRetailRocket製品の推奨事項を示し、もう1つはオンラインストアプラットフォームに関する独自の推奨事項を示しました。 製品推奨の新しいシステムの導入により、店舗は収益を17％増加させることができました。 詳細な事例を読む。



＃4。 TriggMine-メールマーケティング自動化およびパーソナライズサービス



パーソナライズされたメールニュースレターの仕組み：

システムは、オンラインストアへの訪問者の行動を分析し、パーソナライズされたメールを使用して販売ファネルに誘導します。 ファネルのどの段階でユーザーが「スタック」したか、または実行したアクションに応じて、個々の文字がリアルタイムモードで生成されます。 ユーザーがバスケットを「放棄」し、いくつかの製品を見てサイトを離れ、いくつかの商品を購入し、関連商品などを購入する可能性がある場合、個人的なオファーのある手紙を送信できます。







費用：

TriggMineは14日間の試用期間を提供します。 料金は、1か月あたりのユニークサイト訪問者の数に依存します。 サイトの月間ユニーク訪問者が100,000人未満の場合、最低料金は300ドル/月です。 非常に大規模な店舗向けのエンタープライズソリューションは、個別に計算されます。



成功事例：

以前は手動の定期郵送のみを使用していた繰り返し購入を生成するための種子と苗の大規模なオンラインストア。 経営陣は、放棄されたバスケットの使用から始めて、よりパーソナライズされた自動レターの追加を試みることにしました。 その結果、同社は3ヶ月でさらに260万ルーブルを獲得しました。 「Habr」では、 この事件は年の初めに発表されました。
