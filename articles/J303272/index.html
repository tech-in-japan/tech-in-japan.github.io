カザンのAndroid開発者向け無料学校





2016年6月15日に、カザンのAndroid開発者向けの学校への生徒の登録が開始されます。



教師-e-LegionおよびDZシステムの開発をリードするGoogle Developers Group Kazanのリーダーは、3週間にわたってAndroidアプリケーションの開発の複雑さを生徒に教えます。 コースプログラムに含まれるもの：



  • 開発の現在の傾向：要件とアーキテクチャ。
  • クライアントとサーバーの相互作用：従来のA / B / CパターンからJobSchedulerまで。
  • RxJavaを使用したリアクティブプログラミング。
  • 最新のアーキテクチャパターン：MVPおよびMVVM。
  • Androidアプリケーションのテスト。




フルタイムで無料の授業料。 クラスは週3回開催されます。

各卒業生は、コースの完了を確認する証明書を受け取ります。 優秀な学生がe-Legionチームで働くよう招待されます。



ウェブサイトの詳細情報： http : //www.e-legion.ru/lp/android
