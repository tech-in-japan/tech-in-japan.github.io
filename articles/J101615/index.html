Firefox 4ベータ3リリース

Firefox 4.0 Webブラウザの3番目のベータ版がダウンロード可能です。



Firefox 4ベータ3の新機能： リリースノート



*ロシア語を含む34の言語で利用可能。

* Windows 7でのタッチコントロールのサポート。

*デフォルトのタブは、WindowsおよびOSXの上部にあります。Linuxでは、適切な設定を適用する必要があります。

*スマートロケーションパネルでは、既存のタブを検索して切り替えることができます。

* [停止]ボタンと[更新]ボタンは、Windows、Mac、およびLinuxでは1つにまとめられています。

*ブックマークバーはデフォルトのブックマークボタンに置​​き換えられました（元に戻すことができます）。



他の変更の中で：



* JavaScriptの値の新しい表現。Firefoxは「グラフィック」やアニメーションで使用される「重い」コードをより効率的に実行するようになりました。

* Web開発者はCSSトランジションを使用してコンテンツをアニメーション化できるようになりました。

*新しいレイヤーシステムにより、応答性とスクロールがより生産的になりました。

*より最適化されたエンジン、およびその結果、改善されたJavaScript。

* XPCOM要素の登録を変更して、プロセスの分離を改善しました。

*新しいアドオンマネージャーと拡張機能管理API（それらのユーザーインターフェイスは最終リリース前に変更されます）。

*外部関数JS-ctypesのインターフェースの大幅なAPI改善。

*プラグインAdobe Flash、Apple QiuckTime、またはMicrosoft Silverlightが失敗した場合のWindows、Linux、およびMacのブラウザークラッシュに対する保護。

* CSSトランジションの部分的なサポート。

* WebGLの完全サポートを実装（デフォルトでは無効）。

*ビデオフォーマットHD HTML WebMのネイティブサポート。

* WindowsでDirect2Dをレンダリングするための実験的なサポート（デフォルトでは無効）。

* Web開発者はWebsocketを使用できます。

* Web開発者は、API「HTMLストーリー」を使用してページをリロードすることなくURLフィールドを更新できます。

*フレームの「遅延」構造により、ページのよりレスポンシブなレンダリング。

*リンクの履歴の表示は非同期に実行され、ページの読み込み中の応答性が向上します。

*新しいHTML5パーサー。

*より多くのHTML5管理フォームのサポート。
