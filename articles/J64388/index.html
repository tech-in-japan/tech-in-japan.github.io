Java SDK 1.2.2、GWT 1.7リリース

変更点の完全なリストはこちらから入手できます 。



-最後に、ローカルデータビューアが登場しました。アプリケーションを作成する方がはるかに便利です。

-静的ファイルの数が3000に増加しました。

-ORMのインデックスなしフィールドのサポート
