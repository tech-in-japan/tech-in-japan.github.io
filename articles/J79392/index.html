Intelは別の値下げを発表

いつものように、これは新しいプロセッサのリリース後に発生し、Intelは前のラインの「古い」CPUの価格引き下げを発表しました。



この減少は、クラークデール家の公式発表直後の1月に発生します。 予算と中コストのソリューションは安くなります。 たとえば、Pentium E6500、E5400、およびE3300は価格が10ドル下がり、それぞれ74ドル、64ドル、43ドルかかります。



新製品に関しては、最初の5つのチップが以下の価格で販売され始めます。

Core i5-670-284ドル

Core i5-661-196ドル

Core i5-660-196ドル

Core i5-650-176ドル

Pentium G6950-87ドル



さらに、2月28日までにCore i7-930がリリースされ、価格は284ドルになります。3月には、1,000個分のCore i7-980Xがシェルフに表示されます。
