Opera TV Store：HTML5 TVアプリカタログ





「スマートフォンやタブレット向けのアプリケーションを開発している場合、Opera TV Storeは大きな前進となるでしょう」と語り、Operaは新しいプロジェクトOpera TV Storeの HTML5アプリケーションカタログを発表します。 彼のプレゼンテーションは、昨日ラスベガスで開催されたCES会議で行われました。



アプリケーションとドキュメントをテストするための完全に機能するOpera TVエミュレーターが、開発者向けのサイトにすでに投稿されています。 エミュレーターは、Opera Devices SDKのすべての機能をサポートしています。

  • HTML / CSS / JavaScript / DOM
  • SVG、キャンバス
  • W3Cウィジェット
  • HTML5 <video>および<audio>
  • 制限付きRAMシミュレーション
  • Opera Dragonflyでのデバッグ
Opera TV Storeの背後にある考え方は、大きなHDTV画面上で直感的で使いやすいインターフェースでHTMLアプリケーションへのアクセスを提供することです。 インターフェイスは、標準のD / Uリモートコントロールからの制御用に特別に最適化されています（ビデオを参照）。







アプリケーションカタログはOperaブラウザで作成され、アプリケーションはバックグラウンドで実行され、テレビ番組を見ながら（たとえば、新しいメールが到着したとき）通知を発行できます。 したがって、Operaブラウザーは、世界で最もクロスプラットフォームなブラウザーであることを改めて証明します。 すでに任天堂のWiiやPlaystation 3のゲームコンソール（Cell Linux経由）にも移植されており、現在はOperaもテレビで公開されています。



現在、ノルウェーの会社はOEMテレビメーカーと協議中です。 ソリューションの技術的完成度と、MIPSおよびARMプロセッサを含む最も最小限のハードウェアで実行できることにより、それらを賄うことができます。 テレビへのOpera TV Storeの実装は、開発ツールOpera Devices SDKの標準セットを使用して実行され、割り当てられたメモリの最大量を制限する機能（ Out of Memory System ）を含むすべての機能を備えています。
