Windows Phone Toolkitの8月の更新がリリースされました！





木曜日に、MicrosoftはWindows Phone用Silverlight Toolkitの8月の更新をリリースしました。 このライブラリの更新バージョンは、 ここからダウンロードできます 。





変更のリスト：

  • Windows Phone 7.1 OS用に設計されたスムーズな新しいLongList Selector
  • 複数選択リスト
  • ロック可能なピボット
  • ハッタイル
  • エキスパンダービュー
  • ローカライズされたリソース
  • DateTimeConverters
  • DateTimePickers
  • HeaderedItemsControl
  • PhoneTextBox
  • トランジションのパフォーマンスの改善
  • ContextMenuのパフォーマンスの改善
  • Listpickerは複数選択をサポートします
  • さまざまなバグ修正


ツールキットに含まれる要素：

  • AutoCompleteBox
  • ContextMenu
  • デイトピッカー
  • DateTimeConverters
  • デイトピッカー
  • エキスパンダービュー
  • GestureService / GestureListener
  • HeaderedItemsControl
  • ハッタイル
  • リストピッカー
  • ローカライズされたリソース
  • ロック可能なピボット
  • LongListSelector
  • 複数選択リスト
  • ページ遷移
  • PerformanceProgressBar
  • PhoneTextBox
  • チルト効果
  • タイムピッカー
  • ToggleSwitch
  • ラップパネル




UPD：MSIパッケージには最新バージョンのツールキットがありません（PhoneTextBoxだけは表示されませんでしたが、さらに違いがあるかもしれません）。「％Program Files％\ Microsoft SDKs \ Windows Phone \ v7.1 \ Toolkit \ Aug11 \ Bin」をコピーしますライブラリの新しいバージョン。 ソースからコンパイルするか、XAPから例を取得することで取得できます。
