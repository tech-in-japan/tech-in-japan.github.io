Skiff Readerは、本や新聞を読むためのデバイスです。 より大きく、より薄く、より柔軟に！

ブックリーダー市場は長い間新しいものではなく、消費者を驚かすことはすでにかなり困難です。 しかし、このデバイスは私にも驚きました。 一枚の絵はどんな言葉よりも優れていると思う：





明らかに、Skiff Readerのデザインにはガラスがありません。スクリーンはLGが提案した技術に従って金属箔に基づいて作られています。

画面サイズは11.5インチ（解像度1600x1200）で、物理サイズと解像度の両方でクラス最大です。ただし、デバイスは驚くほど薄く、約7 mmです。





読書コンテンツは、そのようなデバイス用に開始される特別なSkiff電子読書サービスを通じてネットワーク経由で受信されることになっています（デバイスはこのデバイスのラインを開きます）。





デバイス仕様：

*メモリ-4GBフラッシュ（ユーザーデータに3GB以上使用可能）

*メモリ拡張-SDHC 2.0互換SDカードスロット

*画面-11.5インチ（292.1mm）@ 1200 x 1600（174ppi）、タッチスクリーン; 柔軟な金属ベース上のシリコンTFT

*ワイヤレス-モバイル3G; Wifi

* USB-USB 2.0（ミニUSBポート）

*オーディオ-スピーカー; 3.5mmオーディオジャック

*バッテリー-充電時間が2〜3時間、実行時間が約1週間のリチウムイオンバッテリー（ネットワークを定期的に使用する数千ページ）

*寸法-9.0 x 11.0 x 0.27インチ（228.6 x 279.4 x 6.8 mm）

*重量-17.5オンス（498 g）

*オペレーティングシステム-Linux



このデバイスは、1月7〜8日に開催されるラスベガスのConsumer Electronics Show（CES）で世界に紹介されます。
