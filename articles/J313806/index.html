VMware vSphere 6.5：次世代ITインフラストラクチャ

VMwareは10月18日に、業界をリードする仮想化プラットフォームの最新バージョンであるvSphere 6.5のリリースを正式に発表しました。 新しいリリースの最も特徴的な機能は、ユーザーの作業を大幅に簡素化すること、強力な組み込みのセキュリティツール、およびアプリケーションを起動するためのユニバーサルプラットフォームとして使用できることです。







1. VMware vSphere 6.5：一般リリース機能



vSphere 6.5は、近年のデジタルトランスフォーメーション（デジタルトランスフォーメーション）およびクラウドコンピューティングへの顧客の移行を加速します。 プラットフォームの新しいバージョンは、次の規定に基づいて開発されました。



1.企業のIT環境はますます複雑になっています。

2. ITセキュリティに対する脅威は急速に増大しており、攻撃はより高度になっています。

3.従来のアプリケーションとサービスの両方をサポートする必要があります。



簡素化されたエクスペリエンス（劇的に簡素化されたエクスペリエンス）



vSphere 6.5は、ユーザーエクスペリエンスをより高いレベルに引き上げます。 このプラットフォームにより、管理が容易になり、運用効率が向上し、その結果、顧客を使用した市場へのアクセスが高速化されます。



バージョン6.5のvSphereの主要で基本的なビルディングブロックは、vCenter Server Appliance（VCSA）です。 vSphereアーキテクチャは、プラットフォーム設計の基本原則-簡単な展開と管理を実装しています。 主要な機能を単一の一意の場所に配置することにより、運用の複雑さが軽減されました。



vSphereホスト管理（完全に統合されたvSphere Update Managerを使用）、バックアップと復元、組み込みの高可用性VCSAなどの機能が、新しいユニバーサルモデルに含まれています。





vSphere Coreビルディングブロック



vSphere 6.5では、VM管理用の新しいRESTベースのAPIが導入され、アプリケーションの高度な仮想インフラストラクチャ管理が提供されます。 新しいAPIは、より少ないコード行でより多くのことを実行できるようになりました。



最後のコンポーネントは、グラフィカルユーザーインターフェイス（GUI）自体です。 待望の新しいHTML5ベースのvSphere Clientは、最新のユーザーフレンドリーで機能的なデザインです。 VMware LabsのFlingの一部として、多くのクライアントがこのvSphere Clientを既にテストしており、そのレビューはほとんど肯定的です。





HTML5ベースのvSphereクライアント：新しいGUIにより高速な作業とプラットフォーム間の互換性が実現



包括的なビルトインセキュリティ



vSphere 6.5は、安全なインフラストラクチャを自然に提供します。これは、単純なポリシー駆動型のオペレーティングモデルによるデータセキュリティとアクセスと同様です。 これらの3つの領域すべてを保護することは、企業の現代的なデジタル変革、およびあらゆるビジネスの発展に不可欠な部分です。



データへの不正アクセスを防ぐために、vSphere 6.5は仮想マシンレベルで新しい暗号化オプションを提供します。 VMwareのアプローチは、多用途でスケーラブルです。 使用するゲストOSに関係なく、VMドライブを暗号化できます。



戦略的なストレージ構造（vSphereストレージポリシーフレームワーク）によってスケーラビリティが確保されます。 vSphereは、新しいvMotion暗号化機能と組み合わせて、保存時と移動中の両方でデータを保護できます。



vSphere 6.5には、ユーザーアクションに関する詳細情報を提供する高度な追跡機能（ログ）もあります。 これにより、新たに発生した異常やセキュリティリスクを簡単に調査できます。



vSphere 6.5は、安全なソフトウェア定義のデータセンター（SDDC）の中核であり、他のSDDC製品とシームレスに統合し、ITインフラストラクチャの完全なセキュリティモデルを提供します。





vSphere 6.5完全組み込みセキュリティモデル-セキュアデータ、セキュアインフラストラクチャ、およびセキュアアクセス



ユニバーサルアプリプラットフォーム



vSphereは、従来型と新世代の両方で、企業で使用されるほぼすべてのアプリケーションをサポートするユニバーサルプラットフォームです。 明らかに、これらの2つのカテゴリのアプリケーションは互いにまったく異なりますが、どちらも適切なレベルの企業規模、パフォーマンス、および可用性を備えたインフラストラクチャの構築を必要とします。



バージョン6.5では、VMwareはvSphere Integrated Containersも導入しました。これは、vSphereユーザーが現在の既存のIT環境にコンテナーを追加する最も簡単な方法です。



その結果、コンテナの操作は仮想マシンの操作と同じくらい簡単になりました-プロセスの変更や使用したツールの交換は不要です。 つまり、VMware vSphere Integrated Containersは、既存のエンタープライズITインフラストラクチャを再設計することなく、必要なコンテナーを追加するのに役立ちます。 このサブシステムの3つの主要なコンポーネントは次のとおりです。



-Component Engineは、コンテナーでの直接作業を提供します（コアコンテナーランタイム）。

-ハーバーコンポーネントは、コンテナイメージを登録します。

-アドミラルコンポーネントは、コンテナ管理ポータルです。





vSphere統合コンテナアーキテクチャ



vSphere Integrated Containersは、VMware NSXおよびVMware Virtual SANと緊密に統合されたDocker互換インターフェイスを提供します。 これにより、クラス最高のネットワーク自動化とスケーラビリティ、およびデータストレージと管理が提供されます。 vSphere 6.5では、独自のエンタープライズインフォメーションセンター（オンプレミス）または外部パブリッククラウドを含む、あらゆるクラウドのアプリケーションを管理することもできます。





vSphere Integrated Containers：開発者と運用ITスタッフにとっての2つの世界のベスト



開発者向け-Docker互換インターフェース、コンテナ管理ポータル、コンテナ登録。



オペレータ向け-vSphereの使いやすさ。新しいツールを使用したり、新しいテクノロジを習得したりする必要はありません。ソフトウェア定義のデータセンター（SDDC）のすべての機能を活用できます。



ただし、vSphere 6.5はSDDCの「中心」だけではありません。 プラットフォームは、VMwareのクラウド戦略の中核です。 そのため、最近発表されたVMware Cloud FoundationとVMware Cloud AWSはvSphere 6.5に基づいています。



上記を要約し、新しいバージョンの主な利点を強調するには、次のように言えます。



-vSphere 6.5により、企業ユーザーはアプリケーションを実行および統合し、クラウドおよび機器を備えた単一のオペレーティング環境でセキュリティを管理および保証できるようになりました。



-現在のアプリケーション、クラウド、ビジネスに最適なプラットフォームとして、vSphere 6.5は、VMware製品に基づいて企業のITインフラストラクチャを構築するクライアントの投資を保護します。



2. VMware vSphere 6.5：新しいリリース機能



上記のように、VMware vSphere 6.5は、高度な自動化と追加の管理ツール、強力な組み込みセキュリティツール、コンテナーなどの新しいタイプのアプリケーションのサポートにより、簡素化を実装しています。



アプリケーションの場合、vSphere 6.5プラットフォームは現在、3Dグラフィックス、ビッグデータをカバーし、自然なクラウドサポート、コンテナー化された機械学習、およびSaaSを含んでいます。 リリースの主なコンポーネントは次のとおりです。



VMware vCenter Server Applianceは、vSphere環境を構築するためのビルディングブロックです。 すべての基本機能は単一の仮想アプライアンスに含まれています。



REST API-運用ITスタッフと開発者の両方の作業を簡素化し、改善します。 RESTテクノロジーに基づいた新しいAPIにより、最新のアプリケーションの自動化と仮想インフラストラクチャ管理を改善します。



VMware vSphere Clientは、HTML5で記述された新しいvSphere Clientです。 日常の操作を実行する際の生産性と使いやすさの面でユーザーの期待に応えるツールソフトウェアを使用することにより、システム管理を簡素化および改善します。



VM暗号化は、新しい仮想マシンレベルの暗号化ツールです。 VMware vMotionで移動される仮想マシンデータと同様に、メディアに保存されているデータへの不正アクセスから保護します。



セキュアブートは、vSphere環境への不正な画像編集とコンポーネントのロードを防ぐ新しいツールです。



VMware vSphere Integrated Containers -Docker互換のインターフェイスを提供し、vSphereのお客様が既存のインフラストラクチャを再設計せずにビジネスプロセスを再構築できるようにします。
