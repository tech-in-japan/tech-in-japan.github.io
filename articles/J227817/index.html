Android用の安全なIMの選択

私は安全なメッセンジャーを探し、人々がこれまでに何をしてきたかを見ることにしました。 準備の度合いが異なる20人以上のインスタントメッセンジャーをテストしました。 「彼らの」プロトコルを持つメッセンジャーのみが興味を持ちました。 XMPPの世界では、すべてが正常で期待されています。

私は主に、あらゆる形式の暗号化を使用したテキスト通信のためのインスタントメッセンジャーに興味がありました。 Google Playで「セキュアチャット」、「セキュアIM」、「セキュアコミュニケーション」、「セキュアメッセージング」のフレーズを検索しました。

無料のソリューションのみをテストしました。



tl; dr：Skypeはまだ代替品ではありません。



実際に、ここにすべての私の発見があります：



1. TextSecure

  • インターネットとSMSの両方で機能します。 対話者の認証なしで暗号化を使用し、指紋さえありません。 メッセージ内の鍵のアイコンに従う必要があります。 テキストについて通知しません
  • 登録するには、電話番号が必要です。 登録はオプションです（ただし、SMS経由でのみ機能します）
  • メッセージのみ
  • Androidのみ
  • オープンソース




2. TigerText

  • より多くのエンタープライズオプション。 電話番号とメール（「ドメイン用」モード）の両方でインターネット上で動作します。
  • 最大30日後のすべての送信および送信された自己破壊（デフォルトは5）
  • 対話者の検証はなく、主要なスナップショットもありません
  • メッセージ、写真、音声録音
  • 登録するには、電話番号またはメールが必要です
  • Android / iOS




3. チフリー

  • キーは最初の起動時に生成されます。 素敵なインターフェース
  • キーの指紋があります
  • メッセージ、通話、送信ファイルおよび連絡先。 グループチャット
  • 登録するには、電話番号が必要です
  • Google Playの説明には、AES256-CGM + 512ビット（？）ECDHキーが使用されていると書かれています
  • Android / BlackBerry




4. バイポール

  • Vipoleサービスに関連付けられています。 キーを生成するには、指で運転する必要があります。 IMだけでなく、他のVIPoleサービス-メモ、パスワードマネージャー
  • たとえば、ローカル履歴を暗号化するために、多くのお金が必要です
  • アカウントのパスワードとキーのパスワードを設定できます。 プログラムをクラッシュさせるか、データをクラッシュして削除するキーに偽のパスワードを設定できます
  • 指紋チェックなし
  • 登録にはメールが必要です
  • メッセージ、通話、ビデオ通話、ファイル転送、グループチャット、通話
  • すばらしいビジネスソリューションのように思えます。
  • AES-256、RSA-3072
  • Android / Windows / MacOS / Linux




5. 見えない

  • 対話者の確認なし、キースタンプなし
  • メッセージ/オーディオビデオコール/グループオーディオビデオコール/ファイル
  • 無料のアカウントでは、最大5人までの電話会議を行うことができます。 無料版にはビデオ会議はありません
  • ログイン登録
  • 4096ビットNTRU
  • Android / MacOS / Linux / Windows




6. シッチャー

  • IM +の作成者からのIM
  • メッセージ、グループチャット、ファイル
  • メッセージの自己破壊
  • 対話者の検証、鍵の指紋さえもありません
  • 電話での登録
  • Android / iOS / WinPhone




7. Whistle.im

  • メッセージのみ。 登録するには、電話もメールも必要なく、ユーザー名とパスワードのみが必要です。 キースタンプユーザー確認
  • Android /ブラウザ
  • 暗号化ライブラリのソースコードが利用可能




8. Surespot

  • ログイン/パスワードのみを使用した登録。 PGPとそのキーサーバーによって使用されます
  • 指紋がチェックされていません
  • メッセージ、ボイスメッセージ（有料）、写真
  • 521ビットECDH + AES256-GCM
  • オープンソースのクライアントとサーバー
  • ウェブサイトの詳細暗号化
  • Android / iOS




9. ウィッカー

  • 指紋はチェックされません。
  • メッセージは最大6日間自己破壊します。
  • メッセージ、写真、音声録音。
  • 登録するには、ログインするだけです。 メールなし。
  • AES256、ECDH521 TLS
  • Android / iOS




10. アントックス

  • Toxについては、すでにすべてを知っていると思います。 どこかでNATが失敗し、通信がまったく失敗しました。 これは、サーバーなしのP2P通信の最大の問題です
  • オープンソース
  • クロスプラットフォーム




11. Seecrypt SC3

  • AES 256およびRC4 384
  • メッセージ/音声通話/ファイル
  • Androidのみ




12. 電報

  • 視覚的な指紋が付いた非常に便利なメッセンジャー。 指紋チェックなし。 2つの操作モード：保護されたチャットと保護されていないチャット。
  • AES-IGE + SHA1
  • メッセージ/ファイル
  • オープンプロトコル、オープンソースクライアント、サーバーではない
  • Android / iOS / Windows / MacOS / Linux /ブラウザ




結論



すべてがやや悲しい。 Chiffryを喜ばせたが、彼はまだ生で所有権を持っている。 Surespotは説明とオープン性に満足しており、Vipoleに幾分興味を持っています。 しかし、他の誰もが使い心地が悪く、不快に感じることはありません。 XMPP + OTR / SIP + ZRTPを引き続き使用します。
