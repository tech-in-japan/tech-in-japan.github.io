スマートフォンレビューMeizu MX3

ハブラリュディ、おめでとうございます！ フラッグシップのMeizu MX3デバイスの販売がついに開始されました。本日、このスマートフォンの詳細なレビューをお届けできることを嬉しく思います。 Meizu MX3の長所、短所、およびすべての機能を理解します。









Meizu MX3の機能



  • オペレーティングシステム ：Android 4.2とFlyme 3 Shell
  • 通信 ：GSM / GPRS / EDGE / 3G（WCDMA）/ HSPA + 850/900/1800 MHz（2G）、850/2100 MHz（3G）
  • 画面 ：Sharp New Mode 2、対角5.1インチ、解像度1800×1080ピクセル、PPI 415、コントラスト1000：1、最大輝度450 cd / m、輝度2048階調、TOLテクノロジー（単一モジュール-ガラス+センサー）、省エネPSRテクノロジー、保護ガラスゴリラガラス2
  • プラットフォーム ：Samsung Exynos 5410、8コアプロセッサ（4コア1.6 GHz Cortex-A15および4コア1.2 GHz Cortex-A7）、28 nmプロセステクノロジーLP HKMG
  • GPU ：PowerVR SGX 544MP3（533MHz、3コア）。
  • RAM ：2 GB、DDR3
  • データストレージ用メモリ ：16/32/64 GB
  • カメラ ：8メガピクセル（BSIセンサーSony Exmor、IMX179 CMOSチップ、感光性エレメント1.4μm）、撮影速度：20フレーム/秒。 5レンズ、F / 2.0絞り、アングル74°、インスタントフィルター機能、ビデオ録画30 fps 1080p、パノラマ撮影、ジャイロフォーカス、HDR機能、顔検出、ジェスチャーショット
  • フロントカメラ ：2MP（ソニーセンサー）、ビデオ録画30 fps 1080p。
  • マルチメディア ：Wolfson WM5102サウンドチップ、Dirac HDテクノロジー、3つのマイクを備えたSmart PAスピーカーシステム、富士通4チャンネル画像処理チップ
  • サポートされている形式 ：FLAC、APE、AAC、MKA、OGG、MP3、MIDI、M4A、AMR、WAV、MP4、3GP、MOV、MKV、AVI、FLV、MPEG、M2TS、TS、JPEG、PNG、GIF、BMPなどその他
  • インターフェース ：Wi-Fi 802.11 b / g / n、2.4GHz / 5GHz（最大速度150 Mbps）、Bluetooth 4.0、microUSB（USB-OTGサポート）、NFC（MEIZU MX3 16GBを除く）、ヘッドフォン用3.5 mmヘッドセット
  • ナビゲーション ：GPS / A-GPS / GLONASS、電子コンパス
  • オプション ：タッチセンサー、光センサー、重力センサー、電子コンパス、ジャイロスコープ
  • バッテリー ：2400 mAh、ソニー、電源管理用デュアルチップ
  • 寸法 ：139.0 x 71.9 x 9.1 mm

外観、デザイン



MX2とMX3の類似性にもかかわらず、後者は前年のフラッグシップとはまだ異なります。 注目に値する最初のことは、寸法です。昨年のモデルでは、厚さは10.2 mmでしたが、新しい9.1 mmでした。 小さな違いのように見えますが、Meizu MX3ではサイドエッジがより面取りされているため、ガジェットがMX2よりもはるかに薄いという印象（および感覚）を与えます。 このため、スマートフォンは理想的には手元にあります。 別の問題は、ケースサイズの増加（139x71x9.1mm対124x64x10.2）ですが、画面の対角線も4.4インチから5.1インチに増加したため、何もする必要はありません。











ディスプレイフレームよりも薄くなりました。 さて、左右でわずか2.9 mm、上下で14 mmです。 とてもエレガントに見えます。 フロントパネルは黒なので、画面がオフのとき（黒いスマートフォンについて話している場合）、フレームはまったく見えません-幻想的な要素のない1つの暗い平面です。











ケースは、サイドフレーム（ちなみに、MX2のほぼ2倍の薄さ）がプロセッサの負荷が重い場合でもほとんど加熱されないように作られています。 縁取りには垂直研磨が適用され、これは電話の製造で使用される「ハードウェア」をさらに強調します。

パフォーマンスとメモリ



Meizu MX3は、サムスンの最新プラットフォームの1つであるExynos 5410を使用しています。8コアプロセッサ（4コア1.6 GHz Cortex-A15および4コア1.2 GHz Cortex-A7）、28 nmプロセステクノロジーLP HKMG。 Cortex-A15アーキテクチャと最新の28nm LP HKMGプロセスにより、モデルの計算能力は前世代と比較して1.8倍に向上しています。



グラフィック部分はPowerVR SGX544 MP3チップ（533MHz、3コア）であり、1秒あたり2億8,000万の三角形の速度でレンダリングし、79億ピクセルを効果的に埋めることができます。これは、1秒あたり1270回の高解像度画面の更新に相当します。 新しいGPUのパフォーマンスは、前世代と比較して267％向上しています。



デバイスのRAMは2 GB（第3世代のデュアルチャネルDDR3）です。 これにより、最大12.8 GB / sの速度でデータスループットを提供できました。 このボリュームの空き容量は約1.2 GBです。







内蔵メモリ16/32/64 GB。 128 GBモデルは当初、国際版も含めて計画されていましたが、現在のところ、同社はいつ登場するかについての情報を提供していません。 約3 GBがシステムと基本アプリケーションによって占有されています。つまり、16 GBモデルでは32 GB〜29 GBなど、約16 GBが無料です。
