3月15日、ヨーロッパでの決済ビジネスとインターネット取得の機能に関するウェビナーを開催します

3月15日水曜日、モスクワ時間の16:00に、ヨーロッパのライセンス企業に基づいた決済ビジネスの作成とインターネット取得の機能に関する国際ウェビナーが開催されます。 支払いプロバイダーPayOnlineのチームがイベントの共同主催者として行動します。



背景のビット。 最近、ヨーロッパで認可された支払い機関と電子マネー機関の設立はますます興味深いものになりました。 なぜヨーロッパで？ 2009年に、決済事業者PSD、2007/64 / ECの活動を規制する指令がここで施行されました。



この8年間で、欧州の規制当局は認可企業との十分な経験を蓄積しており、決済機関と電子マネー機関、および銀行と国際決済システムはこれらのクライアントの特性を理解しています。



上記の種類のライセンスのいずれかを所有している会社は、顧客に次のサービスを提供できます。



  • クライアント-法人および個人（IBAN割り当てのあるものを含む）の電子マネーアカウント/支払いアカウントを開設および維持する。
  • プリペイドおよびデビットバンクカードを含む金融商品を発行します。
  • 口座を開設せずに送金する（送金）。
  • 銀行カードからの支払い、送金、SWIFT / SEPAの支払いなど、さまざまな方法で顧客に有利な支払いを受け入れます。


ウェビナー中のヨーロッパでの支払いの受け取りの機能の詳細については、次のとおりです。



  • アンナ・イワノワ、PayOnline System LLC国際開発部長
  • アンドレイ・ボリセンコフ、開発ディレクターAdvapayOÜ


ヨーロッパでの決済ビジネスの創設は、銀行が多くの新しい課題に効果的に対処することを可能にするツールを提供します。 欧州の決済機関または電子マネー機関の設立は、ロシアの銀行または決済会社のビジネスの多様化のための追加の方法になる可能性があります。



ウェビナーでは、ヨーロッパでのインターネット取得の機能、価格設定、銀行との連携などの機能についても検討します。



すべてのパフォーマンスは、ロシアおよび国際企業の実際の実践に基づいています。 これらのケースは、ビジネスの発展の例として役立ちます。
