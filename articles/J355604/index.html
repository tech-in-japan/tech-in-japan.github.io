Spartan BrowserがWindows 10プレビューに表示されます





Microsoft は 、Windows 10のプレリリースバージョンがより頻繁にリリースされ、その約束が実現されたと述べました。 3月30日、前のビルドから2週間もたたないうちに、Project Spartanブラウザーに焦点を当てた新しいビルド10049がリリースされました。 ビルド番号は、以前のリリースに比べて6つだけ増えました。



ビルド10049はWindows 10のいくつかのバグを修正しますが、最大の変更点はMicrosoftの新しいミニマルなブラウザーであり、コード名はProject Spartanです。 これは最初の公式リリースですが、以前は他の方法でアクセスできました。 目新しさのために、多くの重要な機能が欠落しています。履歴やファイルのダウンロードはありません。 すでに利用可能なSpartanの主な違いは次のとおりです。



  • 音声コントロールCortana。 音声アシスタントは天気と株価を表示できますが、カレンダーとフライトスケジュールのデータはまだ利用できません。 Cortanaは現在、米国のユーザーのみが利用できます。
  • ページでは、テキストのメモを残してスタイラスで描画し、受信したものを電子メールまたはソーシャルネットワーキングサービスで共有できます。 タグをOneDriveクラウドストレージに保存することもできます。
  • ページ要素ではなくコンテンツにユーザーの注意を集中させる読み取りモード。 この機能を使用すると、WebページをPDFファイルとして保存でき、資料を閲覧リスト（閲覧リスト）に収集できます。


ジョー・ベルフィオーリは、彼の現在の状態では、スパルタンは完全にはほど遠いことを特に強調しました。 これはユーザーへの配布用のリリースではありません。ブラウザーにはさらに多くの追加および改善が加えられ、その後のみ公開されます。







SpartanはWindows 10のデフォルトブラウザーになり、スタート画面とタスクバーで利用できるようになります。 Internet Explorer 11は互換性のために残りますが、そのアイコンを起動するには、意識的に検索する必要があります。



最初のページレンダリング計画は次のとおりです。互換性が必要な場合、Spartanは古いInternet Exporer Tridentブラウザーエンジンを使用し、Internet Explorerは新しい兄弟のEdgeエンジンを使用できます。 後にこのアイデアは放棄され 、SpartanではEdgeのみ、Internet ExplorerではTridentのみとなります。 したがって、ブラウザ間の違いはより明白です。 Internet Explorerは互換性を維持するためのツールになります。たとえば、特別な企業サイトやテストのために実行する必要があります。 Spartanは、すべての主要な標準と互換性のある、日常使用のためのブラウザーです。 「トップ10」では、下位互換性が重要な企業と、スピードとシンプルさに関心のある一般ユーザーの両方の要望を考慮します。



ブラウザーをダウンロードするには、Windows Insiderプログラムに参加して 、Windows 10のプレビューバージョンのビルド10049をダウンロードする必要があります。コンピューターに既に数十のプレビューバージョンがインストールされている場合は、更新システムから新しいブラウザーをダウンロードするだけです。 https://windows.uservoice.com/forums/285214-project-spartanで、 Project Spartanに提案を残して投票することができます。



Spartanは、この夏の終わりにリリースが予定されている、将来のWindows 10オペレーティングシステムのメインブラウザーのコード名です。 MicrosoftはInternet Explorerブランドを廃止し、新しいエンジンと最小限のデザインを備えた新しいブラウザーを支持しています。
