「休日の読書」：IaaS、ITインフラストラクチャ、クラウドテクノロジーに関する25の有用な情報源

本日、クラウドテクノロジー、クラウドセキュリティ、サービス開発、およびITインフラストラクチャに関する25の最も人気のある興味深いブログの選択を準備することにしました。 これらのソースは、仕事の空き時間を読むのに最適です。

企業や愛好家のブログ



ダイバーシティ - ダイバーシティ投資家およびディレクターのベン・ケペスのブログ。 彼の関心分野は、エンタープライズソフトウェアと金融セクターソフトウェアから、プラットフォーム全体とそのアーキテクチャ機能にまで及びます。



Google Cloud Big Data and Machine Learning Blog -Googleのブログ。 詳細な製品分析、技術分析、比較、および最新の機械学習技術の使用に関するヒントとライフハック。



Sher.ly-ブログでは、クラウドとデータセキュリティの問題、および従業員の生産性を向上させるためのヒントについて説明しています。



Bartonsのブログは、Dellのシニアアーキテクトのブログです。 開発、オープンソースプロジェクト、テクノロジーに関する多数のインタビュー。



MemorableURL-分析、コンピューターサイエンス、エンタープライズソフトウェア、標準などに関する資料。



Rackspaceブログ-Rackspaceのブログ。 著者は、最近の会議のニュースからハイブリッドクラウドの作成に関する実用的なヒントまで、さまざまなタイプの資料を収集しています。 同社はまた、企業文化についても語っています。



Rackspace Developer Blog - Rackspace開発者からのニュースと洞察、およびデータベースの操作に関する実用的なヒント。



CloudPundit：大規模コンピューティング -著者は、特にクラウドテクノロジーとIaaS、およびDevOpsの哲学とクラウド管理プラットフォーム（OpenStackなど）について語っています。



All Things Distributedは、AmazonテクニカルディレクターのWerner Vogelsのブログです。 サービスおよびITインフラストラクチャの技術的な実装からディープラーニングまで、さまざまな資料。



CloudScalingは、CloudScalingプロジェクトのRandSc Bias共同設立者です。 問題解決、技術比較、実用的なアドバイス-これらはすべてここにあります。



小片ですか？ -クラウドテクノロジーが企業の組織構造に与える影響に関するSimon Wardleyのブログ。



Red Hat Stackは、同名のプラットフォーム専用のRed Hatブログです。 新機能の分析、個々のコンポーネントの動作原理、および生産性向上のヒント。



分析、インタビュー、ニュースリソース



CloudAveは、ITインフラストラクチャ、SaaSのアプリケーション開発に関するオンラインマガジンです。 著者はいくつかの問題で互いに意見が分かれており、興味深い議論につながっています。



ZDNetは、仮想化、データセンター、クラウド内のサービスの場所に関するニュースを掲載したリソースです。



One Million by One Million Blog -IBM、Intel、VMwareなどの一流の専門家との定期的なインタビュー



データセンターの知識 - データセンターの仕組みに関する雑誌。 クラウドの世界からのニュースも添付されています。



登録 -分析のメモを含むデータセンターの世界からのニュースとインタビュー。



CIO-クラウドテクノロジーのニュース、分析、研究、実用的なヒント、インタビュー、会議のビデオ。



ロシア語で読む



最初の企業IaaSブログは、クラウドの機能の概要、機器をセットアップし、クラウドプロバイダーのサービスのコストを計算するための実用的なヒントとコツ、およびITの世界からのニュースです。



1cloud社のHabréのブログ -データストレージテクノロジー、仮想サーバー、ITインフラストラクチャ、およびサービス開発の効率を改善するためのヒント。



HabrにあるNew Cloud Technologies社のブログ -アプリケーションとサービスの開発について：設計からコードまで。



VAS Expertsブログ -ロシアのDPIシステムの開発者は、トラフィック分析に関する彼自身の経験と知識を共有しています。
