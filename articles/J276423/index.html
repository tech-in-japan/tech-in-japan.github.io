Konstantin Sakhnovによる公開講座「あなた自身のゲームを作成する方法」に招待します

みなさんこんにちは！



3月2日にはビジネスインフォマティクスの高等学校でHSEは" あなた自身のゲームを作成する方法 "のトピックに関するゲームデザインの無料公開講義になります 。 内容とタイミングの点で非常にユニークです。有料教育プログラム「ゲーミングインターネットプロジェクトの管理」の私のクラスの1つに相当します。



このプログラムのスーパーバイザーであり、ゲームスタジオRJ Gamesのゲームデザイン部門のディレクターであるKonstantin Sakhnovによる公開講演を行います。 私のチームと私はこの面白いgamedevイベントにみんなを招待します！ 講義の詳細については、記事の後半で読むことができます。











このイベントは、9か月間の専門的な再教育の教育プログラム「 ゲーミングインターネットプロジェクトの管理 」と、現在継続中の教育プログラム「 ゲーム開発の基礎 」の枠組みの中で開催されています。



講義は、自分のゲーム製品をどこから始めたらよいかを理解したいデベロッパーと、ゲーム開発者の知識を体系化し、開発方法論への新しいアプローチを研究することを目的としたゲームスタジオの既存の従業員にとって興味深いものになります。 そしてもちろん、ゲームに興味を持ち、ゲーム業界で働き始める機会を検討している人々 。 ところで、一部の学生は、トレーニング中にゲーム会社で仕事を見つけることに成功しています。



講義で議論されるもの：



  • ゲームの作り方 ：初心者開発者とプロデューサーの両方が新しいプロジェクトを開始する場合。

  • ロシアの大企業と外国企業は、オンラインゲームを開発する際にどの段階を区別しますか。
  • スタジオRJゲームの例の各段階でチームを行うために必要なこと 。
  • 各段階で準備される設計文書の種類 ：ビジョン、コンセプト、dzdok、および参照条件の違いは何ですか。
  • どのブロックから、プロジェクトのコンセプトドキュメントを作成して、管理者が最初に承認するようにする必要があります 。




講義の終わりに、誰もが私と他のプログラム「 ゲーミングインターネットプロジェクトの管理 」のためのゲームの開発について興味のある質問をすることができます 。







無料で講義にご参加ください！ しかし、ニュアンスがあります-場所の数が限られているので、 登録してください 。
