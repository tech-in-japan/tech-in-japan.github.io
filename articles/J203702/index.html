要約：



第1章 「Zend Framework入門」では、開発環境を構成する方法について説明します。 その中で、PHPアプリケーションサーバーをセットアップし、MySQL DBMSをインストールし、Zend Frameworkの実際の演習で使用するデータベースを作成します。これは、以降の章で説明します。

第2章 「Zend Frameworkを使用した最初のアプリケーションの作成」では、Zend Framework 2プロジェクトの作成について説明します。 モジュール、コントローラー、ビューを作成して、Zend Framework 2でMVCアプリケーションを開発する重要な側面について学びます。 作成するモジュールは、この本の以降の章で拡張されます。

第3章 「通信アプリケーションの作成」では、読者にZend / Formコンポーネントを紹介します。 この章では、最初の登録フォームを作成し、Zend Frameworkコンポーネントを使用して登録ユーザーのログインと認証のメカニズムを構成します。

第4章 「データ管理とドキュメント共有」では、Zend Frameworkのデータおよびファイル管理の概念について説明します。 この章では、サービスマネージャー、TableGatewayパターン、ファイルアップロードの処理と共有など、Zend Frameworkを使用するさまざまな側面を紹介します。

第5章 「チャットとメール」では、アプリケーションでJavaScriptを使用する方法について説明します。 Webアプリケーションの開発におけるJavaScript機能の使用を説明する例として、グループチャットの実装を使用します。 さらに、この章では、Zend \ MailコンポーネントとZend Framework Event Managerを使用したメールの送信について説明します。

第6章 「メディアの共有」では、開発者がZend Frameworkで画像とビデオを管理および共有する方法を紹介します。 この章では、画像とビデオの操作にさまざまなZend Framework外部モジュールを使用します。

第7章 「Luceneライブラリを使用した検索」では、Zend Frameworkを使用したLuceneライブラリに基づく検索エンジンの開発について説明します。 この章の冒頭で、ZendSearch \ Luceneモジュールのインストールについて説明し、その後、データベースおよびドキュメントファイルでの検索メカニズムの実装について詳しく説明します。

第8章 「簡単なストアの作成」は、eコマースシステムの概要を示しています。 この章では、ショッピングカートの開発プロセスを示す簡単なオンラインストアを作成します。 支払いを処理するには、PayPal Express Checkout支払いシステムを使用します。

第9章 「HTML5サポート」では、Zend Framework 2のHTML5仕様のサポートについて説明します。ZendFrameworkの以前のバージョンとは異なり、バージョン2はさまざまなHTML5機能の包括的なサポートを提供します。 この章では、HTML5を使用する2つの重要な側面-新しいデータ入力要素と複数のファイルのアップロードについて説明します。

第10章 「モバイルアプリケーションの作成」では、Zend Framework 2およびZend Studio 10開発環境を使用した「ネイティブ」モバイルアプリケーションの作成について読者に紹介します。アプリケーション。
