10月Ciklum .NET土曜日キエフ

10月の一連の.NET Subbotnikの一環として、次の恒星のイベントがキエフで開催されます。

10月20日、Ciklum SkyPointは、イベントの参加者向けに、街の忘れられない眺めと、すべての.NETスペシャリストにアピールする優れたレポートの両方を公開します。



プログラム内：

1. Alexander BeletskyによるASP.NET Web APIフレームワークの概要

ASP.NETバージョン4.5は、改善されたWebForms、MVCサポートを含む多数の新機能を獲得しましたが、著者が.NETでRESTful Webサービスを作成できる新しいASP.NET Web APIフレームワークと呼ぶ主な機能の1つです。 レポートでは、Webサービス、RESTアーキテクチャ、およびWeb APIの主な機能の進化に焦点を当てます。



2. Anton VidishchevによるWindows Azure Webサイトでの継続的な展開

このレポートでは、Azure Webサイトの要素と、クラウドTFSおよびGITの展開と設定の機能を詳細に調べます。

。

3. Dmitry MindraによるTypeScriptの概要

誰もがMictosoftの新しい言語について聞いたことがあるので、JSでスケーラブルなサービスを開発できます。 JS開発者は、モジュールパターンを使用して、クロージャコンパイラを使用して、jslintでコードをすでにチェックしています。 TypeScriptを検討してください。 型チェックと静的分析、明示的なインターフェイス、および単一の言語とコンパイラ内の最適な機能を組み合わせています。 この言語を詳細に検討してください。



4.5 Dmitro Suvorov なしでは生きていけないResharper機能

R＃は、.NETの世界で最も人気のないツールの1つです。 それを使用する開発者は、便利な機能の無限のリストについて長い間語りますが、ほとんどの人は、それが遅く、絶え間なく、非常に役に立たないツールだと考えています。 しかし、スペシャリストからの有能な紹介で再び彼に会ったので、あなたは彼とは完全に異なっていることがわかります。

ドミトリーのレポートは、彼がR＃なしではもはや生きられない理由についてです。

5. C＃による関数型プログラミング by Sergey Kalinets

C＃はオブジェクト指向プログラミング言語と見なされますが、多くのパラダイムをサポートしています。そのうちの1つは関数型プログラミングです。

Sergeyはレポートで、関数型プログラミングパターンを使用することで開発者にとってより快適な条件を作成し、プログラムコードをよりクリーンで効率的にする方法を例で示します。 基本的なものから高度なものまでのC＃、LINQの基本的な構成、並列および非同期コードを検討します。
