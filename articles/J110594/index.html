PyCharm（IDE for Python / Django）がバージョン1.1に更新されました



PyCharm 1.0のリリースから2か月後、最初のメジャーアップデートであるバージョン1.1がリリースされました。 バージョン番号はわずかに増加していますが、多くの新機能と改善がリリースされています。 以下に短いリストを示します。

  • 自動補完が自動的に表示されるようになり、Ctrl-スペースを押す必要がなくなりました。
  • これで、エディタータブを2番目のモニター（ほとんどChromeのように）にプルできます。
  • デバッガーには、条件付きブレークポイント、例外のブレーク、新しいコンソール、およびリモートでデバッグする機能が登場しました。
  • テストの実行で、nose、doctests、およびpy.testの新しいバージョンを操作する方法がわかりました。
  • 多数の新しい検査、クイックフィックス、ローカルリファクタリングが登場しました（たとえば、「ラムダを関数に変換する」）。
  • LinuxおよびMacでは、コマンドラインからプロジェクトとファイルを開くことができるようになりました。
  • パッケージには、FTP / SFTPを介してファイルを操作したり、問題追跡システムと統合したりするためのプラグインが含まれています。
まあ、 はるかに便利です 。 いつものように、30日間の試用版と、オープンソースプロジェクトおよび教育プロセスで使用するための無料ライセンスが利用できます。



jetbrains.comで新しいバージョンをダウンロードできます。
