マンゴー-すべての開発者向け！





2か月以上前のMIX11で、Windows Phoneコードネーム「Mango」についてモバイル開発者の世界に語りました。 約1か月前、誰もがMangoを使用した電話エミュレーターを含む、Windows Phone用の新しい改善された開発ツールを受け取りました。 同時に、新しいOSのほぼすべての機能を世界中で公開し、顧客が利用できるようにしました。 Nokiaとのコラボレーションは勢いを増しています。 開発者に内在する懐疑論にもかかわらず、フィンランドの会社のパートナーは、私たちを愉快に驚かせることをやめません。



創業以来、Windows Phone 7の開発者のエコシステムは20,000以上のアプリケーションを作成してきました。



今日、私たちは次の大きな一歩を踏み出しています。



Windows Phone Developer Tools Beta 2は、App Hubに登録されている各開発者が利用できるようになります。



しかし、それだけではありません。 開発者向けにWindows Phoneコードネーム「Mango」アップデートの配布を開始しています。 はい、はい、このアップデートはお使いの携帯電話用です。



最も活発な開発者は、今日アップグレードの招待状を受け取り始めます。 今後2〜3週間で、アップグレードインフラストラクチャは、登録されているすべての開発者に対して完全に機能するように計画されます。



このすべての詳細については、Windows Phone 7開発チームのブログをご覧ください 。
