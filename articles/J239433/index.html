Apache CordovaはWindows 8.1およびWindows Phone 8.1のサポートを追加します

Windows Universal Appsは 、クロスプラットフォームモバイルアプリケーションApache Cordovaを作成するための一般的なフレームワークになりました。







古いwindows8プラットフォームはユニバーサルWindowsプラットフォームに置き換えられ、Windows 8.0、Windows 8.1、およびWindows Phone 8.1ですぐにアプリケーションを作成できるようになりました。



ここから開始できます： msopentech.com/blog/2014/09/25/apache-cordova-gains-windows-8-1-and-windows-phone-8-1-support-2-2

直接リリース： cordova.apache.org/announcements/2014/09/22/cordova-361.html



Windows Phone 8.1の例（コマンドラインから実行）



1. npmからフレームワークをインストールします 

npm install -g cordova


2.アプリケーションを作成する 

 cordova create awesomeApp cd awesomeApp


3. Windowsプラットフォームを追加します 

 cordova platform add windows


4. Windows Phone 8.1エミュレーターで実行する 

 cordova emulate windows -- --phone
