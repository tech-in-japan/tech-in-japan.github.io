ロシアの開発者向けサポートプログラム

昨年、わが国でWindows Phoneが発売されたことで、ロシアの開発者は世界中のユーザーにとって興味深い便利なアプリケーションを作成しました。

8月1日、新しいWindows 8 RTMオペレーティングシステムの最終バージョンを準備しました。その中には、単一のWindowsストアアプリストアがあります。

明らかに、今がWindows Phoneの最高のアプリケーションの作成者がグローバルなWindowsストアの最初の開発者になる時であり、私たちはヘルプとサポートを提供する準備ができています。



8月中にロシアの開発者をサポートする特別なプログラムを開始します Windows Phoneで。



プログラム規約：

  • プログラムは8月3日から8月31日まで有効です。
  • このプログラムは、ロシアまたはアメリカのWindows Phoneマーケットプレイスで5つ星のうち4つ以上の評価を持つアプリケーション作成者に有効です。


私たちが提供するもの：

  • Windows 8向けのカスタマイズされたアプリケーション設計のアドバイス
  • Windows 8用のアプリケーション開発に関する個別相談
  • Windowsストアで公開するためのアプリケーションの準備に役立ちます


プログラムに参加する場合は、Windows Phone Marketplaceのアプリケーションへのリンクを付けてアドレスbestapps \ @ \ microsoft \ .com \に書き込みます。



Windows 8およびWindowsストアは、Windowsユーザーの世界最大のエコシステムにアクセスするためのユニークな機会を提供します。 最高のロシアの開発者がこの新しい技術競争で成功することを非常に期待しています。
