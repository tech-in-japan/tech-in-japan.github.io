ASUS Eee PC 1201N-ステレオタイプからの脱却





ASUSは本日、12.1インチネットブックEee PC 1201Nが、デュアルコアIntel Atom 330プロセッサーとNVIDIA Ionグラフィックス（GeForce 9400M）の存在下で競合他社と大きく異なることを正式に発表しました。



1 GBのメモリを搭載したIntel Atom 280 +チップセットIntel 945GSE + ICH7Mの束の上に構築され、最良の場合、USBポートの色と数が互いに異なるクローンデバイスにうんざりしていると思います。 この意味でのEee PC 1201Nは、新鮮な空気の息として。



新機能：



OS： Windows 7 Home Premium;

ディスプレイ： 12.1インチ、LEDバックライト、解像度1366x768ピクセル。

プロセッサー：デュアルコアIntel Atom 330;

RAM： 3GBまで拡張可能な2GB。

グラフィック： NVIDIA Ion（GeForce 9400M）;

ウィンチェスター：250GB + 500GBの無料オンラインストレージEee Storage。

ポート： VGA、HDMI、3 USB 2.0、イーサネット、ヘッドフォン/マイク。

メモリカードのサポート： MMC / SD（SDHC）;

ワイヤレスモジュール： Wi-Fi 802.11n、Bluetooth 2.1 + EDR;

バッテリー：最大5時間のバッテリー寿命。

サイズ： 295x206x33 mm;

重さ： 1.45 kg;



ネットブックは12月中旬から黒と銀の色で利用可能になります。 推奨コスト-499ドル。
