1981年から2009年までのインターフェース設計オペレーティングシステムの進化

「 1981〜2009年のオペレーティングシステムインターフェイスデザイン 」の翻訳





グラフィカルユーザーインターフェイス（GUI-グラフィカルユーザーインターフェイス）は、ユーザーがコンピューターのハードウェアコンポーネントと非常に快適で便利な方法で対話できるようにするツールです。

長年にわたり、OS / 2、Macintosh、Windows、AmigaOS、Linux、Symbian OSなどの多数のオペレーティングシステムで、さらに多くのグラフィカルインターフェイスが作成されました。

80年代以降のこれらのシステムのインターフェース設計の進化を見てみましょう。

このトピックは、グラフィックデザインの分野で重要な成果と段階のみを示し（一般的なオペレーティングシステムではなく）、今日まですべてのシステムが存在するわけではありません。







最初のGUIは、70年代のゼロックスパロアルトリサーチセンター（PARC）で開発されました。 この開発により、コンピューターグラフィックスの革新の新しい時代が始まりました。

新しいグラフィカルインターフェイスを使用した最初のパーソナルコンピューターは、1973年に作成されたゼロックスアルトでした。 これは商用製品ではなく、主に大学での研究を目的としていました。









1981-1985



ゼロックス8010スター（1981）


これは、ソフトウェアアプリケーションやグラフィカルインターフェイスを含む統合デスクトップコンピューターとして導入された最初のシステムでした。 このコンピューターは、Xerox Starとして知られており、後にViewPoint、後にGlobalViewに名前が変更されました。









Apple Lisa Office System 1（P1983）


Lisa OSとしても知られていますが、この場合、その略語はその名前のOffice System（Office System）にはあいまいすぎます。 これは、ドキュメントを操作するためのラップトップを作成する目的でAppleによって作成されました。

しかし、残念なことに、このシステムはApple Macintoshオペレーティングシステムによって「殺され」、当時はよりアクセスしやすくなりました。

Lisa OSシステムの1983年のLisa OS 2および1984年のLisa OS 7/7 3.1へのアップグレードもありましたが、これらの変更はシステム自体にのみ影響し、インターフェースには影響しませんでした。















VisiCorp Visi On（1984）


Visi Onは、IBM PC用に設計された最初のインターフェースです。 このシステムは大企業向けであり、非常に高価でした。 インターフェイスはマウスを使用し、インストーラーとヘルプシステムが組み込まれていましたが、アイコンは使用しませんでした。















Mac OS System 1.0（1984年に導入）


System 1.0は、Macintosh用に作成された最初のオペレーティングシステムです。 彼女はすでに最新のオペレーティングシステムの詳細をいくつか持っていました。ウィンドウの原理に基づいており、アイコンが含まれていました。 ウィンドウはマウスでドラッグでき、ファイルとフォルダーはコピー先にドラッグアンドドロップすることでコピーできます。









Amiga Workbench 1.0（1985）


リリース後、Amigaはその時間を追い越したようです。 GUIには、たとえば、カラーグラフィック（4色：黒、白、青、オレンジ）が含まれており、ほとんどの場合、マルチタスク、ステレオサウンド、およびいくつかの状態（強調表示および非強調表示）のアイコンをサポートしていました。















Windows 1.0x（1985）


今年、Microsoftはついにユニバーサルインターフェイスを入手し、最初のGUIベースのオペレーティングシステムであるWindows 1.0をリリースしました。 システムには32x32ピクセルのアイコンとカラーグラフィックがありました。 ただし、最も興味深い革新は（後で消えましたが）、アニメーション化されたアナログ時計のアイコン（矢印付き:)）でした。















GEM（1985）


Digital Research、Inc.が作成したGEM（グラフィカル環境マネージャー） （DRI）はウィンドウタイプでした。 もともとはIntel 8088およびMotorola 68000マイクロプロセッサに基づいたCP / Mオペレーティングシステムで使用するために作成されましたが、後にDOSで使用するために改良されました。 ほとんどの人は、GEMをAtari STコンピューターのGUIとして覚えており、IBM互換コンピューターのAmstradシリーズにも使用されています。 また、Ventura Publisherや他のいくつかのDOSプログラムのエンジンを務めました。 また、他のコンピューターにも移植されましたが、他のコンピューターでは人気がありませんでした。









1986-1990



IRIX 3（1986年にリリース、1984年に最初にリリース）


64ビットIRIXオペレーティングシステムは、UNIX用に作成されました。 GUIの興味深い機能は、ベクターアイコンのサポートです。 この機能は、Mac OS Xが起動するかなり前にこのシステムに組み込まれていました。









GEOS（1986）


GEOS（グラフィック環境オペレーティングシステム）は、Berkeley Softworks（後のGeoWorks）によって開発されました。 もともとはCommodore 64用に開発され、geoWriteと呼ばれるグラフィカルワードプロセッサとgeoPaintと呼ばれる描画プログラムが含まれていました。









Windows 2.0x（1987）


このバージョンでは、ウィンドウ管理が大幅に改善されました。 これで、ウィンドウのオーバーラップ、サイズ変更、最大化、拡大、縮小が可能になりました。















OS / 2 1.x（1988）


OS / 2はもともとIBMとMicrosoftの発案でしたが、1991年に両社はWindows OSおよびIBMがOS / 2の開発を継続するための独自のGUIテクノロジーを備えたMicrosoftと分割しました。 OS / 2で使用されるインターフェイスは「プレゼンテーションマネージャー」と名付けられました。 このバージョンのGUIは、モノクロアイコンのみをサポートしていました。















NeXTSTEP / OPENSTEP 1.0（1989）


Steve Jobsは、研究室や大学に最適なコンピューターを作成するというアイデアに興味がありました。 その後、このアイデアにより、NeXT Computer Incというスタートアップが誕生しました。

最初のNeXTコンピューターは1988年に導入されましたが、1989年にNeXTSTEP 1.0 GUIがリリースされ、後にOPENSTEPになりました。

インターフェイスアイコンが大きくなり（48×48）、より多くの色が使用されました。 当初、GUIはモノクロでしたが、バージョン1.0以降、カラーモニターがサポートされました。 このスクリーンショットは、最新のインターフェースがどのように見えるかを示しています。









OS / 2 1.20（1989）


GUIの次のバージョンでは、多くの領域でいくつかの改善が行われています。 アイコンが見やすくなり、ウィンドウが滑らかになりました。









Windows 3.0（1990）


このバージョンでは、Microsoftの開発者はGUIの真の利点をすべて理解し、大幅に改善し始めました。

オペレーティングシステム自体が標準を維持し始め、386アーキテクチャの拡張モードでは、640キロバイト以上のメモリとより多くのハードディスク領域が必要になり、Super VGA 800×600やXGA 1024×768などの解像度が可能になりました。

同時に、MicrosoftはアーティストおよびグラフィックデザイナーのSusan KareにWindows 3.0アイコンデザインを設計し、GUIのユニークなイメージを作成するように依頼しました。















1991-1995



Amiga Workbench 2.04（1991）


このバージョンのGUIには多数の改善が加えられています。 カラースキームが変更され、体積設計要素が導入されました。 デスクトップは、独自の解像度と色深度を持つ2つの画面に垂直に分割できますが、今日は少し奇妙に見えます。 標準解像度は640×256でしたが、ハードウェアでもより高い解像度がサポートされていました。









Mac OSシステム7（1991）


Mac OSバージョン7.0は、色をサポートする最初のMacシステムでした。 灰色、青、黄色の色合いがアイコンに追加されました。















Windows 3.1（1992）


このバージョンのWindowsには、プリインストールされたTrueTypeフォントが含まれていました。 その時点で、実際にWindowsを公開プラットフォームとして使用することを決定しました。

このような機能は、以前はAdobeのフォント管理システムであるAdobe Type Manager（ATM）を使用するWindows 3.0でのみ利用可能でした。 また、このバージョンには、赤、黄、黒の明るい色合いを含む「ホットドッグスタンド」と呼ばれる配色が含まれていました。

このスキームは、色覚障害のある人々がテキストおよびグラフィック情報を認識しやすくするために作成されました。









OS / 2 2.0（1992）


これは、多言語インターフェイスをサポートすることを目的とした最初のGUIでした。また、使いやすさとアクセシビリティに関するテストを実施した最初のGUIでもありました。 インターフェイスは、オブジェクト指向設計を使用して作成されました。 各ファイルとフォルダーは、他のファイル、フォルダー、およびプログラムに関連付けることができるオブジェクトによって表されていました。 また、ドラッグアンドドロップテクノロジとテーマを変更する機能もサポートしました。















Windows 95（1995）


Windows 95は、ユーザーインターフェイスを完全に再設計しました。 これは、ウィンドウを閉じるために各ウィンドウの隅に十字のボタンが表示されたWindowsの最初のバージョンでした。

アイコンとコントロールのさまざまな状態が追加されました（利用可能、アクセス不可、選択済み、チェック済みなど）。 また、有名な「開始」ボタンが初めて登場しました。

Microsoftにとって、これはオペレーティングシステムとGUIの統合の両方にとって大きな前進でした。















1996-2000



OS / 2ワープ4（1996）


1996年、IBMはOS / 2 Warp 4を導入しました。これにより、デスクトップの外観が大幅に改善されました。

アイコンはデスクトップ上にあり、ユーザーは自分のファイルやフォルダーを置くこともできました。 表示されたシュレッダーは、Windowsのごみ箱またはMac OSのごみ箱（？）（ごみ箱）の類似物でしたが、コピーを保存する代わりに、その中に置かれたドキュメントをすぐに削除した点が異なります。















Mac OSシステム8（1997）


このバージョンのGUIの標準アイコンは256色でした。 Mac OS 8は、擬似3Dアイコンとも呼ばれるアイコンのアイソメ画像を使用する最初のシステムの1つにもなりました。 ここで最初に使用されたプラチナグレーのテーマは、このシステムの将来のバージョンの特徴となっています。









Windows 98（1998）


アイコンのスタイルはWindows 95に似ていましたが、システムは既に256色以上を使用してグラフィカルインターフェイスを表示していました。 Windowsエクスプローラーはほぼ完全に変更され、Active Desktopが最初に登場しました。









KDE 1.0（1998）


これは、KDEチームがリリースでプロジェクトを説明した方法です。「KDEは、UNIXを実行するコンピューター用の最新のデスクトップ環境です。 KDEは、MacOSやWindow95 / NT環境の類似物を連想させる、Unixシステム用の高速で便利なインターフェースの必要性を満たそうとしています。 完全に無料でオープンなコンピュータープラットフォームは、変更用のソースコードを含め、誰でも自由に利用できます。









BeOs 4.5（1999）


BeOSオペレーティングシステムは、パーソナルコンピューター用に開発されました。 もともとBeInで書かれました。 1991年にBeBoxマシンで起動しました。 その後、モジュラーI / Oシステムを使用した対称マルチタスク、ほぼ完全なマルチスレッド、ほぼ完全なマルチタスク、BFSとして知られる64ビットジャーナリングファイルシステムなど、新しいテクノロジーとハードウェアの利点をまとめるために作成されました。 BeOS GUIは、明確でクリーンで不格好な設計の原則に基づいていました。









GNOME 1.0（1999）


GNOMEインターフェースは主にLinux Red Hat用に作成されましたが、他のLinuxディストリビューション用の新しいバージョンも登場しました。









2001-2005



Mac OS X（2001）


2000年代初頭、Appleは新しいAquaインターフェースを発表し、2001年に同社は新しいMac OS Xオペレーティングシステムを導入しました。

デフォルトの32x32および48x48ピクセルのアイコンは、アンチエイリアスと半透明性を使用して大きな128x128ピクセルのアイコンに置き換えられました。

ただし、このGUIのリリース後、多くの批判が現れました。 どうやら、ユーザーはこのような劇的な変化に対応する準備ができていなかったが、すぐに新しいスタイルを採用し、今日ではこのGUIがすべてのMac OS Xシステムの基盤となっています。









Windows XP（2001）


Microsoftは、新しいプラットフォームごとにユーザーインターフェイスを完全に変更しようとしましたが、Windows XPも例外ではありませんでした。 GUIのスタイルを変更できるようになり、ユーザーはインターフェイスの外観と動作を完全に変更できるようになりました。 デフォルトでは、アイコンのサイズは48x48ピクセルで、何百万もの色合いが使用されていました。









KDE 3（2002）


バージョン1.0以降、KDE環境は大幅に改善されました。 すべてのグラフィックとアイコンが研ぎ澄まされ、パフォーマンスが統一されました。









2007-2009（現在）



Windows Vista（2007）


これは、競合他社に対するマイクロソフトの回答でした。 多くの3Dとアニメーションも追加されました。 Windows 98以降、Microsoftは常にデスクトップの改善を試みてきました。 Windows Vistaでは、ウィジェットといくつかの機能強化がActive Desktopの放棄とともに導入されました。









Mac OS X Leopard（2007）


Mac OS Xシステムの第6世代のAppleは、ユーザーインターフェイスの水準を再び引き上げました。 GUIの基本は、光沢のあるスクロールバーとプラチナグレーとブルーの色を備えたAquaインターフェイスです。 新しいインターフェイスの詳細は、3Dドックと多くのアニメーションおよびインタラクティブ機能により、さらに自然でボリューム感が増し始めました。









GNOME 2.24（2008）


GNOMEは、「コンピューターの見栄えを良くする」という目標に沿って、バージョン2.2.4でテーマと壁紙を作成するために多大な努力を払ってきました。 バージョン2.24に含まれているデスクトップの背景として使用する最も魅力的な画像を収集するために、コンテストが開催されました。









KDE（v4.0-2008年1月、v4.2-2009年3月）


KDEバージョン4では、アニメーション、アンチエイリアシング、効率的なウィンドウ管理システム、デスクトップウィジェットのサポートなど、環境とインターフェイスに多くの新しい機能強化が行われました。 デスクトップアイコンは簡単に変更でき、各デザイン要素は簡単に構成できます。 最も顕著な変更は、Oxygen Projectチームによって提供されたアイコン、テーマ、およびサウンドで発生しました。 これらのアイコンは、最もフォトリアリスティックになりました。 そしておそらく、KDEの歴史の中で最大の改善は、環境がWindowsとMac OS Xプラットフォームの両方で自由に実行できるようになったことです。
