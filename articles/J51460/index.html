ASP.NETトレーニング

開発者向けのASP.NETテクノロジ「 ASP.NETでの Webアプリケーションの開発」に関する無料トレーニングは、2月13日にモスクワのマイクロソフトオフィスで開催されます。 このトレーニングは、ASP.NETの使用経験がない開発者向けに設計されています。





注意！ 登録は締め切りました。 最初の15人が確認を受け取ります。 最初の15のいずれかが拒否された場合、確認が次の行に送信されます。



トレーニングに参加するには、HTML / JavaScriptの基本を理解し、Webアプリケーションの基本を理解し、Cライクな言語でプログラムを作成した経験があることが推奨されます。



C＃はプログラミング言語として使用されます。 C＃の知識は望ましいですが、必須ではありません。



トレーニングを実施- ガイダール・マグダヌロフ 。 トレーニングは10.00から始まり、18.00で終わります。



トレーニングプログラム：



  1. ASP.NETの紹介

    • 前提条件のインストール
    • ASP.NET Webアプリケーションの紹介
    • Visual Studio 2008およびASP.NET Webアプリケーションの作成


  2. ユーザーインターフェース開発

    • ASP.NETのサーバーコントロール
    • イベント処理と制御の相互作用
    • サーバーとクライアントに関するASP.NETページ


  3. データを操作する

    • データソース
    • データを表示するためのコントロール


  4. ASP.NETクライアント側

    • ASP.NETでのAJAXサポート
    • サーバー制御
    • AJAX Control Toolkitプロジェクト


  5. ASP.NETアプリケーションのデバッグ

    • Visual Studioデバッガーでの作業
    • ASP.NETのデバッグメカニズム
    • クライアントコードのデバッグ


  6. ASP.NETアプリケーションの配布

    • コンパイルと配布
    • Webサーバーとアプリケーションの構成
