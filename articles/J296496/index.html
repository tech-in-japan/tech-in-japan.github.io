2020年までに、Googleは「クラウド企業」に変わるはずです

Googleクラウド部門の責任者であるUrsHölsleは、今後5年間で、企業のクラウドビジネスは広告よりも多くの収益を生み出すと考えています。 彼は2020年までにGoogleから「クラウド企業」を成長させることを期待しています。



このような大胆な声明は、企業のビジネスの大きな変化を示しています。 実際、2014年の結果によると、Googleの収益の89％は広告によって提供されていました。 2015年第3四半期、すべてのプラットフォームでのGoogleの広告収入は168億ドルに達しました。 これは、2014年の同時期と比較して15％増加しています。



さらに、同社のクラウドビジネスは、クラウドコンピューティング市場の主要プレーヤーであるAmazon Web ServicesとMicrosoft Azureにまだ負けています 。 ただし、UrsHöltsleは、Google Cloud Platformがおそらく業界で最も高い成長率を持っていると考えています。 したがって、しばらくすると、遅延は大幅に減少します。 Hoelsleが言ったように、同社にはすでに多くの企業顧客、さらに「幸せな企業顧客」がいます。



彼は、クラウドコンピューティング市場の発展とスマートフォン市場を比較します。2007年にはiPhoneが初めてでしたが、スマートフォンの需要増加にも弾みをつけました。 そのため、遅いスタートにもかかわらず、 Android OSは世界で最も人気のあるオペレーティングシステムになりました。 Hoelsleは、 Google Cloud PlatformはAndroidシステムの道をたどると信じている、とBusiness Insiderは報告している 。



彼によると、Googleは生産性の水準を引き上げながら、顧客にサービスをより低い価格で提供できるようになるという。 Google Cloud Platformには、Google検索エンジンとGmail、YouTube、および他のすべてのサービスを運用する同じデータセンターが対応しているため、規模と技術の両方があります。 毎日、何十億人もの人々がこれらのサービスを使用しています。



クラウドサービスが長年にわたって存在していたため、Googleは競争力を高める機能をいくつか追加し、2013年に製品全体の名前が変更され、Google Cloud Platformと呼ばれました。 クラウドへの企業向けオプションの追加に伴い、同社はコカコーラ、ゼネラルミルズ、ベストバイなどの大口顧客を引き付け始めました。



まだGoogleではありませんが、Amazon Web ServicesとMicrosoft Azureは、価格の低下と一流の機能を備えた古い企業の顧客を徐々に魅了しています。



たとえば、Amazonは最近、OracleデータベースをAmazon Web Servicesに移動するように設計されているため、 Oracle Cloudの 「キラー」と呼ばれるサービスを開始しました。



一方、Oracle はいくつかの競合サービスAmazon Web Servicesの立ち上げを準備しています。 オラクルは6か月以内に複数のクラウドサービスを開始しますが、そのうち最大のものはElastic Compute Cloudです。 機能とその名前の点でも、クラウドでコンピューティングパワーを提供するAmazon Elastic Compute Cloudに似ています。



オラクルは2015年初頭にクラウドプロジェクトの作業を開始しました。開発者によると、これはAmazon Web Servicesの「キラー」になるはずです。
