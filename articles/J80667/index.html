マザーボードASRock H55DE3の概要

Asrockは、Intel Socket 1156プロセッサ用に10個のマザーボードを導入しました。6個はP55チップセットに搭載され、4個は新しいH55に搭載されました。 この4つのうちの1つのマザーボード、つまりAsrock H55DE3を使用して、このレビューに精通します。 それに加えて、Asrockは2つのmATX、H55MおよびH55M Proマザーボード、およびこのH55 Proラインのトップエンドマザーボードもリリースしましたが、H55チップセットに基づくほとんどのマザーボードに特徴的なグラフィックス出力ポートがなく、これらのマザーボードとそれらのマザーボードの主な違いはP55チップセットに基づく前身。 H55チップセットをベースにしたAsrockマザーボードはすでに販売されており、その価格はSocket 1156のマザーボードの最初の波の背景に対して非常に低く、Socket 1366のマザーボードと大差ないことに注意してください。







革新のうち、追加の電力を必要としないUSB 2.0とeSATAコネクタの組み合わせに注目したいと思います。 このボードは、他の多くの最新ボードと同様に、IDEハードドライブとDVD、およびフロッピードライブを接続するための廃止されたコネクタのサポートを失いました。

マザーボードに付属のパッケージは、低価格帯に属していることを明確に示しています。これには、1組のSATAケーブル、背面パネルのダミープラグ、説明、およびドライバーと専用プログラムを備えたディスクのみが含まれます。 一方で、それは良いかもしれません-あなたは時々不必要なアクセサリーを過剰に支払う必要はありません。

マザーボードは、ダークブラウン、PCI、PCI-E、およびDIMMスロットで白と青で作られています。 一般的に、ボードはきれいに見えますが、ノースブリッジとブリッジ自体にラジエーターがないことにまだ慣れる必要があります。







Asrock H55DE3ボードは、LOTES製のプロセッサソケットを使用していますが、これは明確なプラスです。Foxconn製のSocket 1156ソケットは、プロセッサの対応するコンタクトへのソケットピンの圧力が異なるため、プロセッサの動作中に過度の加熱が発生し、場合によってはソケットが故障することがあります電圧の大幅な増加を伴う加速。







ボードにはDDR3メモリ用の4つのスロットがあります。 これらのコネクタの下部のラッチは、長いビデオカードのテキソライトに接触しません。 ただし、互換性の観点から、別の状況が考えられます-このマザーボードの幅は21cmです。つまり、切り捨てられたATX形式です。そのため、メモリコネクタはプロセッサソケットに十分に近いため、大きなヒートシンクを備えたメモリモジュールと組み合わせた大型のプロセッサ冷却システムのインストールが複雑になります。 私たちのケースでは、かなり大きなThermaltake SpinQ VTクーラーとメモリをそのようなラジエーターだけで使用した場合、幸いなことに、この競合は発生しませんでした。 1つのメモリモジュールの最大容量は4 GBで、合計で最大16 GBのシステムメモリを取り付けることができます。 サポートされる最大メモリ周波数は、通常モードで1600 MHz、オーバークロックモードで最大2600 MHzです。







拡張スロット：2つのPCI-E x16、2つのPCI-E x1、2つのPCI。 上部のPCI-E x16スロットはフルスピード（x16）で動作し、下部のスロットはx4モードでのみ動作します。 このボードは、デュアルチップビデオカードを使用する場合に、CrossfireXまたはQuad CrossfireXテクノロジーを使用して2つのビデオカードを結合する機能をサポートしています。 このマザーボードの本格的なPCI-E x16スロットとPCIスロットは、可能な限り間隔を空けてください。たとえば、外部サウンドカードの所有者に喜ばれるはずです。 ビデオカードと音声は互いに無駄になりません。







背面パネルには、キーボードを接続するためのコネクタ、6つのUSBデバイス（2つのコネクタがeSATAと組み合わされています）、ネットワークポート、オーディオ入力および出力、および画像出力、D-SUB、DVIおよびHDMI用の3つのコネクタがあります。 6個のUSBソケットでは不十分な場合、マザーボードに接続するためのはんだ付けされた出力が3つあるため、背面パネルにスプリッタまたはUSBソケット付きの追加ストリップを購入する必要があります。 Intel Core i5 / i3およびPentium G6950プロセッサに組み込まれたグラフィックコアは、DirectX 10、ピクセルシェーダーバージョン4.0をサポートし、ニーズに合わせて最大1759 MBのシステムメモリを選択できます。 また、統合されたビデオカードを備えたマザーボードの生産の競合他社に続いて、Intelはこのグラフィックコアに高解像度ビデオ再生のハードウェアアクセラレーションを追加しました。







ボードの右下には、4つのSATAIIコネクタがあります。 このマザーボードのサウンドについては、VT1718Sを介したかなり珍しい7.1チャンネルオーディオコーデックに適合しています。 外界との通信用-Realtek RTL8111DLギガビットイーサネットコントローラー。

Asrock H55DE3は、ICS 9LPRS142AGLF周波数発生器を使用します。 SetFSBプログラムではこの特定のジェネレーターはサポートされていませんが、番号で隣接するジェネレーターは優れています。 マザーボードや周波数ジェネレーターに共通する唯一のアドバイスは、明らかに安定した周波数であっても、ワンクリックで大きすぎる周波数ジャンプを避けることです。これにより、システムがクラッシュする可能性があります。







プロセッサ電源システムには、メインプロセッサコアに電力を供給する4つのフェーズと、プロセッサと同じカバーの下にある統合メモリコントローラとグラフィックコアに電力を供給する1つのフェーズが含まれます。 各フェーズには、1つのチョークと2つのMOSFETがあります。 MOSFETの冷却は提供されませんが、少なくとも深刻な加熱はありませんでした。さらに、電源システムにラジエーターが存在しないため、プロセッサクーラーとの競合を排除できます。 Clarkdaleプロセッサとペアになったマザーボードを使用する場合、この電源システムは十分すぎるほどです-負荷に余分なドローダウンや電力サージはありませんでした。

テスト中、使用されたオペレーティングシステムはMicrosoft Windows XP SP3でした。 マザーボードのBIOSバージョンは、2009年12月7日付けのP1.10です。



テストの中で最も興味があったのは、新しいi3 / i5がデュアルコアプロセッサのリーダーとして認められたWprimeと、帯域幅とメモリレイテンシに非常に敏感なWinrarです。プロセッサコアとQPIバスを使用してそれらの間で通信します。

Core i5 670プロセッサの公称乗数は26です; Turbo Boostテクノロジを使用すると、マルチスレッドモードで27、シングルスレッドモードで28になります。 したがって、プロセッサの周波数は、その種類に関係なく、負荷で3730 MHzを下回ることはありません。 この頻度では、Wprime 32Mの結果は12.9秒でした。 最新のクアッドコアIntelプロセッサー（Core 2 Quad）と最新のAMD Phenomプロセッサーは、この結果を達成するために2900〜3100 MHzの周波数で動作する必要があるという事実に注目したいと思います。 本当に印象的です。







組み込みのWinrarテストは興奮を引き起こしません。 もちろん、前世代のデュアルコアプロセッサを超える利点は非常に大きいですが、クアッドコアプロセッサと競合することはできません。 クラークデールは、ハイパースレッディングを有効にすると2060KB /秒、無効にすると1300になります（テスト中にターボブーストは無効になりました）。 比較のために、公称3200 MHzのAMD Phenom II x4 955は2400KB / s、Core i7 860-3000以上を獲得しています。















Thermaltake SpinQ VTクーラーは、今日の基準では最も生産的ではなく、電圧が1.32Vにわずかに上昇したため、安定性を維持しながらCore i5 670〜4500 MHzをオーバークロックできました。 この周波数で、組み込みWinrarベンチの結果がメモリ周波数とシステムバスの周波数に依存することを調査しました。











得られた結果に基づいて、結論は、主なことはシステムバス（BLCK）の周波数であることを示唆しています。その加速からの増加は非常に顕著です。 Asrock H55DE3が高周波数を達成する上で障害になるかどうかはまだ不明です。 私は急いで皆を喜ばせました-私はしませんでした、彼女は227 MHzのクロック発生器の周波数に従いました。 この周波数は、過度の冷却を行わないSocket 1156の既存のプロセッサをオーバークロックするのに十分です。PentiumG6950は、乗数が21のみで、Turbo Boostテクノロジーをサポートしていません。







4500 MHzの周波数でエベレストセットのメモリとプロセッサキャッシュの速度をテストします。







ちなみに、統合グラフィックの使用は結果にまったく影響しませんでした-外部グラフィックの使用と比較した違いはエラーに適合しました。

長いWprimeと短い4877でテストした場合、空冷でこのプロセッサインスタンスで達成されたベンチマークの最大周波数は4793 MHzに達しました。











空冷中のボードの適切な動作と適切なオーバークロックは、極端な冷却中にボードをテストする十分な理由のように思えました。







残念ながら、プロセッサに供給される最大可能電圧は1.6 Vに制限されており、オーバークロック全体が5400 MHzの周波数に制限されています。







この頻度で、CPU-Z検証に合格するだけでなく、短いWprimeを計算することもできました。







安定性をさらに要求する残りのテストは、5300 MHzの周波数で合格しました。































おわりに



Asrock H55DE3マザーボードは非常に良い印象を与えました。 私が思ったように、最初にその画像出力コネクタと現在流行している冷却ラジエーターの欠如を見て、オフィスのコンピューターだけでなく、オーバークロック用の非常に生産的なコンピューターでもお勧めできます。 その利点には、ターボブーストテクノロジを使用して増加した乗数を修正する機能、BLCKシステムバスの周波数での十分なオーバークロック、BIOSとWindowsの両方からのオーバークロック機能、およびボードの低価格が含まれます。
