コードスクールコースへの2日間無料アクセス

Code SchoolはEnvy Labsのプロジェクトです。このフレームワーク内には、さまざまなテクノロジーとプログラミング言語に関する約30のオンラインコースが既に実装されています。





学習プロセスは、すべての演習がブラウザで直接実行されている間、ビデオを見て学習したことを繰り返すことになります。 次のレベルに進むには、前のレベルのエクササイズを実行する必要があり、エクササイズに対してポイントが付与され、バッジが公開ページに追加されます。



コースは、グレッグポラック、ジェイソンヴァンルー、およびその他のEnvy Labsの従業員が指導します。 彼らは話していることに精通しているので、コースを見るのは素敵で面白いです（そして彼らはまた良い音楽を持っています）。 エラーも発生しますが、まだ修正されていない場合は、送信されたエラーのリストを作成するだけでなく、修正を試みます（ 例 ）。



ほとんどのコースにアクセスするには、有料のサブスクリプションを取得する必要があります（月額25ドル、場合によっては1株あたり9ドルのサブスクリプションを申し出ます）。 購読していない場合は、有料コースの最初のレベルを受講できます（または無料コースを受講できますが、現在は7つしかありません）。



幸いなことに、 New RelicはGeek Pride Dayのギフトを全員に贈ることを決定し、5月25日と26日（ EST ）にすべてのコースへの2日間の無料アクセスを支援しました。



したがって、愛するhabrayuzer、あなたはこれからの週末を便利に、そして楽しく過ごすための素晴らしいチャンスを持っています。 コースにアクセスするには、Code School Webサイトにリクエストを残す必要があります。
