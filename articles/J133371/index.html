価格と品質：予算のラップトップを選択する

通常、Habréのレビューでは、最も「洗練された」Lenovoラップトップモデルを選択します。 どちらが論理的なのか。ThinkPadX220やIdeaPad U300などのモデルでは、最新のテクノロジーがすべての栄光に見られます。 それらから、モバイルコンピュータ業界のさらなる発展を推測することができます。 個人的には、IPSディスプレイ、ハイブリッドディスクシステム（HDD + SSD）、およびプロフェッショナルモデルだけでなく、通常のカスタムラップトップでも金属の使用に依存しています。



一方、予算のラップトップは、私やコンピューター出版物からの同僚のレビューからはあまり注目されません。 おそらくこれは完全にふさわしくない。 確かに、信頼性が高く、便利で強力なラップトップを作成することは、同時に妥当な費用がかかりますが、クールな新製品をゼロから開発するよりも難しい場合があります。 この記事では、状況を修正し、予算セグメントの現在の3つのLenovoノートパソコンについてお話します。 私の選択には、超小型のThinkPad X121eラップトップ、ThinkPad Edge E420の主力製品、Lenovo G570デスクトップホーム交換品が含まれます。



ThinkPad X121e



11.6インチのディスプレイを備えたこの超小型ノートパソコンは、ロシアではまだ発売されていません。 その前身であるThinkPad X100eのレビューにより、実際にHabréでLenovoのブログが開かれました。 より有名なThinkPad TおよびXシリーズと比較すると、このラップトップは明らかに「見えません」-ケースはグラスファイバーと金属ではなくプラスチックであり、パフォーマンスはネットブックよりも優れていますが、最高のサンプルには達しません。 しかし、これは理論上です。 実際には、モデルは非常に素晴らしく、強く、非常に快適なキーボードを備えていることが判明しました。 そして、最も重要なことは、モデルのコストが15000ルーブル以下だったということです。それ以前は、この価格の新しいThinkpadは購入できませんでした。 ThinkPad X120eの現在利用可能な変更は、より経済的で「ホット」ではないAMDプロセッサーを使用し、14〜2万2ルーブルの価格で、後者のバージョンでは統合3Gモジュールで購入できます。







ThinkPad X121eの変更点は何ですか？ まず、このモデルで初めて、ユーザーはIntelプロセッサーとAMDプロセッサーのどちらかを選択できるようになります。 ここでの選択は、他のラップトップとほぼ同じです。Intelはもう少し強力で、AMDは少し安いです。 第二に、モデルのデザインが大幅に変更され、ラップトップはEdgeシリーズにより似たものになりました。







モデルの技術的特徴は次のとおりです。

プロセッサー ：AMD E-350 1.6 GHzまたはIntel Core i3-2357M 1.3 GHz

RAM ：最大8 GB DDR3

ハードディスク ：最大320 GBまたは128 GB SSD

グラフィック ：AMD Radeon 6310MまたはIntel HD 3000

画面 ：11.6インチ1366x768ピクセルマット

寸法 ：289x208x23.5 mm

重量 ：1.4kgから

バッテリー寿命 ：最大6時間



まとめると。 最初のThinkPad X100eがオフィスタスクで問題なく感じたが、写真を処理する際に永遠について真剣に考えていた場合、更新されたX121eは両方に簡単に対処できます。 そして、バイヤーはついにIntelプラットフォームとAMDプラットフォームを選択できるようになりました。これは朗報です。



ThinkPad Edge E420



この14インチのラップトップの価格は、Intel Core i3プロセッサーを搭載したバージョンの場合、19000ルーブルから始まります。 最も高価な-Intel Core i5プロセッサと4ギガバイトのRAMで-約2万5千ルーブルかかります。 どのラップトップを選択するかとよく聞かれますが、ゲーム用のモバイルコンピューターではなく、予算が限られている場合は、この特定のモデルをお勧めします。 ThinkPadシリーズは、常に信頼性の向上を特徴としており、低価格にもかかわらずEdgeラインナップも例外ではありません。 強力なディスプレイマウント、つや消し仕上げの頑丈なプラスチックケース、防湿機能付きキーボード：これらはすべて、慎重に取り扱わなくても、ラップトップの生存率を高めます。







重量2.2キログラムのこのラップトップは、非常にモバイルのままであり、同時にフルサイズのキーボードと光学式ドライブを備えています。 X120eと同等のコストで、このモデルは、重量を犠牲にしますが、作業時に最大限の快適さを必要とする人に適しています。 便利なキーボードに加えて、通常のように大型のタッチパッドが追加されます-ジョイスティックとトラックポイントが重複しています。







仕様：

プロセッサー ：Intel Core i3またはCore i5

RAM ：最大8 GB DDR3

ハードディスク ：最大500 GB

グラフィックス ：Intel HD 3000

画面 ：14インチ1366x768ピクセルマット

寸法 ：343x233x28.7 mm

重量 ：2.2kgから

バッテリー寿命 ：標準バッテリーで最大6.2時間、延長バッテリーで最大11.7時間



Lenovo G570



まずは、Lenovoのノートブック製品ラインの簡単な概要で、内容を明確にします。 そのうち3つしかありません。これらは、おなじみのThinkPadとIdeaPad、および予算シリーズ（Lenovo Essential）です。 昔々はレノボ3000と呼ばれていましたが、それでもこの名前は時々価格表に現れます。 Lenovo G570は、最も手頃な価格のラップトップを提供する予算シリーズを正確に指します。 しかし、安い-必ずしも必ずしも悪いわけではありません。これは通常、メディア内のこれらのモデルに関する肯定的なフィードバックによって確認されます（たとえば、 こちら ）。

このモデルのコストは、14,000ルーブル（Intel Pentiumプロセッサを搭載したバージョンごと）から始まり、約25,000ルーブル（Intel Core i5）で終わります。 Lenovo G570プラスチックケースに初めてアルミニウムインサートが追加されました-タッチパッドの周囲にパームレストがあります。 このモデルは自宅で使用するように設計されており、HDMIを介して外部モニターまたはテレビに接続できます。







今年、17インチモデルが最初にGシリーズの14および15インチモデルに追加されました。 22,000ルーブルのコストで、このラップトップにははるかに強力なグラフィックアクセラレータRadeon HD6650Mが搭載されており、ゲームモデルと見なされる可能性があります。 この一連のモバイルコンピューターは、最大数の変更でも注目に値します。最大6 GBのRAMと最大750 GBのディスクストレージが提供される最大バージョンでは、あらゆる好みと予算にオプションがあります。



もちろん、これらの3つのモデルは、レノボのノートパソコンの予算の範囲に限定されません。 また、Intel Atomデュアルコアプロセッサを搭載したIdeaPad S100ネットブック、AMDプロセッサと統合グラフィックスを搭載した12インチIdeaPad S205についても言及できます。 重いカテゴリには、13インチのIdeaPad V370（1万9000ルーブル）と、14インチのGシリーズラップトップ（価格タグが18,000ルーブルのLenovo G470）が含まれます。 したがって、誰もが最も手頃な価格のカテゴリで好みのLenovoノートパソコンを見つけることができます。
