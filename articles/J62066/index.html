qutIM 0.2ベータ版-今バナナ

痛みを伴う3か月の待機の後、開発チームは、ジュビリー300回目のコミットで、すばらしいクロスプラットフォームqutIMメッセンジャーのベータ版を私たちに提示しました。 この間、多くの新しく興味深いものがすべて行われ、多くの書き直され、壊れ、修正され、新しい機能が追加され、作業が最適化されました。 アルファ版のリリース以来、9万人以上がそれをダウンロードしました（通常のビルドとsvリビジョンを除く）。ベータ版では、この数を増やしたいと思います:)



（写真への代替リンク）





最も楽しいニュースは、おそらく、プログラムのコアがいわゆる大きなレイヤー（CL、チャット、履歴など）に分割されたことです。これらのレイヤーは、プラグインの形で実装に置き換えることができます。 たとえば、これにより、プログラム自体を編集したり、カーネル自体として最新のアップデートを使用したりすることなく、このシステムの環境に最適なMac OS Xの連絡先リスト、チャットウィンドウ、通知などを独自に実装できます。およびqutIMプラグイン。 また、この機能を使用して、プログラムの「パーツ」のより簡単で代替的な実装を作成できます。



多数の修正および追加されたエラーに加えて、サポートされるプロトコルの数が増加しました。 現在、完全なアセンブリにはソーシャルサポートが含まれています。 VKontakteネットワーク。

また、twitterプラグインの作業が開始され、MSNプラグインを開発するための連絡先リストがコンパイルされています 。



シンプルなプラグインのリストも補充されました。その中に、リリースに含まれる可能性が高いPlugManを強調したいと思います。 プラグインを使用すると、qutIMから直接ワンクリックでコンテンツ（スタイル、翻訳、プラグインなど）をインストールできます。 現時点では、既存のコンテンツのパッケージを転送および作成する準備を進めています。



他のプラグインは、フォーラムの対応するセクションで見ることができます。



プログラム自体の改善点のうち、次の点に注意できます。
  • 複数のバグ修正（約300）
  • 会議とプライベートを1つのウィンドウで表示できるようになりました
  • 読み込み時間を大幅に短縮
  • ギルアカウントを登録する機能がついに追加されました
  • 改善され完成した翻訳
  • 信じられないほど改善された歴史の速度
  • ジャバーの大幅な改善
  • OpenSSLライブラリでの壊れた作業:)（しかしGNUTLSですべてがうまくいきます）
  • 最新のMail.ruの標準の機能をMRIMプラグインに追加する作業が開始されました
  • そして、はるかに




ベータ版は既にダウンロードできます：
  • Windows向けビルド（ICQのみ）
  • Windows向けのビルド（たくさんあります）
  • カーネルのソースといくつかのプラグインを含む.tar.gz


DebianまたはUbuntuで使用するには、次のリポジトリのいずれかを接続できます（ベータ版は午前中に公開されます）。

deb http://qutim.org/debian/testing testing main # Debian squeeze

deb http://qutim.org/debian/stable stable main # Debian lenny

deb http://qutim.org/debian/jaunty jaunty main # Ubuntu jaunty



wget -O - qutim.org/debian/archive.key | sudo apt-key add - #
