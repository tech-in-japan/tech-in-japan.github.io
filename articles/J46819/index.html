MDC 1.0.2.1ベータリリース

今日はMDCの次のリリースでした。 今回、バージョン1.0.2.1ベータ版が光を放ちました。 それへの道は長く厄介でした。 bugs.mdc.ruでユーザーが表明した意見やコメントを考慮に入れようとしました。 このリリースでは、Mac OS XとFreeBSDの最終的に登場したバージョンを強調したいと思います。 バージョン1.0.2.0 win32のユーザーは、自動更新システムを使用して1.0.2.1にアップグレードできます。









以下は、リリースに含まれる変更のリストです。

  1. Mac OS Xバージョン
  2. FreeBSDバージョン
  3. Ubuntu 32および64のパッケージをダウンロードできるようになりました。
  4. これで、FreeBSD 32および64のパッケージをダウンロードできます。
  5. win32の更新バグを修正＃79 ;
  6. ローカルおよびサーバー設定を操作するための改善されたメカニズム。
  7. LinuxでのMDC起動時間の短縮。
  8. プロキシ設定のバグを修正。
  9. 履歴ウィンドウ＃116のリンクに関するバグを修正。
  10. ロード履歴の表示がパーセンテージで表示されるようになりました。
  11. コレクションからアバターを選択する機能。
  12. フォント設定のバグを修正。
  13. ポップアップメッセージのバグ＃54を修正。
  14. 消えないログインウィンドウ＃108のバグを修正。
  15. アカウントを追加するときのバグを修正。
  16. ICQのパスワードは8文字に切り捨てられます＃129 。
  17. トレイメニュー項目、連絡先リストに追加され、連絡先シートウィンドウが開きます。
  18. トレイアイコンを1回クリックするだけで連絡先リストが表示されるようになりました
  19. 現在、トレイ内の新しいメッセージのアイコンを右（Mac OsXの場合は左）をクリックすると、未読メッセージの送信者のリストを含むメニューが表示されます。
  20. Linux、Mac OsX、win32のウィンドウの可視性と動作に関するバグを修正＃59 ;
  21. MDCが表示されているときにトレイアイコンをクリックすると、最小化されます（まだwin32用ではありません） ＃89 。


ArchLinuxのパッケージは準備中です。



Windows、Linux、FreeBSD、Mac OS X向けのMDCはこちらからダウンロードできます 。
