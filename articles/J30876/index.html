Microsoftが2007年にYahooを買収する10の理由

1.検索エンジン

不気味なマイクロソフトの検索エンジンは、同社が現在直面しているほぼ最大の問題です。 検索エンジンがアイドル状態のとき、広告が表示されないという事実にあります。 MicrosoftがYahoo検索を購入した場合、これはユーザーを取り戻すための大きな一歩となります。





2. Yahoo!の序曲 検索マーケティング。

マイクロソフトは、Overtureとして知られる会社とmsn.comに広告を表示することに同意しています。 現在、彼女は自分で広告を表示しようとしていますが、事態は非常に厳しく進んでいます。 Microsoft Adcenterのようなほとんどの広告主は、検索エンジンから来る人が少なすぎることを認めています。



3. Yahoo! パブリッシャーネットワーク

私の意見では、YPNは最高のコンテンツネットワークです。 彼らには優秀な管理チームと、 質問がある場合に生きている人が答える電話番号があります。 Yahoo!と同じレベルのユーザーサポートを持っている人はいません。 パブリッシャーネットワーク



4. Flickr

マイクロソフトにも同様の製品があるかもしれませんが、Flickrのシェアは大きいです。



5. del.icio.us

Googleがすべての「両面」広告スロットを購入している間、Microsoftはすぐに主要なプレーヤーを獲得することができます。



6.人々。

マイクロソフトには、新しい業界に精通していない昔ながらの人がたくさんいます。 彼らは、ZendやMysqlなどの競合するオープンソースプロジェクトに貢献し始めています。 Jeremy Zawodny（Mysqlに関する本の著者でありツールの作成者）などのYahooからの人々の購入により、Microsoftは検索業界で地位を失うだけでなく、サーバーテクノロジーZendu（php）やその他のオープンソースプロジェクトでも市場シェアを失ったことを理解します。 Windows Serverの市場シェアを維持したい場合、優れたYahoo開発者を獲得することで、オープンソースプロジェクトとの統合への正しい道が開けると信じています。



7.ビデオ。

MicrosoftにはSoapboxがあり 、YahooにはYahoo!があります。 映像

これらを組み合わせて、ビデオ市場で高いシェアを獲得できます。 一般に、その重要性には1つの理由があり、それは「ビデオでの広告」という2つの単語で説明できます。 私はベータテストに参加しました。ベータテストでは、Googleがコマーシャルのコマーシャルをチェックし、その2か月後にYouTubeが購入しました。 広告市場のシェアに加えて。



8.コミュニティ。

YahooにはYahoo Groupsがあり、サイズはMySpaceに似ています。 groups.yahoo.comを聞いたことがありますか？ ああ、おそらくあなたはそれらを忘れていました。 それらを購入すると、広告を表示できる人がすぐに表示されます。 広告市場のシェアはさらに増加し​​ます。



9.ビジネスディレクトリ

マイクロソフトはこの分野での努力を放棄しました。 また、Yahooは常に有料のディレクトリを用意しています。 これは、ネット上で最も尊敬されるディレクトリです。 300ドルの年会費でさえ、ウェブマスターは彼に加わるために並んでいます。 同時に、同様のGoogleプロジェクトと同様に、これはDMOZを中心とした取るに足らないプロジェクトです。



10. Yahooが注文をもたらしました

彼女との統合がさらに容易になります。



これは起こりますか？ 私の意見では、これは明らかです。 マイクロソフトが新しいインターネット経済で競争したい場合、検索市場で大きなシェアが必要です。 そしてヤフーは彼らの唯一の道です。
