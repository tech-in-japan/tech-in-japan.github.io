ペンのふりをするマウス

コンピューターのマウスの設計では、概念的に新しいものを思い付くことは困難です。 しかし、日本人は成功しました。 日本企業Elecomの人間工学の専門家が特殊な形状のマウスを誕生させ、通常のボールペンをつかむのに指が馴染みのある位置を占めるようにします（少なくとも、この位置に指を近づけます）。 この図は、従来のペンと新しいScope Node M-SN1ULマウスをつかむときの指の位置を比較しています。







エンジニアは、人差し指が落ちて中指が戻るように、意図的に1つのボタンを前に押して下に下げました。 親指も離れるように、マウスの左側に肥厚があります。 指のこの位置は、手で書くことができる人にはもっと馴染みがあると考えられています。 したがって、このようなマウスで作業する方が簡単で便利であり、手が疲れにくい。







さらに、レーザーセンサー（解像度1600 dpi）は、「仮想ペン」がマウスの底の紙（親指と人差し指の間）に接触する場所に近づけられました。 おそらく、開発者は、この方法でこのように画面上でカーソルを移動する方が便利になると考えています。



Scope Node M-SN1ULマウスは、約64ドルで日本の店舗でのみ販売されています。
