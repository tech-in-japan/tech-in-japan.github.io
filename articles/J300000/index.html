ブロックチェーンのmerc兵-彼らはどのようになりますか？





こんにちは おそらく、 Dash暗号通貨の機能についてすでにご存知ですか？ 以前に公開した資料で、特にダッシュの重要な利点の1つ、つまりプロトコルに組み込まれた分散管理と資金調達システムについて書いた。 一番下の行は、投票の結果に応じて、このシステムが新しいブロックに対する報酬の10％を分配することです（ビットコインでは、すべて100％が鉱夫だけになります）-Dashエコシステムの開発を目的としたプロジェクト。 これについての詳細はこことここにあります 。



これまではこれは純粋に理論的なものでしたが、最近ダッシュネットワークに予算案を個人的に投稿し、投票の結果として250 DASHの資金を受け取りました（現在のレートでは約12万ルーブル）この記事ではそれについて話してください。





だから...通常、暗号通貨セクターの急速な発展に参加する機会は、何らかの「投資」を行う必要があることを意味します（ほとんどの人にはお金がありません）。 私の経験では、代替の側面、つまり、何も投資せずに、ダッシュなどの分散自治組織（DAO）から「給与」を受け取る可能性について語っています。 Dashには、分散管理システムだけでなく、（特に重要な）自律的で分散型の統合された資金調達システムがあります。 「ブロックチェーンのmerc兵」になる機会に興味がありますか（このシステムでの合計支払い額は1か月あたり7.400 DASH（現在のレートでは> 50.000 USD）です）。 それでは行こう！



そのため、毎月Masternodeオペレーター（それぞれ1票ずつ）が投票を実施します。これは、分散ネットワークのプロトコルに組み込まれています。 彼らは個々のプロジェクトに対して「賛成」と「反対」の両方を投票できます。 このシステムが2015年に発売されたとき、良いオファーがある程度不足していましたが、現時点では、限られた開発予算をめぐるかなり激しい競争が目撃されています。 たとえば、必要最小限の票を獲得した提案の一部は、他のより人気のある代替予算提案に取って代わられたため、お金を受け取れませんでした。



4月の投票のステータスは次の表で確認できます。





5月の現在のオファーのステータスは、 http ： //dashvotetracker.comで簡単に追跡できます。



私は2014年からDash暗号通貨コミュニティのアクティブライフに参加していますが、この間、特定の肯定的な評判を得ることができました（多くのプロモーション資料を作成し、記事を書いたなど）。これにより、投票時に必要なネットワークサポートを得ることができました。 私はボランティアの原則に基づいて活動を行いましたが、場合によっては「半製品」の購入やフリーランサーの雇用などにお金を費やす必要があります。



長い間私の仕事の結果を追い続けてきた人々は、私のイニシアチブをサポートしたいと思っていました。 それをダッシュ​​フォーラムで議論することにしました 。



私の提案に加えて、Dashプロジェクトに利益をもたらす可能性のある他の多くの興味深いプロジェクトがサポートされました。 それらの中には次のものがあります：

  • コア開発チームのメンバーを励ます;
  • DashのLamassuビットコインへの統合。
  • マイアミでのビットコイン会議のためのDash'n'Drink装置の作成に対する払い戻し。
  • マーケティングとプロモーションの予算。
  • Raspberry PiベースのROKOS製品にダッシュを追加。
  • WordPress WooCommerceを搭載したオンラインストアでDASH支払いを受け入れるためのプラグインを作成します。
  • 次世代のDash Evolutionの仕様の開発。
  • 「軽量ウォレット」を実行するために必要なDash Electrumサーバーを維持するための補償。
  • dash.orgドメインの購入に対する補償。
  • ...など。




このシステム内では、支払いは1回限りまたは毎月繰り返し可能です。現在、Dash Budgeting Systemが最終決定されており、取り消せない契約や他の多くのツールでも機能します。



誰でもネットワークからオファーを投稿できますが、すべてのオファーが資金を受け取るわけではありません。 もちろん、初心者への前払いをサポートする人はほとんどいません。初心者には、「お金をくれ、すべてがクールになる」などの約束しかありません。 この種の試みは、評判のない不可解な人々によってすでに行われており、成功裏に「吹き飛ばされた」。 同時に、多くの「サードパーティ企業および専門家」が資金提供されました。それ以前はDashプロジェクトとは関係がありませんでしたが、プログラミングからジャーナリズムまで、他の分野で前向きな評判を得ることができました...



Dashプロジェクトを提供するものがあり、競争力のある価格と明確な正当性を提供する準備ができている場合、Dashブロックチェーンから直接お金を得るのは大したことではありませんでした。 Dashウォレットターミナルだけでなく、適応された便利なWebインターフェースからもオファーを配置できます-オファーを公開した次のサイトを使用することをお勧めします： www.dashwhale.org/budget



ただし、投稿に必要な5つのDASH提案を使用する前に（有料配置はスパム攻撃に対する保護です）、予備的な議論を含むdashtalk.orgフォーラムでトピックを作成し、提案に関するコミュニティの意見や推奨事項を確認することを強くお勧めします。 （あなたの提案の議論とロビー活動のために、コミュニティの大部分はロシア語を話さないので、英語を話すことが望ましいです）



手順に関するヘルプが必要な場合は、いつでもフォーラムですべてを明確にすることができます（ロシア語のセクションもあります）-個人的な電子メールで私に書いてください-私はできる限りお手伝いします！
