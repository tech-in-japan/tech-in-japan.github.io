ロシアAIカップ2012



2012年10月29日、 ロシアのAIカップ2012：CodeTanksと呼ばれるプログラマ向けのコンテストを開始しました。 いいえ、ここでは速度のためにアルゴリズムの問​​題を解決する必要はありません-今回は参加者が戦車の人工知能を書き、戦闘に参加します。





なに？



ロシアAIカップは、オドノクラスニキとサラトフ州立大学のチームによる新しいプロジェクトです。 AIカップは、Mail.Ru Groupが才能のあるITプロフェッショナル向けに開催する競技会のリストの3番目です。毎年恒例のロシアコードカップスポーツプログラミング競技会と、 ロシアデザインカップのデザイナーとデザイナーのための競技会があります。

競争の一環として、参加者はゲーム戦略をプログラミングする巨大な戦闘ロボットの軍隊の作成に手を試すことができます。 初心者のプログラマーと専門家の両方がこのコンテストに参加するように招待されています。 特別な知識は必要ありません-基本的なプログラミングスキルで十分です。

サポートされている競合言語-C ++、C＃、Java、Pascal、Python 2、Python3。競合ウェブサイト-http://russianaicup.ru

最小限の（最高速度！）Java戦略の例を次に示します。 



public final class MyStrategy implements Strategy { @Override public void move(Tank self, World world, Move move) { move.setLeftTrackPower(1.0D); move.setRightTrackPower(1.0D); move.setFireType(FireType.PREMIUM_PREFERRED); } @Override public TankType selectTank(int tankIndex, int teamSize) { return TankType.MEDIUM; } }
