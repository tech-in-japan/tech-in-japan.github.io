印象とギャラクシーネクサスの初見（ビデオ）

みなさん、こんにちは

プレゼンテーションから1日も経たないうちに、 GoogleとSamsungの待望の新製品を手にすることができました。 あまり厳しく判断しないでください ガジェットのファームウェアは非常に未加工であり、香港で表示されたものとはかけ離れています。 Google電話ケースはプラスチック製で、前のGoogle Nexus Sのようにわずかに湾曲した画面を持っています。







スマートフォンの画面が大きすぎることについて多くの話がありました。実際、対角線は4.65インチです。 しかし同時に、デバイス自体は「ジャイアント」または「シャベル」ではないようです。







画面にボタンを配置するためにディスプレイが拡大され、作業面はNexus Sとほぼ同じままでした。 背面パネルはSamsung Galaxy SIIのカバーに似ていますが、少し粗めです。 今回、Googleの電話は間違いなくロシアに届きます。公式の技術的特徴は以下のとおりです。

  • ネットワーク ：HSPA +、HSUPA GSM 850/900/1900/1700/2100、EDGE / GPRS 850/900/1800/1900、LTE（地域によって異なります）
  • プロセッサー ：デュアルコア1.2 GHz（おそらくTI OMAP 4460）
  • メモリー ：1GB（RAM）+ 16GB / 32GBの内部メモリー
  • 画面 ：4.65インチ、スーパーAMOLED、解像度1280X720（HD）
  • OS ：Android 4.0（Ice Cream Sandwich）、ビデオは以前のテストバージョンを示しています
  • カメラ ：メイン：5 MP、LEDフラッシュ、ゼロシャッターラグ、shot2shot、フロント：1.3MP
  • データ転送 ：Bluetooth 3.0、USB 2.0、Wi-Fi 802.11 a / b / g / n（2.4 GHz / 5 GHz）、NFCテクノロジー
  • ビデオ ：コーデック：MPEG4 / H.263 / H.264、再生：1080p @ 30フレーム/秒、記録：1080pフルHD @ 30フレーム/秒
  • 寸法 ：135.5 x 67.94 x 8.94 mm、重量135 g
  • バッテリー ：1750 mAh Li-on
  • オプション ：加速度計、コンパス、ジャイロスコープ、近接センサー、圧力センサー


一般的に、電話は非常に快適な印象を与えます。 ロシアでの新規性がまだ不明な場合、価格も報告されていません。
