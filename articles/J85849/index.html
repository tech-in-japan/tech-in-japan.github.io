Acer Aspire Revolutionに代表されるNvidia ION

6か月前、Acer Aspire Revolutionネットトップをテストしましたが、現在の理由から、この資料を公開することにしました。 ネットトップはロシアで積極的に登場し始めているため、Acerから製品をhaberograter裁判所に提出することが重要であると考えています。







したがって、プラットフォームの作成者は、サイズが最小で機能が最大の新世代のITテクノロジーを目にします。



Intelは、Nvidia IONプラットフォームのリリースに先立って、それを死んだと呼び、費用対効果が高くありません。 しかし、プラットフォームをよく知った後、インテルの立場を完全に拒否しました。







Nvidiaをサポートした最初の企業の1つは、ノートブックの売上高で主導的地位を獲得したAcerでした。 同社は、優れた価格/性能比で顧客を獲得しています。 品質も引き上げて、エイサーがリードしました。 そして今、エイサーはネットトップ-エイサー革命を導入した最初の1つです。 Acer Revolutionには、コストと構成が異なる3つのバリエーションがあります。



1）NVIDIA ION搭載Intel Atom N230、8GB SSD HDD、1GB RAM、Wi-Fi、Linux、キーボードとマウス、VESAマウント。

2）NVIDIA ION 160GB HDD、2GB RAM、Wi-Fi、Vista Home Premium、キーボードとマウス、VESAマウントを搭載したIntel Atom N230。

3）NVIDIA ION、160GB HDD、2GB RAM、Wi-Fi、Vista Home Premium、キーボードとマウス、VESAマウント、ゲームパックを搭載したIntel Atom N230。



160 Gbのハードドライブを搭載し、ゲームパックを搭載していない製品について知ることができます。 IONが第2世代を受け取り、Intel Atom 330プロセッサがプラットフォームに新しい命を吹き込んだことに注意してください（IONを購入するときは、そのようなプロセッサを搭載した製品に注意することをお勧めします）。







コンパクトなボックスには、6つのUSBポート、カードリーダー、スピーカーとマイクのコネクタ、eSata、HDMI、Lan、VGAなど、考えられないすべてのものが含まれています。 スタンドにネットトップを取り付けると、全体の寸法が180x180x30 mmになります。



さて、ここにCPU-Zが示すものがあります：







次に、システムBIOSからの写真を紹介します。驚くことに、BIOSはラップトップというよりもコンピューターのようであることが判明し、さらに嬉しくなりました。











このタブには、システム、プロセッサ、クロック周波数、およびRAMの量に関するすべての情報が表示されます。















ここで、Acerのエンジニアは、インテル独自のテクノロジーを有効/無効にする機能を残しました。







そして、ここでは組み込みのコントローラーがオン/オフになります。











ケースの寸法の温度は通常の範囲内であり、同時にネットトップでノイズが発生します。







提示されたBIOSの最後のタブから判断すると、この「小さなおもちゃ」も分散できることがわかります。 Acerのエンジニアは2つの動作モードを用意しました。デフォルトは、システムバスの周波数が533 MHz以上で、生産性は667 MHzです。これは、周波数1600 MHzと2000 MHzに対応します。 それとは別に、プロセッサとRAMは非同期でオーバークロックできることに注意してください。このような状況では、マイナスよりもプラスになります。 BIOSからタイミングを変更することはできず、MemSetはWindowsからは機能しないため、これは便利です。 たとえば、800 MHzの周波数を超えた場合-既に802 MHzでタイミングがCL5からCL6に切り替わり、パフォーマンスが低下します。







プラットフォームは、1.6 GHzにデフォルト設定され、2.0 GhHzにオーバークロックされた2つのモードでテストされました。 最大オーバークロックは2060 MHzでした： http : //valid.canardpc.com/show_oc.php?id=578892

Super Pi1Mは、プロセッサの周波数とメモリのバンドルに応じて、シングルスレッドテストです。 クロック速度の増加に伴う生産性の向上を適切に反映しています。 結果が小さいほど優れています。







記録は非常に遠いですが、それにもかかわらず、パフォーマンス状況は数字Piの「カウント」の変更から変わりません。 結果が小さいほど優れています。







繰り返しになりますが、最新のPCでの数値は非常に小さいですが、Crysisでプレイすることはできません。











また、wPrimeは、Intel Hyper-Threading Technologyがどのように機能するかを示しています。 インテルCore I7のトップセグメントでこのテクノロジーを使用することにはまだ慣れていますが、ここではオン/オフの違いは40秒です。 ただし、プロセッサのパフォーマンスが向上すると、このデルタが減少することを忘れないでください。 しかし、Atomなどのプロセッサでは、ハイパースレッディングテクノロジーが絶対に必要です。







古き良き3DMark03は、プロセッサの影響に負けなかったテストの1つですが、実際のグラフィックステストはビデオサブシステムに依存します。 内蔵のNvidia 9400GTカードは、映画の鑑賞、簡単な写真処理には十分ですが、それ以上ではありません。 適切な設定でゲームをプレイすることはできません。 また、最初のテストによる図の違いは「飛行機」であり、これはプロセッサに依存する唯一のものです。







5番目のブランドでは、プロセッサーをオーバークロックすることで約450のオウムを取得します。







6番目のマークには、オウムは100匹しかいませんが、その一方で、上記の13パーセントは不要ではありません。



さて、青いテストエベレスト-それぞれデフォルトと加速。

おわりに



IONを知ることによる印象は好意的なものであり、プラットフォームは非常に成功し、Intel Atomプロセッサにとって非常に生産的であることが判明しました。 Acer Revoは、Nvidia IONに基づく他のネットトップメーカーが同等にする必要のある品質基準です。 近い将来、他のメーカーの競争力のあるものが見られることを願っています。 しかし、プラットフォーム全体のわずかなマイナスは、ユーザーコンポーネントの多様性が驚くことではないということだけです。 メモリとハードドライブ、およびプロセッサでのみプレイできます。
