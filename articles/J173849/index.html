Windows 8用のアプリケーションを開発するための無料のオンラインスクール





4月1日から、National Open University Intuitとともに、Windows 8用のアプリケーションを開発するための無料のオンラインスクールを開始することをお知らせします。 。



ここでトレーニングコースにサインアップできます： aka.ms/w8school



コースは6 + 3週間続き、講義、オンライン相談、研究室および学期論文が含まれます。 講義と相談は、Microsoft Russia（Department of Strategic Technologies）の従業員によって行われます。





理論的および実用的な部分



最初の部分は4月1日（コースの開始）に始まり、6週間続きます（5月12日まで）。 第1部では、毎週：

1.マイクロソフトの専門家による新しい講義ブロック（ビデオ録画）がアップロードされています

2.実験室作業の次のタスクが発行されます

3.質問への回答を含むリモートオンライン相談（Lync）を実施しました



講義



講義は、HTML / JSまたはXAML / C＃を使用したWindows 8のアプリケーションの設計と開発の問題を扱う、特定のトピックに関する短いビデオ録画（30分以内）です。 学生は、最も便利なプログラミング言語を個別に選択するか、両方の言語の講義を一度に聞くことができます。



従来、講義は4つのトラックに分かれています。



1.一般講義（トピックの紹介）

2.デザイン

3. XAML / Cでの開発＃

4. HTML / JS開発



実験室作業



実験室での作業の過程で、各生徒は、Windows 8用のWinRTアプリケーションの作成と公開の重要な側面を理解するのに役立つ標準アプリケーション（自由裁量でカスタマイズ可能）を作成する必要があります。また、最終的な順位を得るために、アプリケーションの最終バージョンをWindowsストアに配置します。



講義コースに正常に参加し、Windowsストアにアプリケーションを投稿した学生は、コースの仮想卒業証書を受け取り、コースの2番目のパートに参加できます。



コース講師



このコースは、ロシアのマイクロソフトオフィスによって教えられています。

  • ナタリア・エフィムツェバ
  • マリア・アニキナ
  • コンスタンチン・キチンスキー




コースワーク



コースワークは、Windows 8向けのユニークなアプリケーションを作成するための、1人または複数の学生（1つのグループに3人以下）の独立したプロジェクトです。 + 4週間（6月9日まで）。



5月13日からメインコースを修了した後、最初のパートを無事に通過した学生のために、マイクロソフトの専門家との設計およびアプリケーション開発に関する個別の協議が（遠隔または直接（モスクワ））開催されます。



ペーパーという用語の出力は、6月16日までにWindowsストアで公開されるアプリケーションです。 用語集に合格した学生は、オンラインコースが正常に終了したことを示す仮想の卒業証書を受け取り、学生の作品コンテストに応募して参加し、マイクロソフトから応募のプロモーションを受け取ります。 受賞者は、マイクロソフトの専門審査員によって決定されます。
