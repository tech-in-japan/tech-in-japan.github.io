CES 2014の結果：新しいSamsungカメラ

こんにちは、Habr！



本日、国際家電展示会CES 2014の結果に続く一連の投稿の一環として、Samsungが発表した新しいデジタルカメラについて詳しく説明します。







そして、今年のフラッグシップである最新世代のデジタルカメラSamsung NX30から始めます。 改善された20.3MP APS-C CMOSセンサー、より高速で正確なオートフォーカスのためのNX AF System II、1/8000シャッタースピード、連続撮影（最大9フレーム/秒）-これらは新しいカメラを際立たせる機能です。



80度回転できるチルト可能な電子ビューファインダー（XGA、解像度2.359K）は、被写体自体が何かを遮っていても、新しい角度や視点を見つけやすくします。 水平に180度、垂直に270回転する3インチのAMOLEDタッチスクリーンは、どこからでも写真を撮ることができ、必要に応じてセルフポートレートを作成できます。



サムスンNX30は、スマートカメラ会社の開発における新しい段階を表しています。 NFCとWi-Fiをサポートしており、映像をモバイルデバイスに転送するための多くの機能があります。 Tag＆Goを使用すると、NFC対応のスマートフォンまたはタブレットに写真を即座に転送できます。フォトビーム機能は写真とビデオの両方を送信できます-カメラをデバイスに持ち込むだけで、MobileLinkは複数の写真を一度に（同時に最大4つ）任意のスマートデバイスに転送できますAutoShare機能は、ユーザーのスマートフォンまたはタブレットの写真を完全に自動的にバックアップします。 リモートビューファインダーRemote Viewfinder Proを使用すると、モバイルデバイスを使用してカメラを制御できます。シャッタースピードや絞りなどのインジケーターは、リモートで手動で設定できます。



次世代のDRIMeIVイメージプロセッサと強力なDSPチップは、フルHD / 60pの写真とビデオのキャプチャ品質を提供します。 カメラISO 100-25600の高い光感度により、暗い場所でもまともな写真を撮ることができ、OIS Duo画像安定化技術-動いている物体の鮮明なショットをキャプチャできます。 OLEDカラー機能のおかげで、写真は優れたコントラストを誇っています。





人気のDropboxクラウドストレージサービス（ロシア向け）がNX30にプリインストールされています-これは、Dropboxに直接写真を送信できる市場で最初のカメラです。 ちなみに、他の多くの国では、写真をFlickrサービスに直接配信できます。



3Dの写真やビデオの最新の撮影のサポートなしではありません。カメラはSamsung 45mm F1.8 2D / 3Dレンズと組み合わせてこれを行うことができます。 互換性のある3Dデバイス（たとえば、Samsungの3D TVやモニター）で、結果の「ボリューム」コンテンツを再生できます。



サムスンNX30の機能は、コンパクトでありながら強力なカメラを必要とするオペレーターにとって優れたソリューションです。 ステレオサウンドでフルHD品質で録音できることに加えて、このモデルには標準の3.5 mmマイク入力が装備されています。 録音された音声のレベルはカメラ画面に表示されるため、ユーザーは撮影中にパラメーターを直接追跡できます。 さらに、サウンドトラックの品質を最適化するために、手動で設定することもできます。 フルHD / 30p出力のHDMIライブストリーミング機能により、カメラをディスプレイ、レコーダー、またはHDMIインターフェイスをサポートする他のデバイスに接続できます。



直感的な操作は、Samsung NX30の主要な機能の1つです。 2つのカスタムモードカスタムモードはモードダイヤルを使用して選択します。さらに、ユーザーはカメラのメモリに保存された独自のプリセットをさらに10個作成できます。 切り替えホイールは、連続撮影、ブラケティング、自画像タイマーなど、最も一般的に使用されるモードへのアクセスも提供します。



Samsung i-Functionを使用すると、シャッター速度や絞りなどの設定にすぐにアクセスできます。 また、新しいi-Function Plusシステムにより、経験豊富な写真家がコントロールキーを再プログラムできます。
