レビュー、habrのヒント、推奨事項をありがとう。 ほとんどの場合、ソースコードのフォーマットに問題がありました。 残念ながら、発生した主な問題は、ソースコードの配色です。

何らかの理由で、サイト管理者は HTMLの 完全なサポートを導入しませんでし た。最初に発生した問題は HTMLの 書式設定 でした。

その理由は、 style 属性は サポートされていない が、< font color = '...' /> タグはサポートされているためです

おそらく多くの人がすでに記事をHabra向けに変換するためのユーティリティを実装しているでしょう-私も参加して、 HTML を変換するための小さなユーティリティを出すことにしました 。



たとえば、ハブに次のコードを投稿した場合：





    <p style = "color：red">

        赤

    </ p>

    <p style = "color：blue">

        青

        <span style = "color：green">

            起動するには緑色）

        </ span>

    </ p>



その結果、次のようになります。

赤

青

起動するには緑色）

変換を2、3だけ行う必要がある場合、これは手動で行うことができますが、わかりやすくするために記事で配色が非常にうまく使用されている場合、すべてを手動で変換するのは非常に面倒です。



そのようなコードを変換するために、アドレスに小さなユーティリティを投稿しました。



http://htmlchangeonhabra.akhmed.ru/

-これまでのところ非常に簡単です-推奨事項を受け取ったら改善します。



上記の HTML コード を入力する と、次のコンテンツのコードが取得されます。





<FONT色=赤>赤<BR> </ FONT> <FONT色=青>青<FONT色=緑>そして起動するには緑）</ FONT> <BR> </ FONT>





この受け取ったコードを投稿すると、目的の結果が得られます。

赤

起動する 青 と緑）     



ちなみに、このシンプルなユーティリティの助けを借りて、ハブに投稿することができました。
