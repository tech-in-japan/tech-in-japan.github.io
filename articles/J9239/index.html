Sergey Burkov：ピザフォアグラの人気はどのくらいですか？

モスクワのGoogle Development Centerの責任者であるSergey Burkovが、今日のGoogle Developer Day、ロシアの駐在員事務所の仕事の分野などについて語っています。



Google Developer Dayはグローバルに開催されます-カリフォルニアの本社を含むいくつかのオフィスで。 独立した開発者に、さまざまなAPIの助けを借りて、Googleと連携する可能性を伝えるために開催されます。



現在の開発者の日は2番目の行ですが、ロシアにとっては新しいイベントです。最後の「開発者の日」はカリフォルニアで開催され、会社のマッピングサービスに専念しました。 このイベントは他のAPI専用です。 「開発者の日」の助けを借りて、独立した開発者にプラットフォームで何ができるかについてできる限り伝えたいと思います。 私たちはインターネットをエンドユーザーにとってより便利なものにしたいと考えており、私たちは自分ですべてを行うことはできないことを完全に理解しています。 多くの新しいアイデアが、まだ私たちと一緒に仕事をしていない開発者の頭に浮かぶので、私たちは彼らにプラットフォームを提供します-そして、彼らは新しいアプリケーションを思いつきます。



モスクワ開発センターの主な業務分野は何ですか？ あなたはこの質問によく答えますが、それでも...



私たちの目標は、まず第一に、米国で入手可能なすべての製品をロシアのユーザーが利用できるようにすることです。 別の重要な部分は、会社全体の新しい技術の開発です。



製品のローカライズに関して：ロシア語版のGoogleニュースを開き、バグ修正システムを検索エンジンに追加し、ロシア語辞書とiGoogle用のいくつかのガジェットを備えたGoogleツールバーをリリースしました。 そして今-検索とマップレットを含むロシアの地図、すなわち プログラマは、ロシア語の「マップレット」の作成に取りかかることができます。



ロシアのGoogleオフィスのプレゼンテーションで、あなたは労働条件と会社の雇用について詳しく話しました。 現在、空室状況はどうなっていますか？



それ以来、何も変わっていません。Googleは従業員を募集していますが、最高の従業員のみを採用しています。 バーは非常に高い：いくつかの段階で行われるインタビューで多くの質問がされます。 この場合、面接プロセスは非常に民主的です。複数のエンジニアが申請者に質問し、それぞれがその後レポートを作成します。 これらのレポートの平均データに基づいて、採用が決定されます。



面接を受けた専門家の数は十分ですか、それとも「人員不足」ですか？



ロシアには、優秀な開発者が十分にいますが、誰もがGoogleで働いているわけではありません。



一方では、複雑なタスクを処理できる十分なスペシャリストがいます。これは一般に、Googleマップで目立つはずです。 しかし、もっとあれば、私たちはそれらを働かせます。 面接に合格した全員を連れて行きます。制限はありません。



現在、何名の学生が会社でインターンシップをしていますか？ 彼らはどの大学から来たのか、何に取り組んでいるのか、どの秘密にアクセスできるのか？



私たちは、主要な大学から数人の学生を訓練しました。 たとえば、モスクワ州立大学から。 たとえば、mehmatを使用します。 彼らは、スタッフが取り組んでいるのと同じことに取り組んでいます。 インターンシップを取得するには、仕事に応募するときとほぼ同じ複雑な「フィルター」を通過する必要があります。



Googleに1年以上携わっています。 この時期の最高の成果は何だと思いますか？



おそらくカード。



そして人格の面で？



個人的に...ほとんどの場合-オフィスが成長しているという事実は、才能のあるプログラマーと非常に良いチームを作成することができます。 Googleで慣例となっているように、作品を正しく整理することは可能です。



今、あなたはあなた自身のプロジェクトを持っていますか、それとも会社にすべての時間を与えていますか？ 結局のところ、「80/20」というルールがあります。



はい、従業員は自分の時間の20％を自分のプロジェクトに費やすことができますが、これらのプロジェクトはまだGoogle向けに作成されています。 このルールのおかげでGoogleニュースが登場しました。 主なアイデアは、エンジニアがマネージャーに新しいプロジェクトの許可を求めるべきではないということです。



私に関しては、私の労働時間の150％が管理に費やされています。



つまり、80/20の豪華さは開発者のみが利用でき、管理者は利用できませんか？



（笑い）一般的に、私はそれを利用できますが、ただ...加算されません。



Googleに参加する前に持っていたスタートアップ-それらについて少し教えてください。



最初はビルボイノベーションズがありました。アレックスフリードは私のパートナーであり、ロシアの出身で体育学部の卒業生でもありました。 同社はコンピューター用のペダルを製造していました。 ペダルは、障害のある人、関節炎に苦しむ人、一般に-マウスをクリックするのが難しい、またはキーボードのキーを押すのが難しいと感じる人に必要です：デバイスは、マウスをクリックするか、キーを押すことでプログラムできます。



Friedとともに、暗号化会社であるInvincible Data Systemsが作成されました。 この会社はMicrosoft Encrypted File Systemの類似物を作成し、Microsoft製品よりも前に作成されました。 その後、Invincible Data Systemsはより大きな「プレーヤー」であるVASCO Data Securityに売却されました（昨年、VASCOは世界で最も急成長した10社の1つです）。



3番目の会社はDulanceで、製品の検索エンジンを作成しました。 Googleがそれを購入しました-そして、私たち全員がモスクワオフィスのバックボーンを作り上げました。



そして、これらすべての会社を設立する前に、準結晶の物理学を教えました。 準結晶とGoogle？..



（笑）いやいや。 これはGoogleが必要としない純粋な科学です。



一般的に、純粋な科学についての良い話がありますが。 前世紀の40-50年代には、数論に関与していたハーディという名前のそのような数学者がいました。 彼は、数論は純粋な科学であり、純粋な数学であり、誰にとっても役に立たず、実用的な応用がないことを自慢しました。 彼はそれを非常に誇りに思っていましたが、完全に数論に基づいた暗号が登場しました。



今日では、Panoramioの購入について知られるようになりました。 ロシアには大企業が興味を持つスタートアップがいると思いますか？



はい、彼らは少数です-しかし、彼らは表示され、それは非常に素晴らしいです。



4〜5年前でさえ、ロシア、中国、インドには才能のあるプログラマーがたくさんいると考えられていましたが、アイデアはありません。 人々は「オフショア」の注文を実行しましたが、プロジェクトを作成しませんでした。 現在、ロシアの状況は変わりました。興味深い企業があります。



なぜこれがすべて起こっているのですか？



正直なところ、わかりません。



結局のところ、アメリカのスタートアップは比較的新しい現象です。60年代にシリコンバレーが登場しました。 すべての革新はゼネラルエレクトリック、ゼロックス、AT＆Tなどの内部で行われました。 その後、社会はどういうわけか進化し、ビジネスモデルが出現しますが、それがどのように起こるかはわかりません。



「Googleの陰謀論」についてどう思いますか？ 会社が収集する情報量が増加しているという懸念を表明するかなりの数の出版物があり、それらはその後「他の目的」で使用されます。



世界では、一般に、本当の根拠がない多くの理論があります。 人々は空飛ぶ円盤について何かを思いつきたいので、これは同じオペラからのものです。



はい、Googleはユーザーに関する多くの情報を処理しますが、特定の人々に「縫い付けられた」ものではありません。 弊社に保存されている情報は一般化されています。 人が「パリのピザ」をリクエストするとき、私たちは彼が誰であるかには絶対に興味がありません。私たちは、フォアグラのリクエストよりもピザのリクエストの方がどれだけ人気があるかにより興味があります。 ユーザーに結び付けられている情報は最小限に抑えられます-そして、そのアクセスに関する厳格な企業ポリシーがあります。 このアクセスは確実に保護され、すべての従業員がアクセスできるわけではありません。



私たちは人々をスパイするつもりはありませんが、もちろん、一般化された情報に興味があります。検索エンジンを設定し、よりターゲットを絞った広告を出すのに役立つからです。



会社は何をしたか 検索、コンテンツ広告？



間違いなく、検索エンジン-そして、それは最も労力がかかります。 入力フィールドを備えたホワイトページのシンプルさは明らかです。背後には非常にトリッキーなアルゴリズムがあり、背後には何千人ものエンジニアの作業があります。



それはGoogle AdwordsとAdsenseではありませんか？ 結局のところ、利益を上げるのは彼らです。



それらも優れていますが、それでも基礎は検索です。
