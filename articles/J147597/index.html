Digitimes：Windows 8 OEMの価格

ソースを引用し、内部情報で知られているDigitimesリソースは、コンピューターメーカー向けにWindows 8のさまざまなバージョンの価格を発表しました。 。



このように：

  • Windows 8-60〜80ドル
  • Windows 8 Pro-80〜100ドル
  • Windows RT-50〜60ドル




最後の2つのバージョン（Win 8 ProとWindows RT）の価格は、Officeパッケージのコストを考慮して示されます。



また、メーカーはマイクロソフトの新しいシステムに大きな期待を寄せていますが、クリスマスの販売期間中にノートブックの販売を促進するはずですが、Windows 8のインストール数の注文は依然としてかなり控えめに見えます。 これは、Windows 8の多くの機能（タッチスクリーンなど）が新しいコンピューターをサポートするため、通常のラップトップよりも価格が高くなるためです。 Windows 8ベースのマシンの低価格化は、2013年の第2四半期にのみ予想されます。



Windows 8の小売価格はまだわかっていません。
