クラウド内のActive Directoryとバックアップ、サイトトラッキング、その他のWindows Azureアップデート





Windows Azureプラットフォームは、再び更新に満足しています。 今回は、Windows Azure Active Directoryサービスの公開、新しいWindows Azureバックアップサービスの発表、Webサイトの新しいトレース機能、および他のサービスの更新をお知らせします。



すべての更新の詳細は、 次のリンクのScott Guthrieのブログに掲載されています 。 以下に、提示されたノベルティの簡単な説明があります。



クラウド内のActive Directory



Windows Azureのお客様は、フル機能のクラウドサービスActive Directoryにアクセスできるようになりました。これにより、複数のサービスに対するシングルユーザー認証（シングルサインオン）のタスクを解決し、シングルユーザーディレクトリを維持し、企業内のActive Directoryとディレクトリデータを同期することができます。











Windows Azure Active Directoryは、クラウドでの本格的なディレクトリ実装です。 同時に、プラットフォームは、SAML 2.0、OData、WS-FEDなど、フェデレーションを提供するための一般的なオープン標準をサポートしています。 数日以内に、他の一般的な標準のサポートが追加されます：OAuth 2.0 / OpenID



Windows Azureブログでは、新しいプラットフォームのリリースと予備的なテスト手順について詳しく説明しています。 特に、次のデータがそこに与えられます：



  • Windows Azure Active Directoryは、Office365、Dynamics CRM online、Windows Intune、Windows Azure自体などのMicrosoft製品の認証を提供するためにすでに広く使用されています。
  • このサービスは、世界中の14のマイクロソフトデータセンターにあるソリューションを提供しています。
  • 過去90日間で、サービスは65億件以上の認証要求を処理し、99.97％以上の信頼性を維持しています。
  • サービスの開始以来、290万人以上のビジネス担当者がこのサービスを使用し、2650億件以上の認証要求を完了しました。


さらに、Windows Azure Active Directoryのパブリックリリースに伴い、REST APIを介したディレクトリへのプログラムによるアクセスを提供するWindows Azure Active Directory Graphサービスも公開されています。 ディレクトリグラフの詳細については、 こちらをご覧ください 。



Windows Azure Active Directoryの使用に関する詳細なドキュメントとガイドは、 ここにあります 。 このリンクでサポートされたプロトコルに関する情報を見つけることができます。 詳細なドキュメントはこちらをご覧ください 。



新しいWindows Azureバックアップサービス



他のニュースに加えて、Windows Azureファミリーの新しいサービス-バックアップも提供されます。 このサービスは、クラウドにWindows Serverバックアップを保存するための安全なインフラストラクチャを編成する可能性を提供します。



Windows Azureバックアップは、 Windows Server 2008 R2 SP1およびWindows Server 2012 、 Windows Server 2012 Essentials 、およびSystem Center Data Protection Manager 2012 SP1に基づくシステムからWindows Azureクラウドへの情報のバックアップをサポートしています。







Windows Azureバックアップストレージは、すばやく簡単に作成でき、セットアップが非常に簡単です。 増分バックアップがサポートされます。これにより、トラフィックとストレージスペースが節約され、複数のバージョンから選択したポイントに復元することができます。 データ圧縮と暗号化を整理するためのポリシーとすべての標準Windows Serverバックアップツールがサポートされています。



サービスのプレビュー中にアクセスするには、 次のリンクにリクエストを送信する必要があります。 確認は1〜2日以内に届きます。



詳細については、次のマニュアルから入手できます。



  • チュートリアル-Windows Azureバックアップエージェントを使用したバックアップのスケジュール
  • チュートリアル-Windows Azureバックアップエージェントを使用してファイルとフォルダーを回復する


Windows Azure Webサイトの監視と診断の改善



このWindows Azureの発表の波は、Windows Azure Webサイトに基づくソリューションの監視および診断システムの大幅な改善も示しています。



特に、FTPを介した後続のアクセスでログファイルに情報をトレースして保存する機能が追加されました。 さらに、分割されたトレースデータが利用可能になったときに受信するためのリアルタイムストリーミングオプションが提案されています。







新しい関数は、プラットフォームの既存の機能および.NETなどのライブラリトレースと完全に統合されます。 トレース機能は、Windows、Mac、Linuxのすべてのプラットフォームでサポートされています。



詳細については、Scott Hanselmanのブログの次のリンクを参照してください 。



Windows Azureのその他の革新



メジャーアップデートに加えて、このリリースでは、Windows Azureプラットフォームに小規模な主要新製品が導入されています。



  • Windows Azure HDIsight （Hadoop）-診断メトリックをリアルタイムで監視する機能を備えたダッシュボードが追加されました。他のWindows Azureサービスも同様です。
  • 操作のログ -指定されたサブスクリプションのサービスに変更を加えたユーザーの名前も操作のログに入力されます。
  • Windows Azure Media Services-ビデオ形式をエンコードするための事前定義済みプリセットの拡張セット。
  • Windows Azure仮想マシン -新しいVMのOSディスクのデフォルトサイズが増加し、VMの作成時にユーザー名を指定する機能が追加されました。


もう1つのニュースは、 VMDepotの Windows Azure コミュニティの Linux仮想マシンイメージのベースが200タイトルに増加したことです。



Windows Azureプラットフォームに関するニュースは近日公開予定です。
