SQL Server 2014



Windows Server 2012 R2のリリースと共に、SQL Server 2014データ管理システムの予備バージョンがTechEd Europe 2013でリリースされました。

Windows Azureで新しいMicrosoft製品を試す



幸いなことに、これらのすべての新製品、製品の新機能は、コンピューターにインストールしなくても個人的に試すことができます。



現在、Windows Server 2012 R2、SQL Server 2014、およびVisual Studio 2013 Previewは、Windows Azureギャラリーの仮想マシンイメージとして利用できます。



Windows Azureサブスクリプションを使用するか、 無料試用期間にサインアップして、Windows Azure仮想マシンで新しいサーバーを起動してください。







これを行うには、新しい仮想マシンの作成を開始し、ギャラリーで必要なサーバーを選択します：Windows Server 2012 R2、SQL Server 2014またはVisual Studio 2013。







仮想マシンの構成を指定し、管理者名とパスワードを入力します。







DNSと仮想マシンの地理的位置（データセンター）を指定して、仮想マシンモードを構成します。







仮想マシンの作成が開始されます。 作成を待ちます。数分かかる場合があります。







作成後、仮想マシンの管理パネルに移動し、「接続」機能を使用してRDP経由で仮想マシンに接続できます。







仮想マシンに接続することにより、Visual Studio 2013 Previewなどのインストール済みのMicrosoft製品を使用、調査、およびテストできます。
