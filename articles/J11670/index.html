Web 2.0-> Life 2.0

現在、誰もが「Web 2.0」のようなよく知られた用語を持っていますが、誰かがその無形性を証明し、それをマーケティング担当者にとって新しいトリックと呼ぶ特定の意味を入れています。

私については、これらの紛争は常識の粒を運んでいません。 結局のところ、この用語がWebのコンテキストにおいても普遍的ではなく、普遍的であることは既に明らかです。

実際、Web 2.0が意味することは、グローバリゼーションの構成要素の1つにすぎません。または、必要に応じて、情報発信の哲学や人々の一般的な生活様式を根本的に変える世界の変化の一部です。



Web 2.0は、インターネットの単純化、最適化、変換、および社会化です。 しかし、インターネットだけでなく、多くの分野で同様のプロセスがあり、またそうあるべきです。 たとえば、仕事を考えてみてください。Googleの従業員や、主に新技術の分野で働いている他の多くの会社が、ちょうど9時に働いて、ちょうど6のままで、ほこりっぽい退屈なオフィスに座っていないのはなぜですか？！ はい、それは知的人的資源と労働時間の効率的な使用ではないからです！



プールの近くにラップトップを置いて1時間座っている方が良いでしょう。しかしこの時間に、彼はあなたが何百万ドルも稼ぐことができるアイデアを出し、そして何よりも世界を前向きに変えるでしょう。 一日中オフィスに座って比garbage的に「ゴミに苦しむ」と言う必要はありません（これは確かに単純化されたモデルであり、現在のモデルの不完全さを示すためだけです）。 経営者は、従業員が安心してお金を稼ぐことができるようにすべてを行う必要がありますが、同時に彼は会社で仕事をすることを喜んでいるので、彼は私たちが世界をより良く変え、新しいものを作成し、無限にしないことを知っています大量の古紙をスタンピング！



誰かがこれはユートピアだと言うでしょう、誰もが自分のために大きな作品をつかむ方法についてのみ考えています。誰もが誰にとっても恩恵を受けることができる新しい何かの開発に興味がないということです。 しかし、ウィキペディアの例、および他の多くのソーシャルネットワークは、そうではないことを示しています。 しかし、どうして遠くまで行こうか、欧州連合の例を見てみましょう-これがすべての自発的な連合ではないなら、何か新しいもの、より良いもの、集団的利益のための同盟ではないものをつくるなら、それは何ですか？



なぜ私は上記の行を書き、変化が人間の生活のすべてまたはほとんどすべての分野に影響する、または少なくとも影響を与えるという事実に書いたのか

ソーシャルオリエンテーション、無料、便利さ、シンプルさなど、Webで新しい原則について話すのはなぜですか。 -次に、これらの原則はインターネットにのみ適用されますか？



しかし、実際の生活では、すべてをそのままにしておきましょう。

変更は、人間の生活のすべてまたはほとんどすべての領域に影響を与えるはずです。



たとえば、広告やマーケティングなどの分野を考えてみましょう。

今日の広告の総量は慢で愚かで、鮮やかな絵やキャッチーなスローガンを見た後、すぐにこのまたはそのサービスや製品を購入する愚かな人々のために設計されています。

マーケターとPRの人々は、彼らの製品についてとんでもない話とスローガンを思いつきます。そして、彼らにとっての主なことは、すべてを減らすことへの欲望です。



今日の広告は、商品やサービスの実際の価値や必要性について話をするのではなく、より多くの可能性を売りたいという欲求によって推進されています。

簡単に言えば、広告主は多くの場合、さまざまな形で顧客をだます。

彼らの哲学は私たちが販売することですが、人が本当にこれまたはそのサービスを必要とするかどうかは私たちにとって重要ではありません。 長い間、このアプローチは非常に普通であり、世界中で正当化されていました。



しかし、ネットワークが変化するように、世界も変化しています。 はい、以前と同じように行動できますが、新しいレベルの開発に進み、新しい哲学を受け入れることができます。 これは困難であり、この移行は、特にロシアやウクライナなどの国の現実では、全国規模で多くの時間がかかります。

しかし、これは特定のプロジェクトの枠組みの中で、特定の人々に対する新しい哲学の使用を妨げるものではありません。 この哲学は人々に理解可能であり、実践が示すように、あなたは成功することができます！
