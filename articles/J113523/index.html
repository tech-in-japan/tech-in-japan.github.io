Oracle Essbase 9-インストールとセットアップ

このトピックの目的は、製品であるOracle Essbaseを使用してHabrコミュニティを実践する最初の知り合いです。 ここでは、この製品のアーキテクチャについて読むことができます 。



現在（2011年2月）、現在のバージョンは11.1.2です。このバージョンでは、OracleとHyperionが再び統合に向けて大きな一歩を踏み出し、時々アプリケーションインフラストラクチャが複雑になりました。 このバージョンは、すでにシステムに精通しており、すべてのアーキテクチャ機能を理解している人に適しています。 インストールプロセスについては、テーマ別ブログで詳しく説明しています。



以前のバージョン9.3を使用してOLAP Oracle Essbaseエンジンをよく理解することをお勧めします。OLAPカーネルに関しては、バージョン11で追加のコンポーネントを設定するプロセス（負荷がかかる）に時間がかかるため、およびリソース。



したがって、インストールプロセスは3つの段階に分けることができます。







  1. 「シェアウェア」ソフトウェアの入手
  2. 設置
  3. カスタマイズ


最初のステップ





Oracleの「シェアウェア」ソフトウェアを入手するには、 EDeliveryポータルを使用することをお勧めします。 このポータルでは、フォームに記入し、Oracleライセンス契約の条件に同意する必要があります。 ところで、このポータルは、オラクルからソフトウェアの全範囲にアクセスするために使用すると便利です。



最初に行うことは、ドロップダウンリストからソフトウェアの種類を選択することです。この場合、「Hyperion Performance Management」です（新しいバージョンが必要な場合、ブランド変更により「Oracle Enterprise Performance Management」と呼ばれます）。 バージョン-9.3.1、Windows 32プラットフォーム







4つのファイルを取得する必要があります-Olapサーバー、そのクライアント（Excel経由でデータにアクセスするため）、管理コンソール（動作するにはリレーショナルデータベース（OracleまたはMs SQL）が必要、統合コンソールは多次元ストアフロント（OLAPキューブ）を作成するためのツールです）リレーショナルソース。



これらは、ファイルV14762-01、V14763-01、V14782-01、V14792-01です。



















ステップ2のインストール





インストールプロセスは、複雑で意味のあるアクションではありません。 「ウィザード」はタスクを適切に処理します。インストールプロセスの完了後に設定および確認する必要がある設定（システム環境変数）は3つだけです。



％ARBORPATH％= C：\ Hyperion \ AnalyticServices



％ESSLANG％= Russian_Russia.MS1251@Default



％HYPERION_HOME％= c：\ hyperion \



ステップ3-セットアップ





ソフトウェアを構成するには、対応するディレクトリにあるProgrammer-Configuratorを使用する必要があります







その中で、次のことを順番に行う必要があります

  1. Essbase管理コンソールのリレーショナルソースの構成
  2. EAS用のTomcatをインストールする
  3. EssbaseをWindowsサービスとして登録する


単一の認証サービス（Shared Services）の登録ポイントは、「機能」をもたらすため、意図的に無視されます。MSADとの統合は、OLAP学習の追加ステップです。







EssbaseおよびEASのデフォルトのユーザー名とパスワードはadmin / passwordであるため、このウィザードに合格すると、引き続き質問に答えます







デフォルトのポートは使用したままにします。







次に、EASのビジネスルールリポジトリのデータベースのタイプを指定します







私はOracleを使用していますが、テストにはMS SQL 2000 sp4 / 2005の方が優れています







Tomcat Javaアプリケーションサーバーを終了する







提案に同意し、構成を完了します







構成が完了したら、EASおよびessbaseサービスを開始する必要があります







正常性を確認するには、EASコンソール（admin / password）に移動して、Sample.Basicデモアプリケーションを開く必要があります。







管理者/パスワードとしてログイン







esbaseサーバーを接続する











右クリックしてデータをSample.Basicにロードします











キューブを計算する







結果-Essbase SpreadSheetを通してExcelを見ることができます
