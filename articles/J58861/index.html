EffeLoggerまたは同じ「ピナリック」

McBernar habrayzerの記事「個人の時間管理」を読んで、私は「ピナリク」のアイデアが本当に好きでした。 しかし、このためのプログラムを書くのではなく、 紙のピナリックを手に入れたら私は私ではありません:)



だから、私たちはあなたの効果のためにEffeLogger-Loggerに会います！



1対1の運用原則は、論文「ピナリク」の原則を繰り返しています。つまり、私たち自身が1日をどれだけ効果的に過ごしたかを評価します。 使い方は非常に直感的です。彼らは適切な日にクリックし、評価を選択しました...それで終わりです:)



このプログラムは、PyQt4を使用してPythonで記述されたクロスプラットフォームです。 Linuxの場合、PyQt4をインストールするだけで、PyQt4自体が他のすべてを引き出します。 Debianベースのディストリビューションの場合、これはチームによって行われます 

  $ sudo aptitude install python-qt4


Fedoraの場合-チーム 

  $ sudo yum install PyQt4


それ以外は-わからない:)



Windowsの場合、こちらからwww.python.org/ftp/python/2.6.2/python-2.6.2.msiからPython 2.6.2をダウンロードしてインストールする必要があります。したがって、こちらからwww.riverbankcomputing.co.uk/static/DownloadsからPyQt4をダウンロードしてインストールする必要があります。 /PyQt4/PyQt-Py2.6-gpl-4.4.4-2.exe



更新：

一方、作業は本格的で、プログラムはバージョン0.2に成長しました:)

変更点：

  • コメントを設定する機能
  • ウィンドウの位置/サイズを保存する
  • タイトル内の数字はコンテンツと一緒にスクロールしなくなります
  • 現在の日付表示
  • 色を設定する




ここで取り上げます： effelogger.googlecode.com/svn/branches/0.2/effelogger.py



Unixの場合、もちろん、スクリプトに追加の実行権限を与える必要があります。 

  $ chmod + x effelogger.py




そしていつものように、提案や提案は大歓迎です。
