ブラウザベースの2Dオンラインゲームを作成するためのRPG JS v2フレームワーク

最近、CanvasEngineエンジンに基づいたRPG JS v2フレームワークの2番目のバージョンがリリースされました。







基本情報



投稿者：サマリウム

ライセンス：GPL

言語：英語、フランス語



はじめに



-RPG JSは、ブラウザベースのRPGを作成するためのフレームワークです。 このフレームワークを使用するには、ゲームインフラストラクチャの作成を容易にする最小限のプログラミング知識が必要です。 （スクロール、衝突、移動など...）



-RPG JSはHTML 5およびJavascriptで記述されています。 ビットマップ画像、スプライトなどを操作するため。 EaselJSライブラリを使用



-RPG Maker XPのスクリプトは、RPG JS用のカードに変換できます



-ブラウザとの互換性：IE、Google Chrome、Firefox、Safari、Opera







メリット







-クライアントをダウンロードする必要はありません

-データベースをバインドする機能（AJAXリクエストを使用）

-サイト上の他のコンテンツとリンクする機能

-マウスを使用する機能

-RPG Makerに存在しないコマンドを追加する

-戦術RPGの統合

-MMORPGのような本格的なゲームを作成する機能

-iPhone / iPad / Archosでプレイ可能

-オープンソース



説明







RPG JSはRPG Makerと同じ構造を使用します。 マップをダウンロードし、プロパティとすべてを指定します:-)。 実際、マップ上で発生するイベントを示す必要があります。 次に、ページとコントロールの外観でイベントを指定する必要があります。

現時点では、フレームワークはすでに実装されています：

1）本格的なシングルプレーヤー（RPGに基づく）

2）イベントとの相互作用

3）マウス、キー、タッチスクリーンを使用した戦術RPG（小さなバグがあります）



エンジンの機能を示すビデオ：





また、このフレームワークには便利なジェネレーターが27.9ドルで入手できます。





タイルとオブジェクトのセットからマップをすばやく作成するための直感的なインターフェイス、衝突を考慮したプロパティ、ヒーローの動き、ゲーム以外のオブジェクトとの相互作用があります。

簡単に言えば、このジェネレータを使用してプログラミングの分野で十分な知識を持っていない人なら誰でも、ゲームの基本要素を作成できます。

多くのオブジェクト、武器、鎧、スキルなどを備えた大規模なゲームデータベースのおかげで、ゲームのヒーローとその周辺の世界の冒険を簡単に思いつくことができます。

たとえば、武器の感度、得られた経験、「火」、「氷」、「風」などの要素を使用して示すことができます...

ジェネレーターの公式ウェブサイトで利用可能な機能に関する詳細情報。
