Data Fest Conferenceからのレポート





3月の初めに、2日間のData Fest 2会議がMail.Ru Groupオフィスで開催され、実用的および理論的の両方のデータ分析の分野で関連するあらゆる種類の問題に専念しました。 さらに、会議の枠組み内でハッカソンが開催されました。その参加者は、Dota 2トーナメントの結果をできるだけ正確に予測しようとしました。また、さまざまな開発と研究プロジェクトが提示された研究者向けのピッチポスターセッションも行われました。 Data Fest 2のすべてのパフォーマンスのビデオ録画に注目してください。



Rostislav Yavorsky、高等経済学部、「経営判断にネットワークデータ分析を使用する方法」。 ロスチスラフのスピーチは、「組織の実際の構造の視覚化」と「専門家コミュニティの分析と視覚化」の2つの部分で構成されています。 レポートのビデオ録画 。







アレンAI研究所、Peter Clark「情報抽出から知識のある機械まで」。 ピーターのプレゼンテーションは、アリストプロジェクトと、情報の抽出を自動化する技術に捧げられました。 レポートのビデオ録画 。



スタニスラフセメノフ、データサイエンティスト、Kaggleトップ3、「Telstra Kaggleコンテストについて」 スタニスラフは、彼の報告書で、コンテストの参加者が直面している問題とその解決方法について話しました。 レポートのビデオ録画 。







高等経済学部デロイト分析研究所、セルゲイニコレンコ、「素朴なベイズからトピックモデリングまで」。 セルゲイのプレゼンテーションは、テーマ別モデリングの紹介に専念しました。 レポートのビデオ録画 。







セルゲイジェラシモフ（VMKモスクワ州立大学）、アレクサンダーメッシェリャコフ（宇宙研究所RAS）、「天体物理学におけるデータ分析の問題」。 このプレゼンテーションでは、天体物理学の緊急問題の解決における機械学習とビッグデータの役割について説明します。 レポートのビデオ録画 。







Roman Chebotaryov、CROC、「モデル品質評価指標の選択」。 ローマンは、古典的なデータサイエンスと実際のビジネスニーズの接点での緊急問題にレポートを捧げました。 レポートのビデオ録画 。







ミハイル・トロフィモフ。 ミハイルは短いスピーチで、聴衆にMLトレーニングのような概念を紹介しました。 レポートのビデオ録画 。



DeepHackの創設者であるMIPTのMikhail Burtsevは、「最近のディープラーニングのヒット」と述べています。 マイケルのレポートは、ディープラーニングの分野における興味深い研究の概要です。 レポートのビデオ録画 。







Ivan Lobov、データ中心アライアンス、「ディープラーニング研究の現在の傾向」 Ivanは、トレンドの1つ-ニューラルネットワークにおける注意と記憶の研究について話しました。 レポートのビデオ録画 。







Dmitry Kropotov、VMKモスクワ州立大学、Bayesian Methods Group、「大規模な関数の合計を最適化する方法」。 Dmitryは、ディープラーニングで使用される確率的最適化手法の短いレビューを公開しました。 レポートのビデオ録画 。







Dmitry Bugaychenko、クラスメート。 ドミトリーは2つのスピーチを行いました。最初のセッションでは、彼はOdnoklassnikiが開催する将来のハッカソン、SNA Hackathon 2016について詳しく話し、2番目のスピーチは実際のソーシャルネットワークでのデータ分析のトピックに捧げました。 最初のレポートのビデオ録画 。 2番目のレポートのビデオ録画 。







Vladimir Gulin、Mail.Ruグループ。 ウラジミールは彼のレポートで、「クリック」データを使用して検索エンジンのランキングを改善する方法について話しました。 レポートのビデオ録画 。







Alexander Schus、Mail.Ruグループ。 Alexanderのスピーチは、さまざまなネットワークサービスのアカウントをハッキングするという分野での緊急の問題を解決することに専念しました。 レポートのビデオ録画 。







アレクサンダー・セメノフ、高等経済学部、「通信におけるソーシャルネットワークの分析」。 アレクサンダーは、ネットワーク分析が今日のテレコムでどのように提示されるかについて話しました。 レポートのビデオ録画 。







パネルディスカッション「データサイエンティストの採用の特徴」。 ビデオ録画 。



パネルディスカッション「生産研究の実施」。 ビデオ録画 。
