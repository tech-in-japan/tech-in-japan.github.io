Opera Mobileレポート：2009年6月









Opera Miniユーザーは月に100億以上のWebページを閲覧します



2009年7月27日、ノルウェー、オスロ-Opera Miniユーザーが月間で閲覧する月間Webページが初めて100億を超えました。これらのデータは、2009年6月のOpera Miniブラウザの使用に関するデータに基づくモバイルインターネットのステータスとトレンドに関するレポートに含まれています。



完全なレポートは、 www.opera.com / smw （英語）にあります。



マスタデータを報告する



-2009年6月、Opera Miniブラウザーは約2650万のユニークユーザーを使用しました。これは、5月より4.2％多く、2008年6月より143％多くなっています。これらの数字は、Opera Miniを選択したユーザー数を反映しています携帯電話にインストールします。



-6月に2650万人のユーザーが104億以上のページを閲覧しました。 したがって、彼らは毎日3億4600万以上のWebページを開きました。 5月と比較して、この指標は8.4％増加し、昨年6月と比較して224％増加しました。



-先月、Opera Miniユーザーは1億6,840万メガバイト以上のトラフィックを生成しました-2009年5月より5.6％、2008年6月より236％。OperaMiniブラウザーはデータを最大90％圧縮できます。 情報が圧縮されていなかった場合、ユーザーは約1.6ペタバイトのデータを携帯電話にダウンロードする必要があります。



-インドはOpera Miniのユニークユーザー数の点で中国を上回っています。 6月のこの指標の上位10か国は、ロシア、インドネシア、インド、中国、ウクライナ、南アフリカ、アメリカ、イギリス、ポーランド、ナイジェリアのように（ユーザー数の降順で）表示されました。



-ノキアの携帯電話は世界中でOpera Miniを使用する頻度が高く、この指標ではSony Ericssonデバイスが2位になっています。 米国では、ユーザーはほとんどの場合、BlackBerryモバイルデバイスからブラウザーを開き、サムスンはこの地域で2番目です。



-Opera Miniのロシアのユーザー数は、2008年6月に比べて68.5％増加し、閲覧したページは140.3％増加しました。 各ユーザーの平均月額は401ページです。



-ロシアからのダウンロード数では、Odnoklassniki.ru WebサイトがYandexに次いで2位になりました。 さらに、6月にRambler.ruが最も人気のあるトップ10のリソースに入り、My.Opera.comがトップ10を去りました。 Vkontakte.ruは、訪問数のリーダーであり続けています。



-6月のOpera Miniブラウザのウクライナユーザーの数は71.5％増加し（2008年の同月と比較）、閲覧したWebページは769.2％増加しました。 平均して、ウクライナのユーザーは1か月あたり589ページを見ました。これは、上位10か国で最も高い数字です。



-5月にGoogleがウクライナから最も訪問したサイトのトップ10に戻り、6月に再びこの評価を残しました。 初めて、depositfiles.comがこのリストに掲載され、Vkontakte.ru、Yandex、Mail.ru（降順）がユニークビジター数の上位3位になりました。



-Nokia 6300は、ロシアおよびウクライナのOpera Miniユーザーの間で最も人気のある携帯電話であり、ブラウザの使用に関しては、このデバイスは全世界のリーダーです。 Opera Miniの2650万人のユーザー全員がNokia 6300の携帯電話を持ち、それらを縦に並べると、「タワー」が国際宇宙ステーションに到達し、地球に4回戻ってきました。 その高さは、赤道の長さのほぼ半分（0.44）になります。 2650万台のNokia 6300電話は、ロケットランチャーとボーイング747航空機を搭載したスペースシャトル全体と同じくらいの重量があります。



OperaのCEOであるJon von Tetzchner氏は次のように述べています。 「プラットフォームとして、Web標準は、あらゆるタイプのデバイスでシームレスに表示されるサイトを作成するためのより効率的な方法を提供します。」 今日のユニファイドウェブの戦略はこれまで以上に重要であり、より多くの人々が携帯電話を使用してネットワークにアクセスするほど、その影響力は強くなります。



Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社に関する詳細： www.opera.com
