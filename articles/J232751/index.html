Inspector Buddy-Nokiaスマートフォンの心臓部を持つロボット

先週、インディーズの開発者Robert Oshlerが作成したWindows PhoneスマートフォンベースのロボットプロジェクトであるInspector Buddyは、 2014 Accela Construct App Challenge開発者グランプリを受賞しました。







Inspector Buddyは手頃な価格の（安価ではないが）ロボットで、住宅や非住宅の施設を検査し、事故や労働災害から保護する過程で不動産検査員を支援するように設計されています。 この投稿では、プロジェクトの詳細を共有します。その中心はNokia Lumia 521（Lumia 520スマートフォンのキャリアバージョン）でした。



Inspector Buddyの作成の背後にある主なアイデアは、手が届きにくい場所や潜在的に危険な場所を調査するときに、人（この場合は不動産検査官）を予算の半自動ソリューションに置き換える試みです。







プロジェクトの技術的基盤は、WowWeeのRoboMeロボットです。これは、約78ドルでAmazonや他のオンラインストアで誰でも注文でき、Windows Phoneの安価なスマートフォン（この場合、$ 69でNokia Lumia 521ですが、 WPの別のスマートフォン）およびWindowsのタブレット/ラップトップ/ PCをベースとして使用できます。 ただし、通常のロボットを検査官バディに変えるソフトウェアフレームワークでない場合、これはすべて単純なおもちゃになります。 Accela ConstructとCivicID APIが統合されたWindowsとWindows Phone用の2つのアプリケーションで構成されています。 このソフトウェアを使用すると、ライブビデオ放送に加えて、Inspector Buddyは写真を撮影し、ビデオを記録することができます。プロジェクトを専門ソフトウェアに統合したおかげで、不動産調査レポートに自動的に添付されます。







Inspector Buddyには少しのバックグラウンドがあります。 ロボットは、ロバートオシュラーの過去のプロジェクト-ナノボットの継続であり、同じ技術的基盤に基づいています。



音声認識とスピーキングスキルの原理を使用して、Nanobotは人と簡単なゲームをプレイし、家を探索し、保護し、オブジェクト（本の表紙や音楽レコードなど）を認識し、ある人から別の人への物理的な送信者になることさえできます。 これはすべて、Windows Phone向けの多数のNokiaスマートフォンとアプリケーションの形で実装された「頭脳」のおかげです。 アプリケーション自体は、ロバート自身の経験に加えて、Cortanaおよびその他のMicrosoft APIと密接に統合されています。







ロバートのプロジェクトの継続性は、Inspector BuddyがNanobotと同じコードベースを使用しているという事実によっても証明されています。



「Accela開発者コンテスト（国営企業向けのデジタルおよびWebソリューションの分野で働く国際企業、著者のメモ）について知ったとき、不動産検査で使用できる研究ロボットには特定のニッチがあることに気付きました。そして、私はこれらの目的のために以前のNanabotプロジェクトを適応させることができます-ロバートは言います。 「たとえば、このようなロボットは、ネズミ、ヘビ、有毒なクモ、屋根裏部屋、地下室、エアダクトなどの危険な居住者による攻撃などの民家を検査する際の典型的な危険から検査官を保護します。」



「さらに、ロボットのサイズと重量が小さいため、検査員はアクセスできず老朽化したエリアをチェックして、怪我や事故さえも回避できます」と彼は付け加えました。







ロボットは、VSN Mobilの特別なV.BTTN Bluetoothコントローラーを使用して、ジェスチャーで制御できます。 これは、コントロールやその他のコマンドをアクティブにするワンボタンインターフェイスと、ユーザーの手の動きを判断するための組み込みの加速度計を備えた、非常にシンプルなブレスレットです。







Inspector Buddyは単一のテストサンプルの形式で作成されますが、Robertはこのプロジェクトを単純なDIYソリューションに開発し続け、大規模な州組織だけでなく小規模な民間企業もアクセスできるようにします。 プロジェクトの開発については、 Robertの個人Webサイトで読むことができます。



ノキアLumiaの独自のロボット



Windows Phoneベースのスマートロボットを作成したい人のために、ロバートはクラウドファンディングプロジェクトを立ち上げました。これにより、誰でもNanobotのアナログを作成できるソフトウェアにアクセスできます（ロボットは別売りです）。



Nanabot Developer プロジェクトのWebサイトで登録して、 Windows Phone向けのRoboMeオープンソースツールのパーソナライズされたテクニカルサポートを受け取ることができます。また、Robertが開発した他の便利なツールにアクセスできます。
