1） 2009年9月4日、深刻な競合他社であるApple iPod Touchが正式に発表されました。これは、新しいSamsung M1ポータブルメディアプレーヤーです。

新しいプレーヤーの最も顕著な利点は、大型のAMOLEDディスプレイです。 AMOLEDディスプレイは、TFTディスプレイよりもはるかに低い消費電力と応答時間を持ちながら、高コントラストと明るい色が特徴です。





ディスプレイの対角線は3.3インチで、アスペクト比は16：9です。 1.6 Wステレオスピーカーは、ケースの側面に組み込まれています。 ムービーを表示するために、ユーザーは最初にソースファイルを変換する必要はありません。 ただし、M1はSony PSPおよびApple iPodで表示するために変換されたファイルもサポートし、フラッシュプレーヤーを備えています。 高品質のビデオは、強力なnVidia Tegraプロセッサによって提供され、HD形式720p（WMV / H.264）でビデオを表示できます。



Samsung M1には「ナチュラルサウンドエンジン」が含まれています-DNSe 3.0、最大32 GBのmicroSDカードスロット、FMラジオ、ビデオ録画、ファイル転送、ワイヤレスヘッドフォンの使用、友人とのゲーム、音楽のストリーミングが可能なBluetooth機能ホームまたはカーステレオに。

プレーヤーは、アルバムコレクションのダウンロードをサポートしています。これは、音楽コレクションを合理化して視覚的にするのに役立ちます。



Samsung M1の機能：

•ディスプレイ：3.3インチ、タッチ、AMOLED、WQVGA解像度

•プロセッサ：nVidia Tegra

• TouchWizインターフェイス

•内蔵メモリ：8/16/32 GB

• DMBレシーバー

•加速度計

• Bluetoothワイヤレスインターフェイス

•オーディオ形式：MP3、WMA、OGG、ASF、M4A、AAC、Flac

•ビデオ形式：WMV、ASF、H.264、MPEG4、DivX、XviD、SWF

•プレーヤーの寸法：98.9 x 54 x 9.9 mm

•重量：91グラム



これらすべての機能に加えて、M1にはFMラジオ、電子書籍リーダー、組み込みのゲームとウィジェット、ボイスレコーダー、フォトブラウザー、オーガナイザーがあります。 プレーヤーはテレビに接続できます。

おそらく、Samsung M1プレーヤーは、メディアプレーヤーに特別に適合したLinux OSバージョンの1つを実行していますが、それについては知りません。 残念ながら、サムスンはこれについて沈黙しています。



サムスンM1プレーヤーは、今年10月にロシア市場に参入します。

最も安いSamsung M1 8GBは236ユーロ、16GBでは290ユーロ、32GB以上では324ユーロの費用がかかります。



2） 2009年9月15日、おそらくApple iPod Touchの主な競合他社の販売が開始されます-これはもちろんMicrosoft Zune HDメディアプレーヤーです。

彼については多くのことが書かれています-たとえば、 「公式に発表されたZune HD」および「Zune HDを購入-Microsoftの負債を支払う」を参照してください 。

Microsoft Zune HDの仕様は、 Samsung M1メディアプレーヤーの仕様と非常に似ています。 主な違いは次のとおりです。ZuneHD Wi-Fi接続の可用性と、プレーヤーを制御する別のオペレーティングシステム。

マイクロソフトは直接話さないようにしていますが、開発者のフィードバックによると、動作と外観の面では、Microsoft Zune HDプレーヤーはプレーヤーのニーズに特に適合した制御下で実行されていると言えます。WindowsEmbedded CE 6.5はコアですWindows Phoneスマートフォン用のOSが作成されたことに基づいて、2009年10月6日からWindows Mobile 6.5でのみ販売が開始されます。



Microsoft Zune HDの主な機能：

•マルチタッチ対応のOLED画面（3.3インチ、480 x 272）。

•プロセッサ：nVidia Tegra。

• FMおよびHDラジオ。

• HDビデオ出力。HDTVデバイスでプレーヤーのビデオを視聴できます。

• Wi-Fi;

• Webブラウザー。

• Xbox LIVEおよびXbox LIVE Marketplace（Apple AppStoreなど）との統合。



Microsoft Zune HDの主な利点は、強力なnVidia Tegraプロセッサとソフトウェアプラットフォームです。 マイクロソフトは、できるだけ多くのゲーム開発者をZune HDプレーヤーに引き付けることを約束します。

そして実際には、Apple iPod Touchだけでなく、 Microsoft Xbox 360ゲームコンソールに加えて、 Sony PSPポータブルゲームコンソールのライバルにもなろうとしています。

マイクロソフトは、ゲーム開発者に、Microsoft Windows、Microsoft Xbox 360、およびMicrosoft Zune HDプレーヤー用のゲームを作成できるMicrosoft XNAツールのユニバーサルセットの使用を提供しています！
