修正、修正、修正、およびVivaldiに関するいくつかの単語







みなさんこんにちは！



数日前にVivaldiブラウザの3番目のベータ版をリリースしましたが、今日は多くの修正と改良を加えた次のテストビルドを示す準備ができています。 まず、次の重要な変更点に注意してください。



  • ブックマークバー項目でCtrl、Shift、および中マウスボタンを使用する機能が追加されました。
  • Macバージョンでは、ピンチツーズーム機能が追加されました。
  • Linuxバージョンでは、マウスの中ボタンでクリックしたときのクリップボードからのコンテンツの貼り付けを修正しました。
  • ブラウザインターフェイスの明るいテーマと暗いテーマを切り替えるショートカットを割り当てる機能が追加されました。
  • ReturnおよびTransition関数のいくつかのエラーを修正。
  • ブックマークマネージャーの作業は、それらの多くによって加速されます。
  • 非ラテン文字を含むURLの表示を修正。


などなど。 変更点の完全なリストは、いつものように記事の最後にあります。 そして、Vivaldi Technologiesの創設者でありディレクターであるJon von Technerからの一言。



Vivaldiが他のブラウザと異なる理由



今日、私は最近読んだ記事に照らして、会社としてのヴィヴァルディについてお話したいと思います。 ニュースをフォローしていない人のために、 Firefoxはタブ （以前は「Panorama」と「Tab Candy」と呼ばれていました）のグループ化を拒否していることを指摘したいと思います 。 私はFirefoxのユーザーではありませんが、同様のOperaブラウザー機能を追求して、タブのグループがFirefoxに登場したことを非常によく覚えています（ 愛好家の歴史への小さな逸脱 ）。 ほんの数年前、これら2つのブラウザーは強力な新機能でユーザーを引き付けようとして、真っ向から向かいました。 今日、両方のブラウザは機能を放棄し、平均的な控えめなユーザーに焦点を当てています。



ヴィヴァルディには、別の目標があります。 最も気難しいユーザーを満足させる最大数の機能を備えたブラウザーを作成しようとしています。 そのため、お客様のリクエストを理解するために、ブログ、フォーラム、ソーシャルネットワークで積極的に連絡を取ります。 そして、私たちはブラウザ市場でVivaldiをユニークなものに近づけようとしていると思います。 vivalditips.com （ユーザーが作成したVivaldiのWebサイト）をざっと見ても、ブラウザーにどれだけ追加されているかを確認するには十分です。 しかし、私たちは最初の安定版にさえ到達していません！ 注意する必要があります-これらのすべての機能がすべてのVivaldiユーザーから要求されるとは限りません。 人々はこれらの機能の限られたセットのみを使用することを知っていますが、各セットは一意です-結局のところ、実際にはすべてが異なります。



そのため、今日Firefoxを使用していて、このブラウザの最新の変更に失望している友人や知人がいる場合は、Vivaldiを見せてコミュニティに招待してください。 使い慣れたブラウザから削除された機能をVivaldiで見つける可能性があります。また、好きな新しいものが表示される可能性があります。 Vivaldiユーザーコミュニティに積極的に参加することで、彼らはブラウザの開発に影響を与えることができます（まだ友人のためにブラウザを作っているためです）。 「平均的なユーザー」の基準を順守していません。各ユーザーに個別の機能を提供するよう努めています。



それだけです 以下のリンクを使用して、新しいアセンブリをダウンロードできます。

  • Windows 64ビット （実験的）
  • Mac 10.7+
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
