Microsoftは、Windows、OS X、Linux向けのクロスプラットフォームエディターを発表しました

マイクロソフトは本日、Visual Studio Codeの発売を発表しました。 これは、最新の.NETスタックを操作するためのクロスプラットフォームエディターです。 Visual Studio Codeは、 OS X 、 Linux、およびWindowsで実行されます 。 アプリケーションはすでにダウンロード可能です。



Visual Studio Codeは、自動補完、スニペット、バージョン管理システムのサポートなど、最新のエディターに存在するすべての機能を誇っています。 エディターは、JavaScript、TypeScript、Node.js、C＃、ASP.NET 5などのいくつかの言語とテクノロジーをサポートしています。







マイクロソフトは、歴史上初めて、クロスプラットフォームエディターを導入しました。 Visual Studioのフルバージョンは引き続きWindowsでのみ利用可能ですが、この発表は、同社の以前の行動と同様に、Microsoftが他のプラットフォームのサポートを目的としていることを示しています。
