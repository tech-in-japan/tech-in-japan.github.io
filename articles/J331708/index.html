Stepikコンテストの結果と新しいアダプティブオンラインコース

11月から5月にかけて、 StepikコンテストのコンテストがStepikプラットフォームで開催され、参加者は適応学習システムのレッスンを作成し、Stepikチームが現在開発および開発を行っています。 コンテストのコースについては以前に書きましたが、この記事では結果を共有したい（新しい無料コースです！）勝者について話します。







コンテストに参加するには、ノミネート（応用コンピューターサイエンス、理論的コンピューターサイエンス、データサイエンス、Python、Java、JavaScript）の1つでトピックを選択し、20を超えるタスクといくつかの理論的なレッスンを英語で作成する必要がありました。 作成した教材を適応学習で使用することを計画しました。教育コンテンツとタスクは、生徒の知識と準備のレベルに基づいて、生徒ごとに個別に選択されるため、作成するレッスンには特別な要件がありました。



  • 1つのレッスンでは、1つの複雑なタスク、1つのレッスンで少なくとも1つのタスクを収集する必要がありました。
  • すべてのレッスンは互いに独立していなければならず、それぞれのトレーニング段階の生徒にアドバイスを提供することができました。
  • プラットフォームで作成されたすべてのレッスンを相互接続し、ユーザーがそれらをナビゲートできるようにするために、すべての資料に特別なトピック（タグ）を付けてください。


当初、2017年2月11日にコンペを終了したかったのですが、休日の期間、そして学生セッションが潜在的な参加者の時間を大幅に制限することを考慮しなかったため、締め切りを 3月31日まで延長し、失敗しませんでした：レッスンのほとんどは先月に作成されました。



レッスンとコースを作成しました。 選択プロセス。



コンテスト中、参加者はプラットフォーム上で61のコースを作成し、最も人気のあるトピックはApplied Computer Science（16コース）、PythonおよびJava（各11コース）でしたが、41コースが登録段階に達し、そのうち29コースが要件を満たしました。 コンテストは英語で行われ、参加者の地理は非常に多様でした-ロシア、ウクライナ、イギリス、バングラデシュ、エジプト、カナダ、ポーランド。



3月31日に登録を終了し、10日後にプラットフォームの学生の間で投票するためのコースを提示しました。 これは、正直な投票と、作成されたレッスンが適応学習に適しているかどうかを確認するという2つの目標を一度に達成するために行われました。 すべてのレッスンを組み合わせて大規模な適応コースを作成し、そのうちの4つがありました。



  1. コンピューターサイエンス （2つのノミネートを組み合わせることを決定）;
  2. Java 、そして今ではさらに多くのタスクを備えたAdaptive Javaです。
  3. JavaScript
  4. データサイエンス 。


最後に、サンクトペテルブルク州立大学のヴィタリー・ポルシュコフの学生のレッスンだけがすべての基準を満たしていたため、彼らはPythonコースの作成を開始しませんでした。



誰もが作成したコースをアダプティブモードで取得し、好き嫌いを入れることができます。 マークアップを徹底的にチェックした後（はい、そうでした！）、不正な評価のその他の方法で、結果をまとめ、次の勝者を発表しました。



1位：Artyom Burylov（ロシア、ペルミ）、 Javaでの関数型プログラミング

2位：Andrey Kochkin（ロシア、モスクワ）、 暗号化

3位：マーク・ザスラフスキーと彼のチーム（ロシア、サンクトペテルブルク）、 Git

3位：Dmitry Levanov（ロシア、キーロフ）、 コレクションフレームワーク

3位：Dmitry Knyazev（ロシア、モスクワ）、 Javaのコレクション



その後、関係する専門家によってコースが評価され、学習したレッスンが提案された適応モードにどの程度適合するかが明らかになりました。

その結果、次の著者が追加候補に含まれました。



Python：Vitaly Polshkov（ロシア、サンクトペテルブルク）、 Pythonでの関数型プログラミング

コンピューターサイエンス：Dmitry Mitin（ウクライナ、キエフ）、 Scala

コンピューターサイエンス：アントンブラギン（ロシア、サンクトペテルブルク）、 Docker

データサイエンスギリアトティ（英国、ロンドン）、 ロジスティック回帰

データサイエンスエフゲニアボロンツォワ（ロシア、ウラジオストク）、 ジュリア入門



すべての結果は、 adaptive.stepik.org / resultsでも公開されており、受賞者には賞品（または獲得プロセス）およびStepik社の証明書が授与されます。







Stepikコンテストのコンペティションでは、アダプティブモードで受講できる質の高い多数のコースが提供されただけでなく、Stepikの教師陣に素晴らしいITプロフェッショナルが集まりました。 それらのいくつかはすでに新しいコースを準備しています、そして誰かが慈善のために賞品の一部さえリストします！
