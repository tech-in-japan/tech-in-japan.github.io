Yandex Live Requests

みなさんこんにちは！

しばらく前に、興味深い仕掛けに遭遇しました： http ://export.yandex.ru/last/last20x.xml-Yandex検索エンジンへの最新のクエリ。



彼は座って、おそらく1時間、それを更新しました。人々がYandexに尋ねるのを見るのは非常に興味深い問題です。

見つかりませんでした
「Requests to Yandex-Live」という投稿があったことを覚えています。彼らはそこにリクエストのあるマップを示しましたが、何らかの理由でリンクが機能しません。



google.ruへのリクエストがタイルで表示されているページを見たことがあることを思い出し、Yandexについてのみ、すべての結果を伴う独自のものを作成することにしました。







最初は自分用に作成しましたが、多くの人がこのことを気に入ったので、一般的なレビュー用に出すことにしました。 落ちないことを願っています。



仕組み：ページを開く-最新のリクエストが表示され、ajaxによって新しいリクエストのタイルがランダムに更新されます。 なぜなら さまざまな言語のリクエストが発生します 。たとえば、トルコ語では、Yandexトランスレーターがhttps://tech.yandex.ru/translate/を台無しにしました。 彼はロシア語で書かれていないすべてを翻訳しようとしています（以下の小さな活字で）。



美しさのために、タイルの色を赤、青、または緑、デフォルトの多色に変更できます。



もちろん、キャッシュはわずか2秒です。 そして、Yandexが頻繁なリクエストで403を発行していることに気付きました。



ここでこの作成を確認できます： http : //nikitchenko.ru/zaprosy-k-yandex-online/



よろしくお願いします！



明けましておめでとうございます！ :)
