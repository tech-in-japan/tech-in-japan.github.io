ジュニアシニア：Snom D715 IP Phoneレビュー





友人たち、私たちはドイツの会社Snomの製品についてあなたと知り合っています。この夏については、この夏についてもっと話しました 。 そして今日は、低価格モデルであるSnom D715に属するIP電話についてお話します。

服で



古いモデルと比較して、D715の外観は控えめです。 ディスプレイはそれほど大きくなく、4行のテキストまたはメニューに収まります。 ディスプレイの下には4つの多機能ボタンがあり、ATMで行われているように、その目的はディスプレイの内容によって異なります。



















以下はナビゲーション用の4方向ジョイスティックボタンで、その両側には確認ボタンとキャンセルボタンがあります。 そしてその隣には、音声メッセージを聞くためのボタンがあります（メッセージがある場合に点灯します）。







電話の右上隅には、着信時に赤く点滅し、不在着信がある場合にのみ点灯するライトインジケータがあります。







5つのプログラム可能なボタンがあり、追加の画面はありません-手動で紙に署名するか、プリンターで印刷する必要があります。







すべてのSnom電話には便利な携帯電話があり、大きな手にも小さな手にもよく合います。







従来、D7xxxラインの場合、D715はスタンドを2つの異なる位置に再配置して、電話機の角度を変更できます。











D715は白でも入手可能です。







心



他のレビューで述べたように、D3xxおよびD7xxのすべての電話機の基本的な機能セットは、モデルの高コストに関係なくほぼ同じです。 電話機の機能は、主に定量的に異なり、定性的ではなく、たとえば、プログラム可能なボタンの数、画面解像度、ポートの数などによって異なります。



そのため、機能セットの点では、D715は以前の「兄弟」にほぼ劣っています.IPv4 / IPv6プロトコルをサポートし、組み込みのXMLミニブラウザー、2つのRJ45ポートを備えたギガビットスイッチ、ヘッドセットを接続するためのUSBポートを備えています。 電話機はPoEプロトコルをサポートします。つまり、電源なしで動作し、ネットワークケーブルを介してエネルギーを受け取ります。















ユーザーとその対談者の両方にとって優れた音質は、マイクとスピーカーの考え抜かれた設計だけでなく、一連のコーデックの助けによっても保証されます。 電話機はブロードバンドオーディオ処理をサポートし、組み込みのノイズリダクションがあります。



電話帳には、最大1000件の連絡先が含まれています。 D715は、着信番号が通話中の場合、自動的にコールバックできます。 組み込みのブラックリスト、匿名の通話ブロック、サイレント（DND）モード、通話保留、通話転送があります。 同時に、電話機は最大4つのコールを処理でき、3者間電話会議が可能です。



確かに、いくつかの粗さがありました。ローカライズされたロシア語のファームウェアには時々英語のメニュー項目が出てきます。



D715には、企業の通信ネットワークおよび豊富な設定で実装するための十分な機会があります。 最大4つのSIP識別子がサポートされています。 Bluetoothモジュールは内蔵されていないため、ヘッドセットはRJ11またはUSB経由でのみ接続できます。または、外部BluetoothモジュールをUSBに挿入できます。 HTTP / HTTPSサーバーが組み込まれているため、電話はパスワードで保護されたWebインターフェイスを介してリモートで完全に構成でき、加入者機器TR-069 / TR-111を管理するプロトコルで管理できます。















TLSおよびVPNでサポートされています。 D715はファームウェアの更新と設定を自動的にダウンロードでき、USBポートを介してローカルコンピューターネットワークに接続できます。



要約すると、Snom D715は、多くの電話を受ける必要のない普通の従業員向けの大容量電話であると言えます。 同時に、単一の企業ネットワーク内の他のSnomモデルと同等に機能し、その機能と設定のセットは豊富であり、あらゆる通信スキームを実装できます。
