Web 2.0設計ガイド公開





Web Design From Scratchは 、 包括的なWeb 2.0設計ガイドを公開しています 。 このガイドは、新世代サイトで最もよく使用される15の設計手法のリストであり、具体的な例とスクリーンショットが含まれています。



シンプルさ

ページには、できるだけ少ない要素を含める必要があります。 2つの可能な解決策のうち、よりシンプルなものが常に優れています。



ページの「クリーニング」の結果、訪問者の目はコンテンツに集中します。 ナビゲーションが簡素化されました。 重要なものにユーザーの注意を引くのは簡単です。



Web 2.0サイトの設計では、「描画情報」または「情報ピクセル」（データインク）という用語を使用できます。 一般に、タフト自身は、情報デザインの有効性は、画面上の「情報」ピクセルと総ピクセル数の比率で表されると言っていました。



中央マーキング

以前のデザイナーのファッションと比較して、今ではブラウザの中央にはるかに多くのサイトがあり、フルスクリーンに拡大されたり、左にシフトされたりしていません。 Web 2.0の設計はよりシンプルで経済的であるため、このようなサイトには常に多くのきれいで空白があります。



少ないスピーカー

数年前、3列設計が一般的でしたが、多くの場合、4列も見つかりました。 今日、2つの列は誰も驚かず、3つの列は一般的に受け入れられている最大値です。



別の上部セクション

ページの上部には、他のすべてのコンテンツから分離された水平セクションがあります。 もちろん、このアイデアは新しいものではなく、長い間使用されてきましたが、今では本当に認識されるようになりました。



カラーブロック

上部のセクションに加えて、他のページ要素も明確に強調表示できます。 これは、ナビゲーション領域、メインコンテンツ領域、またはその他の領域にすることができます。

明るい色の問題は、ページの他の要素から注意をそらしすぎることです。 そのため、場合によっては、通常の白い色から逸脱しないほうが良いでしょう。



簡単なナビゲーション

サイトの各ページに表示されるナビゲーションメニューは、大きく、明確でシンプルでなければなりません。 ハイパーリンクについては、ページ上の他のすべてのコンテンツと明らかに異なる必要があります。



大きいサイズ

Web 2.0サイトでは、フォントサイズは通常、前世代のサイトよりも大きくなります。 このルールは、シンプルさと機能設計の原則に基づいています。



大きなテキストの挿入

非常に大きな見出し、ロゴ、ラベルが多くのサイトで見られます。 これらのラベルは通常、グラフィック要素として実装されます。



クリアロゴ

はっきりとした、力強い、そして強いブランドは、目立つ、キャッチーなロゴで作られています。
