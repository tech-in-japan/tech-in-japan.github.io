Acer Iconia TAB W500（パート2、ソフトウェア）を確認する





最初の部分（鉄の説明）はこちらです。

このパートでは、ソフトウェアを詳細に説明し、タブレット全体で作業するようにします。



タブレットはすでに使用されているため、プリインストールされたプログラムのいくつかは飲酒されています（それらについて簡単に説明します）。








BIOS



BIOSに入るにはキーボードを接続する必要があります。 ログイン-[削除]ボタン。

ここで説明する特別なものはありません。

4つのタブ：

1）情報（一般的な説明）。

2）主なもの（日付、メモリサイズ、ブートモード、SATAモード）。 次に、ネットワークブートをオフにしました。

3）ダウンロードするデバイスのリスト。

4）終了します。




Windows 7



スタート。

最初の開始（チェック中に店舗で行われた）で、システムは長い間考え、それ自体を調整します。 契約に同意し、パスワードでアカウントを作成するように求めます。 再起動後、キーボードを備えたこのような画面が表示されます。 両手でそれを書くことはできませんが、指で6〜7文字のパスワードを入力するのは簡単です。



最初のログイン後、Windowsコンピューターの標準である地獄が始まり、メーカーのプログラムがプリインストールされています。 Windows、通知、何か... ...（しかし、このトピックの次の段落でこれに戻ります。

このタブレットでのWindows 7のコールドスタートには、約45秒かかります。

Windows 7〜800MBのRAMを消費します。


アップデート。

最初にしたことは、ネットワークに接続してアップグレードすることでした。

Service Pack 1はタブレットに搭載されており、2回の呼び出しで更新をダウンロードする必要がありました。 最初の-32個の更新（84 MB）、2番目の更新は覚えていませんが、少ない（〜15個）。 更新により、ウイルス対策MS Security Essentialsもインストールされました。 その後、自動更新をオフにしました。



ハードドライブ

空き領域は最初は14 GBです。 ごみ箱（最大300 MB）を減らし、スワップファイルを無効にし、休止状態を無効にし、不要なプログラムを削除した後（ただし、Office 2010といくつかの小さなプログラムをインストールした）、空き容量は15.7 GBに増加しました。 現在（2011年7月4日）-14.1 GB。 データがドライブCに保存されていないことに慣れていますが、ここではマイドキュメントに慣れる必要があります。

システムログは定期的に（おそらくrand（）関数を使用して）「ドライバーはコントローラーエラー\ Device \ Harddisk1 \ DR1を検出しました」（コード11）を書き込みませんが、chkdiskは何も表示しません。 このテーマのフォーラムでは、膨大な数のバージョンがあります。これがどれほど重要かはわかりませんでした。 ラップトップ（3年間のサービス）では、これもありますが、すべてが問題ありません。


スクリーン。

デフォルトの画面解像度は1280x800です。 テキストのスケールは125％です。 150％を試しましたが、一部のウィンドウで問題が発生しました（たとえば、IE9インターネットオプションウィンドウの高さが大きくなりすぎて、[OK]をクリックすることができませんでした）。 原則として、125％です。 小さくも大きくもない。

レビューの最後の部分で書いたように、タブレットを回転させると、画面が最初に「オフ」になり、次に回転します。 ターンは約3秒続きます。 私はほとんど使用しません（本を読む場合を除く）。

マルチタッチにより、4本の指を同時に操作できます。 仮想ピアノを置いて曲（コード+ソロ）を演奏できると思います。


仮想マウス\ペン。

指さし-マウスの左ボタン。

指でTyyyyyk-マウスの右ボタン（この場合、指の周りに円が形成されます）。

すべてがシンプルです。

おなじみの矢印へのポインタ、および中央に円のある小さな十字。 3秒後に滑らかに消えます。 また、指を押す場所から触れると、まるで指の下に水があるかのように出てきます。

「ペンとタッチデバイス」の設定を掘り下げて、2ボタンマウスの精神を呼び起こすことができます。 それと一緒に右ボタンを使用する方が簡単です（スピリットボタンの右にあります）。

また、ジェスチャーを賞賛したいと思います。 フォトギャラリーで写真を回転したりめくったり、ブラウザでスクロールしたりするのはそれほど簡単だとは思いませんでした。 独自のジェスチャーを作成することはできませんが。


仮想キーボード

3種もあります。

最初のものは小さいですが、スケーラブルで再配置可能です。 マウスが入力フィールドを指すとすぐにアイコンが表示されます。 最も普通のキーボード。

2番目は大きく、スケーラブルではなく、静的です。 画面のちょうど1/2を占めます。 残念ながら、このキーボードは最初のように印刷領域に収まりませんが、プログラムウィンドウを上に移動します（つまり、プログラムは画面の1/2の高さになります）。 これが起こることから、私は見当がつかず、これはマイナスです（私とM $にとって）。 その上に印刷するのは便利ですが、あなたがそれを見たときだけ:)

正直なところ、私はこのキーボードで通常の.txtファイルに書いた最初のレビューの1/3です。 キーボードは好みの問題です。

3番目は手書きです。 私の指は快適ではなく、速くもありません。 何糸ペンは高速であり、高速になります。 テキストは驚くほどよく認識されます。 補正用のジェスチャーはシンプルで便利です。 線の太さは設定で調整できます。













探検家

エクスプローラー（および他のプログラム）では、2本の指を\離すだけでビューを変更できます。

複数のオブジェクトを選択するには、チェックボックス（チェックマーク）をオンにしました。フォルダオプション->表示->チェックボックスを使用してアイテムを選択します。 スクロールバーから、約1週間後に離乳しました。 今、私はエクスプローラーウィンドウ自体をめくっています。 その前に、スクロールする代わりに、私は常にフォルダ\ファイルを開きました。

また、リムーバブルメディアの起動からすべてを削除しました。


栄養

最初、バッテリーは3回トレーニング（完全充電、完全放電）されました。一般的に、バッテリーの性能は電源プランと光センサーに依存します。 後者は、（私の意見では）コントロールパネルで削減できます。 バックライトなしで、経済的な計画で、静かな操作（テキストの印刷、写真の表示、ささいなことなど）では、バッテリーは7〜8時間保持されます。 標準プラン（バックライト50％）〜4時間。

たとえば、今日（2011年7月4日）、タブレットは11:30から18:00に動作しました。 ファイルをダウンロードし、このトピックを書き、Visioを入れて、少し（最初の5分間）HDビデオを視聴しました（これよりずっと低い）。 「経済的な食品計画に星を数える」などのテストは実施しませんでした。
