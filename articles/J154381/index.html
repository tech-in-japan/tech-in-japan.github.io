ATNSOFT Key Managerを使用してWindowsでキーとネイティブホットキーをオーバーライドする

ATNSOFT Key Managerプログラムについてお話したいと思います。 これは、コンピューターでの作業が少し快適になるため、作業が少し速くなり、一般的に少し幸せになるプログラムです。 プログラムは、一般的に言えば、バージョン番号1.5にもかかわらず、湿っていて不具合がありますが、私の意見では、多くの便利な機能を備えています。 プログラムは有料です。 有料プログラムについて書くのは流行りませんが、かなり安いです。 時間とともに提供される機能は、単純にプログラムのコストを、節約された作業時間のコストから支払います。 まあ、代替ソリューションを検索する時間のコスト。

プログラムの開発とは関係ありません。 ライセンスはすでに取得されているため、実質的に興味はありません。 以下では、自分が使用するもののみを説明します。 プログラムには他の機能もあります。



ボタンのオーバーライド


ノートパソコンのキーボードにデジタルブロックがあります。 これは素晴らしいことですが、ロシア語のキーボードレイアウトでは、「。」 コンマが印刷されます。 はい、ロシア語の数学の観点からはこれは正しいですが、私はプログラマーとして働いているため、コードにIPアドレスまたは小数を入力する必要があります（セミコロンではなく）。 Windows XPで、この問題を解決する方法を見つけました。 Windows 7では解決策が見つかりませんでしたが、実際には解決策を探しませんでした。 Googleを数回使用し、ヒントを試してみました-うまくいきませんでした。 私はこのベンチャーを放棄しました。

キーマネージャーは、「。」によるように構成できます。 テンキーは常にドットを印刷します。 一方、Numロックをオフにしても、ドットが印刷され始めました。 「しかし、Numロックをオフにしてデジタルキーボードを使用することはありませんが、逆にNumロックをオフにするとイライラします。」 そして、数字キーパッドのキーに0〜9の数字が常に印刷されるように設定しました。 PuTTYコンソールにログインして、数値がテンキーに印刷されるようになったとき、特に嬉しかったです！

私のラップトップのキーボードでは、InsertキーはFn + Deleteとして作成されます。 コンソールには常にCtrl + InsertとShift + Insertが必要なため、あまり便利ではありません。 再定義された組み合わせ：Ctrl + DeleteとShift + Deleteは、それぞれCtrl + InsertとShift + Insertとして機能します。 他の組み合わせでは、Deleteキーは通常どおり動作します。



プログラムを起動する


Appsキーは、常に私のキーボードで最も役に立たないボタンの1つでした。 「カスタム修飾子」としてマークしました。 おそらくこれは、彼女がCtrl、Alt、またはShiftの原則に取り組み始めたことを意味します。 つまり それ自体では、何も意味しませんが、他のボタンの動作を変更します。 これで、Apps +ボタン（または2つのボタン）がさまざまなプログラムを起動します。Apps+ F、FはFirefoxを起動し、Apps + PはPuTTY、Apps + S、C-WinSCP、Apps + F、A-Farを購入します。

RCtrl +ボタンでハングアップしたプログラマーのタスクが少ない：RCtrl + F-すべて、RCtrl + V-CamStudio、RCtrl + NumPad追加-電子メールメッセージの作成。

これまでのところ、私が任命したのはこれだけです。慣れてきました。



特殊文字を挿入する


テキストに特殊文字を追加するのが好きです。 LWin + '-強調、LWin +各カーソルは対応する方向の矢印を印刷、LWin + c-©を追加、Lwin + Tabはタブを挿入（タブはブラウザーまたはメッセンジャーのテキストフィールドに挿入するために使用できます）。



マウス


マウスには2つの追加ボタンがあります（X1とX2）。 デフォルトでは、それらは「フォワード」として機能します。 通常、進むボタンは必要ありません。 しかし、マウスのスクロールをクリックするのは不快です（クリックなしでホイールが自由に回転します）。 中央のボタンのクリックとしてX2を再割り当てしました。

LAlt +スクロールでサウンドが調整されるようになりました。 LAlt +右ボタンで音のオン/オフを切り替えます。 キーボードにはサウンドコントロールボタンがありますが、ホイールを使用する方が便利です。



テキストを入力または貼り付けます


PuTTYコンソールでは、多くの場合、同じコマンドを実行する必要があります（アプリケーションを再起動し、画面にログを表示します）。 多くの場合、コードにテンプレートのコードを挿入する必要があります（xhrリクエスト、エラー表示付きの検証呼び出し、PHPプレースホルダータグ、Pythonオブジェクトの印刷出力デザイン）。

キーマネージャは、バッファからテキストを貼り付けてテキストを入力することもできます。 前の段落はすべてRAlt +文字で実行されるようになりました。



おわりに


私の意見では、 ATNSOFT Key Managerプログラムは非常に便利ですが、最初に書いたように、グリッチがないわけではありません。 そして、他のさまざまな機能を追加できます。 コストは低く、彼らは無料で（一見）アップデートを提供することを約束しました。 この素晴らしいプログラムの開発者に感謝し、彼らの継続的な成功を願っています！
