Windows 7用IE11の公式リリース





2時間前にWindows 7用のInternet Explorerの公式リリースが行われましたが、これは20種類の Internet Explorerをサポートする必要があるすべてのWeb開発者にとって重要なイベントです。



そのため、95言語のIE11は既にダウンロード可能です。 今後数週間にわたって、Windows 7ユーザーはIEの新しいバージョンに自動的にアップグレードします。 アップデートは、IE11 Developer PreviewおよびRelease Previewをインストールしたユーザーから本日から始まります。



MicrosoftはIE11ユーザーに記録的なページレンダリング速度を約束しています。 たとえば、SunSpiderテストのWindows 7でのIE11は、IE10を除く他のブラウザーよりも30％高速です。







IE11は、はるかに効率的なガベージコレクターとJITコンパイラーを実装しています。 Microsoftは、ポップコーンを作成し 、Chrome、Firefox、IE11で結果を比較するためのテストを実行することにより、ブラウザーの高い生産性を検証することを提案しています。



IE11は、他のブラウザーではまだサポートされていないWeb標準をサポートしているため、IE用の美しいオリジナルアプリケーションを作成できます。 たとえば、 World Dataデモでは、 ECMAScript Internationalization API（バージョン1.0）を使用して何ができるかを示しています。これにより、国家標準に従ってデータをソートおよびフォーマットできます。 計算は、サーバーにアクセスせずに、クライアントのブラウザーのJavaScriptで実行されます。







マイクロソフトは、現在サポートされているIEブラウザーの20のバージョンすべてでサイトをテストするために、 modern.IEツールキットを使用するWebサイト開発者を提供しています。 無料の仮想マシンをダウンロードして、Windows、Linux、およびMacプラットフォームでテストすることもできます。
