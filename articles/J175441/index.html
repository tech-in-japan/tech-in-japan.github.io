Windows 8およびSystem Center 2012 SP1の新しいMVAコース

こんにちは、同僚の皆さん！

あなたと私は4月-第3四半期を始めました。学校や研究所の時間を思い出せば-これは1年または学期の最後の四半期-集中トレーニングと試験合格の時間です。 そして、この一般的な波で、 マイクロソフトの仮想アカデミー-MVAの助けを借りて、すべての愛好家に科学の花崗岩をかじる機会を提供することも決定しました。

たくさんの新しいコースが到着しました：







1）Windows 8の2つのコース： 「システム管理者向けのWindows 8の技術概要」および「エンタープライズでのWindows 8の展開」 。 1年目には、ITスペシャリストの観点から興味深い新しいクライアントOSの機能を検討します。 ユーザーとの対話のプロセスを変更する問題、およびインフラストラクチャエンジニアの観点からの8-kiのすべての新しいメカニズムが考慮されます。 2番目の例では、具体的な例を使用して、応答ファイルの準備から構成済みOSのイメージの作成と適用まで、展開のすべての基本段階を示します。



2）Alexei KibkaloのWindows Server 2012の2つのコース： 「Windows Server 2012 RDSとApplication Virtualization 5.0の紹介」コースでは、リモートデスクトップサービスとApp-V 5.0テクノロジの新しいバージョンの機能を明らかにします。 Windows Server 2012 Active Directoryドメインサービスの概要コースでは、Active Directoryの展開を簡素化および高速化し、仮想マシンのドメインコントローラーの信頼性の高い動作を保証する変更について説明します。



3）そして、MVAで初めて、Windows Intuneを使用してクライアントコンピューターとデバイスを管理するトピックに触れます。 Windows Intuneクラウドサービスの概要コースは、この分野のMVPであるValery Volobuevによって作成されました。



4）Alexander Shapovalと私、George Gadhihievのコース「プライベートクラウドの新機能」では、 System Center 2012 SP1およびWindows Server 2012 SP1に基づいてプライベートクラウドを構築および管理する際の主な興味深いポイントを説明します。

また、4月1日に新しいコンペティション「Learn、Learn、Learn」が開始されたことを思い出してください。このコンペティションでは、ソフトライン、LANIT Networking Academy、Mikrotest、ZvezdaおよびCトレーニングセンターのマイクロソフトテクノロジーコースを20％割引で利用できます。
