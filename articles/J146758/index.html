JQuery 2.0はIE 6/7/8をサポートしません

JQuery開発者は、ライブラリの新しいバージョンをリリースする計画を公開しています。



  • jQuery 1.8 ：1か月後に公開されます。
  • jQuery 1.9（2013年初期） ：バージョン1.8で廃止された多くのインターフェースを削除。 それらの一部は、プラグインまたは代替APIとして残ります。 IE 6/7/8のサポートは引き続き維持されます。
  • jQuery 1.9.x（2013年以降） ：パッチ、バグ修正、新しいブラウザのサポートなどが追加されたバージョン
  • jQuery 2.0（2013年初期、1.9のすぐ後） ：jQuery 1.9と同じAPIをサポートしますが、イベントモデル、IE7の「属性」、HTML5shimなどのIE 6/7/8の「奇異」を削除します。 。


目標は、サポートされているAPIスイート内で1.9と2.0を交換できるようにすることです。 jQuery 2.0がリリースされると、開発者は使用するバージョンを選択できます。IE6/7/8サポートが必要ですか。 JQuery 2.0はより小さくなり、パフォーマンスが向上します。



または、jQuery 2.0に直接切り替えることができますが、古いIEを使用する訪問者に対してjQuery 1.9のダウンロードを有効にします。 



<!--[if lt IE 9]> <script src="jquery-1.9.0.js"></script> <![endif]--> <!--[if gte IE 9]><!--> <script src="jquery-2.0.0.js"><</script> <!--<![endif]-->


jQuery 2.0用のoldIEプラグインはありません。これにより、ライブラリコードが複雑になります。
