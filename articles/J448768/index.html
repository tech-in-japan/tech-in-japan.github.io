マルチレベル照明制御：ソリューションと製品の弾力性





マルチレベル照明制御は、照明システムのシンプルでエネルギー効率の良い制御を実装するように設計されています。複数の場所から照明をオンまたはオフにする、グループで照明をオンまたはオフにする、一般的な中央オンまたはオフにする必要がある場合に使用されます。



ハードウェアの復元力、したがって実際の長期運用の観点から、いくつかの基本的なソリューションと製品を検討してください。



マルチレベル照明制御システムの例



レベル1制御-建物内のすべての光源（複数の場所から制御される光源を含む）。



第2レベルの制御-1階の左翼のグループに結合された光源、1階の右翼のグループに結合された光源、1階の左翼にグループに結合された光源、2階の右翼にグループに結合された光源床。



レベル3制御-1階全体でグループ化された光源、2階全体でグループ化された光源



レベル4制御-家全体でグループにまとめられた光源。



そのようなシステムを構築できるソリューション



  1. PLC
  2. インパルスリレー。
  3. 独自設計のモジュール式照明制御デバイスに基づいた、非プログラマブルロジックの技術的手段（KTS NPL）の複合体。


CTC NPLに基づくマルチレベル照明制御の記事でCTC NPLについて読むことができます。



電気機械式照明制御装置は、36 mm幅のDINレール（2モジュール）に設置するためのコンパクトな制御モジュールです。









運営管理



制御は、2つのノーマルオープン接点でロックせずにダブルボタンを使用して実行されます。







KTS NPLの開発の理由



KTS NPLが開発された理由は、PLCを使用せずにマルチレベル照明制御システムの機能を実装したい顧客の技術的なタスクでした（予約が非常に高価なため）。



コテージのマルチレベル照明制御システムの機能の例







照明制御デバイスに基づいたフェールセーフシステムを検討する



構成：



1.照明制御デバイス。



機器のコスト：ソースごとに47ドル。

電気的耐久性：AC-1で100,000サイクル。



照明制御デバイスの1つが故障した場合、照明制御システムの他のすべてのデバイスは引き続き動作します。



つまり、照明制御デバイスが故障しても、1つの光源または1つのグループスイッチを除き、照明は引き続き動作しますが、マスターは新しい機器を設置して動作させます。



フォールトトレラントPLCシステムを検討する







構成：



  1. プログラマブルロジックコントローラー。
  2. 冗長プログラマブルロジックコントローラー。
  3. I / Oモジュール
  4. 冗長I / Oモジュール
  5. 冗長デバイス（スタンバイPLCおよびスタンバイI / Oモジュールへの制御切り替えを提供）。
  6. 中間リレー。
  7. アクチュエーター（リレー/コンタクター）。


機器のコスト：1つの光源に対して237ドル。

電気的耐久性：AC-1で100,000サイクル。



PLCまたはI / Oモジュールに障害が発生した場合、冗長デバイスはリアルタイムで制御を冗長PLCおよび冗長I / Oモジュールに切り替え、事故を通知します。



これは、PLCが故障した場合、照明は引き続き動作し、マスターが新しい機器を設置して動作させることを意味します。



非冗長PLCシステムを検討する



構成：



  1. プログラマブルロジックコントローラー。
  2. I / Oモジュール
  3. 中間リレー。
  4. アクチュエーター（リレー/コンタクター）。


機器のコスト：単一の光源で69ドル。

電気的耐久性：AC-1で100,000サイクル。



PLCまたはI / Oモジュールに障害が発生すると、マスターが新しい機器をインストールして動作させるまで照明は完全に動作を停止します。



住宅部門で最​​も一般的なPLCベースのシステムを検討してください



構成：



  1. プログラマブルロジックコントローラー
  2. 入力/出力モジュール
  3. 入力リレー


設備費：1光源につき41ドル。

電気的耐久性：AC-1で25,000サイクル。



PLCまたはI / Oモジュールに障害が発生すると（電気的耐久性が4倍低いため、これは以前のバージョンよりもはるかに速く発生します）、マスターが新しい機器をインストールして動作させるまで照明は完全に動作を停止します。



パルスリレーに基づいたシステムを検討する



構成：



  1. インパルスリレー。
  2. グループ管理モジュール。
  3. 中央管理モジュール。


設備費：1光源につき73ドル。

電気的耐久性：AC-1で100,000サイクル。



リレーの1つが故障しても、他のすべての照明制御リレーは動作し続けます。



つまり、パルスリレーが故障しても、1つの光源または1つのグループスイッチを除き、照明は引き続き動作しますが、マスターは新しい機器を設置して動作させます。



一見、パルスリレーは照明制御デバイスとそれほど違いはありませんが、そうではありません。パルスリレーにはいくつかの制限があります。



  1. 切り替え制限：1分あたり5〜15シフト/ 1日あたり100シフト。
  2. パルス幅の制限：50 ms-1 s。
  3. 振動は、自発的な切り替えにつながる可能性があります。つまり、必要に応じて、そのような制御キャビネットにコンタクタを設置することはできません。
  4. 隣接するパルスリレーのオン/オフを切り替える場合、制御キャビネットの換気と冷却が必要になる場合があります。
  5. 制御レベルの数が増えると、回路構築の複雑さが増します。


おわりに



耐障害性のあるマルチレベルPLCベースの照明制御システムは、住宅部門ではかなり高いコストがかかり、パルスリレーに基づくシステムには重大な制限があり、照明制御デバイスに基づくシステムは黄金色です。
