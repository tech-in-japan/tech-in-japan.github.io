Ashaのプレミアム開発者プログラム-ノキアの限定報奨金

約2年前、Nokia Launchpadに登録することができた開発者は、条件なしで無料でLumia 800を受け取りました。これらの時代は過ぎ去り、ノキアはケーキではなく 、新しいボーナスはそれほど魅力的ではありませんが、それでもオファーには利点があります。

このプログラムに関する情報がノキア自身のhabr-blogに掲載されなかった理由はわかりません-おそらく賞品の「堅実性」に関連する否定的な波が予想されるためです...

Ashaのプレミアム開発者プログラム 。



詳細はこちらwww.developer.nokia.com/Developer_Programs/Asha_developer_program.xhtml

任意のストア（ Nokia Store、Windows Phoneストア、Google Play、BlackBerry World、Apple App Store、その他 ）で2つのアプリケーションを公開している開発者は、プログラムに参加できます。

ボーナスとして提供するもの：

  • Asha 310、WiFi搭載デュアルSIMスマートフォン（推定コスト〜102ドル） www.developer.nokia.com/Devices/Device_specifications/Nokia_Asha_310
  • リモートデバイスアクセスの拡張。 テスト用のデバイスへのリモートアクセス（プログラムの参加者は使用可能なデバイスがさらに多くありますが、これが数だけに適用されるのかモデルが異なるのかわかりません。同じAsha s40デバイスの数だけが使用できると思います）
  • それらの2つのチケット。 サポート（概算コスト〜198ドル）


しかし、権利に加えて、 www.developer.nokia.com / Developer_Programs / Asha_developer_program_terms_and_conditions.xhtmlにも責任があります。

結果として得られるデバイスは、デバイスを受け取ってから6か月以内にアプリケーションを公開し（Java SDKまたはNokia Web Tools 2.3-少し奇妙で限定的なhtml + javascriptを使用） 、公開する必要があります。 （ちなみに、別のボーナスは、アプリケーションが特定の基準を満たしている場合、追加でアドバタイズできることです）。



個人的には、この提案は興味深いです。Asha 開発者コンペティション www.developer.nokia.com/ashadevelopercompetitionは 6月30日まで開催されます。もちろん、実際のデバイスでテストする方が良いです（そして、4000rのこのような弱い特性の電話を購入すると、ヒキガエルが窒息します）
