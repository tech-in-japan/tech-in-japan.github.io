eMachines E732G：同じ、しかし安い

最も強力な鉄片のレビューを書くことほど面白いものはありません。 そして、それらについて読むことも、たいていとても面白いです。 しかし、購入に関しては、よく読んでいるあなたは、本当の経済的機会について考え、より簡単なオプションを選択します。 （少なくとも実際に解決されたタスクのフレームワーク内で）悪化することが判明するという事実からはほど遠いが、より安い-間違いない。



ビデオカードからウィンターシューズまで何でも選択するときに、同様のアプローチが機能します。 そして、最も厄介なのは、旗艦のレビューは数十ドルであるが、賢明な中間農民について読むことはめったにない。 最近、妻のためにラップトップを購入することに気をつけて、私は「優しく」eMachines E732Gの勝者について話し、同時に家庭用モバイルコンピュータを選択する一般的なアプローチについて話すことにしました。







現在までに、Acer TravelMate 3022の妻は4年前に購入しました。 コンピューターの標準でかなり立派な年齢で、彼は悪くはありませんでした。彼は剥がれず、すべての「ネイティブ」パーツが配置されていて、修理を求めませんでした。 交換を検討しなければならなかった理由は2つあります。 1つ目は、昨年1年半でラップトップが小さな子供の側から毎日好奇心にさらされたという事実によるものです。そのため、キーを失い、強力なポークを得て、ゴム製の脚がなく、テーブルから激しくクラッシュしました。 キーは元の位置に戻りました。ゴム製の脚です。はい、冗談を言っていましたが、落下によるひび割れはありませんでしたが、画面の右下隅に小さな斑点が現れました。 つまり、あなたはそれを使用することができますが、みすぼらしい感覚が残っています。 2番目の理由はもっと深刻です。最初からラップトップを使用していたバッテリーの容量が増えたことで、実際に長寿命が命じられました。4時間ではなく、40分間ドラッグし始めました。 そして、通常のものは時間を強さから守ります、それは率直に言って十分ではありません。 アパート内で使用する場合、それ以上必要ではないと思うかもしれませんが、そうではありません。ラップトップが育児室にある場合、接続された電源コードは飛ぶリスクを半桁増加させます。 したがって、漫画と母親のサーフィンには3時間だけが必要です。子供は慎重に眠りにつくふりをします。 より良い-もっと。 もちろん、新しい大容量バッテリーを購入しようとしましたが、モデルが古くて人気が低かったため、見つけることができませんでした。 その後、私は決めました-時が来ました。





外観上、デバイスが4年前のものであるとすぐに言うことはありません。



ラップトップから次のものが必要でした：



-少なくとも3時間のバッテリー寿命

-最新の、可能な限り少ない「カットオフ」デュアルコアプロセッサ（ 安価なCPUシリーズは通常の速度ですが、省エネ機能は最小限です。最初の要件の実装に悪影響を及ぼします ）

-適度な重量で可能な限り画面の対角線を大きくする（ 目を細めることなく一定の距離から見るため ）

-802.11nのサポート（ 私はこの標準で長い間ホームネットワークを持っています。ハードドライブにコピーせずに、無線でNASを使ってHDで映画や漫画を見るのは非常に便利です ）

-子どもたちが押すのが大好きな最小限の追加ボタンを備えた便利なキーボード。サウンドを削減し、20個のプレーヤーウィンドウを開きます。

-価格は25,000ルーブル以内です。 私が貪欲だからではなく、子供の熱意が日々高まっているというだけであり、ラップトップで何か悪いことが起こったとしても、それは残念なことではありません。 この段落への追加は、可能な限り汚れが少なく、同時に外部の影響に対する耐性がある場合です。



私はすぐに言います：私はすぐにエイサーのラップトップの間で検索し始めました。 宗教的な好みではありません。 私は他のブランドも尊敬していますが、3人のエイサーは自宅で暮らしており、誰もが気に入っています。 私はすでに1つについて言及しました.2つ目はAcer D250ネットブックであり、これは私と妻とすでに地球のかなりの半分を取り囲み、3つ目は17インチのスクリーンを備えた純粋にデスクトップのPackard Bell LJ65です。 パッカードベルの名前に注意を払ってはいけません。これは同じ「エイサー」であり、より「高価な」デザインです。 私はもう一度強調します：非常に価値のあるモデルは他のブランドの下でも生産されていますが、これまでのところエイサーは私を失望させません-良いものから良いものを探したくありません。 少なくとも自分の費用で:) 4年後にバッテリーで死ぬとは思いません。ほとんどの友人にとって、バッテリーはさらに早くオークを与えました。



13〜15インチの対角線でいくつかのオプションを検討した後、彼はもっと-安いことに注意しました。 Core i5 / i7のオプションは拒否されました-少し高価であり、タスクを考えると、あまり有用ではありません。 Core i3を搭載したラップトップに焦点を合わせることにしましたが、比較的少ないことがわかりました。 実際、信頼できる会社には、プロセッサの周波数とビデオカードのタイプが異なる5741と5742の2つのAspireしかありません（重要なニュアンス：何でも欲しいのですが、欲望のオブジェクトが2週間以内に持ち込まれたら...）。 考えた後、24,500ルーブルでAcer Aspire 5742G-373G32Mikkに立ち寄りました（アップグレードから2週間でモスクワのノートパソコンの価格がわずかに下がったことを強調する価値があります-明らかに、売り手は急激なルーブル為替レートについて吐き出し、安全でなくなったようです）。 32ナノメートル（読み取り、コールド、長時間再生）のコアi3-370（2.4 GHz）プロセッサ、15.6インチスクリーン（1366x768）、3 GBのメモリ、320 GBのハードドライブ、ディスクリートRadeon Mobile 5470アダプタが非常に優れているように見えました。ラップトップでプレイする人はいなかったので、それほど重要ではないようですが、ウェブサーフィンを加速するために最近GPUがどのように使用されているかを見て、痛くないと決めました。 キーボードは目が痛く、目障りな重さ2.6 kgです。



私は実際にそれを取りに行くつもりでしたが、知識のある人は私にeMachinesのラップトップを見るように勧めました。 最初はinりました。はい、eMachinesはAcerブランドでもありますが、その下で販売されたものは、Ikeaツールボックスを備えたグレープボックスの愛の果実に非常に似ていました。 私の妻は、もちろん、ザンカディアからの多くの移民のように気取らない（私は明らかにします-私自身はそうです）が、それほどではありません！ しかし、よく調べてみると、eMachinesの外観が著しくまっすぐになっていることがわかりました。 もちろん、私はまだ鑑定家ですが、そのような女性を見せることは恥ずべきことではないようです。

私は何か他のものに驚いた：eMachinesシリーズで、Aspire 5742Gのトータルクローンを見つけましたが、10％安くなっています！ 彼の名前はE732G-373G32Mnkkであり、実際には、違いは上部カバーのロゴとカードリーダーの位置にあります。 残りは双子ですが、価格は24,500ではなく、22,000です。不審な男なので、漁獲量を探し始めました。 たぶん、より少ない保証？ いいえ、両方とも1年間です。 サービスセンターは同じです。 給与の半分で疲れた中国人が夜勤でそれを行うことはできますか？ はい、それは劣らず堅実に見えます。 キーボードは1対1です...私はそのようにeMachinesを考え、考え、取りました。 桃やメロンはリンゴではなく「macbook」の蓋に描かれているため、熱心な大好物であっても10％割引になると思います。 この場合、違いはそのような規模です。



家で、部隊を妻と息子の粘り強い手に渡す前に、彼は少し手をいじりました。 事前にインストールされたWindows 7（「Home Basic」ですが、すぐに64ビット）に疑似有用なソフトウェアが散らばっていなかったことに非常に満足しています。 実際、彼らはNorton Internet Securityを解体するだけでよく、私はあまり好きではありませんでした。事前インストールされたいくつかのアーケードを除き、それ以上はありませんでした。







ネットワークに接続するときに、宣言された特性との不一致がありました。価格タグには、イーサネットはここでは100メガビット、実際にはAspire 5742Gのようにギガビットであると書かれていました。 ワイヤレスでも「トリック」されると思いましたが、Atherosアダプターは802.11gしかサポートしていません。 まあ、何もありません、800ルーブルのnサポートを備えたminiPCIアダプターが見つかるため、ストアでこの欠点に対処しました。すぐに変更します。





イーサネットポートの場所は不便です。 誰も使用しないことを保証します。



彼は組み込みのWindows puzomerkaを立ち上げ、率直に言って少し悲しい。 私のPackard Bellはかなり強力なラップトップだと思っていましたが、平均的なeMachinesは一度にそれを行います。

そして、WinRarに組み込まれたベンチマークを立ち上げたとき、私は本当に退屈でした。eMachinesのCore Core i3-370は、Core 2 Duoに対して800倍ではなく1567ポイント押し出されました。明らかに、300メガヘルツ（Core 2 Duo T6500 2.1 GHz）このような増加はありません-ここでは、アーキテクチャの改善とハイパースレッディングテクノロジーの存在の両方が感じられます。 一般的に、家族で最も強力なラップトップは私のものではないことが判明しました。







少なくとも、Core i7-965 Extreme Editionを備えた政府のテストステーションが存在することで、自分自身に対する敬意を維持することができます。



負荷がなければ、ラップトップはほぼ無音で、負荷がかかっても、一般的には悲鳴を上げません。 冷却システムは右側から熱い空気を取り除き、内側の鉄はあまり熱くありません-少なくとも膝はまったく熱くありません。 プロセッサとビデオチップの周波数は完全にリセットされます：Word + ICQモードでは2.4 GHzではなくCore i3が最大900 MHz、GPUは750ではなく最大175 MHzで駆動します。標準の6セルバッテリーから、サーフィンモードのラップトップの寿命は4時間弱です。 SDのCinemaでは、3時間半後に息切れし、2時間後にはHDになります。 まったく問題ありません。







画面の明るさのマージンは単純に巨大で、すぐに半分の明るさにします。これで十分です。 空からの内蔵スピーカーは星をつかみません-通常のものは大声です。 3つのUSB 2.0ポート、HDMI、カードリーダー、DVDカッター-すべて揃っています。
