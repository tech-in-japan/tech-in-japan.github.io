Cloud Expo Asia Hong Kong-アジア最大のIT展示会からのレポート

Cloud Expo Asia Hong Kongの展示会と会議は、クラウド、クラウドセキュリティ、モノのインターネット、データセンターの4つの領域を組み合わせています。 アジアには常に問題を解決するための独自の特別な方法があり、中国の「雲への道」を見る誘惑は非常に素晴らしかったため、勇敢な新しい世界を探検する気持ちの良い予想で10時間の飛行さえも過ぎました。



東部の文化では、3匹の猿のイメージがあります。サンサルは、悪を見たり、悪いことを聞いたり、悪について話したりしないために、目、耳、口を覆っています。 この場合、人自身が悪から保護されると考えられています。 しかし、西洋の解釈は、自分自身ではなく、彼の周りの世界に焦点を当てているため、旧世界と新世界では、猿はしばしば悪の象徴になります。 結局のところ、見ていない、聞いていない、それに反応していない、彼らはその広がりに貢献しています。 同じオブジェクトが表示されますが、コンテキストに没頭しているレベルに応じて、その意味の解釈は根本的に異なります。 したがって、このメモを面白くて役立つものにするために、私はサンザーとは異なり、会議の2日間、猿が各人形でカバーしているすべてのもの-視覚、聴覚、音声-を積極的に使用しなければなりませんでした。 そして、これはそれから来たものです。









みざる（見ざる）-見えない



このイベントは、市内の島部にある香港コンベンションアンドエキシビションセンター（HKCEC）で開催されました。 展示会が占める地域は、Infosecurity Russiaの最後の2つまたは3つ、またはサンフランシスコでRSAをホストするMoscone Centerのホールの半分に匹敵するように感じました。 会議中に、クラウド、クラウドセキュリティ、スマートIoT、DataCentreの4つのエリアが1つのサイトに集められました。 総スペースのほぼ半分がDataCentre World専用でした。 これは、サーバーラック、光ファイバー通信システム、ラックサーバー、通信機器の領域でした。一般的に、データセンターの建設を想像することはできません。 クラウドとクラウドセキュリティのセクションは、展示会のほぼ半分を占め、100を超えるスタンドのうち14のみがスマートIoTに残っていました。



Microsoft、HGC（Hutchinson Global Communications）、中国のテレコムおよびクラウドプロバイダーChinac、Alibaba Cloud、China Telecomは、最大かつ最も明るいスタンドで最も有利な場所（最初の行）を取りました。 この事実を展示のほぼ半数が「鉄」で占められているという事実と比較すると、中国香港ゾーンでは、「クラウド」は主にデータセンターとそこから販売されるプライベートおよびハイブリッドクラウドサービスであると想定できます。



博覧会を見ると、中国が香港のボトルネックからアジア太平洋地域へのクロールに苦労しているように感じました。 これは、大規模な電気通信プロバイダー（RedToneなど）とAlibabaCloudなどの大規模な営利企業の両方に適用されます。 後者は、私にとって、この展示会での予期せぬ発見の1つになりました。



私の同僚や知人の大部分は、Alibabaは中国起源の商品やさまざまな品質のオンラインストアとしてのみ知られています。 一方、Aliyunとしても広く知られているAlibaba Cloud部門は、中国だけでなく、アメリカ、ヨーロッパ、シンガポール、香港にも独自のデータセンターを持っています。 次のDCは2017年にマレーシアで開きます。 その中で、Apsaraリレーショナルデータベースに基づいてDBMSに焦点を当てたIaaSサービスとPaaSの両方を提供しています。 最後に、Alibaba Cloudは2028年までオリンピック大会とeコマースプラットフォームのクラウドインフラストラクチャの公式プロバイダーになりました。



現在、APAC地域の主要なクラウドサービスプロバイダー（主にIaaSおよびPaaS）は、Microsoft、AWS、Google、およびIBMの巨人であり続けており、これらはすぐに前進しています。 しかし、アリという名前の中国のアトラスは、クラウドで肩をより広く広げており、メインスポンサーであるCloud Expo AsiaのKeynote TheaterですでにMicrosoftの隣に座っています。



















きかざる（聞かざる）-聞こえない



会議で講演者のリストを勉強し始めたとき、私は2匹目のサルの運命から逃れることができないことに気付きました。 常にパフォーマンスが行われている14の劇場では、すべてを聞く機会がありませんでした。 最後に、クラウドセキュリティアライアンス、クラウドイノベーション、サービス＆アプリケーション、サービスプロバイダー＆クラウドエコシステム-テクノロジー＆ビジネス＆DevOps、主に基調講演の聴衆に焦点を当てました。 パフォーマンスは30分ごとに開催されたので、少なくとも1時間に1回は、興味深いトピックとさらに興味深いトピックを厳しく選択する必要がありました。 クラウドセキュリティアライアンスの物語は、アジアのクラウドの主なものは彼らの地上性であるという私の仮説を確認しました。



州および非常に大規模な企業は、物理的に構築されたデータセンター、データセンターで働く人々、および国の経済への他の投資に転換し始めると、クラウドテクノロジーに関心を持ちます。 クラウドサービスプロバイダーのマイクロソフト、アマゾン、およびグーグルのポートフォリオは市場のニーズを満たすには不十分になりつつあり、「ローカル」データセンターが登場しますが、コロケーションサービスではなく、レンタルコンピューティングパワーの柔軟な料金で導入されます。



クラウドセキュリティアライアンスの活動分野の1つは、クラウドテクノロジーの積極的な使用に対する各国の準備状況に関するレポートの作成です。 この評価は、データセンター（人間、エネルギー、生産、政治など）の建設のための生産リソースの可用性と財務に基づいています。 2番目の要因は、物理コンピューティングシステムの所有コスト、国内生活の一般的な基準、クラウドリソースのレンタルコストと自社サーバーおよび情報システムの取得/レンタルのコストの違いを考慮に入れています。 政治的リスク、地震や津波などの自然の脅威も、準備指標の作成時に考慮されます。 総合評価によると、日本とシンガポールが第一位であった展示会で報告が示されました。 1番目と2番目の地域の両方で、スペースが大幅に不足し、生活費と熟練した専門家のコストが高いため、IT資産の所有コストが発生します。 それらをクラウドに持ち込むことで、顧客コストを削減することが提案されています。



この辛くて香ばしいソースの下で、カンファレンスのスピーチの90％が開催されました。 現場からは、クラウドインフラストラクチャへの切り替え時に企業の支出が繰り返し削減されるという噂がありました。 プロジェクターは、既に幸せなプレゼントとさらに幸せな未来の写真をスライドさせて見せました。 ベンダーは既に移行を完了しているブランドとジャグリングしました。CapitalOne、Suncorp、DBSなどの巨人が言及されましたが、声明室からは聞こえませんでした。「私たちはすでに雲の中にいます。 興味深いことに、展示会での2日間の作業中に聴衆からの質問やコメントは1つもありませんでした。 満席の聴衆（ほとんどの場合、少なくとも50％は満席）がスピーカーに喜んで拍手を送り、会場から逃げ出してお土産の新しい部分を集めました。 人々は雲を夢見て、そのような馴染みのある堅実で自信に満ちた感動的な地球に駆け戻りました。















いわざる（言わざる）-しゃべらない



会議で最も興味深いのはコミュニケーションです。 そして、私自身の経験から、プログラムのこの部分は最も時間がかかります。 質問に対する回答を迅速かつ効率的に受け取るには、短いインタビューに勝るものはありません。 もちろん、スコポラミンはまだありますが、会議での使用はやや困難です。 しかし、彼はそこでは必要ありません。 人々は喜んで情報と経験を共有し、アドバイスを与え、エコーだけが寒い土地に届くような出来事について興味深い詳細を話しました。



情報セキュリティ専門家のPISA専門家協会の同僚は、香港証券取引所への攻撃について熱心に語りました。 香港でロシアの銀行が標的型攻撃の別の波を受けていた頃、サイバー犯罪者は取引所で取引されるアプリケーションがインストールされたモバイルデバイスを標的にしました。 アプリケーションコードの脆弱性を悪用することにより、攻撃者は実際の所有者に代わって交換操作を実行できました。 攻撃による金銭的および評判上の損害は、PISAの積極的な参加により、香港株式取引市場で機能する30以上のアプリケーションの大規模な監査が実施されたほどの割合でした。 同僚は、研究の結果が今年7月にパブリックドメインに表示されることを約束しました。



情報セキュリティサービス（SOC / MDR / MSSP）に関与する企業の代表者から、アジアのMSSP市場に関連する独自の仮定と仮説の確認を聞くのは興味深いものでした。 MSSP / SOCが外部委託する3つのコンポーネント（テクノロジー、プロセス、専門家）のうち、アジアには、情報とサイバーセキュリティの分野の専門家である第3の強い赤字があります。 そして、最初の路線の形成が多かれ少なかれ高揚するタスクであると思われる場合、2番目の路線の専門家を地域に輸送するか、学習に時間がかかり、費用がかかります。 ワークショップでの西側の同僚の一般的な意見は、鉄だけでなくSOCAチームで有資格のサービスを提供できる「ブルーカラー労働者」の層を作成するためにも、地域に深刻な投資が必要であるというものです。



しざる（しざる）悪ではない



このサルは時々サンザーに加わりますが、ほとんどの場合、3匹が残ります。 シザルが彼の胃やstomach径部を覆っているという事実のために、彼女は宗教的な旧世界で愛されなくなったバージョンがあります。 一方、アジア文化の4位は不幸だと考えられているため、三位一体は三位一体のままです。 いずれにせよ、悪の不完全さを積極的に主張するシザルは、自分自身が彼の犠牲者、無知、仮定、ステレオタイプの犠牲者になりました。 彼女の過ちを犯さない唯一の方法は、知識を収集し、私たちが見聞きするすべてのものを考慮に入れることです。そうして初めて、私たちの決定は正確かつタイムリーになります。



香港は、一見矛盾するものを組み合わせているという事実によって覚えています。 ガラスとスチールで作られた家は、緑豊かな公園と美しい山の斜面に隣接しています。 二階建てのバスと路面電車からの騒音は、動物園の水ジェットの雑音の間で見分けがつかなくなります。







建物の恐ろしい密度は、木々への特別な注意に隣接しています。香港では、それぞれに個別のナンバープレートが装備されており、法律で保護されています。







この都市国家がこの地域の経済で主導的な地位を占め、西洋世界のアジア文化の媒体になったのです。 ビジネスと自然との関係で細部に注意を払っていることは、中国のドラゴンがビクトリア湾の近くの石のジャングルを飛び回ることを可能にしたまさに春だと思います。 そして、このドラゴンが今や雲の中に飛んでいるので、その翼からの影がAPAC地域の境界をはるかに超えて広がるほど高くて速い。
