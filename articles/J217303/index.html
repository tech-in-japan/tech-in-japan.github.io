Dell PowerEdge R820 12Gサーバーの概要

Dell PowerEdge R820は、中規模企業や大企業の仮想化だけでなく、大量のコンピューティングとストレージリソースを必要とするアプリケーション向けに設計されたユニバーサル2U高速ラックサーバーです。 R820サーバーには48個のDIMMと最大1.5TBのDDR3 RAM、7個のPCIe 3.0スロット、追加のGPUアクセラレーターのサポート、およびPCIeスロットを使用せずにメインマトリックススイッチに接続するための多数のDell Selectネットワークアダプターがあります。 ストレージ用の着信データを高速で配信するためにフロントパネルにすばやく接続された4つの追加のExpress Flash PCIe SSDを使用して、I / Oの速度を上げることができます。







Express Flashドライブが初めて、第12世代のPowerEdge R720サーバーモデルに導入され、企業のテストラボの主力となりました。 このR820サーバーにはExpress Flashドライブが装備されていないため、これは高性能と運用上の信頼性の最適な組み合わせを探している人にとっての主な違いです。 兄弟のように、R820はIntel Xeon E5プロセッサを搭載しています。 R820サーバーは、R720でサポートされている2つとは異なり、Xeon E5-4600ファミリーの4つの中央プロセッサをサポートしています。 追加のストレージオプションには、デュアルRAIDコントローラと最大16台の内蔵ハードドライブのサポートが含まれます。



R820のストレージの柔軟性、および2Uフォームファクターを備えた4つのIntel CPUの直接処理能力は、多くの場合R820サーバーに注目を集めています。 要するに、R820は、従来の2Uサーバーよりも小さいシャーシ領域でより高いパフォーマンスを発揮します。 かなりの計算能力、マルチユーザーアクセス、または数十台のコンピューターを必要とするビジネスにとって、このサーバーの利点は明らかです。 クアッドコアプロセッサを搭載したDell R820 サーバーは 、スペースを大幅に節約し、電力消費を削減し、サーバーのメンテナンスに関連するコストを削減できます。



PowerEdge R820サーバーには、4つの2.7 GHz Intel Xeon E5-4650プロセッサーが搭載されており、各プロセッサーには64 GB ECC DDR3が搭載され、合計サーバーメモリは256 GBです。 さらに、サーバーには、管理者がエージェントの助けを借りずにサーバーを使用、更新、監視、保守できる帯域外リモート管理機能用のiDRAC7 Enterprise管理ボードが装備されています。 サーバーには、多層データストレージ用のPerc H710 RAIDカードが装備されています。 HPサーバーのアナログはProLiant DL560 Gen8です



Dell R820は、Dell Webサイトで6,900ドルの開始価格で完全に構成可能です。

ハードウェア


R820サーバーは、最大8個のプロセッサに対応しており、現在、モデルに応じて、プロセッサあたり最大8個のコアを含む、Intel Xeon E5-4600プロセッサフ​​ァミリの多数の代表製品で利用できます。 検討しているサーバーは、Intel Xeon E5-4650プロセッサを使用しています。 R820サーバーは、アクティブ冷却を備えた最大2つの75W NVIDIA Quadro 2000 GPUアクセラレーターを使用することもできます。これにより、最大1 GBのGDDR5メモリが提供されます。 Dell PowerEDGE R820サーバーは、R720よりも少ないアクセラレーターを提供します。 最大7つのPCIeネットワークカードのサポートにより、追加のアップグレードおよび構成オプションが提供されます。







デルは、4つの内部メモリ構成を備えたR820サーバーを提供しています：2.5インチドライブ8台、2.5インチドライブ8台+ 2.5インチExpress Flash PCIe SSDドライブ4台、8 + 8 2.5インチまたは16ドライブ2.5インチのドライブ。 PowerEdge R820は、175 GBまたは350 GBのExpress Flashドライブと互換性があり、プライマリストレージとして、またはPERCコントローラーと組み合わせて階層型CashCadeストレージとして使用できます。 1.5 TBのUDIMMおよびRDIMMメモリモジュールに加えて、R820サーバーはLRDIMMサーバー用の低負荷メモリモジュールをサポートしています。 LRDIMMはバッファーを使用して、待ち時間を増やして負荷を減らし、密度を高めます。 この構成では、サーバーには、Perc H710 RAIDカードに接続された4つの46GB 15K SASハードドライブが含まれています。



Dell Selectネットワークアダプタの機能には、独自のドーターネットワークカードを備えた多数の組み込みネットワークアダプタカードの選択が含まれます。 R820互換ドーターカードには、Base-T、10Gb Base-T、および10Gb SFP +などの1GbEおよび10GbEポートカードが含まれます。







R820サーバーは、N + 1構成のホットスワップ可能な冷却ファンをサポートしているため、技術者は一度にすべてのファンを交換できます。 第12世代のPowerEdgeサーバー全体と同様に、R820にはフレッシュエア冷却システムが搭載されており、エネルギー消費や関連する冷却コストを増やすことなく、サーバーを35°C / 95°Fを超える温度で動作させることができます。 第12世代のPowerEdgeプラットフォームでは、PCIeネットワークカードなどの追加機器を使用するサーバーの冷却をカスタマイズできます。センサーが過熱点を検出するように構成されていない場合、自動冷却システムの過熱につながる可能性があります。



PowerEdge R820サーバーは、ユーザー設定の要件に応じて、750ワットの交流電流、1100ワットの交流電流、または1100ワットの直流で、自動検出およびスイッチングを備えた最大2つの冗長電源をサポートします。 第 12世代のDell PowerEdgeサーバーには、プロセッサの速度を特定のレベルに下げることで、データセンターのエネルギー制限を設定する機能があります。 これは、電力または温度が制限されている状態で新しいサーバーを使用する場合に特に便利です。 問題のサーバーには2つの1100 W AC電源が装備されています。



設計と製造


PowerEdge R820サーバーのフロントパネルは、16個のコネクタとオプションのロック可能なコネクタを備えた2.5インチのフォームファクタです。 左側には、電源ボタンとインジケータ、トラブルシューティング用のマスク不能割り込み（NMI）、VGAディスプレイの接続、シンプルなLCDコントロールパネル、iDRACで使用するvFlash SDメディアリーダーと2つのUSB 2.0ポートがあります。







デバイスの背面パネルは、2つの活発に接続された電源と対応するインジケータへのアクセス、Select Network Adapterを介したネットワーク機能、および2つのUSB 2.0ポート、iDRAC7ポート、ビデオコネクタ、シリアルコネクタを備えています。 Quick Resource Locatorは、ケースの複数の場所で指定されたサーバーモデルの管理とサポートに関する資料にリダイレクトするQDコードです。







R720サーバーとR820サーバーを区別することは外見上困難であるという事実にもかかわらず、ハウジングカバーを開くとすぐに違いに気付くでしょう。 2つの追加プロセッサをサポートするR820サーバーには、マザーボード上にクイック接続接続で接続されたドーターネットワークカードが装備されています。 ロック要素を開くと、ユニット全体がマザーボード上のプロセッサとRAMコネクタにアクセスできます。 PowerEdge R820構成では、最大48個のDDR3 DIMMを使用できます（R720の2倍）。 このモデルには16 GBのDIMMメモリがあり、合計256 GBのRAMを提供します。



スライドおよび固定ガイドは、特別なツールを使用せずに、ねじのない四角または丸穴のある4つのサポートがあるラック、またはツールを使用してねじ穴のある4つのサポートがあるラックにインストールできます。 2つのサポートを備えたラックに設置するための固定レール（Telco）。 ケーブル管理用の追加のブラケットは、特別なツールを使用せずにスライドレールの左側または右側に取り付けることができます。

特別なツールを使用せずにR820サーバーをレールに取り付けるには、サーバーを前面パネルと背面パネルのハンドルで保持し、サーバーをレール上でそっと下げて、取り付けピンと対応するコネクターを揃えます。 経験から、この方法は他のガイドシステムよりも便利で高速であることがわかっています。



制御と管理


R820およびR720を含む第12世代のPowerEdgeサーバーは 、エージェントの有無にかかわらず管理するためのOpenManageプラットフォームの一部です。 OpenManageプラットフォーム構成の中心には、ライフサイクルコントローラを備えたiDRAC7があります。 エージェントベースのソリューションとの統合のために、OpenManage Server Administraton（OMSA）は、コマンドラインインターフェイスまたはブラウザベースのJava GUIを備えた直接制御システムを提供します。







iDRAC7 Enterpriseは、PowerEdge R820サーバーでiDRAC7 Expressと一緒に利用できます。iDRAC7Expressは、エンタープライズバージョンで利用できる多くの機能を備えています。 Dell OpenManage Power Centerは、ラック、列、およびデータセンターの電力管理を提供します。 R820サーバーには、重複したSDカードを使用する仮想化システムに、中断のないダウンタイムハイパーバイザーを使用する機能があります。



おわりに


PowerEdge R720サーバーと同様、フロントパネルにすばやく接続されたExpress Flash PCIeドライブを使用する場合、R820サーバーは他のサーバーよりもデータを保存する利点を示します。 バックプレーンの高速ストレージ機能に加えて、R820 サーバーの長所には、最大1.5 TBのDDR3メモリを提供する最大48個のDIMMを接続する機能が含まれます。 メモリに大量のデータを保存する機能により、CashCadeから保存されたデータへのアクセス速度が向上するだけでなく、コンピューティング機能が向上します。

PowerEdgeの第12世代サーバーシャ​​ーシ設計は、サーバーがシステムのマウントとトラブルのない操作を備えた柔軟な物理プラットフォームであり、メンテナンスが容易な要素が自由に利用できることを証明しています。 使用可能な構成の印象的な範囲により、サーバーは顧客の要件に最適に対応でき、7つのPCIeスロットは更新のための十分なスペースを提供します。 使用と管理に関して、iDRAC7を使用すると、R820サーバーを最も簡単な方法で運用できます。



ほとんどの場合、R820サーバーの計算能力と密度は必要ありませんが、必要な場合があります。 ストレージの観点から見ると、CPUは、追加のコアを備えた多数のPCIeストレージデバイスを使用する場合に非常に役立ちます。 パフォーマンスを向上させるために処理能力を使用できるFPGAおよびその他の外部メモリコントローラーは、R820でその機能を発揮します。

おわりに


第12世代Dell PowerEdge R820サーバーは、さまざまなハードウェアと実績のある管理機能のセットの機能的な組み合わせを提供します。 Xeonプロセッサの高性能、最大1.5 TBのRAM、オプションのExpress Flash多層内部ストレージを考えると、R820サーバーは利用可能な最速のコンピューティングプラットフォームの1つです。
