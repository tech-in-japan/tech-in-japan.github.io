iCamp-2008でのOpera Softwareとの競争





Opera Softwareはコンテストを発表し、その結果は8月3日にiCampで発表されます！



インターネットの発見 '2008



Opera Softwareは、人々のコミュニケーションを支援するための最高のWebアプリケーションの競争を主催しています。 新しいタイプのWebメールシステムからソーシャルネットワーク、インスタントメッセージングから電子討論、共同執筆散文からマルチユーザーバーチャルリアリティまで、何でもかまいません。 あなたの想像力を制限したくありません。 アイデアが凝っているほど良い。 車とテレビは、商業的価値も未来もない、役に立たない偏心としてブランド化されていたことを思い出してください。



受賞者には貴重な賞品が贈られます。 これ-私たちは言いませんが、Operaはすべての賞を受賞しています。



Opera Softwareは、閉じたフォーマットから解放され、誰もがアクセスできるウェブ標準とインターネットの未来を信じています。 したがって、私たちの要件は、現在または将来のWeb標準で記述されている技術のみを使用することです。 たとえば、SVGは許可されていますが、Adobe Flashは許可されていません。 より正確には、W3C、IETF、および同様の組織の現在の推奨事項と作業ドラフト、および標準化の提案（HTML、CSS、SVG、DOM、MathML for CSS、XML、ECMAScript / Javascript、SMTP、NNTP、IRCなど）を使用できますp。）。 これには、最新の標準化の提案（DOMファイルI / Oおよびウィジェット）が含まれます。 W3Cサイトwww.w3.orgおよびIETF www.ietf.orgに加えて、 dev.opera.comの開発者向けにWeb標準に関する記事と役立つヒントをお勧めします 。



開発は、Opera Softwareがこれまでに公開したブラウザーの少なくとも1つで動作するはずです。 これには、あらゆる種類のアルファリリース、予備リリース、および実験リリースが含まれます。 アイデアに最適なブラウザに応じて、次のブラウザを無料でダウンロードできます。



Windows、UNIX、およびMac用のOpera 9.51 www.opera.com/download

Opera Mini 4.1 www.operamini.com/download

Opera Mobile 9.5ベータ版www.opera.com/products/mobile/download



ただし、特にこのコンテストでは、最新のWeb標準の一部を実験的にサポートするパーソナルコンピューター向けのOperaの特別リリースを行いました。 このリリースはOpera 9.51に似ていますが、さらにVIDEO要素、3次元キャンバス、DOMファイルI / Oをサポートしています。 Windows、UNIX、およびMacのリリースをダウンロードできます。また、ドキュメントを読んでサンプルを見つけることができます： labs.opera.com/news/2008/07/18 実験リリースでは、通常のリリースと同じレベルの安定性が保証されないことに注意してください。 それにもかかわらず、興味深いアイデアが彼らの助けを借りて実装されるのを見てうれしいです！



元のアイデアは、充実感や安定性よりも重要であることを忘れないでください。 非常に興味深いアプリケーションのスケッチでさえ、慎重に実装された通常のプロジェクトよりも価値があります。



作品を送信する必要がありますが、オープンソースライブラリの使用は許可されています。 プログラムコードや画像など、他の人の作品を使用する場合は、著作権を行使する責任があります。



iCamp 2008に到着した人は誰でもコンテストに参加でき、8月1日から2日までのモスクワ時間の午前0時前に作品を提出できます。 エントリーはメールでのみ受け付けられます。 Opera Softwareは、提出された作品を公開する権利を留保します。 チームは参加できますが、優勝チームには賞品が1つしか与えられません。 プロジェクトでWebホスティングが必要な場合、参加者はiCamp 2008の期間のホスティングのパフォーマンスを担当します。適切な場合には、 my.opera.comおよびwidgets.opera.comでホスティングを使用できます。



通常のコンピューター（少なくとも1台はLinuxと1台のMacを実行）とOpera Miniを搭載した携帯電話で作品を見ることができますが、iCamp 2008に持ち込んでプロジェクトのデモを行うと、自家製を含むあらゆる機器を使用できます。



作品の送付先：icamp2008@opera.com
