SMSスパムを取り除く方法





スパムという言葉は従来からメールスパムに関連付けられており、その対処方法は多かれ少なかれ明確です。スパムに送信したり、フィルターを設定したり、スパマーのホスティング事業者に苦情を申し立てたりできます。 ただし、スパムは、受信トレイ内の紙くずや、携帯電話の不要なSMSでもあります。 また、電子メールニュースレターとは異なり、SMSには「登録解除」ボタンがないため、後者をどうするかは必ずしも明確ではありません。 SMSスパムに対処する方法は次のとおりです。この記事で説明します。



SMSスパムの購読を解除する方法



あなた自身がニュースレターにサインアップしている場合：



各自尊心のある会社は、ユーザーが自発的にニュースレターの購読を解除し、広告メッセージの送信を実際に停止できるようにします。 購読解除の条件は、受信したメッセージの追加情報として、または具体的には会社のWebサイトで会社が指定する必要があります。

例：「不要なサービスの登録を解除するには、テキスト「STOP」または「STOP」（大文字と小文字を区別しない）を含む無料のSMSをサービスが提供されている番号に送信します。」



登録を解除できない、または登録しなかった（実際にはSMSスパム）：



携帯電話会社のサービスを使用して、携帯電話から短い有料番号へのSMSの送信をブロックします。 原則として、このサービスは無料で提供されます。 オペレーターは、そのようなクレームを処理するための特別な番号と電子メールを作成しました。

  • 911@mts.ruまたは受信したSMSを無料番号1911に転送する-MTS苦情サービス
  • www.spb.tele2.ru/help_complain.htmlまたは番号0611-TELE2の苦情サービス
  • www.stopfraud.megafon.ru/feedbackまたは番号0500-Megafonの苦情サービス
  • * 110 * 20＃CALL-カメレオンサービスの無効化（広告メール）または電話0611-ビーライン苦情サービス
  • www.mtt.ru/mtt/def-電話番号で、オペレーターの名前を確認して、集中した苦情をすぐに住所に送信できます




または、送信者の番号をブラックリストに登録する必要があります。このサービスを有効にすると、選択したコンテンツプロバイダーの有料SMSサービスを完全にブロックすることができます（着信通話とメッセージをブロックします）。



そして、最も重要なこと-そのようなSMSに返信しないでください！ これは、アカウントからお金を盗む詐欺である可能性があります。



Bluetoothスパム



多くのショッピングセンターでは、中に入るとすぐにBluetooth経由で広告を受け取ります。 いずれにせよ、あなたはそれらを取る必要はありません。 また、それらを表示しないようにするには、たとえば、ファイルを転送する場合など、直接必要な場合にのみbluetoothをオンにすることをお勧めします。 同時に、バッテリーを節約します。 別のオプションは、他のデバイスによるbluetoothデバイスの検出を無効にすることです。



スパムについて苦情を申し立てる人



残ったSMS-okを自分で取り除くだけでなく、スパマーを罰したい場合は、次のオプションがあります。



スパムに関する苦情については、サービスプロバイダーに直接お問い合わせください



ロシアのFAS（連邦独占禁止サービス）に連絡してください。FASは広告法の順守を監督および監視しています。



-ウェブサイトwww.fas.gov.ru/contact-fas/feedbackまたはメールdelo@fas.gov.ruで。 また、苦情は直接郵送するか、郵便またはファックスで送信できます。



ROSKOMNADZORにお問い合わせください



-通信、情報技術、マスコミュニケーションの監督のための連邦政府サービス-ウェブサイトrkn.gov.ru/treatments/ask-question



いずれにせよ、苦情を申し立てるには、特定の情報を提供する必要があります：送信者の名前と番号、メッセージテキストと時間（送信/配信）。 また、送信される情報の正確さとアラートを受信する能力のために、名前、居住地などの個人データも提供する準備をしてください。



苦情について送信されたメッセージに応じて、SMS、手紙、または書留の形で、アクションリクエストの承認を確認する必要があります。



訴訟が法廷に出される場合、モバイル事業者、ディストリビューター、または他のスパマーは、あなた自身ではなく、ニュースレターを受け取ることに同意することを証明します。 スパムメールに本当に同意しなかったことを「良心を落ち着かせる」ようにしてください（サービス契約などで）。



スパマーや詐欺師に捕まらない方法



スパムの配信を停止する必要がないように、スパムを回避する方法に関するヒント：

  • 疑わしいサイトに携帯電話番号を入力しないでください
  • 不明/短い番号からSMSに送信されたリンクをたどらないでください
  • あなたが彼らの信authentic性をしっかりと確信していない限り、未知の番号から送信された挑発的なSMSに応答しないでください
  • 広告メッセージを送信するための条件を明記した携帯電話会社の契約を注意深く読んでください 。 そのような契約に盲目的に加入することで、スパムを受信するために自発的に加入します。




スパム対策電子メールポリシー



スパムとの闘いは、すでに受信したメッセージへの加入者からの苦情に限定されるべきではありません。 スパムは芽で止められ、すべての自尊心のあるSMSメールサービスはスパムと戦うだけでなく、それを防ぐべきです。



AtomPark Software Inc.のePochta SMSアンチスパムポリシーには、次の手段が含まれています。

  • 発信メッセージトラフィックの継続的なチェックと制御。
  • スパムのメッセージの内容の厳密な監視（完全な機密保持）。
  • 短い市外局番からのメッセージをブロックする。
  • 郵送やスパマー送信者のブロックに関する苦情への即時対応。
  • 携帯電話事業者および連邦サービスとの協力。
  • 未承諾SMSを送信する組織または個人との協力を避けます。
