ニンテンドーDSiゲームコンソールのOperaブラウザー









Operaテクノロジーを使用して、ニンテンドーDSiゲーミングコンソールは強力なインターネットアクセスツールになります



オスロ、ノルウェー-2009年4月2日。



Opera Softwareとニンテンドーは、ニンテンドーDSiブラウザーの発売を発表しました。これは、最新のニンテンドーDSiハンドヘルドゲームコンソールのユーザーがWorld Wide Webへのアクセス速度を確認する方法、Webサイトとの連携方法、コンテンツの共有方法を変更するために設計された次世代インターネットブラウザーです。



ニンテンドーDSiブラウザーは、4月3日明日ヨーロッパで、4月5日米国で発売されるニンテンドーDSiコンソールのユーザーを対象としています。 Nintendo DSi Browserは、Nintendo DSiコンソール専用にOpera Softwareによって開発されたもので、Nintendo DSi Shop Webサイトから無料でダウンロードできます。



Opera Softwareの新しい開発により、ニンテンドーDSiがWebリソースを操作するために備えているタッチスクリーンのすべての利点を使用できます。 したがって、ブラウザは、グローバルネットワークで作業するための軽量で直感的なツールであり、あらゆる年齢層およびトレーニングレベルのユーザーにとって自然なものです。



Opera SoftwareのエグゼクティブディレクターであるJon von Techner氏は次のように述べています。 「ニンテンドーは最先端の製品のみをリリースすることで技術的な妥協をせず、ニンテンドーDSiブラウザーはこの会社の高い品質基準と革新に完全に準拠していると確信しています。」



新しいブラウザの詳細とこのページのデモをご覧ください 。

Opera Software ASAについて



Opera Software ASAは、PC、携帯電話、その他のインターネットデバイスでWebページを表示するための新しい可能性を切り開きました。 Operaが開発したクロスプラットフォームブラウザーテクノロジーは、その機能、標準への準拠、小さなサイズで知られ、ユーザーにより高速で安全で動的なインターネットアクセスを提供します。 Opera Softwareのメインオフィスはノルウェーのオスロにあり、世界中にオフィスがあります。 同社は、OPERA証券コードでオスロ証券取引所に上場しています。 詳細については、 www.opera.comをご覧ください。
