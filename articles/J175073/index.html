Evernoteは4年間ロシア語を話します！



ちょうど4年前の2009年4月2日、Evernoteをロシア語でリリースしました。 これはまだ非常に若いサービスのローカライズの最初の経験でした（Evernote-今日ご存知のように、2008年6月24日に登場しました）。 私たちは、Webクライアントと当時存在していたすべてのEvernoteアプリケーションを翻訳することにより、責任を持って国際舞台でのデビューに近づきました。 また、手書きおよび印刷されたテキスト認識技術により、ロシア語のサポートも行いました。



今日、EvernoteはロシアとCIS諸国で150万人以上の人々に使用されており、この数字は増え続けています。 私たちのサービスを彼らの生活の一部にしたすべての人に感謝しています。



今日は、4年間の歴史のハイライトについてお話したいと思います。 古いユーザーが覚えていて、新しいユーザーがEvernoteがこれまでにどのように開発してきたかを知ることが面白いことを願っています。



2009年4月2日。 Evernoteはロシア語で話しました







2009年4月までに、900,000人以上がEvernoteを既に使用していたため、ベストモバイルアプリケーションのノミネート（iPhoneのクライアントごと）でCrunchies 2009を受賞し、国際市場への参入を検討しました。 Evernoteはロシアのルーツを持つ会社であり、創業者と多くの従業員がロシア語を話すため、ローカライズの最初の国に関する決定は満場一致で行われました。



ちょっとした秘密を発見する価値があります。すでに4月1日にローンチする準備は完全に整っていましたが、リスクを冒さないで別の日を待つことにしました。



2009年6月25日。 共同メモ帳がEvernoteに表示されます



他のユーザーや世界中のユーザーにノートブックを開く機能は、Evernoteサービスの重要なイベントになりました。 コラボレーションと知識共有機能は常に進化しています。たとえば、今日では、中小企業向けの本格的なナレッジマネジメント製品であるEvernote Businessがあります。



この新機能は、さまざまな情報を他のユーザーと共有し始めたユーザーの間で人気を集めています。 今日のインターネットでは、多くの開いているノートブックを見つけることができます。 たとえば、 ベジタリアンレシピ 、 管理資料 、 システム管理者向けキット 、 旅行のヒント 、 生産性の ヒント 、 仕事、 Macユーザー向けのヒントなどです 。



2009年7月29日。 Evernote翻訳コミュニティの到来



ボランティアの助けがなければ、Evernoteを母国語のユーザーがアクセスできるようにすることはできません。 Evernoteのファンはこのサービスの多言語への翻訳を熱心に取り上げ、ロシアからのボランティアが多くの努力をしました。



2009年12月16日。 Evernote for Android







かろうじて登場したAndroid版Evernoteは 、当社のサービスで最も人気のあるアプリケーションの1つになり、今日ではGoogle Playの編集者が選択した製品の1つです。



現在、Androidアプリケーションはロシアで最も頻繁に使用されるEvernoteクライアントです。 サービスへのすべての呼び出しの3分の1を占めます。 2番目は、Windows用のWebクライアントとアプリケーションで共有され（それぞれ14％）、iPad用のEvernote上位3位（10％）を閉じます。



2010年7月14日。 Evernoteギャラリーが表示されます







Evernoteの発売後すぐに、オープンで無料のAPIを開発コミュニティに導入しました。これにより、誰でもEvernote機能をアプリケーション、サービス、デバイスに追加できるようになりました。 Evernoteプラットフォーム上のソリューションの数は急速に増加し、 Evernote Galleryを紹介することにより、最高のパートナー製品を実証する場所を作成することにしました。



今日、 ABBYY Text Grabber + TranslatorやExlibris FB2などのロシア語のアプリケーションが存在することを非常に嬉しく思います。



2010年11月11日。 ロシアのユーザー向けの四半期ごとのプレミアムサブスクリプション



プレミアムサブスクリプションを1か月と1年間購入するための標準オプションに加えて、特にロシアのEvernoteユーザー向けに、3か月間追加の支払いオプションを提供しています。 このオプションは、パートナーソフトキーのストアで引き続き利用できます。



2011年2月18日。 ロシア語を話すEvernoteサポートコミュニティの作成



Evernoteユーザーが長期にわたって新たな問題を伝え、議論する公式の場は、英語のフォーラムでした。 ロシア語を話すユーザーのために、私たちは彼らが困難について話し、他の人を助け、従業員に質問をすることができる別のプラットフォームを立ち上げました。



現在、 サポートコミュニティには6,000人以上のユーザーが登録されており、約3500件のディスカッションを作成し、11,000件以上のコメントを残しています。



2011年4月27日。 モスクワで開発者と会う







多くの才能ある開発者がロシアに住んでおり、Evernoteプラットフォームとの統合を通じて製品の機能を拡張してほしいと本当に望んでいます。



2011年4月、モスクワ（デジタル10月センター）に開発者を集め、パートナーシップソリューション、API機能、および毎年恒例のEvernote Devcup Developer Championshipの推進について話しました 。



2011年6月16日。 Evernote for Windows Phone 7を終了します







Evernoteでは、常にさまざまなプラットフォーム向けの本格的なクライアントを作成するよう努めており、2011年までにアカウントに13バージョンのEvernoteがありました。 Windows Phone向けの第14回Evernoteアプリ！ ところで、最近バージョン3.0に更新し、インターフェイスを大幅に再設計し、多くの便利な機能を追加しました。



2011年8月18日。 EvernoteがSkitchを買収し、無料でAndroidバージョンをリリース







2011年に、当社はEvernoteアプリケーションのファミリー全体の登場という、会社の開発の新しい段階を発表しました。 この方向への最初のステップは、画像にメモやコメントを追加するための便利なツールであるSkitchの買収でした。 当時は、Mac用の有料アプリケーションでした。 Skitchは無料で、Evernoteと統合され、Android、Windows、iOS、およびWindows 8 Touchのバージョンがあります。



2011年11月16日。 Evernoteの明確な出力







Evernoteファミリーのアプリケーションの開発を継続し、ブラウザ（最初はChrome、次にFirefox）用の新しい拡張機能-Evernote Clearlyを導入しました。これにより、インターネット上の記事のテキストを邪魔されないWeb要素からクリアして読みやすくすることができます。 さらに、ユーザーはテキストの表示をカスタマイズしたり、記事の重要なポイントを強調表示してEvernoteに保存したりできます。



2011年12月7日。 Evernote HelloとEvernote Foodがリリースされました







Evernoteに特定の種類の情報を追加することの利便性に取り組み、料理の印象と連絡先の詳細という2つの最も人気のあるレコードのカテゴリに特別な注意を払いました。 Evernote FoodとEvernote Helloの特別なアプリケーションが開発されました。







食べ物は、思い出に残るディナーのレシピや印象を保存して表示したり、レストランを検索したりするのに便利な場所です。 Helloは、出会うすべての人々に関するデータを保存するための優れたソリューションです。



2012年5月3日。ロシア語でEvernoteに関する最初の電子書籍







Evernoteについては、電子と紙の両方で多くの本が書かれていますが、それらのほとんどは英語と日本語です。 ロシア語でEvernoteの生産性を向上させる最初の本は、ガニスルタノフの著作「象に餌をやる方法」でした。



2012年5月7日。EvernoteはPenultimateを買収







Evernoteファミリーのアプリに追加されたもう1つのアプリは、人気のiPad手書きアプリであるPenultimateです。 Evernoteと統合し、手書きのメモを検索して同期できるようにしました。 最後から2番目も無料になりました。



2012年8月24日。EvernoteとMoleskineが共同ノートブックを作成







モレスキンのノートブックは、その品質とデザインにより、クリエイティブな人々の間で長い間象徴的な存在でした。 これらのノートブックには欠点が1つしかありませんでした。紙のままでしたが、 Evernoteスマートノートブックで修正されました。 メモ帳ページには、Evernoteを使用した手書きメモの認識を簡素化する特別なマークアップがあり、テキストのデジタルコピーを確実に保存できます。 ノートブックの操作は、iOSおよびAndroidのモバイルアプリケーションでサポートされています。



ノートブックの急ぎの需要のため、彼らはかなり長い間ロシアに旅行しましたが、それでも1月に彼らはロシア市場に現れました。 最近では、Indinotes.comストアはEvernoteユーザーに対して10％の割引を提供しており、これを引き続き利用できます。



2012年9月13日。Evernoteのエキスパート



ロシアでEvernoteエキスパートプログラムを開始しました。これにより、サービスのアクティブで経験豊富なユーザーを特定し、知識を共有できる条件を作成することができました。



Angelica Medvedeva （旅行）、 Armen Petrosyan （生産性）、 Daria Hubert （健康的な食事）、 Marina Eliseeva （時間管理）、およびEvgeny Vyskub （起業家精神）に、彼らの経験を他の人と共有してくれたことに感謝します。 近い将来、新しい興味深い参加者でエキスパートプログラムを補充する予定です。



また、Evernoteエキスパートに質問したり、VKontakteグループの関連分野でサービスを使用するこの側面やその側面について話し合うこともできます。



2012年10月26日。Evernotefor Windows 8 Touchがリリースされました







Evernote for Windows 8 Touchは、アプリケーションストアの初日からWindowsストアで利用でき、Evernoteの主要機能と組み合わせて、このプラットフォームのオーガニックインターフェイスをユーザーに提供します。



2012年12月4日、ロシアの百万人のユーザー

12月には、ロシアからのEvernoteユーザーの数が100万人を超え、成長を続けています。 これは私たちにとって非常に重要なマイルストーンです。私たちは常にロシアを重要な国の1つと考えており、それに加えて歴史的に特別な関係を築いてきました。



2013年2月28日。ロシアでEvernote Businessを開始







ユーザーの66％がビジネス目的でEvernoteを使用しているため、小さな会社やチーム内で情報を保存および交換するための快適な条件を作成し、 Evernote Businessを導入しました。



新製品には、基本サービスとプレミアムサブスクリプションEvernoteのすべての機能が組み込まれ、中小企業向けに特別に作成された多くの新機能も提供されました。 Evernote Businessの立ち上げ以来、世界中の3,000以上の企業とチームが彼らを武装させ、従業員の生産性を高め、重要な情報を社内に効果的に広めています。



「私たちは一緒に長い道のりを歩んできましたが、まだ最初の段階です。」 私たちのチームは、すべてのデバイスと生活のあらゆる分野で外部デジタルメモリをさらに便利かつ効率的に使用できるよう、引き続き取り組んでいます。



Evernoteを選択してくださったすべてのユーザーに感謝します。Evernoteがサービスを改善し、友人や同僚に推奨するのを助けてくれました。 Evernoteのさらなる歴史を一緒にしたいと思っています！
