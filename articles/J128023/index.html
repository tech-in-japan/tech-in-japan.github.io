Adobe Dreamweaver CS5にコードヒントを追加します。

すべての良い一日。

スクリプト、ウェブサイト、ポータルの開発に長い間Adobe Dreamweaver CS5を使用しています...

そして、ある日私は疑問に思ったが、どうやって自分のヒントをDreamweaverに追加するのか？



いつものように、私はGooglaから始めましたが、そこでは適切なものが見つからなかったので、自分でそれを見つけ出すことにしました。





何を手に入れたいですか？


Dreamweaverでは、このような行$ cl = newを入力すると、プログラマーによってクラスのセットが表示され、 $ cl->を入力してクラスを宣言した後、このクラスのメソッドが表示されます。

そこにメソッドとクラスを追加したいだけです。



何かが長い間うまくいかない場合


長い検索の後、 C：\ Program Files \ Adob​​e \ Adob​​e Dreamweaver CS5 \ configuration \ CodeHints \ BuiltinCode \にフォルダーが見つかりました 。このフォルダーにはいくつかのファイルがありますが、 PHP_CORE.phpが必要です 。

ファイルには、編集のための許可を設定する必要があります。



その内容を確認した後、すべてが非常に明確になりました。このファイルのすべての内容を表示するのではなく、一部のみを表示します。
