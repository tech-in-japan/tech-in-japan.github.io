Visual Studio Online“ Monaco”およびWindows Azure Webサイトを使用して、ブラウザーでC＃、PHP、Node.jsコードを直接編集する





ウェブサイトを開発していますか？ Windows Azure WebサイトとVisual Studio Onlineを使用すると、いつでもどこでもこれを行うことができます。



Visual Studioは、サイト作成者にいくつかの興味深い機能を提供します。 WebサイトをVisual Studioから直接クラウドに公開したり、リモートでデプロイおよびデバッグしたりできます。



Visual Studio 2013のリリースには、すべてのWindows Azureサブスクライバーが利用できる別のエキサイティングな新機能があります。



ブラウザ内の新しいコードエディター！



これは、多くのCMSシステムが長年にわたって努力し、独自のエンジンとエディターを作成してきたことです。



仕組みを見てみましょう。



お気に入りのテクノロジーASP.NET MVC、Node JS、PHPのいずれかでWindows AzureにWebサイトを作成するか、Joomla、Drupal、WordPress、Djangoなどの既製のエンジンとフレームワークを使用できます。







Windows Azureでサイトを作成するプロセスは5分以内で完了します 。







サイトが作成された後、 構成設定で、サイトファイルにアクセスするためのアカウントを構成します。 FTP、ブラウザでのVisual Studio Onlineを使用したコードの編集、およびVisual Studioからのソリューションの展開に使用されます。







次に、Visual Studio Onlineの使用を有効にして、コードを編集します。







[ ダッシュボード ]タブに戻ります。 メニューの右側に、Visual Studio Onlineを使用してブラウザーでコードを編集する機会があります。







リンクをクリックすると、Visula Studio Onlineが表示され、ブラウザーでコードを直接作成および編集できます。 これらはすべて、IntelliSenseサポート、自動保存、コンパイルおよび実行機能を備えています。











また、プロジェクトにGITまたはTFSを設定して共同作業を行うこともできます。



Visual Studio OnlineはすべてのWindows Azureユーザーが利用でき、動的で、いつでも、どこでも、どのデバイスからでも作業でき、24時間365日オンラインでサイトを利用できます:)
