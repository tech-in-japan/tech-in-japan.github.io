新しいHexletコース：React、Ansibleなど

こんにちは、Habr！



4か月前、 私たちは新しいHexletのベータ版の立ち上げについて書きました。これはブラウザで実際に練習するプログラマ向けの教育プラットフォームです。 今後数か月で、Hakesletはベータ版からリリースされます。今日は、いくつかの新しいコースについてお話したいと思います。 ただし、最初に、サービスの更新について簡単に説明します。



  • SlackでHakesletコミュニティの公開チャットを開きました！ そこで、他のユーザーや教師と通信できます。 特定のコースやトピック用の特別なチャンネルがあり、英語を学習するためのチャンネルもあります。
  • コースの新しいレッスンのリリースに関する通知がメールで届き 、サイトの通知に表示されるようになりました。
  • このサイトでは現在、現在開発中のコースを計画しています（詳細は以下を参照）。 これらのコースには事前にサインアップできます。
  • 「 サンドボックス 」セクションには、Hakesletの他のユーザーによる無料形式の無料コースがあります。 独自の実用的なコースを作成することもできます ！
  • レッスンに合格すると、ポイントが付与され、メインページでトップユーザーの位置を確認できます。


そして今-新しいコース：



Ansible Basics







構成管理およびオーケストレーションシステムは、現代の開発者およびシステム管理者の武器の重要なツールです。 このコースの目標は、一般的な構成管理ツールであるAnsibleの使用方法を学ぶことです。 前に書いたように、AnsibleはHakesletの開発と展開に積極的に使用されています。



ReactJS







React Basicsとそれに基づくシンプルなアプリケーションの作成。 私たちはReactが大好きで、 ブラウザ開発環境が書かれています。 ところで、記事「 Unconventional React Review 」の記事翻訳を読むことをお勧めします。



PHP5の紹介







このコースは私たちにとって重要な実験です。 実際のところ、インターネット上のPHPコースの99％が言語の構文についてのみ話し、プログラミングについては話していません。 言語は単なるツールであり、現代の開発者の武器であり、通常は2つ以上の言語です。 このコースでは、このギャップを埋めて、言語を使用してコードを記述する方法に主な時間を割こうとします。 このコースは初心者向けに設計されているという事実にもかかわらず、学習プロセスで説明される概念は、多くの経験豊富な開発者にとっても有用であり、新しいものですらあります。



正規表現







すべてのプログラマーは、毎日の練習で正規表現に直面しています。 彼らは多くを怖がらせますが、正規表現は少数の単純な概念に基づいており、それらを使いこなせるように習得することができます。 このコースでは、これらの基本概念を習得するお手伝いをします。



現在、開発中のコースは次のとおりです。

  • ドム
  • SQLの基本
  • Rubyの紹介


すでにそれらを購読することができます、そして、あなたは新しいレッスンのリリースに関するメール通知を受け取ります （あなたのプロフィールのメール設定をチェックすることを忘れないでください）。
