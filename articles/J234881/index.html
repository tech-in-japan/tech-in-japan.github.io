ローマウドビチェンコへのインタビュー。 Code Jam TOP 10またはオリンピックの成績

良い一日、親愛なるhabratchiteli！



おそらく、あなたの多くはスポーツプログラミングオリンピックについて聞いたことがあるでしょう。 近年、このタイプのプログラミングには多くの競争があります。 これは、 Google Code Jam 、 Topcoder Open 、およびRussian Code Cupです。 今日、私は幸運なことに、特別なサウロンプログラムのもとで、AYcampの一部であるローマウドビチェンコと、彼らで真剣な成功を収めた人の一人とチャットできました。 彼はミンスクに住んでおり、昨年ベラルーシ州立大学を卒業し、ヤンデックスで働いています。 彼は、彼がオリンピックに向けてどのように準備しているか、そして彼らがどのように彼の人生を助けたかを語った。



私は猫の下で興味があるすべての人に尋ねます。







ローマの成功例：


  • 銀メダリストIOI 2007および2008
  • VKOSHP-2007の絶対勝者

  • ロシアAIカップ2012の受賞者
  • ロシアコードカップ2011、2012、2013のファイナリスト
  • 学生世界選手権ACM ICPC 2013の銀メダリスト
  • Challenge24 2014のファイナリスト
  • Google Code Jam 2014のファイナリスト




Q：ローマ、オリンピアードのプログラミングを始めたのはいつですか？


A：5年生でオリンピアードプログラミングを始めました。 学校には、コンピューターサイエンスを含むさまざまなサークルがありました。 私は彼に行き始め、すぐに問題を解決することを学び、オリンピックに参加し始めました。



Q：両親はあなたの選択にどのように反応しましたか。コンピューターで多くの時間を過ごしたことであなたを責めませんでしたか？


A：最初は、私の両親は普通の趣味として普通に扱っていましたが、それでも長い時間コンピューターで過ごすことは許されていませんでした。 たとえば、私の母はコンピューターで1日1時間しか費やさないように制限を設定しました。





Q：オリンピアードのキャリアはどのように始まりましたか？


A：サークルでは、IntAlプログラミング言語、そして非常に単純なアルゴリズムを学びました。 しかし、その後、先生は私が非常にうまくいっていることを見て、1年後、彼は両側の長方形の面積と残りの面積の計算が非常に簡単であるという事実から始めて、オリンピックの方向の非常に簡単な問題を解決することを提案しました。 その後、私は地域のオリンピックに行き、それが私にとって非常に興味深いことに気づき、その後、オリンピックの問題を解決し、アルゴリズムを研究し始めました。





Q：オリンピックでの成功から本当に強い喜びを感じたのはいつですか？


A：学校ではすべてが順調に進みました。最初は地域に行き、次に共和党のオリンピアードに行きました。数年後、国際オリンピアードと国際オリンピアードの訓練キャンプに行きました。 おそらく最も大きな喜びの1つは、共和党オリンピアードで学位を取得し、国際オリンピアードのトレーニングキャンプに行ったことです。 おそらく、3年目にプログラミングチャンピオンシップの決勝に達したときでしょう。



Q：過去に戻る機会があった場合、トレーニングで何か変更しますか？


A：いい質問です。 多分、私は11年生の最後の学校の国際オリンピックに十分な準備ができていなかったので、そこで銀メダルを獲得しました。 おそらく、もっと積極的に準備する価値がありましたが、さまざまな理由で準備に多くの時間を費やしませんでした。 そのため、私は銀メダルを獲得しましたが、十分な力とスキルはありましたが、金メダルのようなものです。 おそらく、これは何らかの形で変更された可能性がありますが、全体的に私は自分自身に満足しています。







IOI-2008で2番目に残ったローマ




Q：つまり、常に最高の結果のみを目指し、最初の場所のみを目指しますか？


A：そうではありません：私は自分の強みを多かれ少なかれ客観的に評価し、理解しようとします：ある状況では勝利に頼ることができますし、他の状況では、どこで、例えば、いくつかの一般的なオンライントレーニング。



Q：現在どのように改善していますか？


A：現在のレベルでは、私はもうトレーニングをしていません。 私はすでに大学を卒業しているので、仕事をしていて、いわば、自分が持っているものを持っている競技会に参加しています。



Q：そして、どの時点でオリンピアードのプログラミングに関して最も「ポンプ」したのですか？ それはキャンプ、教科書、または他の何かでしたか？


A：それはコーメンの教科書でした。 8年生または9年生になると、私は絶対に覚えていません。私は本当にやりたかったのですが、国際オリンピックの合宿に行くことができませんでした。 そして、私はスポーツの怒りを感じました：「くそ、なぜ、私はできる、私はそれが必要だ！」 その後、両親はコーメンの本を購入しました。これは、アルゴリズムに関する非常に健全な古典的な教科書であり、夏の間、わざと本を読み始めました。





同じ本




Q：オリンピアードのキャリアを始める価値はありますか？


A：オリンピアードプログラミングを本当に始めたい場合、教育プロセスは反復的である必要があります。問題を解決しようとしても成功しません-上級の仲間や教師から解決方法を学びます。 何か新しいことを学び、新しい問題を解決しようとしますが、理解できないなどなど...経験から、教科書に座ってアルゴリズムを学習し始めてから問題を解決しようとすると、ほとんど何もうまくいかないでしょう。



Q：学年のオリンピックに向けてどのように準備しましたか？


A：学年...学年では、レッスンに加えて、毎日いくつかの問題を解決しようとしました。 1日数時間かかりました。 合計で、週末とともに週に約20〜30時間。 幸いなことに、私たちの学校には優れたディレクターがいて、学校にそのような生徒がいて良かったので、ほぼ一年中クラスに参加することができました。 （笑）これにより、トレーニングにもっと時間を割くことができました。 訪問に問題はありませんでしたが、金メダルで学校を卒業しました。





Q：どのようにリラックスしましたか？


A：私は大好きで、今でも学校で友達とボードゲームをするのが大好きです。たとえば、植民地主義者と遊ぶのが大好きでした。 さらに、彼らはマンチカンと他の古典的なもので遊んだ。 彼はまた、サッカーとバレーボールをし、夕方にはよくスタジアムに行きました。



ローマはマジック：ザ・ギャザリングを演じます




Q：学生時代には？


A：学生時代、すべては非常に複雑です。なぜなら、私は本当にワールドカップの決勝に行きたいと思っていたからです。 1日2回5時間のコンテストを開催する日がありました。 2年目に決勝に行きたいと思ったとき、私たちは週40〜50時間トレーニングし、それは報われましたが、何らかの理由でまだ行けませんでしたが、次のことに成功しました。



Q：私が覚えている限りでは、3年目にはあまり上手く話せませんでした。


A：はい、3年目にタブレットの真ん中のどこかに行きました。 しかし、それは興奮と関連していたと思います。 私たちは、競技を通してこの感覚に対処することができませんでした。 私の考えはすべて次のようなものでした。 すでに誰もがタスクに合格しており、誰もが既に2〜3を持っていますが、何もうまくいきません！ パニック、パニック、パニック-何をすべきか？」これは無意識ですが、集中することが非常に困難になります。



Q：あなたはほんの1週間前にGoogle Cod Jamでロサンゼルスにいましたが、結果に満足していますか？ そしてCISは全体としてどのように行動したのでしょうか？


A：CISは成績が良く、多くの人がテーブルのトップにいます。 10位になったのは、パズルを思い付かなかったからではなく、小さなイデオロギーの誤りのためでした。 正直なところ、解決方法を思いついたことに非常に失望していますが、間違ったアプローチを取りました。

Q：そして、どのツアーが最も困難で、問題を解決するためのアルゴリズムはありますか？


A：はい、おそらくありませんでした。 過度のストレスなしにツアーを解決しようとしています。ある時点で次のラウンドに到達するための十分な努力がなければ、それでいいのです。 ソリューションアルゴリズムについてはどうですか。私は複雑で圧倒的なタスクを追いかけようとはしませんが、よりアクセスしやすいタスクでより自信を持ってポイントを獲得するよう努めています。



Q： さまざまな予選大会に参加しようとしていますが、可能であれば決勝に進出します。 しかし、あなたのオリンピックのキャリアは仕事を妨げますか？


A：いいえ、干渉しません。 Yandexには、以前と現在の両方のオリンピアドニクがたくさんいるので、「競技会に行ったり、座ったり、職場でコードを書いたりしないでください」（笑）





コードジャムの開始前


Q：ヤンデックスとABBYYは素晴らしいキャンプを開催しました。ここは本当にクールです。 あなたは先生としてここに来ました、あなたはそれが好きですか、そして何があなたを内部的に動かしますか？ Yandexでの仕事がなかったら、子供たちに教えることに時間を費やすつもりはありますか？


A：私は子供たちに教えるのが本当に好きです。これは私の人生で好きな活動の1つです。 残念なことに、私たちの国では、教師が非常に高い給料を支払われていないことがありますので、私は完全に教えることに専念する準備ができていません。 プログラミングも好きなので、大企業で働き、面白いプロジェクトをして、良い給料をもらいたいです。 もちろん、私は子供たちに教えることも大好きなので、このようなサマーキャンプに参加できてうれしいです。 トレーニングキャンプやキャンプでは、子供たちに怖くないことを納得させ、質問することを恐れる必要はありません。最悪の事態はすぐには答えられないが、1、2日で答えられるからです。 一般に、多くの人が私にVkontakteを書いたり、この問題やその問題を解決する方法を電子メールで送ってくれて、喜んで答えます。



Q：誰かに平等になりたいですか？


A：全体として、はい。 Ivan MetelskyやGena Korotkevichのようなベラルーシ出身の人たちがいます。彼らはさまざまな競技で活躍しています。 時々私は彼らと相談し、戦略的、戦術的思考の観点から例を挙げます。 ロシアには素晴らしい人が一人います-これはセリョーザ・コペリオビッチです。



Q：主にどの言語でプログラムしていますか？


A：C ++で問題を解決し、Pythonでいくつかの補助プログラムを作成します。



Q：ヤンデックスにどのように乗り込み、オリンピックはあなたを助けましたか？


A：ヤンデックスがミンスクにオフィスを開設することを決めたとき、これは私の3年目の直後に起こりました。彼らは小さなチームで私たちのところに来てインタビューを始めました。 したがって、私とオリンピックに積極的に関与していた12人以上の人々がインタビューに合格し、私たちに非常に満足しており、オリンピックは主にミンスク事務所の最初のスタッフだったと言えます。 それ以来、もちろん、開発者だけでなく、採用担当者や他の専門家でさえ、さまざまな他の開発者で補充されています。 しかし、オフィスの始まりはオリンピック競技大会のチームによって正確に設定されたため、オリンピック競技大会は私を助けたと言えるでしょう。







Q：あなたのようなオリンピックと、普通の開発者の間で、不快感や意見の相違はありますか？


A：実際にはありますが、重要ではありません。 他のアクティビティと同様に、ある開発方法に投票する人もいれば、他の人もいます。 オリンピアディアンは、問題を解決するスピードと表現力によって特徴付けられます。 つまり、実行する必要のあるタスクに出くわすと、オリンピアードはオリンピアードのすべてでそれを行うため、オリンピアードはより早くそれを開始する可能性が高くなります。 ただし、これは常に良いとは限りません。 時には、考える前に、考え、計画し、設計する必要があります。 これらのオリンピアドニキのほとんどは、方法がわかりません。



Q：Yandexでの仕事と、今後のIT業界でのキャリアについてどう思いますか？


A：今、私はミンスクで行われている小さなプロジェクトをリードしています。 原則として、コードを書くプロセスと小さなチームの管理、小さなタスクを全員と何らかの計画に分配するプロセスを組み合わせるのが好きですが、残念なことに、最近ではタスクと計画の分配プロセスにますます多くなり、ますます少なくなっていますコードを書く過程で、私はこの傾向が好きではありません。コードを書くことができて、それが好きで、多かれ少なかれうまくやって、管理や管理活動のためにそれを残したくないからです。 将来的には、Yandexフレームワーク内で十分に小さなプロジェクトを管理して、残りの従業員の仕事に追いつき、自分でコードを作成できるようになることを願っています。





職場のローマ


Q：現在どのようなプロジェクトをしていますか？


A：カードの背景を改善しています。 他のサービスで使用されるもの：渋滞、ルーティング、パノラマなど。 基本的なマップの背景、つまり 家、通りなどの基本的な写真。 最近、この基板の可読性を改善することに取り組んでいます。 私たちのマネージャーは、より視覚的で理解しやすく、使いやすくオリエンテーションに便利な方法を考え出し、これを実装しています。 特に、夏の初めに「地図上のランドマーク」と呼ばれるプロジェクトを立ち上げ、現在はYandex.Mapsでいくつかの建物を見ることができます。「しかし、これはそのような通りの25番の建物です」コンサートホール、映画館、または薬局であること。



Q：Habrを読んでいますか？


A：はい、読んでいます。 （笑顔）



Q：何を読んで、そこで何が役に立つと思いますか？


A：私はVkontakteフィードでHabrahabrを読みました。 私はこのグループに登録しており、そこに出てくる見出しを見て、興味深い記事を開いています。 通常、機械学習、Python、およびすべてのほぼピトナイトに関するニュースは、私にとって興味深いものです。



Q：このボーナスの質問を続けてください：ポーカーに対するあなたの態度と、このゲームがどのように脳を「ポンピング」するか、カリキュラムに含めることは価値がありますか、これから何かを学ぶことができますか？


A：面白い質問です。 私の態度は一般的にコンピューターゲームと同じです。 この職業はまず第一に本人次第です。1ダイムを費やすことなく、自分の楽しみのためにポーカーやオンラインゲームを安全にプレイできます。または、一部のチェッカーやチェスで財産を失うこともできます。 もちろん、ゲームはギャンブルが多い、またはギャンブルが少ないですが、それでも中毒の程度は人によって異なります。 ポーカーに関しては、このゲームには数学的基盤があります。 勉強するように彼女に勧めたわけではありませんが、ポーカーをプレーし、そこで何が起こっているのかを理解することは非常に価値のある職業でした。



Q： ローマ、インタビュー、今後の成功に感謝し、Habréでこのインタビューを読んでください！


A：わかりました、ありがとう。 何か質問したい場合は、「質問」-連絡してください（笑）。
