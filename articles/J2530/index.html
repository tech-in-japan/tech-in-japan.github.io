Secunia Software Inspectorはコンピューターに穴を見つけます

Secunia- Software Inspectorのオンラインユーティリティを使用して、ローカルコンピューターにどのパッチが欠落しているかを確認できます。



このユーティリティは、javaを使用して、Windowsおよびその他のMicrosoft製品に最新のセキュリティ更新プログラムがインストールされているかどうか、およびFirefox、Opera、Skype、Adobe Reader、QuickTime、iTunes、Macromedia Flashなどのプログラムのバージョンがローカルコンピューターにインストールされているかどうかを確認します。プレーヤー、SunのJava JRE、Winamp など 。



Secunia Software Inspectorは、標準チェックと高度なチェックの両方を実行できます（システム検査を徹底。プログラムファイルのような標準タイプだけでなく、すべてのフォルダーがチェックされます）。 このユーティリティを使用するには、Javaマシンとブラウザの1つ（Internet Explorer（6.x以降）、Opera 9.xまたはFirefox（1.5.x以降））が必要です。 「穴」の存在を確認するには、Windowsの最新バージョン（2000、XP、および2003）のみを使用できます。
