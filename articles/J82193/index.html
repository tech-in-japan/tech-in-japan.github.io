Snow Leopard 64ビット

おそらく誰もがSnow Leopardでの64ビットアーキテクチャへの移行について聞いたことがあるでしょう。 しかし、古いカーネル拡張機能との互換性のために、デフォルトでSLが32ビットモードでロードされることを誰もが知っているわけではありません。 ただし、SLは依然としてデフォルトで64ビットカーネルをロードするように強制できるため、パフォーマンスがいくらか向上します。 Leopard / Snow Leopard 32ビット/ Snow Leopard 64ビットのパフォーマンスの比較はこちらをご覧ください 。



デフォルトを64ビットカーネルにロードするには、ターミナルを開き、次のコマンドを実行する必要があります。

sudo nvram boot-args = "arch = x86_64"

32ビットカーネルに戻るには：

sudo nvram boot-args = ""



そして今、パフォーマンスに関して最も興味深いのは、64ビットカーネルで作業しているとき、ビデオを見るときのCPU消費がはるかに少ないことです-たとえば、VLCで失われた映画のシリーズを見るとき、CPU消費は〜19％から〜8％に低下しました。 HD 720pを表示する場合-60％〜28％。 このプロセッサの節約は、Macbookユーザーにとって特に役立ちます-CPU負荷が少なく、バッテリー寿命が長くなります。



UPD：iTunesもCPUの消費を抑え始めました-32ビットモードでは10％ではなく3％
