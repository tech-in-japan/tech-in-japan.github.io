Google I / O 1日目：まとめ

私はGoogle I / Oを訪れることができました。 この投稿は、提出された内容の要約です。 コンテンツを1つの文にまとめると、「Androidはどこにでもあります」というようなものになります。





Androidロゴ「アイスクリームサンドイッチ」[出典： ガーディアン ]

  • Androidニュース

    • Android Honeycomb 3.1のリリースを発表しました。 [ リンク ]
    • Androidの次のバージョンは「Ice Cream Sandwich」と呼ばれ、Android Gingerbread（電話）とAndroid Honeycomb（タブレット）を組み合わせます。 したがって、Androidの1つのバージョンが携帯電話とタブレットの両方で実行されます。 [ リンク ]
    • Androidでは、まもなくUSBホストとして使用できるようになり、カメラ、キーボードなどを操作できるようになります。
    • AndroidはアプリケーションとともにGoogle TVに登場します。 [ リンク ]
  • クラウドコンテンツ：

    • 現在：アプリケーションと本
    • 新機能： Googleミュージックベータ版 。 [ リンク ]

      • アプリケーションからダウンロードする（WindowsおよびMac）
      • 最大20,000曲。
      • ベータ中は無料です。
      • WebまたはAndroidアプリケーションを介して音楽を聴く。
      • オフライン：最近再生した音楽を自動的にキャッシュします。 オフラインで使用するためのアーティスト、アルバム、またはプレイリストの手動指定。
    • 新機能：レンタルを含む映画。 オフライン表示用に「遅延」（キャッシュ）できます。 [ リンク1 、 リンク2 ]
  • ユーザーに対する更新の保証：更新を提供するための基本原則を開発するグループが形成されました。 [ リンク ]

    • グループのメンバー：AT＆T、Verizon、HTC、Samsung、Motorolaなど。
    • 機器が許可する場合、18か月間更新を保証します（潜在的な抜け穴！）。 比較のために：Appleは毎年新しいiPhoneを発売しますが、前のモデルはもう1年販売しています。 この方法で、少なくとも24か月間更新を取得できます。
  • Androidアクセサリ[ リンク ]：

    • Android上のデバイスを接続するためのオープン（無料）標準。
    • Androidデバイスはアクセサリを自動的に認識し、対応するアプリケーションを自動的にダウンロードできます。
    • ケーブルを介した接続。 すぐに：Bluetoothのサポート。
  • Google @ Home：ホームオートメーション。 [ リンク ]

    • オプション：ビデオストリーミングに十分な帯域幅を備えた低コスト（低電力）ワイヤレス通信の標準。
    • スピーカーを内蔵したAndroid @ Homeでの音楽のストリーミング。
    • Googleは、Lightning Scienceと提携して、Androidデバイスでワイヤレス制御できるLED電球を開発しました。 [ リンク ]
  • Honeycombを備えた新しいタブレット：Samsung 10.1タブ（3.0で動作）。 [ リンク ]
