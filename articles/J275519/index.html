ブックマークバーのアイコンとアセンブリVivaldi 1.0.375.3のすべての塊

みなさんこんにちは！



さまざまな理由から、先週、毎週アセンブリをリリースすることができませんでした。 一方、ブラウザにかなりの数の変更を加えて、テスト用に提供したいと思います。 あなたはすぐにMac OSXユーザーを混乱させる必要があります：あなたのバージョンは今日のビルドにありません-最後の瞬間にポップアップする自動更新のバグ。 今週リリースされる可能性が高い次のビルドのために修正を試みます。



はっきりと見える変化のうち、ブックマークバーのリストに表示されるアイコンは次のとおりです。









並べ替えオプションなど、まだ多くの作業が必要です。 しかし、最初のステップはすでにとられています。



Expressパネルの背景画像も追加しました。









また、ブログやフォーラムで報告した修正済みエラーを簡単にリストすることもできます。 これらは以前はアドレスバーで機能していなかったJSアドレスであり、Ctrl + Shift +スクロールを使用するときにページの代わりにインターフェースをスケーリングし、パスワード保存ダイアログを常にポップアップします。 また、ホイールでタブをスクロールする（カーソルがタブバー上にあるとき）、グローバルページスケール、設定の多くの新しいオプションなどの新しい機能を追加しました。



変更の完全なリストは、記事の最後にあります。



それだけです 以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac 10.7+
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
