AlibabaのAnt Financial子会社は、別のラウンドの資金調達後に600億ドルと評価される可能性がある

アリババによって別の事業として設立されたアリ・ファイナンシャルは、31億ドルを得るために投資家と協議中であるとウォール・ストリート・ジャーナルは書いている 。 専門家は、同社が2017年に中国証券取引所にIPOを行うと考えています。







中国の公営企業Alibaba Groupは、Alibaba.comが登録された1999年以来、電子商取引市場で事業を展開しています。 会社の構造には、主な資産に加えて、Alibaba.com B2B取引プラットフォーム-Alibaba Pictures中国映画会社、AliExpress.comオンラインストア、Taobao.comオークション、Alibaba Cloud Computing、Alipayオンライン電子決済プラットフォームが含まれます。 。



2015年、Alipayはブランドを変更し、Ant Financialとして知られるようになりました。 同社自体がAlipayサービスを管理しています。ユーザー数、投資サービスYu'e Bao、オンライン信用格付けシステムSesame Credit、分割払いプログラム、デジタルバンクMYBankの点でPayPalの3倍です 。 2015年7月の時点で、同社は650億ドルから160万の起業家および企業のマイクロローンを発行しました。



2015年7月、Ant Financial はラウンドAの資金調達を終了しました 。 同社は、中国最大の年金基金である国家社会保障基金（NSSF）を含めることに投資しています。 ラウンドの結果によると、同社は400億ドルと評価されました。



新しいラウンドの結果に続いて、Ant Financialが31億ドルを集めることに成功した場合、その見積もりは500〜600億ドルになります。
