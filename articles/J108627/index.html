2つのフラッシュドライブとUSBハブ

私たちは長い間、マーケティング記事であなたを拷問しましたが、最後に、人間の観点からいくつかのデバイスを紹介する時です。 今日では、2つのフラッシュドライブ（8Gbおよび4Gb）とUSBハブになります。







長い間猫を UFO の尾に触って触らせずに写真や説明を表示します。





多くの人にとって、Apacerはこれらのフラッシュデバイスに精通しています。







しかし、長年にわたって、他の多くのモデルがリリースされています。 今日はそのうちの2つを紹介します。 なぜそんなに少ないのか聞いてみませんか？ 線量情報へ。 これらが好きなら、私たちは続けて、すべてのモデルラインについて徐々にあなたに話します。







だから、最初のフラッシュドライブ。



Apacer AH522（8Gb）













フラッシュドライブはプラスチックと金属でできています。 重量はわずか18 gです。寸法はごく普通の59.1 x 21 x 7.2 mmです。 ここのUSBコネクタは内部にあり、特別なレバーを使用して引き出します。







このモデルには、容量が8 GBと16 GBの2つのバージョンがあります。 このデバイスは、それぞれ30 Mb / sおよび12 Mb / sの速度でデータの読み取りおよび書き込みが可能で、青色のLEDステータスインジケーターも装備しています。







ちなみに、製品の所有者には、ドライブに保存されているパスワードで保護されたファイルだけでなく、記録された情報の圧縮による容量の実質的に5倍の増加のために、メーカーの公式WebサイトからApacer Compression Explorer（ACE）独自のソフトウェアをダウンロードする機会が与えられます



そしてもちろん、私はテスト結果をもたらします。 悲しいかな、彼らはデバイスの説明に記載されているよりもわずかに低いです。

8Gb-1000r。

16Gb-1800こする



Apacer AH128（8Gb）













Apacer Handy Steno AH128は、男性用のフラッシュドライブとして位置付けられています。 アクアマリンだけでなく、赤と銀で入手できます。 ケースは黒い縁取りで囲まれ、ドライブには亜鉛合金のバックル付きの革製ストラップも装備されています。







これは男性用のモダンでスタイリッシュなアクセサリーで、水やほこりから保護されています。

Handy Stenoシリーズは、フラッシュドライブチップとボードを1つのユニットに組み合わせ、湿気、ほこり、機械的損傷からデバイスを保護する、Chip-on-Board（COB）テクノロジーを使用して生成されたデータを保存するハイテクデバイスに代表されます。 格納式コネクタにより、デバイスはエレガントで使いやすくなります。

そして最後に、USBハブ。



Apacer PH250









USB 2.0ハブPH250は、コンピューターまたはストレージデバイス間のデータ交換用に設計されており、複雑なネットワーク接続をインストールする必要はありません。 高速USB 2.0およびUSB 1.1をサポートする3台のデバイスを同時にUSBコネクタに接続できます。



PH250の主な機能：



*インターフェイス：高速USB 2.0およびUSB 1.1

*コネクター：3つのコネクター（USBタイプA）と1つのコネクター（USB mini B）

*データ転送速度：最大480 Mb / s

*対応OS：Windows 98SE / ME / XP / 2000 / Vista、Mac OS 10.4 +

*寸法：70×20×20 mm

*重量：40g
