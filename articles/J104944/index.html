Drupalogies 1年！





昨年の9月26日（ 開発者の誕生日）に、プロジェクト「Drupalogy.ru-Drupalで作成されたサイトの展示」が開始されました。





最初のバージョンは訪問数が少なく、追加されたサイトは評価されましたが、サイトはほとんどコメントしませんでした。 カテゴリはほとんどありませんでしたが、制限はありませんでした。





サイトの最初のバージョン



しばらくして、多くの修正が考慮され、リソースが変更されました。さらに、彼は最終的にロゴを取得しました。 Drupalの機能を説明する最も人気のある画像が、ロゴの基礎となっています。







人気のある検索ディレクトリでのサイトランキングの表示、ブラウザサポートなどの便利な機能が追加されました。 追加されたサイトの新しいデータにより、設計が変更されました。





サイトの2番目のバージョン



今年の初めに、2番目のプロジェクトが登場しました-Drupalogia pdfマガジンは、新参者と知識を共有することにした独立した開発者によって作成されました。 ゼロリリースの後、プロジェクトが始まりました。人々は記事やレビューを書く時間がありませんでした。 熱意は尽きていませんが。



少し前、私はDebian初心者ユーザーガイドの執筆をほぼ終えました。これには、DebianでのシステムとLAMPのインストールと設定といういくつかの重要な章が含まれていました。



今年何が待っていますか：





Drupalogy.ruサイトの展示





展示会に「有用性」サイトを追加するための多くの提案を受け取り、いくつかを選択しました。

  • 有料または無料のテンプレートを使用するサイトへの「テンプレート」マーカーのインストール（私は、このオファーで互いに独立した人々から約20通の手紙を受け取りました）;
  • 「エンジン」に取って代わったサイトの「Drupalではなくなった」セクション。




Drupalogy Magazine





  • 彼のサポートを続けます。 私は認めることを恥ずかしく思いますが、計画されていたすべてが適切に行き詰っていました。 ユニットには多くの著者がいます。 できるだけ早い時期に、ジャーナルの出版が再開されます。 現在、サイトは閉鎖されていますが、雑誌はダウンロード可能です。 現時点では、pdfは25,000回以上ダウンロードされています！




Debian GNU / Linux Starter Webサーバーガイド





  • 材料が補充されます。




労働交換



  • Drupal開発者のための労働交換。 オープニングは10月中旬に予定されています。
