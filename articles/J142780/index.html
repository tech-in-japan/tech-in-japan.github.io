Windows 8 HackathonおよびMay Application Excellenceラボ

友だち、Windowsストアの最初のアプリケーションを選択する5月のラボを発表するときです。







Windows 8. Application Excellenceラボ



AEL Labとは何ですか？ これは、Windowsストアで最初の製品の1つを獲得するユニークな機会です。 おそらく既にご存知のように 、Windows 8のアプリケーションストアは、ロシアを含む新しい市場にすぐに拡大します 。 ただし、だれでもそこに着くことができるという意味ではありません（少なくとも最初は）。



私たちは、Windowsストアに本当に素晴らしいアプリが欲しいです-そして、あなたがまさにそれをするのを助けたいです。 これを行うために、私たちは実験室での仕事を整理し、その中であなたのアプリケーション（ロシア語で！）



実験室での作業中にできること：

  • タブレットを含むさまざまなシナリオでアプリケーションをテストします。
  • Metroスタイルに従ってアプリケーションの設計とレイアウトを改善する方法を理解します。
  • メニューのレイアウトと使用、レイアウト、タイルの「活性化」に関する推奨事項を取得します。
  • 契約の実装例を見てみましょう。 検索、共有、設定。
  • アプリケーションの収益化戦略について話し合う。
  • 技術的な質問への回答を取得します。




実験室での作業の結果によると、最高のアプリケーションはWindowsストアにアクセスできます。



5月28日から6月1日まで、モスクワでラボが開催されます（原則としてリモートシナリオも可能です）。



入り口で

Windows App Certification Kit（Visual StudioまたはWindows SDKを使用してインストール）を使用して内部認証に合格した、Windows 8用の動作するMetroスタイルアプリケーションが必要です。 アプリケーションはすべてのテストに完全に合格する必要はありませんが、テストが必要です。



出口で

アプリケーションを改善する方法に関する推奨事項を含むレポートを受け取ります。 最高のアプリケーションの開発者は、最初にWindowsストアにアクセスできます！



ラボに参加する方法は？



上記の説明から、これは非常に限られた数の幸運な参加者を含む非常にコンパクトな形式であることを既に理解しているはずです。 したがって、実験室での作業に参加したい人の間で、追加の選択を行います。

  • 5月13日より前に、 「 [W8 AEL] Name of your application 」という見出しの付いた予備のアプリケーションを電子メールrush4apps@microsoft.comで送信する必要があります。 レターで、アプリケーションを説明します。アプリケーションが解決するタスク、アプリケーションの対象者、およびコミュニケーションの連絡先。
  • 5月18日まで 、同じ住所に同じ見出しで次の文字を含む手紙を送ります。

    • アプリケーションの説明
    • アプリケーションのメイン画面のスクリーンショット
    • WACKによるテスト結果
  • 5月22日まで、マイクロソフトの専門家は、著者が研究室の仕事への招待を受け取る最適なアプリケーションを選択します。




アプリケーションとアプリケーションデータをより早く送信するほど、実験室での作業に入る可能性が高くなります。



Metroスタイルアプリの原則とデザインパターンを順守すればするほど、チャンスは大きくなります。



アプリケーションまたはゲームがより賢明で有用であるほど、チャンスは高くなります。



ハッカソン



アプリケーションの準備を支援するために、 Windows 8用の別の開発ハッカソンを開催することにしました。 Windows 8 Consumer Previewがインストールされ、開発ツール（ Visual Studio 11ベータ版 ）が搭載されたアイデア、ブランク、ラップトップでハッカソンに来てください。



ハッカソンとは何ですか？



Hackathonは、会社の他のマイクロソフトの開発者や専門家の開発に1日を費やす機会であり、何をすべきか、他に注意を払う価値があることを教えてくれます。 ハッカソン-ラボではなく、ステップバイステップのガイドではありません。 これは、実際のプロジェクトの実際のコーディングです。 事前に準備を整え、アイデアを蓄え、一人で、友人や同僚と一緒にプロジェクトに取り組みましょう。 SDKのドキュメントと例を必ず確認してください。



はい、はい！ あなたのアイデアを実現するために来てください！



最高のアプリケーションは、5月末に特別な実験室作業を行い（上記を参照）、Windowsストアに最初に行くことができます！



ハッカソンへの行き方



ハッカソンは、5月5日に10から17にかけて、クリラツコエ（モスクワ）のオフィスで開催されます。
