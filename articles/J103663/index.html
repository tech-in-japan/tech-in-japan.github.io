VMware View 4.5の発表

最近、VWware（製品マーケティング）のディレクターであるRay Mallempati、Raj MallempatiがVMware View 4.5のリリースを発表しました。 私は発表のテキスト全体を見て、ここに私が最も興味を持っているものがあります：



1.「ローカルモードでクライアントを表示-デスクトップ仮想化のための業界初の統合オフラインおよびサーバーホスト型ソリューション。 エンドユーザーは、仮想デスクトップをデータセンターからチェックアウトし、ネットワークに接続せずに効率的かつ安全に作業できます。 さらに、チェックアウトされた仮想デスクトップは、他のソリューションとは異なり、既存および新規のすべてのPCで実行されます。 さらに、チェックアウトされた仮想マシンは完全に暗号化され、ポリシーが関連付けられています。 これにより、組織はBYOPCまたはEOIT（従業員所有IT）のユースケース向けにデスクトップ仮想化を迅速に展開できます。



最後に、仮想ワークステーションのコピーを仮想サーバーからラップトップにダウンロードし、出張に出かけ、エンタープライズネットワークに接続しなくても静かに作業できることを待ちました。 そして、戻ったら、変更を同期します。 これにより、モバイル従業員であっても、VDIソリューションを優先して物理デスクトップを完全に放棄できます。



2. Windows 7の完全サポート-View 4.5およびThinApp 4.6でのWindows 7の完全サポートにより、組織は半分のコストと時間でWindows 7に移行できます。 お客様のWindows 7への移行をさらに支援するために、ThinApp 4.6で導入された重要な新機能は、IE6を仮想化してWindows 7で実行できることです。IE6ベースのアプリケーションの移行はこれ以上簡単ではありません。



さて、追加の言葉なしですべてが明確です。 現在、Windows 7はほとんどのお客様にとって企業標準になりつつあります。 したがって、VMwareのVDIでのこのOSの公式サポートは素晴らしいニュースです。



3.ロールベースの管理-ITタスクを適切な管理者に配布できます。



もちろん、役割の委任は良いことです。 これがどのように行われるかを見てみましょう。 これまでのところ、VMware Viewは、たとえば、非常に広範囲にわたって権限を委任できるQuest vWorkspaceからはほど遠いものです。



一般的に、待つ=）
