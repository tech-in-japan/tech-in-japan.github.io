HP Jornada 690

Toshiba Libretto 50ctについての記事に触発され、私は自分自身を非常に小さく、安価で、長時間プレイし、Linuxを使用することにしました。 その結果、選択はHP Jornada 690で決まりました。
  • プロセッサー：Hitachi 7709（SH3）133 MHz
  • 画面：カラーTFT（実際はSTNに近い）640 x 240ピクセル64kカラー
  • バッテリー、電源：リチウムイオンバッテリー、約8時間動作（バッテリーは古いため、4〜5時間保持）
  • 拡張スロット：PCカードタイプII（16ビット）、コンパクトフラッシュタイプI
  • 通信：IrDA、COMポート（ストラップなし）、モデム

Linuxをインストールする



デフォルトのWinCE 2.11で、1日遊んで、Linuxをインストールするために2GB CFカードを購入しました。 すべてが非常にシンプルであることが判明しました（http://jlime.com/）：
  • カードをカードリーダーに挿入し、
  • FAT16（26mb）、ext2（1800mb）、swap（残りは約150mb）の3つのセクションに分かれています
  • FAT16でブートローダー、構成、zImageをロードし、構成を少し調整し、root = / dev / hda2をroot = / dev / sda2に変更しました。それ以外はブート中にカーネルパニック
  • ext2のセクションでは、jlime.comからダウンロードしたアーカイブのすべてを書き直しました
  • カードをデバイスに挿入し、デバイスのStorageCard2に移動し、shlo.exeを起動し、ライセンスに同意し、すべての情報が削除されること（および何かを削除するものは作成しませんでした）がすべて読み込まれるのを待ちます


2つのグラフィックシェルが与えられ、iceWMを選択してお楽しみください。 ボードにはすでにいくつかのプログラムがあります：AbiWord、GIMP1、いくつかのゲーム、ファイルとアーカイブのマネージャー、2つのブラウザー（DilloとKonquerror）、メーラーといくつかのコンソールアプリケーション。

結論として



多くの人から、なぜこんなデバイスが必要なのかと聞かれました。なぜなら、それは誰にとっても古くて役に立たないからです。 これはおそらくテクノ-ランドフェチかそのようなものです。 しかし、結果として、3000ルーブル（電源と2Gbカードを含まない）で約5時間動作する非常に機能的なキーボードの小さなデバイスが手に入りました。

これまでのところ、モデム、comポート、赤外線ポートについては何も言えません。 PCMCIAイーサネット/ wifiカードを購入する予定で、それからもっと面白くなるでしょう。



IRポート （追加）



どちらの手も、IRが本来どおりに機能しないものではありません。 irda ircommモジュールがカーネルに読み込まれ、ircomm0があります...デバイスのリストにNがあります。できる限りのことをすべて試して、irdadumpを起動しました（赤外線ポートに流れることを教えてくれるユーライトを理解しているため）-電話から送信し、テレビからいくつかのリモコンを試しましたが、結果はありません： 「0個のバッチが受け入れられました。」

WinCEでは、IRコートを介してダイヤルアップを構成しようとしましたが、結果もありません。電話はサイレントです。 電話の赤外線ポートが応答したのは、同期のために理解したように、WinCEでプログラムを起動したときだけでしたが、電話は彼らが望んでいたものを理解しませんでした。 誰かが何かを言うことができる場合-私に教えてください。
