Windows Camp Lab Lab Recordsのお知らせ//開発者向けのラボと便利なリンク





Microsoftの専門家が新しいWindows 10およびユニバーサルアプリケーションについて語ったWindows Camp 2015イベントは、実験室作業の形式で価値のある継続を受けました。 参加者は、実際にユニバーサルアプリケーションを開発する基本的な原則を知り、直接イベントに参加するか、オンラインで接続する機会を得ました。



そして、 Windows Camp // Labsイベントからのエントリの公開についてお知らせできることを非常に嬉しく思います。







研究室での作業の記録はこちらから入手できます 。



ラボの作業（コードサンプル、ロシア語と英語の説明）はリポジトリgithub.com/evangelism/WinDevWorkshopにあることを思い出させてください。



指示の完全なセットは、 PDFブックとして入手できます。



Windows Camp 2015スピーカーによるスピーチの録音は、 チャンネル9にあります。



Windows 10プラットフォームの開発の特定の側面に興味がある場合は、 github.com / microsoft / windows-universal- samplesリポジトリまたはIoTスクリプト用のgithub.com/ms-iot/samplesの公式例を参照してください。



github.com/evangelism/HackSamplesリポジトリで、ユニバーサルアプリケーションの機能を示す興味深いプロジェクトを見つけることもできます。



Windows 10でのさらなる開発作業をお楽しみください！ そして良いコード！
