Visual Studio 2013のソフトウェアライフサイクルを開発、テスト、および管理するための一連の24のラボ

Visual Studioを使用する多くの開発者およびテスターは、定期的に更新される仮想マシンでBrian Keller（ http://blogs.msdn.com/b/briankel/ ）を知っています。この仮想マシンには、Visual Studioの最新バージョンと、開発ツール機能。 この仮想マシンには、開発ツールの機能を評価したり、以前は知らなかった、または使用しなかった新機能の使用方法を学習するのに役立つ実験室のセットaka.ms/VS13ALMVMが付属しています。



ラボは英語であり、この一連のラボと仮想マシンについて話すたびに、これらのラボをロシア語で利用できるようにするよう求められました。



すでにhabrで書いた4つのラボについて： habrahabr.ru/company/microsoft/blog/235157 本日、 アーリマンのおかげで、これらの実験室の作品をローカライズするプロジェクトを成功裏に完了したことをお知らせいたします。 Meet-Visual Studio 2013の効果的なソフトウェア開発、テスト、共同開発、ライフサイクル管理のトピックをカバーする24のローカライズされたラボの完全なセット ！



Visual Studio 2013を自分でダウンロードしてインストールするか、Brian Kellerがブログエントリaka.ms/VS13ALMVMからダウンロードする手順に従って準備した仮想マシンを使用できます。 これはHyper-Vベースの仮想マシンであり、 Windows Server 2008 x64、Windows Server 2008 R2 SP1、Windows 8（プロセッサーがSLATをサポートしている場合）、Windows 8.1、Windows Server 2012、およびWindows Server 2012 R2でそのまま使用できます 。 また、仮想マシンを起動するたびに、その時間は2013年7月9日に設定されるため、ラボ作業を便利に実行できるようになります。

仮想マシンには何が含まれていますか？

  • Microsoft Windows Server 2012標準評価
  • Microsoft Visual Studio Ultimate 2013
  • Microsoft Visual Studio Team Foundation Server 2013
  • Microsoft Office Professional Plus 2013（Word、PowerPoint、Excel、Outlook）
  • Microsoft Visio Professional 2013
  • Visual Studio 2013のMicrosoftリリース管理
  • Microsoft SQL Server 2012スタンダード
  • 24のラボのユーザーデータの例
