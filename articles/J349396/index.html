3CX用のYeastar Neogate TGXXXシリーズVoIP GSMゲートウェイの構成



このマニュアルでは、Yeastar NeoGate TG100 / TG200 / TG400 / TG800 SIP GSMゲートウェイを3CX IP-PBXに接続する方法について説明します。



Yeastar Neogateゲートウェイは、クライアントとしてSIPサーバーに接続できます。 サーバーに登録するだけでなく、組み込みのSIP登録サーバーもあります。 したがって、3CXシステムはゲートウェイでSIPオペレーターとして登録できます。 これを行うには、通常のSIPトランクが3CXで作成されます。 これはまさにこのマニュアルで使用されている方法です。



GSMゲートウェイのセットアップ



このガイドは、ファームウェアバージョン53.18.0.46を搭載したYeastar NeoGate TG200 V2ゲートウェイインターフェイスに基づいています。



1.ユーザー名adminとパスワードpasswordを使用して、192.168.5.150（デフォルトで事前定義）のゲートウェイインターフェイスに移動します。



2.更新が利用可能な場合、ゲートウェイのファームウェアを更新することをお勧めします。 現在のファームウェアバージョンは、ステータス-システム情報セクションで確認できます。 Yeastar Webサイトから最新のファームウェアをダウンロードし、[システム-ファームウェアの更新]セクションで更新します。 ファームウェアのアップグレードは、TFTPサーバーを使用して行われます。







3. [システム-LAN設定]セクションでネットワークに一致するゲートウェイのIPアドレスを設定し、ゲートウェイを再起動します。







4. [ゲートウェイ-モバイルリスト]セクションのゲートウェイインターフェイスで、GSMポートに署名します。 そのため、将来それらとの作業が容易になります。 [通話時間の設定]セクションですぐに、通話時間のさまざまな制限を指定できます。







5. [モバイルグループ]セクションで、1つのGSMオペレーターのゲートウェイに割り当てるポートをグループ化できます。 ただし、このガイドでは、各ポートを個別に使用します（そのオペレーター用）。 したがって、gsmgroupグループを削除する必要があります。







6. [VoIPトランク]セクションでは、2つのデフォルトSIPアカウントがすでに作成されています。 設定を編集するか、新しいアカウントを作成できます。 この場合、このアカウントがユーザー名として使用するGSM行番号を指定して、デフォルトアカウントを編集するだけです。 デフォルトのパスワードを変更することもお勧めします。



注意！ GSMアカウントの名前（名前）を変更した場合、アイテム4のGSMポートの名前と一致しないようにしてください。







7. [モバイルからIP]セクションで、デフォルトルートを削除し、モバイルネットワークから最初のモバイルオペレーターのIPネットワークへのルートを作成します。 同様に、ルートは他のオペレーター用に作成されます。 この場合、3CXの1つのSIPトランクがゲートウェイの1つのGSMポートに対応することを思い出してください。 ただし、1つのSIPトランクが複数のGSMポート（通常は1つのオペレーター）で機能する場合、他の構成も可能です。







8. [IPからモバイル]セクションで、デフォルトルートを削除し、IPネットワークから最初のモバイルオペレーターのモバイルネットワークへのルートを作成します。 同様に、ルートは他のオペレーター用に作成されます。







3CXセットアップ



1. 3CXインターフェースの[SIPトランク]セクションで、新しいSIPトランクを追加します。 一般的な国を選択し、ユーザー名とパスワードで認証を提供する一般的なVoIPプロバイダーを選択します。



2.表示される設定インターフェイスで、次を指定します。



  • トランクの名前（これは、GSMオペレーターまたは方向の名前かもしれません）

  • GSMゲートウェイIPアドレス

  • トランクでの同時呼び出し（整流）の数（上記で合意したように、各3CX SIPトランクはゲートウェイの1つのGSMポートに対応します）-1

  • GSM Gateway Setupセクションのセクション6からのトランク接続資格情報。







3. [パラメーター]セクションで、PBXオプションを介したプロキシオーディオストリームを無効にし（PBXとゲートウェイが同じネットワーク上にある場合）、オプションを設定しますRe-Inviteメソッドをサポートし、Replacesヘッダーをサポートします。 [OK]をクリックして設定を保存します。







4. SIPトランク3CXは、GSMゲートウェイに正常に登録する必要があります。





5.新しいトランクの送信ルールを作成します。 PBXを介した通話のルーティングに必要な条件に応じて、ルールをトリガーするルールを指定します。







これで、3CXと連携するためのYeastar Neogate TGXXX VoIP GSMゲートウェイの構成が完了しました。



SMSメッセージを使用する



Yeastar NeoGate TG100 / TG200 / TG400 / TG800 GSMゲートウェイを介してSMSメッセージを送受信するには、 Yeastar MySMSユーティリティを使用します。







また、SMS-SMS設定セクションで、組み込みのゲートウェイ機能SMS to E-mailおよびE-mail to SMSを使用することもできます。
