@unfollowrがShorty Awardsを受賞しました。％username％！



Habrachelovek、この投稿は素晴らしいアプリケーション@ unfollowr （その目的について-以下）へのヘルプ投稿であり、劣らず素晴らしい人@ ibobrikであることをすぐに警告します。 技術的な問題やその他の問題はすべて、catの下で意図的にクリーンアップします。 助けたい人はTwitterのアカウントのみが必要です。





今年は、第2回ショーティアワードコンペティションが開催されます。今回はミンスク市の開発者@ibobrikが参加します。 アプリカテゴリでは、彼の@ unfollowrアプリが紹介されています。これは、多くのTwitterユーザーが使用していると思います。



@unfollowrを使用すると、あなたから退会している人を追跡できます。 その人があなたの読書をやめた後、あなたはすべての未購読が名前でリストされている通知を伴う個人的なメッセージを受け取ります。 ブロックされているスパマーでさえ、彼から隠れません。 通知を購読するには、@ unfollowrのフォローを開始してください。



@Unfollowrは現在2位ですが、共同の努力により、彼が1位になるのを支援できます。 彼が勝った場合、@ ibobrikはニューヨークを見ることになります。もちろん、彼は本当に望んでいます:)。



そのため、開発者を支援したい場合は、 このページからツイートを作成し、その後に示す必要があります。理由は、例えば、控えめな「I like it」または「it rocks」です。 リンクからテキストをコピーするだけで、任意のクライアントからツイートを送信することもできます。



PS ShortyAwardsページから投票すると、デフォルトで通知に登録されます。 チェックマークを外すことを忘れないでください;）。



UPD ：「 because ... 」の後に何も書かないと、ツイート「#apps because ...でShorty Awardの@TwitterUserをノミネートします」では、投票はカウントされません。 （ ダイブありがとう）

UPD1 ：@ibobrikがリンクを整理しました。これをクリックして[更新]をクリックすると、Twitterページから直接投票できます。
