（パート1）PSPをWindows用のリモコンとゲームパッドに変える

あなたはSony PSPコンソールの幸せな所有者ですか？ 追加費用なしで、わずか数分でコンピューターのリモートコントロールを取得する方法を説明します。 そしてもう少し時間を費やすと、素晴らしいゲームパッドを手に入れることができます。









このベンチャーを実装するには、安価な方法と比較的安価な方法の2つがあります。



最初のケースでは、コンソールはミニUSBケーブルで動作します。 明らかに、「ソファから音楽を切り替える」ことに頼る必要はありません（ もちろん、延長コードを転送することもできます=） ）。 しかし、非常に快適なゲームパッドであることがわかりました。

2番目のケースでは、Wi-Fiルーターが追加で必要になります（ 幸いなことに、3つごとにルーターがあります）。 したがって、リモコンがいっぱいになり、ゲームパッドがさらに便利になります。

注：コンピューター自体にWiFiがあるかどうかは関係ありません。 通常のツイストペアケーブルを使用してルーターに接続できます。 主なことは、コンピューターがPSPと同じネットワーク上にあることです。



PSP Slim＆lite、Windows 7搭載のラップトップ、D-Link DI-524ルーターを使用しています。



ステップ1：必要なものをすべてインストールする



以下が必要です 。

  • カスタムファームウェア（5.03 Gen-Cなど）を備えたプレイステーションポータブルコンソール。
  • Windows 7 / Vista
  • コンソールを接続するためのミニUSBケーブル。
  • Wi-Fiルーター（2番目の方法の場合）
  • PSPdispプログラム


アクション ：

  1. ミニUSBケーブルを使用してPSPを接続します。
  2. PSPdispインストーラーを起動します。
  3. インストールはデフォルト設定で実行されます。
  4. 署名されていないドライバーのインストールを確認します。
  5. インストール中に、プログラムはコンソールを検出し、ユーティリティをメモリカードにコピーする許可を要求します。 これが行われない場合は、フォルダーX：/ PSP / GAMEへのパスを手動で指定します。

  6. PSPdispのインストールを完了します。


ステップ2：セットアップ



  1. PSPをコンピューターから切断します。
  2. PSPをオンにします。
  3. 「ゲーム-メモリスティック」に移動し、PSPdispを実行します（リストの最初にあるはずです）。
  4. プログラムはこれを与える必要があります：

  5. 三角形をクリックします。 コンピューターの内部IPアドレスを入力します（たとえば、192.168.0.3）。
  6. 接続方法を選択します。 USBを選択した場合、ケーブルをPSPとコンピューターに挿入します。 WiFiの場合、コンソールをWiFiポイントに接続するための標準ダイアログが表示されます。


WiFi接続の場合：


  1. Windowsシステムトレイで、PSPDispアイコンを右クリックします。 メニューから[オプション-ワイヤレスLANを使用]を選択します。
  2. メニューで「有効」を選択します。 プログラムは、コンピューターの内部IPアドレスを報告し、それを介して接続するよう要求します。


ステップ3：最後のステップ。 確認する



  • トレイのPSPdispを右クリックします。 [コントロール-マウスエミュレーション]を選択します。



  • 詐欺のパフォーマンスを確認します。アナログを動かします。 カーソルが移動するはずです。 カーソルの近くの画像の一部がPSP画面に表示されます。 「十字」は右クリック、「正方形」-左をエミュレートします。






原則として、少なくとも少し英語を知っている人は、プログラムをさらに設定できます。 最後の手段として、ヘルプがあります。



この段階で、トピックの最初の部分は終了します。 次のパートでは、特定のアクションのボタンをプログラミングするPSPDispの詳細な構成について説明します。 例として、Windows Media CenterとAimpの構成について説明します。 また、PSPから本格的なゲームパッドを作成する方法も示します（「Need for speed：Underground 2」の例を使用）。
