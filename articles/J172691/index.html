IT学生向けの無料セミナー

2008年以降、主にMSTUの学生向けのコースを実施しています。 バウマン、MAI、MIIT。 現在、無料のセミナーがあり、学生IDを提示するだけで誰でも参加できます。



各セミナーは、このテクノロジーをどう処理し、どのように対処するかについての短くて有能なストーリーです。 タスクは、特定の方向に最初のステップを実行し、頻繁なミスを防ぐ方法を示すことです。 簡単に言えば、サブジェクト領域へのエントリのしきい値を削除します。 所要時間-1時間半から2時間。最後に論文が発行されます。これは教師または将来の雇用主に感謝される場合があります。



以下は、今後のイベントのお知らせです。 簡単に説明すると、これらが主なものです。

GNU / Linuxのサーバーテクノロジー

4月1日

これは、セーターを着た狂信的なひげを生やした人々からの純粋なGNU / Linuxプロパガンダです。 確かに、彼らはセーターやあごひげなしになりますが、まあまあです。 セミナーでは、Linuxの無知を恥ずかしく思うのをやめ、一般的にLinuxについて詳しく知ることができます。 さまざまなLinuxディストリビューション、アプリケーションサービス、およびそれらの機能が検討されます。 また、最新のクラスタリングおよび仮想化テクノロジーも影響を受けます。 計画はこれです：

  • GNUプロジェクトとLinuxカーネル。 Linuxの使用条件、ライセンスの種類。
  • サーバーとしてのGNU / Linux：サポートされているハードウェアプラットフォーム、典型的なアプリケーション、運用および運用機能、長所と短所。
  • 一般的なLinuxサーバーのディストリビューションの概要：RHEL、Centos、Ubuntuサーバー、SLES
  • ネットワークシステムとしてのLinux、基本サービスの例：DNS、DHCP、NTP、ルーター、ファイアウォール、VPNコンセントレーター、Linuxベースの暗号化トンネル、wwwサーバーおよびファイル共有サービス、MySQL、PostgreSQL、メール、異種環境のファイルサーバー（ NFS、SMB / CIFS）。
  • 仮想化プラットフォームとしてのLinux。
  • システムの継続的な運用を保証する、信頼性の高いクラスターおよびその他の方法。

仮想化テクノロジーとクラウドコンピューティング

4月3日

外は21世紀であり、ハイパーバイザーまたはハイブリッドクラウドとは何かわかりませんか？ 大丈夫です、このトピックの読者の99％はこれについて知りません:)。 真剣に、このセミナーは、異なるハードウェアおよびソフトウェアプラットフォームでの仮想化とソリューションのトピックに関する非常に詳細な教育プログラムです。 仮想化は、ほとんどすべての最新テクノロジーに浸透しており、さまざまなレベルで現れています。 まずは指の説明から始め、x86ベースの仮想化の機能に進み、Xen、VMWare、KVM、Hyper-Vに触れてから、PowerプラットフォームとARMプラットフォームの機能を見ていきます。 さまざまな種類のクラウド-Google Gmail、Saleforce、Google Apps Engine、Microsoft Azure、Amazon AWSを見てみましょう。 また、厳しいロシアの現実のために「クラウド」を構築する際のレーキについても説明します。
