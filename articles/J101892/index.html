Ubuntu 10.10の指ジェスチャ

Mark Shuttleworthは彼のブログで、2010年10月10日（10.10.10）にリリースされるUbuntu 10.10 Maverick Meerkatがマルチタッチインターフェイスをサポートすると発表しました。



Canonicalは、この目的のためにUTouchと呼ばれるフレームワークをMaverickに作成しました。







これまでのところ、開発は、幸運な所有者が結果を評価できるようになったDell XT2 （写真）をサポートすることを目指しています。 このリリースは、Apple Magic Trackpadなど、さまざまなメーカーの膨大な数のデバイスをサポートする予定です。



開発チームは一種の「タッチ言語」を作成しましたが、これは以前にこの目的のために作成されたすべてのものを上回ると言われています。



現在、個々のジェスチャだけでなく、チェーンにリンクして「文」全体を作成する機会があります。 基本的なジェスチャー、またはプリミティブは、個々の動詞に似ており、そのシーケンスははるかに複雑なアクションを記述することができます。



新しいコードは、GPLv3およびLGPLv3ライセンスでLaunchpadに公開されています。 サブシステム用のモジュール（XやGtkなど）は、互換性のあるライセンスの下で配布できます。 好奇心と共感のある人のために、PPAがあります。詳細は、開発者向けの公式発表に記載されています。



新しいフレームワークは、マルチタッチに関連するX入力プロトコル用に最近公開されたPeter Huttererアップデートを使用します。 ジェスチャ処理とジェスチャイベント配信機能を追加します。



Maverickのかなり多数のアプリケーションがすでにジェスチャースクロールをサポートしています。 Evinceは、開発者がアプリケーションに追加できるいくつかのより高度なトリックを示すためにさらに開発されます。 Unityのウィンドウ管理もジェスチャーをサポートするため、10.10 Netbook Editionのユーザーは、タッチスクリーンまたはマルチタッチパッド（マルチタッチパッド？）を使用して、ジェスチャーを完全に楽しむことができます。



楽しみたい場合は、デスクトップにUnityをインストールできますsudo apt-get install ubuntu-netbook



を選択し、ユーザーを選択するときに適切なセッションを選択します。



10.10以降の計画には、開発者向けの通常のAPIの形成が含まれ、ジェスチャーを処理するためのシステムサービスを提供します。



マークは、11.04ですべての主要なアプリケーション（ブラウザー、メール、ファイルマネージャー、チャット、写真マネージャー、メディアプレーヤー）を「タッチ」したいと考えています。



このニュースはCanonicalの公式ブログにも投稿されています。
