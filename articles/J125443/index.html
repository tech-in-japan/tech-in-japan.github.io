中国のWindows 7タブレット

中国でタブレットPCを所有してから数週間経ちました。 Androidベースのデバイスの豊富さを研究すると、質的に異なるものが必要であることに気付きました。 ある時点で、タブレットフォームファクターのラップトップを探し始めました。Pickegg.com （ Dingoo A320、販売で有名な）でのキャンペーンの結果、375ドルでWindows 7搭載 の中国産業の奇跡的な製品を購入しました。







タブレットの内部には、 Windows 7 Ultimateがインストールされた本格的なPCプラットフォームがあります 。 Windows Updateはデフォルトでオンになっており、動作していますが、OSが海賊版になっているのではないかと思います。

外観



タブレットの前面には、[スタート]メニューを開くWindowsボタン、ライトセンサー、カメラ、タブレットの電源オン/スリープ、バッテリー充電、ワイヤレスアダプターの組み込みを表示する3つの明るいアイコンがあります。



上部には、電源/スタンバイボタン、マイク、無用のSIMカードスロット、microSDカードスロット、Gセンサーロックボタン、スタートメニュー呼び出しボタン、および内蔵スピーカーの音量と明るさを調整できるロッカーがあります手動バックライト。







右側には、19ボルトの電源を接続するためのコネクター、RJ-45を介してVGAモニターと有線ネットワークを接続するための複合コネクター、およびUSBポートがあります。







下には、ボルトを覆うステッカーが2つだけあります。







左側には、USBポート、ヘッドフォン出力、わかりにくいプラグ、ワイヤレスアダプタ用のスイッチがあります。







タブレットの背面には、通気口と内部にリセットボタンのある小さな穴があります。



使用する



実際、タブレットの内部には長所と短所のある通常のネットブックがあります。 特に、上部の換気口の1つから温風が吹き出されます。



iOSおよびAndroid用に特別に設計されたタッチコントロールとは異なり、Windowsは当初マウスとキーボードの使用を暗示していました。 ただし、Windows 7にはマルチタッチサポートが導入され、OSの制御機能がOS自体に組み込まれました。 ほとんどの場合、馴染みのあるプログラムを使用して作業するのは快適ですが、入るのがそれほど簡単ではない小さな要素があります。 Total Commanderの使用を放棄して標準のファイルマネージャーを採用しなければなりませんでしたが、驚いたことにAdobe Lightroomでの作業は苦労しませんでした。 必要に応じて、Bluetoothを含め、マウスやキーボードをいつでも接続できます。



本格的なUSBポートとオペレーティングシステムにより、使い慣れたデバイス、カメラ、外部サウンドカードを接続できます。これは私の選択の重要な基準でした。



タブレット自体はiPadに比べてやや重いため、片手で長時間保持するのは不快な場合があります。ひざまずいたり立ったりします。 画面は良好で明るく、側面からの視野角は180度に近くなりますが、画面の上下ははるかに小さくなります。 光沢のある前面は瞬時にフラップし、幸いなことに背面はつや消しです。 分解せずにケース素材を決定することは難しいと思います。 前部はプラスチックのようで、ボタンは正確にプラスチックで作られていますが、後部はアルミニウム合金に似ています-それは心地よい涼しさと粗さで満足しています。 ケースは非常にモノリシックですが、ツイストテストは4つで実行されます。







バッテリーの寿命は、Wi-Fiが含​​まれているかどうかにもよりますが、充電器を近づけておく方が良いでしょう。



結論



私たちの中国人の友人は別の興味深いデバイスを作成することができました。 一般的に、タブレットの印象は好意的であり、その機能を維持しながら、ネットブックよりもコンパクトではるかにインタラクティブであり、このレビューはそれに書かれています。 ただし、少量の「指向け」ソフトウェアによって、マウスやキーボードがないことを後悔する場合があります。



iPadやAndroidタブレットと比較するのは完全に正しいとは限りません。なぜなら、一般的な形式にもかかわらず、内部にはさまざまなクラスのデバイスがあるからです。 ただし、タブレットの目標は達成されており、うまく機能しています。
