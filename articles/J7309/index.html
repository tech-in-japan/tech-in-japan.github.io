Firefoxとそれが食べられるもの。

たまたま、私は最初からモジララインのインターネットブラウザを使用していました。 グラフィカルインターフェースを備えた最初のOSはFreeBSDでした。NetBSDコミュニケーター以外のグラフィカルブラウザーについては知りませんでした。 次に、Opera 6、Mozilla 0.9、Mozilla 1.0、Seamonkey 1、Firefox 1、Opera 8、Firefox 1.5、Opera 9、Firefox 2、Firefox 3（a1、a2、a3、a5）がありました。 かなり豊富な経験だと思います。



現在のブラウザFirefox 2.0.0.3は非常に満足しており、機能を実行する以上に、オペラのインターフェースの速度が著しく低下しています。



実際には、機能について。 ページhttp://firefox.niksite.ru/firefox-config.htmlから判断すると、現在約30の拡張機能がアクティブになっていますが、最も興味深いのは、幸せなユーザーのブログで見つけたものです。 なぜチェーンを壊す-私は、おそらく、彼らと私について説明します。



  • Greasemonkeyは、JavaScriptコードをページに埋め込むことができる強力な拡張機能であり、かなり大きなコレクションがhttp://userscripts.org/にあります。 最も有用なのは、次のことです。
  • Greymonkeyが独自のJavaScriptをページに追加するため、 スタイリッシュは最も便利な拡張機能の1つです。したがって、この拡張機能は独自のCSSをページに追加します（これは、ページだけでなくブラウザー全体にも影響します）。 他のスタイルのうち、論理的なネストはhttp://userstyles.org/で取得されます。 次のスタイルが私にとって最も有用であることが判明しました。
  • abcTajpu -Unicode文字を便利に入力できます。たとえば、-を-に、（s）を©に、7/8を⅞に、EURを？に変換します。
  • Adblock Plus-広告をフィルタリングし、非常にうまく処理します。 外部ソースからデータベースを更新することができます。この拡張機能にElement Hiding Helperを追加すると、画像/スクリプト/そのようなアドレスからの基準だけでなく、個々のDOM要素でも広告をフィルタリングできます。
  • FxIF-存在する写真のプロパティにEXIF情報を表示します
  • マウスジェスチャ -既にマウスを使用している場合は、最大限に使用します。 マウスでの前方/後方遷移は、LMB-> RMB / RMB-> LMBとは異なります
  • NoScript-不要なスクリプトをカットすることにより、ブラウザをわずかに高速化します
  • OpenSearchFox-任意のページの検索フィールドをブラウザーの検索フィールドに追加できます。 さて、lib.aldebaran.ruでアートブックを見つけるには、このサイトを読み込む必要はありません。入力フィールドを探して、submitを押します。 代わりに、Ctrl + K、Alt +下矢印、lib.aldebaran.ruに配置し（最初の文字-「l」を押すと）、検索語を入力します
  • 検索エンジンの整理 -サブフォルダーを上記の検索エンジンドロップダウンメニューに追加します。
  • 個人メニュー -画面スペースを無駄にしたくない。 [ファイル]、[編集]、[表示]パネルなどは不必要に頻繁に表示されるため、この拡張機能を使用して1つのメニュー項目に最小化し、ブックマークの隣に配置しました
  • 完全に削除 -このXPath要素を使用して、サイトのページから要素を削除できます。
  • ロシア語のホットキーのバグ修正 -この拡張機能がないと、Linuxのメインキーボードの組み合わせがロシア語のレイアウトで機能しなくなる
  • 画像をフォルダに保存 -選択したフォルダに画像を簡単に保存できます。 （メニューを表示せずに）ダブルクリックして、選択したフォルダに画像を保存すると特に便利です。
  • ScrapBook-オフライン表示用の別のセーバーですが、既にページ全体で、スタイル、スクリプト、グラフィックなどが含まれています-リンクのいくつかのレベル。 すべての保存されたフルテキストインデックス、もちろん構造化。 本当に便利な拡張機能
  • Tab Mix Plus-タブなどでやりたいことすべて。 私は個人的にタブの開閉にトリッキーなオプションのみを使用します-新しいタブは現在のタブの右側に直接開かれ、閉じるとき、スイッチは左から開いている場合は右に、そのような右のタブがもうない場合は左になります
  • Web Search Pro-選択した単語をページのセクターにドラッグして検索します（特別な検索フィールドではなく）
  • ロシア語/英語のスペル辞書 -入力フィールドで英語とロシア語のスペルをチェックします。 letterの文字で。
  • NextPlease-同じキーボードの組み合わせを使用して、フォーラムおよび検索エンジンの次/前のページに移動します。 Ctrl +右/左があります
  • MR Tech Local Install-拡張機能をローカルドライブからインストールできるようになりました。さらに、FireFoxのこのリリースで動作するように設計されていなかった場合でも、拡張機能をインストールできるすばらしいアイテム「make compatible」も表示されます
  • MR Tech Likn Wrapper-過度に長い単語は、たとえこれらの単語がリンクであっても、水平スクロールを引き起こしません
  • Locationbar²-住所フィールドをスラッシュに基づいて送信に分割します。サイトのサブセクションを通過すると本当に便利な場合があります
  • Firebug- web-construct`orovの素晴らしい拡張機能; HTML / JS / CSSコードをオンザフライで変更でき、このコードを使用して便利に配置できます（ブラウザの目的の要素の上にマウスを移動し、次に-このコードに関連付けられているHTMLとCSSを参照してください）
  • スナップリンク -一度に複数のリンクを選択して、バックグラウンドタブで開くことができます。 時には非常に便利です。
