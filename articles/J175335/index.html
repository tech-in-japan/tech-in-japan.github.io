JavaFX 2をサポートするIntelliJ IDEA 12.1

こんにちは、Habr！



伝統的に、私たちは毎年春を祝うために、IntelliJ IDEAのユーザビリティの修正と改善を伴う更新を行っています。 今年も例外ではありませんでした！ IntelliJ IDEA 12.1が最終的にダウンロード可能になったことをお知らせいたします 。







ご存知のように、昨年末に、IntelliJ IDEAはJava 8、AndroidのUIデザイナーであるPlay Framework 2、コンパイラーの2倍以上の速度、データベースを操作するための新しいツール、そしてもちろん、ダークカラーテーマDarculaのサポートを獲得しました。 IntelliJ IDEA 12.1では、これらの機能のほとんどは、この間に受け取ったコメントを考慮して改善され、多くの問題が修正されました。





JavaFX 2

IntellIJ IDEA 12.1には、Javaの新しいリッチクライアントプラットフォームであるJavaFX 2のサポートが拡張されています。 サポートには、FXMLマークアップの編集、カスタムCSSファイル、自動補完、検索とナビゲーション、リファクタリング、パッケージ化ツール、SceneBuilderとの統合が含まれます。



デモをご覧ください：







ダルキュラと網膜

新しいダークカラースキームDarculaについて非常に多くの肯定的なレビューを受け取ったことに非常に驚きました。 多くのユーザーがDarculaをIntelliJ IDEA 12の最も重要な技術革新と考えています。これを考慮して、ダークスキームをさらに改善するために改良を試みました。



さらに、Retinaディスプレイ（および「高密度」ディスプレイもサポート）のサポートを改善しました。 必ずチェックしてください 。



IntelliJ IDEA 12.1のその他のイノベーション



  • Windows用のフルスクリーン
  • Spring Frameworks 3.2およびPlay Framework 2.1（Ultimate）のサポート
  • Groovy 2.1のサポート（新しい注釈、コンパイル設定などを含む）
  • Scalaサポートの改善（新しいコンパイラーのサポート）
  • Gradleとの統合の改善（gradleラッパー、タスクなどのサポート）
  • Adobe Gaming SDK （Ultimate）のサポート
  • ソースマップを介したCoffeeScriptおよびTypeScriptのデバッガー（Ultimate）
  • Sassサポートの改善（カスタム関数定義、オートコンプリート、名前変更リファクタリング、ネストされたプロパティなど）（究極）




すべての改善点について学び、IntelliJ IDEAをダウンロードするには、当社の公式Webサイトにアクセスしてください。



レム！ 既にIntelliJ IDEA 12を所有している場合は、IntelliJ IDEAを無料でバージョン12.1にアップグレードできます。古いバージョンを使用している場合は、ここでアップグレードオプションの詳細をご覧ください。
