Windows 10でMetroアプリケーションをアンインストール/復元します





OSの古いバージョンのサポーターがいくらいようと、Windows 10に切り替える時がまだあります。この点に関しては、プリインストールされたMetroアプリケーションを扱うことを提案します。 同意して、企業が私たちに課しているすべてのアプリケーションを絶対に使用する人はほとんどいません。 結局のところ、それらのほとんどはちょうど彼らの場所を取り、誰かが一般的に悩まされます。 この記事では、Windows 10のアプリケーション-不要な-からシステムを「クリーン」にする方法を紹介します。



削除する



それでは、不必要なアプリケーションの削除に移りましょう。 Powershellスクリプトを提供します。その結果、インストールされているすべてのMetroアプリケーションが表示されます。すべてを削除するか、問題に選択的に対処するかは、ユーザー次第です。



スクリプトは次のようになります。

スクリプト.bat 
 @echo off cls echo Apps echo. echo press any key to continue... pause > NUL echo echo. 回復



誤って削除した場合、または後で1つまたは別のアプリケーションを使用することにした場合は、Windowsストアから復元するか、次の提案された回復オプションを使用できます。



たとえば、上記のスクリプトによって削除された「ストア」アプリケーションを復元したい場合、Powershellの管理者から次の行を実行する必要があります。 



 Add-AppxPackage -register "C:\Program Files\WindowsApps\*Store*\AppxManifest.xml" -DisableDevelopmentMode


同様に、「*」の間に適切な名前を指定することにより、他のアプリケーションを復元できます。 特定のN番目の数量の作業を再開する必要がある場合、スクリプトを使用することをお勧めします（必要な行を削除/追加します）。



アプリケーションを復元するスクリプト

または、プリインストールされたユーティリティのセット全体を返す場合は、Powershellで管理者として次の行を入力します。 したがって、実験することを恐れないでください。初期状態を復元するのは簡単です。
