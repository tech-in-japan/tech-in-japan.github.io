iTunesをWindows XP x64で動作させる方法。

序文の代わりに。



多数の非人道的な実験の後、私の32ビットの古いWinXPが最終的に死亡したのは夕方でした。 新しいものを試してみたかった（まあ、すぐに4Gb RAMが私のマシンに現れるかもしれないという事実に目を向けて）、そして私たちのオフィスには対応するライセンスがあるので、WinXPの64ビットバージョンをインストールすることにしました。

新しくインストールしたシステムのiTunesディストリビューションが単に存在しないことが判明したとき、SUDDENLYと呼ばれる新しくインストールしたWindowsの多くの長所と短所は重要ではないように見えました...



診断。



Appleは、「最も人気のあるオペレーティングシステム」のユーザーへの忠誠心のために、x64向けのメディアプロセッサのリリースについて心配しているだけではありません。Vistaバージョンは健在です。 それにもかかわらず、XP x64はこの点で孤児と見なされており、Apple Webサイトで提供されるすべてのバージョンのiTunesインストーラーは、そのインストールを断固として拒否します。 これは、私の兄弟であるiPodとiPhoneが兄のサポートなしで放置される可能性があることを意味します。 おそらく、この事実の悲劇全体について語る価値はありません。

もちろん、通常のiPodの所有者は、ガジェットを管理するための代替ソリューションを検討するのが理にかなっています。たとえば、 Floolaは、iPhoneとiPod Touchのサポートの欠如という1つの欠点を除いて 、個別のレビューに値する素晴らしい機能的な製品です。 私は、iPhoneの所有者として、1つの希望を残しています-全能のGoogle ...

幸いなことに、iTunesのインストールの問題に遭遇したのは私が初めてではないことがわかりました。 iTunesとWinXP x64の相互敵意を克服するために、インターネット上でいくつかの指示がありました。 残念ながら、それらには2つの欠点があります：英語;）と、与えられた解決方法のわずかな陳腐化。 そこで、私はこのトピックについて別のトピックを書くことにしました。そこでは、外国の方法論と個人的な経験を組み合わせました。 私は特別に予約をします-提案されたソリューションの100％の稼働容量を保証しません。 これは非公式の決定であるため、ご自身の責任で使用してください。



レシピ。



したがって、64ビットのiTunesを準備するには、次のものが必要です。

1. 7-zipアーカイバ。 ただし、 ポータブル版も適しています。

2. mscaパッケージOrcaのスクリプトテーブルのエディター。

3. QuickTimeの最新バージョン。

4. Windows Vista用のiTunesディストリビューションの64ビットバージョン。

5.（オプション）GEAR SoftwareのCDを操作するためのドライバー：iTunesはCDの取得/書き込みにこれらのドライバーの32ビットバージョンを使用し、これらのドライバーをインストールすると、iTunesでCDを操作する機能が有効になると想定されます。



手順。



1. Orcaと7-zipをインストールします。 7-zipを使用して、iTunesインストーラーをホテルのディレクトリに解凍します。



それで、私たちは中に何を見ますか？ iTunes自体に加えて、操作用の追加コンポーネントのセットもあります。



最初に、パッケージAppleMobileDeviceSupport64.msiをインストールします。 インストールプロセス中に、システムはそのような未検証のコンポーネントをインストールするかどうかを確認しますか？ もちろんです！ ;）おめでとうございます、私たちはiPhone-iPhone用のドライバーセットをインストールしました:)



2.非常に簡単な手順-QuickTimeをインストールします。 覚えておいて、私はあなたにそれを別々にダウンロードするように頼んだか？ 問題は、解凍されたアーカイブのQuickTimeがインストールされていないことです（まったく不明な理由のため）。 これが、QuickTimeをApple Webサイトの別のディストリビューションからインストールする必要がある理由です。 QTのインストールプロセスをさらに説明する意味はありません。すべてが通常どおりです。ライセンス契約を読み、すべてに同意し、最後に「次へ」をクリックします。



だから、今では、不思議な理由でAppleによって導入されたOSバージョンの制限を除いて、iTunesをインストールすることを私たちを傷つけるものはないようだ。 iTunesインストーラーをVistaより下のWindowsバージョンで実行できないのは、この制限です。 この制限を克服しましょう。

3. OrcaエディターでiTunes64.msiパッケージを開きます。



外国のインターネットでは、アドバイザーは主に次のアドバイスに限定されています。

「... LaunchConditionタブでVersionNT64パラメーターを見つけて、> = 600 to> = 501 ...の代わりにその値を設定します...」



私の場合、これは機能しなかったため、もう少し複雑な2番目の方法を適用する必要がありました。

  • 「LaunchCondition」タブを完全に削除します。 これを行うには、それを右クリックし、「テーブルをドロップ」を選択します。
  • 次に、左側の「InstallUISequence」タブに移動し、右側の 「LaunchConditions」 パラメーターを見つけます。 コンテキストメニューから[行の削除]を選択して、オプションを削除します。
  • 左側の「InstallExecuteSequence」タブを選択し、「LaunchConditions」パラメーターも見つけて削除します。
  • ファイルを保存して閉じます。


4.これで、 iTunes64.msiを安全に起動して、アプリケーションをインストールできます。 主なことは、「インストール後にアプリケーションを実行する」フラグをクリアし、インストール後すぐにiTunesを起動しないことです！ まだやることがあります。



5. iTunesを起動するたびに誤ったインストールに関するメッセージを回避するには、GEARからドライバーをインストールする必要があります。 問題は、このパッケージが最近更新されたが、古いバージョンが見つからず、同時に、iTunesの起動時にチェックするファイルの1つが新しいバージョンから削除されたことです。 解決策は次のとおりです。ドライバーをインストールした後、フォルダーC：\ Windows \ Sysnative \ driversをc：\ windowsフォルダー（まあ、またはそこにインストールした場所）に作成し、その中にGEARAspiWDM.sysという名前の空のファイルを作成します。



6.だから、私たちはすでにほとんどの道を進んでいます。 次のステップは、iPhone所有者向けです。

  • iPhoneを初めて接続すると、システムはデバイスを検出し、ドライバーを提供するように見えます。 Appleドライバーを使用するにはiPhoneが必要です。 これを行うには、[ デバイスマネージャー]を開き、[イメージングデバイス]セクションでiPhoneを見つけます（ロシア語MUIがインストールされたWindows XP x64がインストールされている場合、このような言語の混合に驚くことはありません:)）。
  • 「Apple iPhone」プロパティをダブルクリックし、「ドライバーの更新」を選択します。
  • 提案された対話では、インターネット上のドライバーの検索、自動検索、および「指定された場所の検索」を徐々に拒否します。 「検索しないでください。 必要なドライバーを選択します。「選択して、「次へ」ボタンをクリックします。
  • 「デジタルスチルカメラ」と「AppleモバイルデバイスUSBドライバー」の2つのオプションが提供されます。 「Apple Mobile Device USB Driver」を選択し、「次へ」をクリックします。
  • インストールされたドライバの原因不明に関する別の恐ろしいメッセージの後、「続行」をクリックし、プロセスが完了するのを待ちます。


このiTunesのインストールは完了したと見なすことができ、プログラムを実行してその魅力をすべてお楽しみください！



最後に、iPhoneの電話帳から連絡先を保存することを強くお勧めします。Outlookとの同期に問題があり、古いシステムのOutlookデータベースのバックアップのおかげで電話帳を保存できました。
