Skype 2.0用Skype Gateway 3CXゲートウェイ

3CX Ltdの公式ディストリビューターであるロシアの会社ICE Partnersは、Skypeから固定電話および携帯電話に通話を転送できる3CX Gateway for Skype 2.0プログラムの最終バージョンがリリースされたことを発表しました。 これは、すべてのロシアのIPテレフォニーオペレーターに欠けていたものです。



私から少し：

3年前、私がInfosvyaz社のプログラマーとして働いていたとき、クライアントから質問がありました。 ユニバーサルシティナンバーを使用して、SkypeまたはSkypeから通話を転送できますか。 その後、解決策はないように思われました。 Skypeは、独自のプロトコルおよび独自のコードです。 入手可能な情報源をざっと調べて、私は残念ながら答えました-ほとんど不可能です。

昨年ラップトップを購入し、最初にコンピューターを起動したときに、Skypeクライアントへのアクセスを使用するアプリケーションの許可を求められました。 インターネット上で、新しいバージョンのスカイプにはソフトウェアインターフェースがあり、他のアプリケーションが音声とビデオを送信できることがわかりました。 オペレーターがSkypeで作業するためのVoIPサービスの登場は、それほど長く待つ必要はないと自分自身で決めました。



2009年の夏を通して、3CXのSkypeゲートウェイのベータテストが実施されました。 その結果によると、9月の初めから、最終バージョンが利用可能になりました-3CX Gateway for Skype 2.0 Final。



Skype用3CXゲートウェイは、3CX Ltdが開発した3CX電話システム用のソフトウェアゲートウェイであり、任意の内線番号からSkypeネットワークへの通話を送受信できます。



これで、Skype Call MeボタンをWebサイトに配置したり、パートナーに企業のSkypeアカウントを通知したりできます。すべてのSkype通話は通常の着信通話と同様に処理されます。 それらは記録、モバイルへの転送、キューイングなどが可能です。

内線番号で、ユーザーは自分のSkypeアカウントへの着信転送を有効にできます。 同時に、Windows MobileまたはiPhoneベースのコミュニケーターを使用することが可能になりました。 Skypeのモバイルバージョンをインストールするだけで、これらのデバイスでオフィスコールを受信できます！



3CX Gateway for Skypeは、3CX電話システムと同じサーバーにインストールされます。 これにより、1台のコンピューターで複数のSkypeアプリケーションを実行して、マルチチャネル回線を整理できます。



Skype用3CXゲートウェイの機能：

-着信および発信Skypeコールのサポート。

-通話を受信するメインのSkypeアカウントを指定する機能。 次に、補助アカウントに切り替えます。メインアカウントは常に新しい通話を無料で受信できます。

-Skypeの最新バージョンのサポート。

-3CXインターフェースを介したSkypeアカウントの追加。

-Skypeアカウントの残高の表示。

-ロギングを呼び出します。
