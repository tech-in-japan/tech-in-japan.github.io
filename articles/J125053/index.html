eラーニングトラッカーの概要（パートII）

普遍的な情報化の時代に、教育はよりアクセスしやすくなっています。 オックスフォードまたはケンブリッジで、著名な教授の講義をオンラインで視聴したり、ウェビナーに参加したり、教育を受けたりできます。 このトピックでは、貴重な資料を無料で提供することがある教育用トラッカーに触れたいと思います。 BitMe.org、BitSpyder.net、LearnBits.me、Elbitz.netについてはすでに詳しく説明しています。 それで、新しいレビュアーは：



1. ScienceHD



ドキュメンタリー、ショー、テレビ番組を愛する人のための興味深いリソース。 原則として、SciHDでは高品質の720p / 1080pです。 古代文明の歴史から現代のサイエンスフィクションまで、誰もがここで何かを見つけます。 リリースは、シーンの一種の「混合物」であり、TheBox.bzであふれ、排他的であり、トラッカーの外に広がることは禁じられています。 評価を維持することは非常に簡単です：シードと多くのフリーランス（ダウンロードはカウントされません）のボーナスシステムがあります。 また、登録時に、指定のアカウントに10 GBが書き込まれます。 招待は大学院クラス（35 GBが与えられ、5つの配布が作成され、1 GB以上がダウンロードされます）以上でのみ利用できます。 同時に、1つの招待は1,000,000ボーナスを「費用」とします（ダウンロードした100 GBの「借方記入」に相当）。 いずれにせよ、 招待フォームは希望する人のためにほとんど常に開かれています。

2. 金庫



「ビジネス」トラッカーには、検索エンジンの最適化、商取引、マーケティング、投資などに特化したほぼ30,000のディストリビューションが含まれています。 一般的に、何らかの形で財政とその増加に関連しているすべてのもの。 「グループ購入」があります。興味のある素材は、ユーザーの費用でバケットによって購入されます。 評価は新しいディストリビューションで最もよく得られ、フリーランスのディストリビューションはほとんどありません。 質問に答えることで 、IRC（サーバー：irc.thevault.bz:6667、チャンネル：＃thevault-invites）への招待を取得できます。

3. MyAnonaMouse



オーディオブックに最適なトラッカーはまだ存在しません。 ここでは、科学文献からフィクションまで、常に読んだり聞いたりすることがあります。 ミュージシャンには、ガイドとメモがあります。 座席、ボーナスの維持のしやすさ、親しみやすい雰囲気、素晴らしいコミュニティ、ブラックジャックに対するボーナスシステムが魅力的です。 最も興味深いディストリビューションをダウンロードできるのはVIPだけです;ボーナスのステータスは「購入」できます：28日間で5000。 ボーナスも招待と交換されます（3回の招待に対して50回のボーナス）が、ScienceHDの場合のように、 招待フォームはほとんど常に開いています。

4. オタク



最新のプロジェクトはTBZチームで、ThePlace、TheBox、TheOccult、TheVault、TheEmpireなどのリソースを作成しました。 自己改善に関する文献やビデオの存在（ビデオチュートリアル、電子書籍、ドキュメンタリーに加えて）で注目に値する：スポーツから魔法や幻想までのセクション。 毎週のフリーリッチにより、評価は比較的シンプルに保たれます。管理者は、指定されたもののみがカウントされる複数のディストリビューションを選択します。 分布は毎週変わります。 詐欺の 宝くじもあります。これは、20 GBを特定の人に獲得するチャンスを与えます。評価が低いためにアカウントをオフにしようとしているメンバーのみが参加できます。 01から49までの6つの数字（毎週4回の試行）を選択する必要があります。1つの数字が一致した場合、賞金は1 GBになります。 登録は現在開いています。
