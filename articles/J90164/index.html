SunCalc-ソーラー電卓

私は余暇に開発した、小さいが有用な投影法であるSunCalc太陽電卓を皆さんと共有したいと思います 。







SunCalcは、選択された場所の地図上に、日中の太陽の軌跡と照明フェーズ（およびいくつかの楽しいもの）を表示します。



彼のアイデアは、私がクリミアで日没時に海に沈む太陽のフレームを撮影するための良い場所を見つける必要があった後に生まれました（ビデオのために、彼らはここで最後にそれを撮影しました）-これのための既存の手段はすべて非常に時代遅れであることが判明しました。 想像できる限り最高のものを作り、天文学の公式がたくさんある良いページを見つけ、無料の夕方に仕事に取り掛かったので、ここに結果を掲載しました。



特徴



  • 夜明け、日没、選択した時刻の太陽の位置（黄色、オレンジ色、赤色の線）、太陽の軌跡の投影（オレンジ色の弧）、年間を通しての散布（黄色の領域）を含む美しいベクトルグラフをマップに描画します。 グラフ上の点が円の中心に近いほど、その瞬間の地平線上の太陽が高くなります
  • 日没/夜明け線にカーソルを合わせると、対応する位置の広がりが表示されます
  • タイムスライダーで照明（暗闇、夕暮れ、昼光）を色付きで表示します
  • 夜明けと日没だけでなく、 3種類のトワイライト （市民、ナビゲーション、天文学）および完全な暗闇の時間も見ることができます。
  • 選択した場所の週間天気予報（曇り、降水量、霧）へのリンクを表示します
  • 場所を特定できます（適切なフィールドをクリックします->場所を検出します）
  • 検索、自動検出、地図のクリック、マーカーのドラッグによって場所を選択することもできます
  • パーマリンクはブラウザの特定の場所/時間および前後に機能します




エゴは何のために使用できますか



まず、純粋に日常的な状況で。 ヘッドランプをまだ購入していない場合、森を離れるのにどれくらいの時間が必要ですか？ ビーチが右側の崖の影を覆う前に、夕方にどれくらい日光浴できますか？ 家の下の遊び場でサッカーをするために外に出たとき、いつ太陽が私の目に直接見えなくなりますか？ この年に借りるアパートの照明は何ですか？ このような問題において、それ自体が繰り返し重宝しています。



第二に、写真愛好家！ 私を含めて。 たとえば、夕暮れがいつ、どのくらい続くか（夜景の場合）、太陽が沈む方向、または太陽が昇る場所（適切な角度を選択するため）、被写体を写真に撮る必要がないように、被写体の写真を撮る時間を知ることが重要です。または、彼は完全に隣の建物の影などに落ちませんでした。



第三に、一般教育と自然な好奇心を満たすためです。 オーストラリアはなぜ夏よりも冬が暑いのですか？ 本当に白い夜とは何ですか？どの都市で、いつ観察されますか？ 極地の昼と夜はどうですか？ 太陽の軌道は北から南にどのように変化しますか？ モスクワからキエフまでの夜明けは何分ですか？ 等



いくつかの実装の詳細



  • すべてのアプリケーションコードはブラウザ側で実行されます
  • Google Maps API v3は地図と検索に使用されます
  • マップ上のベクターグラフィックス用-Raphalライブラリ
  • 位置情報-W3C Geolocation API 、 Google GearsまたはGoogle IP Geolocation
  • jQueryといくつかのjQuery UIコンポーネントが使用されます（時間のスライダー、日付のDatepicker、ようこそウィンドウのダイアログ）
  • jQuery AddressプラグインはディープリンクとAjax履歴に使用されます
  • セッション間で、状態はクッキーに保存されます
  • 驚いたことに、 Wolfram Alphaは天気予報に最も適していることが判明しました
  • 改善のためのアイデアのために、 UserIdea / Reformal サービスのページ（すでにHabréの著者によってカバーされています）が開始されました
