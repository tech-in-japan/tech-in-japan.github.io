►梱包と設備

最近、一部の中国企業では、デバイス自体のこの特性を強調するために、タブレットの入った箱を薄くする方法があります。



非常に薄いので、電源装置が干渉しなくなり、別の箱に入っています。 私たちの小包の郵便はspareしまないので、私はすぐにアイデアが最高とはほど遠いことを言わなければなりません。 幸いなことに、私のアダプターは傷つきませんでした。



ここで、デバイスの最初のマイナスを見つけます。 Chuwi V88は、通常のMicro USBの代わりに、充電用に別のポートを使用します。 もちろん、このアプローチは充電時間の短縮をもたらします（強力な電源が提供されている場合）が、この場合、もちろんどこにでも充電器を持ち歩く場合を除き、デバイスを離れて/職場で再充電することはできません。



上記のタブレット自体と充電器に加えて、ボックスにはUSB-OTGケーブル（3GモデムまたはUSBフラッシュメモリを接続するため）、コンピューターに接続するためのワイヤー、および少しの古紙が含まれています。



バンドルは最小限ですが、タブレットを完全に使用するために必要なものはすべて揃っています。 箱が薄いという事実にもかかわらず、それは耐久性のある段ボールで作られており、タブレット自体を損傷する可能性は無効になります。

►ハードウェアプラットフォーム、パフォーマンス

Chuwi V88タブレットは、1.6 GHz（1.8〜2 GHzのオーバークロックが可能）でクロックされるRockchip RK3188クアッドコアプロセッサとMali-400 MP4グラフィックコアに基づいています。 これは、2013年7月現在、中国のタブレットで使用されている最も生産的なソリューションの1つです。 さらに、デバイスには2 GBのRAMが搭載されており、「食いしん坊」のおもちゃを起動するときにChromeで閉じるタブを忘れることができます。

その主なタスクの1つである、つまり、Chuwi V88はブラウジングによって強打に対処します。 ページの読み込み時間は3〜4秒で、スクロールはスムーズで、サーフィンに最適です。

►オペレーティングシステムとソフトウェア

このデバイスには、Jelly Beanとも呼ばれるAndroid OS 4.1.1がプリロードされています。 この記事を書いている時点では、最新のものはAndroid 4.2.2ファームウェアであるため、すべてのテストは既に行われています。以下では、タブレットにインストールする方法を説明します。

実際には命令自体（オリジナルはここにあります 、私は自分の言葉とロシア語でそれをもう一度言いました）：

  1. 最新のファームウェアをダウンロードする こことプログラムRKBatchTool v1.7（ドライバーと一緒に） こちら 。
  2. RKBatchToolプログラムと共にアーカイブ内にあるドライバーをインストールする途中で、ファームウェアを使用してアーカイブを解凍します。
  3. タブレットをコンピューターに接続するときに、大容量記憶装置モードが設定されていることを確認してください。 これを行うには、[設定]-> [メモリ]に移動し、左上隅の3つのドットをクリックして、ダイアログボックスで[コンピューターへのUSB接続]を選択します。
  4. タブレットをコンピューターに接続し、管理者に代わってRKBatchToolプログラムを実行します。
  5. Rockhip Batch Tool 1.7がロードされると、「Connected Devices」セクションのピンクのインジケーターが点灯します。これは、Chuwi V88が検出されたことを意味します。
  6. 「FW Path：」行の反対側のボタンをクリックして、ファイルupdate.img（以前にダウンロードおよび解凍したファームウェアイメージ）を開きます。
  7. [切り替え]ボタンを使用して、タブレットをファームウェア更新モードにします。 その結果、画面が空白になり、フラッシャープログラムのインジケータの色がピンクから緑に変わります。
  8. [復元]をクリックすると、デバイスのフォーマットとその後のファームウェアが開始されます（インジケーターが黄色に点滅します）。
  9. ファームウェアの終了後、標準のAndroidリカバリが起動し、/ data / cacheおよび/ mnt / sdcard / partitionsを個別にフォーマットします
  10. ソフトウェア更新後の最初のダウンロードには十分な時間がかかる場合があるため、しばらくお待ちください。 さらに、デフォルトでは、中国語がタブレットにインストールされますが、3つの手順で母国語に変更できます。左側のメニューで設定アイコンをクリックし、右側のメニューで「A ...」という記号の行を選択します。 2行目を選択し、リストの最後にロシア語を見つけます。


タブレット用のAndroid 4.2.2のリリースで、Googleは電話インターフェイスをデフォルトで設定し、タブレットに変更する機能を持たないという非常に不快な驚きを私たちに提示しました。



この迷惑な欠陥を解消するには、外国の愛好家によって開発された特別な調整をインストールする必要がありますが、これはCWMリカバリを使用してのみ実行できるため、最初にインストールしましょう。

ステップバイステップの手順：

  1. ダウンロードする Oma Rocksの代替ファームウェアとそのインストールに必要なツールを使用してアーカイブします。
  2. デバイスの電源を切ります。
  3. 「ボリューム+」キーを押しながら、USBケーブルを介してタブレットをコンピューターに接続します。
  4. ドライバーをまだインストールしていない場合は、今がその時です。
  5. 管理者に代わって、RKDevelopTool_v1.37フォルダーにあるRKAndroidToolアプリケーションを起動します。 起動後、アプリケーションの下部に、デバイスが検出されたという確認が表示されます（RKAndroid Loader Rock Usbが見つかりました）。
  6. 完全にカスタムファームウェアをインストールする場合は、まずEraseIDBをクリックしてから[実行]をクリックしてください。 私は「ひざの上で」組み立てられたそのようなファームウェアのファンからは程遠いので、CWMリカバリのみを設定しました。 これを行うには、6番目を除くすべてのチェックマークを削除して、[実行]をクリックします。
  7. すべての操作の結果、Chuwi V88は再起動します。CWMリカバリにすばやくアクセスするための小さなユーティリティをインストールするだけです（ルートが必要です）。「Volume +」ボタンを事前に押してタブレットの電源を入れてリカバリを開始することもできます。


ここでルート権限の取得についてお読みください （2番目の方法）。 リンクが別のタブレットの指示につながることに驚かないでください。これは、今日のレビューのヒーローのように、RK3188プロセッサーに基づいて構築されているため、ルートアクセスを取得する手順は同じです。



タブレットを使い慣れたインターフェイスに戻すことは残っています。 ダウンロードする 微調整して、ユーザーメモリのルートに「スロー」します。 次に、リカバリ-> sdcardからzipをインストール-> sdcardからzipを選択し、プリロードしたアーカイブを選択します。 その後、もう一度デバイスを再起動して...出来上がり！ できた



純粋なAndroid 4.2.2について話すことは、ネットワークがこのOSのレビューでいっぱいなので、時間の無駄です。 標準のランチャーとインターフェースの驚くほどスムーズな操作（中国のデバイスの場合）に注意してください。 タブレットには、複数のタスクを同時に実行するとフリーズする機能があります（市場からアプリケーションをインストールし、音楽を閲覧して再生します）。これには4つのコアがあります。 それほど面倒ではありませんが、不快な後味が残ります。
