Gitでの作業に役立つVisual Studio Codeの拡張機能

Visual Studio Codeにはデフォルトでgitがありますが、組み込み関数だけでは便利で迅速な作業には必ずしも十分ではありません。 gitで効果的に作業するには、プロセスの視覚化も必要です。



以下は、gitでの作業を便利で楽しいものにする拡張機能です。



ギトレンズ



GitLensは、組み込みのVisual Studio Codeのgit機能を拡張します。 拡張機能には、多数の機能と柔軟な設定があります。 ワンクリックで、著者、注釈などの情報を1行ずつ取得できます。







Gitの歴史



Git履歴を使用すると、変更履歴を表示できます。 それはかなりユーザーフレンドリーなインターフェースを持ち、非常に高速です。







Gitグラフ



この拡張機能は、リポジトリブランチを参照したり、履歴を検索したりするのに役立ちます。 プルリクエストを受け入れてコードレビューを行う開発者にとって、この拡張機能は不可欠になります。







Gitインジケーター



Visual Studio Codeの下部ペインにあるgitアクティビティの簡単なインジケーター。
