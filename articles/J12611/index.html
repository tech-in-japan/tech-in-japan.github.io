Visual StudioコードネームOrcas

Visual Studio 2008の2番目のベータ版が本日発表されました（コードネームOrcas、内部バージョン9.0）。

VS 2008には.NET Framework 3.5（これまではBeta 2も含まれています）が含まれています。これは、多くの興味深い新技術のサポートを意味します。
  • LINQは、データクエリの言語拡張機能です。
  • ADO.NET Entity Framework-ADO.NETのORMソリューション。
  • Acropolis-複合Windowsアプリケーションの構築。
  • Astoria-REST Webサービス（JSONを含むデータのシリアル化を使用）。
配布パッケージには、Visual Studio自体に加えて、SQL Server Express 2005（SQL Express 2008が最終バージョンになることを想定しています）およびSQL Server Compact 3.5（クライアントワークステーション、タブレット、PDAへのインストール用）が含まれています。

また、VS 2008 Beta 2にはGo-Liveライセンスがあります。 今日、商用製品とソリューションを構築できます。

次のバージョンのVisual Studioをダウンロードできます 。
  • Visual Studio 2008 Standard Editionベータ2
  • Visual Studio 2008 Professional Editionベータ2
  • Visual Studio Team System 2008-チームスイートベータ2
  • Visual Studio Team System 2008-Team Foundation Serverベータ2
  • Visual Studio Team System 2008-テストロードエージェントベータ2
  • Visual Studio 2008 Beta 2のMSDNライブラリ
さらに、プレインストールされたVisual Studio Team SuiteまたはTeam SuiteとTeam Foundation Serverを使用して、仮想マシン（Virtual PC / Server形式）をダウンロードできます。

MicrosoftがVisual StudioのExpressバージョン（完全無料、商用利用を含む）を発行するという伝統を続けているという事実は、喜ばせざるを得ません。 「エクスプレス」のセット全体もダウンロードできます 。
