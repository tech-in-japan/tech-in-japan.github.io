ReadyBoost-スピーディキャッシュ

どこでも、どこでも、このオペレーティングシステムをscったりand笑したりするのは良い習慣になっています。 一方、Windows Vistaはすぐに使用できるものを求めているため、上級ユーザーには適しています。 Vistaは単なる「箱入り」バージョンですが、他のOSにはわからないようなチューニングの機会を提供します。



モバイルランダムアクセスメモリは現在高価であり、アップグレード後に512MBの「チョコレートバー」を置く場所です。 Windows Vistaユーザーは幸運です-フラッシュドライブを準備します。 これは、システムパフォーマンスを向上させるために設計されたWindows Vistaの革新的なReadyBoostについてです。 新しいテクノロジーにより、ユーザーは、システムドライブやアプリケーションファイルをキャッシュするための追加スペースとして、USBドライブ、カードリーダーでサポートされている任意の形式のフラッシュカードを使用できます。



ReadyBoostを使用する前に、オペレーティングシステムは特定の技術要件に準拠するようにデバイスをテストします。テストは次のとおりです。2.5Mb / sの速度で4 KBのランダムリードと1.75 Mb / sの速度で512 KBのランダムリード。 Vistaはデバイスを承認しないため、ReadyBoostはテストから離脱する必要があります。



離乳



そのため、デバイスを接続し、「マイコンピュータ」に移動し、デバイスを右クリックして、コンテキストメニューで「プロパティ」項目を見つけます。 次に、[ReadyBoost]タブで、[接続時にデバイスのテストを停止する]にチェックマークを付け、デバイスをオフにします。



[スタート]メニューに入力してレジストリに移動します。引用符なしで「regedit」コマンドを実行し、HKEY_LOCAL_MACHINE / SOFTWARE / Microsoft / Windows NT / CurrentVersion / EMDgmtのブランチを見つけます。次のキーの-デバイスステータス= 2、ReadSpeedKBs = 1000、およびWriteSpeedKBsも値を1000に設定します。デバイスを接続し、プロパティに移動し、[ReadyBoost]タブに移動して[このデバイスを使用]を選択し、下のスライダーを設定します。 喜ぶ。



よくある質問：



行動の原則。 要するに、ReadyBoostはフラッシュメモリを使用して、そのようなメモリにデータを読み書きする際のアクセス時間が非常に短いため、OSやアプリケーションのロードを高速化します。 ReadyBoostは高速を必要としません。さらに重要なことは、フラッシュメモリへの微視的なアクセス時間です。これは、HDDの10分の1の場合もあります。



パフォーマンス。 4kbブロックを読み取ると、パフォーマンスはHDDと比較して最大10倍になりますが、大きなファイルを読み取ると効果は失われます。 ただし、ReadyBoostは、アプリケーションをダウンロードするとき、何十もの小さなライブラリに複数のアクセスがあり、その成長が目で見てわかることが多い場合にのみ使用されます。



安全性 ReadyBoostはキャッシュに過ぎず、スワップをHDDにバックアップするため、フラッシュドライブをいつでも取り外してデータを失うことはありません。 すべてのデータは、AES-128アルゴリズムを使用して暗号化されます。



最低限。 ReadyBoostは256 MBを使用できます（フォーマット後は250）



最大。 ReadyBoostは、最大4GBの物理フラッシュメモリを使用できます。これは、圧縮中の約8GBのデータに相当します。 FAT32は小さなメモリカードをサポートするために使用され、このファイルシステムの制限は4GBです。



耐摩耗性。 予備テストの結果によると、フラッシュドライブはこのモードで10年以上使用されます。



制限事項 2つのフラッシュドライブを使用できません。 外付けUSB 2.0 HDDは使用できません。 外部カードリーダーを使用することは望ましくありません。この場合、遅延が増加します。 MP3プレーヤーを使用することは望ましくありません-多くのプレーヤーはcなアルゴリズムに従って動作し、これにより遅延が増加します。
