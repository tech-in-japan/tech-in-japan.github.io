隣人はシャンピーでささやきます[FreePBX]

「ささやき」とは、「経験豊富な従業員」が「クライアント」-「若い従業員」という対話に接続し、若者が対話を行うのを支援することです。 「経験者」は当然、若者だけが聞いています。クライアントはこれについて知る必要はありません。



驚いたことに、Googleのマシンは最初のリンクからはほど遠い「考える」方向を示しました。 FreePBX Webマズルを使用したアセンブリでは、 extensions_override_freepbx.confファイルに次の行を埋め込むことでこのメカニズムが実行されます。 



[app-chanspy] exten => 555,1,Macro(user-callerid) exten => 555,2,Authenticate(1234) exten => 555,3,Read(SPYNUM,agent-newlocation) exten => 555,4,ChanSpy(SIP/${SPYNUM},wqv(-1)) ;--== end of [app-chanspy] ==--;


このメカニズムは、通常のChanSpyを置き換えます。 使いやすい：経験豊富な方は555をダイヤルし、パスワード（1234）を入力し、「若い」の内線番号を入力して会話に参加します。
