Firefox 26リリース

デスクトップ向けFirefoxがダウンロード可能になりました。 このバージョンの新機能：





  • すべてのJavaプラグインはデフォルトで無効になり、それらに関連付けられたコンテンツを有効にするには、特別な領域（ Click-to-Play ）をクリックする必要があります。 以前に脆弱性のためにブラックリストに登録された特定のバージョンのJavaプラグインのみでコンテンツがブロックされていた場合、Javaコンテンツはデフォルトでブロックされます（Flashを除くすべての外部プラグインのバージョンをブロックする予定でしたが、最終リリースでは変更が適用されました） Javaのみ）;
  • MemShrinkプロジェクトの継続：表示されていない画像はデコードされないため、ページの読み込み時間（および重要なことに、RAMの消費）が大幅に削減されました。



  • 対応するGStreamerプラグインがインストールされている場合、 LinuxでのH.264のサポートが含まれます。
  • Password Managerは、スクリプトによって生成されたパスワード入力フィールドをサポートするようになりました。
  • Firefoxインストールディレクトリへの書き込み権限を持たないWindowsユーザーがFirefoxの更新を実行できるようになりました（最初にMozillaサポートをインストールする必要があります）。
  • コンテンツセキュリティポリシー（CSP）の実装は、仕様に沿った施行ポリシーとレポート専用ポリシーの両方のケースを含む多くのポリシーをサポートするようになりました。
  • Social APIは、SocialMarkをサポートするシステムのブックマークサービスのサポートを追加します。
  • Windows XPでMP3デコードサポートを追加しました。 Firefoxは、WindowsのすべてのバージョンでMP3デコードをサポートするようになりました。
  • Firefox OS App Managerアプリケーションマネージャーが追加され、開発者にFirefox OS用のHTML5アプリケーションのテスト、デプロイ、デバッグをFirefoxから直接簡単に行うためのツールセットが提供されました。
  • 「レスポンシブデザイン」ツール（Ctrl + Shift + M）に2つの新しい機能が追加されました。タッチイベントのサポートと、このツールを使用して変更されたWebページのスクリーンショットを撮る機能。
  • インスペクター（Ctrl + Shift + I）は、擬似要素スタイルをサポートするようになりました。
  • CSSのimage-orientationプロパティのサポートが追加されました。 度単位で回転角度を設定することに加えて、from-image属性を指定してEXIFメタデータを考慮することができます。
  • 単一の画像を表示する場合、Firefox は画像に含まれるEXIF方向情報を考慮します 。
  • Firefoxは、 AppCache Webサイトで使用されたときに確認フォームを発行しなくなりました。
  • IndexedDBは、ユーザーの確認なしに作成され、最も古い要素が混み合った状態で固定サイズのプールにデータを保存する一時ストレージとして使用できます。
  • OS XのAudio Toolbox MP3バックエンドのサポートが追加されました。
  • 更新KB2670838を使用したWindows 7（MSIE 10のインストールに必要）またはWindows 8.1でのテキストのレンダリングに関する問題が修正されました
