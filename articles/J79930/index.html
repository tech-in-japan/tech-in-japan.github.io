あなたのビジネスを助ける





しばらく前に、私たち（webmaniacs）は独自のプロジェクトを作成し始めましたが、ハブでテストをスキップするという提案もありましたが、テストは非常に成功しました-レビューを受け取り、多くの人々がそのようなサービスに興味を示し、いくつかの間違いが示されました。



私たちはほぼ半年かけてアイデアを思い浮かべ 、機能の拡張、バグ修正、いくつかのことを考え直し、プロジェクトの2番目のバージョンであるlabourist.comを提示しました 。





基本的な考え方は、中小企業（緊急事態、フリーランサー、セールスマネージャー、スタジオ）のオンラインサービスアシスタントです。

私たちのビジネスライフ全体は、何らかの形で私たちが参加する「プロジェクト」です-顧客側またはパフォーマー側では、多くがこれらのプロジェクトをノートブックで「リード」します（ポストイットもモニターに残します）、最も「高度な」ものは使用しようとしますソフトウェア開発サイクルや巨大で遅いCRMシステムで使用される追跡ツールは、これは私たちのほとんどにとってうまくいきませんでした-そして今、簡単で簡潔なオンラインサービスを提供することにしました。 マーケティング用語にカルマを費やすのではなく、機能を簡単に説明します。



だから：

  • 支払いの管理に重点を置いたプロジェクト管理（タスク、タイムライン）（誰がいくら、いつ、どれだけの費用を支払うか）、2種類のプロジェクト-収入と支出、プロジェクトの記事、支払いと連絡先。 今後のイベント、カレンダーの通知。
  • ドキュメントフロー-バージョン管理、関係者への通知をサポートするプロジェクトに関するドキュメント。
  • 連絡先ストレージは、すべてのクライアントプロジェクトのビジネス連絡先のクロスリストであり、タスク、支払い、ドキュメントをクライアントに追加できます。
  • 機能的な「スクリーンショット」があります。 顧客や請負業者とのコミュニケーションに時間を浪費しないように、システムツールを使用してプロジェクトの状況の「スクリーンショット」を表示するだけで、「なぜそんなに？」を明確に説明できます。




プロジェクト全体がSSLでラップされ、最も簡単な登録（確認または追加データなし）、デモプロジェクト、アフィリエイトプログラムもあります。



このプロジェクトはまだ止まっておらず、機能は近い将来に優先度順に計画されています。



  • API
  • レポート（PDFを含む）
  • 追加の通知方法
  • 追加機能を備えたプレミアムアカウント
  • 計画されている「機能」の1つはまだ宣伝しませんが、非常に役立つと思います。




このプロジェクトは、西側のいくつかのプロジェクトとどう違うのですか？ （2つの類似物が見つかりましたが、それらはまだ別の方向で機能しています）



  • ローカリゼーション-言語と精神の両方;

  • 使いやすさ。
