ACER Aspire V7-482PGウルトラブックでmSATAからダウンロード

このマニュアルでは、パーティションをWindows 8からSSDディスクに転送するプロセスについて説明します（システムを再インストールせずに）。これはmSATAに接続され、一部の操作をキャッシュするために定期的に使用されます。 転送プロセスは簡単ではありません。起動可能なフラッシュドライブを作成するにはいくつかのスキルが必要です。 また、ストレス耐性に問題がある場合は、これを行わずに、通常2.5インチドライブから起動することをお勧めします。 すぐにスクリーンショットの品質をおaびします。古いスマートフォンしか利用できませんでした。



計画を実施するには、より大きなmSATA SSDが必要です。 システムはKingston 24Gb SSDをインストールしました。 個人的に、mSATA PLEXTOR PX-128M5Mをインストールしました。







パーティションをコピーおよび編集するには、このウルトラブックで実行できるブートプログラム（Acronis、Paragonなど）を使用できます。 アクロニスが直接起動しなかったため、ブータブルWINPE 4.0イメージを使用して、そこからアクロニスを直接起動しました。 WinPEビルドはTechAdmin1.4でした（トレントにあります）。 以前作成したマルチブートフラッシュドライブとGrubローダーおよびエミュレータfiradiskディスクを使用して、イメージから直接ダウンロードし、次のようにmenu.lstに書き込みました。



title WinPE WIN8





map --mem /grub/fw32.ima (0)





map --mem (md)+8 (9)





map /grub/winpe4.iso (255)





map --hook





write (9) [FiraDisk]\nStartOptions=cdrom,vmem=find:/grub/winpe4.iso;\n\0





root (255)





chainloader







何らかの理由で、ロード後の環境ではイメージが表示されないため、WinPEで使用されるプログラムからイメージからUSBフラッシュドライブに使用しました。

フラッシュドライブからダウンロードするプロセスについては詳しく説明しません。これは別のマニュアルの情報です。



転送に直接進みます。



1）HDDにDATAパーティションを作成する




デフォルトでは、ハードディスクにはWindows 8がインストールされるパーティションが1つあるため、ダウンロードとは無関係にデータパーティションを作成するには、2つの部分に分割する必要があります。

これを行うには、ウルトラブックを通常モードで実行し（まだ実行していない場合）、Microsoftアカウントを登録するなどします。 Windowsとドライバーを更新できます（重いプログラムをインストールしない方が良いです）。 つまり、システムを基本的な作業に対応できる形式に変換します。

Windows 8のデフォルトでは「シャットダウン」は通常の休止状態であるため、標準メニューからではなくシステムの電源をオフにします。つまり、保存された構成が読み込まれます。 Windows 7のように標準シャットダウンの場合、コマンドプロンプトで「 shutdown -p 」と入力する必要があります。このシャットダウンは、マニュアル全体で使用します。



Akronis / WinPE / Paragonなどをダウンロードします。 これを行うには、BIOSに移動し（ウルトラブックをオンにしたときにF2ボタン、電源ボタンをクリックした直後に「スパム」する必要があります）、ブートモードをレガシーに転送します。







F10を押して再起動し、もう一度F2を「スパム」します。 次に、ブート優先順位のリストで、F6ボタンを使用してフラッシュドライブを一番上に移動します。 F10を押して再起動します。

次に、Acronisを起動します。 リカバリ、ESP、Acer、プッシュボタンリセットの4つのセクションが表示されます。 WindowsパーティションはAcerパーティションです。 パーティションのサイズを最小（50〜60 GBの領域）に減らして、ディスクの右側に空き領域が残るようにします。 空き領域にNTFSパーティションを作成し、「DATA」と呼ぶことができます。 結果は次のようになります。







UEFIモードで再起動し（BIOSでレガシーアイテムをUEFIに変更します）、パーティションが表示されていることを確認します（その前に、システムパーティションの復元プロセスは完了しますが、大丈夫です）。 パーティションイメージの保存に使用します。



2）キャッシュからSSDを無効にする




次に、キャッシュSSDを職務から「解放」する必要があります。 そのようなアクションは私がこれを行うのを助けました：

  • セクションCのシステムリカバリを無効にします：（マイコンピュータ->システムのプロパティ->システムの保護->システムの復元->セクションC：->構成）。
  • ExpressCacheサービスを無効にします（コントロールパネル->管理ツール->サービス-> ExpressCache->スタートアップの種類：停止、停止ボタン->適用をクリック）。 私はそれが何を役に立つのか理解していませんでしたが、mSATAでは動作しないかもしれません

  • Intel Rapidプログラムを削除します。システムのSSDをブロックするだけです。 削除後、システムを再起動します。




すべての手順が完了すると、SSDがディスクの管理リストに表示されます（管理->コンピューターの管理）。 次に、SSDを「正しく」マークする必要があります。 キーボードでWin + Rの組み合わせを押し、「 diskpart 」と書き込みます。 DISKPARTコンソールが開きます。 その中に一連のコマンドを記述します。 



 //    list disk //   ( N   SSD) sel disk N //     clean //    GPT convert gpt //    EFI create partition efi size=300 //   EFI  FAT32,    format fs=fat32 label="System" quick //      1024KB create partition primary align=1024 //    select partition 2 //   NTFS,    format fs=NTFS label="WinSSD" quick //    assign exit




その後、「WinSSD」セクションが「マイコンピュータ」リストに表示されます。

それでは、システムパーティションのコピーを始めましょう。 上記のように、Windows 8をシャットダウンしています。



3）パーティションをSSDにコピーする




次に、パーティションを直接コピーするか、最初にすべてのAcronis TrueImageパーティションのイメージを作成します。 2番目のオプションを選択しました。

作成した「DATA」セクションを除くすべてのセクションのイメージを作成します。イメージファイルはこのセクションに配置されます。 結果のイメージは、外部デバイスに保存できます（たとえば、ネットワーク経由でサーバーに転送されます）。 その後、HDDで何かが発生した場合に使用できます。 これらのパーティションがないと、SSD自体はmSATAを介して起動しません。

次に、Acronisをクリックしてパーティションを復元します。 保存したイメージを選択し、ESPとAcerの2つのセクションのみを復元します。 リカバリ設定で、それぞれSSDのパーティションを選択します。 パーティションのサイズはSSDのサイズよりも小さくなりますが、SSDから正常にロードした後、Acronisでパーティションを後で拡張できます。

回復後、UEFIを介して通常のWindows 8を再起動します。コピーされたウィンドウのあるSSDのパーティションが[マイコンピューター]リストに表示されていることを確認します。



4）UEFIブートローダーの編集




今最も難しい部分は、UEFIブートローダーを編集して、コピーしたWindows 8からSSDにパーティションを追加する必要があることです。 その結果、両方のバージョンのウィンドウが使用可能になり、ダウンロードするパーティションを選択できます（誰かにとって便利な場合があります）が、この選択を削除して、デフォルトでSSDからパーティションをロードできます。

管理者としてCMDコンソールを起動します（それ以外の場合はアクセスできません）。

1つ目は、作成したパーティションにブートファイルをコピーすることです。 「cmd」コンソールで、次のコマンドを実行します。



bcdboot c:\Windows /se:







e：現在のWindows 8のSSDパーティションの文字。



すべてがうまくいった場合は、続行します。

次に、UEFIブートローダー（Windowsブートマネージャー）を編集する必要があります。つまり、コピーしたセクションをブートリストに追加します。 



 //       bcdedit /copy {current} /d "Win8SSD" //     bcdedit /v //     ,   , -   {ed97fee6-fa6f-11e2-56a7-084e31ce3863}     2 ,  e:    SSD bcdedit /set {ed97fee6-fa6f-11e2-56a7-084e31ce3863} device partition=e: bcdedit /set {ed97fee6-fa6f-11e2-56a7-084e31ce3863} osdevice partition=e:




これで、新しいセクションがダウンロードリストに表示されます。 Windowsをシャットダウンし、ウルトラブックを起動し、ブートリストを参照し、流行を選択して、何倍もの速度をお楽しみください。 「DATA」セクションを使用してデータを保存できます。







次に、SSDパーティションからデフォルトブートを割り当てるか、選択を完全に削除できます。 たとえば、WIN + R " msconfig "と入力します。 また、Acronisをダウンロードしてパーティションをフルサイズに拡張したり、SSDの空き領域に2番目のパーティションを作成したりすることもできます。



mSATAコネクタを備えた他のラップトップでも同様のアクションを実行できます。主なことは、SSDが独立したドライブとして認識されることです。
