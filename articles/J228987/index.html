Prestigio Visconte-Windows 8.1上のビジネスタブレット

今日、電子機器市場にはあらゆる味と色のガジェットがあふれています。 これはタブレットにも適用されます。 また、ユーザーのニーズは日々高まっているため、メーカーはますますWindows 8に切り替えているため、デバイスの機能は何度も拡張されています。 はい、そしてそのようなデバイスはよりよく買われます。 この人生の休日を避けるために、Prestigioは最初のWindows 8タブレットであるPrestigio Multipad Visconteをリリースしました。







そもそも、私は間違いなくテクノロジーに「幸運」です。 もう1つは、私の代わりに、従業員が職場で配る無料のコンピューターとラップトップを喜ぶことです。 もちろん、私も幸せですが、どういうわけか、完全にではありません。

事実、私はAppleテクノロジーだけに出会っている。 レコーディングスタジオのiMac、メディアプロダクションスタジオのMac Pro、現在のMacBook Pro（網膜ではない）のいずれであっても。

原則として、MacBookで生活できます。 美しいタッチパッド、クイックスタート、あなたの指先での設定。 ただし、手自身は最初にWindows 8.1をParallels Desktopにインストールし、次に完全な「8」をBoot Campに完全に入れるように描かれています。

習慣に影響し、PCでの25年以上の経験。



多くの人が知っているように、Appleは、コンピューターでMicrosoftのOSを使用したいという顧客の希望をあまり歓迎しません。 したがって、Boot Campのタッチパッドドライバーは、控えめに言ってもあまり良くありません。 サードパーティのTrackpad ++ドライバーは、Win8の起動から3分後に読み込まれます。 この時まで、右クリックを行うことはまったく不可能です。 PageUp \ PageDown、Home \ Endキーがないことは、まったく奇妙に見えます。 これらの欠点に我慢しなければなりません。



さらに、私の仕事は、地下鉄と運転の両方で、モスクワ周辺を常に旅行することです。 もちろん、信頼性の高い航空アルミニウム製のMacBook。 この金属の素晴らしさはすべて肩に担がなければなりません。 同時に、コンピューターに必要なものはほとんどありません。オフィスアプリケーション、ブラウザー、Skype、Adobe Illustratorのまれなセッションです。

徐々に、簡単なアイデアが思い浮かびました-私は仕事のためにラップトップは絶対に必要ではありませんが、タブレットが必要です！ 完全な「Windows」で。 マルチタッチ付き。 そして簡単に。 Microsoft OSを搭載した他のホームデバイスであるデスクトップ、ミニサーバー、抵抗性スクリーンを備えた古いタブレットSamsung Q1Uを作ることができました。 後者は、率直に言って、長い間引退する時間がありました。





1か月でモスクワ周辺を移動する計画



そして今、より最近では、星は最も成功した方法で落ち着きました。私はまさに探していたものを手に入れました。 大きな画面サイズと完全なマルチタッチを備えた軽量のタブレット（ああ、これらのジェスチャー！）





どのタブレットについて話しているのかまだわからないふりをしましょう。



休暇中の同僚が、テストのためにPrestigioから新しいタブレットを受け取りました。 岡のほとりにあるカシモフの町で、私の祖母との2週間の休暇中にタブレットに関する記事を書く予定でした。





この町はまだ20世紀に住んでいるようです。



悲しいかな、いつものように、サイトのセマンティックコアを追加し、カタログに新しいカードを追加する必要があり、ディレクターでさえも手放しません。 一般的に、彼はオフィスでタブレットを安全に忘れて、バス停に向かいました。 もちろん、同僚が私たちに電話して、バスにタブレットを緊急に持ち込むように頼みましたが、誰がモスクワの郊外をこのような暑さで引きずりたいですか？

Windows 8.1の最も顕著な欠点の1つは、新しいタイルインターフェイス用に最適化されたプログラムのリストです。 それほど大きくなく、多くの場合、Metro UIスタイルのゲーム、メディアプレーヤー、その他のエンターテイメントアプリケーションが十分にありません。 ただし、Microsoftはオペレーティングシステムを放棄しているわけではなく、Windowsストアには毎日新しいアプリケーションが登場しています。







すべてのワイヤレスモジュールが動作し、ワイヤレスキーボードまたはヘッドフォンを選択できます。
