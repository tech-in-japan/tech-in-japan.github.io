HTML / CSSレイアウト用のメモ帳++

ご挨拶、同僚！



トピック「レイアウトの高速化 -Notepad ++のヘルプ」およびVadim Makeevの記事「Zen Coding 2.0。 コンセプト」は、長い間頭の中で生まれていたhtml / cssレイアウトデザイナーに必要なNotepad ++エディターの機能を拡張するというアイデアに私を押し付けました。



実際、私はこのエディターで「Zen Coding」の概念を実装したいだけでなく、他にもいくつかのことをしたいと思っています。 そして、これらすべてを、プラグインとタイプセッターに必要な機能がプリインストールされたNP ++の別のエディションで、 GPLリックの下でリリースします。



ポイントについて：

  1. Zen HTMLを埋め込む
  2. Zen CSSを実装する
  3. プロジェクト管理用のプラグインを作成します。
  4. クラス、識別子、ルールブロックのグループでの作業を簡素化するために、 関数リストの類似物を作成します。
  5. レイアウトに不要な機能をすべて無効にします。


このエディターのプラグイン（おそらくC ++）の作成を手伝いたい人はいますか？



誰かがこの善行に参加したいという願望を持っているなら、PMで私に書いてください。



プロジェクトの本質は、Zen CodingをNP ++に移植するだけでなく、エディターの機能を微調整することになるため、コメントと集合的な議論を包括的に歓迎します。
