Web開発およびGitHubと連携するためのChrome拡張機能





みなさんこんにちは。 サイトの開発、分析、GitHubとの連携、日常生活で使用するChromeの拡張機能の一部を共有したいと思います。 ほとんどの読者にとって、これらの拡張機能のリストは公開されるものではありませんが、初めて拡張機能について学び、それらをインストールする人は、間違いなく新しいツールでの作業を楽しむでしょう。



ツールの詳細な説明を含む長いフットクロスを作成しないでください。非常にシンプルで理解しやすいものです。 したがって、それらの使用の利点について簡単に説明します。



GitHubを使用するには



オクトツリーは、私の意見では、最も便利で便利な拡張機能です。 ページの左側に表示されるツリー形式でソースコードを表示できます。 新しいナビゲーションは、標準よりもはるかに便利です。 Octotreeは、多数のネストされたディレクトリを持つ大規模プロジェクトで作業している場合に特に役立ちます。



GitHub Repository Sizeは、個々のリポジトリのサイズとその中の各ファイルのサイズの両方を表示する便利でシンプルなプラグインです。



OctoLinkerは、単純なコード要素をクリック可能なリンクに変換することにより、迅速なコードナビゲーションを提供する拡張機能です。 このリンクをたどると、ソースをすばやく見つけることができます。 プログラミング言語に応じて、拡張機能はソースファイルを表示するか、ユーザーを外部サイトに移動します。 リンクに変換される要素にはピンクのドットが付いているため、インタラクティブなリンクがある場所をすぐに確認できます。



GitHub Hovercardは、GitHub Webサイトのどこからでも、ユーザー情報、リポジトリ、質問、コミットへの即時アクセスを提供します。 情報にカーソルを合わせるだけです。



npmhub package.jsonファイルがあるgithub.comのリポジトリを表示すると、この拡張機能はパッケージ内の依存関係を分析し、READMEファイルのすぐ下にそれらの説明を含むリンクを表示します。



github npm statsは、NPMサイトでパッケージのダウンロード統計を表示します（存在する場合）。



洗練されたGithubは、最近公開されたブランチを表示し、リクエストをやり直すことができ、コメントに応答したユーザーのアバターを表示します。



等尺性の寄与 -この拡張機能は、以前のものほど有用ではありませんが、非常に興味深いものです。 GitHubの通常の寄与度チャートと等尺性ピクセルアートのバージョンを切り替えることができます。



サイトを使用するには



Eye Dropperは、Webサイトで使用されている色を表示し、独自のパレットを作成するための優れたツールです。 私はほぼ毎日それを使用しています））。



全ページスクリーンキャプチャ 。 3〜5台のスクリーンに着陸したのに、フルスクリーンにする方法がわからないことを想像してください。 印刷画面を5回押して、エディターで1つを下に作成しますか？ まあ！ FPSCは、現在のページのスクリーンショットを数秒で取得できる優れたツールです。 特別なサービスをダウンロードせずにサイトの最適化を確認する必要がある場合、および作成したサイトを完全にキャプチャする必要がある場合に非常に便利です。



WhatFont-この拡張機能を使用すると、拡張機能をクリックしてカーソルを必要な場所に移動するだけで、サイトで使用されているフォントを簡単に見つけることができます。 シンプルでエレガントです。 「Esc」ボタンを押すと無効になります。 テキストの色、高さ、フォントサイズ、その他のパラメーターも表示されます。



PerfectPixel-この拡張機能は、ピクセル単位の精度でサイトを開発するのに役立ちます。



CSS Peeperは、デザイナー向けに特別に設計されたCSS分析ツールです。 この拡張機能の目標は、設計者が設計に集中し、コードを掘り下げる時間をできる限り少なくすることです。 実際に試してみれば、すべてが理解できます。

Google Translatorはもう 1つのツールであり、私の仕事のほとんどはこれなしではできません。 これはページではなく、拡張機能です。 これにより、選択したテキストを翻訳者のページに移動することなく、ページ上で直接翻訳できます。 追伸 長いテキストは翻訳しませんが、問題ではありません。
