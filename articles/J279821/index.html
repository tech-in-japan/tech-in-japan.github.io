IntelliJ IDEA用のPerlプラグインのバージョン1.4がリリースされました









JetBrains製品のPerlプラグインの新しいバージョンは、公式リポジトリからダウンロードできるようになりました。 主な機能： HTMLの最適化とサポート:: Mason



そして今、もう少し詳細：



新機能



HTMLサポート:: Mason



判明したように、古き良き時代にはHTML :: Masonフレームワークで書かれた多くのプロジェクトがあり、市民は積極的にサポートを求めました。 結果：



  • HTMLコンポーネントの解析:: Mason
  • オートハンドラーとdhandler、コンポーネントのフォルダー、ファイル拡張子、グローバル変数、カスタムタグに名前を付けるための構成。
  • 自動パディングと構文の強調表示
  • コンパイルされたコンポーネントの構造に応じた変数の正しい解決
  • コンポーネント階層の正しい解像度とその視覚化
  • テンプレートから呼び出された場合のコンポーネント、メソッド、ローカルコンポーネントのアクセス許可、ナビゲーション、リファクタリング


新機能



  • タグ
    • ラベルの許可、自動補完、ナビゲーション、リファクタリング（非推奨のラベルの使用はサポートされておらず、計画されていません）
    • 未発表および未使用のタグの検査
  • ヒアドキュメント
    • 他の言語を注入するためのオプションを備えたHere-docのマーカーの自動補完
    • 1行で連続したhere-docをサポート
    • backref here-docサポート
    • 補間された要素がある場合、here-docで言語インジェクションを自動的に無効化できるようになりました
    • 他の言語で挿入されたhere-docでフォーマットが正しく機能するようになりました
  • Perl 5.22の正規表現修飾子n
  • 通常の文字列をサポートされている言語で手動で挿入できるようになりました


ここでは修正のリストは提供しませんが、興味のある人は変更のリストに慣れることができます。 このリリースでは多くの内部最適化が行われ、プラグインは著しく高速になりました。
