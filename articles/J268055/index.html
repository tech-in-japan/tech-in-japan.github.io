Vivaldiアセンブリ1.0.288.3のExpressパネルの位置情報のサポートとサムネイルの修正

みなさんこんにちは！



次回の1.0.288.3の週次ビルドでは、修正されたバグの印象的なリストをお届けすることにしました。 Expressパネルでページのサムネイルを作成する機能を復元する作業について多くの人が喜んでいます。 しかし、他にも興味深いことがあります。 特に、ジオロケーションのサポートを追加しました。



ジオロケーションのサポート



はい、この便利な機能（マーケティング担当者の言うとおり）をVivaldiブラウザーに追加しました。 これで、このテクノロジーWebサイトを使用する各犬は、あなたがどこにいるかを判断し、あなたの場所のすぐ隣に多くの便利なサービスを提供できます。 ただし、ブラウザを最大限に制御できることをお約束しました。適切なポップアップダイアログボックスを使用して、位置情報データへのアクセスを提供するかどうかを自分で決定できます。









Sketches Expressパネル



Expressパネルでサムネイルを作成する機能を修正する第1段階を完了しました。新しいセルが追加されると、サムネイルが自動的に作成されます。 対応するボタンを使用してスケッチを個別に更新する機能も再び機能し始めました。 実際、まだデバッグと改善が必要な作業が多くあり、ミスもありますが、正しい方向への第一歩を踏み出し、そこで止まらないように計画しています。



既知の問題



代替のFFMpegライブラリはブラウザによって選択されなくなったため、次のバージョンで修正する予定です。



今日は以上です。 以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
