Bodo Schaefferからの財政的独立への道（パート1）

Milfgardの このリストの有名な本のメモを書き続けます。 今日、それはボド・シェーファーによる非常に有名な本「金融の独立への道」です。



著者は、個人的な成功とお金が一般的にどのように機能するかを語っています。







この本は、最初の100万（ドル）を稼ぐ4つの方法を説明しています：

  • 収入の一定の割合を節約する
  • あなたはお金を節約しています
  • あなたは収入を増やします
  • 達成された収入の増加ごとに一定の割合を節約します


本に記載されている原則の適用を開始してから7年後に、最初の100万を獲得できます。



私たちは本当に何が欲しいですか？



お金は、常に私たちがそれに結び付ける意味を正確に持っています。 緊張した財政状況は、ストレスのあるお金について考えさせます。 それは悪循環です。人々はこのビジネスでお金を稼ぐ方法を知らないので、人々が彼らを喜ばせることをしません。



解決策は次のとおりです。お気に入りの趣味を取り入れて、その上でキャリアを築きます。 ただし、これは、何が本当にあなたに喜びをもたらし、どの分野で才能があるのか​​を時間をかけて分析する場合にのみ機能します。 裕福な人々はこう言います：「一日中働いている人は誰でもお金を稼ぐ時間がない。」 明らかに、考える時間を与える必要があります。 「自分を知り、何があなたを喜ばせるのかを見つけてください。 そして、どうやってお金を稼ぐことができるかを考えてください。 毎日この質問を自問し、毎日最良の答えを探してください。



人生を最小化するのではなく 、 最適化することが重要です。 危機の時代の多くの企業は最適化という用語を採用しましたが、通常これはすべてのアイテムのコストを単純に削減することを意味します。 最適化は、時間、機会、才能、お金、従業員などのリソースの最も効率的な使用を目的とした意図的なプロセスです。



計画：自分で人生を計画しなければ、誰かがあなたのためにそれをするということです。 これはあなたの雇用主、顧客、親族または税務署であるかもしれません。 結果があなたを驚かせる前に、計画を立てて、いくつかのことを事前に行う方が良いです。



独立の4つの柱：

  1. 夢。 あなたの時間とお金が制限されていなかったらどうしますか
  2. 目標 。 何になりたい、何をしたい、何を持ちたい
  3. 道徳的価値。 あなたにとって本当に重要なことと、なぜこれが重要なのか。
  4. 戦略 それを得るための十分な知識と能力を持っていますか。 何を持ちたいですか？


これらの点を定期的に見て、これらの質問に対する答えを考えてください。



責任



誰があなたを責め、あなたは力を与えます。 それは非常に簡単です：あなたが他の誰かがあなたの問題のせいにすることを決定した場合、他の誰かがあなたの人生をコントロールします。 責任を持ち、あなたの人生を管理します。 実際に何が起こるかがあなた次第ではない場合（一部の悪役が立っている車に衝突した場合）、起こっていることに対する反応を制御するだけです。



責任とは、最善の答えを出すことができることを意味します。 応答という言葉自体が責任を意味します。 英語で 言語はまったく同じです：責任=応答+能力。 紛争による紛争または戦争による挑発への対応は、最適な解決策ではありません。



一般的なスキームは次のとおりです。外部からの影響が発生します→回答→結果を得る。



しかし、多くの場合、人々は自分自身ではなく、遺伝子、親、環境などの外部要因の結果を非難します。



あなたの人生に責任を持ちなさい。 7年前の自分のことを思い出してください。趣味、目標、職業、財政状況は何でしたか。 そして、7年後にあなたが誰になりたいかを考えてください。



制御ゾーン。

あなたが直接コントロールし、あなたが影響を与えることができるものがあります。 しかし、あなたが興味を持っていることは他にもありますが、影響を与えることはできません。 これらは互いにネストされた2つの同心円であり、制御の円は関心の円よりも小さくなっています。







制御ゾーンは常に拡大しています。 子供の頃、私たちにはほとんどコントロールがありませんでしたが、年齢とともに何らかの影響が生じます。 そして、このゾーンは何のために拡大しているのですか？ -責任、知識、経験。 責任を負い、新しい知識を獲得し、異常な経験を積むとすぐに、コントロールゾーンが拡大します。 新しい異常な体験は、快適ゾーンを超えた場合にのみ得られます。



問題があると、コントロールゾーンが大きくなる可能性が高くなります。 したがって、彼らは私たちの関心のあるゾーンにありますが、私たちのコントロールゾーンの外側にあるため、問題です。 問題の各解決策は、制御ゾーンの拡張です。



百万は奇跡ですか？



100万が奇跡ではないように、5つのレベルで変革を行う必要があります。

  1. 彼らの状況への不満。 行動する必要があります。
  2. 目的の結果が得られません。 自問してください：「どうすれば最小限の労力で望ましい結果を達成できますか？」
  3. インスピレーションの例を見つけました 。 例を挙げる人がいると、生きて成功するのがずっと簡単になります。
  4. あなたは世界観を変えています。 変更点と世界はあなたとは異なるように見えます。
  5. あなたはあなた自身の性格を変え、あなた自身を見ます。 たとえば、売り手が自分を売り手と見なす場合、顧客を喜ばせる必要があります。 しかし、彼が自分を専門家だと考えるなら、クライアント自身も彼に興味を持っています。


奇跡は、第3、第4、および第5レベルで変更があった場合に発生し、これらの変更を準備できます。 本はこれに役立ちます（スピードリーディングのテクニックを確実に学習する必要があります）。 自分で読んで、成功した人々に読んで読んだ本のトップ5について聞いてください。 成功日記をつけてください 。 あなたの成果について毎日彼に手紙を書いてください。 ポジティブに焦点を合わせ、自分自身について書くのに十分な自分自身を考慮してください。 セミナーへの参加 ：専門能力開発のために支払う価格は、無知のために支払う価格よりもはるかに低いです。 1年に4回のワークショップという明確なルールから始めることができます。 チャンスをください。 本当にやりがいのあることは、私たちにとって非常に恐ろしいことです。 新しいステップを踏む前に十分な恐怖を感じない場合、このステップは十分に重要ではありません。



上に非常に単純なものをリストしていますが、単純なルールの体系的な実装に関しては、すでに困難になっています。



Bodo Schaefferは、収入の少ない人々の類似点を指摘しています。

  • 彼らは富が何であるかを決定することはできません（特定の数と欲望）
  • 彼らは富を目標にしない（大きな目標の代わりに小さな目標を設定する）
  • 彼らは富を必要としません（すべての目標は絶対に必要でなければなりません）
  • 彼らは立ち上がらない（決して、決して、決してあきらめない）
  • 彼らは責任を負いません
  • 彼らは110％生きる準備ができていません（彼らは彼らが食べるものに満足し、成長しません）
  • 彼らにはメンターがいません
  • 彼らは美徳よりも自分の弱点に焦点を合わせます


お金についてどう思いますか？



お金について考えていることを考える限り、持っているものが手に入ります。 状況を分析するには、次の質問に答えてみてください。

  1. あなたが望んでいたよりも多くのお金を使ったことは今までに起こりましたか？ なんで？
  2. 体重を減らそうとしたが失敗したことはありますか？ なんで？
  3. 保存することを約束しましたが、しませんでしたか？ なぜ
  4. 服を買わないように少なくとも3か月試したことがありますが、うまくいきませんでしたか？ なんで？
  5. 銀行口座を超えたことはありますか？ なんで？
  6. あなたは確固たる決定を下したが、それでも何もしなかったということは起こりましたか？ なんで？


著者は、最も重要な3つの信仰の教義を特定し、それらを書面で書き留めることを提案しています（選択可能なリストを提供しています）。 私は6点しか得られませんでしたが、その中で私にとって最も重要な3点を強調しました。

  1. お金は私の指から流れています
  2. お金は良いものを生み出す
  3. お金がすべてではない
  4. お金はたくさんの良いものをもたらします
  5. 貧困は悪い、惨め、悪い
  6. お金よりも重要なことがあります。


しかし今、これらの信念が私たちの生活にどのような影響を与えているのかを考える必要がありますか？ 私たちの信念は、私たちが最も時間を費やす人々によって大きく影響されます。 一緒に時間を過ごす3人のリストを作成し、彼らの経済的信念が何であるか、彼らがお金をどのように扱い、誰が私たちにアドバイスを与えるかを理解する必要があります。 基本的に ：あなたがなりたいと思っている人ではない人からのアドバイスは絶対に受けないでください。



Bodo Schaefferは、あなたが誰になりたいのか、何をしたいのか、何をしたいのかを考えることを提案します。a）7年後。 b）3年後。 次に、経済的信仰の教義がこれらの目標の達成を妨げるものについて考える必要があります。 個人的に、私は私の個人的な日記でこの練習をしました、そして、それは7年のリストと3年のリストの間に大きな違いがなかったことがわかりました。



負債



義務とは、私たちが今未来に生きたいときです。 まだ何も獲得していませんが、獲得していないという事実を利用したいと思います。 主な推奨事項は、消費者の借金をしないことです。 彼らは破壊的で愚かであり、働く意欲を破壊し、悪循環に陥ります。 一方、それ以外ではビジネスの急速な成長がほとんど不可能な2つの重要なサポートポイントがあります。それは、他の人と他の人のお金です。 したがって、事業を始めるための借金は必要ですが、彼ら自身の「必要」を提供するための借金は奴隷です。



Bodo Schaefferは、古代のバビロンで、すべての奴隷の2/3が、借金を集めるだけで、自分で奴隷になった例を示しています。 これらは普通の市民で、彼らは自分たちで保釈金を受け取り、結果として奴隷になりました。 著者は、この点について十分に詳細に説明し、借金を取り除く方法に関する包括的な推奨事項を示します。 原本を読んで、すぐに演習を行うことをお勧めします。



いくつかの推奨事項：他の人の話を聞いたり、共感を避けたり、常に現金を持っている必要があります。破産の適切な時期を選択することをお勧めします。



収入を増やす方法。



私は値するだけ稼いでいます。 110％で働く：8時間の仕事に対して給料が支払われた場合、10時間働いたように善を行います。 権利ではなく義務に興味を持ちます。 「あなたの国があなたのために何ができるかを考えないでください。あなたの国のために何ができるかを考えてください」（J.F. Kennedy）。



できるだけ早く、 すべてを遅滞なく実行します。 あなたの直観を信じて、間違いを恐れないでください。あなたの迅速な決断の51％が正しければ、あなたは金持ちになります。 自分のスピードでみんなを打つことでスポーツを作りましょう。 3分以内に手紙に答え、すぐに応答し、延期に我慢しないでください。



やる価値のあることはすべてうまくやるべきです。 電子メール、電話、会議室の椅子など 驚くべき成果だけが他の人を感動させるので、常に100％を配り、素晴らしいことをすべてしてください。



不可欠になります。 あなたの直接の責任ではないことに対しても責任を負います。 各企業には1人以上の従業員がいますが、それなしでは何もできません。 それらは不可欠です。 影響力の範囲を広げ、プロジェクトの組織に携わり、「会社は私です」というインストールを受け入れます。 ただし、すべてを自分で行う必要はありません。そうしないと、会社の奴隷になります。 責任を受け入れ、自分のタスクと権限を他の人に委任する準備をしてください。



問題を回避しないでください。 それらが発生した場合、それらを解決するために自分自身を呼び出します。 難しい課題が割り当てられたときにボランティア。



スペシャリストとしての地位を確立します。 他のすべての人ができることと同じくらいできるなら、あなたは他の誰よりも価値がありません。 あなたがすべてを好むとき、あなた自身は顧客を探す必要がありますが、専門家としての地位を確立したとき、クライアント自身があなたを探します。 スペシャリストと見なされるために何をしましたか？ 3年以内にあなたはあなたの分野の専門家になるという事実に基づいて決定を下してください。 今日、あなたが最高のカテゴリーの専門家であり、それに応じて行動することを想像してください。関連する文献を読んで、一流の専門家のように考えてください。 しかし、スペシャリストになるには、これに時間を費やす必要があります。 毎日のワークロードで教育を継続する時間を確保します。



起業家の給与。 あなたが起業家である場合、あなたの給料を支払い、あなた自身のお金から企業のお金を分けてください。 会社の売上高の悲観的な予測に基づいて金額を決定し、個人口座に振り込みます。 毎月の金額とボーナスに違いはありません。「10月に私は良い仕事をしたので、ボーナスを自分で支払います。」 これは間違ったアプローチであり、個人の財政のたるみ、そして最悪の場合は負債につながります。



すべてのお金を会社に入れないでください。 これは多くの人が犯す間違いです。彼らは会社以外の状態を作り出しません。 本当にあなたの会社を発展させたいなら、他の人のお金を使ってください。 ローンでもローンでも構いませんが、消費者ローンではなく、ビジネスローンです。



収入の構成要素：

  1. プロレベル。 あなたはあなたの分野でどれくらい良いですか
  2. あなたのエネルギー。 あなたの職業上の成功にどれだけのエネルギーを費やすつもりですか？
  3. 開発。 あなたの製品やサービスに満足している人は何人ですか？
  4. 自尊心。 あなたはどれほど自信がありますか？ 他の人はあなたのことをよく考えていますか？ 彼らはあなたをスペシャリストとして評価していますか？
  5. アイデア。 あなたは創造的な人ですか？ すぐにアイデアを書き留めていますか？ あなたの個人的なアイデア工場はどのように発展しましたか？
