VMware ESX / ESXi 3.5 Update 5の出力

VMware ESXおよびVMware ESXi製品の3.5行目の5番目のアップデートがリリースされました。

すべての変更と修正は、リリースノートに記載されています。

-ESX 3.5 U5の場合-www.vmware.com/support/vi3/doc/vi3_esx35u5_rel_notes.html

-ESXi 3.5 U5の場合-www.vmware.com/support/vi3/doc/vi3_esx3i_i_35u5_rel_notes.html



特に、次のOSのサポートが追加されました。

-Windows 7 Enterprise（32ビットおよび64ビット）

-Windows 7 Ultimate（32ビットおよび64ビット）

-Windows 7 Professional（32ビットおよび64ビット）

-Windows 7 Home Premium（32ビットおよび64ビット）

-Windows 2008 R2 Standard Edition（64ビット）

-Windows 2008 R2 Enterprise Edition（64ビット）

-Windows 2008 R2 Datacenter Edition（64ビット）

-Windows 2008 R2 Webサーバー（64ビット）

-Ubuntu Desktop 9.04（32ビットおよび64ビット）

-Ubuntu Server 9.04（32ビットおよび64ビット）



このソフトウェアのライセンスを持っている人向けの配布ダウンロードページへのリンク：

-ESX 3.5 U5-downloads.vmware.com/d/details/esx350u5/dGViZGp0cGJkZXBo

-ESXi 3.5 U5-downloads.vmware.com/d/details/esx_3i_installableu5/dGViZGp3dGJkZXBo



ESXi 3.5 U5の無料バージョンのダウンロードページへのリンク（登録が必要）：

www.vmware.com/tryvmware/?p=esxi&lp=1



ESXi 4.0とは異なり、ESXi 3.5は32ビットカーネルに基づいていることを思い出させてください。

比較的古いハードウェアを仮想化サーバーに変えるために使用できます。

64ビットプロセッサを使用する場合は、ESXi 4.0を使用する必要があります。



ESXi 4.0 U1の無料バージョンのダウンロードページへのリンク（登録が必要）：

www.vmware.com/tryvmware/index.php?p=free-esxi&lp=1
