あなたのウェブカメラは新しいFirefox 53で動作しますか？







Firefox 53の最近の更新では、一部のWebカメラのサポートがなくなっているか、WebRTC APIに何らかの変化があった可能性があります。 わかった。 テストでこれに気付きました。 Firefox 52でテストに関係するすべてのWebカメラからビデオが正しくキャプチャされた場合、FF 53では一部のカメラ、つまり組み込みカメラが抜け落ちました。 このバグは、Windows 7およびWindows 8の内蔵カメラでのみ再生されることに注意してください。Windows10ではUSBカメラでは再生されず、Mac OSではまったく再生されません。



FFコンソールでは、バグは次のようになります。 



MediaStreamError { name: "InternalError", message: "Starting video failed", constraint: "", stack: "" }

テスト結果は、バグがChromeとFirefox 52では再現されず、8台のカメラのうち3台で再現されることを示しています。
