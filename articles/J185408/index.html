新しいIntel Perceptual Computing SDKがリリースされました



インテルは、Intel Perceptual Computing GOLD SDK R2の新バージョンのリリースを発表しました。これは、ジェスチャ制御、顔および音声認識、および「ヒューマン」インターフェイスのその他の属性を使用するアプリケーションを開発するためのツールです。 新しいリリースコンポーネントの中で：

  • Havok Vision SDKのサンプルとトレーニング資料
  • その他のC＃ドキュメント
  • Javaアプリケーション開発サポート


Intel WebサイトのIntel Perceptual Computing SDK 2013セクションでは、より詳細な製品情報を見つけたり、さまざまなトレーニングやその他の資料（もちろんSDK自体を含む）をダウンロードしたりできます。

結論としては、Perceptual Computing SDKの機能を示す短いビデオです。これは、実際には現時点で本格的なIntel Perceptual Computing Challengeのプロモーションビデオです。
