DotNext + SpbDotNet + MskDotNet





DotNetコミュニティは再び連絡を取り合っています。 .NETコミュニティはまだ生きており、開発中であることをお知らせします！ 誰もがすでに.NETコミュニティについて知っているので、ツリーに考えを広めることはしませんが、すぐにビジネスに取り掛かりましょう！



MskDotNetとSpbDotNetの新しいイベントを発表する前に、兄であるDotNextと地域コミュニティとの関係について少しお話ししたいと思います。 行こう！



DotNext、モスクワ2017



最近、DotNextは2日間の会議の実施を開始しました。次の会議は、モスクワで、伝統的に、Slavyanskaya Radissonホテルで開催されることも例外ではありません。 最もハードコアな.NET会議は、11月12〜13日に開催されます。







このプログラムは、いつものように高密度で恒星です。 スピーカーは、アンドレイ・アキンシン、ディノ・エスポジート（DotNextでレギュラーになった）、ゴールドスタイン一家、そして多くの同様に有名な人々です。 また、今回は自分自身と会うことができると言いたいと思います... Mark Siman、長い間DotNextにドラッグできませんでした。 Markのブログと有名な.NETのDependency Injectionのすべてのファンには、必ず会議に参加してサインをもらいましょう！



MskDotNetミーティングの目標の1つは、全員がステージに参加してレポートの準備を支援することです。 非常に多くのクールな開発者がオフィスに座っており、さまざまな理由で外に出ません。 もちろん、そこからグローバルコミュニティは苦しんでいます。 クールなスピーカーによって伝えられる実際の経験は、その重みに見合うだけの価値があります。これはどこにも描画できません。または、このためには、大量の文献やその他の情報源を検索する必要があります。



すべてのスピーカーが受ける個別のボーナスは、人前で話すスキルであり、これは人生で非常に役立ちます。 その後、ローカルユーザーグループのどのスピーカーがDotNextで発言したかを思い出してみましょう。



DotNextのMskDotNetおよびSpbDotNetスピーカー









地元のコミュニティからの講演者がDotNextで繰り返し話しました。 それとは別に、Julia TsisykとNikita TsukanovをMskDotNetのオーガナイザーとして、Mikhail ShcherbakovとAnatoly KulakovをSpbDotNetのオーガナイザーとして区別できます。 それらはすべて、DotNextで繰り返し実行されました。 Mikhail Shcherbakovは、セキュリティレポートでDotNextで常に確認できます。 モスクワで、DotNextはトピック「ASP.NET Core：Attack Prevention Mechanisms 2.0」について講演します。



  • Elizaveta Golenok は 、MskDotNetでMonoアプリケーションのデバッグに関するトピックについて講演した後、DotNextで講演します。 彼女の成功を祈ります！

  • Anatoly KulakovはSpbDotNetでStructured LoggingとRoslyn Code Analysisのレポートで講演し、最も近いモスクワでDotNext は実行中のアプリケーションのメトリックを収集するプレゼンテーションを行います。

  • ローマン・ネボリンは有名な講演者であり、F＃とC＃の両方で機能主義についてよく話します。 彼はSpbDotNetとDotNextで何度も演奏しています。 11月12日に、Roman は F＃でタイププロバイダーについて講演します。

  • Vladimir Kochetkov は SpbDotNet のアプリケーションセキュリティ理論について語り 、すぐにモスクワのDotNextで、さまざまな文法の言語でのインジェクション攻撃に対する.NETアプリケーションの脆弱性の問題について話します。

  • 2017年2月、Valery PetrovはSpbDotNetで.NETメモリモデルに関する講演を行いました。 そして、近い将来、DotNext はこのトピックの開発を続けます。





MskDotNetのDotNextスピーカー



スピーカーは、DotNextを使用して地域コミュニティに来ることがあります。







有名なAndrei Akinshinは、MskDotNetとSpbDotNetで、メモリ、ベンチマーク、その他のグッズに関するさまざまな筋金入りのプレゼンテーションを行いました。 SpbDotNetで、Dmitry Nesterukは繰り返し行動しました。 Adam SitnikはMskDotNetで講演し、.NETおよびYegor Bogatovのパフォーマンス関連の機能について、Holo Lenseに関するレポートで語りました。



もちろんこれについては、誰もやめようとはしていません！ SpbDotNetとMskDotNetの両方で新しいゲストが期待されているため、指を離さないでください。 誰がいついつ-カードを公開するまで。 ニュースをフォローしてください。 それまでの間、MskDotNetの次の会議の発表を見てみましょう。
