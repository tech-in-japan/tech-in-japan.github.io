本「Microsoft Visual C＃。 詳細ガイド。 第8版

こんにちは、habrozhiteli！ 最近、ジョン・シャープの本を翻訳しました。



この本は誰のためですか？



この本の読者は、Visual Studio 2015および.NET Frameworkバージョン4.6を使用したC＃プログラミングの基礎を学びたい開発者であると想定されています。 この本を読んだ後、C＃言語を完全に理解し、それを使用して、Windows 10オペレーティングシステムを実行できる適応性のある広くスケーラブルなアプリケーションを作成できます。



この本を読むべきではない人



この本は、以前にC＃言語を使用していた開発者を対象としており、プログラミングの完全な初心者を対象としていません。 したがって、その資料は主にC＃言語の使用に基づいています。 この本は、ADO.NET、ASP.NET、Windows Communication Foundation、Windows Workflow Foundationなど、Windows上で実行されるエンタープライズレベルのアプリケーションを作成するために利用できる多くの技術の詳細な説明を目的としていません。



本の構造



本は4つの部分に分かれています。

‰

パートI、「Microsoft Visual C＃とMicrosoft Visual Studio 2015の紹介」は、Visual Studioプログラミング環境で作業するための基本的なC＃構文とテクニックの紹介です。

‰‰

パートII「C＃オブジェクトモデルの基本情報」では、C＃での新しいタイプの作成と管理の詳細、およびこれらのタイプによって参照されるリソースの管理方法について説明します。

‰‰

パートIII「C＃での拡張可能型の定義」には、C＃が提供する要素の詳細な説明が含まれており、複数のアプリケーションで再利用可能な型を作成できます。

‰‰

パートIV「C＃を使用したユニバーサルWindowsプラットフォームアプリケーションの作成」では、ユニバーサルWindows 10プログラミングモデルと、C＃を使用してこの新しいモデルのインタラクティブアプリケーションを作成する方法について説明します。



この本は、多くの重要な分野でスキルを習得するのに役立つように書かれています。 初心者プログラマーと、C、C ++、Java、Visual Basicなどの他のプログラミング言語からC＃に切り替えるユーザーの両方に役立ちます。 本の多くの章には、実際に学習した概念を実際に検証することを可能にする実用的な例が含まれています。 ただし、どこに焦点を合わせても、システムにサンプルアプリケーションをダウンロードしてインストールする必要があります。



著者について



John Sharpは、ソフトウェア開発およびアドバイザリー会社であるCM Group Ltdのシニアスペシャリストです。 シャープは、これらの両方の分野の専門家です。 彼は多数の出版物の著者であり、30年の経験を持つ教師です。 彼はPascalでCP / Mオペレーティングシステム用にプログラミングし、さまざまなバージョンのUNIXオペレーティングシステム用のC / Oracleアプリケーションを開発し、C＃およびJavaScriptで記述された分散アプリケーションを開発し、最後にWindows 10およびMicrosoft Azure用のアプリケーションを開発しました。 Johnは、Microsoft .NET Frameworkを使用してアプリケーションを作成する主要な専門家であり、「Windows Communication Foundation 4 Step By Step」（Microsoft Press）の著者でもあります。
