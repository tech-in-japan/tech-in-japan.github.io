Windows 8は芸術作品です。 しかし、それはLinuxではありません

英語ネットワークの広大さの中で、私は通常Linuxに座っている人の1人がWindows 8を使用した経験を見つけました。 その投稿からの印象をあなたと共有したいです。 いいえ、Windowsでの永遠の襲撃、ベンデカピアンの予言、その他の無駄はありません。 すべてがとても甘いです。



だから



先日、Windows 8をインストールしました。 教えましょう-かっこいいです。



いいえ、皮肉なし-Windows 8は間違いなく成功しました。



すぐにロードでき、優れたインターフェースです。ラップトップの口histを箱から取り出してすぐに、最後の通知ライトであるアプリケーションがすぐに起動するまでキャッチしました。 くそー、私はファイルコピーダイアログも気に入っていました。



真剣に、8番目のWindowsのコピーダイアログはきれいです。 まだ見たことがないが、クールなインターフェイスソリューションが好きなら、ぜひチェックしてください。 マイクロソフトのスタッフが私に感銘を与えたと言うことは、何も言わないことです。



しかし、私はまだLinuxに戻ります。 なんで？ Linuxについて、別のOSを心から楽しんでいるのに、それでも元に戻したいのは何ですか？ Windows 8をすべてのコンピューターにインストールして、幸せを感じてみませんか？



昨日、私は数時間トラフィックで立ち往生しました、そして、私はそれについて多くを考えました-Linuxに正確に私を引き付けるものは何ですか？



Linuxはオープンソースですか？ いいえ、そうでもありません。 いいえ、オープンソースは素晴らしいですが、それがLinuxを使用する理由ではありません。 デスクトップPCでも、クローズドコードを作成します（私は開発者です）。



たぶん便利なパッケージ管理？ 必要なものは何でも、この簡単なapt-getインストールをすばやく見つけてインストールできます。 いいえ、そうでもありません。 ただし、パッケージ管理とリポジトリはLinuxにとって大きな勝利です。



セキュリティ 異なる分布？ クールなデスクトップ？ いや。 これらはすべて素晴らしいことですが、これは私をLinuxに引き付けるものとは異なります。



それから私は夜明けしました。 これが気に入っています：右クリック-パネルを削除します。



この可能性自体は、必要なものだけが残り、それ以上何も残らないまで、インターフェイスから不要な要素を徐々に削除します。 つまり、ただそこに座って画面を見て、気に入らなかったので右クリックして、「ちょっと、thing_on_my_worker_table！」 もう必要ない！ 無料！」



これらすべてのパネル、バー、ドック、ウィジェット-これらはすべて自由に削除できます。 シンプルでいつでも。



これは本当にLinuxの設計哲学の一部であるように思えます。 はい、Linuxソフトウェア。 「好きじゃない？ だから削除！」

それが大好きです。 本当に、本当に大好きです。



それがどれだけクールかを証明してほしいですか？ 右クリックして、Windows 8でパネルを削除してみてください。

しかし、普通のユーザーとして、私はどういうわけか自分でLinuxをインストールしました。 一般的な開発用。 そして、外国の著者によるメモを読んで、私は彼を理解しています。 最初は、WindowsからLinuxに移行したばかりの人と同じように、私にとっては珍しいことでしたが、そのようなチップに夢中になりました。 また、リポジトリもあります。 素晴らしいこと。
