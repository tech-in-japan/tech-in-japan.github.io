コンテスト「ノーザンライツ」に参加する

Intel IT GalaxyとMicrosoft VirtualAcademyの共同プロジェクトである新しいコンテスト「 Server Lights 」にご参加ください。 新しいテクノロジーを学び、経験を共有すると、Intel IT GalaxyとMVAから特別賞を獲得できます。







競技期間：4月22日から5月14日まで。 受賞者は基準に従って選択され、5月22日まで発表されます。



コンテストに参加するには：

1.無料のMVA教育ポータルでSystem Center SP1 コースを受講します（ 登録が必要です）。

2.更新、ハードウェアおよびソフトウェアソリューションの選択、新しいプラットフォームへの移行の前提条件のいずれかのトピックで実際の問題を解決した経験について、 IT Galaxyのブログ （ 登録が必要）で伝えます。



投稿を書く前に、IT Galaxyブログの投稿要件をお読みください。



賞品



1位： MVAコースを修了した最高のブログの著者は、3000ポイントのIntel IT Galaxyプログラムを受け取り、提供された賞品と交換したり、以前に蓄積した賞品に接続したりできます。 また、MVAからの特別賞-「サーバー」ポールで耳を温めるゼンハイザーHD 518ヘッドフォン。

2位：マイクロソフトのマウスとキーボード、およびIntel IT Galaxyの管理者タンバリン。

3位： MVAのTシャツとIntel ITGalaxyのマグ。



すべての前提条件を満たしたすべてのコンテスト参加者は、インテルIT Galaxyプログラムの500ポイントとMVAからの100アワードポイントを受け取ります。
