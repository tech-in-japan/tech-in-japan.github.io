Raspberry Pi開発者が小学生向けのプログラミングコンテストを発表

正直なところ、このニュースがHabrvsk-Raspberry Piの愛好家全員を考慮してHabrから渡された理由はわかりません。



次のニュースの 7日に、Raspberry Piの公式Webサイトに公開されました。



シェフィールドのゲームブリタニアで、本日、子供と若者向けのプログラミングコンペティションの開始を発表しました。 時間が経てば、毎週このような競技会を開催する予定ですが、これは北半球の学校の休みに合わせて8週間続きます。



要約：


  • 2つの年齢カテゴリ：13歳以下、および14歳から18歳。
  • 各カテゴリの2位には1位の賞金1,000ドルと2位の賞金200ドル。
  • 目標は簡単です。RaspberryPi用に作成したプログラムで審査員に感銘を与えてください。




目標は簡単です。RaspberryPi用に作成したプログラムで審査員に感銘を与えてください。 Raspberry Piをまだお持ちでない場合でも参加できます。プラットフォームのパブリックエミュレーターのいずれかを使用するだけです。



詳細な用語：


スポンサー



スポンサーは、イギリス、CB23 7NU、カルデコテ、ミッチェル・ウッド・ハウスのラズベリーパイ財団です。



規約



2012 Raspberry Pi Foundation Summer Programming Contestは、2012年7月7日9:00:00 UTCに始まり、2012年9月1日9:00:00 UTCに終了します。 申請書を提出することにより、各参加者（または必要に応じて参加者の両親または保護者）はここに提示された公式規則に同意し、公式規則に定められたすべての要件に従って競技への参加が尊重されることを保証します。 これはスキルに基づく競争であり、勝者を決定する際にチャンスは何の役割も果たしません。



誰が参加できるか



コンペティションは、すでに18歳であるか、コンペティションの開始日にまだ参加しておらず、その日にフルタイムで勉強している人にのみ開かれます。 スポンサーとそのディストリビューターの従業員、およびその近親者（住んでいる場所に関係なく、配偶者、両親、子供、兄弟とその配偶者）、またはそれらに関連していない従業員の家に居住する人は、参加する資格がありません。 禁止事項に違反した場合、競争は有効です。



参加方法



競争の目的は、Raspberry Piコンピューターで実行され、審査員に感銘を与えるアプリケーションを作成することです。 各アプリケーションは、アプリケーションフォームと、アプリケーションのすべてのソースコード、ファイル、データを含むアーカイブファイルで構成されます。 参加するには、2012年8月4日以降にwww.raspberrypi.orgにアクセスしてください。 アプリケーションに記入し、アーカイブとともに提出します。



参加者のカテゴリ：（1）13歳以下、および（2）14〜18歳。参加者は、コンテストの開始日にカテゴリを指定し、このカテゴリの年齢要件を満たしている必要があります。 1人の参加者からのアプリケーションの数に制限はありません。



アプリケーションは、参加者が作成したオリジナルの資料から完全に作成する必要があり、個人または法人の著作権、人格権または知的財産権を侵害してはなりません。



賞と審査



各カテゴリの勝者には1,000ドルが贈られます。 各カテゴリで最大5人の受賞者に200ドルが贈られます。 審査員は自由裁量で参加者から勝者と賞受賞者を選択します。審査員の決定は最終的なものであり、競技に関連するすべての問題を拘束します。



スポンサーは、アプリケーションおよび/または参加者（参加者の年齢、身元、住所、および作成者を含む）の信and性と独創性を検証し、これらの公式ルールの遵守。



ライセンス



コンテストに参加することにより、すべての参加者は、アプリケーションを複製、配布、および表示するための取消不能、永久、無償、世界規模の非独占的なスポンサーライセンスを提供します。



責任の制限



コンテストに参加することにより、参加者（または必要に応じて参加者の両親または保護者）は、スポンサーとそのパートナー、支店、子会社、広告代理店、エージェントとその従業員、マネージャー、ディレクター、および代理人をあらゆる請求から解放し保護することに同意します。この競争への参加から生じる損失および損失。



条件



このコンペティションはイングランドとウェールズの法律の対象となり、それらに従って開催されます。



何らかの理由で、スポンサーの制御を超えた何らかの理由で計画されているように、競技が機能しない場合、スポンサーはその裁量で競技をキャンセル、中断、または中断する権利を留保します。 スポンサーは、その裁量により、競技中いつでも公式ルールを変更する権利を留保します。
