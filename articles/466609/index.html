Algorithm advantages



  1. There is no C2 server, calls go to blockchain browsers.
  2. Commands for EBS are unconditionally trusted.
  3. The medium for sending commands is blockchain, which does not allow blocking the sending / sending of commands.
  4. Blockchain is an extremely stable system (this does not apply to cryptocurrency volatility).
  5. Low cost of providing work.
  6. Simple implementation of the algorithm (open source data and code).
  7. No need to generate transactions or download blockchain.


Algorithm Features



  1. Simplex communication.
  2. Dependence on the stability of the browsers.
  3. Certain (albeit minimal) financial costs.
