25ドルのRaspberry Piが米国で販売開始





Raspberry Pi mini-PCは、これまで以上に人気があります。 このミニチュアコンピューターに基づいて、非常に興味深いシステムが作成され、その一部はHabréで説明されています。 先日、これらのデバイスのファンに朗報がありました。事実は、Raspberry Pi Model Aが米国で販売されたことです。 確かに、再販業者alliedelecは数時間以内に利用可能なすべてのコピーを分解したので、Raspberry Pi Model Aの新しいバッチを待つ必要があります。



Raspberry Pi Model Aは、その兄であるRaspberry Pi Model Bとそれほど変わらないことを思い出してください。しかし、25ドルモデルにはイーサネットポートがなく、USBポートが1つとRAMの半分しかありません。 ただし、このモデルは、Raspberry PiモデルBよりも約30％少ないエネルギーを消費します。



以前は、Raspberry Pi Model Aがヨーロッパで販売されました（その年の2月）。 アジアでは、このモデルが先週登場し、現在、米国に住むHabrasocietyの代表者が自宅でこのデバイスを購入できます。 確かに、すでに述べたように、商品の新しいバッチを待つ必要があります。



開発者によると、1週間に数千のデバイスモデルが生産されているため、長く待つ必要はありません。 これまでにすでに20,000以上のRaspberry Pi Model Asが販売されています。



Raspberry Pi Model Bに関しては、販売された100万台のデバイスのマイルストーンが今年の1月に達成されました。
