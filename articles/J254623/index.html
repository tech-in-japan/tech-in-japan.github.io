Callbackhunter、UpToСall、RocketCallback、HookMyVisit、およびLeadhackerコールバックサービスの比較

「 X秒以内にご連絡します！ 」 ほとんどすべての人が、コールバックを提供する特定のサイトのポップアップウィンドウに少なくとも一度遭遇しました。 このウィンドウは、「 コールバックウィジェット 」、「 コールバックウィジェット 」、「 インスタントコールフォーム 」、さらには「 リードジェネレーター 」という異なる名前で呼び出されます 。 そのようなウィジェット（20以上）を提供する多くのサービスがあり、それらは引き続き表示されます。









この技術の有効性については好きなだけ議論できますが、非常に人気があり、安定した需要があるという事実は残っています。



人気ですか？ ほんと？
はい、そうです。 たとえば、1か月ほどで、プロジェクトが開始されてから、ウィジェットが 1000以上のサイトにインストールされました。


ウェブサイトに同様のウィジェットをインストールすることにしたと想像してみてください。 同様のサービスを提供する豊富なサービスのために、少なくとも2つの質問が発生します。どちらを使用するか、選択する際に正確に何に従うべきか、という質問です。 実際、このため、私たちは小さな比較レビューを行うことにしました。



したがって、比較のために、5つのサービスを選択しました。



  1. UpToCall 。
  2. RocketCallback 。
  3. コールバックハンター
  4. HookMyVisit 。
  5. Leadhacker 。


次の特性によってサービスを評価します。



  1. ウィジェットの外観 。 カスタマイズの可能性：写真、配色など
  2. ウィジェット表示アルゴリズム 。 独自の基準に従ってインプレッションをカスタマイズします。
  3. 通話料金 。


なぜこれらの特性とサービスが正確なのですか？
統計によると、ユーザーの70％以上がウィジェットの動作とコンテンツを制御したいので、これらの特性を採用しました。 そして、価格の問題はほぼ最初の場所にあります。 サービスの選択は、入力することで行われました。


無料の関税（UpToCallを除く）でウィジェットをテストしましたが、一部の機能は見落としていたようです。 それでは始めましょう！

部分的にはどういう意味ですか？
たとえば、ウィジェットの色を変更できても、角の半径を変更できない場合。


表からわかるように、部外者は、カスタマイズに関して最も最小限の機能を備えたCallbackhunterとHookMyVisitです。 Callbackhunterが次のテキストの変更を発表することは注目に値します：

「私たちは、200万人を超える視聴者を持つ数千のサイトでA / Bテストを使用してテキストを磨きました。 したがって、現在のテキストは訪問者をベルに変換するのに最適です。」


テキスト編集へのアクセスを提供しない他のサービスには、同様のものがある可能性があります。



フリーテキストの編集とスタイルの欠点は、ユーザーがテキストにいくつかのタイプミスをしたり、テキストコンポーネントを追加しすぎたりすることがあり、ウィジェットの見栄えに影響することです。



モバイル版にはマイナスもあります-サイトがモバイルデバイスに適合していない場合、ウィジェットの表示は多少曲がります。

HookMyVisitとLeadhackerはディスプレイ設定に遅れをとっています。 詳細な表示設定にも大きなマイナスがあります。ユーザーは最適な値を設定できません。その結果、変換が非常に低くなる可能性があります。



たとえば、平均訪問時間が2分間の場合、ユーザーは2.5分後に表示するようにウィジェットを設定できます。これにより、ほとんどの訪問者にはウィジェットが表示されなくなります。

UpToCall通話コストの明確なリーダーであり、RocketCallbackがそれに続きます。 コールバックハンターはコストが高いことで際立っていますが、通話時間ではなく通話料金を支払うことに注意してください。クライアントとの平均通話時間が16分を超える場合、彼がリーダーになります。



1分間のコストを計算するとき、最も高価な料金を採用しました。 ドルでの価格は、1ドル= 60ルーブルのレートでルーブルに変換されました。

1位 -UpToCall（10.5ポイント）。

3位 -Leadhacker（5ポイント）。

4位 -HookMyVisit（2.5ポイント）。



比較から次の結論を引き出すことができます。



  1. 合理的な通話コストで制御されたウィジェットが必要な場合、選択肢はUpToCallになります。
  2. サイトからの呼び出しが少ないが、それらが長い（16分以上）場合、 Callbackhunterは間違いなく最良の選択です。
  3. UpToCallとCallbackhunterの間に何かが必要な場合は、 RocketCallbackとLeadhackerが急いで支援します。
  4. ミニマリストにとって、最良の選択はHookMyVisitです。




各サービスに独自のチップが含まれており、レビューに反映されていないため、評価が完全に正しいわけではないことを理解しています。
