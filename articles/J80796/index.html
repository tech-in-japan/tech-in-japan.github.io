タイミングのためのDojo Control

タイミングのためにDojoモジュールを共有したい。 私たちの研究所での研究中にこのコントロールが必要でした。 コントロールのタスクは、タイミングのための明確で高速なインターフェースを学生に提供することでした。











最初に、あなたの人生を簡潔に簡素化し、時間と分を選択できる2つのドロップダウンリストの形式でHTMLハードコードを作成したいという要望がありました。

1.より美しく、便利なインターフェイスを作成する機能

2.現在の時刻を設定/取得するためのこのようなコンポーネントのより便利な使用

3. JavaScriptを使用してコンポーネントの新しいインスタンスを簡単に作成する機能、およびこのコンポーネントの完全な制御

4.優れたトラフィックの節約

それでは始めましょう。 コンポーネントは何をすべきですか：

1.次の形式で時間を受け入れます：H：M、分数（たとえば、125 = 2：05）

2.時間を選択するときに、非表示フィールドに入力して、フォームをサーバーに送信するときに時間を取得できるようにします

3. JavaScriptを使用する-コンポーネントの現在時刻を設定/取得するための関数



開発プロセス全体を検討してください。



1. Dojoがインストールおよび構成されます。 最初に、Dojoが正しくインストールおよび構成されていることを確認してください。 この記事では、インストールと構成のプロセスについては詳しく説明しません。このプロセスはかなりよく文書化されているからです。 Dojoが/ js / dojo / dojoフォルダーにインストールされているとします。

2.コンポーネントと連携するようにDojoを構成します。 Dojoと競合しない別のディレクトリにコンポーネントを配置します。独自のDojoコンポーネントについては、フォルダー/ js / dojo / switlleを作成します

ここで、このフォルダーからコンポーネントを使用するには、このパスを登録する必要があります。 Dojoを接続した直後にこれを行います。

最初の行に入力タグが含まれ、2番目のスパンに含まれるという事実に注意を払ってはなりません。 タグタグの名前は絶対に再生されません。とにかく、このデザインはテンプレートコンテンツに置き換えられます。 ただし、timeminとtimeの2つの異なる属性により、異なるセッターが発生します。 timeminでは、分数を渡し、時間では、時間の文字列表現を渡します。
