Virtual Movable Type、主なものについて簡単に

Virtual Movable Typeは、人気のある出版プラットフォームの高速で便利な作業アセンブリ向けにパッケージ化および最適化されています。 このアセンブリには以下が含まれます。

  • 可動式
  • Movable Type Action Streamsのプラグイン
  • Linux
  • アパッチ
  • MySQL
  • Memcache（最初は無効）
  • Fastcgi
  • Imagemagick
  • 必要な追加のPerlモジュール


このアセンブリは、 JumpBoxと組み合わせて開発され、次の仮想マシンで実行されます。

  • Parallels
  • VMware
  • Amazon EC2
  • Microsoft仮想サーバー


誰に、そしてなぜこの獣は役に立つのでしょうか？



この製品は、Movable Typeの詳細を知りたい初心者から、テンプレートで多くの作業を行い、すばやく編集してテストする必要があるプロのユーザーにとって、誰にとっても便利です。



この製品の大きなマイナス点は、仮想マシンで実行されることです。仮想マシン自体は既にコンピューターから多くのリソースを消費しています。 したがって、弱いコンピューターに置くことはお勧めしません。 その大きなプラスは、作業の速度と、完全に調整された、エラーのないエンジンです。 デンワーへのインストール中に、インストール中とインストール後の両方で多くの問題が発生する可能性があるとします。 また、Virtual Movable Typeでは、これに該当することはありません。
