W3Cは要素をHTML5に戻しました

HTML5仕様エディターであるIan Hickson が広く使用されている<time>要素を削除するという最近の決定（ bug 13240 ）は、多くの開発者にとって困惑させていました。 W3Cコンソーシアムの代表者も同じ感情を経験しました。 ヒクソンの物議をかもした決定の数日後、彼らは現状を回復した。



Microsoft CanadaのPaul Cottonは、W3Cを代表して、誤解のために全体の状況が生じたことを保証します。 何らかの理由で、悪名高い「13240バグ」に関する情報がHTML WGワーキンググループにタイムリーに届きませんでした（明らかに、トラッカーで不正なBugzillaコンポーネントを使用したため）。 情報が受信されなかったため、これは手順に従って決定を取り消すための基礎となります 。



W3Cの決定により、仕様編集者のIan Hicksonは2011年11月8日までに必要な変更を行う必要があります。それ以外の場合、W3Cスタッフは独自にこれを行います。 Bug 13240はLast Call WGステータスに移行されるため、将来、仕様エディターはワーキンググループをバイパスして決定することはできなくなります。 この問題は、 HTML WG参加者の個人的な会議でも議論されます。
