最初のVisual Studio 11トレーニングコースがリリースされました



Visual Studio 11 Developer Previewチュートリアルの最初のバージョンが本日リリースされました。 このコースには、Visual Studio 11および.NET Framework 4.5に含まれる革新​​、TFS 11でのアプリケーションライフサイクルの管理方法、およびWindowsの将来のバージョン用のMetroスタイルアプリケーションの作成方法を理解するのに役立つラボが含まれています。





コースはwww.microsoft.com/download/en/details.aspx?id=27738からダウンロードできます。



コースに含まれるもの：



Visual Studio開発環境：

  • Visual Studio 11プレビュー


言語

  • C＃およびVisual Basicでの.NET 4.5の非同期プログラミング


ウェブ：

  • ASP.NETおよびVisual Studio 11 Previewの新機能
  • ASP.NETおよびWeb Forms 4.5の新機能
  • WCF Web APIを使用したRESTful APIの作成


アプリケーションライフサイクル管理：

  • 適切なソフトウェアの作成：ストーリーの生成とVisual Studio 11を使用した利害関係者のレビューの収集
  • Team Foundation Server 11の柔軟なプロジェクト管理
  • Team Foundation Server 11でプログラマの生産性を高める
  • IntelliTraceとVisual Studio 11を使用して、運用環境でアプリケーションの問題を診断する
  • Microsoft Test Manager 11の研究テストとその他の革新
  • Visual Studio 11の単体テスト：MSTest、NUnit、xUnit.net、およびCode Clone


Windows Metro：

  • BUILDカンファレンスのWindows 8 Developer Preview用のラボ構築アプリケーション。 注：コースにはwww.buildwindows.com/labsへのリンクが含まれており、実験室の作業自体は含まれていません。




go.microsoft.com/?linkid=9779649でコースをダウンロードできます。 注意深く見ると、2つのファイルが表示されます。 37 MBファイル（VS11TrainingKitOctober2001.Setup.exe）には、すべてのコンテンツとラボが含まれています。 2 MBファイル（VS11TK_WebInstaller_Preview.exe）には、コンテンツインストーラーアプリケーションが含まれています。 起動時にこのアプリケーションを使用すると、すべてのコンテンツをダウンロードするのではなく、特定のラボを選択できます。 また、将来的には、トレーニングコースのコンテンツを更新するときに、このアプリケーションを使用して新しい追加を確認してダウンロードする機会があります。



来月半は、新しい実験室作業を追加します。
