従業員向けのPKIユニフォームまたはソーシャルパッケージではありません

こんにちはHabrahabr！

当社は、情報セキュリティの分野におけるソリューションで最もよく知られています。 Rutoken電子識別子やGuardantソフトウェアセキュリティキーなどの製品を製造しています。

通常、私たちの記事は本質的に純粋に技術的なものですが、今回は、従業員の社会的利益に関する内部会計システムについて説明します。



質問の背景について一言。 私たちは現代のロシアのIT会社であり、他のIT会社と同様に、すべてを自動化することが大好きです。 とりわけ、当社では旅費、劇場旅行、フィットネスの補償があります。 長い間、従業員はチケットを収集して保管し、正しい日に会計部門に提供してからお金を受け取ることを余儀なくされました。 最終的に、この手順にうんざりしていたため、このプロセスを自動化するための最も簡単なソリューションを作成することにしました。 結果として得られたシステムを当社で正常に使用し、今日ではオープンソースに移行しています。



このシステムとそのソースは、誰でも利用できます 。 単にインストールして、会社で使用を開始できます。

便宜上、 ドキュメントを準備し、インストール可能なプロジェクトリリースをリリースしました。



システムとは何ですか：

システムには、従業員、秘書、会計士、管理者用のインターフェースがあります。

従業員のインターフェイスは、可能な限りシンプルで簡単です。 チケットまたは小切手を撮影し、ブラウザウィンドウに単純に転送する必要があります。





主に従業員が使用するモバイル版もあります。







使い方はとても簡単です。チケットを撮影し、フィールドに記入してボタンを押しました。







ダウンロード後、会計士は申請を承認し、月末に処理1（ダウンロードも可能）を使用して、従業員への支払い用のデータを給与とともにダウンロードします。



すぐに使用できるシステムには、メール通知システムであるActive Directoryとの統合があり、ユーザーを作成して制限を設定できます。 デフォルトでは、旅行、社交イベント、フィットネスの3つのカテゴリが作成されますが、独自のカテゴリを追加することを妨げるものはありません。

このシステムは、次のテクノロジーを使用して開発されています。

  • .Net Framework 4.5およびASP.NET MVC5
  • Entity Framework v.6 + MS SQL Server
  • HTML5 +ブートストラップ3
  • JQuery、AngularJS。




システムがユーザーの承認と認証にActive Directoryを使用するため、正しく動作するには、Windowsドメイン内のターゲット展開サーバーのメンバーシップが必要です。

展開パッケージを使用したシステムの初期インストール中に、データベースが作成され、必要なデータが入力されます。



これは最初の1つですが、これがオープンソースに移行する最後の製品とはほど遠いことを確信しています。 コミット、ブランチ、フォークを歓迎します。
