便利なUnityコマンド

Ubuntu 11.04は、ネジを打つための新しい興味深いコマンドを学ぶ絶好の機会を提供します。



起動の可能性を確認します（ハードウェアにバグがある場合は、launchpad.netでこれを行うように求めます）：

/usr/lib/nux/unity_support_test -p







何かがうまくいかなかった...



ccsmから単一設定を削除します（CompizConfig Settings Manage）：

unity --reset





このコマンドは、KDE ​​/ Qtイベントループをオンにしたときに役立ち、2秒ごとにcompizを再起動しました。 かなりループ。



左パネルで初期アイコンを設定します。

unity --reset-icons







すべての統一設定を削除します（ ターミナルからのみ 、便利なxtermから、それはgdmの回復コンソールでもあります）：

gconftool-2 --recursive-unset /apps/compiz-1

unity --reset







トレイにあるお気に入りのPulseAudio Device Chooser



このすばらしいユーティリティのメニューに入る他の方法は知らないので、次を実行する必要があります。

gsettings get com.canonical.Unity.Panel systray-whitelist

['JavaEmbeddedFrame', 'Mumble', 'Wine', 'Skype', 'hp-systray']







そして、「padevchooser」（関連）と「deluge-gtk」（関連なし）を追加します。

gsettings set com.canonical.Unity.Panel systray-whitelist "['JavaEmbeddedFrame', 'Mumble', 'Wine', 'Skype', 'hp-systray ', 'padevchooser', 'deluge-gtk' ]"







gdmのデフォルトセッション


ファンが/ usr / share / xsessions /にショートカットを作成するため。

背景：長い間、デフォルトのログインセッションを設定するには、/ usr / share / xsessions /にあるgnome.desktopを独自のものに置き換えるだけで十分でした。 gnome.desktopの名前をunity.desktopに単純に変更することはできないため（特別です）、解決策を探す必要がありました。



/etc/gdm/custom.confに次の行を追加します。

SessionDesktopDir=/usr/share/xsessions/





実行（xbmcの場合）：

sudo /usr/lib/gdm/gdm-set-default-session -d xbmc







このメカニズムは複数回変更されているため、このメカニズムが再び変更されるという仮定があります。
