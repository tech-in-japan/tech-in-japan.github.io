ロシアの教育分野における課題

記事では、教育の基本的な問題と、同時にこの問題を修正することの重要性を検討したいと思います。 いいえ、改定させてください。 そのような問題の存在を正当化し、解決策を提供したいと思います。 私たちは多くを変えることができると思います。











2つの仮定



1. 教育=>スペシャリスト

トレーニングが生産に必要な領域を考慮して実施された場合、それは良いでしょう。



2. スペシャリスト=>強力な経済

実際、適切な場所に有能な人が存在することは、私が思うに、本当に不足していることです。 時には、企業で人々がしていることを聞いて、不思議に思うことがあります。



問題は何ですか？





事実、VKontakteは勉強するより座るのが面白いということです。 WoWをプレイすることも興味深いです。 もちろん、自己学習する方法はありますが、生徒に選択がある場合は、遊ぶか考えるか、どのように学習するか、何を選択するかです。



したがって、新しい技術の採用は、教育の質を向上させる重要な側面です。 しかし、ここではもちろん、学校でのMacの軽率な購入は役に立ちません。いくつかの追加の考慮事項に従う必要があります。



どうする



  1. インターネットサイトをベースとして使用します。

    1. さまざまなテーマのプログラムは、さまざまなメーカーのもので、競争やコンテンツコンテンツを提供できます。
    2. もちろん、サイトはIE6（私がいた20校のうち1校でのみ見られます）を除く、幅広いハードウェアとブラウザーで動作するはずです。
    3. サイトのためのサイトは間違っています。 機会をオンラインで使用する必要があり、紙のオプションをコピーしないでください。
    4. 情報は、関連するために絶えず更新されなければなりません。 それは教科書を転載するよりも安いです（おっと、出版社はそれを好まないでしょう）。
    5. すべてのサイトをいくつかの要素と組み合わせる必要があります。

      1. おそらく、学校/大学/他者が適切なソフトウェアを選択して比較できる単一のポータルです。
      2. Journalアプリが使用されるすべてのものから統計を収集できるように、パフォーマンス情報を受信するための単一のAPI。
      3. システムに入るには、同じユーザー名とパスワードを使用する必要があります。
      4. モバイルプラットフォームと同様に、インターフェイスを構築するためにいくつかのガイドラインが必要でした。
  2. 最新の教育方法を使用します。 「コンピューター化された能動学習」についてです。 以下は、単純化された方法論の例です。

    1. 先生はレッスンの理論的な紹介をします。 素材の要約がシステムで利用可能です。
    2. 教師は、学生が主題を実際にマスターできるようにするゲームを開始します。 ロシア語、数学、物理学、化学、経済学、歴史に適しています...
    3. インタラクティブテストとゲームの成功により、結果を要約できます。
    4. 主な効果は、学習に対する最も高いモチベーションです。
  3. すべてのプログラムの実装は、それを使用したい教師によって開始される必要があります。 上記の約束は効力を発揮しません。 上記から、訓練と説明だけが行くことができます。


*トレーニングシステムがさまざまなメーカーによって提示されていることの重要性を、単一の共通スペースで再度強調します。



さらに、教科書を1〜2回発行するよりも、すべてのロシアの学生にネットブックを提供する方が安価です。 残念ながら、現時点では、このような決定は、書籍や文房具のロシアのメーカーが注文を引いていることを意味し、ネットブックの外国メーカーはそれらを受け取ることになります。



おわりに



このリストに関するHabrasocietyのコメントを聞き、共同で補足することは非常に興味深いでしょう。 将来、モスクワの学校でこのような教育プロセスを実施した経験の観点から、特定のポイントをより詳細に強調します。 もちろん、1-2のトピックは「私はPR」に該当しますが、州や私たちの誰かが新しい方法で子供たちを教育し始めるまで待つ必要はないと思います。 それらを伝えて見せることが必要です。
