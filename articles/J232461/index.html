SyntaxTreeを購入した後のVisual Studio Tools for Unity 1.9の最初のリリース





MicrosoftがUnityVSでSyntaxTreeを購入しているというニュースがありました。 無料版はいつ登場するのだろうと思いました。



そして今、この便利なツールのリリースは、MSで約束されているように、自然に無料でリリースされるようになりました。



Unity向けVisual Studio 2010ツール

UnityのVisual Studio 2012ツール

Unity向けVisual Studio 2013ツール



これは、SyntaxTreeの買収以来のツールの最初のリリースです。



主なイノベーション：

  1. デバッガーが高速化されます。
  2. より高速な起動 。
  3. C＃構造の処理の改善 。 イテレータをデバッグするとき、または変数がクロージャ内で利用できるときに、ローカル変数のウィンドウが適切に満たされるようになりました。
  4. ゲームを起動して、ワンクリックでデバッグします 。 この機能は、開発者から最も要求されている機能の1つです。デバッグの目的を変更するだけで、デバッガをアタッチしてゲームを開始できます。 この機能は、Visual Studio 2012および2013でのみ使用できます。
