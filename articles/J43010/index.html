そのため、サムスンはモバイルオペレーターAT＆Tとともに、新しいSamsung SGH-i907 Epixスマートフォン（OS Windows Mobile Professional 6.1。）のリリースを正式に発表しました。









このデバイスは、Samsung SGH-i780と非常によく似ています。 QWERTYキーボードと、開発者によって「光学式マウス」と呼ばれる小型のタッチパネルが装備されています（通常のラップトップのようにデバイスを制御できます）。 製品の技術的特性に関しては、次のとおりです。



  • ネットワークGSM / GPRS / EDGE（850/900/1800/1900 MHz）およびUMTS / HSDPA（850/900/1900/2100 MHz）で動作する能力;
  • 256 MB ROMおよび150 MB RAM。
  • 最大32 GBのmicroSDHCカードを接続するためのスロット。
  • Wi-Fi 802.11 b / gおよびBluetooth 2.0（A2DP）;
  • USBインターフェース
  • aGPSおよびAT＆T Navigatorサービスのサポート。
  • 統合された2メガピクセルのデジタルカメラ。
  • MP3、WAV、AAC、AAC +、eAAC +およびWAVで動作する内蔵オーディオプレーヤー。
  • バッテリー容量は1800 mAhで、その充電は、7時間の継続的な交渉またはスタンバイモードでの14日間のバッテリー寿命に十分です。
  • 全体の寸法は117 x 61 x 13 mmです。
  • 重量は129 gです。
