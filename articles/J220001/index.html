Kenju fork Kendo UI Web（GPL3）

数日前、TelerikのKendo UI JSフレームワークの開発で1つの重要なイベントがありました。 彼らは、Apache v2ライセンスの下で製品のOpenSourceバージョン-Kendo UI Coreをリリースしました。 実際、すべてがそれほど単純で明確ではないことが判明しました。



以前は、Kendo UIライブラリ全体がGPL v3ライセンスでリリースされていたため、商用製品で無料で使用することはできませんでした。商用アプリケーションでは、Kendo UI商用ライセンスはこのライブラリに基づいて開発されました。 しかし、オープンソースにとっては、このライブラリは本当の贈り物でした。 もちろん、他にも同様のライブラリがあり、膨大な数のファンがいます。 剣道の利点は次のとおりです。

1）jQueryの完全サポート

2）多数のウィジェット 。ほとんどの場合、1つのJSライブラリ（jqueryをカウントしない）で取得できます。

3）品質と快適なテーマ

4）優れたTwitter Bootstrapサポート。 剣道には、CSSフレームワークを同時に操作するための独自のブートストラップテーマもあります。







Apache v2ライセンスを使用すると、商用プロジェクトでKendo UIライブラリを使用できます。KendoUIコアのバージョンには通常のすべてのウィジェットが含まれているわけではありませんが、次のものは含まれていません。

1） 編集者

2） グリッド

3） スケジューラー

4） ツリービュー

5） アップロード



もちろん多くのアナログがありますが、アナログは追加のJSライブラリであり、異なるデザインスタイル（css）であることを忘れないでください。



フレームワークの以前のバージョン（GPL v3）はパブリックアクセスからすぐに消えるので、 githubに kendoui.web.2013.3.1119.open-sourceのバージョンを投稿しました

剣道という名前はTelerikによって登録されているため、フレームワークの名前をKenjutsuに変更し、削減後、より印象的なKenjuを得ました。

フォークの主なタスクは、GPL v3ライセンスに基づいてフレームワークの状態を保持することです。GPLv3ライセンスに従って、不足しているウィジェットを使用できます。
