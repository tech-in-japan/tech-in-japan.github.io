DIYタキシードペンギンソフトトイ



1年ほど前に、私は女の子、つまりUbuntuにLinuxをインストールしました。 彼女はプログラマーなので、熱心でした。 しかし、Linuxの何よりも、彼女はお守り-タキシードペンギンが好きでした。 そして、適切なぬいぐるみを手に入れたいという願望が表明されました。



まず、購入する機会を求めてインターネットに急いで行きましたが、適切なオファーが見つかりませんでした。 私はロシアでいくつかの申し出を見つけましたが、彼ら（私はウクライナに住んでいます、ハリコフ）はあまり私に合わず、その上、私は彼らの外観が好きではありませんでした。 1つの選択肢は、かわいいペンギンのキエフで見つかりましたが、身長はわずか10センチです。



一般的に、ギフトは無期限に延期されました。 しかし、3月8日の終わりまでに、私は自分でこのおもちゃを作ろうとすることにしました。



残念ながら、ほとんどの作業が完了してからHabrに着いたので、写真を使っておもちゃを作成するプロセスを文書化しませんでした。 ただし、Habrはお針子のフォーラムではないため、これが最適な場合があります。



パターン




それはすべてパターンの検索から始まりました-そして、問題はすぐに解決されました： www.free-penguin.org 。 これは、Tuxのオープンパターンを作成するプロジェクトです。 ここでは、GPLの下でそれらをダウンロードできます。 アルファ版の状態と2005年の最後の変更にもかかわらず、非常に使いやすくなっています。



ギャラリーがあります 。 非常に異なる人々によってこれらのパターンによって作成されたおもちゃの写真は非常に励みになりました。 非常にきれいな標本がいくつかあります。



印刷の利便性のために、パターンをわずかに編集し、A2ワットマン紙に印刷し、0.5センチの余白で切り抜く必要がありました。



素材


ここでも、すべてがシンプルに思えます。 フリースが見つからなかったので、黒（30cm x 1.5m）と白（20cm x 1.5m）のフェイクファーと黄色のキャンバス（20cm x 1.5m）を撮りました。 パディングには合成防寒剤（2m x 1.5m）を使用しましたが、回数4で過剰に使用しました。



ミシン


その後、楽しみが始まりました-縫製。 合併症は1つしかありませんでした。実際に縫うことができませんでした。 ここで言うことすら何もありません-私は何回も縫い、裂き開いていました。



唯一のトリック（最初に縫わない人には明らかなことです）は、すべてを裏返しに縫い付けて最後の瞬間にのみ縫わなければならないということです。 実際、この最後の瞬間が最も困難でした。



結果


一般に、夕方にミシンで座っておもちゃを作るのに2週間以上かかりました。 材料のコストは25グリブナ（約100ルーブル）です。



結果は報われました;贈り物は良い効果がありました。 そして、これは、フリースの代わりにフェイクファーを使用しているために彼がチクチクするという事実にもかかわらず、そして一般的には、なんとなく奇妙に見えました。



完全な秘密は、手作りの贈り物が常に価値があるということです。



あとがき


はい、この記事はHabrの主題に完全には当てはまりませんが、ITとほとんど直接関係があります。



この記事が、あなた自身の手で愛する人へのおもしろい贈り物を作るきっかけになることを願っています。 次の3月8日まではまだ時間があります。
