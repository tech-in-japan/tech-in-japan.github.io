Kickstrap-テーマとボーナス付きのフォークTwitter Bootstrap



Kickstrapは、 Twitter Bootstrap 、 HTML5ボイラープレート 、 IcoMoonおよびFont Awesomeのアイコンを組み合わせ 、実績のあるjQueryプラグインChosenおよびjGrowlを追加し、 Bootswatchのさまざまなカラースキームとテーマのサポート、IEの初期バージョンの追加CSSメッシュおよび健全なサポートを追加すると発生します。



このプロジェクトは、デフォルトで緑がかった配色、選択された友好的なドロップダウンリストを作成するための追加のJavaScriptプラグイン（ Kickstrapのデモ ）およびブラウザーにメッセージを表示するjGrowl （MacOSXのGrowl（ Kickstrapのデモ ））により、標準のTwitter Bootstrapと区別されます。



HTML5の防弾のHTML5 ボイラープレートテンプレートは、Modernizrおよびビルドスクリプトで動作するすべてのcrossdomain.xml設定とともに、HTMLベースとして使用されました。



Kickstrapは完璧なスタイル（ さまざまなテーマ ）で、12 +標準のグリーンがあります。 プロジェクトのテーマのサポートはhttps://github.com/thomaspark/bootswatchから適応されており、テーマ自体は対応する.lessファイルに接続されています。



OLCGグリッド （One Line CSS Grid）のサポートもプロジェクトに追加されます。これには、3つのクラスのみが含まれます-oh =「1/2」、ot =「1/3」、tt =「2/3」ですが、同時に、ネストされた異なるブラウザーをサポートしますおよび画面サイズ。 グリッドのデモンストレーションは、これらすべての機能を完全に示しています。



Kickstrapは、 グリフィコンの代わりに Awesome IconsとIcoMoonを使用します



IE 6をサポートするには、もともとTwitter Bootstrap v2自体のために作成されたBootstrap-IE6プロジェクト開発が使用されます。



プロジェクトは非常に活発に開発されており、オリジナルのBootstrapの新しいバージョンをサポートしています。Kickstrap0.9 RCは4月15日にすでに計画されています。
