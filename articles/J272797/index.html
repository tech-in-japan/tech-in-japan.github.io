2015年12月、Microsoft Virtual Academyの新しい無料コース





本日、 Microsoft MVA Virtual Academyの無料コースの休日前のレビューを準備しました。 これまでと同様に、Microsoftの専門家からの新しい資料は、ITプロフェッショナルとソフトウェア開発者の利益のために引き続きリリースされます。



ヒット！ 子供にプログラムを教える方法

待望の珍しいコースでは、コンピューターとプログラミングで子供を魅了するのに役立つ多くのマイクロソフトテクノロジー（だけでなく）を調べます。 方法論的に包括的であり、テクノロジーに関する包括的なストーリーがあると主張することなく、このコースは正しいリンクとアイデアを提供します。そのおかげで、子供と一緒にコンピューターで勉強し、作成し、多くの時間を過ごすことができます



新しいもの！ 学生および学童向けMicrosoft Azure Cloudの概要

ビデオ資料は、DreamSpark / Imagine Accessプログラム内のクラウドサービスの可能性を最大限に活用したい学生および学童を対象としています。 Azureポータルにアクセスする方法、コンテンツ管理システムに基づいてサイトを展開する方法、WebMatrixのブートストラップテンプレートおよびVisual StudioのASP.NETに基づいてゼロからサイトの開発を開始する方法について説明します。



新しいもの！ Office 365開発の概要

世界中の10億人のOfficeユーザーは、Officeクライアントで1日あたり平均2〜3時間を費やしており、400ペタバイト以上のデータがOffice 365でホストされています。Web開発者の場合、Office 365プラットフォームはこれらのユーザーはビジネスソリューションを使用し、日常のタスクを自動化し、カスタマイズされたソリューションを作成します。 アプリケーションモデルを使用して、コンテキストモードでOffice 365のソリューションを直接作成する方法を学習します。 さらに、OneDrive for Business、Mail、Calendar、People、Sitesなどのサービスを使用するために、スタンドアロンのWebおよびモバイルアプリケーションからOffice 365 APIを呼び出す方法を学習します。



ヒット！ Microsoft Azureの基本

あなたは経験豊富なITプロフェッショナルですか？ Microsoft Azureについてもっと知りたいですか？ Bob Taborに参加してください。 この最初のコースの一環として、Microsoft Azureプラットフォームが存在する理由と、それが若い企業やFortune 500企業にとってどのような問題を解決するかを学習します。



ヒット！ JavaScript開発のエクスプレスイマージョン

JavaScriptを開発している場合、このコースは、近い将来に何が待っているかを理解するのに役立ちます。 複雑なWebソリューション、Windowsアプリケーション、クロスプラットフォームアプリケーション、ゲームなど、JavaScriptでのアプリケーション開発における新しいトレンドと現代の機会の概要を説明しようとしました。 また、将来に目を向け、JavaScript標準の次のバージョンであるECMAScript 6の可能性を検討し、現在JSをすでに適用できる新しい領域に触れます。



F＃の関数型プログラミングの魅力的な紹介



関数型プログラミングは、プログラミングに対する別のアプローチであり、通常の命令とは非常に異なり、プログラマーの間で急速に人気が高まっています。 F＃は最も人気のある（TIOBE評価による）関数型プログラミング言語であり、他の.NET言語と組み合わせて使用​​して、プログラマの生産性と幸福度を大幅に向上させることができます。 このコースでは、F＃言語と関数型プログラミングの概念に精通し、プログラムの作成プロセスをさらに楽しむ方法を学びます。



NEW！ Windows 10：展開、管理、セキュリティ

ITプロフェッショナル向けの最初のWindows 10テクノロジーコースでは、企業全体に新しいOSを展開し、MDMソリューションで管理し、Windows 10デバイスを保護し、企業情報の漏洩を防止する詳細についての洞察を提供します。 このコースでは、新しいWindows 10構成機能（プロビジョニング）、Assessment and Deployment Kitの展開ツールの変更、およびインプレース機能のアップグレードについて説明します。 Windows 10をAzure Active Directoryに組み込む利点、MDMで利用可能なオプション、Microsoft Passportの次世代ID管理システム、Windows Hello生体認証サブシステムの概念などを学びます。



ヒット！ Windows Server vNextの技術概要

コースの一環として、独立した専門家がMicrosoftのサーバーOSの次のバージョンの最も興味深い技術革新と改善について話します。 Hyper-Vの新機能に精通し、WindowsおよびNano Serverコンテナーとは何か、ストレージレプリカがどのように機能するか、どのシナリオで、RDSの新しいローカルディスクを使用してWindowsベースのストレージを構築できるかを学びます。



Azure IaaSクラウドでのDockerコンテナーの使用

このコースでは、Dockerコンテナー管理システムの構築とIaaSタイプのクラウドシステムでの使用の問題に対処します。 KVM、Xen、OpenVZ、LXCなど、さまざまな仮想化、エミュレーション、およびコンテナ化システムの概要と比較を簡単に確認できると便利です。 このコースでは、Dockerの実用的なアプリケーションとその主な機能についても説明します。



Windows ServerおよびSystem CenterでのLinuxワークロード管理
