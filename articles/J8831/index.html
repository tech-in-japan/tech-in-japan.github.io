Firefoxの2つのクールなもの





エピグラフ

私があなたに提示したい2つの部分を別の投稿に載せると、2つの小さなニュースが届くので、それは1つの大きなニュースであり、Firefoxに関するものです:)



したがって、この号では、「Google、 Mouse、 Firefox」および「Firefox for Clerks :)」



マウスなしのGoogle検索

少し前に、Googleは（a）キーボードなしでGoogleで検索できる素晴らしいプロジェクトを発表しました（このことについてのニュースはハブで流れていたようですが、Googleタグでの検索では結果が得られませんでした-歓迎します）。 私はニュースを見ましたが、それがすでに機能しているサービスだとは思いませんでしたが、機能していることが判明し、それとのコミュニケーションを簡素化するためでさえ、作者に知られていない親切な魂が検索拡張機能を作成しました

それはすべて簡単です。フォルダC：\ Documents and Settings \ UserName \ Application Data \ Mozilla \ Firefox \ Profiles \ ProfileName \ searchpluginsに保存して 、Firefoxを再起動します。

Firefox検索のキーボードショートカットを思い出させてください：



Ctrl + Kの組み合わせを使用して検索を開始できます（Macユーザーの場合はCmd + K ）

検索エンジンの選択Ctrl + 上 （Mac： Cmd + 上 ）およびCtrl + 下 （Mac： Cmd + 下 ）。

結果を新しいウィンドウで開きます： Alt + Enter （または、Firefoxを構成して、常に新しいウィンドウで検索結果を開くことができます：アドレスバーにabout：configと入力し、browser.search.openintabパラメーターをtrueに設定します）




そして、新しく発見されたGoogleの場合：



J-次の結果を選択

K-前の結果を選択

EnterまたはO-選択した結果を開きます

/ -カーソルを検索バーに移動します

Esc-行からカーソルを削除します




しかし、これは私たちにとって単なるウォームアップでした-さて、主な楽しいニュースに移りましょう：



ドキュメントファーム

このような状況に遭遇したことはありません-20のタブがFirefoxで開かれ、システムは率直に言って、飛びません。ここでは、あなたにとって非常に興味深いものを見つけますが、情報の思いやりのある所有者によって何らかの形式のオフィス形式で詰め込まれています： ppt 、 doc 、 rtfまたは（地獄は冗談ではない） ps

アプリケーションをダウンロードするには、歯を握りしめて、見つかった情報を楽しんでください。

広告で言うように、私もかつて同じでしたが、 Firefox Docufarm拡張機能を発見しました

一度インストールすると、すぐに上記のすべてのファイルタイプを表示する機会が得られます。作成者が正しく理解していれば、オフィスアプリケーションをインストールすることもできません。 これは、ブラウザでドキュメントドキュメントを直接表示する例です。





他に何を追加するか... 1つだけ。Officeがインストールされていないコンピューターで実験したことはなく、公式サイトには情報がほとんどないため、コメントを歓迎します。



PSは、Yandexが発見したドキュメントでこの拡張機能をテストし、このような問題に遭遇しました。すべてのオフィス形式のファイルは強制的にダウンロードできます（説明方法はわかりません）が、Googleはそのような問題を引き起こしません
