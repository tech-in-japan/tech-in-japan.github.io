Jira Java APIを使用してScriptRunnerからJiraのクエリを更新する3つの方法

この記事では、Jira Java APIを使用してJiraでクエリを更新する3つの方法について説明します。

次のJira Java APIメソッドを使用します。



  • Issue.setCustomFieldValue（CustomField customField、オブジェクト値）
  • CustomField.updateValue（FieldLayoutItem fieldLayoutItem、Issue issue、ModifiedValue modifiedValue、IssueChangeHolder issueChangeHolder）
  • IssueService.update（ApplicationUserユーザー、IssueService.UpdateValidationResult updateValidationResult）


Jiraで使用可能なすべてのタイプのカスタムフィールドを「ボックス」から更新するためのスクリプトの例を示します。表には、検討中のメソッドの違いが示されています。









Adaptivist ScriptRunner 5.3.7プラグインのスクリプトコンソールでJira 7.7.0のすべてのスクリプトをテストしました。 この記事の目的は、「クリーンなコード」を書くことではなく、クエリを使用する原則を述べることでした。



スクリプトテキストはここからダウンロードできます 。
