PDC10会議の最初のリリース





PDC10カンファレンスは数時間で開始されるという事実にもかかわらず（モスクワ時間20:00での生放送 ）、このイベント専用の楽器の最初のリリースがすでに登場し始めています。



以下は、リリースされた更新済みツールとダウンロードリンクのリストです。



-WCF Data Services（2010年10月CTP 1） -.Net 4.0およびSilverlight 4.0プロジェクトのオープンODataプロトコルに基づいてWebサービスを構築するためのツール。

-Microsoft Sync Framework 4.0 2010年10月CTP-同期をサポートするアプリケーションを構築できるフレームワークの新しいバージョン。 新しいリリースでは、Odataのサポート、Windows Phone 7とSQL Azureの同期のサポート、診断ダッシュボード診断ツールの追加、更新された例、iPhoneフレームワークの使用例の追加が発表されました。

-Windows Azure AppFabric SDK V1.0-10月の更新-クラウドサービスWindows Azure AppFabric Access ControlおよびWindows Azure AppFabric Service Busと対話するアプリケーションを構築するための一連のツールの新しいバージョン。

-Silverlight 4用WCF RIAサービスV1.0 SP1ベータ-Silverlightに基づく分散アプリケーションを構築するためのツールの新しいバージョン。 リリースの詳細については、 このブログ投稿をご覧ください。

-WCF RIA Services Toolkit 2010年10月-WCF RIA Servicesで使用する追加ツールのセット。



PDC10の開始を楽しみにしています。PDC10では、注目を集める発表、新技術、多くの興味深いレポートがリアルタイムでオンラインで利用できることが期待されています。
