Google Web FontsのProsto

無料のフリーフォント「Prosto」（ 2月20日にHabrahabr で言及したものと同じ）が最近Google Web Fontsコレクション（ 「 Prosto One 」と呼ばれます ）に追加されました。彼らは熱心にフォントを使用し始めました。



1月にHabrahabr で説明したフォント「PT Mono」は、以前同じコレクションに受け入れられました。



これにより、現在使用中のGoogle Fonts APIのすべての利点が利用できるようになりました。たとえば、より小さな文字のサブセットを選択して、ファイルサイズとダウンロード時間を半分にすることができます。
