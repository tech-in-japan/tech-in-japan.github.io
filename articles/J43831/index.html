Windows 7の新しいタスクバーインターフェイスに関する興味深い事実

Windows 7の新しいインターフェイスのトピックは、前回のPDC 2008の後でHabréで既に議論されました。特に、新しいタスクバーのインターフェイスを議論する際に激論が繰り広げられました。 PDCに配布されたWindows 7のプレベータ版の配布キットには、提示されたインターフェイスが含まれていなかったという事実も、状況に影を落としていました。











しかし、よく知られた ITのエバンジェリストであるLong Zhengは、Windows Core User ExperienceのシニアプログラマーであるChaitanya Sareen（Chaitanya Sarin）となんとか会い、PDCの舞台裏に残されたタスクバーについての新しい事実を語りました。
