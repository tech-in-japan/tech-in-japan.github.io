デスクトップWindowsおよびMac用のSkype 6.0

Skypeは、Skypeクライアントの新しいバージョンであるSkype for Desktopのリリースを発表しました。これは、2つの新しいリリースを組み合わせたものです。

Skype 6.0 for Windows-リリース6.0.0.120

Skype 6.0 for Mac-リリース6.0.0.2946







Skype for Desktopは、最近のスマートフォンやタブレット向けのSkypeバージョンの急速な開発により、別のユニットに分離されました。 近い将来、Skype for Windows 8は新しいオペレーティングシステムに統合される予定です。



デスクトップ用のSkype 6.0の新機能の中で：

-MicrosoftおよびFacebookとの統合。 これで、MicrosoftおよびFacebookアカウントでSkypeにログインし、すぐにチャットを開始できます。 必須のSkype登録は不要になりました。

-Windows Live Messenger、Hotmail、Outlook.comのユーザーとSkypeから直接テキストチャット。



Windowsデスクトップ用のSkype 6.0の革新の中で：

-Skypeは、タイ語、クロアチア語、スロベニア語、セルビア語、カタロニア語、スロバキア語の6つの言語にローカライズされています。

-新しいユーザーインターフェイス

-オンラインユーザーのカウンターが削除されました-開発者は、この要素はそれほど重要ではないと判断しました

-プロフィール写真の履歴

-テレメトリーの改善。



Macデスクトップ用Skype 6.0の新機能：

-個別のウィンドウでチャットを開く機能

-Retinaディスプレイのサポート
