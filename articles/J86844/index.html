Vuurmuur-ネットワークの警備で猫と戦う





Vuurmuur-このような「猫」の名前の後ろには、iptables用のかなり強力なGUIアドインが隠されています。 他のiptablesアドオンとの主な違いは、 Ncursesで記述されたコンソールインターフェイスの存在です。 したがって、管理はまだSSHまたはコンソールを介して簡単に可能です。 Vuurmuurは、シェーピングの操作方法を知っており、トラフィックモニタリング機能をサポートし、個別のログを維持し、2.4コアと2.6コアの両方でうまく機能し、ロシア語を「話す」ことさえできます。







特徴



運営


  • いつものように-iptablesの知識は必要ありません。 かなり:)
  • ルールの簡単な構文
  • 整形
  • Ncurses GUI、Xウィンドウの必要なし。
  • 簡単なポート転送
  • 簡単なNATセットアップ
  • 安全なデフォルトポリシー
  • sshまたはコンソールを介したフルコントロール（PuTTY for Windowsを含む）
  • 他のツールと統合するためのスクリプト
  • vuurmuurを使用せずにシステムにiptablesルールを設定するbashスクリプトを構築する機能
  • IPスプーフィングツール
  • 不要な化合物を破壊する可能性
  • QUEUEまたはNFQUEUEを使用してSnortと連携する
  • サポートされる言語：英語、ドイツ語、ノルウェー語、フランス語、オランダ語、ポルトガル語、 ロシア語
  • ロシア語の明確で便利なヘルプ


モニタリング


  • ログをリアルタイムで便利に表示
  • 接続をリアルタイムで便利に表示
  • ログ表示時のフィルタリング
  • 基本的な交通量
  • 古いログを検索


経理


  • 監査：すべての変更が記録されます
  • すべての新しい接続と疑わしいパケットを記録する
  • トラフィックカウント


設置



ジェンツー


emerge -av layman

layman -f && layman -a sunrise

ACCEPT_KEYWORDS="~ARCH" emerge -av net-firewall/vuurmuur







Debian / Ubuntu


#Debian

echo "deb ftp.vuurmuur.org/debian etch main" >> /etc/apt/sources.list

#Ubuntu

echo "deb ftp.vuurmuur.org/ubuntu feisty main" >> /etc/apt/sources.list



apt-get update

apt-get install libvuurmuur vuurmuur vuurmuur-conf







手動インストールのファンの場合：debパッケージは公式FTPにあります
