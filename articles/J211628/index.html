7.35％や11.82％ではなく、正確に10％なのはなぜですか？

まあ...まず、簡単にそれを取る。 たとえば、1か月に2435人のtugriksを獲得しています。 これらのうち、243個は脇に置きます。 これは非常に初歩的なので、乗算と除算については何も聞いていなかった7歳の娘にこれを教えました。



第二に、ここでは単純な数学が発生します。 たとえば、1か月間、 x tugriksを獲得します。 これらのうち、 0.9 xを使い、 0.1 xを節約します。 年末までに、あなたは：

  • 12 xを獲得しました。
  • 費やした（12 * 0.9）x = 10.8 x ;
  • 保存（12 * 0.1）x = 1.2 x


悪くないでしょ？ 食欲を少し抑えて、1年でほぼ11の給料を受け取り、将来のために給料の1/4を延期しました。 この方法でお金を節約すると、12年で節約できます... 12の給料！ 通年の給料！ 一年中働けない！ 美しさ!!!



そして第三に-はい、あなたは自分自身を少し絞らなければなりません。 しかし、私には思えますが、収入が10分の1に制限されるのは現実的です。 個人的には、できました。

その結果、判明したように、私は自分の銀行を作成しました。 貯金を「銀行」と見なしたとき、あらゆる種類のプロセスが頭の中で続いていました。 そして、彼女は次の段階にこぼれました。 しかし、彼については少し後で。



要約すると何ができますか？ 稼いだお金の一定の割合を節約するために規律を始めます。 なぜ「稼いだ」から？ なぜなら、投資活動からの現金ギフトの形で利益があるからです（預金、誰かが利子を支払うなど）。 なぜ「修正」されたのですか？ 小さな利益からお金を節約しない誘惑があるので（「お金が少ない！」）、そして大きなものはヒキガエルを同情させます（「ええ！そんなにお金を節約しますか？」）。 そして最後に-それは「規律」です！ それがなければ、すべての活動は長期的な影響をやめます。

財務報告とは何ですか？



  • 地面にしっかりと立つ 。 お金がいくらで、何が必要なのかがはっきりとわかりました（またはすぐにわかります）。 そして、たとえば、別の仕事に転職することにした場合、私は必要な給与を明確に知っています。 そして、学校や車などにどれだけお金を使ったか自信を持って言えます。収入についても同じことが言えます。
  • 金融の合理化 。 最初の1か月間のレポートを確認すると、すぐに「ブラックホール」が表示されます。 特にありませんでしたが、このトピックは多くの人に関連しています。 喫煙する場合は、「喫煙」列を開始します-1年間、金額は公平です。 あなたが飲み物にもたれるのが好きなら、その結果も素晴らしいです。 あなたがバー、女性、カジノが横行するライフスタイルが好きなら、これは遠い将来の財政的幸福の確立にも貢献しません。 ところで- エンターテイメントが必要であり、あなたは自分自身を禁止することはできません！ それらがなければ、これは人生ではなく、重労働になります。 しかし！ 重要なのは、それらを合理化し、正当化された合理的な量に制限することです。たとえば、収入の5％以下です。 そして、彼らはあなたにとって完全に「合法」になり、あなたの良心は苦しみません。
  • 活動の目に見える結果 。 あなたは煙に費やす量を減らし始めました-しかし、神はそれが価値があるかどうかを知っていますか？..あなたは統計を見ました-はい、もっとお金がありました！ あなたはローンに別れを告げました-結果はここにあります！ あなたはある種の企業に投資しました-それは1年か2年でそれはそれ自身のために支払いました！ 例えば、彼らはかつて私にこう言っていました：「庭はそれ自体にお金を払っていない！ どれくらい費やしますか？」 そして、彼はどれだけ持って来ます！ 私は時々冬の終わりにジャガイモ/ニンジン/トコジラミを買いますが、トマトはまったく食べません。 一年中、私の4つの口でどれくらい費やしますか？

ステージ2：目標を設定する



これは非常に重要な段階であり、最初は取るに足りない抽象的と思われます。 延期額が具体的になり、それをどうするかを決める時が来たとき、私はその重要性を感じました。 最初は、この金額を棚（カード）に保管するだけだと思っていましたが、インフレの小さな歯を感じました。 それらを保護するために何かを考え出さなければなりませんでした。



今、最高のことは条件に同意することです：



  • 収益 -それは彼らの直接労働によって得られます。 仕事を変えるか、個人起業家の場合は勢いを増すことによってのみ増加できます。
  • 費用 -予算から直接流出するすべて。 これにはローンの支払いが含まれます。
  • 資産は売却可能なものであり、それが「資産」から「収入」に変わります。 不動産、株、預金（期間終了時に％の支払い）はすべて資産の例です。 賃貸不動産は、 受動的所得を生み出す資産です。 売却された不動産は、「資産」から収入に移されます（残念ながら一度限り）。
  • 収入 -収入、受動的収入、および売却された資産。
  • 借金はあなたが支払わなければならないものです-そして、それは「借金」から「費用」に流れます。 不動産、レンタカー、ローンはすべて借金の例です。




あなたの借金（たとえば、クレジット）は、常に（銀行にとって）誰かの資産です。 あなたの収入と収入は常に誰かの費用です。 あなたの資産は収入を生み出します。 あなたの借金は予算からあなたのお金を吸います。 ソファの下の銀行に横たわっているあなたの収入は、資産が収縮しているためです。



明らかに、私は理想的には：



  1. すべての負債を取り除く-お金がブラックホールに流れ込むのをやめる。
  2. 同じ期間に定期的に費用を上回る収入を生み出す信頼できる資産を持っている-たとえば、1,000人のtugriksの毎月のレンタル費用の場合、賃貸不動産、自動車などは、少なくとも1,000人のtugrikをもたらす必要があります。これはすべて修復する必要があります）;
  3. 一定の販売期間中の現在の費用をすべてカバーできる「雨の日」に信頼できる資産を確保するために、月に同じ1,000人のtugrikで、6,000人のtugrikで販売できる銀行の金属または不動産の準備ができているため、半年は穏やかです生きること
  4. 両方の種類の資産が損失のリスクで増加することを可能にする資金の準備を持ちます。たとえば、証券取引所への投資、企業への参加などのために、月にさらに100人のtugrikを持ちます。
  5. 借金を作成する必要がある場合（クレジット、不動産、または家電製品で車を購入する）、それを完済するために機能する資産も必要です。




まあ...これらは私の夢です。 私は彼らと仕事をしています。



すべての著者は、これらの夢を明確な方法で明確にすることを強く求めています。 「今後30年間で毎月1,000人のtugrikを生成し、10,000のtugrikの資産ストックを持ち、毎月1,000個のtugrikのリスクを抱える資産を作成するために」などです。 これらの数に達するとすぐに、私はお金をもうけることをやめます。」



したがって、タスクはさまざまな資金を作成することです。 各基金には、それを超えることができない独自の最小値と、私たちが努力しなければならない最大値があります。 最小値に達すると、次のタスクを並行して解決できます。 最大値を超えるとすぐに、超過した値は目標と最大値に達していない次の資金に転送されます。



それは一種の資金のラインを作成します。 最年少は最も控えめな（そして収益性の点で）が、最高の優先度です。 それらを埋める（または最小しきい値を超える）場合は、先に進むことができます。 「さらに」は古いファンドです。 収入の増加と自由基金の出現により、生活水準を上げるか、新しいグローバルな目標（たとえば、さまざまな形での慈善団体）を作成できます。



UPD ：ファンドのグループは「投資のポートフォリオ」を形成します。 たとえば、ポートフォリオN1は準備目的で機能し、最も信頼性の高い資金で構成されています（現金準備金、信頼できる銀行への預金、銀行の金属があります）。 残念ながら、投資の世界では、「最大の信頼性」=「最小の収益性」です。 したがって、N2よりリスクの高い資産のポートフォリオが必要です。したがって、より収益性が高くなります。 私は3つのポートフォリオに分割することを好みます-最も信頼性の高い（「エアバッグ」と呼ばれることが多い）、受動的収入を生み出す（「お金を稼ぐ」資金は賃貸用不動産、定期的に売却される資産など-「加速ゾーン」）そして（子供の希望と大人の夢を満足させるために）大きな目標を果たす。

そして、次のようになります-「現在の年の少なくとも1つの算術平均コストの現金供給、目標は1つの最大コストです」。 これは、毎月、今年度の費用を再集計することを意味します。 今年、最低支出が1,000人のトゥグリーク、中央値が2,000人のトゥグリーク、算術平均が3,000人のトゥグリーク、最大で7,000人のトゥグリークがあると仮定します（今年の実質比率）。 現金が3,000人未満のtugriksに蓄積されている場合、問題は解決されません。 3000から7000までの場合、より大きなタスクを解決できますが、ここにお金を入れ続けます（ここでどれだけ、どれだけ別々に決定されますか）。 今月7300人のtugriksが累積し、1000人のtugrikを配置する予定である場合、この基金から300個のtugrikを取り出し、さらに1300を次の充填に欲する基金に投資します。 毎月、私は現金準備金を管理し、7,000人のtugriksの数でそれを維持しています。

フローティングモードなど。 数年後には、食欲が減る可能性があります（子供が成長し、現在は別々に住んでいます）-その後、多くのお金が解放されます（新しい資金を作成することに決めない場合）。 または多分他の方法-より多くのお金があり、私たちはより多くを使います（または私は子供の家族を助けます、今私達は皆のための資金を共有します）。 次に、最新の資金から古い資金に戻す必要があります（または、資金よりも低い追加利益を下げる）。
