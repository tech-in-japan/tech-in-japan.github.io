SVN、Apache、Trac for Windowsをインストールして設定します。 パート3-Trac

このパートでは：
  • Pythonをインストールする
  • Tracをインストールする
  • プロジェクトのTracインスタンスを作成します

Pythonをインストールする

  1. python-2.5.msiを実行します
  2. インストールディレクトリを指定しますC：\ usr \ local \ python

  3. その後、すべての設定をそのままにしておきます
  4. インストールが完了するまで待ちます。

Tracをインストールする

Tracを適切にインストールするには、SetupToolsを事前にインストールする必要があります。
  1. setuptools-0.6c7.win32-py2.5.exe
  2. mod_python-3.3.1.win32-py2.5-Apache2.0.exeで 、Apache Cへのパスを指定します：\ usr \ local \ Apache2
  3. pysqlite-2.4.0.win32-py2.5.exe
  4. svn-python-1.5.2.win32-py2.5.exe
  5. Genshi-0.5.1.win32-py2.5.exe
  6. Trac-0.11.1.win32.exe


foo-storeプロジェクトのTracインスタンスを作成します

すべてのTracプロジェクトはD：\ trac-projects \ディレクトリに配置されます svnリポジトリDにあるプロジェクトファイルfoo-store ：\ repos \ store1
  1. 環境を初期化するには、コマンドラインを実行してコマンドを実行します。 
      trac-admin d：\ trac-projects \ foo-store initenv
  2. コンフィギュレーターの質問に答えます。
    • プロジェクト名[My Project]> foo-store
    • データベース接続文字列[sqlite：db / trac.db]> 、空のままにします
    • データベース接続文字列[sqlite：db / trac.db]> 、空のままにします
    • リポジトリへのパス[/ path / to / repos]> D：\ repos \ store1 、SVNリポジトリへのパスを指定します
  3. プロジェクト環境が正常に作成されたことをおめでとうございます！
  4. Tracインスタンスの準備が整いました。
すべてのプロジェクトで繰り返します。
