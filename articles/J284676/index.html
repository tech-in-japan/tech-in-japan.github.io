ACTAが承認される場合があります



AP通信経由の写真：ポーランドのSejmの会議



米国議会によるSOPA （ 英語 ）とPIPA （ 英語 ）の立法イニシアチブの採択に全世界が抗議した一方で、ACTA協定の署名は最終段階に入っています。 偽造防止貿易協定のより詳細なレビューは、勝利を収めて敗北したSOPA / PIPAインターネットコミュニティよりもはるかに厳しい取り組みです。 ACTAの交渉は高機密モードで行われ、詳細は文字通り一文字ずつ明らかにされると言う価値があります。



貿易協定の名前は有形物について話していると仮定するのは論理的ですが、実際には、協定は法律と同等であり、知的財産に関してはるかに広い制限を課しています。 このテーマについてはかなりの情報が利用できるので、すべてを1つのマテリアルで構成するようにします。





歴史と詳細に目を向ける前に、関心を高めるために、 ACTAは税関職員がソフトウェア、音楽、ビデオなどの侵害情報についてメディアを検査できるようにしていることに注意してください。



このような監査の合法性と合憲性（これまでのところ米国のみ）は、2008年にサンフランシスコ第9地区の連邦控訴裁判所[ link ]の決定によって承認されました。 ファイルまたはディスクの暗号化は保存されません-「忘れられた」パスワードの場合、名目で国への入国が拒否される場合があります。



ACTAの採用はプロバイダーにも影響を与えます。プロバイダーによると、文書によると、サブスクライバーとダウンロードするコンテンツを制御するための対策を講じる必要があります。 ただし、物理的な検索と比較すると、そのような結果はまったくひどくはありません。 さらに、一部の国では「3つの警告の規則」が既に機能するように試みられています。著作権侵害に関する3回目の警告の後、プロバイダーは問題のあるユーザーのインターネットアクセスを制限する必要があります。



ただし、SOPA / PIPAとは異なり、ACTAの活動は遠方の外国に限定されません-現在、22の欧州諸国が協定に署名しており、米国、日本、スイス、オーストラリア、ニュージーランド、韓国、カナダ、メキシコが率いるリストを補完しています。 1月26日の最後の和音に、デンマーク、ラトビア、リトアニアが加わりました[ リンク ]。 エストニアは、慎重さを示しながら、盲目的に署名することを拒否し、この問題を政府の本会議の議論に持ち込んだ[ リンク ]。



ラトビアでは、事実上の事後情報として、ピーター・ヴァイバーズ大使が日本への実地訪問で政府を代表してACTA協定に署名したという小さなメモの形で配布されました。 メモはすべてのニュースポータルで決して公開されず、ページの下部にスローされました。 契約草案が適切にカバーされているという当局者の声明は真実ではありません。



物語



欧州委員会および他のACTA加盟国政府は、2008年5月22日にWikiLeaksの契約草案がリークされた後にのみ、合意に関する交渉への参加を認めました。 人権擁護家からのその後の要求に応じて、「偽造防止貿易協定」という名前のみを含む文書を受け取りました。この文書では、テキスト全体が黒く塗られています[ link ]。



2年半の間に行われた11ラウンドの交渉では、さらに多くの参加者が引き付けられたほか、民法、国境管理、インターネット上の犯罪の防止の問題も引き付けられました。 しかし、交渉の最中であっても、文書の全文は欧州委員会でさえ入手できませんでした。 ソニーピクチャーズ、タイムワーナー、およびベライゾンの民間企業は、NDAの下でテキストにアクセスできました。



結果



これまでのところ、ハッカーグループを除き、ポーランド人だけがACTAの宣伝に対応した人たちです。 1月25日、ワルシャワでの大規模な街頭抗議に1万人以上が参加しました。 ポーランド政府は、抗議者による「恐mail」に屈するつもりはないと述べた。 ドナルド・タスク首相は、「ポーランド議会が合意を承認する前に不満を述べた人たちは十分な時間を述べた」と述べたが、これは控えめに言っても、前述の合意の秘密を考慮して真実とはほど遠い。



この物語は3月31日に終了し、欧州議会でのACTAの批准に関するヒアリングが予定されています。 そして、それが起こったら...



結論の代わりに



したがって、ACTAの現在のエディション：

  • 著作権侵害の個人的なアイテムを検索する国境管理の権利を与えます。 令状は必要ありません。
  • プロバイダーおよびホスティング事業者は、サブスクライバーによる著作権の遵守を監視し、最初の要求に応じて情報を提供する必要があります。 令状は必要ありません。
  • 特定の国の法律がそのような責任を規定していない場合でも、協定のすべての国の知的財産権の侵害に対する刑事責任を導入します。
  • 何らかの手段でDRMの回避を犯罪化します。 他の目的で使用できる場合でも、DRMを無視できるソフトウェアの配布を禁止します。
  • 補償を受ける権利のない偽造商標のあるアイテムの押収および清算を認識します。
  • ACTA運営委員会に、参加国による考慮と批准なしに直ちに発効する協定の修正を発行する権利を与えます。


上記に照らして、SOPA / PIPAの立法イニシアチブは、全体主義法自体の力に恵まれた貿易協定から注意をそらすように設計された単なる操作であるように思われます。
