分散検索エンジンYaCyバージョン1.0





フリーソフトウェア運動の活動家は、新しいプロジェクトを立ち上げました：検索エンジンYaCyは、将来、Google、Yahoo / Bing、Yandex、および他の非公開の商業検索エンジンと競合する可能性があります。 YaCyバージョン1.0は5年間の開発の後にリリースされ、現在では誰でも最終的にネットワークに参加できます。



YaCy検索エンジンはP2P原理に基づいて動作します。つまり、検索インデックスの保存と検索クエリの処理は、中央サーバーではなく、Freeworldのピアの分散ネットワークで実行されます。 GNU / Linux、Windows、またはMacOSでソフトウェアをインストールした各ユーザーはネットワークに入ることができます;現在、ネットワーク上には約1000のノードがあります。



Freeworldリアルタイムピアネットワーク図





独自のサイトがある場合は、少なくともインデックスを作成してください。そして、インデックスを独自のFreeworldサイトに配置します。 これを行うには、インデックス作成後に、プログラムのDHT配信およびDHT受信機能を無効にします。 必要に応じて、すべての興味深いサイトのインデックスを作成し、ローカル検索エンジンを使用できます（ スクリーンショット ）



分散ピアネットワークとオープンソースコードは、検索エンジンの安定性を保証し、検閲の試み（SOPA、PROTECT IPなどを参照）または主観の非難から保護し、ユーザーの個人情報を確実に保護します。 リクエスト履歴の保存、トラッキングCookieはありません。



プレスリリースのリリースにより、検索エンジンのデモ版は現在大きな負荷にさらされています。 ポータルが数日耐えられるように、プログラムをインストールした後、静的IP についてフォーラムに通知すると、ポータルからの検索要求がリダイレクトされます。



プロジェクトの著者によると、検索エンジンはウェブのインフラストラクチャの重要な要素であり、数十億人のユーザーと彼らが探している情報との間の重要なリンクです。







「このような重要な機能のために、いくつかの大企業に依存して自分のプライバシーを犠牲にすることはできません」とYaCyのプロジェクトリーダーであるMichael Christen氏は述べています。







ある意味で、YaCyは無料のソーシャルネットワークDiasporaの類似物であり、閉鎖型の集中型Facebook / Google + /などの代替手段として位置付けられています。 YaCyのサポートは、Free Software Foundation Europe（FSFE）によって提供されます。
