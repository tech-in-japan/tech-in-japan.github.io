ソーシャルメディア向けの効果的なプレスリリースを書く方法

私の意見では、ソーシャルネットワークには、有能で効果的なPRの環境として、未だ実現されていない大きな可能性があります。 今日、ほとんどのPR専門家はソーシャルネットワークを完全に無視するか、従来のPRの手法とアプローチを使用しようとして、完全に関心と懐疑心を失います。



ソーシャルネットワークにおけるPRの科学には未来があり、私たちにとってはそうでないとしても、誰がその法律を策定するかは確かです。 「どれだけ多くの素晴らしい発見があるかについて...」。



Muhammad Salimによる記事「ソーシャルメディアプレスリリースの書き方」の翻訳を公開しています。



プレスリリースでソーシャルネットワークの視聴者に興味を持たせることほど難しいタスクはありません。ここで重要なのは、彼らがあなたの会社の出来事を聞くのが嫌だということではありません。 プレスリリースを作成するための古典的なアプローチを次に示します。



1. [会社/ブランド名]が[製品/サービス]のリリースを発表



そのようなプレスリリースは退屈であり、スパムとして認識される可能性があります。なぜなら、目標は明らかにあまりにも明白であるためです-新製品の宣伝です。



2. [会社/ブランド名]が発表[会社の財務ニュース]



通常、このようなプレスリリースには、会社が別の会社と合併する、追加の資金を受け取るなどの発表が含まれますが、このタイプのプレスリリースにはエンドユーザーにとって有益な情報は含まれません（もちろん、プレスリリースGoogleがYouTubeを買収した）。



3. [会社/ブランド名]計画[イベント]



このタイプのプレスリリースには、会社がそのようなことを計画している情報が含まれています。たとえば、医療会社がこの病気の治療法を計画しています***。 ソーシャルネットワークのほとんどのユーザーは、このような情報は興味がなく、自分にとっては不必要であることに気付くでしょう。



4. [会社/ブランド名]完成（a）[プロジェクト]



このようなプレスリリースの唯一の目的は、会社の業績について自慢することです。 ユーザーには、「もちろん、10万人目の登録者がいるのは素晴らしいことですが、私にとっては以前のことですか？」と言う権利があります。



標準的なプレスリリースがソーシャルネットワークで失敗する運命にある理由



標準のプレスリリースがソーシャルネットワークで成功する可能性がゼロである主な理由は3つあります。
  1. 彼らは誤って焦点を合わせており、会社自体がそのようなプレスリリースに焦点を合わせています。
  2. それらは公用語で書かれており、それ自体が不信を引き起こします。
  3. 彼らは非人格的であり、あまりにも多くの聴衆のために設計されています。
  4. もちろん、あらゆる方法であなたの会社を宣伝したいという欲求には何の問題もありません。結局、私たち全員が生計を立てなければなりません。 ただし、ソーシャルネットワークの視聴者は、明示的に何かを販売しようとする試みや、洗練されたマーケティングテキストを試みる試みに対して非常に否定的であることに留意してください。


プレスリリースを機能させる方法



プレスリリースは、まず、読者に興味深いストーリーとして提示するのが最善です。 さらに、この形式のプレゼンテーションは、ジャーナリストがプレスリリースに興味を持つ可能性を大幅に高めます。



ソーシャルメディアで成功するために設計されたプレスリリースは、新しいサービスの立ち上げを発表したり、結果を達成したりするだけでなく、（最も重要なことに）達成された（または達成される）テクノロジーに関する情報を含める必要があります成功、これらのテクノロジーがなぜ機能したのか、そしてその結果が聴衆にとってどのように役立つか。



「代替プレスリリース」を書くためのルール



効果的なプレスリリースを書くためのルールは、多くの点でブログのルールに似ています：正しい見出し構造、興味をそそるスタート、グラフィックの使用、およびブログのすべての基本的な法則を使用するコンテンツ。 ただし、以下では、ソーシャルメディア向けに特に最適化されたプレスリリースを作成するためのゴールデンルールを公開します。

  1. このタイトルは、3種類すべての読者（永続、検索およびソーシャルネットワークからの読者）向けに最適化されています。
  2. 素材の主な意味は、簡単に読んだ後でも明確である必要があります。
  3. テキストのフォーマットは魅力的でなければなりません。
  4. 水なし-本質だけ。
  5. 他の人が自分に役立つ何かを学ぶことができるように、成功を達成するための技術を要約してみてください。
  6. 会社のオープン広告ではなく、読者の利益に焦点を当てたプレスリリースは、誰も読まない標準的な書面によるプレスリリースよりもソーシャルメディアではるかに大きな反応をもたらします。 読者の興味にもっと関心があるなら、読者はあなたが何をするかに興味を持つでしょう。
