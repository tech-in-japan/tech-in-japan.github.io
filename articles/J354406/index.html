あなたは去ることができません：難しい、そしてしばしば間違った選択について

私たちのほぼ全員が、少なくとも一度は別の都市で新しい仕事を提供されました。 15〜20年で少なくとも10件の提案を受け取りました。 私はサンクトペテルブルクにいる間、そのうちの一人を受け入れて、アメリカのプログラムに行きました。 そのような要求の大部分はモスクワからのものでした。 また、2006年にサンクトペテルブルクを離れたいと思っていたので、最終的に面接に行ったサマーラに引っ越すことを考えたことも覚えています。







記事のタイトルから理解できるように、今回は仕事の変更と移転のトピックを推測し、読者の意見を聞きたいと思います。 ほとんどの場合、次のいずれかを選択できます。



  1. 現状（つまり、何も変更しないで、自分の家に留まるオプションがしばしば勝ちます）;
  2. 居住都市で新しい雇用者を探します。
  3. 新しい雇用主のオフィスで働くために別の都市に移動する。


私は一般的な問題を検討し、 受動的な検索 （「ぶらぶら」し、選択は現状と新しい仕事の間、つまり特別な場合）と現在の雇用主の移転プログラム （多くの場合、これは要素を使用して行われます）農奴制、移動する権利も獲得しなければならず、N年後には献身と感謝の念を発散する。 移住も特別です。この場合、彼らは最初に将来の居住国を選択し、次にこの国の会社を選択します。 2番目の選択肢は、新しい雇用者が現在の都市にいる場合、すぐに話し合い、さらなる検討から除外することです。 ほとんどの場合、せいぜい収益はわずかに増加します。 それほど頻繁ではありませんが、より興味深いタスクとテクノロジーが提供されますが、6か月または1年ですべてが再び起こります。 結局のところ、両社はほとんど同じプロジェクトにほとんどの場合関与しているため（CISには数十の成功した食品会社があります）、したがって、従業員の収入の観点からのリミッターを含む同じ経済を持っています。



一般的な場合、すべては次のようになります。現在の職場で絶望的な状態を保つためです。 通常、不満の理由は次のセットで構成されます：レガシーコード、プロジェクトが疲れている、上司が不十分である、彼らは少ししかお金を払っていない、明るい未来について朝食を食べている、など。この段階では、「何かを変える必要がある」、つまり 問題の認識が発生します。 その後、解決策の探求が始まり、新しいものへの欲求と、前向きな変化を受け取らずにあなたが持っているものを失うリスクとの間の闘争が始まります。 したがって、各オプションをサポートする引数を検討します：現状と新しい雇用者のオフィスの別の都市への移動。



現状の選択肢を残すための引数（夢、しかし何も変更しない）：



  1. 生活の質の高さ -多くの人がそれを持っていますが、手放すのは難しいです。 引っ越しの際に彼らが最も頻繁に育てたいと思う高水準の生活（お金とそのために買えるもの）と混同しないようにお願いします。 たとえば、モスクワの生活水準は高く、おそらくロシアで最高ですが、かなりの数の人々にとって、生活の質はそれほど高くありません。 交通渋滞と地下鉄旅行へのストレスから選択する必要がある場合、どのような品質について話すことができますか？ 引っ越してすぐにアパートを買う余裕のある人はほとんどいないので、大都市での人生の最初の年の「楽しい」思い出があるでしょう。 彼らは仕事と帰りの道で1日2〜3時間の標準になりました-それについて考えないほうがいいです。 環境に関しては、状況はそれほど明確ではありません。 モスクワよりも環境がさらに悪い工業都市に住んでいるときの1つのことですが、状況はまったく異なります。自分の家に住むことに関しては、きれいな水を飲む能力ときれいな空気を吸う能力です。
  2. 間違いの恐れ。 これが取り返しのつかない状況につながる可能性は低いため、彼はしばしば誇張されています。 試用期間を過ぎることはできず、能力に自信があるかどうかは問題ではありません。 優秀なスペシャリストはすぐに新しい仕事を見つけます。 私は新しい都市が好きではなかった、私たちが望んでいた幼稚園の問題を解決しなかった、または妻が仕事を見つけることができません-これはすべて修正可能であり、ただ戻ってきます。 人々は他の国から戻ってきていますが、モスクワ-ピーターについては何と言えますか？
  3. あなたが参加していた社会を壊してください。 これは完全な休憩ではなく、21世紀に行うのは困難ですが、会うことに慣れている古い友人の会社がある状況や、助けを必要とする親を残すのは難しい状況です。 これには習慣も含まれ、そのいくつかは長年にわたって開発されてきており、現在の居住地と密接に関連しています。
  4. 明らかではない素材の獲得 。 労働市場をよく知っている（そして彼らも知っている）地元の人々が稼ぐことができるのと同じくらい新しい職場ですぐに稼ぐことはまずありません。 家を借りる費用に加えて、すべての隠れた費用（サービスへの出費、故郷のチケット、少なくとも最初は有料の幼稚園）を正直に計算し、住宅の費用と住宅ローンで購入する見込みを考慮すると、非常に悲しくなります。 幸いなことに、この現象は一時的なものであり、6か月後または1年後、あるいはそれ以前に、仕事に対する公正な報酬の問題を提起したり、単に転職したりすることができます（提案にheしないでください。よく支払いながら）。
  5. 変更の恐怖 。 これは、すべての人々に固有の、深くて古代の現象です。 「コンフォートゾーン」を離れることは困難ですが、これなしでは変化を達成することは不可能です。


移動の引数：



  1. より高い生活水準 。 テレビの映画、知り合いの話、観光旅行の印象-これらすべてにより、モスクワの生活水準はロシアの他のどの都市よりも高いと結論付けることができます。
  2. 野望は 、地元の制約が地元の市場にしばしばもたらす視点を超えています。 モスクワの支店や国際企業での仕事でさえ、しばしば地理的条件に基づく差別を意味し、活動の範囲、責任範囲、収入を制限します。
  3. 機会があれば新しいものを試してください 。 22歳のとき、家族がいなかったので（「また別の時間があった」ので）、「去る」という決定の6か月後に米国に移ることは難しくありませんでした。 たとえば、35歳のときに妻と複数の子供がいるときに何か新しいことを試してみるのは、もっと難しいでしょう。 ここで、少なくとも、あなたはあなたの妻に同意する必要があり、また、子供たちが居住地を変えることによるストレスについても考えなければなりません。
  4. 大都市の機会 。 これは仕事だけでなく、段落1のこれについても当てはまります。 ここでは、新しい社会化について詳しく説明します：子供向けの体育学校、路上で壊れた眼鏡がないこと（地域でよく見かけます）、博物館や劇場（最初はこの時間はほとんどありませんが）、より高い年金（生き残ることを夢見、ロシア連邦の年金制度を信じています）。
  5. 後半の欲望や事情 。 それは起こります。 たとえば、彼らは1つの興味深い仕事を提供しました。私は別れたくありません。恋には遠くに信仰はありません。2番目に行く必要があります。 「後半」が「少なくとも壊れたトラフの代わりに新しいトラフ」を望んでいるとき、それは異なって起こります。 はじめに。
  6. エコロジーと気候 。 北部の工業都市からの移動は、サンクトペテルブルクへの移動（雨とくすみ）について話している場合でも、環境と気候に関する状況を改善できます。


さらに、上記で定義した各パラメーターについて、両方のオプションの重みと値を設定し、それぞれについて要約し、比較することができます。 それがより判明した場所-そのオプションと選択。 実際には、衝動的なテクニックを含む他のテクニックを使用してそれを行う人はほとんどいません。 何らかの方法で、状況とリスクを分析する段階の後、「変更と移転」または「自分の場所にとどまる」を選択することができます。 私は1999年11月に個人的にこの選択をしました。



この選択には大きな欠陥があります-現時点（2018年4月）では、今日考えられない他のオプションがあるため、それは間違っている可能性があります。 少なくとも3つの追加オプションが表示されます。



  1. フリーランス 。 これで、海外を含む注文を個別に見つけて、仕事の面で自給自足できるようになりました。 ちなみに、これは通常の負荷がないメイン作業に加えて多くの練習をします。 1999年のロシアでのフリーランスだったかもしれませんが、私はそれについて何も聞いていません。 米国のコンサルティング業界について学ぶのはさらに驚くべきことでした。そこでは、自分で仕事をすることで1.5倍から2倍稼ぎ、年間11か月ではなく6から7か月働くことができます。
  2. 自身のビジネス 。 このオプションは1990年代に可能でしたが、異なっていたため、多くの若い企業は間違いなくそうではありませんでした。 これはすべて当たり前のように見えますが、成功した例、コミュニティ、ビジネスインキュベーター、シリアル起業家がいます。
  3. リモートワーク 。 これも新しい現象と呼ぶことはできませんが、1990年代後半には、お互いを探して需要と供給の交差点を世界規模で提供するリモートワークの市場はありませんでした。 今、市場があり、それは起こり、発展し続けます。


ビジネスを行う準備ができている人はほとんどいません。これは神経質で常に興味深いビジネスではありませんが、このオプションも検討する価値があります。



フリーランスとリモートワークについて話す場合、経験からわかるように、これらのオプションはスペシャリストの10〜30％が利用できます（基準：彼らはリモートで仕事をしたいと思っています）。 フリーランスとリモートワークの主な違いは、フリーランスが起業家精神の第一歩であるということです。実際、あなたはあなただけが働く会社のオーナーです。 このサイトから拡張できます。成功した例を知っています。 リモート作業は通常の作業であり、リモート形式で行われます。 まったく同じように、リモートで、個人に追加の要件を課します。その主なものは自己組織化です。



私は、100％リモートで働くロシアの専門家の正確なデータを持っていません。 私たちは2〜3％について話していると思います。 10倍の成長の可能性。 これは、州の立法枠組みが当時の要件に追いついていないという事実に一部起因していますが、これも許容されます。 しかし、人々の市場の未発達の主な理由。 誰かがそのような機会を知らず、別の人は試してみるのを恐れ、3番目の人は準備ができていますが、「主な仕事」を失うリスクはありません。 これはすべて、リモートインタラクションに対する両当事者の不信（最初は隠すべきものがあります）によって増大する可能性があり、現在の結果が得られます。 しかし、雇用主のオフィスを参照せずに、独立して居住地を選択できるのは、まさにこの仕事の形式とフリーランスです。 あなたが望む場所に住むことを夢見ませんでしたか？ これにあなたが好きなことをする機会を追加すると、幸せな人のその非常に小さな割合に陥ります。 実際、優れたエコロジー、発展したインフラストラクチャ、低コストの生活、高品質の生活ができ、メガシティなどの専門家のレベルで稼ぐことができます。



これはすべて、遠隔地で働いたことがない人にとっては知覚するのが難しいように思えるかもしれません（恐怖ルール）。私はそれを自分で試したので、問題を客観的に検討しようとしました。 しかし、6.5年の遠隔作業の後、私はモルフェウス（マトリックス）の乗組員の一人のように感じました。 年に数回、私はソフトウェア会社のオフィスを訪問し、毎回「これらの人々はここで何をしているのか、どうすればここで働くことができるのか」と自問します。



結論の代わりに



結論は簡単で簡潔です。 「誤った選択」に陥るのではなく、問題を包括的に検討し、作業と開発のすべてのオプションを評価します 。 そして、あなたの選択をして、あなたのわだちを選択してください。 たとえば、モスクワまたはサンクトペテルブルクから電話がかかってきた場合、コミュニケーションを開始し（=移動の準備ができている）、この都市からの他のオファー、および自由と責任の準備ができている場合は、地元または外国の会社のリモートワークのオプションを確認します。 フリーランスのオプション、あなたのビジネス、そして私がリストしていない（そしてそれらは）他のものを忘れないでください。 多くの人生の状況がありますが、主なことを理解することが重要です。



  • 現状を選んで常に変化に屈しているなら、おそらくあなたはあなたが望む方法ではなく、あなたの周りの人々が期待する方法で生きているでしょう。
  • あなたの人生を変える機会は、外部から来る可能性があります（多くの企業が面接さえ求めない35年から40年後）。その場合、あなたは奴隷になり、選択肢が制限されます。
  • ポジションがより積極的であれば意思決定が容易になり、外部からの最初の接触の場合でも、リーダーになり、自分で状況を管理するために、あなた自身の手でさらに主導権を握る必要があります。 このアプローチには、より多くの意識と時間が費やされますが、自分の運命を決定することよりも重要なことは何ですか？


このトピックの内部議論の中で、私たちは何人かの従業員に誤った選択を拒否し、リモートで働き始めた理由について話すように頼み、どこに住み、どの条件で働くかを独自に決定することにしました。 結果はVKまたはFBにあります。 私は2012年からリモートで仕事をしていますが、私たちのチームが「ウダレンカのベテラン」Vovaを持っていることに非常に満足しています。



Maxilectは、リモートチームワーク、最新のスタック上の複雑で興味深いプロジェクトを提供する唯一の会社ではありません。私たちは常に重要な分野（Java / Kotlin、Python、JavaScript開発者、テスト自動化ツール）の人々を探しています。 私たちは労働市場に的を絞っており、それが変化していることを理解しています。 少なくとも一部の時間はリモートで働く既存の機会に、新しい会社が現れます。 CISから多くの移民がいる創業者の間で、国際企業、特にスタートアップが積極的な役割を果たしています。



私たち、そして私は、すべての読者は、移動することに同意した人々の意見を知りたいと思うでしょう。 または、別のオプションを選択しました。 あなたが去るか、留まるかを決めるとき、あなたにとって個人的に決定的なことは何でしたか？ リモートワークの可能性を検討しましたか？ フリーランス？



情報を事前に感謝し、すぐに会いましょう！
