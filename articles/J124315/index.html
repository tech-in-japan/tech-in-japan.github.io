Google Docs ViewerがZIPおよびRARをサポートするようになりました

Googleドキュメントビューアでは、他の15の形式に加えて、ZIPおよびRAR形式のファイルを表示できます 。 誰かがGmailでZIPまたはRARを送信した場合、「表示」をクリックするだけでブラウザでアーカイブの内容を表示できます。











これにより、アーカイブ内のすべてのファイル、ファイルタイプ、およびこれらのファイルのサイズを確認できます。











ファイルにカーソルを合わせると、「アクション」をクリックしてメニューを呼び出すことができます。 Google Docs Viewerでサポートされているファイルを参照し、PDFがサポートされているファイルを印刷できます。 Googleドキュメントのすべてのファイルをダウンロードおよびダウンロードできます。











他のアーカイブ内のZIPおよびRARアーカイブもサポートされています。 たとえば、ZIPファイル内にRARファイルがある場合（上記の例のように）、このファイルをクリックするだけでこのアーカイブにアクセスできます。



サンプルのZIPファイルを表示してみてください。



この機能は、モバイルデバイス用のGoogleドキュメントでも使用できます。 各ファイル（埋め込みZIPまたはRARを含む）には、表示用のリンクがあります。
