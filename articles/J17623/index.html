＆outerTpl



メニュー出力テンプレートを指定します。



値：使用するチャンクの名前。



可能な変数：



  • [+ wf.classes +]-指定する場所

    使用されるCSSクラス

    （クラス= ""を含む）
  • [+ wf.classnames +]-CSSクラスの名前のみを含む（class = ""を含まない）
  • [+ wf.wrapper +]-表示される場所

    メニューの内容。


例：

<ul id = "topnav" [+ wf.classes +]> [+ wf.wrapper +]



＆rowTpl



一連のメニューの出力テンプレートを指定します。



値：使用するチャンクの名前。



可能な変数：



  • [+ wf.classes +]-指定する場所

    使用されるCSSクラス

    （クラス= ""を含む）
  • [+ wf.classnames +]-CSSクラスの名前のみを含む（class = ""を含まない）
  • [+ wf.link +]-リンクのアドレス（href）
  • [+ wf.title +]-タイトルリンクのテキスト
  • [+ wf.linktext +]-リンクタイトルテキスト
  • [+ wf.wrapper +]-サブメニューを表示する場所
  • [+ wf.id +]-一意の識別子（id）の出力
  • [+ wf.attributes +]-追加の出力

    リンク属性
  • [+ wf.docid +]-のドキュメント識別子

    現在のアイテム
  • [+ wf.subitemcount +]-要素の数

    フォルダー
  • [+ wf.description +]-フィールド値を表示します

    説明。
  • [+ wf.introtext +]-フィールド値を表示します

    イントロテキスト


例：

<li [+ wf.id +] [+ wf.classes +]> <a href = "[+ wf.link +]"

title = "[+ wf.title +]" [+ wf.attributes +]> [+ wf.linktext +] [+ wf.wrapper +]
