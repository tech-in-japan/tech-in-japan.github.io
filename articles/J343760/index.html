Termuxターミナルエミュレーターを使用した電話でのプログラミング

読者へのご挨拶！ この記事では、Androidフォンを使用してプログラムを作成する方法について説明します。 私はすぐに言います-ルート権は必要ありません。



何が必要ですか？



最初に必要なものは、 TermuxアプリケーションがインストールされたAndroid電話です（ Google Playで利用可能）。 端末を完全に使用するには、十分なメモリが必要です。 インストールされた多数のパッケージで、アプリケーションは1.5 GBのメモリを占有します。

また、快適なコーディングのために、電話機がOTGサポートを備えており、キーボードにコードを書き込むためのUSB接続を備えたキーボードが必要です（はるかに便利です）。 そうでない場合は、 ハッカーのキーボードをダウンロードしてください。 Ctrlキーがあり、これは私たちにとって重要です。







テルマックスを選ぶ理由



Termuxには、C / C ++、Python、Golang、PHP、Luaなどのプログラミング言語を操作するのに十分なパッケージがあります。

データベースもありますが、私はmariadbとpostgresqlについてのみ知っています。



開始する





アプリケーションが起動するたびに、ターミナルを操作するための基本的なコマンドを示す挨拶が表示されます。

  • pkg search-パッケージを検索するコマンド。 例：pkg search vim
  • pkg install-パッケージをインストールするコマンド。 例：pkg install vim




コーディングには、テキストエディターが必要です。 ここにはさまざまなオプションがありますが、私はVimを選択し、その操作方法を簡単に説明します。

vimをインストールします。 

$ pkg install vim


vimを使用した例として、Cで「Hello World」と記述します。 

 $ vim main.c


その後、Vimが起動します。 コードの記述を開始するには、「i」キーを押す必要があります。

コードを書きます： 

 #include <stdio.h> int main() { printf("Hello World!\n"); return 0; }


その後、Escキーを押してコロンを入力し、wq（ "：wq"）と記述します。 これにより、ファイルとその変更が保存され（w-書き込み）、閉じられます（q-終了）。



知っておくことが重要です！


w-変更をファイルに保存する

wa-すべてのファイルへの変更を保存する

q-ファイルを閉じる

qa-すべてのファイルを閉じる

wq-変更を保存してファイルを閉じる



次に、このコードをコンパイルする必要があります。 まず、コンパイラーをインストールします。 

 $ pkg install gcc


次に、以下を記述します。 

 $ gcc -Wall main.c -o program


main.c-コードを含むファイル

プログラムは私たちのプログラムです。



これで、プログラムを実行して、動作するかどうかを確認できます。 

 $ ./program


または 

 $ sh program


結果： 

 Hello World!




終わり



これは、電話でのプログラミングの概要です。 継続が必要な場合は、コメントを記入してください：データベース、コード例の操作、他のPLでの実行、端末自体の操作など。
