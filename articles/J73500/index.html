Opera Miniのおかげで、ユーザーは年間80億ドル以上を節約できます









Opera Miniのユーザー数は1か月あたり3,500万人を超えており、処理されるデータの量は2ペタバイトです



2009年10月27日-ノルウェー、オスロ Opera Softwareは本日、Opera Miniブラウザの使用統計に基づいた最新のモバイルインターネットレポートをリリースしました。 ブラウザのおかげで、モバイルデータサービスの総節約額は年間81億ドルに達します。



月次レポートには、世界および個々の地域におけるモバイルインターネットの動向に関する情報が含まれています。 そのフルバージョンは、 www.opera.com / smw （英語）で入手できます。 一般的なレビューに加えて、9月のレポートには、ロシアおよびその他のCIS諸国のモバイルWebの状態に関するより詳細な情報が含まれています。



一般レポートデータ

-2009年9月、3560万人以上がOpera Miniブラウザを使用しました。 これは、8月よりも11.5％、2008年9月よりも150％増加しています。



-3560万人のユーザーが150億のWebページを閲覧しました。 したがって、8月の結果は7.2％、2008年9月は230％を超えました。 Opera Miniユーザーは、平均して1日あたり最大5億のWebページを開きます。



-9月、Opera Miniユーザーは2億2,700万MB以上のトラフィックを生成しました。これは、8月より8.7％、2008年9月より224％増加しました。 Opera Miniでは、データを最大90％圧縮できます。 したがって、ブラウザで処理されるデータの合計量は2.1ペタバイトです。



-Opera Miniユーザー数の上位10か国のリストは、前月と比較して変更されています。 現在では、ロシア、インドネシア、インド、中国、ウクライナ、南アフリカ、アメリカ、イギリス、ポーランド、そして初めてベトナムが訪れています（降順）。 ナイジェリアはベトナムに10位を失い、評価を残しました。



-ロシアの年間ブラウザユーザー数は47.1％増加し、閲覧したWebページは109.6％増加しました。 平均して、各ユーザーは月に7 MBのデータを携帯電話にダウンロードし、Webページの平均ボリュームは18 KBです。



-ウクライナのOpera Miniユーザー数は1年で54.9％増加し、閲覧したWebページは195.1％増加しました。 ウクライナの住民は、ブラウザを使用して、携帯電話で平均10 MBのデータをダウンロードし、1か月あたり569のWebページを閲覧しています。



民主的なモバイルインターネット

Opera Miniを使用すると、モバイルインターネットでお金を節約でき、より多くのユーザーをモバイルインターネットに引き付け、オペレーターに追加の収入をもたらすことができます。

-上位10か国からのOpera Miniユーザーのモバイルトラフィックの節約は、約6億7200万ドル、または年間81億ドルです。 ほとんどのユーザーは、サービスとアクティビティの使用のコストに基づいて、米国と英国を節約します。



ロシアおよびその他のCIS諸国におけるモバイルインターネットの動向

-Opera Miniユーザーの数によるCIS地域の上位11位には、（降順で）ロシア、ウクライナ、ベラルーシ、カザフスタン、アルメニア、ウズベキスタン、モルドバ、キルギスタン、アゼルバイジャン、タジキスタン、トルクメニスタンが含まれます。



-過去1年間で、上位11のCIS地域でOpera Miniを使用して表示されたWebページの総数は133％増加し、ブラウザユーザーの数は55％増加しました。



-トルクメニスタン、キルギスタン、ウズベキスタンは、CISでのユニークなOpera Miniユーザーの年間成長率のトップ11ランキングのリーダーです。



-NokiaとSony Ericssonは、CISのOpera Miniユーザーの間で最も人気のある携帯電話ブランドのままです。

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
