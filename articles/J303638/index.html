Joker 2016 Java Conference：ピーター、10月14〜15日



10月14日と15日に、 Joker 2016 Java Conferenceがサンクトペテルブルクで開催されます。 ロシア、アメリカ、ヨーロッパのJavaスピーカーがジョーカーに来て、彼らの考えやアイデアをあなたと共有します。



ジョーカー4番 より大きく、より強く、より興味深い



これは、サンクトペテルブルクでの4回目のジョーカー会議です。 スピーカーやレポート、スポンサー、スタンド、参加者、専門家が過去よりも多くなるため、すべての訪問者が何かしなければなりません！ 形式は同じままです。2日間、60分のレポート、30分間の休憩-頭が過熱せず、静かに全員と話す時間があるようにします。 私たちはジョーカーの過去にロングブレイク形式を最初に試しましたが、誰もがそれをとても気に入ったので、間違いなくそれを残します。



スピーカー



ジョーカーのスピーカーチームはもちろん、今日の会議の主な誇りです。 現在（6月20日）、12人のスピーカーから確認を受けています。これは、予定されているスピーカーの数の約30％です。







ジョーカーで初めて



いつものように、いくつかの興味深い英語のスピーカーが一度に私たちを訪ねてくるでしょう。

  • Sam Aaron-ケンブリッジの研究者、 Sonic Piの著者-Live Codingモードで音楽を作成するシステム。
  • Charles Nutter -Javaチャンピオン、Red Hat JVMエンジニア、 JRubyチームリーダー。
  • Rafael Winterhalter -Javaチャンピオン、JVM愛好家、ランタイムでのコード生成のためのByteBuddyライブラリの作成者。
  • Andres Almiray -Javaチャンピオン、Groovy、Griffon、JMatter、DbUnit、およびその他の有名なプロジェクトへの貢献者。




退役軍人のジョーカー



  • Alexey ShipilevはOracleパフォーマンスエンジニアで、JVMとJDKのガッツについての一連の興味深いスピーチと記事の著者であり、ロシア語を話す最強の（最強ではないにしても）スピーカーの1人です。 ジョーカーで4回目のパフォーマンスを行います。
  • Oleg Anastasievは、分散システムのエキスパートであるOdnoklassnikovの主要な開発者です。 Jokerで3回目の公演を行います。
  • Vladimir Krasilshchikは、Vert.x、ロギングおよびモニタリングの専門家であるLuxoftのアーキテクトです。 ジョーカーで4回目のパフォーマンスを行います。
  • Vladimir Ivanov -Oracle JVMエンジニア、G1 GC、invokedynamicエキスパート。 Jokerで3回目の公演を行います。
  • Vladimir Sitnikovは、フルスタックプロファイリングとPostgeSQLのエキスパートであるNetCrackerのパフォーマンスエンジニアです。 Jokerで3回目の公演を行います。
  • Alexey Zinoviev -Java / EPAMのBigDataトレーナー。 Jokerで3回目の公演を行います。
  • Kirill Tolkachevは、マイクロサービスの専門家であるAlpha Labのリーディングエンジニアであり、デブリーフィングポッドキャストの編集者および恒久的なホストである DevOpsです。 ジョーカーで2度目に出演します。
  • Alexander Tarasovは、DevOpsとコンテナー化のスペシャリストであるAlpha Laboratoryの主要なエンジニアです。 ジョーカーで2度目に出演します。

スピーカーになる



受付は開いています。 ロシアのJavaコミュニティについて何か伝えたい場合は、レポートを提出してください 。 プログラム委員会にとってトピックが最も興味深いと思われる講演者が優先されます。 他のすべてが等しい場合、過去の録音を提供するスピーカーが優先されます（成功しました！）パフォーマンス：過去のクールなプレゼンテーションは、プログラム委員会が伝統的に指導してきた重要な基準の1つです。 初心者も恐れるべきではありません-私たちは間違いなく皆と話をします。



学生へ



昨年は別の学生の日をやりましたが、今年は大きな問題です。 「大きな」ジョーカーの一部として、おそらく別の学生トラックを作成します。 学生について最終決定を下し次第、すぐに発表します。
