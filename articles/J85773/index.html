Manli ION-優れた機能を備えた小型コンピューター

文字通り人類の生活の最初の10年に入った21世紀は、すべての可能な機器を小型化することを目的としたコンピューター革命をすでに達成しました。 超薄型MP3プレーヤー、PSP Slim、クレジットカード、超薄型携帯電話、Aplle、Sony Vaioラップトップよりも少し厚めのフラッシュドライブ、最近ではバッテリー容量の良い薄型軽量で快適なミニチュアラップトップ、そしてiPadも登場しました。過去10年間のみ。 しかし、進化は飛躍的に進んでおり、グラフィックの巨人Nvidiaから新しいプラットフォーム、ネットトップが手に入ります。 DVDプレーヤーほどの大きさの小さな箱の中のコンピューターは、他にどのように見えるでしょうか？ そのため、消費者はパフォーマンスを要求しています-このアプローチで、NVIDIAはプラットフォームを作成しました-ION。



ロシアの店の棚では会えないというネットトップが私の手に落ちましたが、アジアの国にいると簡単に見つかります。 「赤ちゃん」のメーカーは、マンリの製品であるアジア市場で強力なプレーヤーです。 アジアのトラは強さを増しており、世界市場の大部分を占領しようとしています。そのため、マンリの目は寒く、厳しいロシアでした。 この記事では、ManliのNvidia IONベースのネットトップを紹介します。



しかし、最初に、一見簡単そうに見える質問に答えてみましょう。 なぜIONが必要なのですか？ 8年生の化学の教科書を開いて、イオンの定義を見つけました。 イオンは、単原子または多原子の荷電粒子です。 この定義は、すでに何が内部にあるかについての最初のアイデアをすでに与えています。 本当に原子があります-Intel Atom。 そして、それは本当にシングルコアまたはデュアルコアになります。 モバイルで生産的なプロセッサが内部にあることはすでに称賛に値しますが、IONの料金はいくらですか？ 答えも非常に簡単です。 初めてこのような小さなサイズの製品に出会うと、内部にはIntelの統合されていないグラフィックが表示されますが、これは最新のアプリケーションでは動作しません。 Nvidia ION-これは、ネットトップに組み込まれたGPUのコード名です。GeForce9400に基づいています。出力では、10インチネットブックのサイズのコンピューターが得られますが、生産性は2倍です。







当然、プレスのエンジニアリングモデルは、識別マークのないOEMボックスの手に渡りました。 内部には、ネットトップ自体、電源装置、透明な白いスタンドなど、作業に必要なものがすべてあります。 控えめに言っても、装備はあまり豊富ではありません。 あらゆる種類の「不必要なもの」が付属しているマザーボードの後に​​は、少し困deしているように感じますが、その一方で、パッケージにはそれ以上何もないはずです。







電源デバイスは、現代のネットブックの小型、軽量、中国語の従来の充電器に似ています。 ネットワークからデバイスへの電源コードは、ほとんどの最新の電源に適しています。 ユニークなコネクタでスマートになっていないメーカーに感謝します。







元の形で作られたネットトップの略。 その設計の主な欠点は、ネットトップの固定がないことです。 率直に言って、私はそれを数回落としました。







私の意見では、箱自体はとてもきれいに見えます。 厚さわずか20 mm-プラズマテレビの隣の壁に掛けることができます。







青い花に基づいたシンプルなデザインは、迷惑なだけでなく、女性にもポジティブな感情を引き起こす可能性があります。 このようなガジェットを使用すると、最も冷酷なIT担当者の心をつかむことができます。







ネットトップの上端の垂直位置では、ユーザーはカードリーダー、USBポート、およびマイクとスピーカーを接続するためのコネクタにアクセスできます。 上部のカードリーダーの存在が便宜上正当化される場合、コネクタを備えた残りの存在は少なくとも美的に満足のいくものではありません。 これらのワイヤがネットトップからどのように上昇し、部屋にぴったり収まるかを想像してください。







バックエッジには、LANコネクタ、アナログVGA、4つのUSB、およびHDMIがあります。 HDMIの存在は、「このこと」を通して、大きな対角線でテレビでHDムービーを見ることができるので、私を喜ばせました。

Atom 330プロセッサ、つまり2つの物理コアがあり、ハイパースレッディングテクノロジーのおかげで、仮想コアの数は4つです。 2ギガバイトのDDR2標準メモリは、Windows 7オペレーティングシステムでの快適な作業に十分であり、250 GBのハードドライブにより、映画や音楽の小さなコレクションを保存できます。 テストに移りましょう：







Super Pi 1Mプロセッサのテストは1分35秒で完了しました。 最新のプロセッサの場合は非常に長い時間であり、モバイルのプロセッサの場合も長い時間です。 Intelはコアの数を増やしましたが、現代の標準ではパフォーマンスが十分ではありません。 熱放散が非常に低いことは明らかですが、より高温でより効率的なプロセッサの冷却を改善するために、10 mm幅の広いボックスを使用することをお勧めします。







しかし、マルチコアテストwPime 32mでは、コアの数が大きな役割を果たし、Piの数を計算するのに66秒かかりました。 比較のために、Intel Atom 230（Atom 330の半分）は2倍の時間がかかります。







画像のレンダリングは、強力なPCにとっても難しい作業です。 イオン原子にはほとんど強すぎると言えます。結果は、CineBench R10テストで1550ポイントです。 Intel Core i7 920プロセッサを搭載した最新のコンピューターで得られるよりも約7〜8倍少ない。







DDR2メモリーはほとんど時代遅れですが、このメモリー規格を持つコンピューターの所有者はたくさんいます。 上記のスクリーンショットは、読み取りと書き込みのメモリテストの結果を示しています。 結果は、読み取りと書き込みでそれぞれ2640 Mb / sと2839 Mb / sです。 そのようなプラットフォームの場合、結果は非常にまともです。 CL5 5-5-15のタイミングでメモリを配置できないのはなぜですか？ これにより、パフォーマンスが大幅に向上します。 CL5とCL6の遅延を伴うメモリの価格は実質的に同じであるという事実に照らして、この側面は私にとって謎のままです。







次に、IONプラットフォームのグラフィック機能を確認しました。 3DMark05合成ベンチマークは、2159 3DMarksの結果を示しました。 ゲーム用PCにはほとんどありませんが、ここでは、2年前にリリースされた小さな子供やお気に入りのゲームを楽しませるおもちゃがここで快適に動作します。







1172 3DMarksは3DMark06テストになります。 1年後にリリースされたベンチマークは、3Dシーンの照明を担当するシェーダーでより積極的に機能し始めました。 結果は当然ながら記録ではありませんが、Crysisをプレイするためにそのようなデバイスを使用する予定はありません。







しかし、私はハードドライブの仕事が好きでした、64 Mb / sは2.5インチのハードドライブのまともな指標です。 また、トレントトラッカーには250 GBの容量で十分です。







そして最後に、2313 PCMarksの結果を持つ包括的なベンチマークPCMark05。 このようなプラットフォームでは良い結果が得られます。 それとは別に、ビデオのエンコード速度-170 kb / sに注意してください。このプロセッサの場合、これは期待される結果です。



結論



私はいまいましいIONが好きですが、何に惹かれているのか、完全には理解できません。 私はその寸法が好きです、それは小さく、それを運ぶのに便利です。 彼はハンサムです。 彼に会ったITテクノロジーから遠く離れた人々は、確かに彼に興味を持つでしょう。 静かです-おそらくこれは多くのユーザーの主要な要件の1つです。 Manli IONは、トレントをダウンロードするために安全に一晩放置できます。 しかし、多くの利点があるので、「より速く、より高く、より強く」-さらに生産的になりたいです。 プラットフォームのパフォーマンスはその弱点ですが、作成者はIONのパワーを高める簡単な方法を残しています。 BIOSデバイスには、プロセッサのオーバークロックを担当するパラメーターがあります。 システムバスの周波数を533 MHzから667 MHzに切り替えるだけで、Intel Atomのクロックが2000 MHzになります。 広告のスローガンを覚えておいてください-無料で25％。
