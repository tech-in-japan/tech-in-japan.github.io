10か月後、TrueCrypt 5.0の新しいバージョンが誕生しました

暗号化と暗号化のためのプログラムは数多くありますが、 TrueCryptには興味深い機能がいくつかあります。そのおかげで、ほぼ2年間、2台のマシンで動作していました。 そして、新しいバージョン5.0には、さらに多くの利点が追加されています...



最新バージョン4.3aは2007年5月にリリースされ、120万回以上ダウンロードされています。 10か月後、バージョン5.0が表示されます



TrueCryptの簡単な機能（第5バージョンの太字の新製品はマークされています）：
  • オープンソース
  • 最新のすべてのアルゴリズムを使用した暗号化：AES、Serpent、Twofish、2重および3重の組み合わせ
  • ハッシュ関数：RIPEMD-160、SHA-1、 SHA-512 、Whirlpool。
  • Windows Vista / XP、 Mac OS X 、Linuxで動作します
  • Windowsで2倍高速の読み取り/書き込み速度
  • WindowsおよびLinuxでコマンドラインおよびGUIを使用して作業する
  • 起動前認証でWindows起動パーティションを完全に暗号化する機能
  • LRW記録モードをより信頼性の高いXTSに置き換える
  • コンテナをファイル、パーティション、またはディスクとして作成する機能
  • 特定のファイル（ディスク）が暗号化されていることを証明できないため、ヘッダーすらありません
  • 最もおいしいことは、検出できない別の内部に隠された暗号コンテナを作成する機能、および/またはメインコンテナからパスワードを知っていてもその存在を証明する機能です（「熱暗号解析」に対する保護方法:)
  • 暗号化は完全に透過的で、リアルタイムです
TrueCryptのウェブサイトは現在プレッシャーにさらされているので、そうは思われないことに注意してください...
