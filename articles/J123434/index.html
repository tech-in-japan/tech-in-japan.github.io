TechEd Conferencesブランドの歴史





多くのITプロフェッショナルはTechEdブランドに精通しています。 TechEdは、世界中でマイクロソフトが主催する世界的に有名な会議です。 TechEdブランドのカンファレンスでは、何千人ものマネージャーやエグゼクティブ、プログラマー、IT専門家、各分野の専門家が集まり、カンファレンス当日に数十または数百もの技術レポートを提供します。



この記事では、TechEdのブランドとしてのストーリーを簡単に紹介し、この名前で開催されるカンファレンスの最も興味深い瞬間について説明します。



開始する



それはすべて、1993年にIT標準によって開始され、TechEdという最初の会議が米国フロリダ州オーランドで開催されました。 今年、マイクロソフトは会議の参加者に新しい頭脳を発表しました-Windows 3.1オペレーティングシステムは後に非常に人気がありました。 さらに、主要なトピックはMS DOSオペレーティングシステムであり、当時は人気のピークでした。 その期間の興味深い製品のうち、Microsoftが会議の直前に導入したMicrosoft Visual Basicを区別できます。 後で、Visual Basicも大成功するでしょう。



最初のTechEdは非常に真剣に始まりました-3,000人の技術専門家が訪問しましたが、主催者にとっても驚きでした。 専用のホテルでは、参加者の流入にほとんど対処できませんでした。 すぐに終了したため、事前登録がなくても多くの専門家が来ました。 一般に、最初のTechEdは大成功でした。



1994年と1995年の次の2つの会議は、米国ニューオーリンズで開催され、新しいイベントの成功した開始を確保しました。 1994年、同社は、Windows NT、Access、FoxPro、VBA、Visual C ++、OLE、MAPI、SQL Serverおよびその他の技術に加えて、新しいWindowsプロジェクト「Chicago」（将来のWindows 95）について既に話し合いました。 1995年、カンファレンスは1つのイベントで400以上のレポートを開発者に提供することで、すべての記録を突破しました。 1995年のTechEdは5日間続きました。



1996年、ロサンゼルスでTechEd会議が開催され、すでに7,000人の参加者が集まっていました。 このイベントは、マイクロソフトの取締役会長兼CEOであるビルゲーツ自身によって開かれました。 ゲイツの全体講演では、インターネットと、それがWindowsおよびWindows NTオペレーティングシステムにどのように統合されているかに焦点を当てました。 会議のもう1つの主要なトピックはActiveXテクノロジーでした。







1997年はTechEdの歴史において重要な年でした。 まず、会議はオーランドで再び開催され、5月5日から9日まで開催され、記録的な9,000人の参加者を集めました。 そして、第二に、会議は開発の新たな一歩を踏み出しました-初めて会議のヨーロッパ版がTechEd EMEAという名前で組織されました。 このイベントはフランスのニースで開催されました。 3,700人が参加しました。 組織されたライブ会議の参加者は、ビル・ゲイツによって迎えられました。



1998年以来、TechEdカンファレンスで、マイクロソフトはASP、Internet Explorer、DHTML、およびIISサーバーでのインターネット技術の開発を積極的に推進してきました。 1999年、Tech DNAはWindows DNAの技術を導入し、後に.NETプラットフォームに変換しました。



2001年のTechEd会議での議論の中心的なトピックは、サーバー側、新しいC＃プログラミング言語、および新世代の開発ツールであるVisual Studio.NETを含む.NETプラットフォームでした。



2002年、TechEd North Americaは4月にニューオーリンズで開催されました。 これはTechEdカンファレンスの10周年でした。 エリック・ラダーは今年の会議で全体報告書を作成し、2002年の少し前に開始されたVisual Studio.NETの新製品について話しました。



2003年のダラスでの会議で、新しいWindows Storage Server 2003製品が発表され、コードネームがYukonであるSQL Serverの新しいバージョンと.NET Frameworkの2番目のバージョンについて議論しました。



2004年は、Longhornというコードネームの次世代Windowsの議論の年でした。



既にTechEd North Americaブランドで開催されていた2008 TechEdは、Microsoftを代表してBill Gatesが最後のスピーチをした歴史的なイベントで記憶に残りました。彼はデータセンターと新しいWindows 7オペレーティングシステムの開発計画について話しました。

私たちの日々



今日の中央TechEd North Americaは、何千人もの参加者を集める巨大なイベントです。 TechEd NA 2011カンファレンスでは、世界中の84か国から10,000人の専門家が集まり、500を超える技術セッションとレポートを聞き、250の実験室作業に参加しました。



最新のTechEdイベントの興味深い発表：
  • TechEd Europe 2009-新世代のMicrosoft Exchange Server 2010がリリースされ、SQL Server 2008 R2の予備バージョンが利用可能になりました
  • TechEd North America 2010 -Windows 7 SP1、Windows Server 2008 R2 SP1、およびExchange Server 2010 SP1のプレリリースバージョン、Windows Azureの開発の発表、Communications Server“ 14”の詳細が公開されています。
  • TechEd Europe 2010 - Cloud Powerイニシアチブの開始が発表され、Dell、Fujitsu、Hitachi、HP、IBM、NECとともに、Hyper-V Cloudプロジェクトが開始され、Hyper-V Cloud Fast Trackが開始され、Forefront Endpoint Protection 2010の予備バージョンが利用可能になりました。
  • TechEd North America 2011 -2010年版に代わるVisual Studioの新バージョンの詳細が初めて発表され、Windows Azureクラウドサービスの多数の拡張が発表され、リリース日に関する発表が行われ、新製品バージョンの予備バージョンが利用可能になりました：SQL Server Denali、WP7、Hyper- V、System Center 2012。


地域会議







TechEd EMEA /ヨーロッパ


1997年以降、欧州TechEd EMEA会議がほぼ毎年開催されています。 会議は1998年（ニース）、1999年と2000年（アムステルダム）、2003年（バルセロナ）、2004年と2005年（アムステルダム）、2006〜2008年（バルセロナ）、2009年と2010年にTechEd EMEAがベルリンで開催されました。



2011年、TechEd Europeカンファレンスは2012年に復帰するために休憩を取りました。 6月25日から29日まで、次のTechEd Europe 2012がアムステルダムで開催されます。



2003年、TechEdはさらに一歩踏み出し、会議は大幅に拡大し、新しい国々が新しい大陸にもたらされました。 今回、オーストラリア、インド、イスラエルがTechEdを受け取りました。



TechEd Australia、TechEd New Zealand


ITプロフェッショナル向けの最大のカンファレンスであるオーストラリア版は、オーストラリアのブリスバーンで2003年にその歴史を開始しました。 その後、毎年恒例のTechEdのオーストラリア版がキャンベラ（2004）、ゴールドコースト（2005、2007、2009、2010）およびシドニー（2006、2008）の都市で開催されました。



オーストラリアの会議と一緒に、会議は毎年近隣のニュージーランドで開催されました。 TechEd New Zealandのイベントは、ほぼ常にニュージーランドのオークランドで開催されました。 2010 TechEd New Zealand Conferenceで、数十の論文が発表されました。



2011年8月末に次のTechEd New Zealand 2011が開催され、9月のTechEd Australia 2011が開始されます。



TechEd India


TechEd India Conferenceの歴史は2003年に始まりました。 それ以来、会議は開催されなかった2008年の例外を除き、毎年開催されています。



2009年、Steve BallmerはTechEd Indiaで講演し、米国外のTechEd地域会議でのBallmerの最初の講演でした。 2009年には、会議では28のテクニカルトラックと112のレポートと24の実験室作品が訪問者に提供されました。



最後のインドのイベントは、2011年3月にインドのバンガロールで開催されました。



TechEd Israel


2003年にエイラート市で開催された最初のTechEd Israel Conferenceでは、1,500人が集まりました。 TechEd Israel 2010会議は2010年11月に開催され、3,000人以上が集まりました。会議の主なトピックはOffice 365、Lync、およびSharePoint 2010テクノロジーでした。



2005年、TechEdカンファレンスは再び地理的存在感を高めました-アフリカ版のTechEd Africaが発売されました。



TechEd Africa


TechEd Africaカンファレンス（2007年までTechEd South Africaと呼ばれていました）は、最近TechEd North Americaと呼ばれた元のTechEdの次の国際的な類似物になりました。 最初のアフリカのTechEdは、2005年に南アフリカのサンシティで開催されました。 次の2006年と2007年に、会議はサンシティで開催されましたが、2008年から2010年にかけて、TechEd Africaの開催地は南アフリカのダーバンに変更されました。



2011年、10月17日から20日まで、 TechEd Africa 2011はダーバンでの扉を再び開きます。



TechEd中東


これら2つの会議は、TechEdブランド会議の最新のものです。 最初に、2010年3月、アラブ首長国連邦のドバイでTechEd中東会議が開催され、2010年9月にブラジルのサンパウロで南米TechEdラテンアメリカが開催されました。



TechEd Middle East 2011は、2011年3月にドバイで開催されました。



TechEdの話はこれで終わりではありません。 2011年には、ブランドは再び拡大されます。 TechEd Russiaという名前の新しい会議が、ITプロフェッショナル向けの世界最大のイベントの舞台に登場します。



もっとTechEd


これらの会議に加えて、TechEdブランドの世界にはさらに多くの地理的な場所があります。 これはTechEd China、TechEd Japan、TechEd Brasil（別名TechEd Latina America）です。 ほぼ全世界がTechEdカンファレンスでカバーされており、毎年世界中の何万人もの専門家が集まるイベントです。



そして、2011年に、この大規模なカンファレンスファミリーが別の国（そして実際には地域全体）であるロシアで補充されることを非常に嬉しく思います。



TechEd Russia

このイベントのプログラムには、150を超えるテクニカルレポートとインタラクティブセッションが含まれており、参加者は主要な専門家や開発者に関連するIT問題、製品や技術を実際に知っているための実験室での作業、Ask a Expertゾーン、イノベーションの展示などと議論できます。



最初のTech ∙ Ed Russia 2011カンファレンスの主要トピック：クラウドコンピューティングとサービス、Windows Phone 7モバイルプラットフォーム、およびカンファレンスのもう1つの重要なトピックは、後日発表されます。
