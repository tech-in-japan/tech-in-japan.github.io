Firefox 3.6の新しいハードウェアAPI：オリエンテーション

Firefox 3.6に参加する新しい機会の1つは、Webページが利用可能になったときに、空間内のコンピューターの向きに関する情報にアクセスする機能です。 デモンストレーションでわかるように（Habrahabrでは翻訳で<video>を使用できないため、元のソースでのみ）、この情報を使用して、コンピューターが移動しているか、どの方向に回転しているかを確認できます。



APIの使用は非常に簡単です。 単純なイベントリスナーを追加するだけです。
window.addEventListener（ "MozOrientation"、関数（e）{

/ * 3つの値：ex、ey、ez * /

}、true）;
APIを最初にモバイルブラウザの今後のリリースの望ましい部分として構築したので、デスクトップシステムでもAPIを利用できるようにしました。 最近の多くのMacbookとThinkpadには、コンピューターの位置を表示するデバイスとドライバーが含まれています。 Linux、Mac、およびドライバーとデバイスを備えた一部のPinkpadのサポートを追加しました。 （一部のMacBookでは、実際の向きとは反対の向きを決定することに注意してください。現在も作業中です。）



詳細については、 Doug Turnerによる 2つの ブログ投稿と、 Eric Shepherdによる最新のオリエンテーションドキュメントを参照してください。
