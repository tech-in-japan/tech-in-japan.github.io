* nixのコンテンツブロックの同期と修正

Opera Linkで何かを変更してから長い時間が経ちました。 このスナップショット10.70では、コンテンツブロックを同期する機能を示しています。 これにより、ブロックされたアイテムのリストを簡単に作成および管理できます。



このスナップショットには、Linux / FreeBSD用のいくつかの修正も含まれています。



既知の問題

  • 既存のURLが同期しない場合があります：同期を有効にする前にブロックされたアドレスのリストが既にある場合、Opera Linkを入力して作業を開始する前に、同期のためにコンテンツをブロックすることにより、ブロックされたアドレスのリストにURLを追加する必要がある場合があります。


ダウンロードする

  • Windows MSI / Windowsクラシック
  • Mac（ユニバーサル） / Mac（Intelのみ）
  • Linux / FreeBSD
