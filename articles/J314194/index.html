UX / UIデザイナーのブックマークに興味がある

他の人の優れた作品を見たとき、あなたはどう思いますか？ きっとあなたは私たちと同じように彼らを賞賛するでしょう。 しかし、もっと重要なのは、どのように人々がこれを達成したのか、どのように成功したのか疑問に思われるかもしれません。 彼らはどこからインスピレーションを得ましたか？ 彼らはインターネットで何を検索し、閲覧しましたか？ UI / UXデザイナーがどのような興味深いものをブックマークできるかを見てみましょう。



プロトタイピング



1.ジャスティンマインド



高精度のプロトタイプを作成するためのツール。 モバイルアプリケーションのプロトタイプを作成する方が便利です。 ジェスチャーのサポートにより、他のツールとは一線を画しています。







2.モックプラス



Mockplusは、プロトタイピングツールの新星で​​す。 バージョン2.2では、その機能と使いやすさが大幅に改善されています。 HTMLエクスポートのサポートを提供します。これは、Mockplusが他のツールよりも優れていることを示唆しています。 この開発動向を過小評価することはできません。







3. Axure



JustinmindとMockplusに言及しましたが、もちろん、Axureのことを忘れることはできませんでした。 これら3つのツールは常に比較されています。 しかし、Axureは「兄貴」であり、その主導的地位を維持しています。







グラフィックデザイン



1. Behance



デザインコミュニティのニュースに積極的に興味を持っているなら、おそらくBehanceについて知っているでしょう。 これは、世界中の優秀なデザイナーが作品を投稿する最大のコミュニティの1つです。 ここでは、あらゆる好みのグラフィックデザインに関する多くの素晴らしい作品を見つけることができます。 一流の仕事は、常にデザインを学ぶための最良のツールです。







2.ドリブル



Dribbleは、UIデザイナーにとってもう1つの便利なリソースです。 最高のデザイナーがサイトの生活に積極的に関与しているため、作品を研究し投稿するための素晴らしい選択肢と考えることができます。 参加するには、招待状を受け取る必要があります。 また、6年間Dribbleに携わっ​​たデザイナーのアドバイスが必要な場合もあります。







ウェブデザイン



1. Awwwards



Awwwardsは、優れたデザインのクリエイティブサイト専用の優れたリソースです。 毎日その日のサイトがここで選択されます。 新鮮で独創的なアイデアが必要な場合は、Awwwardsでたくさん見つけてください。







2. DesignMoo



ここでは、最高のデザイナーがボタン、アイコン、ページ効果などのPSDソースを投稿しています。



DesignMooの目標は、デザイナーを結び付けるプラットフォームを作成することです。その中で、ソースコードを共有し、適切なオプションを探し、それらに自由にアクセスできます。







3. Webブレンド



Web Blendは、デザイナー、プログラマー、および最新のテクノロジーに夢中な人々のコミュニティーです。 ここでは、Webデザインの世界からの最新のニュースとトレンドを見つけることができ、便利なツール、サイト、アプリケーションへのリンクを見つけることができます。







4. PrimeLiber



PrimeLiberは、豊富なトピックのセットを持つロシア語のリソースです。 ここでは、UI / UXデザイン、SEO最適化、プログラミング、および製品開発とデザインに関連するその他の問題に関する記事を見つけることができます。







フォントデザイン



タイポグラフィが大好き



タイポグラフィは、著者がフォント、テンプレート、情報、印刷業界での経験を共有しているブログです。 フォントデザインに関する最良かつ最も興味深い記事を以下に示します。



ひらめき



インスピレーションが尽きてきたら、携帯電話とiPadを脇に置き、著名なデザイナーのテクニックとスキルを説明した本を読んでください。 多くの創造的なアイデアをご覧ください。 ところで、私たちはすでに選択をしています-UI / UXデザイナーにとって最高の本とリソースです。



デザイナー向けのプロフェッショナルリソース （この記事はロシア語）の選択も、強力なインスピレーションの源になります。



おわりに
