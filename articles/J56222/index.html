ロシアのUbuntu LoCoチームが公式ステータスを付与

2009年3月31日20:00 UTCに、Ubuntu LoCo評議会の会議が開催されました。この会議では、特にロシアのUbuntu LoCoチームを公式にする問題が解決されました（会議の転写： http ://irclogs.ubuntu.com/2009/03/ 31 /％23ubuntu-meeting.html（21:07-21:40 ））



Ubuntu LoCo Teamとは何ですか？

Ubuntu LoCo Team（地元のLo Cal Co mmunity-ローカルコミュニティ）は、Ubuntuユーザーのコミュニティが地域ごとに結集したコミュニティです。 LoCoは、ネイティブ言語でのUbuntuのサポート、ローカルユーザーの一般的なサポート、その地域でのUbuntuのプロモーションなどのプロジェクトに関与している可能性があります。 しかし、LoCoが行う最も重要なことは、隣人を集めてお互いに「直接」助け合うことです。



ロシアのUbuntu LoCoチームとは何ですか？

ロシア語のUbuntu LoCo Teamは、ロシア語を話すUbuntu Linuxユーザーのコミュニティであり、地球のロシア語を話す人々の間でUbuntuを促進するために集まっています。



現在、ロシアのUbuntu LoCoチームの作業は、Dmitry Agafonov（ https://launchpad.net/~dmitry-agafonov ）とAlexey Molchanov（ https://launchpad.net/~alexey-molchanov ）によって調整されています 。 ランチパッドグループ： https : //launchpad.net/~ubuntu-ru/
