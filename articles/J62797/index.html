いくつかのeSATAフラッシュドライブ

今回、 ウォルトンチェーンテックはハイブリッドUSBフラッシュドライブの分野で卓越し、 APOGEE Prontoと呼ばれる一連のデバイスの市場での差し迫った外観を発表しました。





新しいフラッシュドライブにはeSATAおよびUSB 2.0インターフェイスが装備されており、重量はわずか18グラム（サイズ79.5 x 30 x 10.3.3 mm）、容量は32および64 GBです。 現在のすべてのOS（Windows 98 / SE / 2000 / ME / XP / Vista、Mac OS 9.x / 10.0.x / 10.1.x / 10.2.x / 10.3.xおよびLinux 2.4.x）と互換性があります。



USBデータ転送速度は、毎秒32 MB（読み取り）および毎秒25 MB（書き込み）です。 eSATAの場合、この数値ははるかに高く、読み取りと書き込みの場合、それぞれ93 MB /秒と52 MB /秒です。



しかし、Walton Chaintech APOGEE Prontoの販売と価格のタイミングはまだわかっていません。
