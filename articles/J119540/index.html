Debianデスクトップへのリモートアクセス

こんにちは Debian OSを備えた専用サーバーを持っていますが、これは私から数百キロ離れています。 SSHを使用して管理します。 しかし、今はうんざりしているので、デスクトップに視覚的にアクセスしたいと思っています。 最適なもの、最適なものに関する記事とヒントの山をシャベルで削り、いくつかのオプションを選択しました。



X11転送 ：便利に、簡単に、はっきりと話す。 Windows用のPuttyとXサーバーが必要です（Xmingは許可されます）。 しかし、私はMac OSを使用しているため、このオプションは私には向いていませんでした。



VNC ：これについていくつかの記事を見つけましたが、X11と同じくらい簡単であることがわかりました。 すぐに、Debian用のVNCサーバー、VNCのChickenというVNC4ServerとMac OS用のVNCクライアントを見つけました。



そして、インストールを進めましょう...



VNC4Serverをインストールする





1. VNC4Serverをインストールします 

apt-get install vnc4server xinetd




2. VNC4Serverの最初の起動 

 vnc4server


VNC4Serverの起動時に、リモートアクセス用のパスワードの入力を求められます。

パスワードを入力すると、サーバーが起動し、アクセス用のパラメーターを次のように書き込みます。 

 New '178.63.15.15:1 (root)' desktop is 178.63.15.15:1




3.サーバーを停止します 

 vnc4server -kill :1


ここで、1は接続パラメーター内のコロンの後の数字です。



4.ファイルの編集〜/ .vnc / xstartup 

 nano ~/.vnc/xstartup


次の行のコメントを解除します。 

 unset SESSION_MANAGER exec /etc/X11/xinit/xinitrc


最終的には次のようになります。 

 #!/bin/sh # Uncomment the following two lines for normal desktop: unset SESSION_MANAGER exec /etc/X11/xinit/xinitrc [ -x /etc/vnc/xstartup ] && exec /etc/vnc/xstartup [ -r $HOME/.Xresources ] && xrdb $HOME/.Xresources #xsetroot -solid grey #vncconfig -iconic & #x-terminal-emulator -geometry 80x24+10+10 -ls -title "$VNCDESKTOP Desktop" & #x-window-manager &




5.ファイル/ etc / X11 / xinit / xinitrcの属性を変更し、実行可能にします 

 chmod 755 /etc/X11/xinit/xinitrc




6. VNCサーバーを起動します 

 vnc4server




VNCのチキンとのリモートデスクトップ接続





1. VNCのチキンを起動し、VNCログインウィンドウが表示されます

2. [ ホスト]フィールドに、サーバーのIPアドレス（VNC4Serverが起動時に提供した）を入力します（ 例：178.63.15.15）

3. [ 表示]フィールドに、VNC4Serverが起動時に提供した番号を入力します（ 例：1）。

4. [ パスワード]フィールドに、VNC4Serverを最初に起動したときに指定したパスワードを入力します

5. 接続をクリックします



そして、もしあなたがデスクトップを持っているなら-おめでとう！ :)

目を切る素敵なメッシュと、十字の形のカーソルがある場合は、 GNOMEをインストールする必要があります。 これを行う方法を以下に説明します。



GNOMEのインストール





1. GNOMEインストールを実行する 

 apt-get install gnome


インストールは長くて退屈になります...必要に応じてカスタマイズしてください。



2. GNOMEをインストールした後、VNC4Serverを再起動します 

 vnc4server -kill :1


それから 

 vnc4server




3. VNCクライアントを使用して再接続します。 これで、カラフルなDebianデスクトップと2人のパパがいるはずです。 良い使い方をしてください;）
