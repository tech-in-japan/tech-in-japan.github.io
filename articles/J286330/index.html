典型的なWindowsタスクの高速化

すべてが以前より良くできます。 /ヘンリーフォード/


私は毎日平均10時間コンピューターで作業しています。 この記事では、Windows 7を構成して一般的なタスクの実行を高速化する経験について説明します。これは役立つ場合があります。



一般的に、タスクに費やす時間を最小限に抑えることになります。 タスクの完了に要する時間が短いほど、単位時間内に実行できるタスクの数が多くなり、生産性が向上します。



一般的なタスクの実行を高速化するために、いくつかのタイプのアクションを強調しています。

  1. 追加のソフトウェアを使用する
  2. Windowsインターフェイスとアプリケーションの構成
  3. ホットキーを使用する
  4. システムの高速化（未使用の機能を無効にするため）




1.追加のソフトウェアを使用する



1.1。 Launchy www.launchy.net

Windows 98に戻って、プログラム起動ウィンドウの使用を開始しました（ [スタート]→[実行] ）。 この組み合わせを使用して、基本的に3つのアプリケーションcmd.exe、notepad.exe、calc.exeを実行しました。



Launchyユーティリティに出会うまで、この機能の制限に耐えました （彼らはGeektimesでこのユーティリティについて書いています）。

このユーティリティの主な利点は次のとおりです。

  • 開始コマンド入力ウィンドウはウィンドウの中央に表示されます（画面の左下隅に目を移動する必要はありません-これは21インチ以上のワイドスクリーンモニターに特に当てはまります）。
  • 最初の文字と部分文字列を入力した後に開始できます
  • Launchyが通過するフォルダーのリストを選択し、名前の一部を入力することで、後で起動できるアプリケーション/ファイルの名前を記憶できます
  • スキンのサポートがあります（下の写真はQuickSilver2スキンを示しています）






システムへのインストール後、 Alt + Spaceの組み合わせを使用してLaunchyが起動します。 プログラムの設定により、この組み合わせを変更できます。



私はすでにWin + Rの組み合わせを知っていて慣れているので、同じ組み合わせでLaunchyを「ハング」させたかったのです。



ただし、アプリケーション設定でWin + Rを設定しようとすると、この組み合わせが既に使用されているという通知が表示されます。



私のように、 Win + RでLaunchyを起動したい人は、システムによるこの組み合わせの処理を無効にする必要があります。 これを行うには、レジストリに次のパラメーターを入力します。 

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced] "DisabledHotkeys"="R"


レジストリを自動的に変更するには、 このファイルを使用できます 。 行われた変更をキャンセルするには- これ 。



代替： AutoHotkey （ toxicdreamに感謝）



1.2。 プントスイッチャー punto.yandex.ru

キリル文字とラテン文字を入力する場合、 こんにちは、私はこのGhbdtnかHELLOのように書くと状況は避けられません。

このような場合にテキストを書き換える代わりに、Punto Switcherを使用します。

プログラムをインストールした直後に、自動変換をオフにし（予測できない場合もあります）、手動モードで起動し、ホットキーを使用します。

Shift + Break / Pause 前の段落と同じですが、選択したテキストの断片について（フレーズ、文、または誤ったレイアウトで入力されたいくつかの段落を修正する必要がある場合にこの機能を使用します）
Alt + Break / Pause 選択したテキストフラグメントの大文字と小文字を逆にします（ PRIVET → Hello ）

1.3。 UNIXユーティリティ unxutils.sourceforge.net

UNIXシステムとかけがえのないUNIXコマンドラインユーティリティに精通しているにもかかわらず、私はWindowsで仕事をしています。 Windowsで同様の機能を得るために、Windowsに移植されたUNIXユーティリティに出会うまでCygwinを試しました。



必要なユーティリティを％SYSTEMROOT％\ System32 \ディレクトリに配置すると、UNIXシステムで以前に操作したのとほぼ同じ方法でコマンドラインから操作できるようになります。



1.4。 Locate32 Locate32.cogit.net

このファイル検索エンジンには、ファイルや文字のコンテンツ（Windows SearchやGoogleデスクトップなど）のインデックスを作成できないという事実にもかかわらず、いくつかの利点があります。

-ローカルおよびネットワークドライブ上の高速検索ファイルとフォルダー

-高度な検索フィルター（RegExpを含む）





代替手段： すべての検索エンジン （ ditgeekに感謝）、 MasterSeeker



1.5。 AkelPad akelpad.sourceforge.net

標準のWindowsメモ帳（notepad.exe）を使用してテキストファイルを操作する場合、大きな欠点があります。ファイルの操作速度が非常に遅く、そのボリュームは数メガバイトを超えます。

私はメモ帳の代替手段を試しましたが、テキストファイルを使用した最も簡単な作業では、次の利点があるため、AkelPadを使用しました。

  • 大きなファイルをすばやく開く
  • クイック検索とテキストの置換
  • 列の選択
  • 追加のプラグインをインストールする機能（構文の強調表示、行番号付け、...）


（他のエディターと比較して）追加の便利な機能は、標準のWindowsメモ帳を置き換える機能です。



2. Windowsインターフェイスとアプリケーションの構成



2.1。 メモ帳で任意のファイルをすばやく開く

かなり頻繁に、メモ帳でファイルを開く必要がある場合があります（たとえば、* .html、* .csv、または新しい未知のファイル拡張子）。 この操作を簡素化するために、メモ帳で編集するアイテムを右クリックしてコンテキストメニューに追加します。

はっきり表示




次の情報をレジストリに入力すると、このアイテムが追加されます。 

 [HKEY_CLASSES_ROOT\*\shell\Edit with Notepad\command] @="notepad.exe %1"


レジストリを自動的に変更するには、 このファイルを使用できます 。 行われた変更をキャンセルするには- これ 。



2.2。 デスクトップでは、一般的なアプリケーションやフォルダーへのショートカットに加えて、さらに4つのショートカットを追加します

5秒後に電源をオフにします（5秒。遅延をキャンセルするため、誤ってアクティブにしたため、特に遅延を行います） 

 %SystemRoot%\System32\shutdown.exe /s /f /t 5


10分（600秒）後のシャットダウン 

 %SystemRoot%\System32\shutdown.exe /s /f /t 600


システムの再起動 

 %SystemRoot%\System32\shutdown.exe /r /t 1


再起動/シャットダウンのキャンセル 

 %SystemRoot%\System32\shutdown.exe /a


これらのショートカットを画面の右上隅に配置すると、次のようになります。



ショートカットのアイコン 
 %SystemRoot%\system32\SHELL32.dll


2.3。 [スタート]メニューで、よく使用するアプリケーションやフォルダーにアクセスするためのショートカットを追加します





最も一般的な使用例：左手の親指で[スタート]ボタンを押します。 メニューが開きます（ほとんどすべてのアプリケーションの上部）。右手は既に自動的に[スタート]メニューの上部に移動しています（手がマウスにかかっている場合）、またはキーボードの上下キーを押してからEnter（手を置いている場合）キーボード）。



新しいレターを作成するモードでOutlookを起動するコマンド 

 "C:\Program Files (x86)\Microsoft Office\Office14\OUTLOOK.EXE" /c ipm.note


2.4。 Windows計算機の外観を変更しています

「エンジニアリング」ビューを選択します-これにより、アプリケーションウィンドウが拡張され、結果フィールドが約2倍になります。

「数字による数字のグループ化」を追加-これにより、大きさの順序をすばやく（「目で」）評価できます。

3.ホットキーを使用する



多くの人がメインのホットキー（Ctrl + C、Ctrl + Vなど）を知っているので、このセクションでは、あまり知られていない組み合わせを挙げて、日常の操作の多くを実際に単純化および高速化します。

ホットキー 説明（使用例）
Ctrl + Shift + N エクスプローラーまたはデスクトップに新しいフォルダーを作成します
Shift +マウスの右ボタン

（ファイルまたはフォルダー上） 		追加の項目「パスとしてコピー」がコンテキストメニューに表示されます。 このアイテムを選択すると、ファイルへのフルパスがクリップボードにコピーされます（ファイルにフルパスをコピーした後、メッセージにファイルを添付する必要がある場合に使用します。残りは、メーラーのダイアログボックスでCtrl + Vを押すだけです） さらに、「送信」アイテムでShiftキーを押すと、追加の宛先フォルダーが表示されます
Shift + Win +左/右 アクティブなウィンドウを隣接するモニターにリダイレクトします（複数のモニターのユーザーに関連）
Shift + Delete ゴミ箱に移動せずにファイルを削除する（明らかに不要なファイルからゴミ箱を空にしないようにするのに役立ちます）


4.システムの高速化（未使用の機能を無効にするため）



システムに十分なリソースがない場合、システムの動作が遅くなります。⇒ソフトウェアも生産的な作業スキルも役に立ちません。

この記事のこのセクションでは、ハードウェアのためにシステムを高速化し、より高速なプロセッサ、より多くのメモリ、SSDを推奨することについては触れません。もちろん、システムの速度と反応を向上させるのに役立ちます。

代わりに、有用な結果をもたらさずにシステムリソースを消費するWindowsのコンポーネントと機能を無効にして、リソースを解放するオプションを検討することをお勧めします。
