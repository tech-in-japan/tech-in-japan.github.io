サイトをチェックするための7つの無料サービス（知らないかもしれません）



多くの場合、さまざまなオンラインサービスを使用して、サイトの可用性とその表面テストとチェックを確認する必要があります。

同僚の簡単な調査が示したように、これらのサービスのほとんどすべてが異なります。 見せてあげてください 。



LoadImpact： http ：//loadimpact.com、50の同時接続の無料テストがあるサイト負荷テストサービス（最大5kの同時接続）。 冷酷で容赦のない国内のホスティング業者を1か月あたり0.01ドルでホストしていて、サーバーにサイトが1,000しかない場合、独自のサイトをチェックしてみてください。



BrowserMob： http ://browsermob.com。4つの異なる場所からターゲットサイトのダウンロード速度を確認し、それぞれの詳細なダウンロードメトリックを表示できる追加のシンプルだが素晴らしいサービス。 BrowserMobの主なタスクは、サイトの自動クロスブラウザーテストです。



Alertra： http ://alertra.comは、11の異なる場所から無料（および応答時間）でサイトをチェックできるサイトの稼働時間監視サービスです。 すばやく確実に動作し、キャプチャやその他のゴミはありません。



site24x7： http ://site24x7.com、すでに23の地理的に分散したチェックポイントを使用しているAlertraのようなサービス。



シグナルを取得します： http : //www.yougetsignal.com 。ホスト/ IPで開いているポートをチェックできる優れたサービスです。 多くの場合、NATを介したポート転送の確認に役立ちます。



それでは、アクセシビリティチェックからもう少し複雑なものに移りましょう。 私はw3c製品については書きませんが、とにかく誰もがそれを知っていて使いたいと思っています。



CSE HTMLバリデーター： http : //www.onlinewebcheck.comは 、有名なvalidator.w3.orgに代わるもので、独自の分析コアに（見かけ上）構築されています。 エラーや潜在的な問題を修正するための非常に詳細で理解しやすい推奨事項と比較して有利です。



Built with： http : //builtwith.com/は、指定されたサイトを分析し、Webサーバー、CMS、どの統計サービスが使用されているかなど、サイトに関する全体の関係書類を提供します。 多くの指標があり、競合他社の大まかな分析に非常に適しています。 また、サービス所有者が監視対象のすべてのパラメーターの統計を発行するTrends.builtwith.comを掘り下げることは興味深いことです。 たとえば、大多数のサイトは引き続きGoogleアナリティクスに古いブロッキングJSコードを使用しています。
