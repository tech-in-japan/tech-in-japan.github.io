MRLの概念の選択の準決勝の結果

2012年9月12日に、Skolkovo MDモバイル診断デバイスコンセプトセレクションの準決勝が行われ、7月初旬に開始されたファンドプログラムにより、ロシア、ベラルーシ、ドイツ、スイスの50チーム以上が参加しました。 すべての参加者は重要な仕事をしましたが、気付かれることはありません。 情報技術と医療の分野の専門家で構成される審査員団は、13のチームのリストを決定しました。これらのチームのコンセプトは、規定された要件を最大限に満たしています。 これらのチームは、2012年12月12日に行われる最終決勝に進みました。 ファイナリストのリストには、12のロシアのチームと1つのロシアとスイスの合同チームが含まれています。



ファイナルに達したチームは、医療機器の開発において幅広い経験を持っています。



  • AngioScanチームは、6台のモバイル医療機器の開発経験があり、その導入は革新的な非侵襲的診断法の分野で医療にとって非常に重要でした。
  • グローバルセンサーネットワークチームのメンバーは、バイオセンサー開発の分野で多くの開発を行っています。また、マイクロチャネルマトリックステクノロジーを使用する生体液の分析も行っています。
  • RWMグループは、RWM-Allianceによって作成されました。RWM-Allianceは、ハイテクおよびハイテク分野での仕事で知られており、ロシア市場で数年間発展しています。 チームは、Biosing Instrumentsテクノロジーに基づいたデバイスを提供しています。
  • Keeperチームでは、ストレスの多い状態を監視するためのデバイスと、スイマーの身体状態をリモートで監視するためのデバイスというプロジェクトが正常に実装されました。
  • RUSSWIチームは、医療診断の分野で豊富な経験を持ち、医療、遠隔医療、および電子機器の分野で成功裏に完了したプロジェクトのリストを持っています（15以上のプロジェクト）。
  • FRUCTチームの主な活動は、モバイルデバイスのソフトウェア開発、クロスプラットフォーム開発、モノのインターネットとインテリジェントスペースに基づく将来のサービスの研究開発です。
  • Geliusチームはそれほど前に結成されませんでしたが、チームメンバーはすでに、アプリケーションプログラムと工業的に生産された多くの医療機器とシステムの開発において豊富な経験を持っています。
  • AstMedComチームは、医学の分野で革新的な研究を計画および組織化した豊富な経験を持っています。
  • SNEZHINSKチームは 、電気化学液体分析装置の分野およびセンサーデバイスの分野での開発で知られています。
  • DynAcSysチームのメンバーは、生体システムのメカニズム、認知神経シミュレーターの数学的およびソフトウェアモデリング、パターン認識システムおよび人工知能システムの作成と実装に関する長年の経験を持っています。
  • パーソナルサーモグラフチームは、熱診断の分野での研究、市場での診断デバイスの開発および発売において豊富な経験を持っています。
  • NMSISチームは、予防医学の分野における自動化された情報システムの開発、実装、および保守を専門としています。
  • I-Bolitチームのメンバーは、心臓血管系の機能を監視し、心臓ブロックの病態を特徴付けるために、EHF穿刺療法の迅速な診断方法と機器の開発の経験があります。




12月1日までに、チームはデバイスの既製のコンセプトを提出する必要があります。 セレクションの勝者は、12月12日にDigital October Centerでの最終イベント中に決定され、コンペティションの準決勝進出者全員が招待されます。

各チームはプロジェクトを最終聴衆と審査員に提示することができ、最高の解決策を提案した勝者には最高で900万ルーブルの賞品が授与されます。



チームに関する詳細情報は、 community.sk.ru / foundation / itc / md / p / teams.aspxにあります。
