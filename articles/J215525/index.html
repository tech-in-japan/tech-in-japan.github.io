PonoMusic-デジタル音楽エコシステム

2014年3月11日、KickstarterキャンペーンがPonoMusicチームと有名なミュージシャンNeil'a Young'a- Pono Musicによって開始されました。 11分のビデオには、プロジェクトの技術的な詳細がほとんど記載されておらず、スティング、ジャックホワイト、トムペティ、マムフォードアンドサンズ、レッドホットチリペッパーズ、フーファイターズなど、非常に有名な音楽アーティストの音楽を聴く印象があふれています。 キャンペーンはすでに必要な$ 80万を集めており、すでに1日も経たないうちに$ 1.1kkを超えています。







実際には、技術的な本質とプロジェクトの目的について少し。 彼には2つの重要なポイントがあります。これはPonoPlayerオーディオプレーヤーとWebサイトwww.PonoMusic.comです。 目標は、アクセシビリティと、それを楽しむためのツールの両方の点で、高品質のデジタルオーディオを新しいレベルに引き上げることです。



PonoPlayerは次の形式を再生できます。

  • CDロスレス品質の録音：1411 kbps（44.1 kHz / 16ビット）FLACファイル
  • 高解像度記録：2304 kbps（48 kHz / 24ビット）FLACファイル
  • 高解像度の記録：4608 kbps（96 kHz / 24ビット）FLACファイル
  • 超高解像度記録：9216 kbps（192 kHz / 24ビット）FLACファイル


パラメータ：

メモリー容量：64GBを追加する機能を備えた64GBの内部メモリー

重さ：128g

寸法：5x2x1インチ

バッテリー：8時間のリチウムイオン







残念ながら、内部のデジタル-アナログコンバーターの機能に関する情報は見つかりませんでした。デバイスが、よく知られているオーディオプレーヤーと同じ品質でおなじみの44.1k / 16 FLACを再生するかどうかは疑問のままです。



www.PonoMusic.comは、実際、非常に高品質のオーディオを購入できる場所になります。 これはクラウドだけでなく、ファイルをダウンロードできます。 1枚のアルバムの推定コストは15〜25ドルです。



このキャンペーンは多くの種類の財政支援（誓約）を提供し、一部（400ドル）で有名なミュージシャン（すでに部分的に分解された）によって署名されたPonoPlayer限定シリーズを入手できます。 PonoPlayerの通常バージョンは300ドルで入手できます。 約束日は2014年10月です。



無料翻訳PonoPlayerテクノロジーの特別なところは何ですか？ ：

PonoPlayerは、音質に対する妥協のないアプローチで開発されました。 Ayre（ www.ayre.com ）のエンジニアチームと協力して、世界クラスのオーディオテクノロジーをPonoPlayerに組み込みました。 Ayreチームは、PonoPlayerの開発への貢献について次のように説明しています。

  • PonoPlayerで使用されるデジタルフィルターには最小限のフェーズがあり、不自然な（デジタル）リードがありません。 生成されるすべてのサウンド（および音楽自体）には、メインサウンドの後に常に残響やエコーがあります。 自然界では、従来の線形位相デジタルフィルターが生成する音の前にエコーや残響があるような音はありません。 これが、デジタルサウンドが不自然でラフなサウンドであるという評判の理由の1つです。
  • すべてのスキームはゼロフィードバックスキームです。 フィードバック（フィードバック）はエラーが発生した後にのみ修正できるため、すべてのエラーを修正することはできません。 広い帯域幅と低出力インピーダンスを備えた特許取得済みの超線形回路を使用すると、人為的にフィードバックを行う必要がなくなります。
  • 使用されているDAC（Digital-to-Analog Converter）チップは、オーディオおよびエンジニアリングコミュニティによって、現在入手可能な最高のサウンドチップの1つとして認識されています。
  • ヘッドフォンに使用される発信バッファーは完全に独立しているため、個々のパラメーターと設定をすべて最適化して、完璧な音質を実現できます。 総出力インピーダンスは非常に低いため、ほとんどのヘッドフォンを使用する場合、PonoPlayerは完全にフラットな周波数応答と広い音量範囲を提供します。


オリジナル
PonoPlayerは、音質に対する「妥協のない」アプローチで設計されました。 私たちは、Ayre（www.ayre.com）のエンジニアリングチームと提携して、PonoPlayerに世界クラスのオーディオテクノロジーを取り入れました。 Ayreチームは、PonoPlayerデザインへの貢献について次のように説明しています。

  • PonoPlayerで使用されるデジタルフィルターの位相は最小限であり、不自然な（デジタルサウンドの）プリリンギングはありません。 作られたすべての音（音楽を含む）は、最初の音の後に常に反射やエコーがあります。 音の前にエコーや反射がある音は自然にはありません。これは、従来の線形位相デジタルフィルターが行うことです。 これが、デジタルサウンドが「不自然」で耳障りなサウンドであるという評判がある理由の1つです。
  • すべての回路はゼロフィードバックです。 フィードバックはエラーが発生した後にのみ修正できます。つまり、すべてのエラーを修正することはできません。 広い帯域幅と低い出力インピーダンスを備えた独自の超線形回路を使用することにより、不自然なサウンドフィードバックの必要がありません。
  • 使用されているDAC（Digital-to-Analog Converter）チップは、オーディオおよびエンジニアリングコミュニティで、現在入手可能な最高のサウンドを提供するDACチップの1つとして広く認識されています。
  • ヘッドフォンの駆動に使用される出力バッファーは完全に独立しているため、個々のパラメーターと回路値、および部品の品質はすべて、最高の音質に最適化されます。 出力インピーダンスは非常に低いため、PonoPlayerはほぼすべてのヘッドフォンを使用して、完全にフラットな周波数応答と広い音量範囲を実現します。
