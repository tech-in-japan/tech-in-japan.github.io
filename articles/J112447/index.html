VLC Media Player 1.1.6が光を放ちました

VLC 1.1.5の前バージョンのリリースから2か月後、5800万ダウンロード以上の後、優れたクロスプラットフォームの無料VLC 1.1.6マルチメディアプレーヤーのマイナーリリースが行われました 。



まず、セキュリティに関連する小さな革新と重要な修正（特に、 Real demultiplexerのバッファオーバーフロー ）を紹介します。 バージョン1.1.6の新機能：



  • コーデックおよびデマルチプレクサのセキュリティ更新
  • RTP / PCM 24bit、オーディオ/ L24のサポートを追加
  • WindowsおよびMacOS用のMPC SV7 / SV8のサポートを追加
  • Webm / VP8デコード速度の向上
  • MacOSのMidiサポートを追加
  • ほとんどの言語翻訳の重要な更新
  • Windows用のprojectM視覚化サポートが追加されました
  • KDEとPulseAudioの相互作用に関する改善
  • 字幕処理の修正と改善
  • 視覚化とプログラムインターフェイスの改善
  • コーデックの更新
  • その他多くの異なる修正


公式サイトを使用して、 Windows 、 MacOS 、 iOS 、およびその他のプラットフォームの最新バージョンをインストールできます 。



新鮮なソースコードも世間の注目を集めています。



また、Ubuntu MaverickのユーザーはPPAで満足することができます。



sudo add-apt-repository ppa:ferramroberto/vlc

sudo apt-get update

sudo apt-get install vlc mozilla-plugin-vlc







または、公式のユニバースリポジトリの更新を待って更新します。

sudo apt-get update

sudo apt-get install vlc vlc-plugin-pulse mozilla-plugin-vlc







PSロックされたDVDをUbuntuで再生するには、libdvdcss2をインストールする必要があります。 これを行うには、 この指示に従うか、リポジトリに「 medibuntu 」を追加してlibdvdcss2をインストールします。
