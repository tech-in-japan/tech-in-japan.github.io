チュリュモフ・ゲラシメンコ彗星で迷子にならない方法

2014/08/15 17:30 UTCに公開された元の記事



ロゼッタは、彗星の前で最初の最初の三角形軌道をほぼ完成させました。 （装置の操作は、彗星の重心の周りではなく、その照らされた側にあるポイントの周りで行われます。）毎日、さまざまな位置から核のビューを取得します。 見事なレリーフのディテールはどこにでもありますが、このクレイジーな形の世界ではナビゲートするのは本当に難しいので、同じオブジェクトを別の写真で見ると言うのは難しいです。 多くのオブザーバーが同じ問題を抱えていることがわかったため、Rosettaの新しい家をナビゲートするのに役立つ画像を収集しようと決めました。



表面のオブジェクトに名前がある場合、この試みははるかに簡単になりますが、もちろん、正式な名前とタイトルはありません。 Rosettaが到着したばかりで、マッピングには時間がかかります。 レリーフのさまざまな機能に名前を付けるのは魅力的ですが、多少ar慢であり、正式な名前が表示されるとまったく役に立たなくなります。 したがって、私が興味を持っているオブジェクトのいくつかについて説明するために説明を行い、将来的にはそれらに正式な名前を付けることができることを望みます。



最初に、彗星が宇宙でどのように回転するかを見てみましょう。 彗星は、狭い地峡によって接続された大小のローブで構成されています。 彗星の回転軸は、大部分が地峡に接続されている点に位置しています。 彗星の座標系はすでに指定されており、北極と南極は回転軸が核を離れる点にあります。 大きな分数は主子午線であり、小さな分数は180番目の子午線です。 このアニメーションで、グリッドが彗星の表面にどのように配置されているかを見てみましょう。





OSARISチームのESA / Rosetta / DLR / MPS MPS / UPD / LAM / IAA / SSO / INTA / UPM / DASP / IDA

チュリュモフ・ゲラシメンコ彗星の地理と日光



したがって、アニメーションの左側を見ると、彗星のカラー画像が表示されます。 緑色の斑点には注意を払いません。 赤と黄色の領域を見てください。 これらの色は、彗星のさまざまな領域が太陽によって照らされる頻度を示します。 赤い領域はほとんど常に点灯しています。 それぞれのローブに1つずつ、永遠の光の2つの極があることがわかります。 一部の部分が常に照らされている場合、つまり、常に日陰にある領域。 北緯30度以上の彗星のほとんどは常に照らされています。 したがって、南緯30度以下の彗星の部分にはあまり見えません。



アニメーションから、彗星の写真を撮って、光が上から来るように配置すると、北も上からになることが明らかになります。 これが機能しないのは、Rosettaが北極自体を削除するときだけです。



もちろん、NavCamの最新の写真を見ると、それらのほとんどがポールを見ていることに気付きました。 これは、最初のラウンドでNavCamが撮影したすべての写真です。 一番上の行はポールショットであるため、ほんの少し（経度180度）が一番上になるように配置しました。 下の列はわずかな角度で作られているので、北がほぼ上になるように配置しました。 （右上の写真を配置する行がわからない。）





ESA /ロゼッタ/ NavCam /エミリーラクダワラ

ロゼッタの最初の軌道からのチュリュモフ-ゲラシメンコ彗星のNavCam画像



今、私たちは少し指向し、OSIRISカメラから昨日の画像を撮ることができます。 これは実際には、左右のステレオペアの画像のペアです。 これは、OSIRISカメラレビュー全体をカバーする彗星の最初の画像です。 OSIRISには現在、地球外で最大の検出器があります。これは4メガピクセルのカメラで、各画像のサイズは2048ピクセルです。 赤青メガネをかけて3Dビューを楽しむか、左右の画像を別々に見るだけです。

3Dでのチュリュモフ・ゲラシメンコ、2014年8月7日



私たちは何を見ていますか？ 大きなシェアを小さなシェアに向かって見ます。 北極はわずかに左に傾いています。



まったく異なる3種類の地形が見えます。 1つ目は、非常に滑らかな表面です。 私の意見では、マントルが何かの下に隠れているように見えます。 ある場所では、衝突クレーターに似たものに気づきました。 他の衝突クレーターがあったと思います。そのクレーターの端は塗られているか破壊されています。







いくつかの場所には玉石畑があります。 地峡には特に注目すべき分野が1つあります。 私はすでに説明だけを行うと言いましたが、この石畳のチェーンがネックレスのように見えるという考えを取り除くことは決してないようです。







NavCamの一部の画像では、奇妙な垂直溝のグループも確認できます。 一部の場所では、地峡の滑らかな表面の上にいるかのように見えます。 他の場所では、彼らは彼によってわずかに覆われているように見え、それから突き出ている。







しかし、本当に注目を集める表面は、地峡に向かって見える小さな部分の壁の場所です。 ストライプまたはリボンです。 階層化されているとは言いません。 これは、ひだ、小さな尾根、くぼみがある顕著な表面です。 複数の交差する表面が可能です。 急な斜面で横方向のを見ることができます。これは、より滑らかな表面からわずかな割合で光の極まで始まります。 この表面は、黒い空間を背景に印象的なシルエットを作成するかなりまとまりのある素材で構成されています。







以下は、NavCamのより一般的なビューです。







他の場所では、彗星には高い丸い縁のある物体がたくさんあります。 驚くほど丸い。 衝突クレーターでない限り、このような高い丸いエッジがどのように作られるか想像するのは難しいです。







しかし、これらが衝突クレーターである場合、それらが形成されて以来、それらに多くのことが起こりました。 尾根の上部にある興味深いオブジェクトを参照してください。







日曜日から、ロゼッタはわずかに低い軌道、約80 kmに入り、残りの8月はこの高度から観測を行います。 連絡を取り合いましょう！
