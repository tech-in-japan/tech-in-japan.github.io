PHP内部



  • [RFC]関数呼び出しで末尾のコンマを許可する -関数引数のリストの最後にコンマを使用できるようにすることが提案されています。 



    var_dump( $whatIsInThere, $probablyABugInThisOne, $oneMoreToCheck, );
  • PHP 7.2はまだリリースされていません。次のバージョンの作業はすでに本格的です。 たとえば、7.3 では、 スパースの条件付き定数伝播の最適化が実装されています。 これにより、インタプリタはこの機能を簡素化できます。 



     Class C { public $i; } function fn(int $x) { $c = new C; $c->i = 1; if ($x) { $a = [1, 2, 3]; } else { $a = [3, 2, 1]; } return $a[$c->i]; $c->i++; return $x; }


    これに： 



     function fn(int $x) { return 2; }


    私たちはJITに関するニュースも待っています。
