Ubuntu 16.04にBigBlueButtonをインストールする

良い一日、今日、私は最後に私の最初の投稿を書くことにしました。







BigBlueButton、BBBについてはあまり説明しませんが、BBBは次のとおりです。







  • 無料（GNU Lesser General Public License）。
  • ビデオオーディオ会議用。
  • プレゼンテーション用。
  • ウェビナー用。


始めましょう。







背景のビット。







私たちの場合、Ubuntu 16.04 x64を使用していますが、同じことをお勧めします）







完全なBBB機能には、次のものが必要です。







  1. 静的外部IP。
  2. バインドされたドメイン名。
  3. SSL証明書。
  4. 転送ポート。


パート1



クイックインストールスクリプト



推定インストール時間は15〜20分です。

インストールは簡単で邪魔になりません。







この場合、外部IPを使用してインストールします。 







wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200


外部ドメイン名bbb.example.comを使用する 







 wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com


外部ドメイン名bbb.example.comを使用し、SSL証明書を取得する

電子メールinfo@example.comを使用して暗号化しましょう 







 wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com -e info@example.com


SSL +最新のHTML5クライアント 







  wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com -e info@example.com -t


SSL + GreenLight 







 wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com -e info@example.com -g


上記のすべて 







 wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com -e info@example.com -t -g


インストールが完了するまで待機します。







それから私は、読者を苦しめず、最初の部分と一緒に2番目の部分を書かないことを考えていましたが、手動インストールで別の部分があることを思い出しました。







パート2



Dockerのクイックインストール



BBBウェブサイトには、Dockerバージョンでは取得できないもののリストがあります







  • NginxにはSSL証明書がないため、FireRTを使用してWebRTCオーディオ/ビデオを使用する必要があります
  • プロセスは（systemdの代わりに）Supervisordを使用して開始されます
  • HTML5クライアントはビデオをサポートしていますが、WebRTCを使用したデスクトップ共有はサポートしていません


つまり、彼を本番に送ることはできません。







それでも説明しますが、突然誰かがテストに興味を持つようになります。







ちなみに、SSLを使用してリバースプロキシの背後に隠した場合は、すべてが開始されるはずです（自分でチェックしていません）。 







 docker run -p 80:80/tcp -p 443:443/tcp -p 1935:1935 -p 5066:5066 -p 3478:3478 -p 3478:3478/udp bigbluebutton/bigbluebutton -h LOCAL_IP_ADDRESS
