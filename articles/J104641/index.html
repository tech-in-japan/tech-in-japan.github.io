Intelソフトウェアネットワークサイトの概要



Intel Software Network （ISN）サイトのロシア語セクションは、ロシアのソフトウェア開発者の注意を不当に奪われています。 そして、私はこの情報資源とその内容について叫びたいと思っていました！ 非常に多くの有用な情報が無駄になっています！





私はインテルの従業員ではありませんが、ISNコミュニティの生活に積極的に関与しています。 コミュニティは、Intelプログラマと私たち人間の両方を結び付けます。 簡単な登録に合格すれば、誰でもISNのメンバーになれます。 ただし、ブログ、記事、その他の資料に精通するには、登録は不要です。 彼らは開いていて、誰でも彼らに会うことができます。



「誰もが記事を読むことができ、私にとっても成果です！」 もちろん、成果ではありませんが、プログラマーとしては楽しいことです。 AMD Webサイトの記事を読んでみてください。 http://developer.amd.com/documentation/articles/Pages/default.aspxページを開きます 。 たとえば、「 アーカイブされた64ビットコンピューティングの記事を見る 」セクションに進んでください 。 「FigWam」が呼び出されますが、パスワードの1つが適切ではありません。 「 アーカイブされた並列プログラミングを見る 」なども同じです。



メインページhttp://software.intel.com/ru-ru/からサイトナビゲーションを開始することをお勧めします。 初めてサイトにアクセスする場合は、メインページから利用できるセクション「ブログ」、「フォーラム」、「ツール」を掘り下げることをお勧めします。 定期的にサイトにアクセスすると、メインページで新しい記事、ブログエントリ、ニュースのリストをすぐに知ることができます。 一般に、サイトのロシア語版の開発者は、メインページへのすべてのリンクを収集してメインページを非常にうまく実装しましたが、過剰にロードしませんでした。



ISNに含まれるセクション：



1.プログラミング、コードの最適化、トラブルシューティング、さまざまなツールの使用に関するロシアおよび外国の著者による記事。 したがって、外国の著者の記事はロシア語に翻訳されます。 他の多くの資料と同様に、記事については、コメントを残したり、著者と議論することもできます。 例の記事タイトルは次のとおりです。

  • Intel Parallel Composer-並列プログラムをデバッグするための拡張機能
  • Cilk ++でN Queens問題を解決する
  • 明らかに浮動小数点数について
  • Intel Parallel Inspectorを使用してマルチスレッドエラーを見つける秘Find
  • 潜在的なキャッシュ最適化を見つけるためのツール


技術的な性質だけでなく、インテルとISNコミュニティのメンバーの生活に関する記事もあります。

  • ソフトウェア開発者向けのISNコンテスト
  • Intelソフトウェアネットワークサポートへメール

2.ブログ。 私の知る限り、ブログはロシアのISNメンバーによってのみ管理されており、エントリは他の言語から翻訳されていません。 ブログのセクションはこれからのみ恩恵を受けると確信しています。 私の意見を説明します。 ブログにはしばしばユーモアと専門用語があります。 翻訳する際、これらのコンポーネントを保存することは非常に難しく、興味深いライブテキストは読者にとって理解できない奇妙なメモに変わります。 当然、ロシア人作家のユーモアはロシア人にとって完全に理解可能であり、不明瞭なものに会うことを恐れずにブログを安全に読むことができます。 いくつかのサンプルエントリを次に示します。

  • 私は若く、ソ連90年代に再び住んでいますか？ （祝日後のせん妄）
  • 1000 cuのリマインダー
  • VS2008記録済みコンポーネントのインストール
  • ソフトウェアの性別の変更
  • Rise of FlightまたはGoodbye Piracy！


3.フォーラム。 ISNフォーラムでは、1つの重要な詳細に気付くことができます。 彼らは（今のところ）、たとえばRSDN.ruほど人気が​​ありませんが、Intelの従業員はそこにいます。 そして、これはいくつかの問題を解決する上で大きな助けとなります。 フォーラムの例を次に示します。

  • モバイルシステム用のソフトウェア
  • グラフィックス

  • アカデミックコミュニティフォーラム


4. セルゲイビジャノフはISNでコラムをリードしています。 あなたはすでに彼の名前を知っているかもしれません。 かつて、彼はComputerra-Online（Computerra.Ru）の編集長を務めていました。 現在、彼は3DNews.Ruの主要なアナリストです 。 彼のコラムはISNの表紙に掲載されており、かなりのユーモアで定期的に新しいメモを見ることができます。



もちろん、ISN Webサイトで見つけたすべてがリストされているわけではありません。 ここでは、インテルが開催するさまざまなコンテストに関する情報が定期的に公開され、さまざまなツールの説明がロシア語で提供され、プレゼンテーションがレイアウトされ、ウェビナーが開催されます。



ISNのソフトウェア開発者のロシアコミュニティの魂はDmitry Oganezovです。 ISNのロシア部分のサポートと開発を管理するのは彼です。 さらに、彼は抽象的なリーダーではなく、コミュニケーションのために簡単にアクセスできる生きている人です。 これは個人的な経験に基づいて宣言します。 彼と一緒に、あなたは簡単に問題を議論し、資料の出版で助けを求め、それを得ることができます。 そのため、ISNに参加すると、サイトのソウルレスメカニズムに遭遇するのではなく、常に人々から真のサポートを得ることができます。



はい、最も重要なこと。 ISNは、閉じられた、いわゆる読み取り専用のコミュニティではなく、開かれたコミュニティです。 ここでブログを書いたり、記事を公開したり、インテルの開発者と問題を話し合ったり、他の企業や同僚の専門家とだけ議論したりできます。



ISNには独自のカルマの類似物があります。 ここでは、これを「ステータスポイント」と呼びます。 ポイントの数に応じて、参加者はいくつのランクに分割され、各ランクは特定の色の帯に対応します。 緑、茶色、青、黒のベルトがあります。 最高レベルに達した参加者はブラックベルトの所有者となり、名誉の殿堂に置かれ、Intelの高性能ラップトップが授与されます。



繰り返しますが、オブザーバーとしてだけでなく、参加者および著者としてもISN Webサイトに招待します。 ISNは皆を待っています！
