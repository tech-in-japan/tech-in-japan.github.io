ユニバーサルアプリケーションにAdDuplexを使用する





今日、アプリケーションに広告バナーを配置することは、ソリューションを収益化する一般的な方法です。 Adduplex -Windows / Windows Phoneアプリケーションでのコンテキスト広告の実装に便利なサービスの1つです。



SDKと使いやすいAPIの存在により、必要なツールセットをすばやくインストールし、数ステップでアプリケーションの広告要素を取得できます。



これが実際にどのように起こるか見てみましょう。



登録



AdDuplexアカウントをまだお持ちでない場合は、 http： //www.adduplex.comで登録する必要があります 。 登録には1分もかかりません。



次に、広告バナーを配置するアプリケーションをAdDuplexに登録します。 「新しいアプリ」をクリックします。







適切なフィールドに入力します。 Adduplexでは、ストアでまだ公開されていないアプリケーションURLを指定しないようにすることができます。







これで、アプリケーションページにアイテムを追加するときに指定する必要があるAppIDを取得しました。







4つのステップで、広告バナーを作成します（テキストまたは画像付きテキストの形式）：







保存し、確認後、ブラウザで通知を受け取ります：







ストアにアプリケーションへのリンクを追加して、アプリケーションの登録を完了します。







アプリケーションに広告バナーを追加します。



AdDuplexをアプリに追加します。



Visual Studio Extensions Galleryで、XAML（C＃/ VB.NETおよびC ++）のユニバーサルアプリケーション（Windows 8.1 / Windows Phone 8.1）用のAdDuplex Universal SDKを見つけてダウンロードします。



このSDKをインストールし、「ソリューションエクスプローラー」ウィンドウでプロジェクトへのリンクを追加します

「参照...」->「新しい参照の追加...」



次のウィンドウ「Windows 8.1」/「Windows Phone 8.1」->「拡張機能」で「Windows 8.1用AdDuplex SDK（XAMLエディション）」を選択し、Windows Phone 8.1用アプリケーション「Windows Phone 8.1用AdDuplex SDK（XAMLエディション）」を選択しますそれに応じて。



必要なすべてのコンポーネントを接続した後、アプリケーションの操作に進みます。



プロジェクトでMainPage.xamlファイルを開き、AdDuplex.Controlsを接続します。



Windows 8.1 

xmlns:ad="using:AdDuplex.Universal.Controls.Win.XAML"


Windows Phone 8.1 

 xmlns:ad="using:AdDuplex.Universal.Controls.WinPhone.XAML"


ページにAdControl要素を追加します。



Windows 8.1 

 <ad:AdControl x:Name="adDuplexAd" AppId="YOUR_AppId_HERE" Size="500x130" />


Windows 8.1アプリケーションのバナーサイズ：160x600、250x125、250x250、292x60、300x250、500x130、および728x90。



Windows Phone 8.1 

 <ad:AdControl x:Name="adDuplexAd" AppId="YOUR_AppId_HERE" />






このSDKは、スタンドアロンのWindows 8.1およびWindows Phone 8.1（XAML）アプリケーションでも使用できます。



ロシア語の詳細な手順は、次のリンクから入手できます。



おわりに



Adduplexは、コンテキスト広告の配置に価値のあるツールとしての地位を示しており、実装に特別な困難はありません。 数分以内にアプリケーションで広告バナーを取得できます。



AdDuplex開発者は、HTML / JavaScript、Windows Phone 8.0などを使用して、モバイルアプリケーションの詳細なドキュメントを作成しました。 また、UnityでAdDuplexを使用する例も含まれています。



特に、AdDuplex SDKが統合されたアプリケーションでは、 競争に参加することにより、AdDuplexネットワークで500,000インプレッションを受け取ることが可能になりました。
