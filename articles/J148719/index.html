ZeroNights 2012でCFPがオープン





ZeroNightsは、情報セキュリティの技術的側面に特化した国際会議です。 会議の主な目標は、新しい攻撃方法、脅威に関する情報を広め、それらに対して防御すること、さらに、セキュリティ実務者間のコミュニケーションのためのプラットフォームを作成することです。

場所と時間：ロシア、モスクワ、2012年11月19〜20日。



あなたの経験、知識、スキルを共有したい場合、これは簡単に行えます。あなたのトピックの説明と希望の形式を送ってください。 プログラム委員会（DCG＃7812）がそれをレビューし、決定を下します。 CFPは10/10/12まで続きます。 同時に、公開と選択はいくつかのラウンドで行われるため、CFP期間全体を通して、受け入れられたレポートに関する情報を公開します。



レポートの主題はイベントの分野に関連するものであれば何でもかまいませんが、今年は特定の好みがあります。



1.応用ハッキング


ハッカー攻撃を実際に注目させるに値するすべてのもの。 製品の脆弱性を見つけることと、完全に攻撃を実行するために他の脆弱性を適用し、追加の要因を考慮することはまったく別です。 このトピックは、ポップなAPTについてではなく、すべてにもかかわらず機能し、非常に顕著な損傷を引き起こす実際の攻撃方法についてです。



○ビジネスシステムのハッキング：ERP / RBS / SCADA

○直接利益攻撃プロセス

○企業部門への攻撃-実際のケース、隠蔽方法など

○端末およびATMへの攻撃（ATM）

○組み込みデバイス保護の攻撃と弱点



2.モバイルセキュリティ


モバイルの世界における防御メカニズムの攻撃と弱点：



○GSM / LTE / 3G / CDMAなどのプロトコルに対する攻撃

○OSの脆弱性（Android / iOS / Windows Phone）

○ファームウェアの脆弱性

○デバイスへの攻撃



3.「基礎」研究


私たちは引き続き技術的な会議を開催しているため、攻撃手法の分野における新しい実用的な研究は非常に興味深いものです。



○OSセキュリティメカニズムのバイパス

○脆弱性の悪用

○ファジングテクニック

○ウイルス対策エンジン

○リバースエンジニアリング

○Webセキュリティ

この会議にはいくつかのプレゼンテーション形式があり、それぞれに独自の条件があります。



＃1 40分レポート


この形式では、イベントの2つの部屋の1つで、プレゼンテーションに40分、質問に5分が話者に与えられます。 スピーカーは交通費（モスクワへの道とエコノミークラスに戻る道）を補償され、ホテルの部屋は会議会場の隣に3泊予約されます。 旅行の報酬は1000ドルを超えてはなりません。 追加の条件はすべて個別に説明します。



＃2 FastTrackレポート


この形式では、講演者はプレゼンテーションのために15分、別の部屋で質問のために5分を与えられます。 スピーカーは旅費の補償を受けておらず、宿泊施設は提供されていません。 アプリケーションはfasttrack@zeronights.orgに送信する必要があります。



＃3 マスタークラス


スピーカーは、トレーニング資料、プレゼンテーション、その他の資料を準備します。 別の部屋でマスタークラスを実施するために、少なくとも2時間割り当てられています。 スピーカーは交通費（モスクワへの道とエコノミークラスに戻る道）を補償され、ホテルの部屋は会議会場の隣に3泊予約されます。 旅行の報酬は1000ドルを超えてはなりません。 追加の条件はすべて個別に説明します。



アプリケーションはcfp@zeronights.ruに送信する必要があります。



スピーカーのリクエストには次のものが含まれている必要があります。



●姓と名。

●メールアドレス

●簡単な伝記：

○職場;

○位置;

●スピーカーの居住地。

●レポートのタイトル。

●レポートの短い説明（500文字以内、サイトの説明）。

●完全なレポートの説明（技術的な詳細）。

●0日間の脆弱性の説明はありますか？

●レポートのステータス（以前に表示/公開されたかどうか、もしそうであれば、どこで、いつ）。

●ツールは会議で発表されますか。

●パーソナライズされたスピーキング要件。



特定の製品の販売を目的としたマーケティングレポートや非技術的なレポートは受け付けていません。



レポートはロシア語または英語で提出する必要があります。



スピーカーは、ロシアの首都での良い気分、快適な会社、多くの忘れられない経験を保証されています。
