openSUSE 11.2リリース





新しいopenSUSEは、KDEとGnomeの両方のユーザーと、より厳しい環境を好むユーザーの両方に多くの快適な変更をもたらしました。



新しいバージョンであなたを待っているものの詳細は、カットの下で見つけることができます





保守的なGNomeユーザーは、緑色の新しいテーマ、新しいGnome、およびNetworkManagerの更新されたフロントエンドを楽しむことができます。これらでは、モバイル接続でのシステムの更新を禁止できます。



KDEユーザーは新しいKDE 4.3を喜んで見るでしょう。 新しいopenSUSE KDEshnikovは、テーマ、ファイルダイアログ、通知を含むFirefoxのKDEへの統合、およびOpenOffice.orgとKDE4の統合を歓迎します。



LiveCDにも変更が加えられました：gnomeを搭載したディスク上で、Gimpやその他の多くのプログラムがデフォルトで含まれている場所、および追加のローカリゼーション（ロシア語など）。 また、多くの要望により、pptpパッケージとNetworkManagerの対応するプラグインがLiveCDに含まれるようになりました。



さらに、このリリース以降、すべてのLiveCDはハイブリッドであるため、CDにカットできるだけでなく、コマンドを使用してUSBフラッシュドライブに書き込むこともできます。



dd if=openSUSE-LiveCD.iso of=/dev/sdf bs=4M





ここで、 / dev / sdfはフラッシュドライブです。



管理者はzypperの変更に満足します。zypperにはシステムを更新するためのさまざまな戦略があり、デフォルトの戦略はすべてのパッケージをダウンロードしてからインストールすることです。



また、管理者にとっては、インストール後にデフォルトでsshdがデフォルトで起動しないことを思い出してください。YaST>システム>システムサービス（ランレベル）で独自に起動できるようにする必要があります

Sambaは、以前のリリースで使用されていたsmbpasswdではなく、passdbとしてデフォルトでtdbsamを使用するようになりました。 データベースの変換と新しい形式への切り替えの詳細な手順は、リリースノートに記載されています。



MySQLソケットは/ var / run / mysqlに移動しました。



SaX2から削除されたキーボードおよびマウス構成のダイアログは、DEの機能を使用して構成したり、hal設定を編集したりします



パッケージ管理システムも最適化され、rpmがバージョン4.7に更新され、すべてのローカライズパッケージとカーネルソースにnoarchアーキテクチャが追加され、DVDとリポジトリのスペースが節約されました。



DVDから11.2をインストールするユーザーは、通常の-defaultではなく、カーネルデスクトップカーネルがインストールされていることに気付くでしょう。 このコアは、デスクトップ使用の応答性を強化するために特に最適化されています。 このカーネルでは、サイズによる最適化がオフになり、FULL_PREEMTIONがオンになり、パラメーターHZ = 1000が設定されます。 LiveCD -desktopでopenSUSEをインストールしたユーザーの場合、カーネルはリポジトリで利用可能です。



配布のハイライトの1つ-YaSTにも変更が加えられました。 最初に、YaSTはマスコットを見つけました-Yasti aardvark、2番目に、YaSTのQtバージョンはQt4で完全に記述され、KDE4環境に完全に適合します。



ディスクレイアウト用の大幅に改善されたYaSTモジュール。インストール中にディスク暗号化を設定できるだけでなく、btrfsなどの実験的なファイルシステムを選択できるようになりました。



最も便利な方法を選択してダウンロードする

そして、忘れないでください-たくさん楽しんでください！
