強力な装甲車Dell Latitude XT2 XFR





デルは「装甲」タブレットLatitude XT2 XFRを公式に発表しました。 極端なフィールド条件で動作するように設計されたこのマシンは、そのクラスで最も軽いものの1つで、わずか2.45 kgです。



ラップトップの機能は、12.1インチの対角線と1280x800ピクセルの解像度を持つ180度のマルチタッチディスプレイです。 衝撃および防塵と耐水性のモデルは、MIL-STD-810GおよびIP-54規格の要件を満たしています。



その他の特徴：



OS： Windows Vista BusinessまたはWindows 7 Professional / Ultimate

プロセッサー： Intel Core Duo ULV SU9600-1.6 GHz;

グラフィック： Intel GMA 4500MHD;

RAM：最大5 GB、DDR3-1066 MHz;

ウィンチェスター： SATA 160 GBまたはSSD 64/128 GB。

ワイヤレスモジュール： Wi-Fi 802.11a / g / n、Bluetoothおよび3G;

ポート： 2つのUSB 2.0、eSATA / USB 2.0の組み合わせ、RJ-45、VGAおよびFireWire、ヘッドフォン、マイク。



このラップトップには、3メガピクセルのWebカメラ、指紋スキャナー、DVDライター、ギガビットイーサネットネットワークカード、マルチカードリーダーも搭載されています。



この経済性はすべて31 x 424 cmのケースに収められており、4、6、または9セルのバッテリーを搭載でき、すでに3,600ドルの基本価格で販売されています。
