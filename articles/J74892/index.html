無料のウイルス対策Panda Cloudのリリースは無料で利用できます

2009年4月からパブリックベータとして提供されているウイルス対策ソフトウェアは、クラウドベースのアプローチにより、通常の無料のウイルス対策ソフトウェア製品よりも多くの宣伝を受けています。 Cloud Panda Antivirusが動作するすべてのコンピューターシステムからの情報は、他のすべてのユーザーと自動的に共有されます。







また、バージョン1.0では、ベータバージョンおよび通常のウイルス対策プログラムに追加の改善が加えられています。



Panda Cloud Antivirus 1.0は、PCのパフォーマンスへの影響を最小限に抑えながら、ローカルにインストールされたシグネチャ製品よりも効率的にマルウェアを処理およびブロックする保護モデルの100％クラウドベースアーキテクチャにさらなる改善を導入します。 バージョン1.0は、ベータ版よりもスループットを向上させ、効率に影響を与える方法をすでに誇っています。 従来のウイルス対策ソリューションは、コンピューターリソース（メモリ、ディスク領域など）の約15％を消費しますが、Panda Cloud Antivirus 1.0は3％しか消費しません。



Panda Cloud Antivirusを試してみたいユーザーは、ほとんどのMicrosoft Windowsシステムで利用可能なCloud Antivirus Webサイトから無料でダウンロードできます 。 Panda Cloud Antivirusは、 PC Worldという最高の無料アンチウイルスソフトウェアとして認識されています。
