Web 2.0サイトの統計は常に誇張されています

ごく最近、アメリカの本格的な分析会社の専門家は、数年後にはMySpaceソーシャルネットワークのコストが100〜200億ドルに達すると予測しました。 なぜ投資家はそのような見積もりを信じているのですか？ おそらく、このサイトの視聴者が誇張されているからでしょう。



たとえば、8月上旬に、MySpaceで重要なイベントが発生しました。1 億番目のアカウントが登録されました。 しかし、このニュースに関するコメントでは、神話上の「1億人のユーザー」に関する言葉がいたるところに登場しました。 しかし、これはばかげています。 これらのことについて少し知っている人なら誰でも、アカウントの数とユーザーの数が同じであることは明らかです。 1つ目は他の理由よりもはるかに多くの理由があります。多くの登録ユーザーはずっと前にサイトにアクセスしませんが、他のユーザーは別の名前で登録することを決めました。



MySpaceは、かなり高い離職率を誇るソーシャルネットワークです。 したがって、ここでのアクティブユーザーの数は1億人よりはるかに少ないです。 行の特定のID番号の選択を調べると、 おおよその調査でも、識別子の17％がシステムから削除され、アカウントの31％が作成されてからログインしたことがなく、別の18％のユーザーが比較的長期間サイトにアクセスしたことがわかります。 先週のオーディエンスのみを取得した場合、アカウントの総数のわずか34％です。 月次オーディエンスを考慮すると、この割合は43％に増加します。 言い換えると、1億のアカウントのうちMySpaceの場合、アクティブなのは3400万から4300万だけです。 そのため、マスコミのユーザー数の数値は2.5〜3倍に誇張されています。



統計による同様の誇張は、アマチュアYouTubeビデオを含むWeb 2.0の2番目の「現象」の特徴でもあります。 ここでのみ、ビデオのインプレッションの統計がまとめられています。 非常に頻繁に、著者自身がこれを行って、ビデオを最も人気のあるもののリストに上げ、それによってその本当の人気を高めます。



Web 2.0サイトの所有者は、おそらく物事の本当の状態を知っていますが、当然のことながら、膨らんだ数字の使用を好みます。 数十億ドルになると、出席者の数は魔法のように重要になります。



Skypeの所有者がビジネスを販売する準備をするとき、彼らはサイトからのプログラムのダウンロード数について、そして登録ユーザーの数については常に話しましたが、アクティブユーザーの数については話しませんでした。 結果はわかっています。 ご存知のように、Skypeは23億ドルで販売されました。
