AWS：RDSが無料利用枠パッケージに表示されます

こんにちは



10月1日に、t1.microのようなRDSインスタンスがAmazon Web Servicesの無料利用枠パッケージに登場したという事実を多くの人が見落としていたかもしれません。



MySQL、Oracle BYOL、またはSQL Server Express Editionを月に750時間無料で実行できるようになりました。 RDS無料利用枠には以下も含まれます。

  1. データベース用に20 GBのストレージ
  2. 1000万回のI / O操作
  3. データベースキャストおよびバックアップ用の20 GBストレージ




ニュースは素晴らしいです！ 特に、AWSにタッチしてライブを行いたいが、ブラックボックスの料金を支払いたくない場合に。



AWS無料利用枠の条件とサービスについてお知らせします。 新規登録ユーザーは、年間を通じて無料で（翻訳なしで）以下のAWSサービスを使用できます。

AWS無料利用枠
Elastic Compute Cloud（EC2）


  • 750時間のAmazon EC2 Linux†マイクロインスタンスの使用（613 MBのメモリと32ビットおよび64ビットのプラットフォームサポート）-毎月継続的に実行するのに十分な時間*
  • 750時間のAmazon EC2 Microsoft Windows Server‡マイクロインスタンスの使用（613 MBのメモリと32ビットおよび64ビットのプラットフォームサポート）-毎月継続的に実行するのに十分な時間*
  • 750時間のElastic Load Balancerと15 GBのデータ処理*
  • 30 GBのAmazon Elastic Block Storage 、200万のI / Oおよび1 GBのスナップショットストレージ


シンプルストレージサービス（S3）


  • 5 GBのAmazon S3標準ストレージ、20,000のGetリクエスト、2,000のPutリクエスト*
  • ダイナモドブ
  • Amazon DynamoDBのストレージ100 MB、書き込み容量5ユニット、読み取り容量10ユニット。


リレーショナルデータベースサービス（RDS）


  • MySQL、Oracle BYOL、またはSQL Server（SQL Server Express Editionを実行）を実行するための750時間のAmazon RDSシングルAZマイクロDBインスタンス‡‡-DBインスタンスを毎月継続的に実行するのに十分な時間*
  • 20 GBのデータベースストレージ
  • 1000万I / O
  • 自動化されたデータベースバックアップとユーザーが開始したDBスナップショット用の20 GBのバックアップストレージ


Simpledb


  • 25 Amazon SimpleDBマシンアワーと1 GBのストレージ


シンプルワークフロー（SWF）


  • 1,000件のAmazon SWFワークフローの実行を無料で開始できます。 合計10,000個のアクティビティタスク、信号、タイマー、マーカー、および30,000ワークフロー日も無料で使用できます。


シンプルキューサービス（SQS）およびシンプル通知サービス（SNS）


  • Amazon Simple Queue Serviceの 100,000リクエスト
  • Amazon Simple Notification Serviceの 100,000リクエスト、100,000 HTTP通知、1,000メール通知


クラウドウォッチ


  • 10個のAmazon Cloudwatchメトリックス、10個のアラーム、1,000,000個のAPIリクエスト


データ転送


  • すべてのAWSサービス全体で15 GBの帯域幅を集約
  • これらのサービスに加えて、 AWS Management Consoleを無料で利用して、AWSでアプリケーションを構築および管理することができます。
