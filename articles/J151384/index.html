新しいアップル製品：乾燥した結果

Apple は 、Steve Jobsが関与しなくなった新製品を発表しました。 新しいデバイスの特性を見て、他社のソリューションとどれだけ競合できるかを考えてみましょう。 だから、私たちは最終的に何を持っていますか？ 主なものから始めましょう-iPhone 5。



iPhone 5












iPhone 5の機能：



  • ディスプレイ：ワイドスクリーン、4インチ、1136 x 640（326 PPI Retina）、コントラスト800：1;
  • プロセッサー：Apple A6;
  • ネットワーク：HSPA +、DC-HSDPAおよびLTE。
  • 通信：Bluetooth 4.0、802.11 a / b / g / n;
  • ナビゲーション：A-GPS、GLONASS;
  • メインカメラ：8 Mpx iSight、LEDフラッシュ、1080p @ 30ビデオ;
  • フロントカメラ：1.2 Mpx FaceTime、720p @ 30ビデオ、ジオターゲティング。
  • メモリ：16/32/64 GB;
  • センサー：傾き、加速度センサー、加速度計、バックライト、デジタルコンパス;
  • ヘッドセット：リモコン、マイク、ケース付きのEarPods。
  • バッテリー：225時間の待機、40時間の音楽、10時間のビデオ、8時間の会話またはインターネットの使用。
  • コネクタ：Lightning（8ピン、80％削減）、3.5 mm。 ジャック
  • 寸法：123.8 x 58.6 x 7.6 mm。;
  • 重量：112グラム。
  • 価格：（まだ知られていない）、12月15日以降のロシア。




iPhone 5はアルミニウムとガラスでできています。 カメラはパノラマ機能を受け取りました。 画像安定化の改善。 顔認識機能を追加しました。 FaceTimeでフロントエンドを使用することは、電話で可能になりました。 体に3つのマイクを持ち込みました。 iPhoneを含むすべてのデバイスで、コネクタを30ピンから8ピンに変更しました。 多くの批評家は、iPhone 4Sと比較してiPhone 5の方がiPhone 4と比較して4Sよりもさらに改善が少なく、ジョブズのリズムに追随できないという新しいリーダーシップを非難していると述べています。 また、多くの人がiPhone 5とLumia 920を比較し、ノキアのデバイスは次回のアップグレードでより収益性の高い購入になるとの意見を表明しました。



重要な点：iPhone 5はnanoSIMを使用し、iPhone 4SはmicroSIMを使用するため、SIMカードを再利用する必要があります。



新しいiPod Touch








新しいiPod touchの機能：



  • ディスプレイ：ワイドスクリーン、4インチ、1136 x 640（326 PPI Retina）、コントラスト800：1;
  • プロセッサー：Apple A5;
  • 通信：Bluetooth 4.0、802.11 a / b / g / n;
  • ナビゲーション：Wi-Fiベースの場所。
  • メインカメラ：5 Mpx iSight、ダイオードフラッシュ、1080p @ 30ビデオ;
  • フロントカメラ：1.2 Mpx FaceTime、720p @ 30ビデオ、ジオターゲティング。
  • メモリ：32/64 GB;
  • センサー：傾斜センサー、加速度計。
  • ヘッドセット：リモコン、マイク、ケース付きのEarPods。
  • バッテリー：40時間の音楽、8時間のビデオ。
  • コネクタ：Lightning（8ピン、80％削減）、3.5 mm。 ジャック
  • 寸法：123.4 x 58.6 x 6.1 mm。;
  • 重量：88グラム。
  • 価格：300ドル、10月からロシアで。




iPod touchの背面には、ボタンホールをクリックするボタンがあります。 ループがアイレット、特別なリストストラップに取り付けられています。 一部のメディアは、このプレーヤーのデザインは主にティーンエイジャーにアピールし、そのようなソリューションは人々を「より深刻に」「怒らせる」と指摘しています。
