Raspberry Pi Zeroが5ドルで販売開始：店舗へのリンク





Raspberry Pi Foundationを作成する主な目的は、安価な小型コンピューターを作成することでした。 2012年には、A +モデルが20ドルでリリースされました。目標は達成されたようです。 しかし、これは制限ではないことが判明しました。 今日、Raspberry Pi Foundationは、わずか5ドルの価値があるミニマリストのRaspberry Pi Zeroモデルを発表しました。



1 GHzプロセッサを使用すると、ゼロはモデルBよりも40％高速です。



Raspberry Pi Zeroは、ウェールズ（英国）の工場で製造されています。



プロセッサー ：ARM11、1 GHzのBroadcomm BCM2835

メモリー ：512 MB RAM

ストレージ ：microSD

GPIO ：26/40穴

USBコネクタ ：USB On-the-Go（OTG）、Micro USB

ビデオ出力 ：ボード上の2つのピン（TVとしてマーク）を介したコンポジットビデオ、ミニHDMIコネクタ（1080p60）を介したHDMI
