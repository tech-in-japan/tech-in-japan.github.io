Internet Explorer 8のアクセラレータを書く

Internet Explorer 8のアクセラレータを書くのは、思ったよりずっと簡単です。 Remix'09の一部として開催されているIE8のアクセラレータコンテストについては、おそらく誰もが既に聞いたことがあるでしょう。 そこで、興味深く有用なアクセラレータを書くことにしました。 そして、それは2つでも判明しました（必要に応じてダウンロードしてください）：

a） Habrahabrで検索 （ 投票へのリンク ）

b） Habrametrでユーザーを表示する （ 投票へのリンク ）

これらのアクセラレータが気に入ったら、 IE8 アドオンコンテストWebサイトで投票することができます（もちろんそこに表示されている場合）。 まあ、あなたはいくつかの改善や何かを提案することもできます-私には個人的に;）

M. Chernomordikovのアクセラレータ作成に関する有用なスクリーンキャストは、リンクを参照してください 。

そのため、アクセラレータを実装することが判明したため、これは（各アクセラレータファイルに対して）2つのxmlファイルを書き込むことです。

そのため、アクセラレータの説明「Habrahabrでの検索」：

<? xml version ="1.0" encoding ="utf-8" ? >

< openServiceDescription

xmlns ="http://www.microsoft.com/schemas/openservicedescription/1.0" >

< homepageUrl > outcoldman.habrahabr.ru </ homepageUrl >

< display >

< name > habrahabr </ name >

< icon > habrahabr.ru/favicon.ico </ icon >

</ display >

< activity category ="Find" >

< activityAction context ="selection" >

< execute action ="http://habrahabr.ru/search/{selection}/" >

< parameter name ="q" value ="{selection}" />

</ execute >

</ activityAction >

</ activity >

</ openServiceDescription >



* This source code was highlighted with Source Code Highlighter .





homepageUrlとdisplayがこのアクセラレータの説明である場合、 アクティビティはアクセラレータの機能です。この場合、 選択 （強調表示）テキストを受け取り、アクセラレータメニューをクリックすると、 executeからアクション urlに移動します。 次のファイル、アクセラレータの説明「」

<? xml version ="1.0" encoding ="utf-8" ? >

< openServiceDescription

xmlns ="http://www.microsoft.com/schemas/openservicedescription/1.0" >

< homepageUrl > habrometr.ru/users/outcoldman </ homepageUrl >

< display >

< name > habrametr </ name >

< icon > habrometr.ru/favicon.ico </ icon >

</ display >

< activity category ="Share" >

< activityAction context ="selection" >

< preview action ="http://habrometr.ru/habrometr_88x120_{selection}.png" >

</ preview >

< execute action ="http://habrometr.ru/users/{selection}/" >

</ execute >

</ activityAction >

</ activity >

</ openServiceDescription >



* This source code was highlighted with Source Code Highlighter .
