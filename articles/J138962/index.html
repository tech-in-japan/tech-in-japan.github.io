USB接続のシンプルなオーディオDAC（サウンドカード）









結局のところ、外部USBサウンドカードの作成は簡単で安価です。 この記事では、それをどのように行ったかを説明します。



背景：



数年前、インターネット上のフォーラムの1つで、オーディオDACに関するトピックに出会いました。 オーディオカードをはんだ付けするというアイデアを非常に得ました（！）そして、さまざまなデザインの説明を読み始めました。 複雑なものは、それらを繰り返すことを私に忌避しました（コンピューターからI2Cスクエアバスを取得する場所やS / PDIFを取得する場所はわかりませんでした）回路と高価なコンポーネント（これが最も説得力のある議論でした）。 このテーマに関する資料はロシア語で、今ではほとんどありません...

数ヶ月後、PCM2702チップ上で、そして最も重要なことには、コンピューターへのUSB接続を備えたシンプルなデザインを見つけました。 私は超小型回路のSSOPケースを恐れていませんでしたが、価格を恐れていました-1個あたり500ルーブル以上。 また、そのような高価なチップを経験不足で台無しにすることを恐れていました（過熱、静電気、気にしない？）。 私は他の解決策を探し始めました。 そして、PCM2705でデザインに出会いました。 これもUSBコーデックですが、PCM2702と比べて仕様が低くなっています。

あるフォーラムのフリーマーケットでチップを見つけました。 私は自分と友人を1つずつ注文しました。 正確な価格は覚えていませんが、1ルーブルは150ルーブル以下です。



スキーム：



このスキームは、元のソースとほぼ1対1で繰り返されました。 そして、彼はそこにほとんどきれいなデータシートを持っています。









料金：



回路基板の彼自身のバージョンを作りました。 私はすでにLUTをマスターしました。











最初の実行：



はんだ付け（5ミリメートルの刺し傷でははんだ付けできないと思っていましたが、電子レンジを使用したアイデアは DI-HALTに感謝します）。









震えている手で、コンピューターに接続して... OSは新しいデバイスを見つけました。 ドライバーをインストールしました。 接続されたヘッドフォン-歌う！ さらに、ラップトップに組み込まれている音ほど悪くはありません。 そしてさらに良い！ 少なくとも私は低音の違いを聞いた。 HFには気づきませんでした。 しかし、私のヘッドフォンは最高の品質ではありません。

私も友人をはんだ付けし、それを接続しました...動作しません。 クォーツストラップのコンデンサを変更しました-それは助けにはなりませんでした、クォーツ自体を変更しました-それは機能しました！















使っていますか



私はそれを使用します。 時々、より良い音が欲しいときにオンにします。 私はそれをもっと頻繁にオンにしますが、それを使用するのは不便です-ケースは決してしませんでした、私はラップトップを前後にドラッグします...



アップグレード：



低ノイズの安定装置を備えた外部電源を使用すると、音が良くなります。 USBバスの電源にはさまざまな干渉が含まれています。 抵抗器R7、R8で実験することもできます-コンデンサーC12、C13を減らして増やします-低周波伝送が改善されます。

S / PDIFを出力することはまだ可能でしたが、プリント回路基板にトラックを貼り付ける場所がなかったため、そのときは必要ありませんでした。 そして、それは超小型回路の5番目のピンにあります。

両面の適切に設計された回路基板は、この設計に有害ではありません。 銅の層全体が「グランド」の下に割り当てられるため、これによりリターン電流の経路が短くなり、干渉レベルが低下します。 現時点では、このDACの隣に携帯電話があり、着信コールまたはメッセージを受信すると、おなじみの「あなたはあなたが...あなたはあなたが...あなたはあなたは... s ..... "。



PCM2705が見つかりません...



アナログPCM2705はPCM2704-2707ラインです。 それらについて簡単に：

PCM2704：28ピンSSOP、ヘッドフォンおよびS / PDIF出力、外部ROMインターフェイス

PCM2705：28ピンSSOP、ヘッドフォンおよびS / PDIF出力、シリアルプログラミングインターフェイス

PCM2706：32ピンTQFP、ヘッドフォンおよびS / PDIF出力、I2Sインターフェース、外部ROMインターフェース

PCM2707：32ピンTQFP、ヘッドフォンおよびS / PDIF出力、I2Sインターフェース、シリアルプログラミングインターフェース



どれでも使用できますが、品質は同じです。

データシートを回路とボードとともにアーカイブに添付します （Sprint Layout 5で開きます）。
