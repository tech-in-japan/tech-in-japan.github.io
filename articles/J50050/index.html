Last.fmのホットキー



今日は、有名なプログラムLast.fmを改善します。

知らない人のために、このプログラムはあなたが聴いているものを監視し、あなたが潜在的に好きになるべき音楽、ある種の個人的なラジオ局をピックアップします。 彼女のおかげで、私は自分のコレクションに12を超える新しい未知のトラックを追加しました。 このプログラムは無料で、 http：//www.last.fmから入手できます 。

ただし、このプログラムには大きなマイナスがあり、これはすぐに100年前になりますが、開発者は頑固に欠陥を修正するすべての要求を無視します。 そして、このマイナスはグローバルなホットキーの欠如です 。 つまり 音楽を聴いて並行してゲームをプレイしている場合、次のトラックに切り替えるには、プログラムウィンドウを展開して[スキップ]ボタンをクリックする必要があります。 これは惨めです！

したがって、今日はこの欠点を修正します。 これには次のものが必要です。

  • 実際のプログラムLast.fm
  • オートホットキー


アカウントにLast.fmをインストールして構成した後、およびAutoHotkeyをインストールした後、 Last.fmの設定を開き、言語を英語に変更します。 これは、開発者がロシア語のホットキーが機能しない可能性があるとは思わなかったためです。

次に、 このスクリプトをダウンロードして実行します。

その結果、5つのグローバルホットキーがあります。

  • alt + shift + PageDown-次のトラック
  • alt + shift + end-ミュート
  • alt + shift + L-お気に入りに曲を追加
  • alt + shift +上 -音量を上げる
  • alt + shift + down-音量を下げる


ホットキーが常に機能するように、このスクリプトをスタートアップに追加します。

必要に応じて、割り当てられたキーを便利なものに変更できます。 AutoHotKeyスクリプトプログラミング言語は習得が非常に簡単で、プログラムに詳細なヘルプが添付されています。

作業を中断せずにプログラム機能を使用します;）
