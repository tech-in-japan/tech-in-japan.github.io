ITインフラストラクチャとセキュリティを監視、管理するための無料のSolarwindsユーティリティ





私たちはSolarwindsを熟知しており、長い間協力してきました。多くの人は、ネットワーク（およびそれだけではなく）監視用の製品も知っています。 しかし、あまり知られていないため、ネットワークデバイスの制御、インフラストラクチャ、データベースの管理、さらにはインシデントの処理に役立つ無料のユーティリティをWebサイトからダウンロードできます。 実際、このソフトウェアは有料製品の個別の断片です。 すべてのユーティリティは、試用版ではなく100％無料です。 猫の下の説明とダウンロードへのリンク。



このレビューでは、次のユーティリティ：



  • ネットワークインフラストラクチャ管理。
  • ITインフラストラクチャ管理。
  • ITセキュリティ管理
  • データベース管理;
  • ヘルプデスクサービス組織。


ネットワークインフラ管理



ipMonitor無料版







シンプルなツール。 インターフェイスはアドレスの範囲を設定し、それらをポーリングして、リモート監視に使用できるノードを提供します。 可能な監視ツール：ping、WMI、SSH。 サーバーでは、CPU、メモリ、ディスクなどの標準チェックを実行できます。 これは、古いipMonitorの無料版（無料版）で、最大50ノードをサポートします。 小規模オフィスやテストタスクの場合、何もありません。 説明付きの短いビデオがあります 。



フローツールバンドル







フロートラフィックを送信するためのデバイスの複製、テスト、および構成。 このツールは、レプリケーター、ジェネレーター、およびコンフィギュレーターの3つの部分で構成されています。 レプリケーターは、netflowトラフィックを受信して​​、1つ以上のレシーバーに転送できます。 ジェネレーター-テスト、ファイアウォールの設定の確認、およびその他の目的のためにnetflowを生成します。 コンフィギュレーターは、ネットワークデバイスを自動的に構成して、ターゲットデバイスにnetflowを送信します。 説明付きの短いビデオがあります 。



Traceroute NG







このユーティリティは、システムtracertのアドオンです。 ネットワークルート分析を実行し、TCPおよびICMPプロトコルを使用してパケットを実行できます。 説明付きの短いビデオがあります 。



ポートスキャナー







スキャンされたIPアドレスごとに、開かれたポート、閉じられたポート、フィルタリングされたポートのリストを生成します。 TCPおよびUDPでスキャンでき、スキャン後、収集したすべてのデータをファイルにアップロードできます。 コマンドラインを介して機能します。

説明付きの短いビデオがあります 。



ネットワークデバイスモニター







ネットワークデバイスの監視。 このユーティリティには、SNMPを介した監視用のさまざまなテンプレートが付属しており、レポートとアラートを設定できます。 ただし、制限があります。1つのデバイスの監視のみがサポートされます。 説明付きの短いビデオ 。



GNS3ネットワークエミュレーター







ネットワーク環境エミュレータ。 このユーティリティは、Windows、Mac、およびLinuxで動作します。 20以上のネットワーク機器ベンダーがサポートされています。 さまざまなネットワークトポロジをエミュレートして、すべてがどのように機能するかを確認できます。 説明付きのビデオがあります 。



Wiresharkの応答時間ビューア







応答時間による簡単なパケット分析のためのWiresharkアドオン。 約1200のアプリケーションのトラフィックを検出します。 アプリケーションの応答時間の計算機のように機能します。 説明付きのビデオがあります 。



ネットワークアナライザーと帯域幅監視バンドル







このユーティリティは、NetFlow、J-Flow、およびsFlowトラフィックを監視するように設計されています。 インターフェイスには、デバイス間の相互作用、アプリケーション、ドメイン、エンドデバイスなど、さまざまなスライスの形で表現があります。 さらに、最大60分間のデータを記録できます。



TFTPサーバー







グラフィカルインターフェイスを備えたTFTPサーバー。 Windows上でサービスとして実行されます。 説明付きのビデオがあります 。



IPアドレストラッカー







スキャン、変更のコミット、IPアドレスの競合の特定。 これは、シニアIPアドレスマネージャーの弟であり、200万の有料ソリューションとは異なり、最大254個のアドレスを管理できます。



リアルタイム帯域幅モニター







ネットワークインターフェイスの帯域幅監視とメトリックの視覚化。 また、Network Performance Monitor（NPM）のジュニアソリューションです。



通話詳細記録トラッカー







CCM CDRログを検索、フィルター、およびソートします。



IP SLAモニター







シスコデバイスからIP SLAデータを収集し、視覚化します。



高度なサブネット計算機







アドレス空間設計のためのIP計算機。



Kiwi Syslog Server Free Edition







syslogおよびSNMPの分析、受信、およびアーカイブ。 説明付きのビデオがあります 。



Windows用FTPボイジャーFTPクライアント







Windows用の高度なFTPクライアント。 FTP、FTPS、およびSFTPをサポートします。 このクライアントを介して、スケジュールされたファイル転送を実行できます。



SFTP / SCPサーバー







このユーティリティはWindowsサービスとして機能し、複数のデバイスへのファイルの同時転送をサポートします。



ITインフラ管理



Azureのコスト計算機







ユーティリティは、複数のアカウントの財務情報を同時に表示できます。 インターフェイスには、年、四半期、月のコストの内訳があり、さらに、使用されていない有料リソースが表示されます。 WindowsおよびMacプラットフォームで動作します。



Windows用イベントログフォワーダー







Windowsログからイベントをパックし、必要に応じてsyslogとして送信します。 柔軟なフィルタリングがあります。 Windowsログのどのイベントを制御する必要があるか- 最後の記事で書きました 。



Solar-PuTTY







高度なパテ。 これには、マルチタブインターフェイスと保存されたセッションの便利な検索機能があります。 説明付きのビデオがあります 。



VMモニター







VMwareおよびHyper-Vのパフォーマンスの監視。 インターフェイスには、基本的なパフォーマンスメトリックと仮想マシンのステータスが表示されます。



サーバーヘルスモニター







物理サーバーを監視するように設計されています。 すぐに使用できるのは、Dell PowerEdge、HP ProLiant、IBM eServer xSeriesサーバー、およびVMware ESX / ESXiハイパーバイザーです。 SNMP、WMI、およびCIM相互運用プロトコルをサポートします。



ストレージパフォーマンスモニター







データアレイの監視Dell EMC、NetApp、IBM、Pure Storage。 説明付きのビデオがあります 。



交換モニター







主要なExchangeサービスの監視。 説明付きのビデオがあります 。



PowerShellのリモート実行イネーブラー







ローカルおよびリモートのWinRMサービスを自動構成するためのユーティリティ。



Active Directoryの管理バンドル







3つのユーティリティのパッケージ：未使用アカウントの検出、未使用ワークステーションの検出、ADでのユーザーアカウントのインポート。 説明付きのビデオがあります 。



WSUSエージェントの診断ツール







リモートサーバー上のWindows Updateサービス（WSUS）の可用性を確認するためのユーティリティ。



Webトランザクショントランザクション







模擬テストを実行するためのユーティリティ。 Webアプリケーションでは、1つのトランザクションがサポートされています。



WMIモニター







WMIを介してWindowsサーバーを制御するためのユーティリティ。 1つのデバイスのみがサポートされているため、サーバーです。



Windows用のSNMPイネーブラー







リモートWindowsサーバーでSNMPカウンターを構成します。



ITセキュリティ管理



イベントログコンソリデーター







便利なイベント検索と条件付きアラート設定を備えたWindows Log Consolidator。 説明付きのビデオがあります 。



ファイアウォールブラウザ







ファイアウォールルールをテストするためのユーティリティ。 Cisco、Check Point、およびNetScreenデバイスからの構成のインポートをサポートします。



Active Directoryのアクセス許可アナライザー

データベース管理

ヘルプデスク組織



Web Help Desk Free Edition







インシデント、問題、およびサービスレベルを管理します。 ADおよびレポートとの統合があります。 これは、完全に機能するバージョンの弟です。



これらのユーティリティのほとんどは、ローカルの問題を解決するのに適しており、一部はプロダクションシステムとして使用できます。 過去2〜3年で、Solarwindsは積極的に製品を開発してきましたが、現在では、組織のレベルを監視するための普遍的なシステムと見なすことができます。 この役割の監視システムを探している場合は、Solarwindsを含めることを検討してください。
