サマラでのWebProfessionals会議のプログラム全体を公開しました





WebProfessionalsカンファレンスはまもなく開催されます-4月3日、サマラで。 本日、すべてのレポートを含む完全なプログラムがイベントWebサイトで公開されたことをお知らせいたします。 次のリンクでスケジュールを見つけることができます。



WebProfessionalsの訪問者は、次のトピックに関するプレゼンテーションを待っています。



  • Windows Server 8およびWebプロジェクト向けのIIS 8の革新。
  • IISサーバーおよびWindows Azure上のASP.NET/PHP/Python/Ruby/Node.JS上のプロジェクト、フレームワーク、CMSを操作します。
  • Visual Studio Web開発ツールとASP.NETプラットフォームの新しいバージョン。
  • 負荷の高いソリューションの実用的な例に関するストーリー。
  • Windows Azureクラウドプラットフォーム
  • Metro UIのスタイルのデザインに関するDenis Kortunov（Parcsis / Turbomilk）からの特別レポート。


会議への入場は無料です！ 急いで登録してください 。



さらに、カンファレンスでは、主要なマイクロソフトの専門家やさまざまな企業の招待された専門家とチャットできます：Abbyy、1C-Bitrix、Parcsis / Turbomilk、Inln、ロシアのDrupalコミュニティ。 このページでスピーカーの完全なリストを見つけることができます 。



このカンファレンスの特別ゲストは、Microsoftウェブプラットフォームツールの開発に携わっている米国レドモンドの開発者であるPranav Rastojiです。 Pranavは、ASP.NET 4.5、Visual Studio 11、WebMatrix 2、Windows Azure、Node.JSの3つのトピックに関するプレゼンテーションを行います。 プラナバのセッションの1つは、聴衆とのライブコミュニケーションの形で紹介されます。
