ゲームワールドでのゲームコーディングナイトハッカソン

10月初旬、モスクワはゲーム開発者の最大の会議とイグロミール展を開催します。 開発者向けのイベントをもう1つ追加することを決定し、ゲームワールドの一部として特別なナイトハッカソンGames Coding Nightを開催します。 ハッカソンは、2014年10月2日から3日の夜に開催されます。



ゲームの開発に挑戦したことがない場合でも- 登録して 、来て、翌日には、WindowsまたはWindows Phone、さらには仮想現実の眼鏡やKinect向けの自己開発ゲームを自慢できます。







ハッカソンでは、さまざまなゲームプラットフォーム（HTML5、 Unity 、XNA、 Monogameなど）でゲームを開発する機能、Kinect for Windowsのプログラミング方法、ゲーム業界の第一人者からのヒント、 自分で仮想現実の眼鏡を作る方法を学びます。 ピザ、コーヒー、ジンジャーブレッドを好む人と一晩中コーディングできます。 ハッカソンの専門家の中には、Tatyana Smetanina、Dmitry Andreev、Dmitry Soshnikov（Microsoft）、Valentin Simonov（Unity）、Alexander Bragin（Mail.ru）などがあります。



10月2日の夜 19:00に水族館ホテル（クロッカスエキスポ、ミャキニノメトロ）で始まります！ スケジュールとハッカソン（およびこれは開発ツール（ Visual Studio 、 Unity3Dなど）がインストールされたラップトップ） の登録ページでの詳細情報 。 ハッカソンで開発される最高のプロジェクトは賞品を受け取ります！



開発したいゲームアプリケーションのアイデアを思いつく方がよいのですが、これも必要ありません！ 私たちは、あなたが学び、後で-おそらく-お金を稼ぐことができるゲームアプリケーションのアイデアを思い付くのを助けます。 ハッカソンで開発されたアプリケーションは、 Imagine Cupコンテストに提出されたプロジェクトの開始、スタートアップの開始、および履歴書の良い行であり、プライドの別の主題になります。
