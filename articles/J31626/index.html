ビル・ゲイツの出発は、オープンソースソフトウェアとのマイクロソフトの和解の時代の始まりを示していますか？

ビル・ゲイツが立ち去ることで、マイクロソフトはよりオープンソースになりますか？



これは難しい質問です。 オープンソースとプロプライエタリなソフトウェアの両サイドのオブザーバーは、永続的なリーダーの辞任は会社のGPL時代の始まりを意味するものではないと主張しているが、レドモンドでのポジション変更のプロセスを加速する可能性が高い。



Open-Xchangeのヘッドで元SUSEのマーケティングディレクターであるRafael Laguna氏は、次のように述べています。 Linux 「ゲイツが去った後、「マンモス」だけが残っていた-バルマー。 近い将来に彼が出発すると、Microsoftは間違いなくオープンになるはずです。 ただ義務付けられました。」



別のオブザーバーは、ゲイツを辞めることは、同社がオープンソースポリシーを採用するのに役立つと指摘した。



「ほとんどの場合、Microsoftのコア戦略はWindows Everywhereでした。 ビル・ゲイツはこの原則の創設者であり、同社が500億ドルの収益を上げるのを助けました。 ただし、500億から1,000億に成長するには、明らかにWindows以外のシステムの世界を取り込む必要があります」と、同類ソフトウェアのディレクターであるバリークリストは言います。 「この目標は、ゲイツなしの彼らにとってより達成可能であると思います。 そして、私たちはすでにこの主な兆候を観察しています。」



オープンソースソフトウェアのスポンサーの1人は、ゲイツの早すぎる出発がプロプライエタリソフトウェアの終わりの始まりを示すと信じています。



「ビルゲーツはソフトウェアライセンスから利益を得る最初の方法の1つであり、この方法を使用して地球上で最大のソフトウェア会社を構築することができました。 [しかし]ソフトウェアライセンスの販売は当たり前になっています」とOpenxchangeのCTOであるJuergen Geck氏は、Microsoft Exchange製品と競合しようとしています。



もちろん、Microsoftが主力製品のソースコードを開くとは誰も期待していませんが、Microsoftの開発者でさえも、「ポストゲート」時代のオープン性とソースコードの公開への移行を期待しています。



「戦略とオープンソースコミュニティの存在を考慮に入れて、Microsoftがすべての機能を再検討することを強く感じています。 商用ソフトウェアをリリースし続け、知的財産権を非常に真剣に受け止めるという意味で、基本的にマイクロソフトは現状のままだと思います。 しかし、ASP.NET AJAX Control Toolkitのような、より多くのオープンソースまたは部分的にオープンソースのソフトウェアがCodePlexで見られると信じています。」ヨーク。 「Microsoftは、.NET内および一般的にマイクロソフトの利益の範囲外の両方で、オープンソース企業、コミュニティ、および個人の愛好家とより緊密に連携することになると思います。」 また、Brustは次のように述べています。「Novellがスポンサーで、Linuxプラットフォーム向けのNovellが提供するSilverlightオープンソース実装であるMoonlightと、Microsoftが開発支援を提供するためにこのチームと提携しているという事実。 スコット・ガスリーは、私が知る限り、この種のオープンソースの世界との建設的な相互作用に対して非常に信頼できる人物であり、彼の影響力が大きくなると信じています。」



Linux Foundationのリーダーの1人は、Microsoftの新しいエリートが何をするかは予測できないが、オープンソースソフトウェアは、同社の市場とビジネスのやり方に既に大きな影響を与え続けると述べた。



「すべてはBallmerとOzzyに依存しており、社内で非常に興味深い議論が行われると予測しています。 Linux Foundationのエグゼクティブディレクターであるジムゼムリンは、次のように述べています。 「将来は分散コンピューティングにあり、Web 2.0アプリケーションの開発はLinuxとオープンソースに基づいています。」



他の人々は、ゲイツが取締役会の議長および主要株主として会社に対する影響力を保持し、戦略的問題に関するビジョンを前進させると主張しています。



「ビル・ゲイツは彼の退任に向けて素晴らしい計画を持っていました。今日、会社は多くの偉大なリーダーに率いられています。 彼は非常に重要なシンボルであり、私たち全員が見逃すであろうカルトの人物です。 しかし、取締役会の議長はマイクロソフトの著名な人物であるため、彼は間違いなく会社の戦略的決定に影響を与えるでしょう」
