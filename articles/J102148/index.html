PS3はハッキングされており、バックアップゲームを作成/アップロードできます

Sonyの戦いは長い間夢見ていたものでした。権威あるサイトPSX SceneとOzModChipsのチームは、PS Jailbreak USBドングルを使用すると、内蔵ハードドライブからPS3ゲームのバックアップを作成およびダウンロードできることを確認しました。 次のビデオが証拠として提供されています。







このドングルを使用するエクスプロイトは不明のままですが、自作シーンの専門家は、このデバイスがコンソールをペイントし、デバッグモード（いわゆるusb-jig ）に入るファイルを使用することを提案しています。



アメリカ、ヨーロッパ、日本地域のコンソールのFATおよびSLIMバージョンのサポートを宣言しました。 ドングルの作成者は、すぐに脆弱性を修正できる（またはUSBポートが無効になる:)）ため、ファームウェアバージョンを更新しないことをお勧めします。



更新：自作シーンの一部のプレーヤーは、このハックはリークされたバージョンのPS3 SDKを使用して作成され、コンソールのデバッグバージョンでのみ機能すると主張しています。











デバイスをPS3に接続すると、[ゲーム]メニューにバックアップマネージャーの項目が表示され、内蔵HDDにゲームのダンプを作成できます。 イメージの作成にかかる時間は約8〜10分です。 このプロセスは次のようになります。









このサイトでUSBドングルを注文できます： http : //psjailbreak.com （一時的に置きます）
