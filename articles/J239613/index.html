Sypex GeoがVK APIバインディングを追加しました

Sypex Geoでは、労働者の要求に応じて、VK APIで使用される都市、地域、国のIDにバインディングが追加されます 。



Sypex Geoとは何ですか？



Sypex Geoは、地理位置情報分野のソリューションの1つです。IPアドレスに基づいて訪問者の地理的位置（国、都市、地域）を決定します。 独自のオープンバイナリデータベース形式を使用し、REST API Webサービスも利用できます。これにより、常に最新のデータベースを使用できます。 Sypex Geoは、IPアドレスの場所が変更されると、定期的に更新される独自のデータベースを使用します（無料データベースは月2回、商用データベースは月4回更新されます）。



ジオロケーションの例



  • 訪問者がいる地域の支店の連絡先を訪問者に表示します。
  • 訪問者の都市または地域によっては、オンラインストアにこの地域の特別オファーが表示される場合があります
  • 同社は、商品やサービスに異なる価格を設定して、競合他社が強い決済の価格を下げることができます。
  • オンライン広告（コンテキスト、バナーなど）の地理的ターゲティングを実施します。
  • さまざまな国からの訪問者については、商品とサービスの価格を異なる通貨で示すことができます。
  • さまざまな国からの訪問者は、メインサイトの全国ミラーにリダイレクトできます。
  • 市場調査の実施。


IPアドレスによる訪問者の位置情報
