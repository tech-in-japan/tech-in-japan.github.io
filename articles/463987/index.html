Applications for e-books on the Android operating system (part 2)

In the first part of the review of applications for electronic books on the Android operating system, the reasons for which not every application for the Android system will work correctly on electronic books with the same operating system were outlined.







It was this sad fact that prompted us to test many applications and choose those that will work on reading rooms (albeit with limitations).



Briefly, the causes of difficulties in the work of applications on the "readers" are as follows:



  1. "Readers" have a black and white screen; color display in applications should not be fundamentally important;
  2. Reader screens are also updating rather slowly, so applications should not display fast-changing content;
  3. Applications should not be paid, because e-books do not have a Google Play store; necessary to install paid applications (but paid content in the application is not excluded!);
  4. Applications should be fundamentally compatible with readers, even if all three of the previous conditions are met.


The first part of the article presented the results of testing office applications with links to workable versions of APK installation files.



In this (second) part, the results of testing two categories of applications related specifically to the process of reading books will be presented: Bookstores and Alternative applications for reading books (that is, not pre-installed on sold e-books).



As a result of testing the applications, their common problem was revealed: in their overwhelming majority they are not optimized for electronic books in terms of image contrast.



As a rule, they provide measures to limit excessive image contrast on smartphones and tablets. Usually for this purpose, the background of the text is made not completely white, but slightly darkened (gray, yellowish, with imitation of the pages of an old book, etc.); and the letters in the text are not made completely black, but dark gray.



But e-books do not need to limit the contrast, since their screens already have lower contrast compared to smartphones and tablets. An additional limitation of contrast may result in a lack of contrast.



In this regard, users are recommended in those applications where it is possible to adjust the background color and characters (letters), set the background as light as possible, and the color of the letters as black as possible (or vice versa - for fans of the "inverse" image).



Applications were tested on ONYX BOOX e-books with versions of Android 4.4 and 6.0. Before installing the application, the user needs to make sure the application is compatible with the version of Android under which his e-book works.



The following data is indicated in the application description:



  • name (exactly as indicated on the Google Play store; even if it contains spelling or stylistic errors);
  • developer (sometimes applications with the same name may be released by different developers);
  • application purpose
  • Required Android version
  • Link to the finished APK file verified at the MacCenter;
  • A link to this application in the Google Play store (for a more detailed acquaintance with the application and reviews; you cannot download the APK installation file there);
  • a link to download the installation file for the APK of the application from an alternative source (if any);
  • A note indicating the possible features of the application;
  • some screenshots of a running application.
