Windows Phoneがトルクを起動するのに最適なプラットフォームになった理由

はじめに：先月、 ご存知かもしれませんが、モバイル向けポータルのPocketGamer.bizは、有名なZeptoLabチーム（Cut the Ropeの開発者）の元メンバーであり、ロンドンのゲーム開発スタジオLextreのCEOであるVictoria Pavlovaにインタビューし ました 。







このインタビューでは、ビクトリアはWindows Phoneプラットフォームでのスタジオの最初のゲームであるTorqueアーケードレーシングゲーム（他の国ではPerfect Shiftと呼ばれていました）の目を見張るような成功だけでなく、この成功の理由も説明しています。通り過ぎる。 WPの開発経験のみを共有したいという要望、特に開発パスを開始したばかりの人々と共有したいという要望があり、ブログの新しい形式を試してみました。



以前ZeptoLabでCut the Ropeプロジェクトに携わったVictoria Pavlovaは、今日、ロンドンに本社を置く小規模ゲーム開発会社Lextre StudioのCEOです。 スタジオのデビューは、アーケードシミュレーター「トルク」でした。



LextreはこのゲームをWindows Phoneプラットフォームのみでリリースすることを決定し、最初の2か月で200万ダウンロードのレベルに達しました。これは、2014年の終わりにiOSおよびAndroidプラットフォームでTorqueをさらに起動するための良い出発点となりました。







なぜWindows Phoneに賭けたのですか？



「トルク」は、私たちが市場にリリースした最初のゲームでした。 したがって、思いやりのあるユーザーから質の高いフィードバックを得ることが非常に重要でした。 これがおそらくWindows Phoneに注目した理由です。WindowsPhoneでは、ゲームコミュニティは他のプラットフォームほど大きくはありませんが、高品質のゲームに非常に興味を持っています。



ゲームのリリースは、ユーザーエクスペリエンスを向上させるだけでなく、ゲーム内のユーザーを引き付けて維持するためのメカニズムをより正確に理解するのにも役立ちました。 ユーザーが常にゲームを楽しめるように、ゲームプレイをカスタマイズする方法を見てきました。



Windows Phoneゲームコミュニティのおかげでこの貴重な経験を得て、Torqueをすべての最新のモバイルデバイスで完全に動作するクロスプラットフォームゲームに変えることができました。



一方、このアプローチは、業界で受け入れられているものとは非常に相反します。ほとんどの開発者は、より多くの聴衆、すなわち、 ユーザーが密集している2つのプラットフォームの1つであるiOSまたはAndroid。 その後、ゲームを再描画してWindows Phoneに移植します。



その結果、初心者の開発者は、大規模なプラットフォームでの日光浴の場で苦労することがよくありますが、Windows Phoneには優れたコンテンツがありません。



この厄介な状況に対処するため、パーフェクトシフト用のまったく新しい3Dエンジンを開発しました。これは、ほぼすべてのWindows Phoneデバイスで理想的にサポートされています。 つまり、Microsoftモバイルプラットフォームのファンは、AndroidおよびiOSでゲームがリリースされる前に最高のゲーム体験を楽しむことができます。



Windows Phone向けに開発するメリットは何ですか？



Windows Phoneを選択した開発者は、視聴者がどれほど活発で興味を持っているかをすぐに理解するはずです。



「Torque」などのブレークスルーは、Windows Phone専用にリリースされたゲームがこのプラットフォームのユーザーの注意を引くだけでなく、プレイヤーが喜んで退場するという例外的なフィードバックを得ることができることを示しています。



つまり、開発者は、現在および将来、ゲームをさらに良くする方法について完全なアイデアを得ることができます。







Kabam [モバイルゲーム開発者、約 Per。] 1つのゲームのリリース後、Windows Phoneプラットフォームでのプロジェクトのさらなる開発計画を放棄しました。 WPが行き止まりだと主張する人々に何を言いますか？



Windows PhoneプラットフォームでのTorqueの独占的発売を決定したという事実により、わずか2か月で200万ダウンロードを獲得することができました。 この状況は、Windows Phoneプラットフォーム上のスマートフォンのすべての機能を使用する本当に良いゲームに対する需要がどれほどあるかを示しています。



さらに、このプラットフォームのみが、スタートアップの自然な進歩のための「トルク」の大きな機会を提供できました。



最終的に、App StoreとGoogle Playの裏通りには、数千もの高品質のゲームがあります。 そして、彼らはおそらく二度と人気になることはありません。 また、Windows Phoneに新しく魅力的なデバイスが登場したことで、プラットフォームの能力は向上するだけで、さまざまなアプリケーションやゲームが改善されます。



ZeptoLabでの以前の仕事で、有名なCut the Ropeに取り組み、これまでに6億回以上ダウンロードされました。 このゲームを最初にWindows Phoneで公開したらどうなると思いますか？



正直に言うと、そのようなステップを決定した場合に何が起こるかを想像するのは非常に困難です。 Cut the Ropeは、スマートフォン市場が現在の市場と大きく異なっていた4年前にリリースされました。

しかし、同じ戦略がすべての人に適しているわけではないことを認識しておく価値はあります。 最終的に、それはすべて、視聴者を理解し、特定のゲームに適切なタイミングで適切なアプローチを選択することに帰着します。



私たちの場合、他のプラットフォームでのリリースのかなり前に、Windows Phoneで「Torque」を本当に魅力的で格好良いゲームとしてリリースする良い機会を見ました。







Windows Phoneでゲームをリリースすることについて何を学びましたか？ このプラットフォームで引き続きリリースされますか？



私を信じて、開発プロセスの間に、私たちは非常に強力なサポートを提供してくれたマイクロソフトのおかげや、プロジェクトを本当に愛していたゲーマーから受け取ったフィードバックのおかげなど、多くの新しいことを学びました。



Windows PhoneプラットフォームでのTorqueの大成功に触発されて、私たちはもちろん将来の新しいプロジェクトのためにWindows Phoneを検討します。
