エンタープライズグレードの専用サーバー

読者の皆様、当社の新製品であるColoBridgeの専用サーバーのラインを最初にオープンしたDedicated X16に慣れることをお勧めします。





この機器は、ドイツのフランクフルトにあるSatab 70 Type II Databurgデータセンターの技術サイトに収容されています。 すべてのColoBridgeサーバールームには、無停電電源装置、空調、ガス消火、および不正アクセスに対する保護の冗長システムが装備されています。 外部では、サイトは第1層のレベル3およびGlobalCrossingのインターネットオペレーターのネットワーク、および最大のトラフィック交換ポイントに接続されています。MSK-IX（モスクワ）。 DE-CIX（フランクフルトアムマイン）; AMS-IX（アムステルダム）; エクイニクス（アシュバーン、米国）; LINX（ロンドン）。 内部ネットワークインフラストラクチャは、ギガビットインターフェイス上に構築されます。







Dedicated X16の主な機能の1つはハードウェアプラットフォームです。これにより、1つのエンクロージャーで複数の物理サーバーをホストできます。 このソリューションの特徴は、ハードドライブとサーバーモジュールの両方のホットプラグ（ホットプラグ）のサポートです。 このため、ハードウェアの問題が発生すると、モジュールはすぐにバックアップモジュールに置き換えられ、サーバーのダウンタイムが最小限に抑えられます。 問題のあるコンポーネント（プロセッサ、マザーボードなど）のローカライズは、クライアントサーバーの外部で既に行われています。 ケースには2つの電源が装備されており、それぞれがすべてのモジュールに対応できるため、サーバーに冗長な電力を供給できます。



標準サーバー構成：



  • プロセッサ ：サーバークアッド、ハイパースレッディングをサポートするIntel Xeon L3426（8仮想コア）。
  • RAM ：16 GB DD3 ECC Registered（最大32 GBまで拡張可能）。
  • ハードドライブ ：合計で、最大6つのメディアをサーバーに接続できます（標準パッケージには2つのメディアが含まれています）。 接続オプション：2 x Intel X-25M G3 SSD 80 GB、2 x SATA Seagate Constellation。500 GBまたは1 x SSDおよび1 x SATA。 ドライブは企業向けに設計されています。
  • ネットワーク ：2 x 1 Gbpsの冗長ネットワーク接続。
  • KVM over IP ：電源管理、コンソール、仮想メディア接続、ハードウェア監視。
  • バックアップ ：外部ファイルストレージに100 GB。


さらに、シスコの機器に基づいたVPN接続サービスと、SMSを介したサーバーへのネットワークアクセスのブロックも利用できます。 最新のハードウェアのおかげで、Dedicated X16はあらゆるタイプの仮想化と現在のほとんどすべてのオペレーティングシステムをサポートしています。 必要に応じて、ColoBridgeの技術スタッフが問題についてアドバイスし、サーバーへのプロジェクトの展開を支援します。
