ビデオムービーを.m®形式に変換する（Mobiclip）

この記事では、PDAの画面でFullHD形式で映画やクリップを受信および再生する方法を説明します。 ブレーキ、フリーズ、相互のオーディオまたはビデオトラックの遅延、およびその逆-オリジナルの品質と、自宅のFullHDパネルのようにこのフォーマットを表示することによるすべての利点があります。 絶対に重要ではありません。また、画面が176x220からWVGAまでのWindows Mobileベースの最も弱いデバイスを持っているという事実もあります。







Mobiclip （.mo形式）は、ビデオファイルの再生、フルスクリーンモードでのフラッシュアニメーション、曲の再生を可能にする非常に高度な技術です。 また、本格的なフラッシュビデオゲームをプレイすることもできます。 アニメーションコール（フラッシュ）、ビデオコールを使用します。 カラオケ機能もあります。

Mobiclipは、ローエンドデバイス（16 MHzのプロセッサ（ARM7はこの周波数で実行されます））で動作するため、本当に進歩的なテクノロジです。 確かに、冗談はMobiclip Playerが無料であるのに対して、Mobiclip Converterは開発者の間でのみ配布され、多額の費用がかかるということです。 また、Mobiclip Converterはベータ版のみです。



もちろん、ビデオを.mo形式に変換するには、重量が21323 KbのMobiclip Converter自体が必要です（既にアーカイブにあります）。 さらに、コンバーターは、製造元のWebサイトから無料でダウンロードできるMicrosoft .Net Frameworks 1.1パッケージなしでは起動せず、奇跡パッケージ23697 Kb（既にアーカイブに含まれています）の重量があります。 ビデオを変換するには、893 KbのVirtualDubが必要です（VirtualDubアーカイブ1.9.0.30586ロシア語バージョン+プラグインで既に）。

したがって、必要なプログラム（Mobiclip Converter、Microsoft .Net Frameworks 1.1、VirtualDub）をダウンロードしてインストールした後、作業を​​開始できます。



ビデオを.mo形式に変換する手順 ：

1）VirtualDubを実行し、ソースファイル（video.aviなど）を開きます

2）[ビデオ]メニューで、[フル処理モード]を選択します

3）[ビデオ]メニューで[圧縮]を開き、リストからビデオコーデックを選択します-Actimagine Mobiclip Video Codec

4）[設定]ボタンをクリックし、スライダーを75％から70％に再配置します。スライダーを使用して、画像の品質、つまり画像の品質を調整します。 ご理解のとおり、画質を試すことができます

5）[ビデオ]メニューで[フレームレート]を選択し、ラジオボタンを1フレームおきに処理する（2で間引く）にすると、フレームレートが30 fpsから25 fpsに変更されます

6）[ビデオ]メニューで[フィルター]を選択します。 メニューの「フィルター」フィルターの束では、それらを試すことができますが、必要なものだけを考慮します。 [追加]ボタンをクリックして、サイズ変更フィルターを追加します。 サイズを変更します。「New Width」の行で、たとえば640を指定します。 新しい高さの行で、480と書きます。「フィルターモード」の行で、正確なバイキュービックを設定します。 その後、「表示」ボタンをクリックして、何が起こったかを確認できます。 Windows Mobile（表示範囲は176 x 220から480 x 800まで）に関して、幅と幅の値は異なる場合があります。

7）[フィルター]メニューで[追加]ボタンをクリックし、[回転]フィルターを追加します。これにより、画像を回転するように求められます：左90、右90、約180-左90、右90、約180現在、PDAの幸せな所有者は、通常の形式でビデオを見ることができます。

8）「オーディオ」メニューで、「フル処理モード」を選択します

9）[オーディオ]メニューで[圧縮]を開き、圧縮なし（PCM）であることを確認します

10）[オーディオ]メニューで、[変換]を選択します。 [変換]ウィンドウで、選択ボタン48000 Hz、高品質、16

ビット、ステレオ（またはどこでも「変更なし」にチェックマークが付いています）

11）[ファイル]メニューで[処理パラメーターの保存]をクリックし、たとえばMobiclip.vcfなどの名前を指定します。 したがって、すべての設定を保存し、次回にそれらを再インストールする必要がないようにしました。

12）最後に、[ファイル]メニューで[AVIとして保存]をクリックし、videoclip.aviなどの名前を指定します。 私たちは、目の前で変換プロセスがどのように行われているかを見ます。 VirtualDubを閉じる

13）ここで、videoclip.aviを保存したディレクトリを開きます（どのプレーヤーでも表示できます）。 videoclip.aviを右クリックし、ドロップダウンメニューで[Mobiclip（.mo）に変換]を選択します。 videoclip.aviが配置されている同じディレクトリで変換した後、videoclip.moが表示されます

おめでとうございます、これらの地獄のような労苦の後、私たちがしたことを見ることができます！ videoclip.moを開いて、持っているものを見てください（Windows OS向けのMobiclip Playerはコンバーターの一部です）。 次に、ActiceSyncとXデバイスなどを使用して、videoclip.moをmobiclipフォルダー（つまり、ディスク上のメモリカード：mobiclip）のmicroSD（HS）（誰がどのカードを持っているか）に転送します。 または、単にメッセージとしてデバイスに送信します。

フラッシュアニメーションを.mo形式に変換する手順 ：

.swfフラッシュアニメーションを.mo形式に変換するには、Mobiclip Converter、Microsoft .Net Frameworks 1.1、およびMacromedia Flash MXなどのフラッシュアニメーションを作成するためのツールが必要です。



1）アルファ効果、グラデーション塗りつぶし、またはマスクレイヤーを使用しないでください

2）ビットマップを背景画像として使用しないでください

3）1つのオーディオストリームのみを使用する

4）Macromedia Flash MX自体については、[変更]メニューで[形状]を選択し、[線を塗りつぶしに変換]を設定します。 事実は

5）ライブラリの[プロパティ]ウィンドウで、[インポートしたJPEGデータを使用する]のチェックを外します

6）[ファイル]メニューの[Flash]セクションで[公開]を選択し、サウンドに次の値を設定します：Raw、Mono、44 Khz

7）フレームレートは4以上である必要があります。最適でないと、音声付きのportacが存在します。

独自のフラッシュアニメーションを作成し、Macromedia Flash Playerで表示した後、右クリックして、ドロップダウンメニューで[Mobiclip（.mo）に変換]を選択します。 結果のフラッシュムービーは、コンピューターで表示できます。



注 ：

上記の点により、ネットワークからダウンロードしたフラッシュアニメーションは私たちにはあまり適していません。MobiclipPlayerは気まぐれすぎることが判明し、現代のフラッシュアニメーションに典型的な効果の多くを理解していません。

ただし、Masyanなどのネットワーク上で.swf形式のフラッシュアニメーションをダウンロードしてリメイクすることができます。そのためには、Macromedia Flash MXを実行し、[ファイル]メニューから[インポート]を選択し、ソースファイル（Masyanya.swfなど）を指定する必要があります 次に、上記の手順に従ってすべてをやり直す必要があります。
