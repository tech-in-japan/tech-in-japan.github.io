Linux Mint 15「オリビア」

現在最も人気のある Linuxディストリビューション（distrowatch.comによる）は、Ubuntu 13.04に基づいてリリースされています。



Linux Mint 15にはデフォルトでMATEシェルとCinnamonシェルが付属しています。これらはそれぞれGNOME 2とGNOME Shellのフォークです。

KDEとXfceを備えたディストリビューションバージョンもありますが、通常は少し後にリリースされます。

  • gcc 4.7.3
  • glibc 2.17
  • メサリブ9.1

  • python3 3.3.1

ディストリビューションはUbuntu 13.04に基づいているため、サポートはそれぞれ2014年1月まで続きます。

アプリケーションソース：








MATE 1.6


  • Gatesに置き換えられたMateConf構成管理システム。
  • MateCorbaとMateComponentはDbusに置き換えられました。
  • ユーザーセッションを管理するためのSystemd-logindのサポートが追加されました。
  • Cajaファイルマネージャーの変更：サイドバーの実装の改善、サムネイル生成の新しい仕様のサポートの追加、サムネイルを表示するための新しい領域の追加など。
  • 画面の中央に新しいウィンドウを開くオプションがMarcoウィンドウマネージャーに表示されました。
  • 設定コントロールセンターの追加機能：Marcoウィンドウマネージャーで複合モードを有効にし、Atl + Tabでウィンドウ間のクイック切り替えをアクティブにするためのオプションがウィンドウ設定ブロックに追加されました。
  • アプリケーションのデフォルトリストでは、ターミナルエミュレータを選択できます。
  • Atrilドキュメントを表示するためのプログラムにXPS形式で作業するためのバックエンドが追加されました。
  • 通知の出力を調整して、通知を表示する画面を示すオプションがデーモンに追加されました。
  • 新しいテーマとアイコンが追加されました。


















シナモン1.8


  • 記述子の概念が導入されました - デスクトップ -統合されたミニアプリケーション。
  • この環境には、プログラムの起動、クロック、フォトフレームの3つの基本的なデスレットが含まれています。 さらにダウンロードできるのは、 xkcdコミックと「バイナリクロック」を表示するためのデスクレットです。
  • Nemoファイルマネージャーの機能が拡張され、ツールバーとメニュー、2ペインモード、クイックナビゲーション用のメニュー、ファイルのリストを表示するコンパクトなフォーム、サイドパネルなどの機能が保存されます。
  • この構造には、スクリーンセーバーの独自の実装が含まれています。その機能は、画面ロック中に表示されるメッセージを残す機能です。
  • 使用可能なすべての設定モジュールが単一の構成センター-Cinnamon Settingsに統合されました。
  • アドオンの開発者向けに、desletとアプレットの設定を操作するための新しいAPIが導入されました。
  • ウィンドウマネージャーのMutterの代わりに、彼のフォークMuffinが関与しています。
  • 使用されたレンダリングエンジンのチェックがマフィンウィンドウマネージャーに追加されました。
  • ユーザーが要素の配置に最適なフォームを選択できるようにするデスクトップレイアウトテンプレートのサポート。






















少しだけ追加します。インターフェイスと速度が大幅に改善されたことは喜ばしいことですが、残念ながら、ロシア化には多くの要望が残されています。



RCからアップグレードできます。mintupdateで第1レベルと第2レベルのアップデートをインストールするだけです。
