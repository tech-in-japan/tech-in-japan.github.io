情報隠蔽方法

ハードディスク上の情報は、いくつかの方法で隠すことができます。 簡単なものから始めて、複雑なものに移って、それらについてお話します。 隠れているが見つけられない人のための記事。 したがって、隠蔽方法のみが示されています。 特定の方法についてより詳細な情報が必要な人-彼はそれを見つけ、そして最も重要なことに、調味料を混合することによって独自の独特の風味をもたらします。 また、特定のレシピは、隠すのではなく、求める人に役立ちます。

この方法では、情報をC：\ Users \ Walter \ AppData \ Local \ Temp \ 018iasywq8 \ user \ 10ha1pg1vythz21ds778b0ycq9r2に移動します。 彼らはこの方法を使用します...まあ、原則として、コンピューターがネットワーク上で動作しており、ボタンでオンになっていることを知っている人。

どうすれば見つけることができますか？ 最初の段落のように、「アーカイブ内の検索」のみを示します。

＃5 フォルダー名の変更

＃6 暗号化されたアーカイブ



現在、情報を表示する方法についてではなく、情報を検索する方法について説明しています。 したがって、暗号化自体とパスワードについては説明しません。 実際、最新のアーカイバには、暗号化中にアーカイブ内のファイルの名前を隠す機能があります。 ここで何か面白いことがあります。 ユーザーは少なくとも数時間かけてこの方法を学び、理解しました。



見つけるために？ アーカイブのみを見つけて、そのような困難な方法で暗号化されていると判断できます。 その中のファイルは決して見えません。 つまり、コンピューター上の情報を隠すという事実が明らかになります。 この記事ではこれで十分です。



＃7 特別なプログラムのインストール



情報を隠すためのプログラムの方法は、上記または基本的に以下と同じです。 しかし、そのようなプログラムの存在自体は、情報を隠すという事実をすでに語っています。 そして、もちろん、それをアンインストールすると、すべてのファイルが完全に表示されます。 この方法は、コンピューターに興味がある人に役立ちます。 通常、このようなプログラムにはパスワードが設定されているため、非常に簡単です。ネットワーク経由でコンピューターにアクセスする場合でも、アンインストールすることはできません。



しかし... ...平均的なユーザー向けに設計された既存のプログラムはすべて、情報を隠すという事実を明らかにし、そのような情報自体よりも注目を集めています。 彼らはフォルダに巨大な赤い看板を置き、自動実行とトレイで実行し、「私の隠しファイル」のような愚かな名前を持っています。



この分野で最も有名なベンダーの1つであるシマンテックは、巨大な看板が付いたフォルダーを作成します。 一部の攻撃者が、検索前に盗まれた2カセットレコーダーを自宅に隠すことを決定した状況を想像してください。 彼らは店に行き、パスポートを受け取り（海外でプログラムを取得する方法です）、高価な超金庫を購入します。 彼らは彼を彼のアパートの真ん中に置き、そこに2カセットプレーヤーを置きました。 軍人が来て、家中を回ります。2カセットプレーヤーはいません。 当然、彼らは尋ねます：「アパートの真ん中はどんな安全ですか？」そして彼らは彼らに言いました：「それはあなたのビジネスではありません！ これは私たちの私有財産です！「もちろん、戸口と金庫の指は開いています...そのようなプログラムよりも悪いのは、碑文でデスクトップにすべての秘密情報を配置することだけです：「私の秘密ファイルはここに保存されます!!!」



＃8 仮想ディスク



この方法は非常に一般的で、非常に非効率的です。 たとえば、True Cryptは、数百メガバイトのサイズの暗号化されたファイルを作成し、ディスクとしてマウントします。 この場合、現在必要なファイルだけでなく、すべての情報がネットワーク上および秘密監視の受信時に表示されます。 そして最も重要なこと-情報を隠すという事実を隠す必要がある場合、これは最も原始的な方法です。 暗号化された、たとえば700 MBのファイルを見つけることはまったく難しくありません。



＃9 論理ドライブを隠す



この方法は悪くありません。大きなファイルがないことを考えると、検索しても情報は見つかりません。 ポイントは、Windowsによりドライブの接続と切断が可能になることです。 ディスクが切断されている場合、ディスクエディタが起動するまで表示されません。 そして、これはもちろん難しいことではありません。 また、操作中にすべての情報が開かれます。 また、ネットワーク上およびリモート攻撃の両方で確認できます。 欠点は、その場で既存のドライブを分割することは安全ではないことです。 つまり、フォーマットするときにこの点を考慮する方が良いです。



＃10 ステガノグラフィー



手動で、または特別なツールを使用して短縮することができます。 通常、情報はグラフィックファイルとビデオファイルに隠されています。 配色をカットできる場所で、違いはわかりません。 この点から始めて、隠された情報がどのように見つけられるかを書くのではなく、それを検出するためにどのような専門家レベルが必要かについてお話します。



どのツール情報が非表示で、どのファイルで非表示になっているかがわからない場合は、専門家のレベルを非常に高いと判断します。 隠された情報を見つけることができる人々のカテゴリについては別に、以下で話します。



＃11 NTFSデータストリーム（ADS）



NTFSでは、できるだけ多くのデータストリームを単一のファイルにチェーンできます。 このようなデータは肉眼では見えず、別のOSを接続したり、セーフモードを使用したり、DOSを表示したりすることは役に立ちません。 一方で、検出することはそれほど難しくありませんが、...専門家のみです。 非専門家はそのような情報を見つけられません。



一方、この方法はあまり一般的ではありません。 情報がNTFS自体のサービスデータにかかっている可能性があると考える場合、一般的な検索は非常に複雑になる可能性があります。 他の方法とADSを組み合わせると、タスクは単純に難しくなります。 さまざまな方法とADSの組み合わせに応じて、中高レベルの専門家が必要になります。

誰が隠された情報を見つけることができるか



これについて言えば、アマチュアは自分の博学を誇示し、神秘的なフレーズ「特別なサービス」で他の人を感動させます。 一部の、特に「おおよその」ものは略語NSAを与えますが、他のものは復号化（国家安全保障局）、SVR、FSB、GRUなどを知っています。



私は常に2つの理由でばかげています。 情報や一般ユーザーのペルソナに関心を持たないのは、世界の5〜6都市に存在する特別なサービスです。 したがって、これらのサービス、その機能、および方法については、控えめに黙っております。



実際には、情報はユニフォームの「商人」から隠されており、次の「アクション」の間にPCを奪います。 発見できる秘密を支払わずに標準額を支払うために、情報を隠すことができます。 または、ライセンスのないソフトウェアが「従業員」の同じカテゴリから隠されている場合。 この分野の「商人」の知識は通常あまり深くないため、この場合は2 +の方法で十分と考えることができます。



あなたはあなたの人生の個人的な、純粋に親密な部分を誰からでも隠すことができます。 自宅であれば、ほぼすべての方法が機能します。もちろん、自宅の1つが同じNSAの代表者でない限りは機能しません。 管理者がいる職場では、より単純な方法に必要な管理者権限は通常ないため、状況は複雑です。 そのため、単純なもののうち、6、8が残ります（たとえば、TrueCryptが管理者権限なしで動作するように構成されている場合）。 そして実際の10、11からですが、もちろん、そのような情報を職場で維持することは愚かです。 さらに、ActualSpyなどのさまざまなユーティリティを使用して従業員のコンピューターを密かに監視している組織が増えています。 そして、隠蔽の意味は一般に失われます。



しかし、これらはすべて日常的でありふれたものです。 真の専門家はインテリジェンスサービスではなく、トヨタ、ガスプロム、ノキア、アドビ、Oriflameなどの企業に座っています。 本当の価値は、数十億ドルの超競争環境がある情報です。 このような企業は、競争力のあるインテリジェンスと反知能に予算のかなりの部分を費やしており、その専門家、方法、技術はペンタゴンとMOSSADよりも数年先を行っています。 そのような専門家にとって、上記の方法はほんの短い時間の問題です。
