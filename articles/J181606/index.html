3から15

夏にはそりを、冬にはカートを準備します。 Copyirazmはより強くなります。 それでは、困難な時期に備えて、Windows互換の代替オペレーティングシステムを作成する必要があります。 したがって、ReactOSプロジェクトチームは誇らしげに次のリリースを発表します：



ReactOS 0.3.15





このリリースの最も重要な革新：



  • USBマウス、キーボード、およびストレージデバイスのサポートの 「人種的に正しい」実装。
  • 書き換えられたセッション管理サブシステム。
  • 代替ReactOSメモリ管理モジュールは、メモリ管理のほぼすべての責任を引き受けました。
  • SATA-1 / 2 / AHCIをサポートする更新されたUniATAドライバー。
  • WinDbgを使用したReactOSコンポーネントのデバッグの予備サポート。
  • Virtual PC（および潜在的にHyper V）は、サポートされる仮想化プラットフォームのリストに戻りました（ ビデオドライバーのバグを除く）。
  • AutoHotKeyアプリケーションテストスイートの導入によるさまざまな改善。
  • 配布キットに統合されたドライバーのDriver Verifierユーティリティによるスキャンの結果に応じたバグ修正。




ディストリビューションをここからダウンロード 。 変更点の完全なリストはこちら 。



いくつかの新鮮なビデオ









最後に



従来のリリース配布オプションに加えて、LinuxTagでReactOSのデモに使用されるハイブリッドディスクイメージがあります。 このイメージには、より堅牢なデバイスの初期化と検出を提供するいくつかの追加のUSB修正、およびデフォルトのLautusテーマが含まれています。 スキンのサポートはまだ完全には実装されておらず、LinuxTagハイブリッドドライブでのLautusの使用は主にデモ目的で行われていることに注意してください。



PS残念ながら、 コンソールアプリケーションの作業の大幅な改善と、UniATAドライバーの最新バージョンは、いくつかの理由でリリースに含まれていませんでした。 がっかりしないでください。ReactOSの「 ナイトビルド 」で簡単に試すことができます。
