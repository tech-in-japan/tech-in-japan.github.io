ビジネスでのパテントトロール-VirnetXがAppleに6億2,500万ドルを提訴





VirnetXは些細なパテントトロールとしてよく知られていますが、今月はiMessageおよびFaceTimeでの特許の違法使用を証明することができました。 合計すると、米国の裁判所は Appleに625百万ドルを支払うように命じました。これは、前回の訴訟でAppleが支払った合計額よりもはるかに高い額です。



VirnetXの収益の大部分は、特許のライセンスと訴訟によるものです。 過去10年だけで、Cisco、Avaya、Siemens、Microsoft、および他のいくつかの企業を含む、いくつかの大規模IT企業がVirnetXの被害者になりました。



そのため、2010年、VirnetX は 、Windows Server 2003、XP、Vista、Live Communication Server、Windows Messenger、Office Communication、およびOfficeスイートのいくつかのバージョンでのVPN特許の違法使用により、Microsoftから2億ドル以上を獲得できました 。 そして2014年、同社は Skypeで独自の特許を使用したことでMicrosoftに対して2400万ドルの訴訟を勝ち取った。



「ju審員は私たちが常に言ったことを見ることができました。Appleはライセンスの規則に違反し、VirnetX特許の技術を使用した罪を犯しています」とVirnetX裁判所の弁護士は記者と共有しました。




それどころか、アップルの代表者はこの決定に驚き、特許が無効であると主張しました。



「私たちは、裁判所の決定に驚き、失望し、上訴するつもりです。 当社の従業員は長年この技術を開発しており、知的財産を保護するために特許を取得しています。 米国特許商標庁は、4つのVirnetX特許すべてが無効であると宣言しました。このようなケースは、特許システムの改革の必要性を確認するだけです」とアップルは言いました。




以前のAppleに対する3億6800万ドルの訴訟で、VirnetXはiOSで使用されているFaceTimeテクノロジーとVPNサービスの使用の違法性を証明することができました。 その後、Appleは独自のサービスを更新しましたが、VirnetXは変更が不十分であることに気付きました。
