LLCが133倍少ない税金を支払う方法







今日カバーしたいトピックはかなり複雑でボリュームがありますので、多かれ少なかれ明確で理解可能な方法でそれを提示するために努力しなければなりませんでした。 記事の最後に到達した人は、タイトルの質問に対する答えを認識します。 私を信じて、そこにある。



IT分野でのLLCの課税には固有の特性があり、個々の税金とは大きく異なります（前の記事「 個人の13.3倍の税金を節約する方法 」を参照）。 ビジネスの新人が知っておくべき5つの重要な違いを以下に示します。







1.非稼働のLLCは税金や手数料を支払うことができません



登録後の会社がまだ本当に機能していない場合は、しばらくの間、予算を支払うことができません。 これを行うには、次の条件を満たす必要があります。







  • 会社の貸借対照表には課税対象の不動産（不動産と輸送）がありません。
  • ディレクターが州に登録されていない（ リーダーのみの創設者に許可されている）か、無給休暇であり、他の従業員がいない。
  • UTIIへの移転は発表されていません（このモードでは、事業の遂行に関係なく税金を支払う必要があります）。


このような「一時停止されたアニメーション」の期間は法律によって制限されず、レポートをゼロにすることを忘れないでください。 LLCがその年の間に生命の兆候を示さない場合（アカウントに動きはなく、報告はあきらめられません）、アートによると。 国家登録法N 129-FZの21.1では、実質的に活動を停止し、機能していないというステータスを受け取っていると認められています。 このような法人は、税務調査官の主導でUSRLEから除外できますが、実際にはこれはめったに起こりません。







同様の状況では、非就労者の個人起業家は、依然として自分自身に義務的な保険拠出を行う必要があります（2017年には27,990ルーブル）。







2.事業所得の受領時に、LLCの設立者は配当税を支払います



配当に対する個人所得税は LLC税に直接関連していませんが、創業者にとっては、これらは会社の活動に関連する税金でもあるため、税負担を計算する際に考慮する必要があります。 個人の配当率は、LLCの税制に依存せず、次のようになります。







  • ロシア連邦の居住者の場合は13％。
  • ロシア連邦の非居住者は15％。


居住者の地位はロシア市民権の有無とは関係がなく、配当支払い前の過去12か月間で決定されます。 この期間中に合計183日以上ロシアに滞在している場合、ロシア連邦の居住者として認識されます。 あなたの不在の時間を数えない正当な理由は、他国での治療または訓練です（理由の完全なリストについては、ロシア連邦税法207条を参照してください）。







ビジネスから収入を受け取ると、個々の起業家は支払いをせず、配当の分配のために確立された慣習に拘束されません。







3.組織は個々の起業家より少ない税制上の優遇措置を持っています



罰金の大きさと査察官の厳しさから判断すると、州は、法人が民間の起業家よりも事業を成功裏に遂行し、その利益が高いと考えています。 したがって、LLCの税制上のメリットは少なくなります。







  • 当局は、法的形態に関係なく、中小企業全体に彼らを紹介することを約束しましたが、個人の起業家だけが免税期間の枠内で働くことができます（STSまたはPSNでゼロレートで2年間）。
  • 組織は特許（PIT）を購入することはできません。これは、場合によっては税負担に対して最も収益性の高いオプションです。 これには正当な理由はないので、この問題はビジネスのイニシアチブで定期的に議論されますが、これまでのところ役人はそれを満たそうとはしていません。
  • 一般的な場合のOSNOのLLCの所得税率は、OSNOの個人の個人所得税の13％に対して20％です。 確かに、一部の地域（チェリャビンスク、ノボシビルスク、ペンザ、ウリヤノフスクの地域）では、地域の法律により、認可されたIT企業のみが15.5％に料金が引き下げられています。


4. IT組織は、割引料金で従業員に保険料を支払います。



一般的な場合、従業員（年金、医療保険、社会保険）の拠出率は30％です。 2017年のIT組織の削減率は14％、2018年は21％、2019年は28％です。







特恵関税は、IT分野のすべての企業に割り当てられるのではなく、次の条件を満たす企業にのみ割り当てられます。







  • 収益の少なくとも90％は、開発されたプログラム、データベース、またはそれらの適応、修正、インストール、テスト、保守のためのサービスの開発と実装から受け取られました。
  • 少なくとも7人の従業員数。
  • 通信省の認定 。


同様の活動に従事している個々の雇用主には、これらの特典は適用されません。







5. STS収入およびUTIIモードでは、組織は計算された税を50％以下削減できます。



ここでは、保険料の支払い分だけ納税額を減らすことができる利点を念頭に置いています。 LLCは雇用主によって自動的に認識されるため、設立ディレクターのみが州に登録されている場合でも、税は半減することはできません。







労働者のいない個々の起業家は、自分のために支払われた拠出金の全額に対する税を減らすことができます。したがって、小所得の場合、まったく納税することはできません。
