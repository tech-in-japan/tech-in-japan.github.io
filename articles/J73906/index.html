Zotac MAG：nVidiaイオンネットトップ





Zotacは 、ネットトップの最初のラインであるZotac MAG を発表しました。



Zotac MAGは、高性能nVidia Ionチップセットと周波数1.6 GHzのエネルギー効率の高いIntel Atomプロセッサーをベースにした、サイズが186x189x38mmのコンパクトなコンピューターです。シングルコアモデルHD-NS01とAtom 230 、デュアルコアHD-ND01とAtom 330が 提供されます。





ノベルティには2 GBのRAM、160 GBのハードドライブ、統合されたnVidia GeForce 9400Mグラフィックスカードがあり、eSATA、6つのUSB 2.0ポート、ギガビットイーサネット、VGA、HDMI（1080p HD標準と完全互換）、 S / PDIF光出力および6-in-1カードリーダー（MMC / SD / SDHC / MS / MS Pro / xD）。 さらに、ネットトップにはWiFiモジュール標準802.11nが組み込まれています。



MAGの主な機能の1つは、 オペレーティングシステムがプリインストールされていないことです 。これにより、コストを削減し、ユーザーフレンドリーなシステムを使用できます。WindowsXP / Vista / 7およびLinuxデスクトップオプションとの互換性が保証されています。







新しいネットトップは、設置とインストールも簡単です。 コンピューターを配置するには、 3つの方法があります。ワークステーションの表面に置くか、コンピューターを垂直に置くか、配送に含まれるVESAマウントを使用してモニターの背面にマウントします。







最初のZotacネットトップは11月に発売され 、その価格は約250ユーロになると予想されています。



テストから判断すると、「強打のある」MAGは1080p（約50〜60％のCPU負荷-CUDAが役立ちます）に対応し、慎重にエネルギーを消費し（最大35 Wに達する）、かなり静かなクーラーを備えています。



一般的な情報：
  • NVidiaイオンGPU
  • Intel Atom 330デュアルコアまたはIntel Atom 230シングルコア
  • 2 GB DDR2-800メモリ
  • HDD 160 GB 5400 rpm
  • 寸法：186mm x 189mm x 38mm
  • 重さ：1.7kg
  • VESAスタンドとモニターの背面に取り付けるためのマウントが付属
  • プレインストールされたオペレーティングシステムなし
  • ギガビットイーサネット
  • 統合されたWiFi 802.11n
  • 6-in-1カードリーダー（MMC / SD / SDHC / MS / MS Pro / xD）
  • NVidia CUDAテクノロジー
  • NVidia PureVideo HDテクノロジー
