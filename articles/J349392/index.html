Sublime Text 3は生きています。 （セットアップと操作）

Sublime Text 3は、クロスプラットフォームのテキストエディターです。



初心者向けの簡潔なガイドです。 ここでは、最も必要かつ重要なすべてを収集します。 （必要に応じてリンクで詳細を確認できます）



巨大なデバッグ機能を必要としないPug / Lessのレイアウトで考慮します。 したがって、スピードと機能の両面で競合他社を打ち負かすのはこのエディターです（私の意見では）。







1.プログラムのインストールとアドオンの監視（パッケージ制御）



  1. Sublime Text 3をインストールします。ここではすべてが簡単です- ダウンロードして実行します。



ctrl /⌘+ shift + pまたはメニュー（[ツール]> [コマンドパレット]）を押します。



ここで、アドオンのダウンロード、インストール、アンインストール、表示などができます。





2.プログラム設定：



  1. タブの代わりにスペースを使用します 。

    「Translate_tabs_to_spaces」：true

  2. タブサイズは4スペースです。

    Tab_size：4



    なぜだろう？

    誤ってスペースバーとタブを一緒に押したり、パグアセンブリを破損したりすることはありません。

    また、均一なチームワークのために。

  3. 行の末尾に空の文字を含めることはできません 。

    これを行うには、 TrailingSpacesプラグインを使用します。



-おそらく必須のMustHave設定はここで終了します。 個人に移りましょう：



  1. 全画面モードF11 +メニューバーを非表示（alt）。 OSパネルの状態に気を取られずに、仕事でモニターを最大限に使用できます。

  2. 改行を防ぎます。 おそらく最高の物件がリストされています。 pug構文で混乱することはありません。 画面サイズに関係なく、常に予想されるコードとネストを表示します。 水平スクロールは、Shift +ホイール（またはtouchPanel）によって実行されます

    Word_wrap：false

  3. テーマ。 これについては何も言えません。 好きなものを見つけてください（できれば暗いアウトラインを使用してください）。 テーマを使用します：Material-Theme-Darker.sublime-theme。

  4. 構文の強調表示。 まあ、問題はないはずです。 拡張機能と強調表示が一致することを確認した場合（PugのJadeではなく、PugのPug）

  5. 表示→サイドバー→開いているファイルを隠す -プロジェクトツリー用のスペースを解放します。 このフィールドは既にタブと上部の3つのドットで複製されているため



3.便利な「ホットキー」キー：



  1. 最初の段落では、 ctrl（⌘）+ Z （⌘-以下、単にctrl）などのすべての標準的な組み合わせをスキップすると言います。 キャンセル、保存、繰り返し、タブを閉じる、タブを復元するなど...

  2. おそらく、次の最も一般的な組み合わせは次のとおりです。

    crtl + P-開いたプロジェクトのファイルを検索できます。 オープンスタイルの巨大なツリーを取り除きましょう。

  3. 次に重要なこと：

    ctrl + D-選択したテキストのコピーを検索します。 マルチ編集に最適です。 そして、重複を検索します。 特に大きなファイルと大きなフラグメントの場合。 マルチカーソルの場合は、Ctrlキーを押しながらマウスを使用します。

  4. ctrl + L-行全体を選択し、完全に削除できるようにします。 ctrl + Dでうまく機能します。

  5. 検索...まあ、おそらく最初はctrl + F-ファイル検索です。 次に重要なのは、ファイルセットのフォルダーでのctrl + shift + Fの検索です（フォルダーを右クリックして[フォルダーを検索...]を選択することで呼び出すことができます）。

  6. ctrl + shift +上/下 -行を上/下に移動します（スワップします）。 スタイルと変数を扱うのに便利です。

  7. キーボードショートカットに関する小さなチュートリアルです。 これらの組み合わせは、おそらくコンマで見たでしょう。 だから私は見た...そして今、私はそれらを使用する方法を知っています。 これを行うには、これらの組み合わせを順番に押します（一般的なmodキーを放すことはできません）

    便利なものを次に示します。

  8. ctrl + K、ctrl + 4-ネストが4を超えるすべてのブランチを非表示にします。矢印のアナログを折りたたみます。 （ ctrl + K、ctrl + J-すべてを展開します）

  9. ctrl + K、ctrl + B -SideBarを非表示/表示します。 （F12でうまく機能します）
