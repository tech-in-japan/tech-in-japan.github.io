SQL Server "Denali" CTPおよびPASS 2010会議からの多数の発表をダウンロードできます。





昨日、 PASS Summit 2010カンファレンスで、 Denaliというコードネームの次世代SQL Serverの予備バージョンが発表され、ダウンロード可能になりました。



SQL Serverの新しいバージョンの発表に加えて、他の多くの発表が行われました。
  • 負荷の高い並列コンピューティングと情報処理向けに設計されたSQL Server 2008 R2 Parallel Data Warehouseの新しいエディションを発表しました。
  • SQL Server 2008 R2 Parallel Data Warehouseの新しいエディションに基づいてHP Enterprise Data Warehouse Applianceプラットフォームを作成するためのHPとのパートナーシップを発表しました。
  • SQL Server用のMicrosoft Certified Master （MCM）認定プログラムが発表されました
  • 新しいアトランタクラウドサービスが発表されました。これにより、SQL Serverのインスタンスを集中的に監視し、構成、パフォーマンス、セキュリティの問題を特定し、クラウドを介してサーバーを管理できます。
SQL Server Denaliの新しいバージョンは、膨大な量の革新をもたらします。 それらをリストするには多くの時間がかかりますが、ここでは最も重要なもののみをリストします。



配備
  • Windows Server Coreのサポート。
  • 異なるサブネット上の SQL Serverクラスターフェールオーバークラスターの構成のサポート。
  • SQL Serverインスタンスの可用性を高めるために設計されたHADR （高可用性および災害復旧）と呼ばれる新しいソリューション。
管理
  • SQL Server Management Studio エディターの多数の改善 ：SQL Server 2005 SP2のデータベースに対するIntelliSenseサポート、TSQLのデバッグ時の多くの改善。
  • SQL Serverの1つのインスタンスから別のインスタンスにデータベースを簡単に移行できる新しい包含データベース機能。 このようなデータベースのユーザーはログインに関連付けられておらず、データベース転送を容易にするSQL Serverの特定のインスタンスへの他の接続はありません。
  • SQL Server起動オプションの管理は、 SQL Server構成マネージャーを介して利用できます。
開発
  • 全文検索の改善：IFilterメカニズムを使用した個々のドキュメントプロパティの検索。 検索の各部分と検索された単語のシーケンス間の最大単語数を指定できる新しいNEARオプション
  • UTF-16 補助文字照合のサポート。
  • 要求で返される行の数を制限するためのOFFSETおよびFETCHのサポート 。
  • 3つの新しいタイプの空間データが追加されました： CircularString 、 CompoundCurveおよびCurvePolygon ;
  • 地理データのOGCメソッド、地理インスタンスのOGCメソッド、地理インスタンスの 拡張メソッド 、 幾何インスタンスの 拡張メソッド、地理インスタンスの SQL MMメソッド 。
  • 空間データを操作するための静的メソッドが追加されました。 拡張静的地理メソッドと拡張静的ジオメトリメソッド 。
  • 整数値の自動シーケンスを作成できる新しいSEQUENCEコマンドのサポート。 この機能を使用すると、列で自動生成を使用せずに識別子を挿入できます。
安全性
  • 検索プロパティリスト、CREATE SERVER ROLE、ALTER ANY SERVER ROLEの新しい権限GRANT、REVOKE、DENYが追加されました。
  • カスタムサーバーロールのメカニズムが追加されました。このメカニズムは、新しいCREATE SERVER ROLE 、 ALTER SERVER ROLEおよびDROP SERVER ROLEコマンドを使用して制御されます 。
  • HASHBYTES関数は、 SHA2_256およびSHA2_512アルゴリズムをサポートするようになりました 。
  • 新しい包含データベースエンジンには、管理者が知る必要がある独自のセキュリティルールがあります。 必要な情報は、次の2つの記事に含まれています： 包含データベースの設計と実装、および包含 データベース に対する脅威 。
統合サービス
  • Integration Servicesには多くの変更が加えられています 。これについては、 こちらをご覧ください 。
変更されていないSQL Serverの部分



SQL Server Denaliの次の部分は、SQL Server 2008 R2以降変更されていません：Analysys Services、Data Quality Services、Reporting Services、Replication、Service Broker。
