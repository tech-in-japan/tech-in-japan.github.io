モバイルTVの仕組み

おそらく、多くの人々は、車で見ることができ、観光客がひどく固定されているソビエト連邦からの小さなモバイルTVを覚えています。 小型のポータブルテレビでは、スローモーションであってもあいまいな画像と干渉がほぼ保証されていましたが、成功しました。 それ以来、多くのことが変わりました-そして今、電話でテレビデータを受信するための3つのオプションがあります。



アナログ受信機
  • ソ連時代の標準技術。 格納式アンテナとアナログTVチューナーを備えた限られた数の電話が利用可能です。
  • バッテリーの寿命を大幅に消費します。
  • 強い干渉と歪み、運転中の悪化。




DVB-H
  • モバイルデバイスへのテレビ放送用の特別なネットワーク：私たち（国内）は、これを構築して運用する必要があります。 テクノロジー対応の電話モデルの選択は非常に限られています。
  • バッテリー寿命は最も経済的に消費されます。
  • 時速150キロメートルまでの速度で運転する場合、受信は良好です。
  • 放送用の固有周波数の割り当てが必要です。




ストリーミングTV
  • 標準的なオペレータネットワークを使用します。 カバレッジはネットワークに対応しています。 実装が簡単で、多くの電話でサポートされており、ロシアで使用できます。
  • 常時受信と複雑な後処理：多くのバッテリー寿命を消費します。
  • モビリティは非常に制限されています。動きが速いほど、速度は低下します（ネットワーク機能）。




アナログ受信機



ソ連の時代から、多くの小型化が行われ、テレビチューナーを内蔵した電話が登場しました。 小さなアンテナを引き伸ばすことで、一定のノイズ、あいまいなジオメトリ、想像を絶する色の画像を得ることができます。 また、サムスンにはアナログTVチューナーが組み込まれたモデルもありますが、モバイルデバイスの主要メーカーはこの機能を大規模に実装していません。



モバイルデバイスには、デジタル放送の標準であるDVB-Tがあります。 動きのある鮮明な画像が必要な場合は、別の信号で作業するだけでなく、デバイスに「搭載された」特別なソフトウェアハンドラーも必要です。



DVB-H-モバイル機器向けデジタル放送規格



モバイルの場合、いくつかの特別なデジタル放送規格が作成されています。 この多様性は、世界のさまざまな地域でモバイルTVの作業が開始された時点で、特定の周波数がすでに占有されていたためであり、ほとんどの標準は、独自のネットワークの構築を必要とし、特定の周波数スペクトルに結び付けられた独立したシステムとして開発されました。 ロシアでは、ヨーロッパと同様に、DVB-H（デジタルビデオ放送-ハンドヘルド）形式が選択され、アメリカと北米ではMediaFLO、ワンセグ（ISDB）は南アメリカと日本で使用され、中国と韓国では多くの隣人が選択を行いましたT-DMBを優先します。







その結果、デジタルモバイルTVをサポートする多くの電話は、国家標準のみを理解します。 これはすべて、チップのコストと、あらゆる場所で新しい技術を促進したい携帯電話メーカーの欲求の両方に影響します。

フォーマットには多くの共通点があります-モバイルデバイスに適合しています。 DVB-HはDVB-T（デジタルビデオ放送-地上波）標準の論理的な継続であり、追加機能をサポートし、自律電源を備えたポータブルモバイルデバイスに適合しています。



DVB-Hは、タイムスライシングテクノロジーを使用して、時間間隔で小さなパケットセットのIPデータグラムを量子化します。 これらのセットはそれぞれ、有効なデータの191ビットごとにリードソロモンコードで保護された64ビットの修正コードがある修正コードを含め、最大2メガビットのデータに到達できます。 受信モバイルデバイスは、ユーザーが選択したチャンネルに対応するデータセットが送信される時間間隔にのみ含まれます。 この短い間隔の間に、高速で送信されたデータは受信デバイスによってバッファリングされ、ダウンロードされたデータと再生中のストリーミングビデオの両方を含むことができます。 これにより、消費電力を大幅に削減し、モバイルデバイスの継続時間を延長できます。 動作条件に応じて、このバッテリーの節約は最大90％です。



2009年12月以来、モスクワの住民は2つの放送局からDVB-H信号を一度に受信することができました。 その後、これらのプロジェクトは商業化されず、首都を越えませんでした。 これはおそらく、ロシアの通信大手が加入者の電話にテレビ画像を配信する別の可能性を見たという事実によるものです。

ストリーミングビデオテクノロジーを使用したスト​​リーミングTVは、ロシアの通信事業者の大部分と世界中の数百の事業者を立ち上げました。 このサービスは主に大都市でのみ需要があります。 「ロシアの携帯電話を介したテレビサービスの需要は、このサービスをサポートする高価なモバイル機器を購入する必要性によって依然として制約されています」とPWCアナリストは言いました。 しかし、個々のソリューションにより、コストが3000ルーブル未満のデバイスや、「クラシック」になった40代および60代のノキアの携帯電話でサービスを提供できます。 ストリーミングTVユーザーの地理的最大数は、中央連邦地区、ヴォルガ連邦地区、および北西連邦地区に集中しています。 専門家の推定によると、2012年の初めまでにロシアのサービスユーザーの数は約200万人に達しました。



ストリーミングTVに使用される主なプロトコルはRTSPです。 これは、ほとんどの携帯電話の組み込みメディアプレーヤーでサポートされています。 SymbianおよびAndroid搭載のスマートフォン。 RTSPは、マルチメディアデータを扱うシステムで使用するために設計されたアプリケーションプロトコルであり、クライアントがサーバーからのデータフローをリモートで制御できるようにし、「開始」、「停止」などのコマンドを実行し、見つかったファイルへの時間アクセスを可能にしますネットワーク接続ストレージ。 3Gネットワ​​ークの平均ビットレートは約170kbpsで、プログラムを高品質で視聴できます。 また、EDGEネットワークでは、ビットレートが平均80kbpsに達するため、満足のいく品質でニュースチャンネルを見ることができます（たとえば、写真に急激な変化はありません）。 多くのストリーミングTVプラットフォームは、ビットレートを、サービスへのアクセス時に使用可能なデータ転送速度に適合させます。



iOSデバイスのユーザーには、HLSプロトコル（HTTPライブストリーミング）が使用されます。最近では、最新のMicrosoftオペレーティングシステムを搭載したデバイスでSmooth Streaming Technologyがサポートされています。 Smooth Streamingは、小さなコンテンツ（通常2秒）を配信し、それぞれが適切な時間を持ち、期待される品質レベルで再生することを確認するというシンプルだが強力なコンセプトを使用しています。 フラグメントがこれらの要件を満たしていない場合、次のフラグメントは低品質で配信されます。 逆に、条件が許す場合、後続のフラグメントの品質はより高いレベルになります。 このエンコードモードでは、複数のビデオビットレートを一度にブロードキャストできます。これにより、クライアントは、接続速度、プロセッサー能力、画面解像度に基づいて、最適化されたビデオストリームをHD-1080p品質までリアルタイムで取得できます。



DVB-HとストリーミングTVの比較



DVB-Hテクノロジーの主な問題は、ロシア全土に放送を展開するには膨大な投資が必要になることです。 また、ストリーミングTVの場合、カバレッジ境界は3G / 2Gネットワ​​ークゾーンと一致します。 もちろん、EDGEで動画を見るのは大画面で常に快適であるとは限りませんが、小さな画面では非常に可能です。



DVB-Hのもう1つの敵は、3台の電話から選択する必要があるという事実です。 かなり高価ですが、一流ではありません。 もちろん、DVB-Hデバイスが広く流通している場合はもっと多くありますが、現在、メーカーは大量にサポートされていない電話機能のコストを負担するインセンティブがありません。



モビリティDVB-Hは、モスクワで最大150 km / hの速度で信頼できる受信を示しました。 3GストリーミングTVサービスのモビリティにより、状況は異なります。 実際には、送信/受信速度は多くの要因に依存し、多くの場合、理論的な限界から非常に遠いことが判明しています。 2つの主な要因は、3G信号によるテレインカバレッジの品質と加入者の速度です。 理論的には、3Gでは、オペレーターによって割り当てられた別々の場所で最大2メガビット/秒の速度が利用可能である必要があります。 さらに、2 Mbit / sは、移動速度が3 km / h未満の座りがちなユーザーのみを対象としています。 加入者の速度により、情報転送の速度は常に数倍に低下します。



これは、3Gのアクセスシステムのまさにその原理の制限によるものです。 モバイルオブジェクトの場合、UMTSネットワークは、最大144 kbit / s（12-120 km / h）および最大384 kbit / s（3-12 km / h）の伝送速度を提供しますが、実際には約20 kbit / sしか得られません。 ダウンロードしたデータを表示するには2〜3分間で十分ですが、電車、地下鉄、公共交通機関、車は依然としてストリーミングTVの贅沢品です。



DVB-Hは、最高品質のサービスを提供する真のモバイルテクノロジーです。 ただし、加入者が専用の電話を購入する準備ができており、通信事業者がインフラストラクチャに数百万ドルを投資している場合のみです。 それらと他の両方は、そのような決定的なステップの準備ができていませんでした。 その結果、世界中で放送されるモバイルTVのシェアは低下し、ストリーミングは増加しています（予測によれば、2011年の40％から2013年の53％まで）。
