Broadcom Crystal HD：A to Z HDビデオのハードウェアアクセラレーション





多くの人が、HD（720p）H.264ビデオがネットブックで見やすいことを知っていると思います。 Intel Atomのパワーは、単にデコードするには不十分です。 私は、Acer Aspire Oneを出張や長期旅行のビデオプレーヤーとして使用していますが、これにも出くわしました。 この問題を解決するにはいくつかの方法があります。

  • 低解像度/ビットレートのトランスコード
  • ソフトウェアのトリックを使用する
  • ハードウェアアクセラレーションを使用する


最初の方法は遅すぎて屈辱的です（iPodなどのポータブルプレーヤーでは一般的だからです）。



2番目は普遍的ではなく、制限されています。 CoreAVCコーデックとCrystal Playerバッファーを再生前にインストールする2つの方法を認識しています（ HDTVのパフォーマンスを最大にするためのネットブックのセットアップを参照）。 原則として、これは720pのかなり適切なソリューションです。 ただし、2つの大きな欠点があります。100％のプロセッサ使用率と1080pの場合の完全な障害です。



三番目は私にとって最も「正しい」ようです。 Nvidia IONでネットブックをすぐに購入できます。CUDA計算や3Dゲームのプレイなどの追加機能を提供するユニバーサルソリューションです（ただし、小さな画面での楽しみは疑わしいです）。 残念ながら、Nvidia ION搭載のネットブックは最近登場したばかりで、まだ高価です。 また、「ショーオフ」のないネットブックをすでに購入している場合はどうすればよいでしょうか？



抜け道があります！ Broadcom Crystal HDと呼ばれ、EEE PCまたはAspire Oneで「高速化」することは非常に現実的で安価です。



Broadcom Corporationは、1年前にMini PCI Expressインターフェイスを備えたCrystal HDカードをリリースしました。これは、H.264およびMPEG 2.4（およびFlashと噂されています）のハードウェアアクセラレーション再生を提供します。 美しさは、マイクロ回路を空きPCIeネットブックスロットに挿入できることです。 次に、この奇跡をどのように得るか、どのように構成し、どのように機能するかについて説明します。



これらのすべてについて以下でさらに説明しますが、非常に曲がった手がある場合、お金とネットブック全体の両方の損失につながる可能性があります。



私の例でお話します。 私の構成：

Acer Aspire One AOD 250 ：

Intel AtomシングルコアN270（1.60 GHz / 512 KB / FSB 533）、10.1インチSD（1024x600）、2 GB RAM

Windows 7 + Xubuntu 10.04（ベータ）



何が必要ですか


  • 無料のミニPCIeスロットの存在
  • PayPalで約30ドル、または既に購入したCrystal HD
  • 購入する場合-2か月待つ意思*
  • Windows用の特定のソフトウェア
  • 忍耐と強い神経（もちろん）


*ロシアの郵便が引き続き機能する場合



ウィキペディアからミニPCIeへのリンク 。



通常、Yotaのない安価なネットブックには3Gカード用の空きスロットがあり、必要なものです。 私の場合、RAMを変更するためのコンパートメントの左側の背面にありました。 古いAOD150にも同様のものがあることは確かですが、そこにはコネクタがありません（つまり、ボードを直接貼り付ける場所）。 スロットはないが、本当にマイクロサーキットを挿入したい場合は、ネットブックを分解し、wifiカード（ほとんどの場合mini PCIeを使用）をCrystal HDに交換し、USB wifiドングルを購入してワイヤレスインターネットの問題を解決します。



私はeBayでカードを買いました 。 一部のラジオ市場で販売されている可能性があります。 実行する時間はありませんでしたが、PayPalにはお金がありました。



ソフトウェアのリストをさらに説明します。



Crystal HDを入手


eBayでチップを購入します。 例えばここに 。 私は2か月前に購入しましたが、配達を含めて23ドルでした。 今、明らかに、彼らの供給は終わりに近づいており、彼らははるかに高価です。 70010（旧）、70012（私のもののような）、70015（最新で最も機能が優れた）の3種類のモデルがあることに注意してください。 70012はかなりうまく機能します。

私たちはそれが配達されるのを約2ヶ月待ちます（彼らは私をシンガポールから送ってくれました）。



鉄の仕事をする


喜んでCrystal HDをネットブックに挿入したとき、Windows 7もXubuntuもそれを見なかったため、最初は激しく中断しました。 チーム



lspci | grep Broadcom







バターとシシを見せた。 動揺する必要はありません、友達！ おそらく、この場合、超小型回路に税関で何かが照射されたのではなく、飛行機の静電気が照射されたのではなく、ネットブックの古いBIOSバージョンでは汚いのです。 直感的かつランダムに、AOD250のBIOSを1.06から1.27に再フラッシュしました。 奇跡！ マップが決定しました



lspci | grep Broadcom







発行



Multimedia controller: Broadcom Corporation BCM70012 Video Decoder [Crystal HD] (rev 01)







Windows 7は、起動時にマルチメディアコントローラーにドライバーをインストールしようとしましたが、もちろん何もできませんでした。 必要なソフトウェアのリストに移動します。

  • Broadcom Webサイトのドライバー
  • MKVをプレイするための方法-Haali Media Splitter
  • サウンド再生用-AC3 フィルターおよびCoreAAC
  • Windows 7をお持ちの場合-Windows 7 Preferred Filter Tweaker
  • コーデック診断用-GraphStudio


CoreAVC + Haali Matroska（Media）Splitter（私のような）を持っている人、CCCP、K-Liteコーデックパックなどを持っている人 -すべてをルートに持って行きます。

ダウンロードして、ドライバー（32ビット）をインストールします。

この順序であることに注意してください。



コーデックを扱う


Haali Media Splitter（MKVコンテナ用）を配置します。 AC3Filter（ドルビーAC3用）とCoreAAC（AAC用）を配置します。



もちろん、その後の幸運な人にとっては、すべてがすぐに機能するかもしれませんが、私は明らかにそれらの中にはいませんでした。 最初の720p mkv（h.264）Windows Media Player（およびMedia Player Classic Home Cinemaも）を開いたとき、よく知られているブレーキと100％CPUを手に入れました。 こんにちは、 コーデック地獄 。

GraphStudioでこのファイルを開いて（[ファイル]-> [メディアファイルの追加]）、h.264ビデオストリームが組み込みのWindows 7コーデックによってまだデコードされていることがわかりました。 BroadcomのDirectShowフィルターが立っていました。 これを確認するには、リストの「Broadcom Video Decoder」を見つけます。グラフ->フィルターの挿入...



もちろん、再起動しても何も変わりませんでした。 Windows 7 Preferred Filter Tweakerが役に立ちます！

起動することにより、h.264とMPEG2をMicrosoftからBroadcomに切り替えます。 どうやら、将来、このツールの開発者は、Xvid、VC-1、およびDivx for Broadcomの宣言されたサポートを追加する予定ですが、執筆時点ではそうではありませんでした。 [適用]をクリックします。次のウィンドウで、どこにでもドーズを配置します（後で干渉しないように、すべてのMicrosoftコーデックをオフにします）。 出来上がり！ GraphEditは、Broadcomがh.264、サウンド-CoreAAC、Windows Media Playerなどをデコードし、ビデオをサウンド付きで自信を持って表示することを示しています。







おそらく、Windows XPを使用している人には、このような問題は発生しないでしょう。



BroadcomでXvidをプレイすることはできませんでした。GraphEditで「ブロック図」を収集することもできました。 たぶん私は何かを逃し、あなたは成功するでしょう。 正当な理由により、Windows 7 Preferred Filter Tweakerでは、開発者はBroadcomをXvidデコーダーとして追加しませんでした。 xvid.orgから通常のXvidをインストールすることに満足しました。



Xubuntu用にCrystal HDをセットアップすると、別のトピックが表示されます...

「優れた」再​​生品質とは、ビデオを音声、ラグやジャークがないこと、色飽和度、通常の通常の映画表示と同期させることを意味しました。 標準の3セルバッテリーの動作時間は、720pの場合は2時間、1080pの場合は1時間55分でした。 Crystal HDは非常に熱くなっています（熱を放散する特別な熱ステッカーが付いているだけではありません）。
