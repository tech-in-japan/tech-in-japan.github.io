私はとても怠け者なので、WindowsではなくGNU / Linuxを使用しています。

ソフトウェアのインストール方法は、トラフィック消費の観点から不経済になる可能性があることを警告します。

あなたの注意を引くものはすべて私によって検証されましたが、 サタンがコマンドの実行を呼び出した後に発生する可能性のある問題について、私は一切責任を負いません。

新しくインストールされたKubuntuがあります。

プログラムをどのように配置しますか？ コンソールを通して、他の人を通して、コンソールを通して。 結局、コマンドをコピーしてEnterキーを押すだけで済みます。

斜体とコールアウトは、コンソールに記述する必要があることを意味します。


太字で入力すると、起動時に入力する必要があります（Alt + F2）。

プログラムの名前は赤でマークされています 。



開始するには、[追加と削除] / [プログラム]メニューに移動します。 「ソフトウェアソース」ボタンをクリックして、どこでもチェックマークを付けます。 2番目のタブで、CD-ROMを取り出し、他の2つのソースにマークを付けます（ただし、プログラムはWeb上で新鮮です）

次のようになります。







そして、残りのリポジトリ（これらはソフトウェアのソースです）は、次のようにファイルに直接追加されます。

Alt + F2を押して、次を入力します。

kdesu kate /etc/apt/sources.list







ここで、「追加/削除」に戻り、検索メニューに「 やくあけ 」と入力し、ボックスにチェックマークを付け、「変更を制御」（翻訳）

集中的な幸福のために何が必要でしたか？ もちろん、 ミラー、請求書、魔法の粉は、以前のシステムのデータにアクセスします。 はい、2番目のシステムとしてWindows XPがあります。当然、ntfsパーティションがあります。 それらからコピーするだけでなく、書き込みもしたいです。 ntfsに書き込むためのプログラムも組み込まれています-ntfs-3gと呼ばれます

番号1を使用する方法は、fstabファイルに入り、パラメーターを書き込むことです。 私は怠け者です。

番号2を使用する方法は、ntfs-3gのフロントエンドを設定することです。

sudo aptitude install ntfs-config


以下を開始します。

sudo ntfs-config


そして、2つのドーズを置きます：







ハードドライブ上のデータが、たとえば私のような他のアプリケーションで使用されていることがあります。音楽をオフにするのを忘れていました。







それでは、レイアウトをCtrl + Shiftに切り替えましょう。

ファイルを開きます。

kdesu kate /etc/X11/xorg.conf



そして、「InputDevice」セクションを次の状態に編集します。

セクション「InputDevice」

識別子「汎用キーボード」

ドライバー「kbd」

オプション「CoreKeyboard」

オプション「XkbRules」「xorg」

オプション「XkbModel」「pc105」

オプション "XkbLayout" "us、ru"

オプション "XkbVariant" "、winkeys"

オプション "XkbOptions" "grp：ctrl_shift_toggle"

終了セクション

Ktorrentはアイドルではなかったので、たくさんの映画や音楽を手に入れました。 Amarokは、大規模な音楽コレクションに適しています。 より速いものが必要な場合-それから大胆です。

Amarokは、起動時にmp3コーデックをインストールするよう丁寧にお願いします。 私の直感では、ロシアの音楽のほとんどはクレタスによって表示されます。 さて、たとえあなたが食べられたとしても、2つの選択肢があります。 ここにあります。

最初の方法は、音楽エンコードプログラムeasytagを変換し、異端のcp-1251に再び来るすべての音楽を変換することです。 そしてすべては、Amarokの作成者がutfが世界を支配すると言ったからです（バージョン1.4以降）。 私はそれが好きではありません。選択の自由が取り除かれたことは非常に残念です。

音楽は非常に頻繁に私に届き、プレーヤーと電話は一般的にエンコーディングに対応しているため、2番目の方法を選択します。cp-1251を理解するようにAmarokに教えます。 これを行うには、更新されたライブラリの取得元をaptitudeに指示する必要があります。
