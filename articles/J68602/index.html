Nokia 5800およびN97のトリック

記事「 Nokia 5800およびN97の微調整に関する究極のガイド 」 の翻訳

一般的なヒント

  1. ダイヤルモードでは、 ＃を 2秒間押すと、「サイレント」モードと「通常」モードが切り替わります。 0を押し続けるとWebブラウザーが開きます。 1を押し続けると、ボイスメールが開きます。
  2. 右上隅の前面カメラの目の下に、「ミュージックプレーヤー」、「画像およびビデオクリップギャラリー」、「インタラクティブエクスチェンジ」、「ビデオセンター」、およびWebブックマークをすばやく起動するためのボタンがあります。
  3. 緑色の通話ボタンを押し続けると、音声コマンドモードが有効になります。
  4. Outlookで許可されているのと同じ方法でSMSメッセージをグループ化します。 これを行うには、メニュー項目を表示します：メッセージ->オプション->オプション->その他->ディスプレイ。 グループのメッセージ->はい。
  5. メッセージと不在着信に関する通知を設定するには、メニュー項目を確認します：[設定]-> [電話]-> [インジケータライト]
  6. 画面のタップが正確でない場合は、タッチスクリーンを調整する必要があります-オプション->電話->タッチ入力->キャリバー。 センサー スクリーン
  7. 制御機能を有効にする：
    • 電話を水平または垂直に回します。
    • 着信通話で通話をオフにするには、電話を裏向きにします。
    • アラームをオフにするには、電話を裏向きにします。
  8. テーマおよび変換効果の3D効果をオン/オフするには（電話の速度が大幅に低下します）-オプション->個人->テーマ->標準->機能->テーマ効果->オン/オフ
  9. クイック終了キー：中央の白いボタンを押したままにして、終了したいプログラムのアイコンを押したままにし、ドロップダウンメニューで開く/終了を選択します（これらの目的には便利なtaskmanプログラムを使用することをお勧めします）
  10. 手動入力を使用する場合、ダッシュはスペースを左から右に挿入し、文字を右から左に削除します。
  11. メニュー->オプション->整理でメニューを整理します。 再配置するアプリケーションの場所をドラッグアンドドロップできます。
  12. Nokia 5800ダイヤル*＃0000＃のファームウェアを更新するには、インストールされているソフトウェアに関する情報が表示されます。 次に、メニューから「更新の確認」を選択します。 最新の更新プログラムは、ノキアから携帯電話に送信されます。更新プログラムをダウンロードしてインストールします（GPRSまたはWi-Fi接続が必要です）
  13. 時計をクリックしてメニューを開き、アラームを設定します。
  14. 通常モードのインジケーターまたは日付を押​​して、ドロップダウンメニューでモードを選択します。これは、「カレンダーを開く」項目のないS60を搭載したすべての電話機でシャットダウンボタンを押した場合とほぼ同じです。
  15. 最初の段落と同じですが、言い換えれば。
  16. 作業リストまたはカレンダーの項目をクリックすると、すぐにこの項目に移動します
  17. 左上隅（バッテリーインジケータの近く）にあるアクティビティアイコンをクリックできます（Nokia 5800では、これらのアイコンは右上隅に表示されます-約翻訳者）
  18. 緑色の通話ボタンを長押しすると、ボイスメールが開きます
  19. 中央のボタンを長押しすると（メニューボタン（一部のS60では陰陽ボタン）を押すのに似ています）、すべてのアクティブなアプリケーションのメニューが開きます。
  20. 電話データのバックアップコピーを作成するには、[アプリケーション]-> [ファイルマネージャー]-> [バックアップ]を選択します。 毎日/毎月の自動バックアップを設定できます。
  21. インターネットを閲覧しているときに、ページを拡大する必要がある場合は、拡大したい領域をダブルクリックします。 通常、ページを目的の場所にドラッグするだけで十分ですが、スクロールバーを使用することもできます。 この絶好の機会は、ページサイズのアイデアを提供します。
  22. 通常モードとサイレントモードを切り替えるには、ダイヤルモードで＃を押し続けます。
  23. 開いているすべてのアプリケーションのリストを表示するには、メニューボタンを押し続けます。
  24. 手書きトレーニング-オプション->電話->タッチ入力->手書きトレーニング
  25. 電話のハードリセットを行うには、赤、緑のボタンとカメラの電源ボタンを同時に押して電話をオフにします（Nokia 5800-ノートトランスレーターで成功しませんでした）。
  26. Red Oneを入手してRed Nokia BH-503ステレオBluetoothと組み合わせて、素晴らしいワイヤレスソニックエクスペリエンスを手に入れましょう。

省エネのヒント

  1. ワイヤレスネットワークとBluetoothを使用していない場合、それらの連続スキャンを無効にします。
  2. また、「デュアルモード」ではなくGSMモードを選択して3Gを使用しないでください-設定->通信->ネットワーク->ネットワークモード-> GSM
  3. ディスプレイの輝度を下げる
  4. 使用しない場合は、常にアプリケーションを閉じてください。

メモリのヒント

  1. メモリが30 MB未満の場合は、電話を再起動します。ソフトリブート（ダイヤルモードダイヤル5 **（Nokia 5800 —ノートトランスレーターで成功しませんでした））を実行するか、電話の電源オフボタンを使用して再起動します。
  2. Handy Taskmanを使用して、アプリケーションを閉じるか、未使用のアプリケーションを「殺す」

カメラのヒント

  1. 自動ではなくISO感度を低く設定すると、ノイズを低減できます
  2. 画像を表示しながら、iPhoneのように左右にスクロールできます。

映像

Nokia 5800に最適な品質でビデオを変換するには、次のようにmp4形式を使用する必要があります（すべて16：9形式）。
  1. Xvid @ 678kbps解像度480×272
  2. Xvid @ 794kpbs解像度640×352
  3. H.264 @ 512kbps 400×224解像度
  4. H.264 @ 550kbps解像度400×224（推奨）。

ミュージック

  1. コンピューターからメモリーカードにコピーした後に曲が見つからない場合-音楽プレーヤーのライブラリを更新する[オプション]-> [音楽ライブラリを更新する]

用途

  1. テーマは常に電話カードにインストールしてください。 一部のテーマ（たとえば、クリスマスツリー）は、携帯電話のメモリからでは機能しないためです。
  2. アプリケーションを正しく削除する-オプション->アプリケーションマネージャー->アプリケーションのインストール->削除するアプリケーションを選択する
  3. テーマで3D効果を有効にする-オプション->個人->テーマ->一般->機能->テーマ効果->オン/オフ
