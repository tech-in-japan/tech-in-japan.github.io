SQL Server 2008 R2 SP1、SQL Server Denaliの追加ツールがリリースされました





SQL Server 2008 R2の最初のサービスパックの最終バージョンはダウンロード可能です。 SP1パッケージには、以前にリリースされたサーバーの6つの累積的な更新プログラムが含まれており、さらにいくつかの新機能が提供されています。



さまざまな言語およびエディションのフィックスパックは、 次のリンク （エクスプレスエディションのバージョン ）からダウンロードできます。



さらに、 CTP3の次の予備バージョンで最近リリースされた次世代のSQL Server Denaliデータベースサーバー用のツールがダウンロードできます 。 使用可能なツールには次のものがあります。
  • コード名「Denali」CTP3のMicrosoft SQL Serverの製品ドキュメント（ リンク ）;

  • Express Core CTP 3（ リンク ）;

  • PowerPivot for Microsoft Excel CTP 3（ リンク ）;

  • セマンティック言語統計CTP 3（ リンク ）;

  • レポートビルダーCTP 3（ リンク ）;

  • SharePoint Technologies CTP 3用Reporting Servicesアドイン（ リンク ）。

さらに、 Juneauというコードネーム （SQL Server Management Studioを置き換える）の統合データベース開発環境の新しいバージョンは、Web Platform Installerツールから入手できます。
