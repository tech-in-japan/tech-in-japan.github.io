Tシャツに写真を配置する





長い間、多くの男性が「世界一の父親」という碑文の付いたTシャツを購入しています。 個々のTシャツは新しいものではありませんが、自分で作成できるようになりました。 現在、Tシャツに配置できる画像は通常、コンピューターに保存されています。 デジタル写真を使用すると、熱転写シートに画像を印刷して、Tシャツに適用できます。

必要条件



Tシャツに写真を配置するには、次のものが必要です。



  • Windows Vistaを実行しているコンピューター。
  • Windows Live Photo Galleryなどの写真管理および編集ソフトウェア。 Windows Marketplace Webサイト（英語）を使用して、このようなソフトウェアを検索できます。 「写真編集」というフレーズを検索します。
  • カラーインクジェットプリンター。
  • デジタル写真（コンピューター上）。
  • コットンTシャツ。
  • 熱転写紙。
  • Tシャツをアイロンするアイロンと平らな表面。
