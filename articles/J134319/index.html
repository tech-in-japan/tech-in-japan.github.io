STALKER社は消滅します



おそらく、多くの行商人は、ウクライナの企業GSC Game Worldが2007年から制作している世界的に有名なSTALKERシリーズのゲームに精通しています。 STALKERシリーズの最初の部分である「チェルノブイリの影」の公式発行は、CISで95万部、西部で70万部に達し、東ヨーロッパの企業にとって非常に高い成果でした。 多くの出版物やゲーマーは、ゲームを「カルト」と見なしました。



今日、GSC Game Worldの創設者兼CEOであるセルゲイ・グリゴロヴィッチは、労働者団体の会議で会社の閉鎖を発表し、彼の決定を個人的な動機で説明したことが知られました。



社内の情報筋によると、作業チームの一部（約50人が社内で働いている）が、会社の閉鎖とグリゴロビッチの不満を、2012年にリリースされる予定のゲームSTALKER 2の開発に結び付け、一部は企業の1つに興味があると強調しました情報源によると、約2か月前に始まった権力構造は、これが彼の仮定に過ぎないという予約をしました。







「クリアスカイ」と「プリピャチの呼び声」と呼ばれる「チェルノブイリの影」の前編と続編は、それぞれシリーズのファンがSTALKER 2に大きな期待を持っていたため、最初の部分ほど成功しなかったことを思い出す価値があります。 CISの領土で作成された最も成功したゲームの1つの運命について-開発は最終的に停止されるのか、それともゲームをさらに開発するプロジェクトに参加する会社がありますか？



UPD： 1C-Softklab Ukraine Sergey Galenkinのマーケティングディレクターのブログにあるhabrazer Yakiv_Gluckの投稿から、GSCの詳細を確認できます 。 特に、侵入者の攻撃のバージョンは彼によって確認されていません。
