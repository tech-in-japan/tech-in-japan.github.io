Mimo 720S USBの概要

MIMOモニターは最近、「ミニUSBモニター」シリーズから新しいモニターを導入しました。 このモデルは、MIMO 720-S「タッチスクリーンスライダー」と呼ばれます。 素敵なデザイン、軽量、タッチスクリーン機能の存在、USBポートのみからの電力は、注意すべきパラメーターの良いセットです。 Mimoミニモニターは、コンパクトなラップトップのデスクトップ機能を拡張するのに役立ち、小型で軽量であるためラップトップバッグに簡単に収まります。





技術仕様Mimo 720S



画面サイズ：7 "

解像度：800 x 480

明るさ：350 cd / m2

コントラスト：400：1

ポート：USB 2.0

寸法：折り畳まれた：7 "x 5.25" x 1 "（厚さ）、オープン：7" x 4.5 "x 5.25"

機能：タッチスクリーン技術、90度ディスプレイヒンジ、ビルトインスタンド、CE FCC RoHS



ミニディスプレイの外観

Mimo 720Sミニディスプレイは、最小限のコンポーネントセット（USBモニター、USB電源ケーブル、取扱説明書、ソフトウェアディスク）を備えた明るくコンパクトなパッケージで提供されます。













まず、デバイスの設計に注意を払ってください。 まず、Mimoには、最小限のスペースを使用しながら、フォルダーの形に折り畳むという興味深い便利な機能があります。 第二に、モニターは、水平位置と垂直位置の両方で画像を表示できます。 第三に、デバイスの重量は約400 gであるため、ラップトップバッグに簡単に入れて、そこにあることを忘れることができます。



Mimoディスプレイの回転は90度ですが、画像の切り替えはインストールされたドライバーを使用して行う必要がありますが、自動方向変更は提供されません。











USBモニターの本体は、高品質のマットプラスチックでできています。 ケースには、電源、輝度を上げる、輝度を下げるという3つの機械的なボタンがあります。



輝度は広範囲にわたって調整可能で、300 cd / m2に達します。 背面にはUSBコネクタがあり、会社のロゴが付いたゴム製のフラップの下に隠されています。 付属のUSBケーブルには、一部の外付けHDDと同様に、コンピューターに接続するための2つのコネクターがあります。 実際には、これを使用するだけで十分です。 USBケーブルの長さは140 cmで、ラップトップまたはホームPCに接続するのに十分です。







ソフトウェア

ソフトウェアとドライバーに関するいくつかの言葉。 Mimo 720Sを接続する場合、モニター用とタッチスクリーン機能用の2つのドライバーが必要です。 インストールの問題は発生しませんでした。 インストール後、3つのアイコンがトレイに表示されます：DisplayLink（モニターの画像の回転とメインモニターの拡張または複製機能の調整に役立ちます）およびタッチスクリーン機能の設定と操作用の2つのアイコン（設定と調整用、指でクリックしたときのマウスボタンの切り替え用）









設定をさらに詳しく調べた後、メインモニターのマウスカーソルを制御するまで、多くの追加のタッチスクリーン機能を構成できます。 後者の場合、ラップトップでタッチパッドを使用するときと同じレベルの利便性を構成できませんでした。 多くの場合、単にマウスを動かすのではなく、画面領域を選択します。 より正確なキャリブレーションが必要になる場合があります。



MACユーザーは、タッチスクリーン機能を使用するには、追加の30ドルの有料ドライバーをダウンロードする必要があります。



利便性と画質

画質は優れていますが、傑出したものを誇ることはできません。 同時に、明るさの調整範囲がとても気に入りました。 直射日光がモニターに当たると、高輝度（8レベル）で快適にモニターを使用できます。 テキストは明確で読みやすいです。 画面に光沢がないため、グレアは見られません。 これはプラスだと思います。 必要に応じて、1台のコンピューターに複数のモニターを接続できますが、コンピューターのハードウェアの要件は、1台のディスプレイの1.2GHzおよび512MB RAMから、6台の1.6GHzデュアルコアおよび1GB RAMに増加します。 、極端な角度での明示的な色反転は観察されません。







結論

モニターは、ラップトップ画面の追加拡張として最適です。 その上に、icqクライアント、スカイプ、Photoshopパレットとツール、twitterクライアント、その他のユーティリティを配置できます。 Outlookとブラウザの両方を使用できますが、インターネット上のほとんどのページは800ピクセルを超える解像度用に設計されているため、利便性は低下します。



洗練されたデザイン、軽量で1本のUSBケーブルのみによる電力-小さな画面サイズのラップトップで使用するための優れた品質セット。 必要に応じて、Mimo 720Sをデジタルフレームとして使用できます。
