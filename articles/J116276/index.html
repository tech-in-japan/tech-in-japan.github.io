音声付きHDMIを介してPCをテレビに接続します

問題



私は、2つのDVI出力のみを備えたNVIDIAグラフィックカードと、光（TOSLINK）出力を備えたサウンドカードを備えたコンピューターの所有者です。 最近、私は大型の液晶テレビを購入し、ホームシアターを組織したいと考えました。 ただし、妻は、まず5.1の列を配置する場所がないと言い、次に、良い列をとると、列が高価になると言いました。 したがって、私は問題に夢中になりました。どうすればコンピュータをテレビに接続して、画像に加えてオーディオカードから音を再生することができますか。 明らかに、次の計画のデバイスが必要でした：







そして、幸いなことに、それが見つかりました！



解決策



たとえば、 Amazonで約30ドルで購入できます。 DVI +オーディオ（SPDIF＆Toslink）to HDMI変換アダプターと呼ばれます。 それは2つのバージョンで中国で作られています-コンパクトではありません。 私はコンパクトなものを手に入れました：











宣言された仕様（中国語のスペルは保持されます）：
  • 最大1080pのDVI入力
  • HDMI 1.1およびHDCP 1.1をサポート
  • オーディオ入力DTS-HD / Dolby-trueHD / LPCM 7.1 / DTS / Dolby-AC3 / DSD
  • HDMI解像度24/50 / 60fs / 1080p / 1080i / 720p / 576p / 576i / 480p / 480i
  • 最大伝送帯域幅5.1 gbps
  • 入力ビデオ信号0.5〜1.5ボルトpp
  • 出力DDC信号5ボルトpp
  • 出力ケーブル距離<= 20m AWG26 HDMI 1.3標準ケーブル
  • 最大動作電流480mA
  • 電源アダプターの形式、入力：AC（50HZ、60HZ）100V-240V; 出力：DC5V / 0.8A
  • 寸法（LxWxH）85x70x33 mm
  • 重さ185g
DVIは入力であり、2つのS / PDIFの1つ-同軸または光学、HDMI出力へ。 HDMI側からのオーディオ出力は、理由が不明であり、サウンドがおそらく重複しています。 オン/オフスイッチの目的も不明です。どの位置でもデバイスが機能し、電源コードを抜くだけでデバイスをオフにできます。 2つのLEDが点灯します。入力で赤、出力で青です。 このデバイスには、アメリカのコンセント用の5ボルト電源アダプターが付属しています。

ドキュメントには、すべての画像形式（480i、480p、720i、720p、1080i、1080p）がサポートされていることが記載されています。 そして、本当に、カンニングをしないでください。 次の構成で作業を確認しました。
  • Windows 7 x64 SP1
  • ビデオNVIDIA GeForce GTX 470（2xDVI）
  • Creative X-FI Titanium Audio
  • NONAMEケーブル：光オーディオ2メートル、DVI-DVI 1メートル、HDMI-HDMI 3メートル-各100ルーブル
Windowsは、アダプターを「Display Device on：DVI」と定義しています。







許可のリストでは、最初に使用可能な最高値ではなく、最大1600x1200、および1280x1024が推奨されます。 1280x720、つまり 720p、それはうまく動作しますが、私はそれがさらに涼しくなります。 これを行うには、NVIDIAコントロールパネルに1920x1080を手動で追加します。







60Hz（より正確には59）のリフレッシュレートでは、問題が発生し、テレビが画面上で点滅し、互換性のない信号の受信について時々書き込みました。 しかし、50Hzは問題なく巻き上げられ、安定して動作します。 動作中、アダプターはホームルーターやモデムのようにわずかに熱くなります。 金属ケースは効果的に熱を放散します。 私は数時間映画を見ましたが、問題は生じませんでした。



HDCPサポートに関しては、NVIDIAドライバーは異なる意見を持っています-それはありません、それだけです。 私はHDCPを必要とせず、理解しませんでした。



最初は音がしません。 ボリューム設定またはサウンドカードの専用GUIでS / PDIFアウトを有効にする必要があります。 なぜなら テレビにサラウンドサウンドが含まれている可能性は低いため、これ以上調整する必要はありません。 品質については、PCに接続された優れたヘッドフォンと同じように主観的に聞こえると言えます。 中断やその他のトラブルは認められませんでした。



最大のタスクは、アダプターをUbuntuで動作させることですが、これは調査のための別のトピックです:)



まとめ



中国語で説明されているデバイスは正常に機能し、それほど要求の厳しいユーザーから苦情を引き起こすことはありません。 誰もが幸せです-私と私の妻の両方。
