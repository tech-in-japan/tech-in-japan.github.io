LibreOffice Mathを使用して技術科目に関する講義を簡単に記録する方法

問題：

私は定期的に地下鉄で仕事に行き、若い人たち、おそらく学生たちが、Macbook、タブレット、iPhone、iPadを持っているのを見かけます。 まあ、それは、彼らが最新の生活をしていることは明らかです-そこのツイッター、確かにフェイスブック、インスタグラム。 たぶん、 コースラの講義を見ているかもしれません。 これらすべてのことで、学生は手書きの講義ノートをノートで読みます-略語、表、下線付きの用語の定義などを使って そして、これらの手書きのノートブックは私に認知的不協和を引き起こします-さて、2012年にラップトップで講義を書くことが不可能なのはなぜですか？



結局のところ、ソリューションは長い間知られています-OpenOffice / LibreOffice。

あなたがヒューマニストなら、すべてはここで単に非現実的です-あなたの講義には、おそらく式もグラフも図面もテキストも表もありません。 講義に座って、教師の話を聞き、Wordにテキストを入力します。 ほとんどの人の印刷速度は、手書きの速度よりも高速です-これは事実です。 さらに、オートコンプリートなどの機能により印刷速度が大幅に向上します。「innovation」という言葉がすでにテキストに何度か登場している場合は、「inno」を書き始めるとすぐに、OpenOffice / LibreOfficeがそれを希望するものに追加します。 Enterキーを押して、単語を入力します。



あなたがエンジニアであり、数式を扱うのであれば、長い間、無料のOpenOffice.orgまたはLibreOfficeの現在の無料GPLバージョンを思いつきました。 これらのパッケージの特徴は、数式の信じられないほど便利なエディタがあり、構文上、TeXに非常に似ていることです。 MS Officeには、数式を入力するためのツールもありますが、私にとっては便利ではありません。そこでマウスを突く必要があります。

LibreOfficeに組み込まれた数式エディターでは、次のようになります。

lint from vec H d vec l = int from S vec jd vec S + int from S {{partial vec D} over {partial t}} d vec S







書き込みには15秒かかります。 構造が似ている構造がすでに講義に含まれている場合は、単にコピーし、貼り付けて変更を編集します。 さらに時間がかかりません。



私の意見では、記録方法は非常に明確で論理的です-結局、intは積分、fromは下限、toは上限、lintは表面積分であることは明らかです。 もちろん、マークアップ言語は解釈されます。つまり、入力するとすぐに数式が表示されます。 intを入力-積分の符号が表示され、...から...にintを入力-積分の限界が表示されます。
