Evernote API-Habrの記事の見出しを並べ替え、記事にタグをタグ付けします

こんにちは、Habrasociety様！



この短いトピックでは、Evernote APIの最初の知り合いについてお話したいと思います。

だから、それはすべて私が賢明に過ごすことにした私が仕事で十分な自由時間を持っていたとき、約6ヶ月前に始まりました。 基本的に、私はもちろん、Ebranoteの言及に出会ったHabrahabrに関する記事を読みました。 このサービスが気に入ったので、仕事で読む時間がなかったメモや記事へのリンクを保存するために積極的に使用し始めました。 お気に入りのChromeにEvernote Web Clipperプラグインをインストールした後、Evernoteの記事へのリンクを送信するのは非常に簡単で迅速でした。 そして、いくつかの迷惑がありました...



時間が経つにつれて、「後で」読む記事が増え、時間が短くなり、タグを付けず、興味のあるトピックに関する記事を一般的なノートブックで見つけたため、不便を感じ始めました。恐ろしいメモのリストが問題になっています。

ほぼ同時期に、EvernoteはそのAPIを開き、このAPI自体を使用することで、少なくともメモの一般的なヒープと区別することができるようになりました。 もちろん、私の最初の欲求は、Habrからのすべてのメモを処理し、どこにでもラベルを付けるアプリケーションを作成することでした。 数晩過ごした後、目的のツールを入手しました。今では、Evernote Web Clipperを使用して作成されたメモを見つけるのは難しくありません。 プログラムは「Article Title / Blog Name / Habrahabr」という形式のメモのタイトルを解析し、このメモにタグ「！Habrahabr」と「Blog Name」をタグ付けします。



おそらく、Habrasocietyの誰かがこの小さなコンソールアプリケーションの恩恵を受けるでしょう。
