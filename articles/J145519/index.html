Virtual Treeview V5.0.0 RC1コンポーネントの待望のアップデート

多くのDelphi開発者は、このすばらしい無料のコンポーネントを知っており、使用してツリー 、 リスト、およびテーブルを表示します 。

The Bat！Users 手紙のリストを見て、彼に毎日会いましょう。

このコンポーネントは、ノードでの作業の高速性と設計のセットアップの幅広い可能性によって区別され、ユニコードをサポートします。



コンポーネント4.8.7の最新バージョンは2010年10月にリリースされましたが、更新プログラムへの期待は低かったです。

しかし、それでもアップデートは公開されました！

Delphi XE2のユーザーは喜ぶようになり、64ビットシステムのユーザーも喜ぶようになりました。





では、新しいバージョンで何が変わったのでしょうか？
  • Cバージョン5.0は、Delphi 5/6およびC ++ Builder 5/6をサポートしなくなりました
  • Delphi XE2および64ビットコンパイラのサポートが追加されました。
  • Ctrl + Aですべての要素が選択されるようになりました。
  • 編集モードでは、Tabキーを押すと、次のノードにフォーカスが置かれると同時に、Windows 7エクスプローラーのように編集モードになります。
  • ヒントのテーマサポートが追加されました。
  • 「SelectionTextColor」色の新しい設定を追加しました。
  • Windows 2000のサポートを終了しました。
  • フラットスクロールバーのサポートが削除されました。
  • 廃止されたThemeServicesによるRAD Studio XE2の警告を修正しました。
  • tsUseExplorerThemeモードを使用する場合のWindows7 / Vistaのスタイルでの選択領域の描画を修正しました。
  • Delphi XEがインストールされている場合のDelphi 7のコンパイルエラーを修正しました。


詳細な変更ログ、アーカイブを参照してください。



Virtual Treeview V5.0.0 RC1は、 Google.Codeの プロジェクトWebサイトからダウンロードできます 。



PS コンポーネントを使用したソリューションの例は、プロジェクトギャラリーで見ることができます。
