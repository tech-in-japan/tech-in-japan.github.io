Intel / AMD Switchable graphics [Ubuntu 12.04-12.10]

良い一日！



最後に、いわゆる適切に機能することがわかった。 Ubuntu 12.04 LTSの「ハイブリッドグラフィックス」（12.10にも当てはまります。この「マニュアル」に従って、他のDebianベースのシステムでも動作します）



ネットワーク上には解決策がありますが、正直言って、どれだけ試してみても、私を助けてくれませんでした。

たとえば、1年前に「Ubuntu Forums」に Alexislavieからマニュアルがありました。正直なところ、この方法はシステムを3回連続で「殺しました」。 ほとんどの場合、xorgアップデートの問題が示唆されています。 はい、現在のドライバーは非常に古いです。 この指示は真実に最も近いものですが。 実際、私が説明した方法は文字通り少し異なります:-)しかし、うまくいきます。



だから、最終的に成功を収めたのは何ですか：



1）2013年5月10日にすべての更新を含むUbuntu 12.04 LTS（x64）をインストールしました（実験の純度のため-最初から）。



2）次に、 メーカーのWebサイトでドライバーを取得します （私の場合、私のVKラインで最後に利用できるのはamd-catalyst-13.4-linux-x86.x86_64.runです ）。



3）不足しているパッケージをインストールします。 

sudo apt-get install dpkg-dev execstack debhelper dh-modaliases lib32gcc1 libc6-i386 dkms


4）次に実行します：（Alexislavieの指示に感謝！） 

  cd /usr ; sudo ln -svT lib /usr/lib64


5）ダウンロードした.runファイルに実行権限を付与します。 

 chmod +x amd-catalyst-13.4-linux-x86.x86_64.run


6）パッケージを作成します。 

 sudo sh ./amd-catalyst-13.4-linux-x86.x86_64.run --buildpkg Ubuntu/precise


7）インストール 

 sudo dpkg -i fglrx*.deb


8）Xserverを構成します。 

 sudo aticonfig --initial -f


9）再起動します。



10）ビデオカードの切り替えを確認します。 

 sudo aticonfig --px-dgpu


11）再起動します。



12）構成を編集します。 

 gksu gedit /etc/X11/Xsession.d/10fglrx


13）次のディレクトリをLIBGL_DRIVERS_PATHに追加します： / usr / lib / x86_64-linux-gnu / dri / （区切り文字 " ： "）。



14）再起動します。 今、離散VCで、



VKスイッチングは、次のコマンドによって実行されます 

 aticonfig --pxl # List current activated GPU sudo aticonfig --px-dgpu # Activate discrete GPU (High-Performance mode) sudo aticonfig --px-igpu # Activate integrated GPU (Power-Saving mode)


変更を有効にするには、Xserverを再起動する必要があります！ 残念ながら、「ホットな」解決策はありません:-(



便宜上、.bashrcにいくつかのエイリアスを追加しました。 

 alias gpuamd='sudo aticonfig —px-dgpu && sudo pkill X' alias gpuint='sudo aticonfig —px-igpu && sudo pkill X'




たとえば、次のように使用されるレンダリングを確認できます。 

 glxinfo | egrep render




はい、ほとんど忘れていました。CCCメニューの対応するアイテム（VKの切り替え）を使用しないでください。

何らかの段階でシステムが起動せず、「ブラックスクリーン」でフリーズする場合、コンソールに移動してxorg.confを削除（または単にバックアップを復元）し、再起動します。



fglrxパッケージを既にインストールしている場合は、たとえば次のように、それらからシステムを確実に削除する必要があります。 

 sudo apt-get purge fglrx*




これが誰かを助けることを願っています:-)



私の場合、トレースが使用されます。 鉄：

CPU：Intel Core i5 2430m（Intel HD3000 GPU 0x0116）

VGA：AMD Radeon HD 6650m 2GB GDDR5



PS：Ubuntu 12.10のアカウントに-製造元のWebサイトでは、ベータ段階ではありますが、12.10のサポートが記載されています。

PPS：システムが停止したり、データが失われたりした場合、何かが「間違った」場合、保証できません。 あなた自身の危険とリスクですべて。
