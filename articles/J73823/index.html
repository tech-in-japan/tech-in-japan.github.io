ロシア語のWindows Liveカスタムドメイン



ドメイン向けのYandexメールの開始に関するニュースをフォローしています。 何らかの理由で、 8月4日以降、 ロシア語版の Windows Liveカスタムドメインが利用可能になったことはほとんど言及されていません。



このサービスにより、ドメインをWindows Liveサービスにバインドし、ユーザーアカウントを登録できます。



無制限の数のメールボックスが宣言されており、それぞれ5ギガバイトです（500がすぐに利用可能です。リクエストに応じて追加できます）。 さらに、アカウントはLive Messangerで機能します。 メールが不要な場合は、Live Messenger専用のドメインに接続して、サーバーにメールを残しておくことができます。







さらに、 Windows LiveカスタムドメインSDKを使用して機能を拡張する独自のアプリケーションを作成できます 。 また、閉じたメンバーシップと開いたメンバーシップを持つドメインを作成できます（誰でも登録できます）。
