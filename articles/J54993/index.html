Windows Mobile 6.5-Windows Mobileウィジェット

Windows Mobileプラットフォームの開発者だけがい標準コントロールを手に入れたわけではありません。 プラットフォーム自体の開発者チームもこれを理解し、（ネイティブおよびマネージドメソッドに加えて）WM用アプリケーションを開発する「第三の」方法を開発しました。



彼らはWebアプリケーション（RIA、今ではそれらを呼び出すのが流行しているので）のサポートを生み出し、それらをWindows Mobileウィジェットと呼びました。 通常の個別のアイコンを割り当てることができ、下のメニュー（！）などに完全にアクセスできます。





ウィジェットはIEの最新バージョンに基づいていますが、タスクマネージャーでは巧妙に偽装されています:)フラッシュとActiveXの完全なサポートがあることは興味深いです。 たとえば、Media Playerを自由に埋め込むことができます。



興味深い点-これらはすべて最新のドラフトwww.w3.org/TR/widgetsに基づいています。 MSは標準に従うことを試みています:)



そして最後に...透明PNGのネイティブサポートがあります！ OMG、信じられない！ :)
