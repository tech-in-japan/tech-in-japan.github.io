Chromeの拡張機能-外部のダウンロードマネージャーでファイルをアップロード（ctrl + alt +クリック）

この拡張機能により、サードパーティのファイルダウンロードマネージャーをChromeから呼び出すことができます。 外部プログラムでリンクを開くには、CtrlキーとAltキーを押しながらリンクを開く必要があります。



作業には、追加プログラムchromeget.pyが必要です-LinuxおよびMacOSの場合。 Windows版 -chromeget.exe（pythonを含むため、アーカイブに2メガバイトかかります）。



このプログラムはPythonのシンプルなシングルスレッドWebサーバーで、ポート40,000で発生し、127.0.0.1からのリクエストのみを処理します

リクエストは次のようになります： 127.0.0.1 ：40000 / http：//link.url / ...

このようなプログラムを実行します：python /path/to/chromeget.py download_manager、download_managerはあなたのものです

ダウンロードマネージャー（例：uget-gtk、gwget、kget、wgetなど） （Linuxでは、パラメーターが指定されていない場合、既知のプログラムの一部を見つけようとします）

Windowsの場合：c：\ somewhere \ chromeget.exe“ c：\ program files \ flashget \ flashget.exe”（これはflashget 1.96、flashget 3.xでのみ動作します。コマンドライン）



このことを常に使用する場合は、chromeget.pyを自動実行するのが理にかなっています。
