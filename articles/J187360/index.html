sms.ruサービスを操作するためのyii拡張機能

今日は、yiiのコンポーネントに関する情報を共有したいと思います。これにより、 sms.ruサービスを介してSMS メーリングを送信できます。



インストール：



githubリポジトリからyii-smsをダウンロードします-github.com/pimax/yii-sms ： 



cd protected/extensions git clone git://github.com/pimax/yii-sms.git




protected / config / main.phpで、次の行を入力します。 



 'components' => array ( 'sms' => array ( 'class' => 'application.extensions.yii-sms.Sms', 'login' => 'username', //    sms.ru 'password' => 'password', //  ) );




使用法：



SMSを送信する 


 Yii::app()->sms->send('79251234567', ' '); Yii::app()->sms->send('79251234567', ' ', ' ', time(), $test=true, $partner_id);


SMSステータス： 


 Yii::app()->sms->status('sms id');


SMSの費用： 


 Yii::app()->sms->cost('79251234567', ' ');


バランス： 


 Yii::app()->sms->balance();


1日の制限： 


 Yii::app()->sms->limit();


荷送人： 


 Yii::app()->sms->senders();


ログインとパスワードの検証： 


 Yii::app()->sms->check();


以上です！
