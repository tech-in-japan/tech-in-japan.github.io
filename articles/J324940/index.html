3月のProduct Huntの便利なツールのセレクション

誰もがProduct Huntリソースに精通しており、革新的でありつまらない、便利で愚かな新しいプロジェクトが毎日そこにいくつあるかを知っています。 このストリームを並べ替えるのに十分な時間がない場合のために、過去数週間にわたって選択したいくつかのカテゴリ（CSS、パフォーマンス、メッセージング、API）からの興味深い発見の小さな選択を提供します。









CSS



Animistaは、 Ana Travasの事前作成済みCSSアニメーションの広範なコレクションです。 開発者のサイトでは、さまざまなデザイン要素（背景、テキスト、ボタン、画像など）に適用し、方向を設定し、必要な設定（期間、遅延、間欠動作）を選択することにより、それぞれを実際に評価できます。 お気に入りに最適なオプションを追加してダウンロードし、それらのコードを生成することもできます。



ワッフルグリッドは、Lucas GruwezのFlexbox上のシンプルでコンパクトなグリッドシステムです。 これにより、ユーザーのニーズに応じて簡単にカスタマイズ可能な、多くのデバイスのレイアウトを簡単に調整できます。



Dima BloverとAlexander Isoraは、仲間の職人にWebページの背景として使用できる180個の線形グラデーションのコレクションを提供しています。 現在、このサイトで入手可能：プレビュー、PSDコード、および個々のサンプルをダウンロードするためのPNGバージョン、およびSketchとPhotoshopのフルセットのグラデーションを含むパッケージ。



性能



Product Roadmap Builderは、ProdPadチームによって開発されたロードマップツールです。 アプリケーションは、フィードバック、バックログ、仕様、ブレーンストーミング結果を保持しながら、ワークフローを適切に理解して整理するだけでなく、結果のスキームを効果的に視覚化する機会をチームに与えます-同僚に情報を伝え、同時に顧客を感動させます。 このシステムは、他の一般的なプロジェクト管理ツール（Trello、JIRA、GitHub）と統合されます。



Wildfire拡張機能を使用すると、Webページ（オンラインとオフラインの両方）でアクションのシーケンスを記録し、それに基づいてシミュレーションを作成して、さらに編集することができます。 セッションを記録または再生すると、バックログが自動的に作成されます。 自動化ツール（iMacros、Ghost Inspectorなど）を使用する開発者、またはSeleniumなどで作業するテスターに​​適したオプションです。



Loom Workflowは、同僚やクライアントとの視覚的なコミュニケーションのための強力なツールです。 拡張機能を必要なサービス（Gmail、Intercom、Trello、GitHub、Basecamp、JIRA、Google Docs、Hacker News、InVision App、Dropbox、Product Hunt、GitLabなど）と統合して、画面から直接特定のプロセスを示すビデオを記録しますすぐに他の人と共有します



メッセージングシステム



Parloは、Webページおよびアプリケーションでテキストおよび音声チャットボットを作成するためのプラットフォームです。 既製のソリューションとして使用することも、開発者の裁量で設定することもできます。 多数の一般的なエンタープライズシステムと統合します。 このプロジェクトは現在ベータテスト中です。



SendBirdは、メッセージングまたはチャットをアプリケーションに統合するのに役立ちます。 開発者は、100,000人の視聴者にサービスを提供し、オーディエンスセグメント用に個別のチャットを作成し、スパムやわいせつな言語に対する自動保護を含むモデレーションツールを適用することができます。 このツールは、リアルタイムビデオ、マルチプレイヤーゲーム、ソーシャルプラットフォームを再生するエンターテイメントまたは情報サービスの開発に役立ちます。



Slackの優れた革新：App Unfurlsを使用すると、他の人と共有するリンクのプレビューを個人的に作成できます。 これは、会社のドメインに保存されている内部コンテンツと外部リンクの両方で機能します。 実際のリンクに加えて、プレビューには1つ以上のアクションボタンが含まれる場合があります-対話性は、チームが時間を節約するのに役立ちます。



API



新しいZapierコマンドラインインターフェイスは、Node JSでアプリケーションを作成するユーザーに、コマンドラインから直接開発プラットフォームとの即時統合を提供します。 750を超えるツールのエコシステムに入ることで、製品の作成、テスト、展開の可能性が広がります。APIを接続して情報を要求し、データを交換し、共同プロジェクトを実施し、自動化プロセスを簡素化し、コミュニティからフィードバックを受け取ります。



パースペクティブは、オンラインコミュニティで健全な雰囲気を維持し、議論をより生産的にするために設計されたツールです。 ユーザーがフィールドに入力したコメントのテキストを分析することにより、サービスは送信前であっても「毒性」のおおよそのレベルを判断して表示します。 このようなアルゴリズムは、ユーザーにフィードバックを提供することで不快なコメントのリスクを軽減し、管理プロセスを簡素化します。



Jsnoxiousは、アプリケーションアーキテクチャで使用できるAPIエンドポイントのコレクションと、ブラウザから直接独自のエンドポイントを作成する機能を提供します。
