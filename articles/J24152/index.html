Opera 9.50 Beta 2

起こった。



以下は、Opera 9.5ベータ1と比較したOpera 9.5ベータ2の変更ログのロシア語の翻訳です。











Opera 9.50 Beta 1との違い



ユーザーインターフェース

全文履歴検索がアドレスフィールドの一部になり、結果とパフォーマンスが向上しました。

アドレスフィールドにセキュリティ警告の新しいスキームが導入されました。 セキュリティを参照してください。

代替タブの動作を追加しました。 現在、タブを閉じたときにタブを操作するための次のオプションが表示されます。

最後のアクティブなタブをアクティブにする（デフォルト）

次のタブをアクティブにする

現在のタブから開かれた最初のタブをアクティブにします

Opera Mini 4およびWiiのブラウザで使用されているものと同様の新しい空間ナビゲーションコントロール。



ホットキー

[戻る]および[進む]アクションの[Ctrl] + [Left]および[Ctrl] + [Right]は、MacだけでなくWindows / Unixにも適用できるようになりました。

ドラフトの新しい自動保存のため、ドラフトの保存のCtrl + Sショートカットは削除されました（変更はそれほど新しいものではありませんが、ドキュメントの前半で言及されていません）。

メッセージ作成ウィンドウの「添付ファイルの追加」の組み合わせは、Ctrl + SではなくCtrl + Oになりました。

Ctrl + Shift + Sに加えて、Ctrl + Enterの組み合わせがメッセージを送信するようになりました。

メールフィルターのビューでの「削除」アクションのコントロールが変更されました。「ゴミ箱に移動」の場合はShift + Del、「フィルタービューから削除」の場合はDelです。これはすべてCtrl + Xの代わりです。

「タブのコピー」の組み合わせが削除されます。

「閉じたタブの復元」の組み合わせがCtrl + Alt + Shift + ZからCtrl + Shift + Tに変更されました



Opera 9.27と9.50 Beta 2の間のキーボードショートカット



メール、チャット、ニュースフィード

IMAPのパフォーマンスと信頼性の向上。

NTFSおよびReiserFSファイルシステムでの高速起動。

クイック検索を使用すると、Opera Mailで最高の検索結果が得られます。

これで、Operaをコントロールパネルからメールを処理するためのデフォルトアプリケーションにすることができます。

既に多くのフィードメッセージが読み込まれている場合のニュースフィードのダウンロード速度が向上しました。

トレントをダウンロードする際のCPU使用率が削減されました。

番号付けされたバグが修正および最適化されました。



画面とスクリプト

MathMLのサポートが追加されました。 Dev.Operaの記事を参照してください。

Operaはdocument.allを非表示にします。 Hallvordブログの議論を参照してください。

XMLHttpRequest（AJAX）を使用したWebページの改善された作業。

getClientRectsおよびgetBoundingClientRectのサポートが追加されました。

:: selection疑似要素のCSSカラーおよび背景色パラメーターのサポートが追加されました。



ACID3

encodeURIComponentおよびencodeURIのゼロバイトは正しく処理されます。

Unicodeエスケープは、識別子に非識別子文字を挿入するために使用されなくなりました。

Iframes getSVGDocumentがサポートされるようになりました。

createDocumentTypeは、不正な形式の修飾名に対して例外をスローするようになりました。

修正：NodeFilterがtrue => 1を返す

修正：HTMLTableRowElement.rowIndexおよび.sectionRowIndexは、DOMを介して作成されたテーブル行に対して未定義を返します。

HTMLButtonElement.typeのデフォルトは「submit」になりました。

修正：メインドキュメントツリーの外でフォームコントロールコレクションが名前でインデックス付けされていなかった。

修正：Range.surroundContents（）。

修正：範囲を折りたたまないようにinsertNode。

修正：removeNamedItem（）とremoveNamedItemNS（）が見つかりませんエラーをスローします。

修正：NodeIteratorは、動的な変更の下でも機能するようになりました。

修正：適切な1900年のオフセットを行うためのDate.UTC（）。



その他

履歴とメール検索全体が個別のプロセスになりました。

Opera Linkは、Expressパネルのブックマークとセルに加えてノートを同期するようになりました。

新しいsearch.iniには、Google ImagesおよびWikipedia検索エンジンが含まれています。

Gmail 2との互換性が改善されました。

安全性

不正防止の新しいバージョンが実装され、デフォルトで有効になりました。

拡張検証（EV）証明書のサポートが追加されました。

ルート証明書の自動更新が追加されました。

アドレスフィールドに新しいセキュリティ警告システムが導入されました。

グリーンフィールドのゴールデンキャッスル-拡張検証証明書を使用した安全なサイト用

黄色のフィールドに銀の南京錠-シンプルで安全なサイト用

灰色のボックスに疑問符-問題があるHTTPSサイトの場合

警告なし-通常のサイト用

赤い詐欺アラート-ブラックリストに登録されたサイト用

Operaは、ローカルホストのローカルサーバー、ローカルネットワークサーバー、リモートインターネットサーバーを区別するようになりました。 ローカルサーバーはリモートリソースを使用できますが、その逆はできません。

Windows固有の変更

集中的なグラフィック操作を必要とするWebページでのレンダリングパフォーマンスが向上しました。

Silverlight 1.1のウィンドウレスモードが機能するようになりました。
