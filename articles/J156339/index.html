Windows 8で古いMobility Radeonカードのドライバーを取得する方法

比較的最近、AMDはWindows 8用の特別なドライバーパッケージ、 AMD Catalyst Legacy 12.6 （ AMD Webサイトのニュースへのリンク）をリリースしましたが、Mobility Radeon HD 3400を搭載したラップトップでは動作しませんでした（動作しないことを意味しません-チェックしてください）このトピックで説明されている手順に進む前に）。



このサイトを訪れたとき、古いモバイルカードのセクションではWindows 8がまったく言及されていないことがわかりました。 Windows 7用のドライバーを入手できます。必要な場合、このドライバーは製造元のものです。8種類の組み込みドライバーが非常にうまく機能する場合、それはユーザー次第です。 しかし、私はすぐに、Catalyst Control Center（グラフィカル設定ユーティリティ） が動作しない可能性があると言います。 Windows 7のドライバーを取得する方法は、猫の下で教えてくれます。



  1. AMDのWebサイトにアクセスして、ご使用のアーキテクチャのWindows 7用ドライバーの最新バージョンをダウンロードします。
  2. コンポーネントのインストールを開き（ウィンドウ+ X->コントロールパネル->プログラムとコンポーネント-> Windowsコンポーネントのインストール）、. NET Framework 3.5コンポーネントを選択します（現在はサイトからダウンロードできず、デフォルトでは無効になっており、そのようにオンになります）。誰かが本当にそれを必要とする場合、インストールディスクからオフラインでそれをインストールする不便な方法-私は覚えてコメントを書き込もうとします）。
  3. AMDのWebサイトからダウンロードしたドライバーをインストールしようとしています。 （彼によれば）エラーなしでインストールする必要がありますが、最後の[OK]をクリックするとすぐに、実際には何も変更されておらず、システムはMicrosoftのドライバーを使用していることがわかります。 重要：ドライバーは機能しませんでしたが、インストール（または少なくともハードドライブのどこかで解凍）が必要です！
  4. modderをダウンロードしてインストールします： http ://www.hardwareheaven.com/modtool.php
  5. ここでmodderを実行すると、どの互換性オプションを設定しても、未知の理由でクラッシュします。 したがって、参照によってファイルをダウンロードし、解凍して、modderがインストールされたディレクトリ内の同じ名前のexeファイルに置き換えます。http ：//www.driverheaven.net/driverheaven-tools-discussion/178628-possible-fix-9-2- later.html
  6. ええ、今、あなたはすでにこの実行可能ファイルを実行し、公式ドライバーが解凍されているフォルダーへのパスを表示できます（私はそれを持っていますC：\ AMD \ Support \ 12-6-legacy_vista_win7_32_dd_ccc_whql）、変更ボタンをクリックして少し待ちます。
  7. C：\ AMD \ Support \ 12-6-legacy_vista_win7_32_dd_ccc_whql \ Packages \ Drivers \ Display \ W7_INF、またはそこで公式ドライバーを解凍した場所に移動します。 そこにはたくさんのファイルがあります。 * .infで終わるものに興味があります。インストールできます。 ファイルを右クリックして「インストール」を選択すると、インストールが行われます。 このディレクトリ内のファイルの名前は異なる場合があります。たとえば、私のドライバーはCL145534.infと呼ばれていました。 さらに、すべてのinfsをそこにインストールできるわけではありません。私の場合、そこにあるCW145534.infはインストールされていません。 インストールが成功したことを示すインジケータは、「インストールに成功しました」という言葉が付いたダイアログボックスの外観と考えることができます。 サブスクリプションの解除によりドライバーをインストールしたくない場合は、ポイント8を参照してください。それ以外の場合はスキップします。
  8. このアイテムは、Windowsがサブスクリプションの解除によりドライバーのインストールを拒否する場合に必要です。 彼はまた、デスクトップの右下隅に永遠の碑文「テストモード」を与えてくれます（この碑文を削除する方法をコメントで伝えることができれば、それは素晴らしいでしょう）

    1. 昇格した特権モードでコマンドラインを開きます（新しい複雑な開始メトロメニューを開き、キーボードでcmdと入力し、[コマンドライン]アイコンを右クリックし、下部のポップアップメニューで[管理者として実行]を選択します）。
    2. そこで次のコマンドを入力します。 

      bcdedit /set {current} testsigning on bcdedit /set {current} nointegritychecks true
    3. キーボード（windowsボタン）+ Rを押し、そこにgpedit.mscと入力してEnterを押します。 開いたウィンドウで、ユーザー構成->管理用テンプレート->システム->ドライバーのインストールのパスに沿って進み、右側のペインで「デバイスドライバーのデジタル署名」項目をダブルクリックし、「オフ」を選択して「OK」をクリックします
    4. コンピューターを再起動し、手順7に戻ります（管理者と経験豊富なWindowsユーザーは、gpupdateコマンドとその後に2つのスペースがあり、ポリシーを即座に更新できるため、これは機能しないため、再起動する必要があります）


  9. ビデオカードドライバーのプロパティを開きます。 （ウィンドウ+ X->デバイスマネージャー->ビデオアダプター->ビデオカード（ATI Mobility Radeon HD 3400シリーズ）->右クリックして[プロパティ]を選択し、[ドライバー]タブに移動します）。 「更新」ボタンをクリックし、「このコンピューター上のドライバーを検索する」オプションを選択し、「既にインストールされているドライバーのリストからドライバーを選択する」オプションを選択します。 Windowsに同梱されているものを含むいくつかのドライバーがあります-その名前には「（Microsoft-WDDM v1.1）」という行が表示されます。 これは必要ありません。カードと呼ばれるものが必要です。たとえば、ATI Mobility Radeon HD 3400シリーズがあります。 そのようなものがない場合は、ステップ7に戻り、インストールを忘れた他のinfを探します。 ドライバを選択した後、[OK]をクリックして、インストールが完了するまで、インストールが完了すると再起動します。
  10. 再起動後、ドライバが実際に同じであることを確認します（前の段落で行ったように、「provider」フィールドでMicrosoftからAdvanced Micro Devicesに名前を変更する必要があるビデオカードドライバのプロパティを開きます）。
  11. Catalyst Control Centerが機能しないことを確認します（デスクトップを右クリックすると、ドロップダウンメニューの最初の項目はCatalyst Control Centerになります。この項目を選択した後、彼は数秒間考え、別のダイアログボックスに「何がわからないか」という意味のメッセージを書き込みます設定するので、開始しません）。 CCCがあなたのために働くなら、コメントでそれについて書いてください！
  12. 再度インストールしようとしています ： http : //www2.ati.com/drivers/legacy/amd_catalyst_12.6_legacy_win8.exe 確立された場合-Catalyst Control Centerを取得する必要があります。 そうでない場合は、少なくとも、AMDの動作するドライバーがあります。
  13. お楽しみください。
