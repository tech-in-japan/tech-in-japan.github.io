vim用のPerlサポートプラグイン

プラグインのインストール 



mkdir ~/.vim cd ~/.vim curl http://www.vim.org/scripts/download_script.php?src_id=21048 -o perl-support.zip unzip perl-support.zip rm perl-support.zip echo "filetype plugin on" >> ~/.vimrc yum install perl-Perl-Critic perltidy




新しいファイルを作成する







ファイルの拡張子が.plまたは.pmの場合、ドキュメント用のヘッダー付きでファイルが作成されます。

プラグインは、厳格な使用、警告の使用、utf8の使用も規定しています。

自分用にカスタマイズ-難しくありません。



vimエディターを終了せずに真珠スクリプトを実行する



コマンド\ rr




空のフィールドにカーソルを移動して、\ rrと入力します（テキストの入力を除くすべてのモードで）。

プログラムが実行されます。



エディターを離れずにperl tidyで注文する



コマンド\ ry




1つのコマンドでコードを美しく標準的な外観にします。



Perl-Criticの助けを借りて「その場で」批判します



\ Rpcコマンド




\ rpcを入力した後-真珠評論家は、彼が底に着いたものを教えてくれます。

また、Perl Best Practicesの本のページ番号を提供します。これを修正する方法を理解するために読む必要があります（必要な場合）。
