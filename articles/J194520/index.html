ジャガーは仮想テクノロジーを使用して販売を整理します

ジャガーランドローバーの車両は、取り付けられたコンポーネントによって互いに区別されます。これは、特定のコンポーネントセットを搭載した車がどのように見えるかを顧客に視覚的に示したいディーラーにとって、常に問題です。 この問題の解決策として、Jaguar Land RoverとIBMは3DアプリケーションJaguar Land Rover Virtual Experienceを開発しました。これはPC上で動作し、モーション認識テクノロジーを使用して、ディーラーのデモルームで自動車の3Dモデルのコンポーネントを表示および変更できます。







ジャガーランドローバーバーチャルエクスペリエンスプロジェクトは、現実とデジタルの世界がどのように結びついてショッピングプロセスを変えるかを示しています。



「 私たちの車はすべてカスタムメイドであるため、車とそのコンポーネントを選択するのがユーザーにとってどれほど便利であるかが重要です 」と、ジャガーランドローバーのデジタルテクノロジーディレクターであるネイサン・サマーズは述べています。 「 何かを開発している場合、車を補完することは有益であり、通常の販売プロセスに有機的に適合するはずです。」



Jaguar Land Rover Virtual Experienceを使用するには、ユーザーはタブレットを使用してシステムに登録し、車を選択する必要があります。車の投影は大画面に表示されます。 次に、ジェスチャを使用して、車の画像を制御し、たとえばドアを開けて内部を表示するなど、その機能を調べます。 エンジンの音を聞きたいですか？ ボタンをクリックするだけです。 「 私たちには1つの目標がありました。最も現実的でインタラクティブで楽しいソリューションを提供することです 」とサマーズは強調しました。







ジャガーランドローバーバーチャルエクスペリエンスはラップトップで実行されるため、ジャガーの担当者は、さまざまな国で開催される自動車展示会など、どこでも使用できます。 このアプリケーションは、ユーザーの好み、特に購入プロセス中に車のどのような特徴がより頻繁に注目されるかに関するデータを収集するのにも役立ちます。



IBM Researchの小売調査部長Sima Nadlerによると、ジャガーランドローバーバーチャルエクスペリエンスは、デジタルテクノロジーがストアショッピングをどのように変えているかを示す例です。「 デジタルは現実の世界に存在し、ジャガーはその意味で最前線にあります 」 。



Jaguarは、Jaguar Land Rover Virtual Experienceを新世代の車両の設計ツールとして使用することも計画しています。 設計者は、仮想自動車モデルを作成して、特定の設計上の決定が自動車の外観と運転体験にどのように影響するかを、組み立てラインを離れる前でも確認できます。 「 脆弱性を迅速にプロトタイプ化し、積極的に特定するために、テクノロジーを学習する必要があります 」とサマーズは説明しました。



上記を要約すると、小売業者は、使用されるITに大きく依存するユニバーサル販売チャネルを編成する戦略を使用して、オンラインショッピングとオフラインショッピングを組み合わせようとしています。 残念ながら、IBMがマーケティングディレクターを対象に実施した新しい調査では、マーケティングディレクターの35％のみが複数の販売チャネルにわたるマーケティングイニシアチブを統合していることが示されました。 したがって、企業は約850億ドルの潜在的な売り上げを失い、CIOがマーケティング部門の同僚とより緊密に連携する必要があるもう1つの理由です。
