DXVAをサポートする新しいバージョンのDivx H.264デコーダー

DXVAをサポートするH.264デコーダーの棚に到着しました。 バージョン7から、DivxにはH.264デコーダーが含まれ、DXVAをサポートする新しい修正が登場しました。 したがって、4つのDXVA H.264コーデックを区別できます。



  • PowerDVD H.264
  • Media Player Classic Home Cinema H.264（ffdshowに基づく）
  • Microsoft DTV-DVD H.264（Windows 7から）
  • Divx H.264


最近、衛星HDチャネルからのストリームでH.264デコーダーをテストしました（テストの開始と終了 、その後Divxなし）。 このようなデータは、ほとんど解読が困難です。 ストリームが途切れており、高速で高品質のインターレース解除が必要です。 このような品質テストでは、テストによりDXVAコーデックの著しい優位性が示されました。 パフォーマンスのギャップは非常に大きく、Core2 Duo E8500のCPU使用率はパーセントで測定されます。インターレース解除が必要な場合、パフォーマンスが強力なCPUでもボトルネックになるため、これは重要です。 つまり この種のビデオ素材には、DXVAコーデックが不可欠であると言えます。



以前のバージョンと同様に、DirectShow H.264デコーダーに加えて、MKV DirectShow SplitterとAAC DirectShow Filterがあります。 それらにMKV Media Foundation Splitterが追加されました（Windows 7のみ）。 Media FoundationパイプラインはWindows Vistaで導入され、たとえばWindows Media PlayerやWindows Media Centerで使用されているため、このようなスプリッターの存在は、MKVがこれらのプログラムを理解するのに役立ちます。



PS外部DirectShowフィルターを接続するメカニズムにより、指定されたすべてのDXVAコーデックをMedia Player Classic Home Cinemaで使用できます。 Microsoft H.264はWindows 7に組み込まれており、PowerDVDとDivxをインストールする必要があります。MPC-HCからコーデックを表示できます。
