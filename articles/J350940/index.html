4月のC ++会議

みなさんこんにちは！

今日、私はすぐに2つの会議について話したいです。 どちらの会議も、C ++コミュニティ（サンクトペテルブルクのC ++ロシア 、ミンスクのCoreHard Spring）によって開催されます。



サンクトペテルブルク







C ++ロシアは4月19〜21日にサンクトペテルブルクで開催されます。

今年の会議プログラムは興味深いレポートと実験が豊富です。 参加者がすべてを好きになることを本当に願っています。 ハーブ・サッター（スカイプ経由）、アンドレイ・アレクサンドレスク、ジョン・カルブ、デイブド・ヴァンデブールデ、ジョナサン・ボカラ、ミハイル・マトロソフ、アントン・ポルキン、エフゲニー・ズエフ、その他の素晴らしいスピーカーが会議で講演します。



おそらく、C ++ Russia 2018に関する私にとって最大のニュースは、Andrei Alexandrescuがそれについて話すことです。 私が書いたように、Andreiは参加者からの上位3つのリクエストに含まれています。 2006年、Andreiの本Modern C ++は私のキャリアを5年先に決定しました。 この会議レポートは、C ++に関するものではなく、プログラムを高速化するアルゴリズムと技術に関するものです。



今年の実験の1つは、スピーカーとの夕食会です。ソーシャルイベントでは、別のチケットを購入して、よりカジュアルな雰囲気の中でスピーカーとチャットし、食事を共有できます。



ミンスク







4月7〜8日に、C ++コミュニティCoreHard.byは、C ++言語と関連する「ハードコア」技術に特化した第8回国際2日間会議CoreHard Spring 2018を開催します。 ベラルーシ、ロシア、ウクライナ、ドイツ、オランダの大手IT企業のスピーカーが集まり、開発とテストの経験について話します。 C ++ 11/14/17および標準化、STL、Boost、アセンブリドッキング、C ++ for web、ゲーム開発、コード生成、静的分析、プロファイリング、Windowsでの関数のインターセプトなどに関するレポート。 基調講演者-C ++標準化委員会のメンバー-アントンポルキン、ニコライジョスティス（WG21ワーキンググループ）、ウーターヴァンオイジェン（SG14ワーキンググループ）。 15人のスピーカー、3つのレポートストリーム、専用マスタークラスの日（4月8日）：



  • 軽快でモダンなC ++、アントン・ポルキン
  • C ++ 11、14、17で正しくプログラミングします：言語と標準ライブラリの機能、Vadim Vinnik
  • C ++ 17-新しい言語とライブラリの機能[英語]、Nicolai Josuttis




会議の公式ウェブサイトで会議プログラムとチケットを探してください。 リモートで参加したい人のために-オンライン放送が利用可能です。 プロモーションコードCOREHARD_COMMUNITYは、レポート当日（4月7日）のチケットを20％割引で提供します。



PSコミュニケーションや、プラスの会議や会議に関する最新ニュースをお望みの方は、 CoreHardチャットとC ++ロシアチャットをご利用ください。
