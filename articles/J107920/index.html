Googleは、Facebookに連絡先をエクスポートしたいユーザーについて考えを変えるよう求めています

GoogleとFacebookの対立はますます興味深いものになっています。



背景を思い出させてください：GoogleはGoogle Contacts API APIの利用規約に変更を加えたので、それらを使用するすべての人は相互性を保証する義務があります。 その後、FacebookはGoogle Contacts APIを使用しないフォームを作成することで制限を回避しました。ユーザーはこれを使用して連絡先を自分のディスクに手動でエクスポートし、結果のファイルをFacebookにアップロードできます。



次に、連絡先をエクスポートするユーザーにGoogleが書き込む内容を見てみましょう。











連絡先を引き渡す



ちょっと待って 連絡先情報を返品できないサービスにインポートしてもよろしいですか？



他のサービスへのデータの再エクスポートを許可していないサイトからこのページに移動しました。本質的には、友人に関する連絡先情報を閉じています。 したがって、データをインポートした場合、それを返すことはできません。 データをインポートする前に確認する必要があると考えています。 このデータ保護主義には絶対に反対しますが、選択はあなた次第です。 最終的には、データを制御する必要があるためです。



もちろん、Google連絡先へのエクスポート機能を使用して、いつでも連絡先をダウンロードできます。



このメッセージは、Googleコンタクトの友人に代わって書かれました。



[_] Facebookからデータをエクスポートできるようにしたい。 データ保護主義について私に代わって苦情を申し立ててください。 （Googleはあなたの名前やメールアドレスを共有しません。）



[_]データのエクスポートを開始します。 それらが別のサービスにインポートされるとすぐに、このサービスではそれらをエクスポートして戻せないことがあります。




はい、Googleは連絡先情報をインポートした後、それを再エクスポートする機会がないことを明示的に警告し、データを使用して保護主義に関する公式の苦情を登録することを可能にします（目的は不明のままです）。



ただし、それでもデータをエクスポートできます。 警告を読む前に、おそらく多くのFacebookユーザーがこれを行うでしょう。
