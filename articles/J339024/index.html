Zyxel Nebula：クラウドクラウドエコシステムの主要機能の概要

中小企業の場合、コストが高いため、ネットワークの構築は困難です。 Zyxelは、この問題を解決するNebulaクラウドシステムを提供し、ハードウェアだけでなくソフトウェアコンポーネントにも焦点を当てたネットワークを作成および管理するためのソリューションです。 その機能、主要な機能、およびデバイスについて以下に説明します。







はじめに



ワイヤレスアクセスポイント、スイッチ、ルーターなど、Nebulaブランドのすべての製品はクラウドベースです。 従来のソリューションとは異なり、アプリケーションまたはWebポータルを使用して、リアルタイムで集中管理、構成、診断、および監視できます。 したがって、特定のデバイスとネットワーク全体の両方を制御できます。 また、高度なスケーラビリティにより、新しいデバイスを接続するだけで、ネットワークの新しいセクションを簡単に展開できます。 これは特別な知識を必要とせず、ITスペシャリストの参加なしでも実行できます。デバイスを接続するだけで機能し始めます。 Nebulaを使用すると、複数のネットワークセグメント間で、インターネットを介した信頼性の高い安全なVPN接続を簡単に確立できます。



システムの主な機能：



-大規模ネットワークを迅速に展開するためのビルトイン管理ツール。



-直感的で自動化されたネットワーク管理インターフェース、およびネットワーク管理を大幅に促進する機能の継続的な更新-初めて作業する人がすばやく学習できるようにする。



-すべてのコンポーネントの一元的な可視性。これにより、ネットワークを最適化することにより、ソフトウェアと機器のコストを削減できます。



-ネットワークコンポーネントの良好な互換性を提供する1つのベンダーのソリューションの完全なポートフォリオ。



-クレジットおよびオンデマンドのライセンスモデルは、ビジネスの現在のニーズに応じてシステムを使用する柔軟性を提供します。



-デバイスとネットワーク自体は、ライセンスの有効期限が切れても機能し続けます。



ネットワークアーキテクチャ



Nebulaは、Software as a Service（SaaS）モデルを使用して、インターネット経由でネットワークを作成および管理するためのアーキテクチャを提供します。 つまり、ネットワーク管理のためにローカルシステムを物理的に構築する必要がなくなります。 すべてのNebulaデバイスは、セキュアなTLS接続を介してクラウドから管理されます。 したがって、インターネットのある場所から何百ものデバイスを管理し、中央のコントロールパネルからポリシーとネットワーク設定を変更できます。



NebulaはAmazon Web Service（AWS）ベースのインフラストラクチャとサービスを使用しているため、データとトラフィックはAWSクラウドセキュリティによって保護されています。 デバイスからのデータは2つに分けられます-サービス情報（たとえば、構成、監視データ、統計など）は、専用のNETCONFプロトコル（詳細は後述）を使用した安全な接続を使用してクラウドに転送され、ユーザートラフィック（たとえば、 Webサーフィンまたはアプリケーションの使用）は、クラウドを経由せずにすぐにターゲットサーバーに送信されます。



NETCONF標準



Nebulaは、クラウドを通じてネットワーク設定を安全に変更するためにNETCONFプロトコルを使用する業界初のソリューションです。 このプロトコルを使用してデータを送信する場合、TLSプロトコルが使用され、データのセキュリティが保証されます。 NETCONFの実装前は、コマンドラインインターフェイススクリプトとSNMPが使用されていました。 これらのソリューションにはいくつかの欠点があります。たとえば、トランザクション管理や信頼できるセキュリティメカニズムがありません。 NETCONFプロトコルは、これらの欠点に対処するために設計されました。 NATの障壁を克服するためにTCPをサポートし、上記のプロトコルよりも信頼性が高いと考えられています。 また、使用するトラフィックが少なくなります。これは、クラウドを介してネットワークを管理するときに重要です。 したがって、今では星雲の最適なソリューションです。



星雲コントロールセンター



Nebula Control Center（NCC）は、ネットワークを操作するための広範なツールと、ネットワークで発生するプロセスの明確なビジョンをユーザーに提供します。 コンピューター、スマートフォン、およびタブレットで使用可能なWebインターフェイスを介して、ネットワークパフォーマンス、デバイスステータス、および全体的なネットワークステータスの分析をすぐに確認できます。 情報はNebula Control Centerに自動的に入力されます。管理者が行う必要があるのは、ログインするだけです。 Nebula Control Centerには、デバイスとユーザーを保護するだけでなく、ネットワーク全体のセキュリティを強化するために必要な情報を提供する多くのセキュリティツールもあります。 次に、NCCが提供する主な機能について説明します。



役割ベースの管理



ネットワーク所有者は、それを操作する管理者に異なる役割を割り当てることができます。 各役割には、ネットワークを管理し、ゲストアクセスを設定するための1つまたは別の機能セットがあります。 これは、そうする権限を持たない人が有害な可能性のある特定のアクションを実行できるという事実から保護します。 ロールを適切に配布することにより、管理者がネットワークを正しく構成することも保証されます。



リアルタイム監視



Nebulaでは、24時間365日発生するネットワークステータスを監視します。 管理者は、設置された機器の操作など、ネットワーク内のすべてのアクティビティに関する情報を受け取ります。 このようなレポートは保存して、後で新しいデバイスのインストール中またはネットワーク拡張中に使用できます。 すべてのレポートは柔軟に構成されているため、管理者は必要な情報のみを受け取ります。



ネットワーク管理ツール



多機能ダッシュボード、マップ、フロアプランなどによって統合されたこれらのツールを使用すると、Nebula Control Centerを通じて選択した領域を微調整できます。 目的のサイトを選択するだけで、詳細なネットワーク分析、認証管理、設定の検証、VPNトンネルなどの要素を受け取ります。



アラートを構成する



このツールは、管理者が複数のネットワークデバイスを管理するのに役立ちます。 そのうちの1つに構成の変更があると、管理者はすぐに通知を受け取ります。 それらはネットワークのサイズに関係なくネットワーク全体から取得され、設定とセキュリティポリシーを最新の状態に保つことができます。



不正な設定保護



誤った設定または誤った設定によって引き起こされる通信の中断を防ぐために、NebulaデバイスはNebula Control Centerから直接設定を要求し、クラウド経由で設定を受け取ることができます。 これにより、常にネットワークを動作可能な状態に保つことができます。



ログイン監査



Nebula Control Centerは、システムに登録されている各管理者のログイン時間とIPアドレスを自動的に記録します。 これにより、システムに変更を加えた人と、いつ変更されたかを追跡できます。



SSLサポート



クラウドサービスと管理者間の通信は、SSLプロトコルを使用した安全なチャネルを介して実行されます。 これは、すべての機密データが保護され、第三者によって傍受されないことを意味します。



タイムアウト後に接続を閉じる



Nebulaクラウドアーキテクチャでは、接続は特定の期間のみアクティブになります。 指定された期間内に開いている接続を介してトラフィックが送信されない場合、ユーザーはシステムから切断され、その前に対応する警告を受信します。 その後、ユーザー名とパスワードを再入力してセッションを復元できます。



星雲モバイルアプリ



iOSおよびAndroid用の独自のモバイルアプリケーションは、ネットワーク管理を大幅に高速化するツールを提供します。 たとえば、QRコードスキャナーを使用すると、バーコードをスキャンするだけで新しいデバイスをすばやく登録できます。 登録された各デバイスについて、シリアル番号やMACアドレスなどの詳細情報、およびインストールされている場所を確認できます。 各場所について、詳細情報も表示されます。 たとえば、ネットワーク負荷を調べて、特定のネットワークデバイス（ワイヤレスアクセスポイントまたはルーター）に現在接続されているデバイスの数を調べることができます。 インストールしたデバイスの写真を撮って、アプリケーションに画像を残すこともできます。 これはすべて、ITスペシャリストがネットワークステータスをリアルタイムで監視するのに役立ちます（そして大幅に促進されます）。

ワイヤレスアクセスポイント



Nebulaワイヤレスアクセスポイントは、企業だけでなく、オフィス、学校、病院、店舗、レストラン、その他の公共の場所に設置するために設計されています。 これらは、MIMO、スマートアンテナ、DCS、負荷分散、スマートクライアントステアリングなどの最新の802.11ac規格と技術をサポートしています。 これにより、生産的なWi-Fiネットワークを構築し、信頼できる信号カバレッジを整理できます。 それらはすべてクラウドを通じて制御され、自動構成機能を備えているため、ワイヤレスネットワークの展開と保守が大幅に容易になります。 以下では、Nebulaブランドのアクセスポイントの特徴を理解することができます。



機能：



•MIMO 2x2 802.11ac APアクセスポイントは、最大1.2 Gbps（NAP102）までの速度をサポートします。

•MIMO 3x3 802.11ac APアクセスポイントは、最大1.75 Gbpsの速度をサポートします（NAP203、NAP303、NAP353）。

•二重最適化されたアンテナ（NAP203）。

•スマートアンテナ（NAP303）。

•全天候型の保護をIP66に提供するエンクロージャー（NAP353）。

•自己構成、自動展開（ゼロタッチ）。

•エンタープライズクラスのセキュリティと無線通信の最適化。

•DCS、負荷分散、スマートクライアントステアリング。

•Facebookアカウントを使用したシステムへの登録のサポート。







スイッチ



レイヤー2トラフィックをサポートするNebulaレイヤー2ネットワークスイッチは、ブランチオフィスに展開し、クラウドを介してそれらを管理するための優れたソリューションです。 Nebula Control Centerを使用すると、1つのテンプレートを使用するだけで、使用可能なすべてのポートをリモートで監視および構成でき、マウスを数回クリックするだけで複数のスイッチを構成できます。 また、構成と管理の簡素化、ネットワーク全体のステータスの表示、およびブランチネットワークの展開を大幅に加速するリアルタイムモニタリングなど、多くのクラウドの利点も利用できます。 ACL、VLANベースのQoS、PoEスケジューリングなどの高度な設定により、ネットワーク管理の効率が大幅に向上します。 以下では、Nebulaブランドのネットワークスイッチの特性に慣れることができます。



機能：



•PoEサポートとPoEなしの8/24ポートを備えたギガビットL2スイッチ。

•高速ネットワーク（NSW200-28P）に接続するための10GEアップリンクポートの可用性。

•ACLおよびVLANの便利な設定。

•DHCPサーバーガードとIGMPスヌーピングのサポート。

•375ワット（NSW200-28P、NSW100-28P）/ 180ワット（NSW100-10P）の電力バジェットによるPoEテクノロジーのサポート。

•ネットワークトラフィックを監視するためのポートミラーリング。

•インテリジェントPoEテクノロジーとネットワークトポロジ。

•RADIUSまたは802.1X認証、静的MAC転送。







セキュリティゲートウェイ



Nebula Network Security Gatewayは、組織に信頼性の高いネットワークセキュリティを提供します。 彼らは、IDP（侵入検知システム）などの次世代ファイアウォール機能を使用します。これは、中小企業に高レベルの保護を提供します。 Nebula Security Gatewaysはクラウドを念頭に置いて設計されており、自動的に設定を受け取り、サイト間VPNを構成し、インターネット経由でソフトウェアの更新とセキュリティ署名を自動的に受け取ります。 管理者は、Nebulaクラウドインターフェイスを使用して、ネットワーク全体にセキュリティポリシーを設定し、すべてのブランチのネットワークを簡単に制御できます。 以下に、Nebula独自のネットワークセキュリティゲートウェイの特性を示します。



機能：



•クラウドからネットワーク、セキュリティ、およびアプリケーションを完全に制御。

•簡単なサイト間VPNセットアップ

•ネットワークセキュリティは、次世代ファイアウォール、IDP、およびアプリケーションパトロールによって提供されます。

•組み込みのDHCP、NAT、QoS、およびVLAN管理機能。

•静的ルーティングとDynDNSサービスのサポート。

•セキュリティルールとアプリケーション管理。

•Nebulaクラウドでの認証サポート。







免許



Nebulaクラウドサービスは、いくつかのサブスクリプションオプションで利用できます。 ただし、主なことは、ライセンスの有効期限が切れても、ユーザーはサービスの基本機能を引き続き使用できることです。 これにより、ネットワークは常に稼働したままになります。 次に、提供されるライセンスの種類について説明します。



クレジットライセンス



Nebulaデバイスの初期運用コストを削減するために、1年間のNebula Control Centerライセンスが付属しています。 このライセンスの有効期限が切れると、ユーザーは星雲ポイントでライセンスを更新できます。 ライセンスが新しいデバイスに使用されているかどうかに関係なく、さまざまな種類のデバイスやモデルにライセンスクレジットを使用できるため、クレジットライセンスはサービスのアクティブ化に必要なSKUとライセンスキーの数を減らします。



Nebula Control Center Limited Lifetime License



制限付きライフタイムライセンスは、年間ライセンス更新なしでNCCを使用してデバイスを構成および監視し、追加のライセンス費用なしで以前に購入したNebulaデバイスを新しいデバイスに置き換えるために、フル機能のサービスを長期間必要とするユーザーを対象としています。

さらに、制限付きライフタイムライセンス方式は、従来のコントローラーのライセンス方式と類似しており、理解可能で便利です。 これは、ライセンスの購入に予測可能なコストを提供し、チャネル内の顧客とパートナーの両方に柔軟性を提供し、長期的にコストを削減するクレジットライセンスモデルに加えて開発されました。



ライセンスNebula Security Serviceライセンス



ライセンス購入の初期コストを削減し、登録を簡素化し、Nebula Security Gateway（NSG）をアクティブ化するために、各NSGにはIDPおよびApplication Patrol Nebula Security Serviceライセンス（NSSIDP）が1年間付属しています。 Control Centerライセンスに加えて、NSS-IDPライセンスを購入する必要があります。 会社が複数のNSGを使用している場合、ライセンスの有効期限も同期できますが、Nebula Control Centerサービスのライセンスと同期することはできません。



同期ライセンスの有効期限



Zyxel Nebula Control Centerは、同じ日にすべてのライセンスの有効期限を自動的に調整します。 企業が機器の追加ライセンスを購入すると、古いライセンスと新しいライセンスの有効期間が再カウントされ、すべてのライセンスの有効期限が同じになるように調整されます。その結果、会社のすべてのライセンスが同じ日付まで有効になります。 Zyxel Nebula Serviceの使用には、ライセンス同時終了の条件が適用されることに注意してください。



乾燥残留物中



Zyxel Nebulaは、ハードウェアとソフトウェアのエコシステムであり、これを使用してネットワークを一元的に構築し、簡単なツールで1か所から管理できます。 Zyxel Nebulaファミリには、ワイヤレスアクセスポイント、スイッチ、セキュリティゲートウェイ、クラウドベースのコントロールセンターが含まれます。 構築されたネットワークは、公共の場所（レストラン、学校など）と同様に、いくつかの支店を持つ中小企業での使用を目的としています。 絶えず拡大している企業の場合、新しいデバイスのインストールとその設定が容易なため、ネットワークのサイズを大きくすると非常に便利です。実際、機器は自動的に構成され、ネットワークのメンテナンスとIT担当者のメンテナンスの両方が節約されます。 Zyxelは柔軟なライセンスモデルも提供し、サブスクリプションの有効期限が切れた後でも基本的なネットワーク機能の機能を保証します。 これらすべてにより、NebulaはSMB（中小企業）企業にとって非常に魅力的です。
