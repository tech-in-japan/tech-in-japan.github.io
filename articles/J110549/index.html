iPhone開発者向けに開始されたWindows Phone 7 Webサイト

本日、 Windows Phone 7 Interoperability Webサイトの最初のバージョンがオープンし、さまざまなモバイルプラットフォームの開発者向けの資料がホストされます。



現時点では、リソースには、ソフトウェアモデルとデザインのイデオロギーであるWP7プラットフォームに関するiPhone開発者向けのいくつかの記事とビデオが含まれています。



多くのiOS開発者は、既存のアプリケーションとゲームの移植に興味を持っています-たとえば、最近のMobile Developer Dayで、JoyBitsの代表者は、人気のDoodle Godゲームを移植するのに数日しかかからないと言いました-今日、それは世界で50の最も人気のある有料WP7アプリケーションの1つです！



将来的には、Androidの開発者を含む興味深い資料や、クライアント側とサーバー側の異なるプラットフォームでアプリケーションを作成するためのガイドラインがサイトに掲載される予定です。 同じプロジェクトの一環として、 HTML5 Labsが今週登場しました。



現在、Windows Marketplaceアプリケーションストアには4000を超えるアプリケーションがあり、その成長率はAndroid Marketplaceの同様の速度を超えていることを思い出させてください。 Windows Phone 7トレーニングコースはロシア語で無料で利用できます 。
