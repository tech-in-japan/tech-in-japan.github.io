Wolfram Programming Cloudが利用可能になりました



Wolfram ResearchのCEO、Stephen Wolframによる翻訳投稿。

元の投稿： Wolfram Programming Cloud Is Live！



26年前の同じ日に、私たちはMathematica 1.0をリリースしました 。 そして今日が歴史的な日になることを非常に嬉しく思います。新しいWolfram言語 （Wolframプログラミング言語）に基づく製品ラインの最初のオブジェクトであるWolframプログラミングクラウド （ Wolframプログラミングクラウド ）をリリースします。







Wolfram言語 、特にWolframプログラミングクラウドで作業する私の主な目標は、プログラミングプロセスを再定義し、可能な限りすべてを自動化して、人がやりたいことを十分な明快さとシンプルさで表現できるようにすることでしたこれを行う方法の微妙な点はすべて自動的に行われました。



過去30年間この概念に取り組んでおり、最初にMathematicaで 、次にWolfram | Alphaで必要な一連の技術を徐々に構築してきました。そして現在、これらすべてがWolfram言語で最終形式で提示されています。 先ほど述べたように 、Wolframは新しいタイプのプログラミング言語です。これは知識ベースのプログラミング言語であり、その哲学は可能な限り多くの知識を言語に組み込んで、すべてを可能な限り自動化することです。可能な限り。



Wolfram Programming Cloudは、クラウドプログラムのプログラミング、開発、展開に特化したWolfram言語アプリケーションです。



どのように機能しますか？ あなたは自分でそれを理解することができます ！ それを始めるのは非常に簡単です。 あなたの名前のブラウザからWolfram Programming Cloudに行き、Newをクリックしてください。 ノートブック （ラップトップ）と呼ばれるものを入手できます（はい、25年以上前にMathematicaシステム用に導入しました）。 その後、コードの入力を開始できます 。







このドキュメントのすべてはインタラクティブです。 何かを入力すると、すぐに計算してノートブックで直接結果を見ることができます。

まもなく、モバイルアプリケーションにクラウドオブジェクトを展開するのも簡単になります。 将来的には、組み込みWolframエンジン（Wolframエンジン）などが登場します。



それで、これはどういう意味ですか？ これは、プログラミングのプロセス全体と経済性を実際に変えるため、非常に重要だと思います。 私たちの会社でどれほどのことが変わっているのかを見ました。 Wolfram言語とWolframプログラミングクラウドが一緒に登場するにつれて、内部でそれらを使用しなければならない場所がますます形成されています。 そして、以前は数週間または数か月の間解決されていたプログラミングタスクが、数日またはそれ以下で突然行われたのを見るのは驚くべきことでした。



しかし、はるかに重要なのは、私が知識ベースの言語であるWolfram言語の概念全体が、プログラミングで初めてそのようなことを行うことであり、それは以前に検討することさえばかげていることです。 自社の例-Wolfram言語のおかげで、この言語の数百万行のコードで構成されるサービス、Wolfram | Alphaを作成できました。



しかし、今日起こっている最もエキサイティングなことは、Wolfram Programming Cloudの発売により、これらのすべてのテクノロジーが誰でも利用できるようになり、大規模から小規模までプロジェクトを実行できるようになったことです。



誰でも簡単にウェブブラウザにアクセスして、無料でWolfram言語でコードを書き始め、さらにWolframクラウドで小規模に展開できるように設計されています。 大規模なプロジェクトの場合、組織内で独自のWolfram Private Cloud（Wolfram Private Cloud）を取得する可能性も含め、さまざまな可能性があります。
