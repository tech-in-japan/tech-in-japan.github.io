Yandex.Maps API

最後の投稿で、私は控えめにYandex.Maps APIと呼ばれるイベントを発表しました。このイベントでは、APIマップの第2バージョンの基本概念に関するレポートを聞いたり、非公式の設定で開発者とチャットしたり、Cookieを食べたりできます。 IPAは4月26日にモスクワで開催されます。 来週の木曜日。



次に、 コンサートプログラムを公開します。



  • APIのアーキテクチャについてYandex.Map 2.0は、APIのビジュアルコンポーネントの開発グループの責任者であるAlexander Chupakhinによって語られます。 APIがどの部分で構成されているか、どのように機能し、相互作用するかを学習します。
  • Yandex.Maps APIの開発者であるAlexander Zinchukは、API 2.0がマップを構成してその動作を制御する方法を説明しています。 地図上でコントロールを追加、削除、移動する方法、使用できるジオオブジェクトの種類、設定方法を学習します。
  • Yandex.Maps APIの開発者であるAnton Korzunovが、API 2.0のグラフィックス機能について説明します。 API 2.0でサポートされる新しいテクノロジー、その方法とその場合にそれらを適用することをお勧めします。
  • Yandex.Maps APIの開発者であるMarina Stepanovaは、地図上に多くのポイントを配置する方法と、APIで新しいツールを使用する方法について説明します。
  • そして最後に、サイト上で自分の地図やその他の画像を表示する方法、およびAPIを使用してタイルを作成する方法とタイルを表示する方法について、個人的に説明します。
