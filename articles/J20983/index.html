Windows Vistaを高速化する方法

私たちの多くは、勤務中または必要に応じて、マイクロソフトの新しいオペレーティングシステムに対処する必要があります。 ご存知のように、ビデオカード、RAM、プロセッサ速度などのシステムリソースが非常に要求されます。 ただし、状況はわずかに修正できます。 既存のハードウェアでWindows Vistaを高速化する標準的な方法があります。 これらは主に、以前のバージョンのWindowsでよく知られているよく知られた方法です。



もちろん、最適化を行っても、数ギガバイトのRAMと高価なビデオカードを搭載した最速のプロセッサほど高速にVistaを動作させることはできません。 しかし、Windows XPがコンピューターで高速に実行されていて、Vistaが遅い場合は、これらのヒントが役立つ場合があります。



きれいな家



システムの「ブレーキ」の主な原因の1つは、バックグラウンドで実行され、RAMを詰まらせる外部アプリケーションが多いことです。 不要なものはすべて取り除く必要があります。



不要なサービスを無効にする

サービスは、通知なしに完全にサイレントにアクティブ化されるバックグラウンドプロセスです。 Vistaはデフォルトでできるだけ多くのプロセスを開始しようとするため、それらの中に不要なプロセスが存在する可能性があります。 それらを破棄するには、 [スタート] -[ 実行 ]からコマンドインターフェイスを実行し、 services.mscコマンドを実行します 。 サービスのリストが画面に表示されます。 それぞれのプロパティをダブルクリックすると表示されます。 そこで、デフォルトのダウンロードを無効にするなど、ダウンロードオプションを変更できます。



セカンダリログオン 、 リモートレジストリ 、 コンピューターブラウザー 、 分散トランザクションコーディネーターのKtmRm 、 タブレットPC入力サービス、 分散リンクトラッキングクライアントなど、ほとんど必要のないサービスの例を次に示します。 他にもたくさんあります。 リストを調べて、名前でインターネット検索を使用します。



ハードディスクの注文

ほとんどのユーザーと同様に、おそらくディスクスペースを食い尽くす余分なプログラムをたくさんインストールしているでしょう。 不要なプログラムを取り除くには、 CCleanerをお勧めします。CCleanerは 、Vistaにネイティブのアプリケーションの追加/削除ツールよりもはるかに高速で優れた動作をします。



ダウンロードの加速

Vistaが起動すると、多数のプログラムが自動的にロードされますが、その中には追加のプログラムもあります。 システム構成ユーティリティまたはWindows Defenderを使用して、混乱をクリーンアップできます。



最初の場合、手順は次のとおりです。

1. msconfigを開きます （メインメニューからクイック 検索機能を使用します。Windowsキーを押し、「System」と入力してEnterキーを押します）。

2. [ スタートアップ ]タブを選択します。

3.システムの起動時にロードすべきではないオブジェクトを無効にし、[ OK ]をクリックします。



Windows Defenderで同じことを行う方法。

1. Windows Defenderを起動します。

2.ツールバーの[ ツール ]メニューを選択します 。

3. ソフトウェアエクスプローラーを選択します。

4. すべてのユーザー向けのアイテムショー 。

5.アプリケーションとそのオプションを選択します：削除（削除）または非アクティブ化（無効）。



ハードディスクデフラグツール

ハードドライブ上のファイルの断片化は、システムの動作中に必然的に蓄積されます。 ファイルが個別にハードドライブの異なる場所に物理的に配置されているという事実により、ファイルへのアクセス時間が遅くなり、システム全体の動作が妨げられます。 ハードドライブ上のファイルを単一のクラスターに結合することを、最適化と呼びます。



Vistaはデフォルトで、週に一度バックグラウンドで自動的にデフラグするということを言わなければなりません。 しかし、大多数のユーザーがファイルを頻繁にデフラグする必要がないため、これはベアサービスのように思われ、このプロセスはシステムパフォーマンスに大きく影響します。 したがって、プロセスの自動開始をオフにし、必要に応じて手動でデフラグを開始することをお勧めします。



最適化を手動で開始するには、 マイコンピュータフォルダを開き、目的のドライブを右クリックして、[ プロパティ ]を選択し、[ ツール ]タブを選択します 。ここで[最適化]コマンドが表示されます。



残念ながら、Vistaデフラグツールは、XP以来慣れ親しんできた多くの追加の重要なオプションを奪われています。そのため、最適化を改善するには、Auslogicsの無料のディスクデフラグユーティリティなどの特別なプログラムに頼る必要があります。



ReadyBoostを使用する



ReadyBoostは、Windows Vistaの最も魅力的な機能の1つです。 簡単に言えば、外部ドライブを使用してキャッシュメモリを保存できます。 たとえば、高速のUSBフラッシュドライブを使用すると、Vistaはメモリダンプをダンプします。 すべての情報は自動的に暗号化されるため、フラッシュドライブの情報が失われた場合でも、あなたの秘密はあなたに残ります。



場合によっては、ReadyBoost機能によりオペレーティングシステムの速度を上げることができますが、これはもちろんオペレーティングメモリのアップグレードに代わるものではありません。



vLiteを使用する



ハードドライブの領域を解放するには、無料のvLiteプログラムをインストールできます。 オペレーティングシステムの配布サイズを約半分に減らし、Windows Media Player、Windows Mail、Windows Photo Viewer、MSNコンポーネント、デスクトップの壁紙、スライドショーなどの多くのコンポーネントを痕跡なしで破壊します。 これにより、ギガバイトのハードディスク容量が節約されます。



古いコンピューターでのVistaの高速化



「Vista」のシックなグラフィックインターフェースは目を楽しませますが、中央処理装置とビデオカードに非常に大きな負荷をかけます。 このため、古いコンピューターは「遅くなる」可能性があります。 アップグレードする必要はありません。システム設定を変更するだけです。



* Aeroを取り除きます。 お使いのコンピューターのビデオカードが弱すぎると判断した場合、オペレーティングシステム自体がこのインターフェイスを無効にします。 ただし、これは発生しない場合があります。 次に、[ ウィンドウの色と外観]ウィンドウ（デスクトップを右クリック）を起動し 、[ 色のオプションの詳細を表示するクラシック外観プロパティを開く ]を選択します 。 そこで、Aero以外のオプションを選択できます。



*それほど過激ではないオプションとして、 ガラスの透明度オプションをオフにします。これは、メモリ使用量の大半を占めます。



*リソースの負荷を減らすもう1つのオプションは、アニメーションやその他の視覚効果を無効にすることです。



*ウィジェットもRAMを使用するため、 サイドバーを無効にします。これを無効にすると、オペレーティングシステムの動作がさらに加速されます。
