GTARadio 4ジャストインタイム

約束通り、私たちは多くの人が尋ねたことをしました。 つまり、最近リリースされたGTA4のWeboramsのサポート。 GTA3 / SAのバージョン以降、本質的には何も変更されていませんが、GTARamaはキャッシュをシステムディレクトリではなくユーザーディレクトリに保存するようになりました。







一般的に、 ダウンロード 、GTAIV.exeでディレクトリに入れ、起動、ジャンル/ムード（下記参照）を選択、GTA4を起動、車に乗り、独立FMをオンにしてお楽しみください。







気分とジャンルの選び方





GTARameを実行している状態で、次のキーボードショートカットを使用します。



気分

Ctrl + F1-Fun-Calm

CTRL + F2-楽しい

CTRL + F3-楽しくエネルギッシュ

CTRL + F4-穏やかな

CTRL + F5-ニュートラル

CTRL + F6-激しい

CTRL + F7-悲しい落ち着き

CTRL + F8-悲しい

CTRL + F9-悲しい、エネルギッシュな



ジャンル

ALT + F1-任意

ALT + F2-ラップ

ALT + F3-リズムアンドブルース

ALT + F4-ロック

ALT + F5-クラシック

ALT + F6-エレクトロニクス

ALT + F7-代替

ALT + F8-ジャズ

ALT + F9-メタル



GTARamをダウンロードします。



PS





正式なトレーニングコースを希望し、C＃でGTARamを書き直したい学生のために、興味深い提案をすることができます。 :)
