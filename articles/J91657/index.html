Mac OS X + Dropboxでスクリーンショットを共有する

ほぼ毎月、スクリーンショットを作成して公開するためのさまざまなプログラムが表示されます。

ポピーでどうやってやったのか簡単に説明します。



デフォルトでは、ホットキーCommand-Shift-3 ... Command-Control-Shift-4を使用してスクリーンショットを作成するためのポップアッププログラムが組み込まれていることは多くの人にとって秘密ではありません（詳細はこちらをご覧ください ）。 スクリーンショットはデスクトップに保存されます。



また、多くの人々は、フォルダを共有できる素晴らしい製品dropbox.comがあることを知っています。

パブリックフォルダーのコンテキストメニューには、「ファイルへのリンクを送信する」という項目があります。



問題は小さいままです。Dropboxを配置し、スクリーンショットの作成をパブリックフォルダーにリダイレクトします。 これを行うには、ドロップボックスをインストールした後、ターミナルを開いてそこに入力します：



defaults write com.apple.screencapture location /Full/Path/To/Folder

killall SystemUIServer









（ 詳細はこちら ）

最初のコマンドでパスを設定し、2番目のコマンドでUIサービスをリロードして、ログアウトせずに更新が有効になるようにします。



これで、Dropboxフォルダーにスクリーンショットが作成されました。スクリーンショットを右クリックしてリンクを取得し、友人に提供できます。
