Meizu MX3の先行予約開始！

ハブラリュディ、おめでとうございます！ 9月2日、北京で新しいスマートフォンMeizu MX3を導入しました。本日、新製品の先行予約コレクションの発売を発表しました。 カットの下で、Meizu MX3に関する主な情報は、スマートフォンの仕様、機能、価格、バージョンです。









事前注文には3つのスマートフォンオプションがあります。



  • Meizu MX3 16 GB-17 990ルーブル
  • Meizu MX3 32 GB-19 990ルーブル
  • Meizu MX3 64 GB-21 990ルーブル


バージョンMeizu MX3 128 GBは後で利用可能になり、そのコストは23,990ルーブルになります。



100％の前払いで事前注文すると、ブランドヘッドセットMEIZU EP-21、およびサイトmymeizu.ruのオンラインストアで購入するための1000ルーブルのクーポン（アクセサリまたはセカンドスマートフォンを割引価格で購入できる機能）および注文されたMX3のロシア全土への迅速な無料配送



Meizu MX3の先行予約





Meizu MX3の主な機能：



  • Wolfson WM5102高品質の音楽チップとDirac HDテクノロジーは、おなじみのメロディーの新しい側面をヘッドフォンで開きます
  • Flyme 3.0-スマートフォンとの通信による新しい体験と最大限の利便性
  • 8メガピクセルの解像度と「シャッター遅延ゼロ」の技術を備えた最新のSony Exmorセンサーに基づくカメラにより、高品質の写真を即座に受信できます。
  • 画像および写真処理用の富士通の強力な4チャネルチップ
  • 新世代のノイズキャンセリングチップを備えた3つのマイクのシステム。 話すときの完璧な可聴性
  • ステンレス鋼フレームとマグネシウム合金パネルは、構造全体と軽量ハウジング全体に高い強度を提供します
  • ディスプレイとケースの端の間の超薄型フレーム（わずか2.9 mm）。これにより、スマートフォンは手に快適に横たわり、大きな画面の対角線になります。
  • Sharp New Mode 2高品質FullHDディスプレイ：優れた視野角と自然な飽和色、スムーズな移行を伴う2048レベルの輝度グラデーション
  • バッテリーの消費電力を制御する特別なチップ
  • 設定メニューから、スマートフォンの保証を失うことなく「スーパーユーザー」権限（ROOT）を取得する機能
  • 明るい色の取り外し可能な蓋：紫、青、薄緑、ピンク、オレンジ

  • オペレーティングシステム ：Android 4.2とFlyme 3.0 Shell
  • 通信： GSM / GPRS / EDGE / 3G（WCDMA）/ HSPA + 850/900/1800 MHz（2G）、850/2100 MHz（3G）
  • 画面： Sharp New Mode 2、対角5.1インチ、解像度1800×1080ピクセル、PPI 415、コントラスト1000：1、最大輝度450 cd / m、輝度2048階調、TOLテクノロジー（単一ガラスモジュールとセンサー）、省エネPSRテクノロジー、保護ガラスゴリラガラス2

  • RAM： 2 GB、DDR3
  • データストレージ用メモリ： 16/32/64/128 GB

  • マルチメディア： Wolfson WM5102サウンドチップ、Dirac HDテクノロジー、3つのマイクを備えたSmart PA音響システム、富士通の4チャンネル画像処理チップ

  • ナビゲーション： GPS / A-GPS / GLONASS、電子コンパス
  • オプション：タッチセンサー、光センサー、重力センサー、電子コンパス、ジャイロスコープ
  • バッテリー： Sony 2400 mAh、電源管理用デュアルチップ
  • 寸法： 139.0 x 71.9 x 9.1 mm
