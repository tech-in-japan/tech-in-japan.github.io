オペレーティングシステム用のAndroid SDK（ https://developer.android.com/sdk/index.html ）をダウンロードします（Windows XP SP3 RU x86を使用しています）-執筆時点でadt-bundle-windows-x86-20131030.zip。 \ Eclipse \ eclipse.exeを展開し、ウィンドウ-> Android SDK Managerを選択し、

フォルダーに移動します。

  • ドライバーのみをダウンロードした場合-usb_driver
  • Android SDKの場合、sdk \ extras \ google \ usb_driver \
