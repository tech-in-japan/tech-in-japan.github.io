Developer Expressは最近、.NET開発者向けのコントロールセットの次のメジャーバージョンであるDXperience v2011 vol 1のリリースを発表しました。 伝統的に、このバージョンは前のリリースから6か月後にリリースされ、2011年の最初のメジャーアップデートです。 また、従来、Windows Forms開発者とASP.NET開発者の両方、およびWPFとSilverlightプラットフォームに基づいて製品を作成する開発者向けに、多くの異なる改善が含まれています。

XAF-ビジネスアプリケーションをすばやく作成するためのフレームワーク



新しいリリースでは、ビジネスアプリケーションを開発するためのフレームワーク-XAF-は無視されませんでした。 最も重要な改善点は、 Workflowアドオンに実装されているWindows Workflow Foundation （WWF）の完全サポートです。 このモジュールは、XAFアプリケーションのワークフローのモデリングを簡素化し、開発者にWWFの概念に関する基本的なアイデアのみを要求します。 アプリケーションのエンドユーザーは、ワークフローを作成および変更することもできます。







2番目の新しいXAFモジュールはState Machineです。 単純なシナリオでは、Workflowの軽量の代替として位置付けます。 ステートマシンモジュールを使用すると、一連の状態と特定の状態間の遷移ルール​​を持つオブジェクトを簡単に操作できます。 たとえば、 タスクオブジェクト：







新しい設計ツールであるModel Merge Toolは、アプリケーションモデルの「レイヤー」間で設定を転送するように設計されています。 モジュール間でデータを転送することに加えて、このツールはモデルにユーザー設定を読み込んで統合できます。 これは、アプリケーションを起動し、その外観（エディターの場所、表示と列の順序、チャート表示オプションなど）を構成し、これらの設定をプロジェクトに接続してすべてのユーザーがデフォルトで使用できることを意味します。 また、「高度な」エンドユーザーから設定を借用して、アプリケーションの新しいバージョンのすべてのユーザーに設定を適用することもできます。 このツールの使用方法を示す小さなビデオが当社のWebサイトで入手できます。



モデルエディターは 、多くのユーザビリティの改善を受けています。

  • ギャラリーからの画像の選択。
  • 視覚的表現デザイナーの基準言語 。
  • マスクをすぐにチェックできる入力マスクエディター。
  • ノードをグループ化する機能。
  • 関連付けられたノードへの迅速な移行のための仮想リンクノード。たとえば、現在のオブジェクトのすべてのビューを簡単に見つけることができます。
  • アプリケーション管理者向けのスタンドアロンモデルエディターは、コマンドラインキーなしで起動できるようになりました（構成ファイルはGUIを介して選択されます）。






最適化に関して多くの作業が行われました。 XAFアプリケーションの起動が速くなり、ASP.NETバージョンではトラフィックが大幅に削減され、Webサーバーでページを生成するプロセスが最適化されました。 テストの結果によると、トラフィックは半分に減少し、ページ生成時間は3分の1減少しました。



XAFと一緒に、XAFアプリケーションの機能テスト用のフレームワークであるEasyTestを提供します。 新しいバージョンでは、組み込みのスクリプト言語EasyTestのコマンドセットを拡張しました。 これで、同じコマンドシーケンスを使用して、アプリケーションのWindowsフォームとASP.NETバージョンの両方をテストできます。

Visual Studio LightSwitchのレポートジェネレーター





皆さんの多くは、Microsoft Visual Studioファミリーの新製品、コードネームLightSwitch について聞いたことがあるでしょう。 この段階では、レポートを生成するための組み込みモジュールがないため、このギャップを埋めることにしました。



バージョン11.1では、 LightSwitchアドオンのXtraReportsアドオンを試して、この新しい開発環境でレポートをすばやく作成できます。 このモジュールの実装中に、レポートの開発がLightSwitchでのアプリケーション開発に採用されたのと同じアプローチに基づいているように、すべてを行ったことに注意する必要があります。







同時に、データバインディング、グループ化、フィルター処理、並べ替え（LightSwitchクエリを使用してサーバー上で実行）など、レポート作成の一般的なタスクをすべて解決できます。結果のSilverlightアプリケーションには、エンドユーザーに必要なすべての機能が含まれます。







この新製品に興味がある場合は、Visual Studio LightSwitchでのレポート作成に関する詳細をさらに開示したいと考えています。
