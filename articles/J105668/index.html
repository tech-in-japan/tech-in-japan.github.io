ロシアでは、Blackberryクライアントサブシステムをハッキングしました

有名な会社であるElcomsoftは、RIM BlackBerryコミュニケーター保護システムのハッキングの成功について報告しています。 これまでのところ、サーバー側ではなくクライアントサブシステムについてのみ説明していますが、これでも大きな成果です。多くの国のプログラマーがBlackBerryをハッキングしようとしたためです。 誰かが成功した場合、これは報告されませんでした。したがって、Elcomsoftはこれを実現した最初の企業と見なされます。



ハッキングが成功した直後、同社はElcomsoft Phone Password Breakerプログラムの新しいバージョンをリリースしました。 このソフトウェアの以前のバージョンでは、Apple iPhoneおよびiPod Touchモバイルデバイスのパスワードで保護されたバックアップをクラックすることが可能でした。 現在、プログラムの新しいバージョンでは、パスワードで保護されたバックアップをBlackBerryでハッキングする可能性もあります。 このようなアーカイブには、メッセージ、電子メール、連絡先、Webページへのアクセスのログ、さまざまな設定が含まれます。



開発者によると、彼らのソフトウェアを使用すると、これらのデバイスのパスワードを「回復」できます。 しかし、自分のモバイルデバイスのバックアップアーカイブからパスワードを忘れた人だけがプログラムで動作しないことは明らかです。 しかし、何をすべきか-それは常にそのようなソフトウェアの仕事の裏返しでした。



ところで、プログラムの新しいバージョンは、システムにインストールされたグラフィックアダプターの使用を提供します。 現在、モバイルデバイスのシステムストレージからすべてのデータを読み取り、暗号化できるのは、メールアカウントとインターネットサービスアカウントアカウントの暗号化キーとさまざまなパスワードの両方を含む、唯一のプログラムであるElcomsoft Phone Password Breakerです。



興味深いことに、Elcomsoft Phone Password Breakerは、Apple iTunesデスクトップソフトウェアまたはBlackBerry Desktop Softwareを使用せずに自律的に動作します。 このソフトウェアは、Windows XP、Windows Server 2003、Windows Server 2008、Windows Vista、Windows 7などのOSで動作します。



製造業者は、このソフトウェアパッケージが法執行官にとって非常に有用であり、犯罪者のモバイルデバイスの以前に閉じられたアーカイブデータにアクセスできるようになると主張しています。 もちろん、ロシアにはBlackBerryやiPhoneを操作する違反者はそれほど多くありません。
