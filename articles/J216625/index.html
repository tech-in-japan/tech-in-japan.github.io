アレクセイ・キブカロと不可抗力

私たちのトレーニングセンターと長い間密接に協力してきたスペシャリストのアレクセイ・キブカロを取り巻く多くの誇大広告は、私が意見を表明するようになった日（1991年）から私、ズヴェズダとSの長でした。



第一に、多くの「ジャーナリスト」が話題から外れていることは印象的です。 ほとんどすべての出版物は、Alexey Kibkaloを開発者（プログラマー）と呼んでいます。 はい、Lesha、「少なくとも1行のコード」を書いた場合、それらはすべて1つのノートブックに収まります。 Alexey Kibkaloは、高級建築家、実装者、トレーナー、コンサルタント、ソリューション開発者ですが、コードではありません！



第二に、「取引の秘密」などの迷惑な見出し。 アレクセイの何かの販売について一言も述べていない公式レポートはありません！ あなたがこれらの文書を信じていたとしても、彼はそれらを「転送」しましたが、それらを販売しませんでした！



私は昨日の前夜に学んだことすべてに非常に驚いています:(。このようなことが起こり得て、アレクセイを職場でも個人でも10年以上知っていたとしても、彼は物質的な目標を追求しなかったと思います！小規模なトレーニングセンターでの無料のウェビナー、明らかに数百万の料金ではない...



はい、Microsoftを辞めたAlexeyには、多くの興味深い有益な知識がありました。 しかし、彼はそれらを最も巧妙かつ合法的に共有しました。 そのため、最新のWindows Server 2012仮想化に関するウェビナーを実施したのは米国での発売後のみでしたが、Alexeyには数か月前から必要な知識がありました。



アレクセイキブカロの責任と品位は、すでに夜（米国で逮捕された日）に弁護士が私に何が起きているかを知らせたという事実によっても示されています。 それで、そのような瞬間でさえ、アレクセイは彼のフルタイムのトレーニングのために月曜日に来ようとしているそれらの10人の学生について考えました...



すべてがすぐに明らかになることを願っています！



少し待ってみましょう...



リンクをたどって、Alexeyが実施したトレーニングとウェビナーに関する情報（ 記録を含む）、 および TechNetでの仮想化に関する彼のブログをご覧ください。 アレクセイの専門家としての簡単な要約もあります。 多くの人は、アレックスが多くの便利なことをして、知識を共有して喜んでいたことをすでに知っています！
