Facebookは、モバイルWebで最も人気のあるソーシャルネットワークになっています。









2010年1月26日-ノルウェー、オスロ



モバイルインターネットの使用に関する月次レポートによると、FacebookはOpera Miniユーザーの間で世界で最も人気のあるソーシャルネットワークになっています。 2009年には、Facebookのユニークビジター数が600％増加し、VKontakteが主導的地位からシフトすることができました。 Twitterは成長のリーダーであることが判明しました。2009年にこのマイクロブログサービスを訪れるOpera Miniのユニークユーザーの数は2800％増加しました。



月次レポートには、世界および個々の地域におけるモバイルインターネットの動向に関する情報が含まれています。 そのフルバージョンは、 www.opera.com / smw （英語）で入手できます。



一般レポートデータ

-2009年12月のOpera Miniユーザー数は4630万人で、11月より11％、2008年12月より159％増加しました。



-1か月あたり4630万人のユーザーが207億以上のページを閲覧しました。 これは、11月より10.1％、2008年12月より112％増加しています。



-2009年12月、Opera Miniユーザーは世界中のオペレーターに対して3億1500万メガバイトのトラフィックを生成しました-11月の結果は10.5％を超えています。 Opera Miniでは、データを最大90％圧縮できます。 したがって、12月にブラウザで処理されたデータの合計量は2.9ペタバイトになりました。 2008年12月と比較して、トラフィックは206％増加しました。



-Opera Miniユーザー数の上位10か国は、前月と比較して変更されています。 12月には、ロシア、インドネシア、インド、ウクライナ、中国、南アフリカ、米国、ベトナム、ナイジェリア、英国が含まれていました。 ウクライナのユーザー数は中国よりも多くなり、ベトナムは急速に成長を続け、英国を追い越し、ナイジェリアは3か月ぶりにトップ10リストに戻り、ポーランドを初めて追い出しました。



-2009年、Opera Miniのユーザー数に基づいて、Facebookは世界で最も人気のあるソーシャルネットワークになりました。 ただし、ロシア語圏の国では、選手権はまだVKontakteに属します。



-Twitterは急成長のタイトルを獲得しました。2009年にこのサービスにアクセスするOpera Miniユーザーの数は2800％増加しました。



-2009年、Friendsterを除くほぼすべてのソーシャルネットワークで、ユーザー数が大幅に増加しました。



-2009年12月までにロシアのOpera Miniを使用して表示されたWebページの数は94.1％増加し、ブラウザのユニークユーザー数は78.7％増加しました（年初からのデータと比較）。 平均して、各ユーザーは1か月あたり425ページを表示し、各ページのサイズは平均21 KB（圧縮後）でした。



-ウクライナの年間のOpera Miniユーザー数は140.6％増加し、閲覧したページは275.8％増加しました。 各ユーザーは平均700ページを表示し、平均ページサイズは19 KB（圧縮後）でした。 Yahoo! YouTubeを押し込み、最もアクセス数の多いサイトのトップ10に戻りました。



「4年前、2005年1月、私たちは最初にOpera Miniを国際社会に紹介しました」とOpera Softwareの共同設立者であるJon von Techner氏は回想します。 -人々は、これに使用するもの（携帯電話またはコンピューター）に関係なく、同じサイトにアクセスしたいと考えています。 Opera Miniユーザーの間でのFacebookとTwitterの成功は、手元のデバイスが変わっても好みが変わらないことを示しています。

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザは、その高速性、コンパクトさ、および他のブラウザに比べてWeb標準のサポートが優れているため、ユーザーおよびメーカーから世界的に認知されています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
