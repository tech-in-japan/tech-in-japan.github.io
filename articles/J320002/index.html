Linux 2017：最も有望なディストリビューション

新しい年が到来しました。つまり、未来に目を向け、最良で最も有望なLinuxディストリビューションを見つける時です。







Linuxディストリビューションは、多くの場合タスク指向です。 したがって、オペレーティングシステムのリストを作成して「彼らは最高です」と言うだけでは機能しません。 ここでは、Linuxの使用分野がいくつか強調されており、2017年にニッチ市場で最初になる可能性が高いディストリビューションが選択されています。



システム管理者向けのベストディストリビューション：Parrot Linux





Parrot LinuxはDebianに基づいており、侵入システムをテストするためのほぼすべての考えられるツールを提供します



すべての管理者には、常に多くの作業があります。 良い道具一式がなければ、彼の日々は絶え間ない強さのテストであり、継続的なレースです。 ただし、多くのLinuxディストリビューションが用意されています。 その1つがParrot Linuxです。 2017年に本格的に人気が出ると確信しています。



このディストリビューションはDebianをベースにしており、不正アクセスからシステムのセキュリティをテストするための膨大な量のツールを提供します。 さらに、暗号化およびコンピューターフォレンジックの分野のツール、クラウドサービスおよび匿名性を保証するパッケージを操作するためのツールを見つけることができます。 開発者向けのものもあり、時間を整理するためのプログラムもあります。 これらすべて（実際、ツールの海があります）は、安定した、実績のあるシステムに基づいて機能します。 その結果、情報セキュリティの専門家やネットワーク管理者に最適なディストリビューションができました。



Parrot Linuxは現在、 Distrowatchのランキングで57位にランクされています。 年末までに、この分布が格付けの最上位まで大きく前進するのを観察できるようになると思います。



ここでは、Parrot Linuxについて詳しく読むことができます。



ベストライトウェイトディストリビューション：LXLE





LXLEは、控えめなリソース要件と優れた機能の優れた組み合わせです。



間違いなく、LXLEは2017年に多くの人が選ぶ軽量なディストリビューションになります。 なんで？ その中のすべてが非常に簡単だからです。 LXLEは、適切な機能と控えめなシステム要件を兼ね備えています。 言い換えれば、これはスペースをほとんどとらないディストリビューションですが、コンピューターで完全に作業することができます。



LXLEでは、Linuxデスクトップリリースに必要なすべてを見つけることができます。 このシステムは家庭に非常に適していますが、最新のコンピューターがその制御下で動作することはできません（完全に関連するシステムは言うまでもありません）。



LXLEはUbuntu 16.04に基づいており（その結果、長期的な技術サポートが提供されます）、ここでLXDEデスクトップマネージャーが使用されています。LXDEデスクトップマネージャーは、その存在から10年以上にわたって多くの人に馴染みがあり、複雑ではありません。



LXDEをインストールすると、LibreOfficeやGimpなどの多くの標準ツールが手元にあります。 唯一のことは、最新のブラウザを個別にインストールする必要があることです。



LXLEは、Distrowatchリストで16位になりました。 すべての兆候は、年の半ばまでに彼女がトップ10に入る可能性があることです。 LXLEの詳細については、 こちらをご覧ください 。



最高のデスクトップ配布：小学校OS





Elementary OS Lokiの利点は、見た目の良さだけではありません。 このOSは安定していて便利で、作成者は細部にまで気を配っています。



おそらく私はこのディストリビューションに偏っているかもしれませんが、 Elementary OS Lokiは不可能なことをし、最初からLinux Mintを投げ、2017年に「最高のデスクトップディストリビューション」といううらやましいタイトルを受け取ると確信しています。 その場合、Linux MintがDistrowatchランキングで常に勝つことを考えると、Elementary OSは非常に印象的な偉業を成し遂げます。



エレメンタリーOSは現在6位にあります（そしてLinux Mintは1位に君臨し続けています）。 Elementary OSがMintを打倒できる可能性はありますか？ はい、そうです。 Lokiは、最も美しいLinuxディストリビューションの1つとしての地位を確立しただけでなく、さらに、このシステムは安定しており、便利で、理解しやすいものです。



Elementary OSデスクトップはMacに非常に似ていると考える人もいるかもしれません。 ただし、このような比較は、ユーザーの潜在的な使いやすさを示すため、システムのプラスになります。 もちろん、Lokiの外観のカスタマイズはOS Xのチンキほど制限されていないため、必要なことを正確に行うことができます。



これが Elementary OSに関する私の資料です。



何が何なのかを知っている人に最適なディストリビューション：Gentoo





Gentooにはユーザーのスキルが必要ですが、このディストリビューションを選択した人の努力は報われます。彼らは必要なものを正確に手に入れるでしょう。



このカテゴリのディストリビューションは、Linuxの知識を誇示したい人に適しています。 このようなオペレーティングシステムは、Linuxを他の人よりもよく理解し、自分のニーズに合わせて大幅に強化できるディストリビューションを探している人を対象としています。 これが彼らがLinuxに期待するものであるとき、 Gentooだけが思い浮かびます。



Gentooは、特定のハードウェアのソースコードからパッケージをコンパイルすることに基づくLinuxディストリビューションです。 OSをインストールした後、ユーザーは必要なものをすべて個別に構築する必要があります。 これには、Linuxのかなり高いレベルの理解だけでなく、忍耐と時間も必要です。 しかし、最終的には、必要なものだけを手に入れることができます。 Gentooは新しいプロジェクトではなく、長年にわたって使用されてきました。 しかし、とりわけ、あなたが自分や他の人に自分がLinuxの専門家であることを証明したい場合、Gentooなしではできません。



最適なIoTディストリビューション：Snappy Ubuntu Core





スナップパッケージを使用することで、Snappy Ubuntu Coreはソフトウェアのインストールを簡素化し、アップデートの頭痛を取り除きます。 その結果、IoTに最適なOSができました



今、私たちは小さなフォームファクターのデバイスについて話しています。 モノのインターネット、またはIoTは、Linux組み込みシステムに比類のない領域です。 IoTに適した多くのディストリビューションがあります。 ただし、2017年はSnappy Ubuntu Coreの年になると確信しています。



スナップパッケージを使用すると、Snappy Ubuntu Coreでソフトウェアを簡単にインストールおよび更新できます。 更新による依存関係や中断を心配する必要はありません。



その結果、IoTに最適なソフトウェアプラットフォームができました。 Ubuntu Snappy Coreは、コンピューター愛好家（Rasberry Piなど）、Erle-Copterドローン、Dell Edge Gateway、Nextcloud Box、LimeSDRのボードにすでに搭載されています。



中小規模の組織に最適なサーバー配布：CentOS





CentOSは信頼できるサーバープラットフォームです



CentOSが中小規模の組織のサーバー上で歓迎されるゲストであるという事実には驚くべきことは何もありません。 この状況には非常に重要な理由が1つあります。CentOSはRed Hat Enterprise Linux（RHEL）のソースコードに基づいています。 このため、CentOSを選択する場合、信頼性の高い実績のあるサーバープラットフォームがあることは間違いありません。



RHELとCentOSの主な違いは（異なるマーケティング戦略に加えて）サポートです。 RHELユーザーには、公式のRed Hatテクニカルサポートが提供されます。 ただし、2004年以来、CentOSの「技術サポート」は熱狂的なファンの巨大なコミュニティです。



そのため、サーバーをオープンソースプラットフォームに切り替える中小企業の所有者である場合は、まずCentOSを検討する必要があります。



エンタープライズサーバーのベストディストリビューション：RHEL





Red Hatは企業顧客のニーズに適合



繰り返しますが、驚くべきことは何もないと言えます。 たとえば、 SUSEは企業のOS市場を獲得するために多大な努力を払っていますが、いつかこうした努力が名声の高みにつながるでしょうが、残念ながら今年はそうではありません。 2017年、 Red Hat Enterprise Linux （RHEL）は、最も人気のある企業向けディストリビューションの中で最初の位置を占めます。



Gartnerによると、RHELは大企業向けのLinuxディストリビューション市場の67％を所有しており、RHELサブスクリプションでRed Hatの収益の約75％を獲得しています。 この状況には多くの理由があります。 そのため、Red Hatは企業顧客に必要なものを提供しますが、さらに、多くのオープンソースプロジェクトの開発に多大な努力を払っています。



Red Hatは、Linuxが何であり、企業セクターが何であるかを知っています。 Red Hatは、Fortune 500の多くの企業（ING、Sprint、Bayer Business Services、Atos、Amadeus、Etradeなど）を信頼しています。 RHELディストリビューションにより、多くの開発者は、クラウドシステムを扱う分野のセキュリティ、統合、管理の分野で新しいレベルに到達しました。



さらに、今年、Red HatはIoTの分野に多くのエネルギーを投資すると思います。 ただし、2017年の終わりまでにSUSEがRed Hatの市場シェアをもう少し食い止めることになっても驚かないでしょう。



選択はあなた次第です



Linuxプラットフォームの最も注目すべき側面の1つは、最終的にはユーザーに選択を任せることです。 数百のシステムから選択でき、その多くは最も洗練された要件を完全に満たします。 ただし、上記のディストリビューションのいずれかを使用している場合、それはあなたを失望させないと確信しています。
