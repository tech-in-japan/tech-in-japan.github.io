Linux 2018：最も有望なディストリビューション

2018年の最も有望なLinuxディストリビューションに特化したlinux.comからのJack Wallenの記事の翻訳をご紹介します。 約12か月前に、同様のレビューを共有しました。 年間で何が変わったのですか？







ジャックは、2017年には、Linuxの世界で、ディストリビューション数の増加に関する傾向を観察できると言います。 彼は今年、サーバーコンピューティングとデスクトップPCの両方で、エコシステムの安定性が向上し、Linuxの普及率が高まると予想しています。



今年オープンソースプラットフォームに切り替える予定の人、または使用しているものを新しいものに変更したい人のために、何を選ぶべきでしょうか。 Distrowatchに行き、ディストリビューションの評価を見ると、そこに見られる主なことは想像を絶する可能性の範囲です。 豊富な選択肢がありますが、必要なものを正確に選択することは容易ではないということです。



それでは、今年どのLinuxディストリビューションに注目すべきでしょうか？ ジャックはこの主題について考えを持ち、この記事でそれらについて話したいと思っています。 過去1年と同様、ここではシステムをいくつかのカテゴリに分けて検討します。 つまり、管理者向けのディストリビューション、軽量およびデスクトップシステム、プロフェッショナル向けおよびIoT向けのLinux、およびサーバープラットフォームについて説明します。



システム管理者向けの最適な配布：Debian



Debianは 、最高のディストリビューションのリストにはめったに見られません。 なんで？ DebianがUbuntuの基盤であり（これが多くの他のディストリビューションの基盤である）、実際、Debianは多くの評価の上位にあるべきであることが明らかになりました。 そして、この配布はシステム管理者と何の関係がありますか？ これは2つの重要な理由で真実だと思います。



  • 使いやすさ。

  • 極端な安定性。



Debianはdpkgおよびaptパッケージマネージャーを使用するため、システムは非常に使いやすい環境を作成します。 さらに、Debianは最も安定したLinuxベースのプラットフォームの1つです。 その結果、多くの環境とユースケースに優れたディストリビューションを提供しています。 たとえば、ここ-サーバーとデスクトップコンピューター、ここ-テストとソフトウェア開発。 Debianには、昨年の同様のレビューParrot Linuxで指摘された管理者向けディストリビューションにあるアプリケーションの数は含まれていませんが、必要なものをすべてDebianに追加するのは非常に簡単です。 幸いなことに、誰でも適切なグラフィカル環境（Cinnamon、GNOME、KDE、LXDE、Mate、またはXfce）を使用してDebianをインストールできます。 その結果、Debianを選択することで、誰でもインターフェースがニーズに合っていることを確信できます。









GNOMEデスクトップを備えたDebian 9.3



Debianは現在、Distrowatchランキングで2位になっています。 配布キットをダウンロードしてインストールし、ニーズに合わせて構成し、安定した便利な操作をお楽しみください。 このシステムは、他のシステムと比べてそれほどファッショナブルではないように見えますが、Debianはシステム管理者の夢を現実にするディストリビューションにすぎません。



ベストライトウェイトディストリビューション：Lubuntu



軽量ディストリビューションは非常に特定の目的に役立ちます-通常は最も強力なコンピューターではない、時代遅れの人々に第二の命を与えること しかし、これは、これらの分布が古いハードウェアの計算でのみ考慮されるべきであることを意味するものではありません。 主な目標が速度である場合、軽量のカテゴリからの配布は、現代のコンピューターで非常に役立ちます。



2018年の軽量ディストリビューションのリストのトップはLubuntuです。 多くのオプションがありますが、このディストリビューションの特徴である開発のしやすさに近いものはごくわずかです。 Lubuntuのシステム要件は、たとえばPuppy Linuxほど控えめではありませんが、Ubuntuファミリーのメンバーが目の前にあり、それと連携することで「大食い」を補います。 ただし、最新ではないコンピューターではLubuntuの配布が遅すぎることを恐れないでください。 それでも、Lubuntuのシステム要件は非常に適切です。 つまり、Pentium 4、Pentium MまたはAMD K8プロセッサ、ローカルアプリケーションでの作業用に約512 MBのメモリ、またはインターネット（Youtube、Google +、Google Drive、Facebook）での通常の作業用に約1 GBについて話しています。



LubuntuはLXDEデスクトップを使用しています。つまり、Linuxの初心者が問題に遭遇することはありません。 ディストリビューションに含まれるアプリケーション（Abiword、Gnumeric、Firefoxなど）は、高速性、利便性、シンプルさが特徴です。

最高のデスクトップ配布：小学校OS



2年連続で、 Elementary OSは最高のデスクトップディストリビューションのリストのトップです。 多くの場合、デスクトップシステムのリーダーはLinux Mintです （これは理解できます）。 しかし、私の意見では、Elementary OSの使いやすさと安定性に勝るものはありません。



これの確認は私に起こった1つの物語です。 Ubuntu 17.10のリリースにより、Canonicalディストリビューションに戻ることを余儀なくされると確信していました。 GNOMEを備えた新しいUbuntuに切り替えて間もなく、エレメンタリーOSの利便性と信頼性が不足していることがわかりました。 その結果、Ubuntuで2週間過ごした後、私はElementary OSに戻りました。









Elementary OSのPanteonデスクトップ-芸術作品



Elementary OSを試す人は誰でもすぐに自宅にいるように感じるでしょう。 Panteonデスクトップは、細部へのこだわり、ユーザーのニーズへのフォーカス、利便性の優れた組み合わせの例です。 同時に、更新ごとに改善されるだけです。

Elementary OSはDistrowatchランキングで6位にランクされていますが、このディストリビューションは今年の終わりまでに少なくとも3行目に達すると思います。



小学校の開発者は、ユーザーが何を望んでいるかをよく知っています。 彼らは、OSで作業している人の話を聞き、適切な方向でシステムを開発します。 ただし、エレメンタリーOSはすでに優れた状態です。 その結果、開発者はシステムにわずかな改善しかできないと感じています。 適切に設計されたインターフェイスを提供するデスクトップディストリビューションを必要とする人は、Elementary OSに勝るものを見つけるのが非常に難しいことに気付くでしょう。 優れた信頼性と使いやすさを備えたシステムを探しているなら、Elementary OSはあなたのために特別に作成されました。



何が何であるかを知っている人に最適なディストリビューション：Linux From Scratch



非常に長い間、 Gentooディストリビューションは、Linuxに精通しており、知識やスキルを他の人に見せることを嫌う人々にとってはディストリビューションのリストのトップにいました。 しかし、Gentooがプロフェッショナル向けディストリビューションカテゴリの真のリーダーであるLinux From Scratchに 1位を獲得する時が来たと思います。 LFSは実際にはディストリビューションではなく、ユーザーが独自のディストリビューションの作成を支援するプロジェクトであるため、これはあまり正直ではないように思えるかもしれません。 しかし、Linuxをどれだけよく知っているかをすべての人に示すために、すべての困難を示さないことに真剣に取り組んでいる場合、自分で配布キットを作成するよりも良い方法はありますか？ LFSプロジェクトのおかげで、システムを文字通りゼロから作成し、ソースコードからすべてを完全に組み立てることができます。 したがって、本当に必要な場合は、 LFS マニュアルをダウンロードして、独自の配布を開始してください。



最適なIoTディストリビューション：Ubuntu Core



2年連続で、このカテゴリでの勝利はUbuntu Coreになります。他のオプションはここにはありません。 Ubuntu Coreは、組み込みシステムおよびIoTプロジェクトで使用するために特別に設計されたUbuntuの最小限のトランザクションバージョンです。 Ubuntu CoreがIoTに非常に適している理由は、システムがスナップパッケージ-基本システムに干渉することなくインストールできるユニバーサルパッケージに特別な注意を払うためです。 これらのパッケージには、動作するために必要なすべてのもの（依存関係を含む）が含まれているため、それらを操作している間、それらをインストールするとOSやインストールされているソフトウェアに違反することを心配する必要はありません さらに、これらのパッケージは、隔離された環境で非常に簡単に更新および実行できるため、IoTの優れたソリューションになります。



Ubuntu Coreのもう1つのセキュリティ関連機能は、ログインメカニズムです。 Ubuntu CoreはUbuntu One sshキーで動作し、すべてがシステムに入る唯一の方法がUbuntu Oneアカウントとsshキーを使用するように構成されています。 これにより、IoTデバイスを操作する際に高レベルのセキュリティを確保できます。

最高のサーバー配布：Ubuntuサーバー



この分野では、すべてが少し混乱しているようです。 サーバー配布の主なものはサポートです。 商用サポートが必要な場合は、一見Red Hat Enterprise Linuxを選択することをお勧めします。 Red Hatは長年にわたって、RHELが最も強力な企業サーバープラットフォームの1つであるだけでなく、世界で最も収益性の高いオープンソースビジネス（年間収益20億超）であることを証明しています。



ただし、RHELは唯一のサーバー配布とはほど遠いものです。 実際、Red Hatはエンタープライズサーバーシステムの絶対的なリーダーでさえありません。 Amazon Elastic Compute Cloudのみでのクラウドインストールの統計を見ると、UbuntuはRHELよりもはるかに優れています。 Cloud Marketによると、EC2のデータによると、RHELの展開数は10万件未満であり、Ubuntuの展開数は20万件を超えています。 これは非常に深刻な結果です。



その結果、Ubuntuはクラウドサーバーの分野で議論の余地のないリーダーです。 そして、ここでUbuntuとコンテナを使った作業のシンプルさを追加すると、Ubuntu Serverディストリビューションがサーバーシステムのカテゴリーで地滑り的な勝利を獲得したことが非常に明らかになります。 また、商用サポートが必要な場合、CanonicalはUbuntu Advantageを提供できます。



誰かが気に入らないかもしれないUbuntu Serverの唯一の機能は、デフォルトでこのシステムがテキストインターフェイスのみを使用することです。 必要に応じて、ここにグラフィカルシェルをインストールできますが、Ubuntu Serverコマンドラインの操作は非常に簡単です（おそらく、これはすべてのLinux管理者が持つべきスキルの1つです）。

選択はあなた次第です



すでに述べたように、ここで提示される有望な分布のリストは、非常に主観的な基準に基づいています。 ただし、アクティビティの特定の領域に興味があり、この領域で最初のOSを探している場合は、説明したシステムを試してください。 それぞれが特定のタスク用に設計されており、それぞれが他の多くの問題よりもはるかにこれらの問題を解決します。 ここで何を話しているのか疑問に思うかもしれませんが、Linuxの世界で誰もが正確に必要なものを見つけることができると主張することはほとんどないでしょう。
