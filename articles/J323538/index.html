Citrix XenServerバージョン7.1ハイパーバイザーがリリースされました

XenServer 7.1では、バージョン7.0と比較していくつかのユニークな機能が導入されています。 現在、Citrix XenApp / XenDesktopとの緊密な統合により、XenServerはCitrixコーポレートモビリティアプリケーションを操作するのに最適なハイパーバイザーであり、10年間フルサービスをサポートしている唯一のハイパーバイザーです。 XenServerバージョン7.1の主な革新は、カットの下にあります。







まず、XenServer 7.1は、インタラクティブモードでのパッチ（.iso形式）のインストールのサポートを開始しました。これにより、仮想マシンを切断または転送することなくハイパーバイザーを更新できます。 システムは、パッチをロードするときに、再起動に関連する変更がどれだけ必要かを判断し、必要でない場合は、プロセスを中断することなくパッチをインストールできます。 この機能は、インフラストラクチャメンテナンスに対するダウンタイムの悪影響を大幅に削減します。 また、XenServer 7.1では、バッチ更新のサポートが開始されました。 さらに、Citrixハイパーバイザーの新しいバージョンでは、LinuxまたはVMware vSphere仮想マシンの移行は、損傷やファイルの変更なしで行われます。



XenServerプラットフォームは、長期サポート（LTSR）を備えたバージョンとして初めて提供され、今日では10年間フルサービスをサポートする唯一のハイパーバイザーです。 XenServer 7.1のライフサイクルは、XenApp / XenDeskopのライフサイクルと完全に一致しており、メンテナンスとインフラストラクチャの統合が簡素化されます。



ライセンスに関して重要な変更があったことにも注意してください-現在、XenApp / XenDesktopのすべてのエディションには、Enterpriseと同様の機能を備えたCitrix XenServerを使用する権利が含まれています。



XenServer 7.1は、アプリケーションおよびデスクトップ仮想化のためのProvisioning Servicesテクノロジーをサポートしています。 ハイパーバイザーに含まれるPVS-Acceleratorテクノロジーは、各XenServerホストの帯域幅要件を最大98％削減するホストレベルのキャッシュであり、仮想マシンのロード速度を向上させます。 新しい簡素化されたPVSアーキテクチャにより、同じ数のワークステーションを同時にロードするために必要なPVSサーバーの数を減らすことができます。





XenServer 7.1でのPVS-Acceleratorによるキャッシュ（50-100 VMあたり）



CitrixとNutanix は最近 、Nutanix Enterprise Cloud HCIシステムのXenServerサポートを発表しました。これにより、Nutanixユーザーは、Citrix XenAppまたはXenDesktopベースのスクリプト用のvTaxを必要とせずに、XenServerの高度な機能に無料でアクセスできます。 これらの機能には、PVS統合、Direct Inspect APIテクノロジー、インタラクティブパッチインストール、およびNVIDIA GRID vGPUテクノロジー（Nutanix Acropolisをサポートしない）が含まれます。 XenServerを制御するには、標準のNutanix Prism Webコンソールを使用できます。





Nutanixの高レベルアーキテクチャ図



XenServerはCitrixにとって戦略的に重要であり、PVS-Acceleratorを含むすべてのCitrixソフトウェア製品の革新の基盤を形成し、HPE、Nutanixなどとの大規模なパートナーシッププロジェクトの実現を支援します。
