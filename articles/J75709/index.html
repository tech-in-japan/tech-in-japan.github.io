Uralユーザーグループ.NETの会議

昨日、2009年11月18日、次のUral .NETユーザーグループの会議が開催されました。 この会議には、 SQL Serverチームの開発者であるNikita Shamgunovというゲストが参加しました。 Nikitaは、有望なリアルタイムデータ処理技術であるStreamInsightについて、およびMicrosoftの開発チームでの作業について話しました。



2番目の講演者はStanislav Chernichkinで 、 ASP.NET MVC 2の革新と、関数型プログラミングとLINQのいくつかの側面について話しました。

カットの下でレポートのトピックについての詳細をお読みください。



StreamInsight



最初の講演者であるNikita Shamgunovは、 SQL Server Engine Platformのシニアソフトウェア開発エンジニアです。 Nikitaは、最近モスクワで開催されたPlatform 2010カンファレンスでStreamInsightに関する講演を読みました。 彼は、 CEPテクノロジーとは何か、StreamInsightの形で実装されていることを教えてくれました。 CEPが必要な領域とStreamInsightの適用方法。 ニキータは、テクノロジーの内部構造について語り、LINQ / C＃およびIObservableに基づいてライブラリを操作するいくつかの例を示しました。



Nikitaの2番目の講演は、マイクロソフトの開発チームの仕事についてでした。 このレポートは以前、MVP OpenDaysイベントで読まれました。 このレポートでは、Microsoftでの開発チームの形成プロセス、作業環境の編成方法、さまざまなチーム内での関係の構築方法を正確に説明しました。 Microsoftが新しい専門家を募集する方法と場所を学びました。 ニキータは、学生との仕事がどのように進行しているか、どのような人々が主にマイクロソフトに興味を持っているかを語った。



報告の後、ニキータが興味深い答えを出した多くの質問がありました。 概して、私はすでに両方の報告を聞いていたという事実を考慮しても、非常に興味深いものでした。 非公式の設定で、ニキータはもう少し話をして、もう少し面白いことについて彼に尋ねました。



ASP.NET MVC 2



会議の後半はスタニスラフ・チェルニッチキンが主導しました。 彼は、 ASP.NET MVC 2の新機能、それらを適用する方法、およびそれらを使用するときに直面した困難について話しました。 スタニスラフは、例としてコードを使用して、MVC 2でエリア、データアノテーション、およびテンプレートを使用することを示しました。



スタニスラフの講演の別の部分は、LINQの使用でした。 彼は関数型プログラミングの分野からのLINQのトリックについて話し、Haskellとの類似性を引き出しました。 面白かった。



ミーティングの最後に、私は少し前に、最近リリースされたASP.NET MVC 2ベータに以前のバージョンと比較して登場した新しい機能について話をしました。 私の短いプレゼンテーションはここからダウンロードできます 。



まとめ



会議は成功しました。 多くの興味深いことが語られ、興味深いゲストがいました、興味深い例がありました。 多くの質問と回答。 エカテリンブルクに住んでいる、または働いている場合は、必ずユーザーグループに登録して会議に参加してください。
