高品質のWi-Fi-現代のおもてなしとビジネスの原動力の基礎

高速Wi-Fiは、ホテルのおもてなしの基盤の1つです。 旅行に行ってホテルを選ぶとき、私たちはそれぞれWi-Fiの可用性を考慮します。 必要または望ましい情報をタイムリーに受信することは非常に重要なカテゴリであり、現代のホテルがそのサービスでWi-Fiインターネットにアクセスする必要があると言う理由はなく、不在が宿泊を拒否する理由になる可能性があります。 同時に、ホテルのWI-FI組織は訪問者の利便性を確保するための必須の手段であり、一時的な居住地を選択するための主要な基準の1つであるため、大規模なチェーンホテルであるかブティックであるかはまったく関係ありません。

ワイヤレスネットワークの構築は最も基本的なタスクから始まります-奇妙なことに、ネットワーク自体を構築します。 プロセス全体を簡素化し、最大の結果を達成する方法は？







まず、アクセスポイントの要件と、シスコがこれらの要件を満たすソリューションについて説明します。 ワイヤレスネットワークには何が必要ですか？



  1. 仮想化と使用する機器の量の削減 -理想的には、当然ながら、仮想コントローラーを使用するすべての設備と利点を維持しながら、高価なハードウェアコントローラーを拒否します。



    Cisco Mobility Expressは、物理的なWLANコントローラを必要としません。 コントローラーは中央アクセスポイントとして機能し、Mobility ExpressはWi-Fiテクノロジーの最新の進歩-ローカルまたはローカル（オンプレミス）制御用の802.11ac Wave 2をサポートします。
  2. 干渉に対する耐性と高い信号品質 -ホテルでは、信号の品質は環境によって大きく影響されます：壁、インテリアアイテム、パイプ、エンジニアリング構造。



    シスコのアクセスポイントは、革新的なCisco CleanAirおよびClientLinkテクノロジーを使用して、常に最高のWi-Fiネットワークパフォーマンスを保証します。 CleanAirは、プロアクティブな妨害防止機能です。 この機能は、干渉源を検出および識別し、ネットワークパフォーマンスへの影響の程度を評価してから、現在の環境で最高の効率を達成するようにネットワークを再構成します。



    ClientLinkを使用すると、Wi-Fiクライアントに接続されている側に信号を送ることができます。 このテクノロジーにより、さまざまなクライアントデバイスが同時に動作するネットワークの問題を解決すると同時に、802.11a / g、802.11n、および802.11acクライアントの伝送速度を向上させることができます。
  3. シームレスなローミングは問題ですが、その関連性は失われません。 シームレスなローミングにより、ホテル内を移動するときにゲストを接続したままにすることができます。 また、滞在中ずっとゲストの同じIPアドレスを維持できます。 このゲストのおかげで、ホテルのネットワークに一度ログインし、ホテルの部屋（ロビー、レストラン、または自分の部屋）でインターネットを使い続けるだけで十分です。



    すべてのシスコアクセスポイントを使用すると、専用のWi-Fiコントローラーをインストールせずにシームレスなローミングを作成できるため、あらゆる規模のホテルでWi-Fiネットワークを構築するコストを大幅に削減できます。
  4. 多数のクライアントと高いデータ転送速度のサポート -最適な負荷分散のために、2.4 GHzと5 GHzの無線帯域を適切に管理する必要があります。



    シスコのアクセスポイントでは、Cisco BandSelectテクノロジーを使用して、クライアントデバイスを周波数で区別しています。 デバイスが5 GHzの周波数でアクセスポイントに接続できる場合、その周波数で動作し、より一般的に使用される2.4 GHz無線範囲をアンロードできます。



    さらに、シスコのアクセスポイントでは、無線リソース管理（RRM）アルゴリズムが使用されます。これにより、無線周波数チャネル、その幅、信号放出電力を自動的に調整し、動的無線環境のカバレッジのギャップを排除できます。
  5. PoEテクノロジーを使用した電源ポイント -不都合な場所にコンセントを設置したり、大型の電源を使用したり、配線を追加したりする必要がありません。



    Ciscoスイッチは、リモートアクセスポイントのPoEテクノロジーをサポートしています。
  6. ゲストネットワークと企業ネットワークの安全な分離 -ネットワークはホテルの訪問者とそこで働くスタッフの両方によって使用される可能性が高いためです！ Ciscoアクセスポイントはポリシー分類エンジンを使用します。これにより、ユーザーの役割（ホテルのゲスト、従業員、訪問者）、ネットワークアクセス方法、およびアプリケーションで使用されるデバイスのタイプに基づいて詳細なネットワークアクセスポリシーを実装できます。



    ポリシーは、さまざまなネットワークセグメントへのアクセス権、接続速度、制限、使用するアプリケーションの優先度（Application Visibility＆Control）を決定します。 これにより、すべての従業員とゲストは自分のデバイスを使用して、企業ネットワークの情報セキュリティに違反するリスクなしに接続できます。

Wi-Fiネットワークの収益化は依然として広く議論されているトピックであり、ホテル事業にとってこのトピックは二重に重要です。 ホテルでワイヤレスネットワークを収益化する方法は？







Cisco CMX（Cisco Connected Mobile Experiences）ソリューションは、Wi-Fiが有用な分析を取得し、ホテル経営者がより良いビジネス上の意思決定を行えるようにします。



ターゲットオーディエンスが1日または1週間にどのエリアまたはどのサイトでより多くの時間を費やしているのか、最も集中しているポイントがどこにあるのか、ビジターの何パーセントが初めてここにいるのか、そして何人が再び戻ったのかに関する情報を提供するヒートマップ。 これは、ビジネス開発に必要な最も価値のあるビジネスインテリジェンスであり、シスコの機器が組み立てて処理できます。



管理者と所有者の両方にとって最も簡単なオプションは、1つのウィンドウですべての「グッズ」を提供するモバイルデバイス用のアプリケーションです。



  • 通常のゲストへの個別の挨拶 -ネットワークはゲストを認識し、ロビーの入り口で彼に挨拶します。 これが通常の顧客である場合、自動チェックインを実行し、番号を報告し、モバイルデバイスをキーに変えることができます。
  • アクションと場所に応じたサービスとプロモーションに関する通知 - 場所データを使用して、特定のプロモーションの申し出とともにゲストのモバイルデバイスにプッシュ通知を送信できます（たとえば、ゲストがプールにいる場合、割引バーでカクテルを試す機会が提供されますまたは、店舗を通過したゲストは、割引が提供されているという通知を受け取ります...）;
  • ホテル周辺のナビゲーション -ゲストの位置は、使用されるアクセスポイントによって決定され、目的の場所（ショップ、プール、レストラン、会議室など）へのパスが表示されます。
  • ビジネスおよびビジネス分析の自動化 -従業員の「モバイルデバイスとその場所の把握」を使用して、すべてのゲストの要望にすばやく対応し、ゲストの場所を把握し、ゲストフローを監視して、スタッフを問題のあるエリアにリダイレクトできます。
