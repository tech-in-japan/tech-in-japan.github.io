2015年のデータ分析と処理の5つの傾向





年の初めは、予測と予測の時期です。 この投稿では、2015年の分析およびデータ処理システムの傾向の概要を収集しました。



毎年、ますます多くの企業が、分析システムと大規模なデータセットを処理するためのシステムの正しい使用が市場で大きな競争上の優位性をもたらすという結論に達します。 この状況は、企業の分析部門の積極的な開発とデータ処理インフラストラクチャへの重要な投資につながります。



さまざまなレポートによると、分析およびデータ処理システムの開発コストの年間増加は20％〜30％です。 2015年のデータ処理システムと分析サービスの予測総費用は、約30億ドルになります。



分析システムへの強い需要と多額の投資は、ビッグデータ業界の急速な発展と膨大な数の新製品とソリューションの出現の強力な推進力です。



デロイトやフォレスターなどの大規模なコンサルティング会社の調査によると、2015年には、データ分析とデータ処理の分野における以下の主な傾向が強調されます。



  • データ処理プロセスの自動化と商品化
  • リアルタイム分析システムの開発
  • センサーデータとさまざまなデバイスを分析するための新しいツールの出現
  • 認知システムと自動意思決定システムの開発
  • データの保管と処理に関連する法律の変更




1.データ処理プロセスの自動化と商品化



ストレージおよびデータ処理システムのメンテナンスコストが高いため、企業はより安価なソリューションを探す必要があります。 そのようなソリューションの1つは、独自のデータセンターの放棄と、 Amazon WS 、 Microsoft Azureなどのデータストレージおよび分析のためのクラウドテクノロジーへの移行です。



データ分析の分野の専門家が急激に不足しているため、テクノロジーに関する深い知識を必要とせず、ビジネス上の問題の解決に焦点を当てた自動化ツールの開発に良い動機を与えています。 そのようなツールの例には、 Tableau 、 IBM Watson、およびAzure Machine Learningが含まれます。



それとは別に、 OpenRefineやDataWranglerなどの生データおよび非構造化データ用の自動前処理システムの開発に注目する価値があります 。



2.リアルタイム分析システムの開発



リクエストの処理とレポートの生成に比較的長い時間を必要とするリレーショナルデータベースとOLAPテクノロジに基づく従来の分析ソリューションでは、現在のビジネス上の問題を解決するのに十分ではありません。



効果的な意思決定を行うには、企業はリアルタイムでデータを分析する必要があります。 これにより、ストリーミングデータ処理とRAMに保存されたデータへの迅速なアクセスのための技術の開発につながります。 これらの問題を解決するために設計された最初の製品の1つは、 Apache StormとCloudera Impalaです。



3.センサーデータとさまざまなデバイスを分析するための新しいツールの出現



ウェアラブルエレクトロニクスとモノのインターネットは、2015年も急速に発展し続けます。 このようなガジェットは、エンターテインメントと健康の監視ですでに広く使用されています。 これらのデバイスからの情報を分析することにより、デバイスの使用効率を高めることができます。



ガジェットの広範な使用は、さまざまな種類のセンサーおよびセンサーの信号の分析に関連する新しいタスクの出現につながります。 トランザクションデータとは異なり、センサーデータはノイズが多く、歪みの影響を受けやすいため、そのような信号の処理には特別なアプローチが必要です。 センサーからの情報の収集と分析に関連するタスクについては、特別なツールが作成されます。



センサーと分析システムの統合に関する標準も積極的に開発され、モノのインターネット全体の開発が大幅に加速されます。



それとは別に、ウェアラブルエレクトロニクスのデータを使用すると、ユーザーの行動や好みに関する詳細な情報を収集できるため、データマーケットでの新しいプレーヤーの出現につながる可能性があります 。



4.認知システムと自動意思決定システムの開発



専門家は、人間の相互作用と人工知能の分野における技術の積極的な開発を予測しています。 特に、ビジネスプロセスにおける自動意思決定システム、テキスト情報、音声、画像、およびビデオを分析するためのシステムの開発。 Google NowやWord Lenseなどの既存の製品に注目できます。



5.データの保存と処理に関連する法律の変更



今日、私たちは毎日非常に多くのデータを生成しているため、各人物の肖像を正確に編集し、好みを明らかにし、個人情報を見つけることができます。 モノのインターネットの開発と新しいガジェットの出現により、データの量は増加するだけであり、データセンターに保存される個人情報のシェアは増加します。



このような状況の発展は、この種のデータの処理に直接関係するサービスや企業に対する社会的圧力の増大につながり、規制当局の介入と、個人情報の使用を制限し、データストレージのセキュリティ要件を決定する新しい法律の出現につながります。



ほとんどの場合、新しい要件により、企業のアーキテクチャ上の決定を再検討せざるを得なくなり、クラウドではなくデータの一部が安全なローカルデータセンターに格納されるハイブリッドシステムの出現につながります。



おわりに



過去の期間における大きな成果にもかかわらず、2015年は、分析システムの開発と、データ処理および分析の分野における革新的なソリューションの新しい機会の出現という点で非常に興味深いものになると約束しています。



2015年の分析およびデータ処理システムの動向についてどう思いますか？ 今年、既存の技術のどれが需要がありますか？ まだ解決されていないタスクは何ですか？ コメントで議論しましょう。
