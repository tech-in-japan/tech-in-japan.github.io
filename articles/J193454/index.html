IBM、NeXtScale System-データセンター向け高性能コンピューティングプラットフォームを発表





IBMはNeXtScale Systemを導入しました。これは、最新のモノブロックラックマウントサーバーの最大3倍の計算能力を提供する汎用コンピューティングプラットフォームです。 このソリューションは、ソーシャルメディアやクラウドサービス、分析、数学的計算などの負荷の高いシステムでの使用に最適です。



これらのサービスとその提供モデルへの迅速な適応は、データセンターの作成に対する需要の増加につながり、顧客は効率を高め、コストを削減するためにこれらの要件を満たす新しいテクノロジーを探しています。 同時に、主要な選択基準は、高性能であり、可能な限り低いエネルギー消費指標です。 NeXtScaleプラットフォームは、スーパーコンピューターの電源を必要とするアプリケーションを実行するために特別に設計されたサーバーソリューションのIBM x86ポートフォリオへの最新の追加です。 新しいプラットフォームは、コンピューティング、大量のデータの保存、グラフィックスアクセラレータのインストールをサポートする、柔軟でオープンで操作しやすいアーキテクチャに基づいて構築されています。



NeXtScale Systemでは、標準の19インチラック形式EIAに最大84個のx86サーバーと2016プロセッサコアを配置できます。 このプラットフォームには、入出力ボードやトップオブラックスイッチなどの標準コンポーネントの使用が含まれており、構成の柔軟性とシステムの迅速な適応を実現します。 IBMはまた、NeXtScale用の強力なソフトウェアスタックを提供します。これには、IBM General Parallel File System、GPFS Storage Server、xCAT、およびIBM Platform Computingが含まれ、システムの計画、管理、最適化のための主要なツールをお客様に提供します。



このプラットフォームを使用すると、オープンスタンダードに基づいた単一のアーキテクチャを作成でき、高いパフォーマンスと効率を実現できます。 このようなシステムは、顧客のデータセンター、既存のプラクティス、およびその他のx86ベースのソリューションと迅速に統合できます。 NeXtScaleは、顧客が運用および投資予算をより効率的に管理できるようにし、最大の計算能力と大幅なスペース節約を提供します。 標準のフォームファクタと標準コンポーネントの使用により、IBMパートナーは、より多くのユーザーに高性能コンピューティングのプラットフォームを提供できます。



「NeXtScaleプラットフォームは、高性能およびクラウドコンピューティングまたはマネージドホスティングサービスの最も要求の厳しいワークロードに対して高いパフォーマンスを提供します」と、NASAダイレクトシステムサポートディレクター（IBMビジネスパートナー）のKevin Rozynekは述べています。 「さらに、この新しいシステムは、コンポーネントの設定と構成の柔軟性の点でより多くのオプションを顧客に提供します。実際、何でもできます。」



NeXtScaleは完璧なソリューションです。

  • 効率、高密度コンピューティング、およびスケーラビリティを必要とする大規模なデータセンターの場合。
  • パブリック、プライベート、ハイブリッドクラウドインフラストラクチャの場合。
  • ビジネスインテリジェンスアプリケーションの場合、たとえば、顧客関係管理、運用の最適化、財務リスク管理、ビジネスモデリング。
  • オンラインゲームやビデオストリーミングなど、ソーシャルネットワークリソースのアプリケーションの場合。
  • 薬や石油と天然ガスなどの分野での高解像度イメージング用。
  • 「産業用」用途向け-コンパクトなソリューションが結果の予測、エンジニアリング計算、設計およびモデリングプロセスの最適化を促進する場合。


IBMは、x3650 M4 HDサーバーも導入しました。これは、最大12 GBの帯域幅をサポートするハードウェアRAIDコントローラーを使用するx3650システムの改良バージョンで、60％高いストレージ密度も提供するため、大量のデータや重要なワークロード。 IBM NeXtScaleおよびSystem x3650 M4 HDは、System xデュアルソケットサーバーポートフォリオに追加された最新製品で、ラックマウントソリューション、Flex System、iDataPlex、BladeCenterサーバーが含まれます。 これらのサーバーソリューションはすべて、新しいIntel Xeon E5-2600 v2プロセッサーを使用します。これにより、パフォーマンスが最大45％*向上します。



「並外れた柔軟性、高いパフォーマンスと効率、標準のフォームファクター、強力なソフトウェアスタックの組み合わせにより、NeXtScaleはお客様のデータセンター向けのワンストップソリューションです。 このプラットフォームは、使用、実装、展開の準備をするのに十分簡単です」と、IBM System xのCEO、Adalio Sanchez氏は述べています。 「NeXtScaleおよびその他の新製品とアップデートの発表は、お客様向けの高性能ソリューションのx86ポートフォリオを継続的に開発するという当社のコミットメントを反映しています。」



NeXtScaleは、業界最速のx86プロセッサと、周波数1866 MHzの高速メモリモジュールをサポートしています。 プラットフォームは、データセンターに統合するか、サーバールームで使用でき、100-127 Vの低電圧で動作する電源システムと互換性があります。NeXtScaleは、動作温度が上昇する（最大40°C）データセンターでも使用できます。冷却システムは運用コストを削減します。 ネイティブ拡張コンセプトにより、ユーザーは、チェックアウト段階およびプラットフォームの動作中の両方で、データストレージ、グラフィックアクセラレータ、コプロセッサなどの追加機能をNeXtScaleプラットフォームに追加できます。 NeXtScaleユーザーは、Ansys、MPI-BLAST、OpenStackなど、多くの業界標準の高性能コンピューティングおよびクラウドソリューションの構成に役立つ基本キットにもアクセスできます。



NeXtScaleプラットフォームは、IBMビジネスパートナーにソリューションを構築するための標準部品の形式で注文できる柔軟な構成システムを提供します。これにより、パートナーは主要なコンポーネントとサービスを追加して追加機能を作成できます。 この高性能ソリューションにより、ビジネスパートナーは顧客ベースを拡大でき、急成長するx86サーバー市場での成長に役立ちます。



NeXtScaleプラットフォームは、特別なシステムの柔軟性を備えており、単一ノード構成で、空または構成済みのシャーシの形式で注文できます。また、テストに合格し、すぐに使用できる統合IBM Intelligent Clusterソリューションの一部でもあります。 Intelligent Clusterシステムを選択すると、顧客は、ストレステストに合格した必要なすべてのスイッチングモジュールとソフトウェアを備えたソリューションを受け取ります。 この構成により、顧客がシステムの構成に通常費やす時間を75％**削減でき、同時に梱包材からの家庭廃棄物のレベルを大幅に削減できます。
