Mac OS X 10.6用のロシア語とウクライナ語を組み合わせたタイポグラフィキーボードレイアウト

こんにちは、Habr。



2週間目のメインオペレーティングシステムはMac OS X（10.6.7）です。 印象...あいまいです。 私はたくさんのものが好きです、私はいくつかを我慢する準備ができています（そして、それらを便利だと思う時が来ることを認めさえします）が、カテゴリに合わないものがあります。 つまり-キーボードレイアウト。 事実、私はタイポグラフィを使用しなければならない（したがって、便利なタイポグラフィレイアウトが必要です）ことに加えて、英語とロシア語に加えて、多くの場合ウクライナ語が必要です。 まあ、私は15年間PCキーボードをタップして、メインの仕事で使い続けています。



Mac OS Xの「箱から出した」状態を見て回ったとき、私は悲しかったです...リクエストが増えた裸のスイッチャーの目を通して、状況はおよそ次のとおりです。

  1. Command + Spaceでレイアウトを切り替えます。 非常に美しい解決策は、レイアウトで大きな半透明のウィンドウを表示することですが、私は珍しいです。
  2. デフォルトのレイアウトは米国です。オフになりません（とにかく、 Q＆Aの質問にはまだ回答がありません）。
  3. 活版印刷の文字はレイアウト上に存在しますが、このセットは私が通常使用するものに対応していません-多くの余分な（私にとって）ものがたくさんあります。
  4. 標準のロシア語のレイアウトはPC-shnoyとは異なります。Shift+ 6、Shift + 7ではドットとコンマを使用できます。真実とPC-shnyオプションがあります。 ウクライナ語ではすべてが悪いです-PCオプションはありません。さらに、文字「I」/「I」は場所に置き換えられます。


システムで何かが自分に合わないとき、オタクは何をしますか？ そうだね。 :)彼らはファイルを拾います。



まず、私は自分のタイポグラフィのレイアウトをポピーに移植しました。









この質問は頻繁に聞かれるので、Ilya Birmanのレイアウトとどのように異なるかを予防的に説明します。

  • 何よりもまず、テキストデザイナーだけでなく、プログラマー、レイアウトデザイナー、ウィキペディアエディターにも焦点を当てています。 これはどういう意味ですか？ 英語で使用可能なほとんどすべての文字は、＃、|、＆、^、<、>などのレイアウトで使用できます。 これで、単一の文字を入力するために切り替える必要はありません。
  • 拡張文字セット。 たとえば、Ilyaに乗算記号（×および偶数）がある理由はわかりませんが、除算記号はありません（÷と⁄←をスラッシュと混同しないでください）。 また、短いスペース、日本円の指定、ウクライナのグリブナ、弾丸、追加の著作権記号、およびハートを配置する必要があると考えました。 ミミミ。
  • 上付き文字のすべての数字（Alt + 1、+数字）、およびユニコードで使用可能な1 / xタイプのすべての小数（Alt + Shift + 1、+数字）を入力できます。
  • Mac版では、Macキーの絵文字付きの記号があります。
  • 分音記号はありません（私の意見では、対応する言語のキーボードレイアウトにまだ場所があります）。


ドラフト形式では、英語、ロシア語、ウクライナ語の3つの言語のレイアウトが作成されました。 それはもう少し快適になりましたが、残念ながら、まだ理想からはほど遠いです。 地獄のタイポグラフィを使用した、切り離されていない英語のレイアウトでは、ミッションが達成されたと考えることはできませんでした。 この問題を解決する方法を探して、 Punto Switcher for Macが検討されました-任意の2つのレイアウトを選択し、キーの組み合わせ（ちなみにCtrl / Alt + Shiftを含む）で切り替えることができました。 このオプションは（自動切り替えなどのフリルをオフにした後）非常に便利なように思えましたが、やはり問題は2つのレイアウトだけです...



今回はファイルが少し長くきしみましたが、必要なものが手に入りました。 Caps Lockキーの機能を放棄したため、Caps Lockを押しても反応しない英語のレイアウトと、ロシア語とウクライナ語の切り替えを実際にCaps Lockで行うロシア語とウクライナ語の2つのレイアウトを作成しました。 キリル文字のレイアウトは次のとおりです。



ロシア語





ウクライナ語





その結果、何が得られましたか？

  1. 素晴らしい（私にぴったりの）活版印刷レイアウト。
  2. ネイティブのスイッチャーハートAlt + Shiftに応じてラテン語とキリル文字を切り替える（オプションで他の組み合わせに調整）。
  3. キリル文字で、Caps Lockによるロシア語とウクライナ語の切り替え。


何を失ったの？

  1. キャップロック。 ここで、LOUDステートメントの場合、Shiftキーを押したままにします。 :)私見、重要ではありません。
  2. 「各ウィンドウの個々のキーボードレイアウト」の設定は、次のような意味を失います。 キリル文字では、すべてがCaps Lockの状態に置かれます。


一般的に、今私は快適で幸せに感じています。 私のソリューションが他の誰にも役立つとうれしいです。
