Python Meetup：3月の会議





3月、プログラミング言語Python Meetup Minskの愛好家の月例会議が開催されました 。 次のレポートがmitapグリッドで読み上げられました。Pythonのマキシムバリシュニコフのスイッチヘッド用 、「 Redis。どのように複雑さに苦しんだか 」Roman Imankulovから、「 Review of the Twisted framework 」Andrey Zhlobich。



いつものように、すべてのレポートのビデオとプレゼンテーションへのリンクはブログで見つけることができます。 さらに、 5月30日に開催される新しいPython Meetup Minskミーティングの詳細はすでに知られています。これはすべて、カットの下で見つけることができます。 素敵な景色を！



1. スイッチヘッド用の Python-マキシムバリシュニコフ、 Wargaming.net / Game Stream。





他の言語で書いているプログラマーの観点からのPythonの簡単な説明。



2. Redis。 どのように複雑さと闘ったのか -Roman Imankulov、 Doist.io。





Redisは、あらゆる開発者の武器になるはずのツールです。 レポートでは、RomanがRedisがTodoistを何十万人ものユーザーにサービスを提供し、ソリューションのシンプルさと優雅さに魅了する方法を説明します。



3. Twistedフレームワークの概要 -Andrey Zhlobich、 Wargaming.net / Game Stream。





レポートでは、非同期envenloopが何であるか、Twistedが何であるか、いつ必要になる可能性があるのか​​、そしてそれから何を期待すべきでないのかを検討し、調べます。 遅延コンセプト。 RDBMSおよび他の同期ライブラリとの相互作用。 アプリケーションフレームワーク ツイストWebなど



PS： 5月30日金曜日、ベラルーシのPython開発者コミュニティが4 時にスタートアップハブImaguruに集まります。 ファブリジウス 、 会議は伝統的に19.00に始まります 。



Pythonプログラムの紹介：

レポート番号1。 Artyom Pevukhin Undev.ru（モスクワ） 「Pythonプログラマー向けGo言語」

内容：レポートでは、Pythonプログラミング言語Goが開発者にとってどのように役立つかを説明します。 Goの基本的なイディオムについて説明し、次の質問に対する回答を示します。「Zen of Python」はGoにどのように適用できますか？ この言語の範囲は何ですか？ Pythonの代わりにGoを使用して何を獲得できますか？



レポート番号2。 マキシム・バリシニコフ、Wargaming（Minsk）のソリューションアーキテクト 「Wargaming：the rear-to the front！」

内容について：次のように言っても、オープンサインの秘密を明かさないと思います：ゲームが成功する（そして成功から伝説になる）ためには、ゲーム自体がどれほど良いとしても十分ではありません。 とりわけ、非常に大規模なゲームには、非常に開発されたWebパーツが必要です。 それについて説明します。



レポート番号3。 Alexey Cherkes Sakrament-IT（Minsk）のシニア開発者 「SWIG-C / C ++ライブラリ用の多言語インターフェイスの作成。」

概要： APIを使用したバインディングとPythonについて簡単に説明します。 Pythonのバインダーのクイックツアー。 次に、SWIGについて説明します。それは何ですか、なぜ必要なのか、機能、ユーティリティの動作スキーム、起動とアセンブリの順序、動作方法、C ++オブジェクトモデルとターゲット言語の比較方法です。 PythonおよびC＃の例。 アプリケーションの経験、賛否両論。



通常どおり、「無料のマイク」が機能します。つまり、事前に準備したプレゼンテーションを必要としない電撃レポートを誰でも作成できます。 主なものは、興味深く有用なものを共有したいという願望です。
