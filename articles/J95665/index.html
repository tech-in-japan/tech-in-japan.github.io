Microsoft Visual StudioはWWDC`10に登場しますか？

過去数週間にわたって、3つの興味深いメッセージがほぼ連続して送信されました。

1. 「WWDC 2010のバルマー」 、

2. 「WWDC 2010にはスティーブバルマーはいません」 、

3. 「Microsoftは、iPhone OSデバイスでBing検索を表示したいと考えています 。 」

会話：

-Microsoft Windowsプラットフォーム上のApple iPhone OSの開発者サポートを備えたMicrosoft Visual Studio 2010ツールのリリースの可能性について。

-そして、さまざまなMicrosoft BingサービスのiPhone OSへの統合は、理由なしに発生することはなく（発火せずに煙がない）、相互接続することもできました。



Microsoft WindowsプラットフォームでのApple iPhone OSの開発者サポートを備えたMicrosoft Visual Studio 2010のリリースでは、（特にAdobe Flash開発者およびiPhone OSのネイティブソフトウェアの要件の禁止後に）関心があるのはAppleであり、Microsoftではありません。

なぜAppleはこれを必要としますか？

特定のオペレーティングシステムの人気は品質の高いソフトウェアの量に依存するため、Apple iPhone OSをできるだけ多くのソフトウェアで満たし、懐中電灯などの装身具だけでなく、仕事や日常生活に役立つソフトウェアで満たすためにGoogle Androidとの戦いにおける強力な切り札になります）。

同時に、Microsoft Windowsプラットフォームで作業する多くの専門家を集めて、iPhone OS用のソフトウェアを開発してください！



おそらく、スティーブジョブズは、Microsoft Windowsで今日書かれたプロ用ソフトウェアを可能な限りiPhone OSプラットフォームに移行することを夢見さえします。 彼自身がD8 Conference：All Things Digitalでこれについてほのめかしました -スティーブは、Apple iPadの未来についていくつか興味深いことを述べました。



「...デバイスで何でもできるので、デバイスはほぼ完璧になりました。 ソフトウェアはどんどん良くなっており、近い将来、ビデオ、写真を編集し、以前はデスクトップシステムでしか利用できなかったことができるようになります...」

「私たちが「農業」国家だったとき、大陸のすべての車はトラックでした。 しかし、時間が経つにつれて、人々は都市に移動し、車に変わり始めました。 現在、PCはトラックのようなものだと思います。 それらを必要とする人は少なくなります。」



そして、Apple iPhone OSの開発者サポートでMicrosoft Visual Studio 2010をリリースするようMicrosoftを説得したのはApple自身であったと推測できます（1997年にAppleがApple MacOS用のMS Office＆MS Internet ExplorerをリリースするようMicrosoftを説得したように）お金-結局のところ、MicrosoftはApple iPhone OSをサポートするMicrosoft Visual Studio 2010のリリースに明らかに興味がない。

マイクロソフトは、Appleに譲歩があった場合、当然、Appleに（お金以外の）パートナーシップ契約への署名を要求し始めることができます。もちろん、Microsoft BingサービスをiPhone OSに統合することは、相互譲歩としてです。

（おそらく、Microsoftは、Apple iPhone OSの開発者サポートを備えたMicrosoft Visual Studio 2010がMicrosoft Windowsプラットフォームのトロイの木馬になる可能性さえ認識していない-より専門的なソフトウェアがApple iPhone OSに転送されるため、エンドユーザーにはMicrosoft Windowsプラットフォームが必要です...）



Microsoftは、WWDC`10開発者会議に参加しているSteve Ballmerについての噂をすでに否定しています。 しかし、WWDCで話すことはマイクロソフトの責任者ではなく、たとえばMicrosoft Developer Networkの責任者である可能性があります。

そしてこれにより、MicrosoftはAppleをサポートすることにあまり強く興味を示さないか、おそらくGoogleの敵に対するAppleとMicrosoftの戦略的同盟を偽装することで感覚を多少緩和することができます（Google Androidを促進する-明らかにApple iPhone OSの強力な競争相手）。



私の意見では、Apple iPhone OSのサポートとMicrosoft OSでのMicrosoft Bingサービスの統合を伴うMicrosoft Visual Studio 2010についてのうわさは、明らかに理由があり、AppleとMicrosoftの間のいくつかの交渉が進行中である可能性があります（おそらく非常に難しい交渉）。

1997年のように、AppleとMicrosoftの間で文書化されていない（秘密の）パートナーシップ契約の署名（または拡張）が行われるかどうかは不明です...

-XXI世紀と現代のグローバルビジネスの中庭では、パートナーと敵の両方との密接な相互作用と秘密の陰謀が関係しており、誰が誰であるかを決定することはほとんど不可能です:(
