Windows 7 RCは、MSDN、MSDNAA、およびTechNetサブスクリプションでダウンロード可能

本日より、 Windows 7 RCはMSDNおよびTechNetサブスクリプション、およびMSDN Academic Allianceサブスクリプションでダウンロードできるようになりました！







学生、大学院生、および教師の場合、これは次のことを意味します：教職員がMSDN AAを購読しており、インターネット経由でインストールイメージをダウンロードできる場合、Windows 7 RCのインストールイメージを提供するように要求するプログラム管理者に連絡できます。



また、約1週間後、Windows 7 RCはELMSシステムから利用できるようになります（組織がこのシステムを使用しない場合は、MSDNAAプログラムの管理者に構成を依頼してください）。
