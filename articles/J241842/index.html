ロシアの新興企業に対する欧州の専門知識：RVCがサポートするFaberNovelからの加速





本日、パリの大手イノベーションセンターで2週間の加速プログラムについて話し合うことにしました。このプログラムは、RVC OJSCと共同で実施しており、ロシアのITスタートアップが参加できます。 このプログラムは2014年12月1日から12日まで開催されます。この間、スタートアップはパリに住み、フランスの最高のコワーキングセンターの1つで働き、リーンスタートアップとグロースハッキングテクノロジーの基礎から始まり、有名なフランスの技術専門家、起業家、ベンチャー投資家（ヨーロッパのインキュベーターやベンチャーファンドの代表者は言うまでもありません）。



このプログラムは誰のために発明されたのですか？



シード/スタートアップ段階にあるプロジェクト、そして何よりもまず、プロの連絡先のネットワークを拡大し、プロジェクトを評価し、欧州の主要な専門家の意見とサポートを獲得し、革新的なエコシステムがどのように機能するかを理解する必要がある創業者海外では、ベンチャーキャピタルコミュニティとの関係を確立します。



プログラムの目的は何ですか？



プロジェクトの戦略的ビジョンを形成し、パートナーと潜在的な投資家を見つけ、グローバルテクノロジービジネス開発の原則を理解し、ヨーロッパでの企業の「上陸」の可能性を評価し、革新的なエコシステムに没頭するために、プログラムに参加する価値があります。



どうなるの？



プログラムの12日間（2日間の休業日を除く）では、参加者はワークショップに参加し、講義を聞き、専門家とコミュニケーションを取り、会議に出席します。 加速プログラム全体を通して、スタートアップはパリのオープンNUMAアクセラレータで働きます。



NUMAはパリで最初のコワーキングであり、2008年にオープンしました。 現在、NUMAは国内最大の革新的なハブです。 各プログラムの参加者には、職場とイノベーションセンターのインフラストラクチャへの無制限のアクセスが割り当てられます。 全員のコワーキングに十分なスペースがあります。総床面積1,500 m2の6フロアを占有し、200のスタートアップがあり、40,000人の訪問者を受け入れます。





外でコラボしているように見える



プログラム内のすべてのタイプの学習活動は、次のブロックに分類できます。



テーマ別セッション


ハイテク遊び場


このトレーニングセッション中に、参加者は、新しい顧客体験と新世代のユーザーの出現に影響を与えるテクノロジーに精通します。 セッション中に、FaberNovelの専門家が有望な技術を紹介し、ビジネス構築への影響について話します。 技術の遊び場には、参加者が有望なガジェットと技術をテストできる実用的な部分も含まれています。 クラスの一部として、2種類のテクノロジーが考慮されます。

  • 接続テクノロジー：Google Glass、Oculus Rift、Drone、Leap Motion、
  • 接続サービス：Estimote Beacon、Square、Nest。


リーンスタートアップ


このセッションは、初期投資をせずに製品を迅速に立ち上げるための「リーンスタートアップ」方法論をスタートアップが学ぶのに役立ちます。 この方法論の創設者はEric Riceです。





そしてコワーキングルームは中を見る



グロースハッキング


これは、スタートアップの爆発的な成長ツールを探索するためのチュートリアルです。 セッション中、専門家は成長ハッカーの事例とベストプラクティスを分析し、参加者は顧客を引き付けるための最も効果的な方法を分析し、FaberNovel Instituteの専門家と一緒にテストサイトでトレーニングします。



ビッグデータ


ビッグデータワークショップでは、最も重要な最新トレンドとビッグデータテクノロジーに没頭します。 ワークショップでは、参加者はビッグデータを使用するための主なシナリオを学習し、利用可能な技術に精通し、自分の教育プロジェクトで実験します。



専門家との会議


FaberNovelグループ企業のパートナー


テーマセッションに加えて、プログラムにはFaberNovel パートナーとのミーティングが含まれます 。 各自は、自身のセッション中に、専門家に関連するトピックに関するプロジェクトと洞察を共有します。 たとえば、FaberNovelの副社長であるKirill Warth（彼はTwitter、LinkedIn、Michelinなど、FaberNovelのクライアント向けの事業計画および企業開発プロジェクトの実施を監督しています）が、効果的な企業戦略を開発するための原則について語ります。 彼は、価値提案とビジネスポジショニングの分野でスタートアップと協力します。



プロジェクト（たとえば、 Monet 2010など ）が世界的に認められているデザイナーのBenoit Talaboは、スタートアップにデザインとUXを伝え、サービスや製品のデザインの改善を支援します。 Marie-Carolyn Lanfranchiは、FaberNovelで革新的なプロジェクトと研究管理の立ち上げを担当しています。 セッションの一部として、プロセス制御の参加者を支援して、製品を迅速かつ効率的に発売します。



加速プログラムの代表者


プログラムの一環として、参加者は50パートナーズファンド （スタートアップが価値提案を正式化し、専門的なベンチャーキャピタルを引き付けるのに役立つ種まきファンド）とその業界の専門家に精通します。 さらに、スタートアップは、業界の専門知識をチームと共有するアクセラレータLe Campingのメンターと話します。



コミュニティとのミーティング（シリアル起業家、スタートアップ、ベンチャー投資家）


このプログラムには、イノベーションコミュニティとのミーティングも含まれています。 これらの会議中、参加者は地元の起業家とコミュニケーションを取り、パリの革新的なトレンドを研究します。 プログラムの期間中、参加者はまた、州基金の代表者を含むベンチャーキャピタル投資家や既存のスタートアップサポートプログラムと会います。



特別なイベント


プログラムの1日は、 API Daysを訪問するために予約されます。APIDaysは、APIテクノロジーと、それが現代企業のビジネスの創造と発展に与える影響に特化したグローバル会議です。 この会議では、業界をリードするスピーカーのワークショップ、プレゼンテーション、パネルディスカッションも参加者に提供されます（API Daysは、1784年に建てられたパリの古い建物で開催されます）。



結果は何ですか？



プログラムはデモデイで終了します。プロジェクトファウンダーは5分間のピッチに参加し、ベンチャー投資家、成功した起業家（FaberNovelのファウンダー、Stefan Distengenを含む）からの質問に答えます。 イベント期間中、スタートアップは有資格の投資家と知り合い、資金調達のさらなる作業のための連絡先を取得します。



プログラムの結果に基づいて、チームはパートナーを見つけ、大企業との関係を確立し、市場と技術で高品質の専門知識を獲得するための共同作業のさらなる分野について議論します。



なぜFaberNovelなのか？



FaberNovelは、2003年にパリに設立された国際的なイノベーション機関です。 私たちは、イノベーション、ビジネスの加速、新しいサービスと製品の開発と実装の分野でのコンサルティングに特化しており、スタートアップインキュベーターやコワーキングと連携しています。 FaberNovelの顧客には、トップ500リストの企業（イブサンローラン、ツイッター、AT＆T、LinkedIn、BNPパリバなど）が含まれます。2008年、EMEA地域（ヨーロッパ、中東、アフリカ）で急成長しているテクノロジー企業50社のリストに加わりました。 2011年、カンヌライオンズクリエイティブインダストリーフェスティバルで銀賞を受賞しました。



何に参加する必要がありますか？



まず、ビジネスの概念を持ち、英語を話す必要があります（トレーニングとコミュニケーションは英語で行われます）。 プログラムは支払われますが、参加者はその費用の半分だけを支払う必要があります-50％はこのプロジェクトのFaberNovelのパートナーであるRVCによって資金提供されています。 プログラムに参加して詳細を知るには、 meet_moscow @ fabernovel.comに連絡してください 。
