Windows 8サマースクール





皆さんはすでに、Microsoft Windows 8から新しいオペレーティングシステムについて聞いています。 おそらく、インストールしてアプリケーションを作成しようとした人もいます。



Windows 8は新しく開発されたWindowsです。 これは、ARMおよびx86 / x64コンピューター（タブレット、スレート、ウルトラブック、デスクトップコンピューター）で動作します。



Metroデザインのサマースクールの伝統を引き継ぎ、 Windows 8で開発者向けのサマースクールを開催することにしました。



UPD：「7月13日から14日までの地下鉄学校の移転」に関する手紙を受け取ることができます。 これは間違った手紙です。 私たちのイベントは地下鉄の学校ではないため、スケジュールを変更することはできず（7月には不可能です）、キャンセルされず、8月5日にゴーリキーパーク（ネスクーニーサド）で開催されます。



簡単な説明



Windows 8向けの開発の基礎に精通し、このプラットフォームでアプリケーションを収益化する可能性について学び、志を同じくする人々のチームを見つけたり考え出し、アプリケーションの開発を開始し、コミュニティとMicrosoftの専門家に尋ねる機会を持っている人は、Windows 8 Boot Campに招待されます。



イベントの形式について少し



イベント形式：レクチャー付きの「ガイド付きハッカソン」。



Windows 8と開発ツールがインストールされたラップトップが必要です。



既成のアイデア、チームを用意することをお勧めしますが、何かを考え、その場で志を同じくする人々を見つけることができます。



まず、Kostya Kichinsky kichikによる 、Windows 8のメトロスタイルのアプリケーションの設計についてすべての開発者が知っておくべきことについての講義から始めます。



その後、参加者はチームに分割され、各チームがプロジェクトのアイデアを簡単に提示します。



その後、 stasusからの「Windows 8での開発に関する一般的な」技術講演が予定されています。



チームはプロトタイピングを開始または継続します。



約45分に1回、15分間収集し、アプリケーションの主要な機能をメトロスタイルで実装する方法を分析します。ライフサイクル、連絡先、タイル、通知、スナップなど-その後、チームはこの機能の追加に集中しますあなたのアプリケーション。



一日の終わりには、チームが結果を発表します。



アプリケーションの開発中、コスティアと私は質問や相談のためにチームが利用できるようになります。
