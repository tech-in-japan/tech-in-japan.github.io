Opera Mobile：Operaブラウザのすべてのモバイルバージョン用の1つのインターフェース





2009年12月10日-オスロ、ノルウェー



-Operaは新しいクロスプラットフォームインターフェイスシェルを導入します

-Opera Mobile 10ブラウザーは、Symbian / S60、Windows Mobile、Android、BREWを含む主要プラットフォームの携帯電話メーカーや携帯電話メーカーが利用できるようになりました



Opera Softwareは本日、携帯電話事業者および携帯電話メーカー向けにOpera Mobile 10ブラウザをリリースしました。 このバージョンは、Symbian / S60、Windows Mobile、Android、およびBREWで使用でき、新しいクロスプラットフォームインターフェイスシェルが含まれています。 すべてのOperaブラウザの外観とユーザーのメリットを統合するプログラム全体の一部として、このシェルにより、電話事業者とメーカーは、ブランドやモバイルデバイスプラットフォームに関係なく、同じインターネットエクスペリエンスをユーザーに提供できます。



新しいインターフェースの構成要素

クロスプラットフォームシェルは、ブラウザのコアに対するアドオンです。 これにより、Opera MobileおよびOpera Mini内のすべてのプラットフォームで作業する機会が与えられ、動的なユーザーインターフェースを作成できます。 したがって、モバイルオペレータと電話メーカーは、最小限の統合で、配布するすべての電話に単一のブラウザインターフェイスを展開でき、実装にかかる時間と費用の両方を節約できます。 エンドユーザーは、インターネットにアクセスするための使い慣れた単一のツールを受け取ります。これにより、電話のオペレーターとメーカーへの忠誠度が高まります。



「1つのタイプの携帯電話を提供するオペレーターはいないため、Operaはさまざまなデバイスやプラットフォーム用のユーザーインターフェイスを作成するソリューションを開発しました。 クロスプラットフォームのユーザーインターフェースのおかげで、さまざまなブランドの要件に合わせてインターフェースをカスタマイズできます」とOpera SoftwareのCEOであるJon von Tetzchner氏は述べています。 「当社のインターフェースはすでに数百万台のモバイルデバイスに展開されており、私たちが受け取った膨大な肯定的なフィードバックは、Operaがインターネットへの快適で効率的なアクセスのためのユニークなユーザーソリューションの開発における認知されたリーダーであることを示唆しています」



Opera Mobile 10は、世界中の1億3500万以上のデバイスにインストールされているスマートフォンと強力な携帯電話向けのブラウザです。 ブラウザには、モバイルWebでの作業をますます簡単にするすべての機能があります。 機能セットには、タブ、お気に入りのリソースにすばやくアクセスするためのエクスプレスパネル、およびパスワードマネージャーが含まれます。 最も人気のあるプラットフォーム上のデバイスのオペレーターおよびメーカー向けの本日のリリースは、さらに多くのユーザーがOperaブラウザーにアクセスできるようにします。

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
