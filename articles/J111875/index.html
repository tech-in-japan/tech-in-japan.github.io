Windows 7およびWindows Server 2008 R2の最終SP1署名済み

本日、MicrosoftはWindows 7およびWindows Server 2008 R2の最初のサービスパック（SP1）の最終RTMアセンブリを社内Webサイトに掲載しました。 署名はバージョン7601.17514.101119-1850でした。これはゴールドになり、Microsoftは近い将来にパートナーに提供する予定です。

ほとんどの場合、SP1は2月にのみ公開されます。 噂を信じるなら、これはMWC 2011で起こる可能性がありますが、このイベントはモバイルデバイス、特に電話専用であるため、この情報は疑わしいものです。



Microsoftは、Tech-Edカンファレンスの夏にWindows 7 SP1とWindows Server 2008 R2 SP1の計画を発表しました。 Service Pack 1には、クラウドコンピューティングの時代に合わせて、クライアントとサーバーのWindowsの現在のバージョンを準備するために設計された多くの新しい仮想化技術と同様に、従来の修正プログラムが含まれます。 これはまず、RemoteFXテクノロジーに関するもので、ユーザーをリモートで接続するときに豊富なマルチメディア機能を提供する必要があります。



以前にSP1のベータ版をインストールしたユーザーは、 先日 、更新プログラムKB976902に注意を払う必要があります。 それらを問​​題なくアンインストールできるように設計されており、また、最終SP1のその後のインストールで問題が発生しないように設計されています。 システムにこの更新プログラムが存在することが、サービスパックをインストールするための前提条件になります。
