200ドルのフルHDスクリーンを備えたスマートフォンですか？ 簡単！

会いましょう！ iOcean X7-1920 x 1080ピクセルの画面解像度を持つ5インチのスマートフォン：





4つのCortex-A7コア、2つのSIMカード、優れた8MPカメラ、およびこれらすべてがわずか200ドルを超える価格で提供されています。



►仕様

画面：5インチ、1920x1080、IPS

プロセッサー：Mediatek MT6589（Cortex-A7 1.2GHz、4コア）

ビデオアクセラレータ：PowerVR SGX544

オペレーティングシステム：Android 4.2.1（Jelly Bean）

RAM：1GB

ユーザーメモリ：4 GB + microSDスロット、最大32 GBのカードをサポート

カメラ：フロント1MP、リア-8MP

通信：2 SIMカード（GSM + WCDMA）、GPS、Bluetooth、Wifi（802.11 b / g / n）

バッテリー：2000mAh

寸法：141 x 69 x 9.68（9.38）mm



クアッドコアMediatek MT6589チップは、2013年にすでに中国製のスマートフォンの標準になっています。 周波数が最大1.2 GHzのエネルギー効率の高いCortex-A7は、優れたパフォーマンス結果を示しています。 プラットフォームの唯一の欠点は、2 GBのRAMを搭載したデバイスがないことです。 当初、MT6589と2GBのメモリ間の互換性の問題についてでしたが、現時点では2GBチップの供給に関する問題についてのうわさがあります。 2GBのRAMを搭載した最初のデバイスは、Dakele Big Cola 2とUMI X2で、これらの販売開始日は5月下旬に予定されています。

多くのメーカーは、出荷時の損傷を防ぐために、非常に厚い段ボールを箱に標準使用しています。

ボックス自体は白で、上部に簡潔なX7の碑文があります。





内部はスマートフォンです。

フロント1MPカメラ：

裏表紙はプラスチックで作られており、生産において若干の誤解が生じました。 最初のバッチでは、蓋が厚すぎて、後で別の蓋と交換されたため、全体の厚さが9.68mmから9.38mmに減少しました。 3000mAhを購入すると、キットには別の黒いカバーが含まれます。



フラッシュ付き背面8MPカメラ：





電源ボタンは右側にあります。





左-ボリュームロッカー：

マイクとmicroUSB入力：





すべてのインターフェイスの場所は非常に標準的です。



ボンネットの下には、GSM / WCDMAネットワークで使用するSIMカード用の2つのスロット（そのうちの1つはmicroSIM）、microSDメモリカード用のスロットがあります。 2000mAhバッテリーと同様。

►画面

画面は本当に良いです。 5インチの1920x1080ピクセルの解像度は問題ありません！

5インチという一見大きなサイズにも関わらず、スマートフォンは手の中に快適に置かれています。



画面の周囲のフレームは非常に薄く、全体の幅にプラスの影響を与えることに注意してください。



この解像度では、ピクセルは完全に区別できません。





どんな傾きでも、画像は鮮明なままです。





視野角は、優れたIPSスクリーンに対応しています。 区別できること-傾けると、黒は可能な限り黒のままになり、グレーは輝きません。







►ゲーム

タッチスクリーンはアーケードゲームに最適です。



3Dゲームでは、MT6589の最適化の問題が原因で問題が発生する可能性があります。 しかし、ほとんどのゲームはまだ動作し、フルHD画面解像度による問題はありません。







►音

イヤホンとマイクの品質は優れています。 スピーカーフォンは良好で、高周波数が優勢です。 振動アラートも存在しますが、それほど強くなく、衣服からはほとんど見えません。



►Webブラウジング

443PPIピクセル密度により、ページ上の最小のテキストでも見ることができます。 MT6577と比較して、MT6589チップによるページの処理速度（ロード/スクロール/スケーリング）は大幅に向上しました。





►Bluetooth 、Wifi

Wifiは、他の中国のスマートフォンと同様に正常に機能します。3か所の中断がすでに可能になった後、1-2の壁を通して信号レベルは安定しています。 Bluetoothは現在動作しています。
