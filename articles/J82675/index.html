ASUS M60J-マルチメディア品種のラップトップ

Intelは一歩を踏み出しました-モバイルソリューションを含むCoreファミリーの新しいプロセッサをリリースしました。 現在、店の棚が再びいっぱいになり、買い手が再び選択する権利を与えられるように、新製品を出すのは残りのメーカー次第です。







ASUSはこれらのエキサイティングなレースにも参加しますが、Core i7のノートブックは昨年同社によってリリースされました。 実際、カットの下-それらの1つの小さな概要。



リリース時点では、新しいIntel Core i7プロセッサで実行されているロシア市場で最初のラップトップでした。 正確には、 Core i7 Q820 プロセッサ （公称クロック周波数は1.73 GHz、ターボモード3.06 GHz）が搭載されているモバイルプロセッサの最上位バージョンであり、「大人」の兄弟からすべてのベストを継承しています。 特に、 Turbo BoostやHyperThreadingなどのテクノロジー（その楽しさはすでに部分的に議論されています）。 ラップトップタスクマネージャーを見ると、友達がどれほど親しくなるか想像できますか？ :)

生産性の高いビジネスでIntel PM55チップセットで実行されるプロセッサは、4GBのRAM（DDR3、1333MHz）と、1GBのメモリとオンボードのCUDAサポートを備えたディスクリートグラフィックカードnVidia GeForce GT 240Mによってサポートされています。 さらに、2台の320GBハードドライブ。







ここでも、箱の中身に驚きはありませんでした。ラップトップ自体、アダプター、マウス、説明書、ソフトウェアディスク、その他すべてです。 まあ、それもありがとう:)



外観



ASUS M60Jラップトップ（およびM60Vpモデル）との出会いでユーザーの目を引く最初のことは、ラップトップのトップカバー（および他の身体部分）の材料です。 はい、はい、これは蒸発メタライゼーション技術を使用して作られた、非常にファッショナブルな微細な研削テクスチャーを備えたアルミニウムです（これについては少し書きました ）。 実際、審美的な魅力に加えて、このようなケースは実際には非常に信頼性があります-私はこれを同様のコーティングを施した私のEeePC 1002HAネットブックの例で確信しました。







ラップトップの寸法は、画面の対角線（16インチ）によって決まり、（mm）392x278x33-44です。 これらはすべて、バッテリーを搭載した3.3 kgというかなり大きな重量です。このデバイスから、デバイスは常に持ち歩くことよりも家庭用（デスクトップを交換することさえも）を目的としていると結論付けることができます。



ワイドスクリーンスクリーン（最大解像度1366x768）は保護ガラスの層で覆われ、保護特性に加えて、物理フレームのスクリーンを奪います-便利なほこりが長持ちします。画面...







画面の真下には、2つのAltec Lansingスピーカーと、散在するタッチボタン（AI Touch Mediaと呼ばれる）があり、スピーカー間のほぼ全体のスペースを占めています。







動作中は青く光ります。それ以外の場合は、単に音楽や音量を操作したり、ブラウザでのナビゲーションに少し貢献したりできます。











キーボード-多くのASUSノートパソコンにインストールされているキーボードと同じ-小さくて静かなストロークの鋭い角と縁の黒いボタン。 このモデルの機能は、デジタルキーボードの存在と考えることができます。



クロムメッキのタッチパッドボタンは指紋スキャナーによって分離されています。指紋スキャナーは、多くの場所でログインおよび認証するための便利なツールになります（パスワードコマンダーなどの特定のソフトウェアを使用）。







ところで、システムに入るには、ASUS SmartLogonプログラムで内蔵Webカメラ（ここでは2メガピクセル）を使用できます。 便利に）







すべての側面を詳細に説明するわけではありません-すべてのユーザーを満足させるコネクタのセットをリストします;）したがって、4xUSB、eSATA、HDMI、DisplayPortおよびD-Sub、IEEE1394、ExpressCard / 54、RJ45、および3つのオーディオコネクタ（SPDIFを含む）。 まあ、Blu-rayドライブとカードリーダー（SD / MMC / MS / MSProフォーマットカード用）がなければ、できませんでした。



性能



テストインスタンスでは、Windows Vista Ultimateがインストールされており、ラップトップのインデックスが5.6（プロセッサが最低評価を受け、残りのスコアが5.9以下）を7.9から評価しました。 しかし、ラップトップはWin7がプリインストールされた状態ですでに販売されているということがわかります。







VLCプログラムでは、提供されたすべての映画が、非常に高品質のBDリッピングに達しました。 スピーカーからの音はかなり良いです...それから、ラップトップは映画を見るのに理想的であると結論付けました。 ゲーム...私も、すべてがうまくいっているように見えました。 もっと正確に言えば、私はクライシスで持っていた唯一のCDを試してみましたが、結果は再生可能以上のものでした。 負荷がかかると、ラップトップはややうるさくなり、アイドルモードでは少しだけ聞こえます。







システムの生産的なコンポーネントがおいしい食事を好むのは論理的です-標準バッテリー（6セル、4800mAh、11.1V、53Wh）はバッテリー寿命が3時間しか持続しません。
