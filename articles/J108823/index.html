ApacheはGoogle Waveに新しい命を吹き込みます

Apache Software Foundation（ASF）は、Google Waveの概念をフリーソフトウェアとして開発し続けることを決定しました。 プロジェクトの説明は、Apache Incubatorの提案に含まれています。 ソースコードは 、Google CodeからASFサーバーに移動します。 waveprotocol.orgのコミュニティもApacheインキュベーターに移動します。



Waveの作成者には、オンラインコミュニケーションの方法と方法に革命を起こし、電子メール、チャット、ブログ、ドキュメント共有など、あらゆるコミュニケーション手段を組み合わせるという野心的な目標がありました。 しかし、Googleはこのプラットフォームの真の可能性を実現することができず、大衆に興味を抱きませんでした。 ユーザーの最初の熱意は無関心に取って代わられ、その結果、この製品の開発を放棄することになりました。



しかし、これは技術の死を意味するものではありません。 Google WavesはZIPアーカイブにエクスポートし、WIAB（Wave in a Box）などの代替環境にインポートできます。 Apache Software Foundationは、WISFをASFプロジェクトとして開発し続けたいと考えています。



プロジェクトの説明には、クライアントとサーバーの部分で実行される予定の特定の機能が記載されています。



どうやら、Apache組織は、一部の大企業のテクノロジーへの関心が明らかになったときに、このプロジェクトを開始することを決定しました。 公開された提案は、Apache Waveプロジェクトのサポートは、米国空軍でのWaveの使用に関連するNovell、SAP、Waveエコシステムのスタートアップ、および多くの独立した個人から期待されると述べました。 これらの人々と組織は、Waveの主要な開発者のようです。



Apacheがプロジェクトを真に復活させ、その周りの開発者の幅広いコミュニティを結集できれば、「波」は彼らが考えていたものを達成できるでしょう-オンライン通信の革命。
