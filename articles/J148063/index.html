VKカップ2012-いかがでしたか









スケジュールによると、 昨日は昨日VKカップ2012ファイナルの最終日が行われ、イベントの印象を共有する機会がありました。



私は自分自身とCodeforcesプロジェクトに代わってもっと書きますが、参加者とコミュニケーションをとることができましたが、すべてのコメントを注意深く読んでください。



VKカップ2012プログラミングチャンピオンシップは、特に若いプログラマーにとって有用で興味深いイベントとして考えられていました。レビューによって判断して、なんとかできました。 選手権のルールには、「登録時の年齢が14歳以上23歳以下」という珍しい項目がありました。 それが、小学生と学生のための競争をするために、プロの経験豊富な参加者の輪から抜け出すことが判明した方法です。 たとえば、決勝戦では初めて多くの人がそのようなイベントに参加しました。



登録は2月末に開始され、参加者数に対する期待をかなり迅速に下回りました。 完了時には、約90か国と約750の都市から約6,000の登録が記録されました。 その瞬間、一部の経験豊富なコミュニティメンバーが年齢制限ルールを表明しました。 あなたはそれらを理解することができます:)



チャンピオンシップの選択と決勝の会場はCodeforcesでした 。 VKontakteの従業員が審査員の大部分を占め、サラトフ州立大学のチャンピオンシップのベテランであるCodeforcesチームも審査員を務めました。



選択グリッドは、資格と3回の予選で構成されていました。 ルールで珍しいのは、最初の試行でそれを行うことができなかった人のために次のラウンドに行く2回目のチャンスを与えるように設計された2つの追加のワイルドカードラウンドでした。











これらのラウンドの最初は、エキゾチックで難解なプログラミング言語の大ファンである悪名高いMasha Nicolette Mikhailovaによって準備されました。 それはサプライズランゲージラウンドでした-参加者がラウンドの最初にのみタスクを渡すことができる言語を学習するときの珍しい形式の競争です。 最初のワイルドカードラウンドの言語は、珍しい要因によって選ばれました。 その結果、参加者はスピードのためにスタックパラダイムを整理し、簡単なタスクを解決しました。 ここでは、たとえば、タスクの1つのウィンガーコード（任意の番号システムで番号を追加）： 



USING: io kernel math math.parser sequences locals unicode.case arrays ; IN: g :: fromdig ( c -- i ) c CHAR: 9 <= [ c CHAR: 0 - ] [ c CHAR: A - 10 + ] if ; :: todig ( i -- c ) i 9 <= [ i CHAR: 0 + ] [ i CHAR: A + 10 - ] if ; :: toint ( string radix -- int ) string 0 [| nc | n radix * c fromdig + ] reduce ; :: fromint ( int radix -- string ) "" int [| str int | str int radix mod todig prefix int radix /i dup zero? not ] loop drop ; readln string>number readln string>number [| n radix | 0 n [ readln radix toint + ] times radix fromint print ] call




それは珍しくて楽しいことが判明し、最も気に入った。 どうもありがとう、 ニコレット ！ ここにラウンドの完全な結果がありますので、これらの問題を解決するためにあなたの手を今試してみてください。



2回目のワイルドカードラウンドは1週間続きました。 その結果、オンライン予選ラウンドの決勝戦の最後のクルマに飛び込むことができ、参加者は真剣に挑戦しました。 ちなみに、このラウンドのタスクのアイデアは再びNicoletteによって提案され、Codeforceチームはインフラストラクチャ全体を準備しました。 ラウンドは丸1週間続いたため、長方形を指定された長方形に詰め込み、 目的関数を最大化する必要がありました。 参加者はこのタスクを気に入っており、先を見据えて、チャンピオンシップの勝者がこのワイルドカードを介してラウンド3に行ったことに注目します。



資格と予選ラウンドは、通常の参加者のCodeforceルールに従って開催されました。 VKontakte、Codeforces、サラトフ州立大学の従業員がタスクに取り組みました。 ニコレットは条件を編集しました。 条件やタスクに大きな失敗はなく、すべてが参加者と審査員の相互の喜びになったと言わなければなりません。 ステージの勝者は次のとおりです。



  • 資格1：Ra16bit（Yuri Pisarchuk）、ベラルーシ
  • 予選2：ラトビア、ロクヤン（エドガーロクヤン）
  • ラウンド1：RAVEman（アントンライチュク）、アメリカ
  • ラウンド2：観光客（Gennady Korotkevich）、ベラルーシ
  • ラウンド3：hos.lyric（保坂和弘）、日本




ラウンド3の結果によれば、50人のファイナリストのリストが決定されました。 ファイナルに参加している国のリストは、ロシア、中国、日本、ベラルーシ、インドネシア、ウクライナ、ポーランド、台湾（そこから参加者が国のステータスを主張）、アメリカ、ドイツ、香港（そして彼らが主張）、インドです。 通常、かなりの数のファイナリストがアジア出身であることが判明しました。



チャンピオンシップファイナルの準備は数ヶ月でした。 ロシアに未成年の外国人グループを連れてくることはそれほど簡単ではないことが判明しました。 審査員チームは、最終ラウンドだけでなく、非公式のCodeRacesゲームの準備も行いましたが、その詳細は事前に参加者に完全に開示されていませんでした。











7月13日、サンクトペテルブルクのまさに中心部、ケンピンスキーモイカ22ホテルのチャンピオンシップのオープニングが行われました。 参加者は、非公式の写真のスライドショーを通じてお互いに紹介されました。 プレゼンターは、例えば、「彼の夢の街について尋ねられたとき、neex.emilはAnkh-Morporkと呼ばれました」、「SergeyRogulenkoは6歳で最初のコード行を書きました！」











ファイナリストには、小さいながらも楽しいお土産が贈られました。 参加者への短いが燃えるようなスピーチで、Ilya Perekopsky（VKontakteの副社長）、I（Mikhail Mirzayanov、Codeforcesの責任者、SSUの代表）が参加者に挨拶しました。 少し後に、ヨーロッパからのフライトの問題をなんとか奇跡的に解決したパヴェルデュロフが会場に現れました。 歓迎のスピーチで、彼は賞金プールの増加を発表し、それによって観客の拍手が打たれました。 したがって、数日で男たちは競争しなければなりませんでした：



  • 1位-$ 30,000;
  • 2位-$ 20,000;
  • 3位-$ 10,000;
  • 4-5位-2000ドル;
  • 6-10位-1000ドル。




同じ日、誰もがサンクトペテルブルクの川や運河に沿って散歩を楽しむことができました。 私たちのガイドがなぜ歴史の哀れな側面をそんなに好むのかと思います。 クトゥーゾフの言及で、私たちのガイドは彼の死の状況を即座に語りました。 私たちは誰が十字架にいたのか、そして王室の死について知りました。 どんな外国人がこれを聞いているのだろうか？ 実際、私はどういうわけかお祝いしたかったです！











VKontakteのリーダーシップは、ファイナリストと完全に接触していました。 パベルはいくつかの計画を共有し、彼らの印象に耳を傾けました。 旅行はかなり長い間続いたが、退屈したものはなかった。



翌日は、主に非公式のゲームイベントCodeRacesに捧げられました。 参加者は、フィールドを走り抜ける車の人工知能を書き、旗を集め、タイヤを発射し、ヒットポイントが終了すると青い炎で燃えるように求められました。 追加のエンターテイメントは、フィールドステーションとサービスステーションの2つの障害物によってもたらされました。 参加者は、ボットを3時間半熱心にコーディングしました。 Pascal、C ++、またはJavaを使用できます。 ほとんどがC ++を選択しましたが、Pascalは完全に注意を奪われた可能性があります。 まあ、それで地獄に:)ところで、はるかに広い言語のリストは競争自体で利用可能です：C / C ++、Pascal、Java、C＃、Python、Ruby、PHP、OCaml、Haskell、Perl、Scala。











同日の夕方、ホテルの宴会場でショーが開催され、トーナメント中に勝者が決定されました。 トーナメントのすべてのゲームは大画面で表示され、参加者は表情豊かに病気でした。 ファイナルに近づくほど、よりインテリジェントな戦略が残ることがはっきりとわかりました。 多くの参加者が障害を回避することを学んだのは面白いが、誰もが壁に衝突した。 4つの戦略が最終段階に達しました：イェプトン（ロシア）、ルドラデバサック（インド）、観光客（ベラルーシ）、イウィウィ（日本）。 エゴール・スヴォーロフ（イェプトン）は決勝前に首位だったが、彼の最強のライバルに対する決勝では、彼の戦略は私たちを失望させ、rudradevbasakが首位になった。 以下では、3つのファイナルのうちの1つのビデオ録画を見ることができます。 上位3位が賞を受賞し、参加することによるエネルギーのブーストがすべてです。















この日は、ロシア語、中国語の風味がなければできませんでした。 中王国の動揺からのファイナリストの一人は、ネフスキーでいくつかの悪党が彼をカメラから奪ったと報告しました。 彼はすべてのイベントで得点し、警察に行こうとしていましたが、VKontakteは参加者に損失を補償し、状況を外交的に静めることにしました。 ちなみに、彼は驚かされず、モデルについて尋ねられたとき、彼はかなり病弱なものを書きました（正確な数字は覚えていません）。 ところで、彼らが後でカメラを買って彼に引き渡したとき、彼はそれを混ぜ合わせて5Dマークを念頭に置いていたと言いましたが、ここで彼はまだ警察に行くように申し出られました。 あなたは信じないでしょう：彼は行きませんでした:)



翌日は、メインイベントであるVKカップ2012の最終ラウンドに専念しました。VKontakteの本部での11時、決勝戦が開始されました。 最初（15分）のタスクEは、悪名高い観光客（ベラルーシのGennady Korotkevich）によって渡され、残りの参加者にどのタスクがより簡単かを示しました。 続いて、theycallhimtom（米国）、tmt514（香港）が続きました。 中国の参加者s-quarkはタスクCを最初に開いた人で、21分で合格しました。 さらに、多くの参加者がこれら2つのタスクを取り始めましたが、vlad89（ウクライナのVlad Simonenko）が間もなくBをオープンしました。これらのタスクは決勝で最も解決されたタスクであることが判明しました。 私たちの同胞はクイックスタートを示すことができませんでした-最初の私たちのリーダーはナタリア（ロシアのナタリア・ボンダレンコ）で、29分でタスクCを解決しました。彼らのほとんどはアジアから来た人たちです。 終了の15分前に、Gennady Korotkevichが最も難しいタスクAを送信し、彼のソリューションは予備テストに合格し、コーディングが終了するまでに順位を決めました。 彼に続いて、中国からのsevenkplusとs-quarkが続きました。 しかし、先の最終テストは一連のテストであったため、結論を出すには時期尚早でした。 コンテストの終わりに向かって、ACM-ICPC（Student Programming World Programming Championships）のエグゼクティブディレクターであるWilliam Poucherが最終ラウンドを訪れました。 彼は偉大な権威と驚くべきカリスマ性を持ち、参加者と話し、写真を撮ってフィナーレを明るくしました。



















連中は昼食に出かけ、ju審員はすべてのソリューションをテストし、最終結果を示すプレゼンテーションを準備しました。 すでにここで、ファイナル中に、参加者はタスクについて単一の質問をしなかったことが判明しました。 このような経験のあるイベントの主催者として、これは例外的なケースであると言えます。 さえ考えがフラッシュ-彼らは条件の清潔さに取り組んでいるときにそれをやりすぎた:)



閉会はケンペンスキーホテルの同じ素晴らしいホールで行われ、他の夜も開催されました。 しばらくの間、参加者はおいしい夕食、生演奏、そして彼らの解決策がすべてのテストに合格したかどうかの興奮を楽しみました。 閉会は、世界のプログラミングチャンピオンであるVKontakte開発者のAndrey Lopatinによって2回行われました。 彼がそんなにうまくいくとは思っていませんでした。 よくやった、アンドレイ！ スピーチはウィリアム・ポーチャーによって開催され、その後、結果は短いプレゼンテーションの形で開かれました。 ロシア語圏の聴衆の失望と中国人の喜びの同時期の溜息を聞くのは面白かった。 彼は3位になり、最初の2人の参加者は中国市民でした。 ホール全体の友好的な拍手に、パベル・デュロフは、受賞者の卒業証書と美しいHP Envy 14 Spectreラップトップでトップ10のファイナリストに授与され、トップ3にはシンボルチェックが行われました。 チャンピオンは、VKカップ2012の碑文が付いたスタンドで巨大な星の形で金色の賞を受賞しました。ホールは受賞者に非常に長い間拍手を送りました。
