Habraiting 2018：2018年の最高の素材

こんにちはHabr。



この投稿は、Habréの記事のライフサイクルに関する出版物の論理的な結論です（問題の技術的な側面に興味のある人のための最初と2番目の部分）。その結果、統計分析のためのかなり興味深いツールが作成されました。 この手法は非常に有用であることが判明し、さまざまなパラメーターで記事を見つけることができます。たとえば、最高の「品質」（視聴数に対する評価の比率）を持つ記事、最も極性のあるコメントを持つ最も「論争のある」記事、最もコメントの多い資料などです。







これからいくつかの利点を引き出し、2018年の記事の統計ランキングをまとめます。 理想的には、新年の初めまでにこれを行うのは良いことですが、賢明な考えは実際に起こり、遅くなります。 しかし、これは一度も見逃した人に役立つ記事を再読するでしょう。 また、テキストの最後にある「秘密のボーナス」は、十分に好奇心が強い人向けです。



猫の下で何が起こったかに興味がある人は、してください。



データ収集



結果をすぐに見たい場合は、この部分をスキップできます。



前のパートと同様に、記事に関する情報を収集するためにPythonとBeautifulSoapライブラリを使用します。 その助けを借りて、出版時期と記事のタイトルに関する情報を入手します。 他のすべてのデータはすでに当社から取得されており、希望する人は前の部分を参照できます。



その結果、次のような機能が得られました。 



def get_article_data(page_link): data_html = get_as_str(page_link) soup = BeautifulSoup(data_html, 'html.parser') str_log = "" # Get datetime t = soup.find("span", {"class": "post__time"}) if t is None: return str_log += t['data-time_published'] + ',' # Get title and link url = soup.find("meta", property="og:url") url_str = url["content"] if url else "-" title = soup.find("meta", property="og:title") title_str = title["content"] if title else "-" str_log += '{},"{}",'.format(url_str, title_str.encode('utf8')) # Extract other parameters (rating, etc) str_log += extract_as_csv(data_html) print(str_log) return str_log


関数の結果は、 '2007-06-12T12：58Z、https：//habr.com/en/post/35001/、 "盗作についてどう思いますか？"、投票：0、票プラス：0、票ミン：0という形式の行になります。 、ブックマーク：0、ビュー：161、コメント：1 '、csvファイルに簡単に保存できます。



さらに、彼らが言うように、技術の問題。 幸いなことに、Habréのすべての記事にはエンドツーエンドの番号が付けられているため、入力して目的の日付範囲を見つけることは難しくありません（ちなみに、「デジタル考古学」を行う人はhttps://habr.com/en/post/1の番号の記事を読むことができます/およびhttps://habr.com/ru/post/2/、2006年に既に公開されています ）。 その後、パーサーをループで起動して一晩放置するだけでした（サーバーをロードしないように、リクエスト間に一時停止が追加されたため、プロセスはあまり速くありませんでした）。



企業ブログには疑問がありましたが、判明したように、リンクを開いたときにリダイレクトが発生し、同じ番号が付けられています。 したがって、すべてのデータが完全にダウンロードされ、何も失われませんでした。



サンプル全体について、さまざまなリクエストを行い、何が起こったかを確認します。 技術とソースコードについては2番目のパートで説明しました。希望する人は、この手法をさらに詳しく学習できます。



結果



それでは始めましょう。 リンクは、追加のコメントなしでさらに公開されます。 もちろん、いくつかの記事は一度にいくつかのカテゴリに分類されるかもしれません。 たとえば、多数のビューとコメントを獲得しました。 パーサーによって一部の記事がスキップされることもあります。たとえば、その時点でサーバーがタイムアウトを返した場合、絶対的な信頼性は保証されませんが、この「評価」は非公式です。



編集：コメントで示唆されているように、一部の記事はまだリストに載っていません。 これは、一部の著者が2017年に記事を作成し、翌年に記事を発表したという事実に一部起因しています。 データは更新されます。追加に従ってください:)著者の一人がランキングで自分自身を見つけられなかった場合-個人またはコメントで書き込み、私は追加します。
