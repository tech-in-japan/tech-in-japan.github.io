2011年の過小評価されているWebアプリケーションのトップ10

私たちは定期的に新しいWebアプリケーションに出会いますが、すべてのアプリケーションが注目に値するわけではありません。 そして、2011年の最後の日には、そのうちの10個についてお話ししたいと思います。



10.ガジェット







リストを1つではなく、すぐに一連のアプリケーションで開始することにしました。 1つの単純な機能を実行するものについて詳しく説明しますが、完全に実行します。 その中には、 すべてのタイムゾーン （ すべてのタイムゾーンの現在の時刻を表示）、 Umbrella Todayがありますか？ （今日傘が必要な場合はお知らせします）およびnotepad.cc （ミニマルなメモ帳）。 Spreederもあり、通常よりもはるかに高速にテキスト情報を読み取ることができます。 SleepyTime-最適な時間に目を覚ますためにベッドに行く必要があるときに発言します。 SimplyNoiseと同様に、いわゆるホワイトノイズを生成できます。



舞台裏にはまだ多くの「スマート」で便利なWebユーティリティがありますが、説明のためにここでは十分なスペースがありません。 しかし、ここではまだ少し後です： 0to255 、 Mixest 、 PDFMyURL 、およびCopy Paste Character 。





9. Aherk！







Aherkは、自分自身を脅迫することができる興味深いWebアプリケーションです。 誰もあなたを見ないような形で、あなたの愛する自分の写真があるとしましょう。 また、絶対にやりたくない仕事もあります。 皿洗いをしてください。



上記の写真をAherkにアップロードします！ 作業を行う期限を設定します。 これが完了したら、プレートを鮮やかに取り上げて、Facebookに光沢のある料理の写真を投稿します。ここで、友達は割り当ての事実を確認する必要があります。 友達やあなた自身の怠があなたを失望させたら、Aherk！ 不運な写真を公開します。



aherk.com



8.アーサナ



http://www.youtube.com/watch?v=liO5VbbIqIs



Asanaは、人々のグループが複数のプロジェクトで協力するのを支援します。 各ユーザーは、グローバルな何かの一部である責任、タスク、およびサブプロジェクトを監視できます。 彼らは、同僚が他のプロジェクトで何をしているかを確認し、毎回サイトに登録することなく、すぐに更新を受け取ることができます。



一連のプロジェクトでチームで作業している場合、Asanaは間違いなく知る価値があります。



asana.com



7.クリプトキャット



http://vimeo.com/24583701



Cryptocatは、プライベートインスタントメッセージングを大幅に簡素化します。 指定された名前でチャットを作成し、自分のアドレスを送信することは、安全なチャットを作成するために必要なすべてであり、最後のメッセージから1時間後にすべての情報がサーバーから消去されます。 これはおそらくスパイには十分ではありませんが、一般ユーザーには十分以上のものが必要です。



crypto.cat



6.チームボックス







Teamboxは、同じチームのメンバー間のコミュニケーションを改善することを目的としています。 彼は、必要に応じて、一度にいくつかの通信方法を提供します。 Twitterでは、@記号の使用を含め、ステータスの更新は次のようになります。 ただし、必要に応じて、より長いメッセージを送信できます。 また、ユーザーはリストで共同作業してページを作成することもできます。



teambox.com



5.思考







毎日あなたを通過するすべての有用で興味深い情報を思い出すのは難しいので、どこかに書くのは理にかなっています。 残念ながら、紙の葉は紛失したり忘れられがちです。 もちろん、人気のあるEvernoteがありますが、それによって上記の問題は関連性を失うことはありません。 ここで、Thinkeryが助けになります。 これは、受信した情報を解釈できるアプリケーションです。 たとえば、Amazonへのリンクを入力すると、システムは特定の製品に興味があることを理解し、その製品に関するデータを追加します。 また、タグを追加するには、単語の前に＃記号を付けるだけです。 そして、これはその機能の広い範囲のほんの一部です 。



thinkery.me



4. Cloudtact







Cloudtactは、すべてのデバイスの連絡先リストを同期して最新の状態に保つWebアプリケーションです。だれかが居住地や電話番号を変更するとすぐにデータを更新するのに多くの時間を費やす必要がなくなります。 ログインして、自分の情報を入力し、友達に同じことをするように頼みます。 その後、Google、Android、iPhone、および/またはMac間でこれらすべてを同期できます。 誰もが自分のデータが正しいことを確認するだけです。 Cloudtactが残りを処理します。



www.cloudtact.com



3. Pixlr







Pixlrは、単純なタスクのためにPhotoshopを置き換えることに成功し、信じられないほど使いやすいです。 写真をインポートするか、最初から始めることができます。 アプリケーションは、レイヤーおよびいくつかのブレンドモードでの作業もサポートします。 さらに、ほとんどすべての通常のPhotoshopショートカットが機能します。



pixlr.com



2. WorkFlowy







WorkFlowyは今年リリースされませんでしたが、一部の読者の要望に応じてリストに追加することにしました。 一見したところ、この素晴らしいリスト作成アプリケーションは、余剰のないプレーンテキストエディタに似ています。 しかし、独自のリストを作成し始めるとすぐに、その要素をナビゲートすることの信じられないほどの便利さがわかります。これは、サイト内をいくぶん連想させます。



workflowy.com



1.これがその後







If This Then That（または略してifttt）は、断然、今年の最も興味深いアプリケーションの1つです（私たちはそれへの広範な指示をコンパイルしました）。 サービスの概念は、条件が満たされたときにアクションを実行することです。 たとえば、彼が天気をチェックして、雨が降っていることが判明した場合、SMSメッセージを受信します（もちろん、最初にそれに応じて設定する必要があります）。 そして、お母さんから新しいメールが届くと、iftttは電話をかけることができます。



これはここ数年で最高のWebアプリケーションの1つであるため、時間をかけて十分に理解してください。 私を信じて、最初の恥ずかしさはすぐに消えます。



ifttt.com
