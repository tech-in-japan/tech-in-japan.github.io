DropBoxを使用して、設定、テーマ、プラグインを保存しますSublime text 2

最近、Sublime Text 2の最終バージョンのリリースに関するトピックを読みました。その機能を検討することに決めたため、文字通りこのテキストエディターに夢中になりました。 彼はすぐに私の開発ツールのステータスを受け取りました。



私は2台のコンピューターで作業していますが、「作業中のすべてのコンピューターでエディターの設定、プラグイン、テーマを同期する方法は？」という質問がありました。



インターネットで少し検索したところ、すべての設定、テーマ、およびプラグインをDropBoxに保存し、それらへのシンボリックリンクをエディターにスリップさせるという非常に簡単なソリューションを提供する興味深い記事に出会いました。 それでは始めましょう。



  1. もちろん、Sublimeテキスト2が開いている場合は閉じます。
  2. DropBoxフォルダー（通常は「〜/ Dropbox /」）にSublime Text 2フォルダーを追加します。
  3. Sublimテキスト2の設定に移動します（通常は「〜/ライブラリ/アプリケーション\サポート/ Sublime \テキスト\ 2 /」、Windows 7では「c：\ Users \ username \ AppData \ Roaming \ Sublime Text 2」）
  4. 作成したフォルダー「〜/ DropBox / Sublime text 2 /」に3つのフォルダーをコピーします。

    • 「インストール済みパッケージ」
    • 「パッケージ」
    • 「元のパッケージ」


  5. オリジナルを削除するか、名前を変更する必要があります
  6. 次に、これらのフォルダーへのシンボリックリンクを作成し、エディターにスリップする必要があります。 異なるオペレーティングシステムでは、リンクは少し異なります。

    • OS X



      1. ターミナルに行きます
      2. エディターフォルダーに移動します（ "cd〜/ Library / Application \ Support / Sublime \ Text \ 2 /"）
      3. 次のコマンドを使用して3つのリンクを作成します
      4. 「Ln -s〜/ Dropbox / Sublime \ Text \ 2 / Installed \ Packages ./Installed \ Packages」
      5. 「Ln -s〜/ Dropbox / Sublime \ Text \ 2 /パッケージ./Packages」
      6. 「Ln -s〜/ Dropbox / Sublime \ Text \ 2 / Pristine \パッケージ./Pristine \パッケージ」


    • 窓



      1. コマンドラインを開き、エディター設定ディレクトリに移動します（ "cd％APPDATA％/ Sublime text 2 /"）
      2. 次のコマンドを使用して3つのリンクを作成します
      3. “ Mklink / D”インストール済みパッケージ““ C：\ユーザー\ユーザー名\ Dropbox \ Sublimeテキスト2 \インストール済みパッケージ“”
      4. “ Mklink / D“パッケージ““ C：\ユーザー\ユーザー名\ Dropbox \ Sublime text 2 \パッケージ“”
      5. 「Mklink / D」元のパッケージ「」C：\ユーザー\ユーザー名\ Dropbox \ Sublime text 2 \元のパッケージ“”


    • Linux



      1. Linuxのリンクは、OS Xとほぼ同じです。
      2. 異なる場合があるのは、プログラムのインストールパスだけです。




  7. Sublimeテキスト2を開始します。すべての設定は、DropBoxで作成したフォルダーから取得されます。




このアイデアがあなたの役に立つことを願っています。
