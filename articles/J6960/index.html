マイクロソフト、Silverlightプラットフォームを発表-Adobe Flash Killer

マイクロソフトは、Adobe Flashを市場から排除したいと考えています。 新しいMicrosoft Silverlightテクノロジー（旧コードネームWindows Presentatoin Foundation Everywhere（WPF / E） ）は、Webサイトから直接ビデオ送信とインタラクティブ機能を提供します。 プラグインは最初からFirefoxとSafariで動作します。つまり、Microsoftはこのテクノロジーを世界標準にするためにあらゆることを行う準備ができています。



Microsoft Server and Toolsの製品管理責任者であるForest Keyは、ラスベガスで開催されたNational Broadcasters Association Conferenceで本日のニュースについて話しました 。 このテクノロジは、新しいWindows Presentation Foundation（WPF）グラフィックエンジンに基づいています 。



マイクロソフトは、コンテンツの発行元およびパートナーのサポートを既に参加しています。 その中には、Akamai Technologies、Brightcove、Eyeblaster、Major League Baseball、Netflixなどがあります。



新しいプラットフォームを広く配布するには、Microsoftが3つの側面からのサポートを必要としています。 コンテンツ発行元に加えて、多数のユーザー（必須のIE更新にSilverlightを含めても問題はありません）や、Silverlightに基づいて新しいインタラクティブアプリケーションを作成することになっている開発者がいます。



将来的には、Windows Server用に計画されているインターネットブロードキャスト用のIIS7 Media Packプラグインなど、他の主要なMicrosoft製品がSilverlight向けに最適化される予定です。 もう1つの重要なコンポーネントは、インタラクティブなオンラインアプリケーションを作成するためのMicrosoft Expression開発ツール（6月にリリース予定）です。これは、最近導入されたAdobe Creative Suite 3に代わるものです。



専門家によると、新しいMicrosoftプラットフォームではAdobe Flashの人気レベルを達成するのは困難ですが、単一のMicrosoft Webインフラストラクチャのユーザーは、すぐにSilverlightとExpressionへの移行を開始できます。



Silverlightテクノロジの作業は2年以上にわたって続けられています。 プラグインのベータ版は4月30日にリリースされる予定で、配布パッケージのサイズは2 MBを超えません。



アドビも帽子をかぶって眠ることはなく、すでにユニバーサルオフラインプレーヤーAdobe Media Playerのリリースを発表しています。
