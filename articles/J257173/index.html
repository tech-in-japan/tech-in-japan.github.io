プレビューおよび無料の全文検索-プラットフォームに統合された高速で柔軟な検索

Azureデータレイク


ビッグデータと分析用のマイクロソフトのツールポートフォリオに新しいツールが登場しました。AzureData Lakeは、ファイルやアカウントのサイズに制限を設けずに、あらゆるタイプとスキームのデータを格納するために使用できるビッグデータのリポジトリです。 Data Lake-HFS HDFS準拠のHadoopファイルシステム、Azure HDInsightが統合され、Revolution-R Enterprise、Hortonworks、Cloudera、Spark、Storm、Flume、Sqoop、Kafkaなどと統合されています。 Data Lakeはまだ利用できませんが、パブリックプレビューが表示されたら登録して情報を受け取ることができます。



Azure SQL Data Warehouse


もう1つのデータツールは、Microsoft Azure SQL Data Warehouseです。 超並列処理（DW） SQL Serverスケーラブルなサービスとしてのストレージ。 機能-ローカルおよびクラウドの両方で機能し、構造化データおよび非構造化データの従量課金コンピューティングとT-SQLサポートを動的にスケーリングできる最初のネイティブクラウドデータストレージソリューションです。

Azure HDInsight、Azure Machine Learning、Azure Data Factory、Power BIと統合します。 これらすべてにより、SQL Data WarehouseはデータとBIをクラウドに保存するための完全なソリューションであると結論付けることができます。 まだ一般公開されていませんが、 登録してお知らせを受け取ることができます。



Azureサービスファブリック


データからコンピューティングに至るまで、プロセスの高度なスケーラビリティと柔軟性を考慮してクラウドサービスを構築できる開発者向けプラットフォームであるAzure Service Fabricの開発者向けプレビュープログラムが発表されました。 Service Fabricは、両方のタイプのサービス（ステートレス/ステートフル）の作成をサポートします。これは、小規模で個別にバージョン管理されたサービスから複雑なアプリケーションを構築するアーキテクチャ上のアプローチです。 Service Fabricは、専用のツールキットを使用してVSに統合されています。 ツールとSDKの詳細をご覧ください 。



Azure App Service Premium


マイクロサービスの歴史は、Azure App Serviceプレミアムモードの発表で続きます。AzureApp Serviceプレミアムモードは、Webアプリケーション、モバイルアプリケーション、論理アプリケーション（a-la IFTTT）、およびAPIの使用を支援するアプリケーションの形でマイクロサービスをホストできるコンテナーベースのサービスです。 App Service Premiumモードは、通常よりも大きいリソースセット（8コアと14 GBのメモリ）からVPNなど、幅広い機能を提供します。



VM、ストレージ、ネットワーク向けのAzureリソースマネージャー


Azure Resource Managerは 、パブリックプレビューでVM、ストレージ、およびネットワーク機能に使用できるようになりました。 ARMは、リソースグループ内に複雑なアプリケーションをデプロイするためのパターンを開発する機能を提供します。これは、Azureでリソースをプールするための新しい概念です。 リソースグループは、ロールモデルを使用して管理できるすべてのリソースと、より高度な制御が必要なタグリソースを1つのコンテナーに結合します。 ARMの発表の中で、Cloud Foundry、Mesosphere、Juju、Apprenda、Jelastic、Scalr、MySQL、Chef、ElasticSearch、Zookeeper、MongoDB、PostGreSQLとのパートナーシップも発表しました。
