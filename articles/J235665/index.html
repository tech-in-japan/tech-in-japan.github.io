ベルリンで発表されたLumia 830、Lumia 730、Lumia 735

ご存知かもしれませんが、最大の家電展示会IFA 2014が9月5日から10日までベルリンで開催されます。 しかし、発表の時はすでに来ています。







「#morelumiaの準備はいいですか？」というスローガンの下で開催されたMicrosoftの特別なイベントの枠組みで、新しいラインから3台のスマートフォンを紹介しました。薄いLumia 830と2つの「セルフィ​​ー」 Lumia 730 DSおよびLumia 735です。



Nokia Lumia 830







イベントの最初の目新しさは、副主力のNokia Lumia 830でした。スマートフォンは、 Nokia Lumia 1020などの前身の成功した設計ソリューションをうまく組み合わせています。 しかし、何よりも、彼は「ビッグブラザー」の旗艦モデルであるルミア930を外観で選びました。





Nokia Lumia 930（写真左）およびNokia Lumia 830



Lumia 830ケースの中心には、春に発表されたフラッグシップと同様に、シルバーまたはブラックのモノリシックな金属フレームがあります。これは、スマートフォンのデザイン要素でもあります。 前部は、わずかに凸状のゴリラ3.0保護ガラスでほぼ完全に覆われており、背面には色付きのポリカーボネート保護カバーがあります。 スマートフォンは真っ黒で、銀色のフレームと白、明るいオレンジ、または明るい緑の裏表紙の組み合わせで表示されます。 同様のデザインの「レシピ」はすでにLumiaの伝統となり、鮮やかなカラフルなデザインの愛好家と保守的なユーザーの両方に集中できます。



新しいスマートフォンの物理的特性に関するもう1つの重要な点は、その厚さと重量です。 これらはそれぞれ8.5 mmと150グラムであり、Lumia 830は現在ラインナップの中で最も薄くて軽いデバイスです。 新しいスマートフォンの長さと幅は139.4 x 70.7 mmです。







技術的なコンポーネントに注意を払う価値があります。 スマートフォンには、Adreno 305グラフィックスチップと1 GBのRAMを搭載したクアッドコア1.2 GHz Qualcomm Snapdragon 400プロセッサが搭載されています。



Lumia 830には、microSDメモリカードで拡張可能な16 GBの内部メモリがあります（最大128 GB）。 さらに、すべてのユーザーは、クラウドOneDriveで15 GBを無料で利用できます。



ClearBlackテクノロジーを搭載した解像度1280 x 720ピクセル（ピクセル密度-296ピクセル/インチ）のクリアな5インチIPSディスプレイは、日光下でも優れた可読性を誇り、超高感度センサーのおかげでさまざまな条件でスマートフォンを簡単に制御できます。







Lumia 830には、PureViewテクノロジー、LEDフラッシュ、ZEISS光学（6レンズ光学システム）、およびこれまでに作成した最軽量かつ最薄の光学式手ぶれ補正システムを備えたメイン10 MPカメラが搭載されています。 このカメラは非常に感度が高いため、暗い場所でも高品質の写真を撮ることができます。 センサーのサイズは1 / 3.4インチ、焦点距離は26 mm、絞りはf / 2.2です。 フロントカメラの解像度は0.9メガピクセルで、自画像やビデオ通話をより効果的に撮影するための広角レンズを備えています。



メインカメラを使用すると、フルHD（1080p）の解像度でビデオを撮影できます。 また、ノキアリッチレコーディングサウンドレコーディングテクノロジーと3つの高性能マイクは、「サラウンドサウンド」形式（5.1）で全範囲にわたって歪みのないサウンドレコーディングを提供します。







2200 mAhの容量を持つ強力な交換可能なバッテリー（今日では主力製品では珍しい）は、最大14.8時間の通話時間を提供します（3Gネットワ​​ーク上）。 Qumiワイヤレス充電でLumia 830バッテリーを充電できます。



スマートフォンは、低消費電力環境でフィットネスアプリケーション（Health and FitnessやAdidas miCoachなど）の情報を収集できるSensorCoreテクノロジーをサポートしています。



新しいLumia 830には、Windows Phone 8.1 Update 1を使用したLumia Denimの最新の更新プログラムが既にプリインストールされています。これについては、後ほど詳しく説明します。



Lumia 830の世界的な販売は、この秋から始まります。 ロシアでは、ノベルティはおよそ10月に登場します。 小売価格も追加で発表されます（予想される費用は、税と割引を除くと約330ユーロです）。

ノキアLumia 730 DS / 735







本日発表されたLumia 730 Dual SIMとLumia 735は、同じモデルの2つのバージョンであり、Skypeでのビデオ通話と自画像（またはいわゆる「自分撮り」）の撮影に焦点を当てた類似の特性とデザインを備えています。 モデル間の主な違いは、Lumia 735はLTEネットワークをサポートし、Lumia 730 DSは2枚のSIMカードの同時操作をサポートすることです。







新しいデバイスの重要な機能は、5メガピクセルの解像度とフルHDビデオの録画機能を備えた強力な正面向き広角カメラです。 このカメラの焦点距離は24 mm、絞りはf / 2.4です。 上記の特性と広角レンズのおかげで、フロントカメラは友人とグループ写真を撮ったり、Skypeでの集合的なコミュニケーションに適しています。 特にスマートフォンのプレゼンテーションのために、高品質のセルフポートレートを撮影するための特別なアプリケーション-Lumia Selfieが開発されました。



新しいLumia 730 DS / 735スマートフォンのメインカメラは、6.7メガピクセルの解像度、オートフォーカス、ZEISS光学、LEDフラッシュを備えています。 センサーのサイズは1 / 3.4インチ、開口部はf / 1.9、焦点距離は26 mmです。

一部の市場では、新しいスマートフォンのユーザーは、携帯電話または固定電話に国際電話をかけるための3か月間無料のSkype Worldwide Unlimitedサブスクリプションも利用できます。



これらのデバイスには、アスペクト比16：9、ピクセル密度316ピクセル/インチの解像度1280 x 720ピクセルの4.7インチOLEDディスプレイが搭載されています。 スマートフォンはClearBlackテクノロジーをサポートし、Lumia 830と同様に、高感度センサーを備えています。

Lumiaのユニークな機能


Lumia Denimアップデートには、多くの排他的な機能も含まれています。 多くの機能はハードウェアに依存しており、これまでのところLumia 930、Lumia 1520、およびLumia 830についてのみ発表されています。他のLumiaデバイスでの特定の機能の可用性に関する情報は、このアップデートの配信と同時に提供されます。



「 Lumia Camera 」-「Nokia Camera」アプリケーションを置き換えます。 改良された射撃アルゴリズム、新機能、より直感的なコントロールが特徴です。 Lumia Cameraを使用すると、ユーザーはカメラをすばやくオンにして、数ミリ秒のフレーム間でポーズをとって連続撮影するため、重要な瞬間を見逃すことはありません。







アプリケーションのカメラボタンを長押しするとアクティブになるモーメントキャプチャ機能は、4Kの解像度と24フレーム/秒の速度でビデオ録画を自動的に開始します。 ビデオの各フレームの解像度は8.3メガピクセルです。つまり、各フレームを静止画像として保存できます。

Rich CaptureはHDRサポートを追加し、ユーザーが最初に写真を撮ってから、設定を調整して最適なオプションを選択できるようにします。



改善されたGlance画面 （スマートフォンロックモードで表示される画面）-それをサポートするLumiaデバイスは、たとえば天気や健康とフィットネスアプリケーションからの日付や通知に加えて、他の情報を表示できます。
