新しいネットワーク世代：導入された最初の5G仕様

12月末に、3GPPコンソーシアムは、第5世代のモバイルネットワーク-5Gの仕様の完成を正式に発表しました。



ただし、これは、既存の非スタンドアロンLTEインフラストラクチャに基づいて5Gネットワ​​ークを展開するための5G New Radio（NR）標準を説明する公式文書の最初の部分にすぎません。 スタンドアロンインフラストラクチャ（スタンドアロン）でのネットワーキングの仕様の第2部は、今年予定されています。



以下に、規格の仕様と機能に関する詳細を説明します。





/ Flickr / japanexperterna.se / cc



事前にリリース



当初、仕様のリリースは2018年3月に予定されていましたが、AT＆T、Sprint、Telstra、およびVodafoneのオペレーターグループの要求に応じて日付を変更することにしました。 Mobile World Congress 2017の前夜、彼らはモバイルインターネットの需要が高まっていると述べたため、可能な限り迅速に顧客のニーズを満たすために作業を開始することが重要です-すでに2019年に。



仕様のリリースをスピードアップするために、リリースは2つの部分に「分割」されました。2017年12月に、非スタンドアロンアーキテクチャの5G機能要件が提示され、スタンドアロンアーキテクチャ（スタンドアロン）のドキュメントの準備は年の半ばまで延期されました。 ただし、このために、特定の技術的側面の実装を「残す」必要がありました。



延期された決定



12月のリリースでは、3GPP は 、たとえばアップストリーム接続でデータ圧縮を実装するなど、いくつかのソリューションの最終決定を拒否しました。 これは、モバイルデバイスがLTEネットワークとNRネットワークで同時に動作する場合に、二重データ伝送を優先し、干渉を排除するシステムを実装するために行われました。



延期されたタスクには、次のものがあります。Nonorthogonal Multiple Access（NOMA）およびeV2X（ 拡張されたVehicle-to-Everything ）での作業。



5Gオプション



エリクソンで述べたように、5Gネットワ​​ークの帯域幅は4Gの10倍になります。 同時に、1平方キロメートルあたり少なくとも100万の接続デバイスをサポートできます。 この機能はIoTテクノロジーをさらに発展させ、スマートシティへの道の別のステップになると予想されます。 5Gネットワ​​ークのスペクトル効率は、ダウンリンクで30ビット/秒/ Hz、アップストリームで15ビット/秒/ Hzになります。



ネットワークステーションは 、最大500 km / hの速度で移動するオブジェクトと通信します。 同時に、使用されている無線インターフェイスは、アイドル状態のときに低電力消費モードに切り替わります（宣言された遷移時間は10ミリ秒です）。 データ送信の遅延については、4 msです。 比較のために、LTEの遅延は20で、3Gでは100ミリ秒です。 情報のダウンロード速度は100 Mbit / sであり、その伝送速度は50 Mbit / sです。



5Gネットワ​​ークテスト



さまざまな国の企業がすでに「5Gレース」に参加しており、新世代ネットワークのテストを開始しています。



イギリス


たとえば、英国では、Vodafone は Huaweiと共同でこれを行いました。 SU-MIMOテストでは 、20 Gbit / sの速度が達成され 、 MU-MIMO -10 Gbit / sが達成されました 。 EEはそれもテストしました。5ミリ秒未満の遅延で2.8ギガビット/秒の安定したデータ転送速度を達成することができました 。



英国文化・メディア・スポーツ省は、同国における5G技術のさらなる発展を目指した戦略計画を作成しました。 公開された文書は、政府の必要な行動を説明し、機器の配置と経済的側面の問題を議論します。



ロシア


昨年、エリクソンとMTS は 5Gネットワ​​ークのプロトタイプを作成し、固定およびモバイル機器の使用を含むテストを正常に完了しました。 企業は25 Gb / sのピークデータ転送速度を達成することができました。 これは、マルチユーザーおよびMassive MIMO、ビームトラッキング、 ダイナミックTDDテクノロジーのおかげで可能になりました。



ロシアでの5Gネットワ​​ークの最初の大規模なテストは、ワールドカップ中に開催される予定です。



日本


サムスンは、日本のオペレーターKDDIと提携して、移動中の列車での5Gのテストの完了を発表しました（速度は時速100 kmを超えていました）。 実験中のデータ転送速度は、ダウンストリームおよびアップストリーム接続で1.7 Gb / sに達しました。



フィンランド


2016年のMobile World Congressで、フィンランドは5Gテストネットワークフィンランドエコシステム（5GTNF）を導入しました。これにより、技術者が市場に正式に参入する前に5Gをテストできます。 このネットワークは、 IT大手のNokia、Ericsson、Huawei、および自動車、エネルギー、製造などの他の業界の企業で使用されています。



たとえば、2017年9月、5GTNFの枠組みの中で、ノキアはWIVEプロジェクトを立ち上げました。 彼の焦点は、メディアおよびエンターテインメント業界における次世代ネットワークの応用です。



実施日



仕様の完全版は2018年6月に表示されます。 大規模な5Gネットワ​​ークの完全な実装は2019年に予定されています。 ただし、プロジェクトの実装プロセスをスピードアップしたい企業があります-AT＆T は 、2018年後半に米国で新世代ネットワークを立ち上げる予定であり、Verizonは同時期に5Gルーターとモデムをアメリカのユーザーに提供する予定です。
