ダンテ、ソックスプロキシを介して動作するソフトウェアを教える

CVSの未加工の生データをマージし、ソフトウェアをコンパイルするために、作業グリッドに緊急に到達する必要がありましたが、悲しいかな、vpnサーバーが落ちました。



私は誰にも栄光を知らないが、「万が一に備えて」私たちの管理者は、キーのみによる認証でsshを介して1つのサーバーを開いた。 それは私を救った...







Danteの作成者のおかげで、まるで本物のVPNにアクセスできるかのようにすべてを行うことができ、まったく消えませんでした。



はじめに-私は小さなdante-clientパッケージをインストールしなければなりませんでした（Debianを持っていますが、他のDanteディストリビューションにそれを置くことが問題になるとは思いません）： 



 sudo aptitude install dante-client




/etc/dante.confを編集し、次の行を追加します。 



 route { from: 0.0.0.0/0 to: 0.0.0.0/0 via: 127.0.0.1 port = 3128 protocol: tcp udp proxyprotocol: socks_v5 method: none }




...そしてVPN-for-the-poorを起動します： 



 ssh -D 3128 user@back-door.big-corp.com




これからのすべて-作業ドラフトの原材料を更新することはまったく問題ではありません。 



 socksify cvs update -d




サーバーにアクセスしてアセンブリを開始します-質問ではありません： 



 socksify ssh me@myserver.big-corp.com




さて、http経由で企業リソースにアクセスするには、Firefoxの優れたアドオンであるFoxyProxyを使用できます



...営業日が節約されました！ :)
