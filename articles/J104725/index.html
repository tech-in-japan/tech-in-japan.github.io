Evercookie-最も永続的なCookie

Samy Mamkarは、Cookieを8か所に保存し、互いに自動的に回復し、あるブラウザーで配信されたCookieが別のブラウザーで機能することを確認できるシステムを開発しました。



このCookieを削除することはほとんど不可能です！ （もちろんすべてが可能ですが、面倒です）



Cookieは次の場所に保存されます。

  • HTTP Cookie
  • ローカル共有オブジェクト（フラッシュ）;
  • HTML5キャンバスを使用して自動生成され、強制的にキャッシュされたPNGのRGB値にクッキーを保存します。
  • ウェブ履歴にクッキーを保存する;
  • HTML5セッションストレージ。
  • HTML5ローカルストレージ
  • HTML5グローバルストレージ
  • SQLiteを介したHTML5データベースストレージ。


これらの場所のいずれかから削除すると、Cookieは残りの場所から自動的に復元されます。 ユーザーが （Flashのローカル共有オブジェクトを介して）ブラウザを変更しても機能します。



説明（英語）とデモ： http : //samy.pl/evercookie/

Cookieを削除し、システムをきれいにしてから戻ってください。
