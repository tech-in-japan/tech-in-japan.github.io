usesthis.comのJohn Resig

7月25日にusesthis.comで公開されたJohn Rezigへのインタビュー





あなたは誰で何をしていますか？



私は JavaScriptプログラマーです。 私はカーンアカデミーで働き、次世代のトレーニングシステムを開発しています。 私はjQuery JavaScriptライブラリの作成者およびチーフ開発者であり、JavaScriptに関する2冊の本-「Pro JavaScript Techniques」および「Secrets of the JavaScript Ninja」の著者です。







どの鉄を使用していますか？



私のメインのコンピューターは、22インチのセカンダリモニターを備えた27インチのiMac（i7プロセッサー、8 GB RAM）であり、メインのラップトップは13インチのMacbook Pro （Core 2 Duo、4 GB RAM）です。 私は、RCNプロバイダーから60Mb / sの速度でインターネットに接続しました（注：彼に月額80ドルかかります）。 Windows XP 、 Chromebook、 MacBookを搭載したPCなど、主にテストに使用するバックアップマシンがいくつかあります。 また、テストには多数の電話とタブレットを使用していますが、それらのほとんどはjQuery Mobileの基盤であるフィラメントグループに属します。

メディアファイルの表示に使用するMac Miniもあります。 そのうちの1つは720pビデオを再生できるプロジェクター（ BenQ W500 ）に接続され、もう1つはシミュレーターの前の19インチLCDスクリーンに接続されています。

コミュニケーションの手段として、私はiPhone 4と最近入手したiPad 2をテストに使用します。これらはKhan Academyの特権です。

キッチンには、iPod / iPhoneのコネクタとスピーカーがあり、料理をしながらiPhoneを接続して音楽を聴くことができます。 私の車にも同様のコネクタがあります。

5つのRAIDを備えた5ベイNAS（ Thecus N5200BR ）に5つの1TBディスク（つまり、約3.6 TBのディスクスペースが使用されています）があります。 すべてのメディアファイルをこのデバイスにコピーし、ハードディスクに障害が発生した場合にバックアップディスクに保存しました。 また、すべてのドキュメントのバックアップはDropboxに保存されます 。



どのソフトウェアを使用していますか？



私はFirefoxを使用しています（ブックマークにGoogle Mailとカレンダーがあります）。 私はGoogle Docsで多くの仕事をし、 Keynoteでプレゼンテーションを作成し、 Textmateでブログに書き込みます。 Skype 、Google Chat（GMail）、IRC（ IRSSIを使用）を介して通信し、 Gyazoを介してスクリーンショットを共有します。

開発には、 Chrome 、 vim （ vim構成ファイル ）、 Textmate 、およびiTerm2を使用します。 Firefox、Chrome、 Opera 、 Safariの複数のバージョンを常に開いています。 VMWare Fusionをさらに多くのブラウザー（ IE 、Firefox、Opera、Chrome、およびSafariのいくつかのバージョンを含む）で実行しています。 MAMPを使用して、テスト用にローカルWebサーバーとMySQL DBを簡単に実行します。 すべてのコードをGit （ Github 、多くの場合閉じたリポジトリ）に保存し、バックアップをDropboxに保存します。

携帯電話には、iPod、時計、メモ、カメラ、設定、NOAA（weather.govを開く）、カレンダー、 Twitter 、 Foursquare 、 Carcassonne 、 MassTransit （ボストンのバスの場所を表示）、 Yelp 、 Facebook 、 iSSH 、マップ、Safari、メッセージ、電話。

ビデオを再生するには、 VLCとBoxee 、 iTunesを使用して音楽を聴きます（Last.fmと同期）。 AirVideoを使用して、iPhoneおよびiPadで映画を視聴します。

iTunes、Amazon MP3、 Netflixストリーミングでファイルを購入し、Usenetでダウンロードします。 Usenetを使用するには、Mac Miniの1つでSABnzbd +を使用します。これはGiganewsに接続し、NZBs.orgまたはNewzbinからニュースを受信します。

私はStarcraft 2 （id：frowny.396、私はSilver Protoss）、 Minecraft 、およびiPhone上のゲーム（CarcassonneやWords with Friendsを含む）をプレイします。 カルカソンヌで私は50位で、私のIDは私のメールアドレスです：jeresig@gmail.com。



自分にとって理想的な構成は何ですか？



私はいくつかの欲求を持っているにもかかわらず、私は彼女にとても近いです。 MacBookに3Gモデムが内蔵されていること、iPadでの入力が簡単であること、Chromeの機能が向上すること、SABnzbd +ファイルが再びダウンロードされないようにスマートであること、iPhoneがより高速なので、MacBookでプレゼンテーションを作成するのにドングルは必要ありません。したがって、Boxeeのクラッシュが少なくなります。そのため、DropboxでファイルをGoogle Docs and Mailにバックアップできるように車に音声入力があります。
