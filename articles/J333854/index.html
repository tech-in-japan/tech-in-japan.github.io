Windows Server仮想サーバープロバイダーのレビュー評価：2017





2017年3月に、MS SQL Serverとターミナルサーバーのサーバーハードウェアをアップグレードする必要がありました。これらは、販売会計アプリケーションが機能するために必要です。 新しいサーバーとストレージシステムのコストは700 trを超えましたが、これは予算に合わなかったため、アプリケーションをホストする仮想サーバーを検討することにしました。



実装コストが高いため、ほとんどすぐに専用サーバーを放棄しました：機器、ライセンス、構成エンジニアのサービス。 そのため、ニーズの実装に既製のプラットフォームを選択することにしました。これは仮想サーバーです。

次の利点を強調できます。



  1. ハイパーバイザーレベルでのフォールトトレランスの実装。

  2. 主にフェイルセーフデータストレージ。

  3. 低バックアップコストと簡単な実装。

  4. ニーズに応じてリソースを拡大します（成長に応じて支払います）。

  5. SLAの可用性。
