YAPC会議::ロシア:: 2015年5月16〜17日にモスクワで開催されるMayPerl





5月16日と17日に、May.Perl会議（YAPC ::ロシア）がMail.Ru Groupのモスクワ事務所で開催されます。 Perlの専門家が世界中から集まり、経験を交換し、分野で新しい知り合いを作り、非公式で快適な雰囲気の中で志を同じくする人々とチャットします。



スピーカーと参加者を招待します。 講演者として話すには、 申請書を提出してください。



リスナーとして参加するには、 登録します。 参加は無料です！



そして、YAPC ::ロシアとは何ですか？



YAPC ::ロシアはかなり古く、有名で尊敬される一連の会議です。 YAPC（Yet Another Perl Conference）の一環として、さまざまな形式のイベントが開催されます。 大陸があり、多かれ少なかれ国家があります。 6月中旬にYAPC :: NAが開催され、9月にグラナダでYAPC :: Europe 、 YAPC :: Asiaなどが開催されます。



YAPC ::ロシアは、Perlに関する今年の主要会議です。 2008年からロシアで開催され、旧ソビエト連邦各地からプログラマーを集めています。 以前は、Perl専用の2つのイベントがありました。キエフとモスクワでは、ウクライナ人はPerl Movaと呼ばれ、ロシア人は5月Perlでした。 その後、彼らは団結し、会場を交互に通過し始めました。 昨年、このイベントは、ティムール・ノザッツェ率いるサンクトペテルブルクのプログラマーSPB.pmのグループによって行われました。



レポートの目的は何ですか？



現在まで、一般的なトピックは厳密に定義されておらず、誰でも彼に興味のあるトピックに関するPerlに関連するレポートを申請できます-管理、テスト、データベースの操作など。 たとえば、PostgreSQLでのpl / Perlの使用に関するレポートがあります。



そして、誰が著名なスピーカーになるのでしょうか？



スピーカーの1人はSawyer Xです。 非常に興味深い、衝撃的なストーリーテラー。そのレポートは、感情の嵐、ポジティブなこと、何かをしたいという大きな欲求を引き起こします。 素材を非常に詳細に、活気に満ちた興味深いもので説明すると、必然的に良質で高品質な例で補強されます。 YAPC :: EuropeでSawyer XがAnyEventのような一見有名なものについて話したとしましょう。 その後、Moscow.PM会議の1つでロシア語の翻訳でこのレポートを繰り返しましたが、誰もがこのことの仕組みを知っているわけではなく、誰もがそれを理解しているわけではないことに非常に驚きました。 Sawyer Xレポートの後、会議に出席したすべての人は、AnyEventが何であるか、それがどのように機能するのか、なぜ必要なのか、どこで使用するのかを完全に理解しました。 そして、人々が帰って来て、脚本を書き直し、「はい、今はすべてが私と一緒にかっこいい」と言ったことを本当に願っています;-)。 このレポートの資料は、非常に体系的で詳細かつ理解可能な方法で提示されています。 あなたの好きな映画やテレビ番組でよく出会う俳優のリストをIMDbから引き出す優れた例があります。 それは起こります、あなたはシリーズを見て、どこかでこの俳優を見たと思いますか？ 覚えていない-ここでは便利なスクリプトが役立ちます。



2番目のスピーカーはピーターラビットソン（りばしし）です。 おそらく、1つのYAPCではありません::ヨーロッパはそれなしではできません。 彼は、すべてのORM Perl、DBIx :: Classのメインまたはほぼ唯一の開発者の一人です。 これは、ほとんどすべての人が使用する主要なORMです。少なくとも誰もが確実に知っています。 彼は広く知られていますが、このORM、その新機能、特定の問題を解決する方法についての彼の報告は見過ごされません。 ラビットソンは優れた講演者であり、彼のスピーチを非常に興味深いものにしています。 ほとんどの場合、彼は自分が話していることを直接説明します。「ここにスライドにコードがあります...」、すぐに起動し、動作の仕組み、および外出中のルールを示します。 ところで、ピーターはロシア語をとても上手に話しています。



他に何？



これまでのところ、会議ではレポートのみが計画されています。 間違いなく価値のある賞品のあるPerl Golfがあります。今、私たちはこのタスクについて考えています。 クレイジーなアイデアが1つありますが、今のところはこの秘密を明かしません。 どうにか実装できれば、とてもクールです。 時間が残っている場合、金曜日の夜にミニハッカソンを開催するかもしれませんが、これは問題です。



費用はいくらですか？



YAPCへの参加::ロシアは無料ですが、毎年、特に危機の時代には、この伝統を維持することがますます難しくなっています。 部屋を借りたり、スピーカーを持ってきたり、食べ物を提供したりする必要があります。 これまでのところ成功しました。 昨年、ほとんどすべてがREG.RUによって後援されました。 今年、Mail.Ru Groupがスポンサーを引き継ぎました。 たとえば、YAPC ::ヨーロッパへの参加には約100ユーロの費用がかかりますが、これは一般にかなりの額です。 さらに、今年はスペインでシーズンの真っat中にいる必要があります。 ところで、有償参加では、参加者の数を正確に計算できます。 無料の会議では、アプリケーションの数が実際に到着した数の数倍になることがよくあります。 人々がお金を稼ぐまで、彼らが行くかどうかは常にわからない。 そして、あなたがすでに支払いをしたとき、あなたは確かに行きます。 さらに、人数が正確にわかっている場合、会議の予測と開催が容易になります-少なくとも必要な部屋のサイズを決定します。 今年は100〜150人を期待しています。



また、今年はライブビデオ放送を開催します。 チャンネルを撮影、カット、レイアウトします。 YAPC ::ロシアで話をしたい人は、当社のウェブサイトのフォームに記入できます。会議のスケジュールもそこに表示されます。
