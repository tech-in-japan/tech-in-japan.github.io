Windows Vista用SP2のベータ版は来週リリースされます

ネットワークは、来週10月29日にWindows VistaおよびWindows Server 2008の2番目のフィックスパックのベータ版が利用可能になると報告しています。



マイクロソフトのWindows製品管理担当副社長、マイクナッシュによると、SP2のベータ版には、最初のフィックスパックのリリース後にリリースされたすべての修正が含まれ、新しいタイプのハードウェアのサポートを提供するための変更が含まれます。 さらに、パッケージには次のコンポーネントが含まれます。



-Windows Search 4.0。 検索プロセスの改善とより関連性の高い結果の発行を目的とした、改善されたバージョンの検索。

-Bluetooth 2.1機能パック。 このパッケージは、最新のBluetooth技術仕様をサポートすることを目的としています。

-メディアをBlu-rayディスクに記録する機能。

-異なるタイムゾーンでのファイル同期の修正を目的としたexFATファイルシステムのUTCタイムスタンプのサポート。
