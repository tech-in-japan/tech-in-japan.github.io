アップグレードまたはセカンドウィンドmacbook MB467（Late 2008）

すべての良い一日。 最近、コンピューター（この場合はラップトップ）のすべての所有者が一度に対処しなければならないという問題に遭遇しました。つまり、作業が遅く、デバイスの通常の操作に必要なリソースが不足しています。



MacBook MB467



宛先：

プロセッサー：Intel Core Duo 2.4 GHz

バス：1066 MHz

キャッシュ3 MB

メモリ（RAM）4096 MB

HDD 250 GB



後：

プロセッサー：Intel Core Duo 2.4 GHz

バス：1066 MHz

キャッシュ3 MB

メモリ（RAM）8 GB

SSD 60GB + HDD 250 GB




多くの人はすぐにコンピューターを新しいコンピューターに交換することを考え始めますが、 現時点では金銭的手段がなかったので、古い馬をアップグレードすることにしました。 プロセッサとディスクの温度 、OSといくつかのプログラムの起動速度に関連するテストの前後に2、3のテストを実施しましたが、結果が予想を上回ることがわかったとき、この記事が生まれました。



そこで、この記事では、RAMの交換、 OptiBayポケットを使用したSSDの取り付け、サーマルペーストの交換、およびアップグレード前後の作業のテスト結果についてお話したいと思います。



CD-ROMの代わりにSSDをプライマリドライブとして使用し、OptiBayポケット内のHDD を使用することに関する SeiFeRの記事は、私をこのベンチャー全体に追い込みました。 また、RAMの量を4xから8 GBに拡張することを決定しました（公式ウェブサイトでは、このラップトップモデルは最大4GBをサポートすると書かれていますが、Mac OS 10.6.8はありませんでした）。プロセッサとビデオカードのサーマルペーストを交換します（ご覧のとおり）テストから、ペーストはプロセッサの温度を大幅に下げます）。



有名なiFixitオフィスからの非常に良い情報があるので、交換用のラップトップの分析のすべての段階を詳細に説明しません（ところで、iPhoneとiPad用のアプリケーションがあります）。



1. OptiBayポケットを使用して追加のSSDドライブをインストールします。

2. 冷却システムの取り外し 。

3. ハードドライブの交換 。

4. 熱伝導グリースの交換 。



私は、ラップトップを分解することは難しくなく、すべてが慎重に、そして力なしで行われるとすぐに言いたいです。 米国での購入以来、ラップトップは開かれていないことに注意してください（購入中に4GBのRAMがインストールされました）。



1. RAMを購入するには、適切なタイミングで特定のものを選択する必要があります。 Macbookを開いてKingston 2GBx2がメモリを使用しているのを見て驚いたので、ためらうことなく同様の4GBx2メモリ（ DataSheet ）を選択しました。





2. Optic Bayについては、私の良き友人が助けてくれました。すべてのマウントを備えた非常に高品質のアセンブリであることがわかりました（中国には注意してください-マウントなしです）。



3.サーマルグリースの使用に関するインターネットでの多くのテストを読んだ後、私の選択はArctic Cooling MX-4に当てはまりました。非常に良好な熱伝達があります（8.5）。



4. SSDから、OCZ AGT3-25SAT3-60Gを選択しました。 私は価格/品質の組み合わせが本当に好きでした。 優れた速度と、将来のすべてのSATA v3へのプラス。 60 GB、多くの人はそれが十分ではないと思うでしょうが、人々の経験によると、OSと必要なプログラムにとっては十分であることがわかります。



したがって、作業を開始するには何をしなければなりませんか：
