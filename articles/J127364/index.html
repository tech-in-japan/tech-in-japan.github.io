Oracle Data Cartridge InterfaceとDomino Java APIを使用したOracle DatabaseでのLotus Dominoデータの紹介

私たちはそれぞれ、異なるシステムを統合する必要に直面していました。 Oracle DatabaseのSQL select文を使用して、Lotus Dominoデータベースのデータを表示する機能を提供する方法について説明します。 おそらく、このトピックは多くの人にとって興味深いものではないでしょう。 しかし、私の意見では、これら2つの商用製品は、それらが提供する機会について書くに値します。



このトピックを理解したい人は、次の2つのドキュメントを事前に理解しておくことができます。

それらに基づいて、次の例を作成しました。その目的は、OracleクエリでDominoアドレス帳のデータを使用する基本的な可能性を示すことです。 両方のソフトウェア製品がLinuxに標準的な方法でインストールされ、ファイルとアクション自体がディレクトリ/ home / user / java / oracle_dominoに配置され実行されることを前提としています。 必要な追加コメントは、両方のドキュメントに記載されています。
