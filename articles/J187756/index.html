コンテストインテルアプリイノベーションコンテスト2013。$ 20,000を獲得！

昨日、 インテルアプリイノベーションコンテスト2013 、具体的にはロシア語の部分がHabrahabrコミュニティで始まりました。 7月24日から11月20日まで、Windows 8デスクトップアプリケーションのアイデアの作成から実装までを行うことをお勧めします。 賞金プールは相当なものです：100,000ドル+ Intel Developer Zoneのロシアチームからの追加賞品。 戦うべきものがあります！









Windowsデスクトップと Windowsストア



コメントで考えられる質問を予想して、すぐにすべての「i」にドットを付けたいと思います。 はい。このコンテストでは、デスクトップアプリケーションのみを受け入れます。 いいえ、Windows UIの熱烈な反対者ではありません。 実際、新しいWindows 8 OSのリリース後、プログラマー界で2つの神話が生まれました。

  1. デスクトップモードでは、マルチタッチやセンサーなど、Windows 8のすべての新機能のサポートを実装することはできません。
  2. デスクトップアプリケーションをWindowsストアで公開することはできません。したがって、販売で収益を上げることはできません。


私たちは両方の神話を覆したいと思います。 そのため、このコンテストでは、デスクトップアプリケーションで、タッチジェスチャ、センサー、およびWindows UIでは利用できない他の多くの機能と技術のサポートを実装できることを実際に確認することを提供します。 デスクトップストアをWindowsストアに公開することもできます。



新しいインターフェースが必要であるという意見があります。 未来はタッチデバイスに属し、古き良きデスクトップは遅かれ早かれ陳腐化するでしょう。 もちろん、ユーザーはデスクトップアプリケーションの使用を一晩中やめません。すべてのユーザーがWindowsストアのアプリケーションの革新的なインターフェイスに対応できるわけではありません。 ユーザー向けのWindows 8デスクトップアプリケーションは、以前のOSからWindows UIへのかなり論理的な移行です。 デスクトップおよびWindows UIアプリケーションの開発の利点とサポートされるテクノロジーの詳細な比較は、 http：//software.intel.com/en-us/articles/windows-8-store-vs-desktop-app-developmentにあります。



競技の条件について簡単に



コンテストの申し込みは、 CSDN 、 ThinkDigit 、 Code Project 、 HabrahabrのWebサイトで受け付けています。 ロシアのコミュニティのステータスを無私無欲にサポートするために、私たちのハブを通じてアプリケーションを送信することを強くお勧めします:)朗報：ロシアの開発者だけでなく、ウクライナとベラルーシの開発者も合法的に競争に参加できます。 ドイツ、イギリス、またはイスラエルに住んでいても、Habrを介してロシア語でアプリケーションを送信することを妨げるものはありません。



競争自体は2つの段階に分かれています。 最初の段階は準備段階です-アイデアのみを受け入れます。 審査員のすべてのアイデア（この段階ではロシア語を話す人）から、最高の125人が選ばれ、その作者は第2段階に進み、開発用のデバイスを受け取ります。 デバイスを受け取った後、提示されたアイデアに基づいてデモアプリケーションの開発を開始します。 革新的なアプリケーションを収集し、メインの賞品を競う30人のファイナリストを選択します。



100,000ドルまたは主な動機の1つ



賞を獲得するには、最高のデモアプリケーションの30人の作成者が英語の追加ビデオを作成する必要があります。 この段階では審査員団は国際的であり、ロシア語や中国語を理解していない審査員はあなたの素晴らしいアイデアを理解し、評価する必要があるため、これが必要です。



第2段階の結果とビデオクリップを作成する追加タスクに基づいて最大ポイント数を獲得した著者は、 20,000ドルの主賞を受け取ります。 また、タブレット、キャンディーバー、および6つの競合カテゴリのそれぞれに最適なアプリケーションが選択されます。



合計で、 $ 10,000の賞金が授与される別の8人の勝者が得られます。 賞金に加えて、9人の受賞者全員がインテルからさまざまなマーケティングサポートを受け、もちろん国際開発者コミュニティの名誉と尊敬を受けています。



教育をバイパスしないでください-あなたは追加の報酬を受ける権利があります



今年、教育分野へのサポートはインテルの最優先事項の1つになりました。 この春、Intel Education Galaxy Webサイトでのコンペティションの一環として、私たちは専門家や教育者から、日々の仕事に欠けているアプリケーション（ https://edugalaxy.intel.ru/?portalid=24 ）について多くの興味深いアイデアを受け取りました。 。 おそらくそれらのいくつかは、私たちの競争への参加の基礎として取ることができます。 これらのアイデアはすでに市場で需要があるため、ユーザーがいなければ製品は残りません。 さらに、アプリケーションは、Intel Developer ZoneおよびIntel Education Galaxyプロジェクトから追加の賞を受け取る場合があります。 視覚化、デザイナー、エンターテインメントによる学習の3つのカテゴリの最高のアプリケーションの作成者は、それぞれ100,000ルーブルを受け取ります。 またはこの金額の広告パッケージ、およびIntel Education Galaxyプロジェクトのマーケティングサポート。



最速かつ最も独創的なスナックモチベーターまたは20の賞品



アプリケーションのアイデアが既に頭の中で熟している場合は、テキストまたはグラフィック形式で視覚的に整理することを急いでください。 8月25日より前に申請書を提出した最高のアイデアの著者20人に、コミュニティ賞のHDD Seagate BackUp Plusが授与されます。 受賞者を決定する際、コミュニティメンバーからの投票数とコメント数を考慮します。 不正行為の可能性を回避するために、アイデア自体の独創性に間違いなく感謝します。
