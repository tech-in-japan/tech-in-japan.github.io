インストール後のUbuntuのチューニングと最適化

この記事では、GNU / Linuxを最初に自分でインストールしたユーザーごとに生じる疑問に答えようとします。 私自身はこれらのOSを数年しか使用しておらず、私自身も同様の質問をしていたので、これらのシステムを理解するのを少しでも手伝うことができると思います。 一見、すべてが複雑で理解できないように見えますが、そうではないことを納得させようと思います。 私は初心者向けのLinux OSとしてUbuntuを選びましたが、このディストリビューションは本当にユーザーフレンドリーであるため、間違いではなかったと思います。完全なローカリゼーションと、もちろん、アプリケーションセンター。ソフトウェアをインストールするために、マウスボタンを1つ押すだけです。 しかし、システムをインストールした後、私の意見では不要な多くのプログラムと設定があります。これらは変更し、より便利で機能的なもので補完するのが良いでしょう。 これがこの記事で行うことです。 実行するアクションと実行しないアクションは、各ユーザーの個人的な選択です。おそらく、以下にリストされているコマンドをいくつか必要とするか、記事に記載されているとおりにUbuntu GNU / Linuxを正確に構成します。

各作業環境（Gnome、KDE、Unity）の特定の設定については説明しませんが、作業環境に依存せず、すべてのユーザーに役立つ設定のみを掲載します。



以下にリストされているすべてのコマンドを端末にコピーする必要があります。 ターミナルは、 CTRL + ALT + Tで呼び出すことができます。



更新システム



まず、リポジトリのリストを更新し、プログラム、ドライバー、カーネル、その他すべての更新されたバージョンのインデックスを検索します。

sudo apt-get update







実際、利用可能なすべての更新プログラムをインストールします。

sudo apt-get upgrade







フラッシュ、Java、オーディオ、ビデオコーデックをインストールします。

apt-get install ubuntu-restricted-extras







ここで、ウィンドウコントロールボタン（最小化、最大化、閉じる）を通常の位置（MACのように左側ではなく右側）に移動します。

gconftool-2 --set "/apps/metacity/general/button_layout" --type string "menu:minimize,maximize,close"







余分なソフトウェアを削除します。



Pitiviビデオエディターをアンインストールします。

sudo apt-get remove pitivi







Empathyインスタントメッセージングクライアントをアンインストールします。

sudo apt-get remove empathy empathy-common nautilus-sendto-empathy







Gwibberマイクロブログクライアントを削除します。

sudo apt-get remove gwibber gwibber-service







UbuntuONEを使用する必要がない場合（リモートサーバーにデータを保存してどこからでもアクセスする場合）、このサービスを無効にして削除します。

sudo apt-get remove python-ubuntuone-client ubuntuone-client python-ubuntuone-storageprotocol ubuntuone-client-gnome rhythmbox-ubuntuone-music-store python-ubuntuone







デスクトップへのリモートアクセスを誰かに提供する必要がない場合は、以下を無効にします。

sudo apt-get remove vino







Windowsマシンへの接続時に問題が発生するため、標準のリモートデスクトップクライアントを削除します。

sudo apt-get remove vinagre tsclient rdesktop







そして、より機能的なRDP Remminaクライアントを配置します。これにより、RDPを介してWindowsに接続できるほか、SSHを介してあらゆるものに接続できます。

sudo apt-get install remmina







システムをクリーニングするためのアプリケーションを削除することを提案します。システムは開発中であり、大きな利点はありません。

sudo apt-get remove computer-janitor-gtk







拡大鏡など、支援技術に関連するすべてのものをシステムから削除します。

sudo apt-get remove gnome-accessibility-themes gnome-dictionary gnome-mag at-spi libgail-gnome- module libatspi1.0-0 libgail-common libatk1.0-data libgnome-mag2 espeak-data espeak





module libatspi1.0-0 libgail-common libatk1.0-data libgnome-mag2 espeak-data espeak







コンピューターまたはラップトップにBluetoothアダプターがなく、使用する予定がない場合、これに関連するすべてのサービスとアプリケーションを削除します。

sudo apt-get remove bluez-cups bluez bluez-alsa bluez-gstreamer pulseaudio-module-bluetooth







依存関係を構築せずに単一の.debパッケージをインストールするために、gdebiアプリケーションをインストールします。

sudo apt-get install gdebi







この手順の後、プログラムによるgdebi .debファイルのオープンのコンプライアンスを復元する必要があることに注意してください。 任意の.debファイルをポイントし、右クリックしてメニューの[プロパティ]を選択し、[プログラムで開く]タブでGdebiパッケージインストーラーを選択します。



デフォルトのビデオプレーヤーと音楽プレーヤーを削除してみましょう。これらは使用するのにあまり便利ではないので、一般的には次のことを決定する必要があります。

sudo apt-get remove rhythmbox totem totem-common







引き換えに、シンプルで便利なデッドビーフプレーヤーを置きます

sudo add-apt-repository ppa:alexey-smirnov/deadbeef

sudo apt-get update

sudo apt-get install deadbeef







また、映画はVLCで見るのに非常に便利です。

sudo apt-get install vlc mozilla-plugin-vlc







目に優しいテーマとアイコンを配置します。ここでも選択できます。この方法で、任意のトピックを配置できます。

sudo add-apt-repository ppa:tiheum/equinox

sudo apt-get update && sudo apt-get install faenza-icon-theme equinox-theme equinox-ubuntu-theme







メニューの追加アイコンの表示をオンにします。

gconftool-2 --type boolean --set /desktop/gnome/interface/menus_have_icons true







ボタンの追加アイコンの表示をオンにします。

gconftool-2 --type boolean --set /desktop/gnome/interface/buttons_have_icons true







Compizデスクトップ環境管理ユーティリティを配置します。

sudo apt-get install compizconfig-settings-manager







これで設定の最初の部分が完了し、出力で得られたものに感心し、システムを再起動します。



ICQクライアントの選択





Ubuntuに切り替える際のもう1つの重要なポイントは、OSのICQメッセンジャーの選択です。 だから、どのクライアントを選択するのですか？ Windowsでは、RnQを使用することに慣れています（一般の人々-ラット）。 残念ながら、Linuxにはまだそのようなクライアントが存在しないため、類似物を探す必要がありました。 また、UbuntuでWineからネズミを実行する可能性も除外しませんでしたが、Linuxに切り替えているため、Linuxの何かを使用することにしました。 標準パッケージに含まれているPidginは私には合わなかったので、Qutimを選択することに決めました。 理由：そのインターフェイスは、以前のバージョンのミランダとQIPを非常によく連想させるので、便利で、ロードされていないインターフェイス、その他のアメニティがあります。 2つの方法を検討してください。



インストールプロセスに進みます。 通常どおり、ターミナルを実行します（Ctrl + Alt + T）。



1）システムにプログラムのソースを追加し、ターミナルでコマンドを実行します。

sudo wget linux.intonet.ru/ubuntu/qutim/hardy.list -O /etc/apt/sources.list.d/qutim.list







次に、このためにパッケージデータベースを更新する必要があります。コマンドを実行する必要があります。

sudo apt-get update







更新後、qutimがデータベースに表示されます。 それをインストールしてください：

sudo apt-get install qutim







2）この方法では、複数のコマンドを別々のステップに分割せずに、一度に複数のコマンドの実行を組み合わせる方法を検討します。 1つのコマンドでリポジトリを追加し、パッケージのリストを更新してQutimをインストールします。 この方法はademaro habrayuserによって提案されました。

ターミナルで実行します。

sudo add-apt-repository ppa:qutim/qutim && apt-get update && apt-get install qutim







絵文字とテーマを追加できます：

sudo apt-get install qutim-emoticons

sudo apt-get install qutim-themes







以上でインストールは完了です。 次に、設定を操作して、クライアントをQIP、Miranda、またはRnQのように見せることができます。 クライアントは本当に使いやすいので、健康のために使用します。

スカイプをインストールする



いつものように、ターミナルからSkype-リポジトリを追加してインストールします。 それでは、始めましょう：



ターミナルを開いてコマンドを実行します（geditの代わりに、nanoなど、選択した他のテキストエディターを使用できます）。



sudo gedit /etc/apt/sources.list







表示されるテキストエディタで、次の行を追加します。

deb download.skype.com/linux/repos/debian stable non-free



ターミナルに書き込みます：

sudo apt-get update







情報が更新されるまで待ちます。 次に、ターミナルでコマンドを実行します。

sudo apt-get install skype







インストール後、Skypeはアプリケーション>インターネットにあります。



Operaをインストールする





デフォルトでは、FirefoxはUbuntuにインターネットブラウザーとしてインストールされます。 個人的には、軽量ですが、好きではありませんが、デフォルトでは、Operaで利用できる機能はありません。 もちろん、さまざまなプラグインで完成させることで機能を拡張できますが、これらの自己プラグインからは非常に重くなり、Operaの速度に著しく劣り始めます。 そのため、Firefoxを削除してOperaをステージングすることをお勧めします。 始めましょう。



まず、source.listにオペレータリポジトリを追加します。

deb deb.opera.com/opera stable non-free







次に、更新用のGPGキーを追加します。

wget -O - deb.opera.com/archive.key | sudo apt-key add -sudo apt-get install debian-archive-keyring







実際、Operaをインストールします。

sudo apt-get update

sudo apt-get install opera







次に、メインメニュー/インターネット/ Operaの途中でOperaの可用性を確認し、起動します。 ライセンス契約に同意し、私たちの健康を利用します。



はい、Firefoxは不要になりました。

sudo apt-get remove firefox







システムを完成させましょう





新しく接続されたデバイスに関する通知を提供します：

wget download.learnfree.eu/repository/skss/udev-notify-0.1.2-all.deb ;

dpkg -i udev-notify* ;

apt-get install -f ;







Gimpグラフィックエディターにブラシとフィルターのセットを追加します。

sudo add-apt-repository ppa:shnatsel/gimp-paint-studio ;

sudo apt-get update ;

sudo apt-get install gimp-paint-studio ;







ソースシートに追加...

ppa.launchpad.net/mars-core/ppa/ubuntu natty main

deb-src ppa.launchpad.net/mars-core/ppa/ubuntu natty main







...そしておもちゃを置きます：

apt-get update; apt-get install marsshooter -y







OSを少し最適化します。





＃で始まるコマンドは、スーパーユーザー権限を必要とすることを意味します。これは、sudo記事のように各コマンドの前に追加するか、ターミナルでsudo suを実行します。これにより、後続のすべてのコマンドがスーパーユーザーに代わって実行されることになります。ハチのアノン。



物理RAMがいっぱいになったときにのみスワップ（スワップパーティション）がアクティブになるように設定します。

# gedit /etc/sysctl.conf







ファイルの最後に行を追加します：vm.swappiness = 0

したがって、使用可能なRAM全体を最も効果的に使用します。



システムに2つ以上のコアプロセッサがインストールされている場合、以下を実行することをお勧めします。

# gedit /etc/init.d/rc







CONCURRENCY = noneを見つけて、次のものに置き換えます。

CONCURRENCY =シェル

これにより、システムのパフォーマンスが大幅に向上します。



スケジュールを少しスピードアップして、優先度20にしましょう。

# gedit /etc/rc.local







これをEXITコマンドの前のファイルの最後に挿入します

sudo renice -n -20 $（pidof X）
