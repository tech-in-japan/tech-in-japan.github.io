JBoss ToolsおよびJBoss Developer Studioの新しいバージョン





今週、JBossはJBDS（3.0.0GA）およびJBoss Tools（3.1.0GA）プラグインの次のリリースをリリースします。 製品の新しいバージョンには、多くのバグ修正、機能の変更、サポートされているすべてのテクノロジーを使用した作業の簡素化を目的とした、より使いやすいユーザーインターフェイスの作成が含まれています。



JBoss ToolsおよびJBoss Developer Studioは、JBoss AS、JBoss Seam、JBoss jBPM、Hibernate、JBoss Drools、JBoss ESB、RichFaces、JBoss PortalなどのJBossテクノロジーと連携するように設計されています。



どちらの製品もEclipse IDEとEclipse WTPプロジェクトに基づいています。 これにより、Sun、Weblogicなど、J2EE技術スタックの他の実装者が提供するソリューションに基づいてJ2EEアプリケーションを開発できます。



JBoss Toolsは、無料でダウンロードできるEclipseプラグインのセットです。 また、JBoss Developer Studioは、J2EEアプリケーションの開発と起動に必要なソフトウェアの完全なセットです。



どちらの製品にも、完全な技術サポートと詳細な更新ドキュメントがあります。

Visual Page Editor -Visual Page EditorがRed Hat Enterprise Linux 5 x64で実行されるようになり、VPE設定を介して不明なタグのテンプレートを追加する機能、新しいJSF 2.0タグのサポートなどが追加されました。

JBoss Developer Studioリリース3.0.0GA変更の概要


コンテキストおよび依存性注入（CDI） -コンテキストおよび依存性注入Project FacetがJBoss Developer Studioに追加され、プロジェクトメニューに新しいオプション（CDIサポートの追加およびCDIサポートの削除）などが含まれます。

ESB -ESB Content Based Routerを変更し、ESB Configuration Editorに新しい機能を追加し、JBoss SOA Platform 5.0などに含まれるESB 4.7を公式にサポートしました。

Hibernate-リファクタリングのサポートがHibernate Code Generation Configurationsに追加され、新しいHibernate XML Mappingファイルの機能が更新され、Hibernate Diagram Viewer、Mapping Diagram Viewerなどが更新されました。

JavaServer Faces（JSF） -PageContextFactoryのパフォーマンスが改善され、JSF 2.0構造コンポーネントのサポートが行われ、Eclipse OpenOn関数の操作などが修正されました。

RichFaces - RichFacesライブラリがバージョン3.3.2.SR1に更新されました。 その他

Seam -JBoss Enterprise Application Platform向けにSeam 2.2に更新された環境設定、Seam BookingおよびDVDStoreサンプルを使用する場合のSeamプロジェクトのアセンブリと検証を修正

Smooks -Smooks Readerページは入力ページに置き換えられ、機能が変更され、Smooksプロジェクトのテストが簡素化されました。

Struts-プロジェクトメニューにStrutsの追加と削除のサポートオプションを追加しました。

Visual Page Editor- 「メモリリーク」の原因となるバグが修正され、JSF 2.0属性のサポートが追加され、アイコンデザインが変更され、中央の要素の直後と次の要素を選択する新しいオプションが追加されました。

XHTMLエディター -タグと属性のツールチップが追加されました。

XML構造化エディター -web.xmlのサーブレットの選択を簡素化し、サーブレットマッピングエディターを改善しました。

一般的な問題 -SSHデプロイメントの追加、Imoduleオブジェクトを使用するための操作をエクスポートするための新しいウィザードの追加、JBoss Tools Paletteは標準のEclipse GEFビューなどに統合されています
