Hackathon SkolkovoとIBMのモバイルアプリケーション

IBM、Skolkovo Foundation、およびApps4All開発者コミュニティは、ハッカソンに参加して、IBM Bluemixクラウドプラットフォームのバックエンドでモバイルアプリケーションを開発するよう招待しています。



開発は、次の4つの主要なカテゴリで受け入れられます。

●企業のモバイルアプリケーション

●ウェアラブルエレクトロニクスのアプリケーション

●拡張現実アプリケーション

●IoTソリューション



24時間以内に、チームは既存のアプリケーションを適応させるか、独自のアプリケーションを考案、設計、開発することができ、専門家審査員によって評価されます。



選択したカテゴリに関係なく、B2E（企業間）またはB2B（企業間）セグメントのビジネスユーザーのタスクに対応するアプリケーションが期待されます。



企業ユーザー向けモバイルアプリケーションの開発プロジェクト（B2E）へのベンチャーファンドや個人投資家の関心が高まっていることからもわかるように、企業モビリティ市場は最も有望なものの1つと見なされています。

また、近年のIT産業のこれらの分野の急速な発展により、ウェアラブルエレクトロニクスおよび追加の現実の分野でのアプリケーションがB2Bセグメントでますます一般的になりつつあることも注目に値します。 さらに、ビジネスユーザー向けのアプリケーションとともに、企業のモビリティ市場におけるこれらの分野の進歩性が注目されています。



過去数年にわたり、私たちは、単なる楽しみだけでなく、日常生活を支援するエンドユーザー向けの多くの便利なアプリケーションを見てきました。 問題は、モバイルアプリケーションの企業の世界は、同じユーザーエクスペリエンスの品質を誇ることができないことです。これは、特定の従業員とのやり取りを設計する一方で、企業ソフトウェアが主に正確で関連する情報を収集して企業の管理に提供することを目的としていたためですアプリケーションには二次的な注意が払われました。 このタスクを別の観点から見て、ユーザーエクスペリエンスを普通の会社の従業員の最前線に置くことをお勧めします。



私たちは、小売、電子商取引、金融と銀行、農業、通信、マーケティングとマーチャンダイジング、人事、調達などの分野のビジネスユーザー向けの使いやすいモバイルアプリケーションを作成するお手伝いをします。そしてサプライチェーン。



ハッカソンは無限のコーディングであるだけでなく、専門家、メンター、同僚、マスタークラス、食べ物や飲み物とのコミュニケーションでもあることを知っています。 したがって、イベント中に、IBM Bluemixプラットフォームの豊富な機能をテストするだけでなく、プロジェクトを専門家チームに提示し、メンターと協力して、プロジェクトの収益化と市場でのプロモーションの選択肢について議論する機会があります。



したがって、IBMと協力するための可能なオプションの1つは、アプリケーションをIBM Cloudに配置することです。これにより、IBM Cloud Marketplaceにアプリケーションを配置できるようになります。 協力のもう1つのオプションは、アプリケーションをBluemixサービスとしてホストすることです。これにより、他の開発者がアプリケーションをアプリケーションの一部として使用できるようになります。



このイベントは、2015年3月28〜29日にSkolkovoイノベーションセンターのハイパーキューブの建物で開催されます。





IBM Bluemixは、幅広い技術（Node.js、Liberty、Ruby、Python、PHPなど）および多数のサービスを使用して、さまざまなタイプのアプリケーションを迅速に開発するために設計されたオープン標準ベースのクラウドプラットフォーム（PaaS）です。



このプラットフォームには、MongoDb、Redis、Mysql、Postgresql、CouchDBなどの一般的なテクノロジーに加えて、IBM Watson、MobileFirst、IBM DevOps、Cloudant、IBM Big Data AnalyticsなどのユニークなIBMサービスが含まれています。



IBM Watsonは、自然言語（現在は英語のみ）を理解できるだけでなく、強力な予測分析と視覚分析を使用して、大量のビッグデータを処理できます。



MobileFirstは、モバイルアプリケーションの開発者にサービスを提供し、クロスプラットフォームアプリケーションの開発を加速し、自動テストとエンタープライズまたはクラウド内の外部データソースとの継続的な統合により品質を向上させます。



IBM DevOpsサービスは、開発者にソフトウェアアプリケーション用のラピッドプロトタイピングツールを提供し、短時間で連続したソフトウェア製品を定期的に作成および生産します。



Cloudant-NoSQLのサービスとしての高性能クラウドデータベース。



IBM Big Data Analyticsサービスは、ツイートや地理空間データからのデータなど、アプリケーション内の大量のデータに対して強力な分析および視覚化ツールを使用する機能を提供します。



これらのサービスやその他のBluemixサービスについては、ハッカソンの前夜に開催される会議で詳しく説明します。 会議の日付、時刻、場所は追加でお知らせします。
