CentOS上のOpenVPNサーバー

OpenOSサーバーをCentOSにインストールして操作するのは複雑すぎるように思えたので、これらのプロセスを可能な限り自動化しようとしました。 サーバーは自動的に構成され、インストール後すぐに動作を開始します。



ユーザーを作成すると、構成ファイルが生成されるか、そのためのWindowsクライアントが生成されます。





ユーザーを管理するコマンド：

openvpnuser -h



すべてのサーバー設定は / etc / openvpn /にあり ます。

メイン設定ファイル： /etc/openvpn/server.conf

クライアント構成ファイルまたはWindowsクライアント自体は / etc / openvpn / clients /にあります。



設置



1. OpenVPNをインストールするためのパッケージをダウンロードします。

wget --no-check-certificate \

sites.google.com/site/smkuzmin/home/openvpn/openvpn_public_version.tgz \

-O openvpn_public_version.tgz



2. / progs / packets /にそれらを解凍します ：

mkdir / progs / packets

tar xf openvpn_public_version.tgz -C / progs / packets /



3. /progs/packets/openvpn/config/openvpnuser.pwlに パスワードリストを書き込みます

-新しいユーザーを作成するときに使用されます。



4. / progs / packets / openvpn / config / openvpnuser.cfgで次を 選択します。

CERT_PASSWORD- インストール中に作成されたユーザー証明書のパスワード。

WINDOWS_CLIENTS_TYPE = 'ZIP | EXE'- 生成されたWindowsクライアントのタイプ

WINDOWS_CLIENTS_TO_DIR = 'YES | NO' - Windowsクライアント をディレクトリ に保存するかどうか

WINDOWS_CLIENTS_TO_FAS = 'YES | NO' - Windowsクライアント をFAS に保存するかどうか

単一の保存方法を選択しない場合は、

クライアント構成ファイル。



5. / progs / packets / openvpn /ディレクトリに移動します。

cd / progs / packets / openvpn /



6. ローカルソースから OpenVPNサーバーをインストールします。

bash -c "。./install"

またはインターネットから：

bash -c "。./install -net"
