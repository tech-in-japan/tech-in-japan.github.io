電信履歴の概要

夏の学校では、彼らはいつも重い文学のリストを尋ねました-通常、私は半分以下に足りなかったので、私は簡単に全部を読みました。 5ページの「戦争と平和」-もっと良いことは...似たようなジャンルの電信の歴史についてお話ししますが、一般的な意味は明確でなければなりません。







「Telegraph」という言葉は、2つの古代ギリシャ語-テレ（far）とグラフォ（私が書いている）に由来しています。 現代の意味では、有線、無線、または他の通信チャネルを介して信号を送信する手段にすぎません...最初の電信はワイヤレスでしたが、長距離で情報をやり取りして送信することを学ぶずっと前から、人々は跳ね返り、ウィンクし、き火を作り、ドラムを叩く-これはすべて電信と考えることもできます。



信じられないかもしれませんが、オランダでは、風車の助けを借りてメッセージ（プリミティブ）を送信しました。風車の多くは、特定の位置で翼を止めました。 おそらく、これはかつて（1792年に）クロードシャフが最初の（非原始的）電信を作成するきっかけとなったのかもしれません。 本発明は「ヘリグラフ」（光学式電信機）と呼ばれていました。名前が示すように、このデバイスは太陽光による、またはミラーシステムでの反射により情報を送信することを可能にしました。





都市間には、お互いから直接見えるように、セマフォの巨大な連結翼が設置された特別な塔が建てられました-電信オペレーターはメッセージを受け取り、すぐにそれをさらに送信し、レバーで翼を動かしました。 インスタレーション自体に加えて、Claudeは独自のキャラクター言語を考案しました。これにより、毎分最大2ワードの速度でメッセージを送信できました。 ちなみに、19世紀にサンクトペテルブルクとワルシャワの間に最も長い路線（1200 km）が建設されました。信号は端から端まで15分で通過しました。

刑務所にいる間、多くの囚人は連絡を取り合う機会があります-高度な技術もそこに到達しました。 以前は、メッセージはしばしばタップによって送信されていました。文字テーブル（6列と6行）を使用して、最初に列がタップされ、次に目的の文字の行がタップされました。 そして、モールス信号から「タップ」への「ダッシュ」は「ポイント」よりもはるかに難しいです;）
電気の電信が可能になったのは、人々が電気の性質をより密接に、つまり18世紀頃に研究し始めたときだけです。 電気電信の最初の記事は、1753年に科学雑誌のページに掲載され、特定の「CM」によって作成されました。プロジェクトの著者は、ポイントAとBを接続する多数の絶縁電線を通して電荷を送ることを提案しました。電線の数はアルファベットの文字数に対応する必要があります：「 ボールワイヤの端で、文字の付いたライトボディが帯電し、引き付けられます。」 その後、「CM」の下にスコットランドの科学者チャールズ・モリソンが隠れていたことがわかりました。チャールズ・モリソンは、残念ながら自分のデバイスの正しい動作を確立できませんでした。 しかし、彼は気高く行動しました。彼は他の科学者を彼自身のアイデアで扱い、彼らにアイデアを与えました。







1人目はジュネーブの物理学者Georg Lesageで、彼は1774年に最初の実用的な静電電信機を建設しました（彼は1782年に粘土パイプに電信線を地下に敷設することも提案しました）。 互いに分離された24（または25）の同じすべての配線には、それぞれ独自のアルファベット文字があります。 ワイヤの端は「電気振り子」に接続されます-電気の電荷を送信します（その後、硬い黒檀の棒をこすります）。別のステーションの対応する電気振り子を平衡から外すことができます。 最速のオプションではありません（小さなフレーズの転送には2〜3時間かかります）が、少なくともうまくいきました。 13年後、物理学者のルサージュは、必要な配線の数を1本に減らした物理学者のロモンによって改善されました。



電気電信は急速に発展し始めましたが、実際に使用されたのは静電気ではない場合にのみ素晴らしい結果が得られましたが、ガルバニック電流-この方向の思考の糧は、アレッサンドロジュゼッペアントニオアナスタシオジェロラモウンベルトヴォルタによって最初に投げられました（1800年）。 1802年の磁気針に対する最初の電流の偏向効果は、イタリアの科学者ロマネージによって気づかれ、1809年にミュンヘンの学者Semmeringによって電流の化学的効果に基づいて最初の電信が発明されました。







その後、ロシアの科学者が電信の作成プロセス、すなわちPavel Lvovich Schillingに参加することを決めました-1832年に彼は最初の電磁電信（および後に-仕事のための元のコード）の作成者になりました。 彼の努力の成果のデザインは次のとおりです。絹の糸に吊るされた5つの磁気矢印が「乗算器」（多数の巻き線を持つコイル）の中を移動しました。 電流の方向に応じて、磁気矢印はある方向または別の方向に進み、小さな段ボールディスクが矢印で回転しました。 2つの現在の方向と元のコード（6つのディスク乗数偏向の組み合わせで構成される）を使用して、アルファベットのすべての文字と偶数を送信することができました。







シリングはクロンシュタットとサンクトペテルブルクの間に電信線を作るよう招待されましたが、1837年に彼は亡くなり、プロジェクトは凍結しました。 ほぼ20年後、別の科学者であるBoris Semenovich Jacobiがそれを再開し、とりわけ、受信した信号を記録する方法について考え、電信草案プロジェクトに取り組み始めました。 タスクは完了しました-電磁石のアンカーに取り付けられた鉛筆が条件付きアイコンを記録しました。



また、彼ら自身の電磁電信（およびそれらの「言語」でさえ）は、カール・ガウスとウィルヘルム・ウェーバー（ドイツ、1833年）およびクックとホイートストーン（イギリス、1837年）によって発明されました。 そして、私はすでに彼について詳細に話したが 、サミュエル・モールスについてはほとんど忘れていた 。 一般的に、私たちは最終的に長距離にわたって電磁信号を送信する方法を学びました。 それは始まった-最初の単純なメッセージ、次に特派員ネットワークが電信で多くの新聞のニュースを送信し始め、それから電信会社全体が現れた。



問題は、大陸間での情報の転送でした。大西洋を横断して3,000 km（ヨーロッパからアメリカ）を超えるワイヤを延長する方法は？ 驚くべきことに、それがまさに彼らがやろうと決めたものです。 イニシエーターは、アトランティックテレグラフカンパニーの創立者の1人であるサイラスウェストフィールドでした。 その結果、重量が3,000トン（530万キロメートルの銅線で構成される）のケーブルの「もつれ」が発生し、1858年8月5日までに、大西洋の底に沿ってイギリスとアメリカの軍艦アガメムノンとナイアガラで最大の傷ができました。 。 しかし、その後、ケーブルが壊れました-初めてではなく、修理されました。







モールス電信の不便な点は、コードを解読できるのは専門家だけでしたが、普通の人には完全に理解できないことでした。 したがって、その後の数年間、多くの発明者は、電信コードだけでなく、メッセージテキスト自体を登録する装置の作成に取り組みました。 その中で最も有名なのはタイプライターのze瀬でした。







トーマス・エジソンは、電信オペレーターの仕事を部分的に機械化する（簡単にする）ことを決めました。







テープはレパーフォレーターで作成されました-電信送信機から来る電信コードの記号に従って紙テープに穴を開ける装置です。



レパートレータは、中継電信ステーションで電報を受信し、送信機を使用して自動的に送信しました。これにより、時間のかかる中継電報の手動処理（フォームに文字が印刷されたテープステッカーと、その後のキーボードからのすべての文字の手動転送）がなくなりました。 reperfotransmittersもありました-reperforatorと送信機の機能を同時に実行する電報を受信および送信するためのデバイス。



1843年に、ファックスが登場しました（電話の前に登場することを知っている人はほとんどいません）-スコットランドの時計メーカーであるAlexander Baneがそれらを発明しました。 彼のデバイス（彼自身がBaneテレグラフと呼んでいた）は、長距離にわたってテキストだけでなく画像（不快な品質ではありますが）のコピーを送信することができました。 1855年、彼の発明はジョバンニ・カセッリによって改善され、画像伝送の品質が最終決定されました。







確かに、このプロセスは非常に手間がかかります。自分で判断してください。元の画像を特別な鉛箔に転写し、振り子に取り付けられた特別なペンで「スキャン」する必要がありました。 画像の暗い部分と明るい部分は、電気インパルスの形で送信され、別の振り子によって受信デバイス上で再現されました。 このデバイスはパンテレグラフと呼ばれ、後に世界中で非常に人気がありました（ロシアを含む）。







1872年、フランスの発明家ジャンモーリスエミールボドーは、繰り返し動作する電信装置を設計しました。彼は、1本のワイヤーで1方向に2つ以上のメッセージを送信することができました。 ボードー装置とその原理によって作成されたものは、スタートストップと呼ばれていました。



しかし、デバイス自体に加えて、発明者は非常に成功した電信コード（Baudotコード）を考案しました。これはその後非常に人気を博し、International Telegraph Code No. 1（ITA1）とい​​う名前が付けられました。 スタート/ストップテレグラフ装置の設計をさらに変更すると、テレプリンター（テレタイプ）が作成され、情報転送速度の単位（ボー）は科学者にちなんで命名されました。







1930年、電話式ディスクダイヤラー（テレタイプ）を備えたスタートストップテレグラフが登場しました。 このようなデバイスは、とりわけ、電信ネットワークの加入者を擬人化し、迅速な接続を可能にしました。 将来、このようなデバイスは「telex」（「telegraph」と「exchange」という言葉から）と呼ばれるようになりました。



私たちの時代、多くの国の電信は、道徳的に時代遅れのコミュニケーション手段として放棄されましたが、ロシアではまだ使用されています。 一方、同じ信号機もある程度電信と見なすことができ、ほぼすべての交差点ですでに使用されています。 だから、アカウントから老人を片付ける;）



電信の歴史における1753年から1839年の期間には、約50の異なるシステムがあります-それらのいくつかは紙に残っていましたが、現代の電信の基礎となったものもありました。 時間が経つにつれて、技術とデバイスの外観は変わりましたが、動作原理は同じままでした。



今何？ 安価なSMSメッセージは徐々に消えつつあります。iMessage/ WhatsApp / Viber / Telegramなどのあらゆる種類の無料ソリューションや、あらゆる種類のICQ-Skypeに置き換えられています。 「 22:22-make a wish 」というメッセージを書いて、人（おそらく地球の反対側にいる人）が間に合うようにすることをお勧めします。 しかし、あなたはもはや小さくはなく、自分ですべてを理解しています...同様の期間の後、将来の情報伝達で何が起こるかを予測しようとする方が良いですか？
