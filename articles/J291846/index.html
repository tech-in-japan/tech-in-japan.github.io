サムスンは、ロシア向けのOSとプリインストールされたロシア語ソフトウェアを搭載したスマートフォンをリリースします

SamsungとNew Cloud Technologiesはパートナーになります。 Tizenオペレーティングシステムを実行している韓国のスマートフォンには、MyOfficeオフィススイートがプレインストールされます。



Tizenは、 Linux OSの3つのバージョン、MeeGo、LiMo、およびBadaに基づいています。 ロシアのオフィスプログラムのパッケージには、テキスト、表、プレゼンテーションの編集者、メールサービスが含まれています。



両方の開発は、現在、技術および輸出管理に関する連邦サービスの要件への準拠が認定されています。



パートナーは、ロシアのソフトウェアを搭載したスマートフォンが、政府機関やセキュリティ要件の高い企業から需要があると考えています。 毎年、20,000〜50,000の認定モバイルデバイスが必要です。 この要求に応えるため、New Cloud Technologies社は2016年にロシアおよびその他のCIS諸国で約1億ルーブルを稼ぐことができると、ロシア企業のゼネラルディレクターであるDmitry Komissarov氏は予測しています。



スマートフォンの価格には、プリインストールされたロシア製品の料金が含まれます。 デバイスの各販売から、ソフトウェア開発者は控除を受け取ります。 Officeサービスは曇ります。 さらに、ユーザーは小さなデータウェアハウスを受け取ります。 その増加のためにお金を払う必要があります。



スマートフォンのリリース日はまだ明らかにされていません。



2015年6月、 MyOfficeプロジェクト は 、通信省およびマスメディアからの資金提供の申請者の中で、「カスタムオフィスソフトウェア」のカテゴリで3位になりました。 Tizenベースのロシアのモバイルオペレーティングシステムプロジェクトは、モバイルオペレーティングシステムカテゴリで2位になりました。



Infowatch Natalia Kasperskyも同様のスマートフォンを開発しています。 InfowatchのエグゼクティブディレクターであるVsevolod Ivanov氏はVedomostiに、情報漏えいを検出して防止するプログラムと統合する予定だと語った 。 リリース後3年以内に、同社はロシアのスマートフォン市場の企業セグメントの少なくとも10％を占める予定です。 Ivanovによると、そのボリュームは100万デバイスに達します。
