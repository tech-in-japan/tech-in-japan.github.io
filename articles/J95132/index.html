Chromeウェブストアで私たちを待っているもの



以前に、Googleによる独自のWebストアの立ち上げを既に発表しましたが、今ではChromiumのアプリケーション開発ブランチで前例のない活動が観察されています。 数百人の開発者がWebアプリケーションの準備を急いでいます。これは、待望のChrome OSのリリースに合わせて開催される可能性があります。



私たちは何とか舞台裏を見て、Chrome Web Storeのいくつかのアプリケーションに触れて、今後数か月で使用できるいくつかの興味深い開発について説明しました。 ただし、すべてのアプリケーションはまだ初期段階にあり、開発者がコードをHTML5標準に適合させると、大幅な改善が期待できます。



以下のアプリケーションを実行するには、Chrome 6の最新の開発バージョンをダウンロードして使用する必要があります。記事の最後にある手順。



IMサポート







上の画像は、ライブWebアプリケーションの生きた証です。 これにより、ウェブマスターは訪問者とリアルタイムで話し合い、発生した問題を解決できます。 Lynxusチームが開発したアプリケーション。 それは非常に深刻で強力なソフトウェアパッケージのように見えます。



ダウンロード： imsupporting.com/chrome/IMsupporting.crx



ピクセルナイト







「これは、ヒーローの目標が生き残ることである古典的なプラットフォーマーです。 同時に、王国を守るためにモンスターの大群を刈ることを忘れないでください。 これは、サーバーとの通信を必要としないローカルアプリケーションです。つまり、実績とゲームスコアはローカルに記録されます。 このおもちゃは、Jens Winterstein（Jens Winterstein）によって開発されました。



ダウンロード： codinsoft.de/pknight_app.crx



ソリティア







ロバートシュルツ（ロバートシュルツ）のアプリケーションはすべておなじみのサナダムシ。 これはすばらしい小さなゲームですが、通常のAdSense広告を追加するだけでアプリケーションを収益化する機能を示しています。 そして再び、HTML5とJavaScriptの機能を示す作業の結果。 ほんの数か月後、開発者は彼らのために新しいプラットフォームで印象的なゲームを発表する準備が整いました。



ダウンロード： worldofsolitaire.com/worldofsolitaire_app.crx



TFデザイナー



このアプリケーションには現在、デモ版またはベータ版が機能していないため、このアプリケーションを最後に残しましたが、多くの利点があります。







「TF Designerは、視覚的でインタラクティブな要素を作成するための新しいWebベースのHTML5アプリケーションです。 それでも開発のプロセスでは、今後2か月で0.1の実験バージョンをリリースすることを望んでいますが、保証はしていません。



TF Designerは、インタラクティブなHTML要素での作業を簡素化することを目的としています。 私たちの製品が、数行すべてのコードを書いて、iPadに適したクールなインタラクティブバナーを作成するのに役立つことを願っています。 または、Javadcriptの限られた知識で、Facebookのゲームのように楽しいゲームを作成します。



手順-アプリケーションの実行方法：



Webアプリケーションをダウンロードしてインストールするには、 Chromeバージョンのdevが必要です。 インストール済みのアプリケーションを使用するには、 -enable-appsフラグを指定してブラウザーを起動する必要があります 。
