Ubuntuでの簡単なフラッシュのトラブルシューティング

ほとんどのUbuntuユーザーは、flashplugin-nonfreeパッケージに付属するいFlashにすでに慣れています。 個人的には、リリース後のリリースはさまざまな問題によって妨害され、フラッシュドライブのフラッシュ、フラッシュドライブの何かをクリックすると、一般的に不安定な動作と安定したブレーキが作動しました。 私の場合、Ubuntu 11.04のNatty Narwhalでは、Flashは非常に嫌悪感があり、使用できませんでした。 しかし、ほんの1週間前に、構成のフラッシュに関するすべての問題から私を救う簡単なソリューションに出会いました。 最も注目に値する-以前はフルスクリーンで480pだったビデオが著しく遅くなり、1080pでも完全にスムーズに再生されます。









万能薬



Firefox Flash-Aidのアドオンを紹介します。 アドオンについてのレビューから判断すると、そのおかげで、Flashは私だけでなく、より良く機能します。 そして、x64だけではありません。 Firefoxだけでなく！ Chromiumを使用している場合、Flashの問題も解決される場合があります。



プラグインの原理は非常にシンプルで、システムに適切なバージョンのFlashを選択し、ハードウェアアクセラレーションを有効にし、構成に最適な設定を選択します。その後、「実行」ボタンをクリックしてアドオンウィンドウから直接実行できるシェルスクリプトを生成します。



順を追った説明


注意！ 多くの人にとって、Flashは完全に機能しなくなったため、「実行」ボタンの代わりに「テスト」をクリックします。 私の愚かなヒントを聞いた後で既にFlashが壊れている場合は、アドオンをアンインストールし、flashplugin-nonfreeパッケージを再度インストールしてください。
  • アドオンをインストールしてFirefoxを再起動してください。
  • アドオンのリスト（[ツール]→[アドオン]）でFlash-Aidを見つけ、[構成]ボタンをクリックします。
  • 開いたウィンドウで「実行」ボタンをクリックすると、ターミナルが開き、管理者パスワードを入力します。
  • スクリプトの実行が完了するのを待って、Firefoxを再起動します。
  • ...
  • 利益!!!
このソリューションが、私を助けてくれたのと同じようにあなたに役立つことを願っています。 コメントの結果については登録を解除します。設定を示すことを忘れないでください。 私の場合、これは、Ubuntu 11.04 x64、専用ドライバーを備えたGeForce 460グラフィックスカードです。



代替ソリューション



roman_tikは、PPAリポジトリを使用してUbuntu x64にFlashをインストールするための代替ソリューションを提案しました。 この方法でFlashをインストールするには、ターミナルで実行します： 

sudo apt-add-repository ppa:sevenmachines/flash sudo apt-get update sudo apt-get install flashplugin64-installer


また、コメントには、Adobe Webサイトの Flashプラグインの動作に関する肯定的なレビューがありました。



PS。 また、同じ著者のFlashVideoReplacerアドオンに興味があるかもしれません。これは、一部のサイトのFlashビデオをシステムプレーヤーに置き換えます。
