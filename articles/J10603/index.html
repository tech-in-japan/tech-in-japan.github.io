id Softwareの最新テクノロジー-id Tech 5™



2007年11月6日にサンフランシスコで開催されたWorld Developers Conference 2007（ WWDC-2007 ）でのSteve Jobsのスピーチは、センセーションを巻き起こしました。 プレゼンテーションが続いた30分で、 APPLEの責任者はいくつかの強力な爆弾を投下することができました。 MacOS X 10.5の新バージョン、愛称LEOPARD 、Windows用Safari 3 「アップル」ブラウザ、PCおよびMacでのEAからの最新大ヒット商品の同時リリース、 iPhoneの販売開始日... ™。



id Tech 5は、Windows、MacOS X、Xbox 360、PlayStation 3をサポートし、次世代プラットフォーム向けの最初の真のテクノロジーとして発表されました。 Enemy Territory：Quake Warsの作成中にJohn Carmackが展開したMegaTextureコンセプトに基づいています。 新しいエンジンにより、アーティストやレベルデザイナーに使用可能なビデオメモリの量によって課せられる制限を完全に排除できます。今では、ゲーム内で必要なだけテクスチャを圧縮できます。



Jobsは、MacOS Xを実行しているMacintoshで発売されたレーシングアーケードのトラックを披露しました。レベルは、合計20 GBのユニークなテクスチャのみで構成されていました。 特別なツールの助けを借りて、アーティストは個々のピクセルのレベルで世界を自由に変えることができ、彼らの行動は生産性に影響しません。 使いやすいエディターに加えて、idは非常に詳細なオープンスペースとクローズドスペースで同等に高速な作業を約束します。



Doomの作者による新しい極秘プロジェクト（それは本当にレーシングアーケードですか？）、もちろん、id Tech 5を使用します。さらに、スタジオは既にライセンスの購入申請を受け付けています。 有望なエンジンに関する詳細は、7月11日から13日にカリフォルニアで開催されるE3 Media＆Business Summitで発表されます。



一言で言えば-「氷が壊れた、ju審員」...まだあるのだろうか？ 最も興味深いのはまだ来ていません！
