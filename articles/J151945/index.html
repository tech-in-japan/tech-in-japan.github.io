Kaspersky Anti-Virus for Linux File Serverをインストールします

ここ数ヶ月、ファイルサーバー上のウイルスの問題に悩まされてきました。 Nod32がサブドメインをブロックすると、カスペルスキーがサイトをブラックリストに登録します。 決して私を満足させることはできず、ある種のアンチウイルスを設定することにしました。



Clam AntiVirusは 、すべてのサーバーに既にインストールおよび構成されています。 数年前に使用しましたが、残念ながら、Trojan-SMS.J2MEクラスのウイルスが常に検出されるとは限りません。



グーグルの結果を研究したが、私は本当に何も見つけられなかった。



疑わしいサイトのリストからサイトを削除するというリクエストでカスペルスキーのサポートサービスに再び目を向けると、 Linuxファイルサーバー用の流行のカスペルスキーに出会いました 。 私はそれをテストすることにしました。



このウイルス対策ソフトウェアのインストールと構成に関するヘルプをGoogleに送信することもできませんでした。 すべての結果は、カスペルスキーのサポートサイトにつながります。



本当に誰も彼らのファイルサーバーに彼らの配布キットを置きませんか？ おそらく他の解決策がありますか？



これらの質問に対する答えは、私にとって謎のままです。 上記の製品に落ち着いて、テストすることにしました。



テクニカルサポートサイトでテストライセンスファイルをリクエストします。 答えは数時間で届きます。

すべてが判明し、私たちのウイルスは問題なく判断します。 もちろん、チェックの結果としてどのウイルスを確認したいのですが、サポートサービスからの回答があり次第、登録を解除します。 残念ながら、この情報はマナでは見つかりませんでした。

ウイルス対策のセットアップとテストは成功しました。 次に、すべてのファイルサーバーをチェックし、新しいファイルのオンラインチェックを設定します。
