ストーカーは相続人を得た

昨年12月、一連のポスト黙示録的な射手STALKERで広く知られているウクライナの会社GSC Game Worldが不明確な理由で閉鎖されたというニュースは、最も成功したプロジェクトの運命に関する多くの質問を提起しました。 定期的に、会社は実際に閉鎖の危機にonしており、出版社との交渉が進行中であるというニュースが現れ、その後、突然すべてがうまくいき、プロジェクトは別の出版社とのコラボレーションを続けることが判明しましたが、いつどのように明確ではありませんでした。



この問題は明らかになりました。



元GSC Game Worldの従業員は、2012年3月にキエフでボストークゲームを作成しました。 プロジェクトのウェブサイトには次のように書かれています：



10年前、GSC Game Worldの壁の中で、有名なSTALKERゲームシリーズの開発を開始し、2011年12月にスタジオが予期せず閉じられ、STALKER 2プロジェクトが凍結されるまで、その作業を続けました。 Stalkerブランドを使用するための条件に関して著作権所有者との合意に至らなかったため、新しいプロジェクトで独立スタジオとして活動することにしました。 この日はボストークゲームの出発点でした。





実際、今日、同社は新しいSurvariumプロジェクトを発表しました。これは、開発者によって次のように説明され、「STALKERアイデアの進化」と呼ばれています。



ゲーム「Survarium」は、近い将来に行われるfree2playの一人称シューティングゲームMMOです。 ゲームのイベントは、地球上の大規模な環境災害を中心に構築されており、その原因は確実に不明です。

侵入不可能な森林はあらゆる側面から都市を攻撃しており、取り乱した動物や鳥は工業団地、軍事施設、倉庫、発電所を攻撃しています。 都市は互いに遮断され、水、電気、ガスはなく、通信は機能しません。 国家が崩壊し、無秩序が世界を席巻している。 今、力は誰が生き残るかを決定しています。 一方、動植物の新種は、意図的に人間の文明を破壊します。 奇妙な植物とキノコがコンクリートと鉄に芽生えます。 世界中の科学者は、異常な状況に対処しようとして失敗し、恐ろしい速度で地球をcreepい回っています。





Survariumは2013年の終わりにPCでリリースされる予定です-別のサイトが既にプロジェクト用に立ち上げられており、これまでのところコンテンツで輝いていません。 ストーカーの最も有名な部分である「チェルノブイリの影」の「長期的な建設」に関する悪名高い物語が二度と起こらないことを願うだけです。
