Vivaldi 1.0.377.10ビルドでのタブのクローズとその他の改善をダブルクリック

みなさんこんにちは！



次の毎週のビルドは一種の異常なものですが、まず、Mac OSXのバージョンを提供する準備が再び整いました。次に、過去3日間でブラウザーコードに23の修正を加えました。



それで、このアセンブリにあるもの。



ブラウザの設定を追加および編集し続けます。 利用可能なオプションの数は増え続けており、ブラウザを完全に制御することを好むユーザーに喜ばれるはずです。 正直に言うと、私はすでに自分自身に迷い始めています-以前に追加されたもの、そして今- しかし、黙って言うよりも2回言う方が良いです。 一般に、ブラウザにはどういうわけか目立たないようにタブの設定があり、ダブルクリックでそれらを閉じることができます。 この機能は長い間要求されてきたことを正確に覚えています。 さて、今-今は：









また、タブを複製するためのキーボードショートカットを作成する機能も追加しました。 あなたも尋ねた-私たちはやった。



ピン留めされたタブのサウンドインジケータの位置が不便であるという苦情もありました。 私たちもそれを修正しました。



ベラルーシのロケールのユーザー向けのニュース。 デフォルトでブラウザにベラルーシ語のブックマークが表示されなかったバグを修正しました。 すべてが正常に機能するかどうかを確認します。



さて、さらに多くの種類の便利で必要な修正。



変更の完全なリストは、記事の最後にあります。



それだけです 以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac 10.7+
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
