バージョン1.0.9

数時間前、次のリリースがいつも待ち望まれていたYiiフレームワーク1.0.9のリリースが発表されました。



いつものように、いくつかの注目すべき革新を簡単に検討してください。

  • CActiveRecord :: beforeFindメソッドとonBeforeFindイベントを追加しました。
  • CActiveRecord :: deleteAllByAttributesメソッドを追加して、より便利で柔軟な削除を実現
  • 可能な限りJOINを回避することにより、遅延読み込みの速度が向上しました。
  • 厳密な解析モードが有効になっている場合、アドレスサフィックスにスラッシュ文字を使用できます
  • CApplication :: timeZoneプロパティを追加しました
  • これらのイベントのリスナーがない場合にイベントをスキップすることにより、AR速度を改善しました。
  • メソッドCLocale :: getMonthNamesおよびCLocale :: getWeekDayNamesが追加されました。これらのメソッドは、それぞれ月と曜日の名前を返します。 さらに、それらは異なる形式で返されます：拡張、短縮、および略語の形式。
  • CDbCriteria :: addCondition（）メソッドが追加されました 。これにより、既存の条件に新しい条件を追加できます。
注意！ バージョン1.0.9へのアップグレードは、ActiveRecordの関連オブジェクトの遅延読み込みに注意する必要がある場合を除き、一般に安全です。 リンクテーブルはJOINを介してメインテーブルに接続されなくなったため、対応する関係のクエリの説明でメインテーブルを参照することは避けてください。



テキスト内の参照は、ドキュメント内の適切な場所につながります。
