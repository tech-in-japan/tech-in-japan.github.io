BaseRide ニジニノヴゴロドからシンガポールへ

Baseride Technologiesが2008年にニジニノヴゴロドで設立されてから、ビッグデータ分析と輸送会社向けの特殊なクラウドソリューションの開発に携わる会社は多くの時間が経ちました。 過去数年にわたり、クリエイターはロシアと海外の両方で多くのソリューションを開発および実装することができました。



High Performance Computing AStar Singapore Instituteのサポート、投資のレバレッジ、および優れた市場の見通しにより、同社はアジア市場での足場を獲得したいと考えています。



最近、同社はシンガポールIncuVestファンドによって60万ドルで投資されたことが判明しましたが、従業員によると、投資は前年の終わりに合意されたことが明らかになりました。

アジアの技術投資会社。

創設者：ロニー・ウィー（ロニー・ウィー）およびナターシャ・フォン（ナターシャ・フォン）。

主な関心事：消費者セクター、eコマース、データ分析、モバイルアプリケーション、ウェアラブルエレクトロニクス、IoTおよびB2B。

いくつかのポートフォリオプロジェクト：GoCoin、Chope、Reebonz、Smartmissimo Technologies Pte Ltd、Sogamo、Gamera Networks、Cinarra Systems、Inverted Edge、Feihu interactive、Vanity Trove、Zelrealm、Matchimiなど。




2013年、Mashableはチームの最初のプロジェクト-Doroga-TV をロシアのトップ20のスタートアップに含めました。 同年、同社はフリートを監視および分析し、モバイル要員の作業を制御するためのいくつかのB2Bソリューションをリリースしました。 現在、BaseRideは、製品をGPS / GLONASSデータ形式のビッグデータを扱うためのユニバーサルプラットフォームとして位置付けています。
