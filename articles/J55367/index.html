Lenovo ThinkStation S20 / D20-Mac Proの確かな答え



おそらく、昨日始まったGame Developers Conference 2009で、Lenovoが新製品を発表し、意見に反して、これが別のラップトップ/ネットブックになるとは誰も予想していなかったでしょう。 秘密のベールは、主に3Dモデラー、デザイナー、および信頼できるギガヘルツと高速メモリを必要とする他の職業の代表者に焦点を当てた2つのデスクトップThinkStation S20およびD20から削除されました。



このモンスターの中には、Intel Xeonプロセッサ（Nehalem、新しいMac Pro と同じ5500および3500シリーズに基づく）および新しいNVIDIA Tesla C1060 GPUまたはATI FireProアクセラレータが概念的に新しいハードウェアで実験したくない場合に隠れています。 選択したOS：Windows VistaまたはRedHat Enterprise Linux 5.2



店頭では、これらの美しさは、来年の第1四半期（またはそれより少し前）に、それぞれ1070ドルと1550ドルの価格で登場します。



カットの下の駅の仕様、テキストの言語は英語です。
