Yandexの「コミュニティメッセージ」と「Webvisor」

こんにちは、ユーザー様！ 少し前、VKontakteは「コミュニティメッセージ」サービスを発表しました。 そして、最初の熊手と松葉杖はすでに動作しているように見えます。むしろ、訪問者の目を通してあなたのサイトを見る方法です。 Yandex.Webvisorを起動します。 そしてそこに：









問題の簡単な解決策。



YandexはWebvisorを使用してソリューションを見つけました- ページのコンテンツを記録します ： 



if (/^(.*\.)?mtproxy\.yandex\.net$/.test(document.domain)) { /*   */ } else { VK.Widgets.CommunityMessages("vk_community_messages", /* ID- */, {}); };


21:30に投稿：なぜこのウィジェットを非表示にしないのですか？ なぜ画面の中央にそのようなスタブがあるのですか？



PS尊敬される企業が相互のバグを解決することを願っています。



PPSはユーザーからの質問を承知の上で、VCをチェックして許可されたドメインを設定* .yandex.netに挿入しようとしました -助けにはなりません。



PPPSこのバグは、開発されたサイトの1つで偶然発見されました-Webvisorを見るのが好きな顧客によって。 このソリューションを複数のサイトで一括登録する必要がありました。
