wireframe.ccは最も簡単なインターフェイスプロトタイピングツールです

これまで、インターフェイスの便利で迅速なプロトタイピングのための多くのWebアプリケーションがありますが、 wireframe.ccが私が試した中で最高です。







主な機能は次のとおりです。



  • 最小限のインターフェイス：作業画面に余分なものはありません-ボタンの小さなパネルとデバイスウィンドウのみです。
  • ブラウザおよびモバイルデバイス用のウィンドウテンプレート（横向きおよび縦向きバージョン）。
  • 要素を追加する非常に便利な方法：ドラッグして描画し、オブジェクトのタイプを選択するだけです。
  • インターフェイスにコメントを簡単に追加します。
  • スマート要素：wireframe.ccは、オブジェクトのサイズによって、それが見出し、フォーム、またはスクロールバーであることを判断しようとします。
  • プロトタイプ（すべての人に短いリンクが与えられます）の便利な保存とTwitterおよびFacebookでの共有。
  • シンプルなカラーパレット。
  • ホットキー：標準のCtrl + C / Ctrl + V、Photoshop Alt +ドラッグしてオブジェクトを複製し、ダブルクリックして編集します。
