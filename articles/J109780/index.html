PHPShop IDE 1.0リリース-統合開発環境

PHPShopソフトウェアのテストおよびhabrayuzereの支援により、統合開発環境PHPShop IDE 1.0.0.10のリリースがリリースされました。



PHPShop IDEは強力で、PHPShop APIに基づいたコード編集プロセスを高速化します。 幅広いユーザーに焦点を当てた開発環境は、初心者には不可欠であり、PHPShopを使用する専門家には便利です。 主な機能は次のとおりです。

  • PHPShop API組み込み関数の強調表示と編集
  • 解析および設定ウィンドウから追加メソッドを編集する機能
  • 一般的に使用されるHTMLおよびPHP関数へのアクセス
  • 構成ファイルとその設定へのアクセス
  • PHPShop API機能リファレンス
  • 新しいモジュールと管理インターフェイスを作成する際のアシスタント
  • 外部XML IDE構成ファイルを介して新しい機能を追加する
  • プロジェクト支援
  • デザインテンプレートの編集
  • デバッグ


新しい機能がリリースに追加されました。

  • すべてのプロジェクトファイル（php、sql、ini、xml、js）のサポートと編集
  • モジュールを作成および編集するための「アシスタント」の新機能
  • ファイル処理の高速化


システム要件：

  • Windows 2000、XP、Vista、7
  • P4 1000、256 RAM
