ダウンロード可能なWindows 8.1およびVisual Studio 2013





やった！ 多くの人が待ち望んでいたこと-Windows 8からWindows 8.1へのアップデートとVisual Studio 2013の最終バージョンがダウンロード可能です。

* Windows 8.1へのアップグレードのお知らせ

* VS2013のリリースのお知らせ

* W8.1アプリケーション用にWindowsストアを開くことの発表



Windows 8.1へのアップグレードは、すべてのWindows 8ユーザーがWindowsストアから直接無料で利用できます。



Visual Studio 2013は、MSDNサブスクリプション（スタートアップの場合はBizSparkからアクセスするなど）、 DreamSpark （学生の場合）、またはスタジオのWebサイト （試用版とエクスプレス版）からダウンロードできます 。





11月にVS2013を起動 し 、10月23日に開発者向けWindows 8.1を起動する仮想イベントにもご招待いたします。
