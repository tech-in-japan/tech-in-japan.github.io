新記録：7日間で1,200万ダウンロード

2009年12月7日-ノルウェー、オスロ



11月23日に、Operaはすべてのインターネットユーザーに、Opera 10.10ブラウザの新しいバージョンをリリースすることにより、Webの再発明に参加するよう招待しました。 今週、Opera 10.10は1,250万回ダウンロードしました。これは、2か月前にリリースされたOpera 10の同じバージョンのダウンロードレコードより25％高いです。



Opera 10.10ブラウザーの最も人気のある機能の1つはOpera Uniteです。これは、ユーザーが写真や音楽などのコンピューターに保存されている個人ファイルへのアクセスを迅速かつ簡単に提供できるパーソナルWebサーバーです。



「このような印象的なダウンロードは、ブラウザがインターネットコミュニティの要件を満たしていることを示しています。 Operaブラウザの最高経営責任者であるJon von Tetzchnerは、次のように述べています。 「Opera Uniteは、時間の経過とともに、Web上の情報にアクセスしてデータを共有する方法に関する偏見を克服すると信じています。」



現在、約4,500万人がOperaデスクトップブラウザを使用しており、携帯電話、ゲーム機、テレビ、ポータブルメディアプレーヤーでOperaブラウザを何千万人も開いています。 OperaはOne Web（One Web）の原則に基づいて製品を開発し、いつでも、どこでも、どのデバイスからでもインターネットにアクセスできることを提唱しています。



Operaブラウザの主な機能：

•Opera Unite-2台のリモートコンピューター間でコンテンツを直接共有

•Opera Turbo-サーバー側でのデータ圧縮により、低速チャンネルでのWebページの読み込み速度が向上します

•視覚化されたタブ-ナビゲーションを容易にするために、開いているページのサムネイルがタブバーに表示されます。

•エクスプレスパネル（スピードダイヤル）-新しい各タブでお気に入りのサイトにすばやくアクセス

•Opera Link-ブラウザ設定の同期とOperaブラウザを備えた他のデバイスを使用した設定へのアクセス

•マウスコントロール（マウスジェスチャ）-小さくて速いマウスの動きによる効率的なサイトナビゲーション



Operaブラウザの最新バージョンをダウンロードします： www.opera.com/browser



Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
