ダウンロード可能なSystem Center 2012 SP1ベータ版

System Center 2012 SP1のベータ版がダウンロード可能になりました。 ベータ版には、すべてのSystem Centerコンポーネントの更新プログラムが含まれており、機能面では、ほぼ完全なリリースです（完全な機能）。 追加機能のごく一部のみが後で表示されます。



ベータ版がもたらすものの詳細な説明は、サーバーおよびクラウドプラットフォーム開発チームのブログ （ Server＆Cloud Blog ）にあります。 私は主観的な観点から最も重要な点のいくつかにしか注意していません。



最も重要なこととして、System Center 2012 SP1 Betaは、 Windows Server 2012の最終（RTM）バージョンの管理を提供します。これには、ネットワーク仮想化、Hyper-V拡張スイッチ拡張機能の管理、論理スイッチの作成などの新しいHyper-V機能が含まれます。



Virtual Machine Managerでは、SMI-Sストレージのサポートに加えて、Windows Server 2012に基づいてファイルストレージを管理できるようになりました。WindowsServer 2012のSMB 3.0のおかげで、Hyper-Vはファイルサーバーの共有フォルダーで仮想マシンのVHD / VHDXファイルをホストできます。 SMB透過フェールオーバーおよびSMBマルチチャネルメカニズムは、それぞれ物理サーバーおよび通信チャネル/ネットワークアダプターのレベルでこれらのフォルダーのフォールトトレランスを提供します。 VMMコンソールから、継続的に使用可能なモードで共有フォルダーを作成するなど、このようなファイルリソースを管理できます。



異質性に関しては、ホストHyper-V以外のVMMはCitrix XenServer 6.0およびVMware vSphere 5.0の管理をサポートしています。



最後に、ハイブリッドクラウド内で、App Controllerは以下を提供します。

  • Windows Azureでの仮想マシンロールのサポート-VM展開、VM管理、VMへのリモート接続。
  • VMMからWindows AzureへのVMの移行。
  • VMMライブラリのサービステンプレートに基づいてWindows AzureにVMを展開します。


一般的に、System Centerの各コンポーネントについて、非常に印象的な革新のリストを見つけることができます。
