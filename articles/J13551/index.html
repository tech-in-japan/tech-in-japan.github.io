ご存知のように、検索エンジン最適化（広告なしで新しい訪問者をサイトに引き付ける主な方法）は、長年数百万ドルの業界になりました。 さまざまな推定によると、検索エンジン最適化市場のボリュームは5,000万ドルから1億ドルまで変化し、これはインターネット上のメディアおよびコンテキスト広告市場のボリュームに個別に匹敵します。



驚くべきことに、このような市場指標では、SEOサービス自体は絶対に形式化されていません。 正式な形式ではないため、最終的には、集客されたトラフィックの量だけに関心がある顧客は、自分が何を買っているのかを事前に知りません。 より正確には、CPC、CTRなどのよく知られた指標に従って、SEOサービスとインターネット広告（コンテキスト広告またはメディアカード）をアプリオリに比較することはできません。

プロモーションと監視は2007年6月に実施されました。

会社のウェブサイトPervostrachへの移行の統計は、無料の出席カウンターの1つを使用して計算されました。



この調査の目的は 、これらのクエリのYandex検索結果の個々の位置のCTRに関するデータを経験的に取得することです。



研究モデルの仮定

1.問題のクエリを含むクエリのインプレッションの総数（検索結果の全ページ）に対する問題のクエリの最初の検索ページのインプレッションの比率は、YandexはRamblerと同じ比率に等しくなります。

2.平日のリクエストの配信は月全体で均一です。つまり、特定の日付と曜日に依存しません。 週末のリクエストの配信に関しても同様の仮定が行われています。

3.週末のリクエスト数に対する平日の平均リクエスト数の比率は2と想定されます。この数は、6月のYandex検索のヒット総数の統計に基づいて計算されます-平日のヒット数は週末の2倍です。

4.特定の検索結果アイテムのCTRデータは、少なくとも3日間のデータから計算されます。 つまり、あるリクエストのサイトが問題の位置に3日未満立っている場合、このケースのCTRは大きなエラーのために計算されません。



決済手順



1. 1日あたりのインプレッション数。

させてください：

• Yah- 6月のYandexでの対象クエリのインプレッション数（検索結果のすべてのページ）、

• Rx- 6月のRamblerでの問題のクエリのインプレッション数（検索結果のすべてのページ）、

• R1-6月のランブラーの問題のクエリの最初のページのインプレッション数 。

次に、Yandexで考慮されるリクエストの最初のページ（つまり、First-Straight Webサイトが提示されたページ）の必要なインプレッション数が、仮定1を考慮して計算されます：

R1 = Rx * R1 / Rx

Rx = 62'911（ ソース ）

1 = 11'973を取得します



6月には30日があり、そのうち20日が平日で10日休みであり、仮定3（週末の2倍の週末）を見て、1日あたりの「保険」のヒット数をサイトで取得します。

平日：478

同様に、他のリクエストの1日あたりのインプレッション数を計算します。



2.実際のCTR （1000インプレッションあたりのクリックのシェア）

6月の各日と問題の各リクエストについて、「ローカル」CTR =（クリック数/インプレッション数）* 100％を計算します



次に、問題のクエリごとに、検索エンジンが1か月間サイトを維持した位置を書き出します。 特定の要求で、サイトが特定の位置に3日未満立っている場合、このケースについては計算しません（仮定4を参照）。



その他の場合、CTRはサンプル平均として計算されます。

CTR =（CTR1 + CTR2 + ... + CTRn）/ n、

ここで、nは、このリクエストに対してサイトが特定の位置に立っていた日数です。



計算は、平日と週末に別々に実行することが重要です。

結論



1.明白で、研究の説明なし-CTRは検索結果の関連性に依存します-ここで非常に明確に確認されます。 CTRの広がりは、0％〜100％のすべての許容値内で可能です。 人々が「最初の星」を探していて、そもそも最初の星のウェブサイトを取得している場合、これはほぼ100％のCT​​R（平均して1か月未満ですが、平日はほぼ100％）です。 一方、非常に「あいまいな」「保険」のリクエストの場合、検索結果のPervostrahaの最初の場所では桁違いに低い結果が得られます。CTRは約5％で、コンテキスト広告と同等です。 したがって、たとえばosagoのプロモーションは、保険のプロモーションよりも収益性が高くなります。 後者は2倍のインプレッションを持っているという事実にもかかわらず、最初は4倍のCTRにより2倍のコンバージョンをもたらします。



2.平日の保険トピックのリクエストのCTRは、週末よりも約30％高くなります。 保険の主な利害関係者は、職場でインターネットを使用して平日に保険情報をより積極的に求めていることは明らかです。



3. 「保険」や「保険会社」などの「あいまいな」要求の場合、最初の場所のCTRの差はわずかです。 「保険会社」の場合でも、反対に、CTRは1位から3位に増加しています。 より具体的なリクエストの場合、掲載順位が下がるとCTRがかなり急激に低下します。
