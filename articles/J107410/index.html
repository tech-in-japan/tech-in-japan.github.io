新しいグループ、古いグループ、Facebookページ

ページとグループの機能と役割の観点からFacebookで発生した「改革」により多くの混乱が生じたため、これらのツールの設計方法と現在の目的について説明します。



ページ主な目標は、フィード内のニュースを無制限の「ファン」に公開することです。ページを作成します： http : //www.facebook.com/pages/create.php
  1. ファンのニュースフィードに投稿できます
  2. 次の方法でページを宣伝できます：a）Facebookでの広告b）ソーシャルプラグイン（ボタンが好き、ボックスが好き）
  3. 統計があり、ページプロモーションの有効性を評価できます。
  4. 複数のユーザーがページを管理できます（無制限の数の管理者を追加できます）。
  5. ユーザーは、ページ管理者の承認なしに、自由にファンになります。
  6. アプリケーションを作成してページに追加できます。
  7. ページにタブを追加したり、固有の情報を持つタブを作成したりできます。
  8. ページ管理者の個人プロファイルはページのファンには表示されません。管理者は常にページに代わって公開およびコメントします。
  9. 特定の国のファンと特定の言語を話すファンのみに投稿を公開できます。


新しいグループ

少人数のグループ向けに特別に作成されており、互いに通信できるようになっています。グループを作成： http : //www.facebook.com/ajax/groups/create_get.php
  1. グループチャットがあります
  2. ドキュメント：グループメンバーは1つのドキュメントを編集できます
  3. グループメンバーは、他のメンバーがグループに情報を投稿したときに電子メールアラートを受信できます。
  4. グループ管理者はグループメンバーにメールを送信できます。
  5. 秘密グループ（Facebook検索では表示されません）、閉じたグループ（検索では表示されますが、ユーザーはグループメンバーの招待時にのみグループに参加できます）、開いたグループ（検索で表示され、誰でもグループへの参加リクエストを送信できます）を作成できます。グループ管理者は、グループへの参加のすべてのリクエストを承認する必要があります！）。
  6. メンバーは、グループ内の新しいメッセージの通知方法を選択できます。 メールやFacebookで通知を受け取ることができます。
  7. グループ管理者の個人プロファイルは、グループメンバーに表示されます。
