RuSSIR 2013：VII情報検索に関するサマースクール

2013年9月16〜20日、カザンは情報検索に関するVIIロシアサマースクールを開催します（ RuSSIR 2013 ）。



カザン連邦大学（ KFU ）と、情報検索方法の評価に関するロシアのセミナー（ ROMIP ）が、Yandex、Mail.ru、 G o o g l eおよびABBYYの参加を得て開催しています。



今年の学校の主なテーマは、音声情報の検索と処理です。 使用言語は英語です。







メインプログラムは7つのコースで構成されています。



音声および音楽情報の検索：

  • 音声コンテンツ検索：課題、テクニック、アプリケーション-Gareth Jones （ダブリン市立大学）

  • コンテンツベースおよびコンテキストベースの音楽の類似性および検索- マーカス・スケジュール ＆ ピーター・ニーズ （リンツ大学）

  • 音楽情報検索の2つの成功したパラダイムとしての歌唱/ハミングと音声指紋によるクエリ-Jyh-Shing Roger Jang （台湾大学）

  • オーディオと音楽の検索における適応性- アンドレアスニュルンベルガー ＆ セバスチャンストーバー （OVGマグデブルク大学）



一般情報の取得：

  • 情報検索モデルの紹介-Massimo Melucci （パドヴァ大学）

  • 大規模な情報検索の手法- パオロボルディ （ミラノ大学）

  • テキスト分類における新しい表現と方法-ManuelMontes -y-Gómez ＆ Hugo Jair Escalante （INAOE、メキシコ）



学校への参加は無料ですが、場所の数は限られています。





そこに着く方法は？

学校に参加するには、申請書に記入し、あなたの仕事-開発または科学研究-を説明する科学記事またはポスターレポートを送信する必要があります。



作品は学校のテーマに対応する必要があり、次の分野のいずれかに関連している可能性があります（ただし、必ずしも正確ではありません）。

  • 音楽情報検索（MIR）;
  • IRおよび理論とモデル。
  • データ構造、アルゴリズム、およびインデックス作成。
  • 自然言語処理;
  • ユーザーインターフェース;
  • セマンティックWebおよび知識データベース。
  • テキストマイニング、情報および事実の抽出。
  • IRおよびデータベース用のモバイルアプリケーション。
  • 動的なメディアおよびデータストリーム。
  • ソーシャル検索、推奨事項、パーソナライズ。
  • IR評価。


学校はRuSSIR YSC 2013カンファレンスを開催し、そこであなたの作品を発表することができます。



筋金入りの方法：完全な出版



テンプレート（ Springer template ）に従って、完全な出版物（フルペーパー）が英語の本格的な科学記事として発行されます。 記事は、未発表のオリジナルの作品を提示する必要があります。



最大ページ数は10です。著者の少なくとも1人は30歳未満でなければなりません。



承認された記事は、科学雑誌「カザン大学の科学ノート」の特別号に掲載されます（その場合、高等認証委員会のリストに含まれます）。



記事の受理は、学校への入学を保証します。 しかし、たとえ記事が受け入れられなかったとしても、それはあなたの申請の可能性を深刻に増加させます。



出版物は、2013年 6月2日までにEasyChairシステムを介してPDF形式で送信する必要があります。



軽い方法：記事のポスター



あなたは普通の開発者で、科学論文を書くという考えは恐ろしいですか？ あなたの仕事はまだ準備ができていませんか？ または、反対に、研究の結果はすでに他の場所で公開されていますか？



その後、参加する必要があるのは、短いメッセージを準備するか、科学界の言語でポスターを作成することだけです。 ポスター記事は、現在のプロジェクトを説明する2ページの英語のテキスト 、または将来の作品、およびおそらく以前に公開された作品の興味深いアイデアです。



ポスター記事を作成することはまったく難しくありません。 順番に見てみましょう。 例として、最近のモスクワでの国際ECIR会議の最高のポスター用紙をご覧ください 。



  1. 既製のSpringerテンプレートを使用します。 記事には、タイトル、著者データ、簡単な要約（要約）を含める必要があります。 注釈は通常、研究の一般的なコンテキストを非常に簡潔に定式化し、研究の最も重要な事実または結果を提供します。





  2. 原則として、記事の最初の章、または「はじめに」では、研究のコンテキストをより詳細に明らかにし、研究中に解決される主な問題とタスクを定式化します。





  3. 次に、問題を解決するために提案されたアプローチを明らかにします。





  4. 可能であれば、実験結果を提供します。





  5. そして最後に、研究について結論を出し、使用された文献の短いリストを提供します。





ポスター記事は、参加者フォームを通じてPDF文書として送信する必要があります。



メンバーフォームを送信



登録フォームに記入するには、いくつかの追加資料を準備する必要があります。

  • ポスター記事（上記を参照）：あなたが完全な記事の著者である場合、簡単な注釈を送信するだけで十分です。
  • CV（CV）。英語の自由形式で記入され、あなたの教育レベル、専門的経験、科学出版物およびその他の成果のリストが必須表示されます。
  • （オプション）推薦状-これは、連絡先情報（電子メール、電話）の必須表示を伴う、指導教員または教師からの手紙である場合があります。


次に、 参加者フォームのページに移動し、記入して送信します。



フォームの送信期限は2013年 6月14日、6月 25日です。



不明な点がある場合は、コメントに答えてみます。
