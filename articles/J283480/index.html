NFC-大衆へ。 Eurovision、音楽、スポーツ会場の新技術

あなたがサッカーファンであろうと音楽ファンであろうと、あなたはおそらく人々が大きな舞台で素晴らしいショーを見るために集まったときに生じる独特の一体感をよく知っているでしょう。



それにもかかわらず、スタジアムの混雑したスタンドと数万人の観客は、特に入り口に巨大な列を守らなければならない場合、またはすでに中にいるときに飲み物を買おうとする場合、最悪の経験の1つになる可能性があります。 同時に、観客だけが苦しんでいるわけではありません。 スタジアム運営会社は、たとえば、現金の取り扱い、人々の流れの管理、いじめへの抵抗などの課題にも直面しています。







NFC、モバイルアプリ、スマートイベントプラットフォームは、アリーナの観客に究極の体験を提供します



しかし、このシナリオに従ってすべてを開発する必要はまったくありません。 適切なテクノロジーを選択することで、キュー自体は過去のままでありながら、セキュリティを大幅に向上させ、場所ごとの収益を増やし、訪問者のニーズをよりよく理解できます。 Gemaltoによって導入されたNFCテクノロジー、モバイルアプリケーション、スマートイベントツールは、Allynis Smart Event Platform（ASEP）の基盤であり、次のものが含まれます。

  • 登録済みチケットとアクセス制御技術のサポート。
  • オンライン支払い（EMV）とプリペイド支払い（予定外）の両方のサポート。
  • 輸送システム事業者向けのアプリケーション;
  • ユーザーエクスペリエンスを向上させるソーシャルメディアの連絡先。
  • リアルタイムのデータ収集と分析。
  • モバイルマーケティング;
  • その他-常に新しい機能や機能を開発および追加しています。


インテリジェントなサブスクリプション



すべてのシーズンチケット所有者をインテリジェントイベントプログラムに登録する機能は、主催者とファン自身の両方に役立ちます。これにより、キューのかなりの部分を取り除き、出席のより良いアイデアを取得し、サービス速度を上げ、ブランド化の可能性を得ることができます-たとえば個々の売り手との割引を組織したり、プロモーションを開催したりします。



シーズンチケット所有者ごとにNFC対応のブレスレットがあれば、イベントにすばやくアクセスでき、現金を使用せずに即座に支払いが可能になります。 モバイルアプリケーションは、必要なものすべてを提供します-残高をチェックしてVIPゾーンにパスする機会から始まり、半分の間にスナックと飲み物を事前注文して場所を見つけることで終わります。 これは、すべてのホームマッチで最も重要なファンにはるかに高いサービス品質を提供できることを意味します。



高度なNFCテクノロジーと専用サービスシステムを使用したGemalto ASEPプラットフォーム：

  • イベント中に顧客サービスの品質を向上させることができます;
  • 既存の収入源と新しい収入源の両方に基づいて、収入を増やすのに役立ちます。
  • 公共の関与の度合いとその忠誠心を高める。
  • セキュリティの向上に貢献します。
  • イベントが開催される会場で行われるすべての金融取引の透明性を提供します。
  • アフィリエイトブランドとの対話におけるイベントの重要性を高めます。


さらに、このプラットフォームを使用すると、従来の支払いシステムを使用したオンライン支払いとプリペイド支払いの両方を同時に処理できます。 これにより、チームはスタジアムで動作する独自の安全な支払いシステムでゲームの収益の大部分を維持でき、ファンはNFCテクノロジーをサポートする小売業者でEMVカードを使用して非接触トランザクションを定期的に実行できます。



Allynis Smart Event Platformは、英国のSaracens Rugby Clubを含む、ヨーロッパ各地の多くのスタジアムや会場ですでに展開されています。 ジェムアルトは、スポンサー付きイベントをサポートするために魅力的な製品を展開するために、世界最大の多数の決済システムおよび金融機関と協力しています。



2016年は、知的イベントやスタジアムでのキャッシュレス支払いの組織向けに設計されたテクノロジーにとって、真のブレークスルーの年となります。 ジェムアルトに連絡して、今後増え続けるこのようなインテリジェントサイトに参加する方法を確認してください。







ロシアは準備をしている



ロシアは2018 FIFAワールドカップに向けて準備を進めており、100万人以上が参加する予定のこの大きなイベントのハイテクサービスの提案を検討する必要があります。 輸送、スタジアムへのアクセス制御、および支払いの分野のサービスをカバーできるウェアラブル非接触ソリューションを使用した「キャッシュレススタジアム」の出現に向かう傾向が見られます。







2015年7月、ジェムアルトは、アルファバンクの支援を受けてニジニノヴゴロドで開催されたロシアの電子音楽とテクノロジーのフェスティバルであるアルファフューチャーピープルフェスティバルで、ウェアラブル決済デバイスのNFCテクノロジーのサプライヤーとして行動しました。 1年前、フェスティバルは非現金の領土となり、ゲストは非接触技術を使用して購入した商品を迅速かつ安全に支払うことができました。 このコンセプトの一環として、フェスティバルの訪問者は、ドリンク、食べ物、お土産など、フェスティバル中のさまざまな購入品を非接触で支払うためのブレスレット付きウェルカムパックを受け取りました。



近日中に開催されるファイナル



ジェムアルトは、世界最大の決済サービスプロバイダーの1つであるVisaと提携して、2016 Eurovision Song Contestの訪問者に非接触リストバンドを提供しています。 このソリューションには、非接触リストバンドのパーソナライズ、パッケージング、実行が含まれ、プリ​​ペイド支払いプログラムとイベント管理に当社のAllynis Smart Platform Eventプラットフォームを使用しています。







ビザの非接触リストバンドは、ストックホルム（スウェーデン）のユーロビジョンビレッジで購入できます。 今年、このコンペティションは、非接触デバイスを使用して経験を積む65,000人までの訪問者を引き付けることを期待しています。 イベントの集大成は、グローブアリーナでのグランドファイナルの5月14日（この土曜日）に行われます。



統合されたGemaltoソリューションにより、Visaは、現金や支払いカードを使用せずに購入するなど、ビジネスでの非接触のやり取りを実証することができます。 Gemaltoが特別に作成したWebサイトを通じて、前払い残高や以前の取引に関する情報など、アカウント情報にリアルタイムでアクセスできます。



支払いを行うためのモダンで機能的なリストバンドには、ジェムアルトの非接触チップが搭載されており、国際決済システムのいずれかによって認証されている通常の非接触EMVカードと同じセキュリティ基準を満たしています。 訪問者は、フェスティバルの領土に設置された非接触端末を介してブレスレットのバランスを補充できます。 ブレスレットは撥水性があり、損傷に強いです。 したがって、それらはフェスティバル期間中の屋外でのアクティブな使用に最適です。



Eurovisionプラットフォームに加えて、ブレスレットを使用して、世界中の非接触型決済に対応するVisa POS端末で支払いを行うことができ、ショー終了後も2016年10月31日まで稼働し続けます。







Allynis Smart Platform Eventは、ウェアラブルデバイスやモバイルNFCなど、支払いのフォームファクターと組み合わせて使用​​できます。 このプラットフォームは、課金管理、アクセス権、およびライセンスされたアプリケーションの広範なポートフォリオも提供します。 完全なソリューションは、「タッチアンドゴー」支払い、リラクゼーションを目的とした会場や場所への簡単なアクセス制御、その他の多くの便利なアプリケーションなど、幅広い機能をサポートしています。 訪問者を改善し、保護します。また、主催者やその他の関係者が、イベントの前後、リアルタイムで機能するインテリジェントなコミュニケーション、マーケティングプログラム、ロイヤルティプログラムを実装できるようにします。



Allynis Smart Event Platform（ASEP）は、銀行決済技術、製品設計、インタラクティブイベントに関する専門知識に基づいて開発されたもので、パートナーや顧客がイベントや会場をよりきめ細かく制御できるようにする特別なサービスです。

  • クライアントがイベントで受け取る前向きな体験の向上
  • 既存および新規に作成されたビジネスフローからの収益を増やす
  • 視聴者のロイヤルティを高める
  • 会場での保護とセキュリティのレベルを強化する
  • 会場での金融取引の高度な透明性の達成
  • パートナーブランドのイベント価値の向上


Allynis Smart Event Platformは、次の機能を含むNFCベースのウェアラブルデバイステクノロジーおよびセキュアサービスプラットフォームです。

  • 識別、チケット販売、アクセス制御
  • プリペイド支払いソリューション
  • トランスポートオペレーターアプリケーション
  • ソーシャルネットワークとのインターフェイス（コンテスト、投稿など）
  • リアルタイムのデータ収集と分析
  • モバイルマーケティング


このソリューションは、イベントの主催者、スタジアムの所有者、およびスポンサーに最適です。次のおかげで、座席（訪問者）あたりの収益を増やすのに役立ちます。

  • 訪問時間の延長;
  • 制御を加速します。
  • NFCの高速支払い（約30％高速）;
  • パーソナライズされたロイヤルティプログラム。
  • クライアントフローのダイナミクスと訪問者の行動パターンの理解を改善します。
