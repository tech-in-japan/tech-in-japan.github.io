Windows Phone用のアプリケーションを10分で作成する方法は？

appmakrとは何ですか？





appmakrは、Windows Phoneなどのさまざまなプラットフォーム向けのモバイルアプリケーションを作成するためのオンラインサービスです。 このサービスの特徴は、アプリケーションを作成するために何もプログラムする必要がないことです-これは、指定したパラメーター（たとえば、RSSフィード形式のニュースソース）に基づいてアプリケーションを生成できる特別なデータ消費シナリオに合わせて調整されています。





どのように機能しますか？



まず、登録する必要があります。 その後、コントロールパネル （ダッシュボード）にアクセスできます。 次に、新しいアプリケーションの作成を選択します。







Windows Phone用のアプリケーションを作成するには、適切なオプションを選択します。





次の画面で、データのソース（RSS / Atomまたは対応するフィードへのリンクがすぐにあるサイト）を指定します-私たちのアプリケーションを作成します

学生ブログ ：







次に、基本設定を行う必要があります：アプリケーションの名前を指定し、提案された画像の数（検索を指定できます）、参照またはディスクからアイコンとロード画面を選択します。







右側では、アプリケーションが電話エミュレーターでどのように見えるかすぐに確認できます。







次に、コンテンツの設定に移動します（[タブ]タブ）。 ここで、コンテンツのソースを調整および追加できます（投稿およびコメントのRSS、およびImagine CupのtwitterのRSSフィードを追加します）。







次の項目は、アプリケーションのカスタマイズです。 ここで、アプリケーションヘッダーを変更したり、色を変更したりできます。







次に、[公開]タブで説明を追加し、アプリケーションを生成します。







その後、アプリケーションは生成のためにキューに入れられます-アプリケーションがアセンブルされるまでしばらく待ってから、すべてのユーザーのマーケットプレイスに配置するだけです！



appmakrによって生成される完成したファイルは、xapファイルに名前を変更する必要がある通常のzipアーカイブです。







必要に応じて、CSSとJavaScriptを使用してアプリケーションをさらにカスタマイズできます。



マーケットプレイスに公開



既製のxapファイルを手元に用意できたので、Marketplaceでアプリケーションを公開します 。 アプリケーションでの作業中にグラフィック要素の一部（アイコンやロード画面など）をすでに準備している可能性がありますが、ストアでの公開にはまだいくつかの要素が必要です。

学生（および大学院生）は、 DreamSparkプログラムを通じて無料でマーケットプレイスに登録できます。 また、10月20日まで、Softkeyには無料のアプリケーションを投稿する特別なプロモーションがあります。







グラフィックオブジェクトの完全なセット（すべてが必須というわけではありません）は次のようになります。幅が99、173、200pxの正方形のアイコン、1000x800pxの背景画像、およびアプリケーションのスクリーンショットのセット。 スクリーンショットを撮るための組み込みエミュレータツールを使用してスクリーンショットを撮ると便利です。



必要なすべての説明フィールドに入力し、価格を設定（たとえば、アプリケーションを無料にする）、市場を選択します（アプリケーションにロシア語のデータのみが含まれている場合は、ロシア市場で公開する必要があります。確認のためにアプリケーションを送信します。



数日中に、アプリケーションはアプリケーションストアで利用できるようになります 。
