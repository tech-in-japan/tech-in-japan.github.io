無料版の新しい.NET Forge CMS 6.0サイト管理システム





.NET Forge CMSは、.NETテクノロジーを熟知しているWeb開発者向けの新しいサイト管理システムです。 この新製品には、コミュニティサイト用の無料版と、1Cと統合されたオンラインストア付きの商用版が含まれています。 .NET Forge CMSは、十分に開発されたORMツール、MVCアーキテクチャ、生産性の高い大規模プロジェクトでのテストを備えた.NET開発者向けのプロフェッショナルプラットフォームです。



「便利なHermitageインターフェースを含む.NET Forge CMS製品の登場を歓迎します。これは、数千のWebスタジオで実証されています。 ASP.NET開発者コミュニティが一般的な商用システムの無料版を利用できるようになったことを特に嬉しく思います。 過去1年間でASP.NETへの関心が高まっていることを考えると、.NET Forge CMSはこの関心を満たすタイミングで登場しました」とマイクロソフトロシアのWebテクノロジー部長Gaidar Magdanurovはコメントしました。



.NET Forge CMS 6.0には、コミュニティとビジネスの2つのエディションが含まれています。 無料のコミュニティ版では、最新のコミュニティサイト、ブログ、個人サイト、その他のWebプロジェクトをすばやく作成して、プラットフォームを節約できます。 Business Editionは、スケーリング、フォールトトレランス、セキュリティが重要なオンラインストア、高負荷サイトを作成するための商用システムです。



.NET Forge CMS 6.0はHermitageインターフェースを統合し、マルチサイトテクノロジー、マネージドキャッシング、オンラインストアのSKUシステムを実装し、新しいライセンスポリシーを適用します。



.NET Forge CMSはHermitageインターフェイスの概念を実装しており、1C-Bitrix：Site Managementで1年間使用されています。 Hermitageは、高度な管理インターフェイスとWeb開発者向けの推奨事項の「パッケージ」を組み合わせることで、その成果を実証しました。開発者は、その後、高速、安全、便利で管理しやすいWebプロジェクトを作成します。







.NET Forge CMS 6.0では、異なるドメイン名で同じコピー上に複数のサイトを作成するマルチサイトテクノロジーが導入されました。 すべてのサイトは1つのアプリケーション内で機能します。 マルチサイトは共有ホスティングでも機能します。



.NET Forge CMS 6.0に実装されているマネージドキャッシング（キャッシュの依存関係）の技術は、変更後すぐにデータを自動的に更新するために使用されます。 このテクノロジーを使用すると、キャッシュの更新を待つことなくサイトの変更を表示できます。キャッシュの更新は、事前に決められた期間にシステムによって実行されます。 管理されたキャッシングは、サイトでのユーザーの便利な作業の最も重要な技術コンポーネントの1つです。



この製品には、オンラインストアでSKU（Stock Keeping Unit）を使用するためのサポートが組み込まれています。 プロパティまたは特性（シャツの色、服のサイズ、車両装備など）に応じて、1つの製品に異なる価格オファーを追加できるようになりました。



.NET Forge CMSは、製品「1C-Bitrix：ASP.NETサイト管理」に基づいています。 「1C-Bitrix：ASP.NET Webサイト管理」の開発が完了しました。 .NET Forge CMSは、独立したスタンドアロン製品として開発しています。



アクティブなテクニカルサポートを持つ商用ライセンス「1C-Bitrix：ASP.NETサイト管理」の所有者はすべて、SiteUpdate更新テクノロジーを使用して、無料で「Business Edition」.NET Forge CMS 6.0にアクセスできます。



Webサイトhttp://www.bitrix.net/en/download/から無料のCommunityエディションと商用版の試用版をダウンロードできます。



.NET Forge CMSの詳細については、 http： //www.bitrix.netをご覧ください 。
