Ubuntu 9.10 Karmic Koalaのネットワークの問題

新しいUbuntu「Karmic Koala」にアップグレードした後、多くはネットワークに問題がありました。特に、多くはNetwork Managerで破損したdsl接続に遭遇しました。 障害は、推測するのが難しくないため、Ubuntu 9.10の一部である新しいネットワークマネージャーです。 再び機能するネットワークを取得するには、launchpad.netリポジトリからNetwork Managerの「デイリービルド」をインストールする必要があります



新しいリポジトリを/etc/apt/sources.listに追加します 

  〜$ sudo -i
 〜＃echo "deb http://ppa.launchpad.net/network-manager/trunk/ubuntu karmic main" >> /etc/apt/sources.list


次に、必要なパッケージを更新します 

  〜$ sudo apt-get update
 〜$ sudo apt-get upgrade


そしてそれはそれです、ネットワークは動作します:)



一時的に動作するインターネットを取得するには、次の行を/ etc / network / interfacesに追加します

auto eth0

iface eth0 inet dhcp





そして、pppoeconfコンソールコンフィギュレーターを使用します。 ファイル/ etc / network / interfacesを更新した後、元の形式に戻ることを忘れないでください。 のみを含む必要があります

auto lo

iface lo inet loopback
