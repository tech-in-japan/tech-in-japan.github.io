振り返ってみる：Microsoft Edgeの1年間



Microsoft Edge Web Summit 2016カンファレンスでは、Edgeチームのエンジニアが新しいブラウザーの開発経験を共有し、サポートされる新しい標準と将来の計画について話しました。



また、会議では、Microsoft EdgeとそのWebプラットフォームの開発をコミュニティに対してより透明にする多くのツールが発表されました。

  • プラットフォームの問題は、バグを報告し、EdgeHTMLの既知の問題について学習するための新しい場所です（記事「 Microsoft Edgeプラットフォームの問題追跡ツールの紹介 」も参照してください）。
  • プラットフォームデータは、CSSプロパティをグローバルに使用するためのAPIおよびテレメトリカタログです（記事「 Microsoft Edge DevでのグローバルCSSプロパティの使用の紹介 」も参照してください）。
  • OSXおよびLinuxでMicrosoft Edgeをテストするための更新された仮想マシン
