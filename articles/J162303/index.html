2つの画面を備えた「ロシア語」のスマートフォンは引き続き光を見る



それでもヨータは、2年前にメドベージェフ大統領との会議でディスクが公開された2つの画面を備えた有名なスマートフォンのリリースを確認しました（ Khabrに関する記事 ）。



Engadgetで公開されたプレスリリースによると、このデバイスには以下が搭載されています。

  • 対角4.3インチの2つのスクリーン（そのうちの1つはE-ink）
  • 1280x720 LCDスクリーン解像度
  • 1.5 GHz Snapdragon S4デュアルコアプロセッサ
  • 2 GB RAM、32または64 GBの非拡張フラッシュメモリ
  • もちろん、LTEのサポート（GSM / UMTS / LTE）
  • メインカメラ：12メガピクセル、前面：HD
  • バッテリー：2100 mAh


エキゾチックから：

  • 電話制御用のゾーンではない追加のセンサー（拡張容量性領域のジェスチャー）
  • ワイヤレス充電
  • 電源ボタンと組み合わせたSIMカードスロット
  • Eink画面のデータは、オープンAPIを使用して表示できます


現時点では、Android 4.2の最新バージョンも約束されています。





この奇跡は2013年の第3四半期に販売される予定です。

最初に、 類似の （ saguroを介した）デバイスのプロトタイプが、不明なメーカーによるIFA展示会で展示されました。 しかし、写真では、デバイスのOSはWin Mobile Windows Phoneに似ていますか、それとも画面が接着された空白ですか？





このデバイスは、美しい広告のレンダリングに最適です。

しかし、多くの質問と疑問もあります。 リリース時までに、Androidのバージョンは最後ではなく、デュアルコアプロセッサでさえほとんどすべてのスマートフォンにすでにインストールされています。

Eink画面の完全な機能は理解不能です（すべてに触れるのではなく、部分的にしか行われないようです）
