NetWrixの新しいフリーウェアソフトウェア





数か月前に、変更の監視とアクセス制御のための無料ソフトウェアの概要を公​​開しました。 それ以来、完全に無料のプログラムの新しいバージョンが登場しています。 おそらく、彼らはあなたの仕事であなたのために役に立つでしょう。



カットの下で、3つの新しいNetWrixフリーウェアについて少しお話しします

  • 特権アカウントを管理するには、
  • サービスの監視と再起動
  • 空きディスク容量を監視します。






1. 特権アカウントマネージャー

NetWrix特権アカウントマネージャー（PAM）は、Active Directory、サーバー、および他のシステムの特権アカウントを維持および保護します。 管理アカウントへのアクセスと自動メンテナンスのために、安全なWebポータルが使用されます-組織のすべての特権アカウントの管理と監査が集中化されます。



PAMは、すべての特権アカウントへの安全な役割ベースのアクセスを提供します。すべての許可されたユーザーは、作業前にパスワードを「受信」し、操作の最後に再びシステムに「降伏」する必要があります。 すべての操作は監視されており、いつでも、いつ、どこで、何にアクセスできる完全な監査ログを使用して、一般的な管理ユーザー（たとえば、「管理者」または「ルート」）の実際のID（従業員名）を確立できます。 さらに、指定された間隔（たとえば90日ごと）で、PAMは特権アカウントのパスワードを自動的に変更し、それらをターゲットシステムと同期します（たとえば、サービスおよびスケジュールされたタスクのパスワードを更新します）。





2. ディスク容量モニター

NetWrix Disk Space Monitorは、システム管理者がドメインコントローラー、ファイル、Exchange、およびSharePointサーバーとデータベースサーバーの空き領域を監視できる無料のツールです。 このプログラムを使用すると、指定したサーバーの空き領域を集中的に監視し、空き領域に関する日次レポートを送信できます。 また、空き領域の重要な値に達すると、プログラムは管理者に電子メールで通知します。 このプログラムを使用すると、無制限の数のサーバー上の空き領域を追跡できます。





3. サービスモニター

NetWrix Service Monitorは、重要なWindowsサービスを監視するための無料のツールで、障害発生後にサービスを再起動することもできます。 このプログラムは、サーバーのグループで自動的に起動されたすべてのサービスを監視し、1つ以上のサーバーに突然障害が発生した場合、電子メール通知を送信します。 再起動機能により、すべての制御されたサービスがダウンタイムなしで動作することが保証されます。 軽量プログラムは簡単に構成できます。インストール、コンピューター名の入力、目的の電子メールアドレスの指定。



これらは、制限や制限なしで管理用に無料で使用できる3つのフル機能プログラムです。



また、4月に、ファイルサーバーNetWrix File Server Change Reporterを監査するためのプログラムを更新しました。 プログラムの無料版も更新されました-フォールトトレラントサーバーのサポートがバージョン3.3で追加されました。
