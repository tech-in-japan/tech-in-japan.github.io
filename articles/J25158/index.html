文字体裁のキーボードレイアウト

労働者の要請に応じて、「活版印刷」キーボードレイアウトのトピックに触れます。 つまり、Windowsシステムツールと、キーボードを使用してさまざまな特殊文字（通常、右Altキー、別名AltGr）を入力できるサードパーティプログラムの両方です。



ほとんどはIlya Birmanのレイアウトを思い出しますが、コンテンツマネージャーやデザイナーのニーズには適していません。 そのメリットは、このようなレイアウトを作成する方法がRunetに示されたことです。



また、最初に、私の意見と同僚の意見では、「自動タイポグラファー」を使用するよりも、特定のスキルを持つ「タイポグラフィ」キーボードレイアウト（1〜2週間で得られます）を使用すると、力のコストと入力したテキストの品質の点ではるかに効果的であると言いますレベデフスキーなど。 問題は、オートタイポグラファーがテキストのタイポグラフィに完全に対応できないことです。 そして、私たちの実践が示すように、入力したテキストのエラーを検索することは、最初からテキストを入力するよりも困難です。 もちろん、これは、長いタイポグラフィのテキストを印刷する必要がある場合、オートタイポグラフが比較的うまく機能し、手動印刷のコストが非常に高い場合にはあまり当てはまりません。



記事では、「タイポグラフィ」レイアウトを作成および使用するためのよく知られたソリューションをリストし、最後に私たちと同僚が自分自身を使用するものについて話します。



最後の注意事項：完璧なレイアウトはありません。 プログラマー-プログラマー、HTMLレイアウト-レイアウト、コンテンツマネージャー-コンテンツマネージャー：誰もが特定のタスクを解決する必要があります。 主に最後のタスクを解決する必要があるため、コンテンツマネージャーの観点から特定のメソッドまたはレイアウトの利便性について話すことができます。 おそらく、これらの方法はプログラマーやレイアウト設計者には最適ではありません。

システムキーボードのレイアウト



システムにインストールできるすべてのキーボードレイアウトは、無料のMicrosoft Keyboard Layout Creatorを使用して作成されます。



利点：
  • レイアウトを使用するには、追加のソフトウェアは必要ありません。
  • レイアウトを作成するためのプログラムは無料です。
  • 最新バージョンの1.4は、Windows Vistaのレイアウトを行う方法を既に知っています。
短所：
  • 管理者権限がないと、システムにレイアウトをインストールできません。
  • 残念ながら、MSKLCにはまだ多くのバグがあり、デッドキー（修飾キー）を使用する場合にはバグがあります。
  • プログラムのショートカットが「Alt + Ctrl + 何か 」に割り当てられている場合、対応する文字は機能しません。

サードパーティプログラム



AutoHotKey



AutoHotKeyは、システム内でキーストロークを再定義し、あらゆる種類のアクションを実行できる、非常に強力なスクリプトプログラムです。 キーボードレイアウトに既製のソリューションを使用することも、独自のソリューションを作成することもできます（将来的に、私たちは現在、同僚と一緒にいると言います）。



利点：
  • プログラムはGNU GPLの下で配布されます。
  • プログラムを使用するには、管理者権限は必要ありません。
  • Windows Vistaで動作します。
  • プログラムは移植可能です-フラッシュドライブから起動できます。
  • スクリプトを自分用にカスタマイズできます。機能は通常のシステムレイアウトよりもはるかに強力です（たとえば、キーを押して一連の文字を表示できます）。
  • アプリケーションのレイアウトまたはショートカットの優先度を個別に設定できます。
  • 軽量（プロセスは最大5〜6 MBを必要とし、通常は3 MB以下）。
  • サポートフォーラムの印象的な英語コミュニティ。
短所：
  • プログラムは、システムで手動で起動するか、自動実行に登録する必要があります。
  • プログラムを自分用に構成するには、スクリプト言語を一生懸命勉強する必要があります。

Windows向けのソリューション

コンテンツマネージャー専用のソリューションを探していたため、Windowsでシステムレイアウトを使用することから始めました。 上記のすべてのレイアウトは、必要な文字の一部が含まれていないという点で不便でした（たとえば、 細いスペース 、マイナス、分、秒の記号）。 また、主に金融関連のウェブサイトで作業しているため、さまざまな通貨の記号が必要でした。



その結果、 Scrypeレイアウト （ Windows XPおよびWindows Vistaのバージョン）が誕生しました 。 ロシア語の活版印刷で受け入れられているすべての文字、矢印の記号「↑←↓→」、最も一般的な世界通貨の記号、およびウクライナとベラルーシのアルファベットの文字が含まれています。



残念ながら、美しい絵を描くにはまだ手が届きませんが、一般的にレイアウトはArtyom Sapeginのレイアウトに基づいていますが 、多くの変更があります：
  • 「AltGr +-」-長いダッシュ、「AltGr + +」-マイナス、「AltGr + Shift +-」-ハイフン、「AltGr + Shift + +」-プラスまたはマイナス。
  • 「AltGr +スペース」は理解できないスペースであり、「AltGr + Shift +スペース」は細いスペースです。
  • 「AltGr +」はアクセント記号です。
  • 上矢印、左矢印、右矢印、下矢印は、「AltGr + q、a、s、z」を使用して入力します。
  • 「AltGr + 8」は乗算記号×です。 ただし、AltGr + xを使用したBirmanバージョンも機能します。
  • 「AltGr + m」は商標マークです。
  • 主な通貨はAltGrにあります：？ -「AltGr + e」、¥-「AltGr + y」、£-「AltGr + f」。
  • 世界通貨の最も一般的な記号は、「AltGr + Shift」を使用してダイヤルされます。ウォン-「w」、ユーロ-「e」、円-「y」、ポンド-「f」および「p」、ルピー-「r」、シェケル- 「S」、ドン-「d」、グリブナ-「g」、バーツ-「z」、セント-「c」。 上記のすべての組み合わせは、ロシア語と英語の両方のレイアウトで機能します。
  • ロシア語のレイアウトでは、ウクライナ語type（「AltGr + e」）、ґ（「AltGr + g」）、ї（「AltGr + j」）、і（「AltGr + j」）、およびベラルーシ語 "（「AltGr + v」） ）、および大文字のバリアントЄ、Ґ、Ї、I、Ўから「AltGr + Shift」まで。
  • 直接および中括弧を入力するために、英語のレイアウトに切り替える必要はありません。角括弧の場合は「AltGr + x、b」、中括弧の場合は「AltGr + Shift + x、b」の組み合わせが機能します。
  • 最も単純な分数は 、修飾キーを使用して実装されます。 したがって、記号を入力するには、最初に「AltGr + 5」を押してから離し、次に「8」を押す必要があります。
