Android 1.5（カップケーキ）



Googleはついに待望のAndroid OSアップデートをバージョン1.5にリリースしました



ユーザーインターフェイスの強化：



  • システム全体：

    • すべての組み込みインターフェイス要素の改善
    • アニメーションウィンドウの切り替え（デフォルトではオフ）
    • 加速度計の読み取り値に基づいた自動画面回転


  • 研磨インターフェース：

    • 着信コール
    • ノートブック、最近の通話、お気に入り
    • SMSおよびMMS
    • ブラウザ
    • Gmailの組み込みクライアント
    • カレンダー
    • 郵便
    • カメラとフォトギャラリー
    • アプリケーション管理




パフォーマンスの改善：



  • より速いカメラ起動とより速い撮影
  • 著しく高速なGPS測位
  • ブラウザのスムーズなスクロール
  • Gmailクライアントでのメッセージの高速スクロール




イノベーション：



  • スクリーンキーボード

    • 横長モードと縦長モードの両方で動作します
    • サードパーティのソフトキーボードのユーザーインストールをサポート
    • カスタム辞書


  • 作業台

    • ウィジェット（基本を含む：アナログ時計、カレンダー、音楽プレーヤー、画像、検索）
    • ライブフォルダー（RSSや電子メールなどの動的フィードを表示するためのウィジェット）


  • カメラとフォトギャラリー

    • ビデオ録画
    • ビデオ再生（MPEG-4および3GP形式）


  • Bluetooth

    • ステレオサポート（A2DPおよびAVCRPプロファイル）
    • 自動ペアリング
    • ハンズフリーの改善


  • ブラウザ

    • WebkitエンジンとSquirrelfish Javascriptエンジンを最新バージョンに更新
    • ブラウザでコピーアンドペースト
    • ページで検索
    • ページのエンコードをカスタマイズする機能
    • インターフェースの改善：検索文字列をURL入力文字列と組み合わせます。 ブックマーク/履歴/よくアクセスするページ画面


  • ノートブック

    • お気に入りのカスタム画像
    • 最近の呼び出しリスト内のイベントの日時を指定する
    • 最近の通話のリストから連絡先カードへのワンタッチアクセス


  • システム

    • 更新されたLinuxカーネル（バージョン2.6.27）
    • SDカード上のファイルシステムの自動チェックと復元
    • SIMカード上のオペレーターアプリケーションのサポート（いわゆるSIMメニュー）


  • Googleアプリ

    • ノートブック、SMS、MMS、GMail、メールクライアントでGoogleトークの友人のステータスを表示する
    • Gmailでメッセージを一括してアーカイブ、削除、タグ付けする機能
    • Youtubeに動画をアップロードする
    • Picasaに写真をアップロードする






このアップデートは、Android Dev Phoneの幸せな所有者に無料でダウンロードできます 。 他の皆は、今週の終わり（2009年5月17日）に始まる（T-Mobileフォーラムの噂による）自動「空気」アップデートを辛抱強く待っています。 これは、おそらく1月以降、G1のすべての所有者が楽しみにしていることです。
