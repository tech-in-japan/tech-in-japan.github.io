FireFoxを使用してサイトの広告をブロックする

広告は良いです。 これは進歩の原動力であり、サイトの作者にお金を稼ぐ方法であり、訪問者が自分にとって興味深い製品を見つける機会になることもあります。

しかし、多くの広告があるとき、それはすでに悪いです、そして、あなたはそれと戦わなければなりません。



最近、私が興味を持ついくつかのサイトにあまりにも多くの広告が掲載されており、それで十分だと判断しました。 彼女と戦う時です。

過去半年間、FireFoxがメインのブラウザになったため、FireFoxで広告をブロックすることに困惑することにしました。



最初のステップは非常にシンプルで明白です-AdBlock Plus 1.0プラグインをインストールします。

このプラグインを使用すると、URLによる外部ページ要​​素（スクリプト、フレーム、画像など）の読み込みをブロックできます。

ブラックリストにマスク「ad.adriver.ru」を追加すると、それだけです。AdRiverの広告は表示されなくなります。



そして、すべてがうまくいきますが、一部のサービスは、同じアドレス空間に情報要素（デザインの一部である写真）と広告をブロックして配置するこの方法に非常に精通しています。 このようなサービスの実例は、mail.ruからのWEBメ​​ールです。AdBlockを使用すると、広告の一部のみをブロックすることができます。



これが第2段階の始まりです-ページ内のコンテンツの動的置換。

プラグインの短い検索により、素晴らしいプラグインFoxReplace 0.9.0が見つかりました 。



一方では、このプラグインの機能はめちゃくちゃ小さいように見えるかもしれません...しかし、私は十分です。

たとえば、win.mail.ruの一番上の広告ユニットを削除するには、次の設定を使用しました。

1.トップバナーブロックを非表示にします。

サイト：| http：//win.mail.ru/*

検索：<div id = "lite_top_banner">

replace：<div id = "lite_top_banner" style = "display：none;">



2.このデータブロックへの読み込みを禁止します。

サイト：| http：//win.mail.ru/*

検索：

置換：ここには何も入力しないでください



ps当然、すべての広告をブロックする必要はありません-これはサイトの作者の財布に悪影響を及ぼし、最終的にサイトのコンテンツの品質に影響を与える可能性があります。 しかし、ここでは、mail.ruなどのモンスターに干渉する広告をブロックすることは理にかなっています。



pps幸いなことに、ハブでは広告が非常にインテリジェントに作成され、作業に干渉しません。 したがって、私は家でそれをブロックしませんでした、著者と私に予算でかなりの金額をもらい、サイトをさらに良くします:)
