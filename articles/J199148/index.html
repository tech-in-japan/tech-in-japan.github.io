ノートパソコンのディスプレイから2台目のモニターを作成した方法







そのため、15.6インチのマトリックスとNTA92C VGA / DVI LCDコントローラーを搭載したHP 625ラップトップが手元にありました。通常のモニターをVESAに固定します。 「週末プロジェクト」の一環として、最小限の時間と金銭的費用で。
「週末プロジェクト」とは、非定型のタスクを解決することを目標に短期間（週末または1〜2週間以内）に行うことです。 主な目標：問題を解決し、経験とスキルを獲得し、「ツールボックス」を拡張すること。
ラップトップディスプレイをコンピューター（VGA / DVI / HDMI）に接続する機能は、さまざまな種類のLCDパネルコントローラーによって提供されます。 NTA92Cコントローラーは私のディスプレイに適していました（ebayやaliexpressなどで購入できます）

  • 乾式壁28x27の金属プロファイル

理論部



ディスプレイは、VESAマウント用にモニターの所定の位置に取り付けられ、耳のように横から突き出ます。 重量が軽くないため、モニターが斜めになります。 スキューを回避するには、一方でカウンターウェイトを追加する必要があります。









すべてを計量して測定した後、私はこれを得ました：







42cm * 920gram + 28cm * 210gram = 24cm * Xgram

X = 42cm * 920gram + 28cm * 210gram / 24cm = 1855グラム


これは、カウンターウェイトを収集するときに注目した図です。 カウンターウェイトの手元の材料から、コインを取りました。









実用部



ディスプレイ



ラップトップのディスプレイは、船体とヒンジと一緒に取り外されます（見た目が良いようです）。 ヒンジには、ラップトップの蓋を開くためのリミッターがあり、それが本当に気になります。なぜなら、今では140度以上の角度で開くからです。









リミッターはフレームの突起です。 私がしたように、あなたはそれを挽くか、ハンマーで打ち返すことができます。









さらに、LCDコントローラーのハウジングを電源アダプターのハウジングから作成しました（はい、電源アダプターを分解しました）。このすべてを接着剤でディスプレイの背面に取り付けました。 これで表示は終了しました。



フレーム









プロファイル28x27から正方形を収集します。 ディスプレイヒンジと60x27プロファイルが正方形の反対側に取り付けられているため、これが最初に必要です。 そして第二に、ボルトのねじの下の金属の厚さを増やすこと。 プロファイルはリベットの助けを借りて正方形に固定され、ボルトの下にネジ山が切られます。







プロファイル60x27では、VESAマウント用に2つの穴を作成し（VESA MIS-D、100、C-つまり、穴は互いに10 cm離れています）、以前に作成した正方形のリベットで固定します。



電源







電源装置のハウジングは、電源装置を60x27プロファイル内に後で配置するために分解されました（ハウジングに収まりませんでした）







私は段ボールから（断熱材として）住宅代替品を作り、銃でプロファイルに貼り付けました。 電源自体は、段ボールの角に接着されています。







その結果、私はそのような「シャベル」を手に入れました。



インストールと接続



VESAのボルト（左の家具から）でメインモニターに「シャベル」を固定します。 カウンターウェイトを取り付けます。 DVIと電源を接続します。





表示はすぐに決定されませんでした。 ここに行きます： コントロールパネル→画面→画面解像度設定→必要に応じて検索してカスタマイズします。 それだけです！ 2番目のモニターの準備ができました！







短所



  • あまり良い行列ではありません。 まず、このために光沢がありまぶしいです。 第二に、あまり良い視角ではありません（幸いなことに、ヒンジはディスプレイを望みの角度に回転させる能力を失いませんでした）。

  • ディスプレイの側面の比率（プライマリ1920x1080、2番目の768x1360）の違いにより、単一のディスプレイ効果はありません。 はい、解像度768x1024を使用しましたが、これは堅牢性の点でははるかに優れていますが、画質は私には合いませんでした（文字が少し浮いていました）。

  • 私のプロファイル設計はあまり良くありません ねじれに対する耐性が不十分です。ねじれは、その機能とそれに負荷をかける方法のために発生します。 この問題は、下部の2つのVESAマウント穴に2番目のプロファイルを取り付け、すべてをリベットで固定することで回避できます。
ちなみに、誰の妨害かわかりませんが、ディスプレイをXubuntuの下でSamsung R519ラップトップのVGA出力に接続すると、ディスプレイの色が非常に奇妙になりました。


結論



次の「プロジェクト」が完了し、最終結果が得られました。

マトリックスの価格での構造の総コストは1000〜1500、コントローラーは〜900ルーブル、コード付きの電源ユニットは〜400ルーブル、消耗品は50〜100ルーブルは1950〜2500ルーブルです。 お金のために（まあ、少し追加するかもしれませんが）、完全に正常なセカンドモニターを購入できます。 私はほとんど何もせずに2番目のディスプレイを手に入れました（小さなドリルで25ルーブルだけを費やし、150ルーブルから200ルーブルのコインを使いました）。 とにかく、残りは何らかの方法でした。 必要なのは、テーブルの引き出しを掘り下げることだけでした。



もう一度やらなければならない場合は、アルミニウムの角または正方形を使用します。
