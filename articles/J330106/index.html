モニタリングおよびデータベース管理用の20の無料ユーティリティと89のスクリプト

あなたは座る-ベースが成長し、あなたが行く-ベースが成長し、眠り、食べ、またはより多くのことをし、ベースが成長し、成長する。 私たちの人生で唯一のものは一定であると非常に賢い人が言った。 主なことは、それらに正しく反応することです。 データベースの異常な状況は、そのような変更が原因で発生します。 残念ながら、金で刺繍され、60カラットの宝石がちりばめられた産業用監視システムに投資することは常に可能とは限りません。 そしてこれにより、無料またはシェアウェアのソリューションに頼ることができます。 この記事では、IDERAから20の無料ユーティリティを収集しました（これだけではありません）。これは、MS SQL、MySQL、Oracleの監視と管理に関するいくつかの問題を解決するのに役立ちます。 行こう！







＃1 SQLチェック

このユーティリティを使用すると、SQLサーバーのパフォーマンスを制御できます。 データベースサーバーへのエージェントのインストールや、データベース自体への特別なオブジェクトの直接注入は必要ありません。





＃2 SQLフラグメンテーションアナライザー

このユーティリティは、断片化されたテーブルを見つけるのに役立ち、デフラグを実行する必要があるときに警告します。







＃3 SQL XEventプロファイラー

セッションと実行可能なSQLステートメントを表示します。 追加の利点-イベントのグループ化、並べ替え、エクスポート。







＃4 SQLヒートマップ

データベース内のすべてのテーブルのコンテキストでのディスク領域の使用の視覚的表現。 将来的に必要になる可能性のある容量の管理とリソースの予測に役立ちます。 追加のディスク容量を必要とするテーブルとインデックスをすばやく識別します。







＃5 SQLページビューアー

データベースオブジェクトとイベントの便利な表示と管理。







＃6 SQL更新統計

テーブルとインデックスの古い統計の定義、統計オプションの表示と変更。







＃7 SQL Statisticsアグリゲーター

問題のあるテーブルを識別するためのIO統計の集計、ディスクアクティビティと実行計画の相関。







＃8 SQL Hekatonメモリチェック

特定のデータベーステーブルに対するメモリ使用率の影響の監視。







＃9 SQLインスタンスチェック

データ収集とデータベースインベントリ。データベースにインストールされた更新プログラムの関連性を確認します。







＃10 SQL BIチェック

SSASのリアルタイム監視、CPU、メモリ、ディスク消費の表示と分析。







＃11 SQL Query Store Optimizer

クエリストアサービスのパフォーマンスの向上。







＃12 MySQL Query Explorer

リソースを集中的に使用するMySQLクエリの上部を監視し、データベースをチューニングするためのクエリ実行の統計を表示します。







＃13 SQLバックアップステータスレポーター

バックアップのないデータベースの識別、バックアップの履歴の表示。







＃14 SQL整合性チェック

オンデマンドでSQLデータベースのCHECKDBを実行できます。







＃15 SQLジョブマネージャー

現在および過去のジョブを表示します。







＃16 PowerShell Plus

PowerShellスクリプトを操作するための特別なIDE。







＃17 Rapid Database Extractor

SQLおよびOracleのデータソースの管理、データベースオブジェクトの操作、SQLクエリの実行によるデータの取得。







＃18 SQL Permissions Extractor

ユーザー権限をコピーするためのT-SQLスクリプトを生成できます。







＃19 SQL列検索
