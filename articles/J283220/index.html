データ転送：素晴らしい速度と新しいテクニック



光ファイバを介して送信される光信号の図（ c ）



インターネットが常に必要とするものを知っていますか？ より高いスループットで。 自分で判断してください：数百ギガバイトの4K映画による「ホーム」革命が進行中です。 ビデオコンテンツの波の背後で、またはそれと並行して、仮想現実が発達します。 スピードの成長だけが研究分野ではありません。 今日は、データ転送の方法について説明します。データ転送の方法は、これまでのところ実際の空想科学小説のように見えますが、近い将来実装することができます。



古き良きケーブル



2014年にデンマーク工科大学の研究者は、単一の光ファイバーケーブルで43 Tbpsの速度でデータを送信しました。 科学者は 、いくつかのコアを持つファイバーを使用しました。 それにもかかわらず、2011年には2つの方法で100 Tbit / sの速度が達成されました。それぞれが15.6 Tbit / sの速度を提供する7本の別々のファイバーで構成される光ファイバーコアを介して。 2番目のケースでは、異なるスペクトルのレーザービームを使用してデータを「パック」し、放射の振幅と位相を変えて、合計370の個別のビームを使用しました。







オランダとアメリカの科学者の科学チームは、すべてを1本のファイバーケーブルから絞り出そうとし、255キロビット/秒の速度で1キロメートルごとにデータを転送しました。 研究者は、7芯の光ファイバーを使用しました（上の写真の左側に示されています）。 通常の繊維との違いは何ですか？ ここでは簡単です。ここでは、光は各コアを他のコアとは無関係に通過します。 写真の黒い点は、コアを互いに分離する空気層です。 このテクノロジーを使用して、2012年9月に、52.4 kmの距離で12個のライトガイドチャネルを備えたケーブルで1秒あたり1ペタビットの実際の伝送速度を達成することができました。



データ電球



数年前、Li-Fi光データ伝送技術（Light Fidelityの略）が登場しました。この技術では、通常の電球と区別できないトランシーバーが非常に高い周波数で点滅し、光束とともに情報を伝送します。 Li-Fiでの情報の送受信は、約150 Mbit / sの速度で実行されます。 しかし、レーザーLEDに基づいたLi-Fiの技術は、最大100ギガビット/秒の情報転送速度を向上させることができます。 このようなデバイスにはマイナスが1つしかありません。部屋の外では機能しません。



pCell：神話か現実か？







PCellテクノロジーは、スペクトルの周波数不足の問題を解決し、輻輳の程度に関係なく、各モバイルデバイスが基地局が提供する全周波数帯域で動作できるようにします。 アメリカのスタートアップArtemis Networksのプロジェクトの著者によると、pCellは、同じ周波数範囲と既存のスマートフォンを使用して、現在の4G LTEネットワークの50倍の速度を提供します。 これは可能ですか？ 昨年末、ノキアネットワークスがpCellテクノロジーのテストを開始することが明らかになりました。 調査結果の公開を待つだけです。



LLCD、またはなぜレーザーが必要なのか







2013年には、 LADEEプローブの打ち上げに成功し、その上に月面レーザー通信デモンストレーション（ LLCD ）双方向レーザー通信システムが設置されました。 その結果、地球から385,000 kmの距離にあるデバイスから地上局への622 Mbit / sおよび地上局からデバイスへの20 Mbit / sのデータ転送速度を達成することができました。



地球上で同様のレーザー通信を使用することは可能ですか？ もちろん。 2011年に、50 kmの距離で26 Tbpsの驚異的な速度で指向性レーザービームでデータを送信する実験の結果が公開されました。 そして、困難は、ビームが方向付けられるべきであるという事実から始まります。 都市部では、この制限により速度の利点がほとんどなくなります。



脳から脳へ



情報を送信するための最も素晴らしい方法の1つは、単純な仮定に基づいています。最終的に、ある人と別の人との間の情報交換について話している場合、追加のチャネルが必要な理由です。 しかし、すぐに2つの脳を接続したらどうなるでしょうか？



デューク大学の研究室の神経生理学者は、3匹のサルの脳を 「ローカルネットワーク」に結合しました。 サルは、神経インターフェースと運動皮質に埋め込まれた700個の電極を使用して、コンピューター画面上の仮想アームの動きを制御しました。 各動物は、腕の動きの3つの軸のうちの2つ（XおよびY、XおよびZ、YおよびZ）を制御する責任がありました。



ボディコンダクター



電気通信会社のNTTドコモは、小さな送信機が人間の皮膚の周りに電磁界を生成するヒューマンエリアネットワーキング技術を開発しました。 皮膚の別の部分で信号を取得するために、約10 Mbit / sの速度で信号を受信する高感度の光学センサーが使用されます。 情報を自分自身に渡す人体により、興味深い技術的ソリューションを実装できます。 USBフラッシュドライブを手に取り、プリンターに触れて、印刷用のドキュメントを送信するというアイデアはどうですか？



宇宙全体を通して



国立加速器研究所の物理学者チームの名前は Enrico Fermi（米国）は2012年に、あらゆる物体を介してデータを転送する方法を提案しました 。ニュートリノは、物質と非常に弱く相互作用する基本粒子です。 ニュートリノのおかげで、科学者たちは240メートルの岩を介してデータを送信することができました。これは他のワイヤレステクノロジーではアクセスできない指標です。 ただし、データ転送速度はわずか0.1ビット/秒でしたが、将来、同様の方法を使用して遠方の空間オブジェクトと通信できます。



ウォッカを使用する



はい、電磁波の使用を妨げる環境での使用に適したデータ送信のこのような珍しい方法があります。 この方法は、揮発性化合物を使用して作業をシミュレートする分子システムに基づいています。 専門家は、ウォッカ蒸気を使用してバイナリコードを送信しました。蒸気の存在は1、不在は0です。メッセージは、机のファンを使用して、空気中のアルコール濃度を分析するセンサーに送信されました。



肉のコミュニケーション



イリノイ大学は、最大30 Mbpsの速度で動物組織を介して超音波信号を送信する実験を設定しました 。これは、肉を介したデータ転送の記録です。



この方法は、水中での信号伝送の技術に基づいていました。 5 MHzのセンサーを使用して、水が入ったタンク内の肉を介して信号を送信します。受信にはハイドロフォンがあり、20-30 Mbit / sの速度で信号を送信することができました。 将来、この技術は、患者の体から生放送するか、人間のインプラントソフトウェアを更新するのに役立ちます。



素晴らしいソビエトインターネット



パイプラインを使用してデータを転送するというアイデアは、ソビエト連邦で発明されました。 最新の技術により、この方法は大幅に改善されました。 Nethercomm は、 10年前に特許取得済みのBroadband-in-Gas （BiG）テクノロジーを使用して、最大10 Gb / sの速度でUWB（Ultra Wide Band）無線形式で信号を送信しました。 パイプラインネットワークでは、UWB帯域で送信される無線信号のリピーターのチェーンがマウントされます。 パイプ内にリピーターが設置されているため、この技術は水道管内では使用できません。 無線信号は、家庭用ガスの流れと並行して既存のパイプ内で送信されます。



上記のデータ転送方法は、近い将来世界を変えるでしょう。 大陸間の広いチャネルにより、すべてのデータセンターを1つの有利な場所に転送できるようになる可能性があります。たとえば、無料の冷却のために南極大陸に転送することができます。 遠くの宇宙オブジェクトとの通信は、高速で利用可能になります。 通常の有線インターネット（ただし、モバイルのような）は信じられないほどの速度に達します。



業界がどのように発展するかは確実にわかりません。 また、記載されているすべての方法が、今後10年間で新しい研究の背景から薄れていく可能性もあります。 いずれにせよ、時間は変化しています。つまり、他の信じられないほどのデータ転送速度があります。
