Mac OS X 10.5.7リリース

利用可能なMac OS Xアップデート：
  • Mac OS X 10.5.7アップデート （デルタアップデート、つまりバージョン10.5.6から10.5.7へのアップデート）;
  • Mac OS X 10.5.7。 コンボ更新 （バージョン10.5から10.5.7に更新）;
  • Mac OS X Server 10.5.7 Comboアップデート 。
私自身の使用経験から、各モデルに独自のデルタ更新アセンブリがあり、合計で729.2 MBを超えるコンボ更新があるため、自宅に複数のポピーがある場合は、コンボ更新をダウンロードすることをお勧めします。すべての車。 さらに、ソフトウェアアップデートで更新された場合、奇妙なことがあります。このプログラムは、新しくダウンロードされた更新の整合性をチェックしないように感じます。 突然何かがなければ、彼女はそれについて話すことを止めず、すぐにそれを置き始めます。 したがって、インストール後、コンピューターが更新されたシステムをダウンロードしたくないという点まで、あらゆる種類の問題が発生します。 このような場合の適切な解決策は、インストール前にサービスパックを正確にチェックするため、Apple WebサイトからダウンロードしたCombo Updateを再インストールすることです。 したがって、特に過去にSoftware Updateプログラムに問題があった場合は特に、Combo Updateをすぐにインストールすることをお勧めします。



このアップデートの新機能は何ですか？



誤解があったのは、 アップデート10.5.7の簡単な説明の Appleが誤って修正10.5.6の詳細なリストを参照しているためです。 問題に気づき指摘した丁寧な解説者に感謝します。 したがって、私は多少不器用な翻訳をすべて、 ロシア語版のAppleサイトの資料に置き換えます 。 興味のある方は英語版にも慣れることができます。



一般的な問題

  • 最新のセキュリティ修正が含まれています。
  • 一部のサードパーティ製カメラのRAW画像の追加サポートが含まれています。
  • NVIDIAグラフィックスカードを搭載した最新のMacコンピューターで、ビデオの再生品質とポインターの動きの明瞭さを改善します。
  • Mac OS X 10.5.6でのDvorakキーボードレイアウトの問題を修正します。
  • Converter、Stocks、Weather、MoviesなどのDashboardウィジェットの信頼性と精度を向上させます。
  • Gmailへのログインに問題を引き起こす可能性のある状況を修正します。
  • 連絡先をYahoo!と同期する際の信頼性が向上します。
  • Mac Pro用拡張スロットユーティリティは、正しいPCIeスロット構成を報告するようになりました。
  • フロー制御が有効になっている特定のイーサネットスイッチに接続すると、ネットワークのパフォーマンスが向上します。
  • Mac OS X Server 10.4がホストするネットワークホームディレクトリの安定性が向上しました。
  • Mac OS X Server 10.4、Time Capsule、サードパーティAFPサーバーなど、Spotlight検索をサポートしないネットワークボリュームのFinder検索結果を改善します。
  • Mac OS X Server 10.5.7 Update Informationで説明されているいくつかのディレクトリサービスとクライアント管理の拡張機能が含まれています 。

ICalカレンダー

  • CalDavにより全体的な信頼性が向上します。
  • MobileMeとの自動同期により信頼性が向上します。

メールクライアント

  • Notesで信頼性と同期の問題を解決します。
  • 送信済みメールボックスからメッセージをリダイレクトするときに、BCCフィールドに誤って入力する問題を解決します。

ペアレンタルコントロール

  • ペアレンタルコントロールとアプリケーションの制限との一貫性が向上します。
  • フルスクリーンゲームやクイックユーザースイッチ機能を使用する場合の、制限時間の誤った操作の問題を解決します。

印刷する

  • サードパーティのプリンターが間違った用紙トレイに印刷する可能性がある問題を解決します。
  • 保護者による制限を有効にし、[プリンターの管理]を選択することで、管理者以外のユーザーアカウントがプリンターを追加および削除できるようになりました。
  • 印刷に関するその他の信頼性と安定性の改善が含まれています。
