「tyzhprogramist」の抽出

最近、彼らは私にラップトップを持ってきました-非常に良いラップトップ-もちろん2004年の基準で（もちろん、そのようなことはあなたが殺したかもしれません ） そして、多くの人に馴染みのマントラで、「tyzhprogrammer * a la」は、何年もの間、安定した動作中のデバイスが突然起動を拒否した理由を解明するよう求められました。 そして一般的に、私は電子技術者ではなく、プログラマーでもありません。そのため、R / Matlab / Pythonに手を出しています。 しかし、私の友人はとても良かったので、私はまだラップトップを持っていかなければなりませんでした。



不運な車はブリーフケースから取り出されました。 彼女は悲惨で嘆かわしい状態でさえありました：体は粘着性の液体であふれていました（そしてそれは何年も前に思われます）。 キーボードのボタンは長年の重ね付けの重さで単純に押され、かつては心地よいマットスクリーンは大胆なプリント、斑点、傷でいっぱいでした。

感染の拡大を避けるために、アルコールを（外部から）使用することが決定されました。 ハウジングは完全にクリーニングされています。 ふたの上にプラスチックが現れました-手触りがとても心地よく、ゴム引きプラスチックに似ています。











バッテリーの容量が増えたことがわかり、充電器は猫に擦り切れていましたが、正常に機能しました。 キーボードと画面を重ねた後、ラップトップの外観は自然で非常にまともな状態に近づきました（そして、元のThinkpadの厳格なデザインが常に好きでした）。











Thinkpad T41の電源ボタンを押すと、クーラーがどれほど楽しく、同時にひどく割れたのかがわかりました。 きしむハードドライブが彼に加わりました-そして今、モーターのデュエット、パフと引き裂きが、その速度で参加しました。 Windows XP Professionalのスタートアップスクリーンセーバーにも登場しましたが、それはそうでした。 Memtest86は、512Mbのメモリがすべて正常であることを示しました。 次に、sysrescueCDをダウンロードしてbadblocksを起動しました-これはエラーメッセージが降る場所です（そして、はい、Access IBMボタンを押してもシステムが非表示セクションから起動しませんでした）。 最初の交換候補が概説されました。 2番目は冷却ファンであることが判明したため、ロード中に「ファンエラー」というメッセージが定期的に表示されました。 そこのサイトで、私はそれを修正し 、クーラーを交換するために、彼らはマトリックスの除去を含む41ステップからの指示を提供しました。 しかし、実際には、すべてがはるかにシンプルであることが判明しました。下のキーボードとタッチパッドを固定しているネジを外すだけで、ラジエーターにアクセスできます。 クーラー全体は、ほこり、羊毛、およびhair毛で詰まっていた-brrr。 しかし、掃除機はこのすべてに対処するのに役立ちました。

















収集プロセスは、反対方向の解析プロセスに似ていましたが、唯一の違いは、不要なネジがまだ2つあったことです。 負荷がかかっている間、心配しているテロダクティルの警告音を発し続けたため、洗浄手順はクーラーをあまり助けませんでした。



それからラップトップの所有者に電話して、新しいハードドライブ（2.5インチIDE Fujitsu 40Gb）と新しいクーラーが必要だと言いました。また、メモリを追加することをお勧めします。空きスロットがあります。私が喜んで答えたのは、もしそうなら、彼らはこれらの「fire」を必要とせず、私はそれらを自分自身に任せることができるということです。一般的に、新しいラップトップはすでに購入されています。



ラップトップは既に私のものであり、他人のものではないので、少し緊張することがあります。 ローカルバザールの2.5インチIDE HDDおよびSODIMM PC2700メモリの価格は喜ばれませんでしたが、必要なクーラーを完全に忘れることができました。状況は、 Aliexpressによって保存されました 。 PC2700、そしてThinkpad T41のクーラーでさえ、その価格は本当に恥ずかしいです。約2週間で来ました。私はハードドライブについて最も心配していましたが、うまくいきました：中国人はすべてを可能な限り最高の方法で送りました。























もちろん、ハードドライブとメモリのインストールに問題はありませんでしたが、ファンをいじる必要がありました。 古いクーラーは、ドリルでしか開けられないリベットでラジエーターに取り付けられていました。 そして、中国製の新しいクーラーはラジエーターの取り付け穴にぴったりと合っていなかったので、私はもう一度ドリルを使って集団農場をしなければなりませんでした-はい、古いソビエトのボルトで。











組み立て後、すべてが正確に15分間正常に機能しました。その後、クーラーの回転が停止しました。 私は、安い中国の工芸品は移植に耐えられず、死んだと思っていましたが、古いクーラーはオンにならず、ラップトップは疲れたファンエラーを出し続けたことが判明しました。 さらに、外部電源に接続した場合、両方のファンが正常に機能しました。 Googleのすべての答えは1つに集約されました。マザーボードを変更する必要があります。 しかし、私はそのようなターンの準備ができていませんでした。 しかし、 あるポーランドのフォーラムで、ヒューズが切れたことが原因である可能性があることがわかりました。 エレクトロニクス分野での私の最高の成果は、0-V-2スキームに従って組み立てられたレシーバーであり、マザーボードと同じスケールのコンポーネントをはんだ付けしたことはありませんでした 。 エレクトロニクスエンジニアが私のような人を呪うようなことをしました。ただ、溶断したヒューズを引き抜き、1滴のはんだで接点を閉じました。 手順の後、クーラーは喜んでハミングしました。







詳細をまとめて、DVDから32ビットLinux CrunchBang Linuxをインストールしました（このPentium-MはPAEをサポートしていないため、すべてのディストリビューションが機能するわけではありません）。 「自分のために」調整するための少しのファイルカービング-そして出来上がり！ -優れたタイピングラップトップが機能し、この構成では非常に適切に機能します。

  • ATI Mobility Radeon 7500
  • 14.1 "TFT 1024x768
  • 1GBのメモリ
  • 60 GB HDD
  • Intelギガビットイーサネット
  • CD-RW / DVD-ROMコンボドライブ
  • 56Kモデムを搭載したIBM Integrated Bluetooth II
  • IBM 11a / b / gワイヤレスLANミニPCIアダプター
