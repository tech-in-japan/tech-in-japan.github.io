「Great Eight Applications」のプロモーションの結果

今日は、Windows 8の8つのアプリケーションのプロモーションの結果を共有したいと思います。新年のアプリケーションコンテストの勝者が賞を受賞したことを思い出してください。



賞品には何が含まれていましたか？



この賞には、400万のオンラインインプレッション、 AdDuplexからの無料の広告インプレッション、microsoft.comのプロモーション、およびパートナーサイトが含まれます。



1.アプリケーションは、microsoft.comリソースのコンテストの特別ページに掲載されます。

2.各アプリケーションは最低400万インプレッションを受け取り、次のデジタル通信チャネルで宣伝されます。

-検索中（YandexおよびGoogle）;

-最も人気のあるVKontakteネットワーク。

-36の人気サイトのモバイルバージョン。

-Microsoft VKontakteのソーシャルリソースおよび55,000人以上のサブスクライバーとその友人のFacebook

3. Windows 8での特別なMicrosoftプロジェクトに関する言及：

-afisha.ru

-タイムアウト印刷

-Cosmo.ru

-Maximonline.ru

-Spletnik.ru

-Woman.ru

4. AdDuplex賞 -AdDuplexネットワークで250,000回の広告表示。



結果と結論



前のキャンペーンの調査結果に基づいて、プロモーションのチャネルの1つはソーシャルネットワーク-VKontakteでした。 この通信チャネルは、 以前のマイクロソフトプロモーションで効果的であることが示されています 。



VKontakteは、適切な数のアプリケーションのダウンロードではなく、最大のカバレッジを提供できます。



VKontakteの広告のスクリーンショット








アプリケーションの特定のダウンロードが必要で、Windowsストアページへの大きなリーチと移行が必要ない場合は、最も効果的なチャネルが必要です。 ダウンロードはモバイルデバイスでホストされます。 WapStartによるモバイルアプリケーションのプロモーションは、ソーシャルネットワークのプロモーションよりもはるかに大きな最終ダウンロードをもたらしました。 VKontakteアプリケーションの平均インストール価格は0.65ドルですが、Wapstartを使用した場合の平均インストール価格は0.15です。



モバイル広告のスクリーンショット










モバイルプレースメント形式では、非常に小さなバナー（「Windows Phone 8、7.5、および7」のターゲットで120x20、168x28、216x36、300x50を使用）で最大限の簡潔さと視覚的アピールが必要です。そしてその本質の理解可能性。



また、アプリケーションのダウンロードを増やすための不可欠なツールは検索であり、YandexはGoogleよりも効果的です（クリック率5ルーブル対8ルーブルで応答率は1.04％対0.83％です）。



GoogleとYandexの広告のスクリーンショット








検索に広告を配置するときは、検索フレーズの選択と広告のテキストの作成に注意を払う必要があります。

1）アプリケーションは「Windows 8 \ RT \ Windows Phoneのアプリケーション」というフレーズの下に配置する必要があるという事実に加えて、競争の少ないトピックでプレゼンスを最大化する必要があります。これは、料理、レシピなどの情報要求、ヒントなどですオーディエンスが要求し、対応するアプリケーションが検索で最も多くのインプレッションとクリックを受け取りました。

2）アナウンスメントのテキストを特定のアクションの呼び出しの形式で定式化することをお勧めします。たとえば、「クリスマスボールを一列に集める-ポイントを獲得してください！ Windows 8およびRT用ゲーム»

3）複数のアプリケーションが宣伝されている場合、一般的な検索フレーズを作成して、アプリケーションごとに引用できます。たとえば、アプリケーションごとに1日あたりXクリックを超えることはできません。



また、アプリケーションを宣伝する際に、PRツールを使用しました。これは、アプリケーションブランド自体を宣伝する雑誌やマスメディアの特別なプロジェクトですが、もちろん、アプリケーションの即時ダウンロードにはつながりませんでした。 したがって、限られた予算で、このツールを優先的に使用することはお勧めしません。
