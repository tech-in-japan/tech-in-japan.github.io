VKontakteとスパマーについて説明します。 テーマは永遠です:)



かつて、 vktools.site50.netというリンクがありました。 彼らはサイトにログインすることを申し出ますので、後で閉じたページを見ることができます。 このページはvkontakte.ru認証ページと1対1で対応しています。 私の気分はあまりよくなく（月曜日は難しい日です）、私はそれらに対処することに決めました...このサイトのホスティングプロバイダーは、多くを理解する必要はありません。



私は彼らにメールの濫用で書いた：

Hi! I from Russia, and I know english not very well :)

This site vktools.site50.net is collecting logins and passwords of users of big russian social network "Vkontakte" (vkontakte.ru). They say, that if you login to network from this page, there will be new useful functions. Of course, there is fake.



Then, they uses users accounts for spam in this social network.



At HTML view you can see this js code:







This code is simple for hide your warning message:

"This website *may be* insecure!! If you think it is illegal, please".



Dont ignore this message. I know many victims of this fake login-page. Please, close this site.



3時間後、彼らは私に答えた：

Hello,

Thanks for letting us know, we have removed this site now

Thank you,

www.000webhost.com





サイトは閉鎖されました:)私はこれに満足していなかったので、このようなサイトをさらに2つ閉鎖することにしました。 Vkontakteは、グループ内の1つのグループのクローンを約10個発見し、それらはすべて別の偽ページvktools.awardspace.usをアドバタイズしました 。 テーマは同じで、伝説は同じです。 私はすでに彼らのホスティング会社awardspace.usに対して虐待を書いています:)彼らの答えは、それほど長くはかかりませんでした：

Hello,



Thank you for your e-mail notification of this abusive activity. Please be advised that we have investigated and disabled the following site/s:

vktools.awardspace.us

Our company provides legitimate web hosting services and has a zero tolerance policy towards such violations. We apologize for the inconvenience this has caused you. If you still have any concerns regarding the matter, please feel free to contact us back.



Best Regards,

Max







また、このサイトは閉鎖されています。

なぜ私がこれをすべて書いたのか-私たちの誰もがスパマーにそのような汚いトリックをすることができます。
