Macworldの陰謀が明らかに

MacWorldは皆を魅了しているようです！ Macworld 2008の9to5Mac.comによると 、Steve JobsはMacBook Airを紹介する予定です。

これはアルミニウム製のラップトップで、サイズはMacBookに匹敵しますが、パフォーマンスの点ではMacBook Proのラインに関連しています。 今、展覧会の領土に掛かっているポスター「2008空気中に何かがあります」の意味が明らかになりました。 このようなデバイスに関する考えは、次のことによって決まります。

  1. ラップトップで可能なWiMAXXアダプター。
  2. 昨年、誘導充電に関するアップルの特許
  3. ポスター "2008空気中に何かがあります" [ Habréでの同じ名前の議論
  4. Adiumの使用レポートでこのフレーズを積極的に言及しています 。 詳細については、こちらの記事全文をご覧ください 。


同じリソース（9to5mac.com）が昨年、Steve Jobsによるプレゼンテーションの前に、新しいiPod Nano 3Gの最初の写真を公開したことは注目に値します。



これがすべて当てはまる場合、市場に固有のデバイスが私たちを待っています。 また、スティーブンノートの2日前にMacworld 2008の真相を突き止めることができたのも興味深いことです。
