第3世代および第4世代Intel Coreプロセッサー（Ivy BridgeおよびHaswell）とNVIDIA GeForce GTX 675MXおよびGeForce GTX 770Mグラフィックスカードの比較





このビデオでは、第3世代と第4世代のIntel Coreプロセッサーの比較テストの結果、およびNvidia GeForce GTX 675MXとGTX 770Mグラフィックスカードの比較を示します。 合成パフォーマンステストのテストに加えて、いくつかの最新のビデオゲームで測定を行い、テストシステムの消費電力と熱放散を確認しました。



テストには、異なるプロセッサとビデオカードを搭載したMSI GT70ラップトップを使用しました。



1）Intel Core i7-3630QM（Ivy Bridge）2400 MHz、6 Mb L3キャッシュ、TDP 45W、HD4000

Nvidia GeForce GTX 675M、4Gb GDDR5、GK104、960コア@ 600 MHz、256ビット@ 900 MHz



2）Intel Core i7-4700HQ（Haswell）2400 MHz、6 Mb L3キャッシュ、TDP 47W、HD4600

Nvidia GeForce GTX 770M、3Gb GDDR5、GK106、960コア@ 811 MHz、192ビット@ 1000 MHz
