（Kiev）2月16日から20日まで、コース「VMware vSphere 5.5-VMware vSphere：Troubleshooting Workshop [V5.5]」が開催されます





コース内の50以上のラボ「VMware vSphere 5.5-VMware vSphere：トラブルシューティングワークショップ[V5.5]」

MUKトレーニングセンターでは、2015年2月16日から20日まで開催される「VMware vSphere 5.5-問題解決/ VMware vSphere：トラブルシューティングワークショップ[V5.5]（H6D07S）」というタイトルのコースの受講者を全員に招待しています。

この5日間のハンズオントレーニングワークショップは、VMware vSphereで問題を見つけて解決するための高度な知識とスキルを学生に提供します。 トレーニングに合格すると、VMwareコマンドラインを使用してこの環境で発生した問題を分析するスキルと能力も向上します。



ラボ（合計50以上）は、コース時間の75％を占め、VMware ESXi 5.5およびVMware vCenter Server 5.5上に構築されています。 コースに参加すると、VMware Certified Professional 5-Data Center VirtualizationのVCP5-DCV試験に正式に合格することができます

コースの主なモジュール：

モジュール1：コースの概要

モジュール2：主要な問題の解決の概要

モジュール3：基本的な問題解決ツール

モジュール4：SSL証明書

モジュール5：ネットワーク

モジュール6：ストレージシステム

モジュール7：クラスター管理

モジュール8：vCenter ServerおよびESXi

モジュール9：仮想マシン



プログラムの詳細については、 training.muk.ua / courses / obuchenie-spetsialistov-v-oblasti-virtualizatsii-5 / 3329をご覧ください。



このコースのトレーナーは、HPの分野における情報技術の分野の認定スペシャリストです。
