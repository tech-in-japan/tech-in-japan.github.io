職場のLED照明

私は長い間読みましたが、私は決定することができませんでした-他の人にとって有用で興味深いものを書くためにそこに何がありますか。

そして今、記事「 ホームCNCルーター - ミリングユーザーvbifkol 」を一度読んで 、私は決めました-ここにあります！



一年前、私がまだ初心者で貪欲なCNC初心者であったとき、私は安くて踏みにじるものを手に入れることができると決めました。 私は一人で小さなシリーズで生産されたマシンを1台購入しました。 初心者のマシンとして、マシンは許容可能な選択肢でした。 しかし、彼らが言うように、貪欲な人は二度支払う。 社交的な人物である彼は、おなじみの広告主にスタノチキについて話し、すぐに最初のトライアル注文を受け取りました。 そして、それは始まりました、そうではありません、そうではありません-速度はありません、精度はそうではありません、作業場は面白いです（33 x 43センチ）。 1年余りのレースの後、マシンを販売し、産業用組立機を使用することになりました。 最後に何が起こったか興味があるなら、



私はヒューマニスト（トレーニングによる物理学者）であるため、専門的なフォーラムの研究から積極的な検索を開始しました。 これらのフォーラムへのリンクを意図的に示しているわけではありません。興味深い場合は、リンクを公開します。 フォーラムを勉強する-私は自分が必要なものに気づいた-そして、どのパラメータが売り手に表明される必要があるのか​​。



そのため、選択は以下に限定されていました。

1. CNCはDSPプロセッサによって制御される必要があります。

2.作業領域は少なくとも90 x 60 cmで、ワークピースの高さは少なくとも100 mmでなければなりません。

3.スピンドルは、少なくとも1.5kWの水冷式でなければなりません。

4. Z軸に沿った高さを設定するためのツールセンサーの存在。



会話中に3人の売り手との通信にほぼ2週間を費やし、売り手の数を減らし、配達、機械の梱包、保証の入手可能性などを含むあらゆる微妙な点を明確にしました。 賃貸施設の特性と200 kg近くの定期的な輸送の不足により、デスクトップバージョンのマシンを使用することが決定されました（テーブルには溶接されたいわゆるベッドが必要なため、デスクトップバージョンは条件付きでした）。

銀行振込でほぼ20万ルーブルを支払ったので、私は神経質に売り手からお金を受け取ることについての手紙を待ち始めました。 3日後、「機械が運送会社に送られました。電話を待ってください」というメッセージを受け取りました。 受け取り次第、4人の友人のために時間が費やされ、友人の1人のジープが打撃を受けました-間違った時間にピックアップドアが開かれました。



1週間後、ホームCNCルーターに関する記事を読んだ後、最終的にはホームに役立つ何かをすることが決定されました。 アクティブなPCユーザーとして、職場の照明の問題に常に夢中になりました。 省エネランプを備えた卓上ランプは光が少なすぎ、ランプは恐ろしい規則性で燃え尽きました。 月に2〜3回90ルーブルのコストでランプを交換することは非常に悲しいプロセスであることが判明しました。



したがって、トピックの具体化は解決されました。



素材とデザインの選択



私は長い間、モニターの背後にある背景照明を整理する問題を解決する方法を考えていました。 オプションがありました-LCDモニターの背面にLEDストリップを貼り付けましたが、このオプションは破棄されました：

1.テープが非常に顕著に熱くなるので、そのような加熱はモニターにとって有益ではないと判断しました。

2. LEDストリップの電源がテーブルに制御不能にハングします。

3.ハイテクユニットが存在する場合。

4. n1-n2の組み合わせは非常に面白くないように見え、n3はそうする理由を与えませんでした。



材料として、10mmの合板を選択することにしました。 合板は、CNCルーターで処理するのに非常に便利な素材であり、価格は非常に低予算です。 合板のシートが最寄りの金物屋で450ルーブル（1.5m x 1.5m）で購入されました。



電子機器を購入しました：

1. 1メートルのLEDストリップ（名前は覚えていませんが、店で最も明るいストリップでした）-260ルーブル。

2. 12ワットの電源-320ルーブル。

3.ネットワークに接続するためのプラグ-25ルーブル。

4.電源を220Vネットワークに接続するための1メートルのワイヤ-12ルーブル。

合計：617ルーブル。
