生きているのはDropBoxだけではありません-人気のあるサービスの10の選択肢

Dropboxは非常に人気のあるクラウドサービスです。 あなたが長い間それを使用しているなら、あなたはたぶん代わりを探しませんでした。 これは驚くべきことではありません-Dropboxは非常に使いやすいので、別のサービスを検索するインセンティブはありません。 ただし、考えてみると、このサービスにはいくつかの重大な欠点があります。

•独自のサーバーをホストすることはできません。

•2 GBのみ無料。

•コンピューターへのリモートアクセスなし。

•オープンソースなし。



しかし、DropBoxサービスには多くの代替手段があります-新しくも良くもありません。



1. Dmailerバックアップ

Dmailerは優れたファイルリポジトリです。 このサービスは、WindowsシステムでもMacシステムでも使用できます。 このサービスは、最大2GBのファイルのバックアップサービスを提供します。 Dropboxの場合のように、すべてのデータはネットワークに保存されます。 DropBoxに対するDmailerの利点のリストは印象的です。

•データを自動的にバックアップする機能。

•1つのファイルで複数のバージョンを一度にサポートする機能。

•複数のアカウントを作成する機能。

•セキュリティ：アプリケーションはパスワードで保護され、AES 128ビットプロトコルで暗号化されます。

•ヘルプ：取扱説明書を使用する機能。

•シンプル：すべてがワンクリックで実行されます。

•18言語でのローカライズ。



2.シンプリシティ

Syncplicityは、データの同期とバックアップのプログラムとして使用できます。 さらに、結果として、サーバーは編集可能なファイルの最新バージョンと以前のバージョンの両方を保存します。 PC上のファイルを誤って削除した場合、Webインターフェースを介して復元されます。 削除されたファイルと中間コピーが占めるスペースは、合計から差し引かれません。



Syncplicityはシステムのコンテキストメニューに統合されており、そこから必要なフォルダーを同期フォルダーのリストに追加したり、ダウンロード用のファイルへの直接リンクを取得したり、一般アクセス用のディレクトリを共有したりできます。



招待された任意の数のコンピューターでSyncplicityユーザーが特定のフォルダーのコンテンツを変更できるようにし、すべての変更が他のユーザーに即座に転送されるようにして、ディレクトリのコンテンツがすべてのPCで同一になるようにすることができます。



Syncplicityは、一部のオンラインサービスとも緊密に統合されています。 たとえば、PC上の特定のフォルダーに写真をコピーするだけで、Facebookのアルバムに写真をアップロードできます。 Googleのオフィススイートとの統合も非常に興味深いようです。Googleドキュメントで作成されたドキュメントは、コンピューター上の選択したフォルダーに表示され、その逆も同様です。 したがって、ファイルはMicrosoft Officeを使用してローカルで編集され、オンラインで編集されます。



無料のアカウントでは、2 GBの空き容量があり、3人の友人を招待することで5 GBに増やすことができます（同時に1 GBのボーナスも受け取ります）。



03.ワラ

Wualaは、DropBoxに代わるLinux向けの安全な代替ツールです。 新規登録ユーザーは、Wualaチームから1 GBをオンラインで取得します。



以前は、ローカルハードドライブのストレージの場所を交換することで、このボリュームを増やすことができました。 これを行うには、コンピューターが少なくとも1日4時間オンラインであることが必要でした。 ネットワークスペースの利用可能なサイズは、コンピューターがオンラインで費やした時間と、Wualaに提供されたローカルハードディスクスペースのサイズに応じて計算されました。



ただし、2011年11月3日、クラウド共産主義は取り消されました。「トレード」で取得したスペースは、もう1年間有効です。 その後、他の方法を使用してスペースを拡張する必要があります。友達を招待する、プロモーションコードを使用して空きスペースを取得する、または購入する。



4. Sparkleshare

Sparkleshareは、オープンソースを必要とする人々のためのソリューションです。 ファイルコラボレーションを整理するための無料のオープンソースユーティリティ。ユーザーのセットアップを簡素化し、プログラムをできる限り操作するように設計されています。 このプログラムは、ユーザーのホームディレクトリにSparkleShareというディレクトリを作成します。このディレクトリに、同期のために多数のリモートディレクトリを接続できます。これらのディレクトリは、異なるサーバーに配置できます。 とりわけ、SparkleShareにはDropBoxに似た通知システムがあります。 PPAリポジトリからユーティリティをインストールできます。



$ mkdir -p ~/.ssh

$ sudo add-apt-repository ppa:warp10/sparkleshare

$ sudo apt-get update

$ sudo apt-get install sparkleshare libwebkit1.1-cil git-core







〜/ .sshディレクトリがない場合、最初のコマンドが必要です。これがないと、SparkleShareがクラッシュします。 LinuxがUbuntuでない場合は、プロジェクトの対応するページからダウンロードしたソースコードからSparkleShareをコンパイルできます。



5.ライブメッシュ

Live Meshは、任意の数のコンピューターを同期し、ファイルストレージに最大5 GBを割り当てる機能を提供します。 コンテキストメニューから新しいフォルダが追加されます。 残念ながら、他のユーティリティとは異なり、Live Meshには、ファイル自体のアイコンに現在の同期ステータスを表示するインジケーターはありません。



少し予想外だったのは、リモートアクセスの可用性です。 別のコンピューターに接続するには、Live Meshのインストールはオプションで、ブラウザーからPCを直接制御できます。

欠点には、サービスに接続されている他のコンピューター上の同期フォルダーがデスクトップ上にのみ表示され、その場所を変更できないという事実が含まれます。



6. SugarSync

SugarSyncは5 GBの空き容量です。 複数のコンピューターとのバックアップと同期に加えて、モバイルデバイス（iPhone、Android、Blackberry、Windows Mobile）のアクセシビリティ機能が特徴的です。



7. Windows Live SkyDrive

Windows Live SkyDriveは、登録ユーザーに無料で提供されるディスク容量の点で議論の余地のない「ナンバーワン」です-25 GB！



任意の形式のファイルを保存できますが、いくつかの追加の利点があります。 したがって、これらがOfficeドキュメントの場合、SkyDriveに統合されたOffice Web Appsの助けを借りて、ブラウザーで直接編集できます。コンピューターにインストールされたOffice 2010では、SkyDriveでドキュメントを直接保存および開くことができます。 写真とビデオはディレクトリに構造化され、ブラウザでスライドショーを表示できます。ファイルへのアクセスはさまざまなユーザーカテゴリに制限できます。関連するWindows Live Meshサービスにより、1台以上のコンピューターでSkyDriveを介してファイルを自動的に同期できます。



8. SpiderOak

無料の使用モデルでは、2 GBがファイルに割り当てられ、クライアントはすべてのプラットフォーム（Windows、Mac、Linux）で使用できます。



SpiderOakとDropboxの主な主な違いは、特別なフォルダーにあるだけでなく、ファイルとフォルダーを同期する機能です。

自動（一定の変更追跡）から2日ごとに1回まで、バックアップの頻度を構成する機能、変更されたファイルを以前のバージョンにロールバックする機能、およびサービスの操作を自動化するための多くのオプションもあります。



9.4共有

繰り返しますが、私たちの地平線には楽しいものが現れました。 SkyDriveと同様に、 4Sharedにはロシア語のインターフェースがあり、無料の関税で提供されるデータストレージの量はSkyDriveを思い出させます-200 MBの1ファイルの最大サイズと、ダウンロードの制限速度を除いて10 GBコンテンツサービス。



アクセスとファイル管理の容易さの観点から、SkyDriveとDropboxの間で何かが提案されます-クライアントはコンピューターにインストールされますが、システムには組み込まれません。 追加サービスのうち、投稿された写真からの仮想フォトアルバムの生成と、ダウンロードされたビデオファイルをサイトで直接表示する機能に言及できます。 ただし、前述の制限とあらゆる方法での十分な量の広告により、ユーザーは月額10ドルまたは年額78ドルで有料料金に切り替えるように動機付けられます。



10. Box.net

Box.netでは、無料料金により5 GBのディスク容量が提供され、ファイルあたり25 MBの制限があります。 アップロードされたファイルをダウンロードするためのリンクを提供できます。モバイルクライアントのセット（iPad / iPhoneおよびAndroid）があります。 残念ながら、高度な機能はビジネスレート（ファイルあたり500 GBおよび2 GB）でのみ取得できます。初期料金はユーザーあたり15ドルで、少なくとも3人のユーザーが必要です。パスワード保護と同期は既に表示されています。ユーザーのコンピューターとGoogle Appsの統合。
