YouTrack 4.1の時間管理とすべてすべて

YouTrackバグトラッカーのアップデートがリリースされました。バージョン4.1には、プロジェクトを管理するだけでなく、非常に便利な機能があります。







時間管理



したがって、バージョン4.1の主な革新は、時間を管理する機能です！ これで、タスク、反復、またはプロジェクト全体の完了にかかる時間を制御し、それを予備的な見積もりと比較できます。 チームがどのようにタスクを実行しているかを追跡できるように、費やした時間を報告します。



費やした時間に関する記録の便利な追加

タスクの全画面に、新しい[時間追跡]タブが表示されました。 ここで、特定のタスクに費やされた時間に関する1つまたは複数のレコードを追加できます。 1つまたは複数のタスクの実行に費やした時間の記録を追加するための便利なコマンドがあります。 次のコマンドを入力してください。

work 1d2hは基本的なロジックを実装しました

推定時間と実際の時間の比較

タスクを完了するのに必要な時間を見積もり、実際に費やした時間を追跡します。 初期推定値に従って残り時間を視覚的に表示して、作業の進行状況を確認します。





タイムレポート

新しいタイプのレポート-時間レポートを使用して、特定のサブシステム、反復、またはプロジェクトで特定の時間に費やした合計時間を追跡します。 たとえば、次の検索クエリを使用します。

プロジェクト：JBC #Resolved -Bugs

JBCプロジェクトの完了したすべてのタスクを含み、エラー（バグ）を除外した経過時間に関するレポートを作成します。 期間を指定し、タスクまたはユーザーごとにグループ化する方法を選択し、レポートをエクスポートまたは印刷します。





特定のプロジェクト/ユーザーの時間追跡を有効化する

ユーザーが[タイムトラッキング]タブを表示したり、経過時間のレコードを追加したりできるようにするには、 タイムトラッキングの読み取り/更新権限を有効にします。 推定/使用時間フィールドは、カスタムフィールドの標準アクセススキームを使用してアクセスされます。 管理領域でプロジェクトのタイムトラッキングを有効にし、既存のフィールドの[推定]および[使用時間]を追加または選択して、タイムトラッキングの使用を開始します。





タイマーフィールド

ワークフローworkTimerを使用して、費やした時間を自動的に追跡します。 作業の開始時に[タイマー]フィールドを[開始]に設定し、最後に[停止]に変更します。 経過時間の記録が自動的に追加されます。



GitHubとYouTrack：より良い連携



YouTrackからのセットアップ

統合のセットアップは、YouTrackプロジェクトのGitHubタブから、YouTrackユーザーインターフェイスで直接行われるようになりました。





統合は計画どおりに機能し、コミットから直接コマンドを実行できます。 たとえば、

＃SDA-2135修正済み

コミットコメントで、YouTrackでSDA-2135タスクがfixedとして自動的に閉じられるようにします。



コミットするリンク

これで、YouTrackタスクからコミットに直接移動できます。 コミットハッシュをタスクの説明またはコメントに追加すると、YouTrackはそれを自動的にコミットリンクに変換します。 この機能を有効にするには、管理領域の[GitHub]タブで接続データを指定します。



アジャイルタスクボード-その他の機能



スイムレーンの展開/折りたたみ

これで、タスクボードのスイムレーンを展開したり折りたたんだりして、すべてのタスクを非表示または表示できます。 すべてのタスクの最後に「スイムレーン」を展開して、次の要素の開発に切り替えます。 タスクをテストする必要がある場合は、スイムレーンを展開します。



複数のタスクに対するアクション

これで、タスクリストと同様に、ボード上の複数のタスクにコマンドを適用できます！ 必要なタスクを選択して、すべてを1人のチームメンバーに割り当てたり、別のスプリントに移動したりするだけです。





追加の「グッズ」

時間推定に使用される新しいタイプの推定フィールド、フロートと期間が利用可能です。 これで、ボードとタイムアカウンティングの両方で1つの推定フィールドを使用できます。 さらに、複数のタスクタイプの値を選択して「スイムレーン」を定義するサポートを実装しました。つまり、スイムレーンが機能とユーザーストーリーの両方を同時に形成できるようになりました。



その他の改善点



  • レポートの新しいページ 。 YouTrackレポート用に新しい別のページを追加しました。このページでは、マトリックスレポートまたは費やした時間に関するレポートを作成できます。


  • ディスク容量不足の警告 。 YouTrackは、サーバーの空きディスク領域が不足していることを警告して、操作の失敗を回避します。


  • 新しいタイプのカスタムフィールド 。 完全に新しいフィールドタイプ、floatおよびperiodを追加しました。変換オプションは、 int float 、 period int 、 period floatです。
