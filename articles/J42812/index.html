ビジュアルデザインツール

ウェブサイト/ウェブサービス/プログラム（下線）の開発を成功させるには、設計段階が非常に重要であることを誰もが知っています。 有能な開発計画はプロジェクトの半分です。 まず、要件、用語などを説明する技術的なタスクが必要であることは明らかです。 しかし、ビジュアルデザインはさらに重要です。つまり、UMLスキーム、データベーススキーム、ユーザーインターフェイス、ページサムネイルなどです。



これらの各問題を解決するには、独自の手段があります。

たとえば、ユーザーインターフェイスは、紙またはPhotoshopで表示できます。 ただし、1つのプログラムで上記のすべてを作成できる方がはるかに便利であることに同意してください。



以下に、このようなプログラムの概要と簡単な注釈を示します。 コメントで、このような包括的なビジュアルデザインを実行できる他のソフトウェアへのリンクをぜひご覧ください。



Microsoft visio



ライセンス：トライアル

コスト： 18,000摩擦。

ウェブサイト： http : //office.microsoft.com/ru-ru/visio/



MS Visio 2007は、あらゆる種類のチャート、グラフ、図を作成するための汎用ツールです。 強力な視覚化ツールが含まれており、幅広いタスクに適しています。



Office.org Drawを開きます



ライセンス： FreeWare

費用：無料

ウェブサイト： www.openoffice.org/product/draw.html



OO Drawはベクターグラフィックスエディターですが、多くのユーザーがDrawを使用して、図面、フローチャート、およびその他の種類の図を描画します。



Axure RP Pro



ライセンス：トライアル

コスト： 585ドル

ウェブサイト： http : //axure.com



Axureは、Webサイトフレームワークの作成とラピッドプロトタイピングに焦点を当てています。 インターフェイスはMS Visioに似ています。 これには、ページデザインとユーザーインターフェイス用のオブジェクトのセットを持つ非常に大きなライブラリが含まれています。 グラフィカルな表現を作成できるだけでなく、イベントのバインド、ブロックの相互バインドなどを行うことができます。 完成したスキームをエクスポートする十分な機会。



SmartDraw 2009



ライセンス：トライアル

コスト： 197ドル

ウェブサイト： http : //www.smartdraw.com/



あらゆる種類のインターフェイスを作成できる非常に強力なツール。 インターフェイスはMS Office 2007に似ています。オブジェクトの巨大なライブラリ（追加の要素はオフサイトで利用可能）、およびすべての種類のダイアグラムタイプ。



Gliffy.com



ライセンス： ShareWare

費用：月額5ドル

ウェブサイト： http : //www.gliffy.com/



gliffy.comは、フラッシュで書かれたWebアプリケーションです。 フローチャート、構造、ユーザーインターフェイス、UMLダイアグラムを作成するための武器ツールを備えています。 これは、ドキュメントの便利な公開と共同開発の手段を表します。 ラスターグラフィックスとベクターグラフィックスの両方にインポートオプションがあります。 無料版では、ドキュメントの最大数に制限があります-5。
