Windows 7のキーボードショートカット

ホットキーのリストをご紹介します。ホットキーを使用すると、Windows 7での作業がさらに便利になります（かっこ内は同じことを行う別の方法です）。



ウィンドウ管理



  • Win + Home ：アクティブなウィンドウ以外のすべてのウィンドウを最小化/最大化します。 （「シェイク」ウィンドウ）
  • Win + Space ：すべてのウィンドウを透明にします。 （画面の右下隅にある小さなボタンの上にカーソルを置きます。）
  • Win + Up ：アクティブウィンドウを全画面に拡大します。 （ウィンドウをストップまでドラッグするか、上のフレームをダブルクリックします）
  • Win + Down ：アクティブウィンドウを最小化するか、フルスクリーンに展開されている場合はサイズを復元します
  • Shift + Win + Up ：ウィンドウの高さを最大化します（ウィンドウの下の境界線をタスクバーにドラッグします）
  • Win + Left / Right ：モニターの左/右ウィンドウを占有するか、サイズを復元します（停止するまでウィンドウを左/右にドラッグします）




タスクバー



  • Win + digit ：この番号でプログラムを実行するか、切り替えます
  • Shift + Win + 番号 ：この番号の下でプログラムの新しいコピーを起動します（ホイールのアイコンをクリックします）
  • Ctrl + Win + 番号 ：この番号で次のプログラムウィンドウに切り替えます（クリックしてCtrlを押しながら）
  • Alt + Win + digit ：この番号の下にあるプログラムジャンプリストを開きます
  • Win + T （+ Shift ）：開いているすべてのウィンドウを切り替え、プレビューをタスクバーの上に直接（逆）順序で表示します
  • Win + B ：トレイにフォーカス
  • Ctrl + Shift + クリック ：管理者としてプログラムを実行
  • Shift + 右クリック ：ウィンドウ/グループメニューを表示


エクスプローラー（およびデスクトップ）



  • Ctrl + Shift + N ：新しいフォルダーを作成
  • Alt + Up ：フォルダ内でレベルを上げる
  • Alt + P ：ファイルビューペインを有効にする
  • Shift + 右クリック （ファイルごと）： [パスのコピー]オプションをコンテキストメニューと[送信]サブメニューの多くの追加項目に追加します
  • Shift + 右クリック （フォルダーごと）：このフォルダーからコマンドラインを開くことができるコンテキストメニューに項目を追加します


その他



  • Win + P ：プレゼンテーションモードのセットアップメニューを開きます（2番目のモニター-プロジェクター）
  • Win +（ + / - ）：画像を拡大/縮小します（画面拡大鏡を呼び出します）
  • Win + G ：実行中のガジェットを切り替える
