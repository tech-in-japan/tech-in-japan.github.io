Slackware 13.37リリース



ええと、Linuxの世界からの今日のニュースが豊富にあることに驚いています。

センセーショナルなUbuntuのリリースに加えて、最も古いディストリビューションの1つであるSlackwareが今日更新されました。

世界はバージョン番号13.37を見ました。



リリースには以下が含まれます。

KDE 4.5.5

Xfce 4.6.2

GCC 4.5.2

Glibc 2.13

GIMP 2.6.11

Perl 5.12.3

Python 2.6.6

Ruby 1.9.1-p431



デフォルトのカーネルバージョンは2.6.37.6です; 2.6.38.4をインストールすることもできます

デフォルトのブラウザーはFirefox 4で、WebMビデオを表示できます。
