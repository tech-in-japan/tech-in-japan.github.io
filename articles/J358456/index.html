英国の暗号化デバイス：細部に至るまで英語による制限

芸術としての暗号化、歴史の流れに影響を与える傑作の作成と作成の芸術。 ソビエト連邦、ドイツ、アメリカ、日本の暗号化装置の歴史...英国のいくつかの暗号化機械に精通する時が来ました。







イギリスは、機械式回転機械から完全な電子機器まで、さまざまなデバイスの幅広い生産を誇っています。 長年にわたり、政府通信サービス（GCHQ）は、時には米国国家安全保障局（NSA）と協力して、その開発に関与してきました。 前世紀の前半までに、暗号マシンを作成して使用することの全体的な重要性は昼間のように明らかになりました.1926年に、特別な委員会が作成されました。その目的は、既存の秘密暗号化システム（まだ機械的）を調査および分析することでした。



1926年はなぜですか？ はい、すべてが簡単です。ドイツが軍事分野で初めて暗号化マシンを使用したのは今年でした。 これは、他の国が独自の暗号化および復号化方法を開発および開発するための一種の推進力でした。 機械の理論的開発における優位性はポーランド人とフランス人に属し、英国はポーランドとフランスの占領後も暗号分析を続けました。



Typex



ドイツのエニグマの既存の商用バージョンがプロトタイプとして採用されました。







Typexは、OGWのウィングコマンダーであるLywoodと英国空軍（RAF）の3人のメンバーのリーダーシップの下、1934年に英国で開発された電気機械式ロータリー暗号化マシンです。 すでに述べたように、それは英国版のエニグママシンでした。 暗号機のプロトタイプは1935年までにすでに準備が整っていました。1937年の初めに、30個のTypex Mark I機の最初のバッチが英国空軍に提供されました。 後に、この装置は改良され、より普遍的なTypex Mark IIが光を見ました。第二次世界大戦中に空軍だけでなく、地上軍と海軍によっても使用されました。 Typexは、「Type X」または「TypeX」と呼ばれることもあります。 後のマシンオプションは、BID / 08/2およびBID / 08/3として知られています。



伝説的なBletchley Park（英国）の博物館では、いわばTypex車の興味深い例を見ることができます。 Typexマシンにはいくつかの異なるモデルがありますが、Mark 23をより詳細に検討してください。



Typexの機能







フロントパネルにはキーボードがあり、珍しいことは何もありませんでした。キーとスペースのある3列。 キーボードのすぐ後ろには、5つのローターホイールを収容した回転ドラム（バスケット）がありました。 キーボードと回転バスケットは、ドイツのエニグマのコピーである機械の2つの要素です。 入力ディスクとリフレクターがあり、その上に5つのローターが配置されました（そのうちの2つは静的です）。 エニグマとの違いは、3つのローターではなく5つのローターであり、ローター自体の設計が改善されています。 ローターを備えたバスケットの後ろには、駆動機構またはアクチュエーターがありました。 エンジンがマシンを起動し、すべてのコンポーネントの作業を同期しました。 左右にある2つの大きな黒いシリンダーは、それぞれクリアテキストと暗号化テキスト用のプリンターです。 テキストは紙テープに印刷されました。



キーボードの左側にある32ピンコネクタは、プラグインスイッチを接続するためのもので、リフレクターを交換できます。 この機能は、後のバージョンのMark 22で可能になりましたが、元のMark IIにはありませんでした。 キーボードの右側にある33ピンジョーンズコネクタにより、エニグマのように、キーボードと入力ドライブの間に一方向のプラグインスイッチを接続できました。







この図は、Typexマシンのスクランブラーがどのように機能したかを示す簡略図を示しています。 右側には入力ディスクとリフレクターがあり、それぞれ26の接点があります。 5つのローターディスクがリフレクターと入力ディスクの間に配置され、2つの右側のローターは静的（暗号化中に回転しませんでした）で、他の3つはエニグマのステップメカニズムと同様の（ただし同一ではない）ステップメカニズムによって駆動されました。

ローターホイール



Typexローターは、ドイツのEnigmaマシンのローターに非常によく似ています。 ローターの周囲-両側に26文字と26接点。 各ローターにはディスクホイールがあり、これを使用して開始位置を変更し、右側のギアディスクで駆動しました。 信頼性を高めるため、各連絡先は2倍になりました。 2-静的ローター。ただし、手動でモーションを設定することが可能でしたが、3つの可動ローターがありました。 車輪をより頻繁に回転させる（不規則なステップ）ために、車輪にいくつかのノッチがありました（5、7、9など）。 ローターは通常、木製の箱に保管されていました。

Typex Mark III



Typex Mark IIIは、Typex Mark IIのポータブルバージョンと見なすことができます。 Mark IIと同じ暗号化ローターを使用し、マシンの外観は同じです（特にフロント部分）。















機械には手動制御が装備されていましたが、ドライブと電気モーターを接続することができました。 60文字が1分で処理され、オペレーターは左手でテキストを入力し、右手でハンドルをひねりました。 マシンの後ろには、回転するプリントヘッドを備えた小さなプリンターがあり、9.6 mmの紙リボンにテキストが表示されていました。



Typex Mark VI







Typex Mark VIは、電動モーターで制御されるTypex暗号化マシンの「縮小版」で、テキストは9.6 mmの紙テープに直接表示されました。 マシン自体はMark IIIよりも小さいですが、前部は長く、プリンターは暗号化ブロックの右側に取り付けられています。







ローターホイールは他のTypexモデルのローターとは異なり、直径が小さかった。 キーボードの左側にある大きな32ピンStrowgerコネクタは、外部電源の接続に使用されました。



Typex Mark VIII







Typex Mark VIIIについてはほとんど知られていませんが、モールス符号電信の記号に対応する紙テープに穴を開けるデバイスを備えたTypex Mark II暗号化マシンの説明がある場合があります。 Typex Mark VIIIはTypex Mark IIファミリーのものですが、機械的な「レイアウト」がわずかに異なります。



ローター付きのバスケット（ドラム）が左に移動し、キーボードのすぐ後ろにあるパンチ用のスペースが解放されました。



Typexマーク22



Typex Mark 22は、最も有名なTypexファミリー暗号化マシンです。 第二次世界大戦中の最高レベルのメッセージングのために英国で使用されます。 Typex Mark IIの心臓部に加えて、キーボードの両側に追加のプラグインスイッチがインストールされています。

Krikhは単純なものではなく、キーと文字レバーがワイヤーでリレーに接続された電気タイプライターを取りました。 Krikhは、導体を引き裂き、それらの間に中間リンクを挿入して、デバイスの外部パネルのプラグを単純に再配置することにより、任意の順序でワイヤを混同する機会を得ました。 デバイスの主な秘密は彼のデバイスではなく、キー-送信者と受信者だけが知っているプラ​​グの場所です。
エニグマのプラグインスイッチと同じ場所に、右側のプラグインスイッチをキーボードと入力ディスクの間に挿入しました。 ただし、（エニグマのワイヤーペアと比較して）単一のワイヤーを使用しているため、より多くの順列が可能になり、したがって、より優れた暗号化保護が提供されます。 これにより、Typex Mark 22は有名ななぞなぞの類似物になりました。 エニグマキーを解読した後、ブレッチリーパークの施設には、膨大な数のドイツ語のメッセージを解読するために使用されるTypex Mark 22暗号化マシンが「限界まで」装備されていました。



一部のMark 22マシンは、SIGABA American暗号マシンとの互換性のために変更されています（Mark 23）。



Typexマーク23



Typex Mark 23のベースはTypex Mark 22で、Combined Cipher Machine（CCM）として使用するためにわずかに変更されました。 キーボードの右側にある追加のコネクタであるロータリーバスケット用に、4つのプロファイルがCCMに追加されました（写真の茶色の棒の形で）。



マーク23は、第二次世界大戦中にアメリカ人と通信するために使用され、アメリカ人は修正されたSIGABA暗号マシンを使用していました。



アメリカ人とイギリス人は、Typexを使用して一方の側でSIGABAを使用して情報の交換を可能にするマッチングデバイスのバリアントを作成しようとしました。 複合暗号マシン（CM）-この技術の奇跡は洗礼を受けました。 興味深いことに、アメリカ人はCCMで修正されるイギリスの暗号化機に精通しており、イギリス人はSIGABAの存在すら知らず、どの暗号化機がアメリカ版のCCMの基礎になるかさえも知りませんでした。





アメリカのシガバ



このように、CCMの2つのバリアントが作成されましたが、マシンの動作に互換性があったという事実にもかかわらず、実際には非常に異なっていました。 両方のマシンの変換キットは、NSAによって開発および提供されました。 変換後、機械には5つのSIGABAローターが装備されました。 変更されたマシンは1943年11月に使用され始め、ご存知のように、決して侵害されませんでした。



Portex BID / 50/1



Portexは、1940年と1950年に英国のシークレットサービスで使用されていた英国の電気機械暗号化マシンでした。 機械はドイツのエニグマに似ていますが、それぞれに不規則なステップのある26個の接点を持つ8個のローターがありました。







テキストは一度に1文字ずつ入力され、アルファベットの付いたディスクは左側に目的の文字が設定されており、右側のハンドルをスクロールする必要がありました。 次に、文字はドラムの左端のホイールから電流を流します。 ホイールを右側に残す電流は、左側に取り付けられているプリンターの駆動に使用されます。 出力はストリップ（以前は濡れていた）に入り、マシンは左側に残ります。



ドイツのエニグマに似ていることに加えて、ポルテックスはスウェーデン・スイス・ハーゲリンのマシンのいくつかの機能を借用しています。 プリンター、アルファベットドライブ、および文字カウンター（これらはすべて暗号化ローターの左側にインストールされています）の設計は、ポータブルM-209およびC-446に似ています。 Portexは、Hagelin M-209とEnigma Gのハイブリッドと呼ぶことができます。



暗号マシンは、英国のサービスだけでなく、カナダ軍でも使用されました。 暗号化マシンの民間バージョンは黒いケースで、軍事バージョンは緑色で、さらに追加のコマンドボタンが装備されていました。 前方には、キーストロークごとに機械的に移動する8つの電気エンコーダーがありました。 左側には、プリンター、テキストを入力するためのディスク（アルファベットの付いたリング）、およびカウンターがありました。 ローターの後ろには、暗号マシンのステッピングメカニズムを制御するためのいくつかのソレノイドがありました。 現時点では、メカニズムが正確にどのように機能したかは残念ながら不明です。



一重項BID / 60



イギリスの電気機械式暗号マシンは、1949年後半または1950年初頭に英国で開発およびリリースされました。 彼女はイギリス版の複合暗号機械（CCM）を置き換えました。 イギリスのシングレットには、互換性のあるアメリカのKL-7のローターと同様に、36個の接点を持つ10個のローター（8個ではなく）がありました。 報告されているように、ローターはKL-7ローターと互換性があり、両方のマシンで使用できます（10個のローターのうち8個、他の2個はダミーになります）。 デバイスのコードネームはBID / 60です。







フロントパネルには、29個のキーとスペースのある通常のキーボードがありました。 後ろには、10個の電動エンコーダーを備えた取り外し可能なドラム（シリンダー）がありました。 正面には紙テープ（5ビット）が巻かれています。 生き残った文書によると、暗号機はイギリス軍によってオーストラリア、ニュージーランドと通信するために使用されました。







一重項ローターは、アメリカンKL-7（1952）で使用されているものと同じです。 KL-7は、米国国家安全保障局（NSA）によって開発されました。ローターは、米国と英国の共同製品である可能性があります。 いずれにせよ、それらは英国でCanvu IslandのEgen Electricによって生産されました。



ノーリーンBID / 590







Noreenは、1960年代初頭に英国で開発されたポータブルスタンドアロンOTT暗号化マシンであり、互換性のあるRockex暗号化マシンの代替（または追加）として外交交渉のために英国連邦で使用されています。 NoreenはBID / 590またはMk-854Mとして知られています。







テープを送るための外部デバイスがなければ、マシンの寸法はわずか33 x 27.5 x 17 cmであり、バーナムの暗号が使用された「最小の」暗号マシンの1つになりました。 Rockexと互換性があり、同じ6レベルのキーテープが使用されていましたが、Rockexとは異なり、ノリーンはパンチテープに直接出力できませんでした。 暗号化（または復号化）テキストは、デバイスのフロントパネルにあるロータリープリンターを使用して紙テープ（9 mm）に印刷されました。 次に、紙テープを空白の紙に貼り付け、テレプリンターで再印刷しました。



Noreenは1960年代に英国で生産され、外務省および連邦務省（FCO）によって使用されました。 契約によると、暗号化機械の技術はカナダに送られ、ここでは外部関係局で使用されました。 マシンはオーストラリアとニュージーランドに出荷されましたが、英国との機密通信のみが目的です。



当初、ノリーンは秘密の暗号化機器でしたが、1980年の初めまで使用されていましたが、その後そのデバイスは機密解除されました。







Noreenは、前面のキーボード、背​​面の機械部品、電子回路の3つのセクションで構成されていました。 マシンにはバッテリーパックまたは電源が搭載されていたため、

MODE動作モードスイッチは右上隅にありました：エンコード用のEN、デコード用のDE。 スイッチをOFFモードに設定することにより、マシンをタイプライターとして使用できます。







通常、外部の給紙装置が右側に取り付けられていました。 Noreenには、マシンの背後にある物理接続が2つしかありませんでした。 デバイスを特別なTEMPESTバッテリーパックに接続するために、端に10ピンコネクタが付いた固定ブラックマルチコアケーブルが使用されました。 テープドライブのスイッチを接続するための3ピンBulginコネクタもあり、エンコードまたはデコードのためにフィーダにテープがあるかどうかを判断するために使用されました。











上記の画像は、2つのテープ形式をキーシーケンスと比較しています。 右側-通常の5レベル、左側-Rockex / Noreenテープ（赤）、「異常な」幅が特徴で、6番目のレベルが追加されます。 エンコードとデコードには別々のテープが使用されました。 彼らは色が異なっていた。 英国では、通常、エンコードテープは緑になり、デコードテープは赤になりました。 カナダでは、リボンはそれぞれ黄色と赤でした。 キーボードは32個のキーで構成されていました（31個のキーと文字とスペース）。







印刷装置







Noreenは、キーボードのすぐ上にある紙テープにテキストを直接表示しました（右から左に移動）。 テープは幅約9.4 mmの狭い紙のストリップで構成され、アメリカのM-209とソビエトバイオレットで使用されていた同様のタイルでした。 一片の紙を水平に黒い紙ホルダーに挿入し（右）、印刷ヘッドを通過させました（左）。 プリントヘッドには2つの行がありました。1行目は文字（前）、2行目は数字と句読点です。
