Microsoft Ignite Conferenceセッションの概要

Microsoft Igniteは、2015年5月4〜8日にシカゴで初めて開催され、エンタープライズテクノロジーに特化した新しいカンファレンスです。 この会議は、最大の聴衆に対して最大の資料を提示することを約束します。 会議のセッション（およびリストによる判断では、618が予定されています）はすでに公式Webサイトで公開されています。 Microsoft Igniteで私たちを待っているものを一緒に見てみましょう。









会議について



「Spark the Future」-Microsoft Ignite電話会議の主催者は、2015年5月4日から8日にかけて、IT界の最新のトピックを1か所で議論する機会を得るために、これを呼びかけます。

Microsoft Igniteは初めて開催されますが、Management Summit、Exchange、SharePoint、Lync、Project、TechEd（「すべてを支配する1つの会議」）などの会議から最高のものを集めます。 宣言されたトピックのリストは疑いを生じさせません。 会議では、クラウドテクノロジーと管理、生産性、ビッグデータとモノのインターネット、ユニファイドコミュニケーション、モビリティ、インタラクションなどについて話します。 会議プログラムには、数百のセッション、トレーニング、実験室作業が含まれ、認定試験に合格する機会も与えられます。 そしてもちろん、Microsoft Igniteは、マイクロソフトの主要なエンジニアや専門家とライブでチャットするもう1つのチャンスです。

会議の聴衆も多様です。 セッションのほとんどは技術コミュニティに焦点を当てています-ITプロと企業の開発者、それにもかかわらず、IT界の絶えず変化する状況について学ぶことに興味があるかもしれないビジネスオーディエンスのための資料は、確かに提示されます。



618セッションの簡単な概要により、注目する製品を強調することができます。 議論の余地のないリーダーはOffice 365とAzureであり、SharePoint、Windows、Exchange、Windows Server、System Center、Skype for Businessがそれに続きます。



Windows 10



Windows 10は間違いなく会議の主要なトピックの1つです。 Igniteから、新しいオペレーティングシステムに関するさらに多くの情報を期待しています。 そして、会議のセッションの説明は、これらの期待を高めるだけです。 Windows 10 Enterpriseの概要に加えて、組織でWindows 10を実装および管理する方法について説明します。 専門家は、Windows 10への移行のためにインフラストラクチャを準備する方法、Azure AD、Intune、およびSystem Center Configuration Managerが新しいインフラストラクチャの管理にどのように役立つかを説明します。 次に、Microsoft ITはWindows 10への移行に関する経験を共有します。

また、モバイルデバイス用のWindows 10-新しいオペレーティングシステムを正常に展開する方法、考えられるユースケースについても説明します。

情報を保護し、Windows 10の脅威に対抗するための新機能について学習します。 Windows 10がIoTに提供するもの 企業におけるWindows 10のユニバーサルアプリケーションの導入と、Visual Studio 2015でのこれらのアプリケーションの開発。

Project Spartanや新しいHyper-V機能など、Windows 10およびWindows Server Technical Preview内の個々の製品に関する情報をお待ちしています。



クラウド技術



従来、会議のテーマではクラウドテクノロジーとソリューションが重要な位置を占めていました。 ここでは、Microsoftクラウドプラットフォームの概念そのものと、その個々の要素に専念するレポートがあります。 コンテナとRemoteApp、Azure Pack、ハイブリッドインフラストラクチャ、仮想ネットワーク、クラウドストレージの構築について話します。

別途、サードパーティ製品での作業などのトピックを強調する必要があります。 この会議では、AzureでのLinuxベースの仮想マシンの動作に関するレポートと、Azure Site Recoveryを使用してVMware vCenterで実行されている仮想マシンを保護する方法について説明します。

ラボ作業では、参加者はWindows ServerとSystem Centerを使用してプライベートクラウドを構築し、Azure PackとLinuxサーバーを使用してIaaSを管理し、Microsoft System Center 2012 R2を使用してExchange Server 2013、SharePoint 2013、SQL Server 2014を管理できます。



ビッグデータ



ビッグデータに言及するしかありません。 Igniteは、AzureでのBig Azureアーキテクチャ計画、HDInsightでのApache StormとHBaseによるリアルタイム分析、Azure Machine Learning、SQLおよびRによる分析に関するセッションを開催します。

プレゼンテーションは、AzureおよびSystem Centerとの相互作用であるSQL Server 2014で行われます。

参加者は、認定試験70-461（Microsoft SQL Server 2012のクエリ）および70-462（Microsoft SQL Server 2012データベースの管理）の準備セッションにも参加できます。



Skype for Business



Microsoft Igniteの主要トピックの1つは、ユニファイドコミュニケーションを発表しました。 このトピックの一部として、Skype for Businessについて詳しく説明します。 まず、多数のレビューセッションを待っています。レビューセッションでは、製品自体とその機能、他の製品（Officeなど）との対話について説明します。 また、組織でのSkype for Businessの実装についても説明します。

開発者がSkype Web SDKを使用してWebアプリケーションを作成するトピックに精通することは興味深いでしょう。 さらに、この理論は、Microsoftパートナーが提供するSkype for Businessの既製のアプリケーションと拡張機能に限定されませんが、実証されます。



開発者のテーマ



前述したように、Microsoft Igniteはさまざまな対象者向けに設計されています。 これの証拠は、開発に専念する多数のトピックです。 ビジネスアプリケーション、モバイルデバイス用アプリケーション、ユニバーサルアプリケーションなどの開発に焦点を当てます。 専門家がVisual Studio 2015とASP.NET 5、AngularJS、NodeJS、およびXamarinについて話します。 また、Office 365およびSharePoint用のアプリケーション開発専用のブロックも用意されます。 さらに、クラウドアプリケーションとソリューションの効果的な開発のためにローカル環境をセットアップするためのさまざまなオプションについて説明します。

実験室の仕事も興味深いです。 参加者は自分自身に没頭し、OfficeおよびSharePointのアプリケーション機能について詳しく知ることができます。 SharePoint Hosted Appsを使用して、開発プラットフォームOffice 365に精通することが可能になります。 ラボでは、参加者はWindows 10向けのアプリケーションを作成したり、Apache Cordovaを使用してクロスプラットフォームモバイルアプリケーションを作成したりできます。これはすべてVisual Studio 2015で行います。また、Application Insightsを使用して、アプリケーションの改善、潜在的な問題の特定と解決の可能性についても話し合います。



これで、Microsoft Igniteセッションの簡単な概要で締めくくりたいと思います。 すべてのセッションの詳細については、会議のウェブサイトをご覧ください 。 Microsoft Ignite登録も利用できます。



会議セッションのリストは、ITプロフェッショナル、実装スペシャリスト、ビジネス担当者、および開発者に役立つ興味深いコンテンツを約束します。 会議の前に2か月が残されており、Microsoft Igniteが最大規模のテクノロジー会議の最高のものを組み合わせることができたかどうかを確認できます。



それまでの間、どのMicrosoft Igniteセッションでどのトピックに参加するのが面白いかについての意見をお聞かせください。
