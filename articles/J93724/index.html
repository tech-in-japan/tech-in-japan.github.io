ブリャンスク市での最初のWeb開発者会議



こんにちは。

結局、5月と6月はさまざまな種類のイベントが非常に豊富です。 したがって、おそらく、私は別の会議-Bryansk Web Dayについて簡単に話します。



指揮を執るというアイデアは自然に生まれました。最初は友人と一緒になって、Webアプリケーションの開発について話したかっただけです。 しかし、彼らは、参加者の輪を広げてみませんかと考えました。 そして数週間後、イニシアチブチームが結成され、彼らは名前を思いつき、聴衆を集め始めました。



これはブリャンスクにとって最初のそのような会議であり、私たちは試験的な打ち上げを実施しており、成功すると思います。

プレゼンテーションについては、技術レポートのみに限定することはせず、参加者にインターネットプロジェクトの簡単なプレゼンテーションを行う機会を提供します。 将来どうなるか見てみましょう。

プログラムの編成はまだ完了していないため、すでに決定されているスピーチのトピックをいくつか紹介します。



  1. 「ウェブサイトとそのアプリケーションのパフォーマンスを最適化する方法」、Andrei Petrenko（シニアソフトウェアエンジニア、ラブプラネット、ブリャンスク）
  2. 「インターネットでビジネスを行うための機能」、Dmitry Kirilkin（Infinity Studio web-studio、Bryansk）
  3. 「PHPでのWebアプリケーション開発における設計とリファクタリング」、Zorin Kirill（ Quaint 、Bryansk）
  4. 「Web開発におけるインタラクティブグラフィックス、形式と例」、Andrey Kholmanyuk （ Quaint 、モスクワ）




UPD：プレゼンテーションを作成したい場合は、これについて非常に満足しています。 現時点では、プログラムにはまだ空き時間があります。



日付： 2010年6月5日。 登録は10:30から始まります。

場所： ブリャンスク、クイビシェフ通り、2。トカチョフ兄弟博物館の会議場。
