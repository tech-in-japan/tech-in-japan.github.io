Opera Mobileレポート：2009年7月









休暇期間はモバイルインターネットにどのように反映されましたか？



2009年8月25日-オスロ、ノルウェー

Opera Miniユーザーは、アルバやオーストラリア、トルクメニスタン、南極で休暇を過ごしましたか？ 今年7月のモバイルインターネットの使用に関するOperaのレポートは、Opera Miniブラウザーを使用してダウンロードされたWebページの数と、ユーザーが休暇を過ごした最も人気のある休日地域との関係を追跡しています。



Operaからの月次レポートは、モバイルインターネットの世界的な傾向に関する情報を提供します。 完全なレポートはwww.opera.com/smwで入手できます（英語で入手可能）。 世界的な動向とさまざまな国でのモバイルWebの使用に関するより詳細な情報に加えて、現在のレポートは、その国で行われた選挙に関連したイランのユーザーの活動も反映しています。



マスタデータを報告する

-2009年7月、Opera Miniユーザーの総数は2,910万人に達しました。これは6月より9.9％多く、1年前の結果は145％を超えています。 これらの数字は、携帯電話へのインストールにOpera Miniを選択したユーザーの数を反映しています。



-2009年7月、2,910万人のユーザーが120億以上のWebページを閲覧しました。 これは、2008年6月より15.4％、2008年7月より223％増加しています。



-7月にユーザーが生成したトラフィック量は合計1億8700万メガバイトで、1か月前より11％増加しました。 Opera Miniでは、データを最大90％圧縮できます。 情報が圧縮されていない場合、ユーザーは1.7ペタバイトのデータを携帯電話にダウンロードする必要があります。 2008年7月と比較して、トラフィックは223％増加しました。



-7月のイランからのブラウザのダウンロード数が大幅に増加しました。 ただし、先月のユニークユーザーの数は、6月に比べて32％しか増加しませんでした。 さらに、この指標は年間で22.4％減少しました。 この地域のユーザー数の同様の減少は、サウジアラビアとイスラエルでのみ確認されました（各国で10％ずつ）。 それでも、イランは中東の地域の中で最も多くのOpera Miniユーザーを抱えています。 イランでは、1年間に表示されるWebページの数は119から188に増加しました。



-Opera Miniユーザーはトルクメニスタン、南極大陸、アルバ、アンドラ、ブルンジ、オーストラリアに旅行しました。 これらの地域では、7月のユニークユーザーの数がOpera Miniディストリビューションのダウンロード数を上回りました。



-Opera Miniのユニークユーザー数の上位10か国（降順）：ロシア、インドネシア、インド、中国、ウクライナ、南アフリカ、アメリカ、ナイジェリア、ドイツ、ポーランド。



-ロシアでOpera Miniを使用して表示されたページ数は2008年7月と比較して135.1％増加し、ユニークユーザー数は63.5％増加しました。 平均して、各ユーザーは1か月あたり415ページを閲覧しました。

Operaの最高経営責任者であるJon von Tetzchnerは次のように述べています。 -最適なモバイルブラウザを使用すると、ユーザーがどこにいても携帯電話からのWebページのダウンロードがさらに促進されると考えています。 したがって、誰もがインターネットにアクセスできるようにするには、ほぼすべての携帯電話で機能するソリューションが必要です。 だからこそ、世界中のユーザーが高く評価しているブラウザであるOpera Miniを作成しました。」

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
