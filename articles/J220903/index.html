Eversync-iOSおよびWindows Phoneサポート付き

EverSyncアプリケーションがiPhoneおよびWindows Phoneユーザー向けに利用可能になったことをお知らせいたします。 このアプリケーションではブックマークを管理できるだけでなく、everhelper.meサービスを通じて他のデバイスやブラウザーとブックマークを同期できることを思い出させてください。 EverSyncは、人気のFVDスピードダイヤルの公式クライアントでもあります。 カットの下で、アプリケーションに関するいくつかの詳細。



EverSync for iPhone-itunes.apple.com/us/app/eversync-bookmarks-and-dials/id857281924?l=en&ls=1&mt=8



EverSync for Windows Phone-www.windowsphone.com/en-us/store/app/eversync/c014ecd7-6cd8-4026-b24f-23447094ff10











アプリケーションには、ブックマークとスピードダイヤルの2つのモードがあります。 ブックマークモードでは、ブックマークを表示、作成、編集できます。 Everhelper.meサービスにアカウントがある場合は、ブックマークを他のモバイルデバイスやChrome、Firefox、Internet Explorerのデスクトップブラウザーと同期できます。 便利な検索により、適切なリンクを数秒で見つけることができます。 Windows Phoneでは、目的のリンクをスタートメニューに添付することもできます。







短縮ダイヤル


スピードダイヤルモードでは、モバイルバージョンのダイヤルを使用できます。これはFVDスピードダイヤルと呼ばれます。 適切なダイヤルアップに必要なすべての機能にアクセスできます。 つまり、セルの作成と編集、サイトのスクリーンショット形式のプレビューの作成、および絶えず更新されるプレビューデータベースからのカスタム画像の選択です。 対話のオフラインバージョンを作成し、他のデバイスと同期することができます（everhelper.meのアカウントが必要です）







アプリケーションがいくつかのタイプの同期を実装することは注目に値します。つまり、データの結合、サーバー上のアカウントデータが消去されたときのデータのダウンロード、それに応じてデバイス上のデータが上書きされたときのダウンロードです。
