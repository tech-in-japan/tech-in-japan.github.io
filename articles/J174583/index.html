Windows Phoneでゲームを開発するための新しいツール



サンフランシスコで開催されたGDCカンファレンスでは、Microsoftプラットフォームでゲームを開発するための新しいツールについて多くの発表が行われました。 現在、DirectXなどの既存のツールに加えて、開発者はUnity 3D、Marmalade、Havokを使用できます。



Windows Phone 8は、ネイティブコードをサポートするゲームを開発するための優れたプラットフォームであり、iOSやAndroidプラットフォームなど、わずか数日でゲームアプリケーションを迅速に移植できます。 Windows 8プラットフォームとの統合により、開発者はほとんどのコードを再利用することもできます。

Windows 8の収益化プラットフォームには、アプリ内支払い（1回限りの購入シナリオと消耗品の購入の両方）など、ゲームアプリケーションでよく使用される重要な機能が多数あり、アプリケーションを収益化する十分な機会を提供します。 これは、既存の広告モデル、有料および試用収益化に追加されるものです。 これに加えて、PayPal、Airplay、およびオペレーター経由の支払いなどのサードパーティ支払いシステムを接続するオプションがあります。 後者のオプションは、オペレーターとクレジットカードのどちらから支払うかを選択できるWindows Phoneユーザーの75％以上に好まれています。オペレーターを介した有料ダウンロードの量は、クレジットカードでのダウンロードの3倍以上です。

Windows Phoneプラットフォームユーザーはプレイを楽しんでおり、ダウンロードの3分の1以上はゲームであり、ゲームはWindows Phoneストアからの収益の60％を生み出しています。 「Top50」リストの開発者の半数以上がゲームアプリケーションを作成しています。



新しいツールと新しいゲーム





新しいツールのおかげで、Windows Phone用のゲームアプリケーションの作成と移植がさらに簡単になりました。

  • Unity Tecnologiesは、 Unity3D for Windows Phone 8オープンベータプログラムの開始を発表しましたすべての開発者向けのオープンベータプログラムは、Windows Phone 8プラットフォームの新しいツールを試してフィードバックを提供し、開発を開始する機会です。 Drift Mania Champoinship 2 （Ratrod Studio）、 Fling Theory （コーディングジャースタジオ）-Windows Phone専用、 Orcs Must Survive （Echoboom）、 Temple Runなど 。
  • マーマレードは最近、すべての開発者が利用できるWindows Phone用SDKの新しいベータ版をリリースし、iOSおよびAndroidプラットフォームから移植された2つのゲームの発表が行われました。これは6th Planet （Monkube）とPropel Man （Spiel Studio）です。
  • Havocは、GDCカンファレンスでWindows Phone 8プラットフォームでのデモを行いました。Havocの従業員であるAndrew Bovelは次のように述べています。「当社では、私たちはすべて、 Windows Phone 8プラットフォームは、Havocを使用してこのようなアプリケーションを作成するために必要なすべての要素を提供し、開発者がハイエンドでエキサイティングなゲームを作成できるようにします。
  • もう1つの人気のあるオープンソースライブラリであるWindows Phone 8 用のCocos2d-xは、本日アップデートを受け取り、Cocos2d-x v2.1 APIと互換性があります。




すべてに加えて、GDCの別の発表についてお話ししたいと思います。 PowerAは、モバイルデバイス専用に設計されたMOGAコントローラを導入しました。 Windows Phone用の新しいMOGA SDKは、まもなく利用可能になります。





Windows 8のゲーム開発ツールに関する詳細が必要な場合は、Unityを使用したWP8用のゲームの作成に関するビデオを必ずご覧ください。
