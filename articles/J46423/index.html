仮想化された市場

仮想化、仮想化、仮想化。 この流行語は、すべての業界およびビジネス出版物のニュースヘッドラインで点滅し、ITプロフェッショナル向けのすべての会議で放送され、大企業のストラテジスト向けの修正案がアイデアになります。 以前は従来の消費者（ホスティングプロバイダーと巨大な分散LANの所有者）にのみ提供されていた製品と仮想化ソリューションは、現在、大衆に積極的に宣伝されています。



人事部のタクシー運転手と少女たちが仮想化について話し始めようとしているとき、情熱はすでにそのような重大なレベルに達しました。 マイクロソフトは、リソースにとらわれることなく、競合企業との真の軍拡競争を開始しました。 Hyper-Vをサポートする大規模なマーケティング攻撃は、無料のESXiサーバーを使用したVMwareの反撃に反対されます。



マイクロソフトからの情報の流れを理解するために、VMwareやその他の企業からの情報の流れを理解するために、私はいくつかのインタビューを受け、アウトリーチイベントの突破口に出席しました。 仮想化が直接あなたに関係していないが、私たちのように、あなたは彼女の周りの世界で誰が誰であるかを理解したい-私たちもこれをあなたのためにした。



カタロニアの首都の数十カ国からの参加者に話を聞いて、Microsoftの管理およびソリューション部門の責任者であるブラッドアンダーソンは、レドモンドでの仮想化の様子を非難した。 彼によると、あらゆる企業のITインフラストラクチャで仮想化を使用すると、1石で2羽の鳥が死亡します。



-サーバーおよびPCのハードウェアリソースのより完全で合理的​​な使用によりコストを削減します。 現在、同社によれば、平均的なサーバーはその容量の25％でのみ使用されています。 そして、複数の仮想マシンがすべてを占有できます。

-資格のある人員のコストを削減します。これは、仮想マシンの管理が簡単になるためです。つまり、非人道的な知識と能力を持つシステム管理者に対する大量の需要がなくなります。 ただし、彼らはアプリケーションホスティング業者とデータセンターの仕事に行くことができ、仮想化Zane Adam（Zane Adam）のマイクロソフトの主任戦略家であると安心しました。



マイクロソフトは、仮想化を可能な限り安価にしたいとアンダーソン氏は述べています。 ただし、ハイパーバイザー（コンピューターハードウェアリソースと仮想マシン間の分散を管理するためのプログラム）Hyper-Vは、ハードウェアベースの仮想化サポートを備えた64ビットマシン用にのみ開発されました。 ロシアの駐在員事務所の仮想化マネージャーであるVasily Malaninは、これは「ほとんどの顧客がすでにそのようなマシンを所有している」ためであり、「Hyper-Vで32ビットと64ビットの両方の仮想マシンで作業できる」と付け加えています車で。 ハードウェア仮想化をサポートしないシステムの場合、同社にはVirtual Server 2005 R2およびVirtual PC 2007の製品があり、64ビット仮想マシンを作成することはできません。



Virtual Server 2005は従来のサーバー用のサーバー仮想化ソリューションであり、これ以上開発されず、Hyper-Vを備えたWindows Server 2008に完全に取って代わります。 また、クライアントシステム用の仮想化システムであるVirtual PC 2007はさらに更新され、今後1年間で大幅に改善されます。



企業と消費者の仮想化市場の間で、境界線を引くことは困難です。 むしろ、サーバー仮想化とデスクトップ仮想化市場について話している。 技術的には、これらの方向は異なります。 そのような仮想化の使用について話すと、すべての仮想化製品を企業インフラストラクチャとホームPCまたはサーバーの両方で使用できます。



VMwareは、1999年に仮想化ソリューションで初めて大規模市場に参入しましたが、その人気のために一般向けに積極的に活動することにはほど遠い状態です。 このカリフォルニアの会社の野望は遠くまで広がっていますが。 現在の彼女のパッケージVMware Infrastructure 3は、仮想化自体を除くすべての機能が劣っていますが、実際はオペレーティングシステムです。



パッケージは、ESX Serverハイパーバイザー（および無料のアナログESXi）、vStorage VMFSファイルシステム、仮想マシンをサーバー間でシームレスに転送するためのVmotionユーティリティなどのコンポーネントで構成されています。



プラットフォーム市場でMicrosoftやLinuxと競合するVMwareは、それに応じて動作します。仮想化の利点のリストははるかに大きく、訓練を受けたより優れたリスナーに役立ちます。 サーバー仮想化の販売と実装の両方のリーダーとしてまだ素晴らしいと感じている同社は、ニッチゾーンも占有しています。 これは、PCや別のOSのシェルの下から「ゲスト」OSを実行する多くの一般的なVMware Workstationでよく知られています。たとえば、Fusionは同じことを行います。 すべてのニッチにおいて、VMwareはMicrosoftと中小企業の両方とうまく競合しています。



たとえば、ロシアの会社Parallels。 プラットフォーム分野でのそのような巨人との競争は「疑わしいアイデア」であると考えています。 そのため、彼らはVirtuozzo Containersパッケージとその分野で最も人気のあるWindowsベースのMacランチャーであるParallels Desktop for Macのオペレーティングシステムの仮想化に焦点を当てています。



上記のすべてに加えて、もちろん、2007年にCitrix Systemsが買収したオープンXenSourceハイパーバイザーなど、他にも多数のソリューションがあります。 これに基づいて、XenServer Enterprise Editionは非常に人気があり、多くの点でVMwareインフラストラクチャに劣らない機能を備えています。 しかし、あなたはそれについて聞くことはほとんどありません-Citrixマーケティングは、マス広告にスプレーすることなく、選択的かつ正確に機能します。
