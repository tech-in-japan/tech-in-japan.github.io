Nokia Mobile Web Serverは、常に必要なWebサーバーです。



Nokia Mobile Web Serverは、人気のあるApache Webサーバーのポートに基づいた、S60プラットフォームの電話用のWebサーバーです。

Apacheには、事前定義されたmodが付属しています：mod_alias、mod_auth、mod_autoindex、mod_dav、mod_dav_fs、mod_dir、mod_log_config、mod_mime、mod_rewrite、mod_setenvif、および（！）Mod_python。



Mod_pythonを使用すると、Pythonスクリプトだけでなく、Python Server Pagesも使用できます。

ポートはApache 2.0.52およびmod_python 3.1.3に基づいています



キットには、電話でWebサイトだけではないことを示す概念デモも含まれています。

次のことができます。

携帯電話をウェブカメラとして使用します。

近隣の他の同様のサイトを探します。

モバイルWebサイト（cellid）の場所を見つけます。

サイトにアクセスして、ウェブ経由でSMSkを送信します。 ブログの投稿を残すか、コメントを投稿してください。
