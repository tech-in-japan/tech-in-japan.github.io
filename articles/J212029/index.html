Toxは、プロジェクトの主な目標であるSkypeの将来の置き換えのために作成された分散型メッセンジャーであることを思い出してください。

-完全にオープンソース

-分散化された作業

-ブックマークと盗聴の欠如

-広告の欠如













メッセンジャーの現在のバージョンは次のことができます。



-メッセージを送信する

-チャットに参加する

-Windows / OS Xプラットフォームをサポート（既にコンパイルされたバージョン）

-電球メッセージルーティング（ Git ）



Toxは今後何ができるようになりますか？



-音声通話

-ビデオ通話

-ファイル転送（GUIでの実装、カーネルはすでにその方法を知っています）
