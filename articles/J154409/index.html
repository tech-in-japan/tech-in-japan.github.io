HPワイヤレスネットワーク

現在、企業ネットワークはラップトップからタブレット、スマートフォンに至るまで、さまざまなワイヤレスデバイスでますます使用されています。



これらの条件下では、モバイルWiFiデバイスから企業データへのアクセスの問題を最も簡単かつコスト効率よく解決する方法に関して疑問が生じます。 HPが提供する1つのソリューションは、HP MSMアクセスポイントです。 また、HP IMCネットワーク管理ソフトウェアは、無線ネットワークセグメントと有線ネットワークセグメントの両方を集中管理できます。



HP MSMワイヤレスアクセスポイントシリーズは、1つのモデル（IEEE 802.11n / a / b / g）と2つの無線モジュール（802.11 n / a / b / gおよびa / b / g）で表されます。 MSM-460およびMSM-466モデルは3x3 MIMOワイヤレスデータ送信を使用しているため、市場で最も生産性の高いデバイスに取って代わることができます。 IP67 / NEMA 4X耐候性評価（MSM-466R）を使用して、アクセスポイントの設計を屋内および屋外の両方で使用できます。 アクセスポイントには、内蔵または取り外し可能な外部アンテナを装備できます。 MSMアクセスポイントを使用するには、イーサネットケーブルが1本必要です。イーサネットケーブルは、Power over Ethernet（PoE）テクノロジを使用して給電することもできます。 PoE標準IEEE 802.3afおよびIEEE 802.3atがサポートされています。 （ 読む 。）



HP MSMシリーズコントローラーは、アクセスポイントとともに、集中管理、アクセスポイント設定の自動ネゴシエーション、およびユーザーモビリティの提供に使用されます。 コントローラアーキテクチャにより、802.11a / b / gアクセスポイントの使用と802.11nへの移行が可能になり、承認されたトラフィックを転送してコントローラをバイパスすることで新しいコントローラをインストールする必要がなくなります。 ロールと識別情報に基づくアカウントプロファイル、およびQoS、認証、暗号化、VLAN設定を備えたVSCポリシーがサポートされています。 コントローラは、10個のアクセスポイント（MSM-710）から800（MSM-760 / 765）および最大2000（Mobility Traffic Manager）のネットワークで使用できます。 内部および外部の認証、許可、アカウンティング（AAA）サーバーがサポートされています。 コントローラには統合されたステートフルファイアウォールがあり、各ユーザーにVLANを割り当てることができます。 （ 読む 。）





HPワイヤレスネットワークの設計を簡素化するために、 RF Plannerソフトウェアが使用されます。



このソフトウェアは、アクセスポイントと典型的な建築材料のすべての特性を備えているため、機器の場所で測定することなくワイヤレスネットワークの予備計画を立てることができます。



HP MSM415ワイヤレスセキュリティスキャナーとHP RFマネージャーシステムは、24時間365日のIEEE 802.11a / b / gおよび802.11nワイヤレスセキュリティの脅威を防ぐIDS / IPS機能でワイヤレスネットワークを保護します。



MSMアクセスポイントは、AeroScoutロケーションソフトウェアと連携して動作します。詳細については、RTLSキーワードを参照してください。



WiFiネットワークの問題をトラブルシューティングするには、 個別のスペクトルアナライザーを使用することをお勧めします 。
