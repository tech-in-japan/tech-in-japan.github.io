Veeam ONE 6.5：VMware vSphereおよびMicrosoft Hyper-Vの新しい運用監視機能

この記事では、新しいバージョンのVeeam ONE 6.5の機能について説明します。VeeamONE 6.5は、VMware vSphereおよびMicrosoft Hyper-Vに基づく仮想化システムのレポートおよび運用監視用の製品です。 まず、Veeam ONEを必要とするユーザーとそのタイミングの概要。



Veeam ONE At A Glance


サーバー仮想化の急増に伴い、仮想環境の効果的な管理がますます重要になっています。 これは、次の理由で発生します。

  • 仮想マシンは、ミッションクリティカルなワークロードをサポートするようになりました。
  • ITプロフェッショナルは、IT部門の責任者から企業幹部まで、幅広い人々が仮想化を使用するプロセスとその価値をビジネスに文書化する必要があります。
  • リソースを効率的に割り当て、仮想マシンインフラストラクチャの成長を制御する必要があります。


Veeam ONEは、システム管理者がこれらすべての問題を単一のシンプルで効果的なツールで解決するのに役立ちます。 管理者は、仮想環境のパフォーマンスと可用性に影響する問題について最初に学ぶ必要があります。 200を超える組み込みアラートと広範なナレッジベースにより、製品ユーザーは問題に関する通知を受け取るだけでなく、トラブルシューティング、問題の根本原因の修正、および迅速な解決に必要なすべての情報を取得できます。





国内のポリシー、規制、および常識には、絶えず変化する仮想環境の完全かつ最新のドキュメントが必要です。 環境を管理するには、ワークロード、リソース、およびそれらの消費に関する情報が必要です。 仮想インフラストラクチャのパフォーマンスを瞬時に把握する必要があります。 Veeam ONEは、組み込みレポート、ダッシュボード、自動レポートなど、これらすべてを提供します。 必要なすべての情報をすばやく取得できます。



Veeam ONEを使用すると、仮想インフラストラクチャの技術的およびビジネス指向の両方の表現を作成できます。 これにより、組織単位、地理的位置、部門、サービスレベル、サービスタイプ、および定義したその他の基準ごとに論理的に結合された、仮想化サーバー、クラスター、仮想マシン、データウェアハウスの動的グループのレポートを監視および作成できます。



Veeam ONEは、1つのソリューションでVMwareとHyper-Vをサポートします。これにより、個々のツールの展開と管理に時間と労力を浪費することなく、レポートを監視および作成できます。



Veeam ONE 6.5の新しいバージョンのリリース


バージョン6.5以降、Veeam ONEの機能は、監視およびレポートの対象としてバックアップインフラストラクチャをサポートするとともに、システム管理者が仮想環境のパフォーマンスと可用性を改善できるようにするその他の多くの改善により拡張されています。 Veeam ONEは、スタンドアロン製品として、またはVeeam Backup Management Suite 6.5の一部として利用できるようになりました。



次に、Veeam ONE 6.5の新しい主要機能を検討します。



vSphere 5.1、Windows Server 2012、およびWindows Server 2012 Hyper-Vのサポート


バージョン6.5は、SMB共有、高可用性（HA）SMB共有、CSV 2データストアなどの新しいHyper-Vオブジェクトを（メトリック、イベント、アラートの標準セットに関して）完全にサポートします。 IOPS I / Oパフォーマンスを示すデータウェアハウスの新しいメトリックのサポート。



さらに、バージョン6.5には、新しいイベントとvSphere 5.1メトリックに基づいたテクニカルサポートナレッジベースの記事で説明されている問題のすぐに使用できるアラートが含まれています。



Veeam Backup＆Replicationとの統合


バージョン6.5では、単一の管理コンソールから、つまりVMware vSphereおよびMicrosoft Hyper-V仮想インフラストラクチャの監視に使用されるのと同じユーザーインターフェイスから、レポートを受信し、Veeam Backup＆Replicationバックアップインフラストラクチャの主要イベントおよびインジケータのリアルタイム運用監視を実行できます。 新しいビューを使用すると、異なるバックアップサーバー上のバックアップタスク、レプリケーションタスク、およびバックアップ検証タスクのステータスを同時に監視できます。



Veeam ONE 6.5には、ユーザーがバックアップインフラストラクチャの主要なイベント（バックアップリポジトリの空き容量の不足、バックアップサーバーのステータス、タスクステータスなど）についてすばやく学習できるようにする、Veeam Backuup＆Replicationキーパラメーターのすぐに使用できるアラートセットが含まれていますユーザーインターフェイスを使用して継続的に監視する必要なく、コピーします。



新しいバージョンから、安全でないデータを含む仮想マシンに関するレポートも製品で利用できるようになりました。これにより、バックアップタスクに含まれない仮想インフラストラクチャマシンを特定したり、主要なメトリックに関するレポートを作成したりできます。 レポートはスケジュールに従ってメールで受信できます。これにより、仮想インフラストラクチャデータの保護ステータスを自動的に定期的に監視できます。



リソース計画（キャパシティプランニング）


バージョン6.5では、負荷の傾向を（処理されたデータのサイズごとに）確認し、バックアップリポジトリのディスクストレージの空き領域を予測できます。 Veeam ONE 6.5では、仮想化サーバーの障害をシミュレートし、データウェアハウスI / Oのパフォーマンスを分析し、特定の仮想化システムのボトルネックを改善する方法に関する推奨事項を提供することもできます。 VMware vSphereのダッシュボードを使用すると、仮想化システムの主要なメトリックをオンラインで監視できます。 現在のバックアップタスクを実行するためのリソース不足が予測で示されている場合、製品はこれらのリスクについて運用上の通知を行います。
