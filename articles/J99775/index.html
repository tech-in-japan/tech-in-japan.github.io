今日から革命が始まる

数年にわたる開発、数千時間、100万ドルを超える自社資金により、MODx Revoution 2.0リリースを発表できることを誇りに思います。



新機能



MODx Revolutionは最新のMODxです。 MODxがリッチなWebサイトとリッチなアプリケーション（RIA）を構築するための優れたプラットフォームである理由はまさにそれであり、これにより、より多くのことを実行し、より良い結果を得ることができます。



MODx Revolution 2.0はさらに強力になりました。 負荷の高いサイトに対応でき、システムの単一インストールからマルチサイトとサブドメインを管理することもできます。



MODx Revolutionの新機能と改善点のリストは長すぎて、それらをここにリストできません。 したがって、私たちの意見では、あなたが愛するものだけを発表します。 MODxを自分で試すか、変更のリストを参照することで、さらに詳しく知ることができます。

  • MODx Revolution 2.0を使用して、より大きなサイトを作成できます。 パラメーターを微調整する機能を備えた新しいキャッシングシステムのおかげで、パフォーマンスが向上しています。 さらに、通常のキャッシュをMemcachedなどの大規模なキャッシュに置き換えることができます。
  • これで、システム環境設定、ユーザー、言語、セッションなど、MODxカーネルの任意の部分を拡張またはカスタマイズできます。 これにより、便利な製品サポートを維持しながら、安全に基本機能を拡張し、サードパーティアプリケーションと統合する機能が可能になります。
  • 開発者は、トランスポートパッケージを使用して、個別のアドオンまたはMODxディストリビューション全体を作成できます。
  • ほとんどすべての関係に基づいて、サイトのさまざまな表現を設定できます。 通信により、マルチサイトやサブドメインをネイティブに作成したり、ルートディレクトリ外でMODxカーネルを安全に起動したりする簡単な方法が可能になります。
  • 開発者は、カーネルトラッキングを介して、より優れたバグトラッキングおよびデバッグにアクセスできます。 システムは、ECHO、HTML、およびFILEを含むさまざまなレベルおよび選択された外部目標でのエラー検出を可能にします。 アドオンでこれを使用して、作業レポート、バグレポート、デバッグ、またはその他のニーズを監査することもできます。
  • 統合された簡素化されたMODxタグは、開発中のコードの可読性を簡素化する複数行呼び出しをサポートし、他のキャッシュされたタグを使用してキャッシュされたタグを呼び出し、MODx EvolutionのPHxのような任意の要素にネイティブおよびカスタム修飾子を接続します、しかし、はるかに効率的です。
  • 新しいMODxマネージャーは、MODx API、ExtJS（Sencha）、およびSmartyテンプレートを使用して開発され、新しいカーネルとAPIの機能を実証します。
  • ユーザーとマネージャーは、アクセス権とアクセス許可の詳細設定でグループ化できるようになりました。 ユーザーシステムを拡張して、APIを介してユーザーデータを保存する機能を備えた外部認証を可能にすることができます。 ユーザーシステムは完全に新しい-すべての権限はABAC（属性ベースのアクセス制御）セキュリティモデルを使用して確実に委任されるようになりました。
  • プロパティセットを使用すると、Managerの1つのセクションからどこからでも、またはサイト全体からでもスニペットを呼び出すためのプロパティを変更できます。 プロパティのセットを作成し、それらを割り当て、1つのセクションから管理します。
  • プロパティは、チャンク、TV、テンプレート、プラグインなど、すべての種類のアイテムで使用できます。
  • MODxディクショナリの管理とカスタマイズ-ショートカット、イベント通知など、ローカル、産業、および地域の用語の言語の意味。これらはすべてディクショナリマネージャです。
  • パッケージマネージャーを使用して、アドオンとカスタムビルドのMODxパッケージをインストールします。 パッケージマネージャーを開き、利用可能なアドオンを探すか、独自のアドオンリポジトリを使用します。
  • Static Resource Managerを使用して、ファイルシステム上のファイルを管理します（ルートディレクトリの外でも）。
  • Symlink Managerを使用して、サイトツリーの任意の場所に他のリソースの正確なコピーを作成します。
