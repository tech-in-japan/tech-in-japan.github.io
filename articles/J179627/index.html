超セキュアフラッシュドライブとハードドライブ

みなさんこんにちは！



habrはすでに超セキュアdatAshurフラッシュドライブについて同様のレビューを行っていましたが、今日はこのメーカーのセキュアデバイスのすべてのモデルについてお話ししたいと思います。



私はそのようなデバイスをかなり頻繁に注文するので、製造業者はこのレビュー専用のプロモーションコードを提供しました（ 5％割引-LS18413 ）。







したがって、このような情報キャリアの中心には、PINコードをダイヤルするためのキーボードがあります。 デバイスは、フラッシュドライブ（datAshur、4-32 Gb）およびハードドライブ（250-3000 Gb）などのさまざまなフォームファクターで提供されます。



datAshur







datAshurは、PINコードを入力するためのキーボードを備えたフラッシュドライブです。 このデバイスの非常に大きなプラスは、作業時に追加のソフトウェアが必要ないことです。Windows、Mac、Linux、Android、および通常のフラッシュドライブを理解する他のOSは、このデバイスを問題なく認識します。



暗号化は、256ビットAES暗号化をオンザフライで実行します。つまり、USBフラッシュドライブに到達するすべてのデータは既に保護されており、メモリを別の「通常の」USBフラッシュドライブにはんだ付けすると、コンテンツを表示できません。



フラッシュドライブの重量は25グラムで、寸法は80x20x10 mmです。現時点では平均的なフラッシュドライブより少し大きいですが、耐衝撃性と耐湿性に価値があります。



フラッシュドライブは、4、8、16、または32 Gbの4つの異なるボリュームで使用できます。 読み取り速度と書き込み速度は非常に良好なレベルです-読み取りでは最大27メガバイト/秒、書き込みでは毎秒27メガバイトです。



次に、ピンコードについて説明します。 USBフラッシュドライブをアクティブにするには、キーボードのピンコード（7〜15桁）をダイヤルし、キーが付いたボタンを押す必要があります。 コードが正しく入力されると、緑色のLEDが点灯し、コンピューターに接続できます。 それ以外の場合は、赤色のLEDが点灯し、PINコードの入力を再試行できます。 PINコードを10回間違えて入力すると、USBフラッシュドライブのデータは完全に消去されます。この場合、デバイスの完全なリセットのみが役立ちます。 メーカーは、管理者とユーザーの2種類のPINコードを提供しています。 管理者のPINコードは、たとえば、このフラッシュドライブを従業員に提供したいが、その場合はファイルにアクセスできる場合に役立ちます。

USBフラッシュドライブを使用した後は、単にコンピューターから切断するだけで十分で、自動的にブロックされます。ブロックするために追加のアクションは必要ありません。 これは税金を受け取った場合に便利です-USBフラッシュドライブを引き出すと心配することはできません。 内部には、USBフラッシュドライブがコンピューターに接続されたときに自動的に充電される内蔵バッテリーがあります。 バッテリーが切れた場合、メモリは不揮発性です。30分間PINコードを入力せずにUSBフラッシュドライブを接続するだけで、バッテリーが再充電されます。



このようなデバイスの価格は、ボリュームに応じて39〜99ポンド（1900〜4800ルーブル）です。 メーカーは3年間の保証を提供します。

手順は次のリンクで見つけることができます： http : //istorage-uk.com/documents/iStorage-datashur-User-Guide.pdf



それでは、ハードドライブのレビューに移りましょう。







ウィンチェスターは5つの異なるモデルで紹介されています。 それらの一部はUSB 3.0をサポートし、一部はSSDを使用します。



このようなハードドライブの動作原理は、上記のフラッシュドライブとまったく同じです。違いは、データの量とデバイスのサイズのみです。



diskAshurおよびdiskAshur SSD









これらのドライブは同じように見えますが、名前が示すように、1つの重要な違いがあります-その1つはSSDを使用します。 また、SSDなしのバージョンは128ビット暗号化で購入でき、256ビットより20ポンド安くなります。 SSDバージョンは256ビット暗号化でのみ利用可能です。 どちらのドライブもUSB 3.0をサポートしており、追加のソフトウェアは必要ありません。 電源はUSBポートから供給され、追加のアダプターは必要ありません。

ディスクのボリュームは、SSDのないバージョンでは250〜1500 Gb、SSDのあるバージョンでは64〜512 Gbです。 そのようなものは、SSDなしのバージョンでは119から269ポンド（5800-13000ルーブル）、SSDバージョンでは169から639ポンド（8200-31000ルーブル）かかります。 同様に、メーカーは3年間の保証を提供します。



製造元のWebサイトのドライブの説明： http : //istorage-uk.com/diskashur.phpおよびhttp://istorage-uk.com/diskashur_ssd.php



diskAshur-diskGの「削除された」バージョンも販売されています。 後者はUSB 2.0のみをサポートし、250〜1000 Gb（SSDなしのバージョン）およびSSDバージョンでは64〜512 Gbのボリュームで利用できます。 「兄貴」と比較して価格やその他の違いはありません。



diskAshur DT









最後に話したいデバイスは、デスクトップハードドライブです。 USB 3.0をサポートし、3つのフレーバーがあります：1 TB、2 TB、および3 TB。 179〜279ポンド（8600〜13500ルーブル）のドライブ料金がかかります。これは、提供されるすべてのデバイスのギガバイトあたりの最低価格です。 デバイスには外部アダプターから電力が供給されるため、作業にはコンセントが必要です。 以前のものと同様に、製造業者はこのデバイスに3年間の保証を提供します。
