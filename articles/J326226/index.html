IPとIT：メインについての永遠の議論

すべての著者は、Habréでの出版を目標としています。私の目的は、一見しただけでは見えないが 、経済、社会関係、そしてもちろん技術開発にとって非常に重要な問題をコミュニティに伝えることです。



私が何度も取り上げようとしたこれらのトピックの1つは、ITにおけるIPの問題です。



（ハブレだけでなく）受信したデータから判断すると、2つの反対の立場があります。「このスキームでは、税について私が知っている唯一のことは、私はそこに申告書を郵送し、四半期に一度送信する必要があることです6 ％ 他のスキームでは乗り越えられない困難があるかもしれません。ここではわかりません。 さらに、私は喜んでこれらの6％を差し引いて、誰からも隠れることなく白人の収入を得られるようにしています。」



2つ目は、税自体は問題ではないが、官僚制度（特に現場）、すべての拠出に対する保証の欠如（最近のものから-FIUや他の資金への凍結支払いがない場合の年金の凍結）、そして最も重要なこと-腐敗は「局所的」でさえ災害であり、それとそれに関連するセクターにとって、例えば、eコマースは一見思われるよりもはるかに深い。







10年前、最近、CDを販売していた人々は「海賊」と呼ばれていました。問題は、「結局、小柄な役人が勝ちました。 彼は同じキオスクの隣に、家賃を払わずに、現金デスクなしで（誰もチェックしなかった）海賊と、キオスクを開きました。 そして、ライセンス製品に対して罰金を科されました。 残念です。」 今、悪名高いAKITがシーンに入り、中小企業に真っ直ぐな汚れを投げかけたときに似たような画像が表示されますが、灰色の電話、冷蔵庫、およびこの色のその他の家庭用品は消えません。



そのような立場は共存できないように思われます。 しかし、彼らは仲良くなります。ある国では、一度に、一般に同じIPステータスになります。



いくつかの考えを言う前に、答えは「簡単な」統計です。



「ロシアの小企業の数は69.8千減少しました。ロススタットによると、2015年上半期には242.6千、2016年上半期には172.8千でした。実際、小企業のダイナミクスを評価してください。統計に基づいた企業は今ではほとんど不可能です...」（ イズベスティア ）。



2012年から2017年のサイトによると、IPの数は次のように変更されました。







2013年には、多くの人が「ダミー」はなくなったと考えています。 FIUへの支払いが急増しました。 はい、「これらは将来への貢献である」という議論がありましたが、予想される年金改革と貯蓄の凍結の後、多くの人々の希望は消え去りました。 はい、「カウント-年金受給者の質問は月40万で、この支払いは完全に控除可能」などの議論がありました。 しかし、一番下の行は、IPと猫を持っていたということです。 それは2013年に閉鎖され、「ダミー」ではなく、彼らは単に「余分な」17千人以上でした。



私が個人的に出会ったいくつかの例を挙げます。



  1. 女の子はフリーランサーです。彼女はテキスト（ほとんどが翻訳）を書くことでお金を稼いでいます。 収益はわずかです-約10,000ルーブル。 IPは、「ビジネスは喜びですが、余分なトラブルにより、最も単純なスキームでもこれを実行したいという欲求が殺された」ようなものでした。
  2. 5月から9月のシーズンにのみIPを使用した2人の男：サイトを介して観光客を引き付け、バイカル湖の東部で小旅行をしました。 「より多くの需要があり、「プライベート」からの仕事が「より多くの利益をもたらした」ので、彼らはIPを閉じた（それぞれが持っていた）。
  3. その他の例については、 記事の説明をご覧ください 。


私は誰かを責めたり、サポートしたりはしません。事実だけです。



質問は異なります。現在のIPは、既存の （法的）関係システムにとって非常に収益性の低い ビジネス形態です 。



いくつかのポイントを挙げます。



  1. 個人の起業家にとって、個人資本と運転資金は同じものです。 LLCの場合よりも、多くの人が何かを現金化する方がはるかに簡単です。 そして最も重要なこと-安い。 中央銀行はこれを好まない（WebMoney、Qiwi、Robokassaのケースを参照）。
  2. IPの開閉-ナシの殻をむくのと同じくらい簡単 。 誰でもできますし、 オンラインでもできます 。 破産手続および他の多くのこと-これは個々の起業家のためではありません。 私が何をしているのか分からない場合は、O。ティンコフとS.ガリツキーの興味深いインタビューを見ることを強くお勧めします。 この問題は、議論された連邦法54（ 「オンラインキャッシュデスクについて」 ）に関連して、2018年以降のIPで特に深刻になります。
  3. IP-予算は常に少なくなります 。 さらに、LLCとIEの罰金は異なり、LLCに対して多額です。


繰り返しますが、私はこれらの論文を評価する準備ができていません。これらは実践からの単なる事実です。 しかし、これは2012年から2017年までのロシア連邦における閉鎖IPの写真です。







なぜ今この質問を思い出したのですか？



3つの傾向に関連して：



  1. いわゆる「自営業市民」グループのセクターを「合法化」したいという公的機関の要望（ITにとっては、これはそもそもフリーランスです）。
  2. オンライン予約オフィス、 alex0nikの正しい発言によると、「大規模ネットワークは大幅に節約されました。 民間の起業家、悲しいかな、いいえ。少なくとも6,000 FN + 3,000 OFD + 1.5 EDS = 10.5の場合、実際はECLZ + TTOの前の状態です。 しかし、興行収入はもっと高価です。 そして、ネットは、キャッシュレジスタ、ドライブ、およびOFDのサービスのボリュームにより、大幅な割引を受けます。
  3. 最後に、テロとの悪名高い戦いに関連して（最近の出来事に照らして-明らかに強化されます）：上記のように-連邦法115およびその「子孫」のIPが礎石です。


したがって、明確にするために： USN 6％のIPは、すべての人にとって理想的ではありませんが、優れたフリーランスのソリューションです 。 しかし、国家機関は、ミニビジネスの実施をさらに（？）簡素化して、正確に「自営業の市民」の出現を固執します。



2015-2016年 細断されたビジネス：小規模および小規模の起業家精神の基準が変更されました：これを個々の起業家に関連付ける人はほとんどいませんが、それについて考えると、興味深い傾向がこれから続きます。これは、スコルコボ、イノポリスなどの傾向と矛盾します。 そうじゃない？



フリーランサーはビットコイン鉱山やその他の暗号通貨をますます深くマスターしているため、財務省は「マネーサロゲート」のステータスがデジタルであることを繰り返しています。 2018年の重要な出来事の直後（またはその前）に、金銭（ 電子に反対 ）は金融商品として認識されます。



私の意見では、カワカマスは多くのコミュニティが国家以外の金融分野に参入したいという欲求であるという、非常に奇妙な見解です。 がん-いくつかの政府機関が小規模ビジネスを「始めから終わりまで」コントロールしたいという要望。 そして、白鳥は静止することができない技術です。 そして、これまでのところ、3つのコンポーネントを最も裕福なIP / LLCパーツのみと組み合わせることが可能です-もちろん、収入は100ルーブル、そしてもちろん数千のフリーランサーです。 残りの部分、特に従業員にとっては、すべてが不条理な現実の映画に似ています。少なくとも「ボタン」がブロックされているか、電子マネーや暗号通貨を規制しようとする同じ間違いによって判断できます。 商品アグリゲーター、またはアリストテレスの友人のような技術的なものでさえ、真実は、あなたが知っているように、より高価です。

彼らはあまり与えません 不協和音。 この面でのIPの主な問題も言われています： 責任 。 LLCとは対照的に、リストにあることを除き、すべてのプロパティ。 人口の一般的な負債の負荷を考えると、最もバラ色の傾向ではありません。



この側面の本質は、IPフリーランサーは原則として融資を受けないという事実に要約されますが、IPは、たとえば電子商取引または同様の分野では非常に均一です。 しかし、IPフリーランサーは、営業担当者と「市場を回す」人なしでは生きていけません。顧客も資本も流れません。 もちろん、常にオフラインであることを正当化することはできますが、それでもサイトを作成し、それを宣伝し、アプリケーションなどに使用します。 オンラインが必要-先験的に。
