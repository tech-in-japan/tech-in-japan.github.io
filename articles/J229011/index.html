最近、 TV用Tizen SDKがベータ版でリリースされました。 プロトタイプTizen TVがサンフランシスコで開催されたTizen Developer Conferenceで公開されてからかなりの時間が経ちました-そして現在、開発者ツールが利用可能です。



Tizen TV SDK 1.0 Betaの主な機能：



-統合開発環境（IDE）

-Webシミュレーター

-デバッガー

-HTML、JS、CSSでのアプリケーションの開発

-Tizen TVのアプリケーション用テンプレート

-コンピューターでテレビのアプリケーションをテストするためのツール

-Webシミュレーターでのogg / ogv、MP4のサポート

-デバッガー（Webインスペクター）

-コード検査

-JSデバッガー

-ネットワークおよびリソースプロファイラー

-プロセッサー：デュアルコア1.5 GHz /シングルコア3 GHz以上

-RAM：2 GB以上

-画面解像度：1280 x 1024以上

-HDD：5GB以上

-OS（32ビットおよび64ビット）：

  • Windows 7（推奨）、Windows XP Service Pack 2以降
  • Mac OS X：Intelベースのハードウェア、OSバージョン10.6以降。
