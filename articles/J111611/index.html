Googleショートカットリンクには完全なAPIがあります

9月にリンク短縮サービスを開始したとき、 goo.glをアプリケーションやWebページに統合するためのコンパニオンAPIはありませんでした。 ただし、これに取り組んでいると述べたように、本日、 Google Code Labsで利用可能な本格的なgoo.gl APIをリリースしたことをお知らせします。 ドキュメントはGoogle Code Webサイトにあり、コード例は「 はじめに」セクションにあります。



このAPIを使用すると、開発者はWebインターフェイスを介して利用できるすべての高速でスムーズなスナックにプログラムでアクセスできます。 APIを使用してURLを短縮および拡張したり、履歴や分析を取得したりできます。 これらの機能は、TwitterやGoogleバズクライアントへのリンクを自動的に削減することから、統計やトラフィックデータの監視プロセスを開始することまで、幅広いアプリケーションで使用できます。 開始するには、Google APIコンソールをご覧ください 。



最速のリンク短縮サービスの1つにアクセスするためのAPIを提供できることを非常にうれしく思います 。 使いやすさの向上と、スパムや悪意のあるコンテンツを自動的に検出する機能の改善に引き続き取り組んでいます。 新しいAPIを使用して、今後のアクティビティでgoo.glの場所を見つけてください。 開発者の方は、 goo.gl APIのドキュメントをご覧になり、どのように機能するかを確認してください。
