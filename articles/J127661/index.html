Speed Install 2.0-Windows用リポジトリ

LinuxおよびMac用のリポジトリは古くから存在しており、これらのシステムのユーザーの生活をより快適にします。



リポジトリの利点：

-希望するカテゴリの複数の競合他社からのプログラムの選択、

-常に最新バージョンをインストールし、

-無限の「次へ」、「同意する」、「同意する」のない自動インストール。



リポジトリを作成する問題は、 トピックでdozentによって、そしてコメントで Goodkatによって既に提起されました。



プログラムSpeed Install 2.0 （正確には2.0.1.1531）はタスクを実行し、Microsoft製品に限定されません。



2008年、Google Chromeをダウンロードして、どのような動物であるかを確認したため、そのまま残っていました。 同じように、開発者が試用のためにプログラムを渡してくれたので、今ではそれなしでどのように進むのか想像できません。 あなたはすぐに快適に慣れます。

 







使いやすさ



インターフェイスは簡単に魅惑的です。 私は「マルチメディア」セクションに入り、100年間Foobar2000を更新していなかったことを思い出し、ボタンでそれを選択し、「インストール」をクリックしました...利益！



以前はプログラムを更新していませんでした。私の頭の中では、

-Googleにアクセスして、サイトを検索する必要があります（オプションとして、プログラムについてでサイトアドレスを検索します）。

-サイトで最新バージョンを見つけ、

-自分と比較して、

-Windowsシステムのビット数を覚えて、必要なビットを選択し、

-ダウンロード、

-実行し、「同意する」、「フォルダにインストールする」、「次へ」のボタンをクリックします。



これには時間がかかるだけでなく、注意も切り替える必要があります。 そして、ここではすべてが簡単でシンプルです。 Speed Installがすべてを行います。

 







カテゴリーから探す



プログラムの検索も非常に直感的に行われます-カテゴリ別に必要なものを簡単に見つけることができます。 根拠がないように、おそらくスクリーンショットを添付します。





 







機能的



怠け者なので、機能性は見ていません。 直観的にすべてがすぐに機能した場合、なぜこれを行うのですか？ しかし、私は記事を書くことを引き受けたので、私は見ることにしました。 無駄ではありません。



まず、更新されたプログラムのリスト（オフラインではない）を作成できることが判明しました。



第二に、インストール後にデスクトップからショートカットを削除できる設定では、きれいなデスクトップが大好きなので私にとって重要です。







そして、プログラムの残りの部分は、その目的の純粋さと明快さを楽しみ続けています。

 







プログラムに含まれるもの



次に、プレスリリースからデータをコピーして貼り付けます。



-データベース内の何百ものプログラム。

-完全自動のサイレントインストール。

-常に更新されるアプリケーションデータベース。

-プログラムの最新バージョンをインストールします。

-多言語インターフェース。

-32/64ビットインストール。

-オフラインインストーラーを作成します。

-プログラムリストを作成、編集します。

-クイック検索。

-使いやすいインターフェイス。

-すべてのプログラムは公式サイトからのみダウンロードされます（ウイルスはありません）。

-HTML / XMLレポート。

-インストール中のショートカットの作成を制御します。

-プログラムで実行可能ファイルを作成します。

-依存関係制御（.Net、Javaなど）。



定期的な更新について確認します。 この記事を書いている間、バージョンは2.0.1.1533に更新されました。

 







計画されていること



-ネットワーク上のコンピューターへのリモートインストール。

-ソフトウェアを追加する機能（スケジュールに基づいたダウンロードとインストール）。

-インストール済みプログラムのリスト。

-更新プログラム（アップデーター）のリスト。

-評価センター（プログラムを評価する機能）。

-アンインストーラープログラム。

-プログラムの基盤が拡大します。



Yandexマーケットなど、プログラムに関する肯定的および否定的なレビューを読むことができてうれしいことを自分から付け加えます。

 







設計



プログラムの設計はおそらく少し時代遅れです。 2-3年。 ヴィンドゥソフスキーの自然はそれ自体が感じられます。 プログラムの公式ウェブサイトでさえ、よりモダンに見えます（もちろん、BBフォーラムを除く）。



ただし、Webデザインの人間として、デザインよりもユーザビリティの方が重要だと考えています。 この点で、開発者はタスクに対処しました。

 







料金



プログラムは無料ではありません。 ただし、支払いを行うと、ウイルスやトロイの木馬ではなく、要求どおりにインストールされていることを確認できます。
