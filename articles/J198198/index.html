既存の政府ポータルからのオープンデータ



公的機関が開示すべきオープンデータについて話すとき、ほとんどの場合、必要な形式のデータがないという事実に遭遇します。 または、少なくとも、州によって作成されたデータベースとレジストリを維持する必要がある人は言ってください。



そして確かに、たとえば郵便局のセクションの境界のデータなど、単に存在しない可能性が最も高いデータがあります。 たとえば、 オープンデータの国勢調査などの世界では、 Open Knowledge Foundationは最も重要なデータセットの1つですが、ロシアでは完全に存在しません。



しかし、このコインには別の側面があります。 市民がマシンで読み取り可能な形式でアクセスできない大規模なデータベースを収集する状態ポータルとプロジェクトがすでに作成されています。 そして、これは、当局がデータがないと言うことができない場合にまさに当てはまります。唯一の問題は、データの品質と便利な形式での可用性です。



さらに、国の資金で作成およびサポートされているポータルとプロジェクトについてのみ記述します。 そして、多くの有用な情報を含むも​​の。

リンク： http : //gosuslugi.ru





州が最優先事項の1つとして宣言していることは、トピックに関するオープンデータよりも明らかであると思われます。 公共サービスのオープンデータの可用性は、たとえば検索エンジンがそれらを配信インターフェイスに埋め込んだり、特別なモバイルアプリケーションを開発したり、より便利で実用的なインターフェイスの下で公共サービスのサイトをリメイクしたり、地域のオンラインプロジェクトについてサービスが提供される当局の場所は非常に便利です。



ポータルTakZdorovo.ru




リンク： http : //takzdorovo.ru





多くの人は、確かに、保健社会開発省の命令によって作成されたこのプロジェクトを覚えています。 それは市民のための多くのサービスを作り、興味深い情報を収集し、ソーシャルネットワークを組織することになっていた。

個人的には、私はこのプロジェクトが好きではないことを認めています。ビジネスや公共団体がうまく作成できるプロジェクトを作成しようとするのは州のビジネスではないと思います。

しかし、いずれにせよ、データベースの形で提示できる多くの情報がポータルで収集されました。 納税者のお金によって収集された情報。



特に、これらはデータです：レシピ、演習、施設の住所、製品のカロリー表など。

この情報を使用できるプロジェクトや、このデータで作成できるプロジェクトは多数あります。 そして、誰が知っているか、彼らは以前に国のお金で行われたものよりもはるかに良くなっていなかっただろう。



LaborAll


リンク： http : //trudvsem.ru





情報ポータル「ロシアで働く」では、最初に保健社会開発省、そして現在は労働と雇用のための連邦サービスが、ロシア中に集められた空席と履歴書の巨大なデータベースを置いています。 このポータルは、そのようなポータルの市場が長い間形成されておらず、国が作成したポータルがこの分野の商業ポータルとどのように競合できるかが不明な場合、さらに価値があります。



ただし、そこからのデータは確かに有用です。 州が収集したデータのAPIまたは定期的なダンプを提供した場合、これにより、非常に多くのスタートアップや個々の開発者向けの求人検索サービスを作成できる可能性が大幅に高まります。







間違いなく、長年にわたって私たちの州は、私たち全員にとって有用であり、私たちが受け取りたいデータがある、より多くのプロジェクトとポータルを作成してきました。 それらを開く方法は？

まず第一に、公開リクエストがそれらに表示されることが必要です。 個人的には、将来の会議でオープンデータ評議会を開催する予定です。



情報文化に代わって、これらのポータルを担当する当局に書き込み、データダンプを公開し、オープンライセンスの下でデータ自体を公開します。



どうすれば役立ちますか？


最初に、それについてより頻繁に書いて話します。 これらすべての機関に対して明確な要求があるはずです-「オープンデータがなければプロジェクトは必要ありません」。



第二に、直接訴え。 私たちが彼らに書く手紙に加えて、あなたは常に彼らからデータを要求する必要があります。 彼らが拒否した場合、我々は政府とオープンデータ評議会への問い合わせを通じて彼らから直接要求する機会があります。



そして、もちろん、オープンデータが必要な他の重要な州のポータルを知っている場合は、コメントでそれらについて書いてください。
