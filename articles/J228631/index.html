レンガ造りのゲーム



ハブにブリックゲームをプレイしたことがない人がいるかしら？ 一般の人々では、このおもちゃはテトリスと呼ばれていました。 2つから無限のゲームまで含まれています。 最近判明したように、ゲームの数を増やすためにテトリスを終了することができます- このハブを読んでください 。



2本の指電池で駆動します。 数多くのフォームファクターがあります。現在、中国人は他のコンソールや携帯電話用にスタイルを整えています。 そして、もちろん、1990年代にロシアで非常に人気のある古典的な「レンガ」。 まだ販売中。



おもちゃの人気は他のコンソールのために低下しています。







セガ遊牧民



以前の記事でSega Game Gearについて話しましたが、Sega Nomadは見逃しました。 読者の一人が述べたように、現在、このプレフィックスは珍しいコンソールの中で最も人気のあるものの1つです。



セガノマドは1995年に180ドルでリリースされました。 約100万部のみが販売されました。



コンソールには、3.5インチのカラーディスプレイと6本の指電池用の取り外し可能なコンパートメントが装備されています。



その主な機能の1つは、彼女専用に作成されたゲームがないことです。 代わりに、通常のSega Mega Drive / Genesisホームコンソールのカートリッジをコンソールに挿入する必要がありました。 多くのセガオーナーがこれを夢見ていたと思います。 さらに、このセットトップボックスをテレビに接続し、将giからの通常のジョイスティックを特別なポートに挿入して、一緒にプレイすることができました。 興味深いことに、ディスプレイ上のテレビに接続していても、ゲームを見ることができます。















バーチャルボーイ



ポータブルコンソールには、任天堂が1994年にリリースしたVirtual Boyが含まれます。 このコンソールの場合、結局のところ、テーブルが必要でしたが、移動するときにはそれを使用できません。それは周辺視力を妨げるからです。



これは、3Dグラフィックスが可能な最初の任天堂コンソールでした。 「バーチャルリアリティヘルメット」に似たテクノロジーを使用します-デバイスに顔を浸し、2つのモノクロ（黒と赤）プロジェクターが各目で別々に画像を送信します。



システムの解像度は384x224ピクセルですが、この解像度のフルLEDマトリックスは含まれていません。 代わりに、2つの1x224ルーラーが高速で移動します。



システムに障害が発生することが予想されていました。 ユーザーのほとんどは首が痛い。 はい、黒と赤の画像は喜ばれませんでした。















Game.com



Game.comのポータブルコンソールは1997年に登場しました。 300,000部のみが販売されました-まあまあの結果。 しかし、それにもかかわらず、このおもちゃには多くの興味深いものがありました。



1）コントラストを調整できる白黒ディスプレイをタッチします。

2）カートリッジ用の2つのスロット。

3）モデムに接続するためのポート。

4）コンセントに接続する機能。



キットにはスタイラスが付属していました。フロントパネルの下にいたのは彼です。







コンソールの英語の概要。







このコンソール用にリリースされたゲームには、Duke Nukem 3D、Mortal Kombat Trilogy、Resident Evil 2がありました。このビデオでは、ゲームプレイはResident Evil 2です。はい、白黒ディスプレイで。











DreamCastのビジュアルメモリユニット



1998年に、DreamCastと呼ばれるセガの5番目のプレフィックスがリリースされました。 Visual Memory Unitが利用可能です。 このカードはあまり一般的ではありません。48x 32ドットのモノクロLCDディスプレイが搭載されており、ミニゲームを直接プレイできます。 「バイオハザード」の場合、このマップにヘルスが表示されたため、ゲーム中に「武器」メニューに移動する必要はありませんでした。 VMUはジョイスティックに直接挿入されました。

ソニーのポケットステーション



メモリカードを別のゲームデバイスとして使用する別の例は、1999年にソニーがリリースしたPocket Stationデバイスです。



デバイスには、液晶ディスプレイとスピーカーが装備されています。 時計とカレンダーもあります。 ユーザーは、赤外線ポートを使用して情報を交換できます。 Pocket Stationのおもちゃはゲームディスクから記録されます。



モノクロディスプレイの解像度は32 x 32ピクセルです。



Pocket Stationは日本市場でリリースされたもので、ヨーロッパやアメリカでは登場しませんでした。



猿とかわいい広告。











ネオジオポケット



Neo Geo Pocketポータブルコンソールは、1998年に日本および他のいくつかのアジア諸国でリリースされましたが、ヨーロッパやアメリカの市場を見たことはありません。 彼女は、彼女がカラーバージョンに置き換えられた1999年まで生きました。



プレフィックスは、6.14 MHzの周波数で動作する16ビットToshiba TLCS-900Hプロセッサを使用しました。 ディスプレイの解像度は256x256ピクセルでした。







ネオジオポケットカラー



上で言ったように、Neo Geo Pocketはカラーバージョンに置き換えられました。 両方のコンソールの合計請求額は約200万を売りました。



カラーバージョンは、6.14 MHzの周波数で動作する16ビットToshiba TLCS-900Hプロセッサを使用します。 カラーディスプレイの解像度は160 x 152ピクセルです。







コンソールで最も人気のあるゲームの1つはSamurai Shodown 2です。







テレビ広告。







面白いスローガン-「ポケット力を手に入れてください。」







ワンダーダーワン



WonderSwanポータブルコンソールは、1999年に日本でリリースされました。 彼女はゲームボーイと競争することになっていた。 WonderSwanの開発者である横井軍平が任天堂のオリジナルゲームボーイを作成したのは興味深いことです。つまり、彼は自分自身で「市場を分割」しました。



興味深い機能の1つ-ゲームは、画面の垂直方向の配置と水平方向の両方で作成されました。







ワンダースワンカラー



Neo Geoと同様に、WonderSwanコンソールはすぐにカラーバージョンに置き換えられました-2000年にリリースされました。 興味深いことに、人気のピーク時には、2つのバージョンのWonderSwanが日本のゲーム市場の8％を獲得し、任天堂と本当に競合していました。



この人気の理由は、まず、価格でした-それはわずか約60ドルです。 次に、Squareとのコラボレーションは実を結びました。ファイナルファンタジー、ファイナルファンタジーII、ファイナルファンタジーIV、ファイナルファンタジーレジェンドがコンソール用にリリースされました。
