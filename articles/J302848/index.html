資金調達



ゲームを開発するための資金はどこで手に入れましたか？

2年半前、私たちは熱意をもって、 TurnOnゲームの初期のプロトタイプに長く一生懸命取り組み始めました。2014年の夏、Microsoft Imagine Cup国際大会で優勝し、 5万ドルの助成金を受け取りました。 私たちがそれをどのように行ったかについては、ハブでこの記事のビデオを見ることができます



ちなみに、ドルが38ルーブルに相当する「素晴らしい」時間であり、当然このお金はある時点で使い果たされましたが、蓄積された負債を分配し、必要な機器を購入し、小さいながらも快適なオフィスをかなり長い期間借りることができました。 したがって、開発の残りの部分では、私たち自身の節約と非常に禁欲的なライフスタイルのために、プロジェクトを非常に強化し、資金を提供しました。



外部投資をせずにゲームを作ろうとしたのはなぜですか？



多くの場合、若い会社（チーム）が最初のプロジェクトに投資したばかりの場合、状況は次のように表されます。



投資は借金であり、投資家はそれを一定の割合で返済することを期待しています。

借りる場合は、一定の時間が経過すると、より多くの収益を得て、このお金と利子を返還できるようにする必要があります。 このためには、少なくとも市場の仕組みを理解する必要があります。



すべてを計算し、何に頼るのかを考える方法（ Zhenyaのソファ分析 ）



だから、想像してみてください。あなたはロシアの初心者向けゲーム会社で、興​​味深いプロジェクトを持っています。

最初に行う必要があるのは、すべてをほぼ計算し、最終的に財務の観点から会社がどのように機能するかを把握することです。 まず、あなたとあなたの会社に到達する収入の割合を計算しましょう：



  1. あるサイトでゲームを販売していると想像してみてください（これは重要ではありません）- インターネットには、通常20〜30％- 通常は30％の 噂がある
  2. あなたが簡易課税スキーム（STS）で運営している会社であるとします-つまり、 6％の所得税を支払う必要があることを意味します
  3. お金はあなたの会社の口座に入金されており、現金を得るには少なくとも2つの方法があります：給与を支払うか、配当を引き出す
  4. 現在の状況では、給与を支払うために（ 州からの特権を使用しないことを意味し、FIGSを取得します）、あなたの会社は約43％（個人所得税の13％、年金基金の22％、5.1％の蜂蜜）を源泉徴収しなければなりません.insurance、2.9％社会保険）。 ゲームの販売からの利益の合計100％が賃金の形であなたに届きます-37％ 。
  5. 2番目のオプションは、投資家と共同創業者への配当です。 税法によると、13％の配当が配当に対して支払われました。 配当の場合： ゲームの販売から得られる利益の100％のうち、55％が配当の形で送られてきます 。
  6. 当然のことながら、会社の仕事の一環として、給与と配当の支払いを組み合わせる必要があります。 または、別の税制を使用することもできますが、原則は明確です。
続けましょう。



ゲームの1コピーの実際の費用はいくらですか



ゲームの費用はどれくらいかという質問に答えて、基本価格をドルで話すのが習慣です。 アメリカでのゲームのコピーはいくらですか。 ジャンルが似ているゲーム、たとえば、基本価格が15ドルのプラットフォームゲームNever Aloneを例にとると、 実際に世界でどれくらいの費用がかかるかを確認できます 。







注：現地通貨をドルに換算したときのゲームの価格が基本価格よりも低い地域は、赤で強調表示されています。 ご存知のように、 ロシアは低価格でリーダーです-15ドルのゲームは5.3ドルかかります 。

ゲームの価格は3地域のみで15ドル以上ですが、EUの税法によると、 VATは通常、販売地域のゲームの販売コピーごとに支払われます。これは、以前は15％でしたが、現在は国によって17から25％ （ リンク ）。







また、ストアでのゲームのライフサイクル中に、開発者としてのあなたがプロモーションや割引の助けを借りて販売を促進することも検討する価値があります。 割引の標準および受け入れられている慣行は次のとおりです。

  • 最初の週のローンチは10％割引
  • リリース後3〜6か月で25〜33％の割引
  • 1年後に50％-66％割引など 下向き


快適で柔らかいソファに座って、私たちは結論づけることができます：ゲームの基本価格が15ドルで、すべての地域で利用できる場合、1年間の販売をすると、ゲームの1つのコピーの1年間の平均価格は8ドル （実際にはそれは多かれ少なかれかもしれませんが、私たちは可能な限り最良のシナリオに従っています）。



潜在的な投資をモデル化します



そこで、「外部投資なしでゲームを作ろうとしたのはなぜですか」という質問に答えようとします。

スタジオが投資家から低予算でありながら、コンソールで快適に感じるクールなプロジェクトへの投資を受け取る状況をシミュレートしましょう。 与えられた：

  • 投資家は3年間で最大20万ドルのゲーム開発に完全に投資します
  • チームにとって非常にクールな条件：投資家に60％ 、会社とチームの40％


覚えておいて

  • 投資家がリスクを冒さずに、このお金を非常に優れた信頼できる銀行に3年間10％で投資すると、最終的に彼は267万ドル（200万ドル* 1.1 * 1.1 * 1.1 = 267万ドル）を受け取ります。
  • また、投資家の間では、良好な投資とはROI> 3の投資であると認められています。 つまり 3年間で20 万ドルが6 万ドル以上をもたらしました。


もう少し高く、ゲームのコピーが実際に1年間でどれくらいの費用がかかるかを計算しました-8ドル （基本価格15ドル ）。 また、配当の形での利益の100％のうち 、 55％が投資家と会社に届きます。

そのような流通で良いゲームさえ売られるでしょうか？

1年前、Sergey Galenkin（SteamSpyの作成者）は、2015年のSteamでの（多くの人が知っている）優れたインディーゲームの平均売上は約3 万コピー （ link ）だったと言いました。

また、2016年には 、優れたインディーズゲームの平均売上は2 万コピー以内になると予想されます（ リンク ）。



また、別のマルチプレイヤーシューティングゲームやサーバーへの何らかのバインドを行わないことにした場合、 高レベルの著作権侵害によって売上が損なわれる可能性があることも考慮する必要があります。 たとえば、プロジェクトでは、リリース後最初の1週間、 Unity Analyticsのアクティブプレーヤーの数に応じて、 1対30 （1対30の海賊を購入した）の割合で海賊版になっていると言えます。

そのため、シャベルと軸を備えた投資家が私たちの後を追わないように、追加の投資を引き付けないことにしました。







出版社なしで去るのはなぜですか？



よく見てみると、経験の浅い若いチームには次の2種類のパブリッシャーがいることに気付きました。

  • 開発に投資する出版社
  • （ほぼ）終了したプロジェクトのパブリッシャー


最初のタイプのパブリッシャーはほとんどありません（または見つかりませんでした）。2番目のタイプのパブリッシャーは、通常、70 / 30、60 / 40、50 / 50のスキームに従って機能します。



そもそも50/50形式+サイト運営者の費用の全額があります。サイト運営者は、広告とプロモーションに一定の金額を費やし、主に利益から返還し、残りの50/50をデベロッパーと共有すると言います。



ここで、2番目のタイプのパブリッシャーからのリスクがない場合に、利益分配の不当な割合で瞬間が生じます。



私の謙虚な意見では、交渉を開始する場所は次のとおりです。

あなたは、開発者としてゲームを作ったので、どれだけのお金とリソースが費やされたかを計算できます。 ゲーム開発予算。 50,000ドルと仮定します。 ゲームのプロモーションにいくら費やすかを正確に伝えるパブリッシャー、たとえば同じ5万ドルを見つけた場合、利益からの50/50形式での交渉の開始点は受け入れられます。



理想的な出版社がすべきこと









  • 開発コストの一部をカバー（品質への投資）
  • 品質と条件を外部から（合理的な制限内で）管理するため、つまり チームをリラックスさせないでください
  • プラットフォームパートナー（Valve、Sony、Microsoft、Nintendoなど）と連携する
  • テスト費用、開発キット、年齢の評価などを支援し、カバーします。
  • プラットフォームでスムーズに起動する（落とし穴と適切な人を知っている）
  • マーケティング戦略とクールなローンチPRを開発する
  • マスコミとコミュニティで正しく働く


出版社が必要かどうかを尋ねると、すでに半分以上の仕事をしていることがわかりました。 そして、残りはデューデリジェンスで学ぶことができるものです。 したがって、このプロジェクトでは、「信頼の飛躍」を図り、出版社なしで仕事をすることにしました。



Steamに乗る方法



これは非常に簡単です：



  1. ゲームを開発する
  2. 開発の途中で、プロジェクトのデモ資料の準備を開始できます。
  3. 100ドル（ロシアでは3200ルーブル）を特別ページ（ リンク ）でSteam経由で慈善基金に支払うと、Greenlightでプロジェクトページを配置することができます。
  4. 記事「 Steam Greenlightでのインディーズゲームのプロモーション 」または同様のテーマの他の記事を読んで、精神的に準備します
  5. ゲームのページを作成し、オープンアクセスで公開します
  6. プロジェクトに投票するための努力をする
  7. しばらくして、Steamでこのプロジェクトの招待状を受け取ります
  8. プラットフォームに完全にアクセスするには、NDAおよび必要なすべての文書に署名します。 以下の手順に従ってください。
  9. ゲームを修正してリリースする

年齢の評価を支払い、取得する必要がある理由



コンソールで初めてプロジェクトをリリースするチームが直面しなければならない落とし穴の1つは、 年齢の評価です。 あなたはおそらくあなたが走りたいほとんどすべての国で年齢の格付けがあると聞いたことがあるでしょう。 しかし、携帯電話やSteam向けのゲームをリリースした場合、この問題は特に心配することではありません。 コンソール上のプロジェクトの違いは何ですか：



  1. モバイルゲーム -評価が必要ですが、開発者のパネルでアンケート形式で無料で受けられます（ゲームにはヌードシーンがありますか？怖い要素がありますか？アルコール、キャラクターはタバコを吸うなど）。 毎年、それはますます簡単になり、最近では、いわゆるIARCシステムが導入されました。これにより、開発者パネル（Windowsストア、Google Play、Amazonなど）のアンケートを通じて、ほぼすべての地域評価を無料で取得できます。 このシステムについては、公式Webサイト（ https://www.globalratings.com/）で確認できます。
  2. Steam-評価はオプションです。 評価がありますか？ -いいね！ 評価なし？ あなたが望むなら、あなたのゲームはまだすべての地域で利用可能です。
  3. コンソール -評価が必要です。 そして、最も重要な地域が支払われます。

ESRB （米国およびカナダ）、あなたのゲームがデジタル形式でしか出ない場合-あなたは幸運です。 ESRB担当者との一定のやり取りの後、特別な調査に合格することで、ポータルにアクセスし、無料で年齢評価を取得できます。

しかし！ 評価のこの無料版は非常に切り捨てられています。 たとえば、暴力の種類を選択したり、暴力の有無を指定したりできます。 そして、私たちの場合、常に過大評価の10（E10-Fantasy Violence）が常に与えられました。



この地域のゲームもディスクで販売される場合は、完全な評価手順を実行する必要があります（2015年以降は12,500ドルの価値があるという噂があります）。 そのようなたくさんのお金を使うことの利点-あなたはゲーム+通常の記述子に対応する完全な評価を得ます。



PEGI （ドイツのないヨーロッパ）、原則として、無料のオプションはありません。 しかし！ プロジェクトによって価格にはばらつきがあります。

  • プロジェクトの重さが450 MB未満の場合は、幸運であり、500ユーロ未満の費用を支払うことになります。
