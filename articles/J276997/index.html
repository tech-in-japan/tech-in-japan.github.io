Visual Studio Code用の独自のテーマを作成する





Visual Studio Codeは、Microsoftの開発者向けの新しいエディターです。



もちろん、現時点では、広く普及している崇高なテキストに比べてサポートされる機能の数は劣りますが、同時に急速に勢いを増しています。 これの確認は、すでにVisual Studio Marketplaceのかなり大きな拡張セットです。



原則として、Sublime Textエディターを以前に試したことがある人にとっては、ビジュアルデザインと一連の基本機能の両方で、これらの製品の共通点を簡単に見つけることができます。



Habrahabrに関するごく最近の記事で、著者は次のように指摘しています。

残念ながら、VS Codeはまだインターフェースの設計をサポートしていません。 しかし、多くのクールな構文強調トピックがあります。 以下はその一部です。




既製のテーマに加えて、要件を完全に満たす独自のテーマを作成できます。



おそらく私は独創的ではありませんが、VS Codeでサポートされているテーマを作成できるオンラインエディターに関する情報を共有したいと思います。



エディターでは、必要なもの（フォント、色、キーワードの強調表示など）を変更できます。 既製のトピックがあります。 デフォルトで提供されるベースに基づいて独自に作成することもできます。 新しいテーマの準備ができたら、コンピューターに保存するだけです。



VS Codeにテーマをインストールする



1.％USER_PROFILE％\。Vscode \ extensionsディレクトリに移動します。

2.トピックの名前で新しいディレクトリを作成します。

3.テーマのディレクトリに次の内容のpackage.jsonファイルを作成します。 



{ "name": "theme-name", //  "version": "0.0.1", "engines": { "vscode": ">=0.9.0-pre.1" }, "publisher": "Your Name", //   "contributes": { "themes": [ { "label": "Theme label", "uiTheme": "vs-dark", // use "vs" to select the light UI theme "path": "./themes/YourTheme_Theme.tmTheme" //      } ] } }


4.テーマディレクトリを作成し、新しいテーマをそこに配置します。



できた メニューの「ファイル」-「設定」-「カラーテーマ」で、テーマを選択して適用できます。
