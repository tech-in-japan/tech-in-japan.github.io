Microsoft Kin One-エクスペリエンス

Habrでは、MicrosoftによるKin OneとKin Twoの立ち上げに関するパーティーについて既に議論しましたが、デバイスのレビューもありました 。



最近、ネットワークの知り合いがKin Oneの使用経験を送ってくれました。Habrasocietyはそれについて読むことに興味があると思います。







さらに-略語のない著者のテキスト。 （ 注意-たくさんのテキスト!!! ）





MicrosoftのKin OneとKin Twoの携帯電話が米国市場で悲惨な失敗をしたことは周知の事実です。さまざまな「情報源」によると、Verizon Wirelessは販売の最初の6週間でこれらのモデルを500〜10,000部販売しました。 それはたくさんですか、それとも少しですか？ ただ言ってみましょう。これは単なる失敗ではなく、災害全体と本当の「ジャングルの恥」です。なぜなら、デバイスの開発に1億ドルも投資されていないからです。 （もちろん、KinaをiPhone 4と比較することはほとんど公平ではありませんが、Kinaは3日間で170万台を販売しました。）



このため、Microsoftはヨーロッパでの秋に予定されているKin 1とKin Twoの「発売」をキャンセルし、米国ではVerizon Wirelessからしばらく利用できるようになると発表しました。 一方、Kin開発チームは、Windows Phone 7オペレーティングシステムを念頭に置いて取り組みます。WindowsPhone 7オペレーティングシステムは、企業のすべての "モバイル"の期待に委ねられています。



これはすべて実際にはどういう意味ですか？ Kin OneおよびKin Twoの既存のバージョンはCDMA EV-DOネットワーク用に設計されており、通常のGSM / UMTSネットワークへの変更はリリースされません。



さて、告白の時が来ました。4月11日（Kin電話の発表の日）に、Kin Oneモデルの写真を見て、突然「I WANT！」と気付きました。 はい、デバイスは非常に珍しく見え、「平均的なユーザー」はそれを購入することはまずありません（米国での販売で証明されています）。 CommunicAsia 2010で発表されたNokia X5-1も私を魅了しました。







しかし、Kin Oneに戻ります。 だから、キンの携帯電話のヨーロッパでの販売はキャンセルされましたが、私はまだそのようなデバイスをコレクションのために手に入れたいです。 そして、それを手に入れるだけでなく、作業コピーを手に入れるために-あなたが電話をかけたり、電話を受けたり、ウェブページにアクセスしたりできるように。 私が住んでいるウクライナでは、同じVerizon Wireless周波数を使用する複数のCDMA携帯電話事業者がいるため、米国からKin Oneを注文して接続を試みようという考えが生まれました。



当然、このアプローチにはいくつかの「落とし穴」もありました。たとえば、Kinの電話がVerizon Wirelessネットワークに「ロック」されているかどうかを事前に把握できませんでした。 明らかな理由により、これらのデバイスの所有者の通常のコミュニティはありません。また、外国のオンラインフォーラムでのレビューによると、Kin Oneを米国以外で使用することを考えた人はいません。







実際、eBayのオンラインオークションを見て、ほぼすべてのKinsが米国でのみ購入者に配送できることがわかりました。 ワールドワイド配信には1つのロットしかありませんでしたが、売り手（ちなみに、私の元同胞）も「ロック」について何も知りませんでした。 たとえば、「電話はVerizonから切断されており、SIMカードスロットがないため、何も保証できません。」 CDMAネットワーク用の電話の地元の販売者と話すことは、もう少し希望を呼び起こしました。彼らによると、アメリカのオペレーターSprintの機器はほとんど常に「ロックアップ」されています。 一般的に、私はチャンスをとることにした。 彼は250ドルをウクライナ系アメリカ人に送金し、5日後に（そう、パッケージが非常に早くウクライナに飛んだことにも驚いた）憧れのKin Oneがすでに手元にあった。







以下にデバイスの印象についてお話しします。そして今、Intertelecomオペレーターとのつながりのある叙事詩についてお話します。 そのため、この会社のオフィスでは、Kin Oneは特に驚きませんでした。明らかに、米国からのあらゆる種類のエキゾチックな商品が定期的に持ち込まれました。 彼らは驚きに驚いていませんでしたが、最初の試みでデバイスをネットワークに接続できず、彼らのせいではありませんでした。



実際、Kin OneはオペレーティングシステムWindows XP、Vista、および7によって単に「Kin Portable Media Player」と定義されています。 つまり、Zuneソフトウェアパッケージを起動して、たくさんの曲、写真、ビデオを携帯電話のメモリにダウンロードできますが、プログラムオペレーターの設定でPRLファイルをダウンロードするための空白の範囲では表示されません。 これには特別なドライバーが必要です-Pink CDMA Diagnostic（PinkはKin OneとKin Twoが作成されたプロジェクトの名前です）; さらに、それらをインストールするには、キーボードからシークレットコードとパスワード（それぞれ## 77647266488と000000）を入力し、表示されるメニューでPC接続モードをUSBパススルーに変更する必要があります。 その後、デバイスはアプリケーションマネージャーでピンクCDMA診断として指定され、Windowsオペレーティングシステムは「fire」を見つけようとしますが、何も見つけられないので、自分で置き換えることができます。 そのため、これらのドライバーをインターネットで1週間以上検索しました。 そして彼らは何も見つけませんでした。



私はすでにインターテレコムのオフィスからキンワンを連れて行き、棚に横たわる運命だと判断し、無駄に費やした250人を思い出しました...しかし、数日後、KinPixiPreCentroTreo800w_etc_DiagDrivers.rarアーカイブ（52 kb）がXDA-Developers.comにアップロードされました。 伝えられるところでは、「診断」モードで接続するために必要なすべてのドライバーが含まれていました。 ダウンロードしました。 やってみた 動作しません。 つまり、Windowsはフォルダー（* .inf拡張子を持つ3つのファイルと* .sys拡張子を持つ2つのファイル）をスキャンし、その中に正しいドライバーがないことを報告します。 一方、オペレーティングシステムのデバイスマネージャーの電話のデバイスIDには値Vid_056Cが含まれており、何らかの理由で、ドライバーVid_05C6がメモ帳で開いたドライバーに拡張子* .infで書き込まれていることに気付きました。 修正、保存しました。 稼いだ！ ドライバーがインストールされ、Kin Oneが[ポートデバイスマネージャー（COMおよびLPT）]セクションに[CAIN] Microsoft KIN Passthru Diag Port（COM1）として表示されます。 Intertelecomのエンジニアは問題ありませんでした。翌日、Kin Oneが接続されました。 結局のところ、電話はVerizon Wirelessネットワークでロックされていませんでした。 数週間の操作の後、デバイスは突然Webページの読み込みを停止しました。つまり、電話をかけたり受けたりする機能は残っていましたが、インターネットは「横たわっていました」。 この問題は、オペレーターのオフィスにPRLファイルをリロードすることで解決されましたが、沈殿物は残りました。 誰が知っている-Kin Oneが次回「ダウン」するのはいつですか？ たぶん明日、たぶん決して。 ここで、電話機をPCのモデムとして使用できないことに注意してください。







次に、私の意見では、Kin Oneに関する10の最も興味深い事実を紹介します。 電話を使うことのニュアンスと、おそらく失敗の理由について話すのが一番いいと思います。

1.日本企業のシャープは、Kinフォンの組み立てに取り組んでおり（2008年にMicrosoftが買収したDanger SideKickフォンも生産しています）、バックパネルに誇らしげに書かれています。 それだけでこのアセンブリ自体の品質は高いとは言えません。半分のバックラッシュが発生するだけでなく、いらいらするだけでなく、バ​​ッテリーカバーがあまりしっかりと保持されません。 ちなみに、電話機の上部はキーボードと接触していないようですが、ボタンのペアにはすでに擦り傷があります。



素材については、Kin Oneはさまざまなプラスチックを使用しています-背面はすぐに汚れた柔らかい感触で作られ、前面は半透明のスモーキープラスチックで作られています。 一般的に、同じ金属のZune HDがクールで高品質なものの印象を残す場合、Kin Oneは次のように認識されます。たとえば、安価なおもちゃのように（ほとんどのKin Oneは「たまごっち」に似ています）。 最も可能性が高いのは、クアルコムQSC6085 CDMAチップセット、GPS受信機、BluetoothおよびWi-Fiアダプター、FMラジオなどの大量のワイヤレスモジュールのために、開発者がプラスチックを使用することを決定したことです。



2. Kin電話は、Zune HDメディアプレーヤーの基盤でもある「第1世代のnVidia Tegra」システムに基づいて構築されています。 これは、後者のインターフェイスが非常に迅速かつスムーズに機能するだけであり、Kin Oneは実際の「ブレーキ」と呼ぶことができます。 たとえば、メインメニューからダイヤルに移動するには、0.5秒待つ必要があります。 ほとんどすべてのメニュー項目とアプリケーションはゆっくりと開き、電話機の電源は非常にゆっくりとオンになります。



3.メールクライアントとWebブラウザの両方に、キリル文字ではなく「象形文字」が表示されます。 また、電話はロシア語の文字を含むメッセージを理解しません-それらは空になります。 テキストにキリル文字とラテン文字の両方が含まれている場合、テキストも空になります。 昨年の冬、ひどいZune HD Webブラウザーに会った後、私たちの現実のKin Oneブラウザーも動作が悪いと思いましたが、すべての期待を超えていました。 プレーヤーは通常、少なくともいくつかのロシアのサイト（10のうち約1）を表示しましたが、Kin Oneはこれを行うことすらできません。 公平に言うと、この電話は米国のみを対象としているため、あまりhimる価値はありません。 さらに、彼はMP3ファイルのタグを非常に適切に「ダイジェスト」します。キリル文字に問題はありません。



4. 2.6インチの容量性タッチTFTスクリーン（240x320ピクセル）は印象的ではありませんでした。 どうやら、Kin Oneは非常に安価なマトリックスを使用しています-太陽の下では「フェード」し、色の表現は理想からはほど遠いです。 たとえば、Sony Ericsson P1iには同じパラメーター（おそらく抵抗のみ）のディスプレイがあり、最新の標準とはほど遠いですが、最高の画質を提供します。 Kin Oneはマルチタッチテクノロジーもサポートしています。KinOneの場合、2本の指で写真やページを拡大縮小するのは不便です。 はい、そして「広がり」がありますが滑らかではありませんが、ぎくしゃくしています。 一般的に、失敗。



5.携帯電話には衛星GPSナビゲーション用の受信機がありますが、ナビゲーションには使用できません-内蔵カメラで受信した写真にジオタグを追加する場合のみ。 Microsoftが独自のマッピングサービスBing Mapsを持っていることを考えると、このアプローチは奇妙です。 Kinの携帯電話にはJavaがサポートされていないため、GoogleマップまたはYandex.Mapsをここにインストールすることはできません。



6.ところで、カメラについて：5メガピクセルのオートフォーカスモジュールがソニーによってリリースされたと言われていますが、高品質の写真やビデオ（640x480ピクセル、30フレーム/秒）について話すことはできません。 LEDフラッシュは非常に弱く「ヒット」しますが、懐中電灯として使用することはできません（これはすでに夢です-すべての「通常の」携帯電話にはこのような機能はありません）。 比較のために：画質の点では、Kin Oneは高価なSamsung S8500 Waveとかなりの予算のPhilips Xenium X830によって簡単に「構成」されます。 それらと比較して、a）これらのデバイスには5メガピクセルのカメラもあり、b）彼らは手にしたばかりだからです。



7. Kinファミリーの携帯電話の主要な「トリック」の1つは、Kin Studioサービスです。これにより、写真、ビデオ、その他の個人データを「クラウド」に保存できます。 サービスに登録してアカウントを取得するには、メールアドレスとパスワードを指定して、その条件を受け入れる必要があります。 電話をオンにするたびに、対応するオファーが表示されます。



だから、私は登録することができませんでした。 判明したとおり、これにはVerizon WirelessのIPアドレスが必要です。 それ以外の場合はありません。 興味深い点。 最初に、デバイスはアドレスとパスワードを受け入れ、条件に同意しようとすると、次のように書きました。「Kinサーバーが過負荷になりました。15分待ってからもう一度やり直してください。」 数週間後、電話は、EV-DOとWi-Fiの両方を介してサーバーとまったく通信できなかったと報告し始めました。 状況は、メニューからダウンロードできる更新プログラムでも同様です。以前は、Kin Oneは現在更新プログラムはないが、サーバーに「到達できない」と主張しています。 ウクライナのKin Oneと一緒にアクセスが「注文」された場所に行かないように、私は単に禁止されているという印象を受けました。



最後に、理論的には、携帯電話はRSSフィードを読み取り、Facebook、Twitter、MySpace、Windows Liveなどのソーシャルサービスと何らかの方法でやり取りすることができます。 「何らかの方法で」、実際にはこれらのオプションを試すこともできなかったので、電話は単に「ログイン」することを拒否し、後でこれを試みることを提案します。 どうやら、テープとソーシャルネットワークはKin Studioとはまったく関係ありません。
