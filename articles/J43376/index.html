PDCの発表：Azure-クラウドの新しいプラットフォーム、Oslo-アプリケーション開発への新しいアプローチ

スティーブマーティンは、 Microsoft PDCカンファレンスのオープニングで発表された内容に関するメッセージで詳細な投稿を既に公開しています。 そして、彼の後ろで、開発者のDevisionシェフであるSomasegarもいくつかの詳細を共有しました。



簡単に言うと、 Azure Services Platformは、数回クリックするだけでアプリケーションをスケーリングできるようにするアプリケーションをホストするためのサービスのセットです。 Azureは、アプリケーションを公開するためのプラットフォーム（アプリケーションはMicrosoftデータセンターでホストされ、サーバー上のアプリケーションとオペレーティングシステムはアプリケーションのダウンタイムなしで更新されます）、Windows Liveサービス、Microsoft .NETプラットフォーム、データストレージ用のSQLサービス、SharePoint ServicesおよびDynamicsを組み合わせますCRMサービス。



このすべてにおいて、Azure Servicesでホストするためのアプリケーションの開発は、Visual Studioでの通常のASP.NET Webアプリケーションの開発と変わりません。 詳細については、Azure Web Developerサイトセクションを参照してください。



Azureに関するニュースに加えて、Osloプロジェクトの完全なサイトがついにオープンしました。 Osloの目標は、コンポーネントの抽象化と宣言的記述の追加レイヤーを使用して、ワークフローとサービスに基づいたアプリケーションの開発者の生産性を向上させることです。 Osloは、アプリケーションが構築されるモデルを記述するための一連のツールと新しい言語を提供します。
