彼のMa下ヴィンペレーターは、突然、東から攻撃を開始しました

ChromeがLinuxで安定して動作し始めてから、この素晴らしいブラウザを使い始めました。

そして、VimエディタープラグインとFirefoxのVimperatorのアクティブユーザーとして、「j」を押してページをスクロールしたときに戸惑う瞬間に我慢しなければなりませんでした。 。



そして最近、まったく偶然に、私は日本人プログラマーによって作成されたプロジェクトに出会いました。もちろん、Vimperatoraの潜在能力を完全には実現していませんが、最も好きな機能は次のとおりです。
  • ナビゲーション：j、k、l、h
  • ヒントモード：（f、F）
入力フィールド内：
  • Esc、Ctrl + [-通常モードへの移行
  • Ctrl + a-フィールドの先頭に移動し、もう一度押してすべてを選択します
  • Ctrl + e-フィールドの最後に移動
  • Ctrl + f-右カーソル
  • Ctrl + e-左カーソル
  • Ctrl + d-右側の削除
  • Ctrl + h-左に削除
通常モード
  • r-このページを再読み込み
  • d-ページを閉じる
  • u-閉じたページのページを復元する
  • H-歴史に戻る
  • L-歴史の中で進む
  • G-ページの一番下までスクロール
  • gg-ページの一番上までスクロール
  • 0-ページの上部までスクロール（水平）
  • $-ページの下部までスクロール（水平）
