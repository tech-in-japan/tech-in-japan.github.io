レイアウト、CSS、Windows Mobile 5.0。 IEおよびOpera Mobileのフォントサイズ

楽しいサイトのPDAバージョンのスタイルに関する軍団。 IE Win MobileとOpera Mobileでは、同じコードを表示するのに違いがあることがわかります。



body {

フォント：100％Verdana、Arial、sans-serif;

色：＃000;

背景：#fff;

}



Opera Mobileのフォントは、IEの2倍の大きさです。 どちらのブラウザーもデフォルト設定です。 katomワーキングソリューションの下。 熊手を踏まないでください。





フィルタリングは非常に簡単です。



/ *オペラモバイル* /

html> body {

フォントサイズ：50％;

}



Opera Mobileはこの構文を理解するため、すべてのbodyタグはフォントサイズを半分に減らします。



ちなみに、視聴記録

ボディ[クラス] {

フォントサイズ：50％;

}

フィルタとして適合しません-IEはそれ自身を理解します:-)
