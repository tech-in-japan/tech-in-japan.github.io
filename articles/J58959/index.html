Windows 3.11

私が見るロゴから誰も失恋しないことを願っています:) Windows 7 RCを揺らしている間に、Windows 3.11を見つけたビンで...私が始めたWindowsは...もちろん、インストールはそれほど簡単ではありませんでしたフロッピードライブはありません。 私はそれをしなければならなかったので...まあ、一般的に、最も直接的な方法ではありません。 VMWareバージョン6にインストールしました。まず、DOS 6.22を起動する機能を備えたブートCDをダウンロードし、パーティションを作成（fdsik）、フォーマット（フォーマットc :)、そこにシステムファイルをドロップ（sys c :)、作成したマシンからハードドライブを開きました（ VMWareを使用して）Win 3.11インストールファイルをそこに残しました。 最初はロシア語をインストールしたかったのですが、インストール中にインストールを見たときに、config.sysにどのcpを書き込む必要があるかを覚えておく必要があることにすぐに気付きました。そのため、ENGバージョンを使用しないことにしました。

そして最後に、私の愛する忘れられないものからのもう一つのスキンショット：



誰かが掘り下げることに興味がない場合は、VMWare仮想マシンでアーカイブを投稿しました： Windows 3.11.7z（〜16Mb、パスワード-outcoldman）

更新：書き込みを忘れた場合、dosが最初にロードされ、winコマンドを実行するために必要なウィンドウを起動します。
