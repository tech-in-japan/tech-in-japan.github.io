Raspberry Piを改善する5つの方法

Raspberri Piシングルボードコンピューターは急速に人気を集めています。 できる限り安く​​するために、開発者は「不要」なものをすべて捨て、可能な限り設計を可能な限り簡素化しました。 一方では、これによりユーザーはシステムを操作する際に不便を強いられます。 しかし一方で、創造性と改善の余地は残されています。



そこで、ここでは、Raspberry Piの5つの便利な変更点について説明します。



1.ケース

すべての方向に突き出たコネクタを備えたボードは非常に無防備に見えるので、何らかの場合に隠したいと思います。 Raspberry Piには、あらゆる好みに応じてさまざまなケースがあります。



標準のプラスチックケースを注文できます。 スタイリッシュでシンプルな、いくつかの色の選択。





自分で印刷することができます。





手元に3Dプリンターはありませんか？ ケースはプレキシガラスのジグソーで切ることができます。 説明書が添付されています。





あなたは過激な「緑」であり、あなた自身が余分なプラスチック片で惑星を散らかすことを許しませんか？ スタイリッシュなエコフレンドリーな木製ケースをお持ちください。

2.ラジエーター

多くの人は、ラズベリーが非常に暑いことに不満を持っています。 最もホットなノードは、プロセッサ、イーサネットコントローラ、3.3V電圧レギュレータであり、過熱問題の解決策は、これらのマイクロ回路にラジエーターを配置することです。





これを行うには、古いマザーボードのラジエーターが必要であり、金属用に見られ、適切なサイズの断片にカットし、紙やすりで磨き、熱伝導性の接着剤が必要です。 操作の詳細は次のとおりです。





著者によると、プロセッサの温度は56.1°Cから38.8°Cに低下しました。 これで十分でない場合は、ファンを上に置き、GPIOポートの1つから電源を供給して、温度に応じてファンの速度を調整できます。



3. MicroSD

Raspberry PiのSDカードスロットはやや不便です。カードはボードの外に突き出ています。 この不便さを克服する簡単な方法があります：ボードの底に別のカードホルダーを置きます。 リビジョンの説明 。





SD-microSDアダプターは追加コネクターとして使用され、ネイティブコネクターのコンタクトパッドに直接はんだ付けされます。 microSDカードを挿入します-これで完了です！ 板を超えても、厚さでも、何も突き出ていません。





標準コネクタは引き続き正常に機能します。 真の奇跡は起こらず、両方のカードを同時に使用しても機能しません。



4.スイッチング電圧レギュレータ

Raspberryには、5 V-> 3.3 Vの線形安定器があります。ボードでは、RG2として指定されています。





システムは大量の電流を消費するため、スタビライザーでかなりの電力が消費されます。 効率を上げて加熱を減らすには、リニアスタビライザーをパルススタビライザーに交換できます。 これを行う方法については、 こことここで説明します 。 新しいスタビライザーは、古いスタビライザーから残ったサイトに接続できます。





または、5 Vおよび3.3 V回路も出力されるGPIOブロックに：





最初のリンクで説明されている変更により、消費電流が410 mAから360 mAに、つまり12％削減されます。 2番目の記事では、消費電力が最大25％削減されたと報告しています。



5.レイアウトフィールド

Raspberry Piは、自動化システム、「スマートホーム」など、さまざまなデバイスの制御によく使用されます。 ここでの大きなプラスは、低レベルのGPIO I / Oポートの存在です。 しかし、これらのポートを使用するのはあまり便利ではありません。ほとんどの場合、外部インターフェース回路、ジャンクションが必要です。つまり、追加のボードを製造するか、「ノズルに」取り付ける必要があります。



デザインにすっきりとした仕上がりを与えるために、特別なブレッドボードを使用できます。





ボードはラズベリーと同じサイズで、GPIOブロックの上に装着されます。 I / O、UART、I 2 C、および電源ポートに簡単に接続するためのソケットとクリップがあります。 すべての空き領域は、必要なコンポーネントを配置できる模擬フィールドで占められています。
