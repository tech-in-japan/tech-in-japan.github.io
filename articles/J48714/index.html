ゲームセット

そのような素晴らしいボードがあります（コンピューターではありません！）ゲーム-The Set（The Set）。 遺伝学者のマーシャ・ファルコは、ジャーマン・シェパードのてんかんの研究中にメモを取ることでそれを発明しました。 このゲームは珍しく、非常に刺激的で脳に負荷をかけることが判明しました。 Habralyudam-まさにそれ！ 注意：中毒が発生する場合があります。















カード



セスはカードゲームです。 すべてのカードには4つのパラメーターがあり、それぞれ3つの値を取ります（合計3 x 3×3×3 = 81カード）。 パラメーターのタイプと値は次のとおりです。









  • 図 :: =楕円| ひし形| 「鼻水」
  • 色 :: =赤| 緑| 紫
  • 塗りつぶす :: =白| ストライプ| 固い
  • 番号 :: = 1 | 2 | 3

エッセンス



ゲームの目標は、3枚のカードの特別な組み合わせを見つけることです。 3つのカードすべての各パラメーターの値は、同じか異なる必要があります。 これらのカードの組み合わせはセットと呼ばれます 。









例：













つまり、2つのカードに1つのパラメーター値があり、3番目のパラメーター値が異なる場合、3つのカードはセットを作成しないと言えます。 2枚のカードには、常に3枚目（1枚のみ）がセットになっていることがわかります。









ゲームの進行



ホストはテーブルに12枚のカードを置きます。 誰かがセットを見つけると、「セット！」と叫び、セットカードのコンポーネントを静かに拾います。 レイアウトされたカードにセットがない場合（これはほとんどありませんが、可能です）、発表者はさらに3枚のカードを出します。 セットなしのカードの最大数は20です。ラウンドは、デッキがなくなるまで続きます。 より多くのセットを集めた方が勝ちです。









ヒントとコツ



  • リーダーも通常プレーしたいので、私たちは独立して、リーダーがテーブルに12枚すべてのカードを置くまで「セット！」と叫ぶことはできないというルールを導入しました。 このルールでは、最も知的なリーダーをリーダーとして割り当てる方が適切です:)
  • ラウンド後のセットを測定し 、デッキのシャッフル、セットのピックアップに問題がないようにするために、1枚のカードを自分用に残して、2枚を共通の山に入れることができます。
  • ゲームは非常に動的です。 神経は緊張していて、緊張は限界にあります。 誰かがしばしば「セット！」と叫ぶことができます。そして、「ああ、ごめんなさい、私は彼を認識しました。」 彼が取ったセットを取り去ることができます。
