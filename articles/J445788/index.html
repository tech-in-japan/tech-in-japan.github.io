DIYクラウドビデオ監視：Ivideon Web SDKの新機能





パートナーが独自の製品を作成できるようにするいくつかの統合コンポーネントがあります。Ivideonの個人アカウントに代わるものを開発するためのオープンAPI、Ivideonアプリケーションと同等の機能で完全なソリューションを開発できるMobile SDK、Web SDK。



開発者がプラットフォームをより柔軟で便利にする新しいドキュメントシステムとデモアプリケーションを備えた、改善されたWeb SDKを最近リリースしました。 すでにSDKに精通している場合は、すぐに変更に気付くでしょう-これで、アプリケーションにAPI関数を埋め込む方法の明確な例が得られました。



それ以外の場合は、日常的なケースとIvideon API / SDKを使用した実装の実装について詳しく説明します。



Web SDK：新機能



Ivideonは単なるクラウドベースのビデオ監視サービスおよび機器プロバイダーではありません。 Ivideon内では、カメラファームウェアからWebバージョンのサービスまで、完全な開発サイクルが進行中です。 クライアントおよびサーバーSDKの作成、LibVLCの改善、WebRTCの実装、ビデオ分析の実行、パートナー向けのホワイトラベルサポートおよびSDKのデモプロジェクトを使用したクライアントの開発を行っています。



その結果、どのパートナーが独自のソリューションを作成できるかに基づいたプラットフォームになりました。 これで、SDK for Webがメジャーアップグレードされました。さらに多くの統合ソリューションが期待されています。



便宜上、「クイックスタート」セクションを最初に追加しました。これにより、デバイス管理を簡単に理解できます。



以下のコードは、Ivideon Web SDKの基本的な使用法を示しています。プレーヤーがページに追加され、パブリックカメラのビデオ再生が開始されました。

また、いくつかの新しい機能を追加しました。



  • ワンタイムビデオリンクのサポート。
  • ビデオの品質とアーカイブの再生速度を制御するボタンがプレーヤーに追加されました。
  • プレーヤーコントロールは、一度に1つずつオンとオフを切り替えることができます（以前は、すべてをオンにするか、すべてを非表示にすることができました）。
  • カメラの音をミュートする機能を追加しました。


デモアプリ



ユーザーインターフェイスライブラリでIvideon Web SDKを使用する方法を示すために、デモアプリケーションで配布しています。 これで、Ivideon Web SDKがReactJSでどのように機能するかを確認できます。



デモアプリケーションは、 こちらからオンラインで入手できます 。 彼の作品には、Ivideon TVのランダムカメラが追加されています。 カメラが突然非アクティブになった場合は、上記のリンクを再度たどってください。



デモに慣れるもう1つの方法は、Web SDKでソースコードを調べ、自分でアプリケーションをビルドすることです。



私たちのアプリケーションは、ユーザーのアクションに対応するコードを表示できます。







異なるエンジンを持つ複数のプレーヤーをページに追加し、彼らの仕事を比較します。







複数のプレーヤーを作成し、それらを1つのタイムラインから管理します。このタイムラインには、複数のカメラの録画のアーカイブが同時に表示されます。







デモアプリケーションは、ブラウザーのローカルストレージの最後のセッションの設定を記憶します：APIへのアクセスパラメーター、カメラ設定など。 再入場すると、復元されます。



デモアプリケーションコードはソースマップを使用してコンパイルされます。デモコードはデバッガで直接表示できます。







統合の例







「 iSKI 」という接頭辞が付いたプログラムのグループには、ほとんどすべてのヨーロッパのスキー国向けの個別のアプリケーションが含まれます。iSKIAustria、iSKI Swiss、iSKI France、iSKI Italia（チェコ、スロバキア、スオミ、ドイツ、スロベニアなど）。 このアプリケーションは、スキーリゾートの雪の状態、山の中のレストランのリスト、サーキットトレイル、および旅行前に目的地の全体像を把握するのに役立つその他の有用な情報を表示します。 同時に、インターネットへのアクセスは必要ありません-オフラインで動作します（カメラからのブロードキャストを除く）。 すべてのアプリケーションは無料で利用できます。



現在、ほとんどすべてのスキーリゾートには、斜面の状況を示すカメラがあります。 アプリケーションを介してカメラをリモートで表示するために、iSKIにSDKを提供し、天気予報、雪の深さ、オープンリフトの数だけでなく、斜面から直接ビデオを見ることができるようになりました。







さまざまなスマートホームシステム。 Ivideonと統合することにより、これらのソリューションはホームを制御し、クラウドアーカイブに最も安全な方法でビデオを保存するため、ホームセキュリティにとってより多くの利点があります。 脅威をリアルタイムで通知し、異常な状況に迅速に対応できるモバイルアプリケーションを介して完全な制御が実行されます。







システム分析作業の売り手とコンサルタントのパーフェクトサービスソリューション。 クラウドベースのビデオ監視システムは、オペレーターによってアーカイブされたデータを監視および記録し、結果はオンラインでアカウントに反映されます。 クライアントは最終的に、特定のイベント（販売プロトコル違反または物議を醸す事件）を伴う短いフラグメントを受け取ります。 Webインターフェースでは、違反データと組み込みのビデオスライスが表示されます。 データ配列全体は、重大なイベントと定期的なイベントの2つのカテゴリに分けられます。 レギュラーはイベントの翌日にオンラインオフィスに表示されますが、重大な違反については、SMSまたはメッセンジャーでレポートを受信できます。
