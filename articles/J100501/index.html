Sony Media Go：インストール段階でも巨大な

ソニーからiTunesのようなアプリケーションが登場してから約1年が経過しました。 pspとPlayStation Storeのクライアントを管理するためのソフトウェア以外のユーザーには推奨されません。

アプリケーションの要件はほとんどありませんが、あまりにも多くのことを提供します...多くの条件を観察した後、それは怪物であり、ソフトウェアの良好なシステム要件を持ち、特定のオペレーティングシステム（既にどのシステムの下で推測した）でのみ動作します。

原則として、すべてが言われています。

しかし、私は少し説明しようとします 。



始めましょう


そのような獣は、口実がなくても、Windowsではなく起動しないことは論理的です、試してはいけません。 これは、それらの下にインストールすることによっても証明されます-私は特別に親ネットブックを取り、下向きの矢印を固定し、大まかに選んだ窓に埃で覆われたXPを選びました。 Wai-Waiは切断し、インストーラーを外部ハードドライブにアップロードして開始しました...インストールには約40分かかりました。



1つのWindowsのシステム要件は制限されていません








この区分の分布は90メートル以上の重さがありますが、

  1. Visual C ++ 2006再頒布可能パッケージをインストールします
  2. Visual C ++ 2008（または2009、私はもう覚えていない）をインストールする
  3. WMF（windows media format chtoli ...）をダウンロードしてインストールしようとすると、ダウンロードする必要があります（それなしでは起動しません）-配布まで+12メートル
  4. 同様に.net framework 2.0（+ 23メートル）で-ダウンロードしてインストールする必要があり、英語以外のロケールを選択する場合は.net frameworkローカライズパッケージが必要です（+2メートル、Media Go自体で英語を入れて言語をロシア語に切り替えることができます）
  5. QuickTimeが必要です-（どれくらいの重さかわかりません）、iPodに付属のディスクからiTunesを取り出して、マシンに配信しました
  6. フラッシュを置く




何が得られますか？


その結果、ソニーからこのようなiTunesを入手できます。



ユーザーが望むもの


すべては問題ありませんが、私と多くのユーザーは次のようなPCアプリケーションが必要です。

  • ps-storeで購入できます
  • 保存とゲームをpspからコンピューターに、またはその逆にコピーできます。 （はい、fsのペンのようになることができます）
  • ps-storeからダウンロードできます。 ダウンロードを一時停止する機能。 pspにはそのようなものはありません。また、ストアから大きなものを「一度に」ダウンロードすることは、一部の人にとっては単純に不可能です（たとえば、実際の速度で約50キロバイト/秒で夜に7時間無制限です）。
  • クロスプラットフォームになります




代替案


または、Webバージョンを開いて、クロスプラットフォームソフトウェアを提供して、DRMコンテンツ（まだお金を払っています）をデバイスにダウンロードします。

ちなみに、psp用の「ネイティブな」psストアが登場する前は、Webバージョンのストアが利用可能でした。 そこで、エンコードされたコンテンツをダウンロードし、別のユーティリティ（ちなみにインストール時にメディアと共にインストールされます）を使用して、デバイスにアップロードすることができました。

このユーティリティは純粋にWindowsであり、その「ページ」はpsn usa Webサイトにあります。
