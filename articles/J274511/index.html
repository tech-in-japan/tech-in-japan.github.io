32ビット版と64ビット版のブラウザのベンチマーク：より速く、より強く、より高い？

現在、ユーザーはいくつかの優れたブラウザーを使用できます。 ほとんどには32ビットバージョンと64ビットバージョンがあります。 私たちはそれぞれ、このブラウザの機能について独自の意見を持っています。 それでも、32ビット版と64ビット版の両方で、各プログラムでいくつかのテストを実行することにより、ブラウザーを比較することができます。



Ghacksリソースのユーザーはこれを行うことにしました 。 テストシステムとして、Intel Core i5-2500、8 GBのRAM、256 GB SSD、Nvidia GeForce GTX 960、およびWindows 10を搭載したPCが選択されました。



32ビットブラウザーと64ビット



いくつかの例外を除き、さまざまなブラウザーにそのようなバージョンがあります。 1つの例外はOperaで、32ビットバージョンのみです。 2番目は、32ビットバージョンを持たないMicrosoftのブラウザーであるEdgeです。



すべてのブラウザーは、完全なサービスパックを備えたWindows 10にインストールされました。 変更は行われませんでした。 ブラウザについては、次のとおりです。



  • Mozilla Firefox 43.0.3 32ビット
  • Mozilla Firefox 43.0.3 64ビット
  • Google Chrome 47 32ビット
  • Google Chrome 47 64ビット
  • Pale Moon 25.8.1 32ビット
  • Pale Moon 25.8.1 64ビット
  • Vivaldi Beta 2 32-ビット
  • Vivaldi Beta 2 64ビット
  • Opera 34.0.2036.25 32-ビット
  • Microsoft Edge 64ビット




テスト



次のテストが実行されました。



1.インストール後にRAMを使用します。

2. 10のオープンサイトでRAMを使用する（（ghacks.net、startpage.com、youtube.com、betanews.com、reddit.com、slashdot.org、producthunt.com、twitch.com、amazon.com、google.com）

3. Mozilla Krakenベンチマーク

4. JetStreamベンチマーク

5. Google Octaneベンチマーク

6. RoboHornetベンチマーク

7. Dromaeoベンチマーク 。



結果はやや予想外でした。 たとえば、Google ChromeはRoboHornetやDromaeoなどのテストにハングアップしました。 Pale MoonはJeSteamベンチマークでクラッシュし、VromadiはDromaeoベンチマーク中にコマンドへの応答を停止しました。



すべてのテストに問題なく完全に合格した唯一のブラウザは、Mozilla FirefoxとMicrosoft Edgeです。



また、Vivaldiがいくつかの点でGoogle Chromeを上回っていることも興味深いです（さらに、このブラウザーはまだベータ版です）。















次に、同じテストに合格した場合のブラウザーの異なるバージョンの動作を見てみましょう。



Mozilla Firefox 32ビットと64ビット





RAMの使用は、最初はほぼ同じレベルです。 しかし、すぐに10サイトをダウンロードした後、64ビットバージョンのFFはこのブラウザーの32ビットバージョンよりも多くのRAMを必要とし始めました。 PCが大量のRAMを自慢できないユーザーには、覚えておく価値があります。



Google Chrome 32ビットと64ビット





ここではほとんど同じですが、開始時、または興味深いことに、同時に10個のサイトをロードした後でも実質的に違いはありません。



一般に、64ビットバージョンは、32ビットバージョンよりも少し優れています。

Vivaldi 32ビットと64ビット
