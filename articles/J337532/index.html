KolibriN 8.Xリリース

（2012）



Kolibri 0.8.0.0-KolibriNの前身であるナイトリービルドに基づいて収集したIMGイメージ。

KolibriN 8.1は、ナイトリービルドの上にインストールするアップグレードパックです。

KolibriN 8.2はISOディストリビューションです。 夜のアセンブリがロードされ、KolibriNはすぐにロールオーバーします。



リリース時のKolibriN8は、IMGアセンブリから完全なISOへの移行をマークしたため、当時のハチドリにとって非現実的なブレークスルーであることに気づかずに、whi吹きに重点を置いて非難されました。



しかし、まず最初に。



OSが起動すると、すべてのデータがkolibri.imgイメージから/ rd / 1仮想ディスクに完全にコピーされます。 一部のプログラムは1.44 Mbに適合しないため、CD-ROMに直接置かれています。 ディストリビューションを収集して、メニューのDOOMへのリンクを追加し、zSeaビューアーに関連付けを登録します。 両方ともCDに収録されています。



キャッチは、私たちがこれを行うことができないということです！ CDは/ cd / 1、/ cd / 2、/ cd / 3などにできるため 推測はオプションではありません。 当時、この問題を解決するツールはありませんでした。



KolibriN8の開発中に、ISO上のファイルを検出し、それらを/ tmp0 / 1仮想ディスクにコピーするインストーラーを作成しました。



現在、それらのパスは常に同じであり、デスクトップの追加のプログラムへのショートカットをメニューで追加でき、最も重要なのは関連付けを登録することでした。



2番目に重要なことは重要でした。ハミングバードのすべてのプログラムを1か所に集め、フォーラム、開発者の個人Webサイト、インターネットの隅々に登りました。



夜間のビルドと比較したKolibriNの変更のリスト：



  • 配信には、これまでに公開されていないLiza電子メールクライアントが含まれます。
  • 今では誰でもGrafx 2を実行して試すことができます。それまでは、パスへの緊密なバインドにより、これは重要なタスクでした。
  • 通知ウィジェットにデスクトップメニューと半透明の影を追加しました。
  • このパッケージには、Doom、Loderunner、Pig、Jumpbumpなどのゲームと、ゲームコンソールNES、SNES、Gameboyのエミュレーターが含まれています。
  • DosBoxおよびZX Spectrumエミュレーターは、何百もの古いアプリケーションとゲームを実行します。
  • アプリケーション配信：zSeaイメージビューアー、PDFビューアー、FPlayビデオプレーヤー、その他多くのプログラム。
  • すべての関連付けが登録され、退屈な壁紙とスキンはデスクトップコンテキストメニューから簡単に変更できます。

ライザメールクライアント

（2012.2013）



暗号化なしでPOPプロトコルで動作するすべてのメールボックスをサポートする必要があります-これはyandex.ru、i.ua、mail.ruです... gmail.comは動作しません。 暗号化を使用します。







ログインとパスワードは、プログラムをテキストファイルに閉じた後に保存されます。 独自の接続設定を設定できます。







ログインに成功したら、メールボックスに入ります。 電子メールヘッダーは自動的にダウンロードされます。 レターの本文をクリックすると、レターの本文が読み込まれます。レターのダウンロードプロセスも進行状況バーに表示されます。 文字のリストと文字自体を表示するパネルのサイズは、「= = = "」でマウスをドラッグすることで変更できます。

KolibriN9エディション

（2014）



開発の1年半後、KolibriN9の新しいバージョンがリリースされました！ KolibriN9はKolibriOSに基づいており、より美しく使いやすい環境を提供します。これは美しいデスクトップの壁紙であるだけでなく、提供されるソフトウェアの数が非常に多いものです。







KolibriOSからもたらされた変更から：



  • 追加のディレクトリ/ kolibrios /を接続する; （KolibriN8の作成の結果）
  • USBサポート（フラッシュドライブ、マウス、キーボード、ハブ）およびそれらの「ホット」接続。
  • FS XFSの読み取りおよびext2 / 3/4の読み取り/書き込みのサポート。
  • プロセススケジューラの時間割り当ての改善。
  • ハードドライブとATAPIデバイスのSATA IDEモードのネイティブサポートが追加されました。
  • システム全体のクリップボードのサポート。
  • システムにvidintelドライバーを含める。統合されたIntelグラフィックスカードを搭載したラップトップでのネイティブアクセス許可のサポートを含む。
  • Intel HD Audioオーディオドライバーの更新。
  • IntelおよびATI Radeonグラフィックスカードのドライバー更新。
  • 一部の構成での読み込みの加速。


そして、KolibriN9自体の変更について：



  • 再設計されたシステム設計により、設計はより美しく、楽しくなりました。
  • Docky（ドックバー）の使用により、使いやすさが向上しました。
  • 追加ディレクトリ/ kolibrios /の接続のサポートによるKolibriNの即時起動。


KolibriN専用に書かれたもの：



  • Game Center-KolibriNの一部として提供されるゲームを起動できるゲームセンター。
  • 外観-システムのテーマとデスクトップの背景をすばやく便利に変更するためのプログラム。

Pixie Player v1.X

（2015）

「KolibriOSの音楽：完全なレビューと最新ニュース」の開発に関する記事


つまり、PixieはminiMP3のフロントエンドであるMP3プレーヤーです。 フォルダーからファイルを開くと、プレーヤーは選択した曲を開始し、このフォルダーのすべてのMP3もプレイリストに追加されます。



ウィンドウには、通常モード、タイトルに折りたたまれたモード、コンパクトの3つのモードがあります。 ユーザーが選択したトピックのうち、明るいものと暗いものの2つが利用可能です。 すべてのプログラム設定は、再起動後に保存されます。 パラメータなしでプレーヤーを起動すると、最後に開いたフォルダーの再生が開始されます。









コンパクトモードでウィンドウを移動すると、フレームが移動するのではなく、ハミングバードで初めてウィンドウ自体が移動します。
