KolibriOSはKickstarterを起動してSummer of Codeをホストします

KolibriOSは、MenuetOS（2004年にフォークが発生）に基づくオープンオペレーティングシステムであり、ロシア語の開発チームがアセンブリ言語のみで実行および開発するために必要なRAMは8 MBのみです。 彼女のディストリビューションには1.44 MBのフロッピーディスクが1枚ありますが、テキストエディター、画像を表示するためのプログラム、ブラウザー、30種類以上のゲーム、さまざまなテスト、システムおよびアプリケーションユーティリティが含まれています。 読み取りモードで完全にサポートされているFAT12 / 16/32-NTFS、ISO9660およびExt2 / 3/4。 一般的なサウンド、ネットワーク、およびグラフィックスカードのドライバーが記述されています。



今年、プロジェクト管理者はGoogle Summer of Code 2013への参加を申請しましたが、彼らの申し出は拒否されました。 さらに、Googleはプロジェクトが気に入っていると書いていますが、予算が限られているため、416人の応募者全員を受け入れることはできません。 GSoC '13で実装される予定のKolibriOSイノベーションのリストは非常に広範囲であり、開発計画にある意味ですでに含まれているため、独自のSOCを編成することが決定されました。 このプロジェクトは非営利であり、寄付ベースであるため、小規模なプロジェクトが資金を迅速に集めることは依然として困難であるため、開発者はKickstarterを選択しました。 キャンペーンは3日前に開始され、これまでに20,000ポンド以上が5,000ポンドから引き上げられました。 資金調達は5月31日に終了します。



KolibriOSをダウンロードして、このページでプロジェクトをサポートするかどうかを決定できます： kolibrios.org/en/download.htm

Kickstarterキャンペーンのアドレス： www.kickstarter.com/projects/kolibrios/kolibrios-help-us-hold-our-own-summer-of-code-2013
