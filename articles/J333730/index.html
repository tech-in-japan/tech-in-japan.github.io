FSBの通知がない機器のインポート方法に関するステップバイステップマニュアル

Habré（およびRunet）には、海外から注文した機器に関するFSBの通知がない場合の対処方法に関する指示が実質的にないことに気付きました。 アメリカからラップトップを配達する過程で得た経験を共有したいと思います。 カットの下の指示。







簡単な背景

私の兄は、古いラップトップを交換することを決めて、モデルの選択を手伝ってくれと頼みました。 サイトのレビューの1つで、コメントでLG Gramのようなエンジニアリングの奇跡を滑らせました。 対角15インチのこのラップトップの重量は1キログラム未満で、中身がよく、見た目もとても良いです。 ロシアでは、このラップトップは販売されていないため、Amazon Webサイトで検索を続けました。 当初、モデルの価格は約1,100ドルでしたが、私は長年海外で買い物をした経験があり、待つことに決めました。 その結果、価格は830ドルに落ち、それが取ることにした。 ラップトップは、メール転送サービスのいずれかの倉庫に正常に到着し、出荷の準備が整いました。 しかし、ここから楽しみが始まります。 次の手紙がメールで届きます：







こんにちは、％ユーザー名％！



税関で転送された小包のデータの予備承認で、後者はロシア連邦のFSBの通知がないため、小包を送信するためにあなたの小包＃******からラップトップを承認しませんでした。 そのため、このようなパッケージを***メソッドで送信することはできません。



保険なしでUSPS Express / Priorityメソッドのみを使用して、このようなパッケージを送信する準備ができています。 ただし、この場合、デバイスへの通知がないため、税関から商品を返送するリスクがあることを理解することが重要です。



個人アカウントの「パック済み」セクションで送信をキャンセルすることをお勧めします。 出荷をキャンセルした後、USPSに申告を再登録し、USPS Express / Priorityの新しい発信小包を作成することができます。



おそらく現時点での解決策の1つは、商品を売り手に返品することです。

私は自然に情報を探し、 ECE Webサイトにアクセスします（非常に不便ですが、どうすればよいのか）。このモデルは実際には見つかりません。 問題を解決するための4つのオプションがあります。







  1. 私が幸運であり、小包が税関によって選択的にクリアされないことを期待して、通常の郵便で送信する
  2. 返品と入金
  3. 法的な通知とインポート
  4. 2017年5月、別の方法が登場しました-観光客を介して商品を輸入し、その住所への配達を注文できるサービス。 私はこれらを使用しませんが、誰かがそれを役に立つと思うかもしれません


一般的に、兄弟はすぐに最初の方法を使用することを申し出て、自分でお金を失うリスクをすべて負いました。 この方法の主な欠点は、ロシア郵便局での頻繁な窃盗のために完全な保険を暗示していないことです。 これは、ラップトップを取得しない追加の機会を与えました。 そして私は、税関ではなく、ロシアの郵便局を恐れていました。そのような機器の盗難の可能性は常に高かったからです。 サービスのテクニカルサポートでは、わかりやすい回答を得ることができませんでした。 その結果、私はすべてを自分で決めなければなりませんでした。 最初に、ストレージ期間を超えて支払った金額の割引と引き換えに、この記事をWebサイトに投稿することを提案しましたが、明らかに15ドルから20ドルは多額であり、拒否しました。







ウクライナのベラルーシを経由して知人を引き渡すなどのオプションや、スペアパーツを含むコンテナを送信するなどのエキゾチックなオプションも検討されましたが、すべてが実行可能ではありませんでした。 たとえば、ベラルーシとウクライナでは、通関制限は100ドル未満です。







では、ロシア連邦の領土に公式に機器を輸入する方法は？ 2つの方法があります。







  1. FSBから通知を受ける
  2. 個人のニーズに合わせて法人または個人の起業家としてインポート


最初の方法は、実際には不可能、または高価です。 私の知る限り、この通知は機器の製造業者が受け取る必要があります。 インターネットには、10000rからこれらのサービスを提供する多くの企業があります。 ご存知のように、このオプションは割引のすべての利点を完全に無効にするため、適切でもありません。 2番目のオプションはより興味深いものです。 ご存知のように、法人はLLCや他の企業としてだけでなく、個人起業家としても理解されています。 それから、私はオープンIPも持っていて、タスクが単純化されたことを思い出しました。







私の最初の旅行は、3つの駅のある地域の税関部門へでした。 役に立たない情報に加えて、彼らは税関では小包の選択が十分に速いことを教えてくれました：小包が通知をオンにする技術を示しているが、この通知が小包に示されていない場合、小包は一緒に保存されているため、開かずに送り返されます提供されません。







次はFSB、すなわちロシアのFSBの中央専門訓練センターへの呼びかけでした 。 すぐに言いたいのは、非常に正気な人々がそこで働いていることです。彼らはすべてを詳細に説明し、文書を正しく作成するのを助けます。 彼らは、民間の起業家は実際に個人使用のために機器を輸入する許可を得ることができると言った。 手順はこのドキュメントで説明されます 。 これを行うには、ロシアのFSBの中央制御安全保障理事会の事務所の住所に手紙を送る必要があります（モスクワ107031、ボルシャヤ・ルビャンカ通り、1/3）。







手紙に同封する必要があります：







  1. ロシアのFSBの中央防衛評議会の議長宛ての申請書
  2. 技術仕様の印刷
  3. パスポートのコピー
  4. IPおよびTINの登録証明書のコピー
  5. 完成した意見（2012年5月16日の第45号のユーラシア経済委員会の決定に対する付録第1号）
