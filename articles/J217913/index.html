Dell Venue 8 Pro Windowsタブレットエクスペリエンス

しばらくの間、DELL Venue 8 Proタブレットを使用して、自分の印象について書きたいと思いました。 はじめに、なぜそれが必要なのか、なぜAndroidタブレットが私に合わなかったのかを説明します。 :)たくさんの文字を読むのが面倒で、TTXタブレットを使用してタブレットまで数ページ下にスクロールします。 写真よりも文字数は増え続けますが、ボックスの写真やタブレットのすべてのボタンをさまざまな角度からアップロードすることはしません。 使用経験について詳しく説明します。







現在、私は3つの主要なモバイルデバイスを家に持っています：Sony Xperia ZR電話、Samsung Note 8.0タブレット、およびAsus U36SGラップトップ。



タスクは次のとおりです。

電話はいつもあなたと一緒です。 メール、インターネット、書籍、あらゆる種類の歩行者ナビゲーション（都市地図、公共交通機関カード、チケット注文など）、さまざまな支払いプログラム（銀行、qiwi、Yandexなど）、緊急の場合-リモートデスクトップ管理（ rpd、vnc、teamviewer）、mp3プレーヤー、ラジオ、目覚まし時計、電卓、evernote ...一般的に、まさにパーソナルデジタルアシスタントと呼ばれるもの。 最近、 スマートウォッチによって成長しています。



タブレット-基本的には自宅で生活し、1〜2日間旅行に出かけ、時には仕事に出かけます。 自宅にいるとき-ソファの上のインターネット、pdf、djvu、cbr / cbzのリーダー、手書きのメモ（筆記用具が好きな理由）、職場のとき-紙が翻訳されないようにメモします。 旅行中-インターネットとリーダー。電話がバッテリーを節約するため、または大画面が必要な場合（pdfは同じで、「4人よりも読みやすい」）。 ナビゲーターとしては使用せず、一般的に彼は電話後の傍観者です。



ノートブック（i3 2310M 2100 Mhz / 8GB RAM / 13.3 "/ 256Gb SSD）-2日以上の旅行にかかります。タスク-テキストの操作、カメラ（ライトルーム）とそのストレージからの写真の初期処理、GPSロガーからのトラックのインポート、快適退屈なときのリモートコントロール-ゲーム（個別のビデオ、非常に強力ではありませんが、タンクの最小設定には十分です）。







ただし、タブレットの「モード」でノートブックタスクを実行したい場合があります。 たとえば、RAWの並べ替え、オフィスドキュメントの完全な表示、またはゲームのプレイ（もちろん、シューティングではなく、文明、ヒーロー、またはクエストがタッチスクリーンで非常にうまく機能します）。 ソファに横たわって、バスに乗って、飛行機に乗って、ハンモックで立って、これをすべてやりたいです...

Androidでは、これは原則としてすべて可能ですが、ソフトウェアの制限があります。 Photo Mate R2は RAWのルックアップとして非常に優れていますが、高速であり、多くの形式をサポートしています（ORFで問題なく動作します）。 しかし、これはPCゲームや本格的なオフィスの問題を解決するものではありません。 そして、ラップトップはキーボードの邪魔になります。

私はすぐに言わなければなりません-GPS、3G、および他のカメラはタブレットでは私には興味がありません。 大丈夫ですが、彼らの存在/不在は選択する際の要素ではありません。 これが私の携帯電話にあるすべてです。



最良の選択肢は、タブレット+ノートパソコンの束をASUS T300のようなものに置き換えることです。 しかし、ヒキガエルは首を絞めました。 彼女は頻繁に介入し、彼女の美しい衝動を破壊します。

しかし、ラップトップにそんなにお金を払っても大胆ではないでしょうか？ あなたはまだスケートを降ろしませんでした 。 あなたはまだまともな古いものを持っています。 とにかく、タブレットは1キログラムよりも重いので、 Compaq TC1100を購入します。




私は強力なタブレットトリナシュカのアイデアが好きでしたが、ラップトップを変更する必要があるときは、この方向に目を向けます。 その間、私は別の方向に目を向け始めました-ペンを使用する機能を備えた通常のWindowsプログラムを実行できる安価な8インチのタブレット。そして水平線上で新しい原子のタブレットが迫りました。



私は販売を待って、それらを見て、Dellを購入しました-それは当時最も安いものでした-私のコピーは宅配便で約210ドルかかりました。 本格的なスタイラスを購入するだけです。最初のリビジョンはだまされており、今では通常のバージョンが登場しています。



もちろん、このプラットフォームの他のモデルのメーカー-Lenovo Miix2 、 ASUS VivoTab Note 8 、 Toshiba Encoreもあります。 これらはすべて価値のあるオプション、特にAsusですが、購入時にはDellの方がはるかに安く、残りは配達で約280ドルから始まりました。 さらに、東芝とレノボはスタイラスを購入しません。



さて、これでタブレットの紹介で終わりました。

2つのポートがあります。microUSBとヘッドフォン用です。MicroUSBは上端にあり、ヘッドフォンは左側のWinボタンの隣にあります。このボタンは奇妙な場所にあり、常に中央の前面右側に手を触れたいと思っています。私はそれを使用しません、私はスクリーンを好みます。



ベンチマークはほとんど表示されないため、立ち上げません。 のように感じます-システムは十分に高速で、以前のアトムよりも大幅に高速です。 私は使用中に明らかなブレーキに気づきませんでしたが、いくつかの低迷があります。



バッテリーの寿命は測定が困難ですが、Galaxy Note 8.0のレベル（約6〜8時間）で動作しているように感じます。 おそらく長持ちするかもしれませんが、私はまだ定期的にプレイしており、このバッテリーはブラウザでページを表示するよりもはるかに速く植えています。

ゲーム中または映画鑑賞中、タブレットは大幅に熱くなります。 私はそれを暑いとは言いませんが、むしろ「とても暖かい」と言います。 保持は苦痛ではありませんが、少し面倒です。



ハードウェアの欠点：

  1. 最大の欠点はmicroUSBです。

    • まず、周辺機器を接続するにはアダプターが必要です。
    • 第二に、彼はフラッシュドライブやキーボードよりも深刻な何かを供給する力に欠けています。 外付けハードドライブ、3Gモデム、または家庭内の他の便利な機器を接続する場合は、追加の電力を供給する必要があります。 そのため、外部ネジはコンセントの近くでのみフックできます。
    • 第三に、タブレットは同じコネクタを介して充電されます。 また、たとえば、フラッシュドライブを使用してタブレットを充電することはできません。


  2. Miracastのみを使用して、外部モニターをケーブルで接続する方法はありません。 そのようなすべてのタブレットの中で、東芝だけがmicroHDMIをインストールしましたが、ASUSとLenovoもmicroUSBとヘッドフォン出力を必要としました。
  3. 絶対にアップグレードすることはできません。 プロセッサ、メモリ、「ハードドライブ」-すべてがマザーボードにはんだ付けされ、1つの部品に置き換えられます。
  4. スピーカーは1つだけです。スピーカーは横にあり、タブレットを横長モードで保持すると、手と重なります。
  5. 5V / 1Aの充電器からは充電せず、2Aにします。 そのため、新しい外部バッテリーを探す必要があります。 インターネットは同様のケーブルで遊ぶことができると示唆していますが： https : //www.dx.com/p/usb-male-to-micro-usb-male-charging-data-cable-w-switch-for-samsung- galaxy-tab-3-black-100-cm-241044-そこで、データ接点はスイッチによって閉じられます。




ソフトウェア部分について


タブレットにはWindows 8.1がプリロードされていたため、更新プログラムを台無しにする必要はありませんでした。

私が気にするOS自体については説明しません-長い間探してきました。 :)

私はタッチスクリーンでの作業が好きで、画面の端からのジェスチャーが便利です。 今では、Androidタブレットでもそれらを使用しようとしています（そして、Windowsでは、ホームボタンを常に模索しています）。

さまざまなプログラム用に画面を分割すると便利です。Samsung用の同様のモードよりも優れています。

奇妙なインターフェイスソリューションがいくつかありますが、スクリーンキーボードは悪くありません。

タッチスクリーンを備えたデスクトップモードでは、特に7台にタブレットを置いていたので、生きることも可能です。 しかし、ここでは、Bluetoothを介して問題なく接続されているため、ハードウェアキーボードとマウスを使用する方が便利です。

適切なゲームは問題なく動作します。 確かに、これまでのところ、BG2：EEおよびHeartstoneに限定されています。 BG2はタッチスクリーンで正常に動作しますが、スクロールには不便な点がいくつかあります。 そして、Heartstoneはタッチスクリーン専用に書かれているようです。 なぜアンドロイド用のバージョンがないのだろうか。
