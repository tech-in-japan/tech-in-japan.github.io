ウィークリービルドVivaldi 1.0.167.2

みなさんこんにちは！



今日はロシアでの休みですが、ヨーロッパとアメリカにあるビバルディのオフィスでは、仕事は止まることなく続けられます。 今日、次のウィークリービルドを準備しました。このビルドでは、TP3のリリース以降に見つかったエラーを修正しました。



バグに取り組むことに加えて、キーボードショートカットの機能を改善しました。 必要な組み合わせの選択がより簡単になり、複数の組み合わせを1つのアクションに割り当てる機能も追加されました。



そして、事前にネットワーク内の絶対的なプライバシーの支持者を安心させるもう1つの重要なメッセージです。 このアセンブリから始めて、統計情報の収集を開始します。 ユーザーが誰であるかをよりよく理解するために、使用しているハードウェアとOS、居住している国、インターフェースに使用する言語を確認する必要があります。 このようにして、お客様のニーズに焦点を合わせ、バグ修正と新機能の追加に優先順位を付けることができます。



仕組み。 このバージョンのVivaldiブラウザーを初めて起動すると、ご使用のブラウザーのバージョンに対して個別のIDが生成されます。 その後、ブラウザは24時間に1回、セキュリティで保護されたHTTPSプロトコルを介して（もちろん動作する場合）サーバーに次の情報を送信します。



  • 一意のID
  • ブラウザのユーザーエージェントデータ
  • CPUアーキテクチャ
  • 画面解像度
  • ブラウザインターフェイス言語




もちろん、これらのデータは私たちだけを対象としています-統計は個人化されておらず、商業的価値を表すものではありません。



以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
