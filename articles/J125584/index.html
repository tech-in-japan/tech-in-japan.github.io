Mac OS X 10.7 LionにNetbeans 7.0.1をインストールします

私は最近、MacBook ProとMac OS Xユーザーの所有者になりました。



私はこの「獣」を楽しみのためではなく、仕事のために買いました。 さて、私はすぐにシステムの構成を開始しました：コンパイラー、インタープリター、IDE、さまざまなライブラリーおよび拡張機能、および開発または単純化に必要な他のソフトウェアのインストール。



この投稿のおかげでシステムを「ライオン」にアップグレードした後、それは完全に無料であり、アプリストアから新しいXcodeをインストールし、MacPortまたはDMGパッケージからアプリケーションをインストールし、時計仕掛けのようになりました。 しばらくして、キューはNetbeans 7.0.1に到達しました。 ダウンロードを考慮に入れて、5分以内にインストールされたJetBrains製品をインストールした後、NetbeansはJavaが表示されず、インストールを拒否したと述べました。



私の場合の解決策は公式バグトラッカーで見つかりました ：

# ,

sudo mkdir -p /System/Library/Java/JavaVirtualMachines/

sudo ln -s /Library/Java/JavaVirtualMachines/1.6.0_26-b03-383.jdk /System/Library/Java/JavaVirtualMachines/1.6.0.jdk









その後、インストールを開始し、「次へ」ボタンを数回クリックして、「完了」ボタンが表示されるのを待ちます。



Netbeansは人気のある無料のIDEであり、他のユーザーがこのような問題を経験する可能性があるため、このバグに関するメモを公開することにしました。



PS一般に、私の記事がだれにも役に立たず、すべてのインストールがスムーズに進むことを願っています。 そして、タスクを割り当てられた開発者であるJulia Novozhilovaはすぐにこの問題を解決し、記事の必要性は自然に消えます。
