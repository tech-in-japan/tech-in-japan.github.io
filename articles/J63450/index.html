Mac mini：臓器移植の検死を行います:)

しばらく前に、ときどき、G4（PPC）プロセッサでMac miniを購入しました。







通常のコンピューターのCD / DVDドライブと比較することで、おおよそのサイズを見積もることができます-Mac miniはもう少しだけです。



標準装備（2005年7月から10月に生産）：







私は解読します：



CPU：PowerPC G4 1.42GHz

RAM：512MB DDR

HDD：80GB 2.5 "PATA

CD / DVD：コンボDVD + CD-RW PATA

ビデオ：Radeon 9200 32MB DDR

ネットワーク：10 / 100Mb Lan Card + 54Mb AirMac（WiFi）+ BlueTooth

ポート：USB2.0 + IEEE1394



彼の手でそれを回して、私はそれが単純なメディアセンター（1080pビデオは引っ張らないので単純なもの、最大720p）であり、ファイルサーバー（急流を駆動するがバッファも使用する）であり、インターネットの閲覧に適していることが判明した。 これらの目的のために、80GBのハードドライブは完全に十分ではなく、DVDドライブは空白を書き込まず（読み取りのみ）、Mac OS Xはしばしば512MBのメモリで非常に思慮深く動作します。 したがって、可能な限り最大にアップグレードすることが決定されました。 ビンとインターネットをざわめくと、彼は次の構成で停止しました（変更箇所は太字で強調表示されています）。



CPU：PowerPC G4 1.58GHz （オーバークロックにより、プロセッサがM.P.にはんだ付けされており、交換時にいくつかの問題が発生するため:)）

RAM： 1024MB DDR（これは設定可能な最大値です）

HDD： 250GB 2.5 "PATA +外部500GB USB（内部は320GBまたは500GBに置き換えることができますが、すでにかなり高価であり、これはSATA対応品よりもはるかに高価です）

CD / DVD： DVD-RW DL PATA

ビデオ：Radeon 9200 32MB DDR

ネットワーク：10 / 100Mb Lan Card + 54Mb AirMac（WiFi）+ BlueTooth

ポート：USB2.0 + IEEE1394



詳細：Hynix PC3200 1GBを使用したメモリ：







ウィンチェスターWD2500BEVE 5400rpm：







NEC AD-7640A Slotted Loader Drive：







始めましょう。 まず、蓋を開ける必要があります。これは、蓋​​が内部ラッチに釘が1本も取り付けられていない状態で取り付けられているためです。 :)インターネットで一括して解体するビデオなので、このプロセスは撮影しませんでした。ここでこのアクションを見ることができます。例： http : //www.youtube.com/watch?v=U2CExAXtnis



どこでもヘラを使うことをお勧めしますが、私はそのような器具を手元に持っていなかったので、許容できる状態でしたので、普通の包丁を使用しなければなりませんでした。 検死自体は10秒以上かかりませんでした。



メモリ、ハードドライブ、およびドライブを交換します。何にでも説明することもできますが、それはすべてが些細なことであるため、多少の驚きもあります。 :)この段階では、ツールには小さなプラスドライバーのみが必要です。 外観は次のとおりです（新しいハードドライブとメモリは既にインストールされていますが、ドライブは削除されています）。







さて、そして最後にスポーツの興味からこの奇跡を分散させます。 ここでは、1つのドライバーだけでは対応できません。また、細い刺し傷のはんだごても必要です。 メモリ、ドライブ、ハードドライブを取り外した後、マザーボードをケースから取り外し、角のボルトを外します。 次に、M.P。の下側のクーラーマウントの近くで、このグループの抵抗器を探しています。







そして、抵抗器番号R351をはんだ付けします。 これでオーバークロックは終了し、周波数は1.58GHzになりました（参照により他の組み合わせがあります）。 ただし、1つの警告があります-ファームウェアは1.5 GHzより高い周波数のプロセッサを認識せず、わずかに収束します-eboutのプロセッサは750MHzとして定義されます。 867MHzプロセッサの最小要件があるため、オーバークロックを削除するか、パッチを適用したディストリビューションをインストールするか、インストール中にコマンドラインから制限をバイパスする必要があります。 他の2つについては後で知ったので、最初のオプションを使用する必要がありました。



オーバークロック前後の速度の小さな比較（BOINCのベンチマーク）：



宛先：



測定された浮動小数点速度903万ops /秒

測定された整数速度2759ミリ秒ops /秒



後：



測定された浮動小数点速度1019,000,000 ops /秒

測定された整数速度3068百万ops /秒



それですべてです。誰かが興味を持ち、役に立つことを願っています。 :)
