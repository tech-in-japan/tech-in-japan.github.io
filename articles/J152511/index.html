Google Web Fontsのロシア文字フォントニュースバンドル

Jovanny Lemonadとしてよく知られているフォントデザイナーIvan Gladkikhが、6月上旬に無料で無料の非公式のUnderdogフォントをリリースしたことを思い出してください。 このフォントプレビューの新しい著作権バージョンは次のとおりです。







今日（9月27日）、このフォント（ Underdog ）がGoogle Web Fonts Webサイトに登場しました （たとえば、 必要な文字を選択して、サイトリーダーに提供されるフォントファイルのサイズを最適化できます）。



さらに、政治的配慮に導かれたGoogle Web Fontsのマネージャーの1人は、 フォント「 スターリン 」の作者に名前の変更を強制しました 。 そして強制。 Google Web Fontsウェブサイトのこのフォントは「 スターリン主義者 」と呼ばれていますが、このサイトは作者の元の名前を保持しています。 名前を変更すると、同じ名前でGoogle Web Fontsから接続できたすべてのサイトでこのフォントの機能が失われる可能性があります（その可能性が高くなります）。それに応じてコードを変更することを忘れないでください。



そして来週、 Yesevaフォントの外観が登場するかもしれません。最終作者のプレビューは次のようになりました。



このフォントの太い線と細い線のコントラストが顕著になることは簡単にわかります。
