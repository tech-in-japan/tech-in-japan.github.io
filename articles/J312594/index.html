LinuxとWindowsでの1Cシステムのパフォーマンスの比較

Linux 1Cインフラストラクチャ1Cの実装は古くからのトピックですが、依然として重要です。 Linux上の1C Application Serverという記事を最近公開しましたが、Windowsのソリューションと比較した実際のパフォーマンスの問題は未解決のままです。 テストは手動モードで実行されましたが、客観性のために、異なるオペレーティングシステム（Linux CentOS 7とMS Windows Server 2012）を使用して同じハードウェアプラットフォームで合格したGilevテストの結果を公開します。



サーバーとして、2つのIntel Xeon E5-2670プロセッサー、8x4 GBのRAM、およびIntel SSDを備えたスタンドを使用しました。

受け入れ可能なテスト結果、展開の容易さ、低ライセンスコストにより、完全な製品である1C Linuxベースのサーバーをすぐに作成できました。



Windowsプラットフォームには、 同様の製品ラインがあります。



各モデルには、タスクに最適な構成を選択できる便利なコンフィギュレーターがあります。 これについてもう少し説明します。



1Cサーバーの計算能力の計算機を作成することは簡単な作業ではありません。 そして、考えられるすべてのケースに対応したユニバーサル1Cコンフィギュレーターの作成はほとんど不可能です。



確かにハブには多くの管理者がいます;彼らは、1C未満のサーバーの計算能力の負荷と要件について独自の考えを持っています（コメントはこの記事に価値を加えるでしょう）。 公式の推奨事項1Cがあります。この勧告では、すべてがすべてに機能します...



しかし、一般的な操作スキームに適用可能な計算可能な基本的なパラメーターがまだあります。 ターミナルセッションに必要なプロセッサおよびRAMリソースの量、特定のユーザー数に対してSQLが必要とするIOPSの数を把握し、多数のテストの結果に基づいて、1C用の標準ソリューションコンフィギュレーターを開発しました。



もちろん、各ケースは一意であり、詳細な事前販売が必要ですが、おおよその計算には、ユーザー数、データベースのタイプ、現在のデータベースのボリュームを示し、必要なライセンスを選択するだけで十分です。



LinuxとWindowsに基づいたターンキーソリューションのコストを比較するために、コンフィギュレーターから小売価格の例を示します。



最大80GBのSQLベースを持つ20ユーザー用のサーバー、ライセンス1C：Linux CentOSに基づくアカウンティング8 PROFは、522 759.43ルーブルの費用がかかります。 Windowsに基づく同様の構成-1,036,279.43ルーブル。



1C STSS Flagmanのサーバーのラインナップは、LinuxとWindowsの両方の3つのモデルで構成されています。









1C113.5-020UL -1Cエントリレベルサーバー、SQLデータベースで最大20ユーザーをサポート。 ディスクスペースの量は、ベースの成長を考慮に入れて3年間年間20％で計算されます。 RAID1アレイは、IntelのエンタープライズSSDに基づいています。 「コールド」データ用にデュアルPSUと追加ディスクをインストールすることが可能です。 PostgreSQL、xrdp、およびhttpdのソフトウェアサービスを選択できます。



1C216.4-200UL-最大200の同時接続を行う1Cインフラストラクチャを提供する2プロセッサプラットフォームに基づくモデル。 ストレージは、同じ原則-成長を考慮したデータベースのサイズ-に従って計算されますが、必要なボリュームの4xSSDのRAID10アレイに基づいて構築されます。



1C217.2-050UL-REFは、工場で復元されたサーバーに基づいて構築された限られた予算のお客様向けのソリューションです（保証交換、デモ資金などの後）。サーバーは、出荷前に新しいモデルと同じ負荷テストに合格します。保証期間が短くなります（1年）。 サーバーは、最大50の接続をサポートし、ライセンスを除くと、203 GBのルーブル、40 GBベースのアレイが必要です。









1C113.5-020UW-サーバー1Cエントリーレベル、SQLデータベースで最大20ユーザーをサポート。 ディスクスペースの量は、ベースの成長を考慮に入れて3年間年間20％で計算されます。 RAID1アレイは、IntelのエンタープライズSSDに基づいています。 「コールド」データ用にデュアルPSUと追加ディスクをインストールすることが可能です。



1C216.4-200UWは、最大200人のユーザーをサポートするWindowsベースのモデルです。 ストレージは、必要なボリュームの4xSSDのRAID10アレイに基づいて構築されます。
