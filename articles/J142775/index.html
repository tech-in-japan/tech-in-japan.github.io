第2回Evernote Devcup開発者コンテスト





第2回の毎年恒例の開発者コンペティションであるEvernote Devcupを発表します。 Devcupの一環として、ソフトウェア開発者とデザイナーは、2,500万人のEvernoteユーザーがアカウントの機能を拡張できるようにする興味深いアプリケーションとサービスを考え出し、実装するように招待されています。 コンテストの賞金総額は100,000ドルを超え、ファイナリストは今年のサンフランシスコで開催されるEvernote Trunk Conferenceで作品を発表する機会を得ます。



次の賞品を用意しました。

  • ゴールド -$ 20,000
  • シルバー -15,000ドル
  • ブロンズ-5,000ドル
  • 6名のファイナリスト -各10,000ドル、およびETC 2012期間中のフライトと宿泊


何を見たいですか？



成功するアプリケーションは、魅力的な機能と美しく直感的なデザインを組み合わせています。 私たちは毎日何万人もの人々が決勝戦に参加することを期待しており、彼らは自分たちの生活の中で重要なことを保存し、思い出すことができます。 これまでに聞いた良いアイデアのすべてをリストすることはできませんが、将来のプロジェクトについての考えに役立ついくつかの一般的なトピックを提供してください。



  • Evernote FoodとEvernote Helloは、生活の特定の領域で使用するための追加のEvernote機能をユーザーに提供するモバイルアプリケーションです。 おそらく、ユーザーが情報を構造化された適切な方法でEvernoteに保存したい他の領域に注意を払う必要があります。 これは、ユーザーが見たい映画のリストと、すでに見たものに関するメモを含むアプリケーションの場合があります。 このようなアプリケーションは、ユーザー自身が追加した情報と外部リソースのデータを組み合わせることができます。
  • Nike +やFitbitなどの製品は、多くの人々が日々の出来事や人生の成果を記録する機会を提供します。 これらのアプリケーションは、私たちに関する多くの情報を生成します。 ここでEvernoteを適用する方法は？ そのようなアプリケーションのどのデータをEvernoteに保存するのが理にかなっており、Evernoteユーザーがすでに蓄積した情報を使用して、そのような毎日のエントリを人間にとってより意味のあるものにする方法は？
  • 昨年のファイナリストの1人は、ユーザーがEvernoteで2種類のノートを作成することを指摘しました。 買い物リストなどの一部は、最初は非常に重要ですが、時間とともに価値を失います。 思い出に残る夕食のメモなど、その他のものは短期的にはあまり重要ではありませんが、後で貴重な楽しい思い出になります。 Evernoteノートの世界をこれらのカテゴリに分類する場合、これらのタイプのノートのいずれかで動作するように将来のアプリケーションの方向を考えることができます。
  • Evernote製品は、世界中の何千万人もの人々によって使用されています。 ユーザーの3分の1のみが米国に住んでいます。 私たちは価値と有用性において普遍的であると信じる製品を作成しますが、特定の国のニーズを満たすアプリケーションは人々にとって有用であると思われるかもしれません。 おそらく、あなたの国で大きな需要があるアプリケーションを作成することができ、その必要性は外部の観察者にはそれほど明白ではありません。
  • Evernoteユーザーは、当社の製品について詳しく知ることができます。 Evernoteの使用を開始したり、新機能を習得したりするのに役立つ書籍、ブログ、ビデオは非常に人気があります。 日本でのみEvernoteに関する33冊の本が出版されました！ おそらく、Evernoteでガイドとチュートリアルを作成することをお勧めしますか？




これらは表面にあるトピックのほんの一部です。 Devcupに参加する予定がない場合でも、どのアプリケーションが必要だと思うかを知りたいと思っています。 4月27日金曜日にモスクワで開催されるEvernote Dev Meetupのデベロッパーとのミーティングで、この（および全体のコンテスト）について話します。



昨年の経験





昨年、世界中から約1000人の開発者が私たちの競争に参加しました。 多くの人々はアメリカから、アジアからは約4分の1で、ヨーロッパからも同様でした。また、南米および中央アメリカからの開発者もかなり目立っていました。 6つの最終チームすべてがETCを訪れ、その成果を発表しました。 これらのプレゼンテーションは、 evernote.com / about / etcで見ることができます。 昨年、メイン賞はTouchanoteに贈られました 。 これは、NFCタグを使用して、実世界のオブジェクトをEvernoteのノートに関連付けることができるAndroidアプリケーションです。 もう1つのファイナリストアプリケーション-Notable Meals-は、Evernote Foodの基盤を形成しました。 第三-Colorstache -Evernoteのノートで特定の色の存在を検索できます。



詳細



Evernote APIに統合されているソフトウェアはすべて、Devcupに参加するために送信できます。 コンピュータープラットフォーム、モバイルデバイス、またはWeb用のアプリケーションを開発できます。



重要な日付：

  • 今日から6月25日まで -チームビルディング、アイデア、アプリケーションの作成。
  • 6月25日は申し込みの最終日です。
  • 7月23日 -6人のファイナリストの発表。
  • 夏の終わり -Evernote Trunk Conferenceでの最終作品のプレゼンテーション、グランプリおよび受賞者の決定。






このコンペティションの開始の一環として、世界中でいくつかのオフライン会議を開催しています。 アイデアの可能性と実現可能性についてスタッフと話し合ったり、他の開発者とチャットしたり、Evernoteの詳細を学ぶことができます。 前述のように、モスクワでは、Evernote Dev Meetupが4月27日金曜日に開催されます。 ここでミーティングについてもっと書きました 。 また、突然直接会えない場合は、ウェビナーを開催する予定です。 開発者の競争については、Evernote Devcup-devcup.evernote.comで詳しく知ることができます。
