Visual Studioオンラインアナウンス

今日、Visual Studio 2013の公式リリースが行われ、Visual Studio Onlineの発表が行われました。 開発者は、Windows Azureでホストされるクラウドサービスを使用し、統合スクリプトを使用してアプリケーションを作成、展開、管理する機会があります。 新しいツールには、Visual Studio Online「Monaco」のプレビューバージョン、ブラウザで直接使用できる簡素化されたコードエディター、Visual Studio Online Application Insigts、アプリケーション正常性分析ツール、リリース管理ツールが含まれます。 http://www.visualstudio.com/でこれらのサービスを登録して試すことができます 。 最大5ユーザーまで、このツールキットは無料です。



Visual Studio Onlineとは何ですか？



Visual Studio Onlineは、チーム開発をサポートするアプリケーションを作成するための統合ツールセットです。 完全なALMサイクルを実装するために必要なすべてのツールが含まれています。





Visual Studio Online "Monaco"（プレビュー）



「Monaco」は、Visual Studio IDEを補完するツールで、たとえばクラウドサービスのコードをすばやく変更できます。 プレビューバージョンを使用すると、Windows Azure Webサイトでブラウザーに直接変更を加えて、アプリケーションの新しい作業バージョンを即座に展開できます。





Visual Studio Online Application Insights（招待によるプレビュー）



Application Insightsは、Visual Studio Onlineの新しいサービスであり、さまざまな面でアプリケーションのパフォーマンスを分析できます。



このツールには、受信したデータに応じて、アプリケーションの特定の側面を改善するための意思決定を可能にする監視ツールが含まれています。





プロジェクトポートフォリオ管理（アジャイルポートフォリオ管理）



Visual Studio Onlineを使用すると、大規模システムを作成するチームがプロジェクトの取り組みを共有し、同時にプロジェクトのポートフォリオ全体の進捗を観察し、複数のチームやプロジェクトのレベルで作業がどのように行われるかを理解できます。 このツールキットには、チャートとデジタルインジケータを作成して、現在の状態をすばやく把握する機能が含まれています。





Visual Studio Onlineにはかんばんボードも含まれています。





商用利用可能



Visual Studio Onlineが商用利用できるようになりました。 同時に、課金サブシステムが商用モードになるまで、チームに5人以上がいる場合でも、Visual Studio Onlineサービスは90日間無料です。 Visual Studio Onlineサービスの価格は、 http：//www.visualstudio.com/products/visual-studio-online-overview-vsで入手できます 。 http://www.visualstudio.com/get-started/set-up-billing-for-your-account-vsでセットアップ手順に慣れることができます 。
