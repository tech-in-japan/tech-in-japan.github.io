PuTTY：既存のセッションで配色をすばやく変更する方法

既存のセッションで配色を展開する方法。 自分用に.regファイルを作成しました 。 UTF-8エディターで開いて、変更する必要があります

[HKEY_CURRENT_USER \ Software \ SimonTatham \ PuTTY \ Sessions \ ％session-name％ ] セッションの名前で 、レジストリにインポートします。

色の設定のみが変更され、他のすべてはそのまま残ります。

30秒で、10回すべてのセッションが新しい色で輝きます。







ここでチェーンスキームを取りました。 元のファイルの欠点は、すべてを完全にエクスポートすることですが、色を変更するだけです。
