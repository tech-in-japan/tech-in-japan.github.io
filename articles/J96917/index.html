ブラウザベースのMMOの例を使用して、トラフィックを取得し、そのコストを最小限に抑えます

何らかの理由で、私たちの時代では、ブラウザゲーム（ブラウザベースのMMO）はスタートアップとは見なされず、それらに対処し、起業家に近づけない人々です。 Seedcamp London Week 2008ファイナルに参加した後、個人的な経験からこの印象を得ました。また、i / oベンチャー2010ファイナルで再び確認されました。



したがって、私はオンラインゲームの初心者クリエーターのために興味深いアイデアを共有しようとします。

それで、あなたがあなたのゲームのベータ版を終了し、多くのユーザーを引き付ける必要があるときが来たと想像してみましょう。 もちろん、理想的には、非常に興味深く、理解可能で使いやすいゲームを入手した場合、1人のユーザーが友人を連れてきて、彼はまだ友達であり、口コミが機能し、朝には1000人のプレイヤーがオンラインになります。 そのような成功を達成することは非常に難しいので、夢を見ることをやめて、地面に降りて=）を開始します。



最も一般的で長年の方法は、もちろん紹介です。 しかし、通常、ユーザーが紹介を参照するようにするには、多くのジェスチャーを行う必要があります：引き付けるリンクを見つけ、リソース上に配置し、最初の紹介が到着するまで座って待って、何かがあなたのアカウントに落ちるようにします。



これらの努力を最小限に抑えるために、Vkontakte、MoyMir、Facebook、Googleバズなどのソーシャルネットワークが役立ちます。 APIの特別な知識を必要としない「共有ボタン」などのオプションがあります。



ゲームで「ソーシャルネットワーク」アクションを実行します。その本質は、このボタンをクリックすると、メモまたは壁に参照リンクが追加されることです。 したがって、友人には、彼の友人にリンクが表示されます。 このため、ゲームですぐにボーナスが発生します。



ボタンコード：



ヴコンタクテ 



<script type="text/javascript" src="http://vkontakte.ru/js/api/share.js?5" charset="windows-1251"></script> <script> document.write(VK.Share.button({ url: "http://referallink", title: "Game", description: "Super game", image: "http://game/img/logo.jpg", noparse: true, type: "round_nocount" })); <script>


私の世界 



 <a class="mrc__share" href="http://connect.mail.ru/share?share_url=referallink">  </a> <script src="http://cdn.connect.mail.ru/js/share/2/share.js" type="text/javascript"></script>


ここでは、サイトのメインページにタグを追加して、メールが名前と画像を正しく取得するようにする必要もあります。 



 <meta name="mrc__share_title" content="Game"> <meta name="mrc__share_description" content="I like this game"> <link rel="shortcut icon" href="/img/new_logo.ico" /> <link rel="image_src" href="http://game/img/logo.jpg" />

  <a title="Post to Google Buzz" class="google-buzz-button" href="http://www.google.com/buzz/post" data-button-style="normal-count" data-url="referallink" data-message="I like this game" data-imageurl="http://game/img/logo.jpg"></a> <script type="text/javascript" src="http://www.google.com/buzz/api/button.js"></script>


フェイスブック 



  <a name="fb_share" type="button_count" share_url="http://referallink" href="http://www.facebook.com/sharer.php">Share</a> <script src="http://static.ak.fbcdn.net/connect.php/js/FB.Share" type="text/javascript"></script>


次の図について取得します。







問題に直面していますが、ユーザーが私たちを欺いたかどうか、そして実際にリンクを投稿したかどうかをどのように確認できますか。 そして非常に簡単に、各共有ボタンには、この方法で配置されたリンクの数を示すカウンターがあります。 このカウンターの値を取得するには、次のリンクをクリックしてください。



ヴコンタクテ 



 vkontakte.ru/share.php?act=count&index=0&url=refferallink


私の世界 



 connect.mail.ru/share_count?func=mrc__shareInit&callback=1&url_list=refferallink


フェイスブック 



 api.ak.facebook.com/restserver.php?v=1.0&method=fql.query&query=select%20url%2C%20total_count%20from%20link_stat%20where%20url%20in%20(%27refferallink%27)&format=json&callback=fb_sharepro_render


Googleバズ 



 google.com/buzz/api/buzzThis/buzzCounter?url=refferallink


そのため、これにより、ユーザーが紹介を呼び始めるようにしきい値を下げることができました。



このようなアクションサイトのトラフィックが増加した後、さらに先に進みます。 したがって、MailruとBigmirでカウンターを設定し、それらも上から移動するようにします。 トラフィックの増加によりサイトがランキングで上昇するにつれて、より多くの人々がこれらのトップから移動する必要があります。



さて、これらのツールを使用して、VkontakteまたはFacebookでコンテンツ広告を購入できます。 ちなみに、国ごとに個別に購入する場合、場合によってはクリックあたりのコストは低くなります。



したがって、500ルーブル（Vkontakte）のN登録を受け取った後、上記のツールを使用したこれらのN登録は、別のM登録をもたらします。 したがって、500ルーブルの場合、NではなくN + Mのユーザーを受け取り、トラフィックのコストを数回削減しました。 この方法でトラフィックを3倍安くすることができました。ゲームへの1回の登録で9ルーブルから3倍になりました。



あなたのプロモーションが成功することを願っています。
