Php



  • Serializableインターフェースの置き換え -Nikita Popovからのオファーで、RFCの形式ではまだ発行されていません。 インターフェイスの現在の実装には、セキュリティを含む多くの問題があります。 シリアル化/非シリアル化の呼び出しを必要とせず、代わりに配列を使用するマジックメソッドでインターフェイスを使用することを提案します。 



    public function __serialize() : array; public function __unserialize(array $data) : void;
