TotalFinder-Finderの便利なプラグイン

Mac OS Xのサードパーティのファイルマネージャーに加えて、システムとの統合性が低く、Finderを置き換える（おそらく、 Path Finderの例外を除く-彼はこの点でかなり進歩しています）、通常のFinderをアップグレードできます:)



過去2か月間、 TotalFinderが存在しました。SIMBLを介して動作するファインダー用のプラグインで、ファイルマネージャーの機能をわずかに拡張できます。



ところで、このプラグインの作成者は、 バイザー （QuakeのようなTerminal.app）としての地位を確立した人です。



長所、短所、落とし穴、スクリーンショット、ダウンロード-habrakatの下。









プロから：

•システムへの統合。 実際の統合は同じファインダーであり、より優れています。

•タブ（Chromiumのソースコードのおかげ）

•乱雑なフォルダーを無効にする機能（ローカル、はい！）.DS_Storeファイル—それらは別のフォルダーにスローされます。

•上のフォルダー！ 最初にアルファベット順にフォルダを作成し、次にアルファベット順にファイルを作成します。 タイプごとではなく、プリスとローカライズされた松葉杖のように、アルファベット順に！

•ホットキーを押すと、画面にフェーダーが表示されます。



将来的に：

•総司令官の2パネル

•MacFUSEサポート



マイナスのうち：

•将来的に支払います。 アルファ、そして（ベータ/リリースで）が支払われます。

•要件：Mac OS X 10.6+およびSIMBL

•これまでの最上位のフォルダーはリストビューでのみ機能します。
