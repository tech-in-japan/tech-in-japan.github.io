ネットワークを分散させる時間

（「ユーザーをリソースに引き付ける」の記事 「方法」に触発されました。）



この記事の内容は、タイトルから明らかです。 しかし、この記事にリストされているいくつかのメソッドが理想的に実装されている実際のプロジェクトについて話をしたかったのです。



会社があり、その部門の1つはあらゆる種類のヨーグルトを生産しています。 会社は長い間志向しており、製品を健康的なライフスタイル、薄さ、そして一般に幸福の製品として位置付けています。 したがって、ほっそりした、美しい、笑顔の女性と紳士、そして幸せな、笑っている子供たちは、彼女の箱から私たちを見ます。



ある時点で、同社のマーケティング担当者は、製品の次のプロモーションのための別の動きを思いつきます。 同社は、「当社製品のパッケージングに参加する」という競争を発表しました。 コンテストの申請書を提出するために、十分に長い期間が決定されました。 そのため、ウイルスは可能な限り多くの人口をカバーする時間があります。 コンテストに参加するには、アンケートに記入し、会社のウェブサイトに自分、あなたの愛する人、友人、知人の写真をアップロードする必要があります。 勝者は、自分の写真に対して最も多くの票を獲得した人になります。



私たちは例を挙げません。 個人的には、投票の必要性について本日2回リマインドされました。 私の家族も例外ではないと思います。多くの（非常に多くの）インターネットがあり、ほとんどの人がヨーグルトを食べ、ほとんどの人が名声を欲しています。 関係する家族、遠い親close、近親者、同僚、部下の全員がコンピューターに呼ばれました。 それらの何人が新しい参加者になったかを言うのは難しいです。 この記事の執筆時点では（申請書の提出期限はまだまだ終わりません）、写真付きのページ数は3998（！）でした。 各ページには8人の参加者がいます。 合計-ほぼ3万2千人の参加者と申請の受け入れが継続されます。



誰かが「たわごと！」 そして、私はチャンスがあると思います。 私は、写真を送った人の十分な数が、金銭的請求権を放棄し、会社の裁量で写真を使用する許可を得て、別のファックスを送らなかったと信じているので。 たぶん私は世間知らずで、すべての消費者は非常にunningなため、広告の地下に小さな活字で印刷されたメモ情報を常に読んでいますか？..



典型的な宝くじ。 なぜ参加するのですか？ まず、勝つことは常にクールです。 かばんの中のかばんをハードルのある長距離に乗せても、しばらくサンドイッチを食べても構いません。 寄付の量と量で競う人がいると聞きました。 私はそのような人々を知らないのは残念です。



第二に、マーケティング担当者の別のアイデアは「シュート」します。 FAXが既に処理されている場合は、（！）写真を使用してWebサイトで製品サンプルを見つけることができます。 単なるサンプルではありません。 クールな3次元オブジェクト。 ねじることができます、すべての側面から見て、いくつかの「ショット」を取ります。 お母さんは1日に2〜3回サイトに行き、ヨーグルトのパッケージで子供を見て、優しさの涙を払います。 母親の同僚は、写真とモデルの利点（主に自分の）と欠点（主に他の人）の議論に関与します。 人口の男性部分はガールフレンド/子孫について自慢し始め、同時に同僚とヨーグルトのパッケージとケフィアのボトルからの赤毛を持つブロンドの美徳について議論します。



各参加者は、少なくとも3〜4人をサイトに「連れて行く」と想定できます。 毎日。 10万人の観客、それぞれ1ルーブル...それは...それはそうでしょう...おおよそ、それは1日10万人のユニークな人々です。 その目は、親しい人やあまり親しい人ではない人の幸せそうな顔の背景に会社のロゴを固定するでしょう。 広告の専門家は私を修正することができますが、私の意見では、これは「長期的な肯定的なイメージの作成」と呼ばれます。



冷蔵庫の中を見るだけです。 ヨーグルトと飲料の会社を最もよく思いますか？ 最近では、競合他社を優先しました。



合計 非常に低コストで、会社は以下を受け取ります。

  • ポジティブなイメージ。
  • 常連客。
  • 民俗品。
  • 売り上げを増やします。
  • コンテストを実施するためのデバッグメカニズム。 次のものを起動するには、メディアへの最初の注入のみが必要になります。




クライアントも敗者ではありません：

  • 楽しんでください。
  • 奇跡の場合、彼は勝ちました。 しかし、いや、少なくとも宝くじを買った。 また、安く。 考えてみて、私はかなりの時間を費やしました。
  • 彼は自分自身を一般大衆に見せ、他の人を見ました。
  • 陰鬱な日常生活を背景に、うれしそうな顔の海を見ました。
  • もう一度、私は有権者のように感じ、世界の運命を決定し、少し-選ばれたもの。
  • 高品質の製品を使用しました。




ところで、最後の点に関して。 製品の品質が悪い場合、会社は助けになりません...この時点で、すでに声明の詳細な説明を展開し始めました。 そして、私はそれについてまったく何も言わないことを望んでいることに気づきました。



この記事を「スタートアップ」に掲載した理由についてお話ししたかったのです。 前述のように、コストは「ごくわずか」です。 しかし、私の知り合いの一人が言うように：「私はすでにこれらの小さな雨を私の家から2日間掘り続けてきました。」



そのようなイベントの組織化は、1つの大企業であっても不可能な場合があります。 1つの共同アクションで複数のプレイヤーを組み合わせる必要があります。 たとえば、製造会社+ポータル（情報スポンサー）+ソーシャルネットワーク+写真サービス。 または：地元のテレビ局+ビデオサービス+ポッドキャストなど、応募者の意見を聞きます。 リストとオプションは無限です。 そのような各募集はスタートアップです。 同時に、それぞれが独自の問題を解決し、それぞれ独自の目標を達成し、一緒になってクライアントに対応します。 ばらばらの商品とサービスを彼に提供する。 誰もが勝っていることがわかりますか？ そのようです。 顧客とトレーダーの両方。 動作中のビンツビン。



私の意見では、主なことはそれを台無しにすることではありません。「Miss City」の仮想コンテストでパウダーをプレイし、曲がったホスティングおよび非稼働サービスを使用することによってそのようなキャンペーンを行わないことです。 美しくやれ。 サイトにジープ広告があります。 私はそれを買いたいので、彼女はそこにいません。 それはジープがクールだからではありません。 彼女はめちゃくちゃ美しくて元気なので、私は彼女を置いた。



この状況で敗者はいますか？ もちろん。 これらは、通路にリーフレットを配布し、高層ビルの壁全体にポスターを配布することにより、商品やサービスの宣伝を停止することを決めた会社の競合他社です。 彼らはこれに多額のお金を払いました。 時間が変化していることを考慮しないでください。 Web 2.0が登場しました。 昨日
