Parallels Desktop for Mac 13：macOS High Sierraに対応





1年前、macOSでのWindows仮想化用ユーティリティであるParallels Desktopの12番目のバージョンがリリースされました。 速い。 シンプル。 便利です。 しかし、私たちは怠idleではありませんでした。 今年、私たちはユーザーとオフラインで会い、ソフトウェアを購入してテストしたユーザーの間で追加のアンケートを実施し、AppleとMicrosoftからの有望な新製品を研究し、ユーザーの問題を掘り下げ、 テーマフォーラムも運営しました 。 これらすべてにより、リクエストにさらに注意を払い、重要な要望を考慮することが可能になりました。 そして今日、Parallels Desktopの新バージョン13の機能のいくつかについてお話ししたいと思います 。 開発者だけでなく、一般ユーザーにも多くの好奇心があります。 カットの下で、最も興味深い革新が説明されています。 それらの約50があります。 お楽しみください！ さて、コメントであなたの印象を共有してください。



ワンクリックWindows 10インストール



以前は、Parallels Desktopのインストール時に問題が発生しました。Windowsを入手して製品の動作を試す場所です。 ユーザーがWindowsを備えたDVDを持っていたとしても、Macにはどこにも貼り付けられませんでした。







新しいバージョンでは、すべてがはるかにシンプルになりました。ParallelsDesktop 13をインストールすると、MacウィンドウでWindows 10をダウンロードしてインストールするように自動的に提供されます。 ユーザーが製品を試用した後、ライセンスキーWindowを入力できるようになります。







タッチパネル



新しいMacBook Proでは、多数のファンクションキーの代わりに、タッチバーが使用されます。これは、基本的に長くて狭いタッチスクリーンです。 Parallels Desktop 13では、タッチバーをカスタマイズできます。 Windowsアプリケーションの起動、特定のフォルダーのオープン、特定の機能の管理などを含みます。







デフォルトでは、Parallels Desktop 13では、頻繁に要求される一部のアプリケーションで既製のキーストロークを使用できます。







Parallels Desktop 13での作業中にWindowsアプリケーションが実行されていない場合、Windowsタスクバーがタッチバーに表示されます（これは、ゲストデスクトップとタスクバーが非表示のCoherenceモードで作業する場合に便利です）。







たとえば、Wordを起動すると、メインの書式設定ボタンがパネルに表示されます。











デフォルトのキーが適切でない場合は、必要なキーを選択できます。ParallelsDesktop 13には、このためのタッチパネルセットアップウィザードがあります。







最も必要な人は常に手元にあります



Parallels Desktop 13は、 People Barを統合しています。 このツールは、Windows 10の9月の新しい更新プログラムの公式リリースの準備をしているだけですが、ユーザーは既に利用できます。 どのように機能しますか？ 私たちのほとんどは、最終的に友人、親relative、同僚、ビジネスパートナーの特定のサークルを開発します。 多くの場合、彼らとチャットするために、メールクライアントとインスタントメッセンジャー（Skypeなど）を常に開いて、目的の連絡先を見つけて手紙やメッセージを書く必要があります。 毎回連絡先リストとメニューを登らないようにするために、特定の人のすべての連絡先でピープルバーの名前パネルを構成できます。 Windows Peopleバーには3つまでしか連絡先を登録できないという事実にもかかわらず、Parallels Desktop 13を搭載したMac Dockには同様の数の制限はありません。











ピクチャーインピクチャー



この機能は、現代のテレビの所有者によく知られています。 面白い番組を見ていると仮定しますが、ここで商業休憩が始まります。 誰も広告を見たくないので、一時的に別のチャンネルに切り替えます。 また、広告終了の瞬間を見逃さないように、前のチャンネルの小さな画像を画面の隅に表示して、視聴した番組が再び開始された場合に目の隅から一目でわかるようにすることができます。 Parallels Desktop 13はまったく同じ機能を実装しています。仮想マシンの寿命を示す小さなウィンドウを表示できます。







一部のタスクの完了の程度を確認して、仮想マシンを切り替えないようにするために、デスクトップに「ピクチャーインピクチャー」の形でそれらを表示します。 ウィンドウのサイズ、透明度、および「常に上」の位置は変更できます。 また、どのVMが現在どのOSまたはアプリケーションを実行しているかを覚えておく必要はなく、目的のウィンドウをひと目でクリックするという観点からも便利です。 そして、これらのピクチャーインピクチャーのコンテンツをより美しく見せるために、GPUでピクセルシェーダーを使用した後処理が使用されます。 Picture-in-Pictureモードの最も便利な機能は、読み取りだけでなく完全に機能することです。 サイズを大きくし、仮想マシンで何かをすばやく実行してから、制御のためにバックのサイズを小さくすることができます。 本当にとても快適です。 それ自体でチェックされます。



性能



Parallels Desktop 13では、WindowsおよびMacでのファイル操作はバージョン12より20〜50％高速に実行できます。 また、USBデバイスを操作するメカニズムを改善しました。ほとんどのシナリオでの速度の増加は40％でした。 デフォルトのメモリ量の仮想マシンでは、スナップショットの取得速度が16％速くなり、ボリュームが100％増加したマシンで実行されます。



最大32個の仮想プロセッサと最大128 GBの仮想RAM（iMac ProのProバージョン）を各仮想マシンに割り当てることができるようになりました。 これは発表されたiMac Proを見越して行われます。iMacProは18コアのプロセッサを搭載し、ハイパースレッディングの助けを借りて36の仮想プロセッサになります。



Intel VTune Amplifierのサポートも改善され、Linux用のハードウェア監視カウンター（perfおよびMozilla rr）に基づくLBRおよびツールのサポートが導入されました。









ハイシエラの準備ができて



9月には、macOS-High Sierraの新しいバージョンのリリースが予定されています。 事前にベータ版にアクセスし、すべてのイノベーションをこのOSの新機能に適合させました。 したがって、新しいOSでは、すべての機能が問題なく動作します。



スケーリングスムージング



Parallels Desktop 12のコピーの46％は、Retinaディスプレイを搭載したデバイスで実行されます。 それらの37％では、少なくとも1つの仮想マシンでネイティブ解像度が有効になっています（スケーリングモード）。 これは、すべてのWindowsアプリケーションがhi DPIに適合しているわけではないという事実によるものです。 アプリケーションはテキストを正しく描画せず、コントロールは互いにぶつかり、メニューは非常に小さく見えます。 Parallels Desktop 13でこの問題に対処するために、画像を後処理することでGPU上の画像の品質を改善しています。 スケーリングモードでは、適切な画像品質を実現できます。







許可の変更



Parallels Desktopの新しいバージョンでは、仮想マシンのウィンドウの解像度を動的に変更するコードが大幅に再設計されました。 現在、再起動時、全画面モードからウィンドウモードへの切り替え時、ウィンドウの手動サイズ変更時、これはスムーズに行われ、ブラックタップのけいれんや点滅はありません。



そして、解像度について話しているので、別の新機能は、主に、さまざまな解像度の画面でプロジェクトがどのように見えるかを定期的にチェックするWebデザイナーとWeb開発者にとって興味深いものになります。 Parallels Desktop 13では、特定の仮想マシンの解像度をすばやく選択するオプションが導入されました。







明るい側に行く



あなたはどちらのインターフェイスが好きですか-暗いですか明るいですか？ Parallels Desktopユーザーにインタビューしましたが、回答者の80％以上がライトオプションを好んでいました。 そのため、ユーティリティの13番目のバージョンでは、インターフェイスを明るい色で作成しました。 しかし、「クラシック」のファンには、スイッチを使用して暗いインターフェイスに切り替える機会がありました。







OpenGLを使用する



私たちのチームはグラフィックスに取り組みました。 現在、OpenGL 3.2に積極的に取り組んでいます。 人気のあるプログラムのサポートを拡大しようとしています。 たとえば、Parallels Desktop 13を使用すると、最も人気のあるDIALux evo設計および照明計画アプリケーションの1つでの作業が可能になります。 このプログラムにはOpen GL 3.2が必要であり、サポートされています。







また、Parallels Desktop 13を人気のあるNorthgardコンピューターゲームにしました。 バージョン12以降、ゲームコミュニティと積極的に協力しています。 私たちにとっては、これをさらに続けるつもりです。 バーチャルマシンで特定のお気に入りのゲームを起動したい場合は、テーマフォーラムのいずれかで遠慮なく投稿してください 。 私たちは来て、意見を検討し、助けます！







みんなのためのツールを備えたギフトボックス



Parallels Desktopには、あらゆる場面で役立つ一連のユーティリティ、Parallels Toolboxが付属しています。 バージョン12で登場しましたが、今回はいくつかの新しいユーティリティ（合計約30）を追加しました。



•解像度を変更する

•オーディオをダウンロードする

•クリアディスク

•重複を見つける

•アニメーションGIFを作成する



たとえば、「GIFを作成」を使用すると、数回クリックするだけで、任意のビデオファイルの断片から目的の解像度のアニメーションを作成できます。 あなたがSMMに従事している場合、または猫とのGIFのファンである場合、それは非常に便利です。 また、「重複の検索」では、コンピューター上のファイルや未使用のアプリケーションの重複を検索できます。 したがって、ディスクの空き容量が不足している場合は、このユーティリティを実行するだけで、すべての重複ファイルのリストが作成されます。 それに加えて、「ディスクのクリーニング」を実行して、不要なファイルを削除できます。







Parallels Toolboxは、Windowsユーザーも利用できます。これは、当社のWebサイトからダウンロードできます。 また、Parallels Desktop 13のすべての所有者は、無料で利用できます。







新しいヘルプ



製品ヘルプを大幅に改訂し、状況に応じたものにしました。 これで、各設定ウィンドウで「？」ボタンをクリックして、ページの特定の設定に関するヘルプを表示できます。



新しいトライアル



試用版は、仮想マシンが起動した瞬間からカウントダウンを開始します。 ユーザーは製品のインストール、設定の確認、仮想マシンの作成、PCのインポート、またはBootcamp VMの構成を行うことができますが、試用期間は仮想マシンのスタートボタンを押してから14日間までカウントダウンします。
