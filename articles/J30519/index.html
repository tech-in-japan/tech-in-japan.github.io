Firefly-サイトのページでのコミュニケーション





Firefly-チャットの興味深いバリエーション。サイトの任意のページでアクティブにできます。 通信は、 マウスとキーボードを使用して行われます。



チャットモードがアクティブになると、チャットの他のすべてのユーザーは、ページ上のマウスの現在の位置を見ることができます。 あなたが書いたメッセージは、それがポインタであった場所に表示されます。

プライベート（ほぼ）会話の場合は、サイトの隅に移動して対応できます:-)。



これはすべて、サイトのページのフラッシュに2番目のレイヤーを重ねることで機能します。



このチャットには：
  • 画像を変更する機能（アバター）
  • Twitterとの統合（そこにもメッセージを送信できます）
  • タイムラインを使用してメッセージ履歴を表示する
  • チャットの人数を表示します




このチャットは、ページの最後に短いコードを挿入することにより、サイトで有効にできます。

< script language = "javascript" src = " static.onfirefly.com/ff.js " > </ script >


サイトにとってのメリットという点では、このチャットは普通の楽しみですが、「サイトのレイヤーでのサービス」という概念の観点では、単に素晴らしいです。
