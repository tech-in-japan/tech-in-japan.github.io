ノキアは私の連絡先を教えてくれません

前にも言ったように、ノキアは二度と戻ってきません。 しかし、Nokiaから別のデバイスへの変更でさえ、巨大なhemoです。 電話、Ovi Suiteアドレス帳、Ovi Contactsから半日連絡先を取得しようとしています。 今のところ、200以上のvcfカードを含むフォルダーを取得することしかできませんでした。



彼は考えも疑問もしませんでした...



そして、それは本当です、私は、どの会社も自分の連絡先でやりたいことをすることを私に禁止することを許可するとは思いませんでした。 はい、正式にはこれは禁止ではなく、単に「そのような機能を提供しない」だけですが、実際には禁止に似ています。 ここで、彼らは、画面上のコンタクティです。 必要に応じて、コピーと貼り付けに座って、必要に応じて、ペンを取り、画面からモールスキンに慎重に書き換えてください...



ただし、 Ovi Suiteでは連絡先のエクスポートは許可されていません 。 私たちは通常、フォーラムに登ります。 私たちはもはや驚いていません-フォーラムはうめき声でいっぱいです、誰も人々を助けません、人々はお互いを助けざるを得ません。 これは私が理解していることです-コミュニティと協力してください！ 国内の専門家のランクがいっぱいになります。



彼は期待していなかった...



一般的に、最も一般的なアドバイスは、連絡先を* .nbuにバックアップしてから、バックアップを切断し、バックアップから情報を抽出するサードパーティソフトウェアを使用することです。たとえば、次の無料ソリューションを使用します。http ： //sourceforge.net/projects/ nbuexplorer / 。



だから私は自由に* .vcfsでいっぱいのフォルダーを手に入れました。 ありがとう、少なくとも何か。



さて、さらに掘り下げます。 MS Outlookは連絡先をエクスポートでき、新しいバージョンが追加されるたびに改善されているようです。 Ovi Suiteに移動し、設定でOutlookとの同期をオンにし、[連絡先を同期...]をクリックします。そして、さらに多くの問題が発生します。 また、常に異なる、非テリングエラーコードが表示されます。 Ovi SuiteはOutlook 2010での操作方法などを知らないが、ポイントはこの星への道は通れない棘によって妨げられたということだ。



しかし、主なものは異なります-私はまだ携帯電話自体を持っていてもいいです-私は少なくともバックアップを作成し、フォルダにカードを捨てることができます。 そして、チューブがない場合は？ Nokiaを紛失/ dr死させた場合、バックアップや同期についてまったく話をすることはできません。 実際には2つの方法があります。新しいNokiaのストアに行く（そして、これが起こるかどうかわからないので連絡先がそこに引っ張られることを祈る）か、永遠に別れを告げる新しい方法で連絡先を集めるFacebookに接続していない人（はい、はい、Oryolの祖母とも）。



そのような終わり...



ovi.comでのdavyebanulistamchtoliアカウント（この記録！設立の歴史を思い出すと、揺れ始めます）もまったく役に立たなかったことを書く価値はありますか？自分の手段でもGoogleメールでも連絡先を取得できませんでした。 残っている唯一のオプションは、引き出した* vcf-kiを一度に1つずつGoogleにインポートすることだと思います（ハロウィン、グーグル、バッチインポートを行う!!!!）。



一般に、私はノキアの人々に対する態度を理解していません。デバイスとサービスの品質をユーザーに維持できなかった場合、送信ユーザーのホイールにスティックを挿入することはすでに卑劣です。 フィンランド人に別れを告げ、二度と会うことはないと確信しています。



PSサイレントモードを有効にするために（Googleを同期中に電話がきしむことがないように）、SIMカードを挿入するように求められます。 さて、理解...



PPSは、SIMカードに連絡先をコピーするというアイデアを投げかけました。 ただし、Nokiaは「姓」フィールドに名前と姓の両方をコピーします。これは文字数に制限があります。 その結果、「Alexander Nikolin」は「Alexander N」という名前の名前のない人になります。



PPPSコメントと提案された解決策の数を見てください。 電話機が紛失した場合、提案されたソリューションはどれも機能しません。 いずれのソリューションも、私の電話モデル（Nokia N86 8MP）で動作しません。 そして彼らは私に「これは意地悪ではない」とコメントで言います。 これは何？



shadowaloneからのPPPSソリューション：Googleでのバッチvcfインポート：

チェックしただけで、すべてがシンプルです

連絡先に入り、すべて選択（すべてマーク）

次に、バックアップの作成->電話からメモリカードへ

その他/連絡先フォルダがメモリカードに表示されます

多数のvcfファイルが表示される場所。 これらをコンプにマージします。

次に、これらすべてのファイルを1つにマージします

cd / dir_of_contacts /

cat * .vcf> ../ ALL.vcf

結果のファイルをGoogleアカウントにインポートします

Windows用

cd / dir_of_contacts /

copy * .vcf> ../ ALL.vcf
