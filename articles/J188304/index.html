モバイル革命の準備はできていますか？

太陽と砂の都市オデッサは、8月の前半にモバイルのメッカに変わります。8月8〜10日に、モバイル開発者とスタートアップ向けの一連のイベントが開催されます。



8月8日-モバイルデベロッパーデー



モバイル開発者の日は、モバイルプラットフォームの開発者の日であり、ウクライナ、ロシア、ベラルーシで年に1回開催されます。 これは、ビジネスがモバイルアプリケーションの開発、開発技術に関するプログラマー向けの特別セミナーに関連している人々のための会議形式のビジネスパートです。



詳細： http : //ua.mdday.ru/



8月9日-トレーニング「Windows Phoneアプリケーションの開発とプロモーションに深く浸る」



トレーニングは、DevRain Solutionsの専門家であるAlexander Krakovetsky（Microsoft Windows Phoneの最も価値のあるプロフェッショナル）とMikhail Galushko（Microsoft ASP.NET/IISの最も価値のあるプロフェッショナル）によって実施されます。 このトレーニングは、Windows Phone向けモバイルアプリケーションの開発、プロモーション、収益化に特化したもので、さまざまなシステム、サービス、アプローチの実際の経験とともに、独自の情報が含まれます。



詳細： http : //appclub.im/revolution/



8月10日-AppClub {build、monetize}革命







AppClub {build、monetize} Revolutionは、海の近くのProvectus ITの居心地の良いオフィスで開催され、約100人のモバイル開発者、専門家、スタートアップの代表者、スタートアップインキュベーターを集めます。 会議では、レポートに加えて、講演者や専門家によるマスタークラス、およびハッカソン参加者からのスタートアップピッチが発表されます。



AppClub {build、monetize}は、モバイルの専門家やスタートアップ向けのクラブです。 クラブは2012年10月に活動を開始し、それ以来7回の会議が開催され、そのうち6回はキエフで、1回はオデッサで行われました。 さまざまなモバイルシステム向けのソフトウェア製品の設計、開発、テストに多くの注意が払われていますが、主な重点はプロモーションと収益化の方法にあります。 AppClubミーティングには、業界の有名な専門家と、製品について話し合い、開発の最初の段階ですでにフィードバックを受け取ることができる若いスタートアップが参加します。 会議では、WP7PUBLISH、Tutorle、mockup.io、WithMyFriends.org、Backendlessなどのスタートアップが既に実施されています。


あなたがスタートアップで、AppClub {build、monetize} Revolutionでそれについて話したい場合-プロジェクトの説明とスピーカー/ファウンダーに関する情報をinfo@devrain.comに送って、事前にプログラムに参加できるようにしてください。



伝統的に、会議の参加者や講演者との非公式なコミュニケーションもあり、必要に応じて、オデッサのナイトクラブの1つでコミュニケーションを継続します。



詳細： http : //revolution.appclub.im



8月8〜10日-ハッカフォン



Hackaphoneは、モバイルアプリ開発者向けのハッカソンです。 お気に入りのプラットフォームでアイデアからモバイルアプリケーションのプロトタイプを作成するフルイマージョンモードで40時間。 メンタリングセッション、マスタークラス、専門家審査員、そしてもちろん、アフターパーティー。



プログラム：



  • 8月8日-アイデアのプレゼンテーション、チームの装備、準決勝進出者の選択。
  • 8月9〜10日-プロジェクトの作業。
  • 8月10日-最終、ハッカソンとアフターパーティの勝者の選択（AppClub {build、monetize} Revolutionで）。


詳細： http : //ua.mdday.ru/hackaphone/
