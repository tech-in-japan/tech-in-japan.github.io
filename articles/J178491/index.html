2013年5月の今後のITイベントのダイジェスト

Portal Aichi-event.rf （ itsobytie ）は、2013年5月の「Digest of Upcoming IT Events」の第4号を発表します。





ナビゲーションを簡単にするために、すべてのイベントは6つのカテゴリに分類されます。

  • I.開発
  • II。 ジュニア
  • III。 立ち上げ
  • IV。 マーケティング
  • V.管理者
  • VI。 雑多




I.開発



カンファレンス： Flash GAMM Moscow 2013
/モスクワ/ 2013年5月14日/費用70〜100ドル

フラッシュゲーム、ソーシャルゲーム、モバイルゲームの開発者と発行者の会議。 ゲーム業界で最も生産性の高いイベントで、カンファレンスの講演者には、Adobe、Unity、Alternativa Platform、ZeptoLab、Game Insight、FDG Entertainment、Player.IO、Spil Games、PlayScape、ToonBox、Marmaladeなどが含まれます。 ゲーム開発の経験、プロモーションの秘密、テクノロジー、出版、成功したプロジェクトの事後分析などなど！



カンファレンス： Game Developers Conference 2013（CWI）
/モスクワ/ 2013年5月15日/ 8,000ルーブルの費用。

「開発者から開発者へ」公式のロシア語フォーラムおよびコンピューター、コンソール、オンラインゲーム、ソーシャルネットワーク用ゲーム、スマートフォンやタブレット用ゲーム、アーケードマシン用ゲーム、携帯電話、その他のさまざまなプラットフォームの出版社。



講義： SQL Saturday
/キエフ/ 05.25.2013 /参加は無料です

SQLスペシャリスト専用のウクライナ最大の会議に参加する機会をお見逃しなく。 世界の永遠の都市の1つであるキエフで初めて、首都での楽しい散歩とSQL Serverの世界での便利な散歩を組み合わせる機会があります。



カンファレンス： DevCon 2013
/モスクワ/ 2013/05/29 / 16 888ルーブルの費用。

この会議は、ソフトウェア開発の分野の専門家、テストの専門家、アーキテクト、開発チームのリーダー、およびIT業界の専門家を対象としています。 2日以内に技術セッションに参加し、実際に技術を学ぶことができます-ブートキャンプ*、ワークショップ、テーマ別ラウンドテーブルをご覧ください。



展示会： Mobile Apps＆Technology Expo（MATE '2013）
/モスクワ/ 2013年5月30日/ 300ルーブルのコスト。

既存のすべてのモバイルデバイスおよびアクティブな開発段階にあるモバイルデバイス向けのアプリケーション開発および実装業界に特化した教育ビジネスイベント：スマートフォンおよびコミュニケーター、ハンドヘルドおよびタブレットPC、GPSナビゲーターおよびGLONASS、車、スマートテレビ、Google Project用アプリケーションガラスなど





II。 ジュニア



フォーラム： ITを見つける
/モスクワ/ 2013年5月16日/参加は無料

わずか5時間で、国内の最高の企業で仕事を見つけ、雇用のすべてのトリックを学び、専門家と個人的にコミュニケーションを取り、ITの分野で最も興味深いイノベーションを見つけ、さらには楽しみを持ち、賞品を獲得することさえできます。



ミーティング： サンクトペテルブルクでの第11回ITトーク
/サンクトペテルブルク/ 05.24.2013 /参加は無料です

あなたが専門的または管理職としてのキャリアを築いている場合、同僚が必要なことを行わない、または必要なことをしたくないという状況に必ず遭遇します。 質問の代わりに、「これを行う方法」と「うまくやる方法」、「なぜ？」、「誰がそれを必要としますか？」、「なぜ私が？」という質問が頭の中に集まっています。 弁証法からアジャイルマニフェストまで、多くの理論と概念が役立ちます。 いくつかの普遍的な原則と1つの興味深い理論についてお話したいと思います。



III。 立ち上げ



ハッカソン： OdHac：国際ハスケルハッカソン
/オデッサ/ 05/05/2013 / 600ルーブルの費用。

他の「Haskellist」と知り合い、プロジェクトの新しい参加者を見つけ、既存の参加者を改善し、新しいライブラリとツールを作成する絶好の機会です。 ハッカソン期間中、参加者は2つの主要なプロジェクトであるFayとHakyllに取り組みます。



ハッカソン： ハックデイ＃26
/ Almaty / 05/10/2013 /参加は無料です

競合他社は24時間でプログラム、アプリケーション、サイト、サービス、映画を作成します。 今回のイベントでは、若い開発者、デザイナー、意欲的なディレクター、メディアプロフェッショナルなど、カザフスタンのすべての地域から1000人以上の参加者を集めます。



競争： ITスタートアップEURASIA III
/エカテリンブルグ/ 05.17.2013 /参加は無料です

ITプロジェクトのアイデアとプロトタイプを使用したチームの国際競争。 2つのプロジェクトがRed Button Capitalから投資提案を受け取り、4つのプロジェクトがETV（インターネットテレビ）のMentors Clubプログラムに参加します。 5つのプロジェクトが、今年7月に開催されるInnoprom 2013展示会でのプレゼンテーションに選ばれます。



ハッカソン： AngelHackモスクワ
/モスクワ/ 2013/05/18 / 600ルーブルの費用。

AngelHackの一部として、開発者は、革新的なアイデアを学び、伝え、開発し、賞品を獲得し、資金を受け取ります。 モスクワで優勝したチームは、加速プログラムAngelHack Accelerator Programに参加し、「コードのラインからスタートアップまで」を支援します。



ハッカソン： ハックデイ＃27
/サンクトペテルブルク/ 05.24.2013 / 500ルーブルの費用。

HackDayはハッカソンであり、48時間でアイデアを実用的なプロトタイプに変え、同僚、投資家、メンター、市場の専門家の幅広い聴衆にそれを示すことができるイベントです。 HackDayは、会議でもパーティーでもスタートアップパーティーでもありません。 これは、実際のプロジェクトの作業の時間と場所です。





IV。 マーケティング



会議： 第2回国際モバイル会議＃MBLT13
/モスクワ/ 2013年5月15日/費用7,000-11,000ルーブル。

今年の会議の主なトピックは、さまざまなビジネス分野でのモバイル技術の使用です。 #MBLTの目標は、日々の生活を変えるだけでなく、潜在的な顧客の行動や思考に影響を与えるモバイルテクノロジーを最新のものにすることです。



会議： マーケティングに関するさらに別の会議
/モスクワ/ 2013年5月16日/参加は無料

あなたは、デジタルマーケティングの分野でロシアと外国の主要な専門家の報告を待っています。 実験的な定量的研究の方法、マーケティングおよび戦略的計画における数学的モデル、マーケティングの問題を解決するための異種データの大規模な配列の使用例などが検討されます。



会議： インターネットライフ2013
/モスクワ/ 2013年5月17日/参加は無料

Internet Life 2013カンファレンスでは、会社の従業員として、最新の技術に関する知識を得ることができます。 また、会議では、企業の生産性を高め、コストを最適化するのに役立つ最新のテクノロジーについて詳しく説明します。



カンファレンス： SPIK 2013
/サンクトペテルブルク/ 05.20.2013 / 4,000ルーブルの費用。

会議の参加者は、Runetでのビジネスのすべての側面、ソーシャルメディアの開発、情報セキュリティの問題、国内のインターネットメディアの構造の最近の変化、国内ドメインゾーンの法的規制、スタートアップの作成とプロモーションなどについて話し合います。
