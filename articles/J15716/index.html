MS Projectの次のバージョンははるかに簡単になります

シアトルでの会議で、MicrosoftはMS Projectの将来のバージョンの機能をいくつか示しました。 報道によると、聴衆は新しいタイムラインのスクリーンショットを拍手で迎え、コピー＆ペーストを使用して他のプログラムに転送できます。 たとえば、PowerPointで貼り付けた後でも、フォントサイズなどの要素を編集できます。



これは、このプロジェクト管理システムをより簡単にし、その範囲を拡大するために、MicrosoftがMS Projectに実装する機能の1つにすぎません。 スティーブ・バルマーは、このテーマについて賢明に述べました。概して、各オフィスワーカーは少なくとも自分の時間を管理しているため、マネージャーと呼ぶことができます。 つまり、MS Projectはあらゆるレベルのユーザーがアクセスできる必要があります。



プロジェクトのユーザーインターフェイスの主な革新は、画面上部にボタンがあるすべてのMS Officeプログラムの標準パネルの外観です。 フィールドに入力する手順が簡略化されます。既存の方法の代わりに、空の列がユーザーの前に表示され、任意の情報を入力でき、そのタイプが自動的に認識されます。



Web Accessの機能を大幅に改善します。これは将来、プログラムの重要な機能の1つになるはずです。



MS Projectの新しいバージョンでは、リリースされる年がわからないため、この名前はまだ発明されていません。 おそらくこれはMS Project 2009または2010になります。
