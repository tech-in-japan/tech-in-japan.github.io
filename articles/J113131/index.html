2月18日、サンクトペテルブルク-2011年Java Tech Day

そのため、このようにコンピューターに座って会議の写真を確認すると、突然、次のSUN（Oracle / Java）Tech Days会議が4月または3月のどこかにあるはずだと気づきます。 昨年の経験から判断すると、1月に登録することができました。 私は運を試し、いくつかの情報を探すことにしました。 一般的には、ここにあります（サイトからコピー）：



「2011年2月18日に、Java開発者JavaTechDay 2011を専門とする会議がサンクトペテルブルクで開催されます。この会議は、サンクトペテルブルクでのSunTechDays会議の伝統を引き継ぐOracle Corporationの後援の下で開催されます。 JavaTechDay 2011は、サンクトペテルブルクのMoskovsky Prospekt 97aにあるHoliday Inn Moscow Gate Hotelの会議ホールで開催されます。 この会議には、米国、英国、サンクトペテルブルクの主要なOracleエンジニア、および他社のJava開発者が参加します。



イベントに参加することで、JavaSE、JVM、JavaEE、JavaME、組み込みJava、JavaFXテクノロジーの開発、Javaの生産性の向上、一般的な問題の解決、そしてもちろんワークショップで同僚とチャットするオラクルの計画について学ぶことができます。



会議への参加は無料で、事前の登録が必要です。

JavaTechDay 2011でお会いしましょう！」



それだけではありません！ :)

  • 第一に、おいしくて新鮮なプログラムによると、私はあえて推測するように、JDK 7、したがってJava SE 7の発表を期待しています。
  • 第二に、仮想化に興味のある人は、1か月前に新しいVirtual Box 4について語られます。
  • JavaTVまたはOracle Java Embedded Clientの実装において、流行のスマートテレビがなければ実現しません。
  • そしてもちろん、Oracleが伝統を取り入れていることに気付かないことは不可能です。3日-2009年、2日-2010年、2011年、会議は1日という性質を獲得しました。
