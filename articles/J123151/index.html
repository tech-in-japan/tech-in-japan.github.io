Linux 11.6を計算する

今日、素晴らしいロシアのCalculate Linuxディストリビューションの[lorMode]バイナリジェンタ[/ lorMode]のジュビリー（プロジェクトの4年）リリースがリリースされました。 トピックリンクを作成する価値はありますが、残念ながらできません。

だから、主な利点：


-OpenRCとNetworkManagerの間でネットワークマネージャーを選択するためのサポートが追加されました。

-コンソールインストーラーで、ネットワーク構成のサポートが追加されました。

-Udevはinitramfsに統合され、デバイスのサポートが向上し、livecdのダウンロードが高速化されました。

-デフォルトでは、Grub 2ブートローダーが使用されます。

-ディスクにインストールされているオペレーティングシステムの認識を追加しました。

-スプラッシュでの画面解像度の自動検出を追加しました。

-ビデオドライバを選択するためのメニューを追加しました。 さらに、nouveauおよびati-driverによってサポートされます。

-デフォルトでは、KMSサポート付きの無料のビデオドライバーが使用されます。

-Nvidiaはプレインストールされたドライバーではなくなり、ユーザーが選択するとインストールされます。

-XZ圧縮形式のlivecdイメージを使用します。

-CD / DVDおよびUSBフラッシュへの書き込みをサポートするハイブリッドイメージへの移行。

-Calculate Scratch Server用のバイナリパッケージリポジトリが作成されました。

-pastebin.calculate-linux.ru Webサービスとの統合を追加しました。

-Calculate Directory Serverで、メール通信履歴の保存のサポートが追加されました。

-Calculate Directory Serverには、ファイアウォールを構成するためのShorewallパッケージが含まれています。

-ポルトガル語とルーマニア語のサポートを追加しました。
