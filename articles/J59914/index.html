悪と腐敗のためのGeoTAGS

意味



エマージェンスとは、システム全体の新しいプロパティが発生したときであり、それを構成する要素のプロパティの単純な合計に減らすことはできません。 私たちの厳しい生活の中で、社会組織に適用されるこの概念は、私たち全員の間の柔軟なつながりの欠如によって否定されます。 情報のスレッドは、お互いの経験と知識をより効果的に活用できます。 ジオタグサービスのGeoTAGS（geotags.amoneron.com）が考案されたので、これを社会全体の組織に適用するとよいでしょう。 誰かが何らかの生活状況に直面した場合、彼はこれを短いメモに反映し、それを自分の街の特定の場所に添付することができます。 地理共有サービスとの違いは、ラベルが状況を特徴付けるものであり（資​​料ではなく）、その内容は他の人が決定を下すのに役立ちます。 主な焦点の1つは汚職です：誰かがこの状況に遭遇した（または、むしろ彼が遭遇したとき:)必要な詳細を残した場合）サービスのマーク-最初に、他の人のために状況を明確にし、次に、人々と状況が知名度を得る、これは多くの場合、私たちに残る唯一のソーシャルツールです。





テクニック



実装-Google MAPs API +タグデータベース。 このサービスは、公開性と自己規制の原則に基づいているため、ログインはなく、誰でもタグを作成および評価できます。 任意の場所と任意のスケールで、最高評価の最初の50マークのみが常に表示されます。 低い評価のタグは削除され、データベースから消去されます。



論理的継続



モバイルプラットフォーム用のアプリケーションがリリースされ、ユーザーの位置に対応するラベルが表示されます。
