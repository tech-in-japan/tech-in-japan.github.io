35種類の便利な仮想化ツール





今日、「仮想化」という言葉は誰もがよく知っており、仮想化テクノロジーは組織のITインフラストラクチャの主要コンポーネントの1つになりつつあります。 ちなみに、本日、 1cloudの例を使用して仮想インフラストラクチャプロバイダーを作成した経験を共有した投稿を公開しました。



このトピックの作業を継続し、仮想インフラストラクチャに関係するすべての人に注意を払うべきツールの選択をまとめることにしました。 ランダムな順序で配置された既存のツールのごく一部のみを以下に示します。



仮想化管理


  • 移行：仮想環境に移行するためのツール
  • OpenFiler：ネットワークファイルストレージのWebベース管理の作成
  • StarWind Virtual SAN：ハイパーバイザー指向の仮想化ストレージ
  • vCenter Converter： WindowsおよびLinuxの物理マシンを仮想マシンに変換できます
  • RVツール： VI SDKを使用して、仮想マシンとESXホストに関する情報を表示するアプリケーション
  • Citrix Essentials：エンタープライズ仮想化のためのXenServerおよびHyper-Vホストサーバーの管理


  • Veeam Backup and Replication： vSphereおよびHyper-Vプラットフォームから仮想化されたアプリケーションとデータを復元する
  • vSphere Mobile Watchlist：スマートフォンを使用して仮想インフラストラクチャをリモートで監視できます
  • ESX Deployment Appliance： ESXサーバーの展開を簡素化
  • Ganeti： Googleの仮想化クラスター管理システム
  • oVirt：シンプルなWebインターフェースを備えた仮想化プラットフォーム
  • Packer：イメージアセンブリツール


  • Xen： x86アーキテクチャプロセッサの準仮想化サポートを備えた仮想マシンモニターまたはハイパーバイザー
  • CloudStack：仮想インフラストラクチャの展開、構成、およびメンテナンスを自動化します
  • Vagrant：仮想開発環境を作成および構成するためのコンソールユーティリティ
  • VirtualBox：軽量で信頼性の高い仮想化ツール
  • FusionSphere： Huawei Virtualizationプラットフォーム


Kvm


  • KVM： x86プロセッサー向けのLinux仮想化ソリューション
  • Mist.io：ハイブリッドインフラストラクチャの管理を支援する完全なソリューション
  • nbsvm：単純な連鎖コマンドを使用してVMを作成、操作、および対話する
  • Witsbits：物理サーバーへの仮想マシンの展開を簡素化するサービス
  • UVMM：直感的なXenおよびKVM仮想マシン管理
  • libvirt：仮想化を管理するためのクロスプラットフォームライブラリ


  • cloonix：仮想ネットワーク設定を構成する
  • Archipel： XMPPを使用したプッシュ通知
  • キムチ： HTML5 KVMの管理ツール
  • SolusVM：商用で最も人気のあるコントロールパネルの1つ
  • Virtual Machine Manager： libvirtを使用して仮想マシンを管理するためのインターフェースを提供します
  • vmmaestro： KVMベースの仮想マシンを起動および停止できる小さなシェルスクリプト
  • kvm-wrapper： KVMベースの仮想マシンの管理を簡素化するシェルスクリプトのセット


仮想環境のセキュリティ


  • VMware vShield Edge：包括的なデータセンター境界保護のための製品
  • VMwareコンプライアンスチェッカーツール：セキュリティコンプライアンスチェック（FISMA、PCI / DSSなど）
  • Catbird Secure：物理および仮想インフラストラクチャのセキュリティソリューション
  • HyTrust：仮想インフラストラクチャのユーザー接続を保護するツール
  • トレンドマイクロ：物理サーバー、仮想サーバー、クラウドサーバーを保護するためのプラットフォーム
