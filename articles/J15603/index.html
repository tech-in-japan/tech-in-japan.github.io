Web Studios 2008の評価：新しい基準、主催者の変更



こんにちは、Habr。 キャッチフレーズの評価「TOP-100 Runet Web Studio 2008」がどのように構築されるかについて、ほんの少しお話したいと思います。



2008年、Taglineグループは、専門家の推定だけでなく、企業の売上高だけでなく評価を行います。 次のような指標を導入する予定です。



専門家の市場シェア/タグラインによる実際

プロフェッショナル/非プロフェッショナル環境での会社の人気のレベル

会社の最高の仕事の質

顧客満足度

ニッチおよび価格ポジショニング

さらに、TaglineはITの分野で多くの有名なスポンサーを引き付け、プロジェクト専用のオフラインイベントを開催する予定です。 市場参加者の専門的なコミュニティを形成するために、多くのステップも実行されます（おそらく、この方向に既に進んでいるグループと提携しています）。 評価と市場調査は2008年2月に開始されます。



私も小さな発表をしたいです：



評価は、他のタグラインプロジェクトと同様に発展します。 しかし、私なしで。 マーケティングおよびPRのディレクターとしてADV / web-engineeringに戻るよう招待されました。 招待が会社の10周年の前夜と一致したことは特に楽しいです。 共同の努力により、市場を真に「爆破」できることを願っています。 ADVは市場のペースに先んじて勢いを増しているので、この機会をすべて持っています。





もちろん、TEGLINEのメンバーに、プロジェクトの評価を可能な限り効果的かつ客観的にする方法についてアドバイスします。 しかし、私が市場のプレーヤーの側に行く限り、格付けは少し奇妙です。



ちなみに、市場形成やさまざまな種類の研究のアイデアは無駄ではなかったと言っておく価値があります-最近、この方向でも機能している多くのプロジェクトがアクティブになりましたスタジオとプロジェクトの特定の評価も表示されます。
