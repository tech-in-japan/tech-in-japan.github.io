Firefox 30リリース

大きな遅れをとって、デスクトップ用Firefoxがダウンロード可能になったことをお知らせします。 主なイノベーション：ホワイトリストにあるものを除くすべてのプラグインのデフォルトのブロック、Linux用GStreamer 1.0のサポート、Web開発者向けのツールの変更、バックグラウンドブレンドモードのサポート。 Firefox 30でのすべての変更のリストは猫の下にあります。





  • ソーシャルネットワーク上のブックマーク、閲覧履歴、共有リンクへの簡単なアクセスを提供するインターフェイスを備えたサイドバーを表示するボタンが追加されました。
  • プラグインに関連するコンテンツの表示に関する厳格な制限 。 Firefox 26で追加されたポストクリックプラグインのアクティベーション機能は、Javaおよび古いバージョンのNPAPIプラグインで動作しますが、特別なホワイトリストに追加されたものを除くすべてのプラグインのデフォルトのブロックに切り替えられます。
  • OS Xは、キーボードショートカットコマンド+ Eのサポートを追加して、選択したテキストで見つかった式を置き換えます。
  • Linux 用 GStreamer 1.0のサポートを追加 （以前は0.10をサポート）。
  • Web開発者向けのツールが拡張されました 。 英語の元記事とspmbtによる翻訳の 詳細 。
  • Web Audioを使用している場合、各ウィンドウで音量のミュート/ミュート解除および調整が可能です。
  • Webアプリケーションでは、関数としてWebIDLコンストラクターを呼び出すことは禁止されています。
  • ボックスの影の領域を含む、境界を超える要素の流出を修正しました 。
  • CSS要素のデフォルトのサポートはbackground-blend-modeです。
  • input - type = "reset | button | submit"でline-heightの使用が許可されるようになりました。
  • ECMAScript6 配列とジェネレーターの内包表記を実装。
  • エラースタックに列番号が含まれるようになりました。
  • キャンバスコンテキスト設定にアルファオプションのサポートが追加されました（ 詳細 ）。
  • パスワードマネージャーを使用してパスワードを保存するように提案する場合、 autocomplete = "off"は無視されるようになりました 。
  • TypedArraysが新しい名前付きプロパティをサポートしていなかった問題を修正しました 。
  • NTLMv1認証が無効です
