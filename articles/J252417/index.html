ブライアン・カーニガンによる本「The Go Programming Language」の発表

アマゾンで、彼らはブライアン・カーニガンとアラン・ドノヴァンによる、「The Go Programming Language」というタイトルの新しい本を発表しました 。 誰も知らない-Brian KernighanはUnixの共著者の1人であり、cronの作者であり、プログラミングの世界で最も象徴的な本の1つ、「 The C Programming Language 」の作者であり、1978年にデニスリッチーと書かれました。



この本は予約注文が可能で、紙版のリリースは2015年8月27日に予定されています。







上記の「Cプログラミング言語」という本は、しばしば技術文献のモデルと呼ばれ、C言語に関する完全で包括的な（多少時代遅れではあるが）資料です。言語のすべての側面をカバーし、読みやすく、興味深い例がたくさんあります。 この本の第2版（1988年）は20以上の言語に翻訳され、大学のCに関する参考書であり、C言語全般の人気と流通に大きな影響を与えました。



ブライアンは後にいくつかの本を出版しましたが、そのうちの2冊はRob PikeとThe Unix Programming EnvironmentおよびThe Practice of Programmingによって共同執筆され、Bell LabsでRobおよびKen Thompsonと協力したことを考えると、驚くことではありませんKerniganがGoのプロモーションに貢献することを決定したこと。 ブライアンの最新の本はD is for Digital（2011）とAMPL AMPL：A Modeling Language for Mathematical Programming（2003）でした。



したがって、このGoの本はCのカルトの本と同じくらい完全で概念的であると信じるあらゆる理由があります。さらに、大学やコンピューターサイエンスコースの教師にとって良い出発点になります-この分野には、権威のある高品質の教育文献がありますとても貴重です



本の2人目の著者は、Google Inc.の開発者であり、golang / toolsの主要な貢献者の1人であり、多くの静的Goコード分析ツール（go vet、oracleなど）の著者であるAlan Donovanです。



出版社はコンピューター文学の著者であり、The Mythical Man-MonthやThe C ++ Programming Languageなどの本も出版したAddison-Wesleyです。
