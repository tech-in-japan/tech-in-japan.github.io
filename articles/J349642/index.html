DEV Labs2018。Java開発者向けのオンライン会議。 3月3日

Javaイベントで新しいシーズンを開くことは、すでに伝統となっています。 これはこの言語に特化したシリーズの9回目の会議であり、今回は通常の形式をわずかに変更することにしました。 DEV Labs 2018プログラムでは、いわばその場でプロトタイプを作成することに集中しようとしました。 さまざまな角度から結果を作成、分析、最適化し、検討します。 Materielも注意せずに放置しないでください。



プログラム内：





Spring 5機能およびリアクティブフレームワーク。 プログラムはオレグ・ユシチェンコによって開かれます。 Spring 5の新機能を検討し、Samuraiという小さなデモプロジェクトを作成します。





Javaプロファイリング：Javaプログラムのパフォーマンスのボトルネックを検索します 。 ウラジミールソンキンとともに、プログラムを最適化します。 JVMフレームワークであるVisualVMのプロファイリングを検討し、Springでアスペクトを作成してパフォーマンスを測定することを検討してください。





光の速さでプロトタイピング：最新のテクノロジーでマイクロサービスを作成します。 パート1. Vyacheslav Lapinのプレゼンテーションは2つのパートで構成されます。 レポートの最初の部分では、新しい技術ライン全体をレビューし、この多様性がプロジェクトですぐに使用できるようになることを確認します。



光の速度でプロトタイピング。 パート2。レポートの2番目のパートでは、マイクロサービスを記述するための典型的なタスクの例として、その開発と製品への出力の完全なプロセスを示します。



会議は3月3日11:00モスクワ時間に始まります。
