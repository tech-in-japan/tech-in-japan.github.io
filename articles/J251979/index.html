Google Summer of Code 2015が始まります





Google Summer of Code 2015に参加しているオープンソース組織のリストが公開されました。 Google Summer of Code（略称GSoC ）はGoogleのイニシアチブプログラムで、学生が参加できるオープンソースプロジェクトを毎年選択します。 プロジェクト自体はすべての応募者から学生を選択し、Googleは学生の奨学金をその仕事に対して支払います（ 5,500ドルの金額）。 今年の完全なプログラムルールについては、 www.google-melange.com / gsoc / document / show / gsoc_program / google / gsoc2015 / help_pageをご覧ください。



GSoCは2005年に最初に開催され、それ以来毎年開催されています。今年は11回目の開催となります。 今年、 416の組織が参加申請書を提出し、その中からGoogleが137を選びました。 今回適用した組織の数は、GSoCの存在のすべての年で最高の組織の1つ（私が間違っていなければ、最大は417でした）、受け入れられた組織の数は最下位の1つ（2007年以来の最低の！）。 さらに、この金額はGoogleが約束した金額に反し、2014年にほぼ対応します。
5.このプログラムに参加する予定のメンター組織はいくつですか？



2005年に40の組織、2006年に100以上、2007年に130以上、2008年に175、2009年に150、2010年に150、2011年に175、2012年に180、2013年に177、2014年に190と協力しました。 2015年には2014年と同様の数の組織を受け入れます。
ただし、このプログラムのお金はGoogleから寄付されることを忘れないでください。ご存知のように、「彼らは才能のある馬を調べません」：Googleは必要と思われる限り寄付します（可能な場合）。



GSoCへの参加を希望する学生は、組織のリストに精通することができます。 今年の学生登録は3月16日に始まり、3月27日に終了します。 ただし、Googleでは、生徒が最後の瞬間まで待たずに、バグ修正やコードの改善により希望する組織を支援し、チャンスを大幅に増やすことを推奨しています。 残念ながら、クリミアでの紛争に対する米国の禁輸措置のため、今年クリミアからの学生はプログラムに参加できません。 今年の全プログラムスケジュールは、 www.google-melange.com / gsoc / events / google / gsoc2015でご覧いただけます。



GSoCについての詳細は、学生の観点から、学生の経験に基づいて作成されたHabréの記事で読むことができます。
