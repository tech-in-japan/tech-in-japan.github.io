Linux Raspberry Piコンピューターは1月に注文可能になります



Raspberry Piコンピューターのベータ版の新しい写真が公開されました
Raspberry Piプロジェクトの公式Webサイトで、超コンパクトなLinuxコンピューターがまもなく利用可能になります。 または、12月にサイトのオークションで購入します（これは最も短気な人のためです）注意-トラフィック！ 仕様はwikiページで入手できます 。 最初は、USBドライブのフォームファクターを維持できると想定されていましたが、最終的には、プロトタイプのサイズとデバイスの最初の商用バージョンはプラスチックカードのサイズに対応します。

デバイスの予想価格は非常に魅力的であり、多くの人々はデバイスが市場に登場することを期待しています。 Raspberry Piが潜在的な顧客の期待に応えるかどうかを見てみましょう。 もちろん、デバイスの対象読者は私たちが望むほど広くはありませんが、いずれにしても消費者を見つける必要があります。 Raspberry Piの可能なアプリケーションは、主に開発者の想像力によって制限されます。 このハイテク製品から、ユーザーがGuruplug / SheevaPlugのアナログを入手できることを期待できますが、この場合にのみ、製造業者は実装に失敗した熱除去メカニズムによるマイクロコンピューターの故障の問題を回避できます。
