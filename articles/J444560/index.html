グローバルハイパースケーラーがクラウドサービス市場を完全に獲得するのは本当ですか？ロシア市場でどのような運命が待っていますか？ オンラインストレージの企業データの最大のセキュリティを確保する方法は？ 未来のクラウドテクノロジーは何ですか？ 3月26日、クラウドテクノロジー市場の主要な専門家が専門のカンファレンス「クラウド」でこれについてすべて話します。 SAPデジタルリーダーシップセンターの「ファッショントレンド」。







Amazon Web Services、Kaspersky、Yandex。Cloud、SberCloud、Mail.Ru Cloud Solutions、SAP、その他の企業のトップエキスパートが集まり、すべての参加者と実際の経験を共有し、クラウド市場を変える主なトレンドについて話し合います。非常に近い将来。 3月26日、モスクワのSAPデジタルリーダーシップセンターで、オンラインブロードキャスト形式でお待ちしています。



会議は、ロシア市場の主要なプレーヤーがクラウドインフラストラクチャの開発における主な傾向としてハイパースケーラーについて議論するパネルディスカッションによって開かれます。 一緒に、IaaSとOnPremがマルチクラウド環境との対立でどれほど早く地盤を失うかを把握したいと思います。



完全なイベントプログラム



講演者の個別のスピーチで、2つの重要な領域を特定しました-クラウドでのサイバーセキュリティとクラウドプロジェクトの実装における実際の経験：



  • SAPのクライアント（Globus、Sherremetyevo、SUEK）は、SAP HANA Enterprise Cloud（HEC）での移行プロジェクトの準備と実装のプラクティスを共有します。
  • Kaspersky Labの仮想およびクラウドインフラストラクチャ保護の責任者であるMatvey Voitovは、クラウドに展開する際のセキュリティシステム構築の機能について説明します。


独自のクラウド製品の開発に加えて、SAPは従来のOnPremソリューションをハイパースケーラーに積極的に配置し、複雑なITプロジェクトでもインフラストラクチャの最大限の単純化と単純化を実現する方法を設定しています。



ロシアのITプロフェッショナルは全員3月26日に集まります！
