JQuery Mobile Alpha 4リリース

昨日、人気のあるjQuery Mobileフレームワークのモバイルバージョンの4番目のアルファバージョンがリリースされました。 開発者自身が書いているように 、これは最新のアルファリリースです。 新しいリリースでは、パフォーマンスの改善に主な重点が置かれ、A3からの150以上の問題も修正され、いくつかの新しい機能が追加されました。



プラットフォームのサポート


まず、jQuery Mobile A4 はWP7をサポートするようになりました 。 そして今では、ほぼすべての最新のモバイルOSで動作します。

  • Apple iOS（3.1-4.2）：iPhone、iPod Touch、iPadでテスト済み
  • Android（1.6-2.3）：HTC Incredible、Motorola Droid、Google G1、Nook Colorでテストされたすべてのデバイス
  • Blackberry 6：トーチとスタイルでテスト済み
  • Windows Phone 7：HTCサラウンドでテスト済み
  • Palm WebOS（1.4）：Pre、Pixiでテスト済み
  • Opera Mobile（10.1）：Android
  • Opera Mini（5.02）：iOS、Android
  • Firefox Mobile（ベータ）：Android




メタデータ


jqmData（）メソッドが追加されました。これは、名前空間をサポートする拡張jQuery.data（）メソッドです。



正規化されたイベントシステム


jQuery Mobileの新しいバージョンは、開発者がユーザーのプラットフォームに応じて異なるイベントをインターセプトする必要がないように設計された仮想マウスイベントを提供します（たとえば、AndroidとBlackberryはタッチイベントとマウスイベントの両方をトリガーします）。



PhoneGapの改善


A3に存在するバグへの対処が修正されました。 $ .mobile.allowCrossDomainPagesフラグも追加され、他のドメインからJQMページをロードできるようになりました。 同様のクエリを次のように実行できるようになりました。file：//dir1/dir2/myPhoneGapApp.html#https：//myHomeServer.com/dir1/dir2/foo



リストフォーム


開発者の要求により、フォームをリスト形式で作成する機能が追加されました。 そのためには、リストを<form>タグで囲み、フォームの各要素にラベル要素を追加するだけです。



カスタム文字列値


i18nをサポートするための最初のステップは、ブートダイアログ、エラー、検索フィールドのプレースホルダー、および自動的に追加された戻るボタンの文字列の値を変更する機能でした。



読み込み中


jQuery Mobile A4を取得するには2つの方法があります。

1. jQueryサーバーへのリンクを使用して、コードをサイトに埋め込みます。

JavaScript：

非圧縮： jquery-mobile-1.0a4.js （130KB）

縮小およびGzip圧縮： jquery-mobile-1.0a4.min.js （19KB）



CSS：

非圧縮： jquery-mobile-1.0a4.css （53KB）

縮小およびGzip圧縮： jquery-mobile-1.0a4.min.css （7KB）



コピー＆ペーストのコード：

<link rel="stylesheet" href="http://code.jquery.com/mobile/1.0a4/jquery.mobile-1.0a4.min.css" /><br/><script src="http://code.jquery.com/jquery-1.5.2.min.js"></script><br/><script src="http://code.jquery.com/mobile/1.0a4/jquery.mobile-1.0a4.min.js"></script>





2. 1つのzipアーカイブjquery-mobile-1.0a4.zip （JavaScript、CSS、および画像）をすべてダウンロードします
