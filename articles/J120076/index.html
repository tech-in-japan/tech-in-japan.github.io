1. TRIMとは何ですか？

データストレージデバイスは、必要なデータと不要なデータを区別せず、すべてのデータを平等に扱います。 これは、従来のHDDと干渉しませんが、SDDアーキテクチャでは、データを書き込む前に特定のフラッシュセルをクリアする必要があります。 セルは個別にクリーニングされるのではなく、ブロック（たとえば、各128KB）でクリーニングされるため、予備クリーニングでは記録効率が低下します。 時間の経過とともに、ディスクのパフォーマンスが大幅に低下する場合があります。



TRIMは、ATAデータセット管理コマンドの属性です。 特定のデータが不要になった（たとえば、削除された）ことをディスクコントローラーに伝えることができます。 SSDの場合、この属性により、コントローラーはフラッシュフラッシュセルを記録時間から解放するようにスケジュールできます。 また、これにより、より均一なセル摩耗のためにデータを絶えずシャッフルすることができます。







ディスクコントローラがTRIMをサポートしている場合、Win 7は、関連するすべてのディスク操作（削除、フォーマットなど）でこの属性を使用します。 TRIMが有効になっていることを確認するには、次のコマンドを使用します。

fsutil behavior query DisableDeleteNotify





DisableDeleteNotify = 0の場合、TRIMは有効です。 残念ながら、これは、ドライバーからのTRIMサポートが必要なため、機能することを意味しません。 SSDがRAIDに接続されていない場合（つまり、AHCIがBIOSにインストールされている場合）、TRIMをサポートする標準のMSAHCIドライバーを使用している可能性があります。 それ以外の場合、すべてRAIDコントローラーのファームウェアに依存します。



SSDとHDDの使用の違いは何ですか？

SSDの詳細は、記録中にフラッシュセルが摩耗することです。 フラッシュセルは電子トラップです。 多重記録では、障壁が薄くなり、電子がトラップから逃げます。 最終的に、論理的な0と1を区別するのに十分な数の電子を保証することはできません。 小さいもの。 プロセス、セルが生き残るより少ない数の書き換え。 したがって、フラッシュコントローラーはセルの状態を監視し、利用可能なボリューム全体に情報の記録を均等に分散しようとします。 これは、HDDとは異なり、SDDでは論理情報が物理セル間で絶えず分散されることを意味します。 フラッシュコントローラは、記録された情報の量も監視し、内部アルゴリズムに従って、たとえばSSDLifeユーティリティを使用してSMARTで表示できる予想寿命を計算します。



ディスクの摩耗を最小限に抑えるために、小さなレコードの数を減らすことをお勧めします。 Win 7自体がディスクのタイプを判別し、スーパーフェッチ、アプリケーション起動プリフェッチを自動的にオフにします。これはアプリケーションキャッシュにすぎません。 たとえば、SDDにあるOSとHDDにあるOSの違い。 これらのパラメーターを手動で変更しませんでした。 パラメータ値はここにあります 。







さらに、SSDの特性により、デフラグの必要はありません。データは常に物理的に断片化され、強制デフラグは摩耗を増加させるだけです。 Win7はSSDドライブ自体のデフラグを無効にします。 また、頻繁に記録されるため、一時ファイルでNTFS圧縮を有効にすることはお勧めしません。



スワップファイルをSSDに保存することは可能であり、望ましいことです。 ページファイルを操作するシナリオは、記録と比較して大量の読み取り（40対1）と比較的大量の記録が特徴であるため、データへのより高速なアクセスの利点は有害です。



HDDからSDDにクローンを作成した後、Win 7で何をする必要がありますか？

1. TRIMが機能していることを確認します。 フラッシュコントローラー、BIOSのAHCIモード、TRIMドライバー（またはデフォルトのMSAHCIまたはRAIDコントローラー）のサポートを使用して、TRIMのサポートを確認してください。

2. SSD、Superfetch、prefetchの最適化がオフになっていることを確認します。

3.理にかなっている場合は、他のヒントを試してください。 たとえば、インターネットでは、シャットダウン時にスワップファイルのクリーンアップを無効にすることをお勧めします。HKEY_LOCAL_MACHINE\ SYSTEM \ CurrentControlSet \ Control \ Memory Management \ ClearPageFileAtShutdown = 0



原則として、最も重要なWindows 7はそれ自体に対処し、ほとんどの場合、特別なことをする必要はありません。 合理的なアプローチは、OSとプログラムをSSDに、データ（音楽、ビデオ）を別のHDDに保持することです。 映画を再生するのにクレイジーな読書速度は必要ないので、通常のハードドライブに保存できます。
