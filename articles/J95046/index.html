Opera 10.60 Alpha 1：スピード、見た目、バグ修正

お気に入りのブラウザの新しいビルドと変更点の印象的なリストがリリースされました。



バージョン10.60のリリースを準備しており、すべてのプラットフォームでディストリビューションの開発速度を同期させることを目指しています。 このアルファ版は、パフォーマンスの向上、インターフェース（UI）の改​​善、そしてもちろんエラー処理のアイデアを提供します。





パフォーマンス ！



開発者は眠らないので、生産性が向上します。 JavaScript、DOM、およびCSSを使用しているときにこれに気付くでしょう。



UIの強化



視覚的な設定に加えて、スピードダイヤルの改善が既に確認されていますが、インターフェイスの作業も行っています。 便宜上、Oボタンのテキスト「メニュー」に注意してください。 また、すべての内部タブのサムネイル用に独自のアイコンが追加されました。





Windows 7では、タスクバーのサムネイルのプレビューがデフォルトで無効になりました。これは、多数の開いているタブでのナビゲーションの不便さのために行われました。 ただし、設定でこれを有効にすることができます（opera：config＃UserPrefs | UseWindows7TaskbarThumbnails）。



カーボンニュートラル



Opera 10.50フレームワークの主要な作業とCarbonをCocoaに置き換えた後、可能な限りCarbonコードを削除しようとします。 パフォーマンスを改善するために、いくつかのものが動的にロードされるようになりました。



また、Macユーザーの場合、JavaはMac OS 10.5および10.6の最新のアップデートがインストールされている場合にのみ機能することに注意してください。 Mac OS 10.4用のJavaはサポートされていません。



Unix



Unixユーザーの場合、これは明らかに10.5xバージョンがないことを意味します。 それにもかかわらず、私たちはすべてのプラットフォームでの同期と10.60のリリースに努めており、Unixを含めて同時（6月中旬）になります。
