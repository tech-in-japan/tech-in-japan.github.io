Cubieboard2 Allwinner A20（Cortex-A7デュアルコア）の新しいバージョン

Cubieboard開発ボード開発プロジェクトが成功した後、Cubieteamチームは開発を継続しました。 また、6月には、製品Cubieboard2の新しいバージョンを導入しました。 Cubieboard2デバッグボードは、修正されたCubieboardの最初のバージョンです。 違いは、SoC AllWinner A20 ARM Cortex-A7デュアルコアの新しいバージョンをインストールすることです。 AllWinner A20パッドと以前のAllWinner A10チップは対応しているため、プリント基板のトポロジをやり直す必要はありませんでした。









Subboardboard2仕様：

プロセッサー： SoC AllWinner A20ARM®Cortex™-A7 Dual-Core（デュアルコアプロセッサー）

GPUARM®Mali400MP2、OpenGL ES 2.0 / 1.1に準拠、HDMI 1080p @ 30fpsまたは720p @ 60fps

RAM： 1GB DDR3 @ 480MHz

ユーザーメモリ：4 GBの内部NANDメモリ、SDは最大32 GB、uSDスロット使用時は64 GBをサポート

SATA：サイズ2.5の2Tから2.5までのドライブのサポート

電源： 2Aで5V DC

ネットワーク ：10/100イーサネット、オプションのWi-Fi（USB RTL8188CUS / RTL8188EUを使用する場合）

USB： 2つのUSB 2.0 HOSTポート、および1つのUSB 2.0 OTGポート

オペレーティングシステム： Ubuntu Desktop 12.04、Android 4.2.2



ボードには、I2C、SPI、RGB / LVDS、CSI / TS、FM-IN、ADC、CVBS、VGA、SPDIF-OUT、R-TPなどを接続するための96ピンがあります。
