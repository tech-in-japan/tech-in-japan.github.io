象はすでにここにいます。 高速、信頼性、強力。 PostgreSQL 8.3！

200人を超える開発者、 300を超えるパッチ、 15か月間の開発者とテスターのハードワーク...そして今、世界最高のDBMSの最新バージョンが産業環境で使用できるようになりました！



2008年2月4日、PostgreSQLグローバル開発グループは、最も開発されたオープンDBMSのバージョン8.3の待望のリリースを発表しました。このリリースは、PostgreSQLのオープンソースシステムからの最も生産的なDBMSとしての地位をさらに強化します。 パフォーマンスに関するイノベーションの中で、特に注目に値するものは次のとおりです。



* HOT（ヒープのみのタプル）

* bgwriterプロセスパラメーターの自動調整メカニズム

*非同期コミットコミット（非同期コミット）

*「スミアコントロールポイント」（スプレッドチェックポイント）

*「同期スキャン」（同期スキャン）

*ディスクスペースの追加削減（ "Var-Varlena"）

* L2キャッシュ保護

*トランザクションカウンター（レイジーXID）の「ロールアップ速度」の速度の低下



PostgreSQL 8.3には、開発者とデータベース管理者が高く評価する多くの興味深く魅力的な新製品も含まれています。 これらの革新の一部を次に示します。



* CSVロギング

* SQL / XML（！）

* MS Visual C ++サポート

* ENUMタイプ

*組み込みのTsearch（！）

* SSPIおよびGSSAPI

*複合型の配列

* pg_standby
