Web 2.0とは何ですか？ 多くの人はまだこの現象が何であるかを明確に説明できません。



Web 2.0は、インターネットのブランド変更です。 システム（この場合はインターネット）がその自然な発達のために自分自身を殺さないようにするには、新しいステータスを取得する必要があります。 新しい名前-「Web 2.0」がこのステータスになりました。



インターネットはテキストのセットとしてではなく、各メンバーの多大な関与を伴う生きたダイナミックな環境として認識されるようになりました。 このために設計されたメカニズムを使用するプロジェクトは、Web 2.0スタイルのプロジェクトと安全に見なすことができます。 ところで、ポイントはAJAXだけではありません。 簡単に言えば、このテクノロジーを使用すると、トランザクション「ユーザー-システム」および「ユーザー-ユーザー」の時間コストを削減できます。



すべての「インターネットユーザー」のネットワーク環境にこのような関与をもたらすものは何ですか？ 通常のWebユーザーが世界を支配し、企業がそれに適応するでしょうか？ 「Webの新しいバージョン」の見通しは何ですか？



今では企業だけが世界を支配しているとは思えません。 世界は世界に支配されており、すべてが相互につながっています。 ユーザーの応答は企業の意思決定に影響を与え、企業の意思決定はユーザーの応答に影響を与えます。 実際、人々はすべての人類にとって正しい道を推測しています。

この方法は、いくつかの非物質的環境にすでに存在します。 現在、企業は特定の製品に関するユーザーの意見を調べるために多額のお金を払っています。 また、Web 2.0の登場により、フィードバックの取得とユーザー設定の特定に費やす労力が減少します。



見込み客についての質問に何を答えるべきかわかりません。 それは単なるコスト削減です。 たとえば、デジタルテレビの展望はどうですか？ ポイントAからポイントBに到達するためにポイントBが不要であることは、誰にとっても明らかです。それを削除すると、パスが短くなります-最適化！
