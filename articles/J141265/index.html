Goを書く人のために

ともだち！



「Googleはプログラミング言語Go 1の最終バージョンをリリースしました」という投稿の続きで、興味が論理的に続き、Habr政権から私たちのti病な要望が聞かれ、今日からGo言語には独自のブログ ハブがあります。



この機会に、「Learning Go」という本はお勧めできません 、その最新バージョンはいつでも著者（ありがとう、 Miek Gieben ）またはリポジトリ（.texソース）からダウンロード （PDF、〜1.5Mb）できます： 



git clone http://www.miek.nl/git/gobook.git


現時点では、これは最も包括的なGo言語のチュートリアルであり、言語の基本と機能について詳しく説明しています。 この本には約120ページ（A4）があり、コード例（回答あり）が含まれており、私の意見では、公式Webサイトのドキュメントを読むよりも、言語に慣れるためのより友好的な方法を提供します。



書いて、あなたの経験を共有し、恥ずかしがらないでください！ 行こう！
