Nokia Lumia 930：主力スマートフォンの詳細

こんにちは、Habr。



ご存知かもしれませんが、2014年6月30日に、イベント/ビルドの一環としてこの春に発表された主力スマートフォンNokia Lumia 930のロシアでの販売開始が予定されています。 クアッドコアプロセッサ、2 GBのRAM、5インチOLEDスクリーン、20メガピクセルのカメラを搭載したモバイルデバイスのコストは、 24,990ルーブル （公式に宣言されたコスト）です。







モデルがロシア市場に参入することを予想して、新しいスマートフォンの機能について話し、新しいフラッグシップスマートフォンや会社の他の新製品の設計を担当したJonne Harjuとのインタビューからの抜粋を共有することにしました。



「鉄」



前述したように、フラッグシップスマートフォンのコアは、クロック周波数2.2 GHzのクアッドコアQualcomm Snapdragon 800プロセッサーになりました。 競争上の優位性は、Adreno 330グラフィックコア、2ギガバイトのLPDDR3 RAM、および内部32 GBドライブによっても示されます。 スマートフォンでのメモリカードのサポートは提供されていませんが、他のすべてのLumiaと同様に、OneDriveストレージには7ギガバイトの空き容量があります。







スマートフォンは、LTEネットワーク、Wi-Fi WLAN IEEE 802.11 a / b / g / n / ac（慎重）、Bluetooth 4.0およびNFCなど、すべての最新の通信規格をサポートしています。



一般に、技術仕様は大画面のLumia 1520スマートフォン（メモリカードと、実際には画面を除く）と同一であるという事実にもかかわらず、イデオロギー的にノキアLumia 930はモデルLumia 920/925のラインの続きです。







スマートフォンには5インチのOLEDディスプレイが搭載されています。 解像度は1920 x 1080ピクセルで、ピクセル密度は441ピクセル/インチです。 従来、画面にはClearBlackテクノロジーが採用されており、感度が非常に高いため、手袋をしていてもスマートフォンを制御できます。 また、画面が第3世代のGorilla Glassを覆っていることも注目に値します。



最高のカメラ付き携帯電話のステータスは、41メガピクセルのカメラを搭載したNokia Lumia 1020によって依然として保持されていますが、新しいフラッグシップは特性の点で遅れをとっていません。 PureViewテクノロジーを搭載した20メガピクセルのカメラは、大型の1 / 2.5インチセンサーと2つのLED付きフラッシュを備えています。 他の多くのLumiaモデルと同様に、光学部品はZEISSの友人によって作られています。







リッチレコーディングテクノロジーには特に注意を払う必要があります。 ユーザーがより良く、よりリアルな音を録音できるようにするため、ノキアは4つの高性能全方向性マイクをNokia Lumia 930にインストールしました。 前面に2つ、背面に2つ。



デジタル信号処理と要素の思慮深い配置を使用して、4つの無指向性マイクを組み合わせて2つの指向性マイクとして機能させることができます。



指向性ステレオ録音では、カメラの前の音がキャプチャされ、さまざまな方向からのノイズは無視されます。 これにより、より明確になります。



Nokia Rich Recordingを使用してどのようなサウンドを録音できるかを知るには、同じ技術を使用するオペレーターNokia Lumia Icon（米国市場向けのNokia Lumia 930のバリアント）で記録されたビデオを見ることができます。







エネルギー消費のトピックに触れると、ノキアLumia 930には2420 mAhのバッテリーがあることに注意してください。 その充電は、ワイヤレス充電標準Qiを含めて補充できます。 完全に充電されたデバイスは、最大16時間の通話時間と最大9時間のビデオ視聴が可能です。



Windows 8.1およびエンタープライズサービス



Nokia Lumia 930は、ロシア市場で2番目のスマートフォン（Nokia Lumia 630に続く）で、Windows Phone 8.1の更新バージョンで実行されます。







このアップデートのテーマは既にHabrahabrで繰り返し議論されているため、主なイノベーションを簡単にリストします。



-音声アシスタントCortana（これまでのところ、米国市場ではベータ版ですが、他の市場では時間がかかります）。

-通知センター+画面上部のドロップダウンメニューから設定にすばやくアクセス。

-改善されたインターフェイス（開始画面で2〜3行のアイコンを選択する機能、「タイル」の下にパーソナライズされた背景を設定する機能、カスタマイズされたロック画面をインストールする機能など）。

-「スワイプジェスチャ」およびより高度なインテリジェンスをサポートするワードフローキーボード（読み取り-単語のコンテキスト予測）。

-更新されたアプリケーションストアとユニバーサルアプリケーションのサポート。

-通話とアプリケーション用の別のボリューム（シック！）;

-ハブ「People」でのソーシャルネットワークTwitterおよびFacebookの機能の統合の改善。

-WebGLをサポートし、HTLM5のサポートを改善したİnternetExplorer 11。

-バッテリー電力を最も多く消費するプログラムを識別するためのバッテリー電力検出。

さらに、近いうちに既に発表されているファイルマネージャーの登場（！）







Windows Phone 8.1のNokia Lumia 930には、ビジネスユーザーにも喜ばれるものがあります。 組み込みのMicrosoft Office Mobileスイート（Excel、Word、PowerPoint、OneNote）とSkypeに加えて、Lyncコーポレートメッセンジャーなどの多くの特殊なアプリケーションを無料でダウンロードできます。



モバイルデバイス管理（MDM）の高度な機能もあります。 これらは、すべての従業員に対して一度にVPN、Exchange、およびSharePointを構成する機能に適用されます（これはすべてとはほど遠い）。 その結果、スタッフは会社のサービスにさらに簡単にアクセスできるようになります。







Lumia 930スマートフォンには、自動的に有効化されるVPNおよびS / MIMEサポート、Wi-Fiネットワークの自動検出、および企業ユーザー向けの特定のアプリケーションをブラックリストまたはホワイトリストに追加する機能も備わっています。



あとがきとして



Nokia Lumia 930の技術仕様については、Nokiaのチーフデザイナーと話をする機会があったため、デバイスの外観については特に言及しませんでした。



Yonne Harjuは、ノキアのモバイルデバイスの設計で10年の経験があります。 彼は、Nokia Designの設計チームの1人の頭としてスタートし、現在はLumiaブランドのすべてのデバイスの設計を担当しています。 彼のポートフォリオの明るいページの中には、Symbianの一連のNシリーズデバイス、MeeGoのNokia N9、そしてもちろんすべてのLumiaスマートフォンがあり、800番目のモデルから始まります。







Lumia 930を既にテストした多くのジャーナリストは、このデバイスは使いやすさと魅力的な外観のバランスが取れていると言います...



Lumiaデバイスの設計に関しては、不必要な詳細なしに、できる限り簡潔で最小限に抑えることを目指しています。



Lumia 930と「オペレーター」Nokia Lumia Iconの場合、デバイスをシンプルにすると同時に、非常に魅力的なものにするためにあらゆることを行いました。 だからこそ、私たちは簡潔なアルミニウムフレームと完璧にマッチするスクリーンと明るいバックパネルの組み合わせに落ち着きました。



背面カバーの平らな枕の形は、Lumia 1020のように突出したカメラモジュールがないためです。このソリューションにより、有名なNokia Lumia 920や大画面Lumia 1520を搭載したフラッグシップスマートフォンのように、ワイヤレス充電器のサポートを返すことができました。設計と先進技術の調和を実現します。







これは、Lumiaラインのデザインの開発における新しいラウンドと呼ばれますか？



間違いなく、Lumia 930は他のNokiaデバイス、特にLumia 925と多くの共通点を持っていますが、同時にそのユニークなキャラクターによって区別されます。 このスマートフォンは、審美的な観点からだけでなく、生産プロセスの観点からも、シンプルさの概念にさらに深く入り込んでいます。 モノリシックアルミニウムケースは、生産に必要な時間が短くなります。 Lumia 930のすべてが可能な限り簡素化されています。







ノキアでは、ほぼ10年間設計を行ってきました。 デザインに何らかの系統性があり、そのピークがルミア線に落ちたと思いますか？



はい、それはかなり長い道のりでした！ 2005年に電話を作成した方法は、現在行っている方法とはまったく異なります。 当時、すべては1年に一定数のデザインオプションを作成し、新しい画像を発明することを中心に展開されていました。 しかし、その後、技術的なコンポーネントとユーザーエクスペリエンスから離れるようになり、それが報われました。各デバイスに最適なコンポーネントの組み合わせを選択することに、ますます注意を払い始めました。 新しいデバイスを作成し、私たちは常に経験から学び、自分自身を改善しようとします。







そして、あなたにとって本当に最高のデザインは何ですか？



Nokiaといえば、最初のN9デバイスが本当に好きでした。携帯電話のデザインをどのように見始めたかという点で会社が変わったからです。 一般的に言えば、シトロエンがDSモデルを立ち上げたときの大胆な振る舞いは驚くべきことです。 当時、この車は信じられないほど未来的でしたが、今日、50年経った今でも壮大です。







スマートフォンの未来をどう見ていますか？



しばらくしてから、スマートフォンがどのような目標を達成すべきか、そしてその本当のコンセプトは何かをよりよく理解します。 その理想的なサイズは何ですか？ 価格はいくらですか？ ユーザーは何を見たいですか？ 彼らは通知によって常に気を散らされたいですか？ 人々はオンラインを維持し、お互いにつながりたいという衝動を克服しますか？ 気分がよくないときにスマートフォンで認識して、この時点でメールをオフにする必要がありますか？ 現在、私たちの業界では、そのような質問への答えを見つけることを追求する膨大な数の研究が行われています。

私たちは「実験世代」であり、その行動が将来必要とされるものの意見を形成します。
