Raspberry Pi用のタングステン言語とMathematicaパッケージが無料で利用可能

新しいWolfram言語（ Wolfram Language ）が実際にテストできることがわかったため、すでにハブに書き込まれた Stephen Wolframからの大規模プロジェクトの発表からわずか3日が経過しました：Mathematica数学パッケージとともにRaspberry Piプラットフォームで利用可能です今日-そして絶対に無料。 Wolfram言語のニュースレターは、 これに関するニュースを含むニュースレターを受け取りました。 以下はその翻訳です（この記事はRaspberry Piの開発者のために書かれました）。









Raspberry PiでMathematicaで3Dグラフを描く



Raspberry Piで作業することの素晴らしい点の1つは、コンピューティングサイエンスの必要な変化について意見を共有する人々のグループと出会う機会でした。 そのような例の1つは、学校での数学教育を機械計算から問題解決に転換しようとするコンピューター数学（ コンピューターベースの数学、CBM ）の方向です。 このプロジェクトのサイトからの抜粋：



労働市場、社会および思考全体に対する数学の重要性は、過去数十年で劇的に増加しました。 一方、数学教育は世界中で危機にinしています。国、産業、教育、そして学生自身が必要としているものからますます逸脱しています。



コンピューターがこのギャップの主な理由です。すべての計算をそれらにシフトすることによってのみ、数学は多くの分野で複雑な問題を解決するために適用できます。 実用的な数学はコンピューター計算の影響を受けました。 今や大衆の間で数学を教える分野でこのような根本的な変化の時が来ました。



computerbasedmath.orgは、この再考を達成することを目的としたプロジェクトです。 すべてのレベルで数学教育を再定義することを目標に、コンピューター計算に基づいて数学のまったく新しいコースを作成しています。手動で計算する歴史的に確立された手法から離れ、現実世界の問題を解決する必要がある状況に移行し、より高い数学の開発の方向を設定します。


本日、ニューヨークで開催されたコンピューター数学（CBM）の会議で、Raspbianシステムイメージの一部としてMathematicaとWolframの無料コピーを提供するためのWolfram Researchとのコラボレーションを発表しました。 これにより、Raspberry Piはあらゆる年齢のCBMの子供たちにとって優れた学習プラットフォームになると信じています。 Konrad Wolframが今日述べたように：「プログラマーはすぐにMathematicaパッケージの全能力を使用することができ、能力の範囲を広げるだけでなく、数学の力と重要性を実証することができます。」









ヘビサイドステップ関数の非常に不適切な使用



後続のRaspbianシステムイメージには、デフォルトでWolframとMathematicaが含まれます。 SDカードに少なくとも600 MBの空き容量がある現在のユーザーは、次のコマンドを使用してインストールできます。 



sudo apt-get update && sudo apt-get install wolfram-engine


Mathematicaパッケージは、教育メニューのアプリケーションランチャーにあります。



WolframとMathematicaをRaspberry Piプラットフォームでうまく動作させるために行った驚くべき努力に対してWolfram Researchチームに感謝したいと思います。 今後数か月にわたって、Wolframでできる興味深いことの研究に専念する一連の記事を公開する予定です。 楽しいでしょう！
