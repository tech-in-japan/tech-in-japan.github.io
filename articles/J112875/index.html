10,000ルーブルのタスク

受賞者-10000ルーブル



変数ベースの計算には多くの6桁の数値が記録されています。 この計算の各桁には最大値があります（基数-1）。 この計算システムの数値は、10進数で記述されます。左から右へ、最上位から最下位の桁までです。 数字はスペースで区切られます。 このシステムの最大排出量は「4 8 15 16 23 42」です

最小数

「0 0 0 0 0 0」は、10進法の数値「0」に対応します。

「0 0 0 0 0 42」は、10進法の数値「42」に対応します。

「0 0 0 0 1 0」は、10進法などの数値「43」に対応します。

入力ファイルには、このシステムに記録された番号のリストが含まれています。 この計算システムの入力ファイルの各番号を、それに続く同じ計算システムの番号で置き換えることにより、出力ファイルをコンパイルする必要があります。

例：

入力ファイル「1 3 12 22 0 11」。 出力では、「1 3 12 22 0 12」。

入力ファイル「2 0 0 16 23 42」。 出力では、「2 0 1 0 0 0」。
