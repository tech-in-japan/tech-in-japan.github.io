マイクロソフトは、2008年2月27日に製品の新しいバージョンを発表します

Microsoftは、待望のWindows Server 2008（別名Longhorn）を含む自社製品の次のバージョンが、ロサンゼルスでのプレゼンテーションで2008年2月に正式に発表されると述べた 。



ワールドワイドパートナー会議でのスピーチの中で、マイクロソフトの最高執行責任者ケビンターナーは、ソフトウェアの巨人が2008年2月27日にロサンゼルスでWindows Server 2008、Visual Studio 2008、SQL Server 2008を公式に発表すると述べました。



以前に考えられていたように、Windows Server 2008が今年末までに登場すると予想した顧客にとって、製品のリリース日は魅力的ではないように思われるかもしれません。 Microsoftは、公式には製品が来年まで表示されないものの、スケジュールを変更しないと報告しています。



Turnerは、3つの製品をMicrosoftの起業家ポートフォリオの「ビッグドッグ」と呼んでいます。 「これは、2008年の1日で最大のマイクロソフト製品の発売です」と彼は言いました。
