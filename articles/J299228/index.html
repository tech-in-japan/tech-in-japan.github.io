メディアはEPAM Systemsの歴史の中で最大の買収を認識しました

EPAM Systemsは 、ベラルーシのルーツを持つアメリカのオフショアIT企業で、米国とインドのソフトウェア開発者であるAlliance Global Services（ AGS ）を5130万ドルで買収しました。 取引は2015年末に完了しましたが、金額はまだ明らかにされていません。



取引は現金で全額支払われます。 500万ドルを含むこの金額は、特別なエスクローエージェントの口座に振り込まれ、特定の義務の履行の保証として15か月間彼と一緒になります。 さらに、EPAMは60万ドルの追加の繰延支払いを行う可能性がある、とCNewsは報告しています。



Alliance Global Servicesは、品質管理、設計、ソフトウェア開発に取り組んでいます。 AGSは主に、医学、バイオテクノロジー、保険、メディアおよび情報ビジネス、テクノロジーセクターの分野の顧客に焦点を当てています。



AGSは、米国とインドのオフィスにいる1.15千人のIT専門家を雇用しています。 したがって、EPAMスペシャリストのスタッフは大幅に増加します。



買収された会社の主な構造は、アライアンスコンサルティンググローバルホールディングスです。 彼女は、米国企業のAlliance Global Services、IncとAlliance Global Services、LLCおよびIndian Alliance Global Services IT Indiaを所有しています。



EPAMは以前、AGSを取得すると、北米とインドの顧客とより緊密に連携し、インドのハイデラバードとプネの2つの新しい開発センターでグローバル開発センターのネットワークを拡大するのに役立つと述べました。 さらに、この契約により、EPAMはソフトウェア開発プロセスの自動化の分野で強化されます。



買収の決定は、EPAM Systems Arkady Dobkin の創設者、社長、取締役会の議長によってコメントされました 。

お客様が現在直面している課題はますます複雑でグローバルになりつつあり、当社のソリューションはこれらの新たな課題に対応しなければなりません。 Epamの場合、これは、お客様がどこにいても、どの地域およびどのタイムゾーンでも、サービスとソリューションの適切な品質を確保する必要があることを意味します。 そのため、Alliance Global ServicesとEpamの間の合意は戦略的に重要です。 私たちは同じ業界で働いており、同様の製造原則を共有しているため、お客様はアジア太平洋地域でサービスを拡大することで利益を得られます。 そして、複雑なカスタムソフトウェア製品とプラットフォームの開発市場での地位を強化する機会を得ます。



EPAMビジネスはいくつかの領域に分かれています。



•ITコンサルティング

•ソフトウェア開発

•アプリケーション統合

•移植とアプリケーションの移行

•ソフトウェアテスト

•EPAMシステムに基づいた専用の開発センターの作成

•デジタル戦略の開発



この買収に関連して、EPAM Systemsの経営陣は、収益の予測を上方修正し、2015年には少なくとも905百万ドル、前年の730百万ドルになると予想しました。



AGSの買収は、EPAM史上最大の買収です。 以前、最大の取引は、2014年6月にアメリカの会社GGA Software Servicesを買収したことでした。 彼女は、医療企業向けのアプリケーション、データベース、数学モデリング、ITサポートの開発に従事しています。 繰延負債を含むこの取引の価値は、4550万ドルでした。



2014年4月、EPAMは香港に本社を置くJoint Technology Development（ Jointech ）と、中国およびシンガポールの子会社を買収しました。これらの子会社は、投資銀行および資産管理の分野で働く国際企業にとって「戦略的に重要な技術」を開発しています。 現金と株式の繰延債務を考慮した取引総額は3680万ドルに上ったと同社は述べた。
