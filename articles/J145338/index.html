Ubuntuのトレントストリーム

Linux / UbuntuでのTorrent Streamソフトウェアのオープンテストが開始されました！





Ubuntuユーザーが利用できるようになった、トレント経由でビデオ/オーディオコンテンツをストリーミング/オンラインで再生するための世界最高のソフトウェアアプリケーション



Torrent Stream Systemおよびこのプロジェクトのフレームワーク内でリリースされたソフトウェアの詳細情報は、 torrentstream.org Webサイトで見つけることができます。

Ubuntu用のTorrent Streamソフトウェアのテストビルドには次のものが含まれます。



TSエンジン -トレントストリームテクノロジー（マルチメディアストリーミングエンジン）のすべての機能と機能を提供するメインソフトウェア製品



TSエンジンは、Torrent Streamプラットフォームのコアコアであり、API（ TS Engine API ）を介して、開発者が独自の製品を作成し、Torrent Streamの機能と機能を統合するために使用できます。



TSプレーヤー -マルチメディアプレーヤー（VLCプレーヤーの修正および改良版）。コンテンツのダウンロードを待たずに、トレントファイルを介してオンラインビデオおよびオーディオを再生できます。

コメント：通常のビデオおよびオーディオファイルのように、異なるトレントクライアントを使用せずに、ダウンロードを待たずにデスクトッププレーヤーでトレントを開始および再生できます。また、ユーザーがコンテンツIDを知っている場合、ソースとして使用して、対応するソースを指定する際のフィールド（コンテンツID-これは本質的にマグネットリンクの類似物であり、TSシステム内のみ）。



アセンブリにデスクトップバージョンのプレーヤーを含めるのは、ほとんどのTSプレーヤー設定が同期されているため、ブラウザーベースのWebプレーヤー（TS P2Pマルチメディアプラグベース）の高度な設定を使用する必要があるという事実によるものです。 たとえば、ビデオアクセラレータを使用するか、Webプレーヤーのホットキーを再割り当てする場合、ユーザーはTSプレーヤー設定に移動し、これらの設定を行い、プレーヤー（できればWebブラウザー）でページを再起動する必要があります。 Webプレーヤーで動作します。



TS P2Pマルチメディアプラグイン -ブラウザープラグイン（VLCプラグインに基づいて構築）

これにより、Torrent Streamテクノロジーを使用するすべてのサイト、およびYou Tubeなどの人気のあるビデオホスティングサイトで、世界で最も機能的なWebプレーヤーを使用できるようになり、bittorrentプロトコルで再生されるすべてのコンテンツが完全にサポートされます。



コメント：おそらくビデオの視聴専用にFlashをインストールして使用した人にとっては、これは非常に優れた代替手段になります。さらに、BitTorrentはアクセラレータとしても機能し、ストリームの安定性とユーザーのインターネットチャネルの最大ダウンロード速度を保証します

ベンチマークテスト



前文：トレントストリームに関する多くの出版物は、しばしば「世界で最高」という単語やフレーズが含まれる論文を発音します。 「アナログがない」など これらの論文がソフトウェアのこの部分に適用されるため、これらの論文を確認または反論するために、完全なファイルのダウンロードを待たずにビデオを再生する機能を備えた、他のトレントクライアントと比較テストを行うことを提案します。



テストの場合：テストトレントファイルを使用する（ トレントをダウンロードする ）か、http-sidingの高品質なサポートを使用して独自のトレントを使用し、プリバッファリング時間インジケーターを比較し（ビデオの再生開始後）、巻き戻し時の応答時間を比較しますそれが他のどこかに実装されている場合。 （トレントストリームソフトウェアが実際に目指しているトレントを使用する場合、高品質のAVoDサービス（Audio and Video on Demand）を作成および受信するには、HTTPシードが必要です）



注意！ http-sidの戻り速度またはすべてのhttp-sideの合計速度がビットレートよりも遅い場合、ソフトウェアは通常モードで動作し、そのようなサポートとの大きな違いは感じません。 この制限は、インターネット接続の速度がビットレートよりも低い場合にも適用されます。したがって、高品質のhttpサポートが提供されるインターネットチャネルおよびトレントで可能なコンテンツのみをテストすることを選択してください。



また 、「事前バッファリング中のエンターテイメントクリップの再生を有効にする」オプションを無効にすることを忘れないでください！ （このオプションを有効にすると、映画の予告編がメインビデオの前に再生されるため、再生の開始速度のテストを実行できません）



基本的なテストに加えて、P2P YouTube拡張機能を使用することをお勧めします。これにより、他のトレントクライアントに対するトレントストリームの機能と利点を視覚的に確認できるだけでなく、同じ比較テストを行うことができます。 （トレントを介したオンラインビデオの再生がFlashを介したオンラインビデオの再生に劣らないようにするとともに、ビットトレントプロトコルのすべての利点を追加する機会があります）
