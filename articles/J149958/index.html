AppCode 1.6：新機能

AppCodeチームは、Objective-C- AppCode 1.6の統合開発環境の新しいバージョンをご紹介します。



AppCodeは、大規模プロジェクトでより高速に動作し、Xcode 4.4およびXcode 4.5プレビューと完全に互換性があり、CocoaPodsプロジェクト、サードパーティプロジェクト、ファイルテンプレート、およびすべての新しいObjective-C関数をサポートします。



リリースの機能の一部を次に示します。



  • インプレースコードリファクタリング：名前の変更とリファクタリングの導入は、不要なダイアログボックスなしでエディターで直接機能し、コメントおよび非コードファイルの置換可能な要素も考慮するようになりました。






  • 一流のローカリゼーションサポート ：Find Usagesコマンド、さまざまなリファクタリングメソッド、およびその他のインテリジェント機能がローカライズファイルで利用可能になりました。


  • ビルド設定エディター ：お気に入りの開発環境を離れることなくプロジェクトを構成します。






  • C ++ 11およびlibc ++のサポート 。


  • 特に、必要なすべての列挙値がswitchステートメントに存在することを確認できるようにする、 新しいコード検査のセット全体 。 プロジェクト内のすべてのローカライズされていない文字列を検索します などなど。






  • 生成が改善され 、新しいメソッドを作成する場所を選択できるようになりました-実装、インターフェース、カテゴリ、およびobjectWithメソッドの生成が簡素化されました。


  • AppCodeにはGitHubのプラグインがバンドルされています。 これにより、GitHubを使用してプロジェクトを簡単に複製できます。 このビデオからGitHubとの統合について詳しく知ることができます。




新機能の詳細については、製品のWebサイトをご覧ください 。



戦闘でAppCodeを試してみたい人は、 30日間のトライアルを利用できます。 AppCodeの更新を受信する有効なサブスクリプションにより、新しいバージョンを無料でダウンロードして使用できます。



オープンソースプロジェクトと教育機関は、引き続き無料ライセンスを申請できます。
