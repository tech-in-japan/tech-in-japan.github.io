LiveVideoArc仮想マシンでのビデオ録画のアーカイブの処理

同じ形式（たとえば、DVDディスク）の同一のビデオ録画が多数あり、それらを簡単に変換および圧縮したい場合は、ここにあります。 これを支援する私のプロジェクトについてお話したいと思います。



LiveVideoArc-これはミートグラインダーで、その入力はさまざまなファイル（ftp経由でダウンロード）を受け取り、出力を処理した後、圧縮されたxvid mpeg4ファイルをダウンロードします。

考えられる質問に答えようとします。

  1. これはプログラムですか？

    いや これはLinuxディストリビューション全体で、処理用のアドオンスクリプトが含まれています
  2. 他のユニバーサルコンバーターよりも優れているのはなぜですか？

    任意のファイルを処理し、処理パラメーターを構成できます。
  3. そして、新しいコーデックと圧縮アルゴリズムが登場するとどうなりますか？

    OpenSuseリポジトリを介してプログラムを更新し、新しいコーデックを取得します。
  4. 無料ですか？

    はい また、GPL3オープンライセンスの下で利用できます。これにより、アセンブリを自分用にコピー、アップグレードし、このアセンブリを配布できます。






アセンブリはSuseStudioで実行され、 SuseGalleryで公開されて、操作が行われました。

そのため、ビデオを処理する必要があります。 このアセンブリを（仮想マシンまたは実際の無料のハードウェアに）ダウンロードし、yast2を使用してネットワークを構成します。 ftpを介してlivevideoarcに接続し、* .VOBビデオを/ home / linux / inboxNumberフォルダーにアップロードします。 処理を開始する信号は、ファイルVIDEO_TS.VOBです。 受信ボックスに表示されるとすぐに、処理が開始されます。

このプロジェクトの独自性は、音が正規化（静音信号を最大限に増幅）することにもあります。 また、新しいバージョンでは、コンピューター画面から記録された.AVIファイルを処理するための1280x1024フォルダーが追加されます。

あなたが明るい心を持っている場合、欲求と機会があります。プロジェクトの将来のバージョンに含まれる可能性のある他の形式（たとえば、mkv、mov、ipad形式、iphoneなどに変換する）で処理するためのサンプルスクリプトを私に送ることができます。

このプロジェクトは 2010年4月から存在し、積極的に開発されています。
