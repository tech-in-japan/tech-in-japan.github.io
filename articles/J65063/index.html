Internet Explorer 6デスクトップ終了インジケーター

Internet Explorer 6は2001年に登場しました。 2003年から2004年にかけて、彼は90％の記録的な市場シェアを獲得し、長い間彼の欠点をすべて見ずに主導的地位を保持しました。 今日、私たちはIE6時代の終わりを目撃しています。 このプロセスを観察するために、 yanajyのアイデアを実装しました 。 これで、デスクトップ上でIE6ロゴの画像が影にフェードインするのを見ることができます。







ダウンロード： ie6-meter.zip （1 kb。）

ソース： ie6-meter-src.zip （236 kb。）

デモ： http : //veg.by/files/ie6meter/









データは週ごとにliveinternet.ru統計から自動的に取得されます 。 このスクリプトは、通常のHTMLドキュメントが参照する画像を生成します。これは実際にはデスクトップとして設定されます。











かなり大きなスケールが使用されます。 左側のロゴの最初のピクセルの調光は25％から始まります。 文字「e」は、5％のシェアで日陰に完全に隠されています。 上部の残りのテールは、最後の5パーセントの段階的な損失を反映します。 写真は毎日更新されます。









Windows XPでは、デスクトップにHTMLページを追加する2つの方法があります。 標準の方法（[画面のプロパティ]-> [デスクトップ]-> [デスクトップのカスタマイズ]-> [Web]）は消えます。デスクトップにインタラクティブなブラウザウィンドウが作成されるためです。 2番目の方法にはこれらの欠点がありません。通常の画像としてのHTMLドキュメントはバックグラウンドで全画面表示されます。 これを行うには、デスクトップイメージを選択するウィンドウ（[画面のプロパティ]-> [デスクトップ]-> [参照...]）で、HTMLファイルの名前を手動で入力するだけです。







Linuxユーザーは、この奇跡をデスクトップ上に置く方法をコメントで伝えることができます-ありがたいです。
