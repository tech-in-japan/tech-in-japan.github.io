手首を軽くたたくと、タブレットは...追加のモニターに変わります

こんにちは、丁寧なhabrachitatel。



Khabrovskの仕事の写真を含むトピックを公開した後、私は散らかった職場の写真の「イースターエッグ」に対する反応、つまり「このWindowsタブレットとは何か、なぜ小さなアイコンがあるのか​​」という質問を待ちました。







答えは「Koscheevaの死」に似ています-結局のところ、この場合のタブレット（通常のiPad 3Gen）は、Windows 7を搭載した仮想マシンがフルスクリーンモードで実行されている追加のモニターとして機能します。 このような2番目の小さな高解像度IPSモニター。



Android / iOSタブレット/スマートフォンをWindows / Mac OS X用の追加のワイヤレスディスプレイとして動作するようにすばやく簡単に教える方法については、こちらをご覧ください。



自宅ではさまざまなモバイルオペレーティングシステムを実行しているデバイスがよくあるため、「タブレット/スマートフォンをセカンドモニターに変えるプログラム」を選択する主な基準は次のとおりです。



  • AndroidおよびiOSのサポート。
  • WindowsとMac OS Xの両方のサポート。
  • 許容速度;




私にとって嬉しい驚きは、選択したiDisplayプログラムが有名な会社SHAPEによって開発されたという事実であり、Habrahabrの製品について（自分の自由意志と自分のイニシアチブで）何度も書い た 。

今後、このプログラムを使用することで得られる快適さのレベルを80〜85％と評価しますが、有名なAirDisplayや他のメーカーの代替ソリューションは、私をさらに失望させました。







公式サイトのプログラムの利点の説明はかなり簡潔で、「st迷」に追い込むことができるのは、Mac OS XバージョンiDisplayを使用している場合、iOSを同時に実行する36（！）デバイスを接続できる可能性についてです。

36のiPadに「ロングコート」のディスプレイを並べてフラッシュモブを実行することを除いて、他の使用オプションを想像するのは困難です。 まあ、またはあなたはiPhoneから「プラズマ」を構築することができます:)

ところで、Windowsバージョンの説明では、そのような機能は宣言されていません。







他のオプションのモニターと同様に、作業領域を2番目のモニターに拡張したり、イメージをミラーリングしたりできます。 デバイスの向きを選択するためのサポートがあります-タブレットまたはスマートフォンを展開するだけです。 とりわけ、ピクセルの「倍増」モードが可能です。 2048x1536画面は1024x768のように機能します。

私はそのような決定の喜びを感じませんでした-もちろん画像は4倍大きいですが、明瞭さは失われます。







プログラムが機能するには、タブレット/スマートフォンとラップトップ/デスクトップの両方にインストールする必要があります。 まあ、両方のデバイスは同じWi-Fiネットワークにあります。



この時点で、私は完全に予想外の困難に遭遇しました
Windowsバージョンはコメントなしで機能しましたが、iDisplayをMac OS Xにインストールした後（ところで、インストールには再起動が必要です）、驚くべき「バグ」に遭遇しました。ラップトップでドラッグアンドドロップが機能しなくなります。 はい、はい！ あなたは何かをつかむことができますが、手放します-いいえ。

カスタマーサポートとの対応により、この驚くべき効果の理由を見つけることができました-切り替え可能なNvidiaグラフィックス（9400M / 9600M GT）を搭載したMacBookのみが影響を受けます。 Mac OS Xのどのバージョンでも、代替のビデオドライバーをインストールすると、このような驚くべき問題が発生します。

幸いなことに、簡単な解決策がありました-システムを1秒だけスリープモードにすると、問題は奇跡的に消えます（次の再起動まで）。 おそらく、このバグは機能を引き出せないかもしれませんが、残念ながら、解決策は見つかりませんでした。




トレイに隠されており、小さなメニューは別として目立たないWindowsバージョンとは異なり、Macバージョンはより美しく、便利です。 特に、パフォーマンス設定と現在接続されているデバイスのアイコンを含む個別のウィンドウがあります。







すべての設定は自動的に記憶され、システムの起動時に自動ロードが行われます。 このプログラムは、Windows XP（32ビットバージョンのみ）、Windows Vista（32ビットおよび64ビット）、Windows 7（32ビットおよび64ビット）、さらにはWindows 8でも動作します。MacOS Xとの互換性-バージョン10.5以降。 プログラムのデフォルトの言語は英語ですが、サポートサービスは新しいリリースでロシア語の翻訳を追加することを約束しました。



デバイスとの互換性について-Android 2.3および4.0、およびiOS 5および6バージョンでパフォーマンスを確認しました。 問題はなく、アプリケーションの新しいバージョンがかなり定期的にリリースされました。



もちろん、ビデオを見るにはパフォーマンスは十分ではありません（他のアプリケーションもあります）が、メッセンジャー、Habrahabrを搭載したブラウザ、またはiTunesウィンドウを「ドラッグ」できる場所として-それはうまく機能します。



私の経験がすべてのタブレット所有者に役立つことを願っています。Nexus10の登場により、誰もが超高解像度の安価な追加画面を手に入れることができます。 ところで、Nexus 7はこの容量でも非常にうまく機能します。 プログラムへのリンクは提供しません。興味のある人は誰でもApp StoreとGoogle Playで簡単に見つけることができます。



記載されている欠点にもかかわらず、私は個人的にテストされたものの中で最も便利だと思います。 あなたがこの場所を読んだ場合-ありがとう、それから私が試したのは無駄ではありませんでした。
