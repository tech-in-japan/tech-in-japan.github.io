ASUS F83Tを確認する



ある日、私の古い東芝Sattelite Proは仕事を拒否し、サービスセンターを着ることにすでにうんざりしていました。 ラップトップを買うことにしました。

彼は自分自身を500〜600ドルの予算に制限しました。 選択はASUS F83Tに落ちました



特徴



画面： 14 "（1366768）

プロセッサー： AMD Athlon Neo MV-40（1.6 GHz）

チップセット： AMD RS780 + SB700

RAM： DDR2-800：3GB（2キングストンスラット）

ハードドライブ： 320 GB

ビデオカード： ATI Mobility Radeon HD 4570

重さ： 2.39 kg

寸法： 34.5 x 24.1 x 3.45-3.78 mm

オーディオ：内蔵ステレオスピーカー、SRSプレミアムサウンド

ワイヤレスインターフェイス： WiFi 802.11b / g、Bluetooth 2.1

バッテリー： 6セル、4400 mAh

メモリは最大4GBまで拡張可能です。



パッケージバンドル



-充電器

-ラップトップ

-バッテリー

-ぼろ

-ドライバーと追加ソフトウェアを含むCD

このセットでOSをインストールし、ラップトップの使用を開始できます。



本体



本体はしっかりと作られています：つや消し表面のアルミニウムカバー。 アルミニウムの考え方は、底面と側面を除いて、ほぼ全身に実装されています。 下-黒いプラスチック。 画面の周りの黒いフレームは光沢があります。





左側




その上にあります（左から右へ）：

-電源コネクタ

-RJ-45イーサネットネットワークコネクタ

-HDMI

-2つのUSB 2.0ポート

-マイク入力

-ヘッドフォン出力

-PCIエクスプレス



前側




スピーカーはケースの左右にあり、中央にはWiFiオン/オフスイッチとカードリーダーがあります。

音はすぐに注意する必要があります-これはその弱点です。 静かにプレイしますが、快適ではないことがよくあります。 それらが下向きでない場合、状況は良くなるでしょう。





右側




右側にあるのはCD / DVD-RWのみです



裏面




ここには1つのUSB 2.0ポートとD-SUBがあります



使用する



ラップトップにはDOSがインストールされていますが、Windows XP、Vista、7用のすべてのドライバーと追加ソフトウェアを備えたディスクがあります。



画面は良好で、最大解像度1366x768をサポートしています。 画面が180度傾かない。

上部には1.3 MPixウェブカメラがあります



キーボードの左側にある内蔵マイクもあります。



キーボードは耐水性で、ボタンは快適で、押すのはスムーズで静かです。





タッチパッドはアルミニウムでできており、最初は触覚としては非常に珍しいですが、1時間の作業の後、慣れます。 ボタンはクロムで作られています。





インジケータはタッチパッドの少し左にあります：電源オン、充電、WiFi（Bluetooth）、ハードドライブへのアクセス。



性能



電源を入れると、フロントパネルのLEDが点灯します。 充電状態以外のすべてのLEDは青色で、充電はピンクです。

クーラーからのノイズは中程度です。



CPU 64ビット、1コア、1スレッド。





購入後、Windows 7 64がインストールされました-ラップトップは非常に活発に動作します。



ATI Mobility Radeon HD 4570 512Mbビデオカードを使用すると、苦情なくHDムービーを視聴できます。 私はネットワークから、HDMI 1080pを介してテレビで見るときでも品質は良く、「ブレーキ」がないことを読みました。





ハードドライブは320 GBの容量でインストールされます。グラフから速度特性を評価できます。





Windowsテスト





私の個人的な意見


ケースはもちろん美しく見えますが、ほんのわずかなタッチで痕跡が残ります。 私は仕事のスピードに非常に満足していましたが、多くの人にとっては弱いようですが、映画を見るためにも仕事のためにも、特にバッテリー寿命が約3〜4時間の場合は非常に良いです。 私はまさにこのコネクタの配置が好きでした。



練習から、HDムービーを見るときの高性能設定では、ラップトップは3時間作動しましたが、まだ20分残っていました。

そのような価格では、600ドルの地域で、購入時に私にとって最適でしたが、今では後悔していません-素晴らしい馬です。

嫌いなのは音だけです。とても静かです。 しかし、これは構造的にはアレンジメントのせいだと思います。



おやつに



シルバーの長方形-ウィンチェスター。
