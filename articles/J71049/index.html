SIMカードの個人ホスティング

SMSメッセージを使用してコンピューターを制御するプログラムに基づいて、新しいメッセージを作成しました 。 今度は、プログラムは着信SMSを受信し、指定された番号From：7XXXYYYYYYY：messageにリダイレクトします 。 彼女は着信コールをピックアップし、指定された番号に、テキスト「 Call From：7XXXYYYYYYY」を含むメッセージを送信します。



このプログラムを使用して、あるモバイルオペレーターから別のモバイルオペレーターへの移行を容易にすることができます。 新しい番号があることを誰かに通知するのを忘れた場合、誰かが古い番号で電話をかけるかメッセージを送信するとすぐに、SMSメッセージを受信することでそのことを知ることができます。



ここからプログラムをダウンロードしてください 。



最初の場合と同様に、プログラムアーカイブにはいくつかのファイルが含まれています：プログラム自体（ SMSAM.exe ）、設定ファイル（ options.ini ）、インストール用のいくつかのバッチファイル、システムサービスとしてのプログラムの削除、インストールなしで起動します。



鉄

GSMモデム、またはGSMモデムモードで接続できる携帯電話で動作するはずです。 モデムがハングしているポート番号を確認するには、コントロールパネルに移動し、[ 接続済み ]列の[モデム]タブで[ 電話とモデム ]パネルを開いてポート番号を確認します。



セットアップと起動

開始する前に、設定ファイルを変更（ポートを指定、番号を登録）し、選択したGSMモデム（携帯電話）で実行されている他のプログラムがコンピューターで実行されていないことを確認する必要があります。



プログラムをサービスとしてインストールするには、 -installスイッチ （ install.bat ）を使用して実行する必要があります。 サービスからアンインストールする-uninstall （ uninstall.bat ）。 システムサービスに-debugを追加せずに起動するには（起動時にウィンドウが作成されないため、停止するプロセスのリストを "kill"する必要があります）。



設定ファイルは、デバイスに接続するためのパラメーターが設定されているiniファイルです。



[設定]

; モデムがハングするポート番号

ポート= com8

レート= 115200

; ミリ秒のスキャン間隔

タイムアウト= 5000

; 通知を送信する番号

replyTo = 7XXXYYYYYYY

; 再報告または番号からの呼び出し時

; 合格した場合にのみ通知を送信する

; 指定された秒数（洪水防御）

; デフォルト5分

retryDelay = 300
