FiztekhおよびYandexのCourseraでの機械学習の専門

年の初めに、YandexとHSEの機械学習コースがCourseraで開講されました。 ローンチ時には、14,000人がサインアップしました。 オープニングの1時間後、ユーザーはSlackでチャンネルを作成し、そこでプログラムについて議論し始めました。 現在、すでに21,000人のリスナーがいます。







2月9日に、すでにFiztehと一緒に私たちのスペシャリストによって開発されている機械学習専門分野の録音が、プラットフォームで利用可能になりました。 リスナーがトピックに没頭できるように設計されています。



専門分野である「機械学習とデータ分析」は、5つのコースで構成され、独自のプロジェクトに取り組みます。 トレーニングは数ヶ月続きます。 2月19日まで登録できます。 これを行う時間がない場合は、3月14日から2番目のストリームにサインアップできます。



このコースの作成者は、PhysTechで教えるYandex Data FactoryのスペシャリストであるYandexの従業員です。 コンスタンチン・ヴォロンツォフもその中にいます。 同僚の何人かに、専門分野の恩恵を受ける可能性があるのは誰か、なぜそれが必要なのかを尋ねました。 また、カットの下-すべてのコースのプログラム。



Viktor Kantor-アルゴリズムおよびプログラミング技術部門の上級講師、FIVT MIPT、Yandex Data Factoryのユーザーデータ分析グループの責任者。 彼はモスクワ物理学技術研究所で「アルゴリズムとプログラミング技術」、「データ分析」、「銀行情報技術」部門で講義とセミナーを行い、「コンピューター言語学」と「画像認識とテキスト処理」部門でも教えています。



私たちの専門分野は、データ分析の分野の専門家のトレーニングでよく見られる問題を解決します。

  1. Pythonおよびデータ分析ライブラリに関する必要な知識を即座に提供するため、将来、理論が実践から脱却することはありません。
  2. 次のような冒とく的な表現を配置しないように、将来必要な数学をすぐに思い出させます。「ああ、これらは行列です。 まあ、あなたはそれらで何ができるかを覚えていないことは問題ではありません-あなたはまだコンピュータ上でそれらを乗算します。」 私たちはあなたに私たちが言った方法を理解してほしい。
  3. 私たちは、実際によく使用されるものについて説明しますが、単にもっと伝えたいだけのものではありません。
  4. 統計を使用してデータから結論を導き、よくある間違いを避けることをお教えします。
  5. 多くの適用された問題を分析します。その例では、学んだことをすべて適用する方法を学びます。




Evgeny RyabenkoはYandex Data Factoryの有力なアナリストであり、物理学および数理科学の候補者であり、MIPTの准教授です。 彼はVMKモスクワ州立大学とFUPM MIPTの学部で応用統計に関する講義を行っています。 Yandex Data Analysis Schoolの講師。



HSEコースと専門分野の違いは、教育のペースだけでなく、取り上げられているトピックにもあります。 Konstantin Vyacheslavovichのコースは機械学習専用です。 これはかなり現代的な科学分野ですが、その存在の長年にわたって、その教育の特定の学術的規範がすでに形成されています：最初に最も簡単な方法が説明され、次により複雑な方法がそれらに基づいて構築され、最終的にどこかで最先端の技術に到達します適用されたタスクで本当に高品質の結果を得ることができます。 大まかに言えば、機械学習は数学的分析として教えられます。



私たちの専門分野では、より複雑で完全なデータサイエンスの全体像を提供しようとしています。ここでは、機械学習が最も重要な要素の1つですが、それだけではありません。 現在、データサイエンスのトピックに関する標準的なコーパスはありませんが、同僚と私は実践者として、適用された問題で何らかの方法で対処しなければならないことについてある程度の考えを持っています。 たとえば、データを収集するための実験とシミュレーション結果を解釈するための方法を構築するためのテクニックに専念する別のコースがあります-これは統計の応用分野です。 機械学習自体に関しては、専門分野では、HSEが検討するトピックの範囲を拡大し、たとえば、教師なしで教える問題に多くの注意を払います。この分野では、クラスタリング、異常の検索、構造の抽出など、テキスト。 いくつかの重要なトピック-たとえば、アルゴリズムの構成-は、実際の重要性に従って、より詳細に検討されます。



適用されるタスクを見るすべてのトレーニングの出発点。 特定のアプリケーション分野に関係なく、データの科学で最も頻繁に発生する最も重要な生産物を検討します。 推奨システムを構築したり、時系列を予測したりするタスクは、機械学習のさまざまな方法で解決できます。 このような問題がどのように数学的なステートメントに帰着するか、どの分析方法を試してみるのが理にかなっているか、そして最終的に最良のものを選択する方法を生徒に教えたいと思います。

専門分野とコースは、解決するタスクが異なります。 私はHSEからのコースが本当に好きです-それは非常に基本的です。 問題の形式化された数学的定式化、アルゴリズムの構造、およびその背後にある数学について説明しています。 私の意見では、このコースは、何らかの機械学習アルゴリズムを使用するだけでなく、その仕組みを理解したい十分に訓練された学生に適しています。 これを行うには、適切な数学的装置を所有する必要があります。

さらに、スペシャライゼーションのプレゼンテーションも少し異なります。 使用するすべてのものが直感的であることを確認しようとします。




コース1.データ分析のための数学とPython



このコースでは、データ分析に必要な基本的な数学的概念に精通し、Pythonの初期プログラミングスキルを習得します。 コースは2つの大きな部分で構成されています。 コースの最初の部分は実用的で、Pythonプログラミング言語に焦点を当てています。 言語の構文とイデオロギーに精通し、簡単なプログラムを書くことを学びます。 また、NumPy、SciPy、Matplotlib、Pandasなど、実際にデータ分析によく使用されるライブラリについても学習します。 コースの2番目の部分は、線形代数、数学的分析、最適化手法、確率論などの数学の分野に専念します。 同時に、複雑な公式の導出や定理の証明ではなく、数学的概念の説明と実際の応用に重点が置かれています。



コース2.タグ付きデータのトレーニング



私たちの注意の焦点は、線形モデル、ニューラルネットワーク、決定木などの実際の分類および回帰アルゴリズムにうまく適用されます。 個々のアルゴリズムの品質を大幅に向上させることができ、応用問題の解決に広く使用されている構成作成などの強力な手法に特に重点を置きます。 特に、ランダムフォレストと勾配ブースティング法について学びます。



予測アルゴリズムの構築は、データ分析の問題を解決する仕事の一部にすぎません。 アルゴリズムの一般化能力の評価、モデルパラメーターの選択、品質指標の選択と計算など、他の段階を扱います。



コース3.データ内の構造の検索



このコースでは、データクラスタリングアルゴリズムについて学習します。たとえば、これを使用して、類似のモバイルオペレータークライアントのグループを検索できます。 マトリックス分解を構築してテーマモデリングの問題を解決し、データの次元を減らし、異常を探し、多次元データを視覚化する方法を学びます。



コース4.データの結論



データ分析方法の知識は賃金に影響しますか？ 銀行の顧客の信用力を評価するシステムは機能していますか？ 新しいバナーは古いバナーより本当に優れていますか？ このような質問に答えるには、データを収集する必要があります。 データにはほとんどの場合ノイズが含まれているため、それらに基づいて作成できるステートメントは常に正しいとは限りませんが、一定の確率でのみです。 統計的手法は、最も正しい結論を構築し、それらに対する信頼度を数値的に評価するのに役立ちます。



少数の観測から未知のシステムパラメーターを推定するにはどうすればよいですか？ そのような推定の精度を測定する方法は？ 質問に回答するにはどのデータが必要ですか？また、既存のデータを使用して回答できる質問は何ですか？ データを結論に導くために必要なすべてを学びます-実験の編成、A / Bテスト、パラメーターを推定し、仮説、相関、因果関係をテストするための普遍的な方法。



コース5.適用されるデータ分析タスク



このコースでは、テキスト分析と情報検索、協調フィルタリングと推奨システム、ビジネス分析、時系列予測など、データ分析のさまざまな分野から適用されたタスクを分析します。 それらの例を使用して、異種データから記号を抽出する方法、この場合に発生する問題、およびそれらを解決する方法を学習します。 顧客のタスクを機械学習問題の正式な声明に減らす方法を学び、構築されたモデルの品質を履歴データとオンライン実験で確認する方法を理解します。 各タスクについて、合格した機械学習アルゴリズムの長所と短所を調査します。



このコースを聞いた後、一般的なタイプの応用問題に精通し、それらを解決するためのスキームを理解します。



データ分析：最終プロジェクト



モデルデータに基づくタスクとは異なり、実際のプロジェクトに取り組むことで、データの準備から最終モデルの構築、品質の評価まで、データ分析のすべての段階を個別に進めることができます。 その結果、プロジェクトが実際に使用できる武器庫に表示され、将来独立して開発できます。



私たちの専門分野の理想的な目標は、リスナーが自分の専門的な経験に合ったレベルでデータサイエンティストの立場でインタビューできるようにすることです。 データの科学をマスターし、その方法を使用して分析問題を解決する方法を学びます-データの収集から最適なモデルの構築とその品質の評価まで。 Coursera 専門ページの詳細とエントリ。
