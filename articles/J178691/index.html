WebMatrix 3の新バージョン：クラウド、TFS、Git、サイトへのリモートアクセスとの統合



次のバージョンの無料のコードエディタと統合アプリケーション開発ツールWebMatrix 3がリリースされましたWebMatrix 3の機能の中で、次の点を区別できます。



  • HTML / CSS / JS、ASP.NET、PHP、Node.jsプロジェクトのコードサポート。
  • 組み込みのデータベースエディターSQL ServerおよびMySQL。
  • ジェイド、EJS、LESS、CoffeeScriptのエディターサポート。
  • 人気のあるオープンソースCMSテンプレートのギャラリー：Joomla、Drupal、DotNetNuke、Orchard CMS、WordPress、phpBB、 その他多数
  • ExpressおよびSocket.ioを備えたNode.jsを含む既製の開始プロジェクト。
  • HTML / JS / CSS / jQuery、ASP.NET、PHP、Node.jsのオートコンプリートとインテリセンスのサポート。
  • モバイル開発 ：モバイルWeb開発用エミュレーター、モバイルWebサイトテンプレート。
  • SEO、エラー、パフォーマンスの問題のためのウェブサイト分析ツール。
  • 開発者コミュニティの拡張機能のシステムとギャラリー 。
  • Node.jsのNPMモジュールギャラリーおよびASP.NETのNugetリポジトリのサポート。
  • バージョン管理システムGitおよびTFSとの統合統合。
  • Windows Azureクラウドとの統合 、PHP、Node.jsおよびASP.NETプロジェクトのクラウドへの公開、サイトへのリモートアクセス。
  • サイトへのリモートアクセスとFTP、WebDeploy、クラウドホスティングへのアクセスを介した公開。


WebMatrix 3は、公式Webサイトhttp://webmatrix.net/から無料でダウンロードできます 。

Windows Azureクラウドとのより深い統合



以前、WebMatrixは、管理ポータルからダウンロードした公開プロファイルを使用して、PHP、Node.js、およびASP.NETサイトをすばやく公開する機能を提供していました。 現在、開発環境をWindows Azureサブスクリプションに統合して、サイトの発行プロセスをさらに簡素化できます。







独自のWindows Azureアカウントを接続した後、クラウドで新しいWebサイトを作成するか、WebMatrixで開いているプロジェクトを公開するために利用可能なサイトを選択できます。







現在、新しいサイトを公開するには、Windows Azure管理パネルを操作する必要はありません。 以下に、プロセスを詳細に説明するビデオを見つけることができます。



  • WebMatrixを使用してサイトをWindows Azureに発行する


GitとTFSを使用したバージョン管理



WebMatrixの最も期待される機能の1つは、バージョン管理システムのサポートです。 3番目のバージョンでは、GitおよびTFSシステム、および公共サービスTFService、GitHub、CodePlexに登場しました。 バージョン管理のサポートは拡張機能を通じて実装されているため、すぐに他のシステムのサポートが期待できます。







Gitなどのバージョン管理システムを操作するためのツールは、コミット、ブランチ、同時作業、クラウドへの継続的な公開などに必要なすべての機能をサポートします。







リモートコード編集



WebMatrix 3の新機能により、クラウドでホストされているサイトやFTP経由でアクセス可能なサイトをリモートで開くことができます。 これにより、WebMatrixの便利なコード編集ツールを使用して、Web上でホストされているサイトをプラークで少し編集することができます。







すべてのファイルはリモートで編集されますが、必要に応じて、サイト全体をローカルマシンにダウンロードできます。
