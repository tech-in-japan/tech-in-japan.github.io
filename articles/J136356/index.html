Java Day SPB 2012の招待





親愛なる友人

Java Day SPB 2012のJava開発者向けのカンファレンスは、2012年2月10日にサンクトペテルブルクで開催され、Java EE、Java SEとJavaFX、組み込みシステム用のJava MEとJavaの3つのストリームで構成されます。 昨年のイベントの後、私たちはあなたの希望を考慮し、Java SEおよびMEストリームの設備を拡張しました。 別の非常に興味深いインタラクティブストリームがもう1つありますが、その作成者とスピーカーがそれについて書いています。



会議への参加は無料です。 事前登録が必要です





JavaDay SPB 2012は、サンクトペテルブルクのMoskovsky Prospekt 97aにあるHoliday Inn Moskovsky Vorota Hotelの会議ホールで開催されます。 この会議は、オラクルのソフトウェア開発担当副社長であるジョルジュ・サーブによる全体講演から始まります。 Saabの責任には、Java仮想マシン、システムライブラリ、およびJava言語が含まれます。 彼のレポートでは、開発者のグローバルコミュニティに最も人気があり、信頼性が高く、スケーラブルなオープンソーステクノロジーの1つとして、Javaのさらなる開発のためのオラクルの既に実装された活動と計画について話します。



Alexis Moussine-Pouchkineは、JavaEE 7-Development＆The CloudでエンタープライズJavaセクターの動向について語ります。 Java EE仕様の次のバージョンの変更と革新について学びます。これにより、「クラウド」システムの開発に新たな可能性が広がります：スケーラビリティ、弾力性、保守性など。



Alexander Kuznetsovのレポート「Java FX 2.0-Application Possibilities and Java FX 2.1 News」では、Javaでクライアントアプリケーションを開発するトピックが公開されます。このトピックは、他の3人のOracle開発者、Sergey Grinev、Sergey Malenkov、Oleg Sukhodolskyによって続けられます。



Sergey Walrus KuksenkoとAleksey TheShade Shipilevのレポート「Performance Optimization Methodology」では、Javaプログラムを最適化する方法が示されていません。 お気に入りのプロファイラーの使用方法については説明しません。 SergeyとAlexeyが「正しいこと」（TM）を行う方法を説明します。foo（）メソッドを100倍加速するのに2週間費やさず、アプリケーション全体で+ 0.001％の速度しか得られません。 プロファイラーを開始する前に何をすべきかについて。 レポートでは、「スループット」、「レイテンシー」、「利用率」、「スケーラビリティ」、「アムダルの法則」などの「スマート」な言葉について説明しています。



しかし、JavaはOracleだけではありません。 Javaは世界中で900万人の開発者です。 ロシアのJava開発者コミュニティの著名な代表者の1人であるアカデミック大学のYakov Sirotkinが、同じレポートでトランザクションからアミノ酸に切り替える方法を教えてくれます。
