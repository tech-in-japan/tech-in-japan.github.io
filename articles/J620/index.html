TCCC 2006オンサイトラウンドは11月15日にサンディエゴで始まります

11月15日は、 TopCoder Collegiate Challenge 2006の最後の対面ステージを開始します。これは、実際には個人競技でのスポーツプログラミングの学生世界選手権です。 個人競技会で開催される他の主要な競技会（ Google Code JamおよびTopCoder Open ）とは異なり 、高等教育機関の学生と大学院生のみがこの競技会に参加できます。 今回は、世界中から3088人のプレイヤーが参加しました。





11月15日から18日まで、米国サンディエゴで、TCCC 2006の準決勝と決勝で48人のプログラマーが集まります。 競技スケジュールは次のとおりです。



11月15日-準決勝1

11月16日-準決勝2および準決勝3

11月17日-ワイルドカードラウンド

11月18日-最終



今回は、ロシアからの代表団の数が最初の場所である16人であることに注意してください。 次に代表的な代表団は中国です（参加者6人）。 TCCC 2006のオンサイトラウンドで、今年のGCJの勝者はピーターミトリチェフ、そしてそこで3位になったアンドレイスタンケビッチ、そして4位になったアンドレイハルヤビンでした。
