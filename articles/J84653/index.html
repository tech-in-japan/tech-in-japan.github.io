Acer Aspire Timeline 3810のレビュー

このデバイスを購入した感想を共有し、同時に最近レビューした類似品と比較したいと思います。







なぜ選んだのですか？ 選択基準は次のとおりでした：軽量、コンパクト、優れたパフォーマンス（すべてのストライプのネットブックが消える）、および好ましくは、長いバッテリー寿命（約2:30前のラップトップは時々十分ではなく、時には最も不適切な瞬間に）。 そしてまだ-私はVistaがプレインストールされた状態でそれを取りたくありませんでしたが、Win7がプレインストールされていれば、それは逆に私にとってプラスになるでしょう。



選択肢



まず、事実を乾燥させます（すべてのオプションが太字でリストされています-私が好んだもの）：
オペレーティングシステム Windows 7 Home Premium 64ビット
CPU Core 2 Solo ULV SU3500 1.4GHz / Core 2 Duo SU7300 1.3GHz / Core 2 Duo SU9400 1.4 GHz
RAM 4096（2x2048）MB DDR3 / 3072（2048 + 1024）MB DDR3
表示インチ 13.3 /14/15.6（画面が異なるモデルの番号はそれぞれ異なります-4810と5810、それぞれ少し低い）
ハードドライブ 250/500 GB（5400 rpm）
ビデオカード Intel GMA 4500MHD / ATI Radeon HD 4330 512 MB
無線インターフェース WiFi、WiMax 、Bluetooth
その他 USBx3、VGA（D-Sub）、HDMI、S / PDIFデジタルオーディオ出力、LAN
毎 1.0メガピクセルウェブカメラ、マイク、3-in-1カードリーダー、マルチタッチタッチパッド
価格 21〜29歳 摩擦（21-41千、大きな対角線を持つモデルを考慮する場合）
ここでのオプションは、通常どおり、メーカーによって決定されるため、いくつかの点で私の選択がおかしいと感じた場合、考えられる説明があります。

最初に決定する必要があるのは、対角線のサイズです。 なぜなら 14インチと15.6インチの画面を備えたモデルには、DVDドライブが内蔵されており（絶対に必要ありません）、重量が0.4kgと0.8kg多いため、13.3インチを選択しました。 さらに、すべてのモデルの画面解像度は同じです（1366 x 768）。 13.3インチバージョンの重量は1.6kgです。

次に、プロセッサ。 ラインナップ内のより若いプロセッサはシングルコアであり、その時間はすでになくなっているため、上位2つが残ります。 それらの違いは、それが価値があるかどうかにかかわらず、約3000rです。誰もが自分で決定します。

そして最後に、ビデオカード。 プレイを気にしない場合は、モデルを個別のモデルで見る必要があります（一般に、このようなラップトップには2枚のビデオカードがあり、バッテリーを節約するためにそれらを切り替えることができます）。

私の意見では、他のすべては記述されたコンポーネントに結び付けられており、それらの選択は明らかにあなたが付属物で何を得るかを決定します。 選択したインスタンスのコストは約25500rです。

外観



開梱されたのは、ラップトップ自体、取扱説明書、充電、バッテリー、あらゆる種類のジャンクです。 私はそれをオンにしたが、非常に静かに動作することに気づいた（重い負荷ではそれがブザーだけでなく、非常に静かに）。 画面の周りにはマット（喜ぶ！）フレームがあります。このフレームでは、印刷物がよく見えず、タッチパッドの周りに目立ちますが、目を引くことはありません（おそらく色による）。 これは人生でどのように見えるかです：







バッテリー



ラップトップには、画面を暗くしたり、貪欲なディスクリートグラフィックカードを無効にしたりする省エネボタンがあります。 このモードでは、ラップトップは6（Wi-Fi、アクティブサーフィンなど）から10（Wordのみが開いています）時間まで稼働します。 省エネなしで、3.5（現代の3Dゲーム）から6時間（同じサーフィン）まで動作します。 数字は、順序を評価する機会を与えるためだけに近似しています。



パフォーマンス。



これは、Counter Strike Source、Team Fortress 2、Left 4 Dead 2でテストされました。解像度-ネイティブ、1366x768、すべての設定が最小限。 CSSとL4D2では、ラップトップは許容レベルを示しました; TF2では、プレイヤーが集まる場所では、fpsは〜18-20に沈み、快適にプレイできません。 L4D2の数値を示します（avg fps約40、レベルごとの最小値-16）。 CSSでは、平均fpsは約75で、これは本当に素晴らしいです。







ケースとキーボード。



キーボードは非常に独創的で、ボタンは脚の平らなパッドです。 すべてが正常に押されますが、声の演技ではキーボードのすべての文字が白色であるため、このために問題がある可能性があると言われます（盲目的に入力した場合、これは欠点になることはほとんどありません）。 プリントはよく目立ちます。 ケースはしっかりしていて高品質で、きしみや曲がりはありません。 蓋は金属製で、触ると心地よく涼しく、汚れますが、光沢はありません。

代替案との比較。



最近、ASUS UL50VtとAcer 1810TZのレビューがすり抜けました。私は自分でTravelMate 8371Gについて追加します。 だから、それらについていくつかの言葉。

ASUS UL50Vtはより強力で、より高価で、より重く、より長く、見た目はより長く動作し、すべて光沢があります。

1810TZは非常に似ていますが、サイズと重量が小さいことに加えて、かなり弱く（SU4100プロセッサと統合グラフィックカードのみ）、光沢のあるプラスチックカバーも付いています！

私はTravelMate 8371Gを手に持っていませんでしたが、特性と外観で判断しました。価格（travelmateは5kほど高価です）とマットスクリーンを除いて、説明したラップトップのコピーです。 これは私にとってそれほど重要ではありませんでしたが、コメント/投票で判断すると、彼らはたいていそれを望んでいます。
