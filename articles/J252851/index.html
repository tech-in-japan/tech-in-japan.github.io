PHPプログラマ向けの30のすばらしいライブラリ

Habrahabrの読者に出版物30 Amazing PHP Libraries for Programmers and Developersの翻訳を提供します。 著者は、多くの労力をかけずにクールなものを実装できるように、いくつかの素晴らしいPHPライブラリをコンパイルしました。 レビューの目的は新しいものを提示することであるため、リストには人気のあるライブラリは含まれていません。



30. Goutte







Goutteは、Webサイトのスクレイピングとデータマイニングに使用されます。 ライセンス-MIT。 サイト/ブログをクロールし、HTML / XMLからデータを取得するための広範なAPIを提供します。



29. GoogChart







GoogChartは、Google Chartを使用して動的なグラフを作成するためのPHPクラスです。 Google Charts API全体には適用されませんが、非常に使いやすくなります。



28. GifCreator







名前はそれ自体を物語っています。 これは、複数の画像からアニメーションGIFを生成するPHPクラスです。 画像とその表示時間を指定するだけで十分です。 サイズと透明度は、最初のフレームによって決まります。



27. dBug







dBugは、ColdFusion sfdumpのPHPバージョンです。 色と構造化された変数情報を表形式で表示します。 スタイルを編集したり、表を折りたたんだり展開したりできます。 dBugは、特定のタイプのデータの出力を強制的に変更することもできます。 たとえば、オブジェクト変数の出力を配列として構成できます。



26. アリス







Aliceを使用すると、テスト目的で数百の偽データ（フィクスチャ）を即座に生成できます。 読みやすく編集しやすい制限付きの複雑なデータを生成できます。 フィクスチャの範囲、オプションのデータ、オブジェクト値、一意の制約を制御できます。



25. PHPGeo







PHPGeoは、2つの座標間の距離を高精度で測定するためのシンプルなライブラリです。 彼女はGaversinusとVincentの式を使用して距離を計算します。 マップまたは場所に関連する任意のアプリケーションと統合できます。



24. ジオコーダー







ジオ認識アプリケーションの開発を計画している場合は、ジオコーダーをご覧ください。 この拡張可能なPHPライブラリは、プロバイダーとHttpAdapterの2つの部分に分かれています。 また、ジオコーディングのための抽象化の追加レイヤーも提供します。



23. カール







Purlは、URLを操作するためのオブジェクト指向ライブラリです。 URLを簡単にインスタンス化し、URLを作成した後にメソッドをチェーンできます。 ライブラリは、パス、フラグメント、およびクエリ操作をサポートしています。



22. スナッピー







Snappyでは、URLまたはHTMLページでスナップショット、サムネイル、およびPDFを作成できます。 このライブラリは、Windows、OSX、Linuxで利用可能なwkhtmltoimageとwkhtmltopdfに基づくWebKitを使用します。



21. ShellWrap







ShellWrapは、PHPで強力なUnix / Linuxツールを使用できるようにするライブラリです。 PHP例外を使用した単純な構文とエラー処理を使用して、コマンドを簡単に呼び出すことができます。 すべての引数は適切にエスケープされ、実行可能ファイルへのパスは自動的に解決されます。



20. ユニレスト







Unirestは、多数のアプリケーションに最適な軽量HTTPライブラリーのコレクションです。 PUT、DELETE、PATCH、GET、およびPOST要求を実行します。 gzip、基本認証、ファイルのアップロード、フォーム、カスタムタイムアウト、各リクエストのデフォルトヘッダーをサポートしています。



19. IniScan







IniScanは、php.iniファイルをスキャンしてセキュリティを確保するために設計されたツールです。 一般的なセキュリティエラーを検索し、結果を報告します。 スキャンルールのしきい値を設定して、指定したレベルを超えるエラーを見つけることができます。



18. PHPキャプチャ







PHP Captchaを使用して、オーディオおよびビデオのキャプチャを生成できます。 文字の回転、フォントサイズの選択、文字セットの選択、ランダムまたは指定された背景、追加の表示テキストおよび文字影をサポートします。



17. ラチェット







Ratchetは疎結合PHPライブラリであり、WebSocketを介したリアルタイム開発、サーバーとクライアント間の双方向通信のためのさまざまなツールを提供します。 すぐに利用できるコンポーネントは多数あります。 数分で独自のチャットアプリケーションを作成できます。



16. TCPDF







TCPDF-PDFドキュメントを作成するためのPHPクラス（オープン）。 すべての標準ページ形式、カスタムページ形式、UTF-8、リンクを含むPDF注釈、テキストレンダリングモード、フォントセット、テキストのストレッチと間隔、ページ圧縮、TrueTypeUnicodeおよびOpenTypeUnicodeをサポートしています。



15. ムニー







Muneeライブラリを使用すると、テンプレートのアセットを変更することなく、CoffieScript、Less、SCSSをオンザフライでコンパイルし、画像のサイズを変更し、JSとCSSを圧縮し、サーバーとクライアント側で結果をキャッシュできます。 全体として、これは非常に高速な要求を達成し、帯域幅を節約するための優れたオプションです。



14. HTML Purifier







HTML Purifierは、悪意のあるコードをすべて削除するように設計されたHTMLフィルタリングライブラリです。 XSS攻撃からコードを保護するために使用できます。 このライブラリは、堅牢なホワイトリスト、積極的な解析を使用し、マークアップが標準になるようにします。



13. 検出器







Detectorは、ユーザー/訪問者に関する多数の情報を決定するためのオープンライブラリです。 正確なWeb分析を構築するのに役立ちます。 ブラウザで使用されるデバイスの種類であるユーザーエージェントと、そのHTMLおよびCSS機能を表示します。 また、データを要求した人（人またはボット）も報告します。



12. おっと







Whoopsを使用すると、エラーや例外に簡単に対処できます。 これは、コードに存在するエラーおよび例外に関する詳細情報を伝える優れたインターフェースを提供します（バックライト付き）。 また、JSON、XML、SOAP、およびAJAXもサポートしています。



11. ゴーフレット







Gaufretteは、ファイルシステムの抽象化レイヤーを提供します。これにより、メディアファイルの保存場所と保存方法を知らなくても、アプリケーションを開発できます。 また、ファイルシステムの決定に加えて、ソースコードを変更せずにファイルの場所を変更することもできます。



10. 想像してみて







Imagineは、画像を操作するための豊富なオブジェクト指向ライブラリです。 開く、保存する、作成する、サイズ変更する、透かしを入れる、コラージュを作成する、画像を反映するなど、さまざまな操作を処理できます。 Imagineの座標系を使用すると、画像にさまざまな形状を描画できます。 色クラスは、画像に色を適用するためのRGB値を作成するのに役立ちます。 また、アニメーションGIFのレイヤーと操作もサポートしています。



9. イディオム







Idiormは、PDOのアドオンであるPHP5の軽量ORMおよび柔軟なクエリデザイナーです。 長くて日常的なSQLクエリを記述する必要はありません。 さらに、XML設定、モデルクラス、またはコードジェネレーターは必要ありません。



8. phpFastCache







名前が示すように、これは非常に効率的なキャッシングシステムです。 データベースの負荷を減らすことにより、動的なWebアプリケーションを高速化するように設計されています。 MemCache、Memcached、files、wincache、mpdo、pdo、apcなど、いくつかのキャッシュ方法をサポートしています（期間は明示的に定義できます）。 サーバーに変更を加えて、キャッシュのために再度PHPクラスを書き換える必要はありません。



7. JpGraph







JpGraph-オブジェクト指向グラフを作成するためのPHPライブラリ。 200以上の組み込みの国旗、400の名前付きの色があり、ガントチャート、いくつかのY軸、アルファブレンディング、内部キャッシュ（タイムアウト付き）を追加でサポートしています。 このライブラリを使用すると、任意のサイズの折れ線グラフ、ヒストグラム、パイ、カード、キャンドル、極座標、レーダー、円グラフ、等高線グラフを作成できます。



読む： 20+グラフとチャートを作成するための便利なオンラインツール 。



6. SEOstats







SEOstatsは、バックリンク、ページランク、ウェブサイトの傾向、キーワード、トラフィック統計、Alexaランクなどの詳細な分析など、さまざまなSEO関連のメトリックを提供する強力なオープンソースライブラリです。 Alexa、Google、Moz、オープンサイトエクスプローラ、SEMRush、Facebook、Twitterからデータを収集します。



5. PHPパーサー







PHP Parserは、静的コードの分析と操作を簡素化するためにPHPで作成されています。 PHPソースコードを抽象構文ツリーに変換します。 構文ツリーを使用すると、コードの問題やエラーを簡単に分析できます。 さらに、構文ツリーをPHPコードに戻すことができます。 画像に例を示します。白い部分は単純なコードを表し、黒い部分には生成された構文ツリーが含まれています。



4. PHPエラー







PHPエラーは、通常のエラーメッセージを、完全な構文強調表示とコードフラグメントを使用してより視覚的なものに変換するために使用されるライブラリです。 これは、単に正しくなく、AJAXに適用されるいくつかのエラーメッセージを修正します。 さらに、手動でオフまたはオンにしたり、エラーメッセージなしで個別のセクションを開始したり、選択したファイルを無視したりできます。



3. PHP-CPP







PHP-CPPは、PHP拡張機能の開発に使用できるC ++ライブラリです。 使いやすく、よく文書化されたクラスを使用して構築され、PHP用の独自の拡張機能を作成します。 C ++で記述されたコードはPHPよりも速く実行されるため、これによりプログラムを高速化できます。



2. エデン







Edenは、ほぼすべてのPHPフレームワークとCMSで動作する再利用可能なコンポーネントのセットです。 スタートアップ、キャッシュ、ファイルシステム、Webサービス、テンプレート、国際化、支払いシステム、クラウドテクノロジーのコンポーネントをカバーしています。 Google、Facebook、Yahoo、PayPal、Tumblr、Foursquare、Amazon Web Servicesなどの主要なWebサービスと連携します。



1. イナゴ







Locustは、使いやすいユーザー負荷テストツールです。 これは、システムが維持する同時ユーザーの数を調べるために使用できます。 Pythonでスクリプトを作成できます。 Locustには、重要なデータをリアルタイムで表示する優れたインターフェイス（HTML + JS）があります。
