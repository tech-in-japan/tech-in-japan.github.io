Javaのゲーム：2016年10月14〜15日のキエフでのJava会議

2日間、秋のキエフはJavaスペシャリストの魅力の中心になります。 10月14〜15日に 、ウクライナのJava開発者コミュニティであるUser Group Ukraineが主催するJava会議JavaDay 2016がここで開催されます。









Javaday 6-どのようになりますか



昨年の5周年記念カンファレンスは、 40人のスピーカーからの興味深いセッションを含む5つのトラックで参加者を満足させました。 今年、このイベントは、 ウクライナのITイベントの中で2位にランクされた JavaDay 2015の成功を繰り返し ます 。 会議、特にイベントは、できるだけ多くの洞察を得たい人を喜ばせます。



会議の目的は、同僚に効果的に学び、集中的にコミュニケーションをとる機会をコミュニティに与えることです。 イベントが開催されるParkovyカンファレンスセンターは、開発者とスピーカーが両方の目標を快適に達成できるようにします。



少なくとも1回JavaDayを訪れる幸運な人は、次のことができることに加えて、

  • Javaでの作業における最良かつ最新の技術を発見する
  • 主要な国際的およびウクライナのジャワスターとチャット
  • 新しい興味深く有用な連絡先を作成する
  • Javaコミュニティーに参加してください
すべての会議参加者は、イベントパートナーからの予想外の楽しい驚き、忘れられない雰囲気、そして組織に対する必然的に非標準的なアプローチを期待しています。



たとえば、昨年、一連のイベントはすべてJavaの20周年に捧げられ、キエフのすべてのJavaDay参加者​​は、会議参加者とオンラインで話し合ったジェームズゴスリングに個人的に尋ねる素晴らしい機会を得ました。







プログラム



主催者は、イベントの面白くてイベントフルなプログラムを形成します。 現在（9月1日）、98％で形成されています。







今年の会議の特徴は、 Spring Frameworkとその周辺の製品のエコシステムで知られるPivotalのスピーカーグループの参加です。 会社の代表者に、これまでで最も人気のある製品について話してもらいました。



  • Juergern Hoeller -Spring Frameworkの著者である彼のレポートでは、Spring 4.3の新しいバージョンと、2017年にリリースが予定されているSpring 5で予想される機能について説明します。
  • StéphaneNic​​oll -Spring Bootの主な開発者の1人が、その機能、範囲、将来の開発計画について話します
  • Christoph Strobl -Spring Data Committerは、RESTアプリケーションを作成するための新しいSpring Data機能とRedisとの相互作用を共有します
  • Alvaro Videla -RabbitMQのコア開発者は、分散システムの構築、分散システムの構築に使用されるアプローチとソリューションについて講演します


バランスを保つために、 Java EEに関する一連の講演もあります。 会議の開催までに、オラクルがこのプラットフォームの運命に関する将来の計画を共有することを願っています。



Java EEの元Oracleエバンジェリストであり、現在はJavaEE Guardianグループのリーダーの1人であるReza Rahmanが 、JUG UA（会議の前に開催予定）の会議で聴衆とライブコミュニケーションを行い、Java EE 8の計画と展望について話します。 JMS 2のマスタークラスとIvar Grimstad （JavaチャンピオンおよびJCPメンバー）が、マイクロサービスを作成するためのJava EEセキュリティ、HTTP 2、JMS 2.1、Java EE Microprofileの計画仕様について説明します。



伝統的に、 Baruch SadogurskyとZhenya Borisovがいなければ、どのようなJavaDayがありますか。 私たちは彼らからの新しいレポート、バトル、パズルゲーム、そしていつものように多くのポジティブなものを待っています！ 報告に加えて、ZhenyaはApache Sparkのマスタークラスを提供します。



主催者は、国内スピーカーの数が年々大幅に増加していることを非常に喜んでいます。 さらに、投票の結果によると、彼らの報告は会議の5つの最高の報告の1つです。 Java開発の地元のスターの中で、私は強調したい：



  • ニコライ・アリメンコフ 、JEEConfおよびXPDaysの主催者
  • Taras Matyashevsky、会議の創設者Morning @ Lohika
  • Egor Bugaenko 、そのレポートはリスナーを無関心のままにすることはできません！
