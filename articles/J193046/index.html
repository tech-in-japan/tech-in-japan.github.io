LEC-2126ネットワークプラットフォームの概要



本日、ファンレスパッケージの産業用シンクライアント/ネットワークプラットフォームの概要をご紹介します。 Lanner LEC-2126をご覧ください 。



あなたがそれを拾うときにすぐに気づく最初のものは重量です。 デバイスのサイズが20 x 15 cmであるため、重量は2キログラムを超えます。 ラジエーターでもあるケースは厚いスチールで作られており、その表面はフィンで覆われています。







次の興味深い点は、デバイスのマザーボードにアクセスするためにドライバーを用意する必要さえなく、底部カバーを取り付ける役割が4つのゴム製の足用ネジに割り当てられていることです。







肉厚〜3-4mm。 メーカーによれば、ケースは完全に防塵です。







このプラットフォームには、Intelコントローラー上の6ギガビットRJ45インターフェイス、組み込みモデム用のSIMカードリーダー付きMini-PCI-eスロット、CompactFlashカード用スロット、および多数のカスタマイズインターフェイスがあります。

詳細仕様：

  • デバイスの寸法：198 x 42 x 145mm。
  • Intel ICH8Mチップセット。
  • デュアルコアCPU Intel Atom D510 1.66GHz。
  • 最大2 GBのDDR2 RAM SODIMM。
  • 1x CompactFlash Type IIソケット。 1 x HDD / SSD 2.5インチベイ。
  • パッシブCPU冷却。
  • グラフィックスIntel GMA3150、VGA出力DB15（2048x1536 @ 75Hz）。
  • SIMカードリーダー付きMini-PCIe x 1。
  • 60W @ 12v 2a電源アダプター
  • 6x GbEインターフェイス。 Intel 82574L x1、Intel 82583V x5
  • コンソールRJ45ポートx 1。 2x USB 2.0
  • ハードウェアモニターWinbond W82327THG
  • 重さ：2.5 kg。
  • 温度範囲：0〜40 C
  • 認証：CEエミッション、FCCクラスA
