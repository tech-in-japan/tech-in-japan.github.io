中小企業や支社で回復力のあるプライベートクラウドを構築するための包括的なソリューション

多くの場合、中小企業や大企業では、1Cサーバー、テレフォニーサーバーなどの重要なアプリケーションのダウンタイムがどれほど重大かを考えることはありません。 ただし、ハードウェアとソフトウェアは遅かれ早かれ邪魔され、システム管理者（さらにはIT部門全体）の作業が沸騰し始めます。 管理は非常に不満であり、ここで問題を解決する必要があり、管理者は言うことができます-我慢して、私たちは問題の解決に取り組んでいます。 理由を分析すると、管理者に問題が警告されたことが判明する場合があります。新しいサーバーが必要で、古いサーバーはほとんど存続せず、完全バックアップもバッ​​クアップサーバーもありません。 また、最新の仮想化ツールは、多くの場合、既製のシンプルなフォールトトレラントソリューションを提供しません。 会社はすでに仮想化を使用していますが、重要なサーバーとデータはまだバックアップされておらず、障害が発生した場合、情報システムのダウンタイムは数分から数時間に達することがあります。 また、このプロセスで重要な役割を果たすのは、システム管理者の資格レベルです。 小規模企業を検討する場合、通常、システム管理者は1人です（そして、それが来るかもしれません）。 しかし、まだ解決策があります！



DEPO Computersは最近、 DEPO Cloud 200ソリューションを導入しました。これにより、比較的小規模な企業でも、高価な専門家にお金を浪費することなく、高いレベルのフォールトトレランスを実現できます。 ソリューションの主な重点は、インストール、構成、およびサポートの容易さです。 DEPO Cloud 200は、仮想マシンの仮想化と高可用性を備えた、フェールオーバークラスターとして動作する2つのサーバーです。 仮想化を管理するためのソリューションとして、Stratus Avanceソフトウェアが使用されます。StratusAvanceソフトウェアは、実装とメンテナンスに高度な資格を持つITスペシャリストを必要としません。







サーバー上のすべてのデータは常にリアルタイムで同期されます。 サーバーはホットリザーブにあるため、メインサーバーの可用性に関する問題が検出されると、2番目のサーバーがその機能を引き継ぐため、管理者が関与する必要はありません。



Avanceは追加のサードパーティソフトウェアを必要とせず、ソリューションは全体としてフォールトトレランス用の同様のソリューションよりも収益性が高いことに注意することが重要です。 また、 DEPO Cloud 200は外部ストレージシステムを使用せず、一般的に高価なサーバーコンポーネントを使用しません。 総コストは、他のメーカーの類似品よりも10〜30％低く、コンポーネントの構成よりも約10％低くなっています。



ソリューション自体はサーバーのハードウェアと非常に密接に統合されているため、ハードウェア障害の可能性も予測できます。 サーバーのRAIDコントローラーも制御されるため、サーバーのメンテナンスが最小限に抑えられます。 制御ソフトウェアは、主電源の電圧と温度からRAMのエラーの修正まで、あらゆる種類のインジケータ（合計で約150個あります）の予防的な監視を常に実行します。 値が標準から逸脱している場合、ソフトウェアはすぐに応答します。 電子メールレポートを管理者に送信し、Stratusテクニカルサポートに問題を報告する。







DEPO Cloud 200ソリューションが配信されると、クラスターはクライアントではなく専門家によって構成されますが、Avanceのインストールと構成はそれと同じくらい簡単です。両方のサーバーで20〜30分しかかかりません。 構成と管理自体は、Webインターフェースを使用して実行されます。







ソリューションに含まれるサーバーは、必ずしも互いに隣接しているとは限りません。 それらは相互に最大5 kmの距離に配置でき、そのようなリモートサーバー配置の重要な要件は10ミリ秒以下のデータネットワークの遅延です（通常の1Gbit / sアダプターまたはIntel X520などを使用してネットワーク接続を編成するのに十分です-しかし、すでに10Gbps）。







仮想化プラットフォーム自体は、最新のCitrix XenServerに基づいています。 同時に、Microsoft Windowsシリーズのほぼすべてのオペレーティングシステムを現時点で仮想マシンにインストールできます（ただし、W2k SP4、XP SP3よりも古いバージョンではありません）。 Red Hat Enterprise Linux 4.x、5.xおよびCentOS 4.x、5.xを使用することもできます。 次のような新しいオペレーティングシステムのサポートが計画されています。 Windows Sever 2012、RHEL 6.x、CentOS6.x。 現在、最大16台の仮想マシンがサポートされており、最新バージョンのAvanceは最大24台のマシンをサポートしています。



まとめると


DEPO Cloud 200ソリューションは、ITプロフェッショナルに追加の負担をかけることなく、フォールトトレランスとサービスおよびアプリケーションの高可用性を提供する必要がある場合に、小規模企業またはブランチで仮想化を実装するのに最適です。 同時に、ターンキーソリューションを提供することも可能です。特定の顧客ごとに完全なインストールと構成を行います。



一般に、このソリューションは、仮想化を実装してプライベートクラウドを構築する際のコストを大幅に削減でき、インフラストラクチャのジョブ数が25から50の企業や支店を対象としています。
