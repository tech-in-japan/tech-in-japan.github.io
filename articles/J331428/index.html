4月上旬に、 SAP Encoderコンテストを発表しました。 参加者は、SAP Cloud Platformに基づいて作成された指示に従ってプロジェクトを提案する必要がありました。 この間ずっと、参加者は決定を準備していました-そして今、それらを提示する時です。 ソリューションは興味深いことが判明したため、視聴に参加することをお勧めします。 参加者のプレゼンテーションに加えて、開発に関する高度な傾向を示すSAPに関する2つのレポートを聞くことができます。 カットの下の詳細を確認し、参加することを忘れないでください！









SAP Coder 2017コンテストの決勝戦の生放送は、6月28日10:00〜14:00（モスクワ時間）に開催されます。 ライブブロードキャストを見逃さないように、カレンダーにイベントを追加します 。


SAP Coder 2017は、最も先進的な開発アプローチの習得に取り組んでいるすべての人を対象とした競争です。 今年、SAP CISはこのコンテストを初めて開催します。 競争の一環として、参加者は独自のアプリケーションを開発して、ロシアのSAP顧客が提供する実際のビジネス上の問題を解決します。 モノのインターネット、機械学習、予測分析、その他の高度なビジネスシナリオ向けのデジタルサービスを提供する革新的なオープンソースのSAPクラウドプラットフォームであるSAPクラウドプラットフォームの開発が進行中です。 サイトでの競争についてもっと読んでください。



放送中に、技術トレンドについて聞き、プラットフォームテクノロジーの分野におけるSAP戦略に精通し、SAPクラウドプラットフォームを使用したビジネス開発の例について学びます。
