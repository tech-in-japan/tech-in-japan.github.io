更新されたWindows Azureモバイルサービス：iOSサポートなど

Windows Azureプラットフォームは絶えず進化しています;今日、開発者はWindows Azureモバイルサービスを新しい機能とツールで更新できるようになりました。 以下にイノベーションの簡単な説明を示します。詳細は、Scott Guthrieの最初の発表に記載されています。



IOSプラットフォームのサポートと個別のiOS SDKリリース



以前は、モバイルサービスのインストルメンタルサポートはWindows 8プラットフォームでのみ利用可能でしたが、RESTサービスを使用することが提案された他のプラットフォームと連携していました。



アップデートのリリースに伴い、iPhoneおよびiPad用のiOSアプリケーションを開発するための新しいツールが追加されました。 これらのツールは、無料のApache 2.0ライセンスの下でオープンソースです。







新しいSDKのおかげで、サードパーティのサービスおよびMicrosoftアカウントサービスを介した情報ストレージおよび認証サービスへのアクセスがiOSアプリケーション開発者向けに簡素化されました。 プッシュ通知のサポートは、新しいiOS SDKではまだ利用できず、近い将来に表示される予定です。



次のリンクで、iOSの新しい開発の機会に慣れることができます。



  • 開発紹介
  • iOS向けモバイルサービスのデータを使ってみる
  • iOS向けモバイルサービスで認証を開始する


サードパーティ認証サービスのサポート：Facebook、Twitter、Google



既に提案されているMicrosoftアカウント認証サービス（以前のアプリケーションで使用可能）に加えて、この更新プログラムはサードパーティの認証サービス（Facebook、Twitter、Google）のサポートを提供します。







モバイル開発者にとって、これらのサービスの使用はさらに簡単になりました。 以下は、クラウドサービスを利用するのに十分なWindows 8とiOSのコードです。







Windows Azureモバイルサービスを使用した承認サービスの使用の詳細については、次のリンクをクリックしてください。



  • アプリをMicrosoftアカウントに登録する方法
  • Facebookにアプリを登録する方法
  • アプリをTwitterに登録する方法
  • アプリをGoogleに登録する方法
  • Windowsストア向けモバイルサービスで認証を開始する（C＃）
  • Windowsストア用モバイルサービス（JavaScript）で認証を開始する
  • iOS向けモバイルサービスで認証を開始する


モバイルサービス内でのWindows Azureテーブル、ブロブ、およびサービスバスの使用



サービスの更新により、開発者はモバイルサービススクリプト内の他のクラウドプラットフォームサービスへの呼び出しを使用する機会が得られます：テーブルとブロブストレージツールとサービスバス統合ツール。



JavaScriptコードを使用すると、モバイルアプリケーション用のWindows Azureのストレージ機能と統合を簡単に使用できます。







これらの機能の使用の詳細については、 次のセクションで説明します。



メールとSMSメッセージを送信する



この更新プログラムは、モバイルサービスサーバースクリプトからプラットフォーム自体のクラウドサービスを使用することに加えて、メールメッセージ（ SendGridを使用）およびSMSメッセージを送信できる機能を追加します。



1か月あたり最大25,000件のメールメッセージを無料で送信できます 。







この記事から、電子メールメッセージを送信する機能の詳細を学ぶことができます。



電子メールメッセージの送信と同様に、更新プログラムにはSMS通知を送信する機能があります。 これを行うには、Windows Azure 開発者に 1000件の無料メッセージを提供するTwilioサービスを使用します。







米国西部でのモバイルサービスの可用性



Windows Azureモバイルサービスはプレビュー段階にあることを思い出させてください。つまり、データセンターでの可用性は限られています。 サービスが開始されたとき、それらは1つの米国東部地域からのみ利用可能でした。 更新により、開発者は別の地域（米国西部）を選択できるようになりました。 サービスの最終バージョンのリリースにより、それらはすべての地域で利用可能になります。



Windows Azureトレーニングキットの更新-2012年10月



開発者は、Windows Azureに関連する別のニュースを歓迎します。 Windows Azureトレーニングキットの包括的なガイドと一連のラボ作業が更新されました。これにより、クラウドプラットフォーム用のアプリケーション開発のあらゆる側面を実際に学ぶことができます。



新しいバージョンでは、7つの新しいデモが追加され、22のプレゼンテーションがVisual Studio 2012およびWindows 8で動作するように更新されました。
