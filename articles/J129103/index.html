モスクワでのOperaスタッフとのミーティング

現在のブラウザ開発は、多大な時間と労力を必要とする複雑なプロセスです。 しかし、このような忙しいスケジュールでも、機会があればすぐにOperaユーザーとのミーティングを手配するように常に心がけています。 実際、このようなイベントはブラウザー開発プロセスの一部でもあり、最も重要なイベントの1つです。ソフトウェア製品に関する直接的なフィードバックを受け取ることによってのみ、ユーザーのニーズと期待に最も完全に応えることができます。



したがって、ユーザーとの次回の会議が9月29日にモスクワで開催されることをお知らせいたします。 会社の主要な専門家に質問したり、最も重要な問題について話したり、運が良ければ、Operaブラウザの将来のバージョンで期待されることの秘密を見つけたりすることができます。 ただし、より正確には、モスクワに来る同社の従業員の専門性により、主にOperaのデスクトップバージョンについて話すことができます。 すなわち：



Ian Standal-副社長、開発責任者、Opera Desktop

HanneNorström-プロダクトマネージャー、Opera Desktop

Knut-Jorgen Rieshog-リードユーザーインターフェースエンジニア、Opera Desktop

Alexandra Davydova-ロシアおよびCISのパートナーリレーションマネージャー



また、会議にはロシアのOpera Softwareのオフィスの従業員が出席します。 ご覧のとおり、委任は非常に印象的であり、会議に参加するすべての人に最大限の注意を払うことができます。 だから、私たちはみんなにチャットに招待します：
