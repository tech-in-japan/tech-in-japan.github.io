重要！ WordPress 2.5.1-システムセキュリティの修正

2.5.1システムの新しいバージョンが公式のwordpress.org Webサイトに登場しました。 このリリースには、いくつかのバグ修正、パフォーマンスの改善、およびシステムセキュリティに関する非常に重要な修正が含まれています。 作成者は、特にオープンユーザー登録が利用可能な場合は、誰でもすぐにブログを更新することを推奨しています。 この脆弱性はまだ公表されていませんが、近い将来に情報が表示されますセキュリティホールに加えて、WordPreee 2.5.1には多くのバグ修正が含まれています。 システムのセキュリティの向上のみに関心がある場合は、次のファイルwp-includes / pluggable.php 、 wp-admin / includes / media.php 、およびwp-admin / media.phpをダウンロードして 、既存のファイルを置き換えます。 リリース2.5.1全体をインストールすると、70を超えるバグの修正が提供されます。 このバージョンは、最も厄介なバグとパフォーマンスの改善に焦点を当てています。 ハイライトは次のとおりです。
  • ダッシュボード、および投稿の書き込みページとコメントの編集ページのパフォーマンスが向上しました。
  • 多くのカテゴリを持つサイトで最高のパフォーマンス。
  • 「メディアアップローダー」の修正。
  • TinyMCE 3.0.7へのエディターの更新。
  • ウィジェット管理の修正。
  • さまざまなユーザビリティの改善。
  • IEのテンプレート修正。
西洋のブログの1つに書かれているように、「2.5.xブランチにまだアップデートしていないのであれば、今がその時かもしれません。」 個人的に、私はまだいくつかのセキュリティ修正を施した古い2.3.1に取り組んでいます。 新しいバージョンに切り替えるのに十分な空き時間がありません。 すでにバージョン2.5.0を取得できたすべての人は、 緊急にその安全性に注意する必要があります！ -Tod のブログ リリースから 2.5.1 。を ダウンロードし ます。 インターネットの私のビジョン。
