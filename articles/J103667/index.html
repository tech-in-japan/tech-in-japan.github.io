インターネットは若いテクノロジーです。1989年にWWWサービスが開始されてからの年齢を数えると、インターネットは21歳にすぎません。 人間が作成した他のテクノロジーと同様に、グローバルネットワークは、その目的（公式にはこれは迅速な情報交換です）に加えて、損害をもたらす可能性があります。 これは、コンピューターを破壊するウイルスに関するものではなく、MPHを増やす提案に関するものでもありません。 インターネットは非常に便利なものであり、実際の生活とますます融合しており、一般的に「仮想」と言うことはできません。 仮想現実に没頭しているSFの世界では、現実とは独立した自分の世界がありますが、ここではすべてが異なります。インターネットは情報を送信するためのシステムにすぎません。 しかし、何らかの理由で、ほとんどの人はそうは思わず、インターネット上で起こっていることは彼らの生活に影響を与えないと思われます。 しかし、過去20年間で世界は異なってきました。今日、27時間でロシアからアメリカに行くことができますが、チケットの価格は手に入らないものではありません。 このビデオでルイスCK は言った。

サーバーは米国にあります。つまり、fox.com、hulu.com、last.fmなどの以前に閉鎖されたサイトにアクセスできます。 彼らはドイツでサーバーを取得することができなかったため、リソースからの応答が届くまでに信号が世界中を2.5回（ロシア>アメリカ>ロシア>アメリカ>ロシア）移動する必要があるため、ロシアのリソースの速度は理想からはほど遠いでしょう。 速度を上げるため、ドイツのサーバーはベータ版以降も動作します。 チャネルは1ギガビットのみであるため、参加者の数は制限されています。 Mac OS XとLinuxの所有者は大歓迎です。 残念ながら、これまでのところLinux用にコンパイルされたのはdebパッケージのみであるため、Ubuntuおよび他のDebianベースのディストリビューションでのみ機能します。 Mac OS Xの場合、元のインストールディスクにあるX11をインストールする必要があります。



このサービスは、外国のウェブサイト（ドイツ、アメリカ、イギリスの3か国が想定されています）にアクセスしたい人や、ネットワーク上の動きに関する情報に誰もアクセスさせたくない人に適しています。 VPNは、Wi-Fi経由で接続する場合の保護にも最適であり、暗号化はスニッフィングを無用にします。 特にモバイルデバイスの場合、PPTP VPNは、匿名性のためではなく、「盗聴」トラフィックから保護するために機能します。
