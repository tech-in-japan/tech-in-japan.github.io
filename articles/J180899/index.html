犬を振る尾のよう。 プロパガンダのABC

情報にアクセスしやすいほど、情報は多くなります。 拡散が速いほど、チェックする時間が少なくなります。 徐々に、情報環境は一種の「ホワイトノイズ」に変わります。 過剰なものを取り除くために、自分の内部に新しい情報フィルターを構築することがますます難しくなっています。スローガン、動揺、プロパガンダを取り除き、実際に背後にあるものだけを残します。 そして彼らは常に私たちの考えを操作しようとしています、そして私はフレーム25について話しているのではなく（それは機能しません）、より地上的なことについてです-政治家、広告主、そして一般的に、すべてが非常に巧妙に（そしてしばしば非常に不器用に）怠け者ではありません。 これらの手法については、記事で説明します。



尾が犬を振る。 （「 犬を振る 」）-大きなスキャンダルや「騒ぎ立てる」ことを避けるために、重要であるが不快なイベントに気付かれないようにするために、英語を話す政治戦略家は「犬を振る」と呼び、ロシア語を話す「尾が犬を振る。」 それは、二次的な重要性の問題について、次のより重要なイベントやアクションが気付かれずに通過する背景に対して、議論の波を時間内に上げることにあります。

そのような問題の例：冬時間への移行の取り消し、ロシアのデパルデュー、ロシアのラダカリーナへのプーチンまたはサーシャグレイの旅行、シベリアクレーンとのフライトなど

この技術は、犬を振る英語の表現の尾に敬意を表してその名前を得ました。これは、次の民俗の一節から現れました：

-なぜ犬は尻尾を振るのですか？

「犬は尾よりも賢いからです。」 尾が賢い場合、彼は犬を振るだろう。




一般的な馬車の 効果 （ バンドワゴン効果 ）-「オーケストラのある馬車」、「一般的なプラットフォーム」とも呼ばれます。 このテクニックは、行動の全会一致または均一性を必要とするフレーズを選択し、誰もがそれを行うという自信を作ります。「普通の人はそれを言うだろう...」、「正気な人はそれを知っている...」、「すべてのロシア人はそれを見る... "、"高等教育を受けた人々がHabréに座っている、あなたはそれを理解するべきである... "など。 この手法により、人は、自分が関係している、または関係したい、または自分にとって重要な意見を持っている社会グループのほとんどのメンバーがそのように考えていると感じ、そうでなければ、特定の価値観、アイデアを持ち、1つの視点を共有していると感じます。



「Labeling」 （ Name Calling ）-テレビ、モスクワのエコー、LJ Navalnyのコメント、および日常生活で見ることができるシンプルなテクニック。 それは、攻撃的な比phor、名前、軽rog的な表現、特定の人格の名前、人々のカテゴリー、現象を選択することにあります。 最近の時代の鮮明な例：自由主義志向の人々に関する「バルクハムスター」、「クリーチャー」、「リベラスト」、現在の政府の支持者に関する「ナシ」、「クレムレボティ」、民族主義者に関する「ファシスト」、「ナチシキ」。 意識的な歪みと名前の歪み：「楕円形/肛門」、「パトラー」など この手法を使用すると、比較的簡単に信用を失い、グループ全体または個人に対して敵意を抱きます。



きらびやかな一般性は、以前のトリックのほぼ正反対です。 それは、社会現象、観念、または人々のグループの名前を、よりポジティブな色のより一般的なものに置き換えることにあります。 この手法は、「平和」、「自由」、「幸福」、「愛国心」、「民主主義」などの言葉に関連する人々の肯定的な感情を活用します。 この技術の助けを借りて、よく知られている現象は、特定の人またはグループに固有の特性、品質、態度に起因しています。 実例は、「正統派」の概念をより一般的で明らかに肯定的な言葉「信仰」、ロシアの「エネルギー超大国」、または逆に「原材料付属物」などの定義に置き換えたものです。



「当局への言及」 （ 証言 ）-広範囲に及ぶ手法は、高い権限を持つ人格を引用すること、またはその逆であり、対象の聴衆から否定的な反応を引き起こします。 声明には、人、アイデア、組織についての価値判断、明確な判断または承認を含める必要があります。 大統領選挙の前に、文脈からしばしば引用された古典の引用、テロリストのドク・ウマロフのビデオアドレスを持ってくる（ところで、ここにラベルがあります-このウマロフが達成しようとしていることと彼が戦っているものを誰も知りませんが、誰もが彼がテロリストであることを知っています）共産主義者との会話でレーニンを引用するなど、与党の不承認



「一般の人々のゲーム」 （ 平凡な人々 ）-対象者と同じ、人々からの人、単純な素人としての個人のプレゼンテーション。 人の伝記を使用する（またはゼロから作成する）ことも、視聴者固有のスラングを使用することもできます。 例としては、忘れられない「トイレに浸る」、イリイチの幼少期に関する話、ウラルバゴンザヴォードの労働者の使節としての任命、チェチェン戦争のベテランであるセルゲイ・マスレニツァに関する最近のカモなどがあります。 このテクニックの助けを借りて、対象の聴衆は、オブジェクトに対する信頼を形成します。これは、「友人または敵」のレベルでの前向きな姿勢です。



「 カードスタッキング 」-この方法は、人または現象の肯定的な側面、またはその逆、排他的な否定的な側面のみが発表され、他の側面は無視される、事実の一方的で優しい表現で構成されます。 「ナヴァリニーは森を盗んだ」、「リモノフはファゴット」、「スターリンの下ではそうではなかった」、あるいは逆に「ブラッディスターリン主義政権」はこの手法の例です。 レセプションの主な目的は、事実の一方的な選択を使用して、魅力や、反対に、あらゆる視点やアイデアが受け入れられないことを示すことです。



「Transfer」 （ Transferは信じられないでしょう）は、ほとんどの人にとって巧みに、目立たず、目立たないようにする技術であり、コミュニケーションのソースが彼らに与えるものに対する権威と名声を広めます。 レセプションの助けを借りて、価値と意義のあるものとのオブジェクトの関連関係が開始されます。 たとえば、現在の政府への愛国心の忠誠心を呼び、国の誇らしげに飛んでいる旗を背景に大統領を排除すること（オリンピックチャンピオンなど）など。



おわりに
