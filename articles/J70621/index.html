Google Sites Opened API

Google Sites wikiサイトビルダーの開発チームは、標準化されたGoogle Data APIの一部となる新しいAPIのリリースを発表しました。 これらのインターフェースを使用すると、サードパーティのプログラムを含むGoogleサイトのコンテンツをインポート、エクスポート、編集、更新できるようになります。



まず第一に、これは法人顧客を幸せにするはずです。 現れた機会を利用し始めるのは彼らです。 以下に例を示します。

  • CRMシステムからGoogleサイトを更新します。
  • Microsoft SharePointやLotus Notesなどのエンタープライズシステムからファイルとコンテンツをインポートします。
  • Googleサイトのページをオフライン編集用にエクスポートし、更新されたコンテンツを再インポートします。
  • 変更の全履歴を維持しながら、バックアップのためにサイトを完全にエクスポートします。


すべてのGoogleサイトのコンテンツは標準のGoogleデータプロトコルを介して利用できるため、機能のリストは続きます。
