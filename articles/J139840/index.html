MTSバックグラウンド-Streamの顧客を犠牲にしてWi-Fiネットワーク

Stream（最近は「Home Internet MTS」に改名）、Comstar、「Internet from MGTS」、またはSistema-Telecom CJSCが所有するその他のブランドのクライアントであれば、隣人や通行人がインターネットアクセスを使用していることを知っておく必要があります。 絶対に、さらに、合法的に。



カットの下の詳細。







それはすべて、思いがけず自分にとって、MGTSホームインターネットのクライアントになったという事実から始まりました。 私たちは一時的に妹のアパートに引っ越しました。そこでは他のインターネットはまったくありませんでした。



私はすぐに言います：インターネットへのアクセスは私にとって不可欠です。 私のパートナーと私は、米国で複数のSaaSアプリケーションを回転させ、同じ場所（githubとbitbucket）でソースコードをホストし、1日に数回バージョンを更新します。サイト、サーバー、データベース、Amazon Shmamazonなど。 あなたはそれを理解しています。



そのため、速度はカテゴリに当てはまりませんでした。 安定性も。 バトルフィールド3をプレイして作業し、コーディングすることは完全に不可能でした。 数多くの実りのないアピールとサポートサービスとの協議の後、私は辞任しました...



しかし、ある日突然ルーターを再起動すると、MTS / FON WiFiネットワーク（明らかに私のものではありません-MTS会社のWiFiです！そして、近隣のネットワークでは不十分です...） 、ルーターの再起動と同時に消えて表示されます。 簡単な操作で、このSSIDは実際に私のルーターからのものであることがわかりました。 さらに、無料で許可なしで。



このネットワークを切断するための数多くの試み、少なくともパスワードを割り当てる（モデムの再起動後にすべてリセットされた）、MTSサポートサービスへの呼び出し、 仲間の被害者からの多くの苦情などについては説明しません。 ポイントに直行します。



あれは何だった



したがって、「公式」Wi-Fiルーター（つまり、Stream / MTS / MGTS / Comstar salonで、写真のように会社のロゴなど）を購入した場合、このデバイスは2つのWi-Fiネットワークを作成します。 1-あなたの家。 第二-彼は誰にでも配布する「MTS / FON」という名前で。



このネットワークでは、約1〜1.5 Mbitがチャンネルに予約されています（有料、注意！）。



MTS-FONワイヤレスネットワークのアクセスポイントになります。



このネットワークについては、MTS Webサイトで読むことができます。 つまり、世界中の無料ネットワーク「FON」（ Wikipediaの説明を参照 ）の一部であり、ユーザーはネットワークへのアクセスを自発的に互いに共有します。 あなたのアパートからこのネットワークを配布することにより、あなたは世界のどこでもFONネットワークへのアクセスを得ていると思われます。



しかし、ここでのキーワードは「任意」です。 この機能について購入する場合は警告されません。 ルーターを再起動すると、ルーターの構成がプロバイダーのサーバーから再び取得されるため、経験の浅いユーザーは、ルーターでこの機能を無効にすることはできません。 「背景」に関する質問に答えるサポートスタッフは、わかりやすいものには答えられません。 そして、彼らは意図的に答えを残しているようです。



そして最も不愉快なことは、コミュニケーションの質がこの「背景」に悩まされていることです（多くの人々がStreamフォーラムでこれについてうめき声を上げています）。 結局のところ、「ブランド」ルーターは、2つのWiFiネットワークを単純にプルしない安価な中国製デバイスに基づいて作られています。 言うまでもなく、控えめに言っても、正当な10 Mbpsのうち、1.5メガビットが「サイドに」与えられていることは気分を害します。 通知も無効化もできません。



どうする



  1. まず第一に、プロバイダーからブランド変更されたモデムを購入しないでください。
  2. アパートに「MTS / FON」というネットワークがあるかどうかを確認します。
  3. 見つかった場合は、モデム/ルーターの設定（192.168.1.1など）に移動します。
  4. ルーターの設定（正確にはモデルによって異なりますが、ほとんどの場合、管理者パスワードを設定した場所と同じ場所にあります）で、「ACS URL」フィールドを見つけます。 これは、ルーターがプロバイダー構成を取得するアドレスです。 私のモデムでは、 d.pr / f7RVのように見えます。
  5. このフィールドの値をどこかに保存します
  6. このフィールドをクリアして、[保存]をクリックします。
  7. その後のみ、ルーターの設定でFONをオフにします。 正確に-繰り返しますが、それはモデルに依存します。
