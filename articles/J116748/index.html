Motorola Milestone 2の概要

Milestone 2（別名Droid 2）は、間違いなく最も美しくエレガントなAndroidスマートフォンの1つです。



このデバイスは、タッチコントロールを備えたサイドスライダーのフォームファクターで作られており、主に物理的なキーボードを手放したくない人に主に向けられています。 マイルストーン2は、HTC Desire Zの主要な競合他社の1つと見なすことができます。

そしてもう少し。







デバイスの仕様







携帯電話は非常に持ちやすく、寸法は116mm x 61mm x 14mmで、重量は169グラムです。



オペレーティングシステム： Android 2.2（Froyo）+ MOTOBLURインターフェイス

通信： GSM 850/900/1800/1900 MHz、UMTS 900/2100 MHz、GPRS / EDGE、HSDPA（最大7.2 Mbps）

プラットフォーム： Texas Instruments OMAP3640、1 GHz

メモリ： 512 MB RAM、8 GBの内部メモリ、microSDスロット（最大32 GB）、8 GBカードが含まれています

インターフェース： USB 2.0（micro USB）、Bluetooth 2.1（+ EDR）、Wi-Fi 802.11b / g / n、GPS / A-GPS

画面： TFT（容量性センサー）、対角3.7インチ、解像度854x480、1600万色

カメラ： 5 MP、オートフォーカス、フラッシュ、HDビデオ録画

センサー：空間内の位置センサー、光センサー、デジタルコンパス

バッテリー：リチウムイオン、取り外し可能、1400 mAh

寸法： 116.3 x 60.5 x 13.7 mm

重さ： 169g



詳細については、 Motorolaの Webサイトをご覧ください。





何が入っていますか？









パッケージは次のとおりです。



-コミュニケーター

-USBケーブル

-8GB MicroSDHCカード

-アダプター付き充電器

-ヘッドフォン

-紙







初めて電源を入れると、携帯電話からMotoBlurアカウントをセットアップするように求められます。 主な重点は、ソーシャルネットワーク（Gmail、Twitter、Facebook、MySpace、Bebo、LinkedIn、Last.FM、Picasa、Flickrなど）との統合に置かれています。



MotoBlurの主な欠点は、それを拒否できないことです。 あなたは単にそれを見逃すことはできませんし、MotoBlurを使わずにメールを使うこともできません。

MotoBlurのもう1つの問題は速度です。 このデバイスには強力な1GHzプロセッサが搭載されているように見えますが、画面のロックを解除するなど、単純な処理が時々遅くなることがあります。



外観





最初に目を引くのは、明るい3.7インチディスプレイ（AMOLEDではありません）で、日光で色あせません。 視野角は優れています。



ディスプレイの縁は「クローム」に、側壁は金属製で、背面カバーと同様にダークブルーの「メタリック」に塗装されています。 ケースは定性的に作られており、どこにもきしみません。











ディスプレイの下には、Androidプラットフォームに典型的な4つのタッチボタンがあります。 検索、リターン、メニュー、ホームキー。 彼らは非常に敏感です。







ディスプレイの上には、近接センサーと光センサーに囲まれたスピーカーがあります。







上部には3.5 mmヘッドフォンジャックと、画面のオン/オフ/ロック解除を行うボタンがあります。







デバイスの背面はゴム引きされています。 こちらがスピーカー、カメラ、フラッシュです。











オートフォーカスとデュアルLEDフラッシュ、4倍デジタルズーム、24〜30フレーム/秒のHD 720pビデオ録画を備えた5メガピクセルのカメラ。

午後は良い写真を撮りますが、暗闇の中で思い出さない方がいいです。











第一世代のマイルストーンの裏側、変更はマイナーです。





キーボード



テキストの入力は一般に便利です。

最初は気に入らなかったが、数日後、私はそれに慣れて、実際に画面の代わりにそれを好む。 キーボードはバックライト付きで、暗闇でも入力が可能です。

議論の余地のないマイナスは、ロシアのレイアウトの欠如です。
