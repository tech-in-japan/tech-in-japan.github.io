Creative Commonsの下でライセンスされたAmazonのベストセラーDeath Machine

これは、以前は音楽アルバムで起こりました。オープンアクセスの作品を出すことは、現在の商品の販売に悪影響を与えなかったからです。 Death Machineの作者はこれを明確に理解していました。 彼らは、 CC BY-NC-NDネットワークで、自分の死を知っている（しかしいつは知らない）人に関するSF小説のアンソロジーが利用可能になった理由を説明しました。



なぜこれを行うのですか？ この本の販売を傷つけるのではないかと心配していませんか？



一言で言えば、いいえ。 あなたは、貴重だと思うコンテンツに対して喜んで支払いをすることを何度も証明しています。 恋に落ちた資料を共有する準備ができていることを示しました。 また、本を購入する機会があるかどうかに関係なく、あなたが私たちの仕事に精通できるよう努めています。 Amazonが配達される国に住んでいるかどうかに関係なく; あなたの人生に紙の山がある場所に関係なく。



ダウンロードして、読んで、コピーして楽しんでください！



さらに、一部の選択されたストーリーはCC BY-NC-SAの下でライセンスされています。 、非営利の条件に準拠し、同じライセンスの下で派生物をリリースする限り、作品を翻訳および適合させることができます。 ポッドキャストはすべてのストーリーに対しても作成され、現在3つのテキストが用意されています （ YasonByとal_oneの翻訳の助けに感謝します ）。



現在、Death MachineはAmazonのナンバー1のSFアンソロジーであり、2010年のBest Booksにもリストされています。 詳細については、 Boing Boingおよびthe Death Machine Webサイトを参照してください。
