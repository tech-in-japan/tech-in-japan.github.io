Blackberry Playbook用のAndroidプレーヤーを確認する





設置



  1. 開発者モードを有効にする
  2. DDPb Installer 1.0.7.0をダウンロードします（インストールエラーはスキップできます）。Blackberryの開発経験がある場合は、blackberry-deployを使用してすぐにインストールできます。
  3. sys.android.barファイルを起動して指定した後
  4. 利益




Androidアプリケーションをインストールする



  1. DDPb Installer 1.0.7.0を使用するか、adb installlを介して
  2. DDPb経由でapkファイルを選択して入力し、adb経由でadb接続IPデバイスに接続する必要があります（Android Playerが開発者モードになっていることを確認する必要があります）。次に、adb install path_to.apk
  3. 次に、ファイルマネージャーを使用して、Android自体からインストーラーを実行できます。




Androidバージョン情報



あなたが写真で見ることができるように、バージョン2.3.3があります

これは、Android Playerの未加工バージョンであり、すべての問題があります。

システム自体の速度が大幅に低下し、一部のアプリケーションが起動せず、グラフィックが時々アーティファクトを生成し、カメラにアクセスできず、興味深い不具合が多数発生します。







しかし！ 不足しているプログラムをPlaybookに配置できます。

スカイプ、ツイッター、ジャバークライアント、メールクライアントなど、これはもちろん、起動した場合はアンドロイドです。

不安定な動作、デバイスの加熱、アプリケーションの起動における部分的な制限にもかかわらず、AndroidプレーヤーはPlaybookの使用範囲を大幅に拡大できます。
