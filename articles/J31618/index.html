Ubuntu MID Edition 8.04ショートレビュー

インターネットタブレット用のUbuntuの新しいバージョンに関する記事の無料翻訳をご紹介します。



Ubuntu 8.04のリリースから2か月後、 Ubuntu Notebook Remixの最初のプレゼンテーションから1か月後、 Ubuntu MID Edition 8.04がついにリリースされました。 これは、インターネットタブレット（MID）専用に設計されたUbuntuファミリの新しいCanonical製品です。 Ubuntu MID Editionは、デスクトップマシン向けのUbuntu 8.04のわずかに切り捨てられたバージョンです。ほとんどのソフトウェアパッケージは、Intel Atomプロセッサとモバイルペリフェラルで最高のパフォーマンスが得られるように特別に再設計されています。 これはMoblinプロジェクトのおかげで可能になりました。







特定の数のパッケージが変更、削除、追加され、インターネットタブレットまたはUMPCで最大のパフォーマンスが得られるようになりました。 たとえば、新しいプログラムの1つは、スケーラビリティを備え、小さな画面用に特別に設計されたGecko Webブラウザーでした。 「 MID Browser 」と呼ばれます。







パッケージには、インスタントメッセージング用のPidgin 、Webカメラで作業するためのCheese 、電子メールクライアントとしてのClaws Mail 、およびビデオとサウンドを再生するためのMoblin Mediaが含まれます。 NeverballやFrozen Bubbleなどのゲームも含まれています。 インターネットタブレットで本を読むために、Ubuntu MIDは標準パッケージの1つでFBReaderを提供します。 キットには、時計、電卓、アドレス帳など、多くの小さいながらも重要なアプリケーションも付属しています。









インターフェースは少し異なりますが、Ubuntu MIDはネットワーク接続にNetworkManagerを使用しますが、XServer 1.4.1プレリリースとLinux 2.6.24はシステムを内部から制御します。







Ubuntu MIDは、ユーザーインターフェイスパーツ、ブラウザー、マルチメディアシステム、アプレットなど、多くのMoblinプロジェクトを使用します。 このすべてのソフトウェアは、間違いなくIntel AtomおよびMenlowプラットフォーム向けに最適化されています。 MenlowベースのUMPCが今年1月にCESで発表されました。







Canonicalは、さまざまな目的のためにUbuntu MID Edition 8.04のさまざまなオプションを提供します。 たとえば、インターネットタブレットを使用せずにこのシステムを実際に試してみたい場合は、 KVM （カーネルベースの仮想マシン）を使用して特別なバージョンをダウンロードしてインストールできます。 同社はまた、McCaslinおよびMenlowプラットフォーム用の特別なアセンブリを提供しています。 Canonicleは、開発中にMcCaslinプラットフォームに基づいたSamsung O1Uを使用し、Menlowのサポートは後に登場しました。 ただし、表示される画像は、むしろエンドユーザー向けではなく、メーカーと開発者向けです。 どうやら、サポートされていないMenlowデバイスでシステムを動作させるには、大幅な変更が必要になるようです。 これまでのところ、最近発表されたNVIDIA TegraまたはVIA Nanoをサポートするものは発表されていません。 Ubuntu MID 8.04のKVMバージョンは471メガバイトです。







Ubuntu MID Edition 8.04は、 Canonicalサーバーからダウンロードできます 。



ディストリビューションを作成する際にデスクトップバージョンから開始したため、同じNokia Maemoのように、プログラムの数に大きな問題はないはずだという事実に興味があることを自分で付け加えます。 ただし、Nokiaのインターフェイスの視覚的なテーマは、はるかに考え抜かれているようです。上記のUbuntu MIDのスクリーンショットでは、インターフェイスは子供っぽくて扱いにくいように見えます。 それとは別に、使用するフォントに鼻を向けます。 だから、事業は良いが、私見、多くが損なわれる必要があります。
