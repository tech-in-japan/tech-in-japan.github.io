例を考えてみましょう： 

 { foo(); // ! function foo() { // .. } } foo(); // ReferenceError


この例では、関数はブロックスコープで宣言されているため、外部からアクセスできません。 また、興味深い事実に注目する価値があります。このような変数が初期化される前に操作できないlet
