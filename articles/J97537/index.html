Toshiba Satellite L500-1WPの概要





少し前（つまり、3日前）、東芝はラップトップにテストを提供する準備ができていると発表しました（ proflink ）。 レビューを書いてみることにしました。 もちろん、使用する時間はほとんどありませんでしたが、その長所と短所をすべて見つけようとしました。



UPD：仕様表を修正しました



ラップトップはこのような長い箱に入っていましたが、結局のところ、デバイス自体は通常のサイズで、側面にはワイヤーの入った箱がありました。







少し乾いた技術仕様：







外観









外側は、MacBookをsomewhatとさせます：同じ金属のマット仕上げです。 しかし、もちろん、それらは非常に異なっています。 一般に、デザインは非常に優れています。繰り返しますが、コミュニティにとって退屈な光沢はほとんどありません。 一般に、一部のデザイン要素は一般ユーザーとして私を怖がらせます。 ラップトップを開く前に最初に驚いたのは、ファンクションキーではなく、フロントパネルのリレー（！）を使用してサウンドを調整することでした。 非常に予期しない解決策ですが、原則として、操作中に苦情はなく、適度に便利です。







蓋には会社の名前が書かれており、蓋自体はラッチなしで開きますデバイスを開いた後、タッチパッドの幅よりも大きい巨大なタッチパッドボタンが目を引きました。 しかし、この欠点は、サテライトL500にフルマルチタッチ（2本および3本の指、スクロール、回転、ズーム）があるという事実によって補われています。







また、組み込みのスピーカーの不器用な外観と電源オン/オフボタンがやや不快に見えました。 他のすべての詳細にもかかわらず、それはどういうわけか時代遅れに見えます。











キーボードはまた、その異常性にも驚かされます。 また、Toshiba Satellite L500には開発者に多大な費用がかかりますが、個別のテンキーがあります。 ただし、そのため、一部のボタンはトリミングされます。 「スペース」は、ネットブックのように非常に小さくなりましたが、時間とともにすべてに慣れることができます。







ちなみに、押されたCaps LockキーとNum Lockキーの表示は非常に面白くなります。 Capsでは、LEDがキーに直接組み込まれています。 A







そしてNumでは、その上で点灯します。 また、Scroll Lockボタンは存在しますが、表示されません。







その中に存在するコネクタに注目する価値があると思いますが、それらのすべてが各ラップトップモデルに固有のものではありません。 右側には、2つのUSBポート、ケンジントン、充電ポート、DVDドライブがあります。 前面パネルは、音量調節とカードリーダー（SD / MMC、MS）、およびインジケータ（電源、作業、バッテリー、HDD、メモリカード、ワイヤレス接続）のみを備えています。 ただし、左側には、従来のVGA、イーサネット、マイク、ヘッドフォンジャックに加えて、HDMIとeSATA + USB（デュアルポート）があります。 ワイヤレスには、通信に十分なBluetoothとWi-Fiがあります。 他のすべてでは、おそらくこれは普通のラップトップです。











画面の上にはWebカメラののぞき穴があり、非常に便利です。 顔に直接「見える」（一部のモデルではそうではない-置き換えられる）。 カメラを操作するためのプリインストールプログラム。カーソルを画面の左側に移動すると、そのメニューが表示されます。 ここで、フレームを作成したり、明るさ、コントラスト、シャープネスなどを調整したりできます。一般的に、すべてはすべてのラップトップと同じです。 ところで、カメラの隣には、ある種のバックライトとして機能する小型の青色LEDがあります。







もちろん、カメラはあまりよく撮れませんが、何かを見るのは本物です。 おおよその品質は次のとおりです。







しかし、私たちは外部の魅力にこだわるのではなく、いわば内部に目を向けます。



鉄とソフトウェア









もちろん、マイナスの1つはラップトップの重量と呼ばれます。これは既に2.94 kgであり、これは15.6インチの画面を備えているためです。 しかし、私の主観的な意見では、これは大きなマイナスではありません。ラップトップはスペースを節約するために設計されており、頻繁に持ち運ぶことはないからです。 そして、彼は約3時間のバッテリー寿命を持っているので、彼と一緒に一日中屋外で過ごすことはうまくいきません。 :)







内部の充填物から判断すると、これはクールなゲーム用ラップトップとミッドレンジデバイスのクロスです。 結局のところ、Intel Core i3と4GBのRAMとATI Radeon HD 5145がここにあります。だから、職場でも自宅でも、スカーフやサナダムシだけでなく、ラップトップに空のお金をかけずに、より要求の厳しいゲームをプレイできます。



しかし、根拠がないように、いくつかのテストを実施し、その結果をレビューに提示しました。



初めて電源を入れると、既にインストールされているWindows 7が構成され、2〜5分後に、独自のデスクトップスクリーンセーバーが表示されます。 ところで、インストール中に、32ビットまたは64ビットのバージョンを選択できます。 ご覧のとおり、McAfee、Skype、eBay、60日間バージョンのMicrosoft Officeがすぐにインストールされます。







Fnボタンを押すと、ツールチップメニューがポップアップします。これは、キー自体のステッカーによっても複製されます。







システムには、ラップトップでの作業に役立つプログラムが既にインストールされています。 たとえば、Toshiba Assistは、多くの機能を理解して構成できるようにします。







バックグラウンドでラップトップの動作を監視し、誤動作の場合に警告を発するTemproプログラムと同様に。 はい、多くの場合、組み込みソフトウェアは迷惑ですが、ここでは本当に必要なプログラムです。







3dMarkは約4000ポイントを与えます。 ほぼすべての最新のゲームをプレイできるため、これは非常に優れています。 NFSでテスト済み：ShiftおよびGTA IV。 十分な設定です。最大設定であっても、ほとんど速度が低下せず、良好なグラフィックが表示されます。







このノートブックには4GbのRAM上のダイがありますが、別のものをインストールすることは可能です。 320Gbハードドライブ。







Windows 7システム自体によると（1から7.9ポイントまで）。 評価は最小値です。







組み込みのサウンドはあまりありません。 少しガタガタしますが、音量は小さいですがステレオです。 ラップトップの場合-見るのはロックオペラではありませんが、おもちゃやYouTubeの場合はそれで十分です。



冷却とノイズに関して：ラップトップにはいくつかのクーラーが取り付けられていますが、ノイズは発生しません。 さらに、ラップトップがテーブルの上にある場合、ノイズはまったくありません。 それがひざまずくと、もちろんプロセッサはより強く熱くなり、クーラーは加速しますが、それでもノイズはわずかで、同じDell Inspiron 1525よりもはるかに小さくなります。



ラップトップの価格は約28,000 pです。 これは、私には思えますが、それ以上のものです。 かつて、私は同じお金で、統合ビデオカード、2GBのRAMを搭載したラップトップを、一般に同じサイズで、特性の面で取りました。 一般に、このクラスのラップトップには価格が十分すぎるほどです。
