フリーウェアはあまり起こりませんか？



私たちはこの質問を肯定的に考えて答えました。したがって、管理者が日常業務で役立つ可能性のある無料プログラムの選択を評価することを提案します。 これらのプログラムはすべて、監査変更、アクセス制御、システム監視、およびセキュリティのためにNetWrixで開発されました。



したがって、以下はシステム管理者の作業を簡素化するのに役立つ10の無料プログラムのリストです。 ダウンロードしてお楽しみください！







1. Active Directory Change Reporter ：NetWrix Active Directory Change Reporterは、 Active Directoryを監査するように設計されています。 この製品は、ADおよびグループポリシーで行われたすべての変更を監視し、すべての変更に対して変更されたWHO、WHAT、WHENおよびWHATに関する情報を反映するレポートとアラートを生成します。 このプログラムでは、レポート送信のスケジュールをカスタマイズできます。 レポートには、すべてのユーザー、グループ、コンピューター、組織単位、グループメンバーシップ、アクセス権、Active Directoryサイト、オブジェクトおよびグループポリシー構成、ADスキーマ、およびその他すべてのタイプのオブジェクトの追加、削除、および変更が表示されます。

各変更に関連して、変更前のユーザーの以前の名前や、変更前のOUへのアクセス権のステータスなど、変更されたパラメーターごとに「変更前」および「変更後」の値が固定されます。 この製品を使用すると、現在の日付のADの「スナップショット」（「ドメイン管理者グループのすべてのメンバー」）または「スナップショット」を作成する特定の時点（「ドメイン管理者グループのすべてのメンバー」 2008年12月31日」

2. ファイルサーバー変更レポータ ：プログラムは、ファイル、フォルダ、およびアクセス許可に加えられた変更を記録し、作成および削除されたファイルを追跡します。 このプログラムは、たとえば、誤って削除されたファイルを記録したり、重要なファイルをすばやく復元したりするのに役立ちます。

ダウンロードする





3. USBブロッカー ：NetWrix USBブロッカーは、コンピューターのUSBポートに接続されたリムーバブルメディア（メモリカード、外部ハードドライブ、iPodなど）の不正使用を防止する集中アクセス制御を提供します。 ウイルス対策およびファイアウォール機能に関係なく、USBポートを介したアクセス制御は、ネットワーク上のコンピューターを保護する重要な側面です。 USBデバイスロックは、マルウェアからネットワークを保護し、企業の機密情報の盗難を防ぎます

このソリューションは、グループポリシーの組み込みメカニズムに基づいており、既存の環境に簡単に統合できます。

ダウンロードする





4. パスワードマネージャー ：NetWrixパスワードマネージャーを使用すると、ネットワークユーザーはシステム管理者なしでアカウントパスワードを安全に管理できます。 これにより、組織はActive Directory環境に強力なパスワードポリシーを実装し、情報セキュリティ要件への準拠を実現し、ID管理の問題を解決できます。

プログラムの主な機能：忘れたパスワードのリセット、アカウントロックアウトのトラブルシューティング、安全なWebインターフェイス、Windowsアプリケーションなどの両方を使用したアカウントの手動ブロック。 新しい無料版はWindowsログイン手順との統合をサポートし、Google Appsはロシア語で動作します。

ダウンロードする





5. Service Monitor- 重要なWindowsサービスを監視するための無料のツールで、障害後にオプションでサービスを再起動します。 このプログラムは、サーバーのグループで自動的に起動されたすべてのサービスを監視し、1つ以上のサーバーに突然障害が発生した場合、電子メール通知を送信します。 再起動機能により、すべての制御されたサービスがダウンタイムなしで動作することが保証されます。 軽量プログラムの構成は簡単です。インストール、コンピューター名の入力、目的の電子メールアドレスの指定。

ダウンロードする





6. ディスクスペースモニター ：最新のハードドライブのテラバイトボリュームにも関わらず、サーバーの空きスペースは、予測できないほど速く枯渇する傾向があります。 NetWrix Disk Space Monitorは、 サーバー （ドメインコントローラー、ファイルサーバー、SharePointおよびExchangeサーバー、データベースサーバーなど）のディスク容量を監視する ITプロフェッショナル向けの無料ツールです。 指定されたサーバーの空きディスク領域を集中的に監視することにより、ソリューションは、指定されたしきい値に達すると、ディスク容量不足に関する毎日のレポートと通知を送信します。 無料版は最大10台のサーバーをサポートします

7. 非所有者メールボックスアクセスレポータ ：無料のNetWrixツールの非所有者メールボックスアクセスレポータは、管理者および他のユーザーが外部メールボックスにアクセスできるすべての事実をキャプチャします。 誰が最後の日、月、年にどのメールボックスにアクセスしたか-ソリューションは正確なデータを提供し、過剰または誤って設定された権限を持つユーザーによる不正なアクションを認識できるようにします。 潜在的に、そのような活動は、取締役および企業幹部のメールボックスからの機密情報の盗難に関連付けられる可能性があります。 レポートを使用した外部メールボックスへのアクセスの監査は、企業ポリシーと情報セキュリティ要件への準拠を保証します。

8. イベントログマネージャー ：プログラムは、イベントログのデータを統合し、ネットワーク上のすべてのコンピューターのすべてのログからのすべてのイベントを反映するアーカイブレポートを通知および作成します。 SEIMプログラムは重要なイベントについて通知し、すべてのイベントを圧縮形式で一元的に保存し、アーカイブされたデータの分析を可能にします。 無料版は最大10台のサーバーをサポートします。

9. パスワード有効期限通知 ：

NetWrix Password Expiration Notifierは、リモートで作業しているユーザーを含むすべてのActive Directoryユーザーのパスワード変更を自動的に通知するように設計されています。 このプログラムは、特定の組織単位またはADドメインのすべてのユーザーを定期的にチェックし、特定の日数が経過するとパスワードの有効期限が切れるユーザーを検出し、カスタム通知を電子メールで送信します。

概要レポートは、システム管理者に電子メールで送信されます。

したがって、プロアクティブモードの管理者は、この製品を使用して、エンドユーザーとサービスアカウントのパスワードの期限切れに関する問題を解決します。



追加の計画とテスト機能を使用すると、構成を確認し、パスワードポリシーへの計画的な変更の影響を受けるユーザー数を確認できます。 これを行うには、ユーザーの注意をそらす必要も、AD設定を変更する必要もありません。
