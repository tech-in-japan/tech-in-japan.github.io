4G LTEインターネットメガホン

2012年5月15日、メガフォンはLTEテクノロジーを使用した4Gモバイルインターネットサービスの提供を開始しました： 4g.moscow.megafon.ru 。



Megafonにテスト用のモデムをくれと頼み、翌日にはそれを手に入れました。









モデムは巨大です。冗談で「ツァールモデム」と呼びました。







しかし、射撃しないツァーキャノンや、鳴らないツァーベルとは異なり、ツァーキャノンはうまく機能します。



モデムは新しい4Gネットワ​​ークと3Gおよび2Gネットワ​​ークの両方で機能するため、4G Yotaモデムとは異なり（私はそれと Yota LTEネットワークについて説明しました： ammo1.livejournal.com/252365.html ）、 4Gネットワ​​ークがない場所。 これは、モスクワ4Gネットワ​​ークがこれまでのところモスクワとその地域の特定の都市でしか機能しないことを考えると、非常に重要です（http://moscow.megafon.ru/help/coverage）。







3G / 2G-インターネットメガホンは携帯電話ネットワーク経由で、4Gはヨータネットワーク経由で提供されますが、4Gメガホンとヨータはまったく別のものです。 基地局は同じように使用されるため、カバレッジエリアは同じですが、インターネットの速度と品質は異なります。 これは、異なるインターネットチャネルが使用されるためです。 同じ基地局で異なる周波数を使用することは確認されていません。



Megafon 4Gモデムは、Megafonオフィスおよびオンラインストアで2,990ルーブルの価格で販売されています（http://moscow.shop.megafon.ru/modems_g__g_33_tree_992_1_61626.html）。

同時に、完全に無制限の4G / 3Gインターネットは2012年7月15日まで無料です。



7月15日から、3つの関税が開始されます。1か月あたり950、1200、1500ルーブル：







モデムに戻りましょう。 モデムはHuaweiによって製造され、E392と呼ばれます。







「64K」とマークされた特別なsimカードがモデムにインストールされています。







このSIMカードを別のモデムにインストールしてみました-3Gインターネットは正常に機能しました。 逆の手順も成功しました-他のSIMカードMegafonはこのモデムで正常に動作し、4Gネットワ​​ークのみが表示されませんでした。 他のオペレーターのsimカードは機能しませんでした-プログラムはメガホンsimカードをインストールするように要求しました。



モデムの反対側には、MicroSDメモリカード（カードリーダー機能）と2つのアンテナコネクタ用のスロットがあります。 どうやら、モデムには4Gと3G / 2Gの無線パスがあり、別々のアンテナを接続できます。







コンピューターに接続すると、モデムは仮想CD-ROMを作成し、その上にMegaphone-modemプログラムのインストーラーが配置されます（通常の3Gモデムの場合とまったく同じです）。 モデムドライバーは、このプログラムと共にインストールされます。

Linuxでのモデムの動作については、 habrahabr.ru / post / 144239 / ＃comment_4841134で説明しています。



Megafon 4Gモデムの速度をテストし、その動作をYotaモデムの動作と比較しました。 テストは午前1時に行われ、モデムは同じ場所、つまりSviblovoの17階の窓に置かれていました。 信号レベルは90％でした。



Megaphoneの最良の結果は、8.5 Mb / sのダウンロード、11.2 Mb / sのアップロードです。







平均的な結果はダウンロード8回、アップロード14回で、SpeedTestとinternet.yandex.ruによる速度は非常に安定しており、テストごとにほとんど変化しませんでした。



Yotaの最良の結果はほぼ同じでしたが、テストごとにダウンロード速度が3.5から9に跳ね上がりました。



2番目のテストは、高速サーバーから107メガバイトのファイルをダウンロードすることです。 テストは22:00に実行されました。

メガホン-128秒（833キロバイト/秒〜6.7 Mbps）

ヨータ-410秒（260キロバイト/秒〜2 Mbps）



トレントのダウンロードでメガホン4Gとヨータを比較することはできませんでした-ヨータはトレントのダウンロード速度を人為的に制限しているようです：速度は毎秒40キロバイトという非常に低い値で変動し、メガホンの速度は約800キロバイト/秒でした。



私は、4Gインターネットが窓ではなく部屋の奥でどのように機能するかを試しました。 同時に、信号電力は90％から30％に減少しました。







ご覧のとおり、ダウンロード速度は高いままで、アップロード速度は大幅に低下しました。



4Gをオフにし、同時に3G-Internet Megaphoneが私のアパートで動作する速度を確認しました。 最大速度4.05 / 0.62、平均2.0 / 0.5。



もちろん、私は、4Gインターネットがモスクワと最も近いモスクワ地域だけでなく機能することを本当に望んでいます。 Megafonからの情報によると、2012年末までに、モスクワ地方の次の都市に4Gネットワ​​ークを展開する予定です。 -ズエボ、クラスノゴルスク（rts）、セルギエフポサド（rts）、シェルコヴォ（rts）、プシュキノ（rts）、ジュコフスキー。



結論として、見通しについてのいくつかの言葉。 Yotaネットワークの使用に加えて、MegaFonはモスクワで4G周波数を所有していますが、連邦周波数セットの競争を要約した後にのみサービスを提供できます。 モスクワでは、MegaFon自身の周波数はTDD LTEであり、ヨタ周波数はFDD LTEです。 MegafonのLTEネットワークが立ち上がると、MegaFonのLTE加入者はTDDとLTE FDDの両方を使用できるようになり、独自のネットワークを立ち上げた後、モスクワのYotaとの協力は継続されます。



間違いなく、今日モスクワのメガフォンからのLTEの提供は、ヨータからよりもはるかに興味深いものです。 同じ価格で、速度はより速く、より安定しています。そして最も重要なことは、4Gインターネットが3Gと組み合わされていることです。3Gはほとんどどこにでもあります。
