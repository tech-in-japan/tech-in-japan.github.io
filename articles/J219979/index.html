Sensu Brush-静電容量ディスプレイ用の最初のブラシ

2012年のSensu Brushは、Kickstarterで65,000ドル以上を調達しました 。 私は長い間タッチスクリーン用のブラシを手に入れたいと思っていましたが、今ではついに私の手の中にあります。 この珍しいスタイラスについての印象を共有する時が来ました！



















ケースは金属製で、研磨された金属とマットブラックの2種類があります。 きちんとした箱の中にSensuを完成させるには、ブラシとブラシが隠されているフォームブロックを使用するための推奨事項を含む小さな指示があります。







現在、Sensu Soloのバージョンもあります。 それは普通の水彩ブラシのように見え、保護キャップを持たず、また古典的なスタイラスに変換することもできません。











まず、仕事の原理とSensu Brushの独自性について少しお話したいと思います。







画面の内側には、電極のグリッドがあります。 電極は人体と一緒にコンデンサを形成します。 電子機器は、このコンデンサの静電容量を測定します（電流パルスを与え、電圧を測定します）。



サムスンは、AMOLEDスクリーンのサブピクセル間に敏感な電極を直接取り付けることができました。これにより、設計が簡素化され、透明度が向上します。



このようなスクリーンの透明度は最大90％で、温度範囲は非常に広いです。 画面は非常に耐久性があります（ボトルネックはクリックを処理する複雑な電子機器です）。 汚染を起こさないスクリーンは反応せず、導電性スクリーンはソフトウェア手法により簡単に抑制されます。 したがって、投影型静電容量式タッチスクリーンはパーソナルエレクトロニクスで広く使用されており、その多くの種類がマルチタッチをサポートしています。



Sensu Brushは基本的に2つのスタイラスで構成されています：



  • 導電性ゴムスタイラス

    市場にはそのようなスタイラスがたくさんあります。 それらは安価で、耐久性があり、低温に強いと考えられています。 それらは指で画面に触れたような全体的な感覚を与え、そのような入力デバイスの触覚応答は最高と見なされ、その落ち込みの程度は簡単に調整できます。
  • ブラシスタイラス

    2011年の美容の高度な研究のおかげで、感覚的なブラシを作ることが可能になりました。 落ち着きのない日本人は、ファッショナブルなかつら用のユニークな人工毛を開発しました。これは特別に処理されたときに電流を流しました。












楽しい部分に移りましょう。 通常のスタイラスモードで停止しません。 他の多くの製品と同様、うまく機能します。 すぐにブラシモードに移動します。

彼らが言うように、百回聞くよりも一度見たほうがいいです。 それでは、ビデオから始めましょう。







私自身は美しく描く方法を知りません、ブラシは私のガールフレンドに非常に速く行きました。 そして、ここにSensu Brushの主な力があります。 彼女は本当にあらゆる年齢の女の子が好きです。



そして6歳の女の子に。







ところで、子供にとって、ブラシは素晴らしい贈り物です。 私は誕生日にそれを持ってきて、小さなレロチカは喜んで絵を描きました。 子供が通常の水彩画の本格的な感覚を受け取ることは特に嬉しいですが、同時に塗料で何も染まらず、水であふれず、タブレットはゲームだけでなく尋ねます。



ブラシは、年長の女の子にも快適です。

欠陥がないわけではありません。



  1. ブラシで押すようなものがない。

    私たちは皆、ブラシを優しくなめることで非常に細かい輪郭を描くことに慣れています。 Sensu Brushでは、このトリックは機能しません。 彼女は圧迫のようなものを持っていません。 アプリケーションでブラシの太さまたはスタイルを選択し、少なくともかろうじて触れて、少なくとも強く押します。 そして、はい、古典的な絵画のためにブラシを濡らして使用することは強く推奨されません。
  2. クリップの欠如

    ジャケットやバッグのポケットにハンドルを固定するのが好きです。 私が言えることは、私は櫛のクリップさえ持っていて、とても便利だと思います。 タッセルにクリップが表示されないので、おそらく彼女は描画の邪魔になるでしょう。 しかし、何らかの理由で、私は自分のバッグに入れて運んでいたとき、常にSensu Brushを引っ掛けたいと思っていました。







  3. ブラシケース

    普通のブラシにキャップを付けようとしたことはありますか？ そして、単一の絨毛がキャップの外にないように？）

    ブラシの最初の不注意なブラッシングの後、ふわふわの毛がいくつか現れ、完璧主義者にとって地獄を保証します。








  1. 筆塗りは本当にクールです。 画面上で絨毛が滑るのを感じるのは本当にいいことです。 すでにこの理由だけで、Sensu Brushを購入することについて後悔はありません。
  2. イラストレーターにとって、ブラシは素晴らしいヘルパーになります。 コンパクトでスタイリッシュ、幅広いアプリケーションを備えており、画像ギャラリーの作成、アルバムのスケッチ、文字通りどこでもスキルの向上に役立ちます。
  3. ブラシを使用すると、フローチャート、図、デザインインターフェイスを簡単に描画できます。 もちろん、これは通常のスタイラスで行うことができますが、このために最初のブラシをガールフレンドに渡した後、2番目のブラシを取りました。
  4. 図面のトレーニング。 私が複数回使用したアプリケーションでは、果物、動物、人物を描く方法を学ぶのに役立つレッスンがありました。 最後になりましたが、常に進行状況を保存し、失敗したアクションを取り消すことができます。








  1. スチール製のロッド、ワッシャー、スレッドから静電容量式タッチスクリーンのスタイラスを作成する方法
  2. キャップ-ビニール袋、湿った塩味のぼろきれ、ペン、電気テープの容量性スクリーン用スタイラス
  3. このビデオでは、静電容量式スクリーン用のさまざまな種類の自家製スタイラスを視覚的に見ることができます。

  4. また、Nomad Brushと呼ばれる産業用のカウンターパートもあります。 価格はそれほど劣りませんが、デザインははるかに劣ります。 しかし、その仕組みについては何も言えません。











私の意見では、Sensu Brushはすごい効果のための一度きりのガジェットではありません。 ブラシは見栄えが良く、Kickstarterスタートアップの生の子孫のような印象を与えません。 これは美しく、簡潔で、完成した製品で、本当に使いやすいです。
