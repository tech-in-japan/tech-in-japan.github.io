テラヘルツアンテナ255 x 341ナノメートル

数か月前、Habréで日本の会社Rohmが製造したテラヘルツモデムについて話しました.1.5 x 3 cmのデバイスは1.5 Gbit / sの速度でデータを送信できますが、開発者は将来的に30 Gbit / sにオーバークロックすることを約束します。 伝送はテラヘルツスペクトル、つまり100 GHz〜10 THzの範囲です。 このようなモデムは、テレビ、プロジェクタ、およびコンピュータモデムで超高解像度ビデオを送信するのに役立つと想定されています。







現在、シンガポールに本社を置くA * STARとインペリアルカレッジロンドンの研究者は、テラヘルツスペクトルでも動作するアンテナを開発しましたが、同時にサイズはわずか255 x 341ナノメートルです。 放射を生成するために、半導体結晶の電気光学効果が使用されます。



どうやら、このアンテナは、スキャナーのテラヘルツ放射発生器（T線）として単に使用されることを意図していないようです。 開発者は、室温の通常の条件下で、現在のシステムの「約100倍強力な」放射線を生成すると言います。



ご存知のように、テラヘルツスペクトルの波には特定の特性があります。それらは衣服、木材、プラスチック、セラミックを通過し、金属と水から反射するため、空港のセキュリティシステムにテラヘルツトモグラフを設置して、荷物と人をスキャンします。 したがって、新しいアンテナは既存のデバイスで広く使用できます。



データ伝送にこのようなアンテナを使用することは、より遠い将来の見通しです。 小型の寸法により、チップ上に直接配置したり、インプラント、たとえば高解像度ビデオを受信するためのディスプレイ付きコンタクトレンズに埋め込むことができます。 ただし、seewald habrayuzerがコメントで正しく指摘しているように、この場合は、瞬きではなく送信機を直接見る必要があります。
