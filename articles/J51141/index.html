Office Communications Server 2007 R2の仮想起動

先日、Office Communications Server 2007 R2の新しいバージョンの仮想起動がありました。



オンラインイベントサイト（これを呼び出すことができます）は、 www.microsoft.com / communicationsserver / r2-virtual-launch / eventにあります。 なぜオフラインではなく、仮想起動ですか？ 答えは簡単です。

1）さまざまな国のより多くの人々がレポートや資料にアクセスできます

2）時代に遅れずについていく:)仮想イベントの人気が高まっています



このサイトには次のものがあります。

-製品セッション

-オープニングセッション（基調講演）

-便利なリンクとファイル（たとえば、5日間の試用版）

-仮想イベントの興味深い実装（技術に興味がない場合でも、プラットフォームを確認する必要があります）



さらに、新機能に関するいくつかのドキュメントが既に利用可能です（現時点では英語）： http : //technet.microsoft.com/en-us/library/dd425311(office.13 ) .aspx



間もなく、ロシア語のオンラインレッスンがTechDays.ruのサイトに表示されるようになります。
