sudoおよびグラフィックアプリケーション。

潜在的に安全でない可能性がある特定のアプリケーションを実行する場合があります。 たとえば、firefox。 ローカルユーザーの権限で実行されているプログラムは、そのホームディレクトリからデータを漏えいさせる可能性があります。 ここでは、Wineの下で実行される安全でないアプリケーションの山を追加できます-例えば、IE 5/6。



セキュリティの側面に加えて、他のものがあります：

-現在のユーザー用に構成された構成とは異なる構成でプログラムを起動する：

-仕事からのプロキシを介してポルノサイトを閲覧するためのff

-プロファイルなしのグローバル設定を使用する他のプログラムで、すぐに作業する必要がある

-別のアカウントのブックマーク、ドキュメント、メールへのクイックアクセス

-自分で考えてください:)



おそらくあなたの一人はすでにこれに遭遇しています：

sudo -u guest firefox

No protocol specified

Error: cannot open display: :0.0







この動作の理由は、Xサーバーへの接続に必要な情報の不足です。



プログラムを正常に実行するには、少なくとも次のものが必要です。

gksudo -k -u user_name eval $ TERM; env TERM = 'xterm' DISPLAY = '：0.0'“ firefox”;
