Expression Studio 3がリリースされました

今日、 Microsoft Expression製品ラインが更新され、 Expression Blend 、 Expression Design 、 Expression Web、およびExpression Encoderの新しいバージョンが含まれています。

製品の完全なセットは、1つのファイル（ http://www.microsoft.com/downloads/details.aspx?FamilyID=7e2f033b-c6b5-4565-93a5-a6048246ce28）でダウンロードできます。

更新された製品サイト-http://www.microsoft.com/expression

更新されたコミュニティサイト-http://expression.microsoft.com



個々の製品および追加の詳細へのさらなるリンク。



Expression Blend 3 + SketchFlow





新機能：
  • SketchFlowを使用したプロトタイピング 。

  • レイヤー、グラフィック、テキスト要素を保持したまま、 Adobe PhotoshopおよびIllustratorファイルをインポートします 。
  • Blend での Silverlight 3およびWPF 効果の直接サポート 。
  • 注釈 -効果的なチームワークのために個々の要素にコメントを残す機能。
  • コントロールパネルは、作業中は開いたままにできる独立したアセットパネルになりました。
  • Make Into Controlオプションを使用して、選択したオブジェクトに基づいてテンプレート/スキンを作成します。
  • 状態の改善（状態） -Visual State Managerの改善、トランジションの編集とプレビュー。
  • 動作 -コードを作成せずにプログラムに対話性を追加する機能。



  • データの改善された作業 -プロトタイピングを容易にするデータサンプルの使用、改善されたデータバインディング（データバインディング）。
  • Silverlight 3のサポート -ブラウザーの外部で作業を追加する機能や、プロジェクトメニューからライブラリを直接キャッシュする機能を含みます。
  • TFSとの統合 -設計者は、チェックイン、チェックアウト、マージの内容を知ることができます。
  • EasingFunction —パラメーターを変更するための既製のテンプレートを含む、アニメーションの操作の改善 。
  • フォントサポートの改善 -SilverlightおよびWPFプロジェクトにフォントを埋め込む機能を含む。
  • ドッキング機能、グラデーションの操作の改善、レイヤーなど、ささいなことに関するワークスペースの改善。
  • コードのサポート -Blend、IntelliSense、オートフォーマットなどのバンから直接編集できます。
  • 拡張性-Blendとともに「神秘的な」Blend SDKを追加;）
Expression Blend 3のダウンロード ： http : //www.microsoft.com/downloads/details.aspx? FamilyID=e82db5e2-7106-419e-80b0-65cce89f06bb



表現デザイン3





新機能：
  • Adobe Photoshopファイルのインポート -レイヤーのサポート。
  • ワークスペースの改善 -すべてのパネルにドッキング機能が追加されました。
  • Silverlight 3サポート -グラフィック要素を作成し、Silverlightアプリケーションで使用するためにエクスポートできます。
  • エクスポート効果 -SilverlightおよびWPFの場合、たとえば、DropShadowまたはGuassian Blur。
Expression Design 3を ダウンロードする ： http : //www.microsoft.com/downloads/details.aspx?FamilyID=10e4a530-2c96-4aff-b00c-a154501876bc



Expression Web 3 + SuperPreview





新機能：
  • SuperPreviewは、さまざまなブラウザーでページの表示をデバッグするための特別なユーティリティです。 こちらもご覧ください 。

  • スナップショットプレビュー -コード編集中に直接ページの外観の更新されたプレビュー。

  • DeepZoom -DeepZoom画像コレクションの挿入の組み込みサポート（SilverlightおよびAjaxバージョン）
  • Silverlight-スキンを選択およびカスタマイズする機能を備えた、Silverlightビデオプレーヤーおよびアプリケーションの挿入の組み込みサポート。
  • FTPサポートの改善- （一度に1つではなく）一度に複数のファイルを転送し、安全なプロトコルをサポートします。
Expression Web 3のダウンロード ： http : //www.microsoft.com/downloads/details.aspx?FamilyID=0a73a3a7-3e06-4125-b3c6-f9c10387e9cc



Expression Encoder 3





新機能：
  • スクリーンキャプチャ -スクリーンキャストをエンコーダーで直接記録できるようになりました。
  • タイムライン上の複数のクリップ -多くの部分からビデオを作成し、タイムライン上のどこにでも挿入できるようになりました。

  • アダプティブストリーミングサポート -IISスムーズストリーミング用。
  • 新しいSilverlightテンプレート -ビデオプレーヤーの更新セット。H264、プレイリスト、チャプター、オフラインモードでの再生、別のウィンドウでのサポートを含みます。
  • エンコーディング設定をプリセットに保存して再利用します。
  • 拡張H264サポート -微調整のための多くのオプションがあります。

Expression Encoder 3のダウンロード ： http : //www.microsoft.com/downloads/details.aspx? FamilyID=b6c8015b-e5de-46c0-98cd-1be12eef89a8
