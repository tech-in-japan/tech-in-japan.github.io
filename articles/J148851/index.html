200万ルーブルの賞金であるWindows Phone用メッセンジャーVKontakte！

VKontakteは、既存のAPIに基づいてWindows Phoneのメッセンジャーを作成するコンテストを開催しています。

コンテストの開始日は2012年8月1日で、仕事の受付期限は2012年9月15日（21:00）です。

賞金は2,000,000ルーブルです。



コンテストグループ ： http : //vk.com/wp_contest



競争条件 （ http://vk.com/page-41437534_43978998 ）：

タスク

VKontakteプラットフォームを使用して、 Windows Phoneベースのスマートフォン用のメッセンジャーを作成するためのすべてのメイン画面とインターフェイス要素を描画しました。 機能の詳細な説明は、 別のページで入手できます 。



あなたの仕事

VKontakte APIに基づいて、 C ＃のタスクで説明されているボリュームにアプリケーションを実装する必要があります。 提案された美学を正確に再現し、アプリケーションの高速化を実現することが重要です：初期読み込み、ダイアログを開く、メッセージと添付ファイルの送信、すべてのアニメーションの滑らかさと思慮深さ。 提案された設計および機能からの逸脱は許可されていません。 競技会への参加のための不安定で未完成の申請は許可されません。



アプリケーションは、すべての一般的なWindows Phoneデバイスと互換性がある必要があります 。 可能なすべての画面解像度をサポートする必要があります。 タスクの追加要件は、インターフェースのロシア語版と英語版の存在です。



Windows Phone Marketplaceへのアプリケーションの公開はオプションです。 開発者は、裁量でアプリケーションの名前を選択する権利を有します。 VKontakteに不慣れなユーザーであっても、アプリケーションは自給自足でなければなりません。 アプリケーションの著作権は開発者にあります。



開発を容易にするため、必要なすべてのAPIメソッドのセットを別のページに用意しました。



PNG形式のグラフィック素材、すべての画面のレイアウト、およびPSD形式のソースグラフィックファイルのセットは、リンクからダウンロードできます： http : //vk.cc/Ra6Tv



応募受付

2012年9月15日の21:00まで、エントリはxapインストールファイルとアプリケーションソースコードを含むアーカイブの形式でcontest@vk.comに送信する必要があります。 すべての参加者の作品のリストとコンテストの結果は、コンテストグループとVKontakte APIグループで公開されます 。



競争は、ロシアだけでなく、 どの国の居住者にも開かれています。



すべての参加者は、コンテストグループに参加して関連情報を入手し、ディスカッションに参加することをお勧めします。

Windows Phoneの開発情報は、MSDNでロシア語と英語で入手できます。
