PowerShell：ファイルをシャッフルする

こんにちは、％habrauser％！

最近、彼らは私に一見シンプルな作業を投げかけました。投票システムには約1000枚の写真を混ぜる必要があります。

これを手動で行うことは、ありがたい仕事です。 素敵な小さなスクリプトが必要です。

PowerShellは理想的なオプションであることが判明しました。Pythonのようなものはインストールしたくありませんでしたが、私の人生で.batファイルと友達になったことはありません。



始めましょう。

写真の名前は重要であり、私はそれらを保持したかったです。 ファイルプロパティでそれらを削除することにしました（WinXpの場合は「概要」、Win7の場合は「詳細」）。 さらに、投票はsharepointに書き込まれ、ファイルプロパティから「名前」を選択できます。 それは非常に便利であることが判明しました。

ファイルプロパティを編集するには、 OLEファイルプロパティリーダー2.1コンポーネントを使用しました（vb7サンプルフォルダーに.netのライブラリラッパーがあります）。この利点は既にマシンにインストールされています。 このコンポーネントは、オフィス文書だけでなく、写真も対象としています。



実際のスクリプト： 

[System.Reflection.Assembly]::LoadFrom('c:\DsoFile\Source\Vb7Demo\bin\Interop.DSOFile.dll') function SetSummary-Name([string] $folder, [string]$pattern = '*.*') { $files = [System.IO.Directory]::GetFiles($folder, $pattern) foreach ($f in $files) { $oled = New-Object -TypeName DSOFile.OleDocumentPropertiesClass $oled.Open($f, $false, [DSOFile.dsoFileOpenOptions]::dsoOptionDefault) $sp = $oled.SummaryProperties $sp.Title = $f.Substring($f.lastindexof('\') + 1, $f.lastindexof('.') - $f.lastindexof('\') - 1) $oled.close($true) } } SetSummary-Name -folder 'c:\photo'
