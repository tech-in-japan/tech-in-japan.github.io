Web Platfom Installerの新しいバージョン

冬に、 MicrosoftのGaidarは 、専用のWeb Platfom Installerサーバーで作業するための新しいツールについて書きました。

このツールは、春にHostingSummiteで機能するのを見ました。 専用のサーバーで数分で、サイトを運営するために必要なすべてのもの（すべてのお気に入りのPHPなど）をインストールする方法が非常に効果的に示されました。

数分で、アプリケーションギャラリーからWordPressなどの人気のあるエンジンを1つまたは複数配置できます。



次に、この製品専用のセクションをサイトに作成しました。

また、2分間または3ワードプレスでDrupalをインストールする方法に関するいくつかのスクリーンキャストも記録しました。

本当に速くて簡単！



その後、ホスティングに異なるエンジンをインストールする初心者ブロガーやウェブマスターによって書かれたチケットの数がわかります。

そして、専用サーバーを使用したときに何が始まります。 （個人的な人はいません！）

この意味で、数回クリックするだけでWebサーバーを管理し、エンジンをインストールできる便利な製品があることは素晴らしいことです！



現在、 VPSのお客様には、Windows Web Server + IIS + mySQL + WPIを含む既製の画像を提供しています。

VPSを注文して、サーバーをほぼ準備完了状態にします。 PHPの最新バージョンと目的のエンジンを数分で配置するだけです。



実際にニュース。 ））その後、MicrosoftはWPI Betaを提供し、最近リリースされたWPI RC（リリース候補）をWindows Web Serverを実行する専用サーバーで使用することを提案します。

アプリケーションギャラリーにはすでに19個のエンジンがあります！ そして、ブログ、ギャラリー、そしてショップ...一般的に、あらゆる好みのために。
