ロシアデザインカップ

こんにちは、Habr。 私はいつもプログラマーのためにこんなに多くの種類の競争があるのに、なぜウェブデザイナーやデザイナーのための競争が少ないのだろうと思いました。 ご存知のように、これまで多くの人がデザイナーの作品の結果をクリップアートとグラデーションのアイコンで撮影した帽子だと考えています。 実際には、大規模で複雑なプロジェクトでは、デザイナーとデザイナーのタスクははるかに広く、より興味深いものです。それらには、インターフェイスとの人間の相互作用の研究、制限付きの作業、ガイドラインの開発、クイックリリースのためのスクリプトの分解の両方が含まれます。 デザイナーは、批判を処理し、優先順位の変化に迅速に対応する能力がなければ、作品を提示するスキルなしにはできません。







Russian Design Cupを思いついたとき、私たちはいくつかのタスクを設定しました。

  • 会社で直面しているタスクの興味深い例を示します。
  • 初期のデザイナーの注意を引き付け、プロジェクトに取り組む際の詳細に関心を持たせる。
  • 関連業界の設計者がUX-UI設計を試せるようにします。
  • そのようなコンテストの開催経験を積むだけです。 最初のパンケーキが「非常にゴツゴツ」ではなく、競争に興味がある場合は、間違いに取り組み、来年もそれを繰り返します。
  • そして、もちろん、一部のコメンテーターを予想して、デザイナーの素晴らしいアイデアをすべて盗み、一晩で実現し、誰にも何も支払うことはありません。


すべての審査員はタスクの作成に参加します。 タスクの複雑さはラウンドごとに増加します。 それらのいくつかは深刻であり、思慮深い分析を必要としますが、いくつかは反対に、可能な限り創造的にソリューションにアプローチできるようにします。 最初のラウンドは1か月間続き、参加者は解決策について考える十分な時間を持ちます。



意思決定を提示するための形式は無料です-自由に実行してください。 脚注、リンク、ワイヤフレーム、グラフィカルインターフェイス、テキストの説明。 デザイナーが彼女の作品を提示する方法は、審査員の評価に影響を与えると確信しています。 評価では、誰の解決策が目の前にあるかはわかりません。 投票は締め切られます。次のラウンドに行く（または行かない）場合は、それについての手紙を受け取ってください。



このプロジェクトはデザイナーのイニシアチブであり、実際に実験であるため、リソースに限りがあり、競争を可能にする最小限の機能のみを作成しました。 プロファイル、パスワード、オンライン投票なしで、各参加者はタスクと結果にアクセスするための固有のリンクを持ちます。



また、アイデアがあります-競争の終わりに、参加者や他の人がすべてのソリューションに精通し、議論し、楽しむことができるように、すべての仕事をどこかに置きます、あなたは何と言いますか？



これが私が伝えたかった最も重要なことです。 規則と組織の詳細は、競技ウェブサイトで見つけることができます。 コメントで質問してください-可能な場合は投稿に回答して更新します。



PRスペシャリストによるスナックテキスト



デザイナーとプランナーにロシアデザインカップコンペティションへの参加を呼びかけます。 ロシアデザインカップは、ロシアや近隣諸国で開催されるこのような最初のオープンコンペティションです。 才能のあるインターフェイスデザイナーが手を試し、スキルを発揮できるプラットフォームになることを望んでいます。 RDCは、全国のデザイナーが自分自身を表現し、新しい視点を開くことを可能にします。 次に、業界はロシアデザインカップのおかげで、才能のある専門家について学びます。 競争力のある割り当ては、ワークフロー中にデザイナーが直面する実際のタスクに基づきます。 最高の作品は、大手IT企業の設計部門の責任者（Mail.Ru Group、ランブラー、ヤンデックス）によって選ばれます。



最も正直な審査員：


  • Yuri Vetrov、Mail.Ru Group、インターフェース設計および設計部長
  • Mail.Ru Groupのデザイナー、オレグ・アンドリアノフ
  • Dmitry Zimin、OK RamblerおよびAfishaのインターフェース設計責任者
  • Yandex Web Interface Designの責任者であるKonstantin Gorsky氏
  • Alisher Yakupov、デザイン責任者、Odnoklassniki Social Network、Mail.Ru Group Project




現在、ロシアデザインカップのウェブサイトは参加者を登録しています。 コンテストへの参加については、すでに500以上のアプリケーションが受信されています。 あなたも宣言してください。



いつどこで？


コンテストは7月1日から8月11日まで、 russiandesigncup.ruのWebサイトで開催されます。

何に参加する必要がありますか？

18歳以上で、Mail.Ru Groupの従業員でも従業員の近親者でもない場合は、競争に参加できます。



どのようなテストですか？


参加者は、資格、資格、最終ラウンドの3つのステージを克服する必要があります。

各段階で、参加者はインターフェイス設計の分野で1から3のタスクを解決するように求められます。



そしてもちろん賞品


3つのファイナリストには、最大構成のAppleテクノロジーが授与されます。 1位を獲得した参加者には、27インチのApple iMacが授与され、2位と3位にはそれぞれ13インチのMacBook AirとiPadが授与されます。



公式のRDC twitter.com/RDesignCupでコンテストをフォローしてください
