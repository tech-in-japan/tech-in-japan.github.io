ドリームブックライトIL1



このミニラップトップには、解像度800x480ピクセルの7インチディスプレイ、1 GHz VIA C7-Mプロセッサ、1 GBのRAM、40 GBのハードドライブがあります（オプションで2、4、8 GBのフラッシュドライブが提供されます）。



短所：ドリームブックにはWi-Fiアダプターがなく、このクラスのラップトップには必須であり、基本バージョンの最低価格ではありません-米国では450ドルです。



事前にインストールされたオペレーティングシステムを選択できることが重要です。WindowsVistaまたはLinux Ubuntuを使用できます。



エロネックス







英国の会社Elonexはまた、約100ポンド（約200ドル）の価格の超小型で安価なラップトップをリリースしました。



このラップトップはLinuxオペレーティングシステムで動作し、充電式バッテリー（充電は3時間続く）、Wi-Fiモジュール、フラッシュドライブ、および「ワイヤレスミュージックサーバー」を搭載しています。 これにより、ラップトップの重量は1 kg未満になります。



Linuxプラットフォームを使用して、同社はデバイスのコストを削減するという目標を追求しました。 また、「独占的なマイクロソフト」によってリリースされていないソフトウェアの使用を奨励する政府機関の希望を忘れないでください。



ラップトップ100台が販売されるごとに、会社は貧しい家庭の子供たちに1台を寄付します。



光人社E8







KojinshaはE8ミニラップトップで消費者を喜ばせました。



この新製品には、Asusと比較していくつかの利点があります。 たとえば、1024x600ピクセルの解像度を持つ回転可能なタッチスクリーンが装備されています。



モデルの仕様には、AMDのGeode LX800プロセッサ、40 GB HDD、512 MB RAM、USBポート、メモリカードスロット、内蔵ステレオスピーカー、および最大3.5時間のバッテリー寿命を提供するバッテリーも含まれます。



E8では、Windows XPがインストールされています。 小売価格-約600米ドル。



NorhTec Geckoラップトップ







タイに拠点を置くNorhTecは、さらに別の安価なラップトップ、Geckoラップトップを発表しました。



Engadgetによると、Geckoは1 GHz Via C7M ULVプロセッサーとLinpus Linux Liteオペレーティングシステム（ほぼ同じXandros）で動作します。



また、ラップトップには4 GB SSDドライブ、追加のハードドライブ（40 GBなど）、統合グラフィックカード、800 x 480の解像度を持つ7インチディスプレイが搭載されます。



彼らは、ラップトップが充電せずに約3.5時間動作することを約束します。 バッテリーは最大8時間使用できます。 もちろん、WiFiが含まれています、3-in-1カードリーダー。

標準のRAMでは不十分な場合があります（256MB）が、必要に応じて1GBに拡張することも可能です。



費用は300米ドル以内です。
