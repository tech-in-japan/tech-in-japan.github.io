Windows用Nokia Map Loader





それでもわからない場合は、N900用のすばらしい無料のOvi Mapsプログラムがネットワーク上でカードを「引き出し」、キャッシュします。 ご存知のように、プロバイダーの場合、さらにローミングの場合は、これにより（友人の経験から）複数のGPSナビゲーターのコストが発生する可能性があります。

Googleマップにも同じ問題がありますが、Googleとは異なり、ノキアは公式マップキャッシュツールであるNokia Map Loaderをリリースしました。 Symbianの場合、このアプリケーションはずっと前ですが、Map LoaderでのN900のサポートは最近実装されました（公式サポートがなければカードを埋めることができるという情報を見ました）。

サポートされている両方のプラットフォーム（WindowsとMac OS）で試してみることにしました。 残念ながら、Linuxのサポートはありません。この場合だけでなく、一般に（残念なことに、もう時間はあります。デバイス自体がLinuxの下にあり、OSのデスクトップバージョンの公式サポートがないのは奇妙です）。 最初はWindows XPで試してみましたが、7台ではヒキガエルがそれを買うために押してくれません。非常に高価です。 しかし、サーフィンの過程で出会ったスリップ情報（リンクなし、失われた:-)）から判断すると、すべてがそこでも機能します。 原則として、ツールチップの不正確さを除いて、すべてがうまく機能します。 ビデオでは、これに焦点を合わせました。

WindowsでMapLoaderを実行するには、Nokia PCSuiteをインストールする必要があります（OVI-suiteはまだN900をサポートしていないため、OVI-suiteと混同しないでください）。

ダウンロードリンク：

Nokia PC Suite-europe.nokia.com/support/download-software/pc-suites

Nokia Map Loader-www.nokiausa.com/get-support-and-software/product-support/maps-support/compatibility-and-download#/nokia-n900

さらにビデオで：



Windowsバージョンに関する小さなコメント。 ツールチップには、デバイスをPC-Suiteモードで接続する必要があることが明確に記載されていますが、実際にはこのモードではプログラムはデバイスを見つけることができません。 次に、デバイスがMap-Loaderを実行してPC-Suiteモードでオンになり、大容量ストレージモードで再接続した場合、何も起こりません。 アクションのシーケンスは次のようになります。デバイスを大容量ストレージモードで接続し、ディスクが表示されることを確認してから、Map Loaderを起動します。

外出先でプレイリストを作成する





何か気になったり、気分が劇的に変化したり、単一のプレイリストが適切ではないことが起こります。 その場でプレイリストを作成する機会のために、私はかつてAppleにとても感謝した。 iPod-これを許可した最初のデバイスでした。 IPhoneをN900に変更しました。NokiaMultimedia Transferのおかげで、iTunesの習慣を変更する必要はありませんでした。 メディアライブラリ全体がそこにあり、必要なプレイリストのみを同期するデバイスを備えており、さらにプレイリストを「外出先で」作成する方法を発見しました。 それがまさにあなたと共有したいことです。

これは英語のビデオです（ブログwww.allaboutmaemo.comから取得 ）。



今ロシア語で。 必要なのは、曲やアルバム、別のプレイリスト、ジャンル、アーティスト、一般にリストの任意の要素を選択し、指/スタイラスで数秒間（右クリックのアナログ）保持して、表示されるメニューで[現在のプレイリストに追加]を選択することです。 。 現在再生中のリストを入力するには、右下隅にある小さな再生ビジュアライザーをクリックする必要があります。 リストをスケッチしたら、保存して、プレイリストのリストの対応するカテゴリに表示できます（taftlogs :-)）。 好きなだけリストを作成できるので、iPodよりもさらに便利です。

このアプリを提供してくれたMartin Grinneに感謝します。 ここから、またはパッケージマネージャーからダウンロードできます。 これはどのように見えるかです：

便利なキーボードショートカットのリスト



ここで簡単に説明しますが、キーボードデバイスとこれには利点があります。そのうちの1つは、作業を高速化し、追加の対話メカニズムを提供するキーの組み合わせです。



一般的な組み合わせ（グローバル）：

1. Ctrl + Backspaceは、タスクマネージャーを呼び出します。

2.通常どおり、テキストを移動するには、Ctrl + Cテキストのコピー、Ctrl + V貼り付け、Ctrl + XおよびCtrl + Aを使用します。

3. Ctrl + O開く。

4. Ctrl + N：新しいアイテムを作成します。

5. Ctrl + S：保存。

6. Ctrl + Z：元に戻す（元に戻す）。

7. Ctrl + Y：元に戻す（やり直し）。

8. Ctrl + F：検索バーを開きます。

9. Ctrl +右矢印：PCと同様にEND機能。

10. Ctrl +左矢印：PCと同様にHOME機能。

11. Ctrl + Shift + P：スクリーンショット。

12. Ctrl + Shift + X：「Xターミナル」を開きます。

13. Ctrl +スペース：キーボードレイアウトを切り替えます。



ブラウザのキーボードショートカット：

1. Ctrl + N：新しいブラウザーウィンドウ

2. Ctrl + R：更新、RSSにも適用

3. Ctrl + B：ブックマークを開きます

4. Ctrl + D：現在のサイトをブックマークする

5. Ctrl + S：現在のページをメモリに保存します

完全な組み合わせではありませんが、便利です。ブラウザでズームします。

エリアをダブルクリックすると、エリアが増加します。増加は「スマート」で、この要素が属するエリアを可能な限り増やします。 たとえば、フォームに入力フィールドがあります。このフィールドをダブルクリックすると、最大値になり、入力しやすくなります。 または、たとえば、小さなボタンの狙撃スコープにかからないようにするには、この領域で非常に正確に2回押す必要はありません。この領域は近づき、無理なくボタンに到達できます。 さらに2回押すと、すべてが所定の位置に削除されます。 非常に便利です（iPhoneの動作は非常に似ています）。

また、マルチタッチの欠如は、たとえばスケールを変更するための代替の動きによって補うことができます、あなたはまだ時計と反時計回りに円運動をすることができます（電話ではなく、画面上の指/スタイラスで自然に:-)）、したがって、削除して近似します。

さて、ズームのハードウェア方法はボリュームコントロールです。 ブラウザでは、キーボードを開いた状態ではあまり便利ではありませんが、スケールコントロールとして機能しますが、キーボードを閉じた状態では必要になります。



さらにいくつかの有用性：

画面上にデスクトップがある場合（アクティブなアプリケーションがない場合）、連絡先の名前をすぐに入力できます。名前によるコンテキスト検索が表示されます。

（同時ではなく）Alt、Ctrlを押す-追加の文字のリストが表示されます。 Altは、CtrlとShiftの間の左側にある小さな青い矢印です。



フロントパネルのLEDが点滅する内容



インジケータランプ：

1.青色の点滅-アプリケーションは、新しいインスタントメッセージ、SMS、または電子メールについて通知します。

2.オレンジ色の点滅-バッテリーは充電中です

3.緑-バッテリーは完全に充電されています



すべてのように？ しかし、ここにはありませんでした。 これらのテンプレートは変更でき、2つの方法を提供できます。

1.オタクの場合、これらのテンプレートを処理する特定のファイル/etc/mce/mce.iniを「ハッキング」する必要があります。このファイルには、およそ[LEDPatternLystiRX51]セクションがあり、およそ次の内容が含まれています。

PatternCommunicationCall = 30; 1; 0; b ; 9d80400002ff03ff02ff03ff71080000; 9d800000

PatternCommunicationIM = 30; 1; 0; b; 9d80400002ff03ff02ff03ff71080000; 9d800000

PatternCommunicationSMS = 30; 1; 0; b; 9d80400002ff03ff02ff03ff71080000; 9d800000

PatternCommunicationEmail = 30; 1; 0; b; 9d80400002ff03ff02ff03ff71080000; 9d800000

PatternCommonNotification = 30; 1; 0; b; 9d80400002ff03ff02ff03ff71080000; 9d800000



ここに太字の小さな文字「b」は青を意味し、「r」または「g」を使用できます。それぞれ赤と緑を意味します。また、「rb」、「rgb」...

詳しくは、この記事をご覧ください。

2.普通の人のための方法。 素晴らしいプログラムがあります-LED Pattern Editor。 LED信号を編集できるパラメーターメニューのアプレット。

点滅パターンと色を編集するためのグラフィカルインターフェイスを提供します。

変更を表示し、設定を保存し、デバイスを再起動して変更を有効にします。

「オプション」->「カスタマイズ」から起動します。



非常に機能的なプログラムであり、尊厳を持って作られています。



便利なプログラムのリスト（続き）



まあ、スナックのためにいくつかの便利なプログラム：

オタクの場合：



自動切断



一定期間のトラフィックが特定の制限を超えていない場合にネットワークを切断できる非常に興味深いプログラムです。たとえば、ネットワーク（WiFiまたは3G）、スキャン間隔（5分など）、および最大応答トラフィック（512キロバイトなど）を選択できます。 したがって、このタイプのネットワークで5分以内に512キロバイト未満のデータが通過した場合、ネットワークは切断されます。 時間とお金を節約するのに便利です（3Gの場合）。



バッテリーグラフ



同様のプログラムは数多くありますが、私はいくつかの理由で個人的に気に入っています。それはシンプルで、うまく機能し、Qtで書かれています。 主な目的は、バッテリーの充電を監視して、電力が消費された時点を把握することです（誰がバッテリーリソースを消費しているのかを把握するため）。 これは2つの部分で構成されます。システムで起動し、時間間隔でmySQLデータベースにこれらの瞬間の状態を記録するデーモンと、このデータを視覚化および分析するためのグラフィカルシェルです。 もちろん、この悪魔がバッテリーを食い尽くすことが判明するかもしれません:-)が、私はこれに気づきませんでした。

ブラウザ交換機
