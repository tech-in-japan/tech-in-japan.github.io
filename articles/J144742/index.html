独学：旅行に招待しましょう！

こんにちは、Habr。



少し前に、ウェブデザインとプログラミング言語を学びたいと思いました。 そして、もちろん、「どこから始めるべきか」という論理的な質問がありました。 さらに、ウェブデザインやプログラミングの経験がなかったため、状況は悪化しました。

どうする？ 自分を勉強する方法は？ どこに移動し、何をつかみますか？



さまざまな試行錯誤を通して、私は自分で話したいと思う便利な自己訓練システムを開発しました（誰かが役に立つとしたらどうでしょう？）。



まず、HabrとHabroの人々に感謝します。記事とコメントから、私は非現実的な量の有用な情報を作成しました。 このトピックの独創的なブロックのアイデアに非常に触発され、それを自己学習の基礎として取り入れました。



ステージ1：方向の選択



まず、何を勉強したいかを決める必要があります。 なぜあなたがそれを必要とするのか、それが何を与えるのか、それがあなたにとって本当に面白いのかどうかについての質問に答えるのを怠laにしないでください。 プロセスを楽しんで、長い間本当に学ぶことができるものを選択することは多くのことを助けます。



たとえば 、縫う方法を学びたいという願望がありました。 私はたくさんのマニュアル、教科書を手に入れ、それらのために座って、プロセス自体が私にとって退屈であることに気づきました。 私は基本を噛むことに悩まされました。 私はもうすぐ、計画された複雑な服をすでに縫い合わせていたでしょう。 それが難点でした-私は裁縫を学びたくはありませんでしたが、むしろ望ましい結果をすぐに得たので、学習プロセス自体は不幸な時間の浪費として認識されました。



学習する内容を可能な限り指定してください。 たとえば、「プログラムの作り方を学びたい」が特定の言語を学びたいという欲求に変わり、「Webの設計方法を学びたい」-特定のアクティビティ（サイトまたはアプリケーション？専ら設計またはレイアウトも）の選択にしましょう。 そのため、さらにトレーニングするための資料を探すのが簡単になります。



ステージ2：地図を描く



それで、あなたは勉強することに決めたものを選択しました。 一枚の紙とペンを用意し、図を描いて、あなたの意見では、トレーニング用に選択したオブジェクトが含まれているものを表示しようとします。 目標を、習得する必要のある小さなコンポーネントに分割します。



例： 私の目標の1つはWebデザインです。 これはかなりボリュームのある概念なので、私はそれを次のサブゴールに分けました。

a）設計理論（ここでは、ユーザーの知覚に関するWeb領域と非Web領域の両方に関係するものすべてを含めました-色の組み合わせ、フォント、使いやすさ）。

b）レイアウト（html、css）;

c）ツールの開発（Photoshop、私の場合）。



ステージ3：ルートを選択する



サブゴールを選択しますか？ 素晴らしい。 ここで、それらの資料を見つけて、必要に応じて調整します。

選択したサブゴールに一致するすべての記事、教科書、テーマ別ブログを探します。 調査結果があなたの知覚レベルに対応し、理解しやすく、学習しやすいものであることを確認してください。



例： 個人的には、テーマセクションとQ＆Aを掘り下げて、ハブのすべてを見つけました。 原則として、コメントはトピック自体と同じくらい便利で、多くの場合より便利です。

見つかったすべてのリンクと推奨事項をテキストファイルにコピーし、それらを調べてレビューを読み、必要かどうかを判断しました。

検索の過程で、サブゴールの最初のリストを調整する必要がありました。たとえば、「レイアウト」は「組版：理論」と「組版：練習」に分かれていました。 したがって、私は最初に決めたリストを得ました：

a）デザイン理論（スティーブサークル「考えさせないで！」）;

b）レイアウト：理論（Jeffrey Zeldman“ Designing with Web Standards”）;

c）レイアウト：練習（Nettutsplusで「HTMLとCSSを学ぶための30日間」）;

d）Photoshopをマスターする（Nettutsplusのレッスン）。



ステージ4：旅行計画



これが楽しみの始まりです（冒頭で述べたように）-創造的なブロックのシステム。

そもそも、トレーニングに費やすことができる1日あたりの空き時間がどれだけあるかを正直に認めることをお勧めします。 「仕事を終えてから無料で」というスタイルで計画しないでください。



第一に、休憩なしに2時間以上何かを熟考することは難しく、生産性の低下を伴います。

第二に、あなたは別の家事や時間を取りたい他の活動を持っているかもしれません。

一般に、今のタスクは、トレーニングに専念できる特定の時間を選択することです。



たとえば 、平日は1.5時間、週末は2.5時間を選択しました。

ここで、例えば、典型的な平日の私のスケジュール：

6.30-7.30-クリエイティブブロック（はい、私のクリエイティブブロックの1つは30分ですが、このためにフローティング境界線を使用して間隔を作成し、予期しない状況を割り引きます）;

7.30-8-仕事に取り掛かります。

8-17/18-仕事;

17/18-19-夕食、家事;

20 / 20.30-21-休憩;

21-21.30-翻訳;

21.30-22.30-サイトの問題。

22.30-23-休憩;

23-23.30 / 00-ルビー;

23.30 / 00-00 / 00.30-お風呂、そして寝る。



ご覧のとおり、時間間隔ごとにたくさんのフローティングボーダーがあります。 私は、さまざまな状況を考慮して、自分自身を「時間の借金」に追いやらないようにしています。

いくつかの休憩のギャップがあります。 これは、脳を少し和らげるのに必要です（私が何かに夢中になっている場合、何かをしたり、ベッドに横になったり、スケジュールや創造的なブロックから何かを続けたりすることがあります）。



そのため、クリエイティブブロックの空き時間を選択しました。 たとえば、1日2時間です。

まず、あなたとあなたの脳に最適な作業期間を自分で決めます。 すべてを完全にマスターして、どれだけ継続的に何かを学ぶことができますか？



例： 私にとっては、これは30分であり、その後、注意力が低下し、回転を始め、お茶を追いかけ、ブラウザを開こうとします。



それはあなたのための時間だとしましょう。

つまり、最適には、2ブロック（1時間ごと）に1日2時間を費やすことができます。 いいね！



第二に、自分のトレーニングに最適な時間を選択します。 誰も気にしないとき？ あなたが引っ張られていないとき？ いつ情報を吸収するのが最適ですか？

この時点でブロックを配置します。 クリエイティブが習慣をブロックするようにしてください（毎日、同時に、定期的にそれを行います）。 その後、適切な時間までにトレーニングに取り組む準備が整い、自分で事前に設定する必要がなくなったことに気付くでしょう。

誰にも邪魔されないときにクリエイティブブロックを配置することを強くお勧めします。 ブロック中は学習しているだけです。 すべて、これは勉強のための時間であり、仕事のために走り回ったり、後半との会話のためではなく、問題を解決するためでもありません。 学習のみが1つのクリエイティブユニットです。



第三に、学習目標とそのサブゴールのリストを取り、各項目にどのくらいのブロックを割り当てたいかを検討します。 したがって、前の例に従って、1時間に2つのクリエイティブブロックに毎日2時間を費やすことができます。 合計：7日* 2時間。1時間= 1クリエイティブブロック。 14ブロックになります。
