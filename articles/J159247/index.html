簡単に言うと、サムスンは新しいメモリモジュールEMMC Pro Class 2000の連続生産を開始します

Samsung Electronicsは、10nmプロセスを使用した次世代の64GBフラッシュメモリを発表しました。 新しいNANDメモリは先月後半に生産開始されました。



EMMC Pro Class 2000シリーズの統合マルチメディアカードは、高度なスマートフォン、タブレット、その他のモバイルデバイスで使用するために設計されています。 パフォーマンスチップは、EMMC4.5インターフェイスに基づいた以前のEMMC Pro Class 1500よりも30％優れています。







新しい高速EMMCモジュールは、来年JEDECによって業界標準として採用される予定です。



メモリには、最大2,000 IOPSの任意の書き込み速度と、5,000 IOPSの任意の読み取り速度があります。 さらに、シーケンシャル読み取り/書き込み速度はそれぞれ260 MB / sおよび50 MB / sであり、クラス10外部メモリカードの同様のパフォーマンス（読み取り-24 MB / s、書き込み-12 MB / s）よりも最大10倍高速です。 間違いなく、このようなチップ性能の向上は、マルチタスク環境でのモバイル機器の動作に非常に良い影響を与えるでしょう。



新しいドライブのサイズは20％削減されました。モジュールの全体寸法は、以前使用されていた12x16 mmのフォームファクタに対して11.5x13 mmです。
