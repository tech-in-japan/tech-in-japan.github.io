非公式のフォールアウトオンラインの2回目のオープンテストが開始





イベントの最初のhabratopic発表はこちらです。



良いニュース-habrakatの下;）





プロジェクトサイト-FOnline.ru



オープンテストの期間：状況に応じて1週間から3週間。



クライアントをダウンロードするためのリンク： fonline.ru/foclient.zip

+ ゲームクライアントをダウンロードするためのミラー。



クライアントアーカイブサイズ：34 Mb



オリジナルのFallout 2のファイルが必要です！







アーカイブに含まれるもの：

-FOnline.exe（ゲームの実行可能ファイル）;

-FOnlineDx9.exe（+ d3dx9_38.dll、前の段落と同じ、Direct3D9を使用してレンダリングのみが実行されます。また、このバージョンでは、フルスクリーンモードでのクライアントの誤った折りたたみによる未修正のバグがありません）;

-Mapper.exe（+ Mapper_ReadMe.txt、ゲーム用のマップを作成するためのプログラム、レビュー用にレイアウト。すでに2つのマップmap1、map2がアーカイブにあります）;

-Manual_Ru.pdf（ゲームのガイド、ロシア語と英語、ゲームの前に読むことをお勧めします）;

-Manual_En.doc（英語のマニュアル。残念ながら、これは古いバージョンであり、ドキュメントの最後にいくつかの重要な注意事項があります）

-新しい音楽トラック。

-子供向けのアニメーション（一部のゲームディストリビューションには含まれていません）;

-その他のファイル。



ゲームで、接続時に「ネットワークプロトコルのバージョンが間違っています」というエラーが表示された場合、このページを見ると、ダウンロードしてクライアントのフォルダーに展開する必要があるパッチがあります。



クライアントのインストール：



アーカイブを都合の良い場所に解凍します。

FOConfig.exeを実行します。 プログラムで、パスをmaster.datおよびcritter.datに設定します（または、これらのファイルを展開したクライアントのフォルダーにコピーするだけです）。他の設定も確認します（このオプションまたはそのオプションの目的について質問がある場合は、FOnline.cfgファイルを確認します） 、それらの簡単な説明がそこに与えられます）。

「Fallout2 \ data \ sound \ music \」フォルダーの内容を元のFallout2から「FOnline \ data \ sound \ music \」FOnlineフォルダーにコピーします。 そうしないと、ゲームにオリジナルの音楽が含まれなくなります。

ここで、ゲームサーバーのアドレスを入力できます。デフォルトアドレスはアーカイブに入力されますが、変更することも、新しいアドレスを追加することもできます。サーバーの状況は、このトピックで確認できます：（サーバー）

FOnline.exeまたはFOnlineDx9.exeを実行します。

残りは、ゲームマニュアルで読むことができます。ゲームマニュアルは、クライアントと同じフォルダーにあります。

インストールと起動について質問がある場合は、こちらに問い合わせてください。

クライアントのインストール：クライアントのインストールと起動。



テストに意味を持たせるために、次のゲームが発明されました。



ゲームの本質は次のとおりです。





シエラ、マリポサ、大聖堂の3つの拠点があります。 最終レベルの各ロケーションには、ベースを組織（BSまたはEnclave）のいずれかに従属させることができる端末があります。 その見返りに、彼らは毎日プレイします（ゲームの時間が12倍になります）、12.00に建物の入り口にさまざまなオブジェクト（武器、弾薬、鎧、道具など）を持ち込み、連絡を取ったプレイヤーのみがそれらにアクセスできます組織と。 次のパラメーターが統計に反映されます（Pip-Boy-> Status-> Statistics）：現在のベースオーナー、ベストベースオーナー（ほとんどのベースを保持）、ベースオーナーシップ（どのベースはどの組織に属します）、最も優先度（どのプレイヤーが複数の従属するか）ベース）。 このゲームがゲームに参加するプロセスを多様化することを願っています。



便利な注意事項：



ゲームウィンドウの解像度に関しては、幅640〜1280、高さ480〜1024の範囲内で任意の形状をとることができます。最大サイズは、カードの最小サイズのフィッティングに関連して設定されます。

ゲームの開始時に、トランシーバー、その使用方法が与えられます。マニュアルで読むことができます。

ゲームのF1キーを忘れないでください。ゲームの使用済みトラフィックとホットキーの統計を確認できます。

エンカウンターに残っているすべてのアイテムは、終了するとゲームから完全に削除されることに注意してください。 Hubbologistと一緒にjarにアイテムを保管します。

納税者以外の特別な場所にいないように、レプリケーションアカウントを時間内に補充することを忘れないでください。

Pip-Boyはプレイヤーの実績統計を追加しました。

現在、音楽は全体としてダウンロードされているため、ダウンロードが遅くなり、30〜40メガバイトのRAMが追加されます。数秒またはRAMの不足が重要な場合は、音楽をオフにします（音楽、サウンドのみを残すことができます）。
