CiklumがZend PHP Framework Conferenceをサポート-Zend Framework Day



経験を共有し、新しい何かを学ぶために-これらはすべて、個人的および職業的に開発することが可能であり、必要である方法の長い間知られている真実です。



そのため、さまざまな都市でさまざまなプロの開発者コミュニティの会議、キャンプ、集会が開催されます。



Ciklumでは、コミュニケーション、会議、経験の交換が、あらゆる人の専門的レベルの向上に寄与すると確信しています。 だからこそ、私たちはそのようなイベントを支援し、主催者にプロのトピックについて実りある会話をする場所を提供したり、ウクライナのキエフ 、 ハリコフ 、 ドネプロペトロフスクなどの都市、およびベラルーシ（たとえば、 ミンスク ）。



そのため、たとえば、 11月12日土曜日、 Zend Framework Dayが開催されます。これは、Frameworks Daysのフレームワーク内で開催される最初のイベントであり、サポートしています。



このイベントは、200人以上の建築家の開発者、ウクライナ、ロシア、および近隣のCIS諸国のプロジェクトマネージャーを「結集」させることが期待されています。 イベントのプログラムには、 「Zend Framework 2.0の紹介」から「ZendFramework 2.0を使用したコンソールアプリケーションの開発」 、 「ZF2を使用した軽量APIの作成 」など 、さまざまな好みのパフォーマンスが含まれます 。 主催者は、出席者全員にとって興味深く有用であることを約束します。



イベントの詳細は、主催者のウェブサイトで見つけることができます。
