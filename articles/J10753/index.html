Microsoft Surface。 未来への展望。

Microsoft が Surfaceと呼ばれるデバイスをリリースしたことを知らない人はいないと思います。 今日、私はその分野とその適用方法のトピックについて少し想像しています。



これは、次のような非常に興味深い機能を備えた新しいデバイスです。

  • インタラクティブな多目的（何かをねじった！=））。
  • デバイスは、自然に最も近い方法で制御されます。
  • さまざまなデバイスの認識、技術的な観点からのそれらからの美しい相互作用。


このデバイスの価格に注意を払うことはありません（非常に顕著に噛みつきます-5〜1万個の常緑樹）。 少し空想して、 MS Surfaceの可能な拡張機能とアプリケーションを提案してみましょう。



外部モジュール 。 モジュールはSurfaceとやり取りできる一種の外部オブジェクトであり、その表面に配置され、マジックテーブル（RFID、Wi-Fi？）によって認識されます。 ハードウェアモジュールは、表面に置かれた小さなスタイリッシュなデバイス（ガラス平行六面体？）のように見えます。これは、作業に干渉することはありません（または、おそらくそれに参加します）。 理論的には、これは単に物理的なシェルを備えた外部プログラムであり、ほとんどのユーザーが理解できるものです（ Put＆Play ？）。



素晴らしいツール。 このようなプラットフォームは、デザイナー、アーティスト、製図工にとっても深刻なツールになる可能性があります。彼らの観点からは、大きなタブレットのように思えるからです。 =）さて、ここではすべてが明確です。 続けましょう。



移行デバイス。 プレゼンテーションビデオでは、MSが電話（2つのモデルの比較特性を示した）、カメラ、その他の機器とどのように相互作用するかを確認しました。 しかし、 SurfaceがSDカードやSIMから直接情報を読み取れないことを疑う人はいないと思います。 時間の経過とともに、この困難な作業に役立つさまざまなデバイスが実装されると思います。 ペアリングの例は挙げません。ここでは、誰もが数十種類のデバイスを考え出すことができます（ユーザーの利便性と対話の自然さを重視する必要があります）。



ゲーム 常にゲームがあり、常にコンピューターテクノロジーを進歩させ、PC、Mac、PDA、その他のデバイスの配信のワイルドペースに貢献しました。 Surfaceのような（制御と革新の一般的な観点から）そのようなちょっとしたデバイスは、私たちのお気に入りの娯楽をバイパスしないと仮定するのは論理的です。 ゲームはメモリカードに格納でき、個別のモジュールとして配布できます。インターネットを忘れないでしょう。 MS Surfaceで見ることができるエンターテイメント：

  • 子供向けの教育ゲーム。 カラフルで安全なおもちゃがSurfaceの表面に置かれ、ゲームが直接読み込まれます。このゲームでは、複数の子供と大人が参加できます。 ゲームの終了時に、レコードは内蔵メモリに記録されます（約64〜128 Mb-レコードがあれば、これで十分です）。 ところで、これはPut＆Playの例です（上記参照）。
  • 外交戦略など。主な機能は、チップまたはゲームユニットのセット（軍隊のフィギュアなど）です。
  • ディアブロ、タイタンクエストなどのロールプレイングゲームは、Surfaceを見ると非常に興味深いものになります。 管理は快適なゲームに非常に便利です。


リストは続きますが、 Surfaceはさまざまなジャンルのゲームに最適です（カードを除く！=））。 これで私の短いリストを終了します。 誰かがSurfaceを使用する他の方法とそのアドオンを提案できる場合は、コメントを歓迎します！ =）
