ソフトウェアの自由の日/サラトフ

すべての人をソフトウェアの自由のinvite宴に招待します！



日付： 2009年9月19日

場所：サラトフ、SSU im。 N.G. Chernyshevsky、12番目の教育用建物（Volskaya 10a、Beloglinskayaとの交差点）。

時間： 10：00-18：00

私たちのサイト： sfd.sgu.ru



Software Freedom Dayは、非営利団体Software Freedom Internationalのイニシアチブで毎年9月の第3土曜日に開催される、フリーおよびオープンソースソフトウェアの世界的な祝賀会です。 世界中のボランティアチームが地元のSoftware Freedom Dayの祝賀会を開催し、人々の注目を集めています。



プログラム



10:00会議のゲスト。 配布資料の展示。

10:30はじめに。 Software Freedom Dayの歴史に関する物語。 参加者とのコミュニケーション。

11:30休憩。 プレゼンテーションとマスタークラスの準備。

レポートとマスタークラスをブロックします。

レポートのテーマ：

12:00-実務経験ソフトウェアライセンスに関する問題の解決。 （ヤコフポポフ）

12:30 -Python-主な機能の概要。 （Y.A. Blinkov）

13:00-オープンソースソフトウェアを使用した並列コンピューティングの組織。 （V.A.ポッチュカエフ）

13: 30-ロシア語を話すコミュニティとLoCo Ubuntuチーム。 （ドミトリーアガフォノフ）

ワークショップのトピック：

12:00 -Bluetooth + Linux +電話。 （エカテリーナポリアコバ）

13:00 -PyQt4を使用してGUIアプリケーションを構築します。 （Y.A. Blinkov）

14:00-休憩。

レポートおよびマスタークラスのIIブロック。

レポートのテーマ：

15:00 -Linuxシステムのシステム管理者の準備。 （V.M.ソロヴィヨフ）

15:30 -Djangoを使用した迅速なWebアプリケーション開発。 （レオニード・シュベチコフ）

16:00 -LiveCD-無限の可能性。 （イリヤ・ザハレンコフ）

16:30 -OpenSIPSは、SIPベースのVoIPソリューションの主要コンポーネントです。 （イヴァン・フヴァトフ）

ワークショップのトピック：

15:00 -TrueCrypt。 安全なデータストレージ。

16:00 -Djangoを使用した迅速なWebアプリケーション開発。

17:00フェスティバルの閉会。 材料の配布。



フェスティバル期間中、ロビーでインストールフェスティバルが開催され、誰でもLinuxまたはWindows用のフリーソフトウェアをラップトップにインストールできます。



注意！ Linux OSをインストールする場合は、ラップトップのハードドライブのシステムパーティションから重要な情報を保存するように事前に注意してください!!!
