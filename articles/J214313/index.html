タブレットの選択



タブレットを持っていなかったため、実験用に安価な中国の7インチLarudeタブレットが特別に注文されました。 これは、7個の「通常のTFTスクリーン、かなり高速なプロセッサ、大量の内部メモリ（1GB RAM + 8GBフラッシュ）、かなり反応の良いタッチ、ラバー」タッチプラスチックおよび取り外し不可能なバッテリー（および開いていない裏蓋）を備えたタブレットです。



理想的な「壁に取り付けられたタブレット」には、次の特性が必要です。

  1. スマートホームのマスターの身長に関係なく、情報が適切に読み取られるように、IPS画面を用意します。
  2. 平らな磁石または金属板を安全に取り付けるために、平らな後壁を用意してください。
  3. 両面テープを高品質に固定するために、「ゴム処理」のない通常のプラスチックまたは金属の背面を用意してください。
  4. ワイヤレス充電機能をサポートするか、別売のワイヤレス充電レシーバーを取り付けるための取り外し可能な背面カバーを用意してください。
