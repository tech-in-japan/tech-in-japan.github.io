Unixバージョンでタブ切り替えを構成する

すべてのプログラムで、タブはCtrl + PgUpとCtrl + PgDnの組み合わせを使用して切り替えられます。 Operaでは、デフォルトではCtrl + TabとCtrl + Shift + Tabのみが機能します。 これは設定できることがわかりました：Ctrl + F12→設定→詳細設定→管理、プロファイルを選択し、「編集」をクリックしてから、2つの新しいショートカットを追加します。



PageDown ctrl-次のページに移動

PageUp ctrl-前のページに移動



これはタブだけでなくOperaインターフェースでも機能します。 画面に収まらない幅の広いページで使用される「右ページ」と「左ページ」のアクションは、同じ組み合わせでハングアップします-競合はほとんどありませんが、オフにすることができます（幅の広いページでCtrl + PgDownを2回押すだけです）。



さらに、タブ設定で「順番に」スイッチを選択することはおそらく価値があります（ちなみに、対応する設定は「Ctrl + PageDownでタブを切り替える」と呼ばれますが、この組み合わせは提供されている3つのプロファイルのいずれでも機能せず、「Opera 9.2互換」はalt + PgDn / PgUpです。
