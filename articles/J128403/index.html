MicrosoftはWindows 8 Developer Previewをリリースしました









マイクロソフトは昨日、カリフォルニア州アナハイムで開催されたビルド会議でWindows 8を発表しました。 新しいOSが、タッチスクリーンとキーボードの両方のために設計された軽量システムで、タブレット、ラップトップ、デスクトップコンピューター間のギャップをどのように埋めるかを説明しました。



Windows 8の機能には、新しいMetroインターフェイス、高速起動時間（8秒未満）、新しいWindowsアプリストア、アプリ間通信、ARMおよびIntelベースのハードウェアサポート、ユーザーエクスペリエンスとインターフェイスの無数の変更が含まれます。それらは前任者とは著しく異なります。



マイクロソフトは、 Windows Dev CenterでWindows 8のダウンロードリンクを公開しています。 開発者のプレビュー中、Windows 8は無料になります。 32ビットおよび64ビット構成で使用できます。



もちろん、これは完成品ではありませんので、エラー、絶え間ない更新、およびプログラムの非互換性に備えてください。 一般的に、自己責任で行動してください。
