オラクルはゼロダウンタイム開発者であるKspliceを買収

Infoworld Onlineは、この動きがオラクルがRed Hatのような企業と競争したいという願望を示していると信じています。



そのため、木曜日に、巨大なOracleがKspliceを購入したことが正式に発表されました。Kspliceは、管理者が重要なセキュリティアップデート（カーネルアップデートを含む）をLinuxベースのサーバーに再起動せずにインストールできるようにする技術の作成者です。 取引の詳細は発表されていません。



オラクルのウェブサイトは、インターネットの巨人がゼロダウンタイム技術を顧客に提供する唯一の企業になる予定であるというニュースを投稿しました。これは、デフォルトでOracle Linuxプレミアム顧客に提供されます（Oracle Linux Premierを購入した人を参照）サポート）。



OS部門のCEOおよび上級副社長に代表されるオラクルのエグゼクティブ：「すでに、同社にはOracle Linuxの7,000人以上の企業ユーザーがいます。 Kspliceの導入により、この数は大幅に増加するはずです。」



Ksplice加入者への電子メールニュースレターでは、個別にゼロダウンタイムテクノロジーは今後配信されなくなると言われています。 Oracle Linux Premier Supportでのみ購入できます。 特に、KspliceはRHEL、SuSe、CentOSをサポートしなくなりました。
