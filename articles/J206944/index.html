CorelCAD 2014の主な機能



CorelCAD 2014は、.DWGファイルの組み込みサポート、業界標準のCAD機能、カスタマイズ可能な2Dおよび3D設計ツール、マルチフォーマットの互換性により、作業を改善します。 CorelCADはWindowsとMac OSの両方に最適化されており、あらゆるプラットフォームで優れた速度、パフォーマンス、適応性を提供します。



.DWGファイルを開いて保存する


すべてのCADソリューションでは、AutoCAD .DWGファイルを使用できますが、多くのファイルには独自の形式があります。 そのようなファイルを.DWG形式に、またはその逆に変換すると、図面の重要な部分が失われたり歪んだりする場合があります。 また、一部のCADアプリケーションは、サードパーティの開発者が所有する形式のファイルを開くことができません。 CorelCAD 2014では、.DWGがメインの描画形式として使用されるため、共同作業の問題が解消されます。

独自の形式の多くのCADソリューションとファイルには、.DWG形式でサポートされていない要素が含まれています。 逆に、AutoCADには、一部のアプリケーションでサポートされていない機能があり、ファイルの変換中に望ましくないタイミングの悪い問題を引き起こす可能性があります。 CorelCAD 2014は、サポートされていないAutoCAD機能に関連するファイル属性を処理します。これにより、.DWGファイル形式を使用してパートナーや顧客と作業する際の完全な互換性が保証されます。



CorelCADは、R12から2013 .DWGまでのAutoCADバージョンで作成されたファイルと互換性があります。 バージョンR12の広範な.DWGおよびDXF形式で図面を保存できるため、このバージョンのみを認識する古い機器（たとえば、レーザー彫刻装置やプロッター）を引き続き使用できます。 さらに、CorelCADを使用して、あらゆるバージョンの破損した.DWGおよびDXFファイルを修復できます。







AutoCAD 2013/2014 .DWGサポート： CorelCAD 2014は、R2013 .DWG形式を独自のファイル形式として使用します。 これにより、他のCADアプリケーションとの完全な互換性、すべての必要な機能の可用性、および.DWG形式のCorelCADファイルの属性のサポートが保証されます。 このレベルのサポートは、AutoCAD 2014またはAutoCAD 2013で作成されたファイルを受信して​​処理する必要がある場合に絶対に必要です。AutoCADを使用する顧客やパートナーと連携しながら、完全なファイル互換性を提供します。

使い慣れたCADインターフェイス：他の一般的なCADアプリケーションを既に使用している場合は、CorelCADに切り替えるためのトレーニングは必要ありません。 そのようなユーザーはすぐに使い始めることができます。 コマンドラインを含む使い慣れたCADユーザーインターフェイスのおかげで、ユーザーは使い慣れたコマンドとキーボードショートカットを使用してすぐに始めることができます。



新しい機会！ テープインターフェイス： CorelCAD 2014では、特定のワークフローの効率を向上させるために設計された2つの新しいテープワークスペースが導入されています。 ワークスペース「3Dモデリング」と「ドラフトと注釈」は、便利なリボンでツールとコマンドを整理し、アプリケーション機能へのアクセスを簡素化します。

使い慣れたリボンインターフェイスオプションは、新しいCorelCADユーザーが他のCADアプリケーションをAutoCADから移行するのにも役立ちます。 さらに、カスタムワークスペースファイル（CUIX）を簡単にインポートし、ツールと関数をCorelCADリボンに直接添付できます。

WindowsとMac OSの両方で高いパフォーマンス


CorelCAD 2014では、WindowsオペレーティングシステムとMac OSの両方で迅速かつ効率的に作業できます。

オペレーティングシステムの選択の自由： CADアプリケーションは、ほとんど常にWindowsオペレーティングシステムで実行するように設計されていますが、CorelCADはこの伝統を変えています。 ユーザーが好むオペレーティングシステム（MacまたはWindows）に関係なく、CorelCADを使用すると、迅速かつ効率的に作業できます。 Mac OSの一部のCADシステムの機能はWindowsの同等のものと比較して制限されていますが、CorelCADソリューションはWindowsとMacユーザーにほぼ同じレベルの機能を提供します（特に指定しない限り、すべての機能は両方のオペレーティングシステムに適用されます）。 WindowsおよびMacintoshオペレーティングシステムでのアプリケーション機能の比較については、15ページの「WindowsとMac OS用のCorelCAD 2014の比較」を参照してください。

改善されました！ Windows 8.1およびMac OS Mavericksのサポート：CorelCAD 2014はWindows 8 for PCをサポートしています。 Macユーザーは、全画面表示など、Mac OS Xの新しい機能を利用できます。 さらに、CorelCAD 2014は高解像度操作用に最適化されており、高解像度モニター上で明確で読みやすいユーザーインターフェイス要素を提供します。

改善されました！ 64ビットシステムの組み込みサポート： WindowsとMac OSの両方のユーザー向けに、CorelCAD 2014は、大きなCADファイルの効率的な処理を可能にする64ビットCAD環境を提供します。 さらに、32ビットバージョンは、より強力でないハードウェアで作業する必要があるWindowsユーザー向けにも含まれています。

改善されました！ アプリケーションの速度とパフォーマンス： CorelCAD 2014 64ビットアーキテクチャは、WindowsおよびMac OSプラットフォームでのアプリケーション速度を向上させます。 また、このアプリケーションはより高速な画像回復を提供します。これにより、大きく複雑なCAD図面を操作する際のスケーリングとパンがスムーズになります。



生産性を向上させるチューニングと自動化


CorelCAD 2014のインターフェイスを設定するには、プログラミングの知識は必要ありません。 タスクを自動化したり、アプリケーションに新しい機能を追加したりすることもできます。

自動化サポート：ユーザーは、LISPおよびMicrosoft Visual Studio Tools for Applications（VSTA）*プログラミングインターフェイスを使用して反復タスクを自動化し、独自の関数、ルーチン、およびプラグインを作成することにより、生産性を向上させることができます。 さらに、CorelCAD用のサードパーティ製アドオンと拡張モジュールをプラグインストアで購入して、アプリケーションに新しい機能を追加できます。

* Windowsバージョンのみ。

Microsoft Visual Studio Tools for Applications（VSTA）のサポート： VSTAマネージャーツールバーを使用すると、VSTAマクロを記録、編集、および実行できます。 Windowsユーザー向けに、CorelCAD 2014は組み込みのVSTA機能を提供しています。これにより、日常的なタスクと繰り返しコマンドの実行を自動化できます。 さらに、VSTA開発環境との統合により、複雑なマクロを作成するための追加機能が提供されます。

改善されました！ LISPサポート： CorelCAD 2014では、LISP手順を操作するためのアイコンメニューサポートが提供されるようになりました。 これにより、CADコンポーネントのサムネイル（SLD）を表示するカスタムダイアログを作成できます。 これは、カスタムライブラリを作成するときに特に便利です。 さらに、CorelCAD 2014は、LISPでのプログラミングの共通フレームワークに準拠しているため、再コンパイルせずに既存の手順を再利用できます。

改善されました！ ユーザーインターフェイスのカスタマイズ： CADを使用するすべての業界には、独自のワークフローがあります。 そのため、CorelCAD 2014はさまざまなセクターやプロジェクトのニーズに合わせて簡単にカスタマイズできます。 プログラミングの知識がなくても、特定のタスク用にユーザーインターフェイスをカスタマイズできます。 その後、カスタムワークスペース構成をユーザープロファイルに保存し、組織全体に展開できます。 これにより、ユーザーインターフェイスを特定のプロジェクトに適合させ、生産性を最大限に高めることができます。 さらに、CUIXファイルをインポートできるようになりました。これにより、ユーザーワークスペースで作業するAutoCADユーザーにとって簡単になります。
