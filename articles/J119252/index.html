ワイヤレスカメラまたはWiFiジャマー？

古くから、さまざまな状況で敵をスパイするために、ストライクボールゲーム用に安価なワイヤレスカメラが購入されました。 しかし、その主なアクションに加えて、副作用が突然現れました...







このようなキットを購入できる店舗サイトの説明は、あまり有益ではありません。

ワイヤレスラジオキット、屋内設置用に設計されたカラーカメラ、2.4G搬送周波数、送信機電力50mW、AV信号をモニターに接続するための受信ベース、および2つの電源（カメラとベース用）。


製造業者が話さないサイトについて。 ただ中国。



仕様：

モデル：RES LYD-208C 2.4G

カラーアナログ入力/音声付きカメラ〜380 TVL、ワイヤレス、IRなし。 電源+ 8V。

最大100mの範囲で直接視認できる範囲（実際にテスト済み）。

動作電流：100mA



カメラの作業チャネル（カメラ自体の2つのジャンパーによって切り替えられます）：

1チャンネル-2.414 Ghz

2チャンネル-2.432 Ghz

3チャンネル-2.450 Ghz

4チャンネル-2.468 Ghz



この赤ちゃんは、壁や床の天井を含む最大50mの範囲のWiFiネットワークを迅速かつ効率的に配置します。 すべてのカメラ周波数が4つのチャネルのいずれかで動作するWiFi周波数と重複するわけではないという事実にもかかわらず、カメラはその周波数のいずれかにワイヤレスネットワークを配置します。 カメラが「十分に届かず」、WiFiネットワークが落ちない場合、データ転送速度はすでに大幅に低下しています。 そして、これは製造業者によって意図的な損害とは考えられていませんでした。 アセンブリと部品の安さのために、カメラの周波数はその範囲内で強く移動し、それによってノイズが発生し、その結果としてそのような結果が生じます。



現場では、Kronaバッテリーは約2時間持続します。



キットの価格は32 cuからです。



実験



この実験では、ビデオモニターとしてアナログビデオ入力を備えた、Windows CE 5.0に基づく車のGPSナビゲーターを使用しました。 ビデオ信号がA \ V入力に表示されると、カメラの画像が自動的にオンになります。



テストスタンド：

-WiFi D-Link DIR 615 Rev Cアクセスポイント（ファームウェアDD-WRTを使用 ）、7dBiのゲインを持つ2つの全方向性アンテナ、送信電力100％。

-Intel Wifi Link 5100 AGNラップトップのWiFiカード。



アクセスポイントは混合モード（802.11b + 802.11g + 802.11n）で動作します。 接続は802.11nモードで確立されます。 何でも構いませんが。



アクセスポイントからラップトップまでの距離<3m。



何が起こっているかの簡単なビデオレビュー（ HDで利用可能）：





また、この実験は、アクセスポイントまで約30 mあり、直接の可視性がないオフィスで実施しました。 結果はすべての期待を超えています。 ラップトップ、スマートフォン、PDAなど -すべてがWiFiなしで残ります。



2.4 Whzの周波数で最大1.5 Wの送信機電力を備えた同様のカメラがあります。 これには、いわゆる「ベビーモニター」/「ベビーモニター」デバイスも含まれます。



競合他社のオフィス、またはオフィスセンターの廊下、吊り天井のどこかに設置されているため、WiFiネットワークの故障を探してかなりの数の幸せな時間をお届けします。 特に、有線イーサネットネットワークがプロジェクト内にない場合。 または、吹き抜けの隣人が庭のガレージのビデオ監視をインストールすることにしました。 そして、Wai-Faiポイントは、誰かが「バグ検出器」\「フィールド検出器」を購入することを推測するまで、顔が青くなるまで変更できます。
