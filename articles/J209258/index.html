無料のWeb開発サービスの概要

投稿とは、便利で必要なオンラインサービスの個人的な選択であり、これを使用すると、本当にクールなものを無料（またはシェアウェア）で作成し、Web上に展開できます。 私は新しくて完全なふりをするつもりはありませんが、誰かが役に立つと嬉しいです。



私たちは皆、新しいアイデアを試したいと思います。manulovについての小さなWebサイトを切り開きますが、node.js、highloader redid、memekesh、mongo、feylover、および世界支配の主張を確実にするために。 良いニュースがあります。酷評された西洋の資本家の寛大さのおかげで、これはすべて無料で入手できます。



Github





世界支配の獲得への道に着手したすべてのITファイターが最初にすべきことは、githubでアカウントを作成することです。 トピックに誰もいません-これは分散型vcs gitに基づくオンラインバージョン管理システムです。 githubには強力なソーシャルコンポーネントがあります。他の人のリポジトリと変更をサブスクライブし、プロジェクトをフォークする（つまり、変更を加えてソースコードのブランチを作成する）ことができます。 プルリクエストを使用してお気に入りのソフトウェアの開発者に変更を提供するために、シンプルなバグトラッカー、コードレビューがあります。 Githubにはsvnと対話する機能がありますが、svnは祖父用です。 リポジトリを索好きな目から隠したいまで、Githibは無料です。 しかし、あなたは隠れて恥ずかしがり屋ではありませんよね？ githubの代替品（例： bitbucket ）は存在しますが、コーシャではありません。



モッキンバード





Mockingbird Mockingbird（モックアップ-アウトライン、レイアウトから）-アウトラインインターフェイスを作成するためのオンラインツール。 ブラウザでボタン、入力フィールド、スライダー、その他のトリプをドロップできます。 インターフェイスのプロトタイプを作成し、レイアウト設計者のタスクに適用すると便利です。 10ページのプロジェクトは1つだけ無料で実施できますが、これで十分な場合がよくあります。



Jsフィード





JSFiddle-js、css、htmlを試すためのオンラインツール。 ワークスペースは4つのウィンドウ、つまりhtml、js、css、および結果です。 最初の3つでは独自のコードを記述し（構文の強調表示があります）、4つ目では操作可能なレンダリング結果があります。 多数のバージョンのjsライブラリ（jquery、angularなど）がサポートされています。 たとえば、さまざまなバージョンのjqueryでこのチップまたはそのチップがどのように機能するかを確認すると便利です。ドロップダウンリストをクリックするだけで切り替えることができます。 すべては愛のためだけ-お金はありません。



SQLフィドル





SQLFiddleは、アイデアとsqlコードをチェックするためのクールなオンラインツールであり、さまざまなDBMSがサポートされています。 Mssql、sqlite、mysql、postgresql、oracleなど。クエリを確認する必要がある場合は、クエリ実行プランを確認し、最適化してください。 システムでの作業は、スキームの作成と実際のクエリとdmlの実行という2つのフェーズに分かれています。 ちなみに、保存されたファイルを作成して実行することもできます。 すべて無料。



ヘロク





Herokuは、古典的なPaaSアプリケーションホスティングです。 従来のvpsホスティングとは異なり、ルートシェルおよびその他のハードコアを備えた仮想マシンへの直接アクセスは提供されませんが、事前構成された環境でアプリケーションを公開するためのツールキットが提供されます。 つまり、一部のWebサーバー、ある種のバランサーは既に透過的に設定されています。環境のタイプとバージョン（node.js、pythonなど）を指定し、git経由でアプリケーションをアップロードする必要があります。 1つのWeb dyno（アプリケーションインスタンスと呼ばれる）が無料で提供されます。 PaaSの優れた点は、プロバイダーが低レベルの構成に煩わされることです。ユーザーはアプリケーションのみに関心があります。 Herokuには、いわゆる アドオン-memcached、mongodb、mysql、postgres、rabbitmq、sphinx、その他の開発者の喜び。 あなたが言う必要がある-私はそれが欲しいとそれが表示されます。 実行可能、カスタマイズ、および保守。



Cloud9 IDE





クラウド9は、ブラウザー内のほぼ完全な開発環境です。 LAMP + Eclipse / Netbeans / Zend / Whateverなど、デプロイメントと構成を気にする必要はありませんが、すぐにブラウザーを開いてコーディングを開始してください。 C9はgithubおよびherokuと見事に統合され、sftpに展開できます。 実際の環境に加えて、完全なLinuxコンソール（ブラウザーに直接）があります。 自由に使えるのは、RHEL 6.4を備えた仮想マシンです（明らかにc9.ioはOpenShift.comでホストされています）。 iPadでも完全にコーディングできます。 無料版の制限は、仮想マシンおよび1つのプライベートプロジェクトに対して128 MBです。開いているプロジェクトの数は制限されていません。



トラビスci





Travisは、githubリポジトリと統合し、コミット時にテストスイートを実行する継続的統合システムです。 たとえば、ステージングブランチとプロダクションブランチでコミットテストを渡すときに、異なるherokuドメインへのプロジェクトの自動展開を構成できます。 .travis.ymlファイルがアプリケーションのリポジトリに追加されます-テストを実行するために必要なものが記述されています。 オープンリポジトリの場合-完全無料。



ソースラボ





SauceLabsは、クラウドでのセレンテストです。 セレンは、ブラウザのテストを制御するものです。 スクリプトでは、クリックする場所と書き込む内容を記述し、セレンはブラウザを起動し、記述されたアクションを実行して結果を確認します。 彼らは、さまざまなブラウザの9000以上のバージョンで100500の仮想マシンをセットアップし、セレンをそれらにねじ込みました。 ソースラボで受け取ったアドレスとアクセスコードを示すセレンテストを書くだけで、さまざまなIE、Opera、Safariなどのクラウドでテストが開始されます。テストにログが書き込まれ、スクリーンショットとビデオが撮影されます。 無料のアカウントでは、win / linuxで100分間、macで40分間、poke penで30分間の自動テストを行います。



クラウドフレア





Cloudflareは、1つのボトルでCDNおよびDDOSを保護します。 ドメインをサーバーのIPに転送する代わりに、Cloudflareサーバーに委任します。サーバーに送信するリクエスト、キャッシュから送信するリクエスト、およびキャプチャをリクエストしてブロックするユーザーを既に決定しています。 これはすべてカスタマイズでき、さまざまな最適化と最小化が含まれます。 ただのおとぎ話。 基本機能は無料です-追加料金とDDOS保護のより高度な機能に料金がかかります。
