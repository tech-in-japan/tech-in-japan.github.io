The main announcements of the Microsoft Ignite 2019 conference





At the largest annual conference Ignite 2019, Microsoft introduced a wide range of new tools and services for business development and more productive work of organizations and employees. In this post we have collected the main announcements. Look under the cat.



By the way, there is an interesting table with all the reports of the conference. Learn it here .



Azure arc







Launch of a preliminary version of the solution for creating integrated hybrid environments. Azure Arc provides the ability to leverage Azure‚Äôs unified services, management tools, and security tools across any infrastructure, including multi-vendor clouds such as Amazon and Google, on-premises servers, and edge computing devices.



New Microsoft Edge







The Microsoft Edge browser on the Chromium engine for Windows and macOS will be publicly available on January 15, 2020 in more than 90 languages. Today it is available for trial download . Read the details here .



Office Universal Mobile Application







The versatile Office mobile application combines Word, Excel, and PowerPoint, eliminating the need to install separate applications. With it, you can create and edit documents, including using the ability to convert photos to text or a table. The application is available as a public evaluation version today for Android and iOS users.



Project silica







Microsoft and Warner Bros. recorded the 1978 Superman movie on quartz glass . The companies presented the first confirmation of the Microsoft Research Project Silica project concept. It uses the latest advances in laser optics and artificial intelligence to store data.



Azure Blockchain Tokens







The solution allows you to tokenize physical or digital assets, manage them and use them on blockchain platforms of various types. Using Azure Blockchain Tokens, blockchain developers will have access to Token Taxonomy Initiative (TTI) standards or the ability to use the template to create the four most popular types of tokens. The service provides access to a set of open APIs for interacting with tokens. Among the scenarios for the use of such tokens in business: digitalization of the supply chain, loyalty programs, workflow and much more. At the moment, the solution is available in a preliminary version.



Project cortex



The first new Microsoft 365 service since the launch of Microsoft Teams. This AI tool is a cross between a corporate content classifier and a digital assistant. An intelligent service is able to automatically find and distribute information on topics, for example, in Office, Outlook and Microsoft Teams applications, identify experts on various topics and create an organization‚Äôs ‚Äúknowledge network‚Äù. The solution will be publicly available in the first half of 2020, at the moment it is in closed testing.



Insider Risk Management for Microsoft 365







Intelligent security solution for your organization. It is able to anonymously analyze and detect cases of atypical and suspicious user behavior (for example, actions with files) using machine learning algorithms, which helps to prevent many risks that cannot be detected by standard tools and methods. While the solution is available in closed testing mode.



UI Flows







A public preview of the process automation service in Power Automate (formerly known as Microsoft Flow). The new solution creates interface robots for automation in any application, including those that do not support integration via API. Thus, UI Flows can significantly simplify the automation of processes in the organization, since it does not require programming knowledge to configure it: the user only needs to demonstrate the execution of the action that he would like to automate.



Stream Voice Recording Improvement



The Microsoft Stream video streaming service now uses machine learning algorithms to detect and eliminate unwanted background noise in any video with one click of the mouse.



Teams chat button in Outlook







Allows you to conveniently correspond by switching between conversations in Outlook and chats in Teams.



Azure Synapse Analytics







A new service that extends the capabilities of Azure SQL Data Warehouse. Azure Synapse Analytics aggregates information from all data sources, storage and big data analysis systems. It will allow companies to use their data much faster and safer.
