Visual Studio用Mono Tools 2がリリースされました





Miguel De Icazaは、彼のブログで、 Visual Studioの Monoプラットフォーム用の開発ツールの新しいバージョンのリリースを発表しました 。



MonoTools for Visual Studioを使用すると、Visual Studioを使用してモノラルアプリケーションを作成し、ローカルWindowsマシンまたはリモートLinux-およびMacOSマシンでデバッグし、 Linuxで展開するパッケージを作成できます。



MonoTools 2の新しいバージョンには、次の変更があります。
  • 新しいソフトデバッガーのサポートが追加されました。
  • アプリケーションの展開システムへの転送速度の向上。
  • Visual Studio 2010のサポートが追加されました。
  • バグの修正とその他の改善。
Visual Studioツール用MonoTools 2は、 公式サイトからダウンロードできます。
