ロシア製の機器。 パート2. LANTASTICa

最初の部分では、 M-Links-私の手にあったロシアのE1-Ethernetコンバーターについて話しました。 次に、サンクトペテルブルクの会社Optical TeleSystemsの製品について説明します。

Lantastika-2Speed-いわゆる「適応ブロードバンド無線光通信システム」。 名前には秘密の意味が隠されています。デバイスの機能は、通常のイーサネットを赤外線に変換することです。







鉄


テクニカルパスポートから、このシリーズのLANTASTIC BOX 100M-AC1、BOX 100M-AC3、BOX 100M-AC3（BOX = Wireless Optical Communication Channels）の3つの製品があることが明らかになります。 それらは特性が異なります。 実際、これはデバイスが動作できる距離で反映されます。 したがって、AC3は、それらによると、互いに最大480 kmの距離に配置できます。 パスポート。 率直に言って、宇宙空間と自動位置合わせシステムはすぐに表示されます-理想的な条件でのみ、そのような距離で作業が可能です。

このデバイスは、受信モジュール（中央の大きなレンズ）、2つの送信モジュール（メインの側面）、保護ケース、および取り付け脚で構成されています。 垂直または水平面への取り付けが可能です。 脚の端にはボールがあり、4つのボルトが付いたリングによってケーシングが引き付けられます。 このマウントは、反対側の粗調整を提供します（同じデバイスが反対側にあります）。 ケーシングには2本の水平調整ボルトがあります。1本は垂直ボルト、もう1本は固定ボルトです。







中央のレンズの後壁には、4つの信号強度インジケータ、パワーインジケータ、ローカルおよびリモート同期のインジケータ、イーサネットリンクインジケータ、ケーブルリード、およびサイトを取り付けるための穴があります。 ローカル同期インジケータは、デバイスが反対側からの信号を確認することを意味します。 赤の場合、接続速度は10 Mb / s、緑-100 Mb / sです。 リモート同期の点灯インジケータは、反対側がビームを見て、それについての信号をローカルビームに送信したことを示します。







電源ケーブルとシールド付きツイストペアケーブルは、デバイスに適しています。 キットには、220V電源（48V電源への直接接続も可能）、予備のポインティングデバイス（サイト）、調整ネジ、台座付きのジャンクションボックス、電源用の接地バスとコネクタ、および2つのポートを持つボックスが含まれます：データ送信用（緑とオレンジのペア）と制御用（RS232、青のペアと画面）。 写真では、これがボックスです。







難しさ


最初のポートをすばやく見つけましたが、余分なペアとして画面を使用することは新しいことでした（おそらく、ちょっとした経験のコストかもしれません）。 最も興味深いのは、茶色のペアを使用することでした。 LANtastIKiファームウェアを更新する必要があります。 正常に動作させるには、ペアを閉じないでください。 私はこれを知りませんでした、そして、カップルがたむろしないように、私はそれをベースボードに打ち込みました。 台座から、蒸気はそれぞれ詰まっていない箱に入り、どこにも閉じられなかったので、その時点で未知の条件が満たされたとしても。 長い間、私たちはこれについて戸惑っていましたが、何も機能しませんでした。 そして、ジャンクションボックスのペアがベースから削除されるまで機能しませんでした。 サプライヤーは状況についてコメントできませんでした。 ある種の素晴らしい。

このデバイスは、シベリアの厳しい条件で動作するように設計されています：湿度が-50から+50です。 -30でのテストは成功しました。 しかし、雪はレンズの上に安全に置かれます。 これまでのところ問題はありませんでしたが、定期的にクリーニングする必要があることが判明する場合があります。

別の問題は同期です。 おそらく、これが初めての経験なので、同僚と私は調整に数日を費やしました。 アルゴリズムは次のとおりでした：最初に、光学照準器の助けを借りて、LantastICを反対のポイントに向け、粗調整ボルトを固定し、次に、反対側の人が自分の目で赤い斑点を見るまでS字型（蛇型の方が正確です）の軌道に沿って微調整ボルトを動かします。 デバイスは約500メートルの距離に配置されました。 このような距離では、スポットの直径は約70 cmであり、これは場所から場所へ移動した場合に確認できます。特定の瞬間に、明確な赤いスポットが消えます。 次に、電話でどのように、どこに移動するかを反対側の人に伝えました。

最初のページでは、構成コマンドの基本情報。 上で述べたように、コマンドSA （自動モード）、 S10およびS100によって構成される2つの動作モード10および100 Mb / sがあります。 反対のデバイスで異なるモードが構成されている場合、同期は機能しません（個人的な苦い経験）。

残念ながら、デバイスのリモート制御は不可能であり、監視もできません-サイト訪問が必要です。

短いテスト期間（2週間）の間に、デバイス間の接続はすでに2回失われました。 1回目はすべてが自然に回復し、2回目は少し離れて方向を修正する必要がありました。 しかし、これは設置面や設計の沈下に対する機械的影響によるものだと思います-テストは継続されます。 製造業者によると、雨や雪は作業の妨げにはなりません。霧は品質を低下させる可能性があります。 ランタスティカは雪でテストに合格しました。 そうでなければ、苦情はまだありません。 LANtASTICaは100 Mb / sをほぼ満たしています（ iperfテストでは80-90 Mb / sが示されました）。 このロシアの喜びは、カップルで約30万ルーブルの価値があります。



メーカーの言葉


LANtastika-3Speedは、すでにEthernet 1000 Base-TX接続、独自のIPアドレスの利用可能性、telnetアクセス、さらにはWebインターフェイスを約束しています。

製造業者は、デバイスの絶対的な安全性を宣言しますが（ドキュメントのどこかに、目との偶発的な接触を避けるべきであると言われていますが）、デバイスの使用を許可する必要はありません。

ドキュメントは美しいです。 必要なツールの特性まで、すべてが詳細かつ明確に説明されています。 「落下することなく、すべてのモジュールの球面ワッシャーと一緒に固定ネジを完全に緩める」などの注意事項が個別に記載されています。 軽微な欠陥やスペルミスについては、目を閉じることができます。
