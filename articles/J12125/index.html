Windows Mobileがミニカーに到達

先日、フィアット自動車の懸念は、フィアット500という簡単な名前で新しいシティカーを実証しました。マシンは非常に普通です：コンパクトなボディ、3つのドア、100馬力のガソリンエンジン。 一般に、このクーペは多くのハイテクブログの注目を集めることはほとんどありませんでしたが、車内にある些細なこと（Designed for Windows Mobileネームプレート）が1つでもなかったとしてもです。



結局のところ、Fiat 500はMicrosoftと共同でイタリア人によって実際に開発されました。 後者のおかげで、この車はBlue＆Meシステム、つまり、高度な機能を備えたナビゲーションシステム、音声制御、Bluetoothアクセサリとの互換性、さらには一部のUSBデバイスなどの機能を獲得しました。 もちろん、これはすべてWindows Mobileオペレーティングシステムに基づいて動作します。 それでも、BSODについてのジョーク愛好家は心配する必要はありません。運転パフォーマンスに影響を与えないマルチメディアシステムの導入を超えたMicrosoftの活動は広がっていません。 いずれにせよ、フィアット500のクリエイターはそう言います。





何らかの理由で、開発者は鉄のコンポーネント（プロセッサ、ナビゲーションモジュールなど）について沈黙しています。 確かに、「ハードウェアレベルで」車がさらに多くの変更に耐えることができます。 ミニカーのコストについては、ヨーロッパでは1万から1万5,000ユーロを要求しています。
