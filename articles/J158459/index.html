Windows 7用のInternet Explorer 10プレビューが利用可能になりました

本日、約束通り、Windows 7用のInternet Explorer 10のプレビューバージョンが利用可能になりました。ダウンロードページへのリンクはこちら 、Windows 7 SP1 x64のロシア語バージョンへの直接リンクはこちら （x86はこちら ）。



午後、リンクはここにありましたが、ページは存在しませんでした。夕方までにリンクは完全に削除され、深夜近くに再びアクセス可能になり、機能するようになりました。



UPD。 注目すべき変更点：

1）ページおよびブラウザ全体のレンダリング速度が大幅に向上しました。

2）幸いなことに、内蔵フラッシュはありません。

3）残念ながら、 Windows 7のクールな凸型スクロールバーは、Webページ上のボタンとコントロールのWindows 8スタイルにフラットになり、Windows 8のそれに完全に類似しています。

3） Windows 8のバージョンから理解できないグリッチが残っていました-複数のタブから1つのタブを閉じると、すべてのタブが閉じたタブと一緒にタブの行全体を占めるサイズに拡大し、再び通常のサイズに 狭まり ます UPD：これは機能の1つです：右端のタブを横切り、5回言うと押すと、最後の5つのタブが閉じます。 hf35に感謝します 。
