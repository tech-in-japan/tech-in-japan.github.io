開発センターS + Sロシア

アクションへのコメントによると、 Hyperpodorkは、Hyper-Vのテストと開発のために、SaaSサービスの開発者がそれを取得したいことを発見しました。

ですから、特に彼らにとっては、BizSparkプログラムの一環として昨年開設されたS + S開発センター（Microsoft Software + Service Incubation Center）についてもう一度思い出したいと思います。





センターは、SaaS / S + Sモデルに基づいたサービスの立ち上げにおいて独立系ソフトウェア開発者（ISV）を支援するために、MicrosoftとParking.Ruによって作成されました。



S + S開発センターは、サービスの立ち上げのためのコンサルティングと準備の段階で無料でサービスを提供します 。 準備プロセス中にデバッグとチューニングにテスト技術インフラストラクチャが必要な場合- 特別な優先条件で提供されます



センターサービスは次のとおりです。



-コンサルティング、SaaS / S + Sサービスの形成、ポジショニング、価格設定ポリシー、市場プロモーション、およびホスティング環境でのこれらのサービスの機能に関する技術的側面。



-私たち自身と世界の他のS + S開発センターの両方での実務経験（メソッド、ドキュメントテンプレート、例など）。



-テクノロジーインフラストラクチャ（プロビジョニングシステム、課金システム）、SaaS / S + S ISVソリューションの展開と実装の準備ができています。



-SaaSカタログにSaaSCatalogソリューションを配置！
