言語指向プログラミングの使用例

言語指向プログラミング（LOP）の考え方は、プログラムの開発中にミニ言語が絶えず作成されるということです。 これらは、メインの開発言語を拡張することも、別々の言語にすることもできます。 LOPの最適な言語はマクロを備えたCommon Lispですが、これはそれについてではありません。 Common LispでLOPを使用する例については、Peter Seibel Practical Common Lispによる素晴らしい本をご覧になることをお勧めします。 LOPは最も簡単で効果的なプログラミング方法の1つであると考えています。 これに最も適切な言語でタスクとサブジェクト領域を説明し、それを実装しようとします。



私はブラウザーゲームをRubyで開発しているため、言語の拡張と組み込みDSL（Rubyを使用するとこれを非常にうまく行うことができます）、および複雑なゲームメカニクスに関連するミニ言語の作成の両方にLOPをよく使用します。 この記事では、メイン言語の単純な拡張機能である組み込みmini-DSLと、2つの非組み込み言語について説明します。 私は自分に近いトピックで例を挙げます。それらがかなり理解できることを願っています。

制限を示す組み込みDSL





ゲームにはさまざまな部屋があり、すべての人が許可されているわけではありませんが、プレーヤーが特定の規則を順守している場合のみです。 たとえば、レベルが10を超えているか、デッキ内のカードの数が8を超えていません。このような制限は組み合わされています。 たとえば、「プレイヤーレベル> = N」などの制限の種類があり、「プレイヤーレベル> = 13」という特定の制限があります。



DSL define_constraintを使用して制約のタイプを指定し、制約とその組み合わせをデータベースに保存できます。 



   define_constraint "deck_sum_between"、 "％Nと％Mの間のカードレベルの合計" do
     （引数['N']。to_i..arguments ['M']。to_i）.include？（context.deck.sum_of_card_levels）
  終わり


   define_constraint "deck_without_duplicates"、 "テイクのないデッキ"は
     ！context.deck.has_duplicates？
  終わり

   define_constraint "user_level_ge"、 "プレイヤーレベル％X以上"
     context.level> =引数['X']。to_i
  終わり




制限の各タイプでは、名前（deck_sum_between）、「％Nと％Mの間のカードレベルの合計」という説明を設定します。この説明から、パラメーターに基づいて、特定の制限の説明が取得されます。 そしてもちろん、プレーヤーまたは他のオブジェクトが制約に適合する場合にtrueを返す制約の実装。 したがって、システムは普遍的であり、ユーザーではなくコンテキストです。

結果として、deck_sum_between（N => 10、M => 20）のように制限を記述したり、オブジェクトのさまざまなプロパティに名前とパラメーターを保存したりできます。

ブースタールール記述言語





ブースターは、収集可能なカードゲームで販売される一連のカードで、特定のルールに従ってランダムなカードがいくつか含まれています。 たとえば、沼地住民の5つの中規模マップと1つの良好なマップ。



カードを生成するための各ルールは、テキストで説明できます。



希少性（1）-不正なカード（番号はルールに設定されています。詳細は後ほど）

希少性（1）|氏族（6,7,8）-氏族6,7,8の不良カード1枚。 「|」 ここでは、論理ORではなく、UNIXパイプラインを象徴しています。



確率でルールも可能です：



クラン（1,2,3）|期待（1,60,2,38,3,2）-クラン1、2または3のカード。 60％の確率で-悪い、38％の確率で-中、3％の確率で-良い。



各ルールは、次のようなActiveRecordスコープメカニズムに基づいて実装されます。 



   def rule_clans（スコープ、ID）
     scope.scoped_by_clan_id（ids）
  終わり
  
   def rule_expectance（スコープ、パラメーター）
     scope.scoped_by_rarity（Expectance.for_expectance（ハッシュ[* params.map（＆：to_i）]）））
  終わり
