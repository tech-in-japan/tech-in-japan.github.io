Oracle Certified JAVA Associate試験の準備

OCAJP 7に提出するためのロシア語ガイドの検索は成功しなかったので、私は自分で書くことにしました。



このビデオでは、どのような証明書が説明されています。 誰もが自分で引き渡すかしないかを決めますが、この証明書はJavaに興味がある人にとってOracle階層の基本です。



私自身、OCPを利用して、まず知識を整理し、次に雇用主の魅力を高めることにしました。



CodeRanchを読むことからいつものように始まりました。 OCA 7に関する本の準備を始めたとき、まだ自然ではありませんでした。 公式ガイドが登場しました。 しかし、値札（$ 30）を考えると、私はまだ購入を控えます。 では、その理由を説明します。



まず、ハンドオーバーに必要な情報のほとんどは、SJCP 6の準備でカバーされます。そのほとんどは無料で入手できます。 フォーラムは、多くのJLS JAVA 7が試験に追加されたことを怖がらせています。 いずれにせよ、JVMの新しいバージョンに追加されたチップは、それらの多くがなく、試験のためにかなり一般的な理解があるため、夕方に学習できます。



私はこの素晴らしい本からJAVAへの旅を始めましたが、試験に適さないでしょう。



91％を合格しました。次の資料を使用しました。

  • K＆B SCJP 6スタディガイド 。 ここで繰り返し言及しました。 本は本当に素晴らしいです。読みやすく、よく構成されており、各章の後に、資料を統合する演習があります
  • ここから SCJP 6に合格するためのテスト。 しかし、私は彼らに助言しません、なぜなら 質問はかなり時代遅れです。
  • Enthuwareによる模擬試験（10ドル）


準備計画は次のとおりです。

  • K＆Bの1〜5章と10章をお読みください。 実際の試験のトピックに従って選択的に読むための第6章と第7章
  • 同じトピックでSCJP 6のテストを解決する（私が言ったようにあまり有用ではない）
  • Enthuwareからのテスト（6チケット、500以上の質問）


次のようになりました：

  • 私は1日1冊の本から章を読もうとしましたが、それは章ごとに2日でしたが、課題と小さな実用的な課題を解決しました
  • 最初はモックテストは簡単すぎるように見えましたが、後で判明したように、各チケットはいくつかの特定のトピックに向けられていました。 すべてのチケットが解決されて初めて、全体像が形になりました
  • 最初は週に10ドルでテストをクリックする予定でしたが、最終的にはたった2回だけでした。 計画は、交通警察のチケットと同じでした。チケットは初めて解決され、その後、すべてのミスが解決され、再び解決されます。
  • 実際の試験の質問と比較して、テストの質問ははるかに複雑です。 最初からチケットを60〜70％解決できなかった場合は、もう一度本を読むことをお勧めします。 交通警察で同じチケットを取得する方法を学ぶことはできません。テストの質問は実際の質問とはまったく異なります
  • ただし、実際の試験と同じトピックに重点が置かれています。 最も難しい質問は、「String、StringBuffer、StringBuilder」および「Exceptonsの処理」に関するものでした。 回答のテストの作成者は、Oracleによって準備が必須であると示されていないトピックが実際に試験に存在することをほのめかすことがよくあります。 本当にそうです。 ガベージコレクターについていくつか質問がありました。 実際の試験のテストがいっぱいになったJAVAのデータ型のキャストおよびディメンションの知識に関する「バイトを短くする方法」などの迷惑な質問
  • 試験の前に、地下鉄で旅行している間、K＆B 6のすべての章の要約を読み直しました。何度も役に立つことがありましたが、微妙なニュアンスに関連する質問がありました。


周囲の騒音を聞かないために、私は閉じたヘッドフォンを私と一緒に持って行きました。彼らは本当にとても役に立ちました。 テストクラスには多くの人はいませんでしたが、彼らは絶えず出入りしていたため、生産的な方法でセットアップされませんでした。 試験は150分間続き、気を散らされたくありませんでした。 ヘッドフォンは観客のすべてのノイズを完全に遮断します。



すべての質問に答えた後、私はまだ40分残っていたので、再確認しました。 迷惑な間違いが1つ見つかりました。



テストの後、1つの肯定的な気持ちがありました。準備を終わらせるのは良いことです。 それ以前は、別のベンダーの認定を取得しましたが、テスト用のソフトウェアとテストに合格したコンピューターには常にいくつかの問題がありました。 今回も、質問の読み込み中に各フェードが消えるにつれて、心臓はより速く鼓動し始めました。 しかし、すべてがOKです。 PearsonVUEのソフトウェアはバグがなく、すべてのテストは問題なく合格しました。



皆さんに幸運を！ 次回は、OCPJP 7の準備と配信についてお話します。
