CISで拡張されたGoogle I / O



親愛なるhabravchane、IT専門家、開発者、および技術に無関心でない他の人々！ おそらくご存知のように、有名な毎年恒例のGoogle I / O会議は、5月10日から11日まで、遠く離れたカリフォルニア州サンフランシスコで開催されます。 最先端のGoogleのデベロッパーが集まり、最新のおいしいテクノロジーについて語ります。 この会議は非常に人気があり、チケットはイベントの数ヶ月前に完売します。 会議に参加したいすべての人を支援するために、GoogleはGoogle Technology User Groupsと協力して、世界中のさまざまな都市でイベント「 Google I / O Extended」をストリーミングしています。
