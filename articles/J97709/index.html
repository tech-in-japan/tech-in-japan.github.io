Windows 8の製品ライフサイクル

次のスライドはあまり明確ではありませんが、MicrosoftがOS開発のアプローチを3つの部分に分割する方法を検討するのに役立ちます。
  • 計画（構造からビジョンまで）-思考は飛躍的なものであり、シナリオよりも重要な機能のリストです
  • 開発（VisionからBetaまで）-機能の設計と開発、SKUと価格オファーの更新、コード交換の開始
  • 準備（ベータからGA + 90まで）-機能およびバグ修正の開発の完了、製品の可用性の追跡、Dell + Windows接続の作成への集中。
