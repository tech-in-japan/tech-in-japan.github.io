会議室を構築する基本原則

最近、テストサイトにクライアント用の会議室のモデルを展開した後、ワーキンググループは同僚から多くの質問を思い付きました。「なぜこれが必要なのですか？」、「なぜこんなに多くの種類の機器があるのですか？」、などなど。会議室の機器の基礎に関する教育プログラム。 私たちは願っています、そして、ここでそれは誰かに役立ちます。



確かに、すべてのハビリテーターは、涼しさに応じて、映写機器、ビデオウォール、オーディオ機器、会議システム、ビデオ会議システム、および他の多くが詰め込まれた現代の会議室にいます。 しかし、誰かが直接会うことができなかったとしても、確かに誰もが映画でこのようなものを見ました。 または、たとえば、スーパーマーケットのキャッシュレジスターに広告がぶら下がっているモニター。 一般的に、誰もがこれが非常に便利なことであり、多くの人々がさまざまなマルチメディア資料をデモンストレーションできることを理解しています。



この記事では、会議室を構築する基本原則について説明します。



したがって、一般的な会議室システムの構造には、次の主要コンポーネントが含まれます。







  • オーディオおよびビデオ信号のソース -会議室（ラップトップ、ドキュメントカメラ）と、会議室から十分な距離にあるサーバールーム（CD / DVDプレーヤー、衛星放送受信機、ビデオサーバー）の両方に配置できます。
  • 情報表示機器 -さまざまなプラズマおよび液晶テレビ、プロジェクター、演壇上のモニターなど
  • 音響機器 -さまざまなスピーカーシステム（天井、壁、床）。
  • 信号の増幅と切り替えは、複数のソースから複数の出力デバイスへのオーディオおよびビデオ信号の送信を同時に行うマトリックススイッチャーである会議室システムの中核です。


優れた会議室の設備は、多くの要件を満たす必要があります。



  • オーディオおよびビデオ信号のソースを操作します 。


ここではすべてが明確です-一部の会議室は、イベントの明確なスケジュールなしで非常にアクティブな生活を送っています。スピーカーには、HDCP保護の有無にかかわらず、あらゆる種類の信号ソースが付属しています。 したがって、すべてに備える必要があります。



  • あらゆるタイプの信号を単一の形式に変換します 。


ここでさらに詳しく説明します。 現時点では、アナログ（VGA）とデジタル（DVI-I、HDMI）の2種類の信号があります。 デジタルルームの方が人気があることは明らかですが、一部のお客様は、 暖かくて古くて良いアナログ信号に基づくシステムを会議室に設置することを妨げません。 そのため、たとえば、信号の利用可能なソースとレシーバがアナログの場合、キャリアとしてアナログ信号を選択し、適切な機器を選択することは理にかなっています。 ただし、出力でデジタル信号を提供する機器の間にビデオ会議コーデックがある場合は、二重変換の結果としての品質の損失を回避するために、デジタル信号をキャリア信号として選択することが論理的です-アナログへの数値とその逆。



フォーマットコンバーターを使用すると、たとえば、コンポーネントビデオとVGAをHDMIに変換できます（たとえば、Kramer VP-434）。 信号変換器に加えて、スケーラーと呼ばれるデバイスがこの目的に役立ちます。 異なるタイプのいくつかの入力（CV、YC、VGA、DVI / HDMI）と特定のタイプの出力（DVI / HDMIなど）があります。 スケーラーの例としては、主力モデルであるKramer VP-747があります-それは何でもできます。



そのため、スピーカーがどのタイプのビデオソースで来るかわからない場合、会議室のスキームに小さなスケーラーを含めることは価値があります。



  • 必要な数のディスプレイソースに信号を出力します。
  • オーディオおよびビデオ信号の切り替えと増幅


伝送距離が長くなると、信号レベルが低下し、画質に大きく影響することに注意してください。 そのような場合、いわゆるイコライザーまたは信号増幅器が助けになります。 たとえば、Kramerの信号増幅器はPT-2Hであり、製造業者によれば、文字通り「床から」信号レベルを上げることができます。 信号品質を損なうことなく、最大45メートルのHDMIケーブルを使用できます。 このようなソリューションは、たとえば、特別な機器を購入する必要があるツイストペアケーブルで信号を送信するよりも経済的に実行可能な場合があります。



切り替えとは、複数のソースから複数の出力デバイスに同時に到着するオーディオおよびビデオ信号のブロードキャストを意味します。 前述のように、マトリックススイッチングはシステム内の信号のスイッチングを担当します。 また、情報およびディスプレイデバイスのオーディオおよびビデオソースの広大で分岐したシステムがある場合、イベントのシナリオは急速に変化するため、信号切り替えを制御するための明確なインターフェイスが必要になります。



信号と同様に、マトリックススイッチャーはデジタルおよびアナログです。 さらに、それらは次元が異なります。 たとえば、Kramer VS-88HスイッチはHDMI信号の切り替えに使用され、8つの入力と出力を備えています。



さらに、適切なボードを追加することで入力と出力のタイプと数を変更できる、いわゆるモジュラーマトリックスがあります。





  • 信号切り替え管理-通常は壁に組み込まれたさまざまなボタン制御パネル 。


したがって、会議室を作成するには、次のものが必要です。



1）ソースからのマルチフォーマット信号を1つのタイプ（キャリア信号）にする;

2）キャリア信号を長距離（またはそうでない距離）で送信します。

3）可能な限り最高品質のディスプレイデバイスで信号を切り替えます。



長距離にわたる信号の伝送といえば。 このために、ツイストペアを介してデジタル信号DVI、HDMI、DisplayPort、およびVGAを送信する技術が使用されます。 Kramerでは、この技術はDGKatと呼ばれています。 ツイストペアケーブルへの切り替えは、50 mからの距離で経済的に正当化されます。距離が短い場合は、信号増幅器を使用する方が適切です（上記参照）。



たとえば、品質を損なうことなくHDMI信号を長距離で送信するには、PT-571（送信側）およびPT-572（受信側）-ツイストペアHDMI信号送信機を使用できます。 彼らの助けを借りて、あなたは70-80メートルの距離で1080pの解像度でHDMI信号を送信することができます！ これらのデバイスは、シールドコネクタ付きのシールドツイストペアケーブル（FTPまたはSTP）でのみ動作するという事実に特に注意を払う必要があります。



非常に長い距離（スポーツ施設など）で信号を送信する場合は、デバイスを使用して、光マルチモードケーブルで信号を送信する必要があります。 たとえば、670T / 670Rでは、最大1,700メートルの距離でHDMI信号を送信できます。
