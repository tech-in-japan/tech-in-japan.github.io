Deep Freeze-「絶対的なシステムの清浄度」





この記事は、Faronics CorporationのDeep Freezeに関するものです。



まえがき



最近、学校では、コンピューターが「共有」されているすべての場所だけでなく、結局のところ、インターネットクラブでは、システムの設定を元の状態に維持するという問題があります。ウイルスが発生したり、マシンがボットネットのメンバーにならないなど。



さまざまなケースで、これはさまざまな方法で解決できます。制限された権限（ユーザー、ゲストなど）を持つプロファイルを作成することによって。 制限のレベルに応じて、グループポリシー（特定の場所への記録の禁止、設定の変更の禁止など）を使用します。



私の場合、ユーザーにパスワードを設定したり、システム設定を変更したり、不要なプログラムや必要なプログラムを削除/配置したりできないように、ユーザーに管理者権限を与える必要がありました。 高度なグループポリシーを使用して禁止するのではなく、単にコンピューターを「フリーズ」することにしました。 つまり 現在のセッションのすべてが許可され、ユーザーには完全な権限がありますが、再起動後、すべてが元々設定されていた状態に戻ります。 そして毎回。 ユーザーが何を変更しても、すべてが無駄になります。再起動後、再び「参照システム」が存在します。



このタスクを完了するために、Deep Freezeプログラムを選択しました。これについては、このトピックで説明します。



プログラムの仕組み



プログラムの動作原理は簡単です。インストール中に、「凍結」（フリーズ）する必要があるパーティションのイメージを作成し、次回の再起動時にパーティションをこのイメージに復元します。 これは、このセクションが脅威、ウイルス、トロイの木馬にさらされないことを意味します。



機能と機能



  • Windows 95-Vista 、 Windows 2k / 2k3 Server 、 Mac OS X 、 Suse Linux（SLED）で動作します
  • SCSI、ATA、SATA、およびIDEドライブのサポート。 FAT、FAT32、NTFS、ext2およびext3パーティション
  • Windows用のエンタープライズバージョンがあり、便利な管理コンソール、追加の管理機能：キーボード/マウスロック、メッセージ送信、リモートオン/オフ/再起動、およびセキュリティの強化によって区別されます。
