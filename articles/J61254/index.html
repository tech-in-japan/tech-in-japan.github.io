研究中のマインドマップ

ほとんどのhabra-peopleはマインドマップのようなものに精通しているべきだと思う、彼らはマインドマップでもあり、接続図でもある。 知らない人のために、私は説明します。 マインドマップは、相互に結びついた多くの考えを表示する階層図です。







リンク図には多くの用途があります。 たとえば、彼らの助けを借りて、データベースを開発し、クラス構造を抽象化し、ブレインストーミングや個人的な費用の計画を立てることができます。 私の研究でそれらをどのように使用するかを説明します。



マインドマップを適用する方法



適用方法は非常に簡単です。あなたの知覚に都合の良い形で輪郭を描かれた資料が地図に記録されます。 最初に主題の基本概念を特定し、次にそれらをそれぞれ解読することを好みます。 一般的なものから特定のものに移り、より深い構造を作成します。

私は弁護士になるために勉強しているので、例が適切であることに注意してください。

たとえば、今日は環境法の試験の準備をしていました。 「環境法」というトピックを作成したら、それに従属する基本概念を決定しました。

  • 業界法
  • 環境法関係
  • 業界の原則
  • ...


次に、これらの概念をそれぞれ解読し、それらの機能を示し、それらを適切にグループ化しました。 結果はそのようなカードです。

メモリカードを使用して教材を学習する方法について説明します。

  1. すべてのノードをオフにして、最初のレベルのみを残します。
  2. 学びたい最初の概念を見て、それに含まれるものをメモリからリストします。
  3. ノードを展開し、自分自身を確認します。
  4. 以降の各コンセプトでも同じことを行います。 エラーの割合が多かれ少なかれ許容レベルに下がるまで繰り返します。


このようにして、論理的に構造化できるほとんどすべての情報を学習および記憶できます。



編集ツール



接続図を編集および表示するためのツールは多数あります。 XMindの無料版が好きで、 andind42.comを使用して地図を公開しています。 原則として、キーボードから完全に制御され、マップの個々のブランチを折りたたむことができるソフトウェアが適しています。

XMindの便利な機能は、マップビューモードを簡単に切り替えられることです。 たとえば、メモを取るには、ツリーモードが非常に便利です（図を参照）。
