フォントサンドボックス





きっとあなたの何人かはすでにあなたの好みに合ったフォントマネージャを見つけており、多くはまだ何百ものフォントでシステムとドロップダウンリストを詰まらせています。それらのほとんどは毎日使われていません。 とりわけ、Windowsの各新バージョンでは、標準フォントの数が次第に増加しています。

  • Windows XP： 133
  • Windows Vista： 191
  • Windows 7： 235
  • Windows 8： 400+


すべてのフォントをめくるという頭痛を避けるために、ワークフローを最適化するために、多くのマネージャーがいて、そのほとんどが支払いを受けており、特定のUIを持ち、めったに更新されません。 以下に、最高の有料および無料の5つのアプリケーションを選択しました。





有料アプリケーションのすべての試用版を試した後、私は有料機能が自分に関係ないという結論に達し、 Nexus Fontを選択しました。無料で、小さく、高速で、直感的で、最も美しいUI（Macなど）すべてが市場に提示され、サンドボックスとして割り当てられたすべての機能を実行します。 それから始めましょう：



1. Nexus Font 2.5







最速の提供プログラム： Nexus Fontは、アプリケーションの起動時にライブラリとフォントを自動的にスキャンしません。そのため、数秒で起動します。 Windows 8の場合、このアプリケーションは速度と使いやすさの点で最高のパフォーマンスを示します。



ライセンス：すべて無料（寄付）

最終更新日： 2012年7月

サポートされているOS： Windows XP-Windows 8



機能：
  • シンプルで直感的なインターフェース
  • すべての要素を表示するためのフォントテーブル
  • 自動アクティベーションフォント;
  • フォントの整理、検索、プレビュー。
  • 独自のフォントセットを作成する機能。
  • 対象のフォントのファミリ全体を選択する機能。
  • バッテリー寿命の可能性（ポータブル）;
  • 重複を検索します。
  • フォントライブラリのバックアップ
  • グラフィック形式でサンプルを印刷するためにエクスポート

ライセンス： 59.00ドル

最終更新日： 2012年10月

サポートされているOS： Windows 2000-Windows 8

  • すべての要素を表示するためのフォントテーブル
  • フォントの整理、検索、プレビュー
  • 独自のフォントセットを作成する機能
  • 破損したファイルの回復、フォントの競合の解決、重複の発見
  • サンプルをHTMLで印刷するためのエクスポート
  • フォントライブラリのバックアップと復元。
