SMBおよびプロバイダー向けの費用対効果の高いHPサーバー

ProLiant製品ラインを拡張した新しいデュアルプロセッササーバーは、予算内で運用しているサービスプロバイダーまたは中小企業に適したソリューションです。



Intel Xeonプロセッサの各新世代のリリースには、標準アーキテクチャサーバーラインのメーカーによる更新がすぐに伴います。 新世代のIntel XeonプロセッサーE5-2600 v3をサポートするサーバーには、多くの革新的な機能が含まれています。 パフォーマンスの向上に加えて、最も重要な技術革新の1つは、新しいDDR4メモリ標準のサポートです。



この標準のメモリモジュールは、最大128ギガバイトの容量を持ち、2133〜4266 MHzの周波数で動作し、電力消費を削減し、パフォーマンスは前の標準のメモリモジュールの1.5倍です。 Intel Xeon E5-2600 v3プロセッサーの最大L3キャッシュは45 MB、最大18コアで、ハイパースレッディングテクノロジーを使用すると、オペレーティングシステムから見えるプロセッサー数は36に達することができます。QPIバス速度は9.6ギガトランザクション/秒に増加します。 Intel Xeon E5-2600 v3をサポートするデュアルプロセッササーバーは、身長1または2Uの使い慣れた構成で利用できます。



HPは、小規模企業や大企業のリモートオフィスおよびブランチオフィス向けの新しいHP ProLiant Gen9サーバーを使用して、コンピューティングプラットフォームの製品ポートフォリオを拡大しました。 2014年12月にリリースされた、1UパッケージのHP ProLiant DL60 Gen9デュアルプロセッサーラックマウントサーバーと2UパッケージのProLiant DL80 Gen9は、基本的な技術的および運用上の指標を備えた非常に魅力的なソリューションとして位置付けられています。 ただし、内部サブシステムの拡張性のみが価格のために犠牲になります。いわゆる「第10」シリーズのサーバーは、低コストのLFFディスク、限られたプロセッサセット（発熱は105ワットに制限されます）、および8モジュールに制限されたメモリ容量（現在32 GBモジュールが利用可能）のみを搭載できますサーバーあたり256 GBに制限されています）。 これらの予算のマシンでは、高品質のビルド、ヨーロッパのアセンブリ、iLOおよびOneViewシステムによるフルコントロールも利用できます。



新しいサーバーは、ワークロード最適化、ソフトウェア構成、およびクラウドソリューションでの使用準備のHP ProLiantファミリ全体の最適化の原則に従って構築されています。 それらのアーキテクチャはシステムが安価になるように最適化されていますが、同時に、ほとんどの問題を解決するために十分な処理能力と管理の容易さを提供します。 これらのコンパクトなサーバーは、プロバイダーや中小企業に特に適しています。 2UシャーシのHP ProLiant DL80 Gen9サーバーは、そもそもストレージサブシステムの拡張性が高い点でDL60と異なります。







1UハウジングのモデルHP ProLiant DL60 Gen9は、このクラスのサーバーの標準的な外観を備えています。 輸送用パッケージの内容には、サーバー自体、電源コード、機器のドキュメントとソフトウェア、ラックマウント用の機器とドキュメントが含まれます。











サーバーの前面には4つのLFFディスクドライブ（つまり、3.5 "フォームファクター）があります。ホットモードでは交換されませんが、システムはホットスワップ可能なHDDおよびSSDもサポートします。オプションでサーバーを注文する場合空のディスクシェルフを持つドライブをホットスワップしない場合（つまり、ドライブがない場合）、空のHPスライドを持つサーバーを取得します（以前のようにブランクはありません）。







標準のDL60サーバー構成には、周波数が最大3 GHzの1つまたは2つのIntel Xeon E5-2600 v3プロセッサーと3つの拡張スロットPCI Express 3.0が含まれます。







サーバーは、最大6つのファンモジュール（プロセッサの数に応じて）のインストールを提供します。 デフォルトでは、ファンはホットスワップを意味しませんが、この機能は個別に注文できます。







HP Smart Socketメカニズムにより、「fireを壊す」（つまり、マザーボード上の接点を壊す）リスクなしに、プロセッサを簡単にインストールできます。 これは、たとえばシステムをアップグレードするときに必要になる場合があります。







サーバーのRAMの容量は最大256 GB-最大2133 MHzのクロック周波数で動作する最大8つのHP DDR4 Smart Memory DIMMです。 HP SmartMemoryの組み込み機能により、生産性が向上し、ダウンタイムが短縮され、電力消費が削減されます。 DDR4メモリを使用すると、チャネルごとに2つのモジュールがインストールされている場合でも最大周波数をサポートするため、従来のメモリに比べてスループットを14％向上させることができます。







HP iLOサブシステムは、HP ProLiantサーバーの標準コンポーネントであり、システムの初期セットアップを簡素化し、そのステータスの監視、電源と温度条件の最適化、およびリモート管理を提供します。 マイクロプロセッサ、安全なメモリ、ネットワークインターフェイスが含まれています。 HP iLOは、サーバーおよびオペレーティングシステムから独立して動作します。 オペレーティングシステムがインストールされていない場合でも、HP iLOサブシステムからSNMPアラートを直接送信できます。



HP iLOでは、次のことができます。



•サーバーへのネットワーク接続がある場合、世界中のどこからでも安全なリモートサーバーコンソールにアクセスします。

•4人のサーバー管理者の共同作業に共通の.NETリモートコンソールを使用します。

•仮想メディアをサーバーにリモート接続します。

•サーバーの電源状態を安全にリモートで監視します。

•SNMPアラートを使用して、エージェントを使用しない管理を実装します。

•アクティブな保護システムのログをダウンロードします。

•HP Insight Remote Supportに登録します。

•iLO連携を使用して、単一のiLO Webインターフェイスを介してサーバーの無制限の大規模グループを管理します。

•スクリプトを使用して、展開の自動化やプロビジョニングなどのさまざまなタスクを実行します。

•リモート管理ツールを使用して、iLOサブシステムを管理します。







標準サーバー構成には、HP DynamicSmartアレイB140iソフトウェアRAIDコントローラーが含まれます。 内蔵SATAコントローラーHP Dynamic SmartアレイB140iは、通常、ブートデバイスの接続に使用されますが、予算構成で唯一のコントローラーにすることもできます。 また、パフォーマンスの向上（SAS 12 Gb / s）、キャッシュメモリを使用したデータセキュリティの向上、ローカルドライブ上のデータのハードウェア暗号化の組み込み（HP Secure Encryptionライセンスが必要）のHP Smartアレイハードウェアコントローラーを選択することもできます。 ビルトインと本格的なPシリーズRAIDコントローラーの中間オプションとして、HBAアダプターを使用できます。これは、RAIDモード（キャッシュなし）でも機能します。







HP Smart Storageバッテリーは、電源障害が発生した場合に書き込みキャッシュデータを保護します。







PCI Express 3.0拡張スロットは、必要に応じてサーバーI / Oサブシステムの帯域幅を増やすのに役立ちます。 簡単にアクセスできるPCIコンタクトストリップにより、拡張カードのインストールプロセスが便利かつ迅速になり、システムのメンテナンスが容易になります。







サーバーProLiant DL60 Gen9は、ホットスワップなしの550 Wエントリーレベルの電源、またはオプションのゴールド800ワット電源（冗長）または900ワット（冗長）をサポートします。 このような電源の効率（効率）は92％に達します。







サーバーの背面には、モニターを接続するためのポート、3つのUSB 3.0ポート、1つのUSB 2.0 microSDカードスロットもあります。 サーバーのUSBポートに含まれているmicroSDカード用のカードリーダーも利用できます。これにより、2枚のカードを組み合わせてRAIDミラーを作成し、たとえばハイパーバイザーのフェールセーフローディングが可能になります。











ProLiant DL60の背面には2つのギガビットイーサネットポートがあり、そのうちの1つは従来のネットワーク機能とともにHP iLO管理ポートの役割も果たします。







2Uパッケージの古いHP ProLiant DL80 Gen9は、優れたI / Oスケーラビリティとより大きなストレージサブシステムを提供します。 このサーバーの合計ディスク容量は、DL60モデルの24 TBに対して72 TBに達する可能性があります。 したがって、ITインフラストラクチャの基本的な操作に加えて、このサーバーはギガバイトあたりの価格が許容範囲であり、ディスクシステムの大容量であるため、たとえば長期のストレージやバックアップなどのストレージ容量を必要とするアプリケーションに適しています。







DL80には2つのシャーシオプションがあります。8台または12台のSAS / SATA / SSDドライブをサポートします。 8ドライブシャーシでは、4つの追加ドライブにドライブケージを後で取り付けることができます。







したがって、このスケーラブルなサーバーは、ボード上に4または8 LFFフォームファクターの非ホットスワップ対応ディスクドライブ、または8または12ホットスワップ可能なHDDまたはSSD LFFフォームファクタードライブを搭載できます。







スロットから抽出されたタグを使用すると、サーバーのシリアル番号、ヘルプコード、およびiLO資格情報に関する情報をすばやく取得できます。







HP ProLiant DL60と同様に、DLL80 Gen9モデルには最大3 GHzのクロック周波数を持つ2つの12コアIntel Xeon E5-2600 v3プロセッサーを搭載できますが、拡張性が高くなります。サーバーは3つではなく6つのPCIe 3.0スロットを提供します。 DL60とのもう1つの違いは、グラフィックコプロセッサー（GPU）をインストールできることです。







体内の透明なプラスチック製の仕切りは、空気の流れを導き、視界を複雑にすることなくシステムコンポーネントを見ることができます。







このプラスチック構造は取り外しが簡単で、サーバーの内部にアクセスできます。







ファンもホットスワップ可能です（オプション）。 最大6個のファンモジュールにより、最もホットなDL80構成に十分な冷却容量が確保されます。 ProLiant Gen9 DL60およびDL80サーバーは、周囲温度40°Cおよび湿度80％（ASHRAE A3標準）に耐えることができますが、個々の構成には制限があります。 このような外部環境への要求は、サーバールームの気候設備を大幅に節約し、冷却システムに障害が発生した場合の通常のサーバーサービス時間を延長できます。







DL60の場合と同様に、プロセッサマウントシステムは安全にフィットし、ツールなしでプロセッサを交換できます。







サーバーのRAMの容量も最大256 GBで、最大2133 MHzのクロック周波数で動作する最大8つのHP DDR4 Smart Memory DIMMです。







最大のデータ保護、パフォーマンス、および可用性を実現するために、サーバーにHP SmartアレイP440またはP840 RAIDコントローラーを搭載できます。







DL60と同様に、HPスマートストレージバッテリーは、停電の際にディスクアレイコントローラーのキャッシュデータを保護します。







最大6つの拡張スロットPCIe 3.0のサポートにより、1枚の標準幅のグラフィックカード、ネットワークカード、または選択した他のデバイスをインストールできるため、I / Oシステムのスループットが向上し、必要なスケーラビリティが提供されます。 PCIe 3.0スロットへの便利なアクセスにより、システムのメンテナンスが簡単になり、構成を拡張できます。











80 PLUS Gold認定のHP冗長電源は、最大92％の効率を実現し、HPの電源管理機能と組み合わせて、電力消費と運用コストを削減します。







DL60と同様に、DL80サーバーには2つのイーサネットポート、3つのUSB 3.0、1つのUSB 2.0、ビデオポート、およびmicroSDスロットがあります。 専用のiLOポートもありません-iLOトラフィックは、統合ネットワークアダプターの最初のネットワークポートを通過します。 microSDメモリーカードは、ハイパーバイザーや一部の種類のオペレーティングシステムの読み込みに使用できます。

結論



HP ProLiant DL60およびDL80は、予算組織に適した非常に魅力的な価格性能比サーバーであり、柔軟な管理機能により、プロバイダーとSMBの両方に最適なソリューションになります。 拡張可能なDL80構成は、成長企業のニーズを満たすことができます。 一般に、このシリーズのサーバーは、ドメインコントローラー、メール、ファイル、またはバックアップサーバーの役割に適しています。



ただし、その役割はこれに限定されません。 HP ProLiant DL60およびDL80サーバーを使用すると、データセンター向けのさまざまなインフラストラクチャソリューションを作成し、ITインフラストラクチャの管理、監視、展開を簡素化する機能を提供できます。 システム開発者は、電源、冷却、ケーブル管理、コンポーネントへのアクセスの問題を検討しています。 本格的なサーバープロセッサを使用することで、完全な仮想化サポートを利用できます。



プロセッサのマイクロアーキテクチャと電源の分野における最新の開発のサーバーで80 PLUS Goldを使用すると、システムの高いエネルギー効率、コンパクト性、信頼性に貢献します。 保証とメーカーのサポートが存在することで、実装と運用の段階で起こりうる問題から会社を守ることができます。 また、内蔵のHP iLO管理ツールにより、リモート展開、監視、およびサーバーメンテナンスが可能になります。



HPによれば、全体的に、前世代と比較して、新しいGen9サーバーのワットあたりのパフォーマンスは最大3倍であり、取得コストを含む3年間の総所有コスト（TCO）は最大62％低くなっています。 ただし、DL60 / DL80にはGen8世代の先行モデルがなかったことに注意する必要があります-これらはProLiantラインの新しいモデルです。



HPサーバーアーキテクチャの主要な原則は、システムがソフトウェアで構成可能なクラウド環境で、統合インフラストラクチャの一部として機能し、特定の負荷を最適化することです。 これは、多くの方法で、オープンREST APIを使用してサーバーを管理する機能によって実現されます（次の投稿のいずれかでそれについて読んでください）。



ただし、おそらく19年の歴史を持つProLiantサーバーで最も重要なことは、ますます複雑化するIT環境での作業を簡素化すると同時に最新の管理ツールを使用した新世代のシステムの設計であり、同時に非常に幅広い機能を有資格の専門家に提供しています。 DL60 / DL80には、HP iLO 4、HP OneView Standard、Intelligent Provisioning、HP Insight Online、HP SUM、HP RESTful Interface Toolなどの多くの無料管理ツールが標準装備されています。 オプションとして、HP ILO Essentials、HP iLO Advanced、HP Insight Control、HP OneView Advancedを利用できます。 新しいDL60 / 80サーバーを購入すると（部品番号のリストは限られています）、iLO Essentialsライセンスを無料で取得できることに注意してください。



HP iLOのビルトイン管理ツールを使用して、リモート展開、監視、およびサーバーメンテナンスを実行できます。 HP Smart Update Manager（HP SUM）とProLiant Service Pack（SPP）で構成されるHP Smart Updateは、ファームウェアとドライバーの更新を最適化し、ダウンタイムを削減するのに役立ちます。



HP OneView Standard（HP SIMツールの後継）は、非常に高速で便利なインターフェイスで、ほぼすべてのデータセンターコンポーネントの監視とインベントリを可能にします。 このツールは、HPのWebサイトから無料でダウンロードできます。



HP Insight Onlineは、HP ITインフラストラクチャ向けのパーソナライズされたリアルタイムのパーソナルダッシュボードです。 シンプルなWebポータルで、機器を監視し、サービスケースを配置して監視し、利用可能な技術サポートを管理できます。
