ハッピーチケット

1週間以上、私は歩いて公共交通機関に乗りました;チケットは5日以降販売されないため、着陸するたびに料金を支払う必要があります。 そして、学童と学生は、チケットを最初に使うことで、最も頻繁に何をしますか（そして、大人でさえ毎日見ていますか）。 いわゆる「ハッピー」に属しているかどうかをチェックし、コレクションに入れるか、忠実に食べるために「静かに」食べることさえあります。 または多分それを捨てる:)







いずれにせよ、これはある種の娯楽であり、喜びの理由です。 そして、子供時代が私と一緒に1か所で遊んだので、面白くなりました。「ハッピー」チケットを獲得できる確率はどれくらいですか。 「ハッピー」チケットはいくつありますか？



学校では、誰かが研究作業（またはポイントではなく、そのようなこと）を行ったとしても、数式を導き出し、グラフを描くなどを試みたのを覚えています。



しかし、コンピューターの力により、（数学の観点から）最適でない方法で進むことができることが多いため、数式の導出に煩わされないことにしました。



その結果、より速く判明しました:)

Perlでのプログラミングに使用されます。 ほとんどの賢明なプログラマーは、少なくともDelphi、少なくともBasic、少なくともJavaScriptなどで、同じことを数分で簡単に繰り返すことができると思います。

プログラムは、ほぼ100万の組み合わせ（989k、最初のラッキーチケット1001）を通過します。



#!/usr/bin/perl



print "Content-Type: text/html\n\n" ;



$num = 1000;

$s = 0;



while ($num < 1000000) {



$ln = length($num);



$l6 = substr($num,$ln-6,1) if ($ln > 5);

$l5 = substr($num,$ln-5,1) if ($ln > 4);

$l4 = substr($num,$ln-4,1);

$l3 = substr($num,$ln-3,1);

$l2 = substr($num,$ln-2,1);

$l1 = substr($num,$ln-1,1);



if ($l6+$l5+$l4 eq $l3+$l2+$l1) {

print 0 x (6-$ln), "$num\n" ;

$s++;

}



$num++;



}



print " = $s" ;





* This source code was highlighted with Source Code Highlighter .







その結果、出口で55251枚のラッキーチケットに相当する金額を受け取りました 。

少し丸めると、約18枚ごとのチケットが「ハッピー」になります。



興味深いことに、数学的な解決策を探すことにし、非常に興味深いストーリーを見つけました。それを読むことをお勧めします。仕事から10-15分少し気を散らしたいなら、それは十分面白いです。 同じページには、まだラッキーチケットに関する多くの興味深いものがあります。



また、ウィキペディアでは、 簡単な抜粋を見ることができます。「ハッピー」チケットの良い短い配布があります。



ウィキペディアのように、番号55252がストーリーに示されていますが、このトピックを書いている時点では、チケット番号000000は存在しないと思いました。 解説は、そのようなチケットの存在の証拠を提供します。



私はこれすべてで何を言いたいのですか？ はい、実際には、特別なことは何もありません。 仕事中にリラックスする必要がある場合は、ゲーム（ゲームを除く）や精神活動に役立つものをたくさん見つけることができます。



幸せに！



UPD：タイプが「会議」（金額が1つずつ異なる）および「手紙」（2が異なる）のチケットの数をカウントしました。 「会議」 -109494チケット 、「手紙」 -106524チケット。 その結果、約4枚に1枚のチケットが私たちの生活に少し前向きな気持ちをもたらしています。
