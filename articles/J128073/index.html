HTML5とJavaScriptを使用してWindows Phone Mangoアプリケーションを開発する





MicrosoftはNitobiと協力して、Windows Phone MangoをサポートするPhoneGapのベータ版が利用可能になったことを発表しました。



PhoneGapはNitobiのオープンソースプラットフォームであり、標準のWebテクノロジーを使用して複数のモバイルプラットフォーム向けのアプリケーションを開発できます。 Githubで現在利用可能なプラットフォームプラットフォームのリストを参照してください ： http : //github.com/phonegap





たとえば、Web開発者はHTML5、CSS、およびJavaScriptの知識を使用して、Windows Phone用のアプリケーションを開発できます。



Windows Phone用のPhoneGapアプリケーションを構築するには、Windows Phone SDK 7.1 RC（ http://create.msdn.com/en-us/home/getting_started）をインストールする必要があります 。 開発ツールがない場合は、展開プロセス中にそれらの無料バージョンが自動的にインストールされます。



その後、GithubのリポジトリからPhoneGapをzipファイルとしてダウンロードする必要があります。

https://github.com/phonegap/phonegap-wp7を開き、ディスクに解凍します。



次に、GapAppStarter.zipファイル（ルート内）をVisual Studioプロジェクトテンプレートディレクトリにコピーする必要があります。

\マイドキュメント\ Visual Studio 2010 \テンプレート\ ProjectTemplates \



PhoneGapで最初のアプリケーションの作成を始めましょう。



Visual Studioを起動し、GapAppStarterプロジェクトタイプを選択します。







独自の方法でプロジェクトに何らかの名前を付けて、[OK]ボタンを押します。



作成されたソリューションに、Githubからダウンロードしたフレームワーク\ WP7GapClassLib.csprojプロジェクトを追加し、その後、メインプロジェクトにその参照を追加します。



PhoneGapで最初のアプリケーション（F5）を収集して起動します。
