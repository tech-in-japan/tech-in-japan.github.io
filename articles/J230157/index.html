AppleとIBMは企業市場を征服する同盟国です

実際、主なニュースは次のとおりです。 「AppleとIBMはモバイルパートナーシップで合意しました ！ 」



IBMが近年、企業市場でのタブレットおよびスマートフォンの広範な導入に積極的に準備を進めていることは周知の事実です。 「IBMがモバイルエンタープライズソリューションの市場に参入する」を参照してください。

多くのクライアントワークステーションのタブレットやスマートフォンがデスクトップやラップトップを置き換えることができる場所。

そして今、AppleとIBMの間のモバイルパートナーシップの発表後、AppleブランドおよびIBMソフトウェアを搭載したタブレットとスマートフォンは、IBMとAppleの両方のディストリビューターの助けを借りて積極的に企業市場に進出します。このビジネスからBlackBerryを追い出します（これが、企業モビリティの支持者であるBlackBerryのシェアがこのニュースの発表以来3.4％減少した理由です）！



IBMのようなIT市場の大企業は、企業市場との戦いにおいて本質的にAppleの同盟国になり、 Apple iOSプラットフォームに基づいてスマートフォンとタブレットをサポートすることを選択したことを理解することは非常に重要だと思います ！

-これは、今後数十億ドルの企業資金がこのプラットフォームに正確に投資されることを意味します。 また、Apple iOSプラットフォームを搭載したAppleタブレットでは、Windows PCからタブレットに切り替えると、企業クライアントソフトウェアの大部分が転送されます。



企業市場セグメントにおけるWindows-PCからタブレットへの移行の傾向は明らかに増加しています。 一部の企業はすでにこのプロセスを開始しており、多くは従業員をPCではなく、時にはPCに加えてタブレットやスマートフォンに切り替えることを計画し始めています（例：ドイツ銀行は従業員用に20,000のiPhoneを購入し、Siemens-30,000）。

もちろん、企業にはタブレットに移行できない仕事があります。 さらに、最も高価なソフトウェアであるサーバーソフトウェアは、サーバー上でスピンし続けます。 同時に、IBMは高価なIBMサーバーソフトウェアを搭載したIBM POWERサーバーがAppleタブレットとスマートフォンに付属することを保証するために可能なすべてのことを試みます（AppleとのパートナーシップによるIBMの明らかな利点）。



しかし、最も重要なことは、Apple iOSが、Microsoft Windowsの代わりに、企業ビジネスのクライアントワークプレイスで徐々に新しい事実上の標準になる機会を得る可能性があることです。

現在、3つの主要なタブレットプラットフォームがあります。

1. Apple iOS ;

2. Google Android ;

3. Microsoft Windows RT 。

そして、後者は明らかにタブレットの進歩に遅れており、今では企業市場を征服するすべての機会を失う可能性があります:(
