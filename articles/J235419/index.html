ジャンゴ1.7

ほぼ1年の開発の後、待望のDjango 1.7のリリースがリリースされました。



新しいバージョンへの主要な変更

  • Python 2.6のサポートを終了しました。
  • フレームワーク自体に直接移行のサポートを追加しました。 南からの移行手順については、 ここをクリックしてください。
  • アプリケーションのロードメカニズムを修正しました。 現在、models.pyはオプションです。
  • 移行をサポートするために、Field APIの変更、特に必須のdeconstruct（）メソッドが追加されました。
  • マネージャーから直接QuerySetを呼び出す機能が追加されました。 モデル間の通信を使用するときに必要なマネージャーを指定できるようになりました。
  • validateおよびcheckコマンドの代わりに、プロジェクトをチェックしてエラーを診断するための新しいシステムが追加されました。


このリリースへの変更の完全なリストはここにあります。HabrでのDjango 1.7の革新に関する記事も読むことができます。

ここから新しいバージョンをダウンロードできます。また、 pip install Django == 1.7からインストールすることもできます。

リリース1.7に加えて、以前のリリース（バージョン1.4.15、1.5.10、1.6.7 ）のバグ修正も表示されます。
