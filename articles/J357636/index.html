Android Studio 2.0開発環境のリリース（プレビュー）





昨日、 Android開発者向けのサミットで 、GoogleはIntelliJプラットフォーム上の独自開発環境の最初のメジャーアップデートであるAndroid Studio 2.0を発表しました。



Androidアプリケーションの作成者はきっと喜ぶでしょう。最も重要なことはAndroid Studio 2.0に実装されています。



  • アプリケーションのインスタントラン（インスタントラン）
  • より生産的なAndroidエミュレーター
  • 新しいGPUプロファイラ/デバッガ


開発プロセスを大幅に高速化するアプリケーションの即時起動に特に満足しています。 コードエディタとスマートフォンウィンドウがすぐに表示され、1行追加するだけです。アプリケーションはエミュレータですぐに更新された形式で起動し、コンパイルなどは行われません（猫の下のデモを参照）。 これは本当に革命的な機能です！







すべてが非常に高速に動作します。 [ 実行]をクリックします。エミュレータでアプリケーションが自動的にコンパイル、パッケージ化、インストール、および起動されます。 たとえば、コード行を追加すると、すぐに動作中のアプリケーションにデプロイされます！







新しい行を追加する場合、すべてが同じです。コンパイル、パッケージ化、インストールは繰り返されません！



OpenGL ESコードのプロファイリングも、新しいGPUプロファイラーにより簡単になりました。 OpenGL ESコードの実行およびレンダリング中に、セッション全体を記録し、すべてのフレームバッファバッファとテクスチャのステータスを表示できます。











プログラムは、プロファイリング機能がまだ実験的な状態であることを警告します。 この拡張機能をインストールするには、最初にAndroid Studio SDK ManagerにGPUデバッグツールパッケージをインストールする必要があります。







Android Studio 2.0はまだ終了する準備ができていません。初期のベータ版（カナリア）のみがリリースされました。 最新のビルドはこちらです。
