州および民間企業の「有害な」影響からユーザーとホスティング事業者を保護するために、I2Pネットワークが作成されました。 それは何であり、I2Pに接続する方法-カットの下でそれを読んでください。



だから、私はあなたを最新のものにします



I2Pは匿名暗号化ネットワークです。 2003年に、ネットワークセキュリティと匿名性を提唱する開発者のコ​​ミュニティによって導入されました。

実装のタイプごとに 、 オーバーレイネットワークであり、OSIモデルの第6レベルにあります。



ネットワークの各クライアントは他のクライアントに接続し、トラフィックが送信されるトンネルを形成します（Skypeに似ていませんか？）。 クライアントプログラムは、他のソフトウェアにいくつかの対話インターフェイスを提供します。

  • SOCKS-プロキシ-最も一般的に使用されるプロトコル。 I2Pをサポートするほとんどすべてのプログラムと組み合わせてI2Pを使用できます。
  • SAM（Simple Anonymous Messaging）は一種のAPIです。 TCPの上に実装されます。
  • BOB（Basic Open Bridge）はもう1つのAPIです。 実際、SAMのアップグレードバージョンですが、コマンドとデータに個別のチャネルを使用しています。




公式のI2Pパッケージには以下が含まれます。

  • SusiDNS-DNSクライアント
  • Susimail-メールクライアント
  • I2Psnark-トレントクライアント
  • I2PTunnelは、さまざまなTCP / IPサービスがトンネルを使用してI2Pで通信できるようにするI2Pの統合プログラムです


すべてのコンポーネントはJavaで記述されています。 インストールについて気が変わっていない場合は...



接続しています！



  1. JREがインストールされていない場合は、 java.com / downloadを設定します
  2. 開始するには、クライアントをダウンロードしてインストールします： mirror.i2p2.de/i2pinstall_0.7.14.exe 。 Linux /マクロ：拡張機能に注意しないで、次のように実行します： java -jar i2pinstall_0.7.14.exe







  3. クライアントを開始します。

    a。 Windows：

    スタート-> I2P-> I2Pの開始（ウィンドウなし）。

    b。 Unixライク：

    クライアントインストールフォルダーに移動します。 コマンドを実行します： ./i2prouter start





  4. ブラウザの設定でプロキシサーバーをインストールします。
