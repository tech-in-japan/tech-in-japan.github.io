JetBrains WebIDEホットキー

この素晴らしいIDEで一般的に使用されるホットキーのリストを作成しました。

それらのいくつかはメニューにありませんが、それでもそれらはあり、素晴らしい仕事をします。 



 Ctrl + Del-次の行の前のスペースをすべて削除し、カーソルがある行の末尾に移動します。
 Ctrl + W-現在の単語を選択
 Ctrl + D-行の複製
 Ctrl + M-画面の中央に現在の行を設定
 Alt + F7-プロジェクトで変数/関数の使用を検索
 Ctrl + Alt + F7-プロジェクトでの変数/関数の使用のクイック表示
 Ctrl + Shift + Del-ペアのHTMLタグを削除
 Ctrl + Shift + F7-現在のファイルで変数/関数の使用を強調表示する
 Ctrl + Shift + I-変数/関数定義のクイックビュー
 Ctrl + F12-ファイル構造の高速表示
 Shift + F6-変数/関数の名前とその使用
 Ctrl + Y-現在の行を削除
 Ctrl + "["-開始中括弧に移動します
 Ctrl + "]"-右中括弧に移動します
 Ctrl + G-行に移動
 Ctrl + B-変数/関数宣言に移動
 Ctrl + J-テンプレートを貼り付けます
 Ctrl + P-パラメーター情報
 Ctrl + "-"-ブロックを折りたたみます
 Ctrl + "+"-ブロックを展開
 Ctrl + "/"-コメント行/行（再コメント解除）
 Ctrl +スペース-補完
 Ctrl + Insert-現在の行全体を選択して、バッファーに入れます
 Ctrl + Shift + "/"-ブロックコメントの貼り付け


 F11-設定/マーク解除
 Ctrl + F11-ニーモニックラベルの設定/削除
 Shift + F11-ラベルが設定されたウィンドウを表示する
 Ctrl + F-検索
 Ctrl + R-検索と置換
 Ctrl + Shift + U-単語の大文字と小文字を変更する
 Alt + Shift + Insert-列モードに移動します。
 Ctrl + Shift + V-クリップボードから目的のアイテムを貼り付けるためのクリップボードウィンドウを表示します。
 Ctrl + P-関数パラメーターを表示
 Ctrl + /-1行にコメントします。  Ctrl + /もう一度-コメント解除
 Ctrl + Alt + S-設定メニュー
 Ctrl + Z-1回戻る
 Ctrl + Shift + Z-1つ先の操作
 Alt + Shift + C-最近の変更を表示
 Ctrl + Alt + I-行のインデントを自動的に揃える
 Ctrl +上/下-画面を上/下にスクロール
 Ctrl + Shift +上/下-行を上/下に移動






誰かが重宝すると思います。
