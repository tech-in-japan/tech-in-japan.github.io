Photoshop +ロシア語のホットキー

すぐに言っておく必要があります。PhotoshopCSでのみ作業しているため、Photoshop CSでのみ確認しました。 彼の兄がそれを妨げています。



Photoshopでロシア語のキーボードショートカットが機能しないことは誰でも知っているでしょう。

ある時点でそれは私を悩ませ、私は問題を解決する時であることに気づきました。



まず、すべてのホットキーを「Default Keyboard Shortcuts.kys」ファイルに保存し、そこでロシア語の英語の文字を修正しました。 しかし、その後、英語は機能しなくなりました。



それから、Photoshop自体の機能に再割り当てできることを知りました。

結果はこのファイルです

私が最も頻繁に使用するメインキーのみがそこに再割り当てされますが、簡単に独自のキーを追加できます。



それらのインストールは簡単です：

-\ Program Files \ Adob​​e \ Photoshop CS \ Presets \ Keyboard Shortcuts \ Default Keyboard Shortcuts.kysにコピー

-Photoshopを再起動します

-ホットキーを割り当てるためのダイアログで、デフォルトのキーボードショートカットを選択します。



何らかの理由でCtrl + Shiftの組み合わせが含まれるPSホットキーは割り当てられません。
