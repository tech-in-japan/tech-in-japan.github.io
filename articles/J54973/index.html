Silverlight 3 Beta-すべてのリンクと機能

Silverlight 3ベータ版は昨日 、 MIX09カンファレンスで正式に発表されましたが、このバージョンは主にエンドユーザーではなく開発者や愛好家向けです。 最終バージョンは年末までにリリースされることを思い出させてください。



今日新しい機能を試すには、 次のものが必要です。

  • Visual Studio用のSilverlight 3ベータツール

  • Silverlight 3 Beta開発者ランタイム： WindowsまたはMac （以前のものと自動的にインストールされます）

  • Microsoft Expression Blend 3プレビュー

  • Silverlightツールキット

  • .NET RIAサービス

  • Silverlight 3 Betaドキュメント- オンラインまたはCHMファイル



重要！ Silverlight 3をインストールすると、以前のバージョン用に作成されたサイトを実行できるようになります。 ただし、以前のバージョンのアプリケーションを作成することはできません。



Silverlight 3 Betaのコンテンツは、英語でゆっくりと表示されます。 ロシア語でのReMIX会議用に多くの資料を準備します。



3番目のバージョン（または、現在のベータバージョン）の新機能について簡単に説明します。

  1. 高品質のビデオとオーディオのサポート

    • 現在のチャネルに応じて信号品質が動的に変化する、HDビデオ（720p以上）の直接ブロードキャストおよびオンデマンドの可用性
    • 新しい形式のサポート-MPEG-4ベースのH.264、AACオーディオ
    • ハードウェアGPUアクセラレーションを使用したHD品質を含む、フルスクリーン再生の新機能
    • サードパーティのコーデックを作成するための生のAVフォーマットのサポート
    • DRMを介してコンテンツを保護する機能


  2. 新しいグラフィック機能

    • 有望な3Dグラフィックス
    • ピクセルシェーダー効果
    • グラフィックキャッシング
    • 新しいグラフィックレンダリングAPI
    • アプリケーションのグラフィカルテーマのサポート
    • アニメーション効果
    • コントロールの追加のスタイリングオプション
    • テキストレンダリングとフォント処理の改善


  3. 対話型アプリケーション（RIA）を作成するための新機能

    • 60以上の定義済みソースコード管理
    • アプリケーション内にリンクを作成するためのディープリンク
    • 検索エンジン最適化機能
    • データを操作するための新しい機会-要素間のリンク、データフォーム、データ検証、ビジネスオブジェクトのサポート
    • パフォーマンスの向上-ライブラリキャッシング、新しいDeep Zoom機能、バイナリXML、Silverlightアプリケーション間のローカルリンク


  4. ブラウザなしの機能

    • デスクトップおよびオペレーティングシステムのメインメニューのショートカットのサポート
    • ブラウザなしで起動
    • サンドボックス保護モード
    • 管理者権限なしでアプリケーションを簡単にインストール
    • 組み込みの自動更新アプリケーション
    • インターネット接続を自動的に検出し、それなしで動作します
    • Windows 7との統合-タスクバー、マルチタッチ


  5. アクセシビリティ機能

    • すべてのシステムカラーにアクセスして、視覚障害のあるユーザー向けの対照的なインターフェイスを作成
