Internet Explorer 11はWebGLをサポートします

最近では、Windows 8 Blueがネットワークに流出し、Internet Explorer 11の予備バージョンが提供されています。IE11は、一般に認められているWebGL 3Dグラフィックスアクセラレーション標準をサポートしますが、Microsoftは「セキュリティリスク」のためにこの技術の導入に長い間抵抗していました。



カットの下の情報。







IE 11のWebGL APIの痕跡は、Web開発者Francois Remyによって発見されました。 彼は一度にWebGL APIのいくつかの参照に気づきましたが、それらを使用できなかったため、それらは機能しないと考えましたが、彼の同僚はより深く掘り下げ、WebGLが現在のバージョンのブラウザーで実際に動作することを証明しました。 これを行うには、Windowsレジストリに適切な変更を加え、ブラウザーを再起動します。



Windows Registry Editor Version 5.00



[HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main\FeatureControl]



[HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main\FeatureControl\FEATURE_WEBGL]



"iexplore.exe"=dword:00000001



[HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main\FeatureControl\FEATURE_WEBGL_HLSL_SHADERS]



"iexplore.exe"=dword:00000001
