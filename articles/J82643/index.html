Debian GNU / Linux 5.0.4がリリースされました

2010年1月30日に、Debian GNU / Linuxの4番目の安定リリースがコード名「Lenny」でリリースされました。これには伝統的に多くの修正が含まれています。

カーネル2.6のDSA-1915 （mmap_min_addrパラメーターのデフォルト値に関連付けられた特権エスカレーション）、ntp（DoS）のDSA-1948 、およびspamassassinのFH_DATE_PAST_20XXルールの誤検知操作に関するバグリストの＃563245チケットの最も重要な更新を強調したいと思います。番号は3.2.5-7および3.2.5-2 + lenny1.1〜volatile1）です。

いつものように、ISOイメージは今後数日でミラーにヒットしますが、aptは今すぐ更新できます。

システム管理者はdebian-security-announceメーリングリストの使用を推奨しています。これにより、常に最新のアップデートを入手できます。
