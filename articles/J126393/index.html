Googleの支援を受けたロシアのGTUGがGoogle Chromeのハッカソンに招待し、賞品を伴うコンテストを開催

Chrome Hackathons＆Contestとは何ですか？

これは、Google ChromeとChrome Web Appsの世界最高の拡張機能を作成したい開発者の間の競争です。メインのハッカソンは、8月27日と28日に週末に開催されます。サンクトペテルブルクのマイクウェスト（クロームアドボケート）。 ハッカソンは、ウファ、ヴォロネジ、オムスクでも同じ日に開催されます。



競争の目的は？

目標は、Chromeユーザーから収集されたリクエストや希望に基づいて優れたWebアプリケーションと拡張機能を作成することにより、お気に入りのGoogle Chromeブラウザーを私たちとユーザーにとってより便利にすることです 。



拡張機能：拡張機能は既存の拡張機能を繰り返すべきではありません（ただし、既存の拡張機能を改善することができます）。 ユーザーが提案する拡張機能を使用できますが、それでもこれらの拡張機能は、 ブラウザアクションまたはページアクションのカテゴリに分類する必要があります。 この技術的なコンテストには、 テーマやページのオーバーライドなどの拡張機能は含まれません。 拡張機能は、実際のユーザーの問題と技術的な実装を解決することによって評価されます。



アプリケーション：アプリケーションは、既存のものを繰り返さないことが理想的です。 アプリケーションは、実際のユーザーの問題と技術的な実装を解決することにより評価されます。



ハッカソンと競技会はいつ開催されますか？

ハッカソンとコンテスト自体は4段階で構成されます。

  1. Chrome Hackathons-8月27〜28日：5都市での2日間のイベント。
  2. Chromeコンテスト-8月29日-10月8日：1か月にわたって開催されるオンラインコンテストで、開発者はアプリケーションと拡張機能を完成させ、最終版の準備を進めています。
  3. 最終-DemoFest-10月9日：イベントはGoogleのモスクワオフィスで開催されます。
  4. GDD 2011-10月10日の受賞者のプレゼンテーションと授与：イベントはGDD 2011ロシアで開催されます。


受賞者

3人の受賞者にはMac Book Airラップトップが贈られます。これは、 Google Developer Day 2011 Russiaでの授賞式で贈呈されます。 追加の4位と5位には貴重な賞品が授与されます。
