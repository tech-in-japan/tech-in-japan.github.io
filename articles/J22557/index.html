HDPCオールインワン

MIUはまもなく韓国市場に新しい携帯ハイブリッドデバイスであるHDPC（Hybrid Dual Portable Computer）を発売します。 これらの機能に加えて、MP3 / MP3プレーヤー、ナビゲーター、電子辞書、ポータブルTVなどとしても使用できます。さらに、ユーザーは好みに応じてオペレーティングシステムを選択することもできます。 。



以下は仕様です。



-プロセッサー：500 MHz / 1 GHz VIA C7M ULV、4 GB DDR2メモリー;

-OS：Windows XP / Windows CE 5.0またはLinux Qplus;

-4インチ、800 x 480タッチスクリーン格納式内部ディスプレイ。

-1.3インチ30/60 GBハードドライブ（または4 GB SSD）;

-miniSD / microSDスロット;

-802.11b / g WiFi;

-Bluetooth 2.0;

-3または5メガピクセルのカメラ。

-フルQWERTYキーボード、2つのUSBポート、ステレオスピーカー。

-DMB-TVおよび外部GPSレシーバー。

-頻繁に使用する場合は7時間、スタンバイでは90時間のバッテリー寿命


この全体の重量はわずかであり、サイズは163 x 67 x 24.5 mmで387 gではありません。 その費用は約500立方 今年7月に発売されます。
