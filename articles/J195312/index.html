VLC 2.1リリース

VideoLANは、長年の最初の主要なVLCリリースである無料のVLC 2.1ビデオプレーヤー （Rincewind）のリリースを発表しました。



変更のリストは重要です。モバイルプラットフォームへのポートが実装され、多くの新しい形式、デバイス、メタデータがサポートされ、古い形式の再現がさらに改善され、準備が新しいコーデックをサポートし始めました。 約1000のバグが修正され、140人のボランティア開発者からの7000件以上のコミットがメインブランチに受け入れられました。



1. オーディオおよびオーディオモジュールを処理するためのエンジンが書き直されました 。サウンドおよびサウンドデバイスのより良い制御、マルチチャネルフォーマット5.1、6.1および7.1の適切なサポート、Vista、Android、iOS、OpenBSDおよびOSSv4の新しいオーディオ出力オプション、いくつかの新しいサウンドフィルター/エフェクト、周波数の増加離散化、正確性、オーディオエンジンでのリアルタイム設定、メタデータを含む多くの新しいサポートされているオーディオ形式。







2. 新しいコーデックとハードウェアデコード ：G2M4、MSS1、MSS2、TSCC2、CDXL、Ut、VBLEビデオコーデック、Ulead DVオーディオコーデック、Indeoオーディオコーデック、RealAudio Lossless、SCTE-27およびEIA-608字幕、QuickSyncVideo for Windowsによるハードウェアデコードのサポート、GNU / LinuxでのVDPAU経由、AndroidでのMediaCodec経由、OS XでのVDADecoder経由。







3. 新しいビデオサブシステム ：OpenGL ESを介したOpenGL出力、AndroidおよびiOSの色をglslに変換するためのシェーダーのサポート、XYZ色空間でのより深い色深度のインターレース解除のサポート、3Dの新しいアナグリフフィルター、4Kのサポート。



4. 新しいフォーマットのサポート ：断片化されたMP4、Wave / RF64、Ogg、AVI、MP4およびMKVのメタデータタグとカバーのサポートの改善、オーディオFLAC、Atrac、ADPCM、DVタイプ1、AVIコンテナの12ビットDVのサポート、拡張AVI、MKVおよびMJPEGストリームのサポート、AVIおよびMKV形式での最高の録音、AcoustID形式での音声フィンガープリント。







5. アニメのファン向け ：MKV / Flac 6.1から5.1の新しいダウンミキサー、OpenGLシェーダーの正しいカラーマトリックスYUV-> RGB、MKVの検索サポートとスピードの改善、MKVの編集サポート、MKVの字幕とメタデータのサポートの改善、字幕ASSのさまざまな改善。







6. 新しい入力デバイス ：OS X Lion以降の画面から、Microsoft Smooth Streamingのサポート、libavformatを使用する新しいRTMPモジュール、VNC / rfbおよび表示モードのリモートデスクトップのサポート、Blu-Rayでの再生方法の大幅な改善、ダッシュ、v4l2およびHTTP。



7. モバイルデバイスの場合 ：Android 8から2.1へのポート、ARMv6、ARMv7、x86、およびMIPS、5から7のiOSの新しいポート、3GS以降のすべてのiPadおよびiPhone、Windows 8、8.1のWinRTの部分ポートおよびWP8、OpenGL ESによる最適化された出力、デコーダー、エンコーダーおよびOpenMAX ILレンダリングの改善、モバイルデバイス用のオーディオおよびビデオ出力の新しいフォーマットとインターフェイス。



8. 開発者向け ：libVLCおよびほとんどのモジュールがLGPLv2.1 +で配布されるようになりました。







LibVLCメディアフレームワークはほとんどのアプリケーションで使用でき、libVLC SDK開発キットが登場し、libVLC APIが改善され、VLC Webプラグインがウィンドウレス再生をサポートするようになりました。



VLC for Windows 、 Mac OS X 、 sources 、 changelogをダウンロードしてください。
