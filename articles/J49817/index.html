phpDocumentorおよびUTF-8ソース

phpDocumentatorにUTF-8ソースのロシア語テキストを理解させる方法：



1.ファイルSetup.inc.phpを編集する必要があります。 以下を修正する必要があるcreateDocsメソッドがあります

- if (stristr($ret, "utf-8") !== "")

+ if (stristr($ret, "utf-8") === false)

{

$ret = utf8_decode($ret);

}









2.ドキュメンタのすべての（*。*）ファイルのグローバル置換を実行し、置換します

charset=iso-8859-1'/>





に

charset=utf-8'/>







3.オプション（すでに機能しましたが、マニュアルがXMLに出力されるときに問題になる可能性があります）

ドキュメンタのすべての（*。*）ファイルでグローバル置換を実行し、置換します

<?xml version="1.0" encoding="iso-8859-1"?>





に

<?xml version="1.0" encoding="utf-8"?>
