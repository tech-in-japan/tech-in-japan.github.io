Windows 7の配布ライン

マイクロソフトは、Windows 7オペレーティングシステムが配信されるさまざまなバージョン（SKU、ストックキーピングユニット）の構成に関する情報を発表しました。このラインは、単純な原則に従って編成されています。後続の各ディストリビューションは、以前のディストリビューションの改良されたコピーです。それらがリストされています）。



1. Windows 7 Starter（ネットブック用）：実行中のアプリケーションは3つまでです。



2. Windows 7 Home Basic（発展途上国のみ、他では販売されません）：アイコンに「ライブ」プレビューを追加し、グラフィカルインターフェイスを改善し、ネットワークサポートを強化（ワイヤレスとインターネット接続を共有）、モビリティセンター。



3. Windows 7 Home Premium （個人ユーザー向け）：Aero Glassインターフェイスと、ウィンドウを備えた高度なナビゲーション機能、サポートされるメディア形式、Windows Media Centerとメディアブロードキャストの改善、マルチタッチ、手書き認識。



4. Windows 7 Professional （中小企業向け）：ドメイン参加で管理グループに接続し、データを保護し、ネットワーク経由でバックアップ、暗号化ファイルシステム、場所に応じて異なるプリンターに印刷する機能（位置認識印刷機能）。



5. Windows 7 Enterprise（中規模および大規模企業向け）およびWindows 7 Ultimate（以前の2つのSKUの有料アップグレードとしてのみ）：内部および外部ドライブのBitLockerデータ保護、Windows Server 2008 R2の企業ネットワークへの透過的な接続のためのDirectAccess機能Windows Server 2008 R2上のネットワーク上のBranchCache。AppLockerファームウェアを使用した個々のアプリケーションの実行の禁止。



マイクロソフトは、主要な2つのディストリビューション（リスト内で太字でマークされている）を考慮しており、ほとんどのユーザーに対して最も積極的に宣伝されます。



販売の価格と開始日はまだ発表されていません。
