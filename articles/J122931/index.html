Create For Millions-モバイルアプリケーションの開発者とユーザー向けのコンテスト





Hi％username％！



興味深いイベントを共有したいと思います。 今日、私たちは「Create For Millions」と呼ばれるグローバルな競争を開始します 。 競争の参加者は、Series 40プラットフォーム用のモバイルソフトウェアの開発者だけでなく、実際に必要なアプリケーションについて多くのアイデアを持っている一般ユーザーでもあります。 確かに、あなたの多くはそのようなイベントで手を試しました。 なぜこれに参加する価値があるのですか？





開発者へ





開発者に提供するインセンティブから始めましょう。 コンテストの賞金は100万ユーロです。 参加者には、Nokia Series 40スマートフォン用のアプリケーションを次の4つのカテゴリに分けて開発することができます。



  • ニュースと情報-ニュースと地理位置情報アプリケーション。
  • エンターテイメントとゲーム-ゲームとエンターテイメントアプリケーション。
  • ソーシャルネットワーク-ソーシャルネットワーク用のアプリケーション。
  • 知識と教育-健康的なライフスタイル、教育などのアプリケーション。


各カテゴリには10の賞品があり、各カテゴリの1位の賞品は50,000ユーロです。 さらに、最高のアプリケーションの作成者と協力して、インターフェースを研究および改善し、製品の市場投入などを支援します。 「ベストタッチアプリケーション」、「ベストロケーションアプリケーション」、「シリーズ40のベストWebアプリケーション」のカテゴリの開発者には、別々の賞品が用意されています。



見込み





Series 40プラットフォームは、主に開発途上国での次の10億のインターネット接続戦略で重要な役割を果たしているため、非常に注意を払っています。 これらの国ではよくあることですが、ユーザーはまず携帯電話からインターネットにアクセスします。 既に、Series 40モバイルプラットフォームは、ボリュームの面で世界最大であり、5億人以上がすでにそのような電話を使用しています。



Ovi Storeからのすべてのダウンロードのボリュームの4分の1がNokia Series 40の携帯電話に当てはまるという事実を考慮すると（過去2か月間の成長率は35％以上）、見通しが明確になります。



開発者を支援するために、Series 40プラットフォームでNokia製の携帯電話用のアプリケーションインターフェイスを設計する際のすべてのニュアンスを説明するプレゼンテーションに慣れることをお勧めします。







どこから始めたらいいか分からない？ 弊社のサイトからNokia Web Tools SDKとJava SDKをダウンロードしてください。ビデオチュートリアルとサンプルソースコードがあります。



ユーザーへ





そして、ユーザーにとっては少し嬉しい驚きです。 Nokia製の携帯電話用に作成する必要があるアプリケーションがあれば、コンテストのユーザー部分に参加できます。 最も成功したアイデアの著者は、今年10月にロンドンで開催されるNokia World 2011カンファレンスに招待され、そこで彼らのアイデアがどのように実現されるかを自分の目で見ます。 また、イベント中に発表する最新のNokia Series 40電話でアプリケーションを試すこともできます。



必要なのは、Create For MillionsコンテストWebサイトへのリンクをたどり 、指示に従ってアイデアを思いついたアプリケーションのカテゴリを選択し、必要なフォームを登録して記入することです。
