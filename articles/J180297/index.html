新しいスマートフォンリモート管理ソリューション：3CXモバイルデバイスマネージャー（Android + iOS）

先週、3CXは3CX Mobile Device Managerを導入しました。これは、AndroidおよびiOSでスマートフォンを管理するための安価でシンプルなソリューションです。



ソフトウェアの機能は非常に高度であることが判明しました。

-リモートロックまたはデータの削除



スマートフォンが紛失したり盗まれたりした場合、3CX Mobile Device Managerを使用して、リモートでロックしたり、SDカード上のデータを含むすべてのコンテンツを消去したりできます。 インターネットに接続しなくても、不正アクセスに対する保護が可能です。 プラス パスワードの使用条件を厳しくし、パスワードの最小文字数に制限を設定したり、パスワードが間違って入力されたときにデバイスを自動的にロックするように時間遅延を変更したりできます。

-不要なアプリケーションの削除と大量インストール



管理パネルでは、スマートフォンにインストールされているすべてのアプリケーションを表示し、不要なアプリケーションを削除できます（従業員の労働時間を奪ったり、トラフィックが過剰になります）。 AppstoreまたはGoogle Playを使用せずに、接続されているすべての機器にプログラムを一括インストールすることもできます。

-リモート管理





3CX MDMを使用すると、Wi-Fiに新しいパスワードを設定したり、ネットワーク上のすべてのスマートフォンのセキュリティ設定を一度に変更したりできます。 同時に、従業員にはシステム設定の変更がメールで通知されます。

デバイスの検索と追跡、従業員の動きの監視



3CXモバイルデバイスマネージャーは、接続されたデバイスの場所に関するすべての情報を収集し、スマートフォン/タブレットとその所有者がいた場所に関する詳細な統計を保存します。 すべての場所は、Googleマップに完全な住所とともに表示されます。

-リモート企業メールのセットアップ

3CXは、近い将来、人気のあるAqua Mailクライアントを3CX Mobile Device Managerに導入する予定です。 すべての管理者が企業メールをリモートで構成できるため、個人の通信に影響を与えることなく、仕事用メールを完全に制御できます。 したがって、会社は、従業員が退職した後、モバイルデバイスに保存されている添付ファイルを含むすべての企業通信を削除できます。



3CX Mobile Device Managerの試用版では、5台のスマートフォンを無料で接続できます。公式ウェブサイトwww.MobileDeviceManager.comからダウンロードできます。
