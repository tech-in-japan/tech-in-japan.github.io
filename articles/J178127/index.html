ロシア全土のMegaFonネットワークのMeizu MX2

親愛なるHabrausers、あなたと私たちにとって重要なニュースを共有するために！ ロシアの通信事業者であり、Telecom InfoProjectであるMegaFonは、2013年5月6日からロシア全土のMegaFon店舗でMeizu MX2の協力と販売を開始することを発表しました。









2013年4月30日。 ロシアの電気通信事業者であるMegaFonとTelecom InfoProjectは、MegaFonネットワークでのMeizu MX2の協力と販売の開始を発表しました。 32 GBおよび64 GBのバージョンのMeizu MX2スマートフォンは、2013年5月6日からロシア全土の535メガフォンストアで利用可能になります。



MegaFonストアでのMeizu MX2のコスト：



  • Meizu MX2 32 GB-19 990ルーブル
  • Meizu MX2 64 GB-21 990ルーブル


スマートフォンはMegaFonストアおよびMegaFon オンラインストアで購入できます。



Meizu MX2は、2012年末に導入されたスマートフォンで、最高の機能、エレガントなデザイン、高性能、魅力的な価格を備えています。 モデルは完全にローカライズされ、認定されており、1年間の保証が付いています。



Meizu MX2の機能：



  • オペレーティングシステム：Android 4.1とFlyme 2シェル
  • 通信：GSM / EDGE 850/900/1800/1900 MHz、UMTS / HSDPA 900/2100 MHz
  • 画面：Sharp New Mode 2（IGZO）、4.4インチ、1280x800ピクセル、タッチ、容量性タイプ、アスペクト比16：10、347 ppi、1600万色、ゴリラガラスコーティング
  • プラットフォーム：MX5S（Samsung Exynos 4412、A9、1.6 GHzクアッドコア）
  • GPU：Mali-400 MP
  • RAM：2 GB
  • データストレージ用メモリ：16/32/64 GB
  • カメラ：オートフォーカス付き8 MP、明るいLEDフラッシュ、1.3 MPフロントカメラ、メインカメラ用に1080p、フロント用に720pのビデオ録画
  • ヘッドフォンで優れたサウンドを実現するWolfson Microelectronics音楽チップ
  • インターフェース：Wi-Fi（802.11 b / g / n）、Bluetooth 4.0、microUSB（USB OTGサポート）、ヘッドフォンおよびヘッドセット用3.5 mm
  • ナビゲーション：GPS、aGPS、Glonass
  • オプション：光センサー、近接センサー、加速度計
  • バッテリー：1800 mAh
  • 寸法：124.9 x 64.9 x 10.2 mm
  • 重量：142グラム


Meizu MX2の利点と独自の機能：



  • ベースのステンレスフレーム
  • Sharp New Mode 2（IGZO）ハイパワー、ローパワーディスプレイ
  • ユニークな電話機能：「回線から」の通話録音、通話用のカスタマイズ可能な留守番電話、「ブラックリスト」を作成する機能
  • 最も一般的なオーディオおよびビデオ形式、およびビデオの組み込みおよび外部字幕のサポート：FLAC、APE、AAC、MKA、OGG、MP3、MIDI、M4A、AMR、オーディオ用WAV。 ビデオ用MP4、3GP、MOV、MKV、AVI、FLV、MPEG、M2TS、TS
  • スーパーユーザー権限（ROOT）は、設定メニューから利用できます
