職人は自分の手でセガゲームギアを組み立てました





「lovablechevy」というニックネームで関連リソースで通信する改造の世界で非常に有名な人物が、最近自分で組み立てたGame Gearの独自バージョンを紹介しました。 いいえ、これはポータブルで少し「ドップされた」コンソールではなく、E-bayで購入されたものではありません。実際、ここではすべてが自分の手で組み立てられています。 このデバイスは、オリジナルのGame GearゲームとSEGA Master Systemゲームの両方をサポートしています。 MegaDrive | Genesisでゲームがサポートされているのだろうか？ 続編-いくつかのビデオと設定情報。







ちなみに、発表写真で-ゲーム「ファンタシースター」。 私は最初の部分を除くすべての部分でプレーしましたが、これはまさに最初の方法のようです。 誰かがこの役を演じましたか？ 二番目に好きだった。 これはただのゴージャスなゲームで、一度に何時間も何日も友達と過ごしました...



そのため、このデバイスには5インチディスプレイ、バッテリー、新しいケース、その他のグッズが装備されています。 デバイスの消費電力を削減するために、設計者は設計を大きく変更しました。 ちなみに、すべての消防士に対して、元のデバイスの構成を提供します（ Wikipediaからの情報）。



プロセッサー ：Zilog Z-80、クロック周波数3.6 MHz。

RAM ：24 Kバイトのメインメモリ、16 Kバイトのビデオメモリ。

サウンド ：4チャンネル（3ボイス、1ノイズ）。 各チャンネルのサウンドを左右にシフトして、擬似ステレオサウンドを得ることができます。

スクリーン ：3.2インチ、バックライト付き。 解像度160 x 146、20 x 18セル、各8 x 8（可視部分）; 256 x 224、各8 x 8の32 x 28セル（合計解像度）：パレットの色の数は4096、同時に表示される色の数は最大32です。



ちなみに、改造者は自分のデバイスに大きなバッテリーを内蔵しており、各ポータブルデバイスにはこのような容量のあるバッテリーがあります。5700mAは6時間の連続再生に十分です。 デバイスの突然のシャットダウンを防ぐために、バッテリーが少なくなると点滅を開始する特別なLEDがあります。 だからあなたは生き残ることができます。
