ローカルToDoプログラムの概要



このレビューは、主にコンテキストコンテキスト（場所、カイロ）に焦点を当てています 。 多数のケースを整理するのに最も有望で便利なように思えます。



MacOSでは、レビューhttp://habrahabr.ru/blog/gtd/27859.htmlに iGTD（ http://bargiel.home.pl/iGTD/ ）があります。 レビューのプログラムhttp://www.habrahabr.ru/blog/apple/14495.html しかし、私はWindowsを持っていますが 、このシステムにとって快適なものを見てみましょう。 すべてのプログラムは、 Windows XP Pro SP2の下にインストールされます 。



さまざまな理由で、このレビューには私に知られているものが含まれていません：LeaderTask 5.2（ http://www.leadertask.ru/ ）、ToDoList 5.3（ http://www.abstractspoon.com/ ）、Rainlendar 2.1（ http：//www.rainlendar .net / ）、Active Desktop Calendar 7.1.6（ http://www.xemico.com/adc/ ）、Mozilla Sunbird 0.7（ http://www.mozilla.org/projects/sunbird ）。







EssentialPIM Free 2.12

ウェブサイト： http : //www.essentialpim.com/

http://demoblog.ru/archives/229

個人評価：**





気に入らなかったもの ：

何らかの理由でメインウィンドウのカテゴリは、署名のない色付きの四角形で表示されます。色の意味を覚えておく必要があります。

場所（カイロ）によるアルファベット順の一般的なグリッドのみでの並べ替え。


何が好きでしたか ：

今日の最初のブックマーク。

多くのアクションをホットキーに掛けることができます。

サウンド、ポップアップウィンドウ、選択したプログラムの起動によるリマインダー。




C-Organizer Pro 3.7.0

ウェブサイト： http : //www.csoftlab.com/

個人評価：**





気に入らなかったもの ：

すべての場所（kairos）でタスクを表示することはできません。

カテゴリはまったくありません。


何が好きでしたか ：

場所（kairosi）は、まずツリー（フォルダー）の左側に作成する必要があります。

タスクの繰り返し期間を設定できます。

アマチュア機能（1つのボタン）の場合、タスクを仮想「ステッカー」に駆動します。 時々役立つ。

トレイのアイコンに現在の番号が表示されます。






ライフバランス3.3

ウェブサイト： http : //www.llamagraphics.com/

http://www.improvement.ru/zametki/lifebalance/

個人評価：****





気に入らなかったもの ：

エクスポートなし、テキスト形式へのインポート。

リマインダーはまったくありません。


何が好きでしたか ：

作業ファイルの形式はxmlであり、必要に応じてテキストに上書きすることもできます。

私はいつも手のひらを持っています。 完了したタスクをDayNotezに記録します。

Kairos（場所）は個別に導入されます。 タスクのツリー（概要）を個別に。

3番目のタブ（ToDoリスト）で、選択したカイロのケースの結論を個別に示します。

さまざまな色のバックライト。

労力の分布を推定します。

努力と重要性の両方は、連続スライダーとしてマウス（スタイラス）で変更されます。






MyLife Organized 1.9.1

ウェブサイト： http : //www.mylifeorganized.net/

http://www.improvement.ru/discus/messages/14/432.html

個人評価：*****





気に入らなかったもの ：

フォントの色やアイコンをプロジェクトに割り当てることはできません。


何が好きでしたか ：

お気に入りのMindManagerにエクスポートします（コメント、カイロ、プロジェクトの完了率を含むxml形式で）。

オプションでインターフェイスの複雑さのレベルを調整できます。

タスクツリーには画面出力フィルターがあります。

目標（目標）とプロジェクト（プロジェクト）への分割。 プログラムはプロジェクトの割合を計算します。

アウトラインでは、要素には5つの理解可能なグループに分けられた十分なパラメーターがあります。 価値のあるリマインダー設定。

ホットキーをブックマークする可能性。

ToDoには、「私にはこれだけ時間があります」というオプションと、赤い四角が付いた今日のToDoディスプレイがあります。






ツドゥモ0.6

http://demoblog.ru/archives/395

http://softodont.ru/organayzeryi/2007/10/30/tudumo-todo-list/

個人評価：****





気に入らなかったもの ：

絶対にどこでも同期しない。

結論 ：私は手のひらを持っていますが、私にとっての主なプログラムはライフバランスです。 PPCは、MyLife Organizedに考えずに切り替えます。 また、興味を持って（GTDが大好きです:) Tudumoの開発（ http://blog.tudumo.com/ ）に従います。 ほとんどの場合、このプログラムはWVの下で私のラップトップのメインスケジューラになります。
