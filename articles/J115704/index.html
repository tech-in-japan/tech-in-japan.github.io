サイトの表示を確認するために最も必要な20のツール

コンテンツの可用性とサイトでの情報の表示は非常に重要です。 簡単に言えば、特定の人々がそれを見ることができない場合、あなたのサイトにあるものに何が良いですか？ もちろん、小さな活字と低コントラストの色を使用したこの新しいトレンディなデザインは、あなたやあなたの友人にとってはクールに見えますが、視力の低い人、色を区別できない人、または他の問題がある人はどうですか？視覚で-彼らは単にあなたのサイトで少なくとも何かを読むことは言うまでもなく、単に何も見つけることができません。 「必要」ではなく、必要であり、できるだけ多くの訪問者がサイトにアクセスできるようにする必要があります。



しかし、ここにあなたのサイトの可用性を保証するために何をすべきですか？ この記事を読むと、タスクが大幅に簡素化されます。 以下に、サイトをよりアクセスしやすくする20のツールがあります。



翻訳スポンサー： Aiken Studio


波





WAVEは、サイトの可用性を評価するプロセスを支援する無料のツールです。 複雑な技術言語で書かれたレポートを発行する代わりに、WAVEはアクセシビリティのレベルを示すアイコンとインジケーターを埋め込むことでWebページのオリジナルを表示します。



IDI





IDI Web Accessibility Checkerは、Webページがサイトのアクセシビリティ標準に準拠しているかどうかをチェックして、誰もがテスト対象のサイトのコンテンツにアクセスできるようにします。



ビスチェック





Vischeckは、特定の色を区別しない人がサイトのページをどのように見ているかを示します。 Vischeckサービスをオンラインで使用するには、グラフィックファイルをチェックするか、それを実行してWebページをチェックします。 Vischeckをコンピューターにダウンロードし、インターネットに接続せずに使用することもできます。



カラーコントラストチェック





Color Contrast Checkを使用すると、メインとバックグラウンドの色を選択し、色覚異常に苦しむ人がそれらを見るときにそれらが十分に対照的であるかどうか、および白黒画面でどれだけ対照的かを確認できます。



mobiReady



mobiReady-モバイルデバイスの画面で表示するためのサイトの準備を評価するためのメカニズム。業界の慣習と標準を使用します。 このサイトがモバイルデバイスの画面での表示にどれだけ適しているかを判断するために、評価（1〜5）とWebページの詳細な分析を含む無料のレポートを提供します。



スタンフォードWebアクセシビリティチェック





Stanford Web Accessibility Checkerは、Webページを分析し、アクセシビリティ標準に対する潜在的な障壁をチェックするソフトウェアツールです。 確認したいページのアドレスを彼に与えるか、評価のために作業ファイルを送信することができます。 チェックの結果、選択したカテゴリの可用性に関するすべての考えられる問題を含むレポートを受け取ります。



私の色を確認してください





Check My Colorsは、Document Object Models（DOM）のすべての要素の原色と背景色の組み合わせをチェックし、特定の色を区別しない人に十分なコントラストがあるかどうかを判断するためのツールです。



Lynxビューアー





Lynx Viewerを使用すると、Lynxテキストブラウザーでのページの見た目を確認できます。 また、検索エンジンがサイトをどのように見ているかも示します。 これに加えて、視覚障害のある人がページにアクセスできるかどうかを判断するのにも役立ちます。



クイックページアクセシビリティテスター





Quick Page Accessibility Testerはブックマークレットで、クリックするといつでもWebページのクイック分析を取得できます。 ページ上の明らかな問題を強調表示し、潜在的な問題を警告し、Active Internet Application Accessibility Standard（ARIA）を使用してアクセシビリティを改善できる領域を強調表示します。



グレイビット





GrayBitは、ページで観察されるコントラストを視覚的にテストするために、フルカラーWebページをグレースケールに変換するオンラインアクセシビリティチェックツールです。



アクセシビリティカラーホイール





アクセシビリティカラーホイールを使用すると、特定のカラーペアの可用性を確認できます。 スケールまたは灰色のバーの色をマウスでポイントし、選択をクリックして、メインと背景の色を選択します。 応答で「OK」を取得した場合、この色のペアがアクセシビリティの面で適切であることを意味します。 それ以外の場合は、適切な結果（「OK」）が得られるまで、1つまたは両方の色を変更できます。



ヘラ





HERAは、Webページが「Webコンテンツアクセシビリティガイドライン」（WCAG）仕様にアクセス可能であることを確認するツールです。 HERAはWebページの一連のチェックを行い、自動的に検出できるエラー、またはページが対応する個々のアイテムを特定し、手動確認が必要なアイテムも表示します。



アクセシビリティバレット





アクセシビリティバレットは、正規化された方法でWebページのレイアウトを表示します。 有効、除外、および偽のマークアップ、および不適切に配置された要素を強調します。 生成されたレポートには、可用性の問題に関するすべての警告が表示されます。



シンシアは言う





Cynthia Saysは、標準およびWCAG勧告のセクション506に関連する設計上の欠陥を特定するコンテンツアクセシビリティ評価ツールです。



トー





TAWは、W3C Web Content Accessibility Guidelines 1.0に基づくWebページを分析するためのツールです。 オンラインツールに加えて、個別のソフトウェアをダウンロードすることもできます。



WebAnywhere





WebAnywhereは、JAWSやWindows-Eyesなどのスクリーンリーダーがページにアクセスする方法でページにアクセスできるビジュアルインターフェイスではありません。



カラーコントラストアナラ​​イザー





Color Contrast Analyzerは、Webページで使用されている色の組み合わせをリストするFirefoxブラウザー拡張機能です。 原色、背景色、明るさ/コントラストの度合い、および色と明るさの違いをまとめた表に色を配置します。



ワット





WATは Internet Explorerの追加ツールバーとして埋め込まれ、Webページの可用性のさまざまな側面を手動で確認するのに役立ちます。



Firefoxアクセシビリティ拡張





Firefox Accessibility Extensionはまさにその名前のとおりです-Webページの機能的な可用性をサポートし、Webページの構造とスタイルのレイアウトを確認するために使用できるFirefoxブラウザー拡張機能。 これにより、障害のある人がWebページを簡単に閲覧およびナビゲートできるようになります。



アクセシビリティファブレット





アクセシビリティファブレットは、ウェブページでの作業におけるアクセシビリティの問題を解決するブックマークレットのコレクションです。 それらは保存する小さなスクリプトで（たとえば、ブラウザーのブックマークとして）、それらを使用してWebページの1つでアクセシビリティを確認できます。
