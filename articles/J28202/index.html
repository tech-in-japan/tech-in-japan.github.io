Windows Liveライター





こんにちは友人たち。 この記事では、Windows Live Writerなどの便利な機能について説明します。



これは、 Live.comが伝えていることです（そこからプログラムをダウンロードできます）。



Windows Live Writerは、さまざまなブログコンテンツを簡単に公開できるデスクトップアプリケーションです。



主要なブログサービスに投稿する

サポートされるサービスは、Windows Live Spaces、SharePoint、Wordpress、Bloggerなどです。 まだブログがありませんか？ Writerは作成に役立ちます！



苦労せずに素晴らしいブログを作成する

写真やビデオ、タグ、その他のすばらしいコンテンツをすぐに挿入できます。 強力な編集機能には、スプレッドシート、スペルチェッカー、クイックハイパーリンクが含まれます。

ブログエントリを表示する

ライターは、テンプレートとブログのフォーマットを使用して投稿する前に、ブログ投稿がどのように見えるかを示します。



オフラインで投稿を作成する

ネットワークに接続したときに公開します。



そして、あなたは知っています、私は彼に同意します、これは本当に非常に便利で無料のプログラムです。 見たとき、私自身も驚きました。 インターフェースを見てみましょう：







最も興味深いコマンドを検討してください。



印刷 -作品を印刷できます



プレビュー -または公開前に記事をプレビューできます



ブログ投稿 -開発したブログ記事を自動的に公開します。



このプログラムを使用すると、ハイパーリンク、タグ、表、写真、ビデオを記事に簡単に挿入できます。 特別な機能は、カードの挿入です。



Windows Live Writerには非常に優れたヘルプがあり、コミュニティフォーラムではいつでも質問に答えることができます。
