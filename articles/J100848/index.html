暗い部屋と日光の下でのテスト。 手で穏やかな条件で撮影すると、揺れが最小限に抑えられます。 EISはうんざりするため、このモードはこのカメラで唯一成功したモードです。







購入する前に、少なくとも1つの大きな欠点があることを知っていました。この欠点はもちろん電子画像安定装置です。 このような小さな光学系を備えたこのようなコンパクトなデバイスのビデオ品質は驚くべきものであると同時に、 EISを台無しにすることを認識する価値があります。



このカメラは、どのようにエクストリームスポーツのファンに推奨できますか？ 私はそうしません。 このカメラを撮影するための唯一の可能なオプションは穏やかな条件でのみであるように私には思えます。 そうしないと、最終的に、映像を10分間表示した後に脳が発狂するビデオを揺さぶることは避けられません。



しかし、欠陥については十分です、カメラの利点を見てみましょう





カメラメニューはかなり簡潔に作成されており、ユーザーはあらゆるレベルで少数の設定を理解できます。 ディスプレイは、もちろん、平均的なレベルの明るい色で目を引きませんが、それでも、画面に表示される画像に2つのフィルターのいずれかを適用できるため、晴れた日でも非常に快適に撮影できます。



このカメラは、最大32 GBのSD / SDHCメモリーカードをサポートしています。 SanDisk SDHC 8 GB Class 2とKINGMAX SDHC 16GB Class 10の 2枚のカードを自由に使用できます。 私は自信を持って、2番目のクラスでどんな撮影モードでも十分だったと言えます。 しかし、もちろん、映像をコンピューターに放り込むことは、高速ではるかに優れています。



ビデオは、付属のHDMIケーブルを介してカメラまたはTVまたはモニターに直接出力することもできます。



また、標準バッテリーからカメラを撮影する時間についての神話を払拭したいと思います。さもなければ、レビューにはこれに関する非常に主観的な声明が含まれています。 4サイクルの充電/放電の後、テストを実施します。 充電して、窓から写真を撮るために即席の三脚に置きます。 カメラは正確に1時間44分21秒続きました。 悪くないと思います。 eBayでこのカメラのバッテリーのペニーコストを考えると、いつでもスペアキットを入手できます。 付属の充電器またはコンピューターのUSBポートからカメラにUSBケーブルを接続すると、バッテリーが充電されます。 オプションとして、外部充電器を購入できます。



カメラは実際に水中で離陸し、3メートルの深さまで潜り込もうとしました。 はい、カメラには何も起こりませんでした。 私にとって、このモードは間違いなくプラスですが、多少異なる平面上では、例えば大雨のときにカメラを恐れて撮影することはできません。 自然に行くときにどんな条件で自分を見つけることができますか？ このカメラはそのような些細なことを気にしません。 電子の詰め物は厚いプラスチックの層の下に隠されているため、小さな高さからの偶発的なカメラの落下を恐れてはいけません。 背面は、柔らかい肌触りの素材でできています。 キットに付属のストラップと一緒に、これはほとんどの不快な事故からあなたを守ります。 ただし、水中で撮影するときは注意してください。ストラップはあまり役に立たず、カメラは手から簡単に抜けてしまいます。手rowぎは絶対に不便です。



さらにいくつかの欠点に注意する必要があると思いますが、自動ホワイトバランスには多くのことが望まれています。 明るいオブジェクト（たとえば、光源）から影のオブジェクトへの急激な移行により、カメラのロジックが明らかに停止し、撮影の結果が期待どおりにならない場合があります。 おそらく、これは新しいファームウェアリリースとして修正されるでしょう、見てみましょう。 まあ、Kodak PlaySportをカメラとして使用することはお勧めしません。画像の品質は一言で説明できます-許容できますが、それ以上ではありません。
