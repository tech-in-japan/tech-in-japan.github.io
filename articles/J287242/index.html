KommersantとMAIL.RUは、ソーシャルネットワークでのプロモーションに関する実験を実施しました

Habrに沿って登ったところ、KommersantとMile.Ruの共同プロジェクトについて地元の住民に知らせてくれた人は誰もいないことに気付きました。 それで、私はこのギャップを埋めようとします、それはちょうど2ヶ月（！）です。



そのため、Kommersant内での議論は9月中旬に始まり、10月12日に既に実験が開始され、発表されました。 さて、発表されたように？ はい、 Kommersantにこれに関する単純な出版物がありました。 要するに、あなたが非常に怠lettersな手紙を読んだ場合、一番下の行は、ソーシャルネットワークでのインターネットプロモーションで自分自身をテストしたいいくつかの企業（22 + 6はフリーエージェントとして）が選択されたということです。 Kommersantの役割は、そのすべてをページでカバーすることです。 Mail.ru、私が理解しているように、専門知識、技術的なプラットフォームなど



実際、参加企業のリストはこちらです： http : //www.kommersant.ru/money/social_project/



また、そこに会社のレポート（2か月は冗談ではなく、多くが蓄積されています）を見つけることができます。 あなたが判断するためのレポートの率直さ。



ところで、私もこのプロジェクトに参加しています。 残念ながら（または幸いなことに）私は主催者ではないため、すべてのプロジェクトに責任を負うことはできません。 参加者の一人に参加しました （コソゴロフムーンシャイン）。 ちなみに、ムーンシャイナーは22 + 6社すべての中で最も奇妙です。 彼らは密造酒をまったく宣伝していません（まあ、またはほとんど宣伝していません...私は単一のボトルを宣伝しませんでした:)）、彼らはKOSOGOROV SAMIZDATでブログを作りました（そしてこれはハブのこのブランチにも当てはまります）。 これはWEB 2.0ではなく、起業家精神に関する民俗ジャーナリズムのメディアです。 誰でも著者になれますが、ブログで好きなものを書くのではなく、実際のメディアで好きです。 実際、KOSOGOROV SAMIZDATには、ムーンシャイナー自身からなる編集スタッフがいます...まあ、まともなメディアと同様に、料金も支払われます（ところで、それは500ルーブルのレベルです。千ルーブル）。



最初に、このすべてのKOSOGOROV SAMIZDATはLiveJournal（ ru_kosogorov ）に存在し（現在、彼は実際にそこに住んでいます）、そして今、私の努力により、彼はスタンドアロンのブログ（ samizdat.kosogorov.ru ）に移動します。 想定される



他の誰かが言うのを忘れた...はい。 なぜこれらのムーンシャイナーは自分たちが編集者を想像しているのですか？ 彼らの構成は、Kommersantの戦闘演技ジャーナリストの75％であるというだけです...（+私はD-bar / EXPERTから支援していますが、編集方針に参加することはできません）。 私はむしろ「自発的なマーケティング担当者」であり、一部は技術的な側面も持っています。 サイトは再びそれを見たが、おそらく私はここでそれをどうやってやったのかはわからないだろう...このために「 I am PR 」というブランチがある。



PS HABRAのこのブランチに興味があり、Kommersantとmail.ruのこの実験がどのように進行しているかを確認したい場合は、合図してください。 新しいレポート、最近の出版物などの出現についてお知らせします（たとえば、週に1、2回）。 KOSOGOROV SAMIZDATの成功/失敗についても説明できます。 これがすべてわからない場合は、標識を付け、カルマを切り取り、言葉で言います。 :)
