Visual Studio 2012 RCのメニューからCAPSを削除する方法

最近、 Visual Studio 2012 RC がリリースされました。多数のインターフェイスの変更とともに、大文字で入力された要素がメインメニューに表示されました。 誰もがそれを気に入ったわけではありません（私は個人的に気に入っていますが 、いつどこでキャップを適用するか読むことができます ）。 以下は、メニューを古い状態に戻す方法です







古いメニューに戻るには、Windowsレジストリに次の変更を加えます。



HKEY_CURRENT_USER \ Software \ Microsoft \ VisualStudio \ 11.0 \ General \ SuppressUppercaseConversion

REG_DWORD値：1



Visual Studio 2012 RCの新しいバージョンは、特別な短いアドレスvs11.ruからダウンロードできます。 素晴らしい開発を！
