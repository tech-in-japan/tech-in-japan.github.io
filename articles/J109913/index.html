Java用Google App Engineの無料のオープンソースCMS

こんにちは、HabraUser。

今日は、Java Vosao CMS™向けのGoogle App Engine専用に作成された無料のオープンソースCMSについてお話したいと思います。









プロジェクトについて



プロジェクトのウェブサイトwww.vosao.org

ソースはcode.google.com/p/vosao/からダウンロードできます。



Vosao CMSには、次のような最新のCMSに必要なすべての属性が含まれています。

ビジュアル（WYSIWYG）編集、

MediaWiki構文のサポート、

わかりやすいURL

国際化、

親/子ページシステム、

ネストされたプログラム可能なテンプレート、

コメント（節度あり）、

サイトのインポート/エクスポート、

コンテンツのバージョン管理、

セキュリティおよびグループ管理、

リソース管理

reCaptcha、

Google Analytics

プラグイン

入力フォーム



必要なプラグインを書くこともできます



CMSがjqueryを使用していることを嬉しく思います。



良いことは、プロジェクトがロシアのルーツを持つ人々によって開発されているということです。



そして、GAEにVosao CMSをインストールすることから始めましょう





1. GAE Java SDK code.google.com/appengine/downloads.htmlをダウンロードしてインストールします（お使いのコンピューターにJavaマシンバージョン6が既にインストールされていることを願っています）

2. Vosao CMS WARファイルをダウンロードして解凍します。

3.ファイルWEB-INF / appengine-web.xmlのアプリケーション名をGAE上のアプリケーションの名前に変更します

4. GAEでCMSを入力します。

（Linux）$ {SDK binディレクトリ} appcfg.sh update {展開されたCMSのあるディレクトリを指定}

（Windows）> {SDK binディレクトリ} appcfg.cmd update {展開されたCMSのディレクトリを指定}



インストール後、クラウドコントロールパネルを見ました。



インスタンス数平均QPS平均レイテンシ平均メモリ

3合計0.006 40.0 ms 52.8 MBytes



Googleが無料アカウントで最大3つのインスタンスを提供してくれたことを非常に嬉しく思いました!!!



Vosao CMSを構成する



デモ版はオンラインで感じることができます。

ログインページに行きます

ユーザー名：admin@test.comおよびパスワード：adminを入力します

ロシア語を選択してお楽しみください=）





CMSの構成を見てみましょう





非常に優れたエクスポート機能は、サイトのインポートです。これは、更新して別のホスティングに移動するときに非常に便利です。



デフォルトの配信では、フォームとSEOリンク用のプラグインは2つしかありません。 プロセスは非常に簡単なので、残りのプラグインは自分でインストールする必要があります。



Picasaを使用してメディアコンテンツを保存することもできます。
