すべての都市へのジャグ



Java User Group （JUG）は、Javaプログラマー、Javaユーザー、およびIT企業を集めるために設計されたJava開発者のボランティア組織です。 通常、たとえばJava開発者のモスクワコミュニティやベラルーシのJavaユーザーグループなど、地理的に統合されています。 彼らは、Javaユーザーが情報を交換し、Javaの世界の問題と解決策について話し合い、何よりもビールを飲み、ピザを食べ 、 楽しむための出会いの場を提供します。

個人的に、私はfeel辱されたように感じます。私たちの国の広大な領土全体には、モスクワJUG、サンクトペテルブルクJUG、およびオムスクJUGのみが存在することがわかります（マップ上にないものが他にもありますが、それらに関する情報は見つかりませんでした） 。 ウガンダとラゴスにも独自のJUGがあります。



なぜこれが必要なのですか？


誰もが労働市場の状況を知っています。Java開発者の需要は非常に高く、モスクワで「有能な」人材を見つけることが本当に可能であれば、地域ではほとんど非現実的です。

時々、彼らは私に次の質問をします。

-こんにちは、ここでJavaを知っている知り合いがいましたか？ そして、プロジェクトが計画されています。

（私が出身地である「ここ」、今はモスクワに住んでいます）

-うーん...はい、しかし彼らはおそらく忙しいです

-さて、突然誰がそこで育ったのですか？

...



人々はフリーランスに行き、スタートアップを開始します。プロジェクトだけを引っ張るのは非現実的です。 20年の経験を持つ第一人者をあなたのランクに引き付けることができますが、最初の一歩を踏み出した後輩を連れて行く初心者が必要なものであり、JUGプラットフォームは交流して適切な知り合いを作るのに最適な場所です。



独自の水差しを作成する方法


  • 始めるには、 java.netで登録してください
  • 次に、 作成ページに移動して 、プロジェクトの要件を確認し、[ プロジェクトの作成 ]をクリックします
  • Project Nameフィールドに入力します。この名前はURL project_name.java.netおよびDescriptionで使用されます。SourceLicenseを選択して、createをクリックします。


その後、プロジェクトが正常に作成されたことをメールでお祝いします。



プロジェクトが作成されました。java.netが提供するものを見てみましょう

  • バグトラッカー-JIRAまたはBugzillaを選択できます
  • リポジトリ-subversion、mercurial、またはgit
  • メーリングリスト
  • ファイルストレージ
  • フォーラム
  • チャットルーム
  • ウィキ
  • その他の小さなもの




JUGは作成されましたが、現時点ではプライベートであり、 Jugs Listにはなく、 マップにもありません。 プロジェクトを公開するには、マネージャー（Java.net Community Manager）にプロジェクトを開く要求を記載したフリーフォームで書き込む必要があります。 すべてが問題なければ、しばらくして（数時間）彼らはあなたに答えてプロジェクトを開きます。

JUGをマップに追加するには、 Java User Groups International Mapに、特定のタイプのxmlをコミュニティマネージャーのメールに送信する必要があります。次に例を示します。 



<Placemark> <name>Baikal Web Developer JUG</name> <description> <![CDATA[ Up to a maximum of 250 characters of general group description text can precede the JUG Leader and JUG Site information.<br/> <b>Leaders:</b> <a href="http://ilinchik.ru">Anton Ilinchik</a> <br/> <b>Site:</b> https://baikal.java.net/ ]]> </description> <Point> <coordinates>107.600000,51.833333,0</coordinates> </Point> <styleUrl>#jugStyle</styleUrl> </Placemark>


そして、新しいJUGがマップに表示されます





これで、JUGが正式に登録され、リストと地図に載りました。今では中小企業です。レポートを準備し、スポンサーを探し、ビールとピザを購入しています。
