JetBrains WebIDE-提案と提案

提案：
  1. ファイルのグループをFTPにアップロードする機能：たとえば、複数のファイルを選択/マークして、それらを一度にアップロードします。
  2. タブをドラッグする機能。
  3. IDEの言語を選択する機能：たとえば、スペイン語のウィンドウでは、メニューの一部は英語で、一部はスペイン語で書かれています。
  4. Linuxでプロジェクト（ディレクトリ）を開くと、WebIDEは隠しファイル（ピリオドで始まる）を表示し、これはオフになりません。
  5. プロジェクト（ディレクトリ）を開いたときに、プロジェクトが開かれた最後のディレクトリが表示され、さらに良いことに、現在のシステムのルートではなく、最後からの親が表示されると便利です。 または、そのようなオプションを提供します。
  6. 新しいプロジェクトを作成することなく、ファイルを1つだけ作成する機能。 たとえば、1つのファイルのみを作成する必要があり、そのプロジェクトを作成する必要はありません。
  7. ローカルマシンのサーバーにプロジェクトのコピーがない場合、作業する方法はありません。たとえば、複数のファイルを編集する必要がある場合など、ftpに接続してファイルを直接操作できるようにしたいと思います。ローカルマシンのディレクトリ。
  8. 終了時にファイル内でどのブロックが折りたたまれたかを思い出す機能と、同じファイルを再度開くと、これらのブロックが折りたたまれ/展開されます。 理想的には、ファイルを開くと、すべてのブロックが折りたたまれるか、すべて展開されます。
  9. また、WebIDEを編集するときに、上下のブロックまたはこの設定のオプションが拡張されないようにしたいと思います。
  10. 画面を最後の行までスクロールして、コードを記述するウィンドウを常に見下ろさないようにする機能。
  11. タグ付け。 たとえば、書式設定する必要があるテキストをHTMLに挿入しました（段落にマークを付けたり、太字のものを強調表示したりなど）。 テンプレートを規定し、挿入物で選択したテキストをフレーム化するボタンを引き出したいと思います。 たとえば、テンプレート
    < b > | < / b >
    テキストを太字にします（つまり、テキストolrpolporpoを強調表示し、ボタンをクリックすると、選択したテキストの周囲にタグが挿入されます：
    < b > olrpolrporpo < / b >
  12. NetBeansにはコメントのように見える特別なタグがありますが、それらを見つけると、これは折りたたむことができる擬似ブロックであることがわかります。 さらに、このタグには、説明と初期状態の2つのパラメーターがあります。 説明-これは、ブロックが折りたたまれたときに表示されるテキストです（フレームにはこのテキストと展開用の「プラス記号」のみが表示されますが、ブロック自体はありません）。 状態は、ファイルがアップロードされたときにブロックのテキストがどのように表示されるべきかをIDEに伝えるフラグです-最小化または最大化、これはNetBeansでの表示です：
    <エディター-fold defaultstate = "collapsed" desc = "ここにあるものを記述してください" >

    ここにコードの束があります

    </エディター-折りたたみ>
    ブロックを最小化すると、ここ（フレーム内）の内容を説明するテキストのみが表示され、IDEを使用すると、わずか数クリックでテキストの一部をフレーム化できるようにマクロを構成できます。これらのブロックがナビゲータ上に表示されると便利です。ジャンプすることが可能でした
  13. htmlタグ内に属性を挿入して「=」記号を挿入すると、WebIDEは自動的に引用符のペアを挿入します-便利ですが、問題は引用符の後にスペースを入れないことです。 したがって、別の属性の前に属性を挿入する場合、ペンでスペースを挿入する必要があります...それ以外の場合、それらは一緒に接着されます。
  14. phpdocだけでなく、doxygenもサポートしたいと思います。
  15. 挿入された言語の色付けを記憶する機能、たとえば、引用符で囲まれたsqlクエリがあり、言語（Insert Language）が挿入され、必要に応じてsqlクエリが強調表示されますが、次回プロジェクトを開くときは、もう一度言語を挿入して再実行する必要があります引用符で囲まれたクエリのバックライト。
  16. 賢いコメントを折りたたむ能力
  17. 右側のパネルのShift + F11->ブックマークとラベルだけでなく、ホットボタンでブックマークをナビゲートする機能。
