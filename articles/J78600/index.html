VESA （Video Electronics Standards Association、Video Electronics Standards Association）などの協会があります。 かつてMIS （Mounting Interface Standard、Mounting Interface Standard）としても知られているFDMI （Flat Display Mounting Interface）という長い名前の特別なインターフェースを思いついたのは彼女でした。 そして、まとめてVESA FDMIS （ビデオ電子標準協会フラットディスプレイ取り付けインターフェイス標準）と呼ばれます。



一般的に...今あなたがそれを呼ぶものは何でも、これの本質は変わらないだろう-モニター、タブレット、およびその他のスクリーンを備えたパネルを製造するほとんどの企業は、このまさに協会によって厳密に標準化された特別な穴をデバイスの背面パネルに装備している。 簡単にするために、他の多くの人がそうするように、私はそれをVESA標準と呼びます。



VESA標準には、ディスプレイ設定に応じて5つの異なるインターフェイスがあります。 下の表からわかるように、市場で入手可能なほとんどのデバイスが対象です。







数字を見ると、最も一般的なインターフェイスはMIS-Dであると簡単に推測できます。MIS-Dの穴は、一辺が75ミリメートルまたは100ミリメートルの正方形の角にあります。 誰かが意図した目的のためにこれらの穴を使用し、誰かがそれらをただ見ただけで、誰かはそれらについてさえ知らなかった...しかし、誰か、はい、彼らは単に知らない）



だから、私は何ですか...そうそう。 VESA協会のメンバーである外国企業Ergotronは、あらゆる種類のファスナー（何かのためのもの）の開発に携わっており、この機会を利用して、この問題の世界的リーダーです。 多数の特許およびその他のボーナスにより、同社はほとんどすべての場合に人間工学に基づいたソリューションを作成しています-一般に、ウェブサイトで同様のことをたくさん読むことができます）

もちろん、これはすべて美しいですが、好きなように書くことができます。 別のことは、それがいかに厳しい現実にあるかです。 ちょうど1年前、サンクトペテルブルクで開催されたHIT2008展示会（参考として興味深い）で、大規模なオーバークロッカーと改造スタンドを組織しました。 そしてつい最近、私は通信の印象を共有したい一対のデバイスの誇りに思う所有者になりました。
