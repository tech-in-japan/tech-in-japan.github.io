Mozillaが最初の開発者向けブラウザーを起動

以前に報告したように 、Mozillaは開発者専用に作成された最初のブラウザをリリースしました。 新しいアイテムのリリースに関する公式プレスリリースが11月10日に利用可能になるという事実にもかかわらず、今すぐ新しいブラウザを試すことができます。







Firefox Developer Editionに関する新しい情報が知られています：



  • Firefox Developer Editionは、既存のFirefox Aurora開発チャネルを置き換えます。 したがって、開発者向けのアセンブリは、Firefox Nightlyからイノベーションを取得します。 6週間のブラウザー開発サイクルも保持されます：Nightly-Developer Edition-Beta-Release。 したがって、開発者は、イノベーションがリリースされるまでに12週間かかります。
  • 新しいブラウザは別個のユーザープロファイルを使用するため、Firefoxと同時に実行できます。
  • 開発者ツールへのアクセスを加速する更新されたインターフェース。 デフォルトでは、ダークテーマが有効になっていますが、ブラウザは標準テーマへの切り替えとサードパーティのテーマと拡張機能のインストールの両方をサポートしています。
  • 実験的な開発ツール。 新しいブラウザの構成には、開発の初期段階にある開発ツールが含まれています。 それらの1つは、 Firefox Tools Adapterでした。これにより、Firefox開発ツールを使用して、他のブラウザー（iOSのSafari、AndroidのChrome）でデバッグできます。 さらに、新しいFirefoxには、以前に報告された便利なWebIDEツールが含まれています。


Mozillaの革新とブラウザ開発の計画に関する詳細情報は、2014年11月10日に見つかります。
