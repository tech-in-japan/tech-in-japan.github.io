2016年のMicrosoftからのトップ10の主要な発表

伝統的に、私たちは過去1年、Microsoft Corporationからの重要な発表の選択に2016年を費やしています。 カットの下で、私たちは最も明るく、最も記憶に残る10のニュースを収集しました。







1.人工知能の民主化戦略



9月下旬のMicrosoft Ignite 2016で講演したSatya Nadellaは、同社の野心的な目標である「 Democratizing Artificial Intelligence 」を表明しました。 AIテクノロジーを誰もがアクセスし、理解できるようにすることです。



サティヤはまた、当時のユニークなケースをユーバーと共有しました。 同社は、 Microsoft Cognitive Servicesに基づいて実装された、ドライバーと乗客の安全性を向上させる新しいリアルタイムIDチェックテクノロジーを導入しました。 その本質は、各ドライバーが携帯電話のカメラを使用してそれを認識するモバイルアプリケーションをインストールするという事実にあります。



マイクロソフトが10月初旬にロシアでの新しい戦略に向けて最初の一歩を踏み出したことは注目に値します。 ロシア語はSkype Translatorに登場しました 。



秋には、AIの分野における最大のパートナーシップについて、さらに2つのニュースが登場しました。



  1. Facebook、Amazon、IBM、Google、Microsoftは、人工知能のコラボレーションプログラムであるAI on Partnershipの導入を発表しました。



  2. MicrosoftとOpenAIはパートナーになりました 。 OpenAIは人工知能の研究に従事する会社であり、Elon Musk、Sam Altman、Greg Brockman、Ilya Sutskeverが共同で設立した会社であることを思い出させてください。 OpenAIは、研究を改善し、クラウドでのみ可能な新しいツールとテクノロジーを作成するために、Microsoft Azureを主要なクラウドプラットフォームとして選択しました。





12月中旬に、Microsoftは目標を達成するための最初の結果を共有しました ： Cortana Devices SDKの発表、 Skypeでのボットの話、 Microsoft Bot Frameworkの更新、Harman KardonとのパートナーシップによるAIを使用したユニークなスピーカーシステムの作成。 ほんの数日前、同社は、書かれたコンピューターを教えるためのオープンアクセスデータセットの可用性を発表しました 。 これは、Microsoft Machine Reading Comprehension（MS MARCO）と呼ばれ、BingおよびCortanaで匿名の実際のクエリを使用して収集された10万を超える英語の質問とその回答が含まれています。



2.マイクロソフトがXamarinを買収



今年の初め、マイクロソフトは最大のモバイルアプリ開発プラットフォームであるXamarinの買収を正式に発表しました。 その後の会議のフレームワーク（ Build 2016 、 Xamarin Evolve 2016およびConnect（）; // 2016 ）で、今年のいくつかの重要な発表が発表されました。



  • Xamarinは、Visual Studioのすべてのエディションで完全に無料になりました。
  • Xamarin SDKはオープンソースで発表され、XamarinツールのコアであるMonoプロジェクトは、オープンソースのMITライセンスの下で再リリースされました。
  • AppiumサポートがXamarin Test Cloudに登場しました。


3.マイクロソフトがLinux Foundationに参加



マイクロソフトは 、非営利のLinux開発コンソーシアムであるLinux Foundationのプラチナメンバーになっています。 プラチナメンバーには、Cisco、富士通、HP、Huawei、IBM、Intel、NEC、Oracle、Qualcomm、Samsungなどの企業が含まれます。 一方で、このニュースは多くの人に衝撃を与えたかもしれませんが、最近では、PowerShellソースコード、Visual Studio Code、Edge JavaScriptエンジン、LinuxバージョンのSQL Serverの発表に関するマイクロソフトからのニュースが増えていることに注目する価値があります。そして、今年の初めに、UbuntuはWindows 10に統合されました。



4. LinuxでMicrosoft Bot Framework + Azure Bot-as-a-Serviceを起動します



Build 2016カンファレンスの一環として、ボットを作成するための新しいツールであるAPIを含むMicrosoft Bot Frameworkが発表されました。 Bot Builder SDK、 Microsoft Cognitive Services 、およびBot Connectorが含まれており、ボットをSkype、Slack、Telegramなどの1つ以上の通信チャネルにバインドできます。 また、コンテナとASP.NET CoreをサポートするLinuxでのAzure Bot-as-a-Serviceのプレビューバージョンの発売も朗報です。



5. Windows 10周年記念アップデート



8月2日Windows 10のアニバーサリーアップデートが世界中のデバイスで利用可能になりました。







すべての機能の中で区別できるもの：



  1. Windows Ink。画面上にアイデアのメモ、図面、スケッチをすばやく簡単に作成できます。
  2. Windows Defenderによるセキュリティの強化と、Windows Helloを使用したアプリケーションおよびサイトへの安全なアクセス。
  3. 新しいMicrosoft Edgeと拡張機能の出現により、エネルギー効率が向上しました。


6. C＃7.0：プログラミング言語の大規模な更新



2000年に、Microsoft開発チームは、Javaなどのオブジェクト指向プログラミング言語の表現力とC ++のパワーを組み合わせた新しいC＃プログラミング言語を導入しました。 今年は、C＃7.0言語の新しいバージョンに関するニュースがありました。その機能の一部は、Visual Studio“ 15” Preview 4のリリースで一般に利用可能になりました。



9月の初めに、C＃についての興味深い事実をお伝えしました。



7. Visual Studio for Mac、Visual Studio 2017およびVisual Studio Mobile Center Previewのプレビューバージョン



今年、Connect（）開発者向けの最大のカンファレンスの1つ。 // 2016年はニュースでいっぱいでした。 重要な発表は、 Visual Studio for Macのリリースです。 これは、モバイルアプリケーション、クラウドサービス、およびMac用アプリケーションを作成するために設計された完全な開発環境です。 最初のプレビューリリースには、Xamarinを使用してC＃およびF＃でネイティブiOSおよびAndroidアプリケーションを作成するサポート、およびAzureサービスを使用してバックエンドを作成するサポート、そしてもちろんASP.NET Coreのサポートが含まれます。



Visual Studio 2017 Release Candidateのプレビューバージョンが利用可能になりました。 Visual Studio 2017 RCには、以前のバージョンでは利用できなかった修正と新機能の両方が含まれています。 また、このリリースのVisual Studio 2017 RCには、実稼働ソリューションを開発する機能が含まれています。



モバイル開発者にとっての主なニュースは、 Visual Studio Mobile Center Previewの発表です。

Mobile Centerは、Objective-C、Swift、Java、Xamarin、React Nativeで記述されたiOSおよびAndroidアプリケーションと連携し、クラウドサービスとツールを統合して、高品質のモバイルアプリケーションの迅速な開発を目標とする開発者にアプリケーション開発ライフサイクルを提供します。 このツールは、デバイス上のアプリケーションの開発、アセンブリ、テスト、監視、および展開の機会を提供し、開発者がすべてのプラットフォームでアプリケーションをスケーリングし、最新の最も一般的な機能を実装することを容易にします。



8.開発者向けのMicrosoft HoloLens



2016年の初めに、 Microsoft HoloLens Dev Kitの供給が開始されました。 HoloLensのHello Worldを作成するためのガイドと、デバイスの技術的特性を既に共有しています 。







秋に、HoloLens は名誉あるポピュラーメカニクスの2016年ブレークスルーアワードを受賞しました。これは、「過去1年間のテクノロジー界の革新者、国境を越えて世界を少し良くした人」に授与されます。



現時点では、HoloLens向けに積極的に開発を行っているロシア企業がいくつかあります。 拡張現実の最初のロシア語アプリケーションをリリースした HoloGroupと、HoloLensのアプリケーションをリリースした最初のロシア企業となった Zengaltを含みます。



9.マイクロソフトがGitHubで第1位のオープンソース企業に



2016年半ばに、MicrosoftがFacebookよりも先行し、GitHubでナンバーワンのオープンソース企業になったというニュースがありました。 これについて最初に書いたのはRedditユーザーの1人でした。



10.新しいコミュニケーションサービスがリリースされました-Microsoft Teams



Microsoftは、Office 365用の新しいコミュニケーションアプリケーションであるMicrosoft Teamsを発表しました。 チャットの原則に従って編成され、Microsoft Office 365でのコラボレーションの可能性を広げます。MicrosoftTeams Developer Previewも発表されました。



タブ機能を使用すると、通信用に個々のトピックを構成できます。また、頻繁に使用されるドキュメントやアプリケーションにすばやくアクセスできます。 OneNote、SharePoint、PlannerなどのOffice 365サービス、またはサードパーティソリューション用のタブを作成できます。 Exchangeと同様に、新しいMicrosoft Teamsスペースは、TwitterやGitHubなどのサードパーティサービスからの通知と更新をサポートしています。 Microsoft Bot Frameworkもサポートされているため、インテリジェントサービスを企業の作業環境に統合できます。
