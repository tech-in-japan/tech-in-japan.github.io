Googleアナリティクス：Google検索エンジンでのキーワードによるサイトの位置の決定

現在、Web分析ツールを使用して、ユーザーが検索エンジンからサイトにアクセスした際のキーワードなど、さまざまなパラメーターを追跡できます。 ただし、ユーザーがサイトにアクセスした検索クエリに関する情報のみが提供され、訪問者がクリックした検索エンジン内の場所は提供されません。



残念ながら、ほとんどの検索エンジンは、検索結果のどの位置からサイトへの移行が行われたかを追跡する機会を提供していません。 ただし、この機能は検索エンジンGoogleを提供します。 唯一の条件は、Googleによって承認されたユーザーのポジションのみを決定できることです。







許可されたユーザーの場合、Googleはサイトへの直接リンクをリダイレクトリンクに置き換えます。



つまり www.habrahabr.ruに直接移動する代わりに、このアドレスを使用してリダイレクトが実行されます。

http://www.google.ru/url?sa=t&source=web&ct=res& CD = 1＆VED = 0CCQQFjAA＆URL =のhttp ％3A％2F％2Fwww.habrahabr.ru％2F＆RCT = J＆Q = habrahabr＆EI = eYzZS-z_IsiUOPLyyPEP＆USG = AFQjCNEyq837y3ku80b2QBzQxHAFnlN- hg＆sig2 = bPeHds8OYtT6U9ucOemc5Q



cdは、Google検索エンジンでのこのリクエストのサイト位置番号です。



これらのデータを無視しないでください。 この機会を利用して、Google検索システムからの総コンバージョン数の約25％がGoogleに登録されたユーザーのコンバージョンであることが判明しました。



そのため、最初に監視対象サイトのプロファイルのコピーを作成する必要があります。



新しいプロファイルを作成する



[新しいWebサイトプロファイルの作成]ページで、下のスクリーンショットのように[既存のドメインのプロファイルを追加]オプションを選択します。







プロファイルのフィルターの作成



新しいプロファイルを作成したら、その中に3つのフィルターを作成する必要があります。



これを行うには、プロファイル設定に移動し、[フィルターの追加]ボタンを使用してフィルターを作成します。 フィルターを作成するページで、「プロファイルに新しいフィルターを追加する」を選択し、「フィルタータイプ」セクションで「カスタムフィルター」を選択します。



検索トラフィックフィルター



最初のフィルターは「検索トラフィック」と呼ばれ、検索エンジンからのトラフィックをトラフィックプロファイルのみに含める役割を果たします。







Googleトラフィックフィルター



合計検索トラフィックの2番目のフィルターは、Google検索エンジンからのトラフィックのみを考慮します。







Google Positions Filter



3番目の最も重要なフィルターは、検索エンジンGoogleでのサイトの位置を決定するためのフィルターです。



作成するには、2つのフィールドと出力コンストラクターで構成されるカスタムフィルターのタイプ「詳細」を選択します。



最初のフィールド「遷移」は、位置番号と、サイトへの遷移のソースのアドレスから遷移が行われたリクエストを抽出する正規表現で構成されています。



www\.google\.(.*)/.*[?&#]cd=([0-9]+).*[?#&]q=([^&]+)







2番目の「会社検索用語」フィールドは正規表現で構成され、Googleアナリティクス自体によって定義されたキーフレーズです。



(.*)







3番目のフィールド「Designer」は、前の2つのフィールドの値を1つのフィールドに出力することで構成されます。



$B1 (Google position: $A2)







Googleの検索結果でのサイトの位置は、最初のフィールドに表示されます-$ A2

キーワード（キャンペーン検索語）は2番目のフィールドから表示されます-$ B1



デザイナーの結論は、「会社の検索語」フィールドで実行されます。

その結果、Googleアナリティクスのウェブ分析システムのキーワードフィールドに、フレーズ名の代わりに、次の構造が表示されます。

「キーワードフレーズ名」（Googleの位置：「位置番号」）







これらのフィルターを作成した結果、プロファイル設定ページは次のようになります。







結果を見る



これらのフィルターの結果は、Googleアナリティクスシステムの[キーワード]セクションにあります。







開いたページに、キーワードのリスト全体が表示されます。 特定の位置のキーワードを表示するには、「キーワードのフィルター」フィールドに「google position」というフレーズを追加する必要があります。







その結果、Google検索エンジンでの位置を持つキーワードが表示され、訪問者はこのキーワードを使用してサイトにアクセスしました。
