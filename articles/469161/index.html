  • Hollewegs ( Hollewegs



    ) - this road, which is laid in a significant depression. Dirk Kloosterboer used OpenStreetMap and elevation data to analyze the location of the streets called " holloway

  • Very soon, a billion smartphone owners will again be able to use the European Galileo satellite navigation system, this was reported in the press release of the operator. Recall that in July 2019, this system completely failed. It is assumed that her work will be restored this year, but it will work in full force by 2021, when the launch of the new Arian 6 rocket takes place. The orbital constellation will include 24 main and 6 spare satellites. But it is quite possible that the deadlines can be shifted, since the first test flight of the Arian-6 rocket is planned for 2020.
