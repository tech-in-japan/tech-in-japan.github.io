Mozilla Firefoxの市場シェアは20％を超えました。 もう一度。

ネットアプリケーションを参照したLenta.ruの2008年12月2日の最新ニュース：

Mozilla Firefoxの市場シェアは20％を超えました。 今年の11月、Firefoxは市場の20.78％を占めました。 これは、4万サイトのトラフィックの分析に基づいて、ネットアプリケーションによって取得されたデータを参照してComputerworldによって報告されました。

Firefoxはブラウザ市場の5番目の部分を占めています



どこかですでに聞いたことがあります。 Google、わずらわしい。 itnews.com.uaの2008年11月5日のネットアプリケーションに関するニュース：

10月の最初の2週間におけるMozilla Firefoxの世界市場シェアは20％を超えました。

Mozilla Firefoxブラウザーの市場シェアは20％に達しました



それだけではありません。 グーグル、さらに煩わされることなく。 2008年7月31日の Cnews.ruからのネットアプリケーションに関するニュース：

公開時点の最新データによると、Internet ExplorerはすでにWebブラウザ市場の69.88％を占めています。 Firefox 2とFirefox 3の合計シェアは20.68％（Firefox 2-13.75％、Firefox 3-6.92％）です。

Firefoxは世界のブラウザ市場の20％を征服しました



さまざまなニュースサイトに表示されるほぼすべての同様のニュースには、有名な写真が付いています。



「FirefoxはIEから市場シェアを獲得し続けています。」というスタイルの署名を持ちます。 そして、それがブラウザ市場で完結した革命として発表されるたびに。
