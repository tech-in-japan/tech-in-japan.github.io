デュアルSIMをサポートする最初のAndroindスマートフォンの販売開始

一般的なモバイルDSTL1。

Androidオペレーティングシステムを実行します。

アメリカで発売されました。 少なくとも2つのオンラインストアで見つけることができます：Amazon USとArt of Hitech、デバイスのコストは499ドルです。 General Mobile DSTL1の主な機能は、2枚のSIMカードを同時にサポートすることです。これは、Androindに基づくこのような機能を備えた最初のソリューションです。





一般的なモバイルDSTL1は、GSM / GPRS / EDGE（900/1800/1900 MHz）セルラーネットワークで動作し、3G通信規格をサポートしていません。 Sharp 3インチWQVGAタッチスクリーンが装備されており、ハードウェアキーボードはありません。 欠点の中でも、内蔵のGPS受信機がないことに注意することができますが、少なくとも仕様では言及されていません。 しかし、撮影には、ソニーのオートフォーカス機能を備えた非常に優れた5メガピクセルのカメラが必要です。 ワイヤレスモジュールBluetooth 2.0およびWi-Fi 802.11 b / g、メモリカードmicroSDおよびFMラジオ用のスロットもあります。 どうやら、General Mobile DSTL1はオペレーターに関係なく販売されていますが、SIMカード用の2つのスロットがあるため、これは非常に論理的です。



主な機能：

  • ネットワーク：GSM / GPRS / EDGE 900/1800/1900
  • プラットフォーム：Marvell PXA310、624 MHz
  • RAM：128 MB RAM
  • データストレージ用メモリ：256 MB ROM + 4 GB、microSDスロット
  • ディスプレイ：3インチタッチスクリーン、240x400ピクセル、260 Kカラー
  • インターフェース：Wi-Fi、Bluetooth、miniUSB
  • オプション：FMラジオ
  • カメラ：オートフォーカス付き5メガピクセル
  • バッテリー：1200 mAh
  • 寸法：112x54x16 mm
  • 重量：135グラム
