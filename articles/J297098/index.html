4ベンチャー投資家のとんでもない要求

ベンチャー投資家は、スタートアップが閉鎖される前に取引に非常に奇妙な要件を追加することがあるため、スタートアップに対する彼らの力を感じています。 以下は、Inc。のジャーナリストが行った4つの最もばかげた要求です およびRe / Code Jason Del Rey。







1.スタートアップは、投資に関連するすべての支払いを支払います



長年にわたり、ベンチャー投資家は、起業家に対する義務を、投資プロセスに関連する税金やその他の支払いを支払うようにシフトしてきました。 一部の投資会社はこの慣行を止めており、一部の投資会社は投資先の会社からまだお金を受け入れています。



2.スタートアップは、ベンチャー企業の名前または投資家の名前をサイトのメインページに掲載する必要があります。



悲しいですね。 あなたが良い投資家であり、企業の創設者があなたを大切にしているなら、認識の問題はないはずです。 あなたが投資した会社のホームページは、あなたの認識を示すための最良の選択ではありません。



3.投資オファーは限られた期間有効であり、スタートアップは投資家の要求に応じてトップマネージャーを雇わなければなりません。



一部の投資家は、時間によって投資提案を制限し、これに会社の特定の高い地位のために特定の人を雇うという要件を追加するのが正しいと信じています。



4.スタートアップはサンフランシスコ湾に移転しなければならない



サンフランシスコとシリコンバレーは、そこに住んで働いている才能のおかげで、スタートアップの世界で議論の余地のない王です。 しかし、今日、若い起業家は、ニューヨークからストックホルム、デリーまで、世界中のどの都市でも働くことができます。 同時に、一部の投資家は、投資を予定している会社を強制的に移動させることを義務付けていると考えています。
