Synology®がDiskStation DS412 +を発表

今日Synologyがユーザーに提示するもう1つの新製品は、中小企業のお客様向けの4-disk Plus NAS- Synology DS412 +です。 これは、デスクトップパフォーマンスの4つのディスクドライブの中で2012年モデルの最初のモデルであり、 DS411 + IIに取って代わりました。



高いシステムパフォーマンスにより、205.68 MB / sの速度でデータを読み取り、RAID 5 sストレージアレイとアクティブ化されたネットワークポート集約機能を使用する場合、182.66 MB / sの速度でデータを読み取り、データ交換で重要な結果を発揮できます。 その前身と比較して、速度インジケータは、それぞれ85％と73％改善されています。



Synology DS412 +は、高い信頼性と、リブートなしでの長期動作の可能性を備えています。これは、パッシブプロセッサ冷却テクノロジーと統合ファン複製システム（2つの92 mmファンを使用）の組み合わせによって実現されます。 DS412 +は、2つのネットワークポートのうちの1つに障害が発生した場合、またはネットワーク障害が発生した場合にも装備され、ホットスワップドライブをサポートします。



Synology DS412 + NASには以下が装備されています：



  • 2.13 GHzデュアルコアプロセッサ
  • 1GB DDR3 RAM
  • 2つのギガビットLANおよびリンクアグリゲーションポート
  • 2つのUSB 3.0ポート、1つのeSATAおよび1つのUSB 2.0


主な機能：

  • 最大4TBのハードドライブのサポート（別売）
  • ホットスワップ可能なハードドライブ
  • サポートされるRAIDタイプ：Synology Hybrid RAID、Basic、JBOD、

    RAID 0、RAID 1、RAID 5、RAID 5 +スペア、RAID 6、RAID 10
  • VMware / Citrix / Microsoft Hyper-Vとの互換性
  • Haspキーのサポート
  • 接続されているIPカメラの最大数は20です*
  • 低消費電力-60 W（動作モード）、20 W（スリープモード）




USB HASPキーをNASに直接インストールするサポートにより、このドライブは1Cデータベースサーバー機能と同時に1C：Enterpriseコピーを認証する手段になります。 このような枯渇により、1Cデータベースの信頼性の高いストレージが可能になり、データベースへのグループアクセスが提供され、ソフトウェア使用の独立性が確保されます。 Synology HASPパッケージに基づくソリューションの段階的な実装をここに示します 。



中小企業では、DS412 +はファイル共有ストレージ、バックアップサーバー、VPNサーバー、ビデオ監視サーバーの機能に完全に対応します。



ロシアでは、4月上旬にノベルティが利用可能になります。 暫定小売価格-30 500ルーブル（ディスクのコストを除く）



Synology DS412 +の詳細については、 www.synology.suをご覧ください 。
