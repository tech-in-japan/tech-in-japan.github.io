ロシア向けのアプリケーションを無料にしましょう！

残念ながら、Androidマーケットの有料アプリケーションはロシアのユーザーがまだ利用できないため、無料のものに満足するか、「他のソースから入手」する必要があります。



最近、選択したプログラムのAndroidでの起動をブロックできるプログラムを作成しました 。 ご要望に応じて：Gmail、SMS、Googleトーク、メール、マーケット...

つまり 通常の作業では、ロック解除パターンを使用することも、キーロックのみを残すこともできますが、起動時にGmailと言う場合は、パスワードを入力する必要があります。 再入力は、画面をオフにした後にのみ必要です。



私はどういうわけか自分に感謝し、プログラムを0.99ドルでリリースしたかった-それがどうなるか見てみよう。 うまくいく=）多くの人は、そのようなプログラムを非常に長い間待っていると書いている。



ロシア人に違法コピーを強制しないようにするために（セキュリティ関連のアプリケーションは定期的に更新する必要があります）、彼はロシア語を携帯電話に持っているすべての人にデモ版を無料にするアイデアを思いつきました 。



とてもシンプル-ダウンロードして制限なしで使用できます。 最新バージョンに更新し、市場にレビューを残すことを忘れないでください-良いカルマはそこにある最も貴重なものです。



実際には、主なこと： 開発者への呼び出し 。 とても簡単です！ 問題を解決し、高度な同胞を幸せにするための多くの方法を考え出すことができ、限定バージョンを完全に使用することができます。 とにかく、正式に購入する方法はありません。



Far Manager 1.xでアイデアを見つけました-登録キーとしてロシア文字で現在の曜日を入力することができました。
