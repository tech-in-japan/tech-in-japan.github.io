「給与は成功した候補者と話し合っています...」

雇用主が特定の給与額を示したり、「$ <実質的に低い市場>から」を示すことを嫌うのはなぜですか？



オプション：



1. お金がありません！ これは真実ではありません。 これが当てはまる場合、空席は「危機管理者」と呼ばれるべきであり、お金をspareしむべきではありません。事業が大きい場合、小さい場合は、監督がタクシー運転手に再訓練するときです。

2. それにもかかわらず、突然、「ペニーペックのためのカナリア、そして食べるためではなく、歌うために」？ 奇跡への信仰は常に人々の間で生きています。 奇跡は、熱心な学生や郊外や近隣の州の無人居住者の形で現れることがあり、周りを見回したり、経験を積んだり、最も重要なことには自信をつけたりして、自分の能力により適した場所に行きます。 叫びは彼らに続きます：「裏切り者！」、しかし彼らは聞きません。 しかし、conf / blogsにある「press」や「m $％ak」という小辞のようなエコーは、ボスの目の前に戻ったり、会社のイメージに固執することさえあります。 上司が補償を上げる必要性に気づいたが、堆積物はまだ残っています。

3. 「達人」は必要ありません、男の子は十分です-タスクは何もありません...ええ。 ファンド・ディレクターにとっては、彼の矢-キャッシュ-フローの高さからのようです。 しばらくすると、FoxPro / Access / Excel /（ツールではなくすべての停止）で山積みになった小さな男の子で、「システム」が頭を痛めます-それを理解するよりも新しいビジネスを始める方が簡単です。 または、それらの同じ「グル」をすでに深刻なお金のために招待します-stable舎をかき集めます。 通常、レッスンはうまくいきません。

4. 私たちは「原材料」を必要とします-私たち自身を訓練し、成長させます-教師がいます。 尊敬する そのような場所が必要です。 しかし、私はvy望していません。実際、これはアイテムの組み合わせです。 2と3に対応する結果。 さらに、初心者との大騒ぎは、どのように失われたとしても、すべての専門家を刺激するものではありません。

5. 従業員が読んで調べたらどうなりますか？！ 従業員の給与を互いに隠す（異なる車を恐れる）愚かな慣行は非常に一般的であり、（二次的なポイントを捨てる場合）「分割統治」の古代の戦術に基づいています。 原則として、「チームスピリットを強化してください」という呼びかけが伴いますが、これは典型的なことです。
