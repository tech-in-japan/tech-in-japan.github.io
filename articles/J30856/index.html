Googleが突破口を開く

Googleは、従業員が時間の20％を「実験」に費やすことができることで長い間知られています。 現在、同社はいわゆる「エンゲージメントポリシー」に沿った膨大な数の新しいプロジェクトを実験しています。 今年の夏、彼らは多くのことを成し遂げ、Intuitとの提携によるGoogle Apps for Your Domainの立ち上げや、生体認証テクノロジー企業であるNever Visionの買収など、素晴らしい仕事をしました。 彼らは遅くならないようです。



Googleが最近ユーザーに紹介した主なニュースを見てみましょう。



誰でも利用できるGoogleガジェット



Googleデスクトップでのみ使用されていたガジェットは、どのWebページにも挿入できるようになりました。 ウェブサイトの作成者は、訪問者が旅行の旅程を計画したり、Picasaウェブアルバムの写真を表示したりできるようになりました。 スポーツイベント、火災の地図、交通事故の結果はどうですか？ 来年、これらのガジェットはどこにでもあるようです。なぜなら、最もプロフェッショナルなウェブマスターでさえ、興味深く多様なコンテンツをサイトに埋め込むことができないからです。 また、近い将来、 Googleガジェット学生賞コンテストが11月15日に終了するため、新しい「魅力的なオファー」があります。



マルチサイト検索



Time Magazineの記事には、Googleの従業員Marrissa Meyerへのインタビューが含まれています。MarrissaMeyerは、新しいマルチサイト検索の開発について語っています。 Rollyo 、 Yahoo！ 検索ビルダーまたはWindows Live Searchマクロ 。 この検索は非常に便利です。楽しみにしています。



SearchMash



今週、Googleは名前を会社のブランドとリンクするのが難しい新しいサイトSearchMash.comを立ち上げました 。 これは、インターネットユーザーが情報を検索するときに使用するすべての機能を1つのサイトに結合する試みに似ています。 テキストと画像の従来のGoogle検索結果は、Ajaxの「ロングスクロール」上に構築された結果のドラッグアンドドロップ順列（Live.comのように便利）を含む新しい方法でサイトに提供され、1ページに留まり、結果の数に制限はなく、URLを表示するための追加オプションを提供するドロップダウンメニューも表示できます。 最も注目すべきことは、ユーザーが既にURLをクリックしたドメイン内の同じパラメーターを使用した新しい検索の可能性です。 簡単に言えば、関心のある問題について議論しているサイトを見つけた場合、そのサイトの他のすべてのページを同様の議論トピックで簡単に見ることができます。 画像検索結果は、SearchMashページの右側に表示されます。



Ajax Search API 1.0



GoogleのAjax Search API 1.0が本日公開されました。 このAPIは、WebマスターがエレガントなAjaxベースのビデオ、ニュース、マップ検索フォームを埋め込み、カスタムAPIベースのアプリケーションを作成するのに役立ちます。 ビデオ検索の例はこのページで見ることができます 、それがどのように機能するかを見るために絵で任意のフレームをクリックしてください。



本日公開されたバージョンは、ニュース検索（検索結果を複数のソースに制限する機能を含む）、地図検索（カナダ、フランス、ドイツ、イタリア、スペインの地図を含む）、およびイントラネットファイアウォールを介したAPIへのアクセスをサポートしています。 Googleは、APIを介した検索結果でのコンテキスト広告の実験も行っています。 Cの広告主はまだクリックに対して課金されていませんが、それは明らかです。一般的に、このアイデアは真剣に練り上げられている段階です。
