最初の公式W3C CSS 2.1テストスイートのテスト結果





最近、公式のW3C CSS 2.1 Test Suteで Internet Explorer 9 Betaの公開テストデータが発表されました。 Microsoftのブラウザーは、すべてのテストの97.7％で良好な結果を示しました。



今日、 Apple Safari 5.0.2 MacIntelブラウザー公開テストデータがユーザーエージェントで利用可能になりました：Mozilla / 5.0（Macintosh; U; Intel Mac OS X 10_6_4; en-us）AppleWebKit / 533.18.1（KHTML、Geckoなど）バージョン/ 5.0 .2 Safari / 533.18.5）。 ブラウザ開発者自身がテストを実施し、データは公式のW3C Webサイトに公開されています 。 テスト結果は、すべてのテストの85％に合格したことを示しています。



他のブラウザのテスト結果はまだ利用できませんが、W3Cワーキンググループ内ですべてのブラウザによるテスト結果の公開に関して合意に達したため、まもなく掲載されます。



それまでの間、テストで判断すると、Microsoftのブラウザは、WebkitやApple Safari 5.0.2よりもCSS 2.1標準を実装しています。



UPD： LinuxバージョンのFirefox 4.0b6の結果が表示されました -93.8％
