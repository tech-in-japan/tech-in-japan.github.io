Nokia Lumia 530を発表

新世代のLumiaスマートフォン（Windows Phone 8.1ベースのスマートフォン）のラインは、新しいモデルを補充し続けています。 先日、エントリーレベルのスマートフォンが登場し、Microsoft MobileポートフォリオのWP 8.1で最も手頃な価格のデバイスになりました-Nokia Lumia530。ロシア市場での世界販売が1200万部を超えるLumia 520のイデオロギー的フォロワーの予想コストは、5000ルーブル未満です。







この投稿では、ユーザーの観点と開発者の観点の両方から、新しいスマートフォンの良い点を見つけようとしました。



前述したように、Nokia Lumia 530はWindows Phone 8.1で動作し、Nokia Cyanアップデートが適用されます。 低価格にもかかわらず、ユーザーはこのOSのすべての機能にアクセスできます。 そして、これはWordFlowキーボード 、「通知センター」、 ユニバーサルアプリケーション 、HTML5をサポートするIE11、および（これまでのところ地域を米国に切り替えた後-結果として） Cortanaのパーソナルアシスタントです。



OSは、SkypeやMicrosoft OfficeなどのMicrosoftサービスを統合します。 ロシア市場では、YandexおよびVKontakteとの緊密な統合も期待されています。たとえば、公式VKクライアントの新しいバージョンでは、すでにPeopleハブに連絡先を追加する機能が提供されています。







「ファイル」という単純な名前を受け取った公式のファイルマネージャが使用可能になりました。 このタイプのプログラムには従来の機能があります。携帯電話のメモリまたはSDカードに保存されているファイルを検索してアクセスし、コピー、移動、名前変更、削除、共有したり、フォルダーを作成したり、ファイルを整理したりできます。



とりわけ、Windows Phone 8.1の登場により、開発者がアプリケーションにファイル管理機能を統合できるAPIが利用可能になりました。 詳細については、「ファイルアクセスとファイルピッカー」セクションを参照してください。







しかし、技術仕様に戻ります。 ノベルティの価格/性能比は良好です。 Lumia 530は、FWVGA解像度（854 x 480ピクセル）および246ピクセル/インチの密度の4インチディスプレイを受け取りました。



Nokiaスマートフォンの従来のケースは、ミニマルなデザインと明るい色をうまく組み合わせています。 ロシア市場では、オレンジ、白、グレーの取り外し可能なパネルを備えたデバイスが利用可能になります。 Lumia 630と同様に、コントロールはタッチスクリーンに表示され、ボリュームのロックと調整を行うのは物理ボタンのみです。 ノベルティの寸法は、129グラムの重量で119.7 x 62.3 x 11.7 mmです。







このデバイスの中心は、1.2 GHzクアッドコアARM Cortex-A7プロセッサとAdreno 302グラフィックアクセラレータで構成されるQualcomm Snapdragon 200シングルチッププラットフォームです。内蔵メモリは8 GBですが、ユーザーは最大128 GBのmicroSDカードで拡張できます（ ！）。 内部スペースに加えて、ドキュメント、写真などを保存するために、すべてのユーザーにOneDriveで15 GBのクラウドが提供されます。



RAMの量は512 MBですが、WP 8.1ではこれは問題ではありません。Microsoftの同僚はRAMの消費量を確認し、OSを最適化しました。 少量のRAMを搭載したデバイスの場合、Windowsストアのほとんどのアプリケーションとゲームが現在利用可能です。



注：開発者として、RAMが1GB未満のデバイスに独自のアプリケーションを適応させるかどうかを検討してください。 Lumia 520、Lumia 530、Lumia 630などのデバイスがベストセラーデバイスであり、世界中で大きな市場シェアを占めていることを知っておくと便利です。 この傾向は今後も明らかに続くでしょう。 独自のアプリケーションを最適化するには、MSDNでこのガイドをご覧ください。



Lumia 530のメイン（および唯一の）カメラには、解像度5メガピクセルの1/4インチマトリックスが装備されています。 サポートされているカメラの絞り値/絞り-f / 2.4; カメラの焦点距離-28 mm; 最小焦点距離-60 cm。



Lumia 630と同様に、スマートフォンは2つのバージョンで提供されます。1つと2つのSIMカードをサポートしています。 どちらのバージョンも、2G / 3Gネットワ​​ークと、Wi-Fi 802.11b / g / nおよびBluetooth 4.0を介したデータ転送をサポートしています。



ちなみに、Lumiaスマートフォンで2枚のSIMカードを使用した作業を実装することは、当社の誇りの別の理由です。 中生代の時代から、2つのSIMカードを備えた多くのデバイスは1つのアンテナを備えており、スタンバイモードでのみ各アンテナを同時に使用できるという原則に基づいていました。 呼び出しを行うと、非アクティブなSIMカードが使用できなくなり、ユーザーは重要な呼び出しを見逃す可能性がありました。 Lumiaスマートフォンも単一のアンテナを使用しますが、通話は自動転送を使用して処理されます。 通話中に通話が非アクティブな番号に到着すると、利用可能なSIMカードに自動的に転送されます。







GPSおよびGLONASS（および、念のため、中国のBeiDou）のサポートと無料のナビゲーションアプリケーションHERE Drive +の組み合わせにより、スマートフォンをスタンドアロンのナビゲーションデバイスとして使用できます（もちろん、必要なマップを最初にダウンロードすることを忘れないでください）。
