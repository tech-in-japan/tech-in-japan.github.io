モジュール



  1. ブートストラップ

    Bootstrap Frameworkに基づいてサイトを作成するための基本テーマ。
  2. パネルはいじめをキャッシュします

    サイト上のすべてのパネルのグローバルキャッシュを一度にアクティブにします 。
  3. ビューキャッシュいじめっ子

    サイト上のすべてのビューのグローバルキャッシュを一度にアクティブにします 。
  4. CKEditor CodeSnippet

    多くのプログラミング言語のスニペットをテキストに挿入します。 Highlight.JSによって強調表示されます。
  5. 展開する

    他のDrupalサイトにコンテンツをリモートで公開するためのモジュール、いわゆるコンテンツステージング。 スクリーンキャスト
  6. サービストークン

    サービスの開発時にトークンで認証します 。
  7. Ckeditorの言及

    Facebookなどのサイトのユーザーへのリンクを作成します。@を使用し、名前に2文字を入力します。
  8. たるみ

    Slackコミュニケーターにメッセージを送信します。
  9. より良い改訂

    テキストフィールドを使用してリビジョンを作成する理由を説明するだけでなく、ドロップダウンリストを使用して変更のタイプを示します。
  10. Search API Stanbol

    テキストからセマンティクスを抽出します。
  11. 構成ページ

    唯一のインスタンスを持つFieldable Entityの形式で構成ページを作成します。 また、このビジネスのコンテンツ全体を開始しないように、「1回限りの」ページを作成するのにも役立ちます。
  12. レンガ

    複雑な構造を持つページを作成するための段落の相続人。
  13. MMS-多言語をシンプルに

    コンテンツを翻訳するための元のモジュール。 テキストのほとんどすべての編集要素に自動的にねじ込まれ、翻訳を即座に監視、更新、削除できます。 Localeカーネルモジュールのみが使用されることは注目に値します。
  14. セッション制限

    同時ユーザーセッションの数を制限します。
  15. 作家
