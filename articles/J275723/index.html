JetBrains C ++ night：いつ、どこで



サンクトペテルブルクの居心地の良いJetBrainsオフィスは、2月24日18時から営業を開始します。 CLionおよびReSharper C ++チームはあなたの訪問を楽しみにしています。



プログラム



私たちは3つのレポートを計画しており、それぞれがスピーカーと私たち全員と会話することを望んでいます。



  1. ReSharper C ++最適化

    要するに、これはReSharper C ++の開発で使用された最適化に関するレポートです：キャッシング、レイジーコンピューティング、インクリメンタルアップデート。
  2. CLionでの単体テストの名手使用

    Google Testを効果的に使用する方法と、CLionが最大の効率を達成するのに役立つ方法に関する実践的なストーリー
  3. 高性能コンピューティング技術

    このレポートの目的は、IDEから注意をそらし、ハードウェアから最大限のパフォーマンスを引き出す方法について説明することです。 計画は、SIMD、SSE / AVX、およびMPIを使用したクラスターでのコンピューティングです。
