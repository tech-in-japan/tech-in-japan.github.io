スモールビジネスおよびホームオフィス向けの新しいProLiant

企業部門のお客様向けに設計された8ソケットDL980サーバーを含む、HP ProLiantラインアップの第7世代マルチソケットモデルの前半。 これらのモデルと並行して、HPは小規模企業向けにHP ProLiantサーバーの範囲を拡大しています。



9月に、HPはHP ProLiant MicroServerと呼ばれるシリーズで最小のサーバーを導入しました。 このシステムは、標準のミニタワーのサイズの半分（26 x 21 cm、奥行き-24 cm）で、デスクトップに完全に収まります。 そして、彼女の騒音レベルは21デシベル以下です。 通常のラップトップのように。



サーバーはAMD Athlon II 1.3 GHzプロセッサーを使用し、最大8 GBのDDR3 RAMと最大4つの3.5インチドライブのインストールをサポートします。 HP ProLiant MicroServerの合計ストレージ容量は、標準の2 TB SATAドライブで最大8 TBです。 標準のHP ProLiant MicroServer構成には、統合RAIDコントローラー、1つのギガビットイーサネットポートと2つのPCI-Expressスロット（x16およびx1）、6つのUSBポート、1つのeSATAポート、200 W電源が含まれます。 オプションとして、DVDまたはDVD-RW CDドライブがサーバーにインストールされます。



サーバーは、Windows Server 2008 R2 Standard Edition、Windows Server 2008 R2 Foundation、およびRed Hat 5で動作することが認定されており、Ubuntu LinuxおよびWindows XP / 7 / Vistaオペレーティングシステムの動作も正常にテストされています。



HP Proliantファミリーの最小サーバーは、主に中小企業向けに設計されています。 最大15ユーザーの企業。 このような企業のコンピューターネットワークでHP ProLiant MicroServerを使用すると、情報の保存と取得の信頼性、外部の脅威に対する保護システム、ネットワーク全体のパフォーマンスが向上し、ネットワークプリンターでのドキュメントの印刷が簡単で手頃な価格になります。



さらに、このサーバーは、インターネットに接続するためのデータストレージおよびゲートウェイとしてホームネットワークで使用することもできます。 あらゆる種類の写真、フィルム、電子文書は常に手元にあり、確実に保護されます。 サーバーは、モニターとキーボードなしで使用できます。これにより、自宅やオフィスで適切な場所を簡単に見つけることができます。



サーバーのメンテナンスを簡素化するために、専門家がサーバーを構成し、その運用上の潜在的な問題を特定し、インターネット接続がある場所からそれらを排除できるリモートコントロールコントローラーをインストールすることも可能です。 同時に、メンテナンスコストが大幅に削減され、機器のダウンタイムが削減されます。



標準として、サーバーには1年間の保証が用意されており、さまざまな応答時間で最大3年まで少しの費用で拡張できます。



いつものように、パートナーと価格を指定します。
