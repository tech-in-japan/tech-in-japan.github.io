光ネットワーク：インターネット開発の新しい段階

光ファイバデータ伝送技術は、ロシアを含む世界中で長い間人気を博しています。 まず、その高速性と信頼性のため。 それにもかかわらず、ロシアは光ファイバーネットワークの分布に関して、東南アジアの先進国に大きく遅れをとっています。 これまでのところ、そのようなネットワークはモスクワや最大の都市で最も頻繁に見られますが、地域ではそれらが出現し始めたばかりです。

このようなプロジェクトの多くは、Huaweiがアクティブネットワーク機器のほぼ完全なラインのメーカーとして参加して実施されています。 ロシアのファーウェイのネットワークソリューション部門のマネージャーであるShavkat Abdurazakov氏によると、同社は3つの分野でビジネスを行っています。キャリア市場、法人顧客、エンドユーザー（モバイルデバイス）向けのソリューションです。

ほとんどの光ファイバ機器は、電気通信サービスの大規模プロバイダー（MTS、Beeline、Tele2、Megafon、Rostelecom、Transtelecom、ER-Telecomなど）を対象としています。 ただし、独自のネットワークインフラストラクチャを実装するために、独自のファイバーチャネルを所有することは決して前提条件ではありません。 たとえば、小規模の顧客は、通信事業者から波長を借りることができます。 実際、事業者はガラスの光ケーブルの別のセクションだけでなく、彼が利用できる数十個の波長のうちの1つを販売することもできます。 さらに、複数のネットワークサービスと同時に波長を販売することも可能です。



卸売および小売



実用的な観点から、Shavkat Abdurazakovが指摘するように、サービスから利益を得る機会は物理的な光ケーブルや波長よりもはるかに高いため、事業者にとってはサービスを販売する方がより有益です。 「高帯域幅の機器を購入すると、ギガビットサービスを大量に販売できますが、ファイバーは1回しか販売できません」と彼は説明します。







サービスは通常、自社のネットワークを構築するのは非現実的で不利な小規模企業に買収されます。 たとえば、小規模な銀行の場合、異なる都市のデータセンター間に10 Gb / sの容量のチャネルを展開する必要があります。 このケーブルの価格に固有の容量がそのニーズを大幅に上回るため、彼はケーブルを購入しません。 同時に、顧客が大量のトラフィック（ファイバーペアごとに最大16 Tbpsを送信できるようになる）を期待している場合、独自のチャネルを敷設または購入することが最適かつ信頼性の高いソリューションになります。



競争ではなく、補完性



無線リレーや銅線ケーブル接続などの他のネットワークテクノロジーには、光ファイバーとの競合はありませんが、いくつかの利点があります。 まず、これは、光ファイバを敷設することが物理的に不可能であるか、管理上非常に困難であるスパンを展開する機能です。 例には、線路、高速道路、または軍事施設の下にケーブルを敷設することが含まれます。 この場合、最大70 kmをカバーできるリレーリンクが役立ちます。 そのようなスパンは、毎秒最大ギガビットの速度で伝送を提供できます。 「したがって、光ファイバー通信と無線中継通信は決して競合することはできませんが、互いに補完し合うことができます」とShavkat Abdurazakov氏は言います。

ラジオリレー通信とは異なり、銅線ケーブルは「ラストマイル」で使用され、最新のテクノロジーにより、最大500 Mbps-1 Gbpsのデータ転送速度を中央ノードからわずかな距離で提供できます-100-200 m。ご存じのとおり、ファイバーに基づいたGPONテクノロジーは、中央ノードから最大10 kmの距離で、ダウンロードモードで最大2.5 Gbit / s、アップロードモードで最大1 Gbit / sの伝送速度を提供するためです。 10G-PON規格をサポートする次世代の機器はすでに4〜10倍の速度を提供できるため、これは限界からはほど遠いものです。 このような機器は、中国、韓国、および東南アジアの他の国ですでに使用されています。 「数年間、「ラストマイル」が高いデータ転送速度を必要とする場合、光ファイバー規格は遅かれ早かれ銅ケーブルに完全に置き換わるでしょう」とアブドゥラザコフ氏は言いました。 彼によると、10G-PONはすでに近隣のベラルーシのオペレーターによってテストされています。

「ラストマイル」の光ファイバー標準の開発における次の段階は、データを40〜100ギガビット/秒の速度で送信できるWDM-PONです。 実際には、これは4Kムービーをほんの数秒でダウンロードできることを意味します。



朝のコミュニケーション-夕方のサービス



オペレーターとプロバイダー自身は、ラストマイルインフラストラクチャの変更に関心がありますか？ ファーウェイのロシアのネットワークソリューション部門のマネージャーであるShavkat Abdurazakovは、顧客の大部分が高い接続速度を必要とする場合、オペレーターは他に方法がないと考えています。 もちろん、このような関心のエンジンは、関連するマルチメディアコンテンツの可用性です。 そのため、最近ロシア市場への参入を発表したNetflixサービスは、4Kの映画を15 Mbpsの速度で放送します。完全な4Kを取得するには、最大100 Mbpsの速度が必要です。 銅ケーブルを使用すると、広い範囲でこのような速度を提供することは不可能です。







ケーブルおよび空気で



しかし、有線ネットワークテクノロジだけでなく、データ転送速度も向上しています。 パラレルは開発中とワイヤレスです。 今後数年のうちに、ネットワーク4.5Gおよび5Gの大規模な分散が予想されます。 5Gの速度は、ユーザーあたり最大1ギガビット/秒であり、4Kの解像度でビデオを表示できます。 ただし、有線ネットワークは常に静的で家やオフィスなどの接続を提供し、モバイルデータ送信は常に移動するユーザーに焦点を当てているため、光ファイバー技術と直接競合することはありません。

さらに、アブドゥラザコフ氏が強調しているように、ワイヤレスインターネットには1つの機能があります。制限がまったくないことはほとんどありません。 サービスプロバイダーは、消費されるトラフィックの速度または量を常に制限します。 これは貪欲からではなく、物理法則を順守する必要性から行われます。1人または複数のユーザーがチャネル全体を可能な限り占有すると、モバイルインターネットの残りの部分が利用できなくなるからです。 そのため、静的な状況では、特にボリュームコンテンツの消費に関しては、モバイルインターネットが有線のインターネットに置き換わることはありません。

また、Abdurazakovは、5Gネットワ​​ークは接続に低遅延を必要とする新しいタイプのサービス向けに設計されていることを思い出しました。 たとえば、「スマートカー」のプロトタイプは、今日多くの企業で積極的に開発およびテストされています。 このような車はセルラーネットワークに接続しますが、現在、120 km / hの速度で、50〜100ミリ秒の接続遅延（3G / 4Gネットワ​​ーク）で、車は数メートルの走行を管理していますが、安全要件をまだ満たしていません。 5G標準の出現により遅延時間を1ミリ秒に短縮すると、「スマート」デバイスと「モノのインターネット」の開発に対する新たなインセンティブがもたらされます。



ユニバーサルアクセシビリティに向けて



ただし、高速光ファイバネットワーク自体のすべての利点には、1つの制限要因があります。それは、展開の比較的高いコストであり、これが州内のデジタル不平等につながります。 十分に多くのユーザーオーディエンスと収益性を備えた人口密度の高い億万長者都市は、遠隔地よりもオペレーターにとって興味深いものであり、さらに人口密度が低くアクセスできない集落です。 たとえば、2012年現在の米国では、トランスポートネットワークは合計約200万kmの光ケーブルであり、ロシアでは-70万km未満です。 3年以上にわたり、ネットワークは大幅に拡張され、現在ロシアでは長さが100万kmを超えていますが、我が国の広大な領土を考えると、これでは十分ではありません。

しかし、Shavkat Abdurazakovによると、ロシアの状況はすでに良くなっています。 そのため、2014年の終わりに、ロステレコムはHuaweiの助けを借りて、以前は無線中継チャネルが使用されていたサレハルドへの光ファイバーネットワークの建設プロジェクトを実施しました。 現在、Salekhardは1組のファイバーを介して接続されており、最大8 Tbpsのデータ転送速度を提供します。 RostelecomとHuaweiのもう1つの興味深い共同プロジェクトは、現在も進行中ですが、サハリン-マガダン-カムチャツカルートに沿った光ケーブルの水中設置です。 今年の終わりまでに作業を完了する予定であり、その後、高速インターネットはこれらのアクセスできない地域に届くでしょう。

Huaweiは、このようなプロジェクトを世界中で実施し、オペレーターと他の企業の両方と協力しています。 たとえば、アラブ首長国連邦では、Huawei社の機器をベースにした中東の大手通信事業者Etisalatが、 この地域で最大のGPONネットワークの 1つを展開しています 。 GPONネットワークの展開により、インドの大規模ホテルチェーンのタージホテルズリゾーツアンドパレス（TAJ）は、 「インテリジェントホテル」の機能を実装するための単一のプラットフォームを構築することができました。広州のすべての都市部を二国間でカバーし、あらゆるプラットフォームにまたがる情報サービスを提供します 。







しかし、光ファイバー接続によって提供される速度がすでに利用可能な地域でさえ、彼らは村で広範囲の高速インターネットサービスがすでに利用可能であることをしばしば知らないユーザーから常に需要があるわけではありません。 これには、機器を設置するだけでなく、その機能を示して話す必要があるオペレーター側の啓発が必要です。 ファーウェイは積極的に若者を教育し、ロシアの主要大学を含む世界の多くの教育機関と協力しています。 このようなトレーニングの重要な側面の1つは、最新のネットワークテクノロジーの可能性と最新のネットワーク機器の能力を発見することです。 「私たちは、大規模な産業センターだけでなく地域で働く機会を持つ若者が彼らの出身地に戻り、ICTサービスを開発することを望んでいます」とアブドゥラザコフ氏は言います。

光ファイバネットワークの普及は、ファイバ自体の生産コストを削減し、パフォーマンスを向上させる継続的なプロセスによって促進されます。 現在世界中で使用されているG.652d標準の損失レベルは0.3 dB / kmであり、光ファイバーの大手メーカーであるCorningは、今日ではさらに低い損失レベル0.16 dB / kmのケーブルを製造しています。 「コストの主な割合は、ファイバ自体のコストではなく、ケーブルを敷設するコストです」とShavkat Abdurazakov氏は言います。 -だから、サレハルドでは永久凍土でケーブルを張る必要があり、カムチャッカとサハリンでは海底に沿って張る必要がありました。 さらに、金属ケーブル、疎水性含浸、およびファイバーを保護するシースは、それよりもはるかに高価です。」

したがって、コンテンツの入手可能性および政府のイニシアチブに加えて製造技術のコスト削減は、ロシアにおける光ファイバーネットワークの普及のもう1つの推進要因であり、わが国と世界有数の大国との間のギャップを縮小するはずです。
