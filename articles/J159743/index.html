Windows Azure Big Compute



11月13日にWindows Azureチームのブログで重要な発表が行われました。クラウドプラットフォームはまもなく、複雑なコンピューティングタスク（Big Compute）のために高RAM消費のシステムを展開できるようになります。



すぐに、Windows Azureのお客様は、インスタンスと仮想マシンの2つの新しい構成を自由に使用できるようになります。60GBのメモリを備えた8つのコアと16のコアと120 GBのメモリです。 これらの各仮想マシンには、次の技術的特性があります。



  • 2.6 GHzのデュアルIntel Sandybridgeプロセッサー
  • DDR3 1600 MHz RAM
  • ストレージおよびインターネットアクセス用の10 GigEネットワーク
  • RDMAを使用したInfiniBand（IB）40 Gbpsネットワーク


Windows Azureプラットフォームの開発者は、8064コアのクラスターを形成して新機能をテストしました。これは151.3 TFlopsで顕著なパフォーマンスを示し、世界の有名なTOP500スーパーコンピューターで 165位になりました 。



また、このブログでは、ロードコンピューティング向けのWindows Azureの実用例の他の例を提供しました。NAMD分子動力学計算システムのスケーリング、MG-ALFA金融リスクモデリングシステム、Towers Watson保険の金融コストモデリング、 FaST-LMMゲノム研究。



Microsoft HPC Pack 2012プラットフォームのリリースにより、12月に開発者が新しい機能を利用できるようになる予定です。



管理ポータルの多数の革新



スコットガスリーは、彼のブログで、Windows Azure管理ポータルの次の革新的なバッチを発表しました。 それらの中には：

  • 管理パネルからMedia Servicesデータを公開、エンコード、および再生する機能が追加されました
  • Windows Azureストレージの独自のドメイン名を設定する機能

  • 6言語でのポータルのローカライズ（残念ながら、ロシア語は含まれていません）
