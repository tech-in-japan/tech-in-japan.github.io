IBMはUNIXサーバー市場で独占企業になるのでしょうか？

「IBMは70億ドルでSunを買収する」という事実のために、次のような言葉がありました。

「サンマイクロシステムズとIBMの間の取引は、独占禁止規制当局によって阻止される可能性があります。これらの企業はUnixサーバーセグメントを支配しており、この市場の65％を占めています。」



ただし、独自のRISCプロセッサアーキテクチャ（IBM Power、Sun Sparc、HP / Intel Itaniumなど）を備えた最新のUnixサーバー市場を、独立した閉鎖的な市場に直接区別することはしません。

RISC / UNIXシステム市場には、IBM、Sun、HPの3つの大企業しかありません。IBMとSunを結合した後、IBM / Power-HP / Itanium（おそらく3番目はFujitsu / Sparcになりますが、個人的には、富士通はそのSparc部門を閉鎖せざるを得ないと考えています。

ただし、歴史上、RISCアーキテクチャとともに、次のような伝説的な企業に注目してください。

1. Digital Equipment Corporation（DEC） -CPU Alpha AXP 、

2.シリコングラフィックス（SGI） -CPU MIPS 、

3. Intergraphコンピューター-CPU クリッパー 、

4. Siemens Nixdorf Informationssysteme AG（SNI）-CPU MIPS 、

5. Pyramid Technology- CPU MIPS 、

6.タンデムコンピューター-CPU MIPS 、

7. Data General（DG） -Motorola 88k CPU 、およびその他多数。

そして、あなたは彼らが誰に市場シェアを失った（与えた）のだろうと思わなかったのですか？



そして、状況を注意深く分析すると、標準（xp）Intel x86プロセッサーを備えた標準IBM PCアーキテクチャーのサーバーとワークステーションが市場シェアを占めていたことがわかりました。

そして、安価なIntel x86 / Linuxサーバーが、あらゆる種類の企業（ほとんどが中国人）の巨大なクラウドによってリリースされています。

1. Hewlett-Packard（HP） [飲み込んだコンパックとDECとタンデムとコンベックスとアポロ]、

2. Dell Inc. 、

3. IBM [サンマイクロシステムズを吸収、シーケントコンピューターを吸収]

4. 富士通 [飲み込まれたSNIとピラミッド]、

5. Acer [飲み込まれたゲートウェイとALR]

6. Lenovo Group [IBM PC Co.に飲み込まれた]、

7. ラックマウント型システム [SGI＆Intergraph Computerに吸収]、およびその他。

したがって、IBM Power / UNIXの競合他社は、膨大な数のHP Itanium / UNIXを取り囲んでいます。

そして、主なものはHPではありませんが、HPとCompaq / DECの合併でかつて多くの利益を得ていたユビキタスDellであり、DellはSunの顧客を安価なx86 / Linuxサーバー！
