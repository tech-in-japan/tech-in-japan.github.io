Operaフロントコントロールはブラウザコントロールを提供します



ノルウェー、オスロ-2009年4月1日



Opera Softwareは本日、Opera 10に含まれるもう1つのイノベーション-Face Management（Face Gestures）を発表します。 このテクノロジーにより、ウェブカメラの所有者は誰でも自分の顔を使ってブラウザを操作できます。 この開発は、マウスジェスチャと同じアーキテクチャに基づいており、笑顔、点滅、または鼻を動かすのと同じくらい簡単にWebを閲覧できます。







顔の管理は、JavaScriptとHTMLとCSSのみに基づいて最も効果的な画像認識技術を作成するためのOperaとノルウェーシークレットサービス-NIS（ノルウェーインテリジェンスサービス）とのコラボレーションの結果でした。 7年以上の実験の後、顔の管理システムは公開テストの準備ができています。



2001年に、Opera 5.1でマウスコントロールの世界を導入しました。 しかし、私たちはどこに向かっているのかをすでに知っていました」とOperaのエグゼクティブディレクター、ジョンフォンテクナーは言いました。 「その頃には、ブラウザの初期バージョンの1つに組み込まれたキーボード管理システムを既に開発していたため、Opera 8にVoice Controlが登場してからは、ユーザーの顔という未使用のOperaブラウザ管理ツールが1つしかありませんでした。 Operaのフロントオフィスは、ブラウザ制御の新しい標準を設定します。」



Operaのフロントオフィス管理には、新しいタブを開いたり、セッションを復元したり、Webページをナビゲートしたりするまで、45を超える機能が含まれています。 また、フロントエンド管理は、Operaブラウザの組み込みメールクライアントに統合されています。 さらに：フェイスコントロールは、マウスコントロールを部分的に置き換えることもできます。 新しいテクノロジーを有効にするには、F8キーを押してください（Opera 10でのみサポートされています）。

Opera Software ASAについて



Opera Software ASAは、PC、携帯電話、その他のインターネットデバイスでWebページを表示するための新しい可能性を切り開きました。 Operaが開発したクロスプラットフォームブラウザーテクノロジーは、その機能、標準への準拠、小さなサイズで知られ、ユーザーにより高速で安全で動的なインターネットアクセスを提供します。 Opera Softwareのメインオフィスはノルウェーのオスロにあり、世界中にオフィスがあります。 同社は、OPERA証券コードでオスロ証券取引所に上場しています。 詳細については、 www.opera.comをご覧ください。
