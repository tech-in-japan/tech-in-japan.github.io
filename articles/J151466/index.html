GoogleがJ2ObjCをリリース、iOSアプリケーション向けのオープンソースJavaからObjective-Cへのコードコンバータ

Googleは、オープンソースプロジェクトJ2Objcのリリースを導入しました。これは、JavaソースコードからiPhone / iPadアプリケーション用のObjective-Cへの独自のコンバーターです。







J2ObjCを使用すると、追加のコード編集なしで、iOSアプリケーションのビルドの一部としてJavaコードを使用できます。 このプロジェクトの目標は、プログラマーがJavaでUI（データアクセスまたは内部アプリケーションロジック）に関係のない部分を記述し、Android、Webアプリケーション、iOSで使用できるコードを取得できるようにすることです。



J2ObjCはJavaエミュレーターではありません。 代わりに、JavaクラスをiOS Foundation Frameworkを直接使用するObjective-Cクラスに変換します。 Java 6は完全にサポートされており、エラー処理、汎用および匿名クラス、ストリーム、リフレクションなど、ランタイムオプションのほとんどがクライアントサーバー開発者に必要です。 JUnitテストの翻訳と実行がサポートされています。 J2ObjCは、XcodeやMakeなど、ほとんどのアプリケーションビルドツールで使用できます。



詳細については、次のリンクへようこそ： http : //google-opensource.blogspot.com
