新しいHakeslet：ロシア語のオンラインコース（Java、SICP、Objective-Cなど）

こんにちは、Habr！







昨年の秋、Hakesletを開設しました 。Hakesletは、ロシア語の無料オンラインプログラミングコースを備えた教育プロジェクトです。 1週間後、最初のコースが始まりました-OS Xの開発。これは非常に興味深い経験でしたが、このプロジェクトの開発では多くのミスが犯されました。



本日、Hakesletの再起動をお知らせします！ あなたを待っているもの：



  • 新しいサイトと便利で高速なエンジン。 古いものには非常に多くの問題があったため、多くの学生は単純にそれを正常に使用できませんでした。
  • 改善されたコース形式。 最初のバージョンでは、物理的な大学をインターネットでエミュレートしようとしましたが、最初はオンラインコースを行う必要がありました。
  • 教師との個人的なコミュニケーション。 この機能を他の教育プロジェクトとの重要な違いにしたいと思います。
  • 新しい興味深いコースと今後のコースの計画。




すでに最初の3つのコースに登録できます：



  1. Javaアプリケーション開発
  2. コンピュータープログラムの構造と解釈（SICP）
  3. OS X開発




非常に興味深いSICPコースの予告編：

コース「Javaアプリケーション開発」







Java初心者コースおよびOracle Java SE 7認定の準備。 内容はこれについてです。 最初の講義はすでに利用可能で、2番目の講義は6月20日にリリースされます。 合計で約7つあります。 iTunesにはポッドキャストがあります。



最初の講義は、コースの説明とJavaの紹介です。







コース「コンピュータープログラムの構造と解釈」







同名の本の有名なSICPコースのロシア語版。 非常に重要で非常に興味深いコース。 経験豊富なプログラマーと絶対的な初心者の両方に適しています。 それは、現代のプログラミングの根底にあるこれらの基本原則について語っています。 コースは5つのパートに分かれており、各オーダーで5つの講義が行われます。 最初の講義は7月1日にリリースされます。



コース「OS X向けの開発」
