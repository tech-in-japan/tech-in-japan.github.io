Xpom-Xpum！ SDK-Google Chrome拡張機能とアプリケーション用のIDE

これは何ですか



Xpom Xpum！ SDKは、C＃の無料のオープンソースIDE（ライセンス-MIT）です。 目的-Chromiumベースのブラウザ向けに、さまざまな拡張機能やアプリケーションを作成します。



使い方は？



Xpom-Xpumの最新リリースをダウンロードしました！ SDK （この記事の執筆時点では-アルファ）および.NET Framework 4がインストールされていることを確認しました。

あなたの前にはIDEウィンドウがあります。



まず、他の多くのIDEと同様に、プロジェクトを作成する必要があります。



  1. [ ファイル]> [新規]をクリックします 。
  2. プロジェクトのタイプ-「Chrome拡張機能」を選択します。 プロジェクト名を入力します。 プロジェクト名のフォルダーが作成されるため、名前に次の文字を含めることはできません： \ /？ "<> 。

    プロジェクトフォルダを覚えておいてください。 これは、拡張機能のファイルが格納されるフォルダーです。
  3. プロジェクトブラウザー（左）にManifest.jsonファイルが表示されます。 それをダブルクリックします-そして、次の内容のテキストが表示されます： 



    { "manifest_version": 2, "name": "1", "version": "1.0.0.0" }


    このファイルは、Chromeに試用版の拡張機能をインストールするのに十分です（ただし、プロジェクトメニューから、アイコン、バックグラウンドスクリプト、コンテンツスクリプトなど、さらにファイルやフォルダーを追加できます）。
  4. ツールバーの「実行」ボタンをクリックします。



    Xpom-Xpum！ 拡張機能がインストールされたGoogle Chromeのインスタンスを起動します。 chrome：//拡張機能にアクセスすると、拡張機能を確認できます。



    ブラウザがすでに実行されている場合、拡張機能は現在のインスタンスにインストールされます。
  5. Xpom-Xpumの外部で拡張機能を実行可能にするには、CRXにパックします（ただし、新しいChromiumとそれに基づくすべてのブラウザーでは、これらの拡張機能をインストールすることは禁止されています！）、またはプロジェクトフォルダーを指定して開発者モードでブラウザーにダウンロードするだけです。


Google Chromeがない場合はどうなりますか？



Xpom-Xpum！ コマンドライン拡張機能のインストールを正しくサポートしている場合は、Chromiumベースのブラウザで使用できます。



推奨用途：Comodo Dragon、SRWare Iron、Uran、Yandex.Browser。



IDEでブラウザを設定するには：

  1. [起動]-> [起動設定]に移動します。
  2. [リストにブラウザを追加]をクリックします。
  3. ブラウザのEXEへのパスを入力します（たとえば、C：\ Program Files \ Browser \ chrome.exe）。 OKをクリックします。
  4. 重要：[実行設定]ウィンドウのブラウザのリストでこのブラウザを選択します。
  5. [実行設定]ウィンドウで[OK]をクリックします。


IDEセットアップ



Xpom-Xpumで！ SDKでは、プロジェクトの作成ウィンドウに独自のプロジェクトタイプを作成し、構文の強調表示を設定できます。 これについて- ドキュメント内 。



さらに、Xpom-Xpumのソースコード！ C＃で記述されたSDKを使用すると、2010以降のVisual Studio（Expressを含む）で簡単に開き、制限なしに変更およびコンパイルできます。
