ソーシャルグラフ戦争

グーグルとフェイスブックの間の戦争は活気を取り戻しつつあります。 GoogleはContacts API APIの利用規約に変更を加えたため、これを使用するすべての人は相互性を保証する必要があります。



事はこれです。 Google Contacts APIを使用して、Facebookで新しい友達を検索および追加できます。 そして、ソーシャルネットワークが同様の情報（つまり、ソーシャルグラフ-すべての友人間のリンク）を提供しない場合、Google情報の使用を拒否することが義務付けられます。

5.8。 Googleはデータの移植性をサポートしています。 サービスまたはアプリケーションで使用するために連絡先データAPIまたはポータブル連絡先APIを介してコンテンツにアクセスすることにより、ユーザーが連絡先データを選択した他のサービスまたはアプリケーションに基本的にできるだけ早く簡単にエクスポートできるようにすることに同意します適用される法律に従い、Googleコンタクトからそのようなデータをエクスポートします。


ソーシャルグラフは、Facebookで最も価値のあるものです。 最近、情報をエクスポートする機能を立ち上げても、各ユーザーがサイトからダウンロードして、写真、ビデオ、メッセージ、ステータス、イベントなど、友人のリストを除くすべてのデータを保存できるようになりました。



同時に、ユーザーはすべての連絡先を自由にエクスポートして保存できます。 Googleの特別な説明によると、利用規約の新しい変更により、既存の不正が排除され、「情報の真の自由」が確保されるはずです。



「多くのサイトでは、連絡先などの情報を簡単かつ迅速にインポートおよびエクスポートできます。 しかし、Facebookのようにこれを許可しないサイトは、ユーザーを絶望的な状況に陥れます。 Facebookなどのサイトに連絡先をインポートするとすぐに、効果的に設定されたトラップに陥ることに人々は気づかないことがよくあります」とGoogleはコメントで述べています。
