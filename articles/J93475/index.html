仕事を選ぶ。 またはカウボーイ会社に入らない方法

最近、インタビュー方法に関するHabréの記事が頻繁にあります

応募者の選択メカニズム。 「 私はあなたの売り手です-あなたは私の買い手です 」という立場を感じないように、企業の選択について話す時が来ました 。（c）Dolphin

以下はすべて個人的な経験に基づいた私見であり、究極の真実であると主張するものではありません。



だから、もしあなたがばかでないなら、面白い名門の仕事を見つけて、カウボーイ会社に入らないで、次の会社を避けてください：

  • 名前に顕著な擬態（冷静さ）を持つ会社。 たとえば、「Innovative Technologies」、「Super Web Company」、「Modern WEB Technologies」など。 練習では、原則として、そのような企業では、誇大妄想に加えて、それ以上はないことが示されています。 ちなみに、リストされている最後の企業は本物です（ops、カンターに焼き付けられています）。 それは、2人のプログラマー、1人のデザイナー、流動的な人員を雇う2人の秘書、およびメガチップを自分のサイトに結びつけるという提案でさまざまな会社に電話した5-6人の叔母で構成されていました。 強気の校長はこの行列全体を率いた。
  • 「名声のない場所」にある中小企業やガレージ内のほとんどの場所を借りている会社は、もちろん、これが初期のロシアのアップルではない場合。 成功した企業は、通常は地下鉄から徒歩圏内にある良い部屋を借りる余裕があります。
  • あなたが原則にと​​らわれている会社：「これを知っていますか？ これはどうですか？ 素晴らしい-あなたは私たちに合っています。」
  • モルディブのアラガールとテクニカルディレクターのショルダーストラップを数年のうちに、あなたが最初から不審に約束しすぎている会社。
  • 貧弱なサイトを持つ会社（レイアウトの妥当性を確認し、サイトのデザイン自体を評価できます）。 雇用主（インタビュアー）が求職者の服を探している場合、彼の顔を見てみましょう。 会社のウェブサイトがまったく見つからない場合は、忘れてください。
  • たわごとプロジェクトに取り組んでいる、または仕事を始めている企業（および）。 これがおそらく最も重要なポイントです。 たわごとプロジェクトは遅かれ早かればらばらになります。つまり、あなたのサービスはもはや必要ないでしょう。 頻繁な転職は間違いなくあなたの経験をもたらすでしょうが、その後、新しい仕事を得ても、あなたは「大根」ではないことを誰にも証明しません。 あなたの特定の責任に興味を持って、会社が何をするかについて質問してください。
