IconBIT LZシリーズポータブルバッテリー

スマートフォンの繊細さ、そのパワー、ディスプレイ解像度に革命が見られます。 しかし、モバイル機器のバッテリーにはまだ変更はありません。 もちろん、メーカーはさまざまな技術を使用して、バッテリーのサイズを小さくし、バッテリーの寿命を延ばそうとしています。 しかし、残念ながら、実際にはこれはあまり明白ではありません。 おそらく、この傾向は続くでしょう。 製造業者は、ケースの厚さと性能の間で選択する必要があり、より容量の大きいバッテリーを取り付けるために何かを犠牲にします。







今日、いくつかの旗艦はバッテリー寿命の記録を樹立しています。 たとえば、サムスンギャラクシーノート3、LG G2（写真に表示）またはHTC M8のように遠くに行く必要はありません。 しかし、バッテリー容量ではなく、他のパラメーターに従ってデバイスを選択する人はどうでしょうか？ 1つの選択肢は、コンセントでの生活です。 ただし、そのためには、常に充電器を携帯する必要があります。また、スマートフォンをコンセントに接続する機会があるたびに携帯する必要があります。 別のオプションは、外部バッテリーを購入することです。これにより、デバイスの動作時間が長くなります。 そのようなアクセサリは、間違ったタイミングでデバイスが突然放電してオフになった場合に役立ち、外の世界との通信ができなくなります。



しかし、選択する外部バッテリーのモデルはどれですか？ この質問に対する答えは、特定のタスクと個人的な好みに依存します。 主な選択基準の1つは容量です。 容量が小さく小型で軽量のバッテリーを選択できます。これにより、突然放電したスマートフォンをすばやく充電し、いくつかの重要な電話をかけることができます。 タブレットのバッテリー寿命を延ばすタスクがある場合（たとえば、出張中、プレゼンテーション中、休暇中など）、容量の大きいモデルを選択することをお勧めします。 そのため、iconBITはFTB10400LZ 、 FTB7800LZ 、 FTB5200LZ 、 FTB2600LZの一連のポータブルバッテリーをリリースしました。 これについては、以下で説明します。



特徴

  • タイプ： Li-Po
  • 入力： Micro-USB 5V / 1000mA
  • 出力： USB：5V / 1000mA
  • 寸法： 107 x 24 x 22 mm
  • 重さ： 70g


FTB5200LZ

  • 容量： 5200 mAh / 19.2Wh
  • タイプ： Li-Po
  • 入力： Micro-USB 5V / 1000mA
  • 出力： USB：5V / 1000mA
  • 寸法： 107 x 42.5 x 22 mm
  • 重さ： 125g


FTB7800LZ

  • 容量： 7800 mAh / 28.86Wh
  • タイプ： Li-Po
  • 入力： Micro-USB 5V / 1000mA
  • 出力： USB：USB1 5V / 1A、USB2 5V / 2.1A、USB1 + 2 5V / 2.1A（最大）
  • 寸法： 107 x 61 x 22 mm
  • 重さ： 185g


FTB10400LZ

  • 容量： 10400 mAh / 38.48Wh
  • タイプ： Li-Po
  • 入力： Micro-USB 5V / 1000mA
  • 出力： USB：USB1 5V / 1A、USB2 5V / 2.1A、USB1 + 2 5V / 2.1A（最大）
  • 寸法： 105 x 71 x 22 mm
  • 重さ： 230g


包装および機器

電池は透明なプラスチック製の包装紙で納品され、外国語の碑文と以下の主な特徴のアイコンが付いています。







側面は各バッテリーの容量を示します。 既にモデル名（FTB 7800 LZなど）ですが、その中にmAhがどれだけあるかを推測できます。







仕様と注意事項は、ボックスの背面にあります。 このデバイスが製品として認定されていると言及することで、このバッテリーで充電されているデバイスに何も起こらないという確信が得られます。







箱を開梱すると、内部には携帯用バッテリー自体、充電用ケーブル、保証書付きの説明書のみが入っています。







私は、機器が非常に広範囲であるとは言いません。 私の意見では、たとえばAppleデバイス用の追加アダプターはありません。 現在、ほとんどのデバイスはMicroUSBコネクタを使用しています。



デバイス設計

携帯用バッテリーを考えると、すべての人のケースは楕円形の細長い形をしていると言えます。 これにより、ポケットやバッグのコンパートメントに入れると非常に便利です。 ご覧のとおり、デバイスの長さは同じですが、容量に応じて幅が異なります。







すべてのバッテリーには、LED懐中電灯と充電インジケータが装備されています。







各モデルは詳細には意味がありません。 5200mAhの容量のバッテリーに焦点を当てます。 目の前にあるのは、角が丸いコンパクトな長方形のブロックです。 前面にはバッテリーインジケーターがあります。 インジケータの左側には、インジケータをオンにするためのボタンがあります。 4つのダイオードすべてがオンの場合、エネルギーはそれぞれ75〜100％、3つのダイオード-50〜75％、2つのダイオード-25〜50％、1つのダイオード、5〜25％です。 すべてのダイオードがオフのとき、静電容量は5％未満です。







携帯用バッテリーの側面には、このデバイスが中国製であることが正直に示されています。 IconBITは明らかにマーケティング担当者を進歩させたので、「made in PRC」と言い、「made in China」（中華人民共和国）と訳されています。 これは、すべて中国起源のデバイスを示します。 さて、これに加えて、特性と認定バッジがあります。







右側には、通常のUSBコネクタがあります。 動作パラメータは入力時と同じです：電圧5V、電流1A。







反対側の端には懐中電灯があります。







iconBIT FTB7800LZ 7800 mAhおよびiconBIT FTB10400LZ 10 400 mAhは、最も大食いのデバイスを接続したい、または長い旅行中に、1回の外部バッテリー充電でスマートフォンを数回充電したい人向けのバッテリーです。 どちらも少し太くて重いです。片方は185グラム、もう片方は230グラムです。 両方とも、2つのデバイスを同時に充電することができ、最大出力は2.1Aです。



内部

どんな種類のバッテリーが入っているかを見るために、2600 mAhで最も若いモデルを分解することにしました。 トレースなしではなく、かなり難しいことを理解しています。 体の半分をナイフで互いに切り離します。







身体から内側の詰め物を取り除いた後、私たちの視線はそのようなデザインで提示されます。 これは、バッテリーがはんだ付けによって固定されている制御ボードに基づいています。







使用される電解質のマーキングは、有名なLG社が2600 mAhで製造したLGABB41865です。 ですから、能力についてはだまされませんでした。







灰色のフィルムを取り外すと、その下にはただのバッテリーがあります。







USB、microUSBコネクタ、電源ボタンは非常に確実にコントロールボードにはんだ付けされており、ポータブルバッテリーの使用中に脱落する可能性があるとは言えません。



テスト中

バッテリーをテストするために、バッテリー容量が3000 mAhの同じLG G2スマートフォンを使用しました。 便宜上、表のすべての結果を確認することを提案します。

モデル 宣言された容量 有効容量 容量との不一致 満充電の数
FTB2600LZ 2600 mAh 1800 mAh 30％ 満杯ではなく、最大63％
FTB5200LZ 5200 mAh 3810 mAh 26％ 1回のフル充電、2回目は最大27％
FTB7800LZ 7800 mAh 5850 mAh 25％ ほぼ2回のフル充電
FTB10400LZ 10400 mAh 8340 mAh 19％ 2回のフル充電、3回目は最大78％
結果は非常に期待されています。 結局のところ、エネルギーはあるデバイスから別のデバイスに注がれるだけではないことを理解しているため、公称容量の約1/5を考慮に入れない方が良いと考えています。 また、すべてのバッテリーの電圧は3.7Vで、出力は5Vです。 第一に、電圧の増加はエネルギーを消費します。 第二に、外部バッテリーとスマートフォンの変換スキームは、100％の効率で機能しません。 変換はエネルギーを失っています。 そして第三に、充電時に、バッテリーは受け取ったエネルギーを100％節約できません。 その一部は熱の形で放出されます（既知の事実は、充電中に電話が熱くなることです）。

おわりに

4つのiconBITバッテリーはすべて適度にコンパクトで軽量であり、特に2600および5200 mAhの若いモデルでは、スマートフォンやタブレットの所有者がいつでもガジェットが完全に放電されていない場合に役立ちます。 高品質のアセンブリ、スタイリッシュなデザイン、低価格-これらすべてにより、会社のデバイスはご自身またはギフトとして購入できます。 私が気づいた唯一の欠点は、他の物と一緒にバッグに入れて電池を運ぶと、白いプラスチックが簡単に汚れて傷つくことです。 しかし、これはきちんとした人にとっては問題ではなく、他の人がきれいな白いアクセサリーを時々手配することは難しくありません。
