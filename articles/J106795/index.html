DeaDBeeF 0.4.2リリース



foob​​ar2000とdeadbeefのファン、喜んでください！ この素晴らしいプレーヤーの新しいバージョンがリリースされました。 外側では、目立たないプレーヤー（私はそのような簡潔さを好みますが）は、Linuxの下でプレーヤーのユニットをサポートするCUEマークアップを含む膨大な数のフォーマットを再現します。

高速で簡単、手間がかからず、国内の開発者からの快適なプレーヤーです。







変更のリストは膨大です（オーディオプレーヤーの他の開発者の言い訳）：

  • 多くの言語への翻訳を追加しました
  • プラグインを介してメニューを拡張する
  • システムユーティリティを実行してヘッダーをフォーマットする機能
  • MIDIサポート
  • 多くのロスレスデコーダーを更新しました
  • 破損したCUEファイルのサポート
  • 新しいアイコン
  • グローバルホットキーサポートの改善
  • プレイリストリストでゼロから新しいダブルクリックプレイリストを作成する
  • Alt + [1-9]でプレイリストをすばやく切り替える
  • さまざまなヘッダーをフォーマットするための多くのツール
  • より速いコールドスタート
  • 修正されたバグと読み込みの高速化
  • 閉じるときに現在の位置とプレイリストを保存する機能
  • プレイリストをスクロールする
  • そして、他の多くの変更...




新しい設定の一部：









DeaDBeeFの公式Webサイトからダウンロードできます。

Ubuntuユーザーの場合：

sudo add-apt-repository ppa:alexey-smirnov/deadbeef

sudo apt-get update

sudo apt-get install deadbeef
