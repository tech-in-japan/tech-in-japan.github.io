マイクロソフトはAJAXの理解を導入しました

Microsoftは 、AJAXの実装を正式に発表しました。 以前はAtlasとして知られていましたが、よりわかりやすい名前-ASP.NET AJAX 1.0を取得しました。



ASP.NET AJAXはASP.NET 2.0およびVisual Studio 2005で使用できます。この製品のMicrosoft標準ライセンスには、10年間のサポートと24時間の電話サポート（24時間365日の電話サポート）が含まれます。



次のコンポーネントが強調表示されます。

-Microsoft AJAX Library-任意のブラウザで動作し、任意のバックエンドと対話できるクライアントjavascript-library。 Microsoft Permissive Licenseの下でリリースされます。これは、開発者がライブラリを自由に変更する許可、および変更されたコンポーネントをあらゆる目的で無料で配布することを意味します。

-ASP.NET 2.0 AJAX Extensions-ASP.NETとの統合を担当するサーバーパーツ。

-ASP.NET AJAX Control Toolkit-ASP.NET AJAX Extensionsの機能をより完全に使用できる無料のコントロールとオープンソースコンポーネントのセット。



ASP.NET AJAX 1.0の「機能」は、コード名がOrcasになるASP.NETの次のリリースに含まれます。
