STAXの歴史：エレクトロスタットと市場の状況

私はめったにどのブランドにも称賛の叙情詩を書きたくありません;原則として、私は会社を設立したか、高度な技術を導入した個人に興味があります。 それは私が人々に表敬語をskiしないことです。 ただし、この資料は規則の例外になります。 歴史的サイクルの次の資料では、静電ヘッドフォンというフレーズと同義語になった会社についてお話します-これがSTAXです。 会社の創設者に関する情報はほとんどありませんが、エンジニアに関する情報はまったくありません。







一方、スタックス製品は本当に私を夢中にさせました。 このブランドのデバイスに実装されている技術的なソリューションは、市場の法則のおかげではなく、むしろそれらに反して人気があります。 この会社は、前世紀の1980年代半ばに、あらゆる種類の大量オーディオの妥協とマーケティング担当者が勝ち返ったことを知らない、または知りたくないようです。 ほとんどどこでも、STAXではありません。 だから、カットの下で、静電音響再生の本当のsaの物語-会社STAX。



未知の年と突然のスタート



STAXは1938年に設立されました。 この会社は、林直武によって組織されました。林直武は、小さな家族経営から世界的に有名な会社を作り上げました。 このとき、日本は1937年に始まった中国との激しい戦争を繰り広げ、第二次世界大戦にスムーズに流れ込み、1945年に終了しました。



この期間中に同社がリリースしたデバイスについては、ほとんど何も知られていない。 その時点で、防衛事業を回避することができたのは、日本企業1社だけではなかったと思います。 当時の日本の産業は、「すべてが最前線、天皇Hiro仁の勝利のためのすべて」の原則に従っていました。 STAXの戦後の開発のプロファイルを考えると、たとえば以下に示すような日本の軍用機用のヘッドセットの作成に同社が従事していたと思います。







非軍事専門家向けの電子機器では、会社名は戦後すでに関連付けられています。 公式には、最初のSTAXデバイスは次のとおりです。コンデンサーマイク、ユニークな静電ピックアップヘッドCP-20、同様にユニークな湾曲したトーンアームLA-24。 これらのデバイスはすべて、設立から12年後の1950年にほぼ同時に登場します。









製品は一度に複数のバージョンで発売され、プレミアムセグメントに変更が加えられ、デバイスが少しシンプルになりました。 たとえば、CP-20AD / Nカートリッジにはダイヤモンド針が、CP-20ANにはサファイアが使用できます。









言い換えれば、STAXはスリーブのカードのように、当時の世界的に有名なメーカーの最高の例と競合できる人気のあるユニークなプレミアムデバイスを数行入手しました。 同社は電気音響学、材料科学、電気物理学の分野で12年間にわたって基礎研究を行っており、数十年にわたり、市場のリーダーではないにしても、単にプレミアムオーディオのニッチを占めるようになりつつあるように発展しているようです。



すでにその時点で、同社は開発の方向、つまり電気音響学における静電デバイスを選択しました。 リリースされたすべてのCP-20ピックアップヘッドは静電式でした。









彼らは、20から18000 Hzの非常に広い範囲の再現可能な周波数を持っていました。 CPラインの論理的継続は、ヘッドフォンに直接接続されていない最も有名なSTAX製品の1つであるCP-X静電カートリッジでした。







STAXという名前は、会社のプロフィールを明確に示唆しています。 ところで、会社がこの名前を受け取ったのは正確にはわかりません。 さらに、名前スタックス（Yugen-gaisha Sutakkusu）は、もともと静電気に関連するものとして考えられていたことが明らかです。



よく忘れられた新しい



将来の狭いプロファイルにさらに近づくと、STAXは1954年に別の開発に押されました。 同社は静電トゥイーターをリリースしました。 彼らは、前世紀の20年代から音響システムに音響放射の静電原理を使用しようとしたことに注意すべきです。



特に映画館の音響では静電ドライバーが使用されていましたが、当時のこのタイプのエミッターは適切な高分子材料が不足していたため信頼性が低く、30年代半ばまでに静電スピーカーはほぼ完全に静電技術に取って代わりました。 1954年のSTAXの開発により、STAXはこの技術を復活させ、その開発を前もって決定したと言えます。









1960年、STAXの画期的なイベントが開催され、同社は世界初の静電ヘッドフォンSR-1をリリースしました。 今日まで、静電原理はヘッドフォンで最大の忠実度の再生を確保するための基準と考えられています。 林小之隆は、一般的に使用されている「ヘッドフォン」を拒否し、このデバイスを「イヤースピーカー」と呼んでいる文字通り「イヤースピーカー」と呼んでいます。









このようなヘッドフォンには独自のアンプが必要で、1960年に同社はたった2つのモデルSRD-1とSRD-2をリリースしました。 両方ともバイアス電流制御付き。 アンプは、比較的強力な出力トランスを備えた真空管アンプです。 それらの出力信号は、静電放出の実装に必要な高い電位差によって特徴付けられます。 エンジニアが150 Vのエレクトロスタットに対して比較的小さな電圧で開始するのは興味深いことです。次の変更では、放射面の振幅を増加させるために電圧を230 Vに増加します。







ヘッドフォンメンブレンには、厚さ約6ミクロンのポリマーフィルムを使用しました。 当時、それは世界で最も薄い音響放射器の膜でした。 主にこの機能により、世界初の静電ヘッドフォンは次の特性を備えています。



  • 重さ：320 g。
  • 開口部の厚さ：約5ミクロン。
  • 感度：95 dB / 100 V rms
  • インピーダンス：130 kOhm / 10 kHz
  • 周波数帯域：20-20,000 Hz。
  • ベース電圧：150V / DC。
  • 小売価格：6,000ドル/ 50ドル、今日は432ドル。


最初の生産用エレクトロスタットの論理的な継続は、SR-3の改良版でした。









同社は、静電ヘッドフォンの設計と製造を主な活動にしています。 並行して、STAXのエンジニアは、フルサイズのフロアスタンド型スピーカー用のエミッターの作成に努めています。 そのため、1964年に一連の広帯域静電型スピーカーESSが登場し、ラインにはシステムESS-3A、ESS-6A、ESS-12が含まれます。







デバイスには、当時として印象的な特徴があります。



  • 周波数範囲：40 Hz〜20 kHz。
  • 音圧出力レベル：84 dB / W / m。
  • 最小推奨電力：15ワットから。
  • 抵抗：8オームから（400オーム-OTLのターミナル）。
  • 電源：100 V、50 Hz / 60 Hz。


信頼できる指標にもかかわらず、スピーカーはベストセラーにならず、日本の故郷でさえ異国的であると考えられています。 STAXのスピーカーの大多数は同じ運命に苦しんでいます。 今まで、珍しい音楽愛好家は静電型スピーカーにお金を払う用意ができていましたが、60年代には同様の状況がありました。 ESSシリーズは、60年代も別のESS-4Aデバイスを使用し続けた後、商業的な観点からは見込みのない方向として忘却に沈みました。









黄金期



60年代後半から、STAXは集中的な開発と成長の期間を開始します。 彼らのヘッドフォンは日本以外で人気が出てきており、静電技術の分野での生産の急増と研究の強化につながっています。 そのため、1968年から1970年にかけて、約40のシリアル製品が登場しました。そのうち、70年代の平均的なオーディオマニアが必要とする製品のほぼ全範囲が含まれていました。 特に、ヘッドフォン、個別の静電HFセクション、ピックアップヘッド、トーンアーム（特に、UA-9などのカーボン部品を使用）、アンプ（クラスAを含む）、およびダブルモノの原理）。







UA-9



1979年は特に新製品が豊富でした。 その後、同社は、静電ヘッドフォンを標準ステレオシステムSRD-Xに接続するための特別なアダプターである、伝説的なCP-Y / TYPE 2ピックアップであるCA-X PROデュアルモノブロックアンプをリリースしました。









また、1979年には、伝説的なSR-Lambdaパノラマヘッドフォンが登場しました。 ヘッドフォンのメンブレンは厚さ3ミクロンのフィルムで作られており、入念に設計された音響設計により、パノラマ音を実現し、「頭の中の音」の影響を排除することができました。









SRラムダ



今日で最も人気のあるシリーズの1つとなったヘッドフォンは、8 Hz〜35 kHzの周波数範囲で音を再生し、感度は102 dB / 100 V rmsで、最大109 dBの音圧を発生させることができました。









強化された1986 SR-Lambda PRO



80年代は、STAXでさらに飽和状態になりました。 同社は、シリアルローンチ用に75以上の製品を開発および準備できました。 とりわけ、Stax SR-Lambda Proと呼ばれるSR-Lambdaのスタジオバージョンがリリースされました。 既に確立されている静電ヘッドフォンに加えて、StaxはCDP Quattro CDプレーヤーのリリースによりデジタルオーディオの先駆者の1つになりました。







ビニールディスクを再生するためのデバイスの開発は継続され、アンプは別の方向に割り当てられました。 合計で、1980年代に76個のデバイスが設計され、12個の静電型ヘッドフォンモデルを含むシリーズが発売されました。



ファウル寸前の伝統主義



1990年代半ばに、同社は財政難に直面し始めました。 事実、90年代に主なオーディオファンの傾向、つまり再現の忠実性の考え方は、マーケティング担当者や「全能の」オーディオメディアの情報圧力の下でひび割れて変化したということです。 アクセシビリティとユーティリティが流行し、フィリップスやパナソニックなどの巨人が毎年市場に参入したという事実を背景にしたSTAX製品は、どういうわけか冗長で不当に高価であるように見えました。





多くのSTAXデバイスの基本的な要素は何十年も変わっていません。たとえば、POD-X（70年代の開発）では、多様で高価な親油性の「綿毛」と伝説のTelefunken 12AU7ランプがはっきりと見えます
上記に関連して、2000年代の初めまでに、同社は財政的に疲弊し、破産寸前にありました。 困難な経済状況にもかかわらず、ほとんどの部門は新製品の開発を続けました。 1996年、会社は再編成され、更新された構造で活動を継続しました。 正確に何が変わったのか、企業のリソースは沈黙している、林直武は以前と同様に会社を経営し続けた。 ただし、すべての公式ソースでは、1996年以降の期間はNew Staxと呼ばれます。 おそらく、問題はヘッドフォンで動作しなかった会社のすべての部門の廃止です。



再編成は会社をあまり助けなかったし、STAXの財政難は激化し、ゼロの真ん中でピークに達した。 この期間にわたる巨額の努力により、同社は主にヘッドフォン、それらのアンプ、アダプター、その他のアクセサリーを含む一連の64製品を発売しました。 90年代の革新の顕著な例は、ポータブルアンプを搭載した世界初のインチャネル静電ヘッドフォン、STAX SR-001です。 新しいアイテムは保存されず、シリーズは減少し、収益は減少し続けました。





STAX SR-001 mk2-高度な修正

吸収を節約



いくつかの情報は、90年代後半に林直武の創業者および常任リーダーが彼のポストを辞めたことを示唆しています。 ネットワークで利用可能な断片的なデータから、1991年に彼は84歳であり、まだ企業を経営していたことが知られています。 明らかに、彼の辞任は会社の業務の改善に貢献しませんでした。









2011年12月、STAXは中国のスピーカーメーカーEdifierに買収されました。 中国企業は、象徴的な価格120,000,000円、100万ドル強でStaxの100％の株式を取得したと語った。 新しい所有者は、静電ヘッドフォン、アンプ、DACの生産を継続することを決定しました。 会社へのさらなる投資は彼女を破滅から救い、開発と連続生産を継続することを可能にしました。



まとめ



昨年、同社は80歳になりました。 この記念日に、STAXは伝説的なSR-009BKヘッドフォンの限定版をリリースしました。







現在、同社の他の製品は数千個のシリーズで生産されています。 財政状況で経験された困難と、より大きな教育機関の管理への移行にもかかわらず、STAXは、彼らが日本で言うように、顔を失うことはありませんでした。 彼らは市場に追従せず、潮に逆らって泳ぐ機会を見つけました。









今日、彼らは何年も前に選ばれた方向に忠実であり、彼らの製品と電気音響原理そのものへの関心が戻ってきています。 興味深いことに、同社には修理を扱ったワークショップがあり、20年前に製造されたモデルも取り扱っています。これは、製品の品質とビジネスへのアプローチの両方について多くを語っています。
