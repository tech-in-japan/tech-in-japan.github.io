時代とともに歩む投資：Intel Capitalは10億ドル相当の資産を売却し、高度なテクノロジーに投資することを計画しています



出典： barrons.com



インテルの投資部門は、最も有望な開発分野に焦点を当てることを計画しています。 コストを削減するために、会社は二次資産を取り除くことができます。 確かに、2013年5月にIntelのCEOとしてPaul Otelliniに代わったBrian Krzanichのイニシアチブで、このトピックは2015年半ばに取り上げられました。 しかし、問題は会話にとどまりませんでした。



ウォールストリートジャーナルによると 、Intel資産のわずかなシェアを10億ドルで売却することについて話しています。



オテリーニ率いるインテルキャピタルは、世界で最も活発なベンチャーキャピタル投資家の1つでした。 1991年以来、Intel Capitalは57か国の1,440社に116億4,000万ドル以上を投資しています。



近年、市場構造は大きく変化しました。モバイルデバイスは、従来のPC、クラウドテクノロジー、企業システム、情報セキュリティツール、データセンター、モノのインターネットよりも人気が高まっています。



二次資産の売却により、Intelは米国および中国を含む企業にとって最も重要な地域で資金を再配分し、事業をより積極的に発展させることができます。



2014年、 Clouderaの大規模データ処理システムはIntel Capitalから7億4,000万ドルを受け取りました。 総投資額は9億ドルに達し、ClouderaでのIntelのシェアは18％に増加しました。 Intelのおかげで、同社は Hadoopエコシステムのリーダーになりました 。 取引後、Intel独自のHadoop製品の開発は中止されました。



2015年8月、インテルはMirantisに 1億ドルを投資した投資家のコンソーシアムを率いました。 同社は、OpenStackオープンテクノロジーに基づいたクラウドストレージサービスを販売しています。 2013年にインテルのベンチャーキャピタル部門が初めてMirantisに投資しました。



Intelは、そのプロセッサが最新のほとんどのサーバーで使用されており、伝統的にソフトウェア開発者と協力しようとしています。 このコラボレーションの結果として、Intelのテクノロジー販売の増加に役立つ製品が作成されます。



2015年7月、同社は、すべてのクラウドイニシアチブの一環として、多くの企業に投資する意向を発表しました。 これは、エンタープライズデータセンターがAmazon Web Services（ AWS ）やMicrosoft Azureなどのクラウドサービスと比較できるようにすることを目的としています 。



同年、インテルはUBSから専門家を雇い、インドで多くの資産を売却する見込みを分析しましたが、結果はありませんでした。 インテルキャピタルの経営陣は重大な人事異動を経験しており、この取引は早急に検討されました。 ユニットの社長であるアービンド・ソーダーニは会社を辞めました。 Wendell Brooksが代わった。 Sodhaniは35年間会社に在籍しています、 と Bloombergは書いています。



2月、Wendell Brooks社の新社長は、Intelが「Intelが強い地位にある技術分野の全領域にわたって、スタートアップに毎年3億から5億ドルを投資し続ける」と発表しました。



売却の可能性があるというニュースの中で、インテルの株価は0.51ドルから31.76ドルに上昇しました。
