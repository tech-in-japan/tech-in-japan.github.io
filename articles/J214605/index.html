Globex GU5011Bレビュー-MediaTek MT6582のクアッドコアソリューション

昨年の夏、中国の大手MediaTekは、新しい4コアMT6582プロセッサをリリースする計画について話しました。これは、高周波数のMT6589とは異なり、新しいビデオアクセラレータです。 このレビューでは、ウクライナのメーカーの新しいプロセッサを搭載したスマートフォンであるGlobex GU5011Bに焦点を当てます。







MT6582は、中国のメーカーMediaTekのクアッドコアソリューションであり、1.3 GHz Cortex™A7プロセッサとMali-400 MP2ビデオアクセラレータです。

スマートフォンGlobex GU5011Bは、価格帯で非常に深刻な特性と独自の特性を備えたミドルクラスのモデルです。



特徴



スマートフォンはクアッドコアMediaTek MT6582プロセッサーで動作し、クロック周波数は1.3 GHzです。これは、ARM Cortex A7の4コアプロセッサーのMediaTekファミリーの中で最も新しいモデルですが、明らかにMT6589に劣らず、一部のテストでも後れを取っています。 このプロセッサでは、会社はMali-400 MP2ビデオアクセラレータを使用することにしました。



  • 画面： 5.0インチ（16：9）、960x540ピクセル、IPS OGS、240 ppi
  • プロセッサー： MediaTek MT6582（Cortex A7、1.3 GHz、4コア）
  • ビデオアクセラレータ： Mali-400 MP2
  • オペレーティングシステム： Android 4.2.2
  • RAM： 1 GB
  • ユーザーメモリ： 4GB
  • カメラ：フロント-2メガピクセル、リア-5メガピクセル
  • 通信： Sim1-GSM / GPRS / EDGE、Sim2-UMTS / HSPA + / WCDMA
  • オプション： GPS、A-GPS、EPO、Wi-Fi、Bluetooth、加速度計、光と距離センサー
  • バッテリー： 1700 mAh
  • 寸法： 144.5 x 72.5 x 8.5mm
  • 重量： 143グラム


パッケージバンドル



箱のサイズは中くらいで、主な機能は前面にあります。 パッケージの主な色は白で、つや消しの表面に触れると、箱にアンドロイドのロゴが付いた携帯電話の画像は光沢で覆われます。







裏面には、より詳細な仕様と品質証明書があります。 ロシア語で書かれた情報と英語の混合。







箱は3つの部分で構成されており、情報格納式のカバーと電話が付いている箱自体です。







すべてが「無料」で詰め込まれ、多くの空きスペースがあります。空きスペースを使用する方が良いでしょう。 携帯電話自体はビニール袋に入っていて、座席を歩いていました。 しかし、深刻な構成のおかげで、あなたはそれに注意を払っていません。 パッケージには、保護フィルム（接着済み）、保護用シリコンケース、追加のブラックケース、ヘッドセット、microUSBケーブル、USBアダプター、ロシア語の説明書、スマートフォンの公式保証が含まれています。







ヘッドセットに注意を払う必要があります;メーカーはコストを削減しませんでした。

設計



最初に注意することは、スマートフォンの厚さとその重量です。 Globex GU5011Bは、非常に軽量（143グラム）でコンパクトで、明らかな突出/固着ボタンはありません。 スマートフォンの側面の周囲には装飾的な赤いインサートがあり、注目を集めています。 スクリーンはガラスで覆われ、側面に小さな側面があります。







上部パネルには、フロントカメラ、スピーカー、microUSB、OMTP標準のジャック3.5ヘッドセットの出力、近接センサー、および光センサーがあります。















電話の右側にあるのは電源ボタンのみで、パートタイムとロックボタンです。 電話を右手に持っている場合、親指の位置にあります。 ボタンはボートの形をしており、簡単に模索され、スマートフォンの全体的なデザインと完全に融合します。







ロッカーのボリュームコントロールの左側。







一番下のパネルには、メニュー、ホーム、戻るの3つのタッチボタンがあります。 したがって、スマートフォンの音色に赤いバックライトが付いています。







背面パネルは完全にニスが塗られていますが、もちろんこれは最良の解決策ではありません。わずかなタッチで痕跡が残ります。







同じ側​​にメインカメラ、フラッシュ、スピーカーがあります。











スマートフォンの背面カバーは取り外し可能です。左側をこじ開けて取り外すのは難しくありません。







カバーの下には、標準サイズのSIMカード用の2つのスロットとmicroSDスロットがあります。

スクリーン



Globex gu5011bの画面がこのユニットの誇りであると言えます。 対角線は5インチで、960〜540ピクセルの解像度のOGS、IPSテクノロジーを使用して作成されています。 どの視野角でも、色再現を完全に表示します。







スクリーンは、OGSテクノロジー-1つのガラスソリューション（1つのガラス上のソリューション）に従って作成されます。

ネットワークからの情報：

「センサーと液晶層の間に層がないため、この技術を使用して作られたスクリーンは、輝度の増加と大きな視野角によって区別されます。 ディスプレイの厚さを薄くすることにより、デバイスはより薄く、より軽くなります。」
コンテキストでのOGSテクノロジー。






テキストは明るい日光の下でも画面から読みやすく、バックライトは光の明るさの変化にすばやく反応します。











マルチタッチは完璧に機能し、湿気と低温の両方でテストされ、同時に最大5回のタッチを処理します。

インターフェース



スマートフォンGlobex GU5011Bは、変更されたインターフェースでAndroid 4.2.2を実行しています。 クイックアクセスパネルのみがここで再設計されましたが、一般的に設定は純粋なAndroid 4.2と同じです。











製造元は、4GBの電話のメモリは実際には2.05GBであると述べていますが。 追加のメモリカードなしではできません。 アプリケーションをインストールする場所を選択できます。







電話機は、2G（GSM / GPRS / EDGE、850/900/1800/1900 MHz）と3G（UMTS / HSPA + / WCDMA、850/900/1700/1900/2100 MHz）の2つのネットワークで同時に動作します。







Wi-Fiは、標準のb / g / nプロトコルで動作します。 GPSの動作は、強い妨害、テクノロジー、GPS、A-GPS、EPOを引き起こしません。 MT6627チップは位置を決定し、衛星はすぐに決定されます。
