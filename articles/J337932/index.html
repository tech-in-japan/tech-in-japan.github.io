VMworld 2017ヨーロッパ会議。 1日目





VMworld 2017 Europeカンファレンスで最も興味深いものについての話を続けます（ カンファレンスについての話の最初の部分 ）。



最初の日は、感動的な見出しの下での一般セッションから始まりました： 優れた技術者が問題を解決します。 偉大なイノベーターが機会を創出します 。 VMware CEOのPat Gelsinger氏は、テクノロジーが私たちの周りの世界をどのように変えているかについて語りました。 彼は、人々がどれだけ早く新しいものすべてに適応できるかの例を挙げました。 たとえば、無人車両に対する最初の恐怖は、「内部ではどのように機能しますか？」という関心に置き換えられます。 その結果、ITは企業の成長の主な源泉になりつつあります。 そして、100年前に人が読み取り、書き込み、カウントの3つのことを実行できなければならなかった場合、「読み取り、書き込み、読み取り」のようになります。



VMwareは、クラウド（プライベートまたはパブリック）内のアプリケーションの動作を保証するソリューションを提供し、あらゆるデバイスからアクセスできます。 そのようなソリューションの1つにVMware HCXがあります。これは、オンプレミスインフラストラクチャとパブリッククラウドサービス（IBMクラウドやOVHなど）間のリンクを提供し、ダウンタイムなしで仮想マシンをリアルタイムに転送できます（ゼロダウンタイム移行）。



ただし、すべてのお客様がパブリッククラウドに突進する準備ができているわけではありません。 インフラストラクチャの一部がオンプレミスで展開され、一部がクラウドに展開される場合、多くの人がハイブリッドアプローチを実践します。 これらのハイブリッド環境を管理する機能を改善するために、VMwareは、監視、コスト分析、ネットワーキング、アプリケーションセキュリティ、およびアプリケーションへのシングルポイント接続を含むSaaS対応製品のスイートであるVMwareクラウドサービスを提供します。



次に、パブリッククラウドVMware Cloud on AWSについて説明しました。 要するに、Amazonは、世界中の隅々にある仮想データセンターと、VMware SDDC製品のセットがインストールされているエンジニアリング、ネットワークインフラストラクチャ、およびサーバー機器を提供しています。



VMwareは、仮想環境内で実行される最近発表された他の製品、VMware Integrated OpenStack 4.0（OpenStack独自の配布キット）およびVMware AppDefense（アプリケーションセキュリティソリューション）も想起しました。







展示会



一般セッションの終了後、訪問者は解散し始めました。 他のセッションに急いでいる人もいれば、ラウンジゾーンでリラックスしている人もいれば、パートナーソリューションSolutions Exchangeの展示会に行った人もいました。







毎年100を超える企業がVMworldに立ち、あらゆる種類の製品とソリューションのデモを行っています。 たとえば、Dell、HPE、NetApp、IBM、Hitachi、R​​ed Hat、Arista、Nutanix、および有望なスタートアップのスタンドにアクセスできます。 ロシアのルーツを持つIT企業が展示会に参加することは特に楽しいです：カスペルスキーとVeeam。



フックまたは詐欺師によるベンダーは、訪問者をスタンドに招待しようとします。







Intelスタンドでのゲーム。参加者は、ビジネスニーズを解決するために最適な構成のサーバーを選択するのに一定の時間必要でした。







最初に注目したのは、主にその巨大なサイズによるVMwareブースでした（何らかの理由で、「女の子と食事をする人は彼女を踊る」と言ったことを思い出しました）。







ここでは、デスクトップVMware Workstationハイパーバイザーから最近発表されたアプリケーション保護製品であるAppDefenseまで、会社のすべての分野を紹介しました。各ソリューションには、すべての質問に答える専門家がいます。







エンドユーザー向けソリューション（エンドユーザーコンピューティング）が実証されているさまざまな業界向けの様式化されたスタンドについて、特に言及する必要があります。







たとえば、デックストップモード（Samsung DEX）をサポートするSamsungスマートフォンは、銀行セクター向けに実証されました。 クレードルにスマートフォンをインストールしたら、キーボード、マウス、モニターを接続して、本格的なデスクトップコンピューターとほとんど同じように動作できます。 または、Horizo​​n View Clientを実行して、VDIインフラストラクチャに接続し、公開アプリケーションおよび仮想ワークステーションにアクセスして、デバイスをモバイルシンクライアントに変えることができます。







次に、Amazonブースを訪れました。VMwareとAmazonの共同クラウドサービスの実装について詳しく知ることができます。







Cloud on AWSは昨年のVMworldで発表され、最終的にソリューションはエンドカスタマーが利用できるようになりました。











わずか数個のボタンで、仮想データセンターをAmazonクラウドにデプロイできます。 そのような各データセンターには以下が含まれます。

  • VMware vSphereクラスター（少なくとも4つの物理サーバー）。
  • vCenter Server管理サーバー
  • VSAN分散ストレージ
  • NSXソフトウェアの構成可能なネットワーク。


すべてのサーバーには一般的な構成があります。2つの18コアプロセッサ、512 GBのRAM、10.7 TBのフラッシュ用ストレージスペース。 負荷の増加に応じて手動または自動モードでノードを追加することで、仮想データセンターのコンピューティングリソースを増やすことができます（いわゆるElastic DRS）。 Cloud on AWSは、オンプレミスの顧客インフラストラクチャとの統合メカニズムを提供し、単一のvSphere Client（ハイブリッドリンクモード）コンソールからvSphereオンプレミスおよびクラウドクラスターと仮想マシンを管理できるようにします。



VMwareは、構成、パッチ管理、ハードウェアメンテナンスなどのクラウドインフラストラクチャを完全にサポートしています。



VMwareクラウドはAmazonクラウドサービスと同じデータセンターにあるため、EC2、S3などを使用する場合、顧客は広帯域幅、低ネットワーク遅延、トラフィックアカウンティングの欠如という追加のメリットを享受できます。



VMwareは、Cloud on AWSの3つの主なユースケースを特定しています。

  1. 独自のデータセンターを構築せずに、既存の仮想インフラストラクチャを拡張したり、災害復旧のためにバックアップサイトを編成したりする。
  2. 複数の既存のデータセンターの統合、または既存のアプリケーションとVMへの影響を最小限に抑えたクラウドへの完全な移行。
  3. テスト/開発タスクのための、または循環負荷中のコンピューティングリソースの迅速な提供（販売シーズンのピーク）。


現在、仮想データセンターの展開は1つの地域（米国西オレゴン州）でのみ利用可能ですが、近い将来、地域のリストが拡大されます。



HPEブースでは、新世代のProLiant GEN10サーバーが展示されていました。 特に、1U HPE ProLiant DL360 GEN10ラックマウントサーバーが導入されました。







GEN10のイノベーションの中

  • Intel Xeonスケーラブルアーキテクチャプロセッサのサポート。 新しいプロセッサには、より多くのコア（Intel Xeon E5-2699A v4の22コアではなく、古いIntel Xeon Platinum 8150Mの28）、より高いクロック速度、3次レベルのキャッシュメモリ、および最大205 Wのサーマルパッケージがあります。
  • NVMeインターフェースを備えた多数のフラッシュドライブのインストールをサポート。SATA/ SASインターフェースを備えたドライブと比較して、より高速なデータ転送速度と低レイテンシーを実現します。 1Uサーバーでは、2.5インチのフォームファクターで作成されたこれらのドライブを最大10台インストールできます。 より容量の大きい2Uサーバーは既に20です。HPEポートフォリオに最大2 TBのNVMeドライブが含まれていることを考慮すると、総容量を簡単に計算できます。
  • 信頼されたブートテクノロジーのサポート。これにより、マイクロコードおよびオペレーティングシステムの署名されていないバージョンのインストールとサーバーでの起動をブロックできます。また、機密データの漏洩を防ぐローカルドライブをクリーニング（ゼロ）できます。


サーバーメーカーが25 Gb / sネットワークアダプターに徐々に移行し、それに応じてネットワーク機器の100 Gb / sポートに移行することは注目に値します。 最新のハイパーコンバージドソリューションやソフトウェア定義のストレージシステムに役立つ可能性のある10 GbEと比較して、帯域幅が2.5倍増加し、レイテンシが減少することに加えて、これらのアダプターは、通常のSFP + QSFP +よりも場所。







Intelブースでは、Intelサーバー、プロセッサー、フラッシュドライブ、およびVMwareソフトウェア（vSphereおよびvSAN）に基づくハイパーコンバージドプラットフォームがデモされました。







ホームには、一見したところ、2つのIntel Xeon Gold 6152プロセッサ、768 GBのRAM、3D XPointメモリに基づく4つの375 GB Intel Optane DC P4800Xドライブ、および12の「クラシック」NVMeフラッシュドライブIntel DC P4500が含まれています。







ドライブは2.5インチのフォームファクターで作成され、ホットスワップをサポートし、特別なホストアダプターを介してPCI-Eバスに直接接続されます。







そのようなサーバーの1つは、4 KBブロックのランダムロードと70/30の読み取り/書き込み比で、38万IOPSのレベルのパフォーマンスを提供できます。







次に並んだのは、HCI（HyperConverged Infrastructure）ゾーンです。これは、単一のビルディングブロック内でコンピューティングリソースとストレージリソースを組み合わせたハイパーコンバージドソリューションです。 インテル、レノボ、および富士通のパートナーソリューションに加えて、Dell-EMC VxRail 4.0 HCIプラットフォームがブースで展示されました。







VxRailは、DellサーバーとVMware vSANソフトウェアに基づいて構築された完全な「ボックス化」ソリューションです。 現在、VxRailラインには、さまざまな問題を解決するために設計された5種類の構成が含まれています：高密度GシリーズサーバーおよびEシリーズ（エントリーレベル）の典型的な1Uサーバーから、グラフィックアクセラレーターのインストールをサポートし、仮想ワーカーを実行するために最適化されたVシリーズサーバーまでステーション、高性能（Pシリーズ）または大容量（Sシリーズ）サーバー。











VxRailは、他のほとんどのHCIソリューションと同様に、すぐに使用できる形式で顧客に提供されます。 サーバーをラックにマウントし、スイッチを入れ、電源を入れ、簡単な初期セットアップウィザードを実行するだけで十分です。完全に機能する仮想インフラストラクチャである「Voila！」はすぐに使用できます。



HCIおよびその他の若者向けソリューションを普及させるために、Dell EMCは一連の漫画をリリースしました。

Mellanoxブースでは、さまざまなConnectXネットワークと収束アダプターがサポートされ、10、25、40、100ギガビット/秒、さらには200ギガビット/秒の接続をサポートしました。 ポートの数と速度が増加するにつれて、PCI-Eバスの要件がどのように増加しているかに注意してください。







Mellanoxの隣にはSupermicroブースがあり、Intel XeonスケーラブルおよびAMD EPYCプロセッサーをサポートする最新のツインフォーマットサーバー（1台のシャーシに2台または4台のサーバーを収容可能）のデモを行いました。 Supermicroサーバーは、質素な顧客、ハードウェアビルダー、およびアプレットのメーカーの間で常に人気があります。たとえば、NutanixのWeb Scaleプラットフォームは、そのようなサーバー上に構築されています。







サーバーには交換可能なLOM（Lan-on-board）アダプターがあり、マザーボードを交換したりPCI-E拡張スロットを使用したりすることなく、ネットワークポートの必要な構成を選択できます。
