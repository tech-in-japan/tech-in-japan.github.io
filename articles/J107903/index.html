Ciscoのテレフォニー（Call Manager Express）-パート2

2番目の部分では、ルータをPBXに接続するために使用できるデジタルE1トランクの使用について説明します











1.回路のコンポーネント



NMスロットにNM-HDV2モジュールとVWIC2-1MFT-T1 / E1カードが取り付けられたCisco2811ルーターがあり、同じモジュールに8チャネルPVDMが取り付けられています。

また、アナログ電話を接続するためのVIC2-2FXSモジュールがルータに取り付けられています。

PBXとして、E1接続用のTEPRIカードと2つのOfficeServ KPDP-14SED電話が接続された8-DLIカードを備えたSamsung iDCS-500デジタル電話交換機を使用しました。



2.ルーターの基本構成

インターフェイスの状態を確認するには、次を実行できます。

1 serial interface

Router(config-controller)#channel-group 0 timeslots 1-31

必要に応じて、さまざまな帯域幅を持つ複数のシリアルサブインターフェイスを作成できます（タイムスロットの数に応じて）

この操作の後、新しいインターフェイスSerial1 / 0/0：1があります。

次に、ハードウェアループバックを作成し、RJ-45コネクタと長さ約10 cmのツイストペアケーブルから2本の配線を取ります。最初のワイヤをピン1と4に挿入し、2番目のワイヤをピン2と5に挿入します。

E1インターフェイスに挿入すると、CD（キャリア検出）LEDが緑色に点灯し、Serial1 / 0/0：1インターフェイスが起動し、インターフェイスが機能します。



アナログポートの構成は最初の部分で検討されましたが、詳細に停止することはしません。 アナログ電話の場合、次を追加します。

dial-peer voice 1 pots

destination-pattern 101

port 0/1/0

!

dial-peer voice 2 pots

destination-pattern 102

port 0/1/1









3. PRIトランク構成



E1コントローラーをPRIモードにします。

controller E1 1/0/0

pri-group timeslots 1-10,16







タイムスロットの数は利用可能なPVDMに依存します。リソースが不足している場合、エラーメッセージが表示されます。 上記のコマンドを使用すると、音声の最初の10個のタイムスロットを使用できます。 トランクでは、10を超える同時会話はできません。 16番目のタイムスロットは、シグナリングに使用されます。



シグナリングチャネルを設定します-これはSerial1 / 0/0インターフェイスの15番目のサブインターフェイスで、コントローラの設定後に表示されます

interface Serial1/0/0:15



# PRI

isdn switch-type primary-net5

#

isdn incoming-voice voice







発信通話の番号計画を構成します。

dial-peer voice 3 pots

#dial-peer , "3"

destination-pattern 3..

# PRI,

port 1/0/0:15

forward-digits all

# direct-inward-dial

direct-inward-dial







このルーターの構成は完了していると見なすことができます。PBXに接続するには、2つのRJ45とピン配列を備えたクロスケーブルを使用する必要があります。

1-4

2-5

4-1

5-2







さらに、TEPRIカードが正しく設定されていることを確認する必要があります。ネットワークモード（ステーション部分、デフォルトではカードはユーザーレベルにあり、上位レベルのPBXに接続する）に設定し、E1動作モードを設定する必要があります。

これを行うには、ボードからケースを取り外し、表に従ってジャンパーを設定します。







PBXには、内部番号計画も構成する必要があります（私の場合、3xx）



すべてが正しく構成されている場合、PBXで対応する電話番号をダイヤルすると、呼び出し元のアナログ電話の電話番号が電話画面に表示されます。



発信コールは、サブインターフェイスの1つを発生させます。

Nov 16 16:21:58.799: %ISDN-6-CONNECT: Interface Serial1/0/0:9 is now connected to 302 N/A
