オフィスのHP Microserver

ごく最近、HPのサーバー部門は非常に興味深く、やや予想外の製品をリリースしました。 それは話すマイクロサーバーという名前を受け取りました-デバイス自体を見ることさえせずに、一方で、おおよそ何が危機にatしているのかを想像し、他方で、何でも空想できます。 新しいサーバーは小規模なワークグループ向けに設計されており、オフィスでの使用に重点を置いていますが、自宅での使用も禁止されていません。 このモデルに興味を持ったので、Microserverがどのように役立つか、何に適応したかについての記事をいくつか書くことにしました。







注意、たくさんの写真！







投稿を読むときは、いくつかの使い慣れた方法でしかサーバーを使用できないことを理解することが重要です。 デバイスを使用するための独自のシナリオがより便利で、独創的で、好奇心をそそられる可能性があるため、コメントに自由に投稿してください。



免責事項：著者はサーバーの構成とインストールの専門家ではありません。これは自宅やオフィスでのマルチディスクストレージに使用される予定の個人用サーバーです。



最初の部分では、サーバー自体と、それがオフィスでどのように役立つかについて少しお話ししたいと思います。 Microserverのサイズは、少し幅が広いことを除けば、通常のタワーシステムユニットの半分に似ています。 上部には5インチのデバイス用の場所があり、そこには何でもインストールできます（少なくともBlu-Rayドライブ）。 拡張オプションが不要になることはありません。最終的には、そこにもう1台のハードドライブを配置することができます。



ドライブといえば。 ITコミュニティの目には魅力のかなりの部分であるMicroserverは、SATAドライブ用の4つのスロットの存在を義務付けられています。 4台のディスクのNASストレージのネットワーク価格を見ると、マイクロサーバーを購入し、そこに4台のディスクをインストールし、RAIDアレイを上げて、さらにサーバーを受け取って生活を楽しむ方がはるかに有益であることが明らかになります。 その結果、より少ないお金が費やされる可能性があります。



インターフェイスのうち、MicroserverにはギガビットイーサネットとeSATAがあります。 もちろん、2番目のネットワークインターフェイスを個別にインストールする必要があるのは残念ですが、少なくとも必要なコネクタが利用可能である必要があります。 ただし、たとえば、この場合のwi-fiは、USBドングルを使用して追加する必要があります。 すべてに十分なスロットがあるはずです-Microserverケースには、前面に4つ、背面に2つのUSBソケットが6つあり、もう1つは内部にあります。したがって、必要に応じて、高速で大容量のフラッシュドライブなどを隠しておくことができます。 通常、このような柔軟性は通常のコンピューターでは不十分であり、ほとんどの場合、小規模オフィスのサーバーとして機能します。



テストのために、Microserverをモスクワの中心にある通常の2部屋のオフィスに持ち込みました。 5人は小さなIT会社で働いており、かつては非常に強力でしたが今ではほとんど役に立たない古いシステムユニット（HP製）を改造しようとしました。 古いコンピューターでは必要なタスクに対応できなくなったため、Microserverが役に立ちました。 まず、Microsoft Windows Server 2008 R2 Standard Editionがサーバーにインストールされました。これは、仕事のニーズにより適していますが、一般に、ほとんどすべての最新のOSをMicroserverにインストールできます。 必要に応じて、自宅のコンピューターにすることもできますが、ストーリーの後半ではさらに詳しく説明します。



Linksys WRT610Nは、高速ですが、本質的にはホームWi-Fiルーターであり、オフィスルーターとして使用されます。 しかし、小規模オフィスのニーズには、一般にそのパフォーマンスで十分です。 Microserverのセットアップに問題はありませんでした。手順は一般的であり、システム管理者なら誰でも知っています。 Microserverの高度な使用方法は計画されていませんでした。サーバーには予備のハードドライブのペアが装備され、データウェアハウスになりました。 平均データ転送速度は1秒あたり約5 MBでしたが、時々1.5 MBに低下しましたが、これは既にワイヤレスネットワークの機能と接続されています。



オフィスで最高のマイクロサーバーは何ですか？ サーバーの組み込みディスクコントローラーはRAID1をサポートしているため、データ複製を確立し、重要なデータのストレージとしてMicroserverを使用できます。たとえば、会社が設計やビデオ編集を行っている場合は、あらゆる種類のデータベース、ドキュメント、マルチメディアコンテンツです。 今日の2テラバイトのドライブ（少なくとも同じWestern Digital Green WD20EARS）が利用可能であれば、これは問題になりません。 まあ、ディスク容量が少なくなると、eSATAを介して外部ストレージを接続することが可能になります。



Microserverは、特にRAMを追加した場合（最大8ギガバイトまで増やすことができる場合）、特にメールサーバーとして完全に表示できるという意見もあります。 プロセッサーは変更されません-1.3 GHzのAMD Athlon II。 この構成では、サーバーは穏やかにExchangeをプルし、信頼できる「ネイティブ」メールをすべての従業員に提供します。 まあ、もし会社に真剣な簿記があれば、会計データベースをどこに保存するかが明らかになります。



Active Directoryサーバー上で構成し、使用されるすべてのサービスへの単一のエントリポイントのすべての利点を享受する機能によって、この図は完成しますが、これは小さな会社にとってそれほど重要ではありません。



サーバー設計の最も便利な機能には、完全に取り外し可能なマザーボード、使い慣れたディスク用サーバードライブベイ、および厚い金属製のドアがあります。 蓋とドアは簡単に取り外してケース内にアクセスできます。 サーバーの全体的な設計は、思いやりのある心地よい感覚を残します。MicroserverはHPのサーバー標準に従って作られ、本格的な「プロライアント」であると感じています。 たとえば、マザーボードを取り外すには、2本のネジを外すだけで済みます。 これはドライバーなしで簡単に行うことができます;指でネジをねじ込んだり外したりできるように、キーねじ接続が行われます。 ことわざにあるように、「些細なことですが、素晴らしい」。

これで、おそらく、四捨五入します。 ホームサーバーとしてのMicroserverの動作についてお話ししたい第2部では、ここですべてがさらに興味深いものになるかもしれませんが、使用シナリオはさらに私の頭に突入します。 ここには、テキストの編集、HDビデオのストレージ、オルゴールなどのためのシンプルなマシンがあります。
