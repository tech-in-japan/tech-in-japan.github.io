Mac OS Xのウイルスについての真実

Mac OS Xのウイルスに関する優れた記事で、先ほどMacユーザーになった気分についてメモを書いた同じ著者によるものです。

この記事は、単純な言語で、常識のレベルで、同時に狂信性がないという点で興味深いです。MacOS Xの悪意のあるアプリケーションの現在の状況を説明しています。この記事は、Macで読み、記憶し、 「ウイルスがいっぱい。」



ウイルスについての真実



Macのマルウェア（ウイルス、トロイの木馬、ワーム、およびその他の悪意のあるソフトウェア）で実際に何が起こっているかについての情報をインターネットで閲覧すると、熱烈な議論、半分の答え、「終末の日」に関する非常に便利な発言の断片に出くわすでしょうウイルス対策ソフトウェア会社によって発行されました。



問題は、Mac用のマルウェアに関する資料を学習できる場所がほとんどないことです。 しかし、長年にわたってオペレーティングシステムの「詳細な」研究を行ってきたため、このギャップを埋めることに決めました。



リストに進む前に、私はMacを約4〜5年使用しており、その前はWindowsとDOSに密接に関与していたと言わなければなりません。 長い間、どのプラットフォームでもマルウェアに遭遇したことはありません。



Macマルウェアの仕組みは次のとおりです。



01壊れないものはない



コンピューターが外界と何らかの接触を持ち、人々が彼らのためにプログラムを作成している限り、一部のマルウェアがコンピューターに侵入する可能性があります。 一部のソースコードは他のコードよりも安全ですが、ハッキングできなかったものはありません（少なくともコンシューマソフトウェアの現実には）。



02過去には、Macにはウイルスがありました



OS X以前には、Macに感染し、あらゆる種類の不快なことを行うウイルスがいくつかありました。そのため、所有者はひどい痙攣を起こしていました。 ただし、Mac OSがOS Xへの移行中に完全に書き換えられたという事実は、これらの古いウイルスがすべて現代のMacに感染できないという事実につながっています。



03今、あなたのMacは安全です



この特定の時点で、Macに感染する可能性のあるマルウェアは無料で流通していません。 ウイルス、トロイの木馬、テキストロガー、ボットネット、何もありません。 ネットサーフィン、電子メールの読み取り、インスタントメッセージングシステムでのチャットは、青に変わるまで可能です。コンピューターに害はありません。 ただし、明日の朝、誰かがOS Xの未知の脆弱性を悪用するウイルスをリリースし、正午までにネットワークに接続されている世界中のすべてのMacのデータを破壊する可能性があります。



04マルウェアの概念があります



はい、一部の人はMacに害を及ぼす可能性のあるコードを作成できましたが、これらは「オープンワールド」や「実生活」のMacにまだヒットしていません。 そして、はい、それらの多くは実際にコンピューターに何らかの害を与えるために、ある種の神秘的な動きを必要としました。 しかし、それらは現在、この惑星のどこかに存在しています。



05 Mac OS Xには間違いなくエラーがあります



ほとんどのマルウェアは、コンピューターのオペレーティングシステムでソフトウェアエラーを使用して機能します。 また、Macには独自のエラーがあり、その修正のために（原則として、最も悪名が高いまたは目立つ）Appleはオペレーティングシステムのアップデートを定期的にリリースしています。 ただし、これまでのところ、Mac OS Xのエラーはマルウェア開発者にとって十分な肥沃であると証明されていません。



06 Unix内部は間違いなく役立ちます



内部には、Mac OS XにはUnixのバリアントがあり、その上には、すべてのシステムサービスと作業するビジュアルシェル用にAppleによって書かれた大量のコードがあります。 「実世界」のこのバージョンのUnixは長い間使用されており、長年にわたって多くのエラーが発見され、修正されてきたため、非常に安全になりました。



07 Intelの画像の存在は変わりません



WindowsマルウェアはIntelプロセッサ（および互換）で使用されるように設計されているため、Intelプロセッサに切り替えるとMacの安全性が低下するのではないかと心配する人がいるかもしれません。 マルウェアは通常、プロセッサではなくオペレーティングシステムの脆弱性を使用して記述されるため、これは考えられません。 ただし、これは新しいアプローチであり、特定の瞬間に地下室の誰かがこの方法を思いつく可能性が非常に高いです。



08市場シェアは重要ではない



Mac用のウイルスは市場のごく一部であるため、Mac用のウイルスはないと主張されることがよくあります。 ある程度、これは真実かもしれません。 マルウェアを作成している場合、特に有益なボットネットまたはスパムネットワークの一部である場合、または自分の名前を付けたい場合は、おそらくコンピューターの最大シェアを目指しているでしょう。 ただし、このようなプログラムの開発者の多くは名声も愛しています。さらに興味深いのは、Windows用に別の147,281番目のウイルスを作成するのか、Mac用の最初のスーパーウイルスを作成するのでしょうか。 誰かが「こんにちは、私はMacです」という男に良いキックを与えたいと思うでしょう。



09マルウェアの作成に関わる大金



今日の「有害な」プログラムのほとんどは、コンピューターから情報を取得する（銀行のWebサイトにアクセスするときにユーザー名とパスワードを収集するなど）、または数千（または数万）の巨大なネットワークを作成してスパムし、さまざまなWebサイトでDDoS攻撃に参加する。 いずれにせよ、多くのお金が回っていますが、これはアマチュアによって制御されるにはほど遠いです。 そして、Appleユーザーをターゲットにすることが経済的に理にかなっているとき、彼らは確かに銃の下にいるでしょう。



Mac用の10のウイルス対策ソフトウェアメーカーですか？



ここで少し困惑しています。 一方で、実際に存在しないものから身を守るために、苦労して稼いだお金をプログラムに費やします。 ただし、一方で、次の2つの利点があります。a）適切なプログラムを使用すると、愚かなユーザーから別のユーザーへのWindowsウイルスの感染を回避できます。 b）Macの本当のウイルスがようやく現れたとき、ウイルス定義データベースを定期的に更新し、会社がこのウイルスの更新をリリースしていれば、攻撃を撃退する既製のインフラストラクチャが既にあります。



どうして？ ウィルスソフトウェアはありませんが、念のため毎日バックアップを行っています。
