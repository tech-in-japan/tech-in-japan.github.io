GitHub Reflog v1.7.19

GitHubリポジトリとコミュニティアクティビティの記録であるGitHub Reflogへようこそ。 以前の問題はReflogアーカイブ 、 Reflog のロシア語版のアーカイブの問題の翻訳で利用可能です。



今週のGitHubでは、「 The League of Moveable Type 」-すばらしいオープンフォントを心から歓迎します。



推奨される週刊リポジトリ



ryanmcgrath / wii-js
最も人気のある任天堂WiiゲームコンソールにはWebブラウザーが組み込まれていますが、何らかの理由で進行中の開発はほとんどありません。 このJavaScriptプロジェクトは、WiiとHTMLの相互作用のために、安定した、信頼できる、文書化されたAPIを提供します。



このプロジェクトは、実際のアプリケーションの開発には完全に実用的ではありませんが、子供向けプログラミングを教えるための優れたツールであると主張しています。



Wii-JSは、一度に1つのページで最大4つのwiimotesインタラクションをサポートします。 デモページにアクセスして（Wiiを使用）、動作を確認します。


今週の素晴らしいリポジトリ



danlucraft / git.js
これは本当に素晴らしいです。 純粋なJavaScriptで記述されたGit実装。 PythonのDulwichプロジェクトのように聞こえます。



Git.jsはMITでライセンスされています。 node.js用のコンソールgitクライアントと、HTTP経由でリポジトリを管理するためのブラウザー内APIの両方が含まれています。


素晴らしいリポジトリ

  • j2labs / brubeck ：Python用の柔軟なWebフレームワーク。Mongrel2 、Eventlet、 ZeroMQ 、およびDictShieldで実行されます。 それを評価するには、プロジェクトの例を含むページをご覧ください 。
  • arthuredelstein / clooj ：Clojure自体で書かれたClojure用の軽量IDE。 Swingを搭載。 Cloojはクロスプラットフォームであり、スタンドアロンアプリケーションとして、または組み込みエディターとして使用できます。
  • jceb / vim-orgmode ：この素晴らしいプラグインは、vimコミュニティにemacsのorg-modeノートシステムを提供します。 iPhoneのメモには、 webDAVまたはDropboxを介してmobileorgからアクセスすることもできます。
  • Khan / khan-mobile ：新しいKhan Academyモバイルアプリのソースコード。


有望なリポジトリ

  • urbancoding / jenx ：Jenkinsは、MacRubyで実行されているMac OS X用のサーバーモニターを構築します。 このアプリは、Jenkinsでのすべてのビルドのステータスを報告します。 クリックすると、ブラウザで作品が開きます。 Growlもサポートしています！
  • jesusabdullah / pyee ：人気のあるEvent.mitter node.jsシステムのPythonポート。 シンプルでわかりやすい。
  • memo / iSteveJobs ：この役に立たない（しかし面白い）プロジェクトは、Webカメラからのビデオストリームで見つかったすべての顔をSteve Jobsの顔に置き換えます。 C ++ openFrameworksフレームワークを搭載。
  • kennethkufluk / Twitter-Matrix ：このHTML5プロジェクトは、一般的なツイートのストリームを取得し、Matrixスタイルで表示します。 デモページにアクセスして、実際の動作を確認してください。
