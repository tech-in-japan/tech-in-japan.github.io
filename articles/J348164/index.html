優れた開発者、真のエンジニア、真のリーダー-RIP Shawn O. Pearce







2018年1月29日、有名なプログラマー、著者、コミッター、Git、Jgit、libgit、Gerrit Code Reviewなどの多くのプロジェクトの創設者であるSean Pierceが亡くなりました。







優れた開発者、真のエンジニア、優秀なリーダーが亡くなり、素晴らしい人、妻、そしてオープンソースコミュニティ全体が残りました。







ほぼ一年前、彼が肺癌と診断されたとき、彼はただ一つの要求だけを持ち、可能な限りのことをして、少年たちとサラを助けて、移行を助けました。







2006年にGitプロジェクトに参加し、それ以来、このプロジェクトの最も生産的な開発者の1人になりました。 彼の長所には、Git-over-HTTPプロトコル、高速インポートの概念（後に他の多くのバージョン管理システムで採用された）、reflogsの概念など、多くの基本的な機能が含まれます。

-彼も！







彼は最初のグラフィカルなGitクライアントの著者です（tcl / tkのgit-gui、それから初めて彼と仕事をする機会を得たのは、彼は本当に余分なクラスのプロです）。







Eclipse IDEは、JGitの作成時にGitを完全にサポートし、JavaでGitをゼロから書き直しました。







膨大な数の人々がgitを使用しています。これは本当に人気があります。 そして、これはショーンがその創造に手を携えていたからだということは絶対に確かです。 結局のところ、それなしでは、Gitの主流全体が完全に異なってしまうでしょう。







彼が開発者としてオープンソースコミュニティのためにしたことをすべてリストアップするのではなく、彼は特に優れたリーダーおよびオーガナイザーとして記憶されるでしょう。
