Pyastra-PICアーキテクチャ向けのアセンブラーへのPythonコードトランスレーター

Pyastraは、Pythonコードのアセンブラーへのトランスレーターです。 Pythonで記述されたソースファイルを受け入れ、コードにエラーが含まれていない場合は、お気に入りのPICアセンブラー（gpasm、mpasm、またはそれらと互換性のあるその他）を使用してHEXファイルにコンパイルできるアセンブラーファイルを生成します。











サポートされているマイクロコントローラーとプロセッサー



PIC12、PIC14、PIC16ファミリーのすべてのマイクロコントローラー。



例



ソースコード（ポートA、ピン0は無限に点滅）： 


 while(1): PORTA[0] = 1 for i in xrange(0,255): pass PORTA[0] = 0 for i in xrange(0,255): pass
