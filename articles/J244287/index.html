3 in 1：コードアワー、情報学の日、ボランティアの日

子供たちの「手の森」を見るために教育の小さな革命を作りたい場合、ロボットや熱狂的な悲鳴を再生するように目を光らせて、成功したアニメスプライトから目を向けるなら、彼らは私たちの学校の同じ子供たちです。 そして、彼らはあなたが来て彼らにコーディングの方法を教えるのを待っています。 1日、1レッスン、1時間。 とても簡単です。







本日は、 Code.orgプロジェクトのHour of Codeイニシアチブについてお話しし、ご参加ください。 さらに、今年、これはすべて州によって公式にサポートされています 。 特に、12月4日はわが国のコンピュータサイエンスの日であり、12月5日は国際ボランティアの日です。



だから。 Code Hourは、プログラミングトレーニングイベントの国際シリーズです。 私たち全員がこれに参加できます。 必要なのは、提案された方法論に従って、あなたの街の学校の1つでレッスンを実施するために、12月4日から12日までの1週間です。



読んで行動する！



まあ、またはhoc@codabra.orgまたはvk ＆ fbグループであなたの要望について書いてください。



ここに私たちが提供するものがあります。



1.計画、ビデオの例、配布資料など、1つのデモレッスンの実施方法に関する詳細な説明 。

Code.orgは、プログラミングを促進する今日最大の非営利組織の1つです。 2013年のプロジェクトは、谷の有名なビジネスエンジェルであるアリとハディパトロヴィによって設立されました。 現時点では、Microsoft、Google、Facebookなどの企業によって財政的にサポートされています（ 完全なリストはこちら ）。 このプロジェクトの目標は、代数、生物学、物理学など、学校のカリキュラムでプログラミングを同じ標準レッスンにすることです。 とりわけ、Code.orgは、ゲーム形式でプログラミングをプレイすることを子供に教えるための優れた無料のオンラインプラットフォームです。
