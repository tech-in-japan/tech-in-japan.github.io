Windowsのショートカット

このトピックのコメントからすべてのショートカットを収集しました。



1.フォルダまたはファイルを選択してF2を押すと、名前の編集が有効になります。エクスプローラーでF4を押すと、アドレスバーがアクティブになります

2.スクロールやPage Downキーではなく、「スペース」を押すだけでページを下にスクロールします。

3. Shift +スペース-戻る

4.すべてのアクションShift + ...は逆の順序で戻ります（Ctrl + Tab、Ctrl + Shift + Tab、Alt + Tab、Alt + Shift + Tab ...）

5.マウスを使用せずにこれらの同じ「タスクバーのウィンドウ」に入るには、Win + Tabの組み合わせを押すことができます



6. Win + Break-システムプロパティへのクイックアクセス

7. Ctrl + Shift + Esc-プロセスのリストへ

8.エラーウィンドウからメッセージをコピーするには、そのようなウィンドウがクラッシュしたときにCTRL + Cを押すだけです。

9. ctrl + esc-開始します

10. alt +スペース-アクティブなウィンドウのメニュー（閉じる、最小化する）

11.入力を開始してalt + breakを押すと、カーソルが消えるか点滅が止まります

12. Alt + Esc-ウィンドウの順序を変更します-現在のウ​​ィンドウがバックグラウンドに送信されます

13.ウィンドウは、左上隅のウィンドウアイコンをダブルクリックして閉じます。

14. [スタート]メニューをアクティブにすると、スクリーンセーバーが起動しません

15.「ctrl + tab」-要素のプロパティウィンドウのタブを切り替える

16. Win-Bトレイにアクセスするためのショートカット

17.すべてのウィンドウを最小化する-Win + M、Win-Dですべてのウィンドウを非表示にし、もう一度押すと表示

18. Win + f-検索

19. Win + L-ロック

20. Win + u-聴覚障害/視覚障害のある人向けの特別な機能

21. Win + e-コンピューター

22. Win + r-実行
