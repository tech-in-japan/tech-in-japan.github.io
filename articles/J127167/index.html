Apple Developers Community＃8。 iOsスクールの最初のリリース

サンクトペテルブルクでの9月8日19:00は、 Apple Developers Communityの 8回目の会議になります。



私たちの会議の目的は、すでにiPhoneとiPad向けに開発している人々の開発と、新しい開発者を引き付けることです。 その実装のために、理論的および実践的な知識の獲得を目的とした月例会議を収集します。



覚えているなら、6月に私たちはiOs開発者の無料の学校に人を集めることについて話しました。 夏が終わり、9月が来て、トレーニングは終わりに近づいています。 6人以上の開発者が、iPhone用のアプリケーションを開発する必要がある最終的な割り当てをすでに受け取っています。 私たちは、2つの主要なイニシアチブを分離すべきではないと判断し、8回目の会議と学校の卒業式を組み合わせました。



ADCSpb＃8では、既によく知られているレポートに加えて、プログラムのプレゼンテーションを含む同窓生プレゼンテーションがあります。



前回の会議でApp Storeでのアプリケーションのプロモーションについて説明した後、テクニカルレポートに戻ります。

  • DataArtのGrigory Buteykoが、新しい新しいオペレーティングシステムMac OS X 10.7 Lionの機能について説明します。
  • 次に 、 iBearSoftの Andrey Pilyuginが、世界中の人気のあるアプリケーションに導入している新しいテクノロジーについて説明します。
入場は無料です。 登録するだけです。

発表者



グリゴリー・ブテイコ


Gregoryは、 DataArtのモバイルアプリケーション開発の分野の専門家です 。 彼はすでにMicrosoftとSilver Bullet 、 高品質のシャドウを実装するための予期せぬ方法 、および機器なしでメモリを操作するというプレゼンテーションを行っています。 今回は、グレゴリーが新しいMac OS X 10.7 Lionオペレーティングシステムの機能について説明します。



アンドレイ・ピリギン


AndreyはiBearSoftの創設者です。 おそらく誰もが、Bills for iPadアプリがどのようにして米国の売上高でNo.



iOs 5のリリースまでに、Andreyは彼の製品の更新を準備しています。 ADCSpb＃8で、彼はMoneyを財務会計の一般的なアプリケーションとして使用して、導入される技術について話すことに同意しました。



iOs開発者の学校



夏休みには、iPhoneおよびiPad用の無料の開発者コースを開催しました。 7月9日から9月1日までに、8〜10人のグループがObjective-Cでプログラムすることを学んだはずです。



コースプログラムは次のとおりです。

  • 目的c
  • 基盤とメモリ管理
  • ビューとView Controller
  • Interface Builderでのインターフェースの作成
  • テーブルビュー
  • インターフェイス設計の原則（HIG）
  • Webサービスサーバー、JSONとの相互作用
  • 画像表示、ウェブ表示
  • アプリケーションデータストレージ
  • マルチスレッド
  • 便利なユーティリティ機器
トレーニングの最後に、各生徒は最終プロジェクトのタスクを受け取りました。iPhone用の独自のアプリケーションです。 9月2日、彼らの仕事の結果を見ることができます。 各学生は短いプレゼンテーションを準備し、プロジェクトとその開発について話します。



また、子供たちの教師であるスタニスラフクラスノヤロフと、主催会社であるe-Legionの CEOであるAlexander Zverevが出演します。 彼らは、学校とApple Developers Community全体の学習プロセスと計画について話します。
