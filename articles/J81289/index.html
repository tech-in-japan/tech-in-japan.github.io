GoogleツールバーなしでGoo.glを使用する







Googleリンク短縮サービスについて-Goo.gl は 複数 回 作成 さ れて います。

膨大な数のそのようなサービスが増えているという事実にもかかわらず、GoogleはそれがGoogleであるという点で正確に興味を持っている人々（私を含む）-安定性、信頼性、そして悪の企業への愛が良い=）

軟膏のフライは、GoogleツールバーとFeedburnerでのみサービスを使用できることが判明しました。 少し後に、Chromeのプラグインが登場しました。 しかし、他のブラウザーのユーザーは引き続き飛行中です。

「はい、できません！」と思い、兄に頼って、自分のプロプライエタリなソフトウェアなしで新しい機能を使用する方法を尋ねました。 そしてその答えはすぐに見つかりました 。



Goo.glブックマークレット

どのように機能しますか？

このコードを取ります：

javascript:window.googl_callback = function(response){if(response.error_message){alert("An error occured: " + response.error_message);}else{prompt('', response.short_url);}}; var s = document.createElement("script"); s.src = "http://ggl-shortener.appspot.com/?url=" + encodeURIComponent(window.location.href) + "&jsonp=googl_callback"; void(document.body.appendChild(s));
