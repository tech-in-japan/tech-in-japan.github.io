Windows 10はRaspberry Pi 2に搭載されています

記事は、私たちのお気に入りの著者の1人である本物のオタク、Scott Hanselman による記事の翻訳です。



私はRaspberry Piの誠実なファンです。 私は自宅で3つの作品を使用しています。1つはマルチメディアセンター用 、もう1つは3D印刷用、もう1つは再生用です。 Raspberry Pi 2 が 発表されたというニュースの後、今ではさらに良いことがあります。 35ドルで、ポケットに入れられるコンピューターが手に入ります。

  • 彼女はまだ小さいです！ Raspberry Pi B +サイズ 私のために働いたものは働き続けます。 ;）
  • HDMI！ イーサネット！ カメラの下のポート！
  • 電源用のマイクロUSB！


これで ...

  • クアッドコア900 MHz ARM Broadcom Cortex A7とチップ上のBCM2836-3〜6倍のパフォーマンス。 わあ
  • 1ギガバイトのRAM（GPUと共有）






一般に、私はRaspberry Piを別のデバイス（「専用デバイス」）として使用するのが好きです。

もちろん、汎用のコンピューターですが、非常に安価で強力なため、特定の1つのタスクに使用でき、完全に機能します。



たとえば、地下に何かが漏れているかどうかを追跡し、センサーが濡れたことを感知した場合に書き込みます。 子供向けのMinecraft 。 3D印刷用サーバー 。 小さなおもちゃのエミュレータ 。 オープンソースのメディアプレーヤー。

これで、 Windows 10を 実行 している Raspberry Pi 2で ユニバーサル Windows アプリケーション を 作成できるようになり ました 。

つまり -Windows 10 は Raspberry Pi 2 に 搭載されています



Raspberry Pi Foundationは Raspberry Pi 2を発表し、 Windows 10 はこれをサポートします。 そして、これらはすべて、IoT用のWindows開発者プログラムの下で無料で利用できるようになります（少し後で）。 マイクロソフトは昨年、 Intel GalileoでのIoTおよびWindowsサポートのためのWindows Developer Programを発表しました。 今日、WindowsはRPi2開発者により適しています。



これは、理論的には、ユニバーサルアプリケーションをSurface Pro 3およびWindows Phoneで実行できることを意味します。 そして、Raspberry Pi 2上。そして、Raspberry Pi 2は、このアプリケーションが動作する非常に独立したデバイスになる可能性があります。 これらはすべて、おなじみのコード、ツールで作成でき、同じ手法を使用できます。

開発の観点から、 ユニバーサルアプリケーションがWindows 10で使用する必要があることは明らかです。 Raspberry Pi 2、携帯電話、タブレット、Xbox、クラウド、および地球上での優れた開発経験、API、ツール、クロスコンパイルなど。 私はこれらすべてに非常にハッキングされています。
