新しいオンラインストアパートナーネットワーク管理モジュールがリリースされました-PHPShop Partners

PHPShop Enterpriseオンラインストア用の新しいモジュールをご紹介します。これにより、パートナーのネットワークを作成し、アフィリエイトプログラムのボーナスと割引の会計処理を自動化できます。



オンラインストアPHPShop Enterprise 3.5のプラットフォームのベストセラーの最新バージョンでは、PHPShopパートナーモジュールが提供されます。これにより、オンラインストアの所有者がアフィリエイトネットワークを作成し、多くの会計処理が自動的に行われます。



このモジュールは、アフィリエイトリンクでの顧客のクリックを考慮し、注文が正常に完了した場合、指定された支払い割合の形でパートナーにボーナスを加算します。



各パートナーに対して、ボーナス資金の蓄積が実行される個人アカウントと、支払済みおよび未支払の注文の管理を提供しています。 ボーナス額が引き出しに十分であると指定された数値に達するとすぐに、パートナーは資金の引き出しを申請できます。 また、パートナーのオフィスの[素材]タブでは、ボーナスを考慮するために、パートナーのWebサイトからオンラインストアのWebサイトに顧客をリダイレクトする方法を利用できます。



アフィリエイトリンク



パートナーには、ストアに製品を配置する方法がいくつかあります。 最も簡単な方法は、アフィリエイトリンクを使用することです。 



<a href="http:// /?partner=XXX">  !</a>


パートナーIDが含まれています。



JavaScriptを介したストアフロントの接続



別の方法は、Java Scriptを使用して製品のストアフロントをサイトに接続することです。 この方法では、商品の量と配置のカタログの設計と設定を考慮して、パートナーのサイトのどこにでも店頭の製品を表示できます。 この方法では、複数のディスプレイケースを同時に接続できます。 



 <script src="http:// /phpshop/modules/partner/lib/js/phpshop-partner-lib.js" id="phpshop-lib-xml"></script> <script> var PHPShopXmlManager1 = new PHPShopXmlManager(); PHPShopXmlManager1.url = 'http:// '; PHPShopXmlManager1.imgwidth = 100; PHPShopXmlManager1.partner = XXX; PHPShopXmlManager1.category = 11; PHPShopXmlManager1.limit = 5; PHPShopXmlManager1.load(); </script>




設定の説明：

  • .partner = XXX; -パートナーID
  • .category = 11; -商品の出力のカテゴリID。 このパラメーターを省略すると、特別オファーが表示されます。
  • .limit = 5; -商品の引き出しの制限


製品回収スキームは、カスタム製品回収テンプレートを使用してパーソナライズできます。



Shop4Partnerユーティリティ



パートナーのWebサイトに製品カタログを配置する最も簡単な方法の1つは、Shop4Partnerユーティリティを使用することです。 ストアフロントのHTMLコードを生成するには、数回クリックするだけで済み、パートナーサイトで簡単に使用できます。 このユーティリティは、パートナーの識別子を自動的に考慮し、いくつかのデザインテンプレートを備えています。







カタログモジュール



アフィリエイトプログラムで商品を表示するためのシステムとして、アフィリエイトプログラムを通じて商品をブロードキャストするためのカタログモジュールを備えた完全に機能する無料のサイト管理システムPHPShop Microを使用することもできます。



PHPShop Microは小規模なサイトに最適なソリューションです。データベースを必要とせず、ファイルのみで動作するため、無料のホスティングに適しています。 コンテンツ管理システムには120を超える独自のテーマがあり、Java Scriptを介した製品ウィンドウの表示をサポートしています。







PHPShop Microを使用して商品のブロードキャストを開始するには、 リンクからアーカイブをダウンロードし、解凍してパートナーのFTPサイトにアップロードし、カタログモジュールの商品のデザインと出力の設定を決定する必要があります。



カタログモジュールの構成は、phpshop / modules / catalog / inc / config.iniファイルにあります。

  • enabled = "1"; -モジュールがオン（0-オフ）
  • パートナー= "1"; -パートナーID
  • right = "11"; -右の列に出力するためのコンマで区切られたディレクトリID
  • 左= "6.3"; -左列に出力するためのコンマで区切られたディレクトリID
  • limit_left = 4; -左の列の製品の数
  • right_left = 5; -右の列の製品の数


カタログモジュールを有効にしたPHPShop.CMS Microの動作例： micro.phpshopcms.ru

Partnersモジュールwiki.phpshop.ru/index.php/Partner_Moduleの使用に関する詳細をご覧ください。
