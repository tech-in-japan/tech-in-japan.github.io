広告主が協力したい企業はどれですか：Digital Index 2015

AdIndexは、ロシアのPR、広告、デジタル代理店の調査を実施しました。 その結果、デジタルインデックス2015の評価が決まりました。基準は、顧客の間での代理店の人気、確認された協力の経験に基づくプレーヤーの市場シェア、および企業との協力の質と有効性に関する専門家評価であるとみなされました。



AdIndexスペシャリストは、広告主がさまざまなセグメントのステータスを評価する方法、彼らが主な問題と考えるもの、請負業者に最も頻繁に提示されるクレームを見つけました。 市場平均格付けは、推奨される請負業者（市場平均を超える格付けを受けた）のプールを形成するための基準になりました。



研究者はまた、デジタルエージェンシーのクライアントの調査を実施し、以下の質問に回答するよう招待しました。



•オンライン環境でキャンペーンの請負業者を選択する方法は？

•柔軟な価格で誰に行き、非の打ちどころのない顧客サービスのために誰に？

•最高の専門知識を持っているのは誰ですか？



アナリストは、12の試験で知識、協力、協力に対する満足度の評価を形成しました。



1.バナー広告を表示する

2.ビデオ広告を表示する

3.コンテンツ広告

4. SEO

5.デジタルクリエイティブサービス

6.ウェブ制作/サポート

7.モバイル制作

8.デジタルPR / SMM

9.リードジェネレーション

10.プログラムによる購入

11.ダイレクトマーケティング

12.モバイルマーケティング



AdIndexは、すべての市場参加者に対するランキングと公開性に対する非営利のアプローチに注目しています。 以前と同様に、評価は広告主の調査に基づいていました。 顧客は、さまざまな分野で知っているすべての請負業者に（ヒントを付けて）名前を付けるように求められました。オンラインメディア広告の購入（バナー、ポップアップ、その他の同様の形式）。 コンテキスト広告の配置（検索結果および専門リソースへの商用リンク）; インターネット上での広告キャンペーンの開発および実施など。 次に、過去3年間に協力した機関を示し、サービスの品質と価格設定ポリシーを評価する必要がありました。



以下の企業の専門家が研究に招待されました。



1.トップ1000のオンラインディスプレイ購入者

2.総広告費の上位1000（TNSモニタリングに基づくAdIndexの評価）

3.インターネット上に存在することは、間違いなく、適切なレベルのデジタル専門知識を証明している企業です。



最もアクセス可能で明白なメトリックはインターネット（サイト）のページであるため、問題のこちら側を調査し、特定の専門家を調査に招待することの妥当性について彼ら自身の意見が作成されました。 その結果、約400人の回答者を拒否する必要がありました。なぜなら、彼らのリーダーシップの下で行われた仕事のレベルまたは彼らの勢力は、彼らの票を市場にとって重要なものにしないからです。 その結果、3300人の専門家が研究に招待されたと AdIndexは書いています。



受け取った回答に基づいて、誘導された知識（ヒント付きの知識）の評価と協力が形成されました。 顧客の評価は、協力に対する満足度の評価（品質評価）の基礎となりました。 サービスを利用している専門家の少なくとも2％がサービスに関する意見を表明している機関が含まれています。



AdIndexが公開しているサイトは、サービスの質に関する一般的な評価のみを提供します。 知識とコラボレーションの評価は、12月1日火曜日に公開されます。
