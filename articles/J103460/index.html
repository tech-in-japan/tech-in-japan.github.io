東芝が最速のSDメモリを発売

企業は常に競争を続けており、生産性の高いガジェットをリリースしています。ユーザーが利用できるメモリの量やその他の利点が増えています。 文字通り毎週、次の会社が特定の分野で成功を収めたと聞いています。 たとえば、今日では、東芝が最速のSDHC UHS-Iフォーマットメモリチップをリリースしたことがわかりました。 このチップは、SDメモリカード3.0などのメモリカードのフォーマットとハードウェアレベルで互換性があります。 一般的に、開発者は、チップがデータの書き込み速度と読み取り速度の両方の面で世界最速であると主張しています。 これはもはやコンセプトや単一の開発ではなく、販売されたキャリアです。



現在、4、8、16ギガバイトの容量を持つ高速microSDHCが市場で入手可能です。 これらのメモリカードは、モバイルデバイス、ラップトップ、その他のデバイスの両方で使用できることは明らかです。 この形式のメモリカードの生産はすでにストリーム配信されているため、まもなくこれらすべてが届きます。 確かに、これはSDHC UHS-Iの販売が開始される今年の11月よりも早く発生することはありません。



同社は、SDHC UHS-Iカードへのデータ書き込みの最大速度は80 Mb / s、読み取り速度は95 Mb / sであると報告しています。 開発者は、このメモリカードに基づいて、SSDドライブを搭載したラップトップおよびネットブック用に設計された超高速ストレージメディアであるNANDチップを作成することも計画しています。



モバイルデバイス用に設計されたSDHC UHS-Iメモリカードは、兄（または姉？）よりも生産性が低くなります。 一般に、モバイルバージョンのメモリカードの書き込み速度は20 Mb / sに加えて40 Mb / sの読み取りです。 開発者によると、このようなメモリカードは、HD品質でビデオを録画し、高解像度の写真や生産的なメモリを必要とするその他のタスクを記録するのに最適です。



開発者が実際にこのテクノロジーに基づいてSSDを作成した場合、このようなメモリモジュールは生産性の高いネットブックを作成するのに非常に便利です。 ところで、東芝はまた、そのようなチップに基づいて、11月に市場でリリースされたメモリの量を超えるかなりの量のメモリを備えたドライブを作成することが可能であると主張しています。
