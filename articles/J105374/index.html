3つのステップで起動可能なフラッシュドライブ

最初の投稿の公開から1ヶ月以上が経過しました。 住民のアドバイスに耳を傾けた私たちは、マスターを作成してブート可能なフラッシュドライブを作成することにしました。









インターネットを探索すると、次の結果が得られました。 ブータブルフラッシュドライブを作成するのはそれほど難しい作業ではありませんが、マニュアルを読むのに時間を無駄にする必要があります。 一般に、起動可能なフラッシュドライブを作成するには、プログラムアーカイブをダウンロードするように求められます。 ダウンロードされた「ハーベスター」は次のとおりです-バッチファイルは、閉じずに起動され、GUIアプリケーションを実行する必要があります。GUIアプリケーションを閉じずに、パラメーターを指定して実行可能ファイルを実行し、再び最初のコンソールウィンドウなどに戻る必要があります。 ユーザーが必要なアプリケーションを順番に起動するためのメニュー項目として単純に設計されたソリューションがあります。 この場合、一般的なケースで起動可能なフラッシュドライブを作成するアルゴリズムは非常に簡単です-



1.フラッシュドライブにアクティブパーティションを作成します。

2.フォーマット。

3.フラッシュドライブMBRおよびBRへのインストール。

4. Windows 7ディストリビューションのフラッシュドライブにコピーします。



ユーザーがダウンロードした「ハーベスター」では、ユーティリティが各ステップを決定します。 C ++、Delphiなどの言語は存在しないようですが、シェル、スクリプト言語、既製のアプリケーションのみがあります。 したがって、ギャップを埋めることを決定し、ウィザードをMultiSetに追加しました。これにより、3ステップでWindowsの自動インストールが可能な起動可能なUSBフラッシュドライブが作成されます。









ウィザードは、Windows PE、Vista、Windows 7でフラッシュドライブを作成できますが、Windows XPではさらに複雑になりますが、Windows XPで起動可能なフラッシュドライブを作成する機能を追加する予定です。



MultiSetをダウンロードして、起動可能なUSBフラッシュドライブを作成します： multiset.exe （6.5 Mb）
