Qt SDK v1.1.3リリース

Qt SDKのバージョン1.1.3へのアップデートをお知らせします。



このバージョンには、SDKの内部の変更に一部起因する、かなりの数の更新がありますが、機能と品質の変更もあります。 主なものは次のとおりです。
  • Qt Creator 2.3：多くの小さな機能強化。 詳細については、 ブログリリースのお知らせをご覧ください。
  • Qt Simulator 1.2：ユーザーインターフェイスの大幅な改善に加えて、Qt Simulator 1.2には、センサーシミュレーション、NFCタグシミュレーション、ジェスチャーシミュレーションなどのいくつかの新機能があります。
  • 通知API 1.1：QMLバインディングとパフォーマンスの改善。
  • デスクトップアプリケーション開発者向けのQt 4.7.4。
  • MeeGo 1.2 Harmattanベータ：このバージョンのHarmattanは、6月バージョンで使用されたのと同じイメージに基づいて構築されており、新しい機能は含まれていませんが、MeeGo 1.2 Harmattanアプリケーションの開発を継続するには更新が必要です。
  • Symbian Complementary Packageの更新：最新のSymbian Belleデバイスをサポートするための重要なCODA更新。 Symbian Belleを搭載したデバイス用のベータ版で作成されたアプリケーションは、まだOviStoreで表示できないことに注意してください。
  • SymbianのQt Quick Component Updateには機能的な変更は含まれていません。
Qt SDKが既にインストールされている場合は、コンピューターのQt SDKアプリケーションディレクトリからUpdate Qt SDKを実行することにより、最新バージョンにアップグレードできます。 Qt SDKを使い始めたい場合は、以下のインストーラーを使用して1.1.3をダウンロードできます。
  • Microsoft Windows-オンラインインストーラー：15MB- オフラインインストーラー：1.4GB
  • Linux 32ビット - オンラインインストーラー：23MB- オフラインインストーラー：1.2GB
  • Linux 64ビット - オンラインインストーラー：23MB- オフラインインストーラー：1.2GB
  • Mac OS-X- オンライン インストーラー：12MB-オフラインインストーラー：1.2GB
Nokia Developerは、 Qt SDKサイトで更新されたインストーラーも提供しています。



問題がある場合は、バグレポートをhttp://bugreports.qt.nokia.comに送信してください。
