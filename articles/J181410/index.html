PostPCの世界に生命はありますか？ パート5：Evernoteの代替品を探す

2013年3月以降、デスクトップシステムの使用を拒否し、タブレットに完全に切り替えたことを思い出させてください。 最初のテストはiPad 4（他のデバイスとプラットフォームが並んで待機している）で、ジャーナリストの日々のタスクを大型コンピューターほど悪くないものに解決しました。 残念ながら、コード、アダプター、その他のアクセサリー（iPadと周辺機器の相互作用、およびデータのエクスポートとインポートをテストすることになっていた）はまだ届いていないので、少し気を散らさなければなりません。 今日は、Evernoteデジタルノートブック（存在する場合）の適切な代替品を探しています。







予選ラウンド

人気のあるサービスで自分に合わないものを簡単に作成してみます。 Evernoteクライアントインターフェースは直観的であり、最も重要なのは、プログラムがユーザーがデジタルメモ帳で必要とするすべての機能を備えていることです。 これが問題です。製品が冗長になり、ある種の信じられないほどの組み合わせに変わり始めます。 一方、通常のiOSノートでは十分に機能しません。







先週末、約30個のアプリケーションをテストし、そのうちのいくつかを条件付きで有効であると認識しました。 次のプログラムは、予選ラウンドを通過しました： Journals 、 Catch 、 OneNote 、 Outline + 、 Awesome Note HD 、 Springpad 、 Note Taker HD 、 Notability 、 NotesPlus 、 Noteshelf 、 GoodNotes 、 Moleskine Journal 、 NoteMaster for iPadおよびTopNotes Pro 。 Google Keepサービスは、AppStoreに通常のクライアントがなく、Google Keep HD + （広告付きの無料バージョンもあります）のひどいものであり、間違った向きの特殊なブラウザーであるため、いくつかの熟慮の後に破棄されました（アプリケーション画面は位置に関係なく垂直に展開されます）タブレット）。 Notability、NotesPlus、Noteshelf、GoodNotes、TopNotes Proは、iPhone / iPod Touchのサポートの欠如により切断され、Note Taker HDはクラウドサービス（iTunes共有のみがサポートされます）を介したノートの通常の同期が不可能であり、インターフェイスが貧弱なため、距離を失いました。







リストされている他のプログラム（それらの技術的特性はネットワーク上で簡単に見つけることができます ）はiOS向けの最高のデジタルノートブックと呼ばれ、それらの選択はユーザーの主観的な好みによってのみ決まります。 ジャーナル、OneNote、Catchが気に入りました。これらは上位3位に入りました（Evernoteは明らかな理由でリングから外れています）。



Awesome Note HDは、候補者のリストにほぼ入りましたが、最後の瞬間、彼はスポーツマンらしくない行動で失格となりました。 このプログラムは、iPhone / iPod TouchとiPad、およびGalaxy Note 8のユーザーが利用できます。 つまり、Android向けのAwesome Noteアセンブリは存在しますが、Google Playを通じて配布されるのではなく、特定のモデルのデバイスでのみ配信されます。 もちろんうんざりしますが、商業的な観点からは正当化されます。Android用のソフトウェアを販売してまともなお金を稼ぐことができた場合は、指で数えることができます。グリーンロボットのファンは、海賊版ソフトウェアと他の収益化スキームを積極的にインストールします（サービスへのサブスクリプション、広告など）。



ファイナル

ジャーナル

このアプリケーションは、評価やレビューには表示されません。AppZappを介した次の景品の検索中に発見されました。











ジャーナルの費用は0から169ルーブル（66ルーブル以上の支払いはお勧めしません）ですが、開発者はiPad / iPhone用のユニバーサルIPAパッケージを作成しました。 このプログラムを使用すると、チャプターに分割された教科書を作成できます。そのページには、写真、サウンドクリップ、ビデオ、およびタッチスクリーンで作成した図面を含めることができます。







リスト、タグ、またはリマインダーはありません-アプリケーションは非常にシンプルですが、美しいです。 迅速に機能し、Dropboxを介してデータを同期できます（別のデジタルノートブックサービスに料金を支払う必要はありません）。 JournalsプログラムはiOSでのみ使用できますが、残念ながら、その中には検索がありません-大量のデータを忘れる必要があります。 さらに、ページに関連付けられた写真は、タブレットの位置に関係なく、垂直（縦）方向に表示されます。 これはノートブックで作成された図面には当てはまらず、開発者が時間の経過とともに問題を解決することを願っています。



Microsoft OneNote

iOS用のOneNoteクライアントは 、SkyDriveサービスで動作し、以前のバージョンとは異なり、通常のリストと制御（チェックリスト）リスト、およびインターフェイスの完全なロシア化が存在します。 また、欠点もあります：オーディオ録音を行う機能がないこと（少なくともiOSのバージョンでは見つかりませんでした）-写真のみがメモに含まれています（残念ながら、画像とビデオもサポートされていません）。 このアプリケーションは無料で、最大500件のメモを作成できます。この制限に達すると、表示、削除、同期のみが可能になります。 無制限のバージョンにアップグレードするには、479ルーブルかかります。







iOS、Android、およびWindows Phone 7向けのOneNoteビルドがあります。MicrosoftOfficeにはデスクトッププログラム（iOS用クライアントよりも機能的）があり、さらに、ユーザーはWebアプリケーションを使用できます。 ちなみに、タグをメモに割り当てたり、他のユーザーにノートブックへのアクセスを開いたり、テーブルなどのモバイルプログラムでサポートされていないオブジェクトを操作したりすることができます（iPad上のアプリケーションはそれらを表示しますが、編集は許可しません）。







その他の制限があります。OneNotefor iOSでは、新しいノートブックを作成できず、既存のノートブック内にセクションを作成することもできなかったため、OneNote Web Appを使用して作成する必要がありました。 一般に、この製品はまだ湿っていますが、希望する人は、479ルーブルの代替のアウトライン+アプリケーションを購入できます（iPadのみ、 無料の試用版が利用可能です）。 OneNote 2010形式のノートブックを編集し、SkyDrive、Dropbox、Box、またはiTunesを介して記録を同期できます。







キャッチ

機能の最後のファイナリストは、Evernoteに非常に似ており、写真（Evernote自体のようにビデオはサポートされていません）、オーディオレコーディング、リマインダー、タグおよびジオタグを割り当てることができます。 同期には専用のサービスが使用されます（キャッチはパブリッククラウドでは機能しません）：ユーザーは無料で、デフォルトのMy Ideasに加えて、5つのノートブック（スペース）のみを作成でき、250 MBがデータストレージに割り当てられます。



一度に33ルーブルを支払う必要がある「スペース」を1つ追加すると、4つのパッケージには99ルーブルの費用がかかります。 プレミアム料金プランがあります。 Catch Proの費用は月額4.99ドルまたは年額44.99ドルです。このお金で、ユーザーは50のノートブック、投資用の2.5 GB、ノートにオフィスドキュメントを含めることができます。 Premierオプションは月額$ 15.99（その年の料金$ 144.99をすぐに支払う場合）で、200のノートブックを作成でき、使用可能なディスク容量は10 GBに増えます。







ここにはチェックリストがありますが、それらは個別のオブジェクトとしてのみ作成されます-通常のメモに挿入することはできません。







Catchにはチームワークの機会もあります。追加の「スペース」からノートを共有できます（ノート「マイアイデア」は常に非公開です）。 クライアントプログラムはiOSおよびAndroidで使用できます 。他のOSのユーザーはWebインターフェイスを介して作業する必要があります 。







オフラインモード

私にとっては、インターネットに接続せずにノートブックで作業できることが重要なので、それについては別に書きます。 最終的に到達した製品は、クラウドを介して別のデバイスで作成または変更されたコンテンツを同期し、ネットワークに接続しなくても編集できるようにします。 このジャーナル、OneNote、およびCatchはEvernoteと比較して有利です。Evernoteでは、完全なオフラインモード（より正確には、タブレットで開かずにサーバーからノートのコンテンツをダウンロードする機能）はプレミアムサブスクリプションベースでのみ利用できます。 残念ながら、低速のモバイルチャネルでは、特にOneNoteで同期の問題が引き続き観察されます。







まとめ

ファイナリストを表彰台に立て、Evernoteの代替品が自然に存在するかどうかを判断することは私たちに残っています。 Journalsは、検索の欠如により3位の銅メダルを獲得します-それ以外の場合、プログラムは優れており、大量のメモを必要としない場合、通常のiOSアプリケーションに代わる優れた選択肢になります。 2位はMicrosoft OneNoteです。この製品はまだ湿っています（十分なタグがなく、音声を録音する機能もありません）。 彼はCatchコンテストで優勝しました。これは、よりシンプルなインターフェースを備えたEvernoteのほぼ完全な機能的アナログです。 また、不適格なAwesome Noteアプリケーションの高品質にも注意したい：イデオロギー上の理由で拒否しましたが、iPhone / iPod Touch、iPad、Galaxy Note 8（他のAndroidタブレットはサポートされていません）の多くのユーザーはこの製品を気に入っています。



結論

Evernoteを交換できますか？ はい、いいえ。 条件付きで適切なアプリケーションのリストに含まれるアプリケーションはすべて適切です-それはすべてあなたの好みに依存します。 私にとっては、代替手段はまだ見つかっていません。Catchを除き、同様の機能と同じ価格設定ポリシー（Proアカウントで年間45ドル）を使用しているため、石鹸の価格を変更しても意味がありません。 さらに、象アプリはより多くのプラットフォームで利用できます。
