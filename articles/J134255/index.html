xBounds-Android向けワイヤレスビデオカード

すべてのhabrazhitelへのご挨拶！

今日、私は素晴らしいデバイスに関する情報をあなたと共有したいと思います-さまざまなモバイルガジェット用のワイヤレスオーディオ/ビデオカード（デバイスはAndroidとNokia N900でサポートされます）は、テレビまたはプロジェクターにワイヤレス信号をブロードキャストします：



（dreamchip.deからの写真）



そのため、質問の範囲の歴史について詳しく知りたい場合は、 こちらをご覧ください 。



そして、すでに利用可能な情報をいくつかの簡単な要約とともにリストします。

  • アイデア*境界は、Nokia N770 / N800 / N810の「タブレット」に最初に適用されました。
  • 今年、再生が行われました。最初の商用サンプルは、11月の終わりにDroidconNLでリリースされました（詳細については上記のリンクを参照してください）。
  • デバイスのサイズはコンパクトで、HDMI入力に簡単に直接接続できます。
  • xBounds電源ケーブルの利用可能なデモンストレーションのいずれにも気付いていないので、受信機にはすでにバッテリーが内蔵されていると仮定します。
  • ワイヤレス通信の場合、特別な送信機は必要ありません。スティックとソフトウェアだけが必要です。
  • デバイスは、Wi-Fi、USB、さらにはBluetooth経由でも信号を受信できます！
  • 転送すると、ベクターの再レンダリングが行われます-スマートフォンとテレビの解像度の違いにもかかわらず、despiteいピクセルは表示されません。 巨大なFullHDスクリーンでもすべてがスムーズに表示されます！
  • このテクノロジーは完全にソフトウェアであり、ポータブルです。将来的には「GPUスティック」さえ必要ありません。xBoundsアプリケーションをスマートテレビ/セットトップボックスで実行するだけです。
  • 2つの解像度-スマートフォンとテレビ（H.264圧縮は使用されていないため、遅延はIntelのWiDiテクノロジーよりも小さい）で同時にレンダリングされますが、非常に小さな遅延と、それを減らすための努力がまだ続いています。
  • 複数のモバイルデバイスから1つのテレビ（分割画面）への同時ブロードキャストをサポートします。
  • xBoundsソフトウェアクライアントは、Androidだけでなく、モバイルOSとデスクトップOSの両方に簡単に実装できます。AppleのATV / AirPlayには、競争相手がいます。
