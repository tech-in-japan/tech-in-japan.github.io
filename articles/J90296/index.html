SAP StreamWorkとEvernoteを使用したアイデアからソリューションまで



チームミーティング、参加者の活発な活動、多くのアイデア、すべてがうまくいきますが、最終的に行われた決定は実際には実行されず、無駄な時間しか後悔することはできません。



問題は、多くの場合、これらのチーム会議は従業員間の相互作用とコミュニケーションの重要なポイントであり、現実の世界でこれらの会議の間に生じるすべてのアイデアや変化を常に考慮しているわけではないということです。 これで、EvernoteとSAP StreamWorkを使用して、チームがアイデアからその実現に移行しやすくなります。



統合

SAP StreamWorkを使用すると、収集した情報をまとめて収集し、ブレインストーミング、コラボレーション、意思決定に使用できます 。 インスピレーションが思いがけないものになる可能性があるため、SAPはEvernoteをアイデア収集の手段として使用しています。 Evernote APIのおかげで、 Evernoteの SAP StreamWorkアプリケーションへの統合が可能になり、ユーザーはメモを取り、音声録音を指示し、WebページをEvernoteに保存し、その後StreamWorkインターフェースですべてのメモを表示できます。



ビジネスアプリケーション

これは、ビジネス指向のサービスとの統合が初めての経験であるため、この統合に特に満足しています。 そして、これは驚くべきことです。私たちは、ユーザーが個人的な生活の重要な瞬間を思い出すのを支援するツールとしてEvernoteを作成しましたが、発売以来、ビジネスソリューションとの統合に対する無数のリクエストを受け取りました。 これは最初の経験であり、他の経験はすぐに期待されています。



ストレスが少ない

エキサイティングなプロジェクトに取り組んでいるとき、あなたはそれについて考えることをやめることはできません。 オフィス、店舗、フィットネスクラブでは、プロジェクトをより良くする方法について皆さんが考え続けています。 以前は、95％の確率で頭に浮かんだアイデアが忘れられることを知っていたため、そのような瞬間には一定の不安感が残りました。 これはもっと起こります。 携帯電話に乗って、Evernoteでテキストまたは音声メモを作成するだけで十分です。 統合のおかげで、このレコードはすぐにStreamWorkで利用可能になり、同僚を待機します。



仕組み

StreamWorkで、「追加」>「コレクション」>「アイテムを追加」>「Evernote」を選択します。 Evernoteアカウントにログインするよう求められます。 そこでノートブックを選択し、メモを表示してからプロジェクトに追加できます。 その後、同僚はメモを表示し、コメントを作成し、意思決定プロセスで使用できるようになります。 とても快適です。



試して

SAP StreamWorkは無料で利用できます。 アカウントを登録します 。



Evernote API

Evernote APIを使用すると、開発者はEvernoteと対話するデスクトップ、モバイル、およびオンラインアプリケーションを作成できます。 すでにリリースされたアプリケーションは数十あり 、さらに数千のアプリケーションが開発中です。 今日の開発でAPIを使用してください！
