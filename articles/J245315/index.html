PowerPointで直接投票する方法は？



PowerPointプレゼンテーションを話す多くのスピーカーは、聞き手に特定の質問をしたり、調査を実施したり、調査を実施したりするのが良いと考えています。 この場合、結果はプレゼンテーションの有機的な部分であることが望ましいです。



これらの目的のために、 SMS投票サービスを変更することにしました 。









必要な投票の質問のテキストと、この投票の結果を表形式で取得できる単純なhtml-requestを作成しました。

このhtmlテーブルは、Excelのデータソースとして機能します。 接続は、Webデータ接続を使用して行われます。 次に、標準のExcelツールを使用して、PowerPointなどのExcelグラフとのリンクをサポートする任意のプログラムにインポート（リンクとして貼り付け）できるグラフが作成されます。 この参照Excelドキュメントは既にテスト投票用に設定されています。



しかし、1つの問題が判明しました。 既定では、Excelでは1分あたり1回までソースからデータを更新できます。 VBAマクロを使用して、この制限は回避されました。

マクロテキスト 
Public dTime As Date Private Sub auto_open() Run "RefreshIt" End Sub Private Sub Workbook_BeforeClose(Cancel As Boolean) On Error Resume Next If Cancel = False Then Application.OnTime dTime, "RefreshIt", , False On Error GoTo 0 End Sub Sub RefreshIt() i = Int(Sheets(1).Range("H25").Value) If i > 0 Then On Error Resume Next Sheets(1).Range("A1").QueryTable.Refresh If i < 5 Then i = 5 If i < 10 Then dt = "00:00:0" & i Else dt = "00:00:" & i End If dTime = Time + TimeValue(dt) Application.OnTime dTime, "RefreshIt" End If End Sub




結果は投票結果を含むグラフであり、投票はほぼリアルタイムで利用可能になると更新されます。
