HDDからSSDへの切り替え：新しいラップトップのバッテリー寿命を延ばす時間の計算

私は古いHP TC4400タブレットを持っていますが、HDDからSSDに切り替えたときにどのようなバッテリー寿命の向上が得られるか興味がありました。



これらの計算は病院の平均気温と似ていますが、ある程度の価値もあります。







答えは私を驚かせました。 興味深い写真をご覧ください。







最大値は8.7％です。



（平均的な使用で、SSDは何も消費しないと仮定します。動作時間の相対的な増加の式は100/92です）。



もう1つの注意点：ここで、非定型ディスクを比較します。かなり大きなSSDとハイブリッドHDDですが、後者の場合、電力消費は通常のHDDとほぼ同じです。



通常の消費電力SSD = 0.150 * 0.2 + 0.1 * 0.8 = 0.11 Wを考慮に入れます

（20％が能動的に使用され、80％が受動的に使用されるため、SDD電力データ ）。



通常、電力消費量HDD = 0.05 * 2.2 + 0.15 * 1.1 + 0.8 * 0.8 = 0.915 W

（5％の検索、15％の作業、80％のアイドル状態、 したがってHDDのパワー ）



通常のSSDエネルギー消費を考慮すると、動作時間の増加は100 /（92 + 8 * 0.11 / 0.915）= 7.57％になります



したがって、ラップトップがバッテリーで3時間動作する場合、SSDを使用するとさらに13.5分の作業ができます。



計算の純度については、ラップトップでの実際の作業は、SSDがより速く動作し、作業自体の効率が向上するため、約20〜30分長くなると言わなければなりません。



PSもちろん、2005年から私が依存している分析情報（リンク付きの写真）ですが、ラップトップが新しいものではない場合（新しいラップトップのコンポーネント間のエネルギー分布は不明ですが、 基本的な変更は発生していないと思います）、Atomプラットフォームではなく、その後、おそらく私の計算が行います。
