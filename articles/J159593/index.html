Windowsファミリーは27歳になります

今日、Windowsはかなり重要なイベントになりました-このオペレーティングシステムファミリは27年目を迎えました。 このOSの最初のバージョンは1985年11月21日にリリースされました。 それからマイクロソフトの歴史は始まったばかりです。 若いスティーブ・バルマーを見ることができる時代のプロモーションビデオはこちらです。







Windows 1.0のシステム要件は、現代の標準ではばかげていることは明らかです。 たとえば、このオペレーティングシステムの通常の操作では、256 KBのRAMで十分でした。 Windows 1.0のディストリビューションは、2枚のフロッピーディスクに収まります。 ただし、すでにその時点で、Windowsの通常の操作にはグラフィックアダプターが必要でした。 そして、いや、WindowsはいまだにWindowsのようではありませんでした。 使い慣れた、認識可能なインターフェイス要素が、OSの後続バージョンで追加されました。



また、Windows 1.0があまり普及していないことも注目に値します。 また、2番目のバージョンも需要に違いはありませんでした。 しかし、Windows 3.0とNTは、新しい「機能」と機能のおかげですでに広く普及しています。 しかし、もちろんWindows 95が最も人気があり、その後、Windowsファミリーの勝利の行進がオフィス、教育機関、ユーザーの家で始まりました。 失敗した瞬間がありましたが、それでも成功した瞬間がありました。 分析会社Net Applicationsによると、世界中のコンピューターの約70％がWindowsを実行しています。
