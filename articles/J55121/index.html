アプリケーション：find / Perl

perl -i -pe 's / Windows / Linux /;' テスト*


すべてのサブディレクトリで同じ操作を実行するには：

見つける。 -name '* .txt' -print | xargs perl -pi -e's / Windows / Linux / ig '* .txt

15.ワイヤレス速度制御



難易度：中

アプリケーション：iwconfig



無線受信機と送信機が通信できる速度は、信号の強度に依存します。 信号が劣化したときに通信を維持するには、データをより低いレートで送信する必要があります。 通常、無線は利用可能な信号を計算しようとし、最高の伝送速度を自動的に選択します。



信号レベルが不十分な干渉領域では、パケットが自発的に消失する可能性がありますが、受信機と送信機は常に速度を調整します。 アンテナを追加したり、信号を増幅するために機器を移動したりできない場合は、同期速度を下げる必要があります。 これは、再接続が少なくなることを意味し、フリップフロップ接続よりも大幅に高速化できます。 各ネットワークカードには、接続を確立する独自の方法があります。 Linuxでは、速度制限はiwconfigで設定できます。

iwconfig eth0レート2M



このコマンドにより、ネットワークカードは2Mbpsのみで同期します。 最大速度を設定することもできます。ネットワークカードは自動的に同期されますが、速度は遅くなりますが、速くなりません。

iwconfig eth0 rate 5.5M auto



autoディレクティブを使用すると、最大5.5Mbpsの任意の速度で作業できます。 速度制限を解除するには、単に次を入力します。

iwconfig eth0 rate auto



ネットワークカードは、11Mbpsの速度よりも1Mbpsの速度で長距離で動作できます。 1 Mbpsと11 Mbpsの違いは、Orinoccoカードでは12デシベルです。つまり、理論的には、最大作動距離を4倍に増やすことができ、伝送速度が低下します。

22.テキストファイルエンコーディングの変換



難易度：低い

アプリケーション：再コーディング



recodeは、さまざまなプラットフォームで作成されたテキストファイルの転送を簡素化する小さなユーティリティです。 主な理由の1つは、改行の違いです。 改行文字を使用するシステムもあれば、キャリッジリターンを使用するシステムもあれば、両方を使用するシステムもあります。 その結果、あるプラットフォームから別のプラットフォームにテキストを転送した場合、ファイルを開くと、多くの（または逆に、少数の）改行と多くの奇妙な文字が表示されます。



ユーティリティパラメータは少しわかりにくいですが、エイリアスを設定することでこれを回避できます。

エイリアスdos2unix = 'recode dos / CR-LF..l1'

エイリアスunix2win = 'recode l1..windows-1250'

エイリアスunix2dos = 'recode l1..dos / CR-LF'



多くのオプションがあります：recodeは一連のエンコーディング間でテキストを変換できます。
