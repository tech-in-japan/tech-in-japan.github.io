Acer Aspire One A110ネットブックのSSDの高速化







Acerは最近、このネットブックのSSDドライブ用の新しいファームウェアをリリースしました。 ご存知のように、このドライブは非常に低速であり、更新されたファームウェアは状況を改善するように設計されています。



私はまだそのようなモデルのネットブックを持っているので、すぐに動作中の新しいファームウェアをチェックすることにしました。



いいえ、今日は分解しません。 すべてが非常に簡単です。



注意！ ファームウェアはSamsung SSDでのみ機能します。 始める前に、Aspire One A110にそのようなドライブが装備されていることを確認してください。



また、SSD上のデータはフラッシュ後に失われます。 セーブ！



したがって、最初に準備する必要があります。 次のコンポーネントが必要です。



1. Samsung SSDを搭載したAcer Aspire One A110ネットブック

2. Windowsがインストールされているコンピューターを使用することをお勧めします（私の場合、VirtualBoxの下）。

3. USBフラッシュドライブ。

4.ブートディスクを作成するAcerユーティリティ 。

5.必要に応じて、 HP USB Disk Storage Format Tool 。



フラッシュドライブを準備するには、 AOA110SSDFWv1.exeを使用しますが、私の場合はこれでは不十分であり、ブート可能なフラッシュドライブを作成できるHPユーティリティから支援を求める必要がありました。



じゃあ フラッシュドライブから起動します。





ファームウェアバージョンが古いバージョンよりも新しいため、0を押します。 消去なし-方法はありません。





再び0。





さて、ここに他の選択肢はありません:)





8GB SSDがあります。 私はそれを押します。





更新プログラムのメインウィンドウで、 Enterキーを押して、大切な緑の碑文「 PASS 」が表示されるのを待ちます（ただし、「FAIL」がありましたが、「 D 」ボタンを押して再スキャンすることにしました）。



Linuxをフラッシュして復元した後、Firefoxがはるかに速く起動したことに気付きました。

また、「 hdparm -it / dev / sda 」の出力から、約40MB / sがわかります 。



一般に、システムは著しく高速に動作し始めました。 人気のあるHD TuneとCrystalDiskMarkでWindowsユーザーをテストすることを楽しみにしています。
