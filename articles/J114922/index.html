Visual StudioのC ++ / CLI IntelliSense

Visual C ++チームの開発者は、Visual Studio 2010のリリースで、C ++ / CLIプロジェクトのIntelliSenseツールチップが含まれなかった理由について考えを共有しました。



また、Visual Studio 2010 for C ++ / CLIの発信SP1パッケージには含まれません。これは、Intellisenseツールチップを実装するための内部メカニズムを書き換えるプロセスが原因です。 Visual Studio 2010では、VSの古いバージョンからファイルを転送するための大量の作業のため、古いメカニズムの.ncbファイルの通常の使用を中止することにしました。 そして、ツールチップの新しいバージョンの開発が開始されたため、そのような無意味さがあります。 ツールチップの新しいバージョンは、計画された多数の革新を含む不完全さのために含まれていませんでした。 そのため、IntelliSenseの現在のバージョンでは、メタデータのインポートは機能せず、ほとんど意味がありません。



したがって、Visual Studioの新しいバージョンでは、スマートツールチップの新しいメカニズムだけでなく、それに関連する多くの革新も見られるはずです。
