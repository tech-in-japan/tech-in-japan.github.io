現在のコスト環境-消費電力の監視



テレビはどのくらいの電力を消費しますか？それはより多くの電力を消費します-洗濯機または冷蔵庫、アパートでどのくらいの電気が使用されますか？ これらの質問はすべて、現在閉鎖されているGoogle PowerMeterサービスで以前に答えることができました。 ただし、オタクやホームエコノミストは絶望すべきではありません。Googleからのサービスが崩壊したにもかかわらず、エネルギー消費を監視する問題が続いているからです。 このレビューでは、 Current Cost EnviRデバイスを検討することを提案しています 。これにより、デバイスのディスプレイでキロワットの電力が「漏れる」だけでなく、 Current Cost NetSmart （以前のCurrent Cost Bridge）とともにmy.currentcostのオンラインエネルギー消費を監視できます.com











だから、このデバイスは何ができますか





  1. あなたの家/アパートでの特定の時間の電力消費量（ディスプレイ上で6秒ごとに更新）
  2. 消費電力を測定する非接触方式。 より正確には、交流と電圧は200Vから260Vの設定で設定されます。 電力は、製品ごとにデバイスで計算されます。
  3. 433 MHzの周波数の無線チャネルを介してセンサー端末からディスプレイ受信機にデータを送信するための無線方式。
  4. 1日/月あたりの消費電力の見積もり（1 kW / hの料金が設定されています）
  5. 電力消費のダイナミクス、つまり 家/アパートでエネルギー消費者を接続/切断するとき、デバイスはkWのジャンプを表示します。
  6. ディスプレイレシーバーの設置場所での現在の室温の監視。
  7. my.currentcost.comのエネルギー消費量と現在の室内温度のグラフ。 5分ごとに更新されます。 現在のコストのNetSmartアドオンが必要です。 また、グラフには、家の中のすべてのデバイスがスタンバイモードになっているときの最小エネルギー消費量（+リークがある場合）の線も示されています。
  8. 10分ごとに消費電力データをGoogle PowerMeterに転送します。 現在のコストのNetSmartアドオンが必要です。
  9. XML形式のデバイスからの着信データをPC上でさらに処理する可能性がある特別なUSBケーブルを介してPCに接続する機能。 利用可能なソフトウェアのリストはこちらから入手できます 。 このデータのパーサーハンドラーを自分で記述することもできます。








接続と使用手順





  1. ディスプレイレシーバーを接続して構成します。 日付/時刻、電源電圧、関税を設定します。
  2. 無線送信機をオンにして、受信機と同期します。
  3. 計器の直後に端子を相線に引っ掛け、トランスミッターに接続します。 受信機は、現在のエネルギー消費量をすぐに表示する必要があります。
  4. Web監視の場合は、特別なケーブルを介して現在のコストのNetSmartをディスプレイレシーバーとeth経由でホームルーター/スイッチに接続します。 NetSmartはIPとゲートウェイの設定を受け取るため、ローカルネットワークでDHCPを構成してアクティブにする必要があります。
  5. PCで監視する場合は、特別なUSBケーブルを使用してディスプレイレシーバーをPCに接続します。 注意！ ethとPCを同時に接続することはできません！








さらに興味のある人のみ：注文する場所と配達する方法





以前にQ＆Aで書いたように、これらのデバイスをロシアで家庭用に購入することはほとんど不可能です。 そのため、 イギリスのアマゾンでは、ロシアの送電網に最も近いパラメーターで、比較的許容できるコストで製品を見つけました（このデバイスは米国では見つかりませんでした）。 現在の費用EnviR Black-The Smart-R Monitorを48.60ポンドで購入し、 現在の費用「ブリッジ」デバイスをオンライン監視用に29.95ポンドで購入しました。



次の問題は配達です。 事実、Amazonはこのタイプの製品をロシアに配送せず、英国から商品を配送するために仲介業者のサービスを使用する必要がありました。 レビューを読んで、貧弱な選択に失望した後 、私はEbayTodayのサービスを使用しました 。これは、小額の料金（8.55ポンド）で製品を購入し、£15.95でロシアに配達しました（Amazonは2.50ポンドのリクエストで+ + 6.95 +保険に加入しました）。 合計費用は£112.5でした。 クレジットカードで事前に補充された携帯電話の残高から、資金の移動がより簡単かつ迅速になりました。 振込手数料はまったくかかりませんでした。 私の都市への配達には18日かかり、そのうち9日は英国からモスクワへ、9日はモスクワで税関、仕分け、そしてロシア郵便（〜600km）で私の都市へ配達されました。 受領時に追加の支払いは必要ありません。 その結果、私はこのボックスのようなものを手に入れました（画質については申し訳ありませんが、当時は携帯電話以外には何も手元にありませんでした）：





最後の問題は、ディスプレイとNetSmartにそれぞれ3.3vと5vの2つの電源を接続するためのヨーロッパのアダプターの選択でした（無線送信機は2つの1.5VタイプDバッテリーで動作します）。 実際、ほとんどの市販のユーロアダプターは適切ではありません。 地元のラジオ店で30ルーブルのEURO-2アダプターを購入しました。
