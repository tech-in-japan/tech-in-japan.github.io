モバイル端末向けのUbuntu 8.04



Canonicalは、Ubuntu MID Edition 8.04と呼ばれるモバイルインターネットデバイス（MID）用のLinuxディストリビューションの特別バージョンをリリースしました。



このバージョンは、Intel Atomハードウェアプラットフォーム向けに最適化されており、 Intel Moblinプロジェクトの一部として作成され、コンパクトなインターネット端末用に特別に設計されたオープンアプリケーションの全セットも含まれています。



Ubuntu 8.04 MID Editionシステムインターフェイスは、スタイラスを使用せずに指を制御するように設計されています。 興味深いことに、開発者は、マルチタッチ入力（マルチタッチ）のサポートについて一言も言いません。 さらに、Ubuntu 8.04 MID Editionは、HTML、Flash、クラッター、Python / GTK、C / C ++ / GTK、およびJavaを使用して、システムの外観と動作を完全に変更する機能をOEMに提供します。 Canonicalは、これらのメーカーがカスタムエンジニアリングプログラムで協力することを推奨しています。



Web 2.0レベルのWebアプリケーションの操作と使用の基盤を形成する最新のインターネット技術の完全サポートに加えて、Ubuntu 8.04 MID Editionシステムには、WebカメラとGPSナビゲーション機器のサポートが組み込まれています。 このディストリビューションのメインブラウザーはGeckoテクノロジーに基づいており、小さな画面用に特別に最適化されています。



Ubuntu 8.04 MID Editionに含まれるアプリケーションには、クイックメッセージング用のPidginパッケージ、Webカメラを操作するためのGNOME Cheeseパッケージ、デフォルトの電子メールプログラムとしてのClaws Mailパッケージ、および音楽とビデオを再生するためのMoblin Mediaパッケージがあります。 ユーザーには、NeverballやFrozen Bubbleなど、いくつかのゲームも提供されます。 MIDやUMPC（Ultra-Mobile Personal Computer）などのコンピューターで本を読む場合、Ubuntu MIDシステムにはFBReaderプログラムが含まれています。 デザートには、目覚まし時計、電卓、アドレス帳などの便利な小物がユーザーに提供されます。



Ubuntu 8.04 MID Editionは、Linuxカーネルバージョン2.6.24およびX Serverバージョン1.4.1プレリリースのグラフィカルユーザーインターフェイスに基づいています。 開発者のWebサイトで配布をダウンロードできます。
