Nokia X2：次世代Xシリーズの紹介

今日、Microsoft Mobile OY（旧Nokia Devices＆Services）は、第2世代のNokia Xライン、Nokia X2モデル、および改良されたプラットフォームインターフェイス（Android Open Source Projectに基づく）を導入しました。







手頃な価格（推定コスト-税金と手数料を除く99ユーロ）で、新製品は第1世代のモデルと比較して、ハードウェア、設計、ソフトウェア、およびサービスに多くの改良が加えられています。



「鉄」



技術コンポーネントの観点から見ると、Nokia X2は第1世代のNokia Xスマートフォンの進化的開発の結果です。



1.2 GHzのクロック周波数を備えた、より強力なデュアルコアQualcomm Snapdragon 200プロセッサーは、1 GHz Snapdragon S4に取って代わりました。 RAMの量が1 GBに増えました。 ノベルティは4 GBのドライブを内蔵しており、最大32 GBのmicroSDカードもサポートしています。







容量性IPSディスプレイのサイズは、Nokia Xと比較して0.3インチ増加し、現在では4.3インチと非常に印象的です。 前述のように、ディスプレイの解像度は217 ppiの密度で800 x 480ピクセルです。 Xラインで初めて、多くのLumiaスマートフォンで知られるNokiaのClearBlackテクノロジーが適用され、明るい光の下でも最もコントラストの高い画像を提供します。







メインカメラは5メガピクセルになり、オートフォーカスとフラッシュを備えています。 また、他のインスタントメッセンジャーには、Skypeおよびビデオコール用のフロントカメラがありました。 Nokia X2は、Bluetooth 4.0、Wi-Fi 802.11b / g / n、GPS / GLONASSなどの標準をサポートしています。



1800 mAhのバッテリーは、2つのアクティブなSIMカードで最大10時間（2G）と13時間（3G）の通話時間、および最大23日間のスタンバイ時間を提供します。



デバイスの寸法は121.7 x 68.3 x 11.1 mm、重量-150グラムです。



プラットフォームとサービス







Android Open Source Projectに基づくNokia Xプラットフォームのインターフェースは、大幅に再設計されました。 2つの画面と1つのボタンのインターフェース図の代わりに、Nokia X2には3つの画面と2つのホームボタンと戻るボタンがあり、後者ではタスクをすばやく切り替えることができます。







3画面インターフェイスのレイアウトは次のとおりです。

-調整可能なサイズのタイルを備えたメイン画面（Lumiaスマートフォンのメイン画面の類似物。ユーザーは、最も頻繁に使用されるアプリケーションを取り出すことができます）。

-Fastlane画面の右側の「スワイプ」と呼ばれ、最近使用したアプリケーション（ワンタッチで応答する機能など）およびイベントにすばやくアクセスできます。







-Lumiaスマートフォンのように、アルファベット順にソートされたアプリケーションの完全なリストを左の「スワイプ」で呼び出します。



スタート画面では、色ごとにタイルを整理する機能など、より多くのパーソナライズオプションが利用できるようになりました。



Nokia X2には、Outlook.com、Skype、OneDrive c（！）15ギガバイトなどのMicrosoftサービスがデフォルトでプレインストールされており、クラウドに無料でファイルストレージが保存されます。 さらに、OneNoteおよびYammerサービスがNokia Storeアプリストアで利用できるようになりました。



ロシアでは、Nokia X2もYandexとの友達であり続けます。 検索、Yandex.Browser、Yandex.Maps、およびYandex.StoreアプリケーションストアのYandexキーサービスがスマートフォンにプリインストールされることが予想されます。



設計



Nokia X2には、明るくスタイリッシュなケースがあります。外側は、2つのタッチボタンを備えた黒いパネルで覆われ、背面と側面は交換可能なモノリシックカバーに完全に埋め込まれています。 ケースの右側には、ロック/電源ボタンと音量ロッカーがあり、3.5 mmオーディオ出力とMicro USBが上部にあります。







一般に、デバイスの設計はNokia Xから継承されましたが、第2世代では、透明な上部と明るい内部層を持つ新しい2層構造になりました。 結果は、かなり珍しい「深さの影響」です。 最初は、モデルは光沢のある緑、オレンジ、黒のボディカラーで利用できます。 光沢のある黄色と白、およびマットグレーは、少し後にパレットを補完します。
