チョビカムレビュー-世界最小のカメラ

最近では、Habréで小型カメラ Chobi Camの発表がありました 。 これは、ワウ効果に負けて、説明したデバイスをすぐに注文した数少ないケースの1つでした。 そして今、私は上記のデバイスのレビューを公開します。





パッキング



小包を開けた後、カメラが金属製の箱で出荷され、多くの場合中国製品のように斑点のあるポリエチレンに包まれていないことに驚きました。 それは非常に便利です。デバイスの安全性について心配する必要はありません。







メーカーは、言葉ではなくカメラモードで動作する可能性について、HDビデオを記録する可能性に焦点を当てています...







箱の中には、デバイスが入ったプラスチックのパッケージがあります。







プラスチックはまあまあ脆いように見えます。 外側はベルベットのコーティングのように見えます。







プラスチックカバーの下には、アクセサリ用の別のコンパートメントがあります。







キット内のデバイスには、必要なすべての有用なものがまだ残っていることがわかります（後で詳しく説明します）。







鉄



ケースは光沢があります。 ケースにはプロモーションステッカーがあり、小動物は2048 x 1536の解像度の写真と1280 x 960の解像度のビデオを撮ることができ、音声も録音できることを示しています。







ケース自体は非常に簡単に汚れます。 指紋は多面的なままですが、デバイスがキーチェーンとして使用されるという事実を考えると、あまり気にしません。 そして、生地のプリントは非常によく消去されます。







カメラは、写真とビデオのストレージとして最大4 GBのmicroSDカードを使用できます。 容量が1 GBのカードしかありませんでした。 カードは含まれていません。







データをコンピューターにマージしたり、USB経由でデバイスを充電したりできます。







上部パネルには2つのコントロールボタンがあります。

オン/オフボタンは、同時に写真を撮る際のトリガーであり、ビデオ録画を開始/停止することもできます。 1回押すと記録が開始され、長く押すとデバイスの電源が切れます。

2番目のボタンは、写真/ビデオ-撮影モードを変更するように設計されています。







キットの内容：

  • カメラ
  • レース
  • USBデータケーブル
  • USB充電ケーブル
  • わかりにくい円柱状のもの（外部からの大まかな検査による判断）は、ほとんどの場合、単三電池からの充電器です








ケーブルは慎重にマークされています。 ユーザーが充電ケーブルを介してコンピューターに写真をマージしようとするときにst迷に陥らないように。







充電ケーブルは、2.5 mmのヘッドフォンジャックに非常に似ています。







充電器は、透明なプラスチックと金属の奇妙な組み合わせです...







...なぜその理由を理解するのが難しい場合でも、この金属シリンダーが必要です。







充電器のサイズは、通常の指のバッテリーよりもわずかに大きいため、ポケットに入れて持ち運ぶ際に不都合はありません。







試乗



では、この技術の奇跡が何をすることができるのか見てみましょう...

デバイスに関するいくつかの事実：

  • すべての写真とビデオは、_RECフォルダーのメモリカードに記録されます。
  • 画像サイズとビデオ解像度は調整できません。
  • Windows 7では、エクスプローラーで録画されたビデオファイルのプレビューは表示されません。
  • ビデオファイルはかなり大きいです。 1秒あたり約1 MBのビデオ。
  • 写真の平均重量は230 KBです。
  • デバイスは、コンピューターに接続されている場合、Webカメラとして認識されません。




写真


メーカーは、パッケージで写真を撮る可能性を意図的に示していないようです。

写真は奇妙です-品質は4つのすべての手足で不十分です。





わかりました。風景ではうまくいきませんでした。短い距離で撮影してみます...







ご覧のとおり、状況は似ています。 ブレード上のテキストは完全に読めません。







カメラは明らかにマクロ撮影用でもありません。



映像


ビデオに移る...







ビデオは実際にほぼ宣言されたサイズです。

ビデオが録画日を示していることはひどく怒っています。 さて、正しい日付がそこに表示されますが、カメラは、現在は2007年であり、それを修正する方法が明確でないと言います。 デバイスにはドキュメントが付属していません。補助ソフトウェアもありません。そのため、この問題への対処方法は明確ではありません。



ビデオは128 1 x 960です。このため、多くの問題があります。 第一に、エクスプローラーのサムネイルは表示されず、第二にVirtualDubでbyakaになります。







パッケージに少なくともメーカーの連絡先があった場合、デバイスのファームウェアを微調整するように依頼できます（これは理論的には中国語で可能です）が、 どこにもテキストがなく、これらのカメラを製造しているのは不明です。私はこれで生きなければならないようです。 ビデオは壊れていることがわかります（ただし、プレーヤーでは正しく表示されますが、少なくともプレーヤーの開発者がこの方法でユーザーの面倒を見れば、壊れたビデオでも視聴できます。VLCとWindows Media Playerでテストしました。すべて問題ありません）。



これが、Virtual Dubがビデオファイル形式について教えてくれることです...







ビデオ品質も部分的に不十分です。 まず、移動すると、すべてが「浮いている」ため、オブジェクトの輪郭がぼやけます。

低光条件では、品質が悪くなり、オブジェクトが識別しにくくなります。



音もまあまあです。 外来ノイズが聞こえます。

結論



マーケティングが解決する別の例:)

素晴らしいパッケージングの背後には、非常に平凡な実装があります。

デバイスでドキュメントを撮影する機会はほとんどなく、テキストは表示されません。 暗い場所では屋内での撮影はほとんど不可能です。 風景写真の品質も非常に低いです。

このビデオでは、アーティファクトがピンクの縦線の形で表示されます（表示されることもありますが、常に表示されるわけではありません）。 ビデオは壊れていますが、プレーヤーで再生されます。

音質が悪い。 たとえば、交通警察や税務当局とのコミュニケーションのプロセスを撮影することを決めた場合、結果の資料が視聴に適しているかどうかはわかりません。 声が区別できるという事実ではありません。



一般に、デバイスに支払われるお金（デバイスごとに148ドル+イングランドへの配送に20ドル（最後の記事でストアはウクライナに送らない）+ウクライナへの配送に14ドル）は費用がかかりません。
