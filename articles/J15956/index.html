LeopardをTigerの上にインストールします

この記事では、以前Tigerで動作していたiMacにMac OS X Leopardをインストールした方法と、Mac OS Xのインストールに関するヒントを紹介します。



システムを再インストールする前に何をする必要がありますか？

Leopardをオペレーティングシステムの既に実行中のバージョンの上にインストールする方法は3つあります。すべてのユーザーデータとプログラムを保存できるものもありますが、その詳細は以下のとおりです。 主なことは 、どの方法を使用しても、 すべての重要なデータのコピーを作成する必要があることです。



もちろん、最も便利なのは、必要なすべてのファイルを外部ハードドライブにコピーすることです。 ただし、お持ちでない場合は、CDまたはDVDを使用できます。 DVDにバックアップするときに行うことは、外付けハードドライブにバックアップするときよりも慎重に行い、データをより正確に並べ替えることだけです。



Documents、Music、Movies、Picturesフォルダーからすべてのファイルを外部Fire-Wireディスクにコピーし、これらのフォルダーに同じ名前を付けて、名前で並べ替えるときにそれらがすべて隣になるように、先頭に@記号を追加するだけです。



特別なバックアッププログラムを使用する方が便利な場合があります。 SuperDuperをチェックしてください-すべての英語のブログがこのプログラムを推奨しています。 しかし、私はそれを使用しませんでした。



インストールの種類

Mac OS Xのインストールにはいくつかのタイプがあり、それぞれに長所と短所があります。



1. Mac OS Xをアップグレードします

アップグレードすると、以前のバージョンのMac OS Xから変更されたシステムファイルの単純な上書きが発生します。 理論的には、任意のポピーをアップグレードできますが、 Macworldでは、このタイプのインストールは、Tigerのみがインストールされ、それ以外の新しいポピーでのみ実行することをお勧めします。 実際には、多数のシステムプラグインと破損したファイルが更新されない可能性があり、問題が発生します。 さらに、LeopardはMac OS Xの非常に深刻なアップデートであり、多くの新しいフレームワークと機能が導入されました。 したがって、[アップグレード]を選択しない方が良いでしょう。



2.アーカイブとインストール

中間オプション。 一方では、Tigerシステムファイルを使用しない、Mac OS Xの完全に新しいコピーがインストールされます。 一方、古いシステムはアーカイブされ、以前のシステムフォルダーに保存されます。ユーザーとネットワークの設定を保持するチェックボックスをオンにすると、ホームフォルダー（ドキュメント、音楽、写真、ビデオ）のすべてのユーザーデータとインターネットおよびローカル接続の設定が保存されますネットワーク。 さらに、インストールされたすべてのアプリケーションが保存されます-それらを再インストールして設定する時間を無駄にする必要はありません（注：一部のプログラムはLeopardでは動作しない場合があります）。 長い間システムを再インストールすることを気にしたくない人のための非常に便利なオプション。



3.消去とインストール

maximalistsのインストール方法：ハードドライブは完全に消去され、システム全体が再度記録されますが、すべての設定とユーザーファイルは失われます。 しかし、あなたはすでにバックアップを作成しましたか？ それでは、なぜこれらのユーザーファイルが必要なのでしょうか？



私はこの特定のインストールオプションを選択しました。通常、Tigerが私の新しいシステムに入らないように、エラーがない場合は落ち着いてほしいと思いました。 私は頻繁にプログラムをインストールおよびアンインストールして新しいものを確認するため、エラーや削除されていないファイルの可能性は非常に高くなります。 これは、Tigerで作業している場合（システムを再インストールせずに1年半-WindowsのSFのカテゴリから）明確に現れませんでしたが、安全にプレイしたかったのです。 後悔するまで。



結論

私のアドバイスはこれです：完全にクリーンなシステムをインストールしたい場合-Erase＆Installメソッドを使用してインストールします。 新しいシステムはゴミを持ちたくないため、すべてのドキュメント、写真、音楽の保存を確認して整理します。



そして、すべてがうまく機能し、ドキュメントは必要に応じて整理されていますか？ [アーカイブとインストール]を選択します。 いずれにせよ、 必要なすべてのドキュメント、音楽、写真のコピーを事前に作成してください！
