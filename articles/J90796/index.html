ロシアでのVisual Studio 2010の発売





ロシアでのVisual Studio 2010および.NET 4.0プラットフォームの公式発売が行われました。 モスクワのイベントは、グローバルな打ち上げの枠組みにおける最初のイベントの1つでした。 そして、ロシアでは合計でモスクワを除く2つのスタートがあります：サンクトペテルブルクの4月14日とエカテリンブルクの16。



Visual Studio 2010の開始時に何が起こったのか。



注意、カットの下にたくさんの写真。













まず、全体レポートから始まりました。多くのスピーカーが、新しいスタジオ、SQL Server 2008 R2、Silverlight 4、および関連ツールのさまざまな側面について話し、発表しました。

下の写真では、Roman Zdebsky、Denis Kotlyarov、Vitaliy Zaiko、Marat Bakirov、Grigory Barkan。







マルチタッチ機能、新しいインターフェイスのサポート、SharePointの高度な機能、Visual StudioおよびLinq for SharePointのサポートが実証され、5月に注文可能なSQl Server 2008 R2の販売計画が発表されました。 VS2010で、IntelliSenseおよびデバッグプロシージャを使用してTSQLを操作することがより便利になることを示しました。 Marat BakirovがEntity Framework 4の新機能について話しました。







その後、Sitronicsは新しいテスターツールであるTest and Lab Managmentの機能について話し合いました。 開発の重要な側面であるチーム開発も、新しいスタジオでは見落とされていません。 これは、ローマン・ズデブスキー、ウラジミール・グサロフ、ドミトリー・アンドレーエフによって語られました。 チームがより効率的に作業するのに役立つVisual Studio 2010およびTeam Foundation Server 2010の新バージョンについて説明しました。 彼らは多くのことを語り、Test and Lab Managmentの助けを借りて、大規模なチームでのテストプロセスがより簡単かつ効率的になる方法を説明しました。







全体報告はイベントの始まりに過ぎませんでした。 さまざまなトピックに関する3つのレポートのストリームと、招待されたスター、Team Foundation Serverのチームリーダーであるブライアンハリーによるレポートが、多数のゲストを待っていました。

合計で、以下のレポートが公開時に読まれました。

ストリーム「プロフェッショナル向け」：

  • 最新のWindowsアプリケーション。 スピーカー：ミハイル・チェルノモルディコフ。
  • Web開発者向けのVisual Studio 2010。 スピーカー：コンスタンチン・キチンスキー。
  • SharePointおよびOffice 2010のソリューションの開発。スピーカー：ジョージーバーカン。
  • Team Foundation Serverへの簡単な移行。 スピーカー：ブラウンハリー。


チーム開発ストリーム：

  • Microsoft Visual Studio 2010を使用したプロアクティブなプロジェクト管理。スピーカー：Dmitry Andreev。
  • 開発者とテスターの効果的な相互作用。 スピーカー：Marat Bakirov。
  • 建築設計とコード分析。 スピーカー：セルゲイ・オルリク。
  • 開発のベストプラクティス：チームでのアセンブリの自動化とコード管理。 スピーカー：ウラジミール・グサロフ。


ストリーム「SQL Server 2008 R2」：

  • SQL Server 2008 R2のイベントフロー処理と集中サーバー管理。 スピーカー：ボリス・バリシニコフ
  • 非プログラマーの手でビジネス分析アプリケーションを作成します。 SQL Server 2008 R2のPowerPivotおよびレポートビルダー3.0。 スピーカー：アレクセイ・シュレニン
  • MySQL用のSql Server Migration Assistantを使用して、WebアプリケーションをMySQLからSQL Expressに移行する実際的な例。 スピーカー：アレクサンドル・ネヴォリン
  • Intel Parallel Studioを使用してVisual Studio 2010でマルチスレッドアプリケーションの開発を簡素化する方法。 スピーカー：アレクサンダー・ラザレフ（Intel）


主催者の功績として、3つのストリームすべてがオンラインで放送されたことに注意する必要があります。 会議に参加しなかった人はだれでも、ブラウザの便利なプレーヤーで3つのビデオストリームで本会議とその他のすべてのレポートを見ることができました。 放送はテレビを思い出させてくれました。また、ある放送から別の放送にチャンネルを切り替えて、自分にとって興味深いトピックを取り上げることも便利でした。 その隣にあるのは、＃vs2010launchのタグに関する参加者や他の人によるコメント付きのTwitterフィードでした。
