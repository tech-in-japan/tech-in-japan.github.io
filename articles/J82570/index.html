フォネラ+

技術仕様







寸法：93.5mm x 25.5mm x 110mm（アンテナなし）;

認証：WEP 64ビット/ 128ビット、WPA、WPA2、WPA混合;

暗号化：TKIP、AES、混合; ネットワーク標準：IEEE 802.11b / 802.11g（最大54 Mbps）;

アクセスポイント：1つのパブリック（デフォルトではFON_FREE_INTERNETという名前）は暗号化されず、1つのプライベート（デフォルトではMyPlaceという名前）はWPA-PSKで暗号化されます。



説明



既存のブロードバンド接続に接続するためのインターネットポートと、ネットワークに接続するための追加のLANポートが付属しています。 Fonera +はインストールが簡単で、最大54 Mbpsのデータ転送速度を提供し、すべてのWiFiデバイス（ラップトップ、電話、MP3プレーヤーなど）で動作します。 標準範囲は、環境に応じて10〜50メートルです。 最も重要なこととして、Fonera +はBT FONコミュニティで安全なブロードバンドアクセスを提供します。 英国では500,000以上、世界では700,000以上のホットスポットに無料でアクセスできます。 さらに、空港、ホテル、レストランにあるBT Openzoneの3000のホットスポットに120分間無料でアクセスできます。 Fonera +を使用すると、ビデオや写真を使用してポータルへのアクセスを共有およびパーソナライズするための帯域幅を設定できます。





Fonera 2.0g

技術仕様



寸法：93.5mm x 25.5mm x 110mm（アンテナなし）; USBポート：USB Aタイプ（USB 2.0）

認証：WEP 64ビット/ 128ビット、WPA、WPA2、WPA混合;

暗号化：TKIP、AES、混合; ネットワーク標準：IEEE 802.11b / 802.11g（最大54 Mbps）;

アクセスポイント：1つのパブリック（デフォルトではFON_FREE_INTERNETという名前）は暗号化されず、1つのプライベート（デフォルトではMyPlaceという名前）はWPA-PSKで暗号化されます。



説明



USBポートとオプションのLANポートを備えたFonera 2.0g WiFiルーター。 インストールが簡単で、最大54 Mbps 802.11gテクノロジーのデータ転送速度を提供し、すべてのWiFiデバイス（ラップトップ、電話、MP3プレーヤーなど）で動作します。 最大50メートルの標準信号範囲。 内部にコンピューターの機能が搭載されているため、インターネットでお気に入りのタスクを完了するためにコンピューターを常にオンラインにする必要はありません。 たとえば、YouTube、Facebook、Picasa、Flickr、BitTorrent、RapidShareなどのサービスとうまくやり取りします。 ダウンロード、データバックアップのために、USBドライブ（USBフラッシュドライブ、外部ハードドライブ）をUSBポートに接続できます。 その他の機能には、Wi-Fi経由でファイルをハードドライブにバックアップする機能、印刷、リモートWebカメラへのアクセスなどがあります。 Fonera 2.0gはサポートされているオープンソースであり、開発者は追加のアプリケーションを作成して共有することをお勧めします。
