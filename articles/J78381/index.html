LCD LG LF2510または「今すぐDVDが必要な理由」

1台のデバイスに低予算のテレビとメディアプレーヤーが必要ですか？ LG 42LF2510 LCD TVについてです





大型テレビの購入に困惑。 私は長い間検索し、選んだ：どこが安いか、どちらが良いか。 主な特徴は、FullHDのサポート、少なくとも50,000のコントラストでした：1、USBは素晴らしい追加、イーサネットはテレビの目新しさです。 この目新しさは、ソニーのテレビでのみ見つかりました。 USBはイーサネットよりも頻繁に見つかります。 慎重に選択した後、Sony Bravia KDL-40V5610が選択されました。 いくつかの経済的な理由から、より予算に優しいものを採用することにしました。

突然私はMVideoに入り、安価なLGモデルに目を向けました。 私はこのブランドについてあまり信用していませんが、すぐにこのテレビの機能に感銘を受けました。 残念ながら、判明したように、このテレビにはUSBがありますが、これはサービスです。 レビューを読み始めたとき、私が驚いたのは、このテレビがビデオファイルや非常に多くのフォーマットを再生できることです。 そして、最も重要なのはMKVファイルです:)）



したがって、この特定のモデルが購入されました...購入後、 LG 42LF2510は 、USBフラッシュドライブからファイルを再生するオプションが含まれていることに困惑していました DVD（Pioneer AV610）でMKVを読み取れません。

いくつかのリソースが調査されました。すべてのixbt.comに精通しており、すべての外国のavforums.comに精通していません。 USB経由で再生をアクティブ化するための答えが見つかりました。



現在のファームウェアを確認するには->

  1. リモートコントロール（リモートコントロール）の同じボタンを押して、 メニューに移動します 。
  2. オプションを選択します（クリックしないでください）
  3. FAVボタン（リモコンの右下）を7回押します
  4. 点滅モードです。 最初の行は、ファームウェアのバージョンです。


バージョンが3.4よりも低い場合は、ファームウェア3.15にダウングレードできます。 リモートコントロールとテレビの[OK]キーを押して、エンジニアリングメニューを呼び出します。 （またはそれ以降のバージョンのメニュー）。 バージョン3.47または3.48の場合、テレビをダウングレードするとレンガに変わります



そして最初のオプション：

3.15にダウングレードし、USBをアクティブにし、独自のファームウェアまたはそれ以降にアップグレードする必要があります。



2番目のオプション：受信者に特定のコードを送信することにより、サービスメニューを呼び出すことができます。 方法：サービスコンソール

SymbianおよびWinMobile用のエミュレーションソフトウェア



最初の選択肢によると、ロシア語のixbtフォーラムには多くの情報があり、ダイオードによるアクティベーションに関する情報はほとんどありません（最後にFAQへのリンクを追加します）。 フラッシュ中のいくつかのリスクに混乱したため、このオプションを正確に決定しました。



その結果、USBをアクティブにするために次のことが行われました。

  1. 後部の自転車のランプから1つのLEDが取り出されました（必要に応じて、問題なくはんだ付けされました）。
  2. 3.5mmジャックジャックを備えた古いヘッドフォンからのワイヤ。
  3. LEDは燃えるようにはんだ付けされています（極性を観察）
  4. iPhoneでスペシャルが再現されました。 ファイル。
指示は、音量を周期的に変更して再生する必要があることを示しています。 音量ではなく、テレビの受信機までの距離を変更することを好みました。



2〜5秒すると、テレビがパスワードを要求します。 リモコンの「0000」を押します。 サービスメニューに入ります。

-> 2つの設定-DivX-HD（またはSD）を変更し、JPEGを有効にします（セット1）。

ここで、「EMF（Jpeg、MP3）」は値を「1」に設定します（キー「右」と「左」）

DIVXは値を「HD」に設定します（「右」および「左」キー）

他のものを変更しないでください。 テレビが破損し、保証が失われる場合があります。



出来上がり。 すぐにMKVを試しました。 動作します!!! ブレーキなし。 DVDはすぐそばにあります...なぜ今ですか？）
