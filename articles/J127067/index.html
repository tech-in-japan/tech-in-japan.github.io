情報セキュリティの専門家Thomas Schinderがモスクワを訪問し、Tech∙Ed Russia 2011で講演する





すべての会議の主な富は、興味深い関連トピックを提示するスピーカーです。 そして、Tech ∙ Ed Russia 2011はそれらのベストを集めました。 会議参加者の前で話すこれらの世界クラスの専門家について、このシリーズの記事について説明します。



Tech ∙ Ed Russia 2011の基調講演者の1人は、情報セキュリティ、Microsoft ISA Server製品、およびMicrosoft Forefrontファミリのエンタープライズレベルのセキュリティツールの専門家であるThomas Shinderです。







現在、Thomas SchinderはMicrosoft開発チームの1人のリードエンジニアです。 Thomasのチームが開発している製品は、Unified Access Gatewayと呼ばれます。これは、Forefrontファミリー内のビジネスアプリケーションへの安全なリモートアクセスを提供するツールです。 Unified Access Gatewayを使用すると、ビルトインファイアウォールを介したSSLサポートと保護機能を備えた安全なVPNネットワークを展開できます。



不思議なことに、トーマスは深い医学教育を受けており、アメリカのテキサス州イリノイ州バークレーの大学で学んだ。 彼は、米国オレゴン州の健康科学大学に特化して3年間過ごしました。 Thomasは、1990年代初頭にIT業界でキャリアをスタートし、FINA Oil、Lucent Technologies、Sealand Container Corporationなどのフォーチュン500企業のトレーナー、ライター、コンサルタントとして働きました。 2000年、トーマスはhttp://www.isaserver.orgの編集者およびモデレーターとなり、後にISA Server専用の主要なリソースになりました。 2001年12月、トーマスはISA Serverコミュニティの発展に貢献したことにより、Most Valued Professional（MVP）ステータスを取得しました。



1997年から13年間、Thomas SchinderはTACTEAMに勤務し、現在ではネットワークテクノロジーとITセキュリティのトピックに関するさまざまな企業のトレーニングとコンサルティングを提供しています。 トーマスは積極的な作家であり、彼の創造的な人生の間に多くの本を書いてきました。例えば：
  • Microsoft Forefront Unified Access Gateway 2010の展開
  • Microsoft Forefront Threat Management Gateway 2010の展開
  • ベストダムファイアウォールブック期間、第2版
  • Microsoft Forefront Protection 2010 for Exchange Serverの展開
  • 博士 トムシンダーのISA Server 2004の構成
  • エンタープライズネットワーク用のDMZの構築
  • 博士 トムシンダーのISA Server 2006移行ガイド
その他多数。



2009年12月、Thomasはマイクロソフトの従業員になり、シニアテクニカルエディターとしてUnified Access Gatewayチームに参加しました。



Thomasは、情報セキュリティとソフトウェア保護に関する多くの知識を持ち、知識を技術的な聴衆と喜んで共有する興味深いスピーカーです。 Channel9に関するThomasのレポートには、Microsoftイベントでのスピーチの一部が掲載されています。
  • セキュアエンドポイント-DirectAccessおよびMicrosoft Forefront Unified Access Gateway 2010、完全版
  • 高度なエッジセキュリティ-SSL Threat Highwayを閉じて仮想化セキュリティに移行
  • Microsoft Forefront TMGおよびUAGの360度ビュー
  • Microsoft Office SharePoint Server 2007の包括的なセキュリティ-箱にないもの
今後開催されるTech ∙ Ed Russia 2011では、Thomas Shinderが「情報セキュリティ」というトラックで公演し、マイクロソフトの最新のITインフラストラクチャセキュリティソリューションに関する最新ニュースを発表します。



会議は、2011年11月9日から10日までモスクワで開催されます。 最初のTech ∙ Ed Russia 2011会議の登録が開始されました。
