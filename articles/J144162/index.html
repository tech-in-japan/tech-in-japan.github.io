Team Foundation Server Express



Team Foundation Server 11はまもなく登場しますが、これは開発者とプロジェクトマネージャー、テスター、アナリストの両方にとって多くの新しいものを含むメジャーな新しいリリースです。

Team Foundation Server 11では、Team Foundation Server Expressと呼ばれる新しいバージョンが導入されています。



この出版物では、Expressの無料版について説明します。 この出版物は、初心者の開発者とTeam Foundation Serverをまだ使用していない開発者を対象としています。



このバージョンには次のような機能が含まれます。

  • ソースコード管理
  • 作業項目の追跡
  • ビルドの自動化
  • アジャイルタスクボード
  • などなど...


最も重要なことは、このバージョンは、個人および5人までの開発グループにとって完全に無料であることです。 TFS Expressを使用すると、Team Foundation Serverを簡単に開始および探索できます。 独自のサーバーにインストールできます。

Express Editionは、基本的なTeam Foundation Serverと基本的に同じですが、インストールプロセスが初心者ユーザー向けに簡素化され、非常に高速で簡単になりました。



Expressバージョンの制限



Team Foundation Server Expressには次の制限があります。

  • SharePointとの統合なし、
  • 報告なし
  • サポートされるユーザーは5人までです
  • SQL Server Express Editionのみをサポートします（まだインストールされていない場合は自動的にインストールされます）
  • マルチサーバー構成なし
  • アジャイルタスクボードはありますが、計画およびフィードバック管理オプションは提供されません。
  • TFSプロキシをサポートしていません




もちろん、あなたのチームは成長することができ、より多くの機会が必要になるかもしれません。 追加のユーザー用にクライアントアクセスライセンス（CAL）を購入するだけで、ユーザーを追加できます。 さらにTFS機能が必要な場合は、データを失うことなく完全なTFSライセンスにアップグレードすることもできます。



さらに、Expressバージョンを含むTeam Foundation Serverとの統合が、Visual Studio Express製品ラインの製品に追加されました。



Team Foundation Server Expressをインストールする



TFS Expressのインストールプロセスは非常に簡単です。

ほとんどすべてのシステムにインストールできます。 TFS Expressには、構成済みのドメインは必要ありません。 インストーラーは、必要に応じてSQL Server Express Editionもインストールします。





必要なコンポーネントをインストールした後、サーバー構成の更新プロセスに慣れます。





インストールが完了したら、管理コンソールを使用してより柔軟な構成を行うことができます。





TFS Expressは、次のアドレスでWebアクセスも提供します。http：// <tfs-server>：8080 / tfs





Visual Studio Expressから接続する


既に述べたように、MicrosoftはVisual Studio Express 11ラインにTFS統合を追加しました。

チームメニューからTFS Expressサンプルに接続できます。





接続後、TFSの通常バージョンと同じ方法で作業できます。
