特にノキアLumiaとWindows Phoneの完全な失敗の理由

以下は、my-symbian.comのSlashdotからのメッセージの翻訳です（オリジナルが見つかりました）。 ソーステキストは本格的な記事ではないという事実にもかかわらず、価値があります。 私は少し急いでいたので、私はプライベートメッセージを介して送信された修正に満足しています。



WP7.5（Mango）はこのOSの第2世代ですが、まだ多くの制限、禁止事項、深刻なユーザビリティの問題があります。 これは、市場にある他のモバイルOSと比較した欠点のリストです。 特別な機能や能力を比較するのではなく、最も単純なダイヤラーであっても、すべてのスマートフォンの標準となっているものだけを比較します。 Android、Symbian、またはMeeGoには、これらの欠点のいくつかしかありません。 iPhoneのファイル転送機能にも制限がありますが、それでもWP7.5よりもはるかに進んでいます。 MeeGo OSの第1世代を搭載したNokia N9でさえもはるかに優れており、スマートフォンを正常に使用できないようなこれらの壊滅的な問題はありません。





OSの制限

1.実際のマルチタスキングはなく、アプリケーションはバックグラウンドで「フリーズ」します。



2. Divx / Xvidはサポートされていません。 Zuneは品質の低下を伴います。



3. USBモードはありません。



4. micro-SDサポートなし。



5.デバイスごとに最大16Gbのメモリ。



6.ファイルマネージャはありません。 ファイルシステムは完全に不透明です。



7.ファイルを転送するにはZuneが必要です。 Zuneは写真、ビデオ、音楽のみを転送できます。 通常のファイルには、インターネットを使用する必要があります。



8.連絡先情報は、希望するかどうかにかかわらず、クラウドに自動的に転送されます。



9.最大画面解像度は800x480です。



10.音声検索はBingに関連付けられています。



11.まれな例外を除き、着信音にMP3を配置することはできません。



12.「インター」システムでは、VPNのサポートはありません。



13.接続に静的IPを設定することはできないため、アドホックネットワークに接続します



14.クラウドと同期せずにOutlookと同期することはできません。



15.絶対に閉じられたOS、MS Marketplaceからのアプリケーションのみ。



16.フォントサイズを変更することはできません。



17.デバイス内の写真の名前を変更できません。



18. Windows Live IDアカウントでは、最初のインストール後に国を変更できません。



19.集中通知ページはありません。



20.デバイスの電源がオフの場合、アラームは機能しません。 Symbian / Meegoでは、動作します。



21.スタンバイ画面は完全に空であり、時間/通知を表示できません。



22.電子メールに添付できるのは写真のみです。



23.最も一般的なBluetooth rSAPプロファイルはサポートされていないため、ほとんどのカーオーディオシステムにオーディオを転送することはできません。



24.モバイルバンキングやオンライン決済などの安全なアプリケーションに必要なクライアント側の暗号化サポートはありません。



25. A2DPプロファイルが実装されていないため（！）、ビデオからBluetoothデバイスにサウンドを転送することはできません。



26. Bluetoothキーボードのサポートはありません（HIDプロファイルはありません）。



27.電話を裏返しにして着信音や目覚まし時計の音を減らすことはできません。



28.高度に制限されたカスタマイズ可能性。



29. WP8 Apolloにアップグレードできない。



ユーザビリティの問題



30.バッテリーレベル、信号、2G / 3G / Wi-Fi-Bluetoothステータスが常に表示されるわけではありません。



31.タスクマネージャーは、アプリケーションを完了する方法を知りません。



32.ゲームまたはアプリケーションで「戻る」および「検索」ボタンをオフにすることはできません。これらのボタンに触れて、アプリケーションでの作業を中断します。



33.不在着信と新しいSMSの数を確認するには、デバイスのロックを解除する必要があります。



34.メイン画面に移動せずにアプリケーションを最小化して一時停止する方法はありません。



35.メッセージ内の小さなフォントは、45歳以上の人にとって読みにくいです。



36.電話でプレイリストを作成および保存することはできません。



37.プレイリストは、再生中にのみ編集できます。



38.マーケットプレイスでのみ、携帯電話で音楽コレクションを検索することはできません。



39.音楽プレーヤーを閉じることはできません。最小化することしかできません。 ロック画面のウィジェットは多くの不便をもたらし（重要な会議で誤ってアクティブにすることができます）、再起動後にのみ消えます。



40.現在の曲を巻き戻すことはできません。アルバムのどの曲が再生されているかを確認することはできません。



41.自動方向変更をブロックすることはできません（WP7を使用している人は直立しているだけですか？）



42.オンライン連絡先と電話連絡先は混在しており、それらを分離する方法はありません（除外）。



43.ダイヤラーの[検索]ボタンは、連絡先の検索ではなく、通話履歴に移動します。



44.不完全なSMSを下書きとして保存することはできません。



45.通話履歴には、番号の種類のみが表示されます。 連絡先に同じタイプの番号が複数ある場合、その番号の意味を知ることは不可能です。



46.彼からのSMSメッセージからサブスクライバーを保存したり、呼び出したりすることはできません。



47. SMSは一度に1つずつ、またはスレッド全体で削除されます。



48.削除、送信、またはアップロードのために複数の写真を選択することはできません。 一度にこれを行う必要があります。



49.画像のプロパティ（解像度、サイズ、作成日など）は表示できません。



50.アプリケーションはアルファベット順に配置されます。彼のアプリケーション名を覚えていない場合は、検索に時間を費やす必要があります。



51.カレンダーには週単位のビューはありません。月単位はスケーラブルではありません。



52.添付ファイル付きの手紙を受け取った場合、電話はそれをダウンロードします。不要なダウンロードからあなたを救うスケジュールはありません。



53.データ転送とwi-fiが同時にオンになっている場合、送信するチャネルの電話の選択は予測不可能であり、通常はユーザーの習慣に同意しません。



容量制限



54.ライブ壁紙、ウィジェット、テーマはありません。



55.キーボードからの振動フィードバックはありません。



56.スワイプなし。



57.そしてフラッシュ。



58.そしてJava。



59.レコードを呼び出します。



60.そして、ブロッキングを呼び出します。



61.スクリーンショットは忘れてください。



62.壁紙の自動変更はありません。



63.完全に閉じたOSとは、アプリケーションがハードウェアと直接対話できないことを意味します。 便利なアプリケーションの多くを忘れてください。



64. Zune音楽プレーヤーにはイコライザーがありません。



65.モバイルトラフィックのアカウンティングはありません。



66. IEは、テキストをスケールに合わせる方法を知らず、ファイルを読み込む方法を知らず、キャッシュからページを読み取ることができません。



67. IEに基づいていないサードパーティのブラウザはありません。



69.上/下ボタンはカメラのズームには使用できません（OSの未熟の兆候）



69.電子メールメッセージの添付ファイルからzip / rarアーカイブを開くことができません。



70. MMS経由でビデオを受信できません。



71. Office Mobileの機能は、SmartOffice、QuickOffice、およびPolarisよりも少ないです。 恥を知れ、マイクロソフト。



72. GoogleマップとBingは、米国以外のほとんどの国では利用できません。



73.電子メールメッセージのタイムスタンプに年が表示されません。



74.各国で利用可能なアプリケーションの大きな違い、ユーザーはマーケットプレイスに関連付けられています。



75.すべてのBing機能が州外、特にローカルスカウトで利用できるわけではありません。



ダンバーとダンバー



76.音楽の再生、着信音、アラーム、通知など、すべての機能の1つの音量レベル。



77.画面がオフの場合、Wi-Fiはオフになります。 モバイルネットワークが使用可能な場合は、代わりに使用され、明らかな結果が生じます。



78.電話中にテキストメッセージを受信すると、ビープ音が鳴り、音量が最大になります。 他のすべての電話はそれを減らします。



79. 1つの連絡先に対して1つの電話番号しか入力できません。他のフィールドに入力した番号はSMSを受信しません。



80. MMSは3Gを使用していませんが、3GをオンにしないとMMSを送受信できません。



81.電源がオフの間、デバイスは充電できません。



82.有線同期を行うには、デバイスを有線充電に接続する必要があります。



83.巨大な見出しは、画面上の重要な場所を使用し、情報密度を低下させます。



84.デバイスが起動し、着信メッセージのテキストを表示します。これにより、プライバシーが侵害されます。



85.セキュリティコードを入力せずにデバイスを再起動できます。



可能性はさらに暗いですが、WP7.5ではもはやできません

87. USSDコードを使用できない。



88.不在着信、着信、ダイヤル、最近の通話の履歴をフィルタリングできない。 通話は一切ありません。



89.通話履歴をバックアップまたはエクスポートすることはできません。



90.履歴に通話時間の表示はありません。

93.通話履歴には、今日行われなかった通話の通話時間が表示されません。



94.さまざまな種類の通知にサウンドを設定することはできません。



95.音声プロファイルはありません。



96.メニューにはサイレントモード（振動と音なし）はありません。 オンとオフを切り替えるには、設定に移動する必要があります。



97.連絡先をCSVファイルとして受け入れたり送信したりすることはできません。



98.連絡先またはSMSをPCにバックアップできません。



99.連絡先をSIMカードに保存することはできません。



100.目覚まし時計の音は変更できません。



101.アラームの有効期間は変更できません。



その他の欠点



102.トリミングされているため、オーディオとビデオのプレイリストで長いタイトルを読むことはできません。



103. Zuneでは、ユーザーがデバイスからポッドキャストを追加または更新することはできません。



104. Zuneは、インターネット接続でのみインストールできます。



105.目覚まし時計は、ヘッドフォンが接続されている場合、ヘッドフォンで再生されます。



106.転送された電子メールメッセージは編集できません。



107.ヘッドフォンの通知の音量は下がりません。
