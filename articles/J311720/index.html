WordCamp Europe ViennaおよびWordPress Development Vector





今年、ウィーンでWordCamp Europe 2016会議が開催され、2,300人以上のゲストが集まりました。 オーストリアの首都は、このようなイベントを開催するのに最適です。便利な場所、大きな会議室、アクティブなWordPressコミュニティなど、すべての条件があります。 そして、会議後に見るべきものがあります。 開発者の何人かが WordCamp Europe 2016を訪れました。カットの下で-最も興味深いレポートとイベントに関する彼らの物語。



会議形式



会議は3日間続きました。最初の2日間はスピーチを聞き、最後の日は寄稿者に贈られました。 レポートは、開発、デザイン、ビジネス、コンテンツ、コミュニティのさまざまなカテゴリに分類され、3つのストリームで行われたため、誰もが自分にとって興味深いものを見つけることができました。 スピーカーは、高負荷、継続的インテグレーション、REST API、およびCalypsoプロジェクトの問題に多くの注意を払いました。



寄稿者の当日、約600人のボランティアがサインアップしました。 この日は、WordPressの開発に直接参加し、カーネル開発者、翻訳者、および公式カタログのプラグインとテーマをチェックするチームがどのように機能するかを知ることができます。 あなたの街でWordPressコミュニティを組織するためのヒントさえ得ることができます。



素晴らしいボーナスは、WordPressの創設者、Automattic、WordPress.com、Akismetの創設者、Matt Mullenwegのパフォーマンスでした。 しかし、それについては以下をご覧ください。



最も興味深いレポート



PHP7およびWordPress



PHP7のリリースはWordCamp Europeで気付かれることはありませんでした：Dan Blowsは、WordPressをPHP7で動作させる方法について、PHP7の新機能とPHP7.1の機能について報告しました 。



WordPressコアと多くのプラグインは既にPHP7をサポートしています。 著者を引用すると、「PHP 7へのアップグレードは簡単で、おそらくすべてがすぐに動作するでしょう」と言うことができます。 しかしそれにもかかわらず、彼はサイトをPHP7に移行することに決めた場合に注意すべき点に触れました。







スピーカーは、PHPの技術革新と、第7バージョンに切り替えたときに得られる利点について話しました。 Danは、PHP 5.6と比較して、PHP7で印象的なWordPressパフォーマンスアクセラレーション統計を示しました。



いくつかのWordPress REST APIレポート



特に重要なのは、REST APIに関するレポートです。 これは最も重要で活発に開発されている分野の1つです。 WordPress REST API開発者自身からのニュース、難易度と決定に関するニュース-信じられないほど貴重な情報。 ジョー・ホイルとアダム・ホワイトに感謝します！ Ultimate REST APIトークおよびREST APIとJavaScriptを使用してより良いWordPressインターフェースを作成することにより、彼らのパフォーマンスをご覧ください 。







レポートは、APIの使用とその開発および拡張の両方に関連する問題を明らかにしました。



現在、REST APIはまだカーネルの一部ではなく、 プラグインとして存在しています 。 Adamは、RESTをコアに統合するときにWordPressがもたらすメリットをレポートで強調していますが、今日では、新しいAPIを使用してアプリケーションを構築する機会がすでにあります。



REST APIは前例のない柔軟性を提供し、「WordPressはあなたの想像力によってのみ制限されます」という表現が現実のものとなります。



カリプソ管理パネル



比較的新しい開発-Calypsoに十分な注意が払われました。 これは、JSで記述されたWordPress Webサイト管理パネルであり、WordPress REST APIを搭載しています。これは間違いなく人気があります。







デザイナーのカリプソは、プロジェクト全体と開発チームの費用について話し、プロジェクトで効果的なコミュニケーションを組織するためのアプローチを共有しました。 ダビデ・カサリが言ったように：「コミュニケーションは酸素です。 同意しないことは不可能です。







負荷の高いWPサイトの構築経験



WordPressは、負荷の高いニュースポータルの開発に長い間使用されてきました。 TED、TechCrunch、CNN、NBCなどの巨人がサイトを構築しました。 はい、高負荷のサイトを構築するのは簡単ではありませんが、NewsCorp Australiaのスピーカーはそれを正しく行う方法を話しました。



彼らの経験は、1日あたり数百万のビューを持つWPポータルを構築する可能性を証明する別の先例です。 スピーカーは、開発チームとワークフローに関する貴重な情報を提供し、訪問の継続的な統合、環境、統計について話しました。 また、このようなプロジェクトに必要なコストの計算に役立つ情報を共有しました。







顧客とのコミュニケーションの観点からの信じられないほど有用なレポート： NewsCorp Australiaは、WordPress VIPでオーストラリア最大の「ニュース」ウェブサイトをホストするためにWordPressをどのようにスケーリングしたか 。



レガシープロジェクトについて



Andrew Nacinは、WordPressコアの主任開発者です。 このような人気のあるCMSシステムの長年の開発経験に基づいて、カーネル開発者は 、Andrewが固守しているWordPress哲学の重要なポイントを特定し、まとめました。



将来について本当に考えていなかった人々から「継承」された大規模なプロジェクトを取得することは常に困難です。 そして、それも痛いです。 そのようなプロジェクトがある場合、Andrewのレポートは、不適切な投資を必要としない安定した製品に変える方法を理解するのに役立ちます。 レポートのビデオはこちらから入手できます 。



ElasticsearchとElasticPress



MySQLデータベースでの全文検索は、それ自体では最適なソリューションではありません。 また、負荷の高いサイトや大量のデータがあるサイトなど、さらにはそうです。 Elasticsearchは、効果的な検索を実装するためのソリューションの1つです。 仕組み、Elasticsearchの設定方法、データ保護の確保方法、最後にプロジェクトでの使用方法。 Taylor Lovettは、 Elasticsearchを使用したWordPress Searchの近代化レポートでこれらの質問に答えています。



テイラーはElasticPressプラグインについても語っています。 スピーカーはこのプラグインの開発者の1人であり、彼が話していることを知っています。 一般に、コンポーネント開発者とのミーティングは、使用の複雑さや将来の開発計画に関する情報を入手する絶好の機会です。

コントリビューターの日、WPコアの開発方法



会議は2日間に限らず、興味深い報告とコミュニケーションでいっぱいでした。 特に興味のある人のために、約600人が参加した貢献者の日が計画されました。 彼はウィーン大学の情報学部の建物で合格しました。



各参加者は、オープンソースの開発プロセスに精通し、WordPressの開発で可能なすべての支援を提供できる方向を選択しました。 分野には、カーネル開発、プラグインおよびテーマ開発、国際化、設計、マーケティング、サポートがありました。 各グループは経験豊富なキュレーターによって調整されました。 その日のモットーは「Thank WordPress」でした。 コミュニティに参加して、新しい経験を積んで、専門的なレベルを向上させ、WordPressを改善することができます。



WordPressの未来



WordPressは積極的に開発中です。 最も興味深く、有望な分野はREST APIとJSインターフェースです。 さらに、後者は部分的に同じREST APIにより可能になりました。



カーネルには多くの改善点があります。たとえば、最近のリリースでは、国際化が大幅に改善されています。 テクノロジースタック、開発アプローチ、ツールなど、すべてがWordPressの最初のバージョンから大きく変わりました。 前進があり、何か新しいものをもたらす定期的なリリースを見ています 。



良いニュースは、カーネルが既にPHP7をサポートしていることです。 NIX Solutionsでは、すでに新しいPHPでプロジェクトを開発しています。 しかし、第7バージョンへの完全な移行に関しては、ここでの見通しはそれほど明るくありません。 実際のところ、プラグインとテーマはコミュニティと個々の企業によってサポートされており、それらのすべてが穏やかに言えば、PHP7のサポートの実装を急いでいるわけではありません。



ウィーン会議で驚いたことは、欧州共同体の成熟度とWPに対する真剣な態度でした。 このように大きく、活発で才能のあるコミュニティで、WordPressには素晴らしい未来があると確信しています。
