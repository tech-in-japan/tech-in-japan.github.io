.Net CoreとVisual Studio Codeをインストールする



ここに記載されているインストールコマンドは、Ubuntu 16.04 / Mint 18.xディストリビューションに適しています 。残りの部分については、 こちらを参照してください 。



.Net Coreのインストール：



最新のaspnetプロジェクトジェネレーターとの互換性のために、現時点で最新のRC4をインストールします。 

sudo sh -c 'echo "deb [arch=amd64] https://apt-mo.trafficmanager.net/repos/dotnet-release/ xenial main" > /etc/apt/sources.list.d/dotnetdev.list' sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 417A0893 sudo apt-get update sudo apt install dotnet-dev-1.0.0-rc4-004771




Visual Studioコードをインストールする



このリンクを数回クリックするだけで簡単にインストールできます。



C＃拡張機能をインストールする



Visual Studio Codeを実行し、Ctrl-Pを押して、コマンドを入力します。

ext install csharp







左側に表示されるパネルで、対応する拡張機能の反対側にある「インストール」をクリックします（これが自動的に行われない場合）。 Visual Studio Codeは今のところ閉じることができます。



開発環境の準備とアプリケーションテンプレートの作成



オリジナルのサイト（ディストリビューションに付属しているものは適合しません）から最新のnode.jsをインストールします。付属のnpmパッケージマネージャーのために必要です。 

 curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash - sudo apt-get install -y nodejs sudo apt-get install -y build-essential
他のディストリビューションについては、 こちらの手順をご覧ください 。



プロジェクトの初期化



初期化には、 Yeoman scaffolderが使用されます。これは、ファイル構造の展開とプロジェクトテンプレートの生成を含むプロジェクト初期化子です。 アプリケーションのソースコード。 scaffolder Yo、Bowerパッケージマネージャー、Gruntタスクマネージャーが含まれています。 Yoをインストールすると、Bower and Gruntもインストールされます。 ここで、project.jsonの代わりにmsbuildビルドシステムが返される端末に最新のaspnetジェネレーターをインストールします。 

 sudo npm install -g yo bower generator-aspnet


Project Generatorの起動



Yeomanジェネレーターは、プロジェクトの初期化を記述するYoの指示とテンプレートを含むnpmパッケージです。作成するディレクトリ、ファイルとコピーする場所、テンプレートを処理する方法、処理後に配置する場所。 プロジェクトを配置するフォルダーに移動し、ASP.NET Coreプロジェクトジェネレーターを実行します。
