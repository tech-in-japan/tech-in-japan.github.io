フェデックス：パート1.フェデックスワールドワイドパーセルソートセンターのツアー





テネシー州メンフィス。 信号の稲妻が点滅して、ソートが今日遅く開始されることを示します...

注意、たくさんの写真！



青い信号灯はそれほど頻繁に点灯しません。 メンフィス国立空港から半径3マイル以内で雷が検出された場合にのみ点滅し、点灯するとすべての作業が停止します。 雷雨が猛威を振るうとき、金属構造物に囲まれた通路に人員が入ることを許可してはなりません。











私は白いフェデックス郊外のバスの中に座っており、大雨が壁と雷の後ろでざわめいています。 私にとって、この予想外の遅れはささいな不便さであり、FedEx World Hubの何千人もの従業員にとってこれは大きな問題です。 到着した140機の航空機から数十万個の区画を分類する必要がありますが、今のところ物事は動いていません。





ボーイング777は荷降ろしを待っています。



CNETロードトリップの一環として、テネシー州南西部に655,000人の人口を持つメンフィスに到着し、FedExコアを訪問しました。 最初の停車地は小包包装研究所でした。今日の目標は、最大の高速配送センターであるワールドソーティングセンターで、1晩で130万から150万個の小包を処理する巨大な構造です。



最後に、嵐が静まり、信号灯が消えます。 これで作業を開始できます。 夜勤が23:00に始まる直前に嵐が始まったため、小包を動かして前後に走り回ることになった数百人の従業員は、仕事ができる瞬間を待つことを余儀なくされました。 私が安全な部門に入ったとき-空港チェックポイントの簡易版と考えてください-数百人のFedEx従業員が並んで立って、最終的に職場に着くために検査を待ちました。



1時間あたり180,000



一方、ワールドセンターでの夜間のソートは印象的な操作です。 1時間あたり180,000パッケージは、最新のものが処理されるまで並べ替えセンターを通過します。



毎晩約140のフェデックスの飛行機が世界中からメンフィスに到着します。同社は220か国の375の空港の飛行機にサービスを提供しています。同じ飛行機で仕分けして目的地に送らなければならない荷物を積んでいます。 荷物を降ろした後、荷物は9つの入力セクターのいずれかを通過し、次にマトリックスに移動します。従業員の最初の行では、バーコードスキャナーがそれ以外の荷物を読み取れるように、荷物がコンベアに正しく配置されるように制御しますパッケージがコンベア上にある。

マトリックスに入った後、小包はコンベアに沿ってさらに構造内に移動し、まず小包バーコードを読み取るスキャナーを通過します。 この情報に応じて、小包は19個の自動ディスペンサーのいずれかに配達され、その後、2次仕分けセクターに送られ、次に「出口」に送られて、コンテナーに再梱包されて飛行機に再び積み込まれます。



「140人が到着し、140人が飛び去りました」



メンフィスにあるフェデックスワールドセンターは、同社のグローバルネットワーク上でパッケージや手紙の夜間の分類を行う12のセンターの中で最大のものです。 残りは、インディアナポリス、ダラス、マイアミ、トロント、オークランド（カリフォルニア）、東京などの場所に、国中および世界中に点在しています。



フェデックスでは、1晩あたり約400万件のエクスプレス（航空）貨物と、地上を含む合計1,000万件の貨物を処理しています。 しかし、ワールドセンターは事業全体の中心であり、フェデックスは世界最大の仕分けセンターであると主張しています。



毎晩7,000人の従業員が1シフトあたり130万から150万の急行を整理するために働きに出ます。 3,500人の従業員で構成される毎日のシフトでは、約650,000個の小包のみを処理し、配達時間は2日と3日です。



メンフィスへの商用便は午後9時30分に着陸しなくなり、今後はフェデックス機のみが着陸します。 おやすみのとき、雷雨のために信号灯が点灯しない場合、フェデックスの航空機は午後10時30分から着陸のために整列し、午前1時頃まで次々と着陸を続けます。







目標は、最新のパッケージが2:07までに並べ替えゾーンに到達することです。 午前2時30分までに、最初の「発信」機がソーティングゾーンの出口を出始め、午前4時30分までにすべての機体がメンフィスを出ます。 ワールドセンターツアーのコーディネーター、リックアームストロングによると、「140人が到着し、140人が飛び去りました。」



各ライナーを降ろすために、別々のチームが割り当てられます。 彼らはすぐに動作します：彼らは航空機の内部の輪郭に完全にフィットする形状を持つ金属製の容器を取り、それらを降ろして、地面にそっと下げます：







彼らの目標は、フェデックス艦隊の最大のライナーであるボーイング777を1時間7分で降ろすことです。 次に大きいライナーであるMcDonnell Douglas MD-11を降ろすには、35〜40分かかります。







アンロードされたコンテナはローダーに置かれ、入口ゾーンに配達されます：







フェデックスワールドセンターには、行列が配置されている9つのエントリエリアがあります。 そのうちの5つは国内郵便用、残りの4つは海外からの小包、および危険物、重量物、またはミニ貨物用です。 危険物には、腐食性、爆発性または放射性の物質、武器、およびとりわけバッテリーと蓄電池が含まれる場合があります。 フェデックスは、世界最大の危険物のサプライヤーです、とアームストロングは言います。 輸送が許可されている危険物のリストは非常に多く、電話帳程度です。



小包の流れ



小包が入力セクターに入るとすぐに、上、中、下の3つのコンベヤーのいずれかに置かれます。 ほとんどのパッケージの重量は75ポンド（ 約34 kg-約Transl。 ）未満で、Matrixに向かう中央のコンベヤーに落下します。







重い荷物またはスキーやゴルフクラブなどの不規則な形の荷物は、下部のコンベアに落ちます。 上部コンベヤーは、小包用の配送センターに送られる小包および書類用に設計されています（Small Package Sorting System、SPSS）。



入力セクターからの区画がマトリックスに入るとすぐに、単語の文字通りの意味で「ストリーム」が作成されます。 2011年に日本の津波のビデオを見たとき、村が荒れ狂う川によって流されたとき、マトリックスに向かう数千のパッケージは同じように見えますが、そのような恐ろしい意味合いはありません。







区画はグループで実行されます-飛行機から降ろされる各コンテナには約235個の区画が含まれます。 そのような流れは、マトリックスの労働者を圧倒するように思われるでしょう。 FedEx Packaging Labが顧客と協力して、Matrix内のパッケージが段ボールの海に落ちるまで高速で跳ね返り、互いにぶつかって衝突するため、パッケージが安全に梱包されるようにすることは良いことです。 マトリックスワーカーは、1時間あたり何万ものパッケージを並べ替えます。







さらに、小包はコンベアで19台のディスペンサーへの途中の12台のスキャナーの1つを経由して最終的に飛行機に送られ、さらに目的地に送られます。







グローバルコントロール画面では、FedExの従業員はいつでも各FedEx航空機の位置を追跡できます。 画面には、過去数日間の数十の航空機のステータスが表示されます。







小包配送システム



おそらく、マトリックスの方向に移動するパッケージのフロー以外のアクションの最もエキサイティングな部分は、小さなパッケージをソートするためのシステムです。 ここでは、別の建物で、チームは1晩あたり760〜86万の小さなアイテム（ほとんどのドキュメント）をソートします。アームストロングによると、チームは99.97％の精度でソートされます。



これらの小さなパッケージが入力セクターを通過すると、それらはコンベヤーに置かれ、スキャンされます。 しかし、大量の貨物の場合のように、仕分けの代わりに、目的地に対応する1500個のパッケージのいずれかに分類されるまで建物内を移動します。 パッケージが満たされると、システムが自動的にタグを生成し、従業員がそれに署名します。

小さな荷物を並べ替えるシステムは、特に注意をそらし、それぞれが独自の方向に移動し、最終的にパッケージに滑り込む文字や小さな荷物を誘う数十のコンベアに集中している場合は魅力的です。











最も複雑な部分では、FedEx Parcel Processing Centerは巨大なメカニズムであり、何千もの可動部があり、それぞれがシステム全体がスムーズに機能するために時間をかけて潤滑する必要があります。 しかし、これを無視すると、プロセスは非常に簡単です：毎晩数十万個の小包を持ち込み、サイズと目的地でソートし、処理された飛行機に積み込んで送信します。 すごいですね。 そしてほとんどの場合、FedExパッケージは時間通りに到着します。 アームストロングが強調したように、「当社の精度は99％です。 そして明日、この結果を繰り返します。」
