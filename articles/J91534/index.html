iPhoneで地図を表示するときにトラフィックを節約する

マップアプリケーションを開くたびに、iPhoneがネットワークにアクセスし、必要なマップの一部をポンプで送り出すことをご存知でしょう。一方で、多くのスペースと迅速な更新は必要ありませんが、GPRS \ 3Gの野pricesな価格では、これは現実的ではありません。ちょっとしたトリック。



カードはWIFIなしで機内モードで動作することに注意してください



オフラインマップにアクセスするには、iPhone用の何らかのファイルマネージャーが必要です。

Windowsの場合、たとえば、 Winscpまたはifunbox （両方とも無料）は、wifiおよびsshおよびifunbox経由でアクセスするためのwinscpのみ-ケーブル経由



今、私たちはモスクワの地図のキャッシュを取り、それらを入れる必要があります

MapTiles.sqlitedbを\ private \ var \ mobile \ Library \ Caches \ MapTiles \にコピーします

ファームウェア3.1 for private \ var \ mobile \ Library \ Caches \ Maps \ MapTiles \

com.apple.Maps.plistを\ private \ var \ mobile \ Library \ Preferences \にコピーします

Bookmarks.plistをprivate \ var \ mobile \ Library \ Maps \にコピーします



また、iFile \ terminal \ sshを使用してコピーしたファイルの権限を変更する必要はありませんが、必須ではありません




モスクワのキャッシュは、他の都市と同様に、例えばモスクワの地図などの急流からダウンロードできます

またはMAGNET経由でダウンロード



これは、上記のマップのオフラインアクセスに含まれるものです。





このプログラムを使用してカードを作成することもできます



指示が役に立ち、トラフィックを節約できることを願っています！



PS Wifiを使用せずに地図を見ることができるように、iPodでも実行できます。
