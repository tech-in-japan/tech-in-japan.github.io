「短左シフト」問題の解決

キーボードを長時間使用するときの快適さは、生産性と手の健康に影響する重要な要素です。 多くの場合、中断することなく数時間キーボールする必要があるため、実際に適切なキーボードを選択することに加えて（キーの移動高さ、ストロークの硬さ、操作点から停止までのストロークの深さ、キーの表面からの触覚、キーボードシェルの剛性、キーボードレイアウトオプションなど）をお勧めします手間をかけずにキーボードの使いやすさを改善できる簡単なトリックを考えてください。



私の意見では、「集中的な作業（ゲーミング以外）のための、中程度または低キーの移動、中程度の硬度のワイヤレスキーボード」のカテゴリでは、リーダーがLogitechキーボードです。



残念ながら、ローカライズバージョン（ラテン語と第二言語の文字を含む、メモについては記事の最後を参照）のこの会社のフルサイズキーボードモデルの大部分は、「標準レイアウト、2つの長いShift、大きなEnter、長いBackspaceを備えたフルサイズキーボード」のカテゴリに属していません。



このカテゴリに「含まれない」理由は、Logitechのデザイナーとエンジニアが、右側のスラッシュ（前方、後方、垂直バー）に重複キーを配置することで左Shiftを「短く」したいという永続的な願望です。



多くの場合、左の長いShiftを押すには、水平に配置する小指、または小指と薬指、またはその他のオプションのいずれかを使用します-直感的には、左のShiftが十分に長いと仮定し、キーボードの左端の近くを押そうとする必要はありません。 したがって、左の複製「スラッシュ」キーの誤った押下がしばしば発生します。



私はこの不便さに苦しんでいました。特に、左の長いShiftのキーボードから短いののキーボードに切り替える必要があったとき-Logitechの場合。

グーグル-これは困難であることが判明しました。「ユニバーサルキーボードドライバー」などの結果が多少異なるため、この特定の問題の解決策を選択するクエリを作成するのが難しいためです。



ソリューションは非常にシンプルであることが判明しました-KeyTweakユーティリティ。 私が理解しているように、ネイティブサイトはもうありませんが、多くのソフトウェアポータルからダウンロードできます。 インストールは必要ありません-通常は正しいです。これは、特定のコンピューターでユーティリティを1回実行するだけで済む可能性が高いためです（そうでなければ、誰かがキーの低レベルの再マッピングに夢中になります）。



レセプションの本質はスクリーンショットから明らかです：







左の重複する「スラッシュ」キーの低レベルのコードを、左のShiftの低レベルのコードに置き換えるだけです。 したがって、複製キーを押すことは、左Shiftキーを押すことに似ています。



レジストリの設定は次のようになります（ユーティリティがなくても実行できます）。





（レジストリブランチ\ HKEY_LOCAL_MACHINE \ SYSTEM \ CurrentControlSet \ Control \ Keyboard Layout）



これはレジストリの設定であるため、KeyTweakユーティリティは不要になりました。これはドライバーではなく、そのような設定の利便性のための単なるインターフェイスです。



この場合：



  • 左シフト自体で何も変わっていません。
  • 左Shiftキーを押すには、それを押すことができます。「スラッシュ」代替キーを押すことができます。Shiftキーと代替キーの両方を同時に押すことができます。 副議長は、メインの左Shiftと同様に、キーボードショートカット（Ctrl、Altなど）に完全に関与しています。
  • 右シフトで何も変わっていません。
  • スラッシュ（Enterの隣にあります）の付いた右の右キーで何も変更はありません-動作し、動作します
  • Logitechキーボードだけでなく、他のいくつかのメーカーのキーボードでもテスト済み。
  • 設定は特定のPCで1回行われ、キーボードを変更するときに何もする必要はありません。新しいキーボードに左の「スラッシュ」キーがある場合、設定が使用されます。 。


「ローカライズされた」キーボードについての注意はLogitechです-私が理解するように、短縮された左Shiftはローカライズされたキーボードのみの問題です。 同じモデルの場合、ラテン語のみのキーボードには通常の長いShiftがあり（下の公式写真へのリンクを参照）、ローカライズされたキーボードは短い（ロシアの小売店で引用したすべてのキーボードには短いShiftがあります）。 デザインのアイデアの深さは何ですか-わかりません。 もちろん、例外があります-国際化に関係なく左Shiftが長いキーボードモデル。
