アドビがオフラインFlashビデオプレーヤーを準備

主要なオンラインビデオ市場であるAdobeは、Flashビデオをオフラインで再生するためのAdobe Media Playerのリリースを発表しました。



おそらく来週ラスベガスで開催される全米テレビ放送会社協会のショーで、アドビはWebに接続せずに後で見るためにFlashビデオをダウンロードする新しいプレーヤーについて詳しく話します。 このプレーヤーは、WindowsおよびMacで使用できます。



以前はPhiloとして知られていましたが、Adobe Media Playerを使用すると、ユーザーはRSS経由でビデオポッドキャストを購読してから再生できます。 また、このアプリケーションでは、ユーザーがビデオにコメントしたり共有したりできます。



プレーヤーを開発するアドビは、ビデオコンテンツを提供する企業に焦点を当てたため、コンテンツを保護する組み込みの手段を強調しました。



Adobe Dynamic Media OrganizationのチーフストラテジストであるMark Randall氏は、現在WebキャスティングにFlashビデオを使用しているパブリッシャー（現在76％）が新しいプレーヤーを使用できるようになり、新しい機会が提供されると述べています。



Adobe Media Playerは2種類のセキュリティをサポートします。 最初のバージョンでは、ダウンロードしたビデオは広告にしっかりと関連付けられており（削除する機能はありません）、2番目のバージョンでは、コンテンツコントロールシステムが特定のコンピューターまたはユーザーによるビデオのダウンロードを禁止します。



さらに、アドビのスタッフによると、RSSを使用することで、パブリッシャーはテキスト広告、アニメーション広告、ビデオ広告の両方を表示することもできます。



アドビは、ブラウザのパン（Cookie、Cookie）を使用して、ダウンロードしたビデオを見ているユーザーを監視できるサーバーを提供します。 その後、この情報を出版社に送信できます。



さらに、Adobeは、プレーヤーのFlashビデオの忠実度が向上することを発表する予定です。



Adobe Media Playerは、同社の新しいプラットフォームであるApolloで作成されています。 春の終わりに、プレーヤーはベータテストを開始し、リリースは今年の終わりに予定されています。 同社はまた、Linuxバージョンをリリースし、より多くのフォーマットをサポートすることも検討しています。
