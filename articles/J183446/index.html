Rock It Sounds R-20の例での「強化」サウンドの詳細について

Rock It Soundsは、Rock It Sounds R-50デュアルドライバー「フィッティング」のおかげでポータブルオーディオ市場で定着していますが、バランスの取れたアーマチュアに基づいたシングルエミッターを備えた安価なモデルもあります。 この記事でご想像のとおり、R-20モデルに焦点を当てます。





►宣言された仕様

電気音響変換器 強化（単一ドライバー）
周波数範囲 20 Hz-18 kHz
インピーダンス 31オーム
感度 109 dB
ケーブル 1.3 mねじれ
プラグ 3.5 mmストレート金メッキ
ボディ素材 プラスチック


►梱包と設備

Rock It Sounds R-20は、きちんとした黒い段ボール箱に入っています。 前面には小さな窓があり、そこからヘッドフォンを見ることができます。



内部-シリコンチップのわずかなセット（3ペア）、飛行機でヘッドフォンを使用するためのアダプター（突然ですか？）、およびヘッドフォンを保管および輸送するためのハードケース。



ちなみに、カバーは非常に便利であることが判明しました。カバーは中実の織物でできており、内部には、たとえば追加のノズルやアダプター用の2つのセクションがあります。 他のメーカーも注意を払い、同様のカバーで製品を完成させる必要があります。



ノズルの最小数は、便利なケースと飛行機でヘッドフォンを使用するためのかなりまれなアダプターによって補正されます。



►外観と使いやすさ

誰もが光沢のある黒いプラスチックと硬質ゴムの共生を好むわけではないので、デザインは間違いなくアマチュア向けです。 同時に、仕上がりについての苦情はありません。 しっかりと取り付けられたハウジング部品。 左右のヘッドフォンを識別するための刻印はエンボス加工されており、審美的な観点からは見栄えがよくありません。



サウンドガイドは何にも覆われておらず、すべてがうまくいきますが、これに加えて、デバイスの価格はおそらく影響しますが、それらをクリーニングするデバイスはありません。 したがって、耳をきれいにすることを忘れないでください。



ワイヤーの長さは1.3メートルで、Y字型で、ねじれています。 かなりきれいに見えますが、数か月前に書いたSoundMagic E10の強化ケーブルと比較すると、混乱する傾向があります。



スプリッターブロックとプラグ-自信を刺激します。 固定用のスライダーがあります。



耳の後ろに着陸するため、ヘッドフォンが快適に座って脱落することはありません。 また、この装着方法の具体的な利点は、「マイク効果」がほとんどないことです。 しかし、いくつかの欠点がありました。長い音ガイドのため、ケースのほとんどは屋外にあり、その結果、ヘッドフォンは帽子の下に着用するのが不快になります。 遮音性は十分で、地下鉄には十分です。



►音

音質を評価するために、私は次の機器を使用しました： Cowon D20 MP3プレーヤー、 FiiO E10 DAC /アンプ、 Jiayu G2スマートフォン、およびロスレス録音。 ヘッドフォンの意図的な加熱は行われませんでした。



これらのヘッドフォンの第一印象は非常に複雑で、音はなんとなく「フラット」に見え、「フィッティング」の特徴であり、低音ではありません。 しかし、平均を超える音量で音楽を30〜40時間聴くと、​​音が変わりました。 それほど劇的ではありませんが、低音の量が増え、サウンドがサラウンドになり、全体的に明るくなりました。 これが魔法の加熱によるものなのか、耳の平凡な適応によるものなのかはわかりません。



ヘッドフォンは、ジャズ、クラシック、ライトエレクトロニクス、インディーロックが好きな人に最適です。もちろん、ポッドキャスト、オーディオブック、映画鑑賞でも問題はありません。



それにもかかわらず-私はジレンマを抱えています。 低音（ソフト）を備えたSoundMagic E10がありますが、同時に詳細なサウンドと優れたダイナミクスがあり、さらに詳細なサウンドを備えたRock It Souds R-20があります。中域と高音の優れた研究ですが、まだダイナミクスと低音はありません。 したがって、私は交互に耳を傾け、ポータブルサウンドの観点からより高いレベルに移行する予定です。



高感度のおかげで、ヘッドフォンはどのソースでも問題なく再生されます。 かなり騒がしい場所では、Cowon D20プレーヤーのボリュームレベルの26区分（40のうち）（チャネルあたり29 mW）で十分です。



►まとめ

Rock It Souds R-20は、バランスの取れたアーマチュアに基づいたシングルエミッターを備えた低予算のインイヤーヘッドフォンです。 彼らは価格を正当化する以上のものであり、「強化」サウンドを探索するのに最適です。



►費用

ウクライナでは、Rock It Sounds of Eraの公式代表者からヘッドフォンを購入できます。 執筆時点では、 それらのコストは309 UAHであり、これは約39ドルに相当します。 ロシアでの購入先-わかりませんが、EbayやAliexpressをまだキャンセルした人はいません。
