Ubuntu 9.04 Alphaリリース

Ubuntu 9.04の第2アルファ版（Jaunty Jackalope）がリリースされました。 イノベーション：ソフトウェアバージョンの継続的な更新、最新のテストスナップショットバージョンX Server 1.6 / X.Org 7.5、Firefox 3.1-betaおよびLinuxカーネル2.6.28-rc8の統合コード、オープンドライバーNouveauが含まれています。 テスト時には、独自のATI fglrxおよびNvidiaドライバーがX Server 1.6で動作せず、Intel i845およびi865チップに基づくビデオカードにも問題があることに留意してください。

開発の主な分野：起動時間の短縮、消費電力の最適化（アクティブなイーサネットでWifiを無効にするなど）、デスクトップ環境とWebサービスの統合、すべてのプロジェクトサブシステムの開発をBazaarソースコントロールシステムの使用に移行（すべてのUbuntu srcパッケージの場合、数週間個々のユーザーのホームディレクトリのコンテンツを暗号化し、ARMv7プラットフォームに基づいてモバイルデバイスのアセンブリを準備し、デフォルトで x86_64 PIE（Position Independent Executable）アセンブリ内の実行可能ファイル。



現在、実装予定の技術革新のページには、ユーザーとグループを管理するためのLDAP DIT（ディレクトリ情報ツリー）の作成、X.Org構成ファイル用のエディターの作成、ctrl-alt-backspaceを押してXサーバーの再起動を防止する3つの項目のみが含まれています。



3番目のアルファバージョンは1月15日にリリースされ、リリースは2009年4月23日にリリースされます。
