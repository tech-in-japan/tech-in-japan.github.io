夏 Windows 8. Application Excellenceラボ

前週はマイクロソフトからのニュースが豊富でした。これは、 Microsoft Surfaceの発表とWindows Phone 8の発表です 。 これらの発表は両方とも、Habrとブロゴスフィアに関する活発な議論を引き起こしました。



少し前の5月、 DevCon'12カンファレンスのオープニングで、Windowsストアでロシアの開発者による最初のアプリケーションを紹介しました。 本日、以下の発表の一部になりたい人を喜ばせたいと思います:)新しいシリーズのApplication Excellence Labsを開始しました。







そのため、アプリケーションエクセレンスラボは7月23日から7月27日までモスクワで開催されます。

注意！ リモートスクリプト可能（Lync、Skype）



アプリケーションエクセレンスラボ
Application Excellence Labsとは何ですか？
  1. ロシア語を話す専門家による無料相談。
  2. Windowsストアで最初の1つを取得する機会。


Application Excellence Labに参加するには何が必要ですか？

Visual Studio（Windows App Certification Kit）によって内部的に認定されたWindows 8用の動作するMetroスタイルアプリケーションが必要です。 アプリケーションはすべてのテストに完全に合格する必要はありませんが、テストが必要です。



Application Excellence Labsにアクセスする方法は？

Application Excellence Labsは、非常に限られた人数の非常にコンパクトな形式です。 したがって、実験室の仕事に着手したい人の間で、追加の選択が行われます：
  • 7月16日までに 、アプリケーションに「[W8 AEL] Name of your application」という見出しを付けてrush4apps@microsoft.comにアプリケーションのある手紙を送信します。 文字で示します
    • アプリケーションの説明（解決するタスクと対象者が設計されているもの）
    • アプリケーションのメイン画面のスクリーンショット
    • WACKによるテスト結果
    • 連絡先
  • 最適なアプリケーションの作成者には、Application Excellence Labsへの招待が届きます。


Application Excellence Labsに参加した結果、何が得られますか？
  1. アプリケーションの改善方法に関する推奨事項を含むレポート。
  2. 最高のアプリケーションの開発者は、最初にWindowsストアにアクセスできます！


Application Excellence Labs。 書式
誰もがWindowsストアのアプリケーションが本当に高品質であることを望んでいます。開発者として、あなたがまさにそれを実現できるように支援したいと考えています。 このために、研究室の仕事は組織化されており、その枠組みの中で、ロシア語を話すエンジニアとアプリケーションについて話し合い、専門家のフィードバックを得ることができ、その結果、優れたアプリケーションを優れたアプリケーションから作成する方法を理解することができます。



Application Excellenceラボの進め方：
  • タブレットなど、さまざまなシナリオでアプリケーションをテストします。
  • メトロスタイルに応じたアプリケーションの設計とレイアウトの分析。
  • レイアウトの分析とメニューの使用、レイアウト、タイルの「活性化」。
  • 契約実施例の分析。 検索、共有、設定。
  • アプリケーションの収益化戦略の議論。
  • 技術的な質問への回答。


念のために、アプリケーションエクセレンスラボは、7月23日から7月27日までモスクワで開催されます（リモートスクリプトが可能です）。
