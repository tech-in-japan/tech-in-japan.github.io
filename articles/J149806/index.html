802.11ac：新しいWi-Fi規格について知っておくべきこと

新しい802.11acベースのデバイスはすでに販売されている棚にあり、すぐにすべてのユーザーに質問がありますが、Wi-Fiの新しいバージョンを過剰に支払う価値はありますか？ この記事では、新しいテクノロジーに関する質問への回答をカバーするようにします。



802.11ac-バックグラウンド



最新の公式に承認された標準バージョン（802.11n）は2002年から2009年に開発されましたが、そのいわゆるドラフトバージョンは2007年に採用され、多くの人が覚えているように、802.11nドラフトをサポートするルーターはこのイベントのほぼ直後に販売中に見つかりました。



ルーターや他のWi-Fiデバイスの開発者は、プロトコルの最終バージョンの承認を待たずに、その時点でかなり成功しました。 これにより、2年前に最大300 Mb / sのデータ転送速度を提供するデバイスをリリースでき、標準が最終的に紙に印刷され、最初の100％標準化されたルーターが登場したとき、古いモジュールは互換性のための標準のドラフトバージョンに従って互換性を失いませんでした鉄のレベルで（わずかな違いはファームウェアを更新することで解決できます）。



802.11acでは、802.11nとほぼ同じ話が繰り返されます。 新しい標準の採用のタイミングはまだ正確にはわかっていません（おそらく2013年末よりも早いと思われます）が、すでに採用されているドラフト仕様は、現在リリースされているすべてのデバイスが将来、認定ワイヤレスネットワークで問題なく動作することを保証する可能性が非常に高いです



最近まで、新しいバージョンごとに802.11規格の最後に新しい文字（たとえば802.11g）が追加され、アルファベット順に成長しました。 ただし、2011年には、この伝統がわずかに違反され、802.11nのバージョンから802.11acに直接移行しました。



ドラフト802.11acは昨年10月に採用されましたが、これに基づく最初の商用デバイスは文字通り過去数か月に登場しました。 たとえば、シスコは2012年6月末に最初の802.11acルーターを発売しました。



802.11acの改善



802.11nでさえ、いくつかの実際のタスクでまだ明らかになっていないと断言できますが、これは進歩が止まっているという意味ではありません。 数年後にしか使用できないデータ転送速度の高速化に加えて、Wi-Fiの改善にはそれぞれ、信号の安定性の向上、カバレッジ範囲の拡大、消費電力の削減などの利点があります。 上記のすべては802.11acに当てはまるため、以下では各項目について詳しく説明します。



802.11acはワイヤレスネットワークの第5世代に属し、話し言葉では5G WiFiという名前に固定できますが、正式には間違っています。 この標準を開発する際の主な目標の1つは、ギガビットのデータ転送速度を達成することでした。 原則として、まだ使用されていない追加のチャネルを使用すると、802.11nを印象的な600 Mb / sまで加速できます（このため、4つのチャネルが使用され、それぞれが150 Mb / sの速度で動作します）、ギガビットバーそして、引き継ぐことは運命づけられず、この役割は彼の後継者に行きます。



示された速度（1ギガビット）を犠牲にすることなく、標準の以前のバージョンとの互換性を維持することを決定しました。 つまり、混在ネットワークでは、サポートする802.11のバージョンに関係なく、すべてのデバイスが機能します。



これを実現するために、802.11acは最大6 GHzの周波数で動作し続けます。 しかし、802.11nで2つの周波数（2.4および5 GHz）が一度に使用され、以前のリビジョンでは2.4 GHzのみであった場合、ACでは低周波数が取り消され、5 GHzのみが残ります。これは、データ伝送により効率的です。



2.4 GHzの周波数では、信号は長距離にわたって良好に伝搬し、障害物をより効率的に包み込むため、最後の発言は多少矛盾しているように見えるかもしれません。 ただし、この範囲はすでに膨大な数の「家庭用」波（Bluetoothデバイスから電子レンジやその他の家電まで）で占められており、実際に使用すると結果が悪化するだけです。



2.4 GHzを放棄するもう1つの理由は、この範囲では、それぞれ80〜160 MHzの幅を持つ十分な数のチャネルを収容するのに十分なスペクトルがないことです。



IEEEは、異なる動作周波数（2.4 GHzと5 GHz）にもかかわらず、ACリビジョンが標準の以前のバージョンと互換性があることを保証することを強調する必要があります。 これを実現する方法については詳しく説明しませんが、新しいチップは基本周波数として5 GHzを使用しますが、この範囲をサポートしない古いデバイスを使用する場合、より低い周波数に切り替えることができます。



スピード



802.11acの顕著な速度向上は、一度にいくつかの変更が行われるために得られます。 まず、チャネル幅が2倍になるためです。 802.11nで既に20 MHzから40 MHzに増加している場合、802.11acでは80 MHz（デフォルト）、場合によっては160 MHzになります。



802.11の以前のバージョン（N仕様まで）では、すべてのデータは1つのストリームでのみ送信されていました。 Nでは、それらの数は4になりますが、これまでのところ、最も頻繁に使用されるチャネルは2つだけです。 実際には、これは、合計最大速度が各チャネルの最大速度とその数の積として計算されることを意味します。 802.11nの場合、150 x 4 = 600 Mb / sになります。



802.11acはさらに進化しました。 現在、チャネルの数は8に増えており、それぞれの場合の最大可能伝送速度は、その幅に応じて見つけることができます。 160 MHzで866 Mb / sが得られ、この数値に8を掛けると、標準が提供できる最大理論速度、つまりほぼ7 GB / sが得られます。これは802.11nより23倍高速です。



最初は、すべてのチップがギガビット、さらに7ギガビットのデータ転送速度を提供できるわけではありません。 ルーターおよびその他のWi-Fiデバイスの最初のモデルは、控えめな速度で動作します。



たとえば、最初に説明した最初のCisco 802.11acルーターは802.11nの機能を超えていますが、それでも「ギガビット前」の範囲から外れることはなく、866 Mb / sしか実証されていません。 この場合、使用可能な2つのモデルの中で最も古いモデルについて説明し、最も若いモデルは600 Mb / sのみを提供します。



ただし、仕様に従って、ACの最小データ転送速度は450 MB / sであるため、速度は、最もエントリレベルのデバイスでも、これらの指標を大幅に下回ることはありません。



省エネ

省エネはACの最大の強みの1つです。 この技術に基づくチップは、すべてのモバイルデバイスで既に予言されており、これにより、同等だけでなく、より高速なデータ転送速度でも自律性が向上すると主張しています。



残念ながら、最初のデバイスがリリースされる前に、より正確な数値を取得できる可能性は低く、新しいモデルが手元にあるとき、ワイヤレスモジュールのみが異なる2つの同一のスマートフォンが市場に存在する可能性は低いため、増加した自律性をおおよそ比較することは可能です。 初夏に登場したAsus G75VWラップトップなど、最初のツバメはすでに地平線に見えていますが、こうしたデバイスは2012年末近くに大量に販売されるようになると予想されます。



Broadcomによると、新しいデバイスは、802.11nに基づいた対応デバイスと比較した場合、最大6倍のエネルギー効率です。 おそらく、ネットワーク機器のメーカーはいくつかのエキゾチックなテスト条件に言及しており、平均節約額は与えられたものよりもはるかに低くなりますが、それでもモバイルデバイスの数分、場合によっては数時間の操作という形で顕著に現れるはずです。



頻繁に起こるように、自律性の向上は、この場合マーケティングの動きではありません。なぜなら、それはテクノロジーの機能から直接続くからです。 たとえば、データがより高速で送信されるという事実は、すでにエネルギー消費を削減する理由です。 同じ量のデータを短時間で取得できるため、ワイヤレスモジュールは早くオフになり、バッテリーへのアクセスが停止します。



ビームフォーミング

この信号調整技術は802.11nでも使用できましたが、当時は標準化されておらず、さまざまなメーカーのネットワーク機器を使用する場合、通常は正しく機能しませんでした。 802.11acでは、bimforming作業のすべての側面が統合されているため、実際にははるかに頻繁に適用されますが、オプションのままです。



名前付きの手法は、さまざまなオブジェクトや表面からの反射によって引き起こされる信号電力の低下の問題を解決します。 レシーバーに到達すると、これらのすべての信号に位相シフトが生じるため、合計振幅が減少します。



ビームフォーミングは、この問題を次のように解決します。 送信機は、受信機の位置をほぼ決定し、この情報に基づいて、非標準的な方法で信号を生成します。 通常の動作では、受信機からの信号はすべての方向に均一に発散し、バイフォーミングが厳密に定義された方向に送信されると、複数のアンテナを使用して達成されます。



ビームフォーミングは、オープンエリアでの信号伝搬を改善するだけでなく、壁を「パンチ」するのにも役立ちます。 前にルーターがなかった場合

隣の部屋に「入れ」たり、低速で非常に不安定な通信を提供したりすると、ACを使用すると、同じポイントでの受信品質がはるかに良くなります。



802.11ad



802.11adには、802.11acと同様に、覚えやすいが、非公式の名前であるWiGigがあります。



名前にもかかわらず、この仕様は802.11acの隣にはなりません。 両方のテクノロジーが同時に開発され始め、それらには1つの主な目標があります（ギガビットの壁を克服する）。 異なるアプローチのみ。 ACが以前の開発との互換性を維持しようとする場合、ADは空白の用紙から開始するため、実装が大幅に簡素化されます。



競合する技術の主な違いは動作周波数であり、そこから他のすべての機能が続きます。 ADの場合、ACよりも1桁高く、5 GHzではなく60 GHzです。



これに関しては、2.4 GHzは言うまでもなく、802.11acの動作周波数よりも60 GHzの使用頻度が低いため、動作範囲（信号でカバーされる領域）も減少しますが、干渉ははるかに少なくなります。



802.11adデバイスが互いに正確に見える距離を言うのは困難です。 数字を特定せずに、公式筋は「同じ部屋内の比較的短い距離」について語っています。 信号経路に壁やその他の重大な障害物がないことも、作業の前提条件です。 明らかに、制限がBluetoothの場合と同じ制限（10メートル）である場合、数メートルについて、そして象徴的に話しています。



伝送半径が小さいと、ACテクノロジーとADテクノロジーが互いに競合しなくなります。 前者が家庭やオフィスの無線ネットワークを対象としている場合、後者は他の目的に使用されます。 どの詳細においても、この質問はまだ開かれていますが、ADが最終的にBluetoothに取って代わるといううわさがあります。今日の標準ではデータ転送速度が極端に低いため、仕事をしていません。



この規格は「有線接続の置き換え」にも位置付けられています。近い将来、「ワイヤレスUSB」として知られるようになり、プリンタ、ハードドライブ、モニター、その他の周辺機器の接続に使用される可能性があります。



ADの現在のドラフトバージョンは、元のターゲット（1 Gb / s）をすでに上回っており、最大データ転送速度は7 Gb / sです。 同時に、使用されている技術により、これらの指標を改善しながら、標準内にとどめることができます。



802.11acが一般ユーザーにとって意味するもの



テクノロジーが標準化される頃には、インターネットプロバイダーが関税プランの提供を開始し、その開示には802.11acのパワーが必要になることはまずありません。 したがって、最初は高速なWi-Fiの実際のアプリケーションはホームネットワークでのみ見つかります：デバイス間の高速ファイル転送、ネットワークに他のタスクをロードしながらHDムービーを見る、ルーターに直接接続された外部ハードドライブへのデータバックアップ。



802.11acは、速度の問題以上のものを解決します。 ワイヤレスネットワークの帯域幅が最大限に使用されていなくても、ルーターに接続された多数のデバイスが問題を引き起こす可能性があります。 各ファミリのそのようなデバイスの数が増えるだけなので、ここで問題について考える必要があります。ACがそのソリューションであり、1つのネットワークで多数のワイヤレスデバイスを使用できます。



ACを拡散する最も速い方法は、モバイルデバイス環境です。 新しいチップが自律性を少なくとも10％向上させる場合、デバイスの価格がわずかに上昇しても、その使用は完全に正当化されます。 ACテクノロジーに基づいた最初のスマートフォンとタブレットは、年末近くに待機する可能性が最も高くなります。 すでに述べたように、802.11acを搭載したラップトップはすでにリリースされていますが、私たちが知る限り、これは市場で唯一のモデルです。



予想どおり、最初のACルーターのコストは非常に高く、特に802.11nで状況がどのように発展したかを思い出すと、今後数か月の価格の急激な低下は待つ価値がほとんどありません。 ただし、来年の初めには、ルーターの価格は150〜200ドル未満になり、メーカーは現在、最初のモデルを求めています。



少量の情報の漏洩によると、Appleは再び新技術の最初の支持者の1つになります。 Wi-Fiは常に会社のすべてのデバイスの主要なインターフェースでした。たとえば、2007年にドラフト仕様が承認された直後に802.11nがAppleテクノロジーに組み込まれたため、802.11acが多くのAppleデバイスの差し迫ったデビューに備えていることも驚くには当たりません。ノートパソコン、Apple TV、AirMac、Time Capsule、そしておそらくiPhone / iPad。



結論として、言及されたすべての速度が理論的に最大限に達成可能であることを思い出してください。 また、802.11nが実際に300 Mb / sよりも低速で実行されるように、ACの実際の速度制限もデバイスに示されているものよりも低くなります。



各ケースのパフォーマンスは、使用する機器、他のワイヤレスデバイスの可用性、部屋の構成に大きく依存しますが、暫定的に、碑文1.3 Gb / sのルーターは、800 Mb / s（これはまだ802.11nの理論的な最大値よりも著しく高い）を超えて情報を送信できません。
