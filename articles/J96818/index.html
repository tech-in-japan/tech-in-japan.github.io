CardapioはUbuntuのメインメニューの優れた代替品です

コンパクトで便利な多機能メニュー！ 私はいくつかの選択肢を試しましたが、それらの速度は貧弱です。 これには、あらゆる種類のトピック、透明性、その他の美しいものがロードされていません。 それはちょうど快適です。 Ubuntuの代替メインメニューであるCardapioについてお話します。



これは次のようなものです。







主な利点のうち、注目に値するものは次のとおりです。

  • Google検索との統合により、アプリケーションだけでなく、インターネット上の結果も検索できます。
  • リストの一番上に最も必要なアプリケーションを添付する機能（メニュー項目-[この項目をピン留め]を右クリック）
  • メニューアプレットを任意のパネル（垂直、水平）に配置し、パネルのサイズに応じてアイコンのサイズを自動的に変更する機能
  • メニューの名前とアイコンを簡単に変更する機能
  • メニューをサイズを保存するシンプルなウィンドウとしてサイズ変更する機能
  • アドオンのサポート（キットで提供されているもの以外は見つかりませんでした）








設置





Cardapioには公式のPPAがあります。



sudo add-apt-repository ppa:tvst-hotmail/cardapio

sudo apt-get update && sudo apt-get install python-keybinder cardapio









インストール後、CardapioはGnomeパネルのアプレットリストで利用可能になります。 追加するには：パネルを右クリック-> パネルに追加 ...- >リストからCardapioを選択-> 追加
