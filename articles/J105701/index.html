Mono 2.8リリース

.NET Mono 2.8プラットフォームの無料実装の新しいバージョンがリリースされました 。 このバージョンは、大幅な改善をもたらす重要なアップデートです。
  • C＃4.0のサポート。
  • 新しい生産的な世代別ガベージコレクター （下）。
  • 新しいコンポーネント：
    • ASP.NET 4.0
    • 並列フレームワークとPLINQ。
    • System.xaml;
    • System.Dynamic;
    • MEF
    • MVC 2;
    • System.Data.Services.Client（OData）;
    • WCFルーティング。
    • CodeContracts
    • System.Numerics
    • LINQ to SQL
    • DLRはパッケージに含まれています。
  • 埋め込みAPIの2番目のバージョン。
  • OpenBSDのサポート。
革新の詳細な説明はこのリンクで利用可能です。 このアドレスから Mono 2.8をダウンロードできます 。
