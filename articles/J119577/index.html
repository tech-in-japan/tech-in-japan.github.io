Tech-ed North America 2011のすべての重要な発表

米国のこの時点で、Microsoft Tech-ed North America 2011の主要な会議が終了し、その間、Microsoftはいくつかの重要な発表を行いました。 多くの発表が行われた会議の初日の全体報告書は、 このアドレスで入手できます 。







Visual StudioとTFS


Visual Studio 2010のリリース以来、いくつかの結果が発表されており、Visual Studio 2010は製品の歴史の中で最も売れているスタジオになりました。







次世代のVisual Studioのいくつかの革新が紹介されました。 その1つとして、新しいStoryboard Assistantツールが導入され、使い慣れたビジネスアプリケーションであるPowerPointを使用してインターフェイスのプロトタイプをすばやく作成できるように設計されました。 新しいPowerPointアドオンにより、トレーニングを受けていない人でも、アプリケーションのインターフェイスをすばやくスケッチできます。







Team Foundation Serverに革新がもたらされます。 TFS用の更新されたWebインターフェイスが示されました。 新しいインターフェイスは、新しいデザインだけでなく、要素のドラッグアンドドロップをサポートし、サーバー上の変更をリアルタイムで表示します。







TFS Webインターフェースの革新の中で、新しいタスクボードツールが発表されました。これにより、タスクの追跡が改善されます。







新しいWebインターフェイスでは、タッチパネルを使用して、タスクボードの要素を手で操作できます。



新しいTeam Navigatorツールの動作メカニズムが実証されました。これにより、開発チームのタスクを操作でき、作業環境の現在の状態（開いているプロジェクト、ファイル、パネル、タブ）を保存し、環境を復元するなどの機能を提供します。 この便利な革新は、ある新しいタスクの作業を中断して、別の古いタスクをすばやく調整する必要がある場合に役立ちます。



Visual Studioと単体テストツールの緊密な統合が発表されました。 すぐに使用可能なNunit、xunit、およびMS Testは、リストを展開するオプションでサポートされます。







サイトhttp://www.microsoft.com/visualstudio/en-us/roadmapは、開発ライフサイクル管理ツールに関して、次世代のVisual Studioの将来に関する詳細情報を提供します。 革新の詳細な説明は 、Jason Zanderブログでも入手できます。



新しいSystem Center Connector CTPツールが導入され、System Center Operations Manager管理者からのインシデント管理機能をTFS環境に統合できるようになりました。 System Centerに登録されたインシデントを理解している管理者は、問題を開発チームに簡単にエスカレートできます。開発チームは、多くの有用なデバッグ情報を含むTFS要素としてインシデントを受け取ります。



チャンネル9は、C ++開発者向けの新しいVisual Studio機能を示すビデオを投稿しました。



Team Foundation ServerのJava SDKがリリースされました 。これにより、Javaコンポーネントを使用してオンデマンドTFSを拡張できます。



Windows Azure


会議用にWindows Phone 7 v1.2用のWindows Azure Toolkitの更新バージョンがリリースされました。詳細については、こちらをご覧ください 。







マイクロソフトのWebサイトは、プライベートおよびパブリッククラウドテクノロジーを使用したコスト削減に関する調査結果を公開しています（右図）。



Windows Azure AppFabric SDK V2.0 CTP（5月） の新しいバージョンが発表され 、ダウンロード可能になりました 。詳細については、 こちらをご覧ください 。 さらに、 カンファレンスは 、6月にWindows Azure AppFabric SDKの次のバージョンをリリースする計画を発表しました。これには、マルチレベルアプリケーションを構築するための新しいツールが含まれるはずです。

  • AppFabric開発者ツール-クラウドアプリケーションの視覚化、設計、構築のためのVisual Studioツールの改善。
  • AppFabric Application Manager-AppFabricアプリケーションの自動展開、監視、管理の新しい機会を提供します。
  • .NET拡張機能の構成モデルのセット。


Azure SQL Cloud Databaseの新機能が発表されました。これらは後日利用可能になります。

  • SQL Azure管理REST API- Azure SQLサーバーを管理するためのWeb API。
  • 単一のサブスクリプション内で複数のAzure SQLサーバーを作成するためのサポート。
  • SQL Azure用のJDBCドライバーの新しいバージョン。
  • DACフレームワーク1.1
  • サーバーからクラウドへのデータベースの転送を容易にするための管理ポータルでのインポート/エクスポート機能の統合。
  • ヒューストンツールを含むポータル上のデータベースを管理する機能の改善、およびOdataプロトコルを介してSQL Azureデータベースを管理するための新しいサービス。


SQL Server Denali






コードネーム「Denali」という新世代のSQL Serverの次の予備バージョンが、2011年夏頃に利用可能になることが発表されました。 最初にローンチについて知りたい人は、すでにウェブサイトhttp://www.sqlserverlaunch.com/で通知にサインアップできます 。



SQL Server Denaliの新機能には次のものがあります。

  • SQL Server AlwaysOn-サーバーの復元力を強化し、使いやすく、TCOを削減するための新しいメカニズム。
  • プロジェクトコードネーム「Apollo」 -クエリの実行速度を大幅に向上させる新しいデータストレージテクノロジー。
  • SQL Server開発ツールプロジェクトのコードネーム「Juneau」 -データベース、BI、およびWebベースのソリューションを開発するための単一の環境。
  • プロジェクトコードネーム「Crescent」-ビジネスユーザーとマネージャーのデータを視覚化する機能を備えたデータベースにすばやくアクセスするための対話型ツール。
  • Microsoftが買収したZoomixのテクノロジーに基づいた、新しいSQL Server Data Quality Servicesを含む新しい開発ツール。
  • ビジネスインテリジェンスセマンティックモード -多次元および表形式のプロジェクトに基づいて最終製品を作成するための単一のBIモデル。


PowerPivotの更新バージョンを使用すると、Visual Studioに統合し、スタジオ内でデータを操作できます。 PowerPivotは、Excelで20億行のデータを処理することが示されています。







Windows Phone 7


全体報告では、今後のWindows Phoneの「Mango」アップデートに登場する多くの機能の一部が示されました 。

  • すべての機能にアクセスできる完全なLyncクライアント。
  • アプリケーションだけでなく、メインウィンドウでこれらのアプリケーションの特定の部分のタイルを固定する機能、たとえば特定のExchangeメールボックスフォルダーを固定すると、すぐにアクセスできるようになりました。
  • メールクライアントWindows Phone 7のOutlookのような会話のサポート。
  • 保護された手紙へのアクセスと閲覧のサポート。
  • サーバー上のメールを検索します。
  • Windows Phone Marketplaceから企業の電話へのアプリケーションの配布をターゲットにします。
  • すぐに使えるOffice 365のサポート。


ハイパーv






最大の銀行および金融会社HSBCは 、プライベートクラウドテクノロジーに基づく10万の仮想化VDIデスクトップと、Windows 7に基づくマイクロソフトのHyper-V を実装することが発表されました。これは、史上最大の仮想化テクノロジーの実装になります。 さらに、社内のすべてのSQL ServerおよびSharePointサーバーもHyper-Vに基づいて仮想化されます。



NetAppとCiscoがHyper-V Cloud Fast Trackプログラムのマイクロソフトパートナーになったことが発表されました。



会議中に、MicrosoftはHyper-Vプラットフォームでの仮想化のオペレーティングシステムとしてCentOSを公式にサポートしていることが発表されました。 公式声明と詳細情報はこちらにあります 。 この記事では、ロシア語で詳細な説明を提供します。



システムセンター






System Center 2012の新世代は、iOS（iPhone、iPad）、Android、およびSymbianベースのデバイスの管理を完全にサポートすることが発表されました。



System Center Orchestratorツール（旧称Opalis）のデモが行われました。これにより、ワークフローを使用してITタスクを自動化できます。 System Center Orchestratorのベータ版は、2011年6月にリリースされます。



クライアントマルウェア対策および攻撃保護システムの新しいバージョンが会議用にリリースされました 。 Forefront Endpoint Protection 2010は、セキュリティ機能を強化し、System Center 2012と統合します。ベータ版は、ダウンロードとテストが可能になりました。
