Acronis Backup Russia-ファーストルック

この記事では、 Acronis Backup Russiaと呼ばれる、 Acronisデータ保護製品の新世代の代表者（シンプルな「バックアップ」）の機能について説明します。 バックアップのトピックはありふれたものであり、ハックされているため、バックアップ自体の利点には焦点を当てませんが、この製品と従来のソリューションの違いについて説明します。







建築



一見すると、製品のコンポーネントのセットは、従来のソリューションと同じです。



-管理サーバー（Acronis管理サーバー）;

-Webベースの管理コンソール。

-特定のシステム（Windows、Linux、Mac、VMware vSphere、Microsoft Hyper-Vなど）のバックアップエージェント。



ただし、次のような多くの重要な違い/機能があります。



1）最初に、集中管理サーバーは物理的にクラウド（ロシアのアクロニス専用データセンター）に配置されます。

これは巨大な分散型の高可用性システムであり、そのコピーは世界中のデータセンターに展開されています。

エンドユーザーは、管理サーバーをインストールして、ニーズに合わせて構成する必要はありません。 これにより、分散環境の問題も解決されます。たとえば、ユーザーが従属する複数のオフィスを持ち、各オフィスがVPNを介して本社に接続されている場合です。 このような場合、集中管理サーバーをセットアップすることは多くの場合、重要なタスクになり、システム管理者の多大な努力が必要です。



2）次に、管理コンソールは、インターネットに接続された任意のマシンからアクセス可能な完全なWebアプリケーションです。 Webベースの管理コンソールは、ExtJSフレームワークを使用して記述されたJavaScriptアプリケーションです。



ユーザーの観点からは、Webコンソールに移動し、適切なエージェントをダウンロードしてインストールするだけです。 インストール中に、管理サーバーでの正しいバインドのためにユーザーアカウントが要求されるため、追加の設定は必要ありません。



多くのユーザーは、従来はデスクトップであったアプリケーションのWebインターフェースに一定の不信感を抱いています。 多くの点で、この偏見は、vSphere Web Clientを使用したVMwareのおかげで発展しました。バージョン6.0でも私は個人的に拒否を引き起こします（ただし、以前のバージョンよりもはるかに良くなっています）。 インターフェースでは、ユーザーエクスペリエンス（UX）の観点から、デスクトップバージョンでの作業と可能な限り類似した欠点を考慮して、Webコンソールでの作業を試みました。







3）3番目に、バックアップエージェントは管理サーバーへの永続的な接続を必要とせず、自律的に動作できます。

これは、バックアップタスクがエージェントによって「個人的に」受信されるという事実により達成されます。エージェントは、TLS 1.2および追加の暗号化を使用する安全なHTTPSプロトコルを使用して管理サーバーに接続し、タスクおよびバックアップ計画を受信します。 したがって、管理サーバーがエージェントに直接接続することはありません。イニシアチブは完全にエージェントにかかっています。 管理サーバーとの接続がない場合、エージェントは最後の正常な接続中に受信した計画に従って引き続き動作します。



機能性



製品には、高度なバックアップソリューションに期待されるすべての標準機能が含まれています。



•ユーザーのサーバーまたはAcronis（Acronis Cloud）を実行しているクラウドにアーカイブをローカルに保存します。

•アーカイブの暗号化（AES-256およびGOSTを含む）。

•マシン全体と個々のファイルのバックアップ。

•ベアメタルでの復旧（ベアメタル復旧）。

•あらゆる種類のアーカイブからのきめ細かいファイルリカバリ。

•専用エージェントを使用したMicrosoftアプリケーションのサポート（Microsoft SQL、Exchange、SharePointの個々の要素の回復）。

•vSphereおよびHyper-V上の仮想マシンの「エージェントレス」バックアップ。

•その他。

互換性と拡張性



分散アーキテクチャのおかげで、新しい機能の追加は大幅に簡素化されています。 サーバー側のアップグレードは、ユーザーの介入なしで、Acronis側で完全に実行されますが、古いバージョンのエージェントとの下位互換性は維持されます。 つまり、ユーザーはバックアップソリューション全体を更新するという質問に煩わされる必要はありません。新しい機能はサーバー側に自動的に表示され、それにアクセスするには、エージェントを更新するだけです。これは、Webコンソールの[更新]



結論



Acronis Backup Russiaは、最小限の実装コストで済む便利なバックアップソリューションを探している人に最適です。

この製品は、主に中小企業にとって興味深いものですが、もちろん、大規模環境向けのソリューションもあります。
