Microsoft Virtual CD-ROMコントロールパネル

ある程度後悔しているので、良い人は光学ドライブやレーザーディスクイメージとやり取りするための大きなプログラムセット（たとえば、Nero またはアルコール120％プログラムのセット）をたくらむ傾向があることに気づかなければならないことがあります。 、 ISOイメージ （ CD-ROM またはDVD-ROMの正確なコピー） を 仮想ディスク として接続するだけです。 つまり 、Windowsの このイメージ を、ディスクが挿入さ れた追加のCDドライブ （DVDドライブ）のように見せます。



海賊版プログラムが光学ドライブやレーザーディスクではなく、無料のもの （CDBurnerXPやImgBurnなど）で動作する場合、このプログラムは単純すぎる、つまり通常はディスクイメージを作成でき、他のプログラムで作成されたイメージを変換することもできます（たとえば、NRG形式からISO形式を作成できます）が、仮想ディスクの接続には対応できません。



Windows XP HomeまたはWindows XP Professionalでのこれらの問題はどちらも、MicrosoftがWebサイトから無料で配布しているシンプルで非常に小さなアーカイブファイルを使用して解決できます。 そこにあるハイパーリンク「Microsoft Virtual CD-ROMコントロールパネルパッケージを今すぐダウンロード」をクリックするだけです。 このアーカイブは自己解凍型で、 59キロバイト （！）が必要です。インストール手順が記載されたテキストファイル、 SYSTEM32 / DRIVERSディレクトリに コピー する SYSファイル（このSYSファイルには仮想ディスクドライバー自体が含まれます ）、 およびEXEファイルの 3つのファイルが含まれます ドライバー （インストール、アンインストール） と仮想光学ドライブに「プッシュ」されるISOイメージの 両方 を管理するためのコンソール（挿入、削除、新しいドライブ文字の作成、最後に挿入されたイメージの確認など）さらに）。



（ドライバーはISOイメージのみを認識しますが、前述のCDBurnerXPを使用して、たとえば他のイメージをISOに変換できます。）



残っているのは、MicrosoftがMicrosoft Windows XPにこのドライバーを事前に含めておらず、VistaとWindows 7の類似物が近くにないという事実に驚嘆することです。たとえば、sidyushnikov。
