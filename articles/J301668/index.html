10年間のHabrahabr：ブログのトップ10の人気記事

今日、Habrahabr は10歳です -これは懐かしさにふける理由ではありませんか？ 私たちは、最も暖かい瞬間を思い出し、ブログの人気トップ10の記事をその存続期間全体にわたってまとめました。







10. トレーニングコース。 ASP.NET MVCアプリケーションのEntity Frameworkデータモデルの作成



5年前、Contoso UniversityのWebアプリケーションの段階的な作成に関する資料を、C＃の例を使用して翻訳しました。 コースを修了すると、大学向けのシンプルなWebサイトを開発できます。ユーザーは、学生、コース、教師に関するデータを表示および更新できます。 コメントから判断すると、この記事は本当に有用であることが判明しました。



9. Windows 8のWindows To Goテクノロジー



Windows 8のリリース後、新しいオペレーティングシステムに関する多数の記事を執筆しました。 その1つは、起動可能なUSBドライブ（フラッシュドライブまたは外付けHDD）からWindows 8を起動する方法に関するものです。メディアとホストの要件、WTG展開オプション、Windows To Go Creatorウィザードを使用してイメージを作成する手順を説明します。のみ。



8. Windows 8.1およびVisual Studio 2013はダウンロード可能です



この記事は、 DreamSparkおよびBizSparkプログラムの参加者である学生とスタートアップの心をつかみました 。 その中で、Windows 8からWindows 8.1にアップグレードし、Visual Studio 2013をダウンロードするためのホットリンクを共有しました。



7. SkyDriveクラウドストレージが大きな更新を受け取りました



この記事では、SkyDriveクラウドストレージの更新に関する詳細を説明しました。 しかし、それだけではありません。記事の最後にちょっとしたボーナスがありました。25ギガバイトの無料ストレージを取得する方法について話しました。



6. 2015年のJavaScriptトレンド



2015年の初めに、JavaScriptの傾向について説明しました。 今、この資料をもう一度読んで、どの予測が正しいことが判明したかを見るのは非常に興味深いです。 PS記事の最後に書いたように、2015年は本当に楽しかったです！



5. Windows Server 2012 R2の新機能



この資料の主な目標は、新しいサーバーが何を面白くするかについての一般的なアイデアを提供することでした。そして、特定のタスクに最も適切なものを自分で決めることができました。



4. Visual Studio Express-優れた無料のWeb開発ツール



Visual Studio 2013を使用してWebサイトを簡単に作成する方法に関する詳細な資料。この記事のフレームワークでは、そのコメントの1つを思い出せずにはいられません。



3. DirectX 12



2014年3月20日に、GDC 2014カンファレンスの一環として、DirectX 12の次のバージョンの発表が行われましたが、その後、多くの質問がありました。 DirectX 12の新機能について。



2. Windows 10 Technical Previewをダウンロードできます



Topの最初の2つの記事は、予想どおり、Windows 10の発表に関するものでした。初期の段階では、Windows 10 Technical Previewの可用性に関する短い説明をリリースし、予備バージョンのインストール方法について説明しました。



1. MicrosoftはWindows 10を導入しました
