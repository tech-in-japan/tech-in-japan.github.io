NEMA暗号化マシン

第二次世界大戦中、スイス軍は商業用のエニグマKの改良版を使用しました。これはスイスKと呼ばれることもあります。



NEMA（「Neue Maschine」の略）は、第二次世界大戦中にZellweger AG（スイス、ウスター）によって開発され、有名なドイツのエニグマに代わる電気機械暗号化マシンです。 このマシンは、TDおよびNEMA Modell 45とも呼ばれます。



一見したところ、機械には10個のローターがありましたが、そのうち5個だけが電気回路で接続されていました。 また、5つのうち4つは、各側に26ピンのローターをコーディングしていました。 5番目のディスクは反射型で、暗号化中に移動できます（リフレクターが修正されたエニグマとは異なります）。



NEMAは、オープンメッセージを入力するための26文字（AZ）キーボードと、暗号文の目的の文字を強調表示する26個の電球で構成されていました。 しかし、エニグマと比較して、NEMAには文字と数字の間の追加のスイッチがありました。 それらは、電子印刷デバイスへの接続に使用されました。 NEMAでは、暗号化ローターの不規則な増分回転（余分なホイールの追加による）などの設計改善が行われました。 暗号のハッキングは特に困難でした。







NEMAマシンの開発は、ベルンのインストラクター、アーサー・アルダー（キャプテンアーサー・アルダー）が率いる専門家チームによって1941年から1943年にかけて行われました。 開発者の構成には、分析数学の教授Hugo Hadwiger（Hugo Hadwiger）、数学の教授Heinrich Emil Weber（Heinrich Emil Weber）およびPaul Glur（Paul Glur）が含まれます。



最初のNEMAプロトタイプは1944年初頭に導入されましたが、わずか1年後に（いくつかの修正と改善を行った後）承認を受けました。 1946年、機械の生産が始まりました。 しかし、このモデルは1947年にのみ運用を開始しました。これは軍事面でやや遅れました。 NEMAは、戦後、スイス軍と外務省（外交部）によって使用されましたが、HagelinやGretagなどのより高度な暗号化デバイスに置き換えられました。

NEMAモデル



NEMA暗号化マシンは、3つの基本タイプのモデルで作成されました。外務省の訓練、軍事、および機械です。 これらのモデルのほとんどの操作の外部の同一性と類似性にもかかわらず、いくつかの大きな違いにより、マシンの動作が反対になりました。



トレーニングマシン


このモデルは、オペレーターのトレーニングに使用されました。 マシンとその動作原理に関するデータ漏洩のリスクを回避するために、トレーニングインスタンスは実際の軍事モデルとは異なりました。 現存するNEMAのほとんどは、かなり長い間、集中的にトレーニングに使用されているため、このタイプです。 ドイツ語では、モデルはSchulmaschine（ドイツ語-スクールカー）として知られています。 サンプルにはラベルがあります：

外務省の機械


NEMAのこのバージョンは、スイス外務省（外交局）によってのみ使用されました。 私たちが知る限り、このモデルの車は一般に公開されていません。 しかし、「外交的」NEMAは、異なる速度のローターとディスク、および異なる構成のステップメカニズムを使用して製造されていることがわかりました。



動作原理



NEMAの仕事は、エニグマの仕事によく似ていました。 スイス車にのみ、パッチパネルはありませんでした。 代わりに、ローターのノッチの数が増加しました。 キーボードのキーを押すと、回路が閉じられます。 4.5Vバッテリーから、電流がさまざまな回路を通過し、その結果、コードの目的の文字を表示する電球のセットの1つがオンになりました。 電気回路の絶え間ない変更により、複数英数字の置換暗号を作成することが可能になりました。 これにより、安定性と信頼性が向上しました。 エニグマとは異なり、NEMAローターの移動メカニズムはより複雑でした。







NEMAは、リフレクターを使用してワイヤーのペアを接続しました。 その存在はインボリューションによって保証されました。つまり、復号化は暗号化と同じでした。



10個のNEMAコーディングローターの動作をさらに詳しく調べるには、ヒンジカバー（黄色のラベルの左側）を持ち上げる必要がありました。 赤である入力ローター（ETW）を介してディスクに電流が流れました。 すべてのディスクにマークが付けられました-アルファベットの黄色文字。







右側の極端なディスクはリフレクター（UKW）であり、リフレクターとして使用され、26の静的接点で構成されています。 リフレクターと入力ローターの間には、4組の中間ディスクがありました。 各ペアは、両側に26個の接点を持つ電気式コーディングローターと、ローターの動きを制御する機械式ホイールで構成されていました。 各電動ディスクは、任意のステップローターと組み合わせることができます。 ピンおよびフラットコンタクトは、アルファベットの文字に対応していました。 接触すると、隣接するローターの接点が電気回路を閉じました。 ローターの内部では、各ピン接点が平らなピン接点の1つに接続されていました。 各ローターには独自の接続順序があります。







ローターを動かすために赤いレバーが使用されました。

ローターのステップ運動



NEMAローターの段階的な移動は複雑な方法で行われました。 ローターは、幅の狭いスピンドルとドッグを使用して移動しました。 上方への動きにより、ローターが1ステップ移動しました。 その結果、彼は後方に移動しただけです（ Z→Y→X→Wなど）。







設計の複雑さをさらに高めるため、4本の調整ネジが使用されました。







このようなネジの調整には、特別なツールとトレーニングが必要でした。 同様の構成が軍用車両で使用されました。



暗号鍵



NEMA暗号のキーを選択するときは、内部キーと外部キーを考慮する必要があります。 内部キーにより、暗号化ディスクに接続されているステッピングローターが決まりました。 例：



15-C / 12-A / 14-D / 13-B



外部キーは、メッセージの最初のローターの初期位置を決定しました。 例：



BXLRRTVYLZ



レバーを左に動かして最後の位置にすると、初期位置が変わりました。 適切なキーを選択して、10個すべてのローターを前後に動かすことができました。



ほとんどのキーは、しばらく（通常は約1日）だけ保存されていました。 新しいメッセージについては、ローターの新しい開始位置も尋ねられました。 彼らは暗号を可能な限り保護するためにこれを行いました。 実際、同じ設定で十分な数のメッセージが送信された場合、有能な暗号解読者（いくつかのメッセージも慎重に調査した）は、周波数分析を使用してメッセージの暗号を選択できます。 同様のアイデアが使用された初期化ベクトルの原理を思い出す価値があります。 暗号文とともに、暗号化されたテキストの前に初期位置が送信されました。 この原則は「インジケータ手順」と呼ばれていました。 実際、このような弱点は、エニグマのコードを破るのに役立ちました。



ローターは、かなり単純なタイプの暗号化を作成しました。元の平文の要素が要件に従って暗号化されたテキストに置き換えられた基本的な置換暗号です。



可能な開始位置の総数の計算は比較的簡単でした。 軍用機が使用された場合、6つの暗号ローター（AF）があり、そのうち4つが車内にありました。 出力は、360個のローターの組み合わせ（6×5×4×3）でした。 ステップローターでも同じことが起こりました（360種類）。 10個のローターの可能な開始位置の総数の計算は26 10でした。

NEMAのメッセージの暗号化と復号化は、必要に応じてマシンに接続されたランプパネルを使用して、文字で読み取ることができます。







1992年7月9日、NEMA暗号化マシンは正式に機密解除されました。 数年後の1994年5月4日に、トレーニングと運用のNEMAが売りに出されました。 その結果、モデルは暗号化マシンコレクターの獲得ランクに加わりました。
