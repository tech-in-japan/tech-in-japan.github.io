PC上のCisco ASA？ 可能です！

最近、 リソース （サイト名のアンダースコアに注意してください！）に出会いました。その作成者は、Cisco ASA 5500の機能を通常のPCにうまく移行しました。



Cisco ASAが本質的にLinuxベースであり、完全にx86互換であることは秘密ではありません。 どうやら、この事実は多くのオタクのオープニングチャンスについての空想を喚起し、最終的に、2008年、愛好家のグループがCisco ASAの最初のインストールイメージを世界に与えました。 それ以来、プロジェクトは存在し続け、新しいリリースをリリースしています。

カットされた結果の製品の機能の簡単な説明。





イメージは、仮想環境と実際のハードウェアの両方で起動できます。 ハードウェア要件は次のとおりです。

  • 256MB以上のRAM
  • 110MB以上のHD
  • ネットワークカードIntel Pro / 100、Intel Pro / 1000、AMD PCNET32
  • cdrom


結果のデザインは、シスコ試験のテスト、学習、準備に最適です。 サイトでの帯域幅テストの結果からも明らかなように、無謀さはありますが、戦闘で使用できます。 それらからの簡単な抜粋が添付されています：



試験装置：


  • PIII 500スロット1 CPU
  • 256M SDRAM、PC100
  • Intel 82558b NIC（2ポート）
  • インテル82559 nic


結果：


NAT TCP



NAT UDP



WebVPN（クライアントからのトラフィック、IPSECなし）



PS近い将来、qemu / dinagenを使用してCisco IPSで回線をエミュレートした経験を説明する予定です。 それはかなり実現可能であることが判明しました。
