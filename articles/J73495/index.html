wavesandbox.comのアカウントを持つラッキーな人への別のボーナス

GoogleからGoogle Wave Developer Sandboxへの招待状を受け取ってから12日が経過し、適切なフォームに記入しました（興味のある人は誰でも、 wave.google.com / signups / signup1.htmlのように見えます ）。 アクティブ化されたアカウントを待っている間、私はなんとかGoogle Wave Previewの招待状を受け取り、そこですべてを学び、たくさんプレイしました。 そのため、今日、ついにwaveandboxでアクティベートされたアカウントを手に入れたとき、私はWavesを真っ逆さまに急ぐことはしませんでしたが、それが新しいものをもたらすことに興味を持ちました。



空の連絡先リストを見て、「連絡先の管理」リンクをクリックしました。 GmailとGoogle Appsのユーザーが持っている標準の連絡先エディターを持っています。 そして、WavesがオンになっているGoogle Appsの特別エディションが、wavesandbox.comドメインでアクティブになっているという長年の推測（サンドボックス内のURLの形式に基づく）を確認しました。

ここで、いくつかの結論を出すことができます。

  1. 無料の水泳でWavesがリリースされた後、誰でもマウスを数回クリックするだけで、ドメインのGoogle Appsでサーバーをオンにできます。
  2. Wave Sandboxアカウントと一緒に、wavesandbox.comでGmail、カレンダー、ドキュメントにアクセスできました


そして、最初のものが非常に論理的で予想される場合、2番目のものから何が得られるかは完全には明らかではありません。 ただし、GoogleサービスのPremierおよびEduリリースでは、 Googleドキュメントでjavascriptを実行する機能が追加されたことをすぐに思い出しました（ Google Appsスクリプト：Google Appsのユーザースクリプト、Google Appsでスクリプトが作成されました ）。 Basic Editionのサービスではこの機会を感じられないことに非常に怒っていたときのことを覚えています。 したがって、最初に、スプレッドシートに行きました。 そして、ここで彼はツールメニューの切望されたスクリプトアイテムです：
