Ubuntu Oneは無料ストレージを増やし、100万人のユーザーにリーチ

... 2番目は5月に以前に発生しましたが。 しかし、両方のイベントが今発表されました。

知らない人にとっては、 Ubuntu OneはDropboxの類似物であり、クラウドストレージであり、主に異なるコンピューター間でデータを同期することを目的としています。 名前にもかかわらず、Ubuntu OneはさまざまなOS（Ubuntu、Android、iOS、Windowsベータ版）をサポートしており、モバイルシステムでは、クラウドから音楽を1か月あたり3.99ドルでストリーミングする機能があります。

そのため、無料ユーザーのストレージ容量は2GBから5GBに増えました。 この計画は現在、Ubuntu One Freeと呼ばれています。 20GBごとのボリュームの増加には2.99ドル（月あたり）かかります。



Ubuntu Oneアプリ開発者プログラムも開始されました。その目的は、開発者（すべてのプラットフォーム）に、Ubuntu Oneの開発とアプリケーションへのU1の統合に関する最大限の情報とサポートを提供することです。
