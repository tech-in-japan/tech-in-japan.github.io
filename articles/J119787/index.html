管理者権限を持つアプリケーションのクイック起動



Windows 7で任意のコマンドを実行するためのウィンドウは残っているという事実にもかかわらず（ただし、デフォルトでは Win-Rキーの組み合わせでのみ使用可能です）、スタートメニューの検索バーに置き換えられます。 その助けを借りて、新しい、少し不便なメニューで失われたプログラムを検索できるだけでなく、コマンドラインとして使用することもできます。 さらに、ヘルプを使用して、管理者権限でアプリケーションをすばやく起動できます。アプリケーション（またはコマンド）の名前を書き、Ctrl-Shift-Enterを押すだけです。 同時に、名前を完全に記述する必要はありません。アプリケーションが起動し、リストの一番上に表示されます。

さらに、タスクバー（ドッキングされているかどうかは関係ありません）またはスタートメニューにある管理者権限でプログラムをすばやく実行できます。 これを行うには、プログラムをクリックしたときにCtrl-Shiftキーを押します。 デスクトップ上のショートカットを使用すると、そのようなフォーカスはもう通過しません:-(



電気レポート



間違いなくラップトップの所有者にとって有用なものです。 このレポートを使用すると、どのプログラムまたはデバイスがバッテリーを最も使用しているかを知ることができます。 このレポートを取得するには、管理者権限でコマンドラインで次のコマンドを実行する必要があります。

powercfg -energy





約1分後、分析が終了し、このレポートを表示できる場所が通知されます。 または、レポートの場所を自分で決定することもできます。

powercfg -energy -output C:\Temp\report.html







スタートメニューからインターネットを検索する



さらに、検索バーを使用すると、プログラムの検索、ファイルシステムの検査、コマンドの実行ができ、インターネット上の情報の検索に使用できます。 悲しいかな、デフォルトでは非表示になっています。 このメニュー項目を表示するには、グループポリシーエディター（gpedit.mscを実行）を開き、ユーザー構成、管理用テンプレート、スタートメニュー、タスクバーに移動する必要があります。 上部には、[スタート]メニューの[リンクの追加] [インターネットの検索]オプションが表示されます。 それをダブルクリックし、「有効にする」を選択して「OK」をクリックします。 その後、たとえば「スタート」メニューの検索バーに何かを書いてみて、そのすぐ上にある「インターネットを検索」という項目を見ることができます。 このボタンにカーソルを移動して時間を無駄にしたくない場合は、上矢印を押してEnterを押します。

ところで、グループポリシーにはさらに多くの興味深い設定がありますが、それらのほとんどはシステム管理者のみが関心を持つものです。

ただし、Professionalより前のバージョンのWindows 7を実行している場合は、gpedit.mscユーティリティが利用できない場合があります。 それでも、検索でアイテムを追加することはかなり可能です。 1つのレジストリキーを変更（またはDWORDタイプで追加）する必要があります。

HKEY_CURRENT_USER\Software\Policies\Microsoft\Windows\Explorer\AddSearchInternetLinkInStartMenu





アイテムを追加するには、値を1に設定して削除する必要があります-0。さらに速く、これらはすべて既製のreg-filesを使用して行うことができます： add 、 delete 。 ただし、このレジストリキーを変更すると、このメニュー項目はエクスプローラーを再起動した後にのみ表示されます。



タスクバーナビゲーション



有名なAlt-Tabと新しい3次元のWin-Tabに加えて、実行中のアプリケーションを切り替える別の方法があります。 Win-Tを押し、キーボードの矢印を使用してタスクバーをナビゲートできます。 Aero Peekが有効になっている場合（タスクバーのウィンドウのサムネイル）、それらも表示され、矢印を使用してナビゲートできます。 Win-Bキーの組み合わせを使用して、同じ方法でトレイをナビゲートできます。

さらに高速に、Win-＃キーの組み合わせを使用してタスクバー上のアプリケーションを切り替えることができます。ここで、ラティスはメインキーボード（Win-1、Win-2など）の数字です。アプリケーションが起動されず、タスクバーにドッキングされた場合、開始します。



Windows Live Messengerアイコンを非表示にする



それを使用する人は、おそらくタスクバーからそれを削除できないことに悩まされます。 それでも、この問題は非常に簡単に解決されます。タスクバーのアイコンを右クリックし、「Windows Live Messenger」項目を右クリックして、「プロパティ」を選択する必要があります。 次に、適切なタブで、Windows Vistaとの互換モードを設定し、プログラムを再起動します。 それだけです。もう気にしません。ウィンドウを閉じると、トレイが最小化されます。 この方法は、Windows 7では何らかの理由でタスクバーに残っている他のプログラムで役立ちますが、そうすべきではありません。



ウィンドウを最小化する



きっと誰かがAero Shake機能について初めて耳にするでしょう。 実際、非常に便利です-アクティブなウィンドウ以外のすべてのウィンドウを最小化できます。 これを行うには、タイトルのアクティブなウィンドウを取得し、それを振ってください:-)。 ホットキーと最大化されたウィンドウのファンは、Win-Homeの組み合わせを押すことをお勧めします。

さらに、タスクバーの右端のボタンまたはWin-Dキーの組み合わせを使用して、すべてのウィンドウを非表示（および最大化）するか、Win-Mキーの組み合わせを使用して最小化できます。



アプリケーションのコピーをすばやく開いたり閉じたりする



プログラムを一度開いて、そのコピーをもう1つ開きたい場合は、ショートカットやメニューを使用する必要はありません。 Shiftキーを押しながら、タスクバーのプログラムアイコンをクリックするだけです。

別のより簡単な方法があります：マウスの中ボタン（ホイール）をクリックする必要があります。

もう1つの方法は、ホットキーを使用することです。Shift-Win-＃の組み合わせ（ラティスはメインキーボードの数字）は、タスクバーからプログラムの別のコピーを起動します。

さらに、非アクティブなウィンドウをすばやく閉じる必要がある場合、タスクバー上のウィンドウのサムネイルで十字に手を伸ばすことも、このウィンドウを展開することもできません。 同じウィンドウのサムネイルでマウスの中央ボタン（ホイール）を押すだけで十分です。



コマンドラインから複数のスレッドにファイルをコピーする



Windows Vista以降、 robocopyユーティリティはオペレーティングシステムにバンドルされています 。 かなり豊富な機能を提供するという事実に加えて、有用なキー/ MT [：n]もあります。 ファイルを複数のストリームにコピーできます（デフォルト8、最大128）。 このキーを使用してコピープロセスを高速化することについて話すことはできないと思います。



クイックウィンドウコントロール



Winキーとキーボードの矢印を使用して、ウィンドウのサイズを制御できます。 左矢印と右矢印を押すと、ウィンドウは画面の半分に拡大します。 上矢印を押すとウィンドウが全画面に拡大し、下矢印を押すと縮小または最小化できます。

また、Shift-Winと左矢印または右矢印を押すと、ウィンドウを別のモニターに転送できます。
