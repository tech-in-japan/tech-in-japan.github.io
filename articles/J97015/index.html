Panda Cloud Antivirusアップデート

2010年6月3日、PCAV 1.0のリリースから7か月後、Panda Cloud Antivirus 1.1の新しいバージョンがリリースされました。





このリリースの新機能：
  • 設定メニューは大きくはありませんが、既に拡張されています。
  • Windows Avtoranyがデフォルトでブロックされるようになりました
  • ウイルス対策プロセスと構成の自己防衛を追加
  • 簡略化されたフルコンピュータースキャン機能
  • ロシア語を含む9つの新しい言語が追加されました。 （アンチウイルスのインストール時にのみ言語を選択できます。thx dimap ）
変更点の全リストはこちらでご覧ください。
