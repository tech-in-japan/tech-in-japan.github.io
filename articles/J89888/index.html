Spring Design Alex Ebook Reader







個人的には、このデバイスは一見したところ、Barnes＆Nobleストアの垂直に伸びたNookリーダーを思い出させました。 また、下部にカラー画面、上部に読書画面があります。 確かに、彼は直接のライバルであるボーダーズに縛られていますが、彼は標準的な電子書籍フォーマットを理解しています。 上部のe-inkスクリーンのサイズは6インチで、下部の色は3.5インチです。 Android OSで動作し、4月14日にリリースされることが約束されています。 内部には、 ネオンマーベルARMADA 166Eプロセッサがあります。 実際、CrunchGearがこのデバイスをiPadのライバルであり、Amazon Kindleを備えたDroidやNexus Oneの束ではないと考える理由はわかりません。

* USB充電

*交換可能なバッテリー（iPadでは、iPhoneのように、バッテリーは交換できません）

*上部画面モードのみでのバッテリー寿命の延長（明らかです）



欠点



*両方の画面が小さい

*メモリが足りません（ただし、このデバイスで特定のメモリを見つけることができませんでした）







eDGeデュアルブックを奨励







私の意見では、非常に珍しくて興味深いデバイスです。 ただし、リリースされるかどうか不明なMicrosoft Courierとは異なり、eDGeは今すぐ購入できます：499はダークブルーバージョン、539ドルはホワイト、レッド、ライトブルー、ブラックを購入できます。 CrunchGearでレビュー全文を読むことができます。 電子インクデバイスの左側には、読み取りまたは描画用の画面があります。サイズは9.7インチ、1200x825で、ビデオで判断します-スタイルで描画および書き込みできます。 右側には通常のカラータッチスクリーンがあります。10.1インチ、1024x600、マイク、ビデオカメラ。 さらに、デバイスにはWiFi、Bluetooth、2つのUSBポート、SDカード用の1つのスロット、および取り外し可能なバッテリーがあります。 499ドルの価格を考えると、アカデミックな焦点（開発者は学童と学生のためにデバイスを配置している）にもかかわらず、学生は誰もそれを購入しないと自信を持って言えます。 その理由の1つは、レビューのビデオにはっきりと表示されています。7分ほどの間に、レビュー担当者は仮想キーボードを入力し、軽く押しても画面全体がどのように揺れるかを確認できます。 このバージョンでは、私の意見では、デバイスはまだかなり粗雑です。

Exopc







アップルは夢を嫌います。 このデバイスはiPadのように見え、iPadと同じくらいの費用がかかり、 その技術データは iPadのそれに近い-そしてAppleからはリリースされていない。 要するに：Intel atom N270 1.6GHz、2GB PC5400 DDR2 667Mhz、8.9インチ（1024 x 600）LEDバックライト抵抗性マルチタッチスクリーン、32 GB SSD、WiFi、Bluetooth、RJ-45（他にこれを見たことがありますか？）、カメラ、最大3つのUSBポート、ExoPCが開発したシェルを備えたWindows 7。 約束された動作時間は、バッテリーから4時間、スタンバイモードで72時間、総重量は820グラムです。 私の意見では-素晴らしい選択肢。 LANポートとUSBポートの1つ（2つを残して）を捨て、より大容量のバッテリーを挿入するためにそれらの場所を使用し、少なくとも6時間の動作を提供する場合、すでに深刻な競争について話し合うことができます。 また、exoPCに、Windows 7ではなく、Android、Google Chrome OS、またはこのデバイス用に特別に調整された別のバージョンのLinuxを使用するよう説得する場合、バッテリー寿命は最大8時間延長できます。



長所

* FlashおよびSilverlightのサポート

* WiFi +従来のCAT5を使用する機能

* Windows 7 + ExoPC UI

* 3つのUSBポート



欠点



* App Storeへのアクセスなし

*バッテリー寿命はわずか4時間です



ジュジュ（ CrunchPad ）







元々CrunchPadと呼ばれていたJoojooは、Fusion Garageによって開発されました。 英語版ウィキペディアで、Michael ArringtonがCrunchPadを作ったFusion Fusion Garage社とどのように口論したか、そしてそれがすべて終わった経緯についての短い物語を読むことができます。 JoojooはTechCrunchレビューに含まれないと想定するのが論理的だったので、元のレビューの最後の段落「自分のスマートフォン」をJoojooに置き換えることにしました。 技術面（公式サイトより）：12.1インチスクリーン（1366×768）、静電容量式マルチタッチ、4GB SSD、Wi-Fi（802.11 b / g）、Bluetooth、Flashサポート、総重量1.1 kgで最大5時間のバッテリー寿命。 ブラウザベースのオペレーティングシステム（つまり、Google Chrome OSに似たブラウザベースのオペレーティングシステム）として公式に宣言された、機能が低下したLinux上で動作します。 価格は499ドルです。



ストーリー全体、訴訟などに関連して、Joojooの予約注文数は100をはるかに下回っていました。 さまざまなニュースでさまざまな数字を見たと、ウィキペディアによると、50件の注文があるため、100件未満です。 ブログのiPadに関する大きなヒステリーのおかげで、私はJoojooがすぐにニュースでポップアップすることを期待していませんが、4月1日（ええ、まさにその日）にEngadgetブログがJoojooからビデオを投稿しました 。 率直に言って-私は感心していません。 ビデオが偽物ではない場合、
