ゲームの名前の選択について





最近、 私の記事で、 最初のプロジェクトを単独で開発し、終わらせることの難しさを明らかにしました。 しかし、私は1つのかなり重要なポイント- 名前を明らかにしませんでした。 実際、適切に選択された名前は、視聴者を引き付ける上で重要な役割を果たします。



はじめに 問題の科学的記述



一般に、プロジェクトの完成が近づいているときに名前を付けるのが最適です。 ほとんどの場合、これは起こりますが。



それで、自分のゲームの名前を考え出すことに決めて、私はオンラインに行きました。 しかし、価値のある記事はありませんでした 。



どうする？



そして、問題を科学的に提起し、ゲームの名前を分類し、名前の最適なバージョンを見つけて、自分のモデルを考え出すことにしました。



だから、ここに私の小さな分類があります。 例のゲームの選択は頭に浮かんだものであり、それがすべてであり、これが誰の好みをも傷つけてはならないことを警告します。



グループナンバーワン、または抽象芸術の愛好家







内部名のサブグループ



さて、それとも他の何と呼ばれていますか？



このグループには、本当に何かを意味しているように見える名前が含まれていますが、その意味はゲーム自体でのみ解読されます 。 例は悪名高いArmikrogです。プレイヤーは名前から何も作ることはできませんが、ゲーム自体ではArmicrogはゲームが行われる場所の名前です。



長所 ：リクエストに応じて、このゲームのみが、同様のリクエストへのサードパーティリンクなしで表示されます。 際立ったユニークな名前



短所 ：元の名前を覚えるのが難しく、正しく入力するのが難しい。 プレイヤーはゲームの本質に関する情報を受け取りません



真の抽象名のサブグループ



完全な抽象化であり、ゲームの本質も内部コンテンツも明らかにしない、まれな名前のグループ。 例： Rymdkapsel （彼らはここで、それがまだスウェーデン語で「宇宙船」を意味すると言った）。



長所 ：同じ



短所 ：同じですが、名前はゲームにまったく関連していないため、プレイヤーを倒す可能性があります



グループ番号2、またはほぼ命令







このタイプの名前は、ゲームプレイプロセスに直接訴えかけ、指示を与えます。 例：ロープを切る（Rus。Cut the rope-ゲーム中にロープを切る必要があります）。 初歩的なゲームプレイだけでなく、より発展したゲームプレイ（飢えない）、ルールの設定（移動または死）、またはゲームの目的を示す（橋を架ける）の指示もできます。



長所 ：プレイヤーはゲームで何をしなければならないかをすぐに理解します



短所 ：ゲームがあまり有名でない場合、見つけるのは難しいかもしれません。



グループ番号3、または自己名...もう一度







いいえ、ここではすでに抽象化とは異なり、少なくともプレイヤーに何かを伝えることができる固有名について話しています。



キャラクターサブグループ



それでも、キャラクターの名前は、ゲームの内容についてヒントを与えることができます。 実際、ある種のアンボールは花を集める可能性が低く、バスケットボールボールは通常、刀で敵をカットしません（想像力は無限にありますが）。 名前に忍者が含まれている場合、ゲームでは、おそらく速度または器用さが必要になることをすぐに理解できます。

例：スーパーミートボーイ。 接頭辞「スーパー」は、このキャラクターのいくつかの特別な特徴、彼のヒロイズムを暗示しています。



長所 ：プレイヤーはすぐにキャラクターに精通します。 多分キャラクターはゲームについて何かを伝えることができます

短所 ：前のグループと比較して、情報の内容はまだ低い



サブグループ



名前は本質的により一般的で、雰囲気、世界、つまりゲーム全体の雰囲気を説明しています。 例：ノックノック（Rus。Knock-knock-knock、プレイヤーは明らかに来た人と向き合う必要があることを示しています）。 別の種類から少し：Limbo（まあ、 手足のヒントがあります ）



長所 ：プレイヤーがプレイする環境を明らかにします

短所 ：繰り返しますが、関連するリクエストとマージできるため、名前の解釈方法が常に明確になるとは限りません



略語のサブグループ



略語が使用されることもありますが、通常は解読されます。 例：CATS、STALKER



長所 ：略語は注目を集め、人々を陰謀を誘います。デコードにより本質をより詳細に明らかにすることができます

短所 ：単語を略語に合わせるのは難しい場合があります。 名前の検索に問題がある可能性があります。



グループ番号4、またはミックス







実際、新しい名前を形成するために単語を結合することは十分な実践です。 同時に、結果は一意ではありますが、何かを伝えることができます。

例：ネバーフッド

おなじみの単語を変更できますが、検索エンジンが修正されないように：Scribblenauts



長所 ：名前は一意であり、プレイヤーは本質を理解しています

短所 ：名前は非常に複雑になる場合があります。 検索エンジンは自動的に修正されます



あとがき



私は何を選びましたか？ グループ4を選択し、大気が導入され、太陽が存在しないことが将来の更新の重要なポイントの1つとなるプロットが明らかになるように、プロジェクトに名前を付けることにしました。 具体的には-Nullysun 、つまり「Zero Sun」などです。 私が考慮に入れなかった1つの重要な間違いがない場合、結果を公然と言いません：



プレイヤーは名前をよく覚えていましたが、 スペルは単なる災害でした-彼らはしばしばNullysunの代わりにNullisunを書きました。 これはどういう意味ですか？ 名前がオリジナルの場合は、Y、Wなどの文字を避け、U / OO、C / K、KS / Xなどの組み合わせに注意してください。これは、特にロシアの聴衆を含む、または主に依存している人に当てはまります。
