人と機械：相互作用



ロボットエンジニアである私は、少なくともソフトウェアレベルで、人間とロボットの相互作用についてよく疑問に思いました。 Webテクノロジーとインターネットについて言えば、サイトを作成すると、その人との連絡が失われることを忘れてはなりません。 電話で話さない、ライブで会わない。 すべての情報はテキストで提供されるか、せいぜい写真とビデオで提供されます。



UXブースに関するいくつかの記事を読んだ後、ブルジョアがサイトへの訪問者を特定の人的要素だけでなくそれだけで感じさせないことに驚いた。 私のお気に入りは、MySpaceチームのTom Andersonでした。





MySpaceとトムアンダーソン



アメリカ最大のソーシャルネットワークMySpaceを利用してください 。 そこに新しいプロファイルを作成してみてください-MySpaceの創設者の1人であるTom Anderson （MatrixのMr. Andersenと混同しないでください）がすぐに友人のリストに表示されます。 つまり、多くのFAQ、マニュアル、その他のブリザードを表示するのではなく、MySpaceはいつでもあなたに向けられる「コンパニオン」を割り当てます。 外国の教育機関では、初心者には学生が割り当てられます。学生は完全に自発的に教育機関のツアーを提供し、チームへの参加を支援します。



はい、トムのプロファイルには同じルールがあり、よくある質問と回答は他のネットワークでよく見られます。 同時に、これらのルールを読むことによってのみトムを書くことができますが、それはポイントではありません。 トムは人工的に作成されたイメージの一種であり、その活動はMySpaceリソースのモデレーターによって作成されていることを誰もが知っていますが、インターネットテクノロジーから遠く離れた人について考えてください。 結局のところ、彼は少なくとも誰かが彼の面倒をみてくれたこと、そしてそれが潜在意識レベルだけであっても彼が一人ではないことを非常に喜んでいます。 私を信じて、この種の情報の提示は新しい訪問者を定期的な訪問者に変えます。



Flickrはさまざまな言語で歓迎します



世界で最も人気のあるフォトアルバム（またはフォトブログ）であるWeb 2.0リソースFlickrに言及せずにはいられません。 はい、Flickrが原因で、 FacebookやMySpaceのプロフィールに写真を投稿するのをやめました。 Flickrの使いやすさは、2つの巨人を組み合わせたよりもはるかに優れています。 写真はさまざまな場所からアップロードできます-Windows、Linux、Macのアプリケーション、電子メール経由、携帯電話からWAP経由など、はるかに（信じて！）違います。 写真をすぐに編集できます。 レーズンの巨大でありながらシンプルなAPI（ 詳細 ： Flickr API ）。



だから、十分な広告-ポイントに近い。 Flickrでプロファイルにログインすると、常にある種のシークレットモードで、毎回異なる言語で迎えられます。 プロファイルで作業するとき、通常はリソース全体で、ページのさまざまな領域に非常に有益なアドバイスを提供するツールチップが表示されることがあります（必要に応じてオフにすることができます）。 要点は、Flickrがすべての秘密を見つけるために大量のマニュアルを読むことを強制されていないことです（そして3トンの秘密があります）。 これらのヒントの助けを借りて、Flickrを徐々に身近に習得し、すべてをすでに知っているように見えるとすぐに、表示されなくなることのない新しいお菓子を紹介します。 特に2009年は、リソースの傾向が非常に高いです。 Yahooがそれを買ったのも不思議ではない。



Facebook-お誕生日おめでとう！



言うことはあまりありません。 幸いなことに、誕生日にFacebookプロフィールにログインすると、Facebookは誕生日おめでとうございます。 プリミティブ？ しかし、それは機能します！ 私が間違っていない場合、ロシアのリソースMail.ruは同じことをしますが、何らかのハガキのバカがありますが、私の記憶が私に役立つなら...



パンドラ-1羽の鳥と2つの石



Pandoraは、インターネット上で最大のラジオ局の1つです。 ラジオを長時間（約2〜3時間）聞いた後、Pandoraは追跡し、質問をします。「まだ聞いていますか？ 結局、私たちは失ったすべての歌にお金を払うので、空いている部屋を再生しないようにします。」 したがって、彼らはあなたに、これはある種の「生きている」クリーチャーであり、愚かな受け手ではないという感覚をあなたに知らせます。 そして、彼らは本当にお金を節約します！ 1羽の鳥と2つの石-ベール！ :)）一方、 Last.fmラジオは数日間再生されます...



ちなみに、PandoraはロシアのIPアドレスでは機能しません-ライセンスが必要です。 あなたのメールボックスを彼らのウェブサイトに残すことができます-彼らはすべての問題を解決したらすぐに手紙を送ると約束します。



ネットには同様の例がたくさんあります。 Pepsi-DJ Visionの決定が本当に気に入りました（残念ながら、サイトは以前と同じではありません）。 ビデオが画面に現れ、実際の人（あなたの将来の指導者）がいくつかの入門的な言葉を話し、彼に部屋への願いを添えました。 部屋に入ると、DJコンソールが表示され、演奏が始まります。 時々、この同じ人が画面のさまざまな領域に現れ、次に何をすべきかを教えてくれます-あなたはもはや新人DJが一般に放り込まれるような気がしません。



ロシア語のインターネットでは、同様の手法を適用しようとしているリソースのユニットのみを観察しました。 これが近い将来に変わることを期待しています;）
