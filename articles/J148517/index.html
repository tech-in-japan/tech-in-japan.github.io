NetBeans IDE 7.2リリース





すべてのJava、C / C ++、およびPHP開発者にとって朗報：2012年7月25日、 NetBeans IDE 7.2の次のバージョンがリリースされました 。



最も重要な変更：


  • パフォーマンスが向上しました
  • FindBugsの統合
  • JavaFX Scene Builderの統合
  • Javaエディターのさまざまな改善
  • Java EEの強化
  • PHP 5.4などのサポート
  • 更新されたGroovyおよびGrailsのサポート
  • TestNGサポート
  • UPD： PHPでの注釈サポート
  • PHPフレームワークSymfony2、Doctrine2およびApiGenのサポート




変更点の詳細については、 こちらをご覧ください 。
