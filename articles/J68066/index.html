WPFブラウザー

WPF内でWebページを操作および操作する機能は、いくつかの興味深い可能性を開きます。 WPFには既にWebブラウザーが含まれていますが、これはActiveX Control IEの単なるラッパーです。 残念ながら、これによりWPFツールで適切にプレイすることが難しくなります。



GoogleのChromiumプロジェクト、 Awesomiumと呼ばれるシェル、および少しのWPFを組み合わせることにより、次のことができます。







こちらから ClickOnce を使用してこのブラウザをインストールしてみてください。 ソースコードはCodePlexからダウンロードできます。 ClickOnceを使用してこのアプリケーションをインストールした後、キーボード入力に問題があります（機能しません）-私と同じでした。完成したzipアーカイブをすぐにダウンロードして試してみることをお勧めします。 真実は、そこに試すものは何もないということです。

  • Cjc.ThreeDeemium-サンプルアプリケーション。現在3D機能はありません。
  • Cjc.ChromiumBrowser-コントロール-WPF Chromium Webブラウザー。 Cjc.AwesomiumWrapperと2つのライブラリAwesomiumおよびChromium C ++（Awesomium.dllおよびicudt38.dll）に依存します。
  • Cjc.AwesomiumWrapper - Awesomium上の C ++ / CLIシェル（マネージコード）。 古いP / Invokeセットを保存するよりも簡単で、コードを見る方が快適です
  • Awesomium -C ++ヘッダーとAwesomium / Chromiumのライブラリ。
