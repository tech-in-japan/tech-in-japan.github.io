Mac OS X用Google Waveデスクトップクライアント

多くの人がすでにGoogleまたは友人から新しいGoogle Waveコミュニケーションサービスへの招待を受け取っています 。 Google Waveは、メール、インスタントメッセージング、チャット、フォーラム、共同編集ドキュメントなどの機能を組み合わせるように設計されています。



Google Waveのメインクライアントはブラウザです。 ただし、最も安定した最速のGoogle WaveはGoogle Chromeブラウザーで動作することを認めなければなりません。



現時点では、 Apple Mac OS Xユーザーは最も幸運です-Google Waveのブラウザーバージョンのすべての機能を実装および繰り返すWaveboardというクライアントを既に作成しています。







Waveboardプログラムの特徴は、 Mac OS Xオペレーティングシステムのすべての通知システムとの統合です。

  • Growlを使用してアラートを更新する
  • ステータスメニューおよびDockにあるアイコンに更新されたウェーブの数を表示します










Google Gearsは既にWaveboardプログラム自体に統合されているため、個別にインストールする必要はありません。



ブラウザと同様に、 Waveboardはキーボードショートカットもサポートしています。







プログラムの現在のバージョン： Waveboard 0.11

ダウンロードリンク： http : //d.getwaveboard.com/shine/download_waveboard.php

プログラムのウェブサイト： http : //www.getwaveboard.com



Waveboard Proの商用バージョンが最近登場しました。これには次の機能が含まれています。

  • シングルウェーブ印刷（現在、このオプションはブラウザバージョンのGoogle Waveでは使用できません）
  • ステータスメニューから未読のWaveを直接開く
  • ステータスメニューのアイコンを変更する
  • ウィンドウがアクティブになったときの通知の自動リセット
  • WebアーカイブとしてWaveコンテンツを保存する
  • Dockでアイコンを非表示にする機能。
Waveboard Proの商用バージョンのコストは9ユーロです。



Apple iPhoneの場合、 Waveboardクライアントも利用できます。これは、 App Storeで0.99ドルで購入できます。 バージョン1.0の機能は、デスクトップバージョンと同じです。 現時点では、プログラムの2番目のバージョンがレビューのためにApp Storeに送信されており、 プッシュ通知をサポートしています。
