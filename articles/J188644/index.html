アスタリスク用のGoIP SMSサーバー（部1）

この記事では、実装の1つであるアスタリスク電話プラットフォームでのSMS配信サーバーであるGoIPの使用について説明します。



多くのGSMゲートウェイを試してみましたが、GoIPゲートウェイが特に価格/品質の面で最高であると自信を持って答えることができます。 例として、彼らは素晴らしいです、彼らはベラルーシのコールセンターの1つで働き、約100のGSMチャンネルを練習し、1日あたり約3,000のコールを受け取ります。 さらに、中国のサービスエンジニアのサポートは、非常に迅速に満足しました。

このゲートウェイの重要な利点は、SMSメーリングを生成したり、残高を削除したりすることができるSMSサーバーです。 サーバーのローカライズされたバージョンと記事のより詳細なバージョンは、私のWebサイトにあります。

GoIP SMSサーバーのローカライズバージョンはこのページにあります 。または、メーカーのウェブサイトから元のサーバーをダウンロードできます。 アスタリスクプラットフォームとして、 Elastixディストリビューションを使用しました 。 モジュールのインストールは、いくつかの手順で実行されます。

cd / usr / src

tar xvfz goip_install-ru-v1.15.1.tar.gz

cd goip_install

./goip_install.sh

/etc/init.d/httpd restart

インストール時に、データベースのパスワードを指定します。

192.168.1.35/goipに移動します







パスワードadminログイン：ルートルート。 SMSサーバーの管理パネルに入ります。

最初に行うことは、SMSサーバーとGoIPゲートウェイ上にトランクを作成することです。

[プロバイダーの追加]タブに移動します。







GoIPおよびSMSサーバーにSMS送信者を登録します。







各SIMカードには独自のカードがあります。
