更新されたGoogleテストフレームワーク

先日、 Google Testing Frameworkがバージョン1.7.0に更新されました 。

新しいバージョンでは：

*新機能：死のテストはOpenBSDおよびIOSシミュレーターでサポートされています。

*新しい関数：Test :: RecordProperty（）は、テストメソッドの外部で使用できるようになりました。

*新機能：-gtest_list_testsは、各テストのパラメータータイプとパラメーター値を出力するようになりました。

*改善：文字ポインターと文字配列が正しく処理されるようになりました。

*改善：XMLファイルのクラッシュレポートに行番号とファイルが含まれるようになりました。

*「潜在的に重大な変更」：RUN_ALL_TESTS（）は、Clangとの連携を改善するために、マクロではなく関数として実装されるようになりました。

* C ++ 11およびさまざまなプラットフォームとの互換性修正。

*など。
