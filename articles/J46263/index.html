ネットブックの100ドルマークへの着実な動き

昨日、中国の「お気に入りの1つ」販売業者、 dealextremeは、ネットブック/ UMPCセグメントのビジョンを、この製品として206.50ドルの価格で提示​​しました（世界送料を含む）。

このユニットのTTXおよび機器：

-Microsoft Windows CE 5.0オペレーティングシステム（英語）

-7インチの鮮やかなカラーTFT LCDスクリーン

-80MB RAMシステムメモリ（82340KB）

-フラッシュ可能なROMに保存されたオペレーティングシステム（RAMストレージを占有しません）

-超高電力効率（バッテリー充電でオンにした場合、合計で最大15W）

-バッテリーを取り付けた状態で重量はわずか683グラム（24オンス）

-Windows CEは数秒ですぐに起動します（スリープ機能や休止状態機能を使用する必要もありません）

-ARM926EJ-S 266MHz 32ビットCPUにより、1回の充電あたりの超長時間使用が可能（CPUはPDA携帯電話と同様の電力しか消費しません）

-マウス、トラックボール、キーボードなどをサポートする3 x USBポート

-1 xスピーカー/イヤホン3.5mmステレオミニジャック

-1 xマイク3.5mmステレオミニジャック

-1 x RJ45イーサネットネットワークポート

-80キーキーボード

-SDHC SD / MMCメモリカードリーダー/ライタースロットを内蔵（最大16GBをサポート）

-Wifi IEEE 802.11B / Gワイヤレスネットワークアダプターを内蔵

-WIFIはWEPとWPAの両方のセキュリティをサポートします

-MP3 / WMA / AAC / AAC + / CAT / WMV / AVIファイルなどの再生をネイティブでサポート

-内蔵の自動再充電バックアップバッテリーにより、メインバッテリーの消耗によるRAMデータの損失を防止

-メインバッテリーの電力レベルは、コントロールパネルのアプレットを介して正確な割合で表示される

-ヘビーデューティー4100mAhリチウムイオン充電式バッテリーが付属
