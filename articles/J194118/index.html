IFA 2013：アクションカメラ

今年、偶然にも、ドイツで2番目に大きなテーマ別のIT展示会であり、世界で4つの最大規模（CES、CeBIT、Computex、IFA）の1つであるIFA 2013に参加できました。 たくさんの印象、さらに多くの写真や考えがあります。 そして、アクションカメラを使用してIFA 2013の新製品のレビューを開始することにしました。同僚は投稿で多くの注意を払っています。







彼らは比較的最近アクションカメラについて話しましたが、これらのデバイスは急速に人気を集めています。 対応する需要により高い需要が生じます。IT製品のリリースに関係するほぼすべての企業が、「スポーツ」カメラで品揃えを補充しようとしています。 そして、1、2年前に、アメリカンウッドマンラボのデバイス（GoProのみ）しか聞かれなかったので、今では多くのアナログが市場に出ています。 IFA 2013は、アクションカメラが地元のBブランド（ドイツ語など）だけでなく、有名な国際企業にとっても興味深いという事実を目撃しました。 それらを使用して、新製品のレビューを開始します。



JVC


JVCケンウッドのスタンドを訪れて、私はアクションカメラに出くわすとは思いませんでした。 しかし、それは起こりました。 後に判明したように、同社はADIXXIONシリーズに割り当てられた2つの「極端なカメラ」（メーカーが呼んでいる）を既に持っています。GC-XA1は光を早く見ましたが、IFAで会社は新製品GC-XA2を示しました。 このアクションカメラの世界初演は、2か月以上前の7月9日に行われました。







ADIXXIONはJVCカムコーダーラインの派生物であり、GC-XA1とGC-XA2の外観を推測するのは簡単です。 これから先、ソニーも同じように進んだと言います。ソニーのアクションカメラも小さなビデオカメラのように見えます。 そうすることで、つまり 外観に「バザリング」がなければ、JVCとソニーの両方が負けます。スタンドに来て、「これらは次のビデオカメラです」などと言って出て行ったのを見ました。 庭に立っているマネキンでさえ恥ずかしくない。 同時に、GoProのようなオリジナルのデバイスの場合、見た目は健康的です。







GC-XA2の仕様には特に目立ったものは何もないと言わざるを得ません。 保護されたケースを使用すると、カメラを水深5 mまで浸して、摂氏-10度までの温度でカメラを操作できます。 2 mの高さからのほこりや落下からの保護も宣言されています。 しかし、1.5インチの対角線を備えた組み込みカラーディスプレイの存在は、明確なプラスです。

もちろん、GC-XA2にはバッテリーが内蔵されており、SDXCカードと互換性のあるフルHDビデオを録画できます。 Wi-Fiサポートがあります。











GC-XA2は、レンズのより良いマトリックス（通常の5メガピクセルCMOSに対して8メガピクセルBSI CMOS）、異なる等価焦点距離（20.5 mm対25.4 mm）、および異なる開口部（F / 2.4対F / 2.8）が前身と異なります。 興味深いことに、これに伴って視野角が変更されました。GC-XA1の場合は170度に等しく、GC-XA2の場合はすでに160度です。

JVC GC-XA2の仕上がりは非常に優れていることに注意してください。 カメラは「きちんと調整」されていないかもしれませんが、確かに「ステッチ」されています。 デザインの豊富な金属は、アクションカメラが主に購入される状況に対してJVC GC-XA2が本当に徹底的に準備されているという自信を呼び起こします。











欧州価格JVC GC-XA2-約300ユーロ。



ソニー


IFA 2013では、HDR-AS30V / Bアクションカメラの世界初演が行われました。これは、以前のモデルであるHDR-AS15 / Bよりも顕著に改善されました。 Wi-Fiサポートが旧モデルの最も注目すべき機能であった場合、新しい「プラス」はNFCとGPSです。







興味深いことに、HDR-AS30V / Bには、HDR-AS15 / Bで使用されているのと同じレンズとマトリックスがあります。 つまり 同じJVCで新しいモデルの特性が改善された場合、Sonyは「古いモデルでも十分だ」と考えました。 原則として、HDR-AS30V / Bは、11.9メガピクセルの印象的な解像度を備えた1 / 2.3インチフォーマットのBSI CMOSマトリックスと、F / 2.8の最大開口を備えたCarl Zeiss Tessarレンズを使用しているため、ソニーは理解できます。











HDR-AS30V / Bは、その前身とは異なり、SDXCメモリカードと互換性があります-これは良いですが、音が良く（現在2つのマイクがあります）、制御の利便性のために、付属の水中ダイビングボックスはこれ以上ダイビングしないように設計されています5メートル。 比較のために、箱付きのHDR-AS15 / Bは60メートルのダイブに耐えることができます...一般的に、これには重大なことは何もありません。一般ユーザーがこれほど深く「潜る」ことはまずありませんが、モデルからモデルへのアクションカメラの保護機能は、大幅ではないにしても、改善されるだけです。



「私の手の中に」私はSony HDR-AS30V / Bが好きではありませんでした。仕上げ材は高品質のようですが、JVC GC-XA2が持っている安心感を与えません。



私の知る限り、ロシアでのソニーHDR-AS30V / Bの販売は10月に始まり、そのコストはHDR-AS15 / Bを求めているのと同じ12 990ルーブル、またはそれ以上になります。



ローライ


ドイツでよく知られているRollei社は、アクションカメラの分野で非常に強力に活躍しました。 実際、彼女の説明のほとんどすべては、これらのデバイスに正確に当てられていました。







Rolleiアセットには、形状、サイズ、色さえも異なる約10種類のアクションカメラがあります！ また、3S、4S、5S、5S WiFi、S-50 WiFiモデルのアクセサリカタログには、ヘルメットの通常の3Mステッカーと追加のバッテリーから、サーフボードと磁気パッドの特別なマウントまで、26個のアイテムがあります！ ソニーもJVCも、このようなさまざまなマウントを自慢することはできません。 一般的に、第2層の企業の製品は、より低価格でより多くの機能と機能を備えていると感じました。























Rolleiが示すすべてのカメラモデルをリストするのではなく、2、3にのみ焦点を当てます。

Rollei Youngsterは、主にヨーロッパでよく見られる若者に焦点を当てています。 特性は平均的です（ビデオ録画720p、視野角135度、2インチスクリーン、保護ボックスが含まれます）が、価格は低く、ケースのいくつかの色が用意されています。



一般に、明るい色や交換可能なパネルは、アクションカメラの新しいトレンドであると言わなければなりません。 第2層のメーカーは、需要の大きさを十分に認識しており、外観のカスタマイズを含め、ユーザーに最大限の機会を提供しようとしています。 たとえば、新しい秋のRollei S-50 WiFiには、黒、青、黄色の3つのパネルが付属しています。







S-50 WiFiモデルは、Rolleiのフラッグシップアクションカメラです。特性については、14メガピクセルマトリックス、深さ60 mに没入するボックス、ガラスレンズ、拡張温度モード-摂氏+60〜-20度、 175度の視野角、2インチ画面、Wi-Fi経由でスマートフォンから制御する機能。 交換可能なパネルに加えて、このカメラには2つのバッテリーが付属しています。







Rolleiの例は、アクションカメラがどれほど人気が​​あるか、そして消費者の間でこれらのデバイスに対する爆発的な需要を満たすためにどこまで行けるかを示す最もわかりやすい例であるように思えます。 企業が数十のスマートフォンモデルを製造している場合、それは理解できることです。10種類のアクションカメラを販売しようとするのはまったく別のことです。 私の意見では、これは多すぎる。
