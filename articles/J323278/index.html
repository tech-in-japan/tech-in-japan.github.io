Visual Studio 2017の立ち上げのビジュアルブロードキャスト

Visual Studio 2017のオンライン放送開始に参加するために、3月7日と8時に19:00（モスクワ時間）にご招待します ！ お気に入りのツールの新しいバージョンをすぐにダウンロードできるだけでなく、Visual Studioの20年を祝う機会もあります！







開発者ツールの新しいバージョンは、過去数年にわたって変更されており、エンジニアの日常的なタスクを解決するのにさらに便利になっています。 イベントの一環として、伝統的に新しい機能と最も興味深い機能の多くのデモが計画されてきました。 クロスプラットフォーム開発の改善、クラウドサービスとコンテナーとの連携、Team Foundation ServerおよびVisual Studio Team Servicesとのコラボレーションを整理するタスク-これらのツールの作成者は、このことについて多くを説明します。



初日の全体報告は、ブラウンハリー、ミゲルデイカザ、ジュリアルーソン、スコットハンセルマンによって行われます。 エンジニアやツール開発者と協力して、Visual Studio、.NET、Xamarin、Azureの最新のイノベーションを共有します。 全体報告の後、Microsoftの技術専門家は、 Visual Studio 2017のすべての機能と実際に開発中の開発者向けの待望のアップデートを紹介します。



イベントページにご参加ください。従来のハッシュタグ＃vs2017を使用して、Twitterでニュースをフォローすることを忘れないでください。
