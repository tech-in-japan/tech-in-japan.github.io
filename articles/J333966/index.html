ロシアで5Gを待つ時期

すでに今日では、5Gの導入が世界を変えることは明らかです。 「スマートホーム」と「スマートシティ」、オートパイロットとリモートプレゼンスシステム、ホログラフィックコール、拡張およびバーチャルリアリティサービス、およびあらゆる種類のセンサーとデバイスの巨大な公園は、平均的な人の生活を満たします。 そして、これは私たちの近い将来です。 5Gの機能を繰り返しテストし、25Gb / sのモバイルデータ転送速度を達成しました。 MTS無線アクセスネットワーク部門の専門家であるAlexey Maslovに、5Gネットワ​​ークがどのように機能し、最初のデバイスが第5世代ネットワークをいつサポートするかを尋ねました。







みなさんこんにちは。 そもそも、5GがLTEと根本的に異なる理由は何ですか。 したがって、違いは次のとおりです。



⎯ピークスピードが高い場合：LTE-Advanced Proに比べて12倍、3Gに比べてほぼ500倍の利点。

⎯最高のスペクトル効率：DL（ダウンリンク）の場合、それぞれLTE-Advanced Proおよび3Gの1.5倍および7倍の効率。

⎯無線アクセスサブシステムの遅延を数桁減らす：超信頼性の高い通信サービスURLLCでは0.5ミリ秒、高速通信サービスeMBBでは4ミリ秒、LTE / 3Gでは数十/数百ミリ秒。

⎯ネットワークに接続されたデバイスの最大密度は、都市部で1,000,000デバイス/ km2であり、充電せずに少なくとも10年間自律型センサーを動作させる能力。

⎯最大500 km / hのデバイス速度でのモビリティ（ハンドオーバー）のサポート

⎯6〜100 GHzの新しい周波数範囲を使用する場合。

⎯クラウドアーキテクチャ/ 5G機器の仮想化の使用。

これらの利点は、多くの新しいテクノロジーによって実現されます。 いくつかのテクノロジーは既にオペレーターによってテストされており、いくつかは開発中です。





図 1-6。 5Gテクノロジー



図 1-Massive MIMOおよびMassive beam-forming。 一番下の行は、基地局でトランシーバの数を増やしているということです。 古典的なソリューションが2つの送信機と2つの受信機である場合、5Gでは16〜512以上の送信機と16〜512の受信機が存在する可能性が高くなります。 共通の帯域が1つありますが、加入者ごとに個別のビームが形成されます。 スペクトル効率を大幅に向上させます。 このアーキテクチャは、周波数リソースが限られているため、LTEネットワークでも使用されます。



図2マルチサイト調整 これは調整ツールです。5Gでは、ある基地局から別の基地局への移行はありません。つまり、加入者は自分の環境内の複数の基地局から同時にサービスを受け取ります。 機能の一部はすでにLTEに実装されています。



図 3加入者は基地局から遠く離れることができるという事実について話しているが、加入者の近くに小さなセルを配置することにより、既存のマクロ層の容量を再利用して信号レベルと容量を増加させる。



図 図4は、加入者のシグナリングとユーザー情報の分離を示しています。 信号情報は、損失なく端末に届く必要があります。つまり、カバレッジが良好なマクロステーションから送信されます。 また、信号レベルにとってそれほど重要ではないユーザー情報は、遅延のために、カバレッジが少なく、電力が少ないスモールセルの形式である場合、近くのステーションから送信されます。



図 5-デバイスからデバイスへの接続。 オペレーターのネットワークに参加せずにデバイスがデバイス間で直接動作する場合、これは別の方向です。 デバイスがマクロカバレッジエリアを離れると、近くの端末との通信を通じてマクロネットワークからカバレッジを受信できます。



図 図6は、5Gを使用するときに無線インターフェイスで発生する変更を示しています。 ここでは、特定のサービスがサブキャリア間の間隔を柔軟に変更し、それに応じて、使用される範囲と必要なサービスに応じて最小パケット送信時間を変更することが可能になります（たとえば、リアルタイムサービスの高速データ転送または超高信頼通信）。 また、5Gは1つの帯域を使用します。つまり、アップリンク/ダウンリンク帯域への固定分割はなく（デュプレックスダイバーシティなし）、必要な負荷に応じてアップリンクaとダウンリンクaの送信が柔軟に変更されます。 つまり、ストリップをより効率的に使用できます。



5Gの機器を変更する必要がありますか



現時点では、5Gコンセプトが公式に標準化されておらず、指定されていないという事実を考えると、2018年の初めまでにのみ5Gの実装に対するネットワークの準備の程度を評価することが可能になります。 それにもかかわらず、今日、MTSはNSA 5G NR（非スタンドアロン5G New Radio）と呼ばれる中間5G標準の導入を準備しています。これは、既存のLTE無線サブシステムと既存のネットワークコアを、新しいキャリアを追加してモビリティとカバレッジを管理するためのアンカーシステムとして使用します5G。 同時に、5Gネットワ​​ークでは、より多くの基地局を構築し、トラフィックが活発に消費される場所で通信事業者のトランスポートネットワークを強化する必要があります。



最初の段階では、既存の無線サブシステム、ネットワークコア、および5Gのトランスポートを使用する可能性が高くなります。 しかし、それほど単純ではありません。 既存のLTE規格に幅の広い5Gバンドが追加されるため、既存のLTEテクノロジーに新しいバンドのサポートが追加されます。つまり、速度が向上します。 すべての重要なシグナリング情報はLTEを通過し、より広い帯域を必要とするユーザー情報は5Gを通過します。 このため、既存のLTEインフラストラクチャを再利用できます。



しかし、今では、原則として、5Gの完全な実装には、新しいサブシステムを新しい周波数範囲に配置する必要がある無線サブシステムに加えて、近代化、追加機器の設置、トランスポートチャネルの増加、パケット皮質の容量の増加が必要であることは明らかです。 基地局の計算能力も増加します。



5Gの周波数



ロシアでは、5G周波数はまだ不明であり、最初の加入者端末が動作する周波数を承認するという事実はありません。 オペレーターが現在利用可能な周波数では、5Gを開発する意味はありません。 現在、これは必要ありません。端末はありません。また、周波数帯域が小さいため、これによる影響はありません。 2019年のWRC-19サミットでは、5Gバンドを承認する予定です。これにより、2020年までに商用5Gネットワ​​ークの立ち上げが期待できます。表は、どの国がどの周波数を使用する予定かを示しています。



ロシアの難点は、有望な「ヨーロッパ」および「アメリカ」の5G周波数が他のサービスで使用されていることです。 たとえば、潜在的に考慮されている3.5 GHzの範囲では、衛星サービスと固定ブロードバンドアクセスネットワークが現在「座っている」。 明らかに、範囲をクリアすることは迅速で高価なビジネスではありません。 初期段階では、新しい周波数範囲は、後の新世代のセルラー通信のネットワークに使用されます。これは、それまでに時代遅れであった既存の3G / LTE技術の範囲です。



最初の5Gデバイス



現時点では、5Gチップセットでのハードウェアサポートに関する発表がいくつかあります。 ただし、最初の商用5Gデバイスは2019年（ルーター/タブレット、および2020年）のスマートフォンになります。 主要な機器サプライヤは、主に住宅地での固定アクセスのために、米国と韓国の市場で28 GHz帯域で5G（Pre-5G）の先行標準バージョンをすでにテストしています。 このために、米国の事業者は最初の5G仕様の承認をスピードアップしようとしています。 2018年3月に、プレリリースを承認する予定です。



最初の5Gテスト



MTSは、2018年ワールドカップで5Gデモンストレーションを計画しています。 実装時までに、最初の5G仕様は、米国の事業者のネットワークおよび韓国の28 kHz GHz冬季オリンピックで既に採用およびテストされています。 したがって、この範囲の基地局と5G端末の最初のサンプルが作成されます。 おそらく、同じ原則により、2018年のワールドカップ中にロシアで5Gをテストします。 5Gの追加のスペクトルが2019年秋にWRC-19で決定されることを考えると、5Gの完全な実装は2020年以前になります。



5Gの標準化が完了するまで、規制当局が5Gの低域と高域を使用することを決定した後、新しい技術を導入する準備を整えるために、革新的な技術ソリューションのテストを実施します（5Gの既存の範囲の技術的中立性、オークションの実施）。



たとえば、先日、LAA / LBTテクノロジーのテストを正常に完了し、LTE周波数とWi-Fiネットワークの免許不要スペクトルを組み合わせました。 この新技術は5Gネットワ​​ークに応用され、データ転送速度を向上させると同時に、通信事業者の基地局とさまざまなローカルネットワークのデバイスの両方の同じ周波数範囲で動作します。 Ericsson Pico RBS 6402ピコベースステーションを使用した最初のテストでは、2つの周波数帯域が使用されました。1800MHz LTE帯域で10 MHz、Wi-Fiネットワークで使用される5 GHz免許不要帯域で20 MHzです。 テストでは、無料のWi-Fiネットワークでは最大206 Mbit / s、ロードされたWi-Fiネットワークでは最大145 Mbit / sのオペレーターのLTEネットワークでのデータ転送速度を実証しました。 2番目のテストでは、Huawei Lampsite pRRU3911ピコ基地局は3つの周波数帯域を使用しました。Wi-Fiネットワークの無認可スペクトルのLTE 1800 MHz帯域で15 MHz、5 GHz帯域で2x20 MHzです。 オペレーターのLTEネットワークでのデータ転送速度は、LBTテクノロジーを使用しない場合は411 Mbit / sに達し、LBTをオンにすると、無料のWi-Fiネットワークで373 Mbit / sに、Wi-Fiネットワークがロードされて297 Mbit / sに達しました。



現在のオペレーターの生活



現時点では、通信事業者は「デジタル不平等」の問題を解決する必要があります-2019年末までに、「ビッグフォー」はロシアのすべての地域でLTEネットワークを展開するためのライセンス義務を果たさなければなりません。 これは、市場の現在のニーズを満たすために、カバレッジをさらに拡大し、ネットワーク帯域幅を増やし、新しいLTE-Advanced Proテクノロジー（ギガビットLTE、NB-IoT、LAA）を導入する必要があることを意味します。



5Gを起動するには、まず、高速通信サービス（最大20ギガビット/秒）、超低遅延（最大4ミリ秒）、および高ネットワーク容量（1,000,000デバイス/ km2）が必要です。



過去の世代のネットワークに何が起こるか



すでに述べたように、3GPP内の最初の5Gネットワ​​ークでは、5G基地局が共通の機器（Non-Standalone、NSA）を使用してLTE基地局と連動するアーキテクチャを検討しています。 この場合、新しい範囲の5Gキャリアはユーザートラフィックにのみ使用され、制御トラフィックはLTE基地局のキャリアを介して送信されます。 つまり、最初の5G端末はLTEをサポートします。 端末の普及率が低いため、5Gサポートのない既存のユーザーへの影響はありません。 その後、5G端末は、LTE端末の場合のように、古い技術のみをサポートする端末をスムーズにフラッシュします。



ただし、2025年まで2Gネットワ​​ークを完全にオフにすることはできません-2Gのみをサポートする端末がネットワーク上で消えるまで。 まず、これはM2M 2G端末に適用されます。 3Gネットワ​​ークの切断は2Gよりも可能性が高いですが、3Gのみをサポートする端末の普及率が非常に低い場合も可能です。
