最速のホームインターネット-日本（2Gb / s）





先日、日本のプロバイダーSo-net Entertainmentは、提供されるサービスのパッケージに、最大2 Gb / sの速度でネットワークへの接続を追加することが判明しました。 これはホームインターネットであり、研究所が開発したパイロットプロジェクトではありません。 この場合、ソニーはパートナーです。 新しい高速接続サービスはNuroと呼ばれ、東京の居住者といくつかの近隣の県の居住者がすでに利用できます。



会社の経営陣によると、このパッケージは、家庭用だけでなく、レストラン事業の所有者や他の中小企業向けにも設計されています。 同時に、パッケージのコストはそれほど高くありません：2年間の契約を結んだ場合、1か月あたり51米ドル（プラス、機器の設置についてもほぼ同じ、1回限り）。 同時に、「最大2 Gbit / s」が意味することをすべて知っています（まあ、10 Mbit / sも「前」であり、一部の国内プロバイダーはこの手法を積極的に使用しています）。 しかし、日本人の場合、状況は少し異なります。



この特定の場合、「最大2 Gbit / s」とは、プロバイダーが約2 Gbit / sの速度でダウンロードを保証することを意味しますが、ユーザーからのデータ送信は1 Gbit / sを超えない速度で行われます。 このプロジェクト自体は、接続チャネルを指定された速度に「拡張」する光レシーバーの開発と実装のおかげで実現しました（テクノロジー自体はGPON /ギガビット対応のパッシブオプティクスネットワークに基づいています）。 デバイスは光学系に直接接続し、光学/光信号を電気信号に変換します。



上記の会社に加えて、当局もプロジェクトに参加し、市と県に光ファイバーインフラストラクチャを展開するプロジェクトに部分的な資金を提供しました。
