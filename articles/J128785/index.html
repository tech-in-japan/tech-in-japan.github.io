シリコンバレーがロシアのスタートアップにやってくる

こんにちは

しかし、プラグアンドプレイは、デバイスをすばやく特定して構成するための技術であるだけでなく、世界で最も有名なビジネスインキュベーターの1つであることをご存知でしたか？ 皆さんにとって朗報です！ プラグアンドプレイはモスクワに駐在員事務所を開設し、5つのチームが10月に居住者になる機会を持っています。



背景


プラグアンドプレイテックセンターは1996年に設立されました。 最初のインキュベーション企業の1つは、Logitech、PayPal、Googleでした。 2006年以来、ビジネスインキュベーターの専門家は、600以上の新興企業が独立企業になり、播種前および播種段階で7億5,000万ドル以上のベンチャー資金を集めてきました。 2010年だけでも、30社以上がベンチャーキャピタル投資で1億5,000万ドルを集め、合併や買収によるプロジェクトからの15の「出口」がありました。 プラグアンドプレイのパートナーは、DFJ、Sequoia、Canaan Partnersなどの150以上のベンチャー企業です。



デフォルト都市の近未来


10月12〜14日に、プラグアンドプレイの担当者が初めてモスクワに来ます。 この訪問の目的は、ロシアのプラグアンドプレイでの居住の候補者を検討することです。 最終選考は10月13日に行われ、シリコンバレーのプラグアンドプレイの創設者兼CEOであるSaid Amidiがプロジェクトの選定を指揮します。 ファイナルの結果に続いて、5つのプロジェクトが選択され、プラグアンドプレイロシアでインキュベートされます。



余談＃1：「しかし、私はそれと何をしなければならないのですか？」


プラグアンドプレイは、主に世界クラスのベンチャーキャピタル市場の専門家です。 ロシアでP'n'Pの仕事がどのように構築されるかはまだ正確にはわかっていませんが、住民として、あなたは起業家のグローバルコミュニティに自動的に含まれることになると確信できます。 グローバル市場向けのプロジェクト-世界支配への第一歩。



余談＃2：「わかりました。わかりました。 そして、あなたはP'n'Pのどちら側ですか？」


私たちは、グローバルなTechInnovations（GTI）であり、起業家がロシア国内および海外でベンチャー（だけでなく）資本を調達するのを支援するサービス企業であり、ロシア企業が西部市場にアクセスし、西部企業がロシア市場にアクセスできるようにします。 広範な接続と長い武器があります。 私たちのCEO、アーサーバガノフ、アメリカのビジネスエンジェルスネットワークの共同創立者、エンジェルズアライアンスの共同設立者にのみ価値があります。

ロシアのプラグアンドプレイの場合、プロジェクトの予備選択を担当できたことは幸運でした。 フィナーレの前に、スペシャリストとコミュニケーションをとる必要があります。



そして現在に戻る


ロシアのプラグアンドプレイ居住者になるための一連の手順：

1.アンケートをダウンロードして記入します。

2. 10月5日までにPlugandPlay@globaltechin.comにアンケートを送信してください。アンケートが受信されたという確認が送信されます。

3.この時点では、何もしていません。 現時点では、当社の専門家がプロジェクトの初期審査を実施しています。 その結果、私たちはあなたのチームを個人的な会議に招待します（または招待しません）。

4. 15分間の個人的な会議（これも選択です）で、プロジェクトについて話し、質問に答えます。

5.繰り返しますが、何もしません。 最終版への招待状を受け取ります（または受け取りません）。

6.実際、ファイナル。 ロシアおよび国際的な専門家、およびプラグアンドプレイの代表者であるessnoの前での15分間の英語によるプレゼンテーション。

7.利益！ 少なくとも5つのチーム。
