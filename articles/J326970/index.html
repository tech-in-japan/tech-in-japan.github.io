国内のLinuxディストリビューション-サーバーおよびデスクトップシステムの「OS」：州全体の新しいBolgenOS？







最近、国営のロステック社が所有する国立情報センター（NCI）会社は、サーバーおよびデスクトップシステム用の新しいGNU / Linuxディストリビューションを導入しました。 このディストリビューションはOS （OS-RT）と呼ばれます。 今すぐダウンロードできます。 誰がシステムをテストしたか、彼らは退屈な壁紙があると言っています、これで問題はありません。 壁紙以外に、OSは何を提供しますか？



プロジェクトの著者によると、その機能は情報セキュリティに重点を置いています。 より正確には、開発者はディストリビューションを、国家機密である情報を扱うためのプラットフォームとして位置付けています。 Axisにはすでに顧客がいます-これらは、さまざまな当局、国営企業、および州の参加企業です。 自由市場では、OSも制限なしで使用できることが報告されています。 すぐに国の国営企業のすべてのオフィスで？ いくつかの技術的な詳細を見てみましょう。



ハードウェアプラットフォームに関しては、ディストリビューションはオフィスのワークステーション、サーバー、データセンターにインストールできます。 配布キットには、Windowsソフトウェアランチャーが含まれています。



OSは、RHEL / CentOS 7およびFedora Linuxの多くの要素に基づいています。 デスクトップはXfceに基づいています。 利用可能なパッケージには、LibreOffice 5.2.6、GIMP 2.8.16グラフィカルエディター、Inkscape 0.91ベクターエディター、Blender 2.76 3Dモデリングシステム、画像およびドキュメントビューアーが含まれます。 ブラウザはFirefox、メールクライアントはThunderbirdです。 配布パッケージは、VLCプレーヤー、smplayer 17.2.0、mplayer 1.3、ffmpeg 3.1.6、H.264、H.265、VP9、AAC、Opusをサポートするコーデックセット、およびWebカメラ用ドライバーのコレクションのおかげでマルチメディアにも対応しています。



サーバーコンポーネントに関しては、ストレージメディアの中で、開発者はそのようなサーバーアプリケーションとネットワークプロトコルを区別します。



* SSH経由のリモートアクセス。



*ネットワークファイルシステムNFSおよびSamba4。



* Apache Webサーバー。



* FTPサーバーvsftpd。



* CUPSプリントサーバー。



* BINDドメインネームサーバー。



* NTPシングルタイムサーバー。



* OpenLDAPおよび389-dsディレクトリサーバー。



* Kerberos認証サーバー。



* Ansible構成管理ツール。



*監視ツールZabbixおよびNagios。



*グラフィカルインターフェイスを使用したリモートアクセス用のVNCサーバー。



* IPMIおよびSNMPプロトコル。



仮想化環境はKVMハイパーバイザーであり、「負荷分散と安全なクラウドシステムの構築により、地理的に分散した安全な災害対策クラスターを構築するように設計された」クラスターシステムがあります。







プラスは、次のようなソフトウェアでサポートされています。



* 1C：エンタープライズバージョン8.3（サーバーおよびクライアント）;



* SAPプラットフォーム（サーバーおよびクライアント）;



* MyOffice;



*アンチウイルス博士 Web



Kaspersky Anti-Virusがサポートされていないのは奇妙です。



より詳細なシステム仕様は、 このリンクから入手できます。 確かに、このイノベーションを開発した会社のサイトは、現在何らかの理由で存在しています。 おそらく誰もが新しいディストリビューションをダウンロードするでしょう。



ちなみに、Opennetでは、「OS」にはAurora OSというクローンもあることに気づいたユーザーがいました。 はい、このクローンのサイトはOSサイトと同じIPでホストされています。 偶然？ 修辞的な質問。



他に何？ そうそう-システムの開発者は、すでに国内のソフトウェアのレジストリにOSオペレーティングシステムを含めることを申請しています。 現在、このアプリケーションは、通信省の専門家によって検討されています。 まあ、何、結局、代用をインポートします。
