qutIMのパッチ

RST / GHCチームの研究者はqutIMを修正しました。 これは私のブログに書かれました



oscarprotocol.cppでは、49行目を変更する必要があります。

flapSeqNum = rand（）％0x8000;

に

flapSeqNum = 0x0000;



clientIdentification.cppで、87行目を変更します。

quint16 num = rand（）％0xffff;

に

quint16 num = 0x000;



リポジトリからアセンブルする方法は、 qutIMフォーラムで読むことができます

PSあなたはバイオフリークを表現できることに感謝します-これは彼の行いです）
