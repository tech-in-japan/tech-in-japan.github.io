ComputexがWindows 8デバイスを発表

新しいWindowsのリリースバージョンが間近に迫っています。まもなくストアの棚は、このOSを搭載した新しいラップトップ、タブレット、オールインワンでいっぱいになります。 それまでの間、Asus、MSI、およびAcerは新しいガジェットのデモを行いました。





まず第一に、Asusはモノブロックで注目を集めています。モノブロックのモニターは個別に使用できます（！）タブレットとして、2つのOS（!!）、AndroidおよびWindowsのロードをサポートします。 ASUS Transformer AiOには18インチのタッチスクリーンがあり、ドックから「固定」して、通常のタブレットとして使用できます。 ドッキングステーションには、USB 3.0、Thunderbolt、HDMI、ドライブのポートがあります。 デバイスはデフォルトでWindowsを起動し、特定のキーを押すとAndroidが起動します。 Asusは、より詳細な仕様（プロセッサアーキテクチャ-ARMまたはx86を含む）をまだ発表していません。





Acerは興味深い新製品も喜んでいます。 まず、これは27インチのオールインワンAspire 7600Uであり、フルHDディスプレイは最大64（！）の同時クリックを処理できます。 モノブロックスタンドを使用すると、ポートレートモードで90度回転したり、テーブルに平行な水平位置に置いたりできます。 ケースの厚さが約3.5 cmのオールインワンには、2つのUSBポート、複合オーディオ入力、SDカードスロット、およびオプションでブルーレイドライブを取り付けるためのベイがあります。 もう1つの魅力的な機能は、ドルビーホームシアターサラウンドサウンドテクノロジを備えたサウンドサブシステムです。





次に興味深いデバイスは、2つのディスプレイを備えたASUS TAICHIラップトップです。 オープン（ラップトップ）状態では、これはキーボード、タッチパッド、ディスプレイを備えた通常のラップトップです。 閉じると、外部タッチスクリーンが使用可能になり、スタイラス操作もサポートされます。 両方の画面は独立して動作し、2人のユーザーが同時に使用できます。 このラップトップは、Ivy Bridgeアーキテクチャ、最大4ギガバイトのメモリ、SSDを搭載したIntel Core i7プロセッサに基づいています。 両方のディスプレイには、解像度が1900 x 1280のSuper IPS +マトリックスがあります。ASUSTAICHIは、13.3インチと11.6インチの2つの構成で利用できます。





Acerは、13.3インチと11.6インチの2つのバージョンで新しいAspire S7ウルトラブックを発表しました。 新しいアイテムは、タッチフルHDスクリーンを備えた、それぞれ最薄で最もコンパクトなウルトラブックモデルとして位置付けられます。 ウルトラブックは、13.3インチモデルで12時間、11.6インチモデルで9時間の長いバッテリー寿命を提供します。 すべての新しいウルトラブックと同様に、このウルトラブックも新しいIntel Coreプロセッサー（Ivy Bridge）とともに出荷されます。





AsusはTransformer Bookも発表しました。 Transformer Padとの類推により、ディスプレイは通常のタブレットのように個別に使用できます。 画面の対角線が11.6インチ、13インチ、および14インチのモデルが表示されます。 デバイスはIntel Core i7（Ivy Bridge）プロセッサ、SSD、またはクラシックハードドライブを使用します。RAMは4ギガバイトに制限されています。 フロントとリアの5メガピクセルカメラもあります。



フォームファクタは同じですが、ARMプロセッサ（nVidia Tegra 3）を搭載したモデルは、 ASUS Tablet 600と呼ばれます。 解像度1366 x 768ピクセルの10.1インチSuper IPS +ディスプレイを搭載しています。 ユーザーデータ用に2ギガバイトのRAMと32ギガバイトのeMMCを搭載しています。 2メガピクセルの前面カメラと8メガピクセルの背面カメラがあります。 素敵な追加は、GPSおよびNFCモジュール、ジャイロスコープ、デジタルコンパスです。 ほぼ完全なコピーはASUS Tablet 810です。 違いの1つは、Intel Medfieldプラットフォーム、対角11.6インチ、解像度1366 x 768ピクセル、64 GB eMMCメモリです。





同様のモデルがエイサーによって導入されました。 Iconia W510には、キーボードドック付きの10.1インチIPSタッチスクリーンがあります。 製造元によると、Iconia W510は1回の充電で最大18時間動作できます（ドッキングステーションを使用）。 このデバイスはIntelプロセッサーにも基づいており、SIMカードとmicroSD、micro-USB、およびHDMIポート用のスロットがあります。





MSIはラップトップトランスフォーマーのビジョンを導入しました。 MSI Slider S20は、スライダー形式の11.6インチマルチタッチスクリーンを備えたウルトラブックです。 したがって、画面を移動してラップトップのように操作したり、折りたたんでタブレットを使用したりできます。 重量は1.3 kgで、厚さは2センチメートル未満です。 このデバイスは、新世代のIntel CULV Chief River（Ivy Bridge）のプラットフォームに基づいています。



ご覧のとおり、秋には新しいWindowsで興味深いニュースがたくさんあります。 デバイスはさまざまな形式で表示されるため、誰もが最も適切なものを選択できます。



メーカーがx86バージョンのOSを好むのは興味深いことです。ほとんどの購入者は、クラシックインターフェイスを備えた古いバージョンのWindows用に作成された使い慣れたプログラムを使用することを論理的に恐れています。 しかし、他のメーカーからのニュースを待っています。おそらく、ARM上のWindowsでより多くのデバイスを紹介するでしょう。
