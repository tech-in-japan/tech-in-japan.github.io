「スローダウン」：「最も遅いインターネット」を持つ地球上の場所

イングランドの東にはサフォーク郡があります。 そして、ここで、Trimley Saint Martinの村では、イギリス全体（「12月現在」）で「最も遅いインターネット」は0.68 Mbit / sです。



地元の人々は、45分のビデオをダウンロードするのに8時間待つ必要があります。 この速度は、英国の平均の53倍です。



そして、これは世界の孤立したケースではありません。 この記事では、ネットワークの使用が実際のテストである地球上の他の場所について説明します。





/ Flickr / Tristan Schmurr / CC



「速度が低下したゾーン」



サフォーク以前は、イギリスの1.3メガビット/秒の反記録はグロスターシャーのミザーデンの村に属していました 。 ジャーナリストは、ローカルコミュニケーションの「レベル」とエベレストの状況を比較しました。エベレストでは、平均アクセス速度が1.5倍でした。



英国では、他の場所、特に地方では、国の政府が推奨する最低速度10 Mbpsを「遅らせ」ていました。 専門家組織「Which？」によって実施された719千の速度テストのデータによると、11の（調査された389のうち）市区町村はこの要件を満たしていません。 英国の平均ブロードバンド速度は 36 Mbpsですが。



もちろん、先進国では、そのような問題は人口の少ない地域にのみ存在します。 たとえば、アイルランドでも同様の状況が発生します。 一方、世界には「低速」に耐えることを余儀なくされている国全体と大陸さえあります。



今年9月1日の Do Speed Testのデータによると、「最も遅い」ブロードバンドアクセスを持つ国はリビアです。ここでは人口の95％以上がネットワークにアクセスできません。 5つの「遅い」には、ネパール、ナイジェリア、イラン、およびインドが含まれます。



Cable.co.ukは、アフリカの39か国の状況を調査しましたが 、10 Mbpsでは単一のしきい値に達しませんでした。 モバイルインターネットの速度を考慮すると、アフリカ大陸ではこの指標で米国よりも先に進んでいる国があります。 たとえば、ケニアでは、モバイルインターネットの平均速度は 13.7 Mbpsです（米国では10.7 Mbpsです）。



理由



いずれにせよ、誰もがインフラストラクチャとその建設コストに依存しています。 人口密度の低い地域について話すと、そのような投資の回収期間は、市内の状況とは大きく異なります。



説明のために、高速で安価なインターネットで知られている韓国の例を思い出してください。 平方キロメートルあたりの人口密度は500人以上です。 ここでは、開発の密度がかなり高いことを考えると、民家が互いに遠く離れている従来のアメリカの村に比べて、インフラ投資ははるかに「おもしろい」ように見えます。



「減速」のもう1つの理由は、競争の欠如です。 原則として、高速で安価なインターネットを備えた国では、インターネットは開発の主な推進力の1つです。 一方、場合によっては、競争の低さは、到達しにくい地域にインフラストラクチャの構築を必要とする市場に「参入」したい人がほとんどいないという事実によって説明できます（サポートなしでは利益はありません ）。



通信回線の機能に加えて、接続の「速度」の直接的な認識は、コンテンツ配信プロセスの編成方法によって影響を受けます。 たとえば、2015年の時点で、Googleには米国に7つのデータセンターがあり、ヨーロッパに4つ、アジアに2つ、ラテンアメリカに1つ、アフリカに1つではありません。 この状況では、コンテンツはGoogleから条件付きケニア、たとえばロンドンから「取得」され、企業のさまざまなプラットフォームやリソースを操作する際のユーザーエクスペリエンスに悪影響を及ぼします。 最近のインターネット社会レポートは、ローカルインフラストラクチャの欠如が重要な問題の1つであることを確認しています。 そのため、たとえば、Netflixは大陸にサーバーを展開しています。



フィリピンには、国内のインターネットの速度が遅いことに対するいくつかの「言い訳」があります。 共和国は 7100の島で構成されており、ネットワークの敷設が複雑になっています。 また、インターネットアクセスのコストにも影響します。 2014年、島の住民は2 Mbpsの料金で月額約20ドルを支払うよう提案されました。





/ Flickr / reynermedia / CC



それは何に影響しますか



コミュニケーションの質は、多くの社会経済的な結果をもたらします。 「低速インターネット」 は 、若者が農村部を離れることを余儀なくされる最後の要因ではないことが確立されています。



アフリカのラゴスでは、現代の技術エコシステムが出現しているため、インターネットアクセスの問題により、開発者と潜在的な顧客の両方が失われる恐れがあります。 さらに、インターネットユーザーのオーディエンスが10％増加しただけでも、アフリカ諸国で話をすると、国際貿易の量を0.4〜0.6％ 増やすことができます。



この問題は、当局に決定的な措置を促し、政治的闘争の要因の1つにさえなります（フィリピンの例 ）。



ただし、すべての国が「低速インターネット」の問題を解決しようと努力しているわけではありません。 たとえば、オーストラリアはランキングの50行目にあります（平均で11.1 Mbit / s）。



政府は高価な高速ネットワークを作成する計画を持っていましたが、これは加入者にとってより高価な通信につながります。 人々はそのような計画に曖昧に反応し、専門家はそのようなサービスに対する需要はないと述べた。 したがって、この大陸では「低速インターネット」の状況は変わらない可能性があります。
