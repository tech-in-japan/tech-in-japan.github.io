Windows Server 2012などでクラウドへの道を

ご列席の皆様、友人や同僚！



今日は、クラウド、つまりマイクロソフトのプライベートクラウドプラットフォームの中心となる製品であるWindows Server 2012について少しお話したいと思います。多くの人は、おそらく、新しい暦が今年の年末までにリリースされることを知っているだけではありません。 Microsoftオペレーティングシステム-Windows 8およびWindows Server2012 。ただし、これらの製品の予備バージョンにも精通しています。 まだこれを行っていない人のために、私は少し待つことをお勧めします-結局、文字通り数週間で、6月7日以降、製品の新しいバージョンが利用可能になります-リリースプレビュー。 これらのバージョンは、製品の最終バージョンに向けてさらに一歩前進します。



しかし、多くの人にとって、新しいバージョンのリリースは、新機能と新機能だけでなく、行動への合図でもあります。 はい、はい-私は新しい技術を学ぶ時間について話している。 最良の経験は間違いなく実践ですが、技術の第一人者による優れた技術コースは決して不必要ではありません（笑）。

最新のテクノロジーに関する最新の知識が必要な場所の1つは、 Microsoft Virtual Academy（MVA）です。 ここでは、製品に精通するためだけでなく、かなり深いレベルでそれを習得するために必要なすべての資料を見つけるでしょう。 また、このリソースは、マイクロソフト認定試験の準備に最適なツールです。 そして、 マイクロソフトが認定トラックを更新したことを思い出すと、MVAは医師が注文したものです！



それでも、私はより具体的になります-私たちは、Widows Server 2012に関する新しい知識を得ることに興味があります：



1） MVA上のWindows Server "8"の概要トラック （Windows LiveIDが必要）



2）6月20日から21日は、Windows Server 2012のすばらしいJumpStartになります。登録して、このオンライントラックを必ず確認してください。 確かに、これは労働者のためのレッスンです。イベントは翌日のモスクワ時間の20:00から3:00に開催されます。あなたのことは知りませんが、私はすでにショウガの注入と目のマッチを準備しました（笑顔）



これらはすべてリソースですが、私たちの前には、夏に登場する新しいコンテンツがたくさんあります！ MVAの新しいコースを定期的にチェックしてください。そうすればトレンドになります！



PS>また、MCITP：Office 365のステータスを取得する準備をしている、絶対に実用的なオタク向けのOffice 365用の興味深いJumpStartもあります（このためには、2つの試験に合格する必要があります： 70-323および70-321 ）。 ただし、このコースは3日間（5月30日から6月2日まで、また翌日の20:00から3:00まで）行われます。 私はすでに眠れぬ夜を期待しています-しかし、目標は私にとって明確で大事です!!!



新しい技術、「曇り」気分、雲のない天気の開発における強さと忍耐を心から願っています。 ITプロ分野でお会いしましょう！
