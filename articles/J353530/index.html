MTCNAテストに合格する方法（Mikrotik）100％

この記事は、初めてMikrotikから証明書を取得しようとしているが、「不明な」お金や詐欺的なお金を恐れている人に役立ちます。



昨日（2018年4月14日）MTCNAテストに100％合格しました。これをどのように達成したかをお話しします。









熟練した管理者は、これはゴミであり、このテストが最も簡単だと言うでしょう。 これは第1レベルの認定であるため、論理的です。 これが最初のステップです！ しかし、最初の証明書から正しいトレーニング戦略を受け入れた場合、残りは正常に合格できます！



準備する



  1. ネットワーク管理者は、骨髄レベルでOSIネットワークモデルを理解する必要があります。 トランスポートレベルでのデータの単位が少しだと思う場合は、MTCNAを始めるのが早すぎます。
  2. IPを理解する場合も同じです。 アドレス空間の構成要素とマスクの計算方法を理解する必要があります。
  3. 練習。 TP-Linkホームルーター以上の設定経験が必要です。 Cloud Hosted Router仮想マシンを使用して、さまざまなケースを解決します。 登録しない場合、すべてのポートは1 Mbpsの速度で動作します。 テストでは、これで十分です。
  4. 英語の基礎知識。 それでも公式wikiを正しく理解する必要があります 。 少なくともGoogle翻訳者。 このリソースは試験に合格するのに十分です。



    もちろん、教師はコンピュータネットワークの一般的な理論からの質問に答えて、知識のギャップを埋めることができますが、これはコースの目的ではありません。



    それにもかかわらず、あなたは準備ができたと信じられています。
  5. mikrotik.comに登録し、個人データを正しく入力してください。



    [フルネーム]フィールドにニックネームを入力すると、LeoMaXxの証明書を取得するのが楽しくなります。 メールでも同じです。 教師からの招待状があなたのメールアドレスに送信されます。




トレーニング



コースには3日間かかり、4日間テストされます

公式プログラムはウェブサイトmikrotik.comで見つけることができます。



ロシアでは、コースはロシア語で行われます。



実験室での作業には最も安価なRouterBoardが提供されます。









教師が行うすべてのタスクを完了します。 できない場合は、先生に助けを求めてください。 理論をよりよく理解するために、すべてのタスクが与えられています。 作成されていない各ラボでは、試験に合格する可能性が低くなります。



テストで出くわしたすべての質問は、レッスンで教師によって詳細に議論されました。 注意深く聞いてください！



授業の前に、あなたのマインドフルネスが良くなるように「グリシン」を飲んでください。



試験



試験には1時間25問の質問があります。 教師は降伏へのアクセスを開きます。

あなたは考えをまとめ 、落ち着いてmikrotik.comの個人アカウントで「テストを開始」を押します。



質問は英語ですが、最初の質問の上に「ロシア語に翻訳」ボタンがあります。

すべての質問がロシア語に翻訳されるわけではありません。 人々が翻訳を行ったという事実にもかかわらず、翻訳の品質には多くの要望が残されています...先生は未翻訳の質問の翻訳を手伝ってくれました。 私はまだtranslate.google.comに言及しました









質問のテキストと回答を注意深く読んでください。 彼らはトリックになることができます。 そして、何も考えないでください。 質問を文字通り読んでください。



たとえば、「2つのMikrotikを接続できるマスク」という質問があります。 質問には接続のタ​​イプの表示はありません。 しかし、多くの回答者はイーサネットについて考え、ポイントツーポイント接続を忘れて/ 29/30を設定します。 / 32は、大切なチェックマークなしで残ります。 / 31をベットしないでください。 MikrotikはRFC 3021を知りません！



右側では、時間がカウントされ、「tyrnet」が終了した場合は「進行状況を保存」ボタンが表示されます。 ただし、進行状況は自動的に保存されます。









質問への回答が終了し、時間が許せば、すべての回答をもう一度確認してください！ エラーを発見したことを確認しました。



質問のテキスト全体をコピーしようとしないでください。 彼らは、テストが合格しなかったとして「つかむ」と言います。 また、質問を投稿すると、計算されて取り消される可能性があります。 私はこの理論を自分で確認しませんでした。



テストの終了時に、結果がすぐに画面に表示されます。 また、L4 / P1ライセンスのキーがすぐにアカウントに表示されます









証明書を取得するには、少なくとも60％を渡す必要があります。 4％を超えていない場合は、同じ日に別のチャンスが与えられます。 しかし、私たちのグループでは誰もこれを使う必要がありませんでした。 私もこの理論を検証できませんでした。



おわりに



これらのコースは、一枚の紙のためだけでなく、実際にあなたの専門レベルを上げるためにも行かなければなりません。 結局のところ、私たちは大学にいないので、「合格するだけで」テストに合格しなければなりません。



習得した知識を実践する必要があります。



MTCNAの質問のリストは、インターネットで簡単に見つけることができます。 これらの質問に出くわすことを期待する価値はありません。 Mikrotikは常にテストを開発しています。



ただし、このテストからの質問に答える方法を理解している場合（たとえば、ネットワークの交差点を計算できる場合）、60％に合格します。



主なことは恐れることではありません！ MTCNAの過程で超自然的なことは何もありません。
