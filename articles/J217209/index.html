Google Cloud Platform Live：IaaSとPaaSの組み合わせ。 クラウドに関するムーアの法則

本日、 Cloud Platformサービスに関連するいくつかのニュースを紹介します。 関税の削減と簡素化、DevOpsクラウドツール、App Engineの管理仮想マシン（VM）、Google BigQueryを使用したリアルタイムのビッグデータ分析などがあります。



無敵の価格



クラウドプラットフォームは非常に単純な公式を提供します。仮想インフラストラクチャを使用し、現在の負荷に対してのみ支払います。 これは、独自の機器を購入するよりも有益です。 また、多額の頭金を支払う必要はありません。 ただし、クラウドプラットフォームの価格はムーアの法則と矛盾しています。過去5年間で年間8％しか下落していませんが、同時に機器のコストは20〜30％減少しています。



関税を簡素化し、「オンデマンド」でさまざまなサービスのコストを30〜85％削減することで、均衡を回復することにしました。



  1. Compute Engine仮想マシンは、サイズ、地域、クラスに関係なく、少なくとも32％安くなっています。
  2. App Engineの価格設定がより透明になりました。 フロントエンド仮想マシンとデータベース操作の価格は大幅に低下しました。
  3. Cloud Storageのコストは1 GBあたり2.6セント（0.026ドル）です（ほとんどのお客様で平均68％低くなります）。
  4. BigQueryの 「オンデマンド」料金は85％引き下げられました。


アクティブユーザーの割引



ニュースがもう1つあります。 クラウドプラットフォームに一定の負荷をかけるすべての人に割引を導入します。 私たちと一緒に保存するのはとても簡単です。仮想マシンを1か月間使用した後、割引が自動的に開始されます。 1か月間使用すると、「オンデマンド」で新しい料金に加えて30％節約できます（つまり、古い料金と比較して53％）。





4か月後、定期的に仮想マシンを使用したすべての人に追加の割引が提供されます。



現在、新しい関税と割引のおかげで、業界で最低価格が利用可能になりました。また、頭金、固定費、および将来のニーズの予測はありません。



生産性を向上



また、開発者の生産性を向上させるサービスの新機能を導入したいと考えています。



  1. クラウドでサービスを開発、テスト、実行します。 最小限の労力とワークフローの変更が保証されます。 変更をGitに保存するだけで、バイナリコードをビルドしてモジュールをテストします。
  2. すべての仮想マシンのログを表示し、フィルタリングおよび検索ツールを使用します。
  3. バグを分析するときに詳細なスタックトレースを表示します。 ワンクリックで問題の原因となったコードのバージョンを見つけます。 ブラウザーから直接コードを少し変更します。


後で、他の便利な機能を追加します。 ニュースをフォローしてください！



管理された仮想マシン



今日、開発者は、従来の仮想マシン（柔軟性を提供）と管理プラットフォーム（スケーラビリティおよび自動管理を提供）のどちらかを選択せざるを得ません。 しかし、私たちはあなたに挑戦したくありません。 出会い： 管理された仮想マシン ！ 仮想マシンで任意のコードを実行し、アプリケーションにすばやく組み込むことができます。 その際、App Engineは仮想マシンを自動的に管理します。



Compute Engineの新しいプラットフォームのサポート



Compute Engineは、 Windows Server 2008 R2プラットフォーム（機能へのアクセスが制限されています）、およびRed Hat Enterprise LinuxおよびSUSE Linux Enterprise Server （制限なし）をサポートするようになりました。



リアルタイムのビッグデータ



BigQueryを使用すると、任意のサイズの配列にインタラクティブなSQLクエリを送信できます。 起動は数秒で完了し、サービスを構成およびデバッグする必要はありません。 BigQueryストリーミングを使用すると、今日から1秒間に100,000レコードをテーブルにアップロードできます（更新はほぼ瞬時に行われます）。 これは、大量の情報フローをリアルタイムで分析できることを意味します。 同時に、BigQueryは十分な価格を誇っています-5 TB（オンデマンドリクエスト）で5ドル、5 GB /秒（割り当てられた容量）で月額20,000ドル。 これは、他のプロバイダーよりも75％低いです！



結論の代わりに



開発者であり、世界中のユーザー向けのアプリケーションを作成することは、非常に興味深い仕事です。 また、便利であることを確認しました。使い慣れたツールを使用して、クラウドでコードを記述してテストするのが簡単になりました。 管理された仮想マシンは、App Engineに基づいた柔軟性と自動管理の両方を提供します。 これに加えて、大量のデータを分析することができます。



そして、関税引き下げを忘れないでください！ ムーアの法則に従って、さらに手頃な価格になりました。仮想機器の価格は実際の機器の価格とともに低下しました。 さらに、プラットフォームを定期的に使用するだけで、追加の割引が受けられます-長期契約、固定支払い、頭金はありません。 あなたのサービスで最も手頃な料金と最高のインフラストラクチャ！



第1四半期には長い道のりを歩んできましたが、そこで止まるつもりはありません。 6月に開催されるGoogle I / Oカンファレンスで、今後の計画について学びます。
