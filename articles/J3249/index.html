ACCESS Linux Platform 1.0で導入されました

ニュースサイトが沈黙しているのは奇妙です。 ボルシェビキがこれまでずっと話してきたことは起こりました。 昨日、3GSM World Congressで、ACCESSはACCESS Linux Platform 1.0のデモを行いました。







同社は、テキサスインスツルメンツのOMAPプラットフォームで発売されたALP 1.0のデモを行いました。 同社はまた、ACCESS Linux Platformがモバイルデバイス用の新しいシステムの正式名称であると述べました。 さらに、ACCESS Linuxプラットフォームに含まれるACCESS Garnet OS互換性レイヤー（Palm OS Garnetとも呼ばれます）は、ACCESS Garnet VMと呼ばれます。



ACCESSは、製品開発キット（PDK）は既にライセンシーに提供されていると述べました。 また、SDKも発表されます。これには以下が含まれます。



*すべてのプラットフォームコンポーネント（Linuxリファレンスカーネルを含む）

*リファレンスドライバーと開発キット

*アプリケーション開発キット

* Eclipseベースの完全な開発ツールセット

*ドキュメント
