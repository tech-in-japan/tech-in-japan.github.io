Nokia構成ツール-スマート管理アシスタント

みなさんこんにちは！



本日は、USBケーブルまたはBluetooth接続を介してSymbianを搭載した1台以上のNokia製電話機のさまざまな設定をすばやく構成するために設計されたWindowsアプリケーションNokia Configuration Toolについてお話します。



まず、このアプリケーションは、たとえば、すべての従業員の電話を企業のWi-Fiネットワークに接続し、VPNを構成し、すべての同僚の番号をアドレス帳に追加するか、必要なアプリケーションをダウンロードしてインストールする必要があるシステム管理者に役立ちます。 Nokia Configuration Toolを使用して行うことができるさまざまな設定が多数あるため、アプリケーションの使用例は多数あります。



Nokia Beta Labsは、多くのアドオンを備えたNokia Configuration Tool v6.3の新しいベータ版をリリースしましたので、今後そのことについてお話します。 新機能のリストには、すべてのSymbian電話の自動検出、およびメールアカウント、SIP、VoIPサービス、MS Exchangeなどの設定オプションのテキスト説明が表示されているため、管理者は構成に必要な簡単な情報をすばやく取得できます。
