タッチパッドでla iPodを回転させる

ほとんどすべてのラップトップにインストールされるSynapticsのマルチタッチおよびその他の便利なタッチパッドトリックは、多くの場合、タッチパッドドライバーによって実装され、ハードウェアコンポーネントにほとんど依存しないことを誰もが知っています。 それは、異なるシステムでのサポートが異なるだけです。

約6か月前、ネットワークを掘り下げた後、マウスの右ボタンを2本の指で押し、左ボタンを3本の指で押すように設定しました。その後、マウスに手を頻繁に移動し始めました。 しかし、私はもう少し欲しかった。

人間のシナプスに夢中になって、私はCircularScrollingと呼ばれる興味深いものを見つけました。これは「循環スクロール」を意味します。 彼女はiPodプレーヤーの多くに馴染みがあり、長い文書をスクロールするのに非常に便利です。 当然、右端に沿ってスクロールすることは最良の選択ではないため、同じことを望んでいました。

それでは、セットアップに取り掛かりましょう。 途中で、2本と3本の指によるクリックの処理がどのように実装されるかを示します。



お気に入りのエディターxorg.confを開き、タッチパッドを担当するセクションを探します。



Section "InputDevice"

Identifier "Synaptics Touchpad"

Driver "synaptics"

Option "SendCoreEvents" "true"

Option "Device" "/dev/psaux"

Option "Protocol" "auto-dev"

Option "CircularScrolling" "on" #

Option "CircScrollTrigger" "2" #

Option "HorizEdgeScroll" "on" #

Option "SHMConfig" "on" # ,

Option "TapButton2" "3" # -

Option "TapButton3" "2" # - ()

EndSection









バインディングは次のように配布されます。

0任意

1上端

2右上隅

3右端

4右下隅

5下端

6左下隅

7左端

8左上隅
