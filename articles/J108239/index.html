HTC Desire Z：電話、キーボード、ルートアクセスのセットアップ

昨日、私は英国で購入したHTC Desire Zの所有者になりました。約1日、この喜びのすべてのHabrausersとこの良いニュースを共有したいという欲求に圧倒されました。 実際、もちろん、電話を自分の現実に適応させるプロセスについて話したかったのです。 すでにいくつかのレビューがありましたが、私の意見では実際的な経験が不足しています。

設定のインポートから始め、既存のアプリケーションでの作業と新しいアプリケーションのインストールの経験を継続し、省エネに触れて、デバイスをルート化することで終了します（もちろん、オプション）。 同時に、このノートには独立した価値はなく、インターネット上のいくつかのソースから編集されています。 面白い？



マニュアルを読む



HTCからの別れの言葉：放電が激しい場合は、USB経由で充電することはお勧めしません。 私のように、英国で電話を購入し、たとえばSamsungやNokiaなどのネットワークからMicroUSB充電器を購入した場合（元のSamsung充電器はEurosetで約500ルーブルです。おそらくもっと安いオプションがあります）。 また、過剰を避けるために、HTCは接地されたコンピューターにのみ電話を接続することをお勧めします。 それでも、仕事のために電話を準備するプロセスには驚きました。 電話をオンにするには、電源ボタンを押してください。 バッテリーカバーを開け、バッテリーを取り外し、絶縁フィルムを取り外す必要があります。 次に、SIMカードとバッテリーをそこに入れると、電話は作動可能になります（バッテリーは半分以下に充電されています）。 マニュアルでは、価値のあるものは見つかりませんでした。



初期設定



電源を入れた後に電話が私に最初に提供したのは、連絡先のインポートでした。それから出てきたものはすべて、以前の電話からでした。 このメニュー項目についてHTCに感謝します！ SonyEricsson Vivazから連絡先のみがインポートされました。 ただし、ありがとうございます。SIMカードには重要なSMSメッセージが残っていました。

次のステップは、インターネットの構成です。 マスターに、私はWi-Fi接続しかないことを指摘しました（自宅では無制限、はい）。 電話で3G / EDGEを手動でオンにします（電話では、トラフィックは無制限ではありませんが、前払いは60 MBのみです）。 ですから、お金をfallしまないために-明らかに、セルラーネットワークを介したインターネット接続について言及していないことを示すのは理にかなっています。 しかし、私は無制限のインターネットに接続することを考えています-私は月90ルーブルのようなオプションがあります。

もう1つの重要なポイントは、HTCSense.comへの登録です。 特別な利点はありません（盗まれた電話を返すためにそれを使用したり、少なくともそこからデータを削除することさえできません）が、インターネットを介して私のアパートで電話を検索する機能はプラスです。 最愛の人の電話を探して自分の電話を探す必要はもうありません。

位置情報設定：好みに合わせて。 すべてがオンになっています。 ちなみに、私たちの都市のジオロケーションは、望まれるものが多くあります（精度1〜2キロメートル）。



用途



驚いたことに、電話からのすべてのアプリケーションは、機能を即座に実行します。 ソーシャルアプリケーションと音声入力の両方が機能します。 ロシアのソフトウェアキーボードも気に入った。 しかし、ハードウェアキーボードはロシア語で動作することを望みませんでした。 Androidマーケットで利用可能なロシア語キーボードが私の助けになりました。GameWarriorの 「 ロシア語キーボード 」という単語を検索してください 。 いずれかのコントロールで入力メソッド「ロシア語キーボード」をアクティブにすると、音訳ハードウェアキーボードが得られます。 いくつかの「秘密」：

  • w：sh
  • u：w
  • b：q
  • b：qq
  • yu：yuまたはuu
  • うゆ：うゆ


私はそれが不便だと思った：しかし、いや、ほんの数分で手自身が音訳に慣れてしまい、ロシア語の文字でハードウェアキーボードの操作を改善する方法を彫刻し、探す方法を急ぐ必要性を感じなくなった。

ロシア語のキーボードに加えて、電話自体を操作するためのAndroidターミナルエミュレーターと、リモートサーバー（ssh）を操作するためのConnectBotをインストールしました。 Dolphin Browser™HD2ブラウザー、 Alarm Clock PlusV2目覚まし時計、 Evernoteクリッパー、 YouTubeアプリケーション（ここではより簡単にSymbianにありました）が、画像、 ES File Explorerファイルマネージャー、およびいくつかのゲームを補完します。

ご希望の方は、Android MarketからMoreLocale 2プログラムをインストールすることにより、英国の電話のインターフェースをRussifyできます（アイデアについてDaemonIに感謝します ）。



省エネについて



いつものように、電力消費はユーザーの食欲に依存します。 現在の食欲で、バッテリーは1日以上持続しますが、誰がそれを必要としますか？ 私は最終的に2日間に到達することを望んでいます（すべての実験を電話で終了しているため）。 したがって、最後の充電からのレイアウトは次のとおりです。21時間6分の操作後、バッテリーの18％が残ります（20％で省エネモードに切り替えるように電話を設定します）。 電気の大部分は、かなり大きな画面（69％）の作業で消費され、12％は音声通話に費やされました。 これで電話機が充電され、すでに完全に異なる番号が表示されています（[設定]⇒[電話について]⇒[バッテリー]⇒[バッテリーの使用]）。 消費電力に基づいて、適応ディスプレイの明るさをオフにしようと決めました（30％で十分です）。 今日の通話は約10〜15分でした。 インターネットでは、プロセッサのオーバークロックはエネルギー消費に害を及ぼさないと彼らは書いているので、私はすべて同じようにします。 唯一気になるのは、負荷がかかった状態で電話が暖かくなり、過熱を少し恐れていることです。



こんにちはルート



古いLinuxoidのように、電話でのルート権限の取得を無視できませんでした。 そして、電話をかけたり、仕事をしたり、プレイしたりするには電話が必要ですが（Android 2.2ではこれをすべて快適に行うことができます）、root権限を取得することは単に原則の問題です！ 私は、ユーザーが電話でひどいことをすることができるはずだと確信しています。 たとえば、トイレやバスルームで電話をdrれさせ、使用できなくすることができます。 失敗したファームウェアで電話を台無しにすることもできます。 しかし、何らかの理由で、製造元は最初のアクションを許可し、2番目のアクションから保護しようとします。 まあ、いや！

それであなたは行き​​ます！ 危険 理論的には、この操作により、お使いの携帯電話を2万ルーブルから数百ルピーのレンガに変えることができます 。 私は電話を取り戻し、すべてがうまくいきましたが、念のため、古き良き「7回測定-1回切断」と言ったことを思い出してください。 この方法は私が発明したものではなく、すばらしい英語フォーラムの記事を翻訳しただけです。

XDA-Developersフォーラムには、 ダミーのためにHTC Desire Zでルートアクセスを取得する方法の説明があります 。 使用します。 特に、最初の投稿からファイルをダウンロードする必要があります（ダウンロードには登録が必要です）。 これはポイント番号0になります。 さて、または登録したくない場合、そして同時に私を信頼する場合は、私のブログから登録せずにダウンロードできます 。

次に、Androidマーケットからではなくプログラムのインストールを有効にし、ルート化のために電話を準備する必要があります。 これを行うには、[設定]⇒[アプリケーション]に移動し、[ 不明なソース ]をオンにし、[ 高速ブート]をオフにします。 [開発]サブメニューに移動し、[ USBデバッグ]チェックボックスをオンにします 。

ES File Explorer、Astro、またはその他のファイルマネージャーを使用して、すべての準備が整いました。パッケージのポイント0でダウンロードしたVISIONary +パッケージのインストールを開始します。 次に、すべてのアプリケーションのリストから実行します。

ハッキングプロセス：[TempRoot now]をクリックして、しばらくお待ちください。 [今すぐPermRootを試してください！]をクリックし、プログラムにスーパーユーザー権限を付与することに同意します。 ただし、エラーが発生しました。TempRootを再度実行する必要がありました。 PermRootを実行した後、電話機は長時間再起動します（FastBootを無効にしましたか？）。

そして最後に、XDA Developersフォーラムからのいくつかの質問と回答。

  1. プロセスは可逆的ですか？



    はい、プロセスはリバーシブルです。 VISIONary +は、root化も解除できます。 プログラムを再度実行し、「Unroot now！」を実行するだけです。
  2. ルート化により、電話またはメモリカードからデータが削除されますか？

    いいえ、アクセス権のみを変更します。
  3. 電話をラッピングすることで保証を失いますか？

    これは難しい質問です。 ただし、一般的には、すべてをそのまま返すことができます。
  4. ルート化プロセスの完了後、VISIONary +をアンインストールできますか？

    はい、アンインストールできます。 ルート権限は残ります。
