Oracle Linuxのディストリビューション、アップデート、およびエラッタが無料で利用可能になりました

オラクルは、RHEL 6およびOracle Linux 6アプリケーションの一部に関する認定声明に加えて、本番環境での使用を含め、Oracle Linuxのディストリビューション、アップデート、およびエラッタが無料で利用可能になったことを発表しました。



Oracle LinuxはRHELソースコードから構築され、完全に互換性がありますCentOSおよびScientific Linuxに対する重要な利点は、ハードウェアメーカーのサポートと、Red Hatとほぼ同時にリリースされるアップデートのリリース日です。

以前は、アップデートを迅速に入手するには、有効なサポート契約が必要でした。これには、Oracleから年間119ドル、Red Hatから年間349ドル（2ソケット、1ゲストシステムの制限付き）が必要でした。



パブリックリポジトリは次のリンクから入手できます。

public-yum.oracle.com



Linuxの上級副社長でありOracleの仮想化プラットフォームであるWim Coekaertsは、次のように説明しました。

すばらしい点は、サポートなしで完全に更新されたシステムが必要な場合はそれを使用し、後でサポートが必要な場合はサブスクリプションを取得することです。 シンプル、便利、効率的。


ただし、サポート契約を更新する機能を忘れないでください。
