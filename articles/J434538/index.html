宇宙のための戦い：衛星インターネット市場で何が起こっているのか

12月中旬、米国連邦通信委員会（FCC）はLeoSat衛星インターネットプロジェクトを承認しました。 ネットワークは2019年に導入が開始されます。 SpaceX Ilona Mask は 、デバイスを軌道に投入します。 私たちは企業の計画について話し、なぜ軌道上の多数の宇宙船がそんなに良い考えではないのかについて話します。





/ Flickr / NOAA / PD



衛星インターネットLeoSat



LeoSatネットワークは 、78の衛星で構成されます 。 それらは、地球から1400キロメートルの高度に配置されます。これは、他の衛星システム（GLONASSやGPSなど）の軌道よりも10倍低いです。 会社の代表者によると、地球にデバイスが近接しているため、光ファイバーを介した場合よりも惑星の異なるポイント間で1.5倍の速度でデータを転送できます。 たとえば、ロンドンからシンガポールまで、LeoSat信号は119ミリ秒で到達し、ファイバでは181ミリ秒で到達します。



来年、LeoSatは最初の2つの衛星を宇宙に打ち上げ、ユーザーがすぐに利用できるようにします。 2022年までにネットワークを完全に展開する予定です。



各LeoSat宇宙船には4つのレーザーが装備されており、隣接するデバイスと通信します。 また、衛星には10個のアンテナがあります。 Kaバンド電波を使用した地球端末との通信に必要です。 データ転送速度は1.6 Gb / sと予想されます 。



LeoSatのお客様は5Gオペレーターになります。 通信業界の代表者によると 、新世代の地上通信はネットワークには十分ではありません。必要な5G信号伝送速度は、世界の平均インターネット接続速度の15倍です。 衛星は通信システムを補完し、データ転送を大幅に加速できます。



他に誰が宇宙に衛星を打ち上げるか



LeoSatに加えて、さらにいくつかの衛星インターネットプロジェクトがFCCの承認を受けています。 これらのうち、最も有名なものはStarlinkとOneWebです。



StarlinkはElon MuskとSpaceXのプロジェクトです。 これについては、 以前に詳しく説明しました 。 SpaceXはすでに2つの衛星のプロトタイプを軌道に打ち上げており、2019年から2020年に最初の1,600機の配備を準備しています。 現在、同社はネットワークの開発に費やす追加の5億ドルの投資のために取引を終了しています。



Airbus、Virgin Group、Qualcommなど、多くの企業がOneWeb衛星技術に投資しています。 宇宙船のプロトタイプの打ち上げは2019年に予定されています。 2月7日、10個のOneWeb衛星を搭載したソユーズロケットが、 Kourou Cosmodromeから軌道に投入されます。 デバイスの総数は 600個です。



FCCの承認を受けたもう1つのプロジェクトはTeleSat Canadaです。 同社は 117個の衛星を軌道に投入します。 ネットワークは2021年までに展開される予定です。





/ Pixia / フリー写真 / PD



Facebookは、発展途上国向けのインターネットにも取り組んでいます。 2018年7月、同社は独自の衛星を作成していることを確認し、すでにFCCに申請書を提出していました。 別のプロジェクトはLinkSureです。 これは、グローバルな無料衛星インターネットのシステムを開発している中国の会社です。 最初の宇宙船は2019年に打ち上げられ、LinkSureネットワーク全体が2026年までに配備される予定です。



詰まった軌道



非常に多くの衛星システムが恩恵を受けるかどうかについての意見はまったく異なります。 IT業界の一部の代表者は、技術が農村部や発展途上国に住む人々を助けると信じています。 宇宙プロジェクトに従事している非営利組織Secure World Foundationのクリスタルウィルソンは、この場合、地上ネットワークインフラストラクチャにサービスを提供できる専門家を見つけることが重要だと言います。



衛星インターネットは貧しい人々のためではなく、裕福な人々のための可能性が高いと確信する人々がいます。 これは、そのようなソリューションの開発者によっても述べられています。 LeoSat CEO は 、彼の会社が大企業向けのプレミアムサービスを作成していると言います。


ロンドン大学ユニバーシティ・カレッジのマーク・ハンドリー教授（マーク・ハンドリー）も同じ意見を表明し 、スターリンクの仕事をシミュレートしました。 彼によると、たった1つの衛星の打ち上げには数千万ドルの費用がかかります。 したがって、SpaceXのお客様は、最速のインターネット接続速度に対して大金を払うことをいとわない国際企業になります。



一部の技術専門家は、衛星インターネットの概念を疑っています。 スタンフォード大学のザックマンチェスター教授によると 、1990年代後半に、いくつかの企業がセルラー通信を低軌道衛星ネットワークに置き換えようとしました。 宇宙システムは地上システムよりもゆっくりと開発されたため、衛星プロジェクトは顧客を引き付けることができず、閉鎖されました。



マンチェスターは、衛星インターネットの現在の状況は電話サービスプロバイダーの経験と変わらないと考えています。したがって、SpaceXおよびOneWebプロジェクトは、クリエイターや投資家の期待に応えられない可能性があります。



専門家は、衛星インターネット企業がサービスのコストだけでなく解決しなければならないことに注意しています。 2つの重大な技術的困難があります。 最初はスペースデブリです。



FCCによると、確率が45％のSpaceXデバイスの断片は、6年ごとに人に当たります。


そして、地球に近い軌道には他の企業の何百、何千もの衛星が存在することを考えると、地球の住民への危険は増大するだけです。 この場合、残骸は単に軌道を詰まらせます。 Starlink衛星は約5年ごとに交換する必要があるため、スペースデブリの量は膨大になります。



11月末に、連邦取引委員会は、スペースデブリ問題の解決を目的として、多くの規制文書の作成を提案しました 。 そして、ヨーロッパと米国の政府は、デブリを追跡し、軌道を「掃除」するためのシステムを開発しています。





/ Flickr / NASA / PD



2番目の問題は、システムの情報セキュリティです。 衛星の1つへの信号が攻撃者によって傍受された場合、攻撃者はネットワーク全体の動作に影響を与えることができます。 これは、インターネット接続の中断だけでなく、宇宙空間でのデバイスの衝突にもつながる可能性があります。 彼らはすでにこの問題の解決に取り組んでいますが。 オプションは、無線信号を暗号化するか、地上局からのコマンドの必須認証を実装することです。



次は何ですか



衛星インターネットの競争はますます大きくなります。 すでに、一部の企業はライバルを追い越そうとしているため、国体の承認なしに宇宙船を打ち上げています。 たとえば、アメリカの新興企業であるSwamp Technologiesは、インドの宇宙センターから衛星を軌道に送りましたが、FCCからの申請の確認を待ちませんでした。



ただし、この技術の可能性はまだ不明です。 専門家は、衛星インターネットが次の4-8年で本当に現れると見積もっています。 そうして初めて、企業が光ファイバー接続の代替品を見つけることができたのか、それとも衛星の期待が高すぎたのかが明らかになります。
