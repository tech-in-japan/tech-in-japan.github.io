JS自体には型はなく、単語のみであり、現在の値に基づいて取得されます。通常は必要ありません。たとえば、数値の文字列表現がある場合、JSは自動的に文字列を数値に変換します。

var w = '5';

alert（q / w）//結果11

アラート（5 == '5'）; // true;



しかし、比較するとき、たとえば、0と比較するときなど、タイプを正確に決定する必要がある場合があります

アラート（0 == ''）; // true


この比較では、JSは両方の値が空であることを確認し、これを回避するためにそれらを同等化するため、型も考慮する必要があることをJSに伝える必要があります。===これがまさに

alert（5 === '5'）// false値は似ているが、数字は文字列と等しくない

alert（0 === ''）// false数値は文字列と等しくない
