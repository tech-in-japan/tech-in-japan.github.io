Silverlight 4 Betaが利用可能になりました。 中身は？

WindowsおよびMac OS X向けのSilverlight 4 ベータ版が昨日PDC09で発表され、Silverlightが現在インターネットにアクセスできるコンピューターの45％にインストールされていることが発表されました。



新機能には、Webカメラとマイクのサポート、印刷、Google Chromeの公式サポート、ローカルコンピューターとの統合、Officeおよび他のCOMオブジェクトとの対話、HTMLコードのレンダリング、ドラッグアンドドロップのサポート、およびその他の機能があります。



開発者向けのランタイム（このバージョンはエンドユーザー向けではありません）に加えて、他のコンポーネントの更新も利用できます。
  • Visual Studio 2010 Beta 2またはVisual Web Developer Express 2010 Beta 2
  • Visual Studio 2010用のSilverlightツール
  • Expression Blend for .NET 4プレビュー
  • WCF RIAサービス （以前の.NET RIAサービス）
  • Silverlight 4 SDK CHM （オフラインドキュメント）-またはオンライン
  • Silverlight 4用のSilverlightツールキット
  • Silverlight Media Framework
  • ランタイム自体： WindowsおよびMac （ツールに含まれています）
  • Silverlight 4 SDK-ツールに含まれています
Silverlight 4ベータ版は、Silverlight 3のリリースからわずか4か月後にリリースされます。これは、フィードバックに基づいており、世界中の開発者が去ることを願っています。



それで、Silverlight 4 Betaに登場したもの：
  • ビジネスアプリケーション開発

    • Silverlightアプリケーションのコンテンツの印刷

    • フルテキスト、リンク、画像、編集用のRichTextAreaを含む60以上のカスタマイズ可能なコントロール

    • 象形文字と右から左へのスペルを含む言語を含むローカライズサポート

    • サーバー側とクライアント側の共通コード

    • データバインディングの改善

    • MEF（Microsoft Extensibility Framework）のサポート

    • RIAサービスでの更新

    • Visual Studio 2010の完全なビジュアルエディター



  • インタラクティブ機能の新機能

    • ウェブカメラとマイクのサポート
    • サーバー側なしのローカルビデオおよびオーディオキャプチャ
    • バッファおよびドラッグアンドドロップのサポート
    • マウスの右ボタンとマウススクロールホイールをプログラムで処理する
    • Silverlight 3よりも起動パフォーマンスが倍増
    • より動的なインターフェースのためのアニメーションの改善

    • ジェスチャーおよびマルチタッチの他の側面のサポート

    • マルチキャストストリーミングのサポートとWindows Mediaサービスとの統合

    • H.264を含むPlayReadyを使用したコンテンツの保護



  • ブラウザを操作して、ローカルコンピューターと統合する

    • 現在、アプリケーションはサンドボックス（通常）と信頼（ローカルコンピューターへの大きなアクセス権を持つ）に分かれています

    • サンドボックスアプリケーション用

      • Silverlightアプリケーション内のHTML統合

      • 画面の隅にあるリマインダーウィンドウ（Outlookなど）

      • DRM保護を使用してインターネットから切断する

      • ブラウザ外で作業する場合のウィンドウサイズの完全な制御



    • 信頼できるアプリケーションの場合（以前のアプリケーションに加えて）

      • ドキュメント、音楽、写真、ビデオが含まれるフォルダー内のファイルの読み取りと書き込み（Windowsでは「マイ」フォルダー、Mac OSでは同等）

      • COMオブジェクトとの相互作用（Officeとの統合、USBへのアクセスおよびその他の機能）

      • 組織内のグループポリシーによるアクセス制御

      • 全画面モードでのフルキーボードアクセス

      • 追加の許可なしでのクロスドメイン相互作用

英語では、新機能の詳細な説明と 、個々の機能に関するビデオチュートリアルを利用できます。 近い将来、ロシア語を含むより多くの資料が登場します。
