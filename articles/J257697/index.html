新しいアセンブリの開始オプションとその他の改善

みなさんこんにちは！



次回のVivaldiブラウザビルドは、1.0.174.8の番号でリリースされました。以前のように、私たちはあなたの希望の実装に取り​​組みました。 特に、現在のバージョンでは、次に起動したときにブラウザーに表示されるものを指定する機会を提供します：ブラウザーが前回閉じられたタブのセット、指定したホームページ、または単にExpressパネルのいずれかです。 将来、この機能はさらに拡張されますが、現時点では基本的なオプションのみを追加しました。



毎週恒久的になっているもう1つの作業分野は、インターフェイスのパフォーマンスの観点からブラウザーのパフォーマンスを向上させることです。 今日のビルドでは、コードの一部を書き直しましたが、タブの操作にもっと注意を払いました。



特に、タブ切り替えの改善に取り組んできました。 これらのアクションに注意を払い、エラーや欠落を見つけた場合はお知らせください。 さて、この機能はまだ開発プロセスにあることを忘れないでください-インターフェイスの速度と利便性に引き続き注意を払う予定です。



また、このアセンブリに新しいローカライズが登場しました-リトアニア語が追加されました。



以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
