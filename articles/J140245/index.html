Highscreen Yummy Duo-新しいデュアルSIM Androidスマートフォン



2枚のSIMカードをサポートする最も薄いスマートフォンがロシアに登場しました。 Higshcreenブランドで安価なスマートフォンを製造する有名なロシアの会社Vobis Computerは、新しいHighscreen Yummy Duoモデルを市場に投入しました。 デバイスには、4.3インチの対角線、GPSナビゲーター、フロントカメラ、SIMカード用の2つのスロットを備えた巨大な画面が装備されています。 製造元によると、デバイスの厚さは10.5ミリメートルを超えていません。



スマートフォンはMediaTek MT6573ハードウェアプラットフォーム上に構築されており、Gigabyte GSmart G1345、Highscreen Jet Duo、LG Optimus Link Dual SIMなどの携帯電話で既に使用されています。 開発者によると、新しいプラットフォームを使用すると、デバイスの起動時間を5秒に短縮し、グラフィックパフォーマンスを20％向上（Qualcomm MSM7227Tプラットフォームと比較）し、1回の充電から2日までの携帯電話の平均バッテリー寿命を確保できます。



簡単な技術仕様：

  • 4.3インチの容量性、WVGA解像度（480x800）のタッチスクリーン、マルチタッチテクノロジーのサポート。
  • 512 MB RAM;
  • 512 MBの内部メモリ。
  • 最大32 GBのmicroSDメモリカードのサポート。
  • GPS受信機
  • GSM / GPRS / EDGE 850/900/1800/1900 MHz、UMTS / HSDPA 900/2100 MHz、Wi-Fi、Bluetooth;
  • 5メガピクセルのカメラ、オートフォーカス、LEDフラッシュ。
  • ビデオ通話用のフロントカメラ。
  • Android 2.3


製造業者によると、店舗でのスマートフォンのコストは9000ルーブルを超えません。
