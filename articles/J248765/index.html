マイクロソフトがStudent Developer Packおよびその他のGitHub Education Newsに参加





前の投稿を続けます。 Visual Studio Community 2013およびDreamsparkのその他の製品は、 Student Developer Packの一部として学生が利用できます。 さらに、hackpledge.orgは1時間のヘルプを提供しており、GitHubは教師の話を聞きたいと考えています。 カットの下の詳細。



浸透性のバックパックが重くなっています



GitHubは、発売以来、Student Developer Packに最初に追加されたのが、Visual Studioを搭載したMicrosoft DreamSparkです。 既にVisual Studioをダウンロードし、学生開発者向けのMicrosoftプログラム-DreamSparkにアクセスするために登録することができます。



DreamSparkを使用すると、学生はソフトウェア、ストア内の開発者アカウントを使用してWindowsおよびXbox for Windows用のアプリケーションとゲームを公開する機能、eラーニング資料など、さまざまなマイクロソフト製品にアクセスできます。 すべて無料で、これは喜ばしいことです。



Visual Studioをダウンロードし、 Developer Packを使用してDreamSparkにアクセスできます。



無料メンタリングアワー



hack.summit（）の作成者からのニュース。 hack.pledge（）は、4,000人の開発者のネットワークであり、学生向けのコード作成に関する1時間の無料オンラインコーチングを提供しています。 実験室の仕事をする時間がない人に非常に関連しています。



hackpledge.orgで 1時間のサポートを受けてください 。



先生方から聞きたい



GitHubは「GitHubを教育で使用できるすべての方法を聞いてうれしい」と、カリフォルニア大学バークレー校のコンピューターサイエンスとオープンソース研究者が率いる学生チームが参加できるFIRST Robotics Competitionに参加するキャンペーンを行っています。最初のロボットを作成します。 競争の目標は、科学技術に興味を持つ学生たちです。



さらに、GitHubは教師に、GitHubが専門的な活動でどのように使用されるかを説明するように依頼します。 彼らはここに書くために書きます 。
