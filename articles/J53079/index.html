テスターはWindows 7で2000のエラーを発見しました



マイクロソフトは、新しいWindows 7オペレーティングシステムで、ベータテスト中にユーザーが発見した2,000を超えるエラーを修正することを約束しています、とDaily Techは書いています。



ベータテストのピーク週に、Microsoftは15秒ごとにテスターから1つのエラーメッセージを受け取りました。 ベータテスト中に合計で50万件を超えるエラーメッセージが受信されました。これは、各開発者の500メッセージに相当します。



さらに、テスト中に、マイクロソフトは280万の一意の識別子を使用して1,000万のデバイスのインストールデータを記録しました。 デバイスの75％以上が、オペレーティングシステムに付属のドライバーで正常に機能しました。 残りは、Windows Updateサービスに接続した後、または製造元のWebサイトからドライバーをダウンロードした後に機能し始めました。



Windows 7は、Windows Vistaの後継となるように設計されています。 オペレーティングシステムの正確なリリース日は不明ですが、アナリストと市場参加者は、2009年末または2010年初頭に登場することを期待しています。 （ C ）
