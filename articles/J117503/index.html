Ubuntu 11.04 Natty Narwhal beta2がリリースされました-リリース前の最新バージョン

人気のあるオペレーティングシステムの2番目のベータ版を静かにリリースしました。 今回はリリース候補がないことに注意してください。つまり、これはリリース前の最新バージョンであることを意味します。

多くの変更はありませんが、視覚的には目立ちます。



簡単に言うと、これは次のとおりです。

  • Unityによって更新および修正され、パネルがスムーズに消えて、見た目が良くなりました。 一般に、検索と安定性が向上しました。 また、Twitterなどの別の場所で検索することもできます
  • 最後に、彼らはUnityのオプションと設定に注意を払いました。 これで、パネルをいつ表示するか（上隅にカーソルを合わせるか、カーソルを画面の左端に近づけるか）を選択できます。
  • 外観-更新されたアイコンと、RadianceとAmbianceの標準テーマ。 スクロールバーが暗く/明るくなりました（次の変更に関連して常に正しいとは限りません）
  • オーバーレイスクロールバー-美しいが、珍しいアップデートがデフォルトでインストールされるようになりました。 一部のウィンドウには、細いスクロールバーがあります。



    マウスが近づくと、スライダーが表示されます。 Gnome / GTK +を使用するアプリケーションでのみ機能するため、たとえばFirefoxには当てはまりません。 しかし、それは標準プログラムです。 一部のプログラム（ターミナルなど）にはパッチが必要です。


事実、この素晴らしいスクロールバーはGnome / GTKベースのアプリケーション（Natty Narwhal）にのみ実装されるため、Mozilla Firefox（XULはサポートされません）、LibreOfficeおよびその他のQTベースのアプリ、Gnome-ターミナル（特別なパッチが必要）。

Firefoxは明らかに動作しません。GTK+を実際には使用しません（テーマごとに物事を描画するために下位レベルのパーツを使用しますが、実際のウィジェットは使用しません）。



  • 通知。 現在、特定のイベントでは、左上隅の隅が青で強調表示され、注目を集めています。


更新されたソフトウェアも含まれます：

  • バンシー2.0
  • Shotwell 0.9.1
  • Mozilla Firefox 4.0
  • Python 2.7


使用されているLinuxカーネルKernel 2.6.38-8

また、ATIビデオカード用のプロプライエタリドライバーが登場し、要求の厳しいアプリケーションやゲームでの作業がさらに快適になりました。 そして今、プロプライエタリなドライバーの動作モードでは、Ubuntuロゴは、下に4つのドットがあるテキストラベルではなく、ロード時に美しい画像で表示されます。

Application Centerの「試用」オプションは、12個のプログラムだけでなく、すぐにすべてのユーザーが使用できるようになりました。

また、Unityコンテキストメニューを設定するための構成が徐々に表示されます（たとえば、ブラウザーメニューで新しいタブを開くなど）。

システムの全体的な安定性も向上しました（予想どおり）。
