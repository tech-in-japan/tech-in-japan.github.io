2GIS for Windows Phone（実際）

2GISがあるプラットフォームのリストがもう1つ補充されました。 市内地図を持つ組織の無料ディレクトリがWindows Phoneで利用可能になりました。









アプリケーションは既にWindows Phoneストアにあり、ダウンロードできます。





2GIS for WPは次のとおりです。

-ロシア、ウクライナ、カザフスタン、イタリアの200以上の都市へのアクセス。

-アトラクションの3Dモデルを含む美しく関連性の高いマップ。

-組織の最も包括的な正確なディレクトリ。

-WP7とWP8の両方のサポート。



アプリケーションの主な機能は、 2GIS APIで動作することです。つまり、動作するにはインターネット接続が必要です。 はい、これは2GISの伝統に完全には準拠していません。それ以前は、すべてのアプリケーション（ PC 、 モバイルバージョン ）が完全にオフラインだったためです。



ご存知のように、WP7プラットフォームには特定の制限があります-ネイティブコードでアプリケーションを作成する方法はありません。 私たちにとって、これはデバッグされたライブラリの蓄積された荷物を使用することが不可能であることを意味していました。 すべてをゼロから書くことも万能薬ではありませんでした-開発に利用できるツールと必要な人件費は、望ましいパフォーマンスを保証しませんでした。 したがって、API 2GISのオンラインアプリケーションがソリューションになりました。 これはパフォーマンス要件を満たし、WP8とWP7の両方で電話所有者のカードとディレクトリにアクセスできます。



API 2GISについて少し。 実際、APIで実行されているすべての製品（例： Flamp 、 NGS 、 Rugion ）では、データが毎日更新されるため、最新かつ最新の情報が表示されます。



オフラインバージョンの2GIS for WPがリリースされるかどうかの質問を予想しています。 はい、できますが、WP8のみです。 そして、その中に現在のバージョンにはない多くの機能が表示されます。 いつ？ 正確な日付はまだありません。 詳細をお知らせしますので、ご期待ください。



Windows Phoneストアからアプリケーションをインストールすると、迷子になりません！
