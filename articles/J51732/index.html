Apple TVはゲームコンソールになれますか？

Apple TVの所有者の多くは、 Appleセットトップボックスがマルチメディア経済にどの程度適合するかを調べるためのオンライン調査に参加するよう招待されています。ニュースをご覧ください： 「オンライン調査はApple TVを更新する前兆となります 。 」

この点で、Apple TVセットトップボックスの今後のアップデートについて、さまざまな噂が広まりました。「Appleはテレビをリリースしますか？」

Apple TVの内部を見ると、今日ではApple Mac miniに似たミニコンピューターのようなもので、最も安価で、それに応じて最も遅いプロセッサー= Intel Pentium 4M 1Ghz、最小RAM = 256Mbで十分ですシンプルなグラフィックシステム= NVIDIA GeForce Go 7300 [64 Mb VRAM]（Apple TVについての詳細は、 「Apple TV：または」という名前を変更する別の理由をご覧ください）。

オペレーティングシステムとして、Apple TVは、ユーザーがシステムをセットトップボックスとしてのみ使用できる特別なインターフェイスを備えた、大幅に切り捨てられた修正済みのApple MacOS X 10.4 Tigerを使用します。

Apple TVにはDVDプレーヤーがないため、Apple iTunesプログラムを使用して、またはApple iTunes Storeから直接、自宅のコンピューターからすべての映画をダウンロードできます。 まあ、このコンソールの制限の結果として、Appleが繰り返し述べているように、非常に低い需要があります。 さらに、ユーザーがその機能を拡張しようとする絶え間ない試み- 「Apple TV セットトップボックス は既にクラックされています」および「SkypeはApple TVセットトップボックスにインストールされました！」を参照してください。

当然のことながら、このデバイスの需要を増やしたいというAppleの望みは、Apple TVの機能を拡張する1つの方法しかありません。



オプション1：最も簡単で手頃な価格の拡張の1つは、Apple TVセットトップボックスにテレビ視聴機能を追加することです。

これを行うには、アンテナ付きのアナログテレビレシーバーを組み込む必要はありませんが、夏の半ばに米国のすべてのテレビ局がデジタル形式のテレビ番組の放送に完全に切り替わるため、デジタルテレビチャンネルを受信できるマイクロ回路をいくつか追加するだけです記事「世界の終わりに近い：米国は2月にデジタル放送に切り替える準備ができています」と別の記事： 「2月に米国の放送局の40％がデジタルに切り替えます。」

また、多くのアメリカ人がデジタル放送を視聴するための特別なテレビセットトップボックスを購入しようとしているため、Appleは有利な立場にあります。



オプション2：原則として、映画を視聴する機能に対応するために、Apple TVではIntelのIBM PCアーキテクチャを使用する必要はありませんが、Apple iPod touchプレーヤーで使用されるハードウェアと同じもので十分です。

AppleのWebサイトのiPod + iTunesページとApple Storeの電子ストアでは、Apple iPod touchプレーヤーとApple TVセットトップボックスが常に隣り合っているのが不思議です:)

AppleがApple TVセットトップボックスでARMプロセッサーを使用するべきではなく、Appleが将来開発するマルチコアARMプロセッサーさえ使用できる理由を想像してください。記事「新しいiPhoneはクアッドコアプロセッサーを取得しますか？」

将来のApple TVでは、イマジネーションテクノロジーの開発に基づいたPowerVR SGX543マルチコアGPUが使用される可能性があります。記事「イマジネーションテクノロジーズがAppleのPowerVRグラフィックエンジンのライセンスを確認する」を参照してください。

さて、このすべての鉄の詰め物のコントロールとして、Apple TV用の特別なインターフェースを備えたわずかに変更されたApple OS X iPhoneオペレーティングシステムを使用することは残っています。



新しいアーキテクチャへの移行の結果、Apple TVプレフィックスは、映画を視聴するためのすべての最新機能に加えて、Apple App Storeからほぼ2万のアプリケーションすべてを起動できるようになりました（「App Storeには2万のアプリケーションがあります！」ほとんどはゲームです！

さらに、Apple TVセットトップボックスの機能を拡張する可能性は無限にあり、Apple OS X iPhone用に開発されApp Storeで販売されているプログラムにより、新しいビデオコーデックとして追加したり、Skypeセットトップボックスにインストールしたりすることが可能です。便利なプログラム！



いくつかの問題があります。

=これらの2万件のアプリケーションはすべて、Apple iPhone電話およびApple iPod touchプレーヤーの画面解像度用に作成されているため、これは画面解像度です。 しかし、App Storeのプログラム開発者が使用するApple iPhone SDKの最新の開発ツールを使用すると、複数の画面解像度オプションを使用して異なる画面でプログラムを実行でき、Apple iPhone 3Gの画面に固定されていないと思いますApple iPhoneの将来のバージョンでは、常に同じ画面解像度が使用されます）。

=およびこのセットトップボックスのリモートコントロール：もちろん、Apple App Storeのゲームを使用するには、より高価なリモートコントロールをコンソールに接続する必要があります。これには、Apple iPhoneインターフェイスと同様のタッチインターフェイスが必要ですが、同時にリモートに中央処理装置はありませんApple iPhoneのその他の内部、ただしインターフェイスチップと最も可能性の高いモーションセンサーのみ:)
