Yandexは、Microsoftに同意して、株式の相場を引き上げました





Yandexは、Microsoft EdgeブラウザーおよびInternet Explorer for Windows 10のデフォルトとしての検索の使用についてMicrosoftに同意することができました。この情報がWebに掲載された後、Yandexの引用は増加し始めました。 Interfaxによると、月曜日の終値に対して4.67％でした。 今Yandexの1株のために、彼らは12.76ドルを与えます。



Microsoft Corporationは、ロシアの会社との合意により、YandexをWindows 10オペレーティングシステムを実行するデバイスのデフォルトの検索エンジンにします。国内検索サービスは、ロシア、ベラルーシ、カザフスタン、ウクライナ、トルコのWindows 10ユーザーのデフォルト検索になります。 さらに、Yandexはこれらの同じユーザーのスタートページになります。



現在、ロシアのユーザーの約64％がWindowsをさまざまなデバイスのオペレーティングシステムとして使用しています。 Windows 10のシェアは約5％です。 確かに、Windowsの最も一般的なバージョンはWindows 7です。
