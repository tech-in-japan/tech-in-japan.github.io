WordPressのフォトギャラリーまたはNextGenを少しいじくり回す

NextGenプラグインについて多くのことが書かれています。 しかし、私の経験から、彼らが書いていないこと（少なくとも私は見なかった）と少し共有したい。



waterpressで忙しい人は、このプラグインを知っていると思うので、主な機能をペイントしたくありません。 そして、NextGenギャラリーで写真を見るためのフラッシュプレーヤーの種類をペイントしたいと思います。



運転した



プラグイン自体をインストールした後、組み込みのスライドショーを使用して、PicLensプレーヤーを使用して表示できます。





スライドショー



正しく動作させるには、 Imagerotatorも必要です 。



imagerotator.swfファイルをアーカイブからwp-content / uploads /にアップロードしてから、yourdomain.com / wp-admin / admin.php？page = nggallery-optionsページに移動し、[スライドショー]タブと[ギャラリー]タブで現在のファイルへのパスを指定しますスライドショーをオンにしてカスタマイズします。



何が得られますか？ そしてここに何があります：







原則として、このようなプレーヤーは正常ですが、コントロールは小さく、サムネイルはありません。 そして、残りは正常です-私はそれが好きだった。



ピクレンズ



これはもちろん、大文字の作品です。 :) yourdomain.com/wp-admin/admin.php?page=nggallery-optionsページの[全般オプション]タブで、Piclensサポートを有効にします。



これは次のようなものです。







本当に気に入りました。 すべてが良いです-プレイヤーは非常によくできていて、サムネイル、コントロールは便利です、フルスクリーン表示があります。 しかし、設定はまったくありません（まあ、ロシアのサイト、イギリスからのインターフェースを備えたそのようなモジュールは見た目が良くありません）、そしてこの3D表示は干渉しています。 それは何ですか-私は本当に理解していませんでしたが、彼のために何かをダウンロードする必要があることがわかったとき、私はそれを必要としないことに気付きました。 とにかく、個人的にはこれは過剰です。 しかし、それを削除する方法はありません-設定はありません。 しかし、そうであっても、印象は非常にポジティブです。



これらのプレーヤーは標準プラグインを提供します。 しかし、もう1つのことが可能です。



ギャラリービュー



NextGen Galleryviewモジュールをインストールするとき（ こちらからダウンロード）、別のプレーヤーを入手できます。 これはフラッシュプレーヤーではなく、このカヌーはJSで作成されます。 表示するには、ダウンロードしてインストールしたプラグインを有効にし、ページでタグを指定する必要があります：[nggallery id = 1 template = galleryview images = 0]



取得するものは次のとおりです。







画像は、曲がっていることを示しています。 事実は、CSSがドラフトされていないか、何か他のものであるということです。 しかし、彼らはすぐにこれについて警告します-それはトピック自体のCSSに敏感です。 その結果、非標準のテーマが存在する場合（そして誰がそれを標準にしているのか？）、プラグインはファイルで残酷に処理される必要があります。 もう1つのマイナス点は、ミニチュアミニチュアです。 要するに、プラグインは印象的ではありませんでした。



要するに、基本的なキットに含まれていない他のプレーヤーの小さな概要です。



NextGen Smooth Player



標準リポジトリには、NextGEN Smooth Galleryプラグインがあります。 入れて、アクティブにします。 yourdomain.com/wp-admin/admin.php?page=nextgen-smooth-galleryページでは、すべてがカスタマイズ可能です。 次に、ページのギャラリーを表示するには、[smooth = id：1]タグを挿入する必要があります。



結果は次のとおりです。







個人的に、私は彼が一番好きで、彼を使います。 構成可能で、便利で、高品質で作成できます。 唯一の欠点は、画像をそのサイズに適合させず、単に切り取るだけです。 したがって、ページに640x480のサイズのプレーヤーがある場合、同じサイズのサーバーに写真をアップロードします。



その後、さらに4人のプレイヤーが、 広告の後に続き、一緒にいて 、全員同じようにセットアップされます。



NextGEN FlashViewerプラグインを標準リポジトリからインストールし、アクティブにします。 yourdomain.com/wp-admin/admin.php?page=nggallery-flashviewerページで、プレイヤー自身をダウンロードするよう求められます（swfファイルが必要です）。 言うところのファイルをダウンロードしてアップロードすると、すべてが同じページで設定されます。

かなり単純なプレーヤーlistalka。 管理はシンプルで、飾り気はありません。 しかし、彼は特に感銘を受けたわけではなく、あまりにも単純でした。 誰かがこれを十分に持っているかもしれませんが。
