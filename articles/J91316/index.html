エカテリンブルグ：Visual Studioの食べ方

Visual Studio 2010の発売の波がロシア中を席巻し、最後のイベントはエカテリンブルクでのスタジオの発売でした。







イベントがカットの下でどのように開催されたかについて読んでください（注意、多くの写真）。





Visual Studio 2010は、エカテリンブルクのオネーギンホテルで開始されました。 オネーギンは素晴らしいホテルで、イベントの組織は最初から満足しています-居心地の良い雰囲気と登録テーブルでのフレンドリーな女の子。







その間、おなじみの人々がホールに現れ始めました（セルゲイ・ズベズディン、ドミトリー・アンドレエフ、ヴィタリー・ディルムカメトフ、マラト・バキロフ）：







人々はゆっくりと到着し、レポートルームに移動しました。レポートルームには、＃vs2010launchというタグが付いたユーザーツイートから同じテトリスが表示されました。 より多くの人がいました。







最初は全体報告書で、この日に検討される主なトピックをすべて網羅していました。 本会議は、マイクロソフトのゲストであるブライアンハリーによって開かれました。 彼とMicrosoftの同僚は、Visual Studio 2010の最新の革新、新しい開発ツール、コラボレーションへの新しいアプローチ、テスターとテスト部門向けの新しいソフトウェアツールについて話しました。







また、ツールにVisual Studio Team Foundation SeverとTest Managerを正常に実装したSitronicsのビデオも見せられました。 会社の従業員は、新しいツールが問題の解決にどのように役立つかを語った。







全体報告の後、全員が夕食に招待されました。 私は言わなければならない、オネーギンのレストランは失望しなかった、すべてが高レベルにあった。







昼食後、報告は続きました。 サンクトペテルブルクやモスクワとは異なり、エカテリンブルクではレポートのストリームは1つだけでしたが、非常に興味深いものでした。 最初に、ブライアンハリーは別のレポートを読み、Team Foundation Serverをゼロから操作することについて、買収、移行、インストールについて話しました。 BasicモードのTFSの新しいバージョンは、非常にシンプルで気取らないです。 2番目の写真では、ブライアンはTFSが回転しているネットブックを示しており、Visual Studio 2010が毎日の仕事のためにインストールされています。







第二に、テリー・リパーが私たちを訪問するために立ち寄った！ Leaperは、主要なVisual C ++アーキテクトです。 彼は、Windows 7の新機能をサポートするためのVisual Studio 2010のMFCクラスの革新について語りました。Terryが示したように、プログラムに新しいWin7機能のサポートを実装するには最小限のコードが必要です。







米国のゲストからの報告の後、ロシアの同僚は次のように語っています。DmitryAndreevがVisual Studio 2010を使用したプロジェクト管理について話し、Gaidar MagdanurovがWeb開発について元気よく話し、Marat Bakirovが新しいツールを使用して、 Visual Studio 2010およびTFS 2010は、開発者とテスター間のやり取りのタスクを解決します。 残念ながら、これらのレポートからの写真はほとんどありません。







報告の合間に、人々はブライアン・ハリーとテリー・リッパーを捕まえて写真を撮ったり、サインを書いたり、さまざまなトピックについて話し合ったりしました。 そして、すべてがVisual Studio 2010ロゴで飾られた巨大なケーキを食べてしまいました。ケーキが無傷のときは触りたくありませんでしたが、空腹のプログラマーはケーキにぶつかり、すぐに食べました:-)。









一般的に、イベントは大成功でした。 もちろん、いくつかの奇妙な点やオーバーレイがありましたが、それらはすべて笑顔と一般からの理解を呼び起こしました。



そのようなイベントでいつものように、最も重要なのはレポートではなく、多くの人がかなり前から知ることができた情報ではなく、コミュニケーションと興味深い人々でした。 ここに、そのようなイベントで、新しい知人が現れ、重要な事柄が議論され、問題が解決されます。



質の高い興味深いイベントを提供してくれたマイクロソフトに感謝します！ エカテリンブルクは、このような会議やゲストのためにいつも幸せです、再び来てください。
