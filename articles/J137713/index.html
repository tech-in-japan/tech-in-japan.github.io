プライベートおよびパブリッククラウドに関するロシア語の5つのオンラインコース

先週、MSDNで新しい資料について話しましたが 、今日はMVAの新しいコースでクラウドについて説明します（特にハブに関するこの投稿に触発されたシステム管理者からのトピックへの関心に照らして）。 それとは別に、コースはロシア語であり、完全に無料であることに注意してください（サイトに登録するだけで表示できます）。



2つのプライベートクラウドコース：

- プライベートクラウド。 入門コース

-System Center Virtual Machine Manager 2012



そして、公開の3つのコース：

-Windows Azureの紹介

-SQL Azureの概要

-Windows Azureセキュリティの概要



猫の下でコースの詳細：



プライベートクラウド。 入門コース

このコースは、最初にクラウドの概念を理解することを目的としています。 このコースでは、プライベートクラウドを構築するイデオロギーについて説明します。 IaaS、PaaS、SaaSなどの基本的なサービスの種類について説明します。 プライベートクラウドのアーキテクチャと制御に関する開示情報。 クラウドへのアプリケーションの移行へのパスが表示されます。 クラウド内の負荷の種類と、弾力性とスケーリングの違いについて説明します。 このコースは、プライベートクラウドの概要、プライベートクラウドの計画、プライベートクラウドの管理の3つのモジュールで構成されています。



System Center Virtual Machine Manager 2012

少し前まで、System Center 2012パッケージ全体がRelease Candidateに更新されました。 このコースでは、System Center Virtual Machine Manager 2012を使用してプライベートクラウドを管理するイデオロギーについて説明します。 プライベートクラウドを構築する基本的なコンポーネントについて説明します。 サービス（クラウド内の多層アプリケーション）の集中管理が考慮されます。 System Center製品に基づくプライベートクラウド試用プログラムが導入されました。 モジュール：SCVMM 2012の概要、SCVMM 2012を使用してプライベートクラウドを構成する、SCVMM 2012のモデリングとサービスのメンテナンス



Windows Azureの紹介

このコースでは、Windows Azureでの作業を成功させるために必要な基本的な側面について説明します。 モジュール：Windows Azureの概要、Azureアプリケーションの公開、Windows Azureの管理



SQL Azureの概要

このコースでは、SQL Azureの主な機能について説明します。コースの一部として、Azure SQLデータベースの作成と作成されたデータベース（SQL Azureポータル、SSMS、VSサーバーエクスプローラー）を管理するさまざまな方法の段階的なデモンストレーションを取得します。 モジュール：Azure SQLデータベースの作成と構成、SQL Azureセキュリティ、ローカルデータの同期と移行、ビジネスインテリジェンス



Windows Azureセキュリティの概要

このコースでは、物理、ネットワーク、ホスト、アプリケーション、およびデータレベルでのWindows Azureセキュリティメカニズムについて説明します。 さらに、Windows Azureの認証および承認メカニズムの基本的な知識を習得します。 コースモジュール：Windows Azureセキュリティの概要、認証とアクセス制御、データセキュリティと暗号化、ネットワークセキュリティ
