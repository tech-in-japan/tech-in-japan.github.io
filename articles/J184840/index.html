Windows 8.1プレビュー





Windows 8用の更新プログラムの試用版がダウンロードできるようになりました。



インストールするには、アップデートをダウンロードし 、コンピューターを再起動する必要があります。 ロードすると、ストアが開き、システムを更新するために提供されます。



制限があります：システム言語が英語の場合にのみ機能します。

また、 ブログでは、言語パックがインストールされているRTバージョンにアップデートをしようとしないことをお勧めしています。 それは困難です。 彼らはすぐにそれを修正することを約束します。



更新プログラムのリリースバージョンがリリースされたときに完全に再インストールする必要性に関する情報はありません（Windows自体のテストビルドと同様）。ブログには山ほどの質問があります。



UPD：コメントには、ロシア語版のリンクが記載されています。
