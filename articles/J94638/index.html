WinXPの下からリモートでHyper-Vを管理する

ご存じのように、Winows Server 2008の一部であるHyper-Vマネージャースナップインを使用してHyper-Vハイパーバイザーを管理できます。ワークステーションにRemote Server Administration Tools（RSAT）と呼ばれるユーティリティのパッケージをインストールすることにより、リモートで管理することもできます。

しかし、1つの小さな障害があります：残念ながら、Windows 7は地球上を正常に歩き回っていますが、MS Windowsの以前のバージョンはすべて、歴史のゴミ箱に十分な休息が必要であることを証明していますが、多くの場合、たとえば慣性によってWindows XPを使用しています。 残念ながら、RSATはWindows VistaまたはWindows 7にのみインストールできます。

アップグレードしたくない、またはアップグレードできない人にどうすればいいですか？









Andrei Beshkovという名前の同志の1人は、サードパーティ企業であるUtharamの「Hyper-V管理コンソール」と呼ばれる優れたユーティリティについてブログに書いています。 その助けを借りて、標準のHyper-Vマネージャーコンソールでできることと同じことができます。

  • 複数のHyper-Vサーバーを監視する
  • 仮想マシンの状態を追跡および変更する
  • 仮想マシンとスナップショットの作成、名前変更、削除、エクスポート、インポート
  • RDPを介して仮想マシンに接続する
  • ユーザー権利の輪郭を描き、アクションを監査する




このユーティリティは、製造元の公式Webサイトからダウンロードできます。 ユーティリティのインストールと作業は非常に簡単なので、詳細には説明しませんが、Andreyから正直に借りたスクリーンショットのみを提供します。



結論として、私は大さじ2杯のタールを蜂蜜の樽に追加します。まず、ユーティリティは支払われます。第二に、このソリューションはサポートされていないため、Microsoftサポートサービスはこのユーティリティに関連する質問がある場合は、フォレストに直接送信します。開発者はいくつかのサポートソフトウェアを約束しますが、その品質を判断することはできません-私はそれを使用しなかったためです。一般的に、Windows 7にアップグレードすることをお勧めします;）
