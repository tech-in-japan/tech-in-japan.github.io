ThinkServerが欲しい：あなたのアイデアは私たちの鉄です





すべての読者へのご挨拶！



2014年のすべての可能な結果が正確に要約され、対応する結論が出され、計画が調整され、開始された年度のタスクの概要が示されたので、小さなキャンペーンを開催することにしました。 または実験。 またはテスト。 あなたに合った名前を決定します。



ご存知のように、今日、当社は当社のウェブサイトで問題なく慣れ親しむことができる幅広い機器を市場に提供しています。また、興味深い有望なプロジェクトを支援し、すべてがうまくいくと心から信じています。特定の努力。 しかし、良いアイデアや人生の始まりを実現するためには、十分ではありません。たとえば、優れたサーバーです。

あなたの壮大なプロジェクトは世界を征服する準備ができており、必要なのは初期段階（または初期段階ではない）でほんの少しの助けだけです。 それとも、ビジネスを構築するためにどの機器を選択する必要があり、自分ですべてを確認したいので、他の人のレビューやストーリーを信頼することに慣れていませんか？



サーバーを提供する準備ができました！



厚い契約の9ページ目の下部に小さなテキストがなくても、隠された手数料や手数料は一切発生しません。 サーバーは、払い戻しの対象として、数か月（2〜4か月）使用するために提供されます。



あなたから必要なものは次のとおりです。

  • あなた自身についての短い物語：あなたが誰であるか、あなたが何をするか。
  • 自分の目標は何ですか？
  • 現在、あなたの「プロジェクト」はどの段階にありますか。
  • 「空気のような」サーバーを必要とする目的と、それを使用する目的。
  • 1つの物理サーバーを提供することを理解しています。




このテキストを読むとすぐに、今すぐアプリケーションを送信できます。 mrakaev@lenovo.comまでご連絡ください。件名に「 ThinkServerが欲しい 」と明記してください。

すべての手紙を考慮し、読むことをお約束します。 しかし、残念ながら、絶対に誰もが機器を受け取ることができるとは約束できません。 また、片手で1台のサーバーを提供することに注意してください。

サーバーへのアクセスはどのように許可されますか？

物理サーバー（「手元」）が提供されます。



どのようなサーバーを提供していますか？

現在、2つの構成を使用できます。





ThinkServer RD540 （2 x Xeon E5-2650v2、64Gb DDR3、2 x 300Gb 15K SAS 6Gbps、4 x 600Gb 10K 6Gbps、LSI 9270CV-8iコントローラー、2 x 800W PSUホットスワップ）





ThinkServer RD640 （2 x Xeon E5-2650v2、128Gb DDR3、2 x 100Gb、2 x 300Gb 15K SAS 6Gbps、4 x 600Gb 10K 6Gbps、2 x 500Gb 7.2K 6Gbps、LSI 9270CV-8iコントローラー、2 x 800W PSUホットスワップ）



機器について話し合っていますか？

はい、通信段階で。 私たちはあなたに最適なオプションを見つけようとします。



誰が参加できますか？

ロシア連邦の市民は誰でも参加（申請を提出）できますが、「紙のアイデア」の段階を離れ、展開と実装の準備ができている（または既にある）小規模プロジェクトを優先します。 私たちは、デバッグプロジェクトを持っている人、プロジェクトを「現場」で完全にチェックするために稼働中のサーバーが必要な人、完全に新しいサーバーを購入するかレンタルサービスを使用するかを決めていない人を探しています。



サーバーはどのような条件で提供されますか？

一時使用契約。



サーバーが気に入ったら、たとえば購入して自分用に保管できますか？

テストのためにあなたに渡されるモデルはそうではありません。 ただし、パートナーからまったく同じものを購入して、プロジェクトを透過的に移行できます。 これをお手伝いします。



更新しますか？

機器の伝送の拡張について、各参加者と個別に議論する準備ができています。



サポートはどうですか？

当社のサーバーでの作業中、当社の機器の本格的な購入者とまったく同じ技術および保証サポートが提供されます。 そのため、コインのこちら側を評価する機会があります。
