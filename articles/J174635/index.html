柔軟性の練習-Lenovo IdeaPad Yoga 13

もちろん、かわいい女の子の助けを借りて読者を誘惑する良い伝統はとても良いです。 しかし、私は財団から離れて、Lenovo IdeaPad Yogaが子猫からラップトップを変えるという話を始めることにしました。 記事やノートパソコンが気に入らない場合は、少なくとも子猫を見てください。みんな大好きです。 だから、カットの下で-IdeaPad Yogaとは何ですか、何が面白いのか、Windows 8にチャンスがあるのか​​という質問への答えです。



Windows 8のリリースでは、G8への攻撃に熱心な多くの批評家が気付かなかった別のイベントが発生しました。実際、Windows 8の競技場である新しいクラスのデバイスの出現です。回転またはスライド式スクリーン。 もちろん、これはWindows 8のリリース前でしたが、タッチスクリーンで7-kiインターフェイス（または、神が許してくれました、XP）を使用する価値はありました。 私自身は、ほぼ完璧なデバイスを次のように定義しました：x86 Windowsベースのデバイス（仕事の詳細-.netプログラマーによって決定されます）、薄く、軽く、バッテリー寿命が長く、キーボード、タッチパッド、および画面が良い。 そして、奇妙な欲求-ネットサーフィン、ニュースの閲覧、テレビ番組の視聴などのために、ノートパソコンを「主力」モードから「コンテンツを消費する」モードに切り替える機能から。 iPadや彼のようなタブレットはすべてクラスとしてマークされました。 私は、ブタに鳥を投げるための2つのデバイス（ラップトップ、意味のある何かを行うためのタブレット）を持ちたくありませんでした。 クラシックラップトップは「コンテンツを消費する」には適合しませんでした。 ソファの上に堂々と横たわっているので、私は小さな初歩的なタッチパッドを感じたくなく、役に立たないキーボードをどこに取り付けるかを考えたくなかった。 Windows 8の最初の発表により、デバイスは雨の後にキノコのように見え始めました。これは私の要件により適したものであり、そのうちの1つはすぐに変革への最も型破りなアプローチに注目を集めました。 Lenovo IdeaPad Yogaについてです。 長い間、私は販売中のデバイスの外観を楽しみにしていました、そして今、ついに、私は誇りに思う所有者になりました。 使用の経験はまだ素晴らしいものではありません-2か月ですが、いくつかの結論がすでに得られています。



したがって、物語の主人公は、ウルトラブックトランスフォーマーLenovo IdeaPad Yoga 13です。1.5kg、ケースの最も厚い部分の厚さ16 mm、Intel Core i5（i3 / i7に変更があります）、4Gb RAM（ボードにはんだ付けされていませんが、スロットの形） 、128Gb SSD。 最もおいしい部分の1つはスクリーンです。 13インチ、解像度1600 x 900ピクセルのIPSディスプレイ。 キーボードは島で、キーは短いストロークで快適です。 優れたジェスチャサポートを備えた印象的なタッチパッド（1秒ごとに誤検出のない良い手段）。 順番に行きましょう。

オプションと外観



鈍い茶色の箱に入っています...ラップトップ自体のあるスタイリッシュな黒い箱が同封されています。 パッケージには何も含まれていません-ラップトップと充電器。 ちなみに、これはAsusovsky（およびもちろんMBA）ウルトラブックのトレンドを追うことで実現できます-プラグが電源に接続されていて、1本のワイヤしか出ていない場合。 ブロック自体は薄くて軽いです、少なくともそのおかげです！

ラップトップはファッショナブルで実用的なアルミニウムまたはマグネシウム合金ではなく、鈍い灰色のプラスチックで作られており、キーボードの周りにソフトタッチコーティングが施されているにもかかわらず、見栄えが良いです。 個人的には、MBAを繰り返し、デバイスの前端と側端を「狭める」という愚かな傾向に従わないのが本当に好きです。 私はそれが好きです。手に持ってうれしいです。クールなHTML5プロジェクトを書くためにスターバックスに行くのは恥ずべきことではありません。

別の「トリック」がキーボードとタッチパッドの周りのフロントパネルを覆っています。 「肌の下」の質感と触覚を持つ柔らかいプラスチックでできています。 少し珍しいように見えますが、実際には、特に印刷中にブラシが置かれている領域では、触ると非常に心地よいことがわかります。

外観について言えば、どのポートが外部にあるかに注意する必要があります-これらは2つのUSB（2.0および3.0）、HDMI（ミニではなくフルポート）、メモリカード用のスロット（MMC / SD）です。 また、ラップトップの端には、ボリュームコントロール、画面反転ロックボタン、Novoボタン-BIOSエントリまたはリカバリモードでの起動のボタンがあります。

キーボードとタッチパッド



ホーム、エンド、PgUp、PgDown、CtrlおよびFnをそれぞれの場所（左側のCtrl）に別々にレンダリングした、従来のIdea / Think-Padレイアウトのキーボード、カーソルキーは大きい。 レイアウトでは、おそらく小さな右Shiftでのみ障害を見つけることができます。 キーボードには隠れていますが、おそらくこのデバイスの軟膏で最も目立つハエです。 一般的なモノリシックで非常に高密度のアセンブリにもかかわらず、キーボード上の1か所-0キーの下で-「集中的な」タイピング中にわずかに曲がります。 これは、もちろん、その使用を妨げず、多くの不快感を引き起こしません-しかし、彼らが言うように、スプーンは見つかりましたが、沈殿物は残りました。 これは、最低価格のカテゴリのラップトップでは許されますが、ヨガをプレイする場合-競合他社は深刻なので、アセンブリでのそのような愚かな誤算は不可解です。 さて、話した-それから良いことだけがあります！



Lenovoのエンジニアにうなずきを承認させたキーボードの1つは、一連のファンクションキーです。 どの-最後に！ -「逆転」しました。 つまり Fnなしでは、マルチメディア機能（ボリューム、画面のバックライトなど）が機能し、Fn、F1、F2などでは既に押されています。 プログラマー、怒らないでください-私はあなたの憎しみを理解していますが、IDEの外ではこれらのF1はすべて本当に必要ではないことを認めなければなりません。 もちろん、Alt + F4の場合はF4、F5の場合はF5を除きます。 ここで、Lenovoはすべてを正しく行いました。通常どおり機能するのはこれら2つのキーで、F4は現在のアプリケーションを閉じ、F5は更新要求を送信します。 この操作モードで寿命がない人のために-BIOSでは、機能シリーズのデフォルトの動作をクラシックに切り替えることができます。

はい、キーボードのバックライトはありません-また、「さて、あなたは何ですか...」という質問を提起します

タッチパッドはただの誇りです。 最後に、ラップトップメーカーは、小さなタッチパッドの奇妙な複合体を取り除き始めました。 今では本当に快適に使用できます-サイズは適切で、マルチタッチジェスチャは（誤検出なしで）正常に機能し、表面は素晴らしく、ボタンは表面全体です。 苦情はまったくありません。

スクリーン



とりわけ賞賛！ 1600x900 IPSマトリックスは、その鮮明さと画質に印象的です。 常に画面を使用するときは忘れますが、別のラップトップからヨガに戻ると、すべてが明確になります。 さらに、タッチスクリーンは最大10の同時クリックを認識します。 センサーの品質に関する苦情はありません-すべてのジェスチャーは明確に機能し、誤検知や苦情はありません。 ラップトップモードでも作業習慣がどのように変化するかを観察するのは面白いことです。ダイアログボックスが表示されたら、マウスカーソルを動かしてクリックするよりも、単に手を挙げて[OK]画面をクリックする方が便利です。



自治



言うのは難しい 経験はまだ大きくありませんが、クラスで最悪ではありません。 最長8時間のバッテリー寿命が要求されます。実際、快適な画面の明るさ、Wi-Fiのオン、およびVisual Studioでの重いプロジェクトのコンパイルを含まない「オフィス」操作モードでは5〜6時間のようです。

性能



良いもの。 「それだけです」（C）。 最新のi3でさえ私の問題のほとんどを完全に解決することを考慮に入れると、i5には余裕があります。 また、システムがSSDで動作することを考慮すると、すべてが正常になります。 完全な無意味さの観点から合成テストは実施しませんでしたが、おそらくインターネットでモバイルi5のテストを見つけることができます。 おそらくメモリを8 GBにアップグレードしたいのですが、これは大きな問題ではありません。 メモリはボードにはんだ付けされていませんが、4GBストリップを挿入するだけで、さらに8GBに簡単に交換できます。

タブレットモード



最後に、最も興味深いのは、変換メカニズムです。 レノボは非常に興味深い道を歩みました-ただ画面を曲げるだけでいいのなら... 確かに、このようなターンでは、キーボードとタッチパッドは結果の「タブレット」の下部にありますが、事前にオフになっているため、偶発的なクリックはありません。 ただし、電源をオフにした場合でも、スペースを占有し、ラップトップの重量を維持することを妨げることを考慮する価値があります。 一般に、1.5 kgの質量では、誰もが重量で「タブレット」を使用することはほとんどありません。 しかし、キーボードは本体に埋め込まれており、底面は柔らかいタッチのプラスチックで覆われているため、結果として得られるデバイスは、膝、テーブル、ソファ、飛行機や電車のテーブルなどに非常によく収まります。 しかし、私たち全員が理解しているように、奇跡は起こりません-ヨガからのすべての欲求を持つiPadは機能しません。 したがって、薄くて快適なタブレットが必要な場合は、Windows 8（Acer W510 / W710など）でタブレットを見るのが理にかなっています。 しかし、ネットワークからコンテンツを便利に消費するための画面を短時間取得する機会であり、必要に応じてすぐに「作業」モードに戻るという、まさに望みどおりの結果が得られました。



一般的に、ヨガはさらに2つの操作モードをサポートします：スタンド-ラップトップがキーボードで横になっているが、画面が押されておらず、オーニングが斜めに持ち上げられているとき-ラップトップがキーボードと画面上にあるとき。 原則として、これら2つのオプションは映画鑑賞モードでの作業に適していますが、積極的に使用しているとは言えません。



熱と騒音



ヒートシンクとエアダクトは後縁に作られているため、ラップトップは柔らかい表面に安全に設置できます。 オフィスモードでは、ヨガは熱くならず、体温は膝の上に保つのに非常に快適です。 空気は涼しいですが、それにもかかわらず、ファンは常に作動しています。 確かに、非常に静かに動作するため、あまり干渉しません。 静かな部屋で作業すると、彼の声が聞こえますが、私が言ったように、不快感は生じません。 LenovoはBIOSのパッチを適用すると脅して、ファンが常にオンにならないようにしましたが、このような薄いケースでは達成が難しいと思います。

まとめると



では、Lenovoはどのデバイスを手に入れましたか？ 基本的に、仕事でコンピューターを使用する人向けです。 エンターテインメントのデバイスとしてのみ、ヨガはあまりバランスが取れていません-体重とオフラインモードはタブレットではなく、キーボードはタブレットモードでの生活を妨げ、価格はiPadではありません。 しかし、優れた画面、まともなバッテリー寿命、短時間で画面を「回転」させてa-laタブレットを入手するクールなトリックを扱うウルトラブックが必要な場合、デバイスはほぼ完璧であることがわかります。 また、Windows 8についても言及したいと思います。ヨガでは、栄光のすべてが展開されます。 現代のUI、タイル、ジェスチャーの必要性と有用性に関する質問はすべて消えます。すべてが「タッチエクスペリエンス」に調和してうまく収まるようになっています。 ヨガに「移動」した後、メール、ブラウザ、ツイッターなどの社会的なもの、テレビ番組、ニュースなど、現代のUIアプリケーションを使用し始めた時間のほぼ80％に気付きました。

そのため、「主力製品」とファッショナブルなクールなガジェットをすべて1つにしたい場合（これはオタクが求めるものです！）-Lenovo IdeaPad Yogaは素晴らしい候補です。
