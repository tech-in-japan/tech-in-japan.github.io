ラズベリーパイ財団の創設者は、「ラズベリー」のディスプレイを示しています





Raspberry Pi Foundationの創設者であるEben Uptonは、最近「ラズベリー」用の新しいハードウェアのデモを行いました。 これは、Raspberry Piのアクセサリと追加要素のエコシステムを補完するタッチスクリーンです。



「Raspberry Piを作っている間、ディスプレイはすぐに表示されると言っていましたが、ディスプレイはすぐに表示され、ディスプレイは実際に表示されました」とUpton氏は言います。 このディスプレイは、今年の終わりまでに、または来年の初めにリリースされる予定です。



Raspberry PiをベースにしたUptonによれば、以前よりもさらに多くのデバイスを作成できます。 最初に頭に浮かぶのはタブレットです。 確かに、ディスプレイの対角線はそれほど大きくありません-たった7インチですが、市場には現在、このような対角線を備えたタブレットPCが非常に多くあります。 タブレットは、Pi-Padと呼ばれることになっています。



ディスプレイはRaspberry Pi専用に作成されているため、追加のジェスチャーなしで非常に簡単に接続できます。



特に、UptonはRaspberry Pi Model A +（完全に更新されたミニPC）をリリースするという会社の意図を発表しました 。 彼らは、新しい構成について後で通知することを約束します。 Model B +モデルは、かなり前に導入され、多数のピンといくつかの他の改良が加えられたことを思い出してください。
