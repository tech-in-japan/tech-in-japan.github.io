Tucan Project Linuxの対応物-tucaneando.com/index-en.html



Tucan Managerは、rapidshareやmegauploadなどのホスティングサイトでのダウンロードとアップロードの自動管理用に設計された無料のオープンソースソフトウェアです。



主な機能：



*完全にPythonで書かれています。

*マルチプラットフォーム（GNU / Linux、FreeBSD、Microsoft Windows ...）。

*プラグインで簡単に拡張できます。

*軽量かつ高速。

*プレミアムアカウントのサポート。

*必要に応じてキャプチャ認識。

*交換可能なリンクの管理。



Ubuntuの場合はこちらをご覧ください： www.getdeb.net/app/Tucan
