iNT PROおよびiNETTRADER-iPhoneおよびiPadを使用した株式取引用アプリ

最近まで、ロシアのiPhoneユーザーは携帯電話で証券取引を行うことができませんでした。 ただし、この問題は解決されています。 証券会社のNETTRADER.ruは 、iPhoneおよびiPadを使用してMICEX証券取引所で株式を取引するアプリケーションを開発し、そのようなソフトウェアを発行する世界で最初の証券会社およびロシアで最初の証券会社になりました。 閲覧アプリケーションのリリースは2010年5月に行われ、取引アプリケーションは数か月後の2010年10月に行われました。



各アプリケーションについて順番に。



iNT PRO-取引ターミナル。 このプログラムでは次のことができます。


•ターミナルのデモ版または完全に機能するバージョンを使用します。

•証券のデータをリアルタイムで表示します（デモアカウントでは、実際のデータを表示できますが、遅延があります）。

•証券の売買のための取引注文を作成します。

•注文の履歴、注文のステータス、注文ごとに完了したトランザクションを表示します。

•現在のポートフォリオを表示します。

•Silver SurferやFast Surferなどのニュースを表示します。



このプログラムは、すでにNETTRADER.ruでアカウントを開設している人のみが利用できます。



iNT PROのスクリーンショット：







iNETTRADER-表示端末。 このプログラムでは次のことができます。


•関心のあるツールに関する情報を便利なグラフィカル形式で追跡します。

•カテゴリ別の見積もりを見る：ブルーチップ、電気通信、エネルギー、石油およびガス、貿易、冶金、銀行、化学、機械工学、開発。

•Silver Surferのニュースと貿易の推奨事項に従ってください-独自の開発NETTRADER.ruの独自の意思決定システム。

•現在の取引に関する情報（わずかな遅延を伴う）および取引履歴に関する情報を表示する（このためには、iPhoneを「横向き」にする必要があります）。



iNETTRADERのスクリーンショット：





このプログラムは、引用符を表示したいすべての人を対象としています。 会社との口座開設はオプションです。



両方のアプリケーションは無料です。 ここからダウンロードできます 。



NETTRADER.ru会社には、IEブラウザで動作する独自の取引端末NETTRADER.proもあります。端末NETTRADER.mobile.proはPDAおよび携帯電話用のバージョンです。 2011年3月、トレーディングターミナルの新しいバージョンがリリースされました。 その主な機能は、完全に新しいインターフェース、便利で直感的、そして任意のブラウザーで動作する機能です。
