Synapseは、Alt + F2の代替として最適



Synapseは非常に優れたランチャーであり、いくつかの非常に優れた機能を備えています。



そのため、プラグインの標準セットで使用可能な機能をリストします（そうです、そうです、あなたは正しいと聞きました）。



*アプリケーションを起動する

*バンシー-バンシーで音楽を再生する

*コマンドの実行（「sudo apt-get update」に注意）

* Devhelp-Devhelpを使用してドキュメントを検索

*辞書-単語の定義を見つける

*ディレクトリ検索-フォルダーを開く

* Gnomeセッション-ログアウト、シャットダウン、再起動

*ハイブリッド検索-類似ファイルの検索

* Rhytmbox-Rhythmboxで音楽を再生する

* UPower-コンピューターを安楽死または休止状態にする

* Zeitgeist-Zeitgeistを使用した検索



すべての味と色のそれらのためのサポートもあります。

ところで、デフォルトでは、Ctrl + Spaceで呼び出されます。



シナプスのインストール：

sudo add-apt-repository ppa：synapse-core / ppa

sudo apt-get update

sudo apt-get dist-upgrade＃Lucidにのみ必要

sudo apt-get install synapse



このランチャーはZeitgeistデータを使用するため、インストールすることをお勧めします。

sudo add-apt-repository ppa：zeitgeist / ppa

sudo apt-get update

sudo apt-get install zeitgeist-datasource-bzr zeitgeist-datasource-emacs zeitgeist-datasource-eog zeitgeist-datasource-geany zeitgeist-datasource-gedit zeitgeist-datasource-rhythmbox zeitgeist-datasource-totem zeitgeist-itsource-totem tem



さて、完全な失策のために、FTS Zeitgeist拡張機能を追加しましょう



マーベリックの場合：

sudo apt-get install zeitgeist-fts-extension



Lucidの場合：

sudo apt-get install zeitgeist-extension-fts



テストで頑張ってください!!!
