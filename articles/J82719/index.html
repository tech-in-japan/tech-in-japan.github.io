2月1日からの新しいビルドOpera 10.50（3218/8215/6204）

アルファ版以前のOpera 10.50の新しいビルドがリリースされました。 開発者のブログでは 、アセンブリを説明する記事はまだ確認されていません（ところで、Unix用の新しいアセンブリもあります）。



開発者のブログ投稿が表示されたら、その翻訳をここに追加します。

無料翻訳はオリジナルに近いです。



新しいビルド。



こんにちは

既にお気づきのように、私たちは大量のエラーを修正しているので、皆さんのために新しいビルドをアップロードしています。 でテスト！

ページスケールコントロールの変更（これまでは勝者のみでしたが、間もなく他のプラットフォームでも使用できるようになりました）:)



bug-wizardで見つかったエラーを必ずお知らせください。 バージョン10.10からのリグレッションを探します-これが私たちにとって最高の優先事項です。



顕著な変化

  • Facebookが再び機能する
  • 新しいページビューコントロールボタンがステータスバーに追加され、スライダーがスケールを制御するために追加されました（これまではWindowsのみ）
  • タブ管理（MDI）に取り組んだ
  • 個人用パネルのグリッターなど、テキストを読みやすくするためのスキンの修正（Windows 7 / Vista）
  • 安定性の向上
注意：このバージョンは開発中です。最新の変更が含まれていますが、さらに、クラッシュやデータ損失の状況などの重大な問題がある可能性があります。 実際、それはうまくいかないかもしれません。



ダウンロードする

  • Windows MSI / Windowsクラシック
  • Mac（Intel） OS 10.5以降
  • Unix


私は理想的な翻訳のふりをしません。 修正が受け入れられます。

  • 何か（画像\トレント\など）を保存する際のハングを修正しました。
  • トンボが修理された-私は幸せです！
  • 彼らはステータスラインをもう少し狭くしました-今ではより正確に見えます。
  • 新しいページビューコントロールボタンが本当に気に入っており、スケールのスライダーが長い間欠けていました。
  • の。 オペラ座は今では継ぎ目でばらばらになりません。 何が責任を負うべきか正確にはわかりませんが、これらは現在修正されたPrestoレンダリングのバグだったと思います。
  • クラッシュしない間（pah-pah-pah）。
