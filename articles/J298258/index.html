米国で年間13万ドルを超える給与をもたらす10のITスキル





労働市場と就職活動に特化した有名なリソースDice.comは、最近、2015年秋に米国で実施された調査結果を公開しました。 アンケートの結果、この業界で需要がある特定のスキルを持つ人々の平均給与のリストが作成されました。 しかし、まともな給料で良い仕事を得るためには、いくつかのスキルが十分でないかもしれないことを忘れないでください-あなたも経験が必要です。



10.人形-平均給与は年間131 121ドルです。

Puppetは、Puppet LabsのITオートメーションソフトウェアです。PuppetLabsは、DevOpsと呼ばれるテクノロジーの世界で大きなトレンドに参加している多くの若い企業の1つです。 これは、ソフトウェア開発の方法の1つであり、ソフトウェア開発者と情報技術の保守に携わる従業員のチームが、技術の迅速な展開プロセスを可能にする特別な技術を使用します。



9. MapReduce-平均給与は年間131.563ドルです。

MapReduceは、コンピュータークラスター内の大量のデータを分析するために使用される分散コンピューティングモデルです。



8.豚-年間平均給与は132.850ドル。

現在非常に人気のあるPigツールは、スクリプトを実行するためのプログラミング言語およびランタイム環境です。 このツールを使用すると、Hadoopから情報を取得し、質問への回答を見つけ、利用可能な情報を使用できます。



7.シェフ-年平均136.850ドルの給与。

Chefは、上記のDevOpsトレンドの一部であるIT自動化ソフトウェアです。 シェフはパペットのライバルです。



6. CloudStack-平均給与は年間138,095ドルです。

クラウドコンピューティングも別の傾向です。 ITの世界では、クラウドオペレーティングシステムの闘争があります。 それらのいくつかはオープンソースであり、その一例はCloudStackです。これは、Citrixが作成し、現在Apache Software Foundationが所有しているクラウドサービスインフラストラクチャを作成、管理、展開するためのソフトウェアです。



5. OpenStack-平均給与は年間138.579ドルです。

別の無料のオープンソースクラウドオペレーティングシステム。 IBM、HP、Red Hat、Ubuntuなど、多くの開発者がサポートしています。



4. PaaS-平均給与は年間140.894ドルです。

PaaSは、開発者がクラウドでアプリケーションを作成できるようにする「サービスとしてのプラットフォーム」です。 Microsoft Azureクラウドコンピューティングプラットフォームは、おそらく最も顕著なPaaSの例です。



3. Cloudera-平均給与は年間142.835ドルです。

Clouderaは、Hadoopの商用バージョンです。 Hadoopは、低コストのサーバーに大量のデータを保存するための無料のオープンソースプロジェクトですが、そのすべてのフリーソフトウェアにとって、このソフトウェアはそれほど使いやすいものではありません。 そのため、いくつかの企業がより良いバージョンのHadoopを作成し、Clouderaが最も人気のあるバージョンの1つになりました。



2.カサンドラ-平均給与は年間147.811ドルです。

Apache Cassandraは、無料の分散データベース管理システムです。 これは、さまざまなタイプとボリュームのデータを保存できるnoSQLシステムです。 AppleはCassandraを積極的に使用して、10ペタバイト以上の情報を保存しています。



1. HANA-平均給与は年間154,749ドルです。

OracleのライバルであるSAPが開発した高性能NewSQL HANAプラットフォーム。 HANAは、インメモリコンピューティングテクノロジーとして知られるストレージテクノロジーのもう1つのトレンドの発案です。
