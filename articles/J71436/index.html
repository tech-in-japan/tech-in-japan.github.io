幸せなハードドライブ所有者へのアドバイス

ハブラこんにちは ！ 今日は、現代のハードドライブの小さな便利な設定について説明します。これにより、動作中のノイズを大幅に減らすことができます。







AAMについて聞いたことがない人のために-猫へようこそ。



事は実際には新しいものからは程遠いものであり、多くの人はそれを認識しています。 しかし、さらに多くの人が彼女のことを聞いていませんでした。



だから、私は猫を尾で引っ張らない-ほとんどの現代のハードドライブには、 AAM （自動音響管理）のような機能があり、それは静寂とパフォーマンスの危機にあるバランスを可能にする。 物理的な意味は、読み取り/書き込み時に磁気ヘッドのブロックによって放出される音響ノイズのレベルを制御することです。 パラメーターを変更すると、目的のシリンダーが見つかるまでヘッドがディスクの半径に沿って移動するときに、トラックの検索時間に影響を与えることができます。



ほとんどの場合、プログラム（Windowsの設定を定期的に変更する方法はわかりません）には、0から255（場合によっては128から255）のスケールがあります。 0（または128）は無音（最小パフォーマンス）に対応し、それ以上のことはすべて、より速く、高く、強くなります！

一部のメーカーはこの機能をまったく搭載していませんが、他のメーカーは工場でのパフォーマンスを向上させるために最大値を設定しています。 ちなみに、0（128）に設定するとブレーキがドライブから外れるとは考えないでください。 ディスクの速度は一定であるため、線形読み取り速度は変化しません。 ポジショナーの速度が低下するため、目的のセクターへのアクセス時間を増やします。 AAMの最小値により、アクセス時間が平均30％増加します。 このような数値を恐れないでください。常に「高い」パフォーマンスが必要になることはほとんどありません。 しかし、ドライブからの大幅に低減されたノイズに加えて、消費電力とデバイスの温度、およびその寿命をわずかに獲得することが可能です。







Windows の AAMパラメーターは、 HDTune Proソフトウェア、 HDDLife Pro 、 Victoria 、 MHDD 、 Hitachi Feature Tool （Hitachiドライブだけでなく）など、さまざまな方法で変更できます。 私はHDTune Proを使用します（無料版ではありません）-極値のノイズの違いはプログラムで直接推定できるので、驚くでしょう。



IXBTで AAMの詳細を読むことができます。



この機能の代わりに、厚い防音壁、特別なクレードルおよびボックスを備えた重いケースを使用し、すべてのネジまたは特別なガスケットを介してディスクを固定し、システムユニットを耳から離すなどです。



これはニュースではない人のために-無関心であってはいけません それらに何を追加すべきか、何が余分なのか、何を変更すべきなのか？ どのデバイスレビューがより興味深いですか？ 建設的な批判と新鮮なアイデアはあなたのために私が良くなるのに役立ちます。
