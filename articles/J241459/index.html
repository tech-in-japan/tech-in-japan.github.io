stackoverflow.comおよびgithub.comのロシアのITスペシャリストの統計

私はすべての女性、紳士、そして上記の2つのカテゴリーに属すると分類する人々に挨拶します。



変化する運命の意志により、私は時々、あらゆる種類の組織のあらゆる種類のデータのあらゆる種類の分析を実行する必要があります。 そして、これらの分析の一部を皆さんと共有したいと思います。



質問はおおよそ次のとおりでした-ロシアのITスペシャリスト（もちろんロシアに常駐している）がグローバルITコミュニティにどの程度の存在感と影響を与えているのでしょうか？



以下は、ロシアのITスペシャリスト（一般的にITに関心のあるすべての人々を含む）が2つの最大の国際ITコミュニティ（stackoverflow.comとgithub.com）の分析を使用して、世界コミュニティに与える影響について説明する分析の一部です。



そして、分析自体は次のとおりです。



Stackoverflowプロジェクトの対象者は3 580 212登録ユーザー（2014）、GitHub-3 955 191登録ユーザー（2014）です。 Stackoverflowコミュニティでは、2013年と2014年、GitHub-2012年、2013年、2014年の統計が分析されました。 すべてのユーザーが居住国を指定しているわけではないため（統計は完全ではありません）（Stackoverflowの場合は約25％、GitHubの場合は約70％が居住国を示していません）、しかし、利用可能なデータは一般的な状況と比率を理解するのに十分です。



Stackoverflow



ロシアを居住国として指定したユーザー-11,319人。これはユーザーの総数の0.38％です。 ユーザーの約25％が居住国を示していないことを考慮すると、このリソースのロシア人の実際の数は15,000ユーザーの地域、またはユーザーの総数の約0.42％であると想定できます。 同時に、2014年のリソース上のロシアのIPアドレスを持つユニークな未登録訪問者の割合は、ユニークな未登録訪問者の総数の約1.6％（11位）でした（比較のために、オランダ-1.75％、ブラジル-2.3％、インド- 12.5％、中国-1.46％）。



登録ユーザーの総数によると、ロシアはスタックオーバーフローの上位10位にランクされています（2014年のデータ）。

Github



2012年7月の時点で、リソースに対して6,826,827のコミットが行われました。 ロシアを居住国として指定したユーザーは、コミットの総数の約1.8％を占めました（10位、比較：ブラジル-1.9％、フランス-2.7％、日本-3.8％）。



2013年時点で、居住地を指定したユーザー（全ユーザーの約26％）は約2850万件のコミットを行いました（コミットの総数65,000,000件の約44％）。 ロシアは、このコミット数の約3.5％を担当しています（8位、比較：ブラジル-3％、フランス-4.5％、日本-4％）。 同時に、ロシア連邦のインターネットユーザー総数の約3％がリソースに登録されています（9位、比較：ブラジル-4％、フランス-4.5％、日本-2.5％）。



2014年の状況は、居住地を示す約1050,000人のアクティブユーザー（合計3955191人のユーザー）のうち、ロシア人が1.5％（16319人のアクティブユーザー）を占め、最も影響力のあるユーザー（影響基準は、 10人のフォロワー）ロシア人のシェアは0.91％です（絶対数-78,470のうち719）

結論と仮定：



1.グローバルITコミュニティにおけるロシアの影響力のあるITスペシャリスト（ロシア在住）の割合は、一貫して約1％です。



2.グローバルITコミュニティのIT分野（影響力のある専門家を含む）に関心のあるロシア人（ロシア在住）の割合は約1.5％で、全コンテンツの約2〜3％を生み出しています。



3.英語の無知または知識不足は、世界の舞台でロシアのIT専門家の存在度を高めるための重大な障壁となり得る。 この結論は、少なくとも中級レベル（B1-B2）の英語の知識が絶対に必要なStackoverflowのロシアからの登録ユーザーの割合が0.5％未満であるという根拠に基づいて行うことができます。 同時に、英語の知識が初期レベル（A0-A2）でのみ必要なGitHubの登録ロシアユーザーの割合は約1.5％です。



4.ロシアのユーザーは絶対的にこれらのリソースの上位10位にいますが、相対的な指標（たとえば、人口10万人あたりのユーザー数や人口10万人あたりのコミット数など）は他の国に遅れています。 たとえば、（2012年のデータ）ロシアの100,000人あたりのGitHubコミット数は88です（比較のため、ウクライナは143、ベラルーシは247、エストニアは697、スイスは1437）。
