Revolution RはMicrosoft Rに改名され、開発者と学生が無料で利用できます。



MicrosoftがRevolution Analyticsを買収してから9か月で、RはSQL Server、PowerBI、Azure、Cortana Analyticsとの統合は言うまでもなく、Revolution R OpenおよびRevolution R Enterpriseの多くの更新をリリースしました。



アメリカの企業Revolution Analyticsは、統計データ処理ソフトウェアのメーカーです。 同社は、Rプログラミング言語の商用化とそれを使用したソフトウェアソリューションの作成を専門としています。

Revolution Analyticsの主な製品は、マルチスレッドコンピューティング向けに最適化されたR言語ランタイム（統計データ処理およびグラフィックス用のプログラミング言語）であるRevolution Rと、「ビッグデータ」概念のフレームワーク内の並列処理用のライブラリセットです。



数日前、MicrosoftはMSDNおよびDreamSparkサブスクリプションを通じてRサーバー製品へのアクセスを許可しました。



Microsoftからの公式発表は1月12日に行われました。 同社によれば、Revolution R OpenはMicrosoft R Openに名称変更され、 無料でダウンロードできます。また、Revolution R EnterpriseはMicrosoft R Serverに名称変更されました。



Microsoft Data GroupのコーポレートバイスプレジデントであるJoseph Siroshは、Machine Learning Blogで新製品Microsoft R Server Developer Editionのリリースを発表しました。 同じエディションには、 以前に発表された Microsoft Data Science Virtual Machineが搭載されます。 これにより、Rユーザーは、環境を設定することなく、クラウドでのデータマイニングとモデリングの作業をすばやく開始できます。 Microsoft R Server Developer Editionには商用機能が含まれていますが、無料で配布されています。



Microsoft R Serverの商用ディストリビューションは、SQL Server RサービスとしてSQL Server 2016に統合されます。 SQL Server 2016より前は、Revolution R Enterprise for Windowsはスタンドアロン製品として出荷されます。



Microsoft R Serverを開始するには、次のリンクのいずれかからダウンロードできます。

全員： MRANからMicrosoft R Openをダウンロード （無料）

学生向け： DreamSparkからMicrosoft R Serverをダウンロード （無料。Microsoftアカウントが必要）

開発者向け： Visual Studio Dev EssentialsからMicrosoft R Serverをダウンロード （無料。Microsoftアカウントが必要）
