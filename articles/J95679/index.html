パート1 「鉄」





これはビジネス指向のワードプロセッサです。 かなり強力なテキストエディター、スプレッドシートエディター、カレンダーなど。 （詳細は本文で後述）。







通常のラップトップのフォームファクターを備えています。



仕様：



-6MHzプロセッサー（hd64180rf6x）



-33KBのメモリ



-3.5インチフロッピーディスク（1.44 Mb）で動作する組み込みフロッピードライブ



-9インチモノクロディスプレイ（22 x 80文字）



-パラレルポート



-赤外線ポート



-モデム



-Bookmanカードマルチメディアカードとの互換性



左側には、電源（9V）用のコネクタ、モデムRJ11、トグルスイッチ、ディスプレイコントラスト調整器があります。 輝度制御がなく、画面にバックライトがないことに注意してください。







右側に合うのはフロッピードライブのみです。







蝶番を付けられたふたの下の背部に平行および赤外線ポートがある。







注意深く見ると、デバイスを装着しやすいように、ケースに小さなくぼみがあり、指の形をしています。



ネームプレートステッカーに加えて、下部に2つのコンパートメントがあります。 バックアップバッテリー用（標準3V CR2032）。 2番目のカバーの下には、ブックマンカード用のスロットがあります。 カードにはさまざまな情報（ゲームを含む）があります。 残念ながら、私はカードを持っていません。











バッテリーコンパートメントはキーボードの上にあり、1400 mAhのNiCadバッテリーを搭載する必要があります。 キットにはバッテリーがありませんでしたので、残念ながら、コンセントにアクセスしなくてもデバイスを使用できるようになるまでの時間は言えません。







ディスプレイの下に大きな情報ステッカーが貼り付けられており、「マシン」の利点がリストされています。 率直に言って、このステッカーはある種のおもちゃのような外観をデバイスに追加し、おもちゃのラップトップがどのように子供に販売されているかを知っています。







内部の内容を確認します。



3本のネジを外すと、キーボードを取り外すことができます。







さらに2本のネジを使用すると、ディスプレイの上部カバーを取り外すことができます。 細い柔軟なリボンケーブルを使用して画像を送信します。







写真でわかるように、マザーボードのクラシックコネクタには、完全に通常の3.5インチフロッピードライブが使用されています。珍しいことではありません。







すべての要素は、マザーボードの片側にあります。 Bookmanカードリーダーボードは個別にあり、ループによってメインボードに接続されています。







メモリはボードにはんだ付けされており、メモリを増やすための追加のコネクタは提供されていませんが、おそらく必要ありません。







実際には、Hitachi 64180プロセッサ（ボード上の遠くの大きなマイクロ回路）。







パート2 「ソフトウェア」。



そのため、デバイスの電源を入れると、画面に挨拶が表示されます。







クイックブート後、メインメニューが使用可能になります。 なぜなら マニピュレータが提供されていないため、ポイント間の移動はキーボードのナビゲーションキー（矢印）を使用して実行されます。 ところで、彼らは並外れて、左に、上に、下に、右に位置しています。 一般に、キーボードレイアウトはおなじみのqwertyであり、コードファンクションキーもあります。これと組み合わせて、いくつかのキーが追加機能を実行します。 キャンセルキーとメニューがあります。 バックスペースキーは、通常の矢印の代わりに-BSとマークされます。







スクリーンショットを見るとわかるように、メニュー項目は12個のみです。 もう少し詳しく見ていきましょう。



最初の段落の下には、かなり強力なワードプロセッサがあります。 テキストのフォーマット、ページサイズの設定、ブロック内のテキストのコピー、次のページ、ヘッダーとフッターへの移動などのためのさまざまな機能があります。 ドキュメントは.wpt形式で保存できます。 すぐに印刷することもできます。







スプレッドシートエディタを使用するには、ディスケットを挿入する必要があります。 これがないと、明らかにファイルサイズが大きいため、テーブルは起動しません。 さまざまなテーブルアクションも使用できます（ソートなど）。







ファイルは.wpr形式で保存され、.wptに変換することもできます。



次のメニュー項目はアドレス帳です。 アドレス帳に記入することで、アドレス帳の連絡先を使用してFAXを送信でき、送信時に電話番号を入力しないでください。







4番目の項目は、ファックス送信メニューです。







5番目のメニュー項目は通信で、TTY端末が使用可能で、接続、電話番号、およびファイル送信のさまざまな設定があります。







次のアイテムは、ブックマンカードで動作するように設計されています。 カードがないため、実際にはうまくいきませんでした。カードがないことを示すメッセージが表示されます。デバイスをオフにしてカードを挿入することをお勧めします。







7番目の項目はカレンダーで、特定の日時を参照してさまざまなタスクを追加できます。正確な時間は30分です。







現在人気のあるToDoリストは、8番目のメニュー項目に実装されています。 メモを作成し、何をする必要があるか、終了日を示すことができます。 特定の日付と時刻に信号を設定することもできます。







なんて世界時間のないビジネス、そして確かに一般的な時間です。 9番目のポイントは彼だけに当てられます。 世界時計、アラーム、現在のタイムゾーン（都市）、日付と時刻の設定-それだけです。











次の項目のゼロはファイルマネージャーです。 保存されたファイルの表示、ディスケットの確認、ファイルのコピー、ファイルを.txt形式および.txtから.wptに変換できます。 プリンター上のファイルを一覧表示できます。







[ディスクアプリケーション]メニュー項目には、文字Aの番号が付けられています。ところで、適切なキーを押すと、特定の項目に移動できます。



このメニューを使用すると、ディスクからいくつかのアプリケーションを起動できますが、それは私にはわかりませんでした。



最後のセットアップメニューでは、プリンターへの接続を設定し、パスワードを設定し、高速充電モードをオンにしてデモモードを開始できます（デモはメインメニューからCode + Dの組み合わせで利用できます）。











Code + Sの組み合わせで利用可能な計算機もあります。 特別な計算機は目立ちません。







特定の機能の主な機能を詳細に説明するヘルプ（コード+ H）もあります。
