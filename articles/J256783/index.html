Frontend Dev Conf'15は4月18日にミンスクで開催されました

4月18日、Frontend Dev ConfはフロントエンドとUIの専門家を3回目に集めました。 参加者は300人以上の開発者でした。 このイベントは、友好的な雰囲気、楽しい舞台裏での会話、スピーカーとの激しい議論の中で開催されました。















FDConf'15は、従来2つのストリームで構成され、14の外観が含まれていました。 今回は、プログラムと技術レポートの記入にさらに重点が置かれました。 報告のレベルは会議プログラム委員会によって監視されていました：ディマ・ドゥディン、ヴォヴァ・ダシュケヴィッチ、イゴール・ミアスニコフ、ユラ・タテンコ、ミシャ・ラルチェンコ、ヴァレラ・カミンスキー。



いくつかのパフォーマンスについて簡単に



デニス・ミシュノフは、「時間の幻想。 60秒が1分ではないとき」と彼は、時間の知覚に関する考えに刺激を与えました。



Andrey Salomatinは、レポート「非同期呼び出しのデバッグを停止して生活を開始する方法」で、難しい質問に簡単な言葉で答えることができました。



イリヤクリモフは、トピック「スローJS：露出のある黒魔術のセッション」で、実例と理解可能な解決策を提示しました。



Mapbox GL JSの作成者であるVladimir Agafonkinは、レポート「WebGLでのモダンインタラクティブマップの仕組み」で、オープンライブラリが内部からどのように機能するかを文字通り指で示しました。















コーヒーブレイク中、参加者はレポートで聞いたことを話し合い、スピーカーとの個人的な会話の中で直接質問に答えました。



そのため、たとえば、イリヤクリモフによって引き起こされた「青いもの」は、参加者の間で騒動を引き起こしました（彼はそれを呼んでいます）。 イリヤはアフターパーティーでも彼女と別れませんでした。





















休憩の最も活発な参加者は、トリッキーなクエストタスクを解決し、XBOX Oneのために戦いました！ 特に、プログラム委員会からのタスクを試さなければなりませんでした。たとえば、エニグマによってエンコードされたメッセージを解読するためです。













会議が終了した後、全員がTNT Rock ClubのFDConfとアフターパーティーに行き、そこでメインの賞が受賞されました。



今後のイベントはすべて公開されません。 伝統により、FDonfアフターパーティーにあったものはすべてそのまま残ります。



























ハッシュタグ#fdconfの下で、ソーシャルネットワークで多くの楽しい瞬間を見ることができます。











会議議事録



ビデオレポートFrontend Dev Confは4週間後にオンラインになります。 下のプレゼンテーションや講演者に精通することができます。来年の雰囲気とインスピレーションに来てください。



















http://www.slideshare.net/FDConf/ss-47288087でのプレゼンテーションは利用できません。








Frontend Dev Conf'16でお会いしましょう！
