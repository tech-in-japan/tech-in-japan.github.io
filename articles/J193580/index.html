2GIS-個別、スパマー-個別

言葉にならないスローガン2GISは、「フェンスから昼食まで」 作成できます。 それは私たちが持っているものを非常に正確に反映しています-都市に関するすべての有用な情報。 一部のユーザーにとっては、フェンス、他のユーザーにとっては-正確な時間、他のユーザーにとっては-何か他のものです。



私たちは関連性があり、正確であり、企業に関するほとんどすべてを知っているため、彼らは私たちを愛しています。 残念ながら、誰もが心からこの愛を持っているわけではありません。 2GISを自分の利己的な目的や他の人の不利益のために使用できると決めたユーザーがいます。 これは私たちに合わないので、終わらせない場合は、そのようなユーザーの生活をできるだけ複雑にすることにしました。



もちろん、スパマーに関するスピーチ。 カットの下では、 オンラインバージョン 、 API 、および次に何が起こるかについての電子メールアドレスの消失についての小さな物語があります 。





カスペルスキーによると、電子メールトラフィックのスパムの割合は70％です。 最近まで、2GISは、スパマーがメーリングリストを補充するためのソースでした。



2002年頃にディレクトリに電子メールアドレスが表示されました。 今では、その数は確実に誰も言うことはありませんが、確かにそう多くはありません。 時間が経つにつれて、リリースからリリースまで、その数は増加しました。 これは、もちろん、コールセンターだけでなく、一般的な技術の開発によるものです。 今日、2GISには約2,000,000の企業があります。 これらは約70万の電子メールアドレスであり、「電子郵便配達員」にとって非常に興味深いものです。 パーサーの助けを借りて、彼らはデータを引き出し、郵送またはさらなる販売に使用しました。 彼らの行動はAPIに負担をかけ、ディレクトリにある会社に損害を与えました。



私たちは積極的にこれと戦うことにしました。



1つの選択肢は、電子メールアドレスを非表示にすることでした。 ただし、決定する前に、これがユーザーや企業に与える影響を確認する必要がありました。 実験のサイトとして、最も動的なバージョンであるオンラインを選択しました 。



仮説検定



実際、電子メールが必要ですか？ 彼らはそれらを使用していますか？ 彼らは会社とのコミュニケーションの効果的なチャネルですか？ 削除する前に、これらすべての質問に答える必要がありました。



データ分析から始めました。 決して嘘をつかない統計では 、500件の電話の場合、郵送先住所が1回クリックされることが示されました。 重要ではありません。



私たちはインターネットを見ました。 そこには、カタログや価格など、サービスや価格をすばやく見つけるのに役立つその他のものが掲載された企業のWebサイトが見つかりました。



より専門的なリソース（ Flamp 、 Banks.ru ）とソーシャルネットワークがインターネット上で見つかりました。そこでは、会社の代表者とチャットしたり、あらゆる方法で賞賛したりscったりすることができます。 それも滑らかです。



安全にプレイすることにしました。 自分自身、自分自身の友人、友人の友人にインタビューしました。 電子メールを最後にクリックしたとき、誰も覚​​えていませんでした。 そして、多くの人（大多数）は、@が手紙に書かれなかったと言った。手紙は恐らく奈落の底に落ちるだろうからだ。 また、原則として、特定の専門家のメールを呼び出したり検索したりする方がはるかに簡単で効率的です。 すべてが再び明確になりました。 ほぼ。

結論



ディレクトリ内の電子メールを非表示にすることによる悪影響はありませんでした。 まあ、正直なユーザーの場合。 また、API 2GISの負荷は約30％減少しました。 そして、これらの負荷の30％がパーサーの作業によって作成されただけであることがわかります。



同時に、2GIS-onlineには、 PCおよびモバイルバージョンを使用するためのシナリオとは異なる独自のユーザーシナリオを持つ、独自の対象者がいることを理解しています 。 そのため、メールが単純に隠れることはありません。



メールの代わりに、会社に連絡するためのフォームが表示されます。 実際、メッセージは以前に指定されたのと同じ電子メールに送信されますが、2GISサーバーを経由します。 電子メールアドレスであるtidbitはプログラム自体に保存されないため、これによりPCバージョンのハッキングが役に立たないタスクになることを願っています。
