テストおよび展開ツール

アプリケーションとの互換性の問題がありますか？ これを行うには、これらの問題の解決に役立つツールがあります。



ツールキットと例：



Microsoft Deployment Toolkit（MDT）2008アップデート1 （571 KB-20.7 MB）

Microsoft Toolkit（MDT）2008は、OCデスクトップおよびサーバーアプリケーションを配布（インストール）するためのツールです



Microsoft Application Compatibility Toolkit 5.5 （404 KB-20.6 MB）

Microsoft Application Toolkit（ACT）には、Windows 7、Windows Vista、Windows Update、Internet Explorerに展開する前にアプリケーションの互換性の問題を評価および軽減するために必要なツールとドキュメントが含まれています。



Microsoft Assessment and Planning Toolkit （27 KB-219.3 MB）

Microsoft Assessment and Planning Toolkitは、さまざまな移行方法について現在のITインフラストラクチャを評価するのに役立つツールです。
