エフィカMX



私は長い間、小さな「ホームコンピューター」を探していました。主な選択基準は次のとおりです。

  • 無音
  • 非USB WiFiの存在
  • 720pビデオ再生
  • 独創性
  • 300ドル以下の値札


Googleの発行を解析し、Habrを読んで、今日のPowerPC \ Amigaプラットフォームの開発に興味を持ちました。 しかし、奇妙な組み合わせは、このラインのフォロワーとしてGenesiの製品にもたらしました。 Genesiは、ARM™およびPower Architecture™に基づいた低コストでエネルギー効率の高いコンピューターを設計および製造しており、Webサイトで読むことができます。

一般的に、石鹸は石鹸であり、他の潜在的な候補に興味がある人は誰でも-猫へようこそ。



したがって、スマートトップはEfika MXです。





寸法：160mm x 115mm x 20mm

重量：250グラム

箱の中には：

プロセッサ-ARM™Freescale i.MX515（特に好奇心が強い人のために、これはiPadのARM Cortex-A8 800MHzおよびFreescaleウェブサイトとWikipediaの技術仕様）、要するに、この割合にはOpenGL ES 2.0をサポートする3DアクセラレータがありますOpenVG 1.1は、720pビデオをハードウェアでデコードし、すべての種類のフィルターとハードウェア処理を備えた1366 x 768の解像度でプライマリ画面に24ビット画像を表示できます。

512Mb DDR2 RAMは、ハードドライブの代わりにボックスにインストールされます（これは日常のタスクには十分です）-8Gb SSD（そのうち約2Gbがシステムによって占有されます）、モニター（TV）へのHDMI出力。 Efikは、WiFi（内蔵アダプターは802.11b / g / nをサポート）を介してインターネットに接続し、NASをイーサネットに接続できます。 また、2つの（はい、はい、ハブをインストールする必要があります）USB2.0コネクター、SDHC対応のカードリーダー、およびマイク付きのヘッドフォンジャックもあります。



また、2010年の今年8月に、同社は10 インチのスマートブックを市場に投入し、同様の構成（3G、ウェブカメラ、およびBluetoothモジュール用のSIMカードスロット）で、管理者は価格についていまだに黙っています。
