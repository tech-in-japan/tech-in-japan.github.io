マイクロSDカード内の頑丈なコンピューター





何百万人ものハッカー、スパマー、詐欺師があなたの個人的なオンライン情報を捜しているので、あなたのパスワードが複雑であるにもかかわらず安全であると期待することはできません。 私たちのほとんどは、ますます多くのオンラインサービスに登録するため、パスワードを失うことを心配しています。



ただし、Googleはインターネット上のセキュリティについても懸念しており、最も価値のある情報を最も合理的な方法で保護できるようにしたいと考えています。 Googleは最小のコンピューターであるProject Vaultを作成しました。



このキャッチーなタイトルは、毎年恒例のGoogle I / O開発者会議の2日目に紹介されました。



Project Vaultは、Google ATAP（Advanced Technology and Projects）によって開発された完全に安全なコンピューターで、microSDカードにパッケージ化されており、PCでも携帯電話でも、どのシステムにも挿入できます。



技術的には、Vaultはコンピューターですが、通常のコンピューターではありません。 むしろ、パスワードを入力せずに友人と通信してネットワークにログインする新しい方法です。



今日、私たちはトリッキーまたは覚えやすいと思うパスワードを選択します。 Googleは、あなたのパスワードを、私たちが考えているよりもさらに個人的なものにしたいと考えています。



プログラムまたはサイトがパスワードを要求する前であっても、小さなチップを使用して認証できます。 要するに、Vaultはプライマリ認証（つまりパスワードを使用）をセカンダリにします。 この場合、何もする必要はありません。



Project Vaultを使用すると、GoogleはデバイスをIDに変換します。 Vaultは、チャットアプリケーションとメッセージを暗号化または保護し、追加の検証レベルを提供して、デバイスが本物であることを確認できるようにします。



試用版のデモを行い、ATAPチームは、Vaultがインストールされたスマートフォンを使用して2人が互いに安全に通信する方法を示しました。 チップがインストールされると、通信プログラムは、読み取り/書き込み、入力/出力の機能を持つ2つのファイルで構成される仮想システムを開きます。



Vaultは、チャットメッセージを暗号化し、特別な安全なチャネルを介してコードとして送信します。 Vaultが電話でそれらを送信することはないため、誰もあなたをスパイすることはできません。 メッセージを受信した後、電話機は自動的にメッセージを復号化しますが、同時に、通信に参加している電話機はどれもキーまたはアルゴリズムを認識できません。

同社によれば、Vaultで動画を暗号化することもできます。



Project Vaultには次のものがあります。

-ARMベースのプロセッサ。

-送信機;

-近くのデバイスとの通信用のClose Communication System（NFC）。

-4ギガバイトの「分離され、密閉された」内部メモリ。

-その非常に安全なオペレーティングシステムは、プライバシーとデータ保護に重点を置いています。

-暗号化サービスのセット。



Vaultは、特別に作成されたリアルタイムオペレーティングシステム（RTOS）を使用し、ハッシュ、グループエンコーディング、サブスクリプション、ハードウェア乱数ジェネレーターなどの一連の暗号化サービスを使用します。 これはすべて、あなたの情報と友人とのコミュニケーションを完全に安全に保つために行われます。



Vaultは、Android、Windows、Linux、OS Xなど、あらゆるコンピューターや携帯電話と互換性のあるストレージデバイスであるため、あらゆるオペレーティングシステムで動作します。



ここでは、すべての作業がデバイス自体によって行われ、デバイスの構成に人が関与することはないため、よりスマートに呼び出します。 これにより、非常に使いやすくなります。



Project Vaultは現在「非常に実験的な段階」にあり、使用準備中ですが、システムのソースコードは既にGitHubで利用できるため、開発者は調査を開始できます。
