Microsoft Xbox One完全解析

ですから、今日も新世代のコンソールをご紹介します。 PlayStation 4の解析に成功した後、新しいMicrosoftデバイスが登場しました。 Xbox Oneは11月22日に発売され、Microsoftはすでに1日あたり100万台のコンソールを販売していると報告しています。 前世代のコンソールは8年前の2005年に登場しました。 合計8,000万台のコンソールが販売されましたが、Microsoftは2015年までXbox 360をリリースし続けます。







技術仕様



  • CPU-1.75 GHz 8コアx86-x64 APU中央処理装置
  • GPU-768コアの12のコンピューティングモジュールを備えたAGCNアーキテクチャに基づく、APUのAMD Radeon GPU。 D3D 11.2 @ 1.23 TFLOPS
  • RAM — DRAMとESRAMの2種類のメモリ。 DDR3のボリュームは8 GB（2133 MHz）です（残りの3 GBはオペレーティングシステムのニーズに制限されているため、5 GBのみが使用されます）。 ESRAMの容量は32 MB、RAMの帯域幅は68.3 GB / sです。
  • ハードドライブ-500 GB、取り外し不可、外付けハードドライブを接続可能
  • ブルーレイおよびDVDドライブ
  • Kinect 2.0コントローラー。 高解像度カメラ-1080p。 顔認識。 IRセンサーにより、撮影の品質が向上し、暗闇でも撮影できます
  • 3つのUSB 3.0と2つのHDMI（入力および出力）
  • 有線ギガビットイーサネット（1 Gb / s）、ワイヤレスWi-Fi（IEEE 802.11 a / b / g / n、IEEE 802.11ac）


Xbox Oneは、4K解像度と7.1サウンドをサポートしています。







コンソール機器には、電源、Kinect 2.0コントローラー、ゲームパッドが含まれます。 最初の分割払いでコンソールを購入した幸運な人は、コントローラーに「Day One 2013」という銘が表示されます。 おそらく遠い将来に、そのようなゲームパッドは素晴らしい金額でオークションで販売されるでしょう。 Microsoftの広報担当者は、新しいゲームパッドの開発が1億ドルのコストを超えたと語った 。 スクリーン、カメラ、臭気カートリッジ、さらには内蔵プロジェクターを備えたプロトタイプがありました。
