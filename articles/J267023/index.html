3D再構築のためのモバイルアプリケーションの比較

夏に、サムスンは新しいタブレット-Galaxy Tab S2をリリースしました。 そこでは、とりわけ、タブレットカメラを使用して3Dモデルを作成するためのオリジナルアプリケーションが登場しました。 この機能のリリースはまったく気付かれていませんでしたが、微妙な愛好家（私のような）にとってはもちろん、これは非常に興味深い技術です。

スマートフォン用の現在の3D再構築アプリケーションを詳しく調べて比較することにしました。



用途



  1. Samsungからの3Dキャプチャ。 Galaxy Tab S2のみで利用可能です。 Samsungアプリケーションストアでは、アプリケーションへのリンクを提供することはできません。 そのため、開発者からのビデオへのリンクは次のとおりです。 ボーナスとして、他のアプリケーションと比較して、非常に興味深い3次元エディターがあります（これは、サムスンを多額のお金で個別に販売することができます）。
  2. スキャン3D
  3. TRNIO 。 現在、iOSでのみ利用可能です。
  4. オートデスクからの123Dキャッチ 。 すべてのプラットフォーム用のバージョンがあります。




シナリオと結果



モデルを構築するには、アプリケーションを実行し、カメラをオブジェクトに向け、モデルを円周するだけです。 比較結果は表にまとめられ、結果のモデルはsketchfabに投稿しました 。 比較のために、デスクトップアプリケーションPhotoscanで取得した再構築結果も追加しました。

以下のパラメーターが比較されました。

  1. 再構成の品質。 ここでは、123D Catch、次に3D Captureによって最良の結果が示されました。 写真スキャンでこのような低品質が判明した理由は明らかではありません。 私の経験では、通常ははるかに高いです。
  2. 再構成の速度。 3Dキャプチャが先にあります。 1分から20分の間の差は、20分から2時間の間よりもはるかに大きいことに注意してください。 再構築が1分間しか続かない場合、取得が不十分なモデルを簡単に再撮影して、高品質を達成できます。 再構成が20分間続く場合、そのような機会は、おそらくほとんどありません。 そのため、たとえば、同じ照明の下で同じ条件ですべてのアプリケーションのテストを実施しました。 しかし、2時間の操作後の123Dキャッチは、何も発生せず、再取得する必要があることを教えてくれました。 そして、再撮影の機会は、数日後に異なる照明でしか現れませんでした-これは少し不正です。
  3. 簡単に撮影できます。 最も便利なのは、ユーザーがスタート（3DキャプチャーおよびTRNIO）を押すだけでよい完全自動モードです。 SCANN 3Dはもう少し複雑で、各フレームを個別に撮影する必要があります。 123D Catchの場合、ユーザーは正確に十分な照準を合わせなければならないため、撮影が大幅に複雑になります。
  4. クラウドの使用。 ここではすべてが明確です-クラウドを使用する場合は、インターネットが必要です。 また、旅行中に多くの写真が撮影されるため、インターネットを使用することが重要になる場合があります。
  5. 共有方法。 これまで、3次元モデルを交換するために広く受け入れられている方法はないため、アプリケーションは異なるアプローチを使用します。

    • モデルを郵送します。 素晴らしい方法ですが、ほとんどの場合、受信側は支援なしでそのようなファイルを開くことができます。 それほど難しくはありませんが、誰もがそれを行う方法を知っているわけではありません（または、可能です）。
    • 内部ソーシャルネットワーク。 作成されたモデルは、アプリケーション自体で表示できます。友人は、電話からアプリケーションを入力することでモデルを見ることができます。
    • そして最後に、モデルをsketchfabに配置できます。 非常に便利なサービス-モデルは任意の方法でねじれたりズームしたりできます。Androidやayoを含むブラウザーから機能し、すべてがシンプルで無料です。

結論



当初は多くのSF作家であり、次に科学者だった3D再構築は、広く利用可能なテクノロジーに変わったと言えます。 現在不足しているのは、ブラウザ、広く受け入れられている転送およびストレージ標準などのインフラストラクチャです。 しかし、今から5年後には、3次元の画像を単純な画像と同じように撮ることが普通になり、簡単になると思います。
