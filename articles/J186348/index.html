Ubuntuでは、マークテスト付きのDota 2が利用可能になりました



ニュースがまだHabréに載っていないのはおかしいので、ニュースを一般の人と共有することが私の義務であると考えています。



そのため、ValveのカルトDota 2ゲームは、マークテストでLinuxで利用可能なゲームのリストに登場しました。

報告されているLinuxの最小システム要件は次のとおりです。



-Ubuntu 12.04;

-IntelまたはAMD 2.8 GHzプロセッサー。

-4GBのRAM。

-nVidia GeForce 8600 / 9600GT;

-ATI / AMD Radeon HD2600 / 3600;

-OpenGL 2.1;

-4GBの空きディスク容量。

-OpenAL互換のサウンドカード。



Dota 2はMacでも利用可能になりました。



-OS X Lion 10.7;

-4GBのRAM。

-4GBの空きディスク容量。

-nVidia 320M以降。

-Radeon 7000以降。

-Intel HD 3000以降。



両方のバージョンはテスト中です。 バグレポートでは、 ここに送信することをお勧めします 。
