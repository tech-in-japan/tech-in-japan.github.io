ゴビー-一緒に作ろう

ゴビーとは何ですか？



Gobbyは、1つのセッションとマルチユーザーチャットで多数の開いているドキュメントをサポートする無料の共同（共同）エディターです。

Windows、Mac OSX、Linuxで動作します。 このエディターを使用すると、複数のユーザーが同じドキュメントを同時に編集できます。

Gobbyを使用すると、いくつかのテキストマテリアルをすばやく話し合ったり編集したりできます（たとえば、その方法など）。





インストール方法



Linuxユーザーの場合、すべてが非常に簡単です。ユーザーは、オペレーティングシステムのリポジトリからGobbyを直接インストールできます。



Debian / Ubuntuユーザーは次のようにインストールします：

sudo apt-get install gobby







他のプラットフォームにインストールするには、公式Webサイトのインストール手順を ダウンロードして使用できます 。問題が発生することはありません。



Gobbyはクライアントサーバーシステムで動作します。ユーザーの1人がサーバーになり、残りのユーザーはそのIPアドレスを知ってサーバーに接続します。 何らかの理由でGobbyで働きたい人が専用のIPアドレスを持っていない場合は、GobbyをHamachiと組み合わせて使用​​できます。



特徴



  • データ暗号化を使用したリアルタイム通信
  • 各ユーザーは、自分のテキストを他の参加者のテキストと区別する独自の色を持っています
  • エディターは、多数のプログラミング言語の構文強調表示をサポートします
  • パートナーとのコミュニケーションのための非常に便利なIRCのようなチャット
  • パスワードセッション保護
  • 1つのセッションで多くのドキュメントを開く
  • ドキュメントのドラッグアンドドロップモード
  • ドキュメント同期を呼び出す
  • Zeroconfのサポート
  • Unicodeサポート
  • Gobbyはクロスプラットフォームプロジェクトです。 Microsoft Windows、Linux、Mac OS X、およびその他のUnixライクなオペレーティングシステムで動作します
  • Gobby-GPL 2ライセンスの下で配布されるフリーソフトウェア

私のパートナーはコレクターです



2人が一緒に記事を編集しているため、2人目は最初の記事の内容を追跡します。 したがって、「修正者」が犯した間違いのほとんどを発見し、修正に役立つ可能性が高いです。 たとえば、この記事はGobbyで書かれており、パートナーを修正する機会がありました。

一緒に学ぶ



友人とプログラミング言語を教える際の有用な経験は、Gobbyに関連していました。この素晴らしいクライアントに集まり、クラス、関数を書き、コメントし、モデルの構築について議論し、時には興味深い解決策を思いつきました。 Gobbyは、コード例を記述しやすく、話しやすいため、パートナー（学生教師モデル）の助けを借りて特定のプログラミング言語をすばやく学習するのに適した環境であると考えています。

ACEも同様のGobbyテキストエディターで、テキストのコピーと貼り付け、ドキュメントの読み込みと保存などの標準機能をサポートしています。

CoWordはMicrosoft Wordの拡張機能であり、複数のユーザーが同じドキュメントを同時に編集できるようにします。
