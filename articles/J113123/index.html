Ubuntu 11.04 Natty Narwhal Alpha 2リリース

最初のアルファ版がリリースされてからちょうど2か月が経過し、現在Ubuntu 11.04 Natty Narwhalの2番目のアルファ版がすでにリリースされています。



すべての新しいリリースと同様に、多くのパッケージが更新されています。 それとは別に、注目に値します：LibreOffice 3.3（OpenOffice.org 3.2に置き換えられました）、Linuxカーネル2.6.38-rc2、X.org Server 1.10およびMesa 7.10。



カットの下-最初のアルファと比較した外部の変更。 そして、Canonicalの批判を始める前に、これがアルファであることを最初に覚えておいてください。 :)



団結



Ubuntu 11.04のデフォルトのデスクトップ環境であるUnityは大きく変わりました。



現在、Dashの元のバージョンが使用されています。







ファイルとアプリケーションの表示が多少簡単になりました。







サウンドメニュー



サウンドメニューに返されるプレイリストのサポート：







Ubuntu Oneダッシュボード



Ubuntu Oneアカウントを簡単に同期および管理できるように、Ubuntu Oneダッシュボードアプリが追加されました。







グリッド



現在、デフォルトでは、Compizプラグイン「グリッド」が有効になっています。 ウィンドウを上にドラッグして全画面に拡大し、左または右にサイズを変更できます。







GTK +グリッパー



自動サイズ変更に慣れていない場合は、新しいグリップ要素を使用できます。







アプリケーションセンター



これで、アプリケーションを評価し、すぐにTwitterに投稿できる小さなレビューを書くことができます。







Ubuntu Classicデスクトップ



Unityグローバルメニューアプレットは、トップパネルのコンパクトメニューボタンの横にあります。







設置



2番目のアルファの使用を本当に開始したい場合（開発者でない場合はもう一度考えてください）、10.10からのアップグレード手順およびシステム自体へのリンクは、 http ：//www.ubuntu.com/testing/natty/alpha2から入手できます 。
