Project Araモジュラースマートフォンプロジェクトは、サードパーティの開発者に公開されています





Google Motorolaの Project Modular Project Project Araは、ますます活発に開発しています。 少し前、プロジェクトチームは、 スマートフォンを2015年に発売することを発表しました。 そして現在、このプロジェクトは、81ページのモジュール開発者キット（MDK）を公開することにより、サードパーティの開発者に公開されています。



実際、プロジェクトチームは、信頼できる分岐したエコシステムをスマートフォン向けに作成することを目的として、サードパーティの開発者向けにプロジェクトへの幅広い道を開いています。 「Araプロジェクトの成功は、無数の開発者の豊富で活気に満ちた分岐したモジュールのエコシステムに基づいています」とプロジェクトチームは言いました。



これらの同じ「無数」には、必要に応じてHabrasocietyの代表者が含まれる場合があります。 幸いなことに、スマートフォン自体の構成は数千にもなります。 公開されたドキュメントには、ミニからラージまでのモデルのこのような構成の例が記載されています。 スマートフォンを組み立てるための主なプラットフォームには、小型、基本、中型、大型の3つのタイプがあり、これらに基づいてファブレット（タブレット電話）を設計することができると以前に述べられました。







デバイスの有効な構成の例が公開されています。ここにいくつかの例を示します。







あなたが理解できる限り、企業はスマートフォンの最終設計を決定する権利を留保します。いずれにしても、開発者は上記で提供された例に集中する必要があります。 さらに、スマートフォン用のモジュールは、長方形の形状や6061アルミニウムのベースなど、特定の要件を満たす必要があります。







他の構成のモジュールもありますが、血液中の酸素レベルを決定することにより、この医療モジュールのように外部に作成する必要があります。







このリンクからProject Ara MDKをダウンロードしてください 。 しかし、 Project Araに関するロシア語のサイトは、あなたが理解できる限り、非公式のリソースです。
