Sony Vaio VPCF11M1E / Hノートブックを確認する





エントリー



ノートパソコンSony Vaio VPCF11M1E / Hを購入したときの印象についてお話したいと思います。 チェコ共和国で買いました。

ロシアでは、ソニーはVPCF11S1R / Hに同じ充填剤を提供しており、その外観はVPCF11M1R / Hと一致しています。 詳細については、公式ウェブサイトをご覧ください 。 私が発見した唯一の違いは、ロシアで販売されている802.11nがないことです。

私は、特殊なソフトウェアを使用したパフォーマンス測定の結果ではなく、ラップトップを使用したときの自分自身の印象について詳しく話すつもりです。

「ソフトウェアのないコンピューターは役に立たないハードウェアだから」、購入のソフトウェア部分についても詳しく説明します。

ただし、最初にまず最初に。



選択肢



適切なパフォーマンスは、購入したラップトップの前提条件でしたが、他のすべてのパラメーターは他のパラメーターよりも犠牲になります。

HDビデオが大好きで、ATIビデオカードの操作経験もあるため、Nvidiaグラフィックカードを搭載したラップトップを選択することを主張しました。

以前は、ノートパソコンとの長期にわたるコミュニケーションの経験はすべてデルに限定されていました。数世代のLatituds、今では家族にも秋に購入されたDell 1555があり、時々彼と比較していきます。

最初は、Intel Core i7 720QM、620M、またはCore i5 520Mに焦点を当てました。 Arrandaleファミリーのプロセッサーを搭載したラップトップがチェコ共和国に登場し始めたばかりで、一般的にあまり選択肢はありませんでした。Corei5は主に430Mで表されます。

別の方法として、Dell 1557および1558を検討しましたが、ビデオカード（ATI Radeon HD4570）は好きではありませんでした。 デルを支持して、彼らは良い保証（次のビジネスの日）、スクリーン、そして実際に私のポジティブな使用経験を述べました。

チェコ共和国では、Sony Fシリーズ520M + 4GBおよび720QM + 8Gbの2つの構成のみが販売されていますが、2番目の構成はほぼ40％高価です。



構成



OS-Windows 7 Home Premium 64ビット

チップセット-Intel PM55 Express

プロセッサー-Intel Core i5-520M

メモリ-4Gb DDR3 1066

ハードドライブ-500 Gb（5400）

光学ドライブ-CD、DVD（読み取りおよび書き込み）、BR（読み取り）

ビデオカード-NVIDIA GeForce GT 330M（1Gb GDDR3）

画面-1920x1080

ネットワーク-10BASE-T / 100BASE-TX / 1000BASE-T、802.11 a / b / g / n

重量-3.1 kg







外観検査









ラップトップの表面はマットで、蓋と内側の両方があり、キーボードの前の表面は粗いです（古いモデルには「ゴムのような」プラスチックがあります）。 基本的に、ポートはラップトップの側面にあり、前面にはカードリーダーとwi-fi用のスイッチのみがあります。 カメラは中央にあるため、ノートパソコンを開いているときに「付着」する可能性が高くなります。 操作中のスタートボタンは緑色に点灯し、スリープ中はオレンジ色に点滅します。









最初の打ち上げ



私はチェコ共和国でラップトップを購入しましたが、これに関連して、チェコのWindowsでテテアテテであることを少し恐れており、Windows 7のUltimateエディションでのみ言語を変更できるという事実が懸念の原因となりました。 しかし、無駄に。 最初の起動時に、変更できない言語が要求されます。 一般に、戻るボタンのないウィザードは愚かですが、ここではそれを熟考する機会があります。 また、ワイヤレスネットワークの設定を入力する要求に失望しました。接続できない場合（私の場合のように、これまでルーターに自分の知らないアドレスを登録する必要があったため）、入力されたデータが失われるためです。

最後のインストールプロセスは、「Another Vaio is born！」というフレーズによって楽しまれました。

これは、Windowsのパフォーマンスを評価する方法です。







ソフトウェア



最初の印象は、「すべてがそんなに速いわけではない」というものでした。 ただし、Sonyおよびそのパートナーからプレインストールされたプログラムの一部を切断/アンインストールした後、すべてが何らかの形で生き生きしました。 一般的に、プリインストールされたソフトウェアの量は印象的であり、残念ながら、その多くは非常に有用とは言えません。 例：Vaio Gate-画面上部のパネルとマックのドックの下の支柱

3番目のバージョンのChromeはプリインストールされていましたが、更新することはできませんでした。

Nvidiaの公式サイトでは、GeForce 330M（および一般的な300のモバイルシリーズ全般）のドライバーはまだ提供されていません。 同時に、テレビで画像を表示することができませんでした（わずかに縮小する必要がありますが、インストールされたドライバーにはそのような機能が見つかりませんでした）。 Ubuntuにはビデオドライバーにも問題があります-専用ドライバーをインストールしてアクティブにした後、ビデオは機能しません。



印象と操作経験



  • 画面：映画は見栄えが良い。 オフィスアプリケーションでは、最初はDellよりも少し悪いように見えましたが、注意深く比較すると、それらは異なるが互いに価値があると呼ぶことができます。 しかし、ソニーを支持して-光センサー-非常に便利です。
  • キーボード：デジタルパーツの存在は私を喜ばせ、Insert \ Delete \ Home \ End \ PageUp \ PageDownの場所は非常に便利です（Dellとは異なり、上ではなく横にあります）。彼らの場所に詳しくない（コーナーではない）。 Ctrl + F4を押すのはあまり便利ではありません。





  • タッチパッド：デルのタッチパッドよりもはるかに便利で、手触りがやや粗い。





  • 重量は重要ですが、多かれ少なかれ受け入れられます。 一般的に、アセンブリの印象は悪くありません。Dellの1555と比較すると、Vaioの利点に気付くことができます。これは、古いLatitudeを思い出させてくれます。 ソニーが明らかに劣っているのは、片手でラップトップを開くことの不便さです。
  • USBデバイスおよび電源ワイヤの非常にタイトな挿入/取り外し。
  • 私の意見では、ポートの位置は最適ではありません：前面に近すぎるため、スペースが限られている場合、マウスで作業するとき、USBにつまっているもの、接続されているヘッドフォンなどが干渉します。
  • ノイズ：一般的には許容できますが、ビデオカードの集中的な作業の場合、音はなんとなくうなりますが、これはやや面倒です。
  • スリープからの脱出は即座に行われ、自宅のWi-Fiも遅滞なく検出されます。
  • サウンド：サウンドカードは非常に満足のいくものでした。古いAudigy 4とは違いがありますが、それほど重要ではありません。
  • 古いデスクトップ（AMD 4850e + ATI HD 3200）で問題が発生したものを含め、HDビデオは完全に再生されます。
  • 充電せずに作業：約250分-Wi-Fi経由でインターネットをサーフィン、200分-再度Wi-Fiを使用して画面をオンにしてラジオを聴く
  • 好奇心から：モニターオフボタン。 さらに、それは完全にオフになりません-そしてそこにあなたは写真の痕跡を見るためによく見ることができます
  • ビデオカメラが弱い（0.3 MPix）
  • ネットワークを介したファイルのコピー速度は最大70 MByte / secであり、他のハードウェアで30を超える速度は観測していません。




結論



ラップトップの全体的な印象は好意的であり、私は私が望むものをたくさん手に入れました。 私が発見した長所と短所は上記のとおりです。 おそらく、私にとって理想的な解決策は、Zシリーズの同様の詰め物で、わずか13インチの画面で、重量が約400キロのラップトップでしょうが、コストは購入した価格のほぼ2倍です。
