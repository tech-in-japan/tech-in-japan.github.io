待望のApple Watch



前日、Appleのプレゼンテーションは世界に時計を見せました。 失望-ウェアラブルデバイス市場の新しい代表者に対するユーザーの反応を1つの言葉と呼ぶことができます。 まず第一に、価格、バッテリー寿命（バッテリーの寿命は1日以内）、そしてiPhoneの失望の必要性。



ユーザーだけでなく、投資家も失望しています。午前中、同社の株式は2％以上下落しました。 パシフィッククレストセキュリティーズのアナリストは、Appleがブランドのファンに最初の数ロットの時計を販売できると考えていますが、市場で大きな成功を期待する理由はありません。 そして投資会社のパイパー・ジャフレは、デザインとブランドを考えれば、Apple Watchはウェアラブルデバイスよりも普通の時計と競合する可能性が高いと示唆しました。



状況は、すぐにいくつかの代替スマートウォッチを提供した競合他社の即時かつ正確な応答によって悪化しました。 Googleは15秒のビデオで応答し、ウェアラブルAndroidデバイスのさまざまな形と機能を示唆しました。







Apple Watchは、4月10日から米国とヨーロッパで購入できるようになります。ロシアでのリリース日はまだ不明です。 時計は次の3行で利用できます。

-スポーツを350〜400ドルで見る。

-550〜1100ドルのベースモデル。

-プレミアムWatch Editionモデルは10,000〜17,000ドル。



それまでの間、息を切らして全世界がスマートウォッチのリリースを待っています。AppleWatchとApple Watch Editionの代わりに購入できるものを選択してください。





Unity 5は非売品です！







この見出しの下では、2つのニュースが同時に送信されます。Unity5 Personal Edition は特定の条件下では無料ですが、Unityは販売に関する噂を公式に否定しています。



噂自体は、新しいバージョンのエンジンで利用できる無料機能のリストの深刻な拡張について知られるようになった後に現れました。



噂は部分的には真実でしたが、正確には、Unityは、開発者に新しいエンジンのほぼすべての機能パッケージを無料で提供しますが、これは前年の会社への投資またはその収益が10万ドル未満であった場合のみです。 それ以外の場合は、ライセンスを購入する必要があり、そのために月額1,500ドルまたは75ドルを支払う必要があります。



「このバージョンは、プロファイラー、オクルージョンカリング、レンダリングテクスチャ、フルスクリーン特殊効果、物理的に正確なシェーディング、リアルな照明と見事な視覚効果を作成する新しいツールセットを備えた強力なグラフィックス、オーディオを操作する新しいシステムなど、Unityエンジンとエディターのすべての強力な機能を提供します」、-会社の公式プレスリリースに記載されています。



有料版でのみ利用可能な機能に関して：

  • Unity Cloud Build Proサービス;
  • チームライセンス
  • ゲームのゲームパフォーマンスレポートのパフォーマンスを監視するサービス。
  • Asset Storeレベル11へのアクセス。
  • Unity Analytics Pro分析サービス。
  • エンジンのプレビュー版およびベータ版へのアクセス。


有料版と無料版の両方で、売り上げに対する利子は会社に請求されないことに注意することが重要です！





Unreal 4が無料になりました！



一週間ではなく、寛大さの魅力に過ぎませんが、それは見方です。 Unreal 4 は無料で開発者に提供されましたが、たった1つの「しかし」です。Unreal4エンジンでのプロジェクトの収益が3000ドルを超えると、Epic Gamesは四半期ごとにプロジェクトの売り上げの5％を支払う必要があります。



古い19ドルの月額サブスクリプションは初心者でも不可能ではありませんでしたが、より多くの開発者がより多くの機能と利用可能なエンジンを持っているほど、業界全体にとってより良いものになります。



2つの人気のあるエンジンと2つの非常に似た「動き」-Unity 5またはUnreal 4？！ 開発者がより優先するのは誰なのか、時間だけがわかります。 どのオプションがお好みですか？





Facebookからの匿名ログイン







Facebookは、アプリケーションでの匿名承認の可能性を発表しました。 開発者は、特別なページでベータテスト機能への参加をリクエストできます 。 ソーシャルネットワークによると、新機能はコンバージョンを増やし、人々が匿名を維持しながら、アプリケーションをすぐに使い始めるのに役立ちます。



Facebookがこのモードでユーザーデータを提供するかどうかはまだ不明です。 ただし、Facebookを介したこのような承認により、ユーザーはさまざまなデバイスを介してアプリケーションにログインできます。 さらに、ユーザーは「匿名ログイン」を通常の「Facebookログイン」に簡単に置き換えることができます。



Facebook側のこのステップは非常に興味深いようです。 ますます多くの人が自分の個人データを左右に​​渡さないようにしているので、この機会はきっと彼らにアピールします。
