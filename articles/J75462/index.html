WM 6および6.1で利用可能なWindows Mobile Marketplace



Windows Mobile 6.5の新しいバージョンで開始されたオンラインストアプログラムWindows Mobile Marketplaceは、もともとWM 6.5プラットフォームのサポートを目的としていました。 WM 6および6.1の以前のバージョンのサポートが発表され、最終的にMarketplaceでそれらと連携する機会があります。



Windows Mobile 6以降をベースにしたスマートフォンを使用している場合は、スマートフォンからアドレスmp.windowsphone.comにアクセスすると、ストア内を移動できる小さなクライアントをインストールできます。 店舗が開設された時点で、約250のアプリケーションが登録されていましたが、今日では、1か月後、すでに800を超えるアプリケーションがあります（数は国によって異なります）。



残念ですが、これまでのところ、ロシアのアカウントでは無料のアプリケーションしか利用できません。
