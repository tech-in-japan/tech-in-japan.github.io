ゲームカンファレンス

今日、情報技術の発展を決定する最も重要な分野の1つは、電子エンターテイメントの分野です。 特に、ゲーム。 世界中で、何億人もの人々がコンピューター/コンソール/モバイルゲームに年間数十億時間を費​​やしています。 ゲーム業界には何十万人もの専門家と膨大な財源が関与しており、一部のゲームは大衆文化に大きな影響を与えています。



Mail.Ru Groupでゲームを作成することは、当社のビジネスの重要な分野の1つであることをご存知でしょう。 そのため、 開発者向けの会議の同様のレビューを継続して、専門のゲーム会議に関するこのレビューを書くことにしました 。 前のケースと同様に、私たちが言及していないイベントに関する情報はコメントで歓迎されています。





QuoVadis







この会議では、ワークショップ、ディスカッション、講演会が開催され、そこで講演者とイベントのゲストが、ゲーム業界の現状とさらなる発展に関する経験とビジョンを交換することができます。



日程と場所 ：4月8日〜10日、ドイツ、ベルリン。



ウェブサイト： http : //die-entwicklerkonferenz.de/index.php



デブガム







この国際会議は、オンラインゲームおよびモバイルデバイス用ゲームの開発とプロモーションに特化しています。 このイベントには、開発者、出版社、技術専門家など、1,500人以上が参加しています。



日付と場所 ：5月15〜16日、モスクワ。



ウェブサイト： http : //devgamm.com/moscow2014/



北欧のゲーム







第10回国際会議は、スカンジナビアのゲーム開発会社および出版社の代表者、ならびにゲーム業界の主要な国際参加者によるプレゼンテーションで満たされます。



日程と場所 ：5月21日〜23日、スウェーデン、マルメ。



ウェブサイト： http : //nordicgame.com



OIGC（オタワ国際ゲーム会議）







600人以上のデベロッパーがカンファレンスのゲストとゲーム業界の発展に関する彼らの経験とビジョンをお互いに共有します。



日程と場所 ：5月25〜27日、カナダ、オタワ。



ウェブサイト： http : //oigc2014.com/



白い夜







この会議は、冬と夏に年2回開催されます。 モバイルゲームの開発とプロモーションの問題に取り組んでいます。 最後の会議には、700を超えるゲストと22か国からの参加者が参加しました。



日付と場所 ：6月26〜27日、サンクトペテルブルク。



ウェブサイト： http : //www.wnconf.com/

Cgamesアメリカ



今年は第19回国際会議CGamesが開催されます。 このイベントで議論される主なトピックは、ゲームの人工知能、ゲームデザイン、グラフィックス、視覚化、教育および科学目的でのゲーム技術の使用、オンラインゲームとモバイルゲーム、ゲームビジネスとマーケティングなどです。



日程と場所 ：7月28〜30日、米国ケンタッキー州ルイビル

GDC（ゲーム開発者会議）







このブランドの下で、GDCヨーロッパ、CDC中国、GDCネクストなど、いくつかの国際会議が毎年開催されています。 特に、GDCヨーロッパは世界最大のゲーム展示会および会議です。 ここでは、ビジネス関係を確立し、さまざまな開発会社や出版社と経験を交換できます。



日付と場所 ：GDC Europe、8月11〜13日、ドイツ、ケルン。 GDC中国、10月19〜21日、中国、上海。 GDC Next、11月3〜4日、米国ロサンゼルス。



ウェブサイト：

http://www.gdceurope.com/

http://www.gdcchina.com/

http://www.gdcnext.com/

GEM（ゲーム、エンターテイメント、およびメディア）







さまざまな国で開催された国際会議。 会議の主なトピックは、コンピューターゲーム、エンターテイメント、デジタルメディアの革新と開発です。 イベントには、開発者、専門教育機関の代表者、製造業者、および役員が参加します。 会議では、デジタルエンターテインメントの未来、社会への影響、教育におけるその使用について説明します。



日付と場所 ：10月22〜24日、カナダ、トロント。



ウェブサイト： http : //ice-gic.ieee-cesoc.org/2010/
