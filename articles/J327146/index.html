YiiConf 2017、モスクワ6月16日

今年の夏、2017年6月16日、モスクワはYii PHP フレームワークに特化したWeb開発者の会議を開催します。













スピーカーは、フレームワークを正しく操作する方法について話し、ベストプラクティスを提示し、高負荷下でYiiベースのアプリケーションを最適化する実際的な例を示します。







フレームワーク開発チームのメンバーとトレーニング資料の著者は、すでに参加申請を提出しています。







  • アレクサンダーマカロフ（@samdark）、コアチーム：

    • Yii。 次は？ 。
    • 内部からのオープンソース 。
  • Pavel Klimov（@ klimov-paul）、コアチーム：

    • プロジェクトのインストールとデータベースの移行：典型的なタスクとエラー 。
  • Dmitry Naumenko（@SilverFire）、コアチーム：

    • ドメイン駆動設計は、ほぼ複雑です。
    • Yii 2のレシピ 。
    • XDebug、XHprof、Blackfireを使用したPHPのプロファイリング 。
  • Dimitri Eliseev、ElisDN.ru、コースおよびスクリーンキャストの著者：

    • セマンティックプログラミング：コードを配置する場所 。
  • ミハイル・アルテミエフ：

    • アプリケーションの作業フレームワークを10分で作成してデプロイします 。
  • 依存性注入、コンテナおよびサービスロケータ 。


Carsten Brandt（@cebe）からレポートが届きます。







レポートまたはマスタークラスを申請することもできます。 スピーカーの場合、 YiiconfおよびDevconfへの参加は無料です。







会議の最後に、フレームワークの講演者と著者に質問できる質問と回答のセッションを手配します。 会議中に、フリップチャートを使用して、自分の考えを表現したり、興味深いトピックに関するストーリーを作成したりできます。







参加費用が増加します！ 現在、価格は4000 pからです。 パッケージに応じて 。







ウクライナの居住者への注意：市民のパスポートでロシアに自由に行くことができます。 ビザや追加の書類は必要ありません。







ハッシュタグ#yiiconfを使用して、Twitterの会議ニュースをフォローできます。







すべての質問については、2017 @ devconf.ruでメールで主催者に手紙を書くことができます。







YiiConfでお会いしましょう！
