将来を見据えて、すべてが非常に優れていることがわかりました。 スマートフォンの価格は4,990ルーブルです。 近い価格で、かなりの数のエントリーレベルのモデルを見つけることができます。 しかし、第一に、Yummy DuoのニッチはデュアルSIMフォンです。 そして、第二に、電話はWi-Fi、GPRS、3G、Bluetooth、GPSレシーバー、さまざまなセンサーなど、あらゆる通信を備えています。 まあ、価格は、それほどではありませんが、競合他社よりも低くなっています。



Highscreen Yummy Duoの両方の世代についてお話します。新しいモデルは古いモデルと非常に大きく異なるためです。 私は主に新しいモデルをテストしたので、比較のためだけに古いモデルに言及します。

仕事の印象





スクリーン


正直なところ、私はすでにSamsung Super AMOLEDに慣れているので、800 x 480スクリーン、さらにはTNについても非常に懐疑的でした。 しかし、1週間を使用したので、画面はかなりまともだと言えます。 もちろん、ピクセルは4.3インチの画面でも目立ちますが、特に明るさを自動に設定している場合は、目が疲れません。 視野角があれば、すべてがTNで予測可能です-それらは小さいです。 しかし、画面からの通常の読み取りでは、これは干渉しません。 ナビゲーターとして使用することはかなり可能です-明るさのマージンは大きく、すべてのオブジェクトと碑文は区別できます。 唯一のことは、そのような解像度では、たとえばWebページ全体を見るために小さな印刷を行うことは完全に不可能であることです。 しかし、これは、一般的に、ほとんど必要ありません。 要約として-予算モデルであってもTNマトリックスを放棄してIPSに切り替えることをメーカーに推奨します。これにより、コストが300〜500ルーブルを超えることはほとんどありません。 残りは非常に正常です。



CPU


今では古いモデルについて話すことはほとんど適切ではありません-そのプロセッサはまともな解像度でビデオを再生することさえできません。 新しいモデルでは、プロセッサは著しく高速です。 いくつかのテストビデオをチェックしました。 Full HDを明確に扱っているわけではありませんが、720p H.264は非常に魅力的です。 確かに、いくつかの特に重いプロファイルでは、まだ速度が低下します。 しかし、MX Playerがハードウェアアクセラレーションがオフになったと報告したため、何かを理解できなかったのかもしれません。 MT6575Mプロセッサの仕様は720pハードウェアをサポートしているため、他のプレーヤーが必要な場合があります。



通常の作業については、ここですべて順調です。 アプリケーションは非常に高速に実行されます。 Yandex.Navigatorは、速度を低下させず、マップをスムーズに移動および回転させます。 私はゲームをテストしていませんが、興味のある人はw3bsit3-dns.comのこの電話専用ページで確認済みリストを見つけることができます（以下のリンク）。 デバイスは、実際には負荷がかかっても加熱しません。



記憶


今日の標準ではRAMの量は少ないですが、問題はありませんでした。 強力なマルチタスクは待機する価値はありませんが、アクティブなアプリケーションでは迅速に機能します。



約512 MBのROM-もちろん、これは奇妙な動きです。 私が理解しているように、このモデルではすべてのペニーがカウントされました。 少なくとも4 GBを提供しない他の理由は見当たりません。 一方、Androidにはこのボリュームが十分にあり、microSDカードは32 GBに置くことができ、悲しみを知りません。 面白いですが、microSDカードが取り付けられていない場合、カメラは撮影を拒否します。 市場のアプリケーションも、デフォルトでmicroSDに直接インストールされます。



GPS


コールドスタートにはかなり時間がかかります-A-GPSを約2分間オンにした場合でも（おそらく、ヤードでテストするための場所を選択できませんでした）。 ただし、再起動すると、都市内のまったく異なる地点であっても、座標はほぼすぐに決定されます。 位置決め精度に問題はありませんでした。 GPSをオンにした電話は、実際には加熱されていません。 たとえば、Samsung Galaxy Noteは長時間のナビゲーションの後、非常に暖かくなります。



バッテリー


私は電話を非常に長い間使用せず、特別な測定も行いませんでした。 しかし、日中の一時的な使用（主にインターネットの閲覧）で、問題なく請求が2〜3日間続くのは驚くべきことでした。 同じことがw3bsit3-dns.comの所有者によって確認されています。 また、非常に食いしん坊のYandex Navigatorを起動し、3時間放置しました（画面がオンになり、マシンの明るさが変わりました）。 この間、バッテリーは30％放電されます。 電話のナビゲーションを知っている人は、これが非常に良い指標であることを確認するでしょう。



カメラ


私は特に前部カメラをチェックしませんでした、そして、後部カメラはよく撃ちます。 後処理のアーティファクトに気付きませんでした。 しかし、暗い場所では、写真はうるさいです。 ビデオは最大720pで撮影します。

Android


新しいモデルにはAndroid 4.0.4がインストールされています。 アップデートがリリースされないことを正式に発表しました。 しかし、予算の電話では、これはそれほど重要ではないと思います。 デバイスにはブランドシェルなしでクリーンなAndroidがインストールされていますが、これはプラスだと思います。 おなじみのセットの多くのプログラムがそうでないことに驚いた。 内部メモリの量に制限があるためだと思います。 ユーザーは192 MBしか使用できません。 しかし、おそらく、最小限のプログラムのセットのおかげで、すべてが迅速に機能します。

無線モジュールと2枚のSIMカード


デバイスには1つの無線モジュールがあります。 2枚のSIMカードの動作原理は、iXBTの古いYummy Duoモデルのレビューで説明されています。「通常の電話のように、1つの無線モジュールしかインストールされていません。 したがって、スタンバイモードとの関係でのみ、本格的な「dvuhsimovosti」について話すことができます。 その後、両方のSIMカードが実際にアクティブになり、同時にアクセスされます。 ただし、1つのSIMカードを使用して接続を確立するか、データ転送プロセスを開始するだけで、もう1つのSIMカードはすぐにアクセスできなくなります。 同時に、2番目の番号の発信者には、回線で待機するための招待は聞こえませんが、電話が切れて、加入者が利用できないというメッセージを受け取ります。



ただし、新しいYummy Duoには、「SIMで通話が失われない-SIMでアクティブなデータ送信が行われている場合でも」というフレーズが記述されている、より新しいチップセットがあります。 あるオペレーターを介してデータをロードするときに、別のオペレーターを介した着信コールが問題なく着信することを確認しました。 ただし、上記のように、2つの同時着信コールは機能しませんでした。







コンピューター接続


デバイスには標準のマイクロUSB入力があります。 また、古いモデルでケースの内側にコネクタを打ち抜くケースがあった場合、新しいデバイスの入力はいくらか信頼性が高くなります。 ただし、強度についてはテストしませんでした。



この電話機は、中国語のルーツを提供する「Linux File-CD Gadget USB Device」として定義されています。 古代のメモリカードへの書き込み速度は5〜10 MB /秒でしたが、これはまったく問題ありません。

携帯電話はきれいに見え、手に心地よくフィットします。 「にきび」が付いている裏表紙は、手触りが快適で、滑りません。 フロントパネルの3つのボタンはすべてタッチセンシティブで、個人的にはあまり好きではありません。 電源ボタンはケースの上部にあり、押して画面をオンにするのは不便です。 さらに、どういうわけか非常にはっきりと努力せずに押されています-これはおそらく特定のデバイスの機能です。 フロントパネルのボタンのバックライトがあまり好きではありませんでした。すべてのアイコンが均等に照らされているわけではありません。 一方、多くの予算のAndroidスマートフォンでは、フロントパネルのキーがまったく強調表示されていません。



タッチスクリーンはうまく機能し、システムは非常に反応が良いです。 しかし、それでもクリックが機能しない場合があります。 この画面に適応しなかったか、実際、クリックが記録されない場合があります。 マルチタッチはうまく機能します。



新しいYummy Duoモデルでは、SIMカードスロットははるかに便利に配置されています。 それらを引き出すのは簡単です。 確かに、このためにはバッテリーを取り外す必要があります。

「中国では安い」というコメント



はい、中国では安くなります。 なぜなら、人が中国のオンラインストアで郵便で配送して購入する場合、実際には中国で購入するからです。 価格には関税と付加価値税は含まれていません。



また、そのような購入は保証およびサポートされていません。



つまり 自分で購入でき、多くの人が購入できます。 しかし、大規模な小売チェーンでの取引は機能しません-これは、私たちの法律の観点から、実際には密輸です。



会社が中国からデバイスを正式に輸入し、それらの関税を正式に清算し、州に税金と関税を支払い、商品を大規模な小売チェーンに正式に入れた場合（これらはいわゆる「ホワイトデバイス」）、価格は次のとおりです。



中国の価格= 100ドル（例）。

商品が国境を越えるとすぐに26％（関税、付加価値税-18％、配送、保険など）=すでに126ドル。

ディストリビューターは、利息+最低10％= 125ドル* 1.1 = 139ドルを受け取ります。

小売チェーンは、利息+最低30％= 139ドル* 1.3 = 181ドルです。



つまり、小売ネットワークでは、 どの製品も中国よりも80％〜100％（ほぼ2倍）高価です。 これは売り手の気まぐれではなく、ロシアの合法的なホワイトエレクトロニクス市場の現実です。
