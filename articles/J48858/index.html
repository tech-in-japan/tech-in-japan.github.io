電話-Mac用の新しいSIPクライアント

Mac-電話用のVoIPプログラムの紹介。 これはObjective-C / Cocoaで書かれたネイティブプログラムです。 私はほとんどのソフトフォンがひどいインターフェースを持っているのが好きではなく、それ以外に視覚的に鉄の電話のように見えることを試みます。 クリックする場所を理解するために、普通のユーザーはコンピューター画面に携帯電話を描く必要がないように思えます。 これが何が起こったのかです。



名前または電話番号を入力すると、システム全体のアドレス帳が検索されます（ところで、通常の方法で複数のコンピューターとiPhoneの間で同期できます）。 検索結果が表示されます。





電話。





設定













サポートされているコーデック：

  • G.711ファミリー（PCMA、PCMU）;
  • Speex / 8000（狭帯域）;
  • Speex / 16000（広帯域）;
  • Speex / 32000（超広帯域）;
  • iLBC;
  • GSM
  • L16。


DTMF送信機能、ロシア語および英語のインターフェースが利用可能です。 システム要件：Intelプラットフォーム上のMac OS X 10.5 Leopard。
