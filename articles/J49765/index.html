米国政府の変化

オバマ氏の革新は、米国政府の発足だけでは終わりませんでした。

ユーザーエージェント：*

不許可：/ cgi-bin

不許可：/検索

許可しない：/query.html

禁止：/ omb /検索

許可しない：/omb/query.html

許可しない：/ expectmore / search

許可しない：/expectmore/query.html

不許可：/結果/検索

許可しない：/results/query.html

許可しない：/ earmarks / search

許可しない：/earmarks/query.html

不許可：/ヘルプ

禁止：/ 360pics / text

不許可：/ 911 / 911day / text

不許可：/ 911 /ヒーロー/テキスト

翌日、このファイルは次のようになり始めました。
