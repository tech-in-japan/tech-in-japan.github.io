Windows Vista / 7上のGlazing Opera 10.50

Opera 10.50 for Windowsの主な革新の1つは、Aero Glassガラスインターフェイスを含むAeroエフェクトの完全サポートです。 ただし、これはすべてのブラウザ要素に広がっているわけではありません。 Operaデザイナーは、ガラスとクラシックグラフィックスのバランスをうまく保つことができましたが、お気に入りのプログラムのインターフェイスを完全にガラスにしたい人もいます。









会う：Z1 -AV69によって作成されたZ1-Glass 。



実際に、すべてのツールバーをs薬で覆うZ1-Glassをスキンします。
my.opera.com/community/customize/skins/info/?id=9281

インストール：

ダウンロード時に自動的にインストールされ、インストール後、ブラウザ設定で他のスキンと切り替えることができます。



スキンは、アドレスバーのいくつかのオプションで使用できます。
my.opera.com/Z1-AV69/blog/2010/02/11/3-different-addressfield-options-in-z1-glass-1-52

インストール：

1. Z1-Glassスキン（標準の場所： ％appdata％\ Opera \ Opera \ skin \ ）でアーカイブを見つけ、そこからskin.iniファイルを抽出します。これは少し編集する必要があります。

2. 20行目から、英語の説明があります。 すべてが明確であれば、それを実行し、すぐにステップ4に進みます。

3. 35行目から、 [Dropdown Addressfield Skin]セクションがあります。 パネルの背景画像を定義する3つの異なる行が含まれています。 オプションを選択するには、それぞれ2行をコメントアウトし、1つ（必要な）を残す必要があります。

オプション：

  • 標準：



    Tile Center = border/addressfield.png

    #Tile Center= border/addressfield_alternate.png

    #Tile Center= border/addressfield_classic.png



  • 組み合わせ：



    #Tile Center = border/addressfield.png

    Tile Center = border/addressfield_alternate.png

    #Tile Center= border/addressfield_classic.png



    組み合わせたバージョンの場合、代替のアドレスパネル構成も必要です。 完成したパネルをダウンロードして、 ％appdata％\ Opera \ Opera \ Operabar \ toolbarに保存し、オペラ設定で有効にできます。
  • クラシック：



    #Tile Center= border/addressfield.png

    #Tile Center= border/addressfield_alternate.png

    Tile Center= border/addressfield_classic.png



4. skin.iniへの変更を保存し、アーカイブに戻します。

5.ブラウザを再起動します。



さらに、スピードダイヤルとも呼ばれるグレーズとスピードダイヤルを使用できます。
my.opera.com/Z1-AV69/blog/2010/02/04/transparent-background-when-viewing-pictures-with-z1-glass

インストール：

1. スタイルファイルをダウンロードします 。

2. オペラでは：config＃Image％20Styleは標準ではなく新しいファイルを指定します。

3.スピードダイヤル設定で、背景画像の使用を無効にします。

4.ブラウザを再起動します。

Habrauser alergiは、スピードダイヤルグレージングに特別なアクション（スキンのインストール以外）は必要ないと示唆しています。

最後にそれがうまくいかなかったという事実は、おそらく手の曲がりを相殺するでしょう;）

変更されたimage.cssは、他のいくつかの目的を対象としています。
