私は独創的ではなく、劇場はハンガーから始まると言います。 このメーカーでは、Acer Aspire S3のパッケージングが非常に一般的です。 ボックスのデザインでは、緑とグレーの色の組み合わせが優先されます。 箱に収められたウルトラブックの美しい写真に加えて、デバイスの詳細な仕様が記載されたステッカーを見つけることができます。 内部では、すべてがややランダムに配置されています。 ウルトラブック自体が一番上にあり、薄い厚紙で覆われています。 しかし、その下には段ボールの洞窟全体があり、空きスペースがあり、最小限の便利な追加があります。 すべてが安全に梱包されているため、輸送中の損傷の可能性は非常に低くなっています。











キット（充電器について言えば）は小さいです。 記憶に加えて、ウルトラブックと国際保証についての情報を掲載したパンフレットが数冊しかありません。 ソフトウェアディスクはありません;システムリカバリ用にハードドライブに隠しパーティションが用意されています。 USB-Ethernetアダプターやウルトラブックケースなどのボーナスは提供されません。 私はすぐにウルトラブックのすべてのメーカーに希望を表明したいと思います-価格は推奨されるIntelから非常に遠いので、キットに快適に追加することは約1000ユーロの価値のあるデバイスにとって不必要ではありません。

Acer Aspire S3キーボードには、いくつかの特定の機能があります。 第一に、鍵盤は別々です。ボード全体が手元にないため、楽しくて便利です。 第二に、Enterキーは何らかの理由でスラッシュキーとスタイル的に組み合わされていますが、これは作業には影響しません。 第三に、カーソルブロックは小さく、非常に便利ではありません。この問題はマウスの存在によって簡単に解決できますが、誰かが頻繁にPageUpキーとPageDownキーを使用すると、中毒プロセスは速くなりません。 キーはグレーで、ケースの色、それらの銘は白いペンキで塗られており、よく読めます。 バックライトなし。

システムに関する一般的な情報は、レビューの冒頭に記載されていますが、現在では最も重要なコンポーネントに関する詳細な情報が記載されています。







ウルトラブックAcer Aspire S3はIntel Core i5-2467Mプロセッサーを使用しています。 Intel Core iプロセッサの第2世代であるSandy Bridgeに属します。 ハイパースレッディングテクノロジーをサポートする2つの物理コアは、4つのデータストリームを同時に処理できます。 プロセッサの公称クロック速度は1.6 GHzです。 アイドルまたは低負荷では、この数値は800 MHzに減少し、TurboBoostモードでは、最大周波数は2.3 GHzに上昇します。 プロセッサの熱放散はわずか17ワットです。















Intel HD Graphics 3000グラフィックスカードは、プロセッサの一部です。 グラフィックパーツの周波数は、負荷とグラフィックリソースを使用するアプリケーションに応じて、350 MHzから1.15 GHzまで変化します。 ビデオメモリは最大1024 MBのRAMを使用します。







Acer Aspire S3のストレージシステムは、容量が320 GBの従来のHitachi SATA II HDDと容量が20 GBのキャッシングSSDの2つの部分で構成されています。 SSDはデータの記録には使用できず、ウルトラブックが休止モードのときに情報をキャッシュし、データを保存するために使用されます。 このソリューションにより、Intelの状態を観察することができます-ラップトップの蓋を開くと、ほぼ即座に目覚め、作動する準備が整います。















マザーボードのウルトラブックにハンダ付けされたRAM標準DDR3は、アップグレードでき、交換できませんが、シングルチャンネルモードで動作します。 すぐに目に見える欠点は何ですか：1つのメモリチップが飛び出しました-マザーボード全体を交換し、作業をスピードアップする能力の欠如。 デュアルチャネルモードでは、チップセットはメモリからの読み取り/書き込み結果を約50％高くします。











Windows 7がインストールされているデバイスのパフォーマンスを測定する最も簡単な方法ですが、最も正確ではありませんが、組み込みのパフォーマンス評価機能を使用することです。 可能な最大スコアは7.9で、Acer Aspire S3 Ultrabookは4.5ポイントを獲得します。 最も素晴らしいパフォーマンスではありません。 Windows 7システムの弱点は、ビデオカードです。

ハードドライブの速度は、通常のラップトップドライブのパフォーマンスに対応します。SSDは、上で書いたように、情報の保存や読み取り/書き込み操作には直接関与しません。

統合グラフィックシステムのパフォーマンスは2年前の個別のグラフィックソリューションのレベルにありますが、それでも、平均的な設定で最新のゲームをプレイできます。

RAMは十分な量で表されますが、単一チャネルシステムのため、パフォーマンスのかなりの部分が失われます。

Futuremarkの新しいテスト-PowerMarkで測定しようとしたバッテリー寿命。 テスト操作スキームは非常に単純です-バッテリーを100％充電し、使用シナリオを設定し、外部電源をオフにし、バッテリーがなくなるまで待ちます。 バッテリー充電の15％が残っていると、選択したシナリオに典型的なタスクを実行するときに、ランタイムにテストが表示されます。 プログラムは、バランスの取れたスクリプト、生産性、エンターテイメントを保護しています。 バランスを選択すると、バッテリーは2時間29分続きました。 大したことではありませんが、新しいテストは100％信頼できるものではないため、引き続きレビューに使用し、結果の妥当性の理解は間違いなく得られます。
