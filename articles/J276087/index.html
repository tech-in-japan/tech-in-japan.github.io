OracleはJavaブラウザプラグインのサポートを終了します

オラクルは、多くのユーザーが熱心に受け入れることができる変更を発表しました 。 このソフトウェアの将来のバージョン9から、Webブラウザ用のJavaプラグインのリリースを中止することです。 プラグインは、Java Jだけでなく、Oracle JDKやJREなどのソフトウェアパッケージから削除されます。 アプレットの作成者とプラグインのコンテンツは、 Java Web Startテクノロジーに切り替えるよう招待されます 。







Google ChromeやMicrosoft Edgeなどの人気のあるWebブラウザーは、Javaプレーヤーの操作に必要なNPAPIインターフェイスの使用を既に放棄していることに注意してください。 今年もMozilla Firefoxに同じステップが提供されます。 Javaコンテンツを再生するためのプラグインは、以前は攻撃者がマルウェアをリモートでインストールするために使用していた多数の脆弱性の存在について、セキュリティ研究者から繰り返し批判されてきました。



製品のプラグインサポートを制限および削減しようとする現代のブラウザーベンダーにより、Javaブラウザープラグインに依存するアプリケーションの開発者は、Javaアプレット（ブラウザープラグインに依存する）からプラグインのないJavaへの移行などの代替オプションを検討する必要がありますWeb Startテクノロジー。 Oracleは、JDK 9でJavaブラウザプラグインを廃止する予定です。このテクノロジーは、将来のJava SEリリースでOracle JDKおよびJREから削除されます。


同社は、アプレット開発www.oracle.com/technetwork/java/javase/migratingfromapplets-2872444.pdfから移行するメカニズムを強調した特別な文書を公開しました。
