100ワットのUSB 3.0は独自の充電器に終止符を打ちます

Intel、Hewlett-Packard、Microsoft、ST-Erricsonの代表者を含むコンソーシアムは、USBを介した電力伝送の新しい仕様の準備ができたことを発表しました。これにより、接続あたり100ワットを転送できます。



新しい標準は、ラップトップや小型デスクトップコンピューター、外付けハードドライブ、プリンターなどの多数のデバイスの充電と電力を大幅に簡素化し、最大5 Gb / sの転送機能も提供します。 このグループは、新しい規格のケーブルとコネクタが独自の充電器の廃止に役立つことを望んでいます。これは、ユーザーの利便性だけでなく、家庭ごみの量の削減も意味します。



この標準は、USB 2.0および3.0への追加であり、以前のバージョンとの完全な下位互換性があります。 USB 3.0仕様は2008年11月に初めてリリースされ、それ以来、開発およびプロモーションチームは、エンドユーザーにとって重要な機能を考慮して、標準の改善に継続的に取り組んでいます。







新しい標準によれば、ケーブルはかなり多くの電力を伝送します。 比較のために、USB 2.0は最大2.5ワット、USB 3.0は最大3.5ワットを転送します。 USB Power Deliveryは、ケーブルの方向を変更せずに切り替えられた電源を可能にし、6つの電源モードがあり、その間にDevice Policy Managerの制御下で「スマート」スイッチがあります。 バッテリー充電1.2仕様もサポートされています。



プロファイル1は2 Aから5 V、プロファイル2-2A / 5Vまたは1.5A / 12V、プロファイル3-2A / 5Vまたは3A / 12V、プロファイル4-3A / 20Vを送信でき、プロファイル5は100ワットの上限に達します-5A / 12Vまたは5A / 20V。 microUSBコネクタを使用すると、転送できるのは3 A以下です。プロファイル0は予約されています。







送電には、特別な認定USB電源供給ケーブルが必要です。 仕様には、ケーブルの機能と接続構成を確認するメカニズムが含まれています。名前のないケーブルを介して100ワットの電力を伝送することは、かなり危険で火災の危険があるためです。



新しい標準を使用するための多くのシナリオを想像することができます：家庭用電気ネットワークに接続されたラップトップは、別のデバイスのバッテリーを充電します。 外部電源をオフにすると、ラップトップは外出先でより穏やかなモードに移行したり、外部デバイスへの給電を完全に停止したり、その逆にエネルギーを取り始めたりできます。



USB 3.0プロモーショングループのメンバーであるBrad Sununderは、彼が提供する仕様に統一されたユニバーサルチャージャーを導入することで、年間の廃棄電子機器の削減に貢献したいと考えています（ Steve Wozniakのショルダーバッグを思い出してください）。 ただし、独自の基準にもっと傾倒している企業が新しい化合物の仕様を実装するかどうかは疑問の余地があります。
