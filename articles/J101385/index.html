マフィアIIのデモ版



カルトゲームの待望の継続のデモ版は、日の目を見ました。



デモにはBuzzsawレベルが含まれています。このレベルでは、mafiosi Vito Scarlettiと彼のパートナーが地元の犯罪当局を殺すために出発します。 ゲームは20世紀の40〜50年代に行われます。



ゲームのフルバージョンは、PC、Xbox360、およびPS3で8月27日にリリースされます。



さて、 Steamからダウンロードすることで、今すぐPCのデモ版を試すことができます。



ゲームの最小システム要件：



: Windows XP (SP2) / Windows Vista / Windows 7;

: Pentium D 3Ghz / AMD Athlon 64 X2 3600+;

: 1.5 GB;

: Nvidia GeForce 8600 / ATI HD2600 Pro;
