X Window Systemに置き換わるものは何ですか？

昨年の重要なLinuxイベントの1つは、 X Window Systemを置き換えることを目的としたWaylandディスプレイサーバーに基づくGnome 3.22グラフィカル環境を備えた25番目のFedoraのリリースでした。 しかし、なぜ、そんなに長い年月の後に、そのような必要性が生じたのでしょうか？

「ヒューストン、問題があります。」 それは木星に私たちを吹き飛ばします。

「再びxorg.confをいじくり回しましたか？」

-はい。 ヒューストン、最後の3日間はひげが何とか成長しました。

  • X Window System -1984年の夏にMITの深部で作成されたX Window System 。プロトコルはXsで定義されていますが、技術的にはすべてですが、プロトコルにすべてを縮小することは正当化されません。 実際、プロトコル自体はウィンドウを描画しないため、今日のUnix / Linuxシステムでグラフィックを制御するものはすべてプロトコル以上のものです。
  • X11-1987年にさまざまな形式でリリースされた Xの11番目のバージョンは、今日まで残っています。 特に、 XFree86がそこから芽生え、そこから真に無料のX.Org Serverが登場し、GPLライセンスがあるはずです。 このテキストでは、X11はプロトコルを示しますが、Xとは対照的に、コンポーネントの全体的な組み合わせ（ソフトウェア全般）を意味します。
  • X.Org Server -Xのサーバー部分の無料の標準的な実装。現在はLinux用のXサーバーです。
