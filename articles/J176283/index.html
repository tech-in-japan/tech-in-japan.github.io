HTC 8S-新しい感覚

Windows Phone 8のNTSデバイスのラインナップは多数とは言えませんが、その代表は両方とも、そのユニークな外観と優れたデザインで際立っており、名誉ある賞を受賞しています。 今日は、新しいモデルであるHTC Windows Phone 8Sを見ていきます。HTCWindows Phone 8Sは、新しいモバイルOSの世界への優れたエントリチケットとして機能します。



NTSデバイスWindows Phoneは、システムを念頭に置いて設計されており、インスピレーションの源は、親しい人とのやり取りに基づいたシンプルでモダンなインターフェイスでした。 最終的に、電話はこのために特別に設計されており、スマートフォンはプロセスをより便利にするだけです。 そこに何がありますか、彼を見ることさえすてきです！









HTC 8Sは実用的なマットプラスチックで作られており、汚れや小さな傷を完全に防ぎます。 このデバイスは、ほとんどすべての条件下で新品のように見え、底部に取り外し可能な部品が存在するにもかかわらず、ビルド品質に問題はありません。







その隣にはBeats Audioのロゴがあります。これは、ヘッドフォンの音をより飽和させるアルゴリズムです。 さらに、スピーカーは近所にあり、後壁には5メガピクセルのカメラとLEDフラッシュがあります。 デバイスの前面は、予想どおり、スーパーLCDテクノロジーを使用して作られた4インチディスプレイで占められています。 コントロールボタンの構成はWP8の標準であるため、驚くことはありません。







最新の品揃えの条件では、8Sは非常に小さく見えますが、その控えめな寸法（120.5 x 63 x 10.3 mm）は、コンパクトでありながら機能的なスマートフォンを長い間見ているすべての人を間違いなく満足させます。 8Sの重量もわずか-わずか113グラムなので、ジーンズだけでなくシャツのポケットも引き出せません。 寸法を比較するために、現在のスマートフォンといくつかの比較写真を撮りました。







コントロールキーの標準セットは、周辺に配置されています：電源オン、2ポジションカメラボタン、ボリュームロッカー。 デバイスを徹底的に検査すると、「SIMカードを実際に挿入する場所はどこですか？」という質問が思わず尋ねられます。これはキャッチーな下部インサートにあり、爪でカバーをそっとこじ開けることで簡単に分離できます。 その下に、microSIMおよびmicroSDカードのスロットが隠されています。 要素は取り外し可能であるという事実にもかかわらず、その背後にはきしみやバックラッシュは見られませんでした。







内部メモリは4 GB、最大拡張カードは32 GBであるため、写真、ビデオ、音楽を保存するための空き容量を心配することができます。 同時に、RAMは512 MBであり、デバイスはQualcomm Snapdragon S4プラットフォームで動作します。 WP8のデバイスは、Androidのデバイスよりもリソースに対する要求が大幅に少ないため、このキットは迅速で快適な作業に十分です。







私たちはそれが機能するとは考えず、Marketplaceで利用可能なアプリケーションを使用していくつかのテストを実施しました。 それらは、人気のあるAnTuTu Benchmarkパッケージおよび無料のWP Benchユーティリティになりました。







それで、この小さな、しかし非常に快適なデバイスは何ができますか？ もちろん、モバイルOffice 2013、SkyDriveクラウドサービスと他の機能との統合など、ブランド化されたすべての「チップ」とWindows Phoneサービスが提供されます。 プラットフォームの新しいアプリケーションはほぼ毎日表示され、エコシステム内の標準画面解像度（800x480ピクセル）は、潜在的な互換性の問題を無効にします。







デバイスの写真側はメインカメラで表され、最大2592x1944ピクセルの解像度で写真を撮ることができます。 必要に応じて、フラッシュを使用できます。すべての画像はメモリカードに保存されます。 写真を撮るという点では、オン/オフカメラキーは非常に役立ちます。これは、Microsoftスマートフォンの重要な機能の1つです。 さまざまな条件で複数の写真を撮影しました。各写真をクリックできます。























写真とビデオの場合、主な効果（b / w、ネガ、セピア）が利用可能で、画像の明るさ、コントラスト、彩度の調整もできます。 さらに、顔検出はデフォルトでオンになっているため、ポートレート撮影に便利です。
