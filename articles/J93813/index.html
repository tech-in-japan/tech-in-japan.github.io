Django 1.2リリース

起こった、仲間！ リリース日がほぼ2か月遅れ、絶えず延期されていたにもかかわらず、本日、2010年5月17日、壮大なフレームワークの最終バージョン1.2がリリースされました。



最も顕著な変更が発生したことを簡単に思い出し、より詳細な情報へのリンクを提供します。





  • 1つのプロジェクトのフレームワーク内で複数のデータベースを操作するためのサポートが追加されました（ Habrに関する記事 ）
  • キャッシングテンプレート（ Django Adventの記事）
  • テンプレートタグ{％if％}は比較演算子を理解することを学びました（ Django Adventの記事、Habréの翻訳 ）
  • 管理領域では、彼らはJS自転車を離れ、ついにjQueryの使用を開始しました（Habrの記事 ）
  • CSRFメカニズムの更新（ Django Adventに関する記事、Habréに関する翻訳 ）
  • ユーザー通知システムが完全に書き直されました（ Django Adventの記事、Habréの翻訳 ）
  • 日付、時刻、数値などの形式のサポートが追加されました。 ロケールごと


もちろん、それだけではありません。 変更点の完全なリストは、いつものように、 リリースノートにあります 。 よく知られているリンクがいくつかありますが、決して忘れてはならないものです。

  • ほとんどの変更は、 Django Adventの一般的な科学形式で読むことができます。
  • ジャンゴ開発者のフレンドリーなロシア語コミュニティは、 ジャンゴフォーラムでイヴァンサガラエフと通信します （ イサガラエフもハブレにいます）
