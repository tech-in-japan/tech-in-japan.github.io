Pure Storage：Hour // mが到着しました

こんにちは、Habr！ オールフラッシュ市場を監視している多くの人は、Pure Storageについて聞いています。 それらはコメント、ベンダー比較、評価で言及されました。 1つの問題-ロシアでは純粋ではありませんでした。



そして今、私たちはあなたに次のことを伝える準備ができています：Pure Storageが私たちに到達し、ロシア市場に参入しようとしています！









小さな紹介



JetのストレージとIBSの領域を開発しています。 私の活動分野の1つは、ITインフラストラクチャの開発のための新しいベクターの検索と適応です。 これらのベクトルの中には、フラッシュ技術への積極的な移行と、データの保存と処理における関連する変更があります。



今では、高性能コンピューティングにメカニカルディスクを使用することは過去のものであることを誰もが理解しています。 フラッシュの時代が来ています。 しかし、ご覧のとおり、今日のオールフラッシュは、ほとんどの場合、従来のディスクアレイを搭載した単なるSSDです。 私たちにとってさらに興味深いのは、20年前に技術の負担を抱えていない、このビジネスのスタートアップであるまったく新しいプレーヤーです。



現在、Pure Storageには3つのストレージラインがあります。



  • FlashArray // m-要求の厳しいアプリケーションと従来のシステムの両方に適した汎用AFアレイのライン（これまでのところ、ロシアでは// mラインのみが利用可能になります）。
  • FlashArray // x-最もパフォーマンスが重要なアプリケーション向けのNVMeフラッシュアレイ。
  • FlashBladeは、水平方向にスケーラブルなファイルおよびオブジェクトAF配列です。


Pure機能とは何ですか？



5年以上前にPure Storageに気付きました。 それから皮肉な笑み以外に、これらの人は引き起こしませんでした。 Linuxを搭載したSupermicroの鉄の断片、コードやサードパーティのアプリケーションを使用したシャーマニズムダンス、美しいアイデアであり、最良の実装ではありません。 しかし、時が経ち、すべてが変化し、Pure Storageはアメリカとヨーロッパを獲得し始めた非常に興味深い製品を作成することができました。



なぜPureは私たちにとってとても興味深いのですか？ まず、これはもともとFlashの下で作成されたアーキテクチャです。 彼らは長年にわたって、機械的なディスクではなく、フラッシュのすべてのニュアンスを考慮した製品を作っています。 たとえば、圧縮と重複排除は常に機能します。オフにすることはできず、コード自体、データを操作するメカニズムに接続されています。



第二に、簡単かつ迅速に動作します。 RAIDグループ、 LDEV 、プール、階層化、パーティション化、スペアディスクを覚えていますか？ これは何もありません。 アレイを立ち上げ-容量とパフォーマンスを得ました。 ドライブの追加-より多くの容量とパフォーマンスが得られました。 何も考える必要はありません。すべてがr-a-b-o-t-a-e-tです。



さらに、アーキテクチャ自体も興味深いものです。 一方では、これは古典的なデュアルコントローラーアレイであり、他方では、データがRAMに書き込むためにキャッシュされず、すべてが外部NVMeモジュールに保存されます。 したがって、外出先でコントローラを取り出し、他のコントローラに変更し、ライン内、ライン間でアップグレードできます-すべてダウンタイムなしで。 また、パフォーマンスを損なうことなく、システムは通常モードで50％を超えるロードを許可しません。



そして最もおいしいのは、販売とサポートの点で純粋なビジネスモデルの違いです。



常緑樹



プログラムは、常にアレイを常に更新することを目的とした「常に最新のストレージ」です。



  • サービス契約の枠組み内で3年ごとに、Pure Storageは無料で顧客の現在のモデルとコントローラーを交換します（サポート中は常に新しいコントローラーがあります）。
  • 新世代のアレイでは、ディスクの交換や移行は不要です。
  • ドライブの無期限保証（サポートがある場合）;
  • 顧客が容量を4倍に増やすと、古いディスクは新しいものに無料で変更されます。
  • すべてのライセンスが含まれます（新しい機能がリリースされた場合でも）。


したがって、連中は次のように述べています。「アレイを一度購入すれば、再度購入する必要はありません。」 明らかに、彼らはサポートとキャパシティ拡張で稼ぐでしょう。 アプローチは興味深いです、あなたは何も言わないでしょう。



正しいサイズの保証。 容量保証



重複排除および圧縮技術を使用して容量の節約を保証するプログラム。 構成を計算する段階で、アレイに保存するデータを指定します（たとえば、VMware 30TB、Oracle 15TB）。 次に、インテグレーターとベンダーは、提示されたデータの効率係数が書き込まれる提案（5：1など）を準備します。 この場合、Pure Storageは販売前にこの比率を法的に保証します。 実際のものと一致しない場合は、追加のディスクが持ち込まれます。

多くのメーカーが同様のものを提供しているようですが、多くの場合、それは単なるマーケティングです。 カルトのストレージの有効性を純粋に確立します。法的保証を提供し、提案書に綴ります。 それが実際にどうなるか見てみましょう。



Pure1



アレイを購入すると、クラウド監視サービスPure1に無料でアクセスできます。 インシデント発生時にメーカーによってケースが自動的に開かれるプロアクティブなサービスだけでなく、クラウドからの予測分析も含まれます。 多数の診断メトリックが、インストールされたベース全体からPure1 Meta機械学習システムに30秒間隔で届きます。これにより、問題が実際に発生する前に防ぐことができます。 製造業者によると、このテクノロジーは、優先度1の170以上のケースが開かれる前に防止するのに役立ちました。









1つの製品に集められた上記の機能により、この配列をまったく異なる方法で見ることができます。 Pureは、ストレージ市場で確立された現状に違反し、ディスクアレイに対する態度を逆転させ、Flashテクノロジーへの移行を促進することができます。



ロシアのピュアはどこですか？



唯一の問題は暗号化です。 長い間、Pure Storageは暗号化されたシステムのインポートの制限のためにロシア市場に参入できませんでした。 現時点では、ロシアと中国向けにOSの特別なバージョンがリリースされており、FSB通知はすでに受信されています。 これで、Pureが市場に参入する準備が整いました。8月に最初の出荷が予定​​されています。



JetはPure Storageのトップサービスパートナーです。 現在、独自のデモスタンドの取得に積極的に取り組んでおり、7月を目指しています（ロシアのベンダーのスタンドよりもさらに早くなる可能性が非常に高いかもしれません）。 その後、私たちはあなたに本当のGeek Pornを提供する準備ができています-最も急速に成長しているFlashスタートアップの配列をテストする完全なレポートです。 彼らが言うように、切り替えないでください！ ;）



テスト後、ビジネスのノベルティを体験することもできます。すべてのお客様がデモスタンドを利用できます。 ニュースをフォローしてください。
