Windows仮想サーバー：新機能





過去数か月にわたって、Windowsベースの仮想サーバーレンタルサービスを改善するためにかなりの作業を行ってきました。 仮想Windowsサーバーのユーザーは、次のイノベーションを利用できます。

  • コントロールパネルが大幅に改善され、新しい機能が追加されました。
  • すべてのWindowsベースの仮想サーバーには、vhdx形式の仮想ディスクが装備されています。
  • Windows Server 2012 R2 Standardの新しいバージョンをインストールできます。
  • 特定のタスク向けのソフトウェアがプリインストールされたOCテンプレートを選択できるようになりました。


この記事では、すべての変更と新機能について詳しく説明します。



コントロールパネルの変更



コントロールパネルは次のようになります。







現在では、サーバーリソースの現在の使用状況とその連続操作の時間（稼働時間）に関する情報が表示されます。 管理者パスワードは、パネルから直接変更できます（「パスワードの変更」ボタン）。



サーバー画面のスクリーンショットは、パネルの右下隅に表示されます。



Vhdxサポート



すべての仮想マシンにvhdx形式の仮想ハードディスクが搭載され、多くの利点が得られます。



仮想ハードドライブを使用すると、アクティブなメタデータの更新中に電源障害が発生した場合に読み取り不能なファイルを取得するリスクが伴います。 vhdx形式は、ジャーナリングされているため、示された問題を解決します。すべてのメタデータの変更は、最初にジャーナルの静的ゾーンに書き込まれ、次にメタデータテーブルに書き込まれます。 メタデータテーブルの更新中に電源障害が発生すると、不完全なトランザクションに関する情報がログに保存され、それに基づいて完全なディスクフォーマットを返すことができます。



ダイナミックドライブと差動ドライブのブロックサイズが4 KBに増加し、生産性が向上しました。



さらに、vhdx形式では、最大64 TBのサイズの仮想ディスクを作成できます。



Windows Server 2012 R2標準



Windows Server 2012 R2 Standardエディションは2013年10月17日にリリースされました。 前のエディションと比較して、大幅に拡張され、補足されました。 主要な革新には以下が含まれます。



  • 第2世代の仮想マシン（Gen2）は、ゲストOSの読み込みを高速化し、読み込み時間を短縮します。 第2世代の仮想マシンは、SCSI、SCSI DVD、またはネットワークを介して仮想ハードドライブから起動でき、高速起動も可能です。
  • これで、仮想マシンをオンラインでエクスポートできます。
  • Hyper V R2は、個別の仮想マシンに割り当てられる入出力操作（IOPS）の最小数と最大数を構成する機能を追加します。
  • リモートデスクトップの一連の機能を大幅に拡張しました。 接続パフォーマンスの向上。WindowsServer 2012と比較してトラフィックを50％削減します。これは、より優れた圧縮を提供する新しいコーデックを使用することで実現されます。
  • また、別のユーザーのアクティブセッションをリモートで制御または表示する機能が追加されました。
  • GPOは、IPv6のサポートを拡張し、イベントログをアップグレードし、ポリシーキャッシュを追加しました。
  • Powershell機能が拡張され、DNS、DHCP、Hyper-V、ファイルサービスなどを管理するための新しいコマンドレット。
  • ここで、SMBトラフィックの帯域幅をより詳細に制限できます 。
