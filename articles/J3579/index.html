Mozilla Firefox 2.0.0.2

Mozilla Firefox 2.0.0.2の公式バージョンが公式サーバーに登場しました。 公式リリースによると、新しいバージョンはブラウザーとWindows Vistaの互換性を大幅に向上させ、SSLv2ライブラリのバッファーオーバーフロー、キャッシュからデータを受信する可能性、サイトアドレスの偽装、またはブラウザーの一時停止などのいくつかのセキュリティ問題も修正しました（ 詳細 ）。



Mozilla Firefox 2.0.0.2をダウンロードします。

-Firefoxセットアップ2.0.0.2.exe 、 ミラー （5.73Mb、ロシア語版）

-Firefox Setup 2.0.0.2.exe 、 ミラー （6.33Mb、ロシア語版）



開発者はまだ1.5.0.xブランチのサポートを中止していないため、Mozilla Firefoxバージョン1.5.0.10も登場しました。

-Firefox Setup 1.5.0.10.exe （4.92Mb、ロシア語版）

-Firefox Setup 1.5.0.10.exe （4.94Mb、ロシア語版）



数日以内に、新しいバージョンが組み込みの更新システムを介して自動的に配布されるようになります。そのため、インストールファイルをすぐにダウンロードするのは、コンピューターに新しいものがすべて揃っていることに満足しており、トラフィックを気にしない人だけに意味があります（通常、組み込みシステムによる更新には1メガバイト未満しかかかりません）。
