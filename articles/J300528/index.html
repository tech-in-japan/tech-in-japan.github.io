自発的なゲームデザイン

「歴史の陰謀は、私が海の真ん中の遠い島にいたときでさえ、訓練をあきらめなかった方法です。」スペルと句読点は著作権です。 背景は、Universariumブログにあります。







「新年の前にWGからユニバーサリウムまでのゲームデザインコースに申し込んだのですが、どこからも離れることはなく、何も前兆はありませんでした。 幼少の頃からセーリングと海賊の歴史を好んでいた生涯教育のサポーターであり、やがてベアボートスキッパーIYTのささやかな称号を受け取り、5,000 nmの傷を負いました。 そして今、2016年1月25日の夜の同じ「出航」の知人を通して、私は珍しい申し出を受け取りました-カナリア諸島からカーボベルデへと戻って大西洋を渡る帆走カタマランの移行。 実際の遠征は経験豊富なスキッパーのみが対象で、夜勤と食事を用意しています。 海洋の詳細に入ることなく、このルートは少しおかしいので、拒否することはできませんでした。







彼は1日で決断を下し、1月27日に飛び立ちました。また、2つのプロジェクトの制作について遠隔交渉を続けました。 マーケティング、リテンション、ゲームのビジネスパフォーマンスの改善-これは私が過去数年間行ってきたことであり、数千万の「ソーシャルネットワーク」が友人の推薦で私を見つけました。 出発の準備をしており、乗組員が集まるのを待っている間に、談合は、談合での風のうなり声の下で、空港の建物とテネリフェ島のマリーナで活発に行われました。 旅行の私の計画を知ったクライアントは少し緊張していましたが、海で私にお金を求めないことに気付いたとき、私は彼らに船上でいくつかの基本的な統計を出荷するように頼み、彼らは落ち着きました。 彼らが私の帰国の1ヶ月前に待つことに決めたとき、私はゲームデザインにコースがあるべきであることに気付きました！ 最初の「宿題」を手渡し、Canarから出航しました。



穏やかな5日間、穏やかな風を楽しんで、イルカを戯れ、海の景色を楽しみながら、自分自身について話をする時が来ました。 私は業界の人ですが、ゲームデザインに直接関与したことはありません。 彼らはどこからでもゲームデイブにやって来ます-ファーストフードや建設現場からですが、私にとっては清水のダウンシフトでした。



個々のイベントの言語での職歴：

2009年まで、私は経済教育を受け、お金のために働きました。 セールスマンとして4年、広告とマーケティング、車、アパート、自分のビジネスで5年、しかしどんな方法でも幸福。 彼はたくさんプレーしたが、常に十分な時間はなかった。 彼は完全に落ち込んで、2008年の危機が解決するまで、軽率なものとして独力で運転しました。彼は自分の会社を売却し、ゲームに興味のあるIT会社に参入し、Silkroad Online、Atlantica online、DofusのCISでローカライズと運用の契約に署名しようとしました。 うまくいかず、ソーシャルゲームの現象を研究しながら、彼はタイに住んでエネルギーを得るために去りました。



乱暴に休み、2010年の夏に核融合で、彼はモスクワに戻っていくつかの会社でゲームディズを獲得しました-拒否。 「現実にはこのルーチンを想像できないので、あなたは働くことができません。」 「事業開発のディレクターではなく、あなたへのマーケティング。」 これがそのような「ガラスの床」です。 最終的に、彼は市場の知識とそれが発展している場所を理解することで目標を達成し、最終的に私にはお金は要らないと述べ、「プロジェクトをください。詳しく見る」と言いました。 彼らは、お金、プロジェクトマネージャーの位置、激怒したコンピューターの不具合と3つのゆるいチームボディを備えた「死んだ」ブラウザーを提供しました。 基本的に、私の仕事は投資家にそれがうまくいかなかった理由を説得力をもって伝えることでした。彼らとの経験の恩恵は莫大でした。 しかし、4か月間の闘争と暴力の後、私は貴重な経験を積んで、すでに完全に別の人間になりました。プロセス全体を徹底的に知っていて、最も重要なことはゲームを開発しない方法を知っていました。 何らかの理由で、ネガティブな体験をどのように評価するかわかりませんが、西洋ではすべての成功事例には失敗が含まれます。







プロジェクトを閉じてから、2年間、通常のマーケティングと広告に携わり、宣伝したゲームの1つに投資し、経験を積み、ソーシャルゲームのデジタルパートナーパブリッシングを共同設立しました。 興味深いプロジェクトのある小さなチームを検索し、トラフィックオーバーフローの一般的に受け入れられている慣行に反して、直接的な広告でソーシャルネットワークに公開しました。 最も成功したケースは「魔法のゆりかご」、「グライダーの戦い」、「コントラシティ」です。



2012年、彼は会社の所有者から権力の全負担を引き継ぎ、さらに「2年間」純粋に経営に従事しました。 チームとの接触を確立し、解雇し、仕事を再編成し、シャンパンのために公園でブレインストーミングセッションを開催しました（そのために警察への楽しいドライブがありました）、財政を掘り下げ、チケット売り場のギャップと遅刻の統計に挨拶を送りました。 一般的に、CEO、ほとんど創造性、新しいプロジェクト、興味深い機能、ハイアートスタイル。

2014年の春、インドに開発を移すために行きました。 私は、開発オフィスの開発にしっかりと従事するという提案を受け入れませんでした...完全に終了しました。 私は意識的にアメリカ人がギャップイヤーと呼ぶものを始めました-石を集める時間です。 何かを計画するのではなく、新しいものを探す時、目が見えるところに行く、ルーチンを破る時。 夏にはたくさんの帆、新しい知人、パーティーがあり、冬にはサーフィンと英語を学ぶためにフィリピンに行きました。







現在、私は州下院での強力な能力が必要な自分のプロジェクトを担当しています。 したがって、超動機の一部-私は多くのアイデアを持っています、私はどこでお金を手に入れるか、チームを組み立てる方法、開発プロセス全体を知っています。 GDDの書き方も知っていますが、アイデアの山から本番用のドキュメントを作成する何かが欠けています。 実際、コースのコンセプトの1つを駆使して、コースの完全なドキュメントを作成しました。 プロトタイプはすでに機能しており、それは素晴らしいことです！ :)

学習の動機からカーボベルデに戻る時が来ました。2月6日の朝、カタリマから黄色いカタマラン（サハラ砂漠からの細かい砂を運ぶ風）がまだ夜明けの最初の光線で輝く、カーボベルデ諸島-サントの霧の輪郭に近づきましたアンタンとサンビセンティ。 これらの島々で都市や村のカーニバル行列を見たり、クレーターが小屋や地元の農民の畑がある心地よい谷の壮大な景色を眺めることができる眠っている火山の頂上に登るのにたった2日間しか費やしません。 「火山のように生きる」ことは幸せで繁栄します。そのような土地は非常に豊かな収穫を提供します。 噴火の間。



楽しみの一部を意識して見逃しています-私にとっては、wi-fiを見つけ、ビデオを見て、持ち帰ることができるのは私だけです。 その後、島の南側から迂回し、島を東に向かって出発します。2日間は、私たちが好奇心として撮影されている貧しい漁村のみです。 東から最も遠いサル島には、インターネットを除くすべてのものがあります-途中の1.5メートルのサメ、市内のガソリンスタンド、魚市場、静かなぼろぼろのバーの壮大な港。 新しいビデオを見て、他の学生の作品をチェックします。



その後、海に行き、11日間、向かい風と3メートルの波に逆らいます。 この間ずっと、私たちは海岸に会うことはありません-視界内の単一の船ではありません。 3日間、私たちは癌の熱帯地方を襲撃し、最後にそれを越えようとします。 私たちは、ボートの最大能力に近い嵐をうまく乗り越え、帆、バッテリー、舵を失いますが、危険を冒して、友好的なモーリタニアに燃料を補給することなく、残りの燃料をカナリアに引き寄せることにします。 合計、約2000マイルのタック。 乗組員は飛行機のチケットを心配していたので、私はゲームのデザインについて講義をしていました。

私は次の宿題の前日にインターネットを手に入れましたが、何とか飛行機に乗ることができました。 残念ながら、トップ10に入ることはできませんでしたが、結果によるとトップ50に入りました。 もちろん、これをどこかですでに聞いたり行ったりしているので、いくつかのタスクを行うのは簡単でしたが、条件が許す限り多くのことを学び、すべての責任を持ってトレーニングに取り組みました。 すでに地面に戻ったので、私はまず何よりも望む結果を達成するために仕事に何時間も費やしました。 したがって、あなたが自分自身を教えるこのコースは、あなた自身のモチベーションに比例します。







結論として、コースの適合性に留意し、ゲーム設計の問題における理論の欠如またはWGの無能に対する主張を避けたいと思います。 書誌は理論であり、このコースは業界で働くためにより重要な何かを提供します。 生産パイプライン（フロー）の人々の相互作用を示し、正気なスキルを書くことを教え、他のチームメンバーの希望を考慮し、すぐに結果に導きます-明確なドキュメントとチームワーク。 商業上の成功も忘れないでください。WGでの襲撃は楽しいものです。 私はセッションゲームのファンではありませんでしたが、7〜8年前に、誰も信じていない戦車に関するゲームを信じて見たとき、彼を尊敬しました。



本当の芸術は、他の人たちと同じように風の中を歩くことではなく、反対です！「コースの学生です」 Game Design：Beyond the Game 。
