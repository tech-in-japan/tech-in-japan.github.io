Amazonは、AWS Activateで簡単に開始できるように、最大​​15,000ドルのスタートアップを提供しています

AWS Activateは、Amazon Web Servicesを使用してクラウドで簡単に開始できるようにするためにAmazonが開発した特別なスタートアッププログラムです。



プログラムへの参加は無料です。 私が理解しているように、活動する国や地域に制限はありません。

唯一の条件は、スタートアップがまだAmazonからサポートを受けておらず、AWS無料利用枠を使用していないことです。





パッケージには2つのタイプがあります。

  • セルフスターター-すべての人に
  • ポートフォリオ-アフィリエイトアクセラレータまたは財団と連携するスタートアップ向け。




サポートされている資金のリストはこちらでご覧いただけますaws.amazon.com/activate/portfolio-detail



ご想像のとおり、$ 1000から$ 15`000のサービスを利用するためのローンは、ポートフォリオパッケージでのみ取得できます。 このパッケージでも利用可能：

  • 1〜12か月のビジネスレベルのサポート（ビジネスレベルでのAWSサポート）
  • AWS Technical ProfessionalおよびAWS Essentials初心者向けのトレーニング
  • AWSスタートアップコミュニティナレッジベースへのアクセス
  • プログラムパートナーからの特別オファー




「全員」（セルフスターターパッケージ）の場合、すべてが控えめです。

  • AWS無料利用枠
  • 開発者向けの1か月のサポート（開発者レベルでのAWSサポート）
  • AWSテクニカルプロフェッショナルトレーニング
  • AWSスタートアップコミュニティナレッジベースへのアクセス
  • プログラムパートナーからの特別オファー
