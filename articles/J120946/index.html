Windows MarketplaceモバイルWebサイトとMy Phoneのシャットダウン

本日、MSから上記の件に関する手紙を受け取りました。 彼が驚いたと言うことは何も言わないことです。 最初の考えは「MicrosoftプロファイルはWM7を使用できるようにストアコードを書き換える」、2番目の考えは「MicrosoftはPDA市場の敗北を認識し、ロールアップし始めています」でした。



手紙の最初の段落は次のようになります。

「Windows Mobile 6.xユーザーの皆様、



Microsoftは、Windows Mobile 6.xのMy Phoneをシャットダウンしています。 Windows Marketplaceモバイルサイトも閉鎖されています。 これはあなたに関係するかもしれないので、以下の詳細に注意してください。





私がかつてWMのアクティブユーザーだったため、この手紙が届いた。



すぐに言っておく必要があります。WP7を目にしていません。このシステムについては何も言えません。 しかし、以前のバージョンのWMでは、私は疲れました-神は皆を禁止しました。 私の最初のハンドヘルドはiPaq 3130で 、そのようなカメラがありました。 キットは、優れた状態の盆地として費用がかかります。 どのwinmobileが搭載されているか覚えていませんでしたが、二重の感情がありました。 一方では、カラー表示、計画機能、および辞書に非常に満足しています。 一方で、デバイスを制御する人間工学の完全な欠如に非常に怒っていました。



長年にわたって、状況は1つのイオタを改善していません。 個人用と企業用の両方でiPaqがあり、WMにはNokiaとFujiSiemensがありました。 最後のストローはHTC HD2で、これは多かれ少なかれHTC Senseインターフェースのフレームワーク内でのみ使用できます。 iPhoneの後、それは...まあ、何と比較すればよいのかさえわからない...おそらく、ニバのホイールの後ろにバーと女の子がいる長いリムジンの客室からの乗り換えで。 私は彼を6か月未満苦しめ、後悔したことのないAndroidスマートフォンを購入しました。



これらすべての年、特にWMを吐く前の最後の3年間、私は1つの考えに苦しめられました。 ハニー、Windows Mobileと呼ばれる流産に15分間かけがえのない注意を払って、ポルノのアタビズムのように見せてください。 通常のインターフェイスデザイナーを3〜5人雇って、6か月間部屋に閉じ込めます。 その後、フィードバックを収集し、さらに6か月間のルールを作成します。



ここで待って ほとんどすべてのレビューで賞賛されているWP7が出てきましたが、何かがコサックが誤って処理されたことを演劇のささやきでささやきます。 厳しい統計によると、WMデバイスの売り上げは、何もないよりもわずかに好調です。 Habr（ 1、2 ）に関する2、3のレビューでさえ、信じられないほど左目を台無しにします。 現在、MSはWP7デバイスをXBoxユーザーと、リンゴと緑のエイリアンの支配にうんざりしているユーザーにのみ販売しているように思えます。



なぜこれがすべて書かれたのですか？

さらに、MSはハンドヘルドマーケットを自然に怒らせたと思います（スペードをスペードと呼ぶ場合）。 今では、WPがインストールされている携帯電話市場の少なくとも20％を獲得するために、彼らは信じられないほどのパフォーマンスを達成し、頭上で宙返りをしなければなりません。 iPhoneが最初に登場したとき、私はキーでMSトップマネージャーの意見を読みました。「Appleの電話はwinmobileの競争相手ですか？ 笑わせないで、スリッパも。 彼は貼り付けと貼り付けの方法すら知らない。」そして、2010年末には何が見えますか？ -「...ロシア語のキーボード、マルチタスク、コピーアンドペーストがないことに混乱していない場合（これらはすべて3年前にiPod TouchとiPhoneで行われていました）...」



今、彼らはコピー＆ペーストを調整しているように見えますが、実際にはWP7は競合他社より3年遅れています。 2010年の終わりに、iOSは企業のハンドセット市場で時々WMを支払いました。 そして、2位はBlackberry、3位はAndroidです。 WMの独自の企業市場には7％程度が残っており、これは一般市場の20分の1です。 壮大な失敗。



私は、MSがモバイルアプリケーションを徹底的に取り上げ、今後数年間で完全に悲惨な状態から自信を持って実質的なシェアに転換できるようにしたいと考えています。 私はWindowsが企業の作業環境であるのが好きで、Windowsがすべてをすべてに統合するのが好きです。 そして、少なくとも好奇心からWPハンドセットの購入に興味を持たせることができれば、彼らは正しい方向に進んでいると思います。
