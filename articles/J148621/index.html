マウンテンライオンへの私の質問



そこでAppleは、「Mountain Lion」というコードネームの新しいオペレーティングシステムをリリースしました。 すべてのアップル製品と同様に、ファンの軍隊はそれを楽しみにしていました。 私も例外ではありません。 究極の真実であると主張していないこの記事では、新しいOSのマイナス面を指摘しようとします。



私について少し：Appleと初めて知り合ったのは2007年にiPod nanoを購入したときで、音楽業界のエンジニアリングとデザインの頂点であるように思えました。 時間が経つにつれて、私は徐々に艦隊をアップルに移しました。 Leopard、つまり10.5でMac OS Xを使用した経験があります。



ランチパッド


昨年のLeoの主な「トリック」の1つはLaunchpadでした。これはiOSと同様にグリッドアイコンのメニューです。 私はすぐに彼に恋をした、そして彼を通して立ち上げたプログラムの99％を言わなければならない。 マウンテンライオンでは、彼はどこにも行っていませんが、悪化しています。 MacBookのすべてがきちんと見えるなら、iMac 27には目の前にある種の不満があります。Appleは私たちを非の打ちどころのないデザインに慣れており、これを見ることは少なくとも奇妙です。







Twitter統合および通知センター


iOS 5では、通知センターが昨年追加されましたが、これは非常に人気のある機能であることが判明しました。 iOSが以前はAndroidであったものを使用するのはおそらく史上初めてですが、通知センターは非常に便利であり、デスクトップとモバイルオペレーティングシステムの合併の一環として、OS X Mountain Lionに追加することが決定されました。 Twitterとの統合に加えて、すべてを同じiOS 5に追加しました。通知センターに戻りましょう。新しいメッセージが届くと、そこに表示され、ヘッダーをクリックしてMail.appに行きます。 論理的ですか？ 論理的です。 では、たとえば、Twitterの新しいリマインダーに関する通知が到着した場合、クリックが開くと、OS XのTwitterではなくTwitter Webサイトが開くのはなぜですか？ これは非論理的で、非常に派生的です。



新しいドック


ドックが変更されました。 以前にどのアプリケーションが実行されているかを明確に確認した場合（それらの下にかなり目立つハイライトがありました）、今では小さくて鈍いものが表示されています。







バッテリーインジケーター


はい、彼も変更を受けています。 以前であれば、クリックするだけで、おおよその残りのバッテリー寿命を常に観察することができました。 そして、これは、コミュニティの反応から判断して、多くの不便を引き起こしました。



壮大な失敗


そして今、問題はAppleにあり、6月12日以降に購入したMacの所有者に壊れたコードを配布することを許可しました。 最初から、リリースの数日前にサイトで報酬プログラムが突然開始され、入場した人は誰でもアンケートに記入して大切な鍵を受け取ることができたときに、すべてがうまくいきませんでした。 しかし、その後に起こったことを言葉で説明することはできません。 リリース後初日に受け取ったすべてのコードが使用されました。 誰かがトレントから吐き出してダウンロードし、誰かが（ばかげた20ドルで）購入し、私のような誰かが待っていました。 もちろん、文句を言う必要はありません。誰もがそのようなことを申し出る可能性は低いですが、あなたは思わず「スティーブの下ではそうではなかった」と思います。



少し不平を言う


Snowball（Mac OS X Snow Leopard）の後、多くの大好物が鳴り始めました。 OS X 10.9に何があり、何がなくなるか、悲しいかな、まだ誰も知られていない。
