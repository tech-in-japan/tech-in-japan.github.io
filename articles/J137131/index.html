SkitchおよびEvernoteと統合されたAndroid向けDolphin Mobile Browser





 アプリケーション ： イルカ

プラットフォーム ：Android

価格 ：無料

タイプ ：モバイルブラウザ



旅行するとき、カフェにいるとき、またはソファに座っているときは、モバイルデバイスからインターネットにアクセスする方が簡単な場合があります。 Androidをお持ちの場合は、最近SkitchとEvernoteの2つの追加を受け取ったDolphinモバイルブラウザーがこのオペレーティングシステムに適しています。 Webページのスクリーンショットでメモを取るか、Evernoteにテキストを保存できます。



Skitchアドオン

これで、AndroidデバイスのDolphinで表示しているWebページのスクリーンショットを撮り、Skitchを使用して必要なメモを作成できます。 何かを丸で囲んだり、地図上にルートを描いたり、自分自身に簡単なリマインダーを作成したりすることができます。その後、メモとともに画像をEvernoteに保存したり、友人と共有したりできます。





開始するには、 Skitchをインストールしてください 。 次に：

  1. DolphinのSkitchアドオンをダウンロードします。
  2. メモを作成するページを表示したら、ページを右にスライドしてアドオンパネルを開き、Skitchアイコンをクリックします。
  3. すぐにページがSkitchで開き、そこでテキスト、矢印、円、線などを追加できます。 次に、結果を保存するか、友達と共有します。


Webページからテキストをコピーする

Skitchを使用してWebページやスクリーンショットにメモを追加するだけでなく、表示しているページからEvernoteアカウントにテキストをコピーすることもできます。 記事、研究、ツアーオペレーターの申し出、おもしろそうなすべてのものを保存して、任意のコンピューターまたはモバイルデバイスからこの情報にアクセスできるようにします。 これを行うには：



  1. DolphinのEvernoteアドオンをダウンロードしてログインします。
  2. 覚えたいテキストが見つかったら、それを選択して、右パネルのEvernoteアイコンをクリックします。 情報が追加され、Evernoteアカウントと同期されます。


Evernote API



Dolphinの新機能は、オープンで無料の開発者インターフェースであるEvernote APIによるものです。 Evernoteと統合された機能豊富な製品やアプリケーションを作成することもできます。 詳細については、 開発者向けサイトをご覧ください。 スタッフが必要な支援とサポートを提供させていただきます。
