FacebookおよびGoogleサイトでEcwidオンラインストアを開く方法

Webスタジオ、参照条件、コード、プログラムはありません。 独立して無料で。 迅速かつ不要なトラブルなし。 わずか1時間で独自のオンラインストアを開く方法に関する簡単なヒントをいくつかご紹介します。

手順は3つの方法に適合します。

-Ecwidでストアを作成する

-Facebookページでのセットアップの保存

-Googleサイトページでのストアのインストール

公式ヘルプと知識ベースからのすべての情報。 したがって、ストア開発者にとって、これらのヒントは明白で、初歩的で基本的なものです。 ただし、オンライン取引に興味を持っているが、これまでのところ特別な知識や経験がない人には、このヒントが役立つことを願っています。









Ecwidでストアを作成する



1. www.ecwid.comにアクセスし、大きな青い今すぐサインアップボタンをクリックします。

2. [ ログインまたは登録]ページで、右側の[フォーム]フィールドに入力します（テキストファイルに入力したすべてを保存することを忘れないでください）

-名前

-メール

-パスワード

3. [ 登録 ]ボタンをクリックします。

できた 独自のストアが作成されます。 アクティベーションコードを記載したメールがメールに送信されました。 コントロールパネルが開きます。







4. Ecwidから電子メールを受信し、 ストアをアクティブにします 。 （それ以外の場合、Ecwidは冗談を言っていると判断し、しばらくすると登録を削除します）

5.「 カタログ」タブをクリックします。 カテゴリを作成します。 製品を店に置きます。 製品の説明と写真がまだ準備できていない場合。 テストカテゴリと製品-野菜と果物を残します。 青い[ 保存 ]ボタンを押します。

6. [ 設定 ]タブをクリックします。 ストアの名前、注文先の電子メール、言語、支払いシステムなどを設定します。 青い[ 保存 ]ボタンをクリックします。

7.青い[ Go to store ]ボタンをクリックします。 外観を確認してください。



しかし、まだまだあります



Ecwidストアをサイトにインストールする必要があります。 その一環として、彼は働きます。 これを行うにはいくつかの方法があります。



1. 既に独自のWebサイトまたはページがある場合（エンジン上またはエンジンなし）、Ecwid管理パネルの[ コンソール ]タブをクリックします。 サイトにストアをインストールするには、サイトのページにEcwid ウィジェットコードを挿入します。

-製品カタログ

-カート（推奨）

-水平カテゴリメニュー（オプション）

-または垂直カテゴリメニュー（オプション）

-検索（オプション）

例： Narod.ruのShop1 Websiteskin

websiteskin.narod.ru/shop

2.サイトが一般的なCMSエンジン上にある場合。 Ecwidストアをプラグイン（モジュール）としてインストールする既製のモジュールがあります：

-ワードプレス

-Drupal

-Joomla

-MODx







3. サイトがまだない場合。 そして、あなたはCMSエンジンを理解していません。 ソーシャルネットワークでサイトまたはページを作成します。

-Facebook ページにアプリとしてEcwid Storeをインストールします

-Googleサイトにガジェットと同じストアをインストールします。

例： FacebookページのWebsiteskin Store 2

www.facebook.com/technograde

例： 3つのGoogleサイト上のWebsiteskin Store

sites.google.com/site/technograderu



Facebookページの作成



1. www.facebook.comにアクセスして、アカウントを作成します 。

2. [登録]ページで、右側のフォームフィールドに入力します（テキストファイルに入力したすべてを保存することを忘れないでください）

-名前

-メール

-パスワード

-ポール

-生年月日

3. [ 登録 ]ボタンをクリックします。

できた Facebookアカウントが作成されました。 アクティベーションコードを記載したメールがメールに送信されました。 アカウントが開きます。

4. Facebookからメールを受信し、 アカウントを有効にします 。 （それ以外の場合、Facebookはあなたが冗談を言っていると判断し、しばらくすると登録を削除します）

5.ページを作成します。 （ Ecwid Storeは、アカウントではなく、ページに埋め込まれています ）。 これを行うには、一番下のアカウントのフッターにある[ ページの作成 ]リンクをクリックします 。 Facebookでは、いくつかのタイプのページから選択できます。

-ローカルビジネス、または場所

-会社、組織

-ブランドまたは製品

-アーティスト、音楽グループ

-エンターテイメント

-コミュニティの目標またはコミュニティ

必要な画像をクリックします 。 フォームが表示され、記入してください。 アクティビティの種類、名前、住所、名前を設定します。 [ 開始 ]ボタンをクリックします。 必要なデータを入力してください。 [ 作成 ]ボタンをクリックします 。

6. 完了 。 Facebookページが作成されました。 設定して記入します。 ロゴと表紙の写真をアップロードします。 いくつかのニュース、リンク、コメントを追加します。 インターネット上のページのURLアドレスを必ず指定してください 。 Facebookユーザーだけでなく、すべての人がこのアドレスを使用できるようにアクセスを構成します。



FacebookページにEcwidストアをインストールする



ストアは、リンク apps.facebook.com/ecwid-shopを 1回クリックするだけでFacebookページにインストールされます 。 これの前に、Facebookアカウントにログインする必要があります。 また、Ecwidストアの管理パネルも開きます。

1.次のいずれかを実行します。

- リンクをブラウザのアドレスバーにコピーして Enterキーを押すだけです

- トップメニューのWebサイトwww.ecwid.comで、[ 機能 ] > [Facebookアプリ ]をクリックします 。 写真付きのAwesome e-commerce Facebookアプリページが表示されます。 右下のオレンジ色の大きな[今すぐインストール ]ボタンをクリックします。







- トップメニューのWebサイトwww.ecwid.comで、[ ヘルプ ] > [ナレッジベース ]をクリックします。 FrontPageページが表示されます。 Ecwidの追加方法セクションの左側にあるFacebookリンクをクリックします。 Facebookナレッジページが表示されます。 Facebookにストアを追加する方法の見出しの下で、同じFacebookリンクapps.facebook.com/ecwid-shopをクリックします。

2. FacebookはECWIDアプリケーションをインストールする許可を求めます。 [ 許可]ボタンをクリックします。

3. FacebookがストアIDを要求するので、Ecwidストアの管理パネルに移動します。 [ コンソール ]タブで、右側の5桁または6桁の数字を見つけます。 Facebookの設定フォームにこの番号を入力してください。 [ 保存 ]ボタンをクリックします他の設定は、ページ上のカテゴリと製品の場所であり、後で設定できます。







4. 完了。 EcwidストアがFacebookページにインストールされます。 Facebookページのタイムラインに ショップアイコンが表示されます。 ショップをクリックします。 灰色のAjaxギムレットアニメーションが表示されます。 そして、Ecwidストアがページにロードされます。



Google Sitesサービスでサイトを作成する



1. www.google.com/sitesにアクセスして、Googleサイトサービスを使用してサイトを作成します。

2.既にGoogleアカウントをお持ちの場合は、電子メールとパスワードを入力し、青いログインボタンをクリックします。

3.まだGoogleアカウントを持っていない場合は、赤い[ サインアップ ]ボタンをクリックします。 [アカウント登録]ページで、[フォーム]フィールドに入力します（入力したすべてをテキストファイルに保存することを忘れないでください）

-メール

-パスワード

-国

-生年月日

サイトを作成する際、Googleサービスはアドレスの電子メール名の前半を取得します。

3. [ アカウントの作成]ボタンをクリックします 。

完了しました 。 Googleアカウントが作成されました。 アクティベーションコードを記載したメールがメールに送信されました。 アカウントが開きます。

4. Googleからメールを受信して​​、 アカウントを有効にします 。 （それ以外の場合、Googleはあなたが冗談を言ったと判断し、しばらくすると登録が削除されます）

5. Googleサイトサービスsites.google.comのアドレスを入力し、Enterをクリックします。

別の方法-アカウントのトップメニューで、[ その他 ] > [すべてのサービス]> [Google製品]ページが開きます[情報の共有と共有]セクションで、 Googleサイトの下部にある最後のリンクをクリックします 。

6. [サイト]ページで、赤い[ 作成 ]ボタンをクリックします。

開いたページで、サイトテンプレートを選択し、サイトの名前を指定し、キャプチャを入力して、赤い[ 作成 ]ボタンをクリックします。

7. 準備完了 。 サイトが作成されました。 そのアドレスは、 sites.google.com / site （サイト名）です。デフォルトでは、サイトにはホームページ、サイトマップがあります。 アカウントにログインしている場合、サイトの上部にコントロールパネルが表示され、[ ページの変更]、[ページの作成]、[その他]のタブが表示されます。

8.「 ページの作成」タブをクリックします 。 ページ名を入力-Ecwidストア。 場所を選択-ページを最上位に配置します。 赤い[ 作成 ]ボタンをクリックします。

完了しました 。 作成されたストアのインストール用のランディングページ（ランディングページ）。



GoogleサイトにEcwidストアをインストールする



ストアは 、URLリンクapp.ecwid.com/jsp/STORE-ID/gadget-v2の ガジェットとしてGoogleサイトページにインストールされます。 これを行う前に、Googleアカウントにログインする必要があります。 また、Ecwidストアの管理パネルも開きます。

1.次を実行します。

- トップメニューのWebサイトwww.ecwid.comで、[ ヘルプ ] > [ナレッジベース ]をクリックします。 FrontPageページが表示されます。 [Ecwidへの追加方法]セクションの左側にある[ Googleサイト]リンクをクリックします 。 Googleサイトのナレッジページが表示されます。 [Googleサイトにストアを追加する方法]の見出しの下に、 ガジェットURLとのリンクがあります 。 このリンクを任意のテキストエディターにコピーします。

2. リンクのURLアドレスにストアIDを書き留め 、Ecwidストアの管理パネルに移動します。 [コンソール]タブで、右側の5桁または6桁の数字を見つけます。 テキストエディタで、リンクのアドレスに文字STORE-IDの代わりにこのNUMBERを入力します。 URLリンクを番号と共にコピーします。

3.編集モードでGoogleサイトに移動します。 Googleサイトメニューで[ Ecwidストア ]リンク項目をクリックし、ランディングページに移動してストアをインストールします。

4. Googleサイトの上部のコントロールパネルで、[ ページの編集 ]タブをクリックします。 [ 挿入]、[フォーマット]、[テーブル]、[レイアウト ]の項目を含むページ編集メニューがロードされます 。 そしてテキストツール。

メニュー項目[ 挿入 ] > [追加のガジェット] （左列の下部にあるサブアイテム）をクリックします







5. [ ページにガジェットを追加]ポップアップウィンドウが開きます。 左側で[ URLでガジェットを追加]を選択します 。 フォームウィンドウで、テキストエディタからコピーしたEcwidガジェットのURLをストアの登録番号で入力します。 [ 追加 ]ボタンをクリックします。 ページ編集ウィンドウでは、タイトル「 Google Gadget」とストア名が付いた灰色のブロックがコンテンツフィールドに表示されます。 青い[ 保存 ]ボタンをクリックします。







6. 完了 。 EcwidストアがGoogleサイトにインストールされます。 サイトメニューで、[ Ecwid Store ]をクリックします。 灰色のAjaxギムレットアニメーションが表示されます。 そして、Ecwidストアがページにロードされます。



しかし



ストアの作成はほんの始まりにすぎません。 そして、すべての困難はまだ来ていません。

そして、ストアのインストールのそのような容易さと速度は、インターネットでの取引が簡単であることを意味しません。 しかし、ストアプログラムとソーシャルネットワークの開発者が良い仕事をしたということだけです。 使いやすさを最大化し、電子商取引に入るためのしきい値を下げるため 。
