Photoshop CS3で透かしを作成する簡単なスクリプト

Photoshopでテキストの透かしを作成するための簡単なスクリプトが必要になる場合があります。 画像の解像度と色を考慮します。



時々、写真に「©3fonov」を付けます。 毎回それを行うのは時間がかかり、Photoshopでこのプロセスを自動化する一連のアクションを記録しました。 結局のところ、すべてがそれほど単純ではありません。 第一に 、碑文の色を変更する必要がありました：時には背景と融合しました。 しかし、これはそれほど悪くはありません。 サイズで戦うことははるかに困難でした。 アクションでは、アイテムを介してのみ設定できます。 このため、異なる解像度の画像は、絶対サイズ（ピクセル単位）が異なる銘刻を作成しました。







アクションの助けを借りて、この問題は実際には解決されませんでした。 したがって、私は勇気を出し、透かしを作成するための2つの簡単なスクリプトを作成しました。 あなたのニーズだけに。



このプロセスでは、右端に沿って下から上に向けられた任意のテキストでレイヤーを作成します。 色は、エリアの平均照明に基づいて選択されます。エリアが暗い場合、碑文は明るいです、逆も同様です。 ラベルサイズはピクセル単位で設定されます。 ヘッドセットを指定することもできます。 これを行うには、C：\ Windows \ Fonts \フォルダでそれを見つけ、そこから名前を取ります：







スクリプトは、最初のテキストが開いているドキュメントに単純にテキストを追加し、2番目のテキストが目的のサイズに変更し、指定したフォルダーにプレフィックス「w_」を付けてJpeg形式で保存する点が異なります。 圧縮品質は0〜12の範囲で指定されます。多ければ多いほど良いです。 12と11の違いは実際には存在しませんが。



これらの設定はすべて、スクリプトファイルの最初に示されています。







スクリプトファイルをフォルダーc：\ Program Files \ Adob​​e \ Adob​​e Photoshop CS3 \ Presets \ Scripts \に配置すると、Photoshopの起動時に自動的に読み込まれ、File \ Scriptsメニューから使用できるようになります。







それ以外の場合は、同じメニューの「参照...」コマンドを使用して開く必要があります。
