マイクロソフトは、Windowsの一般的なバージョンのコアサポートの有効期限が切れることを警告します

Windows 7、Windows Server 2008、Windows Server 2003、およびその他の製品の基本サポートは6か月で有効期限が切れます。



2014年の後半になって、MicrosoftはWindows 7のすべてのバージョンを含むいくつかの主要製品のコアサポートの期限切れで顧客を「トロール」することを決定しました。主なサポート期間は通常5年であり、その間、会社はセキュリティに関して無料のパッチと修正プログラムを提供します、および機能の面で。 この期間の終わりに、マイクロソフトはさらに5年間の延長サポートを提供します。この期間中はセキュリティパッチも無料で、他のすべてのパッチには特別な（有料）ライセンスが必要です。 延長サポート期間の終了後、マイクロソフトは、セキュリティまたは機能のために、有料または無料の更新プログラムを提供しません（ただし、Windows XPユーザーにはまだ方法があります （ 回避策 -約Transl）。）サポートは完全に終了していますが）。



2015年1月13日、メインサポート期間は、 Windows 7のすべてのバージョン （Enterprise、Home Basic、Home Premium、Ultimate、Starter）およびService Pack 1がインストールされた対応するバージョンで終了します。本質的に、MicrosoftはService Pack 2 for Windowsのリリースを予定していないことを意味します7. Windows 7の延長サポート期間は2020年1月14日に期限切れになることを思い出させてください。 一部の専門家は、MicrosoftがWindows 7の人気を理由にWindows 7のサポートの終了日を変更する可能性があると考えていますが、この件に関するMicrosoftからの公式声明は今のところありません。



2015年1月13日には、Windows Server 2008およびWindows Server 2008 R2のすべてのバージョン、およびWindows Storage Server 2008のすべてのバージョンのコアサポート期間も終了します。



2015年7月に、Windows Server 2003の延長サポート期間が終了します。 この製品は、Windows XPなどのアカウントから完全に引き落とされます（顧客がカスタムサポート契約を締結していない場合）。 多くの小企業がまだWindows Server 2003を使用しているという事実を考えると、MicrosoftはWindows Server 2012またはAzureへの移行を説得することを計画しています。 マイクロソフトの専門家によると、このような移行の実行には約200日かかるため、ソフトウェアの巨人は今すぐ頭をかき回すよう求めています。



また、Service Pack 1を適用したOffice 2010（2014年10月）のサポート期間は終了します。さらに更新を受け取るには、Service Pack 2にアップグレードする必要があります。
