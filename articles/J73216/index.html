QRコードを収集します！ ポイントをゲット！ 賞品を獲得しましょう！



WEB-Studio 2009の最もアクティブな訪問者になります。



展示会WEB-Studio 2009の枠組みの中で、賞品のラッフルが開催され、誰もが参加できます。



ルールは非常にシンプルです 。QRコードは、セミナーやマスタークラスの会場の参加者のスタンドに配置されます。 QRコードは、携帯電話、コミュニケーター、iPhoneのカメラで簡単に認識される2次元バーコードです。









1.リーダーをインストールします。



2. QRコードを読み取ります。



3.最初のポイントを取得します。





読み取られたQRコードごとに、ポイントが付与されます。 参加者のスタンドから読み取ったコードは、2ポイントをもたらします。 セミナーまたはマスタークラスに参加し、そこでコードを検討すると、3ポイントを受け取ります。



注意！ 今すぐ最初のポイントを獲得できます！ これを行うには、参加者のサイトにアクセスし、そこでQRコードを見つけて読む必要があります。 参加者のサイトから読み取られた各QRコードは、貯金箱に1ポイントをもたらします。 このページのコードを読んで、最初のスコアを取得することもできます。



QRコードを読み取ることができるようにするには、以下を行う必要があります。



モバイルデバイスから、 http ： //qrpr.ru/readersにアクセスします。

リーダー-携帯電話用アプリケーションをダウンロードしてインストールします。
