Dropbox 0.7-LAN同期をサポートする新しいバージョン、およびフラッシュドライブ用のDropboxPortable

今日、 Dropboxの新しい安定したリリースがリリースされました。これは、人気のある「ファイルホスティング」およびデータ同期サービスです。 新しいバージョンの番号は0.7.79で、主な革新はLAN Syncサポートです。 この興味深いサービスに慣れていない人には、 ビジュアルビデオ （英語）を見るか、 短い説明 （ロシア語）を読むことをお勧めします。



LAN Syncを提供するものは何ですか？



LAN Syncは、ローカルネットワーク内の同期速度を大幅に向上させる、長年にわたって期待されていた革新です。 以前は、すべての変更はDropboxサーバーを介して排他的に送信され、同期速度は外部チャネルの幅に制限されていました。 これで、Dropboxはまずローカルネットワークに変更されたファイルがあるかどうかをチェックし、存在する場合はそこからダウンロードします。 確かに、この関数は、Dropboxサーバーへの新規/変更されたファイルの最初のアップロードの必要性を排除しません。



または、何か新しいものがありますか？



また、バージョン0.7では、OS XのFinderツールバーにボタンが追加されました。さらに、改善の標準セットが発表されました。
  • 最適化されたRAM消費。
  • 生産性の向上。
  • 安定性が向上しました。
上記から、Windowsのクライアントは、RAMの食欲が約25％減少したことにすぐに注目しますが、これは一般に悪くはありません。



バージョンは安定していると宣言されていますが 、メインページのリンクはまだ更新されていません 。 彼らは1ヶ月以内に自動更新を通じて新しいバージョンを配布することを約束します。 ただし、ここからダウンロードできます。
  • 窓
  • Mac OS X
  • Linux x86_64
  • Linux x86




DropboxPortable-オンライン同期が十分でない場合



ちなみに、 DropboxPortableのようなDropboxへの便利なアドオンの存在を誰もが知っているとは限りません 。 Dropboxは、フラッシュドライブの単なる置き換えではなく、その補完でもあることがわかりました。 DropboxPortableは、フラッシュドライブでインストールせずに動作します。 このフラッシュドライブにデータを同期できます。 真実は常に安定しているわけではありませんが、さらに、 Windowsのみです。
