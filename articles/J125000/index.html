無料の集中コードライブラリ：Microsoft All-In-One Code Framework





適用された問題の解決策を開始する場合、コードでの実装の簡単な例を見ると便利です。 誰かにあなたに例を書いてもらうようにしたいのです。そうすれば、彼らが彼らに武装しているなら、彼らの特定の仕事を理解することができます。



Microsoft All-In-One Code Frameworkプロジェクトの目標は、すべてのMicrosoft開発テクノロジーの一般的なタスクのコード例を提供することです。



プロジェクトチームは、これらの例を空から発明しません。 どの例が必要かを判断するために、開発者フォーラム、ソーシャルメディア、およびマイクロソフトサポートへの連絡が検討されます。 これらすべてのクエリに基づいて、最も一般的なコードサンプルがライブラリに公開されます。



さらに、サンプルのリクエストサービスを利用できます。 開発者は、必要なサンプルの件名および/または既存のサンプルに投票してリクエストを送信できます。 最も一般的なクエリについては、Microsoftのエンジニアがコードサンプルを開発します。 無料で。 そして、これらの例はライブラリに追加されます。



現在、 Microsoft All-In-One Code Frameworkには650を超えるサンプルが含まれており、そのリストはプロジェクトのWebサイトで直接表示できます。 より便利な検索を行うには、 サンプルのカタログで別のプログラムナビゲーターを使用するか、 Visual Studioに追加します。
