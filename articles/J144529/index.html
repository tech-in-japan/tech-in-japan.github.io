Hack2ownでOperaをハック！









ブラウザーをハッキングするための競争は、当然のことながら大衆に大きな関心を寄せています。一部の人にとっては、日常業務に最も安全なブラウザーを選択することが正しいことを確認するチャンスです。ブラウザで発見された脆弱性に対する賞。 残念ながら、不明な理由により、このようなコンテストの主催者はまだOperaブラウザをハッキングの対象と見なしていませんが、この誤解は今後修正される予定です。



ポジティブテクノロジーズが組織する実用的な安全性に関するポジティブハックデイズ国際フォーラムの一環として、5月30日と31日にモスクワで開催されるHack2ownコンテストで、誰もがOperaをハッキングして、当然の賞を獲得できるようになりました。 競争の参加者は、正面攻撃とプラグインのギャップの使用の両方で、人気のあるブラウザーの強度をテストする機会があります。



Hack2ownブラウザーハッキング競争は、3つのステージで構成されます。



最初のラウンドでは、参加者は、人気のあるブラウザGoogle Chrome 19.0.1084、Opera 11.64、Mozilla Firefox 12、Microsoft Internet Explorer 9の最新の安定バージョンの脆弱性を悪用しようとします。



2回目のラウンドでは、以前のバージョンのGoogle Chrome 16/17/18/19、Opera 11.64 / 11.60、Mozilla Firefox 10/11/12、Microsoft Internet Explorer 8/9、およびApple Safari 5.0 / 5.1.1を試してみることができます。 /5.1.2。



第3ラウンドでは、ブラウザ用の一般的なサードパーティコンポーネントの最新バージョン（Adobe Flash Player（11.2.202.235）、Adobe Reader（10.1.3）、Java（7 update 4））の動作が許可されます。 ブラウザのリストは、第2ラウンドのリストと同じです。



各ラウンドでの勝利に対して、金銭的な報酬が割り当てられます：



最初のラウンドでの勝利のために-137,000ルーブル 。



第2ラウンドでの勝利-75 137ルーブル 。



第3ラウンドの勝利のために-50 137ルーブル 。
