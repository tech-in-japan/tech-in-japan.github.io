モトローラショールズ

モトローラは厳しい時代を迎えています。 Linux + Java（別名motomagx、ezx）に基づく独自のプラットフォームは、その優れた可能性にもかかわらず閉鎖されています。 現在、MotorolaはAndroidに注目し、それに基づいて多くのデバイスをリリースする予定です。 そして最初の兆候はモトローラショールズです。







デバイスのいくつかのバージョンがリリースに向けて準備されています。これは、CDMAバージョン、3Gバージョン、およびQWERTYキーボードのないバージョンです。 Sholesは、コードネームÉclairのAndroidバージョン2.0を入手するはずです。 コミュニケーターの最初の配達は10月に計画されており、ロシアで正式にデバイスが登場する可能性はまだわかっていません。





Motorola Sholesの主な技術的特徴：



ディスプレイ：3.7インチタッチスクリーン、解像度854x480ピクセル、最大1600万色のディスプレイ、物理サイズ45.72 mm x 81.34 mm。

通信：USB2.0、3.5mmオーディオジャック、Bluetooth 2.0 + EDR、Wi-Fi（IEEE 802.11g）

カメラ：オートフォーカスとフラッシュを備えた5 MP

重さ：169g

サイズ：60.00 x 115.80 x 13.70 mm

メモリ：512MB / 256MB（ROM / RAM）、microSDHCカードスロット（最大16 GB）

通話時間：420分

スタンバイ時間：450時間

機能：GPS、8GB MicroSDを含む

モトローラがアンドロイドインターフェースをリサイクルするかどうか（これはすでに良好です）、特別なアプリケーションが開発されるかどうかはわかりませんが、モトローラのLinuxでの非常に古い経験（たとえば、モトローラA760、5年以上！）を念頭に置いて、デバイスはビジーになるはずです。 ちなみに、ヘッドセットジャックは標準のミニジャック3.5 mmです。
