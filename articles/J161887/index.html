HTML / CSS3 / JSの3Dチェス





フランスの開発者Julian Garnierは、HTML / CSS3 / JSを使用した3Dチェス実装を導入し、すべての3DアニメーションはCSS3を使用して行われました。



念のため、繰り返しますが、これは単なる3Dアニメーションではなく、本格的なチェスです。 JavaScriptライブラリの中で、Julianは照明にPhotonのみを使用し、 Chess.jsをゲームのルールの実装に使用しました。 著者自身が報告しているように、3Dライブラリは開発に関与していません。



フィギュアは彫刻家のヨーゼフ・ハートヴィッヒのセットから取られ、いくつかのデザインテーマが利用可能です。 二人用のゲーム。 デモは現在Webkitでのみ利用可能です。



ジュリアンは以前、同様のツールを使用して太陽系のモデルを導入していました。
