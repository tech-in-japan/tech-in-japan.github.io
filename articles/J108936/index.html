2つのタッチスクリーンを備えた異常なラップトップ：Acer Iconia





少し前までは、これと同様の東芝製のラップトップがすでにHabréに表示されていましたが、コメントでは、デバイスを職場で試した人からかなり多くの否定的なレビューがありました。 おそらく、キーボードを持たず、2つのタッチスクリーンを備えた同様のラップトップモデルは、前モデルよりも便利です。 確かに、これはまだ実用的な概念であり、まだ店頭にはありません。 Acer Iconiaの詳細-続編。



そのため、このモデルには2つのタッチディスプレイがあり、どちらも対角線が14インチで、解像度が1366 * 768ピクセルです。 ディスプレイは、その強さで有名なブランドGorilla Glassのガラスで覆われています。 デバイスのディスプレイが最大10ポイントのマルチタッチをサポートしていることは注目に値します。



このデバイスの動作コンセプトは、同社の新製品に焦点を当てた次のエイサー記者会見で実証されました。 間違いなく、このラップトップは記者会見の目玉でした。 同様のモデルと同様に、操作中にディスプレイの1つをキーボードとして使用できます。 デバイス構成は非常に興味深いです。プラットフォーム全体がIntel Core i5-480M / i5-560M / i5-580M（2.66 GHz）に基づいており、ラップトップには4ギガバイトのRAM、統合Intel HDグラフィックス、最大750ギガバイトの内部メモリ、ワイヤレスモジュールが装備されていますWi-Fi接続802.11b / g / n、Bluetooth 3.0、さらには3G、さらにWebケーブルがあります。 デバイスには、USB 3.0、VGA、HDMI、RJ45、S / PDIF、2xUSB 2.0のさまざまなポートが多数あります。



その結果、私たちはあなたが仕事と遊びができるかなり強力なデバイスを手に入れました-ラップトップは本当に普遍的です。 興味深いことに、他社の同様のモデルとは異なり、このラップトップは、ラップトップの作業でテストしたジャーナリストによると、非常に便利です。 残念ながら、市場に2台のディスプレイを搭載したラップトップのリリース日はまだわかっていませんが、少なくとも2000米ドルはかかることが知られています。
