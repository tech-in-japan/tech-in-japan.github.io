オリジナルサムスン

8月24日、Samsung Galaxy Note 10.1タブレットの販売が開始されました。 正式な販売開始の前日、私はこのデバイスに慣れる機会がありました。 合計で、現時点では5日目に使用しています。 最初の印象や考えを共有する時が来ました。











今年の2月に、Samsung Galaxy Note 10.1をバルセロナモバイルコングレスで初めて見ましたが、率直に言って、まったく感銘を受けませんでした。 スタイラス、つまりS-Penがタブレットの体内に収まらなかったという事実は、少し戸惑いを引き起こしました。 オリジナルの5.3インチノートでは、彼の兄は比較に行かなかったため、正直なところ、このモデルについて考えるのを忘れていました。 はい、サムスンは販売しませんでした。 そのため、6か月が経過し、注10.1がまだ棚に載っていました。 しかし、構成、機能、外観の一部がまったく異なるデバイスであることが判明しました。



第一に、今回のケースにはS-Penの場所があります。 さらに、スタイラスをスロットセルから取り外すと、ディスプレイのロックが自動的に解除されます。 とても素敵で便利なニュアンス！ グラフィックタブレットのメーカーの中で最も評判の良い会社の1つであるWacomによってS-Penによって開発されました。







Wacomとのコラボレーションにより、Samsung Galaxy Note 10.1ディスプレイも作成されました。 ディスプレイには、一見すると1つまたは2つの問題があります。まず、Amoledではなく、Samsungデバイスでよく使用されている問題です。 1280x800。 比較のために、古典的なGalaxy NoteにはSuper Amoledディスプレイがあり、5.3インチディスプレイの解像度はまったく同じ-1280x800です。 つまり、Galaxy Note 10.1のディスプレイは、同じ解像度でほぼ2倍の大きさです。 しかし、なぜですか？ そして、なぜディスプレイはAmoledではなく、PLSなのですか？ 結局のところ、このデバイスは、Mali 400グラフィックアクセラレータと最大2 GBのRAMを搭載したクアッドコアExynos 4412プロセッサの構成において明らかにフラッグシップです。 さて、サムスンは欲張りからでも怠lazからでも、旗艦ではない画面をインストールすることはできませんでした！







これらの質問は、おなじみのアーティストに両方のGalaxy Noteを見せたとき、彼ら自身で消えました。 彼はサムスンの可能性のある論理を非常にわかりやすく説明してくれました。 事件は次のとおりでした。 Amoledディスプレイが過度にジューシーであることは誰もが知っています。 ギャラクシーノート10.1に1つの絵を描くアーティストを単純に誤解させてから、他の非Amoledディスプレイに別の絵を描くことになります。 許可に関しては、すべてが論理的でシンプルであることが判明しました。 実際、非常に高い解像度では、正確なペン入力は困難です。 つまり、アーティストは、描画に必要な正確なピクセルを取得するために、常に図面を拡大する必要があります。 ちなみに、私の友人はこのデバイスを非常に気に入っており、SketchBookXで数枚の写真を描いてPhotoshop Touchで処理するという喜びで一日中過ごしました。







PLSマトリックスに関しては、実際にはまったく悪くなく、技術的特性をより詳細に検査することが判明しました。これは何らかの理由で、生産の安さから考慮し始めました。 そのため、たとえば、PLSのピクセル密度はIPSのピクセル密度よりも高く、PLSはより優れた演色性と視野角、より高い輝度、より低い消費電力を持ち、sRGBカバレッジがいっぱいであることが判明しました。







後に、私の友人の議論を熟考しながら、私は突然、AppleがGalaxy Note 10.1のディスプレイ解像度について当惑したことを本当に責めていることに気付きました。 アップル社はディスプレイの解像度を上げる傾向を生み出し、積極的に開発しています。そのため、私たちの消費者は、確かに、高解像度ディスプレイが最も重要な要素であると考えています。 しかし、これは必ずしも利益のための高解像度ではないことが判明しました。 新しいiPadでは、ディスプレイの超高解像度のために、iPad2にはない問題があったという事実については話していません（たとえば、ディスプレイは入力時にキーストロークを認識する時間がない場合があり、しばしば「いつ」または「プロ」の代わりに単に「and」または「about」を印刷します）。







さて、デバイスに移りましょう。 タブレットを一目見ただけで、スピーカーの位置は非常に快適でした-スピーカーは前面に配置されているため、ほとんどの場合、手のひらで握り締めることができます。 同時に、スピーカーは非常に大きいです。 周波数範囲では、本格的なポータブルスピーカーを置き換えることができるとは言いませんが、それにもかかわらず、彼らは音楽を大声で再生します。 はい、ちなみに、ビデオ通信用のスピーカーとフロントカメラ（1.9メガピクセル）の両方の位置から判断すると、タブレットは主に水平方向で使用するように設計されていますが、もちろん、全画面回転インターフェイスがサポートされています。



すべてのハードウェアボタンは、上面に配置されています。 オン/オフボタン（ディスプレイロック解除ボタンでもあります）、ボリュームスイング、SIMカード（フルサイズ）、micro-SDカード用のスロットがあります。 3.5ヘッドフォンジャックもあります。 コンピュータの充電および切り替え用のコネクタは下側に配置されており、通常のマイクロUSBではなく、特殊なSamsung 30ピンです。 正直なところ、デバイスにマイクロUSBコネクタがまったくないのは残念です。 したがって、私たちは別のコードを運ぶ運命にあります。 また、日中にタブレットを充電する必要があるわけではありません。 違います。 Galaxy Note 10.1には7000 mAhの容量のバッテリーが搭載されており、問題なく負荷をかけてデバイスを使用すると9〜10時間持続します。 しかし、タブレットをデータ交換のためにコンピューターに接続する場合、たとえば、同じ30-ninコードがなければ機能しません。 Galaxy Note 10.1にはLEDフラッシュ付きの非常に優れた5MPメインカメラがあるため、このようなニーズが生じる可能性があるため、これはマイナスです。 彼女はFullHDではなく、単純なHD（720p）のビデオを撮影しますが、かなり良い品質です。







私が注目したいもう1つの素晴らしいニュアンス：Galaxy Note 10.1は電話通信の機能をサポートしています。 正直なところ、同じiPadや他の一部のAndroidタブレットでは電話や電話ができない理由を理解できませんでした。 つまり、マーケティングの観点からは、もちろんすべてが明確ですが、一方で、デバイスはデバイス用のユーザーではなくユーザー用であるためです！ もちろん、10インチのタブレットを使って耳をつかむことはうまくいきませんが、スピーカーフォンやヘッドフォンを使用することは非常に便利です。 いずれにせよ、私はメインの電話SIMカードをGalaxy Note 10.1に挿入し、タブレットを、とりわけ電話として使用していました。 違和感はありませんでした。 しかし、真実のために、電話ではスピーカーフォンをオンにするか、接続されたヘッドセットを介して話すことが多いと言います。



そしてもう1つのニュアンス：赤外線ポートは上面にあります！ これは、Galaxy Note 10.1を使用するまったく異なる方向です。 つまり、Galaxy Note 10.1は家電を管理するためのハブになるように設計されていることがわかりました。 スマートリモートアプリケーションは、さまざまなブランドの多くのテレビやプロジェクターのサポートが既に組み込まれている赤外線ポートの操作を担当します。 そのため、Galaxy Note 10.1をリモートコントロールとして機能するように設定しても問題はありません。 ただし、この方向は今後も続くと確信しており、赤外線ポートのおかげで、Galaxy Note 10.1は他のデバイスを制御するために設計されたデバイスの中でも特に、デバイスの全ラインの最初のモデルになる可能性があります。







さて、Galaxy Note 10.1ソフトウェアについて話しましょう。 そして、プレインストールされたOSとして、タブレットはAndroid ICS 4.0.4を搭載していることを除いて言うべきことがあります。 Galaxy SIIIのAndroidでも、サムスンはこのオープンOSのサムスンアセンブリを他のバージョンから有利に区別するいくつかのチップを導入しました。 ここで最も記憶に残っている機能はポップアッププレイで、これにより、プレーヤーで再生されたビデオを、開いている他のアプリケーションの上に小さなウィンドウで表示できます。 ただし、4.8インチSIIIディスプレイでは、この機能はあまり適切に見えませんでした。 しかし、Galaxy Note 10.1 Pop Up Playでは非常に便利に見えます。







しかし、それだけではありません！ Galaxy Note 10.1上のAndroidでは、本格的なマルチタスクが最初に登場し、2つのオープンでアクティブなアプリケーションで同時に作業できるようになりました。 正直に言うと、これは突破口です。 これらすべての年月の間、Androidのマルチタスキングについて知らされてきました。同時に、バックグラウンドに移行されたアプリを穏やかに安楽死させ続けました。 そして最後に、最初のデバイスが登場しました。このデバイスでは、開いているブラウザまたはソーシャルネットワークエージェントの上でビデオを再生できるだけでなく、たとえば同じブラウザ、メモ、またはオフィスを開いたままにすることもできます。 現在、マルチウィンドウモードをサポートしているアプリケーションは、ブラウザ、ポラリスオフィス、Sノート、ビデオプレーヤー、ギャラリー、および電子メールの6つだけです。 残念ながら、このリストにはまだYouTube用のプレーヤー、同じ電話、またはSMSメッセージ用のクライアントがありません。 10インチディスプレイで同時に他の多くのものを開きたいです。 しかし、最も重要なことは、最初の一歩を踏み出したことです。新しいファームウェアでは、時間の経過とともに、マルチウィンドウモードで動作するように構成されたアプリケーションの数が増えると確信しています。 幸いなことに、充填装置の力がこれを可能にします。







この予備審査をまとめると、私の意見では、サムスンは率直に革新的であり、元のデバイスであるAppleとの訴訟の文脈で判明したと言います。 ここで、Appleは間違いなくピッキングの理由を見つけません。 ペン入力は非常に重要で有望な方向であるように思えます。 私がもう一度言うのが面倒ではない唯一のことは、Android Play Marketのアプリケーションベースをサポートしているにもかかわらず、Samsungが変更されたAndroidではない独自のOS、つまり独自のOSを本当に必要としているということです特定のデバイスの場合、少なくとも特定のデバイスの行の場合。 私の意見では、Note行はこの新しいOSの最適な候補です。 ギャラクシーノート10.1の販売開始の少し前に、サムスンが2012年末から2013年に最初のTizenデバイスのリリース日を延期したことは、この点で残念です。 まあ、私たちは待ちます。
