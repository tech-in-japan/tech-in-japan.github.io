ロシアでのノキア開発者ワークショップのトレーニング

2013年4月22〜24日にペトロザヴォーツク州立大学で開催されるロシアのNokia Developers Workshopのトレーニングにご参加ください。

イベントの一環として、Windows Phone 8およびSeries 40 Ashaモバイルプラットフォームの開発者向けに2つの専門的なトレーニングが予定されています。









Windows Phone 8の開発者向けトレーニング。

このレッスンでは、ノキアLumia 920など、Windows Phone 8をベースにした最新モデルのプログラム開発の詳細について学習します。

このプログラムには、現在のプログラミングトピックとWindows Phone APIに関するいくつかの簡単なレビューが含まれており、アプリケーション開発に関する実践的な演習が続きます。 前提条件は、参加者がオブジェクト指向プログラミングの実践的な経験を持っていることです。

トレーニングの目的は、Windows Phoneプラットフォーム用のアプリケーションとサービスの独立した開発のために参加者を準備することです。



Java ME for Ashaの開発者向けトレーニング。

イベント中に、最新のユーザーインターフェイスAPIと、位置情報サービス、地図、センサー、収益化のための最新のツールとAPIがデモされます。

目標は、開発者にとって意味のあるNokia Ashaシリーズデバイスの最新アップデートをJava MEプログラマーと共有することです。



イベントへの参加は無料で、座席数は限られているため、リンクでイベントに登録することを忘れないでください： www.fruct.org/nokia13
