Windows 8.1評価版およびWindows Server 2012 R2評価版のフルバージョンへのアップグレード

今日は、Windowsの評価版からフル版にアップグレードする2つの方法についてお話します-Windows Server 2012 R2ベースのサーバー用とWindows 8.1クライアント用。 方法は根本的に異なります。サーバーOSの場合、移行はMicrosoftによって単純なスケジュールされた操作として考えられました。クライアントOSの場合、ベンダーの考えは、ユーザーにシステムの完全な再インストールを強制することです。 以下に提案する方法を使用すると、再インストールせずにWindows 8.1の評価版を完全なものにアップグレードできます-ちなみに、評価版のみを使用して正直に自分自身を証明する必要がある場合は、Windows 8.1のすべてのバージョンからアプリケーションとデータを保存した評価版に逆に移行します。

ただし、簡単なものから始めましょう。 最初の質問は通常尋ねられます-なぜですか？ 実際には、インストールから6か月後にポップアップメッセージと黒いデスクトップ背景で評価ステータスのみを通知するWindows 8.1評価クライアントシステムとは異なり、サーバーシステムは1時間に1回再起動しますが、これは既に完全に不快です。



Windows Server 2012 R2 Evaluationをフルバージョンにすばやくアップグレードする方法。



有効または期限切れの試用期間で、Windows Server 2012 R2評価版をインストールしました。 OSには、特に評価版から完全版へのOSエディションのアップグレードを定期的に行う機会があります。 次のコマンドを使用して、インストールされているエディションを確認できます。 
DISM /online /Get-CurrentEdition
次に、コマンドラインからこのOSをアップグレードできるエディションを確認できます。 
 DISM /online /Get-TargetEditions




スクリーンショットは、Windows Server 2012 R2 Standardの試用版をインストールし、試用期間が終了したことを示しています。 2番目のコマンドは、OSをフルバージョンのStandardまたはDatacenterにアップグレードできることを示しています。 これを行うには、シリアル番号を入力するだけです-TechNetページからの番号-KMSクライアントセットアップキーは完璧です。 次のコマンドを使用します。 
 DISM /Online /Set-Edition:ServerDatacenter /AcceptEula /ProductKey:W3GGN-FT8W3-Y4M27-J84CP-Q3VJ9




このチームで、Standardの評価バージョンをDataCenterのフルバージョンに変換するプロセスを開始します。 Standardのシリアル番号とServerStandard Editionを指定した場合、Standardのフルバージョンを入手できます。 再起動後、コマンドを再度実行すると結果が表示されます 
 DISM /online /Get-CurrentEdition




これで、KMSサーバーでシステムのライセンス認証を安全に実行し、サーバーを更新する重要なタスクを完了できます。



Windows 8.1評価版の評価版を完全版にアップグレードする方法は？



残念ながら、クライアントOSの場合はそれほど単純ではありません。 弊社では、評価版をダウンロードしてインストールし、公開されている完全なキーに変換することは許可していません。 Windows 8.1の試用版はEnterpriseエディションでのみ利用可能で、ロシア語版はありません。インストール後に英語をインストールし、ロシア語のインターフェイスを追加する必要があります。 以前にWindows Server 2012 R2 Evaluationで実行したWindows 8.1 Enterprise Evaluationで同じコマンドを実行しても、エディションを変更する可能性はありません。



次の論理的なアイデアは、Windows 8.1 Enterpriseのフルバージョンのイメージをダウンロードし、アプリケーションと設定を保存してインストールプロセスを「上に」実行することです。 しかし、残念ながら、失望が待っています-データを保存して試用版を完全版に更新することはできません。完全な再インストールが提供されます。



ただし、あきらめないでください。 愛好家は、評価版、コア版、プロフェッショナル版のいずれであっても、Enterpriseバージョンを他のものの上にインストールできるツールを長い間知っています。 Windows 7の時代に機能していた同じ方法で、プロフェッショナルからエンタープライズに切り替えました。 OSに既に完全なエンタープライズがあり、すべてのアプリケーションと設定を保存した状態で更新インストールを実行することをOSに伝えましょう。 レジストリエディターを開き、2つのキーの意味を確認します（赤で強調表示）。



EditionIDとProductNameをフルバージョンの同じ値に置き換えます。



これで、再起動せずに、OSの更新プロセスを再開できます。 今回は、アプリケーション、ファイル、設定の保存など、さまざまな更新パスにアクセスできます。



インストールウィザードはこの手順を実行し、すべての設定を保存してアップグレードプロセスを開始します。



KMSサーバーを使用すると、OSをライセンス認証して結果を楽しむことができます。



必要に応じて、同じメカニズムにより、エンタープライズからプロフェッショナルまたはエンタープライズ評価へのダウングレードが可能になります。 誰かがこの情報が役立つことを願っています。
