歴史の深さへの小さな余談



BlackBerry Enterprise Serverの最初のバージョンは1999年に登場し、Exchange Server 5.5で機能しました。 今日の管理者の多くがExchange Server 5.5を見つけていないようです。BlackBerryEnterprise Serverについて言えることは、ロシアですぐに登場したことです。

2004年11月に、第4版のブランチが登場しました。このブランチは、今日まで、多くの人がどこで（もちろん、その後のサービスパックでも）使用しています。 これはBlackBerry Enterprise Serverの最初のバージョンになりました。このテキストの作成者は、2007年にそれを実質的に知っており、認定さえしました:)

BlackBerry Enterprise Server Expressはエンドユーザーに何を提供しますか？



• メール ：

o HTMLサポート付きのメールをプッシュする

o双方向の同期（デバイス-スマートフォンとサーバースマートフォン）

oメッセージを「実行用」にマークする（電子メールにフラグを立てる）

oスマートフォンからサーバー上のメッセージをリモート検索

oスマートフォンからサーバー上のメールフォルダーを表示および管理する

• カレンダー ：

o両側のワイヤレスカレンダー同期（デバイス-スマートフォンとサーバースマートフォン）

o同僚の空き時間情報を表示する機能

o会議を転送する機能

o会議に添付されたファイルの表示、編集、保存

• 連絡先 ：

o両側の連絡先のワイヤレス同期（デバイス-スマートフォンとサーバースマートフォン）

oサーバーでのワイヤレス連絡先検索

o連絡先とのフォルダー、個人用メーリングリスト、および連絡先とのパブリックフォルダーの同期（後者には、含めるには別の要求が必要です）

• ドキュメントとファイルへのアクセス ：

oファイルを表示、ダウンロード、編集、保存、添付する機能

oすべての一般的なファイル形式のネイティブサポート

oオーディオファイル（.AWE、.WAV、.MP3、およびオーディオWMA）のサポート

o Open Document Text（ODT）、Open Document Presentations（ODP）、およびOpen Document Spreadsheets（ODS）のサポート

• セキュリティ ：

oパスワードをリセットし、スマートフォンをブロックする機能。 スマートフォンのデータをリモートで消去する

このソリューションはホスティングバージョンで利用できるようになり、顧客がセルフホスティングのための鉄への投資や、ロシアで製品を使用するためのライセンスを取得する際の問題を回避できます（ここでは想定されていないRIMライセンスと混同しないでください）。
