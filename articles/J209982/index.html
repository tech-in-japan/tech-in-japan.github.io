Gartner Report：エンタープライズアプリケーションのPaaSマーケットリーダー

Gartner Agencyは、エンタープライズアプリケーション（aPaaS）のPaaS市場（サービスとしてのプラットフォーム）に関する新しいレポートを発表しました。 ガートナーは、PaaS内のクラウド環境で企業ソリューションをホストするための市場の提供物を分析し、その発見を会社の古典的な「マジッククアドラント」で提示しました。







ガートナーは、aPaaS市場で最大のプレーヤーであるSalesforce.com、Microsoft、Google、Engine Yard、SAP、IBM、Red Hatなどを評価しました。 各ソリューションの利点と注意事項の詳細な説明は、レポートに記載されています。 有名なAmazonプロバイダーは、主にIaaSインフラストラクチャを提供するため、レポートには含まれていませんが、Amazonのサービスに基づいて構築されたaPaaSプラットフォームを調査しました。



分析の結果、Salesforce.comとMicrosoft（Windows Azureプラットフォーム）がaPaaSのマーケットリーダーに選ばれました。 Windows Azureの場合、Gartnerは次の説明を提供しました。



ベンダー（Microsoft）のアプローチは、「すべての核となるクラウド」の原則に焦点を当てており、頻繁な更新と、新機能と改善の積極的な開発を伴います。 広範囲の目標は、.NETアプリケーションをPaaSサービスの形式で開発するために必要な包括的なインフラストラクチャを提供することです。 Windows AzureのPaaS機能はaPaaSを超えており、SQLデータベース（dbPaaS）、メッセージングサービス（Windows Azureサービスバス）、分散キャッシュ（Windows Azureキャッシュ）、iPaaS機能（Windows Azure BizTalkサービス）が含まれます。 このプラットフォームは、多くのモバイルプラットフォームのサポートを提供するサービスの形で、モバイルアプリケーション用のクラウドベースのバックエンドである新しいWindows Azureモバイルサービスも導入します。 他のクラウド環境（Microsoft Dynamics CRM OnlineやSharePoint Onlineを含むOffice 365を介したSaaSなど）におけるMicrosoftの存在も、クラウドベンダー戦略に機能を追加します。



Microsoftプラットフォームの主な利点は、次の特徴です。



  • Windowsアプリケーション開発者と.NETに精通している開発者は、Windows Azure互換の習慣と知識に安心します。 これにより、Windows Azureを独自のプラットフォームと見なすことができる何百万人もの開発者を引き付ける機会が提供されます。
  • マイクロソフトは、パブリッククラウドとプライベートクラウドに同じ機会を提供することに重点を置いています。つまり、企業の開発者は、クラウドプラットフォームが既存の環境とうまく統合でき、必要に応じてクラウドからアプリケーションを転送できるという自信を持ってWindows Azureを使用できます自身の能力;
  • マイクロソフトはプラットフォームの包括的なビジョンを持っています 。これには、主要なaPaaSサービスに加えて、統合サービス、インメモリコンピューティング、メッセージング、およびサービスとして提供されるその他の機能のサポートが含まれます。 さらに、Windows Azureサービスには急成長中のIaaSプラットフォームが含まれており、開発者は必要な抽象化レベルを使用して、同じタスク内でPaaSとIaaSのアプローチを組み合わせることができます。
  • Windows Azureは、.NETとともにNode.js、Java、PHP、Python、Ruby、およびその他のアプリケーション開発プラットフォーム（IaaSのLinuxを含む）をサポートしているため、Microsoft以外のプラットフォームで開発した経験のある開発者はWindows Azure環境を使用できます。


Gartnerは、Microsoftプラットフォームに関するいくつかの注意事項も強調しています。



  • Microsoftの提供には、GUIツールの形式のモデルに基づいたアプリケーションの迅速な開発のための不十分なサポートが含まれています（Visual Studioツールキットにもかかわらず）。 会社の製品（Dynamics CRMの一部）で利用可能なxRM機能は、Windows Azureと一緒に使用できますが、ベンダーはそれらを積極的に宣伝していません。
  • マイクロソフトは、会社のトップマネジメントの変化と、Officeに関連する主要なビジネスとの交差点により、会社の努力をそらす可能性があるため、不確実性に直面しています。 IaaS、PaaS、およびSaaSの多くの領域の開発は、同時に会社の焦点を侵食します。
  • パブリッククラウド（Windows Azure）およびプライベート環境（Windows Server）の開発者に同じ機会を提供することはまだ稼働中であり、マーケティングの約束は現在の製品の現実とは異なります。


他のaPaaSサービスプロバイダーの詳細と説明は、ガートナーのレポートに記載されています。
