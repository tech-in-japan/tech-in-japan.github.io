III国際会議「FOSS Sea 2010」

親愛なる同僚！



9月16日から17日までオデッサで開催される第3回国際会議「FOSS Sea 2010」にご参加ください。 Software Freedom Dayは会議で計画されており、 Install Festは無料/オープンソースソフトウェアの初心者ユーザー向けに開催されます。



会議の目的は、効果的なコミュニケーションと情報交換を確保し、フリーソフトウェアの開発の見通しについて議論し、科学と教育におけるオープンソースソフトウェアの使用に関する新しいプロジェクトを開始することです。



公式部の会場はONUのビッグフィジカルオーディトリアム（Pastera、27）で、 非公式 の会場はクライミングクラブ「オデッサ」です。



9月12日まで、参加者はレポートの要約、電撃戦レポート、またはマスタークラスを会議のいずれかのセクションに提出できます。フリー/オープンソースソフトウェアの主題の紹介。 教育および科学分野の無料/オープンソースソフトウェア。 公共および公共部門向けの無料/オープンソースソリューション。 ビジネス部門向けの無料/オープンソースソリューション。 フリー/オープンソースソフトウェアに基づくハードウェアソリューション; 開発者とフリー/オープンソースソフトウェアのユーザーの相互作用; 無料/オープンソースソフトウェアを使用した科学プロジェクト。 Webスペースの無料/オープンソースソフトウェア。 高性能システムとクラスター。



スポンサーを招待します。 このイベントへのサポートは、オープンテクノロジーの分野で有資格の専門家と協力したい企業や、ウクライナでフリー/オープンソースソフトウェアをサポートおよび開発する企業にとって興味深いものです。



詳細と登録-ウェブサイトwww.foss-sea.org.uaで 。
