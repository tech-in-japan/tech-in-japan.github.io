ダウンロードリンクが利用可能になりました* Ubuntu 15.10 Wily Werewolf





Ubuntuチームは、Ubuntu 15.10 Wily Werewolfの最終バージョンの提供をまだ発表していませんが、リリースはすでにダウンロード可能です。

私はあなたに思い出させます：

Ubuntu 15.10は、Ubuntu Desktop、Ubuntu Server、Ubuntu Core、Kubuntu、Ubuntu Kylin、およびXubuntuなどの他のすべての製品で9か月間サポートされます。



Ubuntu 15.10 Wily Werewolfの主要な変更：

•Linux 4.2.xカーネル。AMDの最新のAMD GPUのサポートが追加されました。

•Intel Broxtonのサポート。

•F2FSファイルシステム暗号化のサポート。

•NV-DIMMのサポート：

•多数の修正と新機能を備えたUnity 7.3.2

•ソフトウェアセンターからSteamをインストールし、Big Pictureモードでマニピュレーターとペアリングするのに十分なワイヤレスマニピュレーターSteamコントローラーの組み込みサポートを提供しました。

•BlueZ 5.34スタックを使用して、リリース2.0に更新されたBlueman Bluetooth Connection Configuration Manager。

•Mesa 11.0.x

•Xorgサーバー1.17 .x、

•Pulseaudioがバージョン6.xに更新されました

•Compizは、MATE環境との統合の改善に取り組みました。

•オプションとして、Unity8 for desktopsの紹介セッションが提供され、音楽プレーヤー、音楽プレーヤー、Dekoメールクライアントなどのデスクトップバージョンのアプリケーションへのアクセスが提供されました。

•事前設定されたサービスを迅速に展開するためのインターフェイスであるJujuを更新しました。 何百ものサービス（Juju Charms）がJujuで使用できるように準備されており、インストール後すぐに使用できます。 これらのサービスには、さまざまなDBMS（MySQL、PostgreSQL、Redis、Cassandra、Mongodb、Membase）、Webアプリケーション（wordpress、drupal、Node.js、Django、Ruby on Rails）、サーバーシステム（lamp、tomcat、vsftpd）、監視システムがあります、クラウドプラットフォームなど

•サーバーエディションには、カーネルネットワークスタックをバイパスしてパケットを直接処理する高性能ネットワークアプリケーションを作成するためのDPDK（Data Plane Development Kit）フレームワークの実験的サポートが含まれています。

•ソフトウェア-LibreOffice 5.0.2、Shotwell 0.22、Chromium 45、Rhythmbox 3.2.1、GNOME Terminal 3.16、Empathy 3.12.10、Nautilus 3.14.2、Totem 3.16、libvirt 1.2.16、QEMU 2.3、Open vSwitch 2.4.0、 Ceph 0.94.3
