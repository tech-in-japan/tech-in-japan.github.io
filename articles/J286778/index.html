MicrosoftはInternet Explorerを殺すつもりはない

マイクロソフトは現在、「Project Spartan」というコードネームの新しいブラウザに取り組んでいます。 このブラウザーは、会社の新しいオペレーティングシステムであるWindows 10で動作し、OSの以前のバージョンの多くのユーザーは、ブラウザーの新しいバージョンがリリースされたときにInternet Explorerに何が起こるかに興味を持っています。 一部のメディアは、MicrosoftがInternet Explorerのブランドを「殺す」という見解を表明していますが、実際にはそうではありません。



Windows 10開発者は、Internet Explorerも新しいOSの配布に含まれることを確認しています。 しかし、今やIEはProject Spartanの後、傍観者になります。 同様に、リリース後の新しいブラウザには別の名前が付けられます。







Microsoftの広報担当者は、Business Insiderに「Project SpartanはWindows 10専用に設計された次世代ブラウザです」と語った。 同時に、IEでの作業は継続されます。



どうやら、Microsoftは徐々にIEブランドから遠ざかり、多くのユーザーに否定的な関連性をもたらします。 同時に、企業はユーザーの注意を新しいプロジェクトに引き付けます。



Project Spartanは、Windows 10と同時に初めて発表されました。ブラウザには新しいデザインとエンジンがあり、Cortanaとの統合もあります。 この統合により、ブラウザはユーザーがWebページをより効率的に操作できるようになります。Cortanaは、ユーザーの電話番号、住所、その他の有用な情報をページで検索できるようになります。



Project Spartanを搭載したWindows 10は、2015年の天文学的な夏の終わり（9月23日）にリリースされ、190か国で111の言語でリリースされます。
