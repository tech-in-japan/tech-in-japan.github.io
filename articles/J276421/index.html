マイクロソフトは、ユーザーがWindows 10にアップグレードすることを強くお勧めします

Microsoft は 、Windows 7およびWindows 8.1のユーザー向けに、Windows 10へのアップグレードのステータスを「オプション」（オプション）から「推奨」（推奨）に変更しました。 1月初旬、同社は既にWindows 10が2億台以上のデバイスにインストールされていることを発表し、ユーザーがWindowsの新しいバージョンにアップグレードするための将来のインセンティブも示しました。 本日から、このOSバージョンのアップデートが、上記のOSのすべてのユーザーのアップデートのリストに表示されます。







同時に、このアップデートは依然として「オプション」のリストに含まれており、ユーザーはこのOSのギガバイトのインストールファイルのダウンロードを拒否し、以前と同様にその後のインストールを行うことができます。 また、Microsoftは、ユーザーがWindows 10での作業を好まなかった場合、インストール後30日以内に以前にインストールされたWindowsのコピー（ダウングレード）に戻ることを許可しています。



以前に、Microsoftは既に元のWindows 8のサポートを拒否しており、ユーザーにWindows 8.1またはWindows 10へのアップグレードを推奨していると書きました。また、2017年に、新しいマイクロプロセッサの所有者に対するWindows 7およびWindows 8.1のサポートの早期終了を発表したことも示しました。 Windows 10への移行を促進しました。Windowsの古いバージョンのユーザーにとってのもう1つの打撃は、今年1月以降のInternet Explorer 7-10の安全でないバージョンのサポートの早期終了です。
