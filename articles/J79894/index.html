インターネット社会から抜け出す-Web 2.0 Suicide Machine

最近、「 Web 2.0 Suicide machine 」と呼ばれる面白いインターネットサービスのネットワークに出会いました。 「インターネットでの社会生活」に悩まされている人々が、数回クリックするだけでFacebook （現在、IPによる管理によりブロックされている）、 Twitter 、 Linkedin 、 Myspaceのアカウントを削除できるようにすることを目的としています。 一般に、肯定的なレビューと「成功したクレンジング」の数から判断すると、ソーシャルネットワークに夢中になっているタカは、サービスを正常に使用し、満足しています。



「私の人生は25％向上しました」、「本物の人間のように感じ始めました！」、「私のガールフレンドはあなたに感謝しています！」などのコメントは特に喜ばれました。 このサービスがOdnoklassniki、Vkontakte、LJなどのゾンビサービスと連携しないことは残念です。 ただし、これは、この概念に興味を持ち、かつてのCISの居住者向けに同様のサービスを作成する可能性のある「私たち」開発者にとってのチャンスだと思います。
