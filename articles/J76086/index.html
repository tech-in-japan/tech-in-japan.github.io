12月8日、モスクワでのWeb標準をサポートするオープンコミュニティ会議

過去1年間、ロシアでは、クライアント側のWebテクノロジーに関する単一の専門会議が開催されていないという事実を考えると（もちろん、これらの同じ会社の製品を主に宣伝することを目的としたさまざまな会社の個別のイニシアチブではないことを意味します）、 WHTパーティーと呼ばれるイベントは、 今年の主要な業界イベントになる可能性がすべてあります。 :-)まあ、もちろん、ミンスクでのWeb Standards Daysの後。 しかし、ミンスクはロシアではありません...



WHTパーティーとは何ですか？ これは、2つの部分を組み合わせた共同イベントです。 最初の「深刻な」部分は、 Web Standards開発コミュニティ（TSA。少し前に、このコミュニティはWSGのロシア部門として知られていました）の公開会議です。 2番目の部分「デザート」は、 WebHiTechコンテストの授賞式です。



TSA会議はミニ会議の形式で開催され、誰でも参加できるという意味でオープンと呼ばれています。 これを行うには、参加のリクエストを残してください 。 残念ながら、座席数は限られています。TSAミーティングに200〜220人の最初の入札者を招待する準備ができています。 もちろん、「クリティカルレベル」に達するとすぐに登録を終了します。 しかし、これがあまりにも早く起こるという事実ではありません。 :-)



登録されているすべての参加者は、WHTパーティー内のイベント（ミニ会議自体、コーヒーブレーク、ビュッフェテーブルとその後の無料コミュニケーションを含むWebHiTech授賞式を含む）に参加することは完全に無料です。 登録することで、イベントのどの部分があなたにとって最も興味深いかによって、いつでも来ることができます。



WHTパーティーは12月8日火曜日にモスクワで開催されます。 イベントのプログラムはhabracicの下にあります



ミニ会議は、WebHiTech'aの受賞者の授与である午後2時頃-午後6時頃に開始する予定です。 夕方、少し後に、TSAミーティングの参加者とスピーカー、受賞者、組織委員会のメンバー、WebHiTechコンテストの審査員との間で相互に自由なコミュニケーションをとる時間を設けます。



TSA会議の予備プログラムは次のとおりです。

  • 会議のオープニング//ヴァディム・マケエフ、アルテミー・ロモフ（10分）
  • Webフォントと Web用フォント// Vadim Makeev（40分）
  • すべてを時間通りに再配置// Andrey Sumin（30分）
  • レイアウト中の分業// Andrey Shitov（30分）
  • VML、SVG、Canvas：次は何ですか？ //ニコライマツィエフスキー（20分）
  • IEでの式の体系化// Roman Komarov（20分）
  • すべて1行で。 Webページのコードの最適化について// Dmitry Shilnikov（20分）
  • 純粋に技術的な文脈でのWebHiTechコンテストの結果について簡単に説明します// Artemy Lomov（20分）
  • 閉会//ヴァディム・マケエフ、アルテミー・ロモフ（10分）
TSAの会議の状況をリラックスした、非公式で、民主的で友好的なものにするための願望に満ちているという事実に注意してください。したがって、上記の報告プログラムは正確なスケジュールではありません 。 実際、他のレポートがこのリストに追加される場合があります。 公演の期間は目安です。 レポートの順序は変更される場合があります。 しかし、途中のどこかで間違いなくコーヒーブレイクがあります。 :-)



そのため、詳細を読んで、イベントの公式ページで登録してください。
