Microsoft Azure StorSimpleはこの問題の解決策ですか？



Microsoft StorSimpleサービスは、ローカルにインストールされたストレージデバイスにアクティブなワーキングセットを格納することでこの問題を解決します。現在使用されていない情報を送信し、暗号化された形式でMicrosoft Azureクラウドに送信します。
