Web開発の棚の状態。 25ツイート中の1人の専門家の意見

以前の翻訳で言及された記事の翻訳 。



多くの専門家は、ウェブの内と外をよく知っています。 その中にはジョーヒューイットもいます。 過去10年間、彼はNetscape、AOL、Firefox、およびFacebook（現在勤務している）などの業界大手に勤務しています。 ヒューイットはまた、伝聞によってiPhoneについて知っています。 iPhone用の優れたFacebook Webアプリケーション（ネイティブアプリケーションがまだないとき）を最初に作成し、次にネイティブクライアント（iPhoneプラットフォームで最高のアプリケーションの1つ）を作成したのは彼でした。 したがって、Joeがケースで何かを言うとき（彼は時々これを行います）、人々は注意します。 そして今日、彼は発言することにしました。



今朝、AppleのCEOであるSteve JobsからのFlashに関するメールに返信して、Hewittは自分の考えを共有するためにツイートしました。 私はジョーに記事にそれらを入れることができるかどうか尋ねました、彼は答えました、「もちろん、なぜ。」。 多くの人が知っているように、HewittはAppStoreユーザー契約との競合により、iPhoneアプリケーションの開発を停止しました。



本日、ジョブズの手紙によって開始されたツイッター上の反Flash宣伝の波を見て、ヒューイットは言った：



-W3CのFlashに対する不満に対処します。これは、標準自体の弱さのために開発者がプラグインを使用せざるを得ない遅さでした。



-さらに、ブラウザが「非標準」の革新的なAPIを提供するたびに、開発者の皆さんに苦痛を感じます。



-ブラウザメーカーは、配布後にW3Cが標準化できるカスタムAPIを提供する必要があります。 W3Cからの革新への期待は死のようです。



したがって、ヒューイットは、Flash（およびその他のプラグイン）が存在するのは、W3C（Web Standards Administrative Board）が新技術の策定と採用が遅すぎるためだと考えています。 彼は、ブラウザーメーカーにW3Cの制限から脱却し、新しいAPIの実験を開始するよう奨励しています。



その後、彼はMicrosoftを擁護し始め、Microsoftは（米国政府を含む誰もがこれは不正な方法で行われたと考えていますが）彼がキャリアを始めたJoe（Netscape）を獲得しました。



-10年前、Microsoftを襲ってIEの開発を禁止し、手綱をW3Cに移しました。 これは何から来たのですか？



-覚えるには若すぎる人たちのために、DOJ（Department of Justice）とW3Cが邪魔をしている限り、IEは4.0から6.0の途方もないペースで成長しています。



-さあ。 ActiveXに加えて、1996年から2000年にかけて、Microsoftは他のブラウザメーカーよりも速くWebを開発しました。



-MSがIEを放棄した理由はわかりませんが、Web開発者が彼らに革新をやめ、単にW3C標準に従うように頼んだことは知っています。



ヒューイットは、Microsoftに対する独占禁止法がInternet Explorerの開発を停止したと考えています。 1996年、JoeがIEの真の革新が始まったばかりだと言ったとき、このブラウザは10％の市場シェアさえ持っていませんでしたが、Netscapeはほぼ90％でした。 そして、キャッチアップとして、IEは革新する必要がありました。 彼がWeb全体をキャプチャし、Microsoftが馬に乗って製品を放棄するまで。

その後、Hewittはオンラインストア（はい、Apple AppStoreを含む）の開発に注意を払います。



-なぜオンラインストアがウェブを脅かし、私のような開発者を怖がらせるのですか？ 「邪悪な」独自の技術は、ウェブをばらばらに破壊します。



-私は再びウェブ開発者になりたいと思っていますが、2020年まで待たなければならない場合、ブラウザが最終的に2010年にCocoaができることを提供できるようになるまで、そして何が起こるのでしょうか。



「「悪」の専有技術はウェブをばらばらに密輸します。」非常に反抗的な引用です（記事のタイトルでそれを使用しないことを今でも控えています）。 繰り返しになりますが、Hewittは、WebがCocoaのような非Webテクノロジーの機能にまだ到達していないと主張します。今後10年には到達しません。



-マイクロソフトは、非標準のhtml / javascript / css拡張機能の特許を取得し、他のブラウザーがそれらを実装できないようにしましたか？



「[W3C]がその標準を規定していないのは事実ですが、開発者は非標準のAPIを使用する人々を恥じています。」 それが問題です。



彼はここで、Microsoftの標準のいくつかが委員会（W3C）で採用されなかった理由を主張しますか？ その後、彼は、Web開発者を、管理委員会によって承認されていない技術を使用している他の開発者を批判したとして非難します。



-Webアプリケーションの作成に戻るためには、Cocoaを忘れなければならないことにthatしている。 私がどれほど深刻な機会を失う必要があるのか​​がいらいらしています。



-どうあるべきか：ブラウザーはさまざまな革新をもたらし、ユーザーが最善を選択します。その後、W3Cはユーザーの選択を標準化し、敗者は欠落したAPIを導入します。



これがヒューイットの議論の本質です。 Webテクノロジーの開発は十分に速くありません。なぜWeb標準を満たすために、それほど強力でない言語を使用する必要があるのでしょうか。 繰り返しになりますが、彼はブラウザーが革新を開始し、W3Cがこれらの革新を受け入れることを望んでいます。



-Cocoaの機能が気に入っています。UIのプログラミングにCのような言語が好きではありません。



HewittがCocoaを欠いている理由は、たとえそれが最高のWeb言語であると思ったとしてもです。 Cocoa iPhoneがiPhoneアプリを作成することを意味します]。



-ユーザーは、特定のブラウザーテクノロジに対してサイトをさらに開発する開発者を増やすかどうかを選択する必要があることを理解します。 開発者は実際に勝者を選択します。





また、開発者がトレンドを逆転させるために積極的な措置を取ることを奨励します。



-IE6は2000年にはかなりクールでした。 最新のブラウザと比較するのは公正ではありません。



IE6を批判する人々への注意-それが出てきたときに何が起こったのか覚えておいてください。



-残念ながら、現時点では[Web言語ではなく] Cocoaをお勧めします。 言わないでおきたい。



-Cocoaは、完璧ではありませんが、Web言語よりも優れています。



-それで、別のブラウザを作成します。 くだらないこと。 些細なことではないことを知っていますか？ AppStoreごとに新しい電話を購入します。



興味深い考え。 ヒューイット氏は、さまざまなブラウザ向けにWebをプログラミングし、さらに多くのhemo核-さまざまなAppStoreのアプリケーションをプログラミングするのは面倒だと言います。



-私はActiveXを非難しませんが、IEのhtml / css / javascriptはかつて非常に高いレベルでした。



再び、当時のIEを保護します。



ここから始めて、HewittはWebアプリケーションとネイティブアプリケーションのトピックについて熟考します。



-はい、正確に。 開発者がユーザーに希釈されたクロスブラウザサイトを作成するのではなく、異なるブラウザを起動するように強制した方が良いでしょう。



-モバイルWebでこのようなことが発生します。モバイルWebでは、AppStoreがブラウザを展開せず、ブラウザを置き換えますが、ネイティブアプリケーションはWebアプリケーションとは対照的です。



-「Xでベストビュー」-ネイティブアプリケーションの開発のためにWebを腐敗させて以来、「別の電話を購入する」ほど悪くはありませんでした。 iPhone用の本格的なネイティブおよびWebアプリケーションを開発しようとした人は、iPhone SafariがCocoa iPhoneと比較して冗談であることを知っています。



Hewittは、プログラミング言語が優れているという理由だけで、AppStoreがWebに取って代わったと考えています。 そして、ウェブが腐敗しているのはあなたのせいであり、許可しました。 彼の最新のツイートは特に強力です。Hewittは、ネイティブテクノロジーとWebテクノロジーの両方で豊富な経験を持ち、iPhone Safariはネイティブアプリケーションでできることと比較して「冗談」だと考えています。 これは、多くの開発者が傍観者にささやいていることですが、率直に話すことはめったにありません。



最後に、ヒューイットは彼の声明のいくつかを明確にします。



-私はFlashに残酷でしたが、W3C、Microsoft、Mozillaによって開始されたリレーレースの10年間の継続について、Macromedia / Adob​​eに感謝しなければなりません。



-実際、WebKitチームがそれほど有能でなければ、それはくだらないことになるでしょう。 それだけです、外に出て、聞いてくれてありがとう。
