Ubuntu用DrWeb

友人Vadik別名keyは、彼が実装したアイデアを投げ出し、私は彼をフォローしました。 たぶん誰かもそれを好きになるでしょう。 このアイデアは、無料のDr.Web LiveCD製品からDr.Web Anti-virus for Linuxにdrweb32.keyキーを簡単に追加することです。



必要な計画を実施するには：

1） www.freedrweb.com/livecdで Dr.Web LiveCD（〜64mb）をダウンロードします

そこからdrweb32.keyを掘り下げる必要があります。原則として、ディスクに書き込み、そこから起動し、キーをUSBフラッシュドライブにコピーできますが、怠zyなものはバイパスします。

Archive Managerを使用してUbuntでminDrWebLiveCD-5.0.0.isoイメージを開き、/ boot / module /ディレクトリに移動して、white.moを引き出します。



2）ターミナルにwhite.moをマウントします

#mount -o loop -t squashfs white.mo /tmp/





そして、パス/ tmp / opt / drwebに従って、drweb32.keyを取り出し、アンマウントします

#umount /tmp/







見ますが、キーは編集しません

Expires=2010-03-07 (15:14) UTC





2010年！ 医者が注文したもの！



3） download.drweb.com/linuxでOSのバージョンをダウンロードし 、Ubuntu 8.04のバージョンを取得しました。 ダウンロードしたすべてのファイルを1つのフォルダーに格納します。32ビットのディストリビューションを使用している場合は、蟻塚に棒を付けたテナガザルのように、マウスでファイルを1つずつdebファイルに挿入できます。 私は64ビットシステムを持っているので、その中の端末も呼び出しました

#dpkg -i --force-architecture *.deb





アーキテクチャの不一致を呪って、パッケージが配信されました。



4）以前に受け取ったdrweb32.keyキーを/ opt / drwebディレクトリにコピーし、xdrwebを呼び出します。デスクトップに起動するショートカットを作成できます。



5）Dr.Webスキャナーでは、最終更新時刻と合計ウイルスシグネチャを評価した後、[サポート]タブに移動し、[更新]をクリックすることができます。



6）Dr.Webの品質作業を確認した後、ストアestore.drweb.comにアクセスし、登録されたキーを購入して、国内メーカーをサポートします。



質問は「なぜ？」

A：なぜLinuxのウイルス対策ですか？

A：Linux用DrWebを使用して、USBフラッシュドライブまたはWindowsウイルスのP2Pで共有されているディレクトリを確認できるようになりました。 高度なLinuxoidがテスト用ではなく、それ自体にWindowsウイルスを格納するのは良くありません。



A：なぜDr.Webなのですか？

A：同意します！ より良いクラマフ、さらに良い「1つの頭が良い、2つの頭がさらに良い」。
