NETGEAR WNDR3800ルーターのレビュー

長い間、私はノームのルーターを苔で生い茂ったものを、よりサラブレッドで技術的でシンプルなものに交換したいと考えていました。 将来のデバイスについては、倒錯した要件はありませんでした。つまり、10台中8台のルーターがおそらく私に合っているでしょう。今日は、8台のうち1台についてのみ検索を停止しました-NETGEAR WNDR3800誰も言わないでしょう。







現在、家庭向けソリューションの選択肢は非常に多くなっていますが、...

...しかし、当初は、その性能特性に依存して、主にルーターを「味と色」にしました。その結果、成長するホームインフラストラクチャと成長する可能性のある外部チャネルに対して、将来的に「十分」であることが保証されました。



このデバイスの外観は、その前身と、特に「弟」WNDR3700に似ています。Boomburumが詳細に語っています。多くの有用な情報とスクリーンショット（実際に購入したもの）があるため、レビューをご覧になることをお勧めします。 しかし、デバイス間の違いの規模は、私の投稿をご覧ください=）



印象は、実際には、多くも少なくもあります。 デバイスとして、ルーターは現代のコミュニケーターの詰め物や現代のビデオカードの速度のように私を決して襲いません。 しかし、ルーターのように、この仕掛けは100個すべてにとって印象的です-現在、少なくともNETGEAR兵器庫からは、家庭で使用するための最も先進的で強力なものの1つです。



豊かな構成、金色の蓋、革のトリム、狭いWi-Fi波を使用したスーププレートのリモートヒーティングの問題でもありません。 これは、これが私の記憶の最初のルーターであり、「sel-pahehal」スタイルの普通の人向けに作られたものであり、多くのツールと「最初に組み立ててみよう」というモットーを持つコンストラクタではありません。 RT-Nシリーズを搭載したASUSでさえそれほど印象的ではありませんでしたが、今ではそうではありません。



同社は、デバイス（少なくともルーター）のラベル付けの問題を変更しました。WNDRXX00のような不明瞭なインデックスから、もう少しわかりやすいNXXXに切り替えました。 たとえば、現在、N150、N300、N450、N600、N750、N900があります。最後の3桁は、ワイヤレスの最大理論データレートを示しています。 マーケティングがないわけではありません-両方の帯域（2.4 GHzと5 GHz）の速度を要約しています。 したがって、配置は次のとおりです。

次のような構成から始めましょう。ルーター自体、電源アダプター、パッチコード、説明書+ソフトウェアCD、垂直取り付け用の特別な「スタンド」。







特に、「 プレミアムエディション 」という碑文が箱に表示されているため、それほど厚くありません。







スタンドは、ルータを「水平」にしたり、壁にネジ止めしたくない場合に、ルータを垂直に立てるのに役立つトリッキーなデバイスです。











セルフタッピングねじ用の穴があるため、オプションで壁に取り付けられます。 合計-3つの信頼できるインストール方法。







ルーターは、Burumの投稿のデバイスとほとんど同じように見えますが、配色が異なる点を除きます：インジケーター領域には黒ではなく赤い境界線があり、数字はわずかに異なります（電源、2.4 GHzおよび5 GHz、USB、WAN、4xLAN、Wi-FiおよびWPS）。 カバーは同じで、側面は同じです。







ポートには、5つのRJ-45コネクタ（4xLanと1つのWAN）があります。それらはすべて、ギガビット、フラッシュドライブまたはプリンタ用のUSBポート（バージョン2.0）、電源「穴」、電源ボタンです。 USBポートが1つしかないことを心配しないでください。USBハブを使用できます。







充填は多少異なります-WNDR3700では8 MBのメモリがあり、WNDR3700v2では16 MB（および5 GHz帯域用の修正されたアンテナ）でしたが、ここ（N600では）でメモリサイズは既に最大128 MBまで増加しています。 内部では、N600はAtheros AR7161 MIPSプロセッサー（680 MHzのクロック周波数で動作）を採用しています-WNDR3700v2と同じです。 残りはほとんど双子の兄弟です。



背面には、いくつかの技術情報（WANポートのMACアドレス、シリアル番号、コネクタ署名、管理パネルにアクセスするための工場ログインおよびパスワードなど）と、工場設定にリセットするための赤いボタンのある穴があります管理パネルでスマートすぎるか、デバイスがクラッシュした場合（これに遭遇するまで）。



控えめで不要なフリルなし。 外部アンテナは単に接続することはできませんが、デバイスには実際には必要ありません。内部（各範囲に2つ）がすべてのルーブルを消費します。 デバイスはMIMOテクノロジーの式を使用します-各範囲に2Tx / 2Rx、アンテナは各ワイヤレスユニットの近くのボードに直接はんだ付けされます-必要に応じて、外部アンテナはMMCXコネクタ（各ワイヤレスコントローラーに2つ）を介して接続できます。 スイッチとして、RTL8366SR（ローカルおよびワイヤレスネットワークを担当）が使用され、Atheros AR9220（2.4 / 5 GHz）とAtheros AR9223（2.4 GHz）がワイヤレスコントローラーの役割を果たしました。 ただし、これはすべて、深く浸透しているファン向けです-ストックの機会は人々にとって簡単です=）



指示は愚か者向けであるため、電源、WAN、LAN、開始キーを挿入します...ルーターが最初に提供したのは（標準のユーザー名とパスワードを入力し、基本設定を入力した後）、ファームウェアを更新することでした。 さて、ネイティブ1.0.0.16は良いのですが、デバイスは更新を切望しているので、彼の渇きを癒してみませんか？ 私の場合、ファームウェア1.0.0.24が利用可能でした（そして、それとともに1.0.0.80から1.0.0.84のGUI /言語モジュール）-更新には1分もかかりませんでした。 この記事の公開時点では、ファームウェア1.0.0.32はすでに利用可能でしたが、その中には革新も変更も気づいていませんでした。



新しいインターフェイス（Genieと呼ばれる-現在、すべての新製品に搭載されています）は多くの点で古いインターフェイス（設定の階層が似ています）に似ていますが、とにかくずっと良くなりました。 ただし、WNDR3700管理パネルの外観が新しいファームウェアのいずれかで変化していることを除外しません。 あなたはそれを楽しいと呼ぶことはできませんが、それは見えます（ロシア語は、もしあれば）です：







a）シンプル、b）真剣、c）アクセス可能、d）過負荷ではない; 他の多くのデバイスの管理者には言えません：同じASUSの場合、すべてが率直に言って...奇妙な配色ですが、多少便利ですが（ただし、「リロード」ボタンの横にある厄介な丹毒がアクセスをブロックし、顔を指す（これは十分に頻繁に発生します）だけで推測されます）。 D-linkにはすべてがあります：小さなフォント、最も普通のアクションで頻繁に再起動、2000年代の初めからとんでもない混乱、不格好な翻訳と外観...残りは原則としては良くありません（異なるルーターの管理画面のスクリーンショットをグーグルで検索できます） ）







MCCのルートには一種のダッシュボードが表示されます。接続に関する情報、ワイヤレスネットワークに関する情報（パスワードが平文で書かれているのは奇妙です）などが表示され、インターフェース（現在の時間に慣れている）は2つのセクションに分割されました： 「そして」高度な-それらの設定はほぼ同じであり、2番目のセクションではかなり多くの設定があります。 それ以外の場合、以前と同様に、設定はセクションにグループ化され、セクションはグループ化され、サイドメニューで破棄されます。 一般に、それは非常にしっかりしていて、すてきで快適です。そして最も重要なことは、主婦ですらそれを理解するでしょう。



「自宅でインターネットが機能するように」状態にルーターを設定する（ほとんどすべての一般的な接続タイプがサポートされます：静的IP、DHCP、PPTP、L2TP、およびPPPoE）には2〜3分かかります（自動設定後、認証データを入力するだけで十分です）帯域幅の割り当てとWi-Fi保護、ゲストアクセスのセットアップなどの詳細な構成には、少し時間がかかります。 ただし、セットアップ自体はまだシンプルです。何も探す必要はなく、すべてが論理的で目に見えるものです。







最初に、ルーターには既にワイヤレスネットワークが構成されています（デフォルトでは、組み込みのアクセスポイントが古い802.11a / b / gプロトコルと連動し、20または40 MHzのチャネル幅の自動選択を行う場合、組み合わせた動作モードがありますが、各無線モジュールは個別に構成できます） 、さらに、接続に必要なすべての情報（はい、SSIDとパスワード）をプリンターで印刷する準備ができています。 まず、これらのパラメーター（デバイスの背面に記載されている管理者パスワードなど）を変更してから、タブレットをWi-Fiに接続し、speedtest.netにアクセスしました。 奇妙に思えるかもしれませんが、Wi-Fi接続の速度はプロバイダーが指定した速度よりも劣っていません（つまり、ルーターは「強制終了」しませんでした）。コンピューターとメディアセンターでのデータ転送テストは、「古典的な」方法で接続された正しい選択-速度とワイヤーは価値がありました。



原則として、ルーターに必要なもの（見栄えの良い、管理パネルはシンプルで、ワイヤレス接続の速度は良好です）から、すべてをすでにお伝えしました=）したがって、彼がまだ何をするかを簡単に説明します。







組み込みのアクセスポイントは、WPA / WPA2 / WPA-PSK / WPA2-PSK、WEP（64、128、および152ビット）およびWPSのすべての必要なワイヤレスデータ暗号化テクノロジーをサポートします。これらのデバイスのフロントパネルには個別のボタンがあります。

通常のtelnet / sshが見つからず、LANポートの1つをWANのブリッジとして使用する機能もありません。これにより、プロバイダーのセットトップボックスを使用してIPTVを視聴したい場合に不要なスイッチを省くことができます（ちなみに、これは単なる「ホーム」オプションです低価格セグメントを含む多くのルーター-たとえば、同じASUSとDLink）。 明らかに、マルチキャストトラフィックはIGMPプロトコルを介してプロキシできます。 Clear Channel Selectorテクノロジーのサポートがあります（最近までは、最も高価なデバイスでしか見つかりませんでした）-最もトラフィックの少ないWi-Fiチャネルの自動および動的選択。
