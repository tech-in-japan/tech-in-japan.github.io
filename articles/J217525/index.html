この情報を理解しやすくするために、メモリカードの名前、コード、製造国、プラスチックボックスの存在を1つのテーブルにまとめました。



私の個人的な経験に基づいて、メーカーが宣言した32 GBのカードサイズにもかかわらず、異なるカードで利用可能なボリュームはわずかに異なると以前は想定していました。 したがって、すべてのメモリカードをNikon D7100カメラでフォーマットしました。 ディスクプロパティとAIDA64テストパッケージを使用して、使用可能なボリュームを決定しました。



受信したすべての情報も集計されました。 データが平均を下回るセルは赤で、データが平均を上回るセルは緑でした。 最大値と最小値は太字でさらに示されています。 この設計は、以下のすべての表に有効です。



トランセンドアルティメットメモリカードは最大の空き容量を提供でき、東芝SD-T032UHS1カードは最小です。 差は947 MB​​で、これは相対的な意味で使用可能な容量の約3％です。 Nikon D7100カメラなどのRAW写真の標準では、これは最高品質の30枚を少し超える写真に相当します。 同意します。1つの宣言されたボリュームのカードを購入するとき、そのようなボリュームを失うことはあまり快適ではありません。



メモリカードの速度特性を調べるために、次のテストパッケージを使用しました。

•AIDA64エンジニアエディション

•ATTOディスクベンチマーク

•CrystalDiskMark



テストは、ASUS Maximus V GeneマザーボードのUSB 3.0ポートに接続されたTranscend TS-RDF8Kカードリーダーで実行されました。



クリスタルディスクマーク



順次読み取りテストでは、カードを2つのカテゴリに分けます。 最初の読み取り速度は約90 MB /秒、2番目は50 MB /秒未満です。 最初のカテゴリ内では、グループ内で最低速度、つまり〜77 MB / sを示したキングストンとメモリカードを個別に区別できます。

なぜちょうど2つのカテゴリーなのか？ 最大データ転送速度が最大104 MB / sのUHS-I仕様にもかかわらず、SDR50とSDR104の2つの標準があります。 したがって、SDR50のインターフェイスの最大データ交換レートは50 MB /秒ですが、SDR104の場合は104 MB /秒です。 したがって、SanDisk ExtremeおよびToshiba SD-T032UHS1メモリカードは、他の全員がUHS-I SDR104に属している場合、UHS-I SDR50に属します。



連続記録テストの結果は、2つのカテゴリに分類することもできます。 同時に、Kingston SDA3、SanDisk Extreme Pro、Silicon Power Superior、Sony SF32UX、Toshiba Exceria Type 2、Transcend Ultimateなど、読み取り速度の速いほとんどのカードは、高い書き込み速度を示しました。 Kingston Ultimate、OltraMax OM032GSDHC10、およびTranscend Premiumカードは、シーケンシャル書き込み速度が読み取り速度よりも大幅に遅いのに対して、先頭グループから2番目に飛びました。

順次書き込みテストで読み取り操作の平凡な結果を示したメモリカードは、誰も驚かなかったでしょう。



サンディスクエクストリームプロカードは、CrystalDiskMarkテスト結果のトップ、そして東芝SD-T032UHS1のアウトサイダーとなりました。



ATTOディスクベンチマーク、読み取り （クリック可能）

テストは、256 MBの最小タスクサイズ、0.5〜8192 KBのブロックで実行されました。



力の調整は、CrystalDiskMarkテストで以前に取得した結果と完全に一致しています。 同時に、このテストでは、メモリカードが小さいファイルと大きいファイルの操作をどの程度適切に処理するかを評価できます。

2つのカード（SanDisk Extreme ProおよびToshiba Exceria Type 2）が、データブロックのサイズに関係なく、平均以上の結果を示すことができたことは注目に値します。 同時に、32Kマークの後、SanDisk Extreme Proがリードするとき、Toshiba Exceria Type 2は小さなファイルでより良く機能します。 もちろん、マルチメディアファイルを扱うには、大きなファイルを扱うためにカードを最適化することがより重要です。



ATTOディスクベンチマーク、記録 （クリック可能）



書き込みテストの結果では、データブロックに関係なく、1枚のカードで他のカードよりも高い結果を示すことはできませんでした。 キングストンSDA3は他に類を見ないほどこれに近いものでしたが。 このテストでは、最大の結果を達成するためにKingston SDA3およびSanDisk Extreme Proカードに注目できます。 Silicon Power Superior、Sony SF32UX、Toshiba Exceria Type 2、およびTranscend Ultimateカードはもう少し先に進み、Kingston Ultimate、OltraMax OM032GSDHC10、SanDisk Extreme、Toshiba SD-T032UHS1、およびTranscend Premiumによって結果が完成します。



AIDA64 Engineer Edition 、 線形読み取りおよび書き込み



AIDA64 Engineer Editionの線形読み取りおよび書き込み結果は、他のテストで以前の結果を再度確認します。 したがって、それらの説明については詳しく説明しません。 テストのリーダーにとって、書き込み操作の速度は読み取り操作の速度とほとんど同じであることに注意してください。



AIDA64 Engineer Edition 、 ランダム読み取りおよび書き込み



ただし、ランダムな読み取りと書き込みを再度テストすると、パワーのバランスが多少変わります。 書き込みの結果が読み取りに匹敵するカードは、SanDisk Extreme ProとToshiba Exceria Type 2のみです。他のリーダーでは、ランダム書き込み速度は、シーケンシャル記録と比較してほぼ2倍低下しました。 もちろん、ランダム記録はシーケンシャル記録ほどメモリカードにとって重要な特性ではありませんが、この事実を無視することはできませんでした。



AIDA64 Engineer Edition アクセス時間



アクセス時間もメモリカードの主要な機能ではありません。 ただし、多くのタスクでは、決定的な役割を果たすことができます。 したがって、このパラメーターでテストすることにしました。

ランダム読み取りアクセス時間がすべてのカードでほぼ同等の場合。 記録のランダムアクセス時間はさらに異なり、最高のカードと最悪のカードの差はほぼ10倍です。
