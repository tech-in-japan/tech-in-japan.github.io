Windows LiveはLJとYandexになりました

YandexやLivejournalなど、20を超える世界最大のオンラインサービスがWindows Liveサービスアフィリエイトプログラムに登録されています。 現在、Microsoftの発案者は、ユーザーhome.live.comのスタートページのコンテンツをそこから抽出する道徳的および法的権利を持っています。



Windows Liveは「オンラインライフの中心」として位置付けられていますが、実際には、すべてのページのコンテンツをユーザーが登録されているすべてのサービスに結合する別の普遍的な「更新フィード」です。 「友人のさまざまなお気に入りのサイトにアクセスして、さまざまなソーシャルネットワークでの生活を知る必要はもうありません。すべてがWindows Liveに統合されています」 。



SUPの経営陣はまた、この点で喜びを表明し、近い将来、マイクロソフトとの新しい共同プロジェクトを発表することを約束します。
