Else Mobile：「他のことを試してみる時間」？









美しい未来的なガジェットの写真のため、この投稿への注意を遅らせたに違いありません。 はい、同じことが私に起こりました。 しかし、それは何ですか？ ただのレンダリング？ 別のコンセプト？ 誰かの夢？ いいえ、これは私があなたに伝えたい新しいデバイスです。





イスラエルの会社Emblaze Mobile（リリースされていないスマートフォンEdelweissで知られている）は、最近、ロンドンで新しい発想のElse Firstを発表しました。 したがって、これはAccess Linuxプラットフォームで実行されるコミュニケーターです。 最初に目を引くのは、インターフェイスそのものです。 彼の演説の中で、アミール・クペルバス（エンブレイズ・モバイルのCEO）は、もちろん「直観的」かつ革新的であると述べました。 プロトタイプと写真付きのビデオを見ると、インターフェイスを少し低く評価できます。



仕様書



  • 寸法：13 x 56.6 x 115.6 mm
  • 画面：LCD 3.47″ FWVGA 480×854、262,000色
  • プロセッサー：TI OMAP 3430
  • 標準：WCDMA / HSDPA / GSM / GPRS / EDGE / 3G
  • Bluetooth 2.0、802.11 b / g、GPS
  • 内蔵フラッシュメモリ16GB、256MB RAM
  • カメラ：5メガピクセル、オートフォーカス、8倍デジタルズーム、画像安定化
  • 3.5mmヘッドフォンジャック
  • 変位、傾斜、光センサー
  • 1450 mAhバッテリー




さらに、ソフトウェア、さらにはSDKを備えたストアが約束されています。
