モブリンの行き止まり

Eee Keyboard PCについてはすでに書いています。EeeKeyboard PCは、Intel Atomプロセッサーをベースにしたマイクロコンピューターを内蔵した新しい概念のワイヤレスマルチメディアキーボードです 。 Eee Keyboard PCは、Windows XPオペレーティングシステムが搭載された完成品として発売されようとしています。 ただし、EeeキーボードPCに関しては、ますます興味深い詳細が登場しています。 キーボードはMoblinバージョンにも存在することがわかりました。







MoblinはMob ile Lin uxであり、LinuxモバイルOSの特別なバージョンを開発するプロジェクトであり、Intelがほぼ完全に監督しています。 Moblinプラットフォームの目標はGoogle Androidの目標とほぼ同じです。Androidが電話のOSであれば、MoblinはいわゆるMID（モバイルインターネットデバイス）、モバイルインターネットデバイス。これは、Intelが数年にわたって推進してきたアイデアであり、現在、数十のメーカーがIntelの開発と構築を進めています。







Moblinは1年以上にわたって、多くのラップトップおよびネットブックで正常に動作しています。たとえば、FoxconnラップトップおよびInventechスマートフォンは、前回のComputex展示会で展示されました。 すべての新しいデバイスと同様に、これらは文字通り今春登場したMoblin 2プラットフォーム上に構築されています。 Intelの従業員が作成したオペレーティングシステムは、 Intel Atomプロセッサで動作するように特別に設計されており、主にデスクトップコンピューター用に設計された典型的なオペレーティングシステムを実行するデバイスよりもわずか数秒でロードし、消費電力が少ないように最適化されています。 そのため、Intel AtomプロセッサをベースにしたMoblinオペレーティングシステムを搭載したASUS Eee Keyboard PCのバージョンがあります。







高度な省エネ機能に加えて、Moblinオンボードキーボードバージョンには別の重要な利点があります。Moblinは、わずか数秒でロードできる非常にモバイルなオペレーティングシステムです。
