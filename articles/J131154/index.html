Jelastic Cloud PaaS-MySQL、MariaDB、PostgreSQL、MongoDBデータベースの使用に関する統計

ユーザーのリクエストに応じて、 Jelasticプラットフォームでのデータベースの使用に関する最初の統計を投稿します。 統計は最も簡単な方法で収集されました。 すべてのユーザー環境で指定されたデータベースの数を計算しました。 統計サンプルでは、​​1000を超える環境が考慮されます。







統計に基づいて、MySQLは現在リーダーですが、他のデータベースも非常に人気があります。



地域ごとのデータベースの分布 ヨーロッパおよび北米。







要約表

ヨーロッパ 北米 合計
MySQL 47％ 40％ 46％
MariaDB 13％ 16％ 14％
PostgreSQL 15％ 15％ 19％
モンゴッド 20％ 28％ 21％


結果は、グローバル市場での分布を正確に表したものではありません。 これは、Jelasticプラットフォーム内の分布統計です。



このような統計レポートを毎月投稿する予定です。 また、すぐにウェブアプリケーションサーバーで統計を公開します。

これらの統計情報にまだ貢献していない場合は、 Jelasticプラットフォームで環境を作成することですぐにそれを行うことができます。
