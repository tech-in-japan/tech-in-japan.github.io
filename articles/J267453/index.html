ウェビナー：OpenMPを使用したC / C ++プログラムの並列化の基礎







こんにちは、Habr！



FlyElephantチームは引き続きウェビナーを実施しています.9月28日17:00に、OpenMPを使用したC / C ++プログラムの並列化の基本を確認し、FlyElephant機能に精通し、例を使用してプラットフォームを操作する方法を学ぶウェビナーに皆さんを招待したいと思います。 ベータテストプログラムと新しい機能についてお話しましょう。これらは近い将来利用可能になります。



これとは別に、ウェビナーの一環として、プロジェクトがプラットフォームを無料で使用できるようにする特別なオープンな研究支援プログラムを発表しています。



こちらからウェビナーに登録できます 。



FlyElephantは、科学計算とデータ管理のためのプラットフォームです。 単一の場所で、さまざまなコンピューティングリソース（マルチコアシステム、HPCおよびGPUクラスター）、プログラム、テンプレート、アルゴリズム、およびデータを利用して、コンピューティングの問題を迅速かつ効率的に解決できます。 また、ユーザーはプロジェクトで共同作業し、計算結果の変化を追跡し、結果のソリューションをパブリックまたはプライベートクラウドサービスとして公開できます。



これで、プラットフォームがテストモードで起動され、コア数が1から32、RAMが最大448 GBのサーバーでOpenMPをサポートするC / C ++で記述されたコンピューティングプログラムを実行できるほか、ストレージ内のデータを操作したり、計算結果を表示したりできます。



コメント、提案、コメントは私たちにとって非常に重要です。 この投稿のコメント、またはsupport@flyelephant.netのメールで回答させていただきます。
