ワンクリックでクラウドPHPホスティング！

Jelasticプラットフォームが「ポリグロット」になりつつあることをお知らせいたします。2012年の第4四半期には、PHPクラウドのローンチが計画されています。 Jelastic PHPサービスのユニークな機能は垂直スケーリングです。大規模なサーバーに過剰な支払いをしたり、リソース不足に苦しむ必要がなくなります。 この機能のおかげで、負荷が増加すると、アプリケーションは必要なリソースを自動的に受け取りますが、逆にそれぞれ減少します。 これはクラウドホスティングへの本当に新しいアプローチです：プラットフォームは非常にシンプルで使いやすいので、初心者でも簡単にアプリケーションをデプロイできますが、サポートされる機能は非常に幅広いため、最も高度なPHP開発者でもプラットフォームを好むでしょう。



アプリケーションのライフサイクル全体（デプロイメント/テスト/本番）で大きな利点が得られます：アプリケーションをJelasticにロードした後、コードを変更することなく本番でテスト、スケーリング、デプロイでき、データベースを含む環境を複製できますデータをトラブルシューティングまたは修正し、アプリケーションをダウンタイムなしで更新し、チームの各メンバーの環境への異なるアクセスレベルを決定します。









次のような利点を強調する価値もあります。

  • 水平スケーリング：マウスを数回クリックするだけで、アプリケーションの最大リソース消費制限を設定できます。
  • 繰り返しますが、使いやすさ：幅広いソフトウェアスタックがサポートされているため、追加のインストールや構成は必要ありません。
  • 汎用性：任意のPHPプロジェクトを簡単にデプロイできます（サポートされている形式は.tar、.tar.gz、.zip、および.bzip2です）。
  • すべての主要な拡張機能を備えたNGINXおよびApacheアプリケーションサーバー。必要に応じて、任意のモジュールを追加できます。
  • リモートアクセスでのさまざまなSQLおよびNoSQLデータベース（MySQL、MariaDB、PostgreSQL、MongoDB、CouchDB）のサポート。
  • すべての構成ファイルへのフルアクセス。
  • PHPのバージョンを選択する機能。
  • git / svnリポジトリから直接デプロイ/更新する機能。
  • さらに、メモリを合理的に使用するための組み込みmemcachedノード。


無料のベータ版Jelastic PHPは2012年の第4四半期に発売され、商用発売は2013年の第1四半期に予定されています。 住民のフィードバックのおかげで、Javaプラットフォームの機能をユーザーが望んでいたとおりに作成しました。 そして、最初に予備ベータを開始する目標は、開発者にとって重要な機能のリクエストを受け取るために、解決したい実際の問題に関するフィードバックを収集する必要があることです。 そのため、ベータテストに参加するすべての人を招待します。 これを行うには、 php.jelastic.comにメールアドレスを残すだけで、プラットフォームの起動に関する通知を受け取ります。
