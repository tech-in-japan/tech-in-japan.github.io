Sybabasコンソール（Popcorn、BBK、eGreatなど）でNASAテレビを見る

イントロ



資金がまったく不足しているため、北大西洋宇宙機関（NASAの一般の人々）は、アメリカの納税者の「マインドシェア」を求めて絶えず戦っています。 これらの目標は、ケネディ宇宙センターのケープカンナベラルでの楽しい観光プログラム、カスタムフィルムと漫画（Fly to the Moon）、NASA自体とすべての宇宙ミッションのウェブサイト、ブロガーの軍隊、さらにミッション中に絶食するロボット自身によって提供されますTwitterで（ 今どこにいる？Travelin '2.49km / s（5577mph）。月から1,737km。T-2min#lcross ）。 NASAには独自のテレビもあり、ISSからのライブ放送をHDで視聴したり、インストールされたWebカメラからのビデオストリーミング、レビュー、記者会見、フライトコントロールセンターからのライブレポートを見ることができます。 NASAの高度に圧縮された（コンテンツではなく解像度の点で）バージョンがNASA TV Webサイトでインターネットにブロードキャストします。

しかし、時々コンピュータの悪鬼のように座ってモニターを見つめたくない（妻はこれを理解していない、それは彼らに違いはない-あなたはブログで書いて、ポルノを見、彼女のポピーのポピーアドレスをルーターの許可リストに追加し、ホームIPの問題を修正する-電話で、またはテレビを見ながら）、ビールのボトルを開けてソファーに転がり落ちて、厳しい労働シフトの後に疲れた典型的な家族の父親を描いています。



問題



HTPCの所有者には問題はないと思います-ストリームのアドレスを取得し、プレーヤーに送り込んで視聴しました。 テレビでの私のビデオは、HTPCではなく、優れたメディアデバイスPopcorn Hour PCH A110によって追い出されています。 SyabasのNMT（ネットワークメディアタンク）がすべて同じソフトウェアを使用しているため、これはBBK、eGreat、およびその他のプレーヤーのほぼ完全な類似物です。 原則として、NMTはほとんどのストリームを再生できますが、NASA TVでは待ち伏せがありました。 ストリームのプロパティを見ると、アドレスhttp：//が表示されるという事実にもかかわらず、これは実際にはmms：//ストリーム、つまりnmtが再生できず、学習しない独自のMicrosoftプロトコルです。 たとえば、MSの発案者ではないVLCは、mmsをプレイするタスクに静かに対処しているため、その位置は完全に、率直に言って、理解できるものではありません。



解決策



幸いなことに、同じVLCを実行し、mms：//からhttp：//にストリームをトランスコードするように構成できるコンピューターがネットワーク上にあります。 行こう！

  1. VLCを起動し、NASAからストリームを開きます。ctrl+ s（ストリーミング）を押し、[ネットワーク]タブに移動して、アドレス行にストリームリンクを挿入します： http : //playlist.yahoo.com/makeplaylist.dll? id=1369080& segment= 149773 、[ストリーム]ボタンをクリックします。

  2. [出力ストリーム]ウィンドウが表示されます。 httpオプションが必要です。出力グループでHTTPにチェックマークを付けます。アドレスとポートフィールドに必ず入力してください。これは、VLCがストリームを駆動するインターフェイスです。 そのネットワークカードのIPアドレス。

  3. プロファイルフィールドでは、事前に構成されたMPEG-2を選択するか、カプセル化、ビデオおよびオーディオコーデック、ビットレートなどを個別に構成できます。

  4. 便宜上、VLCには、今行った設定を実装するパラメーターの完全な行が表示されることに注意してください。 これは、コンピュータなしで実行したい場合に非常に便利なことです-たとえば、サーバーまたはルーターでトランスコーディングを実行することにより（wl500gPは理論上、そのようなタスクを処理できるはずです）。 VLCマニュアルは印象的であり、この行を自分で作成するためには、1晩以上にわたってマニュアルを吸わなければなりません。 別の便利なアプリケーションは、デスクトップにショートカットを作成することです。 VLCにショートカットを作成し、そのプロパティでエグゼキューターの後にストリームアドレスとパラメーターを追加する必要があります。

    「C：\ Program Files（x86）\ VideoLAN \ VLC \ vlc.exe」

    Playlist.yahoo.com/makeplaylist.dll?id=1369080&segment=149773

    ：sout =＃transcode {vcodec = mp2v、vb = 800、scale = 1、acodec = mpga、ab = 128、channels = 2}：複製{dst = std {access = http、mux = ts、dst = 192.168.15.100 ：8989}}

    T.O. ショートカットは、VLCへのパス、ストリームのアドレス、パラメーター文字列の3つの部分で構成されます。 次回、VLCを起動することが可能になり、彼自身がビデオのトランスコードを開始し、httpプロトコルを使用してネットワークに送信します。

  5. 次に、コンピューターがテレビを放送していることをメディア拡張機能に伝える必要があります。 これを行うには、コンテンツを含む通常のhtmlファイルであるプレイリストを作成します。



    <a href=”http://ip-of-computer:port/” vod>NASA TV</a>









    このファイルは、sambaによってコンソールにアップロードすることも、USBフラッシュドライブで実行することもできます。 さらに、リモコンで操作してテリーを見て、ディスク/フラッシュドライブに移動し、「その他のファイル」表示フィルターを選択して、プレイリストを選択します。 そこで、リンクをクリックして、woo-a-laをクリックします。
