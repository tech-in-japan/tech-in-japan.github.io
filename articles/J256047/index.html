ファルコン2リリース





待ち時間は終わりました！ Phalcon 2.0が登場しました！



1年以上の開発の後、Phalcon 2.0の最終リリースのリリースを発表できたことを非常に嬉しく思います。



プロジェクトをフォローした人は、これが偉業であることをよく知っています。



  • 彼らはまったく新しいプログラミング言語Zephir



    を作成しました。これにより、開発者はPHPの拡張機能を作成できます。
  • Phalcon 1.3.xのほとんどを書き換える必要がありました。以前と同じ機能を維持するようにして、下位互換性を保証し、移行を簡素化しました。
  • Zephir



    、開発を切り替えた後に大幅に調整され、開発者により多くの機能と機会を開発者に提供します。
  • バージョン2.0で実装された多くの追加機能があります。そのため、共同作業者に非常に感謝しています。




誇りに思う結果：

  • Phalcon 2.0拡張機能も、以前と同様にPHP（およびそれ以上）と互換性があります。
  • Zephir



    使用すると、開発者はCの知識がなくてもカスタム拡張機能を作成できます。


設置



このバージョンは2.0.0ブランチからインストールできます。まだZephir



インストールされていない場合は、次の手順に従ってください。 



 git clone http://github.com/phalcon/cphalcon git checkout 2.0.0 cd ext sudo ./install


標準のインストール方法も機能します。 



 git clone http://github.com/phalcon/cphalcon git checkout 2.0.0 cd build sudo ./install


Zephir



既にインストールされている場合： 



 git clone http://github.com/phalcon/cphalcon git checkout 2.0.0 zephir build


インストールスクリプトを実行すると、以前のバージョンのPhalconが削除されることに注意してください。



Windows DLLは、ダウンロードページから入手できます 。



バージョン2.0のインストールとアップグレードの詳細については、アップグレードガイドを参照してください。



次は何ですか



当社のリリーススケジュールでは、2か月に1回のアップデートのリリースが提供されています。もちろん、セキュリティの修正は可能な限り迅速に行われます。 目標は単純です。コミュニティが最も要求する機能を実装したいと考えています。



マイナーな変更と修正は可能な限りリリースされます。 また、コミュニティのLTSリリースモデルへの移行にも取り組んでいます。



最後に、PHP7の開発を開始します。これは私たち全員のターニングポイントであり、Zephirは第7バージョンがリリースされるまでに開発者のすべての要件を満たします。 この問題の解決には多くの準備が必要であり、まだ多くの仕事がありますが、私たちはこれまで以上にやる気があり、新しいPHPが提供するすべての利点をZephirに実装することを決意しています。
