AppCode 2.必要に応じて完了

本日、AppCode 2.0が正式に利用可能になったことをお知らせいたします。 このリリースでは、人気のあるKiwiテストフレームワークのサポートを追加しました。 新しいAppCodeには、DashとIngredientsという2つの外部ドキュメントブラウザが統合されています。



AppCodeは、新しいリファクタリング、高度なコード生成機能、コード記述とその解析の改善されたサポートなどを含む「スマート」な機能をすべて取得していなければ、それ自体ではありません。



新しいリリース機能の一部を次に示します。



  • Move / Copy / Clone 、 Pull Members Up / Push Membersなど、リファクタリング の拡張および改善されたセット 。







  • Clangコンパイラーから2,000を超えるメッセージが追加され、エディターに直接表示されます。 バッチモードでコード分析を実行すると、Clang Analyzerからの診断が表示されます。

  • コアデータのオブジェクトがデバッガーに表示されるようになりました 。

  • テストやインストゥルメントを実行する機能など、 iOSデバイスのサポートの改善 。

  • さらに多くのコード生成オプション ：if -respondsToSelector：チェックでコードを簡単に囲むことができ、自動的に-IsEqual：および-hash、-description、-copyWithZone：、-initWithCoder：および-encodeWithCoder：メソッドを追加します 。







  • その他多数 ...


現在、AppCodeはXPathとEmmetをサポートし、Xcodeの最新バージョンとの完全な互換性を保証します。 IDEはさらに高速になり、バージョン管理システムのサポートが拡張され、Retinaスクリーンのアイコンが追加され、集中力が向上しました-暗いDarculaユーザーインターフェイス。



最新のリリース、 修正についての詳細を学び、最も賢いObjective-C IDEをダウンロードしてください！



レム！ 2012年3月20日以降にライセンスを購入した場合、AppCodeをバージョン2.0に無料でアップグレードできます。



喜んでプログラム！
