デザイナー向け2013をビルドします。 Windows 8.1インターフェイスの更新と、デザイナーおよびデザイナー向けの有用なレポートの概要





この比較的小さな記事では、Windowsストアのデザイナーとアプリケーション開発者に特に注意を払うために重要なWindows 8.1インターフェイスの変更の概要を説明します。 最後に、設計の対象者に役立つ（ビルド開発者にとっても、また役に立つ）ビルド会議からのレポートの選択があります。





インターフェースの概要



まず、Windows 8.1のプレビューバージョンを自分で設定していない場合は、 preview.windows.comから設定できます。 ビルド会議の前夜、WindowsチームはJensen Harrisと主な変更点の概要を説明したビデオを準備しました。







主な変更点と新機能



新しい表示モード。 デザイナーとデザイナーが遭遇する最も重要な変更の1つは、アプリケーションの表示モードに関するものです。 Windows 8で、フルモード、スナップモード、塗りつぶしモード、ポートレートモードとランドスケープモードの3つのモードが明確に強調表示されていた場合、Windows 8.1ではこれらの区分が消去されます。 これで、アプリケーションの幅は実質的に任意であり、デフォルトで最小幅は500ピクセルに等しくなりました。これは、スナップモードよりも大きくなります。 ただし、アプリケーションの作成者は、サポートされているより小さな幅を宣言できます（スナップモードであった320ピクセルまで）。







その結果、現在ユーザーにとってより重要なものに応じて、画面上に同じアプリケーションをさまざまな比率で配置でき、同時に3つ以上のアプリケーションを表示できます。



もう1つの結果は、ポートレートモードと、アプリケーションを横モードで表示する「モード」との間に違いはないが、同じサイズ（おそらく別のデバイス上）であるということです。



もちろん、レスポンシブデザインの概念のサイトで今日行われているのとほぼ同じ方法で、画面をさまざまなサイズと条件に適応させるための戦略を検討する必要があるため、デザイナーにとっては、これは追加の複雑さです。



これは、アプリケーションパネル、ヘッダー、検索バー、スクロール、ポップアップ要素とダイアログ、および設定パネルが異なる状態でどのように動作するかに特に注意を払う必要があることも意味します。



アプリケーションの幅を320pxに維持する場合、この状態では、スナップモードと同じ推奨事項が引き続き適用されます。 特に、垂直スクロール、小さなサイズのインデント、見出し、戻るボタンなど。



新しいタイルサイズ。 Windows 8.1では、小さい正方形と大きい正方形の2つの新しいタイルサイズが追加されました。 アプリケーションでは、使用可能なすべての形式をサポートできます。







小さなタイル：70x70px。 大きなタイル：310x310px。 また、正方形のタイルには3つの異なるサイズがあるため、正方形の名前の代わりに小、中、大が使用されることに注意してください。



通知は小さなタイルではサポートされていませんが、バッジ（小さなアイコン）は機能します。 大きくて幅の広いタイルの一般的な推奨事項：サポートしている場合は、静止画像だけでなく、それらをライブにします。



検索の更新。 Windows 8.1では、Windows 8のリリース以降に受け取ったすべてのフィードバックを考慮して、一般的な検索エンジンが大幅に再設計されました。







それでも共通の検索コントラクトに統合できますが、スクリプトの重要な機能である場合は、アプリケーション内にガイドラインを実装することをお勧めします。 特にこのために、新しい検索ボックスコントロールがHTMLおよびXAMLで開発されました。これは、以前に検索コントラクトと統合した方法とAPIが似ています。 コントロールは、結果のヒントと提案をサポートします。



こちらの更新されたガイドもご覧ください ： msdn.microsoft.com/en-us/library/windows/apps/hh465233.aspx



ハブ 異種要素を共通のスクロール機能を備えた単一のシステムに結合できるようにする長い要求のコントロールが、ついに実装されました。







これで、特別なコントロール（ハブ（ HTML 、 XAML ））ができました。 他の多くの新しいコントロールについては別の記事で説明しますが、別の重要なコントロールであるWebViewにも大きな更新が加えられたことに注意してください。



システムとの統合：人、イベント、目覚まし時計 。 現在、連絡先との統合に加えて、ユーザーのカレンダーと統合する機会もあります。







さらに、アプリケーションは目覚まし時計として機能できます。



概要ガイドの追加の変更については、 msdn.microsoft.com / en-us / library / windows / apps / bg182890.aspxを参照してください 。 特に、アプリケーションで必要に応じて、Windows 8.1で音声を合成する方法について学習できます。



また、Windows 8.1はUSB、Bluetoothデバイスでの作業を改善しました。 POSデバイス（バーコードスキャナーなど）、および3Dプリンターでの作業をサポートします。 また、Windowsストアで変更を確認することを忘れないでください。
