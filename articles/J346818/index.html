発音が常に混同される英語の単語

毎日、キャリアと学生は英語の単語の発音を間違え、それについても考えません。 英語の初心者にならず、正しく話すために、発音が最も困難な単語のリストを用意しました。 それらをロックしましょう、男たち！







ACAI

突然、ニューヨークにいる間に、地元のスムージーショップに立ち寄ってアサイベリーのカクテルを味わうことに決めた場合、英語を話すことはありません：ah + SIGH + berry、ACK + ah + eeまたはah + KAI 。 ストレスは常に3番目または最後の音節であることに注意してください：ah +ため息+ EE [asʌɪˈi]。



該当する

英語の形容詞「applicable」は「app-application」で始まり、強調する場所を思い出させてください：APP + lic + able ['æpl+kəbl]、ただし+ PLICK + ableではありません。



北極

最初の子音「s」にもっと注意を払う。 英語学習者はしばしばこの手紙をスキップし、結果はART + ickになります。 「南極-南極」の子音、正しいオプション：ARC-tic ['ɑː®ktɪk]-北極。



キャッシュ

いいえ、これはフランス語に由来するinな言葉ではありません。 現金+ AYを話す傾向は、「cachet ['kæʃeɪ]は記号である」という類似の単語の発音にありますが、「cache [kæworde]はキャッシュです」。



電気

どうやら、これは2016年のアメリカで最も愛されていない言葉です。 e + LEK + tor + al [ɪ'lekt（ə）r（ə）l]-選択的/選択的、決してe + lek + TOR + al。



2月

実際、今月はFEB + ru + ar + ree ['fɛbrʊəri]-2月と発音されますが、FEB + ru + reeまたはFEB + ar + reeとは発音されません。



フォルテ

音楽の達人は、この単語を+ TAYと発音できます。 また、作曲家のノートが音楽を生き生きと大声で演奏することを意味する場合は、すべてが大丈夫です。 それにもかかわらず、シートからメモを読むことがあなたの長所である場合、ここでも「フォルテ」を意味し、転写のみがすでに[ˈfȯrt]です。



フォイヤー

そして再び、フランス語を話す習慣。 ただし、foy + AYでゲストに会っても、第一印象は良くなりません。 覚えておいてください：ロビーはFOY + ur [ˈfȯi（ə）r]です。



ガラ

反対に「ay」という音を配置する必要がある単語の例。 ガライベントに参加する場合、これはGAY + luh ['geɪlə]ですが、GAL + uhではありません。



ジャンル

この単語はよく使われますが、「ジャンル-属/種/ジャンル」がZHON + ruh [ʒɔŋrə]になることを知っていますが、JEN + nerではありません。



ジャイロ

豚肉とトマトソースが付いた美味しいギリシャのシャワルマは、ジャイロスコープのようには聞こえません[ˈdʒaırə、skəʊp]。 有効なオプションは、YEE + roh ['yērəʊ]またはZHIHR + oh ['zhirəʊ]です。



ハイパーボール

ジョークはありません。hy+ PER + ba + lee [haɪ'pɜːb（ə）lɪ]という単語は、4つの音節で構成され、ギリシャ語のルーツによる「ボウル」とは関係ありません。 興味深い事実：誇張の反意語は「リトート-リトタ」と見なされ、LYE-tuh-teez [laɪ'təutiːz]と発音されます。



責任

次の法的用語の意味を混同しないでください。「責任-責任/義務」、したがって「責任-責任」、LIE + a + ble ['laɪəb（ə）l]; しかし、「名誉ˈ損[ˈlaıbǝl]-名誉/損/中傷」。



図書館

2月という言葉と同じ話ですが、大学教授や政治家の間でさえ「r」の音が連続して発音されると、多くの不便が生じます。 ただし、常にLIE + brare + ee ['laɪbr（ə）rɪ]はライブラリです。



モーヴ

2018年の流行のモーブが好きかどうかにかかわらず、MAWV [məuv]は言うが[ˈmov]は言いません。



ミーム

Grumpy Catがインスタグラムを征服するずっと前に、1976年に英国の科学者が「ミーム-文化遺産の単位」という言葉を発明しました。 私たちは発音よりもこの言葉を頻繁に読みますが、私たちは次のことを意味します。



ミスシーヴァス

スペルと密接に関連する別の発音ミス。 ただし、mis + CHEEV + i + ousのように余分な「i」を追加しないでください。MIS+ chiv + ous ['mɪsʧɪvəs]のみ-いたずら/いたずらです。



しばしば

信じられないかもしれませんが、少なくともMerriam-Websterによれば、単語中の子音「t」の発音は正しくないと考えられています。 OFF + enと比較したオプションOFF + 10は容認できず、議論の余地があると思われますが、多くのアメリカ人や他のニューティブはそう言います。



ニッチ

NEESH [niːʃ]は非常に一般的ですが、NITCH [nɪtʃ]は、特定の場所や活動分野に関して確立された人気のあるオプションです。



核

アメリカ大統領：ブッシュ、クリントン、カーター、アイゼンハワーは、この一見単純な言葉の発音に明らかな問題を抱えていました。 NEW-clear ['njuːklɪə]ではなくNEAR-clewであることが判明しました。核です。 これは、メタセシス（音節/音の順列）という言語現象によるものです。



ピザ

ペパロニとパルメザンまたはエビとモッツァレラチーズを使っていますが、英語のピザはPI + zaではなく、PEET + suh ['pi [tsə]のみです。



プレリュード

教会の短い音楽が暗示されることもありますが、最初の音節は「祈り」という言葉のように発音されません。 名詞と動詞「前奏曲」はラテン語の「前奏曲」に由来します。これは前もって十分に演奏することを意味します。 したがって、PREL + yood ['preljuːd]は序曲であり、冒頭/紹介です。



関係なく

非アクティブな人々は「関係なく」ではなく「関係なし」を使用しますが、後者を使用することにした場合、re + GUARD + less [rɪ'gredləs]のみで、[ˌɪrɪ'gɑːdləs]ではありません。



発音

ちなみに、誰もが「発音」という単語を発音しているわけではありません。発音が正しいのは、おそらく誰もが「名詞」という音節を持っているように見えるからです。 したがって：pro + NUN + see + a + shun [prəˌnʌn（t）sɪ'eɪʃ（ə）n]。



不動産業者

脳が単語を綴ろうとしている別のケースは、便宜上母音を追加することです：REE + la + tur。 そのため、たとえばイギリスで新しい住宅を探す場合、表音番号を簡単に渡すことができます。 注意してください、正しいオプションはREEL + tur ['riːltə]です。



レジーム

これは政治的選好、システム、権力の有無を指しますが、「政権」は決してRUH + geemと発音せず、RAY + geem [reɪ'ʒiːm]のみと発音します。



シャーベット

最後に、シャーベットとシャーベルトで状況を明らかにしましょう。 ソルベ（東洋の飲み物）という言葉は、トルコ語の「şerbet」とペルシャ語の「sharbat」に由来するため、最後の音節には子音「r」はありません。 砂糖と氷を加えたフルーツデザートは「シャーベット」-sor-BAY ['sɔːbeɪ]ですが、「sherbet」-SHOR-bet ['ʃɜːbət]ではありません。



ステータス

生涯を通じて「ステータス」をSTAH-tusと発音した場合、あなたは一人ではありません。 ただし、修正するのに遅すぎることはありません。STAY+ tus ['steɪtəs]。



スーツ

あなたのホテルの部屋は、おそらく作業服とは対照的に、かなり甘いです。 sue-TAYのようなことを言うことすら考えず、ただSWEET [swiːt]します。



一時的

一時的という言葉はつかの間/つかの間を意味しますが、誤って、多くはtran + ZEE + entと言いますが、TRAN + shent [ˈtrænʃənt]が必要です。



トライアスロン

トライアスロンに参加するアスリートは素晴らしいですが、彼らはこの種の競技をトライATH-a-lonとして発音せず、トライ+ ATH + len [trʌɪ'aθllən]のみと発音します。



バレット

ヒーローがよくバレットに目を向けるダウントン修道院を思い出してください。 この単語は、フランス語のval + AYよりも英国のVAL + it ['vælɪt]のように聞こえます。
