PDAをUSB WLANアダプターに変える

状況：wi-fiルーター、組み込みwi-fiアダプターのないラップトップ、windows mobileベースのPDAがあります。 質問：ポケットアシスタントをラップトップのワイヤレスアダプターとして使用できますか？ はい、できます！



最初のステップ。 Wi-Fiルーターで、TCP / IPパラメーターを設定します。

IPアドレス：192.168.0.1

サブネットマスク：255.255.255.0



ステップ2 PDAで、ワイヤレスアダプターの静的IPを設定します。

[スタート]> [設定]> [接続]> [ネットワークカード]> [ネットワークアダプター]> [CF8381PNワイヤレスカード]> [特定のIPアドレスを使用]

IPアドレス：192.168.0.2

サブネットマスク：255.255.255.0

デフォルトゲートウェイ：192.168.0.1



ステップ3 PDAで、パケットルーティング機能をアクティブにします。

ここでは、レジストリエディタ（Resco File Explorerなど）が必要です。

HKLM \ Comm \ Tcpip \ Parms \ IpEnableRouter = 1

デバイスを再起動し、wi-fiをオンにします。



ステップ4 ラップトップで：

USB同期のキャンセル：

ActiveSync>ファイル>接続設定...> USB接続を許可

PDAをUSB経由でラップトップに接続します。

PDAのリモートNDISホストネットワークアダプターでデフォルトルートを設定します。

route -p add 0.0.0.0 mask 0.0.0.0 169.254.2.1 metric 1

DNSサーバーのアドレスを設定します。

[コントロールパネル]> [ネットワーク接続]> [USB接続（Windows Mobileベースのデバイス）]> [プロパティ]> [インターネットプロトコル（TCP / IP）]> [プロパティ]>次のDNSサーバーアドレスを使用：192.168.0.1。



ステップ5 楽しんでください！
