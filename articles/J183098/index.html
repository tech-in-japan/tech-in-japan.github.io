タブレットとスマートフォンを使用してデルと連携する





過去数年にわたって観察されてきたモバイルテクノロジーの急速な発展により、多くの現代のタブレットは3-4年前に購入した平均的なラップトップに劣らず、多くの点で後者よりも先に進んでいることがよくあります。 スマートフォンの携帯電話のパフォーマンスも年々向上しており、このクラスのほとんどのデバイスは長い間、単純なダイヤラーから多機能コンバインに変換されており、ユーザーに関連する膨大な数の異なるタスクを解決できます。



数年前、Intelは企業ポリシーへの興味深いアプローチを提案しました。企業の従業員には、仕事で個人用デバイスを使用する機会が与えられました。 ほぼすぐに、このアイデアは他の主要なプレーヤーによって取り上げられ、ポリシーはBYOD（「Bring your own device」）と呼ばれました。 BYODコンセプトを導入するには、セキュリティの問題に細心の注意を払う必要がありましたが、このアプローチでは従業員の機動性が向上し、勤務時間と空き時間をより効率的に使用できるようになりました。

この記事では、BYODポリシーの実装に役立つデルのソリューションのいくつかについて説明し、コンセプト自体の機能についても説明します。





BYODポリシーについて少し



現在、ほぼすべての高校生は、強力な、少なくともデュアルコアプロセッサ、内蔵カメラ、および多数のアプリケーションを搭載した高度なスマートフォンを持っています。 学校を出て高等教育機関に入学した後、タブレットやスマートフォンのような「個人の動物」は近くに留まり、卒業までに、若者はガジェット（娯楽だけでなく勉強にも）に慣れてきます。同じデバイスを使用して仕事をすることは、彼らにとって非常に魅力的です。 現代の若者は、彼らが高校生であろうと学生であろうと、非常に速く人生をすべるので、新しい職場で働くときの機動性は彼らにとって非常に自然に思えます。



より多くのモビリティと自由は、BYODポリシーが提供するものです。これについては以下で説明します。 2013年のガートナーによる統計調査の結果によると、ロシア、ブラジル、インド、およびその他の多くの活発に発展している国々は現在、BYODトレンドの発展のピークにあります。 これを確認するには、以下に示すインフォグラフィックを表示します（一部の写真はネタバレの下に隠れています。インフォグラフィックの言語は英語です）。

次に、BYODの概念自体について少し説明しましょう。 このアプローチを実装するには、慎重な計画とセキュリティの問題に多くの注意を払う必要がありますが、個々の従業員と会社全体がこのタスクに費やした時間、お金、およびその他のリソースに対して支払う以上のボーナスを受け取ります。



BYODの実装に関連するアクティビティを計画する際に注意する必要がある問題の短いリストを次に示します。

  • エンタープライズネットワークに接続されるモバイルデバイスの数。
  • ビジネス情報のセキュリティと個人ユーザーデータの機密性とのバランスを維持します。
  • 暗号化およびクラウドソリューションの使用。
  • ガジェットの購入、デバイスの費用の一部支払い、および会社の費用負担による電話とインターネットアクセスの優先条件。
  • サポートされるデバイスのリストは、ソフトウェアとハ​​ードウェアの最小要件に関する情報で補完され、従業員が利用できるようにする必要があります。
  • デバイスの紛失または従業員の解雇の場合に手順を提供する必要があります。


BYODポリシーを使用する場合の主な利点は、従業員にとって快適な作業環境とリモートで作業できることです。 また、ガジェットの購入とそのメンテナンスのコストを削減することで、かなりのお金を節約できます（ただし、デバイスの紛失や故障の場合、特定の数のデバイスを「ホットスワップ」に利用できるようにする必要があります）。 多くの企業の経験から、提案されたコンセプトの実装により、同様のデバイスをセットアップする際に従業員が互いに積極的に協力することで、サポートサービスをオフロードすることも可能になります。 追加コストのうち、BYOD戦略の開発、ITインフラストラクチャの追加要素の導入、およびモバイルデバイスの予備の維持に注意することができます。



BYOD企業ポリシーを導入するためのオプションの1つは、サポートされているデバイスのリストから従業員が自分に適したデバイスを選択できるようにすることです。 デルは、特にラップトップ、 LatitudeおよびXPSシリーズのラップトップ、ウルトラブック、タブレットなど、新しいコンセプトに完全に適合する幅広いデバイスを提供しています。 ブログで、Sergey VillanovはすでにウルトラブックXPS 12 、 XPS 13 、およびDell Latitude 10タブレットについて語っています。 これらのデバイスは、デルの他のソリューションと非常によく統合され、さらに「独自のデバイスを持ち込む」戦略の利点を最大限に活用します。

Wyse PocketCloud-クラウドにファイルを保存する機能を備えた、異なるクラスのデバイス間の安全なデータ交換







デルは、モバイルガジェットからWindowsおよびMac OS Xを搭載したコンピューターまたはラップトップにリモートでアクセスできる、いくつかのエレガントなソリューションを提供しています。 そのようなソリューションの1つがPocketCloud Remote Desktop Proです。 このアプリケーションのおかげで、ファイルと画像の両方、およびコンピューターにインストールされているプログラムにリモートアクセスできます。



iOSおよびAndroid用のプログラムの利用可能なバージョン（少なくともバージョン2.1）、Windows 8、Windows 7、Windows Vista、Windows XP、Windows Server 2003、Windows Server 2008、Mac OS X 10.6およびMacを搭載したコンピューターおよびラップトップに接続できますボード上のOS X 10.7。

このアプリケーションは、スマートフォンやタブレットでの使用に最適化されており、マルチタスクを完全にサポートし、マウスとキーボード用の高度で正確で応答性の高いタッチパッドを備えています。 オンスクリーンキーボードは、必要なときに自動的にアクティブになります。 RDP仕様は自動的に最新バージョンに更新されます。 128ビットのデータ暗号化により、アプリケーションのセキュリティが確保されます。



PocketCloud Remote Desktopの長所とアプリケーション設定の機能はw3bsit3-dns.comフォーラムで活発に議論されています 。このプログラムの複雑さに関する追加情報は公式wikiで見つけることができます 。



また、DellのWyse部門によって開発されたPocketCloud ExploreおよびPocketCloud Webアプリケーションにも注意を払うことをお勧めします。 最初のアプリケーションは、モバイルデバイスからコンピューターへ、またはその逆に情報をコピーし、クラウドを使用することを可能にし、AndroidおよびiOSでガジェットをサポートする負担に加えて、Windows 8でタブレットでも動作します。2番目のアプリケーションは、Webベースの管理インターフェイスです。必要なファイルとフォルダーをコンピューターからクラウドに、またはその逆にコピーできます。



Enterproid Divide-個人用スペースと仕事用スペースの共有



BYODコンセプトのフレームワーク内の別の有用なソリューションにより、仕事とレジャーの両方でモバイルデバイスを使用できるようになり、従業員の個人情報から雇用者を保護すると同時に、企業データの高度なセキュリティを維持できます。

Divideは、AndroidとiOSに基づくデバイス上に2つの分離されたプロファイルを作成し、仕事に必要なものを個人用アプリケーションから分離します。 このプラットフォームはすでにHabréで言及されていましたが、書かれた内容を少し補足するようにします。







Enterproidのソリューションは、バージョン2.2より古いAndroidを搭載したタブレットとスマートフォン、およびiOS6以上のオペレーティングシステムバージョンを搭載したAppleデバイスをサポートします。



ガジェットの企業プロファイル内のすべての情報は、256ビットキーの対称AESブロック暗号化アルゴリズムを使用して暗号化されます。



このソリューションでは、システム管理者はWebインターフェイスを介して、リモートでビジネスアプリケーションをインストールし、セキュリティポリシーを管理し、デバイスの座標を決定し、デバイスが失われた場合にデバイスの企業プロファイルを「消去」できます。 デバイスの企業プロファイルはVPNを介してクラウドに接続され、個人プロファイルのユーザーアプリケーションはインターネットに直接接続されます。 システム管理者は、デバイスの個人プロファイルにアクセスできません。



Divideの強みの中で、個人と企業プロファイルからのアプリケーションの迅速な切り替え、およびEnterproidとQualcommの緊密な協力のおかげで、ハードウェアレベルで既にプラットフォームセキュリティをさらに改善するための有望な見通しに注目したいと思います。 特に、両社のチームは、AndroidタブレットとスマートフォンのDivideプロファイルをさらに安全に分離するために、 TrustZoneテクノロジーの機能を積極的に使用しています。 AMDがTrustZoneテクノロジーを採用し 、モバイルデバイスの方向に関心の範囲を円滑に拡大しているという昨年のニュースも、喜ぶべき理由となっています。

モバイルデバイス上のビジネス情報のセキュリティを確保するのに役立つ別の可能なソリューションは、仮想化です。 しかし、ハイパーバイザーの必要性、およびデバイスのハードウェアに対する高い要件により、技術開発のこの段階でほとんどのデバイスにハイパーバイザーを使用することは不可能です。 Habréについては、Androidに基づくデバイスの仮想化の分野でのサンクトペテルブルクのアカデミック大学の学生の業績について（ パート1 、 パート2 ）についてすでに書いていますが、開発の最終段階にはまだ程遠いです。

PowerConnect ClearPass Wシリーズ-デルの完全なハードウェアおよびソフトウェアソリューション







デルは、BYODコンセプトの実装を支援するソフトウェアソリューション、およびこの企業ポリシーに適合する高品質のガジェットに加えて、モバイルデバイスからビジネス情報への安全なアクセスを提供するいくつかの統合ソリューションを提供します。 そのようなソリューションの1つが、以下で説明するPowerConnect ClearPass Wシリーズです。



この包括的な製品は、高レベルの保護とスケーラビリティを提供し、Windows、Mac OS X、Android、およびiOSを搭載したさまざまなデバイスを企業ネットワークに統合することも可能にします。

PowerConnect W ClearPass100およびPowerConnect W ClearPass2500の直感的で使いやすいインターフェイスにより、技術トレーニングを受けていない従業員は、ゲストアクセスまたは個人デバイスへのアクセスを簡単に提供でき、ネットワークの整合性に影響を与えずにITテクニカルサポートサービスに負担をかけずに管理できます。



サーバーの技術仕様、およびデルのこの統合ソリューションの機能に関する詳細情報は、以下のネタバレを読んでください。

PowerConnect W ClearPass100およびPowerConnect W ClearPass2500-仕様と機能
セキュリティとスケーラビリティ：

  • 各ユーザーには一意のユーザー名とパスワードが与えられます。 このソリューションにより、既存のActive Directoryディレクトリと簡単に統合できます。
  • ゲストおよびデバイスアカウントの場合、組み込みのリモート認証ダイヤルインユーザーサービス（ RADIUS ）データベースが使用されます。
  • ショートメッセージサービス（SMS）の統合により、範囲外の追加チャネルでワイヤレス通信用のセキュリティキーを送信できるため、暗号化によるスケーラブルなゲストアクセスが可能になります。
  • 50〜数千ユーザーの範囲の高いスケーラビリティは、会社の発展に大きなチャンスを与えます。 エンドユーザーに必要なライセンスを購入すると、機能を構築するための基礎として使用できます。
  • PowerConnect ClearPass Wシリーズは、FTPまたはFTP-over-SSLおよびSMBプロトコルに基づくファイル共有を使用する場合など、すべてまたは個別の構成コンポーネントとレポートのリアルタイムまたはスケジュールされたスケジュールでのバックアップをサポートします。
  • 一時アカウントを削除するか、自動的に期限切れにすることができます。
  • ゲストアカウントのアクティビティ、およびゲストアクセスのすべてのケースを提供する従業員は、レポートを作成することで監視できます。




ゲストアクセスに関連する機能：

  • 自己登録のための使いやすいポータル。
  • ゲストクライアントアクセスを備えたさまざまな一般的なデバイスと、パーソナルデバイス（マルチメディア電話からラップトップ、タブレット、デスクトップコンピューターまで）のサポート。
  • さまざまなタイプのユーザーにさまざまなアクセス特権を割り当てる機能。
  • カスタマイズ可能なログインページにより、ブランドを使用して独自のユーザーエクスペリエンスを作成できます。


これに加えて、BYOD内で接続されたデバイスでは、使用するデバイス、実行中のアプリケーション、およびそれらの場所に応じてユーザーグループプロファイルを微調整できます。 企業の情報が失われた場合、それを使用してガジェットから企業情報をリモートで削除できます。
