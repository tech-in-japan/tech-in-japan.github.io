Vivaldiアセンブリ1.0.196.2だけでなく、空間ナビゲーション

みなさんこんにちは！



Vivaldiはノルウェーとアイスランドの両方にオフィスを構えていますが、6月にボストン近くのグロスターにあるアメリカのオフィスに集まることにしました。 昨年の記事から、私たち全員がどこに集まっているかについて詳しく知ることができます。 さて、今-実際にブラウザで何が起こるかについての詳細。



まず、空間ナビゲーションの改善に取り組み続けました。 おそらく変更は目立っていませんが、しかし、それらは:-)



次はキーボードショートカットです。 私たちは、この機能を順番に、またはより正確には、ユーザーになじみのある形で提供することに引き続き取り組んでいます。 改善プロセスはもちろん無限ですが、今日この機能がすでに完全に機能することを願っています。



すてきなことには、クイックコマンドのリストへのリンクの追加が含まれます。



マクロにはパーソナライズされた良いニュースがあります。Tab+ Tを使用して1つではなく2つのタブを開いたときのバグを修正しました。 今は誰も開きません。 冗談。 確実に開きます。 :-)



64ビットバージョンのWindowsのユーザーは、自動更新の動作を確認できますが、このアセンブリは実験的なものであり、改善の点でまだ多くの作業が必要であることを思い出してください。 フィードバックとエラーメッセージをお待ちしています。



そして、特にそれに対する多くの要求があったという事実に関して、もう一つのことを言う必要があります。 「貼り付けて実行」機能がアドレスバーのコンテキストメニューに追加されました。 実際、この機能にはまだ多くの作業が必要なので、ニュースをお待ちください。



また、新しいアセンブリでは、エスペラント語と西フリジア語の 2つの言語への翻訳を追加しました。 翻訳はまだ部分的ですが、愛好家は常にそれを更新しています。



今のところすべてです。 以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
