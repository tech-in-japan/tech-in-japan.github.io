新しいVivaldi 1.0.344.34ビルドの別のウィンドウにタブを転送します

みなさんこんにちは！



2番目のベータバージョンのリリースに近づいています。新しいアセンブリでは、ブラウザーを不正に閉じる原因となったエラーを修正することに加えて、新しい機能を追加しました（または実装を開始しました）。新しいウィンドウでタブを開きます。 より正確には、すでに開いているタブを別のウィンドウに移動します。



タブを別のウィンドウに移動する



タブのコンテキストメニューに、新しいウィンドウまたは別の既存のウィンドウでタブを開くことができる新しいアイテムを追加しました。 すでに開いているすべてのウィンドウのリストもコンテキストメニューに表示されます。









これは関数の最初の実装にすぎませんが、今後は使いやすさの観点から改善していきます。 たとえば、多くのユーザーの要求に応じて、Vivaldiウィンドウの外側にタブをドラッグして新しいウィンドウを作成する機能を追加する予定です。



今のところすべてです。 記事の最後に変更点の完全なリストがあります。



以下のリンクからアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット


変更の完全なリスト：

  • VB-11444フォルダーが選択されているときにメモを作成するとクラッシュする
