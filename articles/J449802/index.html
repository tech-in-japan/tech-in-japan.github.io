キャリアプログラマー。 パート1.最初のプログラム

Habrの読者の皆様、私はあなたの注意に、私が将来この本にまとめる予定の一連の投稿を提示します。 私は過去を掘り下げて、自分がどのように開発者になり続けたかについて話をしたかったのです。



ITに参加するための前提条件、試行錯誤のパス、独学、幼稚な純真さについて。 私は幼児期から物語を始め、今日で終わらせます。 この本が、IT専門の勉強をしている人に特に役立つことを願っています。



そして、すでにIT部門で働いている人たちは、自分の道に似ているかもしれません。



この本では、私が読んだ文献、トレーニング、仕事、スタートアップのスタートアップの間に交わした人々とのコミュニケーションの経験について言及しています。



大学の教師から始まり、大規模なベンチャー投資家と数百万ドルの会社の所有者で終わる。



現在までに、8〜10の3.5の章が用意されています。 最初の章が聴衆から肯定的な反応を見つけた場合、私は本全体を出版します。



私自身について



私はジョン・カーマック、ニコライ・デュロフ、リチャード・マシュー・ストールマンではありません。 Yandex、VKontakte、Mail.ruなどの会社では働いていません。



私は大企業での経験がありましたが、それについて間違いなくお話しします。 しかし、ポイントは、開発者への道の歴史のように、そしてさらに、商業開発の私の12年のキャリアの間にあった勝利と敗北のように、大きな名前ではないと思います。 もちろん、一部の人はITの経験が豊富です。 しかし、私は私の現在のキャリアにあったドラマと勝利は説明する価値があると信じています。 たくさんのイベントがあり、それらはすべて多様です。

- マイクロソフト最優秀人物 2016

-マイクロソフト認定プロフェッショナル

-認定スクラムマスター

-私はC＃/ C ++ / Java / Python / JSを良いレベルで知っています

-今日の主な仕事はUpwork Freelanceサービスです。 彼女を通して、私はNLP / AI / MLを扱う会社で働いています。 100万人のユーザーのデータベースがあります

-AppStoreおよびGooglePlayで3つのアプリケーションをリリースしました

-現在開発中のプロジェクトの周りに自分のIT会社を設立する準備ができています



開発に加えて、私は人気のブログで記事を書き、新しい技術を学び、会議で講演します。 私はフィットネスクラブで、家族と一緒にリラックスします。



この本のトピックに関しては、おそらく私のすべてです。 次は私の話です。





物語。 開始する



私が7歳のときに初めてコンピューターとは何かを学びました。 私はちょうど1年生に行き、IZOのレッスンで、段ボール、発泡ゴム、フェルトペンでコンピューターを作るための宿題をもらいました。 もちろん、両親が助けてくれました。 80年代前半のママは、工科大学で勉強し、コンピューターが何であるかを聞いて知りませんでした。 トレーニングコース中に、彼女はパンチカードをパンチして、研究室の大部分を占める巨大なソビエトマシンにロードすることさえできました。



すべてをこまめに行ったため、5時に宿題を完了しました。 厚手のA4厚紙を見つけました。 泡の円は古いおもちゃから切り取られ、ユーザーインターフェイスはフェルトペンで描かれました。 デバイスにはボタンがほとんどありませんでしたが、母と私は必要な機能を割り当てました。レッスンでは、「オン」ボタンを押すことで「スクリーン」の隅にライトが点灯し、同時にフェルトペンで赤い円を完成させる方法を先生に示しました。



コンピュータ技術との次の交差点、私は同じ年齢のどこかで起こった。 週末には、祖母や祖父とよく会いました。祖父と祖父はさまざまなジャンク品を取引し、喜んで1ペニーで買いました。 古い時計、サモワール、ボイラー、バッジ、13世紀の戦士の剣など。 こうしたさまざまなものの中で、誰かが彼にテレビとオーディオレコーダーを搭載したコンピューターを持ち込みました。 幸いなことに、私の祖母は両方を持っていました。 もちろんソビエト生産。 チャンネルを切り替えるための8つのボタンを備えたTelevision Electron。 さらに、Vegaの2カセットレコーダーは、フィルムのオーディオテープを再記録することさえできました。





ソビエトのコンピューター「検索」および周辺機器：テレビ「電子」、テープレコーダー「ベガ」、およびBASIC言語のオーディオカセット



このシステム全体がどのように機能するかを理解し始めました。 コンピューターを完備し、オーディオカセットが2枚、非常にボロボロの説明書、「BASIC Programming Language」という見出しの別のパンフレットがありました。 子供の頃にもかかわらず、私はコードをテープレコーダーとテレビに接続するプロセスに積極的に参加しようとしました。 次に、テープの1つをレコーダーコンパートメントに挿入し、「進む」ボタンをクリックして（つまり、再生を開始します）、テキストとダッシュからのわかりにくい擬似グラフィックがテレビ画面に表示されました。



ヘッドユニット自体はタイプライターのようなもので、かなり黄色く目立った重さしかありませんでした。 子供の興奮で、私はすべてのキーを押して、具体的な結果を見ずに、散歩に走りました。 私の前に、すでに年齢のために書き直すことができなかったプログラムの例を含むBASIC言語のマニュアルをすでに置いていましたが。



幼少時代の思い出から、私は両親が私に買った、他の親rememberと結成したすべてのガジェットを確かに覚えています。 最初のガラガラは、有名なゲーム「オオカミが卵を捕まえる」でした。 すぐに調べて、最後に待望の漫画を見て、もっと何かを望んでいました。 それからテトリスがありました。 当時は1,000,000枚のクーポンに相当しました。 はい、それは90年代前半のウクライナで、私の優秀な学問のために、彼らは私に100万を与えました。 大富豪のように感じて、私は両親にこのより複雑なゲームを命じました。そこでは上から落ちてくるさまざまな形の数字を正しくレイアウトする必要がありました。 購入の日、テトリスは両親から連れ去られ、両親は2日間彼から離れることができませんでした。





有名な「オオカミは卵とテトリスを捕まえる」



それからゲーム機がありました。 私たちの家族は小さな家に住んでいて、叔父と叔母も隣の部屋に住んでいました。 私の叔父は軍のパイロットだったので、彼はホットスポットを通過したので、謙modeにもかかわらず、彼は非常に粘り強く、実際の敵意の後、ほとんど恐れていませんでした。 90年代の多くの人々と同じように、私の叔父はビジネスに参入し、彼はかなり良い収入を持っていました。 そのため、彼の部屋には、インポートされたテレビ、VCR、そしてSuborプレフィックス（Dendyと同様）が登場しました。 彼がスーパーマリオ、トップガン、ターミネーターなどのゲームをプレイするのを見たとき、それは息をのむようなものでした。 そして、彼がジョイスティックを私の手に渡したとき、私の幸福は際限がありませんでした。





8ビットのプレフィックス「Subor」と伝説の「Super Mario」



はい、90年代に育った普通の子供たちのように、私は一日中庭​​で過ごしました。 パイオニアボール、バドミントン、そしてさまざまな果物が育つ庭で木に登っています。

しかし、マリオを制御し、障害物を飛び越えて王女を救うことができるこの斬新さは、盲人のメガネ、ポーン、クラシックよりも何倍も興味深いものでした。 したがって、コンソールに対する私の真の関心を見て、両親は私に九九を学ぶ仕事を与えました。 その後、彼らは私の夢を実現します。 彼女は2年生で教えられ、私はちょうど1年生を終えました。 しかし、それは言われている-完了。



独自のゲーム機よりも強いモチベーションを考え出すことは不可能でした。 そして一週間後、私は簡単に「家族九」、「六三」などの質問に答えました。 テストに合格し、切望された贈り物を買いました。 後で学習するように、コンソールとコンピューターゲームは、プログラミングに興味を持つ上で重要な役割を果たしました。



だから年々過ぎました。 次世代のゲーム機が登場しました。 まずセガ16ビット、次にパナソニック、次にソニープレイステーション。 ゲームは、私がうまく行けたときの私の娯楽でした。 学校や自宅で横棒があったとき-ジョイスティックは私から取られたので、もちろん私は遊ぶことができませんでした。 そしてもちろん、あなたが学校を卒業し、あなたの父親がまだ仕事を辞めてテレビを取り上げていない瞬間をつかむことも、一種の幸運でした。 ですから、私がゲーマーだったとか、丸一日ゲームをしたとか言うのは不可能です。 そのような可能性はありませんでした。 むしろ、中庭で一日を過ごしました。そこでは、何か面白いものを見つけることもできました。 たとえば、完全に野生のゲーム-エアガンで小競り合い。 私たちの時代では、あなたは中庭でこれに会うことはありません、そして-それは本当の戦争でした。 ペイントボールは、私たちが組織した虐殺と比較して、子供にとっては楽しいものです。 空気には厚いプラスチック弾が搭載されていました。 そして、他の子供を至近距離で撃ったところ、彼の腕または胃の床に打撲傷がありました。 彼らは生きました。





子供の頃からおもちゃの銃



映画「ハッカー」に言及することは不必要ではありません。 彼はちょうど1995年に出てきて、20歳のアンジェリーナ・ジョリーと共演した。 映画が私に強い印象を与えたと言うことは、言うまでもありません。 結局のところ、子供の思考は額面どおりにすべてを知覚します。



そして、これらの人たちがどのようにATMをクリアし、信号機を消し、街中の電気で遊んだか、私にとっては魔法でした。 それから、ハッカーのように全能になるのはクールだと思いました。



数年後、私はハッカー誌のすべての号を購入し、ペンタゴンをハッキングしようとしましたが、まだインターネットにアクセスできませんでした。





映画「ハッカーズ」のヒーロー



私にとって本当の発見は、すでに15インチのチューブモニターとIntel Pentium IIプロセッサをベースにしたシステムユニットを備えた実際のPCでした。 もちろん、彼は彼の叔父に買収されました。叔父は、90年代の終わりまでに、自分でそのようなおもちゃを作ることができるほど高くなりました。



ある種のゲームを初めてオンにしたときは、それほどエキサイティングではありませんでした。 しかしある日、裁きの日が来て、星が一列に集まり、家にいないおじさんを訪ねてきました。 私は尋ねた：



「コンピューターの電源を入れることはできますか？」

「はい、あなたは彼と一緒にやりたいことをしてください」と愛情のある叔母は答えました。



もちろん、私は彼と一緒にやりたいことをやりました。 Windows 98デスクトップにはさまざまなアイコンがありました。 WinRar、Word、FAR、スカーフ、ゲーム。 すべてのアイコンをクリックした後、FARマネージャーに注目しました。 わかりにくいブルースクリーンのように見えますが、実行できる（ファイル）の長いリストがあります。



それぞれを交互にクリックすることで、何が起こっているのかを把握しました。 何かがうまくいきましたが、何かがうまくいきませんでした。 しばらくして、「。exe」で終わるファイルが最も興味深いことに気付きました。 彼らはまた、マントを付けることができるさまざまなクールな写真を起動します。 だから、おじさんのコンピューターですべての利用可能なexeファイルを起動したのだろう。そして、彼らはかろうじて面白いおもちゃから耳を引っ張って、私を家に連れて行った。





同じFARマネージャー



それからコンピュータークラブがありました。 私の友人と私はしばしば彼らに行き、ネットワークでCounter-StrikeとQuakeをプレイしましたが、それは家ではできませんでした。 よく両親に30分クラブでプレイするためにちょっとしたことを頼みました。 シュレックの猫のように私の目を見て、彼らは私にもう一つの有利な契約を提供しました。 私はトリプルなしで学年を終え、彼らは私にコンピューターを購入します。 契約は年の初め、9月に調印され、切望されたPCは6月にすでに到着しており、契約の対象となります。



がんばりました。 彼は、勉強から気が散らないように、愛するソニーのプレイステーションを感情で売りさえしました。 私はまあまあ勉強しましたが、9年生は私にとって重要でした。 鼻からの血、それは良い成績のみを取得する必要がありました。



すでに春に、PCの購入を期待して、おそらく私の人生で最も重要な出来事が起こりました。 私は先を考えようとするので、ある日父にこう言いました。



「お父さん、コンピューターの使い方がわかりません。」 コースにサインアップしましょう。



すぐに言ってやった。 発表のある新聞を開くと、父は「コンピューター講座」という見出しの小さな活字を見つけました。 私は教師に電話をかけ、数日後、私はすでにこれらのコースにいました。 コースは市内の反対側、3階にある古いパネルのフルシチョフで開催されました。 1つの部屋には3台のPCが連続しており、実際にはそれらを学習したい人がいました。



最初のレッスンを覚えています。 Windows 98が長時間ロードされた後、教師が発言しました。



-だから。 これがWindowsデスクトップです。 プログラムアイコンがあります。 [スタート]ボタンの下部。 覚えて！ すべての作業は開始ボタンから始まります。 それを左クリックします。

彼は続けた。

-ここ-インストールされたプログラムが表示されます。 電卓、メモ帳、Word、Excel。 「シャットダウン」ボタンをクリックして、コンピューターの電源を切ることもできます。 試してみてください。

最後に、彼はそのとき私にとってより難しい部分に移りました。

-先生は、デスクトップ上で、ダブルクリックして起動できるプログラムを見ることができると言いました。

-ダブル！？ -どう？

-試してみましょう。 マウスの左ボタンでメモ帳をダブルクリックして、メモ帳を起動します。



ええ、今。 その時点で最も難しいことは、マウスを1か所に保持し、同時にすばやくダブルクリックすることでした。 2回目のクリックで、マウスが少しひっくり返り、それにショートカットが付きました。 それでも、レッスン中にこのような乗り越えられないタスクをマスターすることができました。



次に、Word、Excelのトレーニングがありました。 ある日、私は自然の写真や建築物の記念碑を見ることを許されました。 それは私の記憶の中で最も興味深い活動でした。 Wordでテキストをフォーマットすることを学ぶよりもはるかに楽しい。



私のPCの隣で、他の学生が勉強しました。 何度か、プログラムを書いている人に出会って、このプロセスについて精力的に議論しました。 私も興味がありました。 映画「Hackers」と退屈なMS Officeを思い出して、私はプログラミングコースに移行するように依頼しました。 人生のすべての重要な出来事と同様に、これは自発的に起こり、興味をそそりました。



私は母との最初のプログラミングレッスンに到着しました。 理由は覚えていません。 どうやら彼女は新しいコースに同意し、支払ったはずです。 庭の春でしたが、すでに暗くなっていました。 私たちはミニバスのガゼルで街全体をドライブして郊外まで行き、悪名高い

フルシチョフのパネルは、床に上がって私たちを入れました。



彼らは私を極端なコンピューターの後ろに置き、完全に青い画面と黄色い文字でプログラムを開きました。



-これはTurbo Pascalです。 先生は自分の行動についてコメントしました。

-見て、ここで私はドキュメントを書きました。 読んで



私の前は黄色のキャンバスで、まったく理解できないテキストでした。 私は自分で何かを見つけようとしましたが、できませんでした。 中国語の文法とそれだけです。



最後に、しばらくして、コースリーダーからA4の印刷されたシートが渡されました。 それは私が以前にプログラミングコースの男たちのモニターを垣間見たという奇妙なことに書かれていました。

-ここに書かれている内容を書き換えます。 彼は先生に命じて去った。

私は書き始めました：

プログラムSumma;



キーボードで英字を探しながら書きました。 Wordでは、ロシア語のトレーニングさえ受けましたが、ここでは他の文字を学ぶ必要があります。 プログラムは1本の指で入力されましたが、非常に熱心に入力されました。

begin、end、var、integer-それは何ですか？ 私は1年生から英語を勉強し、多くの単語の意味を知っていましたが、すべてを合わせることができませんでした。 自転車に乗った訓練を受けた熊のように、私はペダルをこぎ続けた。 最後になじみのあるもの：

writeln（「最初の数字を入力してください」）;

次に-writeln（「2番目の数字を入力」）;

その後-writeln（ 'Result ='、c）;





最初のTurbo Pascalプログラム



ふう、書いた。 彼はキーボードから手を離し、指導者がさらなる指示のために現れるのを待ちました。 最後に、彼は立ち上がって目を画面上に走らせ、F9キーを押すように言いました。

-今、プログラムはコンパイルされてエラーがないかチェックされます、と第一人者は言いました

エラーはありませんでした。 それから彼はCtrl + F9を押すように言ったが、これは初めてステップバイステップで説明する必要があった。 Ctrlキーを押しながらF9キーを押します。 画面が真っ暗になり、最後に「最初の番号を入力してください」というエントリが明確に表示されました。

先生の指示で、7を入力しました。2番目の数字。 3と入力してEnterキーを押します。



画面に「結果= 10」という行が即座に表示されます。 それは陶酔感があり、私の人生でこれまでに経験したことがありませんでした。 それはまるで宇宙全体が私の前に開いていて、ある種のポータルに入ったようなものでした。 暖かさが体を通り抜け、笑顔が私の顔に現れ、私の潜在意識の非常に深いところに、 それが私のものであることに気づきました 。 非常に直感的に、感情のレベルで、私はテーブルの下のこの賑やかな箱に大きな可能性を感じ始めました。 あなたは自分の手でどれだけできるか、彼女はそれをするでしょう！



それはある種の魔法です 青い画面上のわかりにくい黄色のテキストが便利で理解可能なプログラムに変わったように、私の頭はまったく合わなかった。 まだ重要です！ 驚いたのはカウント自体ではなく、書かれた象形文字が計算機に変わるという事実でした。 これらの2つのイベントの間には、その瞬間に深byがありました。 しかし、直感的に、私はこの鉄片がほとんどすべてを行うことができると感じました。



ミニバスで家に帰る途中で、まるで宇宙にいるかのようでした。 この写真は、「結果」という碑文で頭の中で回転していました。このマシンがどうしてできたのか、紙なしで自分で何かを書けますか。 私が興味を持ち、興奮し、同時に刺激を受けた何千もの質問。 私は14歳でした。 その日、職業は私を選んだ。
