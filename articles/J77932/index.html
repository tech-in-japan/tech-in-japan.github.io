高さ100％またはフルハイト





ブログのテーマを作成すると、ページのサイズが小さく、主要部分が最小限に圧縮され、フッターがそれに押されたときに問題が発生しました。 メイン部分は、ページの高さ全体に接続することにしました。 もちろん、CSSを使用するだけでなく、JavaScriptを使用してJavaScriptを使用せずにこれを行う方法は多数あります。 純粋なCSSを選択しました。





この手法の利点は純粋なCSSであり、すべてのJavaScriptが有効になっているわけではありません。 ただし、デザインの概念で直接コンテナの境界線を使用する場合は便利です。





これがこのCSS、「公序良俗に反する」です:)。





< style type ="text/css" >

html,body{

height:100%;

margin:0px;

padding:0px

}



.height100 {

height: auto;

min-height: 100%;

}

</ style >
