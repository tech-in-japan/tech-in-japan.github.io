Dental Cloud Dental Automation Serviceの概要

以前に、サービスのスクリーンレビューを公​​開しました-可能な限り軽量なバージョンです。 今こそ、 Dental Cloud （バージョン01.11.15）の機能の優れた概要を説明するときです。







このサービスは、歯科医、手術、小規模な個人診療所の練習を自動化するように設計されています。 製品のロジックは、クリニックの3つの主要な領域の自動化に基づいています。



  • 運営
  • 治療
  • コミュニケーション（機能開発計画に関する投稿）


関税



関税は、有機的なビジネスの成長に焦点を当てています。 料金表は、次の3つの料金表で表されます。



練習 -条件付きで独立した歯科医を対象とした無料の関税。 データベース内の100人の患者の制限（料金表の開発に関する投稿）。

キャビネット -関税は、歯科医の独立した仕事の最初のステップと、診療を合理化し、アシスタントまたは管理者と協力したい人に焦点を当てています。

クリニック -関税には、治療計画で一緒に働く医師の可能性が含まれます。 関税は、小規模な個人診療所を対象としています。



すべての関税で同じ機能を残したことが重要です。 関税は、ユーザーには明確に表示されないライセンスのセットであり、管理者のライセンス、医師のライセンス、医師のライセンス-共同作業であることが重要です。 ユーザーに過負荷をかける理由



運営







訪問カレンダー



訪問のカレンダーは、サービスの主なワークスペースであり、予約の可能性が実現され、指定された瞬間に働く医師のスケジュールと訪問が表示されます。 カレンダーは、「医師による」フィルタリング、特定の稼働日のクイック表示をサポートしています。



価格表

クリニックまたはオフィスのカスタマイズ可能な価格表。 さまざまなレベルのネスト、カテゴリごとのサービスのグループ化がサポートされています。 セクションの自動番号付け、サービスのカテゴリ。 名前またはコードによる必要なセクションまたはサービスの便利な検索。



請求と支払い

請求-印刷する請求書の作成、管理、および送信。 未払いの請求書は常に管理者の前にあります。 部分的に請求書を支払う機能、近い将来に患者が請求書を支払うためのリマインダーのオプションがサポートされています。



分析

分析レポートの形式で、常にビジネスに関する最新情報を入手できます。 私たちは、診療所に関する一般的なレポート、各従業員の分析、従業員の専門性を監視する機能を実装しました。 このようなアプローチは、診療所全体の全体像を把握し、スタッフの動機付けプログラムを開発する際に分析を適用するのに役立ちます。



治療







歯のカード

治療歴を記入するとき、事前設定された診断のセットでグラフィック歯科チャートを使用します。 診断を選択するには、歯の特定の領域をクリックするだけです。カードが印刷されると、診断は自動的にテキストシンボルに変換されます



治療日記

診療所への患者訪問のための標準治療日記の記入-苦情、病歴、客観的、診断、治療、エピクリシス、結果。 これらのフィールド用のカスタマイズ可能なテンプレートは、医師が情報を入力する時間を大幅に節約できます。



治療コース

診療所の価格表による治療コースの迅速な形成。 各サービスについて、このサービスが適用された歯が割り当てられ、ステータスが表示されます-完了/未実行。これにより、1つの治療を複数の訪問に分割できます。



ファイルストレージ

さまざまな種類のデジタルファイルを患者のカードに添付します-X線、歯の3D画像、テスト結果、文書と契約書、その他の医療文書。 各患者のカードのファイル数は制限されていません。



コミュニケーションズ







オンライン記録

クリニックのウェブサイトに予約ウィジェットを配置して、新しいクライアントを受け取ります。 クライアントは、自分の医師を選択し、記録のための自由時間を確保し、予約することができます。 すべてのアプリケーションは自動的にDental Cloudに移動し、ワンクリックで確認されます。



SMS通知

予定された訪問についての患者への自動SMS通知は、欠勤の数を減らし、医師の勤務スケジュールを最適化するのに役立ち、可能であれば、失敗した訪問を再スケジュールします。 CIS諸国、ヨーロッパ、北米のすべての携帯電話サービスプロバイダーがサポートされています。
