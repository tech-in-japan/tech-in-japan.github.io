プレースホルダー

HTML5にはプレースホルダー属性がありますが、現時点では、Webkitエンジンを備えたブラウザーでのみ機能します。



ハブ上のいくつかの投稿を見たところ、自転車を発明した人や、この属性をクロスブラウザにする別のスクリプトがパスワードなどのフィールドにアスタリスクを表示することがわかりました。



実際、彼はこのすべてをサポートする控えめなjQueryコードを書きました。



-何を言ってるの？

ここで触ることができます 。 必ず送信ボタンをクリックして、プレースホルダーテキストがない場合はアドレスバーを確認してください。



-そして、使用方法は？

簡単です：



options = {

color: '#ccc', // placeholder'

className: 'placeholded', //

handle_send: true, // .

handle_password: true // password-

};

$("#myinput").blurfocus(' ', options);







-「$」が表示されます。 jQueryが必要ですか？

はい、もちろんです。 できれば新鮮。



-どこでも動作しますか？

IE7.8でテストしました。 FF3.5; Opera 10.01。 このスクリプトでは、ChromeとSafariは必要ありません。 プレースホルダー属性を理解します。 （スクリプトはこれを提供します）



-さて、どこからダウンロードしますか？

こちら 。



UPD：すべてのオプション（color、className、handle_send、handle_password）はオプションです！ この例は、標準値のみを示しています。



UPD2：新しいバージョンlinkがあります。 スクリプトを挿入するだけでよいという点で異なります







CSSテーブルのプレースクラスの説明



<style type="text/css">.placeholded {color: #;}</style>







同時に、必要なすべてのフィールドに、目的のテキストを含むプレースホルダー属性を追加します。 DOMReadyイベントのスクリプトは、プレースホルダー属性を持つすべてのフィールドをバイパスし、非Webkitブラウザーにメジャーを適用します。
