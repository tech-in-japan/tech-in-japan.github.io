Monoblock IdeaCentre A700：メディアセンター+テレビ

このキャンディーバーを自宅で1か月間テストしました。 当初はデスクトップにインストールされていたため、すぐに家庭用テレビの代わりになり、そこに留まり、同時にテレビ、メディアセンター、音楽プレーヤーとして機能しました。 このオールインワンで完全ですが、非常にまともなキーボードとマウス、さらにはWindows Media Center用に最適化されたリモートコントロールがありますが、A700の主な利点はタッチスクリーンです。 長い間、私はキーボードを数回しか使用していませんでした。タッチスクリーンは一連の典型的なタスクに非常に適していることが判明しました。 どれ？ たとえば、ビデオの再生を開始または一時停止するには、音楽やテレビ番組でも同じです。



その結果、予想外のことが起こりました。このコンピューターは、家庭用テレビやハードドライブを内蔵したセットトップボックスのライバルであることが判明しました。 ほとんどの場合、両方を置き換えますが、同時に「コンピューター+モニター」バンドルよりはるかに高価ではありません。 このソリューションはどのくらい便利ですか？ それを理解してみましょう。









技術仕様Lenovo IdeaCentre A700

プロセッサー ：Intel Core i3 2.53 GHz（Core i5および4-core Core i7に変更があります）

メモリー ：2 GB（最大8 GB）

ディスプレイ ：23” 1920x1080ピクセルマルチタッチタッチ

ハードドライブ ：500 GB

ビデオカード ：ATI Radeon 5650 1 GB

光学ドライブ ：DVD-RW

コネクタ ：6xUSB、eSata、FireWire、オーディオ、HDMI入力/出力、アンテナ入力、イーサネット、SPDIF、ビデオ入力

ワイヤレスネットワーク ：WiFi 802.11n、Bluetooth

オプション ：ウェブカメラ、TVチューナー、SD / SDHCカードリーダー

寸法 ：568x430x71 mm

重さ ：14.8 kg







ほとんどすべてのモノブロック（Lenovoと他のメーカーの両方）には、親classicにクラシックデスクトップPCとラップトップの両方があります。 A700はラップトッププロセッサとモバイル（ただし強力な）グラフィックスカードを使用します。 しかし、ハードドライブは3.5インチで、少し高速です。 ラップトップとのもう1つの類似点は、A700については何も購入する必要がなく、キーボード、マウス、リモートコントロールなどすべてがすでに含まれていることです。 そして、インストール自体はそれほど時間がかかりません：特定の角度に重点を置くと、完了です。







モデルのパフォーマンスは、サイトDesktopReviewのこの資料で推定できます。 確かに、Intel Core i7プロセッサを搭載したわずかに強力なモデルがそこでテストされましたが、ビデオカードはまったく同じです。 この構成のモデルのコストは33,000ルーブルであり、「フルスタッフィング」は5万強で購入できます。 この場合、同じCore i7、1テラバイトのハードドライブ、6ギガバイトのRAMを使用できます。

A700には、eSataを含む、便利にしか使用できないほとんどすべてのインターフェイスが装備されています。 コンポジットビデオ入力またはHDMIを介して、外部デバイスをデバイスに接続することもできます。







一部のコネクタはサイドパネルに配置されています。







デバイス制御ボタンはタッチセンシティブです。 ビデオソースの切り替え、音量と明るさの調整、画面のオフに使用されます。







別のキーを押すと、ディスプレイの下にある元のバックライトが点灯します。 キーボードをコンピューターの正面に設置すると非常に便利です。 キーボードとマウス自体はワイヤレスであり、Bluetooth経由でPCに接続します。 背面には、Bluetoothデバイス用の同期ボタンがあります。







リモコンは赤外線で、独自のUSBレシーバーが付属しています。







メディアセンター

先ほど言ったように、基本的なタスクでは、キーボードとマウスはまったく必要ありません。 A700を使用してビデオやテレビを見たり、音楽を聴いたりする場合、リモートコントロールまたはタッチスクリーンからコンピューターを制御できます。



A700は、オリジナルのMultitouch表面弾性波技術を使用しています。 画面上のクリックへの応答は即座に行われ、使用するのに非常に便利です。ここでの制限は、おそらくこのタイプの制御に常に適合しているわけではないWindowsインターフェイスに関するものです。

若いA700モデルの1つをテストしましたが、その性能は、デバイスを本格的なホームコンピューターとして使用するのに十分です。 ゲームでは、オールインワンも良好な結果を示していますが、最新のデスクトップビデオカードよりも劣っています。







テレビ

A700はもともとケーブルテレビネットワークに接続されていました。 チャネルはWindows Media Centerに合わせて調整されました。WindowsMedia Centerは、選択された（つまり、連邦政府の）チャネルの番組ガイドも提供してくれました。 Media Centerが約40のチャネルを検索するのに約10分かかり、チャネルのリストを通常の状態に戻すにはさらに20分かかりました。



PCでのテレビチャンネルの切り替えは、通常のテレビよりも少し長くかかります（約2〜3秒）。 このためチャンネルをクリックするのはあまり便利ではありませんが、メニューを使用すると、いつでも希望のテレビ番組に切り替えることができます。



対角線が十分に大きい従来のテレビの場合と同様に、放送またはケーブル放送はデバイス上で見えますが、あまり印象的ではありません。 ディスプレイの解像度は標準テレビの解像度よりも一桁大きいため、画像は非常にぼやけています。 ただし、A700は、必要に応じて、より新しいデジタルテレビサービスで使用できます。



テレビ+セットトップボックス

したがって、テストの次の段階は、ビーラインテレビセットからA700セットトップボックスへの接続でした。 セットトップボックスは、HDMIを介して接続され、高解像度ビデオを含む1080i解像度のテレビ番組を完全に送信しました。 もちろん、コンピューター上でIPテレビを直接見ることは理論的には可能ですが、このオプションの既成のソリューションは見つかりませんでした。



コンピューターの電源が入っているとき、画面の下のボタンを使用して、PCと外部ソースを切り替えることができます。 ただし、このソリューションにはいくつかの欠点があります。リモートコントロールを使用してA700をオンにすると機能しません。 付属のリモートは、Windows Media Center用に最適化されています。 このシェルを使用すると、リモートコントロールからコンピューターの電源を切ることができます。 外部ソースを使用する場合は、外部ソースをオフにすると、ディスプレイが自動的にオフになります。 ただし、ケースのボタンを使用して画面またはコンピューター全体をオンにする必要があります。



結論

IdeaCentre A700は、コンピューターとテレビの両方が必要な場合に最適なソリューションです。 オールインワンの幅広い機能は多少の不便を引き起こし（特に、PCをリモートでオンにした場合）、状況によってはネットトップまたはフルデスクトップまたはメディアプレーヤーを備えた独立したテレビがより便利な場合があります。 しかし、A700は家庭向けの興味深い普遍的なソリューションであることが判明しました。 同時に、価格は個々のコンポーネント（コンピューター+ 23インチモニターまたはテレビ+キーボード、マウス、オーディオシステム）の価格と大きく異なりません。 コンパクト化のために少し余分にお金を払ってもかまいませんが、その結果、仕事（必要な場合）とゲームとエンターテイメントの両方に適したターンキーソリューションが得られます。
