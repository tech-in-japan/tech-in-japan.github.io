AppleはPsystarに到達しました



この春、 私たちは Psystarのコンピューターに気づきました。このコンピューターは当時OpenMacと呼ばれていましたが、すぐにOpen Computerと改名されました。 おわかりのように、Psystarが販売している製品は、Mac OS Xと互換性のある部品から組み立てられており、Mac OS Xの修正版が装備されています。このアセンブリは、OSハードウェアへのオペレーティングシステムのインストールをサポートするOSx86プロジェクトの作業を使用して作成されました。 これらすべてに400ドルの価格が割り当てられ、これはMacの安価な代替品と呼ぶことができます。 その後、Psystarはサービスの範囲を拡大し、ソフトウェアの更新とサーバーアセンブリの両方を提供しました。



以前、私たちはPsystarの行動に対するAppleの反応を見たことはありませんでしたが、遅かれ早かれこれが起こると繰り返し言われました。 Appleは7月3日、意欲的なフロリダの会社を著作権侵害で訴えました。 これは、Psystarが修正されたMac OS X Leopard 10.5.4アップデートを顧客向けにリリースした1日後に起こりました。



ロバートの名前でしか知られていない会社の代表者は、Open Computerの販売開始後、他の人のハードウェアへのMac OS Xのインストールを禁止するAppleライセンス契約は独占禁止法違反であると述べました。
MicrosoftがWindowsをDellにのみインストールすることを許可している場合はどうなりますか？ ホンダがあなたの車を購入した後、示された道路でのみ運転を許可された場合はどうなりますか？
したがって、Psystarは法廷でAppleと会談したいと考えているが、彼らの意見では、Appleがその権利を主張することは難しいだろう。 同社は、防衛の一環としてAppleが機器を高騰させていると非難する予定です。



AppleInsider経由



AmaxSkv UPD： Psystarは、一部の人が言うように、Appleの「独占」からこのような世界の擁護者と見なされるべきではありません（ただし、独占はありません）。 Psystarは違法行為を行うだけでなく、他人の仕事から利益を得ています。 結局のところ、Mac OS Xは彼らによってではなく、OSx86プロジェクトの活動家によってOpenMacに適応しました。 彼らは単に愛好家の仕事を取り、それらに連絡することさえ考えずに、商業製品で使用しました。 同胞Netkasの投稿を読むことができます。彼は違反と彼の作成のライセンス（EFIエミュレーター）について話しています。 また、こうした事件の後、AppleはOSx86に目をつぶるのをやめるかもしれないのではないかと心配しています。



さらに、Psystarのディレクターは、すべてのプレスコールを後でコールバックすることを約束していますが、もちろんそうではありません。 OS XをPCで使用する場合は、hackintoshをビルドします。これはPsystarの場合よりも優れた方法です。Engageのレビューから判断すると、多くの問題があります。 さらに良いのは、Macを購入することです。しかし、それらは優れていますが、:-)
