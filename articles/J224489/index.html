カフェでの昼食の価格のためのヨーロッパの専用サーバー！

みなさんこんにちは！



フランスのホスティングプロバイダーOVHに関するhabr に関する記事は既にいくつかありました。そのため、会社とそのサービスについては詳しく説明しませんが、私が思うに、非常に魅力的な1つの提案にのみ注目します。





背景



昨年後半、OVHは新しいサーバーラインをリリースしました。 技術的特性の標準的な改善に加えて、価格も大幅に引き下げられたため、新しい機器を注文したいという大勢の人がいました。 数週間後、彼らは需要にほとんど対処できないことに気づき（まず、古いサーバーを投入して低コストで新しいサーバーを取得したい人たち）、新しいサーバーの設計を一時的に閉鎖しました 。







しばらくして、特別オファーがKimsufi Webサイト（OVHが所有するエントリーレベルのサーバーを備えた別のサイト）に配置されました。これは月額わずか2.99ユーロのサーバーです。 しかし、多くの来場者は非常に失望しました-フランスの居住者だけが注文を出すことができました。



ここで最も興味深い



そして今、6か月後、同様の提案が再び登場しました！ 月額わずか4.99ユーロ（現在のレートで約235ルーブル）で、2 GBのRAM、500 GBのハードドライブ、および（ほぼ）無制限の100 Mb / sのAtom N2800ベースの専用サーバーを作成できます。



ご想像のとおり、使用可能な機器は非常に迅速に分岐するため、それを調整するには、サーバーの可用性を常に監視する必要があります。



ボーナス



ただし、Nagios / Icinga（および、場合によっては他の監視システム）のユーザーには、小さなボーナスがあります- 単純なphpスクリプト （はい、Perl / Python / Rubyの愛好家は私を許します）、サーバーをチェックします。



サーバーを注文する場合は、EUに居住していない人は税金を支払う必要がないことを忘れないでください。 文書を送信するときは、アカウントのVATを削除するよう依頼してください。



良い週を！
