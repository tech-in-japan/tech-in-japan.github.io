KDE Software Compilation 4.4 Beta1がリリースされました

待望の新しいリリースが、誰でも利用できるようになりました。 開発者は多くの変更を準備しました。 このリリースは、待望の機能をもたらし、Google Summer of Code 2009のフレームワークで作成されたプロジェクトを蓄積します。KDEのフレームワーク内で20以上のプロジェクトが提示されました。 次のリリース4.5では、バグとの戦いに焦点を当てることが約束されています。

最もおいしいイノベーションの特定：

  • Nepomuk Semantic Searchには 、さまざまなアプリケーションを検索するための新しいインターフェースである新しい高速バックエンド（Virtuosoストレージバックエンド）があります。 このブランチから、アプリケーションへの広範な実装が開始されます。
  • Plasmaデスクトップのさらなる開発。 KNetworkmanagerおよびPlasma Netbookを更新しました。 いくつかの新しいウィジェットが利用可能です。 一部のウィジェットはシステムトレイに配置できるようになりました。 新しいウィジェット管理インターフェース。 プラズマテーマの制御は、システムメニューに移動しました。
  • デスクトップ管理メニューの変更。 また、デスクトップに画像をドラッグするだけで、デスクトップの背景画像を変更できるようになりました。
  • KWin Oxygenウィンドウテーマのオプションとアニメーションを制御するためのオプションがさらにあります。 Aero Snap-基本的なタイルのサポート。 タブによって制御されるウィンドウのグループ化。
  • パネルのポップアップのアニメーションのサポート（OpenGLを使用）。 動的スクロール。


このリリースはQt4.6プラットフォームに基づいており、DEの一般的なパフォーマンスとメモリ消費にプラスの影響を与えます。 さて、新機能。
