Yandex Data Analysis Schoolで新しい登録が開始されます

今日、4月10日、次の、すでに6番目の、Yandexデータ分析学校への入学が始まりました。 今年、入学するための条件と規則が少し変更されました。すべての変更について詳しく説明します。







1.モスクワの「データ分析」部門と「コンピューター科学」部門

過去数年間、これらの学部への入学は2段階でした：オンラインでの距離テストと学校の教師との対面インタビュー。



当初、通信テストは簡単でした。たとえば、昨年、テストに合格しなかった候補者はわずか15％で、600人以上が面接に参加できました。 学校の管理者が各候補者に数時間を費やすことがますます困難になっていることは明らかです。



今年、面接に参加するには、フルタイムの筆記試験の課題を解決する必要があります。 数学とプログラミングのプログラムの一環として、テスト、試験、インタビューの両方が実施されます 。



誰もがアンケートに記入し、当社のウェブサイトで今すぐオンラインテストに合格できます。通信ステージは5月20日まで開催されます。



筆記試験の場合、5月21日から6月10日まで2つか3つの日付が割り当てられ、各受験者は適切な日を選択できます。



志願者が筆記試験に合格した場合、面接の時間について個別に同意します。 インタビューは7月10日まで開催されます。



2.サンクトペテルブルクのコンピューター科学センター

モスクワと同様、このセットは現在3つのステージで構成されています。筆記試験が追加され、すべての分野で同じです。 申し込みは、筆記試験の予備日である6月15日まで受け付けられます：4月14日、5月19日、6月20日。 すべてのインタビューは、6月30日までに開催される予定です。



申請は現在、2年目以上の学生からのみ受け付けています。 中学生がセンターと大学で研究を組み合わせるのは非常に困難です。



3.地域部門「データ分析」および「コンピューター科学」

支部での入学手続きは、支部での筆記試験が6月後半に開催されるという点でのみモスクワのものと異なります。



したがって、筆記試験はモスクワとサンクトペテルブルク、キエフとミンスクで開催されます。 不在で勉強する予定の人は、これらの都市のいずれかで筆記試験を受ける必要があります。



4.モスクワのバイオインフォマティクス学科

昨年、数学と生物学の両方の専門分野の学生がバイオインフォマティクス部門に入りました。 応募者を分割し​​ました。ある年には数学者のみ、別の人には生物学者のみを募集します。



数学の学生のみがこの学部に入学できます。 入学は引き続き、遠隔テストと部門長との面接で構成され、筆記試験はありません。 このページでアンケートとテストを検索できます 。



5.モスクワ州立大学の力学と数学の学部との協力

数年にわたり、学校ではMIPTおよびHSEとの共同修士プログラムがありました。 今年、モスクワ州立大学の学校と力学の新しい調整プログラムが登場します。



詳細については、当社のWebサイトをご覧ください 。 プログラムに参加するには、学校と機械学と数学のマスターの専門「コンピューター代数」の両方に独立して入学する必要があります。 このプログラムの学生には、個別のカリキュラムが作成されます。 すべての非居住者の学生には寮が提供されます。 プログラムの詳細については、マジストラシー・マジストラシーの公開日に議論されます。



学校に関するすべての質問をアドレスに送信してください ： yaschool@yandex-team.ru
