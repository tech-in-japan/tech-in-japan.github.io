マイクロソフト、Webロール向けのWindows Azure Acceleratorを発表





Microsoftは、Webロール向けWindows Azureアクセラレーターの最初のリリースを発表しました。 この無料ツールを使用して、IISサーバーを実行しているWindows Azureで実行されているWebサイトをサポートするサーバーファームを作成できます。



Acceleratorを使用すると、Windows Azureで多数の小さなWebサイトを簡単にホストでき、それらを簡単に更新および管理できます。 また、Windows Azureでホストされる大規模プロジェクトの管理を簡素化します。



アクセラレータには、1つ以上のサイトをホストするWebロールを作成するVisual Studioのプロジェクトテンプレートが含まれています。 このテンプレートを1回だけ展開することで、Web配置を使用してWindows Azureでサイトのホスティングをほぼすぐに開始できます。 Windows Azureにサイトを配置するには、約30秒かかります。 したがって、Webロール用のWindows Azureアクセラレーターを使用すると、Windows Azureにデプロイされた多数のWebサイトを便利な管理インターフェイスを備えた柔軟なホスティングに結合できるため、サポートのコストが削減され、大規模プロジェクトでもトラフィックのピークに簡単に対応できます。



Windows Azure Acceleratorには以下も含まれます。
  • ドキュメント：プロジェクトテンプレートに含まれており、CodePlexにも投稿されています。これにより、研究が簡素化され、アクセラレータを使用できます。
  • 新規または既存のWebロールにコードを埋め込むことができる2つのモジュール。 さらに、これらのモジュールは、新しいソリューションを開発するための基盤として使用できます。
  • 依存性チェッカー：インストーラーには、ASP.Net MVC 3またはVisual Studio 2010用のWindows Azureツールなど、必要なすべてのコンポーネントでアクセラレーターが正常に動作することを確認できる、新しく改善された検証プログラムが含まれています。 Microsoft Web Platform Installerと統合され、サイトの展開が簡素化されます。
