Zalman VE-200-仮想CDドライブを備えたポータブルHDD用のUSB / eSATAボックス

最後に、待望のZalman VE-200がロシアで発売されました。







2.5インチHDD用のこのボックスは、一見したところ、特に目立ったものではありません。他の多くのボックスと同じUSBおよびeSATAインターフェースです。しかし、競合他社とは異なり、このボックスには仮想CDドライブ、マウントされている* .ISO、特別なパパに折り畳まれています。しかし、これは後で、そして今： 重いレビュー。



なんで？



オペレーティングシステム/コンピューター診断プログラムを頻繁にインストールしない場合は、このボックスは必要ありません。 しかし、そうすれば、さまざまなバージョンのオペレーティングシステム、さまざまな蘇生器、MHDD、およびその他のユーティリティを備えたディスクのスタックを蓄積している可能性があります。 レイプされているコンピューターにDVDドライブが装備されている場合、これはまだ何もありませんが、装備されていない場合、通常はトラブルが多く追加されます。 ディスクイメージをUSBフラッシュドライブにアップロードするユーティリティが必要です。これらのユーティリティは常に機能しません（たとえば、Ubuntu Serverを2つのユーティリティで苦しめました。1つはオフサイトに投稿されていますが、ディスク）、たとえば、USBフラッシュドライブ上の蘇生器はリセットしません。 そして、このボックスはあなただけのために作成されました。通常のISO-shkiが挿入される仮想CD-DVD-Blurayドライブがあります。 ボクシングには、バックライト付きのLCDスクリーンがあり、その上でホイールロッカーを使用して、ボックスをUSBに接続した直後に目的の画像を選択できます。 また、ドライブからコンピューターを完全に起動することができ、ドライバーを必要とせず、WinCEからMacまでのハードウェアと互換性があります。



特徴



外部インターフェイス： USB 2.0、eSATA 3 Gb / s

内部インターフェイス： SATA 3Gb / s

重量（ディスクなし）： 99 g

寸法： 135 x 79 x 13 mm

ドライブ： 2.5 "HDD / SSD

モスクワでの販売開始時の小売価格： 1280ルーブル（45ドル）

互換性のあるハードウェア： PC、Mac（検証済み）、WinCE、組み込みOS



外観と機器



ボクシングは、従来のCDドライブよりも少し小さい箱に入っています。 すべてがきれいに詰め込まれています。ロシアの郵便局でさえ、私はそれを転送することを恐れません。







すべての仕様と機能は背面に描かれていますが、レビューを読んだ後、そこを見る必要がないことを願っています。 しかし、読み終えていませんか？ 見てみましょう。







ボクシングには黒と銀の2色があり、これまでのところ黒だけが私の手に入りました。 私の意見では、彼はより美しいです。 残念ながら、手触りが快適なボクシングアルミニウム-黒いインスタンスは非常に簡単に汚れており、指紋を収集しやすいことが判明しました。 シルバーは、私にはそう思われますが、この欠点はあまりありません。

フレームの左上には、ドライバーとボルトが入ったバッグがあります。 ドライバーのサイズと磁化されていないという事実を考えると、意図した目的に使用することを強くお勧めします。ボルトを紛失しやすくなります。 キットにはハードケースも付属しています。外側は合成皮革、内側は柔らかいベルベットです。 一般的に、多くのボックスに非常に欠けているもの。 そして、サイズを比較するために、右側にこのボックス用に特別に購入した新しい実験ハードがあります。 また、ボックスには50 cm USBケーブルと標準のeSATA-eSATAケーブルがあります。 そこには、HDDのサイズにほぼ対応する、マニュアルには示されていない透明なステッカーもありましたが、どこに取り付けるかは考えられませんでした。 彼女の写真を撮ることはできませんでした、彼女はあまりにも透明です。







そして今、競合他社との違いについて：上のボックスには、ドライブパラメータを表示し、ISOファイルを選択するための小さなLCD画面があります。 最初はそうでしたが、画面はすでに非常に傷が付いており、爪に軽く触れるたびに新しい傷が現れます。 しかし、すべてがそれほど悪くないことが判明しました。画面は薄いフィルムで密封されており、購入後に簡単に取り外すことができます。

カバーは、コントロールとコネクタを覆わないように特別に作られています。 ケースから出ることなくディスクを使用できます。 後で見るように、危険な加熱や結果はありません。







ただし、大丈夫、HDDを印刷して両方のデバイスをドッキングしようとします。 すべては問題ありませんが、ディスクが何によっても固定されておらず、SATAコネクタによってのみ保持され、ボックス内にHDDにドッキングされたスクリーンの蓋が自由に挿入されていることに少し驚いています。 蓋は、マイクロドライバーでバッグに入れられた2つの小さなボルトで固定されます。 箱の本体は、側面がプラスチックの曲がらないアンテナで閉じられ、ボルトがその下に隠れるように作られています。







ボックスの左側には、ISOファイルを選択したり、HDDの動作モードを切り替えたりするためのプッシュホイールロッカーがあります。







上部にはeSATAコネクタ、隠されたファームウェアリカバリボタンのある穴、書き込み保護スイッチ、ミニUSBがあります。







特徴



私の古いメーター長のUSBケーブルで、ボックスは起動しましたが、システムに認識されなかったため、自分の短いコードでコンピューターに接続する必要がありました。 電源をオンにした直後-「 www.zalman.com 」という碑文が表示され、画面にサービス情報が表示されます。



トップライン

閉じた/開いたロックは記録ラッチの状態を示し、円の存在（画面の粒子のため-歯車のような）はISOファイルが接続されていることを示し、黒い背景HDOの文字はHDD-ODD-DUALモードを示します。 HDDの温度は右側に表示されます（監視プログラムの読み取り値と一致するため、温度センサーから見たところ）、チェックマーク/感嘆符/クロス、ハードドライブの良好、危険（警告）または重大な状態を示します。

ボトムライン

ここではパーティション番号が表示されます。または、オプティカルモードまたはデュアルモードがオンの場合、現在のフォルダーまたは選択したISOファイルが表示されます。

画面の上-ディスクアクティビティを示すLEDがあります。







ディスクにISOがない場合、ホイールを押すと、HDDモデル、HDDのシリアル番号、ボックスのシリアル番号、ファームウェアバージョンが順番に表示されます。







ディスクを仮想ドライブとして使用するために、ディスク上に「_ISO」フォルダーを作成します。登録は重要ではありません。 すべてのISOファイルをそこに記入し、その後、たとえばコードをぴくぴくさせて、ボックスをオーバーロードします。すでに任意のファイルを選択できます。 残念ながら長い名前は切り捨てられ、ロシア語はサポートされていません（ISOはスペースから表示されますが、使用を禁止していません。メニュー構造はそれほど複雑ではありません。マニュアルを1分間勉強した後、質問は発生しません。







テスト



さまざまなオペレーティングシステムとの互換性をテストするために、Windows 7x64、Ubuntu、LiveCD、Knoppixなどのいくつかのディスクイメージがテストされました。 問題なく、CDドライブが認識し、ネットブックを使用して起動することもできます。BIOSでUSB CD-ROMから起動を選択するだけです。 Asusマザーボードを使用すると、BIOS設定に入る代わりに、F8キーを押して目的のドライブを選択し、ポップアップメニューからコンピューターを起動できます。







エミュレートされたドライブの速度を確認するのは興味深いことでしたが、残念ながら、HD TuneはCD-ROMでは動作しません。 FARマネージャーでコピーして作成したいくつかの画像からnulへの検証が容易になり、速度が3 MB / s（Microsoft Officeの画像など、画像に数千の小さなファイルがある場合）と最大20 MB / sからスムーズに変化することを確信しました-より大きなファイル。 CDドライブとUSBフラッシュドライブの速度と比較すると、Windowsのインストールが速くても、その違いは目に見えます。







Windowsのインストール中にLCD画面の温度が42度に上昇しました。 何かが時々落とされたコンピューターに接続されたドライブは、8時間の苦痛の中で40度以上には温まりませんでした。平均で約38人が踊りました。



レビューとレビューによると、このボックス用に購入したHDDは慎重に選択されており、1年半後でもWD6400BEVTがかなり合理的な購入であり続けることを確認できます-高レベルの加熱とノイズ（少し厄介な振動を除く）、信頼性と速度が非常に高いまとも。 7200のディスクは、騒音と発熱が大きいため、私には考慮されていませんでした。 唯一の顕著なマイナスは、ディスクへの最初のアクセスが遅くなることが多いため、ハードドライブが頻繁に停止することです。

読み取り速度テストの例を同封します。グラフは、ディスク速度がUSB帯域幅によって制限され、25〜30 MB /秒でスピンすることを示しています。







ファームウェアの更新後、テスト結果は少し悪くなりましたが、その差はエラーの範囲内に収まる可能性があるため、後悔しません。







eSATAはホットスワップをサポートしています;マニュアルには、最初にeSATAインターフェイスを接続し、次にUSBを接続する必要があると記載されています。 逆の順序で切断します。 残念ながら、すべてのボクシングの可能性が消え、碑文のみが画面に残ります：「eSATAモード」とアクティビティLEDのみが機能します。 温度計もオフになります。

もちろん、HP Proliant MicroServerで測定されたeSATAテストを調べます。 もちろん、速度の差は非常に大きいです。







ファームウェアの更新



はい！ ドライブはファームウェアの更新をサポートしています。新しいバージョンはzalman.comで簡単に見つけることができます。 工場出荷時のドライブには、ファームウェア1.00.00 22N、1.00.00 57がWebサイトに投稿されていました。更新するには、ZIPアーカイブからISOファイルを解凍し、_ISOフォルダーに転送し、ボックス画面のホイールでこの画像を選択します。 再フラッシュは、ディスクをフォーマットする必要があった後（データは消失していませんが、必要ですが、必要です）、ほぼ30秒続きます。 唯一の顕著な変更は、ロシア語の文字がスキップされず、代わりに正方形が表示されることです:)
