Firefox + Vim == Vimperator



特に、両方の製品を熟知しているVimperator拡張機能は 、FirefoxにVimスタイルのコントロールを追加します。



  • Vimスタイルのキーボードショートカット（h、j、k、l、gg、G、0、$、ZZなど）
  • Exコマンド（：quit 、： open www.foo.com 、...）
  • すべてのコマンドのタブ補完
  • 「Hit-a-hint」リンクのサイレントナビゲーション（「f」を押した後、リンクは1文字または2文字のショートカットで強調表示され、リンクをたどることができます）
  • ブックマークと履歴の高度な自動補完（URLだけでなくページタイトルも検索）
  • Vgetスタイルのステータスバーとwgetスタイルのプログレスバー
  • ミニマリストGUI（set guioptions =で不要なメニューとツールバーを非表示にします）
  • 機能：javascriptファイルのソースおよび〜/ .vimperatorrcファイルを使用して設定を保存する
  • シンプルなクイック検索（：open fooはgoogleでfooを検索し、open ebayターミネーターはebayでターミネーターを検索します）Firefoxキーワードブックマークと検索エンジンをサポート
  • 多くのチームでは、数量修飾子がサポートされています（3Hは3ステップ戻ります）
  • エラー時にビープ音（：nobeepをオフに設定）
  • ページにラベル「a」を設定するマーク（ma）のサポート、「a」に移動します）
  • QuickMarksサポート（go {a-zA-Z0-9}を使用して以前にマークしたページにすばやくジャンプ）
  • コマンドのサポート：マップ（およびスクリプトを書く人のためのフィードキー（））
  • すべてのコマンド、マッピング、オプションを説明する詳細なヘルプ
