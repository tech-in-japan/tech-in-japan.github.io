古いDell Latitude XPi CD

最近、祖国の大箱のどこかにある珍しいラップトップに関する投稿が定期的にハブに表示されます。 私はまた、本当に引退せずに人類のために働いている私の老人の恥ずかしがり屋の人々に遅れずについていくことを決めました。 だから、Dell Latitude XPi CD。







ご注意 たくさんの写真。







このユニットは数年前に友人のアルクソントからペニーのために購入されました。 もちろん、最初は、10年前に子供の頃からラップトップを喜んでいた。 ユニットはよく保存されており、マトリックスは明るく、バッテリーは新しいものと同じように保持され、ドライブは読み取り、ネジには不良がなく、擦り傷や傷はほとんどありません。 90年代のお気に入りの粗い黒のプラスチックで作られています。 今、彼らはそれからほとんど何もしません、すべてが光沢があります。 画面も素敵なマットです。



そして、時間が経つにつれて、ラップトップは審美的な喜びだけでなく、実用的な利点ももたらし始めました。 長年にわたり、com-portと特定のdosソフトウェアのサポートにより、使用される鉄はますます少なくなっています。 古いラジオはラジオ局をプログラミングするためのメインツールになりました。



老人のキーボードはオリジナルで、日本語の文字と少し変わったキーレイアウトがあります。 キー自体は大きなストロークで非常に高くなっています。 キーボードは作りませんでしたが、現代のロッカーではキーが吊り下げられていないように感じます。 金属元素は見えますが、固まりはありません。





ふたには普通の、しかし快適なラッチが付いています。





わかりにくい、ロゴとラッチの間にあるカバーの赤い挿入物。





ラップトップの前面。 左側には、ハードドライブスライドのラッチがあります。 ガラスのピンクの窓が最初の赤外線ポートです（はい、このラップトップには2つの赤外線ポートがあります）。 ドライブとフロップ。





左のビュー。 左から右へ：ヒンジ付きの足、オン/オフボタン、2つのPCMCIAスロット、その上の換気スロット、ネジカバー、盗難防止用の穴と標準のオーディオコネクタがあります。





背面図。 左から順に：LPT、COM、PS / 2、VGA、2番目の赤外線ポート、ドックコネクタ、電源コネクタ。





また、この写真では、すべてのコネクタが保護プラスチックカーテンで閉じられていることがわかります。





右の図は、バッテリーコンパートメントを除いて特に目立ちません。





トップビュー。 ふたのLEDインジケータは両面で、ふたが開いているときと閉じているときに見ることができます。





下からの眺め。 換気やコンパートメントの山がない、ボルト穴も豊富ではありません。





運転 かなり気まぐれで、クローズドセッションで工場のスタンピングまたはCD-Rのみを読み取ります。 また、このラップトップにデータを簡単に転送する方法は他にないため、無数のブランクを破壊する必要があります。





トラックボールにも注目したい。 トラックボール付きのラップトップはほとんど見ませんでした。 非常に便利なことですが、慣れれば手袋をはめたままで作業できます。





カバー、キーボード、および内蔵マイクのインジケータ。





ネジは顕著に取り外され、最初にスライドのスライドが動き、次にハンドルが伸びます。これにより、ネジを取り外すことができ、容量は1.3 GBです。





ハードドライブには日本語の文字もあります。





このような「mobilrek」は、ラップトップ自体と同様にかなり重い（太いネジ）です。





ラップトップはヒンジ付きの脚の上に立っており、バッテリードアは開いています。





バッテリーを取り外します。 バッテリーの隅にある白いストライプの列は、実際の充電インジケーターです（近くに小さなボタンが表示されます）。





このLED充電インジケータは、バッテリーが完全に充電されていることを示しているようです（その充電は1時間半から2時間の操作に十分です）。





電源投入直後のBIOSでは、RAMのバイトが考慮されます。 読み込み中...





BIOSの直後に、dosovoパート98ウィンドウが開きます。





もちろん、多くの人にとって馴染みのある瞬間にブート画面が点滅します。





コンピューターのプロパティ。 快適な作業には40メートルで十分です。 彼の若い同僚、8 GBで投げたり回したりするのが難しい7人。





また、システム自体をアンロードすることなく、システムのプロパティをBIOSで表示できます。 キーの組み合わせを使用すると、BIOSにアクセスして何かを変更し、OSに戻ることができます。 便利に。





BIOSから、老人にはP-150、40 Mb RAM、1.1 Mbビデオが装備されていることがわかります。 97年間、これは確かに良い構成です。





したがって、プログラミングステーション用の作業ソフトウェアに加えて...





許容可能なfpsで320x240の解像度で最初のkwakを駆動することはかなり可能です。





ラップトップは時計のように機能するという事実にもかかわらず、老人は快適な作業に十分なネットワークを持っていません。 9xウィンドウに対応したPCMCIA LANアダプタはどこにもありません。 また、予備のバッテリーを購入したいと思います。 しかし悲しいかな、これらはもはや生産されておらず、販売されていません。



私はあなたと写真を共有したいと思っていたので、テーブル上の作業の混乱を整理して写真家を招待する時間はありませんでした。 コメントの誤りや誤りを報告してください。 このラップトップについて詳しくお聞かせいただければ幸いです。
