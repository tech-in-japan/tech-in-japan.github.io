「サウンド」について簡単に



ポッドキャスト「サウンド」は2013年に登場しました。 彼の仕事は、オーディオテクノロジーと音のエコシステムについて幅広い聴衆に語ることでした。 プロジェクトの存在中、それぞれ約30分の43のエピソードを記録しました。



さまざまなデバイスの違いについて議論し、スピーカーを購入する際に注意を払うことが重要なことについて話しました。 また、いくつかの神話を解き明かし、Hi-Fi業界の動向について話しました。



次に、最も重要なトピックのいくつかを検討してください。



神話について



[ listen 、 read ]ポッドキャストの最初の問題は、ポータブルおよび固定オーディオシステムに関連する誤解に対処しました。 最も一般的な神話の1つは、小さなインイヤーヘッドフォンがオーバーヘッドよりも低音を再生することです。



実際、低周波数の伝送はイヤーパッドのサイズに依存せず、一部の「ギャグ」は低音パワーが大型ヘッドフォンに匹敵します。 低周波の「損失」の主な理由は、絶縁不良です。 したがって、インイヤーヘッドフォンの交換可能なヘッドに注意を払うことが重要です。サイズが適切でない場合、音が十分に聞こえない可能性があります。



最初の問題で話したもう1つの誤解は、カラムの製造に「間違った」材料を使用することに関するものです。 伝えられるところでは、ガラスまたは金属で作られた音響は木材よりも悪い音です。 「型破りな」素材には、音を歪ませる過剰な量の共鳴と振動があると考えられています。



この意見は、スピーカーケースをスピーカーをサポートする単純な「ボックス」と誤って考えているためです。 実際には、ケースは特定のスピーカーとその機能のために特別に開発された複雑なシステムです。 このため、コラムのデザインの素材は、関係するエンジニアの経験ほどの役割を果たしません。



3番目のエピソード[ listen 、 read ]では、プロのオーディオ機器は家庭よりも高価であるという見解に反論しました。 誤解は、「ホーム」と「プロ」の概念の混乱に関連しています。 それらは、2番目の方が最初の「アマチュア」のテクニックよりも良いとは限りません。



プロの機器には他のタスクがあります。そのような音響から、彼らはあらゆる施設（レストランやカフェ）に適した中立で「感情のない」音を期待します。 したがって、イベント用のシステムは家庭には適さず、これらのセグメントを相互に比較する価値はありません。


4番目と13番目のエピソードでは、ホームシアターに関連する誤解について話しました。 これらの最初の理由は、リビングルームにできるだけ多くのスピーカーを設置することです。 実際には、ホームシアターシステムに最適なスピーカーの数は部屋の大きさに依存します。 ほとんどの場合、サブウーファーまたは5.1スピーカーなしで4スピーカーシステムで十分です。



ポッドキャストの次の問題で、他のいくつかの神話について話しました。



  • 2番目の問題[ listen 、 read ]は、D / Aコンバーターに焦点を当てています-その中で、ヘッドフォンとプロのオーディオ機器に関連する神話についても議論しました。
  • [ listen 、 read ]の第5号では、ベースの演奏に関する誤解に反論しました。
  • 第7版[ listen 、 read ]では、音響を選択する際に技術仕様とグラフィックスのみを信頼すべきではない理由を説明しています。


専門家との会議



ポッドキャストの頻繁なゲストは、デザインエンジニア、サウンドエンジニア、起業家であるYuri Stanislavovich Fominでした。 [ 聞く ]問題の1つで、彼はアマチュアが自分の音響を作成する方法を説明しました。 彼の意見では、ターンキースピーカースキームとDIYキットがWin-Winのオプションになるでしょう。 これらのセットについては、ポッドキャストの別の号[ listen 、 read ]で詳しく説明しました。 彼はまた、製品回路がどのように開発されているか、つまりどれだけの時間とお金がかかり、音響エンジニアが何をするのかという知識も共有しました。



Yuri Fominに加えて、招待された専門家のリストには、Audiomania Artyom Faermarkの創設者兼CEOが含まれています。 20番目の[ listen 、 read ]および34番目の[ listen 、 read ]の問題で、彼は会社のブランドの作成および生産機能の歴史について話しました。



問題の1つで、サウンドエンジニアのEvgeny Pereyaslavtsevと話しました。 彼と一緒に、フィルムオーディオのプロセスについて話し合いました。 ユージーンは、彼の作品の異常な側面の1つ、つまり、自然界には存在しないサウンドの作成について話しました。 プロジェクトの1つで、彼は地面から急に成長するブドウの音を合成しなければなりませんでした。それは水を注ぐ音、目玉焼き、紙のガサガサ音から生成されました。



招待された専門家に関するその他の問題：



  • 「サウンド」の第10エピソードのゲスト[ 聞く 、 読む ]はF-Labエンジニアのアナトリー・ドミトリエヴィッチ・アルセノフでした。 ポッドキャストで、彼はデジタル形式の出現の歴史とオーディオ圧縮の原理について話しました。
  • 第24号[に耳を傾ける ]には、Hi-FiおよびHi-End機器の専門家Alexey Shubinが出席しました。 オーディオ形式（FLAC、MP3、DSF）、およびビデオについて説明し、4Kムービーについて、そしてダビングせずにホームシアターで映画を観る方が良い理由について話しました。

Audiomaniaプロジェクトについて



ポッドキャストのいくつかのエピソードは、Audiomaniaのインストールに捧げられています。 [ listen 、 read ]の第12号では、オーディオシステムのインストールに関する私たち自身の経験と発生する問題について話しました。 また、スピーカーから最高のサウンドを得るのに役立つヒントも提供しました。



その1つは、可能であれば、設置スペシャリストと一緒に家の内部を計画する段階でオーディオシステムの配置を検討することです。 専門家は、設計者が考えていないかもしれないニュアンス、たとえば音響用のワイヤの配置や操作中の機器の冷却を考慮します。



別のエピソード[ 聞く 、 読む ]では、Audiomaniaオフィスのショールームに2つの映画館がどのように作成されたかについて話しました。 これらの最初のものは「部屋の部屋」デザインです-元のオフィススペースの壁と天井に縛られていない吸音多層モジュール構造。 これにより、部屋が騒音から隔離され、音響処理の問題が解決されます。 このショールームでポッドキャストの1つを記録しました。





2番目の部屋は、「見えない」セラソナースピーカーを紹介するためのものです。 内部では見えませんが、同時に標準システムより音質は劣りません-音響はドルビーアトモス効果をサポートし、内蔵サブウーファーを含みます。



Audiomaniaプロジェクトに関する問題：



  • 第41号[ listen 、 read ]は、映画館での仕事の結果に当てられています。その中で、デザインのプロセスと結果の印象について議論しました。


ビニールについて



ビニールについても話しました。 ポッドキャスト「サウンド」の2つのエピソードは、現代のプレートとその場所に完全に捧げられています。 第18号[ 聞く ]で、ビニールプレーヤーについて説明しました-どれくらいの費用がかかるか、どのように選ぶか、そしてターンテーブルをセットアップする方法です。 この問題では、MCピックアップヘッドとMMピックアップヘッドの違いが簡単な言葉で説明されています。前者は音がより詳細で、後者は音量が大きいです。



35回目のエピソード[ listen 、 read ]で、レコードの購入とプレーヤーのケアについて説明しました。 この号では、「ターンテーブル」とビニールのメンテナンス用アクセサリーに関する情報を見つけます-プレート、ブラシ、帯電防止封筒を洗浄するためのデバイス。
