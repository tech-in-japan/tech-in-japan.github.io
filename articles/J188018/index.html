Windows NTは今日20になります





今日、システム管理者にも注意すべき点があります。伝説のWindows NTオペレーティングシステムは20年前です！







Windows NTの最初のリリースは、1993年7月27日に行われました。 これは、新しいコア上の根本的に新しいシステムであり、 OS / 2を介したMicrosoftとIBMの間の協力が終了した後に作成され始めました。 レドモンドの鼻血は、新しい安定したオペレーティングシステムを作成する必要がありました。これは今後数十年のプラットフォームであり、彼らは信じられないほどの努力をしました。 リードプログラマーのDavid N. Cutler率いるチームがNTカーネルを作成した劇的な歴史は、 「Showstopper！ マイクロソフトでWindows NTと次世代をつくるための猛烈な競争 。



デビッドカトラーは以前、200人の開発者でデジタルリサーチセンターを指揮し、そこで新しいCPUとVMSオペレーティングシステムのアーキテクチャを作成しましたが、1988年にこのプロジェクトは終了しました。 CutlerはすぐにBill Gatesからオファーを受けたため、20人のDigital従業員のグループとともにMicrosoftに移り、NTのVMSおよびUNIXオペレーティングシステムから多くのアイデアを具体化しました。 VMSとNTの重要な類似点については、 こちらを参照してください 。



NTカーネルは、Windows 8、Windows Server 2012、Windows Phone 8、Windows Azure、さらにはXbox Oneを含め、依然として広く使用されています。Mary-JoeFowley 氏はこう書いています 。



David Cutlerは現在もMicrosoftで働いており、Xbox Oneのオペレーティングシステムの主要な開発者の1人であり、それ以前はWindows Azureで働いていました。



1993年7月のNTの最初のバージョンは、Windows NT 3.1と呼ばれていました。 NT 3.5、3.51、および4.0が続きました。 Windowsの新しいバージョンでは、この番号付けが継続されます。
