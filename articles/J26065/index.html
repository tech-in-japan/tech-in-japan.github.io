ReMIXカンファレンスの印象

マイクロソフトによるロシアでの最初の会議が開催されました！ この行動は、下院のシックな円柱状のホールで行われました。 ハイテクと豪華なシャンデリア、ホール、クリスタルクリアな円柱の上昇の組み合わせは非常に興味深いです。 イベントの優れた技術組織に注目する価値があります！ 必需品はすべて大きなスクリーンに表示され、ホールのどのコーナーからでも見ることができました。 レポート中の組織株-私はまったく気付きませんでした。 これはとてもいいです！


レポートについて簡単に：

-Microsoftは現在phpをサポートしています。 php.Netの存在について聴衆から質問がありました。 現時点では、この方向にプラットフォームを拡張する見通しは明確ではありません。

-Microsoft Server 2008でのApacheの作業の最適化に取り組みます。MicrosoftのプラットフォームでLAMPコンポーネントをジャグリングする可能性を作成する予定です。

-鮮明なレポートは、MicrosoftのコマンドラインであるPower Shellに関するものでした。 通常の方法でサーバーを構成するためのオプションがさらに多くあります:)

-IE 8は標準を理解することを学んでいます。 会議中、Microsoftの代表は、会社が過去に犯した間違いに気付いていたと主張し、現在はその修正に積極的に取り組んでいます。

-SilverlightはFlashの真の代替手段になりつつあります。 毎日、150万人のユーザーがプラグインをインストールしていますが、毎日この数字は増え続けています。

イベントの集大成は円卓会議でした。 最初は有望でしたが、残念ながら建設的な対話はうまくいきませんでした。 議論の焦点はあいまいでした。

この日をありがとう！ すべてが非常に前向きで生産的でした！ この会議が伝統になることを願っています！
