マイクロソフト、ロシアのWindows Azureへのアクセスを拡大





マイクロソフトは、ロシアの顧客向けのクラウドサービスの開発において次の重要なステップを踏み出し、ロシアにおけるWindows Azureサービスの買収機会の拡大を発表します。 現在、あらゆる規模のロシア企業がポータルhttp://www.windowsazure.com/ru-ru/を購読できます 。



現在、Windows Azureサービスの無料試用サブスクリプションはhttp://www.windowsazure.com/ru-ru/で入手できます 。 90日間の期間の後、顧客はクレジットカードまたは請求書で支払うことにより、サービスの使用を継続できます。



このオファーは、Windows Azureクラウドプラットフォームの大規模なアップグレードのわずか2週間後に利用可能になりました。 これには、既存のサービスの機能の拡張と新しいプラットフォーム機能の両方が含まれます（多くのサービスが暫定版で利用可能です）。



  • Windows Azure 仮想マシン （Infrastructure as a Service、 IaaS ）： Windows Azureでオンプレミスの仮想マシンイメージを実行し、オンプレミスとクラウド間で仮想ハードディスク（VHD）を移動できるようになりました。 この場合、Windows ServerまたはLinuxの独自のイメージを使用するか、既存のテンプレートを選択できます。 新しい機能は、既存のコードを変更せずにアプリケーションをクラウドに移行するのにも役立ちます。 これにより、パートナーソリューションのWindows Azureプラットフォームへの移行が大幅に加速され、SharePointとSQL Serverの展開の可能性が広がり、時間と費用を節約できます。 Microsoftは、Windows Serverプラットフォーム（バージョン2008 R2、2008 R2、SQL Server 2012 Evalおよび2012 RC）およびLinux（OpenSUSE 12.1、CentOS-6.2、Ubuntu 12.04、SUSE Linux Enterprise Server 11 SP2）を含む、サポートされているオペレーティングシステムのリストも拡大しました。
  • Windows Azure仮想ネットワーク ： Windows Azureでは、仮想プライベートネットワーク（VPN）を構成および管理し、ローカルITインフラストラクチャに安全に接続できます。 IT管​​理者は、仮想ネットワークを使用して、IPアドレス、ルーティングテーブル、セキュリティポリシーの構成などのネットワークトポロジを管理しながら、LANをクラウドに拡張できます。
  • Windows Azure Webサイト ：この新しいサービスを使用すると、.NET、Node.js、およびPHPに基づいてサイトとアプリケーションを作成し、GitやFTPなどの標準テクノロジーを使用して展開できます。 さらに、Windows Azureでは、WordPress、Joomla！、DotNetNuke、Umbraco、およびDrupalエンジンを使用できるようになりました。これは数回クリックするだけで実行できます。
  • Windows Azure Media Services ：このメディアブロードキャストプラットフォームは、コンテンツのエンコードや保護からストリーミングや分析まで、メディア企業やコンテンツプロバイダーのビジネスタスクを解決するために必要なすべてのコンポーネントを提供します。 Android、iOS、Xbox、Windows Phoneを実行しているデバイスを含む、HTML5、Flash、Silverlightをサポートする接続デバイスと通信する機能を提供します。
  • Python および Javaの サポート ：以前に.NET、PHP、Node.jsが含まれていたサポートされているライブラリのリストは、JavaおよびPythonライブラリによって拡張されました。 更新された言語ライブラリは、 http ： //GitHub.comのオープンソースライセンスで入手できます。
  • 更新された SDK ： Windows Azure の更新された開発キットには、MacおよびLinux用の新しいコマンドラインツールが含まれています。 Visual Studio 2010およびVisual Studio 2012用の新しいWindows Azure SDK for .NET 1.7開発ツールも提供されましたWindows Azureサービスを使用するには、 Microsoft Visual Studio 2012 （無料版を含む）およびMicrosoft Visual用Windows Azureツールをダウンロードできます。 スタジオ


すべての和解業務は、ロシア連邦および他のすべての欧州諸国でマイクロソフト製品のライセンスを供与するマイクロソフトアイルランドオペレーションズリミテッドによって実施されます。 ロシア企業は、エンタープライズライセンス企業ライセンス契約を締結するか、SoftlineのAzure.softline.ruなどのパートナーサービスに加入することにより、Windows Azureサービスを購入することもできます。 Microsoft BizSpark、MSDN、Cloud Essentialsアフィリエイトプログラムに参加している企業の場合、Windows Azureへのアクセスは無料です[1]。



このドキュメント （docx）でWindows Azureへの無料アクセスの取得に関する詳細情報を見つけることができます。



Windows Azureブログのmsdn.ruおよびtechdays.ruリソースで 、クラウドプラットフォームと新しいサービスと更新に関する追加情報を取得できます。



特に、Windows Azureクラウドテクノロジーに関心のあるロシアの開発者、技術者、マネージャー向けに、2012年7月6日に無料のオンラインセミナーが開催されます。 このセミナーの枠組みの中で、2012年6月に発表された新しいクラウドテクノロジーとシナリオに関するWindows Azureクラウドプラットフォーム、ロシアおよびその他の国からのサービスの使用に関する最新ニュースを見つけることができます。 ワークショップへの登録はすでに開かれています。
