15分でYotaルーターまたはwimax2wifi

多くの場合、オフィスをインターネットに迅速または安価に接続する必要があります。 この目的のために、新しいモスクワ-サンクトペテルブルクのワイヤレスインターネットプロバイダーであるヨータは、私たちの何人かが失敗した公開テストに非常に適しています:)



asus wl-500gp http://oleg.wl500g.info/ルーター用の高度な機能を備えた代替ファームウェアを作成したOlegと、リバースエンジニアリングを実行し、Linux http://code.google用のyota-modemドライバーを作成したXanderの2人に感謝します。 com / p / madwimax / 、 yotaルーターを作成できるようになりましたhttp://wl500g.info/showthread.php?t=17005



UPD：これらの操作はもはや関係ありません。 Olegの最新のファームウェアはデフォルトでYotaをサポートしています。設定はWebインターフェースにあります。 リンク-http://code.google.com/p/wl500g/ パスカル著 。



モデムのアクティベーション手順はやや混乱しやすく、wl-500の専門家だけが精通している準備作業が必要です。 この記事の目的は、アクティベーションプロセスを3つの簡単なステップに単純化することです。

  1. 準備
  2. ファームウェア変更
  3. モデムアクティベーションスクリプトの実行。




準備



必要なもの：

  1. インターネット付きケーブル（ルーターでパッケージをダウンロードする必要があります）
  2. ルーターに常駐する不要なフラッシュドライブ（その上にfireがあります）
  3. ヨタモデム
  4. oleg.wl500g.info/1.9.2.7-9/WL500gp-1.9.2.7-9.zipを使用したOlegのファームウェア
  5. ルーターキットのディスクからのASUSファームウェア復元アプリケーション
  6. USBフラッシュドライブを下のスロットに接続し、
  7. トップへのモデム
  8. WANでのインターネット付きケーブル




ファームウェアの交換



  1. モデムの背面にあるリセットボタンを押したままにします
  2. 数秒間切断する
  3. 電源インジケータが点滅した後、リセットボタンを放します
  4. ASUSファームウェア復元アプリケーションを起動します
  5. 解凍されたOlegのファームウェアを選択します
  6. アップロードをクリックします。




ヨタモデムのアクティベーション



  1. ここでは、ルーターにインターネットが必要です。 インターネットで手動設定が必要な場合は、Webインターフェースから入力します。
  2. 起動後->実行-> telnet 192.168.1.1をロードした後
  3. ユーザー名adminパスワードadmin
  4. 以下のコードをクリップボードにコピーします
  5. 表示されたウィンドウを右クリックして貼り付け
  6. フラッシュドライブとインターネットの速度に応じて、約10〜15分待機しています。 最後に、ルーターが再起動します。 ロード後10秒で、青いランプが消え、モデムの接続インジケーターが点滅します。




umount /opt

umount /tmp/mnt/disc0_1



echo -e -nd\n1\nn\np\n1\n512\nw | fdisk /dev/scsi/host0/bus0/target0/lun0/part1 < /tmp/fdisk.cfg



mke2fs -j /dev/scsi/host0/bus0/target0/lun0/part1



mount -o sync,noatime,rw /dev/scsi/host0/bus0/target0/lun0/part1 /opt



cd /tmp

wget http://lvk.cs.msu.su/~lasaine/madwimax/wl500g/libusb-1.0-mipsel.tgz

wget http://lvk.cs.msu.su/~lasaine/madwimax/wl500g/madwimax-latest-mipsel.tgz

wget http://lvk.cs.msu.su/~lasaine/madwimax/wl500g/udhcpc-script.tgz

cd /

tar -xzf /tmp/libusb-1.0-mipsel.tgz

tar -xzf /tmp/madwimax-latest-mipsel.tgz

tar -xzf /tmp/udhcpc-script.tgz

rm /tmp/libusb-1.0-mipsel.tgz /tmp/madwimax-latest-mipsel.tgz /tmp/udhcpc-script.tgz



mkdir -p /usr/local/sbin/

echo "#!/bin/sh" > /usr/local/sbin/post-mount

echo "mount -o sync,noatime,rw /dev/scsi/host0/bus0/target0/lun0/part1 /opt" >> /usr/local/sbin/post-mount

echo "mount -o bind /tmp/mnt/disc0_1 /opt" >> /usr/local/sbin/post-mount

echo "insmod /lib/modules/tun.o" >> /usr/local/sbin/post-mount

echo "(while true; do /opt/sbin/madwimax -qof; sleep 10; done) &" >> /usr/local/sbin/post-mount

chmod +x /usr/local/sbin/post-mount



ipkg.sh update

ipkg.sh install ipkg-opt



flashfs save

flashfs commit

flashfs enable



reboot







できた！ ルーターはYotaを介してネットワークに接続されています！
