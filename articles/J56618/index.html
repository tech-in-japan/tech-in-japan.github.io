WineのトリックまたはWineでのWindowsコンポーネントのインストール

Habréにはまだwinetricksプログラムとそれが与える利点の説明がないことに驚いています。



多くの場合、Windowsプログラムを実行するために何らかのコンポーネントが必要な状況に遭遇しています。 たとえば、.NET、Mono、またはDirectX。



あなたの手でそれらをインストールする-長くて退屈な-あなたはディストリビューションを探し、ダウンロードし、パラメータを指定する必要があります...特にワインのインストールが初めて実行され、多くをインストールする必要がある状況では。



Wineコミュニティはこの問題を解決しました。







このタスクを容易にするために、優れたwinetricksプログラムが登場しました 。 最新バージョンでは、コンソールとグラフィカルインターフェイスの両方があります。 インターフェース自体は非常にシンプルです-パッケージを選択してインストールボタンを開始します。



設置





プログラムが正しく機能するには、実際のwine 、 cabextractおよびwinetricks自体が必要です。



1つ目と2つ目は配布パッケージを介してインストールできますが、WT自体は手動でインストールする必要があります。



winetricksの最新バージョンは、次の場所で入手できます。



winezeug.googlecode.com/svn/trunk/winetricks（リンクについてTiGRに感謝します ）。 以下は古い安定版です。



$ su

Password:

# wget -c www.kegel.com/wine/winetricks -O winetricks

# mv winetricks /usr/sbin/

# chmod a+rwx /usr/sbin/winetricks

# exit









wineが動作するユーザーの下でプログラムを起動する必要があります。そうしないと、ソフトウェアが間違った場所にインストールされてしまいます。 ルート権限は必要ありません。



実行します。



コンポーネントを選択してGUIを開くには：



$ winetricks









半自動インストールの場合（コンポーネントの名前と説明は--helpキーで表示されます）：



$ winetricks "components names"











Winetricksは現在、Mono、FireFox、.NET Framework、IE、DirectXなど、100種類のコンポーネントとアプリケーションを知っています。
