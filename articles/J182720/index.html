Windowsストアのアプリケーション統計

Windows 8のアプリケーションの作成者として、ストアの詳細な統計（公開されたアプリケーションの数、最も人気のあるカテゴリ、アプリケーションの価格など）に興味があります。 Microsoftはこの情報を公開せず、プログラムで受信するためのAPIもありません。 そのため、apps.microsoft.comで公開されているアプリケーションの個々のページを分析する小さなユーティリティを作成しました。 以下に、この分析の一部を示したいと思いますが、他の開発者が同じように興味深く有用であることを発見したいと思います。



公開されたアプリケーションの数



現在、Windowsストアには約86万のアプリケーションがあります。 一部は特定の地域でのみ公開されているため、ユーザーが利用できるアプリケーションの数は国によって異なります。







次のデータは、American Windows Storeから取得されます。



アプリケーションカテゴリ







今年の初めに、教育カテゴリが公開されたアプリケーションの数で1位になりましたが、徐々にゲームが1位になり、自信を持ってギャップを広げました。



申込価格







公開されたアプリケーションのほぼ95％は無料であるか、価格が2ドル未満です。 アプリケーションの数が増えても、この状況は実際には変わりません。



アプリケーションの品質







アプリケーション価格の場合と同様に、アプリケーションの数がほぼ2倍に増加したにもかかわらず、過去6か月間の公開アプリケーションの品質評価はほとんど変化していません。 ただし、少なくともある種の評価を持つアプリケーションの数は10％減少しました。
