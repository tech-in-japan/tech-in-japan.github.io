ユーロセットのコミュニケーターの修理経験-プロサービス

この悲しい話は、プロサービス会社（ユーロセットの子会社）でコミュニケーターがどのように修理されたかを物語っています。



概して、このサービスセンターは長い間修理されており、高品質ではないといううわさが長い間Webをローミングしています。 しかし、この記事の著者は実際にこれを検証することができました。



ストーリーは非常に簡単に始まりました。インターネットでHTC Touch Proコミュニケーターを購入したからです。 このデバイスの前に、フォーラムや古き良きFidoで多くのものを購入したので、購入する前に確認する経験はしっかりしていました。 しかし、Touchには奇妙な「グリッチ」があり、買収時にはそれを検出することができませんでした。ある程度長い会話の後、デバイスの底にストリップが現れ、暗い背景でしか見えず、他のアプリケーションのコンテンツを表示しました。 多くの場合、それは「キーボード」-「メッセージ」という色あせた碑文のように見えました。



また、PCTのフルセットと認証にもかかわらず、コミュニケーターにはレシートのない署名のない保証カードがありましたが、最初は私には普通のようでした（店で購入するとき、保証カードには販売日がなく、保証を「延長」することができました）。 利用可能な唯一の情報は、デバイスがモスクワのオンラインストアで購入されたということです。製造日（保証カードがない場合の保証期間の計算元）は、2008年8月8日です。



欠陥とその外観の理由は数週間後に安全に検出され、その後、新しく購入して仕事に非常に必要なデバイス（管理サーバー）は、家から車で5分のところにあるPro-Serviceサービスセンターに引き渡されました（別の場所に移動します）時間がありませんでした）。 はい、Pro-Serviceは公認のHTC c / cです。 この時間から、ユーロセットとの楽しいコミュニケーションが始まります-12/19/2008。

外見では、「プロサービス」サービスは非常にまともに見えます。これは、インターネットを介してデバイスの修理のステータスを確認する機会（ステータスが更新されないため実際に存在しません）と、このステータスを発表する美しい声で女の子を呼び出す機能の両方です（ステータスは常に発表されています） 「修理中」、「予備部品待ち」、「まだ準備ができていません」など、さまざまなバリエーションがあります）。



1月初旬、スイッチガールへの別の電話の後、修理された電話を手に入れることができることが判明しました。 到着すると、デバイスを修復するために「ソフトウェアの交換」が行われたこと、またはより簡単にフラッシュが行われたことを知ります。もちろん、マシンをc / cに渡す前に何度か自分で行いました。 さらに、エンジニアは誤動作の本質すら読んでいませんでした。これは、前述したように、会話の一定時間後に表示されます。 もちろん、タイムカウンターをチェックした後、エンジニアはまったく電話をしなかったことがわかりました。つまり、彼は欠陥を見ることができず、3週間、デバイスは「ソフトウェア更新」でc / cにありました。 デバイスには、削除しなかったファイルがいくつか残っています。 つまり、彼らは電話で何もしませんでした。



彼は苦情を書いた、私はユーロセット再生部門の代表から連絡を受け、デバイスを修理するには何らかのスペアパーツが必要であると述べたが（これは指定されていない）、注文することができますが、約2週間かかります。 その時点（2009年1月15日）でデバイスがほぼ1か月サービスにあったので、私は同意します-私はまだそれを交換するために簡単なFujitsu-Siemens T830を少しのお金で購入しました。



2月が来て、3月がそれに続きました。 デバイスのステータスは「スペアパーツを待機中」です。これは、デバイスを修復できないと書かれた論文を提供する提案です（どのような場合でも必要ありません。主に、3か月の修復後、修復したデバイスを取りたいからです）。



3月、デバイスはまだ「スペアパーツを待っている」ので、最終的にPro-Serviceサービスセンターからピックアップして別の場所で修理することにしました。そのために、レクラメーション部門に電話してデバイスを返送してください（2009/03/18）。 今日（24.03）、宅配便業者はまだ彼を受付に連れて行っています。



Euroset / Pro-Service会社とのコミュニケーションの経験がここにあります。 初歩的な非識字、合理的な修理条件の不遵守、そしてこれが完全に機能しない官僚制の背景にある場所があります。 ユーロセットで機器を購入する人々に心から同情します。
