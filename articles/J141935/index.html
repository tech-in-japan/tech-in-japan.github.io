チーム開発およびアプリケーション開発サイクルのためのTeam Foundation Server 11の革新に関する無料のウェビナー



親愛なる同僚、開発者、プロジェクトマネージャー、開発チームのヘッド、そして興味を持っています！



無料のウェビナー （ブラウザからアクセスできるオンラインイベント）に参加してください。チーム開発を編成するTeam Foundation Server 11の新機能と、プロトタイプの計画から既成ソリューションのテストまでのアプリケーション開発サイクル全体について説明します。



ウェビナーは4月19日にモスクワ時間17:00に開催され 、誰でも利用できます。 セッションには45分かかります。その後、ご質問にお答えします。 ウェビナーは録画され、録画はイベントの数日後に無料で視聴できます。



マイクロソフトイベントページの次のアドレスでウェビナーに登録するように皆にお願いします 。



Webセミナーの45分以内に、Team Foundation Server 11の新機能を紹介し、より効果的なチーム開発を実現します。 Visual Studio 11の新機能、新しいPowerPoint StoryBoardingツール、Team Web Access Webツール、バージョン管理メカニズムの革新、コードレビュー機能、およびクラウドでのプロジェクトの構築に精通します。



ウェビナーでお会いしましょう！
