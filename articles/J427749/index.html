AppleとSamsungは、古い携帯電話を遅くしたことで初めて罰金を科した





スマートフォンの「計画的陳腐化」の実証済みの事実に対して、アップルとサムスンはイタリアでそれぞれ1000万ユーロと500万ユーロの罰金を科されました。 これは、ユーザーに新しいモデルの購入を促すために、古いガジェットの特別な減速のために、ガジェットの最大のメーカーにとって最初の罰金です。



2017年12月にスマートフォンが強制的に減速したという事実はAppleによって認識され 、1か月後に米国で集団訴訟が提起されました 。 その後、対応する調査がイタリアとフランスで始まりました。



調査は、国家競争当局によって開始されました。 判明したように、一部のスマートフォンソフトウェアの更新はデバイスのパフォーマンスに悪影響を及ぼします。



独占禁止法当局の声明によると、「AppleとSamsungは悪いビジネス慣行で運営されています」、そしてオペレーティングシステムの更新は「深刻なクラッシュにつながり、生産性を大幅に低下させ、それにより電話の交換をスピードアップします」。



古いデバイスの速度を落とすことは、更新プログラムをインストールすることによるランダムな「副作用」であり、企業はそのような効果をまったく望んでおらず、あらゆる可能な方法でそれを回避しようとしています。 しかし、これはそうではありません。 調査の結果、両社は、新しいソフトウェアの影響に関する十分な情報を顧客に提供していないことが示されました。



OSやプログラムの更新をスマートフォンにインストールすると速度が低下することを人々が知っていれば、多くの人がそのような更新を拒否するでしょう。 しかし、製造会社はこれを行う機会を彼らに与えておらず、更新をインストールした後、変更をロールバックする簡単な方法はありません。



たとえば、Samsungは「Galaxy Note 4の所有者に、Galaxy Note 7の新しいバージョン用にGoogleのAndroidオペレーティングシステムの新しいバージョンをインストールするよう勧めました」。 ただし、新しいバージョンのAndroidをインストールすると、Galaxy Note 4スマートフォンの動作が悪化することがわかりました。



同様に、AppleはiPhone 6の所有者にiPhone 7向けに設計されたオペレーティングシステムをインストールするようアドバイスしました。これにより、古いモデルの所有者に問題が生じました。



これは明らかにAppleとSamsungの意図的な行動であり、計画された陳腐化の概要に適合し、ユーザーにスマートフォンの新しいモデルを購入するよう促しています。



両社はそれぞれ最高500万ユーロの罰金を受け取り、規制当局の決定を顧客に知らせるイタリアのサイトに通知を掲載するよう命じられました。 さらに、Appleは、リチウム電池の「本質的な」特性に関する明確な情報を顧客に提供しなかったため、さらに平均500万ユーロの罰金を科されました。



12月、Apple はオペレーティングシステムに 、バッテリーが長時間使用されている場合にiPhoneプロセッサのクロック速度を強制的に低下させる秘密の設定があることを認めました。 ティムクックは、ABC Newsとのインタビューで、電話の強制的な減速は、ユーザー自身の利益のために、またデバイスの使いやすさを改善するために実際に行われたと述べました。



隠された機能について知られるようになったとき、アップルは計画された陳腐化のためではなく、バッテリー設定を考慮してスマートフォンを不注意に切断/再起動することを避けるために古いスマートフォンを遅くすることを保証しました。 バッテリーの充電/電圧と周囲温度が考慮されます。 路上が寒く、バッテリーが古ければ古いほど、Appleはプロセッサーの速度を低下させます。 プロセッサのスローダウンアルゴリズムは、2017年1月にリリースされたiOS 10.2.1アップデートで機能し始めました。



その後、同社は保証なしでデバイスを交換するためのバッテリーのコストを79ドルから29ドルに引き下げました。 少数のスマートフォンモデルでのみコストを削減。 AppleはiOSにもバッテリーステータス情報を追加したため、ユーザーはプロセッサのスローダウン機能を無効にできます。



イタリアの反トラスト局は、フランスで同様の調査が開始されたのとほぼ同時に顧客の苦情の調査を開始しましたが、それはまだ保留中です。 フランスの法律によれば、販売を促進するために製品の寿命を意図的に短縮することは、年間売上高の最高5％または懲役の罰金に処せられます。
