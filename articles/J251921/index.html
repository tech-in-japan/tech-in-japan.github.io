富士通サーバーパークニュース



ますます複雑化するビジネスタスクと増え続けるデータに関連する最新の現実には、ITインフラストラクチャの近代化が必要です。 この近代化の重要な側面の1つは、サーバーインフラストラクチャの統合です。 多数の低パフォーマンスモデルではなく、少数の生産性の高いサーバーモデルを使用すると、スペース、電力消費、ライセンス料金を節約し、管理を簡素化できます。 この記事では、さまざまなフォームファクターで実装された標準アーキテクチャーで富士通PRIMERGYサーバーのラインアップを更新する方法について説明します。



「クラウド」時代では、機器の性能に対する要件が絶えず増加することに加えて、リソースを提供することの利便性の要因が最初にきます。 クラウドインフラストラクチャの急増には、サーバーハードウェアに投資されたすべてのルーブルに対する最大限の利益も必要です。



現在、富士通のポートフォリオには多くのx86サーバーモデルが含まれています。 これらはいくつかのカテゴリに分類できます-SMBおよびブランチ向けの費用対効果の高い拡張可能なサーバー。 ユニバーサルでスケーラブルな高性能ラックサーバー。 高性能コンピューティング用に設計された統合インフラストラクチャのプラットフォーム。 高性能、クラウド、ハイパーコンバージェンスコンピューティング向けの高密度プラットフォーム。 インメモリコンピューティングプラットフォームとミッションクリティカルなアプリケーション。 富士通は、ほぼすべてのカテゴリのサーバーのラインアップを更新しました。



タワーサーバー



エントリーレベルは、新しいPRIMERGY TX1310 / 1320/1330モデルで表されます。 これらは、最初のサーバーとして、1Cアプリケーション、メールサーバー、ブランチサーバーなどに使用できます。 Intel Xeon E3プロセッサに基づいて動作し、最大32 GBのRAMをサポートします。 これらのサーバーは、2.5インチまたは3.5インチのホットスワップハードドライブを使用できます。 TX1330は、ホットスワップ可能な電源もサポートしています。 これらは、キャビネット内だけでなくテーブルの下にも設置できるエントリーレベルのサーバーです。「タワー」フォームファクターで組み立てられ、ノイズレベルは22〜24 dBにすぎません。 多くの場合、専用のサーバールームがない小規模企業にとって、この点は非常に重要です。





富士通PRIMERGY TX1320 M1



平均負荷の場合、TX2540 M1は2つのIntel Xeon E5-2400 v2プロセッサに基づいて設計され、最大192 GBのRAMをサポートします。 このサーバーは、中規模企業（25〜50人）向けに既に配置されており、デュアルプロセッサマシンのパフォーマンスの向上を必要とする仮想化とロードされたアプリケーションの展開に適しています。 TX2540 M1は、その「タワー」設計と低ノイズレベルのおかげで、テーブルの下に設置することもできます。



ラックサーバー



PRIMERGYサーバーのラックセグメントで、富士通は本日4つのモデルを提供します-RX1330 M1、RX2520 M1、RX2530 M1、RX2540 M1。 このシリーズの3台の車は、RX2520 M1を除き、以前の製品ラインにアナログを搭載しています。 これは前任者のいない絶対的な新規性であり、小規模企業向けの手頃な価格のデュアルソケットソリューションです。 RX2530 M1、RX2540 M1のこのラインの古いモデルは、昨年9月に発表されたIntel Xeon E5-2600 v3プロセッサーに基づいています。 これらの各プロセッサには、1つのソケットに18個のコア（合計36個のコア）と45 MBのキャッシュがあります。 1.5 TBのDDR4 RAMをサポートします。 また、ラインの古いモデルでは、Fujitsu DynamicLom Technologyのサポートが登場しました。 これは、ネットワーク接続を担当するチップがマザーボードに配線されていることを意味します。これにより、お客様が現在必要としているネットワークインターフェイスに迅速かつ便利に適応できます。 現在、DynamicLomテクノロジーは、1ギガビット/秒の2および4ポートまたは10ギガビット/秒の2ポートをサポートしています。 将来的には、メーカーは10 Gb / sの4ポートと40 Gb / sの1ポートをサポートすることを約束します。 また、このチップは、最大8つの仮想ネットワークポート、つまり、最大8つの仮想ネットワークカードをサポートします。これにより、オペレーティングシステムは物理ポートと同様に動作できます。



RX1330 M1サーバーは、SMBに焦点を絞った低コストのソリューションです。 可能な使用シナリオは、1Cアプリケーションサーバー、メールサーバー、またはブランチサーバーです。 このモデルはIntel Xeon E3プロセッサーに基づいており、最大32 GBのRAM、最大10個の2.5インチホットスワップハードドライブ、および2つの電源をサポートします。



RX2520 M1は中負荷で動作するように設計されており、仮想化およびバックアップ用のサーバーとして使用できます。 Intel Xeon E5-2400 v2プロセッサで動作し、最大192 GBのRAM、ホットスワップ可能な機能を備えた12個の3.5インチハードディスクまたは16個の2.5インチハードディスクをサポートします。





富士通PRIMERGY RX2530 M1



古いモデルのRX2530 M1およびRX2540 M1は、高負荷に最適であり、データセンターに適しています。 Intel Xeon E5-2600 v3プロセッサを搭載し、最大1.5 TBのRAMと最大12個の3.5インチハードドライブまたは28個のホットスワップ可能な2.5インチハードドライブをサポートします。 また、SATA DOM / USB / UFMブートローダーのインストールも提供します。



最高の負荷に対して、RX4770 M1ハイエンドモデルが設計されています。これは、ERPまたはCRMシステムのサーバー、データベースサーバー、または仮想化として使用できます。 このモデルは、Intel Xeon E7-4800 v2プロセッサー（最大4プロセッサー）に基づいており、最大6 TBのRAM、2.5インチのSASハードディスクドライブを最大8台、またはホットスワップ機能付き2.5インチのPCIeハードドライブを最大4台サポートします。 ハードドライブをPCI-eに直接接続できるため、パフォーマンスが向上します。



クラウドとクラスター



クラウドインフラストラクチャおよび高性能クラスターPRIMERGY Scale-Out Systemsのサーバーファミリも更新されました。 新しいブレードモデルBX2560 / 2580は、製造元によって統合サーバー、アプリケーションサーバー、インフラストラクチャサーバー、または仮想化サーバーとして位置付けられています。 Intel Xeon E5-2600 v3に基づいており、最大16 TBのDDR4 RAMスロットまたは最大24 TBのECC RAMスロットがあり、最大1.5 TBをサポートします。 これらのモデルは、最大2台のホットスワップ可能なSAS / SATA 2.5インチハードドライブまたは1.8インチSSDを使用します。





富士通PRIMERGY BX2560 M1



また、DynamicLomテクノロジーとSATA DOMまたはUFMブートローダーのインストールもサポートしています。 これらのモデルは、2月から富士通パートナーに注文できます。 2つのノードCX420 S1（Box in Box）を備えたサーバークラスターは、非常に手頃な価格のWindows Server 2012に基づく中小企業向けのソリューションです。



ソフトウェア



富士通は、サーバーにOEMソフトウェアも提供しています。 特に、Windows Server Foundation 2012 R2オペレーティングシステムには、小規模オフィス向けの最新モデルが付属しています。 このシステムにより、Microsoftは新しいサーバーで大規模メーカーのみに配布できます。 このオファーは、Windows Server Standard 2012のみをインストールできる小規模メーカーの製品よりも費用対効果に優れています。お客様は、ソフトウェアサポートパッケージを富士通から直接購入できます。 特に多くのアプリケーション向けに、富士通は事前に構成されたソリューションを提供しています。 これらには、たとえば、VMware vSan Ready Nodeが含まれます。 Microsoftや他の企業のソフトウェア製品にも同様のソリューションがあります。 別の構成ソリューションは、最大50台の仮想マシン用のETERNUS DX100 S3ストレージシステムに基づくvShape 50です。 特にSymantec Backup Exec 2014ソフトウェアを使用したバックアップの場合、RX2520サーバーに基づくターンキーソリューションがあります。



結論として、富士通の標準アーキテクチャサーバーは市場で低価格で競争力のある価格で販売されており、その設計は最大の信頼性を提供するように特別に設計されており、業界で最高のコンピューティング密度を提供していることも注目に値します（ユニットマウントラックあたり）。
