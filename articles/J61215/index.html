Zend PHP 5認定エンジニアになる方法

それは起こった、乾杯！ 今、 私はZCEの名誉ステータスを持っています:)



そのため、認定の準備の経験を共有すると同時に、PHPの認定に関する小さな教育プログラムを実施したいと思います。







認定はZendによって実施されています。この会社はPHPのコア開発の最前線にあり、引き続きこの分野の新技術を積極的に推進およびサポートしています。



PHPの人気により、このような問題が発生しました。多くの開発者がいますが、実際にプロはどれですか？ 分解するのはそれほど簡単ではないので、産業認証規格を作成することが決定されました。



このような困難な作業のために、 PHP開発者のコ​​ミュニティのリーダーが集まって、彼らは順番にPHP4のテストの最初のバージョンを固定し、少し後にPHP5のテストに置き換えられました。 私が実際に引き渡しました。



ちょっとした発言があります。 西洋では、特に州では、認定された専門家が認定されていないよりもはるかに高く評価されていますが、ロシアでは、あなたのステータスを確認する紙の存在はかなりプラスですが、前提条件ではありません。



疑問が生じますが、なぜ認証が必要なのですか？

私にとって個人的には、挑戦であり、テストでした。その挑戦のために、自分で準備することにしました。 行くことができるかどうか、そして実際に-知識/経験/スキルのレベルを評価することは有用です。 まあ、ZCEのステータスも不必要ではありません:-)



すぐに言ってやった。 私はサイトに登録し、同じ場所で認定バンドルを取得しました。これはオールインワンセットのようなもので、費用は165ドルで、認定の権利、10回のテスト、 認定ガイドが含まれています。 PEARSON VUE Webサイトで、地域のテストセンターを選択し、試験の日時を設定しました。 PEARSON VUEには、ほぼすべての大都市に地域テストセンターがありますので、問題はないはずです。



次に、テストの形式について詳しく説明します。 英語の試用テスト（および認定自体）。 70の質問、すべてについてのすべてが90分与えられます。 ほとんどないようですが、実際にはこれで十分です。 多くの質問は率直に言って、「サブエンファシスを使用して」、コンマがスキップされ、括弧がどこにあるかを言うので、非常に注意する必要があります。 特に、関数の知識、文字列を操作するためのすべての基本的な（そして非常に多くのエキゾチックな）関数に関する質問を思い出しました。 そしてもちろん、 print（）とecho（）の違いを理解してください:-)

しばらくの間、あなたはマニュアルに成長する必要があります。



たとえば、このスクリプトは何を出力しますか？



<?php

echo '3' + '0.5' , print('2') + 1;

?>









当然、データベース、SQLで「手探り」する必要があります。 この領域で新しいPHP5機能を使用する複雑さを理解することは、単に必須です。 試験では、PDO、MySQLi、SQLite、トランザクション、準備されたステートメントに関する質問が肺に出くわすかもしれません。



質問は「オープン」または「クローズ」のいずれかです。 特別な入力フィールドで「開く」と答える必要があります。 通常、「クローズ」回答には4つの回答がありますが、その中から指定された数の真の回答を選択する必要があります。



XMLドキュメントの解析などの緊急のトピックもtopicしみません（SimpleXML、DOM、SAX）。 SOAP、REST、XML-RPCについては、少なくとも1つが他とどのように異なるかを知る必要があり、実践が望ましいです。



このセクションをどうにかして回避する前に、ストリームとネットワークプログラミングのトピックに関する豊富な質問に特に「喜んで」いました。



XSSとCSRFの違いを知っていますか？ まあ、そうでない場合は、特別な文献を読むことを歓迎します:-)これは認証用です。 htmlentities 、 htmlspecialcharsおよびstrip_tags関数などを使用することのすべてのニュアンスを説明する準備が必要です。 など つまり、セキュリティは最後の場所ではありません。



もちろん、PHP5ではオブジェクトの処理が大幅に改善されたため、試験のOOPとデザインパターンの知識は役立ちます。



当然のことながら、PHPを長期間使用していても、その機能の多くを認識していない場合があります。 したがって、文学を読むことは、悲しいかな、すべて英語で必要です。

私は必要な本とリソースのリストを投げようとします：



  • PHP ArchitectsのZend PHP 5認定試験ガイド第2版。
  • Zend PHP認定実践テストブック
  • PHPArchitectのPHPデザインパターンガイド
  • ロバート・リチャーズによるプロPHP XMLおよびWebサービス
  • PHPおよびMySQL Web開発。 第4版。




一般的に、できれば深く理解する必要があります。 「覚えるのは難しく、戦うのは簡単だ」ということわざを覚える時間です。:-)



すべてを考慮したようです。 PHP認定は知識を向上させ、整頓する絶好の機会であると付け加えます。 誰もが努力するものを手に入れます。
