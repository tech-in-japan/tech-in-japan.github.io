Windows 2008 Server RDP署名

昨日、私は署名付きRDP接続ファイルを作成する必要に対処しなければなりませんでした。



標準のRdpSignユーティリティを使用しました。 署名されたファイルは動作を停止しました。 Googleにアクセスして、素晴らしいリンクを見つけました： www.tech-archive.net/Archive/Windows/microsoft.public.windows.terminal_services/2008-06/msg00063.html



簡単に説明すると、署名を追加した後、ファイルは正しく保存されず、Unicodeヘッダーはファイルに書き込まれません。 （2バイトの「ユニコードヘッダー」（0xFF、0xFE）。）修正-作成したファイルをメモ帳で開き、保存します。

たぶん誰かが役に立つでしょう。
