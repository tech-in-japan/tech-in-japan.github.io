バージョン7.1への前post無料アップデート

このトピックは、個人的なWindows-XP、Vista、Windows 7を実行しているPCを保護する必要に直面していない場合、UnixライクおよびモバイルOSのユーザーにとっては興味深いものではありません。



先週、Agnitumは、Outpost製品の無料バージョンのすべてのユーザーにバージョン7.1への移行を提供しました（ Outpost Freeの公式Webサイトを参照）。



検討すべき3つの理由（または無料の統合された前post基地の使用を開始）：



1.無料です。 代わりに、Agnitumは登録メールのみを受け取り、1年間キーを送信します。



2. Outpost Firewall Free 6.5のユーザーには、2009年4月のバージョン6.5のリリースからアップグレードされた保護が提供されます。



Outpost Free 7.1の主な変更点：
  • Windows 7の互換性
  • すべての製品モジュールの改善された自己保護
  • ファイアウォールで作成されたネットワークルールの数の制限が削除されました
  • システムガード（構成内の「システム保護」）-Windowsブロッカードライバーのロードとの戦いを保証するプリセット
  • ウイルス対策とスパム対策-必要に応じて、ファイアウォールとHIPSの基本機能に加えてそれらを使用します。


3. 2010年11月にリリースされた統合アンチウイルスであるOutpost Security Suite Free 7.0のユーザー向けに、バージョンを更新すると以下が提供されます。
  • 単一の「マルウェア対策」の一部としての2番目のウイルス対策モジュール（「ウイルス対策+スパイウェア対策」）、以前は無料版で無効にされていた
  • ウイルス対策モジュールの自己防衛の改善
  • 自動更新（デフォルト）と手動更新をオンデマンドで選択する機能+いつでも更新機能を利用できる
    • そして、あなたは払えません、はい。
