USB 3.0 PDKが利用可能

USB 3.0 xHCI Peripheral Developer Kit（PDK）が利用可能になりました

USB 3.0 PDKは、周辺機器の開発者を対象としています。

xHCIベースの2枚のPCIeカードで構成されています。



キットの内容：

  • PCIe NECカード
  • PCIeカードFresco Logic
  • SuperSpeed USBケーブルメーター
  • PDKドキュメントとUSB30CVツールキット（xHCIドライバー付き）


戦いに熱心な開発者の熱意をわずかに冷やすことができる唯一のことは、セットあたり1,500ドルの価格です。



注文するには、 フォームに記入する必要があります



私たちはたくさんの良い興味深いデバイスを待っています！
