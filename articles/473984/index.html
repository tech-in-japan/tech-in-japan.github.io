Data Science Digest (October 2019)





Welcome all!



Meet the latest issue of the digest with a selection of interesting and useful materials under the cut.



Articles



  • The heading "Read articles for you." July - September 2019 - reviews of scientific articles from members of the Open Data Science community from the channel #article_essense. If you want to receive them before everyone else - join the community !
  • How to Learn Data Science for Free is a good article that describes how to learn Data Science, including resources.
  • The State of Machine Learning Frameworks in 2019 - analysis of preferred frameworks in the industry: what they choose and where everything is moving.
  • Google ML / AI Comic - a fun comic strip from Google.
  • Naive Bayes Classifier From Scratch in Python - article on the algorithm of the "naive" Bayesian classifier, how it works and how to implement it from scratch in Python.
  • Develop k-Nearest Neighbors in Python From Scratch - article on the k-nearest neighbors algorithm: how it works and how to implement it from scratch in Python.
  • MLOps Tooling - A small overview of several tools (Kubeflow, MLFlow, SageMaker, Dask and Rapids) for creating and managing machine learning models.
  • Turn Python Scripts into Beautiful ML Tools - article about the new Streamlit framework for quickly creating custom ML applications.
  • Overview of feature selection methods - a review of existing feature selection methods for teaching tasks with and without a teacher.
  • TensorFlow 2.0 + Keras Overview for Deep Learning Researchers is an excellent notebook to start learning TensorFlow 2.0 + Keras with many examples.
  • Learning a unified embedding for visual search at Pinterest - article about how Pinterest visual search works.
  • Applying deep learning to Airbnb search - An article about what happened at Airbnb when they switched from standard machine learning approaches to deep learning.
  • Intelligent Scanning Using Deep Learning for MRI - An article about how GE uses TensorFlow to determine specific anatomy during an MPT brain scan to improve speed and consistency.
  • Using Deep Learning to Inform Differential Diagnoses of Skin Diseases - an article about a system that can distinguish between 26 skin diseases.
  • A large collection of good Data Science materials .
