Happy Farm Demo Day：ハッピーハーベスト！

親愛なる友人！



7月20日にビジネスインキュベーターで開催されるHappy Farm Demo Dayにご参加ください。 15:00から。







DemoDayの一環として、HappyFarmの常駐スタートアップ（24チーム）がプロジェクトを投資家に紹介します。 また、プログラムには、卒業式とHappy Farmの最初の誕生日も含まれています！



形式：スタートアップ（24社）の3分間のピッチ、パートナーと投資家によるスピーチ、非公式のコミュニケーション。

Happy Farm Demo Dayは幸せな機会です。

投資家向け -CISで最も有望なITスタートアップについて学ぶ-直接！

スタートアップ向け-HappyFarmインキュベーター内でビジネスを発展させるユニークな機会と、9月に始まる第3サイクルに関するすべての詳細について学びましょう！

ジャーナリスト向け -シリコンバレーでの起業家精神とベンチャービジネス（イゴールショイフォット、イゴールリアベンキー、オーデッドヘルモニ、ジョンノルドマーク）のスターとして、HappyFarm Boardのメンバーと、ウクライナ、ロシア、ベラルーシ、その他のCIS諸国の投資家やスタートアップとチャットするため

参加者：国際ベンチャーファンド、投資家、ビジネスエンジェル、ウクライナおよびCISのビジネスインキュベーターの代表、スタートアップ、ジャーナリスト、ブロガー。



場所： c。 Schastlivoe（キエフからボルィースピリに向かって7 km）、st。 Lesia Ukrainka、14歳、HappyFarmビジネスインキュベーター。 アクセス方法： http : //happyfarm.com.ua/eng/bi/contacts/

登録が必要です。 場所の数は限られています。 こちらから登録してください： http : //happyfarmdemoday.eventbrite.com/
