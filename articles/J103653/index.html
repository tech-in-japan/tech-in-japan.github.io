9月18〜19日、キエフのWeb標準



この秋、キエフのWeb開発者は幸運です。9月18日と19日の週末に、 Web Stanadrds Daysの会議とModern Layoutというマスタークラスの2つのイベントが一度に待っています。



Web標準の日



大きなテント「 Web Standards 」がキエフに届きました！ モスクワ、サンクトペテルブルク、ミンスク、リガでの様子は 、当社の主力Webサイトのイベントページで確認できます。 キエフでは、次のプログラムが期待されています。

  1. インターフェイスの設計について、およびAdobe Fireworksについて少し説明します。 ヴァディム・パツェフ （ヤンデックス）
  2. Webパフォーマンスの概要：W3C、Webkit、IE9、および未来。 Nikolay Matsievsky （WEBOソフトウェア）
  3. ウェブが曲がっています。 再生SVG。 Vadim Makeev （Opera Software）
  4. ...および他のいくつか、プログラムが指定されています。
ミーティングは、10月40日のアニバーサリーアベニュー70にあるミールホテルの会議室で開催されます。 会議への参加は無料です。簡単なフォームを送信して、サイトに登録するだけです。



モダンなレイアウト



さて、日曜日に、謙虚な僕はスリーインワンのワークショップを開催し、セマンティクス、スタイル、グラフィックスなどの現代の組版の側面について話します。 マスタークラスの構造は次のとおりです。

  1. HTML：セマンティクス、ネーミング、テクニック、コードを扱う原則
  2. CSS：スタイル管理、ハッキング、後方互換性
  3. グラフィックス：スライス、最適化、結合
  4. ...および質問への回答
参加するには、 SmartMe Webサイトに登録する必要があります。 今回の参加は支払われますが、熱狂はありません。 マスタークラスページのすべての詳細。
