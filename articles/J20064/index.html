Zonbu-OS Gentoo Linuxを搭載した珍しいミニコンピューター

警告：トピックは、 この投稿の情報への追加です。



ゾンブと呼ばれる珍しいコンピューターが発売されました。 目新しさは、統合されたグラフィックコア、VIA C7プロセッサ（1.2 GHz）、512 MBのメモリ、4 GBのフラッシュドライブ、Wi-Fiモジュールを備えたマザーボードで構成されるミニPCです。 開始するには、ユーザーはモニターとキーボードをマウスで接続するだけです。



一見、すべてが非常に標準的なように見えます。デスクトップセグメント専用のASUS EeePCアナログのようなものです。 ただし、それほど単純ではありません。



Zonbuは、情報を保存するために特別なサービスを使用するため、主に興味深いです。 インターネットに接続した後、コンピューターはデータを暗号化した後、自動的にフラッシュドライブからサーバーに情報を転送します。 ドライブの内容に対するその後のすべての変更も考慮され、サーバーに保存されます。 Zonbuが機能しなくなっても、データは失われません。 必要に応じて、ユーザーは他のコンピューターからサーバー上の自分の情報にアクセスできます。 残念ながら、このサービスは無料では提供されていません。月に1回、約375ルーブル（15ドル）をフォークする必要があります。



Zonbuに付属するソフトウェアのリストには、Gentoo Linuxオペレーティングシステムと約20種類のプログラムが含まれています。 コンピューターを個別に購入し、サーバーに2年間のサブスクリプションを発行すると、デバイスの価格は約2.5千ルーブル（$ 100）になります（1年の場合-5,000ルーブル）。 サブスクリプションなしでは、Zonbuは7.5千ルーブルのどこかで買い手に費用がかかります。





•プロセッサー：1.2GHz、VIA C7 ULV

Intel互換の超低エネルギー使用

•コンパクトフラッシュローカルストレージ：4GB

•グラフィック：VIA CX700M2 UniChrome PRO II、VGA出力、最大2048 x 1536（1600万色、75 Hz）をサポート

•オーディオ：内蔵スピーカー、マイク、ヘッドフォンポート

•キーボードおよびマウス用のPC互換ポート

•6つのUSB 2.0ポート

•イーサネット：10/100 Mbpsビルトイン-ブロードバンド対応

•Wifi：ビルトイン802.11b / gはオプションとして利用可能

•電源：20ワットAC-DCアダプター

•測定：4.875 "x 2.175" x 6.675 "または（124mm x 56mm x 170mm）

•重量：2.1lbs（0.97kg）
