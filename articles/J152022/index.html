管理者向けの10のWindows Phoneアプリ

残念ながら、管理者の仕事では、何かがうまくいかなかったときに定期的に状況が発生しますが、この何かを緊急に修正する必要があります。 この時点で、すべてをすばやく確認して修正するために大きなコンピューターを使用していない場合、タスクは複雑になります。 このような状況では、スマートフォンを使用できます。この場合、SSHまたはRDPを介したリモートマシンへの接続は問題になりません。



不可抗力がない場合でも、大きなデスクトップやラップトップの代わりに、ごく普通のことであれば、モバイルで軽量のスマートフォンを使用できます。



この投稿では、Windows Phone向けの10の便利なアプリケーションについて説明しました。これらのアプリケーションは、システム管理者の生活を楽にします。



モバイルサーバーの統計

1.99ドル

これは、Windows Server / PC用のリモート監視ツールであり、モバイルデバイスからのサーバー統計のリアルタイム監視を可能にします。 アプリケーションを機能させるには、 www.mobileserverstats.comサイトから無料のサーバーコンポーネントをインストールする必要があります。



ダウンロード可能なサーバーコンポーネントには、最も必要な統計（システム、プロセッサ、ディスク、サービス、実行中のプロセス、ユーザー、グループなどの状態）のコンパイル済みWMI要求が含まれていますが、再構成して新しい要求を追加できます。 アプリケーションの唯一のマイナス点は、ロシアのマーケットプレイスにないことです。



sshクライアント

無料で

簡単な名前は、アプリケーションの本質をすぐに明らかにします。SSHクライアントを使用すると、コマンドラインからサーバーに接続して管理できます。 このアプリケーションでは、複数の接続プロファイルを保存し、秘密鍵を使用した認証をサポートできます。 アプリケーションは、SSH1、SSH2、およびTelnetプロトコルをサポートします。



クールなリモコン

無料で

これは、Windows XP / 2003 / Vista / 2008 / Windows 7で動作するリモートデスクトップクライアントです。モバイルサーバー統計と同様に、リモートマシンでクライアントを使用するには、無料のサーバーをインストールする必要があります。



アプリケーションは、すべてのシステムの組み合わせを含むキーボードの操作を完全にサポートし、マルチモニターシステムを操作する機能を備えています。 リモートマシンに接続するには、手動接続設定を使用するか、ローカルネットワークのスキャン後にリストから選択します。



便利な追加機能は、Cool Remoteのサーバー側をインストールした後、Webブラウザーを介してリモートマシンに接続できることです。



RemoteDesktop

69ルーブル

Cool Remoteとは異なり、このアプリケーションはRDP接続を使用するため、Windows XP、Vista、7、および2003 Serverを実行しているコンピューターに接続するために追加のサーバーアプリケーションは必要ありません。 RemoteDesktopは、ネットワークレベル認証（NLA）テクノロジー、複数の画面解像度、およびより快適な操作のためのズームをサポートしています。



Petri ITナレッジベース

無料で

これはナレッジベースwww.petri.co.ilの便利なモバイルクライアントで、Windows Server、Exchange、Active Directory、SQL Server、Windows 7、仮想化、ルーティングなどに関する何千ものトレーニング記事、ヒント、ガイドが含まれています。



グループポリシー検索

無料で

グループポリシー検索は、グループポリシーとWindowsオペレーティングシステムのさまざまなバージョンのレジストリに関するMicrosoftの最も包括的なガイドです。 このアプリケーションを使用すると、スマートフォンのオペレーティングシステムのシステムレジストリまたはコンピューターで動作するグループポリシーで必要なパラメーターを直接見つけることができます。 アプリケーションでは、キーとポリシーを検索する製品を指定できます。



SubCalc Subnet Calculator

無料で

これは、小さくても便利なアプリケーションで、IPアドレスとサブネットマスクをすばやく計算できます。 システム管理者である場合、サブネットとネットワークマスクのアドレスを計算するためにこの計算機に注意してください。



Pingdomパルス

無料で

Webサーバーを管理している場合、おそらくWebサーバーの可用性を評価するためのPingdomサービスを知っているでしょう。 Pingdom Pulseを使用すると、現在のステータスや応答時間など、すべてのサーバーの可用性の簡単な概要を確認したり、過去30日間の各サーバーの詳細な統計を確認したり、httpサーバーの可用性をすばやく確認したりできます。



このアプリケーションは、Pingdomの有料アカウントと無料アカウントの両方で機能します。



私のサーバー

無料で

このマイクロソフトの公式アプリケーションを使用すると、Windows Phoneのスマートフォンを使用してサーバーリソースにシームレスに接続できます。 My Serverアプリケーションを使用して、ユーザー、デバイス、アラートを管理し、Windows Server 2012 Essentialsの共有ファイルにアクセスできます。







私のPCを起こす

無料で

お使いのコンピューターのネットワークカードがWake-On-LANプロトコルをサポートしている場合、このアプリケーションは、将来の使用のためにリモートでウェイクアップできるようにします。 必要なのは、すべてのコンピューターのネットワークカードのMACアドレスとそれらが配置されているネットワークのIPアドレスをアプリケーションに追加することだけです。
