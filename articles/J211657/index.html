ケースは、光の下でわずかに光る硬質で滑らかな高品質のプラスチックでできています。 上部には銀色のフレームに挿入されたカメラがあります。 左下隅には、アクティブなスタイラスのある巣があります。

スクリーン



1920 x 1080の解像度のSuperIPS +テクノロジーを使用して作成されたFonepad Note 6画面の物理ピクセル密度は336 ppiです。 これにより、画像の完璧な滑らかさと鮮明さが保証され、優れた色再現と広い視野角が実現します。 ただし、何らかの理由でデフォルトのカラーレンダリングに満足できない場合は、ASUS Splendid独自のユーティリティを使用して、画像の色温度、色相、彩度を調整できます。



音



ここでは、すべてのASUSデバイスと同様に、SonicMasterテクノロジーが使用されます。 Fonepad Note 6には、デバイスの上下に縦向きのステレオスピーカーが装備されています。 スピーカーを通して聞くときの音質は良好で、いくつかは低周波数では不十分です。 ヘッドフォンを使用する場合、状況は劇的に変化し、サウンドはバランスがとれ、中域がわずかに高くなります。



セルラー



タブレットには、3Gをサポートするセルラーモジュールが装備されています。 デバイスは、携帯電話のカバレッジが悪い場所でネットワークを自信を持って保持します。 モバイル3Gインターネットを使用する場合、最高の速度インジケータは次のとおりでした。







スタイラス



ASUS Fonepad Note 6には、正面から見たときに右下隅に隠れたアクティブなスタイラスが装備されています。 スタイラスは非常に太く、持ちやすく、指が疲れません。 スタイラスを取り外すと、デバイスが自動的にロック解除され、ASUS SuperNote独自のアプリケーションが起動します。 さらに、ドットだけでなくスタイラスアイコンでマークされた、追加で作成された仮想デスクトップで開きます。















SuperNoteでは、簡単なメモから複雑な図面、さらにはイベントのリマインダーまで、さまざまな種類のメモを作成できます。 テキストは、手書きとテキストの両方で保存でき、仮想キーボードで入力するか、認識モードを有効にします。 認識の品質と速度は非常に良く、私の曲がった手書きのエラーの割合は驚くほど小さかった。 スタイラスは圧力に敏感で、線の太さが増します。 スタイラスのボタンを使用して、画面フラグメントを選択してスクリーンショットとして保存したり、別のデバイスに送信したり、外国語から翻訳したりすることもできます。 例として、デバイスが認識できた筆跡を示します。







スタイラスを取り出したときに表示される左側の小さな青いショートカットに注意してください。それを引っ張ると、スタイラスまたは指で書き込み/描画できるグリッドが全画面に拡張されます。 線の太さ、色、形状のスタイルを広範囲にカスタマイズでき、単純な幾何学的形状の自動認識を選択できます。 レイヤーがサポートされています。 閉じるときに、結果を画像として保存するように求められます。



































スタイラスをスロットから取り外すと、SuperNoteが自動的に閉じて、コントロールパネルのロックが解除されます。 スタイラスを挿入するのを忘れた場合、5分ごとにスロットにスタイラスがないことを示すリマインダーが表示されます。 これと他の多くの機能を無効にすることができます。















ちなみに、取り外し/取り付け時に音による通知をオフにしても、振動アラートはオフになりません。

スタイラスを使用した切り抜き作成機能は次のように機能します。スタイラスボタンを押したままにすると、画面上に選択矩形の対角線が描画されます。 次に、ボタンを放すと、選択した領域が保存されます。







任意の形状の領域を選択する必要がある場合は、まずツールバーの「一時停止」ボタンをクリックする必要があります。 次に、スタイラスボタンを押したまま画面上に関心のある領域を描画し、回路を閉じた後（青色に変わります）、ボタンを離します。 選択した領域が画像として保存されます。



















ソフトウェア



Fonepad Note 6には、SuperNoteに加えて、独自のリーダープログラムであるマイライブラリ、カレンダー、電子メールマネージャー、非常に便利な自動辞書翻訳機のインスタント辞書が付属しています。 それ以外の場合、アプリケーションのセットは標準です。

性能

おわりに



ASUS Fonepad Note 6には、FullHD解像度を備えた優れた画面、アクティブなスタイラス、セルラーモジュール、優れたパフォーマンスなど、間違いなく多くの利点があります。 同時に、デバイスは4Gネットワ​​ークでの作業をサポートしていません。 ASUS Fonepad Note 6は、スマートフォンとタブレットの間の一種の境界線である移行デバイスとして、同時に7インチタブレットと同等の条件で競合しますが、サイズは大型スマートフォンに匹敵します。 これは、小さな画面のスマートフォンを必要とする人にとっては素晴らしいオプションですが、大きなタブレットは既に大きすぎて不快です。
