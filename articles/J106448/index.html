（a）RSS Moreプラグイン



今月初めに実施した投票の結果に基づいて、すべての回答者のニーズを満たすために（a）RSS Moreプラグインを開発することにしました。



機能的



まあ、すべて同じように、投票に関するコメントで繰り返し表明されている妥協ソリューションを発言する価値があります-最良のオプションは、ユーザーが自分にどのRSSオプションを読むかを自分で決定できるようにすることです-完全または単なるアナウンス。



そのため、プラグインは2つ目の追加のテープを作成します。実際、これが主な唯一の目的です。



別の小さな点ですが、大きな記事の場合、<！-more->タグを再利用すると、完全なRSSモードでも記事をトリミングできます。



ダウンロードする



RSS Moreプラグイン（a）は、Wordpressリポジトリで入手できます： http : //wordpress.org/extend/plugins/a-rss-more/



設置





  1. プラグインでアーカイブをディレクトリ/ wp-content / plugins / a-rss-moreに解凍します
  2. プラグインを有効にする（a）RSS More
  3. RSSフィードを完全に有効にする（ [設定]»[読み取り] ）
  4. テーマが自動フィードリンクをサポートしている場合、それだけです
  5. サポートがない場合は、header.phpファイルに移動し、新しいRSSへのリンクを手動で追加します


<link rel="alternate" type="application/rss+xml" title="Anounce Feed" href="http://domain.com/feed/?announce=1" />

<!-- or -->

<link rel="alternate" type="application/rss+xml" title="Anounce Feed" href="http://domain.com/?feed=rss2&announce=1" />







設定



[設定]»[読み取り]にはいくつかの設定が非表示になっています。
