ITインフラストラクチャの6つの傾向：2018年の予測

今年の第4四半期は、在庫を取り、将来の計画を立てる時期です。 1年前の2017年に、私たちはその年にどの技術が開発されるかというトピックに関する予測を発表しました。 多くの仮定が正当化され、過去数か月はセキュリティと仮想化の兆候の下での時間として本当に記憶されていました。



今日、業界の専門家によると、来年の業界にとって重要となるテクノロジーのリストを再度まとめました。





/写真ケン・ティーガルディン CC



1.サーバー、スイッチ、ディスクの費用が増加する



ここで驚きはありません。 鉄生産者の収益は一年中増加すると予想されていました。 そのため、この事実を新しいトレンドと呼ぶことはほとんどできません。 ただし、状況を詳しく調べることをお勧めします。



IDCによると、2017年の第2四半期に、クラウドインフラストラクチャコンポーネントの販売による収益は、前年比で25.8％増加して123億ドルになりました。 さらに、パブリッククラウドのインフラストラクチャ収益のシェアは現在、市場の33.5％を占めていますが、これについては後で詳しく説明します。



IDCは、サーバー、ネットワーク、ストレージ機器の売上増加を発表しました。 2018年に関しては、Interop ITXテクノロジー会議の前夜に、ITエグゼクティブの調査が将来の投資について行われました。 回答者の24％が、組織は来年、ITインフラストラクチャへの支出を10％以上増やす予定だと述べています。 さらに21％が5〜10％を満たす予定です。



調査の結果から判断すると、企業の資金調達を増やすことは、次の理由によるでしょう。



  • データの増加-回答者の21％は、今後12か月以内にストレージ機器に投資すると答えました。
  • セキュリティの脆弱性に関する驚くべきニュース-回答者の59％が、ネットワークセキュリティを3つの優先事項の1つとして評価しました。
  • 不十分なネットワーク帯域幅-調査対象の19％がネットワーク機器に真剣にお金を費やしています。
  • ワイヤレスの未来-回答者のほぼ半数がワイヤレステクノロジーに投資します。


2.データの量が増え、メディアのサイズが小さくなっています。



このトピックについては、前の段落で既に触れました。 しかし、これは計画の重要なポイントであるため、さらに詳しく説明します。 数年間、彼らはデータ量の増加について積極的に話し続けてきました-2015年、シスコは2017年にデータセンターに保存されるデータの総量が370 EBになると予測しました。 IDCによると 、2013年から2020年までの世界のデータ量は10倍に増加します（最大44兆ギガバイト）。 驚くことではないが、企業はこれらすべてのアレイをどこに保存するのか心配しているのではないか？



機器メーカーのWestern Digital（WD）は、少し前に将来のストレージについて非常に重要な発表を行いました 。2020年までに、全データの70％がHDDに置かれ、現在積極的に開発されています。 WDは、メディアのサイズを増やすことなくディスク容量を増やす方法を採用することにしました。 マイクロ波磁気記録（MAMR）の技術に依存して、同社は最大40 TBのHDDを作成する予定です。 WDは、スーパーホストハードドライブの最初のエンジニアリングサンプルが2019年に企業顧客に利用可能になると述べています。



IBMはフラッシュメモリを搭載しています。 10月、企業はソリューションとソフトウェアモジュールにいくつかの変更を加えました。 コストを削減し、プライベートクラウドの展開を加速することを目指しています。 変更はIBM FlashSystem 900システムに影響を及ぼし、現在では、より多くのデータを保存することでユーザーのコストを削減しています。

3.インターネット接続の速度が増加している



2018年には、IoT産業の成長も予想されます。 スマートシティのインフラストラクチャを開発しているIntelは、この傾向を背景に良好な財務実績を予測しています。 IoTはデータと通信に依存しているため、来年はクラウドコンピューティングと新しい通信規格がさらに注目を集めます。



クアルコムとベライゾンは、テンポの速い未来に向けて意欲的に動きます。 10月のプロセッサメーカーは、モバイルプラットフォーム用のチップセットを使用した最初の5G接続を正常に完了しました。 近い将来、モノのインターネットのデバイス間でのデータ交換の速度の向上が期待されます。 ちなみに、ガートナーによると 、その数は2020年までに200億に達します。



シスコは 、2016年の67％と比較して、ビデオが2021年までにすべてのインターネットトラフィックの80％を占めると主張しています。この傾向に対する答えは、より多くのデータを短時間で処理できる次世代モバイルネットワークの開発です。 そして、変更は2018年に開始される可能性があります。 ロシアについては、今後12か月以内に5Gネットワ​​ークの開発センターを作成します。 ネットワーク自体は、産業施設から断片的に国に展開されます。



同時に、ノキアはIoTと5Gで共同作業を行うためにAmazon Web Servicesと提携しました。 パートナーシップは、ネットワークとITインフラストラクチャをより緊密に統合する必要性から生じました。 これは、モノのインターネットのコンテキストにおけるクラウドコンピューティングの役割の増大を示唆しています。



4.人工知能の開発が続く



どういうわけか、私たちはすでに、クラウドが人工知能技術と機械学習のサービスを積極的に取り入れていると主張しなければなりませんでした。 2018年は、この傾向の新しいレベルを示す可能性があります。これは、ほとんどすべての技術の巨人がこの方向に取り組んでいるためです。



昨年、IBMはAIベースのクラウドプラットフォームを発表しました。 マイクロソフトとアマゾンは傍観しませんでした 。 ガートナーは 、学習し、適応し、自律的に行​​動するための準備を行うシステムを作成することが、2020年までテクノロジープロバイダー間の主要な「戦いの場」になると考えています。



同時に、人工知能は、奇妙なことに、同じ人工知能から来るサイバー脅威との戦いで大きな役割を果たすと予測されています。 ハーバードビジネスレビューが書いたように、「AIの助けを借りて組織化された攻撃に対する保護の主な希望はAIです。」

5.ハイパーコンバージドインフラストラクチャの市場は発展しています



HPEのJesse St. Laurentは、業界が情報の増加にどのように対処するかについての彼の考えを最近発表しました。 彼は、問題を解決する鍵は、データを合理的に使用および処理する技術であるという結論に達しました。 ハイパーコンバージドインフラストラクチャはこれを実装できます。その利点については、 資料の 1つで説明しました。



この傾向は以前にも強調されていましたが、2017年は将来の発展に向けて良いスタートを切りました：市場は新しいプレーヤーでいっぱいになり始め 、ハイパーコンバージドシステムの売上は今年第1四半期に64.7％増加しました。



6.クラウドサービスの人気が高まっています



最初の段落の調査から判断すると、 XaaSはピークにあり、重要なトレンドのままです。 回答者の10％だけが、クラウドサービスの価値を自分自身で認識していませんでした。 残りは、彼ら自身の機器への印象的な投資の実行可能な代替としてそれらを選択します。



2018はテクノロジーにとって非常に重要です。 IDCの予測によると、その終わりまでに、ハードウェアとソフトウェアおよびサービスの両方に費やされるITの40％がクラウドに集中します。



クラウドサービスロシア会議で、ロシアのIDC代表とCIS Dmitry Gavrilov は 、2017年にロシアのクラウドサービス市場は3億6,400万ドルに達すると述べました。



2017年に、クラウドに移行した企業の数を目撃しました。 この傾向は、「戦略的機動」を実行するためにここで大きな柔軟性を獲得している新興企業を回避していません 。 顧客の経験は、たとえば電子貨物輸送プラットフォームやシティコールコンタクトセンターの場合に 、IaaSを使用してコストを効果的に管理できることを示しています。



クラウド市場のその他のいくつかのトレンドは、来年も継続して開発され、カンファレンス「クラウド2017：主なトレンド」で注目されました。これらは、新しいサービスおよび標準ソリューションとSLAの進化によるプロバイダーのポートフォリオの拡大に関連しています。



主な傾向を支援する重要なサブトレンドは、企業をクラウドに移行するためのツールの市場の拡大です。 したがって、2018年には、記録的な数の組織にとってクラウドテクノロジーへの道が開かれると期待できます。
