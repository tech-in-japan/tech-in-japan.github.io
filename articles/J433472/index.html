Unity 2018.3リリース

Unityの新しいバージョンがリリースされました。これはすでにユーザーが利用できます。 Unity 2018.3には、開発者が生産性を向上させ、多面的で美しい世界を作成する機会を提供する、プレハブのワークフローの改善、視覚効果グラフ（プレビュー）、更新された地形システムなど、2,000を超える新しい機能、修正、改善が含まれています。





Unity 2018.3の主な変更と改善



  • 改善されたワークフロープレハブ-Unityユーザーから最もリクエストされたアップデートの1つ。 プレハブは、再利用、制御、およびセキュリティに重点を置いて改善されました。 これらの更新は、ネストのサポートを提供するだけでなく、ワークフロー全体を根本的に変更します。 プロジェクトで作業する人数に関係なく、プレハブを使用する方が安全で効率的ではありませんでした。



    プレハブに関するビデオ（4ピース）








  • ビジュアルエフェクトグラフ -この使いやすく柔軟なノードベースのシステムは、最先端のVFX撮影ツールをイメージして作成されており、アーティストはゲームやその他のプロジェクトの優れた特殊効果をリアルタイムで作成できます。 シンプルで複雑な効果のために数百万のパーティクルを作成でき、上級ユーザー向けのカスタムノードを作成するためのAPIも含まれています。



    2ビデオ


  • 更新された地形システム -新しい地形ツールは、開発者のパフォーマンスと使いやすさを向上させるように設計されています。 操作をGPUに転送すると、ゲーム設計者は、より高速なツール、より大きなブラシサイズ、改善されたプレビュー、および自動シームステッチを使用してタイルの境界線を描画する機能にアクセスできます。 HDRPとLWRPの両方をサポートするための新しい設定が行われました。







  • FPSサンプルプロジェクト -UnityはFPSサンプルをリリースしました。これは、開発者に接続されたマルチユーザーFPSへのアクセスを提供します。 このサンプルをダウンロードし、出発点として使用して、HDRPなどの最新テクノロジーの使用方法を学習したり、次の接続ゲームの出発点として使用したりできます。 これは、Unityの新機能を紹介し、ユーザーを刺激するために更新されるライブプロジェクトです。







未来に私たちを待っています



  • MegaCity Demoは、データ指向設計に対するUnityのアプローチの新しいビジョンを表すデモです。 Entity Component System（ECS）に基づいて構築されたこの開発により、450万メッシュ、10万の独立したオーディオソース、60 FPSで5千の動的車両のレンダリングを備えた大規模なストリーミング環境を作成できます。 このデモはスマートフォンでも表示され、次世代のUnityコンピューティングテクノロジーのスケーラビリティを実証しました。 MegaCityデモの作成に使用されるデータ指向の技術スタック（ECS、C＃ジョブシステム、バーストコンパイラ）要素は2019年に予定されています。



  • Cinecast （実験版）-CineCastは、リアルタイムで映画のように見えるゲームシーケンスから映画シーケンスを作成できる人工知能システムです。 ゲームのリプレイをサポートし、コマーシャルのシーンを「撮影」し、キャスティングとeスポーツを新しいレベルに引き上げます。 Cinemachine上に構築されたこのシステムは、現在のゲームの状況を認識している強力なAIツールを作成者が制御できるようにします。 これにより、何が起こっているのかをベストショットで選択する機会がいつでも得られます。



  • MARSプロジェクト （実験バージョン）はUnityの拡張機能であり、ARに関する約束を満たし、開発者が実際にユーザーがコードを書く必要なく、現実世界のあらゆる環境とインテリジェントに対話するアプリケーションを作成できるようにします。



  • Project Tiny（プレビュー）-新しい高モジュラーランタイムとUnityエディターモードにより、インスタントゲームだけでなく、小型、軽量、高速のゲームを作成できます。 ポータブルでコンパクトなランタイムは、ハードウェアの弱いデバイス、ウェアラブルデバイス、またはWebで動作します。





フルリリースには、45の新機能、250の変更と改善、1915のバグ修正が含まれています。 同じ3つの購入オプションがあります-愛好家（プラス）には月額25〜35ドル、スタジオおよびフリーランサー（プロ）には月額125ドル、初心者向け（無料）は無料です。
