Flash Playerバージョン10.2.152.32の「Press Esc for ...」警告メッセージのパッチ

（ベータ10.2以降）バグ修正の分野では、フラッシュプレーヤーのバージョン10.2が良い結果をもたらしたことが長い間知られています。 プレーヤーにパッチを適用して、2番目のモニターにフォーカスを切り替えたときにフルスクリーンから折りたたまないようにする必要がなくなりました。 「Press Esc for ...」という碑文のパッチの必要性（または望ましさ）が残っていました。



2011年3月4日、プレーヤーの新しいバージョン10.2.152.32が登場しました （2011年1月27日から10.2.152.26でした）。 パッチのバージョンを更新する正当な理由。 現在は1つのパッチが関連しているため、必要なものはすべて、通常、 フォーラムの読者がすばやくすばやく投稿します。videohelp.com/ threads / 304807-How-to-remove-annoying-Press-Esc-to-message-in-Flashブランチ-ビデオ メッセージスレッドの最後に、最新バージョンがよく表示されます。 次に、トピックの最初の投稿に配置されます。 パッチ方法は以前に使用した方法とは少し異なります。すべての言語の警告メッセージはすべて削除されます。 プラグインのバージョンは多言語になり、このパッチはオペレーティングシステムの各国語に適しています（WinXP 32ビットおよびWindows 7 64ビットでテスト済み）。



フォーラムスレッドには、FF-OperaとIEの新しいバージョンのプレーヤー用の2つのパッチが適用されたファイルがあります（文字通り1日で反応しました）。 十分ではありません-Chromeの場合。 ここでは、3つのファイルすべてを単一のアーカイブに投稿します。 （トピックがあなたにとって興味深い場合、それをサポートすることを忘れないでください、さもないと他の読者はそれを見ません。）





ファイルのバージョンを確認します-ファイルを右クリックします-プロパティ-バージョン-ファイルのバージョン。

以下で説明する手順に従って手動でパッチを適用するか、ファイルをパッチを適用したファイルに置き換えることができます（Windowsを除く他のシステムでは、テストされていません-自分でパッチを適用してください）。 （古いバージョンのプレーヤーのこのトピックに関する以前の記事： 1、2 。 ）



1. 16進エディター（16進のファイルコードを検索および変更できるテキストエディター）を使用します。 例：Cygnus Hex Editor FREE EDITION、UltraEdit、その他多くのプロのエディター。



2. IE（Internet Explorer）の場合、必要に応じてfpdownload.adobe.com/get/flashplayer/current/install_flash_player_ax.exeからインストーラーをダウンロードして実行します（IEでは自動的にダウンロードが開始されず、プレーヤーは古いバージョンのままになります）。

Install_flash_player_ax.exeバージョン10.2.152.32は現在ダウンロード中です。 他のブラウザーの場合は、 get.adobe.com / en / flashplayerからダウンロードするか、ブラウザーを再度開きます。



3.すべてのブラウザーとその補助ウィンドウを閉じます（タスクマネージャー（Ctrl-Shift-Esc）で、ブラウザープロセスが閉じていることを確認できます）。



4. 16進エディターでファイルを開きます。

4.1） C：\ WINDOWS \ system32 \ Macromed \ Flash \ NPSWF32.dll （WinXP、または他のOSおよび他のインストールパスに類似）バージョン10.1.102.64。

4.2） C：\ Documents and Settings \ <ユーザー名> \ Local Settings \ Application Data \ Google \ Chrome \ Application \ <browser version number> \ gcswf32.dll （WinXPの場合、または他のOSおよび他のインストールパスの場合）バージョン10.1 .103.19（はい、ここではバージョンが少し異なることが判明しました）。 Google Chromeのバージョンが複数ある場合、最新の最新バージョンが更新され、このフォルダーのファイルが開かれます。

4.3） Win7の場合、Operaが使用するフラッシュライブラリには、タイプ％PROGRAMFILES（X86）％\ Opera \ program \ plugins \ NPSWF32.dllがあり、％PROGRAMFILES（X86）％には通常「c：\プログラムファイル（x86）。」 このファイルがある場合は開きます。



5. IEの場合。 Flash10n.ocxの保護を解除します。

5.1）ファイルC：\ WINDOWS \ system32 \ Macromed \ Flash \ Flash10n.ocx（WinXPの場合、または他のオペレーティングシステムおよび他のインストールパスの場合）を別の場所に移動し、読み取り専用属性をコピーから削除します。

5.2）ファイルFlash10n.ocxを元に戻し（システムを移動しないと、ファイルで何もできなくなります）、HEXエディターで開きます。
