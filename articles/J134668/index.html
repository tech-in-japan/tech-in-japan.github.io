Questliスタートアップストーリー：シベリアからグローバル市場へ

あなたが高品質の製品を持っている場合、それに対する投資家は自分であると考えられています。 しかし、あなたがシベリア出身で、あなたが少しの運が必要な場合、適切な場所に適切なタイミングでいて、単に歩かないようにするために、それを追加したいと思います。



今日、私はノボシビルスク市の一人のヒーローの話をします。その名前はダニラ・コジヤトニコフです。 彼は、タスク（クエスト）を完了しながらお金を稼ぐことができるクラウドソーシングWebプラットフォームであるQuestli.comプロジェクトの創設者です。



進取の気性に満ちた若い男のダニラは、2010年にLe Web 2010の無料チケットの形で運命の贈り物を受け取りました（これがその年の抽選の最後のチケットでした）。 そしてそれだけです。 これで、運は終わり、就業日が始まりました。 宿題のおかげでダニラと彼のチームがこのような驚くべき結果を達成できたため、 TechCrunch Disrupt 2011でオーディエンスチョイスアワードを受賞し、パリのLe Web 2011で講演者になりました 。









運命の贈り物を正しく適切に使用するために、ダニラは何をしましたか？



-10日間でチームと一緒に、彼は長い間考えていた製品のプロトタイプを作成しました-Questli 。

-さらに、私はLe Webに来るベンチャーキャピタリストとビジネスエンジェルの基盤を研究しました。 イベントで利用可能なPresdo通信システムのおかげで、 私は事前にそれぞれに連絡しました。

-私は自分の製品について首尾一貫して話せるように集中英語コースに行きました。

-好奇心の強い投資家向けに、ビジネスモデル、可能な販売チャネル、そして最も重要なこととして、投資する価値がある理由について回答リストを作成しました。

-チームと一緒に、彼らは明るいミーム-「実用的なプロトタイプがあり、投資家を探しています！」という碑文のTシャツを思い付きました。

-彼はLe Web会議のディスカッションパネルで講演者に質問し、主催者とモデレーターから大きな関心を集めました-Loic le Mer。

-潜在的な投資家とのコミュニケーションに利用可能なすべてのオプションを使用して、潜在的な投資家の全体像を把握し、潜在的なパートナーの包括的な分析を実施しました。 （彼は3日目にチューリッヒに侵入し、パリから予定よりも早くLe Webを離れたベンチャーキャピタリストと話をしました）。

-彼はコミュニケーションに柔軟性があり、投資家のアドバイスに耳を傾けました。 その結果、Redalpine Venture Partnersからの資金と、サービス開発に関する多くのヒントを受け取りました。



私にとって、この物語では、鉄の論理と主人公の創造性が重要な役割を果たしており、その背景には彼の運があり、それがダニラをル・ウェブにもたらしました。 もちろん、私は抵抗できず、QUESTLI.comに投資したNicholas Berg（Redalpine Venture Partners）に質問をしましたが、この取引で彼の主な動機は何になりましたか？ シベリア出身の見知らぬ男に彼の財政的手段をどのように委ねることができるだろうか？







そして、私はこの「シベリアナッツ」が何でできているのかを理解するためにダニラ・コジヤトニコフ自身と話をしましたか？ 若い起業家を動かしているのは何ですか？そして最も重要なことは、彼自身の意見で運の秘密は何ですか？







私の側では、そのような物語がロシアに存在することを嬉しく思います。 そして、若者はまず第一に、結果を達成するのに十分な技術的知識、忍耐力、コミュニケーションスキルを持っています。 ダニラの代わりに、あなた方それぞれがそうすることができます、主なことは、全体のポイントが良い状況ではなく、信仰と勤勉であることを理解することです。 Questliの主な成果はまだありません。今、アイデアからグローバルビジネスを構築し、プロジェクトの収益化モデルを実際にテストする必要があります。
