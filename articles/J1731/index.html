Windows Vista RC1は9月5日にリリースされます

本日より、Microsoft は Windows Vista Release Candidate 1（RC1） ディストリビューションを使用して、選択したブラウザーにディスクを配布し始めました 。 RC1への無料アクセスは9月5日に開始される予定です。 これは、ほぼ半額のプログラムを無料でダウンロードする良い機会です。



RC1のリリースは、新世代のVistaオペレーティングシステムの開発がフィニッシュラインに到達したことを意味します。 さて、公式リリースの前に、マイクロソフトは最後の瞬間にいくつかのバグを修正する以外は、何もする時間がありません。



Windows Vistaは11月（ほとんどの場合、11月上旬）に法人顧客に販売され、2007年1月に小売販売が開始されます。 Home Premiumバージョンは270ドルで販売されます。 Windows XPからVista Ultimateへのアップグレードには、270ドルとHome Premiumへの180ドルの費用がかかります。
