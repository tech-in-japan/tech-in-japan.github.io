Microsoft Photosynth：B.オバマ氏のあらゆる面からの就任

私たちの多くは、米国の44人のオバマ大統領の就任式に個人的に出席することはできませんでした。 しかし、そのほぼ直後に、CNNはPhotosynthテクノロジーを使用して、さまざまな角度からアクションを観察できるページを作成しました。つまり、ある写真から別の写真に移動し、同様の領域に自動的に接続されました。







ページを表示するために必要なのは、マシンにインストールされたSilverlight +優れたインターネットチャネル+欲望です:) CNNウェブサイト上のリンク： edition.cnn.com/SPECIALS/2009/44.president/inauguration/themoment



さらに、就任式の写真がある場合は、それらを一般的なデータベースに追加して、Photosynthシーンをさらに完成させることもできます。



この技術の詳細については、Webサイトphotosynth.net/about.aspxをご覧ください。

Photosynthの作成方法については、 mslabs-777.vo.llnwd.net / e1 / inc / movies / Photosynth_HowTo_500k.wmvのHOW-TOビデオをご覧ください。
