SynologyがRackStation RS10613xs +およびRX1213sasを発表





Synologyは 、エンタープライズクラスの新しいNASサーバーであるRackStation RS10613xs +の発売を発表しました。これは、最大400,000ドライブの超高性能で最大106ドライブまで拡張可能です。 さらに、新製品には、信頼性の高い継続的な運用を保証するために、いくつかのレベルの冗長性があります。



RS10613xs +は、SAS / SATA 6GインターフェイスとSSDドライブをサポートし、8つのRX1213sas拡張モジュールが接続されている場合、最大424TBの容量を増やすことができます。 新しいRackStationは、Xeon Quad Core 3.3 GHz CPUと最大32GBまで拡張可能な8GB ECC RAMを搭載しています。 RS10613xs +は、デュアル10GB接続を使用して、RAID 5モードで2,000 MB / s以上の読み取り速度を実現します。 さらに、SSDキャッシュを使用すると、集中的なランダム読み取りにより、デバイスのパフォーマンスが50％以上向上します。



Synology High Availability（SHA）に基づくクラスターソリューションは、ユーザーにとっては感知できないほど、すべてのサービスを、作業を中断することなく、故障したサーバーから相互接続デバイスのグループから別のアクティブユニットに転送します。 最適かつ最大のシステムパフォーマンスを保証するために、RS10613xs +とRX1213sas拡張モジュールの両方に冗長電源とファンが装備され、デュアルデータバスを介して互いに接続されています。



現在、RS10613xs +は、Microsoft Windows Sever 2012 Hyper-VおよびVMware VSphere 5 VAAIの仮想環境を作成する大規模サーバー向けのストレージソリューションを提供し、Citrix XenServer構成もサポートしています。



RS10613xsの主な機能+：

  • Xeon 3.3GHzクアッドコア
  • 8GB DDR3 RAM（最大32GBまで拡張可能）
  • SASおよびSATA IIIのサポート
  • 最大106ドライブのスケーラビリティ
  • 2000 MB /秒を超える読み取り速度、1800 MB /秒を超える書き込み
  • 4ギガビットポート（オプションのデュアル10GBEインストール）
  • ホットスワップハードドライブのサポート
  • VMware VSphere 5 VAAI、Citrix XenServer、Microsoft Hyper-Vをサポート
