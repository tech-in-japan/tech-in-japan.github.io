OpenMac：nonameコンピューターでMac OS Xになるかどうか

Mac OS Xの人気の高まりとAppleのIntelプロセッサへの移行を考えると、これは予想されることです。 これまで知られていなかったメーカーである米国マイアミのPsyStarは、4月14日にMac OS X 10.5“ Leopard”の変更されていないバージョンを実行できるコンピューターの供給を発表しました。



最初のツバメはOpenMacと呼ばれ、すべての人が利用できるコンポーネントに基づいて組み立てられます。 表示価格は399ドルです。 400の場合、Intel Core 2 Duo E4500プロセッサ（2.2 GHz）、2ギガバイトのDDR2 667 RAM、Intel GMA 950グラフィックスを統合したマザーボード、SATAインターフェイスと20倍速のDVD±Rドライブを備えたコンピューターを自由に使用できます。 250ギガバイトのハードドライブ。 これらはすべてAsus TM-211パッケージに組み込まれ 、最も重要なこととして、Mac OS Xを実行できるようになります。 さらに、購入時に、110ドルでGeForce 8600GTに基づいたビデオカードを、さらに50ドルでFirewireコントローラをインストールできます。 原則として、両方とも、自分でインストールしてみることができます。



比較のために、599ドルのMac miniは、1.83GHz Intel Core 2 Duoプロセッサ、ギガバイトのRAM、80ギガバイトのハードドライブ、および小型システムユニットでのFirewireサポートを提供します。



OpenMacの出現は、Appleコンピュータの公式に承認されたクローンが存在した時代を思い出します。 これは、会社がギルアメリオによって管理されていた90年代で、急速に市場シェアを失っていました。 そのため、1997年までは、Apple System 7を実行していて、元のMacよりも高速で安価なPower Computing、Motorola、またはUMAXによって製造されたコンピューターを見つけることができました。 注目すべきは、彼らが大規模なPCキャンプの兄弟のように見えたことです。 Mac互換コンピューターの生産のためにライセンスを発行することが、市場で足場を得る唯一の方法であると信じられていました。



実践では、ライセンスされたクローンはAppleを助けませんでした。ジル・アメリオは解雇され、Mac OS 8.0のリリースでスティーブ・ジョブズが代わりになりました。 時間はジョブズが彼が何をしていたかを知っていたことを示しています。



OpenMacは、当時のクローンとはまったく似ていません。 最初の、おそらく主な違いは、PsyStarが何らかの形でAppleと連携する可能性が低いことです。 PsyStarによると、OpenMacはライセンスされたMac OS Xオペレーティングシステムをインストールし、オリジナルのインストールディスクを提供するという事実にもかかわらず、Mac OS Xのライセンス契約の条件の1つに直接違反します。 Appleによって承認されていないコンピューター上のこのAppleソフトウェア、または他の人を支援するため 。



AppleがOpenMacのような革新を防ぐためにあらゆる努力をすることは間違いありません。 PsyStarのウェブサイトは、ニュースの直後に「Appleの法務部©でモーニングコーヒーが飲まれるとすぐに」応答を停止しました© 。 ただし、サイトの作業の中断は、Appleの商標に問題がないように最初の日にOpen Computerに名前が変更されたと思われる新製品への関心の高まりによって引き起こされる可能性があります。



また、OpenMacが登場する前は、EFIの変更またはEFIエミュレーターの使用を必要とする通常のパーソナルコンピューターにMac OS Xをインストールする方法が既にありました。 また、PsyStar社は、ニュースページへのアクセスに成功したほか、通常のコンピューターにMac OS Xをインストールする際に新しいことを何も言わなかった。OpenMacはEFIエミュレーターを使用してオペレーティングシステムをインストールおよび実行しているため、1つも出荷しないかもしれないOpenMac。ただし、おそらく応募者が不足することはないでしょう。



それでも、あなたはどう思いますか？パンドラの箱はすでに開いており、Appleはアジア太平洋地域の多くのノーネーム企業からのクローンの攻撃に備える必要がありますか、それともスティーブ・ジョブズはすべての競合他社とen深い会社から取り残されますか？ 意見はさまざまです。 誰かがAppleの独占権を擁護していますが、誰かがこれは、iPhoneのロック解除の冒険のように、Appleがその条件をどこでも口述することができない最初の兆候だと信じています。 次に何が起こりますか？
