ウルトラブックソニーVAIO Z（SVZ-1311V）のビデオレビュー





Sony VAIO SVZ-1311-第1世代のIntel Core i7 / i5プロセッサに基づいて構築された新しい13.3インチSony VAIOウルトラブック。 これらは、シリーズの前世代と同様に、超薄型（17 mm）カーボンファイバーケースのエレガントでエレガントなモデルです。 ノートブックには、LEDバックライト、HD +解像度（1600 x 900ピクセル）またはFullHD（1920 x 1080ピクセル）および非常に広い視野角のマットVAIO Display Plusディスプレイが搭載されています。 ドライブとして、新しいアイテムはRAID 0の構成で高速で信頼性の高いSSDドライブを使用します。それらの容量はモデルによって異なり、128または256 GBです。
