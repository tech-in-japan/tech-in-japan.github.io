このディストリビューションキットには、BlueJとGreenfootという2つの新しいプログラミングツールが用意されています。これらはJavaプログラミングを学ぶ人にとって興味深いものです。 2016年10月以来、Raspbian Jessieは新しい作業環境PIXELを使用しています。

軽量分布



ルブントゥ



Lubuntuは、Ubuntuベースの軽量でエネルギー効率が良く、信頼性の高いLinuxディストリビューションです。 このオペレーティングシステムを実行するには、少量の使用メモリ（512 MBのRAMで十分）、つまり軽量OSで十分です。 LXDEデスクトップと一連の軽量アプリケーションを使用し、速度とエネルギー効率に重点を置いています。







LubuntuはLXDE GUIを備えたほぼUbuntuです。 機能的なLubuntuディストリビューションは古いマシンを対象としていますが、新しい機器でも問題なく機能します。 ユーザーフレンドリーなグラフィカルインターフェースに加えて、Lubuntuはユーザーにとって快適な作業を保証する幅広いアプリケーションを提供します：ウェブサーフィン、メディアコンテンツの操作、ゲーム。



Xubuntu



Xubuntuは、Ubuntuベースのシンプルで美的に魅力的なOSです。 XfceはこのOSのグラフィカル環境です。 ラップトップ、ネットブックへのインストールに最適で、要件が低いため、古いマシンにもインストールできます。







Xubuntuには、ライブCD（ディスクから作業できます）と代替インストールCD（インストールのみ）の2つのオプションがあります。 安定した、きちんとした実用的な配布。 現在のバージョンはXubuntu 16.04で、コードネームはXenial Xerusであり、Linux 4.4カーネル、LibreOffice 5.1.2.2、Mozilla Firefox 45.0.2に基づいており、Thunderbird 38.6電子メールクライアントであるThunar 1.6.10ファイルマネージャを備えています。



パピーリナックス



RAMから単純なPuppy Linuxを実行します。OSは古いマシンにインストールされ、ハードドライブはオプションです。 メインOSを変更せずに、追加でインストールすることができます。







Slackwareベースの小型で高速なPuppy Linuxディストリビューション-インストールイメージをUSBフラッシュドライブに取り込み、LiveCDモードで実行します。 システムをセットアップするための多数のツールとプログラムが存在する場合、日常の作業に必要なソフトウェアはすべてありますが、システムは高速ですが、プログラムは別々のモジュールに書き込まれ、必要に応じてプラグインし、メモリを節約します。 古い鉄のための若者のエリクサー。



マンジャロLinux



Manjaro Linuxは、幅広いユーザーがArch Linuxのパワーとシンプルさを利用できるように設計されています。 OSはArch Linux +のすべての利点を吸収しており、初心者にとってより便利でアクセスしやすいです。 デフォルトの環境はXfceです。 複数のコアをサポートします。 Pacmanパッケージマネージャーによる強力な配布。







Manjaro Linuxの主な機能は、カーネルの簡単なインストールと削除です。 ディストリビューションはArchLinuxに基づいていますが、Manjaroにグラフィックドライバーをインストールするのは簡単です。 Manjaroは、速度、パワー、効率、ユーザーリポジトリ（AUR）へのアクセスに固有です。



Arch Linux



Arch Linuxは裸のディストリビューションであり、最小限のパッケージセットとコンソールインターフェイスが付属しており、柔軟に設定できます。 OSをカスタマイズし、ほぼすべてのグラフィカルインターフェイスをインストールして、それらを自由に切り替えることができます。







議論の余地のない利点の中には、パッケージの更新速度があります。 Linuxを学びたい-このディストリビューションはあなたのためです。 Arch Linuxには良いArchwikiドキュメントがあります。 公式リポジトリとユーザーリポジトリが利用可能です-AUR（Arch User Repository）を使用すると、ソフトウェアをコンパイルしてインストールできます。インストールは、ターミナルを介して、テキストモードで手動で実行されます。 パッケージマネージャーがあります-Pacmanはpkg形式です。



小さなコア



Tiny Coreは15 MBのLinuxミニディストリビューションで、低電力ハードウェアでも動作します。 テキストモードでのインストール。 Tiny CoreはBusyBoxカーネルバージョンとFlwmウィンドウマネージャーを使用します。







利点-かなりのアプリケーションリポジトリ、拡張機能（拡張機能）およびバックアップファイルに基づいてシステムを構築する機能。 このディストリビューションは初心者向けではなく、経験豊富なLinuxユーザー向けであると自信を持って言えます。 Tiny Core 8.0は、新しいカーネル、busybox、glibcおよびgccコンポーネントの更新バージョン、およびその他の多くの改良に基づいた最新バージョンです。 新しいカーネルにより、ディストリビューションは最新のハードウェアで正常に動作できます。



Bo提



Bodhiは、Ubuntu Linuxの軽量な修正版で、Mokshaデスクトップ環境を備えた柔軟にカスタマイズ可能なEnlightenmentウィンドウマネージャー（生産性、アニメーションサポート、Unicode）を備えています（動作しないEnlightenment 17機能が改善されました）。 ディストリビューションは、古い低電力ハードウェアでうまく機能します。 直感的でシンプルで便利。







デフォルトでは、5つのアプリケーションのみがBodhiにインストールされます。軽量Midoriブラウザ、Terminalologyターミナル、PCManFMファイルマネージャ、グラフィックファイルを表示するためのePhoto、および軽量ePad pdfビューアです。 他のプログラムをインストールするには、WebツールAppCenterがあります。 最小要件：300 MHzプロセッサ、128 MBのRAM、2.5 GBの空きディスククォータ。
