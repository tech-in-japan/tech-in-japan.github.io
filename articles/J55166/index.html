Subversion 1.6およびTortoiseSVN 1.6がリリースされました





新機能 ：
  • 認証データ管理の改善（KWalletまたはGnomeキーリングでの認証データの保存、SSL証明書へのキーの保存、パスワードを暗号化されていない形式で保存する前の確認要求）
  • 新しいURL形式-ルートの相対URLサポート
  • SVN：エクスターナルエクステンション：SVNのファイルサポート：エクスターナル
  • ツリーの競合の定義-ディレクトリ構造レベルでの競合
  • データの保存方法の改善-リポジトリが占有するディスク容量が少なくなります
  • 新しいPythonバインディング
  • コマンドラインクライアントでのインタラクティブな競合解決の改善
  • updateコマンドの--set-depthパラメーターは、特定のブランチを作業コピーから除外できるようにする新しい除外値をサポートします
  • svnserveのロギングサポート
  • 新しいHTTP URI構文を使用すると、特定のリビジョンにアクセスできます。 
      http：// host / repos / path？[p = PEG] [＆r = REV]
  • コマンドラインクライアントの機能強化：logコマンドは、1回の呼び出しで複数の-rおよび-cオプションをサポートするようになりました。 新しいパラメーター--trust-server-cert
  • APIの変更
  • 修正されたバグ
