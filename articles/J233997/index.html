新しいSamsung Galaxy Note 4に期待すること

こんにちは、Habr！



9月のIFA 2014で、サムスンは、このS-Penラインの伝統であり、高度な充填を行う5.75インチクアッドHDディスプレイを備えた新しいスマートフォンNote IIIの後継モデルを紹介します。 この投稿では、確認されたデータとうわさ、および2011年以降のラインの歴史について詳しく読んでください。







歴史ツアー



この会社については、ブランドヒストリーのフレームワークで既に書いています。 サムスン電子の歴史：干物からハイテクまで 。 そして今、私はGalaxy Noteスマートフォンが時間とともにどのように変化したかを見るよう提案します。



Galaxy Note


2011年には、Samsung Galaxy Noteが登場しました。 5.29インチディスプレイは、その主な特徴的な機能です。 ディスプレイの解像度は800x1280であり、このような画面には十分ではありません。 スマートフォンは、デュアルコアSamsung Exynos 4210、1400 MHzプロセッサで実行されます。



ボーナス-スタイラス。







Galaxy Note II


翌年の2012年10月に、後継機Galaxy Note IIが発売されました。 ディスプレイは5.55インチに成長しました。 SuperAMOLEDディスプレイの解像度は1280×720ピクセルです。 コアも追加されました-Note IIは、クアッドコアExynos 4412、1600 MHzに基づいて構築されています。







Galaxy Note III


次のモデル、ノートIII、ディスプレイは再び成長しました-最大5.7インチ。 解像度は1920×1080で、プロセッサーは3Gを搭載したバージョンの8コアExynos 5 Octa 5420、またはLTEバージョンのSnapdragon 800クアッドコア2.3ghzのいずれかです。 ここでも、以前のモデルと同様に、S-ペンスタイラスがサポートされています。

設計



ボディはマグネシウムとアルミニウムの合金でできているという噂があります。 真正性に疑いがあるこの写真から判断すると、裏表紙はテクスチャードレザーで覆われ、ケースの前部はガラスとプラスチックで作られ、側面は新しいデザインになります。 フロントパーツのデザインは、ノートIIIとは若干異なります。







Upleaksによると、スマートフォンは、白、ピンク、金、黒、青の5つのカラーオプションで表示されます。







CPU



3番目のノートと同様に、新しいファブレットには2つのプロセッサオプションがあります。 ちなみに、彼らの結果は既にAntutu Benchmarkで利用可能です 。



最初のオプションはExynos 5433プロセッサで、4つの64ビットCortex-A57コアと4つのCortex-A53です。 2番目のオプションは、Qualcomm Snapdragon 805、つまり4つの32ビットKrait 450コアとAdreno 420グラフィックスです。







ディスプレイ



新しい注4のディスプレイ解像度は、5.75インチの対角線と一致します。これらは2560x1440ピクセルのQuad HDです。 既に同様の解像度を持っている携帯電話のうち、LG G3とLenovo Vibe Z2 Proのみがあります。



Quad HDディスプレイを搭載したモデルがいくつかありますが、ロシアでは一般的ではありません。これらはBBKが所有するVivoとOppoです。



曲面ディスプレイ？



それとも、ディスプレイはスマートフォンの両端で湾曲したこのディスプレイに似ていますか？







同社は昨年、この開発をCES 2013（フレキシブルディスプレイ）で紹介しました。







カメラ



16メガピクセルのメインカメラを使用すると、3840 x 2160の解像度でビデオを撮影できます。注IIIでは13メガピクセルで、ラインの最初のスマートフォンでは8メガピクセルのみでした。 このような変化は3年以上にわたって発生しています。 メインカメラにはSony IMX240センサーが装備されています。 同じセンサーが、韓国市場向けのGalaxy S5のLTEバージョンで使用されました。



フロントカメラはどうですか？過去数年間、同社は2.1メガピクセルのカメラをよく使用していましたが、今回はこの伝統を変えて3.7メガピクセルをインストールしました。これにより、2560x1440の解像度で自分撮りを行うことができます



おそらく、Note 4はSide Touchテクノロジーを使用していますが、今年の初めにはGalaxy S5に追加されませんでした。 テクノロジーはこれです。ケースの右端の下部にある写真用の物理的なボタンの代わりに、この部分をタップするだけです。







網膜スキャナー



Galaxy S5のリリース前には、Retinaスキャナーモジュールの使用可能性に関する噂がありました。 そして先日、会社のExynos部門がこの画像を公開しました。その上に、おそらくノート4であるかなり大きなスマートフォンがあります。したがって、ノート4ではアイリススキャナーの形の保護が使用されるでしょう。

カバー


この記事に含まれたのは、発表されたアクセサリが1つだけだったためです。超音波を使用して障害物を特定するのに役立つカバーがあるためです。 この技術が視覚障害者の生活を改善するために使えることを願っています。 これが成功したら、 この記事の続きでアプリケーションとアプリケーションの方法について間違いなく書きます。



この出版物によると、カバーは、目の前にある近くの物体を検出し、フィードバックを提供する超音波センサーで構成されています。 ユーザーは、超音波信号の範囲を変更し、ショート、ミディアム、ロングの3つのレベルのいずれかを選択することもできます。 超音波信号の範囲が拡大する場合のみ、それらによってカバーされるエリアが狭くなります。



おそらく、これは会社の歴史の中で最も有用なアクセサリーです。 再び、彼らは彼のために適切な使用を見つけた場合。
