パート2について知っておくべきVIMプラグイン：repeat.vim

Peteris Kruminsによる記事の翻訳を続けています。 オリジナル記事



ここに最初の部分とそのオリジナルがあります。



Repeat.vimは、先週書いたSurround.vimプラグインの重大な欠点を修正します（私は怠け者で、恥ずかしいです-およそPer。）。 サラウンド.vimの問題は「。」にあります。 （期間）。 環境を使用し、「。」コマンドでそれを繰り返したい場合、成功しなかったでしょう。 このプラグインは問題を修正します。



基本的に、surround.vimをインストールする場所には、repeat.vimも必要です。



しかし、ニュアンスがあります。 プラグインは「ys」コマンドの繰り返しを許可しません。 幸い、ysコマンドはcsコマンドに置き換えることができます。 たとえば、yswを複数回実行する場合（単語を引用符で囲む）、cswと入力して使用します。 繰り返します。 ビジュアルモードを使用してコマンドを繰り返す唯一の方法は、マクロを記録することです。



repeat.vimを使用した例を次に示します。 文を入力し、すべての単語を引用したいと想像してください： 



 |foo bar baz quux muux woox (|   )




タイプcsw " ： 



 |"foo" bar baz quux muux woox




次にWを押します。 



 "foo" |"bar" baz quux muux woox




Wはカーソルを次の単語に移動し、ドットは環境コマンドを繰り返しました。



これを4回繰り返すと、文字列全体が引用符で囲まれます。 



 "foo" "bar" "baz" "quux" "muux" "woox"




repeat.vimのインストール方法は？


1. repeat.vimを〜/ .vim / plugin（Unix / Linux）ディレクトリ、または〜\ vimfiles \ plugin（Windows）にダウンロードします。

2. Vimを再起動するか、Unixでは「：so〜/ .vim / plugin / repeat.vim」コマンド、またはWindowsでは「：so〜/ vimfiles / plugin / repeat.vim」コマンドでrepeat.vimを接続します。



お楽しみください！


サラウンド.vimとrepeat.vimを継承します。 次回は、はるかに重要なプラグインについて書きます。 :)
