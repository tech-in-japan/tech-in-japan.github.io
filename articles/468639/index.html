cpanel



The first most popular multifunctional web console in the World, an industry standard. It is used by both site owners (as a control panel) and hosting providers (as a tool for administering Web Host Manager, WHM). Intuitive interface, does not require training, multilingual. There are video instructions.



Base Java: Perl, PHP

Supported OS: Red Hat Enterprise Linux (RHEL), entOS, CloudLinux. It is possible to support Windows through virtualization or through the Enkompass panel from the same developers.

Whm



New



The developers are constantly trying to speed up the panel and in general are actively working to improve it, based on customer requests. So, in the current 82nd version, Installation is completed in 3 minutes. CPanel & WHM update time has been improved: from the penultimate, 80th, it ends in three minutes, and from the earlier in eight. In 2019, disk space requirements for the cPanel & WHM installer were reduced by 10%. There are: compatibility with PCI; automatic backup and restore; a tool that allows you to blacklist and whitelist accounts, IP addresses and entire countries; Free SSL certificate for every website. Now you can include one file in another (add configuration). In general, during this time, the work of cPanel & WHM was accelerated by 90%, the necessary server resources were reduced by 30%.



In April 2019, the developers gently announced , perhaps the most popular request for updating cPanel features - adding the NGINX web server as an alternative to Apache. The function works in an experimental format. Official update documentation.



Prices



Depends on the account level: Solo $ 15, Admin $ 20, Professional $ 30, Premier $ 45 per month. Free trial period. There are affiliate programs. Priority tech support $ 65 incident.



Plesk



A favorite control panel by large hosting providers, which will be understandable even to a novice. A single convenient interface with which you can centrally manage all system services. It is delivered in different publications for a specific hosting and usage scenario.



‚Üí About Plesk on the official website



Base Java: PHP, C, C ++

Supported OS: different versions of Linux, Windows









Plesk



New



New panel features come in the form of extensions compiled into a directory on the site. The interface has been significantly improved: adaptive design, the ability to automatically log clients into Plesk from external resources without re-authentication (for example, from the panel of their hosting provider), the ability to share direct links to screens; Redesigned user interface for task scheduler. Improved database management; there is support for several versions of PHP, as well as Ruby, Python and NodeJS; full support for Git; Docker integration SEO toolkit. The Plesk recovery tool, a command-line utility that can be used to automatically detect and fix a wide range of problems, has appeared. Each Plesk instance is now automatically secured using SSL / TLS. You can reduce the response time of sites and the load on the server using Nginx Caching. A feature called Smart Updates has been added to the popular WordPress Toolkit extension, which analyzes WordPress updates with artificial intelligence to determine if installing the update can break something.



Prices



RUVDS also provides a Plesk panel for its customers, the price of 1 license is 650 rubles per month.



Directadmin



Developers position the panel as the easiest to manage in the world. They try to keep up with the times and use advanced technologies, while there is nothing supernatural in the panel - only basic functions . There are no predefined scripts, but you can create your own (open API). Multilingual interface, but without Russian support (you can use unofficial skins). Weak antispam filter. But - undemanding to server resources and high speed. Multilevel access.



Base PL: PHP

Supported OS: FreeBSD, GNU / Linux (Fedora, CentOS, Debian, Red Hat distributions)
