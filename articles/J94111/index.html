Nokia S60の修正されたファームウェア

このトピックがハブにないことを発見できてうれしいです。ハブラの人々を新しいもので驚かせようと思います。



最近、私は5530（つまり、5800とN97について説明します）のパフォーマンスを小さな手順（スマートの再フラッシュ）で最適化できることを知りました。 しかし、単に点滅するだけでなく、不要なファイルと設定を削除します。

興味深いのは、スマートがはるかに速く、より便利になり、バッテリーの電力を節約できることです。 多くのプログラムと方法は同じ効果を約束しますが、テストされた私の誰もが修正されたファームウェアのような結果を与えません。 カットの下-ファームウェアの1つの概要、説明、スクリーンショット、そしてもちろんリンク。





あなたがすでに推測したように、私はすぐに注意したいです-ファームウェアは「善良な人々」によって作られ、ファームウェアの作者は手の安心のために殺されたスマートに対して責任を負いません。

私の意見では、xorRisont S60バージョン3.0からの変更されたファームウェアを検討します。 これは、他のすべてのファームウェアの一種の組み合わせです。

ファームウェアの説明



  1. ここで5530のファームウェアをダウンロードし、5800のファームウェアをダウンロードし、N97のファームウェアをコンピューターにインストールします。
  2. ここで5530 、 ここで5800 、 ここでN97を選択してダウンロードする必要がある変更されたファイルは、ファームウェアがインストールされているフォルダー（デフォルトではC：\ Program Files \ Nokia \ Phoenix \ Products \ RM-504）に移動され、交換に同意しますファイル
  3. ここで 、JAFファームウェア用のプログラムをダウンロードします（手順とレビューもあります）。
  4. 電話をPC Suiteモードで接続します
  5. インストール後、 BOX DRIVER NOT INSTALLEDメッセージを無視して、デスクトップからJAFプログラムを実行します。これは気にしません。
  6. [BB5]タブに移動し、[Use INI]チェックボックスをクリックします。その後、ウィンドウが表示され、電話機のモデルを選択します。 OK、CRT 308のチェックを外して、装着します：マニュアルフラッシュ、デッドUSB、ファクトリーセット。
  7. 切断せずに電話の電源を切ります。 Flashプログラムを押し、エラーに同意し、電話機の電源ボタンを短く押すことでエラーをオンにします。 次に、プログラムがすべてを行います。 数分後、新しい電話が届きます。


フラッシュドライブ、連絡先、および必要なプログラムのバックアップを忘れないでください。 ご清聴ありがとうございました。
