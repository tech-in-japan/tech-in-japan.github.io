トップレベルのプロバイダーCogent Block Pirate Bayおよびその他の海賊サイト





Cogentは、世界最大のインターネットサービスプロバイダーです。 この会社は1999年に設立され、インターネットアクセス、コロケーション、インターネットトラフィックのトランスポート配信などのサービスを提供しています。 一般に、世界のトラフィックの約20％がこの会社のネットワークを通過します。 最近、世界中のさまざまなインターネットプロバイダーの多くのユーザーが、The Pirate Bayなどの有名な「海賊版」サイトにアクセスできないことが判明しました。 サイトへのリクエストは、Cogentバックボーンネットワークを超えていませんでした。



バックボーンインターネットサービスプロバイダー。その1つは、グローバルインターネットインフラストラクチャの重要な部分であるCogentです。 これらは、ポイントAからポイントBへのグローバルトラフィックの通過を提供します。したがって、バックボーンプロバイダーは、多くのサイトへの要求を許可しないことを決定する場合があります。 この場合、「ネイティブ」プロバイダーが何もブロックしなくても、重要度の低いさまざまなインターネットプロバイダーのユーザーはこれらのサイトにアクセスできません。ロシア語を含むすべての既知の世界の仮想コンテンツ。



これはすべて、Primewire、Movie4k、TorrentProject、TorrentButlerなどの「海賊版」サイトに完全に適用されます。 Cogentは、1週間以上 、ネットワーク上のこれらのリソースに対するユーザーリクエストをブロックしています。 上記のサイトは、CloudFlare IPアドレスで動作します。 IP-104.31.18.30および104.31.19.30。 世界中のほとんどのユーザーはこれらのアドレスに問題はありません-それらへのアクセスはオープンです。 ただし、これは、これらのユーザーのトラフィックがCogentネットワークを通過しない場合のみです。 この場合、指定されたアドレスへのアクセスは閉じられます。



この問題は、米国およびヨーロッパのユーザーによってすでに認識されています。 大規模とは言えませんが、ブロックするという事実から多くの疑問が生じます。 アクセスブロッキングについては、Cogentが実際に行っています。 会社には交通ルートのチェックがあり、ここの写真は明確です。







もちろん、エラーが発生することがあるため、TorrentFreakリソースのジャーナリストはプロバイダーに追加情報を要求しました。 特に、質問は尋ねられました：「ロックがあることは本当ですか？もしそうなら、その理由は何ですか？」 会社の広報担当者は、この問題は研究されると述べたが、一般的にそのような問題と解決策は外部の企業やジャーナリストと議論されていない。 確かに、Cogentの代表者は、彼の会社は上記のリソースのDNSレコードを管理していないと付け加えました。



それでも、これらのサイトはCogentネットワークからアクセスできないままなので、会社自体がリソースの要求をブロックするかどうかという質問に対する答えは簡単です-はい。 バックボーンプロバイダーがこのロックを必要とした理由を理解することはさらに困難です。 ここでは推測することしかできません。 おそらくその理由は、「上位」レベルで違反しているサイトへのアクセスをブロックする多くの著作権保持者の要件を伴う、著作権侵害です。 The Pirate Bayの代表者によると、彼らはすでに問題について知っており、リソースにアクセスできないユーザーにCogentに苦情を申し立てて、問題の解決に成功することを望んでいます。



同時に、リソースの管理に頼ることなくこれらすべてを解決できます。 すべて同じVPNとTorが助けになります。 プロキシの使用もうまくいきます。



アクセスがブロックされているサイト自体がIPを他のIPに変更すると、DNSレコードを更新した後も問題はなくなります。 ただし、著作権所有者の要求でブロックが開始された場合、新しいIPはすぐにブラックリストに登録されます。 現時点では、The Pirate Bay、Primewire、Movie4k、Torrentproject、Couch-tuner、Cyro.se、Watchseriesfree、Megashare、Hdmovieswatch、Torrentbutler.eu、AfdahなどのブロックされたCogentリソースについて知られています。 Movie.to、Mp3monkey、Rnbxclusive.me、Torrentcd、Moviesub、Iptorrents、Putlocker.comおよびTorrentz.cd。







長年にわたり、著作権所有者は、このコンテンツまたはそのコンテンツを公開することにより著作権を侵害していると彼らの意見で判断しているサイトで苦労してきました。 著作権サポーターは、リソースが作成されてからほぼ海賊湾と戦ってきました。 トレントトラッカーの作成者は、かつて裁判を受け、罰金を科され、刑務所に送られました。 Pirate Bayサーバーが押収され、ドメインが分割されました。 それにもかかわらず、サイトは引き続き機能しますが、これらのすべてのアクションがトラフィックに影響を及ぼしていますが、トラフィックは減少しています。



比較的最近、KickassTorrentsは「海賊版」リソースのリーダーの黄色のジャージを受け取りました。 2016年の夏、その所有者はKickassTorrentsに関連する多くのドメインによって逮捕され 、隔離されました。 しかし、すでに12月にサイトが再び機能し始め、前のチームが復活を始めました。 「管理者やモデレーターを含む元のチームのほとんどが私たちに加わりました-これに非常に満足しています」 とプロジェクトの作者は言いました 。



（特定の国で働く）ローカルプロバイダーのサポートで個々のサイトをブロックできなかった権利所有者は、バックボーンプロバイダーに行くことにしたと想定できます。 これが当てはまる場合、おそらくCogentの後、他のトップレベルのネットワークオペレーターが海賊版リソースのブロックを開始する可能性があります。
