Windowsストアアプリの販売統計

Windows 8はまだ若いプラットフォームと呼ぶことができます。 人と開発者はそれを異なって評価します。 誰かが彼女の成功を信じ、誰かが彼女の失敗について話します。 最初の知り合いには、ユーザーに2つの質問があります。a）他のプラットフォームで使用したすべてのアプリケーションはどこにあるのかb）どうしてそんなに悪いアプリケーションがすべてあるのか。



これらの質問に答えるために、1年以上前にWindowsストアで公開したアプリケーションの統計を見てみましょう。



outcoldplayer （以前のgMusicW）は、暇なときに書くホームプロジェクトです。 これは、Google Musicの非公式クライアントです。 Windowsストアでかなり成功したプロジェクトと呼ぶことができます。 かつて彼は、ほとんどの購入済みアプリケーションのリストで1位だったからです（現在は米国の29か所）。







自分のアプリケーションがGoogle PlayまたはApp Storeの有料アプリケーションのトップにいると誰かに言われたら、彼は億万長者だと思います。 これはWindows 8の場合ですか？ 私の簡単な答えは：私はまだ億万長者ではありません。 過去1年間、私はこのアプリケーションで、メインの仕事で得た月給よりも少ない収入を得ました。



outcoldplayerに関する少しの歴史



アプリケーションの最初のバージョンを2013年1月から2月に公開しました。 昨日、40回目の更新を公開しました。 最初は、アプリケーションを無料で配布しましたが、広告付きでした。 ユーザーは特別な広告なしパッケージを使用して広告を削除する機会がありました







2ヶ月前、私はアプリケーションを有料のものとして配布し始めました。 いくつかの理由により：



  1. 広告を常に挿入する必要がある場合、デザインを維持することは困難です。
  2. Microsoft Ad Plaformライブラリの品質が気に入らなかった。 ライブラリはTaskScheduler.UnobservedTaskExceptionに未処理の例外を大量に残していたため、ライブラリの品質が心配になりました。
  3. 広告収入は非常に不安定でした。 1か月で50ドル、さらに500ドルで手に入ります。 広告の記入は非常に小さい32％です。




したがって、過去2か月で-アプリケーションは有料になり、5.99ドルかかります。



Windowsバージョン



私はまだWindows 8をサポートしていますが、アプリケーションのほとんどのユーザーがWindows 8.1に移行したようです







このリストには、公開時にアプリケーションをテストするマイクロソフトの従業員も含まれます。 そのため、Windows 8ユーザーの数はほぼ0％と言えます。 この数値は、Windows 8よりもWindows 8.1よりも高いパーセンテージのWeb統計とは異なります。



成績



平均して、アプリユーザーは満足しています







しかし、もちろん、例えば基本的な主張など、不幸なものもあります



  • 一部のユーザーは、アプリケーションに支払う必要がある理由を理解していません。 結局、彼らは既にWindows 8の支払いを済ませており、Google Musicサービスの支払いもしています。 そして、私はそれらを理解することができます。 誰もが「非公式」という言葉を知っているわけではありません。
  • 一部のユーザーはドライバーに問題があります。 はい、Windowsストアのアプリケーションもドライバーに依存しています。 Microsoft Windows 8 Appsフォーラム（Netflix、Hulu、Youtube、TuneIn）でのいくつかの議論はストリーミングされません 、 0xc00d11d1（0x8007007e）Windows 8.1アップデート 。

アプリケーション更新プロセス



これはWindows 8の大きな問題であり、ユーザーはWindowsストアからアプリケーションを更新する方法を知らず、知らなかっただけでした。 古いバグやGoogleミュージックの最新の変更との非互換性に関する悪いレビューや手紙を常に受け​​取りました。 現時点では、ご覧のとおり、このような問題はありません（最新バージョンは1週間で90％更新されました）







まとめ



これについて少しまとめたいと思います。 もちろん、統計は幸せではありません。 プラットフォームで見られる主な問題：ユーザーは、Windowsストアアプリケーションを使用したくない（その結果、おそらくWindowsタブレットを購入しない）ことがあります。 一方、開発者はWindows Storeプラットフォームでのアプリケーション開発に時間を費やすことを望みません。これは、支払いを希望するユーザーがあまりいないためです。 真剣に、私はGrand Theft Autoを上回っていて、最も有料のアプリのリストで1日あたり約35件の売り上げがありました。 WindowsストアでGTAの価格を知っているので、掛けてみてください。



これは、Windows 8を忘れて、iPadとAndroidタブレットの条件を満たすべきだということですか？ ユーザーとして、2つよりも3つの異なるプラットフォームを使用することは確かに良いことです（より多くの競争がありますように）。 だから私の答えはノーです。 Windows 8の次のアップデートがより成功することを願っています。 [スタート]メニューについては説明しませんが、Windowsストアアプリケーションはデスクトップ上でさまざまなサイズのウィンドウで実行できることを説明しています。 私には思えます-これは、Windowsストアアプリケーションに別の飛躍を与える必要があります。 時間はわかります。
