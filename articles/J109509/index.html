エカテリンブルクの3番目のスタートアップポイント

こんにちはHabrastartaper！



ウラルは、社会の多くの領域に浸透しているロック文化で知られています。 ITに含める。 地元のスタートアップが燃えています！ これは、12月17日にエカテリンブルクのグループChayf 3rd Pointのコンサートで見ることができます。 イベントに登録します。

11月、地方政府の会議で、ITクラスターが作成されました。その主な目的は、必要なインフラストラクチャを開発し、IT専門家にとって好ましい雰囲気を作り出すことです。 スタートアップには多くの注意が払われていますが、会議ではこれについて話します。

最初のイベントから6か月が経過しましたが、すでに成功談があります。 エカテリンブルクのAlexey SulimanovのDataLiteプロジェクトに投資が集まりました。 また、過去のポイントに参加した多くのスタートアップは、開発に必要な刺激を受けました。

プロジェクトのプレゼンテーションを行い、無料の試験を受けたい場合は、申請書を提出してください 。

イベントのプログラムで：

  • Speed Networking形式のクイックデート
  • エレベーターピッチ形式の2分間のスタートアッププレゼンテーション


ポイントに参加する理由




1. 無料の試験を受ける



すべてのスタートアップポイントイベントには、プレゼンテーションの直後に推奨事項を示す専門家が参加します。 プロジェクトの長所と短所を判断するのに役立ちます。 彼らは彼らの能力の枠内であなたに興味のある問題について助言することができます。



2. プロジェクトを幅広い聴衆に提示する



あなたは長い間プロジェクトに取り組んでおり、あなたが側からそれを見るのは難しいです。 イベントでは、会議に参加した人々からいつでもフィードバックを得ることができます。 彼らはあなたの「最初の聴衆」であり、それはあなたのスタートアップをユーザーにとって便利で便利なもので補完するのに役立ちます。 アドバイスに従ってください。



3. プロジェクトを投資家に提示する



投資家は、投資プロジェクトを見つけるためにイベントに参加します。 ミーティングを通じて、10のプロジェクトで270万ドル以上を集めることができました。 また、プレゼンテーションと話すすべてのスタートアップは、投資家への郵送の選択リストに分類されます。 ヤンデックス、ヘッドハンター、Muz-TVなど、100を超える民間の3つの戦略的なものによって受信されています。 20基金も同様です。



4. 新しい友達を作る



多くの興味深い人々が何らかの形であなたに役立つかもしれない会議に来ます。 良い開発には何が必要ですか？ そう、コミュニケーション。 Speed Networking形式を使用して、役に立つ人々と出会うお手伝いをします。



また、イベントで提供するサプライズもあります。 一緒に来て。
