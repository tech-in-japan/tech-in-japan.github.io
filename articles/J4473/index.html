SearchMash：Googleの新しい検索インターフェース

今日、Googleは注目を集める発表をせずに、検索サイトSearchMashをオープンしました。一部の専門家はすでにGoogle 2.0と呼んでいます 。 実際、このサイトを見ると、これは新世代のAjax検索インターフェースであると言えます。 明らかに、ここでグーグルはインターフェースの分野で最も革新的な開発をテストします。それは何らかの理由で公式ウェブサイトでテストできません。



SearchMashには多くの興味深い機能があります。



1. 検索クエリのインスタントセット 。 ユーザーがキーボードでテキストを入力し始めた瞬間に、テキストが検索バーに表示されます。 これで、検索バーに特にカーソルを置く必要はありません。



2. URLのメニュー 。 検索結果のいずれかのリンクをクリックすると、オプションのリストを含むコンテキストメニューが表示されます。このウィンドウで開きます。 新しいウィンドウで開きます。 保存されたコピーを開きます。 このサイトからのより多くの結果。 より関連するページ。 これらのオプションはコンテキストメニューに組み込まれているため、検索結果を視覚的にクリアすることが可能になりました。



3. 画像パネル 。 コンテキスト広告の代わりに、画面の右側にこのキーリクエストの画像検索結果が表示されます（3枚の写真）。



4. 検索結果をページにドラッグアンドドロップします。 検索結果の各フラグメントは、マウスでその番号を「つかんで」別の場所（ページの上下にドラッグ）でドラッグできます。 これにはまだ意味がなく、ただのクールな機能です。 将来的には、このようなドラッグアンドドロップに役立つ機能がいくつか登場するはずです。



5. 「その他のWebページ」リンクは、ページをリロードせずに、より多くの検索結果をページにロードします。



上記の各機能について、サイトで投票が行われ、それらがどれほど有用かが確認されます。



興味深いことに、Googleは3つのメインドメインゾーン（mash-search.com/net/org、mash-serch.com/net/org、mashsearch.com）のすべての子音フレーズを含む60を超えるドメイン名を新しい検索サイトに登録しました。 / net / org、mashserch.com / net / org、mashup-search.com / net / org、mashups-search.com / net / orgなど どうやら、プロジェクトを開始する1〜2か月前に多数のドメインを一括登録することは、インターネットビジネスの標準的な手法になります。
