Zimbra Desktop 2 beta1-すでに！ （ミニレビュー）





主要な電子メールアプリケーションの巨人であるZimbraは、Zimbra Desktop 2電子メールクライアントの第2世代の最初のベータ版をリリースしました 。



Zimbra Desktopは、Mozilla Prismテクノロジーに基づくクロスプラットフォームの電子メールクライアントです。 メールに加えて、オーガナイザー、タスクマネージャー、ポートフォリオ、カレンダーなどの機能が含まれています。 実際、これはMicrosoft Outlookに代わる最良の選択肢です。



Zimbra Desktopは、ネイティブのZimbra Collaboration Suiteサーバー（Exchangeの無料の代替）と、Gmail、Yahooメール、AOL、Microsoft Exenge IMAP、およびその他のPOP \ IMAPサーバーの両方に統合できます。 また、これは「オフライン」操作モードをサポートする最初の顧客の1つです。





zimletフレームワークについて忘れてはなりません。 Zimletは、作業を大幅に簡素化するクライアント用のミニアプリケーションです。 たとえば、zimletトランスレータまたはGoogleマップをインストールすると、住所、文字の電話番号を検索し、それらからマップエリアへのリンクを作成します。 最近、 ソーシャルZimletについて話しました。これは、ユーザーの作業環境をFacebook、Twiterなどのソーシャルネットワークと統合します。



Zimbra Desktop 1と比較した変更点（メジャー）：



•マルチアカウントUI

•仮想メールボックス

•クロスアカウントの共有アドレス帳

•Zimbra Collaboration Suite 6.0との互換性が改善されました。

•改善されたインターフェイス。

•インストールは、Winodws（プログラムファイル）、Mac（/アプリケーション）で実行されます。

•インストール方法が再設計され、WindowsではMSIパッケージ、Macではdmg、LinuxではTGZになりました。

•ZDはマルチユーザーになりました。 システム内のすべてのユーザーに配置できます。







性能：

最初のバージョン（1.0。*）と比較して、Zimbra Desktop 2はより高速になりました。 2番目のバージョンはSQLiteに切り替わりました。 RAM消費量がわずかに削減されます。







個人的な印象：

Zimbra Desktop 1と比較して特別な外部変更はありません。 原理は同じです。 彼は少し少ない記憶を食べます。
