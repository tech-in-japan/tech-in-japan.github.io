SkyDriveの更新とSkyDriveをネットワークリソースとして接続する方法

最近、 Microsoft SkyDriveの無料のクラウドファイルストレージウェブインターフェースが更新されました。 このリソースは引き続き25ギガバイトの無料ストレージを提供しますが、アップグレード中に、単一のダウンロードファイルのサイズが50メガバイトから100メガバイトに増加しました。



Webインターフェイスは、新しいビューを使用してフォルダーを表示し、画像ギャラリーを表示します。







さらに、次の変更が発表されます。
  • Internet Explorer 9を介してサイトをタスクバーにピン留めする
  • すべてのファイルタイプに同じインターフェース
  • 新しいファイル情報パネル
  • 新しい写真ビューア
  • リソース速度が増加しました
  • 100 MBのファイル制限
  • 広告はもうありません。バナーは削除されました。
しかし、最も興味深いのは、SkyDriveストレージをネットワークドライブとして接続して、ファイルのコピー、貼り付け、削除、プレビューを完全にサポートする方法があることです。 これまでのところ、ストアへの書き込みではオフィスファイルがサポートされています。 残りのファイルは、Webインターフェースからダウンロードできます。 Webインターフェースを介してダウンロードされたファイルへのアクセスは、すべてのタイプで利用可能です。 SkyDriveをネットワークドライブとして接続するには、次の手順を実行します。
  1. http://skydrive.live.com/にアクセスします。
  2. Windows Live IDでログインします。
  3. URL https://skydrive.live.com/?wa=wsignin1.0#cid=1234567890abcdefからcid IDをコピーします
  4. Windowsエクスプローラーを開きます。
  5. 左ペインで、[ ネットワーク ]を右クリックします 。 [ ネットワークドライブの割り当て]を選択します 。
  6. URLを入力します： \\ docs.live.net@SSL \ 1234567890abcdef （1234567890abcdefを識別子に置き換えます）
  7. ドライブ文字を選択し、[OK]をクリックします。
  8. Windows Live IDアカウントの入力を求められます。 それらを入力してください。
  9. これで、ネットワークドライブが接続されました！
