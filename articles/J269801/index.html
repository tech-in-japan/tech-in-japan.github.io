ベータ版が近づいています-Vivaldi 1.0.303.40

みなさんこんにちは！



新しい修正が加えられた新しいビルド-すべてが完全なベータ版に移行中です。



タブを操作する



主な修正-タブが多数あり、それらがすべて左または右にある場合、タブのリストを再度スクロールできるようになりました。 また、ボタンを調整してタブを閉じました-スクロールバーと重ならないようになりました。



タブを上下に配置することを好む人には、ニュースもあります。 閉じるボタンは非アクティブなタブで非表示になり、アクティブなタブはサイトアイコンの上に配置されます。 一般的に、機能の利便性を維持しながら、タブを意図せずに閉じないようにしました:-)



Windowsバージョンでインポート



多くのユーザーがOpera 12からデータをインポートすることは周知の事実です。ポータブルモード（USBモード）でインストールされたOpera 12からデータをインポートする機能が追加されました。 Chromiumコアに基づくOperaの任意のバージョンからデータをインポートすることもできます。



今のところすべてです。 以下のリンクからアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット


変更の完全なリスト：



  • VB-10159タブが小さい、小さい、小さいときに閉じるボタンが非表示にならない
  • VB-10224左側の多くのタブの処理が不十分
  • VB-10261タブを簡単に閉じたり、タブの左側を閉じたりすることはできません
  • VB-10296インストーラーの背景画像を更新
  • VB-10273 Opera 12 Windowsスタンドアロン（USBモード）からのインポートが失敗する


毎週のビルドはテストであるため、重要なデータをバックアップすることを忘れないでください。 前のアドレスで見つかったエラーを報告します 。



更新する



そして今日、アップデートが到着しました。 Vivaldi 1.0.303.48の最新ビルドは、開発者のブログからダウンロードできます。
