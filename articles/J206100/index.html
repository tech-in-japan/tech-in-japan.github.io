33 kmの3Gインターネット

私の友人、先祖代々の地所のアイデアの大ファンは、約1年前に村や都市から離れた土地の区画に落ち着き始めました。 彼は木造住宅、浴場を建設し、補助農場を組織しました。 彼には、美しい妻、犬のモデル「ハスコアラバイ」、および猫クシーがいました。 そして、すべてが良いようで、あなたは生きて発展することができますが、インターネットに接続できないのは素晴らしいことです.70 kmごとに市内を訪れ、建設、太陽エネルギー、ガーデニング、オンラインでの友人とのチャットに関するフォーラムを読むことは、不快で費用がかかります。



この問題で、彼は私に向き直りました。 導入条件を指定して、問題を解決し始めました。

  • 3G携帯電話事業者のうち、音声通信はBeelineでのみ機能します。つまり、ウィンドウにアクセスした場合です。 TELE2はまだ捕捉されていますが、3Gはありません。
  • 近隣のUSBモデムを介したインターネットは、平均ページを10分間ロードします。
  • 最寄りの集落は7 km、最寄りの「大規模」（2万8千人）は30 kmです。






機器の選択



3Gは、LTEがまだ到達していないため、経済的に手頃なインターネットアクセステクノロジーから選択されました。CDMAEV-DOは、利用可能な基地局の数を大幅に削減する1つのオペレーターのみを提供します。

中国とロシアのオンラインストアの範囲を確認した後、2つのセットを選択しました。



安い：

  1. 中国のアンテナ -〜700r

  2. アンテナ入力付きHuawei E352モデム-900摩擦


中国人は25dBiのゲインを言うが、この数字は懐疑的だ。 たとえば、同じAntekでは、 同様のアンテナのパスポートdBiは17のみです。 安価な5メートルのケーブルは、少なくとも3 dBiかかります。 その結果、1600rで最大14dBiです。



最大KUの場合：

  1. 衛星信号Supralのオフセットリフレクター、Ø= 0.9m-1280r

  2. Irradiator 3G AX-2000 OFFSET （少なくとも23dBiの0.9mリフレクターを備えたパスポートコントロールユニット）-940r
  3. ケーブルN-Male-> SMA-Male 5D-FB 5m（-1.7dBi）-625r
  4. モデム用アダプターケーブルSMA-Male-> CRC-9-300r
  5. アンテナ入力付きHuawei E352モデム-900摩擦


キットの合計KUは約21-22dBiで、総コストは4045rです。

おそらく、誰かが14dBiと22dBiの違いはたった8ユニットだと言うでしょう。 ただし、dBiは対数単位であり、+ 8dBiは信号電力の6.3倍の差があることを思い出させてください。



そのため、より高価で信頼性の高いオプションを選択することにしました。



準備する



注文したケーブル、照射器、モデムを受け取り、地元の店でリフレクターを購入した後、私たちはその場所に到着し、設置を開始しました。 アンテナを組み立てても問題は発生しませんでした。主なことは、2つのニュアンスを忘れないことです。

  • 照射器は偏光の方向に合わせて取り付けられます（ハウジングのマークは垂直でなければなりません）。
  • 地面に平行に伝播する信号を受信するには、反射面を垂直に対してオフセット角度まで傾ける必要があります。 私たちの場合、角度は〜20°でした。




信号強度を監視するためのアプリケーションとして、 MDMA （Mobile Data Monitoring Application）を選択しました。これは、テストされたものの中で最も有益なものです。 もちろん、モデムは以前w3bsit3-dns.comを使用した方法に従ってオンラインアプリケーションでロック解除されていました 。



私の想像では、アンテナをBSに合わせるのは簡単で楽しいプロセスでした-特別なアプリケーションで信号レベルを見て、パートナーがアンテナを熱心に円状にねじっている間に最大値に注意してください。 その後、アンテナを微調整して壁に取り付けます-そして出来上がり！ 人間の知識のリポジトリ（および猫の写真）のウィンドウが開かれます。 3Gカバレッジマップを注意深く検討しませんでした。 そして、セットアップ-唾を吐くだけ！



熊手歩行



もし私たちの不幸な出来事の説明があなたを退屈させたら、私は「結果」に直接行くことを提案します。







まず、粗いチューニングを試してみることにしました-私は家のラップトップの後ろにいて、外にいる友人が階段の上にいて、アンテナを手に持ってスムーズに回転します。



しかし、ここに奇跡があります-アンテナなしでは、モデムは-97dBmの信号を示しましたが、アンテナではせいぜい-111dBmのほぼ一定のバックグラウンドで-101dBmを見つけることができました。 さらに、レベルまたはSNR（信号/ノイズ）の値が変化を停止するか、-125 dBmに低下したときに、モデムは時々「フェード」します（モデムは小さい数字を表示できません）。 モデムを元の状態に戻すには、再起動する必要がありました。 また、同じ位置では、最初のパスの信号レベルが1であることが多く、次のパスでは信号レベルが完全に異なっていました。 これはすべて作業を大幅に遅くしました。 時々、利用可能なネットワークのリストを確認しました。ほとんどの場合、TELE2のみが表示され、場合によってはBeelineとMegafonも表示されました。



私たちは数時間かけてBSへの道案内を試み、家の片側でリフレクターを回転させ、次に反対側でリフレクターを回転させました。

無駄に。





犬バルは、ゲストが良心的になり、最終的に彼と遊ぶという希望を完全に失いました。



すべてのケーブル接続とコネクタを再確認し、プレートの形状を測定し、照射器を取り外して動かして、「ノックアウト」フォーカスを見つけようとしました。 無駄に。 それから、アンテナをスティックに置いて家からの信号の方向を決定し、正確に回転させやすくすることにしました。 最初の星が空に現れましたが、BSにチューニングするための有用な情報を提供しなかった、弱い、前後にジャンプする信号しかありませんでした。

次のステップは、家の壁の完全なブラケットにアンテナを設置することでした-突然、静止した取り付けなしでは、十分な回転精度がありませんか？ すぐに言ってやった。 しかし、これは驚きです-チューニング結果は他のすべての試みと変わりませんでした。 必死になって、マウントを反対側の壁に移動し、半円に沿ってビーム通過を繰り返し、同じ成功を収めました。

悲しい気分で、真夜中過ぎに寝ました。



朝は新しいアイデアをもたらしました-仮設マストで家の屋根の上にプレートを上げることにしました。 そのため、まず信号を改善し、次に微調整を簡素化します。





総マスト長は8mです。



アンテナを取り付けた後、ラップトップに戻りました。 もう一度、私はMDMAのネイティブインターフェイスを見て回りました... 結局、接続の種類を選択しました： "3G Prefferred"、つまり、排他的ではなく優先3Gモードです！







同じことがすべてを説明します-モデムがアンテナ（TELE2から）と他の奇妙なものなしで信号の3g / 2gおよび-97dBmモードを切り替えたときの定期的な「フェージング」。



「3G-Only」モードをオンにすると、設定がはるかに速くなりました-モデムの「ハングアップ」の頻度が低くなり、バックグラウンド信号レベルがゼロに低下したため、リフレクターを3G BSに向けるとすぐにジャンプしました。



ただし、屋根の上にアンテナがある場合でも、受信した3G信号の最大値は-109dBmであり、これはわずか12kmの村の方向です！ このような大きなリフレクターでは弱い結果になります。



保証期間中に検証のためにキットを返そうとするAnteksと通信するという悲しい考えをすでに持っていたので、どれくらいの時間と神経がかかるかを考えました。 そして私は、インターネットに接続するために、信号が不快であることを確認することにしました。 驚くべきことに、接続はすぐに行きました。 しかし、pingは実行されませんでした。







そのような信号で、私は思った。 しかし、チャネルが非常に不安定な場合、ya.ruという名前はどのように解決できますか？ 他の名前を確認します-それらも解決されます。 ブラウザーを開き、すぐにBeelineスタブページに移動して、残高が不足していることを報告します。 別のSIMカードを接続しています-実際、インターネットがあります！ Speedtest.netは、インバウンド2.19mbpsおよびアウトバウンド0.12mbpsの速度を示しました。 さて、ジュエリーアンテナの位置調整が必要になりました。





屋根の上でさらに30分-信号はSNR = -94で-97dBmに上昇しました！



結果



アンテナなしの3G信号レベルと比較して：





モデムは床から1.5mです。



マストに配置されたアンテナを使用して、信号レベルを28dBm、つまり630倍に増やすことができました！





高さ7mのアンテナ。



〜5Mbit / sダウンリンクと〜1Mbit / sアップリンクの速度を提供することが可能でした。







しかし、面白いことに、ビーラインの3Gカバレッジマップを見て、技術サポートに電話することで、当初考えたように、12ではなく33 km離れた村のBSに実際に接続していることがわかりました。 その方向に近いビーラインには、単に3Gセクターがありません。 そのような距離では、受信信号のゲインは良好です。 確かに、距離があるため、「BS呼吸」の不快な効果が現れました。 セクターの負荷が増加すると、BSは範囲を縮小します。そのため、夕方にはインターネットがしばらく「落ちる」ことがあります。



しかし、いずれにせよ、タスクは完了しました-私の友人は非常に高速なインターネットを得ました。 そして、私は、明確な良心をもって、最終的にバルと遊び、家に帰ることができました。







UPD：ところで、約1週間後、友人がアンテナを調整し、プレートをもう少し傾けました。 -95dBmの信号を受信し、夜間に急流でインターネットは驚くべきダウンストリームである8-10mbit / sに加速します。
