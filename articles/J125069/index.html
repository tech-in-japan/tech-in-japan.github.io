シンプルなツイストペアケーブルテスターの設計

現時点では、私は参加者であり、人々が生まれて以来94 ウクライナでは、小規模であることが判明しました。多くの大学の学部は、10〜11年生の学生と小さな「科学的」作業を行いました。 そのような仕事に参加できて幸運でした。 私は事実を知りました。 TKVT KNURE、つまり上級教師のマリン・A.P.

この記事で説明する簡単なケーブルテスターの作成プロセスを作成するように依頼されました。



私のタスクは、UTPカテゴリ5ケーブルのテストを可能にするシンプルなデバイスを作成することでした（ツイストペアケーブルの任意のケーブルに実装できます） これらのタイプのエラーを診断できます：



-崖;

-短絡;

-クーデタークーデター;

-交差ペア;

このようなエラーでは物理的な接続が残り、干渉への抵抗が変化するため、ペア分割診断を上級の学生に任せました（トーンジェネレーターが必要なようですが、実際には入りませんでした-私にとっては難しすぎます）。



デバイス自体は2つの部分で構成されています（パッシブおよびアクティブと呼びましょう）。

デバイスの論理図：



したがって、テスターのアクティブな部分のタスクは、パルスを物理的および時間的に分配することです。

カウンタの出力（A1、A1、A3、A4）でのトビッシュは次のようになります。



A1 1 0 0 0

A1 0 1 0 0

A3 0 0 1 0

A4 0 0 0 1



各パルスはそれぞれのペアで「実行」され、断線がなく、ケーブルが「受動」部分で正しく圧着されている場合、緑色のLEDが順番に点灯します。



圧着またはケーブルの破損にエラーがあるとどうなりますか？



断線、短絡 ：回路が閉じられておらず、電流が流れないため、すべてまたはいくつかのLEDは点灯しません。



結合 ：電流は他の方向（x1> x2、x2> x1ではない）に流れるため、赤いダイオードがオンになります（色は問題ではありませんが、私は自転車を発明しませんでした）。



ペアの交差点 ：「パッシブ」パーツのLEDは順番に点灯しません。



もちろん、このデバイスは理想とはほど遠いですが、単純な「ダイヤラー」よりも優れています。 残念ながら、私は写真を投稿する機会がありません。ハリコフのラジオ市場で取引されている人はそれほど多くないという事実にもかかわらず、テスターは最も安い中国のカウンターパートよりも2倍安いことがわかりました。

テスター自体は、配線、3つのマイクロ回路（インバーター、カウンター、デコーダー）、8つのLED、2つのソケット、その他あらゆる種類の小さなもの（抵抗器とコンデンサー）に実装されました。
