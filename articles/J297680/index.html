ニュージーランドでの博士課程または世界の果てまで行く方法

博士課程で外国の大学に入学することについてのメガマインドに関する出版された記事を見たことが何度もあり、最終的にこの貯金箱に私の経験を加えることにしました。 2月1日に、ニュージーランドの首都ウェリントン（ビクトリア大学ウェリントン）にあるクイーンビクトリア大学で電力分野の博士号候補として研究を開始します。









彼の学問的および職業的な過去について簡単に説明します。 私は大学で学位プログラム（5年）を卒業し、2012年に電気機器と船舶オートメーションを専攻しました。5年目の終わりに、大手油田サービス会社から招待を受け、その年の12月に仕事を始めました。 同時に、彼は「電力と電気工学」の方向で政務に行きました。 つまり、彼は実際に会社でフルタイムでローテーションベースで働いており、2014年に修了した修士課程でフルタイムで勉強しました。 私は油圧機器を使って仕事を続けていましたが、2015年に国と会社のすべての金融混乱の後、電力と電気駆動の分野に戻ることを決め、博士課程での研究を続けることを辞めました。 雇用主との契約に署名する前でさえ、私は2つのシナリオを見ました：学術環境または研究開発の分野。 どちらの場合も博士号が必要だったので、業界の危機が始まったので、私はこの道を行くことにしました。



ロシアに候補者を書いてはいけないという独自の理由があったので（提案したが）、オフィスを出た後、モスクワに停車してIELTSに合格し、国からの移動を準備した。 大学に入学するのに十分な2週間後（L7.5 / R7.0 / W6.0 / S6.5 / O7.0）に結果を受け取ったので、NZの語学コースに行くことにしました。口語や文章の部分を改善するために、そして一般的に私はモチベーションのために国を離れる必要があることを理解しました。 そして、試験結果を受け取ってから2か月後、私はすでにニュージーランドのオークランドにいました。



語学コースで勉強する瞬間を完全に省き、新しいトレーニングの場所を探し始めました。 一般に、私はニュー・ゼラドニアの大学をオーストラリアの大学に続く代替飛行場としか考えていませんでしたが、それが起こったように判明し、現在の居住地に満足していないとは言いません。 貯金はできたが、ニュージーランドのようなかなり高価な国での4年間の勉強に十分なお金がなかったことに言及する価値があります。 したがって、唯一の選択肢は奨学金を受け取ることでした。 実際、私はオークランド大学の教授に連絡して、面接に招待してくれました。 彼とのインタビュー、研究提案の任務、その後の改訂と新しい提案。 文書、履歴書、卒業証書の翻訳、動機付けの手紙、英語試験の結果、推薦状の送付。 その結果、しばらくして場所の申し出を受け取りました。 問題は、奨学金がなかったことでした。なぜなら、私は夏休みの医者のための奨学金に遅れたからです。 教授が提案したオプションは次のとおりです。 勉強を始めて、最初の年の支払いをしてから、奨学金を申請してください。 外国人のための博士課程には、地元の人と同じくらいの費用がかかります。 ただし、宿泊には約15〜20 kの費用がかかります。 したがって、ご存じのとおり、このオプションは非常に危険です。奨学金で飛ぶ場合、あなたはかなり難しい財政的trapに陥っていることがわかります。 それはそうかもしれないが、それはスタートだった。 1つのオプションは、財政的な観点からは最善ではありませんが、すでにありました。



それから私は頭とのインタビューを行った。 オークランド工科大学（AUT）。 条件はほぼ同じですが、奨学金ではなく、学部の教師と研究者の割合です。 しかし、大学は非常に若く、国際ランキングにリストされていないことを考えると、提案書を書く時間を無駄にしないことに決めました。 同時に、クイーンビクトリア大学と協力しているロビンソン研究所のプロジェクトが目を引きました。 標準的な手順は、教授に手紙を送り、数通の手紙の後、彼は申請書を提出するために青信号を出します。 すべてが多かれ少なかれ標準であり、次のドキュメントが含まれています。

1.推薦状。 学部長の私の学部と彼の副学長の両方は、両方とも私の卒業プロジェクトの監督者です。 誰もが彼らの信verify性を検証するために彼らを呼んだことはないので、通常書かれた手紙で十分でしょう。

2.研究関心フォームの表現、実際にはあなたのカバーレター。 通常、ほぼ同じ手紙が教授に送られます。 なぜこの大学、プロジェクトなどを選んだのですか？

3.資格と経験のフォーム。 学歴および専門職の過去、記事、会議、その他のメリット。

4. 2000文字のアカデミックレター（記事）の例。 1700年頃に送信しました。不要な質問はありませんでした。

5.翻訳された卒業証書とその付属書。

6.英語の試験結果。 ほとんどの大学では、各部で平均6.5以上、6.0以上のIELTSスコアが必要です。 しかし、ハーバードやオックスフォードのよ​​うな例外は常にあります。



書類を準備し、7月1日である締め切りの1週間前に送付しました。 通年、入学のための書類を送信するための3つの日付があります：3月1日、7月1日、11月1日。 1か月後、教授が私に手紙を書いて、私の申請は現在申請委員会で検討されていると言いました。 電話でインタビューを行うことが決定されましたが、オーストラリアへの旅行のために、そのロビンソン研究所の上級研究員が私にインタビューしました。 結局のところ、彼と一緒に仕事をします。 電話で約1時間、決定を待つのにさらに4〜5週間かかるという言葉を奨励します。 実際、1か月後、場所と奨学金の申し出を受けたという手紙を受け取りました。 NZのすべての大学と同様、博士課程の奨学金は2000 NZDです。 黄金の山ではありませんが、過半数ではないにしても、生きることはかなり可能です。 次に、ビザ、旅行、学生の事務処理を申請し、仕事の開始を待ちます。



特にこの経験が他の多くの国に当てはまるので、この記事が国立科学アカデミーでの博士研究のアイデアを与えてくれればとてもうれしいです。 トピックに関する質問があれば、喜んでお答えします。
