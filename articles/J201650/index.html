地政学シミュレータ-作成と開発の歴史

みなさんこんにちは。



私たちのアプリケーションVkontakte RivalRegionsの作成の歴史を紹介したいと思います。

要するに、これは経済と軍事作戦の要素を備えた地政学的シミュレータです。 コンセプトは単純です。既存の政党に住み、組織し、参加する地域を選択し、議員から大統領まで政治的キャリアを開始します。 さらに、キャラクターのスキルを自然に高め、企業で働き、地域の保護に参加したり、近隣地域を征服したりします。 征服後、州を整理できます。



技術部：



フロントエンド-html / css / js（jquery）。 私たちの意見では、モバイルデバイスを介してプレイすることができる非常に便利なクイックインターフェイスであることが判明しました。 サポートされているテーマ：インターフェイスとアイコンの色を変更できます。 フルスクリーンモードでのブラウザーの制限の混乱：チャットと記事（アプリケーションの統合部分）は使用できません。



サーバー側はphpで記述されています。 チャット、戦争、戦闘（ヘクスのフィールドでの戦闘「チェス」-残念ながら、バックグラウンドに後退したトレーニングの形式でのみ利用可能）-node.js + socket.io。 ベースはMySQLです。



すべて次のように機能します。







同時に、オンラインはアクティブな時間中に300〜400前後で変動します。







アプリケーションはリリース以来開発されてきました（革新のリストはwikiにあります ）。したがって、キャッシュはサーバー上の新しいスクリプト/スタイルの完全なセットの物理バージョンの形式で実行されます。 iframeの存在によって増幅されます。 このために、もちろん、管理パネルから起動される「展開」ユーティリティがあります。これは、グラフ、エラー、チャットストリームを備えたインターフェースです。 これ以前は、ファイル名の後にバージョンメソッドが使用されていました（script.js？V = 120）が、驚いたことに100％の保証はありませんでした。



最初は、アプリケーションは多言語として開発されました。以前のアプリケーションのエラーは繰り返されませんでした（その後、やり直す必要がありました）。 開発は英語とロシア語のフレーズを追加して行われ、他の言語への翻訳はプレイヤーに委ねられています。 このためにシンプルなインターフェイスが追加されました。







行を保存し、Enterで新しいフィールドに翻訳すると、約1000のフレーズを十分にすばやく一緒に翻訳できます。現在のローカライズの進行状況と翻訳者のリストはすべてのプレイヤーが利用できます（現在12言語が追加され、31言語が翻訳可能です）。 翻訳に言語を追加するのに数分かかります。言語の数は生産性に影響を与えません。したがって、革命前、ソビエト、シベリアなどの言語の出現です。



マップ（svgファイル）は、phpスクリプトの助けを借りてjqueryプラグインのマップに変換されて手動で作成されます。キーボードとズームを使用して移動できます。 サイト全体または全画面モードで再生する場合、マップは全画面に展開されます。







4月1日、アルファテストが開始されました。 ゲームマップは、ロシア、ウクライナ、ベラルーシのみで構成されていました。 2週間のテストと5月初旬までの休憩。 ベータテストは5月中旬から6月末まで続き、7月1日にリリースされました。 ストーリーを見ると、ゲームがまだファイナライズされていることがわかります。 2-3日ごとに、ゲームに新しい機能を追加します。

ユーザー自身がゲームのファイナライズに参加するのは素晴らしいことです。他の言語へのほとんどの翻訳は自分の手で行われます。



広告



リリース後、ゲームの収益化とプロモーションについての質問がすぐに発生しました。 このアプリケーションでは、VKontakteの投票で金を購入できますが、ほとんどのゲームとは異なり、実際のお金を投資せずに企業で金を獲得することもできます。 ゴルダは、ゲームの国内通貨と交換されます。



私たちの経験に基づいて促進する最も効果的な方法は、内部連絡先広告です。 私たちがそれに費やしたアプリケーションで受け取った投票のほとんど。



公の場で投稿を購入する試みはありましたが、その有効性は低くなりました。300kの公の投稿に対して、150kのカバレッジの場合は1,500を支払い、そのうち200以上のインストールを受け取りました。



また、アプリケーションに関する記録を再投稿する方法、および記録を共有した人の間での賞品の抽選を試みました。 新しいiPod Shuffle用の500以上の再投稿。



新しいアプリケーションをブロックに追加すると、非常に優れた効果が得られることに注意してください。







計画には、新しい地域の追加、経済的および軍事的要素の改善、そしてもちろん西への出口が含まれます。



このゲームは主に政治的シミュレーターと見なされていましたが、驚いたことに、多くのプレイヤーがゲーム内で自分のニッチを占領しました。 プレイヤーの何人かはparty兵の一団に団結し、党のメンバーではありませんでした。彼らの強さを示すために、彼らはモスクワ（当時の最強州の首都）での蜂起を宣言し、政府を解任しました。 多くは敵対行為に参加せず、民間企業を設立するための資金を節約しています（現在開発中）。



喜んで、私は希望、コメント、推奨事項、アドバイスに耳を傾けます。
