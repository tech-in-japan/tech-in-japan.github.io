生産的なユーモア：Acronis True Imageユーザーからのリクエスト

4月1日に、Acronis True Imageの外国ユーザーからのいくつかの面白いリクエストを共有します（国内のユーザーはまだそのようなリクエストに満足していません）。 無料の翻訳で提供します。











ユーザー ：16進エディターを使用して、いくつかのバイトを修正し、MacでWindowsイメージを復元することは可能ですか？ 私はずっとMacを試してみたかったのですが、Windowsのようにすべてを手動で設定したくありません。

おそらく可能です。 バイトだけを修正する必要があります。











ユーザー ：確かに、さまざまなドライブやコンピューターの信頼性に関する多くの情報を持っています。 または、True Imageは、ディスクが破損するまで時間がかかり、前日にバックアップを作成できますか？

あなたがどこに落ちるかを知ってください...私たちは本当に多くの分析を持っていますが。











ユーザー ：見る時間すらない映画や写真がたくさんありますが、それらは多くのスペースを占有します。 バックアップ中にそれらを非常に強く圧縮して、最小の最小ディスクに収まるようにすることは可能ですか？ 後で復元する必要はありません。 私はそれらを見る可能性は低いです。

ハードドライブ用の良い着信音を書いている人を教えてもらえますか？
