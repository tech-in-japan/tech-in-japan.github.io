サムスンは革新的なGear VRメガネでモバイルバーチャルリアリティの世界を開きます

こんにちは、Habr！



Samsung Electronicsは、新世代のハイテクウェアラブルデバイスを導入しました。これは、Samsung Gear VR Innovator's Editionの最初の仮想現実眼鏡です。 このガジェットは、仮想テクノロジーのリーダーであるOculusの開発に基づいて構築された1.5年で作成され、新しいスマートフォンGALAXY Note 4と互換性があります。







GALAXY Note 4は、新しいフラッグシップを搭載した5.7インチの高解像度スクリーンQuad HD Super AMOLED（2560 x 1440）のおかげで、モバイルデバイスの束で重要な役割を果たします。 Gear VR Innovator's Editionを着用すると、ユーザーはすぐに仮想現実に没頭し、360度の可視性と3D効果のあるビデオを提供します。 メガネで直接、通話を受信して​​メッセージを表示する機能が利用できるため、着信またはSMSでスマートフォンを外すことはありません。







ギアVRメガネは96度の視野角を提供しますが、360度回転しても仮想ディスプレイはユーザーの目の前に残ります。 この装置には、通気孔、2つのレンズ、近接センサー、焦点距離を変更するためのホイール、音量ボタン、ケース側面のタッチパッド、戻るボタン、USBコネクタが装備されています。 寸法は198 x 116 x 90 mmです。







Samsung Gear VRは、頭と顔の形をした柔らかい裏地付きの軽量素材で作られています。 デバイスの設計は完全にワイヤレスであるため、ユーザーはコンピューターから多くのワイヤを出さずに仮想世界に完全に参加できます。 近い将来、サムスンはこのデバイスでの視聴に最適化されたコンテンツの幅広い選択を提供します：幅広い映画、ゲーム、360°ビデオがメガネユーザーの武器庫に登場します。これには、ユーザー生成コンテンツの市場のリーダーによる実験ビデオや教育ビデオが含まれます。







米国での仮想現実眼鏡のSamsung Gear VRの価格は199ドルで、ガジェットは今年の終わりまでに販売されます。
