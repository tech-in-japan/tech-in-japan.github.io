完璧なソリューション= PS3 + PS3 Media Server

最近、PS3プレフィックスが私の手に落ち、ゲームがまったくなくなりました。 耳の隅から、メディアサーバーとは何かを聞いたことがありますが、すべてを試すことはできませんでした。 私はラップトップでテレビ番組や映画を見ることは完璧な解決策だと思いました。 進歩は止まりません。

適切な人がPS3で最初に行うことは、テレビに接続することです。 だから私は、32インチでHD 1080pをサポートする自宅のPhillipsを使用しました（1080iを混同しないでください）。





さらに、古くて弱いが、非常に静かな、巨大なHDDを搭載したコンピューターが私の場所に横たわっていました。 Ubuntu 9.10がインストールされています。 PS3メディアサーバーをインストールしました 。 サーバーのセットアップには2分かかりました。複雑な手の動きはありません。 この理由から、サーバー自体はJavaで記述されており、LinuxとWindowsまたはMacで動作します。 プロジェクトアカウントはGoogleコード上にあり、誰でもソースを読むことができます。



デバイスをサーバーに接続する原理を以下に示します。 家庭用デバイスにwi-fiモジュールがある場合、非常に便利です。





フレンドリーウィンドウPS3メディアサーバー：





PS3の電源を入れ、メディアサーバーを探します。ビデオまたは音楽を選択できます。





最後に追加したいのは、ps3は多くのビデオ形式をサポートしていないが、ps3メディアサーバーはリアルタイムでビデオをある形式から別の形式にトランスコードできることです。 トランスコーディングは、コンピューターにインストールされているコーデックに依存するため、必要なすべてのコーデックライブラリを配置する必要があります。 Linuxの場合、Mplayerをダウンロードし（依存関係では必要なすべてのコーデックをドラッグします）、Windowsの場合はk-liteをダウンロードする価値があります。 メインのトランスコーディングパラメーターは（FPS）/（画質）です。プロセッサーの速度と、サーバーとPS3間の接続の種類（イーサネットまたはWi-Fi）に柔軟に調整できます。
