「ホワイト」給与について

「白い」給料を払って、標準的なウェブスタジオがどれだけ与えているか知りたいですか？ カウント！



付随するすべての特典を税制USN（6％）にしましょう。

従業員に、たとえば1000ルーブルなど、きれいな手を受け取るようにします。 （計算を簡単にするため）。



いま

PIT： （1,000 / 0.87）= 1,149

FIUへの貢献： 1,149 * 1.14 = 1,310

合計：+ 31％







2011年以降

PIT： （1,000 / 0.87）= 1,149

FIUおよびCHIへの貢献： 1,149 * 1.26 = 1,448

合計：+ 44.8％



2013年から

PIT： （1,000 / 0.87）= 1,149

FIUおよびCHIへの貢献： 1,149 * 1.34 = 1,540

合計：+ 54％



あらゆる種類の小さな特典と控除がありますが、Webスタジオにとっては現実的ではありません。



「白い」給料を支払うことはいつ利益がありますか？



また、6％の税金があります。これは、Webスタジオの全収入から支払わなければなりません。 FIUおよびCHIへの多額の支払いにより、3％に削減されます。 単純な計算では、各従業員が会社の給与の10倍の収入を得る場合（2011年）に、この減少によりFIUと義務的な医療保険の費用が完全に補償されることが示されています。 同時に、スタジオの総月収は500万ルーブルを超えてSTSに残ってはなりません。
