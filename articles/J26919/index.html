Objective-J？ スタートアップはApple KeynoteのWebバージョンを導入しました

ブロガーのDru Nelson が280 Northのスタートアップと280Slides Webアプリケーションについて書いています。



それがPowerPointやKeynoteの単なるWebベースの類似物であれば、ニュースは大きな波を起こさなかっただろう。 しかし、ドリューが「ボンネットの下」を見て見たとき、すべてがもっと面白くなった... 







  objj_import（ "main.j"、YES）;




main.jソースはさらに興味深いことが判明しました。 



  //
 // main.j
 //エディター
 //
 // Francisco Tolmaskyにより作成。
 // Copyright 2005-2008、280 North、Inc. 無断複写・転載を禁じます。
 //

 import <Foundation / Foundation.j>
 import <AppKit / AppKit.j>

 import "Document.j"
 import "DocumentController.j"

 [DocumentController sharedDocumentController];

関数メイン（args、namedArgs）
 {
     CPApplicationMain（args、namedArgs）;
 }
 ...




Objective – Cを見たことがある人なら誰でも、上記のコードとCocoaアプリケーションで一般的に見られるものの類似性（同一性でない場合）を簡単に見ることができます。



Redditには、会社の創設者が以前はAppleの従業員（iPhoneおよびiTunesチームから） だったというコメントでいっぱいでした。280Slidesは、WebKitナイトリー（および、おそらくSquirrelFish ）の優れたテストサイトです。



280 Northは、JavaScriptでObjective-C（またはObjective-J）を使用して翻訳者を書きました。 ほとんどの翻訳は実行時に行われ、プリコンパイルは行われません（またはほとんど行われません ）！ このライブラリは、CanvasとSVGの両方と（必要に応じて）Flashを使用します-結果は単純に驚くべきものです。
