広告サーバーSepulka.ru





スタートアップとは、夏までに太平洋の島を購入する時間があることを望んで、仕事を辞めてエネルギーと最後の給与のすべてをウェブサイトに投入することですか？ その後、私は典型的なスタートアップを持っています:)しかし、私のスタートアップと古典的なスタートアップを区別するものがあります-私は気にしなかったし、新しいアイデアを思いつきませんでした



それはすべて、昨年12月に偶然にも、よく知られているすべてのサービス（ Izrukvruki 、 Chance 、 Auto.ru ）の助けを借りて、1週間の間にいくつかのアナウンス（私の友人のカップルのアナウンス）を提出する機会があったという事実から始まりました。 ユーザビリティによって経験されたユーザーフレンドリーからのショックは過ぎ去り、多かれ少なかれ世界をより良く変えることが決定されました。



会う： Sepulka.ru広告サーバー



既存のサービスが気に入らない点は何ですか？



まず、登録。 送信前にフィールドの有効性をチェックして、通常のキャプチャで「レベル」にあると便利です（または、少なくともキャプチャが入力できなかったときにすでに完了したフィールドはリセットされません）。 30分待たないように、アカウントの有効化に遅れが生じた場合、すぐにホスピタリにログインして、手紙のリンクをたどることを思い出させることができます...しかし、これにより、モデレーターの生活が容易になり、スパマーが複雑になります。



第二に、広告提出フォーム。 それはただ「威勢のいい90年代」です！ どこでも順番にいくつかのフォームに記入する必要があります。 エラーの場合、戻るフィールドはリセットされます。 勤勉にアップロードされた数十枚の写真が消えます！ もう一度始めましょう...信じてください、これはインターネットでの私の最初の年ではありませんが、ここに写真付きの広告を投稿しようとする気持ちがあります-社会保障機関または私たちの地域の税に行く方法！ しかし、それだけではありません。Izrukvrukiの Webサイトですべてを「正しく」記入し、禅仏教のキャプチャを理解したとき、私への答えは...注意でした。 ...「タイムアウトしました。」



そして、私は悟りを得ました！ 私はあなたが怒るのをやめ、怠realizedなプログラマをprogrammerるのではなく、自分でそのようなサービスを作り、既存のサービスよりも便利にする必要があることに気付きました：



1.登録を拒否します。

年に1回、ほとんどの人に広告を掲載します。 この場合、広告の編集、削除、および一般的な管理の問題は、Cookieを使用して解決できます。 最悪の場合、アナウンスにさらにアクセスするためのコードを提供します（別のコンピューターの場合）。 通常の顧客の場合、登録を自主的に行うことになるでしょうが、これは計画中です...



2. 1つのフォームに収まるようにしてください。

1つのフォームに記入して送信してください-完了です！

例： http : //sepulka.ru/post



3.フォームを送信する前に、すべての検証チェック、写真とビデオのアップロード。



4.写真のアップロードを便利で明確にします。

つまり -フォームをリロードせず、無効なフィールドなどの場合にロードを失うことなく プラス-写真をトリミングおよび回転する機能。

同じ場所の例： http : //sepulka.ru/post



5.ビデオを追加します。

今では珍しいことですが、今でも出席が少なく、これを必要とする人がいました。

ビデオは、http：// youtubeまたはrutubeの形式のリンクによって簡単に挿入され、アバターが自動的に生成されます。

動画からの広告の種類に注意してください ： http : //sepulka.ru/2487 、表示する際の写真と動画の組み合わせに注意してください 。 便利ですか？



6.車、アパート、ペットなどのディレクトリを使用できるセクション（メーカー/モデル、地区/地下鉄、品種など）については、既製のドロップダウンリスト、チェックマークなどを含む特別なフォームを提供し、それらをロードします。パラグラフ3によると、動的に。 さらに、保存するとき、パラメーターは対応するテーブル（車、アパート、動物用の別のテーブル）に記憶され、パラメトリック検索（例：メトロからのエンジンボリューム、メトロからの距離など）に使用されます。 また、広告テキストは、パラメータのテキストコンポーネントからテンプレートに従って形成されます（たとえば、Volkswagen Golf III、1992。1.9ディーゼル、3ドア、加熱シート...-これは手で入力するのではなく、リストとチェックマークから選択します）。 これは、テキスト検索の均一性と品質に貢献します。



7.単一の分母への価格の自動再計算

価格は選択した通貨で入力でき、価格の範囲で正しく並べ替えて検索できるように（ルーブル単位で）再計算されます。 ここでは、為替レートの自動インポートを開始する時間がありませんでした。 私はこの主題へのリンクに感謝します、突然あなたは最近そのようなものに対処しなければなりませんでした...しかし、一般に、価格で考えるべきものがあります-例えば、不動産：価格はメートルと数百の両方で示されています これまでのところ、これをいくつかの数字に減らすことができるかと思います。



8.広告にタグを付ける

タグはユーザーが自由に設定し、さらにロボットが特定のカテゴリの共通タグのセットを付加します。 さて、たとえば、子猫の販売についてのアナウンス...テキストには子猫、猫、猫などの単語が含まれていない場合がありますが、これらの単語でこの広告を見つけるのは悪くありません。 ここで、ロボットがそれらを置き換えます。 現在、タグは表示されませんが、検索で考慮されます。

9.便利な検索を行います。

タグとパラメーターのテキスト表現を考慮した、広告のテキスト全体の単純なテキスト検索（段落6）。 さらに、困難なケースでは、独自のパラメーターセット（車、アパート、犬、猫用）を備えた各カテゴリーごとにパラメトリック検索を行います。 例： sepulka.ru/supersearch/?filter%5Bsearch%5D=%D0%9F%D0%BE%D0%B8%D1%81%D0%BA&filter_rubrica=117.0&filter%5Btab%5D=&sup.x=41&sup.y = 12

（理由によりGETを介してフィルターパラメーターを渡しません。POSTや他のニュアンスとは異なり、ナビゲーションの戻りは影響を受けませんが、行の長さを減らすことを既に検討する必要があります）。



10.検索結果の便利な作業。

必要なアナウンスをメモに追加し、ランダムに並べ替え、印刷します。 rssを使用して、生成されたフィルター（要求）に接続できます。

まあ、これはWebDvOnolです:)そして、メールでアナウンスの著者にメッセージを送信するスパム保護された形式です。
