FullHDビデオカメラPanasonic HDC-SD60のレビュー

こんにちは、％username％！

新しいおもちゃ-FullHDカムコーダーPanasonic HDC-SD60を手に入れたので、急いで感想と機能の簡単な概要を共有しました。









外観といくつかの数字



カメラは非常に小さくて軽く、手にしっかりと収まります。







ボタンのセットは最小限です：

  • 上記-インテリジェント自動モード（iA）と手動モード（手動）を切り替えるボタン、画像安定化システム（OIS）モードスイッチ、ズームコントロールレバー、写真スナップショットボタン
  • 背後-再生、ビデオ、写真モードの切り替え（IMHOはあまり便利なスイッチではありませんが、頻繁に切り替える必要はありません）
  • LCD画面の下-記録、ズーム、メニュー、照明制御、削除ボタン
  • 3つのインターフェイスコネクタが画面のパネルの下に隠れています：HDMI、USB、AV /コンポジット出力、およびオン/オフボタンとメモリカードスロット


他のすべては、非常に反応の良いタッチスクリーンを介して制御されます（親指を持っている人は操作が難しいでしょうが、彼らにとっては珍しい/珍しい形状のスタイラスがあります）

次に、特性について：

イメージセンサー 1/4 "MOS
レンズ口径 F1.8（ワイド）/ 3.3（TELE）
光学ズーム 25倍
圧縮形式 MPEG4-AVC / H.264（AVCHD規格準拠）
画像解像度 1980x1080、品質：HA（17Mbps / VBR）、HG（13Mbps / VBR）、HX（9Mbps / VBR）、HE（5Mbps / VBR）

重量（電池を除く） 255グラム
寸法（WxHxD） 51.5 x 65.5 x 112.0 mm




テクノロジー



このカメラの技術については公式ウェブサイトに美しく書かれていますが、私は嘘をついていないことを付け加えます。 そして、写真をほとんどコピー＆ペーストします：



インテリジェントズームと光学式手ぶれ補正


実際、インテリジェントズームは光学ズーム（25倍）+少しデジタル+最後にシャープネスを追加すると、35倍の光学ズームがある場合と同じになります。



光学式スタビライザーが更新されます（正確に何が新しくなったのかはわかりません）。これまでのように素晴らしく機能します。



広角レンズ




これはまさに以前の部屋では非常に欠けていたものです-この部屋で起こっていることを取り除くために部屋の端に押し込む必要はもうありません（さらに広く...）



おなじみの顔




最も気に入ったカメラ機能。 一番下の行はこれです：最大6人（1人あたり1〜3枚）までカメラを「表示」して撮影します。 この場合、カメラは、左の通行人ではなく、見慣れた顔に自動的に焦点を合わせます。 また、特別モードをオンにすると、6人のうち1人が笑顔になった場合、カメラは写真（5メガピクセル）を撮ります。 カメラは頑固に動物を記憶したくないが、犬の顔を顔として認識することもある。



そして、あなたの名前が誰であるかを忘れた場合、あなたはその人をカメラに向けることができ、彼女はあなたにその方法を教えてくれます;）



トラッキングオートフォーカス




一番下の行は単純です。画面上で誰かに突くとカメラがこのオブジェクトに焦点を合わせ、人物が選択されると、顔が最高品質で見えるように明るさ/コントラストを調整します。





自動撮影モード/シーン選択




要点は簡単です-照明、顔の存在、焦点距離に応じて、カメラ自体が最適な撮影モード（自動、ポートレート、パノラマ、夕方）を選択します



マニュアル撮影モード



AIを信頼しない人や、この同じAIが夢中になり始める条件のためのモード。 ここで選択肢が与えられます：
  • 自動/手動フォーカス
  • 絞り/シャッタースピード/ホワイトバランス設定
  • カウンターライト補正
  • 肌の色の調整
  • ナイトモード（遅いシャッタースピード）
  • マクロ撮影




録音時間



彼はカメラと一緒に32 GBのSDHCメモリーカードを購入し、利用可能な最高品質（1980x1080 17 Mbps）で約5時間のビデオを録画できるようになりました。 これはすべて素晴らしいですが、バッテリーは最大2時間の撮影に耐え、追加および/または増加した容量を購入することはまだできません（ロシアでは販売されていません）。 だから、彼らがイベントの撮影を要求した場合、あなたはあなたと一緒に延長コードを取る必要があります=）



最後に、射撃自体について



すでに述べたように、カメラは1920 x 1080（1080i）の解像度で撮影します。古いminiDVカメラの後、写真は完璧に見えます：木には小枝が見え、犬には毛皮が見え、寄木細工にはすり傷があります。 日中の撮影については何も語りません。写真が美しく、鮮明でジューシーであることがすでに明らかであるため、手ぶれは写真に何の影響も与えません-パナソニックOISによると、長く愛されてきた手ぶれ補正

そのため、夜間または暗い場所での撮影が必要です。 いくつかのオプションがあります：内蔵照明（超高輝度LED）を使用するか、夜間モードを使用するか、何も使用しません。 ビデオを見るのは意味がありません：

ナイトモード
ビルトインライトの有無にかかわらず（夜間、密室で撮影した場合、街灯からの光はほんのわずかでした）
何も使用しませんでした（モスクワ時間の午後8時頃）
まあ、私たちの新しい犬を見てください




写真について少し



前に書いたように、フレームに馴染みのある顔や笑顔の顔がある場合、カメラ自体は最大5メガピクセル（もちろん補間、実際には3メガピクセル）の解像度で撮影しながら写真を撮ることができます。

写真の品質はかなり平凡です（これは通常のカメラと比較した場合です）。 フラッシュがあるので、夜に写真を撮ることができます。



一般的な印象と価格



これに先立ち、Panasonic NV-GS320 3マトリックスminiDVカメラ（および以前のmustek dv3000）を使用しました。これは、ビデオをテープからコンピューターに転送するプロセスに少し負担がかかりました。 これですべてがシンプルになり、ビデオの品質が向上しました（1080i対576i）。

そして最も重要なのは、価格：17,490ルーブル+メモリカード〜4,000ルーブル（執筆時点では、カメラの価格は17,337から21,720ルーブルの範囲でした）
