Microsoftは、Windows 7のリリース後もWindows XPをリリースし続けます



Microsoftは、Windows 7がリリースされた後もネットブックにインストールするためにWindows XPオペレーティングシステムを出荷し続けるとZDNet NewsはMicrosoftの担当者からの声明を引用して報告している。



マイクロソフトは、超安価なコンピューター上のWindows 7がXPよりも人気があると考えています。 その結果、Windows 7 PCは、古いオペレーティングシステムを搭載した同様のマシンを市場から押し出すだけです。



Windows XPの販売は2008年6月30日に停止したことを思い出してください。 その後、Windows XP HomeのみがOEMバージョンのままで、超低予算コンピューターへのインストール用に設計されました。



超予算のデスクトップおよびラップトップ向けのWindows XP Homeの配信は、2010年半ばまで続くことが以前に報告されました。 この日付がWindows 7のリリース後に変更されるかどうかは不明です。



Windows 7は、Windows Vistaの後継製品です。 このOSの正確なリリース日は不明ですが、アナリストおよび市場参加者は、2009年末または2010年初頭に登場することを期待しています。



Windows 7は6つのバージョンでリリースされます。 そのうちの1つは、切り捨てられた機能を特徴とし、ultradecheコンピューターへのインストール用に設計されます。 特別なMicrosoftプログラムに従って、Windows XPで動作するそのようなコンピューターのユーザーは、Windows 7の対応するバージョンを割引価格で購入できます。



Windows 7オペレーティングシステムのベータ版は、2009年1月9日にMicrosoftのWebサイトに掲載されました。 1か月以内に、誰でもダウンロードできました。 © Lenta.ru



PS：Windows XP用のService Pack 4を待ちたいです。
