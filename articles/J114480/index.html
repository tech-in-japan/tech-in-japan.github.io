MicrosoftとSoftlineでクラウドに！





2011年3月4日に、Softlineは、ロシアのMicrosoft駐在員事務所とともに、 モスクワで Microsoft Windows Azureクラウドプラットフォーム上で Microsoft BizSpark Camp スタートアップ向けの会議を開催します。 イベントでは、クラウドでのビジネスの利点について説明するとともに、Windows Azureの主要コンポーネントと、SaaS（サービスとしてのソフトウェア）モデルでアプリケーションを作成するためのシナリオを詳細に検討します。



会議中および会議後には、実験室での仕事を行う機会が与えられます。 会議参加者には、Windows Azureへの30日間のアクセスと 、Microsoftクラウドプラットフォームに関する本が提供されます。 セミナー後に、Windows Azureプラットフォームでのソリューションの実装に関する技術的な質問、プラットフォーム自体に関する質問、ソリューションアーキテクチャなどがある場合は、個々の専門家のアドバイスを受けることができます。 Microsoftクラウドプラットフォームでソリューションを作成したい（または計画している）スタートアップ企業を招待します。
