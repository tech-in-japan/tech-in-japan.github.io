Windows 8でもLokata製品およびサービスファインダーを使用できるようになりました





地元の小売業者の商品やサービスを検索および宣伝するために設計されたロシアのジオロケーションサービスであるLokata.ruプロジェクトについて、Habréにすでに投稿がありました。 LokataのWebサイトとモバイルアプリケーションは 、ジオロケーションと検索エンジンを組み合わせて、顧客が最寄りの店舗で必要な商品を見つけられるようにするとともに、小売業者に関する必要な情報（営業時間、住所、電話番号）を取得できるようにします。 1つのアプリケーションですべての店舗のカタログを収集するというアイデアは、ショッピング旅行を計画している人が時間とお金を節約したい人に委ねられました。



同社は最近、デスクトップとタブレット向けのWindows 8アプリを発表しました。 その立ち上げは、主にモバイルデバイスユーザーに焦点を合わせたマイクロソフトの新しいOSの販売開始とほぼ同時に行われました。



Windows 8はプロジェクトチームの新しいプラットフォームであるにもかかわらず（それ以前はiOSおよびAndroid用のモバイルアプリケーションが作成されていました）、Lokataアプリケーションは記録的な速さで開発されました。 これは、C＃言語のシンプルさ、優れたトレーニング例、およびマイクロソフトとコミュニティからの優れたサポートによって可能になりました。 特に、これがC＃を選択した理由でしたが、同じ機能をC ++、JavaScript、HTML5、VBでも実装できました。 Visual Studioについても言及する必要があります。 Expression Blendと組み合わせて、これはおそらく私が使用した中で最高のIDEの1つです。 コードを書くのはとても便利で簡単です。 ただし、プラットフォームの新規性に関連する軟膏にハエがあります。 たとえば、Bing Maps SDKが利用できない、または一部のドキュメントが不完全です。







Lokataのデザインは、Win8のデザインに適合します。 このアプリケーションは、新しいOSのすでに一般的なスタイルで設計されており、「近くにある」、「最寄りの店」、「お気に入りに追加する」という3つのワークスペースを表しています。 カタログ、小売業者からのオファー、ジオタグが追加されると、ワークスペースが拡大します。



Windows 8用のLokataアプリケーションは、モバイルデバイスの組み込みGPSモジュールを使用してユーザーの位置を特定することを除いて、Webバージョンのサービスの機能をほぼ完全に繰り返します。 いつでも自動検出をオフにして、ご希望の都市と検索エリアを自分で入力できます。 その後、システムは結果を生成し、提案の関連性とあなたから店舗までの距離でソートします。



Lokataアプリケーションの重要な機能の1つは、タブレットまたはデックストップ画面でディレクトリを表示できることです。 現在、サービスデータベースには、500を超える現在のカタログ、パンフレット、さまざまな小売店のチラシがあります。 それらのいずれかを「めくる」には、検索バーに製品またはブランドの名前を入力するだけで十分です。システムは、検索語に出会うすべてのディレクトリを表示します。







ユーザーには、新しい小売店のオファーについて通知するオプションもあります。 [お気に入りに追加]をクリックすると、販売に関するニュース、現在のプロモーション、興味のあるアウトレットやブランドの新しいカタログを購読できます。 マークされたストアがアプリケーションワークスペースに表示されます。







Lokataアプリケーションは 、商品と店舗の検索と選択において常に支援者になるように設計されています。 このサービスはモビリティを目的としているため、アプリケーションの主なユーザーは、Win8を搭載したタブレットの所有者になる可能性が高く、多くの人が範囲を確認し、カフェ、公園のベンチ、公共交通機関の近くにあるお気に入りの店で最高のお得な情報を見つける機会を好むでしょう。
