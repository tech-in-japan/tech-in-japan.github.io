Habrのインターフェイスデザイナー-初年度の結果

みなさんこんにちは！ この投稿では、その年の記録の統計を調べたいと思います。

事実、私はUX / UIデザイナーとして、突然Habrで書き始めました！ ちょうどそうなった、ごみ。 それはほぼ1年前です。 レコードは大幅にリークされました-最初のパンケーキはゴツゴツしています。 しかし、誰かが貴重なものを見つけて、読み取り/書き込みモードをくれました。 ところで、いい人ありがとう。 この投稿を見たら、そっとうなずいてください。



別の忙しい年が終わりに近づいており、在庫を確保する時が来たということはニュースではありません。 事実、私はこれをやったことがありません。 そして、そうする人々を理解しませんでした。 1年は過去の出来事の単なる単位です。 明日の人生は続きます。ゼロからは始まりません。



それでも、1年前の12月14日にHabrに招待されました。 私の最初の過去のハブロゴッドについては、人生の過去の年について世界的に哲学するよりも、線を引くほうがずっと面白いと思います。 それで、私はこの期間に20の投稿を書きました。 1年のうちに読者からの顧客を獲得する方法についての私の話を知りたい場合は、katへようこそ。





ちなみに、Figmaを使用する場合は 、 既製の設計システムに注意することをお勧めします 。 フリーランサーが1か月あたりの注文を増やすのに役立ち、プログラマーは自分で美しいアプリケーションを作成できます。また、チーム作業用の既製の設計システムを使用して、チームが「スプリント」スプリントをすばやくリードできます。



また、深刻なプロジェクトがある場合、当社のチームは、ベストプラクティスに基づいて組織内に設計システムを展開し、Figmaを使用して特定のタスクに合わせて調整する準備ができています。 Web /デスクトップ、および任意のモバイル。 また、React / React Nativeにも精通しています。 Tへの書き込み： @kamushken




背景



私はおそらくここにいます 地獄 私はこれを言うだろうとは思わなかったでしょう:)実際、1年前、プロジェクトは低迷し、見通しは霧で、脳は揺れを必要としました。 特に、私が少なくとも15年間デザインに携わっていることを考えると、自分の経験を定式化してテキストで公開できるという気持ちがありました。



率直に言って心理的な言葉で言えば、第二の理由は「召命への渇望」です。 文字通り取得しないでください。 私にとっては、行われたことに対する承認を得る必要がありました。 インターフェース設計者が長い間プロジェクトを作成するとき、彼は最悪の解決策を提供せず、プロジェクトはベータに達しません-彼は退屈し、彼は疑念にうんざりしています。 または、プロジェクトがベータに達し、良いフィードバックが届き始め、資金が尽きました。 そして、これは私の練習にもありました。



しかし、私がHabrで書く多くの理由を理解するために、私は一年を通して徐々に来ました。 「呼び出し」の必要性私は、数か月後、ある「冷凍」製品を説明しようとしたときに気付きました。 インターフェースで何をどのように行ったかを一般的に説明できることがわかりました。 そして、それは興味深いことが判明しました。 このプロジェクトは古く、残念ながら一時停止しており、NDAの対象外でした。 あなたは基礎として取ることができます。



一般的に、ブログを始めました。 私はこの言葉が好きではありませんが。 私は白衣を着た想像上のエンジニアであり、彼の研究について細心の注意を払っていると信じています。 もちろん、私は最初の一連の投稿でそれらに行ったことはありません。 その後、投稿の統計と聴衆の反応を聞き始めました。 その結果、私はプレゼンテーションの言語と投稿のトピックを調整して、より良い「入ってくる」ものを理解しようとしました。 何でも撃つことができます。 推測することはありません。 プレゼンテーションにさまざまなアプローチを試みましたが、そこでは時々、少しのマーケティングといくつかのトリックを適用しました。 これは、ユーザーエクスペリエンスリサーチラボの白衣を着た同じ架空の研究エンジニアであるという自信につながりました。



私の計画はうまくいき、今年の後半には新しいクライアント、さまざまなプロジェクト、興味深い仕事がありました。 人々は喜んで私のサービスの費用を受け入れ、私は8月中ずっと刺激を受けました。 並行して3つのプロジェクトを主導した方法を今でも覚えています。 就業日は12時間まで落ち着くことができました。 しかし、それは私にインスピレーションを与えただけです！ 私は本当に知的労働の恩恵を享受し始めましたか？ したがって、記事を安定した収入に変換できますか？ これらの質問に対する答えが徐々に出てくると思います。



このスキームは本当に機能しているようです。知識を無料で共有し始めると、良いことが起こります。 私は今でも転向点であることが判明した楽しい驚きを思い切って期待しています。 いいえ、彼らはGoogleで私に電話しません。 興味深いプロジェクトと興味深い条件にしましょう。

そして、このために私は自分自身とHabrに感謝します。
