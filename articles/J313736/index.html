中国人が使用するアプリケーション、ゲーム、オンラインストア





私は最近、10代の若者が米国で使用しているアプリに関するCNNニュースに関するメモを読みました。 彼女は私にとって非常に好奇心が強いようでした。 中国人が使用するアプリケーションやゲームについて話すことで、外国人の関心を満足させると思います。



中国では、アプリケーションエコシステムのユーザーは、主に地理的グループと年齢グループによって分けられています。 この記事では、1番目と2番目のカテゴリの都市の「20〜50歳」の年齢グループのアプリケーションについてのみ説明します。 そのような都市では、生活水準は先進国の水準に近い。



この記事では、10台の電話のうち8台にインストールされているアプリケーションのみを紹介します。 説明に時間を無駄にしないために、このようなアプリケーションには金色のフレームが付いています。 これは、そのようなアプリケーションが非常に人気があり、ほとんどすべての電話で見つけることができることを意味します。







ピンクの枠は、中国で使用されていない同様の中国語以外のアプリケーションを示します。



インスタントメッセンジャー









WeChat（微信）とQQの両方が、モバイル大手のTencentによってリリースされました。 WeChatは、WhatsAppに機能的に近いです。 しかし、これは多くの拡張機能を持つ巨大なアプリケーションです。「Circle of Friends」（朋友圈）と呼ばれるソーシャルネットワーク機能があり、Facebookに非常によく似ています。 中国には多くのソーシャルネットワーク、写真共有プラットフォーム、およびマイクロブログがありますが、それらはすべてWeChatの普及後にほぼ非アクティブになっています。



WeChatのもう1つの重要な機能は、組み込みの電子ウォレットです。これは、ほとんどの小売店で「タッチペイ」ベースで使用でき、オンライン購入やアプリ内購入も可能です。 最近、TencentはWeChatウォレットを積極的に宣伝し、購入ごとに店舗と顧客の両方に補助金を提供しています。



以下で説明するWeChatおよびその他のアプリケーションの例から、中国のアプリケーションがオールインワンアプローチを目指していることが理解できます。



QQは、オフィスのデスクトップコンピューターでより多く使用されます。 競争とWeChatは、西側のSkypeとWhatsAppとまったく同じです。 人々は主にビジネス上の問題についてコミュニケーションをとるために携帯電話にQQをインストールします。



ウォレットと金融アプリケーション







財務面では、ほとんどの人がAlipay（支付宝）と銀行のアプリケーションをインストールします。 Alipayは、Android PayおよびPayPalでも同じことができる別の包括的なアプリケーションです。 さらに、電気やガス、通信料金など、あらゆる種類のサービスに対して支払うことができます。



アリペイはほとんど銀行のように機能し、すべての銀行機能を備えています。資金の引き出しと預け入れ、他人の銀行口座への送金、証券や保険の購入などです。



仮想の財布としてのアリペイの最大の中国のライバルは、WeChat財布です。





ほとんどの店では、AlipayウォレットとWaChatウォレットのアイコンを見つけることができます。購入の際に支払うことができるコードをスキャンしてください。



アリペイには無数の拡張機能があります。 たとえば、WeChatに似たメッセンジャーがあり、Grouponのように、あなたに最も近いレストランやエンターテイメントを見つけるためのページもあります。



オンラインショッピング







モバイルアプリケーションの時代以前は、Ta宝網（淘宝）は中国最大のオンライン市場でした。 その上には、国際的な巨大企業から個人への売り手がいました。 Ta宝網では、オートバイからつまようじまで、中国で合法的に販売できるものがすべて揃っています。 Ta宝網の多くの店舗では配送サービスを提供しており、パリとニューヨークから非常にリーズナブルな価格で商品を購入できます。 Ta宝網はまた、発展途上国の人々に安価で高品質の中国服や家庭用品を購入する機会を提供しています。





駅の中にはZ江の速達会社があります。 従業員は、Ta宝網店のパッケージを毎日処理します。



JD（京东）はAmazonのような会社で、Amazonとほぼ同じ販売を行っています。 彼らは中国のアマゾンとの激しい競争をしている。 JDは電気製品の販売で知られており、Amazonは本の販売で知られています。



タクシー検索







中国では、現在、ユーバーと地元のディディの間で激しい戦いが行われています（嘀嘀）。 DiDiは、大小さまざまな都市をカバーしています。 あなたはDiDiによって認可されたタクシーを呼ぶことができます。 Uberはその場所を求めて戦い、より低い料金を提供しているようです。



ニュース







多くのニュースアプリがあります。 最も人気のあるのはNeteaseとTencentです。



地図とナビゲーション







Baidu Maps（百度地図）は、本質的にGoogle Mapsからコピーされます。 中国以外の国からのデータは不完全であるため、これらは中国国内でのみ有用です。 ちなみに、Googleは中国でブロックされているため、地図アプリケーションの使用は失敗します。



Gaode Mapsアプリ（高德地图）は、最も人気のあるカーナビゲーションソフトウェアになりました。 古いGPSナビゲーターにインストールされ、有料で購入できます。



レストランとエンターテイメントを検索







これら2つのアプリケーション、Meituan（美团）とNuomi（糯米）は、Grouponに似ており、互いにほぼ同一です。



まず、GPSによって現在地を特定し、選択したフィルター（距離、評価、価格など）に応じてストアまたはサービスを提供します。 これらの中国のアプリケーションは、米国の対応するアプリケーションよりも多くの種類の消費者サービスをカバーしており、より詳細な分類を持っていると感じています。 アパートからさらに数ブロック歩くと、80％を超える企業が食品販売、マッサージ、ルームサービス、ジムなどのアプリケーションに登録されていることがわかります。



追加のユーザーを引き付けるために、これらのアプリケーションは、注文ごとに割引を伴うキャンペーンを実施します。つまり、実際にユーザーを実際のお金で「購入」します。



持ち帰り用食品







持ち帰り用の食べ物は主に、サンドイッチやトルティーヤなどの非公式の料理やスナックです。 米国では、そのような食べ物は主にシームレスアプリとGrubhubアプリを通じて注文されます。



中国では、18コースのダイナーからファーストフードの飲食店まで、レストランや食料品店では、顧客の注文に応じて食事をパックして配達します。 以前は、顧客は電話で注文を残していましたが、現在はアプリケーションを使用しています。



最も人気のあるアプリケーションは、Meituan Waimai（美团外卖）とBaidu Waimai（百度外卖）の2つです。これらは、レストランやストリートカフェと連携しています。 彼らは自分の宅配便スタッフを持っています。







これは、中国では興味深い光景です。美しいユニフォームと電動スクーターの宅配便のグループが、大きなレストランの隣の「ダイニング」通りで待っています。 システムから申請書を受け取った彼らのうちの1人はレストランに入り、包装された食品を受け取り、それを顧客に届けます。



ゲームおよびアプリケーションストア





最も人気のある5つのシングルプレイヤーゲームとMMOゲーム。



ゲームは、販売によってお金を稼ぐという点でアプリケーションとは異なります。 今日の中国のゲーム市場の規模と傾向は大きな注目を集めています。 中国人プレイヤーの数は膨大で、どのジャンルのゲームにも天文学的なプレイヤーがいます。 記事では、最も人気のあるジャンルのみを検討します。



中国のモバイルゲーム市場の一般的な状況は次のとおりです。最も人気があり、最も収益性の高いジャンルはMMORPGで、その中には無数のゲームがあります。 他の種類のゲーム、たとえば「3列」、「タワーディフェンス」、およびカードゲームでは、ナンバーワンのゲームは通常独占的な地位を保持しています。



MMORPG







梦幻西游（「 西遊記 」のファンタジー）は、国を動機とするゲームであり、多くのファンがいる古いフランチャイズに属します。 梦幻西游と多くの同様のMMORPGには共通の機能があります。それらはすべて高品質で美しいグラフィックを備えています。 通常、それらは無錫 （武術）の中国神話に基づいています。







使用される戦闘システムに関係なく：JRPGアリーナ、ディアブロスタイル、または2Dスクローラー-それらはすべて同じゲームメカニズムを持っています：レベルアップ、武器検索、基本的なクエストタスク、中国人の心に非常に近い社会的およびチームタスク。





西游は、JRPGの戦闘スタイルと自動戦闘を備えたモバイルデバイスの簡略化を使用します。



以下は、人気のMMORPGのスクリーンショットです。







3列（マッチ3）







開心消消乐（Happy Match）は中国全土で人気のあるゲームで、月間1億元（$ 15百万）の利益を生み出しています。 彼女のスタイルのため、キャンディクラッシュやクッキージャムシリーズなど、世界で最高の3列のゲームと並ぶことはできません。 しかし、モバイルゲームがTencentのWechat / QQでどのように成功を収めることができるかを示す好例です。



欧米では、モバイルゲームのバイラリティは、Facebookを介して接続するプレイヤーに大きく依存しています。 上で書いたように、中国の同様のプラットフォームはWechat / QQです。 また、友人の輪に大きく依存しているすべてのゲームはTencentと密接に連携する必要があります。Facebookとは異なり、Tencentプラットフォームは開発者に開かれていないためです。







中国でのナンバー3マッチ3ゲームは宾果消消乐（カラフルフルーツマッチ）であり、ナンバー1よりもはるかに遅れています。 しかし、3番目は中国ではありません。



Candy Crush SagaとCookie Jamは長い間中国市場に参入しましたが、リーダーボードにはありません。



ランナー







最も人気のある2DランナーゲームはTencentの天天酷Every（Everyday Cool Run）です。 彼女のゲームプレイは悪くありませんが、社会的機能と装備とペットの改善システムにより、ゲームは非常に混乱します。 ランナーゲームは、いつでもプレイを開始および停止できるので、十分にシンプルでなければなりません。







最も人気のある2人の3DランナーはSubway SurfersとTemple Run 2です。これらは優れており、これらのゲームには柔軟なコントロールがあり、新しいコンテンツが定期的に追加されます。 他の中国の3Dランナーは山ほどありますが、リーダーボードには場所が見つかりませんでした。中国の開発者は通常、アクティブなゲームに必要なニュアンスをあまりよく把握していません。



クラッシュ・オブ・クランのようなゲーム







中国では、ご想像のとおり、このタイプのゲームの模倣品やクローンが他の世界よりはるかに多くあります。 ただし、クラッシュ・オブ・クランは依然として中国で第1位です。 そして彼女の成功により、クラッシュ・ロワイヤルのリーダーボードに非常に近づいた。



マインクラフト

収集可能なカードゲーム
