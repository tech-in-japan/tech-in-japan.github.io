近未来のインターフェース

気づいた？ 私たちの目の前で、再び、真の革命が起こっています。 いいえ、いいえ、これはあなたが考えたことではありません-新年の方針はありません！ インターフェイスの分野での革命について話している。 2010年を通してその兆候のいくつかが観察されたという事実にもかかわらず、全体像は誰にとっても明らかではないと思います。







この記事は、インターフェースのトレンドの簡単な概要です。その中で、私たちはすぐに素晴らしい未来に出会えることを確信させたいと思います。



決まり文句から始めて、最もおいしいものを後回しにしましょう。



マルチタッチ



私たちはすでにこのフレーズに慣れています。 マルチタッチは 、iPhone、一部のラップトップのタッチパッドおよびタッチスクリーン、 Microsoft SurfaceおよびTouchTableタッチテーブルに既に搭載されています。一般に、すでに完全に機能する商用化された技術です。 マルチタッチをサポートするデバイスですが、誰もがそれを持っているわけではありませんが、オタクにとってはこの技術は携帯電話と同じくらい一般的になっています。



世界中でこのインターフェースの勝利の行列はわずか5年前に始まったことを思い出したいだけです（ 80年代に発明されましたが）。 Googleトレンドのマルチタッチのクエリ統計をご覧ください。







実際、このインターフェイスからおよそ、その回転を（条件付きで）カウントすることができます。これについては後で説明します。





立体画像と3D画像



午後の100年間、フラットイメージを「測定」しようとします。 しかし、過去2年間、この分野でいくつかのリバイバルがありました。 過去数年にわたって、電子機器メーカーは非常に多くの自動立体視ディスプレイのモデル（つまり、メガネを必要としないモデル）を作成してきたため、別の大規模なレビューに専念することができました。 そして、はい、あなたはすでに予見可能な価格で自宅で3D映画館を装備することができます。



このレビューの文脈では、3D視覚化テクノロジはそれ自体ではなく、以下で説明する他のヒューマンマシンインターフェイスとの組み合わせで興味深いものです。 Appleの特許を覚えていますか？



拡張現実



拡張現実技術はHabrausersにとっても新しいものではありません。それに関するトピックはHabréで定期的に表示されます。



拡張現実が私たちの日常生活にしっかりと入ることはほぼ間違いありませんが、人々はこのテクノロジーの可能性をわずか2年前に認識し始めました。 そのおかげで、仮想現実と物質的な世界がまもなく統合され、「オフライン」と「オンライン」の概念が無意味で見分けがつかなくなる可能性があります。 結局のところ、Dennou Coilシリーズのanimeshnikovになじみのある眼鏡のプロトタイプはすでに作成されています。



次に、再びGoogleトレンドに戻り、マルチタッチの場合と同様に、この分野での最初の進歩が約5年前に始まったことを確認します。 このテクノロジーは比較的広く人気があり、それよりもさらに少ない-2年前です。







ジェスチャアナライザー



2010年6月、Microsoft は Kinectテクノロジーを世界に紹介しました。 実際、モーション認識に基づくインタラクティブシステムは少し前に登場し、 3D視覚化ツールと統合されたものもありました。 しかし、Kinectはインターフェースの時代に2つの理由で新しいページを開きました。 第一に、多数のセンサーと有能なアルゴリズムのおかげで、彼らは見事な精度と認識の汎用性を達成することができました。 そして次に、PCの互換性とSDKの存在のおかげで、開発者のコ​​ミュニティがすぐにテクノロジーを中心に形成され、 libfreenectプロジェクトが登場しました 。その目的は、最も一般的なプラットフォーム用のKinectのドライバーを開発することです。







Kinectに加えて、ジェスチャ認識に関連する別のプロジェクトがあります。これは、息を切らして、世界中のオタクによって監視されています。 そして、このプロジェクトは...と呼ばれています



... SixthSense







SixthSenseは、ボストンで開催されたComputer Human Interfaces 2009カンファレンスで初めて発表されました。その後、BBCニュースはテクノロジーについての短い記事を書きましたが、一般の人は気づきませんでした。 しかし、 TEDの講演に対するPranava Mystryのエキサイティングなパフォーマンスは多くの人に見られ、そのリンクはHabréにありました 。



Sixthsenseテクノロジーは、モバイルコンピューター、カメラ、ハンドヘルドプロジェクター、ミラー、ヘッドフォン、ある種の洗濯ばさみ、指のカラフルなパッチなど、多数のデバイスの組み合わせを一度に使用します。 そしてもちろん、それはsoftwareなソフトウェアなしではありませんでした。最も興味深いのはその中にあります。 その結果、現実の世界とサイバー空間は信じられないほどの方法で相互に作用します。 手でメモを書いて「デスクトップ」に貼り付けるか、実際の紙の文書を指でコンピューターにドラッグするか、普通の紙で車に乗せます。



ただし、これは単なるデモンストレーションであり、実際の技術を実装する前に、おそらく数千、または数百万行のコードを記述する必要があります。 MIT Media Labの代表者は、開発者コミュニティ向けにSixthSenseコードへのアクセスをオープンすることを約束しましたが、これまでのところこれは実現していません。 それでも、このテクノロジーの実装に技術的な障害はないようであり、Kinectと組み合わせると、一般的に奇跡が起こります。 少なくとも、指に色付きの電気テープを常に着用する必要はありません。 また、3D視覚化テクノロジーもここに追加すると？ うーん...しかし、ポケット3Dプロジェクターはまだ存在しないので、私はすでにそれを夢見ていた。







信じがたいことですが、Sixth Senseは、インターフェース開発者が21世紀の最初の10年間に（少なくとも私の主観的な観点から）私たちのために作成した最も興味深いものではありません。 それでは先に進みましょう。



ニューロコンピューターインターフェイス



ニューロコンピューターインターフェース 、またはBCI（脳コンピューターインターフェース）は、いわば脳からコンピューターに直接入力を提供するシステムであり、ユーザーの手やその他の不要な周辺機器をバイパスします。 簡単に言えば、思考の力を制御します。 私の知る限り、この種の最も精巧で手頃な価格のデバイスはEmotiv EPOCです （そうでない場合は修正してください）。



HabreでのEmotiv EPOCの最新の言及は2008年に遡ります 。 その後、デバイスの販売が2009年に延期されたことに遺withの意を表明しました。 それ以来、HabréでのEmotiv EPOCについては何も聞かれませんでした。 一方、デバイスはすでに販売されています。 確かに、これまでアメリカだけでしたが、いつロシアのオタクを止めましたか？ ;）さらに、このものには開発者のコ​​ミュニティ （かなり条件付きではあります）と、軽量の無料バージョンから7.5キロバックの豪華なEnterprise Plusまでの6つのSDKオプションがあります 。 残念ながら、実際のデバイスでデバッグし、ドキュメントにアクセスしてAPIにアクセスする場合、通常のユーザーが提供する300ドルではなく、 Developer Editionの場合は500ドルを支払う必要があります。 しかし、これは宇宙のお金でもありません。将来のインターフェイスを選択し、将来の世代のために彼の偉業を説明する機会を利用する勇敢な実験者がHabréにいることを願っています。



Emotiv EPOCは、BCIを使用する唯一の市販のデバイスではありません。 少なくともg-tecのGamma SysとOCZ TechnologyのNeural Impulse Actuatorがあります。



Gamma Sysは、通常のユーザーよりも研究機関に焦点を当てているようです-メーカーのウェブサイトでは、価格やディストリビューターへのリンクは見つかりませんでした。明らかに、技術的および商業的オファーなどのリクエストを送信することが提案されています



Neural Impulse Actuator（NIA）は、エンドユーザーにとってより魅力的に見えます-100ドルで購入でき、ロシア語のマニュアルもあります。 彼にはSDKがありません。各ゲームの公式サポートは製造元から期待される必要があります（Windowsのみ）。 確かな知識を持つ人々は、NIAは普通のHIDデバイスであり、 自分で fireを書くことができることに 気づきました。 一方で、これはプラスに思えますが、一方で、私が理解するように、このアーキテクチャは、高度なマウスだけでなく、脳のさまざまな部分の活動を測定するための完全なデバイスであるEPOCと比較して、デバイスの機能を大幅に制限します。







使いやすさ



CCC-COMBO BREAKER！ そして今、私はいくつかの言葉を言いたいと思います。一見したところ、いくつかは一般的なキャンバスから抜け落ちます。 この記事は、インターフェイスの分野における革命についての言葉から始まったことを思い出させてください。この兆候は、過去5年間ほど観察されてきました。 上記からわかるように、科学者とエンジニアはデジタルデバイスのユーザーエクスペリエンスを改善するために素晴らしい仕事をしました。 以前に書いたものはすべて、主に新しい周辺機器またはそれらと連携する方法に関連しています。 次に、ソフトウェアユーザビリティなどの概念を見てみましょう。







このトピックも最近人気が出ています。 そして、彼女への関心が高まると感じています。 これまでに、ほとんどすべてのユーザーのニーズをカバーする膨大な量のソフトウェアが作成されました。 そして今、彼らの種類の間でうまく競争するために、プログラムはシンプルで便利で美的なユーザーインターフェースを持つことを強いられています。



そして今、私はテクノロジーに戻り、一般的な結論を導き出します。



おわりに



将来の到来には十分な物理デバイスがあり、それらの使用を簡単で便利にするソフトウェアだけでは十分ではないようです。 もちろん、これも骨の折れる仕事であり、おそらくオペレーティングシステムの開発に匹敵します（ ReactOSは何年間書かれていますか？）。 しかし、この記事を読んでいる人の中には、これらのデバイスをすべて統合し、21世紀のコンピューターインターフェイスを作成するためのオープンソースプロジェクトを組織している人がいるかもしれません。



結論として、少し空想しましょう。 想像してみてください。あなたは椅子に座っていますが、目の前にコンピューターはありません（正確には、見えません）。 代わりに、テーブル（Ikeaの通常のテーブル）に仮想オブジェクト（テキストドキュメント、ゲームウィンドウ、3次元モデル）を表示し、手でテーブル内を移動できます。 または、テーブルの上の空中に吊るします。 ポップアップウィンドウが表示され、閉じます。思考の力でそれを削除しました。 手で机の引き出しを開くと、そこからHabrのあるブラウザーが飛び出します。 ページは読みながらスクロールします。



ジプシーの漫画を作った人を覚えていますか？ 2番目（まあ、3番目）、彼らは6か月で自分の漫画を作成します。 比較：3Dエディター、オーディオエディター、および編集ソフトウェアの代わりに、1つの思考グラバーがあります。 3Dモデルをレンダリングする必要さえありません。 彼らはただ座って漫画を詳細に想像します。 もちろん、独自の微妙さと独自の技術プロセスもあります...しかし、パフォーマンスはまだずっと高くなっています。



今日、そのような空想は遠いものとは無関係に思えます。 しかし、あなたを除いてあなたの周りの誰もがこれで驚くべきことを見ることなく携帯電話とインターネットを使用していることを発見したら、覚えておいてください？ 未来はそれです。 気づかれません。
