立ちながら世界を称賛する方法。 スタートアップを人気にする

世界はコンサートホールであり、新しく立ち上げられたスタートアップは2秒前に終了したショーであり、観客は立っている間に拍手し始めるか、ホールのさまざまなコーナーでゆったりとしたポップスで降りようとしていると想像してください。 本当に洗練されたパフォーマンスが、目の肥えた聴衆に、ときどき控えめで冷たく、時には平凡なものが「ブラボー」と叫ぶのに出会うのはなぜですか？ スタートアップを高く評価し、さらに良くする方法は？ 立ちながら世界を称賛する方法は？







ミシガン大学のスコット・E・ペイジ教授は、コースラのモデル思考コースで知られ、2004年にジョン・H・ミラーと共同で、スタンディング・オベーションと呼ばれるエレガントなモデルを提案しました。これは、セルオートマトンを使用して、観客が立っているときに拍手するかどうかを説明しますいくつかの要因。 この記事では、まずモデルを可能な限り簡略化して説明し、次に簡単な数学的および数学に近い計算、セルオートマトンの構築、そして最終的にはスタートアップやその他の生活領域に関する結論に進みます。



カーテンを下げるエキサイティングな瞬間に一瞬、講堂にいる自分を想像してください。 何が立ち上がって拍手し始めることができますか？ 最初に、もちろん、パフォーマンスを気に入るはずです。 第二に、あなたが立っている間拍手する観客の中で唯一の人である場合（そしてあなたの周りのすべての人が立ち上がって座っている場合）、あなたはおそらく恥ずかしいと感じるでしょう。 したがって、聴衆の一部がすでに立ち上がることを決めていることが必要です。 第三に、あなたが友人や家族と一緒に遊びに来て、そのうちの一人が立ち上がったなら、おそらく彼をサポートしたいと思うでしょう。あなたも立ち上がるでしょう。 第四に、ホール全体が見えない。 居住する場所によっては、屋台の最初の列、ホールの半分のみが見える場合があります。最初の列に座っている場合は、隣人以外は誰も見えません。 したがって、まず第一に、あなたに見えるホールの部分だけがあなたの決定に影響します。



したがって、立ち上がるかどうかに影響を与える4つの要因を特定できます。

  1. ショーの印象。
  2. すでに立ち上がることを決めた聴衆の割合。
  3. 行の隣人の行動。
  4. 最前線に座っている「星」の行動。




オートマトンの小さな理論



パフォーマンスの印象は個人ごとに異なりますが、一般的な場合、パフォーマンスの質に直接依存します。 より良い脚本、演技、セット、演出-より良い印象。 パフォーマンスの印象は式として表すことができます ここで：

i （印象）-パフォーマンスの印象。

Q （品質）-パフォーマンスの品質;

e （エラー）-個々の知覚エラー。



あなたのiがT （しきい値はあなたが上昇するしきい値です）よりも高い場合、あなたの足に上昇します。



また、パラメーターXを導入します。これは、立っている間に拍手し始めなければならない目に見える聴衆の割合です。そのため、立ち上がって気まずくならないようにします。 このパラメーターも個別です。一部の人々は、あらゆる傾向をサポートする準備ができていますが、他の人々は、社会の行動にほとんど何の意味も付けません。 観客スタンディングオベーションの割合は、 Nで示されます。



しかし、これはすべてのパラメーターではありません。 重要な役割は、あなたが一緒に来たグループの誰かによっても演じられます（F-友達と家族を表します）。



視聴者が立ち上がるか椅子にとどまるかを決定するアルゴリズムは次のようになります。

  1. もし 立ち上がる。
  2. もし 立ち上がる。
  3. もし 立ち上がる。
  4. そうでなければ、座ってください。

ホール全体ではなく、ホールの一部しか見えないことを考えると、最も重要なことは、最前列に座っている人々のパフォーマンスに関する意見です。

グループの1人のメンバーが立ち上がると、他のすべてのメンバーがフォローすることが重要です。 観客がパフォーマンスに参加するグループが大きければ大きいほど、スタンディングオベーションになりそうです。



セルラーマシンで遊ぶことができます。ブラウザで入力パラメーターが変更されたときに何が起こるかを確認できます（ Javascriptの実装を作成しました ）。



まとめると。 スタンディングオベーションの可能性を最大化するには、次のことを行う必要があります。

  1. まず、質の高いパフォーマンスを提供します。
  2. 意見の幅を広げるには、パフォーマンスを物議を醸す衝撃的なものにするか、ショーに多様な視聴者を招待します。 また、ショーを好きになる人を正確に知っている場合は、招待することをお勧めします。
  3. ショーに最前列をお楽しみください。
  4. 家族、友人のグループ、または同僚を劇に招待します。
  5. 感心しやすい経験の浅い視聴者を招待します。
  6. 社会に関係なく、自分で何かをすることができる少数の人々と、主に他者の意見によって導かれる人々を招待します。




これはどういう意味ですか？



スタートアップのプロモーションに関しては、これは次のことを意味します。

  1. もちろん、それを良くします。
  2. 広く多様な聴衆に見せ、そして何よりもまず「ターゲット」。
  3. 大統領、俳優、科学者をユーザーとして社会の最前線に引き寄せる。
  4. ユーザーに友人、親relative、同僚を招待するように動機付けます。
  5. アナログを見たことがない（存在する場合）未経験の大衆にプロジェクトを見せてください。
  6. そして、もちろん、それを良くするために。




当然、このモデルは、新興企業だけでなく、社会の各メンバーが二者択一の判断を下す必要がある状況（yes / no）に外挿することもできます。喫煙を開始する必要がありますか？ 大学に行くべきですか？ 家族を始めるべきですか？ 車を買うべきですか？ 犬を飼うべきですか？ スポーツに参加しますか？ プログラミングを開始する必要がありますか？
