ジャンプスタート5月20日：Windows Server 2012 R2およびSystem Center 2012 R2によるデータセンターの仮想化。 仮想マシンとサービスの展開

Windows Server 2012 R2およびSystem Center 2012 R2を使用したデータセンター仮想化に特化した一連のオンラインイベントを提供しています。 これらのオンラインイベントの一環として、物理および仮想データセンターインフラストラクチャの展開、管理、および保守に使用されるWindows ServerおよびSystem Centerの主要なテクノロジについて詳しく説明します。 Windows Server 2008 / R2でHyper-Vを使用している場合、Windows Server 2012 / R2のリリースでMicrosoftハイパーバイザーで発生した膨大な量の変更が表示されます。 組織でVMwareを使用している場合、Hyper-Vの最新バージョンの機能を評価したり、Windows ServerおよびSystem CenterテクノロジーをVMware vCenterおよびvCloud Suiteと比較したりできます。





シリーズの3回目の最終オンラインイベントである5月20日に、一般的な運用原則から仮想化ネットワークのセットアップとHNVゲートウェイの構成に至るまで、Hyper-Vネットワーク仮想化（HNV）テクノロジーを詳しく見ていきます。

さらに、System Center 2012 R2 Virtual Machine Managerでプライベートクラウドを作成する手順と、作成されたクラウドのリソースをクォータに割り当てる方法について説明します。 App Controllerベースのセルフサービスポータルを使用して、プライベートクラウドリソースを活用する方法がわかります。 アプリケーションを展開および更新するためのサービステンプレート（サービステンプレート）とそのアプリケーションの構造を詳細に分析します。 最後に、一部のオブジェクトとサービスがローカルネットワークにあり、一部がMicrosoft Azureのパブリッククラウドにある場合、ハイブリッド構造を管理するためのSystem Center 2012 R2の機能に精通します。
