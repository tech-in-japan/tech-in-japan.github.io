Thinstuff Summer of Code 2011

FreeRDPは Google Summer of Code 2011で受け入れられませんでしたが、本当に必要でした。 Freestuffのスポンサーの1つであるThinstuffは、同様のイベント、Thinstuff Summer of Code 2011を開催するのは素晴らしいことだと判断しました。



GSoCとは異なり、TSoCは学生だけでなくすべての人に開かれています。 実装する必要のある多くの機能があり、開発者にお金を払うことはできます。 このイベントは、FreeRDP作成者の開発に関心のある企業と学生またはコンサルタントとの相互作用を目的としており、FreeRDPの有料作品に関心があります。 このイベントは正式にThinstuffによってホストされていますが、他の企業がスポンサーとして参加するだけでなく、スポンサーになることもあります。



スポンサーを待っているタスクの詳細については、 Thinstuff Summer of Code 2011のWikiページをご覧ください 。 FreeRDP開発のスポンサーに関心のある企業は、連絡先情報とともに独自のタスクをそこに追加できます。



幸運をお祈りし、Thinstuff Summer of Code 2011をお楽しみください:)
