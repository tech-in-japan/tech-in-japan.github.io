PcDuino3 Nano、Raspberry Pi B +の真の競争相手、39ドル

Raspberry Piは、市場の既存のデバッグボードとは異なり、非常に低価格で優れた機能を備えています。 しかし、残りのプレイヤーは眠っておらず、徐々に機能を増やし、コストを削減しています。

9月末、 LinkspriteはArduino互換プラットフォームを備えたデュアルコアAllWinner A20 SoC上に構築されたデバッグボードであるPcDuino3 Nanoを発表しました。 これにより、Arduino UNOのようにArduino拡張カードをPcDuino3 Nanoに接続できます。 さらに、HDDまたはSSDを接続するためのSATAコネクタがあります。

PcDuino3 Nano。これは、Wi-Fiモジュール、LVDS、およびI2CインターフェイスなしのPcDuino3の小型で安価なバージョンです。 ボードはLi-Poバッテリーの接続をサポートしていませんが、古いモデルと比較して別のUSBポートが追加されています。

PCDuino3 Nano





PcDuino3の仕様

  • プロセッサー：SoC AllWinner A20ARM®Cortex™-A7 Dual-Core（GPU）ARM®Mali400MP2 GPU、OpenGL ES 2.0 / 1.1に準拠
  • RAM：1GB DDR3
  • ユーザーメモリ：4GB NANDメモリ、SATAコネクタ、およびmicroSDスロット（最大32GB）
  • HDCPをサポートするHDMI 1.4ビデオ
  • オーディオ：3.5mmアナログインターフェイス
  • ネットワーク：10/100/1000イーサネット
  • USB：2つのUSB 2.0 HOSTポート、および1つのUSB 2.0 OTGポート
  • コネクタ：14xGPIO、2xPWM、6xADC、1xUART、1xSPI、1xI2Cと互換性のあるArduino UNO。
  • カメラ：MIPIカメラのサポート
  • その他：IRセンサー
  • 電源：2Aで5V DC
  • サイズ：91.4mm x 53.3mm（PcDuino3：121mm x 65mm）


PcDuino3 NanoはCubieboard2と似ていますが、フォームファクターは小さく、Arduinoに近く、ギガビットイーサネットインターフェイスを備えていますが、オーディオジャックはありません。 Ubuntu 12.04とAndroid 4.2のサポートが提供され、既製のファームウェアとソースコードが利用可能、I / Oコンタクトを制御するAPIのサポートが提供され、Arduino（UART、ADC、PWM、GPIO、I2C、SPI）でも利用可能です

Mali-400 GPUのおかげで、SoCは1080p 60fpsビデオを再生します。 ArduinoスタイルのプログラミングはpcDuinoのScratchで行われます。 プログラミング言語C / C ++、Python、およびJavaがサポートされています。

PcDuino3の注文は、 LinkspriteまたはCutedigiで39ドルで注文できます。送料は含まれていません。

Linkspriteには、ロシアへの6.75ドルのUSPSファーストクラス配送オプションがあります。 それに比べて、Raspberry Pi B +は現在37.80ドルで販売されています。 フル機能のpcDuino3の価格は77ドルです。 LVDSインターフェイス、デジタルおよびアナログオーディオ出力、WiFiモジュールが含まれています。
