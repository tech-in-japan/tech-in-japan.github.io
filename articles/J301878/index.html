親愛なる小さなスプール：VMware Host Clientの概要

最近、VMwareはvSphere 6.0 Update 2のリリースと同時にVMware Host Client 1.0のリリースを発表しました。VMwareHost Client 1.0を使用すると、 HTML 5ブラウザーを使用して個々のESXiホストを管理できます。 Windows、OS X、Android、またはHTML 5ブラウザーをサポートする他のOSのすべてのOSで使用できますこのリリースより前では、ESXiホストにアクセスするには、Windowsで「ネイティブ」vSphereクライアントを起動する必要がありました。 クライアントのインストールプロセスと主な機能の概要を提供します。







まず、ESXiホストのIPアドレスを Windowsで実行されているブラウザーに入力し、「VMware Host Clientを開く」をクリックしました。 標準認証画面が表示されます。









ESXiホストやコンピューター自体に何かをインストールする必要はありません。 承認後、ホストがvCenterサーバーによって制御されているという警告が表示され、ESXiホストパネルが表示されます。









デスクトップには、サーバーの管理、サーバーのパフォーマンスの監視、サーバーの無効化または再起動など、期待されるすべての機能があります。







画面の左側にあるアイコンを使用して、さらに進んで、ESXiホスト上の仮想マシン、ストレージ、およびネットワークを確認できます。









これらの可能性のそれぞれについて簡単に考えてみましょう。 [仮想マシン]アイコンを選択すると、標準の管理機能を実行できます。









ただし、仮想マシンのクローン作成、vMotionの使用、およびvCenterサーバーを必要とするその他の操作も不可能です。 仮想マシンコンソールを開くこともでき、 Webブラウザからアクセスできます。







ESXiサーバーの最も一般的なタスクの1つは、仮想マシンの作成とインポートです。 VMware Host Client 1.0を使用すると、仮想マシンを最初から作成し、OVFまたはOVAファイルから展開して、既存の仮想マシンを登録できます。







コンピューター上のOVAファイルを使用してホストにDamn Small Linux（DSL ）の新しいインスタンスをデプロイすることにより、この機能をテストすることにしました。 インストールウィザードを使用すると、プロセスは簡単でわかりやすくなりました。









DSL OVAファイルをインストーラーにアップロードし、ストレージとネットワークの詳細を入力して、「完了」をクリックしました。 1分足らずで、実行中のDSLインスタンスの準備が整いました。 これ以上簡単なことは想像できません。







ストレージ機能は、リポジトリの表示やリポジトリ内のファイルの検査など、リポジトリに関連するほとんどの標準機能をサポートしています。









ホストクライアントにはパフォーマンス監視機能があり、ホストのCPU、メモリ、ネットワークアクティビティに関するグラフを表示することもできます。









監視機能を使用すると、パネルを表示して、ホストに関連するイベント、タスク、ログ、および通知を追跡および分析できます。







さまざまなデバイスからホストクライアントを使用してESXiサーバーに接続することが可能です（Samsung Galaxy S5スマートフォン、MacBook、Samsungタブレットから試しました。もちろん、スマートフォンとタブレットの画面サイズは小さいですが、一般的にこれは機能に影響しません：画面をスクロールします）必要な情報を見るのはそれほど難しくありません。









毎回、VMwareは生活を少し楽にする製品をリリースし、ホストクライアントは間違いなくそのような製品のリストに載っています。 間違いなくそれらの製品の1つです。 VMware Host Client 1.0がリリースされる前は、ホストにすばやくアクセスするには、独自のクライアントを事前にインストールしてWindowsが必要でした。 完全に便利というわけではありません。 VMware Host Clientの存在により、任意のデバイスからホストにアクセス、監視、管理できます。 その管理機能により、仮想マシンのネットワーク、ストレージ、ライフサイクルの管理、ESXiホストのパフォーマンスの監視、イベントとタスクの表示が可能になります。 ESXiホストに追加のものをインストールする必要はありません。 これは、仮想化の分野でもう少し便利になる便利なツールです。
