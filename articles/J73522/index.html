Zend Studio 5.5およびWindows 7

Eclipseに切り替えることはできませんでした。 しかし、7000のリリース以降、Windows 7に切り替えました。これには完全に満足しています。 かつて、私は本当に私を悩ませていた問題に出くわしました。ZendStudio 5.5のバージョンは、頑固にW7の下で正しく動作したくありませんでした。 その理由は表面にありました。問題はJREバージョンの不正確な連続性でした。







以下では、Zend Studio 5.5が動作するために実行する必要がある一連のアクションについて説明します。 おそらくこれは誰かに役立つでしょう。 これは私の最初のhabrapostであるため、個人のブログに投稿しました。他の場所にプッシュするのに十分なカルマがありません。 :(





1. Windows用の新しいJREをダウンロードしてインストールします。 リンク



2. WinRARを使用してZend Studioインストーラーファイルを解凍します。



3. Windows \ resourceサブディレクトリで、jreディレクトリの名前をjre backupに変更します。



4. C：\ Program Files（x86）\ Java \ jre6をリソースフォルダーにコピーし、「jre6」の名前を「jre」に変更します。



5. Zend Studioインストーラーを実行し、プログラムをインストールします。 すでにガット化したアーカイブにインストールするEXEファイルは、ZendStudio-5_5_1.exeと呼ばれます。



6.インストール後、「jreバックアップ」をC：\ Program Files（x86）\ Javaにコピーし、名前をjre6に変更します。



7.インターフェイスの表示にまだ問題がある場合は、ショートカットのプロパティで、Vistaとの互換モードを指定します。



すべてが正常に機能するはずです。



これが誰かを助けてくれたら嬉しいです。
