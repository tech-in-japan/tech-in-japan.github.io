シスコを選ぶ理由 この章では、シスコセールスコースを開始します。 私は短いエッセイをそう呼んでいます。



なぜシスコなのか？ この5文字の魔法の言葉には何がありますか？ 本当に良いものはありませんか？

管理者、販売者、インテグレーターに聞いてください-誰もが、シスコのソリューションよりもシンプルで、安価で、優れた、生産性の高いソリューションが存在することを知っているという事実で輝きます...

それはこの素晴らしい決断の左への一歩、右への一歩に過ぎず、代替品を見つけることははるかに難しくなります...

シスコソリューションの強みを簡単に説明します。



届く シスコのソリューションには、このパラメーターに競合他社はありません。 SOHO市場からプロバイダーソリューション、小規模ながら機能的なルーター、大企業のネットワーク管理システムまで、すべてのネットワークソリューションの最も広い範囲。



主な方向：

  1. 多機能ルーター
  2. 強力でスマートなスイッチ
  3. アクティブ保護デバイス-ASA、ACE
  4. 侵入防止システム-IPS
  5. ワイヤレスセントラルアクセスシステム
  6. ユニファイドコミュニケーション（VoIP、ビデオ会議、テレプレゼンス、通話管理システム）
  7. 中央ホスト保護システム（CSA）
  8. アクセス制御システム（NAC）
  9. そして...あなたが考えることができる他のすべて:)




主なことは、シスコがそこに留まらず、「風に鼻を向ける」ことで、有望な分野にお金を投資し、たとえば有望な開発を購入してその決定に統合するなど、新しい分野を考え出します。 したがって、このような巨大なサポートを受けた興味深いソリューションには、「人々」に侵入するチャンスがあります。



信頼性 すべてが壊れます、唯一の質問はいつですか。 信頼性ciscoは、長年の運用実績によってチェックされます。 私は嘘をつきません-そして、シスコはオペレーティングシステムのリリースに失敗し、リリースに失敗しましたが、全体的なフォールトトレランスは疑いの余地がありません。



柔軟性。 同じハードウェアは、オペレーティングシステムとモジュールの詰め込みによって、セキュリティ、ユニファイドコミュニケーションゲートウェイ、サービスなど、まったく異なる機能を実行できます。つまり、何か新しいものが必要な場合は、何も買わずにダイヤルするだけの大きなチャンスがあることを意味します。いくつかのチーム。



相互依存。 曲がった言葉ですが、それは本質を反映しています。 異なる機能を実行する異なる鉄片は、互いに依存し、互いに制御することができます。 これにより、異なるデバイスのセットではなく、ネットワークを生物にできます。



デバッグ 構成とカスタマイザーにとって非常に重要です。ほとんどすべてのシスコデバイスに組み込まれている最も広範なトラブルシューティング機能。



知性。 高価な鉄だけを売るのは難しいです。 アイデアと機会を売る必要がある。 すべてのシスコデバイスには、ネットワークの機能を拡張するために、標準および独自の幅広いテクノロジー、プロトコル、イデオロギーが含まれています。



パフォーマンス。 シスコは多くの市場セグメントのリーダーであり、この高いランクを満たさなければなりません。 したがって、CRSなどのユニークなソリューションが表示されます（このような鉄の1つで、たとえばイギリス全土の通信を提供できます！）。 現在、ファイアウォールのセグメント、ルーターのセグメント、および侵入防止システムのセグメントに10ギガビットインターフェースを備えたトップソリューションがあります...



キリル

聴衆からのレプリカ：

完全に実装されたCRSには、それぞれ40ギガビット/秒の1000を超えるラインカードが含まれ、理論的にはマルチシャーシ構成を介して92 Tb / sの帯域幅に拡張できますが、そのようなサイズのマルチシャーシシステムは決して公開も公開もされていませんでした。 2009年の時点で、最大の実稼働CRS-1システムは8ラインカードシャーシに制限されており、合計で10Tbpsです。

デバイスの開発中は、HFRまたはHuge Fucking Routerというコード名で知られていました。





集中化 シスコのデバイスは個別に管理できませんが、強力なシステム、たとえばcisco security managerを使用しています。 また、あらゆる種類の統計を一元的に収集して分析することもできます-MARS。 誰も会計を集中化するための同様の解決策を提案していません。



もちろん、あるメーカーのソリューションがあらゆる場所で最高のものになることはありません。 たとえば、小規模オフィスのお気に入りのタスク-速度とクォータをユーザーに制限する-は、Ciscoで高価に解決されます。 「なぜ、最も簡単な無料のプロキシがこれを行うことができます！」細心の読者を称賛し、正しいでしょう。 これは、「はるかに優れたソリューションがある」一例です。



さらに、私は扇動を言います：誰もがシスコのソリューションを必要とするわけではありません！ シスコがコストを大幅に引き上げる最も単純なソリューションに対して過払いをするのはなぜですか？ 使用しないのに、なぜ追加機能を支払うのですか？ 考える必要があるのは、これらの「超過」が必要な場合はどうでしょうか。



まあ、典型的な誤解のリストの最後に。 むしろ、それは一つのことです： シスコは非常に高価です！

これは誤りです。 そして、それは顧客が彼のために設計されていないソリューションを提供されるという事実に由来します！ シスコの会社には、ソリューションの選択を可能にする一連のリソース（ AntiCiscoで明示的に収集したリソースの一部）があります。 しかし、キャッチは、国内の「中規模」企業が原則としてSMB（中小企業）の低い範囲に落ちるということです。 そして、彼らはそれらを提供されており、年間売上高は$ 5万、ソリューションは$ 25,000 ...



Cyrill ：ところで、筆者の許可を得て、「 Modern Office Infrastructure-2 」の記事でいくつかのソリューションを使用しました。





たとえば、私は比較をします。顧客には、10 Mbpsのインターネット、それぞれ2メガビットのショップがある10の暗号化されたトンネルが必要です。 正式には、顧客は以下を提供する必要があります。

  1. ASA。 在庫あり-ASA5510、延長ライセンス（5000ドル）
  2. ルーターはプロバイダーに接続します（顧客が2つのプロバイダーを必要とするため）。 ISRの方が優れていることは明らかです。 マージンあり-2811（$ 3500）
  3. ギガビット（$ 6000）で自然にL2 / L3、3560シリーズ内のスイッチ
  4. まあ、セットアップ-最小15％。


合計で、なんらかの割引があり、約12〜13,000ドルです。

また、顧客は2を最大3千ドル使う予定です。 顧客は何と言いますか？ そして、彼はそれを安価で買いに行きます。 私たちはブランドに名前を付けません:)彼は満足します：彼は問題を保存して解決しました。 今。 そして、もしあれば、彼は拡張機能について考えます。



しかし、タスクを詳しく見て、お勧めの写真を愚かに再描画しなくても十分でした！

解決策は次のとおりです。

  1. ASA 5505、無制限のユーザー、およびライセンス-1700ドル
  2. 2xプロバイダーが必要な場合、またはイーサネット経由ではない接続が必要な場合-ISR 18xx（$ 1,500）
  3. VLAN、ASAへのトランク、およびルーティングが可能な場合は、スイッチを古いままにします。 できない場合-Catalyst Express-700ドル
  4. 15％を設定


合計：仕事を含む割引付きの最大3000ドル。 また、後で設定して新しい機能を取得できる一連の追加のアメニティ。



一般的に、私は力で誰かを引きずりません。 私の目標はシンプルです-合理的でバランスの取れた選択を支援することです:)
