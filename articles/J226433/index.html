    $foo()['bar'](); $foo::$bar::$baz; foo()(); $foo->bar()()



    $foo()['bar'](); $foo::$bar::$baz; foo()(); $foo->bar()()



    、および括弧内の式を使用した演算、たとえば(function() { ... })()



    など。
  • RFC：Bare Name Array Literal -JavaScriptでオブジェクトを宣言するのと同様に、キーと配列値を宣言する際に引用符をオプションにすることを提案します。
  • RFC：Bare Name Array Dereference-前の文に続いて、この文では、著者はネストされた配列の要素にアクセスする構文を削減したいので、 $myArray['foobar']['andSoOn']['moreKeys']
