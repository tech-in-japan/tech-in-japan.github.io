Windows内部デバイス

出版社のもう1つの目新しさをご紹介します。マークルッシノビッチ、デビッドソロモン、アレックスイオネスクが新版に参加した伝説のWindows Internals本の待望の第6版です。







Windows Internalsブックの第6版では、Windows 7およびWindows Server 2008 R2コンポーネントの内部コアについて説明しています。



この本はあなたを助けます：

  • オブジェクトマネージャからレジストリまで、システムのコアと制御メカニズムがどのように機能するかを理解します。
  • カーネルデバッガーなどのツールを使用した内部データ構造システムの詳細をご覧ください。
  • 内部からWindowsセキュリティモデルを調べて、データへのアクセスを許可する方法を確認します。
  • Windowsでの物理メモリと仮想メモリの管理方法を理解します。
  • ドライバープロトコルAPIやネットワークアダプタードライバーなど、Windowsネットワークスタックを上から下まで調べます。
  • ファイルシステムアクセスの問題とシステムブートの問題を理解する。
  • 障害を分析する方法を学びます。


この本について多くを書くことはおそらく意味がありません-Windows Internalsは、このオペレーティングシステムのカーネルの詳細な研究のための主要な教科書として、Windowsの管理とアプリケーション開発のすべての専門家に非常に馴染みがあります。



残念ながら、歴史的には、Windowsの次のリリースの数年後、原則としてオリジナルエディションのリリースが出ており、OSの新しいバージョンが関連するときに印刷版が出ることもありました。 それで、今回は起こりました。



この出版物のもう1つの革新は、本が初めて2巻に分割され、6か月間の生産量のギャップが生じたことです。 したがって、提示されたロシア語版は、Windows Internalsの第1巻の翻訳版であり、 第2巻は現在、出版社でのリリースの準備中です。
