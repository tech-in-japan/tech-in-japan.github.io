Return関数の修正およびVivaldi 1.0.275.3ビルドのその他の改善

みなさんこんにちは！



今日、最も重要なもののうち、非常に少ない変更があります-リターンとスケーリング機能を使用してバグを修正します。 修正の完全なリストは以下にあります。



既知の問題



ズームのデフォルトのキーボードショートカットは機能しません。



今日は以上です。 以下のリンクを使用して、新しいアセンブリをダウンロードできます。

  • Windows 64ビット （実験的）
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット


変更の完全なリスト：



  • VB-4815画像の読み込み、読み込み前のページアクションの切り替え
  • VB-5340ページアクションポップアップダイアログが開いたままになる
  • VB-9032再読み込み後にページアクション（ページの変更）が適用されない
  • FastForward / Rewind回帰の修正
  • VB-9104アクティブなページをWebパネルとして追加できません
  • VB-7975 [Mac]現在のショートカット構成でメインメニューを更新する
  • VB-8680ビジュアルタブスイッチャーの改善：フォーカスされたタブの強調表示が改善されました。 ページのファビコンを表示しています。 アニメーションを削除しました。 タブプレビューをこれ以上表示しない。 1つのタブのみが表示されている場合、余分な空白はありません。 キーを押したままにしないでください。 タブが1つしかない場合は、タブサイクラを表示しない
  • VB-9068回帰：UIズームが壊れています
  • VB-9076 [短縮ダイヤルフォルダーに親フォルダーを表示]がボタンの上半分でクリックできない
  • VB-7850ブックマークバーにアイテムを追加できない：ユーザーはブックマークまたはブックマークフォルダーを右クリックし、コンテキストメニューから[ブックマークバーフォルダーに移動]を選択してブックマークフォルダーに移動できるようになりました。
  • VB-8899クイックコマンドが多すぎる文字入力でフリーズする
  • VB-9070ボタン上のテキストが垂直方向にずれている（Windows）
  • VB-7921ブックマークバーのブックマークの名前を変更するときにテキストが重複する
  • VB-9000 UIがページの読み込み時に白くなる
  • VB-9059回帰：キーボードが壊れた状態でズーム
  • VB-8585ブックマークされたURLアイコンの表示が機能しない


毎週のビルドはテストであるため、重要なデータをバックアップすることを忘れないでください。 前のアドレスで見つかったエラーを報告します 。
