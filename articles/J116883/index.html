人材を探しています-マイクロソフトパートナープログラムへの春の登録

Habrovskの多くの住民は、技術の専門家向けのMVP（Most Valuable Professional）のステータスを知っていますが、 Microsoftの学生向けのステータスはMSP（Microsoft Student Partner）です。 この春、 4月17日まで、ロシアの学生と大学院生はこの名誉あるタイトルを競う機会があります！









MSPプログラムとは何ですか？



これは、学生（大学院生）が彼の大学の法人の代表者になったときのマイクロソフトとの協力の一形態です。 MSPは世界中から最も才能があり活発な学生（大学院生）であり、技術に情熱を注いでいます。 大学でのエキサイティングな活動に参加し、他の学生に知識を移転する代わりに、MSPは多くの利点を享受するとともに、 実際の経験を積んで履歴書を魅力的にする機会を得ます。



誰を探していますか？



「理想的な候補者」には、時間管理のスキル、イベントの編成、および大学でのイベントに関する優れたコミュニケーションスキルが必要です。 候補者は人前で話す能力を持ち、率先してチームで働くことができなければなりません。 MSPのタスクの不可欠な部分は、オンラインコミュニティの開発（独自のブログの維持、ポッドキャスト、ビデオの記録）です。 ただし、「理想的な候補者」は探していません。 私たちには、 この世界を変えたいと思う、きらめきのある愛好家が必要です！



なぜMSPになるのですか？



プログラムがパートナーの学生に提供するすべての利点に加えて（以下を参照）、このステータスは世界中のどの学生にとっても名誉です。 MSPプログラムは、Microsoftのツールとテクノロジに関するほとんどの情報を取得し、 新しいスキルを習得し 、卒業してIT業界の一員になったときに良いスタートを切る機会を提供します。



ロシアのMSPプログラムの「卒業生」の一部を以下に示します。
  • ミハイル・チェルノモルディコフとコンスタンチン・キチンスキー（マイクロソフトエバンジェリスト）、
  • ダニラ・コルネフ（Googleエバンジェリスト）、
  • コンスタンチンカルギン（モスクワ州立大学の共同研究室の研究者-Microsoft Research）、
  • Sergey Pugachev（MVP Silverlight、Silverlighter.ruコミュニティの創設者）、
  • Vitaliy Kuznetsov（MVP Windows Live）、
  • イリヤ・グレブノフ（2009年イマジンカップ優勝）、
  • Kirill TropinとDmitry Alekseenko（レドモンドのMicrosoft開発者）。

MSPは何を取得しますか？



最初に、新しい学生はMSPボランティアのステータスでプログラムに参加し、以下を受け取ります。

  • ロシア全土の学生パートナーのコミュニティ内で、特別な閉じたポータルでコミュニケーションをとる能力
  • 継続教育のための電子および書籍形式のマイクロソフトテクノロジー情報
  • 大学でのイベント開催のサポート（お土産、資料、書籍）
  • マイクロソフトのカンファレンスに参加する機会と、パートナーの学生専用のイベント
  • MSDNプレミアムサブスクリプション
  • ウェルカムキット （MSPブランドを誇りに思う製品スイート）
  • 技術リソースとトレーニング
  • MVPおよびMicrosoftとの通信
  • 公式リリース前の新製品に関する情報。
  • ロシアのマイクロソフトおよび海外（レドモンドの本社を含む）でのインターンシップおよび求人に関する優先情報。 仕事やインターンシップに応募する際、すべてのMSPにはメリットがあります 。
大学でマイクロソフトのテクノロジープロモーション活動を成功させたプログラムの最もアクティブな参加者は、 MSPステータスと最もアクティブなMSP Guruを受け取ります。



MSPは何をすべきですか？

  • 教育および研究活動において、マイクロソフトの製品とテクノロジについて得た知識を最大限に活用してください。 勉強不足はパートナーシップの終了の機会になるかもしれません。
  • 学習プロセスでのマイクロソフト製品の使用に関して学生と教員を支援するため 。
  • 学生ユーザーグループまたはセミナーを開催します。
  • Microsoftが大学でイベントを開催し、MSDN Academic Allianceプログラムをサポートするのを支援するため。
  • Microsoft Student Partners コミュニティに参加します 。
  • マイクロソフトテクノロジに関する知識を、対面、イベントでの発言、オンラインコミュニティへの参加、およびインターネットでの知識の共有の両方に広めること 。 マイクロソフトのイベント（イマジンカップ、大学でのイベントなど）に参加するように学生を誘います。 この場合、すべてのMSPがプログラムへの参加時に署名する機密情報の非開示に関する契約の条件を遵守する必要があります。
  • Microsoft Academic Program Coordinatorと連絡を取り、イベントについて報告し、時間通りにヘルプとサポートを受けてください。

MSPステータスを取得するには何が必要ですか？

  • ロシアの大学（大学院を含む）でのフルタイムの学習。トレーニングの終了まで少なくとも1〜2年は残ることが非常に望ましい。
  • 少なくとも学業成績が良い 。
  • マイクロソフトのコアテクノロジーを知っているか、できるだけ短時間でそれらを深く知りたいと思っています。
  • マイクロソフトとの仕事に週に少なくとも4時間を費やす機会があります（MSP Guruステータスの学生の場合は週に8時間）。
  • Microsoft と親しくなる 。
  • 組織的な仕事を恐れないでください。
  • ロシア語と英語でのコミュニケーションを恐れないでください。
  • 万が一の場合に備えて、事前にパスポートを取得しておくと便利です。
プログラムへの参加を申請するには、4月17日まで必要です。
