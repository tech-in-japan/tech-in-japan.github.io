Asus Eee PCおよびMSI Windのデスクトップバージョン



Asus Eee PC超予算ノートブック、Asus Eee Box B202デスクトップが販売開始





現在、香港の居住者はEee Box B202を購入できます。 コンピューターの価格は320ドルです。 以前は、Eee Box B202の価格は300ドルを超えないと想定されていました。



1.6ギガヘルツのクロックで動作するIntel AtomプロセッサをベースにしたEee Box B202を構築しました。 RAMの容量は1ギガバイト、ハードドライブ-80ギガバイトに相当します。 コンピューターは、Linuxカーネルに基づいたオペレーティングシステムを実行しています。



インターネットにアクセスするために、Eee Box B202にはWi-Fiワイヤレスモジュールとイーサネットポートが装備されています。 さらに、Bluetoothモジュールをコンピューターにインストールできます。



Eee Box B202の寸法はA4用紙に匹敵し、コンピューターの厚さは25ミリメートルを超えません。 デスクトップの消費電力は20ワットです。 コンピューターには、白、黒、赤、緑のケースがあります。







また、7月には、予算のMSI Windデスクトップの販売が開始されます。 Intel Atomプロセッサの上に構築されます。 コンピューターのRAMの最大容量は2ギガバイト、ハードドライブ-160ギガバイトです。 デスクトップの費用は199〜299ドルです。 Eee Box B202とは異なり、MSI WindデスクトップにはDVDドライブが装備されています。
