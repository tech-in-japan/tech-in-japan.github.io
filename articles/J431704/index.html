週末の読書：聴力を損なうことなく、街のnoise騒から逃れる方法-17の記事と実践的なガイド

「Hi-Fi World」ダイジェストでは、都市で騒音レベルが上昇している理由と、聴覚を維持し騒音公害を減らすためにできることについて説明します。

都市が大きくなるにつれて





  • なぜレストランやカフェが騒々しくなったのか、どうすればいいのか 。 調査によると、一部のケータリング施設のバックグラウンドノイズレベルは、安全レベルをほぼ3分の1超えています。 この記事では、これが起こる理由、ロンバード効果とは何か、レストラン経営者が静かで快適なスペースを整理するのに役立つプラクティスについて説明しました。


  • 健全なエコシステム：それが何であり、どのように動作するか 。 健全な背景が健康と生産性にどのように影響するかを調べてください。 健全な風景とは何か、エコ音響神話を覆い隠し、読み書き時の集中力に対するノイズの影響について説明します。 さらに、音環境を自分で「調整」し、都市の音質汚染の悪影響を最小限に抑える方法に関する簡単な推奨事項を共有します。


  • なぜ私たちの周りの騒音は便利で危険なのですか？ この記事では、バックグラウンドノイズが精神と健康に及ぼす影響をさらに詳しく調べます。 音響療法、超音波の医学的使用、騒音低減のための携帯機器について説明します。


  • なぜポピュラー音楽がそんなにうるさいのですか 都市騒音公害の重要な要素は、バックグラウンドミュージックです。 この記事では、どのように騒々しくなり、この中で商業ラジオ局が果たした役割について説明します。


  • 音楽のマスタリング、圧縮、音量に関するサウンドエンジニア 。 最近の録音の多くは、不自然に狭いダイナミックレンジに悩まされています。 しかし、これはマスタリングエンジニアを悪魔化する理由ではありません。 経験豊富なサウンドエンジニアのLarry Craneが、圧縮が使用される理由とマスタリングが不可欠である理由を説明します。




私たちはあなたの聴覚を保護し、ノイズを取り除きます





  • 年齢とともに聴覚はどのように変化し、どのように保護するか 。 長年にわたって、人が聞くことができる周波数の範囲は狭くなりました。 これは自然で避けられないプロセスですが、遅くなる可能性があります。 この記事では、聴覚受容体が死ぬ理由、高齢のアボリジニが都市の高齢者よりもよく聞こえる理由 、およびライフスタイルの変化が聴覚の維持に役立つ理由について説明します。


  • 周囲の騒音を減らすのに役立つガジェット 。 不要な音から保護するために、耳栓を使用する必要はありません。 この記事では、アクティブノイズリダクションシステムを備えたヘッドフォン、ヘルメット、マスクについて、市場で入手可能なものから、実際には翻訳されていないプロトタイプまでを取り上げます。




写真： ジョージー・パウエルズ / CC BY



  • 自宅で防音をする方法 。 多くのアパートの建物の防音には、多くの要望があります。 このガイドは、問題を自分の手で扱うのに役立ちます。 防音壁と天井に必要な素材とツール、費用とチェック方法を学びます。


  • 地球上で最も静かな場所の長所と短所は何ですか 。 静寂は、ノイズのように、少なすぎるか多すぎる場合があります。 この記事では、沈黙がどのように有害または有用になるか、日中の「沈黙の休止」が脳をリラックスさせ、無響室に関する神話を破壊する方法を説明します。




都市用のヘッドフォンの選び方と手入れの方法





  • さまざまな状況に合わせてヘッドフォンを選択する方法 。 地下鉄、ジム、自宅では、ヘッドフォンがさまざまなタスクを実行します。 防音は公共交通機関で重要であり、スポーツにぴったりです。 自宅でプロのスタジオヘッドフォンを購入すべきではない理由、ケーブルの寿命を延ばす方法、オーバーヘッドモデルを試着するときに探すべきもの-記事を読んでください。


  • 路上でフルサイズのヘッドフォンを選択する 。 路上でヘッドフォンを選択するときは、快適さ、ノイズの遮断、音質など、多くの要因を考慮する必要があります。 人気のあるクローズドタイプのヘッドフォンのレビューを読んだ後、どのモデルが自然な音の愛好家に最適であり、どのヘッドフォンで地下鉄を降りない方が良いか、そして夏には暑くなるでしょう。


  • イヤーパッドとは何ですか 。 この記事では、インイヤーおよびオンイヤーヘッドフォンのイヤーパッドの形状、購入時の評価方法、およびその素材が音と快適性に与える影響について説明します。 また、個々のカスタムデバイスと、最も柔らかいイヤーパッドが必ずしも使いやすいとは限らない理由についても説明しています。

  • アクティブノイズキャンセリングヘッドフォン：賛否両論 。 ますます多くのメーカーがアクティブノイズリダクション付きのヘッドフォンを製造しています。 騒がしい空間で時間を過ごすことを余儀なくされる多くの人々にとって、そのようなモデルは不可欠です。 しかし、それらには欠点もあります-頻繁な充電の必要性から音質の低下まで。 この資料から、ノイズキャンセリングヘッドフォンを購入する際に何を探すべきか、そして経済的な選択をする方法を学びます。


  • ヘッドフォンのクリーニング方法 。 ヘッドフォンのデザイン、外観、モデルに関係なく、定期的なケアが必要です。 汚れたヘッドフォンの音は悪く、しばしば耳の感染症を引き起こします。 この記事では、一般的なApple EarPodsのお手入れのほか、オーバーヘッドおよびインイヤーヘッドフォンをクリーニングするための一般的な手順を説明しています。




ヘッドフォンが危険なのはなぜですか？





  • 聴覚を損なうことなくヘッドフォンを使用する方法 。 より多くの企業がオープンオフィスに移動するほど、職場でヘッドフォンを使用することを余儀なくされます。 多くの人にとって、これは他の人の会話に気を取られずに集中する唯一の方法です。 ただし、ヘッドフォンを常時使用すると問題が発生する可能性があります。医師は、ヘッドフォンを1日1時間以上着用することをお勧めしません。 この記事では、健康を害することなく職場で音楽を聴く方法と、「ピンクノイズ」の助けを借りてオープンオフィスを静かにする方法を説明します。


  • ベビーヘッドフォン：音量を監視します。 理論的には、子供のヘッドフォンは敏感な子供の聴覚を保護するはずです。 しかし、実際にはそうではありません。 一部の「子供用」モデルの最大音量は、許容レベルを大幅に超えています。 この記事では、子供たちに人気のあるヘッドフォンとその安全性について説明します。


  • 低音のヘッドフォン：低周波の利点と害 。 多くの音楽愛好家は、「ベース」ヘッドフォンを好みます。 エクストラベース、ベースブースト、またはメガベースのラベルが付いたモデルは非常に人気があります。 しかし、人気は質の証ではありません。 このような機器を頻繁に使用すると、聴覚に有害です。 この記事では、低音ブーストをいつ、どのように安全に使用するかを学びます。
