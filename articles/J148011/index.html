Androidゲーム向けのIntelおよびThe Code Projectコンテスト

IntelとThe Code Projectは、NDK for IA（Intel Architecture）のAndroidゲームが参加する新しいコンテストを開催しました。



メインの賞金は25,000ドルで、7月6日に始まり、11月6日まで続き、毎月1,000ドルのプロモーション賞金が発行されます。







コンテストに参加するには、ゲームの説明とコードの一部を含む記事を（確立されたテンプレートに従って）書き、x86エミュレーターでテストして説明し、コードプロジェクトに送信する必要があります。



受賞者は毎月コードプロジェクト自体によって決定され、主な賞はコードプロジェクトとプロジェクトが選択する「影響力のある人々」によって決定されます。 審査員は、コードの品質、構造、およびアプリケーションがエミュレーターでどの程度機能するかに注意を払います。
