Whydown-なぜサイトは嘘をついているのですか？

みなさんこんにちは！ Raspberry Piの販売開始に関連した最近のイベントにより、このユニークなデバイスのメイン販売サイトが今日まで横たわって いることが可能になりました。 「膝から持ち上げる」瞬間を逃さないために、古くて便利なプロジェクトの1つ、 Whydownが緊急に蘇生され、 終了しました 。









まず第一に、彼は何が起こっているのかというそのような技術的な詳細を示す方法を知っています。 



 * Connecting to Whydown server... Successfully connected. * Checking NS records... Found NS server(s): ns2.bigcommerce.com ns1.bigcommerce.com NS records are fine. * Testing DNS servers... ns2.bigcommerce.com picked. Found DNS server IP(s): 174.37.171.130 174.37.171.130 picked. Found raspberrypi.com IP(s): 75.126.1.55 DNS server successfully tested. * Checking TCP and HTTP status... Connected. Connection time: 0.2 sec Reading/writing OK. Download speed: 2.1 KB/s TCP connectivity successfully tested. HTTP status code: 503 Down due to HTTP 503 Service Unavailable


このことは、問題の原因をすばやく計算するのに役立ちます。 また、Whydownは、サイトに関連する最新のインシデントとその反応をツイートの形で表示できます。



まあ、最良の部分は、あなたが嘘のサイトを上げることについての通知を購読することができるということです。 これまでのところ、電子メールのみ-raspberrypi.comが登場する前に急いで記事を投稿するので、Whydownの有用性が誰にでもわかるようになっています。 すぐにTwitter経由になります。



健康に使用してください！
