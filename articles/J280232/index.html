Microsoft DevCon 2016-コンピュータービジョン、SQL Server 2016、データサイエンスなど









友達、DevCon 2016カンファレンスの初日に参加者を待っているいくつかのワークショップの発表を皆さんと共有できることを嬉しく思います！



新しい会議形式では、Microsoftテクノロジーの雰囲気に飛び込むだけでなく、専門家の助けを借りて、Windows開発の最も関連性のあるトピックに触れ、Azureサービス、SQL Server 2016の新しいリリースの機能、およびデータ分析を利用できます。



マスタークラスのトピックと専門家を紹介します。



マイクロソフトのテクノロジーエバンジェリストであり、今年のカンファレンスのWindowsトラックの主任イデオロギー家であるDmitry AndreevによるWindowsプラットフォーム上のコンピュータービジョンテクノロジー 。

MicrosoftとOpen Sourceは、 Alexander Belotserkovskyの友情物語です。彼は、Microsoft Russiaのテクノロジーエバンジェリストでもあり、AzureおよびCommunity trackプログラムを形成しています。



MicrosoftはLinuxが大好きです。この有名なスライドは2015年に公開されましたが、Microsoftとオープンソースの友情の物語はそれよりずっと前に始まりました。 マスタークラスでは、少しの歴史、Microsoftプラットフォームとオープンソースの相互作用の例、実際のプロジェクト「Azureクラウドのオープンソース」、この知識を仕事でどのように活用できるかを説明します。

近年、さまざまなプロセスで蓄積されたデータを使用する方法が増えています。 マスタークラスでは、機械学習とデータサイエンスとは何か、現在存在する最も一般的で強力な方法について説明します：ランダムフォレスト、ニューラルネットワーク、ツリーのブースティング、勾配法、最近傍法。 すべての方法は、Azure MLで解決される実際のタスクの例で明確に示されます。



DevCon 2016



DevConは、ロシアの開発者向けの最大のマイクロソフト会議です。 私たちは6年目を過ごし、伝統的に市内の外のマイクロソフトプラットフォームに無関心な人々を快適なカントリーパークホテルに集め、現在の技術について話し合い、実際に製品の最新バージョンを試します。



2016年、DevConは更新された形式で表示されます。 会議を準備する秘secretをお伝えします。なぜ新しいフォーマットになったのか、 会議グリッドがどのように形成されたのか。 同時に、会議プログラムに関する詳細を公開しています。たとえば、以前のASP.NET Coreの発表で、新しい「集中」形式について詳しく知ることができます。



今年、イベント参加者は、テクノロジーに没頭し、新しい知識を獲得する機会が増えます。 コミュニティから多数の新しい興味深いレポートを準備しました。これらのレポートについては、 以前の発表で詳しく説明しました。 同時に、DevConは単なる会議以上のものであり、豊富なプログラムと、プログラム外の専門家や同僚とのコミュニケーションを備えています。 このサイトでは、メンバーになる方法について詳しく知ることができます。
