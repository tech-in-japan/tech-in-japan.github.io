GMate-geditの便利な拡張機能のセット





GNOMEのデフォルトのエディター、したがってUbuntuはgeditです。 Windowsのメモ帳とは異なり、便利です:)が、gedit-pluginsパッケージをインストールした後でも、非常にシンプルに見えます。 しかし、実際には非常に素晴らしく、拡張性に優れたエディターであり、 ウォーターエクステンションを追加するだけです。



GMateは、Mac OS Xの美しいTextMateに注目し、最もおいしい拡張機能、スニペット、テーマの選択を含んでいます。 当初、GMateはRuby開発者専用に作成されましたが、他の開発者にも役立ちます。



GNOMEとRubyが好きなら、インストールする必要があります:)。



設置



Ubuntu Karmic


Ubuntu on Rails PPAをリポジトリに追加する必要があります： ppa：ubuntu-on-rails / ppa

システム→管理→アプリケーションソース、その他のソフトウェアタブ、追加ボタン



そして、gedit-gmateパッケージをインストールします。

sudo apt-get install gedit-gmate







その他のLinux


パッケージリポジトリを参照してください。



中身は何ですか



言語サポート： Markdown 、 HAML 、 YAML 、 Groovy 、 SASS 、 Cucumber 、RubyおよびDjangoテンプレート言語。



多くの美しいテーマ（ほとんどはTextMateで再作成）。



Ruby on Rails、Ruby、jQuery、ucumberなどの多くのスニペット。



そしてもちろん、多くの拡張機能：

  • 突然開く -正規表現でファイルを開く
  • テキスト用ツール - テキストを操作するための追加ツール
  • ファイルを検索-ファイルブラウザのルートからファイルを検索します。
  • 高度なブックマーク - 高度なブックマーク用のツール
  • 配置 -テキストブロックを列に配置します
  • クラスブラウザ -geditのクラスブラウザ
  • タブ切り替え -Ctrl + Tabでファイルを切り替えることができます
  • スマートインデント -geditのスマートインデントモジュール
  • Railsファイルローダー -ファイルがRailsプロジェクトの一部であるかどうかを確認し、RubyOnRails言語を設定します
  • ファイルナビゲーション - ファイルを簡単に開いて切り替えます。
  • ジェミニ -ペアのキャラクターの巧妙な完成。 （{["'' '"]}）
  • 正規表現による検索と置換 -正規表現を使用してテキストを検索および置換します。
  • Railsのホットコマンド -Rails（またはターミナル）コマンドを実行してください！
  • geditでファイルを開く-検索語でファイルを開く
  • Railsのキーボードショートカット-Rails呼び出しでCtrl + Shift + Rを押します
  • To Doリスト -geditのTextmateからのTODOリストバンドルポート
  • 末尾のスペースなしで保存 - 保存する前に末尾のスペースを自動的に削除します
  • 単語の完成-Enterで単語を完成させます
  • Railsパーシャルプーラー -選択したテキストをRailsパーシャルにプルします
  • ハイライトモード -Ctrl + Shift + Hを押して、選択したテキストをすばやくハイライトします
