新しいHP webOSデバイス





今日（2011年2月9日）、サンフランシスコで、HPは新しいwebOSデバイス（HP Veer、HP Pre 3、およびHP TouchPad）を発表しました。 また、webOS 3.0のリリースを発表しました。





HP Veer ：





OS ：webOS 2.2;

プロセッサー ：Qualcomm MSM7230、800Mhz;

ネットワーク ：HSDPA 7.2 / HSUPA 5.76、クアッドバンドGSM / EDGE：850/900/1800/1900;

Wi-Fi 802.11b / g、Bluetooth、GPS（A-GPS）;

画面 ：2.6インチ、マルチタッチ、320x400 px;

キーボード ：物理的なQWERTY;

メモリ ：8 GB;

カメラ ：5 Mpx;

バッテリー ：910 mAh、内蔵

寸法 ：54.5x84.0x15.1 mm;

重量 ：103g。



HP Pre 3 ：





OS ：webOS 2.2;

プロセッサー ：Qualcomm MSM 8x55（1.4 GHz）;

ネットワーク ：HSDPA 7.2 / HSUPA 5.76、クアッドバンドGSM / EDGE：850/900/1800/1900;

Wi-Fi 802.11b / g / n、Bluetooth、GPS（A-GPS）;

画面 ：3.58インチ、マルチタッチ、480x800 px;

キーボード ：物理的なQWERTY;

メモリ ：8（または16）GB; RAM：512 MB;

カメラ ：5 Mpx、HD、オートフォーカス、2番目の前面;

バッテリー ：1230 mAh;

寸法 ：64x111x16 mm;

重さ ：156 g。



HPタッチパッド ：





OS ：webOS 3.0;

プロセッサー ：Qualcomm 8060デュアルコアSnapdragon（1.2GHz）;

ネットワーク ：3G; Wi-Fi 802.11b / g / n、Bluetooth、GPS（A-GPS）;

画面 ：9.7インチ、マルチタッチ、1024x768 px;

キーボード ：仮想;

メモリ ：16（または32）GB; RAM：1 GB;

カメラ ：1.3 MPxフロント;

バッテリー ：6300 mAh、内蔵

寸法 ：240x190x13 mm;

重量 ：740 g。







腺は春に発売され、価格はまだわかっていません。

ガジェットのライブ写真とビデオ： Veer 、 Pre 3 、 TouchPad 。

Palmロゴはすべてのデバイスから消えており、その代わりにHPロゴが使用されています。



さらに、HPはwebOSをPCに移植する計画を確認しています。
