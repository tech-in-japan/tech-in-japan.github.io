ニュースとリリース



  • Zend Framework 3-人気のあるフレームワークの待望のメジャーリリース。 前バージョンのリリースからほぼ4年が経過しました。 内部では、ユーザーはPHP 7のサポート、更新されたコンポーネント、4倍以上のパフォーマンス、すぐに使えるマイクロフレームワーク、ミドルウェアのサポートなどを見つけることができます。
  • Zend Framework 1のサポート終了-ZF3の発表とともに、ZendはZF1のサポート終了を発表しました。 セキュリティパッチは、今年の9月28日までしかリリースされません。
  • PyroCMS v3-かつて人気のあったCMSのリリース。 このプロジェクトは、実際にはLaravelでゼロから書き直されました。
  • PHP 7.0.8、5.6.23、5.5.37-インタープリターの現在のブランチの更新。
  • PHP 5.5のサポートは終了しました -7月5.5日はブランチ5.5の最後となるPHP 5.5.38をリリースする予定です。
  • PhpStorm 2016.2リリース
  • PHP 7.1.0 Alpha 3-3番目のアルファには、最近受け入れられた文が含まれます。 反復可能型 、 リスト演算子の短い構文 、呼び出し可能からのクロージャの作成などです。
