マイクロソフト、Windows Mobile Marketplaceを終了

ここにある-時代の本当の終わり。 Windows Mobile Marketplaceが終了するまでに2週間もかかりませんでした。マイクロソフトは、「ストアを閉じる前に、すべての可能な更新プログラムをインストールする」ことを推奨するニュースレターでユーザーに再度通知しました。 その後、それらは集中モードで配布されなくなりますが、いずれにせよ、プログラムと更新プログラムはメーカーのWebサイトからダウンロードできます。







Windows Mobileは、Microsoftが独自のハードウェアプラットフォームであるPocket PCおよびSmartphone用に開発したモバイルオペレーティングシステムです。 Windows Mobile 6は2007年2月12日、バージョン6.5-2009年5月（皮肉なことに3年前）に誕生し、最新バージョン-Windows Mobile 6.5.3-は2010年2月2日にリリースされました。モバイル業界の発展における重要なステップ-システムは「おそらく」非常に難しいことですが、「コンピューター」体験を他のデバイスに転送しようとしました。 ただし、Windows Mobileを搭載したMicrosoftがモバイルパラダイムの変化をひっくり返したことは同様に明らかであり、以前の力を取り戻すことができるかどうかは現在不明です。



飲みましょう チャリングラスなし。
