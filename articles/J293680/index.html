SaaSビジネスが死んでいる4つの兆候





早い段階で、SaaSビジネスが運命にあることを明確にする多くの指標があります。 それらのそれぞれを見て、状況を修正する方法を見つけましょう。



1. ARPUは20ドル未満



ARPU （ユーザーごとの平均収入）-ユーザーごとの平均月収。 毎月の収益額をユーザー数で割って計算されます。



低ARPUは低マージンです。 各クライアントの技術コスト（サーバーリソースのコストなど）を無視しても、マージンはゼロに近いままです。 なんで？ それはすべてカスタマーサポートです。



低ARPUでのサポートの必要性は、ビジネスの成長という点で大きな災害です。 スタッフが少ない会社を想像してください。ARPUが低く、最も安いサブスクリプションを購入したが、一流のサービスを必要とする多くの顧客を持つスタートアップです。 より収益性の高いものは、月額3ドルを支払った1000人の顧客、または月額100ドルを支払った30人の顧客にサポートを提供することですか？



これらの「3ドル」の顧客は、最も要求が厳しく、最も忠実ではないことに留意してください。 そうすれば、低ARPUがビジネスを急速に台無しにする理由をすぐに理解できます。



この問題を解決するには？ サービスのコストを増やすため。



B2Bの分野で働いており、初期料金プランが月額20ドル未満の場合、提供されるサービスを過小評価します。 他のビジネスの機能を1グラム単位で改善または促進する場合でも、最低サブスクリプションコストは20ドルを超える必要があります。 物語の終わり。



顧客はそれほどお金を払わないのですか？ これは、あなたが十分に価値のないサービスを提供している、問題を完全に解決していない、または自分自身を間違って配置していることを意味します。



B2Cでは、この意味ではほとんど何もできません。 ほとんどのB2C製品は、損益分岐点に至るまでの長くて困難な旅を運命づけられています。 ストーリーは、B2C製品が100万台目のベンチャーキャピタルの資金に頼ることなく生き残ることができたとき、ユニットです。



2. 100ドル/月未満の所得上限



収益の上限-1つのアカウントの月間サブスクリプションの最大コスト。 これは、お客様に提示する料理の真の価値の優れた尺度です。 収入と顧客の流出の両方がコストに依存します。



高コスト =高収入および低流出。



低コスト =低収入および高流出。



1か月あたり100ドルを超えてリクエストして受け取ることができない場合は、製品の価値に取り組む必要があります。 サービスが顧客のニーズによりよく応える場合、より高い関税を設定し、ビジネス開発を加速できます。



ちなみに、このため、 無制限の計画は会社にとって悲惨なものです 。 彼らは即座に収益の上限を制限します。



3. 25％以上の顧客の毎月の流出



流出 -毎月製品を拒否する顧客（それぞれ、および収入）の割合。 25％を超える場合、毎月ビジネスの4分の1が失われます。 当然、この事実は安定性を追加しません。



高い成長率によって大量の流出がマスクされると危険です 。 新規ユーザーの数が放棄された製品の数を超えている限り、収益は増加しています。 問題は、無限の成長は不可能であり、しばらくすると、新しい顧客の接続スケジュールが低下し、ビジネスが不採算になっていることに気付くということです。



解約を減らすには、より高い価値を提供し、ユーザーとの連絡先の数を増やします。 これは、毎週または毎月のニュースレター、ケース、ウェビナー、ビデオの説明などです。 また、チームとともに製品を使用する機能を追加します。 すべての従業員が製品を使用したときに、クライアントがサービスを拒否することはより困難です 。



4. 1年間の事業でMRRが2000ドル未満



あなたのビジネスが1年以上前であり、MRR（Monthly Recurring Revenue、通常の月収）が頑固に数千ドルを超えたくない場合、ほとんどの場合、顧客の緊急の問題を解決しません。



製品が実際の重大な問題を解決しない場合、死産を復活させようとしないでください。 あなたがお金を稼ぐことができる多くの会社で別の「痛み」を見つけてください。



成功への信仰はどうですか？



あなたのビジネスを評価するときに、これらの兆候の1つを見つけたら悲しいです。 すべてのエネルギーを投資し、莫大な時間とお金を費やすと、不快な現実に立ち向かうのは困難です。



しかし、あなたが望むものを達成するための最速の方法はそれに直面することです。 人生は短すぎて何にも影響を与えません。



人々が止まらず、実行不可能なアイデアの開発にエネルギーを費やし続ける多くの例があります。 それどころか、「停止」と言って方向を変える勇気がある人もいます。 彼らの努力は無駄ではありません。2番目ではないので、3番目の製品は「シュート」し、勝利は以前のすべての失敗に対して完全に支払います。
