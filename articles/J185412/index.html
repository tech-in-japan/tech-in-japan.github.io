PTVSとは何ですか？



要するに、Python Studio for Visual Studio（以降PTVSと呼びます）は、その名前が示すように、Visual Studio 2010以降の無料の拡張機能であり、このIDEに完全なPythonサポートを追加します。 自分で設定することで、ハイライト表示、高度なオートコンプリート、コードナビゲーション、リファクタリング、デバッグ、プロファイリング、Djangoのサポートを備えたPythonコードの編集が可能になり、2回のクリックでWindows AzureにWebサイトを公開できます。 同時に、Visual Studioのすべてのエディションがサポートされ、有料のProfessional、Premium、Ultimate、および無料のShellの両方の拡張機能をインストールできます。 後者と組み合わせて、PTVSはWindows上のPython用の完全な無料開発環境です。このバージョンでは、VS Shell + PTVSの便利な結合インストーラーを作成しました。 Visual Studio 2013 Previewを既にインストールしている場合は、[新しいプロジェクト]ダイアログで次のことができます：







私たちのプロジェクトは、いくつかの点で、マイクロソフト独自のものです。 （Apache License 2.0の下での）オープンソースのソースコードは、誰にも驚くことはありませんが、ソースコードをCodePlexで公開するだけでなく、全員がプロジェクトに協力するよう招待しています 。 はい、はい、サードパーティのコードを受け入れます！



最後に、この点に関してしばしば混乱が生じるため、PTVSはIronPythonではなく、IronPythonのみに焦点を当てた環境ではありません。 ほぼすべてのPython実装（CPython、IronPython、Jython、PyPy、Stackless）をある程度サポートしていますが、優先順位は標準をサポートすることであり、ほとんどのPython開発者であるCPythonインタープリターで使用されています。



次に、PTVSのいくつかの最も興味深いユニークな機能をさらに詳しく検討します。



コードを操作する



おそらくすべての開発者にとって、まず第一に、IDEを選択する上で決定的なのはコードで作業することの利便性です。 そのため、オートコンプリート、リファクタリング、シンボル検索、その他の主要な機能が実装されているPythonコードを解析するための高度なエンジンが、PTVSプロジェクトの一部として作成された最初のエンジンでした。 新しいバージョンごとに、解析の品質が向上し、ベータ2.0も例外ではありません。



今日、動的に型付けされた言語の単純な自動補完で誰も驚かせる人はいませんが、多くの編集者は、言語の動的な機能を使用する些細な例でさえ「スキッド」します。 たとえば、次のコードをご覧ください。 



def f(x): def g(y): return x + y return g a = f(1)(2) b = f(3.0)(a) c = f(u'a')(str(b)) d = (a, b, c)[input()]
