Windows用のInternet Explorer 11アップデートおよびWindows Phone 8.1用のIE11アナウンス

本日、Windows 8.1およびWindows 7用のInternet Explorer 11更新プログラム（4月8日以降のセキュリティ更新プログラムで利用可能）をご紹介します。 自動更新を待たない場合は、4月3日以降にMSDNおよびTechNetから更新をダウンロードできます。



さらに、Windows Phone 8.1用のInternet Explorer 11が本日リリースされます。 Windows Phone 8.1のアップグレードは、今後数か月以内にすべての既存ユーザーに対して開始されます。 来月から、更新プログラムは新しい電話にプレインストールされます。 登録済みの開発者は、来週の終わりまでに更新プログラムをインストールできます。



さまざまなプラットフォームでブラウザの機能を実証するために、 22TracksとFishGLの 2つの新しいサービスも開始します。 それらを使用して、WindowsおよびWindows Phone用のIE11の新機能を調査および評価できます。



今日、人々はラップトップ、タブレット、電話などの複数のデバイスでインターネット上で作業し、一日中交互に使用しています。 WindowsおよびWindows Phone用のInternet Explorerを更新すると、開いているタブとお気に入りが同期されるため、最大限の利便性でデバイスを切り替えることができます。 Windows 8.1およびWindows Phone 8.1用のInternet Explorer 11は同じエンジンを使用し、すべての最新のWeb標準をサポートしています。これにより、開発者は、異なるデバイスで作業する場合に同様に便利なサイトとサービスを作成できます。 たとえば、Windows Phone 8.1用Internet Explorerは、最新の（安定した）WebGL仕様のハードウェアアクセラレーションと、タイトルとアダプティブメディアストリーミング、および権利管理を使用できるHTML5のプロ品質のビデオをサポートするようになりました。





Windows Phone 8.1用IE11の紹介

IE11を使用すると、Windows Phone 8.1のユーザーは、スマートフォン向けに最も近代的で完全に機能するブラウザーを使用できます。 デバイス間でお気に入りやブックマークをレンダリングおよび同期するための既に説明したツールに加えて、Windows Phoneは、Windowsデバイス上のInternet Explorerユーザーが以前に使用できた多くの機能を提供します。 IE11 for Windows Phoneを使用すると、更新された情報を含む「ライブ」タイルの形式で、スタート画面にサイトをピン留めすることができます。 ブラウザは、スマートフォンでの読み取り用にインターネット上のテキストを最適化し、音声を使用してサイトにすばやくアクセスするための音声コマンドを最適化する読み取りモードもサポートしています。



さらに、Windows Phone向けIE11は、InPrivateモードと組み込みのSmartScreenフィルターをサポートするようになり、ソーシャルエンジニアリングやフィッシング攻撃によって導入されたマルウェアから保護します。 もう1つの新しいモードはエコノミーモードです。このモードでは、画像のダウンロードが拒否され、関心のあるページ要素のみを表示できるため、トラフィックを制御し、ダウンロードしたデータの量を60〜80％削減できます。



Windows 8.1用IE11の更新

今日から、更新されたIE11はユーザーの個々の要件に適応し、使用するデバイス（8インチタッチスクリーンタブレットまたはマウスとキーボードを備えた24インチオールインワン）を自動的に決定します。 更新されたブラウザーでは、サイトの表示とインターフェイスを個別にカスタマイズできます。1つの画面に表示されるタブの数、フォントサイズ、メニューを決定します。 また、フルスクリーンモードに切り替えるための設定を制御できるようになりました。これにより、ブラウザコントロールが非表示になるか、画面に残るかが決まります。





F12高度な開発者ツール

IE11の違いの1つは、作成プロセス中にWebサイトおよびアプリケーションを作成、診断、および最適化できる開発者ツールのセットでした。 ブラウザーがリリースされてから経過した時間の間に、F12ツールを使用する開発者は、これらのツールを改善するための多くの要望と提案を表明しました。 この変更は、ツールのショートカットとIEのさまざまなバージョンのエミュレーションモードに影響しました。 さらに、UIツールとメモリツール、およびデバッガーツールが改善されました。 拡張機能の完全なリストは、MSDNで入手できます。







最新のブラウザで生産性を向上させるエンタープライズモード

新しいエンタープライズモードでは、ビジネスユーザーは、レガシーWebアプリケーションへの投資を正当化することなく、Internet Explorer 11の最新のWeb標準、パフォーマンス、および強化されたセキュリティのサポートを最大限に活用できます。 Internet Explorerの古いバージョンからの独立により、企業はWindowsの最新バージョンへのアップグレード、Office 365などの新しいサービス、Sony Vaio Tapなどの最新デバイスの使用への移行がはるかに容易になります。



IT専門家は、IE開発者ブログから IEのエンタープライズモードの利点と機能について、およびTechNetにアクセスして技術文書を入手できます。







エンタープライズモードの詳細については、こちらをご覧ください。



22tracksとFishGL：スピード、美しさ、完璧なタッチコントロールのサポート

22TracksとFishGLは、すべてのデバイスで更新されたIE11の利点を実証するのに役立ちます。 22Tracksは、ヨーロッパの最高のDJが参加する現代音楽を検索するためのサービスです。また、開発者は、1行のコードでWindows Phoneやタブレットで使用できる完全にスケーラブルな機能サイトを作成することができます。 22Tracksの作成と操作の詳細は 、開発者のブログで入手できます。







次に、 FishGLのインタラクティブなデモンストレーションは、WindowsおよびWindows Phoneのサイトとサービスの開発でWebGLを使用する新しい可能性を示しています。







未来に目を向ける

また、本日の更新を発表することにより、次のラウンドの準備ができたことを発表します。ブラウザのさらなる改善と追加に取り組みます。 この点で、ユーザーおよび開発者とチームとの最大限の相互作用に関心があります。 また、 @ IEDevChat 、会議およびフォーラムを通じてフィードバック機能を使用します。この機能では、IEの将来のバージョンに対する希望に関する詳細情報を受け取ります。

Windows 8.1およびWindows 7のユーザーは、2014年4月8日からIE11更新プログラムの自動受信を開始します。 以前にインストールしたい人は、 MSDNまたはTechNetから4月2日からアップデートをダウンロードできます。
