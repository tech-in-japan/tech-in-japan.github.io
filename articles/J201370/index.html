2015年までのAllWinnerプロセッサ開発計画：A23、A6X、A7X、A8X、およびA9X

中国のチップメーカーであるAllwinnerは、AllWinnerのSoCを2015年までリリースする計画を最近発表しました。cortexA7 / A15 big.LITTLEアーキテクチャに基づくプロセッサが2014年に予定されています。 2015年には、Cortex A53 / A57 Big.LITTLEプロセッサ（コアの数は不明です）。





  • AllWinner A23 -Mali-400MP2 GPUを備えたデュアルコアCortex A7 @ 1.5 Ghz、および40nm製造プロセス。 2013年第3四半期に予定されているタブレットは、今月後半に発表されます。
  • AllWinner A6X-クアッドコア皮質A7 / A15（big.LITTLE）@ 2.0GHz、GPU（1.7倍のパフォーマンス向上）（A31のSGX544MP2と比較して？） 2014年の第2四半期に予定されています。
  • AllWinner WXは、ウェアラブルデバイス（スマートウォッチなど）用のクアッドコアプロセッサです。 2014年の第4四半期に予定されています。
  • AllWinner A8X（A80）は、GPUパフォーマンスが2倍に向上した8コア皮質A7 / A15 @ 2.0GHzプロセッサーです。 2014年の第1四半期に予定されています。
  • llWinner A7X（A70） -「経済的なbig.LITTLE構成オプション」は8コアプロセッサーです。 2015年の第1四半期に予定されています。
  • AllWinner A9Xは、A53 / A57コア（64ビット！）を搭載した8コアプロセッサで、A31のSGX544MP2と比較してGPUのパフォーマンスが4倍向上しています。 16または20 nmプロセステクノロジーを使用して製造され、2015年の第4四半期に利用可能になります。



公開された計画は、8コアのBig.LITTLE AllWinner A80プロセッサをベースにした比較的安価なデバイスが近い将来に登場することを意味します（2014年の第1四半期または第2四半期

このプロセッサー用に、AllWinnerはAndroid 4.2.2 SDKを提供しました。



AllWinner A23は、A20 AllWinnerのバジェットバージョンであり、イーサネットとSATAをサポートせず、最大解像度1280×800で、安価なタブレットでの使用を示します。 おそらく、より小さな製造プロセスへの移行（40 nm対55 nm）のために、周波数を1.0 GHzから1.5 GHzに増やしました。 消費電力は、オーディオ再生モードで50 mA、スタンバイモードで2.6 mA削減されました。

OpenMAXのサポートにより、ソフトウェア開発が開発者にとってより簡単かつ安価になります。 CedarXを使用する場合、控除を支払う必要はありません。
