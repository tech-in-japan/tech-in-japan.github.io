WindowsストアおよびWindows Phoneストアにアプリを公開し、賞品を獲得する





既存のモバイルアプリケーションを移植するか、新しいアプリケーションを作成する場合、フォーミュラ1アメリカ合衆国グランプリ2014への旅行で10,000ドルを獲得して、開発者、 MarketBot 3Dプリンター、またはNokia Lumia 1520スマートフォンの独自のモダンコーナーを作成できます。



必要なのは、WindowsまたはWindows Phoneアプリケーションをアプリストアに公開することだけです。



受賞カテゴリー



各カテゴリで、学生、個々の開発者、および中小企業（従業員数が100人未満）に属する会社を表彰します。



Windows 8.1に最適なアプリ



ロータスF1チームに参加して、2014年のフォーミュラ1アメリカ合衆国グランプリへの3日間の忘れられない旅行に参加してください。 賞品は、各グループ（学生、個々の開発者、中小企業）から1人の受賞者を受け取ることができます。 旅行は2014年10月31日から2014年11月2日まで行われます。



各グループからこのカテゴリの上位9位に入った最高の開発者には、 Nokia Lumia 1520が贈られます。



Windows Phone 8.1に最適なアプリ



10,000ドルを獲得して、現代のコーナー開発者を作成してください。 賞品は、各グループ（学生、個々の開発者、中小企業）から1人の受賞者を受け取ることができます。



各グループからこのカテゴリの上位9位に入った最高の開発者には、 Nokia Lumia 1520が贈られます。



最高のクロスプラットフォームアプリケーション



MarketBot 3Dプリンターで優勝してください。

賞品は、各グループ（学生、個々の開発者、中小企業）から1人の受賞者を受け取ることができます。



各グループからこのカテゴリの上位9位に入った最高の開発者には、 Nokia Lumia 1520が贈られます。



最も革新的なアプリ



MarketBot 3Dプリンターで優勝してください。

賞品は、各グループ（学生、個々の開発者、中小企業）から1人の受賞者を受け取ることができます。



各グループからこのカテゴリの上位9位に入った最高の開発者には、 Nokia Lumia 1520が贈られます。



DevConメンバーの中でWindows 8.1およびWindows Phone 8.1に最適なアプリ



あなたがDevCon 2014カンファレンスの参加者であれば、さらに多くの賞品を獲得する素晴らしい機会があります（Nokia電話）。 必要なことは、WindowsまたはWindows Phone用のアプリケーションの開発を開始することだけです。



専門家の指導の下、コーディングナイトハッカソンで開発を終了できます。

ハッカソンの最後に、勝者を発表し、賞を授与します。 また、ハッカソンで優勝したすべてのアプリケーションは、サイトで応募して 、グローバルコンペティションに参加し、他の賞を獲得することができます。



参加する方法は？



1. 2014年6月1日までに、コンテストWebサイトに登録してください 。

2. Windowsおよび/またはWindows Phoneアプリケーションを開発します。

3. Windows / Windows Phoneアプリストアで公開します。

4.アプリケーションは、2014年6月1日までに店舗に持ち込まなければなりません。

5. コンテストWebサイトにアプリケーションを追加することを忘れないでください。 アプリケーションを追加するカテゴリを必ず選択する必要があることに注意してください。 2014年6月1日までに急いでください。
