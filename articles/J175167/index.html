Firefox 20リリース

デスクトップ向けFirefoxがダウンロード可能になりました。 このバージョンの新機能：





  • 再設計されたダウンロードマネージャー。 これで、個別のウィンドウではなく、ツールバーでダウンロードプロセスを表示および追跡できます。


Windows 7のダウンロードマネージャー






Mac OS Xのダウンロードマネージャー






Linuxダウンロードマネージャー






  • 「プライベートモード」の新しい実装は、「個別のプライベートウィンドウを開く機能を提供します。すべてのタブは、訪問およびローカルキャッシュの履歴でデータの沈下なしに機能します」（ Opennet ）。 また、コンテキストメニューに、プライベートウィンドウでリンクを開く機能が表示されました。
  • ブラウザ自体をフリーズせずにハングしたプラグインを閉じる機能を追加しました。 ハングしたプラグインが11秒以内に応答しない場合、Firefoxはブラウザーではなくプラグインの再起動を提案します。
  • Firefoxの一連のさまざまな改善の継続： ページの読み込み 、 ファイルのダウンロード 、 シャットダウン 。
  • ユーザーのウェブカメラとマイクへのアクセスを整理するためのgetUserMedia APIのサポート（ユーザーが機器へのアクセスを提供することに同意するかどうかを確認する特別な警告が表示されます）;
  • ECMAScript 6標準ドラフトに示された新しい要素の実装の継続： clear（）およびMath.imulの呼び出しが追加されました 。
  • 「 開発者ツールボックス 」 -Webコンソール、デバッガー、「インスペクター」、スタイルエディターなど、単一のウィンドウ内で開発者ツールにすばやくアクセスするための統合インターフェースを導入。
  • 新しいJavaScriptプロファイリングツール。
  • キャンバスに調光モードのサポートが追加されました 。
  • オーディオとビデオのタグのさまざまな改善 ： playbackRateのサポート、 いくつかのOggストリームの論理和のサポート
  • Windows 7以降では、 H.264、AAC、およびMP3のサポートが追加されています。 この関数は実験的であり、有効にするには、 configに進み、パラメーターmedia.windows-media-foundation.enabledを見つけてtrueに切り替えます 。

    Firefox 21では 、デフォルトでこの機能が含まれる予定です。
  • Escキーを押すとWebSocketとXMLHttpRequestが切断されるバグを修正しました 。
  • UnityプラグインがHiDPIモードで表示されない問題を修正しました 。




重要 ：このバージョンのCSS3では、Flexboxはデフォルトでは有効になっていません （ただし、 計画されていました ）。
