グラコン-市民選挙管理

こんにちはKhabrovchane！



あなたの政治的見解がどうであれ（多くはすでに「政治家」という言葉にうんざりしている）、今や私たちの国で起こっていることに無関心な人はほとんどいません。 誰が投票しても、選挙は可能な限り正直であることは私たち一人一人にとって重要です。 これを実現するために、数か月前、私たちはオブザーバー、弁護士が選挙を監視するのを助けるサイトgrakon.orgの作成を始めました 。



私たちは有名になったり、お金を稼いだり、次のFacebookになろうとはしていません。 目標の1つは、声の盗難を止めることです。 私たちのチームは、さまざまな分野の専門家である100％のボランティアで構成されています。 誰もこの仕事の代金を払ってくれません。 私たちは、私たちの声が勇敢に公然と盗むという事実をもはや容認することができません。 政府の男性が国の資源をポケットに引きずり込み、時には人々に骨を投げかけているという事実は、長い間誰も驚かなかったが、今では彼らは私たちからの声、私たちの自由を盗んでいる。 これは、国の政治生活に積極的に関与している人だけでなく、私たち一人一人にも当てはまります。



Khabrovites、今ドラゴンはあなたの助けを必要としています！



文字通り、夜間、夜間、週末の1か月間の作業のために、作業リソース（http://grakon.org）をまとめました。 先週、私たちは小額の資金を調達しました。そして今、プログラマーを雇い、広告を購入し、信頼できるホスティングの支払いをすることができます。 リソースを選挙に完全に備えさせるには、次のものが必要です。

  1. JavaScriptおよびCSSプログラマー、
  2. Python / Djangoプログラマー、
  3. ハッキングサイトに対する保護の専門家。


さらに、私たちには常にボランティアが必要です。

  1. サイトテスト（Seleniumなど）を自動化できるテスター、
  2. 不正行為メッセージを確認するモデレーター、
  3. Webサイトプロモーションのヘルパー。


無料で、私たちのように、または有料で働きたいかどうかは関係ありません。 最も重要なことは、昨日-スタートアップで常に起こるように、今、またはむしろ、助けが必要です。
