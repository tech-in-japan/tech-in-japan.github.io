Peregrine SemiconductorのUltraCMOS：CMOSテクノロジーのマイクロ波の機会





私は、頭字語CMOS （ CMOS ）の大部分がロジックチップと電界効果トランジスタに関連していると信じています。 マイクロ波部品の製造には、主に周期表の///--/ Vグループの元素（GaAs、InP、SiGe）が使用されます。 Konの開発-シリコンオンサファイア（Si＆Al2O3）テクノロジーを25年以上にわたって開発してきたPeregrine Semiconductorは 、数十年にわたって市場を支配してきたGaAsテクノロジーよりも優れたパラメーターを持つUltraCMOS 11テクノロジーを開発しました、とRon * Coff氏は言います。



KNSテクノロジーの開発の推進力は、耐放射線性の向上であり、これは宇宙（Space）および高信頼性（Hi-Rel）アプリケーションに必要です。 そのため、たとえば、「サイリスタ効果」（ SEL ）はまったくありません。 また、低放射線量（ ELDRS ）に対する過敏症もありません。 現在、このような製品はE2V部門を通じて提供されています。 Peregrine Semiconductorは、現在最大40 GHzの民間製品を提供し、水準を引き上げ続けています！



現在、Peregrine Semiconductorは、最大8.5 GHzのスイッチ、最大4 GHzのデジタル減衰器、2次電力変換器、最大5 GHzのPLLチップ、最大13.5 GHzの分周器など、信頼性の高い市場向けに製造しています。



民間市場向け：最大40 GHzのスイッチ、最大8 GHzのデジタル減衰器、最大3 GHzのデジタル化可能コンデンサ、最大6 GHzの可変しきい値を備えたパワーリミッター、ミキサー、モノリシック振幅および位相コントローラー、GaN FETドライバー。



多くのパラメータにおいて、Peregrine Semiconductorチップは、Skyworks、RFMD（Qorvo）、Hittite（ADI）などの有名なメーカーと競合しています。 それらの中で最も成功したものを考えてください。



1.マイクロ波スイッチ（ RFスイッチ ）



マイクロ波スイッチは、創業以来生産されており、50を超える製品の最も幅広い製品群を持っています。 それらは、セルラー通信、無線および測定機器、アンテナシステム、ケーブルネットワークで使用されます。







Peregrine Semiconductorスイッチの利点は、低コスト、低消費電力、高い直線性、および多くの場合、動作温度範囲の拡大です。 欠点のうち、最速かつ最も強力なものとはほど遠いものです。 最適なスイッチング時間は145 ns、最大のスイッチング電力は18ワットです。 極の最大数は5（ SP5T ）です。 約200 MHzの周波数で、最適な損失は0.1 dBで、アイソレーションは94 dBです。



PE42020は、0 Hz〜8 GHzのユニークな「True DC」 SPDTスイッチです。 「真」という言葉は、定電圧（最大±10 V）と電流（最大80 mA）を切り替えることができることを強調しています。 これは測定機器で使用され、 ここで詳細に説明されています 。



PE42723-75 ohm SPDTスイッチ5-1794 MHz、入力でIP3 = 88 dBm @（17 MHz＆Pin = 18 dBm）の超線形、3fo = -140 dBc @ 17 MHzと特徴付けられる非常に低い高調波 パスごとの損失は0.3 dB、RFC-RFXポート間の分離は54 dB @ 204 MHzです。 静電3 kV（ HBM ）に対する保護が組み込まれています。 これは現在、ケーブルネットワークのDOCSIS 3.0 / 3.1標準を満たす世界で唯一のスイッチです。



PE42524は、フリップチップパッケージの0.01〜40 GHzワイドバンドSPDTスイッチです。 AN42に 、このようなケースと例のインストールに関する推奨事項が記載されています 。







このスイッチの特徴は次のとおりです。26.5GHzで50 dB、40 GHzで51 dBの高アイソレーション。 優れた直線性-36.5 dBm @ 26.5 GHzおよび低損失-1.8 dB @ 26.5 GHz、3.1 dB @ 35 GHz。 2 kV静電保護（HBM）を内蔵しています。 これは非常に高く登った唯一のCMOSスイッチです！ 他のブロードバンドスイッチの中でも、最大26.5 GHzおよびPE42542 / PE42543-最大18 GHzで動作するPE42522に注意する必要があります。



PE423422-自動車規格AEQ100 Grade2（105°C）に準拠した-40°... + 105°Cの拡張温度範囲を備えたSPDTスイッチ100-6000 MHz 損失-0.25 dB @ 1 GHz、アイソレーション41 dB @ 1 GHz; それぞれ6 GHz -0.9 dBおよび16 dBの場合。



PE42920-デュアル差動スイッチ10-6000 MHz。 損失-1 dB @ 1 GHz、同じチャネルで26 dB、異なるチャネル間で30 dBの分離。 2 kV静電保護（HBM）が組み込まれています。 3.3 Vでわずか100μAを消費



2.デジタル減衰器（ DSA ）



民間のラインには、50オームと75オームの両方の20以上のPE43XXXマイクロ回路が含まれています。 ラインのトップ製品はPE43705で、動作周波数範囲は9 kHz〜8 GHz、減衰は最大31.75 dB、最小ステップは0.25 dBです。これは128の組み合わせまたは7ビット制御に対応します。



信頼性の高いアプリケーションの場合、6ビット制御に対応する最小ピッチ0.5 dBで最大31.5 dBの減衰で、0.25〜4000 MHzの動作周波数範囲でPE94302のみが生成されます。



デジタル減衰器の欠点の1つは、内部キーの非同期動作のために、切り替え時の異常値です。 ペレグリンセミコンダクターはアッテネーターファミリーPE43711 / PE43712 / PE43713（グリッチレス）を開発しましたが、この欠点はほとんどありません。 さらに、これらの超小型回路は、前世代のデジタル減衰器とピン互換性があるため、簡単に交換できます。







3.デジタル調整コンデンサ（ DTC ）および調整スイッチ（ TCS ）。



一種のユニークな超小型回路。 DTCラインには、動作周波数範囲が100〜3000 MHzの8チップPE64X0Xが含まれています。 このような超小型回路には、4または5ビット制御のスイッチドキャパシタのセットが含まれており、16または32の異なる状態を提供します。 最大静電容量範囲：188フェムトファラッドの増分で1.88-14 pF、最小：117フェムトファラッドの増分で0.6-2.35 pF。 これらのチップは、SPIまたはI2Cインターフェイスによって制御されます。 これらは、さまざまなマイクロ波回路のアンテナの調整、調整、修正、最適化に便利に使用されます。



PE613010 / PE613050のチューニングスイッチ（TCS）は、同じ周波数の外部補正回路を切り替えるために設計されています。



4. パワーリミッター



パルスシステムの敏感な回路を保護するために使用されます。 2つのリミッターが生成されます。 上部のPE45450の動作周波数範囲は9 kHz〜6 GHzで、限界しきい値は25〜35 dBmです。 高い入力直線性により特徴付けられるIP3 = 915 MHzで70 dBm。 永久許容電力-10ワット、パルス-50ワット。 静電気8 kV（HBM）に対する保護が組み込まれています。 重要な利点は、調整可能な制限しきい値に加えて、約1 nsの高い反応速度です！







5.ミキサー



現在のラインナップは5チップで構成されています。 現在、最も高い周波数のミキサーは、周波数範囲10〜19 GHzのPE41901のミラーコンポーネントの抑制です。 変換損失は、17 dBmの局部発振器電力で8 dB、中間周波数は0〜4 GHzです。 静電気1 kV（HBM）に対する保護が組み込まれています。 超小型回路は、ADI（Hittite）製のミキサーHMC527LC4（8.5-13.5 GHz）およびHMC528LC4（11-16 GHz）と完全にピン互換です。



6.プリスケーラー



民間市場では、PE35400のみが生産されます-水晶の形で3-13.5 GHzの周波数に対して4の分周器。 残りの7チップ-2、4、および8で最大13.5 GHzの分周器は、信頼性の高い市場向けに設計されています。



7. PLLチップ（ PLL ）



民間市場では、PE83336のみが生成されます。0.5〜3 GHzの周波数の整数PLLです。 より正確には、セラミックケースCQFJ -44および軍用温度範囲-55 .. + 125°の超小型回路です。



残りの11個のマイクロ回路（3個のフラクショナルPLLと8個の整数PLL）は、信頼性の高い市場向けです。 このクラスの最高のパラメーターは、PE97240整数放射PLLが備えています：動作周波数800-5000 MHz、基準周波数100 MHz、正規化された位相ノイズ-230 dBc / Hz。 チップはセラミックケースCQFP- 44で製造されています。 3 GHz（2.7 V電源、5/6プリスプリッター、+ 25°C、500 kHzフィルター帯域）での典型的な位相ノイズを以下に示します。







8.位相回転子とモノリシック振幅および位相コントローラー（ MPAC ）





レビュー記事はこれらの超小型回路に捧げられています。 現時点では、1つの位相シフターのみが生成されています。PE44820-1.7〜2.2 GHzの周波数範囲で、360°すべてをカバーし、位相分解能は1.4°です。 狭帯域で作業する場合、1〜3 GHzの周波数に使用できます。



MPACは、アンテナアレイおよび測定機器での使用を目的としたDohertyおよびPE19601複合増幅器を構築するための3つ以上の複雑なPE461X0超小型回路です。 PE46120 / PE46130 / PE46140は、それぞれ周波数範囲1.8-2.2 / 2.3-2.7 / 3.4-3.8 GHzで動作し、2.8°のステップで90°をカバーする位相シフターに加えて、0.5 dBのステップで7.5 dBの減衰器も備えています。 5.6°刻みで360°と0.5 dB刻みで31.5 dBの減衰器がオーバーラップする位相シフターに加えて、PE19601にはさまざまな動作モード用のスイッチも含まれています。 このチップは、8〜12 GHz（Xバンド）の周波数範囲で動作します。



9.電力変換用チップ（ 電源管理 ）



これらには、それぞれ2/6/10 AのPE99151 / PE99152 / PE99153の信頼性の高い市場向けの高効率（93％を超える）電圧コンバーターが含まれています。 それらは、コンピューター（ FPGA 、 DSP ）の二次電源に使用されます。 すべてのコンバータの入力電圧は4.6〜6 V、出力は1〜3.6 Vです。超小型回路は、CQFP-32パッケージまたは水晶の形（DIE接尾辞）で入手できます。



世界最速の GaN PE29100電界効果トランジスタドライバは、民間市場向けに入手可能です。 2x1.6 mmのフリップチップパッケージで提供され、33 M​​Hzのスイッチング周波数、8 nsの伝搬遅延、1 nsのオン/オフ時間を備えています。
