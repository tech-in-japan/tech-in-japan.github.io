ロシアの子供たちとキャンプOLPC



最近のコンピュータサイエンスを教える方法論はどうあるべきでしょうか？ 彼女はしばらくして何になりますか？ ...教育学の世界では、すべてがまだ残っているようです。 それでも、ほとんどの教師は子供たちを数時間黒板に立たせ、バイナリシステムから10進数に、またはその逆に数字を変換させます。 しかし、以前のように、知識をよりアクセスしやすくしようとしている愛好家がいます。

ゲームの指導方法が本当にうまく機能するという事実は、私たちが長い間知っていることです。 これは、外国語コースでトレーニングを編成するときに使用されます-ほとんどの学校ではありません。 ここには、机、ボード、コンピューターが3週間に1回しかありません。

才能のある子供向けのテーマキャンプは、その効果にもかかわらず、かなりまれなイベントです。 もちろん、アーティスト、数学者、文学者、プログラマーのためのキャンプがあります。 ここでは、ロシアで開催されたOLPCキャンプについてお話しし、最初のキャンプであるデジタルエコロジー2008を振り返ります。





去年の8月でした。 ニジニノヴゴロド教育大学の教師グループは、 5〜7年生の子供たちが一方ではニジニノヴゴロド地域のレッドブックの性質に、他方では現代のテクノロジーに触れられるキャンププロジェクトを開始しました。

イベントのお金は素晴らしい方法で見つかり、興味のある人が見つかりました。 以下で説明する他の機器に加えて、ミレニアム基金のメイキングマイルによって支払われた50個のOLPC XOラップトップを購入することができました。 はい オランダ財団。 彼はロシアの子供たちのためにラップトップを買うためにたくさんのお金を与えました。

次に、「私たちの日々」を支えた人々、彼らの目標、技術的なデバイス、プログラム、テストされた子供たち、そしてこれらすべての主題がそこで何をしたかについてお話します。



ハードウェアとソフトウェア





構成主義は教育学的な概念であり、（簡潔かつ比phor的に）以下から構成されています。私たちの宇宙を素早く理解したいのであれば、その創造に参加してください。 言い換えれば、最大の教育効果は学生の積極的な地位によってのみ達成されます。

魔法の杖ではないにしても、彼が世界を創造するプログラマーのためのコンピューターとは何ですか？ それで、私たちはそのようなスティックを子供たちに渡し、それらの使い方を彼らに教えます。 すぐに簡単に要件を作成します。

  • 杖は学校の机の上に立たないで、携帯する必要があります。 デスクトップマシンではなくネットブック
  • 創造の道具はその所有者から保護されなければなりません。 クラスメートとOLPC XO
  • すぐに私たちの小さなパダワンに大人のジェダイの剣を与えることはできません。まず彼に練習させてください。 C ++ / Java / PHPではなく、Scratch / Squeak / Logo。
  • 抽象的思考はハードに発展します。 そして、最初は、舌を触ったり、触ったり、なめたりできるのが面白いです。 センサー、ロボット、UFO！

結論と結論





コンピューター技術の現在の開発レベルは、誰もが誰とでも対話するためのモバイルまたは他の（個人）デバイス（同じ知識分野に情熱を持っている教師、同僚、友人、または未知のデバイス）を（明示的または暗黙的に）持っていることを意味します。新しい形のコミュニケーションと知識の出現。

現在、生徒と教師の間のコミュニケーションの文化にはギャップがあるようです。 現代の学生は2つの方法で知識を習得しますが、これらは多くの場合、まだ互いに結びついていません。 これは、従来のタスク、レッスン、ゲーム、およびインターネットを備えた古典的な学校であり、無料の非公式のコミュニケーション、検索、および予測できない「トレーニングプログラム」を意味します。 これらのパスを組み合わせることで、学生は最新の情報技術の成果を使用する文化を育むことができます。 これはまさに、現代の学校が生徒たちに遅れずについていくために必要な共生です。
