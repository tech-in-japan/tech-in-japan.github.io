Firefox用拡張機能:: Habraコメントエディター

Firefoxの最初の拡張機能であるHabrahabr Comment Editorをご紹介します。











必要条件



Firefox：3.0-3.5



説明



拡張機能は、コメント入力フィールドに編集ボタンを追加します。

次のタグのコメントフィールドで選択されたテキストをフレーム化できます（「ホット」キーのサポートが実装されています）。

ボタン ホットキー 結果
B ctrl + B <strong>テキスト</ strong>
私は ctrl + I <i>テキスト</ i>
うん ctrl + U <u>テキスト</ u>
引用 ctrl + Q <blockquote>テキスト</ blockquote>


また、ワンクリックで写真（ IMGボタン、ホットキーctrl + P ）とリンク（ URLボタン、ホットキーctrl + L ）を挿入できます！ 9.99で今日だけ！



ここから全部をダウンロードするか、ここから直接ダウンロードできます。

  • 解析絵文字を追加（すべてのコメント用）
  • そして、おそらくWYSIWYGエディターのようなものを実装する


もちろん、コメントで追加や批判を聞いてうれしいです:)



UPD。 Khabrovskの住民の要請により、 < blockquote >タグが追加されました。 これまでの実装は、もう1つのボタンQuote （およびホットキーctrl + Q ）の形式でのみ行われていました。 更新されたバージョンは上記のリンクから入手できます。 アップグレードプロセスはインストールプロセスと同じです。古いバージョンは新しいバージョンに簡単に置き換えられます。
