GoogleはAndroid搭載端末のメーカーにさらに多くを要求し始めました



Googleは、Androidデバイスのメーカーの要件を厳しくします。 更新されたMobile Application Distribution Agreements契約では、パートナーはデバイスに9個だけでなく20個のGoogleアプリケーションをインストールし、メイン画面にGoogle検索ウィジェットを配置すると同時に、Google Playストアアプリケーションと13 Google Chrome、Googleマップ、Googleドライブ、YouTube、Gmailなどを含むGoogleアプリケーション。さらに、パートナーは、音声認識を備えた検索および仮想アシスタントに特定の単語を使用する際の推奨事項を厳密に従う必要があります。



ほとんどの場合、Androidのデバイスのメーカーはオペレーティングシステムの上に独自のソフトウェアシェルをインストールし、基本的なアプリケーションを対応するものに置き換えようとしています。これがおそらく要件を厳しくした理由です。 将来、これらすべてが、すべての新しいAndroidデバイスのインターフェースの統一につながる可能性があります。 同時に、検索大手の組み込みサービスがないAndroid Open Source Project（AOSP）の人気が高まっています。 深刻な競争のため、デバイスメーカーは、デザインだけでなくコンテンツ、さらにAndroidデバイスを統合するためのGoogleによるそのような小さなステップでさえも、ユニークな製品で購入者を引き付けることが重要です。 たとえば、OnePlus Oneは一般的にCyanogenModで実行されるトップエンドのスマートフォンであり、十分に存在しているようです。





Windows 1 10！



はい、誰もが期待したとおり、9ではなく10です。 先日、マイクロソフトの新しいOSのプレゼンテーションが行われ、聴衆は最初から驚きに驚きました-これがその名前です。 彼らはWindows 1という名前を使いたいと思っていましたが、それが採用され、新しいOSの変更の重要性とボリュームを強調するために、10番目の名前を付けることにしました。



新しいOSの主な機能は、PCからスマートフォンまで、あらゆる種類のデバイスで実行を開始することです。 単一のプラットフォームが作成され、すべてのデバイスの単一ストアからアプリケーションを一度にインストールできます。



企業セグメントはspareしみませんでした。 会社の代表者によると、彼らはすべてのデバイス間で同期される印象的な個人データ保護を作成しました。



次に、[スタート]メニューについて説明します。 彼は8人のユーザーの多くの否定的なレビューのために特別な注意を払われました。 従来の[スタート]メニューは部分的に戻りましたが、タイル化されたインターフェイスの一部は保持されています。これにより、スタートメニューのサイズを変更し、アプリケーションに新しい「タイル」を追加できるようになりました。

8 OS上の多くのプログラムがすぐに全画面表示で実行され、何もできないという事実に対するユーザーの不満を考慮しました。 その結果、Windows 10では、プログラムのデスクトップバージョンのユーザーにより多くの注意が払われます。







10番目のOSのマルチタスクは、大幅に変更されました。 下部のタスクバーで実行中のアプリケーションとアクティブなアプリケーション間の通常の切り替えを切り替えるためのインターフェースを変更しました。



Windows 10コマンドラインが完成しました。 通常の外観は残っていましたが、非常に不足していた多くの機能が追加されました。これは、非常に不足していたCTRL + Vの組み合わせを使用する平凡な機能に値します。



これが重要であると思われる主なものであり、Windows 10がすべてのユーザーを喜ばせることを願っています。

先日、PCおよびラップトップ向けのWindows 10テクニカルプレビューが利用可能になり、すべてのデバイスにインストールできるフルバージョンが2015年に登場します。





モバイル広告「VKontakte」



先週、 Apptractor.ruは、VKontakteモバイルアプリケーションの機能を使用した新しい広告プラットフォームに特化したイベントを開催しました。







毎日、Androidデバイスの1,800万人のユーザーとiOSプラットフォームの500万人のユーザーが、モバイルを使用してソーシャルネットワークにログインしています。 図から判断すると、iOSタブレットが全デバイスのほぼ4分の1である場合、Androidタブレットの数は約6％にすぎません。







広告は、テキストの説明、写真、アプリケーションの評価、およびインストールのためにモバイル市場に移動するためのボタンを含む別個の投稿としてユーザーのフィードに送信されます。 Androidインストールの場合、VKontakteアプリケーション自体が追跡し、iOSデバイスの場合、VKontakte SDKの機能を使用する必要があります。 VKontakteのWebバージョンでは、広告会社を管理できます：費用とターゲティング、その結果の監視。 サードパーティのWebサービスを介したアプリケーションからストアへの移行はまだ監視されていません。







このプレゼンテーションでは、Androidの公共料金請求アプリケーションの広告の例を提供しました。 平均CTRは3％で、平均設置価格は約18ルーブルでした。 開発者は、近い将来にVkontakteの新しい広告フォーマットを使用できるようになると約束していますが、今のところは参加のリクエストを残すことができます。
