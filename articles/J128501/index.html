レイジー向けのWindows 8 Developer Previewをインストールする

タスク： Windows 8 Developer Previewをプレイします。



条件：古いWindowsは存続する必要があり、DVDドライブはなく、10ギガの大きなフラッシュドライブもあります。



従来のソリューション：



  1. 仮想マシンを配置します。 遅いため、追加のソフトウェアが必要であり、ハードウェアの仮想化がないと機能しません。
  2. 別のハードドライブを装着します。 残念ながら、2番目のネジをラップトップに挿入することはできません。私にはありません。
  3. ネジの別のセクションを選択します。 そして、これは余分なソフトウェアを入れて、パーティションの周りでタンバリンと踊るためです...怠け者！


さらに、通常、最後の2つのオプションには物理DVDが必要ですが、そうではありません。 ウェイアウト？ VHD！



Windows 8のISOイメージに加えて、Windows 7もインストールする必要があります-以前のバージョンではVHDの操作方法がわかりません。 また、Linuxでは、インストーラーを実行することはできません。







7つのリリースにより、Windowsはようやくイメージから起動する方法を学びました。 つまり、VirtualPCで使用されるVHDイメージを使用します。 さらに、組み込みツールを使用してWindowsでそのようなイメージを直接作成し、使用するためにマウントできます。 これを活用します！



したがって、VHDを作成します。 [コンピューターの管理]-> [ストレージ]-> [ディスクの管理 ]を押して、 [アクション ] -> [VHDの作成 ]メニューを開きます 。 サイズは少なくとも40ギガバイト（インターネットによる）でなければなりません。動的タイプを選択します。そうでない場合は、作成されるまで古くなります。 5400RPMネジ付きのラップトップ上。 すぐにダイナミックディスクが作成されます。 名前c：\ vhd \ w8.vhdを付けて、さらに使用します。 今、それをマウントする必要があります- アクション-> VHDをアタッチします 。



これで、ISOのコンテンツをVHDにコピーできます。VHDは通常のディスクとして利用できます。 ISOは、たとえばデーモンツールを使用してマウントできます。 任意のファイルマネージャーでファイルをコピーできます。Far、Explorer-違いはありません。



コピーした後、コンピューターを再起動し、F8キーを押して回復コンソールに入ります。 ここで、既存のWindowsにログインする必要があり、リカバリツールの選択を示すウィンドウが表示されます。 最も強力で汎用性の高いツールに興味があります-コマンドプロンプト、つまりコンソールは普通です（：回復モードでは、ディスクの文字が時々変わることがあります。それらをチェックする最も簡単な方法は、ラップトップを起動し、コンソールに書き込むことです



notepad







Enterを押します。 ファイルを開くダイアログにすべてが表示されます。 たとえば、CドライブがDドライブになったため、パスC ：\ vhd \ w8.vhdがD ：\ vhd \ w8.vhdに変わりました 。



文字が整理されたら、VHDをマウントします。 これを行うには、DISKPARTを実行します。



diskpart







そして、私たちは彼に言います：



select vdisk file="c:\vhd\w8.vhd"

attach vdisk







メモ帳を再度開いて、VHDの文字（私の場合はF）を確認します。これで、Windows 8インストーラーを安全に実行できます。



f:\setup







必ずカスタムインストールのタイプを選択してください。そうしないと、古いWindowsが失われ、小鳥が発生します。 [カスタム]を選択すると、インストールするパーティションを尋ねられます。 VHDを選択します。 その後、インストーラーは、彼が方法を知らないことを通知します。 彼を信じないでください！ OOKとNextを押すだけですべてが機能します。



それだけです。インストールは通常の方法で行われ、完了するとグラフィカルなブートマネージャーが表示され、どのWindowsから起動するかを尋ねられます。











PSこのインストール方法は、いくつかのチュートリアルを読んだ後に作成されました。



  • VHDからWindows 8 Developer Previewをインストールして起動するためのガイド
  • あまり仮想的ではなく、より多くのマシン-Windows 7とBoot to VHDの魔法


これらのチュートリアルでは、トピックに関する多くのスクリーンショットと、使用されるコマンドの説明があります。
