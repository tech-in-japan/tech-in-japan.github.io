ロシア語プロジェクトがファームウェア1.1.3で登場しました（そして私の電話を殺しました）

遠くから始めましょう。 今日、大衆が待ち望んでいたことが起こりました！ 「iPhone」の最新の2つのファームウェアは、「壊れない」ことで有名でした（特に1.1.3、MultiSimとTurboSimでさえ「受け入れられませんでした」）。 これに感謝するのは、わずか17歳の悪名高いGeoHot（George Hotz）です！



ファームウェア1.1.2および1.1.3で購入したiPhoneは、ソフトウェアで比較的簡単に操作した後にのみ、はんだごてを使用せずにSIMカードで動作するように強制できます。



iPhones.ruからの手順1.1.2および1.1.3のハッキング： http ://www.iphones.ru/iNotes/2156/（サイトが「横たわっている」場合でも驚かないでください -1.1.3のハッキングに関するニュースの後、群衆はそれに落ちました人々、WordPressは我慢できません）。

別の優れたステップバイステップの手順： http : //f.macrostov.ru/viewtopic.php?f =7& t = 27

それはサイトで言われていますが、私はまだあなたに警告します-注意してください、ハッキング方法は完全に「新鮮」です、それはバグがあるかもしれません。 しかし、いずれにしても、iPhoneはほとんど破壊されません。 はい。また、GeoHotメソッドを使用して「解散」したことを知っている複数の友人がすでに登録解除しています。



前述の重要なイベントに関連して、 RipDevのロシアのiPhoneハッカーは、「 1.1.3の脱獄があまりにも曲がっていること」を誓わず、最新のファームウェア「ロシアプロジェクト」を含む「iPhone」の完全なロシア化パッケージをリリースしました。バージョン1.0.5





猫の下で開発者自身からのリリースに関する詳細。



* ファームウェア1.1.3のサポートを追加 -タンバリンとのシャーマニズムやダンスなしで、ロシア語プロジェクトがこのファームウェアにインストールされるようになりました。 唯一の要件は、3.0以上のInstaller.appバージョンの存在です。 特別なテクノマゾヒスティックな傾向がない限り、とにかくこのファームウェアに切り替えることはお勧めできないことに注意することは私たちの義務であると考えています。 現在存在する1.1.3のジェイルブレイクオプションは、最初のリリースから大幅に改善されましたが、依然として多くの問題に悩まされています。 したがって、私たちが知っているすべての脱獄オプションの下でロシアのプロジェクトをテストするのに多くの時間を費やし、そのパフォーマンスが満足のいくものであると認識しましたが、それでも脱獄またはApple自体によって行われた新しいファームウェアのグリッチに関連するいくつかの問題があります。 警告しましたね :)

* コンポーネントCorrの作業を大幅に加速しました。 形式 -SMSアプリケーションは、はるかに速く開き、SMSからの逸脱を引き起こす可能性のあるいくつかのエラーを修正しました。

* 改善されたRiPInstall-電話機自体を再起動できるようになり、アプリケーションのインストールの内部ロジックを若干書き換えました（Caterpillarのリリースに備えて）。

* ロシア語のローカライズにいくつかの変更が加えられました（そしてもちろん、1.1.3サポートが追加されました）。

*パッケージと電話全体の改善を目的とした多くの小さな変更 。





更新プログラムは現在Installer.appで入手できます。 電話機にRPをインストールする方法に関する詳細なガイドを次に示します。 この更新は、すべての登録ユーザーに対して無料です。 プログラムのインストール中に、開発者ページまたはiPhone自体から500ルーブルの「ロシアプロジェクト」を購入できます。



「ロシアのプロジェクト」の主な利点を思い出させてください （これがないと、多くの「国内の」iPhone所有者は生きられません）。

-ユーザーインターフェイスの完全な翻訳

-連絡先リストでロシア文字を削除して、見つけやすくする

-ロシア語キーボード

-主要な通信事業者の色のロゴ

-Notesプログラム用の美しいMarkerFeltフォント

-通常のロシアの外観形式にダイヤルするときに電話番号をリード

-呼び出し元の地域を定義します

-軽微なバグを修正（リダイレクトプレートが常にポップアップするなど）。



一般的に、非常にバグがある場合は賭けに行きます-退会します:)。 1.1.3用のロシア語キーボード（パッケージ全体の前に個別に出てきました）は、今のところ容赦なくバグがありますが、残念ながら頻繁にクラッシュします。



UPD。：RPはすぐに配置を拒否しました。 以前にインストールしたMobile Enhanserとロシア語キーボードを削除する必要があることが判明しました。 その後、インストールされました。



確立されています。 再起動を要求し、電源アイコンでハングしました！ 私は彼にリセットをかけ（2つのボタンを押したまま）、電話はモード回復モードに入りました。 それで、新しい「ロシアのプロジェクト」は私の電話を殺しました！ むしろ、2台の電話でさえ、自宅に2台あります。

まあ...私たちは復元します...



追伸 誰が設定して動作するか-購読解除。 ジェイルブレイク1.1.3はまだ非常に曲がっており、これが当てはまる場合があります。
