同社は、各ニッチで一切れのパンを手にしようとはせず、主に電源（ Hiperpower ）およびメディアセンターのベアボーンシステム（ Hipermedia ）に焦点​​を当てています。 さらに、コンピューターケース（ Hipercase ）、アルミニウムキーボード（ Hipercomm ）、クーラー（ Hipercool ）、およびワイヤー（ Hiperlink ）のカテゴリの製品に対して、多くのさまざまな賞と特許が授与されました。

外観



外観は自信を呼び起こします-光沢のあるプラスチック製の挿入部が前面にあるつや消しの黒い金属ケース。 上部カバーには2つの換気グリルがあります（1つは電源の上、2つ目はソケットの上）-1​​つを通して、マザーボードの赤いテキスタイルが見えます。 対称的にケースの両側には、換気「エラ」も配置されています。 前面（プラスチック）側で、左から右に：

ドライブスロットは何よりも上にあり、その下には、ディスクを取り出すためのボタン、メディアセンターをオンにするボタン、およびリセットボタン（リセット）があります。 右側には真空蛍光（VFD）ディスプレイがあり、その下には6つのマルチメディアボタンがあります。

まあ、非常に右側には、4つのカードリーダーコネクタ（SM、CF / MD、SD / MMC、MS）、3つのオーディオコネクタ（ライン入力、マイク入力、ステレオ出力）、IEEE1394および2があります。 USBポート。 したがって、偽のパネルを閉じると、その栄光のすべての前に、不必要な要素で完全に過負荷になっていないデバイスが表示されます。 そして、それを開くと-機能拡張の可能性があります。 それが判明したように、非常に便利です。











ケースの裏側には、完全に未経験のユーザーが簡単に混乱できる多くのコネクタがあります。 マニュアルから写真を表示する方が簡単です。 写真を投稿しません。なぜなら、 私はそれを「枕の形」にします:(







統合ネットワークコントローラーは、 Realtek RTL8201CL PCIEコントローラーに基づいており、10/100/1000 Mbpsをサポートしています。 Wi-Fiの存在に注意する必要があります。

サウンドコントローラーも統合されています。これは、高解像度オーディオの「チップセット」サポートと7.1オーディオシステムを接続できるRealtek ALC883コーデックに基づいており、 IXBTで非常に良い評価を受けました。



ケースの寸法430mm x 310mm x 53（！）Mm-これは通常のDVDプレーヤーより少し大きいです！ 本格的な_スケーラブル_コンピューターの場合、これは素晴らしいです。 電源（200ワット）の場合、ケースの重量は約3.5kgです。 最近レビューしたLebedev Optimus Maximusキーボードを覚えていますか？ 彼女の体重はわずか2倍です！ :)

リモートといえば。 このメディアセンターは兄弟の間で平均的です。 最年長者は、ステータスによって、機能的なリモートに依存していますが、それについてはインターネットでしか読んでいません。 リモートコントロールによってサポートされることが保証されているという事実にもかかわらず、 Logitech HarmonyOneリモートコントロールセンターと友達を作ることはまだできていません。

次に...リセットボタンは間違いなく正しいことです。 しかし、それは位置しているように見えますが、ややうまくいかないようです。 それは数回で、誤って本やディスクの入った箱を押してしまいましたが、それは最も都合の悪い瞬間でした。 一方、注文に慣れているだけです。







メディアセンターのカバーはつや消しの金属で、簡単に汚れませんが、傷を付けるのは簡単です-すぐに視界を損ねます。 蓋の上には絶対に何も置くことはできません（それでも、プラズマパネルの下のどこかで静かに使用できるように設計されています）が、この方法ではかなり広い領域が無駄に消えてしまいます。 したがって、メディアセンターを直立させておく必要があります。 垂直の壁掛けを考えています。
