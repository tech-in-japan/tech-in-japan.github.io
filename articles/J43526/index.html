Total Commanderに関する小さな記事



Habréでは、小規模な無料プログラムを含む記事が開催され、

ユーザーの人生、それでもトータルとそのプラグインに関する記事を書くというアイデアがありましたが、

ちょうど今登場した時間:-)





ポイント1。 システム



実行中のプロセスを管理し、各プロセスに関する詳細情報を取得することは、 AceHelperを使用すると非常に便利で、 TCBurnerを介してディスクを作成できます。 IEViewはTCに導体を「 埋め込む 」 ことに成功しました-まれにしかありませんが、非常に便利な場合もあります



Windows関数プラグイン：


デバイスマネージャー - デバイスマネージャー

ネットワークコンピュータ-NetworkAlt

レジストリ- レジストリ

イベントビューアー- イベントNT

サービス- サービス

スタートアップ- スタートアップガード



ポイント2。 マルチメディア



ACDSee、IrfanView、XnViewなど、画像を表示するための優れた小さなプログラムには、長い間、非常に緊急の問題がありました。それらは遅すぎて、標準のWindowsビューアは非常に不便です。 そして、このような小さなプログラムがSGViewerプラグインであることが判明しました-非常に迅速で、多くの便利な機能を備えています。

Mmediaが本当に私を助けてくれる瞬間に、ビデオをスクロールしたり、曲を再生したりする必要があることがよくあります。



ポイント3。 ホットキー



私見全体の主な利点の1つは、ホットキーです。 例を挙げましょう

一部の経験豊富なユーザーはそれらを知っていますが、初心者には馴染みがないかもしれません。

Ctrl + Q-隣接するパネルのクイックビュー（注目に値する、IEViewプラグインがある場合、フォルダーはWindowsエクスプローラーによって開かれ、ファイルはデフォルトで開かれるプログラムによって開かれる）

Ctrl + Shift + F1-隣接するパネルの新しいタブでカーソルの下のオブジェクトを開く

Ctrl + Enter-カーソルの下にあるオブジェクトの名前をコマンドラインにコピーします

Ctrl + Home-ディスクのルートパーティションに移動します

Ctrl + D-ブックマーク

Ctrl + Shift + F1-サムネイルモードで表示

Shift + F3-カーソルの下でオブジェクトを起動します（何かが既に選択されている場合は便利ですが、見る必要があります:)



ご清聴ありがとうございました:)
