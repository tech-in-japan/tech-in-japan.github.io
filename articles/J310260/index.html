3CX Enterprise Editionで導入



3CX Enterprise Editionは最も機能的な3CXエディションであり、Proバージョンのすべての機能とバックアップサーバーの追加ライセンスが含まれています。 Enterpriseエディションでは、スタンバイサーバーはホットスワップモードで動作し、プライマリサーバーに障害が発生するとすぐに動作するように切り替わります。 2番目のライセンスは、プライマリサーバーとバックアップサーバーの両方でアクティブ化される1つのキーです。 このソリューションは、フォールトトレランスシステムの可能な限り高速な動作を保証する必要がある大規模な組織で需要があります。 2番目のライセンスは、Proエディションでも使用されている組み込みの3CXフォールトトレランスメカニズムと連動します。 3CX Entrpriseと他のエディションとの比較はここにあり、それに切り替えるコストはここにあります 。







Chrome CallVia3CX Web Call Extension



この拡張機能を使用すると、マウスを2回クリックするだけで、CRMシステム、電子メールメッセージ、または任意のサイトから電話をかけることができます。 ページ上の番号をクリックすると、Windows用の3CXクライアントダイヤラーに自動的にコピーされます。その後、再度クリックしてダイヤルします。 手動で番号をコピーまたは入力する必要はありません！ どのWebページもCRMの拡張になります！ この小さな改善により、多くの電話をかけなければならない従業員の仕事が本質的に簡素化されます。













顧客との作業を大幅に加速し、実行される作業量を増やします。 新しいプラグインは、組織の作業プロセスにコミュニケーションを迅速かつ簡単に統合します。 従業員がすでに使用しているアプリケーションとWebサイトを使用します。 Chromeストアから新しいCallVia3CX拡張機能をインストールできます。







Zendeskカスタマーサポートの統合



FreshdeskおよびNutshellとの統合のリリースに伴い、Zendeskとの新しい3CX統合モジュールを紹介します！ このモジュールを使用すると、ユーザーは3CX ProをZendeskと統合して、次の利点を得ることができます。







  • Click-to-Call-Zendeskの顧客番号のClick-to-Callコール。 呼び出しは、3CXソフトウェアクライアントまたはサポートされているIP電話によって行われます
  • 呼び出しポップアップ-着信時にZendeskのクライアントカードが自動的に開きます
  • 発信者の番号が見つからない場合、システムはシステムの新しい連絡先を作成します。
  • コールジャーナル-すべてのコールはZendeskコールログに記録されます








新しいZendesk CRMプラグインは、3CX管理インターフェイスの[ 更新]セクションにあります。 3CXとZendeskを正常に統合するには、 このガイドをご覧ください 。
