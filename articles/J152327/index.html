VMwareおよびHyper-Vの無料ツール

本日は、仮想環境VMware vSphereおよびMicrosoft Hyper-Vの管理者の日常業務に役立つ5つの便利なフリーウェアユーティリティについてお話します。 これらのツールはすべて、Veeamによって開発されました。Veeamの製品は、VMwareおよびHyper-Vに基づく仮想化システムのバックアップ、レポート、および運用監視のタスクに重点を置いています。 今日は次のことについて話します：



  • VMwareおよびHyper-V用 Veeam Backup Free Edition
  • VMwareおよびHyper-V用 Veeam ONE無料版
  • Microsoft System Center用の Veeam Extended汎用レポートライブラリ
  • VMwareおよびHyper-V用 Veeamステンシル
  • Veeam Management Pack for VMware


それでは始めましょう：



1. Veeam Backup Free Edition for VMwareおよびHyper-V

Veeam Backupの無料バージョンを使用すると、次のことができます。

  • シャットダウンしたり、作業を一時停止したりせずに、仮想マシンをバックアップ、アーカイブ、またはコピーします。
  • 仮想マシンのバックアップからデータを回復します。
  • 仮想マシンとファイルを転送および管理します。


製品には次の機能があります。

  • VeeamZIP ： VMをオンデマンドですばやくバックアップして、 VMを動作、アーカイブ、または転送する機能。



  • インスタントファイルリカバリ（IFLR） ：イメージバックアップから直接個々のゲストOSファイルを復元します。 少数のゲストファイルのみが必要な場合、IFLRにより仮想マシン全体を復元する必要がなくなります。 バックアップでゲストファイルシステムを直接参照し、復元するファイルとフォルダーを選択するだけです。
  • ファイルマネージャー ：仮想マシンとホストの簡単なファイル管理：

    • 概要 ：Windowsエクスプローラーに似た使い慣れたグラフィカルインターフェイスで仮想マシンとホストファイルを参照します。
    • ホスト間のコピー ：たとえば、データストア間で仮想マシンとテンプレートをコピーします。
    • ホストとWindows / Linux間のコピー ：たとえば、ISOファイルをVMwareホストに配信します。
  • VMwareのクイックマイグレーション ：クラスターまたは共有ストレージを使用しておらず、vMotionがvSphereエディションで使用できない場合でも、実行中の仮想マシンを任意のホストまたはストレージに転送します。




2. VMwareおよびHyper-V用 Veeam ONE無料版



Veeam ONEは、仮想環境を24時間リアルタイムで監視し、リソースの使用を最適化し、管理用のレポートを文書化および作成するための製品です。 Veeam ONEの無料バージョンは、完全なデータ収集、複数ユーザーの同時作業のサポート、ダッシュボード、Microsoft SQL Serverデータベースの使用など、完全版と同じ基本機能をすべて提供します。 無料版では、ホストと仮想マシンの数、ユーザー数、またはデータアーカイブのサイズに制限はありません。 フリーウェアの制限は、主にレポート、複雑な通知のカスタマイズ、特定のダッシュボードの分野の高度な機能に関連しています。





3. Microsoft System Center用の Veeam拡張汎用レポートライブラリ



レポートライブラリ（GRL）を使用すると、物理環境と仮想環境の両方について、インフラストラクチャオブジェクトの状態とパフォーマンスを分析できます。 標準のMicrosoft System Centerレポートライブラリで提供されていない機能が含まれています。 Veeamの広範なレポートライブラリには、次のようなレポートが含まれています。

  • アラート統計レポート -2つのモードでトリガーされるアラートの統計の分析-ルール/モニターおよびオブジェクトごと。
  • 汎用パフォーマンストップ（下部）Nレポート -選択したパフォーマンスインジケーターのインフラストラクチャオブジェクト、インスタンス、またはその両方の視覚的表現。
  • パフォーマンスレポート -チャートおよび表のパフォーマンスインジケーターの値の視覚的表現。
  • パフォーマンス詳細レポート -詳細なパフォーマンスデータのトレンド。






4. VMware用Veeamステンシルおよび Visio用 Hyper-V



仮想インフラストラクチャで詳細な図を作成するためのMicrosoft Visioグラフィック要素のセット。 このセットには次の要素が含まれます。

  • ESXおよびHyper-Vホスト
  • データセンター
  • SCVMMサーバー
  • ローカルおよびネットワークストレージ
  • 論理ユニット番号（LUN）
  • 仮想マシンとそのステータス
  • ネットワークインターフェイスカード（NIC）
  • ネットワークなど

Visio要素のセットは、もともとVeeam ONE製品用に作成されたもので、特にVisioダイアグラムの形式でレポートを作成できます。



5. Veeam Management Pack for VMware



無料の10ソケットVeeam Management Pack（Veeam MP）は、すでにMicrosoft System Centerを使用しているが、Veeam MP製品のフルバージョンをまだ使用していないIT管理者向けに設計されています。 Veeam MPは、System CenterでVMware仮想環境を監視するために特別に設計されています。 このソリューションにより、System Centerの管理ポリシーとルールの均一性を維持し、Operations Managerコンソールを使用してインフラストラクチャ全体の状態を監視できます。 Veeam MPは、追加の監視プラットフォームを取得、展開、調査、およびサポートするコストを回避し、既存のプラットフォームを活用します。
