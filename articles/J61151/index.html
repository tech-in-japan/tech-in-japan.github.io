新しいMobile Mail.Ruエージェント：IM +メール+カード



Mail.Ru社は、JavaおよびSymbian向けのモバイルアプリケーションのラインアップを更新し、実際にモバイル「ハーベスター」に変えました。 現在、Mobile Mail.Ru Agentには、MRAプロトコルのサポートだけでなく、ICQ、Jabber、シンメールクライアント、モバイルカード、モスクワとサンクトペテルブルクの道路の最新の交通情報も含まれています。

更新されたバージョンは、 公式サイト （ Java | Symbian ）から、またはモバイルデバイス（ wap.mail.ru ）から直接ダウンロードできます。



さらに、Mail.Ru Agentには、MRAプロトコルとICQの両方を介して、多くのアカウントを操作する機能が含まれています。



Mail.Ruエージェントを使用しても、携帯電話に無料のSMSを送信できることを思い出させてください。



とにかく：MRAはオープンプロトコルです ;）







ダイアログウィンドウには大きな変更はありませんが、スキンのサポートにより、おそらく視覚的にはより快適になりました。







モバイルカードは便利です。渋滞表示モードがあります。 ガジェットにGPS受信機がある場合、またはモバイルオペレーターがそのようなデータを提供している場合、エージェントは現在地を表示します。 さらに、このような便利な機能もあります-あなたはあなたの場所について対話者に知らせることができます。
