SamsungがEMMC Pro Class 1500メモリモジュールを発売

サムスンは再びハイテクを大衆にもたらします。 最近、eMMC（組み込みマルチメディアメモリカード）16、32、64 GBメモリチップの生産開始に関する情報が、国際的なIT大手の陣営から寄せられました。



同社は、eMMC NANDフラッシュチップのパフォーマンスと「容量」を積極的に向上させ、最新のスマートフォン、タブレット、その他のモバイルデバイスの高い要件に対応しています。 現在、サムスンの工場はeMMC Pro Class 1500 NANDモジュールの量産を開始しています。







新しいシリーズは、プレミアムモバイルデバイス向けに設計されています。 Webブラウザ、3DおよびHDビデオのキャプチャと再生、マルチタスク、拡張現実、ソーシャルネットワークの使用、強力なゲームなど、幅広いアプリケーションのシステムパフォーマンスと使いやすさが向上します。



最新世代のメモリチップは、トグルDDR 2.0インターフェイスと、非常に人気のあるJEDEC eMMC 4.5標準をサポートします。これは、SDR-200インターフェイス（200 MHz、最大帯域幅-200 MB / s）など、生産性、効率、セキュリティ、および信頼性を向上させるための機能を標準化しています、キャッシュ管理、動的処理、ファイルクリーンアップ、電源オフ通知）。 順次読み取り速度は140 MBに達し、記録中、スループットは50 MB / sに達する可能性があります。 Samsungは、書き込み中にチップが3,500のランダムな読み取り/書き込みI / Oと1,500 IOPSを実行できると主張しています。 この数値は、すでに前世代のeMMCの機能の4倍です。







ハイエンドモバイルデバイスの薄型ソリューションに対する需要の高まりを受けて、Samsungは20ナノメートルプロセスのおかげで高密度メモリを作成できるようになりました。 新しいeMMCチップは、最大容量64 GB、厚さわずか1.2 mm、重量0.6グラムで、2つ、4つ、または8つのダイに積み重ねることができます。



NANDフラッシュメモリ市場でのサムスンの主導的地位を考慮すると、プレミアムスマートフォンとタブレットは、これらの同じ64 GBのメモリを最大1年間または2年間、最大構成で搭載します。
