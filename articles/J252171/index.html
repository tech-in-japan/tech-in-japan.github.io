3-2-1-行こう！ ロシア、ベラルーシ、カザフスタンでのマイクロソフト開発者ツアー





Windows 10のリリース、Visual Studio 2015のリリース、およびクラウドサービスのAzureとOffice 365の更新を見越して、開発者と個人的にコミュニケーションをとることが非常に重要であると判断しました。 したがって、私たちはさまざまな都市でイベントを開催するだけでなく、ロシア、ベラルーシ、カザフスタンの都市への技術的な遠征に行くことにしました。



イベント、ハッカソン、夜の集まりを行い、個々の企業、学生、スタートアップと会います。 遠征のために、特別装備のトレーラーも用意しています！











3月中旬から4月末まで、トレーラーは最大の都市を巡り、13のカンファレンスと4つのハッカソンを開催します。







イベント



都市スケジュール：

  • 3月18日-サンクトペテルブルク（+オンライン放送）
  • 3月21日-ミンスク
  • 3月25日-ロストフ・ナ・ドヌ
  • 3月28日-クラスノダール
  • 4月1日-サマラ
  • 4月4日-ニジニノヴゴロド
  • 4月8日-カザン
  • 4月11日-パーマ
  • 4月14日-エカテリンブルグ
  • 4月17日-チェリャビンスク
  • 4月22日-ノボシビルスク（+オンラインブロードキャスト）。
  • 4月25日-トムスク
  • 4月28日-アスタナ




イベントでは、開発分野の主要な専門家からWindows、Web、クラウドのトピックに関するレポートを受け取ります。



アプリケーション開発：

  • Windowsプラットフォーム用のユニバーサルアプリケーション
  • Windowsプラットフォーム向けのゲーム開発
  • クロスプラットフォームアプリケーション開発
  • 開発サイクル管理




Web開発：

  • 新しいASP.NET
  • Visual StudioでのWeb開発
  • Microsoft Office用アプリケーション（365）
  • JavaScript / TypeScriptおよびHTML




クラウド開発：

  • Microsoft Azure
  • モバイルアプリケーション用のクラウドバックエンド
  • Azure for IoT
  • クラウド内のDevOps




また、イベントでは、Windows用デバイスの最新モデルの展示会にアクセスし、パートナーのオファーに精通し、非公式の設定で専門家とチャットできます。 また、選ばれた参加者には、予告編に入る特別な機会が与えられます。



ハッカソン



4つの都市でのイベントのメインプログラムに加えて、夜間ハッカソンを実施しています。



  • 3月21日〜22日、ミンスク-ゲームハッカソン
  • 4月4〜5日、ニジニノヴゴロド-IoTハッカソン
  • 4月11〜12日、パーマ-モバイルハッカソン
  • 4月25〜26日、トムスク-モバイルハッカソン




パン



さらに、誰もがMicrosoft Developer Tourからユニークなお土産を受け取る機会があります！ 予告編を背景にクリエイティブな写真を作成し、ハッシュタグ#devtourcontestでソーシャルネットワークに配置するだけです。 コンテストの詳細は、3月15日からサイトで公開されます。



登録



ツアーの参加者の日記も表示されるウェブサイト-msdevtour.ruで、遠征のほか、あなたの街のイベントやハッカソンに登録できます。ロシアの「IT」の場所に関する開発者向けのユニークなガイドが表示されます。サンクトペテルブルクとノボシビルスクでのオンライン放送にプレーヤーが利用できるようになります。



イベントの登録は既に開始されています！
