Raspberry Piで特定された飛行物体を追跡する方法

Raspberry Piを使用すると、識別された飛行物体を追跡できることをご存知ですか？ 最大400 kmの距離にある航空機の無線信号に同調して、すべてのフライトを追跡できます。 これを行うには、安価なUSBテレビと数分間の無料の時間を見つけるだけです。













画像：dump1090-4分の1波長のフレキシブルアンテナと缶アンテナに対してFlightAwareアンテナをテストしています。







この記事では、フライトトラッキングの簡単な紹介、ソフトウェア、ハードウェアの概要、そして最も重要なこととして、必要な用語とスラング表現を紹介します。 また、Dockerとコンテナー化テクノロジーがIoTプロジェクトのソフトウェア管理にどのように役立つかについても説明します。







FlightAware.comなどのサイトでは、クラウドソーシングを使用して何万台もの航空機を追跡できます。世界中の人々がRaspberry Piコンピューター（価格は35ドル）と安価なUSB TVチューナーを使用してタスクを実行します。







個人的な目的でのみフライトを監視することも、 FlightAware.com 、 FlightRadar24 、 PlaneFinder.netなどのサイトで少し貢献して、見返りに詳細なインジケータを取得することもできます-レーダーデータやその他のニシュティク。













画像：FlightRadarは、英国の空域に飛行機を示しています。







なぜフライトを追跡するのですか？







いくつかの理由があります。







  • これは、ほこりの多いパントリーで古いRaspberry Piを使用して実行できる超安価なアプリケーションです。
  • この情報をオンライントラッカーに転送すると、フライト追跡のローカルリーダーになることができます。
  • 詳細な統計を収集し、最適なアンテナとその設置場所を選択できます。
  • ソフトウェアプロジェクトのデータを収集できるだけでなく、航空に関する多くの興味深いことを学ぶこともできます。


そして最も重要なこと-それはあなたがあなたのPiでクランクアウトし、すべての作品から即座に満足を得ることができる単なる楽しいプロジェクトです。 同時に、そのコストはゼロになる傾向があります。







何が必要ですか



  • Raspbian Lite搭載のRaspberry Pi SDカード







  • Raspberry Pi 2または3は、メモリが多く、マルチタスクに適しているため、より適切に使用します。







    • 複数のレシーバーを設置してお金を節約しようとする場合、Pi Zeroも非常に適しています。


  • USB TVチューナー-発行価格は8〜30ドルです。







    • ModMyPiのFlightAware Pro Stickは、ノイズフィルター、SMAコネクタを内蔵した最もクールなバージョンです。 1090 MHzアンテナは別途購入する必要があります。
    • Pimoroniの汎用DVB-T USBチューナー -テスト用のサンプルを送ってくれました。 チップセットは正しく動作しますが、テレスコピックアンテナは最小のコネクタに取り付ける必要があります。
    • NESDR Smart Premium SDR -3つのアンテナ、太いSMAケーブル、および都市環境向けのビルトインノイズフィルターが含まれています。
    • お金を節約したい場合、eBayにはもっと安価なオプションがたくさんありますが、選択するときは注意してください。デバイスはR820Tチップセットで動作するはずです。




結果が必要な場合は、有名ブランドの機器（DVB-Tスティック）を購入することをお勧めします。 上記のリンクはPRアフィリエイトプログラムではありません。







  • アンテナ


ほとんどのDVB-Tチューナーには、1090 MHzで必要なADS-B信号ではなく、陸上でTV信号をキャッチするアンテナが付属しています。







  • 1090 MHzアンテナが必要な場合は、SMA機能を備えた3 dBiのホームバージョンを購入できます。 私がテストしたアンテナは、窓辺に置いた場合、240 kmの距離で飛行しました。







  • FlightAwareアンテナは、窓から外に出したときに、400 kmを超える距離にある市街地で信号をさらに突然キャッチしました。 屋根の上に置くと、結果はさらに涼しくなります。







  • また、PiおよびTVチューナーを十分に確保するために2.5-3アンペアの電源も必要です。 電源のサプライヤであるPimoroniの公式Webサイトで何かを入手できます。


ModMyPi Webサイトでは、必要なキットをすべて1本で購入することもできます。







用語集



したがって、私たちの目標は、特定の範囲のフライトのADS-BブロードキャストをキャッチするようにUSB TVチューナーを構成することです。 最初に、フライト追跡の分野からいくつかの定義と用語を定義しましょう。







ADS-B



現代の航空機には、ナビゲーション機器から情報を収集し、ADS-Bを介して環境に送信する自動トランスポンダーがあります。 この情報は暗号化されていないため、誰でも受け取ることができます-ディスパッチャ、他の飛行機、Raspberry Piの所有者。

ADS-B（自動従属監視放送）は、コックピットのパイロットと地上局の航空管制官の両方が、以前よりも正確に航空機の動きを観察し、航空情報を取得できるようにする技術です。 ソース- ウィキペディア 。







DVB-T



必要なTVチューナーはDVB-Tと呼ばれ、デジタルビデオ放送-地上波を表します。 これは、地上デジタルテレビの欧州規格です。 このデバイスをテレビアンテナとして使用して、お気に入りのテレビ番組を視聴することもできます。 すべてのDVB-Tデバイスを空気信号用に構成できるわけではないため、推奨されるデバイスのいずれかを選択するか、購入前に特性を慎重に検討することをお勧めします。







SMAアンテナコネクタ



ウィキペディア： サブミニチュアバージョンA これらのコネクタは同軸よりも小さく、通常プレミアムまたはターゲットDVB-Tを備えています。 安価なDVB-Tには、おそらく小さなコネクタが付いています。 PigetailはeBayまたは電気店で購入できます。 同軸、SMA、またはRFなどの大きなアンテナコネクタを組み合わせる必要があります。







ダンプ1090









ADS-B信号をデコードするための重要なコンポーネントは、dump1090ソフトウェアです。 この数は、作業の頻度を意味し、dump-実行するコマンド-は、生データを復号化してダンプします。







dump1090アプリケーションは、新しい機能や改良された古い機能を導入したさまざまな人々のために、いくつかの分岐点を持つオープンソースプロジェクトです。 適切なフォークを選択することは非常に困難です。







GitHubのデータを使用して、このアプリケーションの履歴を再作成しました。







  • Antirezは、クリスマス休暇のために2012年にプロジェクトを開始しました
  • MalcolmRobbがバトンを引き受けました-コードで新しいフォークを起動し、いくつかの機能を改善しました
  • Mutabilityが MalcomRobbからforkを起動
  • FlightAwareは、可変リポジトリからのフォークもサポートしています


Dockerを使用してコードをビルドしますが、必要に応じて、ターミナルで個別に同じ方法でコマンドを実行できます。 Dockerコンテナを使用する理由は次のとおりです。







  • 再生可能なビルドスクリプト
  • READMEページよりもはるかに便利な、信頼できるコードアセンブリメカニズムを提供します。
  • 異なるバージョンのコードを切り替えることができます
  • Piには直接何も置かれないため、デバイスは元の形式のままです
  • Piを持っている友人と画像を共有できるようにします


dump1090のほとんどのバージョンには、特定の範囲の飛行機をリアルタイムで表示できるWebインターフェイスもあります。







フライトウェア



FlightAwareは、dump1090からデータを収集するいくつかのアグリゲーターサイトの1つです。レーダーの仮想視覚化を使用して、追跡に役立つフライトの詳細な統計を収集し、作業範囲を調べることができます。







ここに私のプロフィールで見た私の結果があります。 専用のアンテナと防音DVB-Tチューナーを使用しました。













また、FlightAwareプロファイルを表示して、データを増やすこともできます。







MLAT



MLAT（略称Multilateration）は、ADS-Bデータを送信しない航空機を追跡するために多くの地上局を使用できる技術です。 FlightAwareの記事で詳細を読むことができます。







これは、信号の到着時間の差の推定に基づいています。そのまま使用できるため、ADS-B単独でよりもはるかに多くのフライトを追跡できます。

FlightAwareをインストールする



FlightAwareで始めたフライト追跡サイトはいくつかあります。 彼らのソフトウェアはdump1090コードに接続し、データをサーバーに転送します。サーバーでは、すでに統計を確認し、他の参加者とデータを比較できます。







.debファイルをPiに直接インストールできますが、別のDockerfileを作成しました。 2つの利点があります。ソフトウェアの2つ以上のコピーを実行し、Piをフラッシュせずにバージョンを切り替えることができます。







次に、次のコマンドを使用して画像を収集するか、 docker pull alexellis2/flightaware:3.5.0



を使用して画像をダウンロードします。 







 $ cd eyes-in-the-sky/flightaware $ docker build -t alexellis2/flightaware:3.5.0 .


行末のドットに注意してください。見逃さないでください。







FlightAware.comに登録し、ユーザー名とパスワードを設定します。







piaware.confファイルを編集して、次のフィールドを置き換えます。







  • receiver-host（PiのIPアドレスを使用）
  • flightaware-user
  • Flightaware-password（登録時に提供されたデータを使用）


FlightAwareには、MACアドレスでRaspberry Piを追跡できる便利な機能があります。 幸いなことに、DockerではMACアドレスを置き換えることができるため、ソフトウェアのコピーをいくつか実行できます。 その場合は、MACを変更して、コピーごとに一意になるようにします。







次に、イメージを実行してログを確認します。 







 $ cd eyes-in-the-sky/flightaware $ docker rm -f piaware_1 $ docker run --mac-address 02:42:ac:11:00:01 -v `pwd`/piaware.conf:/etc/piaware.conf --name piaware_1 -d alexellis2/piaware:3.5.0

質問への回答



  • 電気の消費量はいくらですか？

    Pi Zeroまたは2/3は、アイドル時に2〜3ワットを消費します。 dump1090アプリケーションは、Piプロセッサを使用します。これは、ゼロでの容量の最大50％であるため、負荷とUSB DVB-Tに追加の電力が消費されることに注意してください。







  • USBバッテリーから操作をクランクすることは可能ですか？

    はい、期間限定です。 バッテリーは通常3日間十分でしたが、フライトを追跡するために使用したときに3時間で残りました。







  • 太陽エネルギーを使用することは可能ですか？

    太陽エネルギーは正しい決定ではないかもしれません。 Raspberry Piは、ソーラーパネルから直接確実に動作しません。 充電コントローラー、適切なサイズのソーラーパネル、数日間続くバッテリーなどの高度な機器が必要になります。

    防水ケースを備えたPower over Ethernetを介してPiを接続するのが最善です。 ここに必要な詳細のリストがあります 。







  • ボックス化されたソリューション-イメージまたはISOはありますか？

    FlightAware WebサイトのSDカードで完全なイメージを見つけることができますが、モジュラーコンポーネントからシステムを構築する場合、dump1090用に開発された他のソフトウェアよりも有利になります。







  • 機器の詳細な概要はありますか？
