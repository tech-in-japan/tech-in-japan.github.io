Appleは新聞でサムスンに謝罪を印刷した



最近、アップルはそのウェブサイトでサムスンに「謝罪」し、サムスンがこれを見ない方が良いように謝罪しました。 裁判所もこれに不満を抱き、アップルにサイトの謝罪を書き直させました 。 公式ウェブサイトにはこれまで謝罪はありませんが、AppleはすでにAppleの新聞で謝罪メッセージを送り始めています。



英国支部は、印刷版のグラフィックユニットにガーディアンを支払い、そこに次のようなテキストを投稿しました。

2012年7月9日、イングランドおよびウェールズ高等裁判所は、Samsung Electronics（UK）LimitedのGalaxyタブレットコンピューター、つまりGalaxy Tab 10.1、Tab 8.9、およびTab 7.7は、Appleの設計権第0000181607-0001の違反を示していないと結論付けました。 高等裁判所の判決の完全なコピーは、 www.bailii.org / ew / cases / EWHC / Patents / 2012 / 1882.htmlで入手できます。 評決は欧州連合全体で有効であり、イングランドおよびウェールズ控訴裁判所により10月18日に承認されました。 控訴裁判所の判決のコピーは、 www.bailii.org / ew / cases / EWCA / Civ / 2012 / 1339.htmlで入手できます。 ヨーロッパのどこでも、会社の登録デザインの禁止は施行されていません。


今回はAppleはそれをからかいませんでしたが、公式の英国Apple Webサイトでの謝罪メッセージは11月3日までにまだあるはずです。
