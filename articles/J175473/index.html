Youtube

ビデオが読み込まれるのを数秒間待つ必要があります> 360p、視聴中のクリッピングが一般的になり、これはかなり広いチャンネルです。







その理由は、プロバイダーがビデオキャッシュサーバーの速度を過小評価していたため、必要なのはそれらへのアクセスをブロックすることだけです。



リクエストがYouTubeサーバーのcdnを通過するには、IPアドレスの範囲（ルーターまたはコンピューター上）をブロックする必要があります。



173.194.55.0/24および206.111.0.0/16



ファイアウォールが有効になっているWindows（xpを除く）では、cmdを管理者権限で開くだけで十分です。 



netsh advfirewall firewall add rule name="youtube" dir=in action=block remoteip=173.194.55.0/24,206.111.0.0/16 enable=yes

iptables -I INPUT -s 173.194.55.0/24 -j DROP
