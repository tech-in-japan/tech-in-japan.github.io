HTML5およびCSS3サポート付きのExpression Web 4 Service Pack 1がリリースされました

Expression Web 4 Webサイトの設計および開発パッケージは 、サービスパックとして更新されました。 このパッケージは、特に、最新のWeb標準をサポートする新しいツールであるHTML5とCSS3をパッケージに表しています。



Service Pack 1には、HMTL5仕様全体のIntelliSenseサポートが含まれています。 さらに、Expression Web 4パッケージの一部のエディターは、たとえばborder-radius（下のスクリーンショット）などのCSS3標準プロパティのサポートを追加しました。







さらに、Expression Web 4は、WebページのスナップショットのソースとしてInternet Explorer 9のサポートを追加します。 また、さまざまなブラウザー用の同時マークアップツールSuperPreviewは、ブラウザーのChromeとIE9のレンダリングの大規模な更新とサポートを受けました。 SuperPreviewは、パスワードで保護されたページのサポートも追加しました。ユーザーはこれを長い間リクエストしています。



Expression Web 4 SP1では、PHPのサポートが改善され、PHP5の何百もの関数のサポートが追加され、ユーザーインターフェイス機能が改善されました。



些細なこととして：InetlliSenseのサポートは、特別な文字とコード（左側）と、ページプレビュー機能の欠落しているほとんどすべてのメインブラウザー（右側）に追加されました。
