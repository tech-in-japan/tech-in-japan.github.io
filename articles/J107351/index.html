Blekko：スラッシュタグを使用した検索エンジン（カテゴリ）

最後に、Richard Skrentaの指揮の下で開発されたオリジナルのBlekko検索エンジンが、パブリックベータテスト用にリリースされました。



リチャード・スレンタは、1982年にパーソナルコンピューター用の最初の大規模ウイルスであるElk Clonerウイルス （当時はApple II）を書いたときにハッカー界で有名になりました。 若い才能はわずか15歳で、彼は学校で勉強しました。



80年代後半から90年代にかけて、SkrentaはAmiga Unix（Commodore Business Machines）、Unix System Labs、およびSun Microsystems（IPレベルで開発された暗号化システム）で働いていました。 後に彼は、Netscapeによってすぐに買収されたオープンディレクトリOpen Directory Projectの創設者の一人でした。 その後、彼は辞め、 Topixニュースアグリゲーターを設立しました。これは2005年にアメリカの出版コンソーシアムに売却されました。



Blekko検索エンジンは、彼の新しいスタートアップであり、元同僚であるMark Andressen（Mosaicブラウザーの作成者であり、Netscapeの共同設立者）の投資資金で作成されました。 一般的に、背景から判断すると、プロジェクトは有望です。



過去のすべてのSkrentaプロジェクトと同様に、Blekko検索エンジンにも「オタク的」なタッチがあります。 主な機能は、 スラッシュタグを使用した検索クエリの改良です。 さらに、スラッシュタグの指定は、オートコンプリートの段階で既に提供されています。 非常に便利で直感的な機能。



このシステムは、前述のOpen Directory Projectディレクトリの影響下で明確に作成されたことに注目してください。 すべてが可能な限り構造化され、各スラッシュタグ（読み取り、テーマカテゴリ）には独自のエディターがあり、登録ユーザーは新しいスラッシュタグを作成し、検索インデックスからサイトを含めることができます。 これは、「個人のテーマ検索エンジン」のようなもので、すべての人に公開することも、閉じたままにすることもできます。







数百のテーマスラッシュタグが既に作成されています。 それらに加えて、組み込みのスラッシュタグもあります。 以下に例を示します。



  • /人-個人ページでの配信の制限
  • /日付-結果を日付順に並べ替えます
  • /ブログ/フォーラム-検索ブログまたはフォーラム
  • /ランク-各ページのランキング評価（ランク）を表示します
  • / dr = 2005-2006-2005年と2006年のページで検索
  • / noporn-「アダルト」サイトからの結果のフィルタリング」
  • /地図
  • /在庫
  • /トラフィック
  • /天気


スラッシュタグは、従来の検索に新しい機能を追加します。 たとえば、[bp / liberal]または[bp / conservative]と入力して、さまざまな政治的観点からBPについて読むことができます。 スラッシュタグの概念については、ビデオで詳しく説明しています。







GoogleもBingもこれを持っていません。
