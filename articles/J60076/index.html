Google Web Application Secrets Book

こんにちは。 私の名前はデニスバルエフで、Googleのウェブアプリケーションに関する本を書いています。 匿名のアルコールクラブのメンバーへの告白のようですね。 私も好きです。 本の作成方法については、あなたの許可を得て、このブログで説明します。 草稿の最初の章から、おいしいオリジナルの匂いがする著作権で保護されたコピーの受け取りから最初の読者からのフィードバックまで、本がゼロから作成される方法について、作家のリアリティショーを提供します！



私はプロではない作家です。 2年生から私はプログラマになることを夢見ていたが、10年経った今でも1人になった。 エンタープライズ管理システム（ERP）および顧客関係管理（CRM）を開発および実装しました。 それから私は生産プロセスの最適化に携わりました。そのため、カイゼン、そして申し訳ありませんが、ムダとは何かを直接知っています。 さらに、私は定期的にさまざまな雑誌に技術的なトピックに関する記事を書きました。そのほとんどは学校から愛されているコンピューター・コンピューターによって出版されたものです。 また、If誌で最高のSFストーリーのコンテストに勝ちました。 それ以来、Yandexは私のストーリー「Latecomers」の音声バージョンへのリンクでいっぱいになっています 。 ブログLifeHack.ruの著者として私を知っている人もいます-Runetで「ハッキングライフ」をテーマにした最初の人の1人。



Googleに関する本を書くことにしたのはなぜですか？ おそらく、私が間違いなく持っている利点の1つ（一般にそれらはほとんどなく、謙虚さは明らかに適用されません）は、単純で理解可能な言語でかなり複雑なことを説明できることです。 このスキルの出現は明らかです。事実は、私自身...あまり理解していないということです:)そして、たとえ自分に何かを説明できたとしても、誰にでも説明できます！ 驚くべきことに、このスキルは私が好きなものに関連してのみ機能します-明らかに、これには何らかの心理的なトリックがあります。 私はGoogleサービスを作成してからほとんど使用してきました。それらは私にとって便利で、パワフルで、モダンで、本当に美しい方法で作られています。 好きですよね 同時に、Googleサービスに関するロシア語の良い本は1つも存在しません。 いずれにせよ、OzonもYandexも、私の家の近くの書店の売り手もそれらについては知りません。 また、ウェブサービスとは何なのかよくわからない人にとっても理解しやすく便利になるように、Googleサービスについてお話したいと思います。 母が自分でGmailアカウントを開設したとしたら、本を書きたいと思います。 Gmailチャットを介して彼女とチャットすることは非常に便利です。



私は以前に1冊の本を書いたことはありません（研究所の最初の年に始めてほぼ終わりに近づいたSF小説を除きますが、絶望的に最後にはまりました。出版社とのコミュニケーションでは、市場で本を宣伝する方法がわかりません。また、本とその出版物を書く方法に現れる百万種類の「微妙な瞬間」については、おそらくわからないでしょう。 さらに興味深いのは、これらのことに専念するブログの読者です！ 本の将来の読者のアドバイスも私にとって有用です-それに現れるべきものと、決して存在しないべきものを一緒に議論しましょう！ 章の抜粋、Googleの世界からのニュース、 GAppsBook.ruのトレーニングビデオを投稿します 。 ちなみに、Googleドキュメントのアンケートフォームに関する最初のビデオは次のとおりです。
