Aegis Secure Keyレビュー

随時、ハードウェア暗号化を備えた保護されたフラッシュドライブに関するニュースがすり抜けました。 時間が経ち、縫い付けられたものが成長し、私が気付いた時が来ました。 まず、Yandexマーケットを開き、高度な検索を通じて、マーケットが提供するものを調べ始めました。 オファーのリストは数枚のシートでした。 詳細を検討するにつれて、基本的な要件を強調しました。 おそらく最も重要なのは、どのOSからも追加のプログラムを使用せずに作業することです。 したがって、ほとんどのフラッシュドライブは、PCに接続すると、パスワード要求とコンテナーのさらなる復号化を伴う組み込みプログラムを起動するため、ほとんどすべてのオファーがビューから即座に消えました。 そして、ご想像のとおり、このアプローチは窓の下でしか機能しませんでした。 その結果、物理的なキーボードを備えたデバイスが視野内で回転し始めました。コードを入力し、USBに挿入して作業しました。 Yandexが提供したのはCorsair Flash Padlock 2だけでしたが、多くのセキュリティパラメータについては、私には向いていませんでした。 インターネット上で検索すると、ロシア語ではそのようなデバイスに関する情報が事実上ないという事実にかなり驚いた。 ハブでスキップされた素晴らしいIStorage datAshurフラッシュドライブに関する投稿 印象的。 しかし、これはタスクではありません。 そのようなデバイスは、ロシアでの販売が禁止されています。 アマゾンのイーベイ-検索では目的の結果が返されませんでした。 そして、Aegis Secure Keyに出会いました。 外観と説明-1人のdatAshurに1人、別の会社からのみ。 Ebay-結果0、Amazon-結果は正です。 16Gb-125ドル。 待ち、通知、ターン、喜びの月。









新しいおもちゃを初めて手に持つとき、最初に気をつけなければならないのは、金属ケースの心地よい重量と軽い涼しさです。 ケースと...インターホンを静かに取り外し、ソフトタッチに似た素材で体全体を覆います。



前面には、10個のデジタルボタン、1個のシステム、3個のLEDがあります。



一方、16gbフラッシュドライブはAES 256ビット暗号化を使用し、USBから充電されるリチウムポリマーバッテリーで動作することを示す有用な情報があります。 仕様は水と衝撃に対する保護を示しているという事実にもかかわらず、私はまだそれを確認する準備ができていません。



ボタンバーに戻ります。 フラッシュドライブの物理的寸法は（ケースと合わせて）80x20x10 mmであるという事実にもかかわらず、ボタンは十分なサイズである5x5 mmで非常に成功したことが判明しました。 私の指は大きくないので、ボタンを押しても問題はありません。 ボタンにはかなりきつい、触覚的な動きがあります。 軽く押すだけで押し心地がいい。 ボタンの素材は、硬くて硬いゴムでできており、その上には、爪などの硬い物体からの痕跡（へこみ）はありません。



さて、それがどのように機能するかを見る時間です！ 私たちはUSBに固執します...、赤いLEDが数回点滅し、次に青いLEDが絶えず点滅します... ...ああ、USBデバイスが認識されません。



指示を開き、工場出荷時のパスワードを確認し、キーを押して、11223344を入力し、もう一度キーを入力します。 緑色のLEDが友好的に点滅し、ドライブをポートに接続するのに30秒かかりました。 新しい機器のインストール、10秒、タスク選択のウィンドウ。 プロパティを開きます。



700メガバイトの映画を投げようとしています。 速度は42mb / sで始まり、21mb / sで終わります。 読み取り速度は12 mb / sを超えません。 明らかに復号化はオンザフライで影響します。 fat32もntfsも大きな違いはありません。



ハッキング：

総当たり攻撃に対する保護を確認してください。 間違った試行を10回行い、点滅せずに赤いLEDが点灯します。コードを入力するための試行回数はこれ以上ありません。 新しいパスワードをプログラムし、USBとVOILに接続し、ディスクをフォーマットするように求められます。

もちろん、私はフラッシュドライブを開けませんので、メーカーを信じます。 製造業者によると、内部のすべてが硬質エポキシ樹脂で覆われているため、チップを取り外そうとするとチップが破壊されます。

熱直腸暗号解析？ 私の知る限り、データ削除コードを設定する方法はありません...



結果。

比較的高品質で信頼性の高い製品で、比較的安価で、コンパクトでスタイリッシュです。 デバイスの紛失または盗難時の優れたデータ保護。 パーティション分割して何が起こるかを確認したかったのですが、ハードドライブを操作するプログラムでは表示されません。 私は質問に答えてうれしいです。
