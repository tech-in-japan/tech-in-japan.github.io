ロシア初のマスセグメントマザーボード

すでに書いたように、マスセグメントの国内サーバー製品のローカライズの度合いは、コンポーネントの生産よりもさまざまなサービスの提供においてより多く表されます。 サーバーコンポーネントは伝統的に中国で製造されており、どのブランドに属していません。 あるロシアの会社は完全に国内のプラットフォームを作成しようとしましたが、このプラットフォームはサーバーの古典的な使用には過剰な特性を備えていたため、製品はニッチであることが判明しました。



しかし、今では、年の初めに、ロシアで設計および製造されたサンプルサーバー2プロセッサマザーボードを受け取りました。 このモデルは、主に特性とコストの点で、大量適用に完全に適していたという事実により、関心を呼び起こしました。 この記事は、ロシアの最初のマザーボードRikor R-BD-E5R-V4-16.EAのレビューとテストに専念しています。



特徴



  • CPUサポート：Intel Xeon E5-2600 V3 / V4シリーズ（最大135W）
  • MBチップセット：Intel Wellsburg C612PCH
  • メモリスロット：16（各CPUは4 x DDR4チャネルx 2 DDR4 DIMMをサポートできます）
  • メモリサポート：最大1024GB DDR4 2133/1866/1600/1333 RDIMM / LRDIMM / NVDIMM
  • PCI-Eスロット：6
  • PCH SATAポート：10 x SATA3.0
  • HDD RAID機能：RAID 0、1、5、10
  • グラフィック：Aspeed AST2400
  • IPMI：仮想メディアオーバーを使用したIntelligent Platform Management Interface v.2.0のサポート

    LANおよびKVM over LANはASPEED AST2400 BMCをサポートします
  • ネットワーク：5 x 1Gb LAN（1 x 1Gb LANはBMC専用管理ポート）
  • USB：5 x USB 3.0（1 x内部）+ 4 x USB 2.0（内部）
  • DOM：SATA DOM電源コネクタをサポート
  • 入力/出力インターフェイス：1 x COMポート、1 x VGAポート、1 x専用IPMIポート、4 x 1000Mbpsポート、4 x USB2.0 + 4 x USB3.0
  • ファン電源ソケット：8 x 4ピンスマートFAN
  • 電源インターフェース：1 x 24ピンSSI電源インターフェース電源インターフェース+ 2 x 8ピンSSI 12V電源インターフェース

おわりに



Rikor R-BD-E5R-V4-16.EAに基づくシステムは、ベンチマークの遅れにもかかわらず、すべてのサブシステムの毎月の負荷テスト中に高い安定性と信頼性を示しました。 マザーボードのIPMIインターフェースはIntelのクラシックです。これは、たとえばAsusマザーボードでも見ることができます。 BIOSは国内の秋に計画されました。 マザーボードの開発と製造（もちろん、コンポーネントを除く）は、ここロシアで繰り返します。



マザーボードに加えて、完全にロシア製の2Uケースがテストのために来ました。 ホットスワップ可能なバスケットを含むすべて（PSUを除く）は、我が国で生産されています。 バックプレーンも。 なぜなら ケースは非常に非常にモデルであり、多くの苦情が出ましたが、この問題は執筆時点ですでに修正されています。 したがって、ケースのレビューは少し後で行われます。
