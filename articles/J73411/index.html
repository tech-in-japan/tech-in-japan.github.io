Conjoon-Zend FrameworkおよびExtJS 2.2のパーソナルオーガナイザー

Conjoonは、メッセージングに特に重点を置いたWebベースのパーソナルオーガナイザーです。 最大限の実用性、使いやすさ、情報の一般的な認識のために、システムはZendFrameworkとExtJS上に構築されています。





私の場合は、 gmail.com 、 FeedReaderおよびMS Outlookをゴミ箱に捨てたと言います。 現在、 Google Chrome + Conjoonのみ！



仕様：

  • Mozilla FireFox > = 3.x
  • Internet Explorer > = 7
  • Safari > = 3.x
  • Google Chrome
  • Opera > = 9.x


パッケージバンドル



  1. 自分自身をインストールし、必要に応じてオーガナイザーを最新バージョンに更新するインストーラー。
  2. ZendFrameworkおよびExtJSライブラリ。
  3. Conjoonアプリ自体


職場



  • 必要なデータにすばやくアクセスするためのメニュー
  • システムメッセージまたはプロンプトと進行状況バーのあるステータスバー
  • データはサブタブに表示されます。
  • 開いているタブまたはモジュール間をすばやく移動するためのブックマークメニュー




クイック起動



  • 手紙や連絡先をすばやく編集する
  • 最近のメールを見る
  • フィードから新しいアイテムを表示する


メールモジュール

  • POP3 、 SMTPプロトコルのサポート
  • UTF-8-エンコード
  • フォルダの移動（ ドラッグ＆ドロップ ）、名前変更、削除、作成の機能：

    • 受信トレイ
    • スパム
    • 発信
    • 送信済み
    • 削除しました
    • ショッピングカート


  • フォルダーを非表示/表示する機能
  • プレビューレターを非表示/表示する機能
  • ビューポートをウィンドウの左または下に移動します
  • メールの作成、保存、編集
  • ビジュアルテキストエディター
  • 単純なドラッグアンドドロップ （ ドラッグアンドドロップ ）でフォルダー間でメールを移動する機能
  • メールのショートカット

    • 最後
    • 既読/未読
    • スパム/非スパム
    • 回答済み/転送済み


  • メッセージ作成中の個人アドレス帳による受信者の自動検索
  • スマイリーのサポート
  • 設定されたアカウントでメールを自動的に受信する
  • メール署名のサポート
  • （理論上）無限の電子メールアカウントのサポート


RSSフィードモジュール



  • ATOM / RSSフィードのサポート
  • 最近の投稿を自動的にダウンロードする
  • ショートカットの読み取り/未読


必要条件



  • mod_rewriteモジュールと.htaccessファイルをサポートするApache
  • PHP > = 5.2.5
  • MySQL 5. *
