Xoom 2およびXoom 2 Media Editionタブレットを正式に発表





これは間違いなくその日の最も衝撃的なニュースではありませんが、それでも本日、モトローラはXoom 2およびXoom 2 Media Editionタブレットを公式に発表しました。 Xoom 2には、1.2 GHzデュアルコアプロセッサ、Android 3.2オペレーティングシステム、1 GBのRAM、16 GBの内部メモリ、およびCorning Gorilla Glassコーティングを施した10.1インチディスプレイが搭載されています。 モトローラはディスプレイのタイプを指定していませんが、表示される角度は178度であるため、IPSのように見えます。 デバイスには、専用のアプリケーションMotoCastが事前にインストールされています。これにより、追加のツール、プログラム、またはワイヤーなしで、タブレットとコンピューターの間で音楽、写真、その他のコンテンツを交換できます。



タブレットには、Wi-Fi b / g / n、Bluetooth、GPSモジュールなど、ほぼ完全な通信オプションセットがあります。 両方のデバイスには、3.5 mmヘッドフォンジャック、microUSBポート、HDMIが装備されています。 メーカーによると、Xoom 2の自律性は10時間であり、Xoom 2 Media Editionはわずか6時間です。 3Gサポートの証拠はありませんが、Xoomの経験から判断すると、Motorolaにはこのオプションを備えたモデルが必要です。







Xoom 2の寸法は253.9 x 173.6 x 8.8 mm、Xoom 2 Media Editionは139 x 216 x 8.99 mmです。 それぞれ599および386 gのタブレット。



棚でのこれらの新製品の販売開始時には、Bluetoothキーボード、ドッキングステーション、スタイラスで構成される一連のワークアンドプレイが提供されます。 モトローラは、これらのタブレット向けの多くのビジネスアプリケーションを立ち上げる予定です。







大手小売業者のCarphone Warehouseによると、Motorola Xoom 2は新しい年の前に発売され、16 GBのメモリとWi-Fiでの修正に約400ユーロの費用がかかります。
