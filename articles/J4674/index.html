Firefox 2：スペルチェックの改善

最初は、辞書は完全ではなく、インターネット上で広く普及している多数の現代用語を知りません。 これは、毎日ブログ投稿を書いたり、Habréのトピックにコメントしたり、メールWebインターフェースを使用してたくさんチャットしたりする場合、あまり便利ではないかもしれません。 ある程度、これは追加の辞書をインストールすることで解決できます（たとえば、 ここから ）。

ただし、スペルチェッカーのパフォーマンスを向上させる方法は他にもいくつかあります。 これらの方法はすべて、ディレクトリ「 C：\ Documents and Settings \ [user] \ Application Data \ Mozilla \ Profiles \ [profile name] 」にあるユーザー辞書の単語のセットを含むテキストファイルpersdict.datを同様に使用します。



一度に複数の単語を追加します



Webフォームのコンテキストメニューから不明な単語をすべて追加する代わりに、メモ帳でpersdict.datファイルを開くだけで、すぐに正しい単語を操作できます。



一般的な用語とタイトルのリストを使用する



よく知られている会社の名前の類似リスト、または一般的なコンピューター用語を含むサイトがあります。 リスト全体をコピーすることにより、FF辞書データベースを非常にうまく補充できます。



カスタムMS Office辞書を使用する



MS Officeを使用している場合は、組み込みのスペルチェッカーにとって未知の単語を定期的にユーザー辞書に追加するようにしてください。 それでは、なぜそれを使用しないのですか？

ディレクトリ「 C：\ Documents and Settings \ [user] \ Application Data \ Microsoft \ Proof 」に移動し、custom.dicファイルの内容をpersdict.datにコピーします。



ファイルで上記のすべての操作を実行する前に、ブラウザを閉じる必要があります。 任意のテキストエディターでユーザー辞書を編集し、新しい行から各単語を入力できます。 同様に、誤って入力した単語を削除することもできます。



すべてのテキストフィールドでスペルチェックを有効にする



翻訳ですでに述べたようにdidaio 、デフォルトでは、スペルチェックはすべてのフォーム要素ではなく、複数行のテキストフィールドでのみ実行されます。 詳細設定ページFF about：configを開いた後、layout.spellcheckDefaultパラメーターを標準値の1から2に変更することにより、単一行フィールドをチェックできるようにすることができます。



現時点では、これらはすべて組み込みのスペルチェッカーを改善するための可能な方法であり、これによりネットワークで通信するための現代の要件を満たすためにボキャブラリーを簡単に拡張できます。
