Flash CS3でAIRアプリケーションを作成する

現時点では、Flex Builderを使用してAIRアプリケーションを作成できます。 アドビは、Flash CS3 IDEでAIRアプリケーションを直接作成できる特別な拡張機能をすでに発表しています 。 しかし、現時点では、アドビを待たずに既に拡張機能を作成した優れたゲイワトソンとグラントスキナーの助けを借りて、この機会をすでに得ています。







それについて詳しく説明します。



ステップ1：Global AIR SWCをインストールします。

AIR SWCは、コンパイルされたAIR APIクラスライブラリです。 Flash CS3は、このライブラリへのパスを知っている必要があります。

  1. FlexBuilder 3“ Moxie”をインストールするか、 Adobe LabsからFlex 3 SDKをダウンロードします。
  2. 次のいずれかのディレクトリにairglobal.swcをインストールします

    -FlexBuilder 3“ Moxie”のメインディレクトリ内のフォルダーsdks / moxie / frameworks / libs / air /

    -Flex Builder 3をEclipseのプラグインとして使用する場合、パスは次のようになります。AdobeFlex Builder 3プラグイン/ sdks / moxie / frameworks / libs / air /

    -FlexBuilder 3“ Moxie” SDKと同様に-frameworks / libs / air /
  3. このファイルのコピーをFlash CS3のメインディレクトリに作成します- 設定/ ActionScript 3.0 /クラス/


残念ながら、AIR SDKにはこのファイルは含まれていません。取得するには、FlexBuilder 3にインストールする必要があります。



ステップ2：AIRComplierパネルのインストール



このパネルはFlash CS3にインストールされており、AIRアプリケーションを設定、テスト、コンパイルできます。

  1. Extension Manager v1.8をインストールします
  2. AIRCompiler Panel（MXP）をダウンロードする
  3. MXPをインストールする
  4. フラッシュを再起動
  5. これらすべてのアクションの後-AIRCompilerパネルはここで利用可能になります- ウィンドウ/その他のパネル/ AIRCompiler


パネルが表示されない場合は、Extension Manager v1.8を使用してインストールしたことを確認してください



ステップ3：パネルを使用する



それだけです これで、コードパネルとAIR SDKがインストールされ、約束どおりにFlash CS3でAIRアプリケーションを作成できます。



AIRランタイムのテスト

  1. AIRComplierパネルで、使用可能なFLAファイルのリストから独自のパネルを選択します。 ファイルが表示されない場合は、保存されていることを確認してください（保存されていないファイルはリストに表示されません）。 そうでない場合は、保存して数秒待ちます。
  2. パネルの「テスト」をクリックします
  3. アプリケーションを初めてテストするときは、adlファイルを選択する必要があります。 MACの下でアプリケーションをコンパイルする場合はadl（復号化なし）を選択し、Windowsの場合はadl.exeを選択します。

    adlへのパスは次のとおりです。

    -FlexBuider 3の場合-sdks / moxie / bin /

    -Flex Builder 3 SDKの場合-binフォルダー内
  4. アプリケーションがコンパイルされ、AIRランタイムで開きます。 原則として、テストは成功と呼ばれます。


ステップ5：AIRアプリケーションを作成する

  1. FLAファイルを選択します。 アクションは前のステップに似ています。
  2. 「パッケージ」をクリックします
  3. すべてが正しく実行されると、swfファイルがコンパイルされたディレクトリにairファイルが表示されます。


いつでも、[設定]ボタンをクリックして、AIRアプリケーションに固有のさまざまな設定を変更できます。 この場合、検索を使用できます。



これらのファイルが利用可能になるための前提条件は、FlexBuilder3「Moxie」のインストールです



パネルは現在「継続的改善」モードになっています。

ここにあなたの願いやコメントを残すことができます。
