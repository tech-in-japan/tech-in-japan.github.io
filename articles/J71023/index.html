任天堂Wii用のOperaブラウザは無料です









2009年9月30日-ノルウェー、オスロ -2009年9月1日から、任天堂の欧州子会社は、Wii用のOperaベースのインターネットチャネルブラウザの無料配布を開始します。 以前は、インターネットチャネルにアクセスするには、Wii Shop Channelサービスを介して500 Wiiポイントを支払う必要がありました。 9月1日より前にインターネットチャネルにアクセスしたWii所有者向けに、10月の任天堂は、保存されたWiiポイントを購入するための仮想コンソールプロジェクトのリストからNESゲームを選択することを提案します。



Opera for Wiiは、家族向けのユニークなブラウザです。 Wiiリモートコントローラーを使用してOperaブラウザーを制御することにより、ユーザーは電子メール、写真、フラッシュビデオ、その他のWebリソースにアクセスできます。 Opera for Wiiは、テレビ画面上のWebページで便利な作業を提供します。 インターネットチャネルサービスの発表中に、独自のスケーリングテクノロジーを備えたWii用Operaブラウザが初めて導入されました。



「私たちは、革新的な任天堂コンソールの成功の一部であることを光栄に思います。 Opera Softwareの最高経営責任者であるJon von Technerは、次のように述べています。 「インターネットチャネルサービスを使用する際の最後の障壁が克服され、現在では、はるかに多くのWiiコンソール所有者が完全なインターネットアクセスを利用できます。」



インターネットチャネルをダウンロードするには、WiiショップチャネルWebポータルに移動するだけで、Wiiゲームコンソールをインターネットに接続する必要があります。 NESゲームを無料で入手する方法については、Wiiメッセージボードを使用して、任天堂のWebサイトで、または任天堂のサポートに連絡してください。



Opera Software ASAについて

Opera Software ASAは、PC、携帯電話、その他のインターネットデバイスでWebページを表示するための新しい可能性を切り開きました。 Operaによって開発されたクロスプラットフォームブラウザーテクノロジーは、その機能、標準への準拠、小さなサイズで知られています。 同時に、高速で安全かつ動的なインターネットアクセスをユーザーに提供します。 Opera Softwareのメインオフィスはノルウェーのオスロにあり、世界中にオフィスがあります。 同社は、OPERA証券コードでオスロ証券取引所に上場しています。 詳細については、 www.opera.comをご覧ください。
