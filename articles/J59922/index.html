6月にはVisual Studio 2010のResharper

昨日、MSDNサブスクライバー向けにVisual Studio 2010をダ​​ウンロードできるようになったことは秘密ではありません。多くの人は、環境がWPF向けに再設計されることを知っています。 しかし、私にとって、そして多くの人にとって、新しいスタジオへの移行の主な要因（私はベータ版への移行についてではありませんが、一般的に）は通常、私のお気に入りのアドオンResharperのリリースを延期します。 そのため、少なくとも2008バージョンでは、jetbrainsがResharperバージョン4.0（VS 2008のサポートを開始）をリリースしたのは、スタジオ自体のリリースからわずか数か月後です。

Visual Studio 2010では、状況が劇的に変化しました。Resharper開発者は、6月にResharperバージョン5.0に触れることができることを保証します （同じ場所のいくつかのスクリーンショットを参照）（Resharper 4.0のリリースを延期したため、100％信頼することはできません）。 まあ、とにかく、スタジオのリリースの前に、お気に入りのアドオンが既に警告されていて、 JetBrains チームに感謝していることは素晴らしいことです！

PS Visual Studio 2010のExtension Manager（アドオンギャラリーのようなもの）を使用して、非常に簡単にインストールできると言われています。
