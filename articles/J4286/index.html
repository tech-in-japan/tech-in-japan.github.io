Microsoftはオフラインのブログエディターをリリースしました

無料プログラムのWindows Live Writer （現在ベータ版）は、オンラインで公開する前に投稿のデザインを評価する機会を提供します。 プログラムはHTMLの知識さえ必要としません。



多くのユーザー、特にダイヤルアップでは、オフラインでテキストを書く方がはるかに快適です。 ここでは、落ち着いて慎重にそれらについて考え、各単語を比較検討し、外観を評価できます。 すべての準備ができたら、数秒間オンラインにして[公開]ボタンをクリックするだけで十分です。 これが、Microsoftの新しいWindows Live Writer Blog Editorの仕組みです。



開発者によると、このプログラムはブロガーがミス、タイプミス、および多数の修正を回避するのに役立ちます。 さらに、このプログラムを使用すると、サーバーがクラッシュしたためにテキストが失われることを恐れずにリラックスできます。 ここでは自動保存が機能するため、テキストはとにかくコンピューターに残ります。



公開前に、プログラムは、フォントサイズとフォント、見出し、フォントの色、背景画像、段落間隔、インデント、引用の外観など、投稿がどのように表示されるかを正確に示します（ スクリーンショット ）。 経験豊富なブロガーは、ソースコードを表示するなどして、プログラムを別の表示モードに切り替えることができます。



Live Writerは、 Windows Live Spacesブランドサイトのブログを更新でき、Blogger、Livejournal、TypePad、WordPressなど、他の多くのブログプラットフォームと互換性があります。 エディターは、R​​SD（Really Simple Discoverability）、Metaweblog API、およびMovable Type APIをサポートしています。 オンラインプラットフォームと統合するには、最初にユーザー名/パスワードとページアドレスを指定する必要があります。その後、ボタンをクリックするだけでブログが更新されます。



便利な写真編集ツールがプログラムのインターフェースに組み込まれています：スケーリング、テキストフロー設定、グラフィック効果の追加（ スクリーンショット ）。 マッピングサービスWindows Live Localにアクセスする機能があります。これにより、惑星の任意のエリアの地図または衛星写真がすばやく検索され、投稿のテキストに挿入されます（ スクリーンショット ）。 この機能は、たとえば、現在の場所を表示したり、政治的な対立に関するメッセージを示す場合などに特に役立ちます。



サードパーティの開発者からのモジュールをWindows Live Writerエディターに接続できることは興味深いことです。これにより、プログラムを使用して他の種類のコンテンツ（写真サービス、ビデオ、オーディオファイルなど）からの写真を編集および公開できます。 Windows Live Writer SDKは、これらの追加モジュールを開発するように設計されています。 プログラムのウェブサイトでは、 最初のプラグインが既に公開されています 。



Windows Live Writerは、最初のブログ編集プログラムではありません。 今年の初めに、Appleは同様のiWebプログラムをリリースしましたが、Apple自身のブログサービスとほぼ密接に結びついています。
