Ruby Open Air（2012年6月16日）ミンスク

6月16日、ベラルーシRuby on Railsユーザーグループのサポートにより、ベラルーシで最初のRuby Open Airが開催されます。非公式のカンファレンス（非カンファレンス）で、国内外のベラルーシのRuby開発者とRuby on Rails開発者が集まります。 主役は、リー・ハンブリー（ドイツ、カピストラーノの維持者）、デニス・ゴリン（オランダ、netzke.org）、ヴラディスラフ・ゴロデツキー（多言語プログラミングエバンジェリスト、RubyShift、ウクライナ）です。 Ruby Open Airは、RubyとRuby on Railsの最新の動向と開発の方向性、最新のソリューション、現在の問題、経験の交換について議論するためのプラットフォームになります。 技術レポートやミニレポート、ディスカッション、ネットワーキングなど、さまざまな形式のプレゼンテーションを活用できます。











Ruby Open Airの会場はTerrasky Manorです。 参加者は、ビールとバーベキューパーティー、浴場、オープンスペースコンサート、本物のRuby開発者向けの極端なエンターテインメントなど、会議以外のイベントとその後の両方の豊富なプログラムを見つけます。



プログラム

10.45-地下鉄カメンナヤゴルカ駅での参加者の集まり、バスによる不動産への配達

12.00-ご挨拶

12.15-13.15招待客の報告（Lee Hambley、Denis Gorin）

13.15-14.45ミニトーク（ライトニングトーク）*

14.45-15.30昼食

15.30-17.00非公式の議論（パネルなし）

17.00-17.30要約、公式パートの完了

17.30-21.00バーベキューパーティー、ビール、非公式のコミュニケーション。

ミンスクへの21.00から23.00の配達は、地下鉄駅Kamennaya Gorkaに手配されます。



Ruby Open Airへの参加は有料です-200,000ベラルーシルーブル。 料金には、往復配達、食事、完全なエンターテイメントプログラムが含まれます。 有料席と無料席の両方の座席数は制限されています（参加者は100人以下）。



Lightning Talksセクションのすべてのスピーカーは無料で参加できます。 belarusrubyonrails.orgの詳細



Ruby Open Airに参加するには、次の2つの手順を実行するだけです。

1. belarusrubyonrails.orgでイベントに登録します。

2. kvitki.byからチケットを購入します （登録時に直接リンクが利用可能になります）。または、スピーカーとしての無料参加について主催者から確認を受け取ります。
