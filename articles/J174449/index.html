コンソールコマンド完了通知

Undistract-Meはシンプルでありながら非常に便利なファーム上のユーティリティで、1つのことを行います。長いコマンド（デフォルトでは10秒ですが、構成可能）の実行が完了すると通知を表示します。







このチームの名前と所要時間も表示されます。 すべてのデスクトップ環境で動作するはずです。

たとえば、何かをコンパイルするときに、常に黒い画面を見つめたり、それでも毎分動作するかどうかを確認したくない場合に特に便利です。 これで、カーネルを再構築しながら、お気に入りの猫のビデオを楽しむことができます！



設置


UbuntuのPPA： 

$ sudo add-apt-repository ppa:undistract-me-packagers/daily $ sudo apt-get update $ sudo apt-get install undistract-me




残りの場合： 

 $ git clone git://github.com/jml/undistract-me.git $ . undistract-me/long-running.bash $ notify_when_long_running_commands_finish_install


プロファイル設定で、「ログインシェルとしてコマンドを実行」オプションを有効にして、ターミナルを再起動します。



BASHでのみ動作しますが、 ZSHのバージョンがあります。
