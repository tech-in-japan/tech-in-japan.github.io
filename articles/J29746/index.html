ポータルとビッグビジネスの新しいエディション

同僚の皆さん、本日、大規模なお客様が高度な信頼性、セキュリティ、パフォーマンスのソリューションを作成できるように設計された「ポータル」と「ビッグビジネス」の2つの新しいエディションをリリースします。



新しいエディションは、フォールトトレラントなプロジェクトを開発する必要性に関する大企業からの増え続ける需要への対応です。 Rostelecom、Xerox、Eldorado、1Cなどの企業は、このサイトをビジネスと評判の重要な部分と見なしています。 サイトの「ダウンタイム」の1分ごとは、損失と損失です。 このレベルのリソースの場合、これは受け入れられません。 そのため、追加の機能、産業用データベースのサポート、および当社からの24時間年中無休の特別サポートを備えた製品の2つの特別エディションをリリースしました。



新しいエディションは、OracleおよびMSSQL産業用データベースをサポートします。これにより、Webプロジェクトと企業インフラストラクチャの統合、高性能、重要なプロジェクトの信頼性を最大限に高める線形スケーリングに対応したクラスター化システムの作成が可能になります。 （MySQLはこれらのエディションでもサポートされています）。

これらのエディションが開発されているプロジェクトのカテゴリの良い例は、Oracle Clusterテクノロジーを使用して教育省の命令によって開発され、教育機関がライセンスソフトウェアを受け取ったときに何百万人ものユーザーにサービスを提供するように設計されたFirst Help 1.0 Webポータルです。











標準の製品配信とMySQLのサポートに加えて、新しいエディションにはOracleとMSSQLのサポート、新しいNTLM認証機能を備えたActive Directory / LDAPモジュールインテグレーター、ドキュメント管理、パフォーマンスモニタリングモジュール、OpenIDサーバー、およびテクニカルサポートの特別なリクエストが含まれます。



編集委員会「Big Business」は、 1Cとの統合をサポートしています。Enterprise8-Production Enterprise Management（SCP）は、1Cの主力製品です。



新しい特別な通信システムは、（標準サポートに加えて）24時間365日の運用サポートを提供します。 緊急事態やプロジェクト開発の重要な分野に対処する際に開発者企業の潜在能力を最大限に活用できるように、インシデントに関するアドバイスはできるだけ早く実施されます。 テクニカルサポートは、24時間年中無休で特別なリクエストに対応します。



製品の以前のバージョン（OracleおよびMSSQLのエキスパートおよびビジネスエディション）のユーザーは、新しいエディション（それぞれポータルおよびビッグビジネス）に無料で移行されます。
