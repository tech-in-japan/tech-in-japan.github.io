Psion NetBook Pro

多くの人は、NetBookの名前の使用に関するNetBooksの夜明けに起こったスキャンダルを覚えているかもしれません（たとえば、ここで説明します： Dell：「ネットブックの自由！」とネットブックの情熱 ）。 このスキャンダルはPsion Ltdによって提起されました。 （より正確には、その現代の「残党」）、それは彼らの時間より先にあった市場でPsion NetBookデバイスを一度に発売し、主に企業セクターに焦点を合わせたため、あまり人気がありませんでした。



この記事は、「純粋な」ネットブックの1つであるPsion NetBook Proについてです。



注意！ たくさんの写真！





サイオン株式会社 かつて、NetBookと呼ばれる2つのデバイスがリリースされました。

  1. Psion NetBook、史上初のネットブック。 マシンはEPOC32を実行しています。 非常に安定した高速デバイス。
  2. NetBook史上2番目のPsion NetBook Pro。 フォームファクターを完全に保持しながら、すべての特性（画面、プロセッサ、メモリ、インターフェイス）を改善しました。 ただし、EPOC32はWindows CEに置き換えられており、安定性が低く、抑制性が高くなっています。


2番目のデバイスがWindows CEで作成される理由（すべてのPsion製品がEPOC / SIBOで使用される前）私は個人的に完全に理解していません。 どうやら、Psionはすでに経済的な問題を経験しており、より人気のあるWindows CEでデバイスをリリースすることで市場で活躍しました。 さらに、NetBook Proは企業部門のモデルとしてのみ位置付けられており、WinCEデバイスを企業のインフラストラクチャに統合することはEPOC32よりもはるかに簡単であるはずです。



外観と仕様



デバイスは比例しており、小型のラップトップのようなサイズです。 スクリーンはワイドスクリーンではないため、マシンの縦横比は伸びません。 そして、画面自体は、ケースのそのような寸法の最大サイズからはほど遠いです。 ケース自体は非常に厚く、バッテリーの背面で最大35mmです。



外観（すべての側面から）：



















すぐに言うことができます：デバイスには多くの拡張スロット、またはむしろ完全なセットがあります：ノートブックPCMCIA、CFカード用スロット、SDは忘れられませんでした。 少し動揺するのは、ヘッドフォンジャックとminiUSBホストジャックとしてのmicroJack（2.5mm）であり、アダプターの使用が必要です。



ビューを開く：











デバイスを開くときに最初に目を引くのは、開閉機構の設計です。 第一に、同じサイズの半分では開示が得られない：上部が下部よりも小さく、第二に、キーボードがデバイスの前面に移動し、画面ユニットがそれに隣接しているため、非常に便利です-画面上の項目を選択するために指/スタイラスをドラッグする必要はありません。 さらに、この設計では開口角の大きな選択を提供していません。実際、角度は1つしかなく、中間状態はありませんが、小さな制限内で調整されています。



サイオン株式会社 常に外に出て、前進し、曲がったものを開くときは、すべてのデバイスにすべての開閉機構を入れてください。 残りのメーカーは、同等の半分と通常のループの形で、単純で不器用なメカニズムを作成しました。 Psionは、より便利な実装実装を試しました。 残念ながら、これらのメカニズムは常に信頼できるとは限りませんでした。 たとえば、Psion Revoでは、ケース内のスプリングが破損し、閉じた位置でロックされました（これは使いやすさにはまったく影響しませんでした）。Psion5 / 5mxではさらに悪化しました。ケーブルは時間の経過とともにほつれ、ループの周りに亀裂が入りやすくなりました。慎重に使用しても脱落する可能性さえあります。



Psion NetBookの場合、メカニズムは成功し、判断できる限り、痛みはありませんでした。 この場合のヒンジは金属製で、プラスチックケースへの取り付けも非常に信頼性が高く、スクリーンケーブルに大きな負荷がかかることにも気付きませんでした。



画面自体は高品質ですが、それ以上のものはありません。 個人的には、私は少しがっかりしていて、何かをより対照的で明るい方法で見ることを期待していました。 角度は小さく、画像は特定の明るさを持たず、太陽の下では非常に平凡に動作します。 画面にバックライトがないと、何も見えません。 しかし、屋内で作業する場合、画面は悪くありません。 最初のPsion Netbookでは、TFTマトリックスはまったくありませんでしたが、STN（パッシブ）であったため、さらに悪化しました。



キーボード：







キーのサイズは快適ですが、キーボードはロシア語のレイアウトにはあまり届きません。ロシア語の文字がわずかに縮小され、すべてのロシア文字が標準的な場所に収まるわけではありませんが、数日使用すると慣れます。 また、特にリモートデスクトップで作業している場合、Fxキーの一番上の行が非常に不足していることがあります。



キーボードはフルノートパソコンです。 キーのデザインと可能性の両方の面で。 ハードウェアキーボードを備えた現代のコミュニケーター（私が扱っていたいずれの場合でも）では、クイックキーボードショートカットを入力する方法はなく、キーストロークを同時に行うとうまく動作しませんが、これはすべてあります。



スペースバーはキーボードで少しうるさいです：それは押されると大声でノックします。 しかし、その理由は簡単でした。キーを押すと、ボードが固定されます。 先端にシーラントを塗ると、ノイズが大幅に減少します。



バッテリー：







完全にノートブックの容量があります：1500-2200mAh、内部にはサイズ18650のリチウムバンクが3つあります。必要に応じて簡単に交換でき、バッテリーの寿命が延びます。 さらに、現在、最大3000 mAhの容量を持つ18650のバンクがあり、通常のバッテリーの容量を大幅に増やすことができます。



バッテリーは機械に非常に巧妙に接続されており、側面に挿入されているだけでなく、固定されているだけです。 まるでデバイスの中にあるかのように配置されており、上部は皮膚を模した柔軟なコーティングで覆われています。 このコーティングは、ほぼ完全に上部カバーを閉じ、デバイスの下部にわずかに行きます。 デバイスを開くと、曲がります：







私にとっては、このコーティングはかなりきれいに見えますが、時間が経つにつれて、端からプラスチックから剥がれ始め、接着する必要があります：







デバイスには、2つのAAAバッテリー用のコンパートメントもあります。 メインバッテリの交換時または完全放電時にメモリの内容を保存する必要があります。







結論として、スタイラス（長さは12 cmをわずかに超える、ほぼ本格的なペン）：







Psions、私見、すべてのPDAから最も便利なスタイラスがあります。 これはPsion 5mxとNetBooks、Psion Revoに当てはまりますが、杖は薄くて不快です。 NetBookでは、原則としてスタイラスは必ずしも必要ではありません。7.7インチの画面には大きな粒があり、ほとんどの場合、指で簡単に操作できるかなり大きなコントロールを提供します（従来のWindows CEインターフェイスでも）。



そして、ここにメインのTTX NetBook Proがあります（一部はここから借りています ）：

  1. プロセッサー：Intel XScale PXA255 400MHz（ARM9 / XScaleアーキテクチャー）。
  2. RAM：SDRAM 128Mb（ユーザーデータの保存とアプリケーションの実行に使用）。
  3. ROM：フラッシュ32Mb（オペレーティングシステムのストレージ）、不揮発性ディスクとして利用可能な小さなパーティションがあります。
  4. オペレーティングシステム：Windows CE.net 4.2。
  5. インターフェイス：PCMCIA Type-I、CF Type I / II、SD / MMCカード、SDIO、USBホスト/クライアント、RS-232、IrDA、2.5mmヘッドセットジャックをサポート。
  6. 画面：TFT 7.7 "SVGA 800x600、タッチ。
  7. 寸法：235x184x35mm。
  8. 重量：1100g。
  9. 電源：リチウムイオン電池、10.6V 1500-2200mAh。


ご覧のとおり、当時（2003年）、デバイスには最新のテクノロジーが搭載されていましたが、今日では特性の点で見た目は悪くありません。 唯一欠けているのはBlueToothとWi-Fiであり、特に企業部門向けに設計されているため、イーサネットインターフェースをこのようなマシンに組み込むことができます。 ただし、拡張スロットが豊富なため、これらの欠点はすべて解決できます。



バッテリーの寿命を強調することは特に価値があります：最大10時間に達する可能性があります。 実際には、デバイスが少なくとも 5時間良好なバッテリーで動作することが判明しています。たとえば、最低輝度で読む場合、NetBook Proは約8時間動作します。 これは優れた指標であり、NetBookファミリーの現代の代表者のほとんどは自慢できません。



このデバイスは、PDAとラップトップのクロスです。 PDAからは、長い稼働時間と即時起動などの強力な機能が採用されました。 ラップトップ、大画面、フォームファクターがラップトップから借りられました。 それにもかかわらず、寸法はノートブックのものに近いものですが、便宜上、何かを犠牲にする必要があります。



柔らかい部分と使用



ボードにWindows CE.net 4.2をインストールしました。 強力で機能的なシステム。 Windows CE用のプログラムの標準セットがプリインストールされています。 Windows CEデスクトップは、Windows 95 / Windows XPのスタイルで作成されています。 ショートカットと、下部にある[スタート]ボタンがあるタスクバーを備えたクラシックなスペース。



一般に、純粋なWindows CEの可能性を考慮すると、Pocket PCシェルを使用したバリアントでは、それがどれほど悲惨であり、トリミングされているかを理解できます。 はい、コアは同じですが、ソフトウェアのセットと機能は非常に異なります。 もちろん、製造元はファームウェアイメージを作成するときにWindows CEのソフトウェアを自由に選択でき、タスクに応じて個々のコンポーネントを切り取ることができますが、Psion Netbook Proは学際的なデバイスであるため、オペレーティングシステムの標準コンポーネントのほぼ最大のセットがあります。



たとえば、リモートデスクトップにアクセスするためのプログラムが既にプリインストールされており、コンソール、一連のコンソールユーティリティ（ipconfig、tracert、route、pingなど）、WordPadがあり、その機能は大規模なWindowsのWordPadとほぼ同等です（そして真剣にPocket Wordを上回る）。

しかし、RDPでは不快な驚きが待っていました。クリーンシステムでは正常に動作しますが、クラック（特定のMacCenter製品）をインストールした後、キーボードはインストールされているレイアウトに関係なくリモートセッションで動作しません。 おそらく問題には解決策があります（ロシア語のキーボードドライバーの抜本的な削除を除く）が、現時点ではそれを理解しようとする時間がありませんでした。



そして最後に、古典的なPocket IE6：







Windows CEには、非常に多様なソフトウェアがあります。 残念ながら、このソフトウェアの少なくとも95％は、Pocket PCおよびWindows Mobileシェルの使用を目的としています。 純粋なWindows CEでは、動作しないか、曲がって動作する場合があります。 幸いなことに、Windows CEに必要なすべてのソフトウェアを見つけることができますが、実験と検索には時間がかかります。 一部のプログラムでは、踊り、必要なDLLを探し、実行可能ファイルのPEヘッダーの互換性フィールドを修正する必要がありますが、最終的には起動します。



しかし、例えば、サードパーティのアプリケーション（この場合、Pocket Acrobat Reader 1.0）の作業：

その歴史は古くても、機械にはあらゆる生命の権利があります。 NetBookの現代の代表に対する2つの主な絶対的な利点は、即時起動と長時間実行です。 これらの2つの機能が重要な場合は、Psion NetBook Proのオプションを検討する必要があります。



残りの部分については、これは古いものの小さな概要ですが、その特徴的なマシンでは興味深いものです。 このレビューは、完全かつ完全な客観性を主張するものではありません。 ご清聴ありがとうございました！
