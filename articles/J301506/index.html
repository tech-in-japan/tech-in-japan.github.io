第13世代、第4波：エントリーレベルのDell PowerEdgeサーバー

2015年11月に発売されたエントリーレベルのPowerEdgeサーバーは、より大きなストレージ容量を提供し、アプリケーションに応じてさまざまな構成で利用できます。 モデルに応じて、DDR4 SDRAMメモリモジュール（最大64 GBのRAM）および最大4つまたは8つのハードドライブを取り付けるための4つのスロットをサポートします。 新しい第13世代のDell PowerEdgeシングルプロセッササーバーは、必要なパフォーマンス、柔軟性、および拡張性を備えた、成長する中小企業を対象としています。







中小企業は複雑で急速に変化する環境で運営されており、誰もがIT専門家をサポートできるわけではありません。 サーバーには、フル機能のリモート管理用のLifecycle ControllerとDell OpenManageソフトウェアが統合されたDell Remote Access Controller（iDRAC8）が搭載されており、サーバーをリモートで管理することもできます。 さらに、Dell OpenManageソフトウェアにより、小規模企業は展開時間を短縮し、管理を最大40％簡素化できます。 したがって、OpenManageは、企業（特に初心者）がITインフラストラクチャにサーバーを迅速かつ効率的に実装するのに役立ちます。 必要に応じて、サーバー管理機能をアウトソーシングに移行できます。







新しいDellサーバーは 、SMBの生産性向上、より強力なアプリケーションへの移行、運用効率の向上を支援します 。 4つのモデルの主な違いは、フォームファクターと構成オプションです。





12代から13代まで。 4つの新しいDell PowerEdgeサーバーモデルが古いモデルに取って代わりました。



R230モデルは、顧客に専用の物理サーバーを提供するホスティング会社に適したオプションです。 強力なDell PowerEdge R230ラックサーバーは、中小企業の分散アプリケーションをサポートするためにも使用できます。



拡張可能なストレージサブシステムを備えたDell PowerEdge R330ユニバーサルラックサーバーは、子会社オフィスを開設する組織に最適です。 これは、中小企業、大規模組織のリモートブランチ、およびOEM顧客を対象としています。



小規模オフィス向けのDell PowerEdge T130パワフルタワーサーバーを使用すると、オフィスアプリケーションとコラボレーションツールを簡単に使用できます。



ラックマウント可能なDell PowerEdge T330タワーサーバーは、企業の将来の成長に合わせて設計されており、大量のデータを保存する必要がある大規模組織の小規模および中規模の企業、部門、リモートオフィス向けに設計されています。



組織が成長するにつれて、組織は他のDellハードウェア、ソフトウェア、およびDellサービスとともに新しいDell PowerEdgeサーバーを追加できます。

提示されたモデルは、新しいIntel Xeon E3-1200 v5シリーズプロセッサとDDR4メモリ（DDR3メモリより33％高速）だけでなく、ほぼ2倍のスループットによっても区別されます。 3.0 実際には、これは、広範なアプリケーションの速度を上げ、システムの応答時間を短縮し、処理時間を短縮することを意味します。







周波数が2133 MHz（DDR3の場合は1600）のDDR4メモリモジュールは、帯域幅が30％高く、消費電力はアクティブモードで30％、スタンバイモードで50％減少します。 64GB DIMMを使用すると、サーバーのRAMの容量は前世代に比べて2倍になります。



IOPS（1秒あたりのI / O操作）は2倍（375ではなく750）でした。これは、以前のPERC8コントローラーを置き換えるRAIDテクノロジーをサポートするDell PERC9コントローラー（PowerEdge RAIDコントローラー9）によって促進されました。 ハードドライブ（または他の種類のドライブ）をインストールするために、サーバーは（構成に応じて）4〜8個のベイを提供します。これは、予算システムにデータを保存するのに十分な容量です。 ネットワークアダプターの幅広い選択肢により、さまざまなタスクに合わせてサーバーをより柔軟に構成できます。



最後に、Fresh Air 2.0テクノロジーのサポートにより、サーバーはより高い温度で動作し、冷却を節約できます。



新しいモデルは仮想環境を展開する準備ができており、Hyper-V、VMWare vSphere ESXi、およびCitrix XenServerを備えたMicrosoft Windows Serverをサポートしています（R330はRed Hat Enterprise Virtualizationハイパーバイザーもサポートしています）。

運営管理



コントロールについては、さらに説明する価値があります。 PowerEdgeサーバーには、Dell OpenManage Server Administrator（OMSA）ソフトウェアとOpenManage Essentials（OME）管理コンソールが付属しており、Lifecycle Controllerソフトウェアで統合Dell Remote Access Controller 8（iDRAC8）を操作できます。 OSMAは、管理にソフトウェアエージェントを使用します。 彼らの助けを借りて、コマンドラインまたはウェブベースのGUIから管理機能を実行でき、OSがインストールされているかどうかに関係なく、iDRACはシステムへのリモートアクセスを提供します。 iDRACによって表示されるシステム概要ページでは、管理者は基本的なiDRAC情報の表示、システムステータスの監視、タスクの実行、ファームウェアのアップグレード、ログの表示、サーバー電源のオン/オフ、iDRACリセットの開始を行うことができます。







[ファンのセットアップ]ページでは、エネルギープロファイル（熱プロファイル）を構成し、デフォルト設定を最大または最小-最大パフォーマンス（パフォーマンス最適化）/最小電力（ワットあたりのパフォーマンス最適化）に変更できます。 ファン速度オフセットはサーバーの冷却を強化し、PWMの最小ファン速度（最大の％）オプションはより詳細にファン速度を制御します。 CPUセクションには、システムのステータスが表示されます。 OpenManage Mobileアプリケーションを使用して、Dell OpenManage Essentialsを操作することもできます。



独立したテストによると、PowerEdge 13G Wave 4サーバー-2つのフロアスタンド（T130およびT330）モデルと2つのラックマウント（R230およびR330）モデルは、アプリケーションの実行時に高いパフォーマンスを示します。 SASまたはSATAドライブを組み合わせて、データを処理する際のスループットとIOPSの速度を高めることができます。 SysbenchおよびSQL Serverのテストでは、これらのシングルプロセッササーバーは4プロセッサシステム、たとえばPowerEdge R230-R930およびR920と競合できます。 また、iDRACおよびOpenManage管理ツールは、展開とメンテナンスを大幅に簡素化します。 そしてこれらすべて-非常に控えめな価格で。
