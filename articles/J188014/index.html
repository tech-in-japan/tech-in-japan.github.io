Twitter Bootstrap 3 RC 1がリリースされました

Bootstrap公式ブログ投稿： blog.getbootstrap.com/2013/07/27/bootstrap-3-rc1



アーカイブのドキュメントとダウンロードは、古いアドレスtwitter.github.io/bootstrapから入手できます。



主な変更点

  1. ブートストラップおよび関連プロジェクトはGithubの新しいアカウントに移行されました： github.com/twbs
  2. 〜1,600のコミット、〜300のファイルへの〜72,000の追加/削除がプロジェクトに追加されました。
  3. すべてのスタイル（レスポンシブを含む）が単一のCSSファイルにコンパイルされるようになりました -アップロードされるファイルが少なくなりました
  4. グリフィコンは、PNGではなくフォントとして接続された別のリポジトリに移動され、アイコンクラスに新しいプレフィックス.glyphicon- *が追加されました。
  5. LESSコードには多くの変更があります。変数の名前付けの新しいスタイル、色の変数（ blueなど）が削除され、セマンティックなもの（@ brand-primaryなど）が採用されました。
  6. スマートフォン（.col- *）およびタブレット（.col-sm- *）用の2つの新しいタイプのグリッドを追加しました。
  7. IE 7およびFirefox 3.6のサポートはなくなりました。 IE 8および9は引き続きサポートされていますが、多くのCSS3プロパティがサポートされていないことを忘れないでください。
  8. このプロジェクトは、ApacheではなくMITライセンスに移行される予定でしたが、Apacheは現在のドキュメントに引き続き表示されます。


また、必要なBootstrapコンポーネントのみを選択できないカスタマイザーコードはまだ機能していません。 彼らはRC2にそれをつけると約束します。



現時点では、「Bootstrapのダウンロード」リンクにエラーが表示されるため、コンパイルおよび縮小されたCSSおよびJSをリポジトリから直接ダウンロードできます： github.com/twbs/bootstrap/tree/3.0.0-wip/dist



プルリクエストの変更の全リスト： github.com/twbs/bootstrap/pull/6342



以前のバージョン2.3.2のドキュメントはtwitter.github.io/bootstrap/2.3.2にあります
