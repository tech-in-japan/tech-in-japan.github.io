Windowsシステム要件の履歴

Cyber​​netNewsの仲間はあまりにも怠laではなく、「最愛の」Windows OSのシステム要件の年表の調査を実施しました。 長年にわたって発生した技術の開発の規模について考えてみましょう。



Windows 1.0

Windows 1.0は1985年11月20日にリリースされました。

-CGA / Hercules / EGAビデオアダプタ（または互換）

-MS-DOS 2.0

-256 KBのラム

-2台のドライブまたはハードドライブ







Windows 2.10

Windows 2.1は1988年5月27日に正式にリリースされました。

興味深い事実：バージョン2.10は、登場したIntel 286プロセッサをサポートするために特別にリリースされました

-MS-DOSバージョン3.0以降

-512 K RAM

-1台のドライブと1台のハードドライブ

-グラフィックアダプター

-Microsoftマウス-オプション



Windows 3.1x

Windows 3.1のいくつかのバージョンは、1992年から1994年の間にリリースされました。

興味深い事実：このOSバージョンの主な違いは、システムインストーラーがMS-DOS以外のDOSシステムでの動作を拒否したことです。

-MS-DOS 3.1以降

-Intel 80286プロセッサー（またはそれ以上）

-1 MB以上のメモリ（640Kコアおよび256K拡張）

-6.5 MBのディスク容量（9 MBを推奨）





Windows 95

Windows 95は1995年8月24日にリリースされました。

興味深い事実：主な改善点は、新しいグラフィカルインターフェイスです。 実際、このGUIの一般的な形式と構造はWindowsでも使用されています。

-Intel 80386 DX CPU

-4 MBのRAM

-ハードドライブに50 MB



Windows 98

1998年6月25日リリースのWindows 98

興味深い事実：Windows 98は、WDM（Windows Driver Model）テクノロジを使用した最初のOSでした。

-プロセッサ486DX-2 / 66 MHz以上（Pentiumを推奨）

-16 MB RAM（24MBを推奨）

-500 MB以上のディスク容量

-VGA以上の解像度でモニターする

-CD-ROMまたはDVD-ROMドライブ

-Microsoftマウスまたはその他のポインティングデバイス



Windows 2000

Windows 2000は2000年2月17日にリリースされました。3つの異なるバージョンのシステムがリリースされ、それぞれに異なるシステム要件があります。

興味深い事実：マイクロソフトはWindows 2000を「信頼性の標準」と宣伝しました。Windowsデスクトップアップデート、Internet Explorer 5、Outlook Expressなどの新機能がこのシステムに導入されました。 Windowsエクスプローラーにも大幅な改善が行われました。

Windows 2000 Professional

-133 MHz以上の周波数のPentium互換プロセッサ

-32 MB RAM（64 MBを推奨）

-700 MBのハードディスク容量（2 GBを推奨）

Windows 2000 Server / Advanced Server

-133 MHz CPU

-256 MB以上のRAM

-2 GBのハードドライブ



Windows me

2000年9月24日リリースのWindows Me

興味深い事実：Windows Meは1年以上の間関連性があり、その後Windows XPが置き換えられました

-周波数150 MHzのPentiumプロセッサ

-32 MBのRAM

-320 MBのハードディスク空き容量

注：これは、MS-DOSをリアルモードで操作するためのサブシステムを備えた最新バージョンのWindowsでしたが、アクセスは閉じられていました。





Windows XP

Windows XPは2001年10月25日にリリースされました。

以下は、XP HomeおよびProfessionalの最小要件です。

-233 MHzの周波数のプロセッサ

-64 MBのRAM

-1.5 GBの空きディスク容量

-SVGAビデオアダプターとモニター

-CD-ROMまたはDVDドライブ



Windows Vista

Windows Vistaは2007年1月30日に公開されました。

興味深い事実：このOSが批判されている主な側面の1つは、そのシステム要件です。

Vista Premium Ready

-1.0 GHzプロセッサー

-1 GBのRAM

-128 MB VRAM

-40 GBのハードドライブ

-15 GBの空きディスク容量



実際、この全期間にわたって発生したシステム要件の進展は目覚しいものがあります。 RAMのサイズのみが256 KBから1 GBに増加しました！ 1つの質問は未解決のままです。次のWindowsオペレーティングシステムのシステム要件はどうなりますか？
