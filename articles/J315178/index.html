ニュースとリリース



  • PHP 7.1.0 RC6-最新リリース候補。 PHPチームは、可能な限りテストに接続するように全員に依頼します。 重大な問題が見つからない場合は、11月末にPHP 7.1の最終リリースがリリースされます。
  • PHP 5.6.28、7.0.13-セキュリティ問題の修正を含む現在のブランチの更新。
  • PrestaShop 1.7 -Symfonyコンポーネントの新しいアーキテクチャとPHP 7のサポートを備えた一般的なEコマースソリューション。
  • Symfony Moscow Meetup-Symfoniacs-＃10- 11月10-17日、モスクワでPHP / Symfony開発者の第10回会議が開催されます。


ツール



  • krakjoe / ui-クロスプラットフォームのlibuiベースのグラフィックアプリケーションを開発するためのPHP 7拡張。 ゲーム「ヘビ」とビデオ デモの実装例が利用可能です。 ドキュメントがあります 。 著者は、PHP 7はグラフィックアプリケーションの開発に適しているだけでなく、非常に高速で効率的であると書いています。
  • dannylinden / aws-lambda-php -AWS LambdaでPHPコードを実行するツール。
  • EasyEngine / easyengine -Nginx、PHP / HHVM、MySQLのWordPressサイトを便利に管理するためのコマンドラインツール。
  • nategood / commando-シンプルで流interfaceなインターフェースでコンソールコマンドを実装するためのライブラリ。
  • jenssegers / imagehash-画像の知覚ハッシュ-画像の類似度を判断できます。
  • simshaun / recurr-繰り返しルール（ RRULE ）を操作するためのライブラリ。繰り返しカレンダーイベントの作業を簡素化します。
  • bernardphp / bernard-さまざまなバックエンドをサポートするバックグラウンド（保留中）タスクを管理するためのライブラリ：Predis / PhpRedis、Amazon SQS、Iron MQ、Doctrine DBAL、Pheanstalk、PhpAmqp / RabbitMQ。
  • woohoolabs / yin - JSON：APIを実装するためのフレームワークに依存しないライブラリ。 neomerx / json-apiの代替実装。
  • aszone / avenger-sh-潜在的に脆弱なサイトを自動的に検索するためのツール。
  • akalongman / php-ip-tools -IPv4 / IPv6アドレスを操作するためのツール。
  • object-calisthenics / phpcs-calisthenics-rules -CodeSnifferのオブジェクト体操規則。
  • spatie / once -PHPのメモ化の便利な実装。
  • Robo 1.0.0-タスクを起動するためのツール。
  • clue / php-socks-server -React PHPに基づいてSOCKSプロキシサーバー（SOCKS4、SOCKS4a、SOCKS5）を実装します。
