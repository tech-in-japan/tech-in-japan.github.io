ASUS EeePC 1005ネットブックのレビュー-シェルでのWiMAX

黒いオフスクリーンのテキストから抜け出す方法：「 あなたは海岸にいます。 あなたの体はリラックスしています。 優しい波の音が聞こえます。 あなたは穏やかで、世界の調和を楽しんでいます。海の表面上をゆっくりと流れる白い巻積雲の雲を見ています。 しかし、モザイクは、最先端のWiMAXモバイルインターネットと、バッテリ寿命の延長されたスタイリッシュなASUS Eee PC 1005ネットブックなしでは完全には完成していません 。







それは理想的な世界で完璧なEee PC 1005の広告になるでしょう。 残念ながら、世界は不完全ですが、1005はこれに非常に近いです。 ロシアで最初のネットブックの1つを手に入れました。WiMAXモジュールが組み込まれていても、多大な労力を費やしてこの素晴らしい新製品の詳細なテストを実施しました。レビューを読んでください。 ）トラフィック。



ASUSが「 SeaShell 」（シェル）と呼んでいる新しいデザイントレンドについては、長い間話をすることができます。ネットブックと貝殻を比較することは、本当に良いコンセプト、あるいはメタファーであることに同意するでしょう。 後者は美しく、保護されており、内部の環境の成功した組み合わせ-真珠。 つまり、光沢のある黒いプラスチックケース、鋭いエッジの欠如、滑らかなラインの優位性です。



この場合、真珠の役割は、シリコン大手のIntelの新しいプロセッサによって果たされます。正確には、クロック周波数1.66 GHz （システムバス周波数-667 MHz）と512 Kb L2キャッシュを備えたIntel Atom 280です。







「軟体動物」の寿命は、 Intel Mobile 945GSEチップセットに由来します（内部はASUS EeePC 1000HEモデルと同じですが、シェル自体が異なることを除いて、優れていることが証明されました）。







ボードにはRAM 1024MB（DDR2 PC2-6400、400MHz。2GBに置き換えることができます）およびデータストレージ用の170GBのメモリ（Seagate ST9160310AS SATA2ハードドライブ上で、最大5400 rpmで回転するプレート、160GB + 10GBが利用可能） EeeStorageのおかげで利用可能）。 G-Sensorのようなハードドライブを保護するためのこのような有用な技術の場所がありました。 ネットワーク技術から、Bluetooth、Wi-Fi（802.11b / g / n）がサポートされています-注意-WiMAX！



ピッキングについて





パッケージバンドルはかなり標準的です：ASUS EeePC 1005ネットブック自体、電源コード付きACアダプター、ネオプレンケース、ソフトウェアディスク、および説明書。



このキットのすべては、「本」を数えません-標準、私は詳細にこだわらないでしょう。 今回は、デバイスに適したアダプタからの電源コネクタのサイズが、以前のすべてのコネクタよりもわずかに小さいことに注意してください-これに留意してください。



本物のMicrosoft Windows XP Homeは最初にプリインストールされていますが、おそらくLinuxを搭載したバージョンもあるでしょう。



外観





デバイスは光沢のあるプラスチックで作られているため（慎重に調べるとメタリックカラーと共通の色を持っています）、遅かれ早かれこのデバイスの幸せなユーザーは指紋などの問題に遭遇します。 ケースはASUS Infusionテクノロジーを使用して作られています。このテクノロジーは、アクティブな使用中に発生する避けられない擦り傷や傷に対する耐性を高めます。 （黒-青、白、ピンクに加えて）-どの海に応じて、さらに3色を選択できます）



デバイスの寸法は非常に控えめ（mm）-262×178x26.5〜35.5ですが、バッテリーの重量は1キログラム強です。







ネットブックには、LEDバックライト付きの10.1インチ画面があり、既に広く普及している1024x600の解像度（WSVGA、16：9）を使用しています。 画面の上には1.3メガピクセルのカメラと側面にあるマイクのペアがあります。 画面は120〜130度以下しか傾きません。







左側には、アダプターを介して電力を供給するための小さなコネクターがあり、その隣にはVGA、1つのUSB（2.0）、および通気口の「えら」があります。







反対側（右側）には、カードリーダー（MMC / SD / SDHC）、ヘッドフォンジャックとマイクジャック、さらに2つのUSB（2.0）ポート、ネットワーク100メガバイトのRJ-45ホールがあります。 残念ながら、USBポート間の距離はまだ非常に小さいため、JetFlashフラッシュドライブなどの「大型」デバイスを使用しているときに問題が発生する可能性があります。







裏側-驚くべきこと-何もありません。



タッチパッド



今回のタッチパッドは非常に珍しく、私の意見では非常に成功しています。 それは物理的に「輪郭を描かれて」おらず、ケースのパネルの上に直接配置されており、「2つのメディアの分離」がないため、そこに詰まる場所はありません。 私はすでにそのような解決策に出会いましたが、その場合、夜はタッチパッドが正常に感じられないという事実のために、いくつかの不快感がありました-最初にボタンを探しました。 すぐにそのような問題はありません。タッチパネルの領域がポイント「にきび」で覆われているため、指で使用するのに非常に便利です。 それらは指の動きを妨げず、長時間使用しても指が痛むことはありません。 このすべての多感覚に追加してください。これはプログラムでカスタマイズでき、別のタッチパッドは必要ないことを理解できます。

2つのキーは1つ、クロムメッキの形で作られています-原則として、これはすでにおなじみの構成です。







タッチパネルの右下には、動作状態、バッテリー、ディスクアクティビティ、Wi-Fi、CapsLockの5つのドットインジケーターがあります。 バッテリーインジケータが充電に応じて変化する可能性があることを除いて、ほぼ全員が青色に点灯します。



キーボード



今回はやや珍しいです。 これらはApple / Vaioのような個別のボタンではありませんが、列のいたるところに配置される単純な「くさび形」のキーではなくなりました。 このモデルでは、それらは長方形で、明確に定義されたエッジを持ち、標準サイズ（17.5x17.5）よりわずかに大きくなっています。 下の行の高さは他のボタンよりも大きくなっていますが、それほど大きくありません。







レイアウトはタイピングのファンにアピールします（WiMAXを介したICQで何かを印刷してみませんか！）。しかし、おもちゃのファンにはアピールしません（フラッシュゲームについて話している）。 事実、矢印ボタンには、あまり成功していないと思われるいくつかの変更が加えられています。 これらは1行に配置され、上下キーは通常のキーの半分のサイズです。 おそらくあなたはこれに慣れることができますが、数日間はこれを行うことができませんでした。 それ以外の場合、レイアウトは非常に成功しています-大きなCTRL、FN、SHIIFT、ENTERおよびBACKSPACEの両方。



テストインスタンスでは、これは単なるサンプルであるため、キーにキリル文字はありませんでした。 もちろん、大量配達では、ネットブックにはロシア語のキーボードが付属しています。



インクルージョンと仕事





ネットブックは、画面の右下にあるボタンを押すとオンになり、中の明るさの青いLEDが点灯します。 少し後に、残りのLEDが「プル」されます。 OSは約30秒後に起動します。 ネットブックの作品の音はほとんど目立ちません。 仕様から判断すると、26dBを超えません。







デバイスのパフォーマンスは、予想どおり、かなり良いレベル（ネットブック向け）です。 Intel Atom 280プロセッサは、すでに説明した以前のモデルで最良の方法で実証されているため、ここでは最悪のパフォーマンスを期待しないでください。 比較してください。



SuperPi Mod 1.5テスト

百万位の小数位までのPi番号の計算時間を比較します。

ASUS EeePC 1000HEはこのタスクに94.84秒で対処しました。

ASUS EeePC 1005は、テストを完了するのに89.594秒かかりました 。







比較のために、 ASUS Aresゲーミングステーションは17.643秒でこのテストに合格し、ディスクリートグラフィックスを備えた優れたラップトップは30秒かかります。



CPU FreeBenchmark 2.2-3つの合成数学テスト。実行時間が短いほど最適です。 省電力モードでは、1000HEネットブックは94.78秒で合格し、1005番目のモデルは同じことを行うために93.22秒かかりました。 高パフォーマンスモードでは、時間はそれぞれ71.39と71.31でした。 本格的なラップトップは60秒でタスクに対応し、このテストのASUS Aresは25.06秒で仕事をします。







ハードドライブについても、非常に優れたパフォーマンスです。 測定はHDTune 3.50プログラムで実行されました-線形読み取りスケジュールは驚くほどスムーズで落ち着いていました。 最大/平均/最小速度（Mb / s）は、それぞれ68.6 /54.7/20.5でした。











ソフトウェア機能

Asus EeePC T91タッチモデルのこのモデルに存在する非標準のソフトウェアに既に遭遇しましたが、メモリで更新する価値があると思います。 まだ私と一緒に？



そのため、レジストリを介して自動実行に登録されたEeeDockingプログラムは、画面の上部（再構成可能）にある小さなポップアップパネルです。 これには、サブグループに分割されたいくつかのプログラムが含まれています。

-Eee @Vibe ：オンラインブロードキャストにアクセスし、さまざまなマルチメディアデータをダウンロードします。

-Eee共有 ：インターネットに接続せずに、他のEee PC、ラップトップ、デスクトップPCとメッセージやデータを交換できます。

-Eee Xperience ：ビデオおよびオーディオの再生品質を最適化するためのアプリケーション。 音声制御の可能性があります（Dr. Eee）

-Eee Tools ：Live Update（自動更新）、Parental Control（保護者による制限、サイトやアプリケーションのブロックが可能）Font Resizer（画面のフォントサイズの管理）などの便利なツールのセット。



Linuxを搭載したバージョン（15秒で読み込まれます！）では、独自のASUS Super Hybrid Engine、Skype、OpenOffice、NOD、およびその他のさまざまなソフトウェアがプリインストールされます。



作業時間





バッテリーメーカーは長年にわたって技術の進歩に大きな負債を抱えてきたことに注目する価値があります。 しかし、EeePC 1005ネットブックは部分的にしか影響を受けません。 なんで？ 一部のモデルには容量の大きい6セルバッテリーが搭載されているため、10時間以上も喜びを伸ばすことができます。 2200mAh（10.8V）の容量の3セルリチウムイオンバッテリーを入手しました。これは約23Whワットです（私の知る限り、2番目の「容量のある」オプションは2900mAhで、容量は64.3Whワットです）。 BatteryEaterで本格的なテストを実施しなかったため、「通話から通話まで」の正確な動作時間がありません。 しかし、私は次のように言うことができます：



運命の悪役は、私の都市（モスクワ地方のジェルジンスキー）の公式WiMAXカバレッジのまさに端に私を住まわせ、好奇心のために、私は幸福を求めて慎重に街を駆け回りました。 魅力的な旅では、2台のネットブック（外部のYotaモジュールSamsung SWC-U200を搭載したASUS EeePC 1002HAとレシーバーを搭載したASUS EeePC 1005）を用意し、テストドライブを配置しました。







結局、私のネットブック（1002HA）は頭痛と安らぎのために横になりたいという欲求を訴え始めましたが、「シェル」は自信を持って信号をキャッチし、渋滞を取り除きました。 このことから、ネットブックは少なくとも4〜5時間動作していると結論付けることができますが、切断された通信モジュールでは今回だけ増加する可能性を排除しません。







「釣り」の品質に関しては、ほとんどの場合、組み込みモジュールは外部のSamsungよりも優れていることが証明されました。 おそらく、ラップトップのアンテナがディスプレイの下のカバー全体を覆っているという事実のためですか？ 知りません







サムスンのデバイスが何もまたは最大1つの「スティック」を見ない異常な領域があり、「シェル」は微妙な信号をキャッチし、特に「活発な」場所では2つの「スティック」で機能しました。







いずれにせよ、両方のデバイスは、100 km / h（モスクワ環状道路上）の速度を含め、良好な速度性能を示しました。 ちなみに、私はこの機会に聞いてみます-多くの人が時速100km以上の速度でインターネットを必要としていますか？

結論



「Seashell」シリーズの新しいモデルは非常に成功し、間違いなくターゲットオーディエンスを見つけるでしょう。 スタイリッシュな外観と軽量さは、男性と美しい女性の両方の使用に適しています-これは、学年度の開始前にバイヤーからの特別な反応を見つけると思います。



高性能で長い作業時間、ボタンによるレースに適した便利なタッチパッドとキーボード-はい、これで利点のリストは終わりません。



最も興味深いイノベーションは、組み込みのWiMAXモジュールです。 この技術には大きな未来があることはすでに明らかであるため、外付けモデムでUSBポートを占有することなく、常にあなたと一緒にいるワイヤレスインターネットのすべての喜びを感じることができます。



マイナスについては-はい、しかし、他の場所と同様です。 光沢はすべての人に適しているわけではありませんが、原則として光沢仕上げに反対していない場合、このモデルはあなたに適した選択肢になる可能性があります。
