SQL Server 2014 CTP2がリリースされました

PASS 2013で、Microsoftの副社長であるクエンティンクラークは 、Microsoft SQL Server 2014の次のリリースである2nd Customer Technology Preview（PTP2）が利用可能になったことを発表しました。



この予備リリースは機能的に完全であることに注意してください。 それと比較して、最終リリースに大きな革新を追加する計画はありません。 主要な革新には以下が含まれます。



•インメモリOLTP（Hekaton）をサポート

SQL Server 2014には、テーブルとストアドプロシージャをRAMに配置する機能があります。これにより、トランザクションアプリケーションのパフォーマンスが大幅に向上します。



•更新された列インデックス

列インデックスは、主に分析アプリケーションで使用されます。 OLAP（星、スノーフレーク）に固有のデータスキームでのクエリの実行を大幅に高速化できます。 列インデックスは、読み取り専用モードのSQL Server 2012の現在のバージョンで既に使用されています。 バージョン2014では、この制限が削除され、列インデックスは、従来のインデックスのようにデータが変更されたときに更新できるようになります。



•リソースガバナーの新しい種類の管理対象リソース

リソースレギュレーターは、SQL Serverにアクセスするさまざまなアプリケーション間でコンピューターの計算能力を分割するために使用されます。これにより、負荷の優先順位付けが可能になり、アプリケーションがデータベースサーバーに割り当てられたすべてを引き継いで、リソースを超えないようにします。 これにより、予測可能なパフォーマンスが可能になります。 以前のバージョンでは、リソースにはメモリとプロセッサ時間が含まれていました。 将来のバージョンでは、I / Oが追加されます。



•回復力の向上

AlwaysOn高可用性グループには、8つのセカンダリレプリカが含まれるようになりました（現在のバージョンでは4つ）。

Windows Cluster Shared Volumesのサポート。



•クラウド統合

オンプレミスのSQL Server用の大きなデータベースファイルをWindows Azureでホストする機能、および新しい暗号化機能を含むWindows Azureでバックアップを実行する機能。



SQL Server 2014の新機能の詳細については、 こちらから PTP2をダウンロードしてください 。
