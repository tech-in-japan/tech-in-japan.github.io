iPhoneのGCC

iPhoneでは、gccコンパイラーを実行できます。 仲間のSaurikとiGuga iGugaに感謝します。





1. Cydiaをインストールします。[ 設定]-> [全般]-> [自動ロック]-> [しない]に移動します。

インストーラーで 、 apptapp.saurik.comリソースを追加します。

インストール->システム-> Cydia Packagerに進みます。

インストール。 8〜10分待つ必要があります。 Cydiaはアプリケーション、写真などを転送します。 iPhoneの大きなセクションで、プログラムをインストールするためのスペースを解放します。



2. Cydiaに移動します。 右上の半円をクリックして（コースを更新し）、 開発セクションからGNU Cコンパイラーをインストールします 。



3. ターミナルサポートカテゴリから、 MobileTerminalをインストールします。



4.公式のiPhone SDKをダウンロードします 。



5. iPhoneで 、フォルダー/usr/lib/gcc/arm-apple-darwin8/4.2.1/includeを削除します。



6.公式のiPhone SDKからincludeフォルダーをコピーして、iPhoneの/usr/lib/gcc/arm-apple-darwin8/4.2.1/フォルダーに貼り付けます。



7.テストCファイルを作成します。



#include <stdio.h>



int main（）

{

printf（“ Hello world！\ n”）;

0を返します。

}



test.cと呼びましょう



8.これをiPhoneにドロップします（たとえば、 / var / mobileフォルダーに）。



9. iPhoneのターミナルに移動します。



10.次のディレクトリに移動します： cd / var / mobile



11.コンパイル： gcc -o test test.c



12.起動： ./test



14 ;-)。 ターミナルに表示されます：



こんにちは世界！



15.シャンパンを抜きます。
