KDEプロジェクトがJavascriptウィジェット開発コンテストを発表

KDEプロジェクトは、 Plasma Javascript Jam Sessionを 発表しました。 参加者は、 Plasma JavaScript APIを使用して記述されたデスクトップ用の最も独創的で興味深いプラスモイドを作成するように招待されています 。 エントリーは2010年3月31日までにjavascriptjam@kde.orgに送信できます。その後、 エントリーは4月2日にkde-look.orgに投稿され、4月9日に受賞者が発表されます。



すべてのエントリのコードは、KDEコアコードベースと互換性のある無料ライセンスの下で配布する必要があります。 データ処理にサードパーティのJavaScriptライブラリとエンジンを使用することは可能ですが、作業自体は、他のプラズモイドからコードをコピーすることなく、完全に独立して作成する必要があります。 個人と開発チームの両方がコンテストに参加できますが、各参加者/チームから受け入れられるのは1つのプラスモイドのみです。



作品の評価基準は次のとおりです。

  • 40％-ユーザー生成の問題を解決するためのプラスモイドの有用性と適用可能性。
  • 20％-アイデアの独創性と一意性。
  • 20％-デザインの質;
  • 20％-技術的なスキルとプラズマ機能の全範囲の使用。


コンテストの勝者には、Nokia N900コミュニケーター、道具一式、およびAkademyやCamp KDEなどのKDE開発者会議に参加するための支払いが提供されます。 準優勝者は、KDE開発者会議への旅費と道具一式を受け取ります。 3位の勝者には道具一式が授与されます。 さらに、コンテストの勝者が考慮されます：最も素晴らしいプロジェクト、最も技術的に進んだプラズモイドおよび最も創造的なソリューション。
