「実験：Pentiumから166メガヘルツを絞ることはできますか？」という記事の著者へ

「アンティーク」セクションの新しい投稿にコメントを書き始めましたが、何かがおかしくなりました...



一般的に、私はソース素材が好きでした。 私も数年前に非常によく似たコンピューターを持っていましたが、いじくり回すのに疲れていたので捨てました。 まったく同じVIAマザー、同じPentium MMX 166がありましたが、32 MBのメモリ、S3 Trio64 VX / DX、およびOpti931サウンドエンジンがありました。 コンプも97歳くらいでしたが、友人から手に入れました。 私のコメントや個人的な経験は何ですか？ そしてここに：

-システム装置内のほこりはまだ収集されています。 ATのミディタワーケースは、現代のタワーほど人間工学的ではなく、ビルド品質も同じです。 ほこりの量は環境によって異なります。きれいであれば、ほこりはありません。

-32メートルのSIMMストリップを4つ入れて、RAMを128 MBに増やすことができました。

-PCI用のUSBコントローラはありませんでしたが、カードリーダーがあり、マザーボードにUSB 1.1コネクタがありました。 ピンアウトを拾って接続しました-うまくいきました。

-ハードドライブはもともと2ギガバイトのWDでしたが、20ギガバイトの別のドライブを追加しました。 BIOSがそれを見るために、彼は別のマシンのFAT16に最初の2 GBのパーティションを作成しました。 そうでなければ、何らかの理由で、ネジがまったく検出されませんでした。

-CD-ROMは24スピードで、CD-RとRWの両方を完全に読み取りました。 著者がなぜそれらを読まなかったのか分かりません...

-ソフトウェアについて。 そのようなマシンで最高のパフォーマンスが得られたのは、MS Windows 95でした。 問題はUSBのみにあり、これはWin 95 OSR 2.1で公式にサポートされていましたが、特定のデバイスには実際にはfireがありませんでした。 そして、はい、95歳未満のドライバーのほとんどはOS / 2と互換性があり、場合によってはIBM自身が作成したことを忘れないでください。 要するに、彼らはUSBについては考えませんでした。

さまざまなLinuxをマシンに配置しようとしました。 私は実際にPuppyRusと古いFedora Core 3をインストールできました。作者がさらに古代のOSと古代のNetscapeをインストールした理由-わかりません。 Opera 9の実行を開始しました。ロシア語のフォントは素晴らしく見えました。 CPUが弱すぎるため、フラッシュは機能しませんでした（ところで、188 MHzにオーバークロックしました）。



ここで彼はハンサムです：





私の最大の目標は、Bluetooth + GPRSを介したインターネットへのアクセスと映画鑑賞の2点でした。 当然のことながら、圧縮されたMPEG4は「スライドショー」モードで全画面表示されるため、通常のMPEG2に満足する必要がありました。 14インチモニターでは、品質は良好でした。

最初の目標は正式に達成されましたが、実際にインターネットを使用することはほとんど不可能でした。 BluetoothおよびUSBサブシステムがCPUをロードしたため、ブラウザーはほとんど動作していませんでした。 また、インターネットは非常にゆっくりと走行しました（GPRSでは、すでにそれほど高速ではありません）。 要するに、私はインターネットサーフィンを放棄しなければなりませんでした。 しかし、映画ではすべてが燃え尽きました。 後で、完全なLinuxを放棄して、ポータブルMOVIXディストリビューションに切り替えることさえできました。 これは、この方法で行われます。システムは、すべてのコーデックを備えた小さなLinuxおよびMplayerが存在するMovixを含むブート可能なCDからロードされます。 OSがRAMにロードされ、CDが取り出され、映画（650〜700 MB）のCDがコンピューターにロードされます。コンピューターは、「ビッグ」マシンでMPEG 2に事前にトランスコードしてお持ちください。 フィルムは遅れることなく、キーボードから状況を完全に制御して完璧に行きました。 唯一の欠点は、CD-ROMが掃除機のように大声で叫んだことです（はい、私はhdparm -Eについて知っていますが、その後フィルムが遅くなりました）。 それでも、現代の速度に甘やかされている人は、どういうわけか、そのような古い車を選ぶのは快適ではありません。 今、私はその場所にUbuntuでPeintium 4を持っています-しかし、それは別の話です。



PS元のトピックの作者がマウスを使ってシステムユニットとキーボードをエレガントに洗濯していることに気付きました。 正確さのために-尊敬！
