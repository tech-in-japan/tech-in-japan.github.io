Internet Explorer 7はブロゴスフィアで批判の波を引き起こしました

Windows XP （SP2またはx64）およびWindows Server 2003用の Internet Explorer 7の 公式リリースから数時間後、新しいプログラムの詳細なレビューがすべての主要機能のスクリーンショットとともにブロゴスフィアに登場しました。 ブロゴスフィアでの「 Internet Explorer 7 」の言及の数は、すぐに4万を超えました 。 多くのレビューが重要だったことを認めなければなりません。



XP用IE7ディストリビューションのサイズは14.8 MBで 、FirefoxまたはOperaの3倍です。 プログラムをインストールするには、2回再起動する必要があり、全体のプロセスには約10分かかります。



インターフェースの変更は印象的です。 たとえば、「進む」ボタンと「戻る」ボタンは、「停止」と「更新」だけでなく、単一のモジュールに結合されます。 設定へのアクセスは、6つの小さなボタンの助けを借りて、メニューからではありません。 ほとんどの新機能（RSSフィード、タブ、検索バー、フィッシング保護のサポート）は、特に過去14か月間にIE7の5つのベータ版がリリースされて以来、 予備バージョンのリリース時に詳細に知られていました。



Internet Explorer 7では 、透過PNG、 CSS 2.1 、Ajaxなどのネットワーク標準のサポートが改善されましたが、これは、一部のサイトがIEブラウザーで異なって表示されることを意味します。



IE7のリリースと同時に、 サイトが拡張機能 （アドオン）で開かれました。その多くはずっと前から知られていました。 最も人気のある拡張機能には、 Avant Browser 、 FlashアニメーションをFlash2X Flash Hunterハードドライブに保存するユーティリティ、 すべてのRoboForm フォームにパスワードを自動的に入力するユーティリティなどがあります。 プログラムは、 Download.comとの契約に基づいて自動的にカタログ化されます。



ブロゴスフィアでは、新しいブラウザー（ 最初の穴が既に検出されている）のセキュリティ問題、速度、IE7ディストリビューションがYahoo Webサイトに最初に登場し、Microsoft Webサイトで公式にローンチされたという奇妙な事実について説明しています。 多くのブロガーは、IE7とFirefox 2の比較テストを独自に実施し、 Firefox 2 がはるかに高速であることを報告しようとしています 。
