A / Bテストは、サイトでのコンバージョンを増やす最も効果的な方法の1つです。 あなたが初心者なら、実験のアイデアを思いつくのは簡単ではありません。 しかし、私たちは支援することを決め、100のアイデアのリストを作成しました。 友達、勉強に進みます。







適切なA / Bテストは非常に複雑なプロセスであることをすぐに警告したいと思います。 全体の難しさは、実験を開始しないことです。 これが最も簡単です。 全体の難しさは、仮説の定式化、テストのアイデアの検索にあります。



専門家が提唱する規則に従って、A / Bテストの潜在的に成功するアイデアを見つけるには、Web分析を使用し、ユーザーの行動を分析し、ターゲットオーディエンスとコミュニケーションを取り、調査を実施する必要があります。 そして、これらのルールは絶対に論理的です。 それらを観察する必要があります。 しかし、小さな会社、小さなチームがあり、必要な専門家がいない場合、上記のすべてを行うことは非常に困難です。



この場合、100のアイデアのリストが編集されました。 記事の最初の部分で51のアイデア。 49-2番目。 サイトで最も関連性の高いアイデアを研究、採用、テストします。



A / Bアクションコールテスト









行動を促すフレーズ（CTAボタン/行動を促すボタン）-ほとんどすべてのサイトに存在する要素。 それは販売ファネルで最も重要なものの1つです。彼らはクリックして、ある段階から別の段階に進みます。 したがって、アクションボタンの呼び出しは、A / Bテストで最も人気のあるオブジェクトの1つです。 さらに、この要素をテストすると、多くの場合、最も重要な結果が得られます。

A / Bテキスト要素のテスト








どのサイトにも膨大な量のテキストコンテンツがあります-見出し、小見出し、製品の説明、顧客の利点の一覧、製品の機能、レビューなど。 これらの要素を使用して、後続の決定に影響を与える最も重要な情報をクライアントに伝えます。 それが、テキスト要素をテストできる理由です。

15.見出しと小見出しのテキストをテストします。 見込み顧客になじみのある言葉を正確に使用し、彼らの言語を話してください。 典型的な例は、潜在顧客がYandex WordstatまたはGoogle Adwordsの特定の単語を使用して製品を検索することが最も多い場合、これらの単語をテキストで使用する必要がある場合です。 また、さまざまな見出しスタイルを試してください-厳格なビジネススタイルの方がうまく機能し、創造的で遊び心のあるオプションを使用することが望ましい場合があります。



16.ヘッダーの長さをテストします：短いヘッダーと長いヘッダー。 サイトに長いテキストが多数ある場合、ユーザーはそれらを単に無視できます。 したがって、あなたは彼らに対するあなたのユニークな申し出の意味を伝えません。 この場合、短いテキストの方が効果的です。



17.タイトルのみのページのパフォーマンスと、タイトル+サブタイトルのページのパフォーマンスを比較します。 タイトルとサブタイトルは、訪問者が注意を払う最初の要素の1つです。 この場合、サブタイトルは、製品に関する重要な情報の追加ソースの役割を果たします。 では、この機会を利用してみませんか？

19.肯定的/否定的な意味合い、質問/肯定的、受動的/能動的なテキストをテストします。 たとえば、アクティブなヘッドライン「ビジネス用のページを作成する」は、パッシブな「ビジネスがインターネット上でより速く開発されている」よりもはるかに効果的です。 または、「手術後の最初の月の生存率-90％」というテキストは、「手術後の死亡率が最初の1か月で10％である」よりも無意識によく認識されます。

A / Bフォームテスト









支払い、ニュースの購読、サイトでの登録-これらすべてのアクションは、フォームに記入することにより実行されます。 フォームはサイトの重要な要素です。 そして、フォームに記入するときに、多数の潜在的な顧客が「脱落」します。 そして、これは、あなたがそれらに取り組んで、フォームに記入して仕事を終える人の数を増やすことを試みる必要があることを意味します。



ちなみに、A / Bテストの前に、ユーザーがどのように満たすかを分析することをお勧めします。 これを行うには、Yandex Metricで利用可能な無料のAnalytics Formsサービスを使用できます。 この情報により、潜在的な顧客が質問や困難を抱えている段階を理解できます。
