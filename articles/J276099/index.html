ローカル検索システムをAndroidアプリケーションに追加します

多くのプログラムには検索機能が必要です。 今日は、レストランのアプリケーションでこのような機能を実装する例を検討します。 私たちの主な目標は、ユーザーが望むものをさまざまな料理のメニューからすばやく簡単に見つける機会をユーザーに提供することです。



ユーザーインターフェイスの統一性を維持しながら、既存のアプリケーションのコンテンツによるローカル検索の機能をどのように追加したかについて説明します。 ここでは、ユーザーインターフェイスに加えられた変更とこれらの変更の理由を検討し、GestureOverlayViewオブジェクトをActivityクラスの要素に追加して独自のジェスチャを作成することについて説明します。 当然、検索には特別な注意が払われます。









レストラン検索結果画面



検索の詳細



アプリケーションに検索機能を追加する前に、いくつかの機能を考慮し、いくつかの質問に答える必要があります。



正確に何を探すべきですか？ 製品のヘッダーとその説明を検索して、ユーザーが結果の最も完全なセットを取得できるようにします。ヘッダーだけでは、どの料理が問題かを正確に理解できるとは限らないためです。 さらに、非表示のメタデータを検索対象のアイテムに追加できます。



検索結果を表示する方法、これに使用するレイアウトは？ リスト（ListViewオブジェクト）から始め、ショッピングカートと同じスタイルで作成しました。 しかし、そのようなプレゼンテーションでは、料理は特に魅力的に見えません-すべての理由は、写真のサイズが小さすぎることです。 リスト内の画像のサイズを大きくすると、ページのスペースが小さすぎて結果を表示できないことがわかりました。 その結果、検索結果をグリッド（GridView要素）に配置し、レストランメニューの主要部分で行うのと同じことを行うことにしましたが、選択した料理に関する詳細を含む大きなブロックを横に配置する代わりに、画面全体に一連の製品を配置しました。 これは、特に、検索結果のあるページと通常のメニュー画面をすばやく区別するのに役立ちます。 料理の詳細を表示するには、ユーザーは検索結果ページで自分の写真をタッチする必要があります。 これに応じて、ページの上にダイアログボックス（DialogFragmentオブジェクト）が表示されます;下の図のいずれかに表示されます。 これにより、ユーザーはダイアログボックスの外側のスペースをタップするだけで、検索ページにすばやく戻り、他の料理の閲覧を続けることができます。 ユーザーは必要なものをできるだけ早く見つけたいので、理想的には、検索は遅延なくほぼ瞬時に実行する必要があります。 さもなければ、彼らは望むものを見つけることができないか、結果を待つのに飽き飽きして、何も探してアプリケーションを離れません。



機密ユーザーデータの処理方法 以前に入力した検索クエリに基づいてヒントを提供する検索システム、またはユーザーが自分に関する詳細情報を入力する必要がある検索を作成できます。 これにより、他の人がユーザーが探しているものを正確に見ることができるか、入力された個人データがどこに送信されるかについての疑問が生じます。 私たちの場合、レストランのアプリケーションについて話しているので、ユーザーがチョコレートケーキが好きだと誰かが見つけても、悪いことは起こりません。 ただし、プライバシーに最も注意を払う必要があるプロジェクトがあります。 このアプリケーションでは、ユーザーは自分に関する情報を入力する必要がなく、検索クエリはログに記録されず、ユーザーの履歴は保持されません。

ジェスチャー処理



検索結果でウィンドウを終了するには、メインメニューで他のページを表示するときと同じように、適切なジェスチャーで左または右にスワイプして移動します。 GestureDetectorクラスはリスト（ListView）ではうまく機能しますが、テーブル（GridView）ではうまく機能しません。 したがって、GestureOverlayViewの使用に切り替える必要があります。



最初に、GestureBuilderアプリケーションを使用してジェスチャライブラリを作成する必要があります。これは、Android SDKに付属のサンプルの中にあります。 特に、必要なのはandroid \ sdk \ samples \ android-23 \ legacy \ GestureBuilderにあります。



アプリケーションを使用するには、デバイスでコンパイルして実行し、それを使用してジェスチャの名前と構成を指定する必要があります。 必要なすべてのジェスチャーを追加した後（この場合、左にスワイプ、左にスワイプ、右にスワイプ、右にスワイプ）、デバイスから「ジェスチャー」ファイルをコピーし、res / rawフォルダーに配置する必要があります。 アプリケーションは、このファイルの正確な場所を教えてくれます。この場合、USB経由でデバイスをPCに接続し、ルートディレクトリを見るだけで十分です。
