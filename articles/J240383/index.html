Firefox 33リリース

デスクトップ向けFirefoxがダウンロード可能になりました。 主な技術革新：（そして再び）メモリ消費の最適化、Windowsのメインスレッドのオフ、OpenH264の統合、アドレスバーでの検索のさまざまな改善、セッション回復の安定性の向上など。 Firefox 33でのすべての変更のリストは猫の下にあります。





  • JavaScript文字列を保存し、文字列データを処理するためのメカニズムが最適化され 、メモリ消費が削減され 、文字列操作のパフォーマンスが向上しました。







  • シスコによって開かれたOpenH264ライブラリの使用により、H.264ビデオコーデック（サンドボックス内）のサポートが追加されました。
  • Windowsの場合、デフォルトではOff Main Thread Compositing（OMTC）が有効になっています。
  • アドレスバーでの検索が改善されました（たとえば、次のページに検索のヒントが追加されました：homeおよびabout：newtab）。
  • 回復セッションの安定性の向上 。
  • コンテンツセキュリティポリシーの新しいバックエンドが追加されました。
  • HTTPSを使用したHTTPプロキシへの接続のサポート。
  • window.cryptoの独自のプロパティと機能が削除されました。
  • デバッガーAPIを支持するJavaScriptデバッガーサービスのサポートは廃止されました（MDNの記事-eng 、 ロシア語 ）。
  • DOMMatrixインターフェースを実装 。
  • CSS3 Counter Styles仕様の@ counter-styleルールを実装。


Firefox開発者ツール

  • Bezier Cubic Curve Editor （ビデオ）を 追加 。
  • インスペクターでDOMイベントを表示します 。
  • 現在のスタイルシートの各@メディアルールのショートカットのリストを表示する新しいサイドバーが実装されました。
  • 再描画可能な要素を視覚的に評価する機能（ ペイントの点滅 ）が追加されました。
  • 検査モードの[ルール]セクションに、@ keyframesルールを編集する機能が追加されました。
  • また、「インスペクタ」でもCSSの強調表示が表示されるようになりました。


その他

  • Firefoxで開くすべてのウィンドウのアニメーションを少し変更しました（ 少し「曲がった」ように聞こえますが、より明確かつ正確に記述する方法がわかりません） 。
  • アゼルバイジャン語にローカライズを追加[az]
  • HTTP 1.1を介したデータ転送の違反を検出することにより、不完全なダウンロードが完了としてマークされる問題を修正しました
