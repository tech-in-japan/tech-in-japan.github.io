-すぐに、私は録音しています！ // IFA 2012のSamsung Galaxy Note 2

サムスンギャラクシーノートのやや奇妙なフォームファクターにもかかわらず、最終的に多くが「試してみました」、そして「スマートフォンまたはタブレット」の販売はかなり高いレベルでした。 しかし、何らかの理由で、次のモデル（注II）のリリースに関するインターネットは、セラミックのどれだけの割合がGalaxy S-IIIに含まれるのかと誰もが不思議に思っていた数か月前よりもずっと少なくなりました。 そのため、新しいNoutのダミーを手にした後、IFA 2012でダミーをオンにした後、あなたは無駄になっていると言えます。 無駄に、無駄に、無駄に！ 注IIは、その前身よりも約12.53％涼しかったことが判明し、いくつかの点で最愛のS-IIIでさえあるかもしれません。







新しいノートはわずかに薄くなりました（ミリメートル、それ以上）、最も重要なこと-5.55インチのディスプレイを備えたスマートフォンについて言えば、よりエレガントです。 これは、ディスプレイのアスペクト比が変更されたために発生しました。 これで、画面のアスペクト比は16：9になります。これは、最初にビデオを見ながら黒いバーから私たちを救い、次にスマートフォンを少し狭くします。そのため、手にとって便利です。



ディスプレイのサイズは5インチ半に増加しました。 Super AMOLEDセンサーの解像度は1280 x 720ピクセルになりました。 視野角、明るさ、コントラスト-すべてが上。 色はおそらく少し酸性ですが、これはAMOLEDの一般的な機能です。





左から：最初のGalaxy Note、Galaxy Note 2、Galaxy S 3



高品質の素材が選択されました。金属エッジングと光沢のあるプラスチックは、SGS-IIIで使用されているものに似ています。 一方で、開発者がテクスチャ付きの裏表紙を放棄したことは残念であり、他方では、ガジェットはこれによりオタク性が低下し、販売レベルが自動的に少し上がります。







コントロールは同じ場所にとどまりました。 右側（サムスンの場合と同様）は電源/ロックキー、左側はボリュームコントロール、上部はヘッドフォンジャック、下部は充電とコンピューターへの接続用のマイクロUSBポートです。



背面パネルには、8メガピクセルの解像度とLEDフラッシュを備えたカメラ用の場所がありました。 画面の前には、光センサー、近接センサー、ビデオ通話用のフロントカメラが並んでいます。



















しかし、ノートと呼ばれるためには、サムスンのガジェットの観点からスタイラスが必要です。 新しいモデルでは、彼はどこにも行っていません-彼はまだ右下隅に隠れています。 ようやく楕円形になり（保持するのが本当に便利になりました）、鉛筆ケースへの固定方法が変わりました。 現在、機械的なクランプは使用されていませんが、ペンを3つの位置にしっかりと固定できるいくつかの磁石があります。割り当てられたニッチに完全に挿入され、4分の3と半分です。



別の話はスタイラスソフトウェアチップに値しますが、ビデオを見るほうが良いですが、ここでそれらをリストします。

  • 8 mmの距離でペンでガジェットを制御します。
  • 手書き認識;
  • 会話中に画面にステッカーを表示して情報を記録する。
  • 写真の一部をすばやく切り取ります。
  • 裏面の写真に「署名」します。
  • 図面内の単語の視覚化。
  • 手書きの数式の認識、方程式の解法、プロット;
  • メディアコンテンツのプレビュー。
  • 描画;




頭に浮かぶのはそれだけです。 たぶん私は何かを忘れました:)







新しいガジェットのパフォーマンスは、単に致命的です。 彼は「手に入れる」ものすべてを吐き出します。 Quadrant Standartでは、1.6 GHzの周波数と2ギガバイトのメモリを備えたクアッドコアExynosプロセッサーのおかげで、約6000ポイントを獲得し、HTC One X、Asus Transformer Prime、さらには「兄弟」のGalaxy S-IIIを追い抜きました。











そして何よりも、Galaxy Note 2はAndroid 4.0では動作しませんが、Jelly Bean 4.1では動作します。これは、まずシステムにいくつかのクールな機能を追加し、次に消費電力を最適化します。



ところで、エネルギー消費について！ ここのバッテリーは3100 mAhで取り付けられており、1日、またはさらに長いバッテリー寿命を期待できます。



唯一の質問は、そのような美しさがどれくらいかかるかです...
