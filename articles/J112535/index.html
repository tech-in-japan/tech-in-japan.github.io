1G、2G、3G、4Gおよびその間のすべて

信じられないかもしれませんが、携帯電話がスマートフォンやスーパーフォンではなく、実際に「電話」と呼ばれるようになったら...彼らはあなたのポケットに入って電話をかけることができます。 以上です。 ソーシャルネットワーク、メッセージング、写真のアップロードはありません。 5メガピクセルの写真をFlickrにアップロードすることはできません。もちろん、ワイヤレスアクセスポイントに変えることはできません。



もちろん、これらの悲観的な日々はすでにかなり遅れていますが、有望な新世代のワイヤレス高速データネットワークは世界中に出現し続けており、多くのものが混乱し始めています。 4Gとは何ですか？ 3Gよりも高いですが、それはより良いことを意味しますか？ 4つの全米通信事業者が予期せず4Gネットワ​​ークを呼び出すのはなぜですか？ これらの質問への回答には、ワイヤレステクノロジーの歴史についての短い説明が必要です。



そもそも「G」は「世代」を意味するため、誰かが「4Gネットワ​​ーク」と呼ばれると聞いたとき、それは第4世代の技術に基づくワイヤレスネットワークについて話していることを意味します。 このコンテキストで「生成」の定義を適用すると、理解しようとするすべての混乱につながります。



1G



物語は、1980年代にいくつかの革新的なネットワークテクノロジーが出現したことから始まります。アメリカのAMPSと、ヨーロッパのTACSとNMTの組み合わせです。 以前は数世代のモバイルサービスが存在していましたが、3つのAMPS、TACS、およびNMTは第1世代（1G）と見なされます。



1Gの時代には、誰もデータサービスについて考えませんでした。これらは純粋にアナログシステムであり、音声通話やその他の控えめな機能を実行するためだけに考案および開発されました。 モデムは存在していましたが、ワイヤレス通信は従来の有線よりもノイズや歪みが発生しやすいという事実により、データ転送速度は非常に低かったです。 さらに、80年代の1分間の会話のコストは非常に高かったため、携帯電話は贅沢品と見なすことができました。



また、1963年にモスクワで発売された世界初の自動移動通信システムAltaiについても言及したいと思います。 アルタイは、車に搭載された本格的な電話であることになっていた。 通常の電話のように（つまり、音声が両方の方向に同時に渡される、いわゆるデュプレックスモード）、単純に話すことができました。 別のアルタイ電話または通常の電話に電話するには、デスク電話のように、番号をダイヤルするだけで十分で、チャンネルの切り替えやディスパッチャとの会話は不要です。 米国で同様のシステムであるIMTS（Improved Mobile Telephone Service）が1年後に実験エリアで開始されました。 そして、その商業的な発売は1969年にのみ行われました。 一方、1970年、ソビエト連邦では、アルタイが設置され、約30の都市で成功しました。 ところで、ヴォロネジとノボシビルスクでは、システムはまだ有効です。



2G



90年代前半には、最初のデジタルセルラーネットワークが急増し、アナログシステムと比較して多くの利点がありました。 音質の向上、セキュリティの強化、パフォーマンスの向上-これらが主な利点です。 GSMはヨーロッパで開発を開始し、D-AMPSとQualcommの初期バージョンのCDMAは米国で発売されました。



これらの初期の2G標準には、独自の密接に統合されたデータサービスのサポートがまだありません。 これらのネットワークの多くは、デジタル形式でステーションにデータを送信できるCSDテクノロジーと同様に、ショートテキストメッセージ（SMS）の送信をサポートしています。 これは実際には、データをより速く転送できることを意味しました-最大14.4 kBit / s、これは90年代半ばの固定モデムの速度に匹敵しました。



CSDテクノロジーを使用してデータ転送を開始するには、特別な「呼び出し」を行う必要がありました。 それは電話モデムのようなものでした-あなたはネットワークに接続されているかどうかにかかわらず。 当時の料金プランは数十分で測定され、CSDは通常の電話に似ていたという事実を考えると、この技術による実用的なメリットはほとんどありませんでした。



2.5G



1997年のGeneral Packet Radio Service（GPRS）の登場は、既存のGSMネットワークに継続的なデータ伝送技術を提供したため、セルラー通信の歴史の転換点でした。 新しいテクノロジーを使用すると、必要な場合にのみデータ転送を使用できます。電話モデムのような愚かなCSDはもうありません。 さらに、GPRSはCSDよりも高速で、理論的には最大100 kBit / sで動作でき、オペレーターは回線の時間ではなくトラフィックを充電できました。



GPRSは非常に都合の良い瞬間に登場しました。人々がメールの受信トレイを継続的にチェックし始めたときです。



この革新により、モバイル通信の世代にユニットを追加することはできませんでした。 GPRSテクノロジーはすでに市場に出ていましたが、国際電気通信連合（ITU）は、「実際の」3Gの仕様を承認する新しい標準IMT-2000を起草しました。 重要なポイントは、固定端末では2 Mb / s、モバイル端末では384 kbit / sのデータ転送速度を提供することでした。これはGPRSの能力を超えていました。



したがって、GPRSは彼が優れた2Gの世代と彼が到達しなかった3Gの世代の間で行き詰っていました。 これが世代間の分裂の始まりでした。



3G、3.5G、3.75G ...および2.75Gも



前述のデータレート要件に加えて、3G仕様では第2世代ネットワークからの簡単な移行が求められました。 このため、UMTSと呼ばれる標準がGSM通信事業者の最有力候補となり、CDMA2000標準は下位互換性を提供しました。 GPRSの先例に従って、CDMA2000標準は1xRTTと呼ばれる独自の連続データテクノロジーを提供します。 公式にはCDMA2000は3G規格ですが、GPRSをわずかに上回るデータ転送速度（約100 kBit / s）しか提供しないことは紛らわしいです。



EDGE標準-GSM Evolutionの拡張データレート-は、GSMネットワークオペレーターが、設備のアップグレードに多額の投資をせずに、2.5Gインストールから余分なジュースを搾り取る簡単な方法として考案されました。 EDGEをサポートする電話を使用すると、GPRSの2倍の速度を得ることができます。 多くのヨーロッパの事業者はEDGEを気にせず、UMTSの実装に取り​​組んでいます。



では、EDGEはどこに行くのでしょうか？ UMTSやEV-DOほど高速ではないため、3Gではないと言えます。 しかし、これは明らかにGPRSよりも高速です。つまり、2.5Gよりも優れているはずです。 実際、多くの人がEDGE 2.75Gテクノロジーを呼び出します。



10年後、CDMA2000ネットワークはEV-DOリビジョンAへのアップグレードを受け取りました。EV-DOリビジョンAは、わずかに高い受信速度とはるかに高い送信速度を提供します。 EV-DO Revision 0と呼ばれる元の仕様では、発信速度は150 kBit / sに制限されていますが、新しいバージョンではこれを10倍高速に行うことができます。 3.5Gが手に入りました！ UMTSについても同様です。HSDPAおよびHSUPAテクノロジーにより、着信および発信トラフィックの速度が向上しました。



UMTSのさらなる強化では、HSPA +、デュアルキャリアHSPA +、およびHSPA + Evolutionを使用します。これらは、理論的には14 Mb / sから驚異的な600 Mb / sまでの帯域幅を提供します。 だから、私たちは新しい世代にいると言うことができますか、それはEDGEと2.75Gとの類推で3.75Gと呼ぶことができますか？



4G-不正行為全般



3G標準と同様に、ITUは4Gを制御し、IMT-Advancedとして知られる仕様に結び付けました。 このドキュメントでは、固定端末では1ギガビット/秒、モバイルでは100メガビット/秒の着信データレートが必要です。 IMT-2000と比較して500および250倍高速です。 これらは、通常のDSLモデムやブロードバンドチャネルへの直接接続を追い越すことができる、非常に速い速度です。



ワイヤレス技術は、農村地域でブロードバンドアクセスを提供する上で重要な役割を果たします。 農地を光ファイバー回線で覆うよりも、数十キロメートルの距離で通信を提供する4Gステーションを1つ構築する方が費用効率が高くなります。



残念ながら、これらの仕様は非常に積極的であるため、世界のどの商業標準にも準拠していません。 歴史的に、CDMA2000およびGSMと同じ成功を達成するように設計されたWiMAXおよびLong-Term Evolution（LTE）テクノロジーは、第4世代のテクノロジーと見なされますが、これは部分的に当てはまります。どちらも、新しい、非常に効率的な多重化スキーム（OFDMA、過去20年にわたって使用してきた古いCDMAやTDMAとは異なります）、どちらにも音声伝送用のチャネルがありません。 帯域幅の100％がデータサービスに使用されます。 つまり、音声伝送はVoIPと見なされます。 現代のモバイル社会がデータ転送にどれほど力を入れているかを考えると、これは良い解決策と考えることができます。



WiMAXとLTEに障害が発生する場合、データ転送速度にあり、理論的にはこれらの値は40 Mb / sおよび100 Mb / sのレベルにありますが、実際には商用ネットワークの実際の速度は4 Mb / sおよび30 Mb / sを超えませんそれぞれ、それ自体は非常に優れていますが、IMT-Advancedの高い目標を満たしていません。 これらの標準の更新-WiMAX 2とLTE-Advancedはこの作業を行うことを約束しますが、まだ完了しておらず、それらを使用する実際のネットワークはまだ存在していません。



それでも、元のWiMAXおよびLTE規格は、従来の3G規格とはまったく異なるため、世代交代について話すことができます。 実際、このようなネットワークを展開している世界中のほとんどのオペレーターは、4Gと呼んでいます。 明らかに、これはマーケティングとして使用されており、ITU組織にはそれに反対する権限がありません。 両方のテクノロジー（特にLTE）は、今後数年間で世界中の多くの通信事業者によってまもなく導入され、「4G」という名前の使用が増加するだけです。



そして、これは物語の終わりではありません。 近い将来にHSPAネットワークをLTEにアップグレードする意向を発表しなかったアメリカの事業者T-Mobileは、HSPA +へのアップグレードのブランド化を4Gとして開始することを決定しました。 原則として、このステップは理にかなっています。3Gテクノロジーは最終的にLTEだけでなく、IMT-Advancedの要件に近い速度を達成できます。 HSPA + T-MobileネットワークがSprintのWiMAXよりも高速な市場は数多くあります。 また、Sprint、Verizon、MetroPCSのいずれも、ライブWiMAX / LTEネットワークを持つ3つの米国の通信事業者は、VoIPサービスを提供していません。 彼らは引き続き音声に3G周波数を使用しており、今後しばらく使用します。 さらに、T-Mobileは今年、LTEに触れることなく、42 Mbpsの速度にアップグレードする予定です。



おそらく、T-Mobileのこの段階が、携帯電話の購入者の間で4Gが実際に何を意味するのかを世界的に再考したのです。 AT＆TはHSPA +への移行を進めており、今年後半に一部の市場でLTEの提供を開始する予定で、これらの4Gネットワ​​ークの両方を呼び出します。 したがって、米国の4つの全国オペレーターはすべてITUから「4G」という名前を盗みました。彼らはそれを奪い、逃げ出し、変更しました。



結論



それで、これはすべて私たちに何を与えますか？ オペレーターはこの戦いに勝ったようです：ITUは最近、4Gという用語を「このテクノロジーの前身であるLTEとWiMAX、および他の進化した3Gテクノロジーに適用でき、第3世代の初期システムと比較してパフォーマンスと機能を大幅に向上できる」と述べました。 そしてある意味で、これは真実だと信じています。今日のいわゆる「4G」ネットワークが2001年の3Gネットワ​​ークに似ていると主張する人はいません。 非常に高品質のストリーミングビデオを転送したり、大きなファイルをすぐにアップロードしたり、特定の条件下でこれらのネットワークの一部をDSLの代わりに使用したりすることもできます。 世代の飛躍のようですね！



WiMAX 2とLTE-Advancedが利用可能になるまでに「4G」と呼ばれるかどうかはわかりませんが、私はそうは思いません。これらのネットワークの機能は、現在存在する4Gネットワ​​ークとは大きく異なります。 正直に言ってください。マーケティング部門には何世代もの名前があります。
