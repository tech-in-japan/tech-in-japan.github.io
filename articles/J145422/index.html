Windows Azureツールの新しいバージョンがリリースされ、Pythonサポートが追加されました

今日、Windows Azureの更新（ Habréの詳細 ）および新しいサービスの大きな発表とともに 、Visual Studio 2010およびVisual Studio 2012用のWindows Azure SDK for .NET 1.7の新しい開発ツールが発表されました。











新しいサービスの発表の詳細は、このファクトシート （.docx）に記載されています。



さらに、開発者は次のような他の多くのツールを利用できます。



  • Windows AzureをサポートするVisual Studio Express 2012 RC for Webの特別無料版
  • WebMatrix 2 RCを更新し、Windows Azure、 Node.jsに基づくプロジェクト 、およびその他の多くの革新をサポート
  • PHP用の更新されたWindows Azure SDK
  • Node.js用の更新されたWindows Azure SDK
  • Python言語用のまったく新しいWindows Azure SDKツールキット-このWindows Azure Toolkitのリリースにより、Pythonはクラウドサービスのプラットフォームとして公式に歓迎されています。 オープンソースのツール

  • Windows Azure SDK for .NET-2012年6月（ ダウンロード ）
  • Web Platform Installerを介したVisual Studio 2010のツールボックスのインストール（ ダウンロード ）
  • Web Platform Installerを介したVisual Studio 2012のツールボックスのインストール（ ダウンロード ）
  • Visual Studio Express 2012 RC for Web for Windows Azureサポート（WebPI）
  • WebMatrix RC（WebPI）
  • PHP用のWindows Azure SDK（WebPI内）
  • Node.js用Windows Azure SDK（WebPI内）
  • Python用Windows Azure SDK（WebPI内）
