ウィークリービルドVivaldi 1.0.124.2

みなさんこんにちは！



Vivaldiブラウザの特別なビルドをリリースしました。番号は1.0.124.2です。 判明したように、以前のビルドのLinuxおよびMacバージョンはFREAK（SSL / TLSの脆弱性）から保護されていませんでした。 一般に、この脆弱性は古くから知られており、たとえば先週リリースされたテクニカルプレビュー2は、この新しい「人気のある」不幸から完全に保護されていますが、私たちの監視のために修正はウィークリービルドに含まれていませんでした。 そのため、必要なパッチを含む新しいアセンブリを準備しました。 更新されたバージョンは、次のリンクからダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット




次の変更も行われました。



  • FREAKの脆弱性に対するboringSSLの修正を追加
  • VB-3917キーボードがタブを閉じるときに正しいcloseTabアクションを呼び出す
  • マウスアウトでツールチップを適切に隠す
  • VB-4117無効にするとタブのツールチップが表示される




毎週のビルドはテストであるため、重要なデータをバックアップすることを忘れないでください。 前のアドレスで見つかったエラーを報告します 。
