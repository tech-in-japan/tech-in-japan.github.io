Sony Vaio Z-ほぼ1年一緒

エントリー



2009年8月に、Sony Vaio Z11MRNラップトップを購入しました。 ケースのステッカーが示すように、デバイス自体は2008年9月9日に生まれました。 だから一年中、私のペットは倉庫の周りをつまずき、そして/または店の窓に立っていました。







将来のデバイスを選択するための基準：

1）小型（理想的には13インチ）および軽量

2）理性の範囲内で強力

3）光沢がない（ケースのすべての部分と画面の両方に適用）





その結果、私はIBM（Lenovo）ThinkPad X200、AcerとSony Vaio Zシリーズ（Z11およびZ21）のペアを選択しました。 上記の3つの基準に加えて、Z'kaに私を「魅了」したのは何ですか？

1）重量1.5 kgの炭素繊維（一般的な炭素）の本体

2）スタイリッシュな外観（ちなみに、2009年8月には、Macの「別個の」ボタンを備えたキーボードを備えたラップトップはほとんどありませんでした。

3）HDMIポート



ラップトップの完全な仕様：



プロセッサー： Core 2 Duo P8400 2.26 GHz

RAM： 4 GB DDR III SDRAM 1066 MHz（それぞれ2048 MBの2つのストリップ）

ハードドライブ： 200 GBシリアルATA 5400 rpm + ハードドライブの Gセンサー電子衝撃保護

ドライブ： DVD + -RW / +-R DL / RAM

スクリーン： 1366x768 13.1インチLEDテクノロジーを備えたXブラック

ポート： 2 usb、IEEE139、メモリースティック™、SD、マイクジャック、RJ-11（モデム）、RJ-45（ネットワーク）、VGA、HDMI、オーディオ出力、Express Card

ワイヤレスインターフェイス： WiFi 802.11b / g、Bluetooth 2.0 + EDR、イーサネット10BASE-T / 100BASE-TX / 1000BASE-T

サウンド： Intel®High Definition

ビデオ： NVIDIA®GeForce®9300M GS 256 MbおよびモバイルIntel®GM45 Express



ここにソニー自身がZ11について書いているものがあります



Vaio Zシリーズは、機能性を犠牲にすることなく、モビリティの原則に基づいています。 独自のコンパクトな12層マザーボードにより、電力と快適性の最適なバランスが実現されます。 Vaio Zのケースはカーボンファイバー製であるため、ラップトップは軽量であるだけでなく（最大1.5 kg）、電源を入れたときに72 cmの高さから落下に耐えることができます。 ノートパソコンのキーボードは、装飾的なアルミニウムパネルで装飾されています。 13インチのスクリーンにはLEDバックライトが装備されています。



購入する



当時、価格はPrice.ruとYandex.Marketで58,000ルーブルから65,000ルーブルまで変動しました。 したがって、47,999ルーブル（および62,000ルーブルのバツ印）の大きな黄色の値札が付いた素晴らしいエルドラドチェーンスーパーでZ11MRNを見たとき、すぐにそれを取りました。 営業コンサルタントは、ラップトップが非常に長い時間窓にあったことを通知しました 店の偶然は、50kでラップトップを買うことを想像していません。 それから、延長保証と不必要な付属品の伝統的な「スチーミング」がありました。そして今、最後に、大切な箱が私の手の中にありました。



第一印象



最初のスイッチオン後、私は非常に大きな「ワウ効果」を得ました。 まず第一に、私は優れたマトリックスを備えたディスプレイに感銘を受けました。 明るく豊かな色、優れた視野角。 これまでのところ、ディスプレイはZ'keよりも良く見えていません。 画面への別の追加-技術仕様には「光沢」と書かれていますが、ストレッチなしのマットと見なすことができます（これは、このように宣伝するX-blackテクノロジーSonevskyのメリットです）。 ケースは優れた品質で、バックラッシュやきしみがありません。 プリインストールされたVistaは高速で実行されました。 当然、他のメーカーのSonevskyユーティリティと試用版ソフトウェアの巨大な束もプリインストールされていました。 VAIO Control Centerを除き、これらはすべて即座に削除されました。 これがないと、Fn + ...ボタンの組み合わせが機能せず、バッテリーケア機能が有効になります。



Windows 7への無料アップデート



この素晴らしいプロモーションを覚えていますか？ =）ラップトップは、その期間中に購入されました。 そのため、Sonyaのウェブサイトで起こっていることを正直にフォローし、7つの「無料」を取得することを期待しました（後で判明したように、配送コストを考慮しない場合）。 最初は、Sevenへのアップデートに関する正確な情報はなく、詳細情報Coming Soonが誇示されていました...その後、予約注文のモデルのリストが表示されました。 当然、卑劣な法則により、私のZ11MRNはこのリストに載っていませんでした。 リストで最も近い「兄弟」はZ21MRNでした。これは、指紋スキャナー（指紋認証用）と7200 rpm HDD（私の5400 rpm）の存在のみが私のデバイスと異なります。 鉄の残りは同じです。 ソニーは、余分な生地（および他のほとんどの企業）をハッキングすることに決めたと思います-新しい軸が必要な場合は、新しいハードウェアを購入します。 新しいハードウェアは必要ありませんが。 今、私は幸運にもアップデートシェアで飛んだと思います club.vaio.sony.ruフォーラムの人々は、このイベント全体に多くの唾を吐きます（私はpraflinkを与えることができません-Sony VAIO公式フォーラムで検索はありません...ほぼ1年後...



Windows 7 vsハイブリッドビデオドライバーvs HDMIオーディオ



さて、おそらく、私の記事の最も興味深い部分は次のようになります。 ラップトップでの技術的特性を見るとわかるように、2つのビデオカードがあります。STAMINAモードではMobileIntel®GM45 Expressがオンになり、SPEEDモードではNVIDIA®GeForce®9300M GS 256 Mbです。 このすべての幸せを一緒に機能させるために、ソニーは特別なハイブリッドドライバーを製造しています。 したがって、NvidiaのWebサイトから9300M GSで最新のfireをダウンロードすることはできませんが、VAIOが完了するまで待つ必要があります。 もちろん、繰り返しになりますが、意地悪の法則によれば、ソニーはドライバーを扱うことを本当に好みません。それはゆっくりと、しばしば不器用に行います。







Windows 7をインストールします。プロセスはスムーズに進み、すべてのドライバーが箱から出されました（7つのビデオドライバーがホイッスラーを「食べた」）。 ただし、小さなバグが明らかになりました。

1）SPEEDモードでは、画面の明るさを調整できない

2）ビデオカード間の切り替えは「バギー」であり、画面は何度もオン/オフ（ストロボのように、よりゆっくりと）し、多くの場合、次のようなことが起こりました。







致命的ではありません。 待つだけでした。 しばらくして、Windows 7のZ21にドライバーが表示されます。 ここでは幸せです、私は=）と思いました...そして、私は間違っていました。 私は、HDMIで接続している32台のPhilips TVで映画をよく見ます。 7歳未満の新しいfireで、私の音は消えます。 症状-テレビをHDMIで接続するときに、トレイのサウンドアイコンをクリックして再生デバイスを見ると、HDMI出力があり、「ボリューム」インジケータバーが「ジャンプ」していると思われます。

しかし、実際には音はありません。 ixbt.comフォーラムで同じ問題を抱えている多くの人々に会います。 私は単純にソニーのサポートに目を向けます。そこでは、「ラップトップモデルにはWindows 7は提供されていません。何もお手伝いできません。」 私はドライバーで実験し始めています：



Z11 for Vista-音があります

Z21 for Vista-音があります

Z21 for Win7-音が出ない

Vin7のZ690（アメリカの同等品）-音なし

Vin7のZ750（アメリカの同等品）-音なし

アフターマーケットドライバーVistaおよびWindows 7 32/64 Notebookreview.com-音が出ない



HDMIを介したサウンド、画面の明るさの調整、パフォーマンスの低下（Windowsによる）、グラフィックモード間の不器用な切り替えなどの選択肢があります。

なぜ私はHDMIケーブルを介してこの音に接続しているのですか？ 答えは簡単です-内蔵スピーカーは非常に小さく、大声ではありません。 要するに、なし。 その結果、私はそのようなデバイスを買いました。 USB経由で接続します（オーディオ信号と電力の両方を必要とします-多くのワイヤが好きではありません）。
