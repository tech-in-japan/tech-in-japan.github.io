ミンスクでのWeb Standards Daysに関するレポート

先週の土曜日、11月28日、「 Web Standards Days 」と呼ばれる最初のイベントがミンスクで開催されました。 この会議は、 Web Standardsコミュニティが主催する一連の定期的な無料の教育イベントを開きます。



会議はベラルーシのホテルの会議ホールで開催され、宣言された330人から約200人の参加者を集めました。 登録が完了し、すべての制限を超えた後でも、すべての参加者が会議に参加することを意図的に拒否しませんでした。 これはCPSU中央委員会の会議ではなく、330人の参加者全員が突然会議に参加することを決めた場合、サイドチェアに座ったり、足を踏み入れることさえできると考えました。







プログラム



会議の2日前に、登録されたすべての参加者は、会議の準備に関する短いダイジェストと更新されたプログラムを受け取りました。現在は5つではなく、7つものレポートがありました（一部のプレゼンテーションのファイルは後で利用可能です）：

  • すべてを時間通りに再配置します。 Andrey Sumin、 PDFでのプレゼンテーション、348 KB
  • エラー。 認識、和解、利益。 Vadim Makishvili、 PDFでのプレゼンテーション、3.53 MB
  • Webフォントと Web用のフォント。 Vadim Makeev、 Opera Show形式（HTML）でのプレゼンテーション、5.47 MB
  • HTML5 身近になります！ Pavel Lovtsevich、 PDFでのプレゼンテーション、1.71 MB
  • WebHiTech。 2009年の結果と2010年の計画。ArtemyLomov、 PDFでのプレゼンテーション、111 KB
  • VML、SVG、Canvas：次は何ですか？ Nikolay Matsievsky、 PDFでのプレゼンテーション、2.90 MB
  • 最新のWebテクノロジーの使用に関するいくつかの問題。 Konstantin Efimov、 PDFでのプレゼンテーション、144 KB
以下に、それぞれについてもう少し詳しく説明します。



すべてを時間通りに再配置する


このレポートは、大規模サイトの全面的な再設計と再コーディングのタスクに直面した人々にとって最も興味深いものでした。 アンドレイは、考えられるツールと一般的なアプローチをリストしました。そのおかげで、 HeadHunter Webサイトでこの困難な取り組みが成功裏に終了しました。



エラー。 意識、和解、利益


Vadim Makishviliによる新しいワンマンショーは、タグ#wstdaysによってtwitterフィードで最も多くの熱狂的な反応を集めました。 それは、バグやエラーなどのことであり、それが開発者自身の作業にどのように影響するかについてでした。 ユーモアと芸術性の健康的な部分で味付けされた非常に実用的なアドバイス。







Webフォントと ウェブ用のフォント


ヴァディム・マケエフは、組み込みの技術、またはますますウェブフォントと呼ばれる技術の開発の歴史的展望の概要を会議のゲストに提示しました。 Web用のシンプルなフォントから、IEを含むすべての最新ブラウザーで現在利用可能な最新技術まで。 最新のFlashおよびJS松葉杖はすべて、Webフォントを使用して交換することをお勧めします。



HTML5 身近になります！


Pavel Lovtsevichは、遠く離れた、しかし非常に切望されていたHTML 5、最新のブラウザーでのサポートのレベル、使用機能、そして実際にはXHTML 2ではなくHTML 5の新機能について話しました。







WebHiTech。 2009年の結果と2010年の計画


Artemy Lomovは、完成したばかりのWebHiTechコンテストの統計を会議のゲストに紹介し、ロシアのウェブサイト構築業界の主な傾向を概説し、今年の授賞式に全員を招待しました。ミンスク会議を部分的に繰り返します。



VML、SVG、キャンバス


ニコライ・マトシエフスキーは、何世紀も深く掘り下げて、最終的に現代のWeb標準であるSVGとCanvasに進化した特殊なWebグラフィックの開発の歴史的視点を表面にもたらすことを恐れませんでした。 ヒントの中には、Googleマップで複雑なマスクを使用する際にCanvasを支持するSVGに対する予期しない批判がありました。







最新のWebテクノロジーの使用に関するいくつかの問題


コンスタンティン・エフィモフは、さまざまなデバイスのマイクロフォーマットや特別なスタイルシートなどの最新技術の実装の特徴と最も一般的な間違いについて、自分のスタジオの経験を語りました。 レポートの主なアイデア-量は常に品質を意味するわけではありません。



印象



一般的な自発性と多くのオーバーレイ（およびそれらがない場合）にもかかわらず、会議が成功したことを願っています。 少なくともこれは、一日の終わりまで会場全体と参加者からの温かいフィードバックによって証明されます。 そして、最も重要なことは、Web標準について説明するだけでなく、質問をしたり意見を表明したりすることに関心があるすべての人の話を聞くことができたことです。 ところで、会議の最も活発な参加者は本を持って帰りました：エリック・マイヤー、ダン・セダーホルム、ニコライ・マトシエフスキー、その他。







来年初めの12月8日の会議に加えて、モスクワ、サンクトペテルブルク、サマラで少なくとも3回の会議を開催する予定です。 twitterチャンネルwebstandards_ruおよびHabréのブログ「 Web Standards 」をフォローしてください。



会議が開かれ、Pavel Lovtsevich、 ジグザグ 、Vadim Makeev、 pepelsbey 、およびゲストと組織を支援した人々の間で、次のhabrausersが気づかれました： bolk 、 chikuyonok 、 engel 、 banzalik 、その他、コメントに記載されることを希望します。 Vyacheslav Oliyanchukに投稿の写真と会議の開催を手伝ってくれてありがとう。



ビデオは少し後に利用可能になり、写真付きの詳細なアルバムは時間とともに更新されますが、すでにYandex.Photos：Web Standards Daysにあります。 ミンスク、11月28日 。 写真がある場合は、wst @ web-standards.ruに写真を送信するか、＃wstdaysタグを付けてTwitterに投稿してください。
