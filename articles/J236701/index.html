Intelモノのインターネット向けの予約済みEdisonボードマイクロコンピューター





CES 2014のIntelは、Intel Edisonプラットフォームのリリースを発表しました。これは、ワイヤレスデータ転送をサポートする、すぐに使用できる汎用コンピューティング環境です。 コンパクトなデバイスやウェアラブルデバイスを作成する発明者、起業家、開発者向けに設計されています。 Intel EdisonプラットフォームのマイクロコンピューターはSDカードのように見えましたが、最終リリースで大きく変わりました。 現時点では、開発者向けのツール（開発キット）がEdisonボードですでに利用可能であり、Linux（ RTOSだけでなく、 小型デバイス向けの特別なYoctoプロジェクト） を起動できます。



技術仕様

  • プロセッサー：SoC「タンジール」、2つのIntel Atomコア（Silvermont）プロセッサー（22nm）プロセッサー@ 500 MHzおよび32ビットIntel Quarkマイクロコントローラー@ 100 MHzを含む
  • RAM：1 GB LPDDR3（PoPメモリ）-2チャンネル32ビット@ 800MT /秒
  • ユーザーメモリ：4 GB eMMC（v4.51仕様）+マイクロSDカード用スロット
  • ネットワーク：802.11 a / b / g / n Wi-Fi（5 GHz、Broadcom 43340チップ）内蔵アンテナまたは外部、およびBluetooth 4.0
  • USB：1つのマイクロUSBコネクタ
  • プログラム可能な連絡先：

    • 2x UART（1つのフルフロー制御、1つのRx / Tx）
    • 2つのI2C、2つのチップセレクトを備えた1つのSPI
    • 1x I2S
    • 4つのPWMピンを含む12x GPIO


  • 周辺モジュールの接続：70ピンコネクタ（Hirose DF40シリーズ-1.5、2.0、または3.0 mmスタック高さ）
  • 電源：3.3〜4.5 Vの入力。 出力：3.3Vで100mAおよび1.8Vで100mA
  • 電力モード：スタンバイ（無線伝送なし）：13 mW; スタンバイ（Bluetooth 4.0）：21.5 mW（2014年第4四半期のBLE）; スタンバイ（Wi-Fi）：35 mW。
  • 寸法：35.5×25.0×3.9 mm
  • 動作温度範囲：0〜40°C


Edisonボードには、LinuxバージョンYocto 1.6で動作するIntel AtomとViperOS（Wind RiverのVxWorksの切り捨てバージョン）で動作するQuark MCUが含まれています。 開発者ツールには、Arduino IDE、C、C ++をサポートするEclipse、Node.JSおよびHTML5開発者向けのPythonおよびIntel XDKのサポートが含まれます。 Quark MCUのSDKおよびIDEはまもなく利用可能になります。 Intel IoT Analytics Platformは、限定的かつ非営利的な使用が可能なクラウドベースのデバイスソリューションです。

外観エジソン





Edisonボードはミニマルなデザインの基本モジュールであるため、Intelは2つのユースケースを開発しました。



Arduino用Intel Edison Board -4つだけのPWMサポートを除き、Arduino UNOコンタクトと互換性のあるBluetoothとWi-Fiを備えたボード（Arduino UNOには6つあります）







Intel Edison Debug Boardは、次のインターフェースと機能を備えた小さなボードです。

  • Edisonモジュールからの1.8 VI / Oのネイティブサポート
  • コンタクトはんだ付け用コンタクトの0.1″グリッドI / O配列
  • USB Micro Type-ABを搭載したUSB OTG
  • USB OTG（電源切り替えあり）
  • バッテリーサポート
  • USBブリッジ-UART、マイクロタイプB
  • DC電源（7〜15 VDC）




さらに、SparkFunはEdisonのスタックスキームに従って接続された独自のバージョンのブロック（モジュール）を開発しました

Edison用のSparkFunブロック。

Edisonおよびデバッグボードの再販業者であるSparkFunは、Arduinoのモジュールとの類推によって構築されたEdison“ SparkFun Blocks”用のスタッカブルブロックの独自バージョンを開発しました。 ブロックのコストは、I2CおよびGPIOバスを使用したベースの15ドルから、OLEDディスプレイとmicroSDカードブロックを備えた9自由度（9DOF）ユニットを含むEdison Sensor Packの70ドルまでさまざまです。





追加のユニットには、ADCサポート、Arduino、PWM（PWM）、コンソール、デュアルチャネルエンジンドライバー（デュアルHブリッジ）が含まれます。 バッテリー寿命のために、SparkFunは400mAh LiPoバッテリーモジュールを導入しました。
