Opera Linkは12月に終了します

My Operaが閉鎖されてから半年、 2年後でも、別のOperaサービスであるOpera Linkが同様の運命をたどりました。







「過度の労働によって獲得されたものはすべて失われる」ことを望まない人のために、私は猫の下で尋ねます。



Opera Sync Services開発チームからのメールは次のとおりです。

古き良きOpera Linkサービスに別れを告げる時です。



3月に、コンピューター、Android、iOS用の最新バージョンのOperaでデータを同期する新しい方法を導入しました。 新しい同期サービスへの移行により、同期手順を簡素化し、コンピューター、電話、タブレット間で同期されるデータ型の範囲を拡大することができました。



** Opera Linkは2015年12月に稼働を停止します**



その後、Opera Linkサービスを使用してブラウザを同期できなくなります。 さらに、 link.opera.com /のOpera Linkウェブインターフェースからデータにアクセスすることはできなくなります。



ただし、Opera Linkを閉じても、お気に入りのブラウザでデータを同期できなくなるわけではありません。 Operaの新しいバージョンをダウンロードすると、デバイスに応じて、ブックマーク、Expressパネル、開いているタブ、閲覧履歴、設定、パスワードを同期できます。



新しい同期方法への切り替えは簡単です。



1. www.opera.com/から新しいOperaブラウザをダウンロードしてインストールします。



2. Opera Linkのユーザー名とパスワードで新しい同期サービスにログインするか、単に新しいアカウントを作成します。 これを行う方法については、 www.opera.com / help / account＃signInをご覧ください。



3.アカウントを登録した後、当社のツールを使用して、Opera Linkから新しいブラウザに古いブックマークを転送します。 このツールは、 sync.opera.com / web /にあります。



新しいブラウザーをインストールできない場合（または望まない場合）、ローカルデータはコンピューターに残りますが、Opera Linkサーバーでは更新されません。 ブックマークを移動して、新しいサービスのウェブインターフェースからアクセスできるようにすることをお勧めします。 移行ツールは、 sync.opera.com / web /にあります。



最新のブラウザで新しい同期サービスをお楽しみください。



Operaをお選びいただきありがとうございます！

同期サービス開発チーム


また、 link.opera.comで次の警告が表示されます。

その後、Opera Linkサービスを使用してブラウザを同期できなくなります。 さらに、Opera Linkウェブインターフェースを介してデータにアクセスできなくなります 。



ただし、心配する価値はありません。 新世代のOperaブラウザーでは、ブラウザーに組み込まれた新しい同期サービスを使用して、ブックマーク、エクスプレスパネル、開いているタブ、閲覧履歴、設定、パスワード、その他のデータを同期できます。 Opera Linkからブックマークを転送する便利なツールも作成しました。



新しいOperaブラウザをダウンロードして、問題なく同期を使用してください。


節約しましょう！
