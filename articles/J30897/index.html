スイートベイ：極端なブランド変更

昨夜、私は「Sweetbay：Extreme Rebranding」というタイトルのボストンアメリカンマーケティング協会のイベントに参加しました。 すごかった！ メモを持ってきます。 （前述のVIAのケースからいくつかの段落を取りました）。



イベント



それはフロリダのKash N 'KarryスーパーマーケットがSweetbayとして生まれ変わった方法のプレゼンテーションでした 。 Kash N 'Karryは、フロリダのモールの西部にある103のスーパーマーケットのチェーンに失敗しました。 Wal-Martとこの市場の主な競合他社であるPublixからの価格競争の激化は、Kash N 'Karryが何かをするか、金融崩壊に直面することを意味しました。 Kash N 'KarryはVIAブランド戦略会社に、彼女が新しいブランドを作成するのを手伝ってくれるよう頼みました。VIAはすでにHannaford Brosスーパーマーケットで成功した経験があったからです。 Delhaize Groupが所有するCo。



未来のブランドの本質を呼び起こすために、VIAは5つのブランドテーマの開発から始めました。 各トピックは、同社が市場で消費者を念頭に置いた戦略的地位を反映したものでした。 広範な調査の後、Kash N 'Karryは、食品取引の狂信者としての地位を確立している企業には市場にニッチがないと判断しました。 Publixの支配的な競争相手は、ブランドポジショニングの一部として食品を含めましたが、Publixにとって、食品はメインテーマではありませんでした。 Kash N 'Karryは、主に食品を専門とするスーパーマーケットのように、消費者の心に落ち着く機会があると判断しました。 VIAは会社のネーミングエクササイズを実施しました。13人の会社のトップマネージャーが作業に関与し、スーパーベイの名前を選択できます。





スイートベイはフロリダ州で育つバージニアのマグノリアであり、地元の人々はそれをよく知っています。 この名前は、甘い（甘い）-デザートまたはスイーツ​​-という言葉のおかげで食べ物のイメージを呼び起こし、ベイ（湾）は、会社自体のあるタンパベイを消費者に思い起こさせます。 その後、VIAはKash N 'Karryのロゴ開発を支援しました。 土壇場で、Delhaize Groupのディレクターは、Hannaford Brosのロゴに一部関連していたため、提案された3つのオプションの2つ目を選択するようアドバイスしました。 Coは、VIAがDelhaizeのためにかつて開発したものです。 ロゴには、スイートベイの文字「Y」の近くにある食べ物、リンゴ、パン、カールの象徴的な画像が含まれており、会社の従業員が仕事に対して持っている情熱を示しています。



テーマを選択し、エッセンスのデザインで仕上げたSweetbayは、新しいブランド戦略の一環として、店舗全体とそのイメージのデザインに着手しました。 さらに、販売予定の食品の品揃え、店舗での販売方法、価格戦略についても慎重に検討しました。 Sweetbayは必ずしもKash N 'Karryの既存の顧客と仕事を続けることを追求していませんでした。彼女は、彼女が提供する食品の大ファンであり、彼女の品質に対して適切な高価格を支払う意思がある消費者を引き付けるつもりでした。 スイートベイは、従業員に対して献身的なことだけでなく、食料問題に関する知識も確認するための強力なトレーニングを実施しました。



Sweetbayは、新しい店舗のオープンを記念して10か月のプロモーションを開始しました。 キャンペーンの戦術には、屋外広告、メーリングリスト、店舗内の資料、ラジオプログラムのスポンサーシップ、印刷広告、サイトサイン、新しいWebサイトが含まれます。



結果



スウィートベイの新しい店舗の売上は、カシュ・N・カリーの以前の売上より30〜40倍増加しました。 Publixは、マーケティングメッセージを再設計し、食品に焦点を当てて対応しました。 Sweetbayは現在、既存のKash N 'Karry店舗を新しい名前Sweetbayに従って改造しており、5〜3年を費やすことを望んでいます。 この物語から私が学んだ教訓は、Sweetbayが市場を実際に分析して空のニッチを特定し、それをターゲットカテゴリーである食品のリーダーとして位置付ける戦略の開発を開始したことです！



スイートベイのマーケティング部長であるスティーブ・スミスは素晴らしい話をしました。 彼は、彼の会社がこれらすべてをどのように経験したかを熱心に説明しました。 さらに、クライアントおよびプロセスの参加者として、スウィートベイのロゴを選択するディレクターの洞察から始まり、デザインの提案が受け入れられて商品のパッケージを整理するようになった理由で終わる、内側からブランド変更がどのように配置されるかについて話しました。 VIA-groupのヘッド兼共同設立者であるJohn Colemanは、会社のブランディングの重要性について素晴らしいレビューを行い、VIA-groupのクライアント戦略の責任者であるJulia Bradyは、ブランド戦略プロセス全体を検討しました。 、スティーブが作った後。
