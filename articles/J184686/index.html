オラクルとマイクロソフトのパートナーシップ：Windows AzureクラウドのOracleデータベース、WebLogic Server、Oracle LinuxおよびJava

2013年6月24日、長年のライバルであり競争相手であるOracleとMicrosoft は、新しい戦略的パートナーシップを発表しました 。 新しい契約の下で、両社は、Microsoftテクノロジーを使用する仮想化環境およびクラウド環境でのOracle製品（Windows AzureクラウドプラットフォームとHyper-Vハイパーバイザー）をサポートするために力を合わせています。







公式のプレスリリースは、 次のリンクで読むことができます。



サポート



この契約に基づき、オラクルは、Windows Azureクラウドプラットフォームを含むマイクロソフトテクノロジー上で動作する製品を認定、サポート、およびライセンス供与します。



6月24日以降、Oracleライセンス製品ユーザーは、 更新されたライセンスにより、Windows Azureクラウド環境をホストするためにそれらを使用できます。



製品



戦略的パートナーシップの一環として、企業は、Windows Azureクラウドに簡単に展開できるように、Oracle DatabaseとOracle WebLogic Serverの既製の事前構成済みバージョンをユーザーに準備および提供する意向を発表しました。



さらに、ユーザーはOracle Linuxベースの仮想マシンのイメージにアクセスでき、Windows Azureのクラウドベース環境に展開する準備ができます。



Java



ご存知のとおり、Windows Azureは、Javaプラットフォーム上でのアプリケーションの開発と展開を、その存在の最初の日からサポートしています。



新しい契約の一環として、MicrosoftとOracleのパートナーシップがWindows Azureのプラットフォームサポートを改善し、JavaプラットフォームとOracleの環境に対する完全なライセンスとサポートを追加すると発表されました。
