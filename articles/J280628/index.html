こんにちは友人！ 述べたように、2月26〜27日に、C ++ Russia 2016という有名な名前でコミュニティ会議が開催されました。



何があり、どのように？ 前方-カットの下で！



コンファレンス



フラッグシップのロシアC ++-もちろん、スピーカーとレポート（ノートブックとそれ自体でハングアウトしますが、それでも...）。 今回は、プログラム部分が2015年モスクワと比較して量的に成長しました。2日間、3トラック、25のレポートがありました。 プログラムのトピックは非常に多様でした。



  • ゴー・ニシャノフ（コルーチン）の魅力的な物語から始まりました
  • 英語のレッスンは、Joel Falcou（Boost.SIMD）、Kirk Shoop（リアクティブプログラミング）、Dori Exterman（並列コンピューティング戦略）によるゲストレポートで思い出されました。
  • すりおろしたスピーカーAlexander Fokin（Reflection）とDmitry Nesteruk（Patterns）とは異なることを研究しました
  • Anton Polukhin（Boost）からのインサイダー情報を聞いた
  • 投手ミハイル・マトロソフと野球をした（カジュアルC ++）
  • 同様に、はるかに、はるかに


プレゼンテーションのスライドがアップロードされ、 ビデオは先日放送されました-見て、勉強して、喜んでください！



ホテルのスペースとコーヒーが許可されたため、約400人が集まった仲間の職業と、スタンドのスピーカーと会社の代表者とのコミュニケーションの良い部分でスナックを食べることができました。



ビジネスには、カスペルスキー、JetBrains、Artec 3D、ProgramPro、Luxsoft、Yandexが含まれていました。 スタンドのチームは、コミュニケーション、チップ、パンに満足しており、オリンピアードプログラミングを使用して情報から爆発する脳をロードしようとしました！



食前酒



会議の前夜、サンクトペテルブルクのオフィスでのホームイベントは、JetBrainsとYandexによって組織されました。



JetBrainsは、 JetBrains C ++ナイトパーティーで人々を集めました。 ここでは、結局のところ、会社の製品ラインについて、開発者であるミニインサイダーと直接話すことができました。 JetBrainsは、Resharper、CLion、およびHPCレポートのバリエーションも導入しました。



Yandex は、C ++で何が起きているのかというトークショーを開催し、外国のゲストをC ++ロシア会議に招待して会話をさせました。 彼らは長い間おもしろい話をしました。 C ++ 17、モジュールとコルーチン、モジュールと概念、モジュールと不良マクロ、そしてもちろんもっと多くについて、楽観的です。



食前酒は非常に高品質でした。 素晴らしいイベントを提供してくれた企業のおかげで、Yandexオフィスでお茶を運転するのは面白かったです）



だから、彼は結果です



実際のC ++マラソンは、冬の最後の日にピーターによって発行されました。 私たちの国のC ++は愛され、高く評価され、知られているようです（少し）。 そして、自分自身を見せ、他の人に会い、知り合いになり、コミュニケーションをとり、何かを学び、彼らの工芸の達人を見て、図書館や言語を作る人たちに、能力とスキルを生み出し、発見とトリックを共有する-これはすべてよく組織されたC + conによって可能になりました+ロシア。 これらすべてのために、私たちは思いやりのあるプログラマーと企業のコミュニティを持っています。 やった！



一方で、世界中から優秀なスピーカーを連れてくることは、実は非常に現実的です。 私の意見では、Gor Nishanovは会議のスターになりました-レポートでも、オープンなコミュニケーションでも、そしてロシア語でも。 また、Boost.SIMDの機能とソースから判断して、Joel Falcouはメタプログラミングの闇の力と共謀しています。 毎日、廊下でDori ExtermanがIncrediBuildの実装が非常に難しいことと、会社がC ++でのモジュールの出現を恐れていないことをどのように伝えるかを毎日見ているわけではありません。
