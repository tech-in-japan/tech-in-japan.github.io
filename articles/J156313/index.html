MacBook Pro 15” 2011年後半の個別グラフィックスアップグレード：ビデオメモリを512MBから1GBに再はんだ付け





Appleコンピューターのアップグレードに関する一連の投稿を続けています。 今回は、MacBook Pro 15”（A1286）2011リリースについてお話します。 私の意見では、非常に価値のあるデバイス：クアッドコアIntel Core i7プロセッサー、Ati 6シリーズグラフィックス、16GBの1333MHz RAMのサポート。 そのパフォーマンス、そして今日では、ほとんどのタスクに十分です。



Appleはこのモデルの4つの構成をリリースしましたが、これらはプロセッサとグラフィックスが異なります。 詳細な仕様については、表を参照してください。 実験は、MD 318（i7 2.2GHz、Ati 6750M（512MB））の完全なセットで実行されました。 512MBの容量のネイティブビデオメモリ（vram）、より容量の大きいチップにはんだ付けされ、1GB vramを取得しました。

すべての構成は、同じプラットフォームと同じEFIファームウェアを備えています（したがって、「ビデオBIOS」は同じです）。 したがって、それらのいずれでも、グラフィックの構成を変更できます。 3つのチップのいずれかと異なる量のビデオメモリ（256、512、または1024MB）を掛けることができます。 構成の選択は、「ストラップ」抵抗器によってボード上で実行されます。



キーポイントについて詳しく説明しますが、BGAはんだ付けプロセスについては説明しません。希望する人は誰でもこの方法を個別に「グーグル」できます。



ボードを分解して入手します。











ボードには4つのHynix H5GQ1H24AFR T2Lビデオメモリチップがあります。 これは、1Gbit GDDR5 SGRAM（128MBシングルチップ）と5Gbpsの転送速度です。







チップをはんだ付けします。







その代わりに、Samsung K4G20325FC-HC04を掛けます。 これは、同じ転送速度5Gb / sのGDDR5 SGRAMでもありますが、すでに2Gbit（256MB 1チップ）の容量があります。 カスタム構成でインストールされるのはこのようなメモリで、グラフィックスには1GBのビデオメモリがあります。



新しいチップ。







結論として、はんだ「ストラップ」抵抗。 結果の構成に従ってインストールします。



ストラップ抵抗。







「ストラップ」の意味は、マザーボードの回路図に示されています。 ストラップの異なる組み合わせは、異なるグラフィック構成（ビデオチップモデル、メモリメーカー、メモリサイズ）に対応します。



ストラップ付きの回路の一部。





これですべて準備完了です。 ケシを集めて結果を楽しみます。



前後のスクリーンショット。









シネベンチのテスト結果。







在庫ピッキングMD318から、カスタムMC723を作成したことが判明しました。 Ati 6770でビデオチップを再はんだ付けすることは可能ですが、残念ながらこれは利用できませんでした。 さらに、ビデオチップはとにかく変わりました。 このポピーは、ビデオチップの誤動作により修理のために持ち込まれました。 所有者に実験を依頼し、ビデオメモリをはんだ付けしました。
