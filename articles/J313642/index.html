マイクロソフトはCognitive Toolkitの更新を導入しました





昨日、Microsoftは公式ブログで、Microsoft Cognitive Toolkitのアップデートを公開しました。これは、深い、畳み込み、およびリカレントネットワークを設計およびトレーニングするシステムです。



さらに、Cognitive Toolkitは強化学習を使用する機能を提供します。 Cognitive Toolkitは、オブジェクト認識、音声、関連する結果の検索などの分野で使用するために提案されています。



Cognitive Toolkitは、GitHubで利用可能な Computational Network Toolkitという別の会社の開発に基づいています 。 また、CNTKは新しいバージョンと名前を受け取ったと直接述べています。「CNTKはMicrosoft Cognitive Toolkitになります」。



CNTKは、Habréの公式Microsoftブログを含む、今年1月にツールとして発表されました。 当初、このツールはGoogleのTensorFlowおよびBaiduのwarp-ctcのライバルとして設計されました。



Cognitive Toolkitの強みは、従来のPCに基づいて動作し、NVIDIA製の強力なGPUでCPUとゲームグラフィックカードの両方を使用できることを示しています。



「Microsoft Cognitive Toolkitは、MicrosoftとNVIDIAの間のテストコラボレーション製品であり、その目標はコミュニティでの粘土教育の進歩です」と同社の公式ブログは述べています。



CNTKコア、およびその結果としてのCognitive Toolkitは、C ++で実装されており、C ++およびPythonで制御できます。 最新の更新（およびCNTKからCognitive Toolkitへの移行）により、Python APIおよびFast R-CNNアルゴリズムのサポートが追加されました。
