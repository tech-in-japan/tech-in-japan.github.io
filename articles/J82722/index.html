NIXP.RU v3.0 Beta-UNIX / Linux / FLOSS愛好家向けのポータル

やっとやった！ おそらく、UNIX、GNU / Linuxシステム、およびオープンソースの分野の愛好家の多くは、2002年（および第3レベルドメイン-2001年）に立ち上げられた古き良きリソースnixp.ruを知っています。 長い間、それは実際には更新されませんでしたが、数年前に私たちは集まり、すべてを再考し、ゆっくりと開発を始めました。 今日、私たちはタイムラインで「v3.0 Beta」と呼ばれる最初の公開マイルストーンに到達しました。







プレスリリースから：



「NIXP.RUの新しいバージョンv3.0は、サイトの機能を拡張して新しい外観をもたらしただけでなく、リソースの非常にパラダイムを変更し、ソーシャル指向のポータルに変えるというアイデアで作成されました。 NIXP.RUで利用可能なすべてのオブジェクト（ニュース、記事、レシピ、プログラム、リンク）に対するコメント、タグ、投票として導入されたこのような革新は、ロシア語を話すオープンソースコミュニティの発展と結集を促進するように設計されています。 NIXP.RUの新しいデザインを準備する際に、「厳格なWeb 2.0」の概念が適用されました。その主なアイデアは、情報出版物の古典的なデザインとソーシャルWebリソースの最新トレンドの組み合わせです。





使用されるテクノロジーについて簡単に説明します： symfony 、Prototype、PHPUnit + Selenium。 詳細はこちら 。 さらに、お気に入りのPerlを使用して、プレーンテキストに保存されている古いnixp.ruから古いデータのヒープを変換しました:-)
