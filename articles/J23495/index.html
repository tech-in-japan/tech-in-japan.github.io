レゴ用MP3プレーヤー

この小さなデバイスは、有名なレゴセットのレンガではありません。 このブリックは本物のMP3プレーヤーです。 珍しいデザインとわずか26グラムという非常に軽量なため、どこでもお気に入りの音楽を聴くことができます。







機能：

* MP3およびWMAをサポート。

*最大2GBのmicroSDをサポート。

*バッテリー駆動（動作時間は表示されていません）;

* Windows 2000 / XP / Vistaをサポート。

*利用可能な色：青、黄、赤、白、緑。

*サイズ：28 x 58 x 17 mm;

*重量：26グラム;



デバイスのコストは45米ドルです。



デバイスのフルネーム： HOMADE MP3 i-Player 。
