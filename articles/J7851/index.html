Windows Mobile 6は音声を認識します：開発者からのビデオ

Channel 9ビデオブログのメッセージのおかげで、Windows Mobile 6.0モバイルデバイス用の新しいOSの最も興味深い機能を詳細に調べることができます。 それを搭載したスマートフォンとコミュニケーターは、確かにすぐに発売されます（ただし、遅れはあります）。



開発者の観点から見ると、最も注目すべき革新の1つは、.NET CF 2.0およびSQL Server Compact Edition 2.0の内部メモリ（ROM）への導入です。 つまり、環境の実行を心配することなくプログラムを作成できるようになりました。



Windows Mobile 6.0の作成者（ WMV 、28分、222 MB）の1人とのインタビューで、一般ユーザーにとって興味深いシステムの新しい機能がいくつか示されています。 デモンストレーションは、HTC Excaliburスマートフォンで行われます。 気難しい人がトラフィックを無駄にしないように、スクリーンショットと簡単な説明を公開しています。



内蔵の音声認識システムにより、電子メールの操作がはるかに簡単になります。 検索は、文字を発音するときにリアルタイムで実行されます。 その人の姓の最初の文字に名前を付けます。内蔵メールクライアントの画面には、その人からのすべての文字のリストがあります。 メッセージの件名での全文検索も同様に機能します。







2つのレベルのドロップダウンメニューを使用して、コンテキストコマンドにすばやくアクセスできます。 便利に実装された「ホット」キー。 たとえば、手紙に答えるには、テンキーの「2」キーを押したままにし、既読または未読としてマークする必要があります-「3」キー。







数回クリックするだけで、自動返信用のテンプレートメッセージの1つを選択するか、新しいメッセージを発声できます。







同期機能が大幅に改善されました。 サーバーのメッセージアーカイブには全文検索があります：検索サーバー機能。 検索の時間間隔を自分で設定します。週、月、3か月、または常時、検索要求を形成し、それがExchange Serverに送信されます。 15秒後、最初の答えが来ます。







メールクライアントには、HTMLメッセージを表示する機能があります。







メールクライアントに加えて、新しいシステムには他の興味深いプログラムが含まれています。 たとえば、個人の財政支出トラッカーを管理するための簡単なプログラム。 ちなみに、情報の音声入力もサポートしています（費用、金額、通貨のカテゴリを明確に述べる必要があります-単語のセットが非常に限られているため、プログラムはトレーニングなしで非常に確実に動作します）。







一般に、音声認識および合成エンジンはシステムに組み込まれており、独自のプログラムで簡単に使用できます。 プログラムコード内で直接認識される必要がある単語を示します。 プログラマーは、デスクトップで経費トラッカーのソースをデモンストレーションします。このトラッカーでは、これらすべてが実装されています。







さらに、インタビューで、Windows Mobile部門のプログラマーは、デバイスに.NETおよびSQL Serverサポートを実装した方法と理由を詳しく説明しています。
