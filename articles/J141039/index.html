4月5日の「Active Directory Audit」の無料ウェビナーに招待します

既に構成されたAD監査（ http://habrahabr.ru/company/netwrix/blog/140569/ ）？

ただし、ご存じのとおり、「ネイティブ」ツールでは、管理者が直面するすべてのタスクを解決できない場合があります。 Active Directory組み込みの監査ツールも例外ではありません。



そのため、 4月5日12:00モスクワ時間に開催されるActive Directoryの監査専用の無料のウェビナーに招待します。 NetWrixのロシアのオフィススタッフは、AD監査の問題について話し、 NetWrix Active Directory Change Reporterソフトウェアソリューションがどのように機能するかを示します。







ウェビナープラン：



1. Active Directoryの監査の目的 。

2. Active Directoryの変更を監査するためのNetWrixのアプローチ 。 ソフトウェアソリューションの概要

3. Active Directory Change Reporterプログラムの作業 ：

*プログラムのインストールと初期セットアップ

*変更およびADスキーマに関するレポートを取得できます

*数回のクリックでADオブジェクトから削除されたものを復元する

*プログラムとビルトインAD監査ツールの違いは何ですか

4.変更およびプログラムの機能の監査に関する参加者からの質問 。



ウェビナーの所要時間 ：1時間（12: 00-13 :00）



NetWrix Active Directory Change Reporterの概要：



この製品は、ADおよびグループポリシーで行われたすべての変更を監視し、すべての変更に対して変更されたWHO、WHAT、WHENおよびWHATに関する情報を反映するレポートとアラートを生成します。 このプログラムでは、レポート送信のスケジュールをカスタマイズできます。 レポートには、すべてのユーザー、グループ、コンピューター、組織単位、グループメンバーシップ、アクセス権、Active Directoryサイト、オブジェクトおよびグループポリシー構成、ADスキーマ、およびその他すべての種類のオブジェクトの追加、削除、および変更が表示されます。

各変更に関連して、変更前のユーザーの以前の名前や、変更前のOUへのアクセス権のステータスなど、変更されたパラメーターごとに「変更前」および「変更後」の値が固定されます。 この製品を使用すると、現在の日付のADの「スナップショット」（「ドメイン管理者グループのすべてのメンバー」）または「スナップショット」を作成する特定の時点（「ドメイン管理者グループのすべてのメンバー」 2008年12月31日」）。
