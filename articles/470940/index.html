MSK VUE.JS meetup # 3 at Mail.ru Group: materials from mitap





In September, our office hosted a big mitap for development on the progressive framework vue.js. The event was attended by 400 people - this is good news. There were 3 reports in the program: the Delivery Club team spoke about the technological side of their new web version of the client site. The guys from MSK VUE.JS shared their experience in organizing the API. And the leading developer of Nuxt.js talked about the main features of the new version of the framework. Under the cutter, we collected materials from the event for you.





"The specifics of rendering vue.js applications on the server"

Alexey Klyuyev, Head of front-end team Delivery Club, Mail.ru Group



A large public client application might need server-side rendering. Possible reasons: the need for search engine optimization and the struggle for productivity.

What challenges await the developer in this case? What decisions will have to be made? How difficult will it be to achieve the set goals in terms of front-end development?

  • Vue.js is preparing v3, what's new for us in Nuxt.js?
  • A bit about how Nuxt.js is built from the inside out.

"Organization of work with API on Vue.js"

Vitaly Kopachev



Report on the approaches to structuring code in a project. How are code writing and entity extraction related? How does this help in work? Analysis of the approach on the example of the implementation of the API service.
