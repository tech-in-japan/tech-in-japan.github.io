Samsungは9月14日にApp Storeを立ち上げました

開始時に、 Samsung Application Storeは 300以上のアプリケーションを提供します。 EA Mobile、Gameloft、Handmark、Capcom、Taito、Com2usなどのコンテンツプロバイダーは、初日にアプリケーションを起動します。



サムスンは、Appleのアイデアを使用する7番目の携帯電話メーカーになります（そのストアは2008年8月に発売されました）。

Google（ Androidマーケット ）、BlackBerry（ BlackBerry Application Storefront ）、Nokia（ Ovi Store ）、LG（ LG Application Store ）、Sony Ericsson（ Play Now ）、Palm（ Palm Software Store ）にはすでに同様のものがあります。
