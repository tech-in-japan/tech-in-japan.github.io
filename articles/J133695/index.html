Microsoftは3年前からスタートアップを照明しています！





11月、 Microsoftは3年間の主力スタートアッププログラムであるMicrosoft BizSparkを祝いました。 このプログラムは、私たちの国の2500を含む世界中の45,000のスタートアップがロシアで利用可能なすべてのマイクロソフトソフトウェアに絶対に無料でアクセスできるようになったおかげです。



ロシアのBizSparkの責任者として、私はプログラム参加者と頻繁にコミュニケーションを取り、BizSparkについて感謝の言葉と肯定的なレビューを聞きます。 このプログラムにより、数万ドルを節約し、開発に専念し、ライセンスについて考えることなく、迅速な技術サポート、推奨事項、有用な知識、およびつながりのために...という事実に感謝します。



BizSparkは最高のMicrosoftプログラムの1つであると心から言うことができます。それは友人に勧めるプログラムです。



この出版物を機に、BizSparkの利点、プログラムに登録して最大限に活用する方法についてお話したいと思います。 また、短期間で重要な結果を達成したプログラムの成功した参加者に注意してください。



BizSparkの利点



そのため、Microsoft BizSparkメンバーは、インターネット上での製品の設計、開発、テスト、配置、およびMicrosoftおよび認定BizSparkコミュニティパートナーからのサポートのために、完全に無料のMicrosoftライセンスソフトウェアを取得します。



  • 3年間、各参加者には以下が提供されます。
  • MSDNサブスクリプション （Visual Studio Team Suite、Team Foundation Server、Expression Studio、Windows Server、SQL Server、Office、Dynamicsなど）を備えた最大のVisual Studio Ultimate開発者パッケージの一部として提供されるすべての開発ツール、サーバーおよびクライアントMicrosoftソフトウェア- ソフトウェア製品の設計、開発、テスト （スタートアップ開発者ごと）。
  • Microsoftサーバー製品 （Windows Server、SQL Server、Office SharePoint Portal Server、System Center、BizTalk Server、Dynamics CRM）のライセンス- インターネット上のサービスをホストするため （ユーザー数を制限せずに）。
  • 3年後、「卒業生」は、BizSparkプログラムを通じて取得したほぼすべてのソフトウェアを保存し、特別な優先条件でマイクロソフトソフトウェアを更新するためのサブスクリプションを購入できるようになります。
  • フリーソフトウェアに加えて、BizSparkメンバーはMicrosoft テクニカルサポート と MSDN リソースへのアクセスを受け取ります 。
  • また、 国際的なBizSpark Connect カタログの 参加者となり、 投資家やパートナーを見つけるのに役立ちます。
  • さらに、BizSparkメンバーは、次のようなBizSpark コミュニティの認定パートナーの特別オファーを利用できます 。投資、特別な条件でのホスティング、 技術およびビジネストレーニング、電子簿記および支払いシステムなどの割引。 BizSparkには世界中に2,000以上のスタートアップパートナーがあり、ロシアには数十のスタートアップパートナーがいます。 Microsoft Startup Portalのカタログを使用して、必要なロシアのパートナーを見つけることができます。
メンバーになるのは簡単です！



参加条件：

-初心者の起業家または新興企業は、 ソフトウェアに基づいて複製可能なソフトウェア製品またはサービスを開発する必要があります。 この製品またはサービスは、あなた自身のビジネスの重要なコンポーネントでなければなりません。

-会社は非公開で、 3年以内に営業し、年間収入が50万米ドル以下である必要があります（収入はロシアの場合）。 法人の登録プロセスのみを行っているか、このプロセスを開始していない個人もプログラムに参加できます。



プログラムへの登録と参加は無料です！



メンバーになるには、リンクを取得してBizSparkコミュニティパートナーまたはマイクロソフトに登録し、登録する必要があります。 ビジネスジャーニーの開始時に役立つプログラムおよびその他のリソースに関する詳細情報- スタートアップ向け Microsoft ポータル （ www.ms-start.ru ）

個別のスタートアップサポートプログラム







Microsoft BizSpark Oneは、深刻な成果と意図を持つスタートアップ向けの国際的な詳細なサポートプログラムです。 特定のスタートアップの固有のニーズに合わせて調整された単一の「エントリポイント」やコラボレーションプランなど、選択された参加者はマイクロソフトと個別に対話する機会を得ます。 これには、共同マーケティング、ビジネストレーニング/コンサルティング、メンタリング、テクノロジー、投資支援、PRなどが含まれます。 現在、ロシアにルーツを持つ4社がプログラムに参加しており、彼らのビジネスは活発に発展しています。



たとえば、クラウドベースのERPシステムとWindows Azureベースのビジネスアプリケーションを開発するプラットフォームを作成したAcumaticaは 、パートナーチャネルを100以上のインテグレーター/リセラーに拡大し、コールバーグクラヴィス＆ロバーツ。 インタラクティブ（クリック可能な）ビデオを作成および投稿するためのクラウドベースのプラットフォームを開発したClickBerryは、ビデオ広告市場の主要プレーヤーとパートナーシップ契約を締結することで米国市場に参入しました。 また、Windows Azureで実行されるNews360ニュースアプリケーションは、サポートされている6つのプラットフォームすべてに搭載されており、インストール数は100万を超えています。 若いスタートアップのFittingRealityは、Microsoft Kinectセンサーを使用したソリューションのプロトタイプを、2011年10月にカリフォルニアで開催されたMicrosoft VC Summitで100を超える主要な国際ベンチャーファンドに紹介する機会を得ました。 現在、マイクロソフトのサポートを受けている会社は、製品の商用バージョンを積極的に開発しています。







2番目のプログラム-Microsoft Startup Accelerator-は、ビジネスの成長を加速させ、BizSpark Oneに参加する可能性を高めるために、2500社を超えるロシアの新興企業の中からMicrosoft BizSparkに潜在力のある少数の若い企業を特定することを目的としています。 マイクロソフトのロシア駐在員事務所は、技術、マーケティング、およびビジネスリソースへのアクセスを提供することにより、これらの企業と個別に協力しています。 過去1年間で、300社以上が初期審査に合格し、70社以上が詳細な審査を受け、25社が選考委員会の対面会議への参加に指名され、9社が参加者として選出されました。 これらの企業のうち2社がBizSpark Oneに参加しました。



スタートアップが市場によって確認されたビジネスモデルを持っている場合、新しい市場を形成する製品を作成している、または既存の市場で大きなシェアを獲得し、Microsoftの技術とMicrosoftとのパートナーシップに興味がある場合は、 rusaccel @ microsoftに会社情報を送信してください。 com 。
