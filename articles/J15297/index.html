RamblerがIMHO VIコンサルタントを採用

Kommersantによると 、Rambler Mediaはインターネット広告代理店Index20の株式をVideo InternationalグループのメンバーであるIMHO VIに売却することに同意しています。 確かに、1つの条件があります。 このオプションを活用するには、IMHO VIは2008年のディスプレイ広告からの収益を2倍にして、2008年には4,000万ドルにする必要があります。現在YandexとMail.ruの広告を販売しています。



IMHO VI Arsen Revazov（写真）の会長は、売却についてはまだ議論されていないと説明した。 最初に、IMHOはIndex20チームに販売の改善と新しい技術の導入について助言します。 「2008年の結果によると、ランブラーがメディア広告で4,000万ドル以上を私たちの助けを借りて獲得すれば、IMHOはIndex20の共同所有者になるでしょう」と彼は言いました。 「オプションのサイズは現在交渉中です。」 おそらく、IMHO VIは現金で何かを支払う必要すらなく、サービスへの支払いとして株式が譲渡されます。



現在、Index20 CJSCはRambler Media Ltd（1％）およびIndex20（Europe）Ltd（99％）を通じてRambler Media Groupによって完全に管理されています。 2006年のRambler Media Groupインターネット部門の収益は、29.9百万ドル、EBITDA-300万ドルに達しました。2007年上半期のこれらの数値は、それぞれ1840万ドルと540千ドルでした。 同社によると、この収益の50％はディスプレイ広告（バナー、ポップアップ、その他の形式）によるものです。



IMHO VIエージェンシーは1998年から存在しています。 2002年、Video Internationalグループに参加しました。 Yandexのホームページとメール、Mail.ruのホームページ（2007年末まで）、Auto.ru、Afisha.ru、Gazeta.ru、Newsru.com、その他のポータルで広告を販売しています。 専門家によると、2006年の売上高は4500万〜5千万ドルでした。



10月、IMHO VIはRambler Media広告スペースの一部の販売に関する2008年の契約に署名しました（Rambler.ru、Lenta.ru）。 契約には、Rambler.ruとLenta.ruのフロントページにある3つの広告スポットの1つ、およびRambler.ruのメールサービスが含まれます。 Rambler Mediaに対するIMHO VIの義務額は、年間1400万ドルです。 したがって、そのオプションを実現するために、IMHO VIはIndex20が2,600万ドル以上を調達するのを支援する必要があります。
