WiFi 802.11 n





5年間の開発の後、新世代のWiFi（ワイヤレスNとも呼ばれる）が9月にリリースされます。

新しいから：

-最大600 Mbpsの転送速度（現在WiFi 802.11gでは最大54 Mbps）

-マルチチャネル入力/出力、MIMOとして知られています

-2.4-2.5または5.0 GHzの範囲で動作

-より多くのカバレッジ。



さらに、802.11nデバイスは3つのモードで動作できます。



-継承（レガシー）、802.11b / gおよび802.11aデバイスのサポートを提供

-混合、802.11b / g、802.11aおよび802.11nデバイスをサポート

-「純粋な」モード-802.11n（このモードでは、802.11n規格で提供される速度とデータ伝送距離の増加を活用できます）。
