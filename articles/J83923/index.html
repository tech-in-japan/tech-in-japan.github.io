フィルターフォージ：ハート型フレーム（レッスン）

このレッスンでは、陽気なフレームの作成方法を学習します-すべての美しい輝く心の中で。 これを行うには、ペンで長く痛みを伴う描画を行う必要はありません。フィルターフォージと呼ばれる美しく強力なツールを使用します。



レッスンの最初の部分では、既に用意されているフィルターをどのように使用するかを詳細に説明します。フィルターはダウンロードして自分の喜びに使用できます。 2番目の部分は、フィルターを作成するプロセスの説明に専念します。 したがって、フィルターの作成方法に興味がない場合は、2番目の部分を安全にスキップできます。



したがって、このフレームを作成します。









必要なもの：

  • この方法で処理したい写真または画像。 このチュートリアルでは、istockphoto.comの写真を使用しました。
  • Adobe Photoshop
  • FilterForge 2.0ベータプラグイン（このプログラムをまだお持ちでない場合は、こちらからダウンロードしてください： www.filterforge.com/download/beta2/index.htmlおよびコンピューターにインストールしてください）




ステップ1-イメージの準備



Photoshopで元の画像を開きます。 背景をレイヤーに変換し、「写真」という名前を付けます。







私の考えでは、ハート型のフレームを作成する予定なので、正方形の見た目は良くなります。 このレッスンでは、600x600ピクセルの画像を作成することにしました。 これを行うには、メニューの[画像]> [キャンバスサイズ]で目的の値を設定します。







ステップ2-選択を作成する



次に、フレームの形状を設定する必要があります。 カスタムシェイプツールの使用：カスタムシェイプ：ハートの形でベクトルオブジェクトを描画します。 この形状レイヤーを「写真」レイヤーの下に配置します。 次に、Altキーを押しながらこれらのレイヤー間をクリックします（クリッピングマスクの作成）。



これで、写真のどの部分がフレームに表示されるかがわかりました。 必要に応じて、フレームの形状を編集し、「写真」レイヤーを移動できます。







フレームの形状を決定しました。 「写真」レイヤーの複製を作成し、「写真」レイヤーと「写真コピー」レイヤーの間でAltキーを押しながらクリックします（クリッピングマスクを解除します）。



次に、「Photo copy」レイヤーを選択し、Ctrlキーを押しながらベクターレイヤーをクリックします（このShape 1があります）。







クイックマスク機能を使用するなど、選択を作成するのに便利な任意の方法を使用できます。 主なものは、結果として、目的のフレームの形で選択範囲があることです。 この心があります。



最後に、「写真コピー」レイヤーが選択されていることをもう一度確認してください。 このレイヤーは、Filter Forgeを使用して処理されます。



ステップ3.1-フィルターフォージ



Filter Forgeでフィルターを作成することに煩わされない場合は、これがレッスンの最終段階になります。 独自のフィルターを作成するスキルを習得したい場合は、ほとんど無限の可能性を開きます:)、ステップ3.2に進みます。



メニューの[フィルター]> [フィルターフォージ]> [フィルターフォージ2]でプラグインを起動します。GlowingHearts Frameフィルターは、 Glowing Hearts Frame.ffxmlからダウンロードして実行します。



既製のプリセットを使用できます（[プリセット]タブ）：







または、フィルター設定を自分で選択することもできます（[設定]タブ）：







目的の結果が得られたら、「適用」ボタンをクリックします。 まあ、それがすべてです。



次の手順では、Filter Forgeのフィルター製造プロセスについて説明します。



ステップ3.2-Filter Forge 2.0でフィルターを作成する



メニューフィルター>フィルターフォージ>フィルターフォージ2でプラグインを起動します...新しいフィルターを作成します：Ctrl + Nまたはプログラムウィンドウの右上隅の[フィルターエディター]ボタンをクリックし、[新しいフィルターの作成]を選択して[フィルターエディターを開く]ボタンをクリックします。







さて、すぐにフィルターに名前を付けます：Ctrl + S（またはメニューの[ファイル]> [名前を付けてフィルターを保存...]）。







ステップ4-ルミナスハートを作成する



画面の右側のリスト（ウィンドウの右上隅にある[コンポーネント]ボタンを押すとリストが開きます）から、Ellipseコンポーネント（パターングループ）をエディターの作業フィールドにドラッグします。 彼に次のパラメーターを設定しました。







色の選択の詳細：[色]パラメーターの横にある色付きの四角形をクリックします。 ピッカーウィンドウが開きます。 HDRパラメーターの横にチェックマークを付けて、明るさの値（HSBグループの3番目のパラメーター）を3に設定します。次に、ピッカーで目的の色を選択します。







この種のものを取得する必要があります。







今、私たちはこの円から心を作ります。 これを行うには、Profile Gradientコンポーネント（Gradientsグループ）とOffsetコンポーネント（Processingグループ）が必要です。



Profile Gradientコンポーネントに対して、Mirrorパラメーターを確認します。残りはすべて変更されません。







Ellipseコンポーネントの出力（緑色の三角形）は、OffsetのSourceコンポーネントの入力（緑色の小さな円）に接続されています。 同様に、Profile GradientをOffset V入力にフックし、OffsetコンポーネントをOffset Hパラメーターにゼロに設定します。

出来上がり！ これで心臓の写真ができました:)







ステップ5-ハートのクローンと再描画



フレームにさまざまな色の粒子を投げるので（この場合、これらはハートです）、ハートを作成したコンポーネントのグループのコピーをさらにいくつか作成する必要があります。 3つのコンポーネントすべてを選択します。 Ctrl + Cをクリップボードにコピーし、Ctrl + Vのコピーを貼り付けます（4回）。 合計で、私はそのようなグループの5つのコピーを得ました。







ハートを多色にするには、各楕円コンポーネントに異なる色パラメーターを設定します



[I ">ヒント：時々フィルターを保存することを忘れないでください（Ctrl + S）。[/ I">



ステップ6-爆撃機コンポーネント（HDR）



次に、Bomberコンポーネント（HDR）（パターングループ）が必要です。 爆撃機だけと混同しないでください。 そうしないと、美しい輝きが得られず、HDRカラーが使用されます。



次に、取得したハート（オフセットコンポーネントの出力）をパーティクル1 ... Bomberコンポーネント（HDR）のパーティクル5の入力に接続します。 （なぜ私たちが異なる色のちょうど5つのハートを作ったのか理解できましたか？）







爆撃機コンポーネント（HDR）を次のパラメーターに設定します（言及しなかったものは変更しないでください）。

サイズ：100;

不透明度カオス：100;

サイズカオス：100;

回転カオス：12;

Hカオス：100;

Vカオス：100;

スカッシュカオス：9;

色合いの量：20;

粗さ：50;

繰り返し：10;

密度：2;



ステップ7-選択を使用する



爆撃機を脇に置き、実際にフレームを取り上げましょう。 Photoshopで以前に設定された選択を使用するには、選択コンポーネント（外部グループ）が必要です。







ステップ8-フレーム幅を設定する



Blurコンポーネント（Processingグループ）を使用して、フレームの幅を定義します。 コンポーネントのリストから作業フィールドにドラッグします。 選択は、ぼかしコンポーネントのソース入力にフックされます。



ぼかし半径を使用して、フレームの幅を制御します。 これを行うには、スライダーコントロール（[コントロール]グループ）を半径入力に接続します。 スライダーのプロパティで、[再マッピングの使用]オプションを有効にします。 SliderコントロールとBlurコンポーネントを接続する線上に長方形が表示されます-これがリマッパーです。 マウスでそれを突き、最小値：1、最大値：40プロパティに設定しますこれらの値を設定することにより、可能なぼかし半径値の範囲を制限しました。 フレームの幅。



次のステップは、Blendコンポーネント（Processingグループ）を取得することです。 フォアグラウンドブラックに配置します。 背景の入力で、不透明度-選択でぼかしをフックします。







結果として生じるライトゾーンは、以前に作成された明るいハートで「爆撃」するゾーンを決定します。 この領域がフレーム内の画像に少し入るには、それを拡大する必要があります。



リスト（調整グループ）からしきい値コンポーネントをドラッグします。 彼の入力ソースにしがみついています

私たちのブレンド。 次のパラメーターのしきい値コンポーネントを設定します。しきい値：19、スムーズ：0。



次に、別のぼかしコンポーネント（半径：7）を使用してフレームにソフトエッジを作成し、レベルコンポーネント（調整グループ）でホワイトポイント：62パラメーターを使用してコントラストを高めます。







ステップ9-心臓うっ血の領域を特定する



Levelsコンポーネントの出力で得たものは、セクション6で説明したBomber（HDR）にChance 1、Chance 2、Chance 3、Chance 4の入力にフックします。したがって、粒子が最も集中するエリアを設定します。 （輝かしい心）。 粒子は、背景にそれほど密に散らばることはありません。 これを行うために、チャンス5の入り口を出ました。







次に、粒子のマスクを作成します。これは、背景の周りに散らばりますが、フレームには入りません。 これを行うには、リスト（調整グループ）から反転コンポーネントをドラッグします。 入力コンポーネントに選択コンポーネントからの出力を入力ソースに渡します（すでにエディターの作業フィールドにあります）。



フォアグラウンド入力でInvertを新しいBlendコンポーネント（Processingグループ）に接続します。 Opacityを6に設定します。そして、このすべての幸せをBomberコンポーネント（HDR）のChance 5入力に送信します。







次の写真が表示されました。







ステップ10-背景を作る



これで、画像をフレームに入れて美しい背景を配置することができます。

リストからPerlin Noiseコンポーネント（ノイズグループ）をドラッグアンドドロップします。 自由裁量でパラメーターを選択します。 私はこれらに決めました：粗さ：28; コントラスト：0; スケール：240; ストレッチ：45; 角度：138;















ステップ11-背景を元の画像と組み合わせる



これで背景は写真と互換性があり、選択を忘れることはありません。 リストからImageコンポーネント（外部グループ）とBlendコンポーネント（HDR）をドラッグします。 （Blend（HDR）であり、Blendだけではありません！さもなければ、美しい背景はそのすべての美しさを失います） また、作業フィールドでInvertコンポーネントを探します（選択範囲を反転します）。 そして、すべてを次のように接続します。







Blendコンポーネント（HDR）の出力は、Bomberコンポーネント（HDR）の「Background」入力に接続されます。



[フィルターの保存]ボタンをクリックします。



ステップ12-結果



「適用」ボタンをクリックします。 出来上がり！ 私たちはこのような不気味な美しさを得ました：







最後に、フィルターを少し改善して、さまざまな背景オプション、ケラレ、およびいくつかのコントロールを追加することにしました。 これらの手順を説明しないことにしましたが、詳細を詳しく読むことができます。 これを行うには、 Glowing Hearts Frame.ffxmlからフィルターをダウンロードして実行します。
