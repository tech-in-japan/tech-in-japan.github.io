企業に添加剤センターを作成する9つのステップ



2018年、UECの領土-モスクワ機械建設企業にちなんで命名 V.V. Chernyshevaは、添加剤技術のハイテクセンターの作成を開始しました/写真：rostec.ru







あなたの製造会社、研究室、設計局、または研究所は、すでにアディティブテクノロジーの可能性について一般的な考えを持ち、それらの実装の問題を検討していますか？ この記事では、企業にアディティブテクノロジーセンターを編成する際に考慮すべき要因、特定のタスクに基づいて選択するテクノロジー、スペシャリストを選択およびトレーニングする方法について説明します。







まず第一に：







1.アディティブテクノロジーが不可欠となる産業と企業の種類を定義します



3Dプリンティングの導入で最も有望な産業は、航空宇宙産業、原子力産業、石油およびガス産業、 機械工学 、エネルギー、造船、 自動車 、建築、医学などの新しいタイプの製品の最適化または作成を優先する産業です。 鋳造および金属加工業界では、3Dプリンターは、鋳造モデル、プロトタイプ、およびツーリングを作成する際の効果的なソリューションになります。







3次元のスキャンとモデリングは、品質管理とリバースエンジニアリングが実行されるあらゆる場所でアプリケーションを見つけます（セクション4を参照）。







3Dテクノロジーは、次の組織で需要があります。







  • 新しいタイプの製品を開発している企業の設計および実験局。
  • パイロット生産を行う技術センター。
  • 研究機関およびセンター;
  • 製造会社と直接働く大学。


2.自分の会社が本当に3Dテクノロジーを必要としているのか、どうすればわかりますか



はい。イノベーションを活用して利益をもたらすことは理にかなっています。 決定を下す前に、以下を考慮することが重要です。







  1. すべての生産上の問題を最大の効率で解決できる普遍的な[付加技術]（https://blog.iqb-tech.ru/3d-printing-technologies）はありません。
  2. 各3Dテクノロジーには長所と短所があります。
  3. 3Dプリンターや3Dスキャナーを選択するときは、タスクから続行する必要があります。


今日の3Dプリントは、従来の方法に取って代わるものではありません。 アディティブマニュファクチャリングは、既存の生産プロセスに統合され、生産サイクルを短縮し、結果として時間とコストを大幅に節約するために、従来の技術では利用できないことが多いソリューションを提供します。







考慮すべき主なポイント：3D印刷は、パイロット、実験、または小規模生産にのみ適しています。 次の目標が設定されている場合、会社はそれを必要とします。







  • 独自の特性を備えた新製品の開発と生産。
  • 研究開発
  • 技術ソリューションのプロトタイピング、テスト、テスト。
  • 製品の最適化/近代化は、付加的な技術（複雑な形状、[重量削減]（https://blog.iqb-tech.ru/slm-asco-case）、薄壁、内部チャネルなど）によってのみ可能です。


別の制限として、3Dプリンターを構築するためのカメラのサイズがあります（これは、技術とメーカーによって異なります）。







従来の方法で目的の性能が得られる場合、新製品の開発、設計の最適化、製品特性の改善が必要でない場合、添加剤センターを組織することは実用的ではありません。









ハイブリッド添加剤技術のセンターは、選択的レーザー核融合技術が使用されるサマラ大学に基づいて作成され、その後、機械的、電気化学的および熱処理が行われます/写真：sgpress.ru

4. 3Dスキャンも必要になります



受信したデータを処理するための3Dスキャナーとソフトウェアは、現代の企業に必要なツールです。 3Dスキャンは、製品ライフサイクル管理のどの段階でも使用できるため、開発段階での時間とコストを削減し、市場での製品の発売を加速できます。









チフビン自動車製造工場のモデルワークショップのマーカーDaniil Zhidkovは、Creaform HandySCAN 3D 3Dスキャナーを使用して鋳造の3次元モデルを作成します。 このポータブル3Dスキャナーを使用すると、サイズや図面が手元にないときにパーツのモデルを作成できます。これにより、デザイナーの作業が大幅に容易になります/写真：tvsz.ru







3Dスキャナーと専用ソフトウェアを使用すると、次のタスクをより効率的に解決できます。







  • 製品および機器の形状制御、入出力制御。
  • 部品の近代化、修理、修復のためのリバースエンジニアリング。
  • CADモデルの取得。


3Dスキャンデバイスは、1メートルあたり20〜50ミクロンの精度を提供します。 これらの精度パラメーターが適切でない場合は、座標測定機を引き続き使用してください。 ただし、測定速度、携帯性、およびコストの観点から、3次元スキャナーではCMMがはるかに遅れています。







5.予算を計画します



追加のインストール価格は、3Dスキャナーの場合、1.5ユーロ（FDM）から数百万ユーロ（SLM）の範囲で、20から13万ユーロです。







プリンターの最終的なコストは、選択したハードウェア構成やその他の多くの要因に依存するため、添加剤センターの整理の初期段階で数字を挙げるのは困難です。 さらに、関連するコスト（ 3D印刷の材料 、追加の機器、エネルギー消費など）を考慮する必要があります。







6.機器を購入する前に3Dテクノロジーがどのように機能するかを確認する



問題の解決策を事前にテストしたいですか？ iQB Technologies ラピッドプロトタイピングセンターでは、ワックス、フォトポリマー、石膏を使用した3D印刷、3Dスキャン、3Dモデリングのテストサービスを注文できます。 ポータブル3Dスキャナーを備えた専門家がロシア全土の企業に旅行することは可能です。







7.追加の機器



添加剤の設置の種類によっては、追加の機器が必要になる場合があります。 たとえば、SLAプリンターの場合、完成品が硬化するUVカメラが必要です。 最も洗練された機器は、 金属3Dプリンターです。







  1. 残留金属応力を緩和するマッフル炉;
  2. 仮眠、ベルトソー、EDMによるサポートの取り外し。
  3. 製品の後処理と表面品質の向上のためのビーズブラストまたはサンドブラストチャンバー。


3Dプリンターを選択した後、当社の専門家とともに、必要な追加機器を見つけることができます。







8. 3Dプリンターの操作に関する特別な要件



前提条件と動作条件には、たとえば、適切な量の電気の供給や空調などの特別な要件があります。 最も厳しい要件は、 金属を使用した3D印刷システムに適用されます-不活性ガス環境で焼結する金属粉末の危険性があるため、安全対策を遵守する必要があります（機械の動作中の気密性を確保し、防護服で作業する）。







金属3Dプリンターの標準要件の例を次に示します。







  • 3相の電圧380 V、動作モード7 kW、ピーク時最大12 kW;
  • 機械への不活性ガスの供給（3-5シリンダー）;
  • 部屋の空調システム（18から23°Cまで）、最適-空気乾燥モードで;
  • 圧縮空気または4気圧のコンプレッサーまたはライン;
  • フィルターおよび製品を洗浄するためのプロセス水。




United Engine Corporationの一部であるODK-Saturn Rybinskの企業は、ガスタービンエンジンの生産に付加技術を積極的に使用しています/写真：rostec.ru







9.添加剤技術センターの専門家を養成します



添加剤研究所の従業員は3Dプリンターの設計者およびオペレーターであり、その人数は機器の構成と作業量に依存します。 プロの3Dプリンティングシステムに1人のオペレーターで十分な場合、産業（SLA、SLS、SLM）には少なくとも2人の訓練を受けた専門家が必要になります。







演算子とコンストラクターが異なる場合に最適です。 設計者は、特殊なソフトウェアで積層造形用の製品を設計し、適切な方法を適用する方法を理解します。オペレーターは技術者に近いため、最終製品の設計者のビジョンを体現しています。 同時に、3Dプリンターのオペレーターは、創造的な人であり、高等技術教育を受けた専門家であり、勉強を愛し、イノベーションに情熱を持っている必要があります。 プリンター設定を正しく選択することにより、高い印刷品質を実現できます。 これは非常に困難な作業になる可能性があります。たとえば、SLMマシンでは、構築プロセス中に変更できる170個のオープンパラメータがあります。







当社のトレーニングセンターでは、3〜4日で産業用3Dプリンターで作業するようにトレーニングできます。 3Dスキャンのトレーニングプログラムと、スキャンデータの処理（Geomagic Control X / Design X / Wrap ）および印刷用のモデルの準備（ Materialize Magics ）の基本的なソフトウェア製品も利用できます。 また、最大12日間続く包括的な包括的なプログラムもあります。







おわりに



この記事が、3Dテクノロジーの選択を決定し、追加の実験室を作成するためのさらなるステップを踏むのに役立つことを期待しています。







3Dプリンティングは、主にハイテク産業でのパイロット、実験、または小規模生産にのみ適していることがわかりました。 付加技術は、新しいタイプの製品を開発したり、現在の品揃えを改善したり、プロトタイプや機器を作成したりするときに、生産プロセスを最適化します。 3Dスキャンにより、品質管理とリバースエンジニアリングタスクを実行する際の開発段階で時間とコストを削減できます。
