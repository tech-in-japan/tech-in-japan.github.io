DropboxでのSamsungからの50ギガの景品

Dropboxは、すべてのSamsung Galaxy S III携帯電話に2年間50GBを割り当てます。 しかし、彼らはおそらくそれについて知っていて、長い間、正直にすべてを手に入れましたが、他のブランドよりもモバイルデバイスの所有者すべてについて話す価値はあります。 おそらくこれはすでにハブに書き込まれていましたが、念のため、「抜け穴をふさぐまでもう一度」は見つけませんでした。 xda-developers.comの職人がパッチをリリースし、これを使用して、あらゆるブランドの電話をSamsungのふりをすることができます。 CWMまたはルートエクスプローラーを介してインストールされます。 つまり いずれにせよ、スマートまたはCWMスマートの下でフラッシュのいずれかが必要になります。



最も単純な場合：

1.データが削除されたアプリケーション（インストールされている場合）からDropboxを削除します。

2. CWM-Samsung-DropBox.zipファイルをダウンロードして、携帯電話のSDカードに書き込みます。

3.リカバリのために再起動し、SDカードから.zipインストールを選択し、CWM-Samsung-DropBox.zipまでスクロールして、インストール、再起動します。

4.インストールされたアプリケーションに表示されるアイコンからDropboxに入り、何かを同期することに同意し（これは重要です）、プロセスが開始するのを待ちます。 その後、中断することができます。

5.電話でブラウザを開き、 www.dropbox.com / gsで Dropboxにアクセスするか、ログイン後にモバイル版以外のページで[開始]リンクをクリックして、すべてのタスクを完了します（6番目のオプション）



実行後、50ギガバイトのボーナスが有効になります



CWMがない場合は、CWM-Samsung-DropBox.zipをSDカードに解凍し、R / W権限を持つルートエクスプローラーを介して、Dropbox.apkおよびDropboxOOBE.apkを/ system / appおよびcom.dropboxpartner.jarフォルダーを/ system / frameworkフォルダーにコピーします

その後、再起動して携帯電話から手順4〜5を繰り返す必要もあります。



UPD：彼らは電話さえ持っている必要さえないと言います、そして、すべてのこれらの操作はWindowsまたはLinuxの下でエミュレーターからされることができます。
