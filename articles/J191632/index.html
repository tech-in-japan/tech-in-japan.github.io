マイクロソフトはMS13-057およびMS13-061を再リリースします

同社は、 MS13-061 （ v。3 ）およびMS13-057 （ v。3 ）の更新プログラムの新しいバージョンを公開しました。 MS13-061（クロスExchange Server 2007-2010-2013 RCE修正）は、Exchange Server 2013の障害およびサーバー上の文字のインデックス作成の停止に関する問題のため、8月のパッチ火曜日以降にリコールされたことを既に書いています。 このバージョンの更新プログラムは、以前のリリースで発生したエラーを修正します。 MS13-061を既にインストールしているユーザーは、最新の修正プログラムで更新するように求められます。







7月のパッチ火曜日のレビューで MS13-057について書きました 。 この更新プログラムは、Windows Media Formatランタイムライブラリコンポーネント（wmvdecod.dll）のクロスプラットフォームリモートコード実行の脆弱性CVE-2013-3127（XP-7-RT）を修正します。 攻撃者は、特別に細工されたメディアファイルを介してこの脆弱性を利用し、コードをリモートで実行できます。 この更新プログラムの最新バージョンでは、以前のバージョンを使用したときに表示された一部のWMVビデオの誤った再生の問題が修正されています。
