firefoxで複数のファイルをダウンロード（multiupload）

multiaploadのフラッシュプラグインとセックスしたことはありますか？ 特にnginxと組み合わせて：「エラー2038-ああ、はい、赤ちゃん、もっと！」。

タイムラインから判断すると、 W3Cは最終的にそれを把握し、2022年頃までにhtml5の推奨を開始する予定です。 そして今日、私はこのサイトに苦しんでいます。

しかし幸いなことに、かなり前に、オペラはWebフォーム2.0のコンセプトを提案し、同じ名前のブラウザーに実装しました。

: input type="file"

: input type="file" min="1" max="9999″ multiple , ..







この機能は、Chromium 3.0.191.0、Safari 4.0.1、およびそれに応じてOperaでサポートされていました。 個人的には、ユーザーの前で良心を払うためにFFだけが必要です。 バグは 2000年以来トラッカーに残っています。もしあなたがmultiaploadとの戦いにうんざりしているなら、簡単な登録をして、投票ボタンをクリックすることを提案します。

恩知らずの子孫はこの問題さえ覚えていないだろうと確信していますが、それは必ずしもそれほど単純ではなかったことがわかります。
