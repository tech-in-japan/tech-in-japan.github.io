No. 18列の追加

質問


この例では、一部の数字が文字に置き換えられています。

これらの数値が何であるかを見つけ、ABCDEの形式で答えを書きます（例：12345）。

決定方法
もちろん、コンピューターでブルートフォースで解決できますが、紙でそれを行うこともできます。追加された数字には8桁があり、合計で9があり、これはキャリーオーバーがあったことを意味するため、合計の最高位のDは1つだけになります。 D = 1



次の放電-ユニットの放電：B + B = 9またはB + B = 19のいずれかですが、9は奇数であるため、転送ユニットは前の放電から来ました。 つまり 2B = 8、次にB = 4、または2B = 18、次にB = 9のいずれかですが、9はすでにオープンで検出されています。つまり、 B = 4のみです。



数十の放電を見ます： E + C = 11であることが知られています。

最後の（右端の）数字9 + 4 = Aを調べます。これは、 A = 3を意味し、転送単位がありました。



小数点の後の4桁目を見てください。前の桁の単位は転送されませんでした（4 + 4 + 1 = 9）および1 + E =C。 E + C = 11、つまり C = 11-E、つまり1 + E = 11-E、2E = 11-1を転送すると、 E = 5 、 C = 6になります。

答え
34615

PS一部の人はあまりにも夢中になったので、それに応じてABCDE値の代わりに合計金額を書き留めました。例外として正しいとカウントされました。
