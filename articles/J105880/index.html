Ubuntu 10.10用Galaxy Live Wallpaper

Ilapというニックネームを持つ男は、Androidのライブ壁紙に感銘を受け、この壁紙を模倣したcompizプラグインを作成しました。 彼によると、それ以前はopenglまたはcompiz用のプラグインの開発経験がなかったことは注目に値します。











この壁紙を自分用にインストールするには、開発者のppaリポジトリを追加する必要があります。



$sudo add-apt-repository ppa:ilap/lwp

$sudo apt-get update







次に、パッケージ自体をインストールします。



$ sudo apt-get install compiz-plugins-livewallpapers compizconfig-settings-manager







次に、壁紙変更メニューを呼び出します-デスクトップを右クリックし、[追加]をクリックして、/ usr / share / backgrounds / Space_by_ilap.jpgを選択します。



プラグインを有効にする必要があります：システム->設定-> CompizConfig設定マネージャー->ライブ壁紙とGalaxy Liveを確認します。



プラグインを有効にするには、キーボードショートカットwin + f4を押します。
