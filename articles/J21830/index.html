ITクイズ。 最強が勝ちます！

3月17日に、ITプロフェッショナルと開発者向けの2回目の全ロシアのコンテスト、 IT Quiz v2.0が始まりました 。 コンテストはItcommunity.ruおよびLinuxcenter.ruによって開催されます。



ITクイズ2.0の競争は拡大しており、新しい質問が含まれているため、開発者は知識をテストし、プロ意識を示すことができます。 昨年の最初のコンテストと同様に、2つのテストで構成されています。 最初に合格すると、ユーザーは面白い個人証明書を受け取ります。 2番目のテストは、プロ意識に関する難しい質問です。 2番目のテストには、5ブロックの質問が含まれます。 彼らは、オープンソースソフトウェア、Windows Server OS、最新の情報セキュリティ技術、Visual Studio、PHPに専念しています。



結果は正解の数によって決まり、賞品の4つのカテゴリの1つに対応します。 各カテゴリで、好きな賞を個別に選択できます。 コンテストの主な賞品はATVであり、コンテストの勝者に対するその他の貴重な賞品であるAsusラップトップ、HTC Touchコミュニケーター、X-boxゲームコンソールです。



「ITクイズの組織に参加しました。ITコミュニティを統合し、ロシアのIT専門家の一般的な知識レベルを特定するイニシアチブをサポートすることが重要だと考えているからです」と、GNU / Linux CenterのCEOであるPavel Frolov氏は述べています。



情報サポートは、IT AcademyとMicrosoftが後援するHabraHabr、bash.org.ru、およびiXBT.comによって提供されました。 2008年5月18日に終了するこのコンテストには、18歳に達したロシア連邦のすべての市民が参加できます。
