Netbeans IDE 8.1-RCそれでも

PHP 、 Java 、 C / C ++などのプログラミング言語向けの素晴らしい、最も重要な無料のIDEの新しいバージョンの安定したリリースが出ようとしています。 それまでは、このバージョンのリリース候補 （RC）をバイパスして、このIDEの変更点を見てみましょう。



開発者自身が直接指摘しているバージョン8.0と比較した主な変更点（フレームワークなどのサポートの変更は考慮されていません） 。

  • Node.jsで直接作業します。

    私にとって、WEB開発者として、これは重要な機能であり、 GulpやGruntの通常のサポートなど、他の多くの優れた改善点を引き出しています。
  • Oracle WebLogic Serverのリモート管理サポート
  • WildFly 9および10アプリケーションサーバーのサポート


Node.js


  • Node.jsプロジェクトのサポート

  • プロジェクト設定からnpmパッケージを直接管理する

  • Node.jsアプリケーションの起動とデバッグ





HTML5 / JavaScript


  • Gulpサポート







  • MochaおよびSeleniumのサポート
  • Jade Template Engineのサポート
  • Gruntの拡張サポート
  • AngularJSおよびKnockoutJSの拡張サポート
  • プロジェクトはpackage.json 、 bower.json 、 composer.jsonを認識するようになりました





Java


  • コードの自動補完の改善
  • より便利で直感的なナビゲーター



  • 設定ウィンドウの改善された制御


Java EE


  • Oracle WebLogic Serverのリモート管理サポート



  • WildFly 9および10アプリケーションサーバーのサポート


Php


  • Phingサポート
  • Composerとの高度な統合
  • Nette Testerによるコードカバレッジ
  • Symfonyインストーラーとの統合





C / C ++


  • C ++ / Javaでの混合開発のサポート
  • 「プロジェクト作成ウィザード」の改善
  • SVN 、 GitおよびMercurialがリモートでサポートされています



  • C ++エディターの改善
  • チェック、ヒント、リファクタリングの改善
  • 「リバースエンジニアリング」のサポート（コールグラフ）


プロファイラー


  • 再設計



  • 新機能：
    • CPUステータスのリアルタイム表示
    • メモリステータスのリアルタイム表示
    • CPU使用率の監視
    • プロファイルされたアプリケーションのスレッドダンプ
  • エンジンの改善：
    • プロファイルされたアプリケーションへの高速接続
    • プロファイルされたメソッドへの発信呼び出しを制限する
    • 選択したクラスのメモリアカウンティング
  • 簡素化された設定
  • IDE統合の改善


その他


  • 拡張Gitサポート
