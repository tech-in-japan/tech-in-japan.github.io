Skypeは再びウイルスを拾いました

いくつかのアンチウイルス企業によると、Skype Inc.の広報担当者であるVillu Arakによる公式声明によると、昨日9月10日、ユーザーは再び未知の攻撃者によって組織されたウイルス攻撃に直面しました。



ウイルスは、最も単純な（そして、悲しいかな、最も効果的な）方法の1つで広がります。Skypeユーザーは、テキストチャットで、jpegイメージへのリンクとできるだけ早くそれを見るための呼び出しを含むメッセージを受け取ります。 ユーザーがリンクをクリックするとすぐに、悪意のあるプログラムがPC（Worm.Win32.Skipi.a）に入り、ウイルス対策ソフトウェアの動作をブロックし、Skypeをサイレントモードに切り替え、連絡先リストへのリンクを含むメッセージの送信を開始します。 したがって、ユーザーは友人からでも悪意のあるメッセージを受け取ることができます。 ワームはシステムに深刻な害を及ぼさないことに注意することが重要です。



これらの企業の代表者によると、F-Secure、Symantec、Kaspersky Labによって製造されたアンチウイルスは、すでにこのウイルスを検出して中和することができます。 ただし、Skype Inc. ワームを駆除するための手順を記載した公式文書をリードしています。



したがって、問題の解決策は次の手順で構成されます。

-PCをセーフモードで再起動します

-レジストリエディタ（regedit）を起動します

-HKLM /ソフトウェア/ Microsoft / Windows / currentversion / runonceブランチを見つける

-mshtmldat32.exeを含むエントリを見つけ、このエントリを削除します

-Windows \ System32ディレクトリに移動し、ファイルwndrivs32.exe、mshtmldat32.exe、winlgcvers.exe、およびsdrivew32.exeを削除します

-Windows \ System32 \ drivers \ etcディレクトリに移動します

-ホストファイルを見つけ、メモ帳で開いてすべての内容を削除し、ファイルを保存して閉じます。

-PCを再起動します。
