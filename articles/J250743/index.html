ウィークリービルドVivaldi 1.0.105.7

みなさんこんにちは！



今日、Vivaldiブラウザの次の週刊ビルドは、番号1.0.105.7でリリースされました。 修正されたバグとブラウザのパフォーマンスの漸進的な改善に加えて、このアセンブリには待望の機能がいくつか登場しました。



まず、テストモードでは、ブックマークバーが追加され、パネル設定で既存のフォルダーから適切なフォルダーを指定するか、独自のフォルダーを作成してリンクを配置できます。 ブラウザウィンドウの下部に移動して、このパネルの場所を変更することもできます。 当然、このパネルは必要ない場合は完全に無効にできます:)



また、このアセンブリはLinux用の32ビットバージョンでも提供されます。



一般的な需要により、バックグラウンドでExpressパネルから中央のマウスボタン（またはホイール）をクリックしてブックマークまたはWebサイトを開く機能を追加しました。



キーボードを使用してブラウザを制御したい人のための別の良いニュース：キーボードコマンド（CTRL + ZおよびCTRL + SHIFT + T）を使用して最近閉じたタブを復元する機能を追加しました。



そして最後に、ブラウザインターフェースが翻訳された言語のリストが大幅に拡大しました。 現在、ローカリゼーションのリストには22の言語があり、以前に追加された言語に加えて：



  • ギリシャ語
  • スペイン語
  • エストニア語
  • イタリア人
  • ラトビア
  • オランダ人
  • ポーランド語
  • ポルトガル語（ブラジルおよびポルトガル語のオプション）
  • スロバキア
  • セルビア語
  • ウクライナ語
  • 中国語（簡体字）




Vivaldiブラウザーユーザーの中からボランティアの翻訳者の積極的な支援のおかげで、ローカライズのリストを拡張することができました（そして、以前に利用可能なすべての翻訳の品質を改善しました）。 現在、Vivaldi Translationチームには、28か国から60人以上がブラウザインターフェースを29の言語に翻訳しています（残念ながら、そのうちのいくつかはChromiumでサポートされていませんが、すぐにブラウザの言語サポートを拡張します）。



Vivaldiブラウザのこのビルドをここからダウンロードしてください：

  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
