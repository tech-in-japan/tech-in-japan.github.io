文字列ノード内の行をスペースで分割し、奇数の各要素を緑で、偶数の各要素を赤で表示する必要があります。

XSLT / XPATH 2.0には、文字列を部分に分割できるすばらしいトークナイズ機能があり、それに応じてxsl：for-eachを使用して、各部分で必要な処理を実行できます。 しかし、私が知る限り、XSLT 2.0はSaxonプロセッサーでのみ十分にサポートされています。 組み込みのXSLTブラウザプロセッサとPHPのlibxsltにはないため、再帰的なテンプレートを使用します。

  2. < xsl：stylesheet version = "1.0" xmlns：xsl = "http://www.w3.org/1999/XSL/Transform" >
  4. <！-一致するだけで文字列、
  5. 次に、ノードのコンテンツがテキストに変換されます。
  6. ただし、文字列内にまだノードがある場合（たとえば、<br />）、
  7. エラーが発生する可能性があります
  8. テキストを使用する場合、
  9. 次に、テキストを一致させる必要があります
  10. コードをわかりやすくするために->
  11. < xsl：template match = "string / text（）" >
  12. < xsl：call-template name = "colorer" >
  13. < xsl：with-param name = "text" select = "。" />
  14. <！-他の値はデフォルト値から取得されます->
  15. </ xsl：call-template >
  16. </ xsl：テンプレート >
  18. < xsl：テンプレート 名 = "colorer" >
  19. < xsl：param name = "text" />
  20. <！-デフォルトの区切り文字-スペース->
  21. < xsl：param name = "delimeter" select = "''" />
  22. <！-デフォルトでは、要素の色は奇数です->
  23. < xsl：param name = "even" select = "false" />
  24. < xsl：変数 名 = "色" >
  25. < xsl：選択 >
  26. < xsl： test = "$ even"の場合 >
  27. < xsl：テキスト >赤</ xsl：テキスト >
  28. </ xsl：when >
  29. < xsl：その他 >
  30. < xsl：テキスト >緑</ xsl：テキスト >
  31. </ xsl：それ以外の場合 >
  32. </ xsl： >を 選択します
  33. </ xsl：変数 >
  34. < xsl：選択 >
  35. <！-文字列に区切り文字が含まれる場合->
  36. < xsl：when test = "contains（$ text、$ delimeter）" >
  37. <！-次に、セパレータに行を出力します->
  38. < span class = "{$ color}" > < xsl：value-of select = "substring-before（$ text、$ delimeter）" /> </ スパン >

  40. < xsl：call-template name = "colorer" >
  41. < xsl：with-param name = "delimeter" select = "$ delimeter" />
  42. < xsl：with-param name = "even" select = "not（$ even）" />
  43. < xsl：with-param name = "text" select = "substring-after（$ text、$ delimeter）" />
  44. </ xsl：call-template >
  45. </ xsl：when >
  46. < xsl：その他 >
  47. < span class = "{$ color}" > < xsl：value-of select = "$ text" /> </ span >

ページ番号（ポケットベル）を表示する必要があります。 ページ番号の前後に、最初または最後のページにない場合は、前のページと次のページへの遷移を表示する必要があります。 また、ページャーが非常に大きくなることがあるため、一度にすべてではなく、次の数ページに移動するためにリンクのみを表示する必要があるという条件を設定します。

このようなかなり大きなテンプレートを取得します。





  1. <？ xml バージョン = "1.0" ？ >
  2. < xsl：stylesheet version = "1.0" xmlns：xsl = "http://www.w3.org/1999/XSL/Transform" >
  4. < xsl：template match = "pages" >
  5. < xsl：call-template name = "page-numbers" >
  6. < xsl：with-param name = "total-results" select = "topics / @ count" />
  7. < xsl：with-param name = "pages-per-page" select = "topics-per-page / @ count" />
  8. < xsl：with-param name = "max-from-current-page" select = "3" />
  9. < xsl：with-param name = "current-page" select = "current-page / @ number" />
  10. < xsl：with-param name = "href" select = "link / @ href" />
  11. </ xsl：call-template >
  12. </ xsl：テンプレート >
  14. < xsl：テンプレート 名 = "ページ番号" >
  15. < xsl：param name = "total-results" />
  16. < xsl：param name = "results-per-page" />
  17. < xsl：param name = "max-from-current-page" />
  18. < xsl：param name = "current-page" />
  19. < xsl：param name = "href" />
  22. <！-総ページ数->
  23. < xsl：変数 名 = "max-page" select = "ceiling（$ total-results div $ $ results-per-per）" />
  25. <！-複数のページがある場合、ページ番号を印刷します->
  26. < xsl： test = "1 <$ max-page"の場合 >
  28. <！-どのページからページ番号の表示を開始するか->
  29. < xsl：変数 名 = "from-page" >
  30. < xsl：選択 >
  31. <！-現在のページ番号が最大距離よりも大きい場合->
  32. < xsl： test = "$ current-page> $ max-from-current-page"の場合 >
  33. <！-現在のページから指定されたページ数だけ削除された最初のページ->
  34. < xsl：select-of select = "$ current-page-$ max-from-current-page" />
  35. </ xsl：when >
  36. < xsl：その他 > 1 </ xsl：その他 >
  37. </ xsl： >を 選択します
  38. </ xsl：変数 >
  40. <！-ページ番号の出力を終了するページ->
  41. < xsl：変数 名 = "to-page" >
  42. < xsl：選択 >
  43. <！-現在のページ番号が最後のページ番号よりも最大距離よりも大きい場合->
  44. < xsl： test = "$ max-page-$ current-page> $ max-from-current-page"の場合 >
  45. <！-これは、現在のページから指定したページ数だけ削除された最後のページになります->
  46. < xsl：select-of select = "$ current-page + $ max-from-current-page" />
  47. </ xsl：when >
  48. < xsl：その他 >
  49. < xsl：値の 選択 = "$ max-page" />
  50. </ xsl：それ以外の場合 >
  51. </ xsl： >を 選択します
  52. </ xsl：変数 >
  54. <！-現在のページが最初ではない場合、前のページへのリンクを含む矢印が表示されます->
  55. < xsl： test = "1！= $ current-page" >
  56. <a href = "{$ href} {$ current-page-1}"> <<< / a >
  57. < xsl：テキスト > </ xsl：テキスト >
  58. </ xsl：if >

  61. < xsl：call-template name = "page-number" >
  62. < xsl：with-param name = "max-page-number" select = "$ to-page" />
  63. < xsl：with-param name = "current-number" select = "$ from-page" />
  64. < xsl：with-param name = "current-page" select = "$ current-page" />
  65. < xsl：with-param name = "href" select = "$ href" />
  66. </ xsl：call-template >
  68. <！-現在のページが最後ではない場合、次のページへのリンクを含む矢印が表示されます->
  69. < xsl：if test = "$ max-page！= $ current-page" >
  70. < xsl：テキスト > </ xsl：テキスト >

  72. </ xsl：if >
  73. </ xsl：if >

  86. </ xsl：when >
