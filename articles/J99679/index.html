HP Virtual Serverショーケース

構築し、構築し、最終的に構築しました！



hp.ru/vitrinaでは、エントリーレベルのHP ProLiant標準アーキテクチャサーバーモデルを販売するオンラインストアフロントが開始されました。 ショーケースでサーバーを注文するには、3つの簡単な手順に従うだけです。サーバーモデル、その構成、および最寄りのHPパートナーを選択します。



その後、注文情報がHPパートナーに電子メールで送信されます。HPパートナーは1営業日以内に連絡を取り、サーバー配信プロセスを開始する必要があります。 オンラインストアフロントのWebサイトには、VATを含むエンドユーザーのルーブル価格が表示されます。 Showcaseの販売に参加しているHPパートナーは、この価格を超えないコストでサーバーを提供します。







初期段階では、HP ProLiant 100シリーズサーバーの4行がインターネットショーケースに表示されます。シングルプロセッサタワーML110G6、デュアルプロセッサタワーML150 G6、および2つのラックサーバ：シングルプロセッサDL120 G6およびデュアルプロセッサDL160 G6です。



ご存知のように、100番めのHP ProLiantシリーズのエントリーレベルサーバーは、小規模ビジネス向けに特別に設計されており、ファイルサーバーとプリントサーバーの役割は、通常、通常のデスクトップPCまたは自己組織化サーバーによって実行されます。 そのため、 オンラインストアフロントを訪れた多くの訪問者は、最初にサーバーを注文し、彼らを支援するために、購入プロセスを可能な限り簡素化しようとしました。 顧客は、多数のHPパートナーの間で有利なオファーを探す必要なく、構成、適切なモデルの価格、および倉庫でのその可用性をすぐに確認できます。



ユースケースの簡単な説明は各モデルのインターネットストアフロントに掲載されており、提案されている構成には、ほとんどのモデルに必要なすべてのサーバーコンポーネントが含まれています。 さらに、Microsoft Windows Server 2008 R2またはMicrosoft Windows Server 2008 R2 Foundationオペレーティングシステムを搭載したサーバーをすべて注文して、最初の90日間は無料のテクニカルサポートを受けることができます。



将来的には、オンラインストアフロントの範囲が拡大すると思います。 そして、誰もがこのような質問への答えを簡単に見つけることができます、 this 、 this 、 this 、 この ようなコメント :)
