Web 2.0の定義の検索について。 反対側から見て

Siddtharthaは、 近くのhabratopicでのWeb 2.0の包括的な定義を求めています。 この定義は、この検索全体の扇動者であるO`Reillyを含め、非常に多くの人が求めています。



Web 2.0の定義は、イデオロギーと技術の2つのコンポーネントに分けることができます。



イデオロギーについて。 無料のサービス、普遍的な愛、友情、平等に対するWeb 2.0のイデオロギーのすべての呼びかけにもかかわらず、このイデオロギーは、主にこれらすべてでお金を稼ぎたい人々によって調査され、策定されています。



Web 2.0のモデルを構築するこれらの人々は、インターネットマーケティング2.0のモデルをすぐに構築します。 Webは変化しています-インターネットマーケティングは変化しています。 つまり、Web 2.0の定義の検索は、基本的にインターネットマーケティングの傾向の検索であることがわかりました。
