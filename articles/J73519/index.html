Windows Mobile開発者コンテスト-2009

MicrosoftとSoftline Venture Partnersは、Windows Mobileプラットフォーム用のWindows Mobile Developers 2009アプリケーションコンテストを実施しています。 10月12日に始まり、12月に結果を発表します。 ウェブサイトwww.wmdev.ruですべての詳細を確認し、応募をコンテストに送信できます。 独立した開発者と企業全体の両方が競争に参加できます。



4つのカテゴリで開発者向けのコンテストを開催しています。



• 審査員の選択

•コミュニティの選択

•Windows Mobile 6.5に最適なアプリケーションまたはウィジェット

•MTSのモバイルデバイスから音楽とビデオへの最適なアクセス





単純な開発者と企業全体の両方が競争に参加できます。 賞品は誰にとっても役立つと確信しています。 Jury Choiceノミネートの受賞者には、モダンでスタイリッシュで高性能のラップトップ 、 Community Choiceの受賞者、Windows Phone コミュニケーターの最新モデル、 Xbox 360エリートの受賞者「 Windows Mobile 6.5のベストアプリケーションまたはウィジェット 」、および最後に、ノミネート「 モバイルデバイスから音楽とビデオへのベストアクセス 」を獲得した開発者-Acer S200コミュニケーター-Windows Phoneベースの最新モデル。



マイクロソフト、ソフトラインベンチャーパートナー、YandexおよびMTSの代表者も審査員であり、Windows Mobileのソフトウェア開発者市場（SPB SoftwareおよびParagon Software Group）のトップ企業に競争で最高のアプリケーションを選択するよう呼びかけました。 審査員は、いくつかの基準に従ってアプリケーションを評価し、総合評価を使用して、「審査員選択」ノミネートの勝者を決定します。 サイトへの訪問者は、競技者の仕事を評価することもできます。 彼らの総合評価は、Community Choice候補の勝者を決定するために使用されます。



受賞者は2009年12月に、ウェブサイトwww.wmdev.ru 、ブログwww.ms4press.ru 、サイトSoftline Venture Partners www.softlinevp.com 、コンテストの公式twitter www.twitter.com/wmdev09で発表されます。
