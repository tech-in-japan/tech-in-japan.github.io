SelectelおよびParallelsパートナー





SelectelがParallelsの公式パートナーになったことをお知らせいたします！ 現在、Parallelsソフトウェアライセンスの全範囲をレンタル用に提供しています。最も人気のあるのは、もちろんPlesk Panelホスティングコントロールパネルです。



Selectelの企業ブログのオリジナル 。



現在、サービスコントロールパネルから注文するための最も一般的な2つのオプションを利用できます。

  • 800ルーブル/月の100ドメインの「パラレルPleskパネル」。
  • 「パラレルPleskパネル」1300ルーブル/月の無制限のドメイン。


ただし、チケットシステムを介したテクニカルサポートのリクエストに応じて、Parallelsから他のライセンスを注文することができます。 最も人気のあるソフトウェアは、コントロールパネルから注文するために追加されます。



「Parallels Plesk Panel」は、ウェブホスティング用の実際の「スイスナイフ」です。これには、サイト管理、メール、データベース、DNS、ユーザーおよび再販業者管理、料金プランとテンプレート、ウェブサイトビルダーが含まれます。



主な機能をリストします。

  • すべてのコントロールパネルの中でオペレーティングシステムを最も幅広くサポート：Windows Server 2008および2008 R2、CentOS、Debian、Red Hat Enterprise Linux、Ubuntu、CloudLinux、およびOpenSuse。
  • インターネットインフォメーションサービスやMicrosoft WebMatrixプロジェクトをParallels Panelサーバーに公開する機能など、Windows Serverの包括的なサポート。
  • Apache Tomcat、Adobe ColdFusion、Ruby on Rails、Microsoft .NET Framework、DotNetNukeを含む開発環境の幅広いサポート。
  • 直感的なWeb Presence Builderは、サイトをすばやく作成するための最もシンプルなインターフェイスを提供します。 さらに、さまざまなタイプの中小企業や個人的な興味のために特別に作成された100以上の既製のテンプレートから選択できます。
  • Parallels Virtuozzo Containers、Parallels Server Bare Metal、VMware、Xen、KVM、Microsoft Hyper-Vなど、多くの仮想化テクノロジーをサポートしています。


先日、「Parallels Plesk Panel」がバージョン11.5に更新されました。新しいバージョンには多くの追加機能があります。

  • 再設計されたユーザーインターフェイス-より便利で効率的になりました。
  • セキュアFTPサポートが導入され、パスワードは暗号化されたプロトコルを使用して送信されるようになりました。
  • 各Webサイトについて、サーバーにインストールされているPHPの任意のバージョンを選択できるようになりました。互換性の問題は忘れてください。
  • ほとんどのApache、NGINX、およびIISオプションは、コントロールパネルで直接構成できるようになりました。
  • 「ハイブリッドWebサーバー」モードが導入されました（PHPアプリケーションの一部はNGINXを使用して起動でき、残りのアプリケーションはApacheで実行できます）。
  • SEOの最適化された設定（ドメイン転送、Webページの適切な停止など）;
  • 新しいバージョンに移行するためのより高速で信頼性の高い手順（プロセスが開始される前に起こりうる問題や制限を警告する移行条件の予備検証のためのメカニズムが導入されました）。


Plesk Panelを使用すると、Webホスティングを迅速かつ最小限の労力で展開できます。 私たちからライセンスを購入することにより、1つのアカウントと統合されたテクニカルサポートで、1つの個人アカウントですべてのサービスにアクセスできます。 （購入ではなく）サーバーとライセンスをレンタルすると、初期コストを大幅に削減し、将来的に容量を簡単に増やすことができます。
