Linux 10.4リリースの計算

予定



Calculate Linuxは、Gentoo GNU / Linuxプロジェクトに基づいて作成され、完全に互換性のあるディストリビューションのファミリーです。 Linuxの計算は、Linuxデスクトップの計算（CLD、CLDG、CLDX） 、 ディレクトリサーバーの計算（CDS） 、およびLinuxスクラッチの計算（CLS）の 3つの領域に分かれています 。



特徴

  • 企業向けのすぐに使えるクライアントサーバーソリューション。
  • Gentooと100％互換性があります。
  • 対話型システムアセンブリのサポート-ディストリビューションに変更を加えます。
  • サポートされる言語：英語、スペイン語、イタリア語、ドイツ語、ポーランド語、ポルトガル語、ロシア語、ウクライナ語、フランス語。
  • HDD、ファイルシステムを備えたUSB-HDD、ext4、ext3、ext2、reiserfs、xfs、jfs、またはFAT32を備えたUSB-Flashへのインストールをサポートします。


主な変更点

  • Calculate Linuxには、GNOMEデスクトップ（CLDG）を備えた新しいCalculate Linux Desktopディストリビューションが含まれています。
  • 認可サーバーのユーザーパスワードを変更するユーティリティ「cl-passwd」が追加されました。
  • ハードドライブで占有されるスペースの量が削減されます。
  • 移動プロファイルデータをリモートサーバーから転送する時間が短縮されました。
  • 〜/ .calculate / ini.envファイルを介してユーザーアプリケーション設定の更新を管理する機能が追加されました。
  • デフォルトでは、パッケージの並列インストールは有効になっています。
  • HP、Epson、およびその他のプリンターモデルの拡張サポート。


CLDではオプション
  • デスクトップビューは、〜/デスクトップディレクトリを表示するように変更されました。
  • OpenOfficeは「Oxygen」スタイルを使用し、ファイルを開いて保存するためのKDEサポートが追加されています。


CLDXではオプション
  • 弱いPCでの作業用に最適化されています。
  • ドメインユーザーセッションに入るときに進行状況バーが追加されました。
  • メニュー構造は、KDEおよびGnomeバージョンとの類推により変更されました。
  • Direct Connect LinuxDC ++ネットワークファイル共有クライアントが追加されました。
  • RSS（Atom）Lifereaアグリゲーターを追加しました。
  • PidginはFacebook、ホットキーのサポートを追加しました。
  • Firefoxはxclearプラグインを追加しました。
  • グラフィカルセッションを開始した後のtty1のキーボードレイアウトを修正しました。


CDSではオプション
  • 〜/ .calculate / desktop.envプロファイルファイルを使用して、サーバーでCLDユーザーコマンドを実行するためのサポートが追加されました。 アーカイブのプロファイルパッキング、LDAPサーバーのユーザーパスワード変更をサポートします。
