そこで、Nokia Xで最も物議をかもした瞬間に到達しました。フィンランドのメーカーは、Androidの本質であるスマートフォンを隠し、その「個性」を見せようとします。 現在のAndroidスマートフォンと比較して、ノキアの仕事は「松葉杖が松葉杖の上に座って松葉杖で追いかける」ことに最も関連しているため、非常に悪い結果になります。 「電話について」のセクションでは、ユーザーの前にAndroidはまったく存在しないが、「Nokia X software platform 1.0」と言われています。よく知られている映画の言葉を思い出します。「だから、彼はこのジャンプで大根を、最初のスカッチャーで分裂するでしょう！」はい、迷彩には特定の問題があります。AnTuTuでさえ、特定の「4.1.2」の存在をすぐに示しました。これは明らかにNokiaのプラットフォームのバージョンではありません。







簡単に言えば、あなたがまだ知識を持っていない場合、モデルのソフトウェアシェルが何であるかを説明します。メイン画面は、Windows Phoneの基本原則に対するノキアのコミットメントを示すための試みです。タイルの連続した垂直キャンバス。各タイルを縮小/増加でき、フォルダーの作成も可能です。追加のデスクトップはありません。そして、ここでは見られないプログラムの「2番目」の完全なリストですら、すべてのアプリケーションとウィジェットは、垂直スクロールのある単一のウィンドウに集まっています。悪いですか？私の意見では、いいえ。 「垂直」について話すと、2012年にSharpスマートフォンのFeel UXシェルでも同じことがわかりました。デスクトップの後ろに隠れているすべてのプログラムのリストはありませんか？だから、これはメーカーのブランドシェルのほぼ半数にとって通常の状況であり、少なくともHuaweiのスマートフォンを覚えています。







画面を左右にスクロールすると、Fasltlaneとも呼ばれるRecentダイアログが開きます。一見、タスクマネージャーですが、RAMの量が限られているため、どうやらこれをまったく信じていません。そして、あなたは知っている、この「ファスレーン」は私が本当に好きだった。まず、7つのアイテムが1つのウィンドウに表示され、Highscreen Zera Fのように3つではありません。2番目に、通常のAndroidでは通知カーテンの後ろに隠れているものがこのダイアログに統合されます。現在のAsphalt 8のキャッシュロードもリストに表示されました。そして、それは非常に便利です。







もう1つは、実行中のアプリケーションの上でFasltlaneを呼び出すことができないことです。プログラムを終了し、画面をスクロールする必要があります。タスク間の切り替え速度が低下します。一般的に、Nokia X SPインターフェースは印象的ですが、予約があります。フィンランド人は、シンプルで直感的なシェルを作成することができました。ここで混乱することはほとんど不可能であり、「親forのためのスマートフォン」にとって理想的なオプションです。多くの人が、経験のないユーザー向けにWPバックグラウンドを推奨しているのは何の理由でもありません。それにもかかわらず、「ティーポット」のスワイプから「裸の」Androidを理解することは簡単な作業ではありません。



悪い点について。作業速度に問題があります。システムの主な欠点は、プログラムからデスクトップにすばやく戻ろうとすると表示されるウィンドウです。また、Quadrantメニューを終了したときに同様の警告が表示されたため、アスファルト8の問題になります。







最初は、低速であることにも気付きませんでした。スムーズに切り替えられるように巧妙に偽装されており、インターフェイスの明らかな痙攣、「遅れ」は見えません。ただし、Highscreen Zera Fに戻るとすぐに、すべてが適切に配置されます。将来的には、これらの同じ年齢がより頻繁に満たされ、ノキアがシステムのその後のアップデートで問題に対処することを願っています。



私はシェルを賞賛しましたが、それはインターフェースの観点からのみで、経験の浅いユーザーの観点からです。 Androidシステムを搭載したスマートフォンに馴染みのある本格的なAndroidインフラストラクチャに関係するすべてのことは、控えめに言っても印象的ではありません。何よりも、現状は、レビューのタイトル「Nokia X Dual SIM：GoogleのないAndroidとは何か」によって特徴付けられています。私は答えます-これは非常に奇妙な製品であり、標準ツールを使用して公式のYouTubeクライアントを入手することさえ非常に困難です。







まず、Google Playはありません。 Nokia独自のサービスに置き換えて、Yandex.Storeからのダウンロードを提供しようとすると、松葉杖の問題に戻ります。このYandex.Storeの[Nokiaが推奨する]タブは、Nokiaサービスに存在しないため、特定のプログラムを探して検索したときに特にm笑されます。さらに、このサービスにはかなりバグがあります。そのため、VKontakteアプリケーションをインストールしようとするとき、電話ソフトウェアを更新することを勧めました。そうしないと、クライアントのインストールは不可能です。私は設定に行き、更新を確認しますが、いいえ-それは空です。これを理解する方法-神は知っています。その結果、Yandex.StoreからContactをダウンロードしました。







一方、Google Playのインストールは、Nokia X / XL Toolsのシンプルなアプリケーションによって許可されています。第三に、主に単純なNokia X SPをターゲットにしているユーザーと同じユーザーにとって、「一歩踏み出す」ことは恐怖から麻痺を引き起こします。まあ、会計士アンナと配管工ヴァシリーは、このような困難に悩まないでしょう。ところで、代替案の質問に-他のシェルをインストールする人はいません。これは作業の速度にプラスの影響を与える可能性があるため、歓迎されます。 Yandex.Shellは非常に優れていることが判明しましたが、Go Launcher EXはアニメーションの滑らかさに満足していませんでした。さらに、代替シェルは標準のタスクマネージャーを提供しないことを覚えておく必要がありますが、Fasltlaneを簡単に元に戻すことはできません。



比較のコンテキストでは、Mediatekのいくつかの追加プログラムを使用して、Highscreen Zera F-Android 4.2.2については実質的に何も言うことはありません。 Nokia Xは、少なくとも特別な暖かさを備えた後、Androidで一見普通に見えるものを知覚するという事実を賞賛することができます。



プレインストールされたNokia Xソフトウェアに関するいくつかの言葉、とりわけ、非標準の音楽プレーヤー、MixRadio、Viber、Skype / Twitter / Facebookクライアント、ASTROファイルマネージャー、WeChatがあります。 MixRadioを本当に評価したかったのですが、プログラムには更新が必要で、NokiaサービスまたはYandex.Storeで検索するために提供されましたが、両方のストアにアプリケーションはありませんでした。

おわりに



Nokia Xに関する多くのレビューを読んだ後、私はこのモデルについて非常に懐疑的でしたが、見た目ほど悪くないことが判明しました。主なことは、Nokia Xにはまだ独自のニッチがあることを理解することです;モデルの潜在的な所有者を2つの主要なグループに分割します。最初-オタク、「ガレージワーク」の愛好家。この観点から、ファームウェアを試したり、「最大限に活用」したりするのが好きな人にとって、Nokia Xは優れた非常に安価なおもちゃです。 2番目のグループは、前述の「相対」です。将来の所有者がAndroidの機能を評価しないことが保証されており、むしろスマートフォンの世界に忠実に参加したい場合は、Nokia Xが適しています。インターフェースの習得が非常に簡単なためです。



ブランドや明るいパネルに対する特別な愛からであっても、他の人にNokia Xを支持する選択をすることはお勧めしません。 Androidスマートフォンの標準機能を背景に、特にHighscreen Zera Fを考慮していなくても、Nokia Xクラフトは、少なくとも当惑させられます。 「AndroidではなくAndroidのような」アプローチで自転車を発明しようとすると、最初は疑わしく見えました。多かれ少なかれ進歩的なユーザーの観点からの結果は悲しい。 Highscreen Zera Fと比較すると、比較的高いパフォーマンス、システムの安定性、および「クランチ」の最初の必要性の欠如により、モデルがより魅力的になります。そして、自律性の利点を忘れてはなりません。 Nokia Xが競合他社より優れているのは、カメラ部分だけです。欠陥の中であまり慰めはありません。高価/安価についてはお話ししません。モデルの価格を思い出させてください。HighscreenZera Fの価格は3,990ルーブル（〜110ドル）、Nokia Xの「ホワイト」小売店の価格は4,990ルーブル（〜138ドル）です。
