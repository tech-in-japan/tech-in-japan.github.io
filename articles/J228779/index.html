802.11ad（WiGig）-範囲とユーティリティ



Wiocity（WiGiGチップセットの開発者の1人）の購入に関するQCA（QuallcomAtheros-Wi-Fiチップセットのリーダーの1つ）の発表の中で、802.11adに対する熱狂の波が生じました。 特に、802.11adでビデオ監視システムを構築するというアイデアがありました-結局、7Gbpsの帯域幅で、多数のカメラから高解像度のビデオを撮影できます！ 私はこれがどれほど現実的かをすぐに理解することにしました。







RFを見てみましょう。

  • Wikiによると、802.11adが最小速度（ 毎秒385Mbps PHY！ ）で動作するための最小信号レベルは-68dBmです。
  • この記事 （ 802.11adの良い記事）によると 、802.11adの最大送信機電力は10dBmです。 ちなみに、これはWi-Fiの10分の1です。
  • 60GHz（ 802.11adが動作する ）でわずか10mの FSPL信号の伝播の損失は88dBです （ 多くのFSPL計算機の1つで計算 ）。 さらに、 酸素分子との共鳴によるさらなる損失がありますが 、これを省略しましょう。この共鳴は100m以上の距離で重要です。




チャネルパワーバジェットの簡単な式を作成したので、802.11adが10mの距離で最低速度で動作するには、約10dBのチャネルゲインが必要であることがわかります 。

詳細とマタン：
  • 予算： TXpower + TX / Rxgains-FSPL> = Rxmin （ 追加の損失を除く ）
  • 私たちの場合： 10dBm +ゲイン-88dB> = -68dBm
  • Get Gains> = 88-68-10 = 10dB




この10dBはどこから来たのですか？ 802.11acおよび802.11adのマーケティング担当者が喜んでペイントするアンテナ増幅またはビームフォーミングのいずれか。 ただし、モバイルデバイスのアンテナは通常、マイナスのゲインを持っているため、ここでキャッチするものはありません。 ビームフォーミング（ 802.11acの結果で示されている ）は、上部で3〜5dBを生成できます。 つまり 5〜7dBは、アクセスポイントの送信アンテナに残ります。



5dBiでアンテナを見つけることは、無指向性であっても問題ではありません（ ただし、ポイント上の指向性アンテナの有用性は限られています ）。 しかし、このようなアンテナを使用した場合のみ、垂直面の開口角は約15度になり、これは非常に小さく、非常に不便です。



そのため、 すでに10mの距離では、802.11adの動作に問題があります。

ビデオのストリーミングに使用できますか？ はい！

  • ビデオをラップトップから外部モニターまたはプロジェクターに2〜3mの距離でストリーミングします。
  • ケーブルやアダプターなしでタブレットからテレビにビデオをストリーミング
  • バックアップのためにビデオファイルを外部ストレージにストリーミングする


監視カメラからビデオをストリーミングしますか？ まあ、ワイヤーをカメラから10m未満の地点に投げることができない場合... :) 2番目のオプションは、高ゲインのカメラの指向性アンテナです （ KU 12-18dBiの60GHzのコンパクト指向性アンテナが宣言されています ）-寿命（および酸素）はどれくらいかを示します実用的です。







一方、このような短距離は、ホームユーザーにとっては大きな恩恵です。壁の後ろにいる隣人は、最終的に「Wi-FiでWi-Fiを詰まらせる」ことをやめます。 同様に、企業のWLAN管理者は、既にロードされている2.4GHz帯域からBYODshnyトライフル全体がどのように削除されるかを確認する機会があります（同時に、5GHzをそれほど速く詰まらせません ）。



一般に、ブルジョアが言うように：「適切な仕事のための正しいツール」（ 写真はクリック可能です ）。 図からわかるように、802.11adは「長距離」アプリケーションを想定していなかったため、実際には発見しませんでした。 ただし、他の目的のために技術を使用しようとする愛好家は常に存在します。 この投稿が彼らとのコミュニケーションに役立つことを願っています。
