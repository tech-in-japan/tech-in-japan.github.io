アンチP2P組織からの隠蔽

もちろん、ここではもちろん、セクションを暗号化してマスクショーや刑罰機関の代表者から保護することが一般的になっています。 しかし、欧米諸国からの情報から判断すると、すべてがそれほど厳しくはなく、著作権所有者の代表である民間企業がこれを行っており、ドア、手錠、四半期の周囲の狙撃兵などを破壊することなく単にあなたを訴えています、裁判所がその内容を確認することを決定するまで待たずに、すべての情報を含むコンピューターを物理的に破壊するのに多くの時間があります。



さらに、完全に合法的なmp3アルバムをコンピューターに置くことができます。正式にCDを購入したため、誰もが私たちの時代には誰もCDを聴かないことを理解しています。 CDがあるのでトレントからアルバムをダウンロードすることもできますが、MP3に転送する方法がわかりません。 絶対に不可能なのは、どんなに受け取ってもアルバムを配ることです。



したがって、暗号化されたパーティションに何があっても、海賊版対策がIPからの配布を検出した場合、すでに問題が発生しており、ディスクがランダムデータで満たされていることを誰も理解できません。 裁判所は何も証明しないかもしれませんが、彼らはあなたがあなた自身が友好的な合意に達することに同意し、両当事者に合った金額を支払うようなhemoをあなたに提供します。 そして、警察、手錠、証人などは、ライセンスのないソフトウェアを使用することに惹かれたい場合（そして、私が理解する限り、あなたはこれから商業利益を得ていることを証明しなければなりません） 。





ご存知のとおり、トレントトラッカーは、接続しているすべての参加者に他のすべての参加者のIPアドレスを提供します。ダウンロードを開始するとすぐに、全員がそのことを知るようになります。これがピア2ピアの原則です。 したがって、著作権者のレーダーに乗らないようにするには、それらへのアクセスとそれらからのアクセスをブロックする必要があります。 このために、特に、IPアドレスの異なる、絶えず更新されるリストがあります。



Ubuntでは、すばらしいファイアウォール（または、それを何と呼ぶべきでしょうか？）を置くことができます。具体的には、この鋭利なもののために： MoBlock 著作権所有者によって何らかの形で接続される可能性のあるIPアドレスのリストを毎日更新し、iptablesを介してこれらのアドレスを使用した発信および着信トラフィックをブロックします。



ソースを/etc/apt/sources.list



追加します（以降、Ubuntu 9.10のすべての例）： 

 deb http://ppa.launchpad.net/jre-phoenix/ppa/ubuntu karmic main




ソースキーを入力します。 

 gpg --keyserver keyserver.ubuntu.com --recv 9C0042C8 gpg --export --armor 9C0042C8 | sudo apt-key add -




更新された：

sudo apt-get update







そして置く：

sudo aptitude install moblock blockcontrol mobloquer







インストールすると、セットアップダイアログが表示されます。

  • 使用されるソースのリストには何もお勧めしませんが、妄想に応じてドーズを入れてください。 デフォルトでは、かなり普通に設定できますが、必要なものとそうでないものを自分で確認することを強くお勧めします。 各リストの説明へのリンクがあります。 後で、もちろん、設定を変更することも可能になりますが、すぐに理解することをお勧めします。
  • いずれの場合でも開かれるポートのホワイトリストでは、ポート22（使用する場合はssh）と5190（ICQを使用する場合。MoBlockはデフォルトでICQサーバーを含むAOLネットワークをブロックします）を追加できます。
  • ローカルネットワークに非標準のアドレスリストがある場合は、それをホワイトリストに書き込みます


インストール後、インターネット-> Mobloquer（監視および設定用）およびトレントクライアントを起動できます。 ブロックされたログを瞑想すると、人々がダウンロードして配布するすべてのリストが何十もの組織のアーカイブに長い間保存されていることがわかります。唯一の質問は時間です。私たちはまだ誰にも興味がありません。
