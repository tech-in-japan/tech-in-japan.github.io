小冊子でPDF文書を印刷する

論文の要約を準備する過程で、okularはpdfブックレットの印刷方法、つまりページを並べ替えてサイズA5のブックレットを取得する方法を知らないことがわかりました。 この問題で私が見つけたのは、psbookとpsnupユーティリティを使用することだけでした。これは、pdfをpsにオーバークロックしなければならなかったため、私には不向きでした。 何らかの理由で、この変換の結果、サイズが50メガバイトのpsファイルが取得されましたが、これは私には不適当でした。



しかし、私はと呼ばれるLaTeXのための素晴らしいパッケージ、見つけpdfpagesを 。 これにより、サードパーティのpdfファイルから技術文書にページを挿入し、あらゆる方法で変換できます。 私はTekhov非常に単純なコンテンツを持つ別のファイルを作成しました： 



 \ documentclass [レターペーパー、11pt] {記事}
 \ pagestyle {プレーン}
 \ usepackage {pdfpages}
 \ USEPACKAGE [a4paper、NOHEAD、includefoot、MAG = 1000] {}ジオメトリ
 \ begin {document}
  \ Includepdf [ページ= { - }、署名= 24、風景] {} AUTOREF
 \ end {document}




コンパイル時に、このファイルはautoref.pdfファイルからページを取得し、必要に応じて再ソートし、ページごとに2を配置します。 Makefileに1行追加すると、すぐに完成したpdfが得られます。両面（長辺）に安全に印刷して、大切な本を入手できます。



このようにして、pdftexで生成されるとは限らない任意のpdfファイルをブックレットに変換できることに注意してください。
