メカニカルキーボード

私たちは長い間あらゆる種類のサイバーグローブとバーチャルリアリティの大量出現を長い間待ち望んでいましたが、それでもオフィスや自宅では毎日、通常のキーボードのような当たり前のことをしなければなりません。



そして、定期的に何かに出くわすので、次第に疑問が生じます-仕事用の従来のキーボードにもっと便利なオプションを見つけることは可能ですか？



それで、しばらく前にキーボードに関する情報の収集に参加し、記事の情報を準備し始めましたが、残念なことに（残念なことに）一度にすべてのニュアンスに関する網羅的な記事があったので、翻訳することにしました。



メカニカルキーボード。 旅行ガイド



事実：あなたの周りに見えるほとんどすべての最新のキーボードは、メンブレンキーボードです。 この技術は、テレビの安価なリモコンで使用されているものに似ています。 製造業者は、大きな利益を上げるためにキーボードをできる限り安価にしています。 はい、高価な「ゲーミング」キーボードと「デザイナー」キーボードを除外しません。 しかし、もともと安価なものとして行われていたものを使用するのはなぜですか？



では、なぜメカニカルキーボードが必要なのですか？

メカニカルキーボードを使って作業を試みた人のほとんどは、感覚が自分自身で話していると主張しています。 現在座っているキーボードでは、キーが最後まで押し下げられて、キーボードが押しを記録するように、膜に接触する必要があります。 これにはある程度の努力が必要であり、指が著しく疲れ、印象は、あなたがプラスチック片全体を一日中叩いているという印象です。 キーのメカニカルスイッチは、ボタンを「最後まで」押す前にキーを押すとアクティブになるように機能するため、スウィッティングに余分な電力を費やすことはありません。 また、さまざまなタイプのボタンを使用して、キーストロークから最も快適な努力と感覚を選択できます。 かつてメカニカルキーボードで働いていた人の多くは、通常のメンブレンキーボードに戻りたくないでしょう。



NBまた、メカニカルキーは膜よりもはるかに信頼性が高いことにも注意する必要があります。たとえば、メーカーはCherry MXキーの5000万回のキーストロークを主張しています。 これは、メンブレンキーボードよりも1桁または2桁大きいです。



用語



キーブロックとゴースト

架空のキーストローク -これはキーボードの2つのキーを押したときで、3番目のキーを押さなかった場合でも機能します。 最新のキーボードでは、メーカーが修正を加え、そのような組み合わせの動作をブロックしているため、最も安価なキーボードでもこれは見られなくなりました。



キーブロックとは、特定のキーボードモデルで許可されている最大数を既に押した場合に、新しいキーをブロックすることです。 したがって、2つのキーを同時に押して、3番目のキーを押しても機能しない場合は、一度に2つのキーのみを処理するキーボードが確実にあります（2キーロールオーバー= 2KRO）。



同時キーストローク数 （キーロールオーバー、「＃KRO」および「NKRO」）

ロック不可の「NKRO」キーボードは、必要な数のキーを同時に押すことができ、すべてのキーストロークがコンピューターに送信されます。 「ゲーミング」キーボードの一部のメーカーは、そのようなキーボードを「アンチゴースト」と誤って呼びました。 また、LogitechとRazerのゲーミングキーボードメーカーでさえ、WASDキーの周りの小さなブロックのロックを解除するだけです。 PS / 2を介して接続されたキーボードのみが完全な非ブロックモード（NKRO）で動作できることに注意してください。 また、MicrosoftとDuckyのキーボードのみが、USBを介してNKROを操作する2つのメーカーです。

（#KRO、「＃」の代わりに数字は同時に処理されるクリックの最大数です）



USB経由で接続した場合、ほとんどのメカニカルキーボードは6KROとマークされます。つまり、6つのキーを同時に押すことができます。 これはほとんどのユーザーにとっては十分なはずですが、一部のゲーム（特に分割画面をプレイする場合）と6つのキーでは、十分でないことが判明する場合があります。

6KROを備えたUSBキーボードでは、6つの通常のキーで最大4つの修飾キーを押すこともできます。 修飾子は、CTRL、ALT、Shift、およびWinです。



キーバウンス

膜のキーを含むすべてのタイプのキーには、びびりがあります。 キーを押して連絡先がタッチを開始すると、キーボードコントローラーは多くのキーを押す/離す操作を登録し始めます。 1回押すと1文字でいっぱいのフルスクリーンが表示されないようにするため、コントローラーでは小さな一時停止が使用されます。 コントローラーは、数ミリ秒待ってチャタリングを解消し、その後クリックのみをコンピューターに送信します。 メカニカルキーボードでは、チャタリングを回避するために必要な時間が著しく短くなります。たとえば、Cherry MXスイッチを使用する場合、5ミリ秒の遅延で十分です。一方、メンブレンキーボードは非常に長い休止時間を必要とします（仕上がりの品質に応じて、キーボードが悪いほど休止時間が長くなります）。



ポーリング率と応答時間

マウスの場合、ポーリング頻度は重要な指標であり、カーソルの滑らかさ/精度に直接影響しますが、これらのパラメーターはキーボードには使用できません。 すべてのボタンが5ミリ秒の遅延応答を持っていると想像してみましょう（理想的な条件に近い）。 たとえあなたがスーパーマンであっても、各プレスは少なくとも5ms遅れます。 したがって、200ヘルツを超える実際のポーリング頻度はまったく役に立ちません。 さらに有害です、なぜなら 頻繁で無駄なキーボードポーリングでプロセッサをオーバーロードします。 USB接続とは対照的に、PS / 2に含まれるキーボードはまったく問い合わせられず、プレスが発生すると信号を送信し、ハードウェア割り込みが発生し、プロセッサがプレスに通知します。



PS / 2またはUSB ？

PS / 2はすべてのカウントで勝ちます：まず、無制限の数の同時クリックをサポートします（フルnキーロールオーバー）。 第二に、PS / 2は一定のポーリングを必要としませんが、ハードウェア割り込みを使用します。 第三に、他のデバイスとのUSBバスの過負荷によりキーストロークが遅延する可能性はありません。 USBデータ転送には、割り込みベース（キーが押されるとUSBコントローラーがキーボードを常にポーリングし、USBコントローラーが割り込みを送信する）と、バス帯域幅の特定の部分が予約されて遅延がないことを確認するアイソクロナス転送モードの2種類があります。 残念ながら、2番目のモードには特別なコントローラーが必要であり、価格の理由だけでメンブレンキーボードでは使用されません。



PS / 2とUSBの両方を使用している場合、PS / 2接続を使用しない理由はありません。



メインボタン



ボタンではなく、ボタン



多くの人は、最終的に何を取得したいかを正確に考えずに、どのボタンが優れているかを尋ねます。 幸いなことに、これは問題ではありません。ほとんどのメカニカルボタンは、どのメンブレンキーボードよりもはるかに優れた感触を与えるからです。 それにもかかわらず、ボタンは単なるボタンではなく、キーボードの中心であるため、最終的な選択は依然として重要です。 キーを押す感覚が気に入らない場合は、キーボードでの作業が嫌になります。



ボタンの応答力は通常グラム（g）で測定されますが、実際には力はセンチニュートン（cN）で測定されますが、1グラムの重量はほぼ0.01ニュートンの力で押し下げられます。 したがって、「55g」と書くのが慣習的です。これは、簡単にするために0.55ニュートンの努力を意味しますが、これは完全に正しいわけではありません。

クイッククリーニング

キーボードはかなり早く汚れます。 実際、コンピューターで作業する前に手を洗う人はほとんどいません。また、キー間の隙間からキーボードに落ちるほこりや髪の毛が常にあります。そのため、少なくとも月に1回はキーボードを掃除することをお勧めします。



1.掃除機またはコンプレッサーでキーの下から汚れを吹き飛ばします。

2.アルコールで湿らせた清潔な布でキーとキーボード本体を拭きます。

注意：一部のキーボード（Filcoなど）でアルコールを使用しないでください。キーラベルが損傷する可能性があります。

3.ヘビースモーカーの場合、キーボードのプラスチックがすぐに黄色に変わることがあります。 Windexタイプの製品（アンモニアを含むガラスクリーナー、アナログ「Mr. Muscle」）で黄色を拭くことをお勧めします。



キーボードを清潔に保つと、長い間目を楽しませてくれます。
