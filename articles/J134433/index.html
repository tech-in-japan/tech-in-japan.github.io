ReactOSダイジェスト＃3：次へ

2011年11月29日に 、ReactOSプロジェクトチームの代表者が第4回年次連邦イノベーションコンベンションに参加しました 。 その結果、チャンネル5のレポートが作成されました（ジャーナリスト、いつものように、やや装飾された現実） 、そしていくつかの新しいパートナーとの有望なコラボレーションの始まりです。











  • ドイツの銀行がプロジェクトに1000ユーロを寄付しました
  • Microsoft Visual C ++ Redistributableのサポートを実装しました。 すぐに、システムにこれらのライブラリを必要とする何百もの新しいアプリケーションが必要になります。
  • scsi-interfaceを備えたスキャナーは既にReactOSで動作していることが判明しました
  • WinRARとTotal Commanderの作者は、チームがソフトウェアに関連するバグを修正するのを助けました。 私たちは彼らに感謝し、このメッセージを読んでいるすべてのソフトウェア作成者にアピールします。私たちのシステムの下であなたの製品のデバッグに参加してください。 私たちはどんな参加にも喜んでいます。
  • Operation PIの一環として、 アプリケーションのプレリリーステストが開始されました。 どなたでも大歓迎です。
  • さらに、開発者は、1年以上コメントが寄せられていないバグのレビューについて支援を求めています。
  • このプレイリストでは、いくつかのおもちゃの動作を確認できます。
  • ReactOS Foundationへの寄付は、ビットコインサイバー通貨で送金できるようになりました 。 私たちはどんな寄付にも喜んでいます。あなたが私たちと一緒にいることを知ることは私たちにとって重要です！




PSプロジェクトは全盛期にあります。 あなたが投資家であり、当社のオペレーティングシステムへの投資を検討している場合は、これを決定する時が来ました。先駆者には排他的な条件があります。

あなたがC / C ++プログラマである場合、開発への参加について考える必要があります。これは、将来、名声、経験、そして確かな給料を得るチャンスです。
