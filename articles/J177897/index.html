Ubuntu 13.04リリース

（ このニュースは新しいものではありませんが、ubuntu 13.04に関するより詳細な投稿にコメントを求めた場合）



今日、Raring Ringtailというコードネームで最も人気のある Linuxディストリビューションの1つがリリースされました。

最近、公式サイトのubuntu.comがこれについて話し合っています。

ダウンロード用の画像は正午ごろ入手可能です。







更新：


  • Linuxカーネル3.8.8
  • スタートアップ1.8
  • Python 3.3
  • CUPS 1.6.2、cups-filters 1.0.34
  • Libreoffice 4.0
  • Unity 7




Unity 7の更新：


  • Ubuntu Oneサービスを使用したデータ同期のメニュー
  • 新しいシャットダウンダイアログ
  • 新しいアニメーション、アイコン、その他の変更
  • Unityパネルのマウスホイールで開いているウィンドウを切り替える機能を追加しました
  • 多くのバグが修正され、パフォーマンスが向上しました






















詳細情報：


ここで 「感じて」 ください 。

ubuntu 13.04のサポートは、 わずか9か月しか続きません 。

このバージョンのディストリビューションにはWUBIインストーラーはありません 。

変更の詳細な概要： wiki.ubuntu.com/RaringRingtail/TechnicalOverview

ubuntuバージョンの更新手順： help.ubuntu.com/community/RaringUpgrades 。
