NNLUGからのUbuntu 11.10ビルド

伝統的に、次のLinux InstallFestのために、Nizhny Novgorod Linux User Groupは、インストールに便利なUbuntuの新しいバージョンを構築する準備をしています。



インストール直後に、すべてのユーザーがシステムのRussificationのパッケージ、オーディオセット、ビデオコーデック、フラッシュプレーヤー、他のプラットフォームからのドキュメントとの互換性のためのフォントなどをダウンロードすることは周知の事実です。



これらのタスクを容易にし、Linux InstallFestとシステムを自宅にインストールするときの両方でインターネットチャネルを保存するために、これらすべてと他のいくつかを含むアセンブリを作成しました。



ここからNNLUGからUbuntu 11.10ビルドをダウンロードします。

Yandex： http : //narod.ru/disk/31588522001/Ubuntu-11.10-desktop-i386-NNLUG.iso.html

トレントUbuntu.ru：http://s1.ubuntu.ru:6969/



Ubuntu 11.10 NNLUG Remixで行われた変更のリスト：



  • GnomeおよびKDEアプリケーションの両方にローカライズパッケージを追加しました。 ダウンロード後すぐに、すべてのアプリケーションがロシア語で「あなたと話をします」
  • Gnome 2のようなクラシックデュアルパネルインターフェイスに切り替えるGnome 3 Classicパッケージを追加しました
  • 必要なすべてのオーディオビデオコーデック 、フラッシュプレーヤー、あらゆる種類のフォントを追加しました
  • LibreOfficeにOpenClipartイメージセットが追加されました
  • 最も必要なアプリケーションを追加： GIMP、クレメンタイン、チーズ （非常に謙虚な意見）
  • 大量のサイズ変更と写真の回転のためのNautilusプラグインが追加されました（非常に便利です！）
  • tzdataパッケージの設定をモスクワ時間に変更しました
  • zipアーカイブ内のロシア語のファイル名に関する問題を修正
  • Nvidiaビデオカード用のキャッシュされたバイナリドライバーパッケージ
  • 文書テンプレートを追加しました。 右クリックして、さまざまな種類のドキュメントを作成できるようになりました！


InstallFestの後、新しいアセンブリで修正されたいくつかの欠点が発見されました。

  • LibreOfficeからドキュメントを印刷する際の問題を修正しました（問題は元のUbuntuに存在します）
  • 4つのローカライズパッケージが追加されました。ハードドライブにシステムをインストールした後、アセンブリがインストールを要求しました（主に英語のヘルプページ）
