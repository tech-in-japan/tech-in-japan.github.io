メタルストーム



リリース年：1991


技術的特徴：


  1. スムーズでスタイリッシュなアニメーション
  2. リアルな重力効果
  3. レイヤード背景
  4. ブレーキなしの視差スクロールがたくさん




何が特別なのか：


スクリーンショットを見てゲームを書き留める前に、実際の動作を確認してください。 このアクション映画のすべてのグラフィックの利点が目に見えるのは動いています。 キャラクターの1つのフレームはそれほど不安を引き起こしませんが、どのように動くかを見て、とてもクールです。

Irem（ゲームの開発チーム）は、このシューティングゲームをプレイアビリティとグラフィックの点で「アサルトスーツレノス」（ジェネシス/メガドライブの「ターゲットアース」ゲーム）のプロトタイプのように見せました。

このゲームの最大の特徴は、重力の方向を変えて天井を歩く能力です。 同様のトリックは他のゲーム（メガマン、ガンスターヒーローズなど）でも見られますが、メタルストームはこの点で先駆者でした。さらに、重力の影響は他の8ビットゲームよりも現実的です。

おそらくこれは些細なことですが、Metal Stormの爆発のレンダリングは多くの16ビット兄弟と競合します。 背景には、前景と同様に2つのレイヤーがあります。 その結果、運転中にリアルな遠近感が得られます。 SNES時代以前は、階層化された背景は非常にまれでした。

複数のオブジェクトが画面に同時に表示されると、ちらつきが発生する場合があります。 ただし、これはめったに起こらず、ゲームプレイの速度を低下させることはありません。

バットマン：ジョーカーの帰還



リリース年：1991


技術的特徴：


  1. 完全に16ビットゲームになります
  2. 大きなスプライトとボス
  3. 素晴らしいカラーワーク
  4. 高速でスムーズなアニメーション




何が特別なのか：


動作中のこのゲームを見て、ピクセル密度は見ていないが、これはGenesis / MegaDriveプラットフォームの初期のゲームの1つであると判断できます（ちなみに、移植版があります）。

最後のボスとの戦いのビデオを見てください。 Sunsoftの開発者は、システムリソースの効率的な使用における倒錯について多くのことを知っていることがはっきりとわかります。 彼らは、NESのすべての制限により、独特の陰鬱で中毒性のある雰囲気を作り出しています。 また、大量の背景アニメーションにもかかわらず、キャラクターがどれだけ速く動くかを示します。 標準レベルでさえ、次世代コンソールのゲームのように見えます。 敗北した敵のレンダリングと武器の発射 Gunstar Heroesを思い出させます。 もちろん、ゲームにはGunstar特有の機能はあまりありませんが、レベルによっては非常に均一なものもあります。

グラディウス＆グラディウス2



生産年：1986年と1988年


技術的特徴：


  1. 画面上で同時に移動する多数の敵とシェル。
  2. ブレーキとフリッカーの最小数
  3. アーケードマシンから転送された詳細なグラフィック。


何が特別なのか：


NESは、その生涯において、スロットマシンから移植された多くのゲームを見てきました。 ただし、Gradiusが行うように鉄製コンソールをレイプしているのはほとんどなく、Gradius IIは日本でのみリリースされています。

典型的な水平スクロールは、多くの敵とそれらが発射するシェルを同時に表示します。 コナミのプログラマーがこれをほとんどちらつきやブレーキなしで実行したことは印象的で、その時のブレークスルーでした。 グラフィックは、このポートを元のアーケードゲームと比較するのに十分なほど詳細です。 音楽はわずかに単調ですが、宇宙の撮影にはまだ十分です。 これはすべて、NES時代の夜明けにリリースされたゲームを称えるものです。 実際のシューティングゲームの熱狂に興味がある場合は、2番目のシリーズGradius IIのビデオを見てください。

コントラ



リリース年：1988


技術的特徴：


  1. 厳しい小競り合いにもかかわらず、ブレーキの最小数。
  2. たくさんのアニメーション爆発。
  3. 多数の飛翔体で擬似3Dで作成されたレベル
  4. 巨大なボス
  5. 日本語版では、レベル間のミッションとビデオ挿入のマップがありました。


何が特別なのか：


コナミのもう1つのアクションゲームであるContraは、アーケードマシンからNESに移され、コンソールの歴史の中で最高のゲームの1つになりました。

ゲームプレイは非常に元気があり、非常にめったにブレーキに隠れることはありませんが、スプライトのちらつきを観察できる場合があります。 このゲームはたくさんのカラフルな爆発でいっぱいで、80年代に撮影されたクールな筋肉マッチョのアクション映画のノスタルジアに突入します。

移動の方向は8レベルで変化します。水平スクロールは垂直に、またその逆になり、その間でプレイヤーは第3シャフトで弾丸と手rena弾が噴出する擬似3Dトンネルに入ります。

サウンドデザインは、ゲームプレイと同じようにドライブしてビープ音を発します。 音楽は静かな雰囲気の曲から強力な合唱曲まで多岐にわたります。 巨大なボスは特別な称賛に値します。 さて、そして最も重要なことは、2人のプレイヤーが同時に悪と戦うことができるということです！

シリウスへの旅



（ロシアでは、Raf Worldとして知られています-約。）

リリース年：1990


技術的特徴：


  1. 詳細かつスムーズなキャラクターアニメーション
  2. フルスクリーンボス
  3. 背景と環境の詳細図
  4. キラー音楽を備えたユニークなオーディオ技術


何が特別なのか：


Sunsoftは、NESで最も器用な開発者がいることを、行為によって繰り返し確認しています。 Journer to Siliusは、その優れたグラフィックス、サウンド、およびゲームの仕組みのおかげで、当時の最も先進的なリリースの1つになりました。 開発者は明らかに、メガマン（ロシアではロックマンとして知られている-約Per。）やコントラなどの不滅の古典からインスピレーションを引き出しましたが、多くの点で前任者を凌ぎ、非常に技術的に高度なゲームを作成しました。

キャラクターアニメーションは、特にランニングやジャンプ中は非常に楽しいです。 ジャンプ中の動きをパルスで計算する方法により、リアリズムと複雑さが増します。 英雄的な成長のゲームのボス、カラフルで詳細な。 ゲームの背景アニメーションは特別な賞賛に値します-それはただ美しいです。 風景の中には、終末論的な未来の暗い世界が最も微妙な色と細部で見えます。 たとえば、第2レベルでは、シーンは、地下構造物の壁に沿ってゆっくりと明滅する赤信号まで処理されます。 このゲームはもともとターミネーターユニバースに基づいて作成されたため、これは驚くべきことではありませんが、作成の過程で、開発者はライセンス権を奪われました。

NESには音楽の能力に感動するゲームはあまりありませんが、Journey to Siliusは珍しい例外の1つです。 このゲームのユニークな機能は、 DPCMオーディオチャネルが他の目的でここで使用されることです。ドラムの代わりにベースサンプルが送信されます。 この場合、打楽器は三角波のチャンネルを使用するため、まったく異なる音色が得られます。 ゲームのサウンドは技術的に優れているだけでなく、その時代の最高のゲームサウンドトラックの1つです。
