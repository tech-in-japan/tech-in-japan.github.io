PressAbout.usは、あなたの場所でプレスと連携します



最初から、有望なインターネットプロジェクトを他のサイトで質の高い方法で提示する必要があります。 これらのサイトには、ブロガー、友好的なコミュニティ（habr）、オンラインメディア、PRエージェンシーがあります。

Runetのプロジェクトは、ニュースメーカーの情報をほとんど気にしません。 そして、外国のプロジェクトの作成者は、マスコミとのコミュニケーションの価値を長い間理解してきました。 メインページの中には、間違いなくプレスまたはプレスルームがあります。 彼らは通常そこに配置します：

  • プレス連絡先
  • プレスリリース
  • 公開された記事のリンク
  • メディアキット-マルチメディア情報
  • ロゴと追加ファイル


これにより、すべてのWebプロジェクトに便利な拡散のボーナスが与えられます。



オンラインジャーナリストにとって、この情報は非常に役立ちます。 これらのデータによると、プロジェクトがどれほど深刻で、インターネットコミュニティにとってどれほど魅力的でありたいかがすぐにわかります。

この問題を提起しました。 プロジェクトに関するサードパーティの情報をフォローしたかったのですが、これはネットワークのさまざまな部分に定期的に表示されます。 そこから、新しいアイデア、批判、協力の機会を引き出しました。 他の人がこれを行うのを防ぐために多くのことを行ってきました。

まず、プロジェクトの1つで、ウィジェット「彼らは私たちについて書いています」が表示されました。



しかし、その機能は十分ではありませんでした。 エントリーをモデレートしたかったので、自分で追加しました。 そして、機能について考えたとき、これは別個のサービスであることが明らかになりました。 しかし、当時、プロジェクトのビジネスモデルはまだ明確ではありませんでした。 時間が経ち、機能が再考され、330のチケットがクローズされ、1314のコミットが開始され、 pressabout.usが誕生しました。



写真付きの機能の簡単な説明：



プレスルーム



ユーザーはプレスルームを作成します。 プレスルームが何に専念しているか、誰がプレスリリースと記事へのリンクを追加するかを簡単に記入します。 サイトに関連するサードパーティの記事が自動的に検索されます。 プレスルームは、ドメインまたはサブドメインにリンクできます。 通常、 press.domain.comにインストールされます 。 これを行うには、pressabout.usでDNSにCNAMEを追加します

例、 PressAbout.usのプレスルーム



プレスリリース



次の形式があります。

  • タイトル-タイトル
  • ソーシャルメディアピッチ-ツイッターなどの短いプレスリリースの説明
  • 要約-要約
  • コンテンツ-詳細なコンテンツ
  • リンク-追加資料へのリンク
  • 引用-レビュー、印象
  • 画像、動画、タグなど


この形式により、すべてのユーザーがプレスリリースから必要な情報を選択できます。







ウィジェット



自分のページで「プレスアバウト」ウィジェットを使用し、 リンクを使用して、任意のプラットフォーム（Wordpress、Blogger、Typepad、またはJavascript）にウィジェットをインストールしたい人。 付属のcssファイルを使用して、ウィジェットの外観を完全に変更できます。







ウィジェットをhtmlとしてダウンロードして、キャッシュすることができます。例：

http://pressabout.us/widgets/yousticker.com.html?theme=modern&include_css=true

また、プレスルーム用のウィジェットを提供し、最新のプレスリリースを表示します。







HTMLウィジェットの例-http ://pressabout.us/widgets/pressabout.us/press-room.html



API



ウィジェットと一緒に、この情報を処理するAPIを徐々に開発しています。 これまでのところ、APIはデータ出力に対してのみ機能します。 例：

http://pressabout.us/widgets/yousticker.com.json?num=100-JSON形式



競合他社の追跡



多くの場合、プロジェクトの開発が「誰が推進したのか」という簡単な言葉でどのように進んだかを判断する必要があります。 これは、pressabout.usを使用して行うこともできます。 pressabout.usロボットは、成長中の人気プロジェクトを常に監視し、それらに関する独立した情報を収集します。 この情報はすべて無料で提供されます。 たとえば、 Instagram.comについてのプレス

各サイトについて、人気のインデックスが計算されます-最も人気のある5つのプレスのお気に入り。
