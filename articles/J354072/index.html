Gamedev-festival DevsGO 6月2日、モスクワ

近い将来、ハイアーズスクールオブエコノミクスのハイアースクールオブビジネスインフォマティクスで、ゲーム業界でいくつかのオープンイベントを開催します。 彼らは自由に訪れることができ、誰でも来て参加することができます。 入り口はリストを介しているため、主なことは事前に登録することです。



  • 4月25日（水曜日）夜の教育プログラムのオープンドア日ゲームプロジェクトの管理。
  • 6月2日（土曜日）から終日DevsGo Game Dev Festival

6月2日DevsGo Game Dev Festival



→ 登録







12:00から夕方まで、イベントのゲストはゲーム業界の代表者からのレポートを聞いたり、講師に質問したり、ショーケースで紹介されたインディーデベロッパーのプロジェクトに精通したり、コンテストに参加して記念すべき賞品を獲得したり、志を同じくする人々の会社で1日過ごしたりすることができます-知り合いや有用な連絡先を作成します。



そして、ここで重要なマークを付けた太字で強調表示し ます。ショーケースでゲームプロジェクトを表示する場合は、コメントを記入してください。 ご連絡いたします。 すべて無料です。主なことはゲームが良いことですが、誰かに行方不明になった場合にチームに参加できるゲストとの会話、フィードバックの収集、ライブチャットを行うと便利です。 展示場所、テーブル、椅子、ソケットを提供します。 ゲーム、ゲームを表示するデバイス、配布資料などがあります。 ゲームはまだ実行されていない可能性があります。主なことは、高品質のプレイ可能なバージョンを使用することです。

13: 00-13 : 45- 「ファイナルステーション：コンセプトからリリースまで」 、Andrey RumakおよびOleg Sergeev、開発者、ゲームデザイナー。



14: 00-14:45- 「市場のニッチを探す」 、ゲームプロデューサー、ゲームプロジェクト管理プログラムのプレーヤーの心理学の教師、アレクサンダーテジャエフ。



14:45-15:30-休憩



15:30-16:35-ラウンドテーブル「ゲーム業界の発展-誰が責任を負い、何をすべきか？」 参加者：セルゲイ・ギメルレイク、デニス・ダヴィドフ、アレクサンドル・ルキチェフ、エレナ・ブガコワ。



16: 35-17 : 20- 「ゲーム業界での話：勝つための話」 、ARPU.GURUのGDナイトであるSergey Kononovは、サンクトペテルブルクのMosigraネットワークを5年間管理したSlash Arena：Onlineプロジェクトの開発に参加しました。
