WordPressは金曜日に8歳になりました。 スクリーンショットのWordPressストーリー





5月27日金曜日、最も人気のあるWordPress CMSはちょうど8歳でした-2003年5月27日、同じ名前のオープンソースCMSが発表されました。 b2ブログプラットフォームのフォークとして登場したものは、数百万人が使用する信じられないほど人気のある自己完結型プラットフォームになりました。 Wordpressサイトは、小学生から最大のオンラインメディアまで、あらゆることを行います。 現在、Mashableを含む約4,500万のサイトがこのCMSに取り組んでいます。 スクリーンショットでWordpressの履歴を再確認し、それがどのように始まったかを思い出してください。 個人的には、バージョン2.3.3からWordpressを使い始めました。 すべての中間バージョンを示すわけではありません。レビュアーには、主要なバージョンと更新のみが表示されます。



Wordpress 0.7.1



2003年5月27日に発表された同じリリース。







Wordpress 1.0.1



このバージョンは2004年1月に登場しました。 この更新には、更新されたインストールシステム、新しいデフォルトテーマ、およびその他のいくつかの改善が含まれています。







Wordpress 1.2



このバージョンのリリース日は2004年5月です。プラグインのサポートがWordpressに登場しました。Wordpressは、このシステムを非常に人気のある重要な要因の1つにしました。







Wordpress 1.5



次の日付は2005年2月です。 ページの作成機能、新しいデフォルトテーマのキューブリックなどの新機能。 このテーマは、5年間デフォルトのテーマのままでした。







Wordpress 2.0



ブロガーへの贈り物は2005年12月31日に作成されました。 新しい管理パネル、WYSIWYGおよびその他の機能。







Wordpress 2.1



Wordpress 2.0のリリースから1年以上が経過し、2007年1月に新しいバージョンが登場しました。ここでは、エディターに自動保存、タブが表示され、さらにサイトのホームページとして任意のページを設定できます。







Wordpress 2.2



2007年5月-Wordpress 2.2のリリース。 ウィジェットが表示され、Wordpressワークフローも変更されます。







Wordpress 2.3



2007年9月 新機能-タグとカテゴリのサポート。 それは今驚くべきことではありませんが、その後-それは何かでした。







Wordpress 2.5



2008年3月-新しい管理パネル、さらに多くの追加の変更と革新。 素材を追加するためのインターフェイスも変更され、多くのユーザーの間で怒りが生じました。 しかし、しばらくすると、これらすべてが再び変わりました。











Wordpress 2.7



2008年12月Wordpress 2.7がリリースされました。 管理パネルは完全に変更され、現在の表示に似ています。 ブログに資料を追加するためのインターフェースも変更され、Wordpress 2.9までほとんど変更されませんでした。











Wordpress 3.0



2010年7月-Wordpress 3.0、キーと呼ばれる最大のプラットフォームアップデートの1つ。 多くの革新と新機能。 サイトに資料を追加するためのインターフェイスは、モジュール化され、変更可能になりました。











Wordpress 3.2



新しいバージョンのリリースは今年6月に予定されています。 開発者によって提示されたベータ版では、新しい資料の追加を含む更新された管理パネル。 また、開発者はIE 6のサポート終了を公式に発表しました。ちなみに、気晴らしフリーライティングモードも追加されました。このモードでは、エディターはテキストの操作を妨げません。
