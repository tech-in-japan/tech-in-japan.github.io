Google Developer Launchpad-Googleスタートアッププログラム

こんにちは、こんにちは！ 少し前まで、Googleの新興企業向けプログラム-Google Developers Launchpadについての情報を共有していました。 次に、このプログラムの可能性と利点について詳しく説明します。







ランチパッドは、開始とスケールの2つのステージ（またはレベル）に分かれています。 Launchpadへの参加は無料で、プログラム自体に時間制限はありませんが、できるだけ早くスタートアップから収益性の高いキャンペーンに成長できることを願っています（誰かが1〜2年かかり、誰かが5年では十分ではありません。すべてはあなたが何をするかにかかっています）。



すでに述べたように、プログラム自体には2つの異なるレベルがあります。 違いは次のとおりです。



開始レベル

これが基本的な段階です。 新しく作成された製品またはそのプロトタイプに焦点を当てました。 たとえば、最初のアプリケーションを作成してGoogle Playで公開した場合、これは開始段階です。



スケールレベル

これは、すでに成功した製品またはアプリケーションをスケーリングする段階です。 ヘルプを取得するか、Scaleステージに移行するための重要な基準は、ユーザーベースです。 Google Playのアプリケーションについて話す場合は、50,000以上の地域のダウンロードに集中することをお勧めします。 現時点では、スケール段階で約30のロシアのスタートアップを支援しており、製品は次のいずれかになります。 Launchpadにサインアップして、Google PlayとGoogleで成長します。



スケールステージのスタートアップの利点



デザインレビュー\ UXアプリケーション（マテリアルデザイン）

このオプションは、ゲームではなくアプリケーションにより適しています。 アプリケーション設計は、材料設計コンセプトの原則に準拠しているかどうかチェックされます。 レビューのために、完成したアプリケーション（apkファイルまたはGoogle Playで公開されているアプリケーションへのリンク）とモックアップの両方を提供できます。 次のように、最初のオプションに集中することをお勧めします ほとんどの場合、モックアップはアニメーション要素を送信しないため、準備をしてください。トランジションの詳細な説明を求められる場合があります。そうでない場合は、迅速なレビューを行い、完成したアプリケーションを再送信する必要があります。 ガイドラインを読み、行われた推奨事項に従ってすぐに実行し、時間とリソースを節約します。



Google Cloud Platform（GCP）-10万ドルのリソース

最大100,000ドルのGoogle Cloud Platformクラウドリソースが提供されます。 これらは自由に使用できますが、GCPへのアクセスを提供するための主な要件の1つは、サーバーまたはクラウドリソースの集中的な使用です。 ちなみに、Google Cloud Platformリソースは、Google Playと他のストアの両方のアプリケーションのユニバーサルバックエンドとして使用できます。



たとえば、The AirとReactive Phoneの アプリは 、アプリケーションインフラストラクチャのリソースを既に受け取りました。



ブログのGCPに関する最新の関連記事： Google Cloud Platform Live：IaaSとPaaSの組み合わせ。 クラウドに対する ムーアの法則とGoogle Compute Engineに対するムーアの法則：同じお金でより強力に 。



Firebaseリソース

Firebaseは、モバイルアプリケーションのバックエンドに特化したサービスです。 仮想マシンを実行し、アプリケーションのサーバー機能（Apache、MySQL、MongoBDなど）にサービスを提供するために必要なコンポーネント/モジュールをインストールできるGCPとは異なり、Firebaseはサーバーインフラストラクチャの保守タスクを簡素化および抽象化します。 。 インフラ管理タスクに関与しなくなりました。







ところで、Android Mでは、オフラインモードのサポートは組み込みのFirebase機能にのみ関連しています。 そのため、Firebaseは強力です。 :) Android Mの技術革新については、Habréの投稿をご覧ください 。



UserTestingリソース（ユーザビリティテスト）

UserTestingは、 実際のユーザーがアプリケーションテストを実行するユーザビリティテストサービスで、ユーザーに発生したすべてのビデオレポートが提供されます。



プログラム参加者には、10回のフルテストと5回のミニテストを実施する無料の機会が与えられます。 通常、1つの完全なテストのコストは49ドルです。



Google I / Oおよびその他のGoogleイベント

Launchpadプログラム（スケールステージ）に参加しているスタートアップは、毎年開催されるGoogle I / O会議のチケットを入手できます。 一部のチケットは無料で提供され、一部のチケットは支払い用に予約されている場合があります。 たとえば、チームの1人のエンジニアが無料チケットを取得し、2人目のエンジニアまたはチームリーダー用に有料チケットを予約できます。 今年のロシアのスタートアップはすでにこのオプションを活用しており、Google I / O 2015を訪れました！







さらに、ロシアで開催される関連イベントにスタートアップを参加するよう招待します。 5月にGoogleのモスクワオフィスで開催されたこのようなイベントの1つに、Android TVのワークショップがありました。 セミナーでは、Creative Mobile、Game Insight、Ivi.ru、Nekki、Playmous、Nivalなどの企業が、Android TV用のアプリケーションの適応に関する経験を共有しました。



また、4月には、MetaBetaとSt. Petersburgの支援を受けてIngriaの支援を受けて、 モスクワで2つのLaunchpad Weekアクセラレータが開催されました 。



開発者サポートと1：Googleの専門家からの相談

開発者は、Googleテクノロジーの無料のテクニカルサポートを利用できます。 サポートは英語で提供され、電子メールで提供されます。 これは、AndroidおよびGoogle Play、Google Apps、Chromecast、Chrome Web Store、Google Glass、およびGoogle + / Google +サインインのテクノロジーに拡張されています。



さらに、技術的な問題だけでなく、ヘルプ情報も入手できます。 たとえば、現在、AdWordsを介してアプリケーションを宣伝するためのキャンペーンの個別相談（1対1）調査のベータプログラムを実施しています。 モスクワとダブリンのGoogleの専門家は、プロモーションキャンペーンの結果を詳細に把握し、その期間全体にわたって監視するのに役立ちます。



早期製品アクセス

Launchpadプログラムのすべての参加者は、Google製品への早期アクセスを得るための招待状を受け取ります。また、製品（Android Mなど）を試用するだけでなく、フィードバックを残す最初の人になる機会もあります。



地域のプログラムと取り組み

私たちは、スケール段階ですべてのスタートアップと緊密かつ積極的に交流し、また、さまざまな地域のイニシアチブに参加するよう招待します。 たとえば、そのようなイニシアチブの1つに、LaunchpadとAppTractorの協力者との一連のポッドキャストがあります 。



8月と9月には、Google Playのアプリケーションの興味深い加速プログラムも待ちます。 :)
