過去のゲーム

ハイテク時代では、リアルなグラフィックス、クールな特殊効果、リアルなサウンドなどを備えたゲームで誰も驚かないでしょう。 結局のところ、それほど前のことではないが、多くの人が目を輝かせて、Quake、Doom、DukeNukem、UFOに「切り刻んだ」。

今では多くの人が持っており、「開発者」を別々に取ると、ほぼ全員がPDAまたはコミュニケーターを持っています。 容量はかなり大きいことに注意してください。 私自身もかつてそのような実験が好きでしたが、今はモバイルデバイスでの前世紀の古き良きゲームについて少しお話したいと思います。 このすべては、週末の怠idleと古いエバラールの使用を多様化する試みによって再び引き起こされます。 はい、ハンドヘルド用に特別に「シャープ化」された古いゲームの使用可能な無料ポートについて説明します。

1.地震

私が出会った最初のゲーム（またはポート）はQuakeです。 最初のバージョン。 プレイするには、元のゲームファイル、具体的には朴、リソースファイルを取得する必要があります。 それら（pak0.pak、pak1.pak）は、古いゲームのあるサイトまたはゲームのディストリビューションからダウンロードできます。 Original Quakeを置いたディレクトリからファイルを取りました。 さらに、デバイス上のファイルをルート、つまりパス-/ Quake /、またはメモリカード、ルート-/ Storage Card / Quake /に書き換えます。 私はreadme.thtを読んでいませんでした。そのため、ゲームが開始されなかったが、奇妙なエラーでクラッシュしたときに問題が何であるかを理解するまで、長い間取り組んでいました。 ファイルをルートに投げた後、すぐに動作しました。 すべてのインストールで。 プレイアビリティについて。

基本的に、これらのゲームポートは、モデルが異なり、誰もが独自の特性を持っているため、指とボタンで遊ぶことができます。 コミュニケーターでは、設定後にメニューを終了できなかったため、横向きモードを設定できませんでした:)私はPDAからスクリーンショットを撮りました。 コントロールを習得するのに約10分かかりましたが、タッチスクリーンとボタンを同時に使用してウォーキングと撮影に慣れました。 グラフィック、サウンド、コントロールの設定が利用できるので、何か気になる場合は修正できます。 実際にはすべて-肉が続いた！



要約：

+古き良き地震、肉の音とモンスター

+優れた管理、ただし飛行機ではない

〜グラフィックは画面上で青白く見えますが、これは問題ではありません

2. DukeNukem

このゲームについて偶然知りました。友人（90倍）を訪ねたとき、彼はすでに「強力な」コンピュータ、スピーカー、およびその他の気まぐれを持っていました。 彼はDukeNukem3Dを見せてくれました。 私はすぐにそれが好きで、美しいグラフィックで、一般的に興味深いものでした。 落ち着き、撃たれ、別れた。 Xsがどれだけの時間を経て、私は再びそれに出会いましたが、ポートの形で。 インストール：元のキットから必要なリソースファイル（DUKE3D.GRP）は1つだけなので、インストールは簡単です。 これをゲームフォルダーにコピーして実行します。Quakeとは異なり、パスに依存するものはありません。 プロセスが開始されました...

ゲームはすぐに横向きに調整され、コントロールは再びわかりにくくなります。何をキャッチするまで、すべてのボタンをクリックする必要があります。 すべてのための5分間、そして戦闘に入ります。

デュークはクワキよりも速いが、それほどではないと言わなければならない。 したがって、鉄の残りの部分が飛んでいきます。 グラフィック、サウンド、制御設定が存在します。



要約：

+風景モード

+弱いデバイスでの良好な速度

〜管理との混乱が少ない


スクリーンショット：



美しいロゴ

3. DoomII

そして最後に-運命。 いいえ-Doooooom。 子供の頃から私のお気に入りのゲーム。 ここでの選択は素晴らしいと言わざるを得ません-携帯電話を含むさまざまなプラットフォームへのネットワーク上のポートがたくさんあります。 Windows Mobileの場合、4個掘ります。 ただし、最も便利/高速/バギーであることが判明した「qfDoom」は1つだけです。 ここでは、元のdoom2.wadファイルが必要です。 それはゲームとともにフォルダに入れられ、すべては不要なジェスチャーなしで簡単です。 このポートの特徴は、コンフィギュレーターの存在です。つまり、最初にコンフィギュレーターを起動し、目的のカードを選択し（最初のカードから開始できます）、HUD（キーボード、いくつかのオプション）と入力し、オプション（FPS、モンスターなし、レスポーン）を実行して、ゲーム自体を開始します。 私のコミュニケーター（195 MHz）でも、すべてが非常に高速に動作します。 モンスターの群集があるシーンを除いて、ブレーキは認識されませんでした。 グラフィックが美しく見える、私はすぐにIDDQD / IDKFAとの戦いを覚えています。 コントロールは便利で、指（スタイラス）を使用でき、ボタンを使用できます。 メニューには、ビデオ、サウンド、コントロールなどのすべての機能もあります。 すでにモンスターの粉砕を開始するには2分で十分でした。 今、私は時々輸送で遊ぶ:)



要約：

+多くの設定

+速度、弱い鉄片でも減速しない

+風景モード

〜指で動きを制御するときの小さなジャム。
