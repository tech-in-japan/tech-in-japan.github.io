最新のIT市場とApple Incの動向

明らかな傾向が現れたため、IT市場は大きく変化したようです（たとえば、1990年代には一般的ではありません）。

その1つは、大企業のオープンソースプロジェクトへの積極的な参加です。

そして2番目の明らかなことは、今日、大企業はもはや狭い市場領域に特化するのではなく、顧客のITシステム全体にサービスを提供する鉄+ソフトウェア+サービスの供給自体で包括的な顧客サービスに従事するよう努めていることです。



1. IBMは長年にわたり、企業に統合ITサービスを提供するIBM Global Servicesユニット（監査会社PricewaterhouseCoopersの以前のコンサルティングユニット）からの利益の大部分-参照： en.wikipedia.org/wiki/IBM

2.昨年、 Hewlett-Packard（HP）はElectronic Data Systems（EDS）を買収し、現在ではHP Company部門のEDSがIBM Global Servicesと積極的に競合しています-参照： en.wikipedia.org/wiki/HP

3.今年、 シスコはネットワーク機器+サーバー+ソフトウェアクライアントを複合施設に提供し、市場でユニファイドコンピューティングシステムモジュールを発売することで皆を驚かせました- 「Cisco Servers：Official Announcement」を参照

4.サンマイクロシステムズを購入するOracleソフトウェア会社がIron + Software + Service複合施設で顧客に提供し始めても驚くことではないようです。

おそらく、OracleはIBMやHPとの競争を恐れることはなく、それらに対処できるようになると判断しました。



残念なことに、Apple Inc.のような大規模なIT企業は、示された傾向に当てはまりません。 1980年代と90年代にIBMやHPと競争しようとしましたが、今日では企業市場で完全に非アクティブになっています。 Appleは企業市場の「重要な」シェアを占めていません-「大人の」IT企業の市場:(

（残念ながら、私はMac OS X Serverが最も成功したサーバーOSの1つであると個人的に考えているため）



今日、Appleはまだ標準のIntel x86サーバーをリリースしています。

しかし、Apple Xserve（x86）/ Mac OS X Serverは、大企業のSun（Sparc）/ SolarisまたはIBM（Power）/ AIXサーバーよりもDell（x86）/ Red Hat Linuxサーバーと競合する可能性が高くなります。

Intel x86サーバーのリリースには100社が関与しましたが、残念ながらAppleはこの市場セグメントのトップ10リーダーの1つでもありません。



また、シリコングラフィックスを購入したのはラックマウントシステムであったという事実（ 「シリコングラフィックス：伝説の歴史の最後の章」を参照）であり、AppleがAppleの企業市場にほとんど関心がないことを確認しているだけではありません（Apple Xserve RAIDの廃止後まで） 、AppleがApple Xserve自体のリリースを停止しても驚かない）:(



大規模なRISC / UNIXサーバーの市場の状況を振り返って追跡すると、Apple Xserve（Power / x86）/ Mac OS X Serverは、市場シェアの選択とシリコングラフィックス（MIPS / Itanium）/ IRIXプラットフォームの具体的な台無しに大きく貢献しています-サーバー。

そして、後者に関連して、正直に言うと、Silicon Graphicsを吸収すべきだったのはAppleでした。これは、プロのTV / Videoエディターの軍隊全体をApple Mac OS Xプラットフォームにドラッグすることにより、Appleの市場シェアを台無しにするために懸命に働きました。

そして、2008年の秋にスティーブ・ジョブズがAppleによって蓄積された200億ドル以上がすぐに使われることをほのめかしたので、AppleからSGIを買うのに十分なお金（ちょうど6億ドル）があるでしょう。

何だろう？



そして、結論はそれ自身を示唆しています：AppleはIT市場での競争相手であるIBMとHPからますます離れており、主に人気のある（ポップ）家電の市場で働いているSonyなどの消費者企業のようになりつつあります人々のために。 そして、将来的には、Appleは次の2つの方向でのみ開発を期待できます。

1.家庭用テレビ、音楽センター、台所用品の発売（Appleの利点は台所に非常に適した食用ブランドです）。

そして、この場合、Appleは将来、Sony社を買収することを避けられません（合計約500億ドル）。



2.そして、私の意見では、2番目の最良の選択肢は、その古典的なコンピューターハードウェア（Xserve、MacPro、MacBook）の段階的な放棄と、IBM PC（Intel x86）互換コンピューターの近い市場からAppleが発明した新しい市場への切り替えです。

+ iPhone互換デバイス（I-net指向）の市場、

+ソフトリリースに集中（I-netに方向付け）、

+もちろん、今日のファッショナブルなインターネットビジネス。

そしてこの場合、Appleは次のような企業の買収に蓄積したお金を費やします。

Adobe、Yahoo！、Amazon。



PS：心を奪わないでください:)

私（あなたかもしれませんが）がAppleの運命を心配し、心から繁栄を願っています:)

同時に、理解しようとしています。AppleがSun Microsystemsまたは少なくともSilicon Graphicsを購入しなかったのはなぜですか。なぜ彼らは企業市場（「大人」のIT企業の市場）でIBM、HP、Dellとの競争に参加したくなかったのですか？
