IPTVとは何ですか？



多くの人がすでに推測しているように、IPTVはIPテレビです。 しかし、IPTV自体の概念はより広範であり、はるかに多くのサービスが含まれていますが、すべてが順番に並んでいます:)



放送はどうですか？ マルチキャストを使用したTBチャネルのブロードキャスト。 次のトピックで既に説明したように、マルチキャストはIGMPプロトコルを使用したブロードキャストの特別な形式です。 通常、チャネル自体はUDPまたはRTPを介してブロードキャストされます。 平均して1つのチャネルは4〜5 Mb / sの帯域を占有します。 ここで最も興味深いのは、1つのチャネルが〜5 Mb / sの帯域を占有し、加入者が100のチャネルを持っている場合、500 MB / sの帯域が必要であるという誤った意見があります-これはそうではありません現在加入者。 1つのチャネルが現在探している場合、〜5 Mb / sがその下に割り当てられます。 したがって、プロバイダーは、ケーブルテレビや地上波テレビとは対照的に、理論的には無制限の数のチャンネルを放送できます。



IPTVコンプレックスには何が含まれていますか？





まず、IPTVにはミドルウェアと呼ばれるソフトウェアが含まれています。これは通常、コンテンツ管理システム、課金、統計、およびサブスクライバーポータルを含む分散ソフトウェアです。



2番目の、しかしそれほど重要ではないコンポーネントは、TVネットワークをIPネットワークにブロードキャストするタスクを実行するヘッドエンドです。 ヘッドステーションは、衛星、イーサー、またはケーブルから信号を受信し、必要に応じてIPパケットのmpegストリームをデコード、解読、カプセル化します。
