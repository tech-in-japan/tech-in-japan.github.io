40歳以上のときにコーダーに戻る方法

同僚やインタビューで何回言ったか。 どうやらHabréに需要があるようです。 誰かが選択に直面しました。 別の人は、意図したパスの正確さを疑います。 燃え尽きると、これが起こります。 誰かが今後何に直面するか、または自分自身を比較することに興味があります。 私を信じて、これは膨大な陰鬱な物語の短いバージョンです。







自己識別 ボルナ



プログラマーと呼ばれるのは大きすぎる。 率直に言って、私はエンコーダー、bydlokoder、またはgovnokoderです。 私は定期的にくだらないコードをくまなく調べ、すくい上げて山積みします。 スピードが必要、もっと安く、もっと。 さて、あなたはそれがどのように起こるか知っています。 食品会社でさえ、LET'S JUST WORKが必要です。 私はこのプロセスを楽しむことを学びましたが、結果は必ずしも幸せではありません。 ホームプロジェクトが役立ちます。







プログラマーになるのは、私が何歳のときに思いついたかは関係ありません。 90年代、私は非常に成功したセールスマネージャーとして働き、ほぼ5年間、夕方に喜びを実践しました。 証明書を受け取ったとき、私は27歳になりました-3か月の勤勉で600ドル。 なんて嬉しい！ これが私の技術であり、ビジネスであり、今では私のものです。







最初の情熱はDelphiで、長い間PHPにとどまっています。 あなたは美のために努力しましたか？ もちろん。 試した：C＃、Java、Python。 そして、給与のために働く必要はなくなりました。 突然、モスクワで火災が発生し、私はスモッグからボブルイスクに脱出し、無期限に残りました。 モスクワは他の都市のように人生のジュースを吸いません。 事前に先祖に行かなければならない場合に備えて、私は事前に「年金」をとることにしました。 時間の3分の1は古いプロジェクトのサポートに、残りの3分の1は有望なテクノロジーの研究に費やされ、残りはヨーロッパ中を旅行することに費やされました。 MeteorのCoffeeScript、Clojure、そして今やJS（ES6 + Node + React + Redux）に恋をしました。







卒業証書なしで仕事を得るのは難しいですか



本のスーツケースをドラッグしてインタビューを行い、知識を証明しました。 うーん 私は最近、アメリカでも12年の経験が卒業証書と同等であることを学びました。 ヨーロッパでは、少ないです。 ウェブサイトでアンケートに記入することは不可能です。雇用主の直接の連絡先を探しています。直接連絡します。 そして、それはさまざまな方法で起こります。







忍耐と方法論、そしてすべてが判明します。 「メロディーを推測する」スタイルの回答が必要な場合、私は通常、ジャークのように感じますが、演奏を拒否しません。 パブリックドメインには十分なコードの例があるように、驚いたことにテストタスクを見ていきます。







大ボスへの道は閉ざされていますが、ここでの野望は長い間尽きてきました。 まれな例外を除いて、私はチームで強調します-彼はとても孤独です、私はこの問題に関する複合体でにやにや笑います







「大きなスポーツに戻る」方法



これはトレーニングのようなものです。 毎日練習をやめると、スキルを失います。 ベラルーシ語には、ペラモガという美しい言葉があります。 古い習慣や思考パターンとの闘いを定義するのに最適です。 新しい知識の習得は難しく、情報の同化における年齢に関連した変化が明らかに役割を果たします。 よりアクセシブルなテクニックを使用します：「新しいことを学びました」という見出しの下にノートをペンで書き、 専門的な 記事を公​​開し、インターンとスペアリングを行います（説明すると、あなたは自分自身をよりよく理解します）、そしてもちろん、YouTubeは独学の新しい世界を開きました。 そうそう、仕事のように、休日のように！ 生きているプロジェクトは非常に貴重な実践です。 そして仲間の助け。 確かに、時々、あなたは狂った祖父のように感じます。







次は何ですか
