より小さく、より良い：Mozilla Firefoxは競合他社よりも速く実行するために重量を失います

Mozilla Foundationは、 Firefoxブラウザーの開発を中心にビジネスを構築し続け、他のプロジェクトを徐々に遮断しています。 さて、 Thunderbirdに別れを告げる時です。 これは、電子メール、ニュースグループ、およびカレンダーを操作するための無料のソフトウェアです。



「Thunderbirdは、Mozillaとそのテクノロジーから分離された後、より良くなると信じています」と、財団の社長であるMichelle Bakerが声明で述べています。



ミシェル・ベイカーは、サンダーバードの運命は完全には解決されていないと指摘しています。 別の会社の一部になることも、別のビジネスで際立ってオープンソースソフトウェアを作成することもできます。



開発者は、いかなる形式のThunderbirdもサポートしなくなるとTechCrunch は書いている 。 このソフトウェアの新しいバージョンの開発は2012年に中止されました。 その後、プロジェクトは3年間サポートされました。 しかし、この間、Thunderbirdはかなり古くなっています。



そして2004年に、この製品は単に市場を「爆発」させました。 起動後最初の10日間で、アプリケーションは100万ダウンロードを受け取りました。 もちろん、時間の経過とともに、ユーザーはモバイルまたはWebクライアントに切り替わり、Thunderbirdの人気は薄れていきました。



Mozillaは、バラストをドロップし、Firefoxを「若返らせ」、Google Chromeなどのブラウザとの競争において、より柔軟でモバイルなものにしようとしています。 これが成功した場合、Firefoxは更新時に関連製品またはそのインターフェースを変更する必要はありません。



同じ理由で、Mozilla は 10月上旬に、来年末までにNPAPI （Netscape Plugin Application Programming Interface）プラグインのサポートがMozilla Firefoxで廃止されることを発表しました。 このプラグインプログラミングインターフェイスでは、Adobe Flashのみが許可されます。これは、この技術が引き続き非常に人気があるためです。 Silverlight、Java、Unity、その他のFirefoxと同じプラグインは、将来サポートされなくなります。 NPAPIの代わりに、ネイティブWeb APIブラウザーインターフェイスを使用することを提案します。これにより、これまでNPAPIを介して行われていたすべてのことを実行できます。



11月の終わりに、Mozilla Foundationは2014年の財務レポートを公開しました 。 この文書から、MozillaはGoogleおよびYahooの検索エンジンとの契約から主な収入を得たことがわかります。会社の総収益に占めるこれらの支払いの割合は90％を超えています。 GoogleはFirefoxのデフォルトの検索エンジンであったため、このファンドは2012年に2億8,000万ドル、2013年に2億8,200万ドルを受け取り、Mozillaは2014年に検索大手との契約更新を拒否し、代わりにYahooと契約を結びました。 それにもかかわらず、Mozillaの総収益は4.9％増加して3億3000万ドルになりました。



昨年11月、MozillaはGoogleとの「単一の」契約を拒否し、より柔軟なポリシーを追求することを選択しました。 その瞬間から、米国のブラウザのデフォルトの検索エンジンはヤフーであり、ロシアではYandexがこの場所を占め、中国ではバイドゥがそうです。 同時に、ヨーロッパの検索大手との合意がないにも関わらず、GoogleはFirefoxのデフォルトの検索であり、支払いは一切ありません。
