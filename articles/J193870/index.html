StableBit DrivePool-重複ファイルのあるディスクアレイ

複数のハードドライブを1つのアレイにまとめるという観点から、RAIDの代替案を常に探しています。 私は多くのことを試してみましたが、何にも完全には止まりませんでした。 RAIDは、気まぐれで、柔軟性がなく、ディスクスペースを必要とするようなものではありません。







私は何が欲しいですか：

  1. データディスクを1つの大きなボリュームアレイにまとめて、ファイルを手動で並べ替える必要がないようにします。これは、複数のディスクで発生します（フォルダにマウントした場合でも）。
  2. アレイを崩壊させることなく、いつでもディスクを追加/削除/交換することができます。
  3. ディスク上の情報が互いに独立しているため、ディスクの1つが死亡しても、他のディスクがアクセス不能になることはありません。
  4. 理想的には、単一のファイルを複数のディスクに保存する何らかの方法。 バックアップは良好であり、私はそれらを行いますが、それらの助けに頼る必要がない場合はより良いです。






ドライブエクステンダー



Windows Home ServerのDrive Extenderは、理想に最も近くなりました。4つのポイントすべてが実装されました。 しかし、WHSを搭載したコンピューターは私と一緒に長く生きることができず、別のホストに移動し、通常のWindows Serverは新しいホームサーバーに落ち着きました（その後WWS2008、その後2008R2、現在は2012）。 はい、そしてDrive Extender自体も長生きしませんでした。MSはスムーズに動作するまで終了しませんでした-不完全なファイルを移動したり、メタデータを台無しにしたりすることもありました。





複合ディスク



通常のWindows Serverを離れた後、しばらくの間、スパンボリュームを使用しました。 最初の段落の実装、2番目の部分-ディスクを追加し（定期的に）、3番目の部分を大きく（非常に規則的ではない）置き換えることができます-1つのディスクが死んだ場合、残りの情報はほとんどすべてのプログラムによって削除されたファイルを回復することができます-深刻な断片化がなかった場合。

欠点-アレイからディスクを削除することも、ディスクを適切に交換することもできません。1つのディスクが死ぬことは、ファイルの回復に伴うシャーマニズムです。

アレイのディスクの1つが死亡した直後に拒否されました。 幸いなことに、彼はちょうど崩れ始め、突然死ぬことはありませんでした。私が情報を静かにコピーして複合ディスクを解析したからです。

Stablebitドライブプール





数か月前、2テラバイトから3テラバイトへの計画的な交換を行っていたとき、私はもう一度自問しました-新しいオプションがあったらどうでしょうか？ そして、はい、彼らがしたことがわかった。 StableBit DrivePool 。 能力の面では、これが実際に必要なものであり、4つのポイントすべてが満たされています。 プログラムを数回テストしてから購入しましたが、利益はわずか20ドルです。



従来のNTFSディスクはプールにまとめられ、システムとプログラムからは見えます。





プールファイルシステムは隠しフォルダーにあります。





ファイル自体は通常どおり保存され、ネイティブのストレージ形式はありません。 OSが突然クラッシュした場合、マザーボードが死んだ場合など、ディスクが生き残った場合、他のコンピューターに接続してデータにアクセスできます。





ディスクは安全に追加および削除でき、フォーマットは不要です。





アレイ内のディスクの数に応じて、複数のディスクにファイルのコピーを保存できます。 2つのディスク-2つのコピーを保存できます。 10個のディスク-10個のコピーを保存できます。





32ビット版と64ビット版の両方のVistaから、Windowsのすべてのバージョンがサポートされています。

StableBit DrivePool 2.Xは、さらに多くのオペレーティングシステムをサポートしています。

Windows 7（32ビット）、Windows 8（32ビット）、Windows 8.1（32ビット）、Windows Vista（32ビット）

Windows Server 2008（32ビット）

Windows Vista（64ビット）、Windows 7（64ビット）、Windows 8（64ビット）、Windows 8.1（64ビット）

Windows Server 2008（64ビット）、Windows Server 2008 R2（64ビット）、Windows Server 2012、Windows Server 2012 R2

Windows Home Server 2011、Windows Small Business Server 2011 Essentials、Windows Storage Server 2008 R2 Essentials *

Windows Server 2012 Essentials、Windows Server 2012 R2 Essentials *
