CSSを使用してSharepointサイトのテーマを作成する

テーマは、Sharepointサイトの外観を変更する最も簡単かつ最速ですが、柔軟性が最も低い方法です。 したがって、タスクがポータルをわずかに再描画する場合-テーマを使用してそれを行います。 最初のトピックを作成するために、Webサイトの構築やSharepointアーキテクチャに関する深い知識は必要ありません。 CSSの知識は十分です。





Sharepoint Webインターフェイスを使用して、完成したテーマを適用できます。 トピックが選択されたページは次のとおりです。





テーマは各Sharepointサイトに個別に適用され、これは管理者によって行われます。



私たちの人生の最初のテーマを作成しましょう。 SharePointサーバーで、C：\ Program Files \ Common Files \ Microsoft Shared \ web server extensions \ 12 \ TEMPLATE \ THEMESフォルダーに移動します。 すべての標準Sharepointテーマが含まれています。 たとえば、テーマが「シンプル」のフォルダをコピーします。 自由に名前を変更してください。 たとえば、「HABR」。







「HABR」フォルダーで、すべてのイメージを削除し、SIMPLE.INFファイルの名前をHABR.INFに変更します。 HABR.INFを開き、どこでも「Simple」を「HABR」に置き換えます。



「C：\ Program Files \ Common Files \ Microsoft Shared \ web server extensions \ 12 \ TEMPLATE \ LAYOUTS \ 1033 \」フォルダに移動し、SPTHEMES.XMLを編集して選択したテキストを追加します。







まあ、それだけです。 テーマを作成しました。 これで、CSSを操作するためのInternet Explorer Developer Toolbarまたは他のブラウザーアドオンのようなものを装備できました。 ところで、IE8には、必要な機能を備えた開発者向けの組み込みコンソールがあります。



そこで、ロゴを変更し、サイトアクションのドロップダウンメニューの色を変更しましょう。







theme.cssに以下を書いてみましょう。



#siteactiontd div、.ms-siteactionsmenu div {

背景画像：なし！重要;

背景色：＃86C5EA！重要;

フォントファミリ：Arial！重要;

フォントサイズ：12px;

}



td.ms-sitetitle {

幅：240px;

高さ：130px;

背景：0 5px no-repeat url（habr_logo.gif）;

}



h1.ms-sitetitle、td.ms-titleimagearea {

ディスプレイ：なし;

}



使用してください！オーバーライドに関しては重要です。



[サイトの操作] / [サイトの設定] / [サイトのテーマ]ページに移動して、テーマをサイトに適用します。
