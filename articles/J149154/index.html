Runpadシェル保護バイパス

良い一日！ Runpad Shellの保護をバイパスして、タスクマネージャーの制御にアクセスしてみましょう。MicrosoftOffice製品ラインに組み込まれたVBAマクロは、この犯罪の助けになります。 それでは、私たちが何を扱っているかを詳しく見ていきましょう。



Runpad Shell-ジョブを制限し、コンピューターセンター、クラブ、インターネットカフェ、図書館、学校、その他の組織で便利なユーザー作業を整理するためのソフトウェアパッケージです。 システムの実装により、不要なユーザーアクションからジョブを保護できます。

Game Classと一緒にインストールされ、柔軟な料金システム（インターネットの有無にかかわらず、毎時およびパケット、トラフィックの有無など、ほぼすべての料金プランを設定できる機能）、データセキュリティ、人事管理、財務および統計レポート、トラフィックカウント（トラフィックの支払いを伴う関税を入力できます）。

VBA- （Visual Basic for Applications、Visual Basic for Applications）-Visual Basicプログラミング言語の実装をわずかに簡略化し、WordBasicなどの以前に使用された特殊なマクロ言語の機能をカバーおよび拡張します。



Runpad Shellは、コンピュータークラブ、インターネットカフェなどにインストールされます。 Microsoft Office WordおよびExcel製品が順番に使用される施設。 ほとんどのマクロは、Office Excelエディターでの実装用に記述されています。プロセスを表示、一時停止、および終了できるタスクマネージャーも含まれます。

このような機能を使用すると、ゲームクラス（gccl.exeおよびgcclsrv.exe）をオフにすることができ、インターネットクラブで無限の時間を過ごすことができます。また、Runpad Shell（rshell.exe）自体をオフにして、コンピューターへのフルアクセスを許可することもできます。



ゲームクラスまたはRunpadシェルプロセスのシャットダウンはサーバーに表示されます。ゲームクラスプログラムには時間追跡がありません。コンピューターがフリーであることが示されます。Runpadシェルプログラムでは、シェルのシャットダウンを示す赤いアイコンが表示されます。



Steamがコンピュータークラブにインストールされている場合、上記にもう1つの抜け穴を追加できます。Steamを介してサードパーティプログラムを起動すると、アプリケーションを起動するためのフルアクセスが開きます。

例：フラッシュドライブを貼り付け（「ファイルマネージャー」があります。その後、 Steamで 「 Steamに関係のないゲームを追加」を選択し、「ファイルマネージャー」の.exeへのパスを指定します。「ファイルマネージャー」を開くか、 「レジストリエディタ」、さらにはポータブルな「タスクマネージャ」。
