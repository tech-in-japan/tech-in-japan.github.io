アップル木曜日

2月3日木曜日、ポクロフスキー大通りにあるHSEビルのサウロンタワーで、モスクワでiOのゲーム開発者向けの次の月例セミナーが開催されました。 AlexGoncharenkoの宴会を導いた 。 彼の質問-イベントについてどうやって知りましたか。300人中298人が回答を受け取りました-Habrahabrの発表から。



聴衆は友好的で、知性的で、教育を受けていました。それは私を驚かせ、人類の明るい未来に対する私の信仰を復活させました。 ほとんどすべてのリスナーは、右手にiPad、左手にiPhone 、ひざにmacAirの女の子がいました。



合計4件の報告が寄せられました。iPhone用のアプリケーションを開発し、appStoreからお金を受け取るための4つのまったく異なるアプローチです。







最初の報告


18-45でショーが始まりました。 聴衆は、最初の2人のスピーカー-2010年のヒット作のクリエーターである伝説のZeptolab兄弟-Cut the Ropeに熱心に挨拶しました。 アプリケーションとは異なり、レポートは最も素晴らしいものではありませんでしたが、彼らが正しい人であることはすぐに明らかになりますが、彼らは笑顔で、最初の行為で観客にロープでトリックを見せました。

Cut the Ropeは2番目の子であり、appStoreの最初の子はPARASUTICと呼ばれ、300,000の有料ダウンロードと獲得戦略の理解をもたらしました。 彼らは、ディズニーが漫画を作成したのと同じ原則に基づいて、カットザロープを作成しました。 これは、優れたアニメーション、プロのサウンドエンジニア、重いスレッド移動モデルのアルゴリズム（開発に最も時間がかかった）、退屈なスクリプト、テスターの友人のチーム、および適切で厚いパブリッシャーの選択です（アプリケーションとパブリッシャーのターゲットオーディエンスは一致しました）。





連中はこの出版社を出版社と呼んでいたが、私の考えでは、アブラモビッチのような姓であると決めた。 私は間違っていましたが、ゲームのプロモーションの大部分を行ったのはパブリッシャーでした。 このアプリは500万回販売されています。 見事に。 出版社はいくらお金を取りましたか？ 不明ですが、報告書の直後、空港ではなく駅に人が立ち去りました。これは憂慮すべきことであり、出版社を選ぶための慎重なアプローチを示唆しています。

兄弟は事件の聴衆からの質問に答え、鼻を上げず、運命と運に感謝した。 私はそれを理解しているので、今彼らは彼らのゲームのキャラクターでおもちゃ、Tシャツなどのリリースで少しバボシックを減らしたいと思っています。 彼らと彼らのヒキガエルがソチ2014エンブレムコンテストに参加しているかどうかを尋ねられたとき、弟は笑い、拍手を送り、年上の弟は思慮深くなりました。



第二報


2番目のレポートは、Game Insightの2人の魅力的な女の子によって表明されました。 同社は、Islandなどのオンラインゲームを所有しており、何百万人もの外国人がプレイしています。 これらの市民は、実際のお金のために仮想の楽しみのためにあらゆる種類のグッズを購入します。 Game Insightは、特定のプログラマーにiPhoneバージョンのゲームを注文し、ストアに入れ、ユーザーからappStoreから60万ダウンロードを受け取り、このイベントから利益を得ていると主張しています。 そして、わずか2か月が経過しました。 正常な結果ですか？





彼らの毎月のゲーム収入は100,000ドル以上です。 女の子によると、ロシア国民はゲームをプレイせず、お金を払っていない。 レポートの質は息をのむほどでした。 HOUSE-2の英雄のスラングは、300 人のSpartansの iPhone開発者の良い隣人に迎えられました。 女の子は多くの質問をされましたが、最も重要なことは尋ねられませんでした-彼らは今夜何をしていますか。



第三報


3番目のレポートはFlexisのIlya Kurylevによって読まれました。 彼は、iPhone Game Centerについて、その利点と多数の欠点について、そして彼ら自身がiPhone用の単一のゲームアプリケーションを作成したわけではないことを、明確かつ知的に語りました。





残念ながら、聴衆はスピーカーに単一の質問をせず、拍手だけで感謝しました。 そして、iDeviceの92％がiO 4.0以降に搭載されている数値をどこで入手したかをIlyaに尋ねます。 私のアプリケーションによって収集されたデータは、50％未満の数値を示しています。 この観点から、今日ではゲームアプリケーションにGame Centerを使用することはほとんど意味がありません。 iOSの第4世代では、サーバー側に多くの穴があり、ロシアのデバイスはほとんどありません。



第4報


最後のスピーカー、私は引用します-Vadim Bashurov、iPhoneアプリケーションの独立した開発者、伝説的な「奇跡のフィールド」の作成者。 これは私だからです。 さらに、私の最初のゲームは「奇跡のフィールド」ではなく、ウクライナとロシアの「海戦」でした。

レポートに関する質問は読み書き可能であり、立派な種類の未知の仲間だったので、iAdライブラリをアプリケーションに追加する際のエラーに目を向けました。

レポートの最も客観的なCコードでは不十分でした。おそらく、会議の形式と画面にはこれがありませんでした。

著者のパフォーマンスの過程で、お金のコンテストが行​​われたことを追加したいと思います。どのアーティストの絵が私のプレゼンテーションの背景として使用されますか？ アレクセイ・ゴンチャレンコは100ルーブルを獲得し、ちょっとしたヒントでマスターを推測しました。 答えを知っていますか？





一般に、この会議は私にとって非常に興味深く、すでに急いで実装しようとしていたアイデアに影響を与えました。



コンテストについて






セミナー中に、参加者はアンケートに記入し、その中で賞を受賞しました-書籍DEVELOPMENT for iPhone。 本は賢い目でハンサムな無精ひげを生やした男によって勝ち取られました。 彼を見て、私は彼が読むべきではなく、自分でそのような本を出版すべきだと思った。 これは、Appleとその魔法のデバイスの助けを借りてお金を稼ぐための4番目の、最も正しいアプローチです。
