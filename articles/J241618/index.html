包括的なスニペットをまとめる25の方法





スニペットは、サイトへのリンクとともに発行ページに表示されるテキストの小さな断片です。 以前は、スニペットにはテキストのみが含まれていたため、1ページで見た10個のアドレスはすべてほぼ同じに見えました。 検索エンジンは、この問題を解決する方法について最初に考えました。つまり、ユーザーに必要なリソースについてできるだけ多くの情報を表示し、必要な場所にすぐに行くようにしました。 デジタルマーケティングの場合、スニペットはユーザーの満足を得るための重要なツールです。 高品質で包括的なスニペットをすばやく作成する方法は？ ごく最近、 PlayMediaデジタルエージェンシーは詳細なインフォグラフィック「Googleで高度なスニペットを使用する10の方法」を公開しました。 今日の投稿でこの知識を実践する方法をお読みください。







今すぐスニペットを使用する必要がありますか？



まず、現在関連性のある傾向に目を向けて、スニペットを一般的に使用することの関連性を判断しましょう。ゲームは価値がありますか？ 現在、スニペットを使用することはどの程度関連性がありますか？ Googleトレンドにアクセスして、スニペットリクエストの人気のグラフを確認します。





出典： http : //www.google.com/trends/



ここでは、ロシアでは2010年から今日まで成長のダイナミクスを追跡できることがわかります。 次のステップは、これが検索エンジンプロモーションとサイトプロモーションのフレームワークにどの程度関連するかをおおよそ「推定」することです。 これを行うために、リクエストの人気のダイナミクスのグラフ「ウェブサイトプロモーション」に注目しましょう。 これらのリクエストの人気を比較することで、スニペットの高度な管理に対する需要の高まりのプロセスを追跡できます。





出典： http : //www.google.com/trends/



ここでは、スニペットの人気がある程度増加しているにもかかわらず、検索ランキングでのウェブサイトのプロモーションの問題における主要なトレンドの1つではないことがわかります。 リクエスト「トラフィック」、「外部リンクのボリューム」、「位置」を「いじって」、主な注意がもちろんそれらに集中していることを確認できます。



「ウェブサイトプロモーション」リクエストの人気に大きな変動があり、このグラフの形成には検索エンジン自体の人気の変動が含まれていないという事実を考慮しても、2010年から現在まで、スニペットの正しい形成はプロモーションの不可欠な部分ですサイト！



これは今私たちに何を与えますか？ スニペットをできる限りユーザーフレンドリーにしたので、トップに位置していない場合でも、他のプロモーションスペシャリストがまだスニペットを手に入れていない場合でも、サイトに取り込むことができます。

方法2：スニペットテキストに影響を与える





Yandexスニペットテキスト



ページの説明を生成するために、Yandexはリクエストの単語を含むドキュメントのテキストのすべてのフラグメントを選択します。 これらの各フラグメントは、いくつかの部分に分割されます-たとえば、クエリの先頭、末尾、および中央にある単語。 次に、プログラムはそれらすべてを互いに比較し、最適なものを選択します-それらはスニペットに分類されます。



...単語が長い文に含まれている場合、ナビゲーションリンクではなく、物語の一部である可能性が高くなります。 これは良いスニペットスニペットです...
