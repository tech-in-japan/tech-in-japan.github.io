自宅のエアホッケー

良い一日、Khabravchany :)



今日、私のお気に入りのバーから再び来て、私はエアホッケーをプレイするためのホームテーブルを作るというアイデアを得ました。 インターネットをさまよった後、私はそのテーブルを作るための非常に良い指示に出くわしました。 この記事の翻訳をお見せしたいと思います。



カットの下にたくさんのテキストと写真。





そこで、YouTubeで覗かれたミニチュアテーブルを作成するというアイデアに触発され、エアロホッケーをプレイするためのフルサイズ（122x244cm）のテーブルを作成することにしました。 このマニュアルでは、このテーブルを製造するすべての段階について説明します（「脚」なし）。



ここでは、エアホッケー用のテーブルの3Dモデルがあります。 プロジェクトのすべての複雑さを理解するのに役立つことを願っています。 図面を表示するには、無料のGoogle Sketchup Viewerアプリケーションが必要です。



ゲームの「コウモリ」を作るための指示を見ることを忘れないでください。



この情報があなたの役に立つことを望み、今すぐ構築を始めてください:)



アクション1



必要なコンポーネント（ミリメートル単位のすべての測定）。

D-long、W-width、H-height。



スプルースからフレームを作成するには、次のものが必要です。

-2つの長辺：D = 2530、W = 45、H = 70（1番）

-2つの短辺：D = 1310、W = 45、H = 70（2番）

-2つの長辺：D = 2440、W = 20、H = 35（3番）

-2つの短辺：D = 1220、W = 20、H = 35（4番）

費用：約37ドル



プレイングサーフェス（MDF）の製造：

D = 2440、W = 1220、H = 6（5番）

費用：約15ドル



コースター：

高さ35 mmの小さな木製チョッパー（6号）

費用：約11ドル



反対側の表面（チップボード）：

D = 2440、W = 1220、H = 12（No. 7）

費用：約15ドル



空気源：



遠心ファンまたは掃除機（No. 8）

テーブルに必要な容量は、毎分約400立方フィートです。 平均的な家庭用掃除機の容量は約140立方フィート/分です。 洗濯機がテーブル上を自由に移動できるように、少なくとも3つの掃除機が必要です。 このような空気源は、家庭での使用には比較的高価です。 より強力な産業用掃除機が存在しますが、入手が困難です。

遠心送風機は高度に専門化されています。 出力電力は、数百から1000立方フィート/分です。 以上。 そのようなスーパーチャージャーは、エアホッケーのテーブルで一般的に使用されています。 遠心過給機も非常に高価です。



吸気口：

-直径約90mmのパイプ1つか2メートル。

-パイプが収まる穴のあるPVCシート250mm x 250mm。 （No. 9）

[チューブが穴に直接収まらない場合は、PVCスリーブ（チューブと穴の周りに収まる）を購入してください]

-2枚の合板250mm x 250 mm（No. 10および11）



ゲート：

-約400mmの長さの木製ビーム

-丈夫な生地400mm x 400mm







2番目のアクション。



フレームの製造。

A：両端に45度の切れ目がある8つの異なる部品を見た必要があります。

B：床上のすべてのコンポーネントを折り畳み、切断の精度を確認します。

C：釘と建設用（ポリウレタン）接着剤を使用して、4枚の短いボード（No. 3および4）を対応する大き​​なボード（No. 1および2）に取り付けます。

D：4辺すべてを接着し、ベースの周りにベルトを巻き付けて締めます。 チップボードはフレームの中央になければなりません。 少なくとも4時間、接着剤を乾燥させるために構造体を放置します。

E：角を斜めに開け、ボルトで締めます。



3番目のアクション。



穴を開ける/プレイ面を準備する。

F：穴を開ける前に、MDFにメッシュを作成する必要があります。 25mmの正方形のグリッドを描く必要があります。

G：作成したグリッドに従って穴を開けます。 直径1 mm（または1/32インチ）のドリルを使用します。 この作業には時間がかかりますが、急がないでください。



合計で、96x48 = 4608の穴を開ける必要があります。



N：グラインダーを使用して、シートの反対側のバリを取り除きます。 こちら側がプレイ面になります。





4番目のアクション。



表面に柱を置きます。

I：高さ35mmの木製ポストが約150-200個必要です。 あなたがテーブルを壊すことなくテーブルの表面に立つことができるなら、あなたは正しい道にいるはずです:)



J：テーブルの端（75 mmまたは3インチ）から3つの「正方形」にポストを配置し、互いに4つの正方形で分離しました。 合計で、9x19-2 = 169列になりました。 しかし、この回路を自分で試すことができます。



K：通常の白い木材の接着剤を使用して、MDFの滑らかでない側面に支柱を接着します。 カウンタートップは互いに対して移動する可能性があるため、ポリウレタン接着剤を使用しないことをお勧めします。 接着剤を硬化させる時間を与えます。





5番目のアクション。



プレイ面をフレームに取り付けます。

L：通気孔を布で覆います。 これにより、接着剤がダクトに入るのを防ぎます。

M：接着剤で表面の端にグリースを塗り、慎重にフレームにスライドさせます。

N：最後に、MDFのポイント3と4で短辺に5本の釘と長辺に10本の釘を打ちます。



アクション6。

ダクト取り付け。

葉をきれいにするためのファンをテーブルのチューブに直接接続するのではなく、テーブルに取り付けることにしました。 過度の振動は、テーブルに直接取り付けた場合にファンを落下させる可能性があるためです。



パーティクルボードシートは安価ですが、欠点が1つあります。十分な強度がありません。 ダクトとテーブルとの接合部を強化するには、力を分散する必要があります。 そのため、大きな面積をカバーするPVCの大きなシート（穴のあいた）が必要になります。 パイプカップリングとPVCシートをしっかりと接続する必要があります。



A：スリーブを青いダクトパイプに接着します。 チューブのもう一方の端をファンの出口に接続します。

P：ドリルで丸鋸ノズルを使用して（下の画像を参照）、2層の合板に穴を開けます。 このシートは合板の間に挟まれるため、穴はPVCシートよりも大きくする必要があります。 ここで、空気入口を設置する場所を決定します（底板の隅に配置することをお勧めします）。

Q：図に従って2層のベニヤ板をドリルします。1枚のベニヤ板には各コーナーに4つの穴があり、2番目のベニヤ板には均等に分布した8つの穴があります。 同じことがPVCシートにも当てはまります。

R：合板の層（4つの穴）をテーブルの穴に接着します。 次に、PVCシートと2枚目の合板を取り付けます。 すべての8ボルトをねじ込みます。







7番目のアクション。



プレイ面をフレームに取り付ける

S：接着剤耐性の布を取り外し、すべての穴がきれいであることを確認します。

T：各ポストとフレームの境界線に接着剤を塗布します。 チップボードをゆっくりと下ろし、フレームにスライドさせます。 重いものでテーブルを押し下げ、接着剤を乾燥させます。





8番目のアクション。 ファイナル



ゲートを作り、仕上げます。

U：テーブルの短辺の中心から15 cmの位置に2つの穴を開けて、プレー面が損なわれないようにします。

V：ジグソーを使用して、これらの穴の間の不要な部分を切り取ります。 サンドペーパーは、ゲートの端にきちんとした外観を与えます。

X：カットの上下に2つの40cmの長さの梁を、それらの間に強力な布地で留めます。 これを行うことで、側面に穴のあるターゲットの周りに一種のポケットを作成し、ポケットからパックを簡単に取り外すことができます。





テーブルの反対側でこのプロセスを繰り返します。



必要に応じて：

-アルミニウムの側面を追加して、衝撃でパックが減速するのを防ぎます。

-フィールドにマークアップを追加する





すべてを正しく行った場合、ファンをオンにしてゲームを開始してください！





楽しんでください:)
