Project Spartanはもうすぐここにあります





Windows 10の予備バージョンの最新の更新により、将来のProject Spartanブラウザーモードでページを表示するメカニズムを有効にすることが可能になり、システム統合を備えたインタラクティブなサイトを作成することも可能になります。





新しいモードでページ表示を有効にする方法



ビルド10041がインストールされていることを確認します。



1. Internet Explorerを起動します

2.アドレスバーにabout：フラグを入力します

3. [ 実験的なWebプラットフォーム機能を有効にする]を選択します

4.ページの下部にある適用ボタンをクリックして、ブラウザを再起動します





Windows 10.通常のIE（トライデント）モードを使用





Windows 10. IEでProject Spartanモードを使用する（EdgeHTML）



ホステッドWebアプリ



Hosted Web Appsは、WindowsシステムAPIを使用して、通常のWebサイトを本格的なプログラムに変える機会です。 このようなサイトをJavaScriptで記述する必要があります。



iOSまたはWeb開発者の一部の開発者は、おそらくiOSモバイルデバイスでホーム画面に追加ボタン機能をサポートするサイトを作成している可能性があります。Webサイトを修正でき、本格的なアプリケーションとして動作します。 一部のシステムAPIを使用して、ファイルをキャッシュし、WebSQL / IndexedDBデバイスのデータベースにアクセスできます。 iOS 8のIndexedDBは現在完全には機能していません 。







Windows 10に戻ると、そのようなサイトをストアに追加できます。 サイトは、連絡先、カレンダー、通知などにアクセスできます。 C / C＃アプリケーションを作成する必要はありません。必要な機能を追加して、Webサイトをアプリケーションに変えるだけです。
