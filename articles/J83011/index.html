Ubuntu 9.10ブートメニューで背景画像を設定する

ご存知のように、9.10では、Grubブートローダーの2番目のバージョンが使用されます。これは、より機能的で最新のものです。 Ubuntuでは、他のユーザーフレンドリーなディストリビューション（openSUSE、Mandriva、Fedoraなど）とは異なり、Grubメニューに背景画像がありませんでした。 9.10の新しいブートローダーでは、これは変更されていません。 しかし、以前の写真のインストールが決して些細な作業ではなかった場合、Grub 2ではこれを行うのが簡単です。

1）少なくとも1024×768サイズの好きな写真を撮影し、GIMPエディターで開きます（システムにプリインストールされているため、検討します）。 画像のサイズを1024×768に縮小します：画像-画像の拡大縮小-1024を入力します（768でない場合、幅は自動的に塗りつぶされます。ペーパークリップのようなアイコンをクリックして、手動で入力します）-拡大縮小します。 これをホームディレクトリにmoreblue-orbit-grub.tga （この名前、TGA形式）として保存します。

2）ホームディレクトリからGrubファイルのあるディレクトリに転送します。

sudo mv moreblue-orbit-grub.tga / boot / grub （ターミナルに入力します）。

3）Grubユーザー構成ファイル（ / etc / default / grub 、スーパーユーザー特権）を編集し、パラメーターGRUB_GFXMODE = 1024×768を設定します。

4） grub構成更新スクリプトを実行します： sudo update-grub2

これで、OSを選択する場合（1つである場合、上記のすべての操作を実行しても意味がありません）、美しくて魅力的な背景画像が表示されます。
