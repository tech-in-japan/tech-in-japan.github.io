PyDev 1.5またはPyDev Extensionsリリースがオープンソースになりました

Pythonでプロジェクトを開発できるEclipseプラグインであるPyDevの新しいバージョンがリリースされました。 新しいバージョンでは、開発者Fabio Zadroznyは、有料のアドオンPyDev Extensionsのすべての機能を無料のオープンバージョンに統合することを決定しました。



新しいバージョンでは：

  • 自動インポートを備えた高度なインタラクティブコンソール、
  • リアルタイムのコード品質分析
  • 自動インポートによるコンテキスト非依存のコード補完、
  • 階層クラス図、
  • リモートデバッグサーバーを実行する機能を備えたデバッグコンソール。


詳細情報は、 PyDev.orgの公式ウェブサイトとFabio Zadroznyブログで見つけることができます。







ダウンロードする


作成者は、インストールする前に、有料のアドオンPyDev拡張機能（インストールされている場合）を削除することをお勧めします。 Eclipse Update Managerのリンク：

  • 安定バージョン： http : //pydev.org/updates 、
  • ナイトリービルド： http : //nightly.aptana.com/pydev 奇妙なことですが、ここにあるのは古いバージョン1.4.8だけです。


zipアーカイブは、Source Forgeのプロジェクトページからダウンロードできます。
