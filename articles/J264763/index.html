Windows 10がユーザーデータを収集する方法





Windows 10がユーザーデータを収集する情報は新しいものではありません。 2014年、マイクロソフトはプライバシーに関する声明を発表しました。これは、使用するプログラム、デバイス、およびそれらが機能するネットワークに関する情報をサーバーに送信できることを意味します。 このデータは、ユーザーID（Microsoftアカウント）、電子メールアドレス、設定、場所の関心などに関する情報と組み合わせることもできます。
