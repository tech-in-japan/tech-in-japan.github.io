AgnitumがOutpost Security Suite Freeをリリース

2010年11月22日に、ロシアの企業Agnitumは、Windowsでのパーソナルコンピューターを保護するための包括的なソリューションの無料版-Outpost Security Suite Freeをリリースしました。 このパッケージには、ウイルス対策、ファイアウォール、プロアクティブなコンポーネント（データ漏洩からシステムを保護する）、およびスパム対策が含まれています。Download.comで人気のあるインターネットセキュリティクラスソリューションであるComodoとは異なりまた、ファイアウォールと予防的な防御の存在。 Outpost Freeはfree.agnitum.com/download.phpからダウンロードできます。



無料版と有料版の違いは非常に重要です。 いずれかの形式の無料版には、ファイアウォール、電子メール保護、予防的保護、ウイルス対策スキャナー、コンテンツフィルタリングなどのすべてのモジュールが含まれています。 同時に、これらのモジュールのほとんどすべての作業は非常に限られています。 特に、無料版とは対照的に、有料版では



-スケジュールスキャンを実行できます。

-ファイアウォールは攻撃を検出でき、

-広告、特定のドメイン名、IPアドレスをブロックすることが可能です。

-追加のプロアクティブな防御コンポーネントが機能し、

-ロシア語のインターフェースがあります。



Agnitumのコマーシャルディレクターによると、Outpostウイルス対策ソリューションは現在、ロシア市場で約5％を占めています。
