エゴール・クリコフ-プログラミングの世界チャンピオン

同僚のEgor Kulikovが、プログラマーの間で最も権威のある国際的な個人チャンピオンシップ-TopCoder Open 2012の優勝者になりました！



Egorは、モバイル製品開発部門のYandexのサンクトペテルブルクオフィスで働いています。 過去に、Yegorは数多くの学校数学オリンピックの優勝者であり、学生プログラミング世界選手権の銅メダリストであり、2010年には別の主要な大会であるGoogle Code Jamで優勝しました。



したがって、Yegorは、Peter Mitrichevに次いで2番目のロシア人となり、TopCoder OpenとGoogle Code Jamの両方を獲得しました。



TopCoder Openは、2003年以来開催されているプログラミングの世界選手権です。 今年彼はオーランドにいました。 競争の最終に到達するために、あなたは難しいマルチステージ選択を通過しなければなりません。 最初の段階はオンラインで開催されます。世界中の何千人もの参加者が、複雑なアルゴリズムの問​​題を解決する能力を実証しています。 24名の受賞者がフルタイムツアーに招待されます。 しかし、選考はこれで終わりではありません。応募者はそれぞれ12人の準決勝に2つに分けられ、その結果、最高のベストがファイナルにたどり着きます-わずか8人です。 この8つに入ることは、すでに大きな成果です。



フルタイムツアーの参加者全員に賞金が贈られ、今年の1位の賞金は15,000ドルになりました。



TopCoder Openですべてのラウンドを実施するためのシステムはほぼ同じです。参加者には、条件付きでシンプル、ミディアム、難しいの3つのタスクが与えられ、それらを解決するのに75分かかります。 参加者が問題を迅速に解決すればするほど、より多くのポイントを獲得できます。 結果の発表の前に、参加者が互いの決定を見て、それらのエラーを見つけることができる別の15分があります。 発見された各テストで、相手のプログラムが正しく動作しない場合、ファインダーは追加のポイントを受け取ります。



このレベルの大会への参加と勝利は、喜びとお金をもたらすだけでなく、多くを学び、素晴らしいキャリアの見通しを与えることを可能にします：Yegor自身がYandexのモバイルの方向性を動かす手助けをします。 Yandex.Directの広告技術、およびその他のPavel Nalyvaikoは現在チューリッヒに住んでおり、Googleでの検索の品質に取り組んでいます。



勝利を収めたイゴールの皆さん、おめでとうございます！
