LinuxでのFlexアプリケーション開発（Ubuntu 8.10）

私の経験を共有したいです。 WindowsでUbuntuに切り替え、Eclipseとさまざまなプラグインをインストールしました。 以前はWindowsでFlex Builderを使用していましたが、FlexSDKとFlex Builder for Linuxをインストールするという質問がありました。 実際に、Linux用のMXMLおよびAirアプリケーションを開発するためのEclipse + FlexSDK + FlexBuilderLinuxの構成方法について説明します...





続行：



ステップ1： Sun JREのバージョンが1.5以上であることを確認する必要があります（これは、「java -version」コマンドを使用して実行できます）。



ステップ2：

$ HOME / EclipseでEclipse（cppまたはjava）<3.4を設定します

-バージョン3.3.x（http://archive.eclipse.org/eclipse/downloads/index.php）をダウンロードします。 私はそれが3.3.xシリーズであることに注意したい、なぜなら FlexBuilderはバージョン3.4では動作しません。

-cd〜

-tar zxvf ./eclipse-cpp-ganymede-linux-gtk.tar.gz

-私たちは日食を抽出します〜/日食自体



ステップ3：

Flex Builder alpha 4 Eclipseプラグインをインストールする

-labs.adobe.com/downloads/flexbuilder_linux.htmlで Flex Builderインストーラーをダウンロードします

-chmod + x〜/ flexbuilder_linux_install_a4_081408.bin

-〜/ flexbuilder_linux_install_a4_081408.bin

-「既存のEclipseフォルダー」として$ HOME / eclipseを選択します。







-「注意して続行」を選択します（Eclipse3.4をインストールした場合はポップアップします）：







ステップ4：

Flex SDK 3.xをインストールします

-opensource.adobe.com/wiki/display/flexsdk/Download+Flex+3からFlex SDK 3.xをダウンロードします

-最新リリースを選択します（3.2.0.3958があります）。

-〜/ Adob​​e_Flex_Builder_Linux / sdksの新しい3.xフォルダーに解凍します。

-cd〜/ Adob​​e_Flex_Builder_Linux / sdks

-mkdir 3.x

-cd 3.x

-unzip〜/ flex_sdk_3.zip



ステップ5：

Flex SDK 3.x用のAIRベータSDKをインストールします

-www.adobe.com/products/air/tools/sdkからAdobe AIR SDKをダウンロードします

-新しく作成したフォルダーに解凍します

-cd〜/ Adob​​e_Flex_Builder_Linux / sdks / 3.x

-tar jxvf〜/ air_1.5_sdk.tbz2



ステップ6：

新しいデフォルトSDKをインストールする

-実行日食

-〜/ eclipse / eclipse

-ウィンドウ->設定-> Flex->インストール済みFlex SDK

-追加

-SDKの場所として〜/ Adob​​e_Flex_Builder_Linux / sdks / 3.xを選択します。







-リストで「Flex 3.x」を選択し、変更を適用します。







以上です。 働くことができます。
