UbuntuでSSHグリーティングを変更する方法

SSHでUbuntuに接続するたびに、同じ鈍いメッセージが表示されます ：

Linux superfast 2.6.20-16-generic #2 SMP Thu Jun 7 19:00:28 UTC 2007×86_64



The programs included with the Ubuntu system are free software;

the exact distribution terms for each program are described in the

individual files in /usr/share/doc/*/copyright.



Ubuntu comes with ABSOLUTELY NO WARRANTY, to the extent permitted by

applicable law.



Last login: Mon Aug 13 01:05:46 2007 from ipaddress removed



geek@superfast:~$





変更方法は次のとおりです。 いくつかのファイルを編集する必要があります。

この「カート」の最初の3つの段落は次のとおりです。

/etc/motd





ビルド番号と保証の免責事項-情報は私にとってあまり有用ではなく、すべてを置き換えました。

最後のログインに関するメッセージを無効にするには（これはお勧めしません）、このファイルをここで編集する必要があります。

/etc/ssh/sshd_config





このオプションを見つけて、yesをnoに置き換えます。

PrintLastLog no





これで、入力してもこのメッセージは表示されなくなります。 セキュリティ上の理由から、そうすることはお勧めしません。

ログインは次のようになります。

This is a superfast system. Please max out the cpu accordingly.

geek@superfast:~$
