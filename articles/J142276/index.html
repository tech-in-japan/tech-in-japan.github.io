6月上旬、サムスンは、最新世代のGALAXY Tab 3タブレットの3つのバージョンすべて、7、8、10インチを世界に公開しました。 伝統的に、新しいデバイスは「重量を失い」、いくらか重量を失いましたが、同時に、より近代的な画面、より強力なプロセッサ、より容量の大きいバッテリーを獲得しました。激しいワークロード。





GALAXY Tabシリーズのタブレットの歴史への短いエクスカーションに飛び込むと、その開発パスは次のようになります。



- 2010年9月 。 Samsung Electronicsの最初のツバメタブレットであるGALAXY Tabが市場に登場します。 このデバイスは視聴者の間で大きな関心を呼びました。これは、画面対角線が7インチで高度な技術特性を備えた同様のデバイス（その解像度が1024 x 700のTFTディスプレイ、Cortex A8プロセッサ1000 MHz、PowerVRグラフィックアクセラレータ）を誇る企業がまだなかったためですSGX540、512 MB RAMおよび4000 mAhバッテリー）。 惨めなAndroid 2.2だけが写真を台無しにし、タブレットの問題にはまったく不適切です。





- 2011年5月 。 彼は長い間待たされ、必死に望まれていました：本格的な10.1インチGALAXY Tabがついに登場しました！ このコンパクトなマシンは、1280 x 800の画面解像度、1000 MHzのデュアルコアNvidia Tegra 2プロセッサ、1ギガバイトのRAM、USBホスト、7000 mAhのバッテリーをすでに誇っていました。 そして-神を称賛-はるかに適切なOSのAndroid 3.1ハニカム。





- 2011年7月 。 10.1インチモデルの弟であるGALAXY Tab 8.9は、その姿を明らかにしました。 タブレット自体に革新はありませんでした。ラインの範囲を広げただけです。





- 2011年12月 。 新年シーズンは、GALAXY Tabファミリーの2つの新しいタブレット7.0 Plus（N）と7.7ですぐに満足しました。 7インチモデルは、1024 x 700の解像度のTFTディスプレイ、1200 MHzのネイティブデュアルコアSamsung Exynos 4210プロセッサー、Mali-400MPビデオプロセッサー、1 GBのRAM、更新されたAndroid 3.2 OS、USBホスト、4000 mAhバッテリーを搭載していました。





それに比べて、画面の対角線が7.7のバージョンはより面白そうでした。最終的に1280 x 800の解像度を備えたSuper AMOLED Plusディスプレイ、GALAXYタブレットのファミリー全体で最速のCortex A9プロセッサー、5100 mAhのリソースを備えたバッテリー8〜10時間のアクティブな使用（infa 100％）。







- 2012年4月 。 4か月も経たないうちに、Samsungは10.1および7.0の対角線を持つGALAXY Tabラインの第2世代を発表しました。 一般に、販売開始時の低価格といくつかの技術的な変更を除いて、第一世代との違いはほとんどありませんでした。





7インチモデルには、ネイティブのSamsung Exynos 4210プロセッサまたはNVIDIAチップセットであるPowerVR SGX540ビデオプロセッサの代わりに、解像度が1024 x 700の同じTFTディスプレイ、Texas Instrumentsのチップセット（Ti OMAP4430、プロセッサ-クロック周波数1000 MHzのデュアルコアCortex-A9）が搭載されましたMali-400MPの代わりに、1 GBのRAM、Android 4.0 / 4.1 OS、4000 mAhバッテリー（7インチモデル）/ 7000 mAh（10インチモデル）。





今年の4月、サムスンのタブレットの第3世代の最初のモデルであるGALAXY Tab 3 7.0が発売されました。







タブレットには、解像度1024 x 600ピクセルのTFTスクリーン、カメラ（メイン3 MPおよびフロント1.3 MP）が搭載されています。 1.2 GHzのクロック周波数と1 GBのRAMを備えたデュアルコアARMプロセッサがパフォーマンスに関与しています。 8 GBおよび16 GBの内部フラッシュメモリと、microSDカードでメモリを拡張するためのスロット（最大64 GB）のオプションがあります。 バッテリー容量は4000 mAhです。 スクリーンの端の縁が薄くなり、タブレットは以前のものよりもコンパクトになりました。 GALAXY Tab 2 7.0の寸法は188 x 111.1 x 9.9 mmに対して193.7 x 122.4 x 10.5 mmです。 重量も40グラム減少しました。Wi-Fi対応バージョンの重量は302グラム、3Gバージョンの重量は306グラムです。
