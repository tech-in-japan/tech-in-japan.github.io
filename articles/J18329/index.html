代替OSの可能性

さまざまなアナリストは、今後数年間は、日中の場所、またはむしろデスクトップ市場のオペレーティングシステムの闘争において決定的になるという見解1、2を絶えず表明しています（ この記事は企業部門について読むことができます）。 実際、今年の初めにWindows Vistaの新しいバージョンがリリースされ、開発に約5年かかりました。 そして最近では、ユーザー向けに戦っているMac OS X Leopardの最新バージョンは、その市場セグメントに当てはまります（一方、ハードウェアと各種ソフトウェアのサポートに関する問題はほとんどありません）。 さらに、Linuxディストリビューションは復活しましたが、ほとんどのリリースには安定した予測可能なリリースサイクルがあるため、これはそれほど注目すべき現象ではありません。



したがって、メインイベントはまさに待望のVistaのリリースでしたが、それでもユーザー（および明らかにクリエイター）の期待に完全に応えることはできませんでした。 販売に関しては、ホイストが記録を更新しているように見えますが、Microsoftの積極的なマーケティングポリシーとコンピューターフリート全体の両方を考慮する必要があります。 それにもかかわらず、企業ユーザーと一般ユーザーの両方が、古いバージョンのWindows XPを引き続き使用しています。 結局のところ、新しいOSをインストールするには、通常、新しいコンピューターを購入するか、古いコンピューターを何度もアップグレードする必要があります。 一方、Windows XPは、最初は専門家やユーザーから好意的に受け取られていなかったが、多くの人がそれを「最高のWindows」と呼んでいたことを思い出してください。 ただし、Windows Vistaにはいくつかの問題があります。



Windows Vistaは何に対して批判されていますか？

  • 高い鉄の要件
  • 低性能
  • 高コストで不便なライセンススキーム
  • DRM
  • 不完全なセキュリティシステム






Linuxのチャンス：2008-2010



もう1つの欠点は、最終バージョンに含まれていない多くの機能です。たとえば、新しいWinFSファイルシステム（ Cairoプロジェクト以降90年代の初めから実際に開発されています）で、 Windowsの第7バージョンに含まれる可能性があります 。 ウィーンというコードネームのこのバージョンは、2010年までにリリースされる予定です。 つまり、代替オペレーティングシステムには、本格的なデスクトップディストリビューション（または複数のディストリビューション）を準備する必要がある2年間があります。 もちろん、他のシステムの開発者もこの機会を利用できます.Appleがハードウェアサポートが制限されているにもかかわらず（PC向けのMacOSを公式にリリースした場合は面白いでしょう（バルマーと言って面白いですか？）



玉座への要求者



Windowsをプッシュした最初の候補は、 最も人気のあるディストリビュー ションの 1つであるUbuntuです。 実際、2008年にはUbuntu 8.04が長いサポート期間（デスクトップでは2011年、サーバーでは2013年）でリリースされました。これは一般ユーザーと企業ユーザーの両方にとって非常に重要です。 Ubuntuのもう1つのプラスは、ベースとなっているDebianの新しいバージョンのリリースです。

NovellがスポンサーとなっているOpenSUSEは 、DistroWatchによると2番目の候補と見なされるべきです。 このオペレーティングシステムに対する態度は、率直に言って-賞賛（企業クライアント向け）から完全な拒否（特にNovellとMicrosoftのコラボレーションに否定的な態度をとるオープンソースファンの間）です。

3つ目はFedoraで 、これはRed Hatによってサポートされています。これは、openSUSEと同様に、エンタープライズ市場でサポートされています。

トップ3以外にも、非常に人気の高い優れたデスクトップディストリビューションMandrivaが残されています。 確かにすべてのLinuxディストリビューションをリストすることはできませんが、デスクトップコンピューターでの使用に最も適した平均的なユーザーを選択しようとしました。





応募者のスイート



ご存知のように、スイートは王様を作り、プログラムはOSを「作り」ます。 実際、今ではオペレーティングシステムは裸のコアや一連の基本的なユーティリティではなく、もちろん、高度に専門化されたプログラムを除き、あらゆる種類のプログラムを備えた本格的な職場です。 実際、カーネルとユーティリティは必要なソフトウェアの最初の層であり、GNU / Linuxと呼ばれています。 2番目の層はデスクトップ環境（「デスクトップ環境」）で、GUIだけでなく、メインオフィスソフトウェアなどの追加プログラムのセットでもあります。

Linuxには、2つの主要なDE- GNOMEとKDEがあります。 第一に、スニーカーに関するいくつかの言葉。これは、ユーザーインターフェイスだけでなくプログラムセットにも影響を与える非常に深刻な変更が加えられた4番目のバージョンが間もなくリリースされるためです。 この場合のgnomeはスニーカーに遅れをとっており（彼は開発哲学を持っていますが、けいれんはありません）、来年3月にバージョン2.22をリリースすることが約束されています。 他の「デスクトップ」については、 Xfceをローエンドマシンで使用できるため、注目に値します。

ソフトウェアの3番目の層は、DEソフトウェアスイートに含まれていないオフィスアプリケーション、ブラウザ、およびマルチメディアです。 主なものだけを見てみましょう。 オープンソースのオフィススイート市場で認められているリーダーは、OpenOffice.orgです。 他の パッケージは彼から非常に強いままです。 Microsoft Officeスイートとの競争に関しては、ここでの状況はオープンソーススイートを支持するものではありませんが 、それらの機能には大きな違いはありません 。

また、ブラウザにも注意を払います。この市場での競争は最も激しいものの1つです。 Fifefoxの3番目のバージョンのリリースは誰もが期待しています。これはInternet Explorerの7番目のバージョンと競合します。 もちろん、Operaの今後のリリースと、 WebKitフレームワークの重要性が増していることは注目に値します。





鉄



増え続けるさまざまなハードウェアのサポートは、Linuxの普及に非常に良い影響を与えます。 デスクトップコンピューターの分野では、状況は改善されており、ドライバーは製造元自身によって開発されていますが、コードや製品の仕様が常に公開されているわけではありません。 最も急速に変化している状況は、製造業者がLinuxプラットフォームの重要性を徐々に認識しているビデオカード市場（および、それに応じてゲーム市場）です。

2007年、Dell、Lenovo、およびHPの3つの主要なコンピュータービルダーは、Linuxがプリインストールされたコンピューターを出荷すると発表しました（各企業は独自の配布キットを選択しました）。



Vindekapets？



もちろん違います。 デスクトップ市場では、中期（2008〜2015年）のWindowsが最も一般的なオペレーティングシステムになります。 現在、GNU / LinuxベースのOSの存在は、国に応じて約1パーセントの範囲です。 機能面でWindowsと完全に比較できるようになったため、このオペレーティングシステムが市場の少なくとも10％を占めることができるようになると、このオペレーティングシステムの計算を開始します。 実際、今後数年間で、代替OSが市場の一部を占める可能性があります。

したがって、今後2年間、Linuxディストリビューションの（マーケティングおよびコンピュータービルダーとのやり取りを含む）競争力のあるリリースを期待しています。いずれにしても、ユーザーとしての競争の恩恵を受けるからです。
