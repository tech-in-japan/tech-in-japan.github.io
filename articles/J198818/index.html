Windows Azure Backupクラウドバックアップサービスは商用利用可能です





データをWindows Azureクラウドにバックアップするサービスはプレビュー段階を終了しており、広く一般に使用できます。



Windows Azureバックアップは、Windows Server向けのクラウドベースのソリューションであり、クラウドへのファイルとフォルダーのバックアップと復元を可能にし、データ損失に対する組み込みの保護を提供します。 このサービスは、IT管理者と開発者に、重要なデータをバックアップおよび保護する機能と、ハードウェアのコストをかけずにどこからでも簡単に復元する機能を提供します。



Windows Azureバックアップには、次の主要な機能が含まれています。



構成と管理の簡素化 。 Windows Azure Backupは、Windows Serverの使い慣れたWindows Serverバックアップユーティリティ、System CenterのData Protection ManagerおよびWindows Server Essentialsと統合して、ディスクまたはクラウドへの単一のバックアップパスを提供します。



ブロックレベルでの増分バックアップ 。 Windows Azureバックアップエージェントは、ファイルの変更とブロックレベルの変更に基づいて増分バックアップを実行し、変更されたブロックのみをコピーするため、トラフィックとストレージの要件が軽減されます。 バックアップの異なるバージョンは、ストレージスペースを効率的に使用し、ブロック間の変更のみを保存します。



データの圧縮、暗号化、および調整 。 Windows Azureバックアップエージェントは、データを暗号化および圧縮してWindows Azureストレージに転送します。 その結果、Windows Azureクラウドには暗号化されたバックアップデータのみを保存できます。 暗号化キーは、クラウド内のWindows Azureバックアップサービスで使用できないため、データを解読できず、クラウドサービスで解読できません。 サービスのユーザーは、調整機能とパラメーターを構成し、バックアップまたは復元中にネットワーク帯域幅をどのように利用するかをサービスに指示することもできます。



データの整合性はクラウドで検証されます。 安全なバックアップに加えて、バックアップデータの整合性が自動的にチェックされます。 その結果、ネットワークを介したデータ送信中に発生する可能性のあるデータ破損は、自動的に検出および修復されます。



クラウドにデータを保存するためのポリシーの構成 。 Windows Azureバックアップサービスでは、ポリシーを使用して、バックアップを削除する必要がある期間を構成できます。 これにより、企業の標準を満たし、ストレージのバックアップを節約できます。
