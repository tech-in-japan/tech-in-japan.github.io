Synology®がRackStation RS812 +およびRS812RP +を発表

信頼性の高いフォールトトレラントでスケーラブルなNAS



Synologyは、中小企業向けの4ディスクラックマウントNASサーバーであるRackStation RS812 +およびRS812RP +を発表しました。 新しいモデルの特徴は、高いパフォーマンス、スケーラビリティ、および信頼性です。



両方のモデルの主な機能の中で、パッシブCPU冷却技術、冗長ファンの可用性、および2つのLANポートのサポートに注意する必要があります。 これにより、単一障害点を排除でき、システムの継続的な安定動作時間が増加します。 RS812RP +モデルの機能は、追加の信頼性と中断のない動作を提供するバックアップ電源です。 ネットワークドライブRS812 +およびRS812RP +は、ハードドライブを「ホットスワップ」する機能を提供し、マザーボードのモジュールに簡単かつ迅速にアクセスするための特別な設計も実装しました。 この一連の対策により、システムのパフォーマンスが向上し、メンテナンスの利便性が向上します。





Synology RS812 +およびRS812RP + NASサーバーは、周波数2.13 GHzのデュアルコアプロセッサに基づいて構築され、1 GBのRAMを搭載しています。 このような強力なハードウェアプラットフォームにより、これらのネットワークドライブは、RAID 5ストレージアレイモードおよびアクティブ化されたネットワークポートアグリゲーション（リンクアグリゲーション）機能で作業しながら、202.31 MB / sの速度でデータを読み取り、175.11 MB / sで書き込むことにより、優れたパフォーマンスを発揮できます。



Synology RX410拡張モジュールをSynology RS812 +およびRS812RP +に接続する機能により、ユーザーはネットワークドライブを8ドライブに拡張できます。 RAMを最大3 GBまで増やすこともできます。 最も完全なiSCSIサポートにより、VMware vSphere、Citrix XenServer、Microsoft Hyper-Vなど、メインサーバー仮想化プラットフォーム向けの包括的なストレージソリューションを構築できます。



Synology RS812 +およびRS812 + RPには以下が装備されています。

  • 2.13GHzデュアルコアプロセッサ
  • 1GB DDR3 RAM（3GBまで拡張可能）
  • 2つのギガビットLANおよびリンクアグリゲーションポート
  • 各4TBの4台のハードドライブをサポート（8台のハードドライブに拡張可能）


主な機能：

  • ホットスワップ可能なハードドライブ
  • マザーボードをすばやく交換する機能
  • VMware / Citrix / Microsoft Hyper-Vnとの互換性
  • Windows ADSサポート
  • 消費電力：46.2W（動作モード）、25.3W（スリープモード）
  • Haspキーのサポート


詳細については、次のリンクにアクセスできます： www.synology.su



ネットワークドライブSynology RS812 +およびRS812RP +はすでにロシアで出荷可能です。SynologyRS812 +の推奨小売価格は52 600ルーブルです（ハードドライブのコストを除く）。
