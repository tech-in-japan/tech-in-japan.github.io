トラフィック分析の秘密

効果的なプロモーションには、ユーザーを引き付けるためのチャネルの選択だけでなく、各チャネルをさらに構成したトラフィック分析も含まれることは論理的です。 セットアップの目標は、投資したリソースの収益を増やし、コンバージョンを増やすことです。



はじめに



すべてのサイトプロモーションアクティビティの全体的な収益ではなく、各プロモーションチャネルの有効性を個別に追跡する方が良いことを誰も疑わないと思う。



この投稿のサイトプロモーションチャネルでは、SEO、コンテキスト広告、メディア、テーマサイトの記事など、ユーザーを引き付ける方法を意味します。



私の意見では、オンラインストアやサービスがトラフィックを引き付けるという目標（購入、登録、ダウンロード）を特定する方が簡単だからです。 そして、すでにこの目標は統計収集システムで設定されています。 目標の設定とは、目的のページのビューを登録することです。たとえば、「ご購入ありがとうございます」などです。 ちなみに、Googleアナリティクスのユーザーは_trackEventを使用してイベントを追跡できるため、柔軟性が向上します。



企業サイトに戻ると、次のようなプレートが必要です。







*個人的な経験から取得したデータであり、政治的な動機はありません



そのような兆候があると、すべてが簡単になり、何にお金を払っているか、どこに注意を払うべきかがすぐにわかります。 当然のことながら、テーブルが詳細であればあるほど、分析は改善されます。 たとえば、同じコンテンツ広告の場合、一般的な「ダイレクト」と「アドワーズ」ではなく、広告グループまたはキーワードに指定できます。



プロモーションチャネルを登録する



最初にすべきことは、すべてのリンクをマークすることです。つまり、リンクにパラメーターを追加します。たとえば、次のとおりです。sitename.ru / super-tovar/？ marker = direct （サイトのエンジンがリンクに外部パラメーターを追加できることは重要です）。 したがって、訪問の統計情報には、さまざまなプロモーションチャネルからサイトへの遷移が表示されます。



私はGoogleアナリティクスが大好きなので、引き続き注力していきます。 Googleアナリティクスはその作業のためにCookieを使用し、最も興味深いのは__UTMZです。













ソース、移行手段（CPC、バナー、メールなど）、広告キャンペーンの名前を保存することが重要です。 また、このCookieには、URLに特別なパラメーターを追加するだけで目的の値を書き込むことができます。 Google リンカーまたはハンドルを使用できます。

sitename.ru/？ utm_source = yandex_direct＆ utm_medium = CPC＆ utm_campaign = ya_direct_camp

これはYandex.Directのリンクの例です。yandex_directはトラフィックソース、CPCはツール、ya_direct_campは広告キャンペーンの名前です。



その後、「すべてのトラフィックソース」ページの分析で、次のようなさまざまなプロモーションチャネルからの遷移が表示されます。









経理担当



今では、各チャネルからの連絡先（通話、手紙）の数を正確に計算することが非常に重要です。 計算には引き続きGoogle Analyticsを使用します。



フィードバックフォームを使用して送信されたメッセージは、イベントを追跡するか、またはsitename.ru/send-message-ok/に人を送信することで読むことができます。「ありがとうございます。メッセージが配信されました。 24時間以内に返信いたします。」 そして、各チャンネルのページへのクリック数を数えるだけです。



ただし、各チャネルの通話数と電子メッセージを計算するには、回避する必要があります。 いくつかの電話番号（モバイルオペレーターの固定電話番号は安価で、IPテレフォニーオペレーターのサービスを使用するか、利用可能な番号を使用することができます）とWebサイトテンプレートへのアクセスが必要です。 アイデアはとてつもなく単純です。__utmzを取得し、ソースを抽出して、各ソースに連絡先の電話番号と電子メールを表示します。 （ソースが定義されていないか、構成されていない場合）別の「デフォルト」のセットを忘れないでください。 私はプログラマーではないので、友人に数行のコードを書くように依頼しました（Yandex.Directのユーザーには特別な連絡先を表示しますが、広告キャンペーンの広告のリンクはマークされています）。



<?php

$output = '';



$__utmz = explode('.',$_COOKIE['__utmz']);

$__utmz = explode('|',$__utmz[4]);



$param = array();

foreach ($__utmz as $value) {

$value = explode('=',$value);

$param[$value[0]] = $value[1];

}



if ($param['utmcsr'] == '(yandex_direct)'){

$output = ': (321) 123-45-67<br>E-mail: contact@sitename.ru';

}



if ($output == '') {

$output = ': (321) 123-45-65<br>E-mail: info@sitename.ru'; }



echo $output;

?>







売上原価の計算



販売統計が収集される場合、各チャネルの1つの販売価格を計算することは難しくありません。



これらの情報をすべて入手したら、ウェブサイトプロモーションチャネルの最適化を開始します。



最も興味深いのは、分析の結果に驚かないことはありません。 そして、私はあなたの決定を聞くことに非常に興味があります：どのように広告費用を最適化しますか？ そして、トラフィックを分析するとき、あなたにとって何が予想外でしたか？
