デスクトップ暗号化ソフトウェアの比較

ごく最近、 モバイル（Android用）暗号化アプリケーションの比較を実施しました。 同様のレビューが必要ですが、デスクトッププログラムが対象です。







比較するプログラムの選択



3つのプログラムすべてが同じウェイトカテゴリに属する​​ようにするため、プロプライエタリなソフトウェア、つまりクローズドソースコードのソフトウェアのみを比較することにしました。 Folder LockとPGP Desktopは、 Cyber​​Safe Top Secretと比較されます。 多くの人が最新のプログラムに慣れていると思います。 しかし、Folder Lockは偶然ではなく、10個の暗号化プログラムと比較して金賞を受賞しました。



フォルダーロックの概要



Folder Lockプログラムの主な機能は次のとおりです。

  • AES暗号化、256ビットキー長。
  • ファイルとフォルダーを非表示にします。
  • ファイルの暗号化（仮想ディスクの作成-金庫）を「その場で」。
  • オンラインバックアップ。
  • 安全なUSB / CD / DVDを作成します。
  • 電子メールの添付ファイルの暗号化。
  • クレジットカード、アカウントなどに関する情報を保存する暗号化された「ウォレット」を作成する




特に個人的な使用のために、プログラムには十分な機能があるようです。 それでは、作業中のプログラムを見てみましょう。 最初の起動時に、プログラムはマスターパスワードの設定を要求します。マスターパスワードは、プログラムでユーザーを認証するために使用されます（図1）。 この状況を想像してください。隠しファイルがあり、他の誰かがプログラムを起動し、どのファイルが隠されているかを見て、それらにアクセスしました。 同意する、あまり良くない。 しかし、プログラムがパスワードを要求した場合、「誰か」は何もすることができません-いずれにせよ、彼があなたのパスワードを拾うか認識するまで。

フォルダーロックプログラムの利点：

  • 英語を話す初心者ユーザーにアピールする魅力的で直感的なインターフェイス。

  • 暗号化されたコンテナのオンラインバックアップと同期の可能性（金庫）。
  • USB / CD / DVD-ROMドライブに自己復号化コンテナーを作成する機能。




プログラムの欠点：

  • ロシア語はサポートされていないため、英語に詳しくないユーザーにとってはプログラムの作業が複雑になります。

  • 電子メールクライアントとの統合はなく、添付ファイルを暗号化する機能のみがあります。
  • クラウドバックアップの高コスト。




PGPデスクトップ



シマンテックのPGP Desktopは、柔軟なマルチレベル暗号化を提供する暗号化プログラムのスイートです。 このプログラムは、システムシェルへの緊密な統合という点でCyber​​Safe TopSecretおよびFolder Lockとは異なります。 プログラムはシェル（Explorer）に埋め込まれており、その機能はExplorerコンテキストメニュー（図14）からアクセスできます。 ご覧のとおり、コンテキストメニューには暗号化、ファイル署名などの機能があります。 非常に興味深いのは、自己復号化アーカイブを作成する機能です-自己解凍アーカイブの原理に基づいて、アーカイブを解凍する代わりに復号化するだけです。 ただし、プログラムFolder LockとCyber​​Safeにも同様の機能があります。

プログラムのセクション：

  • PGPキー -キー管理（所有およびkeyserver.pgp.comからインポート）。
  • PGPメッセージング - メッセージングサービスの管理。 インストール中、プログラムは自動的にアカウントを検出し、AOLインスタントメッセンジャーの通信を自動的に暗号化します。
  • PGPジップ -暗号化されたアーカイブ管理。 このプログラムは、透過的および不透明な暗号化をサポートしています。 このセクションでは、不透明暗号化も実装しています。 暗号化されたZipアーカイブ（PGP Zip）または自己復号化アーカイブ（図17）を作成できます。
  • PGPディスクは、透過的な暗号化機能の実装です。 プログラムは、ハードディスクのパーティション全体（またはディスク全体）を暗号化するか、新しい仮想ディスク（コンテナ）を作成できます。 空き領域の細断処理機能もあり、ディスク上の空き領域を消去できます。
  • PGP Viewer-ここで、PGPメッセージと添付ファイルを復号化できます。
  • PGP NetShareはフォルダーを「共有」するためのツールです。一方、「ボール」はPGPを使用して暗号化され、「ボール」にアクセスできるユーザー（証明書に基づいて識別される）を追加/削除できます。

PGP Desktopプログラムと同様に、Cyber​​Safe Top Secretプログラムは仮想暗号化ディスクを作成し、 ハードディスクパーティションを完全に暗号化できます。 Cyber​​Safe Top Secretプログラムは、Folder LockおよびPGP Desktopプログラムとは異なり、固定サイズの仮想ディスクしか作成できないことに注意してください。 ただし、この欠点は、フォルダを透過的に暗号化する機能によって中和され、フォルダのサイズはハードディスクの空き領域の量によってのみ制限されます。

Cyber​​Safe Top Secretの利点：

  • GOST暗号化アルゴリズムと認定暗号プロバイダーCryptoProのサポート。これにより、個人や営利団体だけでなく政府機関でもプログラムを使用できます。
  • 透過的なフォルダ暗号化のサポート。これにより、プログラムをEFSの代わりとして使用できます。 このプログラムが最高レベルのパフォーマンスとセキュリティを提供することを考えると、このような置き換えは正当な理由以上のものです。
  • 電子的にファイルに署名する機能、およびファイルの署名を検証する機能。
  • キーを公開し、他の会社の従業員が公開した他のキーにアクセスできる統合キーサーバー。
  • 仮想暗号化ディスクを作成する機能とパーティション全体を暗号化する機能。
  • 自己復号化アーカイブを作成する機能。
  • 有料サービスと無料サービスの両方で、あらゆるサービスで機能する無料のクラウドバックアップの可能性。
  • 二要素ユーザー認証。
