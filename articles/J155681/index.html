車輪の上のアポイ

素晴らしいニュースです！



現在、 Apoiアプリケーションは、 Pilot Navigator - NaviPilot Droid社のAndroid OSを実行している車の最新のヘッドユニットの標準パッケージに含まれています。







年の初めから、 Apoi.ruプロジェクトは勢いを増し、本当に人気を博しています。 そして今、また「 自動モバイル 」！ 同じ名前のアプリケーションは、すべてのNaviPilot Droidヘッドユニットの基本ソフトウェアパッケージにプリインストールされています。 以前にスマートフォンまたはタブレットでのみ使用することが可能であった場合、自動マルチメディアセンターNaviPilot Droidの所有者はこの機会を利用できます。









このデバイスは、標準的なヘッドユニットの適度な機能ではなく、インターネットへの直接アクセスによって提供される最も広範な機能を取得したいドライバーを対象としています。 今、あなたはどこにいても、あなたはあなたの車から直接連絡を取り合います。 残念ながら、今日、私たちは交通渋滞に多くの時間を費やし、何時間もアイドル状態になり、貴重な時間を無駄にしています。 このデバイスを使用すると、数時間動けなくても、人生が止まらないことがわかります。車に座っているときでも、外の世界、ビジネスパートナー、友人と連絡を取り合うことができます。 最も幅広いマルチメディアの可能性に加えて、彼は交通渋滞について警告し、交通渋滞を回避することを提案します。 これが不可能になった場合は、ソーシャルネットワークとさまざまなメールおよびインスタントメッセンジャークライアントの両方で通信​​できるようになります。 さまざまなウィジェットとアプリケーションは、オンラインショッピングからインターネットサーフィンまで、交通状況を見失うことなく、多くのエンターテイメントを提供します。



NaviPilot DROIDは、Android OSの基本機能で利用可能なすべての機能に加えて、以下を提供します。



  • DVD / CD / MP3 / MP4を再生する機能を備えたDVDドライブ
  • HD 720Pおよび1080Pビデオ再生システムのサポート
  • 30のラジオ局用のメモリ付きFM / AMラジオ
  • iPod / iPhoneから音楽を再生する
  • デジタルTV信号DVBT / DVBT2を受信する機能（オプション）
  • ゾーン2モード-「セカンドゾーン」-ナビゲーションプログラムと同時に追加のモニターでDVD / TV /ビデオを表示
  • リバースギアが接続されている場合の自動アクティブ化によるリアビューカメラとフロントビューカメラの接続（オプション）
  • プログラム可能なハンドル制御
  • 車両の標準情報（気候制御、駐車センサーなど）をデバイスの画面に表示します
  • SDカード用の2つのスロット、最大32 GB
  • 2つのUSBポート、USB 2.0および外部USB HUBをサポート
  • USBポート経由のGPRS / EDGE / 3G / 4Gモデムを使用したインターネット接続
  • USBアダプター経由のワイヤレスWi-Fi（付属）
  • 音声認識と音声コマンドのサポート
  • システムに表示されるインターフェイスにより、デバイスの機能を拡張できます（OBD II、タイヤ空気圧センサー、駐車センサー、モバイルアクセスポイントなど）






Apoiアプリケーションは 、この素晴らしい自動車デバイスの機能を大幅に拡張しました。 これにより、車の所有者は場所を考慮し、交通渋滞を考慮して目的の組織へのルートを舗装し、目的のカテゴリで最高の企業をすばやく見つけることができます。 また、「レビュー」セクションでは、訪問する前に他の人の意見を知り、「熱心に」感情を共有することができます。



組織の代表者は、車を離れることなく、新しい会社を追加したり、それらに関する情報を編集したり、顧客のフィードバックを使用してサービスの品質を迅速に監視したりできます。



私たちの組織にとって、これは市販のデバイスにアプリケーションをプレインストールする初めての経験です。 私たちは常に協力する準備ができており、将来、このアプリケーションがさまざまなメーカーのより多くのデバイスに表示されることを願っています。



現在、アプリケーションの次のバージョンが準備されており、多くの新機能が追加されます。



  • 現在の割引、販売、プロモーション
  • さまざまなサービスや店舗のレビュー付き記事
  • 組織からのお知らせ
  • プロのアドバイス


私たちは、未来のクルマの創造に貢献できることを非常に嬉しく思います！ そして、これは、機能を大幅に拡張するために、車にさまざまなサービスおよびエンターテイメントアプリケーションをインストールする傾向の始まりに過ぎないと考えています。
