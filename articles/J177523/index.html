あなたの脳が必要です！ :)



Nokia Asha Touchプラットフォームの開発者の競争に参加することをお勧めします。



Javaで記述したり、Webアプリケーションを作成したりする場合、当社のコンテストでは、かけがえのない経験、無料のプロモーション、そしてもちろん、あなたの仕事にふさわしい報奨を得る機会を得ることができます。 賞金総額230,000ユーロは、15人の参加者に分けられます。 12の受賞者は、4つの主要なカテゴリ（「音楽とエンターテイメント」、「ニュースと情報」、「ゲーム」、「ユーティリティと生産性」）で決定されます：1位25,000ユーロ、2位15,000ユーロ、10,000ユーロ- 3位。 さらに3人の参加者に10,000ユーロの特別賞が贈られます（最高のWebアプリケーション、最高の新しいアイデア、最高のUXに対して）。



ちなみに、Nokia Asha Touchプラットフォームの開発は、収益の観点からすでに非常に有望です。1年未満で2,080万台以上のデバイスが販売されました。







コンテストへの参加は思ったよりも簡単です。 たとえば、JavaアプリケーションにはNokia SDK 2.0 、WebアプリケーションにはNokia Web Tools 2.3という非常に便利なツールがあります。 同時に、新しいアプリケーションとAsha Full Touchプラットフォームに移植された両方の参加が許可されます。



競合他社は、Nokia開発者コミュニティから包括的なヘルプを得ることができます。「専門家に尋ねる」セッション、 技術サポート 、 JavaおよびWebAppsプラットフォーム用の広範なUIおよびUXデザインサポート、他のプラットフォームからのプロジェクトの移植手順などのwebinars 。



Nokia Premium Developer Programに参加することもお勧めします 。 資格確認の場合は、無料のAshaスマートフォンと、アプリケーションのテスト用の電話への拡張リモートアクセスと、テクニカルサポートおよびその他の特典のための198ドルのバウチャーを受け取ります 。



Nokia Asha開発者コンテストに参加するには：


1.コンテストに登録します。

2. Asha Full Touch用のアプリケーションを作成する

3. 2013年6月30日までにアプリケーションを公開します。



ヒント：


1.美しいユーザーインターフェイスと豊富な機能を備えた高品質アプリケーションの設計に焦点を当てます。

2. Nokia Ad Exchangeを使用して収益化します。

3.アプリケーションは、コンテストに参加するために、4つの主要カテゴリのいずれかに収まらなければなりません。

4.ユーザーがアプリケーションについてどう思うか知りたいですか？ アイデアプロジェクトにアクセスして最新情報を入手してください。

5.他のプラットフォームからAsha Touchに移植されたアプリケーションを含む参加が許可されます。

6.アプリケーションはNokia Asha 309でテストされます。

7.以前にAsha Full Touch用に開発されたアプリケーションも、新しい機能が追加されていれば競争に参加できます。 目に見える変化のない新しくダウンロードされたアプリケーションは、競争に参加することができません。

8.タッチおよびタイプなどの移植されたアプリケーション、およびフルタッチデバイス用に以前に公開されていない非タッチアプリケーションも競争に参加できます。
