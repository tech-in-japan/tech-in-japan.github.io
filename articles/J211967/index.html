Nimbus Note for Windows-ノートを操作するためのプログラム

みなさんこんにちは！ Nimbus Note for Windowsのバージョンがリリースされたことをお知らせします。これにより、コンピューター上のメモを完全に操作したり、他のデバイスとメモを同期したりできます。 ブラウザー、モバイルデバイス、およびWebパーツ用のアプリケーションは既にありますが、通常のデスクトッププログラムなしでは、このタイプのサービスを完全に呼び出すことは不可能だと思いました。どのモバイルデバイスよりも。 このプログラムの作業はほぼ1年間実施され、その結果は削減された状態で見ることができます。



Nimbus Noteのダウンロード-http: //nimbus.everhelper.me/download.php











機能エディター


エディターには、マーカーや下付き文字/上付き文字からテーブルの作成や編集に至るまで、適切なテキストエディターが依存するほぼすべての機能を使用できます。







Microsoft Word文書とtxtファイルをインポートする


Nimbus Noteを使用すると、Word文書とTXTファイルをインポートして、通常のメモと同じように作業できます。つまり、テキストの変更、画像の挿入などです。







高速


メモを取るアプリは高速である必要があり、これは不可欠なルールです。 プログラムは非常に美しいインターフェースを持つことができますが、作業中にハングした場合、誰もが興味を持つことはまずありません。 私たちはプログラムを最適化するために懸命に努力し、結果を誇りに思う権利を持っています。 このプログラムは、数千のメモがあるアカウントでテストされ、データベースのサイズはギガバイトを超えていましたが、作業のスピードではまったく感じられませんでした。



スマートシンク


特定のメモのみが必要な場合、すべてのメモがダウンロードされるまで待つのはなぜですか？ 理由はありません。 プログラムの同期は段階的に行われます。最初にノートのスニペットがダウンロードされ、その後に添付ファイルが含まれる本文のみがダウンロードされます。必要なノートを選択して、順番にダウンロードすることができます。 また、原則として、すべてのメモをダウンロードする必要はなく、写真や添付ファイルでもダウンロードする必要はありません。 そのような場合、「高速」同期を追加しました。この同期では、メモのスニペットとその本文のみがダウンロードされ、写真と添付ファイルがオンデマンドでダウンロードされます。



構文の強調表示


この機能（これまではベータステータス）は、Habrユーザーにとって興味深いと思います:)その意味を説明する価値はなく、画面上ですべてが明確になっていると思います。 必要なテキストを選択し、[コードブロックとして書式設定]を選択するだけです。 これまでのところ、多くの設計はサポートされていませんが、時間とともに改善されます。







各メモのタスクのリスト


タスクの処理は、私たちとはまったく普通に行われませんでした。つまり、通常、タスクをメモから分離しました。 タスクには、必要なタスクを作成できる別のソケットがあります。







検索と検索アイテム


検索はデータベース全体で実行されますが、別々のフォルダーで検索することもできます。 まだヒントはありません。次のバージョンにプッシュしたいと思っています。 ただし、検索履歴を保持し、必要な結果にいつでも戻ることができる検索項目があります。

メモ自体には検索と置換があり、古き良きメモ帳と同じ機能を実行します。







...および他の多くの可能性


上記に加えて、他の多くの快適な機能があります：プロキシサーバーとの本格的な作業、フォルダーとメモのリストのフォントを選択する機能（デフォルトのTahomaが気に入らない人）、スクリーンショットの作成、ホットキーのサポートなど。



これはプログラムの最初のバージョンであり、考え出された（常に考えられている）すべての機能がそこに実装されているわけではありませんが、それでもすべての機能を気に入っていただけることを願っています。 現時点では、プログラムは英語でのみ利用可能です。私たちのチームは小さく、すべてに十分な人手がないためです（これは確かに言い訳ではありません）が、それでもできるだけ早くロシア語のローカライズを試みます。
