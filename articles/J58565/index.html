Windows Liveの4月の更新！

文字通り3日前、 Windows Liveのホームページに いくつかの改善が加えられました 。 しかし、ユーザーにとって嬉しい驚きはそれだけではありませんでした！ 今日、非常に便利で人気のある機能がWindows Liveサービスに追加されました。



  • Webアクション！

    Facebookで友達とチャットしたり、 LiveJournalでブログを書いたり、 Flickrで写真をアップロードしたり、 LastFMで音楽を聴いたりしますか？ 他のネットワークで何か新しいものを追加したこと、どんな種類の音楽を聴いたか、どんな面白いものを共有したいか、友達に見てもらいたいですか？ プロファイルにWebアクションを追加してください！





  • Facebook、MySpace、Hi5、Linkedinから友達を追加してください！

    クールなWindows Liveサービスと、さまざまなソーシャルネットワークの友人をまだ使用していませんか？ その後、それらを招待します！ これを行う最も簡単な方法は、 http：//profile.live.com/connectから招待することです！





  • 私たちは人々に自分自身を見せます！

    参加して明るい人生の瞬間の写真を見せることほど簡単なことはありません！ これを行うには、 Windows Live Photosサービスに写真をアップロードして、自分にマークを付けます。 誰でもあなたのプロフィールにアクセスし、「ユーザー写真」リンクをクリックして、あなたが写っているすべての写真を見ることができます！



Windows Liveサービスの新機能が気に入っています。 あなたはどうですか？ :)
