デスOS / 2

先週のWindows 3.1の議論で、 danSamaraは次の ように言及しました

あなたの複雑で多面的なシステムは、OS / 2の下で私の仮想マシンで機能しました。


惨めで単純なWindowsが強力で完璧なOS / 2を絞め殺したのはどうしてですか？

開発者の発言：



Raymond Chen： Windows 1.0 の 「キラープログラム」-新しいOSを購入する準備ができていたもの-はAdobe PageMakerでした。 その後、Excelも登場しました。



ラリーオスターマン：実際、PagemakerとExcelはMacIntoshにとってキラープログラムでした。 その後、Windows用にリリースされました。



私の知る限り、Windowsには「キラープログラム」はまったくありませんでした。 Windows 1.0および1.1は、月に数千部の控えめな印刷で販売されました。 Windows 386（バージョン2.0）は、複数のDOSアプリケーションを同時に実行できるため、よりよく売れました。 すべてのアプリケーションを同時に起動できるわけではありません。 実際、GWBasicのいくつかのコピーよりも複雑なものを実行することはほとんど不可能でした。



Windows 3.0がリリースされたときにすべてが変更されました。



Windows 3.0では、さらに2つの重要な新機能が導入されました。
  1. 保護モード286のサポート。アプリケーションは、従来の640KBを超えるメモリにアクセスできるようになりました。 （以前は拡張メモリトリックが使用されていましたが、プロテクトモードでは、すべてがはるかに簡単でした。）
  2. 16色以上のディスプレイのサポート。


アーロン・レイノルズに行ったときのことを覚えています。 私はLan Manager 2.0に従事していましたが、Windows 2.0に関するいくつかの質問を見つける必要がありました。 Aaron、David Weiss、およびRalph Lifeは、Windows 3.0での作業内容を示しました。 私が見た後に言うことができた唯一のことは、「みんな、スティーブはあなたを殺します、あなたはこれについて知っていますか？」



現時点では、MicrosoftとIBMはOS / 2に完全に焦点を合わせていました。 オペレーティングシステムの分野で行ったことはすべて 、OS / 2に行きました。 （例外はNT OS / 2と呼ばれる小さな並列プロジェクトでした。）開発者に余分な時間があった場合にのみ、WindowsとDOSを購入する余裕がありました。



そして、これらの人々はここでWindowsのバージョンを計画しています。 はい、スティーブはOS / 2キラーに取り組んでいることがわかった場合、子牛を産みます。 これは、Windows 3.0がリリースされたときに起こりました。OS/ 2が死ぬほど絞め殺されました。 （子牛については知りません。）



一晩で、Windowsはおもちゃから有望なオペレーティングシステムに変わりました。 新しいアプリケーションがWindows 3.0用にリリースされる前であっても、1か月あたり数百万のコピーを販売しました。



レイモンド・チェン：ラリーはイベントの順序が少し間違っていたので、彼を修正します。



Windows 2.0は、Windows / 386で保護モードをサポートしました。 このバージョンは、Windows 3.0より前にリリースされました。 Windows / 386で、最初に「仮想8086」モードを使用して複数のDOSプログラムを同時に実行しました。 実行中のDOSプログラム間の切り替えは、明示的にオペレーティングシステムに制御を移したネイティブWindowsプログラムとは異なり、自動的に行われました。

Windows / 386が登場したとき、「通常の」Windows 2.0の名前をWindows / 286に変更しました。 両方のシステムのバージョン番号は同じでした。



したがって、Windowsは「リアル」（Windows 1.0など）、「標準」（Windows / 286など）、および「高度」（Windows / 386など）の3つのモードで動作します。 これら3つのモードのオペレーティングシステムはまったく異なる動作をしましたが、「リアルモード」のプログラムは2つの新しいモードに変更を加えずに動作し続けたことは驚くべきことです。 3つのオペレーティングシステムすべてをサポートする1​​つのプログラムを作成できます。





（Windows 3.0の3つの動作モードの最も顕著な違いは、使用可能なメモリの量です）


そして、Windows 3.0が登場し、すべてが変わりました。 彼らは屋根を突破するように販売が跳ね上がりました。 大手ディストリビューター（Egghead？）の1人がWindows 3.0の成功に非常に興奮していたため、Microsoftの従業員全員がアイスクリームを購入したことを覚えています。 （実際にOS / 2で働いていた私のような従業員でさえも）オフィスに座っていると、人々は突然大きな箱のアイスクリームを持ってきて、「これはEggheadのあなたのためです。 Windows 3.0をありがとう。 すばらしい仕事です。」



私は変に感じました：私が取り組んでいない製品に感謝しただけではありませんでした。 製品に感謝しました。それは、私が取り組んだすべてのものを完全に破壊しました！



Joel Spolsky：そのアイスクリームは数ヶ月間延期されました。 私はそれがどれだけ無味であるかにさえ驚いた。 たぶん、エッグヘッドはスーパーマーケットから古くなった商品を取り除いたところでしょう。
