2015年のマイクロソフトのトップ10の最も重要な発表





こんにちは、読者の皆さん！



新しい2015年を見送り、Microsoftが過去1年間に共有した主な発表を思い出します。

カットの下で、あなたは次年度の技術発表を見つけるでしょう。 まとめており、最も印象的なものを共有したいと思います。





1. Windows 10



2015年7月、コンピューター用のWindows 10の販売が世界中で始まりました。

マイクロソフトから新しいオペレーティングシステムがリリースされてから数か月後、「トップ10」がインストールされたデバイスの数は1億1,000万に近づきました。 リンクからオペレーティングシステムの新しいバージョンにアップグレードすることもできます。



Microsoftの以前のバージョンのOCと比較して、Windows 10がさまざまなデバイスにインストールできることは周知の事実です。 これらのデバイスには、パーソナルコンピューター、ラップトップ、タブレット、スマートフォン、ゲームコンソール、さまざまなボード、モノのインターネットデバイスが含まれます。 また、上記のデバイスで実行できるユニバーサルアプリケーションを開発するための単一のプラットフォームを表します。



Windows 10 Mobileオペレーティングシステムは、新しいLumia 950およびLumia 950 XL電話のリリースで利用可能になり、これまでは新しいスマートフォンの所有者のみが利用できましたが、古いLumiaデバイス用の新しいオペレーティングシステムのリリースは2016年にすでに計画されています。



また今年、同社はWindows 10 IoTオペレーティングシステムのバージョンの1つ、つまり、マイクロコントローラー、マイクロコンピューター、およびその他の類似デバイス用のWindows 10 IoTコアを発表しました。





2. Visual Studio 2015リリース



Visual Studioツールは2015年の間に2つの更新プログラムを受け取りました-最初に待望のツールの新しいバージョンがリリースされ、次にVisual Studio Update 1の最初の主要な更新プログラムがすべての人に利用可能になりました。 すべてのツールファミリに十分なニュースがありました：これがTFS更新1の登場、 Visual Studio Team Servicesオンラインサービス（以前のVisual Studio Online）の更新、開発者はVisual Studio Marketplaceを使用してすべてのVSツールの機能を拡張する機会を得ました無料のDev Essentials開発者サポートプログラム。



3.オープンソースのVisual Studioコード



Microsoftコードエディターのリリース後、開発者はLinux、OS X、およびWindowsでVisual Studioファミリツールを使用できました。 しかし、今年、同社はさらに進んで、Visual Studio Codeがオープンソースプロジェクトになり、それを利用したい人は誰でも公式のVS Codeページからhttps://github.com/MicrosoftからGitHubのリポジトリにアクセスできることを発表しました。 / vscode /





4. Office 2016リリース



2015年9月、Office 2016の新しいバージョンが正式にリリースされ、Windows、OS X、Android、iOSのユーザーが利用できます。 この変更は、新しいツールのインターフェースにわずかな影響を及ぼしましたが、ユーザーが1つのドキュメントで共同作業を行う可能性や、企業環境でアプリケーションを展開するためのモデルに影響を与えました。



ところで、開発者にとっては、ユーザーの日常業務中の生産性の向上、つまりオフィスコンテキストアプリケーション（Officeアドイン）の使用に貢献する機会があります。 今年は、ブログのコラム全体をOfficeの開発の方向性に当てました。ハッシュタグ#officeplatformを使用して見つけることができる資料です



5. .NETオープンソース



このニュースは2014年に属しているという事実にもかかわらず、2015年には、.NETの新しい「オープン」履歴のために1つの重要なイベントが発生しました。 以前、同社は.NET Coreライブラリを公開していましたが、今年MicrosoftはCoreCLRのソースコードを公開しました。CoreCLRは、マシンコードへのコンパイルを含むさまざまな機能を担う.NET Coreの主要コンポーネントです。



6.新しいMicrosoft Edgeブラウザー



Build 2015の革新の1つは、Microsoftの新しいブラウザーで、Windows 10のユーザーが利用できます。まもなく、JSConf US Last Call会議（米国フロリダ州）で、 Microsoftが使用するChakra JavaScriptエンジンの主要コンポーネントのソースコードのオープンの発表を発表しましたエッジ ソースコードは、GitHubでChakraCoreプロジェクトとして利用できます。





7. Microsoft Azure



Azureクラウドプラットフォームのインフラストラクチャは、年間を通じて28の大規模なデータセンターに成長し、世界中に分散しています。 これは、他の2つのグローバルパブリッククラウドベンダーの合計の2倍です。



インフラストラクチャの拡大に加えて、Azureで利用可能な開発者およびITプロフェッショナル向けの有用なサービス、機能、およびツールキットの数が増えています。 過去12か月で、クラウドサービスのユーザーは500を超える新しいサービスと機能を利用できるようになりました。



8. RedHatとマイクロソフトのパートナーシップ



次の発表が理由で私たちのトップを襲った。 ビジネスレベルで行われるこのような決定は、多くの場合、技術の変更につながり、開発者やIT専門家に確実に影響します。



たとえば、クラウドテクノロジーの分野におけるMicrosoftとRed Hatのパートナーシップと.NETプラットフォームの進歩により、Red Hat Enterprise Linuxなどの完全なサポートが実現します。 Microsoft Azure上で、Red Hat製品を含む仮想マシンイメージをAzure Marketplaceディレクトリからダウンロードできます。



9.デバイス



今年、同社はブランドデバイスのラインアップをMicrosoft Band 2のパーソナルアシスタントから伝説的なSurface Pro 4に更新しただけでなく、多くの新製品も導入しました。その中で最も予期せぬものは、おそらくSurface Bookと考えられますマイクロソフトの最初のラップトップに変わることができること。



10. HoloLens



2015年は、マイクロソフトにとって革新的なソリューションの年と言えます。 別の確認として、Microsoft HoloLensがあります。MicrosoftHoloLensは、この夏同社が発表した完全に独立した新しいデバイスで、開発者とユーザーの両方から議論と興奮の嵐を巻き起こしました。 拡張現実を作成するための普通の眼鏡ではなく、実際の表面に表示されるホログラムオブジェクトを制御するための作業を可能にする新しい技術。



2015年は素晴らしい年でしたが、2016年のシェアについてはニュースが残ります。たとえば、先に発表したように、 開発者向けのMicrosoft HoloLensは2016年の第1四半期に利用可能になります。
