人気コンテンツプロジェクトBuzzFeedが公開IPOに進むことを決定

Code Conferenceで 、 BuzzFeedの CEOであるJonah Perettiは、公開する予定だと述べました。 これについての話は昨年から続いています。 これについて最初に言及したのは、社長のグレッグ・コールマンでした。 しかし、その後、出口の可能性そのものが考慮され、基本的な決定は行われませんでした。



過去数年にわたって、BuzzFeedは非常に人気のあるプラットフォームになり、コンテンツとリスト記事の単純な集約から、ビデオコンテンツを含むいくつかの形式の組み合わせに移行しました。 リソースの管理は、ビジネスコンテンツとエンターテイメント、さらには政治との調和のとれた共存を組織化することに成功しました。 メディア会社は、ビデオコンテンツの制作に成功しました。 彼女のYouTubeチャンネルの合計視聴回数は1か月あたり約10億回です。



「私たちは国際開発に非常に注力しています。 異なるプラットフォームをカバーしようとしています。 そして、私たちはビデオビジネスのさらなる発展に非常に注力しています」とBuzzFeedの責任者は述べています。



しかし、会社の仕事のいくつかの分野の組み合わせは、あいまいなビジネスモデルを繰り返し批判している、 と The Guardianは報告しています。



批判にもかかわらず、 Andressen Horowitzは昨年8月にBuzzFeedに5000万ドルを投資した。 これは会社の歴史の中で5回目の資金調達でした。 その後、彼女の評価は8億5000万ドルに達しました。 現在、同社の価値は15億から20億ドルです。 利益情報はまだ公開されていません。



昨年、 The Walt Disneyはメディア会社の買収 を検討していました。 おそらく、当事者は価格の問題について合意に至らなかった-BuzzFeedの経営陣は10億ドル以上を要求した。



毎月、約2億人のユーザーがリソースにアクセスします。 BuzzFeedは、最も成功したコンテンツプロジェクトの1つと言えるでしょう。 2014年の同社の年間収益は1億2000万ドルでした。 この金額はブルームバーグの専門家と呼ばれていました。 BuzzFeedはまもなく数千人の従業員に届きます。



メディア会社の経営陣は、基本的に広告主と協力していません。 サイトにはバナー広告を含む広告はありません。 例外はネイティブ広告です。 それは当然出版物の形式に適合し、スポンサーシップ出版物です。
