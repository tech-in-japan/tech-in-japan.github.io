Tcl / Tk：Yandexのダウンローダーコンテキストメニューからの写真（preβ）

比較的速いペースで、彼は画像アップローダーを作成しました。 スクリプトを修正する場所と方法を見つけるために、このような状態でそれを広めました。





大量のファイルに煩わされないように、すべてのコードを1つにまとめました。



Google Code- http://yfotkiup.googlecode.com/svn/trunk/yfotkiup.tclからスクリプトを取得できます。



設置


  • スクリプト自体をダウンロードします 。
  • UTF-8をサポートするテキストエディターで開きます 。
  • 文字列を見つける

    variable login "test"





    variable pass "testtest"





    テストデータを自分のものに置き換えます。
  • defalbum



    変数もあります。 特定のアルバムに写真をアップロードする場合は、 その番号をすぐに設定できます。

    variable defalbum "12345"



  • 涼しく乾燥した場所に置き、道を覚えておいてください。




次に、 %USERPROFILE%\SendTo



フォルダーにショートカットを作成するだけで%USERPROFILE%\SendTo



ます（VistaおよびWin7でそうでない場合は修正します）が、Tclをインストールしていないと思われます。 :)

Windowsの場合、activestate.comからActiveTclをダウンロードする必要があります（win32の直接リンク ）。 そしてインストールします。 tclkitではまだ機能しません。



インタプリタへのパスは標準のまま（ C:\Tcl\



）であり、スクリプトはC:\Tcl\scripts\



と想定しています。



%USERPROFILE%\SendTo



は、「 C:\Tcl\bin\base-tk8.6-thread-win32-ix86.exe C:\Tcl\scripts\yfotkiup.tcl



」を参照するショートカットを作成する必要があります。 たとえば、Yandex.Fotkiという名前（アイコンはここから取得できます ）。
