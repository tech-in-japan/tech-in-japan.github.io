新しいMac miniの解析

iFixitのメンバーはすでに新しいMac miniを受け取っており、古き良き伝統に従って、彼らはそれを分解しました！















電源コードとHDMI-DVIアダプターが付属







本体はアルミニウムの単一部品でできています。 寸法3.6 x 19.7 x 19.7 cm、重量はわずか1.37 kg。 iPadやiMacのように、リンゴのロゴは黒になりました。







さて、あなたはすべて明確です。







前世代と比較して、新しいMac miniは幅が広く薄くなっています。









反時計回りにひねり、カバーを取り外して、RAMとファンを確認します。







RAMの変更は非常に簡単で、基本構成では2 GBのメモリ（1 GBの2つのダイ）です。 サポートされる最大ボリュームは8 GBです。









2本のネジを外し、冷却装置を取り外します。 騒音と振動を減らすため、ネジにはゴム製ガスケットが付いています。 Mac miniのノイズレベルは14 dBです。









Mac miniの前世代と同様に、システム全体は単一のファンで冷却されます。









Wi-Fiアンテナには4本のネジが付いています。







アンテナコネクタがハードドライブの近くに隠れました。









プラスチックカバーを取り外して、マザーボードのコネクタにアクセスします。 ハードドライブ、ドライブ、熱センサー、IRレシーバーを取り外します。













電源を切り、マザーボードを静かに引き出します。









ここで彼女は美人です！









小型スピーカーは2本のネジで支えられています。











AirMac Extremeはスピーカーの隣に隠れています。 802.11nネットワークとBluetooth 2.1をサポートしています。









冷却装置を取り外します。









コネクタと別のアンテナで背面を取り外します。









そのため、右側にはIntel P8600 2.4 GHz Core 2 Duo、左側にはNVIDIA GeForce 320M 256MB DDR3 SDRAMがあります。

反対側には、左上のDelta LFE9249-R LANフィルター、Broadcom BCM57765ギガビットイーサネット、およびその右側のカードリーダーコントローラーがあります。 以下はCirrus 4206ACNZオーディオコントローラーです。









ハードドライブは簡単に交換できます。











12ボルトの7アンペア電源。

多くの人は、Appleがどのように電源に眠りに落ちることができるのだろうと思いましたか？









ドライブは簡単にスライドできます。 ドライブモデルは新品です！

Hitachi-LG Data Storage（HLDS）Super Multi DVD RewriterモデルGA32N







さて、それだけです！
