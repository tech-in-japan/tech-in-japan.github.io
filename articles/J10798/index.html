描く-したくない

衝撃的で技術的なプログラミングの世界チャンピオン、ジョン・カーマックの別の独創的な発明として知られるようになりました。 控えめに言ってもid Tech 5と呼ばれるこのテクノロジーは、パフォーマンスを犠牲にすることなく、将来のゲームであらゆるボリュームのテクスチャを使用できるようにします。 Windows、MacOS X、Xbox 360、PlayStation 3プラットフォームを完全にサポートすることで、このテクノロジーに良い未来がもたらされるはずです-控えめなタイトル（id Software自身によって作成）が強調する「新世代プラットフォームの最初の真のテクノロジー」。

id Tech 5は、2007 Worldwide Developers ConferenceでStephen Jobsのレポートの一部として紹介されました。Jobsは、MacOS Xを実行するMacintoshで発売されたレーシングアーケードのトラックを披露しました。

id Tech 5で作業するアーティストは、特別なツールを使用して、個々のピクセルのレベルで自由に世界を変えることができ、彼らの行動は生産性に影響しません。 使いやすいエディターに加えて、idは非常に詳細なオープンスペースとクローズドスペースで同等に高速な作業を約束します。

もちろん、Doomの作者による新しい極秘プロジェクトでは、id Tech 5を使用します。さらに、スタジオは既にライセンスの購入申請を受け付けています。





貴族のゴミ

「通常のヒーロー」について（彼らは常に回っています！）多くのバラードが作られ、多くのゲームが作られています。 しかし、まだ誰もが姫を救う雪のように白い馬の良い騎士になりたがっているわけではありません。 「異常な」反乱者のために、Firefly Studiosは若い出版社Gamecock Mediaの支援を受けて試みています。 これらの素晴らしい人々の努力のおかげで、ロールプレイングゲームのダンジョンヒーローが来年登場し、下水道、地下墓地、ダンジョンの底での勇敢な冒険について語っています。 開発者は、ゴブリン、インプ、トロル、その他の爬虫類の生活を現実的に示すタスクを自ら設定しました。 グールと友達を作ることで、ヒーローは彼らと一緒に住み、取引し、助け合うことができます...凍傷したトロルがあなたの道を渡った場合...まあ、ダンジョンヒーローのヒーロースキルはそのような問題を解決できます。

ゲームは、PCとXbox 360コンソールの両方でリリースされる予定です。



私たちのようなスパイ

Sony Online Entertainment Corporationは、スパイマルチプレイヤーアクション映画The Agency for PC and PlayStation 3を正式に発表しました。

エージェンシーの陰謀は、近い将来の2つのスパイ組織、UniteとParagonの対立について語っています。 「ファーム」のいずれかのサービスに参加すると、プレーヤーは基本的および二次的な任務を遂行し、報酬、経験値、称号を獲得し、遅かれ早かれ自分のオフィスを作るために母校を離れます。

エージェンシーのミッションは、プレイヤーとそのチームメイトのみが参加できる場所で行われます（合計で4人まで）。 誰かがミッションに参加するたびに、ゲームは彼のために世界のこの部分のユニークなコピーを作成します。 著者は、あらゆる種類の武器（ピストルからスナイパーライフルまで）からの発砲だけでなく、「車輪の上の射撃場」やヘリコプターに衝突する車のような壮大なエピソードで飽和するエキサイティングな戦いを手配することを約束します。

エージェンシーのキャラクター開発アプローチはシンプルに使用されていましたが、新しい分身を作成せずにすべてのコンテンツを試すことができました。すべてのスキルはその瞬間に着る服に依存します。 衣装を変更すると、特定のミッションの目的に応じて、ステルス能力を強化したり、自分の病棟や仲間がオープンバトルで生き残る可能性を高めたり、他のキャラクターに変装したりすることができます。 地元のペットの役割は、NPCの工作員によって実行されます。NPCの工作員は、ボスプレイヤーの軽い手で、衛星を介して敵の居場所に関する情報や今後のミッションに関する重要な詳細を簡単に取得します。

エージェンシーの究極の目標は、ユーザーが運営するスパイ企業（ファンタジーMMORPGのギルドに類似）が領土のために戦う仮想世界を作成することです。 著者はこの戦争を「世界支配モード」と呼んでいます。 PCとPS3のバージョン間の互換性、配布モデル、および可能な支払い方法に関する質問はまだ公開されていません。



天才の場所

都市シリーズSimCityの新しい部分の差し迫ったリリースの公式確認があるとすぐに、電子芸術とティルテッドミルエンターテイメントは、将来のプロジェクトの詳細を浴びせました...

新しいSimCityはSocietiesという字幕が付けられているため、開発者は市民の社会的管理に焦点を当てています。 「SimCityシリーズをどの方向に発展させるべきかを考え始めたとき、なぜ一部の都市が他の都市と異なって認識されているのかと思いました。 たとえば、パリとメッカの違いは何ですか？ それともオックスフォードとミュンヘン？ それともサンフランシスコとソルトレイクシティ？ -Electronic Artsの副社長であり、同社の「Sim-line」を担当しているRod Humble氏は言います。 -感情は明らかに地元の文化や社会の影響を受けています。 そのため、コミュニティ内の原動力に焦点を当てることで、さまざまな素晴らしい場所を作成できることがわかりました。」

6つの「社会的流れ」（たとえば、富や従順など）のバランスを調整することで、ゲーマーは都市の雰囲気と外観を決定できます。 そのため、市長は謙虚に休んで、通りや建物に防犯カメラの外観を実現し、大都市は全体主義的な形を取ります。 プレーヤーが自分の家柄で何が起こっているかを常に知っているように、SimCity Societiesは選択したコースの枠組み内で彼の現在の目標を生成し、住民は上からの決定に行動で対応します。 目標を達成するために、SimCity Societiesは、新しいタイプの建物やその他のボーナスを仮想市長に提供します。 これは、ゲームの開始時に、建物の15％しか利用できないと考える場合に重要です。 著者は、このタイプのゲームプレイは既成のミッションセットよりも興味深いと考えています。
