SAP Cloud Platformを搭載したカウボーイドローンを使用したスマートファーム

SAPは、多くの場合、企業や工業企業を管理するためのERPシステムの唯一の開発者として認識されています。 一方、同社はエンタープライズ管理システムの開発をはるかに超えています。 SAPはR＆Dに投資し、モノのインターネット、機械学習、人工知能、ロボット、拡張現実および仮想現実などの新しい産業に依存しています。



高度な技術を使用した新しいシナリオの1つは、ドローン、モバイルアプリケーション、SAP Cloud Platformを使用した「スマートファーム」のソリューションです。 このシステムは、世界中の農業企業の従業員が多数の牛を監視するのに役立ちます。 このシナリオの作成者は、SAP Labsのモスクワ事務所の従業員、ブルガリアのAlexanderとOleg兄弟です。



本日の投稿では、この仕組みについて説明します。



羊飼いドローンが必要な理由



農業所有地と家畜農場には、人件費を増やすことなく、動物の数を記録し、数を増やし、牧草地と畑を管理し、放牧面積を拡大するといういくつかの目標があります。 したがって、農業では、モノのインターネットのための新しい技術の積極的な研究と実装があります。



ロシア最大の農業所有地の1つが、万平方キロメートルに100万匹の動物を配置する計画を立てているとしましょう。 このような広大な地域からデータを収集するには、より多くの従業員を雇用するか、モノのインターネットの新しい技術的ソリューションを見つけることができます。 ロシアの大規模農場では、チップを使用して動物がいつどのように健康状態を養うかについての情報を収集する慣行がすでに広まっています。 SAPソリューションは、多くのプロセスを自動化し、従業員に管理機能のみを残します。



SAP Cloud Platformファーム管理システムの仕組み



システムは、次のハードウェア部品で構成されています。



-マイコン

-GPS、温度、モーションセンサー

-最大5 kmの通信範囲を持つLoRa WANモデムおよびトランスミッター

-クアドロコプター（ドローン）

-SAPクラウドプラットフォーム







センサーは動物に取り付けられます：位置（GPS）、動き（加速度計）、温度。 センサーは、LoRa WANテクノロジーを使用して、クアドロコプターに搭載されたマイクロコンピューターにデータを送信します。



LoRa WANは、データ伝送用の新しいテクノロジーであり、動物用のワイヤレスセンサーを備えた長距離ネットワーク（1〜30 km）での有効性を証明しています。 LoRa WANを使用するセンサーセットは、最大1000台のデバイスをネットワーク化できます。



無人機はデータ信号を受信するための基地局として使用されます。これは、大きな牧草地ではセルラー通信やその他のタイプの通信がないためです。 彼は群れの上を飛び、「病棟」からデータを収集し、「ベースに」戻ります。ここで、センサーからの情報が処理のためにSAP Cloud Platformにアップロードされます。











開発者によると、現在1つのセンサーのコストは約25ドルですが、価格は常に低下しています。 内蔵バッテリーを使用すると、牛の生涯（約3年間）にわたってデータを転送できます。 ドローンは牧草地や動物の写真も撮ります。 取得した画像を使用して、NDVIインデックスを計算し、牧草地の緑豊かな植生の状態を判断します。



センサーから得られたデータにより、動物の健康状態について結論を出すことができます。 最近の研究では、加速度計の読み取り値が最大9つの異なる牛の病気を認識できることが示されています。 温度センサーは、死亡または病気の動物を検出し、牛群の残りの感染源を排除するのに役立ちます。 GPSと温度計のデータを比較すると、負傷した動物や閉じ込められた動物を見つけることができます。



ファーム管理システムでのソフトウェアの動作



ドローンが収集されたデータとともに農場に戻ると、すべての測定値と写真がSAP Cloud Platformにアップロードされます。 従業員は、受け取った情報の分析をすぐに開始し、さらなる行動の計画を立て、予測を立てることができます。



作業タイトル「農場管理」の下のシステムには、群れを制御するためのいくつかのモードがあります。 この記事では、それらのいくつかを示します。





放牧のために部門を離れた動物を特定する



最初の画像は、特別なインシデントに関するアラートの画面を示しています。 家畜の専門家は、その助けを借りて、放牧のために「エイリアン」セクターに逃げた動物に気付き、動物の地籍数、体調を判断し、「逃げた」牛を正しいセクターに戻すオペレーターにタスクを送ることができます。





オペレーターのタスク割り当て



センサーからのデータにより、病気の動物をリアルタイムで確認し、群れの残りの部分から隔離する決定を迅速に行い、タスクを獣医に送ることができます。





センサーから受信した情報を使用して動物の状態を確認する



家畜企業の従業員は、群れに関する情報を受け取り、コンピューター上でも、スマートフォンやタブレット用のモバイルアプリケーションの形式でも、管理上の決定を下すことができます。



プロトタイプの作成者はSAPの従業員であり、アレクサンダー兄弟とブルガリアのオレグ兄弟です。 彼らは、さまざまな国で使用するためにソフトウェアを適応させるグローバル開発部門であるSAP Globalization Servicesで働いています。
