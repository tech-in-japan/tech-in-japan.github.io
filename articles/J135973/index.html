Evernoteパートナー：iftttはEvernoteを他のサービスと接続します



タイトル： ifttt

プラットフォーム： Web

価格：無料

カテゴリ：タスク管理



ある場所で発生したイベントが計画された一連のアクションの実行につながる場合、インターネットがRub Goldbergマシンとして動作していると想像してください。 これは、 IFTTTサービスのおかげで本当です（「これなら、あれ」-「これなら、あれ」）。 特定のアクションが実行されたときに実行される単純な自動化されたワークフローを作成できます。 セットアップはシンプルで、可能性はほぼ無限です。そして最も重要なことは、iftttがEvernoteをさらに便利なものにしてくれることです。



iftttでタスクを作成する



Iftttは、選択した2つのサービスを接続し、レシピと呼ばれる自動化されたアクションアルゴリズムを作成します。 レシピはトリガー（トリガー）で構成され、 アクション（アクション）につながります。 独自のレシピを作成するか、他のiftttユーザーが以前に作成したレシピを使用できます。 iftttを使用する方法はたくさんあります。 Evernoteの優れたレシピを以下に示します。







Evernoteでiftttを使用する



開始するには、iftttアカウントを作成します。 次に、最初のタスクを決定します（可能性は無限です）。 これらのレシピでは、Twitter、Gmail、Foursquare、およびその他のサービスからのトリガーにより、Evernoteアカウントに新しいメモが作成されます。



Evernoteでiftttを操作するための可能なオプションの一部を次に示します。

  1. Gmailのメールごとに、重要とマークされた新しいEvernoteメモを作成します。 重要な手紙を紛失しないでください。必要に応じていつでも連絡できるように、Evernoteアカウントに直接送ってください。 [ レシピ ]
  2. Googleリーダーの記事にスターを付けるたびに、新しいEvernoteメモを作成します。 Googleリーダーの記事とブログエントリを記憶して、Evernoteがインストールされているデバイスからアクセスできるようにします。 [ レシピ ]
  3. Instagramの写真をEvernoteに保存します。 Instagramを使用して写真を撮って友達と共有する場合、iftttは写真のコピーをEvernoteアカウントに保存してすばやくアクセスできるようにします。 [ レシピ ]
  4. FlickrからEvernoteに写真を送信します。 新しく公開されたすべての写真をすぐにEvernoteアカウントに送信します。 [ レシピ ]
  5. EvernoteのFoursquareでチェックインのアーカイブを作成します。 世界中の動きの履歴を保持し、最近訪れた場所や帰りたい場所に関する情報を記録します。 [ レシピ ]
  6. お気に入りのツイートを適切なEvernoteノートブックに送信します。 Twitterでお気に入りの投稿をお気に入りに追加するたびに、別のEvernoteノートブックに自動的に送信できます。 [ レシピ ]




これらはiftttの使用例の一部に過ぎません。これによりEvernoteがさらに便利になります。 コメントでレシピについて教えていただければ幸いです。
