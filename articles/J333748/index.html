ICO：主な管轄区域と問題





約1年前、私はICOのトピック（誰も知らない-これはトークンの最初の配置である ：略語はIPOから取られている）の研究を始めた：結局のところ-無駄ではない。 今日、ICO市場は$ 1,000,000,000で参入しています（すでに参入しており、今年はまだ終了していません）。 同時に、暗号市場は100,000,000,000を超えました（ただし、現在-850 億という数字の周りで変動しています）。 もちろん、これは兆ではありませんが、すでに深刻です。



このメモでは、3つの質問に答えたいと思います。



  1. ICOはバブルですか？
  2. どの管轄区域で 、なぜICO実施者は登録されていますか？
  3. ICOには未来がありますか？


最初の質問に対する答えは簡単です：はい。 はい、ICOはバブルであり、これにはいくつかの理由があります。



  1. ほとんどのサービス （今日、綿密に分析-54、すべてレビュー-117）には明確な技術的基礎がなく、時には十分に開発された概念さえありません。作成者はホワイトブックとロードマップに限定され、ICOの開発とマーケティングにお金を節約します。



  2. 投機家が強すぎるのは 、市場が非常に小さく、価格を操作するのが非常に簡単だからです（以下に例を示します）。



  3. ドットコムまたはチューリップマニアをICOと比較すると、多くの共通点があります。 これに関する巨大な仕様。 決して終わらないという信念など



  4. ICOに参加するチームは非常に若く、これは最初はプラスの要因と見なされますが、運用上の問題が発生すると、その問題は3倍に増加します。 そして、誰かが資金を盗むとき、私はマッチプールや彼のような他の人についても話していませんが、そのチームワークは無視できる要素ではありません。 ブロックチェーンでは、球体が超技術的に構築されているため、それがより強く表示される可能性があります。これは、見かけ上逆説的に、人的要因からのリスクが増加するだけであることを意味します：DAOの例は誰もがよく知っていると思います。


もちろん、これだけではありません。しかし、評価におけるこれらの基準でさえ、ICOにおける資本の初期蓄積の不安定さを理解するには十分です。 さらに、現在（2017年には、ある意味）、このツールは誇大宣伝になりました：銀行、投資家、および同じIPOからのお金は多くかかりますが、クラウドファンディングからのお金（いくつかの説明的な現象があるので、詳細な分析は行いません） ICO-いくつか：それはすべてトークンの法的ステータスに依存します ）。



2番目の質問への答えはそれほど単純ではありません。2014年、 bitnovostiはこの市場で調査を実施しました。 そして、ここに描かれた絵があります：









研究を完全に「クリーン」にするために、マテリアルの素材は主にビットコインのスタートアップに関するものでした。 しかし、今日の状況はどのようなものか（私自身の考えを基礎として、以下に示すソースから統計を収集することもできます）TOP-30：









検討のために、以下のプロジェクトのみを取り上げました。



  1. それは単なるICOでした（たとえば、ビットコインとリップルにはありませんでしたが、さまざまな理由でありませんでした）。



  2. 企業に関するデータは閉じられていません：ホワイトペーパーおよび/またはWebサイトにあります：今回は、形式でひどいリクエストを行う必要がない場合は、一般ユーザーの代わりに会社のデータを見るために、意図的に深く掘り下げませんでしたSPARK;



  3. 大文字化により、一般的な結論を導き出すことができます。



  4. coinmarketcapのリストによると、私は41の場所に達しましたが、以下は含まれていませんでした。btc、ltc（このコインには、米国のLitecoin AssociationとシンガポールのLitecoin Foundationの両方があります）、monero; EOS、なぜなら 会社は、たとえばArdor（NXTの派生物）のように、すでに以前に組織されています。 バイトボール-プラットフォームは原則的にICOを意味しないため。



  5. さらに、中王国でICOの下に会社を設立するためには、非常に...予想外の投資家である必要があるため、中国のプロジェクトにも触れませんでした。


もちろん、 Waves 、 Golem （Golem Factory GmbHは2017年に設立されたスイスのLLCです）など、他のプロジェクトに関するデータをウェブ上でかなり見つけることができます。



問題は、まだほとんどないことです。



したがって、次のICOプロジェクトのアカウントに資金を提供する前に、次の資料を検討することをお勧めします。



  1. 世界のさまざまな国で暗号資産がどの程度理解されているか
  2. ICO評価の概要
  3. 弁護士の観点から見たICO


シンガポールでの登録 （ちなみに、別の最近のプロジェクトであるSONMも使用されました）は理解できます：6,000から10,000ドルで、政府が友好的である技術的に発展した国（詳細はこちら ）で会社がブロックチェーンのスタートアップのために一般的に設定されます。 さらに、ここでICOで受け取った法定資金への引き出しは、たとえばロシアと比較できません（この司法管轄区と比較するのに適切な場合）。 この点で、香港はシンガポールの完全に論理的な代替品です。 本では「暗号化」に対する態度について詳しく読むことができますが、両国では中立的であるとしか言えません。



米国は新興企業の故郷なので、そこで発生するプロジェクトは司法権を変更しません。 スイス、より具体的には、 ツークはクリプトドリナと見なされています（最初にV.ブテリンのEthereumはこれを「非難する」）。 この意味での英国は、ロシアの新興企業に適しています。



すべてが論理的なように見える？ はい。ただし、これらの管轄区域は当初、中小企業に焦点を合わせていなかった（時には高度な技術にさえ集中していない）ことを理解する必要があります。



要約すると、次のように聞こえます。



  1. イギリス、スコットランド-約1000-1500ドル
  2. シンガポール-（上記参照）約10,000
  3. 香港-2,000
  4. スイス-11,000-15,000


しかし、これらの数字は、Habréの多くにとって合理的なように思えるかもしれません。運用コストは別の問題です。 たとえば、香港とスコットランドでは、非報告に対する罰則が続きます。 しかし、これは多くのことを止めるわけではありません。それでも、ICOは数百万ドル、少なくとも数十万ドルを調達します。



しかし、Jupiterには許可されているが、雄牛には許可されているもの：ICOへのアクセスは、ユニークなサービスではなく、実際のセクターからのあらゆる種類のプロジェクト、または莫大なマーケティング予算のないプロジェクトです（そして、トップICOの場合は10万から500,000の常緑樹から成り立っています）



さらに、誇大広告と取引費用のファンは、直接金融とは言えませんが、考慮しません：たとえば、資金を不法に引き落とす必要がある場合（小学校-開発者、マネージャーに...またはさらに悪いことに-パートナーに）、その後、文書を認証するという問題がすぐに発生します。すべての都市から遠く離れて行うことができます。 モスクワとピーターは重要ではありません。地域のプロジェクトに関するものです。 たとえば、私たちの地域には、文書の翻訳を保証する準備ができていた公証人と、いくつかの公証人の行動に同意した公証人しかいませんでした。 プラス-アポスティーユ。

はっきりさせておくと、元々米国に縛られていたRipple社は、一度に70万人の死んだ大統領に罰金を科されました。 「... Ripple Labsは金融サービス会社として登録できませんでしたが、Rippleネットワーク上のデジタル決済プロセッサーであるXRPを販売しました。」 繰り返しますが、ICOに関する本には例があります。

最後に、最後の質問に答えるために、私は次のように言います。ICOには未来がありますが、下落は避けられません。 そして、いくつかの理由があります。



  1. より厳しい規制
  2. アルファ版およびベータ版を残さないことによるプロジェクトの失望
  3. 法的な観点からのプロジェクトの絶対的な（およびこれに関する別の資料があります）準備不足：この雷は最も聞こえやすいものの1つです。
  4. 技術的欠陥 。
