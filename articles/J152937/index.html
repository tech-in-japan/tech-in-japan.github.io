ビデオごとに手動でhtmlに情報を入力するのではなく、このプロセスを何らかの形で単純化する必要があることは明らかです。 素材のタイトルから、ビデオの名前を取得します。 説明にも問題はありません。 そして、残りのフィールドを埋めるために、「カスタムフィールド」機能を使用します。 Wordpressのこのような機能についてまだ知らない場合は、注意を払うべき時です。 簡単に言えば、任意のフィールドを使用すると、任意の材料に任意のエントリを追加できます。 たとえば、組織のディレクトリを標準ツールにするには、テキストエディターを使用して、組織のすべてのデータ（電話、住所、営業時間）を追加するか、サードパーティのアドオンを使用します。 また、任意のフィールドを使用すると、通常の記事を任意のデータのカタログに変換できます。



任意のフィールドに入力された情報（この場合、マイクロマーキングコード）をテンプレートの出力に使用できるように、このコードを使用します。 



 <?php $image = get_post_meta($post->ID, 'thumbnail', true); if($image !== '') { ?> <meta itemprop="thumbnail" content="<?php echo $image ?>" /> <?php } ?>




または 



 <?php $image = get_post_meta($post->ID, 'thumbnail', true); if($image !== '') { ?> <span itemprop=""><?php echo $image ?></a> <?php } ?>




ここで、 サムネイルは任意のフィールドの名前です。
