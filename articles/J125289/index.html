新しいIEEE 802.22標準が導入されました





この標準と以前の標準、たとえば802.11の主な違いは「範囲」にあります。 単一の送信局の信号は、そこにアンプやリピーターがなくても、最大100 kmの距離で捕捉できます。 印象的でしょ？ 新しい規格は、IEEEの国際協会（電気電子技術者協会）によって導入されました。 おそらく802.22標準は、現在の802.11 a / b / g / nを置き換えるものになるでしょう。



一般に、無線機器の大半は802.11 a / b / g / n標準に基づいて動作するため、ほとんどのユーザーのニーズを満たすため、これが発生するとすぐに発生します。 一方、技術は非常に急速に変化しており、一部の国では、数年前から標準であった同じアナログテレビがデジタルテレビに置き換えられました。 同じことは、標準の802.11 a / b / g / nでも数年間起こります。



802.22ワイヤレス規格が動作する周波数範囲は、54 MHz〜698 MHzのスペクトルをカバーしています。 したがって、データは、単一の基地局に基づいて、最大100キロメートルの距離で最大22 Mbpsの速度で送信できます。 おそらく、この特定の規格は、まだインターネットに接続していない住民がいるグローバルネットワークにリモート地域を引き付けるのに役立つか、ネットワークへのアクセスに多額のお金を払うでしょう。



新しい標準が承認されたので、商業組織はそれを使用して、802.22テクノロジーをサポートする機器を作成できるようになります。 メーカーが試してみると、他の多くの技術で何度も起こったように、新しい標準がすぐにあなたと一緒に私たちの生活に入ります。
