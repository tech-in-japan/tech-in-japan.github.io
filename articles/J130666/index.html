Zend Framework 2.0.0 beta1がリリースされました

Zend Framework 2.0.0の最初のベータ版がリリースされました。 インストールの配布と手順は、 http ： //packages.zendframework.comで入手できます。

彼らが公式のZendウェブサイトに書いているように、これは一連の計画されたベータリリースの最初のベータです。 将来、開発者は「gmail」スタイルのリリースに固執する予定です。 つまり、新しいベータ版ごとに新しい機能と機能が追加され、少なくとも6週間ごとにリリースされます。



2.0.0beta1の新機能：

  • リサイクルされた新しいオートローダー：

    Zend \ Loader \ StandardAutoloader

    Zend \ Loader \ ClassMapAutoloader

    Zend \ Loader \ AutoloaderFactory
  • 再設計されたプラグインブローカー：Zend \ Loader \ BrokerおよびZend \ Loader \ PluginBroker
  • 再設計された例外システム。 新しいシステムでは次のことができます。

    特殊なタイプの例外をキャッチ

    コンポーネントの例外をキャッチ

    SPL例外をキャッチ

    基本型の例外をキャッチ
  • 書き換えられたセッションのメカニズム
  • 再加工されたビューコンポーネント
  • ヘルパーは、可能であれば__invoke（）マジックメソッドを使用するように作り直されました。
  • 再設計されたHTTPコンポーネント
  • 新しいコンポーネントZend \ Cloud \ Infrastructureを追加しました
  • 新しいEventManagerコンポーネントが追加されました
  • 新しいDependency Injectionコンポーネント（Zend \ Di）が追加されました
  • 新しいCodeコンポーネントが追加されました：ReflectionおよびCodeGeneratorコンポーネントの再設計されたバージョンと、新しいScannerコンポーネントが含まれています


ZendのWebサイトによると、上記のコンポーネントは主にZend Framework 2およびいわゆるフレームワークコアの基礎を提供します。 しかし、beta1の機能は、新しいMVCレイヤーがそれらに基づいて構築されることです。

  • Zend \モジュールアプリケーションアーキテクチャを開発するためのモジュール。
  • Zend \MVC。HTTP、EventManager、およびDiのコンポーネントに基づいて再設計および作成されました。


開発者は、ZF2に関心のあるすべての人を支援するために、 テストアプリケーションフレームワークとモジュールテストフレームワークを作成したとも書いています 。
