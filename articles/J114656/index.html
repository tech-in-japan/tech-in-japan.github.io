G'MIC for GIMP-優れたプリセットとエフェクトのセット、独自のPPAを取得

クリック可能




G'MIC（GREYCのMagic Image Converter）は、フィルターとエフェクトがプリセットされたツールのセットです。 初心者でも非常に使いやすいです。



ここでG'MICに提示さ190以上のプラグインのほんの一例です







G'MICはsourceforge.netからダウンロードでき、すべてのOSの最新バージョンが表示されます。 または、Linuxを使用している場合は、次のようにPPAを接続します。



sudo add-apt-repository ppa：ferramroberto / gimp

sudo apt-get update

sudoのGMIC GIMP-GMICをインストールapt-getを



G'MICには悪くないflickrバンドがあります。 そこで、マニュアルを読んで助けを得ることができます。
