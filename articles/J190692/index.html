MicrosoftはSkypeをOutlook.comと統合します









Microsoft は 8月19日に、米国、英国、ドイツ、フランス、ブラジル、カナダでのSkypeとOutlook.comの統合の開始を発表しました。



SkypeとOutlook.comの組み合わせは、2つの最も人気のあるMicrosoftサービスを統合し、電子メールシステムを新しいレベルに引き上げます。 これらの革新は、Gmailとのさらなる競争の柱となるでしょう。 Outlook.comには、Hotmailユーザーの大規模な流入（同社は時代遅れになっています）により拡大しているユーザーベースがありますが、有機的な成長も堅調です。



MicrosoftはSkypeに数十億ドルを支払い、ビデオ通信が急速に成長すると予想しています。 一方、Outlook.comは既に4億を超えるアクティブアカウントを持っているため、これら2つのシステムを組み合わせることにより、大幅な相互成長が可能になります。



もちろん、Gmailはこの「突っ込み」をGoogleハングアウトで解決します。 それでも、その面白さと斬新さにもかかわらず、名声、信頼性、ユーザーの信頼という点で、Skypeに匹敵するものはまだありません。



公式発表で、マイクロソフトはIpsos Public Affairsが実施した調査の結果に注目しました。 回答者の約76％が、メールのやり取りが電話またはビデオ通話に頻繁に使用されることを確認しています。 そして今、Outlook.comのユーザーはこのような機会をすぐに手に入れることができます。



2011年5月10日に、Microsoftは85億ドルでSkype Communicationsを買収したことを思い出してください。 その後、穏やかな期間が続き、統合するためのステップは一般に公表されませんでした。



そのため、2013年2月15日に、 Microsoftがベテランである Windows Live MessengerをSkypeに有利に閉鎖することを発表しました （Skypeを中国市場にのみ残す）。



その後、8月15日に、新しいWindows 8.1にSkypeがプレインストールされるというニュースが出ました。



これで、Skypeが巨人のオンラインサービスに統合されることがわかりました。
