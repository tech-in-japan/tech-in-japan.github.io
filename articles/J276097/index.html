ゲームデザイン：ゲームを超えて

ゲームを作成する上で最も重要なスペシャリストであるゲームデザイナーの仕事は何ですか？ この質問への答えは、伝説的なミリタリーMMOアクションWorld of Tanksの作成者であるWargamingの「 Game Design：Beyond the Game 」コースの講義です。







コース講師、 Wargaming Alexey Kalininのエグゼクティブプロデューサーによると、高品質のゲームを作成することは不可能な3種類のゲームデザイナーがいます。数学者、メカニック、脚本家です。







順番に始めましょう。 ゲームを作成するときに数学者が必要なのはなぜですか？ 大まかに言って、数学者は、アイデアのコストから各キャラクターをゲームに導入した結果まで、すべてを正確に計算できる人です。 アレクセイは不思議な話をします。「GDチームで働いたら。 私たちは皆、朝のいつもの時間にオフィスに来ました。 しかし、あるVasilyが私たちと一緒に働いてくれました。 夕方8時か10時に出発したとき、彼はまだ職場に残っていました。 Vasyaは怪物に見え、エイリアンのように話しました。 プロジェクトマネージャーに次のように尋ねると、「順序と正確さが大好きです。 なぜ彼にこんなことをさせたのですか？」「とても簡単です」と彼は答えました。 「この男は神からの数学者です。 彼は、新しいスキルを計算し、ゲームの革新にどのような影響を与えるかを言うことができる唯一の人です。 彼はゲームとその数学的基礎を知っています。 彼の死を追い払うことも同様です！ それなしでは、新しいタイプの戦闘、新しいヒーロー、武器を導入することはできません。







数学が好きではありませんか？ おそらくあなたの強みは想像力ですか？ ゲームを作成するとき、メカニックなしではできません。計画されたシナリオを翻訳する方法を開発する人です。 ゲームは何十年も前から存在しているため、新しいメカニズムを発明することは非常に難しく、この種のゲームデザイナーの需要はますます高まっています。 ここで顕著な例は、シドマイヤーです。 90年代に、彼は、ウォーゲーマーの間で広く知られているCorsairs、Civilization、Getigsbergなど、互いに著しく異なるいくつかの素晴らしいプロジェクトに取り組みました。







これらはすべて異なるゲームです。 それらは設定とジャンルの両方で異なりますが、それらは一つのことで結ばれています-エレガントで興味深いゲームソリューションは、その後、他の多くのゲームのために何百人ものゲームデザイナーによってクローンされました。



もっとできると思いますか？ その後、脚本家として自分を試してみてください。 同意します。エキサイティングなスクリプトなしではゲームは完成しません。 そして今、新しいメカニックを思い付くのがそれほど簡単ではないなら、全く新しい世界を創造しようとした人がどのような創造性の苦しみを待っているか想像できます！ 正直なところ、脚本家はもはやゲームデザイナーではありません。 ますます、プロの作家がプロットと対話を担当する主要なプロジェクトに招待されます。 なぜこれがそんなに重要なのですか？ BioWarゲームの最後のシリーズであるMass EffectとDragon Ageを思い出しましょう。







プレイヤーはそれらを複数回渡します。 彼らは何度も何度もゲームの世界に戻り、ストーリーを生きるためのさまざまなオプションを試します。他のキャラクターのためにプレイし、プロットフォークで新しい重要な決定を下します。 そして、あなたが自分自身を見つけて知りたいと思うこれらの真に生きている世界のそれぞれ：「もし..？がどうなるでしょうか」は一人によって作成されました。 おそらく、このプロジェクトへの脚本家の参加の割合は、彼が発明した世界を描く50人のプログラマーまたは30人の芸術家によって行われた仕事に匹敵しません。 しかし、ゲームの成功の大部分は、この人が考えているプロットと対話に依存します。



メカニック、脚本家、数学者は誰ですか？ 答えが何であれ、私たちの新しいコース「 Game Design：Beyond the Game 」は、自分のゲームをどこから始め、成功するプロジェクトにするかを決めるのに役立ちます。 コースはすでに進行中です！
