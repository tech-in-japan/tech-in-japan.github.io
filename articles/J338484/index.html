富士通は、人工知能の研究のためのスーパーコンピューターを構築します

先週、日本の産業技術総合研究所（AIST） は富士通を選んで ABCI（AI Bridging Cloud Infrastructure）スーパーコンピューターを構築しました。 これは、人工知能、ロボット工学、自動操縦車、医学の分野での研究のためのプラットフォームになります。



ABCI は 、37ペタフロップスの速度で倍精度で演算を実行すると想定されています。 コンピューターは 2018年に動作を開始し 、日本で最速になります。

中身は何ですか



このシステムでは 、1088個のFujitsu Primergy CX2570サーバーを使用します。各サーバーには2つのIntel Xeon Goldプロセッサーと4つのNVIDIA Tesla V100 GPUが搭載されています。 ローカルI / Oを高速化するために、スーパーコンピューターには3D NANDテクノロジーを使用するIntel SSD DC P4600 NVMeカードが搭載されます。



NVIDIA VoltaアーキテクチャとTesla V100アクセラレータは、動作中に他の要素よりも熱くなるため、液体冷却が必要です。 状況を改善するために、Fujistuは矛盾したアプローチを使用しています-お湯で冷却します。



この方法は、オペレーターが使用するチラーの数を減らすか、まったく使用しないのに役立ちます。 富士通は 2015年に、Primergyサーバーが冷却コストを半減すると述べました。 サーバーのPUE比率は1.06でした。 これは、「チップに直接」（チップに直接）液体冷却の技術を使用して達成されました。



ABCIの別のソリューションは、ラックレベルの冷却です。50kWの液体冷却と10 kWの空気です。 温度を維持し、余分な熱を除去するために、冷却ユニットがプロセッサに取り付けられています。 冷却スキームについては、 こちらをご覧ください 。



置く場所



ABCIプロジェクトの予算は1億 7,200万ドルです。 この金額の1,000万は、システムの新しいデータセンターの構築に使用され、東京大学のキャンパスに配置されます。 データセンターの最大容量は3.25 MW、冷却設備の容量は3.2 MWになります。 データセンターのフロアは具体的です。 ラックの開始数は90個です。18個はデータストレージ用で、72個はコンピューティングタスク用です。



データセンターの建設は今年の夏に開始され、ABCIシステム自体は2018年に開始されます。



ペタフロップスが多い人



スーパーコンピューターの作成は、軍拡競争のようなものです。 最速のスーパーコンピューターは、中国のSunway TaihuLightと見なされています-そのパフォーマンスは93ペタフロップスです。



Tianhe- 2-34ペタフロップスの性能を持つ中国のスーパーコンピューターがそれに続きます。 3位はスイスのPiz Daintとその6.3ペタフロップスです。 続いて、アメリカンタイタン （17.6ペタフロップス）、 セコイア （17.1）、コリ（14）が続きます。 13.5ペタフロップスのパフォーマンスで、日本の上位7つのOakforest-PACSを閉じます。



ロシアはこのランキングで59位になります。 この国の代表はロモノソフ-2で、生産性は2.1ペタフロップスです。



スーパーコンピューターはさまざまな目的に使用されます。 それらの最も強力なものを使用して、科学者は宇宙の仮想モデルを構築しました。 Tianhe-2は、中国の極秘データと国家そのものを保護します。 Piz Daintのアプリケーションの1つは、高エネルギー物理学の分野でのモデリングです。



米国原子力安全局は、セコイアを使用して核爆発をモデル化し、他の科学者は宇宙論的および人間の心臓のシミュレーションを使用しています。



タイタンの助けを借りて、彼らは科学研究を行っています。原子炉内の中性子の挙動のモデルを作成し、気候変動を予測します。 Oakforest-PACSは、スーパーコンピューターに興味のある学生を研究し教育するために日本で使用されています。



エキサフロップスの時代



2018年、米国は200ペタフロップスの容量を持つスーパーコンピューターであるSummitを開始します。 これに対して、中国の科学者は Tianhe-3に答えます -そのパフォーマンスは1エクサフロップスに等しくなります。 このスーパーコンピューターのプロトタイプは2018年に登場します。 2020年には、フランスがレースに参加します。アトスは 、エクサフロップスプロジェクトブルセカナの立ち上げを計画しています。



しかし、専門家は、エクサフロップスへの大規模な移行は、過度のエネルギー消費と過剰な熱につながることに注意しています。 exaflopsで動作するには、世界のコミュニティは、ハードウェア、ソフトウェア、アルゴリズム、アプリケーションなど、コンピューターのエコシステム全体の変化を調整する必要があります。



多くはすでに太陽エネルギーと高度な冷却システムに切り替えています。 しかし、強力なスーパーコンピューターの大量配布には、これでは十分ではありません。



国立研究所のホルスト・サイモン（ホルスト・サイモン）によると 。 バークレーのローレンス、難しさは、同時に多くの科学的ブレークスルーを作らなければならないことです。 最初に、無駄な余分な熱を無駄にせずに、エネルギー消費を削減する方法を理解する必要があります。 そうして初めて競争できます。
