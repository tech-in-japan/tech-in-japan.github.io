GeekBrainsはC ++プログラミング言語を教えます







教育ポータルでは、 C ++トレーニングコースの開始を発表しています。 「十字軍」の学生は、ソフトウェアの設計と開発からゲーム開発まで、あらゆるIT業界で需要があります。 C ++は、複雑な問題の解決に最適です。たとえば、Google ChromeやFirefoxブラウザー、伝説のWorld Of Warcraftなど、負荷の高いシステムはすべてC ++で書かれています。 hh.ruによると、ロシアでは現在、モスクワの1,000人以上を含む、C ++プログラマの空席が3000人以上開いています。







トレーニングには9か月かかります。 このプログラムには、Linux、Git、CおよびC ++の基礎、Cのアルゴリズムと構造のコース、C ++のコンパイルとメモリ管理のコース、オブジェクト指向プログラミングおよびその他の科目の9つの実践的なコースが含まれます。プログラマが直面するタスク。 正常に学習を完了し、最終テストとプロジェクトに合格した学生には、受講した9つのコースごとに州の証明書と証明書が授与されます。







すべての学生は、GeekBrainsで3か月のインターンシップが保証されています。これにより、キャリアを開始するために必要なスキルを身に付けることができます。 優秀な学生は、GeekBrainsパートナーと協力したり、元卒業生が設立したスタートアップに参加したりすることができます。 これを行うには、テストタスクを完了する必要があります。







レッスンは、少なくとも5年の経験を持つ開発者の練習によって教えられます。 クラスは週に数回、夕方にウェビナーとビデオ講義の形式で開催されます。 スキルを強化するために、各レッスンの後に宿題が与えられます。 各生徒には、トレーニングの困難な瞬間を理解するのに役立つ個人指導が与えられます。







学生は実践的なタスクに取り組み、ポートフォリオの3つのプロジェクトを完了します。これには、大量のデータの集約と視覚化のためのプロジェクトと、PNMの輝度アナライザーが含まれます。







クラスは11月5日に始まります。 教育は有料で、詳細なプログラムはこちらにあります 。
