Materials from android-mitap: tools for autotests | Yandex Mapkit 3 | application design | Server Driven UI

Hello! This is a post-report from the android mitap, which recently took place in the Avito office. With colleagues from Ozone, Surf, Revolut and Avito, we discussed Yandex Mapkit 3 integration, how to quickly launch features with Server Driven UI, which tools to choose for auto-tests and how to make the application design memorable.







Come under the cat if you are interested in these topics. Inside the post - all the materials of the meeting: video reports, presentations of speakers, guest reviews of the meeting and links to the photo report.













Reports



Quick feature launch with Server Driven UI. Alexander Sviridov, Ozon



A year ago, Alexander's team began to build the entire UI from widgets on the backend. During this time, several tools have been made with the Server Driven UI approach. The report will be very useful to those who create their own framework or e-commerce platform.
