PHP 20周年





この日、1995年6月8日に、最初のパブリックリリースであるPHP / FI 2.0がリリースされました。 著者はデンマークのプログラマー、ラスマス・ラードルフでした。



これに先立ち、1994年にLerdorfは、訪問者を表示し、HTMLドキュメントテンプレートを処理するオンライン履歴書に登録するためのPerl / CGIのスクリプトセットを作成しました。 ラードルフは、個人用ホームページセットを呼び出しました。





1997年、長いベータテストの後、Cで記述されたハンドラーの2番目のバージョン-PHP / FI 2.0がリリースされました。 世界中のすべてのインターネットドメインの約1％（約5万）で使用されていました。



現在、PHPは世界中のサイトの81.9％で使用されています。



バージョンPHP 3.0は、プログラミング言語の最新の外観とスタイルを決定するために大幅に改訂されました。 1997年、イスラエルの2人のプログラマーであるAndy GutmansとZeev Suraskyがインタープリターコードを完全に書き直しました。 PHP 3.0は1998年6月に正式にリリースされました。



PHP 3.0の強みの1つは、追加モジュールでカーネルを拡張できることです。 その後、拡張機能の作成インターフェイスは、モジュールで作業している多くのサードパーティ開発者をPHPに引き付けました。これにより、PHPは膨大な数のデータベース、プロトコルで動作し、多数のAPIをサポートできます。 多数の開発者が、言語の急速な発展と人気の急速な成長をもたらしました。 このバージョンから、頭字語phpは、廃止された「個人用ホームページ」の代わりに「PHP：hypertext Preprocessor」の略です。



1998年の冬、PHP 3.0の公式リリースのほぼ直後に、Andy GutmansとZeev SuraskyがPHPのコアの処理を開始しました。 タスクには、複雑なアプリケーションの生産性の向上と、PHPコードの基礎のモジュール性の改善が含まれていました。 Zend Engineと呼ばれる新しいエンジンは、タスクにうまく対処し、1999年半ばに最初に導入されました。 このエンジンに基づいており、追加の機能セットを備えたPHP 4.0は、2000年5月に正式にリリースされました。 パフォーマンスの向上に加えて、PHP 4.0には、セッションサポート、出力バッファリング、ユーザー入力を処理する安全な方法、いくつかの新しい言語構成など、いくつかの重要な革新がありました。
