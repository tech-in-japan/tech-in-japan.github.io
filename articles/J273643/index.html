「最適化する！」





今日は最適化についてお話したいと思います。 しかし、ウェブやアプリケーションの最適化についてではなく、日常のインターフェースやホームインターフェースの最適化についてです。 いいえ、クレーンについて、またミキサーを発明することによって信じられない飛躍的な人間性がもたらしたものについてはもう話しません。 しかし、私のトピックは近いです。 私たちが使用することに慣れている私たちを取り巻くものについて話すことに近いので、それらの美しいものを見ることはなくなります-最適化！



ちなみに、Figmaを使用する場合は 、 既製の設計システムに注意することをお勧めします 。 フリーランサーが1か月あたりの注文を増やすのに役立ち、プログラマーは自分で美しいアプリケーションを作成できます。また、チーム作業用の既製の設計システムを使用して、チームが「スプリント」スプリントを迅速にリードします。



また、深刻なプロジェクトがある場合、当社のチームは、ベストプラクティスに基づいて組織内に設計システムを展開し、Figmaを使用して特定のタスクに合わせて調整する準備ができています。 Web /デスクトップ、および任意のモバイル。 また、React / React Nativeにも精通しています。 Tへの書き込み： @kamushken




私はあなたが周りを見回して、自然が私たちを動かし、人生を楽にするさまざまな方法を考えさせてくれることを賞賛することをお勧めします。 すべての栄光の進化！ もちろん、これらのガジェット、スマートフォンなど、私たちの生活を最適化するものはすべて興味深いものです。 かつて私たちについてすべてを学んだ（独自のイニシアチブで）特性をスマートフォンに与えようとしているIT業界のモンスターのアイデアも、「より良く生き続ける」方法を教えてくれるでしょう。 しかし、私はそれについても話していません。



私たち全員がクリエイターであり、イノベーターであることを思い出させてください！ 私たち一人一人で、少しの天才が目覚めることができ、毎日の家事のプロセスをより最適にすることができます。 私たちの多くは人生を頻繁に最適化しているので、それがどれほど深くてエキサイティングであるかを重要視し、正当な認識を持っていません。



人生の生きた実例。 文字通り熱い追求。 最適化のためにすべてが同じであると仮定すると、居住地を変更し、職場に近づきます。 珍しい内部構造の高い16階建てのパネルハウス：階段に入ると、左側に3つのエレベーター、右側に3つのエレベーターがあります。 実際、これは現在亡くなった砂糖工場の元のホステルです。 そして今、多くの本格的な民営化されたアパートがあります。 珍しいことは、左のエレベーターが奇数階に、右が偶数階に移動することです。



これは、オンラインストアで何かを購入するときにコメントで言及しなければならない最初の瞬間です。 2つ目のポイントは興味深いフロア構造です。エレベーターから「脱衣室」に入り、そこから10〜12個のアパートが点在する長い廊下に入ります。 宅配便業者は長時間さまようことができます。 特に待合室にいるとき、彼はアパート番号と電話が入った「記念」プレートに出会う。







それは理解を追加するように思えます：数字とボタンの左の列は、左側のすべてのアパートメントを示し、右側のアパートメントの右側を示していると仮定したいと思います。 しかし、実際にはそうではありません。 つまり、人は間違った道をすぐに許可されます。 それから私は介入することを許可しました：







エレベータに浸る段階でも「迷子になる」チャンスを考えて、私は宅配便業者に戸別のドアを見せて、迷子になった人からの迷惑な電話から自分を救うことによって私の人生を最適化する方法を見つけました。



寿命の最適化の別の例。 3年前に自転車を買いました。 特にサイクルブームと近年の普及を考えると、ありふれたおなじみのことです。 十種競技と安価。 そして、私は一年中乗ります。 注目してください、それは「乗り」ではなく「乗り」です。 この最適化もなぜですか？ かつては往復で最も近い店への旅行に30分を費やしていたからです。 自転車での移動時間の合計は、わずか10分以上になりました。



私はこれらのことをデザイン思考の産物と考えるのが好きです。 別の、より普遍的な平面で。 そして今、最初から私の呼びかけに従って、周りを見ましょう。 以下に簡単な例を示します。



トイレットペーパー。 次のロールが終了したときに、メーカーが段ボールフレームを取り除くプロセスを最適化してくれたことに感謝しましたか？ フラッシュスリーブは素晴らしいです！ トイレに投げて洗い流しただけです。 それは非常に日常的で、デザイン思考の素晴らしい成果だと思う人はほとんどいません。 かつてこの日常は好奇心をそそられました。 私はトイレで「ひじに」手をつけ、この機能をサポートしていないメーカーの完成したロールを家で見つけました。
