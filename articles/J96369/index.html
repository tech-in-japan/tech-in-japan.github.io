XLSファイル生成

当局は、XLSレポートファイル（注文のアップロード、注文した商品、顧客の概要など）の生成を強制しました。 プレーンテキスト（CSV形式、「;」-区切り記号）の生成に問題はありませんでした。 しかし、MAC OSのボスは、彼らが彼女をプッシュしようとしたことを理解したくなかったと指摘します。 解決策は、異なる形式を生成することです。 私はhtmlまたはxmlになりますが、要件の1つは「美しいフォントとカラーセルの塗りつぶし」でした。 一般的にはどこにも勝るものはありません。



公平に言えば、Excel形式の生成は、Habr（ http://habrahabr.ru/blogs/php/18726/ ）およびOpenNET（ http://www.opennet.ru/base/ dev / php_gen_excel.txt.html ） 。3つのテストされた軸（mac、win、ubuntu）で完全に開かれた作成済みXLSファイル。



また、ハブでは、多くの人がPHP Spreadsheet_WriteExcelに関するリンクを急いだと言わなければなりませんが、何らかの理由でそれを使用できませんでした。

その結果、偶然http://phpexcel.codeplex.com/につまずきました-PHPExcel-OpenXML-Excel2007ドキュメントをPHPで作成-スプレッドシートエンジン。

一般的に必要なすべてのもの（IMHO）：通常のドック、多くの例、そして最も重要なこと-多くの機会。



私は自分のタスクで最小限を使用しましたが、それでも：

1）セルサイズの設定

2）テキストの色

3）テキストサイズ

4）フォント

5）領域を色で塗りつぶす

6）フレームワーク

7）数式セルへの挿入

8）アクティブな（クリック可能な）リンクのセルへの挿入

9）異なるシートで作業する



まあ、はるかに。

それを理解するものは何もありません!!! 多くの例では、必要なスクリプトを作成するためにほとんどコピーアンドペーストできます。

/** Error reporting */

error_reporting(E_ALL);



/** PHPExcel */

require_once '../Classes/PHPExcel.php';

// Create new PHPExcel object

echo date('H:i:s') . " Create new PHPExcel object\n";

$objPHPExcel = new PHPExcel();



// Set properties

echo date('H:i:s') . " Set properties\n";

$objPHPExcel->getProperties()->setCreator("Maarten Balliauw")

->setLastModifiedBy("Maarten Balliauw")

->setTitle("Office 2007 XLSX Test Document")

->setSubject("Office 2007 XLSX Test Document")

->setDescription("Test document for Office 2007 XLSX, generated using PHP classes.")

->setKeywords("office 2007 openxml php")

->setCategory("Test result file");

// Add some data

echo date('H:i:s') . " Add some data\n";

$objPHPExcel->setActiveSheetIndex(0)

->setCellValue('A1', 'Hello')

->setCellValue('B2', 'world!')

->setCellValue('C1', 'Hello')

->setCellValue('D2', 'world!');



// Rename sheet

echo date('H:i:s') . " Rename sheet\n";

$objPHPExcel->getActiveSheet()->setTitle('Simple');

// Set active sheet index to the first sheet, so Excel opens this as the first sheet

$objPHPExcel->setActiveSheetIndex(0);

// Save Excel 2007 file

echo date('H:i:s') . " Write to Excel2007 format\n";

$objWriter = PHPExcel_IOFactory::createWriter($objPHPExcel, 'Excel2007');

$objWriter->save(str_replace('.php', '.xlsx', __FILE__));

// Echo memory peak usage

echo date('H:i:s') . " Peak memory usage: " . (memory_get_peak_usage(true) / 1024 / 1024) . " MB\r\n";



// Echo done

echo date('H:i:s') . " Done writing file.\r\n";
