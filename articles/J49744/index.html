Microsoft Web Platform Installerリリース

Microsoft Webプラットフォームインストーラー （WPI）は、インターネットインフォメーションサービス（IIS）6.0および7.0、SQL Server 2008 Express、.NET Framework 3.5 SP1、ビジュアル開発ツールなどのMicrosoft Webプラットフォームコンポーネントをすばやくインストールするためのツールです。 Web Developer 2008 Express SP1および追加のプラットフォームコンポーネント（IISのパス書き換えモジュールなど）。 WPIはWindows XP、Vista、およびWindows Server 2008で実行されます。WebプラットフォームコンポーネントとWPI自体は無料であり、商用開発が可能です。







Web用のMicrosoftプラットフォームは常に進化しており、新しいテクノロジとコンポーネントの数は非常に急速に増加しています。コンポーネントのダウンロードとインストールには時間がかかるため、他の何かに有利に使うことができます。必要なコンポーネントのインストール。 WPIを使用すると、開発用のサーバーまたはコンピューターをすばやくセットアップできます。これにより、経験豊富な開発者がより早く始めることができ、Microsoftプラットフォームを学習し始めたばかりの開発者は、プラットフォームの学習に必要なすべてのコンポーネントを取得できます。 WPIは、インストールのためのいくつかのオプションを提供します。







  • 完全-すべてのプラットフォームコンポーネント
  • ASP.NET Developer-ASP.NETアプリケーションの開発と実行に必要なコンポーネントのみ
  • カスタム-インストールするコンポーネントを選択する機能。たとえば、SQL Serverをデータベースとして使用してPHPをホストする場合、SQL Server Driver for PHPを選択できます。
生活を簡素化するために、WPIユーザーには、カスタムインストールの場合、次のコンポーネントセットがあります。







アプリケーションをインストールするために、 Microsoft Web Application Installerという同様のツールが作成されました 。これにより、最も人気のあるアプリケーションをインストールできます（リストが補充されます）。







スクリーンショットでわかるように、PHPアプリケーションもサポートされており、必要なコンポーネントが存在するかどうかの検証が実行されます。
