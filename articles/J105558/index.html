Skype for Android

Skypeは本日（最終的に）Android用のバージョンをリリースしました。これは、 ボルシェビキのガジェットが長い間話し合ってきたニーズです。

Android向けSkypeでは、次のことができます。

  • 3GまたはWiFi接続を使用して他のSkypeユーザーに電話をかけます。
  • Skype Out経由で電話をかける
  • グループチャットを含むチャット
  • Skypeオンライン番号（Skype-In）への通話を受信する


このバージョンの機能のうち、注目に値するものは次のとおりです。

  • Skype for Androidは、Skype SILK独自のコーデックを使用しており、従来の携帯電話通信よりも優れた音質を提供します。 SILKがどのように異なるか、そしてそれが人間の耳にとってどれほど顕著であるかについては、 こちらをご覧ください 。
  • Skypeはスマートフォンのアドレス帳と同期できるため、異なる連絡先リストを切り替える必要がありません。 さらに、Skypeの実行中は、メインの連絡先リストから直接Skypeを呼び出すことができます。










  • アプリケーション自体の連絡先リストで、それらの一部をお気に入りとしてマークすることが可能になりました。これにより、リストの一番上に移動し、アクセスしやすくなります（携帯電話では、これは非常に便利です）。








  • 電話で話している間に行われた着信Skype通話は、（設定に従って）不在または転送としてマークされます。 Skypeで話しているときに電話がかかってきた場合、この電話を受けることができます。この場合、Skypeの会話は保留モードになります。
  • Skype for Androidユーザーは、自分で電話会議を開始することはできませんが、他の誰かが電話会議を開始した場合、それに参加することができます。


技術要件：

Skype for Androidは、Androidバージョン2.1以降で動作します。

デバイスのプロセッサー周波数は600 MHz以上である必要があり、画面サイズは480 * 320、800 * 480または854 * 480です。 600 MHzおよびその他の画面解像度よりも遅いプロセッサを搭載したモデルでは、Skypeが正しく動作しない場合があります。

Skypeは、HTC Desire、HTC Legend、Motorola Milestone XT 720、Motorola Milestoneモデルでテストされました。
