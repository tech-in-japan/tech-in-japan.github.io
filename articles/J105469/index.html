VKontakte Open APIおよびSoundcloudによるmp3検索を使用したSeesuのオンラインバージョン

Sisuは、音楽を簡単に検索して聴くための小さなアプリケーションです。 last.fm、vkontakte、および最近soundcloudの組み合わせ。 html / css / jsで書かれています。



システムのハブへの最後の投稿以来、 プロジェクトのWebサイトを立ち上げただけでなく、 VKontakte APIのおかげで、デスクトップだけでなくモバイルでも動作するオンラインバージョンです。Android、iPhone、iPad、Opera Miniがテストされますサウンドクラウド）。 さらに、iPhone上のマルチタスクで動作します（基本的にマルチタスクを持たないiPhone 3gを含む）。 オンライン版に加えて、オペラ用のウィジェット、クロム用の拡張機能、アップルダッシュボード用のウィジェットがあります。









インターフェイスは小さな画面用にのみ最適化されていますが、一般的にはまだそれら向けに設計されていません。



ブラウジング、トラックの優先読み込み、soundcloudでの検索（これまでオンラインバージョンでのみ利用可能）、その他さまざまな小さなものなどのイノベーションの中で、「トラックのレベルを上げる」ことを主要なイノベーションと考えています。



sisuインターフェースは、スケーラブルなインターフェースでオブジェクトを 検索および表示するような「 ラスキン 」クジラに基づいて構築されていることを思い出してください。



したがって、検索クエリは検索結果に拡張されます。 検索結果はプレイリストに展開されます。 また、プレイリストのトラックは、詳細な説明に展開されます。 「トラックのレベルを上げる」という革新により、通常は特定のトラックのボタンを表示できるだけでなく、このトラックのコンテンツをコンテキストに応じて表示することもできます。 たとえば、スケーラブルインターフェイスのRaskinモデル内の写真、ビデオ、または同様のトラック。



Sisuの主なプラスは優れた検索であり、主なマイナスはプレイリストの欠如です。
