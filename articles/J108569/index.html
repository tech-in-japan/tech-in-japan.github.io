州はそれ自体が電子文書管理の発展を刺激しています。最近、既存の罰金が真剣に引き上げられ、文書管理に関連する違反に対して新しい罰金が導入されました。 罰金が50ルーブルであった場合、今では数十万に「入る」ことができます。



誤った文書管理会社のためのロシア連邦の行政犯罪のコードによると、ルーブル、そして非常に大きな責任があります。 例えば、13.25条は、書類の保管に関する法律の要件の違反には「…2000年から500から5000ルーブルまでの職員に対する罰金がかかる」と報告している。 法人の場合-200,000から300,000ルーブル ''

そして、彼らが違反を時間内に排除することができなかった場合、再び大きな罰金。 これはアートによって報告されています。 19.5。 同じコード。 法的要件を順守しなかった場合、「20,000から30,000ルーブルの役人に対する罰金。 法人の場合-500,000から700,000ルーブル。」

そして、最も重要なことは、違反が重要でない場合、彼らがあなたを後悔することを期待することです、それは価値がありません-少なくとも先例の権利はありませんが、すでに具体的なケースと決定がなされています。 会計書類の非保存については、会社は20万ルーブルの罰金（ケース番号A32-4676 / 2010-51 / 155-74AZH）、および株式会社の場所での書類の保管-50万（ケース番号A65-16109 / 2009）（出典） -kad.arbitr.ru ）。 後者からは、髪の毛が逆立ちしている-結局のところ、多くの企業は彼らの法的住所がどこにあるか見たことがない。

アート。 25.独占禁止当局に情報を提出する義務。 この法律は2009年8月23日に施行されました。

2010年9月2日以降、税金による罰金が増額されました。 必要な書類を提出しなかった場合（第126条）、罰金が50ルーブルになる前に、書類ごとに200ルーブルを支払う必要があります。 税務当局への報告の不履行（第129.1条）は、現在では1,000ルーブルではなく、5,000ルーブルを脅かしています。1年以内に違反が繰り返された場合、5,000から20,000ルーブルになります。

突然誰かが200ルーブルの罰金は完全にナンセンスだと思ったら、いくつか例を挙げましょう。 2007年に、ダルスヴァヤズをチェックする際、税務当局は、5日間で1400万件以上の請求書を加入者に提出するよう要求し、オペレーターに7億700万ルーブルの罰金を科しました。 2006年、Samotlorneftegazは、予定どおりの井戸の操業に関する毎日のレポート328万を提供しなかったため、1億6,400万の罰金を科されました。 ここで、各ペナルティに4を掛けます。これらの企業が今罰金を科せられた場合に支払う金額はいくらですか。

場合によっては、刑事責任が発生することもあります。 それは、法人の統一国家登録簿、証券所有者の登録簿または預託会計システムの偽造のために提供されています（2010年7月1日の連邦法第170.1条No. 147-FZ「ロシア連邦刑法の改正およびロシア連邦刑事訴訟法151条"）。 また、「...文書の意図的な破壊または偽造、...これらの文書の義務的な保管がロシア連邦の法律により提供されている場合」を含む、有価証券の権利を記録するための手順の違反について（185.2条）。 刑事責任は、統一された州の登録簿に公然と不正確な情報を含めることにより罰せられます（公務員に有効）（第285.3条）。
