Azure CloudでLinuxを使用する



ロシア語のMicrosoft Virtual Academyコース。Linux+ Azureの最初の一歩を踏み出すのに役立ちます。 多数のデモと実用的な推奨事項が含まれています。







次のセクションで構成されています。

1. Azure上のLinux-概要
このモジュールでは、Azureクラウドプラットフォームの基本について説明します。 主な焦点は、仮想マシンサービス、つまり、AzureクラウドでのLinux仮想マシンの起動です。



2. AzureでのLinux仮想マシンのネットワーク
このモジュールでは、Azureでのネットワークインタラクションの主なポイント、つまり、Webサイトに対応する複数の仮想マシンの作成、水平スケーリングなどについて説明します。



3. AzureクラウドでのLinux VMの管理パート1：Azure管理ポータル
このモジュールでは、Azureポータルでの作業の基本原則について説明します。 AzureポータルはWebアプリケーションであるため、ほとんどすべてのデバイスから操作できます。



4. AzureクラウドでのLinux VMの管理パート2：Azure Powershell
このモジュールでは、Azureを操作するためのPowerShellコマンドについて説明します。 このモジュールは、Azureを操作するためのPowerShellコマンドのインストール方法と、それらをAzureサブスクリプションに接続する方法を示しています。



5. AzureクラウドでのLinux VMの管理パート3：Azureコマンドラインインターフェイス
このモジュールでは、Azureのコマンドラインユーティリティを使用する基本について説明します。 このモジュールでは、コマンドラインユーティリティを自分でインストールする方法と、それらをAzureに接続する方法を示します。



6. AzureクラウドでLinux VMを管理するパート4：Cerebrataツール
このモジュールでは、Azure Explorer、Azure PowerShell、Azure Management StudioなどのCerebrataユーティリティについて説明します。 これらのツールはすべて、Azureでの作業をより快適にします。



7. Azureのデータストレージ-ディスクと分散ファイルシステム
このモジュールは、AzureクラウドにLinux仮想マシンデータを保存するために利用可能なすべてのオプション（Blobストレージ、Azureディスク、AzureのGlusterFS分散ファイルシステムなど）をカバーしています。

3.クラウドフェールオーバークラスター構成
