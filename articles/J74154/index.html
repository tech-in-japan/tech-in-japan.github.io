Nokia Communicators用の次世代ブラウザ









2009年11月3日-ノルウェー、オスロ Opera Softwareは、Symbian / S60プラットフォームで動作するNokiaスマートフォンやその他のモバイルデバイス向けのOpera Mobile 10ブラウザーのベータ版のリリースを発表しました。 Opera Mobile 10ベータ版の最も注目すべき革新は、Opera Mini 5ベータ版ブラウザで最初に導入されたインターフェースです。 Opera Mobileユーザーは、お気に入りのWebサイトにすばやくアクセスしたり、タブやパスワードマネージャーを使用したりするために、Expressパネルにアクセスできます。



Opera Mobile 10ベータ版は、タッチパネルとキーボードの両方を備えたコミュニケーターとの連携に最適です。



Opera Mobile 10のベータ版は、 www.opera.com / mobile / nextからダウンロードできます。



Opera Mobile 10ベータ版の開発における最も重要な優先事項の1つは、ブラウザーの速度です。 Symbian / S60の現在のバージョンは、前のバージョンの2倍の速度です。 ユーザーは、Webページの読み込み、スケーリング、トリミングの速度が大幅に向上します。 さらに、ブラウザはOpera Turboテクノロジーをサポートしています。これにより、モバイルデバイスに転送されるWebページのコンテンツを圧縮できます。 このテクノロジーにより、サイトコンテンツのダウンロード速度を向上させ、モバイルインターネットのコストを削減できます。



「強力なスマートフォンとスマートフォンにはOperaのような強力なブラウザが必要です」とOpera SoftwareのヘッドであるJon von Tetzchner氏は述べています。 -Opera Mobile 10のベータ版をリリースすることにより、デスクトップブラウザーを携帯電話に移行しています。 これで、外出先でも完全なWebページをすばやく開くことができます。



Opera Mobile 10ベータ機能の詳細をご覧ください。

更新されたインターフェース。 OperaはOpera Mobileブラウザのデザインを大幅に再設計し、携帯電話ユーザーにとってよりシンプルで直感的なものにしました。 Opera Mini 5ブラウザを開発する際にも同じアプローチが使用されます。



デスクトップブラウザの機能は携帯電話にあります。 タブ、エクスプレスダッシュボード、パスワードマネージャーなどのOperaデスクトップブラウザーの利点が携帯電話で利用できるようになり、ウェブリソースへのアクセスが簡素化され、コンテンツのダウンロード時間を節約できます。



互換性と汎用性。 Opera Mobile 10は、最新世代のブラウザーコアに基づいています。 つまり、モバイルブラウザーのすべてのWebサイトとアプリケーションは、Operaのデスクトップバージョンと同じように機能します。



在庫状況

Opera Mobile 10ベータ版は、ノキア、ソニーエリクソン、サムスンのコミュニケーターが利用でき、Symbian / S60プラットフォームの3番目と5番目のバージョンで動作します。

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社に関する詳細： www.opera.com
