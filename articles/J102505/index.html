Habrのすべての読者への挨拶！

CataXと呼ばれる私のプロジェクトを紹介したいと思います。これは、ローカルデータ検索を便利でシンプル、そして人間にとって便利な方法にするために作成されました。



便利なものは何ですか？



CataXでは、任意のキーワード（タグ）をパーソナルコンピューター上の任意のファイル（またはフォルダー）に割り当てることができ、指定したタグでデータを検索できます。 キーワードに加えて、ファイルにコメントすることができます。 言い換えれば、あなたが好きなようにファイルに「名前を付ける」能力を持っていると想像してください。そして、あなた自身の名前を使って、この単語（またはフレーズ）を書くだけで必要なものを見つけてください。 これがCataXが提供するものです。



どうやって？



必要なのは、メタ情報（タグとコメント）を追加したいデータ（ファイルとフォルダー）を見つけ、それらにインデックスを付け、メタ情報を追加し、オプションで検索方法を指定し、検索するキーワードを入力（または選択）することだけです。



そして、より詳細に？







これは、.NET Framework 3.5を使用してC＃で記述されたCataXの外観です（アプリケーションが機能するために必要です）。 CataXは、Windowsファミリのオペレーティングシステム、より正確には、XP / Vista / 7 / Server 2003 / Server 2008のバージョンで動作します。これらのオペレーティングシステムの32ビット版と64ビット版の両方がサポートされています。
