IT Global Meetup：内部の様子。 サンクトペテルブルクのIT専門家の四半期ごとの集まりに関するPiter Unitedイニシアチブのイデオロギー家

今日のiCharのブログゲストは、サンクトペテルブルクのITコミュニティの協会であるPiter Unitedのイデオロギー家および活動家の1人であるMikhail Ryzhikovです。 年に3回、ピターユナイテッドはIT Global Meetupと呼ばれるITコミュニティの集会を開催します。ここでは、サンクトペテルブルクのIT収集全体が集まり、他の都市の代表者が集まります（次のグローバル集会は今週の土曜日、 11月28日に開催されます）。 Michaelは6年以上ITに携わっており、医療情報システムを開発しており、管理とビジネス分析を専門としています。 彼は、サンクトペテルブルクのITコミュニティの協会がどのように生まれ、ピーターズバーグのIT専門家のお気に入りのイベントがどのように組織されたかを語った。







パイター・ユナイテッド：始まり



Piter Unitedイニシアチブの開始時に、私（私とニコライリジコフ）は、2012年と2013年に参加できた2つの興味深いイベントに触発されました。 これらの最初のものは、その年にキエフで開催されたIT Jam 2012です。 そこに来て、私たちは非常に興味深い形式と雰囲気に染まりました。 イベントは2階の大きな建物で開催されました。 どちらのフロアも大きなオープンスペースで、多くの島に分かれていました。 いくつかの島は企業によって組織され、いくつかはウクライナのITコミュニティによって組織されました。 継続的なコミュニケーションの雰囲気は宇宙で支配され、人々は島から島へと循環し、話し、フリップチャートに何かを描き、お互いに質問し、議論しました！ 一般的に、彼らはお互いとのコミュニケーションを楽しんだ。

2番目の刺激的なイベントは、ヴォルガ川のほとりのウリヤノフスクで定期的に開催されるUlcamp 2013です。 このイベントは、市内外のキャンプ場で開催されます。 キャンプ、森、ビーチでは、バーキャンプが組織されています。 フリップチャートを備えたコミュニケーションスペース。 誰でもバーキャンプでパフォーマンスのトピックを提出できます。これには特別な場所にボードが割り当てられ、パフォーマンスの時間、場所、トピックを示すことができます。 Ulcampは、UlnovskのITコミュニティによって組織されています。 市のIT企業によって作成されました。 そこで、私たちは真剣に、コミュニティの力と利点について考え、これを生きた例で考えました。

2013年、これらのイベントは両方とも1つのコンセプトを形成しました...そして、実際の行動に移りました！



当時、市内にはすでにいくつかのITコミュニティが存在していました。SPbRubyユーザーグループ、DevOps、Linux、UX、テスター、マネージャーです。 私たちはすべての活動家を1つの会議に集め、アイデアについて議論し、 Piter Unitedイニシアチブを開始しました。 Piter Unitedの主な目標は、ITコミュニティの出現と発展のために、市内に好ましいエコシステムを作成することです。 Piter Unitedの主な原則は、宣伝、自己組織化であり、これらはすべて無料でなければなりません。 都市のITスペシャリストが関心のあるコミュニティを見つけて、そのコミュニティのメンバーになり、定期的な会議に参加してほしい。



Piter Unitedの枠組みの中で、IT Global Meetupは年3回開催されます-サンクトペテルブルクのITコミュニティの集会です。 その主な目標は、ITコミュニティを相互に紹介し、新しいコミュニティの出発点となり、コミュニティ間の活動を促進することです。 各コミュニティは、集会への参加に加えて、定期的な（ほとんどの場合は月に一度）会議を開催します。 それぞれにオンラインプラットフォームがあり、参加者にとって重要な質問が提起され、今後の会議が発表され、アイデア、資料、リンクが交換されます。 IT Global Meetupは、その名が示すように、サンクトペテルブルクで開催される単一のITパーティーに地域コミュニティを結び付けるグローバルなイベントです。



最初のITグローバルMeetup：いかがでしたか



最初のIT Global Meetupは2013年12月に開催されました。 9人のITコミュニティが参加しました。 イベントには約80人が参加しました。 当時、私たちは組織的な経験がなく、mitapは小さくて商工会議所であることが判明しましたが、すでに島の原則に従って組織されていました。つまり、島でコミュニケーションが行われ、各コミュニティには独自の島とプログラムがありました。 最初のイベントを処理した後、次の各mitapを前のmitapよりも良くするという目標を設定しました。 私は、私たちが成功していることを敢えて提案します。現在、750人以上のIT専門家が集会に集まっています。 そして、強力な成長の可能性が見えています。質問はサイトにかかっています。 このフォーマットには多くのスペースが必要です！



振り返ってみると、大きく成長していることがわかります。 IT企業をイニシアチブに参加させる方法を学びました。 私たちは自分の投資で最初のmitapを組織しましたが、それでも食べ物とフリップチャートの両方がありました=）



最初のIT Global Meetupの時から、コミュニティの数はちょうど3倍になりました。その数は9から27に増加しました。



機器、サイトの選択、およびイベントの形式で進捗が顕著です。 早い時期に小島が集まって分散した場合、IT Global Meetupには単一のタイムグリッドがあります。 私たちは島の生活の中で同期を達成しました。会場の異なる端で、話し手は同時に話し始めたり話しを止めたりします。 このようにして、イベント参加者は、誰がいついつ聞きたいかを計画できます。



情報サポート、ブランディングの方法を学びました。 サンクトペテルブルクのすべての世界的な集会には、スピーカー、プレス機、およびアクティブな参加者へのプレゼント用のTシャツがあります。



しかし、最も重要なことはおそらく自分への忠誠心です。 ブランドを維持し、ITスペシャリストを会議に正確に引き付け、専門家向けのイベントであり、コミュニケーションと経験の交換を目的とした興味深い形式で運営しています。



難しさ



何らかの理由で、多くの人は、Piter Unitedが参加者の専門能力開発と経験の交換に主眼を置いている非営利のイニシアチブであることを理解/説明するのが難しいと感じています。 このような説明は、企業とのコミュニケーションのレベルでも、コミュニティの代表者または潜在的な代表者との個人的な会話でも行う必要があります。



IT Global Meetupの主催者としてのもう1つの困難は、支援する準備ができている積極的な参加者の不足です。 ドライバーは本当に不足しており、現在取り組んでいます。 mitaps、説明、および一般的な雰囲気の形式を使用して、人々を議論に引き込み、活動の火をつけてみます。 私たちは、コミュニティの参加者に対して、積極的に活動すればより多くの利益が得られることを常に伝えようとします。質問をし、知り合いになり、議論に参加し、議論し、間違いを犯します。



Piter Unitedは自己組織化を推進しています。 コミュニティでは、すべての参加者に平等な権利と機会があります。 コミュニティの関心と有用性は、各メンバーによって異なります。 誰もがディスカッションのための新しいトピックを提案し、発言し、コミュニティラリーの組織に参加できます。



ITスペシャリストの内向性について



ストレッチセーターの内向的なITプロフェッショナルの考えは、現実とはほとんど関係ありません。 プログラマーのプロフィールは、原則として、近年大きく変化しています。 今、私たちのイベントには若い人たちがいます-オープンで積極的です。 たぶん10年前はすべてが異なっていたかもしれませんが、今ではIT専門家は黙秘者のようには見えません。 もちろん、どの地域でも非コミュニケーションの人々の一定の割合が常にありますが=）



からのサポートについて



IT企業を率先して関与させ、企業と知り合い、Piter Unitedとそのイベントについて話すよう努めています。 多くが当社のパートナーになり、ローカルおよびグローバル会議の会場を提供します。 現在、安定したパートナーには約20のIT企業があります。 情報サポートを提供することで、IT Global Meetupの促進に役立ちます。



また、大学内で成長した大学や研究所と協力しています。 IT Global Meetupおよびグローバルラリーへの訪問者には、一定の割合の学生がいます。これらは、Peter Unitedが専門家のコミュニティであり続けるために厳しく管理されています。



現時点では、イニシアチブをサポートするために州と交渉中です。



今後の計画について



年に3回グローバル会議を開催するために、同じペースでイニシアチブをさらに発展させたいと考えています。 ソフトウェアの開発と運用の全サイクル（マーケティングから消費者へのソフトウェア配信まで）をカバーするコミュニティが街に登場するのは良いことです。 現在、多くのコミュニティが非常に不足しています。モバイル開発、データベース、多数の言語（C / C ++、Java）、ITでのマーケティングなどです。また、ITとサブジェクトエリアの交点にあるコミュニティについては、すでに考えられています。 「医学のIT」、「教育のIT」など



次の集会はより広い会場で開催する予定です。これにより、今回はどのコミュニティに合うか、そうでないかを選択する必要はありません。 すべてのコミュニティをイベントに快適に配置し、1週間前に登録を終了しないように（サイトが参加者の流入から破裂することを恐れる）=）



ディスカッション、意見交換、経験のすべての参加者を巻き込み、通常の「一人が話し、他の人が静かに耳を傾ける」ことから逃れる新しい形式を本当に学びたいです。 ディスカッションのモデレート/ファシリテーションに関する無料セッションを開催することも考えています。 そのため、活動家と関心のあるすべてのコミュニティメンバーがこのスキルを習得し、円卓会議と専門的な議論を行うことができました。



コミュニティの利点について



すべての活動家にとって、ピターユナイテッドは趣味であり、専門的な活動ではありません。 私たちはそれぞれITの専門家であり、多くのコミュニティのメンバーです。 コミュニティは、強力な企業と成熟した専門家のツールです。 彼らは専門的視野を広げ、市場を見て企業と知り合い、専門家として自分自身を見せ、志を同じくする人々を見つけ、彼らの専門的活動の困難な瞬間にまともなモチベーションを得ることができます。 すべてのアクティブなITプロフェッショナルがこのイニシアチブに参加し、コミュニティや会議に参加するよう招待します。 ドアが開いています！
