アイソスティック：光学ドライブのふりをするフラッシュドライブ

USBフラッシュドライブをUSBポートに挿入すると、新しいUSB大容量記憶装置と新しいUSB CD / DVDドライブがシステムに表示されます。 これが、 Isostickフラッシュドライブファームウェアの仕組みです。フラッシュドライブは、ブートイメージを含むISOイメージの記録に最適です。 最も重要なことは、 多くのブート可能なISOを記録できることです。また、組み込みのブートローダー（isosel）を使用すると、USB CD / DVDドライブからブートするときにそれらを切り替えることができます。 複数のOSが必要な場合、このデバイスはUNetBootin、grubなどのプログラムを使用するよりも便利です。



フラッシュドライブが作成され、数週間以内に販売が開始されます。この時点で、生産を注文するのに十分なお金を集めます 。 すべてがうまくいけば、作者はファームウェアをオープンライセンスにすると約束し、だれでもそのようなデバイスを自分でできるようにします。







デバイスの内部には、読み取り速度が12.5 MB /秒、書き込みが4-6 MB /秒のmicroSDカード（取り外し不可）があります。 32 GBの容量で、そのようなフラッシュドライブにあるすべてのオペレーティングシステムを書き込むことができます。 FAT32ファイルシステムのみがサポートされていますが、著者はNTFS、HFS +、およびext2 / 3/4 をサポートする可能性を調査しています。



これは、インターネットからISOの形式で何かをダウンロードしたい管理者やその他のオタクにとって理想的なものです。 ただし、ISOだけでなく、そこに任意のファイルを書き込むことができます。オペレーティングシステムの観点からは、これらはCD / DVD-ROM上のファイルになります。



フラッシュドライブの追加オプション-読み取り専用スイッチとアクティビティLED。これらはオフにすることもできます。



現時点では、Kickstarterインキュベーターでフラッシュドライブを購入/投資できます。 75ドルの投資家は、8 GBのフラッシュドライブ、125〜16 GB、175〜16 GB、225〜32 GBを受け取ります。 すべての投資パッケージには、フラッシュドライブだけでなく、ステッカーセットも含まれています。 米国以外の国への配送には、さらに15ドルかかります。
