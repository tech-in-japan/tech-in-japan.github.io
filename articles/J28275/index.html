Windows XPは今日廃止されました

Windows XPのリリースから約7年後の2008年7月1日以降、Microsoftは、Dell、HP、Lenovoなどの主要なPCメーカーへの出荷と同様に、Windows XPを個別の製品として販売することを正式に停止しました。 実際、今日では「廃止された」オペレーティングシステムの公式埋葬のプロセスが開始されました。



この日付は長い間知られていました、そして、トピックは繰り返しHabréで議論されました。 実際、今日革命は起こっていません。 実際、Windows XPライセンスは必要に応じて購入できます。



Windows XPは、システムインテグレーター（2009年1月31日まで）だけでなく、超安価なPCの「予算」ソリューションとして、OEMチャネルを通じて（2010年6月30日まで）長期にわたって提供されます。 同時に、すでに販売されているシステムの標準サポートは今年の終わりまで提供され、いわゆる「延長」サポートは2014年4月まで提供されます。 言い換えれば、Windows XPの所有者にとって、近い将来絶対に何も変わることはありません。



今日、特にひどいことはありませんでしたが、これらすべてのイベントは、新しいコンピューターのアップグレードを強制され、新しい「生の」Windows Vistaオペレーティングシステムに慣れるという、新しい頭痛に直面している約4億人のWindows XPユーザーを喜ばせていません。



Windows Vistaの販売はかなり悪いです。 多くのユーザーは一般に、次のWindows 7システムがリリースされるまでWindows XPを放棄しないと言います。Microsoftは開発を遅らせないことを約束しているからです。
