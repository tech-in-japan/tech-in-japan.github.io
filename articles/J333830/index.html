初のメガアクセラレータ：自由飛行の年







最初のMEGA Acceleratorを無事に完了した2人の卒業生、SensArtとTexelに会いましょう。 1年前、私たちは革新的なアイデアを検索し、実装して、ショッピングセンターなどでのMEGAゲストの購入体験を改善しようとしました。 これらのプロジェクトは興味深く、有望であることが判明し、そのうちの1つがメイン賞を受賞しました。 ここで、彼らのビジネスがどのように変化したか、そして彼らが国内および西部の市場に突入する準備をしていることについて話したいと思います。 また、初心者がアクセラレータ2にアクセスする可能性を高めるために必要なことについて少し説明します。



プロジェクト自体についてのいくつかの言葉。 最初の加速器は2015年12月に発売されました。 約200のスタートアップが予選ラウンドに参加し、そのうち10のスタートアップがフルタイムステージに達しました。 ファイナリストは、アイデアを完成させるための助成金と専門家の支援を受けました。 勝者は、表面をタッチスクリーンに変える技術を備えたSensArtチームでした-Surfancy。 彼女から始めましょう。



界面活性剤



最初のMEGA Accelerator SensArtには、Surfancyという製品が付属していました。これを使用すると、あらゆる表面から触ることができます。 テクノロジーの詳細な説明は、たとえばここにあります 。 その年の可能な「タッチスクリーン」のサイズは50 m2（5x10メートル）に増加し、チームはシステムの速度と精度を高めるために作業を続けています。









チームは、製品の技術面だけに関心があるわけではありません。 現在、スタートアップは企業クライアントにサービスを提供しています。 1つ目は、ブランドに既存のマーケティング資料に基づいてインタラクティブコンテンツを作成し、Surfancyインフラストラクチャで起動する機会を提供します。 たとえば、Megaがサポートするパイロットプロジェクトの一部として、QuikSilverのインタラクティブストアフロントが実装されました。



2番目のサービスでは、企業がさまざまな表面にインタラクティブな視覚効果を追加するシンプルなソフトウェアや、ブランド自身が開発した強力なアプリケーションを含むアプリケーションのライブラリにアクセスできます。 原則として、これらは、割引などを獲得する機会を顧客に引き付けることを目的とした商品やアプリケーションの注文とカスタマイズの可能性を備えた電子製品カタログです。









MEGA Acceleratorへの参加により、チームはあらゆるスタートアップの開発においてRubiconを通過することができました。開発したテクノロジーに基づいた最終製品が市場で需要があることを理解するためです。



「フルタイムアクセラレータの参加者であるトップ10チームに正確に参加することが主な利益であることを理解しています。 そこに到達できる経験は、B2Bスタートアップにとって非常に貴重です。 テクノロジーから製品への移行を行うために、私たちは何をしているのかを再考することができました。 MEGA Acceleratorに参加する過程で、クライアントへのアプローチをほぼ完全に変更し、デバイスの販売からセンサーが必要になったサービスの販売に移行しました。



新しいアクセラレーターに招待されるスタートアップにアドバイスをしたい。 ショッピングセンターのビジネスまたは顧客の問題を解決するだけでなく、同時に問題とその解決策を見つける必要があります。 ニーズ分析にもっと注意を払い、ショッピングセンターの「地上」でできるだけ多くの時間を過ごす必要があります。 最も重要なことは、教室で提供されるすべての情報をアクセラレータ自体に吸収することです。 私を信じて、プログラムへの参加はあなたのビジネスに起こることができる最高です」と、Andrey Litmanovich、サーファーシーの共同創設者は、彼の経験を共有します。









現在、開発者はロシアおよび海外の大企業向けに多数のプロジェクトを準備しています。 正確に何が表示され、どこで、彼らは言っていないが、それが面白いと約束します。



テクセル



昨年のもう1つのファイナリストはテクセルチームです。 彼らは仮想の試着室で加速器に来て、人の3Dモデルで衣服の視覚的なデモンストレーションに賭けました。









今年、コンセプトが変わりました。男たちは市場を研究し、より有望な方向性を選択しました。サイズとスタイルの両方で、服の品質に関する推奨事項を提供します。 主な製品は、ドレスコードソリューションでした。



「コンセプトは非常にシンプルです。 ショッピングセンターにデジタルフィッティングルームを設置すると、訪問者がスキャンされます。30秒もかかりません。 次に、Dress Code AIアプリケーションをダウンロードし、ショッピングセンターで提供されるフィルター処理された衣服のカタログをスマートフォンの画面に表示します。フィギュアの種類（梨、砂時計など）に応じて、サイズとスタイルが自分に合ったものだけです。 また、アプリケーションは、ユーザーが選択したアイテムが販売されているストアへのパスを表示できます。 ドレスコードAIの創設者であるミハイル・ドロホフは次のように述べています。



スタートアップはターゲット市場として米国を選び、本社はニューヨークにあり、開発センターはモスクワにあります。 チームは実用的なプロトタイプを作成し、北米全域に500の3Dスキャナーのネットワークを持つ十分な数のブランドとパートナーと合意しました。



米国ではオンラインショッピングの人気が高いため、Eコマースモデルも使用されています。 ユーザーは3Dスキャナーでスキャンするか（このためには、500の設定のネットワークが必要です）、アプリケーションからデータを入力できます。 その後、彼は、サイズとスタイルが自分に合ったデータベースで利用可能なすべての服を確認し、好きなものを選択して、[購入]ボタンをクリックします。 同社は、少なくとも10,000人の有料ユーザーをプラットフォームに引き付けることを計画しています。









「実用的なモデルを見つけたとき、すぐに市場からの肯定的なフィードバックを感じました。 現在、当社の技術を使用したいロシア、ヨーロッパ、アジアのショッピングセンターから多くのリクエストが寄せられています。 世界最大のスキャナーのネットワークは、私たちと協力したいと考えています。 前回の投資ラウンドを非常に迅速に完了し、使用する予定よりも多くの次のラウンドへの投資申請を既に行っています」とミハイル・ドロホフ氏は述べています。



チームはMEGA Acceleratorに参加したことを喜んで思い出します。



「最も有用なのは、衣料品小売市場の実際のプレーヤーとのコミュニケーションと相互作用でした。 この経験は、アイデアを実際の生活に変換し、クライアント、クライアントの期待、行動を理解するのに役立ちました。



それとは別に、IKEA Centers Russiaチーム、たとえば、MEGA Belaya Dachaのマネージャーとのコミュニケーションに注目したいと思います。MEGABelaya Dachaのマネージャーは、彼の経験と市場のビジョンで私たちを大いに助けてくれました。 彼は、理想的だが機能しない製品よりも、機能する未加工で不完全な製品を用意する方が良いと言いました。



その結果、訪問者とショップに緊急のニーズに焦点を当てたより真の利益が得られるほど、スキャンするために並んでいる人々が増え、パイロットプロジェクトがより成功しました。



私たちが学んだ最も重要な教訓：私たちは他の人に耳を傾け、彼らの経験を信頼しなければなりません。他の人が対処できない何かを恐れることはありません。 MEGA Acceleratorのオーガナイザーに非常に感謝しています。ドレスコードのターニングポイントでした」とミハイルドロホフは彼の記憶を共有しました。
