Unityでトレントを検索する

これはこのテキストの無料翻訳です 。 オリジナルの作成者は、作成をテストすることを提案します。







トレントを検索してUbuntuを頻繁に使用する必要がある場合は、猫の下に飛び込みます...



このアドオンにより、Pirate Bayでトレントを見つけるのがはるかに簡単になりました。ダッシュを開き、トレント検索を選択し、検索するフレーズを入力します。



これは、 都市検索アドオンの作成者であり、 Dashの特別な計算機でもあるDavidCalléの結果です。



特徴



検索結果とともに、配布シードのファイル名、サイズ、タイプ、および数が表示されます。



さらに、検索時に、探しているファイルの種類を選択できます。







結果をクリックすると、ブラウザが開き、Torrentクライアントにドラッグすることもできます：







現在、検索にはPirate Bayのみが使用されていますが、他のサービスを追加できます。



警告



このサプリメントは著作権侵害を呼びません。



海賊版コンテンツはPirate Bayで公開されていますが、Linuxディストリビューションなどの便利なものもそこにあります。



このアドオンは単なる検索ツールであり、著作権侵害を要求するものではありません。



Ubuntu 11.10へのインストール



インストールするには、次のコマンドを実行します。 



sudo add-apt-repository ppa:scopes-packagers/ppa sudo apt-get update && sudo apt-get install unity-scope-piratebay python-lxml
