ダウンロード可能なWindows Server 2012

9月4日に、MicrosoftはWindows Server 2012のグローバルな可用性を発表しました 。MSDNまたはTechNetサブスクリプションの一部としてStandardエディションとDatacenterエディションを入手できます。または、ダウンロードセンターで180日間バージョンの試用版 （これも2つのエディション）をダウンロードできます。



サーバーと同時に、ダウンロードすることが可能になりました：

  • 無料のMicrosoft Hyper-V Server 2012ハイパーバイザー
  • 新しいPowerShell 3.0および更新されたバージョンのWMIおよびWinRMを含むWindows Management Framework 3.0ツールキット。 これらのコンポーネントはすべてWindows Server 2012に含まれており、Windows Management Framework 3.0を使用すると、Windows 7 SP1、Windows Server 2008 R2 SP1、およびWindows Server 2008 SP2にインストールできます。


追加情報に加えて、最近公開されたMVA コースとして 、 Windows Server 2012管理ツール （サーバーマネージャー+ PowerShell）のコース（これまでは英語 ）が登場しました。また、無料の本「 Introducing Windows Server 2012 」も読むことができます 。
