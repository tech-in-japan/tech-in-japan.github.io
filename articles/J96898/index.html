Googleサービスのコマンドライン

誰もが、テキスト入力やファイルのダウンロード用の大きなボタンと特別なコントロールを備えた美しいインターフェイスを好むわけではありません。 コンソールで目的のコマンドを入力する方が簡単かつ迅速な場合があります。







GoogleCLは、コマンドラインのおかげで、Blogger、Picasa Web、YouTube、Googleドキュメント、Googleカレンダーのサービスを使用して作業を簡素化する特別なプロジェクトです。 これはGoogle Data APIのラッパーであるため、最初にPython 2.6 （または2.5）およびGoogle Data API Python Client Libraryをインストールする必要があります 。



Python、Google Data API Python Client Library、およびGoogleCLをインストールすると、コマンドを印刷して、Picasa Webへの写真のアップロード、YouTubeへのビデオのアップロード、Bloggerへの投稿、Googleカレンダーへのイベントの追加、Gmailでの連絡先の検索などを行うことができます。 。

ご存知のように、 Chrome OSには特別な開発モードがあり、経験豊富なユーザーがコマンドラインを介してシステムと通信できるため、これは特に興味深いものです。



以下にいくつかの典型的なコマンドを示します。



Googleカレンダーに「明日午前10時にメアリーに会う」を追加

（Googleカレンダーにイベントを追加）



google picasa get "Album Title" c：\ files \ picasa \

（アルバムの1つからすべてのファイルをダウンロードします）



google picasa create --title "Miami Beach" c：\ files \ photos \ miami \ *。jpg

（写真を新しいアルバムにアップロードします）



google youtube post --title "ローマの夏" c：\ videos \ rome.avi

（ビデオをYouTubeにアップロード）



google docs get --title "Reports。*"

（ヘッダーで「レポート」という単語で始まるファイルをアップロードします）



google docs upload c：\ files \ *。doc

（* .docファイルをフォルダーからGoogleドキュメントにアップロード）



利用可能なコマンドの全リストは、 Google --helpリクエストまたはマニュアルを読むことで見つけることができます。
