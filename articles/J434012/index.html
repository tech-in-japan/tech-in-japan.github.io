オープンソースプロジェクトを開発する人のための無料のPVS-Studio







新しい2019年のお祝いの前夜、PVS-Studioチームは、GitHub、GitLab、またはBitbucketでホストされているオープンソースプロジェクトのすべての貢献者に楽しい贈り物をすることにしました。 PVS-Studio静的アナライザーを無料で使用して、オープンソースプロジェクトを開発する機会が与えられます。



オープンソースコードの改善と信頼性向上を支援します。 私たちの出版物のおかげで、オープンソースプロジェクトで10,000件以上のエラーが修正されましたが、これは明らかに十分ではありません。 私たちのチームは、何千ものオープンプロジェクトを定期的にチェックすることができません。 したがって、2016年に、PVS-Studioの無料ライセンスオプションを提案しました。 唯一の条件は、特別な種類のコメントのコード内の存在です。 このタイプのライセンスの詳細については、「 PVS-Studioを無料で使用する方法 」の記事を参照してください。



私たちの要望に応えて、GitHub、GitLab、またはBitbucketでホストされているオープンプロジェクトの開発に携わるすべての人にPVS-Studioを無料で使用する機会を提供することにしました。 これらのプロジェクトの作成者は、コメントを追加する必要はありません。



全員に、1年間無料のライセンスを発行します。 ライセンスを取得するには、次のことを行う必要があります。



  1. 次のページに移動します： https : //www.viva64.com/en/open-source-license/
  2. ライセンスキーの送信先の名前と電子メールを入力します。
  3. GitHub / GitLab / Bitbucketプロファイルへのリンクを入力してください。
  4. 無料ライセンスのリクエストを送信します。


ライセンスの有効期限が切れた後、同じ方法で新しいライセンスキーを取得できます。



キーは個別であり、GitHub / GitLab / Bitbucketで公開されているオープンプロジェクトを検証するためにのみ使用できます。 無料のライセンスはプロジェクトミラーには適用されません。



コードにコメントを追加するときにアナライザーを無料で使用する古いバージョンは有効です。 このモードには利点があります。 たとえば、学生がGitHub / GitLab / Bitbucketにプロジェクトをアップロードすることなく、プロジェクトをテストするために使用できます。 さらに、以前のバージョンでは、一般的に閉じたプロジェクトでアナライザーを使用できます。



条件



無料ユーザーのサポートは、StackOverflow Webサイトの回答を通じて提供されます。 この条件については、記事「 PVS-Studioを無料で使用する方法 」で詳しく説明しています（「追加：サポート」の章を参照）。 StackOverflowはバグトラッカーではないことに注意してください。 特にアナライザー、その動作モードなどに関連する問題について説明します。 明らかなバグをお知らせするため、引き続きサポートにご連絡ください。



無料ライセンスを提供するアクションは永続的です。 ただし、何か問題が発生したと感じた場合、当社はその条件を変更または終了する権利を留保します。 また、説明なしで特定のキーを無効にする権利も留保します。



前述のように、無料のライセンスを使用する可能性は、プロジェクトミラー（ Clang 、 Chromium 、 KDEなど）には適用されません。 そのようなプロジェクトの開発者へのサポートには、チームからの相当な作業が必要であり、これらの開発者が雇用されている企業が有料ライセンスを取得できれば公平です:)。
