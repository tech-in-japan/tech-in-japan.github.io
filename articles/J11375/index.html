Daemon Tools Pro v4.10.0218

少し前に、有名な仮想CD-ROMデーモンツールの新しいバージョンが誕生しました。 この新規性は、最大34個の仮想ディスク、新しいカスタマイズ可能なインターフェイス、改善された読み取りメカニズム、拡張オンラインヘルプ、ロシア語インターフェイス、およびSPTDドライバーバージョン1.50をサポートします。



3番目のバージョンでDaemon Toolsがインターフェイスのほとんどない優れた仮想CD-ROMであった場合、4番目までにプログラムは無料の機能を失い、追加機能を獲得し、完全に商用のアルコール120％プログラムに似始めました。 Daemon Tools Proを使用するには、Webサイトwww.disc-soft.comに登録し、プログラムを使用するための支払いとして広告を表示するWhenU Save Adwareモジュールをインストールする必要があります。 有料版のAdvancedもありました。これは、とりわけ、CDのイメージを個別に作成できます。 ただし、1つの利点をDaemon Toolsから取り除くことはできません。このプログラムを使用すると、Safedisc、Securom、Starforce、およびその他のコピー保護システムで保護されているディスクのバックアップイメージも使用できます。



デーモンツールをダウンロードします。

-daemon410-x86.exe 、1.8Mb

-daemon410-x64.exe 2Mb、64ビットWindows用

-1049.dll （ロシア語のインターフェースを追加するためのライブラリ）
