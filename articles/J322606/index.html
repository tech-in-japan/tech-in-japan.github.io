Oracle Certified Professional Java SE 8 Programmer 1Z0-810試験の準備と合格の経験

こんにちは。 ここには、認証、認証パス、およびそれが必要な理由について、かなりの数の記事が掲載されています： here 、 here 、 here 。



この記事では、Oracle Certified Professional Java SE 8 Programmer 1Z0-810試験の準備と合格の経験を共有したいと思います。



この場合、私の認証パスは次のとおりです。



  • Java SE 7プログラマーI 1Z0-803
  • Java SE 7プログラマーII 1Z0-804


これらの試験は遠い2013年に合格し、その後、家族の理由で、プログラミングをしばらく中断しなければなりませんでした。 そのため、忘れられた知識を少し使って次の試験の準備をしなければなりませんでした。



そして今、継続的な教育のアップグレードの番です （ Java SE 7をJava SE 8 OCP Programmer 1Z0-810にアップグレードしてください ）。 私の動機は、スポーツへの関心、知識の更新、まあ、さまざまなグッズへの愛です。



この試験には150分、81問が割り当てられます。 試験の費用は150ドルです。 合格レベルは65％です。



テストのトピックのリスト：



  • ラムダ式
  • 組み込みLambdaタイプの使用
  • Lambdasを使用したJavaコレクションとストリーム
  • ラムダを使用したコレクション操作
  • 並列ストリーム
  • ラムダ料理本
  • メソッドの強化
  • Java SE 8 Date / Time APIを使用する


詳細については、 ウェブサイトをご覧ください 。



準備では、次のものが使用されました。



  1. Java 8 in Action：ラムダ、ストリーム、および関数型プログラミング。 ラウル・ガブリエル・ウルマ、マリオ・フスコ、アラン・マイクロフト。 ラムダ式、ストリーム（ストリーム）などのわかりやすいトピック。



  2. Java OCP Book 8 Programmer II Study Guide。 ジャンヌ・ボヤルスキーとスコット・セリコフ。 本には多くのタイプミスがありますので、それらのリストをチェックするのが最善です。 また、各章の最後には、セルフテストに関する質問のリストと、セルフテストのテストを含むリソースがあります（残念ながら、私には使用されていませんでしたが、無駄でした）。



  3. 試験の準備状況を確認するために、有料のリソースが使用されました 。 テストの過程で、システムのさまざまなエラーとバグが明らかになり、会社はすぐに通知されました。 彼らはこれらのエラーを修正することを約束しました。 まあ、一般的に、サービス自体は安定して動作しません。頻繁にフリーズしたり、データベースから切断したりすることさえあります。 特別な注意を払う必要のある問題のいくつかはまったく発生しませんでした（特に、夏時間（DST））。



  4. 良いことも悪いことも含め、あらゆる種類のコードがたくさんあります。


その結果、準備には約2か月（1日2〜3時間）かかりました。

試験の特定のトピックに関する質問があるという事実にもかかわらず、Programmer II Study Guideは全体として勉強する方が良いことに注意したい 各章では、試験で発生する可能性のあるいくつかの落とし穴について説明します。



したがって、精神的に準備し、バウチャーの代金を支払った後、指定された場所で指定された時間に試験に合格しました。



試験中に強調したい質問：



  • 夏時間（DST）、ZoneId、ZoneOffsetについての5つの質問。
  • Files.walkとFiles.findで4つの質問について。
  • Streamsでreduce（）関数がどのように機能するかを知ることに関して、多くの質問があります。
  • 関数から受信タイプと戻り値を確認します。
  • Arrays.stream（）
  • map.computeIfPresent（）およびmap.computeIfAbsent（）に加えて、Map.copmute（）およびmap.merge（）に関する質問がありました。


このような準備の結果、75％のスコアが得られました。



結論は次のとおりです。



  • 私の場合のように、他のソースで質問のタイプを見る機会がある場合、 リソースを使用できます。
  • 試験では、トライアルテストからの単一の質問はありませんでした。 したがって、シミュレーター（有料のものも含む）のみに焦点を合わせるのは100％にはお勧めできません。
  • 彼らの知識を100％確信しているわけではありませんが、変更を延期し、資料を再度確認する方が良いでしょう。


変更の終わりに、印刷された証明書とTシャツを手に入れたいという希望がありました。 しかし、彼らは私たちの国にTシャツを送りません。 また、印刷された証明書について、次の通知がアカウントで送信されました。



重要な注意：オラクルは、環境の保護に役立つプラクティスと製品の開発に取り組んでいます。 ハードコピー印刷された証明書は利用できません。
