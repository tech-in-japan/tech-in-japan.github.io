バージョン1.0リリース

Web Optimizerのオープンソースプロジェクト （1年前）の開発をフォローしている人は、最近のイベントで既に最新です-自動Webサイトアクセラレーション用アプリケーションの安定したバージョン1.0- WEBO Site SpeedUpが投稿されました。



最新バージョンをダウンロード： www.webogroup.com/en/home/download



カットの下-新しいバージョンに関するいくつかの詳細と近い将来の計画。



新しいインターフェース



Web Optimizerアプリケーションの以前のバージョン（0.6.7以前）と比較して、インターフェースはゼロから作成されました。 これで、アプリケーションのすべての作業がコントロールパネルから実行されます。





大幅に再設計されたアプリケーション構成メカニズム。 まず、デバッグモードが使用可能になりました（パスに適切なCookieまたはGETパラメーターがある場合にのみWEBO Site SpeedUpが機能する場合）。 第二に、サイト間で切り替えたり、転送したりできる構成（設定のグループ）が導入されました。





設定自体はほとんど変更されていません。





そして第三に、システム設定の一部（パス、サイトへのアクセス）がシステム状態に転送されました（通常、ページはシステムのすべての機能のチェックに関連して長時間ロードされます）。



ここ（コントロールパネルの対応するブロック）に、ホスティングに関するコメントのリストが表示されます（主に、破損したApacheモジュール）。 このリストにはすでに多くの疑問が生じています。各アイテムのヒントシステムを完成させ、すぐにインターフェイスに追加します。



キャッシュには独自のページもあります。





オンラインデモのアドレス（ www.webogroup.com/en/home/demo）で、新しいインターフェースを実際に試すことができます。



予備のユーザビリティテストを実施しましたが、その結果に基づいた改善の一部が製品に追加されます。 製品のインターフェースに関する提案がある場合は、コメントに書いたり、お問い合わせください 。



ライセンスポリシー



WEBO Site SpeedUpには 3つのバージョンがあります。

  • 非営利バージョン。 営利を目的としないサイト（非営利）で制限なく使用できます。 最初はサイトを高速化するのに十分な設定がありますが、パフォーマンスの制限がいくつかあります。
  • Liteバージョン。 フルバージョンの機能を必要としないが、クライアント側の高速化が必要な商用サイト向けに設計されています。 このバージョンのパフォーマンスは、非営利団体のパフォーマンスよりもわずかに高くなっています。
  • フルバージョン。 すべての設定と最大のパフォーマンス。


完全版が適切かどうかわからない場合は、試用版キーを数日間注文できます。 これについてはメールでお問い合わせください 。



WEBO Site SpeedUpの現在のバージョンは、Web Optimizerアプリケーションの新しいバージョンです。 したがって、Webオプティマイザーのライセンスキーがある場合は、WEBO Site SpeedUpに適しています。



今後の計画



多くのユーザーは、製品の目的（サイトのクライアント部分の高速化）を理解し、その目的に使用します。 しかし、ほとんどの場合、Webサイトの高速化はサーバーの最適化でもあります。 これに関して、今後のバージョンでは、効果的で柔軟なサーバーキャッシングメカニズム（WordPressおよびJoomla！）を追加します。これは、アナログよりも高速に動作し、より柔軟な設定を提供します（たとえば、コメントを送信するときにキャッシュからページを削除する）。



他のすべてのCMSについては、サーバーのキャッシュが既に別のWebアプリケーションに含まれているため、サーバーの負荷を大幅に削減できます。



さらに、もちろん、 公式バグ追跡システムで発表されたすべてのユーザー改善を紹介し、実装します。 合理的な提案がある場合は、このリストに追加してください。
