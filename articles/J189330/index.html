BitByte Festival-ITコミュニティの待ち合わせ場所

9月21日 、ポータルAichi-Event.rfの積極的なサポートにより、Digital OctoberはそのサイトでBitByteを開催します-IT分野の専門能力開発の年次祭。 このフェスティバルでは、IT分野の第一人者と初心者が集まり、技術的およびキャリアの「アップグレード」のためのすべての条件を提供します。





プログラム内：


技術およびキャリアスピーチの並行セクション。 訪問者は、経験豊富な技術専門家から質問への回答を受け取り、HRディレクターからキャリア開発のアドバイスを聞きます。 誰もがマスタークラスでトレーニングを行い、それらに関する市場の専門家からフィードバックを受けることができます。

プレゼンテーションの主要トピック：IT市場の動向と将来、プログラミング言語、モバイル開発、ビジネスの評判、キャリアプランニング。 スピーカーは、成功したケース、テクノロジー、ツールを共有します。





すでにプログラムへの参加を確認しています：


  • イリヤ・バラクニン、紙飛行機「ビジネス評判と新しいメディア：互換性のないものを組み合わせる」
  • RusonixのYuri Ustinov氏「コンテナ仮想化。 雲の中のシンデレラ
  • Petr Kuryshev、ITmozg「Erlang / OTP：not to fall」
  • YORDスタジオ、デニス・ウィットマン「やらないことを学ぶ。 効率を上げる方法」
  • Andrey Sudarikov PlayDisplay「インタラクティブなビジュアルコミュニケーション。 技術開発の視点”




雇用企業の展示会では、参加者は労働条件、現在開かれているIT欠員および従業員開発プログラムについて学ぶことができます。 興味深い点は、大企業のIT部門の代表者と個人的に話し、短いインタビューを受ける機会です。



Google、Oracle、Softline、Mail.Ru Group、Channel One、Yandex.Money、Yulmart、T-Systemsなどの企業がBitByteフェスティバルに参加しました。



イベントITmozgの主催者は、IT分野で仕事を見つけるためのリソースです。

入場は無料です。 BitByteに参加するには、サイトに登録する必要があります： http ://bitbyte.itmozg.ru/

BitByte Webサイトのトレーニングニュースをフォローし、 VKontakteおよびFacebookページを購読してください。 #BitByteフェスティバルの公式ハッシュタグ。
