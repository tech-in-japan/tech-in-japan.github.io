Fedora 25リリース

Fedora Projectは、ディストリビューションの次のリリースであるFedora 25を発表できることを嬉しく思います。Fedoraは、フリーソフトウェアのプロモーションに日々取り組んでいる大規模なコミュニティです。 Fedora 25は、Fedora 25 Atomic Host、Fedora 25 Server、およびFedora 25 Workstationの3つの異なるバリエーションで提供され、それぞれが特定のニーズを満たすのに役立ちます。 Fedora 25には多数の修正と改善が含まれており、このFedoraには以下のパッケージが含まれています。



  • コンテナアプリケーションをインストールして実行するためのDocker 1.12
  • Node.js 6.9.1、人気のあるJavaScriptエンジンの最新バージョン
  • 高速で安定したシステムプログラミング言語であるRustのサポート
  • Pythonの複数のバージョンの同時サポート-2.6、2.7、3.3、3.4、3.5。


主な変更点



Fedoraワークステーションには、次のような重要な変更が含まれています。



  • Waylandを支持してデフォルトでX11を使用できない場合、X11セッションはオプションとして残ります
  • GNOMEとそのすべてのプログラムをバージョン3.22に更新します。 ここで変更点を見つけることができます。
  • 追加のデコーダーをインストールせずにMP3をサポート
  • ブートイメージを記録するプログラム「Fedora Media Writer」を追加
  • これで、GNOMEの新しいバージョンにアップグレードするときに、拡張機能が壊れなくなりました。
  • 新しいパッケージバージョン：PHP 7.0、Ruby on Rails 5.0、GHC（Haskell）7.10、Perl 5.24、Erlang 19、Go 1.7、Glibc 2.24
  • Flatpak形式のパッケージをインストール、アンインストール、および更新する組み込み機能

Fedora 25へのアップグレード



Fedora 24ユーザーは、ターミナルでいくつかのコマンドを実行することで、30分以内に新しいバージョンにアップグレードできます。 



$ sudo dnf upgrade --refresh $ sudo dnf install dnf-plugin-system-upgrade $ sudo dnf system-upgrade download --releasever=25 $ sudo dnf system-upgrade reboot
