BlogEngine.NET 2.0ブログエンジン

1月初旬、ASP.NET BlogEngine.NET 2.0向けの人気のあるオープンソースブログエンジンの2回目のリリースが公開されました。 BlogEngine.NET 2.0のインストールパッケージは、Microsoft WebサイトのWebアプリケーションギャラリーからも入手できます。これにより、数回クリックするだけでインストールできます。



オープンソースエンジンのBlogEngine.NETは、使いやすさと機能の拡張のしやすさという2つの主な原則を念頭に置いて設計されています。 現在、最も強力で使いやすいブログプラットフォームの1つです。



バージョン2.0の主な変更点
  • .NET 3.5の使用
  • Captcha Recaptchaの使用
  • TypePadFilterコメントスパムフィルター
  • Disqusコメントシステムを使用する機能
  • SQL CE 4.0のサポート（データベースはApp_Dataに保存され、データベースサーバーは不要です）
  • 削除された投稿とコメントのごみ箱メカニズム
  • JQueryライブラリーはすべてのページで自動的に接続されます
  • HTML 5ビデオのサポート
  • 構文強調表示のためのビルトインbilbiotek構文強調表示
  • 役割ベースのユーザー権利管理システム
  • JS / CSS圧縮システムの改善
  • 最新のtinyMCE redactorのサポート
エンジンをバージョン1.6から2.0にアップグレードする場合は、 アップグレード手順 （ENG）が利用可能です。



プロジェクトのWebサイトには、エンジンをインストールおよび構成するためのビデオがいくつかあります 。 divochkaのインストールを希望する場合は、 Web Platfrom Installerを自動的に使用できます。
