ISON彗星は危険ですか？

今ルネットでは、来るべきISON彗星の通過と人類に対するその想像上の危険をめぐってパニックの波が人工的に上昇しています。 彼らは、それは必然的に地球に落ち、いくつかのオブジェクトがそれを「伴う」と言います。 それからNASAはこの神話を促進している人々の喜びにオフになりました。







私は長い間、2012年に世界の終わりに魅了された後、妄想が思い付くものを待ち望んでいました。 そして最後に、彼らは生まれました。 古代から人々は、これらの天体現象の起源を多かれ少なかれ理解していたとしても、彗星を恐れていました。 現在、彗星は世界的なパニックを引き起こしていません。 しかし、ISONは科学者自身による大規模なPRによって提供されました。「月より明るい」、「世紀の彗星」であり、これは教育を受けていない大衆を威attemptsする試みの基礎を作りました。 確かに、スケールはローカルです：彗星はひどく恐れているようです、それはロシアでのみです。 欧米のユーフォロジストは、彗星の「護衛艦」のイメージで、単に偽のビデオを楽しんでいます。



C / 2012 S1（ISON）が危険かどうかを見てみましょう。



ISON彗星は、International Scientific Optical Network（ Pulkovo光学観測者の協力 ）にちなんで命名されました。 開拓者のヴィタリー・ネフスキーとアルテム・ノビチョノクは、最初に彼女を小惑星と間違えたため、彗星は名前を受け取りませんでした。



ISONが2012年に最初に発見されたとき、ISONのサイズと軌道の予備的な推定値は、ISONが空で非常に見えることを示唆していました。 したがって、それがビッグコメットになることへの希望がありました。



しかし、それ以来ずっと接近してきたため、惑星間、軌道、地上の望遠鏡を使って研究されました。 したがって、イベントの規模の推定値は大幅に変更されており、非常に成功した状況の結果としてのみ、肉眼でそれを見ることができます。



私たちに飛ぶものを詳しく見てみましょう。



最初の興味深い観測の1つは、火星と木星の間の小惑星帯で回転するNASAディープインパクト惑星間ステーションによって2013年1月に行われました。 彼女は予想される仕事のリソースを大幅に超え、到達できるものすべてを研究し、長い間アイドル状態でした。 実際、ISONの調査はこのステーションにとって最後の大きなものでした。2013年8月に通信が中断され、9月にNASAの専門家がデバイスの最終的な損失を発表しました。







彗星の次の重要なショットはハッブル宇宙望遠鏡でした。



4月と5月に6億5,000万kmの距離から行われた一連の調査により、彗星のより良い眺めが可能になりました。 小さなタイムラプスを作ることさえ可能でした：







彗星が木星の軌道を横切って火星に向かうと、画期的な光景を待つ必要がなくなったことが明らかになりました。 天文学者の予報はより控えめで控えめになりました。彗星の明るさは楽観的な仮定に遅れ、コマ（大気と尾）の形成の活動は期待に応えられませんでした。



9月下旬から10月上旬に、火星の軌道ステーションとローバーによって行われた彗星の観測結果を楽しみにしていました。 しかし、結果として、好奇心は空に彗星さえ見ることができず、MRO衛星の半メートルの望遠鏡は非常に軽薄な些細なことを送りました：







ここで、ハッブルとMROの調査の違いは、最初の彗星は直径約5万kmのコマで、もう1つは直径約2 kmのコアのみであるということです。



今日、彗星の観測は、多かれ少なかれ深刻な望遠鏡のほぼすべての所有者に利用可能です。 たとえば、直径10.5 cmの望遠鏡からの写真：







今日、多くの専門家やアマチュアが彗星を見ています。この目標を設定すれば、世界中からの定期的なレポートや新鮮な写真を簡単に見つけることができます。 もう少し、それは望遠レンズを備えた普通のカメラですでに撮影することができます。 確かに、これには晴れた空、フレアなし、肩からのまっすぐな腕が必要です。



NASAが不作為をシミュレートしている間、ハッブル科学チームは10月9日に新しいISON調査を実施しました（記事の冒頭の写真）。 この研究により、コアの直径-2 kmを明確にし、完全性を失わないことを確認できました。



それでは、軌道について話しましょう。

ISON彗星は、最近のPanstarrsの尾を引いたゲストよりも地球に近い距離を飛行しますが、とにかく、かなりの距離だけ離れています。 ISONと地球の最大収束は12月26日に発生し、距離は約6420万kmになります。 たとえば、これは、最大進入時に地球から火星までの距離を超え、地球から月までの距離は167です。 したがって、火星が地球に落下する可能性が除外されているため、ISONが地球に落下する危険性はありません。



Nasovサービスの軌道図または便利なオンラインサイトsolarsystemscope.comで ISON軌跡を追跡できます。







しかし、「私たちすべての賢明な」理論の支持者は、このスコアに関する言い訳を思い付きました：彗星が爆発し、その断片が私たちに落ちたらどうなるでしょうか？



まず、ブルースウィリスがリグと原子爆弾でISONに着陸するまで、彗星爆発の危険性はありません。 しかし、彗星はいくつかの断片に分裂する可能性があります。 これは、彗星が太陽に非常に近い結果である可能性が非常に高く、そのようなイベントは以前に繰り返し観測されています。



ISONは、星の表面から110万kmの距離を通過するはずです。 重力の影響と潮forces力から、いくつかの断片に分けることができます。 同様のことが、木星の彗星D / 1993 F2（シューメーカー-レヴィ）の落下中に観察されました。 重要なこと：断片化した彗星のすべての断片は、前の軌道で通常の動きを続けました。







また、彗星は内部プロセスの影響下で崩壊する可能性があります。 たとえば、パラノイアの以前の希望-エレニン彗星は、太陽に近づくと解散し、ほんのわずかな塵とガスの雲だけが地球に近づきました。



短周期彗星シュヴァスマン-ワッハマンは、1995年以降崩壊し続けています。







この彗星の3年ごとのラウンドで、ますます多くのデブリがありますが、それらはすべて以前の軌道を続けています。 ちなみに、この「尾の付いた星」は、崩壊したが気化していない彗星に起こることの良い例です。大きな破片は断片化し続け、徐々に核から遠ざかります。 これは、天体の重力摂動の違いから起こり、その隣で破片が飛ぶことができます。 破片までの距離に応じて、重力場はそれらに異なる影響を与えるため、彗星の尾は次第に広くなりますが、前の軌道は保持されます。 テール拡張速度はわずか45 km / hです。



そして、リニア彗星の崩壊により、科学者たちはほとんど「ライブ」を捉えることができました。







これはISONで再び起こる可能性がありますが、シュワスマン-ワッハマンの現在の彗星との違いは長周期です。 彼女の次の帰還は200万年後にのみ予想されるため、1ヶ月で彼女の破片は前の軌道から6000万km離れることができなくなります。 つまり、地球に最も接近した時点で彗星が崩壊した場合、その破片は約7万5千km近く接近します。これは、宇宙規模では取るに足らないものです。



はい、1月の地球はISON彗星の尾に近づきますが、彗星から太陽への接近まで残ったのは尾です。 大きな断片はありません。 彗星の尾自体を通過することは、地球と地球人にとってひどいことではありません。 実際、私たちの惑星は定期的に彗星の軌道を横切っており、同じ「尾」が伸びています。 地球と「尾」流星群の会合の日が観察されます。 おそらく誰もが8月の空の流れ星を賞賛していましたが、誰もが地球がスイフトタトル彗星の軌道を横切る結果を観測しているとは考えていませんでした。 私たちが天気に恵まれていて、彼女自身が空に現れなくても、ISONが私たちにカラフルな光景を与えることを願っています。



それでは、中世の恐怖で育った偽物について話しましょう。

1日以上ユーフォロジストのフォーラムで議論されてきた写真とビデオを分析します。



新しい神話の基礎となったのは、「ISONが2隻の船を護衛する」という現象であり、その本質は「彼らは愚か者のために機能していないようだ」という言葉に簡潔に反映されています。



2013年4月30日、NASAはISON彗星が観測された天球の素晴らしいカラー写真を公開しました。







この場合、すべてのオブジェクトはカラーで表示され、彗星は白黒です。 ご存知のように、カラーデジタル画像は、3つのカラーチャネルを介した3つのショットの結果です。 しかし、彗星と望遠鏡が軌道上を移動し、露出に一定の時間（最大490秒）がかかるという事実により、すべての射撃オブジェクトが所定の位置にとどまるようにチャンネルを減らすことは不可能です。 写真は星に縮小され、彗星は色なしで残されました。 しかし、これはほんの始まりに過ぎませんでした。 完成した公開画像に加えて、色が収集された異なる色チャンネルの3つの画像もハッブル望遠鏡の生画像アーカイブに配置されました 。 これらのフレームのそれぞれで、彗星は異なる位置にありましたが、調査中の長いシャッタースピードは彗星の核のイメージが引き伸ばされることにつながりました。 彗星に対する軌道望遠鏡の位置も変化しているため、さまざまな露出がはるかに遠い背景に対する核のさまざまな位置を捉えました。







誰かがこれらの写真を見つけ、それらを1つにまとめて「護衛付きの彗星」を手に入れました。







ウイルスが起動し、すぐにそれで遊ぶことにした人がいました。 秋には、おそらく彗星の核のレーダーショットを示す2つ（またはそれ以上）のビデオがYouTubeに登場しました。 ビデオは、特定の中国の衛星の仕事の結果として位置付けられています：







2番目のビデオでは「護衛」のデモも行います。「中国のレーダー」の名前のみを明確にします。







まず、「レーダー」を扱います。 Tianlian I（Sky Chain）は、Shenzhou宇宙船とTiangongステーションと地球との安定した通信を提供する一連の静止中継衛星です。 この点で、その機能はアメリカのTDRSデバイスまたは「レイ」に似ています。







どうやら、偽装者は展開されたプレートを見た後、レーダー機能を割り当てることにしました。



しかし、これらのデバイスがレーダーであり、非常に長いベースの電波干渉計のモードで動作したとしても、YouTubeでビューを収集する写真を提供することはできませんでした。 偽造者の間違いは射撃の日付でした。 それを知っていれば、彗星までの距離を簡単に決定し、核の角の大きさを計算し、画像を表示する能力を評価できます。 2013年9月3日、ISON Cometは約4億5,000 kmの距離にありました。 彗星の核の大きさは2 kmです。 これに基づいて、式a = arctan（d / L）でその角度サイズを計算し、約10角マイクロ秒またはマイクロ秒のアークを取得します。 つまり、このような解像度の電波望遠鏡を使用すると、コアがドットの形で表示され、ビデオの詳細が桁違いに高くなります。 しかし、10マイクロ秒に留まります。 それはたくさんですか、それとも少しですか？



今日、天文学では、ロシアのSpectrum-R宇宙望遠鏡とロシア、ヨーロッパ、アメリカの地上ベースの電波望遠鏡のネットワークを使用したRadioastron国際プロジェクトで、電波範囲（およびその他の方法）で最高の解像度が得られました。 彼が達成した最大解像度は、27マイクロ秒のアークです。 彗星の核を観察するのに必要な量の3倍。 Radioastronはユニークなプロジェクトであり、中国での経験は今後も同じことを望んでいます。 Spektr-R宇宙船は地球から数万キロメートル離れており、これにより、地球に土星の直径のレーダー（リングなし）があるかのように、非常に長いベースで電波干渉モードでデータを受信できます。 Radioastronに加えて、中国も世界の他の国にもそのようなレーダーはありません。 もちろん、中国の閉鎖的な性質とその暗い科学の天才について言及することはできますが、もしそのような望遠鏡があれば、有人飛行や月探査プローブのように、彼らはそれを宣伝活動に積極的に使用します。



さらに、電波源を調べることで27マイクロ秒のアークのような高解像度を達成することが可能でしたが、彗星はそうではありません。 地球の近くを飛んでいる小惑星はレーダーでそれらを照射することによって検査されていますが、軌道車は4億5000万kmを超える物体を照射するほど多くのエネルギーを見つけられません。



彗星に関する偽物の作成者が考慮しない次の詳細は、レーダー画像を取得する原理です。 最初のビデオでは、「額」を見ているかのように彗星を示しています（観測軸に対して約45度の角度で飛行しているため、これは事実ではありません）が、レーダーはビデオカメラではないため、信号反射のため、画像はプロファイルに表示されます：







ここで、小惑星1998 QE2を撮影するとき（ちなみに、「伴奏」を使用）、レーダーは光源として機能し、見かけ上は太陽ではありません。 そして、「プロファイル内」にあるオブジェクトを観察します。 超音波スキャンが行われたときに同じ原理が得られます：







「中国のレーダーからの」2番目のビデオには別の誤りがあります。それは彗星の尾の方向を示し、ビデオ中にこの方向は180度変化します。 彗星の尾は常に太陽と反対の方向に向けられているため、尾の方向のそのような変化は、2つの場合に可能です：撮影中に彗星が太陽を回ったか、彗星を回した宇宙船から調査が行われました。 もちろん、どちらも起こらなかったので、たとえそのような品質の画像を与えることができるレーダーがあったとしても、それは私たちに全く異なる何かを見せてくれるでしょう。
