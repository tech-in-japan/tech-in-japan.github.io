Googleがプレゼンテーションで示したもの





数時間前、Googleはニューヨークで大きなイベントを終え、そこで新しいガジェットを発表しました。 これは、Androidの10年のお祝いと一致し、もちろん、新しいPixelスマートフォンに主な注意が向けられました（以前には多くのリークがありましたが、公式モデルは香港でも販売され始めました）。







プレゼンテーションはAppleのプレゼンテーションよりも長く続き、すべてが白い背景に囲まれ、スティーブジョブズシアターの黒いステージとは対照的でした。 正直なところ、Appleではすべてがはるかに高価でプロフェッショナルに見えました。 Googleは新しいデバイスをさらに多く導入しましたが。 そして最も重要なのは、さまざまな分野のすべてです。ここには、スマートフォン、タブレット、ホームハブ、テレビ用のメディアプレーヤー、さらに人工知能を使用する新しい（かなり予期しない）方法があります。







Google Pixel 3およびGoogle Pixel 3 XL



Googleは2017年に約400万台のスマートフォンを販売しました-1年前の2倍です。 しかし、まだ大量認識はまだ先のことです。 比較のために、Appleは昨年約2億1,700万台のiPhoneを出荷しました。 しかし、Googleはここでのポイントは利益ではなく、競争だと言っています。 そして、最大の誇大広告は、同社の新しいスマートフォンのリリースの直前に形成されました。 前回のように、Pixel 3とより大きなPixel 3 XLの2つがありました。 ソフトウェアのすべての変更は別として、主なアップグレードはワイヤレス充電と広角のフロントカメラです（たとえば、写真にたくさんの人を入れたい場合は、よりクールなセルフィーを取るため）。











2つのスマートフォンの仕様は基本的に同じです。 背面カメラは12メガピクセル、前面-8メガピクセルです。 IP68に準拠したほこりや湿気からの保護。 Snapdragonプロセッサは845、6 GBのRAM、ドライブは64または128 GBです（このデバイスでは、 最後に失われるのは256と512 GBの両方を持つ最後のiPhoneです）。 microSDカードスロットは提供されていません。 モデルにはAndroid 9.0 Pieがプリロードされています。







バージョンはサイズ、ディスプレイ、バッテリーが異なります。 Pixel 3は、解像度2160x1080（443 ppi）の5.5インチOLEDスクリーンを備えています。 Screen Pixel 3 XL-6.3インチ（2880x1440ピクセル、523 ppi）。 小さいバージョンは標準に見えますが、大きいバージョンは最新のiPhoneのように、画面上に特徴的な「眉」があります。







ピクセル3のバッテリー容量は2915 mAhで、XLバージョンでは3430 mAhです。 ワイヤレス充電の可能性を開くために、両方のスマートフォンは背面にガラスがあり、すでに宣伝されているマット仕上げが施されています（素敵な、指紋がつかない、市場の他のすべてのガラス製スマートフォンよりも汚れが遅いなど）。 そこに、後ろに-指紋スキャナー。 GoogleはまだAppleのような顔でスマートフォンのロックを解除することを信じていません。 デバイスの端を押すと影響を受けやすくなります。スマートフォンを手に握ると、音声アシスタントのGoogle Assistantを呼び出すことができます。











ガジェットは、ソフトウェア、特にカメラソフトウェアの多くの更新を受け取りました。 多くの評論家によると、Pixel 2はiPhoneがどのように行うかを知っていたものの品質で先を行き、新しいGoogleモデルがオールインしました。 彼らは伝説的な写真家のアニー・ライボヴィッツの支援を求めました。 ライボビッツは、まだ名のないプロジェクトのためにPixel 3で国を旅し、「感動的な人と場所」のポートレートを撮影します。











カメラの機能の中には、AIを使用してぼやけた写真や観客の1人の目が誤って瞬きするなどの一般的な写真の問題を修正するトップショットモードがあります（問題は、写真を撮る前後に追加のフレームを修正することで修正され、レイアウト）。 同じ貯金箱-スーパー解像度ズーム。これは、カメラがズームモードのときに画像を鮮明にするのに役立ちます。 また、Night Sightテクノロジーにより、暗い場所でもフラッシュを使用する必要はないとGoogleは述べています。







もちろん、ARの革新がないわけではありません。 多かれ少なかれ興味深いものから、彼らはマーベルとチームを組み、さまざまなスーパーヒーローのステッカーをPixel 3カメラアプリケーションに追加しました（その後、Pixel 2 / Pixel 1に転送することを約束します）。 Pixelのこれらのステッカーを使用すると、画面に3Dキャラクターを配置して、写真を撮ったり、バックグラウンドでビデオを撮影したりできます。 アイアンマンや帝国のスターウォーズ兵士など、一部はかなりリアルに見えます。 部屋の隅で彼と一緒に写真を撮ると、多くの人はあなたが本当にそのような姿を持っていると信じます。 Googleがプレゼンテーションでそれを示した方法-あなたが彼らと友達であり、文字通り腕の中で写真を撮っているように-はまだあまり自然に見えません。











新しいスマートフォンには、白、黒、「ピンクではない」の3色があります。 Pixel 3の価格は799ドル、Pixel 3 XLは899ドルです。 これは新しいiPhoneよりも明らかに安い（999ドルと1099ドルあります）。 デバイスは12か国（米国、日本、アイルランド、インド、フランスなど）で販売されます。 ロシアはリストにありませんが、いつものように、あなたはアメリカの住所でそれを買うことができます、そして、誰もあなたに尋ねません。







どちらのスマートフォンもGoogleから直接予約注文できるようになっており 、出荷は10月18日に始まります。 無料で6か月分のYouTube Musicが含まれ、さらに所有者は2022年1月までGoogleフォトサービスに無制限にアクセスできます。







Googleホームハブ



音声アシスタントが組み込まれた7インチ画面。 実際、同じ列はGoogle Homeで、接続されたタッチスクリーンのみがあります。 ビデオを再生したり、音声クエリ結果を出力したり、家の他のスマートデバイスを制御したりできます。 グーグルは、このデバイスは家の中でほぼ基本的なものであり、他のすべてのシステム（できればグーグルのアシスタントも含む）が起動されると考えています。 寝室に安全に置くことができるように、同社は特にカメラを装備せず 、画面は周囲の照明のレベルに自動的に調整します。











GoogleフォトとYouTubeサービスのインターフェースは、音声とタッチスクリーンの両方で便利に制御できるように、Google Home Hub向けに作り直されました。 Googleの一部であるGoogleマップおよびすべてのNest製品（サーモスタット、外部カメラ）との統合があります。 デバイス自体のデザインは非常に興味深いものです。GoogleHomeのソフトコラムで、まるでカットされているように、画面が上部に取り付けられています。 デバイスを使用しない場合、デバイスは自動的にデジタルフォトフレームに変わり、Googleフォトの写真が回転します。 この場合、ぼやけた（およびわいせつな！）写真は自動的に除外されます。







主な楽しみは、GoogleがAmazonのスピーカーとYouTubeの接続を依然としてブロックしていることです。これは、企業同士の深刻な敵意のためです。 そして、画面のある列の場合、これは実際、最も重要で最も人気のあるサービスです。 30分間のプレゼンテーションの代わりに、彼らはここに出て、「エコーショーが好きで、YouTubeでしか動作しないので、こんにちは」と言って、マイクを落として出かけることができます。 ロックが有効で、AmazonTubeがまだプロジェクトにある間は、画面を備えたGoogleスピーカーが常に有利です。











先行予約が開始され、列の価格は149ドル、出荷は10月22日に始まります。 ボーナスとして6か月分のYouTubeプレミアムを提供します。







Googleピクセルスレート



Pixel C（2015）以来の会社からの最初のツーインワンタブレット、およびChrome OS上のGoogleからの最初のタブレット。 デスクトップモードとタブレットモードを自由に切り替えることができます。さらに、Google Playの適切なアプリケーションの完全なセットです。 Microsoft Surface Proのライバルとして、コードネームNocturneで密かに作成されました。 前面には2つのスピーカー（ミニステレオ用）、2つのマイク、前面カメラがあり、背面には別のカメラとGoogleロゴがあります。 両方のカメラ-8メガピクセル。 スタイラスの使用をサポートします。スタイラスはパッケージに含まれていません。











上記は、指紋が組み込まれた電源ボタンです。 2つのUSB-Cコネクタがあり、各側に1つあります。 ヘッドフォン用のミニジャックはありません（悲しみ）。 側面-より良い無線通信を提供するアンテナ。







このデバイスは、4 GBから16 GBのRAM、SSD 32-256 GB、最大第8世代Intel Core i7までのプロセッサーを備えた大量の仕様で提供されます。 画面-12.3インチ、3000x2000、LCD。 重量-725グラム。 バッテリーの寿命は最大10時間です。







私が驚いたのは、このツーインワンタブレットにキーボードが付属していないことです。 別途購入する必要があります。 キーボードがなければ、ラップトップにはなりません。 確かに、このソリューションにはプラスがあります-Bluetoothキーボードを既にお持ちの場合、問題なくPixel Slateに接続し、基本的にはアイロンとラップトップを備えています。 キーボードがない場合、BrydgeはPixel Slateスタイル専用のBluetoothバージョンを149ドルでリリースし、Google自体がスロットに接続してタブレットを本格的なラップトップに変えるオプションを199ドルで提供しています。











Pixel Slateの価格は、詰め物に応じて599ドルから1,599ドルです。 キーボードまたはスタイラスは別々に取ってください。 事前注文は開いており 、出荷の正確な時間については伝えていません。 YouTube TVの3か月は、タブレットのボーナスです。







Googleピクセルスタンド



ワイヤレス充電ステーション。フォトフレームとしても使用できます。 スマートフォンはここにはありませんが、置かれています-インテリアの美しさの一部になり、一般的にはより便利になります。 Pixel 3がそのようなステーションに配置されている場合、そのインターフェイスは変化し、遠くから読みやすくなります。 その後、スマートフォンに音声コマンドを与えることができ、たとえば、充電中に写真を変更したり、音楽を入れたりできます。 このモードでは、特別なアラーム設定「サンライズ」もあり、より穏やかな目覚めを提供します。











ステーションにはUSB-Cポートがあります。 ワイヤレス充電では、 Qi標準と互換性のあるデバイスがサポートされます。 事前注文のために開いているピクセルスタンドは、79ドルの費用がかかります。







AIはテレマーケティングと戦います



非常に重要ではありませんが、非常に面白いです。 新しいスマートフォンは、迷惑な電話に対処するための絶対にユニークな方法を可能にします。 これには、Duplex AIが含まれています。これは、「不気味な」音声ボット、人と区別できないすべてのコールセンターの雷雨です。 あなたがそれをオンにすると、AIはあなたに代わって電話に出て、データを受け取り、彼らがあなたから何を望んでいるかを理解します。 そして、それは絞り出されて、画面上の有用な情報を提供します-誰が電話しているか、彼が話したいことの名前。 そして、ボットが話している間、コールを受け入れるか、無視するかを決定できます。 ただし、デュプレックスAIは非常に有名であるため、英語のみでチューリングテストに合格しています。







Googleピクセルイヤホン



また、Googleはプレゼンテーションで何も言わずに黙って、今日その店で新しいヘッドフォンの販売を開始しました。 さらに、それらはすでに完全に配信の準備ができています。つまり、予約注文ではありません。 機能とデザインの点では、これらはまったく同じGoogleワイヤレスヘッドフォンであり、リアルタイムの翻訳機能とボタン1つでのGoogle Assistantのサポートも備えています。 それらは、Bluetoothではなく、USB-Cを介した有線で動作します。 したがって、価格はほぼ1桁安く、わずか30ドルです。
