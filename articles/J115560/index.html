Skalaxiの更新：Windows、クローニング、およびIPアドレス

ご挨拶！

さまざまな困難にもかかわらず、Skalaxiクラウドプラットフォームは開発を続けています。 不快な問題の解決に加えて、私たちのチームが今年の初めから何をしていたかをお伝えしたいと思います。



昨日、私たちは新しいサイトを立ち上げましたが、それは訪問者にとってより明確で友好的になることを願っています-今すぐメインページで必要な製品を正確に選択して、すぐにそのすべての特性を見ることができます。



今日の主なニュースは、Windows仮想マシンのクラウドでのサポートです。 2か月のテストが完了しました。ベータテスターに​​なり、新しい機能のテストに協力してくれたすべてのユーザーに感謝します。 クラウドWindowsは誰でも利用できるようになりました。 このサイトでは、 Microsoft Windows Server 2008 R2ベースの仮想サーバーを1分で作成できます。 Web Server、Standard、およびEnterprise Editionの3つのエディションが利用可能です。 関税とライセンスについても言及したいと思います。 スロットのコストは使用するOSに依存せず、依然として65コペック/時間または468ルーブル/月です。 ただし、Windowsマシンの場合は、ライセンスのコストが追加で支払われます。これは、エディションに応じて、それぞれスロットあたり30、40、または50コペックです。 ライセンスの費用は、マシンの電源が入っている場合にのみ引き落とされます。

マシンに実際のIPアドレスがあり、アクションを実行する必要がない場合、アクティベーションは自動的に行われます。 マシンをリモートで接続および制御するには、vnc、rdp、またはWebコンソールを使用できます;詳細については、ドキュメントを参照してください 。



2番目の重要な新規性は、 仮想マシンのクローン作成の可能性を検討することです。 ワンクリックで、仮想サーバーとそのすべてのディスクの完全なコピーを作成できます。 この機能をバックアップに使用できます。 支払いは付属のサーバーに対してのみであることに注意してください。



それほど重要ではないが最も望ましい変更のうち、動的IPアドレス管理について言及できます。 これで、マシンを停止せずに実際のIPアドレスを追加、削除、変更できます。 また、DNSゾーンコントロールパネルはキリル文字ドメインをサポートするようになりました。



製品の開発だけでなく、インフラストラクチャサービスとクラウド全体の低レベルの基盤も開発することが重要です。 第一に、私たちは現在、機器の別のバッチを試運転しています。現在、2つの6コアIntel 56シリーズプロセッサがサーバーにインストールされています。 次に、ソフトウェアスタックを更新しました。現在、Xen 4を使用しています。これにより、多くの機能を追加できます。



登録時に250ルーブルのクラシックボーナスが有効であることを思い出させてください。
