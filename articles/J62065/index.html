UbuntuでのFreeCall





私はIPテレフォニーの幸せなユーザーであり、特に-FreeCallです。 私はこのサービスの多くの利点すべてにこだわるつもりはありません-私はマイナスにこだわるでしょう：ネイティブクライアントはWindows上でessnoに設計されています 。 （ちなみにWineにインストールすることはまったく難しくありませんが、1C以降は動作しません。）



30分インターネットに登り、上記のプロバイダーからのIP電話を厳しい現実に適応させるための明確なステップバイステップの指示を見つけられなかったため、 ubuntuは自分で書くことに決めました。



まず第一に、ubuntaはEkigaを持参して置くことを忘れなかったと確信しています。 （突然忘れてしまった場合は、 @ホストに名前を付けてください：〜$ sudo apt-get install ekigaが役立ちます ）。

次に、それを実行します。 セットアップウィザードの質問に答えるとき、特に何も書かないでください。 とにかく、すべてを変更してください:)





[ 編集 ] > [アカウント ]をクリックし、選択した名前を右クリックして、もう一度[ 編集 ]をクリックします。

表示されるウィンドウで、次を変更します。



名前： freecall.comで登録するときの名前

レジストラ： sip.voiparound.com

ユーザー： freecall.comで登録する際の名前

ユーザー： freecall.comで登録する際の名前

パスワード： freecall.comで登録するときのパスワード

遅延：デフォルトのまま-3600

「アカウントを有効にする」チェックボックスはそのままにしておくことを忘れないでください



[編集]> [オプション]。 表示されるウィンドウで、次を変更します。

一般>個人情報>名前： Freecall.comへの登録時の名前

[プロトコル]> [SIP設定]> [送信プロキシ]： sip.voiparound.com

[プロトコル]> [SIP設定]> [転送URL]： sip.voiparound.com

プロトコル> H.323設定>リダイレクトURL： sip.voiparound.com











ダイヤルするとき、番号は次のようになります。 主なことは、国際形式で番号をダイヤルすることです。
