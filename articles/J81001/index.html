バックアップディレクトリのスクリプト

Windowsを実行している1台のコンピューターにアーカイブフォルダーがあり、その損失は死に似ています。 さらに苦労することなく、15分間、ディレクトリをアーカイブしてFTPサーバーにアップロードする簡単なスクリプトを作成しました。 おそらく最もエレガントなソリューションではありませんが、誰かを助けることができます。



7zアーカイバーがコンピューターにインストールされ、その役割を果たし、 ftpコマンドを実行し、 -sスイッチの後のパラメーターとして、スクリプトと同じディレクトリにあるテキストファイルftp.txt を転送します（この場合、Cドライブのルートにあります:)



cd c:\"Program files"\7-zip

# garbage backup.zip

7z a -tzip c:\backup.zip c:\garbage

cd \

# ftp

ftp -s:"ftp.txt"











ファイルの内容は次のとおりです。



# IP ftp-

open 192.168.1.1

#

admin

#

admin

#

lcd c:\

#

put backup.zip

#

bye









次に、タスクをスケジューラに追加すると、すべてが機能します。
