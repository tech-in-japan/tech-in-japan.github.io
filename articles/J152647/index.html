予算資金によるイベントのインターネット放送の組織

入門



この記事は、イベントの主催者、トレーナー、教師が、機器やソフトウェアの購入に最小限の費用でイベントのオンラインブロードキャストを開催することを目的としています。



この記事では、無料のオンラインブロードキャストを編成するために必要な手順について詳しく説明します。 必要な機器のリストとインストール方法について説明します。 オンラインブロードキャストを整理するときにオーガナイザーが直面する主な問題と、可能な解決策について説明します。



ユースケース



オンラインブロードキャストは、条件付きで2つのカテゴリに分類できます。



  • インターネット上のオフラインイベントのビデオ放送：会議、セミナー、円卓会議;
  • Web会議：多数の参加者が相互にビデオチャットする。




最初のケースでは、主催者は原則として、参加者が集まる部屋を借りると同時に、このイベントのオンラインブロードキャストを開催します。 この場合、高品質のサウンドとビデオストリームを送信し、ビデオとオーディオ機器を使用する必要があります。

2番目のケースでは、参加者は、組み込みのWebカメラまたはUSBカメラを備えた自宅のコンピューターまたはラップトップを使用して、自宅でWeb会議を開催します。

  • デジタルビデオカメラ

  • ケーブル：オーディオケーブル、HDMIケーブル、バランスケーブル。

設定と接続の手順



オフラインイベントのオンラインブロードキャストを整理するには



オフラインイベントのオンラインブロードキャストを整理するための機器の初期インストールの手順：



1.ビデオキャプチャカードを接続し、必要なドライバーをインストールします（原則として、必要なソフトウェアが含まれています）。 ドライバーをインストールした後、コンピューターを再起動する必要がある場合があります。

2.ビデオキャプチャカードを介してビデオカメラをコンピュータに接続します（HD放送の場合はHDMI経由、SD放送の場合はUSD経由のビデオキャプチャカード経由）。

3. Webサイトwww.adobe.comに登録し、Adobe Flash Media Live Encoder 3.2をコンピューターにインストールします。 （http://www.adobe.com/products/flash-media-encoder.html）

4.ビデオキャプチャカードがカメラからのビデオ画像を見るかどうかを確認します。



これを行うには、プログラムAdobe Flash Media Live Encoder 3.2を実行します。 [デバイス]フィールドで、ビデオキャプチャカードに対応するインストール済みソフトウェアを選択します。







ソフトウェアが正しくインストールされ、ビデオカメラからの画像がキャプチャされると、Adobe Flash Media Live Encoderでビデオカメラからの画像が表示されます。







ビデオキャプチャカードドライバーを正しくインストールしても、正しいパラメーターを選択するまで、Adobe Flash Media Live Encoderの画像が表示されないことがあります。

  • 形式：H.264である必要があります。
  • フレームレート。 Blackmagic Intensity Proビデオキャプチャカードの値は25.00である必要があります。
  • 入力サイズ（可能な最大値を設定）。








5.サウンドを接続します。 これを行うには、ラジオマイクシステムをオーディオミキサーに直列に接続します（ジャックケーブルまたはバランスカメラを介して）。







次に、オーディオミキサーをオーディオケーブルでコンピューターに接続します（一方にはミキサーに接続する2つの赤と白のベルがあり、もう一方にはコンピューターのマイクまたはライン入力に接続するミニジャック入力があります）。 この図は、オーディオケーブルをオーディオミキサーに接続するための入力を示しています。







6. Adob​​e Flash Media Live Encoderで、音源を選択します。







マイクに向かって話すときに音がするかどうかを確認します。 サウンドがキャプチャされると、色の表示が表示されます。







7. SeeMedia.ruインターネットサービスでイベントを登録および作成する

8.作成されたイベントに移動してSeeMediaサービスのセミナーカレンダーに移動し、Adobe Media Live Encoderアイコンをクリックします。









2つのFMS URLおよびストリームリンクが画面に表示されます。







次の2つのリンクをAdobe Media Live Encoderに貼り付けます。









9.図に示すように「ビットレート」ストリームを設定し、「ファイルに保存」の隣のボックスのチェックを外します。 次に、「開始」ボタンを押します







異なるレベルのビットレートを使用すると、インターネットチャネルの帯域幅が異なるリスナー向けに、さまざまなビデオ品質のブロードキャストチャネルを作成できます。

ビットレートの第1レベルを300キロビット/秒でダウンロードし、インターネットに接続していないユーザー、たとえば400キロビット/秒（3Gモデム経由で、遠く離れた地域から）が接続できるようにします。 また、800 Kbpsの2番目のビットレートを使用すると、900 Kbpsを超えるインターネットチャネルを持っているユーザーに高品質のブロードキャストを視聴できます（80〜100 Kbpsが音声伝送に使用されます）。

すべてのユーザーがこの最初の低いビットレートに接続し、必要に応じてより高いレベルのビットレートを選択できるように、最初の（トップ）ビットレートを2番目よりも低く設定することは常に価値があります。 ビットレートの最初のレベルを2番目のレベルよりも高く設定すると、すべてのユーザーが高品質に接続し、インターネットの品質が低いユーザーには画像がまったく表示されず、不満が生じます。



Adobe Media Live Encoderの[スタート]ボタンをクリックすると、次のウィンドウが表示されます。閉じないでください。







10.「Enter webinar」ボタンをクリックして、SeeMediaのウェビナーに移動します。









その後、SeeMediaのインターネットブロードキャストウィンドウに直接移動します







11.ブロードキャストリンクを友人および潜在的なリスナーに送信します。



SeeMediaのセミナーのカレンダーで、「コード」アイコンをクリックします







そして、開いたウィンドウで、リンクをコピーします。







12. Webキャストでウィンドウを表示および配置するための便利なオプションを選択します。



[ワークスペース]ボタンを押して、便利な表示オプションを選択します。

重要な問題またはよくある質問



ウェブキャストにはたくさんの音があります



電気ケーブルの端やきらめく接点の接続が不十分なコンセントの接続不良がある可能性があります。 すべての機器を別のコンセントに差し込みます。



オーディオキャプチャボードのドライバーを構成した後、オペレーティングシステムは、ドライバーが正しく接続されておらず動作しない可能性があるというメッセージを書き込みます



コンピューターにあったすべてのWebカメラのすべてのドライバーを削除し、インストールを再試行します。 ウェブカメラドライバーとビデオキャプチャカードの間で競合が発生する場合があることに気付きました。



ビデオキャプチャカードドライバは正しくインストールされていますが、Adobe Flash Media Live Encoderには表示されません



この場合、最初にビデオキャプチャカードに付属のソフトウェアを実行し、ソフトウェアでビデオを表示してから、Adobe Flash Media Live Encoderを実行する必要があります。 したがって、少なくとも、AVerMedia AVer3D CaptureHDは機能します。



オーディオミキサーを接続すると、誤った低音と高音の周波数が送信されます



低周波数（低音）は最小限に抑えられます。 中域をバランスより少し低くし、高域をバランスより少し大きくします。



ユーザーがウェブキャストに接続できないと不満を言う



ユーザーはポート1935を開いている必要があります。



ユーザーは、ビデオ画像が遅くなり、音が途切れるという不満を訴えます



より広いインターネットチャネルに接続するようにユーザーを促します。 Adobe Flash Media Live Encoderで2つの異なるビットレートを設定します。
