Windowsの場合、私は長い間存在しているため、あなたは望んでいますが、望んでいませんが、毎日使用しているものに慣れています。 90％のケースで、Opera + Google Docs + Skypeで十分だと気づいた後です。 Windowsでは、ロードとアンロードの時間（オフにしたとき）にますます悩まされました。 システムの全体的な速度。



したがって、次のパラメーターを満たす軽量OSを見つけるタスクを自分で設定しました。

-すばやくロードおよびアンロード

-ダミーのように簡単にインストールできます

-すぐにWiFiカードとネットワークを受け取ります（WPA暗号化を使用した非表示のWi-Fiネットワーク）

-不要な問題なしで（または、箱から出して）Operaをインストールできる

-すぐに使えるロシア語のサポート（まあ、または簡単なアドオンを使用）



追加の利点は次のとおりです（そのまま）：

-Skypeによりインストール

-PDFドキュメントビューアー。

-クリーンで最小限のインターフェイス（Windowsでは、エクスプローラーの代わりに、Blackboxポートがあります-BBlean）
