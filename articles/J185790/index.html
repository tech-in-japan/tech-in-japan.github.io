Logitechマウスの交換に関する別の成功事例

ハブラジテラムの挨拶。

障害のあるLogitech G700ゲーミングマウスの交換に成功した私の話をあなたと共有したいと思います。



マウスの使用が1年成功した後、その左ボタンはシングルクリックの代わりに機能し始めました。ダブルクリックではなく2回-3回です。

保証書に記載されているサービスセンターにマウスを移動しました。 3月31日でした。 4月27日に、問題が解決されたことが通知されましたが、現在はスクロールが機能しません。 私は彼らがそれを修正するまで待つことにしました。 6月1日、彼女は同じ障害のある巻物で私に戻った。







Logitechに電話しましたが、マウスがこのような誤動作に置き換わるというニュースに満足していました。いくつかの簡単な条件を満たせばよいのです。



アレクサンダー、私たちはあなたによって提供された保証の下であなたのデバイスと交換します



1.購入を確認するドキュメント（または公式コピー）のスキャンまたは写真

1.a-保証書のスキャンまたは写真



2.手の上にあなたのアピールの番号が書かれた紙の上にあなたのデバイスの写真130603-002919;



3.デバイスの写真と読み取り可能なシリアル番号（P / N、M / N、PID）



4.水の容器に入れたデバイスの写真：

-デバイスの電源を入れ、

-バッテリーカバーを取り外し、

-バッテリーをデバイスに入れたままにする（独自のバッテリーを使用する場合-キットに付属の元のバッテリーを挿入する）

-できれば透明な水タンクにダイヤルイン

-紙の上にあなたのアピールの番号を書いてください130603-002919

-コンテナの横に手書きの参照番号の付いたシートを置きます

-水の入った容器にデバイスを入れます

-写真を撮る







この手紙への返信として、郵便番号と電話番号とともに完全な住所を送信してください。 ただし、ロシアの特定の都市（http://www.ups.com/content/ru/ru/shipping/dropoff/index.html?WT.svl=PNRO_L1）/ウクライナ（ www.ups.com/ content / ua / ja / shipping / dropoff / index.html？WT.svl = PNRO_L1 ）。 これらの都市のいずれかに住んでいない場合は、これらの都市のいずれかのエージェントの連絡先情報（名前、姓、住所、郵便番号を含む、電話番号）をお送りください。




私は一日心配でした。 そして、翌日の昼食時に、私のマウスはすでに死んでいた。 約5分間死んで、インジケータで輝いています。



写真とスキャンをLogitechに送信した後、保証部門が24時間以内にファイルを確認し、48時間以内に追跡番号を受け取ることを約束した手紙を受け取りました。 また、故障したデバイスを返送する必要はないことも通知されましたが、交換品を受け取るまで保管することを強くお勧めします。



追跡は金曜日の午後9時に行われ、4日後に火曜日にマウスを受け取りました。 嬉しい驚きが私を待っていました。 結局のところ、私のマウスは製造中止になり、新しい高度なモデル、G700s充電式ゲーミングマウスが送られてきました。



新しいデバイスの保証期間は3年で、サポートのインシデントの開始から始まります。



そして、ここにボックス付きのデバイス自体があります（古いモデルとは異なることが判明しました）。





読者がトピックに注意を払ってくれたこと、Logitechデバイスの使用に成功したことを感謝し、同様のケースについて議論したいと思います。
