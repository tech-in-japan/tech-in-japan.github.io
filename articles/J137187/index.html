シマンテックの専門家は、AndroidマーケットでAndroidマルウェアを使用した13の人気アプリケーションを発見しました。





昨日、シマンテックからメッセージが公開され、Android.Counterclankマルウェアを含むいくつかの一般的なアプリケーションがこの会社の専門家によって発見されたことが報告されました。 アナリストによると、これは、Android.Counterclankを広めるウイルス作者の標的を絞ったキャンペーンである可能性があります。 配布に含まれるマルウェアを含むアプリケーションは500万人によってダウンロードされているため、現時点では、Androidマーケットにマルウェアを拡散させる最大の試みです。



前述のように、Android.Counterclankは、有名な出版社の13の異なるアプリケーションで見つかりました。 用途は、セクシーガールズパズルからカウンターストライクグラウンドフォースまで多岐にわたります。 これらのアプリケーションは、金曜日の終わりにまだ利用可能でした。 Symantec Security Response TeamのKevin Haleyヘッドは、アプリが実際のパブリッシャーによってロードされる可能性は低いと述べました。 ただし、これらはいわゆる再パッケージ化されたアプリケーションではなく、シマンテックの専門家はこのソフトウェアを見たことがある。







以前は、Android向けの悪意のあるソフトウェアのディストリビューターは、有名な出版社からアプリケーションを再構築し、さらに市場にダウンロードするという戦術を使用していました。 不正なユーザーは、できる限り信頼できると思われる開発者からアプリケーションをダウンロードし、モバイルデバイス用の悪意のあるソフトウェアを受け取りました。 今見たように、戦術は少し異なります。 予備的な見積もりによると、これらの13個のアプリケーションは100万回から500万回ダウンロードされています。 そして、これは明らかに、Androidマーケットの歴史全体における最大のマルウェア配布キャンペーンです（いずれにせよ、シマンテックの専門家はそう考えています）。 この種の攻撃は、2012年の最高のスマートフォンなど、Android OSベースのタブレットや携帯電話に影響を与えることは明らかです。



Android.Counterclankは、モバイルデバイスにインストールされると、デバイスの製造元、ユーザーのブックマークなど、あらゆる種類の情報を収集するトロイの木馬です。 さらに、このマルウェアはブラウザのスタートページを変更します。 このソフトウェアを使用する攻撃者は、侵害されたデバイスにさまざまな種類の広告を挿入し、お金を稼ぎます。 このソフトウェアはかなりの数のアクセスレベルを必要とするという事実にもかかわらず、ほとんどのユーザーはこれに注意を払っていません。なぜなら、このアプリケーションは有名な出版社のものだからです。
