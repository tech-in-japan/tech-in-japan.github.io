Linuxは電話を通じて大衆に？

私の妻には、Motorola ROKR E8電話が贈られました。これはロシアで最初のコピーの1つです。 私は数年前からコミュニケーターを使用しており、特定の軽emptを持つ電話を見ていますが、ここには本当に面白いことがあります。







Linux Javaを持っているという事実から始める必要があります（そうです、あなたは正しいと聞きました）。 モトローラは、長い間Linux電話を宣伝してきました。 プラットフォームはMOTOMAGXと呼ばれます。 これはかなり奇妙に見えます。GUIを含むJavaおよびJavaアプリケーションの上と下のカットオフLinux。







説明からわかるように、公式にはLinuxアプリケーションはこの電話では実行されません。 モトローラは、 Linuxベースの携帯電話用のJavaアプリケーションを開発できるEclipseベースのMOTODEV STUDIO開発環境を起動します。 ただし、ROKR E8にはLinux Java 7.3の新しいバージョンがあり、この携帯電話用のLinuxアプリケーションの開発はかなり現実的になったようです。



モデルの詳細と、そのモデルでLinuxを使用する方法についての考え。





電話の2番目の機能はキーボードです。これは、レベデフのOptimus'aと同様に、現在のアプリケーションに応じて変化します。 キーボードは標準ではありませんが、タッチスクリーンのようなものです。 すべてのコミュニケーターでは、ブラインドに仮想キーを押したときに十分な触覚フィードバックがありません。まともなものを入力することは非常に困難です（そう、iPhoneについて話している）。 ここで、エンジニアは簡単な解決策を見つけました。ボタンを押すと、電話がわずかに振動します-戻りがあります。



4つのキーボードオプションが見つかりました。



1.通常モード。 通常の電話のように、すべての番号が表示されます。







2. MP3プレーヤーモード。 プレーヤーのキーが表示されます







3.カメラモード。 カメラコントロールキーが見える







4.キーボードロック。 一般に、何も燃えず、押されません。







特別なデバイス（半円形のタッチキー）の助けを借りてスクロールすることは私を喜ばせませんでしたが、私のような人々にとっては、ジョイスティックコントロールが重複しています。

確かにすべてのMotorolのように、携帯電話内のインターフェースは便利ではありません。 クイック検索が非常に気に入りました。番号をダイヤルすると、デバイスは電話帳の番号を自動的に検索し、名前も照合します。 一度にWindows Mobileを選択したのは、この機能のためでした。 ROKR E8は、同じ機能を見つけた最初の携帯電話です。

写真やビデオをSMSに直接挿入すると、特にMMSになります。 中間アクションはありません-写真を撮ったところ、写真がすぐそこに挿入されました。



Linuxのコストがかかるという事実にもかかわらず、電話でのマルチタスクは見つかりませんでした。 唯一のもの-MP3プレーヤーはバックグラウンドで再生できます。 マルチタスクOSのすべての機能を使用しないのは十分奇妙です。

APIの正しい位置付けとオープンを備えたLinux Java（MOTOMAGX）は、Google Android、iPhone、Windows Mobileなどのスマートプラットフォームに匹敵し、「非タッチ」デバイスの空いているニッチを占める可能性があります。
