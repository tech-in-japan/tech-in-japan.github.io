セマンティックレイアウトワークショップ5月18日、モスクワ

5月18日に、エクストロポリスカンファレンスホールはセマンティックレイアウトと呼ばれるセミナーを開催し、4月のRIT 2007カンファレンスが主催する一連のトレーニングセミナーを継続します。 セミナーは、セクション「クライアントプログラミング」のスピーカーによって開催されます。

  • Vadim Makeev （あなたの謙虚な使用人、別名pepelsbey 、ブログhttp://pepelsbey.netの著者、RIT-2007でのCSS管理レポートの著者、 WSG-ロシアの参加者）
  • Alexey Rybakov （別名flack 、ブログhttp://flack.ruの著者、RIT-2007で評価されたレポート「 Layout Testing」の著者、 WSG- Russiaの参加者）


セミナーは条件付きで4つのトピックに分けられます。

  • 意味を探す：ドキュメントの内容をプレゼンテーションから分離する
  • コードとドキュメントのセマンティクスのすべての側面の概要：構造、要素の命名、マイクロフォーマット
  • セマンティクスを中心に構築されたコード、ページ、およびサイトの利点
  • 今日から始めましょう：「タグのスープ」を首尾一貫した、理解可能で、意味的に正しいコードに変換する例


退屈なアカデミックなプレゼンテーションスタイルを放棄することを事前に決定しました。

セミナーの基本はライブコミュニケーションです。私たちには伝えるべきことがありますが、おそらく何か質問があります。 教えて答えることができてうれしいです。



今日、セマンティクスは疑わしく、意味がなく、適用できないトピックだと思いますか？

私たちはあなたにその反対を納得させようとします。西欧とロシアの主要な開発者が今日セマンティクスをすでに使用していることと、それがもたらす利点を説明します。



このセミナーは、Web開発の幅広い専門家（プロジェクトマネージャー、部門長、開発者）を対象としています。



セミナーは午前4時に始まり、午後4時に終了する予定です。4つの論理的な各パートの後、無料のランチを備えた大きなランチルームを含む休憩が提供されます。



セミナーの費用は、セミナー開催前の登録と支払いの場合は5900ルーブル （Yandex-Money、WebMoney、非現金支払いは受け入れられます）、またはセミナー当日直接支払いの場合は7000ルーブル （現金は受け入れられます）です。



セミナーへの参加については、Victoria Galimullina、vica [at] list [dot] ru、ICQ：228192946、+7（926）578-68-74までお問い合わせください。

セミナープログラムに関する質問については、スピーカーにお問い合わせください：Vadim Makeev-pepelsbey [at] gmail [dot] com、Alexey Rybakov-alex [at] flack [dot] ru
