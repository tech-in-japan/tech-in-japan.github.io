信号分離と画面ノイズ

場合によっては、複数のソース（テレビなど）で画像を表示する必要があり、干渉が発生することがあります。 信号を分割して干渉を補正する方法に関する情報は次のとおりです。



信号分離。



1つのビデオソースから複数のレシーバーに到達する必要がある場合があります。たとえば、ビデオアイからの信号をドア近くのインターコム（小型テレビ）およびレコーダー（録画用）に出力します。



受信機に2つの信号強度がある場合、通常、BNCコネクタにはFコネクタ付きのティーまたは接続ティーが適しています。









極端な場合、ワイヤをはんだ付けすることができます。



3つ以上のレシーバーがある場合、問題が始まります。 接続オプション：



  • まず、録音デバイスに接続し、その出力（通常は全員が持っています）から、残りの2つ（3つのデバイスがある場合）を接続します。
  • 一致するデバイスを使用しますが、これが常に役立つとは限りません。
  • スプリッター付きアンプを使用します（Amplifier-splitters）。

雷保護。


雷保護装置は、ステーション（ビデオレコーダー、モニター、アンプなど）および周辺機器（ビデオカメラ、アンプ、回転装置など）をケーブル線で発生する高電圧から保護するために使用されます。



  1. 雷放電
  2. 産業干渉
  3. 溶接作業
  4. 強力な発電所






同軸ケーブルとツイストペアを保護するためのさまざまなコネクタを備えた単一チャネルの雷保護があります：BNC、F、UTP。 たとえば、16チャネルSP016Cにマルチチャネルがあります。



雷保護は、ビデオ監視回路、ローカルエリアネットワーク（LAN）、電源および制御回路を保護します。



絶縁体。


  1. 絶縁体は、接地電位の違いによる迷走電流によって引き起こされるビデオの歪みを除去できます。
  2. 迷走電流は、ビデオ監視システムのさまざまなポイントでの（電源ネットワークの不均衡な動作による）接地電位の違いにより発生し、ケーブルを流れ、歪みを引き起こします。
  3. 絶縁体は、異なる接地点間で発生し、主電源の不安定な動作によって生じる高電圧（最大380V AC）による損傷から機器を保護できます。
  4. 絶縁体は、システム回路を「アース」から（アース回路に接続された回路から）絶縁するように設計されています。
  5. ビデオ信号が異なる「接地」電位を持つポイント間でケーブルを介して送信される場合、アイソレータを使用する必要があります。
  6. 絶縁体は、新しく作成されたシステムに簡単に取り付けたり、既存のシステムに追加したりできます。

一致するデバイス。


一致するデバイス、またはトランスフィルターとも呼ばれ、ちらつきのあるフレームを削除するか、画像の「オーバーラップ」、通常はビデオ信号の歪みまたは低レベルのクロック信号によって引き起こされる画像の「​​分岐」。 この理由は通常、長距離で伝送するときのビデオ信号の「減衰」、干渉、ケーブル回線とビデオ機器の調整不良です。 マッチングデバイスとフィルターは、これらの問題を解決するように設計されています。







同軸ケーブルCB001VHの調整デバイスは、ケーブルラインおよびケーブルラインのセクションの抵抗と機器との不十分な調整によって引き起こされる干渉を排除するように設計されています。 入力/出力：BNCプラグ/ BNCソケット; 受動的、電力を必要としません。 同軸ケーブル線の接続点および機器へのケーブル線の接続点に設置されます。 無線信号、他のビデオ信号、近くの電源ケーブル、電話、蛍光灯、変圧器などによって引き起こされる干渉から保護します。



ツイストペアTVB001用のマッチングデバイスがあります。これは、ケーブルラインおよびケーブルラインのセクションと機器との抵抗のマッチングが不十分なために生じる干渉を排除するように設計されています。 受動的、電力を必要としません。 抵抗を調整するためにツイストペアの接続ポイントに設置されています。 4個の接続クリップAP007（粘着テープ）が含まれています。
