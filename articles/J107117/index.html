CSS 2D変換とは何ですか？



CSS 2D Transformsは、開発中のCSS3標準の一部です（現在ドラフト状態です）。



2次元変換を使用すると、CSSを介してさまざまな要素のアフィン変換を記述することができます：変換マトリックスを含む転送、スケーリング、回転、および傾斜。



たとえば、このようなCSSスタイル 



  DIV { 
     transform：translate（100px、100px）;
 }




適用される要素を100px下および右にシフトします。







45度のスケーリングと回転を追加するには、スケールでスケールを指定し、変換で回転角度を指定する必要があります。 



  DIV {
      高さ：100px; 幅：100px;
      変換：平行移動（80ピクセル、80ピクセル）スケール（1.5、1.5）回転（45度）;
 }




重要：CSSのこの部分はまだ開発中であり、推奨候補とはほど遠いため、IE9 PP6の実験モードで追加され、プロパティにプレフィックス-ms-を追加する必要があります（例：-ms-transform）。



セマンティックタグとは何ですか？



セマンティックタグは、html5で導入されたタグで、ページレイアウトをより意味のあるものにします。 html 4.01と比較した新しいタグと変更の完全なリストは、 http：//www.w3schools.com/html5/html5_reference.aspにあります。



現在認識されているタグは、セクション、nav、記事、aside、hgroup、ヘッダー、フッター、図、figcaption、markです。
