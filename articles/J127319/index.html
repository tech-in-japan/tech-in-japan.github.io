Visual Studio 2010用のPythonツール1.0リリース

昨年12月、PTVSチームは、Visual Studioファミリに新しい言語を追加することを1年以内に低コストで約束しました。 9か月後、この約束は果たされました-Visual Studio RTM用のPython Tools 1.0を満たします。

現在、この素晴らしいIDEは美しいPython言語をサポートしています。 より正確には、その実装はCPython、IronPython、Pypy、およびJythonです。 これからは、Intellisence、クラスブラウザー、リンクの検索、リファクタリング、デバッガー、プロファイラーなどを使用できます。一般的に、これらはすべてVisual Studioのユーザーになじみのあるものです。

そして、すべて完全に無料（Visual Studio Shellにインストール）およびオープンソース（Apache 2.0）です。
