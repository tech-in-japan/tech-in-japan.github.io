お知らせ：6月8日は、マイクロソフトの仮想フォーラム「データ。 テクノロジー。 SQL Server 2016»

データの世界で重要なイベントであるマイクロソフトバーチャルフォーラム「データ。 テクノロジー。 SQL Server 2016」 。2016年6月8日に開催されます。 このフォーラムは、データベース管理システムのガートナークアドラント *のリーダーであるSQL Server 2016に 基づいたMicrosoft データ管理プラットフォームの新しいバージョンのリリースと同時に開催される予定です。









今日のデジタルビジネス変革の成功は、データを操作するための高度な技術とツールを使用する組織の能力に直接依存しています。 フォーラムの枠組みの中で、現代のデータプラットフォームに今日どのような要件が提示されているか、どの企業の兵器庫にどの主要技術が必要であるかを議論します。



フォーラムのメインプログラムは、3つの並列技術セッションで構成されています。

  • SQL Server 2016：トランザクションの世界における新しい標準。
  • ビジネスインテリジェンス：SQL、Power BI、R、モバイル。
  • Azure：新世代の分析ソリューション、ビッグデータとIoT。


詳細なプログラムは、イベントWebサイトで入手できます。



バーチャルフォーラムのスペースは、実際の会議の参加者のように感じることができるように編成されています。

  • 各参加者は、パートナーソリューションの展示会にアクセスできます。
  • MVP テクノロジーの専門家とチャットして質問してください。
  • スピーカーに興味のある質問をする 。
  • 必要な資料とプレゼンテーションをすべて入手してください。
  • さらに賞品を獲得する -最もアクティブな人は、SQL Server 2016試験に合格した30の証明書のうちの1つを受け取ります。


これだけではありません。トレーナーの指導の下で、オンラインの実験室作業を使用してプラットフォームのテストドライブに参加することで、日中に動作中のテクノロジーを見ることができます。



フォーラムへの参加は無料です。 事前登録が必要です。
