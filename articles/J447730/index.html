メールマーケティングの進化：QWERTYUIOPからGDPRへ





アメリカ人エンジニアのレイ・トムリンソンが世界初のメールを送ってから47年が経ちました。 アドレスで@ （ アットマーク ）記号を使用することを思いついたのは、彼が今日、ためらうことなく、テクノロジー全体、さらには電子メールの文化にさえ関わったからです。 それが何につながるのかを推測したことはまずなく、40年後に彼は時代の主要な革新者の1人としてインターネットの殿堂に入るとは思わなかった。



過去数十年にわたって、電子メールはコミュニケーション方法からダイレクトマーケティングの主要な形式の1つに進化してきました。 この方法は、消費者の行動を追跡して対応する絶好の機会を提供し、世界中の数十万人のマーケティング担当者に製品やサービスを宣伝および販売するのに役立ちます。



Mail.ru Sendboxサービスチームは、電子メールマーケティングがいかに冷酷なスパムから聴衆と対話するための微妙なツールに進化したかを研究しました。



1969年 。 米国国防総省は、インターネットの歴史を開始した最初のコンピューターネットワーク設計を開始しました 。 1969年、カリフォルニア大学の学生は、最初のコンピューターネットワークであるArpanetを介して2台のコンピューターを接続しました。 生徒は5メートルのケーブルを介して互いにコードワードを送信しました。 これは、コンピュータネットワークを介した情報の最初の送信であると考えられています。



1971年 。 マサチューセッツ工科大学（MIT）のレイモンドトムリンソンの卒業生が初めてのメールを送信しました 。 トムリンソン自身によると、彼はこの手紙の内容を覚えていません。







「 おそらく、最初のメッセージはQWERTYUIOPの精神に沿ったものでした 」と彼は後に述べました。 それ以来、10月2日、世界は毎年Email Dayを祝います。



1971年 。 最初のプログラムは、電子メールを送受信するために作成されました 。 Arpanetの従業員であるLarry Robertsによって書かれました。 1年後、おなじみの@記号がメールアドレスに表示されました。



1976年 。 イギリスのエリザベス女王2世は、他の国家元首に先駆けて最初のメールを送信します。







1978年 。 後に「スパムの父」と呼ばれたGary Twerkは、DECコンピューターを宣伝する400通のメールをArpanetユーザーに送信しました。 手紙で、ゲイリーは新しいデバイスについて話し、コンピューターのプレゼンテーションのためにロサンゼルスでの展示会に招待しました。 史上初のニュースレターは、著者に1300万ドルの利益をもたらしました。



1986年 。 初めて、「スパム」という言葉が使用されました 。彼らは、電子メールで送信された強迫的な広告メッセージを呼び始めました。 その名前は缶詰の肉に由来し、当時は可能なすべてのチャネルを通じて積極的に宣伝されていました。 SPAMを歌った強迫観念は笑の対象となり、別のスケッチ「Monty Python」は缶詰食品の遍在にささげられました。



1988年 。 Microsoft Mailの最初のバージョンが Mac OSオペレーティングシステムに登場しました。 ユーザーはAppleTalkネットワークを使用してメッセージを交換しました。 1991年に、DOSやWindowsを含む他のプラットフォーム用に2番目のバージョンが発売されました。 これは、その後のMicrosoftメールシステム（OutlookおよびExchange）の基盤として機能しました。



1991年 。 スペースから最初のメールを送信しました 。 1991年にSTS-43アトランティスの乗組員によって送信されました。 宇宙飛行士は、当時のMacintoshポータブルで最新のAppleLinkプログラムを使用して、次のメッセージを送信しました。 STS-43クルーへようこそ。 これは宇宙からの最初のAppleLinkです。 ここはクールで、周りにいないのが残念です...クライオとRCSをお届けします！ アスタラヴィスタ、赤ちゃん...私は戻ってきます。」



1992年 。 Microsoftは、Outlook電子メールクライアントの最初のバージョンをリリースしました。 MS-DOS用のOutlook、Windows 3.1x用のOutlook、Macintosh用のOutlookの3つのバージョンが同時にリリースされました。 Microsoftのアプリケーションの主な競合相手はThe Bat！です。 キシナウの会社であるRitlabs、SRLは 、5年後-1997年3月に登場しました。 その後、他の開発者もメールクライアントをリリースしました。 そこで、Mozilla Thunderbird、Opera Mailがありました 。 しかし、それらのどれもOutlookとThe Batの人気のレベルに達しませんでした！



1994年 。 IBMサイモンの発売- モバイルインターネット経由で電子メールにアクセスできる最初の電話 。 したがって、電子メールは、宛先に到達する追加の機会を受け取りました。







1994年 。 大統領令「不公平な広告からの消費者の保護について」が発行され 、これがわが国の広告規制の基礎を築きました。 特に、銀行、金融、保険、投資サービスの広告に制限が設けられましたが、インターネット上の広告の配信は規制されていませんでした。



1995年 。 連邦法が可決されましたが、これはすでにWebを介した広告の配信に一部影響を及ぼしています 。 たとえば、「広告」という用語の法的定義が与えられ、「何らかの形であらゆる手段で配布された」というフレーズが参照されました。 「通信チャネル」を提供または使用することを含め、広告を投稿および配信したすべての人は、広告代理店と見なされました。 したがって、インターネットと電子メールには広告の一般的な制限があります。信頼性と倫理、アルコール、タバコ、医薬品、金融サービスなどの広告の制限です。



1996年 。 最初の無料のhotmailサービスが登場しました 。 彼のおかげで、電子メールの所有者の数が大幅に増加しました。これは、広告メールの潜在的なオーディエンスが大幅に拡大したことを意味します。







1997年 。 Hotmailの開始から1年後、YahooはFour11からRocketMailサービスを購入し、それに基づいてYahoo Mailサービスを開始します。



1998年 。 RuNetでは、無料メールサービスMail.ruおよびsubscription.ruテーマ別メールサービスを獲得しています 。 両方のサービスは、それぞれの業界で標準を設定しています。 同時に、米国は、データ保護法の規定を更新し、すべてのマーケティング郵便物がそれらを受け取ることを拒否する機会を得た。



2000年 。 悪名高い「会話英語センター」が設立され 、2年後、いわゆる「スパミングビジネス」のおかげで全国に雷が鳴りました。 同社は毎日、何百万もの広告レターとそのセンターの電話番号を送りました。 その後、スパムの郵送が反トラスト政策省の注目の的となった。



2000年 。 AOLは一部の電子メールサービスにユーザーフィードバックをブロードキャストし始め、すぐにHotmailおよびYahooとともに、受信者からフィードバックを受信するためのスキームを導入しました。 そのため、 マーケティング担当者は、人々が自分の手紙についてどう考えているかを見つけ、スパムの苦情を測定基準として使用することができました 。 後に、インターネットサービスプロバイダーは、「不要な」電子メールから顧客を保護する方法を導入し、ほとんどのスパムは受信者に配信されなくなりました。



2003年 。 米国では、スパムメール法が登場し 、最初に商用メールを解決しました。 同じ年に、ヨーロッパは、マーケティング通信を許可するための規則を定義したプライバシーおよび電子通信の規制を採用しました。



2004年 。 Sender Policy Framework（SPF）が表示されました 。これにより、送信者の信頼性を確認し、送信者のIPアドレスを制御することでスパムを防ぐことができました。



2007年 IPhone-携帯電話からメールを表示できる最初のAppleスマートフォン 。 これまで、カスタムスマートフォンのメールサポートは一般的に制限されていました。 同時に、RIMのBlackBerryビジネススマートフォンの所有者は、2003年に電子メールで仕事をする機会を得ました。







2009年 。 Return Pathは、送信された商用メールの30％近くがメールボックスに届かないと報告しました。 同年、Merkle氏は、関連性の低さがユーザーがそのようなメールを読まずに削除する主な理由であると述べました。 この点で、無料メールサービスの提供者と作成者によってあらゆる可能な方法で支援されました。 2000年代後半、インターネットサービスプロバイダーは、「不要な」電子メールから顧客を保護する方法も導入しました。 これらのアクションの結果、 スパムメールは徐々に本格的なメールマーケティングに置き換えられました 。マーケティング担当者は、メールの評判を維持する必要があることに気づきました。 手紙はタイムリーになり、受信者に関連するようになりました。



2012年 。 現在、マーケティングメールの40％がモバイルデバイスで開かれています 。 これは、モバイルインターネットの開発によって促進されました。ネットワークへのアクセスは、いつでもどこでも取得できるようになりました。 その結果、マーケティング担当者は、文字が小さな画面にどのように表示されるかに特別な注意を払い始めました。



2012年 。 Michael Persod事件：2012年から2015年の容疑者は、カリフォルニアに拠点を置くImpact Media LLCに代わって複数のIPアドレスとドメインを使用して、Michael Prescott、Michael Pearson、Jeff Martinezとして自己紹介しました。 詐欺師はさまざまな商品やサービスを提供し、取引の成功に対して売り手から手数料を受け取りました 。 2016年に彼が自宅でFBIに逮捕されたとき、報復はこのカリフォルニアの居住者を追い越しました。 連邦裁判官は、Persodに490,000ドルの賠償金と54,000ドル以上の非金銭的損害賠償を支払うよう命じました。



2013年 。 DropboxはMailboxを買収しました。これは、元のインターフェイスと当時のスワイプによる管理のおかげで、 人々が電子メールを使用する方法を永久に変えると思われる有望な新興企業です。 2年後、プロジェクトは終了しました-その結果、ユーザーはスワイプに慣れておらず、通常のメールプログラムを変更する準備ができていませんでした。



2014年 。 別のスタートアップを閉じます。 今回、Outboxは開発者の期待に応えていませんでした。 月にたった5ドルで、特別な人がすべてのオフラインメールをデジタル化し、ネットワーク経由であなたに送信すると想定されていました 。 大衆はこの珍しい考えを非常に慎重に受け入れ、開発を受けなかった。



2018年 一般データ保護規則 （GDPR）がヨーロッパで施行されました 。 EU規制は、EU諸国の国民の個人データの保護を強化することを目的としています。 そのようなデータを扱う企業の責任は著しく増大し、規制違反は多額の罰金で脅かされています。 この措置は、電子メールマーケティングを合理化し、より文明化された基盤に置くのにも役立ちました。



おわりに



電子メールは、最も効果的で便利なコミュニケーション手段の1つです。 この技術は、そのかなりの年齢にもかかわらず、時代遅れではありませんが、ユーザーにとって有用で、時には不可欠なツールおよびマーケティングのための興味深いツールに進化しました。 インスタントメッセンジャーとソーシャルネットワークの両方は、マーケティング担当者が目標を達成するのに非常に役立ちますが、ブランド情報の消費と信頼できるコミュニケーションの面では、メールが引き続きリーダーです。 メールサービスも開発されているため、これが今後10年で変わることはまずありません。新しい技術が登場し、将来の電子メールが通信の競争手段であり続けるのに役立つ「チップ」がますます増えています。
