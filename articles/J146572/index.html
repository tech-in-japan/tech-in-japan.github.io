クラウドコンピューティングで最も影響力のある10社

クラウドコンピューティングにおける10の重要企業の翻訳。



この短い記事では、クラウドコンピューティングの分野で主要な役割を果たしている10社の評価に注目し、それらがトップ10に入る理由を説明したいと思います。



Habréでは、ほとんど毎日のようにクラウドのトピックに特化した投稿が作成されています。MicrosoftやAmazonなど、この分野の本物のサメに特別な注意が払われていますが、誰もが他の会社を知っているわけではありません。 たとえば、 Verizonは 、この通信大手がクラウドコンピューティングの分野を開発し始めたことを知っていますか？



「クラウド」の人気と関連性についてはすでに100以上の記事が執筆されているので、この部分を飛ばして本書に直接進みます。





10位-Verizon




2011年初頭、 Verizonはコロケーションおよびクラウドサービス企業であるTerremarkを 14億ドルで買収しました。 このトランザクションが、その価値を考えると、クラウドコンピューティングの歴史の中で最も画期的なものの1つとして認識されたことはまったく驚くことではありません。



この状況を巧みに利用して、VerizonはTerremarkの共同設立者でテクニカルディレクターであるJohn Considine（John Considine）を誘い込むことができました。これは、「クラウドスタートアップ」 Cloud Switchの創設者としても知られています。

悪くないですか？



「合併および買収」の評価では、おそらくこの会社が主要な地位を競います。



9位-VMware




このリストにVMwareが存在することは、あなたを驚かせるかもしれません。 しかし、彼らの将来のプロジェクトを見て、彼らがクラウドコンピューティングの分野にもたらすものに注意を払えば、あなたはさらに驚くでしょう。



これは、同社がクラウドサービスを提供していないという事実にもかかわらずです。 おそらく、あなたは疑問に思っているでしょう：「この会社はトップ10で一体何だ？」

実際、彼らのvCloudソフトウェアは、クラウドマーケティングビジネスで最高の企業として認められるための十分な議論です。 vCloud Director 1.5は、企業がプライベートまたはパブリッククラウドシステムにアクセスする際の仲介役として機能します。 VMwareは、vCloudを使用すると約100の「クラウド」を作成できると主張しています。

近い将来、この会社が何を待っているのだろうか？ それはさらに何かに成長しますか？



8位-リノード




Linuxユーザーにお気に入りのクラウドプロバイダーについて質問しても、同じ回答が得られても驚かないでください。 はい、 Linodeになります。これは、クラウド企業のランキングで8位に値するものです。



Linodeは、いわゆる仮想プライベートサーバーまたは単にVPSによって固定コストで「クラウド」を作成します。これは、従量課金モデルとは正反対です。

ユーザー自身が指摘しているように、Linodeはあなたが満たすことができる最速のVPSの 1つであり、ほぼ完璧な品質価格比です。



7シート-Salesforce.com




IT分野では、 この会社は10年以上にわたって知られています。

「クラウドコンピューティング」という用語の現代的な理解に近い、最初のテクノロジーの登場は、1999年に設立されたSalesforce.comに起因すると一般に受け入れられています。



現時点では、Salesforceは大きな一歩を踏み出すための良いスタートを切っています。 ほとんどの大企業と同様に、Salesforceも合併に頼りました。 たとえば、1年少し前に 、Ruby開発者向けのホスティングプラットフォームを購入しました 。Herokuは2億1,200万ドルで購入しました 。Herokuは、新しいサービスの導入が容易であることを強調し、顧客が独自のアプリケーションの開発とサポートに専念できるようにしました。

このトランザクションにより、SalesforceはHerokuテクノロジーとその知的資産だけでなく、Salesforceがクラウドコンピューティング分野で自信を持って位置付けられるようにする有名なRubyアプリケーションも取得しました。



6位-シトリックスシステム




Citrixは、クラウドサービスプロバイダーの中で主導的な地位の1つを自信を持って持っているもう1つの会社です。

さらに、Citrixはクラウドソフトウェアを開発しており、近年では、VMware、Microsoftなどのリーダーを含む仮想化ツールの開発者からより多くの注目を集めています。



Cloud.comを2億ドルで買収した後 、Citrixはオープンソースの世界への非常に必要なアクセスを可能にしたため、大きな飛躍を遂げたと言う価値があります。

その後、CitrixはCloud Stack （データセンターコンピューティングリソース管理コンソール）を導入し、オープンソースクラウド市場で最高の製品にしました。



5位-Red Hat




私たちのリストにRed Hatが存在することで、多くの人がこの会社を好まない、 軽んじる 、そしてそのメインプラットフォームであるOpenShiftが独自ではなくAmazonサーバーを使用するため、論争の的になるかもしれません。

しかし、Red Hatがランキングに載っていた理由は、 Linuxユーザーの間で非常に人気があるためです。



OpenShiftに関しては、ユーザーがアプリケーションをAmazonに最も簡単な方法でアップロードするのに役立ちます。

OpenShiftがLinuxユーザーの間でそれほど人気が​​あるのはなぜですか？

実際のところ、ソフトウェアは無料であるため、Red HatはVMwareやCitrixなどの企業と直接競合しています。



4位-Google




最近、 Googleは 、特にクラウド業界で大きな技術的進歩を遂げました。

少し前まで、彼らはGoogle App Engineを公開しました。ユーザーはJavaとPythonで書かれたアプリケーションを開発してダウンロードできます。

現在、 Google Cloud Storageは企業にとって最も推奨されるAmazon S3の代替品です。

Googleドライブ、Googleクラウドプリント、Googleクラウドコネクトなどのサービスは、同社が近い将来に開始する予定のクラウドサービスのほんの一部です。

すべてのユーザーがクラウドからアプリケーションを実行できるChromeOSは言うまでもありません。



3位-マイクロソフト




いくつかの後退にもかかわらず、 この会社はまだクラウドコンピューティングの分野で最高の1つです。

WebアプリケーションとWebサービスを作成するためのプラットフォーム-Microsoft Azureには世界中に何百万人ものファンがいます。



Azureの人気の背後にあると思いますか？

いいえ、これはある種の宇宙ソフトウェアではなく、その開発にダースベイダー自身が参加しました。

すべてが非常にシンプルです-特にAmazonなどの競合他社と比較して、比較的低価格です。

Microsoft Azureが提供するもう1つの興味深い機能であるストリーミングビデオブロードキャストは注目に値します。

AzureがLinuxプラットフォームに参入する準備をしていると聞いたことがありますか？ 実際のところ、これらはこれまでのうわさだけですが、Microsoftがクラウドベースのパイの一部を食い止めることを想像してください。

今日、Microsoftはユーザーに、Hotmail、Skydrive、あらゆる規模の企業向けのOffice 365クラウドアプリケーションなど、最高のクラウドサービスを提供しています。

UTD： Microsoft は 、LinuxをサポートするAzureの更新プログラムを既にリリースしています。



2位-Rackspace




Rackspaceは、クラウドを作成するためにオープンソースメソッドを使用することで知られています。

OpenStackの出現により、Rackspaceはオープンソース市場のリーダーになりました。

この会社はクラウドコンピューティングでどのような役割を果たしていますか？

おそらく、クラウド市場のRackspaceは、モバイルデバイスのOS市場のAndroidのようなものです。

Rackspaceの最も重要なイベント、 NASAとのコラボレーションについてお話ししましょう。これにより、これまでで最高のクラウドソフトウェアが作成されました。

Rackspaceは栄光にとどまるつもりはありません。 今日、この会社は新しいアイデアの検索に従事しており、世界中から最高のプログラマーを招待しています。 最終的に、これはOpenStackの改善につながるはずです。 Rackspaceに感謝します:)



1位-Amazon




本を販売するために設立されたこの会社は、突然何かに変わり、IT業界の世界を永遠に変えました。

今日、クラウドコンピューティングの革新的なアイデアと戦略で知られているAmazonが、この分野の絶対的なリーダーであり続けていると自信を持って言えます。



Amazon Web Services（AWS）は、クラウドサービスの最高のプロバイダーです。 過去6年間で、Amazonがサービスの価格を大幅に下げたことは驚くべきことです。 同社によれば、これは顧客を維持するのに役立ちます。

2011年末、 Amazonの純売上高は4808億ドルに達し、2010年には342億ドルでした。
