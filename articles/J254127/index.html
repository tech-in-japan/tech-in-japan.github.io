誘惑の伝説：Pythonを使用してiOSを自動化する方法





多くの場合、iPhoneで単調で退屈な操作を行う必要があります。これにより、セットアップ、スクリプト作成、および自動化の無限の可能性を備えたデスクトップをviousましく見ることができます。 デスクトップは何ですか-ユビキタスなTaskerを使用しているAndroidユーザーでも、スマートフォンを何でもプログラムできます。 iOSでは、このようなアプリケーションの存在は不可能ですが、小さな抜け穴があります。



この記事では、Pythonista（iOS用のPython（バージョン2.7.5）の開発環境）についてお話します。これにより、本格的なGUIアプリケーションを作成することもできます。 ただし、これをわずかに異なる目的に使用します-ルーチン操作を自動化する単純なユーティリティスクリプトを作成します。







Pythonistaには、iOS機能へのアクセスを支援するライブラリを含む、多くのプリインストールライブラリが含まれています。 例としては、クリップボード、アドレス帳を操作するための連絡先、キーチェーン、場所などを読み書きできるクリップボードがあります。



ビルトインに加えて、サードパーティのPythonモジュールも必要になります。 Pythonistaには、よく知られているpipに対応する2つのものがあります。 これらはpipista 2.0とPypiです。 最初のパッケージを使用してパッケージをインストールするには、スクリプトをルートディレクトリに保存し、次のコマンドを実行する必要があります。 



import pipista pipista.pypi_install('Name_of_library')


このライブラリには、関数pypi_download（）、pypi_search（）、およびpypi_versions（）も含まれているため、pipの完全な代替品と見なすことができます。 2番目のインストーラーには、より明確な要求が必要です。 たとえば、パッケージのバージョンを指定する必要があります。これは何らかの理由で最新バージョンを使用したくない場合に便利です。 



 from Pypi import Installer Installer('Name_of_library', 'Version').install()


このインストーラーには追加機能もあります。







メイン画面からスクリプトを実行する方法



これには2つの可能性があります：* Pythonistaショートカット*および* Launch Center Pro *。 最初のケースでは、すべてが簡単です。デバイスからサイトに移動し、スクリプト名と引数を入力し、ショートカットの作成ボタンをクリックして、標準のSafari機能を使用してこのページをデスクトップに保存します。



2番目のプログラムはさらに興味深いものです。 それからスクリプトを実行するには、イベントを作成し、* URL *フィールドに次の行を書き込む必要があります。 （ある場合）。 時間通りに、または一定の規則で実行する機能もあります。



スクリプトを完全なiOSアプリケーションにパックする機会もあります。 これを行うには、公式サイトからXcodeプロジェクトの[アーカイブ]（goo.gl/jsQK0b）をダウンロードし、標準スクリプトを独自のHello、worldに置き換えます。 その後、このプロジェクトをXcodeでアセンブルし、シミュレーターで実行できます。Apple開発者アカウントを持っている場合は、ガジェットで実行できます。







情報



Pythonistaにはすべての必要なドキュメントが含まれているため、学習するためにインターネットに接続する必要はありません。



組み込みのPythonistaエディターは非常に開発されており、構文の強調表示、自動補完、追加のキーボードキーがあります。 さらに、スクリプト化することもできます。




スクリプト



私たちと私たちの愛する人の生活を楽にするために、いくつかのスクリプトを書きましょう。 おそらくそれらの一部（またはすべて）は誰かには役に立たないように見えますが、Pythonistaを使用して何ができるかについてのアイデアを与え、実験の一種の基盤として機能します。 合計で9つのスクリプトがあり、それらは非常に多様です。







高速ツイート


ユビキタスなTwitterから始めましょう。 ツイートを送信するためだけにアプリケーションを開いたり、サイトにアクセスしたりするのはあまり便利ではありません。 したがって、クリップボードにあるものをツイートするスクリプトを作成します（tweepyライブラリをインストールすることを忘れないでください）。 



 import tweepy import clipboard #  Twitter- consumer_key = "----------" consumer_secret = "----------" access_key="----------" access_secret="----------" #   auth = tweepy.OAuthHandler(consumer_key, consumer_secret) auth.set_access_token(access_key, access_secret) api=tweepy.API(auth) #   text = clipboard.get() if len(text)<=140 and len(text)>0: api.update_status(text)


スクリプトは、使用可能なキーを使用してアカウントに接続します。 それらはTwitterの公式ウェブサイトで入手できます。 すべてのキーを取得するには、アプリケーションを作成し、[キーとアクセストークン]タブに移動して、[アクセストークンの作成]ボタンをクリックする必要があります。 このようにして、必要な4つのキーを取得します。 スクリプトでメッセージを投稿できるようにするには、[アクセス許可]タブでアプリケーションにそのような権限を付与する必要があります。



他のすべての点で、機能は非常に簡単です。 スクリプトはクリップボードから行を取得し、その行がツイート形式（140文字以内）に一致するかどうかを確認して、投稿します。



Instapaperでのクイック保存


同様に人気のあるInstapaperサービスについては、オフラインで読むためにページを保存できます。 次の3行のスクリプトは、クリップボードにあるページをデバイスにインストールされているサービスクライアントに直接追加します。 



 import webbrowser, clipboard addnew='x-callback-instapaper://x-callback-url/add?url='+clipboard.get() webbrowser.open(addnew)


このスクリプトは、いわゆるx-callback-urlを使用します。これは、組み込みブラウザーを介して呼び出すことができるミニAPIアプリケーションです。 この機能の公式Webサイトには、この機能をサポートするアプリケーションのリストがあります。 x-callback-urlリクエストの構造は次のとおりです。 



 x-callback-_://x-callback-url/?=


パスワードジェネレーター




はい、パスワードジェネレーターです。 同様の機能を備えたアプリケーションは数多くありますが、独自のアプリケーションを作成します。 私たちが望むからといって:)。 



 import random, string,clipboard pass = '' for x in range(random.randrange(8,12)): pass += random.choice(string.ascii_letters + string.digits) clipboard.set(pass)


このスクリプトは、選択に対して高い耐性を持つパスワードを作成するのに役立ちます（数字と文字を含む）。 このアルゴリズムの考え方は非常に単純です。ランダムな行（8〜11）から、上記のセットの文字数が空の行に追加されます。 次に、パスワードがクリップボードに配置されます。



現在地をメールで送信


あなたがどこにいるかを説明するよりも、ボタンを押してアドレスを相手に送信する方が簡単な場合があります。 



 import smtplib, location, time from email.mime.text import MIMEText # SMTP- server = "_" user_passwd = "" port = 22 user_name = "@" send_name='@' #     s = smtplib.SMTP(server, port) s.ehlo() s.starttls() s.ehlo() s.login(user_name, user_passwd) #   location.start_updates() time.sleep(10) location.stop_updates() loc = location.get_location() addr = location.reverse_geocode(loc)[0] #     Text = '   : ' + addr['Country'] + ',  ' + addr['City']+', ' + addr['Name'] letter = MIMEText(Text,'html','utf-8') letter['Subject']= ' ' letter['To']=send_name letter=letter.as_string() s.sendmail(user_name,send_name,letter) s.close


このスクリプトは2つの部分で構成されています。最初の部分はメールサーバーで動作し、2番目の部分は現在のアドレスを受信して​​メールを送信します。 2番目について詳しく見ていきましょう。 実際、ロケーションライブラリには「現在の位置を取得」機能はありませんが、頻繁にアクセスした場所のリストを取得できる2つの機能があります。 リストの作成は10秒しか続かないため（time.sleep（10））、現在のアドレスを持つオブジェクトは1つだけになります。 このオブジェクトは辞書です。 キーに必要な値を取得し、テキスト行に美しいフレーズを入力します。これは後で送信します。







FTP経由でサーバーに写真をアップロードする


既知のクリップボードライブラリにより、テキストだけでなく画像も操作できます。 これは私たちに新しい可能性を開きます。 クリップボードからFTPサーバーに写真を保存できるスクリプトはどうですか？ 



 import ftplib, clipboard #        a=clipboard.get_image() filename='out.jpg' a.save(filename) #       con=ftplib.FTP('Host','Login','Password') f=open(filename,'rb') send=con.storbinary(filename,f) con.close()


スクリプトは、最初の2行に書き込むことでわずかに変更できます。 



 import ftplib, photos a=photos.get_image()


その後、最後にキャプチャされた写真はサーバーに送信されず、コピーされます。 そして、最初の2行を次のように置き換えた場合： 



 import ftplib, photos a=photos.capture_image()


その後、iOSは写真の撮影を提案します。



SSH経由でリモートサーバーを操作する


私たちの多くはリモートサーバーを持っています。 ホームメディアプレーヤーまたはファイルワイプを持っている人もいれば、Amazon上のサーバーを操作する人もいます。 iPhoneまたはiPadからそれらを管理する方法は？ FTPクライアントをダウンロードできますが、バックアップなどの同じタスクを定期的に実行する必要がある場合、これはオプションではありません。 Stephen Millardは、リモートコマンドをSSH経由で実行できる[スクリプト]（goo.gl/Tt14cC）を作成しました。 簡略化された形式（入力されたデータとログ出力の正確性を確認せずに）では、次のようになります。 



 import paramiko import console # ,      strComputer = '' strUser = '' strPwd = '' strCommand = '_' #    client = paramiko.SSHClient() client.set_missing_host_key_policy(paramiko.AutoAddPolicy()) client.connect(hostname=strComputer, username=strUser, password=strPwd) #   stdin, stdout, stderr = client.exec_command(strCommand) print stdout.read() client.close()


一連のコマンドを実行するには、「stdin、stdout、stderr = client.exec_command（strCommand）」という行を異なるコマンドで数回コピーするだけです（または、セミコロンで区切られたすべてのコマンドをリストします。-Ed。）。







goo.glでリンクを短くする


多くの場合、リンクを短くするために頼らなければなりません。 しかし、あなたがサイトに行くたびに、コピーして貼り付けてください-それはちょっと退屈です。 幸いなことに、goo.gl APIが存在します。 



 import googl, clipboard client = googl.Googl("") result = client.shorten(clipboard.get()) clipboard.set(result['id'])


スクリプトは、クリップボードからURLを受け取り、変換してクリップボードに戻します。 スクリプトを実行する前に、アクセスキーAPIを取得する必要があります。 これを行うには、[ API ] ページに移動し、[プロジェクトを追加]をクリックして、利用可能なサービスのリストにURL Shortener APIを含めます。 次に、左側のメニューで[APIと認証]タブ、[資格情報]サブメニューを選択し、[新しいキーの作成]ボタンをクリックしてから、ブラウザーキーをクリックします。 キーを受け取ったら、それをスクリプトに挿入します。



スクリプトの実行中、結果変数には{'kind'： 'urlshortener＃url'、 'id'：ShortLink '、u'longUrl'： 'LongLink'}という形式の辞書が割り当てられることに注意してください。 短いリンクのみが必要なので、キー「id」の値がクリップボードに入力されます。

検索エンジンを歩いていくと、ノートブックをクリアする2つの方法のみが見つかります。1つの連絡先を削除するか、コンピューター上の空の連絡先リストと同期します。 いいえ、それは面白くないです。 



 import contacts a = contacts.get_all_people() for i in a: contacts.remove_person(i) contacts.save()

VKからノートブックに友人をインポートする


最後に、最も複雑で長いスクリプトは、VKontakteからノートブックに電話番号をインポートすることです。 他の皆と同じように、VKontakteにはAPIがあります。 pythonには、それを操作するためのライブラリがいくつかあります。 最も有名で最も使いやすいのは、控えめな名前がvkのライブラリです。 



 import vk, datetime, contacts #          iOS def convertdate(date): date = date.split('.') if len(date) == 2: return datetime.datetime.combine(datetime.date(1604,int(date[1]),int(date[0])),datetime.time(0, 0)) else: return datetime.datetime.combine(datetime.date(int(date[2]),int(date[1]),int(date[0])),datetime.time(0, 0)) #        vkapi = vk.API('ID-', '', '') a = vkapi.friends.get(fields='contacts,bdate') a = a['items'] #           for i in a: Temp = contacts.Person() Temp.last_name= i['last_name'] Temp.first_name = i['first_name'] if 'mobile_phone' in i.keys(): try: Temp.phone=[('mobile',i['mobile_phone'])] except: pass if 'home_phone' in i.keys(): try: Temp.phone.append(('home',i['home_phone'])) except: pass Temp.url = [('vk','http://vk.com/id'+str(i['id']))] if 'bdate' in i.keys(): Temp.birthday = convertdate(i['bdate']) contacts.add_person(Temp) #   contacts.save()


Twitterの場合のように、スクリプトの場合、「VKontakte」内に「アプリケーション」を作成する必要があります。 これを行うには、VK Webサイトの[アプリケーション]タブに移動し、[管理]タブに移動して[アプリケーションの作成]ボタンをクリックします。 アプリケーションページで[設定]タブに移動し、[アプリケーションID]をコピーします。 「アプリケーションID」、「ログイン」、および「パスワード」をスクリプトに挿入します。



このスクリプトの仕組みを見てみましょう。 まず、友達のリストを取得します。 デフォルトでは、friends.get（）関数は、countとitemsの2つのフィールドで構成される辞書を返します。 間違いなく2番目の名前に興味がありますが、名と姓だけでなく取得したいので、フィールドパラメータを関数に渡し、知りたいことを示します。 次に、各辞書がユーザーである辞書のリストを確認します。 各反復で、Person型のTemp変数を作成し、次にフィールドを追加します。



連絡先を確認するプロセスで、スクリプトはいくつかの問題を解決します。 電話番号をエクスポートするときに最初の問題が発生します。これは、VK番号で「誰が必要か、知っている」、「秘密」などのように非常に頻繁に見つけるからです。 スクリプトがクラッシュせずにそのようなレコードを処理できるようにするには、tryステートメントが使用されます。 2番目の問題は、生年月日の形式の不一致で発生しました。 VKから受信した辞書には、DD.MM.YYYY形式の文字列として書き込まれ、誕生日フィールドには、datetime.datetime形式のデータを入力する必要があります。 そのためには、スクリプトの最初にconvertdate関数が必要です。 また、生年月日がまったく表示されない場合があります。
