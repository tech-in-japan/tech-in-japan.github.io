Web 2.0を使用するための9つのヒント

Web 2.0とその将来に関する非常に興味深い記事を読むことをお勧めします。



今すぐ始めたらもう遅れてる

聞いたことがある成功したWeb 2.0スタートアップはすべて、長年の仕事の結果です。 今日の「勝者」はすべて先駆者でした。 したがって、今から始めると、あなたは彼らと比べて大きな不利益をこうむります。



一方、成功は不可能だと考えるべきではなく、空想と現実を調整するだけです。 数十億ドルの取引が「あなたを脅かすことはない」ことを自分で理解してください。



コンテンツの保護を忘れる

あなたの計画がコンテンツを提供することであるなら、あなたが最初にしたいことはそれを保護することです。 全体の問題は、Web 2.0では著作権を完全に無視するだけでなく、あらゆる種類のビジネスロジックに違反する準備ができている必要があるということです。 風車と戦う代わりに、 Creative Commonsライセンスについて読んで、あなたに最も合ったものを選んでください。



論争に備える

ビジネススクールでの勉強は、ソーシャルネットワーク（ソーシャルネットワーキング）に関係する人と比べて、あなたを不平等な立場に置きます。 ルールはあなたの利益を守ることを教えますが、ソーシャルネットワークは主に情報の配信（共有）です。 あなたはどういうわけかあなたが形成するコミュニティで勝つ期待される利点と価値の間の妥協を見つけなければなりません。 これは簡単なことではなく、潜在的な購入者の反応は、皮肉の兆候をすぐに追跡します。



あなたのポジションは最初は不利です

従来、顧客に提供される情報は、企業自体によって明確に管理されていました。 ソーシャルネットワークでは、この制御を失うだけです。 顧客は、適切と判断した場合、あなたを批判します。 批判を制御または排除しようとすると、あなたの努力は苦情や不満の単なる別の原因になります。



インターネットでは、ユーザーの対話性は会話とディスカッションの形で提供されるため、予期しない事態に備える必要があります。 丁寧かつ丁寧に。 介入する必要があるときに感覚を養います。 通常、コミュニティのメンバーが同じことについて話すか、影響力のあるキャラクターが何かについて話し合う場合、行動する時です。 コミュニティまたはレミングスのごく一部のみがあなたを批判する場合にのみ、そのような発生は無視できます。



じっとしないで

正直な行動はどのコミュニティでも高く評価されており、Web 2.0も例外ではありません。 可能な限り、あなたはインターネットでの行動だけでなく、あなたのその他の事柄においても、あなたが誠実さを大切にし尊重していることを理解しましょう。 あまりにも物議をかもし、変更可能である-とコミュニティは偽善者を呼び出します。



コミュニティの構築には多くの時間と労力がかかります

既に顧客がいる場合でも、サイトを運営するだけでは十分ではありません。 最も成功したWeb 2.0サイトは、優れた広告、ブログ、郵便フォーラムを通じて宣伝されています。 それは陳腐に聞こえるかもしれませんが、あなたのコミュニティを養うべきです。 そうしないと、誰も使用しない空のサイトになります。



コミュニティとの連絡を失わないでください

コミュニティとのコミュニケーション方法を学ぶ時間が必要な場合があります。また、あなたとの間に良好な関係が構築されたら、それらを維持しようとします。 コミュニティとのコミュニケーションが失われたために、ユーザーが競合他社に移動した例があります。



常に新しいことを考えてください。

90年代の共有ホスティングと同様に、Web 2.0はビジネス参入の低水準を提供します。 言い換えれば、プレーヤーになるために、特にさらなる開発にFOSSを選択した場合、専門知識と大規模な開発は必要ありません。 Web 2.0は数年前から存在しており、経験上、コミュニティは非常に一貫性がなく、ユーザーは問題なくよりおしゃれなサイトにアクセスしています。 したがって、人々を維持するために、開発をやめないでください。 より多くの新しいユーザーを引き付け、既存のユーザーを維持する新しいものを考えてください。



1つの問題があります。サイトのコンセプトが単純であるほど、イノベーションを実装することが難しくなります。 del.isio.usのように、サイト自体がアイデアではない場合、ビジネスを維持するために開発の方向を根本的に変える必要があるポイントに到達できます。



利益の源を知る

あなたが行うことのほとんどは、コミュニティの一部によって拒否されます。 広告を掲載すると、ユーザーから苦情が寄せられます。 Classmates.comのような月額料金を導入すると、利益を上げる可能性のあるコミュニティが失われます。



真実は、お金はWeb 2.0の一部であり、誰も実際には説明できないということです。 4年以内にビジネス指向のソーシャルネットワークに約100万ドルを投資した企業があり、現在eBayオークションでの初期価格は60,000ドルです。



もちろん、最も確実な方法は、プロジェクトがバイヤーを見つけるのに十分な大きさのコミュニティの成長です。 そして、この場合でも、投資された資金を返す保証はありません。



おわりに

これらのヒントが悲観的であると思われる場合、あなたは私が何を得ようとしているのかを正しく理解しました。 プロジェクトを開発するのにまだ1〜2年かかります。 ビジネス現象としてのWeb 2.0は、古典的な方法では制御できません。 それでもWeb 2.0で収益を上げたい場合は、超高速で成長し、コミュニティをさらに速く成長させる必要があります。 同時に来年、Web 2.0は歴史になります。
