Logitech has developed a virtual reality keyboard





VR is becoming an increasingly common technology, which is already used not only for games. Accordingly, the ecosystem of gadgets compatible with virtual reality is developing. Recently, Logitech has revealed a new keyboard, the Logitech Bridge , which can be used with the HTC Vive VR VR helmet.



The keyboard is also suitable for working in normal conditions. If, however, video wear is done while using the Vive VR, then when you hover the system tracker on the keyboard in virtual reality, an exact copy of Logitech Bridge appears. The virtual keyboard is located exactly in the same place as the real gadget. By clicking on the buttons in virtual reality, the user gets an accurate tactile response, so you don’t have to get used to this mode of operation.



VR is usually associated with motion controllers, but many applications in a virtual environment allow you to work with the keyboard. In the event that a user decides to use the keyboard, it can be difficult to get used to working with it in the helmet. Even if a person has a good “blind typing” skill, the difference between sensations in a virtual and real environment does not allow concentrating on the keyboard, you have to get used to it.





Now there is a VR-keyboard, which, according to the developer, works without problems. You can use it in games and other applications. Over time, more and more business applications will appear for VR, so you can’t do without a keyboard. And they, as we see, have already begun to enter the market. True, the price of Logitech Bridge can not be called low - its cost is $ 150. If you remember that the virtual reality helmet itself costs about $ 599, you realize that VR is still expensive. But it is now, over time, the situation will be normalized in favor of ordinary users.



At the moment, the keyboard is available only for developers, and then on request. The deadline for applications is November 16.
