CMSを浸す

比較的新しいCMS-Immerseを検索する記事は見つかりませんでした。 したがって、私は自分でそれを書くことにしました。



公式ウェブサイトからの翻訳：

Immerse CMSは、PHPで書かれた無料のコンテンツ管理システムです。 Media Manager、プラグインアーキテクチャ、AJAX、CSSおよびJavaScriptファイルのインライン編集などの柔軟な利点の多くは、サイトの時間を節約します。



システムの外観と機能は非常に魅力的です。 最も興味深いのは、プラグインとコンポーネントのかなり独創的なアーキテクチャです。 組み込みの管理ツールを使用して、既存のアドオンのコードを手動で変更したり、新しいファイルを書き込んだり（ファイルからロードしたり）することができます。 アドオンとコンポーネントの使用は、タグとして実装されます。 例：

[プラグインアイテム= "vanilla"キャッシュ= "false"メソッド= "welcome_message"]

キャッシュを無効にして、バニラプラグインが呼び出されます。 メソッド文字列は、呼び出しパラメーターとして使用されます（この場合、Vanillaフォーラムの挨拶メッセージ）。

この種のタグは、目的のページに挿入する必要があるphpコードを置き換えます。 言い換えれば、あなたが好きなように、スニペット。

システムはまだ2番目のベータバージョンに存在しますが、それにはかなり因果的な機能があります。



多言語ウェブサイト

訪問者の言語設定に応じて、ページの言語を動的に変更するための「辞書」の機能を使用します。



高度なユーザー管理と権限

特定のユーザー/グループに対してコンテンツを表示/編集する権利を区別する機能。 また、多くの人気のあるブログやフォーラムへの架け橋の開発を発表しました。これにより、サイト全体に単一の登録が作成されます。



管理可能な構造の無制限のコンテンツ

無制限のコンテンツネスト、無制限のサイトサイズ。

（ただし、ほぼどこにでもあります）



検索エンジンに優しいURL

このCMSのクリーンリンクシステムは、Drupalを強く思い出させましたが、そこではすべてがはるかに単純です。 ここで、ページ拡張子を選択できます（またはまったく選択しません）。



高度なサイトユーザー統計

サイト統計収集システム。 ただし、著者は次のように記述していますが、かなり標準的です。

「統計を収集するために多くのウェブサイト所有者が支払います。CMSに組み込まれた強力で無料の収集システムを提供します。」



テンプレートベースのページ

サイトの設計を完全に制御します。 すべてのサービス要素は、特別なタグで記述されます（たとえば、メニューの[component item = "MenuGenerator" current = "[page item = 'id']"]）。 Joomla！を非常に連想させます。



組み込みのファイル管理

ビルトインファイルマネージャー。



WYSIWYG Editorのサポート

人気のあるWYSIWYGエディターのサポート。ただし、これらは個別に配置されます。



インテリジェントキャッシング

キャッシュページを浸漬して、チャネルをすばやく開いて保存します。 特定の要素では、キャッシュを無効にすることができます（また、そうする必要があります）（コメントコンポーネントなど）。



一般的に、印象は最もポジティブです。 システムはJumlaやDrupalよりもはるかにシンプルですが、機能的には後者よりも劣っています。 それに興味があれば、ドキュメントの翻訳に従事します。 :)
