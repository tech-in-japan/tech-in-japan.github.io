ベータ版Firefox 13利用可能



1週間も経たないうちに、 Firefox 12がリリースされ、ベータバージョン13が利用可能になりました。

新しいアイテム



新しいタブで開くページは空ではなくなりました-ユーザーが最もアクセスしたページを示すプレビューを表示できるようになりましたが、このページを自分用にカスタマイズすることもできます。 こんな感じ

デフォルトのホームページから、ブックマーク、履歴、設定などにすばやくアクセスできます。







  • 現在、スムーズスクロールはデフォルトで有効になっています。
  • ECMAScript 6のMapおよびSetオブジェクトの実験的サポートを実装しました
  • FireFoxをユーザーアカウント制御で更新できるようになりました
  • SPDYがデフォルトで有効になりました。これにより、ブラウザーの速度も向上します。
  • CSSターンユニットのサポートを追加

    • CSS column-fillプロパティのサポートが追加されました
    • 開発者ツールの多くの改善が行われました：ページインスペクター、HTMLパネル、スタイルインスペクター、スクラッチパッド、スタイルエディター
    • CSS3拡張構文background-positionのサポートを実装しました
    • ：invalid疑似クラスは要素に適用できます。




    フルバージョンのリリースは6月5日に予定されています。



    ベータ版は、Windows、Mac OS X、およびLinuxで利用可能です。
