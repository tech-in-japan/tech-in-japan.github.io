Teeworlds Modsの概要

素晴らしいタイムキラーがあります-Teeworlds。 少なくとも一度それをプレイしたすべての人は、確かに、標準のゲームモードのMODに出くわしました。



それで、私はあなたにゲームTeeworldsの人気のある改造の私のレビューを与えたいです。



Instagibから始めましょう。











InstagibはTeeworldsのmodであり、その特徴は、レーザーキャノンを使用すると、1回の攻撃で敵を殺せることです。 ブラスターの光線が壁に反射する可能性があります。 新しいバージョンでは、レーザージャンプ（レーザージャンプ）を実行できます。これは、プレーヤーに顕著な加速または単に高いジャンプを与えます。



これはゲームの非公式のmodです（他のすべての人と同様）が、多くの人がそれは常にそうだと考えています:)。 ほとんどの場合、DMおよびCTFモードで再生されます。 これは、人々が別々のクランで団結し、古典的なゲームプレイを完全に忘れる数少ないMODの1つです。 彼はそれを上手に演奏することを学ぶことはそれほど難しくないという事実のために批判されており、初心者はすぐにたむろし始めます。 ;）このような理由から、一部のプロ選手はインスタギブについて否定的な意見を持っています。 批判にもかかわらず、instagibはTeeworldsで最も人気のあるMODであり、チャンピオンシップやその他の競技会を開催します。



次に、 Flagballを検討します 。











フラッグボール-eeworldsmod。主な目的は、 他のプレイヤーのゲートにフラッグを獲得することです（はい、はい、正確に獲得します）。



このmodでは、特別な対称カードが描かれ、各チームには独自のゲートがあり、他のチームのプレーヤーがボールを獲得しようとします（フラグです）。 ボールを獲得する方法は2つあります。 1つの方法は、ボール（旗）を上げて、マウスの左ボタンで打つことです。 別の方法は、旗で敵の門に飛び込み、チームのために自分を犠牲にすることです。 マウスの右ボタンを使用して、フラグを別のプレーヤーに転送することもできます。 楽しみのために、このmodはinstagibと組み合わされています。



Run modも非常に人気があります。











Runは、Teeworlds modであり、その主な目標は、特定のルートをできるだけ早く渡すことです。 ゲーム中、ほとんどの場合、プレイヤーが使用できる武器は1種類だけです-パンくずは、ロケットジャンプ（手rena弾でジャンプ）を実行します。 プレイヤーはお互いを殺したり、干渉したりすることはできません。 このモードには多くのカードがあります。 最初のカードはかなり軽量でしたが、今ではペア通過用のカードさえも思い付きます。 このmodのマップを作成する際、作成者は、プレイヤーがマップをすばやく通過できないように、デスタイルとテレポートタイルを使用することがよくあります。 最高の結果がサーバーに保存され、競合が生まれます:)



そして、私は本当に好きなSurvival modについて言及したいと思います。 :)



サバイバルはTDMベースのMODですが、1ラウンドではプレイヤーは一度しか死ぬことができず、ラウンド数が限られています。











殺されたプレイヤーは自動的に観客のチームに投げ込まれます。 生き残ったプレイヤーが残っているチームがゲームに勝ちます。 限られた数の弾薬がプレイヤーの真のスキルを示しているため、このMODはプレイするのが非常に興味深いです。 このmodをプレイすると、人々はマップコントロールよりもスキルと武器に依存することに慣れています。



最近では、ゲームのプレイがゲームLeft4Deadのゲームプレイに似ているゾンビ modが登場しました。 プレイヤーは、Tシャツゾンビボット攻撃を撃退する必要があります。 攻撃の各波で、彼らはますます強くなります。 ゾンビにはいくつかのタイプがあります：ハンマーを備えたボット、ハンマーを備えたフック、銃を備えたゾンビなどです。



新しい興味深いMODが頻繁に登場しますが、これはプレイヤーを喜ばせます。 可能であれば、Tolsi =）を紹介します。
