IntelliJ IDEAが待ち望んでいるものは何ですか？

まもなく、当社の主力製品であるIntelliJ IDEA開発環境は14歳になります。 私たちのツールを使用してすでに多くの製品が作成されており、あらゆる種類の機能がIDEに追加されているように見えますが、机上の製品が常に新鮮になるように、新しいインスピレーションの源を探し続けています。 次の記念日に、私たちの意見を共有したいと思います。製品開発の開発を決定するのは、今後5〜10年で正確にどうなるでしょうか。



急速に成長している企業として成功を収めるために、独立した専門コンサルタントに頼ることにしました。 選択は、多くの有名なIT企業の協力と独自のプロジェクトで豊富な経験を持つ非常に権威のある国際的なオフィスで行われました。 さまざまな文化的および職業的関心を持つさまざまな年齢、性別の多くの開発者がインタビューされました。 主な結果を武器に、開発者は新しいアイデアを熱心に紹介し始めました。





私たちは「楽しみながらプログラムを作り続けます！」



私たちの絶え間ないスローガン「喜びのあるプログラム」（例えば「喜びのある開発！」）に対する曖昧な意見は、その意味に密接な意味合いを暗示することもありますが、断固として拒否され、完全に拒否されました。 調査結果が示したように、ほとんどのユーザーは喜んでプログラムを作成することを好むため、スローガンには触れないことにしました。





アプリ内購入付きの新しいライトバージョン



ユーザーがHabréで非常に多く書いた過去に私たちが直面していたことは、最終的に反省を経て、次の概念をもたらしました：



IntelliJ IDEAには新しいバージョンが必要です。 はい、そのような製品の2つのバージョンは不当に小さいです。 コイントスによって選択が決定された人にとっては、すぐに困難な時代が来るでしょう。 次のバージョンから、IntelliJ IDEAの新しいエディション（およびIntelliJファミリーの他の製品） -IntelliJ IDEA Liteを紹介します。







このバージョンの重要な機能の1つは、アプリ内購入です。 独創的なものはすべてシンプルで、あなたは間違いなくこの革新を好きになるでしょう。 アプリ内購入は、私たち全員、さらには子供たちも日常生活で使用するものであるため、この機会を製品に取り入れることは自然なプロセスになっています。



機能クレジット


すべてはとても簡単です、友達。 IDEまたはJetBrainsの他のツールを使用すると、特別な仮想通貨であるJetBrains Feature Credits（JFC）を獲得できます。 IDEをより効率的に使用し、知識を深めるほど、より多くのJFCを獲得できます。 現在、コードインデックスの高速化、高度なコード分析、完全自動化されたリファクタリング、新しい配色などのプレミアム機能にアクセスするには、ビルドサーバーでビルドを（順不同で）促進することも必要です。 IDEからJFCに直接リアルマネーを交換することもできます。



より多くのJFCをすぐに受け取るために、IntelliJ IDEAの新しいバージョンの準備を希望する場合、いくつかの推奨事項を作成しました。



  1. 生産性を追跡し、不要なマウスの動きやバックスペースでの不当なクリックを避けます 。
  2. IDE生産性ガイドをご覧ください。 [ヘルプ]メニューを使用して開き、統計情報と他の効果的な開発スキルを追跡します。
  3. IntelliJ IDEA Code Golfプラグインをダウンロードして 、IDEの知識をテストし、新しい知識を学びます。
  4. IDEキーマップ 。 [ヘルプ]タブを開き、キーマップを確認して印刷し、就寝前にキーボードショートカットを繰り返し、学習し、キーボードにヒントを適用します。




もう1つの重要なニュース：アプリ内購入用のサードパーティプラグインもサポートする予定です。





ゲームAPI


開発以外に本当の喜びをもたらすものは何ですか？ もちろん、コンピューターゲーム。 IDEスキームの年齢、性別、または色に関係なく、全員がコンピューターゲームをプレイします。 ゲームは時間を過ごすだけでなく、心を発達させ、より複雑な問題を解決するのにも役立ちます。



現在、IntelliJプラットフォームに基づいたIDEでのゲームの開発と実装を簡素化する新しいIntelliJ Game SDKを開発しています。



新しいIntelliJ IDEA EAPビルドは、2つのゲームを既に取得しており、それらを実行して休憩したり、仕事から気をそらしたりできます。 そして、新鮮なアイデアで戻ってきます。



  1. すべての時間ディガーの時代を超越したゲーム。 新しいファイルを作成するためのメニューを開いて、古き良き掘り手（Ultimateのみ）の再生を開始します。







  2. 2048 19歳のイタリアのWeb開発者Gabrielle Cirulliが私たちに提示したゲームで、インターネットにアクセスできる多くの人々のパフォーマンスをすでに損ない、この記事のリリースを数時間遅らせました。











パーソナライズされたIDE


すべての開発者は同じではありません。 お気に入りの配色、キーボードレイアウト、ウィンドウレイアウトがあります。 自宅や職場、同僚とのどのコンピューターでも、通常の設定でIDEを実行できますか？ 私たちもそうです。 そのため、すべての重要な情報を保存し、いつでもIDEをパーソナライズできるオンラインプロファイルをユーザー用に作成することにしました。 IDEの習熟度やその他の成果を他のユーザーと比較することもできます。
