PHP + Ajax = Webエクスプローラー！

今日、私は非常に人気のある1つのオープンソースパズル手法を克服することを提案します。 このタスクは簡単で、長い間多くの解決策がありました。 しかし、個人的には、質問は未解決のままです。 すなわち：ウェブベースのファイルマネージャ...







eXtplorer -Ajax（ExtJS、 cuckoo ！）とPHPの助けを借りて、私たちの主題は素晴らしい仕事をしています...



これは、 Source Forgeページから翻訳された「O」スライスです



eXtplorerは、Webファイルマネージャーです。 次の用途に使用できます。



*サーバー上のディレクトリとファイルを表示する

*ファイルの編集、コピー、移動、削除、

*ファイルの検索、アップロード、ダウンロード、

*アーカイブの作成と抽出、

*新しいファイルとディレクトリを作成し、

*その他のファイルの変更...



eXtplorerを使用してFTPサーバー（net2ftpなど）にログインし、FTPクライアントを使用する場合と同じように作業することもできます。


他のソリューションよりも優れている点は何ですか？



フォルダのエクスプローラツリービューでよく知られている通常のドラッグアンドドロップ、アーカイブの操作、コードの強調表示によるファイルの編集！



彼に出会ったとき、私はとても幸せでした。 オープンソース、使いやすく、最も重要な機能。 一般的に、この問題に関するリーダーシップの候補の1つです！ ユサ！



Joomla！としても利用可能です。 およびMamboコンポーネント。
