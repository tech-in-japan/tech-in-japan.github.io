Windows Home Serverのスクリーンショット

確かにあなたの何人かは部屋の最も暗い隅（どこか地下室）にファイルサーバーを持っています。 しかし、Microsoftは、新しいWindows Home Server OSのリリースですべてを変更したいと考えています 。 Windows Home Serverは、中央のファイルストレージ、メディアファイルの共有、ホームネットワーク上のすべてのコンピューターのシステム復元ポイントの自動バックアップおよび作成機能を提供します。 少し前に、Windows Home Server RC1がリリースされました。ベータテスターにダウンロードできます（参加することもできます）。 実際、推測するのは難しくありません。彼について話しましょう...

Windows Home Serverの回復



通常のWindowsを実行するサーバーをゼロから作成することは、Windows Home Serverよりもはるかに複雑であることに注意してください。 しかし、この蜂蜜の樽では、もちろん、軟膏にハエがあります：Windows Home Serverは、すべてのホームPCがWindowsで動作する場合にのみ適しています。 つまり、Boot CampにWindowsがインストールされている場合にのみ、MacからHome Serverに接続できます。
