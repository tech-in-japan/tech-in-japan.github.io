SuperDuino：カラーディスプレイと加速度計を備えたArduino互換ミニチュアボード





さまざまな種類の電子機器の開発者は、できるだけ早くこの世界を知りたいと思うエレクトロニクスの専門家と初心者の両方が使用できる新しいモデルのボードをリリースし続けています。



最近の最も興味深いプロジェクトの1つはSuperDuinoです。 これは、カラーディスプレイと加速度計が付属するミニチュアボード（真にミニチュア）です。 開発者は最近、SuperDuinoの大規模なリリースのための資金を調達するためにKickstarterキャンペーンを開始し、キャンペーンが終了するまでに、計画の12倍の資金を調達していました。







SuperDuinoの機能





このボードは、独自のスマートウォッチ、ミニチュアオシロスコープ、電子カレンダーなどの作成など、多数のプロジェクトに使用できます。







ボードのピン配列





開発者はすでにSuperDuinoに基づいてさまざまなデバイスのいくつかのバリアントを作成しており、さらに多くのことができると約束しています。



1.温湿度コントローラー。

2.電圧計と電流計。

3.頻度決定要因;

4.オシロスコープ;

5.デジタルロジックアナライザー。

6.歩数計。

7.スマートウォッチ;

8.通話時間。

9.ポータブルゲームコンソール（Flappybirdの起動を含む）;

10.あらゆる種類のデバイスをスマートフォンに接続するためのスマートハブ。







C SuperDuinoには、PCを介したデバイスの制御、ミニLEDのセットアップ、簡単な信号機の作成など、さまざまな問題をすばやく把握するのに役立ついくつかのマニュアルが付属しています。



製品の特徴：





  • USBプログラマブル（マイクロUSBコネクタ）
  • 16Mhz / 32Khzシステムクロック
  • 個別のプリスケーラと比較モードを備えた2つの8ビットタイマー/カウンター
  • 個別のプリスケーラ、比較モード、およびキャプチャモードを備えた1つの16ビットタイマー/カウンター
  • 独立した発振器を備えたリアルタイムカウンタ
  • 6つのPWMチャネル8チャネル10ビットADC USARTMaster /スレーブSPI
  • バイト指向2線式シリアルインターフェイス（I2C）オンチップアナログコンパレータ
  • Pin Change23 ​​Programmable I / O Linesでの割り込みとウェイクアップ
  • 1.7 "128X160 16ビットカラーTFT LCD
  • USB / CR2032コイン型電池（充電式電池を充電可能）/ 3-12V電源供給
  • 3軸加速度計（シングルクリック、ダブルクリック、上、下、左、右の傾きを検出できます）
  • 2.54mm I / Oピンヘッダー
  • スピーカーオプション
  • Bluetoothアドオン
  • WIFIアドオン
  • MicroSDアドオン
