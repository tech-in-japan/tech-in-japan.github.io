NetBeans IDE 8.0 Betaリリース

数か月前と同じく、素晴らしいNetBeans IDEのベータ版が静かに、大騒ぎせずにリリースされました。



NetBeansは、PHP、C / C ++、Python、JavaScript、さらにはAdaなどの言語でアプリケーションを開発できる高度な開発環境であることを思い出させてください。



NetBeans IDEは無料の開発環境であり、プロジェクトは熱心な開発者のコ​​ミュニティによって開発されていますが、その品質と機能は商用IDEに劣りません。



すぐに使えるサポート：


  • リファクタリングとプロファイリング
  • カラーシンタックスハイライト
  • その場で自動補完
  • すべての一般的なバージョン管理システムの管理




NetBeans 8.0ベータ版の新機能



  • Java SE 8のサポート
  • Java ME 8のサポート
  • HTML5サポートの強化
  • Tomcat 8.0のサポート
  • PHP 5.5のサポート
  • IDE全体の多くの改善






エディター



  • インデント線

    最後に、破線が表示され、タブで区切られたコードのブロックを区別しやすくなりました。 デフォルトで表示されます。 IDEの設定で色を変更できます。







  • 新しいコードヒント
  • 外部ドキュメントへのリンク、ドキュメント表示ウィンドウでアクティブになりました




Java



  • JDK8サポート
  • Java SE EmbeddedおよびJava ME Embedded 8のサポート
  • Javaエディターの機能強化：多くの新しいコードヒント、ツールチップとしてのJavadoc
  • JavaFX Scene Builderとの統合の改善




Java EE



  • 新しいPrimeFacesコードジェネレーター
  • Tomcat 8.0との統合
  • TomEEとの統合
  • さまざまな改善




Php



  • PHP 5.5のサポート





















  • PSR-0およびPSR-1標準のサポート
  • PHP CS Fixerのサポート
  • Twig、Latte、Neonの改善
  • プロジェクト作成ウィザードでのComposerサポート




C / C ++



  • パン粉を追加





  • GDBデバッガーコンソールをコマンドラインモードで使用できるようになりました。 IDEはコマンドを処理し、デバッガーの変更を反映します




プラットフォーム



  • Windowsシステム用のエディタータブをカスタマイズするための新しいエディター

  • 新しい設計チーム管理ユーザーインターフェイス
  • フォルダ内のファイルを拡張子でソートできるようになりました









雑多



  • プロファイラーの改善。 Raspberry Piなどの組み込みシステムで利用できるようになりました。
  • Subversion 1.8のサポートとGitおよびMercurialのサポートに対するさまざまな機能強化
  • プロジェクトグループインターフェースの設計変更
  • さまざまな速度の改善
  • 検索結果のポップアップメニューの改善：選択したいくつかのファイルの「ファイルパスのコピー」、基本的なファイル操作のサポート、「選択>」のサポート




HTML5



  • Cordova 3.3+用のAndroid 4.4 WebKitのリモートデバッグのサポートが追加されました
  • Gruntサポートが追加されました
  • 利用可能なプラグインAvatar.js。 プラグインをインストールした後、Avatar.jsを使用してプロジェクトを作成できます




Javascript



  • AngularJSのサポートの改善
  • Knockoutテンプレートでのコード補完サポート
  • jQueryウィジェットとプラグインを作成するための新しいエディターのサポート
  • JavaScriptコードのデバッグがNashorn（JDK 8+）で行われるようになりました




OS Solarisのインストーラーはサポートされなくなったことに注意してください。 ただし、バイナリインストーラーの代わりにzipアーカイブを使用して、このOSでIDEを実行できます。



これはベータ版のみですが（リリースは4月にのみ予定されています）、新しいバージョンのIDEを試すことができます。 PHPバージョンに関しては、職場で1週間メインIDEとして積極的に使用しましたが、問題はありませんでした。
