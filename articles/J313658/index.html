Node.js 7.0.0がリリースされました。 非同期に会う/バベルなしで待つ





7番目のノードが黄色になりました！ 新機能：



  • V8エンジンはバージョン5.4.500.36に更新され、ES2015仕様（ES6）で定義されたJavaScript機能の98％、および一部は将来のES2017標準をサポートします。
  • 新しいURLパーサーは、WHATWGコミュニティによって準備されたURL設計標準に準拠しています。
  • バッファー、子プロセス、クラスター、ファイルシステム、Promiseの改善された作業
  • 変更の全リスト


バベルトランスパイラーなしでasync / awaitコンストラクトを使用できるようになったので、これは私にとって待望のリリースです。 これはすべてキー--harmonyによってオンになります。



今、あなたはこのようなバベルなしでできます： 



(async function() { let content = await require("request-promise") .get("http://example.com/"); console.log(content); })()


1週間前、Node.js 6.xブランチはLTSステータスを受け取り、その更新は30か月以内にリリースされます。 Node.js 8のリリースは2017年4月に予定されています。
