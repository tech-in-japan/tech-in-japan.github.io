オンラインコースの所有者は誰ですか？ 著作権と落とし穴

私は認めます：最初のオンラインコースの前に、私は著作権を完全に異なる方法で見ました-弁護士の正式な観点から。 私は教師でもあるので、私たちの法律の多くは一貫していないか間違っているようです。 それどころか、何かが嬉しい驚きであることが判明しました-たとえば、作成されたコンテンツに対する従業員の完全な権利。 これらの発見について説明します。



-はい、オンラインコースは、教師が作成する他のコンテンツ（マニュアル、教科書など）とほとんど変わりません。 したがって、よく知られている仮定で、私の記事は一般にあらゆる教育コンテンツに適用されます。









モスクワ州立大学の「公式」スタジオと、近所の控えめな肖像画。



顧客の誤解



雇用主-そして私の大学も例外ではありません-通常、フルタイムの教育者によって作成されたコンテンツに対する権利はデフォルトで組織に属していると考えています。 実際、これは完全に間違っています。 雇用主の機器で作成された作業も、勤務時間中に作成された作業も、自動的に雇用主の財産にはなりません。



例を挙げましょう。工場の機械の通常のターナーが、動作する代わりに動作するのではなく、素晴らしいメカニズムになった場合、プラントは特許権を取得しますか？ もちろん、そうではありません。なぜなら、労働者は自分の労働機能の範囲外で行動したからです。 雇用契約の「外側」で作成されたものはすべて、従業員のみに属します。



同様に、法律はオンラインコースにも適用されます-もちろん、そのようなコースを作成することは従業員の直接の責任ではありません。 その他の場合、コンテンツ（コース、教科書、トレーニングマニュアル-任意）は、次の条件が満たされた場合にのみ雇用主の所有物になります。



  1. 従業員には、オンラインコースを開発するための仕事が割り当てられました。 さらに、割り当てが書面、たとえば、注文、またはメモで完了していない場合、従業員は常に、割り当てを受け取っていないと言うことができます。
  2. このタスクは、雇用契約の一部として実行されます。 たとえば、物理学の教師が生物学のコースを作成するように指示された場合、これは仕事の割り当てではありません。
  3. 割り当ての結果は、理想的には、結果を含む行為またはメモ（スクリーンショット、復号化など）によって送信および記録されます。 それ以外の場合、従業員は、タスクが完了しておらず、コースに合格していないと常に言うことができます。
  4. 従業員は、給与に加えて報酬を受けました。


すべての条件が満たされていない場合、従業員は自分でコースを維持する機会があります。 この場合、彼は自国の大学を呪うのではなく、自分の代わりに商業コースで同じコースを開くことができます。



コース契約



コースの作成者が複数いることもありますが、互いの存在を知らない場合があります。 たとえば、コースにインタラクティブな要素（Stepicのようなタスク検証アルゴリズムなど）が含まれている場合、アルゴリズムの開発者はコースの完全な共著者になります。 コースにビデオがある場合、ディレクターは共著者になります。 音楽がある場合は、作曲者が権利を取得します。 あなたはこれらの状況の何百万も見つけることができます、そして、コースの大部分の創造者はさえ気づいていません。 そして、全員と契約を結ぶ必要があります。









このようなものは、現在それを読んでいる潜在的な共著者の顔を表しています。



幸いなことに、コースの著者/共著者は常に大学に圧力をかけることができます。結局のところ、彼はコースを合格するだけでなく、通常は監督もしています。 著者は必要に応じてテストを修正し、テスターを管理し、フォーラムで複雑な質問に答えます。 universityしないことは大学の利益のためであり、最も意識しているのは著者との複雑な契約を締結することです：コースの開発、資料の転送（ライセンス下または完全な）および一定期間のサポート。 しかし、私はそのような合意（NDAと完全な）を経済学部でのみ見ました。



科学目的で引用します



コースの開発中に、他の驚きに出会いました。 たとえば、民法では、教育目的での資料の引用を許可しています（商用製品であっても）。



  • 引用の範囲は、その目的によって正当化されます。
  • 著者が示されます。
  • 示されたソース（リンクが指定されています）


さらに、テキストの「正当なボリューム」で多かれ少なかれ明確である場合（引用は引用です）、画像とビデオでは明確な答えはありません。 映画科学のコースは、他の人のコンテンツ、つまり映画の断片に完全に基づいているとしましょう。 質問：そのような編集は合法ですか？









鳩の平和とピカソのその後の作品でさえ著作権で保護されています。 現代の芸術家について何が言えますか？



歴史的遺産



さらにいくつかの問題は、ソ連崩壊後の国に特有のものです。 第一に、原作者が原則として確立できない資料がたくさんあります。 それらが署名なしで公開されたとしましょう-古い写真での通常の状況。 ソビエト連邦では、60年代まで、写真の著作権は5年しか続かず、誰も著作権を身に着けていませんでした。 その後、保護期間が20倍以上に延長され、30代の写真がパブリックドメインから突然返されました-まだ著者はいません。 このような状況では、アーカイブイメージ（たとえば、最近の歴史に関するコース）は、自分の危険とリスクでのみ使用できます。相続人はいつでも、イメージの権利を主張して権利を主張できます。



ロシアの現代的なニュアンスがあります。 私たちの法律によれば、公式博物館と文化遺産オブジェクトの所有者には、建物や中のすべての写真を公開する「権利」があります。 これは、理論的には長い間パブリックドメインにあった絵画の使用が、博物館の訴訟で制限または禁止される可能性があることを意味します。 このような慣行があり、エルミタージュはこの点で特に際立っていました。









最後の禁止は、マットレスの広告におけるレンブラントによるドナウ川の使用に関するものでした。 それは正当化されたように見えるでしょう。 しかし、あなたは知っています、「最初に彼らはマットレスのために来ました-私は黙っていました...」



アウトバウンド著作権



コースが所属するもの（教師または大学）に関係なく、他のコースのユーザーと著者が使用できるフレームワークを確立することは常に価値があります。 法的には、これは学生との個別のライセンス契約によって行われます。 ここでは、コースをオープンCCタイプのライセンスで配布するかどうかを指定できます。 狭い質問を明確にする必要があります。コースを他の言語に翻訳したり、カリキュラムに含めたりすることは可能ですか。 他の人の講義でビデオを表示したり、YouTubeにアップロードしたりすることは可能ですか？



「コースライセンス」では、もう1点-学生教材を使用する権利を示すことが重要です。 たとえば、学生がコースでエッセイを書いた場合、著作権は学生のために予約されているため、デフォルトでは（例としてであっても）コースでエッセイを使用することはできません。 したがって、私はコースの最初の資料で、割り当て、アンケート、さらにはコースフォーラムへの投稿に対するすべての回答を、予告なしにコース作成者が使用できることを個人的に規定しました。 もちろん、個人データに加えて-しかし、これは「まったく異なる話」です。
