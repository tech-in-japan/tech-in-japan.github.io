シスコ 認証パス

シスコの認定は、ITの分野で最も権威のある認定の1つと見なされています。 これは、他の多くのベンダーの認定システムとは異なり、シスコ認定試験には、質問、シミュレーション、仮想機器のセットアップのための標準テストに加えて含まれるという事実によるものです。



一般的に言えば、シスコは5つのレベルの資格を識別します（9つの異なる領域があります）。







エントリー （Cisco Certified Entry Networking Technician）は、専門家がネットワークテクノロジーの初期知識を持っていることを確認するエントリーレベルのレベルです。 このレベルは、シスコの2つの領域-CCENT（シスコ認定エントリネットワーキング技術者）とCCT（技術者）に存在します。 最初の方向は、シスコのネットワークテクノロジーの分野での旅を始める専門家向けに作成された9つのステージすべてに共通です。



2つ目は、サービスセンター向けに作成される可能性が高い-CCTの専門家は、診断を実行し、データを復元し、シスコデバイスのコンポーネントを修復および交換できるはずです。 CCTには3つの重点分野があります。

CCT Data Center-データ処理センター（DPC）のネットワーク機器を扱うスペシャリスト向け-Cisco Nexusファミリー、Cisco Unified Computing System（Cisco UCS）機器、Cisco Adaptive Security Appliance（Cisco ASA）ファイアウォール。



CCT R＆S-コアネットワークを使用するプロフェッショナル向け-Cisco CatalystスイッチおよびCisco ISRコアネットワーク、Cisco ASAファイアウォール、Cisco Aggregation Services Router（Cisco ASR）など。

CCT TelePresence-ユニファイドコミュニケーション機器の分野で働く専門家向け-つまり、ビデオ会議システムとテレフォニー。



すべての証明書は3年間有効です。







アソシエイト （Cisco Certified Network Associate）は、機器の設計、設定、中小規模のネットワークの管理とサポートに関する知識とスキルを確認する基本レベルです。 1〜3年の機器を扱う専門家向けに設計されています。 このレベルには、ルーティングとスイッチング、設計、データセンター、セキュリティ、サービスプロバイダー、サービスプロバイダーの運用、ビデオ、音声、ワイヤレスの9つの領域があります。 アソシエイトレベル試験に合格するための要件はありません（CCENTは不要です）。 すべての証明書は3年間有効です。







プロフェッショナル （Cisco Certified Network Professional）-プロフェッショナル。 プロフェッショナル認定の存在は、スペシャリストがこの分野に関する深い知識と、エンタープライズレベルで機器を使用した幅広い経験があることを示しています。 試験合格時にプロフェッショナルのステータスを取得するには-必須要件があります-アソシエイトステータスが存在し、それなしではプロフェッショナルのステータスを取得できません。 つまり、要件を満たしている（プロフェッショナルレベルの試験に合格している）場合でも、ステータスを取得するには、準レベルの試験（または試験に応じて試験）に合格する必要があります。 このレベルには、ルーティングとスイッチング、設計、データセンター、セキュリティ、サービスプロバイダー、サービスプロバイダーの運用、音声、ワイヤレスの8つの領域があります。 ビデオの方向（すべてよりも後に表示されます）の場合、CCNAビデオは現在最高のステータスです。 すべての証明書の有効期限は3年です。







エキスパート （Cisco Certified Internetwork Expert）-エキスパートレベル。 認定は1993年に登場し、現在最も権威のある技術認定です。 CCIE試験は通常、2つのパートで構成されます：筆記（ピアソンVUEテストセンターで）と実践（Ciscoラボ）。 試験には必須の要件はありません。つまり、アソシエイトまたはプロフェッショナルのステータスがなくても受験できます。 この段階では、前の段階と同様に、ルーティングとスイッチング、設計、データセンター、セキュリティ、サービスプロバイダー、サービスプロバイダーの運用、コラボレーション、ワイヤレスの8つの方向があります。 ご覧のとおり、Professionalとは異なり、音声はありません。音声はビデオと組み合わされ、この方向はコラボレーションと呼ばれていました。 前の手順とは異なり、すべての証明書の有効期間は2年です。



ちなみに、統計によると、認定された専門家の約3％のみがCCIEレベルに達しています。 合計で、約50,000人のCCIEステータスのスペシャリストがいます（一部は複数の分野でステータスがあり、4分の1が非アクティブな認定を受けている場合）。



アーキテクト （シスコ認定アーキテクト）-最高レベルの認定。 2010年に登場したこの状態の人は世界中に約12人います（2013年の初めには7人でした）。 このステータスを取得するには、委員会の前に一種の卒業証書を作成し、保護する必要があります。 このレベルには、デザイン、R＆S、ワイヤレス、セキュリティが含まれます。 配信の必須要件は、CCDE認定（Cisco Certified Design Expert）の可用性です。

認証の有効期間は5年です。







スペシャリスト認定 （Ciscoスペシャリスト認定）-アソシエイト-プロフェッショナルレベルのさまざまな分野の専門認定。

これらの認定の一部の試験は、アソシエイト-プロフェッショナルレベルのステータスを取得するために必要です。 これらの専門分野の主な目的は、特定の分野における特定のステータスとコンピテンシーのパートナー企業による買収です。 8つの重点分野があります：ビジネス認定、データセンター認定、モノのインターネット認定、ネットワークプログラマビリティ認定、オペレーティングシステムソフトウェア認定、セキュリティ認定、サービスプロバイダー認定、ビデオ認定。

これらの専門分野の有効期間は通常2年です。

ルーティングとスイッチング 。

最も古く、最も人気のある方向（かつて唯一の方向）は、Cisco機器に基づくネットワーク内のネットワーク機器のルーティングとスイッチングに専念しています。



最初のステップ-CCNA Routing＆Switchingは2つの部分で構成されています。





ステータスを取得するには、 100-101 ICND1および200-101 ICND2の 2つの試験、または1つの一般的な200-120 CCNAのいずれかに合格する必要があります。



準備のためのコースがあります：



相互接続Ciscoネットワークデバイスパート1（ ICND1 ）;

相互接続Ciscoネットワークデバイスパート2（ ICND2 ）;

Ciscoネットワークデバイスの相互接続：高速化（ CCNAX ）。

同様に、2つのコース（それぞれ40時間）を受講するか、1つのコースを加速（60時間）することができます。



すべての証明書は3年間有効です。再認証（ステータスの延長）のために必要です。



CCNA試験に再度合格するか、より高いレベルの試験（ProfessionalまたはExpert）、またはシスコスペシャリスト認定技術試験に合格します。

1月29日まで、ステータスを取得するために試験を受験できます（たとえば、2つの古いエディションと1つの新しいエディション）。1月29日以降は、新しい試験のみが残ります。
