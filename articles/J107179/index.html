WiMax 2X：1つのUSBモデムを介した2人のオペレーター

WiMaxには1つのメインプラスと1つのメインマイナスがあります。高速ですが、カバレッジは不完全です。 この点で、私は多くが携帯電話ネットワークで行うように、複数のオペレーターに一度に接続したいです。 そのような人々のために、2枚のSIMカードを備えた電話を思いついた。 しかし、貪欲なWiMaxオペレーターは、ユーザーが競合他社に接続することを望みません。 オペレーターごとに、独自のUSBモデムを購入する必要があります。 これは不便です。まず、余分なお金を払う必要があり、次に、外出中にモデムを接続するのは不快です。 モスクワにはそのようなオペレーターが2つあり、長い間、2つの異なるモデムに苦しめられていましたが、昨日、テクノロジー企業がメガコーポレーションの欲望に打ち勝ちました。 解決策は非常に簡単であることが判明しました！



接続プロセス全体を5分間のクリップで示しました（はい、それは非常に簡単で高速ですが、どれだけこれに行きましたか）。







ビデオを見たくない人のために、アルゴリズムは：

1） Yota Jingleモデムを使用し、モデムのネイティブソフトウェアから直接インストールします。 ソフトウェアはサイトからではなく、モデムから取得します！

2） Clear Connection Managerをインストールします。 インストールは簡単です。

3） Connection Manager 2501をインストールします。 あまり複雑ではありません。

4）フォルダからすべてのDLLをコピーします

C：\ Program Files \ Clearwire \ Connection Manager

フォルダーへ

C：\ Program Files \ Comstar \ Connection Manager

その後、Clear Connection ManagerはYota Jingleモデムを認識し、Comstarネットワークに接続し始めます。

YotaAccess経由でのみYotaに接続できます。



便利な機能を提供するオプションのステップ。

yotatesterフォーラムにアクセスして、特別なプロファイルをダウンロードします。 フォルダに書き込みます：C：\ Documents and Settings \ Lena \ Application Data \ Comstar \ Connection Manager

Lena-ローカルユーザーに置き換える必要があります。

その後、Comstar Connection Managerで、Yotaネットワークを表示できます。 彼はそれに接続できませんが、時間を節約するために信号レベルを推定できます。



重要な注意事項：

1）これはすべてWindows XPで行いました。 他のシステムで機能するかどうかは明らかではありません。

2）Comstar Connection Managerをエラーなしで起動するには、Yota Access Service（コントロールパネル、管理、サービス）を無効にする必要があります。 サービス設定を入力して、手動で開始します。 それ以外の場合は、モデムを自動的にピックアップしてComstarに接続できなくなります。

3）プロファイルをコピーするには、システムフォルダーを表示する必要があります（コントロールパネル、フォルダープロパティ、ビュー）。



私が欲しいもの：

1）1つのプログラムからYotaとComstarのネットワークに行きたい。 ただし、Comstar Connection ManagerはComstarのみに移動します。 Clear Connection ManagerはYotaにのみあります。 YotaAccessはYotaネットワークのみを認識します。 そのため、今のところ、あるプログラムをオフにし、別のプログラムをオンにして別のネットワークに移動する必要があります。

2）Yota Jingleは最高のモデムではありません。 もっとコンパクトにしたいです。 ただし、Comstarモデムは証明書のためにYotaで動作しません。 また、フォーラムで判断すると、Samsung SWC-U200を再フラッシュする必要があります。 さらに、再フラッシュした後、私が理解しているように、1人のオペレーターでのみ動作します。

3）理論的には、Intel 5150/5350チップセットを搭載したネットブックでより便利な切り替えを行う必要があります。 しかし、私はそれを持っていません、私の友人は持っていません。 やってみたい！
