サムスンの新しいフラッグシッププレビュー：GALAXY NOTE III





9月25日に、サムスンの最新スマートフォンであるSamsung Galaxy Note 3スマートフォンの販売が正式に開始されました。これはコンピューター業界の大手メーカーのもう1つの成果です。



これまでのところ、携帯電話の機能、第三世代のGalaxy Noteの技術革新についてお話ししたいと思います。



スクリーン



5.7インチの画面を備えた携帯電話は、もはや携帯電話やスマートフォンと呼ばれることはありません。これは、スマートフォンとタブレットの間の特定の中間リンクに起因するものです。



Galaxy Noteの前世代と比較して、画面の作業対角線が増加する傾向があります。 開発者は、151.2 mmの同じ高さを維持することに基づいて、望ましい範囲から最大値を絞ることができるようです。一方、コミュニケーターの幅と厚さはわずかに小さくなっています（それぞれ79.2と8.3 mm）。



新製品の画面は、1920 x 1080ピクセルの解像度を備えたフルHDスーパーAMOLEDです。これは、実際には非常に高いディテールとペンティル（粒状性）の欠如を意味します。 Galaxy Note 3には驚くほどバランスの取れたディスプレイがあり、1600万色を表示し、ピクセル密度は386 ppiです。 この画面には、非常に優れた視野角、コントラスト、彩度があります。 ディスプレイの周囲のフレームは、ゲームで時間を過ごしたり、そのような電話で映画を見たい人の希望を満たすために必要な最小サイズです。



注3スタッフィングとパフォーマンス



Galaxy Note 3のような「スマートフォンの製造」における最近の進歩のリリースにより、ポータブルモバイルデバイスが戦術の面で現在の予算のラップトップを上回ることができることはもはや驚くことではありません。 もちろん、これは完全に正しい比較ではありません。デスクトップPCプラットフォームで実行されるはずのタスクがモバイルシステムで異なる場合が多いためです。



スマートフォンでは、インターフェイスとシステム全体のパフォーマンスと応答性が非常に重要です。 OS開発者が定期的に頭脳を改善し、モジュールやコンポーネントを変更しようと努力すると、ハードウェアリソースの必要性が高まります。 確認してください-サムスンギャラクシーノート3はタスク次第です。



デバイスの充填は、Qualcomm Snapdragon 800に基づいています。電話はさまざまなプロセッサバージョンに基づいて配信されます。欧米市場では、LTEネットワーク用のクアッドコア2.3 GHz Krait 400に基づくデバイスの販売と、 3Gネットワ​​ーク-周波数1.9 GHzの8コアプロセッサを備えたExynos 5 Octa 5420プラットフォームに基づいています。 このような周波数では実際には8つのコアのうち4つしか動作しません。 残りの4つのコアのクロック周波数は1.3 GHzです。



グラフィックプロセッサは、8コアのMali-T628チップであり、DirectX 11、OpenVG 1.1、OpenGL ES 1.1、2.0、3.0、OpenCL 1.1標準で動作し、これまでで最も生産性の高いチップの1つです。



ユーザーは間違いなくスマートフォンに3 GBのRAMを見ることができてうれしいです。実際、これは新しいスマートフォンの中で今日の記録です。 システムのパワーは、Android 4.3 Jelly Beanの最新バージョンで完全な「バギー」インターフェースを提供します。



さまざまなバージョンで、データを保存するために16/32または64 GBの内部メモリを備えたバージョンが提供されます。 また、microSDXCカードを介して最大64 GBまでメモリを拡張する可能性もあります。



電話機には従来、2つのカメラがあります-オートフォーカス、顔と笑顔の認識、安定化、LEDフラッシュを備えたメインの13メガピクセルで、最大4128 x 3096ピクセルの解像度で写真を撮影し、1080pで60フレーム/秒でビデオを撮影できます。フロントカメラもあります。 2番目のカメラはビデオコール用で、このタスクには2メガピクセルの解像度で十分です。



Samsung Galaxy Note 3ワイヤレステクノロジーはGSM / GPRS / EDGE 850/900/1800/1900、HSPA + 850/900/1900/2100通信規格で、Wi-Fiネットワークで動作しますa / ac / b / g / n（範囲2 、4および5 GHz）、およびBluetooth 4.0データ転送インターフェイス、赤外線送信機、NFC、GlonassをサポートするA-GPSモジュールがあります。



所有者は、温度、湿度、気圧、コンパス用の追加センサーの存在に満足するかもしれません。 3200 mAhの容量のバッテリーを忘れないでください。 ご存知のように、バッテリー寿命はモバイルデバイスの従来の問題領域であるため、このような強力な充填、スクリーン、および実際の負荷で通常のバッテリーがどれだけ十分であるかを判断することは困難です。 いずれにせよ、所有者はいつでも予備のバッテリーを購入できます。



外観



角の丸い長方形の電話ケースはプラスチック製で、画面はガラスです。 肌のざらつきのある仕上がりの下に模造された裏表紙に満足しました。 バッテリーパックにアクセスするための保護カバーの周囲全体が糸で縫われているようです。 この小さな革新は間違いなく成功したデザインであり、実用的なソリューションです。片手で電話機をより快適に持ち、偶発的な滑りの可能性を減らすためです。







すでに述べたように、Galaxy Note 3は以前のモデルと比べてわずかに小さく、したがってわずかに軽い（注2の183と比較して168グラム）。



スマートフォンの下部には、以前のノートシリーズと同様に、Sペンスタイラスが伸びています。 ペンは十分に固定されており、ケース上で非常によく変装しています。



コンピュータに接続するためのコネクタは、USBホストとMHL 2.0をサポートする最新世代のmicroUSB 3.0形式で実装されています。 インターフェースは古いバージョンと下位互換性がありますが、新しいケーブルを使用する方が良いでしょう。これにより、より高速な充電とデータ転送が可能になります。



ケースに2つの高品質のマイクが存在することで、少なくともよりリアルなビデオ録画が可能になります。また、有線ヘッドセットを使用する場合にも利点となります。



側面のボタンは、Galaxy Noteシリーズの通常の場所にあるため、片手で音量を調節したりロックしたりできます。



ソフトイノベーション



新しい注3では、Samsungがスタイラスの機能を改善したため、面白い空中コマンド機能が追加されました。







画面に触れることなく、テキスト認識機能付きのポップアップメニューにアクセスできます。スタイラスを持ち上げるだけです。 このメニューを使用すると、ウェブコンテンツの興味深い部分を携帯電話のメモリにすばやく保存できます。 このような多くの機能を実行し、日常のタスク（メモ帳/コピー/検索エンジン/ウィンドウマネージャー）の改善になったアプリケーションは、スクラップブックです。







最も興味深いイノベーションは、オペレーティングシステムプラットフォーム上で任意の正方形と長方形を描画する機能です。 フォームはすぐに認識され、ほぼ同じメニューウィンドウが表示されます。 このようなウィジェット内では、ブラウザーページ、個々のアプリケーションを開いて、ビデオを見ることができます。 要するに、これは、特にお気に入りのシリーズをチャット、サーフィン、視聴したい人にとって、便利なマルチタスクです。
