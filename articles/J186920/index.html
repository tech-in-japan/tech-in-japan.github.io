モスクワ市のアプリコンテストのモバイルアプリコンテスト'13

Android開発者を招待して、モスクワのUnified Mobile Platform（EMP）に基づく都市型モバイルアプリケーションの競争に参加してください。



都市の問題を解決し、大都市圏の住民の日常生活をより簡単で楽しいものにする最適なアプリケーションの競争は、 モスクワのIT部門と共同で2013年7月17日から9月20日まで開催されます。







いつ？



申請受付： 2013年7月17日-9月20日。

結果の発表： 2013年10月11日、演技が率いる専門審査員 モスクワ市情報技術局長のArtem Ermolaevは、IV International Apps4All Mobile Developers Forumの枠組みの中で、AndroidプラットフォームでのUrban Mobile Applicationコンテストの受賞者を発表します。



参加する方法は？



ステップ1. コンテストWebサイトに登録し、EMP開発者の個人アカウントにアクセスする必要があります。 この瞬間から、プラットフォームのすべての機能を使用できます。



  • 20のカテゴリの都市インフラ全体で189のデータセットに簡単にアクセスできます。
  • 20を超えるサービス方法を使用する機能-モスクワのモバイル認証から都市の位置情報サービスまで。


ステップ2. EMF APIを使用して、OS上のスマートフォンとタブレットのアプリケーションのアイデアの説明を提供します。

ステップ3.以前に発表されたコンセプトに基づいて作成されたAndroid OSで動作するプロトタイプまたは最終製品を無料でダウンロードするためのリンクを送信します。



誰が参加できますか？



API EMPを使用して、Android OS上のモバイルアプリケーションの形式で都市の問題を解決するプロジェクトを実施するために、誰もがコンテストに参加するよう招待されています。



なぜ参加するのですか？



モスクワには新鮮なアイデアが必要です。私たちはそれらを実装するのを手伝います。 2013年5月、開発者はモスクワのUnified Mobile Platform（EMP）の機能にアクセスできるようになりました 。 EMPは都市のモバイルサービスのエコシステムであり、その目的は都市のオープンデータと電子サービスへのアクセスを促進することです。 このプラットフォームは、モスクワのすべてのモバイルアプリケーションとサービスの作業を提供し、EMPユーザーの合計視聴者は120万人を超えます。



賞品はどうですか？



  • メインの勝者には、15万ルーブルの賞金が贈られます。 その申請はモスクワ政府が推奨する公式サービスになります。
  • 3つの開発チーム（1〜3か所）が、テルアビブのGoogleのスタートアップと開発者のキャンパスに行きます。


ここでコンテストの完全なルールを読んで参加を申請できます 。
