電話部分と通信



Xiaomi Mi Max 3は、2枚のnanoSIMカードの使用をサポートしています。 グローバルバージョンは、ロシアのすべてのバンドGSM、WCDMA、LTEをサポートしています。



無線モジュールは1つしかありませんが、両方のカードが4Gで同時に動作できます。 VoLTEは両方のカードで機能します（オペレーターがサポートしている場合）。 MIUI 9システムがスマートフォンにインストールされたときにテストを行いました。



MegaphoneとTele2の動作をテストしました。 モスクワおよびモスクワ地方では、これらのオペレーターに問題は発生していません。 両事業者の4Gデータ伝送と3G / 2G音声通信は問題なく機能しました。 ヤードでは、両方のオペレーターの速度は許容範囲内でした（メガフォンでは、一般的に禁止されていました）。 YouTube、HD VideoBox、ビデオチャットなどは問題なく機能しました。



音声通信の品質は完璧です。 音声スピーカーの予備音量は。 対談者は騒がしい場所でさえ私をよく聞いてくれました、私もそうしました。 スピーカーフォンの在庫量も良好です。 vibromotorは強力です。 設定では、振動モーターの動作レベルを選択できます。



Wi-Fiモジュールは、802.11a / b / g / n / ac、2.4 GHz / 5 GHz、MIMO 2x2をサポートしています。 他の多くのXiaomiスマートフォンと同様に、2.4 GHzモードでは2.4 MHzのチャネル幅のみが使用されます。 このため、速度は大幅に制限されます。 基地局（Xiaomi Mi Router 3G）から3メートルの位置で、1つの鉄筋コンクリートの壁を通って、スマートフォンは2.4 MHzでチャネル幅20 MHzでほぼ最大速度を示しました。 しかし、5 GHzネットワークでは、MIMO 2x2をフルに使用することですでに高速でした。 これはスマートフォンなので、iperfを使用してもほとんど意味がありません。Speedtestで十分です。

ビデオを再生する



標準のビデオプレーヤーMi Videoの機能は非常に少ないため、テストには使用しません。 検証のために、MX PlayerをHW +モード（MediaCodec）で使用しました。 Xiaomi Mi Max 3はMediaCodecでAC3デコーダーをサポートしていないため、ドルビーデジタルオーディオトラックをデコードする機能は、使用するプレーヤーに完全に依存しています。 MX PlayerはAC3ソフトウェアデコーダーをサポートしています。 MediaCodecのすべての主要なビデオデコーダが存在します。 H.265 Main 10のサポートがあります。



HD VideoBoxが問題なく再生されたさまざまなソースから利用可能なビデオ。 ただし、画面サイズが大きいため、720pリーパーの低品質がはっきりと見えるため、1080pオプションを選択してください。



NASからの1080pビデオ（BDRip、H.264 / H.265、AC3）は問題なく再生されました。 1080p60と1080p50では、H.264とHEVCにも問題はありませんでした。 均一性について不満はありません。



YouTubeは1080p60と1080p50をサポートしているため、再生と均一性に関する苦情はありません。 ズーム機能は、特に新しい18：9スマートフォン（だけでなく）で機能します（トリミングで縦横比を変更せずに塗りつぶします）。

性能



スマートフォンには、最新のQualcomm Snapdragon 636 SoC、最大1.8 GHzの4 Kryo 260 HP、および最大1.6 GHzの4 Kryo 260 LP、Qualcomm Adreno 509 GPUが搭載されています。







システムとプログラムは迅速かつスムーズに動作します。



すべてのCPUコアの最大負荷で、スマートフォンはスロットルなしで約3分間動作でき、最大のパフォーマンスを提供します。 その後、核の温度がしきい値に到達し、速歩が開始されます。 生産性は15〜20％低下します。 これは約11分間続きます。 その後、スロットルが激しくなり、パフォーマンスがさらに低下します。 同時に、パフォーマンスの全体的なレベルは依然として高いレベルに維持されます。 スマートフォンの背面の金属部分は、最上部で最大40°Cまで加熱されます。







明確にするため、Xiaomi Redmi Note 4X（Qualcomm Snapdragon 625、3 / 32）およびXiaomi Mi A2（Qualcomm Snapdragon 660、4 / 64）のパフォーマンスデータを提供します。

充電中



スマートフォンには、Quick Charge 3.0、最大電力18ワットの標準充電器が装備されています。







スマートフォンの仕様では、QC 3.0のサポートが発表されています。 QCを使用すると、対応するアイコンが表示されます。







スマートフォンを完全に放電し、EBD-USBテスターを介して標準メモリを接続します。 スマートフォンは最大18ワットを消費し始めます-これは充電コントローラーのロジックの制限であり、電圧は6 Vをわずかに超えています。電流強度は徐々に低下します。 これは約1時間30分続きます（ステージCC）。 この間、スマートフォンのバッテリーは最大85％まで充電されます。 その後、充電コントローラーはCVモードになります。 45分後、スマートフォンは100％（2時間15分）を報告しますが、実際には充電は継続されます。 フル充電には2時間49分かかります。







充電中、スマートフォンはわずかに熱くなります。



しかし、それだけではありません。 スマートフォンはUSB Power Deliveryをサポートしています。このサポートは公式には発表されていません。急速充電用のPower Deliveryサポートを備えたラップトップから電源を使用できます。Mi Max 3は、9 Vの電圧のプロファイルを使用します。最大電力は18ワットに制限されています。
