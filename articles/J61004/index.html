TreeBrowser（タブのないブラウザ）+デモアプリケーション

2009年夏のMozilla Labs Design Challengeに参加することにしました。コンテストの話題があまりにも美味しくないためです-「ブラウザのタブを置き換えるために提供できるもの」。 開始するには-ビデオ：







カットの下-アイデアとスクリーンショットの詳細な説明。



前提



コントラのアイデアはMozilla Labsで生まれました。なぜなら、タブ（タブ）はたくさんあるとうまく機能しないからです。 画面に収まらず、ユーザーに十分なヒントを与えないため、ページにアクセスしたかどうか、多くのタブの1つで開いているか、新しい方法で開く必要があるかをすぐに思い出せます。



問題はタブだけではないようです-それらに加えて、訪問したページの履歴、ブックマーク（お気に入りのサイト）、現在のタブのページのスタック（戻る/進むナビゲーションボタン）があります。 これらはすべてブラウザに個別のエンティティとして実装されており、どこにでもスケーリングの問題があります。



たとえば、7〜10個のタブがある場合でも、いずれかのタブに切り替えることを自信を持って覚えていますか。「戻る」または「進む」ボタンをクリックすると、どのページに移動しますか？ 現在のページにどのようにアクセスしたか、確実に言えますか？



アイデア



ブラウザには、「タブ」、「訪問履歴」、「お気に入り」という通常の個別の概念がないと想像してください。 代わりに、アクセスするページ用の共通スペースがあります。 このスペースでは、各ページが独自のスケッチで表され、ページはリンク（リンク）で相互に接続されています。 スペース内を移動し、その中のページを拡大縮小して移動し、トピックごとにグループ化できます。







このようなスペースは「マインドマップ」に非常に似ています（これにロシア語の用語が確立されているかどうかわかりません）。 主題をよりよく理解し、知識のある領域をより視覚的にするために、人々が自分で描くグラフ。 以下に、Webで見られるこのような手描きのマップの面白いかわいい例を示しました。







アイデアは、サイトにアクセスするとブラウザがそのようなマップを自動的に作成するということです（もちろん、プロセスに介入する機会を与えます）。 これにより、認知の負荷が軽減され、ナビゲーションプロセスがコンピューターから遠く離れた人々にとってもより視覚的で理解しやすくなります。



このアプローチでは、ユーザーは現在のオンダックでリンクを開くか、新しいオンダックでリンクを開くかを考える必要はありません。タブの概念はありません。 最新のブラウザとメモリは、すべてのデータを含むブラウザウィンドウの完全な状態を保持し、ページ間の高速切り替えのために複数のブラウザインスタンスのプールを使用して復元できます。



すでにアクセスしたページのアドレスを入力すると、ブラウザは新しいページを開く代わりに、単にマップ上の既存のページにジャンプします。 したがって、マップ上には、現在のページから開いたページの履歴が表示されます（つまり、訪問の履歴は現在のセッションと統合されます）。



デモアプリ







このアイデアをデモアプリケーションに変換しようとしました。デモアプリケーションは、ビデオとスクリーンショットで見ました。 次のリンクからダウンロードできます。



http://ai-studio.com/treebrowser/TreeBrowserDemo.zip（350 KB）。



アプリケーションはWindows XP / Windows Vista / Windows 7で動作し、インストールは不要です。アーカイブを任意のフォルダーに解凍するだけです。 試してみて、遊んでください。 どう思う？



開発



デモの機能がかなり制限されていることは明らかです。 実際のブラウザには、次の機能が必要です（または必要な場合があります）。

  • セッションを友人と共有したり、これらのセッションをバックアップしたりできるように、保存およびロードする機能を備えたさまざまなカードのサポート。
  • 特定のサイトセットに基づいてWebサーフィンを開始できるように、トピックごとにグループ化された最も便利なサイト（お気に入りに類似）で「スターターマップ」を作成する機能。
  • マップのさまざまな表現のサポート（たとえば、デモアプリケーションなどの自由形式、または一定期間のページアクセスのシーケンスを表示するのに適した構造化）;
  • 「タイムマシン」モードのサポート。過去の特定の時点でマップを表示したり、ページを開く順序やマップの「成長」をアニメーション形式で再生したりできます。
  • アドレスバー（検索フィールドとしても機能する）で指定された検索条件に一致するマップページを検索および強調表示する機能。
  • キーボードを完全にサポートし、カードを操作するとき。
