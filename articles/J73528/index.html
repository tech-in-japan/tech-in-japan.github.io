Zend Studio 5.5の「可愛さ」

デフォルトでは、Zend Studio 5.5には、コードをマークアップするための2つの配色しかありません。 そして彼らは非常に怖いです。 かつて、私はIDEに何時間も座って快適にする方法に非常に興味があったので、もっとまともな配色を求めて深く掘り始めました。



その結果、約5〜6色の配色が見つかり、作業しやすくなりました。







以下では、アーカイブへの猫のリンクとそれらのインストール方法の簡単な説明の下で、Zend Studio用に見つけた回路をダウンロードすることをお勧めします。



1. zend-schemes.zipアーカイブをここに保存します 。 次のスキームをアーカイブに含めました：Katulus、Komodo、RubyBlue、Navajo-Light、Evening、Experienced、Ambients。



2.これらのカラースキーム（W7の場合）を\ユーザー\ユーザー名\ ZDE \ config_5.5 \スキームにZendカットで入力します。



3. 必ずこのフォルダーに読み取り専用アクセス許可を設定してください 。 それ以外の場合、何らかの理由で、このフォルダーはZendの終了後にクリアされます。



4. Zendを実行し、[ツール]→[設定]→[色とフォント]に移動し、いずれかの配色を選択してお楽しみください。



これが誰かを助けてくれたら嬉しいです。



PS（2010年3月22日）：アーカイブをRapidから別のホスティングに移行しましたが、すべてが正常に機能するようになりました。
