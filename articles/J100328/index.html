ビデオ/オーディオフラッシュプレーヤーの概要

このトピックでは、ハブで見つけることができる無料のビデオ/オーディオフラッシュプレーヤーの長所と短所を祝福しようとします。





Flowplayer



サポートされているメディア形式： FLV、SWF、MP3、MP4、H.264ビデオ、JPGおよびPNG

デモ： http : //flowplayer.org/demos/index.html

長所：

  1. ビデオ形式とオーディオの両方を再生します。
  2. オーディオでは、スクリーンセーバーとして画像を挿入できます。
  3. 膨大な数のプラグイン
  4. リンクをメールで送信するためのプラグイン（単純なmailto :)
  5. プレーヤーをブログに挿入するためのコードを生成するためのプラグイン。
  6. TwitterやFacebookを含む人気のブックマーク/ソーシャルネットワークに公開するためのプラグイン。
  7. 柔軟でカスタマイズ可能なインターフェース。
  8. 全画面モード。


短所：

  1. 高品質のビデオを再生する機能は見つかりませんでしたが、おそらくこれはプラグインを作成することで決定されます。
  2. 英語のみのインターフェース。


アップポッド



サポートされるメディア形式： FLV、F4V、MP4、MOV、M4A、MP4V、3GPおよび3G2、MP3、H.264

デモ： http : //uppod.info/

プレーヤーの古いバージョン： flv-mp3.com/en

長所：

  1. ロシア語のインターフェースとドキュメントがあります。
  2. ビデオ、オーディオ形式を再生します。
  3. ブログに挿入するコードを生成することが可能です。
  4. ビジュアルエディターの柔軟なカスタマイズ可能なインターフェイス。
  5. 全画面モード。


短所：

  1. サイトに登録する必要があります。
  2. 有料のPROバージョンでは、高品質のビデオを再生できます。


Cinema HD Player FS



サポートされるメディア形式： H.264 MOV、MP4、FLV、およびFlashでサポートされるその他の形式

デモ： http : //sonettic-cinema.com/products/players/free/cinema-free.htm#samples

長所：

  1. シンプル。
  2. ブログに挿入するコードを生成することが可能です。
  3. 全画面モード。
  4. 柔軟なカスタマイズ可能なインターフェース。


短所：

  1. 英語のドキュメント。
  2. いくつかの例があります。
  3. 一時停止中の広告。
  4. 高品質のメディアを再生する可能性はありません

Jwプレーヤー



サポートされるメディア形式： FLV、H.264 / MPEG-4、MP3およびYouTubeビデオ

デモ： http : //www.longtailvideo.com/players/jw-flv-player/#jwplayer

長所：

  1. ビデオ形式とオーディオの両方を再生します。
  2. シンプル。
  3. ブログにコードを挿入するプラグインや、高品質のメディア表示に切り替えるためのボタンなど、多くのプラグイン
  4. 各プラグインの詳細なドキュメントがあります。
  5. 全画面モード。


短所：

  1. 英語のドキュメント
