IntelがClDNN [High Performance Deep Learning Library]を開きます

公式のプロジェクトリポジトリは数日前に公開されました。 このニュースについてもう少し詳しく説明し、トピックに関する有用な情報源を提供します。





/ Flickr / fdecomite / cc



clDNN（ディープニューラルネットワーク用の計算ライブラリ）の主な範囲は、画像認識とセグメンテーションです。 高性能ライブラリは、Intel HD / Iris Graphicsを使用した推論の構築を加速するDLアプリケーションで動作します。



clDNNはApache 2.0の下でライセンスされており、 AlexNet 、VGG、GoogleNet、およびResNetをサポートしています（最後の3つについて少し ）。 このライブラリは、CおよびC ++の畳み込みニューラルネットワークで動作するように最適化されたブロックで構成されています。

ディープニューラルネットワーク （DNN）は、自動分析などの分野でのビデオ分析、自然言語処理（NLP）、および問題解決にも使用されます。 DNNは、計算リソースとさまざまなアルゴリズムの実装の観点から非常に要求が厳しいため、この分野での新しい発見は常に大きな関心を集めています。 たとえば、同社は最近、コンピュータービジョンテクノロジーを操作するためのテーマSDKの一部である無料のIntel Deep-Learning Deployment Toolkitを導入しました。



プロジェクト「01」について



「01」は、Intelのオープンソースコミュニティです。 同社のエンジニアは仕事に積極的に参加し、オープンソリューションのコレクションを補充しています。 各プロジェクトには独自のマイクロサイトがあります（例： OpenStack 、 Crosswalk 、 Developer Journey ）。
