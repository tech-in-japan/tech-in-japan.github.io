Vivaldi 1.0.283.8でのWebパネルのショートカットと入力アドレスの改善されたリスト

みなさんこんにちは！



時間が経つにつれて、「時間X」は完全なベータ版のリリースに近づいており、主に既存の機能の「研磨」とデバッグに取り組んでいます。 確かに、機能を拡張することを忘れないでください。



特に、入力されたアドレスのリストの処理を継続しました。 この機能がその目的によりよく適合したことを願っています。



もう1つの興味深い革新は、Webパネルに関するものです。 これで、個々のキーボードショートカットを割り当ててアクセスできるようになりました。



また、私たちのボランティアのテスターであるソプラノに感謝します。 デフォルトのブラウザにプリインストールされているExpressパネルのサムネイルを最適化するという素晴らしい仕事をしました。 あなたが彼らの努力に感謝することを願っています：



既知の問題



はい 残念ながら、Expressパネルのセルが消えるという非常に不快なバグを解決する方法を引き続き探しています。 最寄りのアセンブリの解決策が見つかることを願っています。



今日は以上です。 以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
