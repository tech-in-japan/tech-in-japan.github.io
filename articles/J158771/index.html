SDSではなくSDS：ストレージの経済性



フォーラムのセッション部分での彼のプレゼンテーション「データストレージの新しい経済学」で、Fedor PavlovはSDSのトピックにも触れました。 彼は、現代のストレージシステムの欠点から始めました。 それらを交換することは難しく、時間がかかり、アップグレードに費用がかかります。



「概して、全体的な問題は、ストレージシステムが、開発および近代化戦略なしで、デバイス、さらには耐用年数の限られたデバイスであると見なされることです。 デバイスではなく、ストレージプラットフォームが必要です。 デバイスとしてのストレージシステムへのアプローチは、破滅的なストレージモデルにつながります。 近代化は、1世代のストレージであっても不可能な場合があります。 「ソフトウェアとハ​​ードウェアが相互に抽象化されているストレージプラットフォームに切り替えると、現在の財務モデルを大幅に改善できます。」




しかし、ソフトウェア定義のストレージシステムは、ストレージシステムの従来の財務モデルの問題を解決できますか？







多くのお客様は、特殊なストレージデバイス（アプライアンス）から標準装備の純粋なソフトウェアソリューションに移行することを夢見ています。 しかし、ソフトウェアベンダーに尋ねられます。彼は自分のソフトウェアにどのハードウェアを推奨していますか、テストされていますか、ハードウェアプラットフォームのサポートはありますか、ハードウェアとソフトウェアを統合することは可能ですか？ これらの質問に対する答えはアプライアンスのみです。 円が閉じます。



ストレージシステムの信頼性の高い動作のためには、ストレージシステムのすべてのコンポーネント（容量、計算機、ロジック）が、単一のサポートを備えた単一のデバイスで互いに密接にはんだ付けされていることが非常に重要です。 問題の解決策として、DellはCompellentストレージシステムを提供しています。 それらのコントローラーは近代化されています-さらに、外出先で、そしてライセンスの移動で、次世代に進むことができます。 異なる世代のシェルフ（FC、SAS6、SAS12）を1つのシステムに結合することができ、新しいメディアへのデータの段階的な移行がサポートされています。 これまでに販売されたCompellentストレージシステムの95％が依然としてアクティブなサービスサポートを受けていることは偶然ではありません。



このような経済モデルは、ソフトウェアストレージシステムに対応し、ソフトウェアは一度購入すると、ソフトウェアとハ​​ードウェアの統一サポートが提供されます。 さらに、データセンター間でのデータ移行でも、バーチャライザーは必要ありません。 この場合、SDSは必要ですか？







ただし、Dellはこの傾向を無視せず、従来のデバイスとともに、特にDell XCシリーズソリューションなど、ソフトウェア定義のストレージシステムの広範なポートフォリオを顧客に提供しています。 フォーラムの参加者は、ソリューションの展示会で彼らを知ることができました。

データセンター、モバイルデバイス、企業ネットワークに個人用デバイスを実装するためのアプリケーション、クライアントシステム、ITインフラストラクチャの集中管理用ソフトウェアソリューション、包括的なデータ保護と災害復旧、サーバーおよびハイパーコンバージドインフラストラクチャソリューション、デスクトップ仮想化ツール。







展示会の訪問者は、新しい第13世代のDell PowerEdgeサーバー、フラッシュドライブを備えたSC9000企業ストレージシステム、Dell Storage Center 6.7ソフトウェア、ネットワーク機器、ラップトップ、タブレット、シンクライアント、および他の多くの製品とソリューションに精通することができました。



デニス・トロピンとアンドレイ・コマロフスキーは「デモゾーンのツアー」を実施し、各スタンドで立ち止まり、そこで行われた内容について話しました。



デルのソリューションに基づいたシミュレータエンジニアリングおよび人事訓練センターのインタラクティブゾーンでは、宇宙教育センターで使用される特殊なソフトウェアのプロトタイプをテストして、宇宙船を手動でドッキングした宇宙飛行士のように感じることができました。







ビッグデータのブースは、ビッグデータおよび分析ツールを操作するためのデルの既製ソリューション専用でした。 Fedor Pavlovは、特定のワークロードを対象とし、柔軟なモジュールアーキテクチャに基づいたデータストレージソリューションを開発するための戦略について話しました。 訪問者は、新しいDell Storage SCストレージシステムの利点も確認しました。







ビッグデータのデモスタンドでは、探査用の統合ソリューションも紹介されました。 これは、Dellサーバー、ストレージ、およびクライアントシステムに基づいています。 デルのシステムは、高性能コンピューティングシステムを構築し、要求の厳しい3Dグラフィックアプリケーションの要求を満たすために使用されます。 専門家は、ビッグデータと分析Dell Statistica 13.0を操作するためのプラットフォームの新バージョンの機能についても話しました。



モビリティソリューションは、従業員のモビリティソリューションを紹介しました。 ここでは、Latitude Ruggedの堅牢なラップトップの強度テストに参加したり、Dell Precisionモバイルワークステーションで可能なことや、Dell Endpoint Systems Managementツールを使用して1つのコンソールから多数のモバイルデバイスを管理したり、Dellラップトップをテストしたりできます、タブレット、オールインワン、コンパクトなデスクトップシステム。



セキュリティデモゾーンでは、ゲートウェイからデータベースまで、ネットワークデバイスからアプリケーションまで、ファイアウォールから企業データまで、さまざまなレベルでセキュリティを提供するDell統合ソリューションが示されました。 単一のアプリケーションを使用して仮想マシン、物理サーバー、クラウドのワークロードを保護するアクセス制御と管理のソリューション、およびAppAsureソフトウェアに基づくデバイスを使用して重要なデータをバックアップ、複製、復元する機能が訪問者に提示されました。







さらに、展示会では、最新のDell Networkingネットワーク機器、モジュラースイッチ、Open Networkingソリューションを見ることができました。 現在、デルはオープンネットワークの開発におけるリーダーの1つになり、自社ブランドのスイッチを提供しています。このスイッチは、自社のネットワークオペレーティングシステムの制御下で動作するだけでなく、Cumulus Networks、Big Switch Networks、Pluribus Networksなどの他のメーカーのソフトウェアも制御できます。



Edge Gateway 5000とStatistica分析プラットフォームを組み合わせることで、ネットワークエッジでデータの前処理を整理し、モノのインターネットテクノロジーを使用するシステムの通信チャネルを最適化できます。



クラウドゾーンの目標は、オープンスタンダードに基づくデルのクラウドソリューションがデータセンターインフラストラクチャを最適化する方法を示すことです。 ここで、訪問者はクラウドテクノロジーの実装のためにインフラストラクチャを準備する方法、適切なクラウドプラットフォームを選択する方法、どのハードウェアおよびソフトウェアコンポーネントを優先するか、クラウドインフラストラクチャを構築するためのデルとパートナーの共同ソリューションに精通することができます。



IBS Platformixブースでは、ビデオストリーミングの最適化を備えたDell vWorkspace 8.6に基づくVDIソリューションが展開されました。 システムインテグレーターIBS Platformixは、監査、ソリューションの設計と開発、実装、サポート、サポートなど、現代の企業におけるインフラストラクチャ統合に関連する幅広いサービスを顧客に提供します。
