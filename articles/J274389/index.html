新しいInfoboxCloudクラウドリージョン：サンクトペテルブルク！


2015年、サンクトペテルブルクの自社データセンターにロシアのクラウドインフラストラクチャInfoboxCloudの新しい地域を立ち上げました！ エンタープライズクラスのクラウドは、モスクワとアムステルダムでも利用できます。 ロシアで個人データを保存するための要件を遵守しながら、待機時間が最も短いデータセンターを選択できます。 世界のさまざまな地域でInfoboxCloudの存在を拡大するための戦略を体系的に実施しています。







InfoboxCloudの主要なユーザー ：独自のITインフラストラクチャをクラウドに移行するビジネス、または独自のITインフラストラクチャをクラウド、開発者、Webスタジオ、およびそのインテグレータとともに使用するビジネス。



新しい地域は、以下を含むInfoboxCloudのすべてのメリットを受け取りました。

  • SLA 99.99％ 。 ロシア連邦の法律に準拠した財政的義務と報告を伴う法的保証。
  • 柔軟性 。 必要な構成のクラウドサーバーを作成し、パラメーター（コア数とプロセッサー周波数、メモリーサイズ、ディスクサイズ）を互いに独立して設定し、適切なタイミングでパラメーターを変更するか、自動スケーリングを使用できます。
  • 耐障害性 。 すべてのクラウドサーバーは、単一のハードウェア障害点のないクラスターで動作します。 クラスターソフトウェアは同時に更新されないため、ソフトウェアエラーから保護できます。
  • 信頼性 データのトリプルレプリケーションは 、分散フォールトトレラントストレージシステムCloud Storageで提供されます。さらに、スケジュールされたバックアップを有効にできます。
  • ディスクサブシステムパフォーマンス 。 ディスクサブシステムを高速化するために、PCI – Express SSDおよびDatacenterレベルのSSDでのエンタープライズSSDキャッシングが使用され、すべてのユーザーが追加料金なしで利用できます。
  • ネットワークパフォーマンス 。 クラウドサーバーで外部通信チャネルのパフォーマンス制限を1 Gb / sに設定できます
  • RESTful API 。 コントロールパネルからだけでなく、ソフトウェアからもクラウドインフラストラクチャを管理できます。
  • 2つのタイプの仮想化 -コンテナーと仮想マシン。最も経済的なソリューションとOSカーネルを制御する機能の両方を提供します。
  • WindowsおよびLinux （Windows Server 2012 R2標準およびWeb、Windows Server 2008 R2、Ubuntu 14.04 LTS、Debian 8、CentOS 7および6、Oracle Linux 7、Suse 13などを含む）。
  • IIS、Asp.NET、PHP、SQL Serverが事前に構成されたWindows サーバー 。 Linuxの場合-LAMPなど。
  • そしてもちろん、専門家はロシア語でInfoboxをサポートし、24時間年中無休で働いています

Infoboxからの新しいVPS料金表（U、C、R）。 非標準タスクに最適


2015年に、ユーザーの要望を考慮してVPSの料金表を更新しました 。 ユニバーサルバランスサーバーだけでなく、リソースの量が増加したサーバーも利用できるようになりました。



  • ユニバーサルコンピューティング（U）のVPSは 、ほとんどのタスクに最適です。
  • プロセッサリソースが増加したVPS（C）により、データの迅速な分析、計算の実行、アプリケーションのコンパイルが可能になります。
  • メモリリソース（R）が増加したVPSでは 、RAMにより多くのデータを保存できます。これは、負荷の高いサイトや最新のデータベースにとって重要です。




VPSごとに、コントロールパネルからディスク容量を増やすことができます 。

固定チャンネル速度で無制限のトラフィックでVPSを注文するか、無制限のネットワーク速度で月に3 TBのトラフィックでVPSを注文することもできます。
