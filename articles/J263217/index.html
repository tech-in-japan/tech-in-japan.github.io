Visual Studio 2015および.NET 4.6リリース

Visual Studio 2015および.NET 4.6 がダウンロードできることをお知らせします 。 これら2つのリリースは大きな前進であり、開発者に生産性を向上させ、複数のプラットフォーム用のさまざまなアプリケーションの作成を可能にする新しいツールを提供します。 既にVisual Studio 2015をダウンロードし、 60以上のビデオを使用してこのツールキットの機能を確認できます 。











Visual Studio製品ライン





過去数年にわたって、Visual Studio製品ラインは大幅に拡張され、多くの新機能が追加されました。

最も重要なコンポーネントはVisual Studio IDEです。これは世界中の何百万人もの開発者によって使用され、このツールを使用して優れたアプリケーションを作成します。 昨年、 Visual Studio Communityの完全に機能する無料バージョンのVisual Studioを発表しました。 昨年11月以来、500万回以上ダウンロードされています。

2年前、 Visual Studio Onlineをリリースしました。これは、プロジェクトのバージョン管理、タスク管理、アセンブリ、展開を可能にする開発プロセスを整理するためのクラウドベースのサービスです。 現在、320万人以上の開発者がVisual Studio Onlineを使用しています。

今年初め、 Visual Studio Codeを導入しました。これは、Webおよびクラウド向けのクロスプラットフォーム開発をサポートするMac、Linux、およびWindows用のコードエディターです。 過去3か月の間に、VS Codeは500,000回以上ダウンロードしており、ダウンロードの半分以上はMacおよびLinuxコンピューターからのものです。

Visual Studio 2015および.NET 4.6





本日のVisual Studio 2015および.NET 4.6のリリースは、開発者に何百もの新しい機会を提供し、デスクトップ、Web、モバイル、クラウドアプリケーションなどを作成できるようにします。

新機能の説明は、 Visual Studioリリースノート 、 Visual Studio ブログ 、 .NETブログ 、およびScott Guthrieのブログに公開されています 。



生産性革新





開発者は、ほとんどの時間をVisual Studioを使用して費やしています。 したがって、私たちの使命は、日常業務を促進し、現在の問題を迅速かつ簡単に解決できるようにする革新をこのツールにもたらすことです。 Visual Studio 2015には、デバッグ、診断、コード編集、リファクタリングに関連するものなど、生産性に関連する多くの改善点があります。



デバッグと診断




Visual Studio 2015では、デバッグとプロファイリングが単一の診断ハブに統合されているため、コンテキストデバッグモードでアプリケーションの正しい実装とパフォーマンスパラメーターを確認できます。 これに加えて、実装された機能のパフォーマンスに関する情報が常に手元にあるように、 Perf Tipsの形式でヒントを実装しました。







コードの編集とデバッグ




新しいC＃およびVBコードエディターでは、開発者は電球を含むRoslynテクノロジーに基づいた新しいツールの全機能を使用できます。これにより、コードセクションの便利なソリューションを求め、リファクタリングを支援します。 Roslynとコードアナライザーのおかげで、開発者はアラートとプロンプトをエディターで直接変更およびカスタマイズできます。







XAMLエディターを再設計し、Peekを介したコードのクイックナビゲーションなど、多くの新機能を追加しました。







プログラミング言語




Visual Studio 2015および.NET 4.6のリリースに伴い、C＃プログラミング言語をバージョン6に、Visual Basicをバージョン14に更新しています。 コーディングパターンを簡素化する多くの改善と革新により、C＃とVB.NETの新しいバージョンは、開発の生産性の観点から大きな前進です。

これに加えて、Visual Studio 2015には、C ++ 11/14/17、TypeScript 1.5、Python用のF＃4.0標準ツール、およびその他の機能の広範なサポートが含まれています。







Web、クラウド、モバイル





どのような開発を行っても、Web、クラウド、モバイル、柔軟性と選択が重要です。 Visual Studioと.NETはそのような選択肢を提供し、さまざまなプラットフォームとプログラミング言語の使用を可能にします。

過去2年にわたり、 Roslyn 、 TypeScript 、 CoreCLR、 Python Tools for Visual Studioなど、多くのコンポーネントとプラットフォームのソースコードを公開しています 。 さらに、Visual Studioでは、オープンテクノロジーとプラットフォームの使用がはるかに簡単になりました。



Webおよびクラウド開発




.NET 4.6と共に、Web上のアプリケーション用のプラットフォームの新しいバージョンであるASP.NET 4.6が利用可能になりました。 ASP.NET 4.6は、HTTP / 2およびEntity Framework 6.1.3を含む新しいC＃機能をサポートします。 Visual Studio Web Toolkitには、標準サポートと新しいJSONエディターを備えたHTMLおよびCSSの高度なサポートが含まれています。

Visual Studio 2015にはAzure SDKが含まれており、クラウドアプリケーションをすばやく構築できます。 IaaSやAzureのPaasなど、アプリケーションで使用するモデルが何であれ、.NET、Python、Node.jsを使用してこれらのアプリケーションを作成し、Windows、Linuxにデプロイするか、Dockerコンテナーを使用することができます。

これに加えて、 ASP.NET 5ベータ版を試す機会があります 。 ASP.NET 5には、以前のバージョンに対する多くの変更と改善が含まれています。 ASP.NET 5はCoreCLRの上で実行されるため、将来的には、収集したアプリケーションをLinux、Mac、Windowsなどのさまざまなプラットフォームで使用できます。





モバイル開発




モバイル開発とは、さまざまなデバイスでアプリケーションを実行できることを意味します。 Visual Studio 2015には、iOS、Android、およびWindows用のアプリケーションを作成し、開発者がプロ​​グラミング言語とプラットフォームを選択できるクロスプラットフォーム開発ツールが含まれています。



多くの開発者は、最新のHTML、CSS、およびJavaScript機能を使用して、レスポンシブインターフェイスを備えたWebサイトのモバイルバージョンを作成することから始めました。 Visual Studio 2015には、リッチなWeb開発ツールが含まれています。これにより、モバイルアプリケーションを作成したり、最新のユーティリティやライブラリをデバッグおよび使用したりできます。

Apache Cordovaのツールのおかげで、カメラなどの特定のデバイス機能にアクセスしながら、Webアプリケーションをすばやくモバイルアプリケーションに変えることができます。 Visual Studio 2015には、アプリケーションの機能をテストできるAndroid用の強力なVisual Studioエミュレーターが含まれています。





.NETプラットフォームの豊富な機能を好む開発者向けに、Visual Studio 2015では、デスクトップ、タブレット、電話、IoTデバイス、さらにはHoloLensでも実行できるユニバーサルWindowsアプリケーションを作成できます。 Visual Studio 2015とXamarinの機能の組み合わせのおかげで、開発者は.NETプラットフォームへの投資を節約し、Windows用の既存のコードを使用してiOSおよびAndroid用のアプリケーションを作成できます。

さらに、Visual Studio 2015はC ++開発用のクロスプラットフォーム機能を提供し、iOS、Android、Windows間でコードを共有したり、Unity統合機能を使用してクロスプラットフォームゲームを作成したりできます。





DevOpsとアジャイル開発





現在の開発チームは現在、柔軟性を重視しています。 これに加えて、現在の傾向にはDevOpsの実践に対するサポートが必要です。 Visual Studio 2015、Team Foundation Server 2015、およびVisual Studio Onlineは、DevOpsプラクティスを実装し、プロジェクトの計画時にチームが柔軟な方法論を使用できるようにする統合ツールセットを提供します。

DevOpsの主な機能は、Team Foundation Server 2015（RC2が利用可能、リリースは後日）およびVisual Studio Onlineクラウド環境に含まれています。





Team Foundation Serverは、長年にわたり.NETチームにとって優れたツールでしたが、過去2年間にわたってツールの開発に多大な労力を費やし、Node.js、Java、iOS、Androidなどの他のプラットフォーム向けのソリューションを作成できるようになりました。



計画中




Team Foundation ServerおよびVisual Studio Onlineの柔軟な計画ボードは、チーム作業の編成と進捗の追跡に不可欠なコンポーネントです。 Team Foundation Server 2015には、このツールの外観をカスタマイズし、日常のタスクを簡素化できる新機能が含まれています。







開発とテスト




Team Foundation Server 2015およびVisual Studio Onlineでは、Gitバージョン管理システムに格納されたコードのプルリクエストが許可されるようになり、チームが使い慣れたコラボレーションパラダイムを整理できるようになりました。







リリースのビルドとリリース




Team Foundation Server 2015およびVisual Studio Onlineには、プロジェクトを構築するための新しい自動化ツールが含まれており、ソリューションをコンパイルおよび公開するためのクロスプラットフォームソリューションを柔軟に構成できます。







モニタリング




高品質のコードを作成し、運用環境でアセンブルされたソリューションを展開するだけでなく、既に実行中のアプリケーションを監視することも非常に重要です。 Visual Studio 2015は、クライアントソリューションとサーバーソリューションの両方で、アプリケーションとApplication Insightsテクノロジを迅速に統合し、パフォーマンス、可用性、およびその他の基準を分析できるようにします。








Visual Studioパートナーエコシステム





Visual Studioは間違いなく素晴らしい製品ですが、アドオンを作成するパートナーのエコシステムはさらに強力で便利になります。 過去1年間で、Visual Studio GalleryのWebサイトから1,300万を超えるさまざまなアドオンのダウンロードが行われました。

現在、84社以上のパートナーがVisual Studio 2015の拡張機能を既に作成しています。拡張機能作成システムの改善により、Visual Studioのプラグインの作成がはるかに簡単になりました 。





おわりに



Visual Studio 2015と.NET 4.6は、開発における大きな前進です。 革新性と生産性を組み合わせたこのツールキットにより、最も幅広いソリューションを作成できます。

必ず用意されているビデオをご覧になり、 Visual Studio 2015を今すぐダウンロードしてください 。

Visual Studioの予備リリースのテストに参加したすべての人に感謝します。この製品の開発への貢献は非常に重要です。
