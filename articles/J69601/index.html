Away3d Universe-エピソード1：インストールとセットアップ

ご存知のように、各フラッシャーは3Dでフラッシュサイト、ゲーム、および実験を作成する必要があります:)これで、あなたの手は3Dに到達しました。 「フラッシュには3Dデモはありません、アプリケーションを提供します」というスローガンが好きでした。したがって、この一連の記事（レッスン、チュートリアル-好きなものを呼んでください）で好きなものに導かれますフラッシュの3Dゲーム。

ここに記載されている例を使用するには、SDKバージョン3.2以降のFlex Builder 3が必要です。 もちろん、Away3dはFlash CS3 / 4で使用できますが、開発者にとってFlex Builder 3ははるかに便利なようです。

ツールを決定したら、Away3dを使用してプロジェクトの環境を設定します。

まず、svnからFlash Player 10用のAway3dライブラリを解凍する必要があります-http : //away3d.googlecode.com/svn/trunk/fp10/Away3D/、away3dFP10フォルダーに。 svnから取得するのが面倒な人は、こちらからアーカイブをダウンロードできます。 少し余談ですが、レンダリングの増加が3分の1増加するため、Flash Player 10を使用してコンパイルします（10mプレーヤーでは、ビットマップはプログラムコードではなくプレーヤーレベルで既にラスタライズされています）。

その後、Flex Builder 3を開き、新しいFlex Libraryプロジェクト「Away3dFP10」を作成します。





追加した後、9番目のFlash Playerでプロジェクトがデフォルトでビルドされるという事実により、ビルダーがビルドエラーの全リストを発行することがわかりました。 10番目のFlash Playerのコンパイルモードを変更します。 したがって、最初に行うことは、ライブラリプロジェクトのプロパティに移動し、Flex Library Build Pathカテゴリを開きます。



playerglobar.swcを削除してから、cにある別のplayerglobar.swcを追加する必要がある場所：\ Program Files \ Adob​​e \ Flex Builder 3 \ sdks \ 3.4.0 \ frameworks \ libs \ player \ 10 \ playerglobal.swc：



新しく追加されたplayerglobal.swcのLink TypeパラメーターをExternalに変更する必要があります。



次に、コンパイルパラメーター-target-player = 10.0.0.0を追加する必要があります。



これで、Away3dのインストールが完了しました。 次のパートでは、3次元のシーンを含む最初のプロジェクトを作成します。
