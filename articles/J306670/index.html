前向きな開発ユーザーグループのピクニック：誰にとっても安全な開発





Positive Technologiesは、 開発者やその他のIT専門家向けの非公式の非公式会議であるPositive Development User Group Picnicを開催しています。 その上で、安全に開発する方法を学ぼうとしている人、または安全なアプリケーションを作成した経験のある人に会えることを嬉しく思います。



会議の技術プログラムは、私たちの積極的な講演者の報告によって作成されます。



  • 銀行システムのセキュリティ部門の責任者であるティムール・ユヌソフは、ハッカーがWebアプリケーションをクラックし、よくある間違いを指摘し、興味深い事例を共有する方法について話します。
  • ソースコード分析システムApplication Inspectorの開発グループの責任者であるVladimir Kochetkovは、安全な開発のためのアプリケーションと実用的な技術を保護する方法を紹介します。
  • Application Inspector SSDL Editionソースコード分析システム開発グループの責任者であるValery Boroninが、セキュアソフトウェア開発ライフサイクルの機能と利点について説明します。


さらに、プログラムにはいくつかの驚きがあります。 最近のhabtopopik Volodya Kochetkovaは、彼らが何に関連しているかを推測するのに役立ちます:)



技術セクションの後、参加者はバーベキュー、ビール、そして経験を共有したり、志を同じくする人々とチャットしたりする楽しい部分を見つけます。



セミナーは8月10日15:30に開催されます。 参加は無料ですが、事前登録が必要です。 急いで、席の数は限られています！ 登録を確認した後、参加者に住所を通知します。
