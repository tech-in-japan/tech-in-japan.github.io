IronPython 2.7 Alphaリリース





IronRubyの更新に続いて、 .NETのPython言語であるIronPythonの実装も更新されました。 IronPython 2.7 Alphaの新しいバージョンでは、 CPython 2.7の予備サポートが追加され、その他の変更が行われています。
  • Visual Studio （ Visual Studioスイート用IronPythonツール）との統合を追加。
  • 役に立つIronPython関連情報を含むCPython 2.7ドキュメントを拡張。
  • 追加されたモジュールmmapとsignal ;
  • 多くのパフォーマンスの改善とバグ修正が行われました。
  • ライセンスがMSPLからApacheライセンスバージョン2.0に変更されました。
  • 新しいバージョンには、 .NET 4.0およびSilverlight 4.0が必要です。
