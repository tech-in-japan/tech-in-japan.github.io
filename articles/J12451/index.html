OLEDディスプレイ付き電子時計

中国に本拠を置く深ShenzhenCailiang Electronics がADX18-Aフル電子時計を発売





それは珍しいことではないように思われます-典型的なクォーツ時計、およびフラッシュプレーヤー付きの時計はすでにかなりたくさんあります。 しかし、このデバイスの主な違いは、OLEDディスプレイをダイヤルとして使用することです。





ガジェットは、いくつかの形式で音楽を再生し、画像（ビデオを含む）を表示できます。



仕様：

ディスプレイ： 1.5インチ、260K OLEDトゥルーカラー、解像度128 * 128ピクセル。

時刻： 12および24時間形式、M / D / YからY / M / Dの形式の日付

フラッシュメモリ：内蔵、1または2Gb

サポートされるメディア形式： MP3、WMA、MP4 SMV

画像表示オプション： JPEG、アイコン表示モード、スライドショーを手動でサポートします。

録音：デジタルサウンド録音、形式は256〜320 Kps、WMA：32Kbps-380Kbpsで異なります。

出力周波数： 20Hz-20KHz、SNR：> = 88dB

5つのイコライザーモード、スーパーベース3Dサウンドエフェクト

システムメニューの9言語（ロシア語なし）

電源：内蔵リチウムイオン電池、USB経由またはACアダプタからの充電（付属）

充電時間：約1.5時間

稼働時間：音楽の再生モードで8時まで。

インターフェース： USB 1.1、USB 2.0、USB Plug＆Playをサポート

追加用のオペレーティングシステム。 ソフトウェア： Windows 98 / Me / NT / 2000 / XP以上

電子書籍の閲覧をサポートします。

ステンレススチールケース、レザーストラップ

ヘッドフォン、USBケーブル、ACアダプター、ドライバーディスクが付属



高級バージョンの時計のコスト： 99ドル （2Gbのメモリを備えたオプション）または89ドル （1Gb）。 メーカーから直接、わずか30ドルで予算オプションを購入できます 。 ある小規模な調査によると、中国人はすでにそのような時計のいくつかのバリエーションを生産しており、それらの多くはEbayで販売されています。



本を読んでビデオを見るのが良いかどうかはわかりませんが、おそらく手首のオーディオプレーヤーとして機能し、時計のように見えます。 もちろん、あなたが中国の時計に対して何もない場合。
