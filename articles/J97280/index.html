Internet Explorer 9プラットフォームプレビュー3

今日、IE9開発チームは、MIX10会議での約束に従って、ブラウザの新しいバージョンの更新プログラムを公開するため、次の更新プログラムを公開しました。



IE9 TestDrive WebサイトからIE9 Platform Previewをダウンロードします 。



新しいバージョンでは、HTML5オーディオ、ビデオ、およびキャンバスタグ（ハードウェアアクセラレーション付き）、Web Open Font Format（WOFF）形式のサポート、および作業速度の向上とバグの修正におけるその他の多くの改善が利用可能になりました。



新しいデモも追加されました。







テストセンターでの118の新しいテスト+ ES5の追加の1309テスト。



ACID3は83/100を達成します。 JavaScript の速度も向上しました 。







チャンネル9でさらに利用可能なビデオ：

  • Internet Explorer 9プラットフォームプレビュー3：新しいデモを見る
  • Internet Explorer 9プラットフォームプレビュー3：ECMAScript 5の概要
  • Internet Explorer 9プラットフォームプレビュー3：キャンバスの概要
  • Internet Explorer 9プラットフォームプレビュー3：HTML5ビデオサポートの概要
  • Internet Explorer 9プラットフォームプレビュー3：CanvasとAmazon.comの概要




IE開発チームの新しいブログ投稿もご覧ください。



いくつかの統計：発行の開始以来、IE9 Platform Previewは既に200万回以上ダウンロードされています。
