MicrosoftはSkypeを支持してLive Messengerを辞任する





Microsoftは、Skypeを支持してWindows Live Messengerクライアントを辞任しようとしています。 Vergeは、数か月以内にWindows Live Messengerが無効になり、Skypeと統合されることをいくつかのソースから学びました。



過去数か月にわたり、MicrosoftはSkypeをLive Messengerのサーバー機能に徐々に移行してきました。 Skypeを介して送信されるすべてのメッセージの約80％は、メッセンジャーバックエンドによって処理されます。





情報源によると、同社はおそらく今週中に、Windows Live Messengerの切断を間もなく発表する予定です。

Windows Live Messengerの世界的な放棄と比較して、より収益性の高い動きは、MicrosoftアカウントをSkypeのユーザー名とパスワードにリンクすることです。 MacおよびWindows用のSkypeユーザーバージョン6.0は、MicrosoftまたはFacebookアカウントを使用してログインするオプションを既に提供されています。



VergeはMicrosoftにコメントを求めました。おそらく何かが明らかになるでしょう。



UPD： 噂が確認されました ：「今日、Skypeのブログで、すべてのメッセンジャーユーザーが最新バージョンのSkypeをダウンロードして更新することを奨励していることを発表しました。 今後数か月以内に、中国本土を除く世界中のすべての地域のメッセンジャーアプリを廃止します。 "-"本日、Skypeブログで、すべてのメッセンジャーユーザーに最新バージョンのSkypeをダウンロードして更新することをお勧めします。 今後数か月以内に、中国を除くすべての国でメッセンジャーアプリを無効にします。」
