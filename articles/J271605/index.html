EOIPとDD-WRTで2つのアパートメントを1つのネットワークに結合するか、すべてをリモートプレイに使用

背景：



実際、タスク1とハウス2を1つのネットワークに統合し、ウォーキングブロードキャストとマルチキャストを行います。

目標は、PlayStation TVをPlayStation 4に接続し（メインアカウントなし）、ローカルでリモートプレイを最大限に活用することです。



サービスには、次のアイロンがあります。



家1：

Dlink dir615c2

家2：

Dlink dir615E4



準備、解決策：



両方のルーターに、dd-wrtのファームウェアがインストールされました。 インストール手順は簡単です。このトピックに関する多くの情報がインターネット上にあります。



EOIPを機能させるには、両側に「白い」IPアドレスが必要です。これは、プロバイダーから100 +ルーブルで購入します。 これが不可能な場合は、 vpnトンネル （前の記事で説明しました）を転送し、その上でEoIPを実行できます。



両側のユーザーがネットワーク環境でお互いを見るためには、同じローカルネットワークに属している必要があります。 次の設定を使用しました。



ネットワーク：192.168.100.0/24

家1：ルーター（192.168.100.10）には192.168.100.1-192.168.100.50のプールがあります。

家2：ルーター（192.168.100.70）には192.168.100.51-192.168.100.100のプールがあります



ハードウェアにdd-wrtをインストールし、各DHCPで構成し、クライアントを接続し、クライアントの作業を確認した後、EOIPトンネルの作成に進みます。



これを行うには、Webにアクセスします。 ルーターインターフェイス、[設定] -> [EoIPトンネル ]タブに移動します。

EoIPトンネルを有効に 設定します。

[ リモートIPアドレス ]列に、接続する必要があるルーターの外部IPを入力します（VPNの場合は、VPNトンネルでIPを指定します）。

有効にする ブリッジング ;

*必須。トンネル番号は両方のルーターで一致する必要があります。







[ 管理 ] -> [コマンド ]タブに移動して、ルーターのWebインターフェイスで結果を確認できます。

[ コマンド ]フィールドにifconfigと記述し、[ コマンドを実行 ]をクリックします 。 その後、上げられたインターフェースが与えられ、その中にoet1があります



これで、ネットワーク上の共有フォルダーを表示できるだけでなく、電話からテレビへのビデオを別のアパートに送信できるEOIPトンネルができました。
