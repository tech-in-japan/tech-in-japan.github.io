UMA CMS：開発

こんにちは。 通常、Habrahabrで彼らは何が行われたかについて書きますが、私はそれを行うことを提案します。 アイデアが良いことを願っていますが、何らかの形で十分な批判を待っています。

UMA CMS

超モバイルの高度なコンテンツ管理システム-非常に小さい（約100 kB）

最新の技術的工夫

そして、電話から書く能力


使用例：

あなたはセミナー、会議に来て、たくさんのエキサイティングなことを見て、あなたはあなたのニュースの事実と一緒に、燃える混合物になるという意見を持っています。 携帯電話またはPDA、数クリック、数百万キロの思考が得られ、スタイラスやキーボードの指からの熱の痕跡はまだ冷めていません。


開発に必要：

Javaナレッジ（J2MEの紹介）

PHP 5の知識

Ajaxの一般原則の知識


開発\配布：私が持っているソースをつなぎ合わせることに減らされた、共同のポイントごとの開発。 Sourceforge + Google Code + Habrahabrのトピックによる分析ITメディアとしての配布

要点：CMSは、オンザゴーの投稿アプリケーションで配信され、数秒で（適切な構成で）ホスティングに展開され、最大の機能を提供します。
