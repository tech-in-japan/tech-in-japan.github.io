私は、Web 2.0の動きがインターネット上の匿名性のほぼ完全な消失をもたらすと確信しています。 そして、ここに理由があります。



ジャンルを選択してください



最初に、それが本当に意味するもの、つまりWeb 2.0を見つけましょう。 流行語は、特定の原則と技術的手法の基本セットを隠す魅力的なラッパーです。 Web 2.0のほとんどは、ブロゴスフィアの人気の高まりを理解しています。 ソーシャルネットワークの開発。 タグの発明と製品の作成に対するエンドユーザーの貢献。 これは、無料のネットワーキングおよび友人との無料の交換です。 このような説明は、ポイントに非常に近いと見なすことができます。 正確な定義は偉大な人にも与えられていません。



例を理解してみましょう。 最も成功しているのはDigg.comで、これまでに挙げた多くの側面を組み合わせています。 従来の情報リソースとは異なり、ここでのニュースは参加者自身によって書かれています。 ユーザーが作成したメモと、彼が気に入ったメモは、個人プロフィールにリンクされています。 ユーザープロファイルは友人のプロファイルにリンクされているため、グループのメンバーが互いに推奨するテキストを表示できます。 これに優れたソフトウェアとブログ機能を追加します。



または、強力なソーシャルネットワークであるMySpace（現在はRupert Murdoch Corporationが所有）。 そのユーザーは、リンクファイルシステムのおかげで、音楽ファイル、思考、写真、言葉を交換したり、新しい人と出会ったり、新しい興味を見つけたりすることができます。 友人がスーパーアークティックギターのA Nirvana Marsupialsのいくつかを愛しているとMySpaceが言った場合、同じことが好きかもしれません。



Web 2.0は、オンラインビジネスの進化も意味します。 Googleは検索エンジンから広告プラットフォームに変わりつつあります。 インスタントメッセージングシステムに取って代わる電子メールは、主にビジネスツールになりました。 Ajaxのカウンターパートが出現したOutlookとOfficeは、ますます冗長になっています。 おそらく、すべてをオンラインで実行する能力は、これまでそれほど重要ではありませんでした。



ユーザーのメリット



どうして？ 「動き」が大きくなるほど、ネットワークに影響を与える変更の幅が広くなり、ユーザーにとっては良いでしょうか？ Diggはすでに伝統的な出版ビジネスを危険にさらしています-そのメンバーは、彼ら自身の人口統計学的基準に一致する情報を見つけることでうまくやっています。 リソースの人気は、これの確認です。 MySpaceのおかげで、どこにいても思春期の人々は、頑張って親の世話をバイパスして、最善を尽くすことができます。つまり、ぶらぶらし、異性との気まずいコミュニケーションを確立し、最新の「チップ」と音楽ノベルティについて意見を交換します。



Flickrは、何百万人ものユーザーに無料の写真ホスティングを提供しています。 技術的な実装の成功により、写真の共有がこれまで以上に簡単になりました。 Larry LessigによるCreative Commonsライセンスの使用は、Flickrが他の懐疑的なハッカーやアマチュアコーダーを引き付ける何らかの形で「反対」であるという評判があることを示しています。 Google MailとCalendarの登場により、 Outlookは多くの人にとって不要になりました。必要なものがすべてオンラインで無料で利用できるのに、なぜMicrosoftに支払うのですか？ 世界中のネットワークの成長とモバイルインターネットの発展を考えると、接続なしで放置されることはほとんどありません。



Web 2.0サイトは、優れた機能と最高のオンラインエンターテイメントを提供します。 それは完璧ではありませんか？
