最速のIntel Core

私たちは、液体冷却システムのマニュアルから何世紀ものほこりを吹き飛ばし、最もファッショナブルなケースと最も高級なビデオカードを購入します！ Intelは、史上最も強力なPCプロセッサであるIntel Core i7-5960 Extreme Editionをリリースしました。 EEインデックスを備えたすべての製品と同様に、この新しいプロセッサは、極端なオーバークロックのファンと、新しいコアとその容赦ない動作の可能性に憧れるゲーマー向けに設計されています。



第4世代Intel Core i7-5960Xプロセッサー （コードネームHaswell-E）は8コア（16スレッド）で、3 GHz（ターボモードで3.5 GHz）の周波数で動作します。 おそらく、コアが要因によって壊れていると言うべきではありません。 一般に、オーバークロック機能は非常に印象的です。エンジニアリングサンプルの予備テスト中に、コミュニティでTaPaKaHとして知られるロシアのオーバークロック装置は6.25 GHzの周波数に達しました。



トップモデルとともに、ロック解除された乗算器を備えた「単純に強力な」6コアi7プロセッサーが視聴者に提示されました： 5930Kおよび5820K （それぞれ周波数3.5 GHzおよび3.3 GHz）。 発表されたすべてのモデルは、Thunderbolt 2およびDDR4をサポートする新しいIntel X99チップセットで動作します。 LGA2011v3フォームファクターで実行されます。

Intel Core i7-5960Xプロセッサのコストは、999ドル、5930K-583ドル、5820K-389ドルになります。 すべてのプロセッサはロシアですでに購入可能です-悲しいかな、それはもっと高価です。
