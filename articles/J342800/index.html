YouTrack 2017.4リリース：勤怠集計レポート、マークダウンサポートなど

みなさんこんにちは！ 最近、プロジェクト管理システムの新しいバージョンであるYouTrack 2017.4をリリースしました。お客様とイノベーションを共有するために急いでいます。











YouTrack 2017.4の新機能







  • 勤怠集計レポート。
  • 検索結果の関連性で並べ替えます。
  • カスタム日時フィールド。
  • JavaScriptワークフローをサポートするためのnpmパッケージ。
  • 日本語でのローカライズ。


他に何？







  • タスクフィールドのテキストインデックス。
  • テキストインデックスでの追加言語のサポート。
  • Microsoft ExchangeおよびOffice 365での共有メールボックスのサポート。
  • 電子メール通知を無効にする機能。
  • 実験的機能の一部としてのマークダウンのサポート。


勤怠集計レポート



1つまたは複数のプロジェクトの勤怠集計を追跡できる新しいレポートが追加されました。 新しいレポートでは、タスクはプロジェクト、エグゼキューター、サブシステム、スプリントごとにグループ化できます。

















テキスト検索結果の関連性で並べ替え



現在、タスクによるテキスト検索では、結果は関連性によってソートされています。 これは、タイトル、説明、またはコメントにより多くの一致する単語を含むタスクが結果のリストの上位に表示されることを意味します。







この並べ替えは、検索クエリに明示的な並べ替え属性（たとえば、並べ替え：更新）または式「または」が含まれていない限り、デフォルトの検索結果に適用されます。

















カスタム日時フィールド



日付が「日付と時刻」形式で保存されるカスタムフィールドのサポートを追加しました。 これは、日付のみが格納されるフィールドと、時刻が重要なフィールドを区別できるようになったことを意味します。 この革新により、より正確に時間を追跡できます。

















JavaScript npmワークフローパッケージ



私たちは、ワークフローを扱うプロセスを常に改善しています。 このリリースでは、独自の開発環境からYouTrackワークフローを管理する機能を追加しました。 この機能は、JavaScriptをサポートするエディターで使用できます。 この機能の使用を開始するには、環境にNode.jsをインストールします。 詳細については、 ドキュメントを参照してください。

















日本語でのローカライズ



YouTrackが日本語をサポートするようになりました。 日本語圏の企業は、YouTrackでデフォルト言語として設定できます。







タスクフィールドのテキストインデックス作成



テキストインデックスには、すべてのタスクフィールドに格納されているすべての値が含まれるようになりました。 つまり、テキスト検索では、タイトル、説明、コメントだけでなく、あらゆる属性で一致する単語が検索されます。







さらに、値を検索する属性を省略することができます-検索フィールドに入力するだけです！

















テキストインデックスでの追加言語のサポート



YouTrackは、中国語、日本語、韓国語、トルコ語、ポーランド語の新しい言語のテキストインデックスをサポートするようになりました。 この機能を使用するには、必要な言語でテキストインデックスを有効にしてから、タスクを検索するときに、検索クエリの単語のすべての単語形式が考慮されます。







Microsoft ExchangeおよびOffice 365の共有メールボックスサポート



Microsoft Exchangeサーバー上の共有メールボックスに接続してメールを受信する機能を追加しました。 このようなメールボックスは、多くの場合、1つの外部メールアドレス（support@company.comなど）からの複数のエンジニアによる技術サポートの編成に使用されます。 Microsoft ExchangeまたはOffice 365を使用している場合、YouTrackをそれらと統合して、テクニカルサポートを自動化し、開発者とのやり取りを整理できるようになりました。

















電子メール通知を無効にする機能



特定のタスクの更新を追跡する必要がなくなった場合は、それらの登録を解除できます。 退会するには、YouTrackからの手紙の下部にあるリンクをクリックしてください。 無効なタスク通知のリストは、ユーザープロファイルの[通知]タブに表示されます。 いつでもタスクに再度サインアップできます。

















マークダウンのサポート



YouTrack 2017.4では、実験的な機能の一部としてマークダウンサポートを追加しました。 Markdown構文を使用して、タスクの説明、コメント、およびタスクのテキストをフォーマットできます。 Markdown構文とWikiを切り替えて、使い慣れた形式を選択できます。 マークダウンはデフォルトで有効になっています。この機能を使用したくない場合は、実験的な機能ページでオフにします。











また、その他の多くの改善を行いました-リリースに含まれる完了したタスクの全リストをご覧ください。







YouTrack 2017.4を試して感想を共有していただければ幸いです。 10ユーザー用の無料版をダウンロードするか、クラウドに登録して30日間試用版を使用できます。







YouTrackクラウドをすでに使用している場合、インスタンスはカレンダーに従って新しいバージョンにアップグレードされます。
