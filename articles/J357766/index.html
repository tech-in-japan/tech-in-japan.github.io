年間59.99ドルのAmazonは無制限のクラウドを提供します





Amazon は本日、Amazon Cloud Driveにデータを保存するための2つの計画、Unlimited PhotosとUnlimited Everythingを発表しました 。 2つ目は、10テラバイトまたは100テラバイトでさえありませんが、月額5ドルまたは年額59.99ドルで、あらゆるファイルに真に無制限のデータストレージを提供します。 無制限の写真は同じ無制限のボリュームを提供しますが、画像ファイルの場合は年間11.99ドルです。 この料金プランの写真に加えて、他の種類のファイル用の5 GBのストレージが無料で添付されています。



両方の料金プランは、3か月間無料で強度をテストできます。 ファイルを同期するために、WindowsまたはMac用のデスクトップアプリケーションと、iOSおよびAndroid用のモバイルアプリケーションの両方が提供されています。 Amazonのアカウントがなくても、ダウンロードしたファイルをダウンロードできます。 以前、Amazon Cloud Driveはユーザーに5 GBを無料で提供していました。 これに続いて、20 GBのクラウドで年間10ドルから開始し、500ドルで1 TBで終わる一連の関税が続きました。



以前、Bitcasaはクラウドでの無制限のストレージの同様のオファーを行いました。 興味のある人は、年間100ドルでそのような関税計画を発行するように申し出られましたが、数ヶ月後に彼らは考えを変えました 。 サービス統計によると、「無制限」という言葉は不必要な誇張であることが判明しました。0.5TBのユーザーのみが記入するのに1 TBを超えており、0.1 TBのみが10 TBを超えるのが面倒ではありませんでした。 その結果、昨年11月15日に新しいプラットフォームにデータが移行されたため、そのようなアカウントはすべて削除されました。
