WEBO Site SpeedUpバージョン0.9.5（パブリックベータ）

ほぼ3週間のレイアウト、統合、および集中的なテストの完了後（はい、みんなリラックスして、作業していました！）、自動Webサイトアクセラレーション用のオープンソースアプリケーションの新しいバージョン-WEBO Site SpeedUp 0.9.5（ 詳細 ）がリリースされました。 現在のステータスはパブリックベータです（つまり、このバージョンには重大ではないエラーがいくつか含まれていますが、「戦闘」使用の準備は完全に整っています）。 公式バグトラッカーの有益なエラーメッセージに非常に感謝します。



ダウンロードする

www.web-optimizer.us/web-optimizer/download.html



猫の下で詳細と写真





いくつかの革新は、 アルファ版に関する投稿ですでに発表されています。

  • お気づきのとおり、Google Web（site）Optimizerと競合しないように、製品の名前はWEBO Site SpeedUpに変更されています。 Googleとの競争は必要ありません:)
  • すでに多くの肯定的なフィードバックを収集している完全に新しいインターフェイス。 スクリーンショットもこちらとこちらに掲載されています 。
  • 最もおいしいものから：今すぐにプリセットを変更することが可能です。 デフォルトでは、3セットが利用可能です（もちろん、独自に作成できます）：安全（潜在的に「危険な」設定をすべて除外）、最適（優れた加速を提供しますが、システムを「殺す」ことができます）および極端（まあ、加速は最大ですが、リスクが非常に大きい場合は、さらに構成する必要があります）。
  • コントロールパネルの概念も改訂され、「3文」アプリケーションの最も重要な部分がそれに提出されました。 ブロックは非表示/展開できます。
  • キャッシュファイル用に別のページを追加しました。 このコンセプトは、インターフェースをより便利にするために、まだ最終決定および修正されます。
  • 再帰的なgzipファイルの圧縮と画像の最適化のためのユーティリティを追加しました（後者はsmushのみを使用しますが、punypngは進行中です）。
  • 現在のシステム環境とその設定に関する情報を含む別のページが追加されました。
  • インターフェイス言語の選択が追加されました（製品をロシア語/ウクライナ語以外の母国語に翻訳する準備ができている場合は、 ご連絡ください ）。


+その他の新機能

  • WEBO Site SpeedUpには最適化のための独自のAPIがあります （onBeforeOptimization、onAfterOptimization、onCacheメソッドは新しいバージョンで追加されています）。 これにより、通常のアクションと1回限りのアクションの両方を実行できます（たとえば、アプリケーションのインストール時にインデックスをベースに添付します）。
  • 現在、最新の安定バージョンだけでなく、中間ベータ版へのアップグレードも可能です。 最新のベータバージョンがインストールされている場合、安定したバージョンに「ロールバック」のメカニズムを追加するプロセス。
  • さまざまな環境での.htaccessとApacheモジュールの検出の改善された作業。
  • また、最も予想される改善の1つは、ページヘッダーのlang =“ wo”ラベルが削除されることです（代わりに、 <!--WSS-->



    ページの最後に挿入されます）。
  • 「すべてのブラウザに単一のキャッシュを使用する」機能を追加しました（外部キャッシュメカニズムを使用する場合は役立ちますが、クロスブラウザテクノロジーデータを使用することはできません：URI + mhtml）。
  • 外部リソースにある静的ファイルをキャッシュする機能を追加しました（通常のキャッシュヘッダーが設定されていないスクリプトまたは画像の場合）。
  • フレームの「遅延」ロードを追加しました（ページのメインコンテンツに干渉しないようにするため）。
  • ネイティブプラグイン（WordPress / Joomla！）の場合、サーバーキャッシュが改善されました（バージョン1.0のリリース前に、追加のパフォーマンステストを実施します）。
  • 特定のCookieが存在する場合、サーバーキャッシュからページを除外する機能が追加されました。
  • PrestaShop（1.2.5）のサポートを追加しました。


システムサポート



現在、サポートされているシステムのリストには、ネイティブ（内部拡張のレベル）でWordPress、Joomla！、Drupal、Bitrixが40以上あります（後者はWeb Optimizer経由のみで、WEBO Site SpeedUpインターフェイスが組み込まれています）。



サポートが必要な場合は、コメントに書いてください。



技術サポート



完全なテクニカルサポートは、完全版のユーザーにのみ提供されます。 ただし、 バグトラッカーに登録されたすべてのエラーは、通常24時間以内に考慮されます（通常、同じくらい迅速に解決されます）。



ライセンス条項



WEBO Site SpeedUp（バージョン0.6.0以降のWeb Optimizerなど）には3つの異なるエディションがあります（ バージョン比較 ）：

  • 非営利。 営利目的でないサイト（=非営利）で使用できます。
  • 軽量。 どのサイトでも使用できます。 非商用版とほぼ同じ機能を備えています。
  • 完了。 これらはすべて製品に含まれており、さらに24時間年中無休のテクニカルサポートも提供しています。


更新はすべてのエディションで無料です。 すべての有料ライセンスの有効期間は限られています（年）。 無制限のライセンスを購入することは可能です（これがあなたのケースであるならば、我々に書いてください ）。



ご理解のとおり、アプリケーションの開発を停止するつもりはありません。多くの新機能と個々のユーティリティが一般的なパッケージに追加され、互換性とパフォーマンスが向上します。 有料版のリリースと多くの批判が寄せられた後、高速サイトのファンの数は減らず、彼らの期待は満たされたことを嬉しく思います。



アフィリエイトプログラム



一般的な需要により、WEBO Site SpeedUpの販売のためのアフィリエイトプログラムを追加しました（最大20％以上の手数料）。 すべてが非常に簡単です。webo.inに登録して個別のリンクを取得します（webo.inにアカウントを既にお持ちの場合は、さらに簡単です-最後のリンクに従うだけです）。 このリンクからのすべての販売は「あなたの」としてカウントされます。

また、パートナー向けに多数のグラフィック資料を準備する予定です。 すでにアイデアがあれば、 それらを書いてください..
