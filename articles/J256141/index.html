VIII年次会議「Embedded Technologies 2015」

こんにちは



2015年5月27日に、モスクワはVIII年次会議「Embedded Technologies 2015」を開催します。 今年のイベントの統一コンセプトは、 モノのインターネット（IoT）です。 世界およびロシアのベンダーは、この世界的な傾向と、最新のスマートデバイスを作成するために開かれる機会、およびそれらの実装のための最新のツールとプラットフォームについて話します。

Windows 10 for IoTオペレーティングシステム、 IoT用の Microsoft Azure ソリューション 、およびこの分野のMicrosoftの計画に特に注意が払われます 。 イベント開催時に入手可能なすべての情報を開示します。







このプログラムには興味深いことがたくさんあります。



  • Microsoft、Intel、Advantech、Cisco、およびその他の主要なグローバルIT企業のモノのインターネットの概念、製品開発計画、パートナーと顧客の機会。
  • Microsoftスピーカーからの組み込みOS Windows Embedded 10ファミリの新世代に関するすべての関連情報。
  • 最新の組み込みプラットフォームMicrosoft Windows Embeddedの概要。
  • モノのインターネット向けのMicrosoftクラウドサービス。
  • それらと連携するためのビッグデータとツール。
  • 特殊なデバイスの技術的エコシステム：セキュリティおよび管理ツール、機器、アプリケーション、ソフトウェア開発環境。
  • ロシアおよび世界のメーカーの組み込みシステム向けのハードウェアおよびソフトウェアソリューション。
  • ロシアメーカーのスマートデバイスとソリューションの展示;
  • スマートシティインフラストラクチャソリューションの説明。








今年、会議「Embedded Technologies 2015」は、メーカーと組み込みソリューションを使用する営利企業および政府機関の両方で初めて開催されます 。 会議の参加者は、現代の特殊な電子デバイスの市場の現在の傾向について学ぶとともに、分析ソフトウェアやクラウドソリューションから管理およびセキュリティシステムに至るまで、幅広い関連技術に精通します。



伝統的に、会議は、ロシアのメーカーからのインテリジェントなソリューションとデバイスの展示会を開催します。 博覧会の特別な場所は、スマートシティクラスターが提示される垂直ソリューションのゾーンで占められます。 訪問者は、小売、輸送、医療、サービス、教育、銀行などのためのインテリジェントなソリューションに精通します。



会議に参加するには、専門のインテリジェントデバイスを開発する技術専門家や企業のトップ、ならびに企業 、銀行、小売チェーン、産業企業、広告ビジネス、医療および教育機関のITスペシャリスト 、および現代技術に基づくターンキーソリューションに関心のあるすべての人が招待されます。







会議のスポンサー： Microsoft、Intel、Advantech、Kaspersky Lab、Disti、Congatec、Sensor Systems、パートナー、70以上の情報スポンサー。 イベントのテクノロジースポンサーはMotion Viewです。



会議の主催者： Quarta Technologies、Microsoft Windows Embedded組み込みシステムのロシア有数のディストリビューター（EMEA Embedded Distributor of the Year 2013）。
