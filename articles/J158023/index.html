Python 2.7を使用したアニメーションの作成

この記事では、PythonとPygletを使用してアニメーションを作成する基本を紹介します。 Pygletは3Dグラフィックスで動作するように設計されていますが、この記事では非常にシンプルな2Dアニメーションを作成するために使用します。 特に、Pyglet（アニメーションを作成および再生するためのコードを開発する）を使用するための基本的なテクニックに慣れ、画像のシーケンスを使用してアニメーションを作成します。



それでは、始めましょう...



アニメーションは、次々にすばやく表示される一連のフレームです。 これにより、オブジェクトの動きの錯視が作成されます。



PYGLETをインストールする



Pygletは、Pythonを使用して記述されたアプリケーションでマルチメディアデータを操作するためのAPIを提供します。 これは、複数のプラットフォームで実行されるOpenGLライブラリです。 主にゲームやその他のグラフィックが豊富なアプリケーションの開発に使用されます。 Pygletは、 http ：//www.pyglet.org/download.htmlからダウンロードできます 。 Pygletバージョン1.1.4以降（ある場合）を配置します。 Pygletのインストールに問題はないはずです。



Windowsプラットフォーム


Pyglet 1.1.4.msi以降のディストリビューションをダウンロードしてインストールします。

Python 2.7をインストールする必要があります。



その他のプラットフォーム


Mac OS XのバイナリファイルはPyglet Webサイトに掲載されていますので、Pyglet-1.1.4.dmg以降をダウンロードしてインストールしてください。



Linuxでは、システムの構築に適したPyglet 1.1.4以降をインストールします。

見つからない場合、ライブラリは別の方法でインストールできます。



  • Pyglet-1.1.4.tar.gz以降のアーカイブをダウンロードして解凍します。
  • コマンドラインで動作するようにpythonが設定されていることを確認してください。 それ以外の場合は、PYTHONPATH環境変数でPython実行可能ファイルへの正しいパスを設定します。
  • コマンドプロンプトで、解凍されたアーカイブに移動し、次のコマンドを実行します。 
    python setup.py install
  • 次に、ライブラリフォルダーにあるreadme /インストールファイルに記載されている指示に従います。


設置確認


次に進む前に、Pygletが正しくインストールされていることを確認してください。 これを確認するには、コマンドラインからPythonを起動し、次のコマンドを入力します。 
 import pyglet


インポートが成功すれば、すべて準備完了です！



ピグレットの紹介



重要なコンポーネント


使用する最も重要なPygletモジュールとパッケージのいくつかについて簡単に説明します。 これはPygletフレームワークのほんの一部にすぎないことに注意してください。 Pygletのドキュメントをチェックして、すべての機能の詳細を確認してください。これはこの記事の範囲外です。



窓


pyglet.window.Windowモジュールはユーザーインターフェイスを提供します。 OpenGLコンテキストウィンドウを作成するために使用されます。 Windowクラスは、マウスやキーボードに関連付けられたイベントなど、さまざまなイベントを処理するためのAPIメソッドを表します。 ウィンドウは、通常モードまたは全画面モードで表示できます。 以下は、ウィンドウのインスタンスを作成する簡単な例です。 コンストラクタで引数として幅と高さを指定することにより、サイズを指定できます。 

 win = pyglet.window.Window()


画像の背景色は、OpenGLを使用してglClearColorを呼び出して調整できます。 

 pyglet.gl.glClearColor(1, 1, 1, 1)


このコードは、背景色を白に設定します。 最初の3つの引数は、赤、緑、青を指定します。 最後の値はアルファチャネルです。 次のコードは、灰色の背景色を作成します。 

 pyglet.gl.glClearColor(0.5, 0.5, 0.5, 1)


次の画像は、灰色の背景で作成されたウィンドウのスクリーンショットを示しています。





画像


pyglet.imageモジュールを使用すると、画面に画像を表示できます。 次のコードスニペットは、Pygletウィンドウ内の指定された位置に画像を挿入して表示する方法を示しています。 

 img = pyglet.image.load('my_image.bmp') x, y, z = 0, 0, 0 img.blit(x, y, z)




次のセクションでは、pyglet.imageモジュールでサポートされるいくつかの重要な操作について説明します。



スプライト（スプライト）


スプライトは、ユーザー定義の画像要素構成です。

スプライトは別の重要なモジュールです。 前述のPygletウィンドウに画像またはアニメーションのフレームを表示するために使用されます。 Spriteを使用すると、Pygletウィンドウの任意の場所に画像を配置できます。 スプライトは回転および拡大縮小できます。 同じ画像の複数のスプライトを作成し、それらを異なる場所または異なる方向の1つのウィンドウに配置できます。



アニメーション


Animationモジュールはpyglet.imageパッケージの一部です。 名前が示すように、pyglet.image.Animationは、画像の1つ以上のフレームからアニメーションを作成するために使用されます。 アニメーションを作成するにはさまざまな方法があります。 たとえば、一連の画像から、またはAnimationFrameオブジェクトを使用して作成できます。 スプライトアニメーションを作成して、Pygletウィンドウに表示できます。



AnimationFrame


画像からアニメーションの独立したフレームを作成します。 このようなAnimationFrameオブジェクトのシーケンスからアニメーションを作成できます。 例： 

 animation = pyglet.image.Animation(anim_frames) anim_frames - ,   AnimationFrame.




時計


このモジュールは、指定された時間に呼び出される関数をスケジュールするために使用されます。 たとえば、次のコードはmoveObjectsメソッドを1秒間に10回呼び出します。 

 pyglet.clock.schedule_interval(moveObjects, 1.0/10)




画像を表示する


画像セクションでは、image.blitを使用して画像をアップロードする方法を学びました。 ただし、blitを使用すると、画像を出力する効率が低下します。 作成されたSpriteインスタンスに画像を表示するより良い方法があります。 複数のSpriteオブジェクトを作成して、同じ画像を表示できます。 たとえば、1つの画像をウィンドウ内の異なる場所に表示する必要があります。 これらの各画像には、個別のSpriteインスタンスが必要です。 次の簡単な例では、画像をロードしてSpriteインスタンスを表示し、その画像を画面に表示しています。 



 1 import pyglet 2 3 car_img= pyglet.image.load('images/car.png') 4 carSprite = pyglet.sprite.Sprite(car_img) 5 window = pyglet.window.Window() 6 pyglet.gl.glClearColor(1, 1, 1, 1) 7 8 @window.event 9 def on_draw(): 10 window.clear() 11 carSprite.draw() 12 13 pyglet.app.run()


3行目のコードは、pyglet.image.loadを呼び出して画像をロードします。

この画像に対応するスプライトが4行目に作成されます。

行6は、ウィンドウの白い背景を設定します。

On_drawは、ウィンドウをリロードする必要があるときに呼び出されるAPIメソッドを表します。



次の図は、Pygletウィンドウに画像をロードする例を示しています。 

 import os original_path = "C:/images/car.png" new_path = os.path.normpath(original_path)




このウィンドウには静止画像が表示されます。



マウスとキーボードの制御


Pygletのウィンドウモジュールには、ユーザーがアニメーションの再生を制御できるAPIメソッドがいくつか実装されています。 on_mouse_pressやon_key_pressなどのAPIメソッドは、アニメーションの再生中にマウスやキーボードのイベントをキャプチャするために使用されます。 これらのメソッドは、特定のタスクを実行するために使用できます。



サウンドエフェクトの追加


Pygletは、オーディオおよびビデオコンテンツの再生をサポートしています。 次のコードは、アニメーション表示中にメディアファイルをダウンロードして再生します。 

 1 background_sound = pyglet.media.load( 2 'C:/AudioFiles/background.mp3', 3 streaming=False) 4 background_sound.play()


行3で提供される2番目のオプションの引数は、ブート時にメディアファイル全体をメモリにデコードします。 これは、アニメーション中にメディアファイルを数回再生する必要がある場合に重要です。 API play（）メソッドは、メディアファイルのストリーミングを開始します。



ピグレットアニメーション



Pygletフレームワークは、アニメーションの開発に必要な多数のモジュールを提供します。 Pygletを使用して2Dアニメーションを作成する方法を見てみましょう。



既存のアニメーションを再生する



たとえば、GIF形式のアニメーションファイルが既にある場合は、ダウンロードしてPygletに直接表示できます。 これを行うには、pyglet.image.load_animation APIメソッドを使用します。



既存のアニメーションの表示


以下のサンプルコードは、Pygletを使用してアニメーションを表示する方法を理解するのに役立ちます。 

 1 import pyglet 2 3 animation = pyglet.image.load_animation( 4 "images/SimpleAnimation.gif") 5 6 #   . 7 animSprite = pyglet.sprite.Sprite(animation) 8 9 #   Pyglet 10 w = animSprite.width 11 h = animSprite.height 12 win = pyglet.window.Window(width=w, height=h) 13 14 # r,gb,       15 r, g, b, alpha = 0.5, 0.5, 0.8, 0.5 16 17 # OpenGL     . 18 pyglet.gl.glClearColor(r, g, b, alpha) 19 20 #   21 # pyglet.Window 22 @win.event 23 def on_draw(): 24 win.clear() 25 animSprite.draw() 26 27 pyglet.app.run()




  • APIの3行目で、image.load_animationメソッドは、指定されたアニメーションファイルを使用してimage.Animationクラスのインスタンスを作成します。
  • 行7は、このアニメーションのスプライトオブジェクトを作成します。
  • 行12で作成されたPygletウィンドウは、アニメーションの表示に使用されます。
  • animSpriteは、ウィンドウの高さと幅を設定します。
  • ウィンドウの背景色は、glClearColorへのOpenGL呼び出しを使用して設定されます。
  • 次に、Pygletウィンドウでアニメーションファイルをスプライトに接続する必要があります。
  • Pyglet.windowは、イベントが発生したときに呼び出されるon_draw APIメソッドを定義します。
  • アニメーションを表示するには、行25でdraw（）が呼び出されます。
  • 27行目のコードは、このアプリケーションを起動します。


注：アニメーションファイルは、アプリケーションと同じディレクトリに配置する必要があります。



この例では、アニメーションをPygletウィンドウに表示します。 4行目の行を変更するだけで、任意のアニメーションファイルを使用できます。



次の図は、異なる時間間隔でのアニメーションファイルからのいくつかのフレームを示しています。





画像のシーケンスを使用したアニメーション


Animation.from_image_sequenceは、多数の連続した画像を使用してアニメーションを作成できるAPIメソッドです。 各画像は、アニメーションフレームとして順番に表示されます。 アニメーションフレームが表示される時間は、オブジェクトのアニメーションを作成するときに引数として示されるか、アニメーションインスタンスの作成後に設定できます。



順次表示がアニメーションとなるいくつかの図面を作成して、画面に表示してみましょう。 各画像ファイルは、指定された時間の間、アニメーションフレームとして表示されます。 たとえば、振り子のある祖父の時計の画像を使用します。 アニメーションは、時計の振り子の振動です。 このアニメーションは3フレームの画像で構成され、そのうち2フレームでは振り子が反対側にあります。 これらの画像は次のようになります。





時計の振り子を復活させます。 

 1 import pyglet 2 3 image_frames = ('images/clock1.png', 4 'images/clock2.png', 5 'images/clock3.png') 6 7 #   - 8 images = map(lambda img: pyglet.image.load(img), 9 image_frames) 10 11 #      0,33 12 #  13 #      ,      14 #   ,       1  15 16 animation = pyglet.image.Animation.from_image_sequence( 17 images, 0.33) 18 #    19 animSprite = pyglet.sprite.Sprite(animation) 20 21 #   Pyglet 22 w = animSprite.width 23 h = animSprite.height 24 win = pyglet.window.Window(width=w, height=h) 25 26 #     27 pyglet.gl.glClearColor(1, 1, 1, 1) 28 29 @win.event 30 def on_draw(): 31 win.clear() 32 animSprite.draw() 33 34 pyglet.app.run()




  • image_framesタプルには画像パスが含まれています。
  • 8行目のmap関数を呼び出すと、各画像パスに対応するpyglet.imageオブジェクトが作成され、リストに保存されます。
  • 16行目では、APIメソッドAnimation.from_image_sequenceを使用してアニメーションが作成されます。 このメソッドは、最初の引数として画像オブジェクトのリストを取ります。 2番目の追加の引数は、各フレームの表示時間（秒）です。 完全なアニメーションループの合計時間が1秒になるように、0.33秒に設定します。 したがって、アニメーションで振り子を完全に1回振るには1秒かかります。


注：アニメーションファイルは、アプリケーションのディレクトリ内の別のフォルダーに配置する必要があります。



この例を開始すると、振り子が振動する時計が表示されるウィンドウが表示されます。 アニメーションサイクルは、ウィンドウが閉じるまで続きます。
