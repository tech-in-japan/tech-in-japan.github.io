NetAppとCisco：コラボレーションの強化、新しいFlexPod構成

記事の続きFlexPodとは？ 。 つまり、FlexPodは、Cisco UCSサーバー、Cisco Nexusスイッチ、NetApp FASストレージシステムに基づいて、CiscoおよびNetAppが提供するサーバーおよびネットワーク機器とデータセンターソフトウェアのセットです。



2013年1月24日、ネットアップとシスコは協力の拡大を発表しました。



したがって、両社の製品のより深い統合と、両社のいくつかの新しい開発を使用したFlexPodソリューションの新しい構成を実現します。



  • FlexPodアーキテクチャのより深い統合：シスコとNetAppは、Cisco UCSおよびNetApp FAS製品とFlexPodオープンエコシステム管理を引き続き統合します。

    FlexPodは最大10,000台のサーバーを管理する機能を提供し、組織が複数のFlexPodラックを組み合わせてマルチホップFCoEを使用できるようにします。 これらのソリューションは、ブランチストラクチャを備えたサービスプロバイダーおよび企業向けに特別に設計されており、予測可能なパフォーマンスとテクニカルサポートでインフラストラクチャを拡張できます。

    FlexPodは引き続きインフラストラクチャ管理ユーティリティとの統合をリードし、Cisco UCS Manager 2.1をサポートします。 シスコは最近、統合された統合インフラストラクチャ戦略を一貫して継続および開発することでCloupiaを買収しました 。 さらに、FlexPodのお客様向けにクラウドベースのクラウド管理を有効にすることで、Cisco Intelligent Automation for Cloud（Cisco IAC）がサポートされます。 コンポーネントレベルとは異なり、プラットフォームレベルでのFlexPod全体のより深い統合。ApacheCloudStackを搭載したMicrosoft System CenterやCitrix CloudPlatformなどのパートナーソリューションを使用してFlexPodを簡単に調整できます。
  • また、シスコとNetAppは、ホストレベルのフラッシュテクノロジーとストレージを統合して、統合アーキテクチャ全体のパフォーマンスを向上させます。 これには、NetApp FlashAccelホストアクセラレーションテクノロジーと、 ストレージコントローラー側のNetApp FlashCache / FlashPoolが使用されます。




  • クラウドサービスプロバイダー向けのFlexPod .CiscoとNetAppは、FlexPodソリューションの大規模な拡張の可能性を備えた柔軟なアーキテクチャを開発しました。FlexPodソリューションは、サービスプロバイダーのマルチデータセンターアーキテクチャと連携できます。 このソリューションセットには、Cisco UCSサーバーおよびCisco Nexus 7000シリーズスイッチを備えたNetApp Clustered ONTAPクラスターOSおよびNetApp FASストレージが含まれ、動的なサーバープール、ストレージ、および顧客のニーズに基づいて上下に拡張できるネットワークリソースを作成します。アプリケーション。 このソリューションでは、Ciscoの管理とオーケストレーション、BMC Softwareなどのパートナーのエコシステム、将来的にはオープンソースプロバイダーを含むCisco VMDCアーキテクチャを使用します。





  • ブランチオフィスソリューション：シスコとNetAppは、ExpressPodアーキテクチャの機能を拡張して、コンピューティング、ストレージ、およびブランチオフィスのネットワークリソース要件を満たすパートナーシップを開発しています。 ExpressPodは、ブランチの成長に合わせてFlexPodと同様の統合アーキテクチャを使用します。 このインストールが簡単なアーキテクチャは、組織のデータセンターコアとの高いレベルの互換性を提供し、アフィリエイトが仮想化やクラウドコンピューティングなど、IT業界で最も効果的なトレンドをすべて活用できるようにします。 ExpressPodにはVMware vSphereのサポートが含まれ、まもなくWindows Server 2012 Hyper-Vのサポートが含まれます。





  • FlexPodアプリケーション検証： ミッションクリティカルなアプリケーションカバレッジを拡大し、クラウドデータセンターソリューションを提供するシスコとNetApp。 ミッションクリティカルなソリューションには、Oracleデータベースとアプリケーション、SAP BusinessObjectsビジネスインテリジェンス、SAP HANAプラットフォームが含まれ、MicrosoftプライベートクラウドやCitrix CloudPlatformなどのクラウドソリューションも提供します。
