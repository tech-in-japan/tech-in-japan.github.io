Aiptek PocketCinema V60 picoprojectorレビュー：60インチ、安価

あるカメが言ったように、事故は偶然ではありません。 だから私は、新しいテレビの検索中に、運命の意志により、「ブルースクリーン」の非常に興味深い代替案を知ることができました。 人口の大部分（最近まで私のように）は一般に、プロジェクターの機能についてほとんど知りません。 すぐにそれらは純粋に暗闇の中で「見える」と考えられ、コンテンツのソースは完全にあいまいです-おそらく、最大値はコンピューターへのフラッシュドライブ接続またはHDMI接続です。 Aiptek PocketCinema V60 picプロジェクターは、これらのステレオタイプの誤解を払拭するのに役立ちました。







「picoprojector」という用語自体はオンラインストアでは好まれておらず、これらのデバイスを一般の人が理解できる「ポータブル」という用語にバインドすることを好みます。 実際、これはすべてを言っています-出張や顧客への旅行で問題なくデバイスを使用できるように、自律運転とコンパクトな寸法の可能性を意味します。 後者は、あらゆる種類の会社の代表者、「フィールド」マネージャーなどに役立ちます。 そして、picoprojectorは純粋に、たとえば「働く」使用のためのツールであるように見えますが、自宅ではそうではありません。



数週間前、私はプロジェクターをまったく考えずに、静かにテレビを買うことを考えました。 しかし、テーブルの前の作品を訪れた後、マネージャーの1人が2台のデバイスを減らし、Aiptek PocketCinema V60とAiptek PocketCinema V100が私の場所に現れました。 2番目のモデルについては、「60代」と比べてあまり興味がなかったので、少し後で説明します。



納入範囲



Aiptek PocketCinema V60は、背景が白い中程度のサイズの箱で提供されますが、これはパッケージの「実行可能性」に最適な影響を与えません。 最初は、箱はポリエチレンで密閉されているため、工場からバイヤーの自宅までの途中のプレゼンテーションが保存されます。 側面には図のペアがあり、その識別は私にいくつかの困難を引き起こしました-Aiptek自体について言うか、または...











内部には、HDMI-HDMIケーブル、OTG（！）、MicroUSB、リモート、2つの取り外し可能なフォークを備えた充電ユニットが見つかりました。



















デバイスのポータブルステータスを考えると、論理的なソリューションはカバーの形のアプリケーションでした。 物事は非常に、非常に快適に触れて、優れた出てきました。 確かに、カバー自体に保護が必要かどうかを考えるようになります。 プロジェクターに加えて、OTGケーブルとmicroUSBケーブルをケースに収めることができます。 はい、アクセサリー一式のスペースを見たいのですが、これは「まったく別の話」です。











Aiptekの保証-国際、保証書には条件が10言語で記載されています。 本文の面白い詳細が気に入りました：「万が一、不具合が確認された場合...」 笑顔、そしてのみ。 残念ながら、Aiptekロシア語のウェブサイトでは、CISにある会社の保証センターに関する情報は見つかりませんでした。











ピッキングリストの最後の項目は、よく書かれた指示です。タブレットやスマートフォン向けの乏しいマニュアルの後で、このような「本」の習慣をすでに破っています。 19ページに、プロジェクターのすべての側面が詳しく説明されています。これは朗報です。



外観、コネクタ、およびコントロール



寸法は、プロジェクターの私のお気に入りの部分です。 サイズでは、Aiptek PocketCinema V60はわずかに拡大されたテレビのリモコンに似ています：190 x 165 x 80 mm、重量-わずか190 g。一部のNokia Lumia 920またはSamsung Galaxy Note IIの重量はほぼ同じです。 私の意見では、長方形の形状は大きなプラスです。 プロジェクタは、ジャケット/ジャケットの内ポケットに簡単に配置できます。











デバイスの主要部分は、通常の、ややマットなプラスチックで作られています。 土壌は中程度で、定期的にケースを拭く必要があります。 興味深いことに、V100モデルでは、より実用的なソフトタッチプラスチックが使用されています。 背面パネルと前面パネルは光沢があり、その後の結果はすべて続きます。 プロジェクターの前面と背面に取り組む必要は特にないので、これはそれほど期待はずれではありません。 アセンブリは、5ポイントスケールの強力な4つで行われます。 絞ると、身体はわずかに聞こえるようにクランチしますが、バックラッシュは観察されません。



プロジェクターの上部には、スピーカーと一連のタッチキーがあります。 理由で選ばれたのは感覚のオプションでした。 プレゼンテーション中のプロジェクタの明るさにより、通常のボタンを扱っている場合は簡単に移動できます。 すべてのキーはバックライト付きです。 あまり便利ではないのは、バックライトをオフにできないことです。











右側には、フォーカスホイールと「冷却」グリルがあります。







反対側-電源管理スイッチとキーロック-表示中に誤って「何」をしないようにします。 AV出力とヘッドフォン/スピーカージャック、別のグリル、赤外線ポートを組み合わせた「サークル」もあります。











正面から、換気スロットとレンズが私たちを見ています。 さらに、2台のプロジェクターのいずれにも追加のレンズ保護はありませんでしたが、拒否しませんでした。











コネクタの真の美しさは、microSDカードスロット、DC、HDMI、microUSBコネクタの背面パネルに表示されます。 したがって、プロジェクタは実際にほとんどすべての一般的なストレージメディアをカバーします。 USBフラッシュドライブをOTG経由で接続し、スマートフォンからカードを挿入し、コンピューターに直接接続します。先を見据えて、私はほとんど常にHDMIのみを使用したと言います。







円の下に再びスロットがあり、その隣には三脚にネジ止めするための適度に設けられた穴があります。 写真を少し上げる必要がある場合は、ミニスタンドもあります。











仕事中



プロジェクターは2 GBの内部メモリを提供します-1つの映画と音楽とドキュメントの選択用です。 コンピューターに接続すると、プロジェクターは問題なく認識され、同時に挿入されたmicroSDカードが表示されました。



プロジェクタは、内部ファンによって生成される低ノイズで動作します。 原則として、焦点を合わせると、音自体が感じられます。 しかし、映画を見ている間、プロジェクターは私から1メートルも離れていませんでしたが、話題は決して注目されませんでした。



私の写真の趣味のために、三脚は手元に非常に便利で、その上にプロジェクターが設置されていたとすぐに言う価値があります。 問題の価格は20〜30の「常緑樹」です。 その見返り-天井までの「放送」により、垂直までプロジェクタの位置を調整する究極の柔軟性。



速度の面では、プロジェクタはメインメニューで直接通常の速度で動作しますが、ファイルナビゲーションは遅延します。 そのため、開始ウィンドウには、コンテンツの種類、共通ファイルマネージャー、および設定への移行ごとにタブが表示されます。 一方、バッテリーの状態は右上隅に表示されます-バッテリーレベルまたはネットワーク接続識別子。 接続されたソースがすぐそこに表示されます-メモリカードまたはフラッシュドライブ。 プロジェクターが自律的に動作している場合、「死んでいる」間、画面の中央に大きな赤い放電アイコンが表示されます。

オートパワーオフ。 一定時間後の自動シャットダウン-1/3/5/10/20/30分。



携帯電話の充電。 はい、プロジェクタはポータブルバッテリになり、接続されたデバイスに3,000 mAhを供給できます。これは、主要なスマートフォンを1時間半充電するのに十分です。

私は少し余談します。 プロジェクターにはバッテリーが内蔵されているという事実にもかかわらず、最高輝度で1時間の放送だけが最良の指標ではありません。 したがって、状況によっては、外部電源が役立つ場合があります。 Aiptek PocketCinema V60とLexand LPB-52Wの互換性を確認しました。 以前のブログで、Lexand LPB-78Wバージョンがレビューされました。 だから、私は失望することを急いでいます-microUSBポート経由の接続での充電はサポートされていません。



誰かが知らない場合、Lexand LPB-52WおよびLexand LPB-78Wは、3Gインターネットの「配布」機能を備えた多機能ポータブルバッテリーです。 同時に、各モデルには、ワイヤレスファイルサーバーに変換するためのmicroSDカードスロットがあります。 同じ成功を収めて、フラッシュドライブと外部USBハードドライブを接続できます。 デバイス間の違いは、取り外し可能なバッテリーの容量で表されます-バッテリーが5,200 mAhでどこが7,800 mAhであるかを理解することは難しくないと思います。 これらのデバイスに2つのUSBポートが装備されていると非常に便利です。これにより、3Gモデムの同時接続とスマートフォンまたはタブレットの充電が可能になります。 一般的に、長い旅でも、毎日でも非常に便利なことです。



画像



プロジェクターの「視覚コンポーネント」のパラメーターは次のとおりです。DLPオプティクス、解像度640 x 480ピクセル、コントラスト比500：1、画像サイズ15-152 cm（6-60インチ）、投射距離20-200 cm（8-79インチ）。



おそらく、すぐに画質に集中する必要があります。 カーテンを引くと、日中に映画やドキュメントを見ることができます。 しかし、もちろん、夕方または夜遅くまで待つ方が良いです。 明るさとコントラストは、個人的に不快感を感じることはありませんでした。 一方、写真のソースとしてタブレットが接続されている場合、視覚的な違いは憂鬱でした。 結局のところ、50 lmに何を期待していましたか？ これを要約すると、私と一緒に映画を見た妻は、プロジェクターについて非常に前向きに語り、彼女は明るさや彩度を「追加する」必要はありませんでした。







私は認める、私は見るために排他的に白い表面で気にしませんでした。 アパートの壁紙は、適度に「邪魔な」背景を作るのに十分明るいです。 白い天井での視聴がテストされました-見た目は素晴らしいですが、ポップコーンよりも深刻なものを食べるのは難しいです。 クールな一口は言うまでもありません。







解像度が小さい-何もする必要はありません。 しかし、再び、映画を見るとき、あなたはそれに注意を払いません。 しかし、コンピューターから画像を出力すると、あいまいな文字の輪郭しか見えません。 しかし、ブラウザを開いてTNT Webサイトにアクセスし、灰色のインターンを立ち上げることを私は止めませんでした。



ちなみに、Androidのシステムメニューでタブレットを画像ソースとして使用しようとすると、画面の左側に干渉バーがちらつきます。 全画面モードでは、ブラウザを介して「インターン」を表示すると、はっきりとした波紋が見えました。

私の意見では、結果は良いです。 さらに、私は自分のHDクリップとフルHDクリップのランダムな選択、およびDVDripから1080pまでの高品質フィルムを「運転」しました。 私のファイルは、画像と音声の両方で問題なく再生されました。



オフィス文書に目を向けましょう。 ここで私は、同じ会社の代表者がプレゼンテーションを実演しなければならない場合に画面がどれほど鮮明であるかを見ることに最も興味がありました。 プロジェクターは、3秒で11 MBのPowerPointファイルを開きました。 フレームはすぐには変わりませんでした。 より正確には、基本的に遅延はありませんでしたが、多くの場合、2〜3秒、さらには5〜6秒待つ必要がありました。 スライドを逆の順序で表示すると、遅延が4秒に達しました。



読みやすさに関しては、18サイズのテキストは完全に表示され、問題がありましたが、多かれ少なかれ見分けやすい13サイズでした。



私の論文のファイルはほぼ瞬時に開かれ、プロジェクターはリモコンを押すたびに明確に応答し、ページが即座に変更されました。 数式、表、画像の表示に問題はありませんでした。 確かに、私は何も読むことができませんでした...ズームを使用するまで。 結局のところ、4度の増加があります。 また、ページの約3分の1を含む最大で、すべてが元の12サイズで完全に読み取り可能でした。 雑誌や書籍をPDFで発行する試用版では、開く速度とページングの速度は完全な順序ですが、何かを読むのは難しいことがわかりました。



最後に、4,928 x 3,264ピクセルの解像度で写真を表示してみました。 フレームが変更されるまでに約2秒かかりましたが、画像の鮮明さは理想とはほど遠いものでした。



おわりに



このデバイスを国内の小売店で見るまで、「丘を越えて」の値札は320ドル程度です。 Aiptek PocketCinema V100はロシアで販売されており、その価格は実際に外国のオンラインストアで見たものと変わりません。 したがって、Aiptek PocketCinema V60が指定された制限を超えない可能性があります。



最初の導入後、私はこのプロジェクターをテレビの代替品として積極的に検討しました。 むしろ、単に「大画面」に-私は長い間テレビを見ていません。 そして、Aiptek PocketCinema V60を入手したのは、新しいテレビを購入したいという思いと一致しました。 一方で-約32,000ルーブル（選択したモデルの場合）の1回限りの廃棄物-一方では、1部屋のアパートにはすでに十分なものがあります。 そして、ここにポケットのもの、そして適度なお金さえあります。 一般的に...



何らかの理由で、特別な選択肢は見つかりませんでした。 価格を見ると、最大12,000ルーブルの「フルサイズ」モデルがありますが、コネクタのセットについて深刻な疑問があり、解像度が高すぎません-WVGA。 解像度が約1024 x 768ピクセルの場合、ランプの寿命は約6,000時間です（Aiptek PocketCinema V60では20,000時間）。 Yandex.Marketに連絡して、最低18,000時間、HDMIの存在、800 x 600または858 x 600の解像度を示した-8つのモデルを受け取った。 これらのうち、3つは17 325ルーブルの価格で販売されています。



急に見落とされた場合-指導に感謝します。 しかし、1万から12,000ルーブルを超えないことを覚えておく価値があります。



主なものについて簡単に説明します。 プロジェクターがリモートコントロールに常に反応するわけではないことを気に入らなかったので、ファイルナビゲーションをもう少し速くしたいと思います。 明るさと解像度について-価格を考慮して、それらを欠点に帰するのは間違っているでしょう。 私はこれを言います-表示された画像は、テキストに関してビデオに適しています、14-15のフォントサイズのプロジェクターを使用することは理にかなっています。 プレゼンテーションでは、Aiptek PocketCinema V60をスタンドアロンユニットとして使用しません。 ただし、ネットブックまたはタブレットから画像を表示する場合-これはまったく別の問題です。
