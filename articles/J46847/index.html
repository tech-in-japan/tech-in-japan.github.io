GPU、日本のGPUスーパーコンピューター

世界で最も強力なスーパーコンピューターの最後のリストには、 1つの興味深い展示があります。 これは、calledと呼ばれる日本の技術の奇跡であり、77.48テラフロップスの性能で評価の29行目に位置しています。



Tsubameの独自性は、従来のCPUに加えて、数百のnVidia GT200 GPUが動作することです。 これは、多くの民生用PCで見られる最も一般的なGPUです。



スーパーコンピューターは、東京工業大学の2階にあるいくつかの部屋を占有しています。 主な計算能力は、655個のSun x4600モジュールによって提供され、各モジュールには16個のAMD OpteronプロセッサとClearspeed CSX600アクセラレータが搭載されています。 ただし、それらに加えて、4つのGPUを備えた170台のNvidia Tesla S1070ユニットがシステムに接続されており、各ユニットには240コアが搭載されています。



システム全体でSuSE Linux Enterprise Server 9が実行されています。このアーキテクチャは、既存の分散システムに新しいTeslaモジュールを追加できるように設計されており、スーパーコンピューターの容量が増加します。



日本のエンジニアによると、将来的には、スーパーコンピューターでのGPUの使用が一般的になるでしょう。
