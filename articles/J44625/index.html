Dr.Web Av-Deskの実装

最初に、簡単に説明してから、それが何であるかを説明し、それから助けを求めます。

Dr.Web AV-Deskは、無制限の数の加入者に対してウイルスやスパムからPCを保護するためのさまざまなサービスを提供するインターネットサービスです。

Dr.Web AV-Deskは、このようなサービスを提供するプロセスを集中管理できるソフトウェアです。


一言で言えば、実装プロセスは次のようになります。

-プロバイダーはav-deskサーバーをセットアップします

-管理者コンソールを使用して、一意のIDを持つユーザーが追加されます（たとえば、321a6efe – d11d – b211–8186–980e393f8ac3）

-ユーザーには、 http：// server / blablabla？という形式のリンクが与えられます？Id = 321a6efe – d11d – b211–8186–980e393f8ac3

-リンクを介して、彼はウイルス対策の一意の配布をダウンロードし、自分のコンピューターに配置します



明確なビジネスウイルス対策有料。

そして問題自体は、請求書がないことですが、xml apiで管理できます。

そして今注目されているのは、 「xml apiを通じて、できればphp + mysqlで何かを管理できる請求書をどこで入手できるのか」という質問です。



PS：技術サポートは何も有益なことは言えません。
