MDC Desktop 1.0.3.3のリリース。 それら。 コンセンサス

最近、 MDCの周りに小さなコミュニティが出現しました。これはbugs.mdc.ruで私たちを助け、メールにinfo@mdc.ruを書き込みます。 奇妙なことに、コミュニティの大半は、プログラム開発者として私たちにとって大きな役割を果たしているMDCのさらなる開発に根本的な異議を唱えていません。 それが、現在のリリースが誇らしげに「 1.0.3.3。コンセンサスにちなんで名付けられた 」というタイトルを付けられている理由です。







写真はクリック可能です



デスクトップバージョンMDC 1.0.3.2の最後のリリース以降。 「春の増悪」は97枚のチケットでクローズされました。 以下は、変更とバグ修正の簡単な概要です。

  • スパム対策：自分で質問と回答のオプションを尋ねることができます。
  • 通信の履歴をダウンロードする作業を最適化しました。
  • メッセージウィンドウの最適化（ctrl + cがメッセージウィンドウチャットで機能するようになりました）;
  • 「MDC」メッセージボックステーマの新しいスタイルが追加されました。 このスタイルでは、メッセージの送信に使用されるプロトコルアイコンが表示されます。







    （TributeToQIPスタイルも追加されました）

  • より美しい通知ウィンドウ（新しいメッセージが表示されるときのポップアップ）。 ポップアップを個別に設定できるようになりました：サイズ、アバター、背景、透明度など。



  • Mail.ruおよびICQプロトコルのファイル転送（メッセージウィンドウにドラッグアンドドロップする、連絡先リストウィンドウの連絡先にドラッグアンドドロップする） ;
  • 保存するフォルダを選択したファイルを受信します。

    icqサーバーの動作が不安定であるため、異なるサブネットにあるコンピューター間のファイル転送を完全にテストすることはできませんでした。 現在、ネイティブクライアントでもファイル転送は安定していません

  • MDCのローカライズ。 MDCユーザー（Oleg Zoshchak） （別名open-life.orgのvlas_st）に感謝します... ウクライナ語のローカライズについて 。 言語の選択は、「一般」MDC設定で利用できます。



    MDCの開発を支援したい場合、ロシア語とウクライナ語以外の言語に堪能な場合は、MDCの次のリリースに翻訳を追加できます（詳細はinfo@mdc.ru ）。

  • mdc連絡先のアバターを設定します（連絡先情報ウィンドウを表示しているときに、アバターをクリックできます）。
  • 連絡先情報ウィンドウからフィールドをコピーする機能。
  • Ubuntu 9.04のメニューが読めないバグを修正。
  • 新しいmail.ruアカウントの通知が追加されました。
  • 笑顔の新しいテンプレート。
  • 笑顔で最適化された作業。 メッセージウィンドウでスマイリーをクリックすると、スマイリーが非表示になり、テキスト表現が表示されます。
  • {{{}}}の使用（BBコードの類似物[コード]：メッセージ内のさまざまなコードの例を書くためのもの）;
  • メモリとCPU消費の別の最適化。


MDCがリリースされるたびに、MDCのさらなる開発に役立つ多数のレビューとバグレポートを受け取ります。 もう一度、 bugs.mdc.ruのすべてのユーザーに感謝します。 単一のチケットは無視されません。
