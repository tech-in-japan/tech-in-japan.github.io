Windows 8のスタート画面に関するQ＆A

Windows 8のリリースにより、パーソナルコンピューターのユーザーは2つの戦争キャンプに分割されました。最初は、システムインターフェイスの変更のようなもので、他のものはそれに対して断固として反対しています。



8を超えるすべての点を挙げて、問題に対する私のビジョンを伝え、人々がトピックごとに尋ねる質問の全リストにコメントします。これは1万回答えることは不可能です。











Q：古いスタートアップの何が問題になっていますか？

A： OSの統計を収集しているMicrosoft の人たちは興味深い結論に達しました-彼らは主に検索ボックスにWindows 7での起動を使用しました。 打ち上げ自体はひどく不快で、ほとんど何もすることは不可能でした。 Bill Gatesの会社は、すべてのスタートアップ機能を一連のスーパーバー+検索ボックスに置き換えることができるため、大きく変更する時が来たことに気付きました。 多くの人がそうしましたが、私も例外ではありません。 しかし、スーパーバーの場所は終わったので、エプロビアンの「棚」の左のクローンを置かなければなりませんでした。



Q：本当に価値のある、美しくて快適な何かをすることができます。 なぜすべてがそうですか？

A：マイクロソフトは、タブレット、電話、PC向けのユニバーサルOSを作成するという目標を設定しています。 シンプルで簡潔で快適。 最良の解決策はすぐ目の前にあったことを認めなければなりません。新しいリリースごとに不要になった「このボタン」は、グラフィカルインターフェイスをタブレットに変更するのに理想的でした。



Q：フルスクリーンでの起動はうんざりします。 なんで？

A：次に、スタートメニューを自分の都合に合わせて正確に構成できるようにします。 グループのアイコンとタイル。 私にとって、それは「棚」のネイティブで便利な類似物になりました。

しかし、気に入らない場合は、以前は使用しなかったように使用できません。



Q：検索はどこに行きましたか？

A：検索も同じように機能します-スタート画面でキーボードから何かを入力すると、アプリケーション、設定、ドキュメント、そして重要なインストール済みのMetroアプリケーションごとにグループ化された結果がすぐに表示されます。 この分離はあなただけに良いのか悪いのか。 しかし、新しい検索は何倍も高速で機能的になりました。



Q：タイルの意味がわかりません。

A：タイルは、新しいオペレーティングシステムでデモンストレーションすることを恐れていなかった革新的なアイデアです。

タイルは、アプリケーションがその裁量で最も重要な情報を表示できるようにするショートカットウィジェットです。 これはショートカットの自然な進化です-リンク付きの静止画像。



Q：これらのメトロアプリケーションはすべて、マウスを使用する場合に便利ではありません。 今どうなっているの？

A：ありません。 それらを使用することを強制する人はいません。デスクトップ環境を置き換えることは決してありません。 Microsoftはこれを非常によく理解しています。



Q：地下鉄とクイックスタート以外に何か新しいことはありますか？

：たとえば、エクスプローラーで、最も重要で頻繁な設定でリボンリボンが表示されました-「登録済みファイルの拡張子を表示」および「隠しファイルを表示」のチェックボックスを含む（表示->表示Xをクリックするだけ）

また、単一のファイルマネージャも実行しました。 ファイルをコピーするときの時間決定を改善し、速度グラフを作成しました。

また、新しいタスクマネージャーを作成しました。 チャートには便利ですが、古いチャートも削除していないので、実行ウィンドウにその名前を書き込むことで呼び出すことができます（win + r）。



Q： 2つのコントロールパネル、シャットダウンと再起動ボタンがどこで明確ではありません。 なぜすべてがそうですか？

A：残念ながら、すべてが私たちが望むほどスムーズではありません。 マイクロソフトは、設定を「メトロ設定」と「通常のコントロールパネル」に分割する必要がありました。 また、タッチスクリーンを満足させるために、コンピューターのメイン設定がチャームパネルに転送されました。

イデオロギー上の理由を含む。



Q：なぜ彼らはユーザーに選択肢を与えなかったのでしょうか？

A：マイクロソフトは営利企業であり、少なくとも有望な分野の促進を考えているわけではありません。
