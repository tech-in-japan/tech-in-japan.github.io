Firefox 13は古いバージョンのWindowsでは動作しません





Mozillaの代表者は最近、Windowsの古いバージョンでのFirefoxの新しいバージョンであるFirefox 13のサポートに関する興味深い声明を発表しました。 この場合、「廃止されたバージョン」は95、98などとは見なされませんが、Windows 2000、Windows XP、およびXP SP1です。



Windows XP SP2以降、Firefox 13はすでに動作します。 もちろん、このブラウザは新しいオペレーティングシステムでも動作します。 開発者によると、そのような措置はFirefoxのパフォーマンスを改善するでしょう。 新しいバージョンのリリースは今年6月4日に予定されているため、Mozillaソフトウェアを使用するすべてのWindows XPユーザーは、ブラウザがすぐに動作するために必要なサービスパックをインストールすることをお勧めします。



現在、XPは最も人気のあるOSのままです（いずれにせよ、年の初めにはそうでした）。 ユーザーのWindows 7への移行を促進するMicrosoft Corporationのすべての試みにもかかわらず、多くの人は移行の要点を理解していません（または、一部の企業の場合、この移行は経済的に不利です）。



Mozillaはまた、今年の4月末から（正確には4月24日から）Firefox 3.6のサポートを中止する意向を発表しました。 数週間以内に、Mozillaはブラウザブランチ3.6.xの最新バージョンをリリースし、バージョン3.6.28になります。 このスレッドの他のバージョンはリリースされません。
