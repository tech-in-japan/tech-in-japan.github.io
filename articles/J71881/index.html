Androidは2年でiPhone OSとWindows Mobileを追い抜く

Gartnerのアナリストは水晶玉について相談し、2012年までにAndroidモバイルプラットフォームが世界で2番目に人気になることを発見しました。 スマートフォン市場の発展に関する予測を含む同社の完全なレポートは10月末に開催されますが、ガートナーのKen DulaneyはすでにComputerWorldレポーターと基本データを共有しています。



アナリストによると、アンドロフォンの人気の熱狂的な成長は、多くの異なる要因によるものです。 重要な要素の1つは、多数の「クラウド」機能と関連アプリケーションの形でのGoogleからのサポートです。



また、プラットフォームとしてのAndroidは、他のOSよりも優れています。 たとえば、Apple iPhone OSの場合、Androidの主な利点はオープン性です。



さらに、スマートフォンのインターフェイスは通常2つのアプローチで異なります。iPhoneではすべてがアプリケーションを中心に構築され、Windows MobileおよびSymbianプラットフォームに基づくデバイスでは、通信タスクと機能に主な焦点が置かれます。 Dulaniによると、Androidは両方のアプローチを巧みに組み合わせています。 特に、Androidインターフェースにより、ユーザーはメインメニューに戻ることなく、頻繁に使用するタスクを実行できます。



最後に、最初のandrophoneは1年前に登場し、あまり人気がありませんでした（さらに、深刻なハードウェア制限があります）が、ここ数か月で多くの大手メーカーのいくつかの新しいAndroidスマートフォンが登場しました。 来年、アンドラフォンのモデルはさらに40種類あり、コードネームDonutと呼ばれるシステムの新しいバージョンもリリースされる、とDulaniは言います。



Gartnerの専門家は、これらすべての要因を数学的に書き留め、2012年のモバイル市場のこのモデルのようなものを手に入れることができました。



予測は次のとおりです。



1. Symbian OS -2億3千万台のデバイス、39％

2. Android -7,600万台のデバイス、14.5％

3. iPhone -7150万台のデバイス、13.7％

4. Windows Mobile -6680万台のデバイス、12.8％

5. BlackBerry -65.25百万台のデバイス、12.5％

6. Linux -2,800万台のデバイス、5.4％

7. Palm （webOS）-1100万台のデバイス、2.1％
