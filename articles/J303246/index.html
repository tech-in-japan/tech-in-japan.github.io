Cisco Connected Safety and Security Physical Securityの概要

Cisco Connected Safety and Security（CSS）は、インシデントを防止、準備、対応、または復旧するソリューションを設計および構築するためのアーキテクチャ基盤です。











アーキテクチャは、インシデント管理、市と公安、国境管理、重要なインフラストラクチャと財産の保護、大規模な混雑と大規模なイベントの場所、輸送、司法刑務所システムなどの幅広いソリューションで使用される機能ブロック（図1）を定義します。 。





図1



Cisco CSSソリューションは、ソリューションコンポーネントの配置、システムのスケーラビリティへの柔軟なアプローチを可能にし、高レベルの管理システムを含む関連システムとの統合のためのオープンプラットフォームを提供するIPテクノロジー上に構築されています。



イベント収集



Cisco CSSアーキテクチャのイベントのソースは、CCTVカメラ、アクセス制御システムのアクセスポイント、およびその他のセンサー（IoTセンサーを含む）です。これらの統合は、カメラとACSコントローラーの乾接点またはアプリケーションレベルでの接続を通じて可能です。 イベントは、自動的に（組み込みのビデオ分析アルゴリズムのトリガーなど）、自動（たとえば、バッジをリーダーに提示する）、または手動で（たとえば、オペレーターのコマンドで強制的に）生成されます。

Cisco Digital Video Surveillance Rangeは、すべての一般的なビデオ監視アプリケーションを対象としています。 豊富な設定と動作パラメータのセットに加えて、カメラにはCisco Medianet機能を使用したIPネットワークとの高度な統合手段があります。 これらの機能は、監視カメラがIPネットワークに接続されている場合に監視カメラの自動検出と設定を提供し、設定中のオペレータのエラーを排除し、監視システムの展開を大幅に加速します。



シスコのカメラは、組み込みのビデオ分析もサポートしています。これは、既成のアルゴリズムセットに加えて、会社のテクノロジーパートナーが作成したアルゴリズムをダウンロードできます。 範囲には、すべてのタイプのカメラが含まれます：制御されたパンチルトおよび固定カメラ、ハウジングおよびドームカメラ、および車両への設置用に設計された専用カメラ。



アナログカメラを接続するために、4または8ポートのIPエンコーダーが提供されています。



ACS製品ポートフォリオは、IPテクノロジに基づく統合された物理的セキュリティネットワークに最終機器（ドアロック、ボタン、リーダー、連絡先、視聴覚アラート）を統合するIPゲートウェイによって表されます。



イベント分析



実行中のタスクに応じて、Cisco CSSソリューションでのイベントの分析は自動または自動プロセスです。 最初のケースでは、システムオペレータは、人間の介入なしに、着信イベントを処理するスクリプトを設定して、周辺機器にフィードバックを提供できます。



Cisco CSS Solutionは、次のコンポーネントで構成されています。



-ビデオ情報を処理およびアーカイブするためのソフトウェアCisco Video Surveillance Manager（VSM）。ビデオ、オーディオ、イベント情報の分散処理、ローカルディスクまたは外部ストレージへのビデオアーカイブの安全で信頼できるストレージ、指定されたモーションディテクターからのイベントのビデオ素材の高度な検索を提供します外部システムからの時間間隔またはイベント、オープンプログラミングインターフェイス（API）を使用した他のセキュリティおよび制御アプリケーションとの統合。



-アクセスコントローラ、周辺機器、およびユーザーアクセス権の制御を提供するCisco Physical Access Manager（CPAM）アクセス制御システム用のソフトウェア。 CPAMをCisco VSMと統合して、ACSからの信号をビデオ録画と同期できます。



-統合管理ソフトウェアCisco Physical Security Operations Manager（PSOM）。地理的に分散した組織を含め、組織の物理的セキュリティを確保する技術的手段のための単一の管理コンソールを提供します。 PSOMは、センサーからの情報を集約し、イベントを分析および相関させ、ユーザーが作成した情報処理スクリプトを実行します。



イベントへの反応



インシデントやイベントに対応するときに通信する標準的な方法は、音声を使用することです。 Cisco CSSソリューションを使用すると、標準のマルチメディアコミュニケーションツール（ライブビデオまたはビデオの再生、写真の表示、ビデオ会議への参加、インスタントメッセージング）を補完できます。

ユニットの主要コンポーネントは、Cisco IP Interoperability and Collaboration System（IPICS）ソフトウェアです。 その機能により、さまざまな無線通信システムのユーザーが相互に、および従来の携帯電話のユーザーと通信できます。



次の製品は、Cisco IPICSのコンポーネントです。



-Cisco IPICSサーバー。 ポリシー、トランキンググループ、無線チャネル、メディアリソース、ユーザー認証の管理に使用されます。



-Cisco IPICS Dispatch Console Dispatcher Console。 IPテクノロジーに基づいて、コンソールはスタッフに、従来のコントロールパネルの機能と新しい通信チャネル（モバイルデバイス、パーソナルコンピュータークライアント、テレフォニー）を単一の制御下で組み合わせる機能を提供します。



-モバイルデバイス用のクライアントCisco IPICS Mobile Client。 インシデントのあるビデオや写真を見るなど、リッチなマルチメディア機能と組み合わせてPTT機能を提供するスマートフォン用のアプリケーション。



-Cisco IPICS IP Phoneクライアント用のクライアント。 Cisco Unified IP Phones IP Phoneを使用して、プッシュツートークを介してスタッフと通信する機能。



-Cisco Universal Media Serviceサーバー。 彼は、Cisco IPICS管理サーバーから受け取ったポリシーと設定に基づいて、さまざまなオーディオチャネルを単一のブロードキャスト環境に接続する責任があります。



-P25ネットワークとTetra Peripheral Equipment Interface（PEI）のサポートを備えたゲートウェイ。 IPテクノロジーに基づいたネットワークと従来の無線ネットワークの接続点を提供します。



Cisco Connected Safety and Security Solutionsを使用する利点



-プラットフォームのオープン性により、エンドデバイス、関連システム、高レベル制御システムの統合が可能になります。



-IPテクノロジーの使用により、システムの実装と保守が容易になり、システムの機能が向上します。



-すべてのコンポーネントの仮想化により、透過的な成長と柔軟なシステム設計機能が提供されます。



-スケーリングは、要素を交換する必要なく、システムの成長に自信を与えます。
