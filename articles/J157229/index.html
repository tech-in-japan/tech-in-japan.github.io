特殊相対性効果のある3Dゲーム





マサチューセッツ工科大学には、コンピューターゲームラボ（MIT Game Lab）があります。 どうして彼女はそこにいるのか？ 医師は勤務時間中にどのようなゲームを開発できますか？ そして、彼らは何を示しました：超現実的な3DアクションA Slower Speed of Lightを満たします。 プレーヤーは、光の速度に近い速度で3D空間を移動し、球状のオブジェクトを収集します。各オブジェクトは、固定値だけ光の速度を遅くします。 同時に、特殊効果の相対性理論に従って、視覚効果がリアルタイムで計算されます。



計算された効果：



ドップラー効果 ：可視光の赤と紫のシフト、および人間の目に見える範囲の赤外線と紫外線のシフト。



光収差 ：進行方向の光強度が増加します。



相対論的時間膨張 ：プレイヤーと世界の時間の認識の違い、過去のようにオブジェクトを見る能力。



ローレンツ変換 ：光の速度に近い速度での空間の歪み。



このゲームは、アクセシブルなゲームプレイと理論物理学の分野における真剣な科学的基盤を組み合わせています。 ある意味では、これはそのような品質の組み合わせを持つユニークなゲームです。 高校の物理学のクラスの生徒、そしておそらく大学の生徒に見せることができます。 一般に、彼らはどこでも相対性理論の特別な理論を研究しています。



スクリーンショット


























ゲームは次の構成でテストされました。

  • Intel Core 2 Duo T9900およびCore i7（2.8 GHz）
  • Windows 7およびMac OS X 10.7（ライオン）
  • AMD Radeon HD 6970M、AMD Mobility Radeon HD 4850、Nvidia GeForce 9600M GT
  • 8 GB RAM


一部のユーザーは、2 GBのRAMを搭載したWindows XPでもゲームが起動すると報告しています。



ダウンロード：Windows│Mac







MIT開発者は現在、Unity3Dゲームエンジンの特殊な相対性理論の効果をシミュレートするために、OpenRelativityツールのセットに取り組んでいます。 ドキュメントとともに、2013年に無料ライセンスでリリースされます。
