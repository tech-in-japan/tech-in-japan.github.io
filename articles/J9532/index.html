HTC iPhoneクローンがヨーロッパで近日公開予定



台湾の電話メーカーNTSは、タッチスクリーンを備えたWindows Mobileベースの新しいGSMスマートフォンであるHTC Touchを発表しました。これは、将来のApple iPhoneのようなものです。



HTC Touchは、NTS T-Mobile Wingのような触感に優しいソフトペイント（黒または緑）で覆われているため、101 x 58 x 12.5 mmのコンパクトなサイズで、重量は113 gです。幅はほぼ2です。 4インチ画面ですが、ボタンのないiPhoneとは異なり、HTC Touchには画面の下にナビゲーションバーとボタンがあります。



Windows Mobile 6 Professional（タッチスクリーンを備えたデバイス向けのWindows Mobile 6のバージョン）に基づく新しいHTC TouchFloテクノロジーにより、指先を使用して多くの一般的なタスクを解決できます。 たとえば、ナビゲーションバーから画面の中央まで上にスワイプすると、連絡先のリストが画面に表示されます。 左に打たれると、マルチメディアコントロールボタンに置​​き換わります。 再び左-プレインストールされたアプリケーションへのショートカットのセットが表示されます。







iPhoneの類似性について尋ねられたNTSのスポークスマンJason Gordon（元マイクロソフトおよびモトローラの従業員）は、Steve Jobsが1月にMacWorldでスタイリッシュなAppleタッチスクリーンデバイスを発表したとき、Touchはすでに開発中であることを保証しました。 「そのような技術を開発するには何年もかかります」と彼は言いました。



Touchには組み込みの802.11b / gワイヤレスアダプターが装備されていますが、その中で動作すると、デバイスのリチウムイオンバッテリーに1100μA時間の負荷がかかり、NTSによれば、最大5時間の連続音声通信と200時間のスタンバイ時間が提供されます。



HTCは長い間Windows Mobile電話を製造してきましたが、これまではHP IPaq、T-Mobile Wing、Dash、Cingular 8525などの他のブランドで販売してきました。最近、Windows Mobile 5に基づくHSDPAデバイスであるHTC Advantageを発表しました。ミニチュアPCに似ています-今年の夏に発売されます。 また、夏後半または秋には、Windows Vistaで動作し、クアッドバンドGSM / GPRS / EDGEネットワークをサポートするUMPCデバイスであるHTC Shiftがリリースされる予定です。
