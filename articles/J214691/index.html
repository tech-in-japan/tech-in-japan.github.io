一連の無料のMicrosoft Exam Preparation Webセミナーが始まります



多くのIT専門家は、一生に一度以上認定試験に合格したか、少なくとも合格することを考えています。

試験中に何を期待しますか？ どんな質問になりますか？ それらを正しく解釈する方法は？ 準備する最良の方法は何ですか？ なぜこれが必要なのですか？

顧客からよくこれらの質問が寄せられるため、マイクロソフトの試験に備えて一連の無料ウェビナーを開催することにしました。



マイクロソフト認定資格の新しいラインの最初の試験の1つである70-410 Windows Server 2012のインストールと構成から始めます。

Alexei Kibkaloは、3月7日金曜日の12:00（モスクワ時間）にMicrosoft Exam 70-410の準備について話します 。 最近のマイクロソフトのアーキテクトであるアレクセイは、世界で最も認定された10人のマイクロソフトスペシャリストの1人です。

ウェビナーに登録するには、 n.orlova @ stars-s.ruに 「First Look 70-410」というテーマのメールを書いてください。



MCSA認定の詳細：Windows Server 2012



MCSA認定（Microsoft Certified Solutions Associate）：Windows Server 2012 —認定は、マルチタスクビジネス環境に適用できる基本的なWindows Server 2012スキルを持っていることを確認します。

MCSA Windows Server 2012に合格するには、次の3つの試験に合格する必要があります。



MCSA証明書は、スペシャリストに次の利点を提供します。

•マイクロソフト製品およびテクノロジの使用に関するプロ意識と知識の公式承認。

•マイクロソフト認定プロフェッショナル（MCP）Webサイトの安全な領域を介してマイクロソフトが直接提供する製品および技術情報へのアクセス。

•MCSA証明書、トランスクリプト試験、マップ、およびバッジを使用してMCPステータスを同僚やクライアントに確認します。 MCPプログラムのロゴとトランスクリプトを含むファイルは、ステータスを受け取った後、セキュアなMCP Webサイトからダウンロードできます。

•マイクロソフトが主催する技術専門家向けの会議やその他のイベントへの招待。



試験の費用と種類について



1）試験バウチャーにはいくつかの種類があります。

•割引-試験の購入を5-10-15- ... -100％の費用で節約できます。

•セカンドショット-このバウチャーでは、試験を1回無料で再受験する権利が与えられます（ MicrosoftのWebサイトでプロモーションが行われます ）。 再受験は翌日より早くない場合があります。

2）ウェブサイトregister.prometric.comでは、各試験の基本費用は80ドルで、クレジットカードで支払うことができます。 ここで、興味のある配達の日時を選択することもできます。

3）多くのCTでは、試験は現金または銀行振込で支払うことができます。

4）私たちのCT（モスクワの「星とC」）では、試験の費用は3500ルーブルです。 有料で試験を受験する場合は、3月にSS500offプロモーションコードを使用し、3000rの2回目のチャンスがあるマイクロソフト試験バウチャーを取得してください。
