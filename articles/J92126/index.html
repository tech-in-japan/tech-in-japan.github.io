ニジニノヴゴロドのLinux InstallFest 10.04



5月3日、ニジニノヴゴロドで行われる祝祭の月曜日に、本物のLinux InstallFestが5回開催されます。 Nizhny Novgorod Linux User Groupは、Nizhny Novgorod RadioTechnical Collegeの支援のおかげで、このイベントを6か月ごとに大規模に開催しています。



今年はNTRKジムでイベントが開催されます。 誰もが、配備されたUbuntu 10.04リポジトリとインターネットを使用して、椅子、コンセント、ネットワークアクセスを取得できるようになります。 インストールおよび設定中に質問がある場合は、NNLUG参加者の中からコンサルタントが常に助けてくれます。今年はさらに多くの参加者がいます。



まだ決定しておらず、Linuxについて学びたいだけの人のために、プレゼンテーションエリアでセミナーが開催されます。 さまざまな環境で作業して、インストールプロセスとLinux設定について見て、話すことができます。 日中は、さまざまな無料アプリケーションのプレゼンテーションが行われます。 今年は学校に焦点を当て、教育におけるLinuxの使用について多くのことを話します。 さらに、Nizhny Novgorod RadioTechnical Collegeの教育プロセスでLinuxを使用した2年の経験により、この方向での興味深い開発の可用性が保証されます。



成熟して仕事でLinuxを試してみるときは、Ubuntu 10.04とSchool Linux 5.0がインストールおよび構成されたすぐに使えるコンピューターがあるデモゾーンを見てみましょう。 ここでは、セミナーで聞いたことすべてを試して、コンサルタントに質問することができます。



まあ、「pogamat」をしたい人のために-ゲームエリアが開かれます。 OpenArena、UrbanTerrorおよびその他のエキサイティングな無料ゲームでのGNU / Linuxcenterからの賞品のコンテストが保証されています。 今日から練習を始めましょう！



5月3日にみんなを待っています
