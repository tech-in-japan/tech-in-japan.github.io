ガーディアンチームのプレゼンテーション

セレクションのファイナリストの一人は、モスクワのキーパーチームです。 チームは、正常に実装されたプロジェクトを多数所有しています。 特に、グループはスイマーの生理学的状態を遠隔監視するためのデバイスを開発しました。これは、2008年のオリンピックに向けてロシアチームを準備する際に使用されました。 ストレス状態の遠隔監視およびリハビリテーション用のデバイスのプロトタイプも開発され、スマートフォンに基づいて人体の状態を監視するためのデバイスが開発されました。







チームによって提案されたMDUコンセプトは、専門家の参加なしで使用できる低コストで一般的にアクセス可能な診断装置の開発を目的としています。 同時に、新しいセンサーの組み込みと新しい疾患の診断の両方のために、コンセプトはオープンで拡張可能である必要があります。 これらの規定のうち2つは明らかに衝突し、次のように解決することが提案されています。MDOセンサーをいくつかの個別に着用可能なデバイス （ ONU ）にグループ化する。 各ONUのセンサーの構成は、センサーの近接条件とONUの機能的完全性に基づいて選択されます。 ユーザーは、使用目的に応じて1つ以上のONUを選択して使用できます。



プロジェクトONUでの開発の提案：



  1. 手首の袖口 。 温度センサー; 3軸加速度計; 脈波マイクロフォン; 血中酸素化のレベルを決定する光学ペア（LED光センサー）; レオグラフィック割り当ての電極。
  2. ヘッドバンドバンダナ 。 温度センサー; 3軸加速度計; フォトプレチスモグラム記録用の光学ペア（LED光センサー）; レオグラフィック割り当ての電極。
  3. ECGベスト 。 含まれるもの：ECGセンサー-3〜6リード。 3軸ジャイロスコープ; 3軸加速度計; 磁力計。
  4. 肘の袖口 。 含まれるもの：圧力センサー。 パルスマイク; 膨張式カフ; マイクロポンプ; 温度センサー; ひずみゲージセンサー。
  5. 耳クリップ 。 含まれるもの：血液の相対密度を測定するための2つの光学ペア（LEDフォトセンサー）。
  6. 肺活量計 含まれるもの：吸入-呼気中の気流速度センサー。


BlueTooth4.0（LE）インターフェースを使用する各ONUは、データ処理を実行し、通信機能を提供するスマートフォンまたはタブレットコンピューターにセンサーからデータを送信します。 データはパケットで送信されるため、異なるONU間の衝突がなくなります。



MDUのシンプルで使いやすいインターフェイスは、スマートフォンまたはタブレットのコンピューティングおよびグラフィックス機能を最大限に活用して構築され、ユーザーの操作を最小限に抑えます。
