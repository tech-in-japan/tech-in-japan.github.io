最新のiPhone OS 3.0の詳細な説明

はい、すでに投稿があることは知っていますが、もう少し詳細な要約を書きました。



さらに、真のヤブロマンとして、私は宗教から何か言いたいことがあります:)



だから：消費者と開発者のための新機能の説明！



まだ終わっていないプレゼンテーションの新しい道をたどって、彼は何が変わったのか詳細な物語を書くように私に頼みました。



要するに、多くのことが変わりました。 この投稿では、新しいエンドユーザー機能について、そして次の記事では、開発者向けの変更について説明します（ただし、これはエンドユーザーにとっても重要です）。



だから。 いくつかの変更は皆を待っていました。 パート-と期待していませんでした。 発表順：

  • 待望のカット/コピー/ペースト。 Appleは、タッチスクリーンデバイスにインターフェースを実装するのはそれほど簡単ではありませんが、これをなめるのに多大な労力を費やしました。 一言で言えば、それは判明した。 便利で、美しく、一般的にどこでも動作します。 iPhoneを振ると、テキストの挿入/切り取りはキャンセルされます:)
  • 電子メールで一度に複数の写真を送信できます。
  • すべての主要なAppleアプリケーションで、ランドスケープ（ランドスケープ）モードと対応するキーボードが追加されました。 手紙、SMS、メモを書くほうがずっと便利です。
  • 個々のSMSを管理します。 それらは削除でき、転送できます。
  • MMS 写真、音声メモ、地図座標、連絡先の送信。
  • 実際、音声メモ。 ボイスレコーダー。 編集する機能を備えた、十分に高度な。
  • CalDavおよびICSカレンダーサブスクリプションのサポート。
  • iPhone全体のグローバル検索。 驚くほど作りました！ 文字、音楽、ビデオ、SMS、MMS、何でも検索できます。 非常に高速で便利な検索アクセス。 スポットライトと呼ばれる:)
  • A2DPをサポートします。 簡単に言えば-ステレオBluetoothヘッドセットのサポート。 さらに簡単-フルワイヤレスヘッドフォンのサポート。
  • メモを同期
  • Wi-Fi自動ログイン。
  • Safariの自動入力フィールド
  • 通常の通話記録
  • 暗号化されたプロファイル
  • Youtubeビデオを購読する
  • LDAPサポート
  • ライブストリーミングのオーディオとビデオをサポート
  • カレンダー会議の招待状を作成する




ここで多くの写真を見ることができます： live.gdgt.com/2009/03/17/live-iphone-os-3-0-event-coverage/#more-300



このすべては、この夏ユーザーに無料で提供されます。



しかし、もっとクールなのは、開発者に許可される他の何かです。



この面では、すべてがはるかに興味深いです。 Appleは開発者に誰も望んでいないものを許可しました。 そして、夏までに作成されるソフトウェアを見るのは非常に興味深いでしょう。



発表順：

  • 新しい支払い方法：毎月のサブスクリプション、アプリ内購入。 これで、アプリケーション自体がユーザーからお金を得ることができます。 多くの用途があります：本を販売するブックリーダーアプリケーション。 アイテムを購入できるマルチプレイヤーゲーム。 最終的には、iTunesアカウントからお金を使ってポーカーをプレイできるポーカークラブです。
  • ピアツーピア接続をサポートします。 簡単に言えば、マルチプレーヤーゲームを作成できるようになりました。このゲームでは、Wi-Fiルーターなどは必要ありません。 通常のBluetooth、および周囲の他の人の絶対的な自動検出 しかし、もちろん、これはゲームだけではありません-連絡先の転送、お互いの写真、楽器の共同演奏、一般に-情報の交換。
  • 非常に大きなこと：アクセサリーへのオープンアクセス。 iPhone用の外部スピーカーを製造するとします。 美しいイコライザーを使用してアプリケーションを作成できます。 別の例は、血糖値を測定するためのアクセサリです。 3番目は、洗練されたソフトウェアインターフェイスを備えたFM受信機です。 などなど。
  • Googleマップをアプリケーションに埋め込む機能。 独自のデータを一番上に置くことができます。
  • GPSナビゲーターを実装する機能。 残念ながら、Googleマップ上ではありません-彼らはそのようなナビゲーションのためにそれらをライセンスしていません
  • 待望のプッシュ通知。 サーバーは、イベントについてユーザーに通知できます。 これは、新しいICQメッセージの通知である可能性があります。 サッカーの試合の様子に関するメッセージ。 クラスメートの新しいメッセージに関するメッセージ。 そして、これは対応するプログラムを起動することなく行われます。 音楽を聴くだけで、対応するイベントが発生したとき（2分、5分、または10分に1回ではなく）にこれらすべてのアラートを取得


また、 何百もの異なるSDKが追加されましたが、これについては一言も述べられていません。 ただし、リスト（完全ではない）は表示されます。 最も興味深い：

  • バッテリーAPI
  • アプリケーションで電子メールを操作する
  • 距離センサーアクセス
  • オーディオ/ビデオのストリーミング
  • Shake API
  • インターフェイスを操作するための多くのAPI：ツールバー、テキスト選択、アラートなど。
  • そして、どうやら、非常に、はるかに




プレゼンテーションでは、2週間以内にAppleの要請で開発者が開発したアプリケーションが示されました。 本格的なオンライン3Dシューティングゲーム、Sims3でジャンクを購入してお金を稼ぐ、トロンボーンデュエット、糖尿病患者向けの非常に洗練されたプログラム、血中グルコースを測定するアクセサリー、面白い瞬間をすぐに見ることができるフットボールゲームの最新情報をお届けするプログラムビデオで。 などなど。



結論：Windows Mobile、Android、およびその他のモバイルプラットフォームの開発者の代わりに-私はまず落ち込んでから、ハンドヘルドからあらゆる種類のサブノートなどに再訓練されました。 WMスマートフォンには未来はありません。 Androidにはまだありません。 Symbianはサポートしていません。 たぶんAndroidは他のいくつかのデバイスで何かをするでしょう。 WMは埋めることができます。 はい、すでに。



Appleプラットフォーム、アクセサリメーカー、ソフトウェア実装の利便性、プラットフォームの統一性の開発者の世界の統一性を考えると、iPhone OS向けの開発のさまざまな利点を考えると、他のプラットフォームの専門的な開発は最後の魅力を失います。



Appleは最後の一歩を踏み出す必要があります：AppStoreを根本的に更新し（プレミアムセクション、製品を宣伝するためのより深刻な機会、試用アプリケーションの良いメカニズム）、新しいiPhoneモデルをリリースし、できれば範囲をわずかに拡大します。 今年も同様のことが行われると確信しています。
