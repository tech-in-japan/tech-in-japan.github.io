Windows 7のログイン画面で背景画像を変更する

以前に発表したように、新しいWindows 7オペレーティングシステムでは、ログインウィンドウの背景を簡単に変更できます。 これを行うには、実行可能ファイルのリソースを変更したり、他の特別なアクションを実行したりする必要はありません。 画像を変更するプロセスは非常に簡単で、追加のツールを使用しなくても実行できます。 この機能はビルド7057以降使用可能であり、正しく機能します。



















ログイン画面の背景を変更するには、次の手順を実行する必要があります。



  • レジストリエディターを開き、レジストリキー「HKEY_LOCAL_MACHINE->ソフトウェア-> Microsoft-> Windows-> CurrentVersion->認証-> LogonUI->バックグラウンド」を見つけます。 このセクションには、DWORDタイプの特別な「OEMBackground」キーが含まれています。 この静脈の値「1」は、ログインウィンドウの特別な設計を使用する必要があることを示しています。 キーが見つからない場合は、手動で作成できます。
  • 「％windir％\ system32 \ oobe \ info \ backgrounds」フォルダが存在することを確認してください。 これは、Windows 7がログイン画面の背景画像を探す場所です。 このフォルダーが存在しない場合は、作成する必要があります。
  • このフォルダに必要なすべての画像を入れます。 このフォルダ内の各許可について、特定の名前で画像を配置する必要があります。 目的の解像度の画像が見つからない場合、backgroundDefault.jpgファイルが使用されます。 各画像は256 KB以下であることが重要です。それ以外の場合は、単に無視されます。 このフォルダーに入れるファイルのリスト：

    • backgroundDefault.jpg
    • background768×1280.jpg（0.6）
    • background900×1440.jpg（0.625）
    • background960×1280.jpg（0.75）
    • background1024×1280.jpg（0.8）
    • background1280×1024.jpg（1.25）
    • background1024×768.jpg（1.33）
    • background1280×960.jpg（1.33）
    • background1600×1200.jpg（1.33）
    • background1440×900.jpg（1.6）
    • background1920×1200.jpg（1.6）
    • background1280×768.jpg（1.66）
    • background1360×768.jpg（1.770833）


これらのすべてのアクションを手動で実行したくない場合は、これらの目的のために、インターネット上に十分な数のサードパーティユーティリティを使用できます。 そのような最も単純なユーティリティの1つは、 Windows 7用のTweaks Logon Changerです 。 このユーティリティは、レジストリに必要なすべての変更を加え、イメージを目的のサイズに圧縮して適切なフォルダーに入れます。
