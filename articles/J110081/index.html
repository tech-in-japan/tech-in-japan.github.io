API Mail.Ruでビジネスを始める

こんにちは、habrauzers、



今年、スタートアップ向けのイベントが増え、より頻繁に開催されるようになりました。 今年の終わりは文字通りマラソンでした。「ワーキングウィークエンド」形式のイベントが、週末ごとに国のさまざまな都市で開催されました。



Platform.Mail.Ruプラットフォームチームは、今年の最終イベントであるGet Startedに参加します。



セッションは、モスクワ国際ビジネス高等学校MIRBISで12月17〜19日に開催されます。



起業家セッションの主催者によると、Start Mail.Ruテクノロジーはほとんどのスタートアップにとって理想的です。 そのため、彼らはイベントを「API Mail.Ruでビジネスを開始」と呼びました。 Mail.Ru API機能は既に使用しています

  • ポータル（ポスター、キノポイスク）
  • ビデオプロジェクト（IVI、Tvigle）
  • プロモーションプロジェクト（G-menライブショー）
  • ゲーム（Sky2Fly）
  • 技術プロジェクト（AZ、Imkhonet、Firmbook）
  • サービス（Loginza、Fotorotor）
  • ソーシャルネットワーク（ClassNet、Photostrana）
  • スタートアップ企業ではなくなりました（AlterGeo、Darberry）


My Worldのモバイル版の新しいアプリケーション市場、Mailのアプリケーションについて説明し、アイデアのブレーンストーミングを行います。 すべてのプロジェクト代表者に、当社のテクノロジーを使用して、スタートアップやビジネスモデルの視聴者と効果を高める方法を示します。



そしてファイナルでは、最高の参加者に貴重な賞品を提供します。



12月17日金曜日、モスクワ国際ビジネス高等学校MIRBISの住所、Marxistskaya、34、building 7でお待ちしています。 18.00にセッションの開始、遅れないでください。 :)



Aleksey Terekhov 、Platform Mail.Ruのヘッド、 Alexander Dorzhiev （AlterGeo）、 Ilya Remizov （A-y-yay）からのビデオ招待状をご覧ください 。
