HTC One maxが正式に発表されました

Oneスマートフォンのラインには、新しい代表者が正式に補充されました-5.9 "HTC One max！HTC One maxは、HTC Oneのオリジナルデザインと、1.7 GHzクアッドコアプロセッサを備えたQualcomm Snapdragon 600ハードウェアプラットフォームを継承しました。 Gorilla Glass 3耐衝撃ガラスコーティングを使用したSuper LCD3テクノロジー。



2 GBのRAMに加えて、デバイスには16 GBの内部メモリとmicroSDメモリカード用のスロットが搭載されています。 さらに、One maxのすべての所有者には、2年間Googleドライブオンラインサービスで50 GBが提供されます。 また、3300 mAhのデバイスのバッテリー容量に注意する価値があります。これは、取り外し可能なカバーでさらに1150 mAhだけ拡張でき、デバイスの容量はほとんど増加しません。









HTC One maxには、BSIセンサーと5段階の強度のフラッシュを備えたウルトラピクセルカメラがあります。 デバイスの前面2.1 MPxカメラには広角レンズが装備されており、HDRを撮影できます。 大型ディスプレイと組み合わせて、スマートフォンのマルチメディア機能は、HTC BoomSoundテクノロジーをサポートするフロントステレオスピーカーを補完します。 デバイスの上部にある赤外線ポートに加えて、ワイヤレスインターフェイスは、NFC、Bluetooth 4.0、WiFi 802.11 ac / n / g、およびLTEです。



技術仕様












スマートフォンの最も興味深い革新の1つは、おそらくデバイスの背面にある指紋スキャナーです。 スマートフォンソフトウェアを使用すると、3つの指紋の定義を構成し、電話のロックを解除するかカメラを起動するかにかかわらず、アクションの1つを反応として設定できます。 スキャナーを有効にするには、目的の指で上から下にスワイプする必要があります。



スマートフォンのエルゴノミクスに積極的な変化が生じました。最も使用されているものの1つである電源ボタンは、音量ロッカーの下の右側に移動しました。 背面カバーは取り外し可能で、デバイスの左側にあるボタンで開くことができますが、バッテリーは使用できません。 このクラスのスマートフォンに適した寸法は控えめではありません：164.5 x 82.5 x 10.29 mm、重量は217グラムです。



カバーPower Flip Caseは、一種のスタンドとして機能するだけでなく、スマートフォンのバッテリーに追加容量を提供します。 さらに、実際にはボリュームは追加されません。 バッテリーはケースの前面カバーにあり、柔軟性があります。 ケースはスマートフォンの充電中に充電され、日常の使用中にデバイスを充電します。







スマートフォンはAndroid 4.3およびHTC Sense 5.5で動作します。 販売の開始は10月中旬に予定されており、価格は追加で発表されます。
