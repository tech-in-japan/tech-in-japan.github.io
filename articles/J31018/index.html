振り返ってみると、1975年から1989年までのMicrosoftの歴史。

1975



1月1日 MITS Altair 8800の写真がPopular Electronicsの表紙に掲載され、ポール・アレンとビル・ゲイツにAltairのBASIC言語を開発するよう促しました。



2月1日 Bill GatesとPaul Allenは、最初のパーソナルコンピュータ開発言語プログラムであるBASICをMicrosoftの最初の顧客であるMITSに販売しています。



3月1日 Paul Allenは、MITSのプログラムディレクターとして働き始めました。



4月7日 。 「Altair BASIC-通常のフライト」（「Altair BASIC-稼働中」）-「MITS Computer Notes」の最初の号のタイトルを読んでください。



7月1日 4Kおよび8KバージョンのBASIC 2.0の公式販売が開始されます。



1976



2月3日 ビル・ゲイツは、ソフトウェア著作権侵害の問題を提起した最初のプログラマーの一人です。 MITS Computer Notesで最初に公開された「愛好家への公開書簡」で、ゲイツはこれらの人々がソフトウェアを盗んだと非難し、「良いプログラムを書くこと」を妨げたと言った。 彼は無意識のうちに予言を表明し、「10人のプログラマーを雇い、アマチュア市場に良いプログラムを殺到する機会以上に私を幸せにすることはできない」という言葉で手紙を締めくくった。



3月27日 ビルゲイツは、アルバカーキで開催された第1回Altair World Computer Meetingでオープニングスピーチを行います。



11月1日 Paul AllenはMITSを退職し、マイクロソフトのフルタイムチームに参加します。



11月26日 。 ニューメキシコ州務長官の事務所はマイクロソフトの登録商標です。



1977



2月3日 ポール・アレンとビル・ゲイツは正式なパートナーシップを結びます。



7月1日 マイクロソフトの2番目の開発言語であるFORTRAN-80が登場します。



1978



11月1日 マイクロソフトは、Microsoft ASCIIと呼ばれる日本で最初の国際営業所を開設しました。



12月31日 マイクロソフトの年間売上は100万ドルを超えています。



1979



1月1日 マイクロソフトの本社はアルバカーキからワシントン州ベルビューに移転しました。



1980



6月11日 。 スティーブバルマーはマイクロソフトで働き始めました。



1981



6月25日 。 マイクロソフトは、ビルゲーツを社長兼会長に、ポールアレンを執行副社長に迎え、非公開企業に再編しています。 マイクロソフトは、ワシントンの法人であるマイクロソフト社になります。



8月12日 。 IBMは、Microsoft BASIC、Microsoft COBOL、Microsoft Pascal、およびその他のMicrosoft製品とともに、Microsoftの16ビットオペレーティングシステムMS-DOS 1.0を使用する「パーソナルコンピュータ」を市場に導入しています。



1982



3月24日 。 Microsoft UK Ltd.が登録されています （イギリス）。



6月28日 マイクロソフトは、新しい企業ロゴ、新しいパッケージング、およびディーラーのテクニカルサポート用の資料一式を発表します。



1983



2月18日 。 ポール・アレンはエグゼクティブ・バイス・プレジデントを辞任しますが、引き続き取締役会に残ります。



5月2日 MicrosoftはMicrosoft Mouseをリリースします。



9月29日 。 MicrosoftがMS-DOS 1.00用のWordをリリース



11月10日 。 Microsoftは、グラフィカルな作業環境を提供するMS-DOSオペレーティングシステムの拡張であるMicrosoft Windowsをリリースする計画を発表しています。 Windowsの最初の小売販売は、わずか2年後の1985年11月に始まります。



1984



1月24日 。 マイクロソフトは、Macintoshの導入と同時にBASICとMultiplanの販売を開始し、Appleコンピューターのソフトウェア開発のリーダーになりました。



7月11日 。 Microsoft Pressは最初の2冊の本をリリースしています。CaryLiuによる「Apple Macintosh Book」とPeter Nortonによる「Exploring IBM PCjr Home Computer」です。



1985



8月12日 。 マイクロソフトは10周年を祝っており、年間売上高は1億4000万ドルです。



9月3日 。 マイクロソフトは、ヨーロッパ市場で販売される製品を発売する米国外の最初の製造センターとしてアイルランド共和国を選択しました。



11月20日 。 MicrosoftはMicrosoft Windowsの販売を開始します。



1986



2月26日 マイクロソフトはワシントン州レドモンド市に移動します。



3月13日 。 マイクロソフトの株式は、1株当たり21ドルの初期価格で証券取引所で取引され始め、1日目の終わりには28ドルに上昇します。 最初の提供の結果として、Microsoftは6100万ドルを受け取ります。



1987



4月2日 MicrosoftとIBMがOS / 2を発表 これは、1985年8月にMicrosoftとIBMが「共同開発契約」を結んだ最初の製品でした。



9月8日 。 Microsoftは、最初のCD-ROMアプリケーションであるMicrosoft Bookshelfの販売を開始しました。これは、1枚のCDに収められた10の最も人気のある便利なガイドのコレクションです。



1988



1月13日 。 MicrosoftとAshton-Tateは、Sybaseのリレーショナルデータベース管理システムに基づいたLANサーバー用のリレーショナルデータベースであるMicrosoft SQL Serverを発表しました。



1989



8月1日 マイクロソフトは、Macintosh用の最初の完全なビジネスアプリケーションであるOfficeをCDでリリースします。



11月13日 。 マイクロソフトとIBMは、1990年代にシステムソフトウェアで協力することに合意することで、開発契約の境界を押し広げています。 これには、Intel 386および486プロセッサと互換性のあるMS-DOS、Microsoft OS / 2、およびMicrosoftのLANソフトウェアの改善が含まれます。



12月27日 マイクロソフトは、1990年6月30日、 ジョンシャーリーが社長兼最高執行責任者を辞任したことを発表しました。 1983年8月からこのポストで働いているShirleyは、取締役会のメンバーおよび戦略的プロジェクトのコンサルタントとして会社で働き続けます。
