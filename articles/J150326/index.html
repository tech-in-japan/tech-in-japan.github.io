Firefox 15リリース

通常どおり、Firefoxの安定バージョンのリリースは、 8月28日（火曜日）に行われる公式発表の前に表示されます。

このバージョンの新機能：



  • メモリ消費の別の最適化は、MemShrinkプロジェクトの一部として実行されました。 今回は、ブラウザにインストールされたアドオンのメモリ消費量を削減することを目指しました

  • 組み込みのPDFドキュメントビューアー-PDF.jsを追加しました。 ビューアは、ページのサムネイルのスケーリングおよびナビゲーション機能をサポートし、TrueTypeおよびType 1フォントの読み込みを提供し、グラフやチャートを作成でき、圧縮オブジェクトの操作をサポートします。

    デフォルトでは、PDF.jsはバージョン15で無効になっています。 この機能をテストする場合は、config：でpdfjs.disabledパラメーターを見つけてfalseに切り替えます 。
  • WindowsでのFirefoxの起動速度の改善 。
  • Opusオーディオコーデックサポートの実装。
  • SPDYプロトコルの3番目のバージョンのサポートが追加されました 。

    デフォルトでは、このプロトコルのサポートはバージョン15で無効になっています。 有効にするには、「config」に移動し、 network.http.spdy.enabled.v3パラメーターを見つけてtrueに切り替えます 。
  • 本格的なJavaScriptデバッガーが開発者のツールに統合されています（Ctrl + Shift + S）。
  • 「Adaptive Design」 ツール （Ctrl + Shift + M）が追加され、さまざまな画面解像度のさまざまなタイプのデバイスでWebページを表示できます。
  • 「インスペクター」（Ctrl + Shift + I）で、ページ上の要素のレイアウトを表示するための新しいモードが追加されました。
  • WebGLでの作業の改善：パフォーマンスを改善するために圧縮テクスチャの使用を実装しました。
  • HTML5オーディオ/ビデオ要素を再生するためのバックエンドとしてのGStreamerマルチメディアフレームワークのサポート。
  • オーディオ要素とビデオ要素は再生属性をサポートし、 ソース要素はメディアをサポートします 。
  • Performance APIの一部として、高精度イベントタイマーのサポートが追加され、1000分の1秒の精度で時間を知ることができます。
