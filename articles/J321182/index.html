Swiftでは、これは少し簡単です。プロトコルとその拡張機能で十分です（拡張機能では、デフォルトでいくつかのプロトコルメソッドを実装できます）。

 // Swift class MallardDuck: Duck { func display() { } } class RedheadDuck: Duck { func display() { } }
