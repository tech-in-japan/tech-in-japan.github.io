OpenCLアプリケーション用の新しいインテルSDK

OpenCLアプリケーション用の無料のインテルSDK-2013 R2の新しいバージョンがリリースされました 。 革新のリストはここにあります。

  • Windows 8.1のサポート。
  • Intel HD Graphicsを搭載した新しいIntel Bay Trailプロセッサ、すなわちIntel Atom Z3000シリーズ、Intel Celeron J1000 / N2000、Intel Pentium J2000 / N3000のサポート。
  • Intel VTune Analyzer XE 2013を使用したIntel統合グラフィックスでのOpenCLカーネルトレース。
  • 新しいOpenCL拡張機能cl_accelerator_intel 、 cl_intel_motion_estimationには、ハードウェアアクセラレーションの組み込みモーション推定カーネルがサポートされています。


現在、Windows 8.1は、Windows 8.1 （バージョン15.33.5）用のIntel IrisドライバーおよびHDグラフィックスドライバー、Intel Bay Trailプロセッサー用のWindows 8.1 （バージョン15.33.3）用のIntel HDグラフィックスドライバーのみでサポートされています。

Windows 7および8ユーザーは、以前のリリースを使用する必要があります。

さらに、 OpenCL SDKサイトが更新されました。情報とビデオが追加され、新しい「トレーニング」タブが表示されました。



OpenCL Applications 2013 R2 for Windows 8.1用の無料のインテルSDKをダウンロードします。
