MSDNサブスクリプションは、ソフトウェア会社や個人の開発者、ITプロフェッショナルに多くの機会を提供します。 このような機会の1つは、ソフトウェア開発とテストの目的で、クラウド施設とWindows Azureクラウドプラットフォームのリソースへの低価格での優先アクセスのための無料の月額融資です。 スタートアップ向けのMicrosoft BizSparkプログラムのメンバーはさらに多くを獲得します-サブスクリプションとWindows Azureの商用利用のクラウドパワーを利用する権利があります！



MSDNサブスクライバー向けWindows Azure Platformの利点



MSDNサブスクリプションでVisual Studio Professional、Premium、またはUltimateを所有している場合は、MSDNの特典を今すぐアクティブにして、Windows Azureでアプリケーションの開発とテストを開始してください。 クラウドアプリケーションの構築、仮想マシンでの既存のアプリケーションのテスト、またはデータセンターとWindows Azureプラットフォームでホストされるハイブリッドアプリケーションの作成。 MSDNサブスクリプションに含まれるWindows Azureローンを使用して、テスト機器を取得するコストを排除します。 インスタントトレーニングを活用して、アプリケーションの開発とテストを高速化します。 エラスティックスケーリングを使用して、実際のシナリオをテストします。 MSDNサブスクライバー向けWindows Azureの利点は、成長するニーズに対応します。



Windows Azureクレジットによる柔軟性 。 Windows AzureのMSDNサブスクリプションでは、1か月あたり最大5,000 RUBのローンを取得できます。 Windows Azure ローンの使用方法を選択します 。 仮想マシン、Webサイト、クラウドサービス、モバイルサービス、ストレージ、SQLデータベース、キャッシングサーバーのネットワーク、HDInsight、メディアサービスなど、ニーズに応じて、Windows Azureサービスに使用します。



低価格での仮想化 。 Windows Azure仮想マシンで展開とテストを加速します。 これで、MSDNサブスクライバーが時給を割引にしたため、ローンがさらに節約されます。 Windows Azure がリソース使用の分単位の請求を使用しているという事実を活用してください！ それだけではありません。 仮想マシンを停止すると、仮想マシンの請求はすぐに停止します 。



MSDNソフトウェアの追加料金はありません 。 SQL Server、BizTalk Server、およびMSDNサブスクリプションに付属するその他のソフトウェアを使用して、事前に構成された仮想マシンイメージにアクセスします。 または、MSDNソフトウェアを使用して独自の仮想マシンをダウンロードします。 SQL ServerなどのWindows Server上で実行されるMSDNサブスクリプションソフトウェアに追加料金は発生しないため、Windows Azureのローンは長持ちします。



Windows Azureはオープンプラットフォームであり、Microsoftや他のメーカーのさまざまなオペレーティングシステム、言語、ランタイムを使用して、MSDNサブスクリプションを使用してアプリケーションを作成できます。



注意してください！ MSDNサブスクライバー向けのWindows Azureの利点は、開発とテストのみを目的としています 。 120時間以上継続的に実行されるインスタンス（VMまたはクラウドサービス）を一時停止する権利、またはインスタンスがワークロードに使用されていると判断した場合、当社はその権利を留保します。 ワークロードは、通常のサブスクリプションを使用して完了する必要があります。



2分でWindows Azureをアクティベートする方法は？



MSDNサブスクライバーの利点の1つは、クレジットカード番号を入力および確認する必要がないことです。 MSDNサブスクリプションの一部としてWindows Azureをアクティブ化するには、電話のみが必要です。



http://www.windowsazure.com/en-us/pricing/member-offers/msdn-benefits/にアクセスして、MSDNサブスクライバー向けのWindows Azure情報ページにアクセスし、[ アクティブ化 ]ボタンをクリックするだけです。



サブスクリプションが関連付けられているアカウントでログインし、電話番号を入力します。







SMS確認（または必要に応じて個人電話）で番号を確認し、[購入]をクリックします。 これで、Windows Azureのアクティベーションが完了しました。 すべてが非常に簡単です。



サブスクリプションを管理するためのポータルに移動し、そこで各サブスクリプションを管理できます。







「このサブスクリプションのコストには制限があります。」という通知に注意してください。 この通知は、サブスクリプションに使用のための特定の無料制限が設定されていることを意味し、それを超えることはできません。 この制限により、Windows Azureで安全に作業することができ、1ペンスのお金がお客様から償却されることはありません。



マウスでクリックして、特定のサブスクリプションの設定に移動します。







ここでは、無料のWindows Azure容量の実行コストと、MSDNサブスクリプションに割り当てられた毎月の助成金に関する詳細情報を見ることができます。 情報は、次のローン更新までの日数と、現在の期間に残っている現金ローンについて入手できます。 以前の費用に基づいてシステムが提供する大まかな予測を理解し、現在の月の残りの期間に十分な現在の資金とローンがあるかどうかを理解できます。



MSDNサブスクライバーの優待価格



開発およびテストプロセスを編成する場合、返済不能なローンの一部として割り当てられた容量よりも多くの容量が必要になる場合があります（最初の月は6,600 r、以下は最大5,000 r）。 この場合、費用の最大額を削除し、無料で提供されている容量を超えて必要な容量の支払いモードに切り替えることができます。 「費用の最大額を削除する」ボタンをクリックし、表示されるウィンドウで、無料の月額制限を超えて使用するリソースの支払いに使用されるクレジットカード情報を入力します。







これで、無料で割り当てられたものを超える特別な低価格でクラウドパワーを使用できます。



MSDNサブスクライバーには、ソフトウェア開発およびテスト用の特別な低価格のコンピューティングパワーが提供されます。 MSDNサブスクライバーは、Windows仮想マシンで33％、クラウドサービス、HDInsight、予約済みWebサイトで25％の割引を受けます。







Microsoftサーバー製品のライセンスはMSDNサブスクリプションの特典に含まれており、開発およびテスト目的で無料であるため、サブスクライバーは最終的に最大97％節約できます。







MSDNサブスクリプションおよびWindows Azureの利点の一部として、サブスクライバーはWindows Server、SQL Server、SharePoint ServerまたはBizTalk Serverを搭載した仮想マシンをこれらの価格でいくつでも展開できます。



スタートアップとBizSpark



MSDNサブスクリプションを使用するための追加機能は、Microsoft BizSparkプログラムに参加しているスタートアップです。 Windows Azure for BizSparkプログラムのサブスクリプションは、開発とテストだけでなく、商用運用でのソリューションの起動にも使用できます！



どんなスタートアップでもBizSparkプログラムに参加して、プログラムのすべての利点を活用できます。 提供されたリンクにリクエストを残すだけです。



効率的で安価なクラウドテスト



まとめると、メリットに対する新しいアプローチとサブスクライバーのコンピューティングパワーの価格の劇的な削減により、MSDNユーザーはサブスクリプションをより効率的に適用し、社内のテストプロセスを整理するコストを大幅に削減して、安価なクラウド環境に移行できると言えます。



開発者は、以前に導入されたWindows Azureの1分あたりのコンピューティング能力の請求とともに、投資とメンテナンスを必要とする独自の高価なインフラストラクチャを放棄し、Windows Azureクラウドでソフトウェアをテストするプロセスに切り替えることで、社内のテストプロセスのコストを大幅に削減できます。



BizSparkの新興企業は、ソリューションをさらに安価にクラウドに配置する追加の機会を得ます。
