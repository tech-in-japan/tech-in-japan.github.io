IBMとNvidiaは、深層学習ニューラルネットワークの速度を向上させます







一部の専門家は、ニューラルネットワークの背後に人類の未来があると言います。 または、すべての人類ではないにしても、情報技術に関連する多くの分野。 IBMは、複雑な問題を解決し、多くの問題の解決策を見つけるのに役立つ効果的な認知システムの作成に長い間取り組んできました。 重要な問題の1つは、ニューラルネットワークの詳細なトレーニングです。



このプロセスは十分に高速である必要がありますが、同時に効果的です。 IBMは、IBM Power AIソフトウェアを開発するためにNvidiaとコラボレーション契約を締結しました。 これは、IBM Power 8プロセッサーを搭載した特殊なIBM Power System S822LCサーバー上で実行されるプラットフォームであり、専門家によると、このようなソフトウェアおよびハードウェアプラットフォームは、最新のネットワークに迅速かつ詳細なトレーニングを提供するはずです。



Power System S822LCサーバーの機能は、Nvidia NV Linkインターフェイスの存在です。 このインターフェイスは、CPUとGPUの間の直接の相互作用の手段です。 これらのサーバーのうち、プロジェクト参加者は、 バークレービジョンアンドラーニングセンターの Caffeフレームワークを使用するテストニューラルネットワークAlex Netを作成しました。 Caffeに加えて、IBM Power AIプラットフォームには、cuDNN、cuBLAS、NCCLなどのライブラリが含まれています。 Caffeは広く使用されているフレームワークであり、ディープラーニングニューロシステムの分野で最も人気のあるフレームワークの1つと考えられています。



開発者は、新しいシステムのパフォーマンスは、従来のサーバー上に構築された他のシステムのパフォーマンスよりも数倍高いと主張しています。 パートナーは、822LCサーバーのパフォーマンスは、8世代前のNvidiaグラフィックカード（Tesla M40）を搭載したX86システムアーキテクチャのパフォーマンスに等しいと言います。 一般に、NVLinkを備えたS822LCサーバーは、同様のx86システムよりも約5倍の速度でデータを転送できます。



実際、このサーバーはNVLinkに同梱された最初のPower8システムです。 ニュースの前は、中央処理装置とGPU間のデータ交換がPCIe x16 Gen3インターフェイスのボトルネックでした。 現在、すべてのデータはシステムメモリの一般的なスペースにあり、プログラミングを容易にしていると言えます。



Power System S822LCの構成



-プロセッサIBM POWER8。標準で2.92または3.32 GHzの周波数で動作し、8または10個の完全にアクティブ化されたコアを備えています。

-最大1024 GBのRAM。

-5つのPCIe Gen3スロット。そのうち2つはGPUのみをサポートします。

-1 TB、7200 rpmのSATAインターフェイスを備えた2台のハードドライブ、または480（または960）GBの2台のSSD SATAドライブ。

-2つのCompute Intensive Accelerator GPU K80;

-1つの前面USB 2.0ポートと1つの背面USB 3.0ポート。

-2つの電源。

-19インチラックマウントバージョン2Uの高さ。

-オペレーティングシステム：

Ubuntu Server 14.04.5 LTS

Ubuntu Server 16.04.1 LTS

Red Hat Enterprise Linux（RHEL）7.2







Power AIには、特に、新しいシステム用に最適化されたいくつかの詳細な学習ツールが含まれています。 たとえば、これは科学プラットフォームのTorchとPythonライブラリです。 一般に、Power AIプラットフォームは1台のサーバーと数千台のマシンの両方で動作するため、簡単に拡張できます。



「Power AIは、詳細なトレーニングを簡素化し、他の分析テクノロジーとの連携を促進します。 科学者と研究者は、非常に迅速に展開できる簡単にスケーラブルなプラットフォームを手に入れるでしょう」と開発者は述べています。 また、AIアプリケーションを構築するという点で、新しいプラットフォームが最適であると考えています。 簡単に「インテリジェント」なチャットボットを作成したり、統計を使用して短時間で処理済みの分析結果を取得したりできます。



Human BrainProject 、HPC Nimbixクラウドプロバイダー、トルコのスーパーコンピューターコンピューティングセンターであるSC3 Electronicsなどの組織とプロジェクトは、新しいプラットフォームの試用に関心を示しています。
