高度なLinuxユーザーのための15のUbuntuヒント（翻訳）

英語のオリジナル記事。 翻訳： Boten 、Deniska、 MaxElc



数日前、私（以下-元の記事の著者-約）初心者が自分でLinuxを学ぶためにそれらをダウンロードして読むことができる本について書いた。 今日のLinuxセクションでは、経験豊富なユーザー向けの記事も用意しています。 経験豊富なUbuntu Linuxユーザーの場合は、次のヒントを参考にしてください。





1.コマンドラインをすばやくスマートに管理する

キーボードショートカットやその他のコマンドライン機能を使用して、入力をより簡単かつ迅速に行うことができます。 コマンドの一部やファイルやディレクトリの名前を終了する「タブ」キーについてはすでに知っているはずです。

ターミナルで使用できるキーボードショートカットを次に示します。

Ctrl-a-行の先頭に移動します

Ctrl-e-行末に移動します

Alt-] xは、カーソルを次のxオカレンスに直接移動します

Alt-Ctrl-]] xは、カーソルを前の位置xに戻します

Ctrl-uカーソルから行の先頭までをすべて削除します

Ctrl-kカーソルから行末まで削除

Ctrl-wカーソルから単語の先頭まで削除

Ctrl-yクリップボードからテキストを貼り付ける

Ctrl-lは、現在の行を画面まで移動して画面をクリアします

Ctrl-x ctrl-uは最近の変更を取り消します。 ctrl -__

Alt-rは行に対するすべての変更を破棄します

Alt-Ctrl-eコマンドラインを展開

履歴内のCtrl-R増分逆ルックアップ

Alt-pは履歴内の増分逆ルックアップではありません

!!! 履歴から最後のコマンドを実行します

！abcは、abcで始まる履歴の最後のコマンドを実行します

！n履歴からn番目のコマンドを実行する

^ abc ^ xyzは、最後のコマンドでabcの最初の出現をxyzに置き換えて実行します

また、クールなコマンドライントリックを見つけることができる4つのサイトを必ずチェックしてください。



2.ホットキーを使用してUbuntu Linuxアプリケーションを起動します。

これを行うには2つの方法があります。

* LaunchyやGnome-Doなどのアプリケーションを使用します。これにより、アプリケーション名から数文字を入力するだけで、アプリケーションをすばやく起動できます。

*または、gconfエディターを呼び出し（Alt + F2を押してからgconf-editorと入力してEnterキーを押します）、apps> metacity> global_keybindingsに移動します。 そこで、run_command_Nのいずれかをダブルクリックし、必要なアプリケーションを起動するホットキーを指定する必要があります。 この番号Nを覚えて、apps> metacity> keybinding_commandsに移動し、対応するcommand_N（Nは以前に覚えた番号）をダブルクリックし、起動するアプリケーションの名前を入力します。 たとえば、Firefoxを実行する場合は、単にfirefoxを指定します。







同時に、デフォルトでUbuntuで使用されるホットキーに慣れる必要があります。



3.中断したところから作業を続けます。





Ubuntuを構成して、最後にログアウトしたときに開いていたアプリケーションを記憶することができます。 新しいセッションを開始すると、これらのアプリケーションはすべて既に実行されているので、中断したところから続行できます。

この機能を有効にするには、[システム]> [設定]> [起動アプリケーション]に移動し、[設定]タブに移動して、[ログアウト時に実行中のアプリケーションを自動的に記憶する]チェックボックスをオンにします。



4. Ubuntu Linuxホームディレクトリの下に個別のパーティションを作成します。



Ubuntuの新しいバージョンは6か月ごとにリリースされます。 更新マネージャーを使用してシステムを最新バージョンに更新できるという事実にもかかわらず、更新が正しく機能しない場合があり、この場合はクリーンインストールを好むユーザーもいます。

このオプションの欠点は、ホームディレクトリ内のすべてのデータが失われることです。 この問題を回避するには、Ubuntuの最初のインストール時にホームディレクトリの下に別のパーティションを作成し、必要に応じて必要なディスク容量を割り当てます。 次回Ubuntuをインストールするときは、ホームディレクトリにこのセクションを指定するだけです（マウントポイントとして/ homeを指定する）。

この場合、クリーンインストールの場合でも、すべてのファイルとデータが保存されます。



5.インターネットに接続せずにUbuntu Linuxアプリケーションを更新およびインストールします。





これを行う方法はいくつかありますが、最も簡単な方法はAPTonCDを使用することです。 APTonCDを使用すると、必要なすべてのパッケージを含むCDまたはDVDを作成できます。これらのパッケージは、後でインターネットに接続せずにコンピューターにインストールできます。

APTonCDには、インストールメディアを作成するためのネットワーク接続（またはプリロードされたパッケージ）が必要です。 ただし、メディアを準備した後、アプリケーションがインストールされるマシンのネットワークに接続する必要はありません。 目的のCD / DVDを挿入し、通常どおりapt-getを使用します。



6.新しいフォント、Microsoftフォントをインストールし、フォントの表示を改善する







Ubuntuは、フォントに関してはあまり選択肢がありません。 ただし、Arial、Verdana、ImpactなどのMicrosoftフォントを含む新しいフォントを簡単にインストールできます。 さまざまなサイトを閲覧し、最適なフォントを選択できます。



7. PPAを使用して、最新のソフトウェアをインストールします。

ソフトウェアがUbuntuディストリビューションの一部になるか、リポジトリで利用可能になる前に、いくつかの手順を実行する必要があります。 もちろん、これらの追加手順は追加の安定性を保証しますが、利用可能になったときに最新のソフトウェアを受け取らないことも意味します。

他の製品よりも先に進みたい場合は、お気に入りのLaunchpadソフトウェアの個人用パッケージアーカイブを見つけて、システムに追加できます。 PPA の使用について簡単に説明しました。 このオプションで問題が多すぎる場合は、最新のdebパッケージをダウンロードし、ダブルクリックしてインストールできます（この方法でインストールされたソフトウェアの自動更新は表示されません）。

最新バージョンを使用すると、いくつかの問題が発生する可能性がありますが、ほとんどの場合、重要ではありません。 いつでもUbuntuフォーラムにアクセスして、すぐにヘルプを見つけることができます。



8. rootとして作業します。

rootユーザーは、不正な操作の可能性を最小限にするために、デフォルトでUbuntuでロックされています。 ただし、「注意することを約束する」場合は、次の方法でrootユーザーのロックを解除できます。





1. sudo passwd rootを入力し、rootの新しいパスワードを入力します。

2. [システム]> [管理]> [ログイン]ウィンドウに移動し、[セキュリティ]タブを開いて、[ローカルシステム管理者ログインを有効にする]チェックボックスをオンにします。

これで、ログインウィンドウからルートとしてログインできます。 「sudo su」と入力してパスワードを指定し、rootユーザーに切り替えることもできます。



9. Windows用のアプリケーションとゲームを実行します。





UbuntuでCounter Strikeをプレイしたくない人（もちろんゲームに興味があるなら）、またはPhotoshopを実行したくない人はいますか？ Ubuntuでは、 これは非常に簡単です 。



10.プロファイリングを使用してロード時間を短縮します。

Ubuntu Linux開発者は、起動時間を短縮する素晴らしい仕事をしました。Jauntyは高速で、Karmicはさらに高速になります。 ただし、もう1つの改善の機会があります-ロードプロファイリングです。 プロファイリングにより、Ubuntuは起動時に使用されるすべてのファイルを一覧表示し、ハードドライブ上の場所に応じて並べ替えることができます。 このため、以降のシステムのダウンロード中に、これらのファイルはより速く読み取られます。

以下の手順に従って、ダウンロードのプロファイルを作成します。



* grubメニューで、最も頻繁に使用するカーネルブート項目を強調表示します。

*「e」を押して編集します。

*「カーネル」で始まる行を強調表示し、もう一度「e」を押します。 行の最後に「プロファイル」という単語を追加します。



「Enter」を押してから「b」を押して、ダウンロードを続行します。

プロファイリングを初めて実行するときは、システムの起動が遅くなりますが、その後のダウンロードでは速度が上がることに注意してください。 また、増加は特定のマシンとハードディスク上のファイルの場所に依存するため、アクセラレーションはそれほど大きくない場合もあります。



11.他のUbuntuデスクトップ環境を試してください。

標準のGnomeシェル以外のものが必要な場合は、別のデスクトップマネージャーをチェックアウトするだけです。 環境の完全な置き換えを探しているなら、素晴らしい歴史を持つKDE4が非常に便利で美しいです。 KDEを入手するには、「sudo apt-get install kubuntu-desktop」を実行するだけです。



12.メディアセンターまたはメディアサーバーを作成します。





膨大な音楽、ビデオ、写真のコレクションを簡単に表示および管理できる機会があれば、それは素晴らしいことです。 Windows Media Centerインターフェースに魅了されましたか？ 待ってください。 メディアセンターとしてのUbuntuのすべてのクールな機能を見たことはありません。 Ubuntuにメディアサーバーをインストールすると、電話、PSP、または他のコンピューターからメディアにアクセスすることもできます。
