世界最小のA4インクジェットプリンターEpson WorkForce WF-100W

ベルリンで開催されたIFA展示会でエプソンは、世界で最もコンパクトなA4プリンターであるミニチュアインクジェットプリンターEpson WorkForce WF-100Wを発表しました（2014年4月現在）。 さらに、今日では、1.5インチのLCDスクリーンとフル顔料インクを備えたこの形式の唯一のポータブルプリンターです。



このデバイスは、1分あたり最大7枚の白黒ページと4つのカラーページを印刷でき、寸法は309 x 154 x 61 mm、重量は1.6 kgなので、ブリーフケースやバッグに入れて簡単に持ち運ぶことができます。 WF-100Wにはバッテリー（USBまたは電源コンセントに接続）が内蔵されており、フル充電で最大100枚のA4シート（最小ページ塗りつぶし）で印刷できます。







新製品の最も近い競合他社は、最近発表されたキヤノンPixma iP110 （寸法322 x 185 x 62 mm、重量2.0 kg）とHP Officejet 100 （寸法348 x 175 x 84 mm、重量2.5 kg）です。 ただし、これらのモデルは両方とも、より大きな質量と寸法を除いて、黒い顔料と着色された水性染料の組み合わせインクでディスプレイと印刷を行いません。 WF-100Wは顔料のみで印刷されます。つまり、印刷物は退色せず、水で洗い流され、マーカーで汚れることはありません。



コンパクトなプリンターを作る必要があったため、エプソンのエンジニアは長い間初めて2カートリッジのインク取り入れシステムを作ることを余儀なくされました（私が間違っていなければ、これは2003年のEpson Stylus Photo 900でした）。 同時に、たとえばHP Officejet 100のように、プリントヘッドはまだプリンターの一部であり、カートリッジではありません（このようなデュアルカートリッジプリンターでは、保存がより困難です）。















そしてもちろん、最新のフラッグシッププリンターと同様に、WF-100WはWi-Fi、Wi-Fi Direct、クラウドベースのEpson IPrintおよびE-mail Printをサポートしています。 また、GoogleクラウドプリントおよびApple AirPrintとの互換性もあります。 ノベルティは11月に約300ドルの価格で販売されます。
