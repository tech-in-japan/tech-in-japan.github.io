LinaroプロジェクトはAndroid OSのパフォーマンスを大幅に改善しました

昨日、Redditに投稿があり、LinaroプロジェクトのメンバーがAndroidアセンブリの最適化に関する作業を実演しました。 投稿に添付されたビデオは、Googleのコアを備えたAndroid Open Source Project 4.0.4とLinaroの最適化を備えたAndroid 4.0.4の動作を示しています。 テストのハードウェアプラットフォームは、SoC TI OMAP 4430（Motorola Droid RAZRで使用）でした。 パフォーマンス測定のデモンストレーションで開発者がこれを達成した方法については、ビデオを参照してください（最後の結果は100％の増加を示しています）。

Bernard Rozenkranzer（上のビデオ）は、Googleに彼を雇おうとせず、これらの最適化をAndroid OSPのメインブランチに実装するよう促しました。 CyanogenModのメンバーは、CM9 NightlyブランチでLinaroの成果を実装するための準備をすでに行っています。AndroidOpen Kangプロジェクトでも同じ計画がハッチングされています。 唯一の困難は、すべてのバイオニックの変更がまだ利用可能でないことですが、バーナードはそれらがすぐに公開されることを約束します。
