DUO-または119.95ドルのWacom Cintiq

DUOは、LCDモニターを備えたほとんどすべてのラップトップまたはコンピューターを、完全なタブレット機能を備えたタッチスクリーンに変換できる次世代テクノロジーです。 DUOは、超音波と赤外線を組み合わせた独自の技術を使用しています。







「ベースステーション」として機能する小さなデバイスがモニターの上部に取り付けられ、DUOペンを使用してユーザー入力を絶えず収集します。 KCI Communicationの研究開発部長であるLee Jae-jun氏は、この技術について説明しています。

ユーザーは、8つの異なる言語で画面上に直接書き込みまたは描画ができ、特別なソフトウェアがこれらのラベルをASCII文字に変換します。











デバイスの価格は119.95ドルで、ソフトウェアはWindowsVista®およびWindows7®と互換性があります。



仕様：

  • モニターに直接書き込み、描画
  • さまざまな種類の線と色を使用して重要な情報を強調表示します。
  • 筆圧と感度をサポート
  • 手書き認識（言語：韓国語、中国語、日本語、英語、フランス語、ドイツ語、イタリア語、スペイン語）
  • ほとんどのモニターで使用できる機能、対角線が17インチ以下のラップトップ（より大きなモニター19 "-22"は9月に利用可能になります）
  • USB Plug 'Playデバイス
  • Windows Vista®およびWindows®7との互換性
