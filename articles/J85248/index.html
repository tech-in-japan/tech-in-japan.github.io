SAMSUNG U800 Soul bにJavaアプリケーションをインストールする

アンドロイド、iPhone、およびその他の洗練されたデバイスの流行に照らして、単純なデバイスは無視されます。何も得意ではないが、既存のデバイスに参加したくない（できない）人はどうでしょうか。 終了-自分で終了！



当時、SAMSUNG U800 Soul bを買収しました。 Nokia 6500の代替として選択されました（3Gネットワ​​ークで動作する機能、1Gbのオンボードメモリ、メモリカード用の便利なスロット、そして最後になりましたが、デザイン）。



電話は一度も失敗しなかったので、UTEL-1-1.5 Mbpsのモデムは安定しているため、カメラはまあまあですが、最大の失望は、デフォルトでファイルシステムにアクセスできないことです。 ICQとOperaは、USB設定を切り替えることで、 ビットピムとシャーマニズムを介して配信することができました。



しかし、私は電話と本のリーダーを使いたいと思っていたので、できるだけ早く簡単に本をダウンロードしたかったのです。 そして解決策が見つかりました。







すべてのアクションはWindows 7で実行されました。（XPの場合、Windows Media Playerを更新する必要があります）



電話メニューで、次を選択します。

「設定」->「電話」->「USB設定」->「接続時に確認」



電話をコンピューターにUSBで接続します。 電話機のメニュー「Select USB Mode」で、「Windows Media Sync」を選択します



その後、「Portable Devices」セクションのエクスプローラーに、SGH-U800という電話が表示されます。





入ってみると、カードと電話という2つのフォルダーがあります。



電話を開きます。 Gamesフォルダーを作成します。



JAVA上のアプリケーションを含むフォルダーをGamesフォルダーにコピーします。フォルダーには、拡張子が.jarと.jadの2つのファイルが含まれている必要があります。 （各アプリケーションには独自のフォルダーが必要です。）





その後、電話機をPCから切断します。 コードを入力してください：*＃6984125 *＃、「管理設定」メニューに移動して、項目「4 Internals」を選択し、

パスワードプロンプトで、次を入力します。*＃9072641 *＃。

[内部]メニューで、[7ストレージ設定]の項目を選択します。

「ストレージ設定」メニューで、「2 Java DBの更新」項目を選択します

データを更新したら、電話機のメインメニューに移動し、[エンターテイメント]メニューに移動して、リストに新しくインストールされたアプリケーションを確認します。



次回PCに接続すると、GAMESフォルダー、すべてのファクトリーゲーム、インストールされているアプリケーションが表示されます。





アプリケーションを削除するには、最初にアプリケーションメニュー（ゲームがある場所）から削除し、次に電話からゲームフォルダーを直接削除します（別のフォルダーを見つけます-アプリケーション、インストールされたゲームのすべてのファイルをそこに表示できます）、最初に電話からメモリカードを削除し、Java DBを更新します（参照上記）。



サムスンのFSへのアクセスの欠如について彼らが何と言っても、私はこの電話はシステム全体ではなくファイルシステムにアクセスできると言いますが、それでも...メモリカードにファイルシステムフォルダがない場合は、動作可能なプログラムで作成しますFSはmmc_filesystemを選択します。 このフォルダーにサブフォルダーなどを作成できます。







電話のファイルシステムにアクセスした後、FB2形式の書籍のリーダーを設定しました。 これで、コードの入力に煩わされることはありません（JAVAデータベースの更新）。 ブックをmmc_filesystemフォルダーまたはサブフォルダーにドロップするだけです。

リーダーはTequilaCat BookReaderと呼ばれます。 ファイルと詳細な手順はTequilaCat Webサイトにあります
