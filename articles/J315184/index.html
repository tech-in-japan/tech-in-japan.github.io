エンタープライズITの基盤としてのIBM Power Systems S822LC

Power Systemsは、IBM Power Architectureに基づいた一連のサーバーです。 2015年10月に、POWER8プロセッサを搭載したLinuxモデルS812LCおよびS822LCがこのラインで発売されました。 今年の9月に変更され、今日、フォームファクター、スケーラビリティ、汎用性、価格の組み合わせにより、中規模企業に最も好まれているサーバーの1つとなっています。











過去1年間で、S8xxLCの潜在的な顧客の関心が大幅に高まり、これらのサーバーを使用したプロジェクトを説明する多数の出版物がマスコミに掲載され、それらのアプリケーションの最も好ましい分野がかなりよく決定されました。 最新の変更のS812LCおよびS822LCサーバーは、30を超える基本構成を持つPower Systemsラインのサーバーの一般リストに最新としてリストされています。



S812LCおよびS822LCの最初の検討





IBM Power Systemsの一連のサーバーの3つの新しいモデル-S812LC、ビッグデータ用のS822LC、および高性能コンピューティング用のS822LC



すべてのサーバーはもともとLinux上で実行するように設計されていました。 ジュニアモデルのS812LCは、大量のメモリとストレージデバイスを必要とするワークロードに焦点を当てています。 1つの10コアPOWER8プロセッサーを使用します。 ミドルモデル「ビッグデータ用S822LC」の典型的なアプリケーションは、HadoopとSparkのタスクです。



最も強力なモデルであるHPC用のS822LCは、オンライントランザクション処理などの商用コンピューティング向けに設計されています。 通常、20コア、1 TB RAM、230 GB / sのメモリ帯域幅を使用します。 超並列処理を備えた2つのNVIDIA Tesla K80 GPUアクセラレータも追加されました。



すべてのサーバーは、PowerPCアーキテクチャの直接の後継であるIBM OpenPOWERアーキテクチャに基づいています。これは、20年にわたる継続的な研究によって改善されてきました。 開発の主な目標は、幅広いアプリケーションの生産性を向上させることでした。



2013年、OpenPOWER Foundationコミュニティが設立されました。 IBMは、そのフレームワーク内で、チップセットの設計からファームウェアレベル（ファームウェア）および基本ユーティリティまで、メンバー向けに多くの仕様を公開しています。



その結果、関心のある開発者はシステムの機能を深く探求することができました。 その後、それらの多く（Mellanox、NVIDIAなど）がプラットフォームに重要な追加を行い、その機能を拡張して生産性を向上させました。





S8xxLCサーバーは、多くの主要な開発者とハードウェアおよびソフトウェアのメーカー間のコラボレーションの結果です。



実際、IBM Power Systemsは商用のUNIX業界にニッチを生み出しました。 このサーバーのポートフォリオにより、IBMは、リアルタイム分析とクラウドコンピューティング、つまり、主にLinuxソフトウェアスタックに関連する分野でリードすることを望んでいます。



Linuxは、効率的な仮想化と安全なネットワークの構築が重要な要件である分野に非常に適していることが実証されています。 同時に、リアルタイムで実行される今日の高レベルコンピューティングタスクの要件は、すでにIntelテクノロジーに基づいた大量生産サーバーの能力をすでに上回っています。



データセンターは、データのバッチ処理からリアルタイムの複雑な分析にそれぞれ著しく移行しており、新しい計算モデルが必要です。 これは、HadoopやApache Sparkなどのオープンソースプロジェクトにほぼ当てはまります。 もちろん、ソフトウェア層内である程度の最適化が可能ですが、パフォーマンスは最終的に使用するハードウェアに依存します。

基本的な機能に加えて、オープンソースのクラスター管理Extreme Cloud Administration Toolkit（xCAT）を使用できます。 XCAT 2.11.0はGithubで入手できます。 ファームウェアOpen Power Abstraction Layer（OPAL）は、ハイパーバイザーソフトウェアとOSの独立したレイヤーを提供し、エラーを自動的に検出して修正できるようにします。



NVLinkを備えた新しいPOWER8サーバー



2017年に発売予定の次世代POWER9プロセッサーの詳細を明らかにした直後、IBMはPOWER9とNVIDIA NVLink接続を備えた3つの新しいLinuxサーバーを今年9月上旬に発表しました。





新しいPOWER9プロセッサーを使用した構成では、新しいIBM POWER Systemsサーバーは、より強力な新しいNVLinkツールも受け取ります



Power S822LC for High Performance Computingサーバー（プロジェクトのコード名であるMinsky）は、新しいプロセッサー（NVLinkを備えたPOWER8）を使用して、NVIDIAの新しく最も強力なGPUであるP100 Pascal GPUに接続します。



他のサーバー-ビッグデータ用のベースモデルPower S821LCおよびPower S822LCも、PCIeインターフェイスを介してNVIDIA K80またはP100 GPUに接続できます。 フラッシュドライブおよびFPGA（Field-Programmable Gate Array）に接続するためのIBM Coherent Accelerator Processor Interface（CAPI）が装備されています。



IBM High Performance Computing and AnalyticsのバイスプレジデントSumit Guptaは、ビッグデータ分析とコグニティブコンピューティングが企業ITの中心的な位置を急速に占めるようになったことから、新しいサーバーはIBMが引き続き高性能コンピューティングの分野で機能していることの証明です。



IBMによると、「シリコンレベル」で作成されたNVLinkを搭載したS822LCサーバーは、同等のx86システムで発生するデータ転送の5倍の速度でデータを転送できます。 また、高性能コンピューティングでGPUを使用するためのソフトウェアの複雑さを大幅に軽減します。



典型的なデュアルプロセッサS822LCサーバー構成でNVLinkテクノロジーを使用する。



S822LCは、NVLinkに付属する最初のPOWER8システムです。 これまで、以前に使用されていたPCIe x16 Gen3インターフェイスの制限は、中央処理装置とGPUの間のデータ交換のボトルネックでした。 NVLinkを使用すると、すべてのデータが共通のシステムメモリスペースに格納されるため、プログラミングが非常に簡単になります。



NVLinkプロセッサを統合した新しいPOWER8には、最大3.26 GHzの周波数で動作する10個のコアがあります。 このサーバーのPOWER8プロセッサーは、x86プロセッサーよりも広いメモリー帯域幅（最大115 GB / s）を備えており、スロットごとに最大で半テラバイトのシステムメモリーを搭載できます。



POWER8には大きな内部キャッシュがあるため、高速コアとメモリチャネルの広い帯域幅により、大きな処理能力を必要とするユーザーアプリケーションのパフォーマンスが大幅に向上します。



新しいNVIDIA Tesla P100グラフィックアクセラレータは、浮動小数点コンピューティングを高速化し、最大21テラフロップスのパフォーマンスを提供します。 16 GBのHBM2スタックメモリが含まれ、帯域幅は720 GB / sです。



Linuxサーバーのポジショニング用のS82x



非常にスケーラブルなLinuxクラスターで構成可能なPower Systemsは、ヒトDNA（ゲノミクス）、金融、計算化学、石油およびガス研究、データ分析、その他多くの分野に関連する計算など、大規模なワークロードに対して高いパフォーマンスを提供します。



ビッグデータと分析では、 認知システムがますます重要な役割を果たし始めています。 以前のハードウェアおよびソフトウェアシステムは、通常の数値関係またはシーケンスを分析することにより、データ内の特定のパターンを検出できました。 しかし、彼らはテキスト、ビデオ、オーディオなど、自然言語にカプセル化された重要なデータを分離できませんでした。



それらを検出し、状況の分析と情報に基づいた意思決定の開発に関与させるために、認知システムは自然言語処理と機械学習を使用します。 たとえば、ソーシャルネットワークなどの監視に基づいて、病気の流行の初期兆候を検出するようにシステムを構成できます。



コグニティブシステムは、特に非構造化データを分析します。これにより、ほとんどのアナリストの時間を吸収する重要でない「ノイズ」のレベルが大幅に削減されます。



ただし、より深いレベルの分析と洞察により、認知システムは従来のツールよりも高価になる傾向があります。 集中的なデータ処理には多くの計算能力が必要であり、リアルタイム分析には最大速度に最適化されたインフラストラクチャが必要です。



Linux on Power Systemsアーキテクチャの使用を開始した最初のIBM顧客の1つはAllegiant Airでした。 彼女はリアルタイムで消費者行動の分析に携わり、ショッピングWebサイトへの訪問者に対する個別のオファーを直接生成することを目標としていました。

モビリティ 。 Worklight on Powerは、モバイルアプリケーションの開発を加速し、管理を提供するプラットフォームを実装しています。 これにより、顧客はモバイルデバイスをエコシステムに追加することで、比較的簡単にビジネスを拡大できます。



このプラットフォームには、完全な開発環境、ランタイム最適化用のミドルウェア（モバイル用に最適化されたランタイムミドルウェア）、商用エンタープライズアプリケーションのライブラリ、統合管理および分析コンソールが含まれています。



クラウド 。 IBM Power Systems Solution Edition for Scale-Out Cloudは、低コスト、高性能、高帯域幅のクラウドプラットフォームです。 POWER8テクノロジーの利点と、オープンソースハイパーバイザーの柔軟性、および仮想化管理ツールの豊富な機能を兼ね備えています。



Hadoop Power Systemsプラットフォームは、ビッグデータ分析を簡素化および高速化するために最適化されています。 ビッグデータの高速処理のためのインフラストラクチャを提供します。 これにより、ビッグデータ管理システムのシンプルな展開、使用、管理、サポートなどを通じて、投資収益率（ROI）が向上します。



このプラットフォームは、他の同等のHadoopソリューションよりも優れた機能を備えており、クラッシュや実行時の問題が少なく、信頼性と柔軟性が向上しています。
