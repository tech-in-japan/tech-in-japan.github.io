ヨハネスブルグ証券取引所はLinuxに切り替えます

ヨハネスブルグ証券取引所（JSE） は 、ヨハネスブルグに移動した後、Linuxベースの取引プラットフォームに切り替えることを発表しました。 JSEは現在、Microsoft .NetおよびSQL Server 2000に基づいたTradElectプラットフォームでロンドンから運営されています。2012年前半に、MillenniumITからMillennium Exchangeプラットフォームに移行する予定です。



JSEはTradElectプラットフォームでも動作するロンドン証券取引所（LSE）の例に従いますが、2009年にLinuxおよびSolarisベースのシステムを使用してMillennium Exchangeシステムへの移行プロセスを開始しました。 LSEは、Millennium Exchangeのスリランカの開発者-MillenniumIT-を3,000万ドルで買収しました。



現在、JSEはヨハネスブルグに戻り、南アフリカの顧客がサービスを利用しやすくする予定です。 取引所はこの動きを利用して、ミレニアム取引所に切り替えました。 また、LSEとは異なり、Linuxシステムでのみ機能します。



新しいシステムにより、取引速度が大幅に向上し、現在の取引プラットフォームのほぼ400倍の速度でトランザクションを完了できます。 この改善により、取引所の成長が可能になります。 JSE事務局長のリン・パーソンズは次のように述べています。「取引所が取引速度を上げると、高頻度取引のレベルも上がるという多くの証拠があります。 これは市場を深めるために重要です。」
