汎用性の高いリンク-ユニバーサル接続

Broadcom Limited（旧Avago Technologies）が製造したVersatile Link（VL）ラインの光ファイバーコンポーネントは、世界中の産業用情報伝送システムの開発者にふさわしいものです。 産業、エネルギー、医学のさまざまな分野での光ファイバーの使用については、以前の記事で説明しました。 同じ記事で、産業環境でプラスチック光ファイバ（POF）を介してデータを送信するためのVersatile Linkのコンポーネントラインを確認します。

















ETF LLCはBroadcom Limitedの公式ディストリビューターであり、さまざまな業界の代表者向けの光ファイバーコンポーネント（特にVLコンポーネント）の供給において豊富な経験を持っています。







汎用性の高いリンクコンポーネント-Hewlett PackardからBroadcom Limitedまで



光ファイバコンポーネントのVersatile Linkシリーズは、20年以上前に産業用データ伝送システム専用にヒューレットパッカード（HP）によって開発され、徐々に人気が高まってきました。 コンポーネントは、コアとクラッドの直径が980/1000μmのプラスチック光ファイバー（POF）、およびより長い通信回線用の200/230μmポリマーコーティングシリカファイバー（HCS）を介してデータを送信するように設計されています（ファイバータイプの詳細こちらをご覧ください ）。







その範囲には、その時点で製造された他のすべての光ファイバーコンポーネントとは構造的に異なる、個別の光トランスミッターとレシーバー、および接続製品（コネクターとアダプター）が含まれていました。 少し先を見ると、フィールドでも情報の転送を迅速かつ簡単に整理できるようになったため、多くの点でVLコンポーネントの特別な設計が広く普及した理由となったことがわかります。 また、開発者は、コンポーネントの比較的低いコストとその信頼性に魅了されました。







その後、HPの半導体コンポーネント部門はアジレントテクノロジーの一部となり、後にアバゴテクノロジーに変身しました。 2016年2月、アバゴテクノロジーとBroadcom Corporationの合併によりBroadcom Limitedが設立されました。 同社の製品ポートフォリオに時間の経過とともに重大な変化が生じたにもかかわらず、Versatile Link製品ラインはその地位を失うことはありませんでしたが、逆に、新しい応用分野を拡大、近代化、開発しました。 今日まで、Versatile Linkコンポーネントは、Broadcomの膨大な品揃えの中でしっかりとその位置を占めています。













それでは、Versatile Linkラインのコンポーネントの主な機能を見てみましょう。おかげで、それらは業界で広く使用されています。 特定のコンポーネントの機能については説明しませんが、各グループ（アクティブコンポーネント、接続製品、ケーブル）の一般的な説明と、VLコンポーネントの範囲についてのアイデアを得るのに役立つ情報を示します。 Webサイトinfiber.ru （光ファイバー製品のETF会社のWebサイト）またはBroadcomの公式Webサイトで、VLコンポーネントのカタログをよく理解できます。







トランスミッターとレシーバー



コンピュータネットワークでは、通常、1つのハウジング内のトランスミッタとレシーバである光トランシーバがアクティブな機器として使用されます。 しかし、産業用途では、情報の双方向伝送の必要性が常に生じるとは限りません。 したがって、ボード上の資金とスペースをより経済的に使用するために、個別の送信機と受信機が使用されます。 Versatile Linkラインの一部であるのはこれらのモジュールです。







送信機と受信機の両方のデザインは同じです。 ハウジングは、厳しい動作条件に耐えることができる耐久性のある難燃性プラスチックで作られています。 モジュールには、制御用の4つのピンと、ボードへの追加の固定用の2つのピンがあります。 ピン間の間隔は、標準のDIP8パッケージと同じです（2.54 mm）。 送信モジュールと受信モジュールを識別するために、ハウジングの色は異なります。 現在入手可能なすべてのVLコンポーネントは、RoHSに準拠しています。 最初のモジュールは、0〜70°Cの温度で動作するように設計されました。 その後、工業用温度範囲（-40〜+85°C）で動作するコンポーネントが開発されました。













ケースには、プリント基板に対する光ポートの位置に応じて、水平、垂直、傾斜の3つのタイプがあります。 ボードに取り付けると、水平モジュールポートはボードの表面に平行に向き、垂直ポートは垂直になります。 一部のモデルには、ポートが30°の角度で配置される3番目の変更もあります。 そのような選択の可能性は、システムの設計に追加の自由を与えます。













VLコンポーネントの興味深い機能は、それらを相互に接続してボード上のスペースを節約できることです。 これは、隣接するモジュールの反対側にある対応する溝に挿入されるハウジングの片側にL字型の突起があるために可能です。 4つ以上のモジュールを組み合わせることはお勧めしません。













VLコンポーネントには、特別な光コネクタ設計があります。 入力ポートには2つのフラップがあり、クリックすると光コネクタのコネクタを位置合わせして固定します。 このメカニズムにより、ジャンクションでの損失が最小限に抑えられ、コネクタの偶発的な切断が防止されます。













前述のように、Versatile Linkコンポーネントは、プラスチックファイバーを介してデータを送信するように設計されています。 一部のモジュールは、200/230ミクロンHCSファイバー用に最適化されています。







VLトランスミッターとレシーバーは、最大伝送速度に応じていくつかのグループに分けられます：最大40 kbit / s、最大1、5、10、12、50、または160 Mb / s。 伝送範囲も異なります。これは、アクティブなコンポーネントの特性だけでなく、使用するケーブルと動作温度にも依存します。 VLコンポーネントに基づく通信回線の一般的な長さは数十メートルから数百メートルであり、ほとんどの産業用アプリケーションを満たします。







次に、送信機と受信機の特性をより詳細に検討します。







送信機



エミッタとして、VLトランスミッタは650（660）nmの放射波長のLED（LED）を使用します。 この波長では、プラスチックファイバーの減衰が最小になります。 さらに、このようなLEDは低コストです。 レンズがトランスミッターの出力に取り付けられ、LED放射を光ファイバーに集光します。







ほとんどのトランスミッターはLEDを発し、ドライバーを内蔵していないため、設計の柔軟性に貢献します。 例外は50メガバイトのトランスミッターで、TTL入力のドライバーがあり、一部のモデルには非反転入力があります（高レベルの電気信号はLEDのオンに対応します）が、他のモデルには反転入力があります（低レベルの電気信号はLEDのオンに対応します）。







受信機



受信モジュールには、さまざまな複雑さの光検出器と信号処理回路が含まれています。 ほとんどのレシーバーは、標準タイプのロジックと互換性のある出力を備えています。 50メガビットレシーバーには、TTL出力（非反転または反転）があります。 一部の5メガビットレシーバーでは、TTL出力も利用できます。 160メガビットの受信機にはアナログ出力があります。







トランシーバー



Broadcomは、プラスチック光ファイバーで動作するように設計されたVLコネクタを備えた光トランシーバー（トランシーバー）も製造しています。 ただし、これらは上記のモジュールとは構造的に大きく異なります。













コネクター



VL光コネクタはプラスチック製で、光モジュールのコネクタと互換性のある特別な設計が施されています。







Broadcomは、POF 980/1000ミクロンおよびHCS 200/230ミクロン用のVLコネクタを製造していますが、POF用のコネクタにはいくつかの変更があります。 コネクタは、光ケーブルの端に取り付ける方法が異なります。 一部のコネクタは金属製の圧着リングで固定されていますが、特別なツールで圧着する必要のないモデルもあります。 後者の場合、ファイバはコネクタ本体自体によってしっかりと固定されます。 一部のVLコネクタには、モジュールハウジングの特別な凹部に固定されるラッチがあり、接続に追加の強度を与えます。 同社は、シンプレックス（シングル）およびデュプレックス（デュアル）VLコネクタを製造しています。













VLコネクタの主な利点の1つは、インストールが簡単なことです。 現場では光パッチコード（コード）の製造が可能で、大きなスキルを必要とせず、数分かかります。 重要なのは、VLコネクタの取り付けにエポキシ接着剤を使用していないことです。 必要なデバイス（クリーバー、研磨アクセサリー、および必要に応じて圧着工具）は、石英ファイバーの場合よりも大幅に低コストです。 例として、Broadcom VLコネクタを取り付けるためのビデオの説明を以下に示します（ビデオでは、VLコネクタを備えたトランシーバにのみ適したコネクタを使用しています）。











Broadcomは、ラインセクションを相互に接続するためのVLタイプコネクタを備えたシンプレックスおよびデュプレックスアダプタ（ソケット）も製造しています。













ケーブル



VLコンポーネントでは、主にPOF 980/1000μmおよびHCS 200/230μmが使用されます。 Broadcomでは、POFケーブルにいくつかの変更を加えています。 これらは、直径2.2 mmの黒いプラスチック製シースのシンプレックスケーブルと、ジップコードタイプの2本のケーブル（ジャンパで接続された2本のケーブル）です。 使用するファイバーの特性に応じて、ケーブルには3つのタイプがあります。







  • 標準減衰（0.22 dB / m）のPOFケーブル;
  • 減衰が低減されたPOFケーブル（0.19 dB / m）;
  • 減衰が少ないハロゲンフリーPOFケーブル（0.19 dB / m）。








透明なシースPOFケーブルは、 光学アークセンサーおよび他の同様のアプリケーションにも使用できます。







既製のPOFパッチコードを購入することもできます。 HCSファイバー付きのケーブルとパッチコードは、他のメーカーから入手できます。







デバッグボード



Versatile Linkコンポーネントの機能をテストし、デバイスをすばやくプロトタイプ化するために、Broadcomは、デバッグボード、トランスミッターとレシーバーモジュール、および光パッチコードで構成される便利なデモキットもリリースします。













ユニバーサル接続



Versatile Linkラインには、当然その名前が付けられています（「ユニバーサル接続」）。 VLコンポーネントの光学特性と設計機能は、低コストと幅広い品揃えにより、無数の異なる産業用通信ラインで使用できるようになり、世界中の開発者の20年以上の経験によって証明されています。







EFO会社では、このラインのコンポーネントを購入できるだけでなく、産業用ネットワークでのアプリケーションの問題に関する技術的なアドバイスを得ることができます。 弊社のWebサイトで、VLコンポーネントの範囲について詳しく知ることができます。
