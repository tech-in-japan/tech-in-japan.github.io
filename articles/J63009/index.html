prettyPrint

Githubで著者をフォローしている人は、彼が最近「prettyPrint」と呼ばれる新しいプロジェクトを追加したことに気付いているかもしれません。

「PrettyPrint」は、ColdFusionのcfdumpに似たJavaScript用のブラウザー変数ダンパーです。 デバッグ中に表示するために、テーブルの形式で任意のタイプのオブジェクトを表示できます。

例を参照してください 。



ハイライト：



  • 完全に独立しており、追加のCSSや画像は必要ありません。
  • 無制限のネストを持つオブジェクトをサポートします。
  • DOM要素だけでなく、すべてのJavaScriptデータ型がサポートされています。
  • 再帰的なオブジェクト参照から保護されています。
  • オブジェクトの表示深度を設定できます。
  • グラデーション付きの表見出し。
  • CSSによるカスタマイズを許可します。
  • JSLintで検証済み！


使用例：



var table = prettyPrint( anyRandomThing, { /*optional options object */ } );



// Making the table viewable is down to you...

// eg

document .body.appendChild(table);
