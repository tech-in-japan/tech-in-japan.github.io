難しいメッセージングとインターネットコール：5人のモバイルメッセンジャーはどのようにGPRS / EDGEを消化しますか？

世界中のユーザーがモバイルブロードバンドインターネットに接続する機会が増えているという事実にもかかわらず、モバイルインターネットアクセスの速度が望まれない状況に対処しなければならない場合があります。3Gが切断され、予想される最大値はGPRS / EDGEで。 これは、たとえば、カバレッジが悪い状況で、ネットワークに大きな負荷がかかると発生する可能性があります。 そして、ロシアの広大な広がりを考えると、どこかで3Gが存在しないことさえあります。 したがって、モバイルメッセンジャーが「狭いチャネル」でどのように動作するかを見るのは興味深いものでした。 4つの業界をリードするプログラム-Line、Skype、Vider、WhatsAppは、既に1億インストールの段階を通過していますが、新しいConverseメッセンジャーによってコンパイルされました。









逆


Line、Skype、Vider、WhatsAppが長い間プレゼンテーションを必要としなかった場合、 Converseについて詳しく説明します。



このアプリケーションは無料で、Google Playからダウンロードできます。 近い将来、iOSデバイス用のプログラムをリリースする予定です。 インストールパッケージのサイズは24 MBです。







最初の起動時に、ユーザーは標準登録を行うように求められます。SMSコードの送信先の電話番号を入力します。 ユーザーの識別子（名前/姓またはニックネーム）はGoogleアカウントから取得されますが、後で変更できます。







メインメニューは4つのタブで表示されます。テキストチャット（SMSとの統合が存在する）、コールマネージャー、連絡先（電話帳のすべての連絡先がここに表示されます。「C」ボタンを押すと、 Converseユーザーの連絡先のみが表示されます）、ユーザーのサービスタブ、変更可能あなたの写真。 Doodle Jumpのスタイルで、最初の開始時に表示され、手で描かれたヒントの助けを借りて、プログラムメニューに慣れます。







「プロファイル」タブでは、 Converseアカウントを追加し（すべての新しいユーザーに自動的に2ドルが入金されます）、SMSだけでなく友人/知人をConverseに招待し、いくつかの設定を変更し、最後にプログラムの開発者からのフィードバックを使用できます。 一般的に、すべてがシンプルに見えます。































次に、現在使用できるプログラムの機能について少し説明します。



まず、 コンバースを使用すると、複数の電話を1つのアカウントにリンクできます。 これにより、たとえば、1人の家族だけが通話料金を支払うことができますが、すべての家族はそれを使用します。 この機能は休暇中に便利です。







第二に、一方のチャットが他方のチャットで自動的に複製される場合、複数のデバイスでプログラムを柔軟に使用できる可能性が実装されています。 この機能はSkypeよりも知的に機能するため、たとえばスマートフォンで会話を行う前であれば、タブレットでチャットを再表示する必要はありません。



第三に、通話中に通話の費用が表示されます。



ビデオ通話はまだコンバースでは機能しませんが、これは近い将来の見通しです。



テスト方法


テストに戻ります。 次のように実行されました。5つのプログラムすべてが電話機にインストールされ、3G通信が強制的に切断されました。 同時に、「バックグラウンドでのデータ制限」モードが有効になりました-狭いチャンネルが異なるプログラム間でインターネットに熱心に分割されず、各メッセンジャーは彼に提供されたチャンネルを完全に使用しました。 そのような種類のdul。







制限付きチャンネルの典型的なパラメーターは、以下のスクリーンショットに示されています。







伝説によると、GPRS / EDGEネットワークにいたユーザーは、ネットワークへのブロードバンドアクセスに接続されたスマートフォンにメッセージを送信したり、電話をかけたりする必要がありました。 このアプローチでは、ボトルネックは1つだけです-発信者側にあり、特定のプログラムの機能を狭いチャネルで簡単に評価できます。



各プログラムで、アプリケーション自体のダウンロード速度、テキストチャットの速度、音声通信の品質、150 KBのファイルの送信時間などが推定されました。



結果


テストの全体的な結果：すべてのメッセンジャーが通常狭いチャネルを消化するわけではありません。 そして、具体的な結果は次のとおりです。



  • コンバース ：音声通話を行うことができる唯一のプログラム。 もちろん、彼は完璧な品質ではなく、フレーズには時々長い休止が散在していましたが、それでも断線はなく、会話の本質は失われませんでした。 テキストチャットの作業はプラス記号で4と評価できます-正常に機能しますが、Line、Viber、WhatsAppの作業よりも少し遅くなります。 最後に、メッセンジャーは1分20秒で150 KBの参照ファイルを配信しました。これも良い結果です。 Skypeはテキストチャットとしても機能せず、ファイルをViberに転送するのに3分半かかりました。


  • Line ：クイック起動、クイックテキストチャット、および高速ファイル転送（WhatsAppに次ぐ）が好きでしたが、音声通話の観点からメッセンジャーはすぐに防御的な立場を取りました-不安定な接続とネットワークの問題を報告しました。







    当然、GPRS / EDGEと回線を使用した会話は失敗しました。



  • スカイプ ：おそらく最も重いクライアントは彼の「重さ」を確認しました。 最長のダウンロード時間（15秒）、およびテキストメッセージの送信や通話の不能-これは、SkypeとGPRS / EDGE接続の相互作用の結果です。 コミュニケーションをチェックするための会社のサービスにアクセスできませんでした。


  • Viber ：この人気のメッセンジャーは失望しました-音声通話が機能しませんでした（通話が一方のデバイスで受信され、通話時間が経過したように見え、もう一方の画面ではダイヤラーがまだオンで電話がオフフックであると表示された非常に興味深い状況がありました）、参照ファイルが送信されました非常に長い間、チャットの速度はLineおよびWhatsAppの速度よりも劣っています。
















  • WhatsApp ：そして、逆にこのプログラムは満足しています。チャットは賢く、ファイルはすぐに転送されます。 彼女はまだ音声通話をサポートしています...
