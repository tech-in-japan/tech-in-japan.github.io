PHP内部



  • [RFC] Arrow Functions 2.0-短いラムダでの投票が開始され、投票プロセスが奨励されています。 構文は同じままです： fn($x) => $x * $y



    。
  • [RFC]左結合三項演算子の廃止 -ほとんどすべての言語で、三項演算子は右から左に評価されますが、PHPでは逆に評価されます。 左結合性を完全に廃止することが提案されており、いくつかの三項演算子を使用する必要がある場合は、括弧を使用して順序を明示的に示します。 
     1 ? 2 : 3 ? 4 : 5; // deprecated (1 ? 2 : 3) ? 4 : 5; // ok 1 ? 2 : (3 ? 4 : 5); // ok
  • PHP Internals News＃5 - RFCの著者であるLarry Garfieldとのリスト掲載。
  • PHP Internals News＃6 -PHPの品質保証と拡張のためのPHP 7.2リリースマネージャーであるRemi Colletへのインタビュー。
