最近、 Web 2.0 Expo Berlinから戻ってきました 。 私たちは出展者（出展者）としてそこに行きました-弊社は展示会でWebAsist Webサービスを代表しました。 インプレッションに関する小さなレポート。







展示会は2007年1月5日から8日までベルリンのメッセで開催されました。

Web 2.0 ExpoとInteropの 2つの展示会が1つのホールで一度に開催されました。 これは当初は計画されていませんでしたが、2つの展示会がイベントの約1か月半前に統合されることを（出展者として）通知されました。 彼らが団結することを決めた理由は示されていませんが、両方の展示会だけが計画どおりの参加者と訪問者を獲得できなかったことは容易に理解できました。



実際、このイベントは11月5日にWeb 2.0という用語を生み出したTim O'Reillyによるスピーチから始まりました。 プレゼンテーションは非常に一般的でした-ユーザー指向のシステムを作ることがいかに重要であるか、ユーザーの行動からどれだけ有用なデータを抽出できるか（例として携帯電話で-モバイルプロバイダーが私が誰から、どれだけ、誰から情報を抽出できるかについて） SMSを送信するなど）。 彼はGoogleをデータを正しく集約する会社と呼んでいた:)彼は、タッチコントロールを備えたデバイス（iPhoneなど）の開発に大きな未来を見たと語った。 そして、はるかに。 個人的には、パフォーマンスが気に入りました。

スピーチの断片を見てください 。



私は講演会やさまざまな基調講演に出席しませんでした（私たちは一日中エキスポホールのスタンドで過ごしました）ので、それらについては何も言えません。 しかし、展覧会のこの部分に関する他の人々の意見は、完全にお世辞ではありませんでした。 まず第一に、 それは高価だったためです （フル会議パスで1300ユーロ！）。 しかし、このコストにはビュッフェにサンドイッチとコーヒーが含まれていたため、最も不満だったのはそこで費やしたお金を「打ち負かす」ことです:)
