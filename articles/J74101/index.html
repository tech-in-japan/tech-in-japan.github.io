Ubuntu Moblin Remix



おそらく誰もが、原子ベースのネットブックで使用するためにインテルが直接作成した、注文を運ぶモブリンについて聞いたことがあるでしょう。

誰もがそれを聞いたが、おそらくそれを使用したのはほんの数人だけだった。 英語とロシア語の両方で、最小限の情報があります。



公式サイトには、魅力的なビデオと説明が1つにまとめられています-Moblinはクールです。



私は猫を尾で引っ張らないことに決め、Ubuntu 9.10に基づいたMoblinリリースをダウンロードしました 。



Ubuntu Moblin Remixは、DellがUbuntu Moblin Remix Developer Editionを搭載したMini 10vネットブックのリリースを決定した直後に、突然この世界に名を残しました。

一般ユーザー、特にDellなどの尊敬されるオフィス向けに、ある種の「Developer Edition」をリリースするにはどうすればよいですか



質問はもちろん修辞的です。 ポイントに行きましょう。 できるだけ早く、Ubuntu Moblin Remixに関するセクションが公式のUbuntuフォーラムに作成されました。 そこには、 FAQだけがある意味のあるものから少しの情報を見つけることができますが、基本的にこのセクションには「これは何ですか？」や「なぜUNRより優れているのか？」



さて、それは問題ではありません。 あなたは常に自分ですべてをチェックする必要があります:)





設置



ディストリビューションにはlive-cd機能があるため、最初にこのディストリビューションから起動して、この非常にmoblinの機能を確認します。 第一印象は、「美しい！」、「速い！」、「便利！」という言葉で特徴付けられます。



それが決定され、設定します。

最後に、私は無線LANや他のfireをチェックすることにしました。 一般に、Ubuntuのネットブック（Lenovo S10）でBluetoothを使用するWi-Fiは、バージョン8.04から「箱から出して」静かに動作したため、これについて疑いはありませんでした。 しかし、会社のMoblinネットワークマネージャーのタブに移動すると、Wi-FiとBluetoothがないことがわかりました。



もちろん、何もライブCDには書きませんでした。誰もがfireを積んでいるわけではないと言っています。



インストールは絶対に標準であり、9.04と元の9.10の同じプロセスと変わりません。 すべてが同じくらいシンプルで、最小限の質問があります。



ロード直後の起動画面は次のようになります。



ビデオとは少し相関していませんが、すぐにカスタマイズするよう提案されています。 Twitterとlast.fmから情報を出力するには（そして他に何もサポートされていません）-もちろん、インターネットが必要です。



最後のタブを踏みます。これは誇らしげにネットワークと呼ばれています。



まあ、はい、ブルートゥースも無線LANも持っていません。 そして、決してありませんでした。

この場合、そのような「ギフト」を含むネットブックを受け取った一般ユーザーは何をすることができますか？

まあ、おそらく彼は設定に行くでしょう、それは奇妙なことに、アプリケーションタブに隠されています。



わあ [設定]セクションの[ハードウェアドライバー]アイコン。 そして、ここには何がありますか？ でも、ええ。



すべて、ユーザーはthe迷しています。 ユーザーは、美しいメニューでさらに15分間退屈から抜け出し、ネイティブWindowsを設定するために出発します。

特に倒錯したユーザーは、ケーブルを介して素晴らしいタイネットで接続しようとすることができます。 これらの永続的な人は、別の写真が表示されます。



このような画像は、ネットワーク上のユーザーがDHCPサーバーを持っているかどうかに関係なく表示されます。

設定？ どの設定ですか？ 設定は最後の世紀であり、一般的には吸盤向けであるため、Moblinネットワークマネージャーには設定がありません。 一般的に。 なし。

ExpressCardもUSB GPRSモデムも機能しなかったことに言及する価値はありますか？



Ubuntuに精通している人のためのさらなる手順は、原則として理解可能である。 ndiswrapper以外の場所にダウンロードするか、Linuxのwoodを直接探してUSBフラッシュドライブに放り込んでput、put、put、put ...

結局のところ、モブリンは単なる貝殻であり、その下にはまだ同じ普通のウブンツがあります。

それはmoblinだけで、誰もがこれをしたくないのです。



これに関して、1つの重要な問題が生じます。 システム自体が実際に変更されなかった場合、どのような目的で必要なドライバーをディストリビューションから除外する必要がありましたか？ たぶん、暴徒は家族の呪いや宗教的な規範によって妨げられているのでしょうか？ なぞなぞ。



Ubuntu Moblin Remixと元のUbuntuを区別する唯一のことは、シェルとしてのMoblinと1つの追加されたカブ（ ppa.launchpad.net/moblin/ppa/ubuntu karmic main ）だけです。 Ubuntu 9.10のオリジナルリリースを入れて、このカブを接続し、モバイルを実際のubuntuの上に静かに置くだけで、Ubuntu Moblin Remixディストリビューションのアセンブラーが残したマスターのテーブルの残骸ではなく、それで十分だと思います。



しかし、良いことに進みましょう。シェル自体は本当にゴージャスです。 これは、画面の対角線が7〜10インチのネットブックに必要なものです。

すべてが非常にミニマルで美しく、小さな画面用に最適化されています。 しかし、いつものように-ニュアンスがあり、それらの多くがあります。

タブのスクリーンショットがこれを最もよく示すと思います。





サウンドとバッテリー



最後から始めましょう。 悩みの種であるネットワークタブに従って、音量とバッテリー電力を管理します。







いいね。 スクリーンショットでわかるように、バッテリーの状態を示すタブはプラグとして表示されます。 ネットブックからアダプターのプラグを抜くと、バッテリーインジケーターに変わると思いますか？ いいえ、彼らは推測していませんでした。このタブに移動すると、バッテリーで何が起こるか、充電量がどれだけ残っているかがわかります。 太字マイナス。





ゾーン



通常のウィンドウスイッチです。 alt-tabからも呼び出されます。







用途



次のタブ、アプリケーション。 実際、これは古典的なGnomishメインメニューであり、別のタブに移動しました。







ペーストボード



次にmoblinには、高度なクリップボードである特定のPasteboardがあります。 別のボタンで、ほぼ中央に持って行かれた理由はわかりませんが、おそらくモブリンの開発者は私が知らないことを知っていました。







メディア



さて、写真、音声、ビデオなど、すべてが明確です。 「最近再生された」という見出しの下に、ある種の奇妙な見た目の静止画像があります。 写真の上にカーソルを合わせると表示される、クラシックな緑色の[再生]ボタンの見た目も変わりません。 しかし、これには間違いがあります。

公平には、ビデオの再生が拒否されたことに注目する価値があります。プレーヤーにgstreamerが必要であるという通知とともに、かわいいウィンドウがポップアップしました。 ええ、「ワニは捕まえられず、ココナッツは成長しません。」 インターネットをください-gstreamerはあなたになります。







インターネット



次のメディアが来る...さて、あなたは2つの青い円を持っているどんな種類の関連付けですか？ ダーツのゲーム？ Bluetooth？ ディスクで動作する何か？ 間違っています、これはインターネットブラウザです！

もちろん、Firefoxで表されます。 少し奇妙に見えますが、機能を果たします。

メニューはありません。右クリックメニューも禁止されています。 しかし、ユーザーはどのようにしてテキストをお気に入りのペーストボードにコピーしますか？ 誰もがホットキーについて知っていますか？ ちなみに、Moblinはどこでも正しいボタンには反応しません。PKMはシェルとは別に実行されるアプリケーションでのみ動作します。







人



ここでは、ビデオから判断すると、IMクライアントは見つかっているはずですが、インターネットなしでは見られませんでした。 IMクライアント自体はEmpathyで表されます。







ステータス



次の[ステータス]タブで、インターネットがないときに再び鼻を突かれました。







基本的には以上です。

繰り返しますが、インターフェース自体は素晴らしいです。 それは非常に迅速に機能し、画面スペースは非常に経済的に消費されます。 アプリケーションを起動すると、トップバーは孤独に非表示になり、スーパー/ウィンボタンが押されたとき、またはカーソルが一番上に移動したときにのみ表示されます。



ただし、この「リリース」リリースは、すべての要望に基づいて名前を付けるのは困難です。 それはベータ版ではなく、初期のアルファに似ており、「私たちが作ったクールなインターフェイスを見てください」という言葉が特徴です。



文字通り1年半で、Moblinはこのすべての理由をネットブックユーザーにアドバイスできると思います。



今-いいえ、試してさえいません、悲しいかな、オプションはありません。
