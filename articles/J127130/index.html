Bada SDK 2.0リリース

最近リリースされたSamsung Bada 2.0モバイルプラットフォームに続いて、その開発ツールが登場しました。 新しいSDKはここからダウンロードできます 。また、以下の点がイノベーションから注目されます。



  • マルチタスク。 Badaの主な欠点の1つは取り除かれました（SDK 1.1はバックグラウンドでアプリケーションを起動する方法を説明していましたが、開発者は正直に「疑似マルチタスク」と呼んでいました）。

  • FlashおよびHTML5のサポートの改善。

  • 開発者はプラットフォームバッファにアクセスできました。

  • 新しいナビゲーションスタイルのフッター、ヘッダー、およびTabBar。 UI 2.0と呼ばれるこの機能は、コントロールのブロックを、画面の下部、上部、および水平メニューの種類でそれぞれ作成する機能を表します。

  • 相対レイアウトが可能なレイアウトに追加されました。 説明から詳細を想像することは困難ですが、明らかに、機能はAndroidのRelativeLayoutに似ています。

  • NFC（Near Field Communication）テクノロジーのサポートが追加されました。これは、非接触型支払いの使用に関して最もよく言及されています。

  • アプリケーション間でデータを交換するために、 ServerChannelとClientChannelの 2つの新しいクラスが追加されました。 名前は比較的大きいですが、AndroidのIntentsからの違い（または類似性？）も明確ではありません。

  • Wi-Fi Directを管理するための特別なクラス-ホットスポットなしでデバイスを通信する機能。

  • ターゲット広告をサポートする新しいAPI。 ImageBannerとTextBannerの2つのコンポーネントが提供されています。 クリックで利用できるアクションは次のとおりです。広告主のURLを使用してブラウザにアクセスし、広告主から提供された番号に自動的にダイヤルします（詐欺師の可能性はありますか？）、Samsung Appsストアに移動します。
