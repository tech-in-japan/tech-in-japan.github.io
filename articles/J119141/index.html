ABBYYを使用したスポーツプログラミング

学部についての投稿を書いたとき、5月にABBYY Cup学生オンラインプログラミングコンテストが開催されることに言及しました。 それについてもっと話す時間です。



ABBYYカップは5月15日日曜日の10:00に始まります。 どんな生徒でも手を試すことができます。 参加者には、さまざまな複雑さの5つのタスクが提供され、その解決には8時間かかります。 他のオリンピアードのプログラミングと同様に、最大ポイントを獲得するには、タスクを解決するだけでなく、使用する時間とメモリに関する特定の制限を満たす必要があります。



決定は自動テストを使用して評価されます-プログラムが合格するほど、スコアが高くなります。 攻撃的なミスを排除するために、オリンピック中にオープンテストでコードをテストすることができます。 C / C ++の問題を解決することをお勧めします。 原則として、別の言語（Pascal、Ruby、Python、Perl、Java）を選択できますが、覚えておいてください：すべてのテストに合格できるソリューションが存在することを保証するものではありません（それに応じて、最大ポイントを獲得します） ！



参加するかどうかについて長い間考えないでください！ 登録して、競技が始まるのを待ちます。 ABBYY Cupは、スポーツプログラミングスキルをテストする方法ではありません。 むしろ、このようなコンテストに一度も参加したことがなく、オリンピアードプログラミングの形式に慣れていない学生にとっては、非標準的な課題について急いで考えることなく、解決策を提案する機会です。 何かを最後まで解決できない場合は、とにかくコードを送信してください。 ほとんどすべてのタスクについて、ポイントが付与される部分的なソリューションがあります。



さて、高い結果にコミットしている人のために、貴重な賞品に加えて、ABBYYカップの参加者は当社を訪問する機会があり、モスクワ物理学研究所の画像認識およびテキスト処理部門または雇用時に選択する際の入学試験としてオリンピックの結果を数える機会がありますABBYYで。
