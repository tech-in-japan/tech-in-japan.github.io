Google Summer of Code 2008

明日、3月24日、学生および大学院生からのGoogle Summer of Code 2008への参加申請を受け付け始めます。



Google Summer of Codeは、多くのオープンソース製品の1つであるPostgreSQL、Python、PHP、FreeBSD、その他多数のプロジェクトに取り組むことで、夏に余分なお金を稼ぐ良い機会です（今年参加しているプロジェクトのリストはここにあります： http：// code .google.com / soc / ）。



参加するには何が必要ですか？ 学生または大学院生であること、興味のあるプロジェクトで使用されている言語で基本的なプログラミングスキルと基本的な開発経験を持ち、夏には自由時間（2か月目）を持っていること。 次に、トピックを作成する必要があります（原則として、これは追加モジュール、追加機能、改良であり、まれに個別のサブプロジェクトです）。提案を作成する必要があります。 もちろん、選択したプロジェクトの既存の開発者の1人、理想的にはあなたの将来のメンターのサポートを求めるといいでしょう。



急いで、申し込み受付の締め切りは3月31日です。



PS PostgreSQLの場合、アイデアと手順のリストはここにあります： http : //www.postgresql.org/about/news.931
