トレントからダウンロードする方法、良い評価を得ると同時に匿名のままにする方法

この記事は私の個人的な経験に基づいています 。 繰り返し成功するには、PayPalが必要です 。ロシアとウクライナからの発送が可能です。



torrents.ruの悲しいイベントのずっと前に、私は匿名性を確保すると同時にDemonoidトラッカーを使用することにしました。 私はウクライナ出身なので、そこへのアクセスは閉鎖されました。



その後、私は少量（最大50ドル）を費やし、トレントをダウンロードしたり、サーバーIPからサーフィンしたり（デモノイドの問題にさよなら）できるサーバーをレンタルし、匿名性を維持しながら必要なファイルを静かにダウンロードすることにしました。



まず、サーバー。 私のニーズには低電力サーバーで十分ですが、十分なトラフィックとディスクが必要です。 検索した後、私は最も安いオプションを見つけました-この構成ではサーバーあたりわずか39ドルです：



  • Atom Dual-Core 1.6Ghzプロセッサー
  • メモリ1GB Ram 800mhz
  • 250GBドライブ
  • 2000GBのトラフィック
  • 5つのIPアドレス（alas、同じサブネットから）
  • 100MBチャネル（米国でのみ真）




このリンクでサーバーを注文できますが、覚えておいてください-その数は限られており、すべての人にとって十分ではないかもしれません。



クリーンなCentOS 5ディストリビューションを使用して、コントロールパネルなしでサーバーを注文したところ、メールボックスにルートユーザーパスワードが届き、それを管理するにはsshクライアントが必要になります。 Windowsの場合、 ここから入手できます。Linux、FreeBSD、およびMacOSではバンドルされています。



sshポートをいくつかの非標準に変更することをお勧めします。

echo "Port 30022" >> /etc/ssh/sshd_config

service sshd restart







サーバーIPからサイトに登るには、プロキシが必要です。 3proxyを使用することをお勧めしますが、残念ながらCentOSはパッケージに含まれていません。 同様に、および他の多くの便利なソフトウェア。 これは、追加のリポジトリRpmForge 、 CentALT、およびEpelをインストールすることにより、非常に簡単に修正できます。



yumを使用すると、必要なソフトウェアを配置できるようになります。

yum install 3proxy

パスワードを使用してanonユーザーを作成します。ブラウザのプロキシ設定は、サーバーIPとポート3128になります。



それでは、急流に移りましょう。 コンソール管理を備えた非常にシンプルなクライアントがあります。 もちろん、必要に応じて、Webインターフェースも接続できますが、1GBのRAMではそのような要望はすぐに消えます;）

実際、複雑なことはなく、ユーザーから実行して画面で実行するだけです。

yum install screen rtorrent







次に、kachalkaなどのユーザーを作成し、パスワードを設定します。

adduser kachalka

サーバーからファイルをアップロードまたはダウンロードする最速の方法はFTPです。 個人的に、私はproftpdを使用しています。特に、作成されたユーザーkachalkaを即座に検出するためです。

yum install proftpd

通常、ブラウザを介してすべてのトレントを別のディレクトリにアップロードするため、カチャルカにそれを正確に実行させ、そこにすべてのファイルをアップロードし、FTP経由でダウンロードしたファイルをアップロードするのが賢明です。



そして今、最も重要なことは、sshを介してrtorrentを管理する方法です。

まず、画面を作成します： screen



。 これは、サーバーへの接続を閉じるときにハングしたままにすることができる追加のコンソールです。

画面では、カチャルカのユーザーになります。

su - kachalka





そして、トレントに設定してrtorrentを開始します。

rtorrent -p 20000-30000 directory-with-torrents / *。トレント



rtorrentコントロールメニューが表示されます。 コントロールは非常に簡単です。上/下矢印を使用してダウンロードを移動し、左/右-各ダウンロードの詳細情報を入力します。

追加のキーの組み合わせ：

  • CTRL + q-最初に押すとトラッカーに作業を完了するための信号が送信され、2番目のボタンを押すと送信せずにクライアントがオフになります。
  • CTRL + s-ダウンロードを開始します
  • CTRL + d-ダウンロードを停止し、もう一度押すと削除


残りのそれほど重要ではない組み合わせは、ドキュメントで見つけることができます： man rtorrent



またはコメントで私に尋ねることによって。



最後に、画面の制御について。 コンソールを稼働状態のままにしてメインに戻すには、次の組み合わせを使用します。

CTRL + ad-最初にControlとa、次にd Mortal Combatをプレイした場合、この組み合わせは問題を引き起こしません;）

そして、次のコマンドを使用してメインコンソールから戻ることができます： screen -r
