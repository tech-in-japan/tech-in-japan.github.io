オンラインストアのコンバージョンの増加。 レビューと「安全な購入」サイン付きのウィジェットのA / Bテスト

みなさんこんにちは、私の名前はイリヤです。

2013年以来、KupiVIPホールディングで働いています。



KupiVIP.ruオンラインストア自体に加えて、同社にはKupiVIP E-Commerce Servicesユニット（以下KupiVIP ECS）があり、ファッションブランドのオンラインストアの作成と開発を行っています。 この保有は、「安全な購入」サービスもサポートします。このサービスの目的は、オンラインストアおよびオンライン取引全般に対する顧客の信頼を高めることです。



2013年以来、「Safe Purchase」はKupiVIP ECS（ブランドのオンラインストア：MEXX、TOM TAILOR、ALBA）が作成したサイトでサービスのA / Bテストを実施しています。



2013年、6つのA / Bテストが実施されました。 2014年末から2015年初頭にかけて、3つのA / Bテストが実施されました。



投稿は、テストと調査結果の説明に専念します。



以下の理由から、 Habr Megamindの聴衆にこれらのテストの結果を知らせることが重要だと思います。

  • A / Bテスト結果は、オンラインストアに対する顧客の信頼を高め、コンバージョンを増やしたいストア、特にサイトへの新しい訪問者に役立ちます。
  • レビューの使用と配置に関する推奨事項を提供します。レビューを自分で収集するか、外部サービスを使用するかは関係ありません（「安全な購入」サービスに加えて、独自の特性を持つ他の2-3のロシアのカウンターパートを見つけることができます）
  • 検出されたすべての陽性および陰性結果を含む、受け取った2013年および2014/15年のA / Bテストの実際の数を表示します
  • 同様のテストに関するフィードバックを受け取りたいと思います。たとえば、サイトの「信頼」（信頼を構築する）要素（ウィジェット、レビュー付きブロック、信頼標識など）の最適な場所を探索した人などd。）




Megamindのルールに従って、この投稿ではオンラインストア自体のWebサイトへのリンクを示していません。 ウィジェットの現在の例をグーグルする必要があります。



ABテストの目的、テストの目的、実行方法については、 「A / Bテストは簡単です」という記事をご覧ください。



A / Bテストの実施方法、信頼性のレベルの確認、その他の同様のテストの例に関する情報は、この記事の最後にあります。

結果の解釈





すべてのプロジェクトに共通：



  1. 登録への影響は、MEXXブランドのプロジェクトでのみ顕著でした。これは、テストの期間（MEXX Webサイトではテストが3か月以上続いた）と聴衆の特性の両方に関連します。 将来的には、特別なランディングページのみを残して、別の登録ページを放棄する予定です。 「安全な購入」では、そのようなページ用の特別なウィジェットを作成してテストすることも計画しています。

  2. 2013年と2014-2015年のテスト結果の大きな違いも顕著です。 たとえば、2013年には、ブランドプロジェクトMEXXのサインのみがコンバージョンの大幅な増加をもたらしました。 これは次のように解釈できます。

    • テストされたプロジェクトでは、広告戦略が大幅に変更され、トラフィック量と購入数から店舗にとって有益な高品質のプロモーションチャネルと注文（つまり、領収書の量が追跡され、注文品、配達都市など）に焦点が移りました。トラストサインが顕著な役割を果たし得るもの
    • 2013年、MEXXブランドプロジェクトは小売店とも積極的に協力しました。 このサイトには、小売店のオンラインストアのサイトについて学んだ人々が訪れました。 これらの人々は、オンラインショッパーの経験が少なく、信頼の兆候が彼らに大きな影響を与えました。




さらに、A / Bテストの結果に対する次の要因の影響をより詳細に分析する予定です。使用したブラウザ、居住地域、購入者が来た広告プラットフォームなど。

  2. カートページでトークンまたはSafe Purchaseウィジェットを使用すると、コンバージョンが増加します。

  2. カートページのウィジェットとサインの両方のプラスの効果。

  1. 登録の変換に期待される効果が得られませんでした。

    MEXXブランドのプロジェクトと同じ計画-別の登録ページを放棄し、そのようなページ用の特別なウィジェットを開発し、その後テストを繰り返します。
  2. バスケットでコンバージョンがあいまいに増加しました。つまり、新しい訪問者にとっては改善が顕著ですが、古い訪問者にとってはそれほど明白ではありません。 上記の説明、プロジェクトの視聴者とその広告戦略の考えられる理由を説明しました。






オンラインショッピングの推奨事項：



得られた結果は、顧客から受け取ったフィードバックのよりアクセスしやすい表示の必要性を示しています。 このコンテンツをサイトに置いて、フッターの別のページやサイトの目立たないページに配置するだけでは不十分です。サイトで既に購入した人の意見を顧客に示す必要があります。 ウィジェットは、サイトの注目すべき要素であり、メインデザインから際立っており、注目を集める必要があります。



推奨事項：

  • 既にコンテンツがある場合は、製品カタログ、製品カード、またはカートページの意思決定ページ（ユーザーからのさらなるアクションが期待されるページ）で配置をテストしてみてください（チェックアウト前のページにレビューを投稿できます）あなたは）;
  • コンテンツをよりコンパクトに配置するために、クリックしたときにいくつかのシンプルなウィジェット（写真）を準備できます。利用可能なすべてのレビューが購入者に表示されます。
  • 否定的なものも含めて、すべてのレビューを必ず公開してください。それらに正しく対応し、購入者から通知された問題を解決することを忘れないでください。
  • 顧客からのフィードバックをまだ収集していない場合は、間違いなくこれを開始する必要があります




独自のアイデアを含むさまざまなサービスのソリューションを必ずテストしてください。これらのサービスの1つは、店舗の売上を大幅に増加させる可能性があります。 たとえば、次の月額料金の平均的な店舗の場合：



  • 100,000人の訪問者
  • 1％のコンバージョン
  • 1000注文
  • 5000ルーブルの平均チェック


コンバージョンが10％増加すると、1か月あたりの総売上は500,000ルーブル増加します。 自分に合ったソリューションを見つけてテストするための相当な量。





「安全な購入」のサービス開発計画
