Web Camps Training Kit更新バージョンがリリースされました





Web Camps会議に付随するように開発されたトレーニング資料パッケージの更新バージョンがリリースされました。これは、Web開発のトピックに関してMicrosoftによって定期的かつ世界的に実施されています。



このキットには、マイクロソフトが開発または促進した最新のWebテクノロジーに関するトレーニング資料が含まれています。 その中には、ASP.NET MVC 3、Internet Explorer、HTML5 / CSS3 / JavaScript Web標準、Odataオープンプロトコル、jQueryライブラリ、Webアプリケーション、CMS、新しいWebMatrixアプリケーション開発パッケージがあります。



Web Campsトレーニングキットには、完全な段階的な説明を含むプレゼンテーション、デモ、およびラボが含まれています。 さらに、バンドルされた開発者は、WebMatrix開発環境で無料の本や他の資料を見つけるでしょう。



Web Campsトレーニングキットの概要：



ASP.NET MVC 3
  • ASP.NET MVC 3の基礎
  • ASP.NET MVC 3 Razor
  • ASP.NET MVC 3モデルとデータアクセス
  • ASP.NET MVC 3ヘルパー、フォーム、および検証
  • 検証、データ注釈、クライアント側の検証
IE9およびHTML5
  • 高速なWebエクスペリエンスの構築
  • IE9開発の概要
  • IE9の固定サイト
Odata
  • オープンWeb用のオープンデータ
  • セッション1-ODataの概念
  • セッション2-ODataサービスの構築
  • セッション3-ODataサービスの利用
jQuery
  • セッション1-jQueryの基礎
  • セッション2-Ajax
  • セッション3-テンプレートとjQuery UI
Webapps
  • Umbraco CMS
  • Kentico CMS
  • Dotnetnuke cms
Webmatrix
  • 学習Webマトリックス（3部）
  • WebMatrix eBook
  • WebMatrixホワイトペーパー
  • WebMatrixを使用したWebページの作成：学生向け6部構成のチュートリアル
