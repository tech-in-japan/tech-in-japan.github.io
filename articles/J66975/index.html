elance.comから追い出されたベラルーシの開発者

8月12日に、elance.comは取引所での作業規則を変更しました。 現在、登録時にベラルーシの居住国を示すことはできません。 また、ベラルーシの既存のアカウントはすべて、30日以内にすべてのアクティビティを削減するよう招待されました。



エランスポリシー、8月12日午後12時47分：



親愛なる...



残念なことに、国際制裁のために、Elananceが登録国としてベラルーシをサポートすることはもはや商業的に実行不可能であることをお知らせする必要があります。 このため、Elanceアカウントは無効になりました。 アクティブなクライアントを持っているアカウントの場合、既存のエンゲージメントを終了するまで30日かかります。 お客様のビジネスをより長くサポートできることに失望していますが、それは当社の管理下にはありません。 よろしくお願いします。



詳細については、 www.ustreas.gov / offices / enforcement / ofac / programs / belarus / belarus.shtmlをご覧ください。
