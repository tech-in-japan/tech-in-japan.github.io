ルーターのプレフィックス（USB HUB + USB HDD + UPS）



USBポートを備えたかなり強力なルーター（ASUS RT-N16でした）を入手した後、ルーターを最大限に使用するにはどうすればよいのでしょうか？

必要なもの：

  • ネットワークストレージ
  • 急流ロッキング、
  • そしてもちろん、電力サージ/停電中にこれらすべてが落ちないように。




コンセプト



最も簡単な実装オプションは、USB HDD、USB HUB、UPS（新規または使用済み）をルーターに接続することですが、このオプションは好きではありませんでした。 これらはすべてキャビネットの上にまとめて置かれ、私はそれをすべて1つの箱に入れる方法について考え始め、まともな外観にさえなりました。



コンポーネントの選択



電源/ UPS


まず、ルーター自体とすべての周辺機器には定電圧電源が供給されており、UPSには12 V出力がなく、電源を使用する必要があるため、バッファーバッテリーを備えた電源を優先してUPSの使用を拒否しました。 インターネットで検索すると、いくつかのMean Wellモデルが得られましたが、残念ながら、安価なアナログは見つかりませんでした。

  1. SCP-50-12（ pdfへのリンク ）
  2. AD-55A（ PDFへのリンク ）
  3. ADD-55A（ PDFへのリンク ）
  4. PSC-60A（ PDFへのリンク ）


上記のすべての中で、ADD-55Aは際立っています。 12 Vと5 Vの2つの出力があり、USB周辺機器を接続するのに非常に便利ですが、価格も高く（約1,500 p。）、最も全体的です。 AD-55AはADD-55Aと同じであり、5V出力はありませんSCP-50-12は最も単純です：ダイオードを介した単純な充電回路であり、過放電に対するバッテリー保護はありません。 PSC-60Aは、バッテリーを保護する機能を備えており、寸法は小さくなっています。

ある電気店で、彼はAD-55Aを約1,100 pで見つけました。 全体（152.5 x 97 mm）、および密閉されていないコンポーネントを含む多くの空きスペース（明らかに、ADD-55Aを備えた1つのボード）。

ADD-55AまたはPSC-60Aを購入することにしました。 ADD-55Aを販売するための検索は失敗し、1個のバッチとの接触を希望する人はいませんでした。 PSC-60Aのさらなる幸運：ある会社は、886 pのパッケージ（PSC-60A-C）のバージョンを私に販売しました。

ドライブとUSB周辺機器に電力を供給するために、LM2575チップで5 V、1 Aの自家製DC-DCコンバーターを使用しました。



バッテリー


バッテリーは会社Leoch DJWシリーズによって選ばれました。 前にこれらを使用しました。 次に、利用可能なものから選択し、容量と寸法の点で私に合っています。

なぜなら 長い間、私の電気はほとんどめったに消えず、2、3分間停電することが多かったので、30〜60分間十分な自律性が必要だと判断し、DJW12-3.2バッテリー（12 V、3.2 Ah）を使いました



USBハブ


ここではすべてが簡単です。 私は、4つのUSBポートを備え、最後に1つのUSBを備えた安価なものを購入し、それを使用してハードドライブを接続しました。



HDD


次のように、デスクトップPCではなくラップトップにハードドライブを使用することにしました。 それはサイズが小さく、消費電力が少なく、12 Vの電力を必要としません。ところで、私はすでにUSB-SATAボックスを持っています。 メガボールをするつもりはなかったので、320 GBのディスクを購入することができました。これで初めてです。



本体


ケースはもう少し複雑です-私は自分でやりたくありませんでしたが、小さなPCのケースは高価なので、GIANTA REAのケースを購入することにしました。 彼らは店にいます。 すべてのコンポーネントの寸法を把握し、在庫品を探してG753Aケースを購入しました。その寸法は260 x 180 x 85 mmです。

他に何をやりたいですか



次に、USBポートの下にある切り欠きの不均一なエッジを隠すために、前面部分にステッカーを作成する予定です。 さて、冷却の問題は半分閉じたままでしたが、負荷の下で40°Cは正常ですか？ たとえば、この記事を書いている間、私のラップトップのドライブは41°Cまで暖まりましたが、ブラウザのみが開いていますが、タブがたくさんあります。



配線でルーターとUSBの電源を引き出しましたが、「良い方法で」コネクタを後壁に配置する必要がありました。



UPSと比較したバッファバッテリからのAC-DCの欠点は、その「制御不能性」です。ルータは、バッテリで動作するか、主電源で動作するかを知りません。 この欠点は、端末のステータス情報「AC OK」および「Batt。」をルーターに送信することで補うことができます。 低»PSC-60A、これらの目的のために、USBコントローラーを介して接続し、ルーター用の簡単なスクリプトを作成して、1分ごとに起動して問い合わせを行うことができます。



それでも、すべてをよりコンパクトに配置する場合は、おそらく、ルーターの内臓を統合することもできます。
