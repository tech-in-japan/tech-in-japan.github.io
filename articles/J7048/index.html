Windows Really Good Edition

Windows Vistaへのアップグレードを検討していますか？ 価値のある代替手段があります-Windows Really Good Edition（Windows RG）。 ここでWindows RGのデモバージョンを参照してください 。 すぐに右の列の[全画面]をクリックして、全画面で表示できます。 まず、Wordを試すことをお勧めします（スタート-> Word）。 リストの次へ。 デスクトップ上のアイコンを忘れないでください。 すべてのルールに従って作業を完了します：開始->シャットダウン。
