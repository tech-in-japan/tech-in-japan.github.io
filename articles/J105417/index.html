ロシア向けに導入された無料のOffice Web Apps、新機能





遅ればせながら興味深いニュース。 公式のMicrosoft Officeブログでは、無料のオンラインオフィスOffice Web Appsのユーザーに関する統計とともに情報が表示され、さらに新機能の発表が行われました。



ブログによると、Office Web Appsのリリースから100日で、2,000万人のユーザーがその機能を利用しています。



同じブログエントリは、ロシアを含むさらに7つの国でOffice Web Appsが利用可能になったことを発表しています 。 ロシアに加えて、オンラインオフィスへのアクセスは、オーストラリア、オーストリア、ベルギー、フランス、ドイツ、スイスから受けました。 これ以前は、アメリカ、イギリス、カナダ、アイルランドの4か国のみが接続されていました。



さらに、開発者はOffice Web Appsのいくつかの重要な技術革新を発表しています。



実装されたドキュメント


ユーザーは、iframeベースのコードを使用して、ExcelおよびPowerPointドキュメントをサイトやブログに埋め込むことができます。 同時に、Excelデータを編集してPowerPointプレゼンテーションを表示することもできます。 ブログに例を示します。ここでは、プレゼンテーションをスクロールして、テーブルファイルを編集できます。







Excel Webアプリの革新


ブラウザとOffice Web Appsを使用して、Excelドキュメントにグラフィックを埋め込むことができるようになりました。







さらに、選択された範囲が選択された値で満たされる場合の自動入力機能のサポートがありました。







これに加えて、モバイルWebブラウザーからExcel Web Appsを介してドキュメントを表示することもできます（以前は、WordとPoerPointのみがサポートされていました）。



PowerPoint Webアプリの革新


さまざまなマイクロソフトパートナーのオンラインストレージからクリップアート、無料の写真、イラスト、グラフィックを挿入する機能を追加しました。







さらに多くのプレゼンテーションテーマを追加しました。



Word Webアプリの革新


これで、ドキュメント編集モードで印刷できます。



SkyDriveストレージの革新


無料のSkyDriveストレージでは、保存済みのオフィスドキュメントをデスクトップバージョンのオフィススイートで開くことができます。
