NetWrix Change Reporter Suiteの新バージョンは、ITインフラストラクチャの信頼できる制御を提供します





先週、私たちはすでにActive Directory監査ソリューションの新しいバージョンであるNetWrix Active Directory Change Reporter 7.2のリリースについて書きました。 しかし、これらのイベントに加えて、別の重要なリリースがありました-NetWrix Change Reporter Suiteソフトウェアパッケージには、バージョン3.2への更新が含まれています。 ソリューションは、Windowsおよびsyslog用のActive Directory監査モジュールと新しいイベントログ統合モジュールの更新を受け取りました。







バージョン3.2は何を提供しますか？

  • ソフトウェアパッケージのインストールがさらに簡単になりました-バージョン3.2では、 Advances Services （Windows Server 2008用のSQL Server 2008、Windows Server 2003 以前）のSQL Server 2012 Express Editionの自動インストールと構成が追加されました。
  • Active Directory監査モジュール-NetWrix Active Directory Change Reporter -Windows Server 2012 / Windows 8およびExchange 2013のサポートを受けました。サポートされているすべてのMicrosoftオペレーティングシステムでActive Directory監査が可能になりました
  • モジュールは、Exchange Server（Exchange Change ReporterがExchange 2013をサポートするように）、グループポリシー（グループポリシー変更レポーターがWindows Server 2012 / Windows 8で動作する）、およびADオブジェクトの回復を監査するために更新されました
  • NetWrix Event Log Managerイベントを管理するためのモジュールが追加されました。 その助けにより、Windowsおよびsyslogでのイベントログの統合とアーカイブが実行され、個々のイベントに関するアラートを受信します。 NetWrix Change Reporter Suite 3.2にこのモジュールが追加されたことにより、ITインフラストラクチャの変更に関する30以上の新しいレポートが利用可能になりました。
  • SQL Server Change Reporter 2.5モジュールによるSQL Server 2012データベースの構成とコンテンツの監査
  • ソフトウェアパッケージ全体をWindows Server 2012およびWindows 8にインストールできるようになり、最新のインフラストラクチャで安定した信頼性の高い動作が保証されます。 パッケージに含まれるすべての製品がWindows Server 2012ドメインでの作業をサポートするようになりました
  • また、 NetWrix管理コンソールの更新を受け取りました。これにより、NetWrix Change Reporter Suiteを構成するすべてのコンポーネントの管理が容易になります。
