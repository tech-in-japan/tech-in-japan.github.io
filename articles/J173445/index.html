Tech Peaksをロシア語で見る：Trentoでスタートアップを立ち上げる機会を得る方法





スタートアップの世界のニュースに興味があるなら、今年イタリアの当局が、イタリアの土地で才能を伸ばすことを目標とする高山の町トレントでTech Peaksアクセラレータを立ち上げたことを聞いたことがあるかもしれません。 もちろん、技術および通信プロジェクトの分野の才能。 イタリア人の目標はシンプルで簡単です。トレントで自分のシリコンバレーのようなものを作ることです。 これを行うために、彼らは世界中の人々を探して招待し、彼ら自身のプロジェクト、無料の宿泊施設、食事の支払い、「小遣いのための」奨学金を立ち上げるための助成金を提供します。 現在、プログラムは東ヨーロッパの電話を発表し、東ヨーロッパ、ロシア、バルカン諸国の人々が待っています。 通常のビジネスアクセラレータとは異なり、プロジェクトの焦点は、人材からチーム、チームからスタートアップまでの人々にあります。



Tech Peaksの参加者は、国際チームの一員として働き、世界クラスのメンターから専門知識を受け取り、ヨーロッパのビジネス環境での能力を評価し、最後にプロジェクトを立ち上げるための助成金を受け取ることができます。 ロシアと東ヨーロッパからの何百人もの幸運な人々にとって、これは新しいレベルへの本当のブレークスルーをする機会です。 アプリケーションはまだ受け入れられているので、プロジェクトのロシア人キュレーターであるDaria Batukhtina （habrayuzer darena ）にいくつかの質問をしました。TechPeaksに入ることができるのか、落とし穴は何か、誰がこれを必要とするのか。



-こんにちは。 これは誰のイニシアチブですか？ プロジェクトのアイデアは何ですか？

-Tech Peaksは全体として、ヨーロッパで最も技術的に進歩したトレント地域のイニシアチブです。 そして彼女の主なアイデアは、東ヨーロッパからのICT起業家のためのヨーロッパおよびグローバル市場へのアクセスを開くことです。 このプログラムの創設者は、トレント地域をビジネスや投資家にとって革新的で快適な環境にするために可能な限りのことをする組織トレントライズと、教育活動全体から350人以上の大学院生と研究医師を集めたFBK情報技術センターです。世界の。







Tech Peaksプログラムは、欧州委員会が率いる主要なEU都市のイノベーションクラスターの原則によって組織された最大の専門コミュニティであるEIT Labs（欧州イノベーション技術研究所）と緊密に連携して機能します。 彼らは、実際にプロトタイプやビジネスアイデアをテストするための包括的な実験室としての都市のアイデアを促進する人々です。







-明白な質問：これから誰が恩恵を受けますか？ これは何らかのビジネスインキュベーターですか？ プロジェクトの作者は何に期待していますか？

-プログラムの主催者の主な目標は、国際的な経験と知識との統合を通じて、起業家精神における地元のイタリア人コミュニティの関心を高めることです。 Tech Peaksプログラムおよびその他の活動により、Trentoはイノベーションの中心、それらをテストするための研究所になるはずです。 このため、すべての快適な条件がプログラム参加者のために作成されます...



「それで、彼らは何を得ますか？」

-まず、イタリアへの6か月のビザ。 第二に、プログラム全体の期間中の無料宿泊、および便利な職場、月額500ユーロの食料と小遣いの費用をカバーします。 Tech Peaksからのプロジェクト開発のための助成金も提供されますが、このお金であなたのスタートアップのシェアを譲ることはありません。 それどころか、彼らはさらにグローバルなビジネスを立ち上げるためにあなたがイタリアで会社を開くのを助けます。



-いつ開始され、プログラムの選択はどれくらい続きますか？

-セレクションはすでに本格的です... アプリケーションの締め切りは 4月5日です。したがって、国際的な体験をしてみたい場合は、決断を下すときです。 応募するには、Tech PeaksのWebサイトでアンケートに記入し、3分間のビデオへのリンクを送信する必要があります。このリンクでは、プログラムへの応募を決めた理由とあなたが最高の候補者です。



-英語で申請書を提出する必要がありますか？ 選択基準は何ですか？

はい、もちろんです。 これは国際的なプロジェクトであるため、競争委員会も主に外国人です。 ITプロフェッショナルとプログラマーにとって、どのタスクをどのように解決し、どの程度成功したかを示すことが最善です。 ポートフォリオは成功したプロジェクトと失敗したプロジェクトの両方で構成できることを完全に理解しています。主なことは、「箱から出して」考えることができ、起業家精神に真剣に取り組むことを委員会に明確にすることです。 デザイナーにとって重要な要素は、「影響力を持つエージェント」としての自分の役割を理解することです。なぜなら、潜在的な顧客/ユーザーの蓄積された経験を視覚化するのはスタートアップの人々だからです。 まあ、マネージャー、ビジネス開発者にとって、選択基準は積極的であり、ユーザーから何度も何度も市場から必要なデータを取得する能力です。ビジネスアイデアを試して失敗することは怖くないです。 あきらめると怖いです。 したがって、これらの候補者は、リーダーシップスキルと好奇心、粘り強さと柔軟性を同時に学びます。



-誰が委員会に参加していますか？

-委員会は、技術、デザイン、HRの分野の国際的な専門家、および「次の大きなもの」に完全に発展した香りを持つシリアル起業家で構成されています。 彼らの名前はTech Peaksでまもなく利用可能になります。



-何人が選択されますか？ ロシアからですか？

-合計で、東ヨーロッパとロシアから100人が選ばれます。 はい、これは非常に厳しい選択であることは知っていますが、Tech Peaksプログラムによって提供される機会はユニークです。誰もが参加者に快適な仕事と仕事のために与えられているためです-オフィス、住宅、食料、小遣い、インフラストラクチャ、およびメンタリングベース。 この「種まき」、チーム、またはスタートアップから何かが出たときに、彼らは後でそれを取ります...今日のヨーロッパでは、これはまったく新しい取り組みです。 そして、あなたが半年の間、国際的な起業家精神の完全に予測不可能な雰囲気に没頭し、吸収し、勉強し、スタートアップを作成し、そして申請書を提出する準備ができていると判断した場合！ そうでない場合は、この機会を、より決意があり、変化と個人的な成長の準備ができている他の誰かから離さない方が良いでしょう。



-イタリアに到着した後はどうなりますか？

-適応プロセスが開始されます。 もちろん、最初の10日間は、参加者は異なるネットワーク形式（StartUp Weekend、Lean Start-UP Machine、Teambuildingなど）でお互いに会います。 チームワークは常に共通の関心と同情に基づいているため、最初の日はできるだけ創造的であり、参加者は特定のタスクを実行するときにチームで絶えず変化することを望んでいます。



次に、アイデアとプレゼンテーションの期間が来ます。約1か月半の間、新しく造成されたチームがプロジェクトを審査員に発表します。 そして、メンターと専門家のju審員がアイデアが生きて発展できると決定した場合、チームはスタートアップの株式を返さずに25,000ユーロの助成金を受け取ります。 助成金は公開されているため、チームはイタリアに会社を設立する必要があり、Tech Peaksプログラムの主催者はそれらを支援します。

そして、ワークフローが始まり、チームの残りの3か月は製品のリリース準備をしています。 また、11月末には、デモデイでの作業結果を提供します。

そのため、Tech Peaksプログラムはユニークで、参加者をチームビルディング、アイデアの作成、実際のテスト、プレゼンテーション、開発、評価のいくつかの段階に進めます。 すべてリアルタイムで。 なんらかの理由でスタートアップを取得できない場合でも、独自の起業家としての経験、国際的なスタートアップエコシステムとの関連性、そしておそらくあなたのビジョンや考え方を変えるでしょう...



-クライアントがここで仕事をしているとします。 彼がまったくそれらを終了できない場合はどうなりますか？

-誰もあなたがすべてを落として橋を燃やす必要はありません、それはあなたの欲求に依存します。 ロシアからの注文数を最小限に抑えて、プログラムの下での活動に参加したり、スタートアップに取り組むための十分な時間を確保することができます。 しかし、いずれにせよ、あなたは認めなければならない、あなたは明らかに両方を同時にうまく扱うことができないでしょう。 Tech Peaksプログラムは、参加者に6か月間必要なものすべてを提供するため、幸運で選択に合格した場合は、おそらくそれとその参加の結果を最大限に活用することに集中する必要があります...



-参加者は、出席する講義、会議、イベントを自由に選択できますか？

-最初の2、3か月は、イベントへの参加がほぼ必須です。これは、世界で何が起こっているのかを把握し、ビジネスアイデアの開発または改良への最初の弾みをつける良いチャンスだからです。 ご存知のように、このような環境にとどまることは、プロジェクトでの長年の仕事よりも多くの思考の糧を彼らに与えると、スタートアップからよく耳にします...だから結論を引き出します-あなたがすでにそのような国際的な雰囲気に陥った場合、参加する価値があるのですか？



プログラムの3か月目に、製品開発に重点が置かれたアクティブな作業フェーズが開始されます。 イベントが少なくなり、チームワークが計画されます。 また、先月、ヨーロッパでのグローバルデモデイでのパフォーマンスとプレゼンテーションの準備が行われます。



-参加者には特定のタスク、期限がありますか？ 彼らは彼を「追放」できますか？

-いい質問です。 そして、目標を達成したり問題を解決したりするために、何らかの鞭が必要なのはなぜですか？ Tech Peaksプログラムの各参加者は、独自の内部規律を持っている必要があると考えています。これにより、プロセスから最大限の結果を得られるようになります。 労働時間の管理は起業家の手に委ねられており、見捨て、勉強、試練をやめるまで、誰も彼らを敗者の連中に「追い出す」ことはありません！



これはアルプスでの休暇ではありません。これは仕事です。何か必要な場合は、話し合い、問題を迅速に解決して、両方の当事者が投資するリソースと時間を正当化する必要があります。



-誰がメンターになりますか？ 組織的なサポートとは別に、どのようなサポートが提供されますか？

- メンターとは 、成功するビジネスの創造と投資の経験を積んだ専門家です。 その中には、有名なLean Start-Up Machine方法論の作成者、同名のベストセラーの著者であるHappifyの創始者であるTrevor Ounes、Bitlyの創始者であるAndy Parsons、YahooのベテランであるPeter Stern、UberVUの創始者であるMarvin Liao、Ruby on RailsのVladimir Oane、Gurusの第一人者です。その他多数。 ロシアのメンターもいます。多くの人がアルカディ・モレイニスを知っていると思います。 このリストは絶えず更新されています...彼らの主な仕事は、参加者とインサイダーとグローバル市場のビジョンを共有し、スタートアップの日々の発展を支援することです。 メンターのリストは絶えず拡大しており、あなたは立候補を提供することもできます-そして私たちは人を説得し、何らかの形で彼をこの物語に引き込む方法を見つけようとします（：さらに、彼らの経験と国際的なスタートアップ環境とのつながりを共有できるパートナーの専門家が関与しています。







-参加者が助成金を受け取らなかった場合、彼のさらなる行動は何ですか？

-はい、プログラムの主な目標は、「才能のスタートアップの創始者への転換」を支援し、刺激することです。 したがって、参加者が自分のアイデアの活力を初めて証明できなかった場合は、ピボットを作成するか戦略を変更することで再試行できます。 あなたが助成金を得ることができるとき、プログラムの間にいくつかのピッチセッションがあります。 また、他のチームやスタートアップに参加することもできます。 または、このプログラムでは、ヨーロッパや市場のインサイダーの課題について意見を共有するさまざまな業界の専門家が参加するInspiring Daysが常に開催されるため、何か新しいものを思いつきます。



-助成金を受け取った場合、どのように報告すべきですか？

-はい、報告があるはずですが、主催者は、お金がビジネスの開発に実際に費やされ、不動産や他の無関係なものを買わないことを理解するだけです。 これは会社の通常の会計であり、助成金を受け取った後にスタートアップがイタリアで開く必要があります。



-潜在的な参加者が既にチームを持っている場合、彼らは一緒に行くことができますか？

-はい、個々の専門家とチームの両方がTech Peaksプログラムに参加できます。 後者についてのみ、共同作業の経験、チーム内での役割の分散、およびアイデアについて話す必要がありますが、何が問題なのかを理解するために、プロトタイプまたはそれに似たものを示す方が良いです。
