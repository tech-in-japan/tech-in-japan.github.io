MS Outlookの小さいながらも非常に便利なヒント

Microsoft Exchange Serverという名前の獣がインストールされている職場では、Outlookを起動して、新しくて興味深いものを確認する必要があります。 いつものように、多くのものはそうではありません、動機付けなど。 ナンセンス。 したがって、ウィンドウと作業、作業を最小化します;）



しかし、今、新しいメッセージを作成する必要があり、動きは標準です。プログラムを開き、ボタンを押してメッセージを作成しました。 そして、1日50回。 20回目は、クライアントを前後に動かしてWindows 7のアニメーションを楽しむことはまったく面白くありません。ボタンを押して、クライアントを開かずにメッセージを作成するウィンドウを取得したいと思います。



その結果、ソリューションは非常にシンプルでしたが、非常に便利です。







これは、任意のフォルダーに作成されたショートカットです。 ショートカットパスでは、よく知られているmailto：（ ）をすべての人（またはほぼ全員）に痛々しいほど書き込みます。デスクトップツールバーの新しいパネルとしてフォルダを追加します。 パネルでショートカットのタイトルと署名を非表示にします。 スクリーンショットにあるものを取得します。



少し考えて、このショートカットのプロパティを呼び出し、自分に合った組み合わせのクイックコールを登録しました。







新しい電子メールメッセージだけでなく、タスク、カレンダー、連絡先、メモなどのエントリを作成する必要があると少し考えて、私は特に緊張しなければなりませんでした。 出口はすぐに来ました。 初めてF1が役立ちました:)



実際には、次のように書かれています：



/c ipm.activity — ;

/c ipm.appointment — ;

/c ipm.contact — ;

/c ipm.note — ;

/c ipm.stickynote — ;

/c ipm.task — .







それを実際に実行してください。 再び新しいショートカットを作成します。たとえば、パスに次のように指定します。



"C：\ Program Files \ Microsoft Office \ Office12 \ OUTLOOK.EXE" / c ipm.appointment

この場合、Office12は、2007バージョンのオフィスがインストールされていることを示します。 2003年の場合、Office11があります




誰かが役に立つといいな。 個人的に、私の仕事では、多くの人が新しいメッセージを作成するこのような簡単な方法を疑っていませんでした
