JBossは、JBoss ToolsおよびJBoss Developer Studioの次のバージョンを導入しました

マイケルソロキンの依頼で。



Red Hatの一部門であるJBossは、J2EEアプリケーションの開発に焦点を当てた2つの製品のリリースを発表しました。商用のJBoss Developer StudioとオープンソースのJBoss Toolsです。





製品の主な利点は次のとおりです。



VPE （ビジュアルページエディター）-ビジュアルエディター。

フローエディタ -JSFおよびStrutsのフローエディタ。

ORMマッピングビューアー -ビジュアルマップマッピングエディター。

JBossツールパレット -タグライブラリを含む視覚コンポーネント。

JBoss Application Platform -JBoss Application Sever、Hibernate、およびSeamが含まれます。

jBPM （ビジネスプロセスモデリング）-ビジネスプロセスをモデリングするためのプラットフォーム。



JBossには、J2EEアプリケーションを開発するための完全な製品ラインがあります。 JBoss ToolsおよびJBoss Developer Studioは、JBoss AS、Seam、jBPM、Hibernate、DroolsなどのJBossテクノロジーで動作するように主に作成されています。



どちらの製品もEclipse IDEとEclipse WTPプロジェクトに基づいています。 これにより、Sun、Weblogicなど、J2EE技術スタックの他の実装者が提供するソリューションに基づいてJ2EEアプリケーションを開発できます。



JBoss Toolsは、無料でダウンロードできるEclipseプラグインのセットです。 また、JBoss Developer Studioは、J2EEアプリケーションの開発と起動に必要なソフトウェアの完全なセットです。 JBoss Developer Studioは、JBoss AS、JBoss Tools、Eclipse IDE、WTPを含むインストーラーとして提供され、インストール段階でこれらのコンポーネントを設定できます。



JBossは、オープンソース市場の主要なプレーヤーの1つです。 JBoss Developer StudioおよびJBoss Toolsの各ユーザーは、バグ通知を投稿することで製品開発に貢献したり、 Jiraプロジェクト管理システムで必要な機能に関するリクエストを書いたりすることができます。



どちらの製品にも、完全な技術サポートと詳細な更新ドキュメントがあります。
