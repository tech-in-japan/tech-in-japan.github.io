WindowsストアでのWindows 8用の最初のロシア語アプリケーション





ご存知かもしれませんが（おそらく既に知っているように）、先週Windows 8 Release Previewがリリースされ 、それに加えてWindowsストアアプリストアが更新されました。



この記事では、ロシアからの最初のアプリケーションに注目したいと思います。







まず第一に、これらはコンシューマープレビュー以降に利用可能な2つの人気のあるアプリケーションです。これらは更新され、RPで利用可能になりました。



ロープを切る





ZeptoLabのこの人気のあるゲームには特別な紹介は必要ないと思いますが、これはWindowsストアで最も人気のあるアプリケーションの1つであることに注意してください。



News360





News360は、世界中の関心のあるトピックに関するニュースを選択およびフィルタリングするための人気のあるサービスです。 ちなみに、News360はBizSpark Oneプログラムのメンバーです。



===

Windows 8 Release Previewで利用可能な新しいアプリケーション：



ギスメテオ





人気のある同名の天気予報サービスのクライアントはGismeteo.ruです。 これで、スタート画面の窓の外の天気が変わりました！



イビ





あなたが映画やシリーズを見たいなら、このアプリケーションはまさにあなたが必要とするものです。 これは、ライセンスコンテンツを含む同名のビデオサービスの公式クライアント-ivi.ruです。 あなたのソファで最大のオンライン映画館:)



映画検索





映画を見に行くことを考えている場合、またはこの映画やその映画の詳細やレビューを知りたい場合は、 アクティスワンダーマンが開発したkinopoisk.ru Webサイトの公式クライアントを必ず登録する必要があります。 あなたの指のすぐ下にあるすべての発表とレビュー。



にっぽん





MyEnglishは英語を学び、知識をテストするのに最適なアプリです。 アプリケーションで1日10〜15分を費やしてください。すぐに英語のレベルが著しく向上します。



落書きの神





自分を感じてください...えーと... Doodle-god :) JoyBitsの大ヒット作がWindows 8にも登場しました！ 異なる要素を渡って新しい要素を取得します。



モンテスマの宝3





Alawarの人気ゲームがWindows 8に登場しました。カラフルなデザインと指のサポートを備えたこのジャンルの古典です。



私もWindowsストアで欲しい





Windows 8用のほぼ完成した、または既に完成したMetroスタイルアプリケーション（またはゲーム）をお持ちの場合は、 研究室の仕事のために私たちに来てください。
