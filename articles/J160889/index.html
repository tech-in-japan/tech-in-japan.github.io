手頃な価格のウルトラブックASUS S400CA





Windows 8の新製品の数は、すべての記録を破りました。 Windowsのタブレット、Windowsの新しいウルトラブック、Windows 8のこれまでにないハイブリッド...このような多様性により、美しく高価なエキゾチックなデバイスを見て、実用的で手頃なデバイスを個人的に選択するのが大好きであることを忘れがちです。 最近まで、予算のラップトップはスタイリッシュなデザインや豊富な機能を誇っていなかったため、顧客は難しい選択を余儀なくされていました。つまり、高品質のデバイスを過払いして手に入れるか、節約して耐えるかです。



一見、新しいASUS VivoBookシリーズのラップトップは、手頃な価格の「ハードワーカー」です。 特徴を注意深く研究し、以下を見つけます：薄いボディ、小さな寸法、軽量、容量の大きいバッテリー-これらはウルトラブックの典型的な特徴です。 しかし、ASUSは比fig的に言えば、大衆市場を再編成することを決定しました。 人気のあるタッチデバイスの作成に豊富な経験を持っているため、同社はすべてのシリーズのASUSラップトップに実質的にタッチスクリーンを装備することを決めました。 ASUS S400シリーズは、新しいトレンドの最初の兆候です。 Windows 8のリリースからの感覚のプリズムを通して見ると、非常に魅力的なソリューションです。



本日、ASUS VivoBook S400CAのレビューで、タッチスクリーンを使用して、新しいWindows 8オペレーティングシステムのすべての機能を明らかにしました。





外観







安価なラップトップでは、めったに金属元素を使用しません。 ASUS S400CAでは、光沢のあるプラスチックが見えると期待していましたが、ラップトップはアルミニウムとマットプラスチックの組み合わせに非常に驚きました。 このラップトップでは、フレームは高品質のプラスチックでできており、ヒンジ付きパネルは金属でできています。







ユーザーはケースの厚さにますます注意を払っています。 光学ドライブがないため、厚さを減らすだけでなく、重量も減らすことができました。







ディスプレイカバーはダークメタル製です。 手触りは快適ですが、表面は指紋を簡単に収集します。







ラップトップの優雅さは、側面をシャープにするデザイナーを追加します。 実際、ラップトップはそれほど薄くありません（厚さ2 cm強）が、見た目は素晴らしく見えます。







ASUS UX31Aのレビューで、エッジの鋭さについて不満を言いました。 S400では、角が丸くなっています。



スクリーンマウントは震えず、ヒンジはきつくて信頼性が高く、蓋はほとんど曲がりません-ビルドの品質は非常に高いレベルです。 そして、認識可能なデザインを考慮して、ASUS S400はZENBOOKプレミアムウルトラブックを思い出させます。







ラップトップの重量は記録的なものではありません（1.8 kg）が、電源アダプターははるかに「薄い」。







バッテリーが目立ちません。 底部カバー全体がワンピースのプラスチックプレートです。 ラップトップの「内側」を楽しむには、10本のネジを外す必要があります。 ハードドライブとRAMを交換できます。



キーボードとタッチパッド







キーボードに追加のテンキーはありませんが、アイランドタイプのキーは通常のサイズです。 特に、電源キーの見慣れた場所と、トリミングされていない最適なカーソル矢印に注意してください。 キーには通常のストロークの深さ、十分な反動があります。 使用は便利です。 バックライトなし。



金属基板は十分な剛性を提供します。 キーボードは曲がりません。 手の作業エリアにより、印刷中に快適に対応できます。







大きなタッチパッドは、苦情を引き起こしません。 マルチタッチジェスチャの実行、さまざまな強さのポークおよびタップに対する感度が最適に実装されます。 しかし、タッチスクリーンにすぐに慣れて、私は実際にタッチパッドを使いませんでした。



コネクタと通信







ラップトップには適切なポートセットが装備されています。左側には、2つのUSB、ヘッドフォンとマイクの組み合わせジャック、およびカードリーダーがあります。







右側には、USB 3.0、HDMIおよびVGA出力、イーサネットがあります。 ポートは、互いに十分な距離にあります。







フロントエッジにコネクタはありません。







ただし、LEDインジケータは多数あります。



ワイヤレス接続の場合、ラップトップには802.11a / b / g / n Wi-FiアダプターとBluetooth 4.0が装備されています。 ビデオ会議では、ディスプレイのすぐ上に1.3メガピクセルのWebカメラがあります。



ディスプレイ







対角14インチ、1366x768ピクセルの控えめな解像度の画面には、タッチと光沢のある2つの機能があります。







最大輝度レベルは満足のいくものではありませんでした-人工照明のある部屋にのみ十分です。 そして晴れた晴れた日には、そのようなディスプレイで路上で行うことは何もありません。 目の前に鏡だけが表示されます。







ディスプレイは、グラフィックスを使用した専門的な作業を目的とするものではありませんが、通常の「オフィス」機能で問題なく実行できます。 あなたは映画を見ることができますが、屋内でのみです。 ディスプレイは大きな視野角を誇ることはできませんが、Windows 8を非常に効果的に制御できます。タッチパッドと同様に、ジェスチャーによるシンプルで直感的な制御がサポートされ、複数のクリックが認識されます。



画面の角度は小さく、悲しくなりました。 タッチディスプレイとのやり取りがより便利になるように、ラップトップを完全に拡張したかったのです。



Windows 8およびタッチスクリーンASUS S400



Windows 8は好きなように扱うことができますが、インターフェイスでどの程度の作業が行われたかをすぐに確認できます。 このオペレーティングシステムは、確かにタッチスクリーンに焦点を当てています。 もちろん、通常の「ウィンドウ」（最愛の[スタート]ボタンを除く）に戻る機会はありますが、新しいMicrosoftオペレーティングシステムの主な機能はMetroインターフェイスです。 タブレット（およびトランスフォーマー）の場合、ソリューションは正当化されているように見えますが、ラップトップがあります。



それを理解してみましょう。 Microsoftの新しいオペレーティングシステムには、まったく新しいアプローチの作業が含まれています。 ASUS S400からタッチスクリーンを取り外しても、平均的な仕様と実質的に役に立たないWindowsを備えた手頃な価格のラップトップを使用できます。 マウスまたはタッチパッドを使用してMetroを制御できますが、人間工学は不十分です。 同時に、指での作業は非常に便利です。



ラップトップの重量は、ディスプレイを操作する際の安定性を提供します。 好きなだけ画面上で指を突く-ブナは1ミリも動かず、テーブルから外れません。 手には疲れを感じるほどのキャノピーはありません。 ラップトップの電源を入れ、画面をスクロールし、アプリケーションを起動して通常どおりに動作します。 私の主観的な感覚によると、「有効化-実行-作業」のカテゴリの多くのタスクは、センサーを介してより速く、より便利に実行します。



従来、ASUSは、オプション（Trend Micro Internet Security）と必要（Microsoft Office 2010 Starter Edition）の両方の追加ソフトウェアをインストールすることにより、ユーザーに数ドルの節約を提供します。 今回のASUS WebStorageでは、ユーザーはファイルを保存するための無料の場所を与えられます。



性能









ASUS VivoBook S400の中核をなすのは、クロック周波数1.7 GHz（TurboBoostで最大2.6 GHz）の強力なデュアルコアIntelCore i5-3317U *プロセッサーと統合グラフィックスIntel HD 4000です。

*スクリーンショットは3427U-先行販売サンプルの機能を示しています。



プロセッサは、低消費電力にもかかわらず、あらゆるタスクに十分なパフォーマンスを提供し、少なくとも来年には関連するでしょう。







このサンプルでは、​​6 GBのDDR3 RAMがありますが、販売中には2/4 GBバージョンが見込まれます。 興味深いソリューションは次のとおりです。マザーボードにはんだ付けされた内蔵メモリは2/4 GBですが、最大8 GBまで拡張するための追加のスロットがあります。 これは非常に優れています。なぜなら、より高価なウルトラブックでも、4 GBは非常に魅力的な天井だったからです。







プロセッサに統合されたHD Graphics 4000はDirectX 11をサポートします。統合ソリューションの場合、HD 4000は優れたパフォーマンスを提供します。 ただし、AMD / ATIおよびNvidiaの個別ソリューションとゲームのパフォーマンスを比較することはできません。 最新のゲームを実行できますが、最小限の設定で実行できます。









ハードディスクとして、320または500 GBの容量の標準2.5インチオプションが提供されます。 同時に、毎分5400回転の修正を使用して、十分に低いレベルのエネルギー消費を保証します。







メインハードドライブの速度は印象的ではありません。 ただし、ASUS S400にはSSDドライブに基づく24ギガバイトキャッシュが搭載されているため、システムは迅速かつスムーズに動作します。







Windows 8インデックスは、統合グラフィックスとハードドライブ（SSDを除く）の高い評価を期待していませんでした。



熱性能







ASUS S400は最適な温度範囲を実現しました。 ラップトップは膝の上に置くことができ、ディスプレイを操作するときに便利です-単に入浴するものはありません。 これは日常の使用だけではありません-ストレステスト中に深刻な熱はありませんでした。 さらに、ASUS S400は実質的にノイズがありません。







エアブローは底部ではなくディスプレイ上で行われます。



オフライン作業



100％のプロセッサ使用率、最大画面輝度、アクティブなワイヤレスインターフェイスにより、ウルトラブックは約2時間持続しました。 同時に、Wi-Fi経由のインターネットブラウジング時間は4.5時間でした。 悪くない-そのような指標は平均と呼ぶことができます。



音



ASUS S400CAからは、透明な音は必要ありませんでした。 私が思ったように、最高のサウンドを得るために、間違ったアドレスに移動します。マルチメディアノートブックモデルのサウンドは常に優れているためです。 この場合、ASUS S400MasterはASUS S400CAに満足しています。 最大音量のスピーカーは喘鳴しません。 音量は大きくなる可能性がありますが、音は明瞭です。 ラップトップはマルチメディアソリューションとして位置付けられていないため、さらに奇妙です。 外出先で映画を簡単に視聴したり、ヘッドフォンを使用せずに音楽を聴いたりできます。



おわりに



価格の下落を考えると、タッチスクリーンのラップトップはすぐに標準になります。 ただし、タッチスクリーンのみの使用を強制する人はいません。これは代替手段であり、タッチパッド、マウス、キーボードの代わりではありません。



ASUS S400は主にWindows 8で使用するために作成されたもので、最適化された直感的なコントロールにより、その目的に完全に対応しています。



興味深い簡潔なデザイン、高品質のアセンブリ、信頼性の高い素材、小さなサイズ、タッチディスプレイにより、このラップトップはWindows 8を実行する最も魅力的なソリューションの1つになります。しかし、ラップトップの主な利点はその主な欠点です。ディスプレイの品質が製品の使用範囲を制限します。 それにもかかわらず、私はASUS S400を自宅とオフィスの両方の仕事に推薦できます。
