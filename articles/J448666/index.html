顧客との電子文書管理サービスを選択した方法

研究を共有するハブロフスク。 3月、彼らは最高の電子文書管理オペレーターを探していました。 まあ、最高として。 私たちは、サービスが当社により適しているものを選択しました。 1週間で、最も有名なもののうち7つを研究しなければなりませんでした-1Cとの統合オプションから技術サポートの品質まで、パラメーターで比較しました。 しかし、まず最初に...











それがすべて始まった方法



法律に関する問題を回避するために、法的に重要な電子文書管理にこのサービスを使用することにしました。 トピックに初めて飛び込んだとき、30以上のオプションから選択する必要があることがわかりました。 いずれにせよ、インターネット上で多くが見つかりました。 私はすべてを詳細に扱いたくなかったし、十分な時間もなかった。 したがって、請負業者から何を使用するかを学びました。 これにより、応募者の数を最も人気のある7人に絞り込むことができました。







そこで、サービスを調べました。

1.最低料金



サービスのコストとともにサービスに関するデータの収集を開始しました。 そして、ここで2つのニュースがあります。 良い-すべてのオペレーターからのすべての着信トラフィックは完全に無料です。発信文書に対してのみお金を支払う必要があります。 悪い点は、サイトに表示されている価格を把握するのが非常に難しいことです。 それはすべて、すべてのサービスに対する関税と支払いの原則がさまざまな方法で構築されているためです。







私は、最小限の関税のみが考慮されたことにすぐに注意します。 請負業者はそれほど多くなく、ワークフローの量もそれほど多くありません。最初はあまりお金をかけたくありませんでした。







サーキットディアドック



900ルーブルの最低関税。 100のドキュメントが含まれます。 電子文書管理オペレーターがSKB Konturである相手とのみ文書を交換できます。 同時に、クライアントは1年に1回しか「最小」料金プランを購入できません。







Taxcom / 1C：EDO



オペレータサービスの支払いに依存する2つの作業オプションがあります。 1C-EDOソリューションを使用する場合、価格はお住まいの都市の1Cフランチャイジーによって設定されます。 私たちは一週間でやり遂げませんでした。 2番目のオプションは、1Cを参照せずに直接作業することです。 この場合、私たちが理解したように、1年間の最小パッケージには1800ルーブルがかかります。 150件の送信メッセージを含みます（それぞれ-1つの請求書を含むドキュメントのパッケージ）。







VLSI



最小パッケージは500ルーブルです。 1年あたり、制限は四半期あたり50パッケージです。 パッケージには、あらゆる種類のドキュメントと数量を含めることができますが、それぞれに請求書を1つだけ含める必要があります。 彼らはサービスに接続するための支払いを取ります-500ルーブル。







Synerdocs



レートは2050ルーブルから始まります。 300ドキュメント。 それらは1年間計算されます。







Kaluga.Online / 1C：EDO



最低関税は1200ルーブルになります。 1年に300件の送信メッセージに対して。 上記のすべては10ルーブルです。 PCごと 1つのメッセージ（パッケージ、キット）には、1つの請求書と2つの関連文書を含めることができます。







スフィアクーリエ



サービスに接続するために、小売店でない場合は、300ルーブル（250 + VAT）から別途支払う必要があります。 300ルーブルの最低料金で。 （250 + VAT）発信50を受け取ります。 関税を超える書類はより高く支払われます-それぞれ7ルーブル。 PCごと 関税は1か月間有効です。







E-com



最低料金は4000ルーブルです。 1か月あたり500件の送信ドキュメントが含まれています。

6.ローミングでの交換



最後に、最も興味深いものについて説明します。 モバイル通信のように、電子文書管理では、ローミング中の加入者の相互作用に問題があります。 サービスエリア外のサービスだけではありません。 ドキュメント管理の場合、ローミングには、異なるオペレーターの顧客間でドキュメントを交換する可能性が含まれます。







繰り返しますが、ロシアには30を超えるサービスがありますが、誰もが互いにローミング接続しているわけではありません。 EDIの運用者が誰と既に持っているかは、 こちらで確認できます 。







一方で、ほとんどすべてのオペレーターはローミングセンターに接続しています。これは、異なるサービスの顧客間で請求書の安全な交換を提供するプラットフォームです。 一方、ローミングを構成するオペレーター自身の準備は、すべての人で異なります。 パートナーの経験から判断すると、Kontur。DiadocおよびTaxcom / 1C：EDOの場合、プロセスは遅れています-VLSIおよびSynerdocs。 他のオペレーターのレビューはありません。
