Windows Mobileスマートフォン用のTwitterクライアントの概要

スマートフォンをお持ちの場合、Twitter用の特別なソフトウェアを探していないのは罪です。 WMでこの方向にすべてがクラウドレスであるとは限らないことが判明し、私は過去数週間、さまざまなtwitterクライアントを使用して過ごしました。 この記事では-私の個人的な印象と結論。

クライアントは、HTC S710（Vox）スマートフォンでテストされ、Windows Mobile 6.1 Smartphone Editionにフラッシュされました。 スマートフォンの特徴は次のとおりです。TexasInstruments OMAP 850 200 MHz、64 MB RAM、128 MB ROM（SDRAM）、TFT 320 x 240。

特別なテスト技術はありませんでした。 クライアントをセットアップし、それを使って些細なことをしようとしました。フィードの読み取り、ツイートの送信、失敗したツイートの削除、フォロー、フォロー解除です。 結局のところ、すべてのクライアントが上記のすべてを実行できるわけではないため、それらをより密接に検討してください。





クァック

www.anthonyburns.co.uk/quakk

さまざまな色（エイリアンは灰色、返信は緑色、彼らは青色）でツイートを読む非常に優れたクライアントは、フィードをすばやく更新します。



しかし、ツイートの送信に問題がありました。 Quakkは、ActiveSyncとGPRSを介してコンピューターに接続した場合にのみツイートを送信しました。ありがとうございます。Twitterへの接続は切断されませんでした。 しかし、近くにインターネットを備えたコンピューターがある場合、モバイルクライアントが必要なのはなぜですか？ :)

さらに、Quakkの設定はわずかであり、残りの機能はほとんどありません。ツイートへの応答方法がわからない、プロファイルの監視方法がわからない、ほとんど何もわからない。 しかし、それは美しいようです、はい...

Quakkのすべての機能は次のとおりです。





小さなTwitter

www.tinytwitter.com

TinyTwitterにはいくつかのバージョンがあります：WM（PPCおよびSmartphone）のほか、2つのJavaバージョン（MIDP 1.0およびMIDP 2.0）があります。つまり、理論的には、すべてのWMデバイスおよびJavaサポートのあるデバイスで使用できます。

クライアント設定は非常に広範囲であり、プロキシを使用できます。



インターフェイスはよく考えられていません-各ツイートは少なくとも画面の半分を占め、大きなツイートは画面全体を占めます。 ただし、ここで読むことをお勧めします-過去100時間にアップロードされた高速でバグのあるツイートです。



他のツイッタープレイヤーのタイムラインを読むことができ、フォローする方法を知っています。

しかし、軟膏には大きなハエがあります：TinyTwitterはキリル文字を送信することに致命的に対処しません。 ツイートを送信すると、ストリームでは完全に表示されません。



ツイートの最後にある3つのドットをクリックすると、正しい完全なツイートが開きます。



また、ツイートは削除されません。削除されたと表示されますが、実際にはツイートはそのまま残ります。

ceTwit

www.kosertech.com/blog/?cat=3

インターフェースはそれほど熱くはありませんが、過負荷ではなく、一度に多くのツイートが画面に収まります。 一方、長いツイートは少しトリミングされています。 ただし、ツイートでジョイスティックをクリックすると、ジョイスティックがフルモードで開き、同じモードでテープ全体を読み取ることができます。



色分解もあります。

フリンジ

www.fring.com

それはすぐに私にはひどく不快に思えました。 また、Twitterには独自の問題があります。 まあ、まず、fringは最近、Twitterエクスペリエンスを改善するために設計されたtwitter 2.0アドオンの追加を発表しました。 長い間、このアドオンは接続を拒否し、今日、ようやく機能しました。

悪くはないように見えますが、Fringサービスの広告の一番上の行は絶対に必要ありませんが、削除する方法が見つかりませんでした：



あまりフリンジはできません：ツイートの送信、プロフィールの読み取り、ツイートへの返信。 ただし、ボタンを押すたびに、soooooooooが長く反応するため、実際にはすべてのパフォーマンスが無効になります。

彼はキリル文字でツイートを投稿していますが、優れています。 そして、テープを更新するとき、それは美しい鳥のさえずりを公開します:)

まあ、実際にフリングに対処するために、古いtwitterアドオンをテストすることにしました。 ツイート専用です。 そして、これに非常によく対処します。 ストリーム内の以前のツイートもいくつか表示されますが、テープをスクロールダウンできませんでした。 チャットのように見えます。 新しいツイートは、投稿後すぐに表示されます。



一般的に、フリングはもちろん、主にIMクライアントであり、同様の原則に従ってTwitterでも動作します。 他のデバイスでより高速に動作することを除外しませんが、動作することは完全に不適切です。
