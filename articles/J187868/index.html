Windows 7のIE11プレビュー、modern.ieの更新、Parallels for Macの25％割引

本日、Windows 7用のInternet Explorer 11の予備バージョンがリリースされましたが、以前はWindows 8.1ユーザーのみが利用できました。 新しいブラウザは、高性能、ページ読み込み速度、および新世代のWebサイトの作業に必要な最新の標準のサポートを提供します。 さらに、開発者は完全に再設計されたF12ツールを使用できます。







IE11 Developer Previewを今すぐダウンロードしてお試しください 。



IE11を使用すると、開発者はWebGL上でWebGLを含むプロフェッショナル品質のビデオ、ハードウェアアクセラレーション2Dおよび3D Webテクノロジーを使用できます。



IE7のすべての利点は、今日から7億人以上のWindows 7ユーザーが利用でき、さらに、無料の仮想マシンとすべてのプラットフォームでのオンラインサイトテストサービスを備えたwww.modern.ieの開発者向けの更新ツールも提供します。







さらに高速でスムーズなインターネット



速度は、ブラウザの最も重要な品質の1つです。 IE11コードは、サイトができるだけ早く表示されるように最適化されており、Webサービスはコンピューターにインストールされたアプリケーションの速度で動作します。



IE11は、コンピューターのGPUを使用してリアルタイムのJPG画像をデコードする世界初のブラウザーです。 これにより、ページの読み込みが速くなり、RAMの使用量が少なくなります。これにより、モバイルデバイスの消費電力を削減し、バッテリ寿命を延ばすことができます。 IE11は、グラフィックプロセッサを使用してテキストをレンダリングする世界初のブラウザでもあります。 テキストと写真はインターネットの基盤です。つまり、これらの2つの技術は、アクセスするほぼすべてのページをより速く表示するのに役立ちます。



ページをより速くロードするには、ブラウザがサイト要素のロード順序を最適に選択できる必要があります。 IE11は、 W3C Resource Priorities標準を使用する最初のブラウザーです。 これは、開発者がどのページ要素を最初にロードするかを指定できることを意味します。 また、新しいHTTP 2.0仕様の基盤であるSPDYプロトコルのサポートにより、一部の既存サイトの読み込みが高速化されます。







www.ietestdrive.comの新しいデモを使用して、IE11の速度を自分で評価できます。 レビテーションとローンマークに注意してください。2Dおよび3Dグラフィックスの作業速度を評価できます。 マークアップ表示は、HTML5、CSS、およびJavaScriptの使用を組み合わせたLiteBriteを使用して評価できます。



JavaScriptの高性能



JavaScript「Chakra」エンジンは大幅に改善されました。 以前のバージョンとの互換性と高レベルのセキュリティを維持しながら、さらに高速になりました。 IE11は、最新のECMAScript 6標準で説明されている機能、let、const、Map、Set、WeakMap、および互換性のための__proto__もサポートしています。



JITコンパイラーは、プロパティの多態的なキャッシングや関数のインライン化など、新しい最適化方法をサポートするようになりました。つまり、より多くのコードがJITを通過し、計算に費やす時間が少なくなります。 Internet Explorer 11で作業するエンジニアは、既存のサイトを操作するシナリオに焦点を合わせています。 ただし、WebKit SunSpiderなどのJavaScriptテストのIE11は、競合他社を大きく引き離しています。







さらに、IE11はECMAScript国際化API （バージョン1.0）を使用します。



3Dハードウェアアクセラレーション



Internet Explorer 11は、Web開発者がコンピューターのGPUを使用して表示される複雑な2Dおよび3Dグラフィックを使用してサイトを作成できる新しいテクノロジーであるWebGLをサポートします。 実験にのみ使用されていたWebGLテクノロジーは重要な開発ツールになりつつあり、まったく新しいタイプのインターフェースを作成できます。



たとえば、最近GlacierWorksと提携して、気候変動がヒマラヤの氷河に及ぼす影響と、氷河の融解が生態系に及ぼす影響について説明するインタラクティブなサイトが作成されました。 本日、 WebGLを使用してヒマラヤのインタラクティブな3Dマップを作成することにより、このサイトを更新しました 。 これは、3Dインターフェイスがサイトに既に存在する写真やビデオコンテンツとどのようにやり取りできるかを示す好例です。 このサイトはどのデバイスでも動作します：たとえば、Windows 8のタブレットからwww.glacierworks.orgにアクセスした場合、IEのもう1つの優れた機能であるCSSデバイスの方向性、タブレットを手に持って山岳地帯をナビゲートする機能を評価できます。







IE11は、Windows 7または8を搭載したデバイスでWebGLを提供し、ブラウザーのセキュリティや信頼性を損なうことなくGPUのリソースを使用できます。



プロ品質のビデオ



Internet Explorer 11では、追加のプラグインなしでHTML5ビデオを使用できます。 IE11はクローズドキャプション標準（ TTMLおよびSDP ）をサポートしているため、サイトに投稿するビデオの品質は最高になります。 IE11はHTML Full Screen APIもサポートしているため、視聴中にHTML5ビデオを全画面に簡単に展開できます。 IE11は、ファイルをハードドライブにキャッシュせずにビデオをダウンロードできるため、デバイスのバッテリー寿命を延ばすのに役立ちます。



新しいF12開発者ツール



IE11には、サイトの最適化に役立つ完全に更新されたF12開発者ツールが含まれています。 IEチームのエンジニアは、Visual Studio開発者と一緒に作業しました。 開発者は新しいF12ツールを使用して、パフォーマンスの最適化、エラーの診断、IEによるサイトの表示方法に関する完全な情報を得るために必要なデータをすばやく取得できます。



新しいF12ツールは次のとおりです。

  • UIデバッグおよびメモリ使用量診断ツール
  • ライブDOMエクスプローラーとCSS検査。ページと同時に情報を更新し、動的コンテンツがレイアウトとスタイルに与える影響を評価する機会を提供します
  • JavaScriptデバッグ-ページの更新を必要としないツール








F12ツールが通常のVisual Studioツールに非常に似ていることも重要です。 F12アドベンチャーオンラインレビューで新しいF12ツールをお試しください。



すでに機能しているものはすべて機能し続けます



Internet Explorer 11は既存のサイトと完全に互換性があります。 開発者がサイトをさらに改善できるAPIが登場しました。 一部のAPIも変更および削除され、現在動作しているサイトがブラウザの新しいバージョンでエラーなしで引き続き表示されるようになりました。 IE11は、改善されたCanvas 2D（スムージング、塗りつぶし、破線）、CSS境界画像、デバイス固定ポジショニング、ポインターイベントとCSS Flexbox、ミューテーションオブザーバーなど、 より多くのHTML5およびCSS機能をサポートします 。



IE11には他にも多くの変更が含まれています。 たとえば、組み込みのテキストエディターが大幅に改善されました。 開発者は、クリップボードから画像を貼り付けたり、アクションを元に戻したりできるため、テキストエディターを使用するアプリケーションを簡単に作成できます。

IE11の新機能の完全な概要については、 こちらをご覧ください 。 そしてIEチームはConnectであなたの提案や希望を待っています。



ただし、IE11では大幅に更新されたユーザーエージェント文字列を忘れないでください。



Modern.ieを更新



Internet Explorer 11 Developer Preview for Windows 7のリリースに伴い、サイトwww.modern.ieを更新しました。これは、IEを含むWebサイト開発を容易にするためのツールと資料のセットです。 私たちはインターネットの進化を望み、開発者が新しいものを見つけるためにより多くの時間を費やし、テストに多くの時間を費やさないように、できる限りのことをしています。



したがって、modern.IEの今日から：

  1. 限定オファー：Parallels for Macが25％オフ。
  2. Windows 7およびWindows 8.1用のIE11をテストするための新しい仮想マシン
  3. ほとんどのデバイスとブラウザでサイトのスクリーンショットを無料で取得できる新しい無料ツール




さらに、modern.ie Webサイトスキャナーがオープンソースとして利用可能になりました（Apache 2.0でライセンスされています）。 独自のプロジェクト用にGitHubからダウンロードできます。



限定オファー：Parallels for Macが25％オフ。



Parallelsの友人は、iOS開発者向けのWindowsクイックスタートキットを 4月に提供してくれました。あなたの反応を喜んでいます。すべてが25分で完売しました。 本日、25％の割引で、Parallels Desktop 8 for Macの限定購入の提供を発表しました。 2013年8月19日までに、modern.IEの2500セットの特別なコードのいずれかを入手してください。



仮想マシンでのIE11プレビュー



仮想マシンは、サイトをテストするために不可欠なツールであり、作業環境に依存しないことがわかっています。 modern.IEのWebサイトには、Windows XPのIE6からWindows 8.1のIE11まで、可能なすべてのバージョンの仮想マシンが投稿されています。

  • Windows Server 2008 R2 SP1上のHyper-V
  • Windows Server 2012およびWindows 8 Pro w / Hyper-V上のHyper-V
  • Windows 7上のVirtual PC（VPCは仮想Windows 8を実行できないことに注意してください）
  • Windows、Mac OSX、Linux上のVirtualBox
  • WindowsおよびMac OSX上のVMWareプレーヤー




ここから仮想マシンをダウンロードしてください 。
