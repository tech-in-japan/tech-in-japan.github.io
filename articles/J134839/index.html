Amazon Kindle FireにTWRP 2.0をインストールする

TWRPは、ブートローダー回復モード（回復モード）です。 このモードは、デバイスのバックアップまたはバックアップからの復元に使用されます。



あなた自身の危険とリスクで実行するすべての操作！ 著者は、この手順によって損傷した機器には責任を負いません！



ソースデータ：

-デバイスは完全に充電され、ルート化されている必要があります

-インストールされたAndroid SDK

-fastbootプログラム（ fastboot.exe ）

-adbプログラムを使用するためのスキル



TWRPブートローダーtwrp -blaze-2.0.0RC0.imgを、adbユーティリティと同じフォルダー（Android SDKのplatform-toolsフォルダー内）にダウンロードします。



デバイスをコンピューターに接続します。



コマンドプロンプトで、次のコマンドを入力します。 



adb shell idme bootmode 4002 exit fastboot -i 0x1949 boot twrp-blaze-2.0.0RC0.img


コンピューターの画面に、 <デバイスを待機中>が表示されます 。 Fire自体が再起動しない場合は、電源ボタンを長押ししてオフにし、再びオンにします。 Fire画面にTWRPスプラッシュ画面が表示され、インストール手順が表示されます。 Amazon FireロゴでFireが停止した場合、fastbootを停止し、コマンドfastboot -i 0x1949 boot twrp-blaze-2.0.0RC0.imgを再度指定できます。



TWRPをインストールした後、次のコマンドを入力します。 



 adb shell idme bootmode 4000 reboot


デバイスが再起動し、Amazon Fireの代わりに黄色の三角形のリカバリブートが表示されます。 リカバリモードで起動するには、電源ボタンを押し続けます。 通常のダウンロードの場合は、お待ちください。



ロード後、コンピューターはデバイスが不明であると言う場合があります。 次に、このandroid_winusb.infドライバーをダウンロードし、 \ Android \ extras \ google \ usb_driver \フォルダーに配置し、このフォルダーを指定してFireドライバーを再インストールします。
