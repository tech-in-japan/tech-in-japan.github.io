3.ベースプレーヤーのせいにする



ベース奏者もいないので、 SF2形式のNaturally Decaying Bass Guitarsを使用しました。

SF2 Playerは、ツールとしてLMMSの一部です。 SF2 Playerのプロパティで、ベースファイルを指定し、 PatchプロパティでNaturally Finger Bassを選択する必要があります。 LMMSでは、 JACKサーバーのサポートが十分に実装されていないため、 Ardorでは低音部が標準としてウェーブのエクスポート/インポートによって追加されました。 効果はArdourでのみ処理されました。



4.ミックスについて。



バンドのギタリストはミキシングに関与していました。 彼はHabréでアカウントを作成する予定はありません。彼がシングルの彼の作品の要点を私に描いたからです。

当初は、 Ubuntu Stidio 12.04（LTS）を使用する予定でしたが、これはインストールしてすぐに使用でき、わずかな問題しかありませんでした。 ただし、ここではボーカルに外部者が必要でした（ボーカルに倍音を追加してサウンドを改善できます）が、このプラグインは現在のバージョンのCalfにのみ存在します。 依存関係のため、12.04では配信されませんでした。 重い心で13.10を置くことにしました。 経験は欺かれません。 Ardour 、 JACK 、その他のソフトウェアの暖かいランプフォールに戻りましたが、 Ardor 3と必要な害虫駆除機を入手しました。



4.1。 JACKサーバーのセットアップ


JACK Audio Connection Kit（JACK）は、このテクノロジをサポートするオーディオプログラムを低遅延で接続できるサウンドデーモンサーバーです。 Ardourが動作するために必要です。

Ubuntu Studioには、 qjackctlグラフィカルユーティリティが付属しています。 ここで 、セットアッププロセス全体について説明します 。

qjackctlを実行した後、オプションを選択し、アイテム「Real-time」（ Realtime ）の前にあるボックス（チェックされていない場合）をチェックする必要があります。 強力なコンピューターが使用されると、サンプリング周波数は96000、バッファーあたりの周期3になりました。「インターフェイス」パラメーターは、オーディオデバイスの選択を担当します。 サウンド（ オーディオ ）-デュプレックス。

バッファ内のフレーム/期間の選択に問題がありました。 1024を超えると、 Ardourはエクスポートを拒否したため、1024が残りました。 より小さな値を使用できます。



4.2。 アーダー3.5.74


Ubuntu 12.04とは異なり、 Ardor 3は Ubuntu 13.10のリポジトリに存在します。 ギタリストは、MIDIキーボードのフェーダーやその他のツイストをArdourのトラック設定と同期することにしました。 Mackieインターフェースが正しく動作しないことがわかり（ マスターボリュームのみが移動）、ソースからArdourを再構築しました。 正しく動作させるには、ソース（ /libs/surfaces/mackie/surface.cc ）、クラス「Surface :: Surface（MackieControlProtocol＆mcp、const std :: string＆device_name、uint32_t number、surface_type_t stype）」のコンストラクターでMackieを修正する必要がありました 。 "、_active（false）"を"、_active（true）"に置き換える必要があります。

Ardourの最初の起動後、プロジェクト名とフォルダー（デフォルトではホーム）を指定する必要があります。 彼は自分でJACKサーバーに接続しました。 ジャックとの同期を有​​効にする必要がある場合は、トランスポートコントロールボタンとタイムコードの間にある内部ボタンを見つけます。 クリックすると、その値はJACKに変わります： ArdourとJACKの同期が有効になります。

幅広い機会に満足しました（無料を考慮）。 また、フェーダーの前のプラスエフェクト（後のプログラムで非常に頻繁に）。



4.3。 効果


ふくらはぎプラグインはすでにハブでLADSPAの最高のものと呼ばれていました。 TAPセット（ TAP Reverberator ）からリバーブを追加します。 ただし、いくつかの問題と解決策の検索がありました。 TAPの影響のほとんどと他のいくつかは排除する必要がありました。 Calf Equalizer * Bandが低音、特に底にガラガラを加えたとします。 そしてピッチシフターはただ-ひどいです。 トラックの開始/終了に大きな衝動、トラックの割れ、または他の同様の問題がある場合-何らかの効果を無効にすることは確かに役立ちます。 幸いなことに、主な種類の効果はいくつかの方法で提供されます。

ギター用のGuitarixプラグインの個別のセットがあります。 このセットからのオーバーロードは、私たちの意見ではLADSPAの中で最高です。 ただし、この場合、コンボアンプとペダルのオーバーロードが使用されたため、必要ありませんでした。



4.4。 ドラムミキシング


タイヤの存在に満足-同じエフェクトを使用するために複数のトラックを接続できるオブジェクト。 ミキシングでは、ドラムは次のようにArdourで処理されました。

  • 各機器のバッチがwaveにエクスポートされました。
  • これらのトラックはすべてArdourに追加されました。
  • それぞれにコンプレッサーが掛けられていました。
  • 4つのタイヤが作成されました：大きなドラム、スネアドラム、タム、およびプレート。
  • イコライザーが各バスに掛けられました。
  • タイヤ「ビッグドラム+スネアドラム」、「すべて」、およびマスターを追加。
  • これらのタイヤはalbum.rf / mix / drums-and-bassの推奨に従って処理されました。


Hydrogenは、コンポジションを複数のオーディオファイルにエクスポートすることをサポートしています（楽器ごとに個別）。 JACKを介してArdourに直接接続することもできます。 ただし、コンピューターリソースを節約するには、標準のエクスポート/インポートウェーブを使用することをお勧めします。

flacおよびmp3形式へのエクスポートには、 サウンドコンバータが使用されました。 EasyTagとKid3-Qtを使用してタグを追加しました。 表紙はGimpのおなじみのデザイナーによって描かれました。 表紙のフォントはwebfont.ruから借用しました 。 曲のプレゼンテーションバージョン（最初の詩+コーラス）は、 Audacityを使用して作成されました。
