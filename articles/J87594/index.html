より速く、より高く、より強く-A-Data Fast Memory Review

A-DATAは、RAMの主要メーカーの1つです。 また、品揃えには、フラッシュメモリ、メモリカード、SSDディスク、外部ドライブ、および高性能ソリューションが別個のアイテムです。 A-Dataは新しいDDR3 2200（8-8-8-24-24）1.65Vメモリキットを導入しました。これは製品ラインで最速のメモリです。 今日の彼女について、そして議論されます。







XPG +シリーズは、高性能ソリューションのクラスに属します。 このクラスのモデル専用の特別に設計されたマイクロサイトもあります。



モデルを試してみましょう：AX3U2200PB2G8-DP2

仕様：

チャンネル数：2

モジュールタイプ：240Pin Unbuffered-DIMM Non-ECC

ボリューム：4GBデュアルチャネルキット（2GBx2）

動作周波数：2200MHz

タイミング：8-8-8-24

動作電圧：1.55V〜1.65V



冷却と安定性を向上させるために、ストリップは次の技術を使用します：熱伝導技術（TCT）、DRAMモジュールの重要な部分から直接熱を除去するとともに、ボード自体の銅の二重層（2オンス）を使用する技術。



このようなメモリは、高性能のゲームやワークステーションの一部として使用するためのものです。

問題のモデルは、最も高価で有望なElpida Hyperチップに基づいています。 ご存じのように、低レイテンシーとアグレッシブに低いタイミングだけでなく、高いレイテンシーと高いタイミングでも動作します。 これにより、パフォーマンスを最大化するためにメモリサブシステムを柔軟に調整できます。



















仕様の前半で示したように、テスト対象のモデルは、デュアルチャネルメモリコントローラーを搭載したシステム用に設計されています。 まず第一に、プロセッサソケットLGA 1156を備えたプラットフォーム向けです。800番シリーズにはかなり強力なメモリコントローラが内蔵されているため、より正確には、このようなストリップの魅力を感じるにはCore i7 860/870プロセッサが必要ですシステムバスに対するメモリ乗算器x12。



Intelの公式仕様によると、Core i7 860/870プロセッサの一部として、JEDEC標準化委員会が採用した最大互換メモリモデルは、9-9-9-24のタイミングで1333 MHzの周波数で動作します。 私たちの場合、標準とはまったく異なる8-8-8-22のタイミングで2200 MHzの周波数で動作できるハンドストラップを手に持っており、かなり目に見えるパフォーマンスの向上を約束します。



まず、2つのモードでパフォーマンスの違いを比較しましょう。



1333 MHz 9-9-9-24および2200 MHz 8-8-8-24。



プロセッサの周波数には触れません。

ところで、SPDのA-DATA 2200 XPG plusシリーズには、2200 MHzのXMP設定があります。 つまり、周波数とタイミングの値を手動で設定したくない場合は、BIOSアイテムのExtreme Memory Profile（XMP）で「Profile1」値を選択するだけです。 このアイテムは、MB Intelligent Tweaker（MIT）>>> Advanced Frequency Settings（マザーボードがGIGABYTEの場合）およびAi TweakerまたはExtreme Tweaker（マザーボードがASUSの場合）にあります
