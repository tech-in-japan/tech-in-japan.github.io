標準端末の非常に興味深い代替品





Enlightenmentコミュニティは、ターミナルエミュレーターであるTerminologyの新しいバージョンをリリースしました。 カットの下の詳細



通常の端末の標準機能に加えて、次のものがあります。

-PDF、ビデオ、ベクター、ビットマップファイルをターミナルウィンドウで直接表示

-タブのサポート（非常に古典的ではありません）

-外観のカスタマイズ（壁紙、配色、透明度、フォントなど）

-テキスト/リンクのドラッグのサポート（非常に関連性のある機能、少なくとも私にとって）

-URL、ファイルパス、石鹸アドレスはリンクとして強調表示されます

-リンクによる組み込みコンテンツの表示

-X11とWaylandの両方をサポート

-タッチインターフェースに適合（明らかに、啓蒙プロジェクトの経済的支援に対するサムスンへのオマージュ）

-OpenGLとOpenGL-ES2の両方によるレンダリング（必ずしも要件ではなく、単なるオプションとして）

-マルチバッファ

-ほとんどのrvtxおよびxtermエスケープシーケンス+ Xterm拡張機能をサポート

-などなど...



用語はEFLツールキットに基づいて開発されており、動作するのにEFLおよびlibcライブラリのみが必要ですが、最小限の依存関係を使用し、fbconフレームバッファーの操作をサポートしています。



いくつかのコマンド：



tyls-プレビュー付きのファイルとフォルダーのリスト

typop-小さなウィンドウで画像を表示します。マウスをつつくだけでもかまいません

tycat-端末自体でのみ画像を直接表示する

tyq-typopコマンドにキュー画像を追加します

tybg-背景をサポートされているマルチメディアコンテンツ（画像、ビデオ、gifアニメーション）に変更します

tyalpha-オン/オフ値で、背景の透明度をオン/オフにします



明らかに他のチームがありますが、私は見つけていません



キーボードショートカット：



Shift + PgUp-ページを上にスクロール

Shift + PgDn-ページを下にスクロール

Shift + Insert-クリップボードから貼り付け（Ctrl + V / C）

Shift + Keypad-Plus-フォントサイズを大きくする

Shift +キーボードマイナス-フォントサイズを小さくする

Shift +キーパッド-乗算-フォントサイズのリセット（デフォルトサイズは10）

Ctrl + PgUp-フォーカスを前のターミナルウィンドウに変更します

Ctrl + PgDn-次のターミナルウィンドウにフォーカスを変更

Ctrl + Shift + T-新しいターミナルウィンドウを作成します

Ctrl + Shift + Home-ターミナルウィンドウ間を移動する

Ctrl + Shift + PgUp-端末を水平に分割します

Ctrl + Shift + PgDn-端末を垂直に分割します

Alt + Home-コマンド解釈モード

Alt + Return-最初の選択を貼り付け

Ctrl + Shift + C-現在の選択をクリップボードにコピー

Ctrl + Shift + V-クリップボードから現在の選択を貼り付けます



マウス制御：



マウスの右ボタン-セットアップメニュー

マウスの中ボタン-クリップボードの選択を貼り付け

マウスの左ボタン=選択

スクロールホイール-上下にスクロール

Ctrl +ホイール=フォントの増加/減少
