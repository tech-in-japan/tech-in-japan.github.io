ファイルシステムとしてのYandex.Disk



最近、Yandex は DropBoxに似た新しいサービスを発表しました。 多くはすぐにバケツからそれを注ぎ始めましたが、私は無駄に思うと思います。



個人的には、Linuxoidの利点の1つです。DropBoxの場合のように、追加のアプリケーションをインストールする必要はありません。 サービスは、WebDAVを使用してNautilusファイルマネージャーを介して接続できます。 実際、私はすぐにやった。



ただし、guaアクセスは完全に良好ではなく、誰もコンソールをキャンセルしなかったため、私はこれに限定しませんでした。





Yandex.Diskをファイルシステムとして接続します （Ubuntuなど）。



davfs2パッケージをインストールします。 



# apt-get install davfs2




: 



# mkdir /mnt/yandex.disk




, . (use https, Luke!): 



# mount -t davfs https://webdav.yandex.ru /mnt/yandex.disk/




, : 

# df -h /mnt/yandex.disk/
Filesystem            Size  Used Avail Use% Mounted on
https://webdav.yandex.ru
                       10G  7.7M   10G   1% /mnt/yandex.disk
