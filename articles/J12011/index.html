Web 2.0-起源、指定、開発...

Web 2.0という用語は単に遍在し、日常生活に突入しました...しかし、それはどういう意味ですか、誰もがそれを理解することを意味します。人生の意味は誰にとって...

そして、彼はどこから来たのですか、そして彼の先祖が誰であるとはどういう意味ですか？ すぐに私は特にないことに注意したい それは社会的および専門的なネットワークではありません。 彼はこの言葉からどこに来たのですか？ ティム・オライリーはかつてそれを初めて語り、世界中に膨大な数のサービスが登場しましたが、その作成者はこのコンセプトの一部として作られたと言うことができます。 最も人気のあるエリアの1つは、ソーシャルニュースサイトになりました。 次に、RSS、RDF、Atom、FLVのテクノロジーが登場しました...

だから、私はティムが「2.0」、はい、はい、USB 2.0とやり取りしたと思い込んでいます。 したがって、USB 2.0はWeb 2.0の先祖です（これで終わりです！）。 usb 1.1に関連するUSB​​ 2.0自体は、スループットが高いことを意味します。これは、写真、ビデオ、オーディオ、プリンター、スキャナーなど、USB 2と互換性のある高速動作する新しいデバイスの出現の手段でした...

すべてのデバイスがusb 2.0に直接接続されているわけではありませんが、この標準が非常に長い間登場したとき（3-5年前）、私のデバイスとPCはUSB 2.0をサポートしていると言うのが流行しました。 そのため、USB 2.0は高速化、低消費電力化を象徴し、その結果、世界のデバイスの事実上の標準となりました。



Web 2.0へ。 類似点。 カメラ、USB 2.0を搭載したビデオカメラ-これは、Picasa、YouTubeなどとの類推による場合です。



したがって、Web 2.0-最初のターンでのみ、ブロードバンド、光ファイバーの出現が示されます。これがないと、ギガバイトの情報や他のサービスを備えたYouTubeはありませんでした。



そして、2番目のターンでは、これはインストールの必要がないGoogleマップ、GoogleドックなどのCloudWareなどのソフトウェアを作成する機会です。 この機能は、スループットの向上の結果です。 最新のWeb OSは、RSSあり/なし、プレーヤーあり/なしの通常のインターネットサービスです...このプラットフォーム（Web OS）では、ハードドライブの速度と比較してインターネット速度が必要ですが、200 kbitではなく、2 Mbpsでもありません。すべてが異なる要素を備えた通常の平凡なサイトです。



そして3（これは重要です）。 人々がこの用語にあまり集中せず、根本的に新しいものを作成し、あるサービスやアプリケーションの先駆者になることを望んでいます。たとえば、かつてはeBay、YouTube、Yahoo、MySpace、そしてもちろんAppleとMicrosoftでした。 1.0（dotCOMに精通している全員以外）は行われませんでした。

すべての人に幸運と成功を！ オリジナルを作成してください！
