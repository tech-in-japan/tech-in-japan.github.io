CSSを使用した画像の影。 刷新

これに触発された： habrahabr.ru/blog/css/36860.html

最初に、必要な画像のダブル/カスタムフレームを作成することを思い出させてください{... cut ...}、またはコンテナに画像を配置し、コンテナのインデントと背景画像を設定します。



などなど。 もっと良い方法があります！





いいえ、コンテナ、ボーダーボード、javascriptはありませんが、CSSは1行だけです。



img { background: url(shadow.jpg) right bottom no-repeat; padding: 0 12px 14px 0; }







はい、はい！ まさに。 imgの場合、色だけでなく背景を画像として設定することもできます。 そして... *ドラムロール*どこでも動作します！ （そして、なぜ彼は働かないのですか？）

どうやら、ほとんどのWeb開発者はimgは図面であるという印象を持っていますが、それだけなのに、なぜ背景が必要なのでしょうか？ :)そして、彼らはコンテナに影を刺し始めました...ステレオタイプを破ります！
