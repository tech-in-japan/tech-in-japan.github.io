Windows RT 個人的な経験

多くの場合、またはほぼ常に、デバイスを購入するときに、必要な機能を彼が実行することを期待しています。 すべてのデバイスが彼らの期待を正当化するわけではありませんが、それについて考えると、ガジェットメーカーもソフトウェアクリエーターも、バイヤー自身が責任を負わないことがわかります。 ユーザーは何らかの機能を待っているため、開発者はデバイスに特定の機能を追加します。 ここから、購入する前に、さまざまなデバイスで必要な機能を探し、気に入ったものや友人からアドバイスされたものだけでなく、見つけたものを購入する必要があるという考えが生まれます。 そのような行為は正しいでしょう。 Windows RTで便利なタブレットにできる人を見てみましょう。







Windows RT


Windows RTオペレーティングシステム自体は、2012年10月26日に初めて公開され、ARMプロセッサベースのタブレットコンピューター用のモバイルオペレーティングシステムとして位置付けられています。 残念ながら、x86-x64命令セット（Intel、AMD）を搭載したプロセッサ用に作成されたソフトウェア製品はサポートしていません。 公式ストアからのみアプリケーションをインストールできます。 2013年11月現在、Windowsストアには約12万5千のアプリケーションが含まれています。 これはそれほど多くはありませんが、原則として、仕事に必要なものは十分にあります。 それについては後で詳しく説明します。 このOS上のデバイスは、Lenovo、DellおよびMicrosoft自体の異なる企業によって製造されています。

後者の製品であるSurface RTはしばらくの間私と一緒に暮らしてきましたが、その例で私が思うすべてを表現します。 実際、Windows RTは本格的なWindows 8の簡易バージョンです。同じデスクトップ、コントロールパネル、スタートアップ、ウィンドウ、フォルダー、ショートカットなど。 「デスクトップ」向けの本格的なプログラムのうち、MS Office 2013、ペイント、および「ユーティリティ」の標準セット（ボイスレコーダー、コマンドラインなど）の他のプログラムのみがあります。 作業のすべての機能は、ストアの「メトロ」アプリケーションにより実行されます。 特定の種類のファイルをデスクトップからアプリケーションを介して実行するように割り当てても問題はありません。 タブレットとOS自体の両方を過剰に評価したり、ダウングレードしたりすることはできません。そのため、あなたはそれについて考えることができません。 当初からiPadを購入したかったのですが、Surfaceの方向に選択をしたので、今度はこの急旋回の背後で私が待っていたものを説明します。 もちろん、Windows RTには長所と短所がありますが、ここにいくつかあります。詳しく見てみましょう。



長所


彼らは常に善から始めるので、私はそうします。 Windows RTのタブレットが絶対に役に立たないか悪いと言うのは不可能です。これは真実ではないからです。 Windows RTは「実行不可能なシステム」であるということを、さまざまなサイトやフォーラムで聞いたり読んだりしているでしょう。 人によって意見は異なりますが、そのようなレビューを書く人のほとんどは、適切な評価に必要な情報の量と質が適切ではありません。 まず、Microsoftのオペレーティングシステム上のガジェットは、ゲーム、サービス、アプリケーションなどよりもドキュメント、ファイル、マルチメディアを扱うように設計されています。 最も重要なプラスは、プレインストールされた本格的なデスクトップMS Office 2013です。完全にすべての機能とアプリケーションパッケージを備えています。 Androidなどのサードパーティ製品と比較して、非常に高速かつ適切に動作します。 2013バージョンには、SkyDriveクラウド（まもなくOneDriveに名称が変更されます）にドキュメントを直接直接保存する優れた機能もあり、複数のデバイス上の1つのファイルで便利な作業を提供します。 次の大きなプラスはデスクトップです。 実際、これは、他のモバイルOSの類似の機能と比較できない最高のマルチタスクおよびファイルマネージャーです。 長い間、デスクトップモードとウィンドウモードについて話すことができますが、なぜですか？ 誰もが（ほぼ）個人の自宅または職場のコンピューターにWindowsをインストールしており、それが何であり、なぜそれが良いのかを知っています。 画面をいくつかの作業部分に分割して作業することも便利です。 2つのアプリケーションを開いて、画面上に落ち着いて並行して配置し、同時に作業することができます。 たとえば、一方にはブラウザまたはデスクトップを配置し、もう一方にはチャット用に少し小さいSkypeを挿入できます。 AndroidタブレットやiPadと比較して他に大きな利点はありません。 ささいなこと：Internet Explorerを使用すると非常に便利です。 なんで？ いいえ、彼は非常に速く安定しませんでした。 しかし、それは本格的なデスクトップブラウザのように機能します。 はい、Facebook、Odnoklassniki、VKontakteでゲームをプレイできます。 もちろん、Windows 8にホームPC、Windows Phoneにスマートフォンがある場合、Windows RTのタブレットはデータ同期の点で非常に便利です。 それにもかかわらず、どのようにMicrosoftが独自のエコシステムを持っています。 また、Xboxも接続する場合、RTタブレットが必要です。











短所


この製品の最も快適な側面はありません。 たとえば、満足のいくゲームをプレイしてもうまくいきません。 市場では、文字通り指で良いゲームを数えることができるので、あなたがゲーマーなら、これは間違いなくあなたのためではありません。 タブレットには奇妙な動作もあります。 たとえば、ロックを解除すると、デザイン色の背景が空になります。これは、「Win」キーを押すだけで処理できます。 重要ではありませんが、それでもです。 また、システムは「バッテリーなし」というメッセージを表示することがよくありますが、Surfaceでは削除されません。 ロックされた状態でタブレットから物理キーボードを取り外すと、以降の操作でOSにタッチキーボードが表示されないため、タブレットを再起動するか、タッチカバーまたはタイプカバーを再接続する必要があります。 さて、もう一度、デスクトップ用の本格的なプログラムをインストールする方法がないことを覚えておく必要があります。 これは実際には非常にまれであり、不快感を引き起こします。 デスクトップを開き、Photoshop、Google Chromeなどを実行したいが、悲しいかな。 ちなみに、Photoshopは私が一番寂しかったです。 市場にはAdobeの公式Photoshopアプリケーションがありますが、私のスマートフォンの標準的なフォトエディターよりも機能が少なくなっています。 そのようなプログラムを使用したい場合は、Surface Proを使用する必要があり、これは迷惑な意見としてマイナスではないと見なすことができることに、あなたは正しく気付いています。 本格的なデスクトップブラウザーとしてのパフォーマンスが低いとは思われないInternet Explorerは、依然として不安定なパフォーマンスを持ち、長時間使用すると速度が低下し、Vimeoからのビデオを開いてカットすることを拒否します。 最も悲しいことは、価値のある類似物がないことです。 UCブラウザがありますが、安定性にも違いはありません。 公式のVkontakteアプリは非常に長い間「考えている」。 メッセージまたは返信に関する情報は、ログアウトしてクライアントにログインすることにより、ほぼ手動で更新されます。 多くの場合、大量のテキストをメッセージで送信すると、単にクラッシュします。 コンテンツの表示はひどく、明確ではありません。 これは主にアプリケーション自体の問題ですが、更新を急ぐ必要はありません。 しかし、TwitterとFacebookのクライアントは非常に機能的で、便利で美しいです。 彼らの仕事について苦情はありません。 バッテリーを100％充電し、ロックされたデバイスをちょうど横になっているテーブルに置いた別のケースがありました。 約8時間後、私はそれを手に取り、電源を切りました。 デバイスの電源を入れようとしたときに、電荷がゼロになっていることを大まかに示唆しました。 奇妙ですが、おそらくデバイス自体に問題があります。







表面


タブレット自体について話すことはあまりありません。 特性はオンラインまたは公式ウェブサイトで見つけることができます。 ケースはアルミニウム製で、手になじみます。 私が気に入ったのは、高所で品質を構築し、すべてのボタンがうまく機能する、非常に便利なスタンド、寿命を簡素化する磁気充電コネクタ、microSDスペースの可用性（購入後1週間で学んだこと）、本格的なUSB入力の存在（デバイスから非常に便利です）ドキュメントで動作するように設計されており、多くの場合、フラッシュドライブやミニHDMIで転送されます。 そして今、イライラしているのは、画面とアルミニウムの背面全体が苦痛を伴う生物学的情報（印刷物）を収集することです。 また、ボリュームは非常に満足のいくものではありません。 音量が100％の騒々しい会社では、ほとんど何も聞こえません。 カメラはダメです。 メインカメラとフロントカメラは約0.3メガピクセルです。 Skypeに十分です。 厚さはそれほど大きくはありませんが、普通ですが、片手での重さは持ちやすいとは限りません。 Touch Coverを犠牲にして、私の意見は分かれました。 一方で、素材の理想的なものは、触ると非常に心地よく、繊細です。 一方、タイピングは便利ではありません。 キーストローク、それらの間のギャップを感じないため、特に長いテキストを入力するときに不快感が生じます。 Surface（RTまたはPro）をタイプカバーで使用する場合は、引き続きアドバイスします。
