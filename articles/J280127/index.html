IBM Watsonは人間の感情を理解し、ユーザーの気持ちを知ることで対応します





IBM Watson認知システムは改善されています。 企業の専門家は、「感情を認識するためのモジュール」を大幅に更新し、システムに人の感情状態を認識する機能を提供しました。 分析データに基づいて、システムは、人がどのように感じているかを知りながら、それに応じて応答することができます。



トーンアナライザー 、 感情分析 、 視覚認識の 3つのAPIが一度に追加されました（これまでのベータ版）。 Text to Speech（TTS）サービスが更新され、感情の新しい機会が生まれました。また、 表現力豊かなTTS音声モジュールAPIへのアクセスも開かれました。 これはすべて、人間と機械の間の相互作用の境界を大幅に拡大し、機械は人間の感情に共感するように訓練され、共感を加えました。



IBM Watsonの最高経営責任者（CEO）であるDavid Kennyは、次のように述べています。「人工知能に基づいた動的なアプリケーションとサービスを開発コミュニティに提供することで、IBM Watsonプラットフォームの機能を改善し続けています。 「プラットフォームを簡素化することで、IBM Watsonに基づく基礎の作成、トレーニング、実装のプロセスをより理解しやすく合理的にしています」と彼は付け加えます。



IBMはNode 、 Java 、 Python 、SDK、およびWatsonポートフォリオの一部である最近導入されたiOS SwiftとUnityを操作するための新しいツールを導入し、Watsonインターフェースを改善しています。 開発者は自由にApplication Starter Kitsを入手しました。これにより、開発者はWatsonに基づいてアプリケーションとサービスの開発を加速できます。 すべてのAPIは、IBM Watson Developer Cloud Bluemixプラットフォームで使用可能です。



トーンアナライザー、感情分析、視覚認識



IBM Watson Tone Analyzerは、テキスト通信の全体的なトーンを設定するのに役立ちます。 これは、Watson Developer Cloud APIおよびSDKの最新のアドインの1つであり、Bluemix環境で使用可能です。



感情的な着色 。 文学で説明され、子供の頃から私たちによく知られている人々の感情には、怒り、恐れ、期待、驚き、喜び、悲しみ、信頼などが含まれます。 サービス開発者は、テキスト内の感情を評価するためのモデルを作成しました。 Tone Analyzerは、すべての感情をポジティブな感情、ネガティブな感情、怒りの3つのカテゴリに分類します。 ポジティブな感情には、喜び、楽観、インスピレーション、幸福、満足が含まれます。 ネガティブ-恐怖、嫌悪、否定、屈辱、罪悪感、絶望。 怒りは、攻撃性、怒り、欲求不満、苛立ち、敵意などの要素を伴う一種の否定的な感情です。



社会的要素 。 また、オープン性、同意の表明、誠実さ（誠意）という3つのタイプに分けられます。 開放性とは、新しい体験に対する人の準備です。 同意の表現-妥協と協力を見つける傾向。 誠実、誠意-それに応じて行動したいという願望。



プレゼンテーションのスタイル 。 そして、3つのカテゴリーがあります：分析、信頼、および注意。 分析スタイルのプレゼンテーションは、メッセージの著者が分析し、物事の本質を理解したいという欲求を示しています。 自信は、何かに対する自信の度合いを反映しています。 そして、注意はイベントの漸進的な発展への願望です。



IBM は感情分析を AlchemyLanguage APIパッケージに追加しました。 深い言語処理と分析を使用して、開発者が他の人の感情をよりよく理解できるようにします。 開発者は今、ポジティブな感情とネガティブな感情を識別し、喜び、悲しみ、恐怖、嫌悪感、怒りなどの幅広い感情を強調することができます。 これはすべて、たとえば、ユーザーのレビュー、ソーシャルネットワークへの投稿、通常のコメントの分析に使用できます。 企業は、肯定的なレビューと否定的なレビューの比率を評価することにより、製品に対する顧客の態度をすばやく判断できます。 したがって、ユーザーの基本的な感情を強調し、製品またはサービスを変更することで対応できます。



視覚認識は、画像を認識して分類する機能を提供します。 視覚認識のあらゆる分野でこの技術を使用する前に、モジュールが将来画像を認識する「トレーニング資料」を提供できます。



他の会社にも類似物がありますが、重要な違いもあります。 ここで、IBMの主な目標は、さまざまなタスクに使用できるツールを開発者に提供し、独自のマテリアルでトレーニングして、カスタマイズされたイメージ分類子を作成することです。 たとえば、小売業者は視覚認識を教えて、ソーシャルネットワーク上の写真で自分の服を認識し、視聴者の規模を分析できます。



表現力豊かなテキストからスピーチ



この技術の作業は12年以上にわたって継続されています。 現在、Watson TTS APIの形式ですべての人が利用できるようになっています。 認知システムは、以前は機械では利用できなかった程度の音声を認識します。 これには、音声のトーン、コンテキスト、および感情的な色付けに関するコンピューターの「理解」が含まれます。 この分析に基づいて、コンピューターはそれに応じて応答できます。



以前は、このような状況のコンピューターは、事前に定義された一連の単語に基づいて動作していました。 「コントロールセット」からの1つ以上の単語の出現により、アルゴリズムの1つがトリガーされ、コンピューターが「応答」しました。 現在、「テキストベースの分析」システムは、個々の単語を分析するだけでなく、スピーチの感情的な方向、そのトーン、およびセマンティックコンポーネントも決定します。 Watson TTS APIに基づいて、開発者は独自のアプリケーションとサービスを作成し、人間とコンピューターの相互作用の原理を簡素化および改善できます。



現在、IBM Watsonプラットフォームには、 IBM Watson Developer Cloud of the Bluemixプラットフォームで利用可能なさまざまなサービスがあります。 認知システムコミュニティには、80,000人を超える開発者、学生、起業家、エンジニア、技術者が含まれます。
