::「ビデオカメラの選択を始める場所」のトピックに関するBazaar ::

私は自分が「ビデオカメラにこだわり、撮影と編集をしている賢い人」だとは思っていませんが、重要なことを切り捨てました。 「甘やかす」ためにビデオカメラを購入する場合は、次のテキストが役立ちます。 そして、もしあなたがMacで作業しているなら、最後に読むことが義務付けられたピペットです。 ちなみに、下のテキストは一般的にごくわずかです!!!







要するに、ビデオカメラを選択する上で最も重要なことは、書き込み先のファイル形式です 。



フィルムテープに書き込むカメラ（最も安いオプション）を使用する場合は、キャプチャしたすべてをデジタル化するものを入手する必要があります。 Pinnacle製品には、安価な内蔵PCボードから、カメラやPC / Macに接続された非常に外部の部品まで、多くのオプションがあります。



フィルムカセットはそのようなものです。タンパクの少女のように、特定の種類のカセットを購入し、最終的にはカートを購入する必要があるからです。



別のオプションがあります。カメラがCD / miniCDに直接書き込む場合、すべてがより簡単になりますが、...インストールするためには、まずPC / Macですべてを取得する必要があります。 そして、「レーキ」のプロセスは、品質を改善するものではありません。 しかし、そのようなカメラがプラスされています。ディスクがすべてのルールに従って破棄されて保存されない場合、キャプチャされたものは常に翼で待機しているアーカイブになります。 そして、カメラの品質は原則として非常に優れています。



ディスクとミニディスク-これは、本当にお金を節約する必要があるときの平均的なオプションですが、たわごとは本当に必要ありません。 それは独自の特徴を持っていますが、彼らはそれが習慣の問題だと言います。 ちなみに、テープの場合のように、カメラにはメモリカード用の穴がありますが、写真専用です（品質は明らかです）。 ちなみに、これは単純な誤解です。ビデオカメラのように、ビデオから静止画を撮影して写真を撮ることができるので、これはよりクールです。 しかし、これら2つのカメラを搭載したバージョンでは-確かではありません。恐ろしいカカがまだいるからです。



3番目のオプションはHDDカメラです。 ハードドライブへの書き込みを個別のファイルとして入力します。 このバージョンでは、カメラを60ほど購入することで本当に中断することができます。そして、ビデオ撮影が終わったら、カメラをPC / Macに接続し、カメラのパパを調べ、ビデオファイルを見つけません。 かっこいい？ ここで私も話しています。



いくつかのソニー製カメラはmpg形式で書き込み、一部はmts（私のような）で書き込みますが、さらにオプションがあります。 私は、他のメーカーが独自のいくつかの他のフォーマットを考え出すことができると思います。 スーパーマーケットの絶対にすべてのカメラの売り手はあなたに明確な答えを与えることができないので、購入する前に「選択したカメラが書き込むフォーマット」というトピックに関する「google tyrnet」をお勧めします。 原則として、特にMediaMarktの一部のブティックでは、10日以内に問題なくカメラを渡すことができますが、ばかげた理由で「内部に収まらなかった」（試してみた）ことがあります。

したがって、mpg形式で書き込むカメラを購入すると、Macユーザーは次のことを行います：プレーヤー〜「VLC」が撮影したすべてを見るか、QuickTimeプレーヤーの有料アドオンを購入（またはトレントを検索）して、mpg- iMovie / FinalCutStudio /その他のプログラムのファイル。 ちなみに、iMovie'08はより複雑であると広く認識されているため、iMovie'06にマウントしますが、出力で超高品質を得るためには非常に困難です...



または、mts形式で書き込むカメラを購入して、タンバリンと踊る準備をします（私もそうです）。 ここでは、魔法のMTSトランスコーダーを見つけて購入し（または、私が時間内に成功しなかった壊れたトランスコーダーを探して）、ビデオの音がビデオ自体よりも速い理由を解明する必要があります。 MPEG Streamトランスコーダーというオプションがありますが、Apple WebサイトでMPGサポートを購入する必要があります。 爬虫類...要するに、hemoなど。 ところで、はい、MTS形式の場合。 「ブルーレイ」というディスクを知っていますか？ これらの高価なディスクにビデオが書き込まれるのは、MTS形式です。 どうやって？ これは必要ありませんか？!!! どうして私はあなたをvyましく思いますか...-）））



紳士、本当にあなたのための2つのオプションがあります。 「Srednyachkovy」-カメラはディスク/ミニディスクに書き込みます。 原則として、多くの人々がそのようなカメラを持っているだけで、彼らは何も住んでいません。 また、「クール」オプションは、カメラがハードドライブ（HDD）に書き込みを行う場合で、より控えめなおばあちゃんの場合は、今日最高品質で書き込みます（つまり、ショーオフは最終的に屋根よりも高くなります）。



したがって、すべてのビデオカメラを3つのカテゴリに分割し、正しいカテゴリでのみ泳ぐことができます。 原則として、カメラの場合は「高価なほど良い」というロジックが機能するため、カメラを選択するときは、最小値ではなく、可能な最大値でガイドする方が適切です。
