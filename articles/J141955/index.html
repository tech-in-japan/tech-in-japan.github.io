MegaFonはDDoSからの保護を提供しますか？



最近、朝にサニタリーファイアンスの玉座に座って、突然、光沢のある雑誌で広告を見つけました。「メガホンはDDoS攻撃に対する保護サービスです」。 ただし、このサービスがどのように顧客に実装されているか、およびターゲットオーディエンスの「口径」は、サービスの説明からは明らかではありません。 これはサイト自体では驚くことではありません。 うーん、奇妙なことに、サービスは完全に宣伝されており、「メガホンddos」を検索しても、Habréに結果がもたらされませんでした。 このギャップを埋めることにしました。 私はこの会社の仕事とは何の関係もありません、投稿は支払われませんでした（彼らがここに自分のブログを持っていると思われるようです）



DoS / DDoS攻撃に対抗するサービスはどのように機能しますか？

MegaFon保護装置は、年中無休で情報リソースに到達するトラフィックを分析します。 DoS / DDoS攻撃が検出された場合、保護機器は有用なトラフィックを通過させ、悪意のあるトラフィックをブロックします。



DoS / DDoS攻撃に対抗するサービスの提供は、次の2つの方法で可能です。

  • 保護装置は、自動モードでDoS / DDoS攻撃に対抗します。
  • リモートアクセスは、DoS / DDoS攻撃に個別に対抗する機能を備えたトラフィックに関する統計情報への個人アカウントを介して提供されます。


MegaFonが提供するDoS / DDoS攻撃に対抗するサービスの特徴は次のとおりです。

  • 最大10 Gb / sの攻撃に対抗します。
  • 継続的に更新される署名データベース。
  • 高レベルの保護。
  • 24時間のサービスサポート。
  • 魅力的な料金ポリシー。
