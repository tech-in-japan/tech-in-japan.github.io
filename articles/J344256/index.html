「教区の3つのクラス」またはMicrosoft Virtual Academy（MVA）で学ぶ

.NETやその他のMicrosoftソリューション向けの開発に興味のある多くの人が、 Microsoft Virtual Academyについて何かを聞いたことがあるかもしれません。



実際、Habréにはこれらのコースに関する十分な情報がありますが、検索結果が示すように、MVAがパスではなく完全に言及されているほとんどの関連記事は、何らかの形でMicrosoftとつながり、企業ブログに投稿された人々によって作成されました。



この学習リソースについての独立した主観的な見解を提供します。 ほとんどのMVAコンテンツは初心者向けに設計されているため、すぐに注意します。おそらく、この資料は初心者やアマチュア、プロ向けの可能性が高いと思います。



このレビューでは、ポータルの説明、賛否両論、他のプラットフォームとの比較について簡単に検討します。 まだ興味があるなら、カットへようこそ！









ただちに予約してください。ここにあるのは単に私の意見です。私自身はプログラマーではないので、コミュニティの観点からいくつかのことを間違って評価することができます。



それでは、MVAとは何ですか？ マイクロソフトによると、「国際レベルの無料オンラインコースは、技術的なスキルを獲得し、キャリアの成長を継続するのに役立ちます」







国際レベルの概念はややあいまいであり、さまざまな方法で解釈できます。 さまざまな国の人々にとって高級なコースであり、理解しやすいものです。 2番目の解釈と矛盾がない場合、最初の解釈ではすべてが明確ではありません。 いずれにせよ、このサイトでは、現地の専門家と「衰退しつつある西」 （冗談）の国々の両方が用意した異なる言語のコースを提供しています。



一方、一部のコース（少数派）では、場所は明らかに国際レベルではなく、地元のコンピューターサイエンスキャビネットのレベルにあります。 確かに、西洋の講師とのコースはしばしば専門的に行われ、国内のコースよりも専門的に行われますが、それについては後で詳しく説明します。

パートI：一般情報。





おそらく、Microsoft Virtual Academyのコースの最も重要な利点は、完全に無料であることです。 登録してトレーニングを開始するだけです。



登録後、コースを受講し、最後に証明書を受け取ることができます。 まだ何らかの評価システムがありましたが（私は注意を払っていませんでした）、今ではバッジ-「Member of the MVA Founders Club」に置き換えられているようです。将来的には他のモチベーションシステムがあるかもしれません（ 詳細はこちら ）

修了証がどのように見えるかを気にする人は、それをネタバレの下に隠しました。



コース証明書






彼には何の価値もないと思いますが、私たちは証明書のために勉強しているだけでなく、コース自体に移りましょう。



Microsoftに期限を与える必要があります-原則として、多くのコースがあり、それらはいくつかの一般的な主要分野に分かれています。これがメインページの外観です。







コースに加えて、理論的にはトレーニング計画を立てるのに役立つ専門分野も登場したようです。 トレーニングスキームはほとんどありません-12のみで、それらのほとんどはITプロフェッショナル向けです（開発者向けではないという意味で）。 すべてのコースを修了すると、トレーニングスキームは「メダル」を与えるように見えますが、私はまだ修了していないので、確実に言うことはできません。



コース自体に進みましょう。 ネタバレの下の写真に示されているように、あらゆる好み、多くの方向、異なる言語のコースがあります。

人生でしばしば起こるように、肯定的な側面はあまり強く伝えられませんが、マイナスは常に風味があります（テレビで政治ショーや分析放送を見ます）。したがって、今回はこの傾向にも屈するようですので、プロは短いですが、より詳細な短所。

パートIII：短所





いつものように、マイナス面は、原則として、より強い感情を引き起こします。



1.ブレーキとUI-あまり良いエンジンではありません。 私は専門家ではなく、おそらく私が判断することはできませんが、サイトを気にします-凍結しているのは、開始画面にロード画面を取り付けた理由ではありません。事実、あなたはそれを常に表示し、長い間ハングするでしょう







エンジンには他にも不都合があります。たとえば、通常は新しいタブを開いて期待した結果を得ることができません。



たとえば、コースのコンテンツ（多くの写真の右側にあるブロック）で、現在見ているものとは異なる別の章を選択した場合、別のタブで必要なものを取得できません（別の章を開く）。 代わりに、あなたが残したのと同じコンテンツを受け取ります。



同様に、[戻る]ボタンはうまく機能しません。 たとえば、動画を視聴してからコースの別の要素（テストなど）をクリックした場合、ブラウザのボタンを押して別の要素を開いた後、前に見た動画に戻りたいのですが、いや、そのままですちょうどそれが突っ込んだところ。



2.字幕 -サイトは、MS EdgeなどのMicrosoftブラウザー向けに最適に設計されていると言う必要があります。 Edgeの字幕がフルスクリーンでどのように正常に機能するか、またChromiumの巻き戻し線の下でどれだけ素晴らしく非表示になるかを見てください。 これはフルスクリーンモードであるため、右側の左側のエッジはChromiumエンジンのブラウザーであるということを、私の言葉で理解する必要があります。







3.ハングアップ。 -両方のブラウザで同様に発生する可能性があるものがあります-一部のビデオがフリーズします。 これは初心者向けの新しい機械学習コースです。ブラウザに関係なくフリーズします







4.ローカライズ -ビデオの字幕とテストの両方で、ローカライズに問題があります







それでは、おそらく、追加するもの は何もありません



5.テスト -ローカライズされた横枠に加えて、元のテキストと翻訳をスペースで区切るのを馬鹿げたままにしていない場合、右の図のように明らかな矛盾がまだあります（複数の選択肢について話しますが、1つの選択肢があります）多くの同様のシステムでは、講義資料の知識が常に適切に記述されているわけではありません。







6.証明書 -証​​明書へのリンクを取得できないため、サイトではダウンロードのみが許可されます。



7.品質 -すべてのコースが高品質であるとは限りません。2分間のビデオがあるこのような奇跡のコースがあります。







さて、明らかに不利な点があるので、整理して、他のトレーニングプラットフォームと比較してみましょう。



パートIV：他のオンラインコースとの比較





私はオンライントレーニングの経験がほとんどないとすぐに言わなければならないので、ここで私の意見は完全に主観的です。



1. edX-私の意見では、edXプラットフォームに関するMicrosoftのコースはより堅実に見えます。また、場合によっては、MVAに関するコース自体で、edXの継続教育を提案しています。



2. CodecademyまたはSoloLearn-おそらく、最初にCodecademyまたはSoloLearnのトレーニングの方向を比較することは意味がありません。2番目に、コードに重点が置かれ、MVAではビデオとテストの視聴に重点が置かれます。



3. 直観 -もちろん、それらはいくつかの点で似ています。 両方のプラットフォームは、コースの最後に無料のトレーニングと取得の機会を提供します。 Intuitのみが焦点に焦点を当てています。テキスト講義はテストであり、MVAは焦点：ビデオ-テストです。 まあ、MVAコースはよく行われることが多いことに注意する必要がありますが、一方で、MVAは写真（コースへのリンクのある写真）のような前衛芸術の素晴らしい例を描く方法を教えることはまずありません。







4. Coursera-コースラでマイクロソフトのテクノロジーコースを特に見たことがありませんでした。私の経験はすべて機械学習の専門知識のみに基づいていますが、私にとってはコースラ-良いことは本当の知識を与えて汗をかきます（適切なコース）。 確かに、無料のMVAとは異なり、すべてが多少悪くなります。



競争から：

たとえば、MSTUのTC "スペシャリスト"での遠隔学習。 バウマン。 広告とは思わないでください。ちょうど私がレビューを書いたわけではありませんが、すべてを忘れてしまったので、ここに小さな挿入物として残しておきます。



覚えておいて、MVAコースの1つがさらなる研究のために私を直接怒らせたことを少し高く書いたのですか？ それで、私はもっと徹底的なことを勉強すると同時に、職業再訓練の卒業証書を取得するのが最善であると決めました（その時は自由な時間とお金がありました）。 「エキスパート」について多くの良いことを聞いたので、それらを選びました。



その後、私が持っていたプログラムの枠組みの中で、.NET遠隔学習プログラムを受講しました：ウェビナーの記録、マイクロソフトの教科書（英語ではtrue）、および特定の時間にCAにインストールされた作業マシンにリモート接続してラボを作成し、相談する機能先生。 確かに、私はおそらく失敗し、フルタイムの教育を受けなければならなかったことを正直に認めています。



仮想「お尻」を使用してラボを作成するのは不快ですが、最良の場合、コース全体のラボの3分の1の自宅のPCで完全に再現できたので、当然ながらコースの終わりには良いプログラマーになりませんでしたが、少なくとも楽しんでいます。 ソビエトのエンジニアによって育てられた男性として、私はA.コラブリンの講義の講義を楽しんだため、誰もが講義の仕方を好むとは限りませんが、教育機関のITに少なくとも関連するものがあれば、同じレベルで教えられるでしょうおそらく「成長してプログラマーになる」でしょう。



1つの問題-喜びは高価です。 私に税金控除があったとしても、この幸福は80の木製のものの中に何千もあることが判明しました。



「スペシャリスト」についてのこの挿入は、プログラマーが大学で勉強する必要があるかどうかという質問に答える部分的な私の意見です。 任意のプロファイルで高等教育（さらには中等職業教育）があれば、いつでも他の何か（職業再訓練/高度な訓練）を経て、雇用主の会社でSRO証明書を発行するのに役立つかもしれない1つまたは別の活動に従事する公式の権利を取得できますライセンスであるため、「ビジネスと喜び」を組み合わせることができます。



しかし、トピックに近づきましょう。 「スペシャリスト」からの遠隔コースの背景に対して、MVAが練習の観点から与えるものは近くにありませんでした。 また、最後にスペシャリストが書き出すMicrosoftからのある種の証明書は、より深刻に見えます（しかし、正直に言うと、私はそれらを誰にも渡さなかったのかわかりません）。 まあ、再訓練の卒業証書もどこにも行きません。



さて、MVAを守るサイドバーの最後に、「スペシャリスト」でトレーニングした後、MVAのコースは、コース中に忘れたり覚えなかった「穴」を埋めるのに役立つと言います。 座って、加速モードでビデオを見ると、いくつかのことが起こります。 繰り返しますが、MVAは無料です。これは大きなプラスです。



念のため、繰り返しますが、スペシャリストトレーニングセンターとはまったく関係がありません。数年後、人々が良い言葉で覚えてもらいたかったのです。



パートV：誰に役立つか。





それで、私達は上で言われたことを要約します。

私の意見では、Microsoft Virtual Academyのコースは、すべての小さな粗さにもかかわらず、依然として有用なものです。



まず第一に、彼らはプログラミングの絶対的な初心者やIT分野のその他の関連する問題に役立ちます。 アクセス可能な形式で基本的な知識を無料で取得できるので、ツールとリソースに精通し、Microsoftの専門家が使用する優れた実践方法を調べてください。



しかし、ビデオをダウンロードしたり、サイトで直接倍速で視聴したりすることで、まったく新しいものに慣れたり、忘れられた何かを覚えたりすることができるため、これらのコースは、IT分野の「焦げた海の狼」にとってはそれほど有用ではありませんMVAコースは、より経験のある人々にも役立ちます。



ウラジミール・イリイチが言ったように、何らかの方法で、主なことは「勉強する、勉強する、勉強する」が、MVAの助けを借りて、またはどういうわけかそれほど重要ではありません。



ほとんどの読者にとって、 平日は終わりに近づいていると思うので、 良い週末と幸運を！
