[エラーコード：0x00000000]

  4. Run Shell Scriptブロックで、テキストテキストを上記で取得したCEP証明書のSHA1ハッシュパラメーターの値に変更します。
  5. スクリプトを保存します：⌘Command+ S
  6. 「CryptoPro.workflowで署名」ファイルを実行し、インストールを確認します。
  7. [システム環境設定]-> [拡張機能]-> [Finder]に移動し、[CryptoProで署名]クイックアクションがオンになっていることを確認します。
  8. Finderで、任意のファイルのコンテキストメニューを呼び出し、[ クイックアクション]および/または[ サービス]セクションで、[ CryptoProで署名]を選択します。
  9. 表示されるCryptoProダイアログで、CEPからユーザーPINを入力します
  10. 拡張子が* .sgnのファイルが現在のディレクトリに表示されます。これは、DERエンコーディングを使用したCMS形式の切断された署名です。
