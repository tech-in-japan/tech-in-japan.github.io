NoRetypeを使用してブロガーのレビューを購入した方法、または333ルーブルの価格で訪問者を獲得する方法

NoRetypeプロジェクトは、人気のあるブログのページにWebサービスに関するレビューが表示されるようにします-NoRetype.ru Webサイトに書き込まれます。 それがどれだけ役立つか、実験的にテストすることにしました。

広告の目的は、リンパの新しいソーシャルネットワークでした 。

このリソースは、若者を対象としています。 これに基づいて、NoRetypeマネージャーはwww.mail.ru 、 www.liveinternet.ru +オフラインブログを含む少数のITブロガーから人気のあるブロガーをピックアップするように求められました。

マネージャーから寄せられた回答から判断すると、多かれ少なかれ人気のあるブロガーの膨大な数



最初は、次のパラメーターを含む8つのブログが提供されました。

Yandexブログ/ブログホスティング/価格に掲載する

47位/ LiRu / 3000ルーブル。

1464位/ LJ / 1000ルーブル。

3721位/ LiRu / 800ルーブル。

4697位/ LiRu / 600ルーブル。

323位/ LiRu / 600ルーブル。

217位/ LiRu / 800ルーブル。

見つかりません/ Blogspot / 600こする。

6970 / LJ / 600こする





2人のブロガーがレビューを書くことを拒否し（47位になるものを含む）、そのうちの1人が置き換えられました-その結果、7人のブロガーがlimpa.ruに関するレビューを書きました。 これらのレビューには6,000ルーブルかかります。

Yandexのブログによると、4,000ルーブルのナンバー1のトップブロガーも提供されましたが、私たちは拒否しました。

1件のレビューを除くすべてが詳細であり、著者は実際にサイトを見てテキストを書いた。 選択したブログには狭いトピックがありませんでした。

レビューは私たちに何を与えましたか？ あまり人気のないブログからの7件のレビューで、18人の訪問者を得ました。 したがって、1人の訪問者に333ルーブル、33コペックがかかりました 。 悪い結果を期待していましたが、結果が実質的にゼロになるとは期待していませんでした。



このミニ実験から私が学んだ道徳：

1.一般的なブログでレビューを購入-お金を捨てましょう。 トラフィックなし、結果なし。

2.非常に専門的な製品を提供する、トピックの狭いブログへのより効果的な投稿。 たとえば、seoブログで、seoソフトウェアを提供しています。 （そのような広告の個人的な経験がある人は、共有してください。）

3. NoRetypeサービスには、まだ多くのブロガーがいません。 たとえば、mail.ruでブログを提供することはできませんでした。

4.ブロガーはまだレビューを書くのにあまり有益ではありません。 サービスの費用の少なくとも半分がかかる場合、ブロガーは300〜400ルーブル、1500〜2000ルーブルの地域のトップブロガーを受け取ります。
