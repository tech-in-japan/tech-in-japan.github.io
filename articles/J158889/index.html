Linux用にリリースされたSkype 4.1

一連の良いニュース（ Skypeが世界中で1か月間の無料通話を提供します ）とあまり良くない（ スカイプの脆弱性、アカウントを盗むことができます ）Skypeに関するニュースを急いで共有します：

昨日までに、Linuxオペレーティングシステム用のSkype 4.1のリリースに関する公式ブログ投稿が掲載されました。



イノベーションについては、 ローラから引用します。



  • Windows Live Messengerクライアントとチャットする機能。
  • トークウィンドウが変更されました。
  • セカンダリログインのサポートが実装されます-一度にプログラムの複数のインスタンスを実行する機能（コマンドラインに「-secondary」パラメータを追加することで実装されます）。
  • ビデオ会議の改善
  • WebページのSkype URIのサポート（Webサイト開発者は、Webページから直接チャットや通話をサポートするためにWebページにSkype URIを埋め込むことができます。SkypeURIはSkype for Linuxでサポートされるようになりました。名前と電話番号を含むWebページでは、URIが自動的にSkype for Linuxを起動し、指定されたアドレスへの呼び出しを開始します）。
  • さらに、バグが修正され、安定性が向上しています。




リンク：

  • 公式ブログで実装の詳細をご覧ください
  • 最新バージョンをダウンロードしてください 。 これまでのところ、Ubuntu 12.10ではx86パッケージのみが利用可能であることは注目に値します。




PS

Ubuntu 12.10に正常にインストールするためには、新しいパッケージをインストールする前に古いパッケージを手動で削除する必要がありました。



PS2

MicrosoftによるSkypeの購入後、Linuxシステムのサポートが「無効」になるという考えは実現していないようです。
