新しいNetBeans IDE 7.1がリリースされました

このIDEのファンは、おそらく公式Webサイトnetbeans.orgの ロードマップに示されている日付が現実とは多少異なることに気付いたでしょう。 個人的には、12月14日を楽しみにしていて、毎日何かが変わっているかどうかを確認していました。 そして昨日それが起こりました-新しいリリースが一般公開されています。



NetBeans 7.1の主な革新は、JavaFX 2.0を使用した開発の完全なサポート、Swing GUIビルダーの機能の大幅な拡張、CSS3サポート、SwingとJavaFXに基づくアプリケーションインターフェイスの視覚的なデバッグのための新しいツール、およびGitサポートの統合を提供することです。 さらに、新しいバージョンでは、Oracle WebLogic Server 12cと統合するためのツールが追加されています。 NetBeans 7.1インターフェースは、ロシア語を話すユーザー向けに完全にローカライズされています。



私はWeb開発にNetBeansを使用しているため、次のことに興味があります。

  • CSSエディターでのCSS3サポート。新しいCSS3要素のオートコンプリート、構文の強調表示、コンテキストドキュメントのサポートを含みます。
  • さまざまなブラウザーに固有の新しいプロパティのサポート。
  • PHP言語用の改善されたデバッガー。
  • PHPUnitに基づくテストグループのサポート。
  • Smartyテンプレートのサポート。
  • SFTPおよびFTPクライアントのキープアライブサポートにより、ファイルのダウンロードを高速化します。
  • Gitリポジトリを操作するためのツールの統合。
  • エディターウィンドウの変更履歴を表示するタブ。
  • Subversion作業ディレクトリ内のファイルのロックおよびロック解除のサポート。
