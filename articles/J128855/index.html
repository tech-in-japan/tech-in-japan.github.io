Acer Aspire + AMD A6 =高速！

コンピューターデバイス市場は、依然として成長、変化、進化を続けています。

携帯電話（別名スマートフォン）はオンラインになり、HDコンテンツを再生できます。

タブレットはそのニッチをしっかりと占有し、余暇をより総体的かつ便利に過ごすことができます。本を読んだり、遊んだり、お気に入りのテレビ番組を見たり、祖母に写真を見せたりします-これはすべてのタブレットで行うことができます。

ラップトップでは、ほこりを集めてスペースを占有するかさばる引き出しをほぼ完全に捨てることができます。

今日は、読者にラップトップを紹介したいと思います。 ミッドレンジコンピューターを交換できるラップトップ。 Acer Aspire 5560G-6344G64Mnkk。





どんな感じ？



外は、ラップトップは古典的な15.6インチのラップトップで、ダークグレーのカバーで、色の内側は少し茶色に近く、画面の周りのフレームは黒です。







ラップトップのデザインは、ASUSによる最近の作成を連想させます。 ここでは新しい傾向は見ていません。

バッテリーの「ロック」がきれいに見えない限り。







ディスプレイは、光沢仕上げと1366x768ピクセルの解像度を持つ通常のマトリックスです。

タッチパッドはマルチタッチ、ジェスチャーをサポートし、ダブルキーも備えています。







キーボードはいっぱいで、数字キーパッド、島型のキーがあり、その下にほこりやゴミが詰まります。

キーボードの使いやすさはごく普通です。 長いテキストは問題なく入力されます。

インジケーターは青色（電源インジケーター）、オレンジ（Wi-Fi、バッテリー）で点灯します。

すべてのインジケータがはっきりと見えます。 電源インジケータは電源ボタンに刻まれており、残りのインジケータは下部に配置されています。











ラップトップの蓋は約150度開きます。 これで十分だと思います。 カバーは背面パネルにあるため、コネクタはありません。







コネクタは、他のすべての側面に広がっています。

左側には、充電コネクタRJ45、換気グリル（左に暖かい空気が吹き、右利きの人が喜ぶ）、D-Sub、HDMI、USB（2.0）、ヘッドフォンおよびマイクジャックがあります。









右側には2つのUSBコネクタ（2.0）とドライブ（DVD-RW）があります。









前面には、カードリーダーと前述のステータスインジケータがあります。









蓋には、光沢に加えて、ウェブカメラが収まります。 1.3メガピクセル、標準品質。 補間を使用すると、720pが生成されます（ただし、品質はそれほど高くありません）。





ちなみに、充電器は標準の小さな複合ユニット（メモリとネットワークケーブル）です。





中身は？



このラップトップの内部は目新しいものです。 AMD A6プロセッサ。





引用：

AMD AシリーズA6-3400M（コードネームLlano）は、エントリーレベルのクアッドコアノートブックプロセッサです。 AMD Fusionハイブリッドプロセッサフ​​ァミリ（APU）に属します。 4つのx86コアを備えた単一のチップ上に、統合された高性能Radeon HD 6520Gグラフィックスカードがあります。 プロセッサは1.4 GHzの周波数で動作しますが、Turbo Coreのおかげで、この周波数は2.3 GHzに増加する可能性があります。



A6-3400Mに統合されたメモリコントローラーは、デュアルチャネルDDR3L-1333メモリとPCIe 2.0インターフェイスをサポートしています。




私の側では、このプロセッサを2011ファミリのIntel Core i3プロセッサ、またはIntel Core i5と比較しますが、同じパフォーマンスであれば、AMDはクアッドコアIntel Core i5よりも大幅に安くなり、Intel Core i3（2コア）と同じ価格になります。







プロセッサに統合されたグラフィックスカードに加えて、個別のアダプタAMD Radeon HD 6650Mもあります。 もう一度引用してください。

AMD Radeon HD 6650M（ATI Mobility Radeon HD 6650とも呼ばれます）は、高速のミッドレンジラップトップグラフィックスカードです。 HD 5650の後継であるDirectX 11をサポートします。カードの機能：より高い周波数、80シェーダー、Eyefinity +および新しいUVD3ビデオデコーダーのサポート。 ただし、アダプターは40-nmプロセステクノロジーを使用して製造されており、6700Mおよび6600Mシリーズカードと同様に、Whisterチップに基づいています。 実際、6650MはDDR3メモリを備えた6750Mアダプタです。



480ストリームプロセッサは、5000シリーズカードで使用されているものと同様の「古い」5Dアーキテクチャに基づいています。 したがって、シェーダーの数は、NVIDIA GPUの1次元シェーダーの数と比較できません。 新しいカードでのテッセレーションが改善されます（6800シリーズのデスクトップアダプターと同様）。 ストリーミングプロセッサは、DirectX 11、OpenGL 4.1、DirectCompute 11、およびOpenCLで使用できます。 また、ビデオトランスコーディングなどの一般的な計算の実行にも使用できます。



また、ビデオカードについても一言述べます。これは非常に優れたオプションですが、統合グラフィックカードとディスクリートカードはハイブリッドクロスファイアモードで動作します。 そして、さらにパフォーマンスが向上します！ 以下の「オウム」を見ることができます。
