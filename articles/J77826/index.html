Windows Mobile 6.5（Windows Phone）でのコールバック



2009年12月から、 Moroko PDAプログラムの新しいバージョンが利用可能になりました 。 これは、IPテレフォニーを使用してWMコミュニケーターまたはPDAからWindows Mobile 5-6.5と通信する便利な方法です。





このアプリケーションは、Betamax「ファミリー」のIPテレフォニーオペレーターのVoIPアカウントに使用されます。 その中には、12voip.com、ActionVoip.com、BudgetSip.com、CallEasy.com、Dialnow.com、Freecall.com、Internetcalls.com、Intervoip.com、Jumblo.com、Justvoip.com、Lowratevoip.com、Netappel.fr、 Nonoh.net、Poivy.com、Rynga.com、Sipdiscount.com、Siptraffic.com、SmartVoip.com、Smsdiscount.com、SMSListo.com、Sparvoip.de、Telbo.com、Voicetrading.com、Voipblast.com、Voipbuster。 com、Voipbusterpro.com、Voipcheap.com、Voipdiscount.com、VoipGain.com、VoipRaider.com、Voipstunt.com、VoipWise.com、Voipzoom.com、Webcalldirect.com これらのオペレーターのいずれかに登録が必要です。



プログラムの機能は次のとおりです。

  • 呼び出し順序
  • バランスチェック
  • SMS送信
  • 通話履歴
  • ノートブックへのアクセス
  • VoIPプロバイダーの複数のアカウントの使用。


この新しいバージョンでは、指の入力の方向と充電機能が追加されました。



アカウントの状態を操作できるように、アプリケーションウィンドウの上部隅に残高が表示されます。 さらに、通話中は、1分間の会話のコストが表示されます。



プログラムのデザインは大幅に変更されました。「指指向」インターフェースがアクティブになり、メニューが縮小され、アイコンが変更および拡大され、制御がシステムボタンに移されました。 最近の通話とSMSメッセージの履歴の形式が更新されました。



最も頻繁に使用される番号から電話をかけるのに便利なように、「通話」と「接続」機能が統合されました。



このプログラムは、Wi-FiとGPRSの両方で機能します。 同時に、インターネット接続は、通話要求（約1KB）を送信するためにのみ必要です。 次に、システムは最初の番号にコールバックし、次に2番目の番号にコールバックします。 このタイプの接続は、コールバックまたは「電話間」と呼ばれます。 通話は、発信者と通話の受信者に対して受信され、選択したVoIPオペレーターの料金でのみ課金されます。



すばらしい追加は、プログラムの拡張テストモードでした-1日あたり最大10人のロシア人加入者の接続とSMSメッセージの送信。 登録せずに試すことができます。 しかし、あなたがそれを好めば、 あなたはサイトに登録することができます



また、モロコPDAの新しいバージョンは、Windows Mobile 6.5プラットフォームの開発者向けのコンテストで発表されることを付け加えます。 この点で、クリエイターチームはWMDEV.ruコンテストでのMoroko PDAに対するフィードバック、提案、投票に満足しています。
