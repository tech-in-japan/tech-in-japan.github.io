Ghost：単なるブログプラットフォーム

Khabravchiansへの良い一日。 私のダイジェストの1つで、私はすでにゴーストについて言及しました：

私の知る限り、これはクラウドファンディングによって開発された最初のCMSです。 プロジェクトの作成者であるJohn O'Nolanは、すでにKickstarterで400,000ドルを調達しています。これは、必要な金額の785％です。 Ghostは、オープンソースプロジェクトであり、優れたデザインであり、非常に便利なレスポンシブインターフェイスです。



開発者は数時間前にGitHubにリポジトリを公開しました ...



Forbes、Wired、TechCrunch、Mashable、SmashingMagazine、Envato、その他の多くの人気のある西欧メディアでそれについて書いたからといって、尊敬されているHabrオーディエンスにプロジェクトを見てもらうことをお勧めします。 また、10時間で2,000歳。





Ghostを実行するには、 紳士用キット全体が必要です。

  • Node.js
  • SASSとバーボン
  • Phantomjs 1.9およびCasperjs 1.1
  • Python
  • うなり声




開始するには、プロジェクトフォルダーに移動して、次のように入力します。 

npm install --production
      
      

        
        
        
      

    
        
        
        
      
      

        
        
        
      

    
     


 npm start
