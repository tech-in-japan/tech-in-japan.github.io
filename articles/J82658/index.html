GnuPG：Win、Mac、および* nix用のセキュアメール

GnuPG（GNU Privacy Guard）は、OpenPGP標準（ RFC 2440 ）と互換性のあるオープンPGP実装です。 GnuPGでは、キーペア（デフォルトではRSA）を使用して、電子メッセージを暗号化、復号化、署名、および検証できます。 GnuPGの最も一般的な用途は、電子メールの暗号化とアップロードされたファイルの署名の検証です。 ただし、他のプロトコルで使用することもできます。たとえば、 PSI（Jabberクライアント）はGnuPGをサポートし、暗号化された会話をインラインで許可します。



3つのオペレーティングシステム（Windows、Mac OS X、Linux）のGnuPGのインストールと初期構成を検討してください。

読者は、OSコンソール（CMD / Terminal / xtermのそれぞれ）でいくつかのコマンドを実行するのに十分な知識を持っていることが理解されます。





ダウンロードする



Windows ： GPG4Win -GnuPG + GUI + Claws-Mailの既製のアセンブリ。

Mac OS X ： GPG Suiteをインストールします

Linux ： ディストリビューションによって異なります。 キーワードはgnupg2です。

* nixの下にはさまざまなクライアントがあるため、全員のインストールをスケジュールすることは役に立ちません。

GNUPGをサポートするクライアントのリストはこちらです。



設置



Windows ：標準インストール。 「Claws Mail」にチェックを入れることを忘れないでください。

Mac OS X ：GPGスイートのインストール。

Linux ： ディストリビューションとメールクライアントに応じて:) l。



キー作成





ターミナルを実行します（cmd.exe / Terminal / xterm）。

Windows OSの場合、cd "Folder_Where_installed_GPG4Win"（C：\ Program Files \ GNU \ GnuPG \デフォルトで）

全員の場合：gpg2 --gen-keyコマンドを実行します

いくつか質問があります。 

 必要なキーの種類を選択してください：
    （1）RSAおよびRSA（デフォルト）
    （2）DSAおよびElgamal
    （3）DSA（署名のみ）
    （4）RSA（署名のみ）


1を選択します。 



  RSAキーの長さは1024〜4096ビットです。
どのキーサイズが必要ですか？  （2048）


2048で十分です。 Enterキーを押すだけです。 



 キーの有効期間を指定してください。
          0 =キーは期限切れになりません
       <n> =キーはn日で期限切れになります
       <n> w =キーはn週間で期限切れ
       <n> m =キーはnか月で期限切れ
       <n> y =キーはn年で期限切れ


キーは何時以降に無効と見なされますか？ 「Enter」を押します:)それを永遠にします。



次に、「Y」をクリックします。 そして、3つの質問に答えます。



本名： -ラテン文字で名前を入力します（例：Vasily Pupkin）。

メール： - メールを入力します（例：v.pupkin@gmail.com）

コメント： -会社名からニックネームまで何でも）

すべてが正しい場合は、「O」を押します。

次に、キーのパスワードを尋ねられます。 同じ2回入力します（または、パスワードのないキーの場合はEnterキーを2回押します）。

数秒...そしてキーの準備ができました！ やった！ ）

確認：gpg2 --list-keys

pub 2048R / 5C35B909 2010-01-31

uid Vasily Pupkin <v.pupkin@gmail.com>

サブ2048R / 40ADD978 2010-01-31



使用する





Windows： Claws-Mailを起動します。 メールアカウント（キーが作成されたアカウント）を接続します。

設定モジュールに移動し、そこにGPGモジュールがあることを確認します。

ここで、独自のメッセージを作成し、[オプション]-> [暗号化]を選択します。

ところで、別のアカウントの新しいキーをすぐに作成できます-設定-アカウント設定-モジュール-GPG-新しいキーペアの生成

Mac OS X：メールを起動し、新しいメッセージを作成し、「署名済み」および「暗号化済み」チェックボックスを確認します。 署名/暗号化用のキーを選択できます。 10.6およびClawsメールについては、Windowsに似ています。

Linux：クライアントに依存。 Claws-Mailの場合はWindowsに似ています。
