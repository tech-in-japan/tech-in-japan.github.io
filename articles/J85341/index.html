Apple、Google、Microsoft：誰が繁栄するのか？

Apple、Google、Microsoftの3社は、ハードウェア、モバイルプラットフォームとコンピューターのオペレーティングシステム、ソフトウェア、エンターテイメント、インターネット検索など、さまざまな分野でさまざまなレベルで競い合い、支配のための三者間戦争にしっかりと結びついています。 今日、GoogleがWeb検索を引き継ぎ、Microsoftはオペレーティングシステムとアプリケーションの市場を支配しており、Appleは「ハイエンドハードウェア」、音楽とビデオの販売、メディアデバイス（iPod、iPhoneなど）。 もちろん、状況は変わるかもしれません。







Computerworldの編集者の1人は疑問に思った。言及された各企業は敵の大胆な部分を見ているので、近い将来に最もよく準備されているのは誰で、誰が戦いに負ける可能性が最も高いのか？



りんご



Appleが現在トップであり、状況が悪化しているという事実にもかかわらず、同社はこれら3つの巨人の中で最も脆弱です。 これは、Appleの成功が、一人の人であるSteve Jobsのユニークな洞察と才能に基づいていることに起因しています。



ほとんどの企業では、あるCEOを別のCEOに置き換えても実際には何も変わらず、以前のように会社は存在します。 Appleの場合、これは事実ではなく、AppleはJobsです。 1996年の終わりに彼が会社に戻る前に、Appleはむかついて生き延びようとしました。 彼は不採算なプロジェクト（たとえば、ニュートンなど）を閉じました。 彼の注意深い注意の下で、いくつかの成功したプロジェクトが開発されました。MacOS X、iPod、iPhone、iMac。 彼は会社を回復しました。 昨年、尊敬されているハーバードビジネスレビュー誌は、ジョブズのリーダーシップの下でAppleが市場価値を1500億ドル増やしたため、彼を世界で最も価値のあるシニアエグゼクティブとして認めました。



これはAppleにとって良いニュースでもあり悪いニュースでもあります。 Jobsが動作している間は、すべて問題ありません。 （Vitolからの注意：このフレーズは英語ではおかしく聞こえます： "Jobs is on job"）。 しかし、彼は永遠に会社を経営することはなく、近くにはビジョンと直感で彼に匹敵する人はいません。 Computerworldのプレストン・グララによると、彼の出発により、Appleは長くゆっくりとした衰退の段階に入るだろう。



Appleのビジネスモデルは、GoogleやMicrosoftのビジネスモデルとは異なり、ハリウッド映画スタジオのビジネスモデルと非常によく似ており、深刻な利益を得るには大ヒットが必要です。 Jobsが辞めると、そのようなヒットは表示されなくなります。



グーグル



これら3社のうち、他の企業よりも将来的に繁栄する可能性があります。 世界経済の経済成長エンジンの重要な要素の1つであるインターネット検索をほぼ独占しています。 これにより、Googleは新しい方向性を拡大し、資金を提供するための優れた基盤を提供します。



企業が製品を宣伝する際に自発的で非体系的なアプローチを示すこともありますが、Googleは急速に成長しているセクター、特にモバイルデバイス市場に着実に照準を合わせています。 この点で、Googleは今後数年間に予想されるモバイル広告ブームを十分に活用する準備ができています。 グーグルはオペレーティングシステムとアプリケーションの分野でMicrosoftと真剣に競争することはできないでしょうが、それでもこのビジネスで少なくとも強力なプレーヤーになるために他の分野で十分な収入を得るでしょう。



マイクロソフト



そして最後に、Microsoftがあります。これはAppleとGoogleの間に置くことができます。 Appleとは異なり、Microsoftは成長のためにメガヒットを必要としません。 デッドロックのあるオペレーティングシステムとオフィスアプリケーションの市場に参入し、優れた信頼性の高い企業ソリューションを提供することで、同社は絶えず成長します。 GoogleはMicrosoftのほぼ独占的な地位を破ることはできません。



Windows 7の成功は、新しいオペレーティングシステムのリリースが大きな利益をもたらすことを示しています。 Windows 7の急速な売り上げのおかげで、Microsoftはこの四半期の売上が前年全体よりも多くなりました。 一方、Microsoftは検索分野のリーダーからGoogleを追い払うことはできませんが、Bingはこの件で利益を上げる能力を実証しています。



そのため、今後はGoogleが全盛期を迎え、Microsoftはほぼ同じポジションに留まり、AppleはJobsの力を失ってしまうと想定できます。
