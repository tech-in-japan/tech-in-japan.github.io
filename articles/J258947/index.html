IDEプラグイン、GradleプロジェクトでのJavaコード分析、JavaScriptコードナビゲーションなどを含むUpsource 2.0がリリースされました

こんにちは、Habr！



1週間前、 Upsourceの新しいバージョンをリリースしました。これは、リポジトリの コードレビューとナビゲーションのためのツールです 。



最初のリリースから6か月間、機能を大幅に拡張することができ、作業の結果を共有できるようになりました。



まず、IntelliJプラットフォーム（特にIntelliJ IDEA、WebStorm、Android Studioなど）に基づいた開発環境用のIDEプラグインを作成しました。これにより、コードレビュー、コードへのコメント、プロジェクト内のニュースのフォロー、コードへのリンクの受信、将来的に同僚と交換できます。これはすべて、居心地の良いIDEを離れることなくできます。





人気のある需要により、 ブランチのサポートがSubversionに登場し 、GitとMercurialのタグも登場しました 。





Javaコードの静的解析とプロジェクト内のナビゲーションがGradleプロジェクトで機能するようになりましたが、以前はこれはすべてMavenを使用するプロジェクトでのみ利用可能でした。



さらに、Javaプロジェクトでは、隣接する2つのリビジョンでの Java文字の出現の比較が利用可能になりました。アップソースは、追加された出現数、削除された出現数、変更されなかった出現数を表示します。





コードレビューの作成と実施の段階は著しく簡素化されています。 特に、 レビュー担当者向けの自動ヒントがありました。レビューが何度か繰り返される場合、間違いなくリビジョンをフィルタリングする必要があります 。





最後に、 HTML / CSS / JavaScriptプロジェクト内の JavaScriptコードの発生のナビゲーションと検索が利用可能になりました。 これは、Javaコードだけでなく、IntelliJプラットフォームでサポートされている他の多くの言語でも表示される静的分析とナビゲーションの最初のステップです。 JavaScriptのサポートはまだ実験的であり、問​​題なく機能することはほとんど期待できませんが、次のリリースで改善する予定です。





Upsource 2.0の新機能の詳細については、 完全なチェンジログを確認するか、テクニカルエバンジェリストのTrisha Geeによる短いビデオショットをご覧ください。

価格といえば 。 Upsource 2.0は、最初のバージョンのプランのいずれかを購入した人には無料です。 さらに、Upsourceは最大10人のチームに対して常に無料であることを思い出してください。
