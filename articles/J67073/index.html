有効なHTMLドキュメント

そして、あなたは、本当に有効なHTMLドキュメントが<HTML>、<BODY>、<HEAD>タグを必要としないことを知っています。多くのタグはHTMLドキュメントの必須属性であると考えています。 そして、ここにドキュメントがあります： 

 <!DOCTYPE html> <title> HTML-</title>
<!DOCTYPE html> <title> HTML-</title>







バンパス検証付き。 <meta http-equiv = "Content-Type" content = "text / html; charset = utf-8”>これにより、警告が少なくなりますが、オプションです。

したがって、余分な文字を入力するのが面倒な場合、またはすべてのバイトを保存してユーザーの世話をし、ユーザーが消費するトラフィックの量を減らす場合は、次のHTMLドキュメントテンプレート（既にHTML5ドキュメント）を使用できます。 

 <!DOCTYPE html> <meta charset="utf-8"> <title> HTML-</title>
<!DOCTYPE html> <meta charset="utf-8"> <title> HTML-</title>
