マイクロソフトはソースコードを公開しますNET Framework 3.5

Microsoftは、.NET Framework 3.5ライブラリのソースコードを投稿することを決定しました。

.NET基本クラスライブラリ（システム、System.IO、System.Collections、System.Configuration、System.Threading、System.Net、System.Security、System.Runtime、System）のソースコード（ファイルコメントを含む）を開き始めました。テキストなど）、ASP.NET（System.Web）、Windowsフォーム（System.Windows.Forms）、ADO.NET（System.Data）、XML（System.Xml）、およびWPF（System.Windows）。 来月より多くのライブラリを公開します（WCF、Workflow、LINQを含む）。 ソースコードは、 Microsoft Reference License （MS-RL）の下で公開されます 。



.NET Frameworkライブラリのソースコードは、スタンドアロンインストールを介してダウンロードできます（これにより、任意のテキストエディターを使用してソースを表示できます）。 VS 2008でのデバッグもサポートされています。



VS 2008の最終リリースでは、Microsoftサーバーからデバッグ情報と.NET Frameworkソースを動的にロードする機能のデバッグをサポートしています。



オリジナルの英語ニュース： ironruby.blogspot.com/2007/10/microsoft-opens-doors-of-source-codes.html



VS 2008で言及されているデバッグのスクリーンショット： weblogs.asp.net/scottgu/archive/2007/10/03/releasing-the-source-code-for-the-net-framework-libraries.aspx
