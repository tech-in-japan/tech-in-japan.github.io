VDSのテスト。 行為II。

ちょうど1週間前に、新しいサービスVDSのテストの開始を発表しました！ テストが本格化しており、サービスが「コーミング」であり、新たな興味深い機会を獲得していることを嬉しく思います。 多くの人が、私たちが選んだOSをいつ選択できるかを尋ねました。



だから、私たちはあなたの注意に提示します







新しい配布



「テスター」は、VDSにCentOSだけでなくインストールできるようになりました。 リストには、次のディストリビューションが追加されました。



-Debian 4.0のエッチング

-Ubuntu Server 8.10



新しい「テスター」は、注文時に必要なオペレーティングシステムを選択できます。 このリストをさらに拡張する予定です。 途中、Fedora、OpenSUSE、Gentoo。



ジャバー会議



さらに、VDS専用のJabber会議を開きました！



Jabberクライアントを使用して誰でも会議に参加できます。 会議の参加者は、自分の考えや「発見」を互いに共有したり、新しいサービスのテストに関する特定の機会に相談したりできます。 Jabber-conferenceを使用すると、通常のチャットのように、参加者は「ライブ」で通信してインスタントメッセージを送信できます。 会議に接続するには、 手順を使用できます。



近い将来、VDSサービスを可能な限り便利にする多くの技術革新を導入する予定です。 また、私たちのテスターに​​感謝し、メールアドレスvds@sweb.ruに送られた彼らの提案や推奨に感謝します。 感謝します！
