KWallet統合用Firefoxアドオン

KDEとFirefoxのユーザーとして、保存されたFirefoxのパスワードはFirefox自体に保存され、KDEのパスワードはKWalletに保存されるという事実に常に不満を感じていました。 これに関連して、 統合の必要性に関するBugzilla @ Mozillaのバグにサインアップしました。

そして、救世主-アルゼンチンの開発者Guillermo Molinaが対応するアドオン-KDE Wallet password integrationを作成しました 。これにより、FirefoxのパスワードをKWalletに保存できます。



最初は、UTF以外のページからのASCII以外のデータが誤って保存され、アドオンの作成者に報告されたためにエラーが見つかりました。 彼の功績によると、このバグは24時間以内に修正され、他の問題は見つかりませんでした。すべて正常に機能します:)

addons.mozilla.orgの議論によると、アドオンは32ビットシステムと64ビットシステムの両方で動作するはずです（32があり、すべて正常に動作し、動作します）。



また、小さな警告をしたいと思います。 アドオンをインストールするときに、既に保存されているパスワードがkwalletに正しく書き込まれるかどうかわかりません。 Weaveを使用してパスワードを同期し、最初の更新で、すべてのデータがKWalletで受信および記録されました（ところで、Weave-KWalletのこのような二重同期には問題はありません）。 したがって、過労によって取得されたデータの損失を回避するために、引き続きバックアップすることをお勧めします。 アドオンはまだ実験段階です。



アドオンページに記載されている問題については、 登録を解除してください 。



PS誰かできれば、Fiery Foxブログに転送します



UPD： Murzは、KDE ​​Wallet Password Integration 2.0の新しい2番目のバージョンがFirefox Syncで正常に動作し始めたと報告しているため、バージョン1.xのようにパスワードが同期されなくなることを恐れずにこのプラグインを安全に使用できます。
