マカフィーは数百万台のクリーンなコンピューターを無限に再起動します

本日リリースされたMcAfeeアンチウイルスのアップデート（McAfee DAT 5958）は、svchost.exe（サービス）を考慮したWindows XP SP3での無限の再起動をもたらします。



PricewaterhouseCoopers New York （数十万台のコンピューターのみ）、 大学のキャンパス 、Windows用の医療ソフトウェアを使用している病院 、および世界中のその他の多くの企業が無限に再起動しました。



ブラックユーモアですが、McafeeはWindowsをウイルスとして公式に認識した最初のアンチウイルスです。



アンチウイルスの「治療」：



オプション1：セーフモードで再起動し、mcshield.exeの名前を変更し、再起動し、ウイルスコンソールを実行し、[ツール]-> [DATのロールバック]、mcshieldの名前を変更して、再起動します。



オプション2：名前を変更する代わりに-レジストリHKEY_LOCAL_MACHINE \ SYSTEM \ CurrentControlSet \ services \ McShield \ Start = 4で、同じ方法でロールバックします。



McAffeeからの公式な修正が期待されています。 公式の修正 （4行必要なときに5ページを書くのが面倒ではなかったのですか？以下を参照）...更新5959は既にレイアウトされており、数時間以内に拡散して修正することを約束します。 ただし、無限の再起動ではおそらく更新が許可されないため、手で治療する必要があります。



公式の「治療」の5ページの簡単な改定：

  1. extra.datをダウンロードしてください 。
  2. セーフモードで再起動します。
  3. extra.datをc：/プログラムファイル/共通ファイル/ mcafee /エンジンにコピーします。
  4. 再起動
