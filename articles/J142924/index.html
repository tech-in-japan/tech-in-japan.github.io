ITmozg and Darth Vader Production Present-ビットバイト！

5月19日、ITmozgは、サンクトペテルブルクで最大のITプロフェッショナル向けイベント-Bit Byteを開催します。







Bit Byteは、IT業界の専門能力開発の祭典であり、技術専門の学生と経験のある専門家の両方が、主要なIT企業の代表者や成功したスタートアップとコミュニケーションを取り、ITスターのパフォーマンスを聞くことができます。



プログラムを2つの部分に分けることにしました。これは会議であり、誰でも直接会社の代表者とコミュニケーションできる会社です。



1.専門家によるプレゼンテーション

専門家は、ケーススタディ、IT分野の労働市場の分析、ソフトスキル分野の推奨事項を共有し、技術的な問題の解決について話します。 すべてのパフォーマンスは、次の3つのセクションで開催されます。



•ITセクション。

•スタートアップセクション。

•HRセクション。



1）技術ITセクション。

ここで話をします

  • REST API
  • 高負荷
  • Livejournalが「ソフトウェア開発の落とし穴」について語る


2）スタートアップセクション

ここでは、スタートアップ企業の開発、投資の誘致、製品開発のすべての複雑さについて説明します。



2）人事セクション

人事部門の専門家が、キャリア構築の複雑さ、インタビューの準備と合格方法について話します。



Mail.ru Group、Fotostrana.ru、Oktogo、Livejournal、Sup MediaのOdnoklassnikiプロジェクトの専門家がそれぞれの経験を共有します。 アレナ・ポポワとレオニード・ブガエフ 、 イリヤ・ドロノフ、ロシアのLiveJournalのヘッドは、特別にモスクワからイベントに来ます。

また、サンクトペテルブルクからは、Yandex.Moneyの人事、ビジネスコーチ、コンサルタントであるAlexander Orlov aka Happy PMのセレクションと開発の責任者であるAlena Novoselova氏もおもしろいゲストです。

すべての企業の人事担当者と個人的に連絡を取ることができます。



2.国および都市の最高の企業の求人： EMC、Epam、Mail.ru Group、Embria、Livejournal、Motorola Mobility、Ascreen、「セキュリティコード」、「Institute of Network Technologies」など。







一日中、会社のスタンドは展示エリアで働き、訪問者は会社の代表者とチャットしたり、履歴書を残したり、インターンシップや空席についての情報を見つけたりすることができます。



プレゼンテーションのトピックを含む詳細なプログラムはこちらから入手できます。





3.スタートアップへの注意



このフェスティバルには、成功した都市のスタートアップが展示される展示スタートアップゾーンと、別のプログラムパートがあります。 講演者は、スタートアップのスタッフ募集の特徴、投資の獲得、新しい会社の促進について話します。



あなたがBit Byteの紹介に興味のあるスタートアップの代表者である場合は、お問い合わせください 。
