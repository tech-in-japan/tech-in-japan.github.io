Visual Studio 2.0用のPythonツールがリリースされました

Python開発ツールの新しいバージョンは、Visual Studioからダウンロードできます。 Python Tools for Visual Studioはオープンソースであり、次の機能を提供します：CPythonサポート、IronPython、機能コードエディター、Intellisense、デバッグ、プロファイリング、HPCクラスター、IPython、Django、およびWindows、Linux、およびMacOS用のクラウドベースのクライアントライブラリ。







Visual Studio 2.0 AlphaのPythonツールには、Pythonのコード分析とインテリセンスの改善、PythonのvirtualEnviroment、Linux / OSXからのリモートデバッグ、スクリプト機能としてのデバッグ（クイックデバッグ用）などの重要なイノベーションが含まれています。
