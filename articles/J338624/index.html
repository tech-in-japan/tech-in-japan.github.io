免責事項 ：テーマのスレッドとハブに従って、選択をいくつかの部分に分割することにしました。 数学、アルゴリズム、プログラミング言語に関するブログから始め、今後のリリースではウェブ開発とデザインに移ります。

数学とアルゴリズム





  • ビットプレーヤー


ブログの著者であるブライアン・ヘイズは自分自身を幸運だと呼び 、彼は数学とコンピューターサイエンスの研究に人生を捧げることができました。 彼のブログでは、著者は自分の経験（350以上の投稿）を共有し、生物学と物理学についても書いており、問題を解決し、画像を扱った経験を共有しています。 1973年から1984年まで、ブライアンは1845年以来出版された、米国で最も古い人気のある科学雑誌の1つであるScientific Americanに記事を書きました。 ブライアンの出版物は、コンピューター言語とアメリカの科学者に掲載されています。 2005年に、ブログの著者は、産業施設に関する本「 インフラストラクチャ：産業景観へのガイド」を出版し、2017年の秋には、彼の新しい本「Foolproof、and Mathematical Meditations」がリリースされます。





  • Mathr


ブログ投稿のほとんどは、数学とコンピューターサイエンスに専念しています。 ただし、言及された科学に加えて、著者はグラフィックスにも興味があり（たとえば、蝶から飾りを描く方法を教えています）、針仕事の経験を共有しています（たとえば、Tシャツにクラゲを描くプロセスについて説明しています）。 最近では、著者は複雑な生物学的図面のモデルに基づいたグラフィック画像を含む本「 形態形成 」を出版しました 。 本の白黒画像は、生成された1000個のバリアントから手動で選択されました。





  • ジョン・D・クック


応用数学、統計、およびソフトウェア開発の分野の専門家は、たとえばラプラス分布やパスカルトライアングルなど、シンプルで理解しやすい言語で話します。 ジョンは、 オンコロジーセンターのソフトウェア開発者として、これらの分野の問題を解決した経験を応用することができました。 M. D.アンダーソン 。 著者の成果は、 記事 、 プレゼンテーション、そしてもちろんブログ投稿の形式で入手できます。 現在、ジョンはアメリカ全土の企業にコンサルティングサービスを提供し、遭遇した最も興味深い問題、タスク、および開発について定期的に執筆しています。





  • ケビン・ナドソン


上記のブログの著者は、Kevin Knudsonのブログをすべての人にお勧めします。 また、ケビンは長い間ブログを更新していませんが（最後の公開は2015年に遡ります）、彼は「 My favourite theorem 」と呼ばれるプロジェクトを成功裏に実装しています。 リスニングにおすすめ。





  • 離散分析


上記のほとんどすべてのブロガーは、離散分析を読むことを推奨しています。 このジャーナルの著者は、数学に関する他の出版物とはまったく異なると主張しています。 彼らは、素材の持続的なテーマの焦点と便利なプレゼンテーションに違いがあると考えています。





  • 無限の近所


量子力学や高等数学の分野で知識を広げたい場合、ダンピポニがお手伝いします。 彼のブログでは、著者は退屈な説明から逃れようとし、通常の生活の例を使用して、図でそれらを補強しています。 著者自身がHaskellの例を解析しますが、線形代数、論理、および他の多くのトピックに関する資料は、他の言語で作業していても読むのが面白いです。 Google+または彼のTwitter の著者のプロフィールで、著者の人生について調べたり、記事への興味深いリンクのある投稿を見つけることができます（たとえば、P値を使用して仮説をテストする）。

プログラミング言語





  • NSBlog


マイクアッシュが2005年以来リードしているブログは、特にSwiftで執筆している人にアピールします。 ブログには、テーマ別ワークショップのガイド、バグのレビュー、メモがあります。 マイクは定期的に読者の質問に答え、毎週のQ＆A投稿の本全体をまとめました。 The Complete Friday Q＆A：Volume Iと呼ばれます。





  • エリ・ベンダースキー


Eli BenderskiによるPython、Haskell、Clojureのチュートリアル。 このプロジェクトは2003年に個人的なブログとして始まり、技術雑誌に成長しました。その助けにより、著者は自分の知識を「汲み上げ」、オープンプロジェクトを共有します。





  • スティーブンディール


詳細な説明とコード例を含むHaskellの問題解決の例については、Steven Dealのブログ投稿を参照してください。 さらに、著者は言語の機能に没頭するために個人的に検証されたリソースを提供します。 また、ブログにはあまり多くの投稿はありませんが、それぞれが詳細に注意して書かれています。





  • イカのブログ


ソフトウェア開発者のKarin Meyerは、Clojure、人工知能、ロボット工学について書いています。 ちなみに、最近では2017年5月にブログの著者が本「 Code Shifter 」を出版し、娘と一緒に執筆しました。 この本は、ワンタッチでコードを変更できる12歳の少女、エリザの冒険について説明しています。 これらの冒険の間に、エリザは彼女の友人とプログラムすることを学び、彼女の強みを学びます。





  • 他の何と同じくらい悪い


Learn You Some Erlang for Great Good！ Bookの著者であるFred Herbertは、ブログで間違いを解析し、Erlangでガイドを共有しています。





  • Bartekのコーディングブログ


ポーランドの開発者Bartolomei Filipek（Bartlomiej Filipek、別名Bartek）のブログは、C ++に関する定期的に更新されるリソースを探している人にとって興味深いものになります。 Bartekは14歳のときにコードを書き始めたため、ソフトウェア開発からゲーム開発までさまざまな分野でかなりの経験を積んでいます。 ブログで最適化の投稿、実用的なヒント、最新の標準などを見つけることができます。 たとえば、Bartekによるこの記事は、2016年1月に英語のテーマサイトで最高の1つとして認められました。





  • Fluent C ++


ブログの名前はそれ自体を物語っています。 内部には、クリーンなC ++コードを作成するためのガイドがあります。 ブログの著者であるJonathan Boccaraは、C ++プログラミングと従業員のトレーニング方法に関する5年間の経験を共有しています 。





  • 存在タイプ


カーネギーメロン大学の計算機およびシステムの理論と実践学科の教授であるロバート・ハーパーのブログは、プログラミング言語の理論に没頭したい人にとって興味深いものです。 特に、 SMLおよびグリッドコンピューティングに関する多くの情報を見つけることができます。 Harper教授は、標準MLの定義やプログラミング言語の実践的基盤など、いくつかの本の著者でもあります。





  • 青みがかったコーダー


Chris Dahlのブログでは、Pony、 Wodka、 Haskellなどのさまざまな言語を知ることができます。 これに加えて、Chrisは彼のブログで、OS、ブラウザー、およびネットワーク技術を扱っています。 通常、彼はオブジェクトの詳細な分析を行い、コードと説明を添えて、有用なリソースへのリンクで意見を補強します。 ブログには、12年にわたって蓄積された約300の出版物があります。





  • 内側245-5D


このブログの名前はInside 245sからスムーズに発展しました。 それ以前は、Inside T5、Inside P4、Inside 1712B、Inside 2214、Inside 233、Inside 374、Inside 206-105、Inside 214-1E、Inside 245-5Dと呼ばれていました。 このブログの著者であるEdward Z. Yangは、彼と共に進化しました。 彼は現在、スタンフォード大学の大学院生であり、いくつかの出版物の著者です。 ブロガーはHaskellに最も興味を持っています（たとえば、ブログにはHaskellの作業について図解された紹介があります）。





  • ルビーランド


「Rubyの国」では、最も若くて最も人気のあるプログラミング言語の1つに関する最新のニュース、チュートリアル、専門家の意見をご覧いただけます。 リソースは1日に数回更新されます-これはRubyに関する最高のテーマリボンの1つです。 プロジェクトの作者であるJonathan Rochkindは、RubyのブログとWeb開発に定期的に投稿を投稿しています。また、RubylandのアップデートはTwitterでも公開されています。
