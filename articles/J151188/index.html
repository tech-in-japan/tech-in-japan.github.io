Windows Phone 8上のNokia Maps

新しいNokia Lumia 920およびNokia Lumia 820スマートフォンでのカメラとワイヤレス充電、およびその後の従来の手法について議論した後、これらのデバイスの最も興味深い部分の1つは「ナビゲーション」と呼ばれます。 この欠点を補い、Windows Phone 8でのNokia Mapsの主な変更点と追加点をリストします。







これはピノ・ボネッティによるわずかに改訂された記事で、その元はここにあります 。



オフラインマップのサポート







従来、オフラインマップは、SymbianやMeeGoなどの他のプラットフォーム上の他のNokiaスマートフォンとともに、WP8でサポートされています。 オフラインマップを好む人は、インターネット接続にお金を費やす必要がないため、動作が速く、安価であることに注意してください。 これらの利点に加えて、WP8のオフラインおよびオンラインマップは、Nokia Location Platformと呼ばれる対応するAPIエンジンを介して、他のサードパーティアプリケーションでも使用できます。 このエンジンは、最近までJS（Web）、JME（S40）、Qt（Symbian＆MeeGo）のAPIとして、またRESTサービスの形式で存在していましたが、Windows Phone 8でプラットフォームの機能として利用できます。ハイブリッドフォーム-オンラインとオフライン。



Nokia Lumia 920および820の特徴は、 スタンドアロンの A-GPSおよびA-GLONASSレシーバーを備えていることです。 GPSを搭載した既存のNokia携帯電話は、Nokia MapsのオフラインモードがA-GPSを無効にしているため、デバイスが現在の場所を見つけるまで待機するのにかなり時間がかかります。 現在、Lumia 920および820は、A-GPSおよびオフラインを含む通常の操作を約束します。つまり、場所は数秒以内になります。



カードがオフラインモードとオンラインモードを切り替える場合、オンラインで受信したオフラインマップへのすべての変更は、このモードがオフになった後もキャッシュに残ることを知らない人のために追加する必要があります。



音声プロンプトによる歩行者ナビゲーション







これはSymbianとMeeGoの機能ですが、Windows Phoneプラットフォームではすぐには実行されませんでした。 そのため、Nokia Mapsには、自動車、公共交通機関、歩行者の3種類のナビゲーションがあります。 従来、自動ナビゲーションは、Nokia Driveと呼ばれる別のアプリケーションで行われていました。 公共交通機関を使用したナビゲーション-別のアプリケーションNokia Transport。 そのため、Windows Phone 8のNokia Mapsには歩行者用のナビゲーションもあり、ルートをたどりながらキーポイント、ターン、道路名を音声で読み上げました。



また、歩行者のナビゲーションは、公園、小道、歩道、階段など、歩行者のみがアクセスできるエリアを含む歩行ルートを敷くような方法で行われることに注意する価値があります。



拡張現実







拡張現実または拡張現実。これは、NokiaがNokia Mapsの追加機能としてかなり長い間計画していたことです。 写真でわかるように、拡張現実エンジンはカメラ画像にマップマークをオーバーレイします。 この方向またはこのターゲットまたはアトラクションが配置されている方向を画面に表示します。 歩行者ナビゲーションと合わせて、これは非常に便利な機能のように見えます。特になじみのない都市をさまよう場合に便利です。 この場合、コンパスもケースに含まれているため、前後に走る必要はなく、その場でスピンして画面を見ることができます。



Windows Phone 7の所有者は、拡張現実を別のアプリケーションNokia City Lensとして利用できます。こちらからダウンロードできます



屋内ナビゲーション





Nokia Labsの部門で長らく待ち望まれていたもう1つの機能。 世界では、屋内測位、屋内ナビゲーションとして知られており、最終的に会場マップと呼ばれていました。 空港、ショッピングセンター、ビジネスセンター、スタジアム、地下鉄など、GPSの受信が困難または不可能な大きな屋内エリアで非常に役立ちます。 この機能を使用するには、目的の建物内に無線インフラストラクチャが存在する必要があります。 少なくとも、既存のWiFiネットワークに基づいてインフラストラクチャを作成できます。 せいぜい、特別なWiFiやBluetoothビーコンをインストールする必要があります。



現在、Nokia Mapsは38か国の4,600を超える部屋でのナビゲーションをサポートしており、毎月新しい部屋が追加されています。 繰り返しますが、屋内ナビゲーションは、Nokia Location Platformの一部として外部アプリケーションで使用できます。



私の通勤







これは、この比較的新しい機能が新しいNokia Lumiaにも継承されていることを思い出させるためです。 これは、たとえば自宅から職場まで2回以上運転した場合、常に使用しているルートが自動的に記録されることにあります。 図のように、ルートを手動で設定することもできます。 そのようなルートの場合、必要な時間の1時間前のMy Commuteテクノロジーは、交通渋滞や道路上のその他の状況を考慮して移動時間を計算し、現在の状況を考慮して最適な代替手段を見つけようとします。



Windows Phone 8のリリースにより、このチップがドイツ、ロシア、英国、ブラジルで数か月間利用可能になることが正式に発表されました。 ロシアに関しては、ロシアのすべてがこのビジネスに該当するわけではないという推測がありますが、何が起こるかはわかります。
