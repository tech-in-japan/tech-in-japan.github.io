実例に基づく意思決定とリスク管理システム

この出版物では、企業だけでなく日常生活でも意思決定およびリスク管理システムを使用する実用的な例を紹介します。 私はこのアプローチを個人的に使用し、そのパフォーマンスを確認しました。 少なくとも心理的なレベルでは、幸運なことに、記載されているリスクはどれもうまくいきませんでした。



ウィッシュカード



2016年の夏、妻と私は夏休みの場所について論争しました。 その時までに、「休暇2016」貯金箱には12万ルーブルがありました。 デッキチェアに横たわって2週間はこれで十分でしたが、事実、受動的な観光は私たちを引き付けません。 ウィッシュカードは次のようになりました。



  • 21日以上の期間
  • 最大12万ルーブル 二人用
  • 休暇中の最大のモビリティ （車両とルートの可用性）


昨年の休暇オプションは、標準的な複合フライト、レンタカー、2〜3千kmのルートでした。









前者の場合、切り下げられていない別のルーブル（45ルーブル/ユーロ）が役立ち、後者の場合、マイルのチケットを購入しました。 今回はボーナスがありませんでした。

SWOT分析



2つのバージョンでは、私にとって重要な側面を太字で強調しています。



オプション「Gelendzhik」

強み









  • ヨーロッパの価格と比較して安価なガソリン（価格は平均的なヨーロッパの2倍です）
  • 安い宿泊施設
  • 野外活動のための手頃な高山
  • 北コーカサスの豊かな自然
  • 乗馬やATVなどの手頃な価格の屋外アクティビティ。
  • ソチで開発されたインフラストラクチャ：クラスナヤポリヤナ、オリンピック会場、公園への訪問。


弱点









  • 汚れた海と汚れたビーチ （ガラス瓶、タバコの吸い殻、桃の骨など）:(
  • ビーチインフラストラクチャの欠如：更衣室、トイレ、シャワー。
  • M-4高速道路の渋滞時間（ヴォロネジ、クラスノダール、ロストフオンドン、有料道路区間の料金徴収ポイント付近）
  • クラスノダールからソチのリゾート地への近代的な道路の欠如 （ロシアのリゾート地開発の大きなブレーキと考えています）
  • 海までの大きな距離


特徴









  • グロズヌイ市への観光訪問
  • エルブルスのふもとへの観光客の訪問


リスク









  • 事故や故障が発生した場合の自車の高価な避難（最大2000 km）
  • 低品質の医療サービス




オプション「Zadar」

強み









  • 澄んだ海とビーチ
  • 岩の多い海岸、砂、松を組み合わせた珍しい自然
  • 道路インフラを備えた近代的な道路（ポーランドを除く）
  • 多くの外国の都市や国々を訪問し、地元住民とコミュニケーションを取ります
  • 他の文化を学ぶ
  • 高品質のサービス


弱点









  • クロアチアのエンターテイメント、野外活動、レストランの高価格
  • モスクワ-ポーランドの長い旅（初日と最終日で1300 km）
  • 運転中の困難（外国の都市の方向、道路標識の解読）
  • 高いガス価格
  • 安いホテルの選択（予算を満たすため）
  • 「これまで」「外国」という親Relativeの経験


特徴









  • プリトヴィッツェ湖群への観光訪問


リスク









  • 事故や故障が発生した場合の自車の高価な避難（最も遠い距離で最大2500 km）
  • 車が故障した場合の車の修理工とのコミュニケーションの難しさ、その結果、不必要な修理のための「離婚」
  • 異なる通貨（フォリント、クン、ズロティー、クローン、ユーロ）による予算の不均等な支出（レストランの初期にはヨーグルト入りパンが可能）
  • 保険事故による保険金の支払いに対する保険会社による費用のかかる治療または拒否
  • 事故発生時の法的リスク（外国語で無罪を証明することの難しさ）
  • 道路罰金




SWOT分析の後、ザダルの旅行オプションには、私にとっては魅力的ではあるものの、より多くのリスクが含まれているという結論に達しました。



このようなリスクの説明とそれらへの影響の検索により、合理的な答えが存在することで、自分自身や愛する人を安心させることができます。 最も重大なリスクは、深刻な車の故障と医療処置の必要性です。 両方のリスクは、同じ確率とコストでロシアと海外の両方で発生する可能性があります（ 推測 ）。 しかし、虫垂炎の炎症の場合、「虫垂炎の非炎症」オプションを選択できない場合、私はヨーロッパの診療所を選択するように思われます。

意思決定









たとえば、美しい写真を見てすぐに行動に移ることに感心することができます。 決定の幼児主義は致命的です。 企業での時期尚早な行動を防ぐことが特に重要です。 複雑な決定には、人のグループ（取締役会）を含む詳細な調査が必要です。 新しい1C：ERPシステムを起動するか、新しい機器を購入しますか？ 組織内のビジネスプロセスのリエンジニアリング（リエンジニアリング）または最適化を実行するには？ 新しい組織サイトを立ち上げますか？



私は毎日1週間決断を下し、自分の決断の印をつけて、気分の影響を排除しました。







おわりに



私の例はそれほど大規模ではありませんが、その上で意思決定への実用的なアプローチを示します。 一般的に、次のようになります。



  1. 技術要件を取得する
  2. 利用可能なオプションを調べる
  3. リスクとそれらに影響を及ぼす可能性を説明する
  4. 季節の変動や気分を考慮せずに決定を下す
