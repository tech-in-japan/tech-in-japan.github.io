ASUS N750JVマルチメディアノートブックを確認する

ほんの1か月前、IntelはHaswellと呼ばれる新しい第4世代Intel Coreプロセッサを一般に紹介しました。 また、OMPUTEX2013では、Intelの最新のプラットフォームで実行される最初のラップトップが示されました。 今日のレビューのヒーローはASUS N750JVです。 インデックスNは、このデバイスがトップエンド構成、高品質のサウンドシステム、スタイリッシュなデザインを備えた一連のマルチメディアラップトップに属していることを示唆しています。 名前の7という数字は、17インチ画面のラップトップを持っていることを意味します。 サイズからわかるように、これはデスクトップ交換デバイスです。 この面白いラップトップを詳しく見てみましょう。





技術仕様



プロセッサー： Intel®Core™i7-4700HQ（2.40 GHz、6 MBキャッシュ）

チップセット： Intel®HM87

メモリ： 16 GB DDR3 1600 MHz SO-DIMM

ディスプレイ： 17.3インチ（43.9 cm）アンチグレアLCD（16：9、フルHD、1920 x 1080）、LEDバックライト

ビデオカード： NVIDIA®GeForce®GT 750M（4 GB）およびIntel HD 4600

ハードディスク： 1）SSD SanDisk 24GB、2）HDD HGST 1000GB 5400 rpm 3）ST750LM022 750GB 5400 rpm

ドライブ： Pioneer 6x BluRay Read Combo

ネットワーク： Realtek 10/100/1000 Mbps Atheros WLAN 802.11 a / b / g / n、 WiDiサポート

インターフェース：カードリーダー（SD / MMC / MS / MS Pro）、HDMI、miniDisplayPort、ヘッドフォン、マイク、4x USB 3.0、Bluetooth 4.0

ウェブカメラ： HD

バッテリー：リチウムイオン6セル、6260 mAh、69 Wh

寸法： 41.5 x 27.5 x 2.99 mm

重さ： 3.2 kg



パッケージバンドル



ラップトップが入っている箱のサイズは驚くべきものです。 タブレットやウルトラブックの後、これらの寸法は大きすぎるように見えますが、すべてのラップトップがかさばるパッケージに入っていました。





箱は通常のASUSスタイルで装飾されています。ラップトップの画像を使用したマット印刷です。





内部のほぼ全域がラップトップ自体で占められ、布製の封筒に入れられています。









右側には、充電器とリモートサブウーファーの小さなニッチがあります。





つや消し仕上げと光沢のある端部を持つ充電器は、印象的な寸法を持っています。 19Vの電圧と6.32Aの電流は120ワットです！





外観



昨年のモデルとは異なり、ラップトップのデザインは変更されています。 ASUS N750JVの印象的なサイズと重量は3.2 kgです。 ラップトップのデザインでは、2色が交互に使用されます。カバーと底面は暗褐色で、ケース自体は銀色です。 ボディデザインは完全に金属です。



カバーはブラシ仕上げのアルミニウムでできており、指紋の一部は残っていますが、あまり目立たず、簡単に消去できます。 中央に刻まれたASUSロゴは、中央に刻まれています。









ラップトップの底面は金属製です。





中央には換気グリルがあり、前部にはスピーカー用の穴があります。





別の換気グリルがケースとスクリーンの間にあります。





取り付け設計は簡単です。周囲のネジを外すだけで、ラップトップの内側にアクセスできます。

端のケースの厚さはわずかに薄いため、ラップトップをテーブルから取り出すのに便利で、ラップトップを視覚的に薄くします。 ボディの周囲のエッジは、小さな面取りによって滑らかにされています。





スクリーンは1つの大きなヒンジを使用して取り付けられており、開き角は120度です。 ヒンジにはタイトなストロークがあり、スクリーンを任意の位置にロックできます。 閉じたとき、ラップトップの蓋は磁石の上にあります。 エンジニアは作業量を完全に計算し、ラップトップの蓋はケースを持たずに片手で静かに開きます。



インターフェース



メインコネクタはラップトップの左側に配置されています。 最初は電源コネクタです。 次に、ギガビットRJ45をネットワークに接続します。ヒンジ付きの蓋があり、コネクタのサイズを小さくすることができます。 次に、モニターとテレビを接続するためのHDMIおよびminiDPコネクタ、オフ状態での充電をサポートする2つのUSB 3.0、およびSDXCカードリーダー（SD / MMC / MS / MS Pro）があります。





フロントエンドには、美しい白色のLEDインジケータがあります。





右端にはケンジントン城があります。 付属のサブウーファーの出力は2.5 mm。 ブルーレイコンボドライブだけでなく、映画を見るのに便利です。





次に、さらに2つのUSB 3.0コネクタ、マイク入力、S / PDIFをサポートするヘッドフォンジャックがあります。





音



Nシリーズのラップトップの大きな利点の1つは、高品質のサウンドです。 ASUS N750JVも例外ではなく、優れたサウンドサブシステムを受け取りました。 伝統的に、このラップトップは、ハイエンドのオーディオおよびビデオ機器の製造を専門とするデンマークの企業であるBang＆Olufsenによって認定されています。





ASUSの専門家が開発したSonicMaster Premiumテクノロジーを実装します。これは、音質を改善するためのハードウェアおよびソフトウェアツールのセットです。 特別な「ハイライト」は外部サブウーファーで、高品質の低周波伝送に役立ちます。













このバンドルにはWaves MAXXAudioソフトウェアも含まれています。これにより、サウンドを微調整できますが、個人的にはあまり意味がありません。 音の変化はわずかです。





4個のスピーカーは、ケースの前面のミシン目の下に隠れています。 一般的に、ラップトップの音は優れています。





最大のボリュームは心地よく驚きますが、品質はまったく低下しません。 現時点では、これは私が出会った中で最高の音質を備えたラップトップです。



スクリーンとカメラ



画面の対角線は17.3インチです。 15インチバージョンのN550JVも販売されることに注意してください。 最初は、画面サイズが異常に大きいように見えましたが、すぐに使い慣れます。 おそらくこれは、家庭やオフィスに最適な画面サイズです。 解像度は1920 x 1080ピクセル、1インチあたりのドット数は130です。残念ながら、IPSマトリックスを使用していませんが、公平に、IPSマトリックスを搭載した17インチラップトップは販売されていません。 このスクリーンはEMWテクノロジーを採用しており、視野角を水平方向に150度、垂直方向に120度にします。





スクリーンの重要な利点の1つは、マットな反射防止コーティングです。 画面の上には、解像度2メガピクセルのHDカメラがあり、その隣には白い作業インジケーターがあります。 カメラの端に沿って、ノイズが低減された2つのマイクがあります。





入力機器



私たちの前には、別のデジタルセクションを持つおなじみの島のフルサイズキーボードがあります。 キーはプラスチック製で、ケース自体とまったく同じ「アルミニウム」色です。





キーボードにはキーの標準セットがあります。 一方で、そのようなミニマリズムとシンプルさは目を楽しませ、他方では、サウンドの調整やメディアプレーヤーの制御など、さまざまな追加キーを好むユーザーがいます。





一般的に、キーボードは快適です。大きなリストパッドのおかげで、タイピングが便利です。 バックライトを備えたキーのストロークは小さく、触覚フィードバックがあります。





キーのすべてのキャラクターはレーザー彫刻されており、暗闇で光っています。





バックライトをオフにすることができ、明るさの3つのレベルがあります。





キーボードの下には印象的なタッチパッドがあります。 パネルはケースとまったく同じ色で、全体的なデザインに完全に適合します。 タッチパッドのサイズは10.5 x 7.5 cmで、触ると表面が粗く、快適です。





タッチパッドの全領域を「マウスの左ボタン」でクリックすることができます。 右下の部分では、「右クリック」と2本の指で「クリック」を行うことができます。 当然、MultiTouchおよびすべてのWindows 8ジェスチャがサポートされています：起動、アプリケーションの切り替え、スワイプなど。

電源ボタンには白いバックライトが付いています。





一方、対称的に、アプリケーションのクイック起動を割り当てることができるボタンがあります。





性能



ASUS N750JVは、Intelの新しいHaswellプロセッサで動作する最初のラップトップの1つです。 Intelのtick-to-tack戦略によると、Haswellプロセッサはタックです。 このようなプロセッサは、パフォーマンスが前世代と著しく異なります。 現在の第4世代Intel Coreは、ビデオの視聴時に最大50％電力を消費します。 一般的に、ラップトップの速度は満足です。 ハイブリッドテレロジーHDD + SDDテクノロジーの使用のおかげで、ラップトップは非常に素早くオンになります。 オフ状態からのフルブートにかかる時間は10秒未満です。 スリープモードを約2秒間終了します。スリープモードでは、ラップトップは1回のバッテリー充電で1か月間使用できます。 高負荷になると、クーラーが作動し始めます。 あなたはそれを静かに呼ぶことはできません、結局のところ、ラップトップには強力なプロセッサとビデオカードが装備されています。 キーボードの上のラップトップの上部のみを加熱します。 熱い空気が後ろから吹き出されます。

























4GBのメモリを搭載したディスクリートグラフィックカードNVIDIA GeForce GT 750Mは、 優れたゲームパフォーマンスを発揮します。





新しい統合グラフィックスカードは、Ivy Bridgeと比較してパフォーマンスが2倍に向上しています。 Intelの組み込みビデオに初めてIrisという名前が付けられました。





合計で、システムには3つのドライブ、2つのHDD、1つのSSDがあります。













CrystalDiskMarkを使用してHDDドライブの速度をテストします。









最後に、Windowsのパフォーマンス評価。





バッテリーとランタイム



内蔵バッテリーの容量は6260 mAhです。 フル充電には2.5時間かかります。 ラップトップのコンパクトな設計のため、取り外し可能なバッテリーを捨てなければなりませんでした。

中輝度でFullHDビデオを視聴した場合、ラップトップは4時間続きました。

最小限の画面の明るさとインターネットサーフィンで、動作時間は6時間でした。

ネットワークをオフにし、最小輝度を設定し、省エネプロファイルをオンにして、オフィスアプリケーションでの作業を制限すると、最大7時間に到達できます。





おわりに



ASUSの新しいラップトップは、自宅でもオフィスでも優れたアシスタントになります。 プロセッサとビデオカードの高性能により、ホームコンピューターが置き換えられ、最新のゲームをプレイできるようになります。 ブルーレイドライブと優れたサウンドに加えて、大型で高品質の画面が存在するため、コンパクトなマルチメディアセンターをほぼどこにでも配置できます。 オフィスでこのラップトップを使用すると便利です。反射防止層は、反射する不要な光源を消します。

ラップトップは間もなく発売され、予想価格は55,000 pです。
