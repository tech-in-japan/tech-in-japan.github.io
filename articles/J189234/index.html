約10時間の実行時間を持つ太陽電池を搭載したラップトップを発表









Canadian WeWi Telecommunications Incの子会社である同名の会社が製造する発表されたSOLラップトップの説明には、いくつかの異なるリソースが記載されています。 新製品の魅力は、2つの重要な要因の組み合わせによるものです。米国の350ドルのコスト（開発途上国では50ドル安くなると約束されています）と、ソーラーパネルが組み込まれていることによる高い自律性です。 これらはすべて、Ubuntuの最新バージョンの1つによって制御されており、「THE WORLD'S FIRST TRULY SUN-POWERED LAPTOP」という大きなフレーズが先行しています。



ノートブックの仕様は次のとおりです。

  • プロセッサー：Intel Atom D2500 1.86 GHz Duo Core、Intel 945GSE + ICH7M
  • ハードドライブ：Seagate 2.5” SATA HDD 320GB
  • RAM：キングストン2-4GB DDRIII SDRAM
  • ビデオ：1080p HD Vide、ビルトインIntel GMA3600グラフィックス
  • 画面：13.3 "LCD、WXGA、1366 x 768
  • カメラ：3MP
  • サウンド：Realtek ALC661 HD Audio、ビルトイン2スピーカー| 内蔵マイク+ 1/8インチ入力
  • モデム：3G / 4Gワールド/マルチモードLTE
  • GPS：GPSOne Gen8A
  • WiFi：MIMO 802.11b / gn（2.4 / 5GHz）
  • Bluetooth：統合デジタルコアBT4.0
  • 3 USB2.0、ヘッドフォンジャック、HDMI、LAN（10/100）、カードリーダー（SD / MS / MMC）




これらの比較的優れた特性の背後には、主な「機能」があります-ソーラーパネルによって提供されるラップトップの自律性 デバイスを「バッテリーから直接」起動できると言われています。これは約2時間充電され、負荷と充電レベルに応じて8〜10時間の稼働時間を提供します。

SOLは、2つのバージョンで米国で利用可能になります。最初のバージョンは、上記のように350ドルで費用がかかります。 2つ目は50ドル高額で、記載されているすべての「機能」に対して耐水性（水中）があります。
