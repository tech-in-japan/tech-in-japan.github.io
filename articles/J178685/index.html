Debian 7

Debianバージョン7.0の待望のリリースが行われました。







このバージョンでは、多くの改善が「内部」で行われ、エンドユーザーに表示されます。

現在、パッケージベースには12800を超える新しいパッケージがあります。したがって、リポジトリ内のパッケージの合計数は37493です。Squeezeと比較して、20160パッケージは更新され（パッケージベースの70％）、4125（14％）はさまざまな理由で削除されました。



設置


インストーラーには次の変更が加えられています。



  • UEFIサポート（セキュアブートはサポートされていません）
  • WPAサポート
  • 音声コマンドを使用したインストール
  • Ext4デフォルトアクション


ソフトウェア


配布には、次のソフトウェアバージョンが含まれます。



  • Linux Kernel 3.2と独自のパッチ
  • X.Org 7.7（X.Org Server 1.12）
  • GNOME 3.4（3.2の一部のコンポーネントを含む）
  • KDE 4.8.4
  • Xfce 4.8
  • LibreOffice 3.5（デフォルトで有効になっており、アップグレード時にOpenOffice.orgを置き換えます）
  • MySQL 5.5
  • Apache 2.2.22
  • Python 2.7
  • GCC 4.7.2
  • LLVM / Clang 3.0
  • Iceweasel（Firefox）10
  • systemd 44




マルチアーチ


Debian 7はマルチアーキテクチャに移行しました。 つまり、同じマシンに複数のアーキテクチャのパッケージを一度にインストールできます。 ia32-libsはもう使用されていません。これについては小さなメモがあります。



リポジトリ


Debian Backportsサービスは現在公式にDebianプロジェクトの一部です。 バックポートを使用するには、次のリポジトリを接続する必要があります。

deb mirrors.kernel.org/debian wheezy-backports main contrib
