公式Joomla 2.5リリースはMS SQL ServerとPostgreSQLをサポートします





JoomlaをベースにしたほとんどのサイトはバージョンJoomla 1.5で実行されているという事実にもかかわらず、開発者は依然として違いを生むよう努めています。 先日、このCMSの新しいバージョンであるバージョン2.5がリリースされました。 開発チームは、Microsoft SQL ServerやPostgreSQLなど、Joomlaの新しいデータベース管理システムのサポートを新しいバージョンで導入しました。 以前は、MySQLのみがサポートされていました。



他の新製品の中でも、Joomlaにはついにいくつかの確認ボタンを押すだけで、このCMSを自動的に更新する機会があります。 Wordpressでは、このような機会がかなり前に現れたことは注目に値します。 したがって、初心者にとってはJoomlaでの作業が容易になり、初心者にとっては非常に複雑な更新プロセスに特別な知識は必要なくなりました。 そして、プロフェッショナルにとっては、更新に費やす時間を大幅に削減できるため、このような機会は大きなプラスになります。



開発者自身がさまざまなDBMSのサポートを最も誇りに思っており、この機能により製品の人気がさらに高まると考えています。 製品の新しいバージョンにこのようなサポートを含めることは、ユーザーからの苦情が繰り返し発生することによる解決策です。 たとえば、多くの人が何らかの理由でMySQLを使用できないと不満を漏らしましたが、Joomlaをもっとよく知りたいと思っています。 一般的に、開発者はユーザーに会いに行きました。







新しいバージョンには統合された検索があり、この機能を基本にすることが決定されました（以前の検索は別のオプションでした）。 検索機能は、いわば、Joomlaで大きく前進し、いくつかの商用ソリューションと一致しました。



現時点では、世界で約200万のサイトがこのCMSに基づいて動作していますが、これは非常に優れています。
