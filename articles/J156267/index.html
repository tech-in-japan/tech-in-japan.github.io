新しいSwitchvox 450および470、APIなど...





今日、アスタリスクが独自のPBXを構築するための無料の「エンジン」ではないことは周知の事実です。 これは、多数の商用ソリューションの基礎です。



多くの製品があり、品質の低いものがありますが、最良のソリューションはまったくありません。



ご想像のとおり 、ユニファイドコミュニケーションシステムSwitchvoxについて話します。 格付け機関Frost＆Sullivan 、雑誌InternetTelephonyおよび他の多くによると、2012年に最高と認められたアスタリスクで最も人気のある商用ソリューション...



Digiumには3つのボックスオプションが用意されていることを思い出してください。小規模企業向けSwitchvox AA65、中規模AA305およびAA355用です。



製品のSMBバージョンは、マルチレベルIVR、スイッチボードコールの便利な管理のための操作パネル、外部システムとのAPI統合などを提供します。



サーバーを使用すると、システムを最大400ユーザーに拡張し、最大75の同時通話を行うことができます。 もちろん、これはアスタリスクの制限ではなく、会社はこの「欠陥」を修正することを決定しました。



今日、Digiumは大企業向けのソリューションを提供しています：Switchvox AA450およびAA470



新機能と新機能については、habracatに従ってください。

Digium Phones API







新しい鉄部品の発表前でさえ、Digiumはソフトウェア自体を更新しました。 Switchvox SMBはバージョン5.7になり、最新のIE、Linux用ブラウザー、およびDシリーズ電話用の更新されたファームウェアのサポートが追加されました。 現在、1.3という番号が付いており、デスクトップIPデバイスで独自のアプリケーションを使用できます。



Digium電話APIを使用すると、独自のJavaScriptアプリケーションを実行できます。 ウェブ開発者はこれに関して問題を抱えるべきではないと思います。



なぜ固定電話で独自のアプリケーションが必要なのですか？



Hello Worldを作成しますか？ :)







天気やTwitterフィードなどの外部サーバーから情報を取得しますか？







しかし、本当の価値は、IP電話をWebサイトまたはCRMシステムと統合することです。 標準の発信者IDは退屈です。 電話を受けたら、クライアントについてもっと知る必要があります！ 注文ステータス、通信履歴、連絡先など...







ロシア化Switchvox



Switchvoxは完全にロシア語です。 以前は、Webインターフェースのユーザー側のみが翻訳されていました。







現在、管理者による管理にはキリル文字が付随しています





残念ながら、このローカライズは一時的に開発者によって承認されていないことに注意してください。 しかし、これは一時的な現象だと思います。



価格



そして、はい、結論として価格に言及する価値があります。 推奨小売価格は、サーバー450と470でそれぞれ9,850ドルと12,975ドルです。 ユーザーのプラスサブスクリプション（ライセンス）：各50ドル。 〜システム全体で約3万ドルの合計価格。 たくさん？



500人以上の従業員がいる会社のこの計算は、新しいPBXだけでなく、本当に価値のあるユニファイドコミュニケーションシステムの計算であることは注目に値します。
