Red HatはAPI管理会社である3scaleを買収し、オープンソース製品を提供する予定です





Red Hatは、ソフトウェア開発者が使用するアプリケーションプログラミングインターフェイス（API）を管理するためのソフトウェアを開発する3scale社を買収すると発表しました。



声明によると、GAAPに基づく営業費用は2017年度に700万ドル増加するという事実にもかかわらず、この取引はRed Hat（オープンソースソフトウェアを提供する公開会社）にとって大したことではありません。



Red Hatは、既存の3scaleクラウドサービスを販売するだけでなく、市場にローカルバージョンを導入する予定です。 さらに、Red Hatの副社長兼ミドルウェアディレクターであるMike Peach氏のブログでは、Red Hatが「通常どおりソースコードを開く」と書いており、3scale製品は既存のソフトウェアの機能を拡張します。 。



「3scaleテクノロジーは、JBossミドルウェアの能力を拡張します。 その堅牢で機能豊富なAPIコントロールは、現在Red Hat JBoss Enterprise Application Platform（EAP）、JBoss Fuse、JBoss A-MQ、およびRed Hat Mobile Application Platformに組み込まれている既存の高度な統合機能に追加されます」 「統合プロジェクトを実装しているお客様は、API管理機能に関心を持っています。実績のあるJBoss製品に加えて3scaleを提供できることを嬉しく思います。」



ライバルのApigee社は昨年上場し、Masheryは2013年にIntelに買収されました。



3scaleは2007年に設立され、サンフランシスコとバルセロナにオフィスがあります。 2013年、3scaleはJavelin Venture PartnersおよびCostanoa Venture Capitalから420万ドルを受け取りました。 彼女のクライアントには、キャンベル、オレンジ、テレグラフなどの企業が含まれます。



Red Hatは、DevOpsテクノロジーを実装するためのソフトウェアを開発する企業であるAnsibleの昨年の買収を通じて機能を拡張しました。



さらに、Red Hatは本日、10億ドルの株式買戻しプログラムの開始を発表し、5月31日に終了した四半期の収益を発表しました。 Red Hatの純利益は9,200万ドル（GAAPによるとは異なります）で、5億6800万ドルの収益に対する1株当たりの希薄化前利益の50セントです。 同社の利益はアナリストの予想と一致しており、その収益は予測値を上回っていました。



交換セッション終了後の電子取引中、Red Hatの株価は5％下落しました。
