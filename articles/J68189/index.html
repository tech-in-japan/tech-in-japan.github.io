すべての（現在の）バージョンのWindowsおよびそれらに相当するMacに共通のフォント

はじめに



このリストには、現在関連するすべてのWindowsオペレーティングシステム（実際にはWindows 98以降）に共通のフォントと、Mac OSでの同等のフォントがリストされています。 これらのフォントは、 ブラウザで安全なフォントと呼ばれることもあります 。 これは、Webページを作成するときに使用する小さな参照であり、役に立つと思います。



Webデザインを初めて使用する場合は、次のようなことを考えているかもしれません。「なぜこんな小さなフォントセットに限定する必要があるのでしょうか。 美しいフォントの膨大なコレクションがあります！」という事実は、訪問者のブラウザは、オペレーティングシステムにインストールされているフォントのみを表示できるということです（ 翻訳者のメモ：現在、 CSS 3およびその新しいプロパティは@ font-faceです;ただし、すべてのブラウザーがこの機能をサポートしているわけではありません）、つまり、ページを訪れるすべてのユーザーが選択したフォントの所有者でなければなりません。 したがって、各オペレーティングシステムにあるフォントのみを使用する必要があります。 幸いなことに、 CSSにはこれを簡単にする@ font-familyプロパティがあります。



Windowsの各バージョンに含まれるフォントのリストにも興味があるかもしれません。

1 GeorgiaおよびTrebuchet MSフォントはWindows 2000 / XPに付属しており、IEフォントパックに含まれている（実際、多くのMicrosoftアプリケーションに付属している）ため、Windows 98を実行する多くのコンピューターにインストールされます。



2文字フォントはInternet Explorerでのみ表示され、他のブラウザーでは通常標準フォントに置き換えられます（ただし、SymbolフォントはOperaで表示され、WebdingsはSafariで表示されます）。



3 Book Antiquaフォントは、Palatino Linotypeとほぼ同じです。 Palatino LinotypeはWindows 2000 / XPに同梱されており、Book AntiquaはWindows 98に同梱されています。



4これらのフォントはTrueTypeではなくビットマップであるため、一部のサイズでは見栄えが悪くなる場合があります（96 DPIで8、10、12、14、18、24 ptのサイズで表示されるように設計されています）。



5これらのフォントはSafariで標準スタイルでのみ機能しますが、太字または斜体で強調表示されている場合は機能しません。 Comic Sans MSも太字で機能しますが、斜体では機能しません。 他のMacブラウザーは通常、欠落しているフォントプロパティを独自にエミュレートしているようです（ヒントについてはChristian Fecteauに感謝します）。
