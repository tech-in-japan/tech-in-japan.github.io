MySQL組み込みおよびライセンス

MySQL Embedded + ZeosDBO + Delphi 7バンドルを使用したいのですが、どこでもライセンスを取得することに関して明確な答えが見つかりませんでした。 私が出くわしたすべてのこと、いくつかの不明瞭な合計があり、最悪のことは、mysql.ruで最大の答えが見つかったが、明らかに終わっていないことでした。 SUNマイクロシステムズとのコミュニケーションは結果をもたらさず、彼らはサッカーをし、1人に電話し、そこに電話し、そこから電話して彼らに切り替えさせました。 独自のフォームを介したリクエストも、何にもつながりませんでした。 Vobschemがらくた。

その結果、今日私は集まったが、まだ質問が残っている：あなたの顧客への配布のために、2つの条件のうちの1つが満たされなければなりません：ソフトウェアがGPLの下で配布されるか、すべての職場のための支払いは〜325ユーロ/職場です。



注：この投稿は、MySQLと「その他のデータベース」またはDelphi 7向けに作成されたものではありません。 それぞれに独自の開発ツールと異なるタスクと異なるクライアントがあります。





当然、そのような条件は単に「地獄」であり、受け入れられません。 この金額は、インストールごとに200〜250ドルの金額とは比較にならないものであり、この金額にはクライアントへの訪問、3か月間のテクニカルサポートが含まれます。 つまり 1ライセンスのコストは、プログラムのコストの2倍を超えます。

開発者ライセンスを1〜2000ドルで支払うことに同意しますが、ロイヤリティフリーライセンスの下では、それが意味をなさないので、550〜700ドルの金額で、このhemoの代わりにPostgresを入れて、会社に来てもらいます。 しかし、このプロジェクトは主に社会的であり、単にインターネットを使用できない、またはお金さえも使用できない人々にとっても、すべての人々にとって非常に有用であるため、金額を上げたくありません。



さて、可能な弁護士をいじってみましょう。どの弁護士が助けになるかを突然知っているので、あきらめたくありません。



オプション1： GPLライセンスが必要、良い。 Hello Worldプログラムの開発、ソースコードを開き、簡単なサイトを作成し、GPLライセンスを使用して、いわゆる API 「CARS」などの商用プラグインを作成し、Hello World APIに固定します。 私が理解している限り、Hello Worldが配布されているGPLライセンスに違反していますが、この「HELLO WORLD」はライセンス違反に訴えません。理論的には、MySQLはOPEN API「HELLO WORLD」を使用するため、開発者「CARS」を訴えることはできません。 また、「Hello World」の開発者に苦情を申し立てることはできません。彼はGPLライセンスを提供しており、オープンソースの製品だからです。



オプション2： GPLディストリビューションとは正確には何ですか？ たとえば、組み込みのlibmysqld.dllを含むソフトウェアをクライアントに販売しない場合（このlibのみが必要）、サードパーティまたは公式のリソースからダウンロードするか、システムがこのライブラリを介してADDITIONとして販売される場合。



これまでのところ、他の選択肢は思い浮かびませんでしたが、高官が価格を動かさずに開発者を人間的に扱った方が良いと思いますが、アメリカとヨーロッパではどうすればいいかわかりません。架空の会社に責任を負わせてお金を取るのは簡単ですソフトウェア用ではなく、Nevedoma-Zverushka-Soft社のソフトウェアのインストール用です。



UPD：この投稿は盗むことを試みるものではありませんが、少なくともSUNマイクロシステムズ（およびロシアとウクライナに支部があります）の少なくとも誰かが見て、まだ開いているかどうかの質問に明確な答えを与えることを願っています、「盗難」などの「回避策」なしで、libmysqld.dllファイルを使用します。 「盗み」はしたくないが、法外な「トリビュート」を支払う機会もない。
