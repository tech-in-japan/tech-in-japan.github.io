300万人近くのユーザーが評価したOpera Turboの利点









2009年10月22日-ノルウェー、オスロ -Opera 10が発表されてから（2009年9月1〜30日）、世界中の約300万人がOpera Turboテクノロジーを使用しています。 この間、彼らは低速インターネットチャネルでの表示に最適化された6億6800万近くのWebページをロードしました。



毎日4000万人以上がOperaデスクトップブラウザを使用しています。 Opera Turboのこのような高い人気は、この技術が世界中のインターネットユーザーのニーズと期待を満たしていることを示しています。 Opera 10を搭載したOpera 10は、 www.opera.comから無料でダウンロードできます。



ユーザーが自宅からダイヤルアップ接続経由でネットワークにアクセスするか、空港の公共Wi-Fiアクセスポイントを使用するかに関係なく、低速通信チャネルの問題に直面しています。 Opera Turboテクノロジーは、主にWebページの読み込み速度を大幅に向上させる能力が求められています。 ただし、Opera Turboの人気は、インターネットユーザーがメガバイト料金でダウンロードデータの量を減らしたいという願望によっても説明されています。 Opera Turboがユーザーデバイスに送信する前にウェブページを圧縮する機能により、トラフィックを削減し、時間とお金を節約できます。



Opera SoftwareのCEOであるJon von Tetzchnerは、次のように述べています。 -誰もがこの問題に絶えず直面しています。出張中のビジネスユーザーからインターネットカフェの訪問者まで。 Opera Turboはインターネットのユニバーサルアクセラレータとして機能します。」



Opera Turboテクノロジーは、Opera Softwareが推進するOne Webコンセプトの一部として開発されました。 Opera Turboは、いつでもどこでもインターネット体験を改善および高速化するように設計されています。 このテクノロジーは、インターネットアクセスの制限を解除し、すべての人およびすべての人がグローバルネットワークにアクセスできるようにします。



Opera Turboテクノロジーについて

Opera Turboは、Opera 10およびOpera Mobileブラウザーで利用可能な独自のデータ圧縮テクノロジーです。 Opera Turboは、Webページのほぼ80％の圧縮を可能にします。 したがって、Webサイトのコンテンツのロード速度が大幅に向上します。 トラフィックを請求するとき、ユーザーは同じお金でより多くの情報を受け取ることができます。 Opera 10でOpera Turboを有効にするには、左下隅のアイコンをクリックするだけです。 Opera Turboテクノロジーの詳細については、 www.opera.com / browser / turboをご覧ください。



Opera Software ASAについて

Opera Software ASAは、PC、携帯電話、その他のインターネットデバイスでWebページを表示するための新しい可能性を切り開きました。 Operaによって開発されたクロスプラットフォームブラウザーテクノロジーは、その機能、標準への準拠、小さなサイズで知られています。 同時に、高速で安全かつ動的なインターネットアクセスをユーザーに提供します。 Opera Softwareのメインオフィスはノルウェーのオスロにあり、世界中にオフィスがあります。 同社は、OPERA証券コードでオスロ証券取引所に上場しています。 詳細については、 www.opera.comをご覧ください。
