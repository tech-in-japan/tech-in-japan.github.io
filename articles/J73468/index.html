VistaSwitcher-代替ウィンドウスイッチング

Windows VistaおよびWindows 7では、ウィンドウ間の切り替えに2つのタイプがあります。標準のAlt + TabとWindows Flip 3Dです。

1つ目はほとんどの人にとって馴染みがあり、2つ目はより美しくなりますが、必ずしも便利ではありません。



長い間、これら2つのタイプを何らかの形で組み合わせたいという要望がありました。 そしてつい最近、素晴らしいアプリケーション、 VistaSwitcherに出会いました 。









だから。 アプリケーションはトレイに静かにぶら下がっており、スクリーンショットに示すように、標準のウィンドウ切り替えを便利で便利な方法に置き換えます。 実際、右側には実行中のアプリケーションのリストがあり、左側には空のデスクトップの背景にあるアプリケーションのプレビューがあります。 これらはすべて、角が丸い半透明の箱に入っています。



Alt + Tab、または矢印、またはマウスを使用して、アプリケーションのリストをナビゲートできます。



アプリケーションには32ビットバージョンと64ビットバージョンがあります。
