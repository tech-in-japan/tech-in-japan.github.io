Intel Core i3プロセッサに基づいたToshiba Portege Z830ウルトラブックを確認する

Acer Aspire S3ウルトラブックに関する前回の記事で、Core i3とオフィスタスク専用の小型SSDドライブを備えた軽量マシンを見たいと言いました。



そしてここに彼女は-東芝Z830-A2S-ほとんど夢です。 クラス最高の軽量（わずか1.12 kg）、マット（！）スクリーン、3（！）USBポート。 しかし、何かがあなたに伝えます、そのような理想的なオプションはあり得ません。 私はこの東芝の中に何があるか見ていきます。



ウルトラブックの中で最も控えめなのはパッケージング：キャッチーなダストジャケットや色のないシンプルな段ボール箱です。 ウルトラブックは2つのフォームインサートの間に固定されており、充電器付きのボックスが側面にあります。



充電器は小さいですが、ASUSがネットブックやウルトラブック用の小さなメモリに切り替えたことはまだ驚くべきことであり、他のメーカーはキットのこの非常に重要な部分のサイズを小さくすることができません。



他のほとんどのモバイルコンピューターと同様に、ソフトウェアディスクは含まれていません。 システムを復元するには、Windowsの起動時にメニューを呼び出し（F8キーを押して）、「問題の修正」項目を選択する必要があります。 次に、いくつかの修正オプションが提供されますが、その中にはToshiba Recoveryがあります。

東芝は、Portege Z830ウルトラブックのいくつかの修正版を提供しています。 私は次の仕様のZ830-A2Sになりました。

東芝の登場は個人的に非常に嬉しいです。 はい、もちろんこれはMacBook AirやASUS Zenbookではありませんが、Z830は本当にスタイリッシュで美しいように見えます。 しかし、微妙な違いがあります。



ウルトラブックの左側にあるステッカーはこちらです。 以前のテストサンプルでは、​​ロゴは灰色で、デザイン全体に完全に適合していました。 東芝はその道を選び、IntelとMicrosoftからの恐ろしいフルタイムステッカーを残しました。



これも驚くべきことです。 「CIS諸国のために作られた」は、昇進のための最高のスローガンではありません。 CIS諸国にとっても、彼らが今日行っているのと同様に、他の国々にとってもそうであることを願っています。

電源ボタンは、トップパネルの左側にあります。 その隣にはさらに2つの小さなボタンがあります。エコモードをオンにし、表示をすばやく切り替えるためのボタンです。 Eco Utilityユーティリティを使用すると、ウルトラブックのエネルギー消費を制御でき、Toshiba Portege Z830の使用中に節約した電気量を表示できます。



キーボードは、87の独立したキーで構成されています。 それらの描画は最も成功しておらず、ロシア語と英語のフォントは同じ色で適用され、追加の機能はグレーでペイントされていますが、これは読みにくいです。



東芝ウルトラブックのタッチパッドは芸術作品です。 作業台の小さなサイズは、指紋リーダーとハードウェアタッチパッドミュートボタンの2つの専用キーによって相殺されます。 表面で作業するのが便利です。感度は良好なレベルにあり、キーは特徴的なクリックではっきりと押されています。 タッチパッドの横にLEDインジケーターがあります：ウルトラブックオン、電源オン、ハードドライブアクティビティ、ワイヤレスネットワークアクティビティ、3Gモデムアクティビティ（私のモデルではありません）。





サイドパネルには拡張ポートがあります。 左側：2つのオーディオジャック-ヘッドフォン用とマイク用、およびすべてのSecureDigital（SD / SDHC / SDXC）およびMultiMediaCard形式をサポートするカードリーダー用。 彼はケンジントンロックスロットである右側に孤独なUSB 3.0。





背面パネルはよりリッチに見え、2つのUSB 2.0ポート、1つのHDMIポート、充電ソケットとイーサネットポート、ウルトラブックの珍しいゲストが含まれています。 背面には、熱気を放出するための換気グリルがあります。 私は一度に3つのUSBポートが非常に良いニュースだと言いたい、私が訪れた以前のウルトラブックはそのような富を誇っていなかった。





下部パネルには、ウルトラブックとWindowsに関する情報が記載されたステッカーと、冷却システム用の丸い空気取り入れ口があります。



ウルトラブックのマトリックスは完全ではありませんが、マットです。 通常、現在の世代のほとんどのモバイルコンピューターの解像度は1366 * 768であり、快適な作業には十分ではありません。



Toshiba Portege Z830のサウンドシステムはRealtekコーデックに実装されており、Dolby形式をサポートしています。 音楽を聴いたり、ビデオを見たりするとき、私は否定的な感情を感じませんでした。 音質は、デバイスの適度な寸法にとってまともです。



東芝は数本の普通のネジで蓋を閉めたので、中を撮影できませんでした。真ん中に驚きを残しました-複雑な頭のネジ-中央に盛り上がった6先の尖ったスプロケット。 そんなに見つけられませんでしたが、実際には、たとえばKingston SSDでそのようなソリューションに出会ったことがあります。 どうやら、サービスセンターには、奇跡のドライバー付きの特別なブラックボックスがあります。

写真がないので、そのように話します。 中央処理装置Intel Core i3-2367Mは1.4 GHzでクロックされます。 32nmプロセステクノロジを使用して作られたSandy Bridgeプロセッサフ​​ァミリに属します。 より若いシリーズ（i3）に属しているため、TurboBoostテクノロジーをサポートせず、プロセッサーの周波数は公称値を超えて増加しません。 ハイパースレッディングテクノロジーが完全に存在し、Intel Core i3-2367Mは4つのデータストリームを同時に処理できます。







ビデオサブシステムはプロセッサに統合されており、Intel HD Graphics 3000の名を冠しています。コアで実行される実際のグラフィックタスクに加えて、350〜1100 MHzのクロックで、統合ビデオカードはIntel Quick Sync（ビデオエンコーディング用）、Intel In Tru 3D（ 3Dムービーの再生）、Intel Clear Video HD（ビデオの再生時にアーティファクトを除去するため）。



Toshiba Portege Z830にデータを保存するには、128ギガバイトの容量を持つ社内SSDドライブを使用します。



RAMは2つの部分から組み立てられます-マザーボードにはんだ付けされ、標準モジュールSO-DIMM DDR3の形で取り外し可能です。 非常に興味深い解決策は、自分でメモリを増やすことができないのは残念なことです（その理由は、ウルトラブックの真ん中にある魔法のねじにあります）。 ただし、サービスセンターは、2ギガバイトのバーを4ギガバイトに置き換えて喜んでお手伝いします（もちろん無料ではありません）。





これまでのところ、すべてがスムーズであり、ウルトラブックは大衆に適しています。 Z830のパフォーマンスを確認します。

Windows 7の標準スコア-総合スコア5.2



比較のために、Acer Aspire S3のスクリーンを示します。合計スコアは4.5です。



違いは何ですか？

Intel Core i5がIntel Core i3よりも速いという事実は理解でき、RAMは同じ周波数で動作しますが、奇跡は続きます。 同じグラフィックスカードを使用すると、東芝はWindows Aero環境での作業で0.9ポイント多く得られます。これは、デュアルチャネルメモリモードとSSDドライブによるものだと思います。 ゲームのグラフィックスパフォーマンスもほぼ同じですが、東芝のハードドライブパフォーマンスは優れています。AcerAspire S3には、キャッシュSSDを備えた通常のハードドライブが搭載されていることを思い出してください。

Futuremarkテスト-PC Mark 7およびPC Mark Vantageは、Acerの1871および5054ポイントに対して、2424および5779ポイントを獲得しました。 これは、Core i3の控えめなマシンが、より強力な（プロセッサの点で）競合他社よりも優れている方法です。

ハードドライブの速度は素晴らしいものではなく、通常のSATA II SSDドライブです。



RAMの速度はAcerと同等です-あまり強力でないプロセッサーは2チャンネルで補正されます。



東芝のウルトラブックの温度は、エイサーの温度よりも悪いです。 動作中、ウルトラブックは大幅に加熱され、ファンが鳴り始めます。 加熱中の最高温度は75度で、ひざの上にデバイスを置いておくのはまったく不快になりました。



バッテリー寿命は、Futuremarkの新しいベンチマークであるPowerMarkによって測定されました。 このシナリオでは、バランスの取れた東芝は2時間49分で、Acerよりも20分長くなりました。 薄暗い明るさでドキュメントを操作するときの実際の時間は約5時間でした。



まとめの考え。

私は東芝Z830が好きで、ウルトラブックで見たい多くの機能を共有しました。 プロセッサの速度はほとんどのタスクを実行するのに十分である必要があります.128ギガバイトのソリッドステートドライブは、ドキュメントや必要なファイルを簡単に引き継ぎます。日当たりの良い場所で作業する場合、マットスクリーンは迷惑になりません.3つのUSBポートにより、マウスとUSBフラッシュドライブの両方を接続し、スマートフォンを充電できます。 もちろんマイナスもありますが、マトリックス1600 * 900の解像度、より高速なSSD SATA III、より効率的な冷却システムが必要です。 しかし、これらすべての改善は価格の上昇につながります。 しかし、力と価格の現在の整合において、「最高のウルトラブック」のタイトルのための闘争において、東芝Portege Z830が上に出ます=）。 もちろん、これは私の個人的な意見であり、この特定のモデルを強制的に購入することはありません。 先は第3世代のIntel Coreモバイルプロセッサの出力であり、おそらく好みが変わるでしょう。
