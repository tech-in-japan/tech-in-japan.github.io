マイクロソフトロシアに手紙を送りました

こんにちは



申し訳ありませんが、間違ったアドレスに書いている可能性がありますが、残念ながら、Microsoft Webサイトの「フィードバック」セクションでフォームまたはアドレスを見つけることができませんでした。



私はおそらくある種の間違ったユーザーかもしれませんが、4年前からWindows Mobileベースのコミュニケーターを既に使用しており、昨年になって5回（今日は前回）連絡先を取得しました。 そして、ActiveSyncまたはWindows Mobileデバイスセンターを使用しているときにこれが発生するたびに。 おそらく私の問題の根源は、コミュニケーターの3台のコンピューターへの接続を使用していることと、モバイルデバイスが2台以上のコンピューターに接続されている場合にActiveSync / Windows Mobileデバイスセンターが誓うことです。



しかし、それにもかかわらず、大きな要求：

  • 連絡先の同期インターフェイスに機能を追加して、連絡先を削除または変更したときに、このアクションを実行するための要求が表示されるようにしてください。 少なくともオプションでこれを行うことができます。
  • また、ActiveSyncとWindows Mobileデバイスセンターで、一方向の連絡先の同期を明示的に設定できるようにします。 ActiveSyncは、コミュニケーターがより新しいベースを持っていると判断し、デスクトップコンピューターから連絡先を消去する可能性があるため、現在の連絡先のベースを使用してコンピューターにコミュニケーターを接続することを恐れています。
  • Windows Mobileのファイルから連絡先をインポートする機能を追加してください。
  • 連絡先の同期をキャンセルする機能も追加してください。
私の手紙を無視して開発者に伝えたり、問題を解決するためにどこに連絡する必要があるかを教えてくれないかと思います。



敬具、ノスコフ・ミハイル。
