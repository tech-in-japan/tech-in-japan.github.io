スマートテレビ



おそらく、ハブで最も人気のない方向、つまりテレビから始めます。 インターネットの出現により、テレビへの関心は低下しておらず、変化しています。 インターネットは、ゲームに新しいルールを提供しました。選択するコンテンツ、ソーシャルネットワーク、選択の自由、あらゆる種類の情報の膨大な量です。これらは、構成や複製/認識の方法が大きく異なります。 「最初のチャンネルのニュース」を備えた通常のテレビは、ゲームの新しいルールに対応することを余儀なくされ、さまざまなメーカーがスマートテレビテクノロジーを考案しました。 この運動の先駆者および指導者の一人はサムスンであり、この分野における最新の開発はIFAで発表されました。



私が強調したい新製品の中に、特別なゲストがいます-55インチの対角線と最新のサムスンスマートテレビとスマートハブテクノロジーを完全にサポートするES9500 OLEDテレビ。







優れた画質と驚くべき黒の深さは、薄いフレームと優れた色の演出と相まって、競合他社にチャンスを残しません-スーパーマーケットの棚にある数十の類似モデルの中で、このテレビは着実に目を引き、注目を集めます。 しかし、その主な「キラー機能」は、2人の視聴者が互いに干渉することなく、2つの番組を同時に視聴できることです。 これまでにない短い応答時間と、ヘッドフォンを内蔵した特別な3Dメガネのおかげで、コンテンツはユーザーごとに完全に表示されます。







それに加えて、新しい75インチのフラッグシップES9000について言及したいと思います。これは、スマートTVコンテンツとゲームAnrgy Birdsの分野における最新の開発をサポートし、デバイスに触れることなく初めて制御できます。 75インチの対角線の少なくとも半分で「鳥」をプレイしたことがありますか？ :)







スマート写真





サムスンによって提示された展示会で最も珍しいデバイスの1つは... GALAXY Camera、モバイルスマートカメラでした。 この「超音波」とスマートフォンのハイブリッドは、BSIテクノロジー、21倍光学ズーム、画像安定化、3G、4G、Wi-Fiネットワークのサポート、そして巨大で真にシックな4.55インチHDディスプレイを備えた16MPピクセルマトリックスを備えています。 この素晴らしさはすべて、Android 4.1 JellyBeanオペレーティングシステムの制御下で機能します。







Googleのオペレーティングシステムが提供する機能に加えて、カメラには組み込みの画像エディターと、外部条件と選択した撮影プログラムに応じてセンサーの動作を修正する特別なマルチモードソフトウェアが装備されています。 さらに、カメラはAuto Cloud BackupおよびSamsung Allshareサービスにアクセスできるため、コンテンツを簡単に共有および再生でき、安全性を恐れることもありません。







スマートPC





注意なしではなく、Microsoftの新しいオペレーティングシステム。 Samsungは、タブレット、PCトランスフォーマー、およびWindows 8を実行するスマートフォンで構成されるATIVと呼ばれるデバイスの全ラインを導入しました。



Samsung ATIV Sは、WP8ベースのスリムでスタイリッシュなスマートフォンです。 内部-すべてがAndroidの「アダルト」モデルのようなものです-デュアルコア1.5 GHzプロセッサ、あらゆる種類のワイヤレスプロトコルとインターフェイスのサポート、HD解像度の優れた画面、HD SuperAmoledテクノロジーを使用した4.8インチの対角線。







あなたのようなサムスンATIVタブ。 おそらく推測-Windows RTベースのタブレット。 その自由に10.1インチのディスプレイ、8200 mA / hのバッテリー容量、新しいOSの完全な機能と組み込みのオフィススイートOffice 2013 RTプレビューがあります。











Samsung ATIV Smart PCおよびSmart PC PROは、「タブレット」モードと「本格的な」モードの両方で動作可能なトランスラップトップです。 これらは、Galaxy S Note IIでおなじみのフォームファクター、11.6インチFullHD LCDディスプレイ、S Penによって統合されています。GalaxyS Note IIは、「タブレット」モードでトランスを完全に使用できるようにする独自のソフトウェアパッケージです。 各デバイスには、キーボードドックに2つのUSBポート、mHDMI、microSDスロット、および2つの追加のUSB 2.0ポートが含まれています。







スタッフィングデバイスは異なります-Smart PCの「タブレット」部分には最新世代のIntel Atom USB 2.0ポートが含まれており、バッテリーで最大14時間使用できます。 Smart PC PROの兄は、「タブレット」であるCore i5アーキテクチャ（USB 3.0）に基づいて構築されており、バッテリー寿命は8時間に達します。 そのようなデバイスには悪くないですか？







スマートデスクトップ





別の目新しさは、オールインワンモノブロックの新しいAIO（オールインワン）ラインです。 ラインの古いデバイスには、27インチの解像度2560x1440のスクリーンがあり、そのマットディスプレイはPLSテクノロジーを使用して作られています。内部-最新のCore i7、最大8ギガバイトのDDR3メモリ、Radeonのディスクリートビデオ（2ギガバイトのGDDR5メモリを備えた7850Mモデル）および無線インターフェース。



最新バージョンは23.6インチのフルHDで、内部はマット仕上げ、i5またはi3、最大8ギガバイトのメモリ、わずかに弱いビデオカード7690、1 GB GDDR5、および同じ有線/無線インターフェースのセットです。少し遅れて中間モデルのTTXが発表されるかもしれません。







これらのソリューションの主な「キラー機能」は、マルチタッチ入力（最大10タッチ）および統合されたHDカメラによるジェスチャー認識です。

Windows 8の新機能（これらのデバイスに搭載される予定）を考えると、スタイリッシュな外観、幅広い可能性、使用の柔軟性という優れたカクテルが判明します。 ハイテクの「フィリング」は、写真やビデオの編集、ゲーム、高解像度の映画やその他の作品に簡単に対応でき、コンパクトなボディと優れた画面品質により、選択が容易になります。 :)



スマートディスプレイ









B2Bセグメントは忘れられていません。 IFAでは、 UD22bとNL22Bという新しいスクリーンが発表されました。 1つ目は、特殊な「ビデオウォール」を構築するために設計された、アスペクト比が1対1の特別な22インチパネルです。 そのような製品では、新しい照明技術であるサムスンダイレクトLEDが初めて使用されます。これは、従来のLEDシステムよりもはるかに少ない熱を放出し、エネルギーを大幅に消費しません飛行機！）あなたは本当に素晴らしいビデオのインストールを行うことができます。 キットには、特別なソフトウェアSamsung MagicInfoが含まれています。これにより、この幸福をすべて一元管理できます。



2番目のノベルティ-NL22b-は単なるスクリーンではありません。 これは、製品プレゼンテーション用のソフトウェアおよびハードウェア全体の複合体であり、独自の機能である透明なLEDスクリーンを備えています。 透明なLCDパネルに加えて、このキットには保護ガラス、ミニPC、スピーカーが含まれており、インタラクティブなプレゼンテーションのための完全なセットです。 拡張現実の効果は、特別な眼鏡をかけたり、スマートフォン/タブレットのカメラで「世界をスキャン」する必要なしに得られます。
