新しい年、新しいGitHub：無制限の無料プライベートリポジトリ

本日、GitHubの2つの重要なイノベーションを発表します。これは、開発者がアクセスしやすくするものです。無制限の無料プライベートリポジトリと、企業にとってより便利な製品です。 カットの下の詳細！







  • GitHub Freeには、無制限のプライベートリポジトリが含まれるようになりました。 開発者は初めて、プロジェクトにGitHubを使用して、最大3人の共同編集者を無料でリポジトリに追加できます。 多くの開発者は、サードパーティのプロジェクトで作業するためにプライベートリポジトリを使用するか、公開する前に何かをプライベートに試してみたいと考えています。 今日から、これらのスクリプトやその他の多くのスクリプトがGitHubで無料で利用できます。 パブリックリポジトリは無料のままで、無制限の数の共同編集者が含まれます。
  • GitHub Enterpriseは、Enterprise Cloud（以前のGitHub Business Cloud）およびEnterprise Server（以前のGitHub Enterprise）向けの新しい統合製品です。 クラウドまたはスタンドアロン構成でGitHubを使用する柔軟性が必要な組織は、同じ価格で両方にアクセスできるようになりました。 また、 GitHub Connectのおかげで、これらの製品は確実に相互接続でき、開発者があらゆる環境でシームレスに作業できるハイブリッドオプションを提供します。


さらに詳しく



GitHub Pro（以前のGitHub Developer）およびGitHub Teamは、プロのコーディングおよびコラボレーション機能を必要とする開発者およびチームにも利用できます。 そしてもちろん、オープンソースプロジェクトには、GitHubチームの無料バージョンを含む、パブリックリポジトリでの共同作業に必要なすべてのものが含まれています。



あなたがコードの最初の行を書き込もうとしている学生であろうと、世界中にチームを持つ企業の責任者であろうと、GitHubがコードを記述し、開発者のグローバルコミュニティと協力し、コミュニケーションするのに最適な場所であることを望みます。 今日の変更は、GitHubの将来への大きな投資であり、2019年に考案および作成されたすべてのプロジェクトに非常に満足しています。
