Kerio Control 8.4リリースの新機能





カリフォルニア州サンノゼ2014年10月14日 -Kerio Technologiesは本日、管理しやすいセキュリティソリューションであるKerio Control 8.4の新しいバージョンをリリースしました。 新しいバージョンは、HTTPS、Wi-Fi、ゲストアクセス、IPv6の4つの主要なセキュリティホールを閉じます。 Kerio Control 8.4により、中小企業は最小限の労力でネットワークセキュリティのレベルを上げることができ、これまでになく簡単になります。



「毎日セキュリティを確保することがますます難しくなっています。 Kerio TechnologiesのCEO、Scott Schreiman氏は次のように述べています。 「新しい課題とリスクの発生は避けられません。 Kerio製品は常に一歩先を進み、中小企業がこれらの変化に効果的に対応できるようにします。」

Kerio Controlの新しいHTTPSコンテンツフィルタリング機能は、最大のネットワークセキュリティホールを埋めます。 サイバー犯罪者はSSLプロトコルを積極的に使用して、ネットワークへの安全な接続を介してウイルスやその他の脅威を密かに導入します。 Kerio Controlでは、既存のコンテンツフィルタをHTTPSトラフィックに適用するだけで、ボタンをクリックするだけでこれらの脅威を阻止できます。



Wi-Fiユーザーとユーザー自身のデバイスの数の増加に伴い、企業は従業員とゲストに安全なネットワークアクセスを提供する必要があります。 ただし、WPA共有パスワードは非効率的であり、ネットワークを悪意のある攻撃に対して脆弱なままにします。 新しいKerio Control 8.4ビルトインRADIUSサーバーは、通常実装が困難なエンタープライズレベルのテクノロジーを使用しており、小規模企業でもWi-Fiネットワークで高性能認証と強力なパスワードポリシーを簡単に実装できます。



Kerio Control 8.4は、ライセンスなしでネットワークへのゲストアクセスも提供します。 企業は1つのアクセスポイントを購入し、すべてのゲストに開放したままにすることができます。 同時に、Kerio Controlはイントラネットリソースからゲストトラフィックを分離し、企業の重要な資産とシステムを意図しないゲストアクセスから保護します。



世界中の企業がIPv6に移行しているため、ネットワークセキュリティが低下しています。 IPv6の理解と使用は困難ですが、私たちの場合はそうではありません。 Kerio Control 8.4を使用すると、独自のトラフィックルールを作成し、IPv6に適用するルールを選択するだけで機能します。 余分な労力はありません。

Kerio Control 8.4の詳細については、 www.kerio.ruをご覧ください 。

30日間の試用版は、 www.kerio.ru / support / kerio - controlからダウンロードできます。

  • HTTPS制御
  • IPv6のネットワークプレフィックス変換サポート
  • RADIUSサーバー
