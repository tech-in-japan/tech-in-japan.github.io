新しい遊牧民

運用：この動きのポイントは、ITを顧客に近づけることです。

同僚：それから、北ではなくモスクワの南にあるオフィスに移動するのは理にかなっています-私の主なクライアントはそこにいます...

操作：はい、多分...

私：それからどこに移動すればいいですか？

オペレーション：そして...あなたは、ノボシビルスクにさえ、まったく気にしません。





飛行機に再び座ってこの会話を思い出しましたが、過去6か月間でこれが私の40回目の飛行であると計算しました。 私は大規模な製造会社の「エリアマネージャー」ITマネージャーです。 同社はロシアとCISにほぼ100の支店を構えていますが、同時に地域のさまざまな「充填」が行われています-植物と貿易、貿易のみ、どこかで原材料の購入のみ。 したがって、ITが直面する課題は多少異なります。



個人的にこれはどういう意味ですか？ たくさんのこと。 私はモスクワのオフィスで働いており、出張で働いています。 私は定期的に飛行機や空港に住んでいます（主は私たちにwifiをくれました。ロシアの空港でついに無料で登場するのはいつですか？）



今、あなたの友人/知人/仲間を見てください-あなたは確かに同様のライフスタイルを持つ誰かを見つけるでしょう。 ほとんどの場合、ソーシャルネットワーク、IM、または電子メールなど、リモートで彼とやり取りすることが重要になります。 私たちは「デジタル遊牧民」、「ネットワーク遊牧民」、より正確には「新しい遊牧民」と呼ばれる人々です。



WordSpyのコメントは次のとおりです。

デジタル遊牧民（n。）技術、特にワイヤレスネットワークを使用して、オフィスやその他の固定アドレスを必要とせずに働く人。



クルーニーとの映画「In the Sky」（「Up in the Air」）は私たちのライフスタイルに関するものです。



このような生活様式は特定の仕事の組織を決定します。モスクワ時間の午前11時まで、ウラジオストクに鐘を鳴らしましたか？ 親愛なる友人、明日お会いしましょう-すでに夕方6時です。人々は帰宅しました。 私は...うーん...隣国の1つに緊急の対応が必要なメールを書きましたか？ さらに電話してください-さもなければ、手紙は合理的な時間枠を超えてほぼ100％答えられます-メンタリティ...親愛なる人々、非常に親切で、仕事で非常にリラックスしています...



彼はサービスと保存情報の要件を指示します。すべてのデータは会社のネットワークに保存されます（ラップトップは永久ではなく、通信が非常に重要です）。少なくともTashkentのOmsk、およびAvayaソフトフォンは、通常、企業OSのイメージに含まれ、外部から退出するように構成する必要があります。内部番号を呼び出す方が、モバイルを呼び出すよりも安価です...



また、情報のニーズ（および適切なガジェット）も、生活様式によって部分的に決定されます。空港に座って何か（本または20通のメール）を読むこと。 飛行機を半分目を閉じて映画を見るか、目的地のホテルに行く方法を見つけます。 フライトが遅れるのを待つ間、facebookで大騒ぎします。 はい、それでも、この壮大さがすべて、コンセントから少なくとも7〜8時間、できれば10時間離れたところで機能するように-空港までタクシーで1時間、出発の2時間前に登録して、誰もキャンセルしませんでした。



ノートパソコン？ うん！ バッテリーと企業XPを搭載。 そして、あなたは、タクシー運転手の群衆の中に立って、永遠に「どこに行くのか-タクシー、私たちは-チェックが必要です-安くて、出会わないで」を脇に置いて、ラップトップを手に入れ、片手で（たとえば、2番目のポートレットで）目が覚めるまで待って、 Outlookにアクセスして、ドライバーの連絡先のある手紙を見つけますか？ 複雑なことはありません-ただ快適ではありません。

プレーヤー、リーダー、携帯電話、さまざまな料金など、たくさんのガジェットを持ち歩くだけでなく、手荷物だけで飛行するときは、すべてのキロを大切にし、バッグに入れます。



それでは、少し気を散らしてストーリーを見てみましょう。 遊牧民がいて、座りがちな部族がいました。 遊牧民の間で知識はどのように広まったのですか？ 口頭で。 遊牧民のライフスタイルは、多くの本を意味するものではありません。 そして、スキルは適切です。 解決が現れた-文章が現れ、スキルが変化した-文章の蓄積された知識を読み、維持する能力が開発に必要になった。 遊牧民の間での基礎科学の爆発的な発展は、原則として不可能です。



なぜこれをすべて言っているのですか？ そして、ここに何がありますか：見回す-遊牧民が再び需要があり、それゆえに現れるとき、世界はそのような開発の段階に円を描いて来ました。

インフラも同様です。



新しいデバイス-Amazon Kindleとネットブック-がこのシリーズの最初のツバメでした。 はい、そしてMID自体のイデオロギー-オタクや遊牧民のほかに、そのようなデバイスの機能のほとんどを必要とする人はいますか？



ちなみに、同じiPadは私のニーズにほぼ完璧です。 欠陥がないのではなく、その欠陥-余分なお金を投入するためのマーケティングと政治-MacBookのように、HDMI経由でテレビに接続するために30ドルでケーブルを購入する必要があります。 しばらく待つと、プロジェクターやその他のアクセサリーに接続するためのケーブルが表示されます。 ドック-ところで、USBケーブルは既に含まれています。 そして、次のモデルはおそらくHDスクリーンとWiMaxを搭載するでしょう。



「新しい遊牧民」向けのサービスもあります-Google Voice、Google Docs / Thinkfree / Zoho ... Evernote / Mobile Meなど ソーシャルネットワーク-そこにも。 これは、このサービスが利用できない、または他の人に不便であることを意味するものではありませんが、これらのサービスの主なアイデアの1つは、使用場所からの独立です。



そして、企業市場はそれほど遅れていません-MS Officeは企業ポータル（もちろんSharepointで）とより密接に連携しており、内部ポータルは長い間「企業のクラスメート」と呼ばれていました。 LotusLiveは、実際には「企業のFacebook」です...



Jobsが新しいタブレットを必要とするタブレットで失敗したかどうかを確認したいですか？ それとも、世界がどこに向かっているのか、どのようなサービスが求められますか？ 自分が「新しい遊牧民」だと想像してください-何が必要ですか？
