MS SQL Server 2008は、個人データの保護に関するロシア連邦法の要件への準拠証明書を受け取りました





個人データの保護に関するRF法の要件に準拠するため、Microsoft SQL Server 2008のEnterprise Edition x64およびStandart Edition x64バージョンが最近認定されたことをお知らせします。







Microsoft SQL Server 2008は、機密情報への不正アクセスに対する組み込み保護機能を備えた汎用ソフトウェアツールです。 セキュリティクラス5によると、情報への不正アクセスに対するセキュリティの指標は、セキュリティクラス1Gまでの自動システムを作成するとき、およびクラス3までの個人データ情報システムを作成するときに使用できます。
