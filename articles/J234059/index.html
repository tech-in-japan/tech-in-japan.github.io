Octa-core big.LITTLE SoC AllWinner A80に基づく新しいOptimusBoard開発者ボード





AllWinner A10プロセッサ上の最初のCubieboardバージョン1デバッグボードは2012年10月にリリースされ、続いてCubieboardバージョン2のAllWinner A20プロセッサがリリースされました。 Cubieboard 3またはCubietruck 。 以前のバージョンとは異なり、Cubietruckボードには、アンテナが内蔵されたWi-Fi + BTワイヤレスモジュールが追加で収容されていました。



NanoXionは、2014年初頭に、CubieBoard （Cubiebox）に基づく最初のコロケーションホスティングを開始しました 。

CES 2014で、Allwinner Techは将来のOptimus BoardベースのSoC AllWinner A80デバッグボードを発表しました。



Allwinner Techプロセッサ用の開発キットを作成したShenzhen Merrii Technology（別名WITS）は、 OptimusBoardを開発および導入しました。



A80 OptimusBoard -4つのCortex A15コアとbig.LITTLEモードで動作するCortex A7を備えた8コアSoC AllWinner A80で開発されました。



SoC AllWinner A80





OptimusBoardはAliExpressで345ドル 、Fedexによるロシアへの送料92.61ドルで購入できます。 まだ非常に高価ですが、すぐにコストが下がるはずです。



技術仕様

  • プロセッサー： AllWinner Ultra Core A80 4x Cortex 15、4x Cortex A7 big.LITTLE with PowerVR GC6230 GPU（64-threads）、およびOpenGL ES 1.1 / 2.0 / 3.0、Directx 9.3のサポート
  • RAM：2GB DDR3、プロセッサーは最大8GBをサポート
  • ユーザーメモリ：NANDメモリ8 GB、およびMicroSD最大32 GB
  • SATA：公式発表には情報はありませんが、ほとんどの場合あります。 SoCのすべての以前のバージョンがサポートされています
  • ビデオ：HDMI 1.4aから4K30、UHD / 4K H.264 / VP8 4Kx2K @ 30 fps
  • オーディオ：AC100コーデック、ヘッドフォンジャック
  • ネットワークインターフェイス：Wi-Fi 802.11 b / g / n、Bluetooth 4.0（AP6330）、およびギガビットイーサネット（GMAC）
  • USB：1つのUSB 3.0 OTG !!!ポート、2つのUSB 2.0ホストポート
  • カメラ：16MP MIPI CSI
  • デバッグ：UARTおよびJTAG
  • プログラミング：32ピンGPIOピン、シングルIR
  • オプション：電源およびリセットボタン、LEDインジケーター
  • 電源管理：AXP806、AXP809「スマートな電源管理スペシャリスト」
  • 電源：3Aで5V DC、またはバッテリー駆動
  • サイズ：135 mm x 70 mm


ボードには、電源アダプタとデバッグケーブルが付属しています（UART-> USB）。 ボードにはAndroid 4.4.2、Linux 3.4.39がプリロードされています。 これまでに、このボード用に準備されたOSの選択は終わりに近づいていますが、Allwinner TechはSoC AllWinner A80、5 つのオペレーティングシステムのサポートを約束しています ：

  • Android
  • Chrome OS
  • Ubuntu
  • Firefox OS
  • Windows RT（!!!）


おそらく、他のオペレーティングシステムのサポートにはさらに時間が必要であり、近い将来にサポートされる予定です。
