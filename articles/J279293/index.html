Laravel 5の言葉の数

フォームを作成するときに数字で数字を表示する必要がある場合があり、Laravelフレームワーク専用に設計されたDigitTextモジュールが役立ちます。



任意の言語で任意の数を処理できます









設置



モジュールのインストールは非常に簡単です。



1.コマンドを順番に実行します。 



composer require andrey-helldar/digittext composer update


2.コンポーザーを更新した後、サービスプロバイダーをファイル `config / app.php`に追加します。 



 Helldar\DigitText\DigitServiceProvider::class,


3.次に、同じファイルの「エイリアス」セクションで、ファサードを追加します。 



 'DigitText' => Helldar\DigitText\DigitText::class,


できた！



使用する



数値をテキストに変換するには、次の構成を使用します。 

 DigitText::text($number = 0, $lang = 'ru', $currency = false);




例： 

 DigitText::text(); DigitText::text(64.42); DigitText::text(2866); DigitText::text(0, 'ru', true); DigitText::text(64.42, 'ru', true); DigitText::text(2866, 'ru', true); /* * : *  *   *      * *   *    42  *       */




言語形式



執筆時点で、このモジュールは「ロシア語」、「英語」、「ドイツ語」などの言語形式での作業をサポートしています。



誰でもDigitTextモジュールの開発を支援できます。



モジュールはMITによってライセンスされています。
