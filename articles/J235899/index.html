Kickstarter No. 5の国内プロジェクトのダイジェスト

Ice-Pick Lodgeの驚異的なゲームMor.Utopiaのリメイク、スタイリッシュなメンズブレスレット、リサイクル素材で作られたダイオードテーブルランプ、タブレット用DVD、地獄での生活に関するアーケードゲーム、愛好家が作った潜水艦、永久運動機！



伝統的に、現在有効であり、完了まで少なくとも数日が残っている既知のKickstarterプロジェクトについてお話します。 注目すべきは、今回はデザインカテゴリの2つのプロジェクトのみが既に成功していることです。Moraのリメイクは、Kickstarterの国内デベロッパーから最も収益の高いゲームプロジェクトになると脅かされています。 しかし、他のプロジェクトは財政的な目標からはほど遠い。

以下に、各プロジェクトについて詳しく説明します。











病理学（マリンユートピア）



PathologicはIce-Pick Lodgeからの2番目のKickstarterプロジェクトです。 これに先立ち、チームはノックノックゲーム用に$ 40k +を調達しました。このゲームは今年Steamでリリースされ、非常に成功しました。 現在、「アイスアクシス」は、カルトゲーム「Mor.Utopia」のリメイクを開始しました。 男たちはただクレイジーな映画を持っています！ そして、彼らは当然スタッフの写真（Kickstarter編集者が好むプロジェクト）の選択に落ちました。



「疫病。 Utopia」は、PC（Windows / MAC / Linux）、PS4、およびXbox Oneでリリースされるオープンワールドサバイバルアドベンチャーストーリーです。 これはゲームのリメイク版で、2005年に最も非標準的なゲームプロジェクトとして賞を受賞しました。



最初の週末のプロジェクトは、必要な250,000ドルから148,359ドルを調達し、29日間続きます。







あなたのスタイルのアンカー-男性専用ブレスレット



あなたのスタイルのアンカーはスタイリッシュなメンズブレスレットですが、「キックスターターエディターズチョイス」カテゴリーに入った後、その料金は上がりました。 これがこの著者の2番目に成功したプロジェクトであることは注目に値します。 これに先立ち、彼は本物の革で作られたアクセサリーの生産のための資金をすでに首尾よく集めていました。



このプロジェクトは、たった$ 5,000の申請で39,542ドルを既に調達しており、さらに17日間続きます。







手作りのLEDランプT1



ランプT1は、2ワットのダイオードテーブルランプです。 80％がリサイクル素材で構成されています。 ちなみに、このプロジェクトをKikstarter編集チームに選出しても、手数料は大幅に減りませんでした。



このプロジェクトはすでに5,000ドルを目標として6,609ドルを調達しており、6日間続きます。







ゲームマスター-外出先で冒険をプレイ、作成、共有！



Game Masterは、一人で、または友達と一緒に完成できるゲームおよびダンジョンビルダーです。 これは、D＆Dをタブレットに転送する一種の試みのようです。 また、スタッフの写真も取得しましたが、レジを通過しました。



このプロジェクトは、必要な£130,000のうち£10,429のみを調達し、さらに18日間続きます。







ラップサム



Lapsumは、プレイヤーが地獄のようなトンネルを通り抜け、さまざまな拷問器具をかわす必要があるアーケードゲームです。 このプロジェクトと以前のプロジェクトは、Kickstarterのモバイルアプリケーションにとって非常に難しいことを完全に示しています。 また、ターゲットメディアにはすでにインディープロジェクトに関するニュースがあふれており、支持者はKickstarterのiOSおよびAndroidアプリケーションやゲームを特に好まず、PCやコンソールゲームを好んでいます。



このプロジェクトは、33,300ドルの目標でたった275ドルを調達し、さらに32日間続きます。







3EBC潜水艦



3EBC Submarineは職人のPavel Stepanovのプロジェクトです。彼は既に自分の潜水艦を半分まで完成させており、Kickstarterに水泳装置を完成させるために来ました。 このプロジェクトには、エンジンを備えた溶接フレームの3Dレンダリングと写真があります。 もちろん、自由時間の愛好家がそのような野心的なプロジェクトに従事しているのはクールですが、プロジェクトの視覚的な部分は両足に足を引きずっていて、料金を完全に反映しています。



このプロジェクトは、必要な£5,000のうち£13.00のみを調達し、さらに11日間続きます。







燃料-無料エンジン





燃料-無料エンジンは、私が知らない、笑う、泣く、永久運動機械のプロジェクトです。 著者は構造上の特徴を開示していません。それらは現在特許を取得しています。 しかし、それは質量次元と技術的特性につながります：

  • 設計は、発電機と燃料なしエンジンの2つの部分で構成されています
  • 寸法：ヨーロッパ-1 x 0.5 x 0.7 m。アメリカ-3.2 x 1.6 x 2.2フィート。
  • 重量：ヨーロッパ-最大20 kg。 米国-最大44ポンド。
  • 磁場は無害です（100％）。
  • ノイズレベル-最大40 dB。
  • 接続-単相および三相
  • 開発が完了するまでの推定時間は5〜6か月です。




そして、はい、まだ実用的なプロトタイプはありませんが、彼らはすぐにそれを組み立て始めます。



プロジェクトはこれまでに必要な30,000ドルのうち32.00ドルを調達し、さらに25日間続きます。
