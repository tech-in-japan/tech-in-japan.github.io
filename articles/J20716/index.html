ブラウザを再起動する必要はありません。開いているすべてのページでトピックを更新するだけです。 「返信」または「コメントを書く」をクリックすると、ボタンが表示されます。



オペラでは：

このフォルダーにスクリプトをダウンロードします。 ページを更新すると、スクリプトが機能します。

手順-SIMBLを入力し、フォルダ/ Library / Application Support / SIMBL / Plugins / GreaseKit.bundleに入力します。
