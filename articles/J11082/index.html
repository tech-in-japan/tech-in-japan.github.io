Mac OS X 10.4.10リリース

Appleは本日、 IntelおよびPowerPC Macのバージョンで利用可能なMac OS Xの次のアップデートをリリースしました。



UPD ：TUAWブログによると、10.4.10の更新とともに、新しいフレームワーク、つまりYahoo Syncがインストールされています。

フレームワークは/System/Library/PrivateFrameworks



あり、組み込みのYahoo! Syncアプリケーションが含まれています。 このアプリケーションが実行する機能はまだ明確ではありませんが、おそらくYahoo!の同期に使用されます。 およびiCal。





この更新には、次の変更/改善が含まれています。



Bluetooth

  • BluetoothヘッドセットがBluetoothデバイスのリストから削除された後、サウンド設定パネルで使用可能なデバイスとしてBluetoothヘッドセットを表示できるようになったとき、問題は解決しました。


サードパーティ

  • 次のカメラのRAWサポートが追加されました：Panasonic DMC-LX1、Panasonic DMC-LX2、Leica M8、Leica D-LUX 2、Leica D-LUX 3、Fuji S5 Pro、Nikon D40x、Canon EOS 1D Mk III。
  • 一部の画像が暗くなったり歪んだりすることがあるDNGの問題を解決しました。
  • Mathematica 6と64ビットMacとの互換性が改善されました。


USB

  • スリープから復帰した後のリモートコントロールの使用の信頼性が向上しました。
  • 外付けUSBハードドライブのマウントの信頼性が向上しました。
  • USB経由でIntel Macに接続するときにTomTom GO 910を検出する際の問題は解決されました。


その他

  • Control-Ejectの組み合わせを使用した場合のシャットダウンダイアログの表示に対する応答が改善されました。
  • DVカメラからビデオをコピーするときにフレームがドロップする問題は解決されました。
  • 以前のセキュリティ更新プログラムが含まれます 。
  • 一部のアプリケーションでの日付の計算に関する問題は解決されました。
  • 一部のアプリケーションでの小数の丸めに関する問題は解決されました。
  • ビデオメモリを充填するときに一部のテクスチャが破損する可能性がある場合、Motionの問題は修正されました。




10.4.9より前のバージョンからアップグレードするには、 コンボ更新を使用します。
