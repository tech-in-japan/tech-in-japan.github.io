QHeaderViewを変更する



知らなかった人のために、QtのこれらのテーブルヘッダーはすべてQHeaderViewです。 QTabBar ::タブと同じ状態を持つ1つのサブ要素::セクションがあります。



今回は、伝統から離れてグラデーションを塗りつぶすことにしました（はい、ここで可能です）。 これを行うには、構造qlineargradient、qradialgradient、qconicalgradientを使用します。 グラデーションはオブジェクト境界モードで指定されます。 グラデーションがレンダリングされ、その左上隅が（0、0）にあり、右下隅が（1、1）にある長方形を想像してください。 この場合の勾配パラメーターは、0〜1の小数として示されます。これらの値は、実行時に長方形の実際の座標に外挿されます。 境界ボックスの外側にある値を指定することができます（たとえば、-0.6または1.8）。

次のデザインを使用しました。 

 QHeaderView::section { background-color: qlineargradient(x1:0, y1:0, x2:0, y2:1, stop:0 #616161, stop: 0.5 #505050, stop: 0.6 #434343, stop:1 #656565); color: white; padding-left: 4px; border: 1px solid #6c6c6c; }




それが私たちが求めたものです。
