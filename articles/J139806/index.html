Tinyicon-Js上のファビコンのカウンター

Tinyiconは、ファビコンサイトを操作して新しいイベントに関する情報を送信するための小さなライブラリです。 キャンバスをサポートしないブラウザーの場合、カウンターはページのタイトルに表示されます。









使用する



使い方は簡単です。 接続するか書く 

Tinycon.setBubble(6);


このライブラリは、色の選択、フォント、カウンターの配置などのオプションもサポートしています。 

 Tinycon.setOptions({ width: 7, height: 9, font: '10px arial', colour: '#ffffff', background: '#549A2F', fallback: true });




サポートされていないブラウザーのページタイトルにカウンターを表示するフォールバックパラメーターに注目してください。



サポートされているブラウザ



Tinyconは次のブラウザでテストされ、動作します：

  • Chrome 15+
  • Firefox 9以降
  • Opera 11+
次のブラウザーでは、カウンターはタイトルにのみ表示できます。

  • Internet Explorer 9
  • Safari 5
