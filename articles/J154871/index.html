3. Zen + jQuery UI、DhtmlX、ExtJS、Twitter Bootstrap

ご存知のように、InterSystems Zenは、コンポーネントライブラリを拡張できるajaxフレームワークです。 既製のjQuery UIコンポーネント/プラグイン、DHTMLX、ExtJS、Bootstrap jsコンポーネントを使用して、コンポーネントまたは一連のコンポーネントを作成します。 コンポーネントの動作方法を示すサンプルZenページを作成します。

4. Eclipse-CachéObjectScriptのスタジオ

Caché/ Ensembleプロジェクト向けの本格的なIDEとしてEclipseを使用するアプローチを実装および/または実証するEclipse用プラグインを作成します。 実装する必要があるプライベートタスクの中で：

  • CachéObjectScript構文の強調表示
  • 構文チェック
  • クラスのコンパイル
  • Cachéクラスのコード補完
  • Cachéクラス/プログラムメソッド起動コンソール。
