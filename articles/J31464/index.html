7お使いの携帯電話の急流ロッキング。

携帯電話のインフラストラクチャの開発に伴い、携帯電話での急流の揺れがより重要になっています。 GPRS / EDGEを介してダウンロードする必要はまったくありません。億万長者がいる都市では、無料または安価なWiFiポイントが珍しくありません。





それで、私はあなたの注意を示します。



Symtorrent

第2世代または第3世代のSymbian OSの急流ロッキング。 複数のトレントの同時ダウンロード、OSの再起動後の継続的なダウンロードなどをサポートします。




トランスミッション



Mac OS X、Linux、FreeBSD、およびSolarisで実行されているトレントクライアントのクラッチ管理。 トレントをダウンロードした後、ファイルはWebインターフェイスを介して携帯電話に転送できます（写真はMac OS Xインターフェイスを示しています）。




uTorrent iPhone

iPhoneのWebインターフェイスを介したuTorrentプログラムの管理。




µTorrent mUI

インターネット接続のある携帯電話で動作します。 Opera Miniを介してuTorrentのWebインターフェイスを使用することをお勧めします 。 サービスWebサイトで機能します。 こちらからログインできます。



WickedPsyched iPhoneトレント

iPhone用の別のソフトウェア。コマンドラインを用意することで、linux-clientトレントロックを管理できます。 スタイリッシュ。 オタク向け。




ウィズビット

Symbian S60スマートフォン用。




Winmobileトレント

Windows Mobile 5およびWindows Mobile 6を実行しているすべてのデバイスで動作します。アプリケーションには費用がかかりますが、トレントダウンロードに必要なすべての機能を利用できます。
