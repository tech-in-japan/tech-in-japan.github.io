Eclipse PDT、WTP / Zend Studio for EclipseのjQuery

ここでは、お気に入りのEclipseのJavaScriptパースペクティブで、完全なドキュメントを使用してjQueryのオートコンプリートを取得したかったのです。 ネットを検索したところ、否定的なレビューと古いバージョンのためにインストールしたくないjQueryWTPのみが見つかりました。

したがって、私は自分でそれをしなければならなかった、そして結果はここにある：



要件：Eclipse、Web Tools Platform / Zend Studio for Eclipse / JSDocをサポートするエディター



PSもちろん、NetBeansやAptanaプラグインを使用するなど、他のjQuery接続方法もあります。 たまたまこの方法が選ばれたのです:)







Eclipse WTPのjQueryサポート（jQueryDoc-1.3.js）をダウンロード： Google Project



内部-jQuery 1.3.1の「ヘッダーファイル」。 公式ドキュメントに基づいてJSDocスタイルでコメントし、必要に応じて短縮およびスタイル設定します。

オートコンプリートは機能しますが、F2は助けを得るという素晴らしい仕事をします。



接続手順：



設定：JavaScript /インクルードパス/ユーザーライブラリで、[新規]をクリックし、「jQuery 1.3.1」という名前を入力します。 選択して、「Add .js ...」をクリックし、およそのファイルを見つけます。 それだけです！

プロジェクトでJavaScriptサポートを有効にし、プロジェクトプロパティ、JavaScript / JavaScriptライブラリに移動し、[ランタイムライブラリの追加]をクリックして、[ユーザーライブラリ]を選択し、選択して終了します。



凡例



一部のjQuery関数にはいくつかの使用方法があり、JSDocを使用して基本的に文書化されていません。ここではオーバーロードが便利です。したがって、いくつかの規則を導入する必要がありました。

1：クリック（）-関数を呼び出す最初のメソッド

2：クリック（コールバック） -2番目の呼び出し方法

引数名の中央にある$記号は、名前を付けるためのいくつかのオプションを示しています。 引数の最後の$記号はオプションです。
