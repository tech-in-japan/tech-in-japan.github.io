KPIでのOperaセミナーについて

昨日、キエフ工科大学でセミナー（または会議？）オペラが開催されました。 最初の「ホット」レポートはすでにここにありました 。 私は、警棒を握って、さらに詳しく話そうとします。







すでに知られているように、Navjot Pawera（製品マネージャー）、Ulyana Kumenova（流通マネージャー）、Ilya Shpankov（開発マネージャー）の3人がOperaから参加しました。







HaikonのEducampのように、プレゼンテーション全体がMS PowerPointで収集されるのではなく、CSSで収集されます-この最後に私が夢中になったとき、彼らはそれを「オペラショー」と呼びます。 同様にHaakonは本全体を書きました：XHTML + CSS => PDF、昨日彼らはこれについても言及しましたが、通り過ぎました。



私たちはモバイルインターネットとその中にページを表示する機能について話しました。 私の知る限り、Operaの主な視点は次のとおりです。 「Webはモバイル化されています！」そして、実際、Opera MiniとOpera Mobileは最も賢明なモバイルブラウザーと考えられています（Mobile FFはまだどこか近くにありますが、まだ表示されています） 。 さらに、Opera Mobileは、お金で販売されている唯一の製品です。 ですから、会社の発展の主な方向性は明らかだと思います。



実証済みのDragonfly + Scope-Firebugの類似物で、Operaに組み込まれていますが、リモートデバッグが可能です。



ナビゲーターは、アニメーション化されたPNGを約束し、最も一般的なXMLであることが判明したSVG（Scaleble Vector Graphic）のサポートを示しました。 同時に、SVG画像を使用した単純なJavaスクリプトの助けを借りて、アニメーションとインタラクティブの両方のすべてを実行できます。



描画にキャンバスドキュメントを使用する方法を示しました。 Java Scriptで直接おもちゃを開発するのはそのようなことです。:)一般に、このように進むと、Flashは完全に無関係になります。



Haikonが言及した新しい<video>タグに注目しました。 一般に、私はこの考えに完全に同意します。現在、メディアコンテンツを表示するには、ユーザーはプレインストールされたプレーヤーが必要です-これは間違っています。 そのため、Operaはメディアをページに直接埋め込み、ブラウザの方法を使用して表示することを提案しています。 同時に、すべてが非常に柔軟に表示されます-それは自由にサイズ変更され、反転され、フィルターが適用されます。 （同じJavaスクリプトだと思います）いずれにせよ、今でもFlashでこれを許可していません。



実際、これについてはメインプレゼンテーションが終了し、講演は「高等教育支援プログラム」自体についてでした。 だから：

  1. Operaウェブサイトでコーダーに正しいコードを書くように教えるために、「How to ...」に関する主要な開発者による記事でセクションが開きます。
  2. Operaは、世界中の教育機関（すでに何度も言及されています）の代表者を探して、学生と交流し、さまざまなイベントを開催しています。
  3. これらの代表を通じて、Operaは大学にブラウザの全ラインを完全に無料で配布します（ただし、Opera Mobile以外は無料です）。
  4. Operaは興味のある学生を夏の練習、インターンシップ、仕事に招待します。


それから質問がありました。 最も興味深いのは、「すべてが無料の場合、Operaにはどれくらいのお金がありますか？」:)さらに、Ilyaはこの質問がすべての会議で求められていると不満を漏らしており、プレゼンテーションに追加するかどうかをすでに考えています。 したがって、これはまったく秘密ではなく、さらに、関心のある人のすべての財務報告はウェブ上で利用可能です。



そして今、「黒魔術のセッションとその後の暴言」：

  1. 検索サービス。 つまり、組み込みのブラウザ検索では、Google、Yahoo、その他がOperaにtheir分の一を納めます。
  2. それでも、Opera Mobile。 しかし、小売業の収入はほとんどなく、主な収入は携帯電話メーカーの卸売購入です。
  3. 非伝統的なデバイス用のOpera。 つまり、ヘルプシステムなどのインターフェイスとしてのブラウザです。 イリヤは、「遅かれ早かれ、Operaが画面とキーボードを備えたどのデバイスにも搭載されることを願っています」と冗談を言っていました。


それだけです
