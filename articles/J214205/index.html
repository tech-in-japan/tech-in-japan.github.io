無料のマイクロソフト試験バウチャー

こんにちは

マイクロソフト試験のすべてのファンにとって、素晴らしいニュースがあります！

無料試験バウチャー74-409 Windows Server Hyper-VおよびSystem Centerによるサーバー仮想化





（バウチャーの有効期間は2014年6月30日までです）

www.microsoftvirtualacademy.comでの登録のプロモーション、またはまだ登録していない場合は、 登録できます （Need Live ID）

www.microsoftvirtualacademy.comにログインしたら、 www.microsoftvirtualacademy.com / offers / virtualizationsquaredoffer＃？fbid = n9BV8QW7dMgにアクセスしてください。



[無料バウチャーを取得]をクリックします。

その後、システムがバウチャー番号を生成し、それを電子メールに送信します（MVAへの登録中に表示されます）。





試験の登録は、プロメトリックテストセンター（www.register.prometric.com）から、またはn.orlova@stars-s.ru（Skype：stars-s.nata）宛てに書面で行うことができます。



バウチャーの数が限られていることに注意してください！



PS試験に登録するときに、「Habr from Habr！」と書いたすべての人に、試験前にチョコパイとお茶を無料で提供します:)



UPD。 このバウチャーは、プロメトリックのテストセンターで有効です。 注文するときは、試験に合格するのにより便利な国を選択します。 モスクワのロシアにあるオフィスであることを示していないことをおIび申し上げます。 この投稿の目的は、無料で試験を受ける機会について話すことでした。
