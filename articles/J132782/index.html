マイクロソフト認定プロフェッショナルになるには？ そして、なぜですか？

こんにちは、％ユーザー名％。 Microsoftからの次の認定試験MCITP：Database Developer 2008に合格したので、それほど大きくはありませんが（これまではMCTS、MCPD、およびMCITPのみ）、これらの試験に合格した経験を共有することにしました。



さらに、開発とプログラミングについて説明しますが、これは管理およびその他の認定（オフィス、シェアポイントなど）に完全に適用されます。



「試験の準備方法」という質問に簡単に答えると、次のように答えます。 コードを書いて、性交し ます （ マットのことは言い訳になりますが、歌から言葉を消しません ）



「証明書がもはや流行していないので、なぜそれが必要なのですか？」という質問に対する詳細と回答が必要な場合は、猫の下でお願いします。





なんで？


実際、認定試験に合格する理由はいくつかあります。

1.アパートや家に飾られている壁紙が気に入らない人は、美しいフレームワークで証明書のレイヤーの下にそれらを隠すことにします。

2.人は、対話やプロジェクトの成功ではなく、たくさんの証明書を配置することによって、自分がクールな専門家であることを他の人に証明しようとします。

3.人は問題を本当に理解し、「ショーのために」自分の虚栄心を楽しませたい。

4.その人は、証明書の助けを借りて、雇用主により良い印象を与え、大きな給料を求めることを望んでいます。

5.誰もがそれを手に入れたが、私は赤毛か何か？



これらの原因の組み合わせがより一般的です。



個人的には、最初は3つの要因と4つの要因の組み合わせがありました。 また、4番目の要因は二次的です。 証明書は、履歴書を読んで面接に招待するのを助けるために、候補者の最初のスクリーニングを通過するのをより簡単にするとさえ言います。



2009年10月の退屈な秋の夕方、すべてが仕事で終了し、締め切りが間に合わず、バグが修正され、スプリントの終わりまであと数日ですべてが始まりました。 そして、私たちは考えましたが、強度について自分自身をテストして、.NET Framework 2.0の試験に合格するのではないのですか そして、彼らは提出することにしました。 そして降伏した！ ところで、私がこれまでに受けた最も難しい試験です。



2年でMicrosoftから6つの試験に合格した後（実際、最初の後に-残りはこの推測を確認しました ）、私は、専門家としてではなく、これらの試験の受験と準備：

6.どのような状況でも、実際には、たとえばSQL Server Integration ServicesやMS SQL Server Agent向けに設計されたすべてのタスクに出くわすことはありません。 そして、試験の準備により、この知識を100つか2つの要約の形で体系化することができます （各質問は本質的に、このテクノロジーまたはツールの使い方を非常に簡潔に示す1つの論文です）。



そこで、「なぜ」という質問を見つけましたが、今は「どのように」質問が残っています。 「方法」という質問にはいくつかの文言があります。

1.準備方法

2.合格するための準備方法



準備方法

簡単に言えば、本質は簡単です：1年以上この方向で本当に作業できますが、この場合は、試験で影響を受ける領域を調べてメモリ内で更新するか、脳が青くなるまで特定の試験の準備をするためにスマートテキストを勉強する必要があります。



認定コースに行ったり、スマートな講義を聴いたり、実験室での仕事をしたりできます。 教師が良ければ、コース終了後すぐに試験を受けることができます（個人で確認済み）。



合格するための準備方法


テストの著者からの教科書に加えて、インターネット上で各テストの多くの「リゾルバ」を見つけることができます。 あなたはこれらのプログラムについて大声で戦うクラブとして話すことはできませんが、誰もがそれらについて知っています=）



服用方法


ここではすべてが非常に簡単です。最寄りの認定センターを探し、登録し、支払い、引き渡します。

モスクワについて話す場合は、 スペシャリストをお勧めします。 私はすべての試験にMSTUの近くの訓練施設「ラジオ」で合格しました。 N.E.バウマン。 広告のためにそれを考慮しないでください、しかし、彼らは本当にトップのクライアントに雰囲気と注意を持っています。 そして、彼らは常にクッキーを持っています！

ところで、これらの試験のコースを準備するためにそこに行くこともできます。



試験では、主なことは、神経質にならず、落ち着いて、質問のすべての言葉を注意深く読むことです！ 多くの場合、1つの単語の意味が回答の正確さに影響します。 急いで行く場所はありません。50〜55の質問の時間で十分です。英語がネイティブの場合は120分、ネイティブでない場合はさらに20分です。



試験で最も難しい問題は、C＃またはVB.NETという正しい試験言語を選択することです。



合格した試験と証明書の所持は正確に何をもたらしますか？


まず、試験の準備の過程で、よく知っている分野の知識を体系化します。 検査では、特に考えていないような些細なことに集中せざるを得ないことがあります。

第二に、私は本当に私が知らないそれらの瞬間を見つけます。 そして、私はそれらに対処し、それによって詳細を学びます。 たとえば、最終試験に備えて、NEWID（）だけでなくNEWSEQUENTIALID（）の存在についても学びました。また、hierarchyidタイプの外観とMS SQL Server 2008のその他のニュアンスについても学びました。

第三に、私は自分の虚栄心を楽しませて、自分ができると自分自身に言うことができます！

そして4番目に、私の履歴書では、筆記録へのリンクと証明書のリストを示さなければなりません。 これにより、簡単に履歴書の郵送段階を経て、面接に行くことができます。 ちなみに、私は証明書の可用性のために正確に私が興味を持っているお金の量で私の現在の仕事の仕事を得た。
