PyCon Russia国内料理と国際関係

「自分の人生について話すからといって、ロシアで逮捕されたくはありません。」これらおよび外国人が会議への参加を拒否するその他の理由。 カットの下-人生からのいくつかの物語。







PyCon Russia 2014がまもなく始まり、プログラムの準備が整い、講演者のチケットが購入され、ビザが発行されました。 スピーカーの強力なチームが再びあります。そのうち5人のpythonコア開発者は、アメリカ、カナダ、イギリス、オーストリア、ウクライナ、ベラルーシ、ロシア（モスクワ、サンクトペテルブルク、カザン、エカテリンブルク）から来ています。



しかし、率直に言って、今年はプログラムの形成が過去よりもはるかに多くの血で私たちに与えられました（そして、私たちはそれが簡単だと思った-あなたはPyCon 2013のポジティブな経験に訴えることができます）。 「高い感情」が思いがけなく邪魔された-ロシアの一般的な政治情勢。



PyCon 2014はそれぞれ6月2〜3日で、今年の第1四半期に講演者を招待し始めました。



そしてそれが始まりました...私たちは数十人（！）の人々と交渉していました。彼らはプログラム委員会のメンバーと昨年の外国人スピーカーによって推薦されました。 信じられないほどの数の失敗がありました。



ここで、例えば、ウクライナの有名な開発者からの手紙：

「こんにちは、

招待してくれてありがとう！

そして、あなたの立場に感謝します-ロシアには合理的な人々がいて、栄養士がその中にいることを喜んでいます:)

残念ながら、現在の状況のた​​め、私は近い将来ロシア連邦の領土に入る可能性を考えていません。

再びありがとう！」


私たちは怒っていましたが、さらに働き続けました。



海外からの講演者がさらに興味深いものになりました。 たとえば、米国市民、pythonコア開発者、us.pycon.orgの有名なスピーカーが答えたものは次のとおりです。

「私は長年ロシアを訪問することを夢見ていましたが、2013年に採択された現在の反ゲイ法のために、私は来たいです-残念ながらできません-それは厳しく観察されておらず、違反する可能性は低いことを知っていますしかし、私は危険を冒すことはできません、私は私の人生について私に話すためだけに逮捕されたくありません。

この法律が廃止されるとすぐに、本当に申し訳ありません-あなたに来て、話すことを非常に嬉しく思います。 「今年はこれが起こらないこと、そしてそれはあなたの力ではないことを知っていますが、残念ながら、そうではありますが、私は来ることができません。」



おお サイトに「Gay Friendly」と書くだけです。



ちなみに、このまさに法律は、何らかの理由でゲイだけでなくブロガーも怖がらせています。 コミュニティの別の有名人が私たちに書いているものは次のとおりです。

「PyCon Russiaのヘッドライナーになるように私を招待してくれたことを非常に嬉しく思います。 また、多くのTwistedファンがいることを強くお勧めします！ 数人のロシア人駐在員が現在私と一緒に働いています（すべてのTwistedユーザーです！）そして、あなたが非常に興味深いコミュニティを持っていることを理解しています。

しかし、ロシアでの最近の政治的出来事は、あなたの国に旅行するときの私の個人的な安全について考えさせます。 私にはブログがあり、いくつかの考慮事項は「同性愛のプロパガンダ」と解釈できます。 私は人気のブロガーであり、最近、ロシアで法律が発行され、訪問したブログはメディアと同一視されているという法律を読みました。 そしてもちろん、客観的には、ウクライナとの状況により、ロシアと米国の関係は緊張しています。」



ここにある。 私たちは耳を傾け、支援を求めてエカテリンブルクの米国領事にさえ頼みました-同胞を安心させるように彼に頼み、彼らはすべてのメディアが膨らんでいると言いました、それは実際ここで安全です。 しかし、この日、運が良ければ、不幸な出来事が起こります：エカテリンブルクのアメリカ総領事館の副領事、クリスティーナ・ヘイデン、そして5人のアメリカ人ミュージシャンがセロフの警察に拘束されました（これはスベルドロフスク地域の都市です）。 後で判明したように、彼らは事件のために拘留され、すぐに釈放されましたが、それでも領事はここで私たちのアシスタントではないことが明らかになりました。



一般に、非常に活発な国際生活がここで行われています-ロシアの中央部、ヨーロッパとアジアの国境にあります。



しかし、失敗のすべての原因がそれほど劇的ではありません。 あるヨーロッパ人は、「PyCon Russiaは行動規範をサポートしていない」ため、行かないと書いています。 行動規範は、主に性的差別や嫌がらせを禁止することを目的とした会議の行動基準です。



幸いなことに、会議での行動規則はロシア連邦の法律ではなく、ここで私たちは自分で決定を下すことができます。 良いスピーカーを獲得するために、 pycon.ru / 2014 / conference / cocを紹介しました 。 嫌がらせやセクシュアルハラスメントをせずに今度は試してみましょう。



ちなみに、過去の後、PyConは少なくとも1つの結婚したカップルを結成しました。主催者の女の子がスピーカーと結婚しました。 そして、これは会議の目標ではありませんでしたが、私たちはまだとても幸せです。



女の子といえば。 内部キッチンの別の面白い事実を投稿に追加します。意外なことに、私たちは-注目を集めました！ インドのpython開発者！ この少女は22歳で、初心者の開発者向けに1つのPythonライブラリに関するレポートを15分間読みに行きたかったのです。 見て！ 男性は行くことを恐れており、インドの若い女の子は申請書を送った。 （確かに、申請を却下し、経験豊富な開発者向けの会議を行っています。インドから15分間のレポートを提出するのは費用がかかります。）しかし、とにかく満足しました。



ところで、翌年、 アレックス・マルテッリと彼の妻のアンナ・ロヴェンスクロフトは私たちのところに来ると約束しました。 すでに来年-私たちは女性のニシキヘビの報告を聞く機会があります。 しかし、おっと、私はすでに行動規範を破っているようですので、そこで止めましょう:)



その結果、すべての変動にもかかわらず-私たちは勝ちました！ 私たちは1人のアメリカ人-ブライアン・カーティンを連れてくることさえできました。 これはPython Software Foundationのディレクターであり、次のことを伝えます。

•Python 3のディストリビューションの状況（意図したとおりにすべてがうまくいかなかった理由）

•Python 2.7が2020年末までサポートされることが既に公式に発表されている理由

•pythonコミュニティはどこに住んでいてどこに行くのか



さらに、会議プログラムには、Flaskの著者、コア開発者Twisted、CPython、Elasticsearch、Rackspace、Yandex、Wargaming、Headhunter、Jetbrainsなどの開発者からのレポートが含まれています。 ここにすべてあります： http : //pycon.ru/2014/program/content/



PyCon Russia 2014でお会いしましょう！ ロシアでは、このようなスピーカーに会える場所は他にありません！
