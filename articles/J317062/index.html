Stepikコンテストグローバルコンペティション

Stepik教育プラットフォームは、ITタスク作成コンテストの開始を発表します。 Stepikコンテストでは、生徒の個人的な推奨事項に基づいた独自のオンライン学習システムが実現します。 受賞者には最高$ 10Kの賞金が贈られます。









2015年、Stepikチームは、 適応型推奨システムの開発を開始しました。その目的は、知識とトレーニングのレベルに基づいて、各生徒の教育コンテンツと課題を個別に選択し、トレーニングの特定の段階で重要なことのみを学習することです。 適応コースの最初のプロトタイプは2016年に登場しました。これらは、Pythonプログラミング言語を教えるための2つのコース（ ロシア語と英語）です。



アダプティブシステムのさらなる開発とテストには、広範な知識ベースが必要です。これは、最高の教師、専門家、および経験豊富なStepikの学生が作成するものとして信頼しています。 競争の形式はこの問題を解決するのに非常に適しているように思われました。つまり、賞金を獲得し、IT教育の未来を形作ることに参加する機会です。



コンテストは6つのカテゴリで開催されます。



  • 理論計算機科学
  • 応用コンピューター科学
  • データサイエンス
  • Python
  • Java
  • Javascript


何をする必要がありますか？



コンテストのノミネートの1つで自分のトピックを選択し、20以上のタスクとトピックを明らかにするいくつかの理論的なレッスンを作成します。 これらはすべて、Stepikプラットフォームのコース形式で発行し、リンクを送信する必要があります。 コンテストの特徴：すべての資料は英語のみで受け付けています。



そして今、私たちは最も人気のある質問に答えて、競技の具体的な詳細を明らかにします。



1）締め切りは何ですか？



申し込みは2017年2月11日まで受け付けられます。つまり、レッスンを作成するのに2か月かかります。 その後、3月から4月に、受け取ったすべての資料が評価され、受賞者が発表されます。



2）著作権



著作権はタスクおよび理論的レッスンの作者にあります。Stepikプラットフォームでのコンテスト用に作成されたすべての資料は、Creative Commons Attribution – ShareAlike 4.0ライセンス（プラットフォーム上の他のオープンコースの場合）の下で無料で使用が許可されます。



3）教育コンテンツを作成するのに必要な形式は何ですか？



理論的なレッスンには、あらゆる種類の資料を含めることができます。録画されたビデオ、編集された要約、ドキュメントからの抜粋などです。



タスクはもう少し複雑です：1つのレッスンではトレーニングが適応的であるため（つまり、システムは彼の知識に基づいて学生に教育コンテンツをアドバイスします）、1つのレッスンで少なくとも1つのタスクを1つの複雑なタスクを収集する必要があります。



また、作成したすべてのマテリアルに特別なトピック（タグ）を付けるようにしてください。 プラットフォームで作成されたすべてのレッスンをリンクし、ユーザーがそれらをナビゲートできるようにすることが重要です。 より詳細な説明は、ヘルプセンターの特別な記事Stepik.helpにあります。



4）誰がどのように勝者を選びますか？



コンテストの一環として作成されたすべてのレッスンは、Stepikの学生（および全員）に評価のために渡されます。 評価は各レッスン（好き嫌い）をマークすることで行われ、それらの違いを計算し、このレッスンの視聴回数の割合を考慮します。 コンテストでは、タスクを含む20を超えるレッスンを作成できるため、公正な評価のために、各著者の20のベストレッスンを受講することに注意してください。



適応Stepikメートル法システムを使用して、特別なノミネートが選択されます。 著者は、次の場合に何が良いかを知ることが重要です。



  • 生徒はあなたの課題を見たときにだけコースから脱落しません。
  • タスクを見て、複雑すぎる、または単純すぎると言う人は多くありません。
  • ほとんどの人は最初の試行では解決できません。


5）コンテストで獲得できる賞品は何ですか？



このコンペティションでは複数の賞金が用意されています。1位は10,000ドル、2位は5,000ドル、3位から5位までは2,000ドルです。 また、特別なノミネートも発表します。コンペティションの6つの分野のそれぞれで最高のレッスンを提供するために2,000ドル。
