モスクワのハッカソンAPPS4ALL

こんにちは、Habr！







12月13日と14日にロシア最大のハッカソンAPPS4ALLとCISがモスクワで開催されることをお知らせします！



Nagatino's Coworkingの1日と2日以内に、エキサイティングなプログラミングマラソンが開催され、モバイルアプリケーションまたはWebアプリケーションが作成され、70以上のチームが参加します。 参加者（プログラマー、デザイナー、起業家）は、オープンデータの招待された専門家、パートナー企業の代表者、メンター、投資家、市場の専門家からアドバイスを受けます。



Hackathon APPS4ALLは2段階で開催されます。 開始時には、参加者はhackathon.apps4all.ruサイトにオンラインで登録し、チームの形成で終了することが期待されています。 第2段階では、形成されたチームとハッカソンサイトで直接チームを見つけた参加者がマラソンに参加し、自分のアイデアを実現します。



ハッカソンへの参加は完全に無料です-イベントWebサイトで登録するだけです。



Hackathonパートナーは5つのノミネートを提示しました。



1.「 Samsung Mobile SDKを使用して作成されたモバイルアプリケーション。」 ハッカソンに登録するときは、アプリケーションの作成時にこのSDKのどれを使用するかを記述する必要があります。



2. Samsungの「 Tizenプラットフォームでのモバイルアプリケーション」 。 ところで、すべてのハッカソン参加者はTizenデバイスを手に入れることができます！ さらに、サムスンのエバンジェリストとチャットし、多くを学ぶ機会があります。



3. Microsoftの「Windows Azureのバックエンドを備えたモバイルアプリケーション」 。



4. Intelの「Androidx86用アプリケーション」 。



5. Googleの「Googleテクノロジーを使用したアプリケーション」 。



ハッカソンAPPS4ALLでお待ちしています-じゃあね！
