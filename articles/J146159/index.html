Microsoft Surfaceタブレットはモバイルデバイスを変更できますか？

Microsoft は 2日前にMicrosoft Surfaceを紹介しました。 この名前は私たちの多くによく知られていますが、まだ完全に新しい製品に属しています。 簡単に言えば、Microsoft SurfaceはWindows 8タブレットですが、実際にはそれ以上のものです。 SurfaceがiPadを破壊することはまずありませんが、Microsoftを復活させることができます。







SurfaceはMicrosoftによって完全に開発されています。 これは、Microsoftオペレーティングシステムで実行されるDellまたはHPのタブレットではありません。 同社は、Windows 8を最高の側面からデモンストレーションするために明確に考案しました。 タブレット自体は素晴らしいようです。 iPadと比較すると、明確なライン、I / Oポート、通気口があり、実用的です。 同時に、おもちゃというよりも真剣なタブレットのように見えます。



それでも、鉄はそれほど重要ではありません。 覚えてる？ 仕様だけが廃止されました。 マイクロソフトは今日、Surfaceの技術的特性を単にリストすることに限定されず、Surfaceの機能と設計を賢明に賞賛しています。 シーンから何度も、それはPCのように聞こえた。 もちろんこれはタブレットですが、PCでもあります。仕事、パワー、退屈なデスクトップソフトウェアに関連する略語です。 Microsoftは、Windows 8を成功に導くために、間違いなくこのタブレット、またはPCを作成しました。 または、言い換えれば、Microsoft自身がタブレットを開発したのは、技術的な機器を提供するパートナーを信頼していないためです。



SurfaceのチーフデザイナーであるPanos Panayは、プレゼンテーションで次のように述べています。 MicrosoftがWindows用のハードウェアを作成したことはありません。 決して。 しかし、今では、従来のPCの市場が背景に消えてWindowsを引き寄せているとき、Microsoftはその基準を改訂し、新しい勢力で最前線に駆けつけました。 私はまだ個人的にSurfaceを手に入れることができませんでしたが、これはDell Adamo XPS以来手に入れたい最初のWindowsデバイスです。



私はiPadが大好きです。 私は毎日それを使用しています。Logitechの超薄型キーボードのおかげで、非常に生産的に作業していますが、まだその可能性の約80％しか使用していません。 タブレット形式が好きです。 iPadをMacBook Airに変更するのは、タッチスクリーンデバイス用に構築されたアプリが原因だったのではないかと思いました。 しかし、エイリアンブルーアプリのおかげで、iPadでRedditをさまようことが好きです。 Flipboardは、他の方法では得られない大量の情報を提供してくれます。 そして料理のアプリケーションは私の神ですが、私はそれらをただ崇拝します。 ただし、iPadを使用してTechCrunchで記事を作成および公開することはできません。 そしてSurfaceでは-できます。



ラップトップは、創業以来モバイルコンピュータ業界を支配してきました。 タブレットは、コンピューティングプラットフォームが限られているため、生産性の高いデバイスであることを証明する必要があります。 しかし、これはWindowsです。 オフィスで。 タブレットで。 驚くほどの取り外し可能なキーボード付き。 あなたはそれらを好きでも嫌いでもかまいませんが、WindowsとOfficeは物事を成し遂げる方法を知っています。



初期のユーザーはSurfaceを称賛しています。 テクノロジー愛好家は、クールな製品の感触とシステムの速度が大好きです。 さらに多くの空白スポットがありますが。 Surfaceの価格については何も報告されていません。 キーボードはどのように機能しますか？ 5時間の連続使用後、タブレットはどうなりますか？ これに関して、疑問が生じます-バッテリーはどれくらいですか？ 3G / 4Gがありますか？ 起業家に適していますか？ そして、最も重要なこととして、開発者はMetro用のアプリケーションを作成したいと思うでしょうか？



iPadの利点は、App Storeでホストされている多数のサードパーティアプリケーションがあることです。 Appleは、クレイジーな数の開発者をプラットフォームに誘い込み、利益と公平なアプリケーション配布環境を約束しました。 Microsoftは、Surfaceのアプリケーション配布環境をゼロから作成する必要があります。 ただし、タブレットはWindowsで動作します。 これは、地球上で最も一般的なオペレーティングシステムです。 したがって、この初期段階では、手遅れになる前に、開発者にMetroの成功に参加するようアドバイスします。 Surfaceがクラッシュした場合でも、他のメーカーのWindows 8タブレットが多数表示されます。



SurfaceはiPadの真の競争相手ではなく、設計標準として、またはマイクロソフトのイメージデバイスとして興味深いものです。 今年の終わりに向けて、最初のARMモデルがリリースされます。おそらく400ドルから600ドル程度で、x86モデルは約1,000ドルで販売されます。SurfaceはおそらくiPadを追い越しませんが、 HP、Dell、Lenovo、Acer、Asusの明確な基準を設定しています。 ファッションデバイスとして、主にシボレーがコルベットをリリースするように、MicrosoftとWindows 8に注意を引くように設計されています。



MicrosoftはWindows XP以降、クールなものをリリースしていません。 Windows 7はWindows Vistaに取って代わりました。 モバイルWindowsの運命は、苦しんでいるフィンランドの会社にかかっています。 Zuneが好きな人はいませんでした。 Bingは単なるクローンです。 Hotmailを使用している人はいません。 私はまだWindows Liveの利点を理解していません。 同社の唯一の成功したプロジェクトは、7年前にリリースされたゲームシステムです。 過去10年間、Microsoftは生き残りました。 Windowsが「キラー」Surfaceデバイスを必要とするのと同じように、Microsoftは顧客からの信頼の信用を切望しています。



ほとんどの人と同じように、私は状況の結果を見ているだけです。 私はランダムな会社の勝利に盲目的に喜んでいる無知なファンではありませんが、消費者が利益を得る限り、どの会社がモバイルコンピューター戦争に勝つかは気にしません。 過去数年間で、Appleはいくつかの優れたデバイスを作成してきました。 そして今、MicrosoftはSurfaceと共に、賞賛に値する一流のデバイスも持っています。
