芸術としてのマーケティングについて書いている著名なジャーナリストであるテリー・オライリーは、 スティーブ・ジョブズの死について素晴らしい記事を書いています。 著者の親切な許可を得て翻訳を読むことを提案します。







Jobsに同情する10の理由



私は水曜日の夜にトロント交響楽団のコンサートで、彼の死のニュースがホール全体に広まったときでした。 ほとんどの視聴者は、15分以内に、何が起こったのかが分かるまで音楽を聴くことができるとは考えていません。



彼は私たちの世代の最大のマーケティング担当者でした。 消費者が切望する製品を作成する能力だけでなく、 それらを正しく提示する能力。



かつて誰かは、ジョブズが誰よりも「欲望」の概念をよく理解していると言っていました。 これは、私が正直に言って、彼が作成したほぼすべてのものが欲しかったからだと思うからです。私はすべてのApple製品を持っています。



スティーブ・ジョブズについては多くのことが言えますが、個人的に私を驚かせる彼に関する10の事実があります。



10.彼はフォーカスグループを使用したことはありません。ヘンリーフォードは次のように引用しています。



9.彼は1つの広告代理店-信じられないほどのTBWA / Chiat / Dayに忠実でした。 彼はチーフクリエイティブリーダーのリー・クロウに特に忠誠心を示しました。 （彼らの関係を理解するために、「 Dynamic Duos 」と呼ばれる説得の時代のエピソードを聞いてください。）



8.取締役会全体が彼のショーに断固として反対したとき、彼は1984年の映画の彼自身のポケットから支払うことを申し出ました。 本能は彼を失望させませんでした：彼はこれがこれまでで最も有名なコマーシャルになるだろうと言って、彼は正しいと判明しました。



7.はい、彼はエレベーターで従業員を解雇できるという事実で有名でした。 しかし、彼の同僚の一人が言ったように、「あなたのIQが、あなたが話すすべての人の中で最も高い場合、それはいつもあなたを困らせます」。



6.彼は午前中にオフィスに行き、「誰もが野原にいる！」と叫び、そして彼のすべてのデザイナーを博物館に連れて行き、彫刻の1つについて小さな詳細だけを見せました。 その後、彼らはバスに戻ってオフィスに戻りました。 彼にとって、神は細部にいた。



5.彼は特に1998年のMacで「正しい」色を作る方法を見つけるためにChupa Chups工場に行きました。



4. iPodでの記者会見で、彼はそのシンプルさを次のように自信を持って説明しました。「プラグイン、すごい、それだけです。」



3. 11年間の亡命後、彼がアップルに戻ったとき、彼は見たものにとても恐ろしく、1つの象徴的なものを除いて、自分のすべての株を売りました。



2.ペプシCEOジョン・スカリーをAppleに誘い込もうとしたとき、彼はたった1つのフレーズでこれを達成しました。「甘い水を売りたいですか、世界を変えたいですか？」



1. TBWA / Chiat / DayがJobsにアインシュタイン、レノン、ガンジーなどの世界の偉大な思想家をフィーチャーした「 Think Different 」広告キャンペーンを見せたとき、ジョブズはSteve Jobsだけを拒否しました。



この決定においてのみ、彼は間違っていました。
