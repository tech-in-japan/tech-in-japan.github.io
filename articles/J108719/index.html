Windows Mobile 6.5 Highscreen Nanoコミュニケーター

最近、Highscreen Nanoコミュニケーターが私の手に落ちました。 外部的には、デバイスは見栄えが良く、HTC Touchに似ています。 シルバートリム付きブラックケース。 小さいサイズ（103x54x14.1）。



OS Windows Mobile 6.5はボードにインストールされ、「デスクトップ」Spb Mobileシェルのシェルも標準のシェルの代わりにインストールされます。これもプリインストールされたソフトウェア-SPBキーボード、SPBモバイルTVからです。 ファームウェアに縫い付けられているため、ハードリセット（工場出荷時の設定に戻した）後でも再度インストールされます。







製造元がプリインストールされた高品質のソフトウェアを心配していたという事実は、特に予算のコミュニケーターにとっては非常に良いことです。 私の個人的な意見では、もう1つ不足しています。 また、標準の「ダイヤラー」に代わるものを事前にインストールしておくと便利です。これは、通話とアドレス帳のプログラムです。 これは頻繁に使用されるものであり、WinMobileの標準的なものは、たとえばPhoneEx、Spb Mobile Shell、または類似物には明らかに到達しないためです。 Spb Mobile Shellは最高のシェルプログラムの1つです。 また、設定からアドレス帳まで、必要なほとんどすべての機能が含まれています。









ソフトウェア部分から、キーボードも交換されました。 代わりに、キーボードは再びSpb Mobileからのものです。 組み込みの方がよく、ボタンが大きくなっていることに注意する必要がありますが、それにもかかわらず、ランドスケープ（ランドスケープ）が広がっている場合にのみボタンを通過できます。







最も安価なアナログが260ドルから始まることを考えると、コミュニケーターの価格はWindows Mobile 6.5のコミュニケーターの場合と同様に非常に快適です（230ドル）。



コミュニケーターには、かなり高速なプロセッサー（Marvell PXA300（624MHz）+ NXP5209、RAM（RAM）の容量-128 Mb、内蔵フラッシュドライブ（ROM）-256 MB）が搭載されています。 これは、Windows Mobile 6.5とサードパーティソフトウェアの両方がユーザーにとって快適な速度で動作するのに十分です。







ネットワーク機能のうち、Highscreen NanoにはBluetooth（Bluetooth 2.0 + EDR）およびWiFi（802.11b / g）があります。







電話は予算が限られているため、3Gサポートを必要とする価値はありません。 したがって、電話はGSM周波数でのみ動作します。 GPSレシーバーとモーションセンサーの存在に喜んで驚いた。 Highscreen Nanoには、一般的なSiRF Star IIIチップに基づくGPS受信機も装備されています。







ナビゲーションソフトウェアはプリインストールされていませんが、Yandex.MapsまたはGoogle Mapsに簡単に置き換えることができます。 必要に応じて、NaviTelやiGoなどの追加のナビゲーションソフトウェアをインストールすることもできます。 AGPSがサポートされており、セルラー基地局およびインターネットからのデータに基づいて、衛星を検索する時間を大幅に短縮します。 もちろん、FMレシーバーもあります。これは、パッケージに含まれている有線ヘッドセットを接続するときに機能します。







回転センサーもあります。







電話の画面は非常に標準的です-320x240 2.8 "QVGA。2Mカメラがありますが、フラッシュとオートフォーカスはありません。ほとんどの低予算の携帯電話では、カメラは非常に平凡に動作します。これに関しては、Highscreen Nanoも例外ではありません。手元にカメラがないときの写真。







パッケージには、ソフトウェアディスク、取扱説明書、追加のスタイラス、有線ステレオヘッドセット、同期用ワイヤー、電源、ケースが含まれています。 これは、デバイスを快適に動作させるのに十分です。 ヘッドセットには3.5mmジャックがありますが、これは欠点と考えることができます。 それでも、Highscreenで使用されているminiUSBコネクター付きの交換用ヘッドセットは見つかります。
