10名

今日、革新的な技術で市場に参入するだけでは不十分です。 「あなたが船を呼ぶと、それは航海します」-子供の頃から誰もが知っているフレーズ。 おそらく、大企業が製品の名前を慎重に選択する理由は、サウンド、スペル、グローバルな視聴者の知覚の妥当性、著作権の問題など、膨大な数の要素を考慮する必要があるためです。 今日は、世界的に有名な10のブランドがどのように名前を付けたのか、そしてそれらのいくつかの意味についてお話します。 何かが興味深く、驚くべきもので、かなり予測可能なものになるでしょう。厳密に判断しないでください。



Red Hat Linux：豊富なサブテキスト名



共同創設者のボブ・ヤング（画像中） は 、「赤い帽子」の出現についていくつかの説明をしました 。1.レッドハットは、西洋の歴史において解放と権力闘争の象徴である赤に敬意を表してその名前を得ました。 2. 2人目の共同創業者であるマークユーイングは、大学で祖父の赤いラクロスの帽子をかぶっていて、技術的な問題と機器の取り扱いの経験を解決するアプローチで知られていました。 質問や問題があった人は皆、赤い帽子をかぶって男のところへ行きました。 3. Ewingは、彼の最初のソフトウェアプロジェクトをRed Hat 1、Red Hat 2などと呼びました。 そのため、Linuxプロジェクトが登場したとき、単にRed Hat Linuxと呼ばれていました。 物事はうまくいった:)



Mac OS XとBig Cats



最も有名なアップル製品の1つである有名なオペレーティングシステムは、明らかにラテン語の「10」のように「X」という文字を受け取りました。 その理由は明らかです。OSの10回目のリリースでした。 残念ながら、多くのAppleファンにとって、ほとんどの人はまだ数字を「X」と読んでいますが、最近では人気が高まっています（X-Men、X-Menなど）。 さらに興味深いのは、リリースの名前の「大きな猫」​​の年表です。最初はチーター（チーター、10.0）、次にクーガー、ジャガー、パンサー、トラ、そして最後に「小さなヒョウ」でした-現在のバージョンです。 Snow Leopard（10.6）はすでに発表されており、次にlynxとcougarが順番に来るという噂があります。



ウィキペディア-聞いたことを書く



Wikipedia自体によると 、「Wiki」、共同（UGC）Webサイトを作成する技術、および「百科事典」-厚い本から、言語汚染（単純な用語では「複合語」と呼ばれる）の結果としてその名前を得ました。勉強しながら、誰もが恥知らずにメモをコピーしました。 今日、子供と大人は百科事典の代わりに恥知らずにウィキペディア自体をプラグインします。



Android-隠された、しかし幻想的ではない意味を持つ



Googleによって開始されたコンソーシアムの名前-Open Handset Allianceは、「オープン戦略」と「フリーソフトウェア」への直接の参照を提供します。 ただし、Androidブランドが2005年にGoogleに買収されたことを知っている人はほとんどいません。2008年までのすべての作業は秘密のベールの下で行われました。 この会社が「携帯電話向けのソフトウェア開発」に従事していることしか知られていませんでしたが、Googleはこれを有名な商標だけでなく、搭載されているすべてのAndroid携帯電話の優れた名前にしました。 「-何がありますか？ -Android。 「ああ、わかりました。」-私自身も、このような対話を繰り返し目撃しています。



ThinkPad-シンプルさが勝つ



IBMラップトップの有名な製品ラインは、1992年に導入されました。 一般向けに発表する準備が整った最初のコンセプトはすでに準備されていたという事実にもかかわらず、ラップトップと呼ばれるものについて社内で深刻な議論がありました。 IBMはシンプルさを追求しており、一般的な評議会では、ThinkPadという名前が多数派によって選ばれました。 ただし、各IBM製品には数字のマークを付ける必要があるため、最高の企業ランクはこの名前を高く評価しませんでした。 さらに、適切な質問が生じました-ThinkPadは他の言語にどのように翻訳されますか？ しかし、最初のThinkPadを設計したIBM従業員の主張のおかげで、ラップトップはそのように命名されました。 2005年にレノボの事業のこの部分を売却する前、しかしその後も、ThinkPadの伝統は続いています。



Windows 7-アクションのマジックナンバー



Microsoftの次のOSは別の明白な名前ではないかと噂されていますが、Windows 7は一般ユーザーの間で愛されているようです。 しかし、少し振り返って、この人気のあるOSの以前のバージョンであるVistaを思い出してください。 この名前は、すぐに最も快適な形容詞ではなく同義語になります。 Windows 95？ 合理的な質問をするかもしれません-レドモンドにはファンタジーはありませんか？ Windows XP？ ほとんど何も良い。 Mike Nash は、Windows 7を次の言葉で発表しました 。「これはWindowsの7番目のリリースです。したがって、Windows 7には明確な意味があります。」 明らかに？ かなり。 さらに、今やマジック7の力に頼る機会があります。おそらく、マイクロソフトに100万以上の「幸運」をもたらすでしょう。



Twitter-デジタルTwitter



Twitterの共同作成者であるBiz Stoneは、2006年にJack Dorseyが作成したアプリを見て、2人が鳥同士がどのように話し合うかについて考えました 。「短い情報のバースト-鳥のさえずりが聞こえます。 短いパッセージでは素晴らしいツイッターを作成しますが、多くの場合でもこのノイズを止めることはできず、最も重要なことは、鳥のさえずりのノイズとnoise騒に断固として反対する人は一人もいないということです。」 その後、ストーンは「twttr」という単語を思いつき、チームの他のメンバーが母音を思いついた。 今では、Twitterがテクノロジーと情報の世界を作ったという事実を過小評価することは困難ですが、当初はささいなTwitterでした。



Firefox：シンプル-気に入っています



長い間あなたの記憶に残っていて、重要なことにまだ利用可能な製品の名前を選ぶことは、従業員の大企業がMozillaに集まったということです。 元のバージョンは「Firebird」という名前でした（これは「firebird」だけでなく、アメリカで非常に人気のある車を指すこともあります）が、同じ名前の別のオープンソースプロジェクトのために、このオプションは却下されました。 Firefox、レッサーパンダは、少し後に思い浮かびました。 なんで？ 「覚えやすい。 いいですね。 これはユニークです。 気に入っています。」 そして、最高のものは何ですか？ これは絶対にユニークな名前です-Mozillaの前に誰もこの名前を使用したことがありません。



BlackBerry-スイートアディクション



カナダの企業であるResearch In Motionは、2001年に有名なレキシコンブランディングオフィスに問い合わせて、ワイヤレスメールリーダーの名前の作成と選択を支援しました。 研究によって判断すると、この言葉は多くの人々（企業の神経）の血圧を上昇させるため、「電子メール」にとらわれないようにアドバイスされたのは、レキシコンででした 。 代わりに、彼らは喜び、良い気分、平和感を呼び起こす言葉について考えました。 デバイスの小さなボタンがベリーの種のように見えると誰かが示唆したとき、Lexicon Brandingは、スイカ、イチゴなどのベリーの名前から単語を追加し始めました。 既存の名前を使用するというアイデアを放棄したため、このデバイスは単に「ブラックベリー」と呼ばれ、これまでのところ、すべてのBlackBerryデバイスはデフォルトでその色になっています。



iPod-取り外し可能なコンパートメント



AppleでのMP3プレーヤーの開発中、Steve Jobsは会社の戦略について多くのことを話しました。「Macは他のすべてのガジェットのハブです。」 新しいデバイスの名前を手伝うために雇われたフリーランスのコピーライターであるVinnie Chiecoは、ブレーンストーミングプロセスでこの考えに依存していました 。 そのとき、彼は宇宙ステーションとの類推を思いついた-それを取り外し可能な宇宙船に置いておくことはできるが、燃料補給のために戻らなければならないだろう。 2001年、彼が最終決定を下したのはポッドです。 まあ、そのとき、Appleはすでにすべてに「i」を追加する習慣になっています。 今でも子供たちが知っていることが起こった。
