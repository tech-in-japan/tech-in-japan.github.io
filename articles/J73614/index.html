私のGoogleリーダー

私はとても怠け者です。 そして、他のナマケモノのように、私はあなたがより少なくすることができて、より多くを得ることができる小さな肉焼きが本当に好きです。 時間の経過とともに私にとって最も有用な買収は、RSSとGoogle Readerテクノロジーになりました。 サイトをさまよい回るのに多くの時間を費やすのではなく、あるプログラムのウィンドウを開いてそこに必要なものをすべて表示することができました。 LiveJournal、Habrahabr（申し訳ありませんが、Facebookはニュースフィードブロードキャストをオフにしました）、サービスからの通知、新製品のリリースに関する通知（email2rss）、お気に入りのグループの新しいアルバムのリリースに関するニュースなど、私は多かれ少なかれ常に読んでいるすべてをGRにドラッグしようとします。 d。 など

賃貸アパートでさえ、RSSを使用して検索されます-cian.ruを解析し、私のパラメーターに従って新しい広告を探しています。 ）



しかし、これが「通常の」GRで常に私を殺したものです。これは、（特に垂直方向に）どれだけの空きスペースが費やされるかです。 ロゴがどれだけの空き容量を消費しているか、検索、「サブスクリプションを追加」ボタンを見てください。





そして、垂直ピクセルは800だけです。



どうする？





その過程で、FF： Better GReaderの興味深い解決策に出会いました 。 いくつかの役に立たないことに加えて、 Absolutely Compactを含むいくつかのスキンの選択をGoogleリーダーに提供しますが、ほとんどすべてが必要です。





さて、なぜ画面上部のリンクバーを削除するのですか？ そして、記事のタイトルのフォントとサイズを変更しますか？ そして、このオプションの検索バーはなんとなくぎこちなく見えます。



ロゴを削除し、「サブスクリプションを追加」ボタンを押し、検索バーを移動します。 残念ながら、そのような機能を提供するスクリプトはありませんでした。 GreaseMonkey チェッカーを拾わなければなりませんでした。 数晩、友人の助けを借りて、私のGoogleリーダーと呼ばれるスクリプトが作成されました。 彼は、リストされている3つのことと、それらだけを行います。 私のバージョンでは、これがGoogleリーダーの外観です。





必要に応じて、（Firefox）をインストールできます。

  • クリックしてください グリースモンキーをインストールする
  • クリックしてください スクリプトMy Google Readerをインストールするには


以上です。 それが少し役立つことを願っています。 ）



PS：念のため、ソースコードを提供します。

// ==UserScript==

// @name MyGoogleReader

// @namespace http://denis.fayruzov.ru

// @include https://www.google.*/reader/*

// @include www.google.*/reader/*

// ==/UserScript==



( function () {

var css = '@namespace url(http://www.w3.org/1999/xhtml);' ;

css += '#main, #settings-frame, #settings { top:2.1em !important; }' ;

css += '#logo-container, #lhn-add-subscription-section, .gbh { display: none !important; }' ;

css += '#search { position: inherit !important; }' ;

css += '#search-input { width: 150px; }' ;



if ( typeof GM_addStyle != 'undefined' ) {

GM_addStyle(css);

} else if ( typeof addStyle != 'undefined' ) {

addStyle(css);

} else {

var node = document .createElement( 'style' );

node.type = 'text/css' ;

node.appendChild( document .createTextNode(css));

document .getElementsByTagName( 'head' ).appendChild(node);

}



var init = function () {

var moving_node = document .getElementById( 'search' );

if (moving_node == null ) { setTimeout(init, 100); }

else {

var target_node = document .getElementById( 'viewer-top-controls' );

var brother_node = document .getElementById( 'viewer-details-toggle' );

if ((moving_node != null ) && (target_node != null ) && (brother_node != null )) {

var parent_node = moving_node.parentNode;

parent_node.removeChild(moving_node);

target_node.insertBefore(moving_node, brother_node);

}

}

}

init();

})();
