ASUS Vivoタブ



ASUS Vivoタブ（名前はラテン語の「ライブ」に由来します）には、Intel Atomプロセッサ、2 GBのシステムメモリ、64 GBのユーザーメモリが搭載されています。



「トランス」と同様に、タブレットにはドッキングステーションが装備されており、USBポートを2つ追加し、バッテリーを追加することでバッテリーの寿命を延ばします。



Vivoタブには、1366 x 768ピクセルの解像度の11.6インチSuper IPS +ディスプレイとWacomデジタルペンをサポートするマルチタッチインターフェイスがあります。 タブレットの厚さは8.7 mm、重さは675 gです。2つのカメラがあります。前面の2メガピクセルと、オートフォーカスとLEDフラッシュを備えた背面の8メガピクセルです。 内蔵のNFCセンサーを使用すると、他のNFC互換デバイスに接続できます。



ASUS Vivo Tab RT



ASUS Vivo Tab RTのハードウェア構成には、統合グラフィックスコア、2 GBのシステムメモリ、32 GBのユーザーメモリを備えたクアッドコアNVIDIA Tegra 3プロセッサが含まれています。 Windows RTオペレーティングシステムを実行しています。 その重量は520 gで、ケースの厚さは8.3 mmです。 Vivoタブと同様に、キーボード、タッチパッド、USBポート、オプションのバッテリーを備えたモバイルドックで使用できます。 1366 x 768ピクセルの解像度を持つ10.1インチSuper IPS +スクリーン。 古いモデルと同様に、Vivo Tab RTには2台のカメラとNFCセンサーが装備されています。



両方のタブレットは、Windows 8のリリース後すぐに利用可能になります。

アスースタイチ



蓋を開けると、TAICHIはフルサイズのキーボードとタッチパッドを備えた他のウルトラポータブルノートパソコンのように見えます。 ただし、蓋の背面には別のディスプレイが取り付けられており、蓋を閉じると、ラップトップはマルチタッチスクリーンとスタイラスサポートを備えたタブレットコンピューターになります。 さらに、両方の画面が同じハードウェア構成に接続されているという事実にもかかわらず、それらは互いに完全に独立して動作し、2人のユーザーが同時にTAICHIで同時に作業することができます。 彼らは、デバイスがASUS ZENBOOKと同じくらい軽くて薄いことを約束します。 第3世代のIntel CoreおよびSSDドライブがパフォーマンスを担当します。







ASUSトランスブック



ASUS Transformer Bookは、タブレットの「トランスフォーマー」の伝統を受け継いだラップトップです。キーボードからディスプレイを切り離すだけです。 Transformer Bookのハードウェア構成には、第3世代Intel Core i7プロセッサ、ソリッドステートドライブ、またはハードドライブが含まれます。
