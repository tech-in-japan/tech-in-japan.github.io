IdeaPad Z570-完璧な価値

今回は完全に新しいラップトップの話ではありません。LenovoIdeaPad Z570は昨年導入されましたが、今では関連しています。 「ホーム」モバイルコンピュータのラインでは、Zシリーズは比較的高価なY470とY570と非常に安価なLenovo Gシリーズラップトップのほぼ中間に位置しています。 Y570の独自の利点（ハイブリッドSSD + HDD、独自のスピーカーシステムなど）の一部を奪われたIdeaPad Z570は価格が非常に魅力的であり、ハイエンド構成では非常に優れたパフォーマンスを提供します。



私はまだ高速で、信頼性が高く、長持ちするThinkPad Xシリーズをモバイルラップトップとして使用していますが、家庭ではうまくいきません。 しかし、Z570はこの役割に理想的に適合していました。クアッドコアIntel Core i7プロセッサーを搭載した強力なマシンは、音楽や映画の変換に優れた働きをし、HDMI接続のLCD TVでビデオを再生し、家の他のすべてのデバイスの中央データウェアハウスとして機能します。 非常に強力であると同時に、必要に応じてコンパクトになり、新しい場所に簡単に移動できる、一種のホームサーバーを取得できます。 Z570の構成、価格、最大3万ルーブルの価格を持つ強力なラップトップの能力については、こちらをご覧ください。





IdeaPad Z570の私のバージョンの仕様は次のとおりです。



プロセッサー ：Intel Core i7-2630QM 2-2.9 GHz

メモリ ：4 GB DDR3（2つの2 GBモジュール）

ストレージ ：750 GB HDD 5400 RPM

画面 ：15.6インチ1366x768ピクセルの光沢

ワイヤレス ：WiFi 802.11n / Bluetooth

コネクタ ：ヘッドフォン、マイク、3xUSB 2.0、1xUSB / eSATA、HDMI、VGA、イーサネット、カードリーダー

寸法 ：376x250x20-35.5 mm

重さ ：2.6 kg







ご覧のとおり、このラップトップはハードウェアの点で、ビデオカードの点でのみ、より強力なモデル（特にY570 ）よりも劣っています。 NVIDIA GeForce 540M-ゲームではかなりまともなパフォーマンスを提供しますが、実際のファンにはほとんど適さず、外部モニターまたはテレビが接続されると、フレームリフレッシュレートが急激に低下します。 私の場合、ゲームは最優先事項ではありませんでしたが、強力なアクセラレーターを使用して、CUDAを使用してビデオをすばやく変換できます。



外観







驚いたことに、IdeaPad Z570は、 U300のような非常にコンパクトなモデルを好むガールフレンドの注目を集めました 。 最もコンパクトでかなり重いZ570から遠く離れたところが好きなのはなぜですか？ 議論は殺人的でした：「彼は青い！」 確かに、私のコピーの明るい青色は、かなり控えめなデザインと大きなサイズをはるかに上回る可能性があります。 しかし、このラップトップは非常にモダンに見え、ディスプレイの蓋のカバーとタッチパッドの周りのパネルは非常に傷に強いです。







アイランドスタイルのキーボードには、追加のテンキーが装備されています。 ちなみに、今年のIdeaモデルのFnボタンは、Ctrlキーの右側にあるほとんどのユーザーにとってより身近な場所に移動しました。







このキーボードには1つの主張しかありません。カーソルキーは他のキーボードから分離されておらず、最初は見ずに「見つける」ことは非常に困難です。 しかし、これは習慣の問題です。



このラップトップには、外部ドライブにすばやくアクセスするためのeSata、外部モニターまたはテレビを接続するためのHDMI、および4つのUSBポート（Y570とは異なり、2.0のみ）など、あらゆる種類のポートが装備されています。







最先端には、ワイヤレススイッチ（右）とグラフィックス（左）があります。 後者では、NVIDIAグラフィックカードを無効にして、バッテリーの電力を節約できます。



性能



クアッドコアIntel Core i7プロセッサは2 GHzの標準周波数で動作し、必要に応じて2.9 GHzに自動的に加速します。 パフォーマンスの点では、 IdeaCentre B520のデスクトップクアッドコアプロセッサの機能よりもわずかに劣りますが、合成テストでは、デュアルコアIntel Core i5プロセッサを搭載したY570ラップトップ、および合成テストの前世代のデュアルコアCore i7モデルよりも優れています。



Z570にインストールされたGeForce 540Mグラフィックスは、Y570にインストールされたGeForce 555Mグラフィックスカードよりも、人工テストで約20％、実際のゲームで3分の1以上劣っています。 最後に、WD Scorpio Blueハードドライブは、このモデルのデータストレージに使用されます。これはかなり遅いですが、容量が大きい（750 GB）。 その性能指標は記録破りではなく、おそらく将来、そのようなモデルの所有者はハードドライブを高性能SSDに変更したいと思うでしょう。



価格と仕様







IdeaPad Z570の最小コストはわずか12,000ルーブルです。 この価格のラップトップには、最も基本的なIntel Pentiumプロセッサと統合ビデオのみが搭載されています。 そのため、これは非常に安価な家庭用ラップトップに適したオプションであり、Webサーフィンなどの厄介な問題専用です。 他のすべての構成では、プロセッサ、ハードドライブで使用されるメモリ量が異なりますが、ビデオカードは変更されません。 Intel Core i3を搭載した最も安価なモデルには、約2万ルーブルかかります。 最も高価なオプション-クアッドコアCore i7では、コストは3万ルーブル未満です。 これらの2つの価格タグの間には、オペレーティングシステムのないバージョンを含む、あらゆる好みのその他の多くの構成があります。 一般に、IdeaPad Z570のコストは、同様の構成のY570のコストよりも約5〜8千ルーブル低くなります。



評決



記事の冒頭で述べたように、トップエンドプロセッサを搭載したIdeaPad Z570は、最も生産性の高いグラフィックカードではなく、ホームデスクトップおよびマルチメディアサーバーとして非常にうまく使用されています。 ゲーム以外のタスクでは、このモデルは強力なY570の優れた代替品です。 後者では、より高い値札を理解できます。ビデオカード、特にハイブリッドディスクシステムはこれを非常によく正当化します。 そして、Z570は、予算モデルとトップエンドのホームラップトップの間の黄金の平均であることが判明しました。 非常にリーズナブルな価格で、パワフルで見栄えがよく快適です。
