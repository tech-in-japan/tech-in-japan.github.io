ケンコー延長チューブ





マクロ写真を切望しているが、望遠レンズに十分なお金がないアマチュア写真家には、興味深い代替品があります-標準レンズのいわゆる「エクステンダー」。 このようなアマチュア写真家の一人である私は、この機能を購入した印象、2つのケンコーフィルター、およびebayでの最初の注文の経験を共有したいと思います。



購入とパッケージ



Nikon D3100を購入した後、鏡のおもちゃを手に入れた新人の写真初心者のように、YouTubeで面白いビデオチュートリアルを見たり、HDRIを知ったり、マクロ写真から... 「マクロの奇跡」を試してみたいという落ち着きのない最後の欲求は、有名な写真家のブライアンピーターソンからの衝撃的で印象的なYouTubeビデオを見た後に生まれました。 彼のアドバイスに基づいて、ケンコーエクステンションチューブを購入することが決定されました。 そして、キエフのオンラインストアにこの機能がなかったため、ついに「ebayで何かを購入する」ようになりました。



友人のアドバイスでは、価格以外の売り手を選択する際に私が注意した唯一のことは、もちろん、売り手によってすでに成功した類似品がいくつあるかということだけでした。



さらに、私はすぐに同じケンコーから2つのフィルターを購入することにしました。 売り手は、最初の製品に対して13ドル+後続の製品ごとに10ドルとして配送費用を計算しました。 驚いたことに、ある短い通信で、売り手に13ドルですべてを1箱で送るように説得することができました。フィルターは小さく、サウンドロジックが勝利しました。



また、私はパッケージの品質について尋ねるのが面倒ではなく、最も普通で安価な郵便料金を使うように頼みました。 売り手は、梱包が非常に保護的であり、価格が義務を回避するために3つ以上過小評価され、FedexまたはDHLがないことを確信しました。



結果は素晴らしいです：

*香港から13日間で荷物が届きました

*本当に保護的な包装

*宣言された合計値は72ドル対実数〜210ドル

*香港のサイトと郵便サービスの両方で、小包番号による追跡作業。



それで、箱は本当にしっかりしています：





内部の発泡体、およびすべての製品は、にきびのある同じオイルクロスに慎重に梱包されています。







最初の会議



開梱後の写真。 Kenko延長チューブは、レンズを12、20、36 mm延長する3つの部品で構成されています。 それに応じて。 好きなように組み合わせることができます。 添付は、レンズへの接続を説明し、これらの3つの部分のさまざまな組み合わせの倍率値の表を含む説明です。



ここで読みやすい指示。



レンズのフィルター-それぞれ独自のボックス内：





私のガールフレンドによると、この写真はパウダーの広告に非常に似ています。





実験：延長チューブ



そのため、延長コードなしの最大近似値と、焦点を合わせることができる最短距離では、制汗剤の瓶は次のようになります。





12mm延長ケーブルの場合：



（フォーカス55 + 12mm、iso100、1 / 60秒、f / 5.6）



2つのエクステンション：12mm + 20mm：



（フォーカス55 + 20 + 12mm、iso100、1 / 30秒、f / 5.6）



すべての拡張機能付き：12mm + 20mm + 36mm：



（フォーカス55 + 36 + 20 + 12mm、iso100、1 / 4秒、f / 9、三脚）



最大伸び-その他の例


まったく同じリーフレットですが、リーフ自体とその毛に焦点を当てています。







最小伸び




数十枚の写真を撮った後、私は予備的な結論を出しました。クローズアップで人間の目に見えるオブジェクト（葉やイチゴなど）の写真には、最小限の延長コードで十分です。 同じ構造、背景などの写真の場合 -最大延長コードを使用します。 これは、最小限の延長コードを付けた乾燥した葉の写真の例です。





実験：ニュートラルフィルター



ニュートラルフィルターは、色の再現に影響を与えたり、オブジェクトを歪ませたりしないでください。 その唯一の目的は、レンズに入る光の量を減らすことです。 これにより、シャッタースピード（レンズが開いている時間）を上げることができ、画像を「ねじる」トリックと明るい日光の下で水などのモーションをキャプチャするのに役立ちます。 それが、私がさらに注文した映画の1つが中立的な理由です。 たとえば、次のように、レンガの壁をねじることができます（0.5秒の写真の間にレンズをスクロールしてズームする）。





実験：UVフィルター



UVフィルターは、空き地の風景を撮影する際に、かすみやその他の「乱雑さ」を取り除くことを約束します。 残念ながら、バルコニーからの次の2枚の写真には違いが見られませんでした（フィルターなしで残されています）。





おわりに



ケンコーエクステンションチューブは、マクロ写真の未知の美しさの視野を開いたようです。 しかし、これはあなたを近づける望遠レンズではありません。 このようにレンズを長くする場合、レンズなしでは、被写体に近づく必要があります。 最大限に伸ばすと、被写体から1センチまたは2センチだけ焦点を合わせることができました。 ただし、価格は望遠レンズの数倍です。 まあ、初心者のために-私は満足しています！
