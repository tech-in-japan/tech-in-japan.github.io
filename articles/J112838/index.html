PPA経由のUbuntu 10.04および10.10上のXFCE 4.8





Xfce 4.8は、 Linuxオペレーティングシステム用の軽量な作業環境です。 以前はソースコードでのみ利用可能で、少し後でPPAを介して利用できましたが、ubuntu 10.04のユーザーのみが利用できました。 今、彼女は待望のPPAとubuntu maverick merkatの最新バージョンを受け取りました。 インストールの説明と、 xfceの最新バージョンに関するビデオを2つ紹介します



Ubuntu 10.04 Lucid Lynx用XFCE 4.8（32ビットOSのみ）

sudo add-apt-repository ppa：alexx2000 / xfce

sudo apt-get update

sudo apt-get xubuntu-desktopをインストール



Ubuntu 10.10 Maverick Merkat用XFCE 4.8（32ビットおよび64ビットOSバージョン用）

sudo add-apt-repository ppa：koshi / xfce-4.8

sudo apt-get update

sudo apt-get xubuntu-desktopをインストール
