最初のタブレットNokia Lumia 2520のレビュー

Nokiaの「大型」デバイスに関する噂は6か月間続きました。 多くの噂やティーザーに熱狂し、一般の人たちはどんな種類のデバイスになるのだろうと考えていました（Windows Phoneやラップトップのタブレットにもオプションがありました）。 昨年10月にアブダビで開催されたNokia Worldカンファレンスですべてのカバーがドロップされました。

Windows RT 8.1で。







ほんの少し前、ノキアLumia 2520がロシアに到着しました。この点で、このデバイスの詳細なレビューを手配することにしました。



今後は、Nokia Lumia 2520がすぐに明らかにならないことに注意してください。 あなたが最初に考えたかもしれないように、これはコンテンツを消費するための典型的なモバイルデバイスではありません。 むしろ、（コンテンツ）作成に優れた機能を備えたタブレットです。 ただし、ノキアLumia 2520が最新のラップトップを置き換えることを期待してリリースされたとは言えません。 ただし、多くの仕事や創造的なタスクを解決できるということは、このタブレットが単なるゲームやエンターテイメントのプラットフォームではなく、一種のオールインワンデバイスであることを示唆しています。 しかし、まず最初に。



本体



Nokia Lumia 2520タブレットの設計は、タブレットデバイスのフォームファクターにおけるNokia Lumiaラインの最新モデルのスタイル上の決定を具体化したものです。







ケースの素材は、当社の主力デバイスで伝統的なポリカーボネートです。 タブレットには黒、青、白、赤がありますが、ロシアではNokia Lumia 2520は赤と黒でのみ使用できます。 Lumia 2520の色だけでなく、コーティングのタイプも異なることは注目に値します。黒のモデルはマット仕上げで、赤のモデルは光沢があります。







同時に、赤でもタブレットは軽薄なガジェットのようには見えません。 かなりキャッチーな「ハイテク」であり、競合デバイスの質量とは区別されます。



フロントパネルのほぼ全域が、コーニングのゴリラガラス2を備えた10.1インチのスクリーンで占められています。 これを介して、フロントカメラと情報LEDが見えます。 さらに、フロントパネルに表示される要素は、タッチの「Windowsボタン」とスピーカージャックのみです。











画面の大きな対角線にもかかわらず、タブレットの手にはまったくかさばっていないようで、外出先での使用に便利なガジェットの印象を残します。 これは、デバイスの実際の寸法で確認されます：厚さわずか8.9 mmで267 x 168 mm。 タブレットの重量は615グラムです。











モノリシックな背面では、メインカメラの覗き穴とNokiaロゴのみが目立ちます。 他のすべての制御とデータ転送は最後まで行われました。 オン/ロック解除ボタン、メモリカードとSIMカード用のスロット、追加のアクセサリを接続するためのインターフェイスがあります。











ここには、microUSB 3.0およびmicroHDMIコネクタ、標準の3.5 mmヘッドフォンジャック、充電器ジャックもあります。 後者は、その形状により、古典的なノキアの充電器を思い起こさせます。











ハードウェア機能



前述のように、画面サイズは10.1インチで、解像度は1920 x 1080ピクセル、密度は218 ppiです。 ほとんどのNokiaスマートフォンと同様に、Lumia 2520 LCDスクリーンはIPSテクノロジーに基づいています。 「弟」から、ノベルティはClearBlackテクノロジーも受け取りました 。







追加のレイヤーが存在するため、ClearBlackスクリーンではグレアを最小限に抑え、画像を飽和させて読みやすくすることができます。 実際には、さまざまな照明条件のLumia 2520（公共交通機関でニュースフィードやソーシャルネットワークを読んだり、車のナビゲーターとして使用したり、自宅でカジュアルゲームをしたり、オフィスでこの投稿を部分的に書いたり）ことで、画面の読みやすさは常に快適なレベルにとどまっています。



高輝度と広いマトリックスアングルは、画面品質インジケーターでも機能します。 宣言された輝度値は650 nit（cd /m²）ですが、実際にはインジケーターはさらに高い値に達する可能性があります（たとえば、Hi-Tech @ Mail.ruテストでは、インジケーターは731 nit（cd /m²）でした）。 タブレットに指向性光源があっても、たとえば、斜めに持って作業することができます。







ノキアLumia 2520の充填に関する次の重要なポイントは、その「心と脳」-Qualcomm Snapdragon 800プロセッサです。 ノキアは、このシステムオンチップ（SoC）をタブレットコンピューターに、特にWindows 8.1 RTのデバイスの分野で使用する最初のメーカーの1つです。



システム自体は28ナノメートルの技術を使用して作られています。 2.2 GHzの周波数で動作する4つのKrait 400コア、Adreno 330グラフィックプロセッサ、4G LTE、USB 3.0、およびBluetoothをサポートするHexagon v5 DSP、および2560 x 2048の解像度で画面に画像を表示する機能ピクセル。



Nokia Lumia 2520はデフォルトで、システム、アプリケーション、ユーザーコンテンツ用に2 GBのRAMと32 GBの内部メモリを搭載しています。 microSDカードスロットは、ユーザーの「個人用」スペースをさらに64 GB増やします。



アプリケーションについては後ほど説明しますが、パフォーマンスのコンテキストでは、この構成がスムーズなマルチタスクを提供することに注意する価値があります。 特に、アプリケーションを切り替えるときにアプリケーションがすべての機能を保持しているという事実を考慮してください。



Nokia Lumia 2520のWindows RT 8.1では、アプリケーションが単一の画面を共有する場合、マルチウィンドウモードで作業することもできます。 これは、たとえば、ニュースフィードがソーシャルネットワークのスペースに移動した場合に非常に便利です。 これで、ブラウザーとTwitterイベントフィード（VKontakte、Facebook）を絶えず切り替えたり、並列ウィンドウでリンクを開いたりする必要がなくなりました。 さらに、これらのウィンドウのサイズは自由に調整できます。







ところで、インターネットコンテンツを操作するために、デバイスはWi-Fiだけでなく、上記の4Gモジュールも提供します。 このソリューションは、Windowsのタブレットではまだ一般的ではありません。 デバイス間のローカルデータ交換について話すと、このスコアの「大」のLumiaは、最新の標準Bluetooth 4.0、microUSB 3.0、さらにはNFCのサポートを提供します。







タブレットには2つの内蔵カメラが搭載されています。Zeissの光学系を備えたメインカメラ（！）の解像度は6.7メガピクセル、前面は2メガピクセルです。どちらのカメラもフルHDでの録画をサポートしています。



最新世代のコンポーネント（特にSnapdragon 800システムオンチップ、ソリッドステートドライブなど）とWindows RT 8.1の機能の使用により、このデバイスはエネルギー効率が非常に高くなっています。 これに強力な8120 mAhバッテリーを組み合わせると、最大25日間のスタンバイ時間と11時間の連続ビデオ視聴が可能になります。



Nokia Lumia 2520は特別な充電器から充電され、高速充電機能をサポートしているため、わずか1時間で充電を80％充電できます。

Windows RT 8.1



Windows 8.1とWindows RT 8.1には多くの違いがあります。 最初のものがx86 / x64アーキテクチャ用に設計されている場合、RTはモバイルARM専用です。 これは、「クラシック」アプリケーションがNokia Lumia 2520で実行できないことを意味します。







ただし、タブレットの平均的な購入者の観点から見て、モバイルOSと比較すると、Windows RTにはいくつかの利点があります。



他のシステムとは異なり、Windows RTのファイル管理は、複雑なサードパーティアプリケーションの形式では実装されていませんが、使い慣れた「エクスプローラ」です。



タイル付きの初期画面は、ジェスチャ制御によってシャープ化され、ウィジェットの情報コンテンツとアプリケーションアイコンのシンプルさを組み合わせています。

メインコンピューターにWindows 8.1がインストールされている場合、システムはその設定とアプリケーション（ホーム画面上の位置を含む）をLumia 2520と自動的に同期します。これを行うには、アカウントを使用してログインします。







しかし、Windows RT 8.1について話し始めたところに戻ります。



用途



Windows RTの反対者の伝統的な議論は、「クラシック」アプリケーションの欠如と「ストア」の控えめな品揃えです。 ただし、クラシックWindowsアプリケーションはタッチコントロール用に最適化されておらず、公式アプリケーションストアは2013年末の時点で200,000個の小規模なアプリケーションとは呼ばれなくなりました。 多くの主要な出版社とサービスはすでにプラットフォーム用のアプリケーションをリリースしているか、リリースする予定です。



Gameloft（Asphalt 8）、Disney、Microsoft Studiosなど、多くの「ゲーム」の巨人が自信を持ってWinRTプラットフォームを習得し始めました。 Nokia Lumia 2520のハードウェア機能は、現代のタブレットのヒットに対処するのに十分です。







Lumia 2520では、Nokia以外の上記のすべての補完（Lumiaスマートフォンの所有者になじみのある）。 現在、ブランド化されているアプリケーションには、Nokia Camera、HERE Cards、Nokia Mix Radio、Nokia Stories、その他いくつかのアプリが含まれます。



Lumiaユーザーに愛されている「ノキアカメラ」は、便利なインターフェイス、撮影パラメーターの微調整、高品質のフレーム処理を備えた撮影アプリケーションです。 もちろん、上記のすべてがタブレット用に最適化されています。







このアプリケーションでは、選択を行うことができます。パラメーターの定義をNokia Cameraのアルゴリズムに提供するか、ホワイトバランス、焦点距離、ISO値、シャッタースピード、露出を必要に応じて調整します。



ここにある地図は、Nokiaのスマートフォンだけでなく、さまざまな自動車メーカーの多くのナビゲーションシステムでも実証されている詳細な地図です。 アプリケーションにはオフラインナビゲーションモードがあります。 これを行うには、デバイスのメモリにカードをプリロードするだけで十分です。 オンラインモードでは、ユーザーはトラフィックに関するリアルタイムの動的データを利用できます。 しかし、ここは、運転者だけでなく、歩行者や公共交通機関で旅行する人にとっても便利です。







「Nokia Stories」 -スマートフォンまたはタブレット上のフォトギャラリーの別の外観。 こちらの位置情報を使用して、このアプリケーションには世界地図上に写真やビデオのアイコンがあり、一種の「写真旅行」を作成します。







Nokia Mix-Radioは、ストリーミングオーディオサービスです。 ユーザーは、アーティストや作曲だけでなく、特別な編集コレクションやユーザーコレクションでも音楽を聴くことができます。 サービスはオフラインでも機能します。







また、ノキアのタブレットには、シンプルなビデオ編集ツールNokia Video Director、My Nokiaタブレットに関するインタラクティブな情報ガイド、およびサービスプロバイダーからサービスメッセージを受信できるオペレーターアプリケーションからの通知があります。







仕事のためのツールとしてのノキアLumia 2520



ノキアLumia 2520を別の観点から、つまり作業中のラップトップの代替品として見ることにしたので、特にこのセクションをアプリケーションの範囲外にしました。



まず、このタブレットのすべての所有者は、デフォルトで、Word、Excel、PowerPoint、OneNoteを含むMicrosoft Officeアプリケーションの基本パッケージを受け取ります。 アプリケーションは、x86のアプリケーションと違いはなく、テーブルの作成やプレゼンテーションの提示（microHDMIコネクタを思い出してください）などの基本的なタスクを備えているため、Lumia 2520ユーザーは簡単に対処できます。







さらに、ユーザーはストアから企業のOfficeアプリケーションをダウンロードできます：同僚とのコミュニケーション用のLync 、会議、ビデオ通話、SharePointサイトからニュースを受信して​​ディスカッションに参加するためのSharePointニュースフィード 、Microsoftレポートへの常時アクセスのためのMicrosoft Power BI Office 365





Microsoft Power BI



組織でNokia Lumia 2520を使用するためのもう1つの重要な点は、Nokia Lumia 2520がVPN接続を使用して企業ネットワーク上で機能できることです。 ネットワーク設定は、タブレットの一般設定を通じて設定されます。



Microsoft Dynamics、 Microsoft Dynamics CRM 、 Dynamics AX 2012 Approvals 、 Dynamics AX 2012 Expenses 、 Dynamics AX 2012 Timesheetsが何らかの形で仕事に関係している人たちの助けになります。



Microsoft Dynamics CRMは、組織のCRMデータベースを追跡および管理するための「ダッシュボード」です。 最後の3つのアプリケーションでは、Dynamics AX 2012の承認、レポート、タイムシートを管理できます。すべてのアプリケーションは、Windows Application Storeで無料で利用できます。



また、Windows Server 2012 EssentialsおよびWindows Server 2012 R2 Essentialsでそれぞれ作業するためのMy Server 2012およびMy Server 2012 R2を見つけることができます。 これらのアプリケーションのインターフェースを介して、ユーザー、デバイス、アラートを管理し、共有ファイルにアクセスできます。 さらに、My Serverアプリケーションを使用して最近アクセスしたファイルはオフラインでも使用できます。これは、旅行中や地下鉄などで作業する場合に便利です。



たとえば、個人または企業で使用するための単純なアプリケーションを作成するには、Microsoftアプリケーション「Project Siena」を使用します。 製品カタログまたはインタラクティブなFAQを作成するには、コードを記述するための特別なスキルは必要ありません。 このアプリケーションの使用方法の詳細については、 こちらをご覧ください 。







他に何？ 残念ながら、ノキアLumia 2520を使用してビジネス上の問題を解決するために利用可能なすべてのアプリケーションをリストするには、十分な投稿がありません。 ただし、確かなことが1つあります。ストアにはプロフェッショナルなアプリケーションがあり、十分な量があります。 さらに、 Adobe Photoshop Expressフォトエディターなどのユーザーにとって便利な機能的なアプリ、または患者データSAP EMR Unwiredを操作するための医療アプリケーションなどの高度に専門的かつ高度にターゲットを絞ったアプリケーションのいずれかです。
