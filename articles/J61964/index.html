MyNotifierの提案/販売、フリーランサー向けウィジェット

こんにちは、Habrahabr！



1年半前に、 フリーランサー向けのWebサイトを作成しました。これは、6つの最大のフリーランスサイトから注文に関する情報を収集し、Windowsユーザーのデスクトップに配信しました。 サービスはねじれた状態ではなく、私にお金をもたらしてくれませんでした。 1年半の間、私はサイトを数回再編集し、サーバーを変更しました。 Windowsクライアントの新しいバージョンで2回投げられました。 過剰な支払いをしないように、サーバー上のトラフィックを常に均等化しました...このサービスは、多くの友人や新しい顧客をもたらしました。



サービスについてはここに 、作成の歴史についてはここに書かれています 。



創業以来、多くの水が流れてきました。 私はフリーランスのサイトにはほとんど行きません-仕事や常連客でいっぱいになりました。 alllanceなどのサービスは、私のものと似ています。



6月2日に、Free-lance.ruサイトのデザインが変更され、その結果、情報を取得したパーサーが機能しなくなりました。 毎日修正に1分かかると約束して、サブスクライバーにこのイベントを10日間待機させます。 なんとかして、私はサイトを別の善良な人の世話に移すことにしました。 1つを見つけるために、私はこの投稿を書いています。



技術的ポイント：
  • php + mysql + nginx;
  • データベースには、6つのフリーランスエクスチェンジを備えた320,000以上のプロジェクトが含まれています。
  • 1000人以上のユーザーが登録されています（活気を取り戻すために使用されているため、ほとんどのユーザーがサービスを使用しています）。
  • 同時に、プログラムは平均30人で開始されました（フリーランスが落ちた後-10）。
  • プロジェクトへの300,000以上の移行に関する情報が保存されます。
  • 新しいプロジェクトは、SSLを介してサブドメインからクライアントによって取得されます。
  • 新しいクライアントの開発に関する適切な参照条件を作成しました。


サイトが機能しなくなったり、リンクウォッシュになったりすることに興味がないので、最初に尋ねた人には知らせません。 最悪のシナリオでは、単純に大規模な類似サービスのいずれかにリダイレクトします。 2つのケースがあり、そのうちの1つでプロジェクトを行います。
  • あなたはすでに人類の利益のためにプロジェクトをサポートしており、それを行う準備ができています。
  • または、プロジェクトを購入する準備ができている場合、私はたくさんのお金を提供した人々から選択します。 この場合、手紙は提案された金額の表示で始まる必要があります:-)
プロジェクトまたは財務提案へのリンクは、 anton @ isaykin.ruまでお問い合わせください。



私は、収益化、拡大、または他のサービスへの参加に反対していません。 興味のあるブランチのために、彼をフォローするのが難しくなりました。サービスは私にとって負担になりましたが、それを閉じるだけで、毎日それを使用するユーザーにとっては不公平です。
