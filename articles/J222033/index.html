本当に強力でコンパクトなPC

いくつかの新しいアイテムは非常に憂慮すべきです。 これは、キーボードのないオープンラップトップで、弱いARMプロセッサをベースにした$ 1'995のディスクなし、350ドルのスマートフォンサイズの 「ユニバーサル」かつ「パワフル」なTango PCです。 、職場と自宅の間にまだコンピューティングモジュールを装着している場合。



私は長い間、ラップトップを何かに置き換えることを目指していましたが、タブレットの登場以来、ラップトップはそのようなものとして完全に興味を失いました。

完全にコンパクトなものが欲しかったので、nano-ITXとpico-ITXのフォームファクターを掘り始めましたが、手頃な価格で十分な生産性を見つけることができませんでした。



しかし、Intel Coreの第4世代であるHaswellを搭載した文字通り半リットルがありました。







画像がはっきりしていない場合、このPCは、前述のタンゴのドッキングステーション（108mm x 115mm x 30mm vs 135mm x 135mm x 27mm）よりも少し小さくなっています。



マザーボードに埋め込まれたプロセッサが付属しています。 プロセッサライン：



-Intel®Core™i7-4500U 1.8 GHz-3.0GHz（ターボ）、Intel HD 4400

-Intel®Core™i5-4200U 1.6 GHz-2.6GHz（ターボ）、Intel HD 4400

-Intel®Core™i3-4010U 1.7 GHz、Intel HD 4400

-Intel®Celeron™2955U 1.4 GHz、Intel HD



AMD A8-5545M 1.7G / 2.7GHzに基づいたモデルもあります。



このような奇跡は、完全なセットが必要なメモリ（1.5GBではなく、1.35VのDDR3のみが適切です）とmSATAドライブが必要なアセンブリのモデルとして販売されています。

コストは、プロセッサに応じて、海外サイトで250〜500ドルの範囲です。 合計容量が8GBの2つのメモリスティックのセットは、約100ドルで、ほぼ同じコストとmSATA SDD 128GBで購入できます。 WiFi b / g / nおよびBluetooth 4.0 / 3.0は既に搭載されており、PCIeポートのみを占有します。















接尾辞Hが付いているケースでも使用できます。これは通常の2倍の高さですが、2.5 "SATAドライブのインストールが可能です。







GamingとProには、さらに高いボディとより強力なフィリングのオプションがあります。











別の特別な注意は、プロジェクタを内蔵したモデルに値します。
