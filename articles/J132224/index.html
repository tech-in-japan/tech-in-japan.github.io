Operaモバイルエミュレーター11.5

Opera Mobile Emulatorの新しいバージョンがリリースされ 、現在のバージョンのOpera Mobile 11.5と一致しています。 新バージョンでは、Opera Mobileの起動時にシミュレートできるデバイスのリストと、ランチパッド自体が更新されました。







スクリーンショットはクリック可能です：







現在、Opera Mobile Emulatorは、モバイルデバイス用のWebサイトを作成およびデバッグする最も便利な方法を提供しています。 かさばるSDKをインストールしたり、1ダースのデバイスを購入したりする必要はありません。エミュレーターは、Windows、MacまたはLinuxで個別のシンプルなプログラムとして実行されます。



Dev.OperaのOpera Mobile Emulatorでのテストの詳細については、英語の最新記事「 Opera Mobile Emulator for desktop 」をご覧ください。



Opera Miniでサイトをテストしたい人のために、 Opera Mini Simulatorがあります。これは、サイト上でJavaアプレットとして直接実行されます。 必要に応じて、 Opera Mini 4.2を実行することもできます。 まあ、Opera Miniの特別なバージョンを実行する必要がある場合、または作業中のマシンで直接実行する場合は、このためのMicroEmulatorがあります。
