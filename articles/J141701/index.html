Internet Explorer 10でWindows 8 Consumer PreviewでWebサイトを閲覧する



この投稿では、Internet Explorer 10の新機能について説明します。パフォーマンス、標準、利用可能な機能などの面で基本的なブラウザーモジュールを大幅に改善しました。 MetroスタイルのInternet Explorer 10は、Webを閲覧するための新しい改善された方法であり、アクセス方法ではなく、必要な情報に真剣に集中することができます。 没入感を最大限に引き出します。 同時に、タブ、キーボードショートカット、InPrivateの表示など、使用するすべてのセキュリティ機能とコントロールを提供します。



SkyDriveとWindows 8を使用して、アプリ、ファイル、コンピューター、およびデバイスをクラウドに接続します



ブログの多くの読者は、SkyDriveとMeshサービスを積極的に使用しています。これらは、Windows Liveサービス（Hotmailサービスなど）の一部であり、Windows Liveのメインコンポーネント（Messenger、Photo Album、Film Studio、Mailなども含む）のプログラムです。 これらのサービスを導入し、Windows 7がリリースされたとき、Windowsエクスペリエンスをどのように補完するかについて話しました。 Windows 8を開発する際、ユーザーに対するこれらのサービスの役割を増やす方法、つまり、すべてのWindows 8ユーザーにすべてのコンピューター（および電話）でクラウドサービスを使用する機会を提供する方法を慎重に検討しました。

Windowsランタイムで非同期を使用して高速で柔軟なアプリケーションを作成する



人は本質的に非同期であるため、アプリケーションも非同期に応答することが期待されます。 Windowsランタイム環境（WinRT）では、この非同期性は、高速で柔軟なMetroスタイルアプリケーションの作成において特権的な位置を占めます。 Metroスタイルアプリケーションを構築している場合、ある時点で非同期コードを記述する必要があります。 このブログ投稿では、WinRTで非同期プログラミングが非常に一般的である理由について説明し、アプリケーションでのアプリケーションの可能性とその操作のいくつかの原則に関する基本的な知識を提供します。



アプリケーションでWindows 8コントラクトをアクティブ化する



Metroスタイルアプリケーションの開発を開始すると、Windows 8の新しい効果的な概念であるコントラクトをすばやく見つけることができます。Metroスタイルアプリケーションは、コントラクトを使用して、他のアプリケーションおよびサポートするWindowsオペレーティングシステムとの対話を通知します。 検索、共有など、それらの一部については既に聞いたことがあるかもしれません。契約はアプリケーションの効率を高め、ユーザーが契約を使用する多くのアプリケーションをインストールするときに、システムまたは相互のやり取りを改善します。 この記事では、アプリケーションに契約を追加する際に考慮すべき主要な概念の1つであるアクティベーションについて説明します。

Googleはユーザーのプライバシー設定を回避します



Internet Explorerワーキンググループが、GoogleがSafari Webブラウザーのユーザーのプライバシー設定をバイパスしていることに気付いたとき、私たちは簡単な質問をしました。GoogleはInternet Explorerユーザーのプライバシー設定もバイパスしますか。 この質問に対する答えはイエスであることがわかりました。Googleは同様の方法を使用して、Internet Explorerのデフォルトのプライバシー保護を回避し、Cookieを介してInternet Explorerユーザーを追跡します。



オンラインプライバシーの保護



今朝、米国政府は、インターネット上のユーザーのプライバシー保護を改善するプログラムの一部として提案された新しい「ユーザー機密保持法案」を発表しました。 前述したように、Microsoftは、オンラインプライバシーのWeb標準を作成するために、追跡保護業界のイニシアチブであるW3Cコンソーシアムに積極的に関与しています。



Windows Consumer PreviewのWebプラットフォームコンポーネント

Internet Explorer 10の高度なメモリ保護



Internet Explorer 10では、メモリ保護機能が大幅に強化されており、脆弱性の悪用が難しくなり、悪意のあるインターネット環境でもユーザーを安全に保つことができます。 これらの機能強化により、脆弱性悪用ツールの開発の複雑さとコストが増加し、攻撃者の生活が困難になります。



拡張保護モード



Internet Explorerの各リリースには、インターネットの閲覧中にユーザーを安全に保つための改善が含まれています。 Internet Explorer 10の新しい拡張保護モードは、攻撃者がブラウザーまたはそのアドオンの脆弱性を悪用した場合でもデータを保護するのに役立ちます。



Windows Consumer PreviewのWebSocketオブジェクト



Windows 8 Consumer PreviewおよびServer BetaのInternet Explorer 10およびその他すべてのMicrosoft WebSocketクライアントおよびサーバーコンポーネントは、IETF WebSocketプロトコルの最終バージョンをサポートするようになりました。 さらに、Internet Explorer 10は、W3Cコンソーシアムから勧告候補ステータスを受け取ったWebSocket APIを実装しています。



Internet Explorer 10およびMetroスタイルアプリのIndexedDBの更新



Internet Explorerチームと幅広いWebコミュニティのメンバーは、IndexedDB仕様の改善を続けています。 IndexedDBはW3Cコンソーシアムのワーキングドラフトであり、インターネット接続が切断されている場合でも、ユーザーのデバイスにデータを保存、検索、取得できます。 IndexedDBは、一般的なInternet Explorer 10 WebプラットフォームおよびWindows 8 Consumer PreviewオペレーティングシステムのMetroスタイルアプリケーションのコンポーネントです。



Windows 8でのInternet Explorer 10の起動オプション



Internet Explorer 10には、Windows 8の概念に従って構築された新しいブラウジングエンジンがあり、Metroスタイルアプリのすべての利点を提供します。 Internet Explorerの基本アーキテクチャを拡張してこのメ​​カニズムを作成し、完全なハードウェアアクセラレーション、信頼性の高いセキュリティ、HTML5およびその他のWeb標準の完全サポートを備えた高速ブラウザモジュールを提供しました。 Internet Explorer 10は、オペレーティングシステムで使い慣れたエクスペリエンスも提供します。これは、ワークフローで通常のデスクトップツールを引き続き使用したいユーザーにアピールします。
