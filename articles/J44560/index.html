仮想会議VORT-EX 08

11月18日と19日、モスクワ時間の午後4時から9時まで、マイクロソフトはWeb開発者向けの仮想会議VORT-EX 08を開催します。



会議はインターネットで放送されます。参加するには、ウェブサイトwww.vort-ex.comに登録するだけで十分です 。その後、Microsoftの専門家や他の企業からのWeb開発、Webデザイン、ビジネスのトピックに関する専門家の報告を聞くことができますインターネット。



会議は英語で開催されるため、新しい知識を得るだけでなく、英語の理解も練習できます。



さらに、イベントに喜びを加えるために、参加者はWindows Web Server 2008を無料でダウンロードでき、登録されたすべての参加者は自動的にギフト抽選（携帯電話、ウェブカメラ、Microsoft Pressの書籍、Zuneプレーヤー）に参加します。 ですから、何か新しいことを学ぶだけでなく、記憶に残るお土産を手に入れるチャンスもあります。



登録し、オンライン会議を見て、レビューを書く-あなたがそのようなイベントに興味があるかどうか、どの形式があなたに適しているか、どのトピックがそのような仮想会議でカバーされるべきかを知ることが重要です。



会議についてもう少し詳しくはブログに書いています 。
