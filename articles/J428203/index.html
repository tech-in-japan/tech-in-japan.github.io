チャートを見る：クラウドコンピューティング市場の推定と予測、2018年のデータ





今日のクラウドプラットフォームとアプリケーションは、経済で広く使用されており、ITインフラストラクチャの開発を推進し、ビジネスのデジタル化の新たな機会を生み出しています。 以下では、グローバルクラウドサービス市場の状況を反映したクラウドテクノロジー市場の予測と評価の概要を確認します。



CIOは、成長の触媒および新しいビジネスイニシアチブの成功を可能にするアーキテクチャとしてクラウドプラットフォームに依存しています。 戦略的なIT管理の時が来ました。 後で役立つと思われる結論についてお話します。



Amazon Web Services（AWS）は、2018年の第2四半期に同社の営業利益の55％を占め、AWSは同社の純売上のわずか12％を占めました。



2018年の第1四半期には、サービスが収益の40％を占めました。これは3年前よりも26％多くなっています。 出典： Business Drives Amazon's Profits、Statista、2018年7月27日







80％の企業がアプリケーションを起動するか、優先クラウドプラットフォームとしてAmazon Web Services（AWS）を実験



合計で、企業の67％がMicrosoft Azureプラットフォームでアプリケーション（45％）と実験（22％）をホストしています。 企業の18％がワークロードにGoogleクラウドプラットフォームを使用し、23％が将来の使用に向けてプラットフォームを評価しています。 2018 RightScale研究は、この比較の作成に使用された元のStatistaデータセットに含まれていました。 出典： Statista、2018年に世界中でアプリケーションを起動するために実行されている公的にアクセス可能なクラウドプラットフォームの現在および計画された使用。







企業環境でのMicrosoft Azureの導入は、43％から58％、CAGR-35％（累積平均年間成長率）で大幅に増加していますが、この市場でのAWSの普及率は59％から68％に増加しています



将来のプロジェクト（実験およびワークロード）の計画を立てた回答者は、Googleに最大の関心を示しています（41％）。 出典： RightScale 2018 State of the Cloud Report 表示を簡素化するには、グラフをクリックして展開します。







SaaS市場では、顧客はソフトウェアプロバイダーに200億ドルの四半期収益を生み出し、その数は年間32％で増加しています



マイクロソフトは17％以上の世界市場シェアをリードしており、現在急成長しているコラボレーションセグメントでのリーダーシップにより、現在SaaSのリーディングプロバイダーです。 出典： 四半期のSaaS支出は200億ドルに達し、Microsoftが市場でのリーダーシップを拡大するため、2018年8月21日にSynergy Researchが発表しました 。



金融サービス業界は、プライベートクラウドまたはパブリッククラウドに展開された仮想サーバーの割合が最も高く、実装率の中央値が19％であるのに対し、ほぼ100％に近づいています。



McKinsey＆Companyによる最近の調査では、金融サービス業界が、保険やヘルスケアに先んじて、仮想化や基盤となるクラウドを超えてクラウド開発を進めていることが示されました。 出典： McKinsey＆Company、Nagendra Bommadevara、Andrea Del Miglio、Steve JansenによるITの近代化を加速するクラウドの採用 。







品質管理、コンピューター支援設計、製造管理システム（MES）は、クラウドで最も広く使用されている3つのシステムです。



また、個別の継続的な生産を行うメーカーの60％が、エンドユーザーがオンプレミスインフラストラクチャよりもクラウドを好むと回答していることも示しています。 出典： アマゾンウェブサービス＆IDC：クラウドに対応したIndustry Ready 。 表示を簡素化するには、グラフをクリックして展開します。







予測によると、パブリッククラウドサービスの世界市場は、2018年の1,758億ドルから2019年には206.2億ドルまで17.3％成長します



調査およびコンサルティング会社のGartnerによると、市場は2017年の1,453億ドルに比べて21％成長します。 Infrastructure as a Service（IaaS）は、最も急速に成長している市場セグメントです。 2019年の予測成長率は27.6％で、2018年の310億ドルに対して、市場は395億ドルに達します。 2022年までに、ガートナーは、IaaSパブリッククラウドサービスを利用する企業の90％がIaaSおよびPlatform-as-a-Service（PaaS）とその機能を単一のプロバイダーとともに使用すると予想しています。 出典： Gartnerは世界のパブリッククラウド収益を2019年に17.3％成長させると予測しています。







クラウド環境での展開のためのインフラストラクチャIT製品（サーバー、企業ストレージ、イーサネットスイッチ）の総支出の伸びは、前年比で10.9％に達し、2018年には523億ドルに達すると予測されています。



将来的に、IDCは、オフプレミスインフラストラクチャのコストが5年間で平均年間成長率10.8％で増加し、557億ドルに達すると予想しています。 出典： クラウド環境での展開のためのITインフラストラクチャへの支出は、2018年に10.9％成長すると予測されています（IDC）。







1.3兆以上。 IT支出のドルは、2022年までにクラウドテクノロジーへの移行に直接または間接的に関連付けられます



2018年の19％に対して、2022年までに、主要IT市場での企業支出の28％がクラウドに費やされる予定です。 2018年までの最大の変化は、特にSalesforceがマーケットリーダーとして支配しているカスタマーリレーションシップマネジメント（CRM）ソフトウェアのおかげで、アプリケーションソフトウェアで発生しました。 CRM市場は、従来のソフトウェアと比較して、クラウドのコストの割合が高いポイントにすでに到達しています。 出典： Gartnerは、2022年までに主要ITセグメントの支出の28％がクラウドに移行すると述べています。







IDCは、世界中のパブリッククラウドサービスへの支出が2018年に1,800億ドルに達し、2017年と比較して23.7％増加すると予測しています。



IDCによると、市場は5年間の平均年間成長率（CAGR）21.9％を達成すると予想されており、2021年にはパブリッククラウドサービスのコストは合計2,770億ドルになります。 2018年にクラウドサービスに最も支出すると予測されるセクターは、個別製造業（197億ドル）、専門サービス（181億ドル）、および銀行業（16.7 10億米ドル）。 また、2018年には、継続的な生産および小売企業がパブリッククラウドサービス市場にさらに100億ドルを費やすことが予想されます。 これらの5つの業界は、以前に選択されたソリューションへの継続的な投資により、2021年でもリストのトップに留まります。



5年間の予測期間で支出が最も急速に伸びているセクターは、専門サービス（24.4％CAGR）、通信（23.3％CAGR）、および銀行サービス（23.0％CAGR）です。 出典： 今年 、IDCの1,600億ドルに達する世界的なパブリッククラウドサービスの支出予測



クラウドへの移行の理由の1つは、そのようなソリューションの費用対効果です。 この質問については、Cloud4Y のホワイトペーパーで詳しく説明しています。 TCOの計算：クラウドとサーバーの購入 。
