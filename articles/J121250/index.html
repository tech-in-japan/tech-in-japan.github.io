別のHTPC





2009. 長い間、自宅に2台目のコンピューターを設置したいという要望があり、液晶テレビの出現を加速しました。 HTPC（Home Theater Personal Computer）と呼ばれ続ける「ファーストセカンド」マシンは、スリムなATX形式の昔ながらのAMD Athlon 2500+老人になりました。 しかし、このオプションは、いくつかの欠点を考慮すると、夢の限界からはほど遠いものでした。







私は小さなアパートに住んでいますが、どうやら隣人とは本当に幸運で、騒がしい道路や路面電車からはかなり離れていたようです。 二重窓は修復後に完全な静寂を追加しました。 あなたはすでに推測していると思いますが、私が最前線に置いた最初の事柄はノイズ、またはむしろその欠如です。



2番目に重要な要素はエネルギー消費であり、これは特に毎年増加する電力の価格を連想させます。

3つ目はコンパクトな配置であり、重要ではありませんが、新しいHTPCの望ましい要件です。

そして、もちろん、すべてのニーズに対して十分な生産性があります。



そして、どのようなニーズがありますか？


-720 / 1080pまでのビデオ形式をテレビで表示。

-数百ギガバイトのメディアデータを保存します。

-トレントのダウンロード/配布。

-開発用のホームWebサーバーとして機能します。

-モバイルからパブリックWi-Fiネットワークで作業するときの妄想の治療法として、VPNサーバーの役割を実行します。

-小さな夜間の特定の自動化タスク。通常、スケジュールに従って作業します。



2010.予算と選択戦略はまったく定義されていません。それは、より少ないお金でより多くを獲得したいという願いとして理解されるべきです。



エイジオブアトム。


この特定のラインアップに興味があったのは、その低消費電力と予想されるパッシブ冷却システムの在庫があるためです。

私の街の店での選択はこのクラスではかなり限られており、私はどこかからの配達を待ちたくありませんでした。 選択はZOTAC IONITX-CEに落ちました。







私には完璧な解決策のように思えました：

-Intel Atom 230プロセッサー。

-NVIDIA MCP79チップセット（ION）;

-フォームファクターmini-ITX;

-3 SATA 3Gb / s;

-最大4 GBのRAM。

-外部電源90ワット。

価格はあまり良くありませんでしたが、ファンを1台も使わずにシステムを組み立てる機会を上回りました。



動作するPCの前に、RAMが足りなかったので、余分な費用をかける必要はありませんでした。GEILDDR2-6400 2GBキットの残りのストラップを使用し、2番目のバンクに1GBを挿入しました。 その結果、目立たない2 + 1構成を受けた両方のトリムにはラジエーターがあります。



体。


ここですべてが悲しいことが判明しました。 mini-ITXフォームファクターエンクロージャの選択は非常に少なかった。 私はこのケースを取りました（名前はもう覚えていませんが、写真素材の品質についても謝罪します）：











ケースはそれほどコンパクトではありませんが、5インチのベイがあり、250ワットの電源は絶対に必要ありません。



組立


特別な瞬間はありません-すべてがシンプルです。 ケースのPSUを取り外して取り外しました。 マザーボードをインストールし、RAMバーを貼り付けました...、ああ、ウィンチェスター。



単一のノイズ源を減らすことを期待して、ウィンチェスターはグリーンに選ばれました-容量1.5TB、64MBキャッシュ、プレートの低速回転のWD。



ケースの横にはファンがありましたが、ファンは意図的に忘れられており、電源が入っていません。







つけて 最初の考え-静かで、とても静か。 プラットフォームとソフトウェアの選択の問題を別のストーリーで説明する予定なので、鳥に話を戻しましょう。 長期間の運用（昨年の夏を思い出してください）により、低速でサイドクーラーをオンにする必要がありました。 一般的な温度体制は満足のいくものになりました。



アップグレード


最初に取得したのは、Microsoft Remote Control w / Receiver for MCEでした。







便利なリモートコントロール、唯一のマイナスはバッテリーの大食いと呼ばれることができます、おそらく理由はボタンのバックライトです。



PCI-EのWi-Fiモジュール。 HTPCをラップトップとモバイルのリンクとして使用したかった。 今回は辛抱強く、ebayで注文しました。 1か月後、nをサポートするWi-Fi付きルーターを購入する前に、モジュールが正常に到着し、挿入され、忠実に機能しました。







結論として、説明した構成は、提示されたすべての要件をまだ満たしており、引き続き喜ばれ、マルチメディアに快適さをもたらします。 今日は、おそらく、USB3のサポート、2番目のネットワークインターフェイス、そしておそらくはより高速なプロセッサを追加したいと思います。
