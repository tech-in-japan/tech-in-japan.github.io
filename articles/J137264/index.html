Context Free：画像生成言語





この画像は 、次の説明を使用してContext Freeによって生成されます。 



startshape T // FLIGIZ background{b -1} tile {s 2.5} rule T {3*{r 120 hue 30}S{x .3}} rule S 3{CIRCLE{hue 30}4*{r 20 b.007 sat .1}S[s.3.7y.9]} rule S {CIRCLE{hue 15}9*{r 20 b.05 hue -3}S[s.3.7y.9]} rule S {S{flip 90}}


画像を説明するために、Context Freeは、画像生成専用に作成されたコンテキストフリーの文法を備えたCFDGプログラミング言語を使用します。 大まかに言うと、これはCIRCLEとSQUAREの2つの端末のみを持つ基本的なルールのセットです。 レンダリングは、 Anti-Grain Geometryの Maxim Shemanarevライブラリを使用して実行されます。



コンテキストフリーアートギャラリーのその他の絵画をいくつかご紹介します。 ギャラリー内のすべての作品は、 クリエイティブコモンズライセンスの下で公開されています（オリジナルへのリンクをクリックしてください）。
