Dell Streak 7タブレットのビデオレビュー

こんにちは

先ほど、 Dell Streak 5インチデバイスについて説明しました。 画面の対角線は5インチで、デバイスに適切な名前を付ける方法（タブレットまたはスマートフォン）が完全には明確ではありませんでした。 今日、私たちは会社の別の新製品を詳しく見ていきます。それは間違いなくタブレットです-Dell Streak 7に会ってください：







他のデル製品と同様に、タブレットは、デザインを大幅に改良することなく、品質と真剣な製品の感覚を呼び起こします。 ケースはプラスチック製のモノブロックです。 デバイスのバッテリーは取り外しできません。開く唯一のカバーは、メモリカードスロットとSIMカード用です。



スクリーンを覆うために、 ゴリラガラス技術を使用して作られた傷や衝撃に対する耐性が強化されたガラスが使用されました。 市場でのDell Streak 7の主な競合相手はSamsung Galaxy Tabである可能性が高く、どちらのタブレットにも長所と短所があります。 デバイスDellの主な利点は、強力なデュアルコアプラットフォームNvidia Tegra 2です。







Dell Streak 7の機能：

  • オペレーティングシステム ：Android 2.2（Froyo）+ Stage UI 1.0
  • プロセッサー ：nVidia Tegra T20-デュアルコア@ 1 GHz
  • メモリ ：512MB DDRII SDRAM、16、32、または64GBオンボード。 拡張スロットSD、MMC、SDHC
  • 寸法 ：199.9 x 119.8 x 12.4 mm、重量450 g
  • バッテリー ：2780 mAhの容量
  • ディスプレイ ：7インチ、容量性、マルチタッチ、WVGA解像度800×480
  • サウンド ：3.5 mmジャック、ステレオスピーカー
  • カメラ ：背面5 Mpx、前面1.2 Mpx
  • データ転送 ：2G（GSM）、3G（HSPA）、WiFi 802.11b / g / n（2.4 GHz）、Bluetooth 2.1 + EDR、内蔵GPS受信機


会話中に、メーカーはすぐにデバイスをAndroid 3.0 Honeycom bにアップグレードすることを約束しました。ご存知のように、システムの最新バージョンは特にマルチコアプラットフォームを対象としています。 主な欠点のうち、画面の視野角が悪い、容量がわずか2780 mAHのバッテリーと寸法（厚さ12.4 mm、重量450 g）に注意したい。



Dell Streak 7のWiFiバージョンは5月に発売され、ロシア市場では6月のみにWiFi + 3Gバージョンが登場します。
