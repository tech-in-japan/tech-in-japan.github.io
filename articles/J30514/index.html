PHP用SQL Server 2005ドライバー

SQL ServerがPHPをサポートするという約束を守りながら、MicrosoftはSQL Server 2005のドライバーをリリースしました。



このドライバーにより、PHP 5.xはSQL Server 2005のどのエディションでも動作します。MSDNで 、ドライバーについて議論するフォーラムが作成されました 。 希望とエラーメッセージをSQL Server Connectに送信できます 。

ドライバーのソースコードはCodePlex Webサイトで入手できます。
