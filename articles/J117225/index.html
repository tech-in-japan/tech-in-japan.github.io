RIT ++チケットが抽選されます！

ハブロフスク、ようこそ！



REG.RU会社から、 RIT ++のチケットの抽選-今年の主要な専門家会議から別の驚きを用意しました。 警告：今回は難しいタスクを思いつきました！



コンテスト参加者は、サイトwww.reg.ruのメインページの読み込み/表示時間を最適化するための最適なオプションを提供する必要があります。 独創性を発揮してください-当然、REG.RU開発者は多くのことを試してみましたが、同社は標準および非標準の最適化手法の両方で効果的なソリューションを見つけることができました。



参加するには、2011年4月11日から17日まで、電子メールアドレスpresent@reg.ruに、識別のためにREG.RUアカウントの名前を示すソリューションを送信します。 戦闘に入る前に、タスクを完了するための技術的な要件を理解することを忘れないでください。



-ソリューションには、ページが読み込まれた瞬間からページの表示時間を短縮するHTML / CSS / JSコード、画像、またはその他の推奨事項の1つ以上の最適化を含める必要があります。 ページの表示時間が読み込み開始から少なくとも10％短縮されるように決定されます。



-提案されたすべてのソリューションは、実際にREG.RUメインページの内容について著者が検証する必要があります。



-検証は「実際の条件に可能な限り近い」条件で実行する必要があります。つまり、「サーバー」から「クライアント」へのチャネルの制限速度、現実的な応答時間（少なくとも10ミリ秒）を考慮に入れる必要があります。



-チャンネル遅延をエミュレートせずにローカルディスクからページファイルをロードする場合にのみ最適化がテストされます。一般的な場合、ロードの実際の状況を反映せず、「戦闘に近い」条件下でテストされたソリューションに負けます;



-最適化の前後に、読み込み時間および/またはページ読み込みグラフを含むレポートの図を提供することをお勧めします。



-決定は自由形式で行われます。



-回答は現在の状況に左右されます。



-コンテストの主催者は、この回答がコンテストの条件に該当するかどうかを決定する権利を留保します。



有効なアカウントを持つREG.RU会社のアクティブなクライアント（少なくとも1つのアクティブなサービスを持っている）のみがコンテストの参加者になることができることに注意してください。 賞品は現金同等物と交換することはできません。返却および交換することはできません。



RIT ++のウェルカムチケットに加えて、2つのインセンティブ賞を用意しています。

2位-VPS-2料金で3か月間の無料ホスティング。

3位-VPS-2料金で1か月無料ホスティング。



すべてのREG.RUのお客様には、会議への参加に対して10％の特別割引があります。

コンテストの結果は、2011年4月19日にREG.RU Webサイトで発表されます。



あなたは私たちを驚かせるでしょう。 頑張って
