ID SoftwareはMacおよびLinuxでのquakeliveサポートを放棄します









先日、 ID Softwareが子孫QuakeLiveの Mac OS XとLinuxのサポートを拒否していることが判明しました 。 QuakeLiveは、かなり深刻な変更と改訂されたエンジンを備えた改善されたQuake 3であり、Firefox、Safari、およびIE用のブラウザープラグインという形式のランチャーであることを思い出してください（Windowsでこのリストに非公式にGoogle Chromeを追加できます。しばらくの間、彼はFirefoxのプラグインをサポートしていました。



このような重大な決定の理由は、Google ChromeでのNPAPIサポートの廃止に関する最近のニュースの形で与えられ、 Firefoxは要求に応じてデフォルトでプラグインをアクティブにしなくなりました 。 個人的には、これらは単純な言い訳です。なぜなら、 QuakeLiveは公式にはGoogle Chromeプラグインをサポートしていませんでした。Firefoxで「プラグインを有効にする」ボタンをクリックする必要があるという事実は、それほど怖いものではありません。



また、「 QuakeLive 2.0 」と冗談めかして呼ばれるQuakeLiveのさらなる運命についてはほとんど知られていません。今年の終わりまでに、IDは、たとえばQuake 3のように、ネイティブのWindows-exeランチャーでゲームStandAloneを作ることを約束します。



PS Valveおよび他のゲーム開発会社の最近の議論の中で、John CarmackがLinuxをゲームプラットフォームとして絞るというValveの取り組みに否定的にコメントし、ID Softwareが近い将来Linuxを待つ必要がないことを明らかにした面白い話を思い出すことができます。
