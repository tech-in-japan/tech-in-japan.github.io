Apple Mac OS X 10.5.2アップデートでは、AMD RS780チップセットに言及しています！

Mac OS X 10.5.2アップデートの最初のビルドがリリースされました。 また、インストールファイルのサイズから判断すると、これはMac OS Xの歴史の中でシステムの最大の技術更新であり、350メガバイトのクリーンフィックスです。

最初の外部アセンブリ10.5.2は9C7に指定され、Appleが開発者に特別な注意を払うように依頼するコンポーネントのリストは37項目で構成されています。 その中には、スペルチェック、Docの作業、Rosettaトランスレータ（メモリリークの排除）、Finder環境、iChat、Mail、Safari、Time Machineアプリケーション、ワイヤレスデータ転送のサポートがあります。 すべての変更の完全なリストは、Deep Apple Forumにあります。



この更新により、triakis.kextモジュールが導入されました。このモジュールには、RV670、RV770チップに基づいたビデオカードの操作に必要なコードが含まれています。 したがって、MacBidouilleからの情報が確認された場合、新しいMac Proでは、ATI Radeon HD 38X0または「3番目のATIシリーズの類似物」を待っています。

最後に、アップデートでもAMD RS780チップセットの言及が見つかりました。 オンボードのビデオチップを内蔵し、AMD CPUを使用してマザーボードが設計されているチップセット。AMDのチップセットとプロセッサを搭載した何らかのApple Macモデルの外観です。



たとえば、AMDの内部機能を備えた新しいバージョンのApple TVが登場する可能性があります。 結局、リリースされてから1年が経過したので、更新する時が来ました。

確かに、Appleの弁護士は、デスクトップコンピューターとラップトップコンピューターのみが公式に影響を受けるように、Intelと排他的な契約を結んでおり、Apple TVのセットトップボックスは公式にはコンピューターではない;

また、Apple TVでは、どんな種類のマザーボードとCPUが存在するかを気にする必要はありません。AppleはApple TVのドキュメントでこれについて何も述べていません。 これはVCRなどの消費財であるため、Appleは、たとえば安価なAMD AthlonまたはVIA C7などをローエンドCPUに供給する権利をすべて持っています。 Apple MacOS Xは、おそらく新しいApple TV 10.5.2でスピンしますが、Apple TV専用のインターフェースを使用しています。



別のオプションは、Appleが最終的にすべてのPCクローンに対してMacOS X 10.5をリリースするということです:)



まあ、またはAMDマザーボードとCPU用のドライバーの開発のため、Appleは非公式にHackintoshプロジェクトの開発をサポートしています！



そして最も信じられない選択肢： Appleの幹部は、 「AppleがAMDを購入するべき10の理由」というタイトルの記事を発行した金融出版物である、Seeking Alphaのアドバイスを受けました。
