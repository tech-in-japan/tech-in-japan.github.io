web-standards.ruを開く



今日、良いニュースがあります：約束通り、私たちはフラッグシップWebサイトWeb Standardsをオープンしています。 これは、開発者向けに開発者が作成した記事、ニュース、イベントのお知らせWeb Standards Daysのリソースです。



Web Standards協会の歴史はかなり長く、2006年のWebmasconフォーラムですべてが始まりました。 その後、コミュニティは「WSGロシア」と呼ばれ、開発者や企業にWeb標準を配布する組織であるWeb Standards Groupのロシア語圏でした。 しかし、後に、WSGとの正式な提携を離れ、ロシア語を話す開発者に特化したコミュニティを形成することが決定されました。 Web Standardsプロジェクトの詳細を読んでください 。



クライアントWeb開発に関する記事は、他のブログと同様に定期的に公開されます。 唯一の違いは、他のサイトよりも優れた素材を生産するという目標を設定していることです。 これは、編集者と著者の経験を組み合わせることにより可能です。



オープン時には、次の2つの記事を提供しています。

  • 「 クロスブラウザの丸みを帯びたコーナー 」レフ・ソルンツェフ
  • 「 デザイナーを育てる方法 」オルガ・アレクサシェンコ


イベントセクションは、3年以上にわたってコミュニティで定期的に開催されているイベントの完全なアーカイブです。 イベントでは、常にプレゼンテーションスライドとビデオプレゼンテーションへのリンク、および写真レポートやその他の追加資料を含むレポートプログラムがあります。



このサイトには、クライアントテクノロジーと関連分野に関するニュースフィードがあります。 RSSフィードを購読し、編集用メールに興味深いリンクを送信してください。



記事のセクションに関連資料を補足するために、強力な著者を招待しています。 web-standards.ruの著者になるのは簡単です。記事のドラフトまたは完成したテキストを送信してください。資料の作業に参加者を含めます。



干渉することなく支援してくれたすべての人、およびサイトで積極的に働いたすべての人に感謝します：Slava Oliyanchuk（ miripiruni ）、Mikhail Baranov（ 404 ）、Oleg Roshchupkin（ jahson ）、Sergey Mezentsev（ thebits ）およびOlga Aleksashenko（engel）。



トピックへのコメントで、 wst @ web-standards.ruまたはその場でお待ちしているコメント、レビュー、提案、ニュース、および出版物に非常に喜んでいます。
