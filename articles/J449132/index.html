Android Studioのトップ17プラグイン







Android Studioには数百のプラグインがあり、その数は日々増えています。 これは主に2つの理由で起こります。 まず、Android StudioはAndroidアプリケーションを構築するための公式の統合開発環境です。 次に、Android Studioの基盤となるプラットフォームであるIntellij IDEAを使用すると、両方のIDEと互換性のある便利なプラグインを開発して簡単にインストールできます。 どのプラグインをインストールしますか？







もちろん、単一のプラグインを追加するのではなく、必要なライブラリをインストールするだけで、「裸の」Android Studioでプログラミングできます。 しかし、なぜですか？ Android Studioのプラグインは、作業をより効率的で楽しいものにします。 次に、（少なくとも）パフォーマンスを3倍にできるAndroid Studioプラグインのトップについて説明します。







Android Studioに最適なプラグインのリストには、時間でテストされたいくつかの基本的なツールと、Androidアプリケーションの経験豊富な開発者が推奨するいくつかのあまり知られていない宝石が含まれています。 だから、ここに私たちのお気に入りのAndroid Studioプラグインがあります：







1. ストリング操作









名前が示すように、このプラグインは文字列を使用した面倒な手作業の時間を節約します。 テキストスタイル （camelCase、kebab-lowercase、KEBAB-UPPERCASE、snake_case、SCREAMING_SNAKE_CASE、dot.case、lowercase、Capital Letters、PascalCase）の変更、 エンコード/デコード、増加/減少、並べ替え、フィルター、整列を行うことができます 。







2. コドタ









次のプラグインで何ができるかを見てください。 Codotaプラグインは、機械学習を使用してコードを自動補完します 。 プラグインは、独自のコンテキストとともに、数百万のJavaプログラムのコード例から学習します。 初心者と経験豊富なJava開発者の両方が使用するCodotaは、開発をスピードアップし、コードに現れる厄介なバグを排除するのに役立ちます。







プラグインに加えて、Codotaは、適切な場所にコピーして貼り付けるために、一般的に使用されるJavaコードのスニペットの広範なライブラリも提供します。







3. CPU使用率インジケーター









このシンプルなプラグインは、一見すると思うよりもはるかに便利です。 コードとアプリケーションのコンパイルと実行がより複雑になると、これはプロセッサに影響します。 プロセッサ使用率インジケータの最も便利な機能の1つは、 スレッドダンプを生成して、バックグラウンドスレッドでプロセッサをどのように詰まらせるかを示す機能です 。







4. レインボーブラケット









ブラケットは、常にプログラマにとって頭痛の種でした。 この不足しているブラケットを見つけるのに何時間かかりましたか？ やめて！







座って既に閉じているブラケットとまだ閉じていないブラケットを把握する代わりに、Rainbow Bracketsプラグインを使用してブラケットを色付けします 。







Java、Scala、Clojure、Kotlin、Python、Haskell、Agda、Rust、JavaScript、TypeScript、Erlang、Go、Groovy、Ruby、Elixir、ObjectiveC、PHP、HTML、XML、SQL、Apex、C＃、Dartなどをサポートしています言語。







5. プロジェクトロンボク



Androidアプリケーションの複雑なJavaコードには、多くの定型コード（コンストラクター、ゲッター、セッター）が含まれることがよくあります。 これは、読みやすさとコードサポートの点で面倒で困難な場合があります。 Project Lombokは、IDEに接続してこのテンプレートコードを生成するJavaライブラリであり、コードを簡潔で読みやすいものにします。 前後のコードの違いは非常に驚くべきものです。







コンパイル時のライブラリであるLombokを使用しても、アプリケーションの負荷が大きくなることはありません。 さらに、 サイト上のプラグイン開発者は、ユーザーおよびプラグイン開発者に多くの有用なリソースを提供します。







6. Android Drawable Importer









Androidアプリケーションの開発時にdrawableを使用する場合、これは注意すべきプラグインです。







Androidデバイスのすべてのサイズと画面解像度にリソースを適合させるために、各Androidプロジェクトには描画可能なフォルダーがあります。 すでに正しいフォルダに正しい名前のリソースを提供しているParadiseのデザイナーに恵まれているなら、心からうらやましい。 しかし、ほとんどの場合、あなたはそれほど幸運である可能性は低く、自分でPNGファイルの名前を変更し、目的のフォルダーに移動します。







Android Drawable Importerは、膨大な時間を節約するプラグインです。 これにより、必要なサイズと権限への規制のために個別のリソースをインポートし、必要なフォルダーに自動配置するためのzipファイル全体をインポートできます 。







7. ベクトル描画可能なサムネイル



ベクターXMLファイルをプレビューするには、アプリケーションを実行する必要があります。 Vector Drawable Thumbnailsは、これを取り除き、ワンクリックでベクター図面を表示するのに役立ちます。







8. Android Drawable Previewプラグイン



プロジェクトに描画可能なコンポーネントが多数ある場合、それらをナビゲートするのは非常に困難です。 IDEに簡単な画像プレビューがあればいいですね。 Android Drawable Preview Pluginを使用すると、これが可能になります。







このAndroid Studio用プラグインは、プロジェクトファイルツリーのデフォルトアイコンを要素プレビューに置き換えます 。 しかし、drawableに適切な名前を使用することを忘れないでください、それはまだ重要です。







9. その色に名前を付ける









神話を打ち破ることから始めましょう。 すべての女性がターコイズとアクアマリン、またはナスとプラムを区別できるわけではありません。 このため、色覚異常である必要はありません。







このプラグインの存在と人気は、ほとんどの開発者にもこの魔法の色認識スキルがないことを示唆しています。







Name That Colorプラグインは、Android Studioのリソースファイルで直接クリップボードにある色に名前を付け、最も近い一致する色の名前を付けます。 xmlファイルのlighter_light_pink時代は終わりに近づいています。







10. bundletool



bundletoolは、Gradle、Android Studio、およびGoogle PlayがAndroidアプリバンドルを作成したり、アプリバンドルをデバイスにデプロイされたさまざまなAPKに変換したりするために使用する基本ツールです 。 Googleのこのツールは、APKパッケージをより効果的に使用するために必要です。







11. バターナイフゼレズヌイ



Android ButterKnifeは、 依存性注入ライブラリです。 原則として、これによりコードの可読性が向上し、Viewコンポーネントを検索したりリスナーを追加したりするためにヒープにコードを混在させるのではなく、ロジックに集中できるようになります。













Android ButterKnife Zeleznyは、アクティビティ、スニペット、またはアダプターで選択したXMLレイアウトからButterKnifeインジェクションを作成するためのAndroid Studioプラグインです。 ほとんどのButterKnifeマニュアルでは、Zeleznyのインストールを推奨しています。







12. Android入力



Android Inputは、 Androidデバイスやエミュレーターに直接テキストを簡単に入力できるAndroid Studio用の非常にシンプルですが便利なプラグインです。 最後に使用したデバイスと最後に入力したテキストを記憶します。













13. ADBのアイデア



Intellij IDEAとAndroid Studioの両方のこのプラグインは、IDE でさまざまなADBツールコマンドの便利なキーボードショートカットを追加します。 これはやり過ぎのように思えますが、開発者は、ADB IdeaがAndroidアプリの開発とデバッグを本当にスピードアップできると言います。







14. adb-enhanced



「Androidのテストと開発用のスイスナイフ」と呼ばれるadb-extendedは、さまざまなスクリプトを実行するためのコマンドラインインターフェイスです。 これにより、画面の回転、バッテリーセーバーモード、データセーブモード、スタンバイモード、許可の付与または取り消しなど、 多くの潜在的にエラーのあるアプリケーションの動作をテストできます。







15. ADB WIFI



このプラグインにより、デバッグのためにWiFi経由でデバイスをADBに簡単に接続できます 。 他の多くの小さくても便利なプラグインと同様に、これはコマンドラインで実行できる一連のコマンドの単なるラッパーです。 しかし、このプロセスをより快適にしてみませんか？







16. ここにはドラゴンがいます



ドラゴンズには、IntellijとAndroid Studioプラグインがあります。これにより、「汚れた」Javaメソッドに@SideEffectアノテーションを付けることができます 。 このメソッドを呼び出すと、プラグインは小さなドラゴンアイコンを表示します。













さらに、このプラグインは可愛すぎてリストに含められません。 これらはドラゴンです！
