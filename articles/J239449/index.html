3. DEBIAN LINUX-ローカルインストール



便利な方法でプラグインをダウンロードします。

ダウンロードページから

https://www.google.com/tools/dlpage/hangoutplugin?hl=en



または

https://www.google.com/tools/dlpage/hangoutplugin/download.html?hl=en



または

https://www.google.com/tools/dlpage/hangoutplugin/download.html?hl=ja&platform=linux_ubuntu_x86_64





直接リンク、

現在のバージョン（推奨）

https://dl.google.com/linux/direct/google-talkplugin_current_amd64.deb





または特定の（最新の）バージョンが可能

https://dl.google.com/linux/talkplugin/deb/pool/main/g/google-talkplugin/google-talkplugin_5.4.2.0-1_amd64.deb





インストール： 

 sudo dpkg -i google-talkplugin_current_amd64.deb


または 

 sudo dpkg -i google-talkplugin_5.4.2.0-1_amd64.deb




Debian 7.6 Wheezy（安定版）では、インストールされたgoogle-talkpluginパッケージのすべての依存関係を満たす必要があります。

それでも、dpkgを使用したローカルインストール中にgoogle-talkpluginパッケージの依存関係が満たされていない場合は、次のように修正できます。 

 sudo apt-get -f install


依存関係は、主要なDebianリポジトリからのaptによって「プルアップ」されます。

4.定期的な更新



パッケージを（リポジトリまたはローカルから）インストールするとき、パッケージに含まれるインストールスクリプト（/ deb：// DEBIAN / postrmおよび/ deb：// DEBIAN / postinst）およびパッケージインストールプロセスの最後に実行されると、上記のリポジトリとキーが追加されます。正しいGoogleリポジトリ設定を毎日確認するためのCronタスクスケジューラディレクトリ内のシェルスクリプトファイル（/etc/cron.daily/google-talkplugin-> / opt / google / talkplugin / cron / google-talkplugin）へのシンボリックリンク。 プラグインを定期的に更新するには、タスクをCronまたはAnacronに追加できます。

パッケージはいつでも手動で更新できます。 

 sudo apt-get install google-talkplugin
