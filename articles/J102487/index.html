JVC PICSIO-インタラクティブなエンターテインメント、簡単な制御、ビデオのダウンロードを備えたポケットビデオカメラ

PICSIO GC-WP10およびGC-FM2には、3Xインチのタッチスクリーン、クイック同期およびアップロード機能、ハイエンドHD画像を作成する機能、

低照度条件と耐水性での卓越した記録品質と同様に。



JVC（日本ビクター株式会社）は、2つの新しいPICSIOハンドヘルドビデオカメラ、GC-WP10およびGC-FM2、「All for Fun + Fun for All」（すべての人向けのデバイス）を紹介します。ユーザーエンターテインメントと驚くべきデータ転送機能で他の人を楽しませる能力。 このカメラの機能、簡単な制御と表示のための3Xインチのタッチスクリーン、高品質のフルHDビデオ、低照度条件下での超高感度、防水ハウジング（GC-WP10のみ）。



機能の1つは3Xインチディスプレイで、最大の利便性と直感的なコントロールのために移動可能なグラフィックアイコンを表示します。 2つ目は、ディスプレイ自体のサイズです。これにより、撮影したばかりの素材を、グループでも単独でも最大限の利便性で表示できます。 さらに、タッチスクリーンを押す強さを決定するための統合システムにより、手袋を付けた状態でもデバイスを制御できます。これは、ウィンタースポーツや寒い天候を記録するときに非常に快適です。



PICSIO CMOSセンサーを使用すると、最大5メガピクセルまでの自然な非補間解像度でJPEG写真を作成できます。 さらに、最大1920x1080P / 25 fpsの解像度でフルHDビデオを録画できます。これにより、画像安定化システムとともに、HDMI出力を使用してHDTVで高品質の映像を表示できます。 4倍デジタルズームと膨大な数の異なる撮影モードも利用できます。 ビデオ録画はMPEG-4 AVC / H.264で実行され、MP4ファイルに保存されます。MP4ファイルは簡単に編集でき、時間と品質を損なうことなく多くのデバイスに送信できます。 機能の1つであるTime-Lapseを使用すると、長期イベントまたは自然現象を短いファイルに記録できます。



プリインストールされたソフトウェアのLoiLoScope EX for Windowsを使用すると、直感的かつ迅速に映像を編集してYouTube、Facebookにアップロードしたり、iTunesにエクスポートしたりできます。 ビデオカメラがコンピューターに接続されるとすぐに自動的に起動します。つまり、コンピューター上でのみプログラムをインストールしたり、ビデオカメラを操作したりする必要はありません。 ダウンロードプロセスと編集は、いくつかの簡単な手順で利用できます。 さらに、ビデオカメラは、クリエイティブなビデオを作成するためのあらゆる特殊効果を統合し、その管理は、アイコンを使用した機能への簡単なアクセスと必要な要素の簡単なドラッグアンドドロップに基づいています。 このソフトウェアはWindows 7 Multi-Touchをサポートしているため、タッチインターフェイスを備えた任意のPCを使用するなど、制御できます。 また、DVDを直接作成するためのデータ転送の可能性があります。



MacユーザーはiFrameビデオ録画モードを使用できます。これは、後でiMovieで編集する場合に最適です。 両方のカメラは、コンピューターでビデオを編集するプロセスを可能な限り簡素化するために特別に設計された形式であるIFrame形式で記録する機能も提供します。 IFrame-この形式をサポートするMacおよびPCアプリケーションの両方と互換性があります。



PICSIOビデオカメラには、SD / SDHC / SDXCメモリカードスロットが装備されています。



GC-FM2ポケットビデオカメラには、側面に押し込まれる特別なUSBコネクタが装備されています。 どちらのモデルもUSBコネクターから充電されるため、バッテリーを放電する心配なく、ファイルを転送した後すぐに撮影を続けることができます。 USBケーブルが提供され、ACアダプターがオプションとして利用可能です。



PICSIO GC-WP10ビデオカメラのモデルは、最大3メートルの深さ（IPX8証明書）で防水であるため、水中、高湿度、または雨の中でビデオを作成できます。 プールや水域に落ちたために壊れた膨大な数のカメラについて話すと、GC-WP10はそのような事故に対して非常に必要な保険を提供します。 また、GC-WP10ビデオカメラにはMP3ボイスレコーダーが統合されており、講義からコンサートでのハングアウトまで、あらゆるものを完全に録音できます。



JVC GC-WP10およびGC-FM2は9月1日から利用可能になります
