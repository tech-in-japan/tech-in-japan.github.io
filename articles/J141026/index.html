Ubuntu 12.04 2番目のベータ版がリリースされました

ほんの数時間前に、Ubuntu 12.04の2番目のベータ版がリリースされ、すでにダウンロード可能です。 次のリリースはUbuntu 12.04のフルバージョンで、そのリリースは4月26日に予定されています。



そのため、すぐにwebupd8.orgのWebサイトから小さなデモを行い、 Ubuntu 12.04 Beta 1のリリース以降のすべての変更を示します。







主な変更点について話します。



団結



標準のUbuntuシェルがバージョン5.4から5.8に更新されました（ Unity 5.8 、 Unity 2D 5.8の詳細 ）。これには多くの変更と修正が含まれています。 主なものを検討してください。



ダッシュ






Dashでグローバルな変更は行われませんでした。外観が徹底的に再設計され、すべてがトリミングされ、検索で何も見つからなかったときに新しいホットキーとメッセージが追加されました。



ハド






Canonicalの新しいメニューでは、彼らは再び何も変更せず、外観とアニメーションを改善するだけでなく、Altキーがホットキーで使用された場合の競合を修正しました（ HUDメニューを呼び出すのに役立つことを思い出します）



標準アプリケーション



Ubuntu One






Ubuntu Oneプログラムのインターフェースはかなり更新され、使いやすくなり、すべてが直感的になりました。 サービス自体に変更はありません。



Gwibber






Gwibberの標準ソーシャルクライアントは更新されていますが、表面的な変更のみが行われています。



風景


バージョン12.04から、標準のLandscapeアプリケーションがUbuntuに追加されました。これは、Ubuntu Linuxがインストールされているコンピューターで監視、更新、およびその他の多くのアクションを実行できる強力なシステム管理ツールです。



Ubuntu Application Center


便利な機能が追加されました。アプリケーションのインストール中に、インストールインジケーター付きのアプリケーションアイコンがサイドパネルに追加されます。 便利なことに、毎回Application Centerを展開する必要はありません。

また、システムはバージョン3.2.12に基づくカーネル3.2.0-20.33を使用します。



システム



最も興味深い革新の1つは、2台のモニターの完全なサポートです。 また、ウィンドウが画面の端にくっつくようにする組み込みオプションがありました。



一般的に、一般的なレビューから判断すると、システムは本当に安定しています。 以前はベータ版を置くのが怖かったのですが、今ではこれで事態は改善されています。



設置



Ubuntuを2番目のベータ版にアップグレードするには、ターミナルで次のコマンドを入力します。

update-manager -d





また、私のウェブサイトからUbuntuの最新バージョンをいつでもダウンロードできます ： Ubuntu 12.04をダウンロードします。
