Windows 8アプリケーションデザインストーリー





MSDNでWindows 8の作成とWindows 8のアプリケーションの設計と開発 に関するガイドの更新のブログ作成者である同僚は、最近、独自のアプリケーションの設計と一般的なシナリオのアプリケーションの設計の両方で興味深い話を投稿しました。 以下に、これらのストーリーへのリンクの選択があります。





実際のアプリケーション



人物アプリ



連絡先の管理は、特に連絡先へのアクセスに使用するPCとデバイスの数を増やした後、多くの人にとって困難な作業です。 簡単に移動して接続できるように連絡先をクラウドに保持することは、ソリューションの一部にすぎません。 Windows 8と新しいPeopleアプリにより、クラウドストレージを次のレベルに引き上げ、既に使用している他のサービスに接続できるようにします。 したがって、電子メール、サービスアカウント、ソーシャルネットワークの連絡先は、すべてのWindows 8 PCおよび携帯電話で利用できる、簡単にアクセスできる便利な場所にまとめられます。



メールアプリ



Windows 8 Release Previewのリリース以降、毎日何十万人もの人々が予備バージョンのメールアプリケーションを使用しています。 また、新しいWindowsストアでこのアプリケーションを常に更新しており、さらに更新する予定です。 この出版物では、Mailアプリケーションを内側から見て、その構造と機能、特にMetroスタイルのアプリケーション設計の原則に関連する機能について説明します。 アプリケーションのすべての機能や将来実装される機能の一覧は表示しませんが、その構造とWindows 8との統合に焦点を当てます。



カレンダーアプリ



この記事は、MailおよびPeopleアプリケーションの記事に記載されている資料に基づいており、Calendarアプリケーションについて詳しく説明しています。 これらのアプリケーションを、最も重要なクラウドサービスに接続する単一の補完的な通信スイートに統合するために、素晴らしい仕事をしました。 この記事では、Windows 8との統合、現在のプレリリースバージョンのコンポーネントの一部、および現在開発中の機能について詳しく説明します。 また、システム開発の歴史について簡単に触れ、反復について少し説明します。



写真アプリケーション



Windows 8のいくつかの新しいアプリケーションに関する一連の記事の最後で、新しいフォトアプリについて説明します。 このアプリケーションとMetroスタイルの設計原則により、さまざまなソースから写真を収集し、それらを表示および共有できるアプリケーションを開発することにしました。





典型的なシナリオ



ゲームシナリオ



通常、ゲームはいくつかのコンポーネントで構成されます。 ゲーム自体に加えて、他の領域（実績画面など）、オプションメニュー、ヘルプメニュー、およびユーザーインタラクションのさまざまな領域へのエントリポイントへのアクセスを提供するメニューがあります。 ゲーム自体の内部には、さまざまなメニューにある多数のチームも参加できます。 これはすべてユーザーが操作できるフォーマットされたコンテンツであり、Windows 8 Release Previewのファーストクラスのゲームでは、ユーザーはユーザーインターフェイスのランダムな中間要素をすべてバイパスして、このコンテンツを直ちに表示します。 このセクションでは、ユーザーインタラクションを改善しながら、Metroスタイルでゲームを設計する方法について説明します。



面白いシナリオ



Windows 8は、タッチタブレットから高解像度のラップトップやデスクトップコンピューターに至るまで、さまざまなデバイスと対話するための幅広いオプションを提供します。 Windows 8の新機能の多くは、マルチメディアおよびエンターテイメントプログラム専用に設計されており、多様なコンテンツのパブリッシャーに、視聴者とつながり、対話し、通信する新しい方法を提供します。



ニュースシナリオ



Windows 8は、タッチセンシティブタブレットからフルサイズのラップトップやデスクトップコンピューターまで、幅広いデバイスをサポートしています。 このようなさまざまなフォームファクターを使用して、ニュース発行者はさまざまなタスク用の使いやすいインターフェイスを作成でき、1つのアプリケーションで多くのユーザーにリーチできます。



パフォーマンスシナリオ



Windows 8は、生産性アプリケーションに最適なプラットフォームです。 既存のパフォーマンス向上アプリケーションが正常に動作できるようにする通常のデスクトップ環境に加えて、アプリケーション用の新しいMetroスタイル環境では、新しいタイプのパフォーマンス向上アプリケーションを作成できます。 このセクションでは、Windows 8でMetroスタイルと機能を使用するときに、アプリケーションのパフォーマンスを向上させるさまざまな方法を提供します。
