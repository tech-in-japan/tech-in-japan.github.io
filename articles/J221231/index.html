パブリッシャーは、Microsoft Windowsオペレーティングシステムの主要コンポーネントの内部構造と操作アルゴリズムに関する伝説的なWindows Internalsブックの第6版の第2部を公開しました。 初めて、Windows Internalsは作成者によって2つのボリュームに分割されたことを思い出してください。 昨年、最初の巻が出ました 。それについてはここで書きました 。







Windows Internalsブックの第6版は、 Windows 7およびWindows Server 2008 R2コンポーネントの内部コアをカバーしています。

この本はあなたを助けます：



  • オブジェクトマネージャからレジストリまで、システムのコアと制御メカニズムがどのように機能するかを理解します。
  • カーネルデバッガーなどのツールを使用した内部データ構造システムの詳細をご覧ください。
  • 内部からWindowsセキュリティモデルを調べて、データへのアクセスを許可する方法を確認します。
  • Windowsでの物理メモリと仮想メモリの管理方法を理解します。
  • ドライバープロトコルAPIやネットワークアダプタードライバーなど、Windowsネットワークスタックを上から下まで調べます。
  • ファイルシステムアクセスの問題とシステムブートの問題を理解する。
  • 障害を分析する方法を学びます。


本の最初の部分は、主要な概念を紹介し、本で使用されるツールについて説明し、システムの一般的なアーキテクチャとコンポーネントを説明する2つの章から始まります。 次の2つの章では、基本的なシステムメカニズムと制御メカニズムについて説明します。 最初の部分は、オペレーティングシステムの3つの主要コンポーネントの調査で終わります。まず、プロセス、プログラムのスレッドとタスク、次にセキュリティ、そして次にネットワークです。



残りの主要なサブシステムには、I / O、長期ストレージおよびメモリ管理メカニズム、キャッシュマネージャーおよびファイルシステムが含まれます。第2部で説明します。 2番目の部分は、オペレーティングシステムの起動と停止のプロセスと、緊急ダンプ分析ツールの説明で終わります。
