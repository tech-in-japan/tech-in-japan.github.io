ルッキングシャープと呼ばれる開発者のブログエントリ（「スピードアップ」または「シャープニング」と翻訳できる）は、次のように説明しています（私のコメントは斜体で表記されています ）。



Kestrelの従来の開発と並行して、ブラウザの外観を改善するための別のプロジェクトもありました。 新しいスキンは、ペレグリンの前奏曲である可能性が高く、将来を覗き見ることができます...



一言で言えば、新しい外観は明確であると説明できます。 たとえば、空港で使用されているシンボルなど、直感的な意味を持つ明確な輪郭と明確なアイコンで、同時にスカンジナビアの伝統に沿ったスキンを作成したかったのです （ 興味深いのは何ですか？ ）。 ちなみに、新しい外観はWiiおよびOpera 9.5 for Windows Mobileのバージョンによりよく適合しています。



インターフェイスの配色は、ページの表示から注意をそらすのではなく、同時に要素によって実行されるアクションを明確かつ明確に示すために選択されました（たとえば、「停止」ボタンが赤になります）。



ブラウザ自体の品質と細部へのこだわりがその外観に反映されるはずです。そのため、プロ仕様のオーディオおよびビデオ機器のデザイン要素を使用して、素材、ボタン、タブを作成できました。



デフォルトの設定では、パネルを開くためのボタンを追加しました（ これらはサイドバーです ）。これは、パネルが非表示の状態にある方がはるかに便利だからです。 さらに、パネル選択バーに「 + 」ボタンを追加しました。これにより、パネルをさらに追加したり、不要なパネルを削除したりできます（ 興味深いことに、「 + 」ボタンでパネルを削除するにはどうすればよいですか？ ）。 これにより、Operaは初心者にとって使いやすくなり、製品の開発とともに成長し、残りは新しい、そして新しい可能性を開くことができます。



[新しいタブ]ボタンは、最後のタブの右側にあります。 IE7ユーザーは使い慣れており、簡単に切り替えることができます（ クールなマーケティングの動き、ハァッ ）。後で、Ctrl + Tを使用するか、タブバーをダブルクリックできることを覚えています。 ボタンが閉じたタブを復元するためのボタンと組み合わされた先の尖った形状を持っていることに気付くかもしれません。これは速度とパフォーマンスのヒントです（ 方法は次のとおりです）。
