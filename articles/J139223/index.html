Windows Server 8ベータ

2月29日の夜、Windows Server "8"のベータ版がダウンロード可能になりました。 ISOファイル（Datacenter Edition、重量約3.5 GB）、または既製のVHDの形式でダウンロードできます。 配布キットはDVDから約15分でインストールされ、さらに一部のWS8 コンテンツが既に投稿されています。 新しいサーバーには明らかに何かがあります。



ネットワークサポート、リモートアクセス、VDIスクリプト、複数サーバーのリモート管理、ストレージの操作、セキュリティなど、ほぼすべての領域に多くの改善と新機能があります。 しかし、最大の変更点は仮想化と新しいHyper-Vでした。 私はほんのいくつかに注意してください：

  • ホストごとに最大160個の論理プロセッサ。
  • 仮想マシンごとに最大32個の仮想プロセッサ。
  • 仮想マシンごとに最大1 TBのRAM。
  • クラスター内の最大64ノード。
  • NICチーミングの組み込みサポート（厳密には、Hyper-Vだけでなく、サーバー自体にも適用されます）。
  • ライブストレージの移行。
  • クラスタ外のホストでのライブマイグレーションのサポート。
  • 仮想マシンのファイバーチャネル仮想アダプター
  • サイト/データセンターの復元力のためのHyper-Vレプリカ。


これは機能の不完全なリストです。さらに、製品の最終バージョンがリリースされるまでに、たとえば数値などの一部のパラメーターが変更される可能性があります。



主にプライベートクラウドの構築に関連するWindows Server "8"のリストされた機能の多くについては、 ITCampイベントで詳しく説明します。 また、インターネット上でレポートをブロードキャストする予定です。 時間があります-入ってください。
