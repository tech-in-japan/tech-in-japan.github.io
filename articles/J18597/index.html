UbuntuでのWiFiセットアップ

Ubuntuを6か月以上使用しています。 その中のすべてが私に合っているように見えますが、時々、鉄の不完全なサポート、ほとんどの場合無線LANのような些細なことで人生が影に隠れます。 このhabratopikeでは、ハードウェアでWi-Fiをセットアップした経験を共有して、初心者を数時間のグーグルから、そしてLinuxの経験豊富なユーザーを質問から救おうとしています。 私はsmartov habrayuzerの 要請でこれを行います。 このトピックに関する最も包括的なリファレンスを作成できる場合は、記事にあなたの経験を追加できて嬉しいです。







Intel PRO /ワイヤレス



Win- Intel PROSet / Wireless用の適切なソフトウェアを備えたWiFiカード。

例としてMSI L745を使用すると、すべては私の介入なしで機能しました。 NetworkManagerアイコンをクリックします-エリア内のすべてのネットワークのリストを取得します。

ノートブックASUS Pro 52RLの例でwifiを上げました。 asusサイトでも、どこにも情報はありません。 Googleはロシア語で4つのリンクのみを提供しています。 メディア市場でのみ販売されています。 まあ、ポイントに近い。

Ubuntu 7.10



Atheros AR5006EGワイヤレスカードが搭載されています。 Ubuntaは、動作しない独自のドライバーを検出し、デフォルトで有効にします。 無効にする必要があります。 これを行うには、行を/etc/modprobe.d/blacklist-commonファイルに追加します（そうでない場合は、ブラックリストファイルに追加します）。

ブラックリストath_pci


再起動：

$ sudo init 6


さらに、Windows用のドライバーが必要です。 指定されたカードについては、 こちらで入手できます 。 はい、それを機能させる別のプログラムをインストールすると、すぐにドライバーをインストールできます。

$ sudo apt-get install ndiswrapper-common

$ sudo ndiswrapper -i net5416.inf

$ sudo ndiswrapper -m

$ sudo init 6



新しいwlan0インターフェイスがiwconfigリストに表示されます。 その後、すべてがインテルPROと同じです。
