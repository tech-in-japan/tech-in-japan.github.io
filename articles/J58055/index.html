Windows 8開発が開始されました

最近、Windows 7は標準タイプのオペレーティングシステムの最新バージョンになり、次のOSは完全に「クラウド」の原則に基づいているという話があります。 彼女は彼女がすでにコードネームみどりを持っていると言います。



実際、これらのどちらも真実ではありません。 マイクロソフトに近い情報筋は、この問題に関するいくつかの明確化を提供し 、レドモンドの巨人の本当の計画について話しています。 実際、Windows 8の開発はすでに計画されており、同社はこのプロジェクト専用のプログラマーを採用し始めています。 特に、開発者は分散ファイルシステムレプリケーション（DFSR）の新しいバージョンを探しています。 Windows 8で作業するためのプログラマーの移転について話している他の情報漏洩があります。



Windows 7の最も一般的なリリース日は、2009年秋の終わりです。 OSの次のバージョンが同じ計画に従っている場合、2011年中にそれが予想されます（Microsoft自体は公式に2012を呼び出します）。
