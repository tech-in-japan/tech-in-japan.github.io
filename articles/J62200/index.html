インターネットを教えてください、イオタマスター

仕事上のニーズで有名なYotaからMobileWiMAXモデムを手に入れました。 読者の皆さん、私の印象についてお話しします。



GNU / Linux OSでYota向けにSamsungが作ったモデムから始めましょう。 歴史的に、仕事や日常生活でMacOSXやMicrosoft Windowsなどの代替OSをまったく使用していませんが、GNU / Linux（仕事と個人の世界観）の下でのみ仕事をしています。 MadWiMAXチームのメンバーのおかげで、彼らはGNU / Linuxでiotaモデムをサポートするドライバーを作成しました。 自宅のGentooの下で大きなラップトップでそれを集めようとしましたが、そこでは必要ありませんでしたが、奇妙なものが欲しかったため、マナをどこかに変換したかったので、何が見つかりましたか私はこれを行う方法を理解していませんでしたが、それはそれについてではありません。

公式のネットブックでは、eeebuntuを使用しています。なぜなら、ユーザーを作業中のマシンのGentooネットブックに入れるのは奇妙だと考えるからです。 Ubuntu GNU / Linuxのパッケージが見つかり、配信され、すべてがすぐに稼働しました。 ヨータとドライバー開発チームを尊重します。

多少の安定した接続にもかかわらず、数日に一度、接続がモデムが歪む前にどこかで消えてしまうこともあります。



ここで、カバレッジ、スピード、その他の喜びについてお話します。

カバレッジ カバレッジが私にとって非常に重要な場所はいくつかあります。私たちのオフィスがあるUgreshskaya、私が住んでいるVoikovskaya、そしていくつかのポイントです。 私はそれらだけでなく問題を見つけました。ウグレシカヤでは、地図には表示されていますが、建物自体の良好なカバレッジの場所が窓だけでどのように捉えられているか、さらに特定の1つでも、速度は自宅で毎秒1〜3メガビットです2番目のNovopodmoskovskiyレーンの方がよくキャッチされます。速度は同じです。街の中心部では何か悲しいものがあります。私のお気に入りのパブであるBelfastは、ノヴォクズネツカヤにありますが、私がチェックしたキタイゴロドのカフェでは速度が遅く、メガビット未満です。ウィンドウにのみ信号があります。 私が住んでいる同じボイコフスカヤの公園では、自宅よりもはるかによく捉えられ、速度は毎秒6メガビットのピークでした。 カバレッジと速度で見つけたレイアウトの一部を次に示します。最近のローンチに備えて、これを削除しましょう。

その他の楽しみは、もちろん、営業開始日におけるイオタの崩壊（より正確には、私の個人アカウントへのアクセス不能で表される請求システムの崩壊）です。 同僚はそれをしません。準備するほうがいいです。
