8月29日、CocoaHeadsへようこそ





部屋で緊急、雷！ 明日、つまり8月29日、19：00にMail.Ru Groupのオフィスで、iOS / OS Xの開発者による月例会議（偶然、2年記念日）があります。このイベントは、伝統的に毎月最終金曜日に開催されます。



ここで誰もが彼のレポートを作成し、彼のプロジェクトに関する経験と情報を共有できます。 実際、これは同業者の同志との出会いです。 私たちのドアは、初心者と経験豊富な専門家の両方に開かれています。 明日、3つのレポートがあなたを待っています。



最初に、Marat Saytakovが聴衆にiOS用の適応型インターフェースを構築する全く新しい方法について話します。 これは、いわゆるサイズクラスの概念であり、比較的最近のWWDC 2014で開発者が提示したシーズンの目新しさです。もちろん、Maratは自分の言葉ですべてを語るだけでなく、この新しい概念のために作られたインターフェースを使用するいくつかの興味深い機能に関する情報も共有します。 そして最後に、これらのインターフェースを古いiOS 6にバックポートする方法。



Mikhail Bainovによる2番目のレポートは、iOSアプリケーションでデータを格納するために配列とANSI C / 99構造を使用した実際の経験に当てられます。 テーマはそれ自体で語り、減少も追加もしません。



そして最後に、今年の夏にスウィフトの学校を指揮したアレクサンダー・ジミンは、スウィフトにスタイリッシュでファッショナブルなボタンを作る方法を示し、教えます。







さあ、それは面白いでしょう！ 参加するには、 登録する必要があります 。 ちなみに、「Q＆Aサービス」を開始しました。iOS/ OS Xの開発に関連する質問がある場合は、 @ CocoaHeadsMSKまたはイベントの主催者のメールに質問を送信してください。
