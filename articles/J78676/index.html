Blogger + I'm Sync





1週間前、 I'm Syncというサービスの登録が開始されました。 Bloggerサービスを例に、自分のブログで同期している公開ツールを使用する方法についてお話ししたいと思います。



ブログ投稿コンポーネントを追加するには、次のものが必要です。
  • I'm Syncでブログを登録してください。
  • HTML / JavaScriptガジェットを使用して、ブログページに公開コンポーネントを追加します。
  • デザイン設定でブログの幅を広げます。
結果はhttp://iamsyncdemo.blogspot.comで自分で確認できます。 これで、今後のイベント、写真、ビデオ、連絡先に関する情報をブログに追加できます。



さらに、I'm Syncでは、 http ： //iamsyncdemo.blogspot.com/#mmp=ViewPhotoPage&mmpid=143などの特定のイベントまたは写真へのリンクを提供できます。



I'm Syncを使用できるユーザーについて一言。 まず第一に、これらは企業やコミュニティのウェブサイト所有者です。 このサービスは、個人のWebサイトやブログで使用することもできます。たとえば、あなたや読者の興味を引くイベントを公開するために使用できます。 パブリケーションの任意のコンポーネントを個別に追加できます。たとえば、イベントのみ、写真のみ、または例のようにまとめて追加できます。



デフォルトでは、公開コンポーネントはサイトのCSSを使用します。 ただし、スタイルをオーバーライドする場合は、外観設定で独自のCSSをロードできます。



同様に、I'm Syncサービスを他のブログ（ Wordpress 、 Tumblr ）、コンテンツ管理システム（ Drupal 、 uCoz ）および通常のサイトと組み合わせて使用​​できます。 JavaScriptを埋め込むことができる場所ならどこでも。
