Zealz GK802-Ubuntuをインストールできるミニコンピューター



Androidオペレーティングシステムを搭載したミニコンピューターとメディアプレーヤーをお持ちの方はほとんど驚かないでしょう。 しかし、灰色の塊の中でも際立っているデバイスがまだあります-これはZealz GK802ミニコンピューターで、Androidバージョン4.0.4がデフォルトでインストールされています。

彼の何が面白いですか？

しかし、興味深いのは、4コアのFreescale i.MX6Quadチップ上に構築されており、完全なUbuntuアセンブリがあることです！

►Ubuntuをインストールする

現在、GK802にインストールできるUbuntuのバージョンがいくつかあります。

これはUbuntu 11.10で、GPUおよびVPUとUbuntu 12.04のハードウェアデコードサポートがあります。

Ubuntu 11.10をインストールするには、次の手順を完了する必要があります。

1.次のコマンドを実行して、Ubuntu 11.10、uboot、カーネル、およびモジュールのイメージをダウンロードします。 

wget http://dl.miniand.com/jas-hacks/gk802/ubuntu_gk802.img.gz wget http://dl.miniand.com/jas-hacks/gk802/u-boot.imx wget http://dl.miniand.com/jas-hacks/gk802/uImage_3.0.35-0269_cpu_freq wget http://dl.miniand.com/jas-hacks/gk802/modules_3.0.35-02695.tar


2.イメージを解凍します。 

 gzip -d ubuntu_gk802.img.gz


イメージ、uboot、およびカーネルをmicroSDメモリカードにコピーします 

 sudo dd if=ubuntu_gk802.img of=/dev/<sd_device> sudo dd if=u-boot.imx bs=1k seek=1 of=/dev/<sd_device> && sync sudo dd if=uImage_3.0.35-0269_cpu_freq of=/dev/<sd_device> bs=1048576 seek=1 && sudo sync


/ dev / <sd_device>をシステム内のsdカードへのパスに置き換えます。 私の場合、ラップトップの内蔵カードリーダーを介して接続する場合、パスは/ dev / mmcblk0でした 。

ファイルを解凍するときにmicroSDカードのサイズが十分ではないというエラーが発生した場合は、イメージの最後に重要なデータはありません。

4. microSDカードをコンピューターから取り外し、再度挿入してrootfsをマウントします。 これが自動的に行われなかった場合-コマンドを使用して手動でマウントします 

 sudo mount -t vfat /dev/sdc /media/ubuntu


5.カーネルモジュールをrootfsにコピーします。 

 tar xvf modules_3.0.35-02695.tar sudo cp -r modules/3.0.35-02695-ga09806b-dirty /media/ubuntu/lib/modules


6.メモリカードをアンマウントし、コンピューターから取り外します。

7. G802を分解し、Linuxイメージのメモリカードを挿入し、組み立てて実行します。



電源を入れてしばらく待ち、しばらくするとUbuntuが私たちに会います。

次のようにUbuntu 12.04をインストールします
この指示は、Ubuntu 11.10がmicroSDカードに既にインストールされていることを前提としています。

1. rootfsをダウンロードする Ubuntu 12.04

2. microSDカードのイメージから既存のrootfsを削除します。 

 rm -rf <sd card mount point>/ubuntu/* sync


3.新しいrootfsを解凍します 

 cd <sd card mount point>/ubuntu tar xvf linaro_12_04_gpu.tar.gz sync


4.カーネルをダウンロードして、カードに書き込みます。 

 wget http://stende.no-ip.info/jas/uImage_cpufreq sudo dd if=arch/arm/boot/uImage_cpufreq of=/dev/sdc bs=1048576 seek=1 && sudo sync


5. ubootをダウンロードして、カードに書き込みます。 

 wget http://stende.no-ip.info/jas/u-boot.imx dd if=u-boot.imx bs=1k seek=1 of=/dev/sdc && sync


6. G802を分解し、Linuxイメージのメモリカードを挿入して、組み立てて実行します。



さまざまなLinuxビルドがArmTechフォーラムにあります。



ご覧のとおり、LinuxオペレーティングシステムからG802のメモリカードにUbuntuをインストールできます。 Linuxを手元に持っておらず、Windowsしか持っていない場合は、フラッシュドライブにUbuntuをインストールし、USBから読み込んだ後、GK802のmicroSDカードにイメージをインストールします。 これを行うには、以下を行う必要があります。

1. Ubuntuディストリビューションをubuntu.comからダウンロードします

2. UbuntuをUSBフラッシュドライブにインストールするプログラムをダウンロードします

3.フラッシュドライブを準備し、画像を記録するプログラムを実行します。

4.インストールするオペレーティングシステムの種類、記録するデバイスを選択します。

5.イメージをフラッシュドライブに書き込みます。

6.コンピューターを再起動し、起動時にBIOSに入ります（コンピューターの起動時にログインボタンが表示されます）

7.ブートシーケンス設定で、最初に[リムーバブルデバイス]を選択し、設定を保存して再起動します。

8.開始時に、今すぐUbuntuを試すか、すぐにインストールするかという提案があります。 試すことを選択します。

以下は、Linuxからのインストールと同様に、microSDカードにイメージをインストールする手順です。
