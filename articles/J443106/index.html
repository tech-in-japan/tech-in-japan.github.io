USB4の発表：標準について知られていること

USB Promoter GroupはUSB4を導入しました。 新しい標準の帯域幅は40 Gb / sで、USB 3.2、USB 2.0、およびThunderbolt 3との下位互換性があります。





/ Flickr / Rolf Dietrich Brecher / CC BY-SA



少しの背景



2年前、Intel は Thunderbolt 3を非独占的なライセンスの下でリリースし、ハードウェアメーカーにロイヤリティの支払いを免除することを約束しました。 2019年の初めに、ITの巨人は約束を果たしました 。 このオープン仕様は、USB Promoter Groupが開発したUSB4標準の基礎を形成しました。 そのドラフト版は、 USB-IF非営利組織のメンバーである50社によって既に研究されています。



この規格の正式リリースは今年の半ばに予定されており、USB Type-Cのアップグレードとともに行われます。 さらに、下半期のUSB Developer Days 2019では、標準の作成者は、USB4およびUSB Type-Cの操作に関するテーマ別トレーニングを実施することを約束しています。 ただし、USB4を搭載した実際のデバイスは、2020〜2021年まで市場に登場しません。



USB4の新機能



USB-IFは、USB4の詳細な仕様をまだ明らかにしていません。 ただし、20ギガビットUSB 3.2 Gen 2×2の2倍の帯域幅に加えて、新しいプロトコルを使用すると、画像（HDMI 2.0およびDisplayPort 1.2）を同時に充電および表示できることが既に知られています。 双方向操作は、既存のUSB-Cケーブルを使用して実行されます。 このインターフェースは、最大100 Wの電力での充電もサポートしており 、60 Hzのリフレッシュレートで最大2台の4Kディスプレイまたは1台の5Kディスプレイを接続します。

標準に関する潜在的な問題



ソリューションの欠点の中でも、USB4標準が公開されるため、専門家は新しいケーブルの品質を確認することの難しさを強調しています。 その結果、消費者はデバイスポートを「焼き付ける」ことができる低品質の製品を入手する危険にさらされています。



この状況は 、USB-C 3.0ケーブルですでに発生しています。 メーカーの1つは、商品を販売から早急に撤回しなければなりませんでした。 その後、USB Promoter Groupチームは 、USB Type-C認証標準をリリースしました。これは、ガジェットまたはケーブルの「真正性」を確認することになっています。 おそらく、組織はUSB4の場合に同様の手順を実行する必要があります。



Hacker Newsの居住者は、USB4の一部の機能が仕様で「オプション」として示されており、その実装はメーカーの裁量に任されていることにも注意しています。 これはすべて、顧客の間で混乱を招き、デバイスのケーブルまたはポートが損傷するリスクを高める可能性があります。



USB 3.2アップデート



2019年、USB-IFはUSB 3.2アップデート（USB 3.2 Gen 2×2（前述））の導入も約束しました。 この仕様の主な利点は、最大20 Gb / sの帯域幅の増加と、10 Gb / sの速度での双方向データ転送の可能性です。 以前のバージョンとの下位互換性も残っています。





/ Flickr / GEEK KAZU / CC BY



マイナスの中でも、専門家は標準の複雑な名前を強調していますが、これもまた混乱を招きます。 以前のバージョンがGen 1およびGen 2と呼ばれていたことを考えると、命名法は非論理的に見えます。同時に、Ars Technica は USB 3.2が5、10、20 Gbit / sの帯域幅を示すようになったことに注目します。 これは不正なメーカーが消費者を誤解させ、より悪い特性の製品をより高価に販売するために使用できます。



一般的に、コネクタの未来はUSB-Cにあると言えます。 CES 2019で発表されたデバイスのほとんどは、このタイプのコネクタを提供していました。 来年、すでに新しい仕様をサポートするポートを備えたガジェットが展示会で発表されることが期待できます。
