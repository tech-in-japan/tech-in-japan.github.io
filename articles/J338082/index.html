ブラックボックスの内容：仮想および物理インフラストラクチャのコストはいくらですか

ガートナーのアナリストは、4月にデータセンターへの世界の支出は昨年に比べて増加し、1700億ドルを超えると述べた。 この記事では、そのようなお金がデータセンターの文脈でどこに行くのかを見て、仮想インフラストラクチャと物理インフラストラクチャのコストの構成要素について説明します。

エンタープライズサーバーのコスト



クラウドソリューションとハードウェアのどちらを使用するかは問題ではありません。計算では、直接、間接、および隠れたコストを考慮する必要があります。 これらの費用グループはすべて、総所有コスト（TCO）メトリックに適合します。 物理インフラストラクチャのコストの大部分は機器であると考えられています。 ただし、Intelによると 、ハードウェアのコストはIT資産の総コストの15％に相当します。



ハードウェアに加えて、直接コストには、ソフトウェア、消費電力、および補助システム（加熱、冷却、換気）のコストが含まれます。 間接費は、システム管理者または他の従業員へのRFPの支払いに関連しています。



機器については、サーバー自体を購入し、データセンターの将来のアーキテクチャを検討する必要があります。これは、複数のブロックを備えたスケーラブルなモジュラーシステムであるか、従来のラックであるかです。 さらに拡張するための戦略を検討する必要があります。



サーバーのコストは、パフォーマンスによって異なります。 特定のデバイスを優先する決定は独立して行われるため、この時点でビジネスは節約の範囲を開きます。 ただし、安価で低パフォーマンスのサーバーが常に有益であるとは限りません。 エネルギー効率に注意を払うことが重要です。これは、長期的に節約するだけです。



サーバールームには、メインおよび中間スイッチギア、照明、UPS、暖房、換気、空調、防火、監視、セキュリティなどの電気システムが装備されています。



次の目的地はソフトウェアです。 サーバーには、Windows Server 2016などのオペレーティングシステムが必要です。これは購入する必要があります。 企業は、Microsoft SQL Server、Citrix Presentation Serverなど、企業の問題を​​解決するために他のツールを必要とする場合があります。ネットワークを編成する必要もあります。 専門サイトでローカルネットワークを作成するためのソフトウェアの推奨事項を知ることができます 。



最後に、コストが非常に高い電気に到達しました。 Rackspace の Marco Ponchielliの物語は非常に明快です。 彼は、総コストの中で電気のコストが以前ほど重要ではなかったため、計算のコストに近づくには30〜50年かかったデータを引用しています。



現在、高関税地域では、この期間は2年、平均で4年に短縮されています。 Marcoは、ブレードサーバーに2200万ドルを費やしたIT部門についても話しました。 3年間で、メンテナンスの費用は3,000万ドルに達し、設置のためのインフラストラクチャの近代化は5,400万ドルになりました。



支出の別の項目があります-前提。 ここで、価格の範囲は、オフィスが会社に所有されているかどうか、機械室が建設されているかどうか、および占有するスペースの大きさに完全に依存します。



さらに、フレームを考慮する必要があります。 サーバーがリモートに配置されている場合でも、企業には管理者が必要です。Trudポータルによると 、ロシアの平均給与は約4万ルーブルです。 スタッフのサイズはインフラストラクチャのサイズに依存するため、ここでの計算は特定のケースごとに異なります。



2〜4年以内に、オンプレミスとクラウドソリューションの両方のコストはサービスとサポートに限定されます。 ただし、ネットワーク管理者によると 、約5年後、企業はエンタープライズサーバーソリューションをアップグレードする必要に直面しています。



物理サーバーのライフサイクルでは、これに続いて新しい設備投資が行われます。これは、長期的には隠れたコストの主な項目です。 さらに、新しい物理機器の効率は古いものよりもはるかに高くなっています。 したがって、時代遅れの技術とエネルギーコストの上昇の間には、しばしば等号を付けることができます。



印象的な期間のこれらの投資は、インフレのために時間とともに低下することも忘れないでください。 機器の価格は、需要が時間とともに増加するにつれて上昇しています。



また、無形のコストは、ソフトウェアとデータベースを新しいプラットフォームに移行するコストと、移行のリスクを軽減する意思決定のコストという包括的な状況をまとめるための暫定的な見積もりに含める必要があります。



したがって、ローカルインフラストラクチャの保守コストは年々増加し、短いハードウェアライフサイクルに近づいています。

仮想サーバーのコスト



消費者の観点から見ると、クラウドエコノミーは非常にシンプルです。必要な構成を選択し 、家賃を支払うだけです。 いくつかの側面を考慮するだけで十分です。 たとえば、従業員がクラウドインフラストラクチャを効果的に管理できるように、スタッフをトレーニングする必要があります。 さらに、クラウドへの移行のコストは、隠れたコストの記事に記録する必要があります。 場合によっては、さまざまなプロバイダーと連携する試みの失敗が含まれます。 これらはすべて最終的にお金がかかります。



クラウドの場合のクライアントのコスト計算は非常に透過的ですが、プロバイダーにとっては何を加算しますか？ 概して、状況は企業インフラストラクチャに似ていますが、もちろん、規模ははるかに大きくなります。



高品質のTier IIIデータセンターは 、ディスクアレイ上のマルチプロセッササーバーを高速チャネルを介してネットワークに接続します。 このために、いくつかの交換可能なトランクルータが使用されます。 クラウドインフラストラクチャは、詳細なパケット分析システムによって保護されています。 ネットワークトラフィックを分散させるのは、相互に重複した接続を持つシステムを形成する責任あるスイッチです。 ストレージには、物理​​ディスクとソフトウェアの更新、および負荷分散を処理できる信頼性の高いハードウェアとソフトウェアのソリューションを提供する必要があります。



システムの複製により、高可用性が実現されます。 作業の安定性と最適化のために、ソフトウェアスイートには、エネルギー効率制御、動的負荷分散、コントロールパネル、保護ツールなどの特別なソリューションが追加されています。



すでに述べたように、人的資源はデータセンターの仕事におけるもう1つのコスト要素です。 Clabby Analyticsの調査会社Joe Clabbyの責任者によると、スタッフのスキルと地理的な場所に応じて、データセンターに勤務するスタッフ全体の年間給与は75千ドルから12万5,000ドル以上に及びます。 管理者に加えて、他の何十人もの従業員がデータセンターにサービスを提供していることを忘れないでください。 たとえば、Tier IIIレベルのデータセンターには物理サーバーのセキュリティが必要です。



しかし、この規模にもかかわらず、場合によっては、クラウドプロバイダーは顧客のお金を節約することができます。 これは、データセンターの運用規模により、事業者が機器やソリューションの割引を受けることができるためです。



この場合、機器の設備投資は、1ラックまたはギガバイトのメモリに関して分配されます。 費用は機器の耐用年数にわたって償却されます。 また、冷却システム、UPS、保護システムなど、インフラストラクチャ全体を提供するハードウェアについても忘れてはなりません。 コストはサーバー間で「分配」されます。



仮想プロバイダーのエコシステムにより、使用するサーバーの数が減り、エネルギー消費が削減されます。 さらに、スケールシステムとバックアップシステムがあれば、システムに損害を与えることなくサーバーを再配布および切断することが可能になります。 Uptime Instituteによると、単一の1Uサーバーラックの廃止により、年間2,000ドル以上を節約できます。



Uptime Instituteの調査によると、2015年には、すべてのサーバーの約30％が使用されていませんでした。つまり、プロバイダーの肩に「重荷」を置いています。 世界的には、約1000万のcom睡状態のサーバーがありました。 金銭的に言えば、これは300億ドル以上であるため、クラウドプロバイダーは「バラストをなくす」ことで、自分自身とクライアントのためにまともなお金を節約しています。



したがって、企業はクラウドサービスにますます注意を払っています。 最近のレポートによると、プライベートデータセンターへの投資はIT企業にとって意味を失っています。 代わりに、経営者はビジネスアプリケーションの開発への投資を増やしようとします。 調査対象の組織の70％の代表者は、2017年にセキュリティコストを増加させる計画であり、67％がクラウドアプリケーションを優先し、さらに52％がクラウドインフラストラクチャを優先するとしました。
