初回リリース候補PHP 5.3.0（PHP 5.3.0RC1）

PHP 5.3.0の新しいブランチ（PHP 5.3.0RC1）の最初のリリース候補がダウンロード可能です。 このリリースは、多くの新機能、バグ修正、セキュリティの改善など、PHPの5番目のブランチを改善するための多くの作業の最終段階です。



新機能：

  • 名前空間のサポート
  • 生産性の向上
  • 遅延静的バインディング
  • ラムダ関数とクロージャー
  • 構文拡張：NOWDOC、限定されたGOTOサポート、 短縮された三項演算子形式？： 、__callStatic（）メソッド
  • 循環参照のガベージコレクター
  • mysqlnd-ネイティブlibmysqlの置換
  • Windowsサポートの改善
  • 浮動小数点数のより正確な丸め
  • 新しいエラーレベルE_DEPRECATED
  • php.ini構成ファイル形式の改善
  • ディストリビューションには新しい拡張子が含まれています：ext / phar、ext / intl、ext / fileinfo、ext / sqlite3、ext / enchant
  • 既存の拡張機能、特にext / openssl、ext / splおよびext / dateに対する無数のバグ修正と改善




また、このリリースでは、いくつかの拡張機能が除外され、内部APIが統合されました。 下位互換性の問題がある可能性があります。

  • パラメータ処理APIの統合により、一部の関数は型を操作する際に多少厳密に動作します
  • リモート拡張機能：ext / mhash、ext / msql、ext / pspell、ext / sybase
  • PECLリポジトリに移動された拡張機能：ext / ming、ext / fbsql、ext / ncurses、ext / fdf
  • 互換モードzend.ze1が削除されました


すべてのPHPユーザー、特にPHP 5の以前のバージョンをまだ使用しているユーザーは、このリリースをテストすることをお勧めします。

公式発表： www.php.net/archive/2009.php#id2009-03-24-1
