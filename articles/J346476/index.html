2017年のOpenStreetMapカンファレンス

翻訳者から：OpenStreetMapプロジェクトは動的に開発されており、より多くの参加者が参加しています。 外国では、何が行われ、何がまだ行われていないかを議論する会議が開催されます。 ロシアには多くのOSM参加者がいますが、地域コミュニティはまだ作成されていません。



明けましておめでとうございます！ 過去2017年はOpenStreetMapにとって忙しい年でした。 世界中のコミュニティでは、すばらしい地図会議の状態だけでなく、多くの会議、カートマラソン、地図作成パーティー、セミナーなど、地図製作者を集めてOpenStreetMapの本質を感じさせる多くのイベントが開催されています。



State of the Map （SotM）会議は、毎年開催される国際的なOpenStreetMap会議です。 OpenStreetMap Foundationが主催し、 2007年から毎年開催されています（2015年を除く）。 「地図の状態」と呼ばれる他の多くの地域および地域の会議もあります。 これらの地域のSotMは、独自のスポンサーシッププログラムを実行し、OpenStreetMap Foundationとは別個の予算を持つローカルチームによってのみ編成されることに注意してください。 これらの会議の主な目標は、新旧のユーザーまたは潜在的なユーザー間で情報と経験を交換するためのプラットフォームになることです。



このレビューには、SotM以外の会議も含まれていますが、OpenStreetMapはそれらにも多くの注意を払っています。



この会議を成功させ、OpenStreetMapの作成に貢献したすべての主催者と参加者の皆さん、こんにちは！

SotM国際会議2017年8月18〜20日、日本



今年、 State of the Map国際会議が会津若松で開催されました。会津若松は、OpenStreetMap FoundationのSotMワーキンググループと地域コミュニティによって開催されました。 日本での3日間は、自由で開かれた世界地図での講演、ディスカッション、ワークショップで皆が楽しみました。 私たちのスピーカーの経験についてはここで読むことができ、2018年7月28-30日にイタリアのミラノで開催される国際的なSotM 2018で会えることを期待しています！

State of the Map Asiaでは、毎年恒例のOpenStreetMap地域会議がネパールのカトマンズで開催されました。 会議には、アジアの16か国、北米の2か国、ヨーロッパの2か国から175人のオープンマッピング愛好家が参加しました。 このイベントは、経験を共有し、お互いから学び、アジアにおけるOpenStreetMapの将来の発展について集合的に考える機会を提供しました。

「マップの状態」という名前を使用します。



地域の会議を開催し、「State of the Map」または「OpenStreetMap」という名前を使用したいですか？ OpenStreetMap Foundationの担当者にできるだけ早くお知らせください。 これは、近くを通過する国際または他の地域のSotMとの一時的な競合がないことを保証するためです。 これは、商標の保護にも関連しています。



2018年に地域のSotM会議を計画していますか？ 心配する必要はありませんが、すぐにお知らせください。 このリクエストは、カードパーティーの開催者には適用されないことに注意してください。 近い将来、更新された知的財産ポリシーについてお知らせしますので、ご期待ください！
