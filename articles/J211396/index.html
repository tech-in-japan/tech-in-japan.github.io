Firefox 27リリース

デスクトップ向けFirefoxがダウンロード可能になりました。 このバージョンの新機能：





  • Social APIを使用した複数のサービスの同時起動のサポートを実装しました。これにより、 いくつかの統合されたソーシャルネットワークで通知を受信し、通信し、他のアクションを同時に実行できます 。
  • TLS 1.1およびTLS 1.2のサポートが追加されました。
  • SPDY 3.1プロトコルのサポートが追加されました。
  • 「インスペクタ」は、特別なアドオン（ ビデオ ）をインストールする必要なく、HTMLブロックの無料編集のサポートを追加しました。

    背景画像と色のプレビューが提供されます（対応する色の円がデジタル値の隣に表示されます）。 デフォルトのカラー出力形式（rgb、16進数、名前別）を決定する設定を追加しました。
  • スタイルエディター、デバッガー、インスペクター、およびスクラッチパッドは、 Codemirror HTMLエディターを使用します。これにより、HTMLブロックを完全に編集できます。
  • all：unset要素を使用して、スタイルシートパラメーターをリセットする機能。
  • 「デバッガ」でJavaScriptの難読化を解除する機能が追加されました 。
  • 選択した領域に収まらないフィールドセットのスクロールのサポート （fieldsetタグのオーバーフロープロパティ）。
  • iframeタグの場合、ポップアップを表示する機能を制御するallow-popupsサンドボックスモードディレクティブが提供されます。
  • カーソル「-moz-grab」および「-moz-grabbing」のプレフィックス CSSプロパティから削除されました 。
  • SpiderMonkeyエンジンは、ECMAScript 6ジェネレーターのサポートを追加します。
  • ECMAScript 6で数学関数Math.hypot（）のサポートを実装しました 。
  • Canvas は破線のサポートを追加しました 。
  • Linuxでは、 Azure / Skiaを使用してコンテンツをレンダリングする機能 が提供されます。


追記



新しいAustralisデザインの実装に興味/興奮している人のために： AuroraチャンネルでFirefox 29に新しいデザインを統合する決定がなされました。 第29版のNightlyからAuroraへの移行は今週行われる予定です。

Firefox 29のリリースは4月29日に予定されています。
