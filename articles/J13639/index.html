通常の外付けハードドライブの価格でIomega NAS

Iomegaは、ネットワークドライブの新しいラインを導入しました。その価格は、従来の外付けハードドライブのコストとそれほど変わりません。 Iomega Home Network Hard Driveと総称されるデバイスのディスク容量は、320、360、および500 GBです。 現在、容量が320 GBと500 GBのモデルは、それぞれ149ドルと199ドルの価格で購入できます。





すべてのIomega Home Networkハードドライブモデルは、8 MBキャッシュとスピンドルスピード7200 rpmのSATA-IIハードドライブに基づいて設計されています。 また、通常の外部ドライブとしてコンピューターにオプションで接続するためのUSB 2.0ポートも提供します。 ネットワーク機能は、内蔵10/100イーサネットアダプターの存在によって提供されます。 「システム」を設定するには、ドライブをルーターに接続し、Iomega Home Network Hard Drive（Discovery Tool Homeユーティリティに付属）へのアクセスを提供する必要がある各PCにインストールするだけです。 Mac OSおよびLinuxベースのPCの場合、この場合、ハードウェアの構成と設定は手動で行う必要があります。
