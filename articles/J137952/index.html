品質保証の日へようこそ

品質は、ソフトウェアにとって最も重要な成功要因の1つです。 品質に影響を与える多くの方法があります。 これらは、開発組織のプロセス、およびテストの分野でコード、アーキテクチャソリューション、およびアプローチの品質を保証する方法です。 業界はこの分野で多くの経験を蓄積しており、これらの開発の使用は多くの問題の解決に役立ちます。 これが、Microsoftが3月30日にCareerLabと共同で、ソフトウェア品質に関心のある人向けの会議であるQuality Assurance Dayカンファレンスを3回開催する理由です。



今回、開催チームは会議の内容を大幅に拡大しました。 私たちが追求した主な目標は、品質問題に特化した産業イベントを作ることでした。 品質保証およびテスト組織の分野で最も有名なグルの1人が招待されています- レックスブラック 。

レックスは、業界で最も成功している著者、講演者、実務家の1人です。 ちなみに、彼の最新の本「Managing the Testing Process」は25,000部以上を販売しています。 テストでの25年の経歴には、ASTQBおよびISTQBの名誉会長などの役職が含まれます。 レックスは現在、RBCS、Inc.の社長です。 ソフトウェア、ハードウェア、システムテストの世界的リーダー。

この会議には、Microsoft、Intel、パフォーマンスラボ、Parallels、Quest Software、Selaの専門家が集まります。 テスター、開発者、およびそのマネージャー向けに、3つのレポートストリームが計画されています。 会議参加者は、ロシアで初めてVisual Studio Test Professional 2011テスト環境の新機能、Windows 8のアプリケーションテスト方法、およびVisual Studio 11の新しいユニットテストテクノロジーについて学ぶ機会を得ました。プロセスとテスト管理について非常に興味深いトラックが計画されています。 品質保証の分野における主なエラーについて、テストの有効性と人件費を測定および評価する方法について学びます。



会議の詳細については、www.microsoft.com / ru-ru / events / msqadays / default.aspxをご覧ください。
