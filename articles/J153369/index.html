Samsung GALAXY Note IIのプレゼンテーションからのレポート





昨日、トヴェルスカヤ22にあるサムスンのショッピングギャラリーで重要なイベントが開催されました。それは、ビジネスパーソンと再現したい人向けの非標準スマートフォンの第2バージョン-GALAXY Note IIのプレゼンテーションです。 イベントのゲストは、新しいデバイスのサンプル（だけでなく）をテストし、国際的なIT大手の代表者の激しいスピーチに耳を傾け、ハイテク、モダンなデザイン、設置の雰囲気を吸収する機会を得ました。



同僚と私は事前にその場所に到着し、提示されたデバイスとアクセサリーの小さな写真を撮ることができました。















































13.00にプレゼンテーションが開始されました。 最初に講演したゲストは、Samsung Electronics CIS HeadquartersのJang Sang Hoでした。







「昨年、Samsung GALAXY Noteを導入しました。これにより、スマートモバイルデバイスの新しいカテゴリが作成されました。 1年足らずで、デバイスの販売台数は1,000万台を超えました。 新しいGALAXY Note IIには、改良されたSペン、アクティブな創造性の豊富な機会、AllShare PlayおよびAllShare Castのシンプルで便利なデータ転送機能など、多くの革新的な機能があります。



彼の後に、GALAXY Note IIの革新についての詳細な説明が、Samsung Electronics RUSのモバイルデバイス部門の責任者であるAlexey Dorofeevによって行われました。







第一世代のデバイスと比較して変更が加えられた最も重要なもの：



スマートフォンの画面はさらに大きくなりました（以前のバージョンの5.3に対して、5.55インチの優れたHD Super AMOLED）。 これはサイズに影響を与えませんでした-画面は前面のほぼ全体を占めていましたが、現在は側面の薄いフレームのみに制限されています。 ディスプレイがより長くなったため、アスペクト比が変更されました：16：9、1280x720ピクセル（最初のモデルでは、比率は16:10、1280x800より非標準でした）。 ちなみに、PenTile AMOLEDは本格的なRGB AMOLEDに置き換えられました。



同時にいくつかのアプリケーションでの作業を簡素化するように設計された多くの機能が登場し、それによりスマートフォンの改善されたマルチタスクを提供します。



エアビュー機能 。 写真や画像をそれぞれクリックすることなくクイックモード（プレビューの一種）で表示できます。画像上の投影点をポイントするだけです。





ポップアップノート 。 電話での会話中に画面上のペンをダブルクリックすると、Sノートアプリケーションのポップアップノートが表示され、そこにメモを残したり、重要な情報を記録したりできます。





クイックコマンド 。 手書きコマンドを作成して特定の機能を割り当てる機能（たとえば、コマンド "@ brother"を作成し、自動的に呼び出し音を鳴らしたり、友人にメッセージを送信したりできます）。





創造性のためのツールも大幅に拡張されました。



簡単クリップ 。 任意のソースから画像の断片を切り取り、メモ、メッセージ、電子メールに貼り付けたり、Fragmentsフォルダーに保存したりできる非常に面白い機能。





手書き認識 。 この機能が医師の筆跡を正しく認識できるほどクールではない（チャック以外は可能）が、乱雑で「傾いた」筆跡でも非常に耐えられる。 手書きのメッセージもメールで送信できます。SPlannerオーガナイザーを使用すると、手書きのメモを取り、メモリに保存できます。 一般的に、悪名高い日記の優れたデジタル代替品。





写真のメモ 。 確かに、誰もが写真付きの古いアルバムを持っており、その裏面には「モスクワ、ニューイヤー、1985年」や「リトル・ジョニー：最初のステップ」のようなものが書かれています。 レニングラード、1953」。 GALAXY Note IIに保存されている写真の裏面に、デジタル形式のみの同様の手書きのメモを残すことができます。





サムスンの新しい非フォーマットフラッグシップの最後の仕上げは、生産性の向上です。統合1.6 GHzクアッドコアプロセッサ、Android 4.1（Jelly Bean）OS、16 GBユーザーメモリ+ 2 GB RAM、3100 mAhバッテリー。





プレゼンテーションの終わりに、国内市場でのGALAXY Note IIのリリース日が判明しました。これは、ロシアのすべての地域で現地時間の10月18日19:00に行われます。 http://samsung.com/enで今すぐ予約できます。
