ウェビナーシリーズ「Windows 8向けアプリケーション開発」

Windows 8の公式リリースは間近に迫っています。また、「考える」だけでなく「感じる」こともできる新世代のウルトラブックの登場です。 これまで以上に、新しいモバイルプラットフォーム向けのアプリケーション最適化は、指入力処理の実装、タッチセンサーの使用、エネルギー効率など、より重要になります。 来週から、これらの問題に専念するインテルのエンジニアとのオンラインセッションに参加する機会があります。







詳細については、 Intelのイベントページまたはcatをご覧ください。



10月9日火曜日、シリーズ「 Windows 8とタッチインターフェイス：開発者向けの新しい現実 」の最初のウェビナーが開催されます。 ソフトウェアエンジニアのDmitry Ryzhkov（ Terror ）とユーザーインターフェイスデザイナーのAnatoly Lyubomirovが、タッチインターフェイス、手話の一般的な要件について説明し、Windows 8 UIおよびデスクトップで指入力処理を実装する方法の例を示します。 タッチインターフェイスのサポートがなければ、アプリケーションに未来がないことを確認できます。



10月12日、2回目のウェビナー「 ウルトラブックのセンサー。 理にかなっています！ »ウルトラブックセンサーを含むアプリケーションの開発への遠足を取得します：GPS、加速度計、コンパス、ジャイロスコープ、光センサー。 最も重要なのは、これらのセンサーを使用する方法を学ぶために、これらのセンサーを使用するための多くのオプションがあります。



Windows 8向けの最新のGreen Softwareウェビナーは 、10月16 日に開催されます。 モバイルプラットフォームで最大限のエネルギー節約を保証するコード最適化手法に専念します。 アプリケーションのエネルギー効率を評価するためのバッテリー寿命アナライザーおよびWindowsパフォーマンスアナライザーツールを確認します。



モスクワ時間の午前11時に各ウェビナーの開始。
