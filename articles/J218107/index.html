現在、Intel Galileo上のWindows 8、Microsoftはすでに無料で計画しています！



昨日、4月2日、Microsoft Build Developer Conferenceで、私たちの意見では重要なイベントが開催されました。 Microsoft Executive Vice President Terry Myersonは、Intel Quark X1000プロセッサを搭載したIntel GalileoコンピュータでWindows 8.1オペレーティングシステムをデモしました 。 プレゼンテーションでは、ガリレオが制御する屋外ピアノが披露され、Windows- Windows on Devicesの 「モノのインターネット」のソフトウェア開発者向けの新しいサポートプログラムの開始も発表されました。

エンドユーザー向けのもう1つの良いニュース： Windows for smart thingsは無料です。 これはマイクロソフトの意図です。 もちろん、これはWindowsの価格表の即時変更を意味するものではありません。 また、「Windows for the Internet of Things」のライセンスが、デバイス自体とその使用に関する特定の制限を規定することは疑いありません。







したがって、「Windows Territory」には、PCやタブレットだけでなく、Intel Galileoなどの小型デバイスも含まれるようになりました。 確かに、今や知的物事はさらに多様になりますが、MicrosoftとIntelの巨大な共同可能性は確かにこの有望な市場の発展を加速させるでしょう。
