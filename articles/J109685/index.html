Little ProLiant（Just Right Home Server）



最小のプロライアント。



Hewlett-Packardは9月8日に、大規模なProLiantファミリーの最小サーバーである新製品のリリースを発表しました。 モデルには従来のDL / ML / ...プレフィックスはありませんが、単にHP ProLiant MicroServerと呼ばれます。

このソリューションは小規模な構造（最大10ユーザー）の最初のサーバーとして位置付けられており、価格は500ドルからです。



最初の配送は、ディストリビューターの倉庫に届いたばかりで、サーバーの1つはすでに私の家に落ち着いていました。



乾燥した数字：

  • AMD Athlon II Neo N36L-1.3 GHzで2つのコアを持つAtomクラスメート
  • 最大8GB（2スロット）まで拡張可能な1GB DDR3 RAM
  • 3.5インチSATAドライブ用の4つのスロット
  • 1Gbイーサネット
  • PCI-express x16スロット、PCI-express x1スロット
  • 5.25「インターフェース付きデバイス用スロット
  • 統合されたSATA RAID 0/1
  • 7つのUSBポート 前面パネルに1つ、4つ
  • 外部eSATAポート
  • 150W PSU
  • 最大24 dBのノイズレベルで動作する2つの冷却ファン


最も詳細な詳細は、 QuickSpecにあります。





電源を入れる前に、サーバーを慎重にレビュー、分解、研究しました。 印象は非常に好意的です。素材とビルドの品質が一番です。 おそらく、いくつかの場所での100番目のシリーズは、見た目も感じも悪くなります。 サーバーは、ほこりの多いキャビネットではなく、フロントコーナーに設置するように設計されていることがわかります（それでも、私の家では、サーバーはメザニンの遠いコーナー用に準備されていました）。 つや消しプラスチックで作られた滑らかな形状、きちんと照らされたロゴ。 HPのロゴには、美観に加えて診断機能もあります。ロゴはさまざまな色で点滅し、エラーコードを報告できます。





ロック付きの穴あきドアの後ろには、4つのハードドライブベイがあります。 1つの160GBドライブが基本構成に含まれています。 ドライブレールが含まれており、SATAドライブをねじ込むことができます。 わずかなお金で（ブランド化されたコンポーネントにお金をかけることなく）、8 TBのRAWスペースを取得できます。これはSOHOにとって非常に多くのことです。 ユーザーに対するベンダーの懸念は特に感動的です。ドアには、16本の予備のネジと、六角ネジ用の従来のProliant L字型キーがあります。





マザーボードは小さな可動ベッドの上にあります-メモリ/拡張カードの取り付けに便利です。 残念ながら、ケーブルはHotSwapではなく、手動で外す必要があります。 ボードとシャーシには、適切なLowProfileビデオカード、および/またはイーサネット/ Wifiカードを取り付けるのに十分なスペースがあります。



プロセッサとチップセットの熱は非常に低く、積極的な冷却は必要ありません。 これにより、サーバーは驚くほど静かでした。 製造業者は「聴覚を超えた」ことを欺きませんでした-通常の操作では、うるさいネジだけが聞こえます-それらからの振動はケースに伝達されます。 後壁の巨大なファンは低速で動作し、ほとんど聞こえません。 電源の小さなクーラーのみを混乱させます。





すべてを回収した後、OSの選択に直面しました。製造業者はWindows Server 2008とRHELを公式にサポートしています。 これはもちろんTrue Enterpriseですが、私たちの選択ではありません。 プロセッサはAMD Virtualization Technologyをサポートしています-ある種のハイパーバイザーを起動してみることができます。 インターネットでESXiの実行に関するレビューを見つけましたが、オンボードRAIDサポートはありません。



この同じRAIDは、新鮮なDebianのもとでは動作しませんでした。 混乱するのはひどく面倒だったので、すべてが完璧に機能するUbuntu Serverをインストールしました。



これでサーバーが回転します。



-伝送はトレントをダウンロードして配布します

-FlexGetはトラッカーのRSSフィードを解析し、Transmissionのニュースを提供します

-Time Machineはサーバーにバックアップを追加します

-UPnPメディアサーバーは音楽とビデオを配信します

-AFPファイル共有-Mac用のネイティブプロトコル。 データ交換速度はSambaの2倍です。AFPは90 MB / sの読み取り/書き込みを示しますが、Sambaの場合は45です。



同時に、プロセス/メモリによる負荷は最小限であり、追加のサービスのための十分な空きリソースがあります。



近い将来、2 TBのネジを2本、HDMI出力付きのビデオカードを取り付け、20メートルのHDMIケーブルでテレビに接続してXBMCを起動します。



オフィスアプリケーション



これらの鉄片のもう1つが、私たちの地元のオフィスに取って代わりました。 10人のファイル共有として機能します。 社内のほぼすべてのサービスは集中化されており、以前はメールを使用するか、SharePoint Serverを使用して10メガピクセルのプレゼンテーションを相互に交換する必要がありました。



サーバー内のビンにあるDAT72テープドライブをインストールする予定です-疎外されたメディアへのバックアップ！



鉄の信頼性について心配する必要はありません-旧プロライアント兄弟のように、Microserverの拡張サービスパッケージの1つを購入できます。 最も重要なのは、3Y 6hのCall-to-Repairは、HPサービスへの呼び出しから最大3時間、今後3年間でサーバーが既に機能することを意味します。



小型サーバー-素晴らしいセーリング



Microserverがホームシステムと小規模オフィスの両方で大きな成功を収めることを願っています。 さまざまなSOHO NASに匹敵する価格で、スケーラビリティと柔軟性は比類のないほど優れています。
