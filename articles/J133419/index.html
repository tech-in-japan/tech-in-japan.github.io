Qualcomm Mirasol電子書籍の販売は世界で始まります

電子インクに近い技術に基づいて、電子インクと従来のディスプレイの両方の利点を組み合わせたQualcomm Mirasolディスプレイが Habréで書かれてから、ほぼ2年が経過しました。







そして最後に、プロジェクトは別の実験室実験から市場製品に変わりました。 このようなスクリーンを持つ最初の読者は韓国で販売されました。 この市場で最大の書店と共同でリリースされたKyobo Book Centerは、 Kyobo eReaderと呼ばれています。 小売価格は310ドルで、京坊クラブのカード所有者は265ドルです。



デバイスの残りの特性は、読者にとって非常に優れていますが、デバイスをタブレット（本質的には）として使用する場合は満足しません。



•1.0 GHzでクロックされるQualcomm Snapdragon S2プロセッサ

•RAMの量は512 MB、フラッシュメモリ-2 GBです。

•最大32 GBのmicroSDまたはmicroSDHCカードスロットがあります。

•デバイスには、USB 2.0ポートとWi-Fi 802.11 b / g / nアダプターが装備されています。

•寸法は174 x 119 x 12.9 mm、重量-338 gです。



使用されているOSはAndroid 2.3です。
