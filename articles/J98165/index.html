GuruPlug Server Plusの概要

4月の終わりに、 新しいIT GuruPlug Server Plusから注文しました。 5月にのみ火山とのすべてのトラバースの結果として彼らは彼を毒し、彼は7月にのみ私の郵便局に到着しました。 確かに、1500桁を超える注文はさらに2か月延期されたため、多くの人々はさらに不幸でした。 しかし、それはすべて歌詞です。アウトレットのサーバー自体に移りましょう。



特徴



Sheevaプロセッサー：

  • 1.2 GHz
  • L1キャッシュ：16KB命令+ 16KBデータ
  • L2キャッシュ：256Kb


アルゴリズムのハードウェアサポート：

  • AES、DES、3DES、SHA1およびMD5


メモリ：

  • 512MB DDR2 400MHz
  • 512MB NANDフラッシュブート可能


周辺機器：

  • 2 x 1Gbイーサネット
  • 1 eSATA 2.0ポート
  • 2つのUSB 2.0ポート
  • 内部に1つのmicroSDスロット
  • 外部に1つのmicroSDスロット
  • Wi-Fi 802.11b / g
  • Bluetooth：2.1 / EDR
  • U-SNAP I / O
  • 3.3VI / Oのユーザー5、1.8VI / Oの2の7x GPIO
