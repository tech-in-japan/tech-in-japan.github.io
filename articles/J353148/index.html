テレフォニーとサービスMySkladの統合

サイト



MoySkladは、ロシアの貿易を管理するための最も人気のあるSaaSソリューションの1つです。 2016年、CNews Analyticsによると、MySkladはクラウドサービスプロバイダーの中で収益で7位になりました。 発売時の製品の革新は、貿易および倉庫業務の会計をオンラインで自動化できることでした。



卸売業や小売業、オンラインストア、製造などのビジネス分野に機会を提供するこのサービスは、CRMモジュールなしでは実現できませんでした。 そして、もちろん、このプロジェクトに統合できる電話サービスとして私たちを代表することが重要です。







Iptelefon.suは、通信事業者でありインテグレータです。 Zeon IP-PBXに基づく既製のIPテレフォニーソリューションを提供します。 クラウド内またはサーバー上。



ロシア市場でのビジネスコミュニケーションに関連するすべての注目すべきプロジェクトとサービスの開発を注意深く監視しています。 そしてもちろん、MySkladのようなシステムは、私たちの視野に落ちざるを得ませんでした。 統合モジュールを開発する過程で困難があったことは注目に値しますが、MySkladのサポートが最上位であることが判明しました。APIメソッドが改善され、サービスが改善され、プロジェクトを正常に完了できました。



サービスの簡単な説明



このサービスの利点の中でも、多くのユーザーが指摘しているように、中小企業向けのクイックスタートと過剰な機能の欠如が注目されています。 倉庫の会計と取引の自動化のサブシステム、文書の作成とアップロード、レジ係の職場と他のサービスが提示されます。



オンラインストアの作業を自動化するために、1C-Bitrix、Opencart、WooCommerce、InSalesなどの多くの有名なCMSとの統合が行われました。 また、このシステムでは、ソーシャルネットワークVKontakteで、自動写真アップロード機能を備えたVK製品を通じて、独自のオンラインストアを作成することなく、商品のデータをYandex.Marketにアップロードできます。



CMSとの統合により、オンラインストアからシステムに注文を転送し、製品の残高を戻すことができます。



アウトレットと連携するために、iOSとAndroid向けのモバイルアプリケーションも作成されており、キャッシャーの職場（タブレットまたはスマートフォンで動作するモバイルキャッシュデスク）を整理できます。 もちろん、このアプリケーションは適切な取引機器と連携して現金と領収書を印刷し、必要に応じて、すべての最新のクラウドベースのPOSアプリケーションと同様に、オフラインで作業し、その後データを同期できます。



CRM機能



このシステムを使用すると、卸売りや小売り販売など、さまざまな方向に販売ファネルを構成できます。







機能が含まれます：



  • 連絡先と請負業者のコンパクトなディレクトリ。
  • 顧客との関係の履歴を記録するイベントのフィード、通話、手紙、会議に関する情報、マネージャーと同僚の両方が独自に入力したデータをそこに表示できます。
  • タスクとリマインダーを使用して営業部門を監視し、今日の新しいタスク、変更されたタスク、およびスケジュールされたタスクの電子メール通知を設定します。
  • クライアントのカードから直接文書、請求書、契約書を作成する;
  • クライアントの個々のグループまたはデータベース全体への電子メールおよびSMSメーリング、UniSender、Sendsay、SubscribePROサービスへの接続。 さらに、レターの配信とオープンに関する詳細情報、リンクマップを含むレポートを受け取ることができます。
  • 取引先指標によると、販売目標到達プロセスレポート。
  • Tochka、Alfa-Bank、Modulbank、Tinkoffなどの銀行では、1C形式の明細書と支払い注文をインポートできます。


テレフォニー機能



統合により、呼び出し時にクライアントを判別し、呼び出しを受信し、MyStoreから直接かつ迅速に直接呼び出しを行うことができます。 MyStoreで電話をかけると、顧客カードがポップアップ表示されます-誰が電話しているかすぐにわかります。 新しいリードの場合、会話中にカードを作成できます。 通話履歴はサービスに記録され、ブラウザーで対応する通話のリンクをクリックすることで、いつでも記録を聞くことができます。



必要に応じて、PBXをCRMシステムと統合するときに機能の機能を拡張し、要件に合わせてカスタムリビジョンを作成できます。







設定方法



統合ウィジェットを開き、フィールドに入力します。



  • テレフォニープロバイダーのアドレス-http：// <サーバーアドレス> /api/integration/moysklad.php;
  • MySklad APIアドレス-自動的に生成され、PBXに転送する必要があります（以下を参照）。
  • アクセスキー-「アクセスキーの作成」ボタンをクリックすると生成されます。 また、電話交換機に転送する必要があります。
  • 内部番号は、システムのユーザーに従って入力されます。
  • ウィジェットをアクティブにするには、「接続済み」を確認する必要があります。
  • [保存]ボタンをクリックします。






次に、Zeon IP PBXに移動します。 メニュー>システム>統合。 「MySkladサービスとの統合」タブがあります。



  • 「オン」状態をオンにする
  • ウィジェットからアドレスを入力してください
  • MySkladの「アクセスキー」に対応する秘密キー
  • [保存]ボタンをクリックします






iptelefon.suのテレフォニーを使用する方が良いのはなぜですか？



  • 参加せずにすべての設定を行います。
  • 強力な電話システムが得られます
  • すべてのサービスは1つの「ウィンドウ」で取得できます。 ATS、通信サービス、IP-SIMカード（FMCサービス）、ロシアの電話番号、機器（IP電話、ヘッドセット）
  • 電話技術サポート
  • クイック接続と起動
