Amazonはユーザーにスーツケースを使ってデータをクラウドに転送することを提供しています





今日、Amazonは、インターネットを経由せずに、ユーザーデータをAmazon Web Services（AWS）クラウドサービスに配信する新しい方法を明らかにしました。 2009年以来、最大の小売業者は、低速インターネット接続を使用するユーザーに特別なAWS Import / Exportサービスを提供してきました。このサービスでは、あらゆるメディアで会社にデータを送信し、顧客が利用できるようになったクラウドにアップロードされました。 Amazonは現在、この目的のために、E-Inkスクリーンを備えた別のデバイスを提供しています 。外観はSnowballと呼ばれる巨大なスーツケースに似ています。



Snowballは、内蔵ストレージメディア（どれが明確ではないか）とネットワーク10ギガビットカードを使用して、重量約22 kgの外部の影響から保護されたデバイスです。 AWSサブスクライバーは、特別なインターフェイスを介してインポート/エクスポートサービスを注文する必要があり、その後Snowballがメールで送信されることが理解されています。 「スーツケース」を輸送するときは、6Gの加速度で衝突や落下に耐えることができるため、何も起こらないはずです。また、防水性もあります。 Snowballを受け取った顧客は、通常のネットワークデバイスとして自分のネットワークに接続し、256ビット暗号化をサポートする特別なアプリケーションを介して自分のデータをコピーします。 内部ストレージのボリュームは50テラバイトです。 完了したら、スーツケースをAmazonに返送する必要があります。



Snowballは次のようになります。





Snowballを使用したインポート/エクスポートサービスの注文には、AWSに200ドルのサブスクライバーがかかります。 彼はすべてがどのように機能するかを把握し、コピーを完了して「スーツケース」を送り返すために10日間を費やすことになります。 データをAmazonクラウドにコピーするコストは、ギガバイトごとに0.03ドルになります。



同様のイニシアチブで、Amazonのような統一はありませんが、Google はこの夏に来ました。 オフラインメディアインポート/エクスポートサービスは、ハードドライブ、フラッシュカード、磁気テープなどの別個のストレージメディアのみを、特別に選択されたプロバイダーに送信することをユーザーに提供します。
