Mac OS X 10.8 Mountain Lionが本日リリース

Appleの2012年第3四半期の収益報告書によると、Mac OS Xの新しいバージョンは本日AppStoreに表示されるはずです。 アップグレードには19.99ドルかかります（もちろん、今年の6月11日以降にポピーを購入した場合を除きます）。



新しいバージョンのさまざまな機能についてはすでに何度も議論されているので、イノベーションを簡単にリストします。



  • iCloudとの緊密な統合
  • メッセージ
  • リマインダー
  • 注釈
  • 通知センター
  • ソーシャルネットワークでの共有
  • Twitter
  • ゲームセンター
  • エアプレイ
  • ゲートキーパー
  • 安定性、セキュリティなどのあらゆる改善
