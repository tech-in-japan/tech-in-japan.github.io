セールスリーダー2013-KUB-Micro / 60コントローラー

さらに数日-さようなら、就業日！ こんにちは、休日と待望の休暇！ それまでの間、私たちは、他の皆と同じように、物事を仕上げて要約します...



2013年の結果によると、KUB-Micro / 60コントローラーは、通信市場の販売リーダーになりました。 正月の主人公になるのは彼です。











KUB-Micro / 60は 、通信コンテナやブロックボックス、アクティブなキャビネット、その他の「通信」オブジェクトなどの通信オブジェクトを監視するために「閉じ込められています」。



「ボード」には、上記のオブジェクトを監視するための最も一般的な機能がコントローラーに含まれています。 いわゆる外部拡張モジュール（BMP）により、追加機能を提供できます。 VMRは、最大14個のRS485を介してKUB-Micro / 60に接続されたスレーブです。 さらに、追加のコントロールポイント数または追加機能を提供します。 つまり、KUB-Micro / 60に基づく監視システムは次のように構築されます。 オブジェクトは、必要なセンサーとVMRを備えたKUB-Micro / 60コントローラーです。 コントローラーはデータをディスパッチセンターに転送し、必要に応じてセンターから制御コマンドを受け取ります。













しかし、まず最初に。 KUB-Micro / 60自体の機能から始めましょう。



KUB-Micro / 60コントローラー機能



KUB-Micro / 60には、次の機能ユニットと技術仕様があります。

  • エントランス「カウンター」 -電気メーターのパルス出力から測定値を取得します。
  • 「振動」入力 -ショック固定センサーの接続。
  • 入力「位相」 -相電圧の存在を制御します。
  • 入り口「火災」 -火災探知機の接続。
  • 4つの入力「乾接点」 -ドア開放センサー、煙/火災センサー、分離センサーなどの接続
  • ステーション電源12および60 Vの電圧を測定するための標準ユニット。
  • オプションの標準温度センサー。
  • 「リモートコントロール」を終了します。ハングしたときに通信機器を再起動するか、ディスパッチセンターからさまざまな機器を制御します。
  • 「自動制御」を終了 -機器の制御と自動モードでの再起動。
  • RS232 / RS485ポート「パススルーチャネル」 -インテリジェント機器の接続：電気、水、RS485インターフェースを備えた熱計測デバイス、電源ユニット、フォトレコーダーなど
  • RS485ポート -最大14個の外部拡張モジュールを接続します。
  • 施設で発生する最大64個のイベントを保存するためのシステムログ 。
  • 深さ35日の「ハーフチャック」による電気メーターの測定値を保存するための不揮発性メモリ 。
  • コントロールセンターからリモートで設定を変更する機能 ：温度しきい値、カウンターステータス、天文時間、IPアドレス、サブネットマスク、ゲートウェイなど
  • ディスパッチセンターからのデバイスのリモート「フラッシュ」の可能性 。
  • SNMPプロトコルのサポート。
  • 通信チャネル：イーサネットまたはADSL。
  • 電源： 12/60（48）V




KUB-Micro / 60には利便性と取り付けの容易さが備わっています。すべての接続はデバイスからケースを取り外すことなく行われ、コントローラーを取り付けるプロセスは目的の表面に固定することに限定されます。



デバイスのコストは4850 pです。 VATを使用。



KUB-Micro / 60に接続できる外部拡張モジュールの機能



したがって、すでに述べたように、外部拡張モジュール（BMP）をKUB-Micro / 60に接続して、監視ポイントの数を増やしたり、監視システムの機能を拡張したりできます。 VMRは、センターとの独自の通信チャネルを持たない安価な小型デバイスであり、メインコントローラー、この場合はKUB-Micro / 60を介してソフトウェアにデータを転送します。 1つまたはさまざまなタイプの最大14個のBMPを、RS485通信インターフェースを介してKUB-Micro / 60に接続できます。 RS485では、最大100 mの距離でKUB-Micro / 60からスレーブを削除できます。



ここでは、私たちが開発した最もリクエストの多い外部拡張モジュールのみをリストします。 それでは始めましょう：





IS485（Intelligent Reader）は、セキュリティ、CHIPキーまたは近接カードによるアクセス許可を提供し、サイトの電気錠を制御します。 IS485には、屋内設置用と屋外使用用の2つのバージョンがあります。



PST485（火災警報器および自動消火設備の制御用モジュール）は、メンテナンスフリーおよびオンサイトの施設の両方で使用できます。 意図されています：



  • 防火のため、
  • 人員警告装置（サイレン、ライトディスプレイ）を含めるため、
  • 火災の場合にプロセス機器を制御するため（オブジェクトの停電など）、
  • 一方向の消火設備の監視と制御用。


さらに、PST485は、消火剤への人々の暴露を防ぐために、制御オブジェクトの武装/武装解除および侵入に関するサードパーティのアラームシステムから信号を受信できます。



DVT485（湿度および温度センサー）は、温度と湿度を測定するように設計されています。 3つのバージョンで利用可能：1ポイントで温度と相対湿度を測定するDVT485、1ポイントで温度のみを測定するDTモードのDVT485、2ポイントで温度を測定する2 DTモードのDVT485。



PT485（温度コントローラー）は、施設の温度を制御するために使用されます。臨界温度に達すると、システムはエアコンまたはヒーターを自動的にオンにできます。



EPU485（電力パラメータ制御ユニット）は 、電気変換ユニットの動作を制御するように設計されています。 電気メーターのパルス（テレメトリ）出力からの遠隔読み取り、3相電源入力での電圧と電流の測定、およびステーションの供給電圧、充電電流とバッテリーの放電電流の測定を提供します。



AKB485（バッテリー制御ユニット）は 、12V、6V、2Vの電圧のモノブロックで、バッテリーの主要パラメーターである電圧と温度を測定するように設計されています。 通信施設にある鉛蓄電池または一連の直列接続された電池の定期的な監視を提供できます。 AKB485と一緒にミニチュアMCAモジュールが使用され、バッテリーに直接配置されます。


命名法には他の外部拡張モジュールもあります 。 ここで開発したすべてのBMPを見ることができます 。



さまざまなBMPのコストは、1000から5000ルーブルの範囲です。 したがって、プロジェクトで1つではなく複数の外部拡張モジュールを使用する必要がある場合は、プロジェクトを安くするためにKUB-BSコントローラーを購入する可能性を検討する価値があります。 KUB-BSは、当社製品の中で最も多機能なコントローラーであり、KUB-Micro / 60と複数のBMPで構成される1つのシステムを置き換えることができます。 KUB-BSの詳細はこちら
