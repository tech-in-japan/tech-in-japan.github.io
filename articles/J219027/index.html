4月13日、Yii 2はベータステータスになります

1週間以内に、多くのYii2フレームワークに愛されている開発のロードマップが更新されました 。



ここ数日で、そして今、膨大な数の変更がメインブランチに注がれています。



最も興味深いもののいくつかを次に示します（もちろん現在のアルファと比較して）



  • `Application :: bootstrap`を支持して` Application :: preload`を削除

    配列のインデックスといくつかのクラスの名前が変更されただけでなく、モジュールの読み込みシステムが適切に再設計されました。

  • 「HtmlResponseFormatter」と「JsonResponseFormatter」を追加しました

    これはすべて、RESTful APIボックスからの約束されたサポートの準備です。
  • ドキュメントの膨大な数の変更




現在、Yii2がリリース前に大きな変更を受ける可能性は低いため、誰かがプロジェクトでそれを使用したいが、当然アルファを恐れていた場合、2日間待つだけでした。
