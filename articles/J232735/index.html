Novatek NT96650プロセッサ上のSJ4000アクションカメラ

私が思うに、興味深いことに、SJ4000と呼ばれるアクションカメラが市場に登場しました。このカメラは、GoProから有名なカメラを外部からコピーします。 価格差は5倍なので、このモデルが材料と鉄の品質においてGoProと比較するのは正しくないことは明らかです。 したがって、品質は異なります。 それでも、SJ4000が有名なブランドに注目している優れたメーカーを見るのは興味深いことです。 さらに、このカメラのアクセサリキットは非常に豊富で、GoProと部分的に互換性があります。 このモデルの最大の可能性を可能な限り客観的に示すために、彼はその日のさまざまな時間にさまざまな撮影を行いました。 しかし、まず最初に。







>仕様



マトリックス： CMOS、12MP;

レンズ： 170°A + HD広角;

プロセッサー： Novatek NT96650;

LCDディスプレイ： 1.5インチ（4：3）;

ビデオ解像度： 1920 * 1080/1080 * 720/848 * 480/640 * 480

レコード形式： MOV;

ビデオコーデック： H.264;

サウンド：内蔵マイク

写真形式： JPG、12 MP、7 MP、5 MP。

メモリ：最大32 GBのMicroSDカード。

インターフェイス： Micro-HDMI、Micro-USB 2.0、microSDカードスロット。

バッテリー容量： 900 mAh（Li-ion）、70分間の連続撮影（1080p）;

寸法： 29.8 x 59.2 x 41 mm;

重量： 44 g、バッテリー付き-59 g;

追加オプション：シングルショット、遅延撮影（2秒/ 5秒/ 10秒）、顔と笑顔の認識モード、ホワイトバランス調整、カメラの電源を入れたときにビデオの録画を開始、USBからの充電、バーストおよびインターバル撮影、ビデオループ録画;



>オプション



カメラとアクセサリー用のプラスチック製クレードル内の段ボール箱。 透明フィルムを通して、カメラのシルエットが保護ボックスに表示されます。







キットの付属品セットは驚くべきものです：マウント、プラットフォーム、ファスナー、洗濯ばさみ、さまざまな形と目的のアダプター：









充電器ユニットは1Aの電流を生成し、動作中に過熱せず、1時間でバッテリーを充電します。







グラスファイバー保護ボックスには、トップパネルにラッチがあり、盲目的に押すのに便利な大きな複製ボタンがあります。 製造業者は、このボックスは最大30メートルの激しい衝撃と浸漬に耐えると述べました。 ボクシングに加えて、サウンドトラックに特別な要件が課される撮影用に設計されたスロット付きのカバーがあります。









カメラはさまざまな形状の複数の場所に取り付けることができ、ハンドルマウント、無料のロック用の洗濯ばさみもあります。

>設計と人間工学



カメラは、マルチカラーのフロントパネルを備えた黒で入手できます。 ケースは耐久性のあるプラスチックでできており、ソフトタッチコーティングが施されているため、カメラが手から滑り落ちません。 デバイスの寸法は30 x 59 x 41で、重量は59グラムです。

フロントパネルには、魚眼レンズ、電源ボタン、メニューナビゲーションがあります。







カメラの上部にはシャッターボタン\メニュー項目の選択と撮影インジケーターがあります。







右側には、Micro-HDMI、Micro-USB 2.0、最大32 GBのmicroSDメモリカード用のスロット、およびマイクスロットのコネクタがあります。







左側には、可聴アラームのスピーカーとメニューのナビゲーションボタンがあります。







背面パネルには1.5インチの画面とステータスインジケータ（赤と青）があります。充電中は青と赤のLEDが点灯します。充電が完了すると、赤のLEDが消灯します。ムービーモードはインジケータの点滅（上部の黄色と背面の青）で示されます。







カメラの下部にあるプラグを取り外し、バッテリーを取り外します。バッテリー重量は16グラム、容量は900 mAh、GoProは互換性がありません。









>鉄



カメラにインストールされたNovatek NT96650プロセッサは、故障や過熱なしに、30fpsの周波数で1920 * 1080の解像度のビデオを簡単に処理します。

このプロセッサは2013年にリリースされ、当時は価格と品質の点で最高でした。 DVRの中で多くのベストセラーがこのプロセッサを搭載しています。



カメラで撮影できるもの：

1）モードのビデオ：1080p_30k / s、720p_60k / s、480r_60k / s（カメラはハードウェアで30fpsのみをサポートし、ビデオの60fpsはソフトウェアフレーム複製で取得されます：1つではなく2つの同一フレーム）。

2）5MP〜12MPの解像度の写真。



>インターフェースとソフトウェア



カメラのインターフェイスは、ミニマリズムに簡素化されています：前にある-オンボタン\モード切り替え; top-OK-撮影の開始/終了。 side-メニューを介したボリュームズーム遷移。 すべてのボタンは直感的で、盲目的に簡単に検索できます。







PCに接続すると、カメラはフラッシュドライブまたはWebカメラとして動作モードについて尋ねます。

いくつかの動作モードと機能に注意してください。1）3/5/10分周期のビデオ録画とダビングのモード。 2）フォトタイマー2秒/ 5秒/ 10秒。 3）CAR DVモード（車のレジストラ：カメラは充電器が接続されると録画を開始し、その後停止することができます）; 4）記録中の写真。







>さまざまなモードでの撮影



固定焦点距離レンズの視野角は170°です。 ビデオの撮影には、640 * 480〜1920 * 1080の範囲の解像度を使用できます。

SJ1000は12メガピクセルの解像度で写真を撮ることができます。 中程度の品質の画像が取得されますが、空間的な歪みがあり、エッジの鮮明さが低下します。

>結論：



SJ4000は私に好印象を与えました。 その価格では、このモデルは最高の評価に値します。



長所：

1. FullHD 1080p 30fpsを撮影する（オブジェクトの適切な照明で鮮明な画像）

2. 1.5 "カラーディスプレイビューファインダー

3.ビデオストリーム出力用のマイクロHDMIインターフェイス

4.車のレコーダーモード：周期的なビデオ録画（3、5、10分）、画像回転、充電中のビデオ録画、モーションセンサー

5.コンパクトなサイズと軽量（59 g）

6.防水保護ボックス（最大30m）およびパッケージ内の豊富なアクセサリセット

7. GoProのアクセサリーとの互換性

8.コンピューターに接続したときのウェブカメラモード

9.充電用LEDインジケータ、撮影モード



短所：

1. 60fpsのサポートの欠如（Novatek NT96650プロセッサのハードウェア制限）

2.バッテリーを交換した後、日付と時刻が迷う

3.暗闇で撮影すると、画像の鮮明さが失われます
