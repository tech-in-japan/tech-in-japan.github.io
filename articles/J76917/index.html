科学コラボレーション：ソーシャルメディア、Wiki、Google Wave

ネットワーク上の科学協力は、科学者がコミュニケーションするためのツールとして作成されたため、Webでの科学協力のアイデアは新しいものではありません。 しかし、過去数十年で、このプロセスがWeb 2.0のスタイルでソーシャルメディアとコラボレーション用ツールの開発によって近年ほど激しく発生したことはない、 とカナダのITジャーナリストMatthew Ingram（Mathew Ingram） は書いています 。 彼は科学者向けに特別に作成された多くの新世代サービスをリストしています。



たとえば、科学論文の書誌データを比較および分析するためのメンデレーのオンラインサービス。 PDF形式のドキュメントをアップロードし、プログラムがそれを分析し、参考文献全体を抽出し、データベース内の既存のレコードと一致するものをすべて見つけます。 科学者はこのプロジェクトを気に入っていました。開発者が BBCのインタビューで述べたように、サービスの月間視聴者はすでに70,000人を超えています。



科学コミュニティでは、Wikiベースのエンジンとブログの両方を使用しています。 たとえば、 Project Polymathは、複雑な数学的問題で協力するためのサービスです。 Project Polymathの創設者は、LinuxInsiderとのインタビューで、科学的研究の中でオープンソースの原則について語りました。



または、ここに別のwikiサイトがあります：バイオインフォマティクス組織のコミュニティプロジェクト。 1998年の創設以来、世界中の27,000人の研究者がこの組織に登録しており、サイトはコミュニティ全体の中心であり、そこで質問、モデル、実験、計算生物学に関連する発見を公開しています。 これは、この分野の科学者の一種のソーシャルネットワークです。



そのサイトがソーシャルネットワークに似ている場合、科学コミュニティには独自のLinkedIn： ResearchGateと呼ばれるプロジェクトがあります。これは、関連分野のさまざまな著者の科学作品間のつながりを追跡できる深刻なソーシャルネットワークです。



「科学者間のコミュニケーションは、新しい知識の広がりを加速させます。 科学は協力であり、科学ソーシャルネットワークはこのような協力を支援し、改善します」とResearchGateは宣言しました。



科学者は現在、Google Waveなどの新しいツールとのコラボレーションを改善する方法について考えています 。 彼らの意見では、Volnaの主な強みは自動化にあり、キーワードによる科学文書は自動的にテーマウェーブに「吸い込まれ」、特定の分野の資料の検索を大幅に促進し、専門家間のコミュニケーションを刺激します。
