Ohlohの統計





このグラフは、Ohlohのサイト統計に基づいたオープンソースプロジェクトのアクティブなコミッターの数を反映しています。 コミット、変更されたコード行、プロジェクトの数を比較することもできますが、それでも私にとっては、これが業界の最も反映された状態のようです。 いくつかの結論を下しましょう。



1. C / C ++は徐々に地位を失っています。 Apache HTTPサーバー、Linuxカーネル、Open Office、Firefox、GCCなどのプロジェクトには他のオプションはありませんが、新しい大きな「低レベル」プロジェクトも存在しないことを認めましょう。

2. Javaは新しいリーダーになりました。 Apache、Eclipse、JBoss、SunのほぼすべてのプロジェクトとSourceForgeのかなりの部分が書かれています。

3. Perlの衰退期にある動的言語。 しかし、 Pythonは着実な成長を見せており、 PHPは一時的にそれを追い越しましたが、今ではリーダーシップがJavaScriptに移行し始めています。JavaScriptはWeb（RIA）でさらに強化され、他の言語をサーバーの奥深くまで押し込みます。

4.私はすでにRubyの仕事の市場の状態について書いています。 この言語のオープンソース開発者も少し眠りました。

5. C＃は、C / C ++およびJavaと並んで欠員の面で3つのリーダーの1つであるという事実にもかかわらず、.NETの開発者の間でのオープンソースはあまり人気がありませんが、成長は最近概説されています（主にMonoとCodePlexによる）。
