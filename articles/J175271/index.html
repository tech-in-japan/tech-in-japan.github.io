発表：TurbomilkのDenis KortunovとYegor Gilevが、DevCon 2013の一環としてデザインスクールを開催します





親愛なるソフトウェア開発者と開発者！



DevCon 2013チームを代表して、会議でデザインスクールを発表できることを嬉しく思います

設計の専門家およびTurbomilkスタジオの創設者による「非設計者向けのインターフェースの設計」

デニス・コルチュノフとイェゴール・ギレフ 。



最新のアプリ設計についてもっと知りたいですか？ UIおよびUXデザイナーのすべてを理解したいですか？ ユーザーインターフェイスの相互作用の現在の傾向に興味がありますか？ そうすると、デニス・コルチュノフとイゴール・ギレフのデザインスクールがあなたにとって非常に役立つでしょう！



学校の目的



UIデザイナーの作業は、インターフェイスを実装する開発者なしでは意味がありません。 しかし、多くの場合、デザイナーとプログラマーはお互いを理解していません。



学校の主な目標：



  • 会議参加者にインターフェース設計の基本を紹介する
  • デザイナーと開発者間の相互作用の主な問題とそれらを解決する方法を検討する
  • 現在の傾向とユーザーインターフェイスとのやり取りの方法について話す


学校講師







エゴール・ギレフ



Turbomilkデザインスタジオの共同設立者。 10年の経験を持つアイコンおよびインターフェイスデザイナー。 Pixelpalooza 2004アイコンデザイナーコンテストの受賞者、タイムマネージメントエバンジェリスト、SnailRiderブログの著者。







デニス・コルトゥノフ



Turbomilkデザインスタジオの共同設立者。 ウェブサイトとインターフェースの設計に13年以上携わっています。 以前は、Expressionの方向でMicrosoft MVPのステータスがありました。 サマーラで毎年開催される「 フェスティバル404 」の主催者。



学校プログラム



デザインスクール「非デザイナー向けインターフェイスデザイン」の枠組みの中で、3つのマスタークラスがDevCon 2013で開催され、それぞれ1時間続きます。



  • Windows Phoneアプリケーションインターフェイス：簡単な作り方
  • Windows 8のアプリケーションインターフェイス：バグの処理（エイリアン）
  • UXデザイン：生きている人々のためのインターフェースの作成


すべての学校の参加者はデカールを受け取ります。



DevCon 2013について



DevConカンファレンスは数年にわたって開催されており、すでにロシアのデベロッパーや他国のゲストに愛されています。 この一連の会議の重要な特徴は郊外形式であり、会場はモスクワ地方の最高のリゾートから厳選されています。 同時に、会議のすべての参加者は田舎の別荘に電話をかけ、会議の2日間その中に住み、これらの場所のすべての楽しい利点を受け取ります。



DevCon 2013の主要なテーマは次のとおりです。



  • 最新のクライアントアプリケーション
  • モバイル開発
  • クラウドコンピューティング
  • ソフトウェア開発およびライフサイクル管理ツール
  • 開発技術とプログラミング言語
  • アプリケーションストア向けアプリケーションの開発、収益化、プロモーション
  • マイクロソフトモダンデザイン
  • 企業開発、データ処理


さらに、会議参加者は、メインプログラム以外のさまざまな学習活動に独自にアクセスできます。



  • 関数型プログラミングスクール
  • 集中的なソフトウェアテストトレーニング
  • Runa CapitalのGaidar Magdanurovのマスタークラス
  • アジャイルの第一人者、アスハット・ウラズバエフとヴィクトル・ストレルコフによる、スクラムの理論と実践に関するマスタークラス 。
  • ハッカソン 「コーディングの夜」。
