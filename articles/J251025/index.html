3CX WebMeeting 7.1 Betaの紹介-次世代のビデオ会議

3CXチームは、新製品である3CX WebMeeting 7.1を発表します。これは、高解像度のプレゼンテーションと画面共有のためにWebRTCテクノロジーの改良を使用しています。 これはすべて、オンライン会議の有効性を大幅に高めることができます。 オンライン会議の品質を向上させるために、PowerPoint-HTML5 Converterが3CX WebMeetingに追加されました。これにより、プレゼンテーションの明瞭なプレゼンテーションが保証され、最大限の効果が得られます。 3CX WebMeeting 7.1は、帯域幅の使用率も改善し、オンライン会議の品質に影響を与える可能性のある参加者のインターネット接続の問題を示すインジケーターを追加しました。

3CX WebMeeting 7.1は、3CX Phone System 12.5 （最大10人の参加者）のユーザーが無料で利用できます。または、ローカルバージョンの 3CX WebMeeting Serverをダウンロードできます。



次に、主な新機能と改善点について詳しく説明します。



新機能と改善点



  • ホストされている各3CXポータルの同時参加者の数が3から10に増加しました。
  • ビデオ通信と画面共有の帯域幅が256 Kbpsに増加しました。
  • 現在のネットワークの使用状況をユーザーに通知する接続品質インジケータが追加されました。
  • HTML5形式のプレゼンテーションをプリロードして、狭いインターネットチャネルを使用している場合でも明確な表示品質を提供する機能。
  • 3CXPhoneクライアントから参加者をインポートして、会議の整理を迅速化します。
  • 高品質ビデオの帯域幅制御。


ドキュメント



3CX電話システムv12.5ユーザー向けの3CX WebMeeting：

  • 3CX電話システムで3CX WebMeetingアカウントを有効にするには、手順をご覧ください 。
  • 重要な注意。 3CX Phone System v12.5にアップグレードした3CX Phone System v12ユーザーは、3CX WebMeetingライセンスを無効にしてから再度有効にして、10人の無料メンバーにアクセスする必要があります。
  • 3CXPhone for WindowsおよびMacを使用して3CX WebMeeting Web会議を作成する方法を学びます。


3CX WebMeeting Serverのローカルインストールが必要なユーザー：

  • ローカルバージョンのインストールおよび設定方法については、 3CX WebMeeting Guideをご覧ください。
  • 管理ガイドには、ユーザーの作成と連絡先の追加に関する情報が記載されています。
  • 3CX WebMeetingユーザーガイドは、Webインターフェイスを使用してWebMeeting会議を作成する方法を示しています。
