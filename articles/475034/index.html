Example usage for Arduino.



1. Run SVMonitor, and in the settings for the Web server, specify the IP address and port.







2. Fill such a sketch in MK: 



#include <SVClient.h> void setup() { bool ok = svisual::connectOfCOM("client"); } int cnt = 0; void loop() { svisual::addIntValue("dfv", cnt); ++cnt; if (cnt > 10) cnt = 0; bool odd_cnt = cnt % 2; svisual::addBoolValue("bFW", odd_cnt ); svisual::addBoolValue("bBW", !odd_cnt ); delay(200); }


3. Next, you need to restart SVMonitor and reconnect the MK.



4. Now in SVMonitor we should see the connected MK:







5. Open the browser, enter the address specified in the SVMonitor settings in the address bar and see the following page:
