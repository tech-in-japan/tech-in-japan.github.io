MS Outlook形式がオープンになります

Microsoftは、Open Specification Promiseイニシアチブの一環として.PSTを開きます。 したがって、どの開発者も、Microsoftの許可を求めることなく、少なくともMozillaメールプログラム、少なくともGoogleデータセンターで、クライアントおよびサーバープログラムで.PST形式を使用できます。 これは、Microsoft Office Interoperabilityのマネージャーが企業ブログで発表したものです。



.PST（個人用ストレージテーブル）形式は、MS Outlook用の閉じたユーティリティ形式で、文字、日記などのデータベースを含むすべてのユーザー情報を保存します。 現在、データベースへのアクセスは、 Messaging API （MAPI）およびOutlook Object Modelを介して行われますが、.PST形式を開いた後、Microsoftソフトウェアをまったくインストールせずに.PSTファイルを作成および編集できます。



将来、Microsoftは.PSTファイルの構造、階層ナビゲーションのヘルプ、および個々のオブジェクトとプロパティへのアクセスに関するガイダンスを含む技術文書を公開します。 ドキュメントは執筆の初期段階にあります。
