プログラマーズデーにVisual Studio 2012を起動してください！



今日、プログラマの日に、注目すべきイベントが開催されます-開発者向けの新製品のリリース-Visual Studio2012 。アドレスwww.visualstudiolaunch.ruで、数時間後に仮想イベントのブロードキャストが開始され、そこからこの製品に含まれる新製品について学習します。 Microsoft Visual Studio 2012には、Windows 8、インターネット、SharePoint、モバイルデバイス、クラウドサービス向けの多くの新しい開発機能と、従業員の障壁を取り除き、人件費を削減するのに役立つアプリケーションライフサイクル管理ツールがあります。





Visual Studio 2012の新機能



アプリケーションに新しい時代が到来したことは秘密ではありません。 常に接続されているデバイスとクラウドサービスは、以前はアクセスできなかった機能を提供します。 開発者はどこからでも接続でき、何百万人ものユーザーがすぐに利用できる一流のアプリケーションを作成できます。



そのため、Visual Studio 2012を最も重要なリリースの1つと考えています。 この環境は、最も重要なことは新しいアイデアとその実装の速度である環境で成功できるように特別に設計されています。 新しいバージョンには、非常に多くの変更が含まれています。 それらの詳細については、リンクmsdn.microsoft.com/ru-ru/library/bb386063.aspxで（ロシア語で）学ぶことができます。 また、2〜3分でVisual Studio 2012の機能を理解するのに役立つ短い紹介ビデオも用意しました。







新しい外観



環境を起動した瞬間から、多くの変更が行われたことがわかります。 インターフェイスは完全に再設計され、作業が簡素化され、日常のツールへのアクセスが容易になりました。 ツールバーがより簡単になり、タブの数が減り、コードで検索する新しい方法が登場しました。 これにより、アプリケーション全体のナビゲーションが簡単になり、必要に応じて作業できるようになります。





Windows 8対応



Windows 8のリリースにより、すべてが大幅に変更されました。 Visual Studio 2012には、新しいテンプレート、コンストラクター、テストおよびデバッグツールがあります。魅力的なアプリケーションを最短時間で作成するために必要なものはすべて揃っています。 同時に、Blend for Visual Studioは、新しいWindows 8インターフェイスを最大限に活用するためのビジュアルツールを提供します。





高度なWeb開発ツール



Web開発に関する限り、Visual Studio 2012には必要なものがすべて備わっています。新しいテンプレート、改善された公開ツール、HTML5やCSS3などの新しい標準の完全サポート、ASP.NETの最新の改善点があります。 また、ページインスペクターツールを作成してデバッグを簡素化しました。これにより、開発環境で作成中のページと直接やり取りすることができます。 モバイル市場への参入を計画していますか？ ASP.NETを使用すると、アプリケーションで画面が小さい携帯電話、タブレット、およびその他のデバイス用に自己最適化するコントロールを使用できます。





クラウドサポート



以前は、全員が独自のサーバーを個別に保守する必要がありました。 規模を拡大するには、多大なインフラ投資が必要でした。 これで、クラウド内のほぼ無制限の数のサーバーにアクセスできるようになり、必要に応じてメモリと計算能力を拡張できます。 Visual Studioの一部として、新しいテンプレートや公開オプション、分散キャッシュのサポート、インストールプロファイルの削減など、アプリケーションをWindows Azureに移植するための第一級のツールがあります。





ビジネスアプリケーションサポート



また、新しいデザイナー、テンプレート、展開オプションなど、SharePointでの作業が大幅に改善されていることに気付くでしょう。 パフォーマンスプロファイリング、単体テスト、IntelliTraceなど、SharePointの高度なアプリケーションライフサイクル管理機能を使用できます。 おそらく最も嬉しい驚きは、集中的なコーディングなしで動作するアプリケーションを作成できるツールであるLightSwitchです。





柔軟なプロセスと堅牢なアプリケーションライフサイクル管理



アプリケーションの複雑さが増すにつれて、チームワークの構築を支援するツールが必要になります。 だからこそ、私たちは仕事に柔軟なアプローチを導入しました。 Visual StudioとTeam Foundation Serverを使用すると、既存のワークフローを壊すことなく、計画に従って柔軟なワークフローを実装できます。 環境ではプロセスのすべての参加者からの要件とフィードバックを追跡できるため、すべての従業員を仕事に引き付けることもできます。



また、アプリケーションのライフサイクルを管理する責任を与えることもできます。 Team Foundation Serviceを使用すると、適切なインフラストラクチャを維持する必要なく、ALMのすべての機能を利用できます。 これにより、小さなチームでもバージョン管理、コード分析、柔軟な計画を活用できます。



学習して作成する時間です！



もちろん、これはVisual Studioに関するニュースのほんの一部です。 独自に、 msdn.microsoft.com / en-us / library / bb386063.aspxリンクで、コードの操作、アプリケーションのライフサイクルの管理、および開発の実施時にさらに多くの改善点を見つけることができます。 そして、仮想起動www.visualstudiolaunch.ruに必ずアクセスしてください ！
