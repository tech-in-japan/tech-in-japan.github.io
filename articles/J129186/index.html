Huawei Honorスマートフォンが正式に発表

最近、Huaweiは新しいHonor U8860スマートフォンを導入しました。 インターネットで見つかったすべての情報をまとめてみます。



そのため、Huaweiは中国でHonor U8860スマートフォンを公式に発表しました。 すぐにそこにオペレーターChina Unicomとともに販売されますが、他の市場、たとえばヨーロッパでは、デバイスも表示される場合があり、米国では、このスマートフォンはオペレーターCricketからHuawei Gloryという名前で表示されることが予想されます。 いくつかの噂によると、メーカーはこのデバイスの価格を300ドルと約束しています。

著者から：もちろん、情報は非常に疑わしいため、これまでのところ499ルーブルの先行予約のみが開かれていますが、実際にそのような価格がある場合、このデバイスはこの価格カテゴリに競合他社がいないようです



製造元によると、彼の新製品は最大3日間アクティブモードで動作します（ Androidにとっては悪くありません-およそ著者 ） 。特に大容量バッテリー（1900 mA / h）の存在と最新技術のサポートにより、Huawei Honorはわずかです厚さ（10.9 mm）、重量140 g。



デバイスの技術的特性は次のとおりです。



  • 通信規格のサポート：3G WCDMA / HSPA + 900 / AWS / 2100 MHz; GSM / GPRS / EDGE 850/900/1800/1900 MHz
  • プラットフォーム：Android 2.3.5
  • CPU：1.4 GHz
  • ディスプレイ：4インチ、1600万色以上、16：9、FWVGA解像度を表示
  • メインカメラ：8 MP、HDR、オートフォーカス付き
  • フロントカメラ：2 MP
  • FMラジオ
  • マイク2本
  • メモリ：512 MB RAM、4 GB ROM、microSDカードスロット
  • 通信：WiFi 802.11b / g / n、DLNA、Bluetooth v2.1（A2DPプロファイルを含む）、Micro USB 2.0、GPS
  • その他：電子コンパス、ジャイロスコープ、加速度計、光センサー
