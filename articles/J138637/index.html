Wi-Fiサイト調査

Wi-Fiワイヤレスネットワークプロジェクトには、設計作業の段階および機器の設置前の対象物の無線調査を常に含める必要があります。 これは、正しく実行された場合に、予測可能な特性を備えた実行可能なワイヤレスネットワークソリューションを作成するための十分な理由を得るための唯一の実際の機会です。



無線システムでは、テスト機器を使用せずに電波の伝搬を予測し、干渉の存在を判断することは非常に困難です。 全方向性全方向性アンテナを使用しても、実際には、電波はすべての方向に同じ距離を伝播するわけではありません。 代わりに、壁、ドア、エレベーターシャフト、人などのさまざまな障害物。 異なるレベルの信号減衰を導入します。これが、無線の放射パターンがあいまいで予測不能になる理由です。 その結果、無線アクセスポイントを展開する前に、無線信号の動作と分布を完全に理解するために、Wi-Fiネットワークのカバレッジエリアの無線調査（サイト調査）を実行する必要がしばしばあります。

サイト調査の主な目的は、アクセスポイントの数と位置を決定するために十分な情報を得て、ターゲットエリア全体に必要なカバレッジを提供することです。 ほとんどの場合、必要なカバレッジは、最小データレートを確保することによって決定されます。 無線調査では、他のソースからの干渉の存在も判断され、WLANのパフォーマンスが低下する可能性があります。



サイトの無線調査の要件と複雑さは、サイト自体とその特性によって異なります。 例えば 複数のオープンルームで構成される小規模オフィスでは、無線調査はまったく必要ない場合があります。 ただし、干渉は確認する価値があります。 このシナリオは、おそらくオフィスのどこかに1つのアクセスポイントをインストールすることで実装でき、一般的なタスクのカバレッジは十分であると期待できます。 アクセスポイントが隣接オフィスのWLANからの干渉に遭遇した場合、ほとんどの場合、隣接する非オーバーラップチャネルに切り替えることで問題を解決できます。



大規模オフィス、住宅、病院、格納庫、ワークショップなどの大規模な施設では、通常、詳細な無線調査が必要です。 調査を行わないと、ユーザーがカバレッジが不十分になり、一部の地域でネットワークパフォーマンス（帯域幅）の問題が発生する可能性が非常に高くなります。 展開後に問題が発生して無線サブシステムの再設計が必要になった場合、数十個のアクセスポイントのインストール場所とそのすべての接続を再インストールする可能性はほとんどありません。

Wi-Fiのオブジェクトの無線調査を実施する場合、次の手順を推奨できます。



1.フロアプランを取得する

ラジオ調査を開始する前に、将来のネットワークの全領域の計画を取得します。これには、カバレッジがあるはずのすべての部屋のフロアプランが含まれます。 利用できるものがない場合は、寸法で計画を描き、すべての壁、通路、窓、エレベーターなどの位置を示します。



2.オブジェクト全体を視覚的に検査する

テストを開始する前に、施設全体を調べて、フロアプランの精度を確認します。 これは、無線信号の伝搬に影響を与える可能性のある潜在的な障害物を特定する良い機会でもあります。 たとえば、目視検査は、通常はフロアプランにない金属製のキャビネットやパーティションなど、無線信号に対する障害物を識別するのに役立ちます。



3.将来のWLANユーザーを見つけます。

フロアプランで、有線および無線接続を使用してユーザーの領域をマークします。 さらに、ワイヤレス/モバイルユーザーにローミングが必要な場合と、行かない場合を示します。 ウェブサイトのWi-Fi-ResebnikのCampusネットワークのトレーニングタスクの例に示すように、ローミングゾーンを制限したり、継続的なカバレッジではなくWi-Fiの「ホットゾーン」を整理するモデルに切り替えたりすることができる場合は、より少ないアクセスポイントで対応できる可能性があります。



4.アクセスポイントの予備インストール場所を特定する

以前は、WLANユーザーの位置、予想されるカバレッジエリア、セルのサイズ、ネットワーク上のサービス、無線サブシステム自体の要素を分析することにより、アクセスポイントのロケーションと数を推定して、必要なエリアの適切なカバレッジを確保できました。 継続的なカバレッジを確保するには、隣接するアクセスポイントのセルのオーバーラップを計画する必要がありますが、チャネルをアクセスポイントに割り当てるとき（手動構成または予備計画中）、同一の周波数チャネルを持つポイントは、存在しないか最小限になるように十分に離れている必要があります隣接するアクセスポイントを通過する放射からの干渉。 2.4GHzスペクトルでは、オーバーラップしない周波数チャネル1、6、11が3つしかないことに注意してください。十分なレベルのセルオーバーラップを検討できることも追加する価値があります。

-メインサービスとしてデータ転送サービスを提供する場合、10〜15％のオーダーの重複、

-ネットワークがWi-Fi経由で音声サービスVoIPを提供する場合、20％のオーダーの重複、



Wi-Fiネットワークを計画するための特別なツールは、アクセスポイントの場所を予備的かつ合理的に決定するのに役立ちます。 同様のソフトウェアモジュールは、たとえば、WCS / Wireless Control Systemおよび新しいCisco Prime NCS / Network Control Systemなどの有名なCisco WLAN管理システムに統合されています。 ちなみに、これらのシステムは30日間利用可能なデモバージョン（有効期間が制限された本格的なシステム）です。 シスコからダウンロードできます。

しかし、これは、ラジオ調査の実施を、それを置き換えることなく促進するだけです。 フロアプランでのAPの位置の評価では、調査を実施し、APの推奨条項を検証および調整する必要があります。



アンテナアクセスポイント、データケーブル、および電源ケーブルの設置には適切な取り付け位置を特定する必要があり、アクセスポイントの位置を決定する際には、さまざまなタイプのアンテナの必要性も考慮してください。 たとえば、APを建物の外壁の隣に取り付ける場合、この場合の最善のアプローチは、おそらく建物内で比較的高いゲインの指向性パネルアンテナを使用することです。 統合アンテナを備えたアクセスポイントを使用する場合、天井に最も正しく配置されるような放射パターンを使用することがよくあります（仮天井の後ろではなく、部屋の内側に突出する必要があります）。 当然、この場合、天井の高さは通常のオフィスでは通常でなければなりません。 天井が高い部屋やワークショップ/格納庫では、指向性アンテナを備えたAPを使用します。 このトピックのいくつかの例は、Wi-Fi ResechnikのWebサイトのキャンパスのWi-Fiネットワークの例で説明されています。



5.アクセスポイントの場所とネットワークパラメータの実際のレベルを確認する

これは、実際のテストが開始されるときに発生します。 通常、複数のアクセスポイントが施設の事前に計画された位置に配置され、無線調査を実施するための専用のツール（Ekahau、Fluk / AirMagnetなど）を使用してフィールドテストが実行されます。 調査中に実際のネットワーク上にあるアクセスポイントとアンテナのモデルを正確に使用すること、およびネットワークで見られる無線特性に関して最悪のユーザーデバイスを考慮してテストを実行することが非常に重要です。 また、無線ネットワークの特性を取得するパッシブテストだけでなく、実際のトラフィック負荷を形成してアクティブテストを実行することも非常に重要です（通常、アクティブなラジオ調査機能を備えたツールには組み込みのメカニズムがあります）。 これのみが、ネットワークの将来の動作の実際の図を表示します。



また、2.4GHzおよび5GHzの周波数範囲の兵器庫にスペクトルアナライザーがあることは非常に便利です。 これにより、カバレッジエリア内の干渉パターンを識別し、正確に想像することができます。



6.結果を文書化する

満足のいくテスト結果が得られ、アクセスポイントやアンテナの正しい位置が決定された瞬間から、このデータをサイトプランに入力する必要があります。 また、信号レベル、データ転送速度などのレポートログを保存して添付する必要があります。 各アクセスポイントの予想されるセル境界まで。 これにより、将来のネットワーク再設計作業のための基本情報を入手できます。

ここで説明する手順は正しい方向に導きますが、実際の経験に置き換わるものではありません。 これらがWLAN / WLANの方向への最初のステップである場合、適切な専門知識を持って会社に連絡することは理にかなっています。 たとえば、多くのシステムインテグレーターは従業員を訓練しており、これらの作業を有料で実行できます。
