OpenDataScienceとMail.Ru Groupは、オープンな機械学習コースを実施します

2017年9月6日、データ分析と機械学習に関するOpenDataScienceオープンコースの2回目の開始が開始されます。 今回は、ライブレクチャーが開催され、サイトはMail.Ru Groupのモスクワオフィスになります。













要するに、コースはHabré（ ここが最初）の一連の記事、複製された資料（Jupyterノートブック、こちらがコースのgithubリポジトリ ）、宿題、Kaggle Inclassコンテスト、チュートリアル、個々のデータ分析プロジェクトで構成されています。 ここでコースにサインアップできます。 ここで 、OpenDataScienceコミュニティに参加してください。ここでは、コース中にすべての通信が行われます（Slack ODSのチャネル#mlcourse_open）。 そして、より詳細に言えば、これは猫の下であなたのためです。

コースの特徴は何ですか









このコースの目標は、既存の知識をすばやく更新し、さらに学習するためのトピックを見つけることです。 コースは、このトピックの最初のコースに完全に適合するとは考えられません。 データ分析と機械学習に関する包括的なコースを作成するタスクを設定しませんでしたが、理論と実践の完璧な組み合わせでコースを作成したいと考えました。 したがって、アルゴリズムは数学を使用して十分に詳細に説明され、実践的なスキルは宿題、競技、個々のプロジェクトによってサポートされています。







この特定のコースの大きなプラスは、フォーラム（OpenDataScienceのSlackコミュニティ）でのアクティブライフです。 一言で言えば、OpenDataScienceはロシア語を話すDataScientistsの最大のコミュニティであり、 Data Festの開催を含む多くのクールなことを行っています 。 同時に、コミュニティは積極的にSlackに住んでいます。そこでは、誰でもDS質問への回答を見つけたり、プロジェクトの志を同じくする人々や同僚を見つけたり、仕事を見つけたりすることができます。 オープンコース用に別のチャンネルが作成されました。このチャンネルでは、新しいトピックを習得するのに役立つ3〜400人の人々が同じことを勉強します。







素材のプレゼンテーション形式を選択して、HabréおよびJupyterノートブックに関する記事を作成しました。 これで、ライブ講義とそのビデオが追加されます。







コースの対象者と準備方法



前提条件：専門大学の2年次レベルで数学（線形代数、解析幾何学、数学的分析、確率論、統計）を知っている必要があります。 Pythonで少しプログラミングできる必要があります。







十分な知識やスキルがない場合は、シリーズの最初の記事で、数学を繰り返し、Pythonプログラミングスキルを更新（または取得）する方法について説明します。







はい、英語の知識はもちろん、ユーモアのセンスも傷つけません。

講義



講義は、9月6日から11月8日までの19:00から22.00までの水曜日に、Mail.Ru Groupのモスクワ事務所で開催されます。 講義では、記事で説明されているのと同じ計画に従って、理論全体を分析します。 しかし、講師によるタスクのライブディスカッションも行われ、各講義の最後の1時間は練習に専念します-学生はデータを自分で分析し（そう、直接コードを書く）、講師がこれを手伝います。 現在の評価のコースの上位30人の参加者が講義に参加できます。 ランキングは、宿題、競技会、データ分析プロジェクトの影響を受けます。 講義放送も開催されます。













講師：







  • ユーリ・カシニツキー 。 Mail.Ru Groupのプログラマー研究者であり、HSEコンピューターサイエンス学部の上級講師であり、HSEでのデータ分析の年間教育プログラムの教師でもあります。
  • アレクセイ・ネイトキン OpenDataScienceコミュニティの創設者であり、Digineticaの最高データ責任者であるDM Labs。 以前は、デロイトの分析部門の責任者。 DataFestの主催者であるOpenDataScienceコミュニティのイデオロギーリーダー。
  • ドミトリー・セルゲイエフ 。 Zeptolabのデータサイエンティスト、モスクワ州立大学の数理ファイナンスセンターの講師。

宿題



10のトピックにはそれぞれ宿題が付き、1週間が与えられます。 タスクはJupyterノートブックの形式であり、そこにコードを追加し、それに基づいてGoogleの形式で正しい答えを選択する必要があります。 宿題は、コースの参加者の評価に影響を与え始め、それに応じて、誰がライブで講義に参加できるようになるか、最初のものです。







コースリポジトリで、 10の宿題とソリューションを見ることができます。 コースの新しい立ち上げでは、宿題は新しくなります。







チュートリアル



コース中の創造的なタスクの1つは、データ分析と機械学習の分野からトピックを選択し、それに関するチュートリアルを作成することです。 ここでの例を知ることができます 。 経験は成功したことが判明し、コース参加者自身が、コースで考慮されなかったトピックに関するいくつかの非常に堅実な記事を書きました。







Kaggle Inclassコンペティション









もちろん、どこでもデータを分析する練習をしなくても、競争の中ですぐに何かを学び、その方法を学ぶことができます。 さらに、さまざまなパンの形での動機（「大きな」Kaggleでのお金と格付け、そして私たちが知っている格付けの形での格付け）は、データ分析競争中の新しい方法とアルゴリズムの非常に活発な研究に貢献します。 コースの最初の開始では、非常に興味深い問題が解決された2つのコンテストが提供されました。







  • インターネットでの行動による攻撃者の識別 。 さまざまなサイトを訪問しているユーザーに関する実際のデータがあり、30分以内に訪問したサイトのシーケンスから、それが誰かアリスであるか他の誰かであるかを理解する必要がありました。
  • Habréに関する記事の人気の予測 。 このタスクでは、テキスト、時間、およびHabréの出版の兆候によると、この記事の人気-お気に入りへの追加の数を予測する必要がありました。


個別プロジェクト









Vkontakteの公開 「成人男性向けの機械学習に関するミーム」から。







コースは2.5か月間設計されており、多くのアクティビティが計画されています。 ただし、教師が提案した計画に従って、独自のデータを使用して、最初から最後まで独自のデータ分析プロジェクトを完了する可能性を必ず検討してください。 プロジェクトは同僚と話し合うことができ、コースの終わりにプロジェクトのピアレビュー検証が手配されます。

プロジェクトの詳細については後ほど説明しますが、今のところは、「プロジェクトのために何かを予測する」ためにどのようなデータを使用するかを考えるかもしれません。 しかし、アイデアがなければ、問題ありません。分析のために興味深いタスクとデータをアドバイスします。それらは複雑さの点で異なる場合があります。







コースに登録するにはどうすればよいですか？



コースに参加するには、 このアンケートに記入し、OpenDataScience コミュニティに参加してください（「OpenDataScienceについてどうやって知りましたか？」の「mlcourse_open」と答えてください）。 コース全体の通信のほとんどは、＃mlcourse_openチャネルのSlack OpenDataScienceで行われます。







コースの最初の実行はどうでしたか



最初の打ち上げは2017年2月から6月に行われ、約1,000人がサインアップし、最初の宿題は520人、最後は150人でした。 フォーラムでの生活は本格的で、Kaggleコンテストで数千個の小包が作られ、コース参加者は多数のチュートリアルを作成しました。 また、レビューから判断すると、ニューラルネットワーク、Kaggleの競争、または機械学習の理論にさらに踏み込むことができる素晴らしい経験を得ました。







コースのトップ100のファイナリストにボーナスをもたらしたのは、Mail.Ru Groupのモスクワ事務所のmitapでした。これには、現代のDSに関連するトピックに関する3つの講義がありました。







  • Apache Sparkによるビッグデータ処理（Vitaliy Khudobakhshov、Odnoklassniki）。 ビデオ： part1 、 part2 ;
  • ニューラルネットワークとディープラーニングの基礎（Alex Ozerin、Reason8.ai）、 ビデオ ;
  • 感情分析問題の解決におけるディープラーニング（Vitaliy Radchenko、Ciklum）、 ビデオ 。
