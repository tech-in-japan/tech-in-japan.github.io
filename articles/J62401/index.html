UPD：ソリューションが見つかりました。



問題は、Firefoxの3.1 / 3.5以降では 、カラープロファイルが組み込まれた画像のカラー管理が有効になっていることです。

また、Ubuntuでは、これらの同じプロファイルがデフォルトで欠落しています。

2つの解決策があります。

最初のものは正しい：プロファイルをインストールします。

sudo apt-get install icc-profiles


その後、約：config

gfx.color_management.mode = 2

gfx.color_management.display_profile = /usr/share/color/icc/sRGB.icm




2番目は間違っています。 カラーグレーディングを無効にします。

約：config

パラメーターを設定する必要があります

gfx.color_management.mode = 0
