ラップトップ用のMicrosoftの新しいマウス

Microsoft Corporationは、モバイルPC用の3つの新しいワイヤレスマウスをすぐに導入しました。 これは、コンパクトなBluetoothノートブックマウス5000、4方向スクロールホイールを備えたワイヤレスノートブックレーザーマウス7000、およびモバイルメモリマウス8000です。トランシーバーも1 GBの容量のフラッシュドライブです。



Bluetooth Notebook Mouse 5000の寸法は、わずか58×93 mmです。 マウスには、ラップトップに接続するためのBluetoothインターフェイス（ラップトップにアダプターがある場合はトランシーバーは不要）と4つのボタン（標準ボタンに加えて、Webでのナビゲーションを容易にする「戻る」ボタンが装備されています）。 範囲は10メートルで、バッテリーの寿命は3か月ですが、アイドル時間中にデバイスの電源がオフになっている場合に限ります。 Bluetooth Notebook Mouse 5000の価格は50ドルです。



ワイヤレスノートブックレーザーマウス7000は、2.24 GHz（範囲10メートル）の周波数の無線信号をワイヤレスインターフェイスとして使用します。 このマウスの機能は、Windows VistaでFlip 3D機能を呼び出すための追加ボタンが装備されており、スクロールホイールが4方向に機能するため、Bluetooth Notebook Mouse 5000の機能よりもわずかに高くなっています。 バッテリーの寿命も長くなります-6か月。 しかし、コストはまったく同じです-50ドル。



ユーザビリティデバイスの観点から最も興味深いモバイルメモリマウス8000は、このマウスのトランシーバーも容量が1 GBのフラッシュドライブであるという点で異なります。 8分の1は、無線信号とBluetoothの両方で機能します。 Mobile Memory Mouse 8000には5つのボタンがあり、バッテリー寿命は6か月で、コストは100ドルです。
