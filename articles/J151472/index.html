Windows 8 Summit-最初のWindows 8パートナー会議

Windows 8がすぐに新しいデバイスと一緒に棚に保管されるようになるのは誰にとっても秘密ではないと思います（公式発売は10月26日に予定されています）。 1か月以上もあるようですが、ご存知かもしれませんが、今日ではソフトウェア開発業界で多くの準備作業が行われており、多くの企業や個人の開発者がWindows 8のソリューションと提供に積極的に取り組んでいます。







打ち上げの前夜にすべてを1か所に集めて、近い将来の市場開発の経験、計画、ビジョンを共有できるようにすることが非常に適切であると判断しました。 欲望を実現するために、8月の発売の数日前の10月下旬に、Windows 8 Summitを実行しています。



Windows 8 SummitはWindows 8での最初のパートナー会議です。名前が示すように、主な役割はマイクロソフトではなく、私たちの愛するパートナーに与えられ、彼らの経験から、すべてがうまくいくだけでなくレーキについても語ります、落とし穴と私たちの不完全な世界の他の困難:)。 会議は、ビジネスの聴衆、起業家、スタートアップ、プロの開発者を対象としています。



このカンファレンスには、2つの個別のトラックが含まれます。アプリケーション開発のすべての段階をカバーするビジネストラック-市場分析、設計、開発、収益化、プロモーション。 Windows 8の開発の基本を既に知っている開発者向けのテクニカルトラック。パートナーソリューションの展示会では、非公式の設定で専門家とコミュニケーションを取り、すべての質問をすることができます。
