サイトでバックグラウンドミュージックを作成する方法

今日、疑問が浮上しました：フラッシュを使用せずにサイトにバックグラウンドミュージックを配置するにはどうすればよいですか？そして、 最も重要なことは、ページを移動しても音楽の再生が停止せず、再生が開始されないようにすることです



少しグーグルで考えた後、私は決断しました：ユーザーがサイトにアクセスすると（たとえば、「音楽をオンにする」ボタンをクリックする）、javascriptが小さなウィンドウを開閉し、バックグラウンドミュージックが既に「再生」されます。



したがって、直接実装：



index.html-メイン



< html >

< head >

< title > </ title >

< script >

function openPage()

{

var newwin = window.open( 'bgsound.html' , 'ert' , 'scrollbars=0, resizable=0, titlebar=1, height=20, width=320' );

newwin.blur(); //

}

</ script >

</ head >

< body onload ="openPage()" >

< a href ="otherpage.html" > , ? </ a >

</ body >

</ html >
