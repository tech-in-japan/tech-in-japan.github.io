Windows Phone 7の開発者ツールの最終バージョン

Windows Phone 7プラットフォームが発表されたMobile World Congressカンファレンスの後、半年以上が経過しました。 9月1日の秋の初日に、Windows Phone 7プラットフォームのリリースが成功したことが発表されましたが、今日、バルセロナでのプラットフォームの発表からちょうど7か月後に、Windows Phone 7プラットフォームに基づくアプリケーション開発者向けの最終ツールが公開されました！ おめでとうございます！



既に発表したように、すべてのWindows Phone 7開発者ツールは無料のままです。 Visual Studio 2010のいずれかのエディションを既にインストールしている場合、使い慣れた開発環境から直接アプリケーションを作成できます。 Visual Studio 2010がインストールされていない場合、またはExpressエディションを使用している場合は、Windows Phone環境用の無料のVisual Studio 2010 Expressエディションでモバイルアプリケーションを作成できます。



さらに、インストールパッケージには、Windows Phone用Expression Blend 4、Windows Phone用Silverlight、Windows Phone用XNA Game Studio、そしてもちろんWindows Phone 7エミュレーターも含まれています。



公式サイト-download.microsoft.comから通常どおり開発者ツールをダウンロードします 。



開発者ツールのリリースに伴い、待望のコントロール-PanoramaとPivot-がリリースされました。 この点で、アプリケーションテンプレートのリストは拡大しています。



または、 Windows Phone 7用Microsoft Advertising SDKをダウンロードできます。これは、アプリに広告を配置できるパッケージです。 このリンクからこのパッケージをダウンロードできます。 広告プラットフォームの詳細については、少し後で説明しますが、今のところ、Raj Kapoorからの簡単な発表を読むことができます。



Silverlight Toolkitについても忘れないでください。SilverlightToolkitには、Windows Phone 7の便利なコントロールも含まれています。 このリンクからダウンロードできます。



そして最後に、Rescoは既にWindows Phone 7の独自のコントロールセットをリリースしました。それらを見て、そのうちのいくつかは役に立つかもしれません。



スコット・ガスリーとチャーリー・キンデルは、すでに最終開発者ツールのリリースを祝っています。 それらに遅れないようにしましょうか？
