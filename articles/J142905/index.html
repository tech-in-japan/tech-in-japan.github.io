E1に基づいたモバイル電子文書管理プラットフォームをテストした結果、Apple OSは最速として認識されています

当社は、E1 Euphratesロボットビジネスプロセスオートメーションシステムと、すべての主要なモバイルプラットフォーム（iOS、MS Windows Phone、Android、Symbian）との互換性の包括的なテストを完了しました！



テストの一環として、もちろんその目的は、さまざまな「スマート」電話でのモバイルクライアントのリモート作業の利便性と有効性を確認することでしたが、

さまざまなメーカーの4種類のスマートフォンが「実験的」として選択されました。 その中には、iPhone 4S（iOS）、HTC Titan X310E（MS Windows Phone）、Nokia N8（Symbian）、およびSamsung GALAXY S II（Android）がありました。



スマートフォンのかなり小さな画面は、リモートモードでの作業プロセスを大幅に複雑にします。 したがって、モバイルクライアントの機能をテストするための最も困難な条件を作成しました。





すべてのデバイスは、必要なすべてのコマンドと作業機能にアクセスできるE1ユーフラテスモバイルクライアントの包括的な包括的なテストを受けました。 テスト中に、テストが実行される文書の登録管理カード（RKK）のすべてのフィールドが埋められました。



テストの簡単な説明。
