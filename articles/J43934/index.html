Windows 7タスクバー-Windows 7ビルド6801の新機能と有効化方法。

現時点でWindows 7の最も興味深い機能です！

新しいタスクバーでは、Windows 7は非常にきれいに見えます。



このパネルに配置された各アプリケーションは、アイコンとして表示されます。



このアイコンにカーソルを合わせると、このアプリケーションのすべての実行中のウィンドウが表示されます。



各アプリケーションウィンドウは、タスクバーから直接閉じることができます。



パネルには、アプリケーションが開いているウィンドウまたはタブの数が表示されます。



アプリケーションが開いていて、ウィンドウがアクティブな場合、アイコンの背景は暗くなります。 開いているがアクティブではない場合、明るい背景：



アイコンにカーソルを合わせると、背景が強調表示されます。



この「バックライト」はカーソルの後に移動します。 さらに、アイコンは、アイコンの最も上にある色で強調表示されます。



一部のアプリケーションにプログレスバーが存在する場合（これまでのところ、そのようなアプリケーションはほとんどありません）、タスクバーのアイコン全体に緑色のバーが表示されます



任意のアプリケーションをタスクバーに追加したり、タスクバーから削除したりできます：



一部のアプリケーションは、アプリケーションウィンドウを開かずに管理できます。



設定メニューでは、小さなアイコンを表示してグループ化を無効にできます。





そして最後に、このチップを自分の目で見たいと思っている人、そして既にWindows 7ビルド6801をインストールした/インストールする準備ができている人のためのボーナス：

Windows 7 Pre-Betaをインストールすると、新しいタスクバーは表示されません。 理由はわかりませんが、デフォルトでは有効になっていません。 しかし、ラファエルという名の良い男が気づいたように、システム自体にはそうです。 彼に感謝します！ 新しいタスクバーを有効にする方法は次のとおりです。

x86またはx64バージョンの特別なユーティリティをダウンロードし、Windowsフォルダーにコピーします。 次に、ノートブックを開いて、ノートブックに貼り付けます。



それだけです、新しいパネルが登場しました:)喜び、テスト、そして再び喜び:)
