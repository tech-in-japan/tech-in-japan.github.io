Android ZenPad Ensoタブレット

香港に本拠を置き、コンシューマーおよびパーソナルエレクトロニクス市場に焦点を当てた新会社Ensoは、2010年3月20日に、Androidオペレーティングシステムをベースにした最初の製品であるEnso ZenPadタブレットを、基本価格155ドルで発売しました。



ZenPad Ensoは、132.2 x 87.6mmのデバイスです。 厚さ14.8mm。 そして185グラムの重さ。 LEDバックライトの5インチスクリーンと800x480の解像度。



667MHzの周波数を持つSamsung 6410プロセッサによって制御されます。 データストレージには、1GBの内部フラッシュメモリと8GBのmicroSDが使用可能です。これは含まれています（最大32GBまで拡張可能）。 これらはすべてAndroid OS 1.6に基づいて機能します。



搭載されているワイヤレステクノロジーには、Wi-Fi（802.11a / b / g / n）があります。 UMTS / HSDPA / EVDO（850、1900、2100 MHz）+ GSM / EDGE（850、900、1800、1900 MHz）の特性を備えた外部3Gモジュール（+ 35ドル）を注文することができます。





組み込みのGPSモジュールが必要ですか？ 問題ありません。 このようなモデルの費用は180ドル（および基本価格に対して25ドル）です。



エネルギー消費に関しては、内蔵の2000 mA充電式バッテリーが搭載されており、最大負荷で6時間デバイスのパフォーマンスを保証します。





その結果、GPSと3Gモジュールを備えた最上位モデルの費用はたった240ドル（送料と保険料を含む）になります。 ちなみに、配送は世界中で行われています（現在220か国のリストにあります）。
