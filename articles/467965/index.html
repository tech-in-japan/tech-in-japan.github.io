Vivaldi 2.8 - Menu, please





Hello!



Today's final release is not quite ordinary. Due to the fact that recently we were preparing to release the first mobile beta version of the Vivaldi browser , work on the version for desktop computers was carried out in the polishing mode of existing functions and fixing errors. However, we have something to please users with, so we hasten to go on to describe the innovations of the new version. The list of major new products is as follows:



  • Keyboard navigation in the bookmark bar
  • Adjustable column width in bookmark manager
  • Support horizontal (classic) menu navigation
  • Toggle display of images using a key combination


Now let's take a closer look at these changes - there is something to tell.



Bookmark Buttons



The bookmarks bar is one of the most frequently used, but so far it has had a slight inconvenience - users could not fully use the keyboard navigation on the elements of this panel. It seems like a trifle, but for people who are used to doing miracles with just one keyboard, this inconvenience was very annoying. We are in a hurry to correct and inform that now it is possible to navigate through all elements of the panel only with the arrow buttons, without touching the mouse.







Here is such a width



Another innovation also adds convenience to the browser. Now users can change the width of the data columns in the bookmark manager themselves, and, so as not to get up twice, in the history viewer too. It is enough to place the cursor on the border between the columns in the title bar and drag it in any direction.







Return to the classics



The history of this option is rooted in the distant 2014. And this is not a typo - this request was born in the bowels of the company even before the first public test version of the browser was published. This cannot be said that this was one of the first requests, but the four-digit number of this bug report (VB-2238) leads to nostalgic memories of how it all began. Then, of course, there were other similar requests from users, and finally, we were able to fix this annoying defect: now you can navigate the horizontal classic menu using the keyboard and, which was much more in demand, by moving the mouse cursor over menu items :







Playing chords



This option has been asking for a long time. Well, as it was, she was in the still old tube Opera and users were perplexed: why was she not in Vivaldi? We are talking about the possibility of switching the display of images using a combination of keyboard shortcuts - so as not to drag the mouse to the corresponding button on the bottom of the browser. In general, it was somehow unusual.



Well, now it should become familiar - we added this option to the list of keyboard shortcuts. Abrasive treatment



And, of course, as in all previous versions, we worked very actively with the file and removed the roughness on the numerous functions of Vivaldi.
