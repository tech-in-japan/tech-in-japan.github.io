マイクロソフトはWindows用の一連の更新プログラムをリリースしました

Microsoftは、パフォーマンスを向上させ、Windowsの安定性を向上させる一連のオプション（非セキュリティ）更新プログラムをリリースしました。 17の更新プログラムがリリースされましたが、そのうちKB3064209とKB3058168が最も興味深いものです。 Windows 7-8.1のKB3064209 （ WindowsのIntelプロセッサー向けの2015年6月のマイクロコード更新 ）は、初期の段階（Hvax64.exe、Hvix64.exe、Hvloader.efi、Hvloader.exe、 Hvservice.sys、Kdhvcom.dll、Mcupdate_genuineintel.dll）。







別の更新KB3058168 （ Windows 8.1およびWindows 8 KMSホストが新しいバージョンのWindowsをアクティブ化できる更新 ）は、Windows 8のその後のWindows 10へのアップグレードのための別の改善であり、Cryptxml.dllファイルの新しいバージョンと構成ファイルの更新バージョンを提供しますKMS （キー管理サービス）を使用してWindows 10を正常にライセンス認証するために必要です。







KB3064209



この更新により、特定のIntelプロセッサを使用するシステムの信頼性が向上します。 この更新プログラムは、オペレーティングシステムが正しく機能しない原因となる問題も修正します。


KB3058168



この更新プログラムは、Windows 8.1、Windows Server 2012 R2、Windows 8、およびWindows Server 2012のKMSを拡張し、Windowsベースのクライアントの新しいバージョンのライセンス認証を可能にします。


これらの更新プログラムをインストールすることをお勧めします。



以前に、重要な5月の更新KB3035583についても書きました。これは、Windows 7およびWindows 8 / 8.1ユーザーにGWX（Get Windows 10）ツールを提供し、ユーザーはWindowsのコピーをWindows 10にアップグレードできます。 Windowsの新しいバージョンのコピーを取得して更新するプロセスは段階的に廃止されます。
