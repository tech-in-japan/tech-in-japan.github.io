新しい「チップ」Far Manager v2.0







私は、多くのファイルマネージャーが日常業務でFarファイルマネージャーを使用していると思います。 そして、ほとんどすべての人がすでに知っていることです。今ではすべての人が完全に無料であり、ソースコードで利用でき、 x86およびx64プラットフォームでも利用できます 。 しかし、誰もが絶えず進化しており、新しい「バン」が実装されていることを認識しているわけではありません。



エンパワーメント

以前は、アクセスできなかったフォルダーにアクセスしようとすると、「アクセスが拒否されました」というメッセージが表示されていました。

現在、Far Mangerは賢明であり、UACメカニズムを介して独立して管理権限要求を実行しています。



ファイルをコピーする

以前のバージョンでは、パネルでファイルを選択してクリップボードにコピーすると（Ctrl + Ins）、ファイル名のみがコピーされていました。

現在のバージョン（v2.0ビルド1666）-ファイル名に加えて、これらのファイルへのリンクもコピーされるため、エクスプローラー（デスクトップなど）またはそのようなファイル処理をサポートする他のプログラム（例： DVD書き込みソフトウェア）。 ファイルへのフルパスのコピー（Ctrl + Alt + Ins）は変更されていません。



新しい「おいしいプラグイン」があるかどうかは見ませんでした-仕事のために、7-zip、PicView Advanced、Coloreeで作業するのに十分なプラグインがあります。

非常においしい「グッズ」を知っている場合は、コメントで共有してください。
