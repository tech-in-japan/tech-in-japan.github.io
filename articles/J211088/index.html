首都艦隊間の主な戦いはB-R5RBで行われたが、CFCはシステムへのアプローチを阻止するために急いだ。 サブキャピタル艦隊は、他の主要なN3PLシステムであるI-NGI8およびGXK-7Fに向かい、増援を妨害しました。 スーパーキャピタルジャンプドライブが戦場から逃げるのを防ぐために必要なインターディクターとヘビーインターディクタークラスの船は、戦闘中のドミニックス船からのCFC艦隊によって破壊されたため、これはRUS / CFCを大いに助けました。 さらに、戦闘を他のシステムに部分的に転送することで、Eve Onlineサーバーの負荷を軽減できました。



CFC / RUSが優位を獲得



UTCの午後10時頃、東海岸のアメリカ人がゲームに参加し始め、戦闘に参加し始めました。 これは、CFCがN3PLよりもこのエリアに多くのプレイヤーを抱えていたため、戦いの転機となりました。 戦闘は常に数値の優位性によって勝つわけではありませんが、通常はこれが起こります。 数値的な優位性を獲得したRUS / CFCの資本艦隊は、損失の交換を停止し、失うよりも多くの船を破壊し始めました。 UTC 22:00で、失われたタイタンの数はN3PLで18、RUS / CFCで7でした。 UTC 23:00までに、CFC / RUSの首都艦隊はさらに32人の敵のタイタンを破壊し、この間に5人だけを失いました。



1月28日のUTC 3:20に、N3PL司令官は撤退命令を発行しました。 現在の目標は、必要に応じて通常の資本を寄付し、スーパーキャピタルを置き換えるのが可能な限り高価で困難なものを節約することでした。 また、その瞬間、彼らはRUS / CFCスーパーキャピタルをもはや破壊できないことに気づきました。なぜなら、修理される前に船を破壊するために同時に終末兵器を使用するのに必要なタイタンの臨界質量を持たないためです。 彼らはドレッドノートに切り替えて、できるだけ多くの敵船を墓に運ぼうとした。 その日最後に破壊されたRUS / CFCタイタンは、Circle-Of-TwoアライアンスのChango Atacama船でした。



スパイのおかげで（そしてスパイ活動はEVE Onlineメタゲームの大きな部分です）、RUS / CFC司令官はN3PLが撤退しようとしていることを知り、N3PL船が戦場から逃げるのを防ぐためにパイロットにインターディクターとヘビーインターディクタークラスの船に移るよう命令しました。 これらの小型船を簡単に破壊できる唯一の首都はかなり弱いケリアだったので、プレイヤー数と準資本船の数におけるCFCの優位性が重要な役割を果たしました。 ドレッドノートとタイタンの巨大な砲塔は小型で機敏なボートに乗ることが困難です。戦艦USSアイオワが16インチの銃から高速ボートに乗ろうとしていることを想像してください。 ドローン爆撃機の戦闘機にも同じ問題があります。 N3PLがこれらの船に対処する唯一のオプションは、同程度のサイズのドローンを制御できるケリアを使用することでした。







しかし、彼らは十分ではありませんでした。 残りのN3PL資本とスーパーキャピタルの救済は、EVE Onlineサーバーが毎日の定期メンテナンスのためにシャットダウンした1月28日の11:00 UTCにのみ発生します。 この時点で、すべてのプレイヤーはサーバーから強制的に切断され、彼らの船は消えます。 サーバーの電源を入れた後、パイロットが再びログインしたくない場合、彼の船はゲームに表示されず、他のプレイヤーには見えません。 そのため、戦闘は操業停止まで続きました。 その日最後に失われたスーパーキャピタルは、カデシのクレイダが所有していたNyxクラスの船で、10時40分UTCに破壊されました。 CFC / RUS艦隊は、誰もがサーバーから切断された最後の瞬間まで敵の首都を殺し続けました。 予想通り、N3PLは、EVEの存在が突然中断された中で最も費用のかかる戦闘の後、再度ログインしないようにパイロットに命令しました。



損失：大きさと結果



CCPはブログにバトルポストを投稿しました。 数字は単に驚くべきものです。 75のタイタンが破壊され、そのうちCFC / RUSは16、N3PL 59を失いました。これに加えて、13のスーパーケリー、370のread級戦艦、123のケリエールが殺されました。 CCPの当局者は、損失は11兆ISKに達すると言いました。 それに比べて、以前の最も高価な戦いは聖バレンタインデーの虐殺、または12のタイタンが破壊され、合計損失が1兆に達したUemonの戦いでした。 Uemonの戦いは、長年Zero EVE Onlineの北を保持していたグループである、現在は機能していないNorthern Coalitionに対するRUS / PL戦争の転換点でした。



本物のお金に変換されるこれらの損失の周りに、今ニュースで多くの誇大宣伝があります。 引用された金額は正しいものと間違ったものの両方です。 CCPは、ゲーム内市場でISKのために販売できるプレイ時間を支払うためにカードを販売しています。 現在の価格では、 1ドルは3500万ISKに相当します。 11兆ISKを獲得するには、カードを30万から30万ドルで購入し、国内市場で販売する必要があります。 しかし、ウォール街で大量の株を売るのと同じように、非常に多くのカードを一度だけ売ると市場が台無しになります。



紛失した船は、適切な量のISKを使用しても、簡単に回復できません。 もちろん、多くのアライアンスは、非アクティブなアカウントに予備資本と超資本を持っています。しかし、破壊されたタイタンを新しいものに交換するには1年かかる可能性があります。これは、同盟が戦争中に脆弱なキャピタルシップアセンブリアレイを使用するリスクを回避する可能性が高いためです。そのような圧倒的な敗北の後、N3PLが残りのタイタンに対してさらに注意を払い続けることは明らかです。



N3PLのもう1つの問題は、B-R5RBのステーションに残っているアクセス不能なリソースで、推定2,000億ISKです。。これは同盟の財産であるだけでなく、プレイヤーの持ち物でもあります。もちろん、Pandemic Legionはこれらのリソースをそこから取得しようとします。もちろん、これを行う最も「直接的な」方法は、システムをキャプチャすることです。遠回りの方法は、スパイを使用して同盟に侵入し、ステーションにドッキングし、そこからすべてを静かに取り出す権利を与えることです。いずれにせよ、パンデミック軍団の兵responsibleを担当するプレイヤーは、燃料、弾薬、およびモジュールを入手するために、新しい船の建造を組織するために一生懸命働く必要があります。



このような戦いの結果を予測することは非常に困難です。EVEオンラインの主権戦争のルールの1つは、道徳が最初に死ぬことです。EVE Onlineでの戦争は意志力のテストです。意志が破られると、主権を掌握するさらなるプロセスはかなり冷静になります。CFC / RUSはすでに他のシステムをキャプチャして、プレーヤーに注文を送信することを計画しています。N3PLがこれらの操作に耐えられない場合、それは道徳が抑圧され、戦争がすぐに終了することを意味します。CFC / RUSが他のシステムを奪取しようとする試みが失敗した場合、位置的戦争は継続します。いずれにせよ、B-R5RBの戦いは史上最大かつ最も高価な仮想の戦いとして記録簿に記録されます。
