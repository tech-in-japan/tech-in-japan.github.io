コンピュータ図像学の簡単な歴史





前世紀の70年代以来、最も一般的なコンピューターアイコンは長く困難な道を歩んできました。 この記事は、コンピューターの図像学の発展を示すアイコンのコレクションをまとめる試みです。 1981年から2010年まで、グラフィカルインターフェイスを備えたさまざまなオペレーティングシステムがありましたが、最も重要なものだけが選択され、現代のアイコンデザインに最も大きな影響を与えました。





1981年。ゼロックス8010スター-グラフィカルユーザーインターフェイスを備えた一般顧客向けの最初のコンピューター



1973年、 ゼロックスアルトは、グラフィカルユーザーインターフェイス（GUI）を備えた世界初のコンピューターになりました。 ちなみに、「オフィス」のメタファーは、世界で初めてアイコンのデザインに使用されました。 アルトは科学研究用に作成されたため、幅広い市場にはヒットしませんでした。 合計で約2,000台のこのようなマシンが製造され、その後、ゼロックスアルトの成功により、アップルはLisaコンピューターを作成するようになりました（1983年）。 1981年、Xerox Starがリリースされ、Altoのアイデアを開発し、彼の祖先のすべての革新と発見を取り入れました。 Xeroxのアイコンは、馴染みのあるものを介して、コンピューターインターフェースと人間の相互作用の原理を示しています。 ご覧ください。計算機、ドキュメント、フォルダー、ゴミ箱のアイコンは、過去30年にわたってまったく変わっていません。





1981-ゼロックス8010スター




1983年。AppleLisa-GUIの普及



Appleは、ゼロックスの初期のコンピューターの強い影響を受けて、1978年にLisaコンピューターの開発を開始しました。 アップルは、パソコン市場でニッチを占めることを望んで、コンピューターと初心者のコミュニケーションをできるだけ簡単にするために「オフィス」の比phorを採用しました。 その間、Lisaには非常に高度なグラフィカルインターフェイスがありました。「デスクトップアクセサリ」（そして実際にはウィジェット）、ドロップダウンメニュー、およびフォルダ形式のディレクトリがありました。 ご覧のとおり、Lisaアイコンは、サイズと1ピクセルのストローク、および設定パネルのコンピューターアイコンの使用を除いて、Xeroxコンピューターアイコンとそれほど違いはありません（これらの目的にはギアを使用するのが一般的です）。





1983-リサオフィスシステム1




1984年。AppleMacintosh 1.0-アーティストが作成したアイコン



Lisaのリリースから1年後、 Apple Macintosh 1.0が誕生しました。 これからは、ドラッグアンドドロップを使用してファイルをコピーし、ウィンドウを移動できます。さらに、新しい驚くべきアイコンが登場しました。 今回、アイコンの作成は、今では伝説的なスーザン・カレが主導しました。 スーザンは、MacPaintインターフェースのアイコンを含む、Apple用の多くのアイコンを作成しました（図2）。 Kerの哲学はシンプルです。「良いアイコンは、通常の絵よりも道路標識にはるかに近く、理想的には、できるだけシンプルで簡潔で覚えやすいものにするべきです。 今日のコンピューターにはより多くのグラフィック機能が登場しているという事実にもかかわらず、私はアイコンをできるだけ明確かつシンプルにしようとしています。」 この哲学は、その年のAppleの商業的成功の基礎となりました。

1985年。アタリTOS-等尺性のアイコン



これらの年に、Appleコンピューターだけでなくグラフィカルインターフェイスがあったことに注意することが重要です。 Atari STには独自のTOSオペレーティングシステムがあり、それは同じデスクトップメタファーを使用して最小限のインターフェイスを持ち、それまでにコンピューター標準になりました。 ちなみに、一部のTOSアイコンは等尺性で描かれています。





1985-Atari TOSバージョン1.0




1985年。AmigaWorkbench-4色のアイコン



Amiga Workbenchは、 Amiga 500 PC用に作成されました。 そして、粗雑に描かれたアイコンにも関わらず、その間は非常に進歩的でした。 その中のカーソルの外観を変更することが可能で、アイコンは4色であり、状態を視覚的に変更する機能もありました。 Amigaはデスクトップの哲学を変え、環境内のフォルダーアイコンの代わりに引き出しのある職場を使用しました。





1985-Amiga Workbench 1.0




1985年。Windows1.0x-Microsoftの最初のGUI OS



1985年、Microsoftはついに最初のグラフィカルユーザーインターフェイスをリリースすることに成功しました。 アイコンは、Amigaのアイコンと同じくらい粗雑に見え、それらも白黒でした。 興味深いことに、最初のWindows PainterアイコンはMacPaintからいくつかのキャラクターを直接借用しています。特に、Spray Painterアイコンは印象的です。





1985-Windows 1.0x




1986年。コモドール64のGEOS-代替OS



Commodore 64のGEOSをリストに追加する価値があります。当時、 このグラフィカルインターフェイスはMacintosh 1.0の次に売れた数でした（販売台数で）。 アイコンはマイクロソフトのアイコンよりもはるかに表現力豊かでしたが、Macintoshの哲学-単純な比phorの哲学を公言しました。





1986-コモドールC64 GEOS




1991年。Macintoshシステム7-最初の色のMac OS



システム7では、アイコンが色で友達になりました。 さらに、「クリック可能性」を高めるためにわずかに成長しました。





1991-Macintoshシステム7




1992年。Windows3.1-新しいデザインアイコン！



Windows 3.0（1990）では、Microsoftは既に知っているスーザンケア（以前にMacintosh 1.0用のアイコンを作成していた）のアイコンを使用しました。 Windows 3.1では、スーザンはアイコンの色と外観をさらに強化しました。 さらに、Windows 3.1はTrue Typeフォントがプリインストールされた最初のMicrosoftプラットフォームでした。





1990-Windows 3




1995年。Windows95-スタートボタン



Windows 95では、アイコンをよりカラフルにすることができました。 さらに、さらにいくつかのアイソメオブジェクトが表示されています。 Windows 95インターフェイスの設計は根本的に再設計され、今日までWindowsに存在していた要素の一部が含まれています。 これがタスクバー、メニュー、そして町の話題になった「スタート」ボタンです。





1995-Windows 95




1997年。MacintoshOS 8-明るいアイコン



Mac OS 8では、アイコンはずっと明るく見え、光源の方向がはっきりと見えます。 Macintoshは、シャドウ効果のあるアイソメ図スタイルを実装し始めました。





1997-Macintosh OS 8




2001 Mac OS X v10.0-Macスタイル



Mac OS Xのリリースは、完璧に見える光沢のあるプラスチックゼリーアイコンの外観によって特徴付けられました。 OS Xのアイコンは、2年前にリリースされたOS 9のアイコンと比較して大きな進歩を遂げました（そのアイコンはOS 8のアイコンとほとんど同じように見えました）。 おそらくドックの外観のため、すべてのアイコンは、あたかも光源がその真正面にあるか、少し上にあるかのように描かれました。 新しいAquaグラフィックテーマの不可欠な部分として作成されたアイコンには、複雑な反射、ハイライト、テクスチャがありました。 現在のコンピューターの図像は非常に多様で興味深いので、その調子を整えたのはアクアだったと確信しています。





2001-Mac OS X v10.0




2001年。WindowsXP-明るく落ち着いたアイコン



2001年に、Microsoftは新しいオペレーティングシステムを一般に紹介しました。 ただし、豊富なカラーパレットを使用すると、アイコンは穏やかに見えます。 1つの光源と、半透明の落下する影があります。 スタイルは引き続きアイソメを使用します。





2001-Windows XP




2007年。MacOS X Leopard-Reflection Dock



Macはストライプを投げ、そこにあるアイコンを反映して3Dスタイルのドックを誕生させました。 装飾でのクロム、ガラス、反射の使用は、これまで以上に人気があります。 OS X以降、アイコン自体はあまり変更されていません。





2007-Mac OS X Leopard




2009年。Windows7 —落ち着いた落ち着き



Windows 7のアイコンはWindows XPのアイコンと根本的に異なり、Vistaのアイコンを継承します。 主な違いは、Windows 7およびVistaのアイコンが、Windows XPのアイコンのように右ではなく左に見えることです。 7つのアイコンでは、反射効果が明確に存在しますが、さらに穏やかで柔らかくなります。
