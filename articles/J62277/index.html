Microsoft Small Basic 0.5リリース





変更点の一部を次に示します。



  • 初期化されていない変数：

    コンパイラは、使用されているが初期化されていないコード内の変数を追跡およびレポートできるようになりました。



  • スペイン語のローカライズが追加されました



  • 開発環境の更新：



    * Ctrl + Fで呼び出される検索機能を追加

    *現在の行と行内の位置の表示を追加

    *「名前を付けて保存...」コマンドを追加

    *プログラムコードをフォーマットする機能が追加されました（コンテキストメニューからエディターで右クリックして呼び出されます）



  • そしてまた：

    *多くのバグ修正

    *より小さなインストールファイルサイズ（3.7MB）

    *インストーラーは、.Net framework 3.5 SP1の存在を自動的に検出するようになりました




リリースはここからダウンロードできます。
