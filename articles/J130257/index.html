GDDモスクワ2011でのGTUGプロジェクトの概要

みなさんこんにちは！



先週の月曜日、Google Developer Day Moscow 2011が開催され、最新の開発技術に関するさまざまな興味深いレポートに加えて、イベントの別のセクションがGTUGチームプロジェクトに専念しました。



予想外に、GoogleはAmperkaを招待してこのイベントに参加し、いくつかのGTUGプロジェクトのサポートを提供しました。 あなたはカットの下で最後に何が起こったかについて読むことができます。



知らない人のために： GTUG （Google Technology User User Group）は、世界のさまざまな都市の（言葉の意味では）ハッカーの協会です。 Googleはそのようなチームに情報やその他のサポートを提供し、それらはGoogleテクノロジーを使用して創造的で革新的なプロジェクトを実装します。



今年、多くのAndroidデバイスとArduino ADKは 、ユニークで面白いガジェットを手に入れるために、他の何かで特に人気がありました。



以下でプロジェクトについて簡単に説明します。



ロボワーズ



Robowarsプロジェクトは、ウファのチーム（著者-Gilyazitdinov Rafael irafa ）によって提示されています。 2つの二輪ロボットとAndroidタブレットの楽しい屋外ゲームでした。











各ロボットにはハンマーと大きなボタンがあります-弱点です。 2人のプレイヤーがプレイします。それぞれが自分のロボット用です。 ゲームの目的は、ロボットの相手をハンマーのボタンで打ち負かすことであり、自分に代わるものではありません。 4ポイントを獲得した最初のもの、それ-よくやった。



ロボット自体は、特別に作成されたクライアントを介してAndroidデバイスから制御されます。 分割画面モードのプレイヤーには、仮想マッシュルームジョイスティックと「パンチ！」ボタンが与えられます。



ロボットとの通信は、 XBeeモジュールを介して行われます 。 概略的に、鉄は次のように構成されています。







このプロジェクトは理解可能で楽しいことがわかりました。高価なスーツを着た大人の叔父でさえ、即興の舞台で対戦相手を打ち負かしました。



GDD会員カウンター



このプロジェクトは、モスクワGTUGチームとAmperkaによって実装されました。 彼は準備のために全員の神経を丁寧にたたきました。 時間がないため、イベントの前夜に収集され、必死にテストされました。 会議が午前9時にドアを開けたという事実にもかかわらず、すべては午前11時まで「ほぼ稼働」状態のままでした。



しかし、努力は報われ、最終的に、ホールに入る前に、GTUGがセットアップされました。登録されたゲストの数を示すGoogle+サークルです。 いくつかのスライスされたLEDストリップからの工芸品、クロッカスの部屋の1つで盗まれた分解された延長コード、および食品容器内の少数の制御電子機器については、それはまともなように見えました。 設置時には、数字の正確な意味がまだ理解されていませんが、多くのゲストは急いでデバイスを背景に写真を撮りました。







訪問者の数に関するデータ、インジケーターは近くの受付からHTTPを引き継ぎました。 概略的に、デバイスは次のようになります。







実際、後で気付いたように、ほとんどのゲストはこの数字の目的を理解していませんでした。多くのゲストはカウンターを増やすことを望んで頑固に跳ね返りました。 これについては以前考えていたかもしれませんが、過去のゲストのカウンターをキログラムで作成したかもしれません。 もっと楽しいだろう。



アンドルイド音楽スタジオ



このプロジェクトは、アルマトイのチーム（リーダー-ジャムシェッドナスリジノフ）によって提示されました。 一番下の行は次のとおりです。上のフレームにはレーザーがあり、下のフレームには応答フォトレジスタがあります。 8本のレーザーストリングを形成します。 フレームはAndroidスマートフォンに接続され、各ストリングのサウンドサンプルが記録されます。







ビームが遮断されると、電話機は対応するサウンドを再生します。 このような異常な方法で、作曲ができます。 デモンストレーション用に、音のバンクを1つだけ準備しました。1番目の文字列-「1」という単語、2番目の文字列-「2」という単語などです。 それでも、ゲストの一部は面白い「音楽作品」を抽出しました。



ルーター用のAndroidインターフェイス



ウクライナのキエフGTUGセンターチームは、ネットワークルーターの構成を簡素化するプロジェクトを発表しました。 おそらく、それは最も壮観ではありませんでしたが、同時に最も有用で応用されたGTUGプロジェクトでもありました。







一番下の行は、インターネットサービスプロバイダーの技術者がlaptop屈な地下室でラップトップを持ち運ぶのが難しいということです。 そして、どういうわけか、ルーターを再構成する必要があります。 プロジェクトの作成者は、AndroidとArduino ADKに基づくソリューションを提示しました。これにより、ラップトップをシンプルなタッチインターフェースを備えたコンパクトなボックスに置き換えることができます。



オーディオビジュアライザーADK Led Matrix



ウクライナからの別のプロジェクト（著者-Alexey Zhurbitsky blo ）は、オーディオスペクトルビジュアライザーです。 Androidはオーディオトラックを再生し、オープンアクセサリモードのArduinoは大きなLEDマトリックスを制御します。 すべて一緒に-シンプルだがスタイリッシュな視聴覚インスタレーション











タンチキ



また、ウクライナの別のチーム、ドネツク（著者-Roman Bogdzievich romeo_ordos ）は、Arduino ADKとAndroidタブレットをベースにしたプロジェクトゲームを発表しました。 その中で、AndroidはArduinoによって制御され、その逆も同様です。Arduinoを介した自家製のジョイスティックからの信号はタブレットに送信されます。



ジョイスティックの1つの「マッシュルーム」は毛虫を、もう1つはタワーを担当します。 1つのボタンは火のために取っておきます。



タスクは簡単です-相手を撃つ。
