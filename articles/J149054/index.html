Technetサブスクリプションのトリミング

残念ながら、7月16日以降、Technet StandardおよびTechnet Professionalのサブスクリプションはその価値を大幅に失いました。 そのような行為にinしている人々がまだ見つかっていないのは奇妙です。 以下は、Microsoftが深く隠された広告に書いたものです。

Microsoftは2012年7月にTechNetサブスクリプションにどのような変更を加えましたか？

7月16日、MicrosoftはTechNetサブスクリプションに変更を加えてプログラムの目的をよりよく反映し、ITプロフェッショナルが現在のMicrosoftソフトウェアを評価できるようにし、サポートされていない使用からサブスクリプションの整合性を保護しました。 この変更の一環として、次のことを実行しました。

1）ユーザー契約の量と複雑さを軽減しました。

2）Microsoftソフトウェアおよびその他のサブスクリプション特典を12か月のサブスクリプション期間のみ利用可能にしました。

3）ダウンロード可能な製品のリストを次のように更新しました。

-不要な重複製品インスタンスを削除しました。 たとえば、Outlook、Publisherなどの個々のOffice製品ではなく、Officeパッケージ全体をダウンロードできます。

-ITプロフェッショナルが管理する作業環境での使用を意図していない製品を削除しました。 例：Windows XP Home Edition。

-マイクロソフト製品サポートライフサイクルを通じて延長サポートの対象外となった製品を削除しました。

4）すべてのプログラムの24時間でのプロダクトキーの受け取りに関する標準化された制限、および最大10までの特典レベル。

メール通知なし 今後の変更に関するメールを受信しましたが、これは非常にugいものです。



実際、上記のように、サブスクライバーが使用できるキーのリストから、個々のOffice製品（Access 2007-2010、Outlook 2007-2010など）のほとんどすべてが消えました。 Windows 7からは、Windows 7 Professionalのみが残りましたが、Ultimateバージョンには企業クライアント向けの多くの機能が含まれています。 Windows XPは、Professionalエディションのみに留まりました。 現在、これらの製品は利用可能なキーのリストにハングアップし、0から0個のキーが利用可能であることに注意してください。



一般的に、膨大な数のキーが消えました。 製品はまだダウンロードできますが、これは一時的なものだと思います。 しかし、最も不快なのは、サブスクリプション価格が同じままであることです！



これまでのところ、問題は、変更が行われる前に要求された除外された製品のキーに残っています。 サポートリクエストが送信されましたが、まだ応答がありません。



更新されたTechnetサブスクリプション条件の下で利用可能な製品の最新リストは、 ここからダウンロードできます 。
