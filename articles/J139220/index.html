WirelessHDとWiGigの関係について


2007年に「タンデム」という言葉が登場する前に、高品質（HD）信号をテレビに送信するニーズの高まりに応えるためにWirelessHDが登場しました（ieee802.15.3に基づく）。 しかし、2009年に最大のメーカーがカブに傷を付けて、WirelessGigabitアライアンスを作成しました。Wi-Fiの速度がすぐに小さくなると正しく想定していました。 実際、ieee802.11ac / adはWiGigの下で作成されました（ wiNIKAの詳細記事を参照 ）。これにより、古き良き802.11a / b / g / nを2.4および5 GHz（802.11ac）に「スイング」し、マルチギガバイトと組み合わせることができます。 1つの標準のフレームワークで60 GHz（802.11ad）の怪物。

WiGig がエクスタシーに合併した後のWirelessHD が WiGigに参加したことは驚くことではありません。 したがって、式を単純なWi-Fi + WirelessHD = WiGigに簡略化します。



WirelessHD規格はウルトラワイドバンドブロードキャストに基づいて構築されており、ゼロおよび1を最大7 Gb / sの速度で送信できます（また、圧縮と遅延なしでFullHD 3D 7.1を駆動できます）。 このような幅広のストリップは、壁に「ドラッグ」できません。 しかし、彼女は家具、人、動物などの小さな障害にうまく対処しています。 どのテクノロジーを使用すると、「知的に曲がる」ことができます。



羊の話に戻りますが、残念ながら最近、WirelessHDのビデオ送信者についてはほとんど聞いていません。 彼らは一定のレベルに達しました-FullHD 1080p @ 60Hz、3Dおよび7.1音、そしてこれらすべては圧縮と遅延なしで、壁を介して送信できないことと送信機のかさばり（内蔵アンテナにもかかわらず少なくとも6）は十分な需要を与えていないようですビデオ送信者の鍵となる技術の開発について。

WHDI





次の顧客は、伝統的に、WHDIです。 イスラエルの企業Amimonによって作成された、HD伝送用のオリジナルファームウェアを備えた強力なMIPSチップに基づいています。 802.11a / nに似た原理に従って5Ghzの周波数で動作しますが、はるかに高速-3 Gb / sのワイヤレスチャネルです。 5Ghzでゆっくりと接続するWi-Fiと競合しないように、 MIMO + OFDMテクノロジーとタイムスロットが使用されます。これにより、互いのWHDIとWi-Fiに干渉しないようにできます。



WHDIの現在のバージョンでは、壁を通して3Dおよびマルチチャンネルサウンド（HDMI 1.4をシミュレート）でFullHD 1080p @ 60Hzを転送でき、伝送遅延は1ミリ秒（LCDモニターよりも短い）で、赤外線ポートとUSB-HIDを転送できます。

最新の呪文とはどういう意味ですか？ 赤外線ポートを転送すると、リモートから信号ソースに信号を送信できます。 大まかに言って、レシーバーをテレビに接続し、別の部屋にトランスミッターが接続されている衛星放送受信機があります。 衛星受信機から受信機にリモートコントロールをクリックすると、リモートコントロールが送信機の別の部屋に転送され、送信機から赤外線アイが衛星受信機を見て制御します。 元のIR信号は、ビデオ送信側チャネルを介して送信されます。

USB-HIDはIR転送とほぼ同じであり、レシーバーに接続されたキーボードとマウス（HIDデバイス）を使用してこのPCを制御するために、追加の（HDMIに加えて）USBケーブルでトランスミッターをコンピューターに接続できます。 大ざっぱに言えば、テレビにはレシーバーがあり、キーボードとマウスが接続されています。コンピューターからの信号を見たり聞いたりして、それを制御することができます。



WHDIは圧縮を使用しませんが、特別な特許技術であるVideo Modemがあり、強力な干渉と大幅な帯域幅カットの状態でも信号を送信できます。 ここでロシア語の古くて良い記事について読むことができます 。



現在、2つのWHDI設計があります。



完全 - 本格的 -送信機と受信機はルーターのサイズですが、アンテナなしで5〜6個あり、それらは組み込まれています。通信は主要な障害を克服して最大30メートルです。



コンパクト（いわゆるスティック） 、2〜3個の内蔵アンテナ、トランスミッターはUSBフラッシュドライブよりもわずかに大きく、レシーバーはコンピューターのマウスよりもわずかに大きく、両方とも壁のコンセントからだけでなく、USBポートからも電力を受け取ることができます。 資本以外の障害を克服して10〜15メートルまで送信します。



今日、WHDIは最も有望な規格の1つであり、Asus、HP、Belkin、PowerColor、KFA2などの大手メーカーが確認し、自社ブランドでWHDIキットを製造しています。 また、既にWHDIモジュールが組み込まれたテレビを導入しているLG（および業界の他の多くのモンスターも集まっています）や、WHDIモジュールが組み込まれた7インチのAndroidタブレットをリリースしたLenovoも注目に値します。 CES2012でのWHDIの更新とタブレットのデモを含むビデオは、 ここで見ることができます 。



WHDIは、消費電力の削減、1つのトランスミッターから複数のレシーバーへの同時ブロードキャストのサポート、4K x 2K解像度のサポート、データ転送用のワイヤレスチャネル、WHDIとWi-Fiを1つのチップに統合する予定です。



長所


  • それぞれかなり高速のワイヤレスチャネルでは、圧縮は使用されず、遅延はありません（1ミリ秒は無視できます）。
  • 最高品質のHD品質（生まれたばかりの4K x 2Kを除く）およびIRおよびUSB転送機能をサポートします。
  • 壁を克服し、コンパクトなデザインを使用する能力。
  • 1つの受信機で複数の送信機を使用する機能（代替）。


欠点


  • 5Ghzの周波数で動作し、極端なスペクトルオーバーフロー中に送信を妨害する可能性があります。
  • 組み込みソリューションとモバイルデバイスには、コンボチップをリリースするまで、Wi-Fiに加えて、別個のWHDIモジュールが必要です。

HDビデオ送信者でのWi-Fiの使用の機能：

  • Wi-Fiには300〜400メガビット（もちろん802.11n）の帯域幅制限があります。
