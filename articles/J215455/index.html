サムスン初のUHDモニターU28D590がすでに販売されています

こんにちは、Habr！



サムスン電子は、28インチの対角線を備えた製品ラインUHDモニターで最初のU28D590のロシアでの販売開始を発表しました。 新しいアイテムの主な特徴は、4K画面解像度（3840x2160）です。







モニターが10億色を超える色を表示できるため、スムーズな色遷移による自然な色再現が実現します。 あらゆる種類のメディアプレーヤー、ゲームコンソール、PC、またはラップトップを接続するために、2つのHDMIポートと1つのディスプレイポート（DP）インターフェイスコネクタがあります。 3840 x 2160ピクセルの解像度での最大フレームレートは、DisplayPort経由で接続した場合は60 Hz、HDMI経由で30 Hzになります。 2台のPCを一度にモニターに接続し、品質を損なうことなく両方のデバイスのコンテンツを同時に再生できます。各PCの画像解像度は1920 x 2160です。



間違いなく、新しいサムスンのディスプレイは、労働者のゲーマーにとって望ましいオブジェクトになります。マトリックスの応答時間は1 ms、視野角は水平に170º、垂直に160º、最大輝度は300 cd / m2、静的コントラストは1000：1です。 特別なゲームモードモードは、モニターに表示されるコンテンツに合わせて、モニターの色パラメーターを自動的に直接調整および最適化します。







また、Samsung U28D590の便利な機能には、更新された「Picture in Picture」機能（PIP 2.0）の可用性が含まれます。 これにより、メイン画面がUHDで再生される間、元の解像度でFullHD（1920 x 1080）までの小さなウィンドウの画像を再生できます。 さらに、低解像度コンテンツを再生する場合、新しいUHDアップスケール機能により、鮮明度と画質が最高の4K解像度に向上します。



サムスンU28D590 UHDモニターは、昨日、サムスンブランドの全店舗で発売され、25,000ルーブルの希望小売価格で販売されています。



技術仕様：

対角線： 28インチ

解像度： 3840x2160（16：9）

マトリックス： TFT TN

バックライト： WLED

明るさ： 300 cd / m2

コントラスト比： 1000：1

応答時間： 1ミリ秒

視野角：水平方向に170°; 垂直160°

最大色数： 10億以上

インターフェイス： HDMI（2ポート）、DisplayPort、ステレオオーディオ

消費電力： 32 W

寸法と重量： 661 x 486 x 169 mm、5.65 kg
