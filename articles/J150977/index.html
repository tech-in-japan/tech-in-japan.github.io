IBM PureSystems コンピューティングの新時代





すでに書いたように、4月にIBMは、エンタープライズクラスのアプリケーションのインストールとメンテナンスでIBMの専門家によって蓄積された知識を使用する、展開の準備が整ったPureSystems統合ソフトウェアおよびハードウェアシステムを発表しました。







この記事では、37の研究所と17か国で4年間開発されてきたIBM PureSystemsについて詳しく説明します。

投資額：20億ドル。







ロシアのIBMおよびCISのハードウェアディレクターであるAndrei Filatovによると、ビジネス開発には、企業がITの効率的なデータセンター、内部クラウド、および新しいビジネスアプリケーションの使用の分野で新しいソリューションを実装する必要があります。 しかし、ほとんどの顧客向けのこのようなプロジェクトの実装は、いくつかの理由により困難です。



1）企業には十分な資金がない 予算の65％以上が既存のITインフラストラクチャの管理に費やされています

2）スタッフに必要な経験が不足しているため、新しいソリューションの展開と統合、構成、構成、スケーリング、および管理が増加します。





企業がビジネス問題を解決するためにITリソースを実装および使用するための最小限の労力で迅速かつ効率的に行えるように、IBMは組み込みの専門知識を備えた統合システムのIBM PureSystemsファミリーをリリースします。



IBM PureSystemsは、ファミリー全体の名前であり、現在2つのシステムで表されています。



•インフラストラクチャPureFlex Systemソリューション。組み込みの専門知識により、情報処理に必要なリソースの使用を検出および予測できます。

•PureApplication Systemプラットフォーム。統合された専門知識により、サードパーティのビジネスアプリケーションの展開と読み込みを最適化します。















PureFlex Systemsの中心には、最大14のコンピューティングノードをインストールし、以下を組み合わせることができる最新のシャーシがあります。



•Intel XeonまたはIBM Power *プロセッサを搭載したモジュラー2ソケットおよび4ソケットサーバー

•外部ドライブ仮想化を備えたIBM Storwize V7000ストレージシステムの異なるバージョン

•ネットワークスイッチ（10 / 40Gbイーサネット、FCoE、ファイバーチャネル、およびInifiniBandポートが利用可能）

•PCIExpress拡張モジュールとディスクシェルフ

•4つのオペレーティングシステムから選択：AIX、IBM i、Windows、RedHat、SUSE Linux

•4つの仮想化テクノロジー：Power VM、KVM、VMWare ESX、またはMicrosoft HyperV



* 5つの異なるコンピューティングノードのモデルをIBM Flex Systemにインストールできます。FlexSystem x240およびFlex System x220モデルには、Intel Xeonプロセッサ用の2つのソケットがあります。 Flex System p260には、Power7用の2つのソケットが付属しています。 Flex System p460には、IBM Power7プロセッサー用の4つのソケットがあります。 LinuxをOSとして使用する、Powerプロセッサーを搭載したp24Lモデルも利用できます。 各モデルには約10種類があり、コアの数、アダプターのタイプ、およびその他のパラメーターが異なります。



PureSystemsの機能の1つは、サーバー、ストレージシステム、ネットワークデバイス、および仮想化ツールを使用してITインフラストラクチャの管理効率を最大化するIBMハードウェアおよびソフトウェアシステムであるIBM Flex System Managerノードです。



IBM PureFlex Systemsには3つの基本構成があり、お客様の要件に基づいて簡単に拡張できます。



•PureFlex System Express-中小企業向けのシステム

•PureFlex System Standard-通常のエンタープライズアプリケーションをサポートするためのシステム

•PureFlex System Enterprise-大規模なDBMSと信頼性の高い構成のためのシステム





PureApplication System-PureFlex Systemに似ています。 優れた計算能力とストレージシステムが組み込まれています。 PureApplication Systemには、96〜608コア、1.5 TB〜9.7 TBのRAM、6.4 TBのソリッドステートドライブと48 TBのハードドライブをインストールできるいくつかの変更が加えられています。



PureApplication Systemを使用すると、独自のソフトウェアテンプレート、ミドルウェア、仮想システムリソースを使用して、ソフトウェアの展開と管理を高速化できます。







PureSystemsは、新しいソフトウェア機能「エキスパートモデル」に基づいています。 このアプローチは、技術的な知識と経験を、インターネット経由でダウンロードして再利用可能なソフトウェアパッケージに変換します。 「エキスパートモデル」は3つのカテゴリで提供されます。



•IBMパターン（IBMのモデル）

•ISVパターン（独立系ソフトウェアベンダーのモデル）

•顧客パターン



アプリケーションの展開を大幅に削減することに加えて（たとえば、展開に数日かかったアプリケーションをインストールして10分で作業できるようになりました）、「エキスパートモデル」はインフラストラクチャ全体と展開の管理をカスタマイズするための膨大な「テンプレート」セットを提供することで、オペレーターの作業を大幅に簡素化しますクラウド環境。



PureSystemsのもう1つの優れた機能は、サービスの信頼性、可用性、利便性を向上させるメカニズムの使用です。



•考えられる問題の原因を見つける

•障害が検出された場合にIBMサービスに自動的に連絡する

•システム内のすべての不規則性に関するデータを収集する



「新しいシステムにより、世界中の150を超える顧客がITメンテナンスコストを最大40〜50％削減できると同時に、データセンター領域を拡大することなく2倍のアプリケーションを処理できるようになりました。」



ロシアのIBM System XおよびCISのビジネス開発マネージャーであるKirill Tereshenko氏によると、すべてのハードウェアとソフトウェアは工場ですでに完全に統合され、最適化されています。 また、サーバーおよびストレージリソースの割り当てと分配のシステムは、顧客のアプリケーションによって決定されるワークロード（ピークを含む）に対して最適化されます。 IBM PureSystemsは、電源投入から数時間以内に使用する準備ができており、システム全体を管理するための単一のインターフェースを備えています。
