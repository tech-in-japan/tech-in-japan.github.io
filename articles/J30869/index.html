マイクロソフトは、1995年のようにクラブになってほしい

誰もが5年ほど話しているだけです。 Windows XPのリリースから5年、Windows Vistaの開発と推進に5年かかった。



ただし、Microsoftは、Windows 95の壮大なプレゼンテーションを思い出してほしいと思います。これは、Officeアプリケーションスイートの更新と一致し、レドモンドがこれまでに見た最大のお祝いで終わりました。 レドモンドの青い空の積雲がWindowsディスクの箱に不吉に似ていたこと（ビルゲイツ自身が休日に登場するように命じていたかのように）と、夜明けまでオペレーティングシステムを購入するためにコンピューターストアで並んでいる人々に関するストーリーをリリースしたことを覚えています。



Microsoftによると、Windows Vista、Office 2007、およびExchange Server 2007のリリースは同じレベルのイベントです。

そうですか？ この三位一体は、マイクロソフトが私たちに刺激を与えたいと思うほど壮大ですか？ 「オールインワン」とはできませんか？
