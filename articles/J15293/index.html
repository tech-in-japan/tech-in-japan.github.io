UbuntuにInternet Explorerをインストールする方法（Gutsy Gibbon、Feisty Fawn、Edgy Eft、Dapper Drake）

翻訳、オリジナルはこちら



この記事では、Ubuntu Feisty Fawn、Edgy Eft、およびDapper Drake（Gutsy Gibbonでもテスト済み）でInternet Explorerを3つの異なるバージョン（6.0、5.5、および5.0）でインストールする方法を示します。 これは、たとえば、Linuxに切り替えたが、Internet Explorerでサイトをテストする必要があるWebデザイナーにとって便利です。 また、Internet Explorerでのみ機能するサイトがまだあります。





また、これがあなたのために働くことを保証しません！



1導入ノート



IEs4Linuxというパッケージがあります。これにより、Linuxシステムに3つのバージョンのInternet Explorerを簡単にインストールできます。 この記事では、その使用方法を示します。



しかし、始める前に、最初にこのページを読む必要があります： http : //www.tatanka.com.br/ies4linux/page/Legal_notices IEs4LinuxはGPLソフトウェアですが、Internet Explorerはそうではなく、独自のライセンスを持つ独自のソフトウェアです。 LinuxシステムにInternet Explorerをインストールする場合は、有効なMicrosoft Windowsライセンスが必要です。そうでない場合、Internet Explorerのインストールは違法になります。



2 /etc/apt/sources.listを編集します



IEs4Linuxをインストールするには、/ etc / apt / sources.listを変更する必要があります。



2.1 Ubuntu Feisty Fawn



Ubuntu Feisty Fawnでは、 deb deb.archive.ubuntu.com/ubuntu feisty universeとdeb wine.budgetdedicated.com/apt edgy main linesが必要です（はい、edgyは正しいです。なぜなら、このリポジトリにはまだ共有がありません。 Feistyの場合、幸いなことにEdgyのパピテスもFeistyで動作します）/etc/apt/sources.listで：



sudo vi /etc/apt/sources.list



[...]

deb de.archive.ubuntu.com/ubuntu feisty universe

deb wine.budgetdedicated.com/apt edgy main

[...]



次に、コマンドを実行する必要があります。



sudo apt-get update



パッケージデータベースを更新します。



2.2 Ubuntu Edgy Eft



Ubuntu Edgy Eftでは、/ etc / apt / sources.listにdeb de.archive.ubuntu.com/ubuntu edgy universeとdeb wine.budgetdedicated.com/apt edgy mainの行があるはずです。



sudo vi /etc/apt/sources.list



[...]

deb de.archive.ubuntu.com/ubuntu edgy universe

deb wine.budgetdedicated.com/apt edgy main

[...]



演じる



sudo apt-get update



パッケージデータベースを更新します。



2.3 Ubuntu Dapper Drake



Ubuntu Dapper Drakeでは、deb de.archive.ubuntu.com/ubuntu dapper universeとdeb wine.budgetdedicated.com/apt dapper mainが/etc/apt/sources.listにあります。



sudo vi /etc/apt/sources.list



[...]

deb de.archive.ubuntu.com/ubuntu dapper universe

deb wine.budgetdedicated.com/apt dapper main

[...]



演じる



sudo apt-get update



パッケージデータベースを更新します。



3 wineとcabextractのインストール



次のようにwineとcabextractをインストールする必要があります：



sudo apt-get install wine cabextract

4 IEs4Linuxをインストールする



IEs4Linuxをインストールするには、これを行います（通常のユーザーから）：



cd〜

wget www.tatanka.com.br/ies4linux/downloads/ies4linux-latest.tar.gz

tar zxvf ies4linux-latest.tar.gz

cd ies4linux- *

./ies4linux
