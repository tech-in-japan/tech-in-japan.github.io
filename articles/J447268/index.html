Microsoft Azureセキュリティセンターが新機能を発表

企業をクラウドに移行することでより多くの組織がより速く革新するにつれて、セキュリティを改善することがすべての業界にとって重要になっています。 Azureには、データ、アプリケーション、コンピューティング、ネットワーク、識別、脅威保護のための組み込みのセキュリティコントロールがあり、セキュリティを構成してパートナーソリューションを統合できます。



セキュリティへの投資を続けており、先週ハノーバーメッセ2019で発表されたクールな更新を共有できることを嬉しく思います。その中には、Azureストレージの高度な脅威保護、コンプライアンスダッシュボード、スケーラブルな仮想マシンスケールセットのサポートがあります）（VMSS）。 catの下の完全なリスト。







Hannover Messe 2019で発表された以下の機能がAzure Security Centerで利用可能になりました。



  • 高度なAzure Vault脅威保護-セキュリティの専門家でなくても、顧客がボールトアカウントで発生する可能性のある脅威を検出して対応するのに役立つ保護レベル。
  • 規制順守ダッシュボード -セキュリティセンターのお客様が、サポートされている一連の標準とルールの順守状況に関する情報を提供することにより、順守プロセスを合理化するのに役立ちます。
  • 仮想マシンスケールセット（VMSS）サポート -セキュリティに関する推奨事項を使用して、VMSSのセキュリティステータスを簡単に監視します。
  • 専用ハードウェアセキュリティモジュール（HSM） （英国、カナダ、オーストラリアで利用可能）-Azureに暗号化キーのストレージを提供し、最も厳しいセキュリティ要件と顧客の要件を満たします。
  • VMSSのAzure Drive Encryptionのサポート-Azure Drive Encryptionは、AzureのパブリックリージョンのVMSS WindowsおよびLinuxで有効にできるようになり、標準の暗号化技術を使用してVMSSデータを保護および保存できるようになりました。


さらに、Azure Security Centerの一部として仮想マシンセットのサポートが利用可能になりました。 詳細については、これらすべてのイノベーションに関する記事をご覧ください。
