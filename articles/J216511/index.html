Verimatrix-問題または解決策

今日、急流という言葉を持つ大多数は、著作権の問題と関連しています。 IPTVの発展に伴い、 ピアツーピア放送でも同じことが起こり、著作権者はコンテンツの権利の保護を非常に心配しています。 Tomskの放送局の1つとの最近の事件により、著作権者にとって安全なコンテンツコーディングの重要性について考えさせられました。 トムスク放送局の人気は、他の放送局によるコンテンツのコピーおよび再放送からコンテンツを保護するVerimatrixの技術に従ってエンコードされていないため、約85チャンネルの放送を禁止されました。 そして、 IPTVサービスを提供する会社として、このトピックに非常に興味を持っています。Verimatrixについて詳しく説明したいと思います。







Verimatrixは、デジタルテレビサービスプロバイダー向けのコンテンツセキュリティソリューションのリーダーとして認められています。 同社は、革新的な3次元セキュリティアプローチを使用して、マルチメディアコンテンツをさまざまなデバイスに配信する権利プラットフォームを提供しています。 そして一般的に、同社は放送ビデオのセキュリティ分野での収益の面で世界一の地位を確立しています。 今日、ほとんどの著作権者は、コンテンツがVerimatrixに従ってエンコードされている場合にのみブロードキャストすることに同意しています。 会社にとって大きなプラスであり、放送局にとっては「ストレス」です。 オペレーターが今日Verimatrixで市場に参入した場合、それはすでに重要な競争相手です。 Verimatrixライセンスの支払いは、事業者が言うように、合法的なビジネスから利益を得ており、これは事業者の評判にプラスです。







同社のソリューションは、ほとんどのネットワーク（IPTV、インターネットTV、DBV、ハイブリッド、モバイルTV ）に適したVCAS（ビデオコンテンツオーソリティシステム）です。 業界標準のセキュリティと情報保護（HLS、PKI、AES、SSLなど）を使用して設計されたこのシステムは、信頼性の高いコンテンツ保護を提供します。 モジュラー設計、柔軟性、および高いスケーラビリティにより、VCASを少数の加入者を持つオペレーターとして使用できるだけでなく、マルチリージョンネットワークを持つ大規模なテレコムオペレーターとしても使用できます。







Verimatrixソリューションの主な原則は、未登録のサブスクライバーによるコンテンツの視聴を保護することであり、そのコピーとその後の配信も除外します。 コンテンツ保護自体は、「アクセスキー」の保存、生成、管理のための単一のシステムと、指定されたシステムへの専用アクセスチャネルに基づいています。 Verimatrixは非常に柔軟で使いやすく、システムの残りのコンポーネントのアーキテクチャの制限に関連付けられておらず、加入者ボックスのハードウェア変更を必要としません。 現在までに、このシステムは世界有数のメーカーの200以上のSTBモデルと統合されており、 iOS、Android、STB（HLS）、スマートTV、PC / Macの他のタイプのデバイスもサポートしています。











インターネットTV用VCAS （一部の機能）：

  • ViewRight Web Clientは、HLSに含まれるデバイスの大規模な市場をサポートしています。
  • 特許取得済みのVideoMark透かし技術によるコンテンツの不正コピーおよび配布に対する保護。
  • サポートされているビデオ形式：MPEG-2、MPEG-4 / H.264、VC1、DivX。
  • 複数のネットワークをサポートする高いスケーラビリティ：ブロードバンド、3G / 4G、Wi-Fiなど。
  • サポートされているコンテナ形式：MPEG-2トランスポートストリーム。
  • VODコンテンツの暗号化：便利なインターフェイスを使用した手動モードまたは自動モード。




放送事業にこのソリューションを使用することは誰にとってもビジネスですが、長期的に数十万人の顧客の放送予測を考慮すると、この投資は報われ、コンテンツを効果的に保護することができます。
