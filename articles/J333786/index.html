HPEサーバーにRAMを追加した方法



主にサーバー仮想化の使用が増加しているため、DRAMの品質と信頼性はこれまで以上に重要になっています。 もちろん、多くのIT専門家によると、RAMモジュールはサーバーの最も信頼性の高い要素の1つであり、最後のモジュールに障害が発生することは注目に値します。 仮想化には多くの利点がありますが、最大数の仮想マシンに対して最適なパフォーマンスを確保するためにサーバーで必要なメモリ量が大幅に増加します。 HPによれば、2007年から2011年までの5年間で、すべてのHP ProLiantサーバーにインストールされた平均メモリは、サーバーあたり4 GBから30 GB以上に500％以上増加しました。



現在、クラウドプロバイダーとして、HPE BLADESYSTEM C7000 ENCLOSUREシャーシでHP ProLiant BL460c Gen8 ブレードサーバーを使用しています 。 ここで完全にQuickSpecs、RAMの仕様のみを示します。

メモリの種類の指定とその違いを理解するのに助けが必要な場合は、 Habréに関する有用な記事をお勧めします 。



クラウド用の機器で使用する場合、つまり、スケーラビリティと安価さを損なうフォールトトレランスを必要とするシステムでは、エラー訂正機能（ECC RDIMM）を備えたレジスタメモリが使用されます。



お客様のニーズの増大に応じてクラウド環境のパフォーマンスを改善するために、サーバーのRAM量を可能な限り最大値に増やすことにしました。 論理的な疑問が生じました：どのような種類のメモリを購入するのですか？ サーバーメーカーの申し出に精通してから選択を開始するのは論理的でした。



簡潔な回答を求めて、HPはホワイトペーパー、または2 つ、2011年に1つ、 2017 年に 2つを見つけました。 私たちが自分のために特定したもの ：



  • HPはRAMモジュールのメーカーではありません。
  • HP Qualified Memoryブランドを見ると、DRAMが独自の診断ツールと特殊な診断テストを使用して広範な品質テストとテストを受け、HP ProLiantサーバーの最高レベルのパフォーマンスと可用性を保証しています。







  • DRAMモジュールは、HP Qualified Memoryブランド（ステッカー）を受け取る前に、集中的な4段階の品質管理プロセスを経ています。 製品開発のずっと前に、HPはティア1メモリサプライヤメーカーのDRAM要件を伝えています。 HP Memory Coreグループは、ベンダーから直接DIMMを受け取ります。 HPMQ（HP Memory Qualification）プロセス図は、品質管理プロセスの4つのフェーズをまとめたものです。 私たちの意見では、検証のために受け取ったモジュールのどの部分が除去されるかを説明していないという事実に注意を払うことが重要です。





    その結果、HPMQはチップの位置を変更しません;設計者は間違っていたようです



  • メモリは、熱くなりすぎないようにチェックされます。そのため、データを忘れることはありません。

    HPが検証したメモリは、熱くなりすぎてデータを忘れないようにテストされています。 HPが検証したメモリは、電源の過負荷によってシステムがシャットダウンしないようにテストされています。
  • HP認定メモリには、ProLiantサーバーと同じ完全保証が付いています。
  • サーバー障害が異なるブランド（非HP）のRAMによって引き起こされたと判断された場合、またはサードパーティの部品がHPコンポーネントまたはシステムに損傷を引き起こしていることが発見された場合、損傷したHP部品、人件費および旅費を含む修理費用は補償されませんHP標準保証。 ただし、サードパーティのメモリを取り付けても、ProLiantサーバーがHPの保証から除外されるわけではありません。 この誤解は、サーバーのメモリ選択の議論において、ITフォーラムのユーザーによって「OEMの」議論としてしばしば引用されます。



HPMQおよびHPカスタマーサービスの費用はいくらですか？ Yandex.Marketのスクリーンショットを添付した価格レベルを明確に理解するため。 特に、未知の売り手からのHP RAMの最低価格は平均市場を60％下回るため、これを本格的な市場調査と見なすことを申し出ません。











比較のためにKingston RAMを選んだのはなぜですか？



  • 特定のシステムの各Kingstonサーバーモジュールは、強制障害テストに合格する必要があります。 この独自の手順により、潜在的に欠陥のあるモジュールを拒否し、出荷された製品ロットに落下するのを防ぐことができます。 含めて、加熱中の信頼性チェックがあります。 品質チェックの完全なリストはこちらです。
  • キングストンテクノロジーは、JEDEC理事会の長期メンバーとして、メモリテクノロジーの業界標準に貢献しています。 すべてのKingstonメモリモジュールは、JEDEC要件と主要な半導体メーカーの基本仕様を満たしています。
  • テクニカルサポートとテストドライブ。 ロシア連邦には公式の駐在員事務所があり（たとえば、Crucialとは異なり）、要求に応じて、出荷に必要な数のダイが常に存在します。 テストドライブ用にサーバーメモリを取得し、特定のサーバーで実際の状態でどのように機能するかを確認できます。
  • 生涯保証。 メモリモジュールが故障した場合、保証の下で交換されます。


Kingstonには、RAMモジュール用の多数の異なるオプションがあります。 適切なウェブサイトで見つけることができます。 Kingstonによると、モジュールは関連するすべてのサーバーオプションでテストされているため、潜在的な非互換性の問題は存在しないはずです。 私たちの場合、HP BL460C G8を選択しましたが、動作に問題はありませんでした。



キングストンのメモリモジュールは何を提供してくれましたか？ 大量の調達により、近代化の予算を大幅に削減することができました。 キングストンとHPのRAMの品質管理に大きな違いを特定することはできませんでした。 ほぼ2倍の請求額を支払う理由が見つからなかったため、Kingstonモジュールを選択しました。 また、サービスの消費者に追加コストをシフトさせないこともできました。



この記事では、HPのコンポーネントの高コストに注意を払うつもりはありません。 さらに、HP ProLiantおよびHP BladeSystemサーバーの使用に満足しています。 単一の物理サーバーで多数の仮想マシンをサポートするには、より多くのメモリ、データウェアハウジングの接続、およびより多くのネットワーク接続が必要です。したがって、VMware認定のHPサーバーは、構築されて以来最高の仮想化プラットフォームの1つです。 「仮想化を念頭に置いて。」
