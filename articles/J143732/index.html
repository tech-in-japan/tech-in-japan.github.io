iPad Wi-Fi + 4Gは、iPad Wi-Fi + Cellularに名前が変更されました

法律上の理由により、Appleは最新モデルのタブレットの名前に「4G」のマークを付けることを拒否しました。 公式Webサイトは既に修正されているため、5月12日からタブレットは公式にiPad Wi-Fi + Cellularと呼ばれます。



さまざまな国で多くのユーザーの苦情があったため、名前を変更する必要がありました。 iPad Wi-Fi + 4Gが4Gネットワ​​ークに接続できないことに不満を抱いています。 たとえば、英国では公式の4Gオペレーターがまったくいないため、Appleについて特に多くの苦情がありました、 と ZDNetは書いています 。



Appleは誤解を招く広告に対して罰則を科せられるため、タブレットのラベルを4Gから法的に正しいCellularに変更するのが最善と考えました。 ブランドの統一性のために、英国だけでなく他の市場でもラベル付けが変更されました。



名前の変更は、米国、英国、カナダ、オーストラリア、UAE、シンガポール、ベトナム、タイ、ニュージーランド、マレーシア、アイルランド、香港ですでに確認されています。 他の国々が従う可能性があります。







タブレットiPadおよびiPad 2の以前のバージョンは、名前に変更はありませんでした：Wi-Fi + 3G。
