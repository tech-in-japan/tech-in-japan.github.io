ラバーレイアウトについて

すべてが正常であることを確認する方法を知らないかもしれない人のために。

サイトは特定の値にのみ拡大し、特定の値にのみ絞り込む必要があります。 私は通常、それぞれ1500ピクセルと980ピクセルを使います。



作り方 最小幅および最小高さ。



#site {

margin: 0 auto; /* max-width*/

min-width: 980px;

max-width: 1500px;

}







しかし、IEには含まれていません。 そして、式はブラウザをロードしすぎます...落ち着いて、友達！ 私はあなたを闇からの光へと導こう！

さらに書きます。







そして、ie.jsに



window.attachEvent('onload', mkwidth);

window.attachEvent('onresize', mkwidth);



var minwidth = document.getElementById("site").currentStyle['min-width'].replace('px', '');

var maxwidth = document.getElementById("site").currentStyle['max-width'].replace('px', '');

function mkwidth(){

document.getElementById("site").style.width = document.documentElement.clientWidth < minwidth ? minwidth+"px" : (document.documentElement.clientWidth > maxwidth ? maxwidth+"px" : "100%");

};







上記の方法の特別な長所は何ですか？ そして、一般に、最小値と最大値を1か所でのみ設定する必要があるという事実。



要するに、健康に使用してください！
