Intel 5150 WiMAXカード用のUbuntu 10.04でYotaを構成する





現在、WiMAXは多くのラップトップに組み込まれています。Windowsユーザー向けに、この点では、すべてがシンプルで、すべて工場で動作し、Yota Accessがすべてを実行します。 しかし、Linuxのドライバーはまだ作成されていないため、Linuxを使用している人たちは大げさなままです。 そして、そのような場合にしばしば起こるように、コミュニティはそれから脱出しなければなりません。



Ubuntu 10.04を使用してWiMAXカード（Intel 5150）をセットアップするためのこの手順+ 自動チューニングのスクリプト +





スクリプト構成



気にしたくない場合や、単に何かを理解したくない場合は、 このスクリプトを使用してください。 rootパスワードのみを入力する必要があります。



スクリプトをダウンロードして解凍し、ターミナルでコマンドを実行する必要があります。

sh intel5150-wimax-driver.sh









手動設定



スクリプトが必要ない場合は、手動で。

まず、これらのファイルをダウンロードします。

  • WiMAX-Network-Service-1.4.0.tar.bz2
  • Intel-WiMAX-Binary-Supplicant-1.4.0.tar.bz2
  • NDnSAgentConfig_forDriver.xml
  • NDnSAgentDefaultConfig.xml




  1. libnl-devパッケージをインストールする必要があります。 これを行うには、コマンドを実行します

    sudo apt-get install libnl-dev







  2. フォルダWiMAX-Network-Service-1.4.0およびIntel-WiMAX-Binary-Supplicant-1.4.0を解凍し、/ usr / src /ディレクトリに配置します。 または、コマンドを使用して端末ですべて実行します

    tar -xvjf 'WiMAX-Network-Service-1.4.0.tar.bz2′ –directory /usr/src

    tar -xvjf 'Intel-WiMAX-Binary-Supplicant-1.4.0.tar.bz2′ –directory /usr/src







  3. 次に、ドライバーを配置します。

    cd /usr/src/WiMAX-Network-Service-1.4.0

    make clean

    ./configure –prefix=/usr/ –with-i2400m=/usr/src/linux-headers-$(uname -r) –localstatedir=/var

    sudo make

    sudo make install







  4. サプリカントをしましょう

    cd /usr/src/Intel-WiMAX-Binary-Supplicant-1.4.0

    sudo ./install_supplicant.sh installsudo cp -r /usr/local/lib/wimax/*.* /usr/lib/wimax/







  5. 次に、構成をインストールします。

    sudo cp NDnSAgentConfig_forDriver.xml /usr/share/wimax/NDnSAgentConfig_forDriver.xml

    sudo cp NDnSAgentDefaultConfig.xml /usr/share/wimax/NDnSAgentDefaultConfig.xml









まあ、そのようなもの。



Yotaネットワークに接続する



接続するために実行する一連のコマンド。

sudo wimaxd

sudo wimaxcu ron

sudo wimaxcu scan //

sudo wimaxcu connect network 15





すべてが動作するはずです。



必要なコマンドのリスト：


sudo wimaxd —

sudo wimaxcu status —

sudo wimaxcu scan

sudo wimaxcu connect network 15 - Yota

sudo dhclient wmx0 – ( IP)

sudo wimaxcu ron – WiMAX

sudo wimaxcu roff – WiFi

sudo wimaxcu status link

wimaxcu reset factory — .
