ターミナルサービス用のマルチラインTAPIドライバーのベータ版のリリースおよび3CX v15.5のプレゼンテーションの発表

今週は、多くのCRMシステムとの統合に使用される新しいマルチラインTAPIドライバーについて説明し、3CX v15.5リリースのオンラインプレゼンテーションに招待します。



Windowsターミナルサービス用の3CXマルチラインTAPIドライバー



パートナーおよび顧客からの多数の要求に応じて、WindowsターミナルサーバーにインストールされたTAPI互換CRMシステム用のMulti Line TAPI Driverをリリースしました。



Multi Line TAPIドライバーが機能するには、Windows用の最新の3CXクライアント （3CX v15.5に含まれる予定）および3CX Proエディションが必要です。



3CX Multi-Line TAPI Driverのインストール



  1. 配布キットをダウンロードして 、ドライバーのインストールを実行します。







  2. インストール中に、内線番号とユーザー名を指定するウィンドウが表示されます。 その中に最大5人のユーザー（行）を追加できます。 残りの行は、インストール後に構成ファイルC：\ ProgramData \ 3CXMultiLineTapi \ 3CXTAPI.iniを介して追加されます。



  3. 3CX Multiline TAPIドライバーをインストールした後、Windows用の3CXクライアントの最新バージョンをダウンロードしてインストールします 。



  4. クライアントを起動します。 これでインストールは完了です。


行を追加する







前述のように、ファイルに追加の行が追加されます



C:\ProgramData\3CXMultiLineTapi and edit 3CXTapi.ini







以下の行に、同じ形式に従って、既存の行と同様に新しい行が追加されます。





注： 1,100ユーザーのJamesを登録する場合、 Windows用の3CXクライアントも拡張100で構成および登録する必要があります。 TAPIと3CXクライアントの構成が一致しない場合、 不明なセッションエラーが発生します。



行を追加した後、ファイルを保存し、 テレフォニー Windowsサービスを再起動します。



3CX v15.5オンラインプレゼンテーション



数千のユーザーが3CX V15.5 AlphaおよびBetaをテストした後、最終バージョンをリリースする準備ができました！



この素晴らしいイベントを祝って、V15.5のすべての機能について説明するために、2017年6月7日 17:00東ヨーロッパ時間（+2 UTC）にオンラインプレゼンテーションとQ＆Aセッションを実施します。



質問に対する回答は次のとおりです。NickGalea、CEO、Nick Borg、プロダクトマネージャー、Stefan Walther、ユーザーサポートおよびトレーニング責任者。 質問は、ハッシュタグ＃3CXv15.5を使用して、YouTubeライブチャットおよびTwitterを介して英語で質問できます。 質問自体の前に、 名前とパートナーのステータスを示します。 大きな要求は、質問を簡潔かつ積極的な方法で定式化することです-このようにして、割り当てられた時間内に質問の最大数に答えることができます。







イベントはYoutubeチャンネルとFacebook ページで放送されます 。 Facebookを介してイベントに登録することをお勧めします。



新しい3CXテクノロジーの秘密を共有するのを待っています！
