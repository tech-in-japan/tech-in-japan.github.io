会議の歴史「プラットフォーム。 未来を定義する”





2011年11月9〜10日に、モスクワは最初の国際的なテクノロジー∙エドロシア 2011会議を開催します。 この会議には、ロシアとCIS諸国の情報技術分野の専門家約3,000人が集まります。 Tech ∙ Ed Russia 2011は、有名な年次会議「Platform。」の開発における次のステップです。 未来を定義する」、モスクワで12年間開催されました。



この記事では、Microsoftが毎年開催するロシア最大のIT会議「プラットフォーム」の歴史について説明します。 プラットフォームの12年間で、この会議は何千人ものIT専門家、マネージャー、マネージャー、開発者、システム管理者の出会いの場になりました。 各プラットフォームで、参加者は技術レポートトラックを通じて技術と計画に関するMicrosoftの包括的なレポートを受け取りました。これは年々増加しています。



それがどうだったか見てみましょう。



プラットフォーム2000



会議の歴史「プラットフォーム。 未来の定義」は1999年に始まりました。 その後、会議には短い名前が付けられ、「Platform 2000」という名前で開催されました。 最初の「プラットフォーム」の主な対象者は、情報化のディレクター（当時）とIT部門のマネージャーでした。





osp.ru からの写真



最初の「プラットフォーム」のトピックは興味深いもので、1999年にマイクロソフトは次のように語っています。
  • Office 2000エンタープライズ展開
  • Windows 2000を実装するための機能と方法論。
  • エンタープライズ情報システム管理とTCO削減の戦略的問題。
  • 21世紀の日付の形式、つまり「2000年問題」の解決策のための組織のコンピューターサポートの準備
ご覧のとおり、最初の「プラットフォーム」はIT部門の差し迫った問題を提起し、大規模を自慢できませんでした。 しかし、それはほんの始まりに過ぎませんでした。



プラットフォーム2001







2000年11月、「プラットフォーム」が繰り返されました。今回は、当時のまったく新しいMicrosoft .NETアプリケーション開発プラットフォームが主なトピックでした。 会議は大幅に拡大し、11月22日から24日にかけてラディソンスラビャンスカヤホテルで3日間開催されました。 2番目の「プラットフォーム」にはすでに750人以上が参加しています。





pcweek.ru からの写真



Platform 2001では、3日間にそれぞれ5つの並行レポートトラックが提示されました。 合計で、会議は70を超えるレポート、4つのラウンドテーブル、14の実験室作品（300人の参加者）を開催しました。



.NETテクノロジに加えて、会議では企業情報システムの近代化と効率性の向上、Exchange 2000とSQL Server 2000サーバーテクノロジ、およびWindows 2000オペレーティングシステムのトピックについて議論しました。



そして、歴史とWeb 1.0に少し脱線しました。 プラットフォーム2001サイトは、今日まで正常に動作しています。 http://platforma2k.ru/で見つけることができます。 懐かしい人の多くは、「このサイトを表示するには、Internet Explorer 4.0以降をインストールする必要があります」という碑文を覚えていると思います。 確かに、目撃者によると、このサイトは完全に公式ではありませんが、会議参加者の一人が発行したディスクのコピーです。 ただし、これによりリソースの希少性が低下することはありません。



プラットフォーム2002







3番目の「プラットフォーム」は2001年11月に開催されました。 今回、会議の対象聴衆は、ロシアとCISのさまざまな企業の主要な専門家とIT部門の長でした。



Platform 2002カンファレンスのトピックは、Windows XP、64ビットWindowsアーキテクチャ、会社のサーバーソリューションなどの新しい基本技術でした。 また、レポートの量はわずかに減少しましたが（会議は2日間開催されました）、850人以上が会議に参加しました。



会議では、.NETプラットフォームと新しいVisual Studio.NETアプリケーション開発環境について引き続き議論しました。 さらに、会議の主要なトピックは、最近リリースされたOffice XP、BizTalk Server、Exchange Server、SharePoint Portal Server、Microsoft AppCenter、Content Management Server、およびWindows .NET ServerとSQL Serverの「Liberty」のプレリリースバージョンです。



大きく前進したのは、レドモンド本社と欧州オフィスの会社代表者からの報告でした。



プラットフォーム2003



次の第4回連続会議「Platform 2003」は2002年11月に開催されました。 今回、会議には1000人以上が参加しました。 テクニカルレポートストリームの数が再び5に増えました。
  • 情報インフラストラクチャ：Windows 2000 Server、Windows .Net Server、Systems Management Server、Operations Manager、Application Center、Internet Security and Acceleration Server、Exchange Server、およびMobile Information Server。
  • ビジネスアプリケーションを構築するためのプラットフォーム：BizTalk Server、SQL Server、.Net Framework、Commerce Server、Content Management Server。
  • ビジネス生産性システム：Office、SQL Server、Project Server、Sharepoint Portal Server、Content Management Server。
  • 情報セキュリティ技術：セキュリティ、公開キー基盤、VPNの構築。
  • .Netアーキテクチャでのアプリケーション開発：Visual Studio .NET、ASP.NET、Visual Basic .NET、C＃。
ご覧のように、レビューされた技術の数が増加し、会議はさらに波乱に富んでいます。 そして、会議の発展における非常に重要なステップは、会議への参加が支払われたことです。 それはお金のための最初の「プラットフォーム」でした。 それにも関わらず、参加者の数は減少しただけでなく増加したため、ITイベントとしてのプラットフォームの大きな価値が実証されました。



プラットフォーム2004。未来の定義



2003年、次の5番目のプラットフォームが開催され、今回は追加のスローガン「Defining the Future」を獲得しました。 これは本当に重要なスローガンです。会議の名前は来年も常に取り上げられており、会議で発表された技術は、近い、または少し離れた未来の技術であることを示していました。 会議はレーザーショーによって何百人もの参加者に開かれました。





sql.ru からの写真



伝統によれば、「Platform 2004」は11月に開催されました。 この会議は、Microsoft Corporationの副社長であるKirill Tatarinovによって開会されました。 その他のカンファレンスゲストには、Trustworthy Computingのセキュアな情報システムイニシアチブの主要なイデオロギーの1つであるセキュリティアーキテクトのDavid Leblancと、マイクロソフトの欧州本社のチーフアプリケーションアーキテクトであるEric Van Beverが含まれます。



今回の会議のトピックは、Exchange 2003、Office 2003、Dynamic Systems Initiative、Windows Server 2003、Longhorn、Microsoft Operations Manager、SQL Server Yukon、Visual Studio "Whidbey"の次期バージョン、BizTalk Server 2004などの新製品とイニシアチブでした。



この会議はさらに一歩前進し、2003年には約1,500人が訪れました。



プラットフォーム2005。未来の定義







6回目のPlatform 2005アカウントは、11月30日から12月1日まで開催されました。 この「プラットフォーム」は、ロシアのマイクロソフト駐在員事務所の新しいディレクターであるビルガーステンによって初めて会議が開かれたという事実によって特に際立っています。

Platform 2005の発表の1つは、当時の新しいOffice OneNote製品の公式のロシア化の発表でした。



2004年の会議には合計で1200人以上が参加しました。
