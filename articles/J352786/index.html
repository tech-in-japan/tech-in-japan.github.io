Azureを無料で使用する方法（学生向けのライフハック）

本日、Azureプラットフォームを無料で使用して、切り捨てられた機能を問題なく使用する方法について、興味深いライフハックを紹介します。 結局のところ、すべてが簡単です-あなたは学生である必要があります。







著者に発言権を与えます。



みなさんこんにちは。 私の名前はイリヤです。マイクロソフトスチューデントパートナーの退役軍人の一人です。 利用可能なすべてのAzure学生向けサービスについてお話しします。



Azure for Students Starter



Azureが市場に参入してからかなりの時間が経過しましたが、以前の学生は、新しいテクノロジーについて聞いて試してみたいと思った場合、使用したすべてのサービスの全額を支払うことを余儀なくされました。 また、学生を支援するために、Microsoftは2015年に学生向けの特別オファーをリリースしました。 多くの利点（サブスクリプションは1〜2分で作成され、作成時にクレジットカードデータは不要）と欠点（最大30 MBのサイズの無料WebサイトとSQLデータベースのみがサービスから利用可能）の両方がありました。 率直に言って、これはどんな学生プロジェクトのプロトタイプにとっても十分ではありません。 3年間で、サービスの数は6に増えましたが、16歳以上の年齢制限が現れました。



したがって、Azure for Students Starterは、クラウドテクノロジーに精通したい人にとって理想的なソリューションです。 ただし、提示された技術スタックでは、完全な開発には不十分です。







この提案に関心がある場合は、 ここでそれをよく理解できます 。



Azure無料アカウント



ごく最近、Azure無料アカウントから新しいオファーが登場しました。 約30のさまざまなAzureサービスの使用が含まれますが、いくつかの制限があります。 まず、すべてのサービスのリソースが制限され（1か月あたり750時間の仮想マシンでのコンピューティング、Azure App Serviceでの10個のアプリケーション）、その使用が制限を超える場合、200ドルの節約という形で「ゴールデンパラシュート」最初の30日間）。 次に、サービスの一部（同じデータベースと仮想マシン）が12か月間のみ無料で利用できるようになります。 また、このアカウントを有効にするには、クレジットカード情報が必要です。



制限されたサービスに加えて、25の無制限の無料サービスがあります（たとえば、Azure Active Directory、Azure Service Fabric、コグニティブサービス）。

それらのいくつかを次に示します。



  1. 1 GBのストレージでAzure App Serviceを使用する10のサイト、モバイル、またはAPIアプリケーション
  2. 1か月あたり30,000 Face Transaction APIおよび1分あたり20
  3. LinuxまたはWindowsでの750時間のAzure B1S仮想マシン
  4. Translator Text APIで200万文字
  5. 100個の機械学習モジュールと各実験の1時間の10 GBストレージへのアクセス






その結果、Azure無料アカウントは以前のものよりもはるかに技術的に進んだ提案です。 最初の1か月は200ドルですが、クレジットカード情報が必要です。 このオファーは、Azureの能力を実証したいが制限を超えない教師に適しています。



詳細はこちらをご覧ください 。



学生向けのAzure



生徒はほとんど義務を負わないので、最初の文に惹かれますが、技術はあまり豊富ではありません。 一方、2番目の提案は技術的な観点からはより魅力的ですが、それを有効にするにはクレジットカード情報を入力する必要があります。 さらに、開始$ 200は30日間のみ提供されます。



なぜこれをしているのですか？ さらに、Azure for Studentsは最初の2つの文の共生です！ 開始時の100ドル（時間制限なし）とクレジットカード情報の欠如が絡み合った大規模な技術スタック（Azure無料アカウントと同じ）があります。 取得するには、次の3つが必要です。



  1. ライブID（Windows 10のアカウント）
  2. 電話（Azureアカウントを確認するため）
  3. Microsoft Imagine Membership（無料）






Microsoft Imagineプログラムのメンバーになる方法は？



Microsoft Imagineは、技術的な（だけでなく）大学向けの学生サポートプログラムです。 無料のソフトウェア（特にVisual StudioとWindows Server）と学生Azureへのアクセスを提供します。



それを受け取るには、学生のステータスを確認する必要があります（大学のドメイン/マイクロソフトアクティベーションコード/ Shibbolethアカウントにメールを送信してください）。 Shibbolethと大学のメールはロシアではあまり一般的ではなく、Microsoftアクティベーションコードは、MicrosoftまたはMicrosoft Student Partnersの公式学生イベントでのみ入手できます。 アクティブ化するには、Microsoft Imagineポータルを使用する方法と、 サイトを使用する方法の2つがあります 。 その後、Azureポータルで連絡先情報を提供する必要があります。
