Web Standards Days、ミンスク2009

それでもWeb標準に達していないのですか？ それから私たちはあなたに行きます！



2007年の晩秋にサンクトペテルブルクで開催された Web Standards Developers Association の最初の公開会議は、完全に成功しました。 そのため、協会の地理的範囲を拡大し、ロシアやウクライナ、ベラルーシでこのような一連のイベントを開催することが決定されました。



決定を果たすために、Web Standards Developer AssociationはWeb Standards Daysと呼ばれる新しいイベント形式を提示します。 最初の会議は、2009年11月28日に、ミンスクで開催されます。 Storozhevskaya、15。



最初の会議とその後の会議の主な目標は、Web開発の分野で質の高い教育と経験の交換を提供することです。 このような形式は、個々の商業的利益や技術の促進を意味するものではありません-私たちは、単一のオープンWebを構築するという考えに一致しています。



無料で登録するには、姓と名をwst@web-standards.ruに送信してください。また、ご希望の場合は、ご自身のことばを少しお伝えください。 ちなみに、座席の数は限られています-彼らが使い果たし始めた場合、我々はこれについて警告します。



会議プログラムは、有名なWeb開発技術と新しいWeb開発技術の両方の開発、日々の過ちから利益を得る方法を学ぶ方法、およびハイテクWebHiTechサイトの競争について参加者に伝える5つのレポートで構成されています。



関心のあるすべての専門家を招待して、実際にウェブ標準の開発とその適用に関する議論に参加してください。 会議の講演者と参加者には、 Opera Software 、 Yandex 、 Soup Factory 、 WEBO Software 、 LOVATA Groupの代表者が参加します。



イベントの後、全員がスピーカーと参加者間の非公式のコミュニケーションに招待されます。 時間と場所が指定されます。



イベントのパートナーはOpera SoftwareとLOVATA Groupで、スポンサーシップと組織的なサポートを提供します。また、イベントの写真とビデオ撮影を提供するIT-JOB.byプロジェクトもサポートします。

16：45–17：00-閉会
