OperaはOpera 10ブラウザの第3ベータをリリースします

ノルウェー、オスロ-2009年8月13日 -Opera Softwareは、Opera 10の3番目のベータ版のリリースを発表しました（ www.opera.com/nextから入手可能）。 このリリースでは、パフォーマンスと安定性が主な優先事項になりました。 さらに、Opera Turboテクノロジーは、データを圧縮することでデータ転送を高速化するように改善されました。 以前のベータバージョンよりも効率的にWebサイトを操作するために、ブラウザインターフェースが変更されました。 導入以来、Opera 10は1000万回以上ダウンロードされています。



Opera 10の3番目のベータ版の新機能は何ですか？

タブを表示します。 以前のベータ版をテストした多くのユーザーは、より便利なタブレイアウトを選択しています。 これに対応して、第3ベータ版のOperaは、上下にだけでなく画面の横にもタブを配置する機能を提供しました。 タブのサイズは変更でき、タブで開かれたサイトのサムネイルの表示はオンとオフを切り替えることができます。



ユーザーインターフェース デザイナーのJon Hicksは、Operaブラウザでより効果的に機能するいくつかの新しい要素を導入することにより、ユーザーインターフェイスの更新を続けました。



多言語主義。 3番目のベータ版には、38の言語（ロシア語、ウクライナ語、ベラルーシ語を含む）のインターフェースが付属しています。 これにより、世界中のユーザーがOpera 10ブラウザーにアクセスしやすくなります。



障害保護。 統合されたクラッシュロガーにより、3番目のベータ版の障害に対する回復力が強化されました。



もっとターボ。 Opera Turboテクノロジーは改善されており、現在では低速チャンネルでWebサイトを操作する速度がさらに向上しています。



Opera Uniteテクノロジーは3番目のベータ版に含まれていますか？

2009年6月、Operaは新しいOpera Uniteテクノロジーのアルファ版を発表しました。 この技術は、Opera 10の3番目のベータ版と最終リリースには含まれません。ラボ版のままです。 できるだけ早くベータ版をリリースするためにOpera Uniteの作業を続けています。 新しいQ＆Aリストはunite.opera.com/supportで入手できます



「開発者としての私たちにとって、製品の成功は、ベータ版が100万回以上ダウンロードされたときに明らかになります。 Opera Softwareの最高経営責任者であるJon von Tetzchnerは次のように述べています。 -Opera 10の3番目のベータ版には多くの変更が含まれています。 技術の観点からこのような完全な製品をリリースしたことはありません。これは定期的な改善を受けただけでなく、ユーザーの特別な注意に値します。 このリリースを誇りに思っており、世界中のユーザーが大幅に再設計されたブラウザの新しいバージョンを高く評価することを願っています。

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社に関する詳細： www.opera.com
