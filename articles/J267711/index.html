鉄の利点、またはGPS / GLONASSトラッカーの知識について

当社の主な活動は、 衛星監視用ソフトウェアの開発です。 これに関連して、私たちはGPS / GLONASS機器のメーカーと緊密に協力しています。 したがって、私たちは経験を共有し、GPS / GLONASSトラッカーをテーマにした一連のテーマ記事をリリースすることにしました。ここでは、ナビゲーション機器を分類する方法を説明し、製品のユーザーの間で最も人気のあるデバイスの技術的特徴を詳細に検討します。



最初の記事は、GPSトラッカーとは何か、どのように機能し、何が起こるかという基本概念に焦点を当てています。 目標は、ナビゲーション機器の操作の原則と技術的実装の目的と機能によるGPSトラッカーの分類を読者に知らせることです。







GPSトラッカーとは何ですか？



ハードウェアトラッカーは、車両、人、およびそれがインストールされている他のオブジェクトの動きをリモートで制御するように設計されたデバイスです。 トラッカーは、衛星ナビゲーションを使用して、オブジェクトの位置を正確に特定します。



GPSトラッカーは、スタンドアロンの電子デバイスまたは関連ソフトウェアのいずれかです。



ソフトトラッカー-GPSトラッカーのハードウェアプラットフォームとしてスマートフォン、タブレット、その他のデバイスを使用できるようにするソフトウェア。 アプリケーションは通常のGPS受信機にアクセスし、ナビゲーションサーバーとのメッセージの交換はGSMとポータブルデバイスのWi-Fiモジュールを介して実行されます。 ソフトウェアトラッカーの例としては、 GPS Tagモバイルアプリがあります。



ハードウェアが不足しているため、ソフトウェアトラッカーは電子的なトラッカーよりも優れています。 ただし、トラッカープログラムの機能は、特別に設計された電子デバイスよりも著しく劣っており、主に位置の特定と送信に限定されています。 したがって、ハードウェアデバイスについて詳しく説明します。



トラッカーは、いくつかのコンポーネントで構成されています。



  • GPS / GLONASS衛星は、デバイスがその位置を特定するためのレシーバーを受信します。 衛星からの信号が利用できない場合、固定GSM基地局の信号を分析することで位置を特定できますが、この方法の精度は大幅に低下します。


  • ナビゲーションサーバーとのデータ交換のためのGSMモジュール。 デバイスは、位置を特定した後、収集した情報をGPRSパケットまたはSMSメッセージを介してセルラー基地局に送信します。 その後、信号は情報処理サーバーに送信されます。 PCまたはモバイルデバイスからデータ処理サーバーに接続すると、オペレーターはオブジェクトの正確な位置、その技術的状態に関するデータ、および移動経路に関する情報を見つけることができます。 設置された機器の構成に応じて、システムオペレータはテキストメッセージまたは制御コマンドをオブジェクトに送信する場合があります。


  • 内蔵バッテリー、Wi-Fiモジュール、内部メモリ。 デバイスには、これらのコンポーネントをオプションで装備できます。 GSM信号がない場合、トラッカーは情報を内部メモリ（「ブラックボックス」）に書き込み、できるだけ早く基地局に送信します。 一部のタイプのトラッカーは、衛星またはWi-Fiを介してサーバーに情報を送信できます。 上記のすべてのデータ転送方法を使用できるデバイスが市場に出回っています。


現在のトラッカーは、位置の特定に加えて、他のデータ、たとえば車のオンボードシステムや他のセンサーからの情報を蓄積することができます。



目的別トラッカーの種類



さまざまなGPSトラッカーは、目的に応じていくつかのグループに分けることができます：車両監視、個人追跡、他のオブジェクトの監視。



車両監視用トラッカー



トラッカーと車両の車載システムとの接続の種類によって、デバイスはいくつかの種類に分類されます。



  • 有線トラッカー。


電力は、車のオンボードネットワークを利用して提供されます。 内蔵バッテリーは、車両の電気システムの修理または故障中の自律運転にも使用できます。



外部アンテナを使用して、GPSおよびGSM信号の受信を改善できます。



さらに、このタイプのデバイスを使用すると、燃料レベルセンサー（FLS）、電子タコグラフ、ビデオレコーダー、点火制御システム、客室内および車体内の温度センサー、車両の技術的状態監視システムなど、さまざまなタイプの追加の自動車システムを接続できます。 CANバスリーダー）など



デバイスのセットアップと構成は、ファームウェアをダウンロードして制御コマンドを送信することにより、リモートで実行できます。



利点：このようなトラッカーは多機能で信頼性があります。 このタイプのデバイスの重要な利点は、フラッシュマウントの可能性です。



欠点：インストールと修復の複雑さ。



有線トラッカーのおおよそのコストは、機能によって異なりますが、3000-18000ルーブルです。



人気のあるGPS / GLONASSトラッカーの例は、 GALILEOSKY 3G v 5.1です。







  • OBD IIトラッカー。


OBD II（オンボード診断）コネクタは、車両システムを診断するように設計されており、ほとんどのタイプの現代の車に共通しています。 このコネクタから、オンボードコンピューターと車両の標準センサーから多くのパラメーターを読み取ることができます。



GPSトラッカーのメーカーは、これらの利点を活用する機会を逃さず、OBD IIコネクターに直接接続するデバイスを開発しました。



利点：このようなトラッカーには多くの機能があり、車に簡単に取り付けることができます。



短所：座標決定の精度が低い（外部アンテナを使用しない）。 また、車両の標準センサーからデータを読み取ると、必ずしも完全で信頼できる情報が得られるとは限りません。 たとえば、燃料消費量の監視精度を向上させるには、特別に設計された流量計、燃料レベルセンサーを使用し、「ワイヤー」トラッカーと統合してデータを処理サーバーに転送することをお勧めします。



OBDIIトラッカーのおおよそのコストは1500〜9000ルーブルです。



OBD IIトラッカーの例は、 ATrack AX7です。







  • シガーライター搭載のトラッカー。


異なる車で1つのGPSトラッカーを使用する必要がある場合があります。 これを行うために、シガーライターを搭載したトラッカーのモデルを開発しました。 このようなトラッカーはポータブルであり、多くの場合、防水エンクロージャがあります。 通常、シガレットライターを搭載したデバイスは、高度な機能（追加のセンサーを接続する）を使用しません。 これには取り付けの複雑さが伴います。 外部アンテナの存在はここではオプションです。デバイスは車のフロントガラスの前に配置でき、GPS / GSM信号レベルは非常に許容範囲です。



利点：このタイプのトラッカーはコンパクトなサイズで、車に簡単に設置できます。



欠点：デバイスを「隠す」ことはできません。 隠れたインストールの可能性はありません。これは、不正なドライバーがトラッカーを簡単に無効にできることを意味します。



シガーライターを搭載したGPS / GLONASSトラッカーの例は、 Gosafe G717です。







シガーライターを動力とするトラッカーのおおよそのコストは、6000〜30000ルーブルです。



トラッカー自体の機能はかなり限られたデバイスであることを理解することが重要です。 位置データとセンサーの測定値を蓄積し、その後の処理のためにサーバーに送信することしかできません。 この段階で、ソフトウェアと連携して、無限の単位、ゼロ、およびその他の文字セットが、単純な人が理解できる情報に変換されます。 クラシックな「カートラッカー+ソフトウェア」スキームは、ビジネスや日常生活の多くの分野で使用されています。



運送会社と宅配便。 監視システム（トラッカー+ソフトウェア）を使用すると、フォワーダーは商品の動きをすばやく追跡し、追加のセンサーを介して車とドライバーの状態を監視できます。 さらに、このシステムは不正なドライバーによる不正飛行や燃料流出を防ぐのに役立ちます。



タクシーサービス。 監視サービスデータにより、ディスパッチャは最も近い利用可能な車を決定し、道路状況に関する現在の情報を考慮して、ドライバーに最適なルートを設定することができます。



農業企業と道路サービス。 最新の監視システムの機能により、農業および建設機械の使用を監視して、時間を制御し、燃料の流出を防ぐことができます。



セキュリティサービス。 GPS / GLONASSトラッカーは、車のオンボードアラームシステムに接続されています。 セキュリティサービスの所有者またはオペレーターは、盗まれた車の位置を追跡し、必要に応じて、エンジンを停止する信号を送信できます。



保険会社。 運転スタイルを決定し、保険比率を計算するために、保険会社は顧客の車にインストールされたGPSトラッカーからのデータをますます使用しています。 GPSトラッカーは、収集したデータ（走行距離、速度、加速度、平均燃料消費量、エンジン速度データなど）を監視システムサーバーに送信します。 さらに、ソフトウェアはこの情報を処理し、運転スタイルを詳細に分析できるようにします。 分析の結果に基づいて、保険代理店は保険契約の係数を設定またはダウンできます。 ところで、 OBD IIコネクタに直接接続するトラッカーは、これらの目的に最適です。



スポーツ大会の主催者。 トラッカーを使用すると、ジャッジとマーシャルは、ラリーレイド、セーリングレガッタ、その他の競技会の参加者の正確な位置と速度を追跡できます。 トラッカーから受信したデータを使用して、リーダーを特定し、不正行為を防ぎ、緊急時にSOS信号に迅速に対応できます。



パーソナルGPSトラッカー



パーソナルトラッカーは、人やペットを監視するために広く使用されています。 原則として、それらはサイズがコンパクトであり、自律的な電源が必要です。 パーソナルトラッカーの動作原理は、自動車の動作に似ています。 違いは、外部センサーからのデータを分析するための形状、サイズ、および機能のみです。 たとえば、追加のセンサー（心拍数モニターなど）をパーソナルトラッカーに統合できます。



多くの場合、モバイルデバイスのアプリケーションは、個人用GPSトラッカーとして使用されます。



パーソナルトラッカーはどの領域で使用されますか？



企業のモバイル従業員の監視：宅配業者、販売代理店、出張者など



幼児、高齢者の親Observationの観察。 幼児や高齢者を監視するためのコンパクトな個人用GPSトラッカーの例は、SkyPatrol TT8850です。







ペットウォッチング。 これらは、耐湿性の耐衝撃性ケース、長いバッテリー寿命のための大容量電源の存在によって区別されます。 このようなトラッカーは、多くの場合、広い範囲で家畜の放牧を制御するために使用されます。 また、このタイプのデバイスは、見つけやすいように、犬やペットを狩るための首輪に組み込まれていることがよくあります。 カラートラッカーの例は、 Rilla G19です。 個人トラッカーのコストは、1000ルーブルから始まります。







野生生物の観察（GPSビーコン）。 GPSビーコンは、野生動物の移動を制御するように設計されています。これは、科学者や環境団体にとって非常に重要です。 防水耐衝撃ケースとは異なります。 そのようなデバイスは位置を決定し、通信することはほとんどありません。これにより、デバイスのコンパクトなサイズで、動作中の長期的な自律性（数年まで）を実現できます。



観光、野外活動。 これらを使用すると、旅行やトレーニングのルート情報を分析したり、ルートの地理的なポイントに写真やビデオを添付したりできます。 さらに、緊急時には、救助サービス用の位置データを送信できます。



貨物トラッカー



貨物の監視に使用されるトラッカーは、個人のトラッカーと非常に似ていますが、いくつかの機能があります。

ほとんどの場合、これらのデバイスは輸送用コンテナの監視に使用されます。 コンテナの金属ケースは外部信号の侵入に対する重大な障害であるため、このようなトラッカーには、密閉されたエンクロージャ、大容量バッテリー、強力な外部アンテナ（GPSおよびGSM）が必要です。



コンパクトなGPSビーコンを使用して、重要な郵便物を監視できます。



定常監視



動いている物体の監視に加えて、静止した物体を監視するためにGPSトラッカーが使用されます。 この場合、位置データは重要な役割を果たしません。 ただし、GPS衛星からの信号を使用して時刻を同期できます。 それ以外の場合、静止物体を監視する場合、トラッカーに接続されているセンサーのデータが主に重要です。 これらのデータはソフトウェアによって処理され、表、図、レポートの形式で視覚化されます。
