モバイルGMaps：モバイルの世界地図





モバイルGMapsプロジェクトは進化し続けています。 これは、 Googleマップ 、 Yahoo！などのサービスからの情報を表示する無料の携帯電話アプリケーションです。 Maps 、 Windows Live Local （MSN Virtual Earth）、 Ask.com Maps 、および衛星画像。



Java J2MEを搭載した携帯電話、およびJavaがインストールされている場合は、スマートフォンとコミュニケーターのWindows MobileとPalmがサポートされます。 MGMapsは、Bluetoothを介してGPSモジュールに接続するか、内蔵GPSレシーバーを使用して、地図上のユーザーの現在位置を自動的に追跡できます。 MGMapsの次のバージョンには1.35.00というラベルが付いていますが、まだ開発中です。バージョン1.33は安定バージョンと見なされます。



新しいバージョンは次のような変更を発表しました：





-ディスプレイの反転（水平モード）

-カードの保存-キャッシュ形式が変更され、各ファイルに複数のカード要素を含めることができるようになりました。これにより、メモリカードの使用が改善され、一部の電話機では-RAMメモリ

-カードストレージ-ユーザーはMGMapsCacheフォルダーを表示できます

-OpenStreetMapからのマップのサポート

-検索-フィンランドのサポートが追加されました。 「母国」としてのノルウェー、ベルギー、デンマーク

-バックライトの明るさは、NokiaおよびSony-Ericssonの電話機で調整できます

-終了確認はオプションになりました

-いくつかの問題が修正されました
