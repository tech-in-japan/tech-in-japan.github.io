Nokia副社長から開発者コミュニティへの公開書簡

親愛なるノキア開発者、



過去数週間にわたって、ノキアの発表された戦略について多くのことが言われてきました。 私はあなたの多くからの承認の言葉や懸念、および質問を聞きました。 私とForum Nokiaチームとの対話を続けてください。



この発表の重要性、Nokiaが開発者をどのようにサポートするか、現在および将来にどのような機会をもたらすかを明確にしたいと思います。



まず、発表された内容を思い出しましょう。 戦略の3つの主な側面：

  • Windows Phone向けのマイクロソフトとの幅広い戦略的パートナーシッププラン
  • 第三世界をターゲットに（ 次の10億をつなぐ ）
  • 未来のブレークスルーテクノロジー
Symbianはどうですか？ Qtはどうですか？



明らかに、これらの質問はすぐに発生します。 Windows Phoneが主要なスマートフォンプラットフォームになる一方で、Symbianの恩恵は引き続き受けられます。 新しい投資家を引き付け、既存のSymbianユーザーを維持し、競争力のある新しいスマートフォンを発売できるようにする投資を行っています。



ここ数週間、Symbian製品のリリースプランを確認し、移行期間、つまり2011年と2012年に新しい製品の強力なポートフォリオがあることを確信しています。 これらのデバイスは、サービスや写真やジオロケーションサービスなどの切り札との強力な統合の恩恵を受けます。 ギガヘルツ以上の周波数と高速グラフィックスを備えたプロセッサを含む、より強力なハードウェアが搭載されます。



これらの製品の競争力をさらに高めるために、Symbianのアップデートをリリースします。 最初のメジャーアップデートは夏に行われ、新しいメイン画面、新しいアイコン、高速ブラウザ、新しいNavbar、およびOvi Mapsサービスへのソーシャルメディアの統合を含む、新しいOvi StoreとOvi Mapsが含まれます。



CTIA Wirelessの最新製品であるNokia C7 Astoundでこれらの変更の一部を見たことがあるかもしれません。 これらおよびその他の革新は、メイン画面に表示される自動更新によりすべてのデバイスに届きます。 Nokia Astoundの所有者には、すべてのユーザーと同時に、不足している機能を備えたアップデートも提供されます。



Symbianをサポートする期間を何度も聞かれました。 答えを避けているかのように思えたのは間違いないでしょう。 真実は、簡単な答えを出すことは非常に難しいということです。 Windows Phoneデバイスをできるだけ早く市場に投入したいと考えていますが、Windows Phoneは、初日からすべての言語とすべてのローカリゼーションをサポートするわけではありません。



現在、Symbianが大きな市場シェアを持つ最も人気のあるスマートフォンプラットフォーム（中国、インド、ロシア、トルコ）を含む多くの市場で、マイクロソフトと協力しながらSymbianポートフォリオを可能な限り競争力のある位置に配置し続けますWindows Phoneデバイス開発。 特定の市場は1億5千万台のSymbianデバイスの販売においてより重要な役割を果たし、Windows Phoneスマートフォンが他の市場に登場した後、Symbianスマートフォンを長期間販売します。 そのため、Symbianが廃止される特定の日付を指定することはできません。



私が約束できるのは、Symbianユーザーと開発者を単に落とすだけではないということです。 少なくとも、国ごとに異なるサポート期間の法的義務があります。 最後の電話が販売されてから数年後にサポートを提供する必要があります。 私たちの目標は、Symbianデバイスの自然なライフサイクルが終了すると、ユーザーは新しいNokia Windows Phone電話を購入することです。 したがって、Symbianでの経験を悪化させることは私たちの利益にはなりません。 通信事業者は、Windows Phoneに移行する間、Symbianデバイスの販売とサポートを継続できることを保証しました。 Oviストアは、190か国以上のユーザーが利用できます。 現在、34か国の109のオペレーターが店舗に有利な支払いの受け入れをサポートしており、マイクロソフトが優れた新しいデバイスの開発を開始する一方で、こうした優れたローカリゼーション、差別化、柔軟な支払いオプションを備えたプラットフォームを引き続きサポートします。



Symbian開発プラットフォームであるQtおよび将来のMeeGoテクノロジーは引き続き重要であり、ノキアはこれらのプラットフォームに最適なツールキットとしてQtに投資することを約束し 、主に使用された商用ビジネスライセンスのビジネスを排除する計画の一環として将来の開発を計画していますモバイル市場外の顧客。



さらに、アプリケーション分析、アプリ内広告、アプリ内課金（プログラム内の仮想商品の販売）、新しいブラウザー、ハードウェアのアップグレードを準備しています。 開発者は、まもなくリリースされる予定のAPIに多くの新しく有用なものを見つけるでしょう。 戦略的投資の他の分野でも引き続きQtを使用しています。



つまり、SymbianとQtには多くの興味深いことが期待され、多くの新しいデバイスとプラットフォームの改善が期待されています。ユーザーは素晴らしいプログラムをダウンロードしてデバイスにインストールすると信じています。 これらすべてを合わせると、何百万人ものユーザーの間で収益化とブランド認知のために、あなたの能力を適用するためのQt投資が安全な選択であることを意味しています。



パートナーシップの発表は、Forum NokiaとMicrosoftが将来どのようにサポートするのかと多くの人を驚かせました。 これについてマイクロソフトと詳細に話し合うとすぐに、情報を共有します。 しかし、私たちはあなたの懸念やコメントに耳を傾けます。 ノキアとマイクロソフトは、開発者の移行を可能な限りソフトにするというビジョンと意図を共有しています。



ノキアの戦略の第2の柱である第3世界の国々のインターネット（「次の10億のインターネット」）は、開発者の機会、特にJava開発者への注目の高まりを表しています。 ノキアは1日あたり100万台以上のシンプルな携帯電話（フィーチャーフォン）を販売しています。 任意の基準で素晴らしい数。 開発者は、Javaアプリケーションを約6億台のSeries 40デバイスに配布できるようになりました。



Series 40開発者により多くの革新と改善をもたらすつもりであり、Java開発者がより簡単に、より簡単に、より手頃な価格で作業できるように、使いやすいツールとSDKを開発し続けます。 たとえば、Javaアプリケーションには無料の署名があります。 タッチおよびタイプUI用の新しいSDKはすでに市場に出回っています。 さらに、プロキシブラウジングデバイスのサポートの改善とWebアプリケーションのサポートの計画があります。



世界中の消費者は、ノキアの携帯電話用の新しいアプリケーションを楽しみにしています。



当社の戦略における画期的な技術分野には、MeeGoおよびNokia Research Center（ノキアの将来の技術を研究する世界的な研究所）での作業が含まれます。 MeeGoについては、今後数か月にわたって最新のニュースをお届けします。



最終的に、賞金総額1,000万ドルのCalling All Innovatorsコンテストの勝者はまだ決定されていません。 締め切りは近づいています：3月31日ですので、必ず申し込みを発表してください。 アプリケーションのコンテンツが米国およびカナダの消費者に適している場合、1,000万ドルの賞金プールの一部をお金とギフトで獲得できます。 このような大きなチェックを配り、何百万人ものユーザーがダウンロードしたアプリケーションを見ることが、私の仕事の中で最も楽しい部分です。



今後数週間および数か月間、Symbian、Windows Phone、およびSeries 40のステータスと、Ovi Storeの新しい改善点について引き続きお知らせします。 これらの各分野でお客様と協力できることを嬉しく思います。 その間、Ovi Storeでの次のQtプログラムまたはWebアプリケーションを楽しみにしています！
