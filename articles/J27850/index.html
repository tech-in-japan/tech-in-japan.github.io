CSS定数：最初のテストは既に画面上にあります

数日前、 ダニエル・グラズマンは、彼とデビッド・ハイアットが CSS定数のアイデアを成功裏に実装したというニュースをブログに投稿しました（ 私のブログの記事）。 すでに今日では誰もがCSS定数の動作を見ることができます。 これを行うには、 WebKit Nightly Buildsの最新バージョンをインストールします 。



実際、ここに私の小さなテストがあります：divブロックのスタイルを定数として定義しました。





開発者が彼らの考えを放棄していないことをうれしく思います。 しかし、ウェブマスターがそれを利用できる場合、それは不明です...そして、コードは今のところかなり面倒に見えますが、少なくとも何か:)



定数だけでなく、次の形式の変数も使用できるかどうかは興味深いです。

@-webkit-variables {

div1_height: auto;

div2_height: 100px;

div2_width: div1_height - div2_height; }







これは、レイアウトプロセスの多くの問題に対する優れたソリューションになります。
