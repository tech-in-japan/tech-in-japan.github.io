クリックの女王：歴史上最も優れたキーボードの物語

30年の間、彼女は模倣され、コピーされ、修正されてきました。現代のキーボードの設計の先駆者であるIBM Model Mについて話しています。







IBM Model Mキーボードが手に落ちたときに最初に目を引くのは、そのサイズです。 半キログラムのデバイスの小さなボタンとガラススクリーンを何年もクリックした後、重量で約2.5 kgのプラスチックと金属（厚いスチールプレートを含む）を保持するのは少し珍しいことです。 2つ目はサウンドです。大きなクリック音は、標準的なベージュの周辺機器をコンピューターの世界で最も価値があり有用な骨qu品の1つに変えました。



来年、モデルMは30歳になります。 しかし、多くの人はまだそれを他のすべてよりも高く評価しています。 彼女は最近、Minecraftの作成者であるMarcus Notch Perssonのテーブルに目撃されました。彼女はゲームPCに接続されていました。ビデオカードの1つはおそらく1,000ドル以上かかります。 「モデルMは本質的に史上最高のキーボードです」と彼はインタビューで述べました 。 YouTubeでは、モデルMのセットに多数のデモがあります。このビデオは、そのサウンドを展開して他のメカニカルキーボードのサウンドと比較するプロセスを説明したビデオです。 当初から、Model Mは完璧なキーボードのモデルでした。



「iPadが好きです。すばらしいデバイスです。 「Kindleリーダーは素晴らしいものです。プリンストン大学のITマネージャーであるBrandon Hermite氏は言います。しかし、タッチスクリーンを使用して記事、論文、その他の作品を書くことは決してありません。」 Hermitaは、モデルMの寿命を延ばすために多くの時間を費やしており、倉庫やリサイクルポイントからそれらを受け取り、ClickyKeyboards Webサイトで販売し、モデルMプライベートミュージアムを運営しています。 4000〜5000コピー。







他の多くの人と同様に、モデルMの子供時代の漠然とした思い出はまだありますが、先月、ニュージャージー州郊外に旅行してエルミートに会い、史上最も人気のあるキーボードの魔法を再発見しました。



彼の広々としたオフィスを訪れた当日、約20個のキーボードが高価なワインのように棚にきちんと置かれていました。 その上には、保護用のガラスケースの中に、黒いキーボードが別々に置かれています-プロトタイプのモデルMは、エルミートコレクションの最も古いアイテムの1つです。 大きなバスケットには最近の買収があり、それはまだ整理され、チップ、縫い針、および以前の所有者から残った他のゴミをきれいにする必要がありました。 モデルMを長年初めて見たとき、私たちにとって最も注目すべき瞬間は、その目立たない外観でした。 モデルMは過去の遺物である可能性がありますが、そのDNAはほとんどすべての最新のキーボードで見つけることができます。



70年代と80年代のキーボードは非常に異なっていました。有名なものから逆説的なものまで、まったく不自然です。



QWERTYキーボードレイアウトは、タイピスト向けに19世紀の終わりに発明され、すぐに一般に受け入れられた標準になりました。 しかし、1981年に最初のIBMコンピューターがリリースされたときには、大文字とスペースを入力するためのキーだけがレイアウトに含まれていませんでした。最近のユーザーは、テキストエディター、ターミナル、および「マイクロ計算機」に既に直面していました。 振り返ってみると、1970年代と1980年代のキーボードは、よく知られているものから逆説的で完全に不自然なものまで、非常に異なっていたことがわかります。 そのため、PC / XTとして知られるIBM PCのオリジナルの83キーキーボードでは、最も重要なShiftキーとReturnキーは小さく、どこかに横にシフトし、また不思議な矢印で示されていました。 一般的に、彼女はある種の混chaとしたミニチュアキーのセットとそれらの間の奇妙なギャップの印象を与えました。







1984年8月、IBMはより魅力的なPC / ATキーボードのリリースを発表しました。 前のモデルと比較して、「PC / ATキーボードは大きな前進です」とPC Magazineは言いました。 ATは最新のキーボードのようなものではありません。その中のファンクションキーは、通常の上部1行ではなく、左側の2行にあり、エスケープは補助数字キーパッドの中に隠れており、CtrlとCaps Lockは交換されています。 しかし、この形式でも、現代のユーザーは以前の形式よりもはるかに理解しやすくなっています。







しかし、IBMは単なる許容可能なオプション以上のものを作りたかったのです。 1980年代初頭、同社は10人のタスクフォースを集め、専門家とユーザーの推奨に基づいてより良いキーボードを作成するタスクを設定しました。 前のバージョンの設計は「迅速かつ効率的に行われました-それは深刻なチームワークの結果であるとは言えません」と、そのタスクフォースのメンバーであるデイビッド・ブラッドリーは言います、そのメリットは、一般的に受け入れられている機能Ctrl + Alt +削除する 新しいグループは、初心者のコンピューターユーザーに、より快適なキーボードのテストを依頼しました。このキーボードでは、重要なキーを大きくし、CtrlやAltなどのよく使用されるキーボードを2か所に設置して、手で簡単に届くようにしました。 必要に応じて、多くのキーを取り外して別の場所に再配置できます。 モデルMが誕生しました



1985年に、モデルMはIBM 3161端末の一部として導入され、IBM拡張キーボードと呼ばれました。 PC互換バージョンが来春に登場し、1987年にIBM Personal System / 2の公式標準になりました。 Hermiteによると、最も初期のモデルMは、1985年6月10日に作成された端末のバージョンです。







各Model Mキーボードの背面には一意の識別番号と製造年月日があるため、日付を非常に正確に決定することができます-Hermiteは、20歳以上の人から誕生日に作られたキーボードを検索する注文を常に受け​​取ります。 彼は、モデルMアーカイブプロジェクトも管理しています。これは、会社を通過したすべてのキーボードに関する情報と、他のユーザーが提供した情報（識別番号、製造日、シリアル番号）を保存する巨大なスプレッドシートのアーカイブです。



IBMは私に言っているような漠然とした感覚があります：「それは未来のキーボードだからあなたはそれを愛すべきです」とPC Magazineの評論家は書いています。



Hermiteのコレクションには、旅行代理店で注文するために設計された名前のキーや、キーが3つにグループ化された小さなモデルなど、特定の幅の狭いキーボードが含まれています。 「コンピューターが最初に市場に参入したとき、彼らは自分自身をビジネスデバイスとして位置付けました」と、元IBMマネージャーであるNeil Muskens氏は語っています。 特定のプログラム用のコマンドが付いたステッカーは古いキーボードでも保持されており、ブラウザはキーボードがWordStarやLotus 1-2-3などのプログラムでどの程度うまく機能するかに基づいてキーボードを評価しました。



ある観測者は、モデルMが再びキーのレイアウトを変更したことに失望しましたが、同時に、このデザインが長期間残るという感覚は消えませんでした。 IBMが私に言っているような漠然とした感覚があります：「それは未来のキーボードであるため、あなたはそれを愛すべきです」とPC Magazineでレビュアーを書き、コンピューター技術のさらなる発展が示すように、それを控えめに言っても、本当の予感以上のものでした。



Model Mのレイアウトはそれほど長く変化していないので、今日では当たり前のように思われています。 しかし、そのキーボードの子孫では、最も伝説的なモデルMの「トリック」の1つが船外に残されていました。曲げバネを備えたキー-PC / XTに登場したメカニズムです。 ピストンのように垂直に押し下げられるメカニカルスイッチとは異なり、モデルMの各キーの下には、圧縮、平坦化、または曲げられたスプリングがあり、キーを離すと跳ね返りました。 最近のほとんどのキーボードで使用されている柔らかく静かなゴム製キャップとは異なり、彼らは注目を集めました。 これは必ずしも良いことではありません。モデルMの所有者は、配偶者や同僚が絶え間ないノックに耐えられないと不平を言うストーリーをときどき発行します。 しかし、鑑定家は、バネの抵抗とそれらの大きな「クリック」がキーが最後まで押されたかどうかを理解するのに役立ち、それがエラーの数を減らすと言います。 そして、おそらくより重要なことは、モデルMでの入力は特別で具体的なものです。 タイプライターのように多くの点で、明確なクリックで各文字を物理的に感じることができます。







モデルMが登場してまもなく、その多くが市場に登場しました。 IBMは、新しいキーボードモデルの設計に象徴的な変更のみを加えました。 その結果、モデルMの郷stalは世代から世代へと受け継がれています。 「人々はしばしば私に電子メールで連絡します。1980年代に技術専門の若い学生だった時代を思い出させてくれてありがとう」とHermita氏は言います。 若いお客様は、キーボードのキーを並べ替えることで、学校のクラスメートで冗談を言ったことを思い出します。



1990年、IBMはタイプライター、キーボード、プリンターの米国ラインをLexmarkと呼ばれる新しい会社に変身させました。 6年後、レックスマークはキーボードの製造を拒否しました。これは、マスケンズが、より安価な製品への業界の一般的な移行と呼んでいる間に起こりました。 IBMはスコットランドの工場とMaxi-Switchという会社でしばらくの間、それらを注文し続けましたが、私たちが知る限り、最新のIBM Model Mは1999年に組立ラインを離れました。



このような限定的な提供により、モデルMファンはすぐにお気に入りのキーボードで入力する機能を失います。



約80ドルで、Model Mの公式バージョンを今すぐ購入できますが、IBMブランドはありません。 Lexmarkが退職した後、Muskensは、Unicompを装って他の元従業員と一緒に、このキーボードの製造のための特許と機器を徐々に取得し始めました。 「電子機器を変更する必要がありました」と、Muskens氏は言います。「ヒンジ付きシェルフの材料は1999年に変更されました。しかし、他のほとんどすべては変更されていません。」







一部の人にとっては、これではまだ十分ではありません。 「IBMロゴ付きの製品を販売できるかどうか、常に質問されていますか？ 「いいえ、ロゴはIBMが所有しています」とMuskens氏は言います。 彼は、IBMがまだ既存の商業顧客向けに少数のキーボードを注文しているが、古いロゴが必要な場合は、eBayまたはHermitaのような人に行く必要があると言います。 他の人にとっては、Model Mの固有の優れた品質と多様性は、ノスタルジックな信頼性の概念よりも重要です。一部のユーザーは、ワイヤレスBluetoothデバイスに作り直します。 1人のRedditユーザーが、RazerまたはAlienwareの見事なデザインに似たキー照明を備えたオリジナルの修正を投稿しました。 しかし、このような限定的な提供により、Model Mファンはすぐにお気に入りのキーボードで入力する機能を失います。







「オイルと比較できます。 石油埋蔵量が枯渇したら。 そして、それは災害になるでしょう」とエルミタは言います。 しかし今、これはまだ非常に遠いようです。 最も古いモデルMsはすでに30年間務めており、エルミートは、少なくとも10代または20代続くことを望んでいます。少なくとももう1世代がこのコンピューターの歴史を利用するのに十分です。







モデルMは、エンターテイメント用ではなく、主に産業用として最新のコンピューティングシステムが開発された当時の成果物です。 開発されたPS / 2コンピューターは少なくとも2295ドル（今日の標準では約5000ドル）であり、そのパワーと機能は現代のスマートフォンよりもはるかに劣っていました。 数十年後、コンピューターのパワーは飛躍的に増大し、コンピューターの価格は劇的に低下しました。 しかし同時に、メーカーは長寿命と長寿命という概念を放棄しました：多くのサードパーティ企業がほとんどのコストで最新のマウスとキーボードを販売する準備ができている状況では、メーカーに必要以上の投資を促すことは困難です。



そして、古いテクノロジーを新しいテクノロジーに素早く変更することで、私たちが失ったものを完全に体験する機会が与えられました。 Redditに関する最近のコメントで述べられているように、「これらのナンセンスは本当のゲーミングキーボードです。 どのように彼女をあざけりますか、彼女はまだあなたより長生きします。」
