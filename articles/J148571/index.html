飛行船またはSSDの知り合い

すべての良い一日...

2年以上、Habrahabrを読んでいます。ここから、ITの分野で（ITだけでなく）実用的なすべての情報を引き出します。

しかし、誰もが招待を取得しようとすることに手が届かず、プレゼンテーションのためのトピックを思い付くことができませんでした。

新しいラップトップを購入すると、すべてが変わりました。 通常のHDDに数日間座ってから、SSDをその中に入れました。

私はただびっくりしました。 私の、最速のSSDではないので、違いは単純に大きかった（ラップトップに7200 rpmのフル価格のHDDがあったという事実を考慮しても）。

そして、このことについて何かを書くためにアイデアが熟しました。

しかし、私の記録はどのSSDが優れているかというホリバーではありません。

ここでは、SSD向けWindows 7の最適化に関する人々の経験を1か所で購入、インストール、収集するプロセスについて説明します。

Windows 7がSSDにインストールされている場合、Windows 7自体がシステムパラメーターに変更を加えると多くの人が書いていますが、これらの変更は、率直に言って、十分ではありません。



SSDの購入



良い友人/知人のペアの推薦によってのみ、あるいは会社の評判に基づいて良いものを選択することが可能になったとき、長い時間が過ぎました。

現在、インターネットにはさまざまなプラットフォーム/フォーラムがたくさんあり、この製品やその製品を実際に使用した経験のある人は退会します。

保証が終了してから3日後にランダムに購入したDVDプレーヤーが「死亡」した後、私はインターネット上の人々に相談してから購入します。

多くのトレーディングフロアでは、低価格のSSDに対しても膨大な読み取り/書き込み速度を書き込むだけですが、実際には、「合成」速度測定値を実際の速度インジケーターと比較することはできません。

いくつかのフォーラムで質問し、SSDの要件として「最良価格/品質比」と4 TRの予算を示して、Crucial M4 128 GBという明確な答えを得ました。 私はそれを買って、後悔はしていません。

ちなみに、大容量のSSDを使用することはお勧めしません。 現時点では128GBが理想的なオプションです。

Windows（または他のOS）+必要なプログラム+いくつかのゲームをインストールします-この量で十分です。

しかし、HDD（私の場合のように、内部または外部）をファイルウォッシュとして使用することをお勧めします。 SSDの書き込みサイクル数には制限があり、80％を超えて一杯になったときのパフォーマンスは劇的に低下します。



SSDの取り付け



SSDを通常のデスクトップコンピューターにインストールすることは、通常のSATA HDDをインストールすることと同じであるため、ここでは複雑なことはありません。

ここでラップトップを使用するのはより困難です。 ラップトップメーカーは最近、ラップトップのスタッフィングへのアクセスを制限していませんが、売り手は、特に小規模な地域企業で、さまざまな「保証」ステッカーなどでラップトップの内側へのユーザーアクセスを制限しようとしています。

この場合、保証の取り消しの脅威について気にすることはありません（偶然、違法である可能性があります）。または、サービスセンターに行ってそこにSSDをインストールします（ほとんどの場合、追加料金を支払う必要があります）。

ラップトップを購入したとき、保証に問題があるかどうかをセールスアシスタントに明確に伝えました。 否定的な答えを聞いた後、数日後に私は店に走り、128GBでCrucial M4を購入しました。

私の場合（Asus N55）のインストールは非常に簡単でした。

ラップトップカバーの2つのボルト、HDDをケースに保持しているフレームの2つのボルト、およびフレームに保持しているHDDの4つのボルトを外しました。

SSDのインストールは逆の順序で実行されます。

SSDをラップトップにインストールするプロセスをおおまかに説明するこの魅力的なプロセス（私が撮影したものではありません）のビデオインストラクションを添付します（場合によっては、ニュアンスがあるかもしれません。





SSDにWindows 7をインストールする



Windowsを通常のHDDにインストールする場合と比べて、大きな違いはありません。 一つだけある...

SATAコントローラーのマザーボードの一部のメーカーのBIOSでは、デフォルトモードはIDEエミュレーションであり、HDDにとっては重要ではありませんが、SSDの場合は非常に重要です（AHCIモードを有効にすることがSSDのインストール要件であると言えます）。 AHCIモードの有効化は非常に簡単で、BIOSで実行されます。 具体的には、私の場合、AHCIを有効にするパラメーターは[詳細設定]タブにあります-SATA構成-SATAモード選択



SSD用にWindows 7を最適化する



既に述べたように、SSDの書き込みサイクル数は限られているため、すべての最適化は関連しているため、システムにディスクへの書き込みを可能な限り少なくする必要があります。

まず、SSDコントローラーのファームウェアを更新することをお勧めします（ほとんどの場合、店頭では古いファームウェアのSSDを販売しています）。

SSDの場合、コントローラーはSSDの中心であるため、これは非常に重要です。 多くの古いファームウェアには、メーカーが時間をかけて修正するさまざまな不具合があります。

このプロセスは、製造業者ごとに個別です。 私の場合、すべてが非常に簡単でした。 オフからダウンロードします。 実行可能ファイル形式の重要なサイトファームウェアで、システムドライブのルートに挿入します。

プログラムを起動した後、再起動することを提案します。

再起動後、Windowsを起動する代わりに、フラッシャープログラムが起動し、汚れた作業を行い、コンピューターを再起動します。

Windowsを起動した後、ファームウェアファイルを削除して生活を楽しむことができます。

次に、 チップセットドライバーとSATAコントローラーをインストールする必要があります。



その後、TRIMがオンになっているかどうかを確認する必要があります（必要なドライバーがインストールされている場合は、オンにする必要があります）。

この方法で確認できます。管理者としてコマンドラインを実行し、fsutil behavior query DisableDeleteNotifyコマンドを入力します。

応答DisableDeleteNotify = 0が返される場合、TRIM関数はアクティブです。 TRIMがアクティブでない場合-ドライバーに目を向けます。



次に、不要なサービスを無効にする必要があります。Win+ R-> services.msc

SuperfetchおよびWindows Searchサービスを無効にします（サービスをダブルクリック->「停止」->「スタートアップの種類：無効」）



次に、システム保護（簡単な方法-システムの復元）とスワップファイル（システムなしでシステムを使用するのに十分なRAMがある場合は、HDDに転送します）をオフにします。

システム保護は次のように無効になります。「コンピューター」-「プロパティ」-「システム保護」-「構成」-「システム保護の無効化」

スワップファイルは次のように無効になります：「コンピューター」-「プロパティ」-「システムの詳細設定」-「速度」-「パラメーター」-「詳細」タブ-「変更」-「スワップファイルのサイズを自動的に選択する」ボックスのチェックを外して、SSDを選択し、 [スワップファイルなし]-[設定]を選択します。 再起動します。



次に、休止状態システムをオフにします。 意味がなく、余分なギガバイトのスペースは不要ではありません。

これは簡単に実行できます。コマンドプロンプトでpowercfg -h offと入力します。



その後、次のようにディスクのインデックス作成をオフにします。

「コンピューター」に移動し、SSDのボリュームを選択し（複数のボリュームがある場合は各ボリュームでこれを行う必要があります）、右ボタンでクリックして「プロパティ」を選択し、「プロパティに加えてこのディスク上のファイルのコンテンツのインデックス作成を許可する」チェックボックスをオフにしますファイル»



ラップトップにSSDがインストールされている場合、電源プランをわずかに変更する必要があります。

これを行うには、「コントロールパネル」-「システムとセキュリティ」-「電源」に移動します

[高パフォーマンス]モードを選択し、[電源プラン設定]-[追加の電源設定の変更]-[ハードディスク]-[ハードドライブを切断する]を[なし]に設定します。 これは、コンピューターの前にいなくてもTRIMが機能するようにするためです。



そして最後に-レジストリの変更：
