お知らせRambler.iOS＃9







「今でも、敵はRambler.iOSの動画を熱心に見ています」

ハーバード大学の学生の動機



長い休憩の後、5月25日にDanilovskaya Manufactoryのオフィスで開催されるRambler.iOS＃9の春の会議を発表します。





この問題のヒーローは次のとおりです。



  • 「 記憶のない生活 」というレポートのあるデニス・バトビンキン。 彼のレポートでは、デニスはメモリ不足エラーについて話し、このエラーの原因を分析し、それに対処するための有用なツールを示します。







    トークトーク： GitHub









  • Sergey Krapivenskyが、 静的コードアナライザー 、その使用方法、解決する問題、通常の開発サイクルでの実装方法について説明します。







    トークトーク： GitHub











  • Samvel Majlumyanは、 Swiftをサーバー開発ツールと見なし、一般的なプラットフォームと比較します。 バックエンド開発のマイクロサービスアーキテクチャ、問題、およびベストプラクティスの問題に触れます。







    トークトーク： GitHub









従来、レポートの合間に、ミニゲームが開催されます。 できるだけ面白くするために、今、あなたからの素晴らしい質問を受け入れます 。 メンバーになり、自分の博学を自慢したい場合は、iOSの開発または開発全般に関する質問をrambler.ios@rambler-co.ruに送信してください。 唯一の要件は、質問に明確な回答が必要であり、論理的に到達できることです。







Tech Talksに参加して、RAMBLER＆Coの専門家、Mail.Ru、React Native vs Native、ObjC vs SwiftなどのYandexトピック、Ash Furrowの秘密の質問について議論することもできます。

  • パートナーからのソフトウェアライセンスの宝くじ、およびRAMBLER＆Coのお土産。
  • ライブブロードキャスト-TwitterおよびFacebookのグループでニュースをフォローしてください。
