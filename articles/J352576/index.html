Mikrotik RouterOSを任意のクラウドにインストールする簡単な方法

多くのクラウドプロバイダーは、ISOファイルのダウンロードを許可せず、クラウド自体が提供するOS以外のOSをインストールする機会を提供しません。







この記事では、この制限を回避する方法と、5つの手順でクラウド仮想マシンにMikrotik Cloud Hosted Routerを簡単にインストールする方法を説明します。









  1. 新しい仮想マシンを作成します（UbuntuまたはDebianは重要ではありません）

  2. /tmp



    tmpfsを起動してマウントします 







     mount -t tmpfs tmpfs /tmp/


  3. リンクをたどり、「Cloud Hosted Router」生イメージをダウンロードします。 







     cd /tmp wget https://download2.mikrotik.com/routeros/6.39.3/chr-6.39.3.img.zip


  4. 次の手順では、イメージを抽出して仮想マシンディスクに書き込みます。 







     unzip chr-6.39.3.img.zip dd if=chr-6.39.3.img of=/dev/vda bs=4M oflag=sync


  5. 次に、ハードリブートを初期化する必要があります。 







     echo 1 > /proc/sys/kernel/sysrq echo b > /proc/sysrq-trigger


    再起動後、Mikrotik RouterOSはハードドライブを自動的に準備し、ログインプロンプトが表示されます。
