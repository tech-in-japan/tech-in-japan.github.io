Windows MobileからAndroidへの切り替え。 アプリケーションリスト

WMコミュニケーターからAndroidへの移行の実務経験。

リンクと簡単な説明付きのAndroidアプリケーションの別のリスト。

以下はすべて著者の主観的な意見であり、究極の真実であると主張するものではありません





前文 ：Windows Mobileベースのコミュニケーターと3年間共存した後、WMが不気味で不器用で曲がった怪物のように見え始めました。 コミュニケーターは非常に密に使用されていました。もちろん、通話とSMSですが、これは主なものではありません。 主に-読書、モスクワの交通渋滞での毎日のナビゲーション、ウェブサーフィン、メール、連絡先の保存、カレンダーを使用した作業時間のスケジュール（タスクを残しました）、メモ（フリーハンドグラフィックを含む）、通信IM、VoIPコール、ssh。 さらに、連絡先、カレンダー、およびメモは、USBおよびBluetoothを介してデスクトップコンピューター（BB）およびラップトップ上のMS Outlookと常に同期されます。 必要なデータは、コミュニケーターと職場、そして自宅のラップトップの両方で常に利用可能であり、別のプラットフォームに切り替えるときにこれを拒否したくありませんでした。 私の選択はAndroidでした。 オープンでモダンなプラットフォームなど。 プログラマーの魂に近い。





HTC Wildfireを購入してから過去2か月間、試行錯誤を繰り返して必要なプログラムのリストを作成しました。 これは、疑わしい品質や疑わしいソースからのさまざまなソフトウェアの複数のインストールとアンインストールのおかげで達成されました。 副次的な結果は、コミュニケーターのわずかな不安定性でした-2-3日ごとに部分的にフリーズします。 以前に考えたように、ハードリセット、フラッシュドライブのフォーマット、必要なアプリケーションの再充填が決定されました。



同時に、MS Outlook（HTC Syncを使用してAndroidと同期する）を使用せず、Googleサービスに完全に切り替えることにしました。 この移行で私が失った唯一のものは、同期されたグラフィカルなノートでした。 しかし、画面上を指で書くのは難しく、スタイラスではないので、我慢しなければなりませんでした。



Windows Mobile + MS OutlookからAndroid + Googleへの切り替え



1. Googleアカウントが作成されました。



2.「Windowsでエンコードされたデータとカンマ区切りのデータを含むファイル」の形式でMS Outlookから連絡先をエクスポートし、その後ファイルからGmail連絡先にデータをインポートします。 連絡先の写真を「ペン」で転送する必要がありました。



3. MS Windowsから「Windowsでエンコードされたデータとカンマ区切りデータを含むファイル」の形式でカレンダーをエクスポートし、その後ファイルからGoogleカレンダーにデータをインポートします。 カレンダーのエクスポートには問題があります。ファイル形式の制限により、誕生日などの定期的なイベントを正しくエクスポートすることはほとんど不可能です。 1年のイベントをエクスポート/インポート（たとえば、2010年9月から2011年8月まで）してから、Googleカレンダーで周期的なイベントを編集し、必要な期間と繰り返しの期間を割り当てます。



4. MS Outlookでは、Notesはより複雑です。 多くのメモがあり、それらが異種である場合、AndroidにMementoプログラムがインストールおよび設定されるまでエクスポート/インポートを延期することをお勧めします（以下を参照）。Googleドキュメントにテーブルを作成し、何らかの方法でメモをインポートできます。 たとえば、すべてのメモをスプレッドシートにマージして、Googleドキュメントに一度だけエクスポートできます。



5. BBハードドライブ上のコミュニケーターのフラッシュカードの完全なコピーが作成されました。



6.コミュニケーターは次のようにハードリセットされました。電源ボタンをオフにします。 「音量を下げる」を押しながら、「電源」を短く押します。 表示される画面で、「音量を下げる」を使用して、「ストレージのクリア」項目に移動します。 [電源]をクリックします。 「音量を上げる」をクリックします。 ダウンロード後、標準の起動ダイアログ（言語、WiFiの有効化、Googleアカウントの設定）に応答する必要があります。 「設定」-「電話について」-「システムの更新」-「今すぐ確認」を実行することで、オペレーティングシステムの更新を確認しても問題はありません（Android 2.2は間もなく表示されます）。



7.フラッシュドライブがフォーマットされました：「設定」-「デバイスメモリ」-「メモリカードの取り外し」-「メモリカードのクリア」-「SDカードのフォーマット」-「すべて消去」。

デバイスがネットワーク（WiFiまたはGPRS / 3G）に接続されている場合は、Googleコンタクトとカレンダーが既にお使いの携帯電話にあり、準備ができていることを確認できます。 「連絡先」アプリケーションでは、「メニュー」-「表示」-「Google」のみを選択します。 SIMカードまたは電話のメモリの連絡先を使用していません。 デバイスで連絡先を編集する際の混乱を避けるため、Google Mailの連絡先のみ。



アプリケーションのインストール
