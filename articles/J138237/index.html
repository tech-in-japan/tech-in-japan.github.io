ソフトフォンの代替

ソフトウェアツールからハードウェアソリューションへの「ライフパス」の説明。



過去



2008年以降、彼は自宅で働き始めました。 クライアントとの作業に非常に重要な音声/電話通信の問題は、ラップトップにソフトフォンをインストールし、SIPを介して動作中のIP PBXに接続することによって決定されました。 「ハードウェア」は、最も単純な有線ヘッドセットと追加のUSBオーディオアダプターによって決定されました。 使用したMacbookにはマイク入力がありませんでした。



次の機能が気に入りました。

  • 「手の自由」-電話での会話中にコンピューター上で完全に機能する能力。 CRMで何かを記述したり、ドキュメントで情報を探したりする必要がある場合にクライアントに相談する場合に関連します。
  • CRMまたは一部のテキストソース（サイト）からのコピーによるダイヤル。
  • コンパクトさ。自宅からオフィスに旅行するときに重要です。






ただし、否定的な点がありました。

  • 絡まり、ぶら下がり、あらゆる方法で細かく緊張したワイヤ。
  • 場所への取り付けは、ワイヤーの結果です。
  • コンピューターとソフトウェアへのバインド。 通話または発信通話を受け付けるには、実行中のソフトウェアと目に見える画面が必要でした（ホットキーがソフトフォンに実装されることはほとんどありません）。
  • 柔らかい！ さまざまなプログラムがありますが、作業中のIP PBXとの信頼できる通信に必要な機能を備えたユニットがありました（機能...）。 そして、それらのうち、許容できる人間工学（X-Lite for Mac）を備えたものは1つだけです。 そして、プログラムはしばしば「クラッシュ」しました:-(
否定性は次第に肯定的な特徴を確実に上回るようになりました。



最初の試み。 Bluetoothヘッドセット
Bluetoothを備えたラップトップを使用して、少なくとも何回かワイヤーを外したいという要望がありました。 ただし、試してテストしたすべてのBluetoothヘッドセットでは、十分な音声品質が得られませんでした。 メーカーからの約束の山にもかかわらず、安価なヘッドセット（700ルーブル）と高価なヘッドセット（4000ルーブル）の両方が「デジタル」歪みによって明確に定義されていました。



本当に高品質の音声品質を備えたDECTヘッドセットを試してみたいという要望がありましたが、最も単純なオプションの7000ルーブルの価格は止まりました。



さらに、ヘッドセットはコンピューターとソフトウェアに「結合」を残しました（！）。



2回目の試行。 携帯電話のソフトフォン
iPhoneは良いことですが、実験中にインストールされた多くのソフトフォンのうち、動作中のIP PBX（Zoiper）で動作したのは1つだけです。 残りは、未知の理由により、これを行うことができませんでした。 たとえば、テストされたソフトフォン（3cx）の1つは、地域オフィスの1つにインストールされた「サブPBX」で動作することを望んでいませんでした（コーデックに問題がある可能性があります）。



私は夜にソフトフォンをオフにするのを忘れて、彼の携帯電話のバッテリーを放電した後、実験はすぐにそして永遠に終了しました。 朝はアラームが鳴らず、重要な会議に非常に遅れました。



最後の試み。 IP DECTコードレス電話
無線管のDECTファミリには、PSTNと、SIP経由でLANを介してPBX /サービスの両方に接続できるベースユニットのオプションがあることを偶然発見しました。

さらに、私はシーメンス（ギガセット）からのこうした「パイプ」に関する多くの肯定的なレビューを見つけました。 仕上がりの品質と音質、信頼性、多かれ少なかれ動作しているソフトウェアの詰め込み。



プレゼント



非常にリーズナブルな価格のコードレス電話DECT Gigaset C610A IP



基本：

  • PSTN回線とLANに接続されたベースユニット+充電用の個別の「ガラス」=家の周りの最も便利な部品の配置
  • 異なる設定の6つのIP回線
  • 私に必要な設定。 例：

    • 適切なコーデックの選択（PBXの動作はu-Lawでのみ機能します）。

      利用可能：G.722、G.711 a-law / G.711 u-law、G.726、G.729
    • ルータでさらに「転送」するためのSIPおよびRTPポートの「手動」表示
  • ワイヤーの欠如-直線で300メートル、家で50メートル
  • ラップトップとソフトウェアの分離
  • 有線ヘッドセットを接続する機能（「ハンズフリー」を返す）


「パン」：

  • 留守番電話
  • ウェブインターフェース
  • 回線のダイヤルプラン（たとえば、「9」はPSTN用、「8」はIP 1用）
  • スタンバイモードで180時間、トークモードで12時間（パスポートデータ）
  • ベビーモニターモード-将来のために:-)


どうした
  • ルータの近くの「ベース」はどこか遠くにあり、すべてのワイヤがあります。
  • 4ライン充電ハンドセット：
    • メインの内線番号+市内（モスクワ）および長距離/国際チャネルでの発信コールを使用したメインオフィスのIP PBX
    • ノボシビルスクでの発信通話のためのIP PBXノボシビルスクオフィス（頻繁な作業が必要）
    • 友人への長距離電話のためのsipnet.ru
    • 自宅の電話番号（PSTN）
  • （選択した回線のいずれかを介して）リモートで（コンピューターから）番号をダイヤルして、発信者が表示された状態でコールを受信する可能性。
  • データをvCardにインポートする電話帳-Macアドレス帳から仕事の連絡先をエクスポートし、Webインターフェイス経由で電話に持ち込みました。
  • コンピュータの電源を入れても、画面の電源が入っても、「正しい」ソフトウェアでも問題ありません。




未来



このハンドセットに接続されている有線ヘッドセット（0.5メートル以内のコード）を見つけて、「フリーハンド」を返す必要があります。



PSモビリティ



自宅からオフィスへ、またはその逆に持って行くと、ベース、グラス、チューブ、2回の充電が多くかかります。 しかし、オフィスに別の拠点を置き、電話だけを持ち歩くには、これは非常に便利です。 ハンドセットは「スマート」で、それが置かれているベースに静かに接続します。
