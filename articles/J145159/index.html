環境にやさしく、エネルギー効率の良い京セラ印刷技術

今日、怠laでない人は皆、環境に優しい技術への移行について話している。 しかし、彼らが言うことではなく、彼らが何をするかを見るのは常に興味深いです。 独自の技術により、京セラは廃棄物の発生量を大幅に削減することができました。 どうやって？

まず、京セラはカートリッジを使用しない印刷技術を開発しました。

第二に、コンポーネントの寿命が延びました。





京セラのデバイスでは、カートリッジは個別に固定されており、デバイスの寿命を通じて交換が必要な消耗品はトナーのみです。 従来のカートリッジが60個を超えるコンポーネントから組み立てられている場合、京セラのトナーコンテナは4個だけで構成されています。さらに、従来のレーザープリンターのカートリッジは交換する必要があり、ドラムやその他のコンポーネントがまだ寿命に達していなくても、トナーコンテナは空です。 京セラのECOSYSテクノロジーは、「オプションの」廃棄物の発生を回避し、最終製品の製造および輸送中の二酸化炭素排出量を削減します。



従来の60成分カートリッジ



京セラトナーコンテナ



さらに、TASKalfa *シリーズの最新のカラーおよび白黒高速MFPは、従来の定着器ランプの代わりに誘導加熱定着器を使用します。 誘導加熱を備えた定着器を使用すると、デバイスのウォームアップ時間が短縮されるだけでなく、消費電力も削減されます。 これは、ランプが誘導加熱器よりも多くのエネルギーを消費するという事実によるものです。 これにより、京セラでの印刷はエネルギー効率の高いプロセスになります。



* TASKalfa 6500i / 8000i、TASKalfa 3050ci / 3550ci / 4550ci / 5550ci / 6550ci / 7550ci
