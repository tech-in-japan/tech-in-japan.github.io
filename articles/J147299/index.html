Skype Connectの最初の知り合い

エントリーの代わりに。

私は初めてSkype Connectを扱っていますが、これは私の個人的な経験であり、私のような人々を助けたいと思っています。



期待

Skypeがsipで接続できるというニュースを見たことがありましたが、一時は無料のようでしたが、その後、費用がかかり始めました。 SIPテクノロジーの研究に触発されて、私はこのサービスを自分で取得しようとすることにしました。 広告が掲載されているセクションに入った後、実行するアクションの順序をすばやく調べました。つまり、次のとおりです。

1. Skype Managerにログインします

Skype Managerから：

2.通話チャネルの料金を支払い、SIPプロファイルのアカウントにお金を入金します。

3. SIPプロファイル情報を使用して、会社のPBXまたはVoIPゲートウェイを構成します。

4. Skypeを使用するために、SIPプロファイルにお金を割り当てます。

5.通話を開始します。



「Skype Managerから」という太字の碑文を見ずに、端末が私の支払い方法に最も近いため、2番目のポイントを実装することにしました。 Skypeアカウントのアカウントに10ユーロを投入したことを考え直さずに、コミュニケーターの設定でSkype sipアカウントを作成することを既に楽しみにしていたので、すべての友人が通常の電話のように静かに電話をかけることができます。 しかし、幸福感は長続きしませんでした。 コンピューターに座って、Skype ManagerのアカウントとSkypeの個人アカウントが互いに接続されていないことがわかりましたが、最初から2番目にお金を移すことができますが、逆の手順はありません...



試行番号2

私がこれらの10ユーロについて考えたことは間違っていました-私は最も貴重なもの-時間を失いました！ Skype Managerでの入金方法を選択した後、私は再び支払い端末に戻りました。 そして、彼はすぐにそれが機能するようにすべてを設定することを望んで、コミュニケーターに視線を下げました。 そして、ユーザー名とパスワードを見て、コミュニケーターに入力して、大切な機能をテストすることを決め、親relativeに私のSkype（当時はオンラインではありませんでした）に電話するように頼みましたが、奇跡は起こりませんでした。

Skype Webサイトをさらに掘り下げて、標準プログラムを介してSIPチャンネルに接続されたスカイプアカウントを使用することは不可能であるという碑文に出会いましたが、驚いたことに、すべてが静止したコンピューターで動作しました...



稼いだ！？

少し後に「ビジネスアカウント」の概念を発見し、これがまさに私が必要とするものであることに気付きました。 これで、アカウントからビジネスアカウントを作成し、電話を受けることができるようになります。 いいえ、なぜですか？ ええ、はい、チャンネルは私のアカウントに接続されています、スカイプは新しいものを始めることを申し出ます。 アカウントを探しています。Skypeのすばらしい機能についてメールで手紙を受け取ります。そして、着信コールに添付された新しく作成されたアカウントへのコールを受け取ります。

はい、彼らは素晴らしいですが、なぜ別のSkypeアカウントが必要なのですか？ 私は現在のもののために13ユーロを持っていますが、Skypeのウェブサイトによると、この種のクライアントはプログラムを切断しても、それにアクセスできません。

現時点では、私は目標を達成しました。通話は自分のSkypeアカウントに送信されていますが、希望のアカウントには送信されていません。 追加のサービスを購入せずにSkypeからSkypeに転送すると、見つかりませんでした。 Skypeにステータスを入れて、私が別のアカウントを持っていることを通知してください。



結論

書かれたすべてを注意深く読んでください-これが主なことです。

サイトで厳密に指定された順序で接続します。つまり：

1. Skype Managerにログインします

2. Skypeアカウントにはお金を入れませんが、Skype Managerアカウントにはお金を入れます

3. Skype Connectサービスの支払い

4.着信通話または発信通話用のアカウント（ビジネスアカウント）を作成する

5.彼をサービスにバインドします

6.使用します！
