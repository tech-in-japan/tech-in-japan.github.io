FreeDNS.afraid.orgのスクリプト



似たようなスクリプトがいくつか見られましたが、制限が異なるため気に入らなかったので、完全に自分に合った自転車を作ることにしました。



基礎として、私はLESHIYODESSAからスクリプトを取りました 。 動的DNSレコードの現在のアドレスを保存するためにファイルが使用され、定期的に解析される彼のアルゴリズムはあまり好きではありませんでした。さらに、スクリプトは異なるレコードの更新をサポートしていません。これは、スクリプトを伝播することを提案していますが、これは特定のIPのレコードを更新する問題を解決しません住所へ。 そのため、実際には、いくつかのエントリとは無関係に、DHCP経由で受信したインターフェイスの監視対象IPアドレスを変更するために、定期的な更新（1時間間隔で）を行う信頼性の高いメカニズムでファイルを使用して作業を置き換える独自のスクリプトを作成しました。



FreeDNS.afraid.orgのサブドメインの名前とそのハッシュ、WANインターフェイスの名前の配列を宣言し、そこからIPアドレスを追跡します。 また、レコードの数（Quant）を配列のサイズまたは手動で設定します。 



 :local SubdomainHashes {"U3dWVE5V01TWxPcjluEo0bEtJQWjg5DUz=";"U3pWV5VFTWxPcjlOEo0EtJpOE1MAyDc="} :global DNSDomains {"aaa.xyz.pu";"bbb.xyz.pu"} :global WANInterfaces {"ether4-WAN-Inet";"ether3-WAN-Beeline"} :global Quant [:len $DNSDomains]


ヘルパー変数を宣言します：



SkipCounters-監視対象インターフェースの検証カウンターの配列

LastIPsは、FreeDNSに既に送信されているIPアドレスの配列です。



カウンタの配列を使用すると、ダイナミックDNSレコードを互いに独立して更新できます。



まず、空の変数宣言を作成します：global SkipCounters 、このような宣言により、新しいグローバル変数を作成するか、環境変数とその値で既存の変数を上書きせずに使用できます。



次の構成は、新しく作成された変数に対して機能し、データ型が配列でない場合にチェックされ、型が配列に変更され、変数の値が割り当てられます。 したがって、出力では、必要な値で初期化された配列型変数があります。 



 :global SkipCounters :if ([:typeof $SkipCounters] != "array") do={ :set SkipCounters {""} :for i from=0 to=($Quant-1) do={:set ($SkipCounters->$i) 1} } :global LastIPs :if ([:typeof $LastIPs] != "array") do={ :set LastIPs {""} :for i from=0 to=($Quant-1) do={:set ($LastIPs->$i) ""} }


追跡更新アルゴリズム自体もありません。



dhcp-clientから現在のIPアドレスを取得します。 その後、最も興味深い。



[/ ip dhcp-client get [find where interface =（$ WANInterfaces-> $ i）] address]コマンドは通常、IPアドレスを含む文字列以外のすべてを返すことができるため、通常はCurrentIP変数の値を更新します。 返される値は、IPまたはnilの文字列のいずれかです。または、ランタイムエラーが発生し、コマンドはCurrentIPを更新しません。 したがって、上記の明示的な宣言行を入力します：local CurrentIP "" 。 そして、コマンドを実行した後、CurrentIPは ""、nil、またはIPアドレスを持ちます。



上で書いたように、演算子：lenの安定性が最も高いので、受信データの妥当性をチェックするためにさらに使用します[：len $ CurrentIP]> 0 。 また、カウンタの値を監視し、> = 60の場合、リクエストを強制的にFreeDNSに送信します。 したがって、通信の問題に対するアルゴリズムの安定性が向上します。 シェダーでスクリプトを1分に1回実行するので、必須の更新期間は約1時間で、FreeDNSサービスに大きな負荷はかかりません。



他に注意を払う価値があるもの。 パラメーター "＆address ="は、更新要求のURLに存在します。$ CurrentIP、このパラメーターを使用すると、自動アドレス（要求の送信元のインターフェイス）ではなく、サブドメインのIPアドレスを明示的に指定できます。 



 :for i from=0 to=($Quant-1) do={ :local CurrentIP "" :set CurrentIP [/ip dhcp-client get [find where interface=($WANInterfaces->$i)] address] :set CurrentIP [:pick $CurrentIP 0 ([:len $CurrentIP]-3)] # :log info ("Current SkipCounter$i: ".($SkipCounters->$i)) :if ([:len $CurrentIP] > 0 and ($CurrentIP != ($LastIPs->$i) or ($SkipCounters->$i) > 59)) do={ :if ($CurrentIP != ($LastIPs->$i)) do={ :log info ("Service Dynamic DNS: Renew IP: ".($LastIPs->$i)." for ".($DNSDomains->$i)." to $CurrentIP") } /tool fetch url=("http://freedns.afraid.org/dynamic/update.php\?".($SubdomainHashes->$i)."&address=".$CurrentIP) keep-result=no :set ($LastIPs->$i) $CurrentIP :set ($SkipCounters->$i) 1 } else={ :set ($SkipCounters->$i) (($SkipCounters->$i) + 1) } }
