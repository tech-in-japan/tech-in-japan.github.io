DreamSparkは教師に開放されています

ともだち！ ロシアの学校や大学の教師がDreamSparkプログラムを利用できるようになったことをお知らせいたします。 公式発表は、MS Press Blogにあります。







DreamSparkプログラムでは、12歳以上の学生、大学院生、学童、さらには教師が非営利的な使用（研究、科学など）のためにMSツールに無料でアクセスできることを思い出させてください。



このツールキットは常に新しい製品で更新されており、現在は次のものが含まれています。

  • 開発および設計ツール （Visual Studio 2010、Robotics Developer Studio 2008 R3、Expression Studio 4など）
  • サーバー製品 （Windows Server 2008 R2、SQL Server 2008 R2）、
  • 教育向けの他の多くの興味深いプログラム（たとえば、Kodu Game LabやSmall Basic）。


また、プログラム参加者は、Microsoft 認定試験を無料で受ける （6月30日までまたはバウチャーが終了するまで）、 PluralSightトレーニングコースを受講し 、 WP7アプリケーションを無料で（有料のものを含む）マーケットプレイスに投稿することができます 。



プログラムへのアクセス方法の詳細については、 dreamspark.ru / access.aspxをご覧ください。
