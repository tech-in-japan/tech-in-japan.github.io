比較して人気のパスワードマネージャー

安全なパスワードストレージは、特に最近の大規模なサイトの注目を集めるクラッキングの後、いつでも非常に重要なトピックです。 MtGox交換をハッキングした後、パスワードの1つがネットワークに漏洩した後、深刻な保護方法への切り替えが心配になりました。



パスワードを頭や紙に保存する多くのユーザーが無視している最も重要な点は、各サービスで個別のパスワードを使用することと、簡単で覚えやすいパスワードを拒否することです。 これを可能にするために、品質の異なるかなりの量のソフトウェアがあり、その研究を取り上げました。 今、私は私の研究の結果を共有したいと思います。



したがって、私の選択に該当するプログラムは次のとおりです。KeePass、eWallet、LastPass、1Password、RoboForm 。 誰が気にします-猫へようこそ。



事前に警告します。このレビューは完全または100％の信頼性を主張するものではありません。 読者に提示されたプログラムのかなり完全な印象を与えたいので、そのうちの1つを使用する決定を下すことができます。



それでは始めましょう。



KeePassパスワードセーフ



最初の申請者は、オープンソースのパスワードマネージャーKeePassです。 GPL v2ライセンスの下で無料で自由に配布されます。 主に2つのバージョンがあります。Windowsでのみ動作する「古い」1.xと、.NETで記述され、OS XおよびLinuxのMonoを含む「新しい」2.xです。 両方のバージョンがポータブルバージョンに存在します。 LinuxおよびMac OS X用のKeePassXなど、 KeePassデータベースで動作するサードパーティプログラムもあります。



パスワードデータベースは、AES-256で暗号化され、Dropbox、フラッシュドライブ、その他の便利な手段で同期できるファイルに保存されます。 多方向キー変換を使用することができます。これにより、データベースの暗号化解除に必要な時間が増加します。 これにより、ブルートフォース攻撃に対する耐性が高まります。 他のプラットフォームの一部のクライアントは、Dropboxのデータベース（iOSのKyPassなど）を直接操作できます。 バージョン2.xのデータベースは1.xと下位互換性がないため、古いバージョンのデータベースで動作する多くのサードパーティソフトウェアで問題が発生します（ただし、新しいバージョンから古い形式のデータベースをエクスポートできます）。



KeePassには、ブラウザやその他のプログラムでパスワードを自動的に入力できるオートタイプ機能が組み込まれています。 KeePassには、すべての主要なブラウザー（IE、Firefox、Chrome）との緊密な統合を含む多くのプラグインもあり、多くの追加機能を提供します。



前述のように、KeePassのオープン性により、さまざまなプラットフォーム向けに多くのソフトウェアが作成されています。 モバイルデバイスでは、iOS、Android、WM Classic、Windows Phone 7、Blackberry、およびJ2MEのプラットフォームにKeePassクライアントがあります。 プラグインおよびサードパーティソフトウェアのより詳細なリストは、KeePass Webサイトで入手できます。



eWallet



eWalletは、 IliumSoftwarの有料パスワードおよび個人情報マネージャーです。 eWalletは、WindowsおよびMac OS X（9.99ドル）のバージョンに存在し、iOS、Android（ビューアのみ）、BlackBerry、およびWindows Mobile Classicのクライアントも持っています。



KeePassと同様に、データベースファイルはAES-256を使用して暗号化されます。 データはローカルに保存され、eWalletによるクラウドストレージは提供されません。



デスクトップコンピューター間のデータベース同期は、手動転送によってのみ可能です。 Windowsバージョンは、組み込みのプラットフォーム同期（それぞれActiveSyncおよびBlackBerry Desktop）を使用して、WM ClassicおよびBlackberry上のモバイルクライアントと同期されます。 iTunesとWi-Fiを使用して、MacのバージョンとiOSのバージョンを同期できます。



eWalletのWindowsバージョンは、Internet Explorer、Firefox、Chromeと統合されます。 OS Xバージョンは、Safariとの統合のみを提供します。



1パスワード



1Passwordは、パスワード、ソフトウェアライセンス、その他の個人情報をMac OS Xに保存するための人気のあるAgileBitsソリューションです。 最近、Windows用のバージョンもリリースされ、iOS用のネイティブクライアントも提供されています。 プログラムは非常に高価です-WindowsとMac OS Xのバージョンは39.99ドル、または両方とも59.99ドルです。 iOSバージョンはAppStoreで14.99ドルで入手できます。 読み取り専用のAndroidアプリケーションは無料です。



1Passwordのすべてのバージョンには、Dropboxサービスを使用したデータベース同期機能が組み込まれています。 この機能はオプションであり、データベースはデフォルトでローカルに保存されます。 データベースはAES-128によって暗号化されます。 ブラウザおよびオペレーティングシステムとの組み込みの統合により、キーロガーを介したパスワードの漏洩が防止されます。



Mac用1Passwordは、Safari、Firefox、Chrome、およびCaminoとすぐに統合できます。 Windowsバージョンは、Firefox、Chrome、およびIEと統合されます。 1Passwordの両方のバージョンは、他のアプリケーション（KeePassに似たAutoType機能を含む）に保存された情報を使用するための便利なインターフェースも提供します。



さまざまなプラットフォームとの統合に加えて、1Passwordはそのデータベースにアクセスする別の独自の方法を提供します。 パスワードストレージ（アジャイルキーチェーン）は一連のファイルです。その1つは、データベースを操作するための完全なインターフェイスを備えたHTMLファイルで、ほとんどすべてのデバイスのブラウザーで開くことができます。



顧客評価1Password for iOS-5つ星のうち4つ星は、考慮されるすべてのモバイルプログラムの中で最高です。 多くのレビューは、ユーザーフレンドリーなインターフェースとブラウザ統合のためにOS X上の1Passwordを称賛しています。



ロボフォーム



RoboFormは、この市場で最も古いプログラムの1つであり、Palm OSおよびWindows Mobile 2003の作業バージョンがまだ残っている唯一のプログラムです。RoboFormFreeの無料バージョンは、WindowsおよびMac OS Xで使用できますが、かなり限られています。 RoboForm Desktopの有料版（29.95ドル）は多くの制限を取り除きます。 しかし、最も興味深いのはRoboForm Everywhereパッケージ（年間19.95ドル）で、WindowsおよびMac OS X用のデスクトップバージョン、FirefoxおよびChromeと完全に統合するためのプラグイン、およびすべてのバージョン間の自動クラウドベースのデータベース同期をフルに使用できます。



RoboFormデータベースはAES-256標準を使用して暗号化され、プログラムのすべてのバージョンでローカルコンピューターに保存されます。 RoboForm Everywhereを使用する場合、データベースはRoboFormサーバーにもあります。



メジャーバージョンに加えて、RoboFormは多くのモバイルプラットフォーム用のアプリケーションを提供します。 これには、iOS、Android、BlackBerry、Windows Mobile（6.x、5、2003、さらにはPocket PC 2000および2002）、Palm OSおよびSymbianが含まれます。 iOSおよびAndroidバージョンはクラウド同期をサポートしており、RoboForm Everywerサブスクリプションが必要です。 他のすべてのモバイルバージョンは、追加のソフトウェアを使用してデスクトップバージョンと同期されます。



また、RoboFormは、Windows、Mac OS X、およびLinux上のOperaブラウザー用の別個のプラグイン（または、さらに2つ）を備えたレビューの2つのプログラムの1つにすぎません。



ラストパス



LastPassは、非常によく知られているクラウドベースのパスワードストレージサービスです。 LastPassの基本バージョンは無料です。 プレミアムパッケージは月額1ドルです。



LastPassは、おそらくこのレビューで最も幅広い機能を備えています。 このサービスは、すべての主要なブラウザー（IE、Firefox、Chrome、Opera、Safari）上のWindows、OS X、Linuxで利用できます。 LastPass for AppsはWindowsでも利用でき、どのアプリケーションからでもパスワードを自動的に保存できます。 パスワードデータベースの管理は、LastPass WebサイトのWebインターフェイスからも可能です。 Windowsの場合、バックアップストレージおよびオフラインで使用するためにデータベースをダウンロードできるポータブルクライアントがあります。



LastPassはクラウドサービスであるため、データベースは常にLastPassサーバーに保存されます。 同期自体は必要ありません。 利便性に加えて、データベースをサーバーに保存することもリスクです：少し前に、LastPassがハッキングされ（噂による）、サービス所有者は多くのクライアントがマスターパスワードを変更することを提案しました。 レビューの他のほとんどのプログラムと同様に、LastPassデータベースはAES-256で暗号化されています。



LastPassは、iOS、Android（Dolphin HDおよびFirefox Mobileブラウザー用の追加アプリケーションを含む）、WM Classic、Windows Phone 7、BlackBerry、HP / Palm WebOSなどのモバイルデバイス用の幅広いクライアントを提供します。 LastPassのすべてのモバイルバージョン（iPadを除く）には、LastPass Premiumサブスクリプションが必要です。
