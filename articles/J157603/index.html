I-それほど素朴ではないことが判明しました。これは、D（ジャンプ数）* H（収益化係数）です。 Hについては別に説明しますが、今のところDについて心配しています。Dはすでにはるかに複雑です。



D =（A1 + A2 + A3 + A4）* B * C * H * J +（E * E1）+（F * F1）

  • （A1 + A2 + A3 + A4）* B * C * H * J-オーガニックトラフィック
  • （E * E1）-PRおよび広告
  • （F * F1）-インストールの購入




オーガニックトラフィックは、特別なマーケティング活動なしで、アプリケーションが単独で稼ぐお金です。 この式を見てみましょう。 B、C、およびHは乗数であり、1未満です。 彼らができることは、観客を広めることだけではありません。 Jは、双方向のコミュニケーションのため、潜在的な哲学的石ですが、それについては後ほどです。



Aのまま-私たちが得ることができる基本的な観客。 これらは私たちの顧客ではなく、私たちの訪問者でさえありません-これらは私たちのサインを見た人々です。 看板の表示回数を増やすことで、Aにどのように影響を与えることができますか？ 明らかに-DとIを増やします！ Appleは苦労せずに、標準のフィードバックループを使用しました。



A1とA2は、管理のための最もアクセスしやすいオブジェクトです。 まだトップに到達していない場合は、A4しか残っていません。これは率直に言って小さいです。 Angry Zombies対Alien Doodle Plantsの精神で特別に作成された「偽の」アプリケーションを使用しても、1日に数百のビューについて話します。

Total Freeで最高のトラフィック。 その後-合計有料。 後-トップカテゴリー。 ビューの数が場所に応じて比例して減少することが重要です。 「ビュー」とトラフィックソースの正確な統計は、7つのシールの背後にあるAppleの秘密ですが、常識に基づいて、いくつかの行が表示されます。



  • トップ1-人々は無意識のうちに最高を好むように配置されています。 したがって、最初のアプリケーションは、他のすべてのユーザーに関連して常にダウンロードボーナスを受け取ります。
  • トップ2-4（6 iPad）-ここに到達すると、AppStoreにアクセスするすべてのユーザーにアプリケーションが表示されます。 トラフィックは非常に標準以下ですが、多くのトラフィックがあります。
  • トップ4-8（6-12 iPad）-このページを表示するには、ユーザーは選択したカテゴリのアプリケーションを「フリップ」する必要があります。 これにより、ユーザーの大部分がすぐに遮断されますが、少なくとも何かを購入することを考えている人だけが閲覧しているため、トラフィックがよりきれいになります。
  • トップ8-10-iTunesのページに表示されるトップ10。 そして、デバイスからではなくプログラムからダウンロードする人の割合は大きくありませんが、これは依然として重大なマイルストーンです。
  • トップ10-100-後続の各ページで、アプリケーションを見るユーザーの数は減少します。 20位以降のどこかで、ドロップの速度は低下し、ページを表示する人の数はそれほど減少しません。
  • Top 100-200-100箇所未満、アプリケーションを表示するには、iPhoneで「すべてを見る」リンクをすでに開いている必要があります。 これにより視聴者が再び減ります（裏返すことと、リンクをタップすることです）。 200の場所の後に、iTunesのトップから飛び出します
  • Top 200-300-すべて、iPhoneのトップから飛び出して見えなくなった

（E * E1）-PRおよび広告


Facebookの壁に叫んだ記事、レビュー、ブログ投稿を書いたとしましょう。 これらのメッセージのそれぞれは、特定の数の人々によって見られます-これはEです。そして、それらのいくつかは、アプリケーションをダウンロードするためのリンクをたどります-それはE1です。



多数のレビューと彼ら自身の経験によると、アプリケーションについて書いている専門サイトで最も効果的な出版物です。 質の高いレビューの場合、Eはサイトのオーディエンスの20％にリーチできます。 つまり、1日あたり100,000人のユニークビジターがサイトにアクセスした場合、メインページに印刷されたレビューは20,000回表示されます。



無料のアプリケーションの優れたE1（変換）レートは20％です。 したがって、人気のあるサイトのスーパーレビューでは、4,000の競馬が行われます。 有料アプリケーションの場合、E1は大幅に低くなります（レビューの価格と品質に応じて1％-5％）。



（F * F1）-トラフィック購入


違法な方法（ボットネットなど）を採用しない場合、無料アプリケーションのインストール（CPI）の価格は10 USセントから始まります。 ただし、実際には、米国の通常の価格は、トラフィックの品質と使用する方法に応じて1〜2ドルです。



最初の3日間で40,000のインストールを取得するには（Total TOP 100 USおよびTop 10 USカテゴリに入れる）、インストールを購入するために10件のレビューまたは5件のレビューと20,000ドルが必要です。 あなたが接続を持っている場合、アプリケーションは美しく作られており、それは良いアイデアを持っています-レビューはあなたのために少しのお金または無料で書かれます。 これらがすべて揃っていない場合、有料のレビューにはインストールの購入に匹敵する費用がかかります。



ほとんどの出版社は、確立された連絡先、名前（高品質のレビューを取得するのに役立ちます）、低インストールコスト（独自のアプリケーションオーディエンスとインストールの一括購入を使用するため）の恩恵を受けます。
