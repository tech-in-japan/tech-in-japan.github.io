iLoaderブートローダー経由でRockBoxを搭載したiPod nano 2 gen

エントリー



読者の皆さん、ご挨拶。

本日、 Appleプレーヤーを近代化します-Ipod Nano 2 gen



より最近では、2009年10月に、代替ロックボックスファームウェアWebサイトに、 第2世代Ipod Nanoプレーヤーへの移植の成功に関する情報が掲載されました。







ウィキペディアからの抜粋は次のとおりです。

Rockboxは、デジタルオーディオプレーヤーのファームウェアを置き換える無料のソフトウェアです。 Rockboxプロジェクトは2001年末に始まり、最初にArchos Studioプレーヤーに実装されました。

Rockboxは、多くの場合、元のファームウェアを削除せずに、プレーヤーのファームウェアに代わるものを提供します。 Rockboxの内部デバイスは、拡張モジュールを使用することで機能を強化できます。 機能の改善には、アプリケーション、ユーティリティ、およびゲームが含まれます。 Rockboxは、2000年代半ばにリリースされたプレーヤーにビデオ再生（後付け）を提供できます。 Rockboxには、失明や視覚障害のある人が使用するのに適した音声駆動のユーザーインターフェイスもあります。


私のメインプレーヤーが壊れただけで、私はそれに適した代替品を選択していました。

ためらうことなく、私はebayに行き、60ドルのiPod nano 2 gen 8Gbを注文しました







プレーヤーがチェリャビンスクに旅行している間に、私はインストールオプションの研究を始めました。





作業計画



1） ILoaderブートローダーをインストールする

2） RockBoxを注ぐ

3） 元のファームウェアを注ぐ

4）ダウンロード時に、必要なソフトウェアを選択します



ソフトウェアとファームウェアを準備します



まず、 Linux PCが必要です。

Linuxがインストールされていない場合は、任意のLive CDを入手できます



ダウンロードする



1） iLoaderブートローダー

2） iPod nano 2 genの最新のRockBoxリリース

3） このサイトの iPod nano 2 Gのオリジナルファームウェア

そこで、ドロップダウンメニューでデバイスを選択し、[ ダウンロード ]ボタンをクリックします。

iPod_29.1.1.3.ipswのバージョンをダウンロードしました

4） ファームウェア用のDumperをダウンロードする



iLoaderをインストールする



1）ipodをPCに接続し、/ devディレクトリでドライブの名前を確認します（ / dev / sdd {1,2}があります ）

2） 7z e iLoader-fullfs.7zコマンドを使用してiLoader-fullfs.7zアーカイブを解凍します

3）iPodモデルに必要なファイルを選択します 。

私は8Gbモデルを持っているので、ファイルiloader-image-8gb.binを取りました

4）ipod コマンドのブートローダーをインストールします

dd if = iloaderimage-Xgb.bin of = / dev / sdX; 同期する


ここで/ dev / sdXはipodドライブ番号です（私の場合/ dev / sdd ）

5）iLoaderを更新する必要がある場合は、 次のコマンドで実行できます

dd if = iloaderimage-Xgb.bin bs = 2048 skip = 63 seek = 63 count = 63 of = / dev / sdX; 同期する


6）完全に削除する必要がある場合は、iPodをiTunesに接続し、リカバリプロセスを実行する必要があります。



iLoaderを使用する





ここではすべてが非常に簡単です。必要なファームウェアをディスクにコピーし、電源を入れたときに選択するだけです。



メニューボタン ： iBugger （/iLoader/ibugger.bin）をロードします

iBuggerは、iPodのファームウェアを開発およびテストするためのファームウェアです。

左ボタン ： 元の Ipodファームウェア（/ iLoader / appleos.bin （復号化）または/ iLoader / osos.fw （暗号化）

中央のボタン ：ドライブモード

再生ボタン ： RockBox （/. rockbox /rockbox.ipod）をロードします

右ボタン ：サードパーティのファームウェア（/iLoader/custom.bin）をダウンロードします

これはおそらく、 iPodLinuxの下で将来的に行われます。







Rockboxのインストール



1） rockbox-ipodnano2g.zipアーカイブを解凍します

2） .rockboxフォルダー全体をドライブ/ dev / sdX2のルートにコピーします

3）PCからiPodをアンマウントして取り外します

4） 再生ボタンで対応するメニュー項目を選択し、使用します



元のファームウェアのインストール



1） extract2g.i386を実行可能にします（chmod + x extract2g.i386）

2）ファイルの名前をiPod_29.1.1.3.ipswからiPod_29.1.1.3.zipに変更し 、解凍します

3）コマンドを与えます：

extract2g.i386 -Aファームウェア-29.8.1.3


そして、3つのファイル（osos.fw、aupd.fw、rsrc.fw）を取得します

4）コマンドを与えます：

dd if = osos.fw of = osos.out bs = 2048 skip = 1


5） Ipodのosos.outファイルをiLoaderフォルダーにコピーし、名前をosos.fw （/iLoader/osos.fw）に変更します

6）iPodをPCからアンマウントして取り外します

7） 左ボタンで対応するメニュー項目を選択し、使用します
