DataTalks＃2：データの可視化

DataTalksコミュニティの2回目の会議は、データの視覚化に専念しました。これは、アナリストの手にとって最も重要で強力なツールの1つです。 結局のところ、顧客の決定と結論は、研究作業の結果がどのように提示されるかに依存します。 さらに、視覚化は予備的なデータ分析を行い、新しい関係をすばやく見つけるのに役立ちます。

カットの下には、Wargaming、Ambitious Suits Consulting、Targetprocess、A2 Consulting、およびインターネットポータルTUT.BYの代表者のスピーチの録音があります。 スピーカーは、視覚的で有益なデータのプレゼンテーションの経験を共有しました。

ミーティング自体は、伝統的に「戦車の下」で、Wargaming開発のミンスクオフィスであるGame Stream JLLCのメインオフィスで開催されました。







BI QlikView / Alexander Gonchar / A2 Consultingを使用したビジネス向けデータの視覚化

Alexanderは、ベラルーシのさまざまな企業でQlikView BIシステムを使用する例を示しました。 QlikViewを使用したデータの分析と視覚化は、A2 Consultingのお客様がオンラインでビジネスの状態を監視するだけでなく、収益性を高める方法を見つけるのに役立ちます。





データの視覚化：RおよびTableau Desktop / Alexey Yurkevich、Dmitry Shchegrikovich / Wargaming

AlexeyとDmitryは、RおよびTableau Desktopアプリケーションを使用してデータを視覚化した経験を共有しました。 Rの話で、アレクセイは特別な準備なしでデータの視覚化を開始できるいくつかのパッケージを強調しました。

Dmitryは、Tableau Desktopを使用してデータを視覚化することの利点について説明し、データを顧客に提示する際に起こりうるいくつかの一般的な間違いについても警告しました。



インタラクティブで静的なインフォグラフィック。 多数の聴衆によるデータの認識の特徴/ Vadim Shmygov / TUT.BY

Vadim Shmygovは、彼の最高の作品の小さな回顧展を行い、TUT.BYポータルのほぼ400万人の視聴者のために直感的で関連性のあるインフォグラフィックを作成するために使用するツールとデータについて話しました。





アナリティクスプレゼンテーションの問題点/ビクタースムシン/アンビシャススーツコンサルティング

ビジネスに携わる各アナリストは、自分のアイデアや結論を顧客に正しく、明確かつ明確に伝えることができなければなりません。 ビクターは、基本的なルールについて話しました。これを遵守すると、プレゼンテーションがより効果的になります。





意思決定における視覚的手法/ Natalya Yadrentseva / Targetprocess

データの視覚化パターンがあります。 しかし、それらのアプリケーションは常に望ましい結果につながりますか？ ナタリアは、データを提示する従来の方法が情報の認識を妨げる可能性がある例をいくつか示し、より便利でシンプルな統計を提示するための代替オプションも提示しました。
