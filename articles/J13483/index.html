Mail.Ru Agent 4.10はスパムからあなたを救います

8月23日にリリースされたエージェントのmail.ruの新しいバージョンは、インスタントメッセージの形式でスパムに対抗するように設計された新しい機能で強化されています。



許可されていないユーザーからのメッセージは新しいバージョンでブロックされ、「疑わしい」フォルダーに保存されるため、後で何らかの理由で必須の承認手順に合格しなかった実際のユーザーからのメッセージからスパムを分類できます。



また、スパム対策システムに「ヒューマンファクター」が導入されました。スパマーから明確に受信したメッセージについて苦情を申し立てることができるようになり、Mail.Ru Agentのサーバー部分が不要な大量メールをより効果的に処理できるようになりました。



許可されていないユーザーから手紙で受け取ったリンクはクリック可能ではなく、コピーさえされていないため、スパムメッセージの意味が無効になります。



Agent 4.10は、Mail.ruとともに、Sputnik Mail.Ruという小さなプログラムを提供します。これは、Internet Explorerにツールバーの形で組み込まれています。 ツールバーには、インターネット上の必要な情報をすばやく検索できる検索フォームが含まれています。 また、Sputnik Mail.Ruを使用すると、Mail.ruプロジェクトにすばやくアクセスできます。
