Flight Simulatorの誕生、開発、および死の物語





Flight Simulatorは、これまでに作成されたすべてのシミュレーターの中で最も長く成功したシリーズです。 その開発の歴史の中で、70年代後半に定められたコードと原則が何十年も変わらずに残っていたのは興味深いことです。



1975年、ブルース・アートウィックはイリノイ大学で電気工学の学士号を取得しました。 しかし、サイバネティックスは彼から遠く離れているとは思わないでください：最初は1973年に彼はコンピューター設計を専門として選び、その後、研究の方向を電気工学に変えました。コンピューター技術の開発のこれらの初期の時代にはそのような科学者が理解すると信じていたからです学位。 1975-1976年 デジタルコンピューターラボのグラフィックターミナルを開発するグループでの作業と並行して、ブルースは元のUNIXの作成者であるケントンプソンと通信し、最初のUNIXマシンの1つであるPDP-11で作業することができました。そのテーマは、多層動的コンピューターフライトグラフィックスでした。 Artvikは、コンピューター画面に表示された航空機の飛行モデルを調べました。これは、実際にシリーズ全体の誕生を意味していました。



1976年に科学の理学修士号を取得した後、ブルースはロサンゼルスのカルバーシティにあるヒューズエアクラフトで働き始めました。 PDP-11の経験により、ArtvikはMotorola 6800プロセッサ（利用可能な最初のマイクロコンピュータ）が算術計算を行い、飛行シミュレーショングラフィックを表示できることを証明することができました。 ブルース・アートウィックはプログラムと彼のアイデアに関するいくつかの雑誌記事を書きました。 そのうちの1人の編集者がArtvikに電話し、一部の読者はプログラムを購入したいと言ったとき、ブルースはsubLOGICを見つけることにしました。 彼女の名前は、ArtwickがDigital Computer Labで長年取り組んできたPDP-11の論理回路への言及でした。



subLOGIC Flight Simulator 1（1979）





1978年にStuart Momentと協力して設立された同社は、Motorola 6800、MOS Technology 6502、およびIntel 8080プロセッサ向けのソフトウェアの開発を開始しました。 -80。 彼女のグラフィックは次のように見えました。











最初のシミュレーターは、地表の白黒グラフ（最大36平方、6×6）、1つの平面のみ、飛行プロセスの限定的なシミュレーションを備えていました。 すべての欠点にもかかわらず、シミュレーターは大成功を収め、1981年にApple IIのベストセラーエンターテイメントプログラムに選ばれました。 現在のように、シミュレータのほとんどのユーザーはプロのパイロットではありませんでした。 グラフィックスをTRS-80の弱い機能に合わせるには、ダッシュボードを削除して解像度を下げる必要がありました。 したがって、TRS-80のFlight Simulatorバージョンは、すべてのフライトシミュレータの中で最もシンプルなグラフィックを備えています。 SubLOGICは後に、両方のプラットフォーム用のソフトウェアアップデートをリリースしました。 更新には、改善された地球表面、ヘルプメニュー、および爆弾の光景から表示する機能が含まれていました。



Microsoft Flight Simulator 1.0（1982）





Artvikの仕事は見過ごされませんでした。レドモンドの別のオタクが彼自身の会社を設立し、シミュレーターに興味を示しました。 IBMもFlight Simulatorライセンスの争いに参加しましたが、MicrosoftはIBM PCアーキテクチャに移植するための共同ライセンスを取得しました。Artvikは小さな会社の雰囲気が好きでした。



1982年11月に、Microsoft Flight Simulator 1.0がPCプラットフォーム用にリリースされ、すぐに2番目のバージョンがリリースされました。 Artvikによって開発され、第5バージョンまで使用された更新されたFSワールド座標システム、気象効果、Sopwith Camelとの戦闘モード、時刻、改善されたグラフィックが新しいシミュレータに登場しました。 システム要件も増加しました。ユーザーは、主にFlight Simulatorのためにマシンを更新する必要がありました。 MS FSはベンチマークとしても使用されました。MSFS1.0とLotus 1-2-3を実行できるマシンは、完全にIBM PC互換と見なされていました。











subLOGIC Flight Simulator 2（1983-1984）





IBM PCの移植性の強化の優位性は明らかであり、顧客はそれらを元のプラットフォームに導入することを求めました。 subLOGICは、IBM以外のプラットフォーム用のFlight Simulator 2バージョン1を開発およびリリースしました。 このバージョンは、Microsoft（米国内の複数の場所）のプログラムとして無地であり、Apple II（1983）、Commodore 64、およびAtari 800（1984）のプラットフォームを対象としています。 アドオンの機能が初めて登場しましたが、今日の形ではありませんでした。 プログラムは制限があるため、ターゲットデバイスによって異なります。 1986年にリリースされたAmiga、Atari ST、およびMacintoshのバージョンは、わずかに改善されたため、第3世代に割り当てられることがあります。



















Microsoft Flight Simulator 2.0（1984）





1984年、Microsoftは製品の新しいバージョンをリリースしましたが、最初のバージョンとそれほど違いはありませんでした。 新しいシミュレーターには全米の景観が含まれていましたが、最初のバージョンから空港は追加されていませんでした。 subLOGIC Scenery Disksアドオンとの互換性があり、すぐに米国、ハワイ、日本、ヨーロッパの一部をパッチでカバーしました。











Microsoft Flight Simulator 3.0（1988）





1988年、ArtwickはsubLOGICを去り、新しい会社BAO Ltdを設立しました。 （Bruce Artwick Organization）、フライトシミュレーター用の製品の開発とマーケティングに従事していました。 基本的には、Microsoft Flight Simulatorと、米国連邦航空局向けのプログラム（タワー制御シミュレーター、航空管制官など）でした。 BAOのスタッフは35人でしたが、1995年までArtvikがMicrosoftの資産を売却し、従業員がシアトルのオフィスに移るまで、同社は新しいバージョンのシミュレータでMicrosoftと協力して仕事を続けました。



88年の後半、レドモンド社のシミュレーターの3番目のバージョンがリリースされました。そこには、マップ、ダッシュボード、さまざまな視野角、EGAモードに基づくグラフィックス、3台の航空機-Gates Learjet 25、Cessna Skylaneを表示できる個別のカスタマイズ可能なウィンドウがありましたとソッピースキャメル。 Microsoftシミュレーターで初めて、3人称の飛行機を見ることができました。 subLOGICシミュレーターには、第2世代の第三者視点もありました。 MS FS 3.0シミュレータキットには、古いシーナリーディスクをSCNファイルに変換するユーティリティが含まれていました。















Microsoft Flight Simulator 4.0（1989）





1989年、Apple Macintosh用のFlight Simulatorの最新バージョンがリリースされ、FSの90年代の時代が続き、厳密に1つのプラットフォームに焦点を当てることになりました。



1989年から1993年の間に、多数の追加機能がリリースされました。 BAO Ltd. 場所と航空機の航空機とシーナリーデザイナーを編集するためのプログラムを準備したため、ユーザー自身がアドオンを書き始めました。 その後、Artvikの会社はサウンドとグラフィックスのアップデートをリリースしました。これにより、スピーカーのビープ音ではなく、合成サウンドと最大800×600の画面解像度を得ることができました。 ランドスケープランドスケープの更新、航空機エディター、QBASICのようなAircraft Adventure Factoryフライトスクリプト言語。 MS FSの後続バージョンには、独自のアドオンを作成するためのこのような強力なユーティリティは含まれていません。















Microsoft Flight Simulator 5.0（1993）





フライトシミュレータの5番目のバージョンには実際の座標系があり、4番目のバージョンのランドスケープはFS 5.0と互換性がありませんでした。 初めてテクスチャが使用され、フラットシャドウを使用した以前のシミュレータよりもはるかに高いレベルのリアリズムを実現できました。 現在、各航空機にはダッシュボードがあり、1つではなく、マシンごとにわずかに変更されています。



1995年、MS FS 5.1は最初にCDでリリースされました。 本当に3次元の気象効果があり、地球の衛星画像ライブラリを使用して景観が構築されました。 MS FS 5.1の機能の欠如（たとえば、世界中で合計250の空港）と、BAO Ltdによる1995年のフライトショッププログラムのリリースの組み合わせ。 無料のカスタムアドオンの波を後押ししました。











Microsoftフライトシミュレーター6.0（1996）





Bruce Artwickが手に入れたMS FSの最新バージョンは、対応するオペレーティングシステムをサポートするためにFlight Simulator for Windows 95という名前で販売されました。 シミュレータBAO Ltdのリリースの少し前 マイクロソフトに売却され、ブルース・アートヴィックは現在、コンサルタントの資格しか持っていませんでした。 同じ頃、別のフライトシミュレーターであるPro Pilotを開発するためにsubLOGICがSierraに買収されました。



FS 6.0の主な違いは、モデル、ダッシュボードテクスチャ、より多くのランドスケープテクスチャ、およびいくつかの新しい航空機の改善でした。 奇妙なことに、新しいWindowsの反対者の苦情にもかかわらず、新しいシミュレータのフレームレートはWindows 3.11のバージョンと比較して1.5倍増加しましたが、解像度も増加しました。



MS FS 6.1はFlight Simulator 98と名付けられ、1998年にリリースされました。 DirectXの使用開始、Bell 206BIII JetRangerヘリコプターの外観、更新されたCessna 182、3000の空港、45の先進都市を除き、わずかな改善のみが行われました。 FS98の1000万枚以上のコピーが世界中で販売されています。











Microsoft Flight Simulator 7.0（1999）





当時のFlight Simulator 2000と呼ばれるバージョンには、少なくともPentium 166 MHzの大きなシステム要件がありましたが、クロック周波数が400〜500 MHzのプロセッサが本当に必要でした。 ArtvikはBAO Ltdの売却については正しかった。 必要に応じて、130人が新しいバージョンの開発に参加しました。 GPSはFS 7.0に登場し、インターネットから実際の天気をダウンロードするためのサポート、17000の新しい空港（現在2万の空港があります）。 実際、世界のほぼすべての主要空港の存在について話すことができます。 MS FS 2000は、Windows 95およびWindows NT 4.0をサポートする最新のシミュレータです。











Microsoft Flight Simulator 8.0（2001）





Flight Simulator 2002は、以前のバージョンと比較して、航空交通管制の改善、他の航空機からの人工知能、仮想コックピットの表示時の慣性効果、およびパフォーマンスの改善を受けました。 2001年9月11日のイベントの後、ツインタワーを撤去するため、リリースは10月19日までわずかに遅れました。







Microsoft Flight Simulator 9.0（2003）





Flight Simulator 2004：A Century of Flightと呼ばれるシミュレーターは、ライト兄弟の100周年の年にリリースされました。 新しいバージョンには、いくつかの歴史的な航空機、仮想コックピットでの双方向性、さまざまな景観オブジェクト（ライト、納屋、サイロなど）が含まれていました。 2004年6月10日に、バージョン9.1への更新がリリースされ、忘れられたブリッジといくつかの他のオブジェクトが追加されました。











Microsoft Flight Simulator 10.0（2006）





Flight Simulator Xは、これまでにリリースされたFlight Simulatorの10番目の最新リリースです。 以前のシミュレーションのデフォルト空港であるメイグス空港は2003年3月に閉鎖されましたが、このバージョンでのみ完全に遮断されました。 10番目のバージョンでは、1台の航空機を一緒に制御したり、管制塔で作業したりするための新機能など、マルチプレイヤーゲームのサポートが改善されました。 シミュレーターのデラックスバージョンには、自動生成されたランドスケープを変更できるSDKが含まれています。 シミュレーターがDVDに初めて配信されたとき、システム要件が大幅に増加し、強力なプロセッサーの必要性が特に顕著になりました。











マイクロソフトフライト





しばらくの間、このシリーズのマイクロソフトのソフトウェア開発部門は、会社の財政的問題との闘いの尺度として閉鎖されました。 ただし、2010年8月には、新しいバージョンに関する情報が表示されました。 残念ながら、Microsoft Flightはその名前が示すように、優れたグラフィックスと非常にアーケードゲームプレイを備えた単なる無料（F2P）ゲームです。したがって、すべてのマーケティングにもかかわらず、それを後継、特にシリーズの復活。 このゲームは追加コンテンツ（DLC）を販売することで収益化されるため、以前の追加は互換性がありません。
