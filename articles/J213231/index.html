Nokia Treasure Tag：持ち物を守るスマートタグ

昨日、人々が重要なことを忘れないようにするという大きな使命を託された国際市場でのミニチュア機器の差し迫った発売が発表されました。 文字通り。 Nokia Treasure Tagの詳細はすべて公開されていません。









Nokia Treasure Tag-わずか30x30x10 mm、重さ10グラムのデバイス。NokiaLumiaスマートフォンの所有者に忘れ物を思い出させるように設計されています。 デバイスは、バッグ、キーチェーン、または他の重要なアイテムに取り付けることができます。



Bluetooth 4.0ワイヤレスアダプター（近接プロファイル（PXP）、オブジェクトプッシュプロファイル（OPP）1.1、およびFind Meプロファイル（FMP）をサポート）、NFCマイクロチップ、ブザーがNokia Treasure Tagスマートタグに組み込まれています。 電力はCR-2032バッテリーによって提供され、通常の使用中に平均6か月のバッテリー寿命を提供します。







Nokia Treasure Tagをアクティブモードにするには、NFCまたはBluetoothを使用してスマートフォンと「リンク」するだけで十分です。 後者の場合、BT Smartテクノロジー（BT LEとも呼ばれる）を使用して自動タグ検出が実行されます。



現在、Nokia Lumiaユーザーがタグを付けたアイテムを忘れて家を離れると（実際、スマートフォンはNokia Treasure Tagとの接続を失います）、両方のデバイスが鳴り始めます。







Nokia Treasure Tagを管理するために、 Windows Phone用の特別なアプリケーションがリリースされました。 タグを識別して記憶する基本機能に加えて、さまざまなモードや設定を制御するために使用できます。 たとえば、ユーザーはタグ信号を手動でアクティブにしたり、その逆にサウンドアラートをオフにしたりできます。 また、このアプリケーションを使用すると、デバイスを非アクティブ化するか、「スリープモード」に送信できます。







ただし、タグが添付されたアイテムがまだ忘れられていて、どこに残っていたか覚えていない場合は、いつでもここのカードや拡張現実アプリケーションLiveSightを使用して見つけることができます。トレジャータグ。



逆のシナリオも機能します。スマートフォンを離れた場所を忘れた場合は、トレジャータグのボタンを押したままにします。 電話が半径40メートル以内にある場合、音声信号を発します。







Nokia Lumiaスマートフォンは、4つのNokia Treasure Tagタグを同時に使用して作業をサポートします。これは、ユーザーが頻繁に忘れられるアイテムをいくつか持っている場合に便利です。



Treasure Tagは、白、黒、黄色、青の4つのカラーオプションで提供されます。 ロシア市場での新しいアイテムのコストと入手可能性は後日発表されます（仮に$ 30）。



これまでのところ、Nokia Treasure TagはWindows Phone 8上のNokia Lumiaスマートフォン（Nokia Blackバージョンにアップグレード）と互換性がありますが、まもなくAndroidおよびiOS用のサードパーティアプリケーションをリリースする計画があります。
