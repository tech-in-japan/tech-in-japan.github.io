「プログラマー向けAndroid：アプリケーションを作成する」-Daytelの新しい本

明日は、おそらくコンピューター文学の世界で最も人気のある著者によって書かれた素晴らしいノベルティを見るでしょう。 会う：

Deitel＆Associates、Inc.の Paul、Edie、およびHarvey Daytelによる 「 プログラマー向けAndroid：アプリケーションの構築 」 マイケル・モルガノは、彼らに加わった。 プログラマー向けのiPhone：App-Driven Approachの本ですでにこのチームに紹介されている。

表紙の姓は質の表れです。 世界中の100万人を超えるプログラマーが、C、C ++、C＃、iPhone用Java、JavaScriptでのWebプログラミング、XML、Visual Basic、Visual C ++、Perl、Pythonなどを本から教えてきました。 これらの著者による約30冊の本がすでにロシア語で出版されています。







本の特徴



この本は、Android向けアプリケーションの開発を開始し、Google Playですばやく公開するために必要なすべてを提供します。 この本は、アプリケーション指向のアプローチを使用しています。 アプリケーション開発に特化した各章（第3章から第19章）では、1つのアプリケーションが紹介され、アプリケーション機能が検討され、実行中のアプリケーションのスクリーンショット、テストドライブ、およびアプリケーションの作成に使用されるテクノロジーとアーキテクチャの概要が紹介されています。 次に、アプリケーションをビルドし、その完全なソースコードを提示し、このコードの詳細な分析を実施します。 プログラミングで使用される概念について説明し、アプリケーションの作成に使用されるAndroid APIの機能プロパティを示します。



以下は、本で紹介されているアプリケーションをリストし、それらを作成するために使用される主要なテクノロジーを説明しています。



  • 第3章、ようこそ、Dive-Into®Eclipse、およびADT
  • 第4章、ヒント計算機、Javaを使用したAndroidアプリケーションの開発
  • 第5章、お気に入りのTwitter®検索、コレクション、ウィジェット、およびビュー
  • 第6章、フラグクイズアプリ、メニューとインテント
  • 第7章、Cannon Gameアプリ、フレームごとのアニメーションおよびカスタムイベント処理
  • 第8章、スポットオンゲームアプリ、トランジションアニメーション、タッチトラッキング
  • 第9章、Doodlzアプリ、グラフィックスおよび加速度計
  • 第10章、アドレス帳、アダプターおよびAdapterView
  • 第11章、ルートトラッカー、マップ、およびコンパスAPI
  • 第12章、スライドショー、写真および音声ライブラリへのアクセス
  • 第13章、強化されたスライドショー、オブジェクトのシリアル化とビデオの再生
  • 第14章、天気ビューア、インターネットアプリケーション、Webサービス、およびアプリケーションウィジェット


Android SDK2.x。 この本では、Android Software Development Kit（SDK）2.xに含まれる多くの新機能について説明します。これには、Bluetooth、Googleマップ、カメラAPI、グラフィックスAPI、さまざまな画面サイズと解像度のサポートが含まれます。



タブレットアプリケーション用のAndroid SDK3.x。 タブレットアプリケーションの開発用に設計された新しいAndroid SDK 3.xのセットを検討しています。 このセットは、アニメーション、アクションバー、スニペット、ステータスバー通知、およびドラッグアンドドロップテクノロジーをサポートしています。



Android Maps API 。 Route TrackerアプリケーションはAndroid Maps APIを使用します。これにより、開発でGoogleマップを使用できます。 Maps APIを使用するアプリケーションの開発を開始する前に、 code.google.com /にあるAndroid Maps API利用規約（関連する法的通知およびプライバシーポリシードキュメントの規約を含む）に記載されている規約に同意する必要があります。 android / maps-api-tos.pdf



日食 無料のEclipse統合開発環境（IDE）と無料のAndroid SDKおよびJava Development Kit（JDK）は、Androidアプリケーションの構築とテストに必要なすべてを開発者に提供します。

マルチメディア。 アプリケーションは、グラフィックス、画像、フレームごとのアニメーション、プロパティアニメーション、オーディオ、ビデオ、合成、音声認識など、幅広いAndroidマルチメディア機能を使用します。

最高のAndroidアプリケーション開発テクニック。 プログラムコードを詳細に分析する過程で、Androidアプリケーションの開発プロセスで使用される最高のテクニックが注目されます。 AndroidベストプラクティスリソースセンターWebサイト（ www.deitel.com/AndroidBestPractices）をご覧ください 。



Webサービス Webサービスを使用すると、インターネットで利用できる豊富なサービスライブラリを利用できます。その多くは無料です。 第11章で開発されたRoute Trackerアプリは、組み込みのAndroid Maps APIを使用してGoogle Maps Webサービスと対話します。 第14章で開発されたWeather Viewerは、WeatherBugのWebサービスを使用します。
