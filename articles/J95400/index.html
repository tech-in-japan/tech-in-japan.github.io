Computexを使用したMeeGoに関するニュース

先週、有望なMeeGo 1.0 が発表され 、その周りのアクションは非常に急速に発展し始めました。 たとえば、Computex 2010は最も近い会場となり、新しい発表や発表に最適です。



新しいプラットフォーム（およびIntel AppUp Centerプロジェクト）のサポートは、多くのメーカー、OSサプライヤー、通信事業者によって発表されました。 カットの下の1行の詳細。



モバイルデバイス向けに最適化された新しいオープンソースオペレーティングシステムのアプリケーションは、現在ベータモードのIntel AppUp Centerを通じて配布されます。 国際展示会Computex 2010で、IntelはIntel Atom Software Summit会議を開催し、新しいプロジェクトのさらなる開発の計画について議論しました。 結果：



Acerは、プロジェクトのサポートと、2010年末までにMeeGoベースのネットブックをリリースする意向を発表しました。

Asusは、Intel AppUp Centerに基づくストアを立ち上げる計画を発表し、2011年にMeeGoベースのデバイスの出荷を開始する予定です。

インテルは、InventecとQuantaが開発したタブレットデザインを導入しました。 開発者は、Intel AppUp Centerがアプリケーションへのアクセスを容易にする方法を示しました。





ネットブック、タブレット、その他のデバイス向けにMeeGo 1.0に基づくオペレーティングシステムをリリースする意向は、多数の開発者によって発表されました。



China Standard Software Company（CS2C）は、MeeGo 1.0に基づくOSをリリースする意向を発表しました。これは、深セン（中国）のOEMメーカーであるCZCによって開発および販売されます。

DeviceVMは、Splashtop MeeGo Remixをリリースする計画を発表しました。これは、起動時間の短縮、強力なセキュリティ、カスタマイズ可能なインターフェースを備えたSplashtopの改良版です。

International Systは、使いやすいグラフィカルインターフェイスを備えたオペレーティング環境であるMetasys MeeGoをリリースします。 このシステムは、Intelコンポーネントをベースにした学校の変形可能なPCで使用されます。

Linpusは、MeeGoに基づいたLinpus Linux LiteネットブックのOSバージョンを紹介します。 新製品には高度な電源設定があり、Linpus LiveUpdateを通じてアップデートを受信する機能が提供されます。

Mandrivaは、世界で最も人気のあるLinux製品の1つであるMandriva Linuxの機能を使用して構築された、MeeGo 1.0に基づいた使いやすいオペレーティングシステムであるMandriva Miniをリリースする計画を発表しました。

ノベルは、モバイルデバイス向けのMeeGo 1.0に基づくオペレーティングシステムであるSUSE MeeGoを発表しました。 2011年に使用する予定です。

これもMeeGo 1.0ベースのPixartは、Intelコンポーネントをベースにしたデジタルテレビデバイスと学校のコンバーチブルPCに焦点を当てます。

Red Flagは、ネットブック用のMeeGo 1.0ベースのRed Flag inMiniオペレーティングシステムと、Intel Atomプロセッサを搭載したタブレットやその他のデバイス用のMidinux 3.0を開発および起動します。

Turbolinux Chinaは、車載インフォテインメントシステム用のOSであるGreatTurbo IVIの導入を計画しています。 MeeGo 1.0に基づいています。

VietSoftwareは、教育省とベトナムの支援を受けて、エントリーレベルのネットブックとデスクトップPC向けの便利なオペレーティングシステムAsianux Lite 2.2を導入します。



私が好きではないもの（私は怖いですか？）そこにはあらゆる種類のLinux（オープンソースのオペレーティングシステム）があります-ディストリビューションは雨の後にキノコのように見えるからです。 誰もが自分の特性、ゴキブリなどで異なっています-とにかく、多くの人々がミランダの組み立てを行い、他のすべての人にそれらを課します。 私はあなたのことは知りませんが、そのような混乱を整理することはできません（したくないのですか？）-それがおそらくWindowsで完璧に仕事をしている理由です。



しかし、一方で、これは他のすべての人にとっての主な利点の1つであるように思われます-多くの選択肢を持つ一種の完全な自由。 まあ、それが事実なら、プラスポスト-結局、MeeGoも例外ではないようです:)
