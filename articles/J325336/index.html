核の冬の鉄-寒さ、霧、煙からの保護







通常、軍事用ハードウェアは、信じられないほどの信頼性と活力に包まれています。 ただし、これは、2014年4月にアメリカの駆逐艦ドナルドクックが彼に対してKhibiny電子戦システムを使用した後に「盲目」になり、「耳が聞こえなくなる」ことを妨げるものではありませんでした。 しかし、機内には専門的ではあるものの多くのコンピューター機器がありました-電磁放射に対する保護は本当にありませんか？







一般に、軍事ITのトピックにも興味がある場合は、「軍事」および単純に保護されたハードウェアの分野への小旅行を提供します。







軍事分野の秘密とオープンソースのofさを考えると、一部の情報はすでに無関係かもしれませんが、それでもそれは全体像を示しています。 場合によっては、論理的な推論によって情報の不足を補う必要がありました。矛盾に気付いた場合は修正して補足します。







装甲、耐衝撃性、霧をクリアに保つために



軍事機器の信頼性に関する要件についての長い議論を省略した場合、理想的なサーバーには次の機能が必要です。







  • 耐衝撃性の防弾ケース;







  • ほこり、塩霧、水、火の封じ込めまでの気密性;







  • 可動部品の欠如、独自の受動冷却;







  • -60から+55までの温度変化、および湿度と大気圧のサージに耐える能力;







  • しっかりと組み立てられたコンポーネント、超小型回路の高品質の配線。







  • ハードウェアレベルでの情報の暗号化。







  • 回復の可能性なしにデータを削除する機能を備えたハードドライブ（ 軍用フラッシュドライブに実装されているように）;







  • 自然で人工的な性質の電磁放射に対する保護。







  • 核爆発を乗り切る能力-EMPに加えて、これはガンマ/中性子放射の瞬間的な線量が大きいことです。


また、一部の要件は相互に排他的であるため、機器の欠陥は正しい部屋で補う必要があります。 サーバー用に適切に設計された「シェルター」は、いくつかの機能を実行できます。たとえば、霧や電磁放射からの保護です。







データセンター「a-la Militare」



ほぼ10年前、ロシアの会社Sitronics（現在はNVision Group） は、さまざまな気候条件に焦点を当てた、Dateriumと呼ばれる一連のモバイルデータセンターを発表しました 。 データセンタープラットフォームは、必要なすべてのエンジニアリング機器を備えた、高さ42Uの7ラックの輸送コンテナです。







基本キットに加えて、可用性モジュールが強化されており、電源システムとその配電の完全な冗長性を提供します。 Dateriumソリューションに基づいて、最大28ラックを含むモジュラーデータセンターを組み立てることができます。これは、120平方の有効領域を持つ固定データセンターに相当します。 メートル。













データムはあらゆる場面に適応します。







Dateriumは産業用データセンターとして発表されましたが、移動性、拡張性、展開速度を可能にするこのような「全天候型」ソリューションが、地質学者、石油労働者だけでなく軍隊にとってもデータセンターの基礎となり得ることは明らかです。







これは海外でも信じられています-数年前、DellはTactical Mobile Data Centerと呼ばれる独自のバージョンのモバイルデータセンターも提案しました 。 ベースは変更されたISU-96コンテナーで、その内部には3つの42Uユニットまたは1つの120Uボリュームがあります。













Daterumに対するアメリカの反応-戦術的なモバイルデータセンター







このバージョンのデータセンターは、あらゆる場面に対応しています。







  • 気象条件、砂嵐、塵に対する保護。







  • 統合された消火システム。







  • 冗長換気システムによる空冷。







  • ビデオカメラ、侵入検知器などのセキュリティ監視システム。







  • 世界との通信用-外部銅、光ファイバ、またはBNCコネクタ。


この複合施設は、適切な価格にもかかわらず、米国軍隊および他国からの潜在的な買い手にすでに興味を持っています。サーバーとデータセンターのスイッチがなければ、221,500ドルもの費用がかかります。







単一サーバーではない









OPK製の軍用ラップトップ（「Constellation」の懸念と協力して「United Instrument-Making Corporation」）







このロシア製の軍用ラップトップは、デジタル戦術無線通信システムの一部として使用されます。 彼は生き残るために必要なすべてを持っています：







  • 保護された密閉筐体は、ほこり、湿気、水、火からの保護、および振動荷重、衝撃、および小さな高さからの落下に耐える能力を意味します。







  • -40°Cから+ 50°Cの温度で働く能力;







  • 現場での積極的な使用により、外部接続ポートのコネクタの保護レベルが向上しました。


軍用サーバーは明らかに悪くはありません。防衛産業は、さまざまなレベルの制御の指揮所で動作するように設計されたフォールトトレラントコンピューター「Sunrise」を作成しました。













その特徴：







  • 耐衝撃性および密閉ハウジング;







  • 受動冷却システム;







  • -60°C〜+ 55°Cの温度範囲。







  • 移動中に作業する能力;







  • わずか35ワットの電力は、平均的なオフィスコンピューターの消費量の3〜4分の1です。


コンピュータープラットフォームは、「特殊な目的のコンピューター」の基本要素としても適しています-安全な「軍用」タブレットからモバイルおよび固定ワークステーションまで。 または、KOMDIV64アーキテクチャを備えたロシアのプロセッサに基づいて構築されているため、本格的なサーバーになることもできます。 このアーキテクチャにより、クラスター内のプロセッサを数百から千以上のコア数と組み合わせて、スーパーコンピューターのパフォーマンスを実現できます。







軍隊も



私の国内の軍事用ハードウェアは、ロシア製のプロセッサに常に関連付けられていました。ロシア製プロセッサは、x86、低特性などの一般的なアーキテクチャの欠如で知られていました。 しかし今日、状況は劇的に変化しました。







前述のVoskhodコンピューターには、800 MHzのクロック周波数と2 GBのRAM DDR3 1200を搭載した2つのコアが既にあります。 同じOPKの軍用ラップトップは、動作周波数3.5 GHzの第6世代プロセッサに基づいて実行されます。 このような特性は、一般的な「民間」サーバーの機能とほぼ同じであり、深刻な軍事問題を解決できます 。







別の良い例は、Intel Core i5-520MまたはIntel Core i7-620MプロセッサとnVidia GT 330M 1024MB DDR3グラフィックを搭載した高度な保護機能を備えたGetac X500ミリタリーラップトップです。 それは346千ルーブルかかります 。 このような高コストは、デバイスの深刻な保護だけでなく、各軍隊に必要なチップによっても決まります。







  • さまざまな情報デバイスから受信したインテリジェンスデータの収集。







  • 特殊な軍事ソフトウェアのサポートと、さまざまな形式の大量のグラフィックデータの処理。







  • 敵の位置と彼ら自身の軍隊の正確な位置の決定。


徐々に、軍事機器は信頼できるものではなくなりますが、利便性は常に「四角」になり、その特性は悲惨です。 軍のハードウェアとソフトウェアの機能、利便性、速度は、すでに消費者市場のモデルとかなり匹敵しています。







おわりに
