Photoshopでの効果的な写真処理





Photoshopに関する記事を読んで、基本的に単純な処理タスクのソリューションを複雑にしている著者の数に驚かされることがよくあります。 多くの「記念碑的な」作家、例えばダン・マルグリスもこれに苦しんでいます。 しかし、それは彼には許されません-彼の仕事は、処理プロセスのすべての微妙さとニュアンスについて書き、あらゆる角度と側面からそれを考慮することです。 彼の本の資料のプレゼンテーションのこの特定の機能は、多くの読者をはじきます。



実際、これらの「40アクションのシャープ」メソッドのルーツは非常に単純なものから生まれています。これらのレッスンを書く人は、大量の写真を扱ったことはありません。 つまり、原則として、2枚の写真があり、それらを処理する過程で1晩か2晩殺す用意ができています。 しかし、定期的な注文があり、写真を撮影するたびに、数十フレームを真剣に処理する必要があります。より簡単で便利な処理方法について考え始めます。



今日はそれらについてお話します。 私の仕事で常に使用している5つのシンプルだが非常に効果的なPhotoshopツールについて説明します。







Photoshopで写真を処理する前に、私は常にRAWコンバーターでフレームを使用します。 写真の基本的なカラーグレーディングと一次処理を行うのはそこです。 実際、処理の「スケルトン」を作成し、Photoshopでは写真の詳細をすでに使用しています。



そのため、RAWコンバーターで写真を処理し、Photoshopで開きました。 Photoshopは、あらゆる場面に対応する膨大な数の処理ツールを備えています。 ただし、最も単純で効果的なものについて説明します。



1.ダッジツール/書き込みツール







ダッジツール/書き込みツールの主な機能は、画像の特定の部分を明るく/暗くすることです。 実際、調光を「ペイント」することも、その逆も可能です-画像を明るくします。 それは非常に簡単です、試してみてください：このツールに感謝するでしょう。 Dodge / Burnツールには2つだけの非常に重要な設定があります。



範囲-スコープの選択





このツールは、写真のシャドウ、ハイライト、またはミッドトーンで使用できます。 たとえば、（ ポートレートを処理するとき）あごの暗い部分を明るくし、光に手を触れないようにする必要があります。 この場合、シャドウモードをドッジツールに配置すると、適用する場所の暗い領域のみが明るくなります。



暴露-衝撃強度





効果の強さを正しく設定することは非常に重要です。 Photoshopを試している多くの人は、Dodge / Burn 100％を試します。 そして、画像を暗くし、ブラックの「穴」を開け、明るくします-連続的な露出過剰。 もちろん、そのような結果を得ると、彼らはもはやこのツールに戻りません。 しかし、Dodge / Burnは微妙なツールです。 影やハイライトで作業する場合は、アプリケーションの強度を7〜10％に、中性領域の場合は10〜20％にしてください。 各ケースについて、影響力は個別に選択されますが、このツールを少し操作した後、各ケースで必要なパワーを正確に感じるようになります。



使用する



Dodge / Burnには多くの用途があります：



  • 虹彩を明るくする
  • 男性の肖像画の顔の形の線を暗くする
  • バックライト領域を明るくする
  • 歯を白くする




ダッジツールを使用して、写真の歯を白くするのが最も簡単で効果的です。 少し後で、Dogdeツールを使用して、正しい歯のホワイトニングに関する個別のレッスンを作成します。



2.クローンスタンプ







Photoshopには、画像をレタッチするためのツールがいくつかあり、それぞれが独自の方法で優れています。 しかし、スタンプは使用中の最も汎用性の高いツールです。



その機能は、画像の特定のセクションを取得してコピーすることです。 したがって、たとえば、しわをレタッチすることができます-単に滑らかな肌の領域で「置換」することによって。 これを行うには、Altキーを押して、写真を撮る場所を選択し、画像の目的の部分をクリックするだけで、その部分にコピーします。



スタンプ設定では、2つのパラメーターに注意を払うことが重要です。



モード





これらは、スタンプが機能するモードです。 たとえば、暗くするモードでは、スタンプは選択した領域よりも明るい領域のみを「置き換え」ます。 実際、画像の明るい部分を暗くすることができます。そのため、モードの名前はDarkenです。 また、それに応じて、[明るく]モードでは、スタンプは画像の暗い領域でのみ機能し、明るくなります。



クローンスタンプには多くの操作モードがあります。それらを試してみてください。面白い結果が得られるはずです。

私の意見では、各モードの動作を説明することは意味がありません。Photoshopでは、すべてのツールについて、本質的に同じモードの原理が機能します。



不透明度







不透明度は不透明度を意味します。 簡単に言えば、この設定でパーセンテージを設定するほど、スタンプの「作業」はより透明になります。 たとえば、100％の場合、スタンプは選択した領域を完全に置き換え、50％の場合は半透明になります。 顔のレタッチには、原則として10〜30％が使用されます。そうでない場合、スタンプマークがはっきりと見えます。



クローンスタンプの使用



  • レタッチ




すべての顕現のレタッチがスタンプの主な目的です。 まず第一に、スタンプは肌をレタッチするために使用されます-しわ、目の下の打撲傷、腫れなどの母なる自然の美しい創造物を取り除きます。







たとえば、フレーム内の不要なオブジェクトをレタッチすることもできます。 もちろん、写真の半分を撮らない限り。



スタンプを使用して小さな露出過多を排除することは非常に便利です。 たとえば、モデルの鼻の先に露出オーバーの小さなスポットがあります。 数回クリックするだけで、スタンプを取り、暗くするモードを設定し、このスポットを暗くします。



3.履歴ブラシ









Historiブラシは、写真を処理するためのタイムマシンです。 処理のどの段階でも、ブラシを使用して画像に応じて描画できます。



履歴ブラシには大きな可能性があります。 このツールの動作については、 別の記事ですでに詳しく説明しました。 その中で、履歴ブラシの使用に関する詳細なレッスンを見つけ、必要な画像の領域でのみシャープにする方法を学びます。



もちろん、適用範囲はシャープだけではありません。 今後の記事では、履歴ブラシを使用して写真の色を操作する方法を説明します。



4.白黒





白黒ツールは、画像->調整タブにあります。 または、写真に調整レイヤーを作成することもできます。







白黒ツールの主な機能は、カラー画像をb / wに「正しく」変換することです。 各色の白黒表示を変更できるため、修正します。 したがって、美しくて「おいしい」白黒画像を得ることができます。







ただし、この機能ではB＆Wは制限されません。



このツールを使用すると、非常に興味深いカラー画像を取得できます。 写真に白黒を適用し、オーバーレイレイヤーモードをオンにします。



これで、B＆Wコントロールレバーとレイヤーの透明度を操作することで、非常に興味深い画像を取得できます。 わかりやすくするために、レイヤーの不透明度をB＆W-62％で非常に高く設定し、緑、シアン、青、マゼンタのレバーを最大にしました。



ご覧のとおり、写真はすぐに豊かになり、コントラストが強くなりました（写真をクリックすると拡大します）。







それでは、 Tintチェックボックスに注目しましょう。 オンにすると、必要な色で画像に色を付けることができます。







使用する



カラーを操作するときと白黒を処理するときの両方で、白黒を使用するオプションは大量です。

次の記事の1つで、いくつかの写真を処理する例について、白黒で作業する際の基本的なニュアンスについて説明します。



5.シャドウ/ハイライト









シャドウ/ハイライトは、画像->調整タブにもあります（ちなみに、興味深いツールがたくさんあります。ちなみに、みんなで実験することをお勧めします）



このツールは、露出過度の領域を暗くし、影から光を描画するように設計されています。 最も明白なアプリケーションに加えて、露出過剰とギャップを排除するために、S / Hは、より大きな画像の深みの感覚を作成するのにも役立ちます。 明るい領域に暗いハーフトーンを追加し、暗い領域に明るいハーフトーンを追加できます。 したがって、画像はより大きく、より深くなります。



たとえば、S / Hを使用したこの写真では、子犬の髪のボリュームを追加し、すぐに写真がより面白くなりました。







実際、シャドウ/ハイライトは、あらゆる深刻な処理に不可欠なツールです。 ほぼすべての写真は、S / Hを正しく適用するとより良くできます。



S / Hのすべての設定とその機能についてお話ししたいと思いますが、これは実際には別の記事のトピックです。 将来、私は間違いなくトピック「シャドウ/ハイライト」に戻りますが、今のところは、実験してみてください-異なる設定を試して結果を見てください。 私の経験では、この方法は新しいものをマスターするのに最も効果的です。



ご覧のとおり、これらのツールはすべて非常に使いやすいですが、同時に非常に効果的です。 それらを試してみると、処理中にどれだけの機会が与えられるかを感じるでしょう。



Photoshopのシンプルだが非常に効果的なツールに関する一連の記事を作成する価値があると思います。 そして次の記事では、写真の色を使った真剣な仕事のためのツールについてお話します。
