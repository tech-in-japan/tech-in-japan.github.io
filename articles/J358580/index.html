Zeon IP-PBX + RetailCRM =効率的なオンライン販売

別の興味深いシステムと統合しました。 そして、それについて話すのが待ちきれません。



RetailCRMは、もともとオンライン取引用のCRMシステムとして位置付けられていました。その主な利点の1つは、アプリケーションの受領から配達まで「シングルウィンドウ」モードで注文を処理できることです。 サイト、ランディング、その他のシステムとの統合に関するさまざまな要求を満たすことができるオープンAPIを備えています。









RetailCRMサービスの説明



サイト



このシステムは、さまざまな電子商取引サービスと広く統合されています。 たとえば、Yandex.Kassaを使用すると、retailCRMを介してSMSの形式で請求できるほか、チャット、インスタントメッセンジャー、電子メールなどの便利な方法でクライアントに送信できます。



CRMのもう1つの機能は、トリガーシステムです。 システムを「トレーニング」して、特定のイベントに基づいてさまざまな自動操作を開始できます。 もちろん、それらの多くは、よく知られた主題を持つクライアントへの自動送信レターに関連付けられています。商品はバスケットに入っていますが、注文されていません-特別オファーでレターを書いてください。 クライアントはサイト上のいくつかの製品を見ました。 そして、これらの製品の割引などで、自動的に手紙を受け取ります。



レビューから判断すると、retailCRMはマーケティング担当者に人気があります。 彼女は、各注文、訪問者に関する多くの多様なデータを収集します：売り上げ、注文、利益、注文完了までの時間、クライアントの出身地、平均請求書、キャンセルの理由、支払い方法など。 レポートを使用すると、数回クリックするだけで必要なデータを取得できます。



システムのテレフォニー機能



テレフォニーが接続された標準CRMは、次の機能を提供します。



  1. 着信コールで、マネージャはCRMデータベースから顧客データを見ることができます。すでに存在する場合、スペシャリストは取引も見るので、今後の交渉の準備を改善できます。 それ以外の場合は、新しい注文を作成することを提案します。
  2. 顧客のカード、注文、タスクから発信する。
  3. CRMからのすべての会話の記録を聞く機能。


Zeon IP-PBXの説明



サイト



PBX Zeonは、2回のクリックでさまざまなcrmシステム（クラウドおよびボックス）と統合できる新しい革新的な製品です。 箱入りバージョンとクラウドバージョンの両方で利用可能なAsteriskコア上に構築された企業は、最大300人の内部加入者と、必要に応じて複数の通信事業者を接続できます。 ステーションは、IP電話やゲートウェイなど、さまざまなメーカーのデバイスをサポートします。後者を使用すると、アナログ回線をPBXに接続できます。



追加のモジュールを使用して、機能をアクティブ化できます。



  • コールセンター （さまざまな役割：管理者、スーパーバイザー、オペレーター/さまざまなパネル、オペレーター/番号/コールに関するレポート、作業の監視、キューの処理）
  • 自動リダイヤル （リストのインポートと作成、キューの生成とサポート、カスタムレポート）

一般に、電子商取引ビジネス向けのZeon RetailCRM + IP-PBXパッケージは非常に印象的です。 着信注文の強力なチャネル（店舗、CPAネットワーク、トレーディングフロア、モバイルアプリケーション、および着陸）には、簡単に整理されたインバウンドコールとアウトバウンドコールの両方のコールセンター機能に基づくものを含む電話通信のサポートが補充されます。



設定方法



統合するには、個人アカウントでRetailCRMを使用する必要があります。「設定」>「統合」>「APIアクセスキー」に進みます。 キーをコピーします。







次に、Zeonに移動します。 [統合]メニューに移動し、[RetailCRMとの統合]を選択します。 アドレスを示し、RetailCRM + / api / v5 /でドメインに対応します（例：https：//iptelefon.retailcrm.ru/api/v5/）。 そして、RetailCRMの個人アカウントで受け取った秘密鍵を入力します。 「保存」をクリックします







RetailCRMに戻ります。 「マーケットプレイス」に移動します。







「iptelefon.ru」アイコンを見つけます（下）。 クリックします。







テレフォニー設定メニューが開きます。 「電話」>「拡張コード」に移動します。 従業員の内部数を増やします。 「保存」をクリックします







すべて、セットアップが完了しました。



iptelefon.suのテレフォニーを使用する方が良いのはなぜですか？



  • 参加せずにすべての設定を行います。
  • 強力な電話システムが得られます。
  • すべてのサービスは1つの「ウィンドウ」で取得できます。 ATS、通信サービス、IP-SIMカード（FMCサービス）、ロシアの番号、機器（IP電話、ヘッドセット）;
  • 電話による技術サポート。
  • クイック接続と起動
