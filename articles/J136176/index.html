CES 2012 // Dell XPS 13





さらに6か月間のハンディキャップを取り、デルは、第二波ウルトラブックとともに、このクラスの最初の代表者を紹介しました。 会う：Dell XPS 13 Ultrabook！







まず、ケース。 このフレームワークを放棄したことで、同社のエンジニアは、11インチのフォームファクターに13インチのディスプレイを収容することができました。 XPS 13は13インチのMacBook Airよりも本当に小さいですが、12インチまたは11インチのデバイスのように見えます。 いいね！







車体はアルミニウムと炭素繊維でできており、最も薄い場所のモデルの厚さは6 mm強で、重量は1.36 kgです。 （厚い部分-約16 mm）







コーポレートキー、Dellキー。 もちろん、島型です。 タッチパッドは、その正確さと柔らかさが印象的です。

ウルトラブックの中には、運用作業に必要なものがすべて揃っています。 プロセッサの選択：Intel Core i3-2367M、i5-2467Mまたはi7-2637M、Intel HD 3000グラフィックス、4 GBのRAM、128または256 GB SSDドライブ。
