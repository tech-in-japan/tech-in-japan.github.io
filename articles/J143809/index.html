QuickBlox：サンプルおよびSDKの概要

こんにちは、ハブロフスク！



プラットフォーム-QuickBlox-モバイルアプリケーションのバックエンドに関する記事を続けるために、ダウンロード可能な開発ツールに注目したいと思います。



この記事とそれをここで書くというアイデアは、できるだけ多くのベータテスターをプロジェクトに連れて行くことを目的としています。機能に関する賢明な推奨事項を提供し、QA部門がバグをキャッチするのを支援できる本物のプログラマーです。



この段階でQuickBloxの使用は完全に無料であり、将来ベータ開発者はディスクプログラムに参加し、 QuickBloxチームの協力に感謝することができることを思い出してください 。



SDKとフレームワーク


iOS


iOS開発者向けに、 SDKの最新バージョンをダウンロードできます。 簡単な指示のおかげで、SDKをプロジェクトに簡単にインポートでき、クラスと関数は既に利用可能です。その助けを借りて、APIを使用してほぼすべての操作を実行できます。



また、SDKのすべてのクラスと機能を説明する、かなり馴染みのあるメソッドに関するドキュメントもあります。



Windows Phone


iOSの場合と同様に、開発者向けに、 Windows Phoneのフレームワークをダウンロードできます。 SDKをスタジオに追加するのは非常に簡単で、これでアプリケーションを作成する準備ができました。 SDKドキュメントはこちらから入手できます 。



Android＆Web / Facebook


この段階では、これらのプラットフォームのフレームワークが開発中であり、近い将来、プロジェクトでそれらを確認して使用できるようになることを願っています。 これで、アプリケーションをQuickBloxと統合できる例に従って、モバイル開発者を支援するためのサンプルが使用可能になりました 。



サンプル


QuickBloxメソドロジのサンプルは、 QuickBlox開発者コミュニティー向けにダウンロードおよび変更できるオープンソースプログラムです。 2種類のサンプルを提供しています。

  • スーパーサンプル
  • 簡単なサンプル


名前が示すように、スーパーサンプルは個別のすぐに使用できるプログラムです。 データ（認証キー）を変更し、デザインを変更し、機能を追加すると、アプリケーションのリリース準備が整います。 はい、それはとても簡単です！ はい！ 大規模ですぐに使用できるアプリケーション向けのオープンソースコードを提供します。



単純なシードとは、QuickBloxの機能の動作を記述する小さなコードです。 たとえば、ユーザーログインまたはチャットメッセージ。 開発者にとっては、すべてがシンプルで明快です。



サンプルはここにあり、次のプラットフォーム用に提供されています。

  • iOS
  • Android
  • Web / Facebook
  • Windows Phone
  • Phonegap


サンプルでは、​​次の機能が考慮されます。

  • 地図でチャット
  • 座標の定義
  • ユーザー認証
  • ユーザーのリスト
  • 送信-プッシュ通知の受信
