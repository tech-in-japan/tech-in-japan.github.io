Firefox 3.1の代わりに、Firefox 3.5がリリースされます

3月4日、次のFirefox / Gecko Delivery Meetingでは、特にMozilla Firefox 3.1ブラウザの次の安定バージョン（コードネームShiretoko）のリリースに専念し、このリリースのバージョン番号を3.5に増やす提案が発表されました。

この提案は受け入れられ、 3月6日に発表されました 。



このソリューションの理由は、ブラウザの新しいバージョンでの多数の変更と革新でした。
  • 新しいJavaScriptエンジンTraceMonkey。
  • HTML 5仕様の< video>タグのサポート。
  • 個人情報の安全性の改善された制御（IE8やSafari 4で利用できるような、プライベートブラウジングWebページを閲覧するための新しいモード）;
  • Webページの表示の大幅な改善。
< audio>タグと新しいCSS3セレクタのサポートを新しいバージョンに含めることもできます。



3月12日にリリースされる予定の次のベータは、まだ3.1b3にありますが、4番目は3.5b4になります。 ブラウザエンジンのバージョンは変更されません（Gecko 1.9.1）。



この決定は、IE8の次期リリースとSafari 4ベータの予想外のリリースにも影響されたように思われます。



その結果、知床のリリースはさらに遅れますが、Chrome、IE、およびSafariで最近登場したいくつかの革新を次のバージョンで紹介するためにこの一時停止を使用すると、私はそれだけになります。
