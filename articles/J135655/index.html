MicrosoftはWindows AzureでLinuxを実行することを許可します

マイクロソフトは今春、 Windows Azureクラウドサービスで仮想マシンを実行する機能をテストモード（Community Technology Preview）で起動することを計画しています。



現時点では、Microsoftの「クラウド」での仮想マシンの動作の特性は、状態を保存できないことです。つまり、仮想マシンをオフまたは再起動すると、すべてのデータが失われます。 最近まで、Microsoftはビジネスユーザーからの多数の要求にもかかわらず、サードパーティアプリケーション自体がこの機能を提供することを期待して、この機能をAzureに追加しませんでした。 ソースによると、この要因は、レドモンド社の「クラウド」がAmazon Web ServicesまたはGoogle App Engineの許容可能な代替手段と見なされなかった理由でした。



現在、状況は変わります。仮想マシンはクラウドサービスの機能を使用してデータを保存できるため、プログラマーの作業が大幅に促進されます。 また、既に説明したように、Azureは独自のLinuxイメージを展開し、SQL ServerおよびSharePointアプリケーションを実行できます。 同時に、クラウドコンピューティングの分野におけるMicrosoftの主要な競争相手であるAmazon EC2は、WindowsとLinuxの両方を実行する機会を長い間提供してきました。
