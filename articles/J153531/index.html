新しいImagine Cup 2013をご覧ください！

今年、 Imagine Cupは2倍の素晴らしいものになるでしょう！ 最初に、2013年夏にサンクトペテルブルクのロシアで国際決勝戦が開催されます （ロシアの学生は参加する可能性が非常に高くなります）。 第二に、競技自体が今年大幅に更新され、はるかに良くなりました！ 以下の詳細！









今年のコンペには、世界中で開催される3つの主要なデザインコンペが含まれます。

  • ソーシャルプロジェクトは、ソフトウェアプロジェクトのかつての競争です。 人類の緊急の問題を解決する解決策を提示する必要があります。 医学、教育、飢againstとの闘いなどの分野で美しく活気のあるプロジェクトを提出することは合理的です。
  • ゲーム -このコンペティションはオンラインコンペティションではなくなり、他の2つのプロジェクトコンペティションと一緒にロシアおよびその他の国で開催されます。 ゲームを作成する場合（または既に作成している場合）、Windows Phoneスマートフォン、Windows 8、XBox、またはWebのいずれかのマニフェストで、ゲームコンテストに応募してください！
  • イノベーションは、私たちが長い間待ち望んでいたコンテストです！ 上記の2つのカテゴリのいずれにも該当しないプロジェクトをImagine Cupに絶対に提出できます。


プロジェクトコンペティションは、ロシアで開催されるほか、過去数年間にソフトウェアプロジェクトのコンペティションが開催されます。

  • 申込期限は2013年2月15日です。
  • 地域の決勝戦とオンラインツアー（地域の決勝戦に直接参加できない人向け）-2013年3月
  • ロシア決勝-2013年4月


今年はロシアからの国際決勝戦に3チームが参加します -各プロジェクトコンテストごとに1チーム（ロシアはホスト国としてこの権利を受け取りました。他のすべての国からは1チームのみが残っています）。 同時に、1つのプロジェクトはプロジェクトコンテストの1つでしか提示できません（場合によっては、プロジェクトを2つに分割することもできます）。そのため、プロジェクトが最も有利になるカテゴリを慎重に検討してください。



決勝戦に進むための追加の機会は、特定のテクノロジー（ Windows 8 、 Windows Phone 、 Windows Azure （および、場合によっては他のテクノロジーが追加される））に従って開催されるオンラインテクノロジーコンテストです。 これらのコンペティションでは、 まずimaginecup.comウェブサイトで少しのオンラインテストを経てから、国際審査員による検討のために申請書を送信する必要があります。 最高のアプリケーションを作成したチームは、国際決勝に招待されます。



また、Windows 8、Windows Phone、Azure向けのロシアアプリケーションコンテストも開催します。これは、他のロシアの学生と競い合って賞品を獲得するもう1つの理由です。 もちろん、主な賞品はアプリケーションであり、Windowsストアの数百万人のWindows 8ユーザーが利用できます。



今後数週間で、各コンテストの詳細をお知らせしますが 、更新されたウェブサイトimaginecup.ruで 登録できるようになりました。 登録する際、 地元のアイデアコンテストにも参加するために、あなたのアイデアを簡単に説明してください。 はい。今年は、既製のアプリケーションだけでなく、審査員が最も興味深く期待できるアイデアの著者にも賞を贈ります。 また、アイデアを 「 アイデアの交換 」に置くこともできます。この場合、アイデアに興味がある他の学生があなたに連絡してプロジェクトチームに参加し、Imagine Cupコンペでこのアイデアを提示します。







コンテストの詳細については、当社のWebサイトwww.imaginecup.ruをご覧ください 。 国際大会の公式ルールはwww.imaginecup.comで入手できます。 コンテストに関するすべての質問には、Twitter @shwarsで個人的に、この投稿に対するコメントで、またはrusacad@microsoft.comへのメールでお答えします。



今年はImagine Cupに参加するのに最適です！ 第一に、 あなたは実際にコンペティションの主な賞である国際ファイナルに触れるチャンスがはるかにあります ！ 何らかの形で、3つのファイナリストチームがファイナルに参加できるだけでなく、競技会やロシアのファイナルに積極的に参加している他の参加者も参加できることを願っています。 第二に、Microsoftテクノロジーを何らかの方法で使用する学生プロジェクトは、更新された競争構造に参加できます！
