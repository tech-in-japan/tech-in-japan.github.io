Microsoft Surface RTの長期使用後の印象

数ヶ月前、その好奇心ではなく、iPad所有者の灰色の群衆から目立ちたいという欲求（私自身もその一部でしたが）ではなく、マイクロソフトから新しいアイテムを購入するように勧められました。 Surface RTは非常に珍しくて興味深いデバイスのようでしたが、私の意見では注目に値します。 現在、購入から十分な時間が経過しているため、デバイスの実際の長所と短所に基づいて正確な意見を作成できます。



仕事とレジャーのメインタブレットとして、ほぼ毎日Surfaceを積極的に使用しました。 仕事とは、ドキュメントとメールで仕事をすること、休憩、映画を見ること、朝のコーヒーでニュースを読むことです。



Surfaceを購入した後、私が最初にしたことは、「タッチカバー」キーボードケースを購入することでした。これがないと、タブレットはまったく役に立たないからです。 キーボードとタッチパッド（特にオフィススイート）を使用せずにデスクトップインターフェイスを使用するのは簡単です。 「Touch-Cover」は非常に平均的であることが示されています。キーボードを入力する場合、キーボードを入力する方が画面の半分を占有しないため、キーボードで入力する方が便利です。 さらに、2か月使用した後、カバーは端で非常に摩耗し、コネクタの近くで曲がります。 Surfaceを購入しても、利便性と信頼性が高いため、「タッチカバー」ではなく「タイプカバー」を選択することをお勧めします。



ピル自体に戻りましょう。 ケースは非常に快適に見えますが、実際にはひどく汚れやすい金属でできています。 長時間使用した後でもそれほど摩耗しませんでした-実質的に傷はありません。 残念ながら、この金属は乱暴な取り扱い中に多くのしわになるので、特にSurfaceを落とさない方が良いです。

タブレット画面の対角線は10.6インチで、幅の広いフレームがあります。 Surfaceの後、9.7および10.1インチのタブレットは小さくてコンパクトに見えます。 このような対角線は、キーボードケースで作業する場合はプラスですが、Surfaceの重量を維持する必要がある場合は大きすぎるようです。 ホームボタンとカメラの目はまだフロントパネルにあります。タッチボタンの反応は中程度で、カメラはSkypeでの会話にのみ適しています。



タブレットの側面には、非常に豊富なインターフェイスのセットがあります。 左側のヘッドフォンジャックとmicroSDカードスロットに加えて、右側にMicro HDMIがあり、Surfaceの最も重要な利点の1つは本格的なUSBポートです。 タブレットからスマートフォンを充電する機能が私の好みになったと認めます。 iPhoneのバッテリーは目の前で放電します。タブレットでバッテリーの寿命を延ばす機能は非常に便利なオプションです。 既に述べた充電に加えて、USBはデバイスへのファイル転送の問題を大幅に簡素化します（クラウドサービスは私たちのすべてですが、特定の状況ではシンプルなフラッシュドライブがとても便利です）。また、外付けハードドライブから映画を見る機能も非常に快適です。 私は、MSのこの製品の大きな欠点の1つに言及するのを忘れていました。側面には、うんざりするようなステレオスピーカーがあります。 最近試したどのタブレットのスピーカーよりも静かです。



デバイスのソフトウェアコンポーネントについて説明します。 ご存知のように、このバージョンのSurfaceは、Windows 8のフルバージョンではなく、モバイルタブレットのWindows RTでは動作します。 RTでは、古いバージョンとOSの主な欠点との唯一の大きな違いは、以前のバージョンのWindowsからアプリケーションを実行できないことです。 つまり、Desktop OfficeおよびIE10にプリインストールされているものに加えて、そこに簡単にインストールできるものは何もなく、ソフトウェアの選択はWindowsストアからの提供によってのみ制限されます。 理論的には、タブレットはタブレットアプリケーションで動作するはずであるため、これはユーザーを悩ませるべきではありませんが、キーボードカバーを備えたトランスデバイスの原理そのものがこの状況で多くの利点を失い、メトロ（Microsoft UI）アプリケーションの数はまだ限られています。 この状況で最も不幸なのは、数か月で最小数の新しいメトロアプリケーションがリリースされたことです。 ソフトウェアの重大な不足に対する唯一の解決策は、人気のあるサービスのWebバージョンを使用することです。 Facebook、Evernote、VK-これらはすべてブラウザで使用しなければなりませんでした。

システムインターフェイスの全体的なパフォーマンスを5つのうち3つに評価します。重いアプリケーション（同じExcelまたはPowerPoint）で作業しているとき、タブレットは目立ちます...式に申し訳ありません-遅れています。 不規則なスクロール、UI要素の押下に対する応答の遅延、アプリケーションの長時間起動は、Windows RTを使用する際の最も一般的な問題です。

しかし、悲しいことについては十分です。 Windowsタブレットの利点は、消費よりもコンテンツの作成に重点を置いていることです。 プレインストールされたOfficeを使用すると、ほぼすべての種類のドキュメントを快適に操作できるため、出張中にラップトップの代わりとしてSurfaceを非常によく使用しました。 フラッシュ対応IE10はWindows RTに追加するのに適していますが、フラッシュの必要性はあまりありません。

私の意見では、現時点では、Windows RTはあまりにも粗雑なままです。 リリースから数か月たっても、OSは依然としてSurface RTの弱点のままです。 Windows Blue（8.1）で状況は少し改善するかもしれませんが、根本的な改善を期待すべきではないことは明らかです。



あとがき


膨大な数の欠点にもかかわらず、Microsoft Surfaceは私にポジティブな印象を残しました。 タブレットの便利さとラップトップの機能を組み合わせたデバイスのアイデアは、私にとって非常に興味深いようです。 このような概念は、マルチメディアタブレットとウルトラブックの間の空きスペースを占有する可能性があります。 残念ながら、このデバイスは両方のタスクに対処できるため、タブレットまたはオフィス作業用のラップトップの代替としてSurface RTをお勧めしません。 一方、Windows 8とIntel Atomを搭載したタブレットには多くの人が注意を払うことをお勧めします。 Surface RTとAtomタブレットを直接比較すると、Intelチップ上のデバイスがOSのはるかに優れた滑らかさと速度を実証していることに気付くのは簡単です。そして、最終的には、古典的なWindowsプログラムを使用する機会を奪われません（タブレット上の同じVisual Studioは非常に興味深いようです）提供）。 Windows 8は、OS市場におけるMSにとって大きな一歩ですが、Windows RTの改善は依然として必要であり、RTタブレットに時間とお金を費やすことはお勧めしません。
