ステップバイステップ：FCS（税関）でのIGO（小包）クリアランス

0.購入



オンラインストアで商品を購入し、郵便で配送します。 パッケージを受け取るために郵便局に来てください...そしてもし受け取ったら、それ以上読むことができません。

小包を配達する代わりに、ロシアの郵便局の従業員が税関に拘束されたと喜んで報告した場合。



1.不達



郵便局では、小包に添付された通関申告書の通知とコピーを受け取ります。 通知には、必要に応じて税関の住所が含まれますが、事前に2gisまたは利用可能なサービスを通じて税関の電話番号を見つけて、質問の市民の受付場所と時間を指定することをお勧めします（私たちの市では、別の住所での登録は主要税関ビルにあります）。





2.文書の準備



小包はロシアの税関の粘り強い足に落ちたので、返送するためにいくつかの（大きすぎない）書類を収集する必要があります。

  • 商品の購入に関するオンラインストアの請求書。
  • 銀行取引明細書（重要：日付と金額は口座と一致する必要があります）。
  • 自分とすべての直接の親relative（親子、兄弟姉妹、夫、妻など）のパスポートのコピー、およびあなたの血縁関係を証明する書類（出生証明書、結婚登録）のコピー。
  • 税関が税関申告に従って小包内の商品を特定できない場合、店舗のウェブサイトから商品の写真を印刷する必要があります。
  • 商品の中に輸入制限のあるものがある場合は、消費者権利保護と福祉の監督のための連邦サービスの登録証明書が必要です。ロシアでこのタイプの製品の輸入が許可されていることを証明します。


パッケージに同様の製品がたくさんある場合は、親relative向けのドキュメントが役立ちます。



3.最初の訪問



税関に来て、パスポート、通知、申告書のコピーを窓から提出してください。 次の2つのフォームが発行されます。

  • 税関管理のためのアプリケーション。
  • 説明（ここには、何が何の量で、あなたの直接の親relativeのどれがパッケージに入っているかを書いてください）。


フォームに記入し、小包を開くためのアプリケーションを作成します（彼らは写真を撮って宣言と比較し、犬と一緒に犬のハンドラーも存在します）。









4. N回目の訪問（お金について）



最初の実行から、すべてが一緒に成長できるわけではありませんが、このメモを読んだ後は、多少なりとも準備ができています。 はい、すでに第2項で書いたように、従業員は輸入制限のある商品について疑念を抱いているかもしれません...そして、1か月の期間の商品の検査が任命されるかもしれません。 -あなたのケースで可能であり、あなたにとってより都合が良い場合、送り主への返品で小包の受け取りを拒否する方が良いです。そうでない場合、このコメントで説明されているように、検査のために商品を引き渡す必要があります。

したがって、すべての書類が税関職員に適しており、主な問題が同じタイプの複数の製品を輸入したことだった場合、お金を払う必要はありません。

1,000ユーロの金額を超えた場合（重要：申告書が別の通貨である場合、費用は申請日の為替レートで計算されます）、この金額を超える30％の手数料を支払う必要があります。

MPOが商業貨物として認識されている場合、出荷金額の30％の関税が支払われますが、最も不愉快なのは、商品が税関仲介業者を通じて（契約価格で）処理されることです。

私の銀行のオペレーターはフィールド107（税関当局コード）に記入せず、モスクワのFCSにお金が詰まっているが、市の税関には来ていないため、支払うときは、すべてのフィールドの完了と支払注文の詳細を慎重に確認してください。 システムの詳細を修正するようにモスクワにファックスを送信し、銀行に電話してファックスも送信し、その後モスクワに電話をかけなければなりませんでした。



5.最後の訪問



それで、あなたのお金は税関に達しました。 支払い部門に来る必要があります。 また、一定数のドキュメントとそのコピーが必要になります（！）：

  • パスポート
  • 納税証明書（TIN）。
  • 支払いまたは支払い注文の領収書。


支払い部門では、支払い注文にマークが付けられます。





税関職員は幸せです-あなたは彼らのすべての要件を満たしています。 彼らはすべての必要な文書を発行します。 税関の領収書の令状と通知が与えられます。 郵便で小包を発行する許可に通知がスタンプされます。 解像度をスキャンしに行かず、喜んで電話で写真を撮るのを忘れたので、写真はありません。



6.プレゼンテーション



郵便局に行って、もう一度小包を開いて、投資と申告書を比較し、それから自分のために大胆にそれを拾います。
