Microsoft Exchangeに代わる価値のあるZimbra Collaboration Suite

近年、マイクロソフトのロシアオフィスの閉鎖の可能性に関する情報が積極的にメディアに広まりました。 それはすべて、企業がソフトウェアのライセンスの直接販売を停止し、小売業向けに再設定したという事実から始まりました。 次に、公式のデジタル配信ストアで、 価格がドルに変換されました 。 ロシアに正式に登録された法人を有する企業のみがルーブルで決済を実行できることを考慮すると、ロシアのマイクロソフトオフィスを閉鎖するだけでなく、それに関連する法人も、したがってソフトウェアを使用できない可能性について話すことができます公共部門のマイクロソフトから。 また、Microsoftは公式コメントでロシアでの活動を続けることを約束しましたが 、何らかの理由で、法的法的地位と実行するタスクを指定しませんでした。 起こっている政治的な出来事を考えると、現在の状況は不確実性につながり、最も慎重なITマネージャーはすでにマイクロソフトのソフトウェア製品を置き換えることができる代替ソリューションを検討し始めています。







そもそも代替品を探し始めるべき重要な製品の1つは、もちろんMS Exchangeです。 企業でコラボレーションおよびメールソリューションを使用している場合、おそらく、今日のビジネスで同期されたカレンダー、グローバルアドレス一覧、オンラインチャット、ビデオ通話などにどれだけ関係しているのかご存じでしょう。 ある時点でコラボレーションシステムの合法的な使用が不可能になるという見通しは、ビジネスおよび国家機関の通常の機能にとって深刻な脅威となります。 しかし、良いニュースがあります。 Microsoft Exchangeは、無料のオープンソースソリューションであるZimbra Collaboration Suite Open-Source Editionに置き換えることができます。



現在、Zimbraは電子メールの分野における主要なソリューションの1つであり、電子メールサーバーだけでなく、カレンダーとコラボレーションの機能も提供できます。 Zimbraはシステムリソースを必要とせず、どのLinuxディストリビューションにもインストールできるため、世界中で非常に人気があります。 現在、Zimbra Collaboration Suiteのユーザー数は5億人を超え、その多くはSaaSプロバイダーを通じてZimbraを使用しています。 毎日、ユーザーはカレンダー上で数十万のイベントを作成し、テラバイトの電子メールを送受信し、ファイルを相互に共有します。 Zimbraは、ユーザーがお金だけでなく時間も節約できるように支援します。これは、本当に価値のある、かけがえのない唯一のリソースです。



しかし、Zimbra Collaboration Suite Open-Source Editionが何かのせいにされている場合、実際のビジネス環境での使用にはあまり適していないということです。 この分野で要求されるすべての機能、たとえば、リアルタイムバックアップシステム、データ圧縮および重複排除システムは、開発者によって製品の商用バージョンに慎重に移行されました。 同時に、Zimbraの商用バージョンの高い価格と多数の機能に誰もが満足するわけではありません。Zimbraの実際のビジネス環境ではアプリケーションが見つかりません。







Zimbraの無料版の機能が不足している人向けであり、商用版によって提供される機会は冗長に思えます。イタリアの会社Zextrasは、Zimbra Open-Source Editionに最も必要な機能のみを追加するモジュール拡張のZextras Suiteパッケージを開発しましたが、同時にはるかに安価です。



Zextras Suiteのおかげで、Zimbra Collaboration Suiteの無料版は、高度なバックアップ、災害復旧、マルチテナンシーサポート、管理者委任、テキストおよびビデオチャットサポート、階層ストレージサポート、およびデータアーカイブとデータ重複排除システムを追加して、ディスクスペースを節約できます。 さらに、Zextras Mobile拡張機能により、Zimbraをモバイルデバイスと同期する機会が得られるだけでなく、Microsoft Outlookとの互換性が追加されます。これは、Exchangeを使用してZimbraに移行するユーザーにとって非常に便利です。 このような互換性により、移行が完了した後も電子メールユーザーは使い慣れたプログラムを使用し続けることができます。つまり、移行プロセス自体がよりシームレスになり、企業の従業員のトレーニングに要する時間が短くなります。



MS ExchangeからZimbraへの移行プロセスは非常に簡単です。 これらの目的のために、ZCS Migration Wizardユーティリティは特別に開発されており、カレンダー、メールボックス、連絡先リストをZimbraに転送できます。 使用方法の詳細な手順については、以前の記事をご覧ください 。 いずれにしても、不可抗力の発生を待たずに、無料のZimbra Collaboration Suite Open-Source EditionおよびZextras Suiteの試用版を今すぐダウンロードし、このソリューションがあなたのビジネスに適しているかどうかを個人的に確認することを強くお勧めします。 ルイス・キャロルが書いたように、 「同じ場所にとどまるためにできるだけ速く走らなければならない」と今日、このフレーズはロシアのIT分野に安全に適用できます。
