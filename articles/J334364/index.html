デザイナーと開発者向けの7月の無料Cookie





この号では、最も興味深い無料Cookieの選択。 ほぼ毎日、デザイナーや開発者の作業を最適化する興味深いアプリケーション、リソース、ライブラリが数多く公開されています。 この号では、7月にリリースされた西洋のコミュニティの意見で最も興味深い製品を収集しました。



スケッチ： Supernova Studio、Qwikly、スケッチマテリアル

Webアプリ：用紙サイズ、グラビエント、Canvaカラー、TypeHero、抽象

開発者向け： MDB Angular GMDキット

デザイナーのもの：メンフィスパターン、Travelisto UIキット、フローフォント、Lists.design、WayFXロゴ





ちなみに、Figmaを使用する場合は 、 既製の設計システムに注意することをお勧めします 。 フリーランサーが1か月あたりの注文を増やすのに役立ち、プログラマーは自分で美しいアプリケーションを作成できます。また、チーム作業用の既製の設計システムを使用して、チームが「スプリント」スプリントを迅速にリードします。



また、深刻なプロジェクトがある場合、当社のチームは、ベストプラクティスに基づいて組織内に設計システムを展開し、Figmaを使用して特定のタスクに合わせて調整する準備ができています。 Web /デスクトップ、および任意のモバイル。 また、React / React Nativeにも精通しています。 Tへの書き込み： @kamushken








超新星スタジオ



https://supernova.studio/



Sketchへのプラグイン。デザインをすぐにXcodeに変換し、すべてのリソースを開発者に安全に転送できます。 Androidもサポートされています。 彼らが真実を語れば、このことは長い間待っていて、それは途方もない可能性を秘めています。











Qwikly



https://www.getqwikly.com/



Qwiklyは、Sketchプロトタイプツールにプロパティを与える試みです。 スマートフォンですぐにデザインをテストできます。画面間にはいくつかの標準的な切り替え効果があります。 アイデアをすばやくテストしても、誰にも害はありません。











スケッチ素材



https://websiddu.github.io/sketch-material/



GoogleマテリアルガイドのUIをSketchですぐに組み立てる必要がある場合は、このプラグインが最適です。 テーブル、カード、ボタンなど GMDキットには、仕様に従って作成されたすべてのコンポーネントがあります。











用紙サイズ



http://papersizes.io/



印刷に取り組んでいますか？ シート、封筒、名刺、ポスターなど、すべての国際的なフォーマットと標準が役立ちます。 これらはすべて、最新のタイポグラフィとスタイルのソースを含む興味深いUIラッパーの形式で提示されます。











マテリアルデザインの角度付きブートストラップ



https://mdbootstrap.com/angular/



Angularの別のUIフレームワーク。 高品質で作られています。 開発者によると、400以上の素材要素、600以上のアイコン、74のCSSアニメーションが含まれ、jQueryはまったく含まれていません。











グラビエント



https://www.grabient.com/



一般的なブームの中で：別のオンライングラデーションビルダー！ 以前のものよりも高度です。角度を変更したり、独自の色を追加したり、透明度を調整したりできます。 スケッチにエクスポートしますか？ もちろん。











カンバの色



https://www.canva.com/colors/



新しいデザインプロジェクトを開始し、色のアイデアはまったくありませんか？ 次に、キャンバスであなたに。 既製のパレット（多く）、その適用可能性、例、詳細な説明の形での心理学のビット。











タイプヒーロー



https://typehero.now.sh/



Googleフォントの機能の一種の非公式の拡張。 色を操作したり、さまざまなパレットをテストしたりできます。 フォントペアを収集するための十分な能力がなく、より多くのスペースが使用されないため、より良い結果が得られます。











アブストラクト



https://www.goabstract.com/



Abstractはデザイナーにとってgitとして位置付けられていますが、これまでのところSketchとの友だちのみです。 彼らは残りが近づいていることを約束します。 Webバージョンを使用するか、macOS用にダウンロードできます。 Startanuliは最近、視聴者の関心が高まりました。 私の練習ではまだ必要ではありませんが、この製品には何らかの成功があります。











10メンフィススタイルパターン



https://pixelbuddha.net/freebie/10-memphis-style-patterns-free



これは無料のメンフィススタイルの背景のセットです。 Dribbbleでいいねを集めるのが好きなら、これはあなたのためです。 今、三角形、円、正方形のないショットはほとんど興味がありません。 そして、トレンドに反対する議論はありません...











スケッチ用のTravelisto UIキット



https://dribbble.com/shots/3624797-Travelisto-UI-Kit-For-Sketch



旅行スケッチ用の無料のUIキット。 一般に、他のトピックについては簡単にやり直します。 このコンセプトは、簡潔なカードの形でのトレンドの垂直スクロールに基づいています。 ロバート・アニテイ、ありがとう。











フローフォント



http://danross.co/flow/



Flowは、ラピッドプロトタイピング用の無料フォントです。 魚のテキストの現代的な代替。 3つのスタイルでダウンロードできます。 アブラカダブラでドライブし、スペースで休憩を入れます。 便利に！











リスト



https://www.lists.design/



ただし、Hi-Fiプロトタイプを作成している場合は、実際のテキストなしではできません。 そして、リストプロジェクトが助けになります。住所、姓、都市、役職などです。リストの形式のあらゆる種類のテキスト。 JSONへのエクスポートが可能です。











Wayfxロゴ



https://wayfunction.com/logos/



スタートアップを立ち上げてロゴを発明する時間がない場合-WayFxは急いで支援し、さまざまな最新のエンブレムを無料で提供します（商用利用を含む）。 SVGおよびAI形式をダウンロードできます。











Uxフロー



https://free.lstore.graphics/uxflow/



UX Flowは無料のUIプロトタイピングキットです。 Sketchに適応し、ユーザーのフローパスをすばやくカスタマイズおよび構築する機能を備えた標準画面のユニバーサルセットを提供します。











ギットポイント



https://gitpoint.co/



iOS用の無料のシンプルでモダンでスタイリッシュなGitクライアント。 開発者は、製品には必要なすべての機能などが備わっていると主張しています。











サウンドキット（by Facebook）



http://facebook.design/soundkit



デザートには、モバイルアプリ用の無料のFacebookサウンドキットが含まれます。 主なもの-ユーザーに刺激を与えないように、製品にサウンドを実装して無理をしないでください。
