リットルは、1900冊の本を9,600万ルーブルでデジタル化します

Liters社は、国立電子図書館の基盤と連携するための州契約に署名しました。 そのため、電子書籍ストアは、11000冊の本を9,600万ルーブルでデジタル化することを約束している 、とRoem は書いています 。 Litersとの契約に署名する前に、デジタル化された本は非営利団体National Library Resourceから購入されました。



国家契約の初期費用は1億200万ルーブルでした。 リットルは最初の価格より5.8％低い価格を提示し、契約を受けました。 同時に、同社はBest Buk LLCとNational Library Resource ANOをバイパスしました。 現在の契約額に基づいて、1冊の本をデジタル化すると、顧客に8700ルーブルの費用がかかります。 以前、1冊の本のデジタル化にはNEB 11,000ルーブルがかかりました。



この契約は、（請負業者の側で）著作権保有者とライセンス契約を締結する必要があること、および完成した作品を電子形式で顧客に転送することを示しています。 デジタル形式の本がない場合、請負業者はデジタル版（PDF / AまたはEPUB）を独自に作成する義務があります。 すべての本のデジタル化のために、出演者には3か月が与えられます。



Litersには5万から6万タイトルの作品のリストがあり、そこから1万1千冊の本を選択できることに注意してください。 書籍は、文化省の下での科学評議会の勧告、ロシアブックユニオンの専門家の評価および公開討論を考慮に入れて、RSLおよびロシア国立図書館の読者の調査に基づいてリストに選ばれます。
