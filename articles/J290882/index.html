RVCとR-Pharmによって設立されたこのファンドは、健康診断データの分析用ソフトウェアに1億6000万ルーブルを投資しました。

ロシアのファンドRBV Capitalは、 Image Analysis Ltdに 1億6000万ルーブルを投資しました。 同社は、免疫疾患および腫瘍疾患の治療における磁気共鳴画像（MRI）検査データの視覚化のためのソフトウェアを開発しています。



調達した資金は、Dynamicsソフトウェアプラットフォームのさらなる開発、臨床センターの開発、およびImage Analysisのクライアントベースの拡大に向けられます。



今日、健康診断データの定量分析用ソフトウェアの市場は活発に発展しています。 このソフトウェアは、炎症性疾患や腫瘍性疾患、筋骨格系の疾患、神経変性疾患、心臓灌流、その他多くの疾患の診断に使用されます。



画像分析は、類似の開発のリーダーです。 以前は、医師は質的評価、自分の経験、および患者の反応にのみ依存して、検査の結果を取得する必要がありました。



画像分析の開発により、専門家は疾患の動態のデジタル分析を実施し、中間治療結果を特定できます。 このようなソフトウェアを使用すると、診断の精度が向上すると同時に、コストが削減されます。 さらに、診断プロセス自体が大幅に加速されている、と同社はプレスリリースで述べています。



Image Analysis Ltdの創設者兼CEOであるOlga Kubassova博士：

Dynamics for MRIを使用する臨床および研究会社の数が増えていることを嬉しく思います。 広大な領土と有能な人材が不足しているロシアの状況、特に地域では、当社の製品が大きな需要があると考えています。 会社の現在の製品の開発への重要な貢献はロシアとベラルーシの専門家によってなされました、そして、我々はこの協力を続けるつもりです。


アンドレイ・コズロフ、RBVキャピタルの投資ディレクター、Image Analysis Ltdの取締役会メンバー：

Image Analysis Ltd.の専門チームと仕事をする機会を得たことを嬉しく思います。 現代医学では、データと客観的基準の詳細な分析に基づいたMRI検査の結果の定量的評価の需要が高まっています。 この機会は、画像解析の独自の開発によって提供されます。 会社の発展をサポートするとともに、有望なロシア市場の発展を支援できることを嬉しく思います。


基金の創設者はRVCとR-Pharmです。 RBV Capitalは2014年に設立されました。 基金は、その資金の半分をロシア企業に投資する予定です。
