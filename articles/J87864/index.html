7番目のテープドライブユース



新しいHP StorageWorks DAT 320テープドライブは、主にバックアップの必要性について考え始めたばかりの人向けに設計されています。 原則として、この段階では3つの問題が潜んでいます。



まず、あなたの心では、テープドライブはDr.Albanグループまたは遠い90年代の他の兆候と漠然と関連付けられています。 現在、さまざまな再利用可能なデバイスでデータストレージ市場がほぼ完全に妨害されている時代に、私たちはしばしば古い実績のある技術を捨てています。 同時に、テープドライブで1ギガバイトのデータを保存するコストと他の種類のメディアを比較することはできないという再利用の考えが忘れられています。



2番目の問題は、ハードドライブまたはフラッシュドライブ以外の操作に本当に慣れている（または慣れていない）場合です。 ここでは、外付けUSBドライブオプションが役立ちます。これは、デスク上のもう1つの素敵なボックスになります。 コンピューターの分解/組立手順が複雑に思えない場合は、SASインターフェースを介して接続する内部マウント付きのデバイスを購入することで、少し節約できます。



そして、ここでは第3の波に触れています-デバイスを使って作業を整理する方法です。これまでに遭遇したことがないのですか？ 完全なソフトウェアであるHP Data Protector Express Single Server Editionを活用します。 ほとんどの場合、このソフトウェアは初心者ユーザーの要求の90％を満たすことができます。これが必要なことです。



さて、第7世代は何をもたらしますか？



小さいけれど賢い！



最初の祖先が前世紀の80年代初頭に登場したドライブでは、何が改善できるのでしょうか。 結局のところ、すべてが長い間発明されてきました！



しかし、技術と技術思想はまだ止まっておらず、その結果は次のとおりです。



•予想される容量の増加。 古き良き伝統によると、2回、今ではマッチ箱の形とサイズに似たリボンで、最大320 GB（圧縮2：1）に適合します。

•書き込み/読み取り速度の向上。 データは最大86 Gb / hの速度でこのボックスに送られます。 カセットは4時間以内に記録されます。

•デバイスの消費電力が半分になりました。 これは、理想の追求がその結果をもたらし、ユーザーにとって非常に心地よいものであることをもう一度証明します。



沈黙する習慣は何ですか



自問自答してください。テープドライブの購入と使用について、お金と時間の無駄ではなく、サーバーの問題が発生した場合の大きな助けとして考え始めたのはいつですか。 この質問への頻繁な答え：決して。 サーバーがダウンしたときにいくつかの不快なケースを経験した後でも、オフィスが働き始め、フォークやトーチを備えた同僚の群衆がテーブルを包囲しました。



そして、ここで無駄に！ 結局のところ、すべてのHPテープドライブの所有者は、 One Button Disaster Recovery （OBDR）機能を使用してサーバーファームを基本的に保護することができます。 そのおかげで、OS、アプリケーション、またはデータの損傷によって損傷を受けたProliantは、DVD-ROMなどの最後のバックアップカートリッジから起動し、事故前の状態を完全に復元できます。 これらはすべて、ボタンをクリックするだけです。 この場合、ダウンタイムは数分に短縮されるため、従業員はもう一度お茶を飲むだけで済みます。



その結果、HPの現在の世代のテープドライブは、あらゆる種類のストレージデバイスの中でおそらく最も開発された技術的基盤とともに、ネットワークの最新のインフラストラクチャに適合することができるすべての属性を備えています。
