Asus UL30aで2か月

この記事は、このラップトップの2か月間の「ユーザーエクスペリエンス」に関するレポートの要素をレビューすることを目的としています。 Habréでこのラップトップのレビューはなく、モデルは興味深いので、レビュー自体に加えて、読者が長いコメント（2か月-長い作業を将来判断し、それを出すことができる期間についても読むことも興味深いと思います）私は、このラップトップを日常の仕事/研究で使用するユーザープログラマーとして、製品の1つまたは別のコンポーネントに関する第一印象に基づいた意見をすでに持っています。 それでは始めましょう。



モデルに関する一般的な情報と、私がそれを選んだ理由。



このラップトップはAsus UnLimited Seriesの一部です。つまり、ULV（超低電圧）シリーズプロセッサに基づいて構築された軽量で薄型の長寿命ラップトップを提供します。これにより、かなり効率的なプロセッサ（このシリーズのプロセッサには、celeron、pentium、コア2のソロ/デュオ周波数1.2〜1.6 GHz、さまざまなキャッシュオプションTDPは10Wからです。これはかなり長いです、あなたは同意する必要があります）。 このようなプロセッサーの欠点は、周波数が低下することですが、インテルCentrinoプロセッサーのままであるため、Atomから完全に切り離され、快適に作業できます。



Core Duo 1.6に基づいて構築された古いAcer Aspire 5561の交換に選んだラップトップは、タスクに十分なパフォーマンスを発揮しましたが、作業時間（3h）と寸法/重量（14 "/2.3 kg）の両方が私は容認できませんでした-5時間の作業時間とより少ない重量が必要でした（バックパックで1日中持ちます。すべてが寸法でシンプルでした-14は大きすぎましたが、「小さいほど良い」という原則は私にとってはありませんでした-私は見ました11「モデル-小さな画面から目をつぶしたくありません。12〜13の間で何かが選択されました」 原則として、バックパックに快適に収まり、デバイスのその他の必要なパラメーターがより重要な役割を果たしました-まずは動作時間。



また、ラップトップには最低でもwifi / BT /カードリーダー/ 3USBが必要でした。 VGA / HDMIは歓迎されました-家でテレビと24 "monik、私はそれをカットしたかったです。CD/ DVD-RW-原則的ではありませんでした。価格-まあ、道徳的なしきい値は3万で、Sony、MacBook Pro、Asus U20、 ThinkPad x200（秒）。



合計-Acer、HP、ASUS、LenovoのULVのモデル間ですべてが選択されました（Atomは原則的に私には向いていませんでした-photoshopとVisual Studioは実行する価値のあるメインプログラムでした）。 以前の品質の経験からすぐに言います-エイサーはすぐにマークされました、私はHPコーニーが好きではありませんでした、選択はLenovo U330、Asus UL20、Asus UL30の間でした それから彼は実際にトピック、フォーラムを中断し、より便利なものを理解しようとして店内のラップトップで「遊びました」。 そのときまでに、私はCore 2 Duo（特定のモデルについて話す場合はSU7300）にラップトップが必要であると判断しました。これは、まず、2つのコアとノーカットキャッシュ、そして次に、Win7で仮想マシンとXP Mode 。



UL20は、6セル以上のバッテリーが付属していないこと、HDMIが欠落していること、UL30の動作時間が長いこと、およびキーボードがpbup / down / homeなどにトリミングされていることに気付いたときに「落ちました」正しく食べる。 レノボは、フォーラムでの最高のレビューではなく、バッテリーとバッテリーの寿命のために「落ちました」。 そこで、UL30を優先して選択し、拡張バッテリー（8セル、5600 MAh）とSU7300を搭載したモデルを探し始めました。 UL30には2色のボディカラーがあります-グレーと黒、黒は汚れやすく、グレーは優れています。 グレーが欲しかったのですが、ロシアのSU7300 +グレーはまだ配達されていなかったようです（さらに12月には輸入されなかったので、SU7300 UL30を見つけるのは困難でした）。ちなみに、私は27k木製より少し多く支払った）
