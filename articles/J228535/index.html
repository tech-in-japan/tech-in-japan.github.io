IBM Cloud Server Month：無料のソフトレイヤーを入手





少し前まで、IBMは、Softlayerホスティングを含むIBMクラウドクラウドプラットフォームの開発に10億米ドル以上を投資してきました。 IBM Cloudは、クラウドテクノロジーとサービスのかなり強力な複合体であり、企業の顧客の幅広いタスクを解決できます。



Softlayerは、負荷の高いサイトやアプリケーションに実サーバーと仮想サーバーを提供するホスティングサービスです。 同時に、Softlayerでは、適切なコンストラクターを使用して独自のサーバーを構築できます。 RAMの量、プロセッサコアの数、内部メモリの量-これらはすべて独立して設定されます。



クラウドバージョンのSoftlayerサービスを無料でお試しいた​​だけます。 2日間ではなく、1か月間お試しいた​​だけます 。



サービスに精通するために、初期サーバー構成が発行されます。



CPU：1 x 2.0 GHzコア

RAM：1 GB

ストレージ：25 GB（ローカル）

パブリック帯域幅：5000 GB帯域幅

プライマリIPアドレス：1 IPアドレス

アップリンクポート速度：100 Mbpsのパブリックおよびプライベートネットワークアップリンク



注文を構成するときに、適切なデータセンター（米国、西ヨーロッパ、中央および東南アジアのいくつかの地域）、およびサーバーで利用可能なオペレーティングシステムの1つを選択できます。



試用版サーバーを注文するときは、有効なクレジットカードを指定する必要があります。サービスをオフにせず、1か月経過しても使用し続けると、クレジットカードから資金が引き落とされます。



上記のように、仮想化されたサーバーだけでなく、実際のサーバーも注文できます。







ここでこれを行うことができ 、IBMはこのサービス（Intel Xeonプロセッサに基づく専用Xpressサーバー用）に対して20ドルの割引を提供しています。 さらに、無料でリソースを増やす機会もあります。



このビデオから、IBM Cloud / Softlayerの機能について詳しく知ることができます。







参照用：

2014年1月、IBMは、グローバルクラウド開発に12億ドル以上を投資する計画を発表しました。 その結果、北米、南米、ヨーロッパ、アジア、オーストラリアを含む5大陸の15か国にある40のデータセンターでのIBMのクラウドサービスは、世界中の顧客が利用できるようになります。



さらに、IBMは将来、データ処理および分析のためにPower Systemsの最適化されたパフォーマンスを使用するオファーのリストを拡張する予定です。



提案の予備リストには以下が含まれます。



•Watsonソリューション-Watson Discovery Advisor、Watson Engagement Advisor、およびWatson Development Cloud-ツール、開発キット、APIを含むサービスとして提供され、認知アプリケーションの開発と展開を支援するプラットフォーム。



•サービスとしてのソフトウェア-IBM DB2 BLUやIBM Cognos分析ソリューションなど、Power Systemsサーバーでの使用に最適化された幅広いデータサービス。



•サービスとしてのインフラストラクチャ-Power Systemsサーバーは、オンデマンドインフラストラクチャプラットフォームとして顧客に提供されます。
