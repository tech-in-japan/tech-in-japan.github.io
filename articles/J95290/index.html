アプリケーション開発者の日-プログラマーの日

2010年9月23〜24日にヤロスラヴリで開催されるプロのプログラマーであるApplication Developer Daysのカンファレンスにご参加ください。







このユニークなイベントは、ソフトウェア開発の分野の専門家によって実施され、個々の専門家とITコミュニティ全体を1つのプラットフォームに統合します。 会議は特定の技術やプラットフォームに限定されません。 さまざまなソフトウェアプラットフォームとツールを使用して仕事をするスペシャリストは、同僚にプロジェクトについて伝えたり、経験を交換したり、新しいアプローチや機会について学ぶことができます。



プログラマは、アプリケーションを明らかにするだけの人々です。 そして、長い間、業界会議で少数派であったのは彼らでした。 開発業界は成長しており、プロジェクトマネージャー、テスター、ビジネスアナリストはプログラマーをイベントの範囲を超えて推進しています。 昨年になってようやく、真の技術会議の復活に向かう傾向が現れました。 Application Developer Days 2010がシーズンを開始し、本当に興味深いものになることを約束します。



この会議は、2009年5月にミンスクで開催され、多くの肯定的なフィードバックを集めた昨年のイベントEngineering Software Forumの定性的な継続です。 会議は、主要なCISソフトウェアエンジニアリングイベントの当然の権限を獲得しました。 過去1年間の主要な講演者のリストには、セルゲイ・アルヒペンコフ、アレクサンドル・オルロフ、スタニスラフ・フォミン、アレクサンドル・アレクサンドロフ、アスカット・ウラズバエフ、アンドレイ・ビビチェフ、アレクサンダー・ベイキン、デニス・ペテリン、ユーリ・シリャエフ、ドミトリー・ロバセフなどが含まれます。 この会議は、ハイテク産業のリーダーと各国のIT協会によってサポートされています。



2010年、Application Developer Daysは、ソフトウェア開発、プログラミング言語の選択、成功するアーキテクチャソリューションとその作成に関する推奨事項、最も人気のある技術、有名ベンダーの製品、オープンソースソリューションに関する考慮事項に関連する幅広い問題について議論します。



プログラマー自身に加えて、会議はソフトウェア製品の作成プロセスに関与し、開発者の生き方を理解したいすべての人々、プログラマーの観点からすべてを見ることになるでしょう。



会議形式とプログラム



会議は、3つの並列セクションで構成されています。

会議の期間は2日間です。



参加者は待っています：

  • ソフトウェア工学の主要な専門家による実践的なワークショップ。
  • ソフトウェア開発のさまざまな分野で成功した問題解決の例に関するストーリー。
  • 研究報告書;
  • ラウンドテーブルとパネルディスカッション。
  • 技術的ソリューションの提示;
  • そして、はるかに。


会議報告のテーマ

  • 特定の応用問題を解決する練習。
  • ソフトウェアアーキテクチャ。
  • コード作成テクニック。
  • 開発環境;
  • プログラミング技術;
  • ソフトウェア開発の動向。
会議の公式プログラムは、プログラム委員会による競争力のあるレポートの選択の後に最終的に決定されます。



会議への参加者の登録



参加費用と割引







Application Developer Daysで講演することをお勧めします



会議では、上記のトピックに関するレポートと、ソフトウェアエンジニアリングに関連する他のトピックに関するレポートを受け入れます。

報告手順はここにあります 。



レポートを提出する



受理されたセクションレポートの著者は、会議への参加のための登録料の支払いを免除されます。 複数の共著者がいる場合、実際の講演者は組織料の支払いを免除されます。
