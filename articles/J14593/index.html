Earth Class Mail：紙のメールから-メール

今日、コミュニケーションは非常に発達しています。私たちは全員、毎日複数のメール、SMSを送信し、異なるIMで通信しています...しかし、それでも、通常のメールを使用する人がいます。そして、郵便配達人によって配達されました！ しかし、新しいサービスのおかげで、標準的な（前世紀の）メールを忘れることができます。



Earth Class Mailは、月額13ドルから、紙のメールを電子メールに変換してプライベートなオンラインメールボックスに送信するサービスを提供しています。



仕組みは次のとおりです。 郵便局で住所の変更を登録すると、すべてのメールが会社に送信されます。 Earth Class Mailは、すべての受信メールの外側をスキャンし、スキャンをメール受信ボックスに送信します。



次に、これらのスキャンに基づいて、レターを（および物理的にも）削除するか、レター全体をスキャンして電子メールで送信するか、通常のメールでレターを送信するように依頼できます。 また、会社に依頼して、元の手紙を「アーカイブ」してさらに回復することもできます。



このサービスは、旅行が好きで明確な居住地がない人、および通信の流れに取り組んでいる企業に役立ちます...まあ、または20世紀に住みたくない人に。 私は個人的に（アパートの請求書や税金ではなく）手で書かれた紙の手紙を受け取りますが、とてもいいです。
