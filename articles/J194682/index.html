会議資料DEF CON 1993-2013





DEF CONは、ネバダ州ラスベガスで毎年開催される世界最大のハッカー会議です。 最初のDEF CONは1993年6月に開催されました。



情報セキュリティ、サイバーパンク、およびそれを必要とする人に興味がある人-この情報を見逃した可能性があります。 DEF CONは、最初の会議から入手可能なすべての情報を投稿します。 クイックリンク：

  • media.defcon.orgディレクトリにグループ化されたすべての資料の概要。
  • www.defcon.org/html/links/dc-torrent.html-トレントファイルを含むさまざまな資料。
  • www.defcon.org/html/links/dc-archives.html-年ごとの資料へのリンクがあるポータル。


リンクを使用すると、音楽、さまざまなビデオ、パフォーマンスの記録、CTF競技会のファイル、さまざまなアートなどを見つけることができます。 お楽しみください;）
