iPhoneリーダーの概要





iPhoneを購入した理由の1つは、電子書籍を快適に読むことができることです。 ネイティブのiBooksリーダーはOS 4.0にのみ登場したので、今のところはサードパーティのプログラムを使用する必要がありますが、この記事で最も興味深いものについて説明します。



各読者は、一種の「テストドライブ」を受け、その結果は4つの基準に従って評価されました。

  • インターフェース： プログラム の 外観、それを操作する便利さ。
  • ライブラリ： ストレージ組織、新しい本のダウンロードの容易さ。
  • 設定：設定の 数、柔軟性。
  • パフォーマンス： プログラム の 速度。




1. ShortBook



投稿者： 審査員ショーキ

配布： AppStore、有料、$ 4.99

最新バージョン： 5.7.3

サポートされている形式： FB2



スクリーンショット：







長所：

  • 見積書
  • 辞書
  • 頻繁な更新


短所：

  • テキスト検索がありません
  • サポートされる形式は1つのみ


評価：

  • インターフェース： 8/10
  • ライブラリ： 8/10
  • 設定： 8/10
  • パフォーマンス： 10/10
  • 合計： 9/10


評決：これまでのiPhone の最高のリーダー。 便利な図書館、優れた整理された本の積み込み、スピード-幸せに必要なものは何ですか？



2.スタンザ



投稿者： Lexcycle

配布： AppStore、無料

最新バージョン： 2.1.1

サポートされている形式： EPUB、PDF、FB2



スクリーンショット：







長所：

  • 素敵なインターフェース
  • Wi-Fi経由で書籍を簡単にダウンロードする
  • 多くのテーマ


短所：

  • 不完全なPDFサポート
  • 不便な反転アニメーション
  • サードパーティのリソースからの書籍の不明瞭なダウンロード


評価：

  • インターフェース： 8/10
  • ライブラリ： 7/10
  • 設定： 8/10
  • パフォーマンス： 7/10
  • 合計： 7/10


評決：無料の読者の中で最高。



3. ruBooks



作者： セルゲイ・マカロフ別名（Colel）

配布： Cydia、無料

最新バージョン： 0.921

サポートされている形式： TXT、RTF、FB2、HTML、DOC、PDF、XLS



スクリーンショット：







長所：

  • さまざまな読み取りモードに合わせて背景とテキストの色を柔軟に調整


短所：

  • 書籍の直接ダウンロードはありません。Safariダウンロードプラグインを使用する必要があります
  • 不十分な技術サポート


評価：

  • インターフェース： 3/10
  • ライブラリ： 4/10
  • 設定： 4/10
  • パフォーマンス： 5/10
  • 合計： 4/10


評決：非常に平凡なソフトウェアで、最も必要なプロパティのみがあります。 しかし、彼女には恋人もいます。



4. readMe



投稿者： Unreal Mojo

配布： AppStore、有料、$ 1.99

最新バージョン： 1.6.1

サポートされている形式： FB2、EPUB、PDF



スクリーンショット：







長所：

  • シンプルなインターフェース
  • コンピューターから本をダウンロードするための組み込みHTTPサーバー


短所：

  • 整理されていない図書館
  • 3Dページングブレーキ
  • 不便なステータスバー
  • 高速バッテリー消耗


評価：

  • インターフェース： 8/10
  • ライブラリ： 6/10
  • 設定： 7/10
  • パフォーマンス： 6/10
  • 合計： 7/10


評決： ruBooksとは異なり、この読者にはミニマリズムが心地よい。 まだ多くの欠点がありますが、頻繁に更新することでこの問題を解決できます。 そして、価格は噛み付きません。



5. i2reader



投稿者： Unsanity

配布： AppStore、有料、$ 4.99

最新バージョン： 3.2.1

サポートされている形式： FB2、TXT、EPUB、DOC、DOCX、RTF、HTML、PDF（テキストのみ）



スクリーンショット：







長所：

  • 美しいスタイル
  • 辞書


短所：

  • 頻繁なバグとクラッシュ
  • 十分に発達していないテキストフォーマット
  • FTPおよびWEBサーバーは安定していません


評価：

  • インターフェース： 9/10
  • ライブラリ： 7/10
  • 設定： 7/10
  • パフォーマンス： 2/10
  • 合計： 5/10


評決： ShortBookの直接の競争相手であり、さまざまな機能が過剰に負荷をかけているため、その半分がまったく機能せず、2番目の機能が-記載されているとおりではありません。 これまでのところ、プログラムは明らかに要求された価格の価値はありません。
