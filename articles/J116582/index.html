Ubuntu 11.04 Betaリリース





Ubuntu 11.04の最初のベータ版がダウンロードおよびテストできるようになりました。 公式の発表はまだありませんが、今すぐ新しいバージョンをダウンロードまたはアップグレードできます 。





Ubuntu 11.04 Alpha 3と比較した変更：




1）新しいタイプのインストール：











2）ALT + F2チームの改善







チームの歴史といわゆる「ライブ検索」があります。



3）メインパネルのアイコンのサイズを変更できます（32から64ピクセル）：







4）インストールせずにドライブアプリケーションをテストします。 この関数は、Habréを含む多くのノイズを発生させたため、どのように機能するかを説明しません。







ところで、アプリケーションをインストールするときに、Ubuntu Software CenterはすぐにUnityパネルにアイコンを追加することを提案します。







5）デフォルトでFirefox AppMenuがインストールされます。







LibreOffice AppMenuもインストールされますが、最初に許可を求められます。



6）ダッシュに新しいスクロールバーが追加されました：







7）デフォルトでは、壁紙はわずかに変更され、アイコンも小さな変更が加えられています。



8）改善されたサウンドメニュー。 「プレーヤーバック」、プレイリストの選択、音量調整のサポートに加えて、Skypeでの会話中にマイクの調整が行われました。



多くの小さな変更も行われましたが、説明するのは非現実的です。
