Google I / Oカンファレンスが本日開始

5月19日20:00、サンフランシスコのモスクワ時間は、開発者向けの次の年次会議を開きます





その上で、インターネットの巨人は、新しい製品とサービスの多くの重要な発表をすることが期待されています。



会議のトピックは、Android、App Engine、Chrome、Google Web Toolkit、OpenSocial、Google AJAX API、そしてもちろんGoogle Waveであり、会議のメイントピックは間違いなくAndroidモバイルデバイスのオペレーティングシステムです。 開発者は最終的にシステムの次のバージョンである2.2（Froyo）を実際に表示することが期待されています。 フローズンヨーグルトのデザート。 ソニーは、テレビのタスク用にIntelが用意した特別なAtomプロセッサによって制御される、コードネームDragonpointと呼ばれるAndroidの特殊バージョンをサポートするテレビのデモを行うと噂されています。



Googleは、このイベントに実質的に参加したい人の面倒を見てくれました。 会議の1日目と2日目の営業時間中にこれを行うことができます。 これら2つのパフォーマンスのビデオブロードキャストは、すべての来場者向けにパブリックドメインのwww.youtube.com/GoogleDevelopersで開催されます。



初日-5月19日、20：00〜21：30（モスクワ時間）

2日目は5月20日19時30分から21時00分です。



会議に関する追加情報は、 http ： //code.google.com/intl/en-RU/events/io/2010/にあります。
