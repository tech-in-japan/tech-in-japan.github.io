バラク・オバマの電話

最近、アメリカの会社BlackBerryからの奇跡のデバイスがテストのために私の手に落ちました。彼らは私にロシアで最も洗練された公式に入手可能なBlackBerry®9000 Boldを割り当てました。 判明したように、現在の米国大統領であるバラク・オバマも同じです:)。 おそらく、外観から開始します（写真の下）









デバイスが企業セグメント、厳格な形態、ケースの金属縁取り用に設計されており、背面カバーが革でトリミングされていることがすぐにわかります。これはタッチに非常に快適です。さらに、厳格な黒い色と光沢のあるディスプレイはすべて、これが固体で高品質の電話の印象を残します。 フルセットで言及する価値があり、これはまさに私が手に入れたものです。良いヘッドフォン（3.5 mmジャック）、さまざまな種類のソケット用の交換可能なプラグ、ソフトウェアディスク、PCに接続するためのケーブルで楽しく驚いた充電器、ブラックベリーのロゴが入ったレザーケースもあります。 デバイスは小さくないため（114 x 66 x 15、136 gr。）、後者の存在は喜ばれます。ポケットに入れて持ち運ぶことはできません。 デバイスを操作するときの感覚を説明します。



すぐに過去3年間、私はタッチスクリーンを搭載したWindows Mobileでのみコミュニケーターを使用することを予約しました。そのため、最初はスタイラスで画面を単純に突くことができないことに多少の不満を感じました。 しかし、適応は驚くほど迅速で痛みがありませんでした。トラックボールに非常に満足していました。定性的に作成されるデバイスは指で数えることができ、BlackBerryも成功しました。 メニューとWebブラウザーの両方のナビゲーションの主要な要素はトラックボールです。 また、小さく見えるキーボードが便利であることが判明し、ある程度の適応の後、SMSの高速入力や電子メールの操作に便利になりました。 後者は偶然にもテストの目的でした。



それでは、BlackBerryデバイスからExchangeでメールを操作しているときに何が得られましたか？

このテストドライブの結果から判明したように、多くのことを行いました。 まず、トラフィックを節約します.Exchangeサーバーへの接続は直接ではなく、特殊なBESサーバーと独自のプロトコルのおかげです。ただし、電話はサーバーと常時接続されていますが、圧縮技術により最小限のトラフィックが無駄になり、メッセージがほぼ瞬時に到着します。 第二に、添付ファイルを表示する便利さです。サーバーは、Word文書、Excelスプレッドシート、写真のいずれであっても、現在必要な添付ファイルのみを画面に魔法のように表示します。



たとえば、重い1MBのテーブルが1枚だけ必要でしたが、スマートサーバーはそれを完全に提供しませんでした。GPRSトラフィックの価格の最近の残忍な上昇を考慮して、私が見ることにした添付ファイルを私に送信しました。 第三に、リモートワーク、アドレス帳の同期、会社の企業ディレクトリでの連絡先検索の使用、カレンダーの同期のすべての機能により、オフィス外での生活、出張が大幅に簡素化され、通勤中であってもイベントに遅れずについていくことができます。



メールの操作に関係しないコミュニケーターの珍味から、Opera Miniを完全に置き換えることができる非常によく開発されたWebブラウザーに言及します。トラックボールと、それに応じて、ナビゲーションの利便性のためのカーソルがあれば、それを超えます。 また、「ブラックベリー」では、ICQ、Gtalk、および西部Facebookでの人気のサポートを忘れませんでした。 内蔵プレーヤーはまともなサウンドを生成し、標準のヘッドフォンジャックを使用すると、特に深刻なモデルをスイングするのに十分なパワーがあるため、まともなサウンドを接続できます。



電話番号の音声入力認識機能には満足していましたが、残念ながら、デバイスの他のすべての機能とは異なり、ロシア語化されていませんでしたが、騒々しい通りを歩いているときに音声で数字をダイヤルするには十分です。 組み込みの優れたマッピングサービスとGPSにより、騒がしいモスクワで迷子にならないようになりました。



一般的に、コミュニケーター自身とExchangeメールサービスクライアントとしての仕事の両方からの印象は非常に楽しかったです。 通信に関しては、デバイスは非常に満足しており、オフィスワーカーが現在必要とする可能性のあるすべてのもの、Exchangeメールを操作するためのすべての機能、インスタントメッセンジャーのサポート、GPSナビゲーション、天気予報、在庫情報提供者、ドキュメントの表示と編集DOC \ XLS \ PPT、便利なカレンダー、最新のAJAX \ JavaScript \ CSSテクノロジーをサポートするWebブラウザーで動作します。 オーディオとビデオとしてのプレーヤー、さらにはいくつかのゲームのセットもあります。 ところで、電話から直接十分な組み込みアプリケーションを持っていない人のために、AppleのAppStoreに似た着信音、デスクトップ写真、アプリケーションをダウンロードするサービスが利用可能です。
