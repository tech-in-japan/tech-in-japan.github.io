SMB向け10Gスイッチング：Extreme Summit Switch X620シリーズ

多くの場合、高密度のポートを必要としない組織のネットワークの一部では、超生産的な機器の設置と運用のコストは経済的に実行不可能です。 このような場合、低遅延およびスタッキング機能を備えた中小規模の市場セグメント向けのExtreme Networks Summit X620シリーズスイッチが最適な選択肢です。



このスイッチラインは、中規模および大規模のオペレーターをトラフィック集約ノードにインストールするのに最適です。また、サーバーを接続したり、イントラネットリソースを集約したりするデータセンターにも最適です。 ACまたはDC電源を使用すると、既存の電源スキームを変更せずに、Summit X620を既存のインフラストラクチャに簡単に統合できます。 このソリューションには、100 Mbpsのかなり広い範囲のポート速度と、1および10ギガビット/秒（銅線と光の両方のオプション）があります。 これにより、高速で動作する能力のないデバイスをスイッチに接続できます。 また、選択したFANモジュールのタイプに応じて、フロントツーバックおよびバックツーフロントモードで動作可能なSummit X620冷却システムに注目する価値があります。



サポートされている技術



Extreme Summit X620は、他のExtreme Networksスイッチモデルと同様に、ExtremeXOSモジュラーオペレーティングシステムを実行します。 このOSのアーキテクチャにより、このOSは高い可用性と拡張性を備えており、マルチメディアコンテンツを送信するときや重要なビジネスアプリケーションを操作するときのネットワーク機器の効率が向上します。



Summit X620ラインのスイッチは、さまざまな組み合わせの8〜16ポートのモデルで表され、16,000個のMACアドレスをサポートし、600ナノ秒未満の低いスイッチング遅延が特徴です。



Summit X620ファミリは、ユーザーまたはデバイスに基づいて簡素化された管理とポリシーを実装します。 スイッチは、IPv4 / IPv6のハードウェアベースのACLをサポートします。 誰がこれらのリソースまたはそれらのリソースにアクセスできるか、どのような種類の操作が許可されているか、または逆に、それらを作成することは禁止されています。



スイッチは、Open Shortest Path First（OSPF）ダイナミックルーティングプロトコル、プロバイダーブリッジ（PB、IEEE 802.1ad）、AVB、OpenFlow、およびネットワーク上の時間を同期するためのIEEE 1588 Precision Time Protocolなどの多くのテクノロジーをサポートしていることに注意してください。



モデルの特徴



スイッチの新しいSummit X620シリーズは、Summit X620-16x、Summit X620-10x、Summit X620-16T、およびSummit X620-8T-2xの4つのモデルで提供されます。 デバイス間の主な違いはポートの数とタイプです-これはモデル名のインデックスによって示されます。



Summit X620-16xには、16個の10GBASE-X SFP +ポート、さまざまな電源（AC / DC）用の2つのスロット、およびファンユニット用の1つのスロットがあります。 スイッチングマトリックスの合計帯域幅は320 Gb / sです。 機能は後方ポート互換性です-100Mb / 1Gb / 10GBASE-X SFP +。



ライン内の他の2つのスイッチ、Summit X620-10xおよびSummit X620-8T-2xには、200 Gb / sのスループットのマトリックスが装備されています。 1つ目は10ポート10GBASE-X SFP +、2つ目はボード上にあります-8ポート10GBASE-T + 2つのコンボポート10GBASE-T / SFP +両方のオプションには電源とファンが統合されています。
