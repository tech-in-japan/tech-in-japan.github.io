bashrcの関数。 もう一つの簡単なこと

.bashrcでは、関数を記述できます。



なんで？ エイリアスがない場合、 1行のスクリプトを記述する必要はありません。



たとえば、.bashrcに関数を追加して、外部IPアドレスを取得できます。



wanip() {

wget -q -O - checkip.dyndns.com | awk '{print $6}'| sed 's/<.*>//'

}







コマンドラインで、wanipは外部IPアドレスを表示します。



そして、これを.bashrc addで別のファイルに入れる方が良いです



if [ -f ~/.bash_func ]; then

. ~/.bash_func

fi







.bash_func-関数を書き込むファイルの名前。
