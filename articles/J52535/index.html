開発者危機のアイドル

まず第一に、この記事の著者は、あなたの企業のインターネットチャネルでの空の寄生を、労働時間を犠牲にすることを主張していないことに注意したい。



だから、bashやhabrをすべて同じように読んで解雇されない方法。

あなたは本当に何もすることがなく、イライラするシェフが歩き回り、ダモクレスの剣を頭の給料に降ろす準備ができていると想像してみてください。



ide == Microsoft Visual Studio 2008（2005）の場合



1. ideを実行します

2.ツール-オプション-環境-起動

3. [起動時]リストで、[開始ページの表示]を選択します。

4. [スタートページのニュースチャネル]フィールドにbash.org.ru/rssと入力します

5. [コンテンツのダウンロード]で、たとえば、60分

6.スタジオを再起動して読みます。







ide == Aptana Studioの場合



1. Aptana-メッセージセンター-フィードの構成

2. My Aptanaオプション-起動後に常に表示

3. RSSフィードはプラス記号を挿入してbash.org.ru/rssを記述します

4.右下隅に、おなじみのRSSアイコンが表示されます。クリックしてお楽しみください。







もちろん、さらに進んで、Googleリーダーで選択を収集し、通常のRSSリーダーとしてide'shkaを読むこともできます...一般的に、独学の怠idleさの完璧さに制限はありません。
