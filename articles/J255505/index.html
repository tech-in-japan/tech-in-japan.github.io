ウィークリービルドVivaldi 1.0.151.7

みなさんこんにちは！



イースター休暇と「突然の」不快な退行のため、新しいビルドのリリーススケジュールは少しずれていますが、月曜日に新しいテストバージョンをリリースするという伝統を取り戻すつもりです。 しかし、今週は空のままにしてはいけません-私たちはそれが決してないよりも遅くなったと判断しました。 したがって、今日はVivaldi 1.0.151.7のアセンブリを紹介します。 何が新しくなったのですか？ 今からお話しします。



先週の作業分野の1つは、複数のページを1つのタブに配置する機能で、Operaのマルチドキュメントインターフェイスの懐かしい思い出をもたらします。 しかし、もちろん、新しい現実を考慮してこれを行います。 そのため、ステータスパネルに特別なボタンを追加しました。これにより、複数のページを垂直、水平、またはタイルで配置できます。 ただし、このボタンは以前のアセンブリに表示されていましたが、タブのグループを操作する場合にのみ使用できました。 これで、同じボタンを使用して、選択したタブのグループを1つのウィンドウに配置できます。



もう1つの小さな改善点は、未読の背景タブを示すインジケーターです。 以前のイースタービルドでも導入されましたが、この機能の改善に取り組みました。 おそらくこれは外部ではあまり目立たないかもしれませんが、内部ではかなり徹底的にコードに取り組みました。



また、バグの修正と一連のキーボードショートカットの作業の改善にも取り組みました。 特に、エラー報告ページでもこの機能を使用できるようになりました。 もう1つの注目すべき修正-テキストフィールドでキーボードショートカットCtrl +左矢印を使用しても、閲覧履歴を戻ることはできません。 まあ、少なくともそう願っています:-)



もちろん、私たちは上記の機能だけでなく、他の多くの問題にも対処しました。 修正のリストは以下で詳細に読むことができますが、今のところは新しいアセンブリをダウンロードすることをお勧めします。



  • Windows 32ビット
  • Windows 64ビット
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
