HTMLフォーム要素名について

HTMLのフォーム要素の名前は配列として記述できることをご存知ですか。 たとえば、次のフォームを使用して、ユーザー情報を入力します。 

 <フォームアクション= '/'メソッド= '投稿'>
    名前：<input type = 'text' name = 'user [name]'> <br>
    性別：<select name = 'user [sex]'>
                 <オプション値= '女性'>女性
                 <オプション値= 'man'>男性
             </ select> <br>
    メール：<input type = 'text' name = 'user [email]'> <br>
         <入力タイプ= 'submit'値= 'Next'>
 </ form>


PHPでは、このデータは次のような配列として取得できます。 

     $ user = $ _POST ['user'];


または： 

     $ user = $ _REQUEST ['user'];
