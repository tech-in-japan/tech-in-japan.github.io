グローバル教育：留学のためのロシア政府奨学金プログラム





こんにちは、Habr！



秋には、外国の大学が2016/2017年度の申請者からの申請を受け付け始めます。 アイビーリーグ大学または地球上で最高の大学のトップ200にある大学の卒業証書を取得したい人のために、ロシア政府はグローバル教育奨学金プログラムを開始しました。



ちなみに、これはStudyQAプロジェクトの最初の投稿です。 私たちは、外国の大学の教育プログラム用の無料のスマート検索エンジンを作成します。これは、プログラムを見つけ、大学選択委員会に連絡し、サイトから直接興味のあるプログラムのドキュメントを提出します。 さまざまな興味深いインターンシップ、コース、留学、無料の機会について書いていきます。



この記事では、助成金の受け取りの複雑さと、技術専門分野の行政と大学院での研究を継続したい人のためのこのプログラムへの参加条件について説明します。



グローバル教育とは：



2013年12月28日に、グローバル教育プロジェクトが開始されました。

プログラムの本質は簡単です。 認可された修士課程、大学院課程、または居住プログラムのために227の外国の教育機関のいずれかに入学した候補者に助成金が与えられます。 卒業後、参加者はロシアに戻り、 参加企業の 1つで3年間働く必要があります。



Global Educationは、5つの優先分野をサポートする助成金を提供しています。

  • 科学
  • 教育
  • 薬
  • エンジニアリング
  • 社会的管理。


当初は、研究と宿泊施設のために年間1381800ルーブルのロシア人1,500人に資金を提供することが計画されていました。 しかし、経済状況に関連して、参加者数を半減し、資金を年間270万人に増やすことが決定されました 。



参加条件：

  • ロシア連邦の市民権（現在どこに住んでいるかは関係ありません）;
  • 高等教育を修了した（他国で学士号を取得した場合、卒業証書は鼻孔形成手順を経る必要があります）。
  • 外国の大学からの招待状の存在;
  • 犯罪歴の欠如と、帰国後3年間働くための正式な障壁。
2015年には、4つの競争力のある選択を実施する予定です（1つが合格し、次の締め切りは6月上旬、その後8月〜9月）



文書のパッケージ：

  • アプリケーション（ウェブサイトeducationglobal.ru/kandidatuプログラムに記入）;
  • 申請書（プログラムWebサイトにもあります ）;
  • 費用の見積もり（ 同上 ）。 見積もりは、期間全体に対して一度に計算されます。 すでに外国の大学で勉強している場合、推定は残りの学習期間についてのみ計算されます。 その他の費用の額は固定されており、調査対象国によって異なります（1か月あたり約6万ルーブル）。 詳細はこちら ）;
  • パスポート（ロシア語および外国語）;
  • 犯罪歴のない証明書。
  • 外国の大学での入学/招待/研究を確認する文書。
  • プログラムの費用が記載された公式文書。
  • 名前変更文書、TIN割り当て証明書、保険証明書、ワークブック、雇用証明書（ある場合）;
  • 外国の大学との有料サービスの提供に関する契約。
  • 競争力のある選択の機会を増やす方法：
  • 電子キューへの早期登録（これは奇妙ですが、それでも登録が早ければ早いほど、勝つ可能性が高くなります）。 登録時に、ドキュメントのパッケージ全体を一度に添付する必要はありません。
  • Web of ScienceおよびScopusのリストに含まれる主要な科学雑誌の出版物（科学トラックに提出する場合）。
  • 専門の経験;
  • 外国の大学で勉強しています。
出願時にプログラムリストから外国の大学に留学している場合、競争力のある選択で有利になります。



雇用条件：

  • 卒業後1か月でロシア連邦に戻り、プログラムの組織の 1つで仕事を見つける必要があります。
  • プログラム参加者の約10％がモスクワとサンクトペテルブルクで仕事を得ることができます。 参加者のほとんどはロシア連邦の他の地域にいます。
  • 仕事の場所を変更することは、常に2回以下です。
  • プログラム条件の遵守を拒否した場合、割り当てられた補助金の半分の金額で罰金を支払う必要があります。
自分の会社またはプログラムによって承認されたリストに含まれていない別の組織で働きたい場合、会社はリストへの登録を申請できます。 この場合、組織は少なくとも3年間登録する必要があります。



覚えておくことが重要です：

  • このプログラムには、志願者だけでなく、外国の大学で学ぶ学生にも申請できます。 勝利の場合、参加者には残りのトレーニングの助成金が与えられます。
  • 候補者が一度に複数の大学に入学した場合、優先大学または専門分野を選択することにより、すべてのプログラムに申し込むことができます。
  • グローバル教育プログラムによって提供される助成金がトレーニングの費用を賄うのに十分でない場合、候補者は他の基金からの資金提供を要求できます。
  • トレーニングはフルタイムでなければなりません。
  • クォータ 。


トレーニングの各方向には一定の割り当てがありますが、場所の競争は非常に小さいため（たとえば、現時点で100件未満の申請が提出されている）、割り当ては大きな役割を果たしません。 現在、助成金は7人に授与されています。 受賞者はこちらでご覧いただけます 。

すでに国際試験証明書をお持ちの場合、2015/2016年度の奨学金を登録して受け取ることができます。
