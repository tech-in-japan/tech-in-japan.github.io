Node-Webkit Innovationsバージョン0.3.0

Zhao Chengは昨日 、node-webkitエンジンの新しいバージョンのリリースを発表しました。これは、Intel Open Source Technology Centerの腸で作成されたWebKitとNode.jsの強力な組み合わせで、HTML、CSS、およびJavaScriptでクロスプラットフォームのグラフィカルアプリケーションを作成できます。



新しいバージョンの番号は0.3.0で 、次のようになります。







（スクリーンショットは、 ノードJSキットエンジンに移植したAppJSのデモを示しています。WindowsXPで起動しました。）



ご覧のとおり、 node-webkitの新しいバージョンは、ウィンドウ設計においてGoogle Chromeに非常に似ています。



そして、このバージョンの他の革新のリストは次のとおりです。



  • サポートされている3つのすべてのオペレーティングシステム（Linux、Mac、Windows）で「ネイティブ」GUIにアクセスするために使用できる関数のライブラリが登場しました：ウィンドウ、コンテキストメニュー、システムトレイ、クリップボード、ファイルおよびフォルダー選択ダイアログなどを管理するため。

  • フレームのないウィンドウのサポートが登場しました 。

  • フルスクリーンウィンドウと非表示ウィンドウのサポートが（ アプリケーションの説明 と APIの両方で）登場しました 。

  • バグが修正されており、特に、ユーザーがマウスを動かす瞬間までLinuxおよびWindowsでフリーズすることがあります。

  • ICOおよびPNG形式（透明度を含む）およびJPEGのアイコンのサポートが追加されました。 任意のサイズのファイルを使用できます（1524×938ピクセルのサイズのJPEGをチェックしました）。これは必要なサイズに自動的に縮小されます。

  • MacおよびWindows用の新しいツールバー。 コマンドラインで--developerスイッチを使用すると、このパネルに新しいボタンが表示され、開発者コンソールが呼び出されます。

  • Node.jsエンジンはバージョン0.8.11に更新されました（ただし、これはスクリーンショットで既に示しています）。
