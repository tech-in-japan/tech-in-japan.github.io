Firefox 12リリース

いつものように、伝統によれば、Firefoxの新しい安定ビルドは、4月24日（火曜日）に予定されている公式発表の数日前に利用可能になります。

このバージョンの新機能：





  • ブラウザーの更新プロセスが改善されました-約束どおり、FirefoxはWindowsユーザー向けのサイレント更新モードを受け取りました。
  • 更新されたメディアコントロール：Firefox 12にはHTML5メディアコントロールが改善されました。
  • 開発者ツールの改善：このリリースでは、開発者向けに85以上の改善が行われています。 更新されたWebコンソール、シンプルなJavaScriptエディター、スタイルエディター、HTMLビューアー、およびページインスペクターの3Dビュー。
  • Webページのソースコードの表示で、 行番号を追加しました。
  • 複数行のツールチップ（タイトル属性）のサポートが追加されました。
  • 検索結果のセンタリングのサポートがページ上の検索機能に追加されました（Ctrl + F）。
  • ダウンロードマネージャーにリンクを挿入すると、ファイルのダウンロードが自動的に開始されます。
  • CSS列の塗りつぶしのサポートが追加されました。
  • CSS text-align-lastのサポートを追加
  • Mac OS XでのWebGLの速度に関するバグを修正しました。
  • ECMAScript 6のMapおよびSetオブジェクトの実験的サポートを実装しました。


重要： Firefoxバージョン3.6.Xは4月24日にサポートされなくなりました
