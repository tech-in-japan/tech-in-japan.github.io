NetApp AltaVaultストレージシステム

従来のクラウドバックアップおよびリカバリソリューションは、多くの理由で企業に適さないことがよくあります。 NetAppは現在、顧客にクラウド統合ストレージソリューションのAltaVaultファミリーを提供しています。













まず、現代の企業が継承されたオンプレミスのデータバックアップおよび復旧システム（企業自体のローカルサイトにある）にもはや満足していない理由について。



1.遅すぎることが判明 。 ユーザーは常にインスタントリカバリと最小限のデータ損失を同時に期待していますが、従来のバックアップとリカバリの戦略は時代に追いついていません。



2.高価すぎる 。 重要な企業データは急速に成長しています。 企業は、オンプレミスシステムで増加するデータ保護のコストに対抗するために、あらゆる手段を使用しています。 さらに、データ量が増加するにつれて、必要な帯域幅を確保し、他の制限に対処するためのコストはますます重要になります。



3.彼らはあまりにも多くのリスクにさらされています 。 多くの組織は依然としてテープストレージシステムに依存しています。 このアプローチは、記録およびリカバリ中にデータの一部を失う潜在的なリスク、テープシステムの比較的長いダウンタイム、およびテスト能力の制限のために、今日不完全であると思われます。



4.複雑すぎます。 保護を必要とする重要なエンタープライズアプリケーションの数も増加しています。 複数のアプリケーションのバックアップアーキテクチャの複雑化。 継承された技術もエラーを蓄積する傾向があります。 そのため、従来のローカルバックアップシステムは不必要に複雑になっています。



AltaVaultがバックアップ、バックアップ、復元の問題を解決する方法



NetApp AltaVault（旧SteelStore）は、オンプレミスソリューションと比較して最大90％のコストを削減しながら、あらゆるタイプのクラウドで信頼性の高い安全なバックアップを提供します。





物理デバイスAVA 400/800の設計



AltaVaultは、企業のバックアップインフラストラクチャへの既存の投資を保護し、サービスレベル契約（SLA）に準拠しながら、クラウドエコノミーを最大限に活用する機会をお客様に提供します。



AltaVaultソリューションの効果



業界をリードするデータ圧縮のため 。 AltaVaultは、重複排除と圧縮を「オンザフライ」で使用し、最大30のデータ圧縮を提供します.1。 これは、企業がクラウド内の実ボリュームに格納するデータが少なくなり、より高速に移動できることを意味します。



ネットワークとクラウドを最適化する 。 組み込みのWAN最適化により、転送されるデータ量が削減され、クラウドとのデータ交換時間が最大4倍になります。 したがって、AltaVaultはインテリジェントにトラフィックをさらにインテリジェントに圧縮し、お金と時間を節約します。 同時に、指定されたサービスの品質（サービスの品質、QoS）により、ビジネスに必要な速度でデータがクラウドに出入りすることが保証されます。



高速データ復旧のため 。 AltaVaultは、データの95％がローカルキャッシュに残るため、回復性が向上します。 インテリジェントなプリフェッチにより、AltaVaultは数分でクラウドからデータを回復します。 したがって、AltaVaultを使用すると、顧客はテープドライブを使用した場合よりも最大32倍高速でデータを回復できます。



NetApp AltaVaultを使用すると、企業には次の利点があります。



1.オープンプラットフォームでの企業のスケーリング



柔軟な展開とスケーリング 。 クライアントは、クラウド（Azure、AWSなど）仮想（VMware、Hyper-Vなど）または物理（AVA）の最適な展開オプションを選択します。



AltaVaultファミリーのデバイスは、わずか2 TB（最年少の仮想モデルAVA-v2）で始まり、クラウド内の保護されたデータの57 PB（最高の物理モデルAVA-800）に拡張されます。 これにより、AltaVaultソリューションは市場で最もスケーラブルなクラウド統合ストレージになります。

2.重複排除、圧縮、暗号化



NetApp AltaVaultは、プロバイダーが提供する使用済みエンタープライズアプリケーションとクラウドサービスのシームレスな統合を提供します。



既存のバックアップソフトウェアと互換性があります 。 AltaVaultベースのソリューションは、EMC、Symantec、IBM、Commvaultを含むすべての主要なバックアップおよびアーカイブソフトウェア製品と互換性があります。 ITスタッフの再教育のように、企業はギャップを埋めてソフトウェアを交換する時間を無駄にしません。



クラウドの選択と俊敏性 。 クライアントは、AltaVaultが通常サポートしているため、任意のクラウドを好む場合があります。 AltaVaultは現在、市場に存在する主要なクラウドプロバイダーとプラットフォームの95％をサポートしています。 クラウドプロバイダーは表示および非表示になり、企業はいずれにも接続することを望みません。 汎用性AltaVaultは、単一のプロバイダーへの依存を排除​​します。



3.「装甲」セキュリティおよび規制順守



暗号化とコンプライアンス 。 データはすべての場合に暗号化され、安全です-各デバイス上およびトラフィック中の両方。 通常、最新のFIPS 140-2レベル1仕様および業界標準のTLS暗号化と互換性のある256ビットAES暗号化が使用されます。 暗号化キーは、 KMIPプロトコルを使用してローカルで管理できます。実際、情報センターから離れることはありません。



データアクセスコントロール 。 企業の従業員は、役割ベースのアクセス制御と、ターミナルアクセスコントローラアクセス制御システム（TACACS）およびリモート認証ダイヤルインユーザーサービス（RADIUS）との統合を使用して、AltaVaultのデータにアクセスできます。 このために、適切なアクセス制御リストが使用され、特定のプロトコル、ポート、およびネットワークへのユーザーおよびメンテナンス担当者のアクセスが許可されます。



復旧時間とデータ損失を削減します 。 目標復旧時間（ RTO ）の短縮は、ローカルAltaVaultキャッシュを使用することで達成されます。



緊急時の準備 。 ローカルワークロードを回復するための時間と手順は、システム回復テスト中に、またはAltaVaultを使用する場合のAWS / Azure機能に基づいて決定されます。



4.非常に簡単な展開と管理。



より高速な展開 。 AltaVaultベースのシステムは、通常30分以内に起動できます。 プロセス全体が3つの簡単なステップに削減され、その後、作業モードでクラウドへのデータ送信を開始できます。



よりスマートな管理 。 AltaVaultは、エラーが発生しやすく時間のかかるテープドライブや手動制御を必要とするタスクの処理に費やす時間を削減することに重点を置いています。 リモート管理と監視は、適切なGUIを備えたダッシュボードにまとめられ、管理者が物理的にどこにいても、会社のどこにいても、インターネットチャネルがある世界のどこにいても完全に制御できます。



ワールドクラスのサービスとサポート 。 NetAppとそのパートナーのプロフェッショナルサービスは、会社の決定に基づいて構築されたインフラストラクチャと異種環境の両方に適用されます。 これにより、企業の特定のニーズを満たす最適化されたソリューションが提供されます。 サポートサービスは、顧客が継続的な運用、運用の卓越性、および費用対効果を達成するのに役立ちます。



5. AltaVaultはクラウドストレージの参入障壁を下げます



多くの組織は熱心にクラウド経済を受け入れていますが、すぐに一連のハードルに直面します。 AltaVaultは、クラウドに統合されたストレージアプライアンスとして、もともと顧客がこれらの障壁を克服できるように設計されました。



多くのプロバイダーが存在する環境でのセキュリティとコンプライアンスのサポート 。 組織は、自分のデータや顧客のデータを危険にさらすことはできません。 NetApp AltaVaultは、データを保護し、セキュリティ要件を満たしながら、クラウドベースの戦略を実現するように設計されています。



帯域幅が制限されている場合でも、大量のデータを簡単に管理できます 。 一例として、AltaVault重複排除ツールは、クラウドにアップロードされる建設会社データの量を39 TBから7 TBにほぼ6倍削減し、すべてのデータは1週間でAzureに転送されました。



データへのクイックアクセス 。 AltaVaultの大きな利点は、このデバイスを使用してデータをすばやく回復できることです。 テープでは、このような速度と充満度を達成することは不可能です。 AltaVaultを使用すると、1時間または2時間で、どこからでも完全に復旧できます。
