Windows 7で神モードをアクティブにする方法

あなたもあなたが神になることができることを知っていますか！？ まあ、少なくともWindows 7では（ユーザーによると、これはWindows Vistaでも動作します）。



このモードでは、便利なグラフィカルメニューからすべてのWindows設定を管理できます。 有効化する方法は次のとおりです。



1.デスクトップ上の空のスペースを右クリックします。

2. [作成]メニューに移動して、新しいフォルダーを作成します。

3.フォルダーの名前を次のように変更します： GodMode。{ED7BA470-8E54-465E-825C-99712043E01C}

4. Windows 7 x64（およびWindows Vista x64）でこの機能を使用しないでください-システムクラッシュにつながります。ユーザーによると、これはx64システムでも機能します。 ただし、これは自己責任で使用してください。
