どうやってマックに着きましたか。 パート2. Me、Mac、インターネット

これは記事の続きであり、多くの人がコメントから判断して聞きたいと思っていました。 継続は非常に面白く、場所によってはコミカルでさえあり、インターネットの設定に直接関係しています。 これがまさにあなたが聞きたいと思うものであるかどうか、私にはまったくわかりません。 この記事は、初心者のMacユーザーにのみ興味深いものになるでしょう。 それでも、インターネット、特にwi-fiワイヤレスポイントの設定、および「Macで考えているほど悪くない」ために生じた困難について書いています。





それで、私は家に帰り、インターネットを私たちのすべてだから、最初にネットワークを設定することにしました。 ここですべてが非常に簡単です。何をする必要があるかを正確に知っている場合のみです。 これが私が伝えようとしていることです。



MACアドレスの重要性について



MACアドレスが何であるかを知ることは、次のような多くのインターネットの問題や愚かなことを避けるのに役立ちます。

インターネット自体のセットアップに何度も失敗した後、私はまだプロバイダーに電話して状況を説明しています。



-こんにちは！ だから、私はラップトップを買った、私はそれからオンラインになろうとしている、すべての設定が正しいですが、何も動作しません。



「もちろん、新しいMACアドレスがあるため、何も機能しません。」



私にとって、MACアドレスという言葉自体はその時点では何も言わなかったのですが、私はまだ思っていました。 （本当に-ブロンド:)））




事実の後ですでに学んだように、1つの記事を読んだ後、「ほとんどの家庭および都市のネットワークはDHCP経由のアクセスを使用し、内部IPアドレスはネットワークカードのMACアドレスに関連付けられます。 ほとんどの場合、ユーザーは気にしません：マシン（およびネットワークカード）を変更するときは、再バインドのためにプロバイダーを呼び出すだけで十分です。 今はとても賢いのは私であり、それから多くの困難が生じました。



最初に自分でインターネットを設定しようとしましたが、約1時間半かけて、これをどのように行うことができるかを考え、設定を検討しました。 私は本当に誰も気にしたくはありませんでしたが、そうしなければなりませんでした。 何をどのようにすればよいかを知ることができたのは、私がコンピューターを持っていたキエフの店の2人と、ネットワークの管理者からなる会議の結果だけでした。 だから、レシピはそれを正しく行う方法です。



WIFIポイントの設定



そもそも、私のMacBookでWi-FiポイントのAirMac Expressを購入したと言っておく価値があります。 彼女がなぜだったのか、他の点ではなく、十分な理由のある議論をすることはできません。 私は、いずれにしても、Mac Wi-FiポイントはMacBookと友だちになるべきだという原則に従いました。 それ以前は、wi-fiポイントとの取引がなかったので、プレイの過程ですべてを把握する必要がありました:)



Macのwi-fiポイント経由で接続する原理は、おおよそ次のように実装されています。インターネットがそのポイントに行き、それがコンピューターに送信されます。 したがって、それに応じて設定する必要があります。最初に、Wi-Fiポイントを設定する必要があります。 これを行うには、[Finder]> [プログラム]> [アプリケーション]> [ユーティリティ]に移動し、[Air Port Utility] / [Air Port Utility]を選択します。 まず、Air Portベースステーション自体をネットワークに接続し、イーサネットケーブルをそこに接続する必要があります。これは私の場合と同様です。 Wi-Fiポイントが接続されている場合、スクリーンショットに示すように、左側のリストに表示されます。







次に、ネットワークを作成します。 [続行]> [新しいワイヤレスネットワークの作成]>ワイヤレスネットワークの名前とベースステーションの名前（任意のいずれか）を入力します。>セキュリティレベルを選択し、ワイヤレスパスワードを入力します。 ここで、WPA 2をより信頼性の高いものとして使用することをお勧めします（アドバイスされたように、専門家のアドバイスをお伝えします）。 >インターネットへの接続方法を選択します。 私にとって、これは最初に提案されたオプションです>プロバイダーから受信したTCP / IPデータ、ベースステーションを保護するためのパスワードを入力し、[作成]をクリックします。







後でAir Port設定を表示または変更する必要がある場合、これはこの方法で行われます。 [Finder]> [プログラム]> [アプリケーション]> [ユーティリティ]に戻り、[Air Port Utility] / [Air Port Utility]を選択します。 ベースステーションの画像をダブルクリックすると、その設定に入ります。 [インターネット]タブに移動して、入力された設定（接続方法、IPアドレス、サブネットマスク、ルーターとDNSサーバーのアドレス）を確認または変更できます。 契約で指定された私の標準設定では、ルーターはなく、DNSサーバーに対応していることが判明しました。 ご覧のとおり、iTunesから音楽を転送するAir Portにサウンドシステムを接続したり、ネットワークにするためのプリンターを接続したりできます。 これはまだ必要ありません。Wi-Fiポイントを設定して先に進みます。







コンピューターでインターネットを設定する



ネットワークが作成されました。手順2に進みます。次に、コンピューターをwi-fiポイントに接続する必要があります。 ここで判明したように、賢明に哲学する必要はありません。 システム設定に移動します。 これを行うには2つの方法があります。ドックの対応するアイコンをクリックするか、トップパネルのリンゴをクリックして[システム環境設定]を選択します。 [ネットワーク]メニュー項目を選択します。 このウィンドウが開きます：







[詳細設定]タブを選択し、歓声を上げます！ 乾杯！ -ネットワーク設定に入ります。 [TCP / IP]タブに移動し、[DHCPを使用]を選択して、[DHCPアドレスを要求]をクリックします。 この場合、コンピューターはWi-Fiポイントに接続し、既にIPアドレスを取得しています。 おめでとう、今ではすべてが機能するはずです！







この方法でインターネットにアクセスするため、ネットワーク管理者にコールバックして、ベースステーションのMACアドレスをメインのアドレスとして使用するように依頼するだけです。 ボックスまたはデバイス自体のいずれかにあります。 （私が言われたように）イーサネットIDではなく、エアポートIDを指定する必要があります。



これは、Macでwi-fi接続を確立するための非常に難しい方法です。 ここでの主な困難は、設定がどこにあるかを知ることです。これは、Windowsから移行した人が別の場所で設定を見つけるのが少し難しいからです。



MACアドレスの重要性についてもう一度





Wi-Fiポイントを設定するための訪問中に、マネージャーと私は再度設定を行い、すべてがそこにあることを確認しました。 それから彼は私にプロバイダに電話するようにアドバイスしました-それがまったく機能しない場合は、彼らが来て手動で設定できるようにします。 はい、電話しています。 ラップトップに加えて、wi-fiポイントも購入したので、設定する必要があることを伝えます。 私はMacを持っていると言います。 管理者は「しばらくお待ちください」と言って電話を取り、同僚に状況を伝え始めます。 その後、一時停止し、1分間だけ興奮して笑い声が聞こえます。 私はそれが何であるか理解できませんでした。 厳しいLinuxoidsが決定した-それだけで、それは私たちにやってきた、または状況は彼らに私が知らないBashからの有名な引用を思い出させた...はい、私は見える-私たちはmakovodovが大好きです:)））



何も、最後に笑う人はよく笑います:)私はすぐにこれを確認することができました。 2週間、インターネットは「私は働きたい、したくない」モードで働いており、最も都合の悪い瞬間に定期的に飛び出していました。 最初はある種の障害があり、次に機器が変更され、それから何か他のものが変更されました。 そして今、最後の試験の前日に、インターネットは再び忘却に陥り、表示されなくなります。 プロバイダに再度電話するのを恥ずかしく思いますが、もう一度彼に迷惑をかける必要があります。インターネットが切実に必要であり、質問のもう半分を学ぶ必要があります。インターネット上の教科書です。



私のインターネット接続を復活させる試みが何も起こらなくなったとき、管理者が私を訪ねてその場で設定しました。 それから、Macに一度も座ったことのない人が初めてMacで何かをしようとするとどうなるかを見ました。 私は、Macが最も恐ろしく、不快で、理解できないコンピューターであると管理者が決定したと確信しています。 そもそも、スタートボタンはありません。 結論：何かがうまくいかない場合、Air Portユーティリティ、ネットワークユーティリティ、ターミナル、およびコンソールがMacのどこにあるかを知る必要があります（Finder>アプリケーション/ユーティリティ>ユーティリティを探してください）およびシステム設定。



私たちは座って、何が起こったかを非常に長い間知りました。 すべての設定が正しく入力され、Air Portのライトが緑色になります。これは、ネットワークが見えるが、応答せず、接続がないことを意味します。 すべてが非常にシンプルであることが判明しました。システムでMACアドレスが変更されたために、おそらく何かを再起動した結果、そこでどのように発生したかわかりません。 正しいものを入力すると、すべてが再び機能しました。 したがって、MACアドレスの重要性を再度思い出します。これが発生した場合、何をすべきかがわかります。
