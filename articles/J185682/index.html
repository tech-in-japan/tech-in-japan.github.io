Fujitsu ServerView Suite

これまで、市場には、上記の操作を実行するための統合システムと、1つまたは2つの機能を実装する個々のユーティリティの両方が多数あります。 サーバーのライフサイクルの主要プロセスを効果的に管理できるこれらの包括的なソリューションの1つに、Fujitsu PRIMERGYサーバーに付属するFujitsu ServerView Suiteがあります。 このパッケージの機能のほとんどが無料であり、追加コストを必要としないことが重要です。



ServerView Suiteは、IT運用の柔軟性を高め、生産性を高めながらコストを削減し、24 x 7モードで高いサーバーパフォーマンスとフォールトトレランスを維持するのに役立ちます。

ServerView Suiteの展開



このグループには、Installation Manager、Deployment Manager、Scripting Toolkit、Altiris DS用Integration Packのコンポーネントが含まれます。 別のライセンスが必要なDeployment Managerを除き、他のすべての製品はFujitsu PRIMERGYサーバーに含まれています。



ServerView Installation Manager-サーバー上のオペレーティングシステム（Microsoft Windows、Red Hat、SUSE）またはハイパーバイザー（VMware、Xen）の初期ローカルまたはリモートインストール機能を提供します。 オペレーティングシステムを複数のサーバー（最大5台）に並行してインストールでき、ユーザーの介入なし（無人）でのインストールをサポートし、Fujitsu PRIMERGYサーバーのハードウェア構成を自動的に決定し、適切なドライバーをインストールします。



ServerView Deployment Managerは、大量サーバーの試運転に焦点を合わせた追加の有料モジュールです。これにより、5台の並列サーバーでInstallation Managerの制限を回避できます。 データセンターの展開時に最大80％の時間を節約します。



ServerView Scripting Toolkit -RAID、BIOS設定、およびiRMCを構成できるWinPEおよびLinux（Red HatおよびSUSE）用のユーティリティセット。



ServerView Suite Control



このプログラムグループの主なアイデアは、単一の完全に機能する中央管理ポイントを提供することです。 これは、中央管理ステーション（CMS）-Fujitsu PRIMERGYサーバーに関する情報を収集する特別なServerView Operations Managerソフトウェアをホストする別のコンピューターまたはサーバー（仮想コンピューターを含む）に基づいて作成されます。 管理ソフトウェアのこの中央モジュールは、物理サーバーと仮想サーバー、および関連するインフラストラクチャコンポーネントを監視および分析します。 ユーザーの役割に基づいて、サーバー監視、電源管理、パフォーマンス管理、およびストレージ管理を提供するモジュールからサーバーサービス担当者にアクセスします。



サーバー監視機能（サーバー監視）を提供するServerView Operations Managerは、すべてのサーバーと作成されたグループの概要を提供します。







実際、サーバーリストは、ほとんどすべての管理機能の出発点です。 これから、特定のシステムを1つ表示できます。これにより、選択したサーバーに関する情報を最初に全体として取得できます。







次に、個別に選択されたサブセクションごとに詳細に説明します。







問題の状況について責任者にタイムリーに通知するために、 ServerView Event Managerが提供されます。ServerViewEvent Managerは 、管理者のワークステーションで電子メール、SMS、ポップアップメッセージなどの特別なメッセージを送信して通知を送信できます。 また、イベントマネージャは適切なログを保持し、特定の条件が満たされたときに機能する特別なスクリプトを作成する機能を提供します。 そのため、たとえば、1つまたは別のセンサー（トリガー）がトリガーされる場合、プログラムの起動を構成できます。



さらに、Operations Managerは、プロセッサ、RAM、ファン、ハードドライブ、CMOSバッテリーなどのサーバーコンポーネントに事前障害検出と分析（PDA）の検出と分析を実装しています。



サーバーの電源管理機能は、サーバーとそのグループのエネルギー消費に関する統計を収集すること、およびサーバーまたはサーバーのグループの最大消費電力を制限する可能性にあります。 後者の場合、サーバーは、クロック周波数を下げるか、個々のプロセッサコアを切断して指定レベルの電力消費を確保することにより、パフォーマンスを低下させます。 この機能により、エネルギーリソースを時間内にサーバー間で柔軟に分散し、ラックやデータセンターへのサーバーの設置密度を高めることができます。



Performance Managementは、サーバーリソースの使用と負荷を監視します（CMS経由）。 次に、このようなワークロードを分析することにより、ボトルネックと未使用のリソースの両方を特定できるため、サービスまたは仮想マシンを過負荷のサーバーから空き容量のあるサーバーに移行して最適化できます。



ストレージ管理機能は、サービス対象サーバー自体のストレージシステムを監視および管理するための特別なプログラムであるServerView RAID Managerを通じて実装されます。 これにより、ディスクサブシステムのステータスの監視、イベントのログ記録、RAIDアレイの作成、グラフィカルユーザーインターフェイスを介した管理が可能になります。



ServerView Suite Dynamize



これはServerView Suiteの唯一の領域で、すべてが有料コンポーネントです。 ServerView Virtual-IO Manager（VIOM）とServerView Resource Orchestratorの2つのエディション：Virtual Edition（ROR VE）とCloud Edition（ROR CE）が含まれます。



ServerView Virtual-IO Managerは、MACアドレスとWWNアドレスを仮想化することにより、富士通PRIMERGYサーバーにI / O仮想化を提供します。これにより、ハードウェアサーバー間の仮想マシンの移行が簡単になります。



ServerView Resource Orchestrator Virtual Edition は仮想サーバーと物理サーバーの均一な管理を提供し 、ServerView Resource Orchestrator Cloud Editionのより高度なバージョンでは、本格的なIaaSクラウドインフラストラクチャを構築できます。



ServerView Suite Maintain



すべてのFujitsu PRIMERGYサーバー（低コストTX100モデルを除く）には、ステータスと場所に関係なく完全なリモート管理（リモート管理）サーバーを提供する統合されたリモート管理コントローラー（iRMC）が組み込まれています。 オペレーティングシステムがインストール、オフ、またはフリーズされていない場合でも、サーバーへのアクセスを提供します。 iRMCには、センサーの測定値とシステムイベントの受信と解釈を簡単にするWebサーバーが組み込まれています。



Advanced iRMC Advanced Packの機能（ブレードサーバーに無料で含まれ、他は別途購入する必要があります）は、キーボードとマウスを含むサーバーのグラフィカルコンソールとテキストコンソールのリダイレクト、およびリモートストレージ（ハードディスク、光学ドライブ、またはUSBドライブ）のサーバーへの接続を提供します。



ServerView Update Managerを使用すると、サーバーのファームウェア（ファームウェア）とBIOSのファームウェアを更新できます。また、ドライバーを更新し、ServerViewエージェントを個別のサーバーに直接、および多数のサーバーに並行してインストールできます。 スケジュールおよびサーバーで更新を構成する機能があります。



ServerView Asset ManagerとServerView PrimeCollectは、サーバーのハードウェアとソフトウェアに関する情報を収集、保存、処理するように設計されています。 さらに、ServerView Suiteには、RAM、プロセッサ、ハードディスク、光学ドライブのテストを可能にするアプリケーションであるServerView Online Diagnosticsが含まれています。



ServerView Suiteの統合



最後のグループには、ServerView Suiteを最も一般的な管理システムや他のベンダーの専門ソリューションに統合する機能が含まれています。 特に、Fujitsu ManageNow、Microsoft SCCM、Microsoft SCOM、Microsoft SCOM / SCVMM（PRO Pack）、HP Operations Manager、HP Systems Insight Manager、Nagios、Icinga、BMC Patrolへの統合がサポートされています。 これらのソリューションとの統合は、適切なServerView Integration Packを使用して行われます 。
