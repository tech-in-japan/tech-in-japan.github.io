フリーソフトウェア：危険と魅力

フリーソフトウェア-なぜメーカーはそれらをリリースするのですか？ フリーソフトウェアは高品質で便利で安全だと思うのはなぜですか？ 実際には、これら3つのポイントのうち1つだけを選択する必要があります。 景品への信仰とは何ですか？ または、無料チーズはネズミ捕りでのみ発生するため、火のような無料プログラムから逃げるべきでしょうか？ 理解してみましょう！





2013年、フリーソフトウェア市場を興味を持って見ました。 これは、人気のあるアメリカのリソースCnet.comに助けられました。Cnet.comでは、ダウンロードされた内容と回数に関する詳細な統計情報を含むソフトウェアの選択が常に更新されています。 もちろん、私は新しいソフトウェアに興味がありました。 真新しい。 しかし、そのようなプログラムは、私はほとんど遭遇しませんでした。 しかし、明確で永遠の機能を備えた非常に高度なプログラムは、ユーザーの間で非常に人気がありました。 主なことは、それらのすべてで、1つの共通の特徴が推測されたことであり、それを次のように定式化します。「彼らは私たちのためにお金を稼ぐ方法を見つけました！」 しかし、私たちはお金を稼ぐことを恐れるべきですか？



手始めに、良い。 私が出会った人気のある無料プログラムには、まったく同じ肯定的な機能がありました。 まず、彼らは本当に無料で、直接お金を要求しませんでした。 第二に、それらは非常に使いやすいものでした。 子供でも使用できます。 第三に、彼らは述べられたタスクをほぼ100％解決しました。 まあ、それは素晴らしいことではありませんか？



キャッチは何ですか？



さらに、疑わしいワームは常に私に質問をしました-キャッチは何ですか？ メーカーはどのように、そして何でお金をmakeけますか？ 私が発見した典型的な方法は次のとおりです。



•「デフォルト」インストールモードでの追加のシェアウェアまたは無料ソフトウェアのインストール（ここで、開発者はキットに付属するソフトウェアの作成者からお金を受け取ります）

•明らかでない場合もあるが、プログラムの有料版の広告。 たとえば、プログラムの一部のセクションでは、コンテンツを表示するには有料版が必要であると報告されています。 また、定期的にポップアップウィンドウが表示され、より機能的な有料バージョンにプログラムを更新することを提案します（これは特にウイルス対策ソフトウェアで実行されます）

•Cnet.comなどのリソースをダウンロードするためのプログラムの配置。そこでは、プログラムの開発者が受け取ることができる有料広告がいっぱいです。



私の意見では、このソフトウェアが設計されているユーザーの大多数は、開発者に何らかの方法を提供するでしょう。 これらがすべて些細なものであり、ここでお金を稼ぐ方法がないと思われる場合、あなたはフリーウェア市場の熟練した戦闘機です。 次のプログラムをインストールするときに、すべての問題を解決し、次のプラグインでPCをいっぱいにしないでください。時には、本当のクエストになります。



フリーソフトウェアを安全に使用するには？



フリーソフトウェアに関して私にとって最も怖いのは、セキュリティの問題です。 無料プログラムには他に何がありますか？ 彼女はあなたに他に何を追加しますか？ 彼を知っている人。 したがって、私はすべての新しいプログラムを疑っています。 「プログラムをインストールしました-問題を解決しました-削除しました」という原則に従って使用します。 フリーウェアを使用するためのアルゴリズムとまったく同じです。 考えてみてください。PCで常に使用する無料のプログラムはごくわずかであり、常にPCにあると信頼するべきではありません。



フリーソフトウェアのトップリスト



2013年に、有用な無料プログラムのリストに、アメリカの評価からのいくつかの新製品が追加されました。 これらのプログラムは次のとおりです。



•Wise Care 365 Free（ http://wise-care-365.ru/ ）-さまざまなソフトウェアのジャンクやエラーからPCをクリーニングするための無料プログラム。 私は良いレビューとすべての作業が1つのボタンを押すことで構成されているという事実に魅了されました-「修正」。 つまり、プログラムはPCをスキャンし、考えられるすべてのエラー（レジストリ、古いシステムファイルなど）を探し、これらすべてを一気に排除します。 ユーティリティの人気は理解できます-何も理解する必要はありません。 配信済み-起動済み-削除済み。 PCの方が高速で、ディスク容量が増えます。 このプログラムには有料版がありますが、購入する必要性は疑わしいです。







•Driver Booster（ http://driver-booster.ru/ ） -PCまたはラップトップ上のドライバーを更新するための無料プログラム。 一度に、彼女は信じられないほどのダウンロード数を受け取りました。これはランキングで際立っていて、私の注目を集めました。 一般的な概念のSoftina：起動し、1つのボタンを押しました。 私はこれらの傾向が本当に好きです。 スマートソフトウェアはそれ自体ですべてを行う必要があります。 そして、ホームコンピューターのすべてのドライバーと一緒に1つの[スタート]ボタンを使用してプログラムに割り当てるのは楽しいことです。 一般に、プログラムはそのシンプルさと利便性に非常に満足しています。 配信済み-起動済み-削除済み。







•無料のウイルス対策ソフトウェア（最も人気のあるウイルス対策ソフトウェアは、 http ： //www.besplatnyi-antivirus-skachat-besplatno.ruなどに明示されています ）-このソフトウェアはもう少し複雑です。 第一に、たとえばVirus Bulletinのメンバーである有名なメーカーのみを信頼できます。 さらに、各プログラムには独自のニュアンスがあり、単純な「設定して忘れる」単純な操作を期待するためにここでは機能しません。 広告や追加サービス（Avast）、弱い保護（Microsoft Security Essentials）、誤検知（Avira）などのいずれかで入手できます。一般的に、理想的な無料ソリューションはありません。 あなたはより小さな悪を選ぶだけです。 無料のウイルス対策ソフトを守るために、有料プログラムは通常さらに悪いと言えます。 少なくとも、カスペルスキーやノートン製品での私の経験は、さらに否定的な感情を引き起こしました。







結論



最新の無料プログラムは非常に便利なツールです。 それらを使用することは本当に喜びです、主なことは合理的な予防措置に従うことです。



私は、無料プログラムを使用したい人にいくつかのアドバイスをすることを敢えてします。



•開発者のサイトからのみフリーウェアをダウンロードします（ファイルホスティングやトレントはありません）

•PCにウイルス対策ソフトウェアがない場合、プログラムをインストールしないでください。

•デフォルト設定を使用しないでください。

•無料プログラムが安定して動作し、すべての問題を解決できる場合は、急いでアップデートしないでください（新しいバージョンは新しい広告やその他の不便をもたらす可能性があります）

•数か月以内に無料のプログラムが必要ない場合は、アンインストールします。



これらの簡単な推奨事項により、フリーソフトウェアとの通信が安全になります。
