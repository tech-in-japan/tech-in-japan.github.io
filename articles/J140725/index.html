Android 戦闘アプリケーションの概要

良い一日！



「 iPhone：ポケットからのMiTM攻撃 」という記事に加えて、Androidに関するほぼ同様の記事が誕生しました。



私たちはすでにiPhoneの能力を知っています。 Androidはそれに負けますか？



約25件の戦闘申請が審査されました。 ちょっとした調査の結果をお伝えしたいと思います。 多くのアプリケーションは起動さえせず、一部は電話をしっかりとハングさせましたが、一部は動作しました！



すべてのソフトウェアは、Android 2.3バージョンの携帯電話LG Optimusでテストされました。



したがって、Androidの戦闘ソフトウェアの概要：







1. サメ -同じサメ 。 はい、Androidの下にもあります。 それは完璧に動作します。 問題なく巻き上げられたデバイス上。 * .pcap形式でログを書き込みます。 SDカードで折ります。 このファイルは、WindowsマシンとShark Readを使用した電話自体の両方で簡単に解析されます。 （優れたアプリケーション。特に電話がWiFiアクセスポイントとして機能する場合）





2. DroidSheep / Facesniff -Webセッションの傍受。 非常にシンプルですが、センセーショナルなアプリケーションです。 オープンポイントにしがみついて、開始して待機します...ところで、ポイント自体を一時停止することもできます。



3. WiFiKill-からのアプリケーションにはシリーズが必要です。 あなたがいるサブネット全体をスキャンします。 デバイスのリストを表示します。 「いいね！」を選択し、チェックマークをクリックして、数秒待ちます。 デバイスはインターネットなしで残ります。







4. MACアドレスの設定-MACアドレスを変更します。 パラグラフ3と併用するとうまく機能します。



5. Net Swiss Tool Free / Fing-ワイヤレスネットワークをスキャンし、接続されているデバイスのリストを表示します。 各デバイスを個別にスキャンし、開いているポートのリストを表示できます。 裁判所には、ping、トレース、wake on lan、arp、udp-floodも含まれています。





6. Wi-Fi Analytics-美しいアプリケーション。 利用可能なすべてのアクセスポイント、SSID、mac、暗号化、信号強度を表示します。







7. ホストエディタ -/ etc / hostsを編集できます。 特に電話がWiFiポイントとして機能する場合に便利なアプリケーションです。







8. kWS-Android Web Server -Webサーバー。 パラグラフ7とともに、うまく機能します。



9. RouterAttack / Route Brute Force ADS 2 -Androidでの実際のBruteForce！ 各アプリケーションは、基本アクセス認証を強制しようとします。 ソフトウェア自体は少し湿っています。 しかし、私のポイント、パスワードは12345です-非常に簡単かつ迅速にスクロールしました。 通常の操作では、良い辞書をダウンロードすることを忘れないでください。



10. Router KeyGen-近くにあるルーターから、Androidスマートフォン用に事前にインストールされたWPA / WEPアクセスキーを選択します。 標準のThomson、DLink、Pirelli Discus、Eircom、Verizon FiOSでうまく機能します。



11. Android Network Toolkit-Anti -Universalアプリケーション。 ネットワークスキャナー、スニファー、MITM、リモートエクスプロイト！ プラグインによる拡張可能な機能。 より多くの機能-より多くのお金がかかります。



ほとんどのアプリケーションにはルート権限が必要です。
