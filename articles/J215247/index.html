私が偶然自分のために無制限の（！）インターネットをmtsからusbモデムで掘り出したので、最低料金で

ご注意



従業員が支払うので、読むことはお勧めできません。 記事の具体的な焦点が強すぎます。



自慢



現時点では、完全に無制限のインターネットのすべての魅力を、1週間前に300ルーブル （サーバーの負荷に応じて） 〜1-9 Mbit / sの速度で自由に使用しますが、これは余裕がありません。

今、私の「SIMカード」だけでなく、インターネットのコスト、そして最終的にこのインターネットのエンドユーザー、すなわち 私。 正義は勝利しますが、すぐに店を閉めると思います。



バックグラウンド。 まあ、彼女なしでは不可能です



だから、私はmtsのusbモデムのユーザーであり、経済的な学生として、250ルーブルの最も安い関税を月4GBの悲しいトラフィック制限で接続しました。

かなり偶然、関税に関する最良の取引を求めてMTSのウェブサイトを登ったり、場合によっては自分自身を何らかの選択肢に結びつけたりすることさえあります。 私は不可解なオプションに出くわしました：



ターボナイト

費用： 50ルーブル/月

有効期間： 午前3時から午前8時まで （加入者の居住地域の時間による）

有料トラフィックなし、つまり 完全無制限



5時間でトレントをどれだけダウンロードできるかをカウントし、頭の上の質問に少し反省します。

必要ですか？

私は結論に達し、翌日、1か月間接続して、それがどうなるかを確認しました。

それが判明したのも不思議ではありません。

もちろん、トレントをダウンロードしました。一晩中、15 GB以上をポンプで送り出しました。 そしてすぐに、彼は契約で指定された時間に達すると気付き始めました：朝8時、誰も私からインターネットを奪い去り、まだ完全に請求されていませんでした。

私は大学から来たときにだけこれを知りました。 トレントをオフにするのを忘れたのに気づきましたが、速度はまだ〜8 Mbit / sでした。 そのときは奇妙に思えたので、もちろんためらうことなく、問題はサーバーにあると判断しました。



ハック



「問題」のこのような奇妙な繰り返しを2、3回繰り返した後、私はその理由が何であるか疑問に思い始めました。 トレントをオンにしたまま8時の時点でインターネットが切断されないのはなぜですか？

私は問題を完全には確信していませんが、これは私の見解です！

答えは簡単でした-接続チャネルは開いています。

私は思う：それでは、なぜ、インターネットが常に超高速にとどまらないのか？

そして、一度、私はより良いアイデアを得ました：種をポーリングする速度。

ターミナルを開いて、単純な無限のpingでYandexサーバーをプッシュし始めました： ping ya.ru

そして今、あなたは私が人としてインターネットを使用しているとさえ言うことができます！

cronでタスクを作成したので、今はこのpingを気にしません。大学から来て、座ってオンラインで映画を見るだけです！



使用説明書



1.最も安い関税-250ルーブルを接続します

2.オプションのターボナイトを接続-50ルーブル （価格が上昇した可能性があります）

3. 7.55で目を覚ます

4. 顔が青くなるまで何かをpingする

5.いかなる場合でも、ネットワーク接続をリセットしないでください（！） 常にオンラインにしてください

6. サーフィン！



合計： 300ルーブル + 小さな努力 = USBモデムからの通常のインターネット 。
