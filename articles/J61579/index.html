新しいOperaモバイルブラウザがモバイルインターネットを加速









新しいコアと一連の最新のデータ処理技術を備えたベータ版ブラウザOpera Mobile 9.7を使用すると、携帯電話からインターネットでの作業を新鮮に見ることができます。



ノルウェー、オスロ、2009年6月8日 -Opera Softwareは、Windows Mobileを実行する電話およびコミュニケーター向けにOpera Mobileブラウザーバージョン9.7のベータ版をリリースしました。



新しいバージョンのブラウザの主な利点は、さらに高速であることです。 Opera Mobile 9.7は、新しいOpera Presto 2.2カーネルを使用します。これにより、Webページのレンダリングが高速になり、パフォーマンスが向上します。 ブラウザはOpera Turboテクノロジーをサポートしており、携帯電話へのデータのダウンロードを高速化します。 さらに、一連のウィジェットとそれらを管理するためのアプリケーションがベータ版に含まれています。 これにより、携帯電話からWebサイトのコンテンツに簡単にアクセスできます。



Opera Mobile 9.7 for Windows Mobileのベータ版は無料で利用できます： www.opera.com/mobile/download



Opera Mobile 9.7の主な機能

Opera Mobile 9.7のベータ版はOpera Turboテクノロジーをサポートしています。 主な利点は次のとおりです。

-Webページの高速レンダリング。 Opera Turboのおかげで、Operaのサーバーがレンダリングを担当するようになりました。 Opera Mobile 9.7ブラウザ無料のモバイルデバイスリソースページの読み込み速度と全体的なパフォーマンスの改善

-データ圧縮。 Opera Turboは高度な圧縮技術を使用しており、ダウンロードされたデータの量を最大5倍削減できます。 したがって、セルラーネットワークの品質やモバイルデバイスの技術的特性に関係なく、Opera Turboを使用すると、最新の3G電話よりも高速にインターネットを閲覧してページを読み込むことができます。



新しいPresto 2.2カーネルのおかげで、Opera Mobile 9.7は以前のバージョンよりも約25％高速にWebページのコンテンツを読み込みます。 Presto 2.2カーネルは、Opera 10デスクトップブラウザーでも使用されます。



ウィジェットの人気と普及を考慮して、Opera Widgets Managerと開発者向けのOpera Widgets SDKは、Opera Mobile 9.7のベータ版に含まれています。 Opera Widgets Managerを使用すると、ユーザーは画面を1回クリックするだけでお気に入りのWebリソースとサービスにアクセスできます。 Opera Mobile 9.7のベータ版には、GeoQuiz、Twitter、Bubbles、MyStatus、Google Translateのウィジェットが含まれています。



「他のモバイルブラウザには、Windows Mobile電話での作業に最も速くて最適であると主張するものがあります。 しかし、私は各「モバイル」ユーザーにOpera Mobileをインストールし、速度と互換性の新しい標準に参加することをお勧めします。 Opera Mobile 9.7では、低速で「重い」ブラウザがはるかに遅れています」とOpera Softwareの最高経営責任者であるJon von Tetzchner氏は述べています。 -私たちはOne Webのアイデアを信じており、これによりOpera Turboなどのテクノロジーを作成することができます。 誰もが3G電話を持っているわけではなく、誰もがそれらを必要としているわけでもありません。 しかし、誰もが高速インターネットを必要としているため、Operaはユーザーが3G接続のすべての利点を感じられるようにブラウザーを改善しようとしています。EDGEネットワークで作業することもできます。



Opera Mobile 9.7の速度とブラウザのオープンWeb標準との互換性は、エンドユーザーだけでなく、通信事業者やWindows Mobileベースの機器メーカーにも高く評価されます。 携帯電話プロバイダーには、ネットワークの負荷を減らし、帯域幅を増やす本当の機会があります。 メーカーは幅広い携帯電話でOpera Mobileを使用できます。 実際、Webサイトの処理はOperaのサーバー上で実行されるため、携帯電話がインターネット上で動作するために必要なRAMが少なくなります。 最終的に、これによりコストが削減されます。



Opera Mobile 9.7のベータ版は、Windows Mobileベースの電話でのみ利用可能です

質問と回答のOpera Mobile 9.7



Operaは本日、何を発表しますか？

Opera Turboテクノロジーを搭載したOpera Mobile 9.7のベータ版を紹介します。 このブラウザは、Windows Mobileを実行しているスマートフォンで使用するために設計されています。

注： Opera Turboテクノロジーはデフォルトで無効になっているため、個別に有効にする必要があります。 有効にするには、[設定]メニューの[詳細設定]に移動し、[Opera Turboを有効にする]を選択します。



Opera Mobile 9.7はどのデバイスで動作しますか？

Samsung Omnia、HTC Diamond、Sony Ericsson Xperiaなど、Windows Mobileを実行するすべてのスマートフォン。



Opera Mobile 9.7ベータ版の主な機能は何ですか？

新しいブラウザの主な機能はOpera Turboテクノロジーです。これはSPEEDを意味します。 さらに、新しいOpera Presto 2.2カーネルは、Web標準のサポートを改善し、Webページのレンダリングを高速化します。



Opera MiniとOpera Mobileの違いは何ですか？どちらの場合もデータ圧縮技術が使用されているためですか？

Opera Mobileには独自のカーネルが組み込まれているため、Opera Turboを使用しなくてもブラウザーを動作させることができます。 さらに、Opera Mobileはデータ圧縮モードでのみ機能します。 これにより、JavaScriptおよびAJAXテクノロジーを使用したWebサービスを完全にサポートできます。



Opera Mobile 9.7はOpera Mobile 9.5とどのように違いますか？

Opera Mobile 9.7はOpera Mobile 9.5の後継です。 Opera TurboテクノロジーとPresto 2.2カーネルのおかげで、Webページの読み込みがさらに高速になりました。 車と同様に、Opera Mobile 9.5がポルシェ911である場合、Opera Mobile 9.7はポルシェ911ターボです。



このバージョンが10ではなく9.7と呼ばれるのはなぜですか？

特にOpera Mobile 9.5は当然の価値があり、本当に高品質の製品であるため、Windows Mobileの以前のバージョンと新しいブラウザーの連続性を強調したかったのです。



FlashアプリケーションはOpera Mobile 9.7で動作しますか？ YouTubeで動画を視聴できますか？

Opera 9.7はFlashプラグインをサポートしています。 電話機にインストールされている場合、Flashコンテンツを操作できます。



Opera Mobile 9.7のベータ版の機能は何ですか？

Opera Mobile 9.7とOpera Turboの両方がまだテストモードにあることを考慮する必要があります。 機能の改善に取り組んでいますが、この段階でユーザーにいくつかのニュアンスを理解してもらうようお願いしています。



ブラウザ

-Opera Turboテクノロジーがオンになっていると、ダウンロード機能（ダウンロード）および一部の設定（画像のオン/オフなど）が機能しません。

-解像度が480x800のデバイスで実行されているWindows Mobile 5.0のバージョンでは、縦画面モードと横画面モードの切り替えが正しく機能しない場合があります。 これらのエラーは、以前のバージョンのMicrosoftドライバーでこの解像度が十分にサポートされていないことが原因です。

-一部のエディターの一部の入力ツールは、これらのプログラムがMicrosoft SIPテクノロジーやIME標準をサポートしていないため、Operaブラウザーでは正常に動作しない場合があります。 Operaがそのようなエディターを電話で発見すると、ブラウザーは既知の（デフォルトでインストールされた）入力手段を使用します。 たとえば、プログラムEzInput v1.5では、キーボードとコンパクトなQWERTYモードは機能しませんが、他のすべてのモードではすべてが正常に機能します。 EzInputをバージョンv2.0に更新することをお勧めします。

-FlashLite 3の基本的なサポートのみ。Flashプラグインはデフォルトではインストールされません。

-Opera Mobile 9.7はOpera Link機能をサポートしていません。 ただし、ユーザーはWebインターフェースとlink.opera.comからブックマークにアクセスできます。



ウィジェット追加マネージャー

-Opera Widget Managerはブラウザーのアドオンとして機能します。 ダウンロードするには、まずOpera Mobile 9.7をインストールする必要があります

-Opera Widget Managerを使用する場合は、Opera Turboモードを無効にすることをお勧めします

-画面の向き（縦から横、またはその逆）を変更するときにウィジェットを回転すると、正しく機能しない場合があります。 ウィジェットを使用するには、ポートレートモードを使用します。

-プラグイン（Flash＆Gears）と同時にウィジェットが正しく動作しない場合があります

-ウィジェットの内部リンクをクリックすると、Opera Mobileブラウザーが起動します

-適切なウィジェットを使用してTwitterアカウントに初めてログインすると、ユーザー名とパスワードを認識できない場合があります。 Twitterウィジェットを閉じて、もう一度開きます。



Opera Turboモードは画質を低下させますか？

Opera Turboをサポートするサーバーは、モバイルデバイスに送信される画像を最適化します。 小さな画面の携帯電話では、デスクトップバージョンのブラウザーよりもデータを圧縮できます。 そのため、画像圧縮は、サーバーが実行するコンテンツ最適化プロセスの一部にすぎません。

大きな画像は主に圧縮されています。携帯電話の画面の解像度がQVGA以下の場合、ユーザーは1024x768ピクセルの写真をアップロードする必要はありません。 また、BMP画像をJPEG形式に変換し、アニメーションGIFの最初のフレームのみを表示します。 これにより、トラフィックを大幅に節約できます。



ユーザーはOpera Turboのサーバー設定を変更できますか？

いいえ、ユーザーはOpera Turboのオンとオフのみを切り替えることができます。 このテクノロジーをできるだけ使いやすくしたいと考えています。 同時に、サーバー自体が分析し、特定のデバイスでOpera Turboを使用することが最も効果的な動作モードを確立します。



ウィジェットは今どのように機能しますか？

S60プラットフォーム用のOpera Widget ManagerはOpera Widgets SDKに含まれており、dev.opera.comからダウンロードできます。 Opera Mobile 9.7のウィジェットは、アイコンとフルスクリーン画像の2つのモードで動作します。 Windows Mobileのアセンブリの3番目のモードである対話型アイコンは、S60のOpera Widget Managerとは異なり、正しく動作しますが、サポートされていません。
