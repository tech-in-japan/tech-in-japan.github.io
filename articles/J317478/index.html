12月16日-ロシアでのImagine Cupの公式ローンチ！ 詳細をご覧ください

今年、私たちの主要な学生コンテストImagine Cupは、国際的にもロシアでも重要な変化を遂げました。 これらの変更について詳しく説明するために、 ロシアでのイマジンカップの公式開催という特別なイベントを開催することにしました 。







場所： Microsoft Office、モスクワ、クリラツカヤ通り、17ビル1

いつ： 2016年12月16日、午後3時-午後10時

登録： こちら



イベントで何が起こるか



  • コンテストの公式ルールがどのように変更されたかを説明します
  • 今年のロシアでの競争の様子をお伝えします
  • プロジェクトで使用するのに役立つ最新技術の概要を簡単に説明します：ボットフレームワーク、IoT、Xamarin、Azure MLなど。
  • ご希望の方には、IoTとボットの作成に関する2つの実験室作業を実施します。 ラボ作業に参加するには、Visual Studio 2015がインストールされたWindows 10を実行しているラップトップを持ち込む必要があります
  • 成功したImagine Cupプロジェクトマネージャーが、チームプロジェクトの経験を共有し、学生プロジェクトの作業を分散します
  • イベントの最後に、専門家/メンターとの個別相談の時間を設け、特別なメンタリング/ナノ加速プログラムのために既に形成されたプロジェクトを登録します。


イベントの完全なプログラムはサイトで利用可能です 。



モスクワ出身ではないが、参加したい人のために



  • イベント全体の公式放送は行いませんが、PeriscopeまたはFacebook Liveで自発的な放送を行います。 #imcupタグを使用して検索します。
  • プロジェクトのアイデアを事前に共有し、メンターと話し合うか、ナノ加速プログラムに参加したい人-VKontakteプロジェクトのアイデアを説明してください。
  • #imcupタグを使用してTwitterで質問することができます。回答します


大きな変更



あなたが知っておくべきコンテストのルールの主な変更：



  • 国際大会の主な賞金は10 万ドルの規模に増加しました
  • チームの規模は現在1〜3人です。 過去数年間の傾向を分析し、ほとんどのチームが3人（たとえば、ロシアの2015年と2016年の勝者など）に適合することを認識しました。
  • 今年の主な焦点は、プロジェクトの技術的な実装になります （公式評価基準の50％の重量）。 同時に、Microsoft Azure クラウドテクノロジーの使用は必須です。
  • 競争の1つのカテゴリが3つのカテゴリに置き換わります。 参加するカテゴリを選択したり、プロジェクトをそのカテゴリに合わせたりする必要がなくなります


その結果、最新のハイテクプロジェクトはImagine Cupコンテストに参加できます （Microsoft Azureクラウドを使用している場合）。



イベントでお会いしましょう！
