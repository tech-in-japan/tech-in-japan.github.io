私はすでに無料の集中コードライブラリであるMicrosoft All-In-One Code Frameworkについてhabrで話しました。



Microsoft All-In-One Code Frameworkプロジェクトの目標は、すべてのMicrosoft開発テクノロジーの典型的なタスクのコード例を提供することです。 同時に、プロジェクトチームはこれらの例を空から発明しません。 どの例が必要かを判断するために、開発者フォーラム、ソーシャルメディア、およびマイクロソフトサポートへの連絡が検討されます。 これらすべてのクエリに基づいて、最も一般的なコードサンプルがライブラリに公開されます。



このようなライブラリは、コード例を簡単にナビゲートしたり検索したりする機能がないため、事実上役に立ちません。 また、ライブラリ開発者は、この機会をすぐに別のアプリケーションとVisual Studioのアドオンの形で提供しました。 アプリとアドオンは、オールインワンコードフレームワークのサンプルを使用する便利な方法を提供しましたが、Microsoftのサイトには、オールインワンコードフレームワークに含まれていない他のコードサンプルがあり、内部からアクセスするのに便利です1つのプログラム。 そのようなアイデアを思いついたのは私だけではなかったと思います。



数週間前、 コードサンプルナビゲーターの新しいバージョンがリリースされました。これにより、MSDNコードギャラリーのサンプルや公式のWindows 8サンプルなど、3,500以上のサンプルにアクセスできます。
