Debian GNU / Linux 5.0.6リリース

Debianキャンプは深刻なリバイバルです。 8月の初めに、 Squeezeディストリビューションブランチがフリーズしました 。これは現在テスト中です。その後、安定したリリース状態に移行し、Lennyリリースを置き換えます。先日、次のバージョン7.0ブランチに名前を付けました。 トイ・ストーリーの三部作に注目。

それにもかかわらず、開発チームは既存のファンのランク、リリースの更新を忘れないでください。 2010年9月4日に、コードネーム「Lenny」と呼ばれる6番目のDebian GNU / Linux安定版リリースアップデートが導入されました 。

リリースに含まれていた重要なバグ修正を見てみましょう。

bind9パッケージのDSA-2054 （DNSキャッシュポイズニング）、 wgetパッケージのDSA-2088 （潜在的なコード実行）、 php5パッケージのDSA-2089、2.6カーネルカーネルのDSA-2094 （特権エスカレーション/サービス拒否/情報漏洩）、 opensslパッケージのDSA-2100 （二重無料）。 さらに、 libconfig-inetd-perlパッケージが削除されました。

いつものように、ISOイメージは今後数日でミラーにヒットしますが、aptは今すぐ更新できます。
