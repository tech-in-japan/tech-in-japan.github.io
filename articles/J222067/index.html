ネイティブインプレッションとNAS Synology DS214レビュー

こんにちは、ハラジテリ。 本日、NAS Synology DS214を使用した感想をお伝えしたいと思います。







私の作品が無駄にならないことを願っています。誰かがこの記事を気に入ってくれて、将来の選択に役立つことを願っています。



Runetオープンスペース（約6か月前）の時点で、公式サイトで誰でも見つけられる広告記事と機能と写真の概要を除いて、このデバイスの概要を見つけられなかったため、これを行うことにしました。 このミニレビューでは、コネクタの数、外観などに関する明確な情報は書きません。SynologyWebサイトにアクセスして、元のソースSynology DS214で確認できます。 私の意見では、これはNASaのパフォーマンスの主な指標とはほど遠いため、ハードドライブの速度も測定されませんでした。明らかに、仕事用のネットワークドライブ用に特別に設計された最速のWD Redシリーズハードドライブを使用していないからです24 \ 7。







そもそも、この特定のデバイスで選択肢が選ばれた理由と、NAS（Network Attached Storage）が必要な理由をお話ししたいと思います。



正直なところ、7か月前にはNASが何であり、何に使用されているのかわかりませんでした。 あらゆる種類のネットワークドライブ、RAIDアレイ、その他のハードウェアがたくさんあることは知っていましたが、NASの定義はあることを知りました-それは知りませんでした。 それが何であるかを見つけ、購入を決定するのに1ヶ月かかりました。



当初、目標は次のとおりでした。

-バックアップコピーの保存。

-大量のデータ（ビデオ、写真）の保存。

-どこからでもデータにアクセスできます。

-RAID 1アレイの必須データミラーリング。

-PCのルーチン機能の一部を小さなボックスに入れます（PCを除くすべてのファイルにアクセスするため）。







これらの目的には、おそらく最も安価なネットワークドライブが適しています。 しかし、私は余裕を持ってもっと欲しかったので、この余裕を受けました。



このモデルを選んだ理由はいくつかあります。

-非常に魅力的な価格を見つけたのは私の都市でした。

-外部デバイスを接続するためのUSBポートがあります。

-コンパクトサイズ（パン一loのような）;

-確立された名前Synology。

-RAID 1を作成するためのハードドライブ用の2つのベイ。

-OSおよびソフトウェアのタイムリーな更新。

-NAS機能をインストールおよび拡張するための既製のパッケージが多数あります。

-PCの電源を入れたままにしない、静かで小さなデバイス







個人的に私にとってNASとは何ですか？ 私にとっては、これは24時間365日外部ネットワークに接続されている鉄片です。 これは静かに動作し、電力をほとんど消費せず、一般にその存在を伝えません。 この鉄片には、すべての写真、重要な文書、すべての情報がアップロードされています。 複数のマシンからの自動バックアップを構成しました。 WebDAV / FTPサーバーが起動し、すべての友人がアクセスできます。 また、iPhoneのどこでも写真やビデオを見ることができます。Synologyには、あらゆる好みに対応する幅広いソフトウェアがあります。これについては、以下で説明します。







データおよびバックアップストレージ



これはおそらく私にとって最も重要な機能の1つです。 学生時代に、自分自身の過失または偶然の過失により、文書、学期論文、卒業証書を数回失いました。 その後、いつか似たような状況が必ず起こると思い、私の写真、ビデオ、重要な文書をすべての年の人生で失う可能性があるという考えがしばしば思い浮かびました。 そして、これらは最終的に、記憶、感情、財政です。 写真は、私がNASで持っている最も価値のあるものです（もちろん、バックアップも）など 幸いなことに、これらのファイルの重量は数メガバイトです。



最近、私の友人が、愚かさのためにコンピュータ上で持っていた連絡先をすべて失いました。



Windowsでバックアップを設定するのは非常に簡単です。 標準ツールのバックアップと復元を設定します。これは、ネットワークドライブであるネットワークドライブにバックアップを毎週ダウンロードします。 購入後、1つの問題が発生しました。100Mbpsは通常のバックアップを読み込むには小さすぎ、重量は約150GBです。 したがって、最も安価な1Gbitスイッチが購入され、問題は解決しました。 今では、Windowsがコピーを実行することに気づきさえしません。 1Gbpsネットワーク上のNASは、独自のハードドライブに近い速度で実行されます。







FTP / TFTP / WebDav



個人的にはこれらのプロトコルはあまり使用しませんが、NASeにデータを保存する人（知人、友人）が使用します。 はい、ディスクスペースの一部を友人や知人と共有しています。なぜなら、多くの人が重要なファイルを持っていますが、誰もがバックアップストレージ用のデバイスを持っているわけではないからです。 数百メガバイト、または数ギガバイトで申し訳ありませんが、それは人々をフラストレーションやトラブルから救うことができます。



YOUにコンピューターがある人のために-WebDavを構成しました。 この場合、ユーザーは自分のコンピューターで別の論理ドライブを見るだけで、残りを知る必要はありません。 WebDavを設定するには、チェックマークを有効にしてユーザーに権限を与えるだけです。











FTP / TFTP私はコンピューターが得意な人と共有します。彼らは彼らがどんな種類のプロトコルであるか、それらをどのように使用し、どんな制限を課すかを知っています。

同じように含まれています-1目盛りとユーザー権限があります。



接続されているすべてのユーザーは、デスクトップ上のウィジェットに表示されます。 雑誌でも見ることができ、とても便利です。







私自身はプロトコルをほとんど使用しません。 自宅では、ネットワークドライブであるネットワークドライブにデータをダウンロードしてアップロードし、職場やその他の場所では、非常に美しく便利なWebインターフェイスを使用しています。 ちなみに、どのモバイルデバイスからでも使用できます。



VPNサーバー



NASを理解する前に、VPNサーバーを継続的に使用したことはありませんでした。 しかし、この機会があったとき、私はすぐに鉄片でそれを構成しました。 VPNはどこで使用しますか？ パスワードのないWi-Fiがある公共の場所。 iPhoneには、私のサーバーと接続するvpnクライアントが構成されています-私は保護されています。



VPN Serverパッケージには、PPTP、OpenVPN、L2TP / IPSecの3つの接続オプションがあります。

それは非常に簡単に構成されます-1つのチェックマークとユーザー権限。







トレントとファイルのアップロード



私はこの機会をあまり使用しませんが、ほとんどの人はそれがどのように実装されているかを知りたいと思うでしょう。

このため、Package CenterにはDownload Stationパッケージがあります。これは、ただのトレントクライアントであり、ファイルのダウンロードです。 次のようになります。







ある場所では、ダウンロードされたトレントと通常のファイルが表示されます。 追加は非常に簡単です。プラスボタンはトレントを追加し、プラスは地球上にダウンロードリンクを追加します。 混乱するのは難しいです。



トレントトラッカーのリストでファイルを検索することができます。







フォトステーション



このパッケージは最も快適で便利です。 写真コレクションを整理して、世界中のどのデバイスからでも見ることができます。

パッケージをインストールし、作成したフォルダーに写真をアップロードすると、ファイルのインデックス作成とプレビューの作成が開始されます。これは、数千のコレクションがある場合は最大数日間続きます。 これは、写真デバイスを表示するたびにフルサイズの写真をロードする必要がないようにするために必要です。







すべてのパッケージを説明することはできませんが、簡単に言うと次のようになります。

-PHP + MySQLと組み合わせて完全なWebサーバーを構築するための一連のパッケージ。

-CMSパッケージ（Prestashop、Wordpress、Drupal）;

-メールサーバーパッケージを使用してメールサーバーを静かに設定します。

-ビデオ監視が必要ですか？ 問題ありません-Surveillance Stationパッケージがあります。

-ウイルス対策が必要ですか？ Antivirus Essentialパッケージがあります。

-テレビで視聴するための本格的なマルチメディアセンターが必要ですか？ Video StationおよびMultimedia Serverパッケージに加えて、設定も非常に簡単なDLNAを使用します。







つまり、多くのパッケージがあります。 ユーザーのほとんどのニーズを満たすことができると思います。



デモ版のSynology DSM Demoで、オペレーティングシステムとパッケージのリストについて詳しく知ることができます。

モバイルアプリ



すでに書いたように、Synologyはデバイスを操作するためのあらゆるモバイルアプリケーションを提供してくれました。 それにもかかわらず、私は、単に仕事だけでなく、本格的な仕事を強調します。それはデスクトップ版での仕事と大差ありません。 私はIOSを使用しているため、このオペレーティングシステムのパッケージについて説明します。



DS Audioは、Audio Stationパッケージに類似しています。 デバイスでオーディオコレクションを聞いて管理できます。

DS Videoは、Video Stationパッケージに類似しています。 すべてが似ています。 ビデオコレクションをご覧ください。

DS File -File Stationの類似物。 ファイルを管理し、デバイスにアップロードして、どこでもビデオ/ドキュメントを見ることができます。

DS Finderは、ローカルネットワークでNASを見つけるためのプログラムです。

DS Photo +は、写真を表示するための非常に便利なプログラムです。 すべてが非常に便利で美しく行われます。 Dropboxで行われているように、電話からNASへの写真の自動アップロードが実装されています。 これを行うには、プログラムを開くだけです。

DS Cloudは、Synologyと同等のDropboxです。 個人的に、私はこのパッケージを使用しません。 デスクトップバージョン用のより興味深いCloud Syncパッケージがあります。これにより、Googleドライブ、Dropbox、Baidu Cloudを同期できます。 ポイントは、これらのサービスに追加/削除するすべてのものが自動的にNASと同期されることです。 これは、クラウドサービスの一種のバックアップです。

DS Download -Download Stationの類似物。 ダウンロードを管理します。 停止、開始、追加、削除、統計など。

DS Cam-名前から、ビデオ監視を使用する場合、このパッケージを使用すると、モバイルデバイスからカメラを監視および制御できることが明らかです。

すべてのパッケージは、iPhoneとiPadの両方で利用できます。



個人的には、DS Photo +を使用して、映画やセミナーなどを携帯電話にアップロードする場合に、携帯電話とDSファイルから写真を表示およびダウンロードすることがよくあります。

ポート転送。 この項目をマイナスに追加することにしましたが、実際にはプラスがあります（プロセス自体の容易さ）。 はい。NATで働いていて、白いIPアドレスを持っていない場合は、ルーターのポートを転送する必要がありますが、すべてのユーザーに余裕があるわけではありません。 DS214では、これは非常に簡単です。 いくつかのボタンを押すと、黒いボックスがルーターを定義し、UPnP自体でポートのルールを作成します。 個人的には、私のTP-Link WR842NDルーターではこれで問題ありません。 しかし、DD-WRTはファームウェアをインストールする必要はありません。つまり、ファームウェアを正しく決定しますが、実際にはルールは正常に作成されますが、何らかの理由でルール作成プロセスの最後にエラーを書き込みます。 この場合、ルーターで手動でこれを行う必要があります。







標準ポート。 このマイナスは前のマイナスです。 標準の管理パネルはポート5000で機能します。つまり、別の場所からデバイスに接続するには、ブラウザにhttp ：//myname.myds.me►000と入力する必要があります。 ただし、1つの問題があります。5000のすべてが、プロバイダー/ルーターレベルでポートを開いているわけではありません。 したがって、すべての人が管理パネルにアクセスできるわけではありません。 標準ポート80はWebサーバーによって占有されています。 同様に、パブリックリンクを使用します。 現在Dropboxで行っているように、ファイルを誰かと共有したい場合は、DS214でファイルを右クリックして[ファイルリンクを交換]を選択します。 開いたウィンドウで、リンクの有効期間を設定します。 リンクはhttp：//myname.myds.mehaps000/fbsharing/yG67cCKgのようになります
