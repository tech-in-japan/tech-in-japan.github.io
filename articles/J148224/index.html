レスポンシブWebデザインは、Adaptive Web Designと呼ばれるより広い概念の一部です。 レスポンシブWebデザインとは、Webページのレイアウト（Ethan Marcott、「ゴム」レイアウト、柔軟な画像、メディアクエリ）のみを意味します。



一方、「レスポンシブWebデザイン」には、単なる「ゴム」レイアウト以上のものが含まれています。 実際には、これはプログレッシブエンハンスメントとほぼ同じです。要するに、このコンセプトは、可能な限り幅広い視聴者に最高の機会を提供するよう努めることを意味します。 また、「Adaptive web design」と「Adaptive layout」（Adaptive layout）の概念を混在させることはできません。 これらはまったく異なるものです。



最後に、「アダプティブレイアウト」とは、多くの固定水平寸法（幅）を組み合わせて作成されたレイアウトを意味します。 さて、すべてが正しく行われ、固定幅ブロックが「ゴム」ブロックと組み合わされる場合、アダプティブレイアウトは、その代替手段ではなく、レスポンシブWebデザインの形式の1つと見なすことができるように思えます。 Zeldmanは、2011年に彼のブログエントリの 1つで、すでに少し似たアイデアを表明しています 。
