オペラの新しいビルド。 ユナイテッド・アット・チョイス

Opera Uniteはまだ最終段階にありますが、ビルドは2つのカテゴリに分類されます。それぞれUniteなしとOperaありのOpera 10.0とOpera 10.10です。 Opera 10.10のベータ版のリリースは、メインバージョン10.0のリリース直後に予定されています。



既にopera:config#AutoUpdate|DownloadAllSnapshots



オプションを有効にしている場合opera:config#AutoUpdate|DownloadAllSnapshots



で、以前のビルドを既にインストールしている場合、今日のOperaはバージョン10.0にアップグレードする必要があります。

何らかの理由で、Linuxでうまくいきませんでした。 これは、おそらくユナイテッドのないアセンブリにのみ当てはまります。


ユナイテッドのテストを継続したい方のために、ビルドバージョン10.10をダウンロードする機会があります。ビルドバージョン10.10については、後で自動更新も機能します。



また、主な変更点のうち、OperaがFreeBSD 8で動作するようになったことは注目に値しますが、これまでのところ64ビットFreeBSD 7/8のビルドはありません。 また、フォトギャラリーサービスのゲストビューボタンは、プライベート/個人のアクセス権では機能しません。



Opera 10.0ビルドのダウンロードリンク

  • Macintosh（Intelのみ） / Macintosh（ユニバーサル）

  • UNIX / Linux

  • Macintosh（Intelのみ） / Macintosh（ユニバーサル）
