ユーリミルナーはDSTグローバルV投資ファンドを設立し、そこで17億ドルを調達しました

ロシアのビジネスマンYuri MilnerはDST Global V投資ファンドを立ち上げ、匿名投資家はこのファンドに17億ドルを投資しました。 このファンドの最初の株式の売却は、米国証券取引委員会（SEC）のウェブサイトに示されている今年の7月中旬に行われました。 基金はケイマン諸島で登録されています。



ミルナーが近い将来に追加資金を調達する計画があるかどうかはまだ不明です。 以前のミルナー基金と同様に、DST Global Vはインターネットのスタートアップに焦点を当てる-同社は戦略を変更していない、と情報筋は Vedomostiに語った 。 彼によると、DST Global Vにはロシアの投資家はいない。



合計で、Milnerの資金は、その存在中に約29億ドルを調達しました。 最初のDST Globalファンドの株主は、ビジネスマンのAlisher Usmanovでした。 今年4月、 VKontakteの共同設立者であるLev LevievとVyacheslav Mirilashviliは、DST Global IIおよびDST Global Iファンドの投資家になりました。



2014年6月、ユーリミルナーは DSTグローバルIV投資ファンドを設立しました。 3月、アメリカンフォーブスは、世界で最も成功したベンチャーキャピタル投資家100人のランキングであるミダスリストのトップ20 にミルナーを含めました。
