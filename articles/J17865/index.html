QEMUのOpenMoko

私は、最初の発表の日からOpenMokoプロジェクトを注意深くフォローしています。 ある時点で、モバイルLinuxをテストする私の興味と欲求は自然な怠naturalを上回りました:)エミュレーターとプラットフォームのインストールは驚くほど簡単でした。 私はあなたと私の結果を急いで共有します！



作業プラットフォームはDebian sidまたはUbuntuです。



$ sudo aptitude install gcc-3.4 lynx netpbm libsdl1.2-dev dosfstools subversion

$ mkdir openmoko

$ cd openmoko

$ wget www.rwhitby.net/files/openmoko/Makefile

$作るqemu



トラフィックをNメガバイトに費やす準備をしましょう-qemuなどのOpenMokoイメージ
