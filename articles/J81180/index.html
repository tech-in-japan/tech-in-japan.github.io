Webデータの視覚化：ガントチャート

Webでのデータの視覚化に関する最近の一連の投稿に関連して、ガントチャートを描画するためのライブラリをアドバイスするよう求められました。



ガントチャート （Eng。ガントチャート、テープチャート、ガントチャート）は一般的なタイプの棒グラフで、プロジェクトの作業計画や計画を説明するために使用されます。 これはプロジェクト計画の方法の1つです。





jsGantt



jsGanttは、ガントチャートを描画できる無料のJavascript / CSS / HTMLライブラリです。







ライセンス：BSD



ウェブサイト： www.jsgantt.com

Googleコード： code.google.com/p/jsgantt



SIconガントチャート



SIcon Gantt Chartは、ガントチャートを描画するための別のJavascriptライブラリです。







ライセンス：GPL



ウェブサイト： www.codeproject.com/KB/scripting/SIcon_Gantt_Chart.aspx
