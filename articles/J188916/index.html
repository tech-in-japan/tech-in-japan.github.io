Bmorn K12-Android搭載の変圧器タブレット



Bmorn K12は、10.1インチのタブレットで、すぐにBluetoothキーボードが付属します。これにより、ラップトップとして使用できます。





技術仕様



1280x800の解像度を持つ10.1インチIPS画面

PowerVR SGX544MP2ビデオアクセラレータ（8ストリームプロセッサ）

Allwinner A31クアッドコアCortex A7アーキテクチャ

2GB RAM

Android OS 4.1.1

16 GBのユーザーメモリ+ microSDカードスロット

2台のカメラ：背面5 MPおよび前面2 MP

通信：Wi-Fi（802.11 b / g / n）、Bluetooth

オプション：USB OTG

6000 mAhバッテリー

寸法：266x175x9.5 mm。



1280x800ピクセルの解像度の画面の場合、塗りつぶしは非常に強力です。 8コアPowerVR SGX544MP2は、すべてのゲームおよびグラフィックスパフォーマンステストで優れた結果を示します。



オプションと外観

タブレットは巨大な白い箱に入っており、中国からの配送時に少し押しつぶされました。





アクセサリキット：イヤホン、OTGケーブル、USBケーブル、充電器：





保護フィルムはスクリーンにあまりよく接着されておらず、その下にはいくつかの気泡があります：





背面の金属インサートにより、効率的な放熱が可能になります。





5MP背面カメラ：





インターフェースの一部は保護カバーの下に隠れています：





マイク、ヘッドフォン、充電器の入力は側面にあります。





MiniHDMI、microUSB、microSDメモリカードスロットは保護カバーの下に隠れています：





デュアルスピーカー、背面および側面：



音質は悪くありませんが、最大音量は貧弱です。



異物粒子からメッシュで保護：





別のボックスにBluetoothキーボードがありました：





白、英語のレイアウト：



キートラベルはとても楽しいです。



右側には、接続と充電のステータスインジケータと、「接続」および「オフ/オン」コントロールボタンがあります。





キーボードには独自の小さなバッテリーがあります。



製造元がキーボードに適切なバッテリーを取り付けて、タブレットを接続することでそのリソースを使用できるようになれば素晴らしいと思います。



制御盤：





タブレットをキーボードに保持するために、リクライニングプラスチックスタンドが使用されます。





タブレットを溝に挿入し、スタンドに置きます：





デザインは非常に安定しており、背中に倒れることはありません。





キーボードはタブレットに「固定」でき、外観上はラップトップに似ています。





スクリーン

10.1インチ画面はIPSテクノロジーを使用して作成されており、解像度は1280x800です。

色再現は良好で、最大輝度は高レベルです：





視野角：







性能

Allwinner A31チップは、周波数1.2 GHzの4コアCortex-A7プロセッサーとPowerVR SGX544MP2 GPUで構成されるBmorn K12にインストールされます。 RAMの容量は2ギガバイト、永続的-16ギガバイトです。





模擬テストの結果を見てみましょう。

Antutuテストは、ほぼ13,000ポイントを獲得しています。





象限：





Nenamark2グラフィックステストでは、fpsはロールオーバーします。





Epic Citadelテストでは、結果も優れています。





ベラモ：





ファームウェアの最適化が不十分なため、AntutuおよびQuadrantのテスト結果は、Allwinner A31チップの真の能力を反映していません。



ソフトウェア

Bmorn K12では、Android 4.1.1がプリインストールされています。









ゲーム

PowerVR SGX544MPと2GBのRAMを使用すると、重い3Dゲームを実行できます。





勤務時間

タブレットのバッテリー容量は6000mAhです。 これはあまりありません。

Antutu Battery Testerテストでは、タブレットのスコアはわずか549ポイントです。





しかし、50％の明るさでビデオを再生すると、ヘッドフォンからの音、Wifiはオフになり、タブレットは8時間弱作動しました。これは非常に良い結果です。





結論



強力なPowerVR SGX544MP2ビデオアクセラレータ

2GB RAM

Bluetoothキーボードが付属

不十分なファームウェア最適化



一般に、タブレットをAndroidオペレーティングシステムを搭載したラップトップに変換するという素晴らしいアイデアを得ました。Androidオペレーティングシステムの一般的な印象は、未加工のファームウェアによってのみ損なわれます。
