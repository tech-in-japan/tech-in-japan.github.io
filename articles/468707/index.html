JSONpath - query language for JSON

Fedor Sigaev, Technical Director, Postgres Professional



The report will present the recent development of Postgres Professional - a large JSONPath patch that implements the most important functions of the SQL / JSON standard.

The JSON Path query language will be examined in detail: functions, jsonpath data type, working with indexes, application examples are shown.
