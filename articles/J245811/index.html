新しいリリースと機能：3CX Phone System 12.5リリース候補

みなさんこんにちは！



長い間3CXからのニュースはありませんでしたが、これには十分な理由があります-彼らは積極的に新しい3CX Phone Systemビルドを「見ました」。



そして今、先日、リリース候補バージョン12.5がリリースされました。







最新情報をお知らせします。



-3CX WebRTC Gatewayを介して、ブラウザーから直接顧客に電話をかけます 。 現在、「8-800 ...」回線に料金を支払う必要はありません。従業員への自動ルーティングを使用して、ブラウザから直接通話を行うことができます（オーディオとビデオの両方）。

-Microsoft Exchangeカレンダーとの統合 。 現在、3CXはOutlookとの同期を通じてサブスクライバーのステータスを自動的に変更します（「不在」、「別の場所で作業する」など）。

- コールの「ブラインド」転送が失敗した場合の自動コールバック 。 これで、通話を転送したが、サブスクライバーが応答しなかったか、話中だった場合、通話は後で自動的に戻ります

-デスクフォンからCTIを介した3CXPhone for Windowsへの着信転送を確認

-Macデザイン用の新しい3CXPhone ！ Mac用の新しい3CXPhoneはよりモダンなデザインになり、新しいエコーキャンセルモジュールと3CX WebMeetingsを計画する機能が追加されました。

-Htek IP Phoneのサポート| ハンロン 、すなわち：



-インストールが簡単になり、新しいVoIPオペレーターのサポートが追加されました。 詳細- ここ



その他のチップとグッズv12.5：



  • X86システムのサポート
  • Windows用3CXPhoneのビデオ通話。
  • 3CXPhone for Windowsのチャット機能が改善されました。
  • WebRTCまたはWebMeetingを使用したWindows用の3CXPhoneでのビデオチャット。
  • 3CXPhone for Macを使用した3CX WebMeetingsの計画。
  • 3CXPhone WindowsおよびMacの新しいエコーキャンセルモデル。
  • 集中ファームウェアアップグレードをサポートする新しいPolycomテンプレート
  • ODBCソースを介して連絡先を検索する
  • WebRTCを介した通話のチャット
  • VoIPプロバイダーおよびゲートウェイのインバウンドおよびアウトバウンドパラメーターのインターフェースの改善
  • Windows Server Timeが3CX管理コンソールに表示されます
  • 3CX電話システムはMicrosoft Exchange Serverにインストールできます
  • 管理コンソールから3CXPhoneクライアントをリセットおよび再構成する機能。
  • SBCモジュールを介した接続の改善。
  • 古いHTTP / HTTPS HTTP / HTTPSポート（5000/5001）を保存するか、独自のポートを選択する機能


ダウンロードする



ここから 3CX Phone System 12.5RCをダウンロードできます（Mac OSのクライアントはパッケージに含まれています）。

また、 Android （Google Play）およびiOS （AppStore） 向けの3CXPhoneバージョンがあります。
