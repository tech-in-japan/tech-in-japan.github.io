Intelは、Core Xラインから18コアCore i9 Extremeを発表しました

台湾のComputex 2017で、Intel は新しいプロセッサのラインを導入しました 。 このシリーズはCore Xと呼ばれ、ゲーマーやリソースを大量に消費するコンテンツを扱う人々向けの高性能PCチップで構成されています。 ラインナップには、Core i5プロセッサとCore i9プロセッサのより強力なバージョンと、Core i9プロセッサが含まれています。



/写真クリス・イシャーウッド CC



ラインの最年少モデルは、4つのコアと同じ数のスレッドを備えたCore i5-7640Xです（予備コストは242ドル）。



Core i7 Xシリーズでは、コアの数は4または10で、スレッドの数は8または20です。 Core i9 Xシリーズに関しては、12コアと16コア、それぞれ24スレッドと32スレッドのプロセッサーが含まれます。



36スレッドの最初の18コアデスクトッププロセッサCore i9 Extremeもリリースされます （暫定費用1999年）。



Intel Core i9-7980Xと呼ばれ、AMDのRyzenチップに対する一種の応答です。 このチップはSkylake-X CPUファミリに属し、LGA2066マザーボード（X299チップセット上）と互換性があり、新しい改訂されたキャッシュ階層：24.75メガバイトL3と18メガバイトL2を持っています。 Core i9-7980XEのTDPは165ワットです。



新しいSkylake-Xプロセッサは 、プロセッサチップから熱分配カバーに熱を伝達するための材料として、熱伝導率の高い金属はんだの代わりにサーマルグリースを使用します。 したがって、インテルはプロセッサー製造技術を統合することを計画しています。



以前に導入されたHyper-ThreadingおよびTurbo Boost Max 3.0テクノロジーがプロセッサを支援することが知られています。 後者は、ドライバの機能とCPUに保存されている情報を使用して、最速のカーネルに負荷を分散します。 予想どおり、Turbo Boost 3.0の一部として、Skylake-Xの周波数は最大4.5 GHzまで増加します。



Intelについて


Intelは、マイクロプロセッサやチップセットを含む電子デバイスおよびコンピューターコンポーネントのメーカーです。 会社は1968年にロバートノイスとゴードンムーアによって設立されました。 Intelはコンピューター技術の開発に多大な貢献をしました。多くのポート、バス、標準、コマンドシステムの仕様はIntelの参加により開発されました。
