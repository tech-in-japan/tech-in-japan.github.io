ASUS Eee Slateを確認する

タブレットコンピューターが人気を博し始めた頃、最初の顧客レビューは矛盾した考えを表明しました。それまでのタブレットは、それがインストールされるまで本格的な作業ツールではありません... Windowsオペレーティングシステムです。







いくつかの成功したAndroidタブレットコンピューターをリリースしたASUSは、ユーザーの意見に耳を傾け、市場のビジネスセグメントであるEee Slate B121モデルを対象としたWindows 7タブレットを発売しました。 B121モデルは元々EP121としてリリースされましたが、ロシア市場ではビジネス向けデバイスとして位置付けられ、企業市場に焦点を当てています。したがって、ASUSビジネスノートブックでおなじみの「B」インデックスです。 レビューでは、この興味深いデバイスを詳しく見て、通常のコンピューター以外でWindowsを使用する方法を見つけます。



技術仕様

オペレーティングシステム：Windows 7 Professional

ディスプレイ：12.1 "LEDバックライトWXGA（1280x800）

プロセッサー：Intel Dual-Core i5 470um

RAM：DDR3、1 x SO-DIMM、4GB

ストレージ：64GB SSD

ワイヤレスインターフェイス：WLAN 802.11 b / g / n @ 2.4GHz、Bluetooth V3.0

カメラ：2 MPixel

寸法と重量：312 x 207.2 x 16.95 mm、1.16 kg

インターフェース：

デジタルペン、2つのUSB 2.0ポート、オーディオジャック（ヘッドフォン/マイク）、1つのミニHDMIポート、カードリーダー（MMC、SD / SDHC / SDXC）、電源コネクタ。

バッテリー：

通常モードで4.5時間、HD表示モードで2.4時間

付属品：

ユーザーマニュアル、保証書、ソフトウェア付きDVD、電源、ケース（スタンド機能付き）、クリーニングクロス、それらを交換するための予備のペン先5本とピンセット、単4電池2本付きのオプションのBluetoothキーボード。

納入範囲

タブレット用のボックスは、ネットブック用の同様のサイズのボックスとは異なります。 ここでの品質は、ビジネス環境の標準に対応しており、「環境に優しい」黄色の段ボールはありません。 タブレット自体、充電器、指示書に加えて、購入者はケース、スクリーンを清掃するための布、スタイラス（交換可能なチップ付き）を受け取ります。



ケース自体に関するいくつかの言葉：それは大きくて強いように見え、タブレットの重量をさらに増加させるその重量は、材料の信頼性と品質によって正当化されます。 カバーを数か月着用した後、絶望的に拭き取られることを心配しないでください。



設計









外部的には、スレートは「フロント」スクリーンを備えた他の現代のタブレットに似ています（個性を達成するために変更することはほとんど不可能です-それ以外の場合はタブレットではなくなります）







背面には換気用のスロットが見えます。 前部には1つの突出した部分があります-右側にある半円形のボタンです。 機能的には、Enterボタンとして機能し、実行中のアプリケーションを切り替えるのに役立ちます。







タブレットの珍しい機能は、Microsoft製品への参加を許可するステッカーです。











タブレットには、スタイラスコンパートメント用の場所がありました。 手元にカバーがない場合に便利です（それに応じて、スタイラスに適したマウントもあります）。







タブレットの端にあるボタンはすべて金属です。 とてもスタイリッシュに見えます。 タブレットを手に持ちやすくするために、下側と右側にはコネクタとボタンがありません。







製造業者は、上面に、オン/オフ、仮想キーボード呼び出しボタン、および画面の向きをブロックするレバーを配置しました。



タブレットは大きいです。 本当に大きい。 これをプラスまたはマイナスで書くことはできません。そのような寸法は、タブレットで解決するタスクによって決まります。 それらについては後で説明しますが、デザインの観点から見ると、スレートは男性に適した大規模でスタイリッシュなデバイスです。 手の中のタブレットが注目を集めていますが、その場所はオフィスセンターやスタイリッシュな禁欲的な家です。 本を読んで、片手でタブレットを持ちたい地下鉄車を忘れてください。



技術 可能性



ASUSで慣例となっているように、このデバイスは豊富なコネクタとスロットでユーザーを喜ばせます。

2つのUSB 2.0ポート（マウスとキーボードを接続でき、本格的な作業ツールを備えています）、miniHDMI、コンボ3.5 mmジャック（ヘッドフォンとマイク用）、充電器を接続するためのコネクタ。



もちろん、タブレットのストレージを拡張するSD / SDHC / MMC / SDXCカードスロットの余地があります。

ASUS Eee Slateは、Intel Core i5-470UMで動作する世界で唯一のタブレットです。



























プロセッサの速度、4 GBのRAM、SSDドライブの存在により、タブレットはこのクラスで最も生産性の高いデバイスになっています。 ただし、システムには弱点があります-統合グラフィックス。 ただし、プロの「稼働中の」デバイスでゲームを実行したいという欲求は通常発生しません。







なぜそのような機会ですか？ 多くのアプリケーションでシステムを活用できます。 たとえば、Webサーフィン、複雑なExcelスプレッドシートの操作（最後のキーボードでもASUS Transformerでこれを想像するのは困難です）、および一般的にMicrosoft Officeスイートを使用する場合。 パフォーマンスと大きなリソースを必要とするプログラムとの関係により、デバイスの専門的な使用の程度が決まります。 他のWindows 7タブレットでも、パフォーマンスが明らかに平均を下回る「原子」がインストールされています。 吊りテーブルは、ビジネスデバイスから受け取りたいものではありません。 この点で、ASUS Slateは確かに勝ちです。



スクリーン









ASUS Eee Pad Transformerについてのレビューを読んだり、自分でデバイスを見た場合は、最新のASUSタブレットの画面の品質レベルがわかります。 タブレットファミリの最も高価なビジネス製品に、巨大な「トランス」よりも劣ったスクリーンがあるとしたら、奇妙です。 これは起こらなかったので、印象は最もポジティブでした。



画面は容量性であり、特殊なスタイラスで作業するときに押す力を認識するWacomテクノロジーをサポートしています。 WacomはASUS Slateの明らかな利点です。ASUSSlateは、デバイスをアーティスト、デザイナー、プランナーなどのプロフェッショナルツールのカテゴリに変換します。 主観的に、スタイラスを使用することは非常に便利ですが、ASUSタブレットで作業するためにWacomタブレットを使用することに慣れているユーザーのみが技術の有効性を評価できます。



スレートは多くの点で絶対的なチャンピオンです。 を含む、画面の対角線-1280x800の解像度で12.1インチ。



画面は、IPSマトリックスをさらに改善したAFFS（Advanced Fringe Field Switching）の最も高度なテクノロジーの1つを使用します。 AFFSでは、ピクセル間の距離が短くなり、視野が改善されています。 上記の利点に加えて、画面は優れた色再現と高輝度を提供します。 その結果、ディスプレイは太陽の下で「フェード」しません。

ディスプレイは容量性および抵抗性入力をサポートします-指タッチ（およびMultiTouchでは一度に最大10本の指を使用できます）をスタイラスと一緒に使用できます。



ASUSタブレットで保護用の強化ガラスCorning Gorilla Glassを見つけることはすでに一般的です。これは、金属物と接触しているときでも傷を避けるのに役立ちます。 このようなガラスはASUS Transformerにインストールされており、このクラッシュテストビデオでその機能を評価できます。







ソフトウェア









「世界で最も強力なタブレット」の形成と開発の歴史の中で最も議論された瞬間は、常にWindows 7オペレーティングシステムを使用することの不可避性でした。おさまりません。



そのため、タブレットには64ビットバージョンのWindows 7 Professionalが搭載されています。 タブレットのデスクトップバージョンとの違いはありません。 画面にデータを入力するには、仮想キーボードを使用します。仮想キーボードは、画面の左上隅の端を引っ張って呼び出すことができます。 キーボードは、ニーズに応じて簡単に拡張できます。 そして、あなたはそれを完全に削除して手書きに行くことができます。 それにもかかわらず、手書きはまだ実際に実用的な価値がありません。 システムは、粗雑な手書きを十分に認識しないため、高速コピーブックに頼ることはできません。



残念ながら、タブレットはほんの数時間で手に入りました。 この間、タッチコントロールに対するWindowsの反応の特性に慣れるのは困難です。 正直なところ、ASUS Eee Pad TransformerにインストールされたAndroidオペレーティングシステムは、直感的な応答という点ではるかに優れています。 私たちが遭遇した主な頭痛の種はInternet Explorerの使用で、通常のアクション（ページを指でスクロールする）が予期しない結果を引き起こしました（スクロールの代わりに、常にテキストの断片を選択することしかできませんでした）。 このような不快な機能は、悪意のあるバグよりもマイクロソフトのビジョンに関連している可能性が高く、多くの場所でシステムに見られます。 おそらくこれは習慣の問題ですが、常に空きポートを使用し、少なくともマウスを接続したいと考えていました。 幸いなことに、このためのすべてはデバイスにあり、使い慣れたタブレットよりもキーボードのないラップトップである可能性が高くなります。



バッテリー寿命



ラップトップにとっては悪くない、タブレットにとっては十分ではない、と言ってみましょう。 Windows 7とCore i5は明らかにバッテリーをフルに使用します。ブラウザーとwi-fiのみを使用する場合、3時間の中断のない操作に頼ることができます。 コンピュータゲームまたはHDビデオの電源を入れたら、2時間で充電器を接続する準備をします。

ええ、そうです、背面カバーのスロットは混乱を招く可能性があります。まるで、熱気が強力なro音の流れで吹き抜けているようです。 ただし、実際には、要求の厳しいアプリケーションが完全に搭載されていても、タブレットは手で熱くならず、不快感を生じません。

冷却システムの騒音は聞こえますが、完全に迷惑です。 それは完全に静かな従来のタブレットを使用した後にのみ顕著です。



結論



デバイスが手に入る前であっても、ASUS Eee Pad Transformerと比較したかったのです。 しかし、ASUS Slateに慣れるとすぐに、この欲求は消えました。 ASUS Slate B 121は、通常の意味でのタブレットではなく、ラップトップのクラスにより近いものです。 ASUS Slateは、ポンピングタブレットよりもキーボードのないラップトップです。



ASUS Slateは、優れたAFFSマトリックスでほとんどのラップトップよりも優れていますが、わずかではありますが、技術仕様は劣っています。 ほとんどの場合、4 GBを超えるメモリは必要ありませんが、もちろんプロセッサはより新しいものにしたいと考えています。 ASUSはバッテリー寿命も劣りますが、人間工学的には競合他社を上回ります。必要な場合は外部キーボードを接続できる画面しかありません。 Bluetoothを介して実行し、モビリティをさらに強化します。



私たちの意見では、ASUS Slateはウルトラポータブルラップトップの代わりに使用すると効果的です。 また、このデバイスはファッショナブルなものであり、他の数十個のタブレットよりもはるかに強力な外部効果をもたらすことを忘れないでください。



このタブレットの別の使用分野は、純粋にプロフェッショナルです。 Windowsでのみ効果的に解決できるタスクがあります。 そして、生産性の高いWindowsタブレットで作業することが物理的に便利な分野、専門分野があります。それは、ペンを使用してグラフィックアプリケーションを操作することです。 Wacomに慣れているが、より多くの機能が必要な場合は、ASUS Slateをお勧めします。



多くのニッチな用途があるかもしれません。 消費者は、オフィスの仕事から医療まで、デバイスを使用する環境を選択する権利を持っています。 目的地のリスト全体を詳細にリストすることはしませんが、次のニュアンスに注意を向けたいと思います：多数の高性能+モビリティ+必要なソフトウェア-未来。



私たちは主要なノートでレビューを終了します：ASUS Slateは明らかにその開発の初期段階にあります-その能力のすべてがまだ明らかにされていません。 これは、Windows 7によって課された制限によるものです。そのため、Microsoftの次のリリースであるWindows 8は、アーティスト、デザイナー、建築家だけでなく、ネットブックからより高度なデバイスへの切り替えに関心のある一般ユーザーにとっても、このタブレットをより魅力的にすることができます。
