2番目のPwn2Ownモバイルは東京で開催されます：30万ドル以上の賞金プール

Zero Day Initiativeの一環として開催される第2回Pwn2Ownモバイルコンテストは、11月13日と14日に東京で開催されるPacSec Applied Security Conferenceで開催されます。 賞金総額は300,000ドル以上です。







Mobile Pwn2Ownコンテストは、モバイルデバイスの脆弱性とハッキングの機会を探すためのコンテストです。 今年、彼はAndroid Security TeamとBlackBerryの支援を受けています。
