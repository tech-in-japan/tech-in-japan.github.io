Php



  • 無名関数の短い構文 -内部は、短いラムダの可能な実装について積極的に議論しています。 チルダを使用した構文は、以前は投票によって考慮および拒否されていました。 $x ~> $x + 1;



    、および^($x) => $x + $y



    。 現在、次のオプションが検討されています。 



     fn(params) => expr function(params) => expr (params) ==> expr (params) => expr [](params) => expr


  • カーネルにPCSを含める-PHP 7.3のオファーは公式のRFCではまだ発行されていません。 PCSは、PHPの拡張機能をPHPとCに同時に作成できるモジュールです。 PCSの詳細をご覧ください 。 この機能をカーネルに含めると、PHPの多くの拡張機能を書き換えることができます。 これにより、コードが大幅に簡素化され、バグの数が減り、潜在的なメンテナーの数が増えます。
