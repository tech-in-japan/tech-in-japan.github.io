今週、開発チームのほぼ全員がアイスランドのオフィスに集まり、Vivaldiブラウザーのエラーや欠点の修正を迅速化するとともに、新機能の共同作業を行いました。 今日のビルドには、過去数日間に既に行ったことの一部が含まれています。 修正されたバグを含む変更の完全なリストは、記事の最後にありますが、今では何が新しくなったのかがわかります。



マウスジェスチャー



この機能では、いくつかの新しいオプションが追加されました。 特に、Altキーをマウスの右ボタンの代わりとして使用し、使用可能な入力デバイスを使用してジェスチャを実行できるようになりました。 デフォルトでは、このオプションは無効になっています- ツール->設定->マウス->ジェスチャーに移動して有効にできます。



別のオプションは、ユーザーが待ち望んでいたものの1つです。これまで、この機能をロッカージェスチャと呼んでいます。 この機能の本質は簡単です。左右のマウスボタンをすばやく押す（または逆の順序で）ことにより、表示履歴を前後に移動できます。 この機能はデフォルトでは無効になっていますが、 ツール->設定->マウス->ロッカージェスチャで有効にできます。



さらに、ブラウザー拡張機能のポップアップ通知の表示に関するいくつかの重大なエラーを修正し、バックグラウンドで読み込まれたページのインジケーターの最適なオプションを引き続き検索しました。 さらに、クイックコマンドのいくつかの欠点を修正しました。



それだけです 以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac 10.7+
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
