EvernoteとHTC 7モーツァルト

この秋、HTCはWindows Phone 7.5 Mangoオペレーティングシステムを実行するロシアで最初のスマートフォンを発表しました。 これは私たちにとって特に重要な瞬間です。WindowsPhoneのEvernoteアプリケーションは、さまざまなプラットフォームのサービスクライアントファミリーの中で最年少であり、おそらく最も興味深い珍しいものの1つだからです。 さらに、最近では、ついにロシア語のインターフェースを取得しました。



提供されたHTC 7 MozartスマートフォンをHTCに感謝します。この上で、Evernoteの新しいバージョンの動作をテストしました。





デバイスの特性については詳しく説明しません。HTCWebサイトですべての電話パラメーターを確認できます。 基本仕様のみに言及します。 これは、プラスチックインサートを備えたアルミニウムケースに収納されたモノブロックで、解像度800x480ピクセルの3.7インチタッチSLCDスクリーンと、クロック周波数1 GHzのプロセッサを搭載しています。



電話は、キセノンフラッシュとオートフォーカスを備えた8メガピクセルのカメラを受け取りました。 したがって、HTC Mozartは、Windows Phone 7を実行する写真機能モデルの面で最も先進的なものの1つと言えますが、これは重要です。電話で作成されるEvernoteノートの大部分は、ポインター、ドキュメント、ラベル、およびテキストを含むその他のオブジェクトの写真ですから。 したがって、カメラが被写体にすばやく焦点を合わせる能力は非常に重要です。



準備作業



EvernoteはMarketplaceアプリストアからインストールできます。 アクセスするためのアイコンは、デバイスのデスクトップにあります。 ストアにアクセスするには（そして実際にWindows Phoneで完全に動作するには）、Windows Live IDアカウントを作成する必要があります。これは、 サイト上または電話から直接実行できます。 ただし、Live Messengerインスタントメッセージングサービスまたは他のMicrosoft Liveサービスを使用している場合は、すでにアカウントを持っています。



マーケットプレイスで、虫眼鏡アイコンをクリックし、検索バーに「Evernote」と入力し、「検索」ボタンをクリックして、結果のリストで象とおなじみのアイコンを選択し、「インストール」をクリックします。 プログラムに位置決定へのアクセスを許可するように求められます-これを行うことは価値があり、将来、現在の位置の近くで作成されたマップ上のメモを常に簡単に見つけることができます。



クライアントがスマートフォンにインストールされたら、メイン画面にタイルの形でEvernoteをすぐに取り出すことができます。 これを行うには、すべてのアプリケーションのメニューで、アプリケーションアイコンを押し続けます。 表示されるリストで、「デスクトップ上」を選択します。 デスクトップ自体のEvernoteタイルを最初の画面よりも高い位置に上げるには、指で押しながらアイコンを目的の場所にドラッグします。







通常、最初の起動時に、Evernoteアカウントのユーザー名とパスワードの入力（または作成）を求められます。 その後、アプリケーションはすべてのノートを同期します。 その数によっては、これには時間がかかる場合があります。 Wi-Fi接続がある場合は、この手順をお勧めします。







インターフェース



一部の人は、Windows Phone 7のMetroインターフェイスが異常であると感じるかもしれませんが、画面をスクロールするアプローチが本当に気に入りました。 Evernoteの機能はこれに特によく適合します。 同期が終了すると、すべてのメモを含むホーム画面がユーザーの前に表示されます。 各メモについて、タイトルと作成時間、数行のテキストが表示され、その中に画像または音声の録音がある場合、対応するスケッチまたはスピーカーのアイコンも表示されます。







指をスワイプすると、ノートブックの画面、タグ、最近開いたメモにすばやく移動できます。 後者は、たとえば現在作業中のプロジェクトで定期的に同じレコードに戻る人にとって特に便利に思えます。







画面を回転させるプロセスは、速度を落とすことなく、非常にスムーズかつ自然に進みます。 リストを下にスクロールするときに新しいメモをアップロードする場合も同様です。 Windows Phone向けのアプリケーションの作成者は既に、技術ブログで（ ここで翻訳をレイアウトしました ）インターフェースの滑らかさの感覚をぼやけさせないために、スクロール時にわずかな粗さがないことをどのくらい達成したかを伝えました。



特定のメモを開くと、下部に可能なアクションのリストを含むパネルが表示され、電子メールによる編集と送信に加えてピンボタンがあります。 クリックすると、メモはWindows Phoneのメイン画面に個別のタイルとして表示されます。 これは、たとえば購入のリストや収集されたドキュメントを参照するなど、常に同じメモを参照する場合に非常に便利です。 アイテムはデスクトップの最後に追加されますが、それをもっと高くする方法はすでに知っています。







作成および編集



新しいメモの作成は非常に簡単です。 下部のすべての画面には3つのボタンのあるパネルがあり、新しいメモをすばやく作成したり、写真を撮ったり、検索したりできます。



新しいメモを作成する画面は、3つのフィールド（見出し、メモ帳/ラベル選択パネル、およびメインテキストエリア）に便利に分割されています。







複数の写真と音声録音を一度にメモに追加できます（メモの最大サイズを超えない場合、無料ユーザーの場合は25 MB、有料ユーザーの場合は50 MB）。 このような添付ファイルは、ノート作成画面の保存、写真、音声レコーダーのアイコンが含まれる同じ下部パネルから簡単に作成できます。 添付ファイルは既存のメモに配置できます。 これを行うには、リスト内の目的のノートをクリックし、開いた後、編集ボタンをクリックするだけで、すべては新しいノートの作成と同じです。







Evernoteでは、ユーザーの個人の受信メールアドレスに手紙の形でメモを送信することにより、アカウントにメモを追加できることを思い出してください。 たとえば、重要なレターを送信する場合、アドレスをブラインドコピーに入れるか、受信した通信をそのアドレスに転送できます。 また、多くの人は、いつものように自分で覚えておきたいメモを書いた手紙を送る習慣があります。 習慣を変えることなく、Evernoteに直接メールで送信することができます。その後、Evernoteはいつまでもあなたと一緒にいて、どこでも利用できます。



アプリケーションから直接、Evernoteのメールアドレスを電話の連絡先に追加できます。 これを行うには、メイン画面の1つで、アクションバーの省略記号をクリックして[ 設定]を選択し、アドレスの下の[ 連絡先に追加 ]ボタンをクリックします。 既存の連絡先に住所を割り当てるか、新しい連絡先を作成するように求められます。 2番目のオプションでは、アクションバーの[+]ボタンをクリックし、[保存]をクリックして、新しい連絡先に名前を付けます（たとえば、[My Evernote]）。







写真



スナップショットはEvernoteモバイルクライアントでの作業の非常に重要な部分です。Evernoteは、キャプチャした画像を保存し、任意のデバイスからアクセスできるようにするだけでなく、手書きの画像を含むテキストを表示します。 キーボードから入力するのと同じくらい簡単に、キャプチャしたテキストで検索できます。 名刺、割引カード、ラベル、標識、領収書など、かけがえのないもの。



カメラのアイコンをクリックすると、標準の撮影モードに入ります。 画面全体のほとんどがカメラからのビューブロードキャストで占められており、下部には設定とズームイン/ズームアウト用の小さなボタンがあります。 ビデオを撮影するための別のボタンは、結果のクリップが大きすぎてノートに挿入できないため、Evernoteで非アクティブになりますが、HTC Mozartは1280x720の優れたビデオ形式を撮影することを言及する価値があります。







しかし、撮影プロセスに戻ります。 電話には、デバイスの右側の下部にある興味深い2段シャッターボタンがあります。 軽く押すと焦点が固定され、さらに圧力をかけると写真が撮影されます。 ただし、ケースを引っ張って写真を撮るのが怖い場合は、画面上のどこでも指でクリックするだけです。 撮影後、アプリケーションに結果が表示され、それを受け入れるか、再撮影を試みることができます。



撮影する前に、フラッシュモード（強制、自動または無効）に加えて、将来のフレームのサイズ（最大3264 x 2448ピクセル-写真のボリュームは約2〜3メガバイト）、測光モード（中央、積分またはスポット）を選択できる設定を詳しく調べることができます、光源のフリッカー周波数（50または60 Hz）。 そこでは、事前に設定された一連のパラメータを使用して、撮影モードを現在の状態に変更できます。 これを行うには、「シーン」設定セクションに移動します。 利用可能なオプションの中には、スポーツ、ビーチ、ライト、キャンドルライト、クローズアップがあります。 また、カメラアプリケーションは、標準的な効果の選択を提供します：グレーの陰影、ネガ、セピア、およびソラリウム。 デバイスの「戻る」ボタンを押すと、設定から撮影に戻ることができます。



照明が不十分で、フラッシュがない場合（テキストで領域を照らすことがあります。これにより、その後の認識が妨げられます）、写真は非常に鮮明で、Evernoteサーバーは表示されたテキストを簡単にインデックス付けできます。







ところで、画像の回転のようなアプリケーションでのこのような機会を思い出す価値があります。 Evernote for Windows Phoneで写真が添付されたメモを作成または編集する場合は、目的の画像をクリックするだけです。 編集画面が開き、画像を時計回りまたは反時計回りに回転したり、メモから削除したりできます。 したがって、名刺を突然「逆さまに」外した場合、これは問題なく修正できます。 同じモードから、アクションバーの省略記号をクリックして対応する項目を選択することにより、Windows Phoneの画像に写真を保存できます。







撮影後、ノート作成画面に戻り、ノートを保存したり、ノートに何かを追加または追加することができます。



同期後（自動的に行われますが、メイン画面のいずれかのアクションパネルで省略記号をクリックしてから[同期]項目を選択して手動で開始することもできます）、ノートはすべてのEvernoteデバイスで利用でき、それに含まれる画像はテキスト認識サーバーに送信されます。 認識が完了すると（プレミアムユーザーの場合、これは数秒で発生し、無料アカウントの所有者は少し待たなければならない場合があります）、テキスト付きの写真をEvernoteアカウントで検索できるようになります。



音声



Evernoteモバイルアプリケーションの次の人気機能は、もちろんボイスレコーダーです。 録音ボタンを押すと、対応する画面が開き、録音されたサウンドの音量、タイマー、停止ボタンが表示されます。







完了すると、ノート画面に戻り、追加されたレコードがスピーカーアイコンとその隣のノートのサイズとして表示されます。 クリックすると、ファイルをすぐに聞いて、必要に応じて削除できます。 表示されるサイズは、ノートのサイズを制御するのに役立ちます。 GPRS / 3Gを介して接続している場合、大きなサウンドファイルを含むノートを使用したデータ転送には時間がかかり、予期しない費用が発生する可能性があります。







ディクテーション後、編集を続行したり、メモを保存したり、別のオーディオファイルを追加したりできます。



-一般に、電話を選択してWindows Phoneを見るプロセスを行っている場合、HTC Mozartはミッドレンジのスマートフォンの中で価値のある選択肢のようです。 少なくとも、私が将来のために残しておきたいすべてのものをEvernoteに暗記することで、彼はこれに必要なものすべてを持って非常にうまく対処しています。



Windows Phone向けのアプリケーションは、モバイルプラットフォーム向けの最高のEvernoteソリューションの1つであると考えています。 すでにこのスマートフォンを所有していて、Evernoteをまだインストールしていない場合は、この省略を早急に修正する必要があることを確認してください。
