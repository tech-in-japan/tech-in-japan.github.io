3月の興味深い：モスクワのDevOpsDays

年末年始は終わりました。CESで可能性と不可能性のすべてを発表し、アトラシアンはTrelloを買収し、すべての主要なスマートフォンメーカーがWestWorldのクラムシェルの特許を取得しました。 そして、シャーロックでさえリークされ、見られ、議論されました。 みんなが目を覚まし、働き始め、春の会議を開催する力を見つけた人もいました。 カットの下で、 DevOpsDaysについて少しお話したいと思います。これは、6年で主要な国際パーティーに発展し、2017年3月11日にモスクワで初めてゲストを集めるカンファレンスです。



1.引き続きレポートを送信できます



会議の1か月半前。 また、プログラムにはいくつかのスロットがあります。 5分間の電撃戦レポートで事前登録できる、私のお気に入りの照明を含みます。 練習する良い方法は、テクノロジーを学ぶために+10、メモリに+5を取得することです。 主催者はこのプログラムを真剣に受け止めています。広告はなく、ユニークなレポートのみを約束します。



2.世界中のスピーカー



会議で最も価値のあることは、レポートの内容ではなく、スピーカーです。 あなたの質問からステージから実行する場所はありません。 そして、彼ら自身が傍観者にコーヒーを飲みに来て、ステージから言えない何かについて議論しました。 長年にわたる大規模で進化している会議は、業界の最高の専門家を獲得するための非常に良い方法です。 そして、個人的な経験について注意深く彼らに尋ねてください。 価値は、本当に経験豊富な専門家が良い記事を書くことはめったにないということです。 何かすることがあるからです。 しかし、彼らは1つのレポートを準備し、その年に彼と話し、同僚とコミュニケーションをとる準備ができています。



DevOpsは、2つの並行するトラックで、外国人とロシア人の両方の専門家による最高のレポートを取り上げます。



スピーカーの中で：



アントン・ワイス







Otomatoの技術コンサルティングの共同所有者。 彼は、ハイテクで15年以上の経験があります。 技術教育の専門家。 イスラエル初のDevOp認定コースの開始者および共著者。 多くの国際会議の参加者、経験豊富なスピーカー。 ほとんどの場合、彼はプログラムの開発に携わっていますが、同時に社会学の学位とMBAの卒業証書を持っているため、開発の社会的側面を専門的に研究することができます。


レオン・フェイアー







Leonの20年の経験は、複数のトラフィックの増加に耐えることができる高度なWebベースのシステムの設計と運用に焦点を合わせてきました。 長年にわたり、彼は、National Geographic、WhiteHouse.govなど、世界で最も訪問されたサイトのいくつかをサポートするシステムを設計および構築するユニークな機会がありました。 彼は同僚の間で専門的な皮肉屋と見なされており、実際には、少なくとも100万人のユーザーにとっては、機能するまで何も機能しているとは考えられません。


マイケル・ヒュッターマン





Javaチャンピオン、SCJA、SCJP、SCJD、SCWCD、フリーランスの開発者、建築家、トレーナー。 Java / JEE、ALM / SCM、アジャイル開発方法論を教えます。 彼は国際会議で講演し、自分自身の組織に参加しています。 たとえば、ツールセクション「アジャイル2009」。 Java User Group Cologneに従事し、java.net JUGコミュニティを率いています。 JetBrains Academy、FEST、およびJAVA自体の開発に参加しています。 本と開発記事を書く ： huettermann.net


3.タランツールとhh.ru



そしてもちろん、モスクワでの会議には、テクノロジーの巨人のための講演者がいます。 Tarantoolのインフラストラクチャコンポーネント、高負荷下でのHeadHunterの生存、および直接参加者からのその他の多くのストーリー。 参加者はマスタークラス、コンテスト、賞品を待っています。 今すぐ参加しよう！
