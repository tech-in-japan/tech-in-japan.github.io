TechEd Europe 2013：今日が未来です

プレリリースバージョンのWindows Server 2012 R2、System Center 2012 R2、およびSQL Server 2014が利用可能になり、Windows Azureとともに、これらの製品はMicrosoft Cloud OSアプリケーションのコンセプトおよび統合プラットフォームの重要な要素となります。



マイクロソフトは、 Windows Server 2012 R2サーバーオペレーティングシステム、 System Center 2012 R2管理ツール、 SQL Server 2014情報管理プラットフォームなどの新製品のプレビューを発表しました。 これは、Microsoftのサーバーおよびツール担当副社長であるBrad Andersonによって発表され、毎年開催されるTechEd Europeカンファレンスの参加者と話しました。





「Windows Azureと共に、新しいソリューションは統合エンタープライズアプリケーションプラットフォームの不可欠な要素であり、Microsoft Cloud OSの概念を反映しています。 クラウドOSは、サーバーインフラストラクチャを非常に弾力性のあるスケーラブルな情報環境に変換し、あらゆるデバイスからいつでも同等に効率的に動作する柔軟で最新のアプリケーションを迅速に作成し、それらを管理して、あらゆる種類のデータの可能性を明らかにするのに役立ちます これにより、使用するデバイスに関係なく、ユーザーの生産性を常に向上させることができます。 現在、真に動的な「クライアントツークラウド」情報環境の構築を支援しているのはマイクロソフトだけです」と、マイクロソフトのサーバーおよびツール担当副社長であるブラッドアンダーソンは述べています。



Windows Server 2012 R2



新しいMicrosoftサーバーオペレーティングシステムにより、ローカルインフラストラクチャの「クラウド」モデルの使用にすばやく簡単に切り替えて、パブリックWindows Azureクラウドまたはサービスプロバイダーのクラウドと組み合わせることができ、同時にIT開発への投資を節約できます。 Windows Server 2012で提供される追加および改善されたテクノロジーと機能に加えて、次のものが含まれます。



•

最新の仮想化プラットフォーム：共有アーキテクチャを備えた動的なプラットフォームにより、従来の仮想化を超えて、サービスの提供に最大限の柔軟性を提供できます。



•

最大320の仮想プロセッサ、仮想マシンごとに4 TBのメモリをサポートすることによる高いスケーラビリティ。 ホストあたり1,024個のアクティブな仮想マシン、クラスターあたり最大8,000個の仮想マシン、ボリュームを動的に変更する機能を備えた64 TBの仮想ディスク。



•

リアルタイム移行中にVHD / VHDXファイルを圧縮し、RDMAテクノロジーを使用してこの移行を実装する機能を含む仮想マシンのモビリティ。VMの移動速度を大幅に向上させます。 また、新しいWindows Server 2012 R2は、共有リソース、複数の移行、仮想マシンストレージの移行、パブリッククラウドへの移行、ネットワーク仮想化（SDN）およびそれらの分離なしの移行テクノロジを改善します。



•

VM間でメモリを自動的に再配分するために使用されるダイナミックメモリテクノロジーは、Linuxベースのゲスト環境を完全にサポートするようになりました。



•

シンプルな管理と費用対効果：新しいWindows Server 2012 R2は、たとえば、サーバーアプリケーションやSQLやHyper- V、従来のストレージシステムに基づいています。 新しいWindows AzureテクノロジーHyper-V Recovery Managerは、Hyper-V-Replicaの機能を拡張し、アプリケーションのアップタイムを保証します。 新しいAPIとインターフェイスのセットにより、単一のパネルからさまざまなストレージシステムを集中管理できます。



•

あらゆるプラットフォームのあらゆるアプリケーション：Windows Server 2012 R2は、ローカルインフラストラクチャとWindows Azureのパブリッククラウドの両方で柔軟性、スケーラビリティ、アプリケーションの互換性を提供し、これらのアプリケーションの相互作用を保証するために必要なツールも提供します。 Webプラットフォームとして、新しいサーバーOSはサイトの密度を高め、マルチユーザー環境を効率的に作成し、それらを追跡および管理できるようにします。 同時に、互換性が確保され、HTML 5などのオープンスタンダードやオープンソースアプリケーションがサポートされます。



•

モダンなワークスタイル：Windows Server 2012 R2は、ユーザーがどこからでも、どのデバイスからでも企業データおよびアプリケーションにアクセスできるようにするために必要な柔軟性を提供します。 彼は、デバイス登録（職場参加）の新しい概念を提供します：証明書がユーザーのデバイスにインストールされます。これにより、ドメインに入らずにActive Directoryに登録することができ、ユーザーの権利に従って企業情報への必要なアクセスが提供されます。 Windows Server 2012 R2の別の新機能は、自動VPN接続を使用することにより、個人用デバイスから企業ネットワークへのアクセスを簡素化します。



「Windows Server 2012は、データセンターにITインフラストラクチャを構築するための、最新かつ非常に要求の厳しいオペレーティングシステムです。 Windows Server 2012は、お客様のITサービスでの使用に最も便利です。 それぞれが特定の規模の組織に焦点を合わせた4つのエディションと幅広い機能により、必要なサービスを迅速にアクティブ化し、ビジネスアプリケーションの機能のためのスケーラブルで信頼性の高いコアを構築できます。 新しいバージョンのWindows Server 2012R2から、さらに高い信頼性と機能性が期待されます」と、CROCのシステムソリューションディレクターであるIgor Malyshev氏は述べています。



System Center 2012 R2



System Center 2012 R2は、Microsoft Cloud OSの包括的な管理を提供します。 System Center 2012テクノロジーの開発と改善の継続として、インフラストラクチャ管理製品の新しいMicrosoftポートフォリオは、Windows Server 2012 R2のすべての機能の完全な管理を提供します。 それらの中で：VMのコピーの起動。 VDMXとデータストレージボリュームの動的なサイズ変更 マルチユーザーVPNゲートウェイの準備と管理。これにより、データセンターの容量を迅速に拡張できます。 Linux環境を含む異種データセンターの管理。 System Center 2012 R2は、ハイブリッドインフラストラクチャを管理するための重要なツールであり、たとえば、ローカルインフラストラクチャからWindows Azure仮想マシンへ 、またはその逆へのワークロードのライブマイグレーションを提供します。



System Center Configuration Manager構成ソリューションは、Windows Intuneクラウドサービスと共に、さまざまなプラットフォーム（PC、ラップトップ、タブレット）を管理するための単一の環境を形成し、BYODコンセプトの要件を満たします。



「当社のMicrosoft Windows Server 2012およびSystem Center 2012シリーズの新製品に関する知識は、プレビュー段階から始まりました。 そのため、注文が可能になるとすぐに、インフラストラクチャソリューションの基盤となりました。 Hyper-V 3.0のネットワークサブシステムとストレージサブシステムの革新により、以前はVMWareの代替手段がなかったプロジェクトでMicrosoft仮想化を使用できるようになりました。 Integral Siberia Group of CompaniesのテクニカルディレクターであるSergey Yushkov氏は、次のように述べています。データセンターのコンピューティングパワーの向上に節約したお金を振り向けることで、サードパーティソフトウェアの追加コストを大幅に削減することができました。



SQL Server 2014



他のMicrosoft Cloud OS製品と同様に、Microsoftのデータおよびアプリケーション管理プラットフォームSQL 2014の次のバージョンは、 Cloud of Spotlight戦略に完全に準拠して作成されています。 情報処理、クラウドおよびハイブリッド情報環境での使用、およびデータ分析のための根本的に新しい機会を提供します。



SQL Server 2014の主要な革新の1つは、新しいHekatonの組み込み機能です。 RAMでトランザクションを処理する場合、ミッションクリティカルなアプリケーションのパフォーマンス、可用性、および管理性が質的に異なるため、トランザクション処理速度が50倍以上向上し、機器リソースの使用効率が向上します。 SQL 2014は、ビジネスインテリジェンスの進歩的な改善とともに、ミッションクリティカルなビジネスアプリケーションの管理と、ビッグデータを含む情報とデータのより深い分析の新しい可能性を切り開きます。



「新しいMicrosoft製品の初期の知人プログラムの参加者として、BaltikaとMonolitはMicrosoft SQL Server 2014の予備バージョンを積極的にテストしています。新しいバージョンでは、AlwaysOnレプリカの数を4から8に増やすことが非常に重要です。 Monolit-Info CJSCのチーフエンジニアであるIlya Sivov氏は、Azure Storageで、アプリケーションで新しいHekatonインメモリトランザクション処理テクノロジーを使用する利点を分析しています。



SQL Server 2014は、Windows Azureパブリッククラウド用の新しい災害復旧およびバックアップ機能も提供します。



MicrosoftクラウドOS戦略とプラットフォームの詳細については、 www.microsoft.com / en-us / server- cloud / windows-server / 2012-default.aspxにアクセスしてください。 製品の予備バージョンは、Microsoft Webサイト（ Windows Server 2012 R2 、 System Center 2012 R2 、 SQL Server 2014）からダウンロードできます 。
