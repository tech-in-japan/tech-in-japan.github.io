Nokia Data Gathering：携帯電話で干ばつと戦う

ノキアマップのような携帯電話やユーザーサービスの生産だけでなく、社会的に有用な技術の開発にも携わっていることを知っている人はほとんどいません。



多くの慈善団体および非政府組織は、さまざまな集団からのさまざまな情報の収集と分析に絶えず頼っています。 アンケートプロセスを促進するために、インタビュアーの携帯電話を使用してさまざまな調査を行い、さらに処理するために超高速でデータを送信できる無料のNokia Data Gatheringサービスを開発しました。







この技術は次のように機能します。 組織は、Nokia Data Gatheringサーバーで必要なプロファイルを作成します。その後、すでに完成したプロファイルがインタビュアーの携帯電話に送信されます。 その後、インタビュアーは人々にインタビューし、受け取ったアンケートに記入します。記入後、彼らは自動的にネットワーク経由でサーバーに戻り、すべてのアンケートからの情報が蓄積、分析、マップに添付され、通常の「オフィス」形式にエクスポートされます。





プロファイル作成の例



したがって、遠隔地であっても情報の収集がはるかに高速になり、調査を実施するために、ラップトップ、タブレット、さらには移動性のない紙のアンケートを使用する必要がなくなりました-統計を計算するために分析センターに返す必要があります。 Nokia Data Gatheringには、紙に比べて他の利点があります。情報の収集は、雇用された従業員の手ではなく、サーバーに置かれ、大量のアンケート（余分なお金と時間がある）を再集計し、さらに、より「グリーン」な収集方法です紙に損傷を与えることなく情報を提供-必要なのは、Series 40またはWindows Phoneベースの電話だけです。 ラップトップは、バッテリー寿命、移動性、多くの場合、調査を地理的な場所に結び付けることができないという点でもNDGを失います。これは多くの場合重要です。







現在、Nokia Data Gatheringは、WWF、国連食糧農業機関（FAO）、World Vision、Health Vigilance Foundationなど、200を超える非政府および慈善団体で使用されています。 これらの組織の多くは不明ですが、さまざまな病気や自然災害との闘いに従事しており、通常の生活条件で助けを必要とする人々に提供しています。 この活動はさまざまなデータの収集なしでは不可能であり、 Nokia Data Gatheringテクノロジーがこれらの組織の善行に役立つことを嬉しく思います。
