Vistaのタグが嫌い

Windows Vistaの「革新」の中には、Macから馬鹿げた検索があります。

これがポピーでどのように行われるかはわかりません（すでにhabrahabrの1人の女の子にこれについての投稿を依頼しました）が、Vistaでは名前とコンテンツだけでなくメタデータも検索されます。

スティーブバルマーと別のMicrosoft Vista狂信者は、目を膨らませ、さまざまな場所（エクスプローラー、ファイル保存ダイアログ、画像ビューアー）からファイルにタグを追加する魔法の能力を称賛し、喜んでいます... しかし、これを何度も読むと、おそらく疑問は-ええと...しかし、すべてのファイルになりますか？

したがって、答えはもちろん、NOです。 全員ではありません。 そして、ここで、すみません、ロバが始まります。 ここで、これらのタイプにラベルを追加できます。
  • Microsoft Office Word、Excel、PowerPoint、およびAccessファイル。
  • Windows Mediaオーディオ、Windows Mediaビデオ。
  • TIFF、JPEG。
  • MP3
  • XPS-Microsoft PDF。
  • MSIインストールファイル。


他人に-それは不可能です。 ご存知のように、新しいOSのクールなチップの欠陥を見つけるために、私は何の関係もないように思えるかもしれませんが、考えてみてください。

そこで私はたくさんの面白いビデオを集め、それらに「ユーモア」ラベルを付けることにしました。 いいですね。 それから私はスタートを押し、「ユーモア」を入力し、このラベルですべてをマークしました...図！ 結局のところ、私はwmvビデオ（私は自分自身に衝撃を与えました！）だけでなく、たとえばタグを追加できないaviも持っています。

JPGは可能ですが、PNGはできません！

MP3には可能ですが、WAVにはありません...



これらのラベルはファイル自体に保存され、すべての形式で許可されているわけではないことを理解しています。 しかし、私はそれを気にしません。 これを使用することは単に不可能なので、いろいろ試してください。

Vistaには、これらの機能のプレゼンテーションで最適に機能する多くの機能がありますが、深刻な作業には適していません。 そして、これはそれらの1つです。 Vistaではタグが嫌いです。
