MODXがSQL Serverを完全にサポートするRevolution 2.1をリリース

MODX開発チームは素晴らしい仕事をして、Revolutionの次のメジャーバージョンをリリースしました。 MODX Revolution 2.1が正式にリリースされ、Microsoft SQL Serverのネイティブサポートが含まれています。



「これからは、Microsoft SQL Serverを含むMicrosoftテクノロジースタックでMODX Revolutionを完全に実行できます。 これは、商用サーバーとライセンスソフトウェアに基づいて構築された環境でソリューションを販売できるMODXソリューションパートナーのパートナーに大きな機会を提供します。 「マイクロソフトの市場は、当社のソフトウェアからも重要なメリットを享受しています。柔軟なWebベースのコンテンツ管理プラットフォーム、完全な創造の自由、バージョン間の信頼性の高い更新サイクルです。 マイクロソフトプラットフォームで運用している企業は、商用サポートおよびSLAプログラムによるサポートに依存することで、これらすべてを実現できます。



IDCのレポートによると、Microsoft Windowsは2010年第1四半期に48.9％のシェアでプレインストールサーバーの市場をリードしています。 Microsoftプラットフォームに投資する企業は、既存のインフラストラクチャとサーバーを使用し、既存の専門家のスキル、経験、ツールを使用して、最も柔軟なオープンソースCMSプラットフォームを同時に活用できます。



マイクロソフトのオープンソースコミュニティのチーフディレクター、ジャヌゴラベリノは次のように述べています。



「PHPとMicrosoftの開発者間の緊密な作業の火花が、SQL ServerとMicrosoftテクノロジースタックの完全なサポートを提供するリリースに成長したことを見て、私たちは絶対に満足しています。 私たちにとって、このリリースは、テクノロジーの相互運用性の向上という目標に向けた私たちの旅の別の印です。 開発者とビジネスの選択は、最新のSQL Server JumpInイベントの主な目標でした！ キャンプは11月に開催されました。 そして、このMODXリリースで目標が達成されたことを非常に嬉しく思います。 「この目標を達成することができたMODXチームの貢献とコラボレーションに感謝します。」



さらに、MODX Revolution 2.1では、キャッシュサブシステムの改善と最適化により、サイトのパフォーマンスが向上しています。 これらの変更は、ページの読み込み速度の向上に定性的に影響を与え、サーバーの負荷を軽減し、複数のサーバーおよびデータクラスターに拡張するためのオプションを提供します。



MODX Revolution 2.1で導入された重要な変更は、レガシーコードの数千行をクリーンアップすることで、システムのサイズを35％削減しました。 この点で、Webサイト管理チームは、Revolution 2.1にサイトの更新を展開する前に、既存のWebソリューションをテストすることが重要です。 サイトの確認にサポートが必要な組織の場合、MODXは、MODX Solution Partner http://modx.com/partners/solutionのパートナーと協力することをお勧めします。
