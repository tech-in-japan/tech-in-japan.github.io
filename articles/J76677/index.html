HandbrakeとMetaXを使用してライブラリを作成する

このレビューでは、独自のメディアライブラリを作成する方法に焦点を当てます。 Handbrake 、 MetaX 、 Apple iTunesなどのプログラムを使用したビデオ変換およびメディアライブラリの設計プロセスを検討します。



例では、 DVD-RipムービーをMP4形式に変換して、 Apple TV 、 iPhone 、 iPod Touchなどのデバイスで表示するプロセスを示します。 また、 MP4形式のファイルとその派生物にタグを追加するように設計された無料のMetaXアプリケーションも検討されます。





自宅では、 Mac OS X 10.6.2がインストールされたApple MacBook 13 "を使用しています。しかし、多くのWindowsまたはLinuxユーザーにとって、 Handbrakeは無料のオープンソースアプリケーションであり、そのようなオペレーティングシステムで利用できるため、レビューも役立つ場合がありますMac OS X 、 Windows XP / Vista / Windows 7 、 Ubuntu 、 Fedora MetaXなどのシステムもWindowsおよびMac OS Xユーザーが利用でき、 LinuxユーザーはEasyTAGを使用できます。



最も重要なことから始めましょう。つまり、 Handbrakeを使用したビデオ変換プロセスです。これは、MPEGビデオ（DVDビデオを含む）をMP4またはMKV形式のMPEG-4ビデオファイルに変換するように設計された無料のアプリケーションです。



入力形式：

  • ビデオ ：DVDに似たソース：VIDEO_TSフォルダー、DVDイメージ、または実際のDVD（エンコード済みまたは非エンコード）、PALまたはNTSC。
  • オーディオ ：AC-3、LPCMまたはMPEGオーディオ。




出力形式：

  • ファイル形式：MP4およびMKV
  • ビデオ ：MPEG-4、H.264、またはTheora（1パスまたは2パスエンコーディング）
  • オーディオ ：AAC、MP3、VorbisまたはAC-3およびDTSパススルー（複数トラックのエンコードのサポート）




追加機能：

  • チャプターチョイス（DVDチャプター）
  • 字幕サポート
  • 組み込みのビットレート計算機
  • 画像のサイズを変更する
  • 白黒コーディング




プログラムの現在のバージョンは0.9.4です。

  • Mac OS（10.5 / 10.6） ： 32ビット 、 64ビット 、 Power PC
  • Windows ： XP / Vista / Windows 7
  • Linux ： Ubuntu 9.10（32ビット） 、 Ubuntu 9.10（64ビット） 、 Fedora 12（32ビット） 、 Fedora 12（64ビット）




プログラムの公式ウェブサイト： www.handbrake.fr



ハンドブレーキのセットアップ


[ 全般 ]タブで、[ MP4にiPod / iTunesフレンドリー（.m4v）ファイル拡張子を使用する ]オプションを選択し、[ 音声 ]タブで音声の言語を指定することをお勧めします。 デフォルトでは、指定された言語は、存在する場合、言語リストの最初になります。







MetaXを使用してMP4ムービーにタグを追加する場合は、[ 全般 ]タブで[ ファイルをMetaXに送信する]オプションを選択することをお勧めします。 したがって、変換が成功した後、映画、タグなどの説明を追加できます。 あなたの映画に。





コーディングソースの選択


エンコードソースを選択するには、[ ソース ]ボタンをクリックして、DVD-ROMまたはファイルを選択します。 この例では、ファイル形式MKVを選択しました。 プログラムがディスク/ファイルの構造を分析した後、追加オプションを指定することができます。 エンコード設定の詳細に進みたくない人のために、多くの既製のテンプレート（ プリセット ）があります。 たとえば、 iPhoneまたはiPod Touchの場合、 Appleプリセット-> iPhone＆iPod Touchを選択できます。 ただし、設定をセットアップし、将来の使用のために宛先として保存することをお勧めします。



ビデオ設定：




  • ビデオコーデック ：H.264（x264）
  • 平均ビットレート（kbps） ：900


900 kbpsのビットレート値により、iPhone / iPod Touchだけでなく、Apple TVやMacBook 13 "でも映画を快適に視聴できます。映画をポータブルデバイスでのみ視聴する場合は、ディスクスペースを節約するためにビットレートを600 kbpsに設定することをお勧めします。

  • 2パスエンコーディング ：2 パスエンコーディングオプションを選択することをお勧めします




画面解像度の設定：


解像度設定を変更するには、[ ウィンドウ ]メニューの[画像設定 （⌘⇧P） ]項目を選択します。 表示されるウィンドウで、必要な寸法を指定できます。 アナモルフィックビデオを作成する必要がある場合は、 アナモルフィックパラメータをLooseに設定する必要があります。 iPhone / iPod TouchだけでなくApple TVで映画を視聴する場合は、幅を640ピクセルに設定することをお勧めします。 それ以外の場合は、幅を480ピクセルに設定できます。これにより、ディスク容量も節約されます。







[ プレビュー ]ボタンをクリックすると、画面の幅に正しい値を選択し、黒いバーがないことを確認できます。 この例では、 AnamorphicパラメーターをNoneに設定し、 Aspect Ratio Keepオプションを選択し、幅を640ピクセルに設定します。



サウンド設定：


[ オーディオ ]タブで、最終ファイルに追加するオーディオトラックを指定する必要があります。 ロシアのトラックに加えて、私は映画が英語のオリジナルのトラックを含むことを好みます。 オリジナルの映画を見ることも英語の学習に貢献します。

  • オーディオコーデック ：AAC（facc）
  • ミックスダウン ：ステレオ
  • サンプルレート（khz） ：自動
  • ビットレート（kbps） ：160






映画の場合、 サンプル レートとビットレートをそれぞれ48 khzと160 kbpsに設定することをお勧めします。 テレビ番組の場合は、ディスク容量を節約するために値を44 khzおよび96 kbpsに設定する価値があります。



チャプター設定（チャプター）：


通常、DVDムービーはチャプターに分割されます。 必要に応じて、これらの章を最終的なフィルムに保存できます。 これを行うには、[ チャプタ]ウィンドウで[ チャプタマーカーの作成]を選択し、[ チャプタ タイトル]セクションの各チャプタの横にあるチャプタの正しいタイトルを指定します。







字幕：


プログラムの最新バージョンでは、 SRTファイルを使用して字幕を追加することが可能になりました。 以前のバージョンではこのオプションを本当に見逃していたので、 iSubtitleプログラムを追加で使用する必要がありました 。 たとえば、SRT形式の字幕のコレクションを含む多くのサイトがあります。

  • http://subs.com.ru
  • http://ru.movi​​esubtitles.org
  • http://www.vsesubtitri.ru




SRTファイルをディスクにダウンロードし、[ 外部SRTの追加... ]ボタンをクリックして、変換可能なムービーに追加できます。







それで、基本的な設定は完了し、エンコードプロセスを開始できます。 Handbrakeプログラムを使用すると、複数のムービーを一度にエンコードできます 。 通常、私はこの機能を使用して、一晩で数本の映画をコーディングします。 キューにコーディングを追加するには、すべての設定を指定した後、[ キューに追加 ]ボタンをクリックします。 [ キューの表示]ボタンをクリックして、キューのエンコードプロセスを表示および開始できます。







変換設定の保存：


選択した設定は、[ Toogleプリセット ]ボタンをクリックして保存できます。 現在のテンプレートを含む追加のブロックが、右側のメインプログラムウィンドウに追加されます。 ブロックの下部にある[ + ]ボタンをクリックします。 ウィンドウが表示され、テンプレートの名前と説明、および解像度サイズを保存するオプションを指定できます。







変換プロセス：


メインウィンドウとShow Queueウィンドウの両方、またはMac OS X Dock自体で変換プロセスを監視できます 。







キュー全体の変換が完了すると、ダイアログボックスが表示されます。







プログラム設定で[ファイルをMetaXに送信]オプションを設定している場合、 MetaXプログラムが自動的に開きます。



MetaXは、 MP4ファイルにタグを追加するための無料のアプリケーションです。 外国映画のメタデータは、 Amazon.comカタログから自動的にダウンロードできます。 その他の場合は、タグを自分で編集する必要があります。



現在のMetaXバージョン：

  • 2.4.6 Mac OS X Leopardの場合
  • 2.3.0 for Mac OS X Tiger
  • 2.0 for Windows




MetaXでファイルを開く最初の段階で、ビデオファイルの種類（ TVショー 、 ミュージックビデオ、またはムービー）を選択するウィンドウが表示されます。







この場合、通常のムービーがあるため、[ ムービー ]オプションを選択します。 MetaXは、必要なすべてのタグをネットワークから自動的に取得しようとします。 ただし、映画について必要なすべての情報を自分で追加する方が適切です。



ライブラリの場合、次のフィールドに入力します。

  • タイトル -ロシア語の映画の名前と映画の元の名前
  • アーティスト（表示） -アーティストのリスト
  • 日付 -年
  • ジャンル -映画のジャンル
  • 簡単な説明 -映画の簡単な説明


また、必ずフィルムのカバーを追加してください。 これは、ムービー情報が入力されたMetaXウィンドウの概観です。







この情報は、 キノポイスクやIMDbなどのサイトを使用して取得できます。



MetaXはキューもサポートしているため、一度に複数のフィルムに説明をすぐに追加できます。



必要な情報をすべて追加したら、[ 書き込みと共有 ]ボタンをクリックする必要があります。 必要なタグをキュー内のすべてのフィルムに正常に追加すると、終了に関するウィンドウが表示されます。







その後、たとえば2009-Moon.m4vのようにファイル名を変更できます。 このシステムでは、快適で美しい外観になります。







この映画をライブラリに追加するだけです。 ライブラリとしてiTunes 9.0.2を使用しているため、ライブラリは次のようになります。







その後、映画をiPhone / iPod TouchにApple TVにアップロードしたり、ラップトップから視聴したりできます。
