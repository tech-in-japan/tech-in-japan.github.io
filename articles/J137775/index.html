Java Day SPB 2012でのJelastic

JelasticチームのメンバーであるDmitry LazarenkoとMarina Rightが、2012年2月10日に開催されるJava Day SPB 2012開発者カンファレンスに参加することをお知らせします。 非常に興味深いレポートを用意しました：「フィールドにいるだけでは戦士ではありません。GlassFish3.1.1クラスターの構築方法」。 レポートは、ホリデイインモスコフスキーヴォロタホテル（サンクトペテルブルク、モスコフスキープロスペクト、97a）の「ステンバーグ」ホールで17.45に開催されます。 耐障害性のあるGlassFishサーバークラスターを構築する方法を学びます。 ロードバランサーへのサーバーの登録、スティッキーセッションの編成、サーバー間のセッションの複製、クラスターの管理、クラスター上のアプリケーションの更新、この構成のサーバーによる追加のリソース消費などの問題が考慮されます。 そしてもちろん、すべてがJelasticでどのように機能するかがわかります。

あなたは私たちを個人的に知り、すべての質問について話し合う機会があります。 また、興味深いことをお伝えします。 お見逃しなく！

写真は、会議の他の参加者やゲストの中からすばやく見つけるのに役立ちます。 じゃあね！
