Google Chrome 42無効化されたUnity Web PlayerおよびJavaサポート

プロローグ



今日、Javaアプレットのサイトにある私の製品のWebインターフェースで、「プラグインはサポートされていません」という碑文が付いたいスタブを見つけました。 まあ、あなたは決して知らないでしょう、多分Javaは時代遅れです、私は行ってアップデートします。



私はjava.comに行き、アップデートをダウンロードしてインストールしました ...











どうした



Googleは、Java、Silverlight、Unity Web Player、Flash Playerなどのブラウザーにプラグインを実装するための古くからの実績のあるAPIであるNPAPI（Netscape Plugin Application Programming Interface）のサポートを放棄することを決定しました。 APIは、Netscape Navigator 2.0用に1995年に開発されました。



Googleは以前、NPAPIプラグインをホワイトリストに制限することでこれを行うと警告していた。 ただし、リストには最も人気のあるプラグインのみが含まれていたため、ほとんどのユーザーはそのことに気づきませんでした。



Chrome開発者はNPAPIの代わりにAPIを宣伝していますが、どういうわけかあまり成功していません。



どうする



2015年9月までは、chrome：// flags /＃enable-npapiですべてを有効に戻すことができますが、この機能も非表示になります。 燃え尽きて、偉大な帆船の時代は終わりました...
