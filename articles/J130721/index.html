アドビが買収したTypekit

2011年10月3日に、CSSの@ font-faceディレクティブを使用したWebサイトデザイン用の商用フォントへの有料アクセスに基づくTypekitサービスがアドビに買収されました。 同じ日にTypekitのブログ と Adobe プレスリリースで読むことができます。 Adobeによると、Typekitサービス（現在、New York TimesとWordpressを含む約25万人のユーザー）は、Adobe Creative Cloudプロジェクトの一部となる予定です。



アドビはすでにインターネットフォントトレーダーの間でランク付けされている可能性があることに注意してください。 Typekitが数千を販売する一方で、Adobe Web Fontsのコレクションでは、フォントの数が数百を超えています。 これを、オフラインで使用するために販売されているAdobe Type Libraryコレクション内のフォントの数と比較すると便利です-その数は2300を超えます。



Typekitブログの読者のコメントから判断すると、一部のユーザーは売られすぎの状態に不安を感じ 、価格の上昇やその他のトラブルを恐ろしく期待しています。 しかし、それはそうです。



個人的に、Typekitがあまり好きではなかったのは、率直に言って、ロシア語の文字のフォントプレビューを（少なくともFirefoxで）正しく動作させることができなかったからです。 タイプテスターの操作で。 ラテン文字（英語のスピーチなど）は正常に機能しますが、キリル文字は機能しません。 無料フォントのサイトGoogle Web Fonts with Cyrillicは問題ありません。



アドビに関しては、ブラウザでロシアのストアを開こうとすると、安全な接続証明書が20日前に期限切れになるという警告が表示されます。
