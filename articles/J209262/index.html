MicrosoftはWindows XPのウイルス対策保護を停止します

広く使用されているMicrosoftオペレーティングシステムであるWindows XPの終わり近く。 4月8日-待望の12年OSのサポート終了日-このニュースはほとんどのユーザーを驚かせるはずがありません。



ただし、XPに関するMicrosoftの別の決定は、アップグレードを拒否する頑固なXPユーザーに打撃を与える可能性があります。 同社は今週、4月8日に同社の無料ウイルス対策アプリケーションであるWindows XP用Microsoft Security Essentialsのサポートを終了すると発表しました。



4月8日以降、Windows XPユーザーはセキュリティパッチを受け取れないだけでなく、インストールされたMicrosoft Security Essentialsもウイルスアップデートを受け取らず、マシンを二重に保護しません。



Security Essentials製品に対するMicrosoftのサポートを終了することは、XPの支持者を最新の製品であるWindows 8.1に段階的に移行するための非常に巧妙な方法ではありません。 まだXPにしがみついている何百万人ものユーザーは、別の無料のウイルス対策ソフトウェア製品を見つけるか、XPの終followに続く可能性が高い悪意のある攻撃のリスクにさらされる必要があります。



中国のユーザーの50％以上を含む最近のデータによると、世界中のインターネットユーザーの20〜30％が依然としてWindows XPを使用しています。 専門家は今後のXPocalypseを回避するためにあらゆることを行っていますが、無数の悪意のある攻撃を回避するために、何百万人もの人々がシステムをアップグレードする必要があります 。



コンピューターでまだXPを使用している場合は、4月8日までにWindowsの新しいバージョンにアップグレードすることをお勧めします。 XPを搭載したWindowsの最新バージョンにアップグレードするには、まず「 Microsoft Windows 8 Upgrade Assistant 」をダウンロードして実行し、コンピューターがWindows 8のハードウェア要件を満たしているかどうかを確認する必要があります。 Windows 8がプリインストールされた新しいコンピューター。



コンピューターがWindows 8をサポートできるかどうかにかかわらず、現在のXPユーザーは、新しいシステムに転送したいデータのバックアップにも注意する必要があります。XP（またはWindows Vista）からの更新は、すべてのファイル、設定、または古いOSに現在インストールされているプログラム。
