Visual Studio 2010 SP1を追求して、新しい開発者ツール









おそらくご存知のように、Visual Studio 2010の更新パッケージが最近リリースされましたが、しばらくすると追加のツールが表示されるようになりました。









そこで、まず、Scott Guthrie 氏は自分のブログで 、デスクトップWeb開発者を迅速に準備するための新しい機会が利用可能になったことを発表しました 。 2つのVS 2010 SP1 WebPI BundleおよびVisual Web Developer 2010 SP1 WebPI Bundleツールキットは、Web Platform Installerを介した無人インストール用にリリースされました。 これらのパッケージは、VS 2010またはVWD 2010 SP1開発環境、ASP.NET MVC 3（+ツール）、IIS 7.5 Express、SQL Server Compact Edition 4.0（+ツール）、およびWeb Deployment 2.0のコンポーネントで構成されています。 Web Platform Installerを使用すると、これらのコンポーネントが自動的にダウンロードおよびインストールされるため、カスタマイズされたWeb開発者ワークステーションを数分で入手できます。









第二に、Entity Frameworkの新しいバージョンがインデックス4.1でリリースされました。これは、リリース以来のORMの重要な更新です。 リリース候補バージョンはまだ利用可能ですが、最終バージョンはまもなくリリースされます。 Entity Framework 4.1は、開発者にCode Firstプロジェクトで提示されたすべての革新を提供します 。 EF 4.1のリリースに関しては、MSDN Webサイトで特別なトレーニング資料が作成されました。









第三に、Visual Studio 2010 F＃のローカライズパッケージがダウンロード可能です。 このパッケージを使用すると、統合されたF＃言語ツールのエラーメッセージおよびその他のテキストデータをさまざまな言語にローカライズできます。 Report Viewer 2010コンポーネント用に同じパッケージがリリースされました。









4番目（ およびこれが最も興味深い ）、包括的なトレーニングパッケージVisual Studio 2010および.NET Framework 4トレーニングキットの新しい3月バージョンが利用可能です 。 このパッケージを使用すると、着信する.NETおよびVisual Studio 2010のすべてのテクノロジを探索できます。検討中のテクノロジの例をいくつか示します。C＃、F＃、WCF、ASP.NET、Silverlight、MEFなどです。 トレーニングパッケージには、 50の実験室作業、 22のデモプロジェクト、 16のプレゼンテーション、 12のビデオが含まれています。 3月バージョンとの主な違いは、 Silverilght 4の研究のための新しい研究室とWindows Azureの新しい資料です。
