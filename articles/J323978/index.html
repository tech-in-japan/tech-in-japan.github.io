IntelとGoogleは「クラウド」の友達です

サンフランシスコで開催されたGoogle Cloud Next '17カンファレンスでは、クラウドテクノロジーとモノのインターネットの分野におけるGoogleとIntelの戦略的パートナーシッププログラムBetter Togetherを発表しました。 このプログラムの目的は、開発者にソフトウェアおよびハードウェアコンポーネント、ツール、およびトレーニング資料を提供し、アプリケーションの作成プロセスを加速することです。 同時に、Googleは「クラウド」とその機能へのアクセスを開放し、 IntelはIntel IoT Developer Kitなどの開発ツールを提供し、 Intel NUCの形でモノのインターネットゲートウェイを提供します。









開発者向けの共同オンラインリソース（Intel Developer Zone Webサイトのプロジェクトのメインページ）は既に機能し、トレーニングやその他のイベントを実施する予定です。 次に、企業のスペシャリストは、Intel IoT Developer Kit、Intel NUC、およびGoogle Cloud Platformへの参加を統合します 。



IntelとGoogleの共同プロジェクトは、クラウドに接続されたIoTサービスを作成するためのツールとツールの完全なセットを開発者に提供します-創造性を追加するだけです。 このテーマに関するいくつかの考えはここで読むことができます 。
