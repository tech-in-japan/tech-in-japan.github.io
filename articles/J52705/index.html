インキュベーター-ビジネス算術

インキュベーターとスタートアップの戦いでは、すでに多くのコピーが壊れています。

膝を介してもう1つを破ります。

槍の破壊-槍を構築しないでください？









ハブで「放棄することはできません」という申し出をする方向でのインキュベーターと天使の新しい公共のインキュベーションはすべて、聴衆を著しく引きつけます。 情熱のマグカップは、主に「1万ドルを贈るが、将来のGoogleで50％（または回答者の感応度に応じて20または70）が欲しい？ やぎ！ （スタートアップはカルマを大切にしているため、これは自分自身に追加されます）。 Startupsブログの最新のトピックへのリンクは次のとおりです。1、2。



はい、インキュベーターは通常の冷笑的なビジネスですが、これはそれ自体がマイナスでもプラスでもありません。特に金庫に手を突っ込んだり、カサカサする幸福でポケットをいっぱいにしたりすることはありません。 実際、あなたがあなたに提供する条件に満足しているとしても、それを理解することも重要です-インキュベーターの経済的目標は何であり、それがあなたとどう相関するか。 インキュベーターは、お金を配るときに何を期待していますか？ これについてお話します。



ポイント1。

インキュベーターはマイクロ投資ファンドであり、そのビジネスは、初期のプロジェクトに投資してお金を稼ぐことです。 これは、彼らが将来会社の株式でまともな利益を得たい、または購入した株式よりもはるかに多く売却したいことを意味します。 まともな利益とは何ですか？ もちろん、各インキュベーターには独自の基準がありますが、これは少なくとも、銀行に預金するだけでなく、少なくとも何倍も意味します。

預金に対する銀行の利子は年に5％から12％の範囲であることを思い出させてください。 「はるかに高価な株式の販売」とは何ですか？ 同じデポジットの考慮事項から-インキュベーターが10,000をあなたに投資し、 X で3年間にその株式を売却した場合、年間利益率=

[（X / 10,000）-1の3次ルート] * 100％

そして、この数字は、銀行の年間5〜12％よりもかなり高いはずです。 インキュベーターの関心の下限として少なくとも年に30％を取りましょう（私は大多数が30％でうまくいくとは思わないと思いますが、そのような収益性のためにGorbushkaのラップトップで推測する方が簡単です）。

X =（0.3 + 1）^ 3 * 10,000 = 22,000

合計では、価値の成長を120％または2.2倍確保する必要があります。



ポイント2。

同時に、例外なくインターネット上のすべてのスタートアップは非常にリスクが高いことが重要です-つまり、ビジネスプランに従って成功する実装の可能性は低い（大まかに言うと、100のプロジェクトのうち、計画されたキャパシティでユニットが出てくる）、インキュベーターがリスク調整を行う必要があることを意味しますこれにより、投資された資本の収益率が非常に高い要件になります。 たとえば、インキュベーターが10のプロジェクトに それぞれ 10,000を投資し、 1 つを撃った場合、このプロジェクトでのインキュベーターのシェアのコストは、少なくとも100,000 、つまり10倍に増加する必要があります。 そうしないと、インキュベーターの総投資額10 x 10,000 = 100,000は、利益は言うまでもなく戻りません。 インキュベーターには、少なくともいくつかのプロジェクトが必要です。そうしないと、1つのショットが発射されず、インキュベーターがすべての投資を失う危険性があるためです。 たとえば、インキュベーターに10個のプロジェクトがある場合、業界平均の発火確率（実際には低い）が1/100である場合、10のうち少なくとも1つが発砲する確率はV = 10％です。 インキュベーターに100個のプロジェクトがある場合、この確率はV = 63％です。 （ V = 1-（0.99）^ N ）ここで、Nはインキュベーターでのスタートアップの数です）。 したがって、確率的な理由から、 10プロジェクトのインキュベーターは各プロジェクトのシェアのコストを10倍（プラス900％）増やす 必要があり 、 100プロジェクトのインキュベーターは100/63 = 1.59（プラス59％）を置く必要があります。 これはインフレーションを考慮せず、インキュベーターの累積利益が0であるため、投資したお金を返すためだけに出ます。 ショットの確率が1％を超えると思う場合は、他のパラメーターを簡単にカウントできます。



1と2を組み合わせると、投資済み資金の年間30％を終了するために、 10のプロジェクトを持つインキュベーターは、各プロジェクトのシェアのコストを2.2 * 10 = 22倍または2100％増加させる必要があります。 つまり、10のプロジェクトに100,000を投資し、そのうち9つのプロジェクトが死んだので、成功したプロジェクトを販売して、パッケージで少なくとも220,000を獲得する必要があります。これにより、年間120,000の利益が得られます。

100プロジェクトのインキュベーターでは、 2.2 * 1.59 = 3.5倍または250％の成長が必要です。 インキュベーターは、10,000個のプロジェクト（うち37％が死ぬ）に100 万個を投資し、成功した63個のプロジェクトで株を売却し、 （63 * 10,000 * 1.59 * 2.2）= 2,200,000または1,200,000の利益を獲得します。これは年間30％に相当します。 成功した各株のシェアは、平均で10,000 * 3.5 = 35,000で販売されます。



追加の変数がなくても見やすくなります（「お辞儀したパイ」についての冗談を覚えていますか？物理的な施設はありますか？）小さなインキュベーターにとっては、欲求よりもプロジェクトの大きなシェアが必要になることがよくあります。結局、シェアがたとえば20％の場合、売上は220,000 / 0.2 = 1,100,000になり、これは完全に非現実的です（小さなプロジェクト、危機など）。 そして、シェアが50％の場合、プロジェクト全体を220,000 / 0.5 = 440,000で販売する必要があります。 一般的に、プロジェクトの販売価格はインキュベーターのシェアに反比例し、安価な販売は高価なものよりもはるかに簡単です-奇妙なことに...



もう1つ-単純な数字からわかるように、ショットの確率は重要なパラメーターです。 「有能な」インキュベーターでの発砲の確率は、業界よりも高いことは明らかです。 インキュベーターの「能力」は、さらなる資金調達を求めたり、事業を売却したりする能力にも影響を与えます。そのようなインキュベーターは、必要な知人と、ふっくらした顧客の間で信頼の信用を持っています。 自問してください-私が投資契約の結び目を結ぶインキュベーターは、私のトピックでビジネスの本当のバイヤーと仕事をした経験があります（あなたのスタートアップの最終バイヤーになる可能性のある人を想像する必要があるという意味で-例えばYandex、またはAOL ）またはより高いレベルの投資家; 彼が少なくとも2〜3個のプロジェクトを設計キャパシティに売却または持ち込んだかどうか（1は事故の可能性があります）。 彼は、他のプロジェクトを考慮に入れて、必要な時間だけ私のプロジェクトをサポートする経済的な保証がありますか？



すべての成功。 およびインキュベーター、およびインキュベート。 ロシアのインキュベーターがどのように西洋のインキュベーターと根本的に異なるかについての重要なトピックがまだありますが、それらは異なります...
