Yandex.Money：カードからの直接支払い

こんにちは、Habr！



今日、Yandex.Moneyはユーザーが長い間私たちに尋ねてきたサービスを開始しました。

これで、Yandex.Moneyアカウントからだけでなく、それに関連付けられた銀行カードでもオンラインストアに支払うことができます。 さらに、コンピューターから離れることなくオンラインでバインドできます。ATMへの出張は不要です。



ロシアの銀行のカードをバインドできます。支払いを行うには、カードのCVV / CVCコードと支払いパスワードをYandex.Moneyに入力するだけで十分です。



カードからの支払いの追加手数料は撤回されません。



マップバインディング



カードをYandex.Moneyアカウントにリンクするには、 銀行サービスページ（[サービス]メニューからもアクセスできます）に移動し、[オンラインでカードをリンク]オプションを選択する必要があります。







その後、カードの詳細を入力する必要があります。







カード上で少量のランダムな量がブロックされるため、リンクを確認する必要があります。 SMS通知サービスを有効にしている場合は、電話に届くメッセージから必要な金額がわかります。 また、インターネットバンキングをご覧になるか、銀行サポートサービスにお電話ください。

バインディングを確認した後、カードでブロックされた金額はアカウントに戻されます。







カードがリンクされた後、Yandex.Moneyページのバランスブロック（左上隅）に、カード操作ページにつながるリンクが表示されます。 それをクリックすることで、いつでもカードのリンクを解除したり、サービスの使用に関するFAQを読むことができます。



カード払い



カードをリンクすると、支払い確認フォームの形式が変わります。

店舗で商品を選択して当社のウェブサイトにアクセスした後、支払い方法を選択できます。Yandex.Moneyウォレットから、または銀行カードから直接。



たとえば、次のようになります。







CVV / CVCコードと支払いパスワードで支払いを確認するのは残り、完了です。



落とし穴



事前にいくつかの質問に答えてみます。



接続されたカードではまだ何ができないのですか？
  • これまでのところ、ユーザー間でp2p支払いを行うことはできません。
  • Yandex.Moneyでアカウントを補充することはまだできません。
  • これまでのところ、一部の店舗（エクスチェンジャー、VKontakte、PINコード店など）に支払うことはできません。
このサービスの一環として、複数のカードをアカウントにバインドすることはできません。 しかし、たとえば、Alfa-Bank、OTKRITIE Bank、またはRosEvroBankが貼られたカード（たとえば、口座の便利な補充と資金の引き出しのため）とアフィリエイトの記述方法を持つことは可能です。 したがって、これらの銀行のいずれかの以前にリンクされたカードを使用して店舗に直接支払いをする場合は、上で説明したように、それをほどいてから再度結ぶ必要があります。



サービス内での支払いの制限は 、1日あたり5,000ルーブルと1か月あたり30,000ルーブルです。



利益！


しかし、このサービスにはすでに制限時間よりも実際の利点があります。
  1. 投稿の冒頭で述べたように、 支払い手数料は0％です。
  2. 店に支払いをするとき、あなたはカードの詳細を入力する必要がなく 、彼らの安全を恐れる必要はありません 。 バインド時にのみ指定します。
  3. カードからの支払いは、支払いパスワード（または拡張認証に切り替えた場合はトークン/グリッドカード）およびCVV / CVCコードによって保護されたままです。
  4. すべてのアクションはオンラインで行われます -どこにでも行く必要はありません。
