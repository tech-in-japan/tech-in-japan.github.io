Wi-Fi 6の発表：新しい標準について知っておくべきこと

10月上旬、Wi-Fi AllianceはWi-Fi標準の新しいバージョンであるWi-Fi 6を発表しました。そのリリースは2019年末に予定されています。 開発者はネーミングのアプローチを変更しました-802.11axのような通常の構造を1桁に置き換えました。 他に何が新しいのかを理解しています。





/ウィキメディア/ ヨノラテンゴ / CC



名前を変更した理由



標準の開発者によると、命名の新しいアプローチにより、Wi-Fi標準の名前が幅広い読者に理解可能になります。



Wi-Fi Allianceは、ユーザーがホームルーターで動作できない標準をサポートするラップトップを購入する状況は非常に一般的であると指摘しています。 その結果、新しいデバイスは後方互換性メカニズムを使用します-データは古い標準を使用して交換されます。 場合によっては、これによりデータ転送速度が50〜80％低下する可能性があります。



特定のガジェットがサポートする標準を明確に示すために、アライアンスは新しいマーキングを作成しました-Wi-Fiアイコン。その上に対応する番号が示されています。







Wi-Fi 6はどのような機能を提供しましたか？



Wi-Fi 6のすべての機能と特性の詳細な説明は、Wi-Fi Allianceの公式文書 （入手するにはフォームに記入する必要があります）またはシスコが作成した文書に記載されています。 次に、主なイノベーションについて説明します。



2.4および5 GHz帯域のサポート。 理想的には、2.4 GHzと5 GHzの同時サポートにより、複数のデバイスを使用するシナリオの数を増やすのに役立ちます。 ただし、実際にはこの利点は役に立たない場合があります。 新しいデバイスは互換モードで定期的に動作するため、市場には多くのレガシーデバイスがあります（2.4 GHzをサポート）。



OFDMAサポート。 これは直交周波数分割多元接続（OFDMA）です。 実際、このテクノロジーはOFDMの 「マルチユーザー」バージョンです。 信号をサブキャリア周波数に分割し、個々のデータストリームを処理するグループを選択できます。



これにより、平均速度で複数のWi-Fi 6クライアントにデータを同期的に送信できます。 ただし、警告が1つあります。これらのクライアントはすべて、必ずWi-Fi 6をサポートする必要があります。したがって、「古い」ガジェットも「オーバーボード」です。



MU-MIMOとOFDMAのコラボレーション。 Wi-Fi 5（2014年に承認された古い表記の802.11ac ）では、 MIMO （マルチ入力マルチ出力）テクノロジーにより、異なるサブキャリアを使用して4つのクライアントにデータを送信できました。 Wi-Fi 6では、可能なデバイス接続の数が2倍になりました（最大8つ）。



Wi-Fi Allianceによると、MU-MIMOシステムとOFDMAは、 ダウンリンクで最大11 Gb / sの速度でマルチユーザーデータ転送を編成するのに役立ちます。 この結果は、CES 2018でのテストデバイスによって実証されました 。ただし、ハッカーニュースの住民は 、通常のガジェット（ラップトップ、スマートフォン）にはこのような速度が見られないことに注意しています。



CES でのテストでは、 D-Link DIR-X9000トライバンドルーターが使用され、11 Gb / sは3つのチャネルの最大データ転送速度の合計です。 Hacker Newsの住民は、ほとんどのデバイスが1つのチャネルのみを使用するため、データは最大4804 Mbpsの速度でブロードキャストされることに注意しています。



ターゲットウェイク時間関数。 これにより、デバイスはスリープモードに移行し、スケジュールに従って「起動」できます。 Target Wake Timeは、デバイスがアイドル状態で動作する時間を決定します。 ガジェットが特定の期間（夜間など）にデータを送信しない場合、そのWi-Fi接続は「スリープ状態」になり、バッテリー電力を節約し、ネットワークの輻輳を軽減します。



各デバイスに対して、「目標ウェイクアップ時間」が設定されます-条件付きラップトップが常にデータを転送する瞬間（たとえば、企業ネットワークの営業時間中）。 そのような期間中、スリープモードはアクティブになりません。

Wi-Fi 6はどこで使用できますか？



開発者によると、この技術は高密度Wi-Fiネットワークの展開に役立ちます。 MU-MIMOやOFDMAなどの個別のソリューションにより、公共交通機関、企業環境、小売エリア、ホテル、スタジアムでの通信の品質が向上します。



しかし、ITコミュニティのメンバーは、テクノロジーの採用という観点から、Wi-Fi 6を大きなマイナスと考えています。 Wi-Fi 6への移行の具体的な結果は、ネットワーク上のすべてのデバイスが新しい標準をサポートしている場合にのみ顕著になります。 そして、これには間違いなく問題があります。



Wi-Fi 6のリリースは2019年末に行われることを思い出してください。
