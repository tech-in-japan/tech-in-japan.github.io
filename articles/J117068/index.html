マイクロソフトリサーチアクセラレータ





Research Acceleratorsは、学生、大学院生、教師、科学者向けのツールキットです。 彼らは、研究、学術研究、科学の研究、データの分析と視覚化、および行われた研究の結果の公表を支援します。 多くのツールには、オープンソースコードが付属しているか、対話と拡張のためのオープンインターフェイスがあります。 これにより、Microsoft Researchグループのエンジニアや研究者の最も高度な開発者と協力することができます。



リサーチアクセラレータキットの構成





Wordの化学アドイン


Word用ケミストリーアドインは、 Microsoft Word文書に化学式と画像を入力するプロセスを簡素化します。 これにより、化学情報の入力と処理に特化した個別の「化学ゾーン」をドキュメントに作成できます（これには、データの編集、レンダリング、エクスポートの特別な機会が含まれます）。







Wordの記事作成アドイン


Wordの記事作成アドインを使用すると、Word文書を操作するときに、著作権で保護された資料に関連する大量のメタデータを入力できます。 NLM形式（National Library of Medicine's）でドキュメントを保存できます。





カンファレンスXP


Conference XPは、遠隔教育や他のリモートコラボレーションシナリオのビデオ会議用に設計された無料のソフトウェアです。 ConferenceXPは、高性能のオーディオ、ビデオ、およびネットワークテクノロジーを統合し、多数のリモートユーザーを柔軟なインタラクティブ環境に統合し、生産性の高いコラボレーションを実現します。



Microsoft Office用のCreative Commonsアドイン


Microsoft Office用のCreative Commonsアドインを使用すると、Creative CommonsライセンスをMicrosoft Officeドキュメントに追加できます。これは、ユーザー定義のパラメーターに従ってオンザフライで生成されます。





DryadおよびDryadLINQ


DryadとDryadLINQは、クラスター化されたWindows環境に分散アプリケーションを実装するタスクを簡素化するように設計された高性能な分散データ処理エンジンです。 DryadLINQを使用すると、開発者は、LINQの特別な拡張バージョンと追加のAPIを使用して、マネージコードでアプリケーションを作成できます。





Word用GenePatternアドイン




Word用GenePatternアドインは、遺伝子分析の分野でいくつかのプロセスをモデル化および分析できる生物学的ソフトウェアです。 このプラグインを使用すると、作成したオブジェクトをMicrosoft Word文書に埋め込み、Wordから他のGenePatternソフトウェアと直接やり取りできます。



マイクロソフト生物学財団




Microsoft Biology Foundationは、Microsoft .NETプラットフォームの拡張機能として作成されたツールボックスであり、研究者や科学者の日常業務を支援します。 このプラットフォームはフリーソフトウェアであり、バイオインフォマティクスでよく実行される多数のタスクの補助ツールとして機能します。 多くの大学や企業がMBFを使用しているため、日常業務に費やす時間を減らし、コア研究に集中できます。



NodeXL


NodeXLは、大規模ネットワークを視覚化および分析するための強力で使いやすいツールです。 NodeXLは、MS Excelをグラフ作成、分析、データ視覚化のプラットフォームとして使用します。 このツールは、さまざまなソーシャルネットワークからデータをインポートするための多くのプラグインをサポートしています。





Wordのオントロジーアドイン


このプラグインは、科学および技術テキストのメタデータを使用して作業を簡素化します。 その助けにより、用語、脚注、定義、および記事の主要な構成要素ではないが、理解に必要なことが多いその他の情報ブロックを簡単に扱うことができます。 そのような資料の作成を簡素化することに加えて、検証、HTML、XML、およびNLM形式へのエクスポート、辞書との対話、タグの操作などがサポートされます。





研究情報センター


Research Information Center（RIC）は、Microsoft Researchと大英図書館が共同で作成した仮想的な研究環境です。 製品のソースコードとインストーラーはCodePlexで入手できます。 RICは、Microsoft Office SharePoint Serverのアドオンであり、その機能を拡張して研究グループや学術グループのニーズに応えます。 その支援により、科学プロジェクトのサイトをすばやく作成して埋めることができます。これには、カレンダー、タスクリスト、Wiki、ブログ、プロジェクト参加者の個人ページ、その他の資料を含めることができます。







プロジェクトトライデント：科学的ワークフローワークベンチ




Tridentプロジェクトは、科学的なワークフローを管理するように設計されています。

その機能は次のとおりです。

  • さらなる研究のためのデータ分析と視覚化の自動化
  • 実験の作成、保守、カタログ化
  • 詳細な実験記述の維持
  • 典型的なワークフローとアルゴリズムを記述するライブラリを作成する
  • コンピューターのクラスター時間の分布を含むワークフローの計画
  • 研究チームの調整




善意


Zentityは、デジタルライブラリ（データウェアハウス）の作成と維持に役立つモジュール、ユーティリティ、およびサービスを提供するプラットフォームです。 Zentityを使用すると、研究者は必要な情報に簡単にアクセスして分析し、さまざまな構造要素間のすべての関係を視覚的に確認できます。 Zentityは、.NET 4.0、Microsoft SQL Server 2008、Entity Framework、LINQなどのテクノロジーに基づいています。 Zentity API、Microsoft Live Lab Pivot Viewer（任意の次元のデータの大規模な配列とそのメタデータを視覚化するように設計されたSilverlightアプリケーション）、またはMicrosoft Research Visual Explorerを介してZentityのデータにアクセスできます。
