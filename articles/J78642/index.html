Habréで切望されているトップに入るには？

Habréの一部の人々は投稿を書いてトップに到達しようとしますが、成功する人はほとんどいません。 私の投稿は、トップに到達するためにどのように、そして何を書く必要があるかについてです。 私の経験から。



この記事は、特に初心者にとって、 投稿を書く人だけに役立ちます。







Habréでしばしば「マイナス」と「obsserat」ということが起こりました。 しかし、ご覧のとおり、トップはまだ質の高いコンテンツでいっぱいであり、批判は誰にも害を与えていません。 そのため、一部のユーザーは興味深い資料を作成でき、他のユーザーは作成できないと想定するのは理にかなっています（ただし、学習したいことを教えてくれます）。



まず 、資料がどれほど興味深い/興味のないものになるかを理解するには、誰がそれを読むのか、つまり Habrahabrの対象ユーザー。 昔々、「あなたは誰ですか、ハブロマン」というトピックに関する調査がありましたが、残念ながら、私はそれを言葉で見つけることができませんでした。つまり、ハブルの人々はまず第一に、ロシア、ウクライナ、ベラルーシなどのIT専門家、主に男性です国（これらの統計はここで見つけることができます： http : //habrahabr.ru/info/stats/ ）。 これから結論を1つだけ引き出すことができます。ロシア人男性向けのプログラミングに関する投稿を書く必要があります。 :)



次に、 最も興味深いトピックを知る必要があります 。 2007年に、このトピックに関する調査がありました（ http://habrahabr.ru/blogs/ilhh/35190/ ）。 その上で最も興味深いトピック：インターネットプロジェクト、将来の技術、Webプログラミング、設計、レイアウト、管理。 しかし、それはすでに時代遅れであり、Habrの視聴者はそれ以来少し変わっていることを忘れないでください。今日、いくつかの人気トピックを追加します。ビジネス、自己開発、「日曜大工」、有名人（主にIT専門家）、ガジェット、情報セキュリティです。



約半数のユーザーが、投稿を読んだ後（評価）、タイトルを読んだ後、またはコメント付きで投稿を読んだ後、少し評価します（このトピックに関するアンケートは、 http ： //habrahabr.ru/blogs/ilhh/35229/にあります ）。 したがって、コピーライティングの最初のルールを思い出してください。見出しは記事の最も重要な部分であり、記事の序文に注意を払います。興味があり、記事全体を読む必要があります。 また、人は投稿の最初と最後に最も注意を払うことを覚えておく必要があります。したがって、最初は興味があり、最後に要約します。



Habréでは、彼らは単にあらゆる形の広告を嫌うので、広告したい人へのアドバイスを忘れないでください：控えめにそれをしてください、そして一般的には、トップに投稿を得た後、すでに控えめな広告さえ追加する方が良いです。



良い投稿には、注目を集めるために主な考えが大胆であり、Habréでもユーモアが大好きなので、ジョークで薄められた投稿は、単なるドライテキストよりもはるかによく知覚されます。



もちろん、友情を利用して支援することもできます。たとえば、投稿がトップの前に1票ない場合（トップに到達するには+ 7が必要です）、Habréに登録されている友人の注目を投稿に引き付けることができます。



そしてもう1つの重要なポイント：私の個人的な観察によると、投稿へのコメントは非常に注目を集め、トップに立つのに役立ちます！



結論：興味深いトピックに関する興味深いユニークな投稿を書いて、問題なくトップになってください！
