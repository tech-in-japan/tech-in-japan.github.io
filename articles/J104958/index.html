MeeGoベースのWeTabが発売

Amazonのオンラインストアのドイツ支店では、 Intel＆Nokiaの本格的なLinuxシステムをベースにしたかなり興味深いタブレットの販売が始まりました。







カットの下のデバイスのビデオと特性。





2つのタブレットがあります。 16GBの通常のタブレットと大きなダイヤラ、32GBのメモリとaGPS、3G / UMTSを備えたチョッパー（必要な場合）。 いくつも見えなかったが、どこでも、価格はタブレットで449ユーロ、タブレットで669ユーロでスキップされます。 しかし、残念なことにAmazon.deはまったく異なる考え方をして最年少モデルを579ユーロで販売し、古いモデルを押します。ダイヤラー機能を備えたこの種のデバイスは間違った手に渡ってはならず、すべての顧客は厳しい純粋なドイツのテストを受けます。







パキスタンのi報機関にも知られているように、このデバイスの正確な技術仕様が知られています。 サンスクリット語からのインテリジェンスの複雑で苦痛に長い（接続されたパキスタン人にとって）翻訳によって、このデバイスはそれほど悪くなく、次のものが含まれていることがわかりました。

画面 ：1366 x 768の解像度とmutitouchサポートを備えた11.6インチ

プロセッサー ：Intel Atom 1.66 GHz N450 Pineview-M

RAM ：1 GB

ROM ：16および32 GB + SDHC用スロット

ビデオ ：統合されたIntel GMA 3150

神とのつながり ：Wi-Fi a / b / g / n＆bluetouth 2.1 + EDR

プラグで突くのが好きな人のために ：HDMI＆2xUSB 2.0
