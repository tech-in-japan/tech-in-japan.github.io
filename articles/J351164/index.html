Whatever Hack 3月16日にお誘いします







48時間の機械学習マラソンが3月16日にモスクワのオフィスで始まります。その間、ミームを生成するためのソリューションを作成し、キャビネットをカーペットと区別したり、無料テーマの一部として最高のMVP（最小実行可能製品）を作成するためのスキルを発揮できます。







3つの問題のいずれかを解決し、150万ルーブル（またはルーブルではない）の賞金プールを競うように皆を招待します。







さらに-タスクと詳細について。







ルールは非常に単純です。 各チームには最大4人まで参加できますが、チームメンバー全員の参加は必須です。 参加者は、次の3つのタスクのいずれかを選択できます。







1.ミームの品質の決定









データセットは、写真とそれらのテキストキャプションのリストです。 各画像と署名のペアには、ラベルがあります-彼らがそれを共有したかどうか。 人々が特定の写真と署名のペアを共有するかどうかを予測する必要があります。







2.インテリア写真での家具の検出と分類









データセットは、バウンディングボックスを使用して家具をマークした写真で構成されます。 〜git clone fast-rcnn〜を実行し 、検出器をトレーニングする必要があります。これにより、指定された家具が見つかり、パラメータに応じてアパートを選択できるようになります（フィルターを「祖母のカーペットなし」にする）。







3.無料テーマ









長い間延期されていた問題を解決したり、問題の拡張バージョン1および2を解決したりすることができます。







最初の2つのタスクは客観的な指標になります。 3番目のタスクでは、参加者の決定が審査員によって評価されます。







各カテゴリの5つの最高のチームには賞が授与されます。







  1. 240k₽
  2. 120k₽
  3. 80k₽
  4. 40k₽
  5. 20k₽


ただし、賞品は地下室で入手できます。







Whatever Hackハッカソンは、CISで最大のMLコミュニティであるOpen Data Scienceの誕生日に捧げられ、DbrainとMail.Ru Groupが後援しています。 参加は無料ですが、 登録する必要があります。







集会：モスクワ、3月16日18：00、Mail.Ru Groupのオフィス（Leningradsky Prospekt、39、p。79）。







完了：3月18日。
