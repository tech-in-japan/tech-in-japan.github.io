夏期締め切り：マイクロソフトバーチャルアカデミーのトップ10コース

これは、 Microsoft MVAバーチャルアカデミーのレベルスキルに関する一連の資料の最後の記事です。 前の2つはこことここにあります 。 Windows Server 2012 R2およびSystem Center 2012 R2を使用したデータセンターの仮想化、企業リソースでのアクセスセキュリティ、企業でのWindows 10の展開の詳細、AzureでのLinuxクラスター、Windows ServerでのLinuxワークロードの仮想化： 2012 R2、Microsoftの専門家によるハイブリッドソリューション、Windows Server 2016、Windows Nano Server、モバイルアプリケーション用のOffice 365 API、SharePoint Serverの構成の詳細。









1.一連の3つのコース「Windows Server 2012 R2およびSystem Center 2012 R2を使用したデータセンターの仮想化」



Windows Server 2012 / R2のリリースにより、Microsoftハイパーバイザーに多数の変更が発生しました。 3つのコースでは、物理および仮想データセンターインフラストラクチャの展開、管理、および保守に使用されるWindows ServerおよびSystem Centerの主要なテクノロジについて学びます。 タイトルのリンクをクリックして、シリーズの最初のコースを見つけます。



2.一連の2つのコース「Corporate devices」



企業ネットワークにアクセスするために、会社の従業員は個人用デバイスを使用することが多く、これはドメインに含めることができないことが多いため、従来のアプローチと管理ツールを適用します。 この一連のコースでは、Windows、Windows Phone、iOS、Androidを実行しているモバイルデバイスから企業リソースへの安全なアクセスを提供するテクノロジーについて学習します。 タイトルのリンクをクリックして、シリーズの最初のコースを見つけます。



3. Windows 10：展開、管理、セキュリティ



このコースでは、企業全体に新しいOSを展開し、MDMソリューションで管理し、Windows 10デバイスのセキュリティを確保し、企業情報の漏洩を防止する方法の詳細について説明します。 新しいWindows 10構成機能、Assessment and Deployment Kit展開ツールの変更、およびインプレース機能のアップグレードについて学びます。



4. AzureでLinuxクラスターを構築する



AzureのLinuxクラスター-Linuxオペレーティングシステムを実行し、Microsoft Azureクラウドシステムでホストされ、1つの問題を共同で解決するように構成された多くの仮想マシン。 このコースでは、DRBDベースのフェールオーバーストレージ、CorosyncおよびPacemakerクラスターソフトウェアのセットアップ、フェンシングなど、クラウドでフェールオーバークラスターを整理する方法を学習します。 概して、これらの設定は、クラウドを使用せずに、従来のアーキテクチャのシステムにも適用できます。



5. Windows Server 2012 R2でのLinuxワークロードの仮想化



Windows Server 2012 R2は、LinuxおよびFreeBSD仮想マシンを実行するための高度な機能を提供します。 このコースでは、Windows Server 2012 R2がLinuxおよびFreeBSDの仮想化に使用される理由、Hyper-VがLinuxおよびFreeBSDワークロードの実行に対する強化されたサポートを提供する方法、および仮想化技術に深く没入するために探索する価値のあるリソースについて専門家が説明します。



6.マイクロソフトの専門家によるハイブリッドソリューション [英語]



企業クラウドのセキュリティと透明性について学びます。 コンピューター上の複数のソースからデータを収集して検索するプロセスを簡素化する方法について説明します。これにより、運用上の問題の根本原因を簡単に特定できます。 また、クラウドの使用量を予測し、必要な容量をできるだけ正確に計算する方法も学習します。



7. Windows Server 2016への深い没入 [英語]



柔軟なインフラストラクチャにより、複雑なシステムを迅速、効率的、経済的に統合できます。 このコースでは、Microsoftの専門家がWindows Server 2016の安全性、スケーラビリティ、および効率性の向上について話し、Hyper-Vの背景を見て、Windows Nano Serverの使用について話します。



8. Windows Nano Serverに飛び込む



Windows Nano Serverは、Windows ServerおよびHyper-Vクラウドおよびコンテナー向けに最適化されたWindows Server 2016オペレーティングシステムのコンパクトバージョンです。 これにより、インストールする重要なセキュリティ情報の数を90％以上減らし、必要な再起動の回数を80％以上減らすことができます。 このコースでは、Windows Nano Serverをインストール、展開、および管理する方法を学習します。



9. SharePoint Serverのインストールと基本構成 [英語]



このコースは、SharePoint Server 2013のマルチサーバーアーキテクチャを展開および構成する方法を学びたい人を対象としています。SharePoint2013を1つ以上のサーバーに展開して、展開ライフサイクルのさまざまな段階でSharePoint 2013ソリューションをテストおよび実装するために使用できるさまざまなトポロジを作成する方法を学習します。 。



10.モバイルアプリケーション向けOffice 365 APIのディープイマージョン [英語]



Office 365 APIを使用して、Apache Cordova、ユニバーサルWindowsプラットフォーム（UWP）、Xamarin用のVisual Studioツールを使用したモバイルアプリケーションの開発について学習します。 また、iOS用のネイティブXcodeとAndroid用のAndroid Studioについても説明します。
