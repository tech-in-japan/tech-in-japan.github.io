ライダー・リチャード・ストールマン

リチャード・ストールマンをカンファレンスで講演するよう招待したい場合は、彼の要件を考慮する必要があります。 もちろん、他の世界のスターと同様に、フリーソフトウェア運動の創設者にはライダーがいます。つまり、イベントを開催するための詳細な条件が記載された長いドキュメントです。 ストールマンの性格を知っていれば、彼のライダーは珍しいと推測できます。 ですので、このドキュメントの最新バージョン（2011年10月18日付け）をご覧ください。



GNUプロジェクトの創設者であるFree Software Foundationと「Freedom for Programming Freedom」、「コピーレフト」の概念の著者は、細部に非常に注意を払っています。 会議が適切な飲み物を持っていることは非常に重要です。 お茶が望ましい-ストールマンはいつもポケットにお気に入りのお茶をいくつか入れています。 彼は他のブランドに同意します、「どんなお茶でもミルクと砂糖でおいしい」からです。 リチャードが眠いのを感じるなら、彼は2つのジャーまたは非食餌性ペプシの小瓶を見たいです。 ストールマンは、コロンビアとグアテマラの組合指導者の暗殺に対するコカコーラの国際的なボイコットが現在進行中であることを思い出すので、コカコーラは決してありません。



もちろん、リチャード・ストールマンはイベント全体がフリーソフトウェアの精神を満たしていることを確認しなければならないので、彼は彼の前と後のスピーカーのリストをチェックしなければなりません。 彼は、事前の同意なしに、入場が制限されているか、さらにチケットが販売されているかどうかに断固として反対しています。 リチャードは技術に詳しくない多くの聴衆と話すのが大好きで、もし彼が入場の制限を見つけたならば、彼は話すことを拒否する権利を留保すると言います。 商人には、評判の良い組織からの「出席証明書」の販売を組織することを勧めています。 組織が本当に評判が良い場合、証明書はキャリアのためにそれらを必要とする人々の間で需要があり、残りは無料で行くことができます。



スポンサーは、イベントの参加者に広告資料を配布してはなりません;ステージ上に会社またはその製品の名前が付いたストレッチマークがあってはなりません。 プログラムに感謝を込めて小さなロゴを配置することのみが許可されました。 ストールマンは、スポンサーが同意しない場合、スポンサーなしでイベントを開催する必要があると言います。



パフォーマンスが他のイベントの隣に編成され、さまざまな企業のロゴが広告ポスターや隣接するディスプレイに表示されることがあります。 この場合、倫理上の理由で特に好きではない会社がそこにフラッシュすると、ポスターを布で覆うか、スピーチの期間中ディスプレイをオフにするように強制されるとストールマンは言います。



イベントのタイトルには、「知的財産」、Linux、またはオープンソースという言葉を含めるべきではありません。フリーソフトウェア運動について話しているのに対し、4年後にはオープンソース運動が「具体的には私たちの理想主義哲学に挑戦する」とストールマンは言います。 彼らは意図的に自由について話すことを避けています。 これは、超党派の政治システムにおける2つのライバル政党のようなものです（詳細はこちら ）。 同じことがLinuxにも当てはまります。GNU/ Linuxオペレーティングシステムで使用されるカーネルです（これはそれが呼ばれるべきものです）。 当然、ストールマンは彼の近くでペンギンを見たり、GNU / Linux以外のディストリビューション（Ubuntuなど）を聞きたくありません。 一般に「知的財産」というフレーズは、正しいイデオロギーを破壊する不条理なミームウイルスとしての使用から除外する必要があります。



ステージでは、ストールマンはマイクのみを必要とし、スライドや他のプレゼンテーションを使用しません。 通常のプレゼンテーションは、1時間30分と質問への1時間の回答に続きます。 テーブルには、FSFメーリングリストにサインアップしたい人のために、紙の束とペンが必要です。



ストールマンは、ジャーナリストが正しい用語（GNU / Linux、フリーソフトウェアなど）を使用すると約束した場合、インタビューを行う準備ができています。 彼は、パフォーマンスのオーディオ録音を行うすべての人に、録音に品質の低下したコーデック（つまり、Ogg以外の人）を使用しないように依頼します。 デジタルビデオの場合、Matroska VP8またはOgg Theora（オーディオ用-Ogg VorbisまたはOgg Speex）を選択することをお勧めします。 ストールマンを使用してインターネット上でビデオを配信する場合、これらの要件は望ましいものから望ましいものになります。 商用フォーマットのQuicktime、Windows Media Player、または特許取得済みのMPEG2、MPEG4、MP3を使用することはできません。



招待者は、Stollmanに飛行機のチケットを購入し、このチケットを紛失した場合、新しいチケットの購入に同意する必要があります。 彼のために電車やバスのチケットを購入する場合は、誰にも彼の名前を付けてはいけません。 「兄貴は私がどこへ旅行するのか、あなたがどこへ旅行するのかを知る権利がない。 興行所があなたに名前を言うことを要求するならば、あなたはあなたの友人のいずれにも属さない名前を与えるべきです、とストールマンは言います。 搭乗前にバスまたは電車でパスポートを確認した場合、この交通手段を放棄する必要があります。 米国では、搭乗前に書類をチェックするという方針のため、国際列車を使用することはありません。 ストールマンのチケットを購入するときは、名前も付けず、現金のみで支払います。



同様の規則がホテルに適用されます。 パスポートが必要であり、チェックイン時にすべてのお客様のデータを警察に送信することは受け入れられません（多くのホテルで一般的な慣行です）。 ホテルの部屋の温度は22°Cを超えてはなりません（乾燥した空気の場合は23°Cになります）。そうでない場合、Richard Stallmanが眠りにつくことは困難です。 温度がこのしきい値をわずかに上回る場合、ファンが必要です。 温度が3度高い場合は、空調が必要です。 ホテルの部屋は、ネットワークにアクセスでき、プロキシとファイアウォールなしでSSH接続を自由に確立できる必要があります。 リチャードストールマンはブラウザを使用しません。通常のスタッフはインターネットとウェブの違いを理解していないため、ホテルのシステム管理者にホテルチェーンの特性を確認することをお勧めします。 ストールマンにとって、身分証明書なしでオンラインにアクセスできることも重要です。



ストールマンが入室する部屋は猫の毛をきれいにしなければなりません。 一方、部屋に友好的なオウムがいる場合、彼は「とても、とても幸せ」になります。 しかし、リチャードは彼のために特別にオウムを買わないように頼みます。それは非常に大きな責任であり、意識的にすべての責任を負わなければならない重要な人生決定だからです。



食べ物に関しては、リチャード・ストールマンは特に好き嫌いがありません。 準備が整っていれば、彼はほとんどすべての料理が好きです。 彼はコーヒーと強いアルコールを飲まず、ワインだけを飲みます。 ワインはもっと複雑です-リチャードは特定の種類のワインが好きですが、彼はその名前を覚えていないので、彼に尋ねるのは無意味です。 リチャードは、テスト方法を使用して、その場で酒の問題を解決します。
