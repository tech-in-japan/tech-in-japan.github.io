リカバリー/ var / lib / dpkg / status

今日、Debianでの長い実験中にそれが起こりましたが、誤って削除されました： / var / lib / dpkg / status その結果、apt-getとdpkgおよびsynapticの両方、およびシステムの更新とアップグレードに関連するすべてが故障しました。

ここで、ステータスを復元する方法を説明することにしました。突然誰かが役に立ちます。

/ var / lib / dpkg / statusを復元するには、ファイル/var/backups/aptitude.pkgstates.0をディレクトリにコピーする必要があります- / var / lib / dpkg / status ： cp /var/backups/aptitude.pkgstates.0 / var / lib / dpkg /

その後、コピーしたファイルの名前をstatusに変更すると、問題は解決します。



psこのソリューションは、もちろん、最高のふりをするものではありません。いずれにしても、私がこのソリューションを見つけたほぼ1日を費やしています。 この問題の別の解決策をコメントに残していただければ幸いです。
