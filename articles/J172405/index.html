Radio86RK-ソビエトの即興コンピューター

コンピューターを作ったことがありますか？ まあ、少なくとも一生に一度。 少なくとも完全ではありません。



「もちろんです！」とあなたは言います。



そして、あなたはまったく正しくないでしょう。



いいえ、私は決してあなたの功績を損なうことはありませんが、1986年にソビエト連邦でコンピューターを組み立てることができたのは、経験豊富なラジオ愛好家だけでした。 マザーボード、プロセッサを購入し、RAMスロットを2つと大容量のハードドライブを購入し、500W電源の美しいケースにすべて入れるだけで十分です。 これはすべて、はんだごて、オシロスコープ、多数の回路図の助けを借りずに行われます。







ソ連時代には、実際に家庭用コンピューターを組み立てたと自慢できるのは、Habrの読者のごく少数です。 希少かつ不正な方法で、彼らは希少なマイクロ回路を入手し、プリント回路基板をエッチングし、夜までUV ROMと戦い、電源用のトグルスイッチプログラマーまたは巻線トランスで武装しました。 数ヶ月にわたる骨の折れる組み立てと試運転の後、最終的に、すでに誇らしげに家庭用コンピューターと呼ぶことができるものが判明しました。 そして、もしあなたがあなたの創造にふさわしく美しいケースを見つけたり、作ることができたら、そのようなコンピューターは、まだクラフトスキルをレベル80まで上げていない誰もがうらやましかった。



最も人気のある自家製ホームコンピューターの1つはRadio86RKで、その説明と図は、雑誌「Radio」No。4-6 / 1986の一連の記事に掲載されました。これらの記事の2人の著者、G。ZelenkoとS. PopovはVと共著しました。 B. パノフはその後、完全にコンピューターRadio86RKに専念する本「Home Computer」（出版社「Radio and Communication」）をリリースしました。



Radio86RKは、アマチュア無線家による組み立て用に提供された最初の家庭用コンピューターではありません。 その前身はMicro-80であり、多数のモジュールで構成され、約200個のマイクロ回路で構成されていました。 組み立てて委託することは困難でしたので、あまり配布されませんでした。





Micro-80コンピューターの作者の一人。 http://mrsaing.my1.ruから撮影した画像



しかし、Radio86RKはわずか29のマイクロ回路と少数のその他の無線コンポーネントで構成されていたため、組み立てが簡単で、取り付けにエラーがなければ、電源を入れた直後に動作を開始しました。 ただし、無線コンポーネントに欠陥があるため、Radio86RKが起動しない場合があり、オシロスコープのプローブを拾い、故障を探す必要がありました。



後になって、国家がアマチュア無線デザイナーを生産することを決定したとき（さらにはアマチュア無線家庭用コンピューターのクローンを生産することさえも）、アマチュア無線が住みやすくなりました。



量産されたRadio86RK のクローンの1つは、 Mikrosha PCのレビューでHabréにすでに書かれていました-Radio -86RKのクローンの1つですが、作者は噛まれたコンデンサKMのために作者と正しく働くことを拒否しました。それらに含まれる貴金属のために、一時的に、彼らは色が変わったようです。





ミクロシャ。 wikipedia.orgから撮影した画像



さらに、Mikroshはその祖先と部分的にのみ互換性があるため、Radio86RKの多くのプログラムおよびゲームは、適切な適応なしでは実行できません。



ミクロシャに加えて、Habrのページにはアマチュア無線コンピューターOrion-128の概要があり、その基礎は明らかにRadio86RKによって取られました。 Orion-128には、Radio86よりもはるかに多くのRAMがあり、ROMディスクとドライブが接続されていました。また、特殊なコントローラーを介して最大10Mbのハードディスクを接続することもできました。



しかし、これまでのところ、これらすべての多数のクローンの親、つまりRadio86RKについてのレビューを書いた人はいません。 今日、この迷惑な誤解を修正したいと思います。 それでは始めましょう。



Radio86RKの技術的特性。




プロセッサー：K580IK80A（KR580VM80A）-ソビエトのI8080Aプロセッサーのアナログで、1.78 MHzの周波数で動作します。 実際、このプロセッサーの公称クロック周波数は2.5 MHzでしたが、Radio86PKホームコンピューターの回路を簡素化するために、ビデオコントローラーのクロックジェネレーターを放棄し、プロセッサークロックジェネレーターをそのまま使用することにしました。 ビデオコントローラーが1.78 MHzの周波数しか消化しないという事実により、私はより低いプロセッサー周波数で満足しなければなりませんでした。




ROM：紫外線消去K573RF2またはK573RF5（ブルジョアアナログ-2716）を搭載したチップ上の2 KB。「モニター」制御プログラムが配置されています。 コンピューターのデバッグ中に、「モニター」の代わりに、診断テストがROMに書き込まれ、メモリ、文字ジェネレーター、およびキーボードの正しい動作をチェックしました。 したがって、このチップは通常、ソケットにインストールされます。




RAM：オリジナルバージョン-8チップK565RU3で16 KB。 改訂版では、16個のK565RU6超小型回路上の32KB（2階で互いにはんだ付けされています）。 32 KBのRAMを搭載したRadio86RKコンピューターの通常の動作では、RAMの終了アドレスの変更に関連して「モニター」ROMのファームウェアを変更する必要がありました。




ビデオサブシステム：KR580VG75 + KR580VT57チップの束の上に構築。 キャラクタジェネレータのシンボルは、ROMチップK573RF1にステッチされています。 この超小型回路は、電源に3つの電圧（+ 5、+ 12、-5ボルト）を必要とし、電源電圧の1つが消失すると故障することがよくありました。 回路に小さな変更が加えられたため、多くのハムがK573RF5マイクロ回路に置き換えました。 KR580VG75超小型回路によりカラー画像を実現できるという事実にもかかわらず、64文字に対して25行のテキスト白黒モードのみが利用可能です。 設計を簡単にするために、彼らはカラー画像を放棄することにしました。



KR580VG75チップが不足しているため、一部の無線アマチュアは、それを約20個の一般的な回路を含む別のボードに置き換えました 。 ただし、このような交換後、ビデオメモリ、特にゲームに直接アクセスするプログラムは、コンピューター上で正しく機能しませんでした。 適応なしでは、ROM機能を介してビデオメモリにアクセスしたプログラムのみが機能しました。



モニターとして、従来のテレビまたはビデオモニターElectronics MS 6105が使用されました。




キーボードは回路への接点を持つマトリックスであり、KR580VB55マイクロサーキットの1つに作られた入力/出力ポートを介してCPUによって完全にサービスされました（コンピューターには2つのマイクロサーキットがあります）。 最も頻繁に使用される電卓のキーボードのリードボタンは、滑らかさには違いはありませんが、高い信頼性がありました。




EI / DIコマンドによって切り替えられたINTEプロセッサ出力（「割り込み解像度」信号が生成された）は、音源として使用されました。 コンピューターで割り込みが使用されなかったため、この出力は無料でした。 よく消えないでしょ？




電源：誤動作により、非常に少なく高価なマイクロ回路が故障する可能性があるため、最高の信頼性で設計されたコンピューターの最も重要なコンポーネントの1つ。 実際、Radio86コンピューターは3つの電源電圧（-5V、+ 5Vおよび+ 12V）を使用し、最初に-5Vの電圧を使用し、次に+ 5Vおよび+ 12Vを最初に供給しました。 それらは逆の順序で削除する必要があります。 また、すべての電圧の同時供給と除去が許可されており、そのうちの1つが消えた場合、他のすべての電圧を除去する必要があります。




家庭用テープレコーダーは、プログラムとデータを保存するための外部デバイスとして使用されました。





周辺機器（プリンターなど）は、2番目のチップKR580VV55の空きポートに接続されました。



Radio86RKの外観。



自家製のコンピューターの外観について話すことは完全に正しいとは限りません。

誰もが最善を尽くして、手に入れたものからPK86を主張しました。 一部の人はまったく気にせず、コンピュータをケースなしで完全に放置し、それから彼らはそれを手入れして大事にしました-神は禁じ、お茶を注ぐか、静電気で殺します。



ゼンのアマチュア無線愛好家は、適切なサイズのソビエトの計算機（たとえばIskra 111M ）から発想を隠そうとし、ハンダごてを手に持つだけでなく、プレキシガラスで作業する方法を知っていた人々は、独自のケースを作成しました。





Iskra 111M電卓のハウジングにあるRadio86。 http://sfrolov.livejournal.comから撮影した画像



次に、Radio86RKのインスタンスのケースバージョンを見てみましょう。





プレキシガラス製で、黄色に塗られています。 ボタンと2つのLED用の穴は、上部ケースカバーに切り取られています。



キーボードは、ソビエト電卓のボタンから入力されます。 手動でボタンに文字と矢印を描く必要がありました。そのようなボタンの上には、碑文が消去されないようにする透明なキャップがあります。 ボタンの数字は、各自尊心のある電卓のボタンにあらかじめ印刷されているため、描画する必要はありませんでした。





ケースの側面には何もおもしろいものはありませんが、背面の壁には必要なすべてのコネクタがあります-テープレコーダ、TV、「リセット」ボタン、および電源用のコネクタを接続するためのコネクタです。





電源ハウジングもプレキシガラスで作られ、黄色に塗装されています。



Radio86の電源コネクタと電源ボタンに加えて、PSUケースの前面壁には+ 5Vコンセントがありますが、これは原則として不要です。 しかし、これはバグではなく、機能です。





電源の側壁は巨大なラジエーターです。 これにより、冷却の問題が解決されました。



中身は何ですか？





ケースの上部を取り外します-ネジはなく、引き上げるだけです。





キーボードがコンピューターのメインボードにループで接続されています。





4本のネジを外し、キーボードからケーブルを外すと、Radio86RKコンピューターのメインボードに到達できます。 プリント回路基板は両面で、非常に質的にエッチングされています。 ただし、配線エラーのため、一部のトラックは通常のワイヤを使用して切断および配置されます。





ソケットにインストールされているチップが多すぎます。 たとえば、プロセッサ。





RAMはソケットにも入れ子になります。 はい、2階です。





サインジェネレーター。 国内のチップK573RF1の代わりに、ドイツのアナログU555で作られています。 とても信頼できます。





チップK573RF5（ROM "モニター"）も同様のチップ2716に置き換えられています。





クロックジェネレーターKR580GF24。 ソケットにも。





キーボードI / Oポート 近くには、同じチップ用の空のソケットがあり、8ビットの周辺機器を接続するように設計されています。





ビデオコントローラー



このすべての美しさは、Radioマガジンの説明と図に基づいて、ラジオのアマチュア自身が取り上げなければならないことを思い出してください。 また、骨の折れる組み立てと調整に役立つのは、そこに公開されている参照表です。





こんな感じでした。



原因時間、楽しい-1時間。



多くの人は、Radio86RKホームコンピューターを組み立てることに成功した過酷な叔父たちが演じたことにおそらく興味を持つでしょう。 エンターテインメントなど、かなり適切な数のプログラムがRK86用に作成されたことがわかりました。



それらを見てみたいですか？ お願い！



まず、Radio86をモニターとテープレコーダーに接続します。 次に、お気に入りのゲームのカセットを挿入し、コンピューターのIボタン（「入力」という単語の頭文字）を押し、その後、VKキー（キャリッジリターン、Enterキーのアナログ）を押すことを忘れないでください。再生用のテープレコーダーをオンにします。 モニターは魔法のように画面上のすべての表示を停止し（これはコンピューターの機能です）、プログラムをロードした後にのみ画像が再び表示されます。 ダウンロードが完了しました。



プログラムを開始するには、Gボタン（「GO」という単語の最初の文字）を押し、VKキーを押してコマンドを確認する必要があります。 プログラムがあるメモリの開始アドレスを指定する必要がある場合がありますが、これは非常にまれです。
