Oculus GoBlack-Oculus Goの改善方法

Oculus VRの創設者の1人であるOculus Riftバーチャルリアリティヘルメットの発明者であるPalmer Luckyのブログから。







Oculus Goは、多くの製品と同様に、コスト、リリーススケジュール、品質、およびコンピテンシーの妥協点です。 これは明らかに市場で最高のスタンドアロンVRプラットフォームですが、技術的な知識が少しでもあれば、さらに良くすることができます。



過去数ヶ月間、私はGoの改良に取り組んでおり、結果として得られたデバイスは、その使用に関して大きな前進でした。 GoBlackの後、もう戻りたくありません！ 更新のリストと、それぞれの動機付けの説明を次に示します。



色



Oculus GoのストックモデルのハウジングはABSプラスチックでできており、さまざまな合成材料の混合物がフェースとファスナーとの接合部に使用されています。 デバイスの色はキュートな主流のグレーで、鋳造プロセスの小さな欠陥をうまく隠しています。 私は黒いVRヘルメットの大ファンなので、黒いペンキがアップデートのリストの一番上にありました。 これは、コントラストやその他の歪みを制御する重要な役割を考えると、ヘルメットの内側のすべての表面にとって特に重要です。 ディスプレイ、鼻腔スリット、またはその他の隙間からの光は、これらの表面で反射され、レンズと光学キャビティに到達します。これらの光線が気に入らない場合は、つや消しの黒い表面を理解してください。



ケースを外側からペイントすると、デバイスが重くなるだけで、顔に隣接する部分の問題は解決されないため、すべてをポリエステルの染料で覆うことにしました。 注意 ：これを行うときは注意してください。 プラスチックを染料でコーティングするには、沸点に近い高温の液体で作業する必要があります。温度をわずか数度超えると、プラスチックが変形します。 温度が低すぎる場合、染料はプラスチックに含浸しません。 そして、あなたはやけどをすることができます。





飲まないで！ 繊維を冷水ですすぎ、濃いプラスチック染料を肌に付着させないでください！





左側はOculus Goの元の前部、右側はGoBlackの染料に浸した部分です。 どれがより少ない光を反射するか推測してください。





また、レンズの周りのクリップを交換して、突起が少なくなるようにしました。 ヘッドレストを設計する際は、少量の光でも反射する表面の使用を常に最小限に抑えてください。 これが、Riftのすべてが、レンズに至るまでマットな黒の生地で仕上げられ、開いているプラ​​スチックの面積が最小限に抑えられる理由の1つです。



重さ



Oculus Goはかなり重いです。 紙の上では、その重量はRiftを上回りますが、これはおもに生地ベルトが原因で起こります。生地ベルトは、重量感をあまり加えず、頭の動き中に肌に圧力をかけません。 顔の上に置かれたデバイス自体の重量は400 gを少し超えています。



いくつかの変更により280 gに減らすことができ、ほとんどの浮き彫りはデバイスの前端にあり、最も感じられます。 主観的には、特に移動中に体重が半分に減少したようです。 そして、ここに私がやったことがあります：



バッテリーの交換 ：バッテリーをヘルメットの前面から取り外し、頭の上部に移動し、ベルクロとトップストラップを接続します。 当初、このデバイスは2時間動作する2600 mAhの特に容量の大きい18650バッテリーではありませんでしたが、3500 mAhで3時間以上動作できることにお金をかけることにしました。 スプリングプラグ[pogo-pinコネクタ]を介して接続されているため、再生を続けたいときに新しいバッテリーに簡単に切り替えることができます。 ゲーム中に充電する方がはるかに便利です。これは過熱につながる可能性があります。 一日中ヘルメットを使用できるバッテリーモジュールを作るのは非常に簡単ですが、私はこれをやりたいとは思いません。











熱放散/ヒートシンク ：Oculus Goの冷却システムは受動的であり、Qualcomm Snapdragon 821 SOCからの熱をアルミニウムプレートに分配するヒートパイプを使用します。 残念ながら、このシステムは重く、ヘルメットの前面に配置されているため、使用の快適性への影響が最大になります。 私はヘルメットの重量に厳しくアプローチします-ヘッドマウントディスプレイは、一見小さな重量削減から大きな利益を得るという点でロケットに似ています。



この問題を解決するために、まったく新しい冷却システムを設置しました。 ヒートパイプははるかに短く軽量であり、シングルチップシステムから小さなファンで冷却される小さなアルミニウムフィンの近くのアレイに熱を逃がします。 システムはマザーボードから給電され、アルミニウムプレートを偏光プラスチックに交換しました。 結果は良いように思えます。







すべてのカバーとスクリーンも取り外しました 。 数グラム節約しましたが、私はそのような美学、各チップ上のすべての周辺機器を見る能力が好きです。



プラスチックの量を減らす ：もっと極端な方法を選ぶこともできますが、信頼性、耐久性、そして店のように見える外観が欲しかったのです。 冷却システムとフロントパネルをサポートするために使用された大量のプラスチックをなんとか除去しましたが、私の友人GOROmanはGoの変更でこのアイデアを最大限に引き出しました！



その他



瞳孔間距離の調整 ：多くの理由でこのオプションを拒否しました。 この距離はOculus Goで固定されているため、解像度を上げても距離が長すぎたり小さすぎたりする人には役立ちません。 私の距離は非常に大きく、約70 mmであり、Oculus Goは私にはあまり適していません。画像フィールドの幾何学的歪み、スケール、曲率が正しくないため、目の疲れやぼやけが生じます。 Goを定期的に使用するほとんどの人は、平均に近い瞳孔間距離を持っていると思います。Riftをほぼすべての男性と女性に合うように設計し、それを誇りに思っているのは残念です。 より高い解像度では、適切な光学調整がより重要になっています。 バーチャルリアリティ革命は、目の間の距離に関係なく、すべての人がアクセスできるようにする必要があります。



モバイル用キーボード ：Oculus Goの最も興味深い機能の1つは、 仮想デスクトップのサポートです。 巨大なモニターを使用して、Windowsのほぼどこでも作業できます。 お気に入りのメカニカルキーボードの1つであるPCを組み立て、Happy Hacking Keyboard Professional Type 2 S（未書き込みのキーを使用）（VRには必要ありません、まだ表示されていません！） フルサイズのキーボード+ Windows + GoBlackは、どのラップトップよりも優れています！







6DOF、Oculus Quest、NOLOなど ：現在、主に瞳孔間距離の問題のため、GoBlackはほとんど使用されていません。 Oculus Questに調整機能があることを非常に嬉しく思い、本当に楽しみにしています！ 必要な場合は、私が話した新しい前庭固定のかなりの時間を待ちます。





Oculus Goのサードパーティの追跡システムであるNOLO VRの最新バージョンには非常に感銘を受けました。これは、いくつかのファームウェアの更新後、はるかによく機能し始めました。
