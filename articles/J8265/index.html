サイトのSite24x7または24/7モニタリング

複数のサイトをサポートする場合、,するユーザーの手紙や電話から数時間たってからではなく、すぐに切断されたサイトがあることを知りたいと思います。 また、「ホスティングのサイトは1日24時間、週7日間利用可能である」と標準的に宣言しているホストプロバイダーのうち、真実が伝えられているものと、戸惑っているホストプロバイダーを確認したいと思います。



これを行うには、サイト監視サービスを使用します。 最高の1つはSite24x7です。

アカウントを登録した後、15日間すべての機能を利用できます。この期間後は、特定の制限付きで2つのサイトを追跡できる無料アカウントに滞在するか、プロフェッショナルアカウントに追加料金を支払い、さまざまなより洗練された方法で無制限の数のサイトを監視できます。

Site24x7でできることは次のとおりです 。

  • HTTP / HTTPSリクエストをサイトに定期的に送信して、機能を確認してください。 リクエストは、無料アカウントの場合は60分間隔で、プロフェッショナルの場合は5〜60分（設定）間隔で送信されます。

    サイトの応答時間を追跡し、指定された値を超えた場合に報告します。

    HTTP要求のテストシーケンスを記録し、定期的に再生します。 取引システム、電器店などのパフォーマンスをチェックするときに役立ちます。

    サイトページの変更を追跡します。 これは、ページのコンテンツの変化の割合を設定できることを意味し、この割合を超えると、それに関するメッセージが表示されます。 したがって、サイトのページへの不正な変更、さまざまな厄介なトロイの木馬などをキャッチできます。

    許容される応答時間を超え、サイトのページやその他の監視されているイベントを電子メールやSMSやRSSで変更することで、サイトのアクセス不能に関する即時通知を受け取ります。

    特定の期間におけるサイトのアクセス可能性/アクセス不能時間、最大アクセス時間、最小アクセス時間、平均アクセス時間、および対応する図に関する情報を含む、毎日および/または毎週のレポートを受け取ります。

    もちろん、次のレポートを待たずにいつでもアカウントにログインして、必要な情報をすべて入手できます。

    無料で楽しいアプリケーション-ツールセクションのツール-ドメイン名でDNSおよびIPを検索（DNSルックアップおよびIPを検索）、任意のサイトの可用性を確認（可用性を確認）、最後に、国に正確にホスティングしているWebサイトの実際の場所を決定し、常に）都市（場所の検索）。
