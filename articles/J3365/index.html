Web 2.0

誰もが知っているように、ティム・オライリーはこの用語を定義し、説明しました。

Web2.0プロジェクトとしての地位を確立するリソースが現れ始めました。新しいスタートアップについて書くリソースがあります。

しかし、この定義は十分ではありません...



これが私が気づいたことです。 この任期が到来する前に、ルネット全体が静かに暮らしていたが、今ではある種のレースが始まった。 彼らは、「Web 2.0用のアプリケーションを備えた」サイトに気付き、そこで実行します。 誰もがオライリーの記事を長い間忘れていました。 1,500人の登録ユーザーがピックアップされ、停止されます。 さらに、彼らはサイトの開発の報道とオープニングでの発表、例えばHabréやインターネットなどでのみ視聴者を獲得します...

Web 2.0を支持する人もいれば、それは神話だと言う人もいます。 しかし、これはもはや神話ではあり得ない-世界中の多くの人々がこれについて議論している。 すなわち そのような用語が存在するかどうかは異なりますが、すでに他のユーザーと意見を共有し、受精し、育成し、投稿し、コメントする（つまり、コンテンツを作成する）習慣を形成しています。 プロジェクトの作成者は、作成された頭脳を管理するだけで、たとえば3年前よりもはるかに少ない記述を行います。 私たちは編集者、親愛なるユーザーになりました。 しかし、それは非常に興味深いものになりました。便利なフォーラムではなく、快適なインターフェースと便利な機能に変わりました。 RunetはWeb 2.0があるかどうかを議論していますが、すでに来ており、誰も別の名前を付けることはありません。

その名前でそれを集合的に認識し、Web2.0プロジェクトに2〜3年前に存在していたウェブサイトと何が違うのかという質問に答えましょう。



Web 2.0は...



1-これは、コンテンツ、サービス、およびプログラムであり、その価値はそれらを使用する人々の貢献（記事の評価）によって決まります。 この概念には、無制限の数の放送局が関与し、その中で誰もが主要なプレーヤー（参加者の評価）の1人になることができます。



2-これが最初のポイント+「Web 2.0のスタイルの」必須デザイン（明るい/暖色系、反射神経）です。



3は単なる設計です。



4は、サービスやデザインの内容に関係なく、トラフィックを収益化するためのビジネスモデルです。



5は退屈な議論であり、私は参加に興味がありません。
