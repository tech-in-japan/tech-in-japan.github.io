7月29日から7月31日まで、 オデッサは7回目のWebCamp会議を開催し、Web開発技術、プロジェクト管理、ビジネス開発に焦点を当てます。 伝統的に、イベントはいくつかの領域で構成されています。 今年は、 フロントエンド 、 Python 、 PHP 、 DevOps 、 PM 、 BizDevです。



WebCamp 2016には、ウクライナ全土および近隣諸国から600人以上の参加者が集まります。 会議は中級/上級レベルの参加者向けに設計されており、誰もが経験を交換し、アイデアを共有し、アフターパーティーで同僚と楽しい時間を過ごし、黒海でリラックスする機会があります。



私たちのFlyElephantチームは現在、積極的に会議プログラムの完成に取り組んでおり、あなたはまだ参加する機会があります。 Microsoft、Wargaming、Looksery、Upwork、Provectus、Ciklum、DataArt、Zillion Whales、SoftServe、YouScan、WebbyLab、Coderiver、CartFresh、RevJet、FlyElephant、Netpeak、DataRobot、KeepSolid、VertaMediaなどの企業の代表がレポートを作成します。
