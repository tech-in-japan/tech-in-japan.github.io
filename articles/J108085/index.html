Ubuntu 11.04 Natty narwhalのPPA経由で利用可能なcompizエンジンに基づくUnity

すでに、ubuntu 11.04のユーザーは、既知のすべてのcompizへのプラグインとして実装された統一インターフェースをテストできます。 したがって、このインターフェイスを実行するには、ネットブックエディションセッションで再起動する必要はなく、compizプラグインをオンにするだけです。







すぐに目立つのは、ランチャーが半透明になることです。







もちろん、すべてがスムーズに機能するわけではありませんが、これは新しいインターフェイスの初期のベータ版またはアルファ版にすぎません。

しかし、この動きは、Unityがプラグインになったため、他のすべてのCompizの作業が励まされるという点で励みになります。







たとえば、公開します！



インストールするには、ターミナルを開いて...



sudo add-apt-repository ppa：unity / ppa

sudo apt-get update

sudo apt-get dist-upgrade



そして、compizマネージャーがまだない場合は...



sudo apt-get install compizconfig-settings-manager



誰もがタンバリンと一緒にダンスを楽しんでいます））
