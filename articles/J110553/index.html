Kingmax Hercules Nano RAM Kitレビュー-ナノアーマーの古代ギリシャのヒーロー

RAM市場は、さまざまなプラットフォーム向けのあらゆる種類のソリューションで飽和しています。 現在、ユーザーはさまざまなオプションの真に膨大な選択肢を持っています-低遅延の高メモリ周波数、高遅延の高周波数、低遅延の低周波数、高遅延の低周波数。 実際、選択の問題は、特定のケースとユーザーの予算のサイズを決定することになります。 そのため、製造業者は、大型またはカラフルなラジエーター、照明モジュール、またはファンを備えた冷却システムを装備した形で製品に注目するさまざまな方法に頼ることがよくあります。

Kingmaxはこの秋、追加のラジエーターを必要とせず、2200 MHzの周波数で動作可能なHercules DDR3タイプRAMの最新シリーズを発表しました。 モデルの製造では、 ナノ熱放散技術を使用しました。 メモリチップから熱を除去するラジエーターの代わりに、モジュールはシリコンとグラファイトの特別なコーティングで覆われています。 このコーティングにより、従来のDDR3メモリモジュールと比較して熱放散が10％改善されます。





メモリモジュールは、青と青で作られた小さな箱に入っています。



前面には、2つのモジュールの1つが見える透明な窓があります。 最初に目を引くのは、適用されたナノ熱放散技術と生涯保証に関する情報です。 箱を開けると、両方のモジュールが高密度の多孔質材料にしっかりと固定されており、輸送中の損傷を防ぎます。



箱の背面には、メモリモジュールの主な特徴があります。 それらを以下の表に簡単にリストします。

RAMモジュールは、黒いテキソライトに基づいています。 各メモリモジュールには、16個のメモリチップ（各側に8個）が含まれています。 各モジュールの容量は2048 MBです。 キットの総容量は、両方のモジュールの容量に対応しています。 4096 MB。

各モジュールに貼られているステッカーは、モジュールのモデル、シリアル番号、メモリの種類、周波数、およびボリュームについても通知します。



モジュールの製造では、Kingmaxステッカー付きの6層HJ M1プリント回路基板が使用されました。



メモリチップには、Kingmax KFB8FFJXF-DXX-09Cというラベルが付いています。 実際、メモリチップはPowerchipによって製造されており、この場合は再マーキングについて話しています。



PCBでは、何らかの理由ではんだ付けされていない場所がLEDに割り当てられています。

マザーボードASUS Maximus III Formulaに取り付けられているメモリモジュールは次のとおりです。



Auto Kingmaxパラメーターをロードすると、Hercules Nanoは、周波数が1333 MHz、遅延が9-9-9-24 1Tの通常のモジュールとして定義されます。



XMPプロファイルを有効にすると、10-10-10-30 1Tの遅延で2200 MHzに等しいメモリ周波数が得られます。



次のパラメーターは、製造元のSPDに記録されます。





テスト構成とテスト方法

マザーボード Asus Maximus IIIフォーミュラ（Intel P55、Intel LGA 1156、BIOS 2001）

メモリモジュールの安定性のテストは、memtest86 +バージョン4.10ユーティリティによって実行されました。 動作周波数の最大値は、テストNo. 5が少なくとも10分間合格したことによって決定されました。 最終的な動作メモリ周波数は、2：10乗算器を使用して、BCLK周波数でプロセッサをオーバークロックして達成されました。 メモリ電圧は、1.65 Vに手動で設定されました。



メモリを操作する能力は、工場設定でXMPプロファイルをアクティブ化することで確認されました。また、プロセッサテストサンプルのオーバークロックは周波数BCLKが220 MHzに制限されていたため、メモリは低い「偶数」タイミングでテストされました。 さらに、PSCチップのメモリオーバークロックの特性を考慮して、特定のタイミング、つまりtRCD = tCL + 3での作業がテストされました。



試験結果

最初に、メーカーがXPMプロファイルに記録した設定でメモリセットの安定性がチェックされ、テストが正常に完了すると、正しい設定がさらに確認されました。



「偶数」タイミングでのオーバークロックの可能性を検討して、3つのモードをテストしました：7-7-7-21 1T、8-8-8-24 1T、9-9-9-27 1T。 テスト結果は、XMPに対応する設定でテスト結果が追加されたダイアグラムの形式で以下に示されています



メモリは、8-8-8-24の遅延で1823の周波数で、9-9-9-27で2035の遅延でテストに合格することができました。 このレベルのメモリについては、かなりまともな結果です。



より特徴的なPSCメモリチップのタイミングでの可能性を研究したところ、それを完全に明らかにすることはできませんでした。 主な制限は、低メモリ乗数でした。 Intel Core i7 8XXの2:12とは異なり、Intel Core i5 750プロセッサーの最大値は2:10のみです。



8-11-8-27および9-12-9-27の遅延を持つ周波数の値は、これらの条件下で達成された値を示しています。 ただし、このメモリとCore i7-8XXプロセッサを使用して得られた結果を調べると、1.65から1.8 Vの供給電圧で9-12-9-27の遅延で2400-2500 MHzのマイルストーンを克服した結果を見つけることができます。これはすでに非常にまともな結果です。



上記のすべてから、次の結論を導き出すことができます-Kingmaxは、ラジエーターの欠如と新しいユニークな技術の使用だけでなく、非常にまともなオーバークロックの可能性でも注目を集める非常に興味深い製品を生産することができました。



テスト結果に基づいて、Kingmax Hercules Nano RAM KitはEditor's Choice Awardを受賞しました。
