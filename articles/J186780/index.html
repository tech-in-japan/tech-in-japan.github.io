Intel Integrated Performance Primitives 8.0に対応

多くのソフトウェア開発者は、最適化されたデータ処理とマルチメディアのための機能セットであるIntel Integrated Performance Primitives（Intel IPP）ライブラリをよく知っています。 インテル®IPP機能は、プロセッサーのマイクロアーキテクチャーの機能、特にベクトル化機能を積極的に使用して、アプリケーションの速度を大幅に向上させます。 さらに、プログラマーが既製の「ブリック」を手元に置いており、車輪を再発明する必要がないため、アプリケーションの直接開発が加速されます。 インテル®IPPは、画像変換、統計、コンピュータービジョン、その他多数の分野で広く使用されている数百の機能を実装しています。これらはすべて最適化、デバッグされ、ロイヤリティの負担はありません。

そして今、製品の機能はさらに拡張されています-Intel IPP 8.0の新しいバージョンがリリースされました 。



Intel Integrated Performance Primitives 8は、バージョン7とは大きく異なります-Windows 7からWindows 8に劣らず、より良いことを願っています。

  • IPPは、静的なシングルスレッドライブラリを介して、Windows 8の新しいUI（以前の「METRO」）をサポートするようになりました。
  • IPPパッケージにOSX *サポートを追加しました（以前はOSX * Composerの一部としてのみ利用可能）
  • 新しいHaswellマイクロアーキテクチャの最適化が多くの機能に追加されました。
  • SSSE3以降を使用した画像処理機能の追加の最適化。
  • 主な配布には、以前に別途提供されたIPPの使用例が含まれています。 新しい例が追加されました。
  • Androidで使用するための拡張ドキュメント。
  • そして最後に、IPPは異種混合になります。つまり、Intel Graphicsで実行できるようになります。 プレビューモードでは、対応するバージョンのIPP が利用可能です 。また、非同期C / C ++ライブラリとOpenCLベースのライブラリの2つのバージョンがあります 。


Intel IPP 8.0は、「軽量」ディストリビューションとして配布されます。 サポートされているオペレーティングシステムは、Windows（XP、7/8、サーバー）、Linux（Redhat / CentOS 5/6、Fedora 18/19、SLES 10/11、Ubuntu 12.04 LTS / 13.04、Debian 6/7）、MacOS X（10.8。 2 / 10.9）。 商用版の費用は199ドル（アカデミック$ 99）です。
