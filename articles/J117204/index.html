Flash Player 10.2 + vdpau + 64ビット

ご存じのとおり、64ビットLinuxシステム用のAdobe Flash Playerの実際のバージョンはまだありません。 はい、スクエアがあります。 ただし、更新は行われず、GPUビデオのデコードも行われません。 32ビットバージョンを使用する必要があります。



それは、彼女のGPUデコードビデオがそのままでは機能しないためです。 どうする



実際、64ビットシステムの多くのユーザーにとって、これが明らかになることはほとんどありません。 しかし、あまり考えていなかった人や、私が32ビットシステムから64ビットシステムに切り替えたように、libvdpauライブラリの32ビットバージョンをインストールするだけで済みます。



これを行う方法はすでに微妙です。 32ビットのドライバーパッケージを手動で選択し、目的のファイルを/ usr / lib32に配置できます。 多くはディストリビューションに依存します。 Ubuntuには、簡単な方法があります-getlibsユーティリティを使用します。



wget frozenfox.freehostia.com/cappy/getlibs-all.deb

sudo dpkg -i getlibs-all.deb

sudo getlibs -p libvdpau1







3つのチーム-YouTubeのフラッシュは、フレームのブレーキとドロップを停止します。



メモはオリジナルのふりをしていません。 32ビットライブラリを使用する可能性が思い浮かんだ後、私はGoogleを使用して、 ここで解決策を見つけました。 私は、まだ知らない人に（彼らがまだ存在するかどうか）知らせることにしました。
