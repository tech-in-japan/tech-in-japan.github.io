コンピューターとごみ箱のコンテキストメニューへのコマンドの追加（続き）

Habréの記事を公開した後、Explorerのコンテキストメニューに独自のコマンドを追加します。いくつかのプログラムは何らかの方法でコマンドを追加することができますが、定期的に手紙が届き始めました。著者は追加したコマンドがコンピューターまたはゴミ箱フォルダーのコンテキストメニューに表示されない理由を説明するように頼まれました



質問は非常に自然で、私自身も数年前にWindowsレジストリを調べたときに同様の質問をしました。 各文字に応答しないように、Windowsのコンテキストメニューのトピックに関する短い続きを書くことにしました。



まず、フォルダーとの特定の類似性にもかかわらず、コンピューターとゴミ箱は特別なプロパティを持つ特別なWindowsオブジェクトであることを理解する必要があります。 実際、Windowsにはそのようなオブジェクトが数百あります。 HKEY_CLASSES_ROOTブランチにあるWindowsレジストリには、特別なCLSIDセクションがあります。 このセクションに進むと、中括弧で囲まれた一連の数字で構成されるわかりにくいセクションが含まれていることがわかります。 これらはクラス識別子（クラスIDまたはCLSID）であり、60バイトの数字です。 簡単にするために、この番号を電話番号として表すことができます。 電話番号が1人の人物に対応するように、CLSID番号は特定のオブジェクトに対応します。



ビジネスに取り掛かろう。 そのため、コンピューターアイコン（以前のマイコンピューター）のコンテキストメニューに新しいコマンドを追加するには、その識別子を見つけて、シェルセクションに新しいサブセクションを追加する必要があります。 Computerオブジェクトは、識別子{20D04FE0-3AEA-1069-A2D8-08002B30309D}に対応しています。 したがって、コマンドをコンピューターオブジェクトのコンテキストメニューに追加するには、メニューに表示されるHKCR \ CLSID \ {20D04FE0-3AEA-1069-A2D8-08002B30309D} \シェルセクションに新しいセクション（メモ帳など）を作成する必要があります。 作成されたセクションで、 コマンドサブセクションを作成します。このサブセクションでは、実行するプログラムを既定のパラメーターで記述します。 この場合、これは文字列notepad.exeになります。



フォルダーを使用した例のように、作成したメモ帳セクションのデフォルト値を編集して、より便利な名前を表示できます（例： メモ帳の実行 ）







チェック-任意の場所（[スタート]ボタンメニュー、デスクトップまたはエクスプローラーのアイコン）でコンピューターオブジェクトを選択し、コンテキストメニューに[メモ帳の開始]コマンドが表示されることを確認します。







コンピューターのコンテキストメニューについて少し説明します。 サードパーティのプログラムがそのコマンドをコンピューターのコンテキストメニューに追加した場合、このコマンドを探す場所を知って、それを削除できることは明らかです。 しかし、無理をしないでください。 たとえば、 Manageコマンドを削除する場合、 Manageサブキーを削除する必要はありません。 HKCU \ Software \ Microsoft \ Windows \ CurrentVersion \ Policies \ ExplorerセクションのNoManageMyComputerVerbパラメーターを使用するか、グループポリシーを使用するだけで十分です（このパラメーターの詳細については、Windowsレジストリの電子ディレクトリを参照してください。この場合、安全な方法で不要なコマンドを非表示にします。



NoPropertiesMyComputerパラメーター（電子ディレクトリ内）を使用して、コンピューターオブジェクトのコンテキストメニューから[ プロパティ ]メニュー項目を非表示にすることもできます 。



Cartオブジェクトでも同じことを行います。 ごみ箱の識別子は番号{645FF040-5081-101B-9F08-00AA002F954E}です。 コマンドサブセクションを使用して、見つかったCLSIDの新しいセクションを作成しています。 メモ帳を実行する退屈な例の代わりに、 HabrahabrホームページでGoogle Chromeを起動してコマンドWent to Habrを作成しました（C：\ Users \ tehnolog \ AppData \ Local \ Google \ Chrome \ Application \ chrome.exe http://habrahabr.ru） 。







完全を期すために、[プロパティ]コマンドを非表示にするために追加します。NoPropertiesRecycleBinパラメーターはごみ箱のコンテキストメニューで使用されます。
