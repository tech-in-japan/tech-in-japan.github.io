サムスンギャラクシーAの全金属ラインがロシアに届きました

こんにちは、Habr！



12月5日、モスクワは、Samsung GALAXY Aの全金属ケース、つまり最初の「ツバメ」である5インチのGALAXY A5と4.5インチのGALAXY A3のスマートフォンの新しいラインのプレゼンテーションを開催します。 金属製のスマートフォンは、5メガピクセルのフロントカメラによって区別され、現在、サムスン製の最も薄いモバイルデバイスです。厚さは、それぞれ6.7 mmと6.9 mmです。







Samsung GALAXY A5シリーズの古いモデルには、クロック周波数1.2 GHzのクアッドコアQualcomm Snapdragon 410プロセッサー、5インチのSuper AMOLEDスクリーン（1280x720）、13メガピクセルのメインカメラおよび5メガピクセルのフロントカメラが搭載されています。 また、スマートフォンには2 GBのRAMと16 GBの内部メモリがあり、microSDカードスロットを使用して64 GBに拡張できます。 デバイスはLTEネットワーク（Cat4）をサポートします。 バッテリー容量は2300 mAhです。 スマートフォンは、TouchWizを搭載したAndroid 4.4 KitKatで動作します。 マルチウィンドウモードでは、別々のアプリケーションの2つのウィンドウを同時に画面に表示できることに注意してください。 モデルサイズ：139.3 x 69.7 x 6.7 mm 重量-123 g



仕様GALAXY A5：

携帯電話： 4G：LTE Cat 4 150/50 Mbps; 3G：HSPA + 42.2 / 5.76 Mbps

プロセッサー：周波数1.2 GHzのクアッドコアQualcomm Snapdragon 410

画面： 5インチHD Super AMOLED（1280x720）

オペレーティングシステム： Android 4.4（KitKat）

カメラ：メイン-13メガピクセル、オートフォーカス、LEDフラッシュ。 正面-5メガピクセル

インターフェイスと接続： WiFi 802.11 a / b / g / n / ac、Bluetooth：4.0（BLE、ANT +）、NFC

ナビゲーション： A-GPS / GLONASS

センサー：加速度センサー、ジェスチャーセンサー、デジタルコンパス、磁場センサー

メモリ：内蔵メモリ-16 GB、microSDカードスロット（最大64 GB）。 RAM-2 GB

寸法と重量： 139.3 x 69.7 x 6.7 mm; 123グラム

バッテリー： 2300 mAh（標準リチウムイオン）







Samsung GALAXY A3は、周波数1.2 GHzの同様の4コアQualcomm Snapdragon 410プロセッサー、4.5インチqHD Super AMOLEDディスプレイ（960x540）、8メガピクセルのメインおよび5メガピクセルのフロントカメラ、1 GB RAM、16 GBの内部メモリを受け取りましたメモリ+最大64 GBの拡張用microSDスロット、LTEネットワーク（Cat4）およびTouchWizを備えた同じAndroid 4.4 KitKatプラットフォームのサポート。 スマートフォンのバッテリーの容量は1900 mAhです。 寸法：130.1 x 65.5 x 6.9 mm、重量は110.3 g。



仕様GALAXY A3：

携帯電話： 4G：LTE Cat 4 150/50 Mbps; 3G：HSPA + 42.2 / 5.76 Mbps

プロセッサー：周波数1.2 GHzのクアッドコアQualcomm Snapdragon 410

画面： 4.5インチHD Super AMOLED（960x540）

オペレーティングシステム： Android 4.4（KitKat）

カメラ：メイン-8メガピクセル、オートフォーカス、LEDフラッシュ。 正面-5メガピクセル

インターフェイスと接続： WiFi 802.11 a / b / g / n / ac、Bluetooth：4.0（BLE、ANT +）、NFC

ナビゲーション： A-GPS / GLONASS

センサー：加速度センサー、ジェスチャーセンサー、デジタルコンパス、磁場センサー

メモリ：内蔵メモリ-16 GB、microSDカードスロット（最大64 GB）。 RAM-1 GB

寸法と重量： 130.1 x 65.5 x 6.9 mm; 110.3グラム

新しいアイテムは、白、黒、金、銀、ピンク、水色の複数の色で表示されます。 すでに12月に、Samsung GALAXY Aラインが国内の棚に表示されるはずです。 公式価格はまだ明らかにされていません。
