米国はパテントトロールに対する新しい法律を検討しています

過去1年間、米国議会はパテントトロールの問題を解決するためにいくつかの法律を提案しています。 新しい法案「2013年のイノベーション法」は、以前に提案されたアイデアを組み合わせており、採用される可能性が最も高くなっています。



法律は悪い特許を禁止することはできませんが、少なくとも最も悪質なトローリングを止めるでしょう。 これは明らかに正しい方向への一歩です。 さらに重要なことは、多くの議員が最終的に特許制度に重大な問題の存在と特定の行動の必要性を認識したことです。



パテントトロールは、特許の購入に従事しており、疑いを持たない企業や革新者に対する知的財産の積極的な使用に関するビジネスを構築しています。 この場合、非常に広範な作用範囲の非常に疑わしい特許が使用されます。 たとえば、スキャナーユーザー1人あたり1,000ドルのレートで永久ライセンスを発行するという提案のあるデジタルワークフローの特許 。



それにもかかわらず、多くの企業は、法的な費用とリスクが高すぎるため、訴訟を法廷外で解決することに同意しています。 時には例外があります-一部の企業は法廷でトロールと英雄的に戦い、訴訟に勝ち、勝利を達成します。 負けた側は勝者の費用を支払わないため、勝利はピュロスです（先を見据えて、新しい法律はそれを直すべきです）。



したがって、訴訟は非常に危険で収益性の低いビジネスです。 たとえば、昨年の米国のパテントトロールに対する勝利の後、カスペルスキーは、40社を超える他のすべてのウイルス対策開発者が連絡を取らず、特許ポートフォリオに登録したと述べました。 LK自身は3.5年間訴え、その道徳的満足のために心から支払いをしたに違いありません。



問題は本当に関連性が高まっています。 2つのグラフを与えるだけで十分です）。



1つ目は、「非実務会社」（トロール）が関与する特許裁判所の数を示しています。







第二に-近年発行された発明の特許の数（実用特許、工業デザインの特許、GMOはここでは考慮されていません）、数千。







それで、新しい法律は何をしているのでしょうか。



1. 詳細情報 。 訴訟を提起する場合、必須の要件は、どの特許が侵害されており、どの特定の製品がこれらの特許をどのように侵害しているかに関する情報を提供することです。 この規則により、トロールは大量の請求を提出することが難しくなります。被告ごとに個別の文書を準備する必要があります。



2. 訴訟費用の換算 。 裁判所は、勝者が被った法的費用を補償するために敗者から要求する権利を受け取ります。 これは、トロールが被害者を脅迫することに成功した重要なポイントです。彼らは、法廷で勝ったとしても、トロールが要求する以上の金額を支払うことを理解しています。



3. 透明性 。 法律は、原告が自らの利益のために漏らすことを厳しく要求している。 今日、ほとんどのトロールは特定の特許の下で作成された1日のアフィリエイトです。 会社が子会社である場合、裁判所の敗北後の1日の会社が破産を宣言し、費用の補償を拒否した場合、被告は法的費用について被告に補償するために親会社を裁判所に連れて行く権利を有します。



4. テクノロジーユーザーに対する遅延事例 。 法律は、この製品の製造業者に対する訴訟がある場合、裁判所はシリアル製品の企業ユーザーに対する訴訟の検討を一時停止することを要求しています。 たとえば、2011年には、「公共の場所でWiFiルーターを使用する」ために、数百のホテル、カフェ、レストランがパテントトロールInnovatio IP Ventures、LLCに2,300ドルから5,000ドルを支払いました。 これは、トロールが製造業者に対する大規模な訴訟ではなく、テクノロジーユーザーに対して数百件の小さな請求を提出するための新しい戦術を適用した最初のケ​​ースの1つでした（チャンスはありませんが、ここでは少なくとも10万ペニーを回収できます）。



5. 裁判所への文書の義務的開示の改革 。 現行法の下では、原告は、被告がプロセスへの参加に同意し、裁判所手数料およびその他の費用（多くはリスクを冒さない）を支払うまで、特許番号を指定せず、他の情報を提供しない権利を有します。 特許に関する情報が最初から発見された場合、トロールのばかげた主張がすぐに明らかになる可能性があります。



6. 発行後の検査 。 法律は、すでに付与された米国特許に異議を唱える能力を拡大しています。



電子フロンティア基金は法案を承認し、これまでにリリースされたトロルに対する最高の法案と呼びました 。 財団は、すべての米国市民が2013年イノベーション法への支援を要求する議員に手紙を書くことを奨励しています。
