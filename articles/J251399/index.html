PHDaysの新機能：スーパーコンピューター保護、iOSアプリケーションセキュリティ、エクスプロイトセール





少し前まで、Call For Papers情報セキュリティフォーラムPHDays Vの第1波が開催されました 。 5月26〜27日にモスクワで行われる公演の新しい部分に注目してください（ 最初と2番目の発表はHabrahabrで読むことができます）。 講演者は、iOSアプリケーションのセキュリティを強化する方法、スーパーコンピューターがハッカーを引き付ける方法、およびそれを保護する方法、ゼロデイ脆弱性のエクスプロイトの売り手と買い手の関係について話します。



デバッグ自動化



Alexander TarasenkoがWinDbgツールを使用したデバッグの自動化について説明します。 生徒は、組み込みのWinDbgエンジンを使用して、PythonおよびPykd拡張機能を使用して、実用的なスクリプト作成スキルを習得します。 このレポートは、非標準のデバッグツールの使用を必要とするソフトウェアのコード研究者および開発者にとって興味深いものです。



IOSアプリのセキュリティ



エミレーツ航空のOWASPメンバーおよびセキュリティスペシャリストであるプラテクジャンチャンダニは、iOSアプリケーションのエクスプロイト作成に関するマスタークラスを実施します。 デモでは、スピーカーは典型的な脆弱性を含む特別に開発されたアプリケーションを使用します。 リスナーは、iOSアプリケーションの開発段階でセキュリティレベルを上げる方法を学習します。 入門編の最後で、誰もがアプリケーションのテストに挑戦できます。



スーパーコンピューターを見る



ドイツの情報セキュリティ企業ERNW Felix Wilhelm（Felix Wilhelm）とFlorian Grunov（Florian Grunow）の従業員は、いくつかの有名なスーパーコンピューター（IBMのWatsonなど）で使用されているIBM General Parallel File Systemのファイルシステム、そのアーキテクチャ、脆弱性について説明します。 このシステムの人気は、保存されたデータだけでなく、強力なコンピューターのコンピューティングリソースへのアクセスを獲得する能力にも関心があるサイバー犯罪者の目標となっています。 スピーカーは、2つの実際のIBM GPFSセキュリティエラーの操作を実演します。



エクスプロイトセール



BeeWiseプロジェクトの創設者であり、secYOUreのチーフコンサルタントであるAlfonso De Gregorioは、ゼロデイ脆弱性のエクスプロイト市場で一般的なモラルについて説明します。人気サイトで開発されたツールの売り手と買い手の関係について説明します。



5番目の速度でハッシュをハッキングする



PHDays Vでは、有名なパスワードクラッキングユーティリティJohn the Ripperの開発に参加し、GUIインターフェイスをサポートしているAlexey Cherepanovが、コード生成メソッドを使用したクラッキングハッシュの高速化について説明します。



早くて便利



標準の講義に加えて、PHDays Vプログラムでは、豊富でダイナミックな15分間のスピーチで構成される、広範なFastTrackが計画されています。



フォーラムの訪問者は、基地局のなりすましによるGSMネットワークへの攻撃がどのようにGSM電話を聞くことができるかを学びます-NRNU MEPhIエンジニアリングセンターの従業員がこのハリコフハリコフについて話します。



さらに、Kudelski Securityの暗号作成者でありセキュリティの専門家であるSylvain Pelissierは、GNU / Linux eCryptfsファイルシステムを例として使用して、ファイル暗号化がパスワードの解読に役立つことを示します。



デニス・ゴルチャコフの話から、学生は通信事業者のネットワークで支払い詐欺に対抗する方法について学びます。 Android OSでのウイルス分析、感染したデバイスからのボットネットによるコントロールセンター（オンラインとSMS）の識別、データコレクター、およびバッテリーアカウントのためのハードウェアとソフトウェアの複合体についてです。



スピーカーになる方法



2月16日、 Call For Papersの第2波が始まりました。 作品の受け入れは3月31日まで続くため、今後のPHDaysで講演者になる機会があります。



また、フォーラム（ HITBカンファレンス）のCFP友人にも参加することをお勧めします。
