小さなオンラインストアを開くときに探すべきこと





ロシアでは、電子商取引市場が非常に急速に成長しています。 オンラインストアをオープンした経験はないが、それについて考えている人のために、このビジネスを構築するための重要な機能と、注意を払う必要がある主要なポイントについて説明します。

推奨事項のほとんどは、大規模なオンラインストアを作成する予算がない人向けです。 小さな店の概算費用は50,000から100万ルーブルです。



1.製品の選択





オンラインストアを開く前に、まず行うことは、販売する製品を選択することです。 多くは製品の種類に依存します。



製品選択基準：

  • 商品の物理的特性（寸法、重量、耐衝撃性）


小さくて軽い商品を取引する方が、物流の点でより便利で安価です。 それらは、宅配便または郵便で配達できます。 壊れやすい商品や大型商品を扱うのは危険です。



  • 返品率


大規模なプレーヤーは非常に高いレベルのサービスに市場を慣れているため、衣服を扱うのは簡単ではありません。 商品を試着して返品する機会を人々に提供する必要があります。 この場合の返品数は多くなります。

技術的に複雑な製品や、顧客による追加のカスタマイズが必要な製品も返品されることがよくあります。



  • ノベルティ


製品が長い間市場に出回っていて、それを販売する多くのオンラインストアがある場合、成功するためには、まだ誰も使用していない非標準の方法を使用する必要があります。 例えば、IPhoneの販売の場合、 Yes-creditは市場にクレジットで販売する新しい方法を提供しました。 これにより、同社はこのような人気製品の他の売り手よりも有利になりました。

ラップトップ、家電、電話を販売することは非常に困難です。なぜなら、 これらの市場は主要なプレーヤーによって占有されています。

たとえば、海外で人気のあるトレンド、新製品、およびそれらの組み合わせを監視する必要があります。 多くの興味深い製品は、米国（amazon.com）と中国のオンラインストアで見つけることができます。 人気のあるものを次に示します。

www.tinydeal.com

com.focalprice.com

www.lightinthebox.com

www.miniinthebox.com/en

www.wsdeal.com



  • サプライヤーの入手可能性


以下で詳細に説明しますが、一部の商品はリクエストに応じてサプライヤから倉庫から取り出すことができ、一部の商品は1〜2週間待たなければならず、一部の商品は1万個以上のバッチでのみ販売されることを事前に理解する必要があります。



  • 価格設定ポリシー


サプライヤーは、一部の商品グループに対して厳しく規制された最小小売価格（MRC）を設定します。 もちろん、これは違法ですが、オンラインストアを含むこの市場のプレーヤーは、製品に許容可能なマージンを確立し、許容可能な利益を上げることができます。

このような市場の利点は、どの製品にどのマージンを設定するかをビジネスプランで事前に計算できることです。

低価格で買い手を引き付ける方法がない場合は、サービスまたは他の何かでそれを取る必要があります。 まともなサービスの組織は、最も困難で費用のかかるビジネス記事の1つであることを理解する必要があります。

誰も商品の価格を管理していない場合、投棄の可能性があります。 市場では販売が価格に非常に大きく依存しているため、その減少により、膨大な数のバイヤーを引き付けることができます。 しかし、競争力のある製品に最低価格を提供する機会を見つけることは困難です。 一部の市場では、ダンピングにより、純利益率が大幅に低下しています。 コンピュータ機器、携帯電話、家電製品、その他多くの市場のようなマージンの低い市場でオンラインストアを構築し始めるのは意味がありません。



結論：

ユーザーを誘致する予定の「トリック」がある場合、どの競合他社が「戦う」必要があるかがわかっており、競合他社と戦う手段が明確である場合、厳しく規制されたMRCの市場で働く方がより有益です。

希少な製品を販売することを決めた場合、または販売には特別なトレーニングが必要な複雑な製品を販売するための理解とスキルがある場合は、製品をMRCで管理せずに市場で作業できます。



2.商品はどこで購入できますか？







商品を決定したら、1つ以上のサプライヤーを選択する必要があります。



サプライヤーのタイプとその条件（大まかに、異なる市場では異なる微妙さのため）：

  • 中国またはアメリカからの貨物の購入。 大規模な投資が必要であり、多くのリスクが伴います。
  • 中国およびアメリカのオンラインストアからの商品の単位購入。 市場にとって興味深い製品を見つけた場合は、中国のオンラインストアで注文できます。
  • ロシアのメーカーからの商品の購入。 製造業者には、特定の定期的な注文に対して計算された生産計画が常にあります。 ディストリビューター、小売業者などの大企業のみが実行できます。 主要なプレーヤー。
  • ロシアの代理店からの商品の購入。 条件は、特定の市場、特定の販売業者、およびそれにどのように同意できるかによって異なります。 出荷される商品の最低出荷に厳しい条件を課す者もいれば、口座に預金を預けるよう要求する者もいます。 しかし、小さな会社のために協力のための許容条件を提供する人がいます。
  • ロシアの再販業者（ディーラー）からの商品の購入。 ほとんどの場合、最小ロットの条件はありません;信頼度を上げた後、実装のために出荷される可能性があります。 最良の条件を得るために、ディーラーの代表者に直接会って、彼と交渉することを恥ずかしがらないことをお勧めします。 マイナス面は通常、比較的高い購入価格です。


結論：

オプション3、4、5は、初心者向けオンラインストアの残りの部分よりも優れています。 （代理店、再販業者、ディーラーのサプライヤーに電話します）。



3.注意が必要なサプライヤーの特性





残念ながら、サプライヤーとの契約が完了し、2〜3週間のチェックが完了するまで、サプライヤーの作業の品質を確認することはできません。



  1. サプライヤーは常に現在の在庫残高を持っている必要があります。

    通常、残高情報は毎日すべての顧客に送信されます。 これらのデータに基づいて、オンラインストアはサイト上の商品の在庫を更新します。 サプライヤの倉庫の在庫が予想外に変化すると、オンラインストアは顧客に必要な製品の配達を提供できなくなり、顧客を失うことになります。
  2. オンデマンドでのサプライヤによる商品の予約。

    オンラインストアは、販売する製品が常にサプライヤーから入手可能であることを確認するか、製品を予約できるようにする必要があります。 サプライヤは、ERPシステム（通常1C）で、商品が特定のオンラインストア向けであり、他の人に配送できないことに注意する必要があります。 商品を予約するマネージャーが常に利用可能であることが望ましい。 先進的なサプライヤーには、電子流通システムがあります。 このシステムは、インターネットを介した商品のオンラインショッピングサービスのオンライン予約を提供します。つまり、マネージャーに連絡する必要はありません。 このようなシステムを持つサプライヤは、オンラインストアに最適です。
  3. 製品コンテンツ（写真、説明）の提供。

    サプライヤが商品の写真と説明を電子的に持っているのは素晴らしいことです。 残念ながら、これはまれです。 通常、情報は、紙のカタログまたはスキャンされた紙のカタログのコピーを含むPDFファイルで提供されます。

    サプライヤが電子形式の商用コンテンツを所有していない場合は、自分で作成する必要があります。 このタスクは一見簡単そうに思えますが、安くはありません。 大規模なオンラインストアは数百万を費やし、10〜20人のコンテンツの部門全体を作成します。 コンテンツが盗まれることは非常に多く、この目的のために特別なプログラムが作成されます。 この問題については触れませんが、コンテンツを提供するプロバイダーを見つけることをお勧めします。
  4. 返品を処理します。

    サプライヤは、たとえば購入者の拒否により、オンラインストアが販売しなかった商品を取り戻すことができることが望ましい。 この点は、サプライヤーとの契約で指定する必要があります。 サプライヤーが返品を受け入れない場合、この製品を割引価格で販売する必要があります。


結論：

倉庫で商品を購入できない場合（原則として、購入できない）、商品が常に利用可能であるという保証がない場合、残高の関連性に関する情報は非常に重要です。

条項2および条項3で説明されている特性により、サプライヤーとの作業が大幅に簡素化されます。

販売された商品の収益率が低い場合、パラグラフ4の条件は重要ではありません。



理想的なサプライヤーの特徴：

  1. 倉庫に商品を購入する必要はありません。 サプライヤーの倉庫から直接取引できます。 これにより、限られた予算でオンラインストアを運営し、倉庫で購入した商品の需要不足に伴うリスクを軽減できます。
  2. 商品を個別にまたは小さなバッチで購入する機能。
  3. 購入した商品の量に制限はありません。
  4. オンラインストアが一定の速度に達した場合でも、前払いなしで商品を販売することができます。
  5. クライアントから注文を受け取ったときに、サプライヤから商品を迅速に予約する機能。 これにより、販売した商品がサプライヤーの倉庫から確実に出荷されます。 原則として、商品の保留のリクエストをディストリビューターマネージャーに電話または書面で提出する必要があります。 一部のディストリビューターは、オンライン配信システムの使用を提供しています。つまり、オンラインストアが独自にオンラインで商品を予約できるようにします。




4.商品の販売方法は？





インターネット上の製品はすべて、特別なマーケティングツールを使用して販売されます。 膨大な量の資料がこれに費やされているため、基本的なツールについて簡単に検討します。



  1. コンテンツ広告（Yandex.Direct、Google AdWords）。 検索エンジンで特定のユーザーリクエストに合わせて広告会社を柔軟にカスタマイズできるツール。 製品の競合他社からのオファーが比較的少ない場合に有利です。 慎重に使用する必要があります。 対象外のユーザーのサイトでのクリックが原因で、広告に過度の支出を許可することはできません。 開始する前に、アドバイスを求めるか、専門家に助けを求めることをお勧めします。 このような知り合いが必要な場合は、Facebookに登録し、SEOスペシャリストのグループに参加してください。
  2. SEO。 検索エンジン（Yandex、Google）が特定のクエリのサイトを見つけて、1番目または2番目のページのリストに表示するには、そのサイトは非常に正確で適切でなければなりません。 また、最初の6か月間、サイトはどのような状況でもYandexの最初のページに到達しません。
  3. Yandex.Market。 最低価格で商品を見つけるための最も人気のあるRunetリソースの1つ。 価格ダンピングにより、大規模なオンラインストアと競争することができます（規制されていないMRCを持つ商品に適用されます）。 また、顧客はオンラインストアのサービスを評価できます。 正の評価は、販売の可能性を高めます。 格付けとレビューの膨大な数の改ざんに注意する必要があります。
  4. バナー広告。 原則として、それはブランド化ツールとして使用されます。つまり、特定の顧客を引き付けることを目的としていません。
  5. ソーシャルネットワーク。 興味深いペースの速いツール。 コンテンツ広告と非常によく似ています。 主な違いは、ターゲットオーディエンスをさらに微調整する機能ですが、多くの場合クリック単価が高くなります。 たとえば、年齢、ソーシャルカテゴリ、ステータスなどでクライアントを正確に特定できる場合は、ソーシャルで広告を掲載します。 ネットワークが最も効果的になります。
  6. MarketPlaceの使用。 ロシア語-オンラインマーケットまたはオンラインショッピングセンター。 このようなプラットフォームを使用すると、カタログや店舗に商品を配置できます。 同時に、MarketPlaceは多数の購入者をサイトに引き付けることを保証します。 一部のMarketPlaceは、独自のオンラインストアを作成せずに商品を販売する機会を提供し、倉庫と物流の両方を提供します。 このような機能には、各注文の価値の1〜8％の費用がかかります。



    最も人気のあるMarketPlace：

    wikimart.ru

    tiu.ru

    price.ru



結論：

広告ツールはたくさんあります。 各ツールは別々の科学です。 選択した製品カテゴリで機能する最も生産的な方法を試して見つける必要があります。



5.オンラインストアの作成。





最初の4つのポイントに関するすべての情報を分析したら、オンラインストアサイトを作成できます。



多くのツールがありますが、主なものは次のとおりです。



  • コンテンツ管理システム（CMS）。

    箱入りのソリューションは、いわゆるコンテンツ管理システムです。 箱の形で提供され、ホスティングでのインストールが必要です。 有料と無料のオプションがあります。 有料のもののうち、最も人気があるのはBitrix、NetCat、UMI、HostCSMです。 最も無料のもののうち、最も人気のあるもの：Joomla、Drupal、Typo3、ModX、WordPress。
  • コンテンツ管理システム（CMS）SAAS。

    このタイプのCMSは、ホスティングへのインストールを必要とせず、「サービスとしてのソフトウェア」の原則に基づいて提供されます。 つまり、サービスに登録し、関税を選択して（原則として、毎月の支払いが暗示されます）、独自のオンラインストアを取得するだけで十分です。 最も人気のあるリソース：InSales、e-magazin.biz、 www.shoptus.ru 、 www.webassist.com 、 sellbe.com gollos.com
  • ウィジェットオンラインストアECWID。

    既製のWebサイトまたは単にインターネットページがある場合に適しています。ブログでも十分です。 17ドルで、リソースの任意のページに既製のオンラインストアを埋め込むことができます。 インストールするには、CMS管理インターフェイスを使用してサイトのページにコードを配置するだけです。


結論：

初心者の起業家には、いずれかのオプションが適しています。 多くの場合、SaaSまたはウィジェットを使用する方が収益性が高いことに注意してください。 それらの機能は、最も便利な機能に関心のある大企業によってサポートされています。 フリーランスのストアを注文することは、はるかに危険です。

すべてのツールを使用して、受注を処理できるため、ERPを長期間使用せずに実行できます。



オンラインストア向けの追加サービス：



オンラインストアの作成と運用を簡素化できるさまざまなサービスがあります。



  • 宅配便サービス。

    クライアントへの商品の配送を整理するのに時間とお金を無駄にしないようにします。 契約を結び、作業を開始するだけです。 宅配便業者は、倉庫またはサプライヤの倉庫から商品を受け取り、顧客を運びます。 サービスの費用は、1注文の配達に対して200ルーブル以上です。 同様のサービスがたくさんあります。ここにいくつかの例がありますwww.ru-courier.com 、 www.speedpost.ru
  • 会計自動化

    会計と請求を自動化するために利用できる低コストのサービスがいくつかあります。 最も一般的なELBAの1つ。




おわりに





小さなオンラインストアを開くことは、かなり面倒な作業です。 市場、法律、規制を理解せずにオンラインストアを開くべきではありません。 しかし、ビル・ゲイツがかつて言ったように、「将来的には、インターネット上の企業と廃業した企業の2種類が市場に出てくるでしょう。」
