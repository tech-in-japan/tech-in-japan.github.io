JetBrainsとDjangoがPyCharmの30％の売却を発表し、すべてのお金がDjango Fundに送金されます

PyCharmでDjangoフレームワークを数年間サポートした後、JetBrainsとDjango Software Foundationは、PyCharmライセンスの提携と2週間の販売を行うことに合意しました。その後、販売から得たすべてのお金を、さまざまなプロジェクトとイニシアチブの開発のためのDjango基金に振り替えます。 販売に加えて、パートナーシップの一部は、Djangoの将来のバージョンでの新しいPython型注釈標準（ PEP 484型ヒント ）の導入です。











「Djangoは世界クラスのWebフレームワークに成長しました。 DjangoとPyCharmは、共に素晴らしい開発生産性を提供します」と、Django Software Foundation（DSF）の社長であるFrank Wilesは述べています。 「DSFはDjangoの成長を支援しています。JetBrainsがDjangoの募金キャンペーンをサポートしていることを嬉しく思います。」




「過去6年間、Djangoのサポートは製品の最も重要な機能でした」と、JetBrainsのプロダクトマーケティングマネージャーであるDmitry Filippov氏は述べています。 「Djangoの成功は、Pythonコミュニティ全体にとって非常に重要です。」 この売却により、Djangoとのパートナーシップが強化され、Django Software Foundationが目標を達成できるようになります。




セールへの参加方法は？



販売から2週間以内に、PyCharm Professional Editionの個人向け1年間サブスクリプションを30％割引で購入することにより、Djangoを支援できます。 とても簡単です：



  • PyCharm年間サブスクリプション購入ページに移動します 。
  • 支払いページで、「割引コードをお持ちですか？」をクリックします
  • 30％の割引コードを貼り付けます：ISUPPORTDJANGO
  • ページの残りの必要なフィールドに入力し、「注文」ボタンをクリックします。




オファーのすべての詳細を表示したり、セール専用ページで割引価格でPyCharmを購入したりできます 。



売却の結果として集められたお金はすべてDjango Software Foundationに寄付されます。 この販売は、DSFがDjangoの開発を継続し、Djangoフェローシッププログラムを通じてDjango Coreチームをサポートし、Django GirlsのコースやDjango会議などのさまざまな教育イニシアチブを開発するのに役立ちます。



Djangoフェローシッププログラムの一環としてDjangoにタイプアノテーション（PEP 484）を展開する



DjangoとJetBrainsのパートナーシップの一部は、販売に加えて、Djangoの将来のバージョンで新しいタイプ注釈標準（ PEP 484タイプヒント ）を導入するための共同助成金です。



「DjangoでのPython 3のサポートは、過去数年にわたって私たちにとって優先事項でした」とFrank Wiles氏は言います。 -Python 3.5と型注釈は、特にPyCharmのような強力な開発ツールのサポートにより、生産的な開発に向けた大きな一歩です。 この助成金は、Django開発者とコミュニティ全体がDjangoに型注釈を実装するのに役立ちます。



型注釈により、開発者に直接および自動コード分析ツールの両方でコードの可読性が大幅に向上します。 DjangoにPEP 484を実装すると、開発者はDjangoコードをよりよく理解し、フレームワークのより正確なドキュメントを手に入れることができます。 PyCharmなどの自動コード分析ツールは、静的分析で型情報を使用して、コード検査、自動補完、およびリファクタリングを改善できます。 私たちは、大規模で実際に機能するプロジェクトで初めて標準を実装する予定です。これは、DjangoとPythonの将来への大きな一歩になると信じています。



「Python 3の将来と、特にDjangoのようなフレームワークにとっての型注釈の利点を信じています」とDmitry Filippov氏は言います。 「Djangoのタイプアノテーションは、コード分析、コード補完、ツールチップ、リファクタリングの改善により、PyCharmがDjangoを使用して開発する人々の生産性を向上させるのに役立ちます。」



PEP 484規格は過去1年間安定しており、あまり変更されません。 DSFは、フレームワークに標準を実装するためのDjango Core開発者の作業に対して支払います。



この作業を開始する正確なタイムラインはありませんが、それらに関する情報はdjango-developersのメーリングリストとDjangoのブログにまもなく表示されます 。
