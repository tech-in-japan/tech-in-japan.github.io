jQuery UIブートストラップテーマ

プロジェクトでは、 Twitter BootstrapとjQuery UIを使用して、さまざまなページ要素のインターフェイスのプロトタイプを作成します。 フォーム。



先日、フォームに1つのフィールドを追加する必要がありました。 一連のスイッチ（入力要素、無線タイプ）の形式で、最適な形式が選択されました。



Twitter Bootstrapでは、一連のスイッチは一連のボタンとして表示され、イベントを処理するための追加コードの作成を伴うinput要素とは直接機能しません。 jQuery UIライブラリには同様の問題に対する解決策があり、すべてのイベント処理は既に実装されています。



彼はjQuery UIライブラリーに立ち寄った。 外観の問題を解決するために残っています。 プロトタイプはプロトタイプですが、Twitter BootstrapやjQuery UIとは異なる細長い外観を壊したくありませんでした。



jQuery UI Bootstrapプロジェクトは、Twitter Bootstrapウィジェットの視覚的な実行をjQuery UIウィジェットに転送することを本質としています。 ほとんどの場合、プロジェクトはjQuery UI互換のテーマです。
