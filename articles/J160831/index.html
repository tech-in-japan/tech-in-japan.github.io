Android 4.1でのMiniTV MK808の概要



面白いMiniTVの概要を紹介したいと思います。 このデバイスは最近人気を集めており、愛好家からのファームウェアがすでにいくつか登場していますが、ハブに関するレビューはまだありません このデバイスは、よく知られているアジアの取引プラットフォームで、キーボードと一緒に63ドルで購入しました（キーボードがない場合は52ドルです）。 そのような価格で、興味深い特性を備えたデバイスが得られます。

  • CPU：デュアルARM Cortex A9 RK3066（2コア、1 GHzで実行、理論的には1.6 GHzにオーバークロック可能）
  • GP：Mali-400MP（クアッドコア）
  • RAM：1GB DDR3
  • ネットワーク：WiFi：802.11 b、g、n
  • OS：Android4.1.1ジェリービーン
  • 8GBの内部ストレージ
  • オプション：1 x miniHDMI、1 x USB、1 x miniUSB、1 x microSDカードリーダー








1週間待った後、荷物を受け取りました。 miniPC自体は素敵な透明なプラスチックパッケージに入っています（画像をクリックできます）。



キットには、HDMIケーブル（標準ケーブルは情報の送信をまったく拒否したか、大きな損失を伴って送信したため交換する必要がありました）、miniUSB出力を備えた電源、USB-OTGソケットに接続するためのminiUSB-USBアダプターケーブルが含まれています。 デバイス自体は、ソフトタッチと同様に、タッチコーティングが快適です。



換気口を通して、両面ラジエーターを見ることができます（中心部のオーバークロッカーは、グリルを破り、電気テープで大きなクーラーを巻き、デバイスのオーバークロックを開始することをお勧めします）：



一方には、周辺機器用のUSBと電源用のminiUSBがあります。



反対側には、テレビ/モニターに接続するためのminiHDMIがあります。



ここにはOTG-USBがあり、アダプターを介してOTG（：wiki）をサポートするデバイスを接続できます （これはほとんどのキーボード、マウス、フラッシュドライブです）。



反対側には、microSDカードリーダースロットがあります。



タッチパッド付きのワイヤレスキーボードも注文しました：



キーボードは、特に右手にとってはあまり便利ではありません。なぜなら、 タッチパッドが邪魔になっているため、右手の親指でキーを横に持ってキーに到達するのは困難です。 検索エンジンまたはサイトのアドレスに短いリクエストを入力するには十分ですが。 タッチパッドには特別な特徴があります。感度をまったく失い、タッチに応答することを拒否するか、または過敏性になり、触れないで上げた指に反応します。 ワイヤレスマウスに交換する必要がありました。 サイズ推定のための手の写真：





深く見て



両親のためにテレビ用のメディアセットトップボックスを手に取っているときに、MK808に気付きました。 コンソールのコスト（最も予算のかかるメディアコンソールよりもほぼ2倍安い）と、OSとしてのAndroidの存在と比較すると、GooglePlayで非常に多くのソフトウェアを使用できるようになりました。

デバイスの電源を入れます。 スクラッチからの起動には約15秒かかります。 私たちの前に、標準のAndroidスタート画面があります（退屈な壁紙はすでに私に置き換えられています）：





OSバージョンを見てみましょう。





接続速度を確認してください。 ダウンロードで3.6 Mbit / s、アップロードで0.9 Mbit / sでした。 これは私のADSL接続のほぼ最大速度ですが、標準アンテナを備えたMK808はそれ以上のものを生成しないと多くのフォーラムで書かれています。

合計


ここ数週間、MK808はメディアセットトップボックスの機能を正常に実行しています。 もちろん、マイナーな苦情があります：フラッシュプレーヤーがフルスクリーンモードで正しく動作しない場合があります。ネットワークから切断したら、WiFiのパスワードを再入力する必要がありましたが、これはAndroid自体に適用されます。 前述の鉄の問題のうち、問題はアンテナにあります。つまり、WiFiの低速です。 以下のリンクは、アンテナを削除することでこの問題を解決しています。



http://w3bsit3-dns.com/forum/index.php?showtopic=380744-このフォーラムでは、詳細な議論を読み、マニアのさまざまなファームウェアを見つけることができます。
