Zend Framework 1.6.0



Zend Framework 1.6.0がフィニッシュラインに到達しました-リリース候補版はすでに「1」という番号で入手可能です。







変更点は次のとおりです。

  • Dojoフレームワークの統合（4つではなく29メートルのアーカイブをダウンロードするようになりました）

    • JSON-RPC
    • Dojoデータパッキング
    • 道場ビューヘルパー
    • Zend_FormおよびZend_Viewと統合されたDijit
    • 道場図書館配布


  • SOAP-修復およびグレーディング

    • SOAPサーバー
    • SOAPクライアント
    • 自動検出
    • WSDLアクセス
    • WSDL生成


  • 実験室でのツーリングプロジェクトのプレビュー（/検査室ディレクトリを参照）-それらはどういう意味ですか？！？
  • ユニットテストコントローラー
  • Lucen 2.3（ファイルインデックス）のサポートを追加
  • Zend_Sessionはデータベースのテーブルのハンドラーを保存できるようになりました
  • Paginatorコンポーネントの追加-PEAR :: Pager-Hello
  • FIGletのサポートが追加されました
  • ReCaptchaサービスのサポートを追加しました
  • Zend_Config_Xmlが属性を適切に理解するようになりました
  • DBアダプターは「文字セット」オプションを受け取りました-SET NAMES *
  • Zend File Transferコンポーネントが追加されました（私は見つけられませんでしたが）
  • Zend_Viewの新しいヘルパー、Flash、Quicktime、Object、Pageを追加しました
  • Zend_TranslateのINIファイルのサポートを追加しました




プロジェクトのJIRAで詳細を確認します： http : //framework.zend.com/issues/secure/IssueNavigator.jspa?requestId=10794
