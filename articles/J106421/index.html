成功事例。 成功するiPhone / iPadアプリを構築する





2010年10月21日（木曜日）18.30から21.30まで、高等経済学部のビジネスインキュベーターwww.hse-inc.ruに基づき 、ロシアのApple Developer User Group の次の会議が開催されます。 会議に参加するには、 リンクを使用して登録する必要があります。HSEビルへの入り口は厳密にパスです。



前回の会議で、参加者はiPhone / iPadアプリケーションの開発とローンチに関連するサクセスストーリーを作成者から直接聞きたいという要望を表明し、10月21日にそのようなストーリーをいくつか聞きます。 それらの中には：

  • ビジネスカードリーダーアプリのアビー（Olga Tolstunova）
  • Crystal TVアプリでのCrystal Reality（Mikhail Filippovich）
  • Mobile Balanのアプリに関するXZoneソフトウェア（Andrey Ostanin）
  • SlovoEdアプリケーション（電子辞書）に関するParagon Software（Dmitry Bekhterev）
ミーティングの終わりに、アフターパーティーはカフェ「ポクロフカのコテージ」で開催されます。



そのような会議はすでにモスクワで行われており、私はそれを訪問する機会があったので、カットの下で私の印象を共有します。



そもそも、コミュニティは私の目の前に生まれました。以前は開発者の集まりのようなものでしたが、今では関心のある人々の密接な輪による小規模な会議になっています。 優れた組織に加えて、次のことに注意できます。

  • 面白い人との知り合い 。 その中には、Paralells、Unreal Mojo、Flexisなどの開発者がいます。
  • 経験を積んだ 。 iPhoneとAndroidの開発を比較したレポートを作成しました。 人々は多くの質問をし、私のレポートは客観的には最高になりませんでしたが、満足し、非常に前向きな経験を得ました。 ところで、私はiPhoneで勝ちました;）
  • 品質 。 Steve Jobsのフォロワーのような人々は、プレゼンテーション自体を含め、あらゆるものの品質を応援するのが好きでした。
  • アフターパーティー ビールとたくさんのビジネスとビジネス以外の会話での素晴らしいオートパーティー。


出席者の中には、Android開発の直接の知識を持っている多くの開発者がいましたが、それでもすべてがAppleの方が優れていることに明確に同意したのは興味深いことです。
