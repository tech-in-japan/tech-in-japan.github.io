Intel Core i5 660プロセッサーのレビュー

LGA1566プロセッサソケットは静かに私たちの生活に入り、多くのユーザーのコンピューターで現実のものになりつつあります。 LGA1156用の最初のIntelプロセッサのリリースにより、多くの上級ユーザーがこのコネクタを「埋め」ました。 Intelによる2番目のヒットは、私たちを長く待たせませんでした。 より安価で、より低温で、さらに優れたオーバークロックのプロセッサーが市場に参入しました。 今日、これらのプロセッサの1つについてお話します。 Intel Core i5 660（3.33GHz）は、コードネームClarkdaleの古いデュアルコア32nm Intelファミリーの1つです。 Intel Core i5 660の競争相手は、Coreの最速のデュアルコアアーキテクチャと考えられるInte Core 2 Duo E8600（3.33GHz）です。







記者会見でインテルの代表者は、ティックトゥティックシステムについてITコミュニティに絶えず繰り返します。 この名前は可能な限り選択されており、Intelの実際のチップ製造ポリシーを反映しています。 私たちが知っているように、Intelプロセッサは通常世代に分けられます。 検討中のTik-Takシステムは、Intelが各世代のプロセッサの開発に2つの技術プロセス（2段階）を捧げていると述べています。 1つ目は、新しいアーキテクチャの導入、いわゆる「パフォーマンスブーム」です。 新世代の成功が統合されると、Intelは同じアーキテクチャに基づいたプロセッサーを新しい、より優れたプロセステクノロジーに移行し、それによって、放熱とオーバークロック、さらには古いプロセステクノロジーの可用性と普及率を示すいくつかの指標を即座に獲得します。







クラークデール-プロセッサとチップセットの進化の上位段階の1つにIntelが置かれている点で興味深い。 最近では、マザーボードに通常のヒートシンクがなく、ノースブリッジを冷却していました。 現在は必要ありません-メモリコントローラーはプロセッサに統合されています。 メモリコントローラに続いて、PCI-Eバスコントローラがシステムのブレインセンターに続きました。 プロセッサには、ビデオカードも組み込まれています。これは、統合されたグラフィックコアです。







上の画像は、Intel Core i5 6xxおよびIntel H55を左から右に示しています。 新しいIntelプロセッサに統合された統合グラフィックスの使用をサポートしていることをユーザーに伝えるのは、「H」という名称のチップセットです。 当然、統合グラフィックスコアのクレイジーなパフォーマンスを夢見ることしかできません。 おそらく、多くの人々がインターネット全体を取り巻く物語を覚えています。 Intelプロセッサに組み込まれたグラフィックコアは、3DMark Vantageで非常に優れたパフォーマンスを示し、個別のローエンドソリューションでさえも打ち負かしました。 しかし、実行可能ファイルの名前が（Vantageではなく）Vintageに変更されると、オウムのかなりの部分がどこかで失われました。







Clarkdaleのラインでは、Intel Core i5 660がトップのポジションの1つを占めています。 Intel Core i5 661プロセッサーは、プロセッサー特性の点でi5 660とまったく異ならないため、アナログと見なすことができます。 ただし、i5 661のグラフィックコア周波数は900 MHzよりも高い-i5 660の733 MHzです。したがって、グラフィックコアの周波数が低いため、熱レベルはしきい値73Wを超えません。



32 nmプロセステクノロジを使用して製造された新しいデュアルコアプロセッサと、前世代のデュアルコアプロセッサコア、つまり同等の周波数の競合他社であるIntel Core 2 Duo E8600とを比較するのは興味深いことです。 多くのユーザーの間で人気を博しているプロセッサは、今では負けません。 これは、Intel Core i5 660とInte Core 2 Duo E8600を比較することで私が決定したものです。

Intel Core i5 660の極端なオーバークロック。



最近、インターネットには、6 GHzを超えるプロセッサのオーバークロックの成功に関する情報がたくさんあります。 プロセッサを液体窒素で冷却して、この情報を自分で確認することにしました。







オーバークロックに影響する重要な要素はコールドバグです。 Coldbug-プロセッサのこのような温度、それ以下では動作を停止し、システムは生命の兆候を示しなくなります。 クラークデールのプロセッサは、液体窒素の沸点である摂氏196度に近い非常に低い温度でColdbugを持っていることで有名です。 しかし、極端なオーバークロックに専念している外国のフォーラムの1つで、そのような温度はEvgaマザーボードのみに依存しているという情報を見つけることができます。 会社の開発者の1人は、このような低温に身をかがめることができる秘密のmodを共有しています。
