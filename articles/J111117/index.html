ミッドナイトコマンダー4.7.5安定版の発売

Midnight Commander（MC）は、ファイルとディレクトリツリー全体のコピー、移動、削除、ファイルの検索、シェルでのコマンドの実行を可能にするビジュアルコンソールファイルマネージャーです。 組み込みのビューアとエディタも含まれています。



MCは先週更新されました。 バージョン4.7.4以降の変更 ：



  • Cs Csで以前のクイック検索を繰り返します（テキスト入力なし）
  • 大文字と小文字を区別するファイルのソート順は、ピリオドで始まる他のユーザーのファイルと混ざりません。
  • ダイアログとステータスバーに新しい色を追加しました
  • 新しいPanelSmartGotoParentDirアクションが追加されました
  • 組み込みのブックマークエディターは永続的になり、エディターを終了した後も変更は失われません。
  • MCポリシーに従ってソースツリーの大規模な再編成と完全なインデント
  • 新しい翻訳：バスク語、エストニア語、グルジア語、インドネシア語
  • FISHファイルシステムに接続する前にパネルを非表示にする
  • テキストをクリップボードにコピーしても、現在選択されているテキストはリセットされません
  • ビューアーでファイルを開き、mceditで再発見した後、現在の検索位置を記憶する
  • スピードアップ/ダウンカーソル
  • 新しいダークスキンが追加されました


変更の完全なリストはこのページにあります 。



新しいバージョンにアップグレードする場合は、WebUpd8 PPAを使用できます。

sudo add-apt-repository ppa:nilarimogard/webupd8

sudo apt-get update

sudo apt-get install mc







またはそれなしで行います。
