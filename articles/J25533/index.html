Eclipseのプラグインをインストールする別の方法

通常、Eclipseのプラグインは簡単にインストールできます。Eclipseのフォルダーにコピーされます。 ただし、この方法はあまり便利ではありません。 プラグインのバージョンを管理するのは簡単ではありません。 また、新しいバージョンをインストールすると、フォルダーが乱雑になります。 そこで、リンク（リンク）に基づいて次の方法を提案します。



たとえば、Eclipse用のSQL Exporerプランをインストールするには、サイトから受け取ったアーカイブを、通常のインストール方法と同じように、フォルダー階層を維持したままsqleなどの任意のフォルダーに解凍する必要があります。 その結果、次のものが得られます。







次に、Eclipse IDEがインストールされているフォルダーで、linksフォルダーを作成します。 このフォルダーに、任意の名前のファイルを作成します（もちろん、このファイルには何らかの種類のセマンティックロード、たとえばsqlexplorer.linkが含まれていることをお勧めします）。このファイルには、次のように記述します。



path = d：/ Program Files / Eclipse / sqle
