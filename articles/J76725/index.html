外付けHDDからのインストール



私はすでに1年間DVDドライブを拒否していますが、ネットブックの傾向は決定の正しさを示しています。 しかし、DVDなしでOSをインストールする方法について質問がありました。 フラッシュドライブが所定の場所でなくなってしまい、新しいものを求めて店に行ったとき、Seagate Expansion外付けドライブが偶然目を引きました。 そして、サイズと全体がジャケットのポケットを大きく困らしませんでした。

寸法：141 x 18 x 80 mm

重さ：0.16 kg



しかし、判明したように、USB HDDを備えたWindows 7のインストールは、USB FLASHと同じようには機能しません。 操作性を理解して確認したので、情報を共有しています。

デスクトップ、ラップトップ、ネットブックの両方で機能します。

  • HDDの準備
  • HDDパーティションの切断と準備。
  • Windows 7をコピーする




HDDの準備



ブートディスクを作成し、Windows 7をインストールしてから消去し、必要に応じてHDDを使用し続けることができます。 または、はるかに便利にします。

外付けHDDの4ギガバイトのスペースを気にしない場合は、ディスクを2つの部分に分割できます。この場合、Windows 7ディストリビューションに1つの部分を割り当て、常に手元に置いておきます。



HDDのパーティション化と準備



1.に移動します。

- - ( )





ディスク1を見つけます（システムに複数のハードディスクがある場合は、外付けHDDのサイズに等しいハードディスクを見つけます）



2.外部HDDドライブを選択します。

-右クリック-ボリュームの削除。

-右クリック-シンプルボリュームを作成します。

表示されるウィンドウで、4300メガバイト（[配布]セクションの下）のボリュームサイズを選択し、[次へ]をクリックします。



次のダイアログで、必要に応じてドライブ文字を選択し、「次へ」をクリックします。 新しいダイアログで、新しいセクション（FAT 32）をフォーマットする必要があり、クリックして完了です。



その後、作成されたパーティションをアクティブにする必要があります。

-セクションを右クリックして、セクションをアクティブにします。

これで、外部HDDは次のようになりました。



同じ原則により、ディスク上の2番目のパーティションをフォーマットします。何よりもNTFSでフォーマットします。 ただし、アクティブにしないでください 。



Windows 7をコピーする



Windowsディストリビューションをブートディスクにコピーするのは、これまでになく簡単になりました。



DVDをお持ちの場合：

-インストールDVDを開き、すべてのフォルダーとファイルを選択して、外付けHDDのアクティブパーティションにコピーします。



ISOイメージがある場合：

-Total CommanderまたはDaemon Toolsを使用してインストールISOイメージを開き、すべてのフォルダーとファイルを選択して、外部HDDのアクティブセクションにコピーするだけです。



すべて、外部HDDはブータブルメディアとして動作する準備ができており、十分に再起動し、BIOSでプライマリブートデバイスとして設定されます。 BIOSの種類に応じて、特定の外付けHDD、または単にUSB HDD。 変更を保存し、Windows 7のインストールを続行します。
