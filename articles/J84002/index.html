Opera Miniのアクティブユーザー数は5,000万人を超えました









2010年2月12日-オスロ、ノルウェー



世界で最も広く使用されているモバイルブラウザであるOpera Miniは重要なマイルストーンに達しました。2010年1月のデータによると、アクティブユーザーの数は5,000万人を超えました。 このマイルストーンは、Opera Miniの人気を証明しています。ユーザーは、その機能と独自の機能によりOperaブラウザを選択します。



近年、Opera Miniはすべての地域で前例のない人気の増加を経験しています。 わずか1年で、ブラウザユーザーの数は150％増加しました-2009年1月の2,000万人から2010年1月の5,000万人まで。 これらの指標では、インターネット上のブラウザでアクティブに作業しているユーザーのみが考慮されます。



Opera Miniは、ほとんどの携帯電話で実行され、サーバーベースのデータ圧縮技術を使用する、小型で使いやすいブラウザーです。 このテクノロジーにより、ユーザーの電話にダウンロードされるトラフィックは初期ボリュームの90％に圧縮され、インターネットの速度が向上し、モバイルインターネットのコストが削減されます。



Gartnerの分析会社によると、2013年までに、携帯電話は、今日のコンピューターであるインターネットにアクセスするための主要な手段になるでしょう。 この予測は、ここ数カ月でのOpera Miniの急速な成長を説明しています。 Opera Miniの最初の1,000万人のユーザーを引き付けるには、28か月（2006年1月から2008年4月）かかり、わずか3か月で4,000万から5,000万に増加しました。



「Opera Miniの主なタスクは、オペレーターのネットワークや携帯電話のモデルの帯域幅に関係なく、誰もがインターネットにアクセスできるようにすることです。 Opera SoftwareのCEOであるLars Boilesen氏は、次のように述べています。



Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザは、その高速性、コンパクトさ、および他のブラウザに比べてWeb標準のサポートが優れているため、ユーザーおよびメーカーから世界的に認知されています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
