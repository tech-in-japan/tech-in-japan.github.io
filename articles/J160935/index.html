Virtual Day Cisco Expo 2012

2012年11月20日から22日にかけて、モスクワで重要なイベントが開催されました-第13回Cisco Expo会議。 これは、国内のすべての地域から数千人の参加者を集めている最も印象的なITイベントの1つです。 会議では、100を超えるさまざまなレポートが読み上げられ、最新のネットワークテクノロジーと通信に関する多数のセッションとプレゼンテーションが行われました。 会議のトピックの範囲は非常に広範でした。企業ネットワークインフラストラクチャ、セキュリティ、ワイヤレスネットワーク、データセンターと仮想化、ユニファイドコミュニケーション、電話会議、プロバイダー向けソリューションなど。



11月19日の会議に先立ち、シスコは、Cisco WebExを使用してオンラインで開催された「仮想の日」も開催しました。 日中、さまざまなトピックに関するいくつかのレポートが読まれ、だれでも接続できました。 セッションの1つは、ソフトライントレーニングセンターの教師が実施しました。 スピーチ「Cisco Routers」。 1時間ごとのレポートでは、シスコがリリースするルーターのラインアップを網羅し、その主要な機能を説明する試みが行われました。 この記事では、このレポートの簡単な要約を見つけることができます。



シスコは、ホームオフィスや支店、大企業、プロバイダーなど、すべての市場セグメント向けに設計されたさまざまなモデルのルーターを製造しています。 製造されたルーターのほとんどは、統合サービスルーターに分類されます。 これらは単なるルーターではありません。 従来の機能に加えて、追加の統合機能を利用できます。 たとえば、スイッチはすべてのルーターモデルに統合されており、4つ以上のクライアントデバイスを接続できます（モデルに応じて）。 スイッチポートはPower-over-Ethernet（PoE）をサポートできます。 イーサネットケーブル経由の電源。 このシリーズのすべてのルーターにはセキュリティ機能があります。DMVPNやGET-VPNなど、さまざまなタイプのVPNトンネルを構築します。 IOS Firewall、Zone-based Firewallなどのシスコテクノロジーを使用して、ネットワークを保護できます。 ルーターは、コンテンツをフィルタリングし、侵入防止機能を提供し、さらにウイルス対策保護を提供できます。 さらに、これらの機能の多くは最年少モデルに存在します。 また、多くのモデルはオプションで802.1a / b / g / nワイヤレスネットワークをサポートしています。 統合サービスルーターの多くのシリーズは、IPテレフォニーに対応しています。 すべてのルーターは、グラフィカルツール（SDM、Cisco Configuration Professional）と従来のコマンドラインの両方を使用して、ネットワークに迅速に展開できるように準備されています。 1900シリーズ以降、ルータには追加モジュールをインストールするための拡張スロットがあります。 これらのモジュールは、デバイスの機能を大幅に拡張し、中央処理装置の負荷を軽減できます。 モジュールは、LANおよびWANポートの密度を高め、暗号化タスクを実行し、侵入防止システムの機能を提供し、IPテレフォニーの機能を拡張できます。



結論として、シスコのルーターにはほぼ無限の可能性があると言いたいです。
