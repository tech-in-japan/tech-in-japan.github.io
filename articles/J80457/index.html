Firefox 3.6候補リリースが利用可能になりました

Mozillaコミュニティは、 Firefox 3.6の リリース候補があったことを誇らしげに発表しています。 このプレビューバージョンのブラウザーは無料でダウンロードできます。また 、 Firefox 3.6の ベータバージョンのすべてのユーザーに対して自動更新によって配信されます。 Firefoxに追加された数千の追加の75％以上は、 Firefox 3.6との互換性のために作成者によって既に更新されています。 お気に入りのアドオンがまだ互換性があるとマークされていない場合は、アドオン作成者がアドオン互換性レポートツールを使用してテストするのに役立ちます 。 このリリース候補は随時自己更新される可能性があり、 やがてFirefox 3.6の最終リリースと同一になります。



Firefox 3.6（ Gecko 1.9.2プラットフォームに基づく）は、ユーザーおよび開発者向けにいくつかの新機能を導入しています。
  • ユーザーはPersonasを使用して1回クリックするだけで、ブラウザーの外観を変更できます。
  • Firefox 3.6は、古いプラグインについて安全を守るためにユーザーに警告します 。
  • Firefoxにサードパーティソフトウェアを統合する方法を変更して、安定性を高めました。
  • フォームのオートコンプリートの改善により、フォームへの入力履歴から最適なオプションを選択できます。
  • 同じ名前の<video>要素で接続されたオープンビデオは、フルスクリーンモードで表示できるようになり、 ポスターフレームもサポートするようになりました 。
  • WOFFフォント形式がサポートされています。
  • JavaScriptの実行、ブラウザの全体的な応答性、および初期読み込みの速度が改善されました。
  • スクリプトの実行を遅らせる機能が登場し 、ページの初期読み込みを加速しました。
  • HTML5ファイルAPIがサポートされています。
  • 新しいCSS、DOM、およびHTML5 Webテクノロジーがサポートされています。
アドオン開発者とWeb開発者は、Mozilla Developer Centerで開発者向けの多くの新しいFirefox 3.6機能について読む必要があります。 Firefox 3.6の アルファリリース以降に行われた変更の完全なリストを受け取りたい場合は、 そのリストを見てください （かなり大きい）。



したがって、将来のFirefox 3.6を覗き見したい場合は、候補リリースをダウンロードして試してください ！



既にFirefox 3.6のベータ版をお持ちの場合、リリース候補の前に自動的に更新されるはずです。 ヘルプメニューから「アップデートの確認」を手動で選択することもできます。



いつものように、Mozillaコミュニティはこのリリースについて、またはあなたが見つけるかもしれないバグについて受け取ったフィードバックを喜んで聞きます 。
