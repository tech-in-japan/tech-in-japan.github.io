Midnight Commander 4.7.0-pre3がリリースされました

今日、前のバージョンがリリースされてから1か月が経ち、Linux用の素晴らしいファイルマネージャー（だけでなく:)）mc 4.7.0-pre3のリリースに関するニュースが公開されました。

見つかったエラーを修正し、新しいエラーを追加することに加えて:)（そしてどのようにそれなしで？）以下の重要な変更が行われました：

  • デザインを変更するメカニズムがあります（スキン）
  • 定義済みのキーボードショートカットを変更するメカニズムがあります


これで、ほとんどすべての組み合わせを割り当てて既存のものを置き換え、VIMまたはFARに似たものにすることができます...



バージョン4.7.0-pre2に関する主要な変更点のリスト



全般

  • GLibは、独自の実装の代わりにコマンドラインパラメーターを処理するために使用されます
  • パネルのタイプと拡張子によるファイルの強調表示を実装しました
  • ファイル名の表示は、ファイルのコピー/転送時の警告ダイアログで最適化されます
  • 初めてファイル検索を開始すると、入力フィールドに履歴の値が入力されます
  • 複数のエンコーディングをサポートするようになりました（--enable-charset）、デフォルトで有効になりました
  • 「スキン」のメカニズム（フレーム、カラースキーム）が実装されています
  • キー（パネル、コマンドライン、エディター、ビューアー）を再割り当てするためのメカニズム

VFS

  • tar.xzファイルの動作を修正
  • 拡張によるlzmaアーカイブの認識が追加されました

編集者

  • ファイルを保存するための「名前を付けて保存」ダイアログで、行末のタイプを選択できるようになりました：Windows / UNIX / Mac（CR LF / LF / CR）
  • VerilogHDL、シェルスクリプト（シェルスクリプト）の構文強調表示を更新
  • .repo（yum）、PKGBUILD、.installの構文強調表示を追加

視聴者

  • WindowsおよびMacの行末でファイルを表示する問題を修正（CR LF / CR）

雑多

  • 環境変数（$ HOME、〜ユーザー名など）のサポートがhotlist要素に追加されました
  • ホットリスト要素に追加された自動補完サポート
  • 数字キーを使用したリストのクイックナビゲーション（「4」-5番目の項目を選択するなど）

バグ修正

  • 無効なカラーパレットの行を解析するときのmcクラッシュを修正
  • パネルが水平の場合のパネルエンコーディング選択ダイアログの不適切な配置を修正
  • raneliでのファイル交換の表示に関する9桁の制限が削除されました
  • stickcharsモードでの行の表示を修正（オプション-a）
  • 非匿名FTPログインでのmcクラッシュを修正
  • Ctrl-Oを押したときの「スクリーン」ターミナルでのmcの動作を修正
  • CP866ロケールのサポート（IBM866）
  • configure.ac構成ファイルのglibバージョンの検証が改善されました（GLib 2.6以降が必要です）
  • メニューと履歴リストでマウスホイールを使用してスクロールするとmcがクラッシュする問題を修正
  • Meta + PgDn（Alt + PgDn）によるテキストマーキングを修正
  • カーソルが行末にない場合にCtrl +矢印を使用したときのカーソルの動作を修正
  • エディターのオートコンプリートを修正
  • ファイルのコピー/移動ダイアログでのMeta-d（Alt-d）の動作を修正
  • --with-gpm-mouseオプションのビルドの問題を修正
