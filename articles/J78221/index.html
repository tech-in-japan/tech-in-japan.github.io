* .AIから* .CDRへのエクスポート



先日、プロジェクトをAdobe IllustratorからCorelDrawに変換するという非常に興味深い関連タスクを解決する必要がありました。 キャットの下でこれをどのように達成したかについての小さなガイド。 これらの小さなトリックがhabrコミュニティの誰かに役立つことを願っています。



EPSで保存

画像のシンプルさ（グラデーションの欠如、透明度）を考えると、すべてを非常に簡単に解決できます。 保存するときは、 EPSオプションで * .EPS形式を選択し、サブツリーのファイルのバージョンを選択します。

保存したEPSファイルをCorelで開きます。



アイクリップボード

複雑なイラストの変換はもう少し複雑な作業であり、タンバリンとの非常に小さなダンスが必要です。 Adobe IllustratorからCorel Drawに画像を変換するスクリプトをダウンロードします。
ダウンロード先：
  • 公式サイト
  • CorelDRAW 10/11/12 / X3 / X4のスクリプトバージョン
  • Corel DESIGNER 10/12のバージョン


* .aiから* .cdrへの転送手順は次のとおりです。
  1. 次のパスにあるアーカイバを使用してスクリプトを解凍します。... \％root_folder％\ Draw \ GMS \
  2. Corelを起動し、 ツール/ Visual Basic / Playに移動します
  3. [マクロ]で、AiclipBRD.gmsを選択します（メニューとCorelを閉じないでください）
  4. Illustratorで* .aiファイルを開き、すべてを選択してCtrl + Cを押します。
  5. Corelに移動し、開いているメニューでCopyPaste.PasteAIFormatを選択して、[実行]をクリックします。
  6. 画像が挿入されます。 * .cdrに保存


CorelDraw X4のマクロメニュー

画像を転送する際、透明度のあるグラデーションとレイヤーの転送で浅瀬に気付きませんでした。
