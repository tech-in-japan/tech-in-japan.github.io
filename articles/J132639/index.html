AddThis共有サービスとGoogle Analyticsの統合

今年、Google Analyticsに新しいレポート、 Visitors> Social Featuresが登場しました。 ご想像のとおり、ソーシャルボタン（共有など）を介してサイトとのやり取りを表示する必要があります。

ただし、各ソーシャルボタンを追跡するには、 _trackSocial



メソッドを呼び出してボタンのクリックを追跡するコールバック関数を作成する必要がありました。

これについては、Google Analyticsヘルプで確認できます： http : //code.google.com/intl/en-US/apis/analytics/docs/tracking/gaTrackingSocial.html

そして、それはあまり便利な解決策ではありませんでした。 サイトには多くのソーシャルボタンがあり、それぞれに対して関数を記述する必要があります。



現在、ソーシャルインタラクションを追跡するためのよりエレガントでシンプルなソリューションがあります。 つまり、 AddThisソーシャルボタンサービスとの統合。 このサービスには、ソーシャルボタンのかなり大きなベースと、ウィジェットをカスタマイズするための十分な機会が含まれています。





AddThisウィジェットをGoogleアナリティクスと統合するには、ウィジェットの呼び出しブロックに小さなコードを追加する必要があります。 

 <script type="text/javascript"> var addthis_config = { data_ga_property: 'UA-xxxxxx-x', data_ga_social: true }; </script>


その結果、ウィジェット呼び出しブロック全体は次のようになります。 

 <!-- AddThis Button BEGIN --> <a class="addthis_button" href="http://addthis.com/bookmark.php?v=250"> <img src="http://s7.addthis.com/static/btn/v2/lg-share-en.gif" width="125" height="16" alt="Bookmark and Share" style="border:0"/> </a> <script type="text/javascript"> var addthis_config = { data_ga_property: 'UA-xxxxxx-x', data_ga_social: true }; </script> <script type="text/javascript" src="http://s7.addthis.com/js/250/addthis_widget.js"></script> <!-- AddThis Button END -->


統合の詳細については、こちらをご覧ください ： http : //www.addthis.com/help/google-analytics-integration



訪問者がAddThisウィジェットのソーシャルボタンをクリックし始めると、相互作用がGAレポートに表示されます。

  • 訪問者>ソーシャル機能>インタラクション -サイトの主な指標（訪問ごとの閲覧ページ数、サイトでの平均滞在時間、直帰率など）を、ソーシャルアクションを含めて、または含めずに比較できます。
  • [訪問者]> [ソーシャル機能]> [アクション] -各ソースおよびソースとアクションのさまざまな組み合わせのソーシャルアクション（[共有]、[いいね！]などのクリック）の数を比較できます。
  • 訪問者>ソーシャル機能>ページ -このレポートを使用して、サイトのさまざまなページでのソーシャルアクティビティの数を比較できます。 このデータは、ソースごと、およびソースとアクションの組み合わせごとに表示できます。


AddThisレポートには、GAにはない別の興味深い指標があります。「ウイルス」-このリンクの共有総数に対する共有リンクのクリックの比率。
