推奨される週刊リポジトリ



yostudios / spritemapper


Spritemapperは、既存のCSSファイルからCSSスプライトを生成するためのコンソールアプリケーションです。 PNG画像を1つにマージし、新しいスプライトマップを使用して新しいCSSファイルを生成します。

また、独自のPNGおよびCSSパーサーも含まれています。 必要なのは、Pythonインタープリターだけです。 PIL設定をデバッグする必要があった場合、これがどれほど重要かを知っています。



今週の素晴らしいリポジトリ



jstrait / beats


このプロジェクトは本当に素晴らしいです。

BEATSは、Rubyで書かれたコンソールオーディオシーケンサーです。

音楽はYAMLでプログラムされています。 入力ファイルは、楽器（サンプル）、そのシーケンス、および曲の完全な構造を定義します。 結果の曲は、後で聴く/ミキシングするためにLPCM WAVにエクスポートできます。



素晴らしいリポジトリ

  • tweetdeck / TDOAuth ：iOSアプリケーション用の小さくてエレガントなOAuth v1.0実装。 有名なhomebrewの作者であるMax Howellによって書かれたこのBSDライセンスプロジェクトは、Objective-C開発者が利用できる他のOAuth実装よりも優れていることを目指しています。
  • sporst / SWFREtools ：Adobe Flash PlayerおよびコンパイルされたSWFファイルのリバースエンジニアリングとセキュリティチェックの実行のためのツールのセット。 SWFパーサー、SWFミニマイザー[SWFミニマイザー]、Flashデバッガー、メトリックジェネレーターが含まれます。 GPLv2ライセンスで配布され、JavaおよびPythonで利用可能です。
  • leftnode / get-shit-done ：システム上のドメインのカスタムリストをブロックして作業を妨げる小さなスクリプト）。
  • microsoft-dpe / watoolkitios-lib ：Microsoftが作成したこのライブラリセットにより、Objective-C開発者はiOSアプリケーションでAzure Cloud Storageプラットフォームを使用できます。 デモリポジトリをチェックして、動作を確認してください。

  • enonic / cms-ce ：JavaのSpring FrameworkとJEEに基づくEnonic CMSの新しいコミュニティバージョン。 商用バージョンとは異なり、AGPLv3コミュニティバージョンでライセンスされているものには、プラグインエンジンとLDAP / ADサポートがありません。

  • wbsun / kgpu ：この非常に野心的なプロジェクトにより、LinuxカーネルはサポートするCUDA GPUに並列コンピューティングを配置できます。 「複数の要求を同時に処理するだけでなく、1つの大きなリソースを大量に消費する計算をいくつかの部分に分割し、GPUカーネルで実行する機能も備えています。」

  • jiminoc / goose ：HTMLコンテンツと記事のGravity Labs Instapaperスタイルの抽出ツール 。 Apache v2でライセンスされたJavaで書かれています。 渡したURLを受け入れ、コンテンツを返します。

有望なリポジトリ

  • bobthecow / coda-cli ：Panic Codaエディター（TextMate mateコマンドに類似）を起動するためのコンソールユーティリティで、完全な* nixスタイルのアプリケーションになります。 例については、プロジェクトのページにアクセスしてください 。

  • skammer / vim-css-color ：この素晴らしいプラグインを使用すると、Vim設定でCSSカラーを使用できます。 現在サポートされているのは、RGB、RGBA、およびHTMLカラー名です。 （256色の端末が必要です）。

  • bradfitz / campher ：これで、Goスタック上でPerlを実行できます。 真剣に。 この難解なプロジェクトには、かなりの野心と驚くべきユーモアのセンスがあります。

  • wavded / humane-js ：シンプルさを最大化するように設計された控えめなブラウザー通知システム。 すべてのJSフレームワークおよびブラウザーと互換性のある完全にカスタマイズ可能なシステム。 デモページにアクセスして、実際の動作を確認してください。

  • brixen / poetics ：Rubinius VMで直接実行されるネイティブCoffeeScriptインタープリター。 もっと話す価値はありますか？
