AmazonがClusterKのスタートアップを買収

Amazonはクラウドでの地位を強化しています。 今日、アマゾンは、アプリケーションを起動するためのクラウドインフラストラクチャの世界最大のプロバイダーです。 Venturebeatによると 、このスタートアップには2000万〜5000万ドルが支払われており、将来的にはClusterKテクノロジーが同社のインフラストラクチャに統合される予定です。



ClusterKは、AWSベースのアプリケーションを実行するトリッキーな方法を開発しました。 スタートアップが提供するサービスのコストは、元のAmazonianの10分の1です。 ただし、ClusterKはAmazonのライバルではありません。 AWSが安くなります。



昨日、Amazonはスタートアップ投資家のMarc Parrishに支払いをした。 会社にとって大きな利益であり、Amazonが主導的な地位を獲得し、競合他社を大きく引き離すことができます。 Microsoft AzureとGoogle Cloud Platformの両方は、対象にとどまるために値下げを余儀なくされます。



この契約は、今後数か月および数年で、Amazonがスタートアップアプリケーションをクラウドアプリケーションの低コスト展開のプラットフォームとして使用することを意味しています。



以前、私はAmazonが2lemetryテレメトリのスタートアップを買収することについて書いた。 同社はアミアトも買いました。 そのため、同社がクラウドテクノロジーのモンスターと見なされる立場は理解できます。



ClusterKチームは、アマゾンのシアトル本社に移り、働いていると伝えられています。 Mark Parrishは、ClusterKの創設者兼CEOであるDmitry PushkarevがAWSで重要な役職に就くことを期待しています。



早くも2月、ClusterKは、スタートアップが120万ドルを調達したシード投資ラウンドを発表しました。 ClusterKは2013年にカリフォルニア州パロアルトで設立されました。
