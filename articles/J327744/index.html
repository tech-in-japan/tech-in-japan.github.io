記念日PYCON RUSSIAは7月16〜17日に開催されます

5回目のロシアPyConは、95 km離れたCronwell Yakhonts Tarusa Hotelで7月16〜17日に開催されます。 モスクワから。 PyConがわからない場合は、以下のビデオをご覧ください。PyCon-2016と会議の歴史について簡単に説明しています。





スピーカーのレベルを評価するには、昨年のレポートのビデオやプレゼンテーションを見るか、PyConRu-2016に関するHabréのレポートを読んでください。



参加者の登録は公開されています。 アーリーバードのチケットは14500ルーブルかかります。 このようなチケットは50枚あり、コストが増加します。 学生には特別価格があります。



今すぐ参加しよう！



書式



PyConRu形式は同じままです。自然の中で2日間、ロシア語と外国語のスピーカーによるプレゼンテーションとマスタークラス、Lightning Talksとアフターパーティ。



カンファレンスは、ヤホンティタルサカントリーホテルで開催されます。 ホテルには、スイミングプール、フィットネスセンター、ボーリング、ビリヤード、ビーチ、自転車、ローラーのレンタルがあります。 PyConへの旅行は、あなたにとって有益なイベントであるだけでなく、家族全員にとって素晴らしい週末にもなります（詳細は以下をご覧ください）。



ホテルとの往復シャトルサービスがあります。





PyConRu-2016で、Pythonのコア開発者であり、レイモンドヘッティンガー言語の多くの部分の作者でありメンテナーであるロシアで初めて話をしました。



プログラム委員会



プログラムディレクターは、アプリケーションをレビューし、最も有用で興味深いものを選択し、講演者がレポートの準備を支援します。 一般に、彼らはプログラムを関連性のある興味深いものにします。



今年のプログラム委員会は強力です。Pythonコア開発者、DataRobotのエンジニアAndrey Svetlov 、JetBrains Andrey Vlasovskikhの IDE PyCharmの開発者、Doist Inc Roman Imankulovの開発者、Tochka銀行のサービスステーションDanila Shtan 、 pipedrive.comのソフトウェアエンジニアDmitry Ovchinnikov 、CTO Persolloで、Toptal Andrey Soldatenkoでpython開発者、Spherical Anton PatrushevでCTO。





Flaskフレームワークの作者、Palletsプロジェクト、Pocooチームの創設者の1人、Python言語Jinja2の有名なテンプレートエンジンの作者であるArmin Ronacherは、PyConRu-2016で「How Python was Shaped by leaky Internals」を発表しました



プログラム



会議は毎日2つのストリームで行われ、2日目にはさらにいくつかのマスタークラスがあります。 使用言語はロシア語と英語です。



プログラムが形成されています。 すぐに最初のスピーカーがサイトに表示されます。 スピーカーを知らずに参加を決定することは難しいことを理解しています。 過去数年間のレポートをご覧になり、カンファレンスに重要かつクールな世界クラスの専門家を常に招待するようにしてください。 そして、私たちはロシアで最初にそれをやっています。





PyCon RussiaのプログラムディレクターであるPythonコア開発者のAndrey Svetlovは、PyConRu-2016でpy.testライブラリについて話しました。 そして昨年、アンドレイは私たちにPython言語についての短いインタビューを行いました。

登録



アーリーバードのチケットは14500ルーブルかかります。 このようなチケットは50枚あり、コストが増加します。



チケット価格に含まれるもの：

•会議への参加（2日間）。

•ダブルまたはトリプルルームでの宿泊。

•7月16日の昼食と夕食、7月17日の朝食と昼食。

•すべてのコーヒーブレーク。

•エンターテイメントプログラム。

•7月16日にAnnino地下鉄駅からカンファレンス会場まで、そして7月17日にその逆のバス。



学生の特別価格は9000ルーブルですが、特別価格でチケットを購入するには、学生スキャンをom@it-people.ruに送信し、それに応じてプロモーションコードを送信します。



あなたの愛する人と一緒にPyConに来てください-彼らは何かする必要があります。 ホテルには、スイミングプール、フィットネスセンター、ボーリング、ビリヤード、カラオケ、スパ、さらには5D映画館があります。 子供向けには、遊び場、アトラクションのある部屋、トランポリンがあります。



また、馬に乗ったり、自転車、スクーター、ローラースケートをレンタルしたり、kitを飛ばしたりすることもできます。 近くにはビーチのある川があります（泳ぐことはできませんが、日光浴はできます）。 そして、これはすべて森林の背景に対してです。



一緒に旅行する人のチケットは6,000ルーブルかかります。 訪問レポートを除き、参加者のチケットに含まれるすべてのものが含まれます。
