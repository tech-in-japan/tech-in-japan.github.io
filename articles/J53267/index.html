ロシアのインターネット市場が西側に大きく遅れていることは秘密ではありません。 詳細を説明せず、正確な測定と計算を行わない場合、西部のインターネットには2年間のコマーシャルがあります。 当然のことながら、ロシア人は特に傍観することを望みません。そして、海外のVyakタイプのスタートアップが泥から裕福になっているのを見て、ロシア人は西洋の同僚の経験を完全に引き継ごうとしています。 待って、採用しないで、勇敢にコピーしてください！



一般化された用語については詳しく説明しません。vkontakte.ruのfacebook.com（またはORTの大Year日は星なしの大E日ですが、これは一般に別の話です）のクローンとして誰もが明確だと思います。 本日は、地元のパロディや人気サイトのコピーがあなたのサイトへの数百万の訪問をどのように可能にするかについてお話します。



覚えていれば、1年半ほど前に、odnoklassniki.ruのホームページのスクリーンショットであるが、刑務所のテーマに合わせてパロディがリメイクされた写真がネットワークに表示されました。 その後、特定のエフゲニー・チスチャヤコフが現れ、アイデアをさらに発展させ、この写真からサイトを作成することを考えました。



ここで読むことができます： http : //www.lenta.ru/articles/2008/02/27/odno/

そしてここ： http : //www.hb24.ru/url/?id=1855



これらのサイトでは、彼が他のプロジェクトの広告を掲載しました。すでに想像できるように、パロディからクラスメートへの移行により、これらのプロジェクトへの参加者が大幅に増加しました。



しかし、それはすべてプロローグ、いわばバックストーリーでした。 そして今日、 http：//kvartira499.ru/というサイトに出くわしました。その主なスローガンは、「499ルーブルのアパートを販売します」のようなものです。 あなたのことは知りませんが、すぐに自分の家を売りに出し、99ユーロでチケットを販売したオーストリア人についてのニュースを読んだことを思い出しました。



結局のところ、サイトの所有者（まあ、おそらくアパート）は、まさにエフゲニーチスチャコフです。



次に、このような呼び出しに対する公共の行動を分析し、予測してみましょう。 このようなスローガンを読んだユーザーはすぐに興味を持ち、リンクをたどります。 また、ニュースをフォローし、不動産を海外で販売する同様のケースと報道で彼らの報道を覚えているユーザーは、彼らの運を試し、プロジェクトの創設者と彼のゲームをプレイするリスクがあります。



したがって、最小限のコストで（男の子はいましたか？）開発と組織化のために、特定のポイント（この場合はサイト）に膨大な数のユーザーが流れます。 そして、訪問者に何を見せるかを決めるのはあなた次第です。常に大衆や典型的でありふれた戸口に持ち込みたいと思った有用な情報です。



すぐに予約しますが、サイトを評価するのではなく、現在モデルを検討していたことを思い出してください。 これは、最初のトピックから気を取られたり、他のチャンネルに注意を向けたりしないようにするためです。 最初は、直接的なリンクを提供したくありませんでしたが、伝えたい資料を「ベール」にすることはできませんでした。したがって、ここで提供されるすべてのリンクは、実用的な資料としてのガイダンスのみです。
