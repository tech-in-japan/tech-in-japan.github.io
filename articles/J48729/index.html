VP Suite 3.4。

この製品が提供する選択は非常に幅広いです。



私はそれを約6〜7ヶ月間使用したので、それについては非常に楽しい気持ちがありましたが、十分な否定的なものもありました。 しかし、これについては続けて。

製品が非常に開発されており、基準を満たしていることを嬉しく思いました。



さらに、製造元のWebサイトには多数の例がありますが、これも非常に重要です。 また、便利なインターフェイスと優れたヘルプですが、インターネットからヘルプをダウンロードする必要があります。これはマイナスです。

それ自体がコンストラクターであり、多くの要素が1つのシェルにアセンブルされています。 特定のコンポーネントをインストールできます。これにより、ライセンスが影響を受けます。

Visual Studio、Eclipse、Borland JBuilder、NetBeans / Sun ONE、IntelliJ IDEA、Oracle JDeveloper、BEA WebLogic Workshopなどの開発ツールとの非常に優れた統合を提供します。 かなり良いリストですね。

このバージョンで私が嬉しかったのは、彼らがcopy / paste要素のバグを取り除いたことです。 彼らは良い製品を得ました。

言語のリストはわずかに異なりますが、これはコード生成をサポートし、元に戻すことができる重要な要素でもあります。 Java、C ++、CORBA IDL、PHP、XML Schema、Ada、Python、C＃、VB .NET、オブジェクト定義言語（ODL）、Flash ActionScript、Delphi、Perl、Objective-C、Rubyのコードを生成できます。

反転可能：Java、C ++、CORBA IDL、PHP、XMLスキーマ、Ada、Python、C＃、Javaクラス、.NET dllおよびexe、JDBC。

誰かが私を修正するかもしれませんが、少なくともそのようなデータは開発者のサイトで提供されます。

データベースの操作を担当する部分も非常によく開発されており、サポートされているデータベースのリストは印象的です。 さらに、プロバイダーを提供し、「左」データベースを操作できます。 標準インストールでは、MySQL、MS SQL Server、Oracle、HSQL、Sybase ASE、Sybase SQL Anywhere、PostgreSQL、CloudScape / Derby、DB2、Ingres、OpenEdge、Informix、Firebird、FrontBase、Cache、SQLiteを受け取りました。 はい、チームの仲間は、リストの使用中に成長しました。

数字を与える価値はあると思います。なぜなら、私は今は仕事をしていないという事実にもかかわらず、私はこの製品を長い間賞賛することができるからです。
