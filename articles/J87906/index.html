ヨタスプリングアップデート

Yotaは、モバイルWiMAXネットワーク上のデバイスをサポートするプログラムを更新しました。

新しいYotaAccess 1.5.1リリースは、Intel WiMAXモジュールが組み込まれたラップトップ向けのバージョンと、ASUS Mobile WiMAX / Wi-Fi Centerルーター用のソフトウェアの新しいバージョンです。ダウンロード可能です。

内蔵WiMAXモジュールIntel Echo PeakまたはIntel 5150/5350を搭載したラップトップ向けの新しいバージョンのYotaAccess 1.5.1は、3月18日から自動更新形式で表示されます。 更新されたバージョンは、以前のWiMAX / Wi-Fiモジュールと、Sony VAIOモデルで最初に登場した新しいIntel Kilmer Peak別名Intel 6250モジュールの両方で、ラップトップの信頼性と安定性を高めます。 改善されたWi-Fi統合は、新しいIntel Wi-Fiバージョン13.1.1によって提供され、これもアップデートに含まれています。

ASUSモバイルWiMAX / Wi-Fiセンター。

3月上旬から、ASUS Mobile WiMAX / Wi-Fi Center用のソフトウェアの新しいバージョンも利用可能になり、デバイスの安定性が向上しました。

新しいファームウェアのインストールは、[ファームウェアの更新]ボタンを使用して、デバイスのウェブインターフェースから行います。 同時に、ソフトウェアの新しいバージョンは、コンピューターから（ 参照によりダウンロードした後）、コンピューターが現在ネットワークに接続されている場合は更新サーバーから自動的にインストールできます。

ファームウェアの更新の詳細な手順は、 ユーザーガイドの 31ページに記載されています。
