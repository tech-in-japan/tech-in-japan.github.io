Extreme Networks 802.11ac Wave 2─次世代のワイヤレスソリューション

すべてのITマネージャーは実際に同じ目標を追求します-コストの削減、エンドユーザーの生産性の向上、IT運用の組織のビジネス目標との調整。 これらの目標に貢献する技術は、慎重に検討する必要があることは明らかです。 これらには、エクストリームネットワークスがワイヤレスネットワークの分野で開発しているものが含まれます。









エクストリームネットワークスは、組織のIT部門が顧客、パートナー、従業員間の緊密な関係を通じてビジネスの成果を向上させるのに役立つ管理可能なネットワークソリューションを設計および提供します。





過去5年間、エクストリームネットワークスの株式の価値は大きな変化を受けておらず、昨年、Wave 2の出現により、着実に成長しています。



今日、Wi-Fiは、あらゆる組織やほとんどすべてのモバイルデバイスとアプリケーションにとって、ユーザーにとって最も便利な技術と接続性を備えています。 したがって、802.11ac Wave 2標準に基づくソリューションに対する顧客の高い関心は驚くことではありません。



ワイヤレステクノロジー、無線設計、アクセスポイントの構築、および過去20年にわたる実装の実践の改善により、すでにいくつかのネットワーク世代が生成されています：10 Mbit / s、100 Mbit / s、および802.11ac─1 Gbit / s 実際、802.11ac Wave 2は1.733 Gb / sのスループットを達成しています。



802.11ac Wave 2が高帯域幅を提供するだけでなく、ほぼすべてのデバイスとトランスポート接続をこれまでにないパフォーマンスと低コストで使用できるようにすることが重要です。





過去20年間でWi-Fiパフォーマンスを向上



今日のIT管理者は、より高い効率を求めながら、予算を削減して作業することを余儀なくされています。 この点で、ビジネスの観点から新しい投資を正当化するための議論の検索が前面に出てきます。



以下は、Wave 2を使用するビジネスケースの5つの主要コンポーネントです。



1.高度なワイヤレス技術を使用する利点を理解する



ワイヤレスソリューションについて議論するとき、彼らは通常彼らの帯域幅について話します。 ただし、Wi-Fiの主な問題ではなくなりました。特に今日は1.7 Gb / sのWave 2の生成について話しているためです。 これは、現代のほとんどの企業問題を解決するのに十分です。



さらに重要な問題は、多数のユーザー、デバイス、アプリケーションのサポートであり、その多くはリアルタイムのオーディオ再生とビデオストリーミングを使用しています。



Wave 1と比較してピークスループットが33％改善されたとしても、Wave 2テクノロジーは革新的というよりも進化的です。 これらは802.11nの革新に基づいており、802.11ac Wave 1と比較して根本的な違いはありません。



改善点には、スペクトル効率の向上（エラーの減少）およびMIMO（複数入力/複数出力）─周波数と時間だけでなくスペースを使用する機能が含まれ、スループットが大幅に向上します。



ビームフォーミングと呼ばれる追加の革新も802.11ac規格の一部です。 簡単な説明では、ビームフォーミングとは、特定のアクセスポイントとクライアント間の通信に最適な方向にラジオブロードキャストを特定の時間に集中させ、リアルタイムで調整して無線環境の動きやその他の条件を補正する機能です。



これまでのところ、ほとんどの802.11ac製品は、最大3つのMIMOストリームを提供しており、それぞれが最適な条件下で433.3 Mbit / sの最大スループットを提供していました。 Wave 2は4つのストリームで動作し、1,733ギガビット/秒を提供します。



興味深いことに、802.11ac規格では実際に最大8つのストリームを定義しています。 ただし、実際にこれを行う製品は、その複雑さとコストのために（少なくとも当分の間）一般的になる可能性は低いです。 さらに、現在、ほとんどのクライアント側デバイスは、1つ（まれに2つ）のスレッドのみをサポートしています。



Vawe 2マルチユーザーMIMOテクノロジーにより、複数のユーザーが特定のアクセスポイントからの単一の送信で各クライアントに固有の送信を受信できます。 つまり、他のユーザーにサービスが提供され、潜在的な帯域幅が浪費されるまで待つ必要はありません。 その結果、エンドユーザーの生産性とともに効率が劇的に向上します。

2.設備投資の最大限の活用



2008年の景気後退以降、ほぼすべての企業のIT予算に資本投資（Capital Expense、CapEx）が制限されています。同時に、ITの革新の継続的な流れは、インフラストラクチャと機器の両方の交換サイクルに影響を与え続けています。過去に比べて遅くなりました。 そしてもちろん、絶対コスト、特にROIを慎重に検討することは、今日のすべての資本投資の決定の一部であるべきです。



前述のように、無線LANの場合、製品の新世代ごとに価格/性能比が向上します。 しかし、実際には、これはスループットの増加によるものです。特定の機器（アクセスポイントなど）の価格は変わらないか、わずかに低下します。



802.11ac Wave 2製品は、Wave 1（および802.11nでも）と完全な下位互換性があります。 より高度なシステムアーキテクチャとファームウェアを備えているため、前任者よりも優れた機能を発揮します。 改善は、通常シームレスで安定した企業環境へのインストールにも影響しました。



最小の設置コストでの価格/性能比のこのような改善は、重要な疑問を提起します。つまり、既存のWi-FiソリューションをWave 2に置き換える時期です。明確な答えはありません。 各組織は、減価償却サイクル、利用可能な財源、およびITインフラストラクチャ全体と会社のIT戦略との組み合わせの観点から購入を評価する必要があります。



ただし、802.11g（またはそれ以前）のソリューションは、WLANの技術的な能力と価値の面であまりにも多くの変更が加えられているため、間違いなく交換の準備ができていることは明らかです。 既存の802.11nシステムは、特にMU-MIMOを利用することが理にかなっている場合、Wave 2からのプレッシャーが増大します。



3.運用コストを削減する



定義により、CapExとOpExが総所有コスト（TCO）を構成します。 これは、投資収益率（ROI）を決定し、長期にわたる資本コストを推定するための重要な指標です。





総所有コスト（TCO）構造



802.11ac Wave 2の資本コストの構成要素はかなり単純ですが、運用コスト（Operating Eexpense、OpEx）は、考慮すべきやや複雑な問題です。 この理由は、OpExが監視、管理、サポート、トラブルシューティングなどの側面で構成されているためです。それらには資格のあるIT専門家の注意が必要です。



CapExとは異なり、OpExは時間とともに増加する傾向があります。つまり、インフレや特別なスキルの競争により、人々はより高価になります。 企業はしばしばこの問題に対処し、IT担当者の生産性を向上させます。



したがって、IT部門の従業員の生産性の大幅な向上を約束するのは、Wave 2の導入です。 まず、WLAN操作の改善（帯域幅、容量、音声やビデオなどの時間制限のある通信のサポート）により、技術担当者への呼び出しが少なくなります。 さらに、Wave 2は、改善された管理と分析、結果の可視性、および短いトラブルシューティング時間を提供します。



そして最後に、Wave 2の実装方法に関係なく、─新規導入またはアップグレードとして、─柔軟で安定した安価なシームレススケーラビリティはWave 2の不可欠な部分です。したがって、Wave 2の導入に関連するITスタッフのコスト最小化されています。



4.エンドユーザーの生産性の向上



ネットワーク戦略は、ネットワークが提供するエンドユーザーのパフォーマンスを向上させるという観点から最もよく評価されます。 上記のように、ワイヤレスネットワークのユーザー数は常に増加しています。それは、会社の従業員数だけではありません。



Bring Your Own Device（BYOD）トレンドの広がりにより、同時に複数のアクティブなデバイスを持つユーザーからのサービスに対する需要がかつてないほど大きくなりました。 今日では、2つのデバイス（スマートフォンとタブレット）で同時にワイヤレスネットワークに接続しているユーザーに会うことができます。 3〜4台のデバイスを同時に使用することは珍しくありません。



遅い、特定の場所でアクセスできない、信頼できない、またはストリーミングメディア、リアルタイムテレフォニー、その他の「重い」通信のサポートを含む増大するニーズに対応していないネットワークのようなこれらのデバイス（ユーザー自身など）はありません。



モノのインターネット（IoT）の新しいアプリケーションも、ワイヤレスネットワークに対する追加の要求を提起します。 もちろん、多くのIoTソリューションは比較的少ないトラフィックで動作します。 ただし、ビデオ監視やリアルタイムプロセスなど、非常に重いサービスがあります。 これは、動作するWi-FiネットワークにWave 2レベルのニーズがあることを意味します。



ワイヤレスLANの信頼性と手頃な価格が高いほど、エンドユーザーの生産性は高くなります。 したがって、ほとんどすべての組織でワイヤレスサービスを配信するには、MIMOおよびMU-MIMOストリームがますます必要になります。



Wave 2の実際的な結果は、生産性を最大化することでTCOを削減し、ROIを向上させることにより、新しいテクノロジーへの比較的小さな投資で運用コストを大幅に削減できることです。



5.関連するネットワーク技術



802.11ac Wave 2に基づく製品は、ワイヤレスネットワークチェーン全体を最適化するための機能と技術の最新の進歩を体現しています。 これは、モバイルクライアント、アクセスポイント、コントローラー、スイッチとルーター、制御システムなどに適用されます。



Wave 2は、今日の無線LANの技術的基盤です。 ネットワークインフラストラクチャは、Wave 2に基づく製品に実装された新しい機能や機能にますます依存するようになります。これらは主なものです。

-管理 。 この機能には、コンテキスト駆動型の監視、アラートとアラート、レポート、監査などが含まれます。



-分析 。 これは、不確実性の条件の下で大量のデータを調査するために設計された一連の能力として定義されます。つまり、検索の対象を正確に決定することが困難な場合です。



-高度なセキュリティ 。 セキュリティは、どのネットワークでも常に重要な要素です。 ますます、WPA2 Wi-Fi機能がこれに使用されます（もちろん、これは重要なままです）。 より多くのセキュリティは、識別、アクセスポリシー、コンテキスト、コンテンツフィルタリングなどに基づいたネットワークの他の手段に依存します。



-クラウド 。 有線ネットワークと無線ネットワークの両方の展開における今日の主な傾向は、クラウド内の管理および関連機能です。 クラウドベースの管理の主な利点は、簡単なターンキー実装、シンプルなスケーラビリティ、フォールトトレランス、および強化されたセキュリティです。 CapExの設備投資は非常に経済的なOpExオプションに変換されるため、このアプローチのコストメリットも重要です。



-SDN 。 ソフトウェア定義ネットワークは、今後数年で有線ネットワークと無線ネットワークの両方でより大きな役割を果たします。 SDNを使用すると、ネットワークセキュリティ、ソリューションの整合性が向上し、実装が簡素化され、従来のソリューションよりも包括的で優れた管理と分析が提供されます。
