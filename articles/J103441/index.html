Rad Studio 2011 XE：Delphi、C ++ Builder、RadPHP、Delphi Prismの新機能

ごく最近、Embarcadero RAD Studio XE（2010）の新しいリリースがリリースされました。 バージョン2010と比較して、この環境の新機能に慣れることをお勧めします。







DelphiおよびC ++ Builder





DataSnapを使用した多層アプリケーションの開発


  • サーバーアプリケーションとクライアントモジュールを作成するための新しいウィザード
  • C ++ Builderを使用してDataSnapサーバーアプリケーションを作成するためのウィザード
  • 暗号化および圧縮ストリーム
  • 認証ロールのサポート




追加のユーティリティ


  • AQTime標準プロファイリングユーティリティキット
  • CodeSite Expressロギングおよび高度なデバッグキット
  • インターネットアプリケーションを開発し、IPファミリプロトコルを操作するための一連のSoftware IP Worksコンポーネント
  • ファイルの違いを比較および検索するためのテキスト比較ユーティリティを超えて




クラウドコンピューティング


  • 基本的なWindows Azureストレージシステムを使用するための新しいコンポーネント（ブロブ、キュー、およびテーブルがサポートされています）
  • Amazon EC2サーバーに開発ソフトウェアを迅速にデプロイする機能




IDEでのSubversionサポートの統合


  • SubversionはProject Managerとソースコード変更履歴マネージャー（History Manager）に統合されています
  • リポジトリでの基本的な操作がサポートされています：インポート、サーバーからの更新、ログのコミットと表示
  • 差異を表示して結果をマージするウィザード
  • Open Tools APIを介してSubversionツールを操作するためのソースコードは、オープンソースプロジェクトとして利用できます。




生産性と使いやすさのツール


  • モデラーのコード生成が改善され、より良いコードが得られました。
  • Delphiアプリケーションのメソッドからシーケンス図を生成する機能が追加されました
  • Ctrl-Dのコードフォーマット設定を追加
  • 環境機能にすばやくアクセスするための強化されたテキスト検索とIDE Insightメニュー
  • 変更されたコード行間を移動するためのいくつかのホットキーを追加しました




組立プロセスの自動化


  • FinalBuilder自動ビルド管理パッケージが追加されました
  • ソースコードを自動的にフォーマットするFormatter.exeを追加しました
  • ソースコード監査を自動的に実行し、メトリックを計算するAuditsCLI.exeを追加しました
  • 自動ドキュメント生成用にGenDocsCLI.exeを追加しました




言語、コンパイラ、およびライブラリの改善


  • 更新されたVCL、RTL、およびSTLライブラリ
  • 正規表現を使用するためのサポートが追加されました
  • TStringsおよびTDateTimeクラスの拡張機能
  • C ++ BuilderでのRTTIサポートの拡張、新しいRValueリンクの追加、stdの移動セマンティクス：文字列、およびC ++の他のVCLの改善
  • OpenTools APIに内部タブのサポートを追加しました
  • コンパイラーのパフォーマンスが向上しました




ToolCloud統合


  • Embarcadero ToolCloudと連携する機能が追加されました




RAD Studio XEを購入すると、DelphiおよびRAD Studioの以前のすべてのバージョンのライセンスをプレゼントとして受け取ります！



Delphi Prism XEの新機能





  • Subversionサポート
  • MonoDevelopを使用したWindowsおよびMac OS Xでのクロスプラットフォーム開発
  • .NET 4.0およびSilverLightのサポート
  • IDEは、Visual Studio 2010 Shellに基づいており、そのすべてが暗示しています。
  • 並列プログラミング、関数型プログラミング（新しいif、caseおよびfor）、BigInteger定数のサポート。 拡張メソッドを定義するための新しい構文が追加されました
  • Delphi Prismコードへの自動変換により、C＃コードをインポート、コピー、および貼り付けする機能
  • iPhone OS用のアプリケーションを開発する機能
  • 環境から直接コードを難読化する機能
  • CodeSiteロギングシステムの統合を高める




RadPHP



新しいRadPHPを使用すると、環境から直接PHPサイトを開発およびデバッグできます。 組み込みライブラリを使用すると、FaceBookのアプリケーションを作成して、サードパーティのコンポーネントを介して機能を拡張できます。 データアクセスアプリケーションの開発がかつてないほど容易になりました。
