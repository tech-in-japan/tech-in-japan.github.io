MS SharePoint Server 2010の予備システム要件が発表されました

これまでのところ、あまり多くのニュースはありませんが、重要なことは次のとおりです。

  • IE 6のサポートは終了しています。 それにもかかわらず、彼は終わりに近づき、これを喜ばずにはいられません。 IE 7および8、FF 3.0でサポートされ、Safari 3.0のより良いサポートで動作します
  • MS Office SharePoint Server （MOSS）は完全に64ビットになります 。つまり、32ビットプロセッサのバージョンはまったくリリースされません。 ただし、これは古いニュースです。
  • データベースとして使用できるのは、SQL Server 2005/2008 の64ビットバージョンのみです。
SQL Serverの64ビットバージョンのみのサポートに関する決定は、ライブプロジェクトのテストと観察に基づいて、64ビットデータベースがMOSSの速度とそのスケーラビリティに大きく影響することが明らかになり、32ビットサポートを挿入しないことにしたという事実によって説明されます。 MOSS 2007以来、64ビットバージョンのSQL Serverを使用することを常に推奨されていました。 ええ、鉄の会社を更新する機会です。
