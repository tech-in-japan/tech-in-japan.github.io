この記事では、YouTubeアナリティクス、Googleアナリティクスと統合する利点、およびGoogleタグマネージャーを正しく設定することで取得できる追加データについて説明します。











YouTubeは、世界で3番目に訪問されたサイトです。 しかし、最近では、ますます多くの人々が検索エンジンとしてそれを使用しています。 現在、2番目に大きなグローバル検索エンジンです。 一部のテーマニッチでは、オーガニックトラフィックが年間最大70％増加しています。 ビデオコンテンツはテキストよりも情報量が多いため、これは驚くことではありません。



したがって、競争は急速に成長しています。 マーケティングプランを作成し、分析と密接に連携せずにチャネルを宣伝することは、数年前よりもはるかに困難になりました。

YouTubeアナリティクス



YouTubeアナリティクスは、最も重要なビデオ分析ツールです。 その目的は、個々の動画とチャンネル全体の有効性を追跡することです。 膨大なデータ層（トラフィックソース、表示デバイス、人口統計など）が含まれており、その分析は難しくありません。 以下に、統計を分析するときに注意する必要があるいくつかの機能と、ビデオのパフォーマンスを向上させるためのトリックを紹介します。



視聴者維持率



ビデオの15〜25秒間（特に最初の15秒間）に絶対保持率が30％を超えて急激に低下することはありません。 以下のようなグラフが表示されている場合は、統計で-間違いなく始まりを変えてください。 おそらく、ビデオのロゴが長すぎるか、ユーザーを悩ますものです。 または、単なるミニチュアは無関係で誤解を招くかもしれません。 視聴者維持スケジュールにこのような急激な失敗が含まれている場合は、理解する必要があります。







ビデオの特定のポイントでの保持の急激な増加または減少を慎重に分析します（下のグラフでより明確に確認できます）。 通常、急成長は、素材が興味深く、視聴者のようであることを意味します。 しかし、これは常にそうではないことを忘れないでください。 おそらく、反対に、この場所には十分な情報がないか、引用されている間違いがあります。







多くの場合、ビデオでは人々が脱落する瞬間に出くわしますが、それらを使用しないことは不可能です。 そのような場合は、すぐに何か面白いものや有用なものがあるという情報を注釈として付けてください。 一部の人々は去らないでしょう、保持はわずかに改善します。



加入者



重要な測定基準が満たされ、直接および間接的に多くの影響を受けます。 直接的な影響があります-サブスクライバーの数はランキング要因の1つです（私の意見では、すでにかなり少ないです）。 間接的な効果があります-多数のチャンネル登録者がいるチャンネルを見ると、通常はより忠実です。 このような視聴者は、保持率をわずかに高め、より頻繁に、チャンネル登録者になる可能性が高くなります。



最も多くのチャンネル登録者を引き寄せる最適な動画を選択し、チャンネル登録の提案とともにアノテーションとカードを追加します-消耗はさらに大きくなります。



重要：エンディングを追加することを忘れないでください？Sub_confirmation = 1アノテーションとカードのチャンネルに（たとえば、タンクの場合： https : //www.youtube.com/user/WorldOfTanks ?sub_confirmation=1）。 このリンクはサブスクリプションウィンドウにリダイレクトされます-変換が増加します。



いいね、コメント、嫌い、共有



現在、これらの指標は数年前ほど重要ではありませんが、無視することはできません。 さらに、間接的な効果もあります。あなた自身は、おそらく興味深いトピックに関する長いビデオを見る可能性は低いですが、嫌悪感がたくさんあります。



トラフィックソース



最初のトラフィックがチャネルに表示されるとき、それがどこから来て、どの外部および内部チャネルが機能するかを理解することは非常に重要です。 この指標はトラフィックを増やすために重要だと思います。なぜなら、あなたの動画のどれが他のサイトに喜んで埋め込まれているかを正確に示しており、キーワードによってうまく宣伝されており、メインページに表示される可能性が高く、バイラルなどです



ここで私の記事から主な指標について学ぶことができます 。





YouTube + Googleタグマネージャー



Googleタグマネージャーの開発により、サイト上の動画コンテンツに対するユーザーの操作を簡単に追跡できます。 リソースのビデオビューの統計は次のようになります。







なぜ誰に役立つのでしょうか？

•多くのブロガーやサイト所有者が行動要因を構築して、投稿にビデオを埋め込みます。 人々はビデオを見て、サイトで過ごす時間を増やします。 適切に調整されたGTMは、ユーザーが最後まで視聴するビデオの種類を理解するのに役立ちます。 これにより、サイトで費やす時間を増やすことができます。

•サービス、オンラインストア、またはランディングの場合、GTMを使用するとリマーケティングリストをより効率的に収集できます。 ビデオを見る人は、製品を購入する可能性が高くなります（または、別の必要なアクションを実行します）。

•GTMを使用して独自のビデオを作成するサイトの所有者は、視聴者をよりよく理解できます。



Googleタグマネージャーの設定方法



実際、設定には10分以上かかりません。 上級ユーザー向けの便利な手順は、 Optimizesmart.comで入手できます。

以下のこのガイドでは、GTMを使用したことがない人でもインストールおよび構成の方法を理解できるように、詳細に説明しました。



重要1： iframeを使用してサイトに挿入されたすべてのビデオを追跡します（サイトにビデオを埋め込む標準的な方法）。







重要2：すべてのスクリーンショットは、GTM 2.0の最新バージョンで作成されています（執筆時点で関連）。



1）GTMに登録/ログインします： tagmanager.google.com 、サイトにコードを配置し、Googleアナリティクスアカウントにリンクする新しいタグを追加します（まだ実行されていない場合）。







2）YouTube is present変数を作成します。 変数はネイティブJavaScriptコードです。 以下のコードを挿入します。 ページにビデオがあるかどうかを確認します。

1 // Return "true" if there is at least one Youtube video on the page

2 function () {

3 for (var e = document.getElementsByTagName('iframe'), x = e.length; x--;)

4 if (/youtube.com\/embed/.test(e[x].src)) return true;

5 return false;

6 }











3）「YouTubeプレゼント」トリガーを作成します。 イベント選択-ページビュー、トリガータイプ「DOMモデル準備完了」。 アクティベーション条件-以前に作成された変数「Youtube is present」は「true」に等しい。







ページにビデオが存在する場合も、トリガーがオンになります。



4）次に、タグを作成し、スクリプト自体を直接貼り付ける必要があります。これにより、ビデオ視聴の割合を追跡し、情報をGoogleアナリティクスに送信し、見出しを便利に表示できます。

スクリプト自体はこのリンクにあります。

タグ「Youtube Listener」を作成し、カスタムタグ「HTML」を選択して、スクリプトコードを挿入します。







アクティブ化条件は、以前に作成された「YouTube present」トリガーです。







5）すべての情報を転送するには、イベントアクション、イベントカテゴリ、イベントラベルの3つの変数をさらに作成する必要があります。 タイプ-データレベル変数。

「データレベル変数の名前」フィールドに、それぞれ「eventAction」、「eventCategory」、「eventLabel」と書き込みます。 データ層バージョン-バージョン2。









また、すべてのデータをGoogleアナリティクスに送信するカスタムトリガーを作成する必要があります。 イベント選択-ユーザーイベント。 アクティベーション条件に、ペンでイベントの名前「YouTube」を記入します。







6）すべてのデータを収集してGoogleアナリティクスに送信するために、別のタグ「UA YouTubeイベント」を作成します。 製品選択-Googleアナリティクス、タグタイプ選択-ユニバーサルアナリティクス。







次に、タグ設定について説明します。 識別子にユニバーサルアナリティクスを配置した識別子では、追跡のタイプはイベントです。 カテゴリについては、-イベントカテゴリ、アクション-イベントアクション、ショートカット-イベントラベルを選択します。 アクティベーション条件はYoutubeイベント（5番目のステップで作成されたトリガー）です。







一般に、次の図を取得する必要があります。







このタグは、YouTube動画とのすべてのインタラクションをGoogleアナリティクスに送信します。



7）作業を確認するには、右上隅の「公開」の横にある矢印をクリックして、「プレビューとデバッグ」を選択します。







次に、YouTubeビデオが挿入されているサイトのページに移動し、[再生]をクリックしてから停止します。 3つのタグすべてをアクティブにする必要があります。







すべてがアクティブになったら、Google TMに戻り、「公開」をクリックします。

その後、自分でビデオを再生して停止できます。 すべてのデータは、[リアルタイム]タブのGAの[イベント]セクションに表示されます。







突然問題が発生した場合、または何かが機能しない場合は、コメントに書いてください。喜んでお手伝いします。



YouTubeアナリティクス+ Googleアナリティクス



YouTubeアナリティクスは、ビデオ分析のための豊富な情報を提供します。 ただし、チャネルを分析する場合は、GAとの統合が追加情報の取得に役立ちます。



2人のアナリストをリンクするには、指示に従ってください。

1）GoogleアナリティクスでYouTubeチャンネルの新しいプロパティを作成し、トラッキングIDをコピーします。







2） www.youtube.com/advanced_settingsのリンクに従って、YouTubeの詳細設定を有効にします。







ページの一番下には「GA Identifier」フィールドがあり、最初の段落から識別子を挿入する必要があります。







その後、YouTubeはすべてのデータをGAに送信します。



これは分析に何を提供し、これから何を得ることができますか？



Googleアナリティクスでは、チャンネルを通常のサイトとして認識し、すべての情報を提供します。 ただし、このデータはすべてURLチャネルの一部として表示されることを理解する必要があります 。 たとえば、タンクチャンネルのすべてを測定します： www.youtube.com/user/WorldOfTanks



GAは、このURLと、このURLを含むすべてのカテゴリとフィード（たとえば、 www.youtube.com / user / WorldOfTanks / videos 、 www.youtube.com / user / WorldOfTanks / playlistsなど）の統計情報を保持します。 また、設定でシンプルで覚えやすいURLを使用しない場合（当社の場合はwww.youtube.com/channel/UCVnah-S-sByndtg9cHBxlOA ）、およびそれを含むすべての派生物に関する情報も、チャンネルのメインページに表示されます。



ただし、動画のURLに関する情報はありません。 メインページ、プレイリスト、または他の場所で動画をクリックしても、GAはこれを修正しません。



GAとYouTubeの通信方法がわかったので、取得できる追加情報を確認します。



•広告キャンペーンの評価。 YouTubeチャンネルのメインページで広告を購入する場合は、統合が必要です。 YouTubeアナリティクスは、その有効性を測定しません。 しかし、GAでは、この広告ソースからのユーザーのサイト滞在時間、直帰率などを確認できます。

•コンバージョンの追跡と分析。 多くの人は、チャンネルURLにSub_confirmation = 1を追加すると、サブスクリプションページに直接スローされることを知っています。 たとえば、当社の場合： www.youtube.com/user/WorldOfTanks?sub_confirmation=1

GAは、このURLのクリック数を示します。 このデータとYouTubeアナリティクスのチャンネル登録者数を関連付けることにより、ユーザーの行動を理解できます。 チャンネル登録者とクリックの比率が非常に低い場合、何かがチャンネル登録を強制します。 おそらく、メールにログインするには注釈で尋ねる必要があります。 あなたはそれが無料であることを書く必要があるかもしれません。 チャネルごとに、すべてが個別です。

•特定のページの統計。 GAは各ページのクリック数を示します。 たとえば、「チャンネルについて」ページからメールをメールに送信するコンバージョンを測定できます。 広告を販売していて、チャンネルページに多くのアクセスがある場合、コンバージョンの費用を見積もることができます。



多くの場合、トラフィックを引き付けるために予算を費やすよりも、分析の有能な仕事のほうがはるかに収益性が高い場合があります。 ご質問やご意見がありましたら、お尋ねください。すべてお答えします。
