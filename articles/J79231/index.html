PHPでWindowsプラットフォームの機能を使用する

Microsoftには、 PHPアプリケーションから使用するWindowsプラットフォームサービスのラッパーを開発する特別なチームがあります 。 これらのラッパーは相互運用性ブリッジと呼ばれ、すべての種類のテクノロジとWindowsプラットフォームとの対話専用の同じ名前のWebサイトに投稿されています。 この短い記事では、Windowsプラットフォームで作業するPHP開発者に役立つラッパーについて説明します。



PHPでIE8のアクセラレーターとWebフラグメントを作成する例



Webslices＆Acceleratorsプロジェクトを使用したPHPのサンプル 、CodePlexプロジェクトサイト： webslicesandaccelphp.codeplex.comこのプロジェクトには優れたチュートリアルがあります。



任意のPHPサイトでIE8のカスタムWebパーツとアクセラレーターを作成するために使用できるPHPコードの例。



SQL Serverで作業するためのPHPコードスニペットを作成するためのウィザード



PHP用SQLプロジェクトCRUDアプリケーションウィザード 、CodePlexプロジェクトサイト： sqlcrudphpwizard.codeplex.com 使用例は、このプロジェクトの開発者が作成したビデオにあります。



SQL Serverに保存されたデータを使用した基本操作用のPHPコードスタブを作成できる小さな.NETアプリケーション。 また、Windows Azureでホストされるクラウド内のデータを操作するためのコードの作成をサポートするプロジェクトも含まれています。 調達サイトのジェネレーターとして機能します。



ADO.NET Data Servicesとの統合



Project Toolkit for PHP with ADO.NET Data Services 、CodePlexのプロジェクトサイト： phpdataservices.codeplex.com Channel 9のプロジェクトに関するビデオ。



このライブラリは、ADO.NET Data Servicesのラッパーを提供します。これにより、ADO.NET Data Servicesを介してRESTful APIを提供する.NETアプリケーションと簡単に統合できます。 ADO.NET Data Servicesのサポートの実装は難しいタスクではないため、この方法で既存の.NETおよびPHPアプリケーションを統合できます。



Windows Azure SDK



Windows Azure SDK for PHPプロジェクト、CodePlexプロジェクトサイト： phpazure.codeplex.com



PHPで記述されたWindows Azureインフラストラクチャのラッパー。 このSDKを使用すると、PHPアプリケーションをWindows Azureサービスに統合することを考慮する必要はありませんが、アプリケーションのビジネスロジックコードを記述します。
