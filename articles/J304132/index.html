少しプロモーション写真








































ご覧のように、一般的に、モデルは同一であり、変化がありますが、革命的ではなく、むしろ進化的です。

内部の変更：

  • Allwinner A20（デュアルコア）をAllwinner H8（8コア）に置き換えました。良いか悪いかはあなた次第です！
  • SoCとともにビデオコアも改善されました。Mali400MP2がPowerVR SGX544になり、ビデオサブシステム全体のパフォーマンスにプラスの影響を与えるはずです。 さらに、メーカーはSoC H8をホームエンターテイメントセンターの基盤として位置付けています。
  • 更新されたWi-Fi + BTチップはAP6210がAP6330になりました。 802.11aモードが追加されました。これはアップデートには示されていませんが、チップのデータシートにAP6330があり、FMレシーバーがあります。 これでラジオが簡単になります）


周辺機器：

  • ビデオ：VGAの代わりに、DisplayPortを使用できます。 HDMIはどこにも行っていないので、2つのモニターを接続できますが、同じ画像が表示されます
  • オーディオ：ヘッドフォンジャックに追加されたマイク入力
  • 接続：54ピンコネクタの代わりに-現在70ピン。 CB3と互換性のある54ピン。 別のコネクタに配置された互換性のないピン


CB3に関しては、CB5はAndroid 4.4.4とUbuntu Linaro 14.04を完全にサポートしていると主張しています（CB3のコメントで思い出したように、Linuxのハードウェアビデオアクセラレーションに問題があります。 しかし、他のディストリビューション（たとえばwww.armbian.com ）の使用を禁止する人はいません



HDD-RAID + USB3シールド





しかし、それだけではありません。なぜなら CB5とともに、メーカーは拡張ボードHDD-RAIDを発表しました



このボードを使用して、追加のSATAドライブを接続し、それらをRAID（RAID0、RAID1、JBOD、PMのバリエーション）に結合できます。

実際、2つのディスクをこのボードに接続し、シールドとCB3 / CB5をメインボードワイヤに接続します（そして、メーカーはCubieboard3 / Cubietuckとの互換性を主張しています。私はCB3の所有者として嬉しいです。）

また、このボードにはUSB 3.0コネクタが搭載されており、これも朗報です。
