AsteriskNowディストリビューションを使用したDigium電話のテスト





以前、このメーカーのIP電話をSwitchvoxユニファイドコミュニケーションシステムでテストしました。 今日は、人気のある無料のAsteriskNowソリューションに注目します。



利用可能な最新のディストリビューションはバージョン2.0.2で、認定されたAsterisk 1.8.11-cert1とFreePBX 2.10.0 Webインターフェイスが含まれています。



最初からインストールしても問題は発生しません。電話との高度な統合に必要なモジュールはすでに含まれています。 また、FreeBPXが以前にインストールされていた場合は、システムの（管理者-モジュール管理者）インターフェースから「Digium Phones Config」をインストールする必要があります。



電話のDigiumファミリは通常のSIPデバイスであるため、どのSIP PBXでも使用できます。 しかし、もちろん、テレフォニー（オープンソースアスタリスク）の世界で革命を起こした企業は、従来のSIP IP電話のこのような控えめな機能に限定することはできませんでした。



DPMA


Asterisk用Digium Phone Moduleは、Asteriskの商用モジュールであり、電話とサーバー間の緊密な統合を提供します：自動更新、アプリケーションなど。



ライセンスはメーカーから入手する必要がありますstore.digium.com/productview.php?product_code=804-00032

製品は無料で、数分後にアクティベーションキーが受信トレイに届きます。



サーバーのセットアップ


接続に移動-Digium Phones =>ライセンスの取得/登録







残っているのは、Webインターフェースにデータを入力し、ライセンス契約の条項に同意することだけです。







アクティブ化後、各SIP拡張機能（拡張機能）をDigium電話で構成できます。



基本設定を考慮してください。





  • 簡単モード 。 シンプルモードでは、各電話をカスタマイズすることはできません。たとえば、すべての電話でデフォルトディレクトリが縫製されます。
  • グローバルピン 電話の初期設定時にパスワードとして設定されるデジタルピン。
  • 電話認証方法 。 デフォルトでは、構成中に認証は不要であり、各ユーザーは必要な追加の認証をインストールできます。 MACアドレス、個人パスワード、またはグローバルパスワードにバインドすることにより制限できます。
  • サービス名 。 電話の起動時にユーザーに表示されるサーバーの名前。
  • 登録サーバーアドレス/登録サーバーポート 。 システムのホスト名またはIPアドレス。
  • 桁マップ 。 必要に応じて、数値を入力するための特別なルールのフィールド。
  • Web UIが有効になっています。 携帯電話のウェブインターフェースを有効/無効にします。 セキュリティ上の理由で必要なアイテム、または着信音の設定、輝度レベルの表示、その他のSIP回線などに必要なアイテム
  • 未使用の回線キーのBLF 。 行キーは、ノートブックの連絡先を表示するために使用されます。
  • 着信音 。 システム内のすべての電話の着信音を同時に変更します。
  • アイドル画面イメージ 。 会社のロゴ、PNG画像がサポートされています。
