無料のVeeamツールの選択

今日、Veeamの5つの最も人気のあるフリーウェアユーティリティに関する情報を1つの投稿に要約します。これは、VMware vSphereおよびMicrosoft Hyper-Vの物理環境と仮想環境の両方の管理者の日常業務に役立ちます。 この投稿では、次の情報を提供します。







  • VMwareおよびHyper-V用Veeam Backup Free Edition
  • VMwareおよびHyper-V用Veeam ONE無料版
  • Veeam Agent for Windows Free Edition
  • Veeam Agent for Linux Free Edition
  • VMware、Hyper-V、およびVeeam用のVisio要素セット


猫へようこそ。













Veeam Backup Free Edition



Veeam Backup Free Editionは、仮想環境のバックアップ、復元、および管理を行うための効果的なツールであり、特にVeeamZIP機能に関してユーザーに記憶されています。 VeeamZIPを使用すると、仮想マシンをシャットダウンして、ローカル、ネットワークフォルダー、またはリムーバブルデバイス（USBまたは外部ハードドライブ）に保存することなく、仮想マシンの完全バックアップを作成できます。 無料版のもう1つの優れた機能は、ライセンスが一時的ではなく、VMware環境またはHyper-V環境のVMの数に制限がないことです。









VeeamZIPファイルを作成する場合、Veeam Backup Free EditionはVMデータをコピーし、圧縮して、フルバックアップ（VBK）ファイルに保存します。







Veeam ONE無料版



経験と常識から、問題を予防する方が結果を分析するよりも簡単であることが多いことが示唆されています。 Veeam ONE Free Editionは、VMwareおよびHyper-V環境を監視し、インフラストラクチャに影響を及ぼす前に問題を特定できる無料の監視ツールです。 このツールの主なタスクは、注意を要するイベントのリアルタイム通知、リソース割り当ての優先順位付け、動的リソースグループの作成などにより、IT管理者の頭痛の種を取り除くことです。









管理環境にとって潜在的に危険なイベントが発生すると、通知がトリガーされます。 ユーザーはすべてのVeeam ONE通知を表示できます。 各通知には、問題、その原因、解決策、および追加ソースへのリンクの詳細を記載したナレッジベースの記事があります。







Veeam Agent for Microsoft Windows無料



Veeam Agent for Microsoft Windowsは、Windowsを実行しているコンピューター、ラップトップ、物理サーバーからデータをバックアップおよび復元するためのシンプルで効果的なツールです。 これにより、コンピューター全体と個々のファイルまたはフォルダーの両方を外部メディア、NAS共有フォルダー、またはVeeam Backup＆Replicationリポジトリにすばやくバックアップできます。 ウイルス感染やデータへのアクセス不能（hi、 Wana decrypt0r ）、OSの読み込みエラーなど、緊急の状況では、必要な情報を数分で復元して仕事に戻ることができます。 最新バージョンは、Microsoft Azureでの、またはVM Hyper-Vとしてのバックアップ物理マシンのリカバリ、合成フルバックアップの作成、およびその他の多くの便利な機能をサポートしています。









Veeam Agentバックアップから、個々のファイルとフォルダーを復元できます。 バックアップ内で検索し、元の場所と新しい場所の両方に復元できます。







Veeam Agent for Linux FREE



Veeam Agent for Linux FREEは、物理的なLinuxマシンをパブリッククラウドにバックアップできる当社の武器庫の目新しさです。 エージェントは、ボリューム、ファイル、またはシステム全体のイメージレベルでバックアップを作成します。 使いやすく、経験の浅いユーザー（管理者）にも適しています。 管理は、コマンドラインまたはグラフィカルインターフェイス（GUI）を介して行われます。









現在のバックアップセッション、および完了したセッションの詳細な統計を表示できます。







ボーナス：VMware、Hyper-V、およびVeeam用のVisio要素セット



ボーナスとして、 VMware、Hyper-V、およびVeeamのVisio要素のセットを保持して、クールなチャートを描画できるようにします。 Microsoft Visio 2010、2013、2016でプレゼンテーション、レポート、ドキュメントに使用できます。
