AnyChart Stock-柔軟で高速かつインタラクティブな日時ベースのチャート

みなさんこんにちは。

最終的に、新しいプロジェクトであるAnyChart Stockの開発が完了しました。



これは、時間順に並べられたデータを表示するインタラクティブなグラフを作成するためのコンポーネントです。 最も一般的な例は、財務チャートです。



ランタイムコンポーネントはFlashで記述されています。 構成はXMLまたはJSON（もっと好きな人は誰でも）で指定され、データはcsv形式です。



開発には1年半以上かかりましたが、それだけの価値がありました。







猫の例の下で、いくつかの機能、そしてもちろん、habraskidka！





コンポーネントの使用例：

anychart.com/products/stock/online-demos/forex-charts/index.html

anychart.com/products/stock/online-demos/stock-portal/index.html



Webサイトからコンポーネント自体をダウンロードできます： http : //anychart.com/products/stock



特徴



私たちを他の同様のプロジェクトと区別するいくつかの機能について説明します。



性能


私たちは何とか頭脳を壊しましたが、大量のデータ（gzip圧縮されたcsvファイルから100,000ポイント以上）を表示するのに十分な速度のエンジンを作成しました。

たとえば、このデモには約50,000ポイントがあります： anychart.com/products/stock/gallery/samples/using-huge-amount-of-data-more-than-46000-points.html



XML / JSON設定


チャートの構成を可能な限り作成および変更するプロセスを簡素化するようにしました。



詳細なドキュメントとリファレンスを含むXML構成に、XMLスキーマを追加しました。これにより、構成が非常に簡単かつ迅速になりました。



また、JSONを介してグラフを構成する機能を追加し、もちろん、その構造に関する詳細なリファレンスを作成しました。



ギャラリーのすべてのサンプルとドキュメントがダウンロードされ、好きなように選択できます。 いずれの例でも、設定行をクリックして、このオブジェクトまたはノードのヘルプを開きます。 あなたのためのすべて;-)



jpg / png / pdfへのエクスポート


かなり頻繁にスケジュールを維持したいです。 jpg、png、pdfなどの一般的なすべての形式で保存できるようにしました。 さらに、グラフをpdfに画像またはpdfに組み込まれたswfとしてエクスポートできます。



外部API


JavaScriptを使用して外部からチャートを変更し、そこからイベントを取得するのに十分なAPIを作成しました。 外部APIを使用してできることの例については、こちらを参照してください。

anychart.com/products/stock/online-demos

一般に、すべての機能について話すのはかなり難しい作業であり、数千もの機能があります:)そして、イベントマーカー、およびテクニカルインジケータを構築する機能、そして...一般的に、誰が気にかけますか？
