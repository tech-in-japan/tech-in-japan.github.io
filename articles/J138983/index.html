Mozillian「Boot to Gecko」：Webはプラットフォームです

Mozillovsky Boot to Gecko（B2G） -オープンWeb用の完全かつ十分な（スタンドアロン）オペレーティングシステムを構築するプロジェクト。 その目的は、デスクトップや携帯電話のアプリケーションでWebテクノロジーを最初の選択肢にすることです。 そして、B2Gは独自の単一ベンダーのアプリケーション開発キットを置き換えることができると信じています。 この道で特定の成功を達成し、それらをあなたと共有したいと思います！



Geckoプロジェクトを起動します



まず、プロジェクトのコンポーネントを見てみましょう。



タスク



Boot to Geckoは、オープンソースWebテクノロジーに基づいたモジリアンモバイルオペレーティングシステムです。 これは、2012年2月にMobile World Congressで発表された、オープンWebのハードウェアプラットフォームの開発環境です。



技術



Boot to Geckoアーキテクチャにより、特定の単一プラットフォームAPI上でアプリケーションを実行する必要がなくなります。 HTML5を使用すると、どこでも開発者は、メーカーが管理しているプラ​​ットフォームの規則や禁止事項に煩わされない、Web上でユーザーと対話するための素晴らしいアプリケーションとツールを作成できます。



利用可能なオープンスタンダード



すべてのMozillaプロジェクトと同様に、ソースコードはオープンでアクセス可能であり、プロジェクトは完全にオープンスタンダードに基づいています。 また、オープンな標準（テレフォニー管理、SMS、ビデオカメラ、Bluetooth、USB、NFC）がまだない場合、Mozillaは標準化委員会やサプライヤーとやり取りして標準を作成します。 これについての詳細は、ブログ投稿「 MozillaとモバイルWeb APIの進化 」をご覧ください。

オープンWebデバイスに会う



本日、 World Mobile Congressで発表したように、Open Web Devicesに関する非常にエキサイティングなニュースがあります！ 私たちは、電気通信事業者Telefónicaと協力して行動します（ プレスリリースで発表されたように）、相互作用の目的は次のとおりです。

  • Open Web （Open Web）に基づいたデバイスでのHTML5のハードウェアサポート。これは、スマートフォン（スマートフォン）の機能を備え、通常のスマートフォン（フィーチャーフォン）のコストがかかります。

  • それらのインターフェイスは標準化のためにW3Cに送信され、公開されます。

  • アプリケーション開発者に新たな機会が開かれ、クロスプラットフォーム標準としてのHTML5の役割が拡大しています。
Qualcommチップセットに基づいて、Mozilla、Telefónica、およびQualcommは共同でプロトタイププラットフォームの圧倒的な機能を作成します。 彼女のブラウザはすでにHTML5テスト で 317ポイントを獲得し ています 。 手頃な価格に関しては、TelefónicaのCarlos Domingoによると、Boot to Geckoの携帯電話はiPhoneの10倍も安くなります。
