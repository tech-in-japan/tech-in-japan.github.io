.NET Forge CMSおよびWeb Platform Installerを使用した迅速なサイト開発、パート1





最近では、1C-Bitrixは.NET Forge CMSと呼ばれる新しいコンテンツ管理システムのリリースを発表しました。これは、無料のコミュニティバージョンを含む2つのエディションで提供されます。 このシステムには 、インターネット上の商業活動やその他の活動のために、モダンで高速な機能サイトを作成するための強力なツールがあります。



新しいCMSのその他の利点には、ASP.NETプラットフォーム、オンラインストアのテンプレート、企業Webサイト、ブログ、コミュニティサイト、ビジュアルエディター、Hermitageインターフェイス、Webサイト広告管理システム、評価、投票、ギャラリー、フォーラム、ブログなどがあります。 、脅威やハッキングの試みに対する予防的な保護、形態学的検索、組み込みの更新システム。



.NET Forge CMSが、他の多くのコンテンツ管理システムの中でも特に、Web Platform Installerを介した迅速で自動化されたインストールに利用できるようになったことをお知らせします。 これは、自分のサイトのインストールと設定がはるかに簡単になり、すべてのユーザーが利用できるようになったことを意味します。



以下では、Web Platform Installerを最短時間で使用して、完全に完成した自社Webサイトを展開および構成する方法について説明します。



設置



.NET Forge CMSを自動的にインストールするには、Web Platform Installerがインストールされている必要があります。 最近リリースされた最終バージョンは、 次のリンクからダウンロードできます。 ここでは、製品のローカライズ（14の言語が利用可能）と、オペレーティングシステムに対応するバージョン（32ビットまたは64ビット）を選択できます。 WebPIの最新バージョンをダウンロードする別の方法があります 。これは公式の製品ページで、 http：//www.microsoft.com/web/downloads/platform.aspxから入手できます 。 このページでは、製品に関するもう少しの情報を取得し、オペレーティングシステムのバージョンに対応する現在のバージョンをダウンロードできます。



Web Platform Installerがコンピューターにインストールされたら、.NET Forge CMS Webアプリケーションテンプレートのギャラリーページへの次のリンクをクリックします。 ここで、プロジェクトの説明を見つけたり、フィードバックを残したり、友人の情報を見つけたりすることができます（図1）。





図1 .NET Forge CMS Webアプリギャラリーページ



ページの下部にある[ダウンロード]ボタンを使用して、.NET Forge CMSテンプレートを起動します。 プロジェクトテンプレートは、Web Platform Installerで自動的に起動し、インストールの準備が整います（図2）。





図 2.インストール可能なプロジェクトテンプレート



[インストール]ボタンをクリックし、インストールされたコンポーネントのリストとライセンステキストが表示されたウィンドウで、[同意する]ボタンをクリックします。 コンテンツ管理システムのコンポーネントの自動ダウンロードとインストールが開始されます。 しばらくすると、コンピューターにサイトを配置するためのパラメーターを指定するよう求められます（図3）。





図3 サイト設定のウィンドウ



これらのパラメーターを変更せずに「続行」をクリックできます。 数秒後、データベースを操作するためのコンテンツ管理システムの初期設定を入力するための提案を含むウィンドウが表示されます（図4）。





図 4.データベースパラメータの入力



データを入力したら、「続行」ボタンをクリックします。 数秒後、コンテンツ管理システムが正常にインストールされたという通知を受け取ります（図5）。





図 5. .NET Forge CMSの正常なインストールの確認



すべての作業には数分かかりました。



打ち上げ



すでにインストールされている.NET Forge CMSコンテンツ管理システムパッケージに移動するには、提供されているリンクをクリックします。 これに続いて、作業用のサイトの主要な構成が開きます（図6）。





図6 .NET Forge CMSサイトの初期構成



Web Platform Installerを使用したインストールのおかげで、初期セットアップには多くの労力は必要ありません。ライセンス契約に同意し、必要に応じてサイトに登録する必要があります。 その後、サイト管理者のパラメーターを入力し、メールサーバーのオプションデータを入力するよう求められます（図7）。





図7 管理者とメールサーバーの設定の入力



[次へ]をクリックすると、サイトのテンプレートを選択するページに移動します（図8）。





図8 サイトテンプレートとそのテーマの選択



自分のサイトを選択します。個人用サイト、サービスセクターのプロファイルを持つ企業サイト、プロダクションのプロファイルを持つ企業サイト、またはコミュニティサイト。 次に、ソリューションのデザインテーマを設定し、会社のロゴなどのサイトパラメーターを入力します。 数秒で、企業のWebサイトが使用できる状態になり、コンテンツがいっぱいになります（図9）。





図9 企業ウェブサイトの準備完了



おわりに



会社「1C-Bitrix」は、機能的なサイトを作成するための優れた新しい強力な製品を開発者に提示しました。 .NET Forge CMSには、多くの利点と豊富な機能があります。 このコンテンツ管理システムの無料コミュニティバージョンは、あらゆるレベルの開発者がアクセスできるようにします。 このバージョンは無料のままで、機能が失われることはなく、既製のテンプレートと独自のコードの両方に基づいてサイトを作成できます。



Web Platform Installerのインストールによる.NET Forge CMSの可用性により、このCMSでの作業がさらに簡単かつ高速になります。 既成のウェブサイトで作業を始めるのにほんの数分を費やすだけで十分です！
