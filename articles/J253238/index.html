LinuxでのAndroidアプリケーションのインストールと実行





ご存じのとおり、 Chrome App Runtimeライブラリのおかげで、多くのAndroidアプリケーションをChrome OSでネイティブに起動できます。



chromeos-apk 拡張機能とARChon拡張機能を使用して、Android APKの起動が他のオペレーティングシステムで可能になりました。



この投稿では、Ubuntu 14.04に任意のAndroidアプリケーションをインストールする方法を紹介します。



ソースツール：

  • Ubuntu 14.04 LTS
  • Chrome 39.0（別のバージョン37以降も適しています）。








ARChonランタイムをインストールする

1）ARChonをダウンロードして任意のディレクトリに解凍します。

2）Chromeを開き、Chromeで開発者モードを有効にします：//拡張機能/ ;

3）「アンパックされた拡張機能のロード」をクリックして、アンパックされたARChonをインストールします。



次に、インストールするAPKアーカイブを準備する必要があります。



chromeos-apkをインストールする

1）Ubuntuの場合、まず、lib32stdc ++ 6ライブラリをインストールする必要があります。 



sudo apt-get install lib32stdc++6


2）chromeos-apkは、npmパッケージマネージャー（ node.jsで提供 ）を使用してインストールできます。

3）chromeos-apkを直接インストールできるようになりました： 



 npm install chromeos-apk -g


これでAPKを展開する準備が整いました。



chromeos-apkを使用したAPKの準備

1）APKアーカイブを解凍するには、次の手順を実行します。 



 chromeos-apk path/to/file.apk


2）これでChrome-APK拡張機能が得られました。必要なのは（ chrome：//拡張機能/ ）ダウンロードして実行することだけです。
