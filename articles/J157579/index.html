OCEJPAD 6の準備

Java EE 6 Java Persistence API開発者認定エキスパート試験（OCEJPAD 6）に合格した経験を共有したいと思います 。



OCEJPAD 6について



OCEJPAD 6は、Java Persistence APIを対象とする新しい（比較的新しい）証明書の1つです。 以前は、Java Persistence APIはSCBCDの一部でしたが、JPA 2.0にCriteria APIが追加されたため、1つではなく2つの試験が作成されました。

  • Java EE 6 Java Persistence API Developer認定エキスパート試験1Z0-898（EE6 JPAまたはOCEJPAD 6）
  • Java EE 6 Enterprise JavaBeans Developer認定エキスパート試験1Z0-895（EE6 EJBまたはOCEEJB 6）


この証明書は誰のためのものですか？ 私の意見では、すべてのJava開発者にとって、まれなプロジェクトはデータベースへのアクセスを使用せずに行い、JDBCのみを使用することはもはやファンではありません。 したがって、JPAの強力な知識は大歓迎です。 最も一般的に使用されるJPA実装の1つはHibernateです。



模擬テストはどこで入手できますか？



この認証の既存の模擬テスト（少なくとも私に知られている）はepractizelabsだけですが、支払われており、多くのエラーが含まれていると言われています。 私はそれらを使用していないため、このリソースについては何も言えません。

しかし、本を読むだけでは試験の準備が難しくなるため、Pro JPA 2：Mastering the Java Persistence APIのすべての重要なポイントを含む質問/回答を作成しました。 それらに答えて、私は認証のためにかなり準備ができていました-私は87％で合格しました。 これらの質問/回答はここから入手できます-http://www.examclouds.com/は無料で使用できます。



何を読む？



Pro JPA 2：Mastering the Java™Persistence APIブック（インターネットで無料で入手可能）と、トレーニング用のJPA 2.0仕様を使用しました。 私の意見では、仕様は、Pro JPA 2：Java™Persistence APIの習得を徹底的に研究した後に読むか、最も曖昧な点の明確化を求める必要があります。 この本の次の章には、試験に合格するために必要なすべての情報が含まれています（残りは安全にスキップできます）。





第3章エンタープライズアプリケーション
この章では、EJB 3.1-セッションBean、メッセージ駆動型Bean、トランザクション、依存関係管理に焦点を当てています。 試験では、メッセージ駆動型Beanに関する質問はありませんでした。



第4章オブジェクトリレーショナルマッピング
JPAの基礎の基礎。 受験するつもりがない場合でも、これを知る必要があります。



第5章コレクションマッピング
関係と要素のコレクションに関する章。 最も難しい部分は、異なる値（基本タイプ、エンティティー属性、埋め込み可能タイプなど）によるマッピングです。 しかし、準備してください-このトピックに関する多くの質問があります。



第6章 エンティティマネージャー
エンティティマネージャのタイプとエンティティとの相互作用。 永続コンテキストおよびトランザクションと同様に。



第7章 クエリを使用する
JPQLの知識、エンティティとの相互作用。 クエリおよびTypedQueryインターフェイス、パラメーターバインディング。



第8章クエリ言語
JPQLの詳細な説明。



第9章基準API
クエリを作成する別の方法が検討されます。 通常のSQLに慣れているすべての人にとって、非常に珍しい方法です。 試験には、Criteria APIの知識に関する多くの質問が含まれています。



第10章高度なオブジェクトリレーショナルマッピング
埋め込みオブジェクト、複合主キー、派生識別子のより高度な使用。 継承だけでなく。



第11章高度なトピック
この章には、SQL Queris、ライフサイクルコールバック、検証、ロック、同時実行性、キャッシング、およびPersistenceUtilおよびPersistenceUnitUtilの説明が含まれています。 検証は認証に含まれていません。また、PersistenceUtilとPersistenceUnitUtilについての質問もありませんでした。



第12章XMLマッピングファイル
Java SE 5のリリース前に、より便利なアノテーションの代わりに使用されたXMLマッピングファイルの説明。正直なところ、この章はスキップしました。 Oracleサイトの試験の説明では、トピック-XMLマッピングファイルは指定されていません。 しかし、このトピックに関して質問がありました。 だから私は皆にそれに注意を払うことを勧めます。



SCJP（OCJP）について一言



投稿はJava EE 6 Java Persistence認定に関するものなので、繰り返してSCJPについて話したくありませんでした。 しかし、私は彼女の最愛の人について多くの質問を受け取るので、私はまだ気が散って、いくつかの言葉を言います。



何を読む？



SCJP Sun認定プログラマーfor Java 6試験310-065。 この本は、特にSCJP認定の準備のために作成されました（インターネットから無料でダウンロードできます）。 各章には模擬試験の例が含まれており、試験に合格する場合と非常によく似ています。 だから、他のものを探すことはできません-準備には十分です。



模擬テストはどこで入手できますか？



原則として、本からのテストのモックで十分かもしれませんが、それはすべてあなたのトレーニングに依存します。 したがって、魂がもっと必要とするなら、coderanch.comを見てください-無料のテストの大きなリストがあります。 それらからのすべての質問がSCJP試験の精神に沿っているわけではありませんが、それらはSCJPの準備に非常に適しています（私はすべてに合格しました）。 有料テストにお金を使う理由はないと思います。
