米国のデスクトップ検索でのGoogleの市場シェアは過去3年間で最低を記録





SurveyMonkeyは 、検索エンジンの市場調査を実施しました 。 アナリストは、米国のユーザーの調査から取得したデータを調査しました。 回答者は、使用する主要な検索エンジンを示すように求められました。



SurveyMonkey Audienceを使用して、米国の回答者379人にインタビューしました。 同社のオンラインマーケティングマネージャーであるEli Schwartz氏によると、この金額はサンプルの代表者を検討するのに十分な量です。



ソース： SurveyMonkey



デスクトッププラットフォームでは、男性の77％と女性の71％がGoogleを使用しています 。 モバイルデバイスの場合-それぞれ76％および78％。 Googleはメインの検索エンジンとして、45歳未満のユーザーの82％、45歳以上の回答者の66％のみを選択しています。



この調査の結果は、Google検索エンジンの位置が揺らいだことを示しています。



デスクトップ検索



今日のデスクトップシステムにおける主要な検索エンジンのシェアの分布：



1. Google-75％;

2. Yahoo -8％;

3. Bing -7％;

4. DuckDuckGo -2％;

5. Aol -1％;

6. 質問 -1％。

7. バイドゥ -1％。



2014年10月、3人のリーダーは次のようになりました。



1. Google-80％。

2. Yahoo-8％;

3. Bing-6％。



同時に、2014年10月の分析会社comScoreによると、米国のデスクトップ検索市場におけるGoogleのシェアは 67％でした。 2015年10月-63.9％。 ビング-20.8％; ヤフー-12.5％。



2013年、comScoreは、Googleのシェアは66.9％であると報告しました。 2012年-66.9％; 2011年-65.6％。



したがって、今年、米国のデスクトップ検索市場での同社のシェアは最小値に達しました。



SurveyMonkeyの結果はcomScoreとは異なります。 ただし、Eli Schwartzは、Googleがオーガニックコンバージョンの80〜90％を占めるGoogleアナリティクスアカウントのデータと一致していると指摘しています。



モバイル検索



2015年のSurveyMonkeyの調査では、モバイル検索でのユーザー設定の問題も提起されています。



1. Google-78％;

2. Yahoo-5％;

3. Bing-3％;

4. DuckDuckGo-1％;

5.質問-1％。

6. Aol-1％。



調査の参加者には、モバイルデバイスで音声アシスタント（Google Now、Siri、Cortana）の機能を使用する頻度も尋ねられました。



•47％-使用しない。

•11％-毎日使用します。



GoogleとAppleは音声アシスタントを主要な機能として位置づけていますが、個人ユーザーのみが音声アシスタントを使用しています。



リーダーシップを強化する試み



10月、Google は Yahooと契約を結びました。 これで、Yahooは検索と広告を使用します。 Yahooには、検索向けAdSenseの広告ツールであるWebページと画像を検索するツールがあります。 その結果、訪問者はサイト上で必要なデータを検索でき、リソース所有者は結果ページに関連する広告を表示することで収入を得ることができます。



ヤフーは別の契約に縛られている-今回はマイクロソフトと 。 YahooのすべてのトラフィックをBing検索エンジンに転送します。 ただし、Yahoo Marissa Mayer（Googleの元従業員）の新しいヘッドは、トラフィックの49％に対する権利をMicrosoftから「ノックアウト」しました。 これまでのところ、Bingの51％がいます。



これらの49％はGoogleに転送されますが、これはこの状況で予想されていました。
