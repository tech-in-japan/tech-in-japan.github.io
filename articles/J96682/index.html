リアルタイムの日光-デスクトップ上のリアルタイムの惑星

昔々、MS Windowsに似たものがありましたが、それはずっと前のことで、Windows以外ではあまり美しくありませんでした。 現在、Ubuntuには非常に満足していることがあります。 このプログラムは、デスクトップ上に世界の地図（宇宙からの画像に非常に似ています）と太陽の光を表示します。 本質的に、これは非常に便利なGUIを備えた風変わりな壁紙です。





拡張マップには4つのタイプがあります。 私は最初のものが一番好きでした：







惑星を半球の形で見ることも可能です。







さらに、月の満ち欠けを観察します（画像は正方形です。デスクトップの背景を黒にする必要があります）







最後のユーティリティは、自動起動する機能です。

ここから画像が取得され、一定の頻度で更新されます。

スクリーンショットを撮っている間に、最も楽しいニュースが聞かれました-デスクトップ用のこの装飾のUbuntuのリポジトリが登場しました。 これで、インストールのためにソースからコンパイルする必要がなくなりました。 インストールするには：



sudo add-apt-repository ppa:realtime.sunlight.wallpaper/rsw

sudo apt-get update && sudo apt-get install sunlight







ご清聴ありがとうございました。



UPD。：システムの速度にはほとんど影響しません。 新しいイメージをロードするときの「tyrnet」の速度にわずかに影響する場合があります。
