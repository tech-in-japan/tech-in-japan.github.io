VKontakte、Facebook、Twitter、Youtubeでブランドの返品について話し合った方法について

ピックアップするかしないか？ 出て行けない 拾えません 多くの同僚は、登録商標と同一のユーザー名を返すという問題に直面していました。 角をとってください-必ずしもこれを行う必要はありません。すべてのお客様にユーザー名を返すことをお勧めしません。 さらに、それを行う方法の明確な悪い例はありません-VKontakteグループに関するノキアの行動。 会社はその住所で多くの悪いカルマを獲得しました。



まったく異なるケースがあります。 興味のある名前のユーザーは、非アクティブであるか、不人気であるか、ルール違反のアカウントに至りました。 戻ることにした。 それでは行きましょう...



与えられた 。 WIPO-ROMARINシステムによると、英国、ドイツ、ウクライナの3か国で登録された商標。



必須です。 VKontakte、Facebook、Twitter、Youtubeの4つのソーシャルネットワークでユーザー名を返します。



とても速いtwitter 。 最も簡単で迅速な方法は、名前をTwitterに返すことでした。 Twitterには、そのような呼び出しのためのかなりシンプルで直感的なフォームがあります： http : //support.twitter.com/forms/











商標の所有者から承認されたため、2番目のアイテムを選択しました。 しかし実際には、フォームの最初の2つのポイントに違いはありません。 しかし、誰かに「伝えたい」ときの3番目の点は受け入れられません。 「ウェルウィッシャーズ」Twitterからのそのようなリクエストは考慮したくさえありません。



フォームはシンプルで簡単です。 最も重要なことは、商標登録番号とそれが登録されたシステムを知ることです。 フォームへの記入が完了すると、指定された電子メールアドレスに、仕事が許可されたという通知レターがすぐに届きます。



1日後、リクエストが承認され、選択肢が提示されたという手紙が届きました。ユーザー名を既存のアカウントに転送するか、新しいアカウントを作成できます。



転送はすぐには行われません。 最初は、少し怖かったです。マイクロブログは古いアドレスではもう利用できませんでしたが、新しいアドレスではまだ利用できませんでした。 アカウントの「輸血」は約6時間続いた。 どうやらそれはすべて、マイクロブログが存在するサーバーと、マイクロブログの量に依存しています。

締め切りは2日です。



完全なFacebookではありません。 Facebookには、ユーザー名と商標のための特別な形式はありません。 標準の著作権侵害フォームを使用しました。 http://www.facebook.com/legal/copyright.php?copyright_notice=1







すべてがTwitterのように単純で明確ではありませんが、フォームはロシア語で記入できます。



サポートサービスからの応答もロシア語で送信されます。 ユーザー名は返されましたが、管理者は古いページ名を非常にベールに包まれた新しい名前に変更することを拒否しました。 政権は単にこれに関する質問に答えず、ページ登録の「ヘルプ」の一部を含む最後の手紙を回答として送った。



締め切りは2日です。



奇妙なVkontakte 。 最も奇妙なコラボレーションは、VKontakteとのコラボレーションでした。 検索名は使用されませんでしたが、登録には使用できません。 さらに、私たちとクライアントの両方がそれを登録しようとすると、ブラウザがこのページを登録できなかったというエラーが表示されました。



VKontakteのドキュメントでは、商標とその保護については何もありません。



カンファレンスで、ソーシャルネットワークのプレスサービスの責任者であるVlad Tsyplukhinは、パートナー[woof] corp.vk.comのアドレスを発表しました。ここでは、商標の著作権保護について書くことができます。 このアドレスは、VKontakte Webサイトのテクニカルサポートページには示されていませんが、少なくともそこからは助けが得られます。

グループの名前は、2週間の連絡後に割り当てられました。



非常に奇妙で非常に高速なYouTube 。 ヘルプで明確に表明された位置は矛盾を意味しないため、最初はYouTubeで成功しないと考えました-http ://www.google.com/support/youtube/bin/answer.py? hlrm=en-GB&answer = 151655 「ユーザー名に他人の商標を使用すること自体は違法ではないかもしれません。」



ユーザー名が法律に違反していない場合、YouTubeはユーザーに個人メッセージフォームからの連絡を通じてこの問題を自分で解決するよう求めます。「YouTubeはユーザーと商標権所有者間の商標紛争を解決する仲介者として行動できません。 したがって、商標権者はユーザー名の所有者と直接紛争を解決することを強くお勧めします。 商標権所有者は、YouTubeのプライベートメッセージ機能を介してユーザーに連絡できます。



私たちが興味を持っているアカウントのユーザーは、彼が中国にいて、プロフィールページに1年以上居なかったというプロフィールから判断して、この手紙に返信しませんでした。 そのような場合、YouTubeはサイトの管理者に直接連絡することで問題を解決することを提案します。 これを行うには、法的問題フォームに記入します。 ただし、フォームへのリンクはヘルプで利用できます... ...フォームではなく、YouTubeコミュニケーションサービスhttp://www.google.com/support/youtube/bin/request.py?contact_type=legal2およびQ＆Aフォーラムにアクセスします。



フォーラム自体にアクセスする前に、YouTubeは私たちにとって便利な言語を選択することをお勧めします。 「ロシア語」を選択し、英語フォーラムにアクセスします。 さらに-より興味深い。 「質問を作成する」を選択して、最終的に誰かに状況を尋ねます。 そして、対応するカテゴリがないという事実に直面しています。







その結果、「Accounts」カテゴリを選択して、質問をします。 1日後、YouTubeのボランティアの1人から、私たちが始めたヘルプページへのリンクが返されます。



ループに入った後、完全に絡み合っていない場合、私たちは旅を続けることにします。 商標に関する問い合わせのホームページに戻りますが、すでに英語を選択しています。



このテキストは、すでに法的通知フォームだけでなく、特に商標権侵害の申し立てフォームも参照しています。 対応するリンクをクリックすると、商標のリクエストフォームが実際に表示されます ： http : //www.google.com/support/youtube/bin/request.py?contact_type=legal2&hl= en







最初のオプション-「商標はユーザー名で使用されます」を選択します。 そして、私たちは他のアメリカのソーシャルネットワークに似た形で自分自身を見つけます：







リクエストへの回答は20（！）分で届きました。 YouTubeの管理者は、指定されたユーザーが活動していないため、リクエストが満たされたと報告しました。 彼らは、1時間以内にサービス電子ボックスから連絡し、アカウントへのアクセスを提供することを約束しました。



サービスメールボックスからの手紙は、最初の手紙から15分（！）分後に届きました。 手紙は、目的のアカウントでサイトに入るには、フォームの前半で提供された電子メールアドレスと一時パスワードを使用する必要があることを示していました。 興味深いことに、YouTubeはチャンネルをリセットしただけでなく、古いユーザーの情報を削除し、アカウントに関連付けられたメールアカウントを削除しましたが、ユーザーの統計情報は削除しませんでした。 つまり、動画なしで視聴回数のあるチャンネルを取得し、2009年に登録しました。



結論 ユーザー名を登録商標に戻すことは、一見思われるほど怖くはありません。 繰り返しになりますが、この手順は必ずしも必要ではなく、さらに常に有用ではないという事実に注意を向けます。 そのような操作の1つは、あなたが無頓着にそれを行う場合、関係する聴衆に多くの問題をもたらすことがあります。
