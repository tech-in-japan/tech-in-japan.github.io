Amiga X1000マザーボードの発表

2010年初頭、A-eonは新しいAmigaOS互換コンピューターのリリースを発表しました。







このリリースは2010年の夏に予定されており、世界初のマルチメディアコンピューターであるAmiga 1000の25周年に捧げられています。

数週間前、A-eonウェブサイトが説明と写真とともに開かれました。





ちょっとした歴史


1994-2000


Amiga互換コンピューターの開発は常に挑戦的です。

Amigaブランドを所有し、いわゆる「クラシックAmiga」（Motorolaプロセッサ）をリリースした1994年のコモドールの破産後、Amigaブランドは何度も手を変えましたが、各所有者は利益を最大化しようとしましたが、プラットフォームの開発とサポートに投資しました。



したがって、1994年から2002年まで、時折発表されたアヒルであるにもかかわらず、Amigaの開発はほとんどありませんでした。 この期間にサードパーティ企業によって開発されたアクセラレータは、MotorolaプロセッサからPowerPCへのコンピュータおよびソフトウェアの転送の開始を含む、クラシックモデルの寿命を延ばすことを許可しました。 しかし、あまりにも多くの時間が失われ、PCやMacとの競合を言うのはばかげているでしょう。



2000-2010


私たちの世紀の初めに、PowerPCプロセッサのみに基づいたAmigaの新しいモデルがついに登場し始めました。 過去10年間で、このようなプラットフォームは4つしかリリースされておらず、その一部のみがAmigaOS 4と互換性があり、残りはMorphOS （AmigaOSのフォーク）と互換性があります。

これらのプラットフォームは、熱狂的な企業によって実際に開発されたもので、AmigaおよびAmigaOSブランドをめぐる訴訟に同時にお金を費やす必要がありました。 ボードは非常に小さなエディションで発行され（私は各バージョンで数千枚のボードに過ぎないようです）、主にAmigaコミュニティ愛好家の間で分岐しました。



プラットフォームがリリースされました：

* AmigaONE （2002）

* ペガソス （2003）

* エフィカ （2005）

* SAM 440ep （2007）

同時に、プラットフォームの技術的特徴は非常に平均的であり、最大のパフォーマンスはPowerPC G4プロセッサを搭載したPegasos IIプラットフォームでのみ提供されました。



それは注目に値する 過去10年間、Amiga互換コンピューターのパフォーマンスは低く、プログラマーはプログラムを非常に最適化することを余儀なくされてきました。 これは、最新のソフトウェアのほとんどが、68020 / 14MHzのプロセッサと2メガバイトのコンピューターメモリを備えたAmigaでも実行できるという事実につながります。 AmigaOS4自体は最新ですが、かなり軽い軸であり、メモリで約30メガバイトを占有し、200 MHzのマシン上でも作業容量が快適です（AmigaOSおよびLinuxを起動できるプラットフォームでは、AmigaOSは最も一般的なタスクではるかに高速に動作します） ）

それにもかかわらず、HDビデオの再生など、弱いプロセッサでは物理的に解決が不可能な客観的なタスクがあります。そのため、常に高性能なプラットフォームが求められています。



新しい








新しいボードの仕様の発表：

* ATXフォームファクター

*デュアルコアPowerISA™v2.04 + CPU

*「Xena」XMOS XS1-L1 128 SDS

* 4x DDR2 RAMスロット

* 2 x PCIe x16スロット

* 2 x PCIe x1スロット

* 2x PCIスロット

* 1x Xorroスロット

* 10x USB 2.0

* 1xギガビットイーサネット

* 7.1チャンネルHDオーディオ

* 2 x RS232

* 4 x SATA 2コネクタ

* 1x IDEコネクタ

* JTAGコネクタ

* 1xコンパクトフラッシュ



特性を考慮してください。 Amiga互換マシンは、初めてマルチコアプロセッサと特定のコプロセッサを使用します。 また、周辺機器用のPCI / AGPバスからPCI-Eへの移行が行われ、DDR2メモリが使用されました。 キットの残りの部分（usb、ネットワーク、サウンド、sata）は、以前のプラットフォームですでに見たものであり、大きな違いはありません。



プロセッサーについて：

PowerISA v2.04仕様プロセッサは、たとえばハードウェア仮想化やハイパーバイザーをサポートする最新のマルチコアPowerPCプロセッサです。 特定のプロセッサのバージョンと周波数はまだ公開されていません（開発者がAmigaOSの移植とデバッグ用に1.6 GHzプロセッサを搭載したボードを提供したという情報があります）。



次のステップはXenaコプロセッサーです。 1985年、amigaは、特殊なチップセットを使用して、中央処理装置をオーディオ処理、グラフィックス、およびI / Oからオフロードした最初のものの1つでした。 今では誰も驚くことではありません。現代のPCでは、ビデオカードが計算能力において中央処理装置よりも優れていることがよくあります。 X1000はさらに進化しました。 コプロセッサーとして、プログラム可能なプロセッサー（マイクロコントローラー） XMOS XCoreが使用され 、500 MIPSの容量で8つのスレッドを処理できます。 コプロセッサは、周辺機器を個別に制御したり、DSPを操作したり、他のチップをエミュレートしたりできます（たとえば、Spectrumを愛する人は、ハードウェアまたはプラットフォーム全体でAYチップをエミュレートできます）。

面白いですね。 コプロセッサをプログラミングし、作業をそれに移すことで、プロセッサを大幅にアンロードできます。

Xenaのパフォーマンスはそれほど大きくないと思いますか？ さらに拡張するために、ボードには64個の入出力ストリームがある特別なXorroバス（pci-eスロットの形式で作られています）があり、このバスにより、最大102,400 MIPSの容量を持つ最大256個のXenaコアを含むコプロセッサーとして外部ユニットを接続できます（Intel Core i7に匹敵するパフォーマンス）。 トランスピュータとしてこのようなボードを使用することは可能です。 XCoreチップを含む開発者向けのテスト16チップ（64コア）マザーボードの例。



既製の車には、ATI Radeon R700、2GB RAM、500GBハードドライブ、22x DVDコンボが追加装備されます。 マザーボードとコンピューターの価格はまだ決定されていません。



合計




90年代初頭以来初めて、Amigaは競争力のあるプラットフォームを手に入れ、当初はさらなる開発の大きな可能性をもたらしました（従来のAmigaのすべてのモデルと同様）。



もちろん、このプラットフォームは、私たちの生活のあらゆる分野に浸透しているという点でPC / Macと競合することはありません。 しかし、おそらく最新のハードウェアはAmizhniksの視聴者を拡大し、近い将来、市場の目に見える割合を獲得するかもしれません。 最終的には、代替手段があることは常にプラスになります。



X1000開発者のサイトはhttp://www.a-eon.com/ですが、発表以外には何もありません（FireFoxのAdBlockはサイトの下部メニューをブロックできます）。



PSまた、AmigaOSバージョン4以降を実行するのに適したPowerPCプロセッサを搭載したコンピューターエミュレーターはありませんが、これが他のプラットフォームで「見た目を見る」ことができない理由です。 エミュレートされる最大値は、WinUAE / E-UAEのMotorolaプロセッサ上の「クラシック」Amigaです。 1994年から95年のAmigaサンプル。
