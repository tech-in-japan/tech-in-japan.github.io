ヒント：ネームゾーンのwhoisレジストリからほぼ完全なデータを取得する

ドメイン名のWhoisはそのプライバシーで知られています。有効期限（委任の終了日）を確認するために、このビジネスのパスワードを絶対に公式に購入するように求められました。

このデータをもう少し取得する方法

通常のリクエスト中に、レジストリ内の存在に関するスタブが与えられました

# whois -h whois.name qwerty.name

...

Domain Name ID: 4461890DOMAIN-NAME

Domain Name: QWERTY.NAME

Domain Status: ok







しかし、このドメインに追加する価値があります= <*>。より多くのデータがあるため、名前

# whois -h whois.name domain=qwerty.name

...

Domain Name ID: 4461890DOMAIN-NAME

Domain Name: QWERTY.NAME

Sponsoring Registrar ID: 230REGISTRAR-NAME

Sponsoring Registrar: Regtime Ltd.

Domain Status: ok

Registrant ID: 3281648CONTACT-NAME

Admin ID: 3281645CONTACT-NAME

Tech ID: 3281647CONTACT-NAME

Billing ID: 3281646CONTACT-NAME

Name Server ID: 1605444HOST-NAME

Name Server: NS1.SERVERSTAT.RU

Name Server ID: 1605445HOST-NAME

Name Server: NS2.SERVERSTAT.RU

Created On: 2009-03-25T15:43:51Z

Expires On: 2010-03-25T15:43:51Z

Updated On: 2009-07-18T16:41:11Z







有効期限と同様に、作成時間は秒単位で正確であることに注意してください。
