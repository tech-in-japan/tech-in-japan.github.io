Intel Developer Forum 2012について最も重要なこと



秋が来ました、 渡り鳥 、ソフトウェア、コンピューター、デジタルデバイスの開発者は南に日当たりの良いサンフランシスコまで広がりました2012.フォーラムで行われた最も興味深い発表と発表の一部を紹介します。



フォーラムの中心的なトピックの1つは、Haswellの22nmマイクロアーキテクチャに基づく第4世代のIntel Coreプロセッサで、2013年に市場に登場する予定です。 これらのプロセッサの主な利点は、パフォーマンスが大幅に向上し、同様に消費電力が削減されることです。 Haswellプロセッサは、Intel HDグラフィックシステム、高速化と暗号化のための新しい命令セット、新しいハードウェア保護機能、新しい低電力モードをサポートします。 一般に、Sandy Bridgeと比較したプラットフォーム全体のエネルギー効率は20倍になります。 インテルのプロダクトディレクターであるDadi Perlmutterが演説の中で具体的な例を挙げました。第3世代および第4世代のIntel Coreプロセッサーで立ち上げられたベンチマークです。 グラフは、同じ性能の両方のチップの消費電力を示しています-Ivy Bridgeで17W、Haswellで7.5W（下図）。







Intel Coreに加えて、 Intel Atomプロセッサライン （コードネームClover Trail）も更新され、近い将来に導入されます。 これは、Windows 8用に特別に作成された新しいシステムオンチップ（SoC）です。CloverTrailプロセッサは、32ナノメートルの製造技術に基づいており、変形可能なデザインの小型タブレットやモバイルPC用に設計され、長い作業時間とデバイスへの即時アクセスをサポートします。



更新は、 Intel Xeonサーバーシリーズを待っています。 2013年には、コードネームIvy Bridge-EPと呼ばれるE5ファミリのXeonプロセッサの生産が、世界初の3次元トライゲートトランジスタを含む22 nm製造プロセスに基づいて開始されます。 同時に、E7ファミリの新世代のIntel Xeonプロセッサが登場します。コード名はIvy Bridge-EXで、これも22 nm製造プロセスに基づいて作成されます。 これらのプロセッサフ​​ァミリの利点には、パフォーマンスの向上、新しい信頼性機能、およびハードウェア保護機能が含まれます。 新しいXeon Phiコプロセッサも導入されました。これはDadiの左手（右手）です。







「デジタル環境」のソリューションの分野における最新の傾向の1つは、人間にとって伝統的な「受容的」な、コンピューターデバイスとのユーザーインタラクションインターフェイスの作成です。 同じスピーチの中で、Dadi Perlmutterは開発者コミュニティにIntelとより積極的に協力して、 Intel Perceptual Computing Software Development Kitのベータ版を使用してこの種の相互作用インターフェイスを作成するように呼びかけました。 新しいキットは次の四半期の初めにリリースされ、ハードウェアおよびソフトウェアソリューションの開発者は、Intelプロセッサに基づく既存および将来のUltrabookデバイスとPCにジェスチャーと音声認識を追加できます。







Intelの取り組みを含む私たちの日常生活では、相互にデータをやり取りできるインテリジェントデバイスの役割が拡大しています。 この「物事のグローバルコンピュータネットワーク」には注文が必要です。そのため、Intelは、 Intel Intelligent Systems Frameworkを作成しました。 Intel Intelligent Systems Frameworkには、Intelとそのエコシステムパートナーが開発したいくつかのコアコンポーネントが含まれています。 システムは、Intel Xeonプロセッサー、Intel vPro TMテクノロジーを搭載した第2世代および第3世代Intel Coreプロセッサー、およびIntel Atomプロセッサーを使用します。



開発者にとっての朗報-フォーラムは、開発者と企業が同僚とのコミュニケーションに必要なすべてのツール、コミュニティ、リソースにアクセスするための単一のリソースになるプログラムであるIntel Developer Zoneの立ち上げを発表しました。 その主なコンポーネント：

  • リソース開発。 ソフトウェアツール、トレーニングプログラム、開発者ガイド、コードサンプル、およびプロフェッショナルサポート。
  • ビジネスのためのリソース。 新しいソフトウェアの配布と販売の機会は、Intel AppUp Centerから入手できます。
  • アクティブなコミュニティ。 開発者は、Intel Developer Zoneを使用して、Intelの専門家を含む業界の専門家とチャットできます。




ファッショナブルな「クラウド」テーマは影響を受けませんでした。 インテルは、クラウドサービスプロバイダーの検索と選択を促進するために設計された新しいIntel Cloud Finderの発売を発表しました。 多くの企業がクラウドコンピューティングテクノロジーを実装する準備ができているという事実にもかかわらず、最適なサービスプロバイダーを選択することは非常に難しい作業です。 Intel Cloud Finderは、検索を高速化し、パブリッククラウド環境を実装するためのベストプラクティスと戦略に関する情報を共有するのに役立つオンラインツールとリソースにより、このプロセスを簡素化します。



Intel Developer Forum 2012は既に完了しており、専門家はオフィスと本社に散らばっています。 しかし、私たちにとって重要なことは、フォーラム後、2013年4月に北京で開催される新しいIDFの直前に、次の6か月に十分なほどの新しいトピックがあったことです。
