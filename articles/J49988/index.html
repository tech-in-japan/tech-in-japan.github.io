ラジオVSプレーヤー



次に、プレーヤーとラジオで音楽を聴くことを比較したいと思います。 プレーヤーとは、ポータブルデバイスと、PC、ラップトップ、PDAなどのプレーヤーの両方を意味します。 ラジオ-FMとオンラインラジオ局の両方。 プレーヤーにはいくつかの利点があります。広告がない、かなりの量の曲があり、好きなものだけを再生します。 なぜ人々はラジオを聴き続けるのですか？ お気に入りの曲がラジオで聞こえるのに、どうしてそんなに幸せなのですか？ 秘密は、あなたの行動の結果ではない快楽は運と見なされることです。 そして、運がいいです。 ラジオには教育的で楽しいプログラムもありますが、これはポイントではなくボーナスです。



FM VSオンライン。



よくここでそれは明らかです。 FMケアは時間と技術の問題です。 その結果、インターネットラジオが耳を支配します。 そして、おそらくLast.fmのような社会的に適応性のあるサービスが未来にあるでしょう。 幅広いチャンネルを持つ多くのユーザーが彼を好みます。これは魅力的なラジオであり、あなたの好みに合った音楽だからです。



Last.fmはオフラインです。



私がこのサービスを知っているのは最近でしたが、6年前はかっこいいと思いました。 それが機能する原理を知って、私はすぐに私のライブラリをいっぱいにしました、そして今、運の確率は100％になりがちです:)しかし、本当に私を悩ませた1つの鉱山があります-インターネットへの執着。 道路、出張、スノーボード旅行はプレイヤーと行われ、彼は大幅に失いました:( StationRipper ソフトウェアの問題を解決しました。

あなたがする必要があること：

1. Last.fmで登録する

2. StationRipperを起動します

3. Last.fmタブの設定で、アカウントデータを入力します

4. [プレミアム]タブで、[曲を保持しない]をオンにして、1を入力します。フィールドの横にキログラムではなくメガバイトで入力してください:)。

5. [OK]をクリックして、ステーションのアドレスを入力し、下の入力フィールドに「追加」します



プログラムはタグをまとめて、ファイルに美しい名前を付け、データベースを維持します。 同じ曲が2回表示されません。 ここで、約50曲が始まると、フォルダーの名前変更を停止します... \ output \ [stationname]から... \ output \ [stationname] .1に変更して、プレーヤーに書き込みます。 私たちは数日間、1週間に耳を傾け、新鮮なリッピングとレコーディングを行います。
