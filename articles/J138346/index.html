OS X Mountain Lionは開発者が利用できます

Appleはまだ驚きの方法を知っています！



OS X Lion（OS X 10.7）では、特にiOSとiPadからいくつかの改善を適用し始め、デスクトップOS Xと統合しました。今日、Appleは次のステップであるOS X Mountain Lionを採用しています。 これは、Appleから統合デバイスシステムを構築するための大きな一歩です。



そうです、同社はOS X 10.8の次のバージョンを表示する準備ができています-最新バージョンのリリースからわずか7か月後です。 iOSの進化の速さを考えると、AppleはOS Xが時代に対応できることを示したいと考えています。



今日、AppleはMac開発者にアクセスを許可し、夏までに新しいOS Xをリリースできると確信しています。



システムの小さな概要がすぐにTechCrunchに登場しました 。



主な変更点：

  • iCloud

    iCloudはOS X Lionよりも統合されています。 インストールの初期段階で、iCloudを構成するようすぐに求められます。 新しいバージョンの重要な機能の1つは、クラウド内のすべてのドキュメントのストレージです。 Finderでは、ドキュメントやフォルダなどをリアルタイムで変更できますが、変更は他のデバイスにほぼ瞬時に反映されます。



  • メッセージ

    iPhoneとiPadのすべての所有者に愛されているiMessageサービスは、システムに直接統合されています。 プロファイルAIM、Googleトーク、Jabber、Yahoo Messengerを接続することもできます。 このアプリケーションでは、iPhoneに書き込み、簡単なドラッグアンドドロップで写真を投げることなどができます。 ベータ版が利用可能になりました （OS X 10.7.3が必要です）







  • リマインダー

    To-Doアプリケーションは、システムに緊密に統合され、iCloudを介してデバイスと同期されます。 私が理解しているように、iCalを置き換える可能性があります。



  • 通知センター

    iOSの特徴の1つは、デスクトップで利用できるようになったプッシュ通知です。

    画面の右側にある通知付きのスライドパネルは、iOS 5の同様のパネルに非常によく似ています。これまでのところ、通知には2つのタイプがあります。 開発者向けのAPIは近日中に利用可能になります。







  • ソーシャルネットワークでの共有



    iOSでは、すべての主要なアプリケーションからソーシャルネットワーク上で何かを簡単に共有できます。 OS Xの新しいバージョンでは、そのような機能はすぐに利用できます（apiを含む）。 たとえば、Safariでは、サイトへのリンクをTwitterまたはメッセージで送信したり、写真を共有したりできます。 一般に、システムレベルでの他のサービスとの緊密な統合。 確かに、Facebookはまだ何らかの理由でまだです。



  • Twitter

    人気のあるサービスとの統合は、iOS 5と同様にシステムに直接送られます。他のアプリケーションとの密接なやり取り、そしてもちろん、組み込みの通知。



  • ゲームセンター

    同社は、ゲーム用のソーシャルサービスをオペレーティングシステムに直接統合することを決定しました。 現在、ネットワークには1億人以上の登録ユーザーがいます。 このサービスはデスクトップゲームでも利用できるようになり、開発者はMacマルチプレイヤー（iPhone、iPad、iPod）を作成するようになりました。



  • エアプレイ

    Apple TV経由で画像を共有できる機能がデスクトップでも利用できるようになりました。 不要なジェスチャーなしで、テレビを2番目のモニターとして使用できます。



  • ゲートキーパー

    OS Xの完全に新しいセキュリティ機能。Gatekeeperを使用すると、インストール済みのアプリケーションを制御できます。任意のアプリケーションのインストール（現在）、Mac App Storeからのみインストール、信頼できる開発者からのアプリケーションのインストール（Mac開発者プログラムに参加）できます。 そして、デフォルトでは最後のオプションが選択されていますが、これはかなり議論の余地がある決定だと思います。



  • 中国語版のサポートの改善

    デフォルトでBaiduとの統合、入力方法の改善など。 中国では無効になっているので、Twitterが機能するのだろうか。



  • そして、他の多くのマイナーな追加...

    Launchpad管理の改善、Safari（最終的に）でのアドレス入力と検索のための単一フィールド、標準メールクライアントの優先機能（Gmailなど）、新しい壁紙:)など。
