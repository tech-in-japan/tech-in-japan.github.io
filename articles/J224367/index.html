ASUS VivoTab Note 8を確認する

今日、新しいBay TrailプラットフォームでASUSのデバイスをご紹介します。 先駆者は変圧器ASUS T100でした。 VivoTab Note 8は同じプラットフォーム上に構築され、最新の4コアIntel Atom Z3740プロセッサーを実行しています。 小型の8インチスクリーン、コンパクトなボディ、完全なWindows 8.1を備えたこのデバイスは、さまざまなタスクの優れたアシスタントです。 競合他社との重要な違いは、Wacomのスタイラスです。これにより、デスクトップモードでの作業が容易になります。

パッケージバンドル



ASUS VivoTab Note 8は、面白いマットの小さなマットボックスに入っています。









パッケージバンドルは非常に貧弱です：電源とマイクロUSBケーブル。 もちろん、デバイスの予算クラスが影響しますが、OTGケーブルまたはケースを入れることができます。





電流が2アンペアの標準サイズの充電器。





外観



一見したところ、このタブレットはNexus 7を連想させますが、Nexus 7もASUSによって製造されていますが、よく見ると違いが顕著になっています。 デバイスの寸法は221 x 134 mm、厚さは11 mmです。 重量も390 gに過ぎません。実際、このような赤ちゃんが手のひらに収まる本格的なx86コンピューターであることは今でも珍しいことです。





前面はガラスで覆われ、画面には小さなフレームがあり、上下にわずかに幅があります。 カメラと光センサーは上部にあり、下にASUSロゴがあります。





ケースの背面は、ソフトタッチのプラスチックでできています。 タブレットは手に持つのに便利で、滑りにくく、手から落ちようとしません。 ダークグレーのマット仕上げは、指紋によく対応します。 上下に、両側に間隔を空けたステレオスピーカーグリルが目立ちます。 メインの5メガピクセルカメラは、ASUSネームプレートの真上にあります。





ラップトップの遺産は、たくさんの異なるステッカーに行きました。 将来的には、メーカーがより美的なソリューションになることを願っています。





インターフェース



タブレットのインターフェースに移りましょう。 実際、それほど多くはなく、すべてが控えめです。 上部には、ヘッドセットを接続するための3.5ミニジャックジャックがあります。 充電用のマイクロUSBに近い。 また、さまざまな周辺機器を接続できるOTGもサポートしています。





もちろん、キットにOTGケーブルを入れたいのですが、別途購入する必要があります。 フラッシュドライブ、USBハブなど、すべてのUSBデバイスは通常どおり検出されます。 0.5 A以上を必要とするUSB​​ドライブの一部を以下に示しますが、「固執」しないでください。 たとえば、このドライブは0.4 Aを消費し、システムによって正常に検出されました。





左側には、microSDカードリーダーの開始画面を呼び出すボタンがあります。 最大64 GBのサポートされているカード。 Windows 8.1用のリカバリイメージを備えたメモリカードが含まれています。





右側には、ボリュームロッカーと操作の表示がある電源ボタンがあります。





充電中、インジケータはオレンジ色に点灯しますが、一般的には非常に見えにくいです。 一方では、注意をそらすことはありませんが、他方では、タブレットが充電されているかどうかを見るのはあまり便利ではありません。 また、右側にはワコムのスタイラスがあります。









スクリーンとカメラ



ASUS VivoTab Note 8には、800 x 1280ピクセルの解像度の8インチスクリーンがあります。 一方で、これはFullHDではありませんが、このようなPPI（188）と小さな対角線を使用すると、画像は非常に良く見えます。





もちろん、デスクトップモードでは、インターフェイスは小さくなりますが、作業は快適でした。 表またはテキスト文書では、スタイラスはすでに便利です。 IPSマトリックスのおかげで、画像は飽和していますが、画面は太陽の下で明るさが不足しています。





IPSはまた、良好な視野角と色反転の欠如をもたらします。 疎油性コーティングにより、ガラスは指紋で覆われにくく、掃除が簡単です。





アンチグレアフィルターはうまく機能しません。 明るい光源は画面上で十分に反射し、作業の邪魔になりますが、これはすべての予算のタッチスクリーンの問題です。

画面の上には、ビデオコール用の1.2メガピクセルのカメラがあります。 動作中、インジケータは近くで点灯します。 メインカメラはオートフォーカス付きの5メガピクセルです。 写真の品質は平均です。

サンプル写真。



入力機器



容量性スクリーンは、最大5つの同時タッチをサポートします。 感度は良好なレベルであり、すべてのタッチは問題なくキャプチャされます。 OTGアダプターをマイクロUSBコネクターに接続すると、キーボードとマウスを問題なく接続できます。 周辺機器もBluetoothを使用して接続できます。

キットにはワコムのスタイラスが付属しています。 サイズは、PocketPCの時代のようではなく、通常の太さ12 cmの普通のペンのようなものです。 スタイラスには、右クリックをエミュレートする小さなボタンがあります。 ボタンは親指のある場所にあります-便利な解決策です。





また、長押しで、コンテキストメニューを呼び出すことができます。 スタイラス自体は追加の電力を必要とせず、バッテリーなしで動作します。 デジタイザーは、画面から2 cmの距離にあるスタイラスを既に認識していますが、指への応答はオフになっています。 このため、紙の上に置いてタブレットで書くように、静かに画面に手を置くことができます。 Windows 8の組み込み手書き入力は正常に機能します。 ずさんな手書きでも認識されます。





タイル化されたインターフェイスにはスタイラスは不要で、すべてのコントロールが大きく便利です。 Modern UIのソフトウェアでも同じことが言えます。 ただし、デスクトップモードでは、スタイラスは便利です。たとえば、複雑なテーブルを編集したり、多数のファイルを選択したりする必要がある場合に便利です。 一般的に、使い慣れたWindowsインターフェースのアイコンの小さな断片に入るように、スタイラスなしで良い仕事をしました。





性能



Intelの新しいBay Trailプラットフォームのリリースにより、低コストのAtomタブレットは、バッテリー寿命が長い生産性の高いデバイスとして高い評価を得ています。 このシステムには小さなSoCチップに4つの本格的なコアがあり、ハイパースレッディングを拒否しました。 このチップは、Haswellのような22 nmプロセステクノロジーに従って製造されており、エネルギー消費と熱放散を削減します。 2 GBのRAMは、要求の厳しい日常のタスクには十分です。





ASUS VivoTab Note 8は、本格的なWindows 8.1で動作します。 また、事前にライセンスされたオフィススイートMicrosoft Office Home＆Student 2013がプリインストールされており、デスクトップオペレーティングシステムを使用することで、実質的にあらゆる目的のWindowsプログラムを幅広く選択できます。 システムは約15秒間起動します。 スリープモードからの覚醒は即座に発生します。

32GBバージョンでは、システムディスクの容量は28GBで、インストールされたシステムは15GBを占有します。 その結果、ユーザーは12GBしか使用できません。 これは、同様のデバイスのすべてのメーカーにとって問題です。 それでも、32GBのタブレットを購入すると、システムに必要な数ギガバイトが20GBではなく、最大で使用できないという事実に慣れています。





















バッテリーと持続時間



優れたパフォーマンスに加えて、新世代のAtomプロセッサーは優れたアップタイムを示し、ARMベースのタブレットに追いつきました。

タブレットをエアモードに切り替え、最小バックライトを設定し、10.5時間の作業時間を確保します。 非常にまともな結果。

HDビデオを見るとき、平均画面輝度で、タブレットは6時間続きました。

平均輝度、インターネットでの一定のサーフィン、30秒ごとに1つのリンク、合計8時間を設定します。

標準の充電器を使用している場合、バッテリーを完全に充電するには3.5時間かかります。





おわりに



繰り返しになりますが、ASUSは一見既に形成されたタブレット市場で魅力的なニッチを見つけています。 新しいタブレットは、常に外出中のビジネスマンに最適です。 多くの場合、この消費者グループは特定の企業ソフトウェアを使用する必要がありますが、これは通常Windows専用です。 また、途中でドキュメントを緊急に編集する必要がある場合に備えて、本格的なMS Officeを用意しておくと便利です。 スタイラスの存在により、大きなドキュメントやデスクトップモードでの作業が容易になります。 小型で軽量なため、長時間タブレットを片手で疲れることなく保持できます。 デバイスは1日中使用できます。 すべての基準で、ASUS VivoTab Note 8は学生の優れたアシスタントとして機能します。 実際、人気がなくなったネットブックをタブレットに置き換えます。

推奨価格は13,990ルーブルです。
