ジョン・スキート



スキートは.NETコミュニティで高い評価を得ていますが、それは聴衆が彼を講演者として望んでいることを自動的に意味するものではありませんでした。 Stack Overflowへの回答のスターかもしれませんが、彼はロシア語を話す聴衆の注意を1時間の英語を話すパフォーマンスで保持することはできませんか？ 彼女の基調講演の冒頭で、スキートは「私はロズリンと一緒に真夜中を選んでいたので、ステージで眠りにつくと、何かを投げつけてしまった」と心配しました。 そして、会議の後、フィードバックが示されました：視聴者の中には英国のアクセントによって妨げられたものもありましたが、一般に、彼らの評価は非常に高かったです。



スキートは、数字、テキスト、日付/時刻などの基本的なプログラミングの問題について話し、何とかキャッチーなフレーズを発することができました。



  • 「通常、無意識の偏見は人種差別と性差別の文脈で話されていますが、私たちは皆、異なる偏見を持っています：10進主義。 10進数の方が2進数よりも優れていると無意識に考えています。」
  • 「私が彼をどんなに愛していようと、さまざまな文化が私たちのプログラミングを台無しにしました。 すべて同じ文字セットを使用できますか？」
  • 「UTF-8はすべての問題を解決したわけではありません。 同じ名前の2人のユーザーが異なるブラウザーを使用する場合、1人は同じフィールドに収まり、もう1人は収まらないことがあります。 これで故障を感じます。」

ディラン・ビーティ



この講演者は基調講演をしませんでしたが、彼に言及することは絶対に不可能です-会議で彼は多くの観客を征服しました。 さらに、彼らの一部は報告前でさえありました：ギターやその他の楽器をDotNextの初日後のパーティーで彼と一緒に持ち込み、彼は即興ロックコンサートを行い、「Sultans of Swing」ダイアストレイツから「Seven Nation Army」The Whiteストライプ。



レポートでは、BeattyがRESTのアイデアが厳しい現実とどのように衝突するか、実際に何が起こるかについて話しました。 「開発者が取得するレプリカ」の例があり、ディランはロシア語を話せませんが、DotNext視聴者向けにこれらのレプリカをロシア語化することにしました。 そして、彼はスライド上でそれを翻訳しただけでなく、ラテン文字でプロンプトを書き留めて、「Vash idiotski API peregruzhaet nashu bazu dannikh！」という表現で注意深く読み上げました。聴衆は反応を見なければなりませんでした。







同時に、ディランの報告が完全に無責任な楽しみであると想定すべきではありません。 彼は特に、RESTのアイデアは長期的には優れていると指摘しました（最初はもっと努力する必要がありますが、最終的にはそれだけの価値があります）。そのため、「将来の自分」に対する責任だけです。 ディランはこれを理解していません。彼は2000年からSpotlightで働いているので、自分の古いコードに定期的に出会っており、将来、コーナーを切りたいという欲求がどのように戻ってくるかを知っています。



彼の例では、ビーティーは「楽しい」と「プロフェッショナリズム」がまったく矛盾しないことを鮮やかに示しました。 そして、会議自体を同じようにしようとしました：同時に楽しくて便利です。
