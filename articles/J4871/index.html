来年のウェブ向け占い

Richard MacManusが彼のブログでまとめた今年の結果についてはすでに書いています。 彼の次の記事である2007 Web Predictionsは 、彼の同僚と共同執筆しており、来年と、インターネット業界の発展の主な傾向について説明しています。 トレンドはいくつかの大きなグループに分けられます。



RSS、構造化データ



2006年に世界的に認められた標準となったRSSは 、 引き続き勝利を勝ち取り 、Windows Vistaおよび最も人気のあるメールサービスYahoo Mailの更新されたインターフェイスに統合されます。 さらに、 TechmemeからRSSで広告をブロードキャストするなど、興味深い（革新的とも言える）ソリューションを期待しています。



構造化データ ：MicrosoftとYahooがサポートするオープンなマイクロフォーマット標準、Googleの競合相手、対立と混乱の可能性-このトピックは2007年の中心的なトピックの1つになります。



すでにすべてのコーナーで誇示されているウィジェットは広がり続けます。 助けを借りて実装された商用およびマルチメディアアプリケーションの出現を期待する必要があります。



エンタープライズアプリケーション



このような背景でGoogleとMicrosoftが対立する中、 Webオフィスは引き続き発展し 、小規模な新興企業（Zoho、Zimbra、ThinkFree）の革新に支えられています。



コラボレーションおよびWebアプリケーション -リモートチームのコラボレーションを組織できる仮想オフィスは、企業のIT分野にしっかりと参入します。 企業のブログも増え続けますが、このトピックに関する議論は、機密情報と公開情報の境界を定義するのが難しいため継続されます。



ウェブ開発



リッチインターネットアプリ（RIA）は2007年に大きな力になるでしょう。 AdobeのApolloプラットフォームは最大の関心事ですが、MicrosoftのWindows Presentation Foundationおよび代替ソリューションは割引されるべきではありません。 たとえば、OpenLaszloプラットフォーム。 デスクトップとWebを調和的に組み合わせたアプリケーションは、純粋にブラウザーアプリケーションを無効にし、より多くの機能を提供します。



一方、Googleは間違いなくブラウザ内アプリケーションの宣伝を続け、AJAXの制限に対する解決策を徐々に見つけていきます。 おそらくディストリビューションは、ベクトル（VML / SVG）グラフィックスを積極的に使用するアプリケーションを受け取るでしょう。



セマンティックインターネットは2007年に人気のトピックになります。 RadarNetworksやMetawebなどの企業は、RDFの早期採用に貢献する必要があります。



アマゾンウェブサービス 、その高度な技術（S3やEC2など）は、他の企業に同様のソリューションの作成を促すでしょう。 Googleがこれを行う可能性が非常に高く、Microsoftがそれに従う可能性が最も高い。 Amazon自体は、複数の新しい方向性を開きます。



検索とオンライン広告



ほぼ確実に、私たちはオンライン広告市場の大転換を目の当たりにします。AdSenseは、MicrosoftのadCentreやYahooの新しい広告プラットフォームとの激しい競争に直面します。



市場では、疑わしい現代のCPC / PPCを置き換えるために、 オンライン広告の新しいモデルを必要とし、作成します。



2007年は、いわゆる検索2.0または垂直検索の時代になります。 Technorati、Pluggd、Live Image Search、およびその他の特殊な新しいWave検索エンジンは、情報配信の独自の方法により、Googleやその他の従来の検索エンジンよりもはるかに優れています。 ただし、巨人も着席しません。 Googleのコード検索とヘルス、およびテストサイトでスキップしている技術革新は、同じ方向のアクティブな作業を証明しています。



マイクロソフト対 グーグル



Microsoft Windows Liveサービスはベータ版から抜け出し、MSNのサービスに取って代わり、Windows Vistaのリリースのおかげで、競合他社に熱心になります。



これらのサービスからの脅威に対抗するために、Googleはある種のWebOS / GoogleOSをリリースする可能性が高く、その噂は非常に長い間広まっています。 繰り返しになりますが、LinuxはOSの中核となる可能性があります。



KDE4やXGLとBeryl with 3Dエフェクトなどのオープンソースデスクトップは、NovellとRed Hatからの新しいビルドとともに、引き続き十分な注目を集めます。



ブラウザ



Firefox、IE、および程度は低いがOpera、Flock、Maxthonなどのプログラムが関与する2番目のブラウザ戦争は破産し、長い間激怒し続けます。 Googleがどのように振る舞うかはまだわかっていません：独自の、あるいは異質な、しかし制御されたブラウザーは、この会社をまったく阻止しなかったでしょうが、その投資家はそのような動きを確実に評価することはないでしょう。



ソフトウェアメーカーはWebKitに頼ります。



マルチメディア



インターネットテレビ 、YouTube / Googleビデオ、Skypeの作成者によるVenice Projectなどの代替プロジェクト、およびIPTVとBitTorrentの大規模な導入のおかげ。



ピアツーピアネットワークの場合、来年が最も成功するでしょう。 トラフィックの増加、便利な統合Webインターフェイス、およびメディア企業との協力-これらが開発の主な方向です。



Second Lifeなどの仮想世界は 、広告主、マーケティング担当者にとって重要なプラットフォームとなり、一方で将来のソーシャルネットワークの要素となるでしょう。



仮想マネーにはさらに広い範囲があります。



コミュニティ指向のコンシューマアプリケーションは、従来のビジネスを「損なう」ための成功するビジネスモデルを探し続けます。



2006年のブームの後、 ソーシャルネットワークはメンバーから多くの時間を批判的に奪い、いわば反社会的となり、実生活を妨害します。 これにより、ネットワークユーザーアクティビティが減少する可能性があります。



国際ネットワーク



OLPCは、無料のプラットフォームとWebアプリケーションの使用に基づくシンクライアントコンピューターの採用に大きな弾みをつけることができます。



ブロードバンドインターネットアクセスは、Web 2.0の精神に沿ったサービスのニーズに追いつくのが困難で、地球上でさらに広がります。



機動性



VoIPは他の音声通信と同じように馴染みがあり、通信業界に打撃を与えます。



モバイルインターネットは、広告主にとって魅力的な媒体になります。



ウェブにアクセスするための機器 、電話、その他のモバイルクライアントは、消費者の目にポイントを獲得し始めます。



現時点では、読者が最も重要だと考えている大きなトレンドについての調査で、インターネットテレビとビデオサービスの進歩が進んでいます。
