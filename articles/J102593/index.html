Firefox 4 Beta：新しいアップデートがリリースされました-Web開発者にどのような影響がありますか？

Firefox 4の最新*ベータ版が到着しました（ダウンロードしてください ）。 これには、 Firefox SyncとFirefox Panoramaだけでなく、 何百もの修正が含まれています。 また、新しいウェブ開発環境の簡単な概要は次のとおりです。
  • FirefoxはHTML5ビデオのbufferedプロパティをサポートするようになりました 。
  • HTML5ビデオの「autobuffer」プロパティは「 preload 」に置き換えられました。
  • より合理化されたJavaScriptアニメーションを提供する実験的なAPIが追加されました。
  • CSS実験的拡張： カスタム要素を背景として-moz-elementでレンダリングします。
  • デフォルトのキャッシュサイズが増加しました（50 Mbでしたが、250 Mbになりました。 バグを 参照 ）。
  • 一部のDOMプロパティへのアクセスが高速化されました（.id、.className、 .style、 ...-バグ584287 および 584293を参照）。
____________

* 翻訳者注：これは4番目のベータ版、 Gecko / 20100818 Firefox / 4.0b4です。
