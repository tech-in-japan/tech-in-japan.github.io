ロシアの開発者にオープンする前にWindowsストアに招待します

特に、Windows Phone用の高品質で成功したアプリケーションの開発者向けに、MicrosoftはアプリケーションをWindows 8プラットフォームに移植するためのサポートプログラムを開発しました。



現在、サポートプログラムには以下が含まれています。



  1. 個人的な技術サポート-ロシア語を話す専門家に直接質問することができます;
  2. 相談とUXアプリケーションを設計し、Windows Phoneアプリケーションに設定した高品質の基準と、Windowsストアが作成する要件を満たすようにします。
  3. Application Excellence Labsに参加するためのアプリケーションの優先的な検討-最も近いものは今週 （7月23〜27日）モスクワで開催されます。








Microsoftは開発者に何を期待していますか？



このプログラムは3か月間設計され、9月末に終了します。したがって、プログラムへの参加を有効にするには、遅くとも9月1日までにアプリケーションの動作バージョン、8月1日までにアプリケーションの基本設計、8月15日までにプロトタイプを用意する必要があります。



アプリケーションの実用的なプロトタイプを既にお持ちの場合は、7月23〜27日に開催される今後のApplication Excellence Labsにご参加ください。



プログラムへの参加を確認するために、 win8 @ apps4all.ruにメールを送信します 。このメールには、どのアプリケーションをどのような時間枠で書きたいかが記載されています。



Windows 8の一般リリース後、Windowsストアでは、開発者が100を超える言語で200を超える市場（70を超える国の通貨をサポート）で世界中のアプリケーションを販売できるようになります。 マイクロソフトは、当初発表されたように、27ではなく41の市場でのアプリケーションの立ち上げをサポートします。
