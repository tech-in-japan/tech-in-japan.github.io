GoogleパブリックDNS

ブラウザのアドレスバーにwww.wikipedia.orgを入力すると、Wikipediaにアクセスする以外に何も必要ありません。 おそらく、 ドメインネームシステム （DNS）のバックグラウンドで何が起こっているのか、あまり詳しくないでしょう。



現在、 インターネットを高速化するための継続的な取り組みの一環として、 Google Public DNSと呼ばれる独自のDNSサーバーを立ち上げており、試してみてください。



私たちのほとんどは、DNSに精通していません。 ほとんどの場合、すべてがインターネットサービスプロバイダー（ISP）によって処理されますが、DNSはインターネットに重要な機能を提供します。 たとえば、覚えやすいドメイン名を変換するスイッチと考えることができます。 www.google.com-一意のインターネットプロトコル（IP）番号-たとえば、74.125.45.100-コンピューターが相互に通信するために使用します。



ほとんどのインターネットユーザーは毎日数百のDNSクエリを実行し、一部のページはサイトの読み込みを開始する前にいくつかのDNSクエリを必要とします。 これにより、ブラウジングエクスペリエンスが低下する場合があります。 私たちの調査では、ユーザーにとって速度が重要であることが示されているため、過去数か月間、私たちのエンジニアはパブリックDNSサーバーの改善に取り組んでおり、ユーザーをより速く、より安全に、より安全にオンラインで使用できるようにしています。 ドキュメントに対して行った特定の技術的改善についてお読みになり、 当社のWebサイトでインストール手順をご覧ください 。



経験豊富なユーザーで、ネットワーク設定を変更する方法を知っている場合は、 Google Code Blogにアクセスして、コンピューターまたはルーターにGoogle Public DNSをインストールする方法の詳細な手順と詳細を確認してください。



GoogleパブリックDNSの使用を開始したら、ウェブコミュニティや他のDNSプロバイダーと学んだことを共有して、世界中のユーザーのインターネットエクスペリエンスを向上させる予定です。 Google Public DNSの目標は世界中のユーザーに役立ち、何万ものDNSプロバイダーがサービスを改善し、最終的にすべての人にとってインターネットを高速化するのを支援します。
