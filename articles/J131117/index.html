前文。



多くが掃海艇をプレイしました。 特定の結果を達成しました。 一部の人にとって、これらの結果はわずかになり、結果を改善するための長い試みが始まりました。 最初の移動後にゲームを再起動することが可能で、鉱山が「ugい」点在していることがわかりました。 バージョン5.0および5.1（それぞれWindows 2000およびXP）でチートを使用できます-ゲームを開始し、xyzzyを押してエンターシフトを入力し、座標（0、0）でモニターピクセルをたどります（左上）、または人気のある編集プログラムを使用できますArtMoneyの思い出。 これがすべての始まりです。



可能なすべてを検索しようとした後、ArtMoneyで、メモリを表示および変更する機能が発見されました。 エディターを最初に開いたとき、何も面白くありませんでしたが、初期値領域から少し離れて、非常に面白い場所が見つかりました。 バージョン5.0では、場所はアドレス0x01005700、5.1 0x01005340で始まり、サイズは27 x 32バイトです。







そして、私が最初にしたことは、多くが嫌がらせを受けたと思うかどうかを確認することでした。 最初の動きから地雷を弱体化させることは可能ですか？ いいえ、できません！ これは、ゲームボーイカラーのゲームのバージョンでのみ実行できます







また、バージョン6.0および7.0では、ボタンを押して同じゲームをもう一度プレイした場合。 XPまでのMicrosoft Windows用の掃海艇のすべてのバージョンでは、最初の動きを損なうことは不可能です。



最初の動きが地雷に当たった場合、地雷はフィールドの左上隅に移動します。 左上隅の地雷をクリックすると、地雷は右に「跳ね返る」、つまり、座標（1、0）でセルに表示されます。地雷が既に（0、0）にある場合も同様です。



また、メモリを介して、フィールドのサイズを変更できます。 これは「カスタム」モードから実行できますが、このモードには制限があります。 フィールドを8 x 8より小さく、24 x 30より大きくすることはできません。フィールドを特に大きくしないと（25 x 30にできます）、逆になります。 フィールドを4から4にして、5分間置くことができます。







インターフェイスは、このような寸法用に設計されていません。



メモリ内の地雷原の位置がわかったので、ゲームは、異なる人が同じフィールドを渡すためにゲームにセーブ/ロードを追加することから、掃海艇を25 x 30ピクセルの指標として使用することまで、新しい可能性を開くことができます。 例として、掃海艇のテトリス、ヘビまたはアルカノイド。



掃海艇での作業中に気づいたことがいくつかあります。



このゲームは、ワークグループ用のWindows 3.1に登場しました。 バージョン3.1、4、および5ではゲームアイコンは変更されませんでしたが、何らかの理由で95、98、およびMEで変更されました







95日から始まり、私で終わるように、スケジュールが変更されました。 シカゴでは、グラフィックスは変更されていません。 白黒バージョンは変更されていません。







ベストタイムウィンドウにもいくつかの変更が加えられました。
