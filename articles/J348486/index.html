Kali Linux 2018.1リリース





ローリングリリースのKali Linux 2018.1がありました。これには多くの更新が含まれています。更新されたパッケージ、更新されたカーネル、より良いハードウェアサポート、更新されたツールを提供します。







Kali Linuxは、Webアプリケーションの脆弱性の分析から、ネットワークやサービスのハッキング、システムの修正まで、侵入テストを実行するための多くのユーティリティを含むディストリビューションです。 このディストリビューションは、以前はバックトラックと呼ばれていました。







Kali Linux 2018.1には新しいカーネル4.14.12があり、2つの新しい機能に注目したいと思います。







AMDセキュアメモリ暗号化のサポート。 これは、新しいAMDプロセッサーで使用可能な機能で、DRAMを自動的に暗号化/復号化できます。 これらの機能を追加すると、物理的なアクセスがあってもメモリが読み取り不能になるため、システムがコールドブート攻撃に対して脆弱ではなくなります。 （+偏執病の場合）







メモリ制限の増加。 最新の（および古い）64ビットプロセッサには、64 TBの物理アドレススペースと256 TBの仮想アドレススペース（VAS）の制限があります。カーネル4.14では、4 PBの物理メモリと128 PBの仮想メモリのサポートが追加されました。 （+メモリをペタバイト単位で測定する場合）。







また、このアップデートには、zaproxy、secure-socket-funneling、pixiewps、seclists、burpsuite、dbeaver、reaverなどの一般的なユーティリティのいくつかの エラーと新しいバージョンの削除も含まれています。 Hyper-Vハイパーバイザーの実行に対するサポートの改善。







以前のディストリビューションを更新するには、次を実行する必要があります。 







# apt update && apt full-upgrade


かなり長い間ディストリビューションを更新していない場合は、新しいリポジトリキーを追加する必要があります。 







 # wget -q -O - https://archive.kali.org/archive-key.asc | apt-key add


または、 公式ページから新しいイメージをダウンロードできます 。
