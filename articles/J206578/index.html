Tizenアプリケーションの開発者向けのMarmalade Technologies Ltdからの特別オファー

おはよう、ハブル！



まだ知らない場合は、Tizenアプリケーション開発者がTizen Associationのパートナー企業であるMarmalade Technologies Ltdから6か月間Marmalade SDKライセンスを取得するユニークな機会と、 appbackrから現金報酬を受け取る機会があることを急いでお知らせします。







参加条件は次のとおりです。



  1. 2013年12月31日までにここで登録してください 。
  2. TizenプラットフォームをサポートするMarmalade SDKの最新バージョンをダウンロードします（動作するには、ExpressバージョンまたはXcode 4.1以降を含むVisual Studio 2010以降がインストールされている必要があります）。
  3. Marmalade SDKを使用して、アプリケーションをTizenプラットフォームに開発または移植します。
  4. （このオファーに加えて）完了した（またはほぼ完了した）アプリケーションをappbackrに送信し、アプリケーションのポイントを獲得します。
  5. アプリケーションの準備が完了したら、2014年1月31日までに登録し、 Tizenストアで認定を受けてください。 アプリケーションが以前にappbackrに送信された場合、「検証チームへのコメント」フィールドに「appbackr」を追加することを忘れないでください。
  6. 6か月間Marmalade SDKライセンスを取得し、appbackrから現金報酬（PayPalで最大10,000ドル）を受け取ります。
  7. 最高のアプリはTizenストアとappbackrから宣伝されます。


プロモーションの詳細： www.madewithmarmalade.com/offers/tizen
