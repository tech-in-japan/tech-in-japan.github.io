4月の新しい支払い方法とプレミアムサブスクリプションの20％割引





ロシアでの Evernote の発売の記念日に、ユーザーに驚きを用意しました。



主要なソフトキーソフトウェアのオンラインストアと知り合いになり、Evernoteプレミアムサブスクリプションを購入できるようになりました。 これは、さまざまな支払いシステムと方法を使用してプレミアムを支払うことができることを意味します。 それらのいくつかを次に示します。



  • Yandex.Money ;
  • 個人アカウントQIWI ;
  • WebMoney ;
  • RBKマネー ;
  • Stream.ru ;
  • Dom.ru ;
  • テレペイ
  • HandyBank ;
  • MoneyMail ;
  • Payment.Ru ;
  • 現金および現金なしの支払い。


ここにそのような印象的なリストがあります。 また、Evernoteは銀行カードによる支払いもサポートしていることを思い出します。つまり、ようやく人気のある支払い方法の全リストを網羅し、できればロシアのユーザーの生活をできる限り簡素化したことを願っています。



発表されたパートナーシップと既に言及したラウンドの機会に、Evernoteの年間プレミアムサブスクリプションをSoftkey経由で購入する際に、ユーザーに20％の割引を提供することにしました。 割引を利用するには、 このリンクをたどって年間サブスクリプションの料金を支払うだけです。



割引は4月末まで有効です！



1年後、Evernoteサブスクリプションを更新するには、Evernoteアカウント設定に移動するだけで、利用可能な支払い方法のリストにソフトキーが表示されます。



また、Evernoteサービスは、ノートを快適に操作するために必要なすべての基本機能をユーザーに完全に無料で提供し、その機能を常に拡張していることを思い出してください。 同時に、仕事や生活でこのサービスを積極的に使用する人々は、作成されたノートのボリュームを1か月あたり40メガバイトから500メガバイトに増やし、追加の機能セットへのアクセスを提供するために、プレミアムサブスクリプションを取得しています。



特に、プレミアムユーザーは次の機能を利用できます。

  • 任意のタイプのファイルの追加と同期。
  • iPhoneおよびiPod Touch上のオフラインのメモ帳。
  • PDFファイル内の画像のテキストのインデックス作成。
  • ノートブック内のノートの共同編集。
  • Windows、Mac用のクライアントアプリケーション、およびウェブインターフェースに広告ユニットがない。
  • 1営業日以内のテクニカルサポートからの返信。
  • 画像内のテキストを認識するためのサーバーへの優先順位。
  • すべてのデータのSSL暗号化によるセキュリティの強化。


プレミアムアカウントのメリットについては、 当社のWebサイトをご覧ください 。
