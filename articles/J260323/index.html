IaaSプロバイダー向けのGartnerのマジッククアドラント

今日、インフラストラクチャクラウドサービスIaaSのグローバル市場は変革の状態にあり、多くのプロバイダーは大きな市場シェアを獲得できないことで戦略を変えています。 顧客はプロバイダーを選択する際に注意が必要です。 人々がクラウドへの移行を考えるとき、最初に頭に浮かぶのはサービスとしてのインフラストラクチャ（IaaS）です。 2014年には、IaaSクラウドワークロードの絶対的な増加が初めて、オンプレミスアプリケーションのワークロードの増加を上回りました。 市場は未成熟のままですが、急速に進化しています。 市場シェアの分布は、市場の爆発的な成長にもかかわらず、集中力を高める傾向があります。



2014年はIaaSクラウドプロバイダーのベンチマークの年であり、その多くは失敗として戦略を認識していました。 それらのいくつかは、まったく新しいIaaSプラットフォームを作成したか、以前に作成されたソリューションに大幅な変更を加えています。 他の企業は機会を検討しているか、主要プロバイダーのプラットフォームに基づいたサービスの提供に向けて戦略を転換している最中です。



多くの市場関係者は、クラウドベースのIaaSサービスへの投資を停止または大幅に削減しようとしていることに気付いていますが、他の企業は市場を離れるか、プラットフォームを交換する予定です。 ガートナー社は、購入者にサプライヤを選択する際には非常に注意し、サービス開発計画を指定し、少なくとも12か月間はサービスの大幅な変更または終了を禁止する契約義務を要求するよう求めています。

現在、クラウドベースのIaaSは、開発およびテスト環境、HPC、Webサイト、Webアプリケーション、および重要ではないビジネスアプリケーションに使用されています。



ビジネスに不可欠なアプリケーションにクラウドベースのIaaSを使用している組織の数は増加しており（Gartnerによると2014年の28％）、クラウドへの完全な移行を計画しています。 ほとんどの場合、移行は、IT機器の更新やデータセンターの作成などの資本コストを削減するために行われます。 2015年にGartnerが実施したCIOの調査では、CIOの83％がIaaSをインフラストラクチャ開発に適した選択肢であり、10％がIaaSをデフォルトの選択肢であると考えています（「Flipping to Digital Leadership：The 2015 CIO Agenda」）。



提示された「マジッククアドラント」は、Gartnerのお客様のニーズに対応するIaaSプロバイダーをランク付けしています。クラウドにデータセンターを設置する必要性があり、お客様は運用上の責任の大部分を保持します（その後、管理されたサードパーティサービスを通じてこの責任を外注するつもりであっても）。 ガートナーの顧客には主に企業、中規模企業、あらゆる規模のテクノロジー企業が含まれ、評価結果は一般的な顧客の要件に焦点を当てています。







リーダー

リーダーは、戦略的な事業開発と野心的な計画に適したサービスを提供することで、市場で際立っています。 それらのソリューションは幅広いケースに適用できますが、すべての可能な領域に理想的ではなく、特定のケースでは最良とは限りません。 成功事例の重要なリスト、市場シェア、多くの有名な顧客がいます。



チャレンジャー（チャレンジャー）

応募者は、個々の関連するマーケティングニーズに対応できるように適切に配置されています。 彼らは、アプリケーションの特別なケースを目的とした優れたサービスを提供し、ビジネス上の問題をうまく解決します。 しかし、彼らは市場の課題に迅速に適応することも、十分な願望を欠くこともできません。



先見の明

先見者は未来に対する野心的なビジョンを持ち、独自の技術の開発に多大な投資を行います。 新しい市場参加者でも、ビジネスを変える既存のクラウドサービスプロバイダーでもかまいません。 彼らのソリューションには多くの開発機会がありますが、狭い範囲のユースケースにのみ適用できます。



ニッチプレーヤー

ニッチプレーヤーは、高度に専門化された分野で優れたプロバイダーになることができますが、そのソリューションは幅広い製品ラインで違いはなく、すべての場合に使用できるわけではありません。 彼らはこの市場への比較的新しい参入者であるか、まだ大きな市場シェアを獲得していないかもしれません。 関連市場で確固たる指導的地位を占める人もいますが、IaaSクラウドサービス機能の開発の初期段階にあります。 かなり優れたIaaSプラットフォームに基づいた管理サービスを専門とするプロバイダーは、このカテゴリに含まれます。 顧客ニーズの専門性が高いほど、ニッチプレーヤーのソリューションが適用される可能性が高くなります。





Microsoft Azure

  • Microsoft Azureは、全体として機能するIaaSおよびPaaSの統合コンポーネントを対象としています。 新しい機能とサービスが急速に登場しています。 このシステムは、自律的な作業のための主要なソリューションであるだけでなく、マイクロソフトのオンプレミスインフラストラクチャ、開発ツール、アプリケーション、およびSaaSサービスとのシームレスな統合の可能性を提供します。


  • マイクロソフトブランド、幅広いクライアントベース、開発への多額の投資、積極的な製品開発計画により、IaaSの戦略的プロバイダーの地位を迅速に達成することができました。 Microsoft Azureは急速に成長しており、市場シェアでは2位であり、競合他社のコンピューティングパワーを2倍以上超えています（市場シェアリーダーであるAmazon Web Servicesを除く）。 マイクロソフトは、Amazonと同等の公開価格を維持する必要があり、実際には、割引のあるお客様は、Microsoft Enterprise License Agreementに基づいてAmazonと競合する価格/品質比を受け取ります。
