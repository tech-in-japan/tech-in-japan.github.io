記事を書く人のためのHabra Editor

Habrausersと記事の著者は、 HabraRedactorプロジェクトに注目します。これにより、Habrの記事を書く際に少し新しい見方をすることができます。 Habra Editorは、主に個人用に作成されました。 記事を書くときに私にとって欠けていたものを実装します。



なに



  • テキストの余分なブロックが負担されない広々としたインターフェイス。
  • 幅が広く、ほぼフルスクリーンのテキスト入力フィールド。
  • 便利なツールバー。
  • メイン入力フィールドが大幅に拡大され、テキスト内をより適切に移動できます。
  • すべてのHabrahabrタグがサポートされています。
  • すべてのタグは選択したテキストで機能しますが、現在読んでいるような行のブロックを選択し、そこからリストを作成できます。
  • img、a、user、font、youtubeなどのタグの便利な挿入。
  • habracut'om、このタグの自動挿入に関する問題の根本で解決されました。
  • 真実がまだ確定しているというプレビューがあります。
  • サービスpoison.qsh.ruを強調表示する構文を表示するパネルがあります。
  • 受信したテキストのクリップボードへの自動コピー。
  • 音訳;
  • キャラクターマップ;
  • 下書きを保存します。


ブラウザ



HabRedactorテストは、次のブラウザーで実行されました。

  • Internet Explorer 7;
  • Opera 9.5;
  • Firefox 2.0.0.14;
  • Firefox 3.0。


計画されていること



  • プレビューを完成させます。
  • Habrディスプレイを備えたパネルを作成します。
  • ブラウザのサポートを改善する
  • おそらくドラフトを保存する機能を実現します。


このようなプロジェクトに興味を持っているすべての人からのコメント、希望、バグ報告、批判に感謝します。 おそらく、Superhabrを使用すると、非常に便利に記事を書くことができますが、現時点では、独自のバージョンを提供します。
