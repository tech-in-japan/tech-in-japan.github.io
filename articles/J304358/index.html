仮想化プラットフォーム：より多くの仮想マシン、高速

Lenovo System x3650 M5サーバーは、仮想化パフォーマンスで業界をリードし、消費者が複雑さを軽減し、仮想IT環境をより効果的にサポートできるようにします。

エネルギー効率-節約の基礎



組み込みストレージの高性能と大容量、およびエネルギー効率の高いアーキテクチャにより、Lenovo System x3650 M5サーバーは、サーバープラットフォームのエネルギー効率に関して2つの記録を樹立しました。 Lenovo System x3650 M5は、 ワットあたりの記録 的なパフォーマンスとワットあたりの 全体的なパフォーマンス（ストレージサブシステムを含む）を達成しました：SPECvirt_sc2013_ServerPPW 4.196 @ 132 VMおよびSPECvirt_sc2013_PPW 4.196 @ 132 VM。



仮想化ワークロードのエネルギー消費量に関するストレージパフォーマンスの記録 。 SPECvirt_sc2013_PPWベンチマークに基づくと、より優れたエネルギー効率が示されました。 Lenovo System x3650 M5サーバーの消費電力は、すべてのx86システムの中で最も低くなっています。 レノボは、内部ストレージを使用し、ストレージエリアネットワーク（SAN）を使用しない場合に印象的な結果を達成しました。そのため、サーバーにより、お客様は大幅なエネルギー節約を達成し、電気代とデータセンターの冷却にかかる費用を削減できます。



x3650 M5の4つの大型ファンが、6つのデュアルローター「ホットプラグ」モジュールに置き換えられたことに注意してください。 ガイド付き冷却システムは、それらを2つのゾーンに分割し、750ワットの「チタン」電源と組み合わせて、サーバーがより高い周囲温度で動作できるようにします。



すべての仮想化ソリューションにおけるエネルギーワットあたりのプロセッサ特性の比率に関する記録 。 ベンチマークSPECvirt_sc2013_PPWは、サーバーの最も効率的な電力消費を明らかにし、仮想化環境でのLenovoの優位性を示しました。



アプリケーションサポート：幅広いタスク



デュアルプロセッサデータベースとトランザクションシステムワークロード（OLTP）の記録 。 TPC-Eのテスト結果によると、Lenovoサーバ​​ーは価格/性能比でFujitsu PRIMERGY RX2540 M1よりも30％高速で9％優れていました。 ベンチマークTPC-Eは、サーバーの特性を実際に評価し、データベースとOLTPの中で価格とパフォーマンスの最適なオファーを特定するように設計されています。



TPCが開発したTPCベンチマークEベンチマークパフォーマンステスト（TPC-E）は、オンライントランザクション処理（OLTP）ワークロードをシミュレートします。 TPC-Eテストは、証券会社のOLTPロードをシミュレートします。 テスト中、中央データベースは、会社の顧客の個人アカウントに関連するトランザクションを実行します。 TPC-Eテストは証券会社のビジネスモデルに基づいて構築されているという事実にもかかわらず、論理データ構造、データボリューム、トランザクションタイプ、および実装ルールは、現代のオンライントランザクション処理システムで広く使用されています。



ビジネスプロセスの観点から、TPC-Cは、多くのユーザーがDBMSに格納されたデータを使用してトランザクションを実行するコンピューター環境をシミュレートします。 このようなビジネスアクティビティは、エンタープライズ管理システムであるか、航空券の注文であるかにかかわらず、アクティビティの分野に関係なく、注文入力システムの特性です。 このテストには、注文入力のシミュレーション、請求書の発行、支払い情報の入力、注文ステータスの確認、倉庫の状態の追跡が含まれます。 システムおよびデータベース構造の負荷は、特定の業界の詳細を保持しないように選択されますが、特定の商品またはサービスの管理、販売、または流通に関連する平均的なビジネスを反映します。
