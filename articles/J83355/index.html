VUEを介した認定テストの支払い

はじめに



このノートでは、Pearson VUEシステムを介した認定テストの支払いに関するささやかな経験を共有したいと思います。

まず第一に、ピアソンVUEとその最も近いライバルであるプロメトリックについてのいくつかの言葉。



Pearson VUEはPearson Education Corporationの一部であり、シスコシステムズ、CompTIAなどの世界的な大手ソフトウェアメーカーの製品のコンピューター認証テストを提供する世界的に有名なテストプロバイダーです。

Virtual University Enterprisesは1997年に設立されました。 このグローバルなeテストサービスには現在、120か国以上に3,400以上の認定テストセンターがあります。



プロメトリックは、独立テストの親会社であるドレイクプロメトリックのフォロワーとして、VUEより数年早く市場に登場しました。 プロメトリックの本社は米国のボルチモアにあり、世界中にオフィスがあります。

過去10年間、同社はOracle、Sun Microsystems、HP、SCO、Microsoft、Novellなどの有名なベンダー企業と協力して、テストの専門家でした。 テストは、認定プロメトリックテストセンター-APTCで行われます。APTCは、世界中のプロメトリックとの契約に基づいてネットワークを運営しています。



配達が成功した場合、ベンダー企業は候補者に受け取ったステータスを確認する証明書を候補者に送信します。



シスコシステムズは、テストを実施するためにピアソンVUEを選択しました。その後、このシステムのアカウントを通じて試験の注文と支払いのプロセスを検討します。



シスコシステムズのテスト



認定テストに合格するプロセス全体は、次のポイントに分けることができます。

  1. 方向の選択（必要な試験のリスト）;
  2. 試験の準備;
  3. テストセンターの選択。
  4. 試験支払い;
  5. 選択した試験に合格。


パラグラフ1〜3は個別の議論に値するため、現在のノートでそれらを検討することにはあまり意味がありません。 パラグラフ5は、あなた自身の経験に関する物語であり、関連フォーラムでの議論に適しています。 現時点では、このリストの4番目の項目を検討したいと思います。

なぜ彼は正確に？

支払いの段階で、質問が発生します。支払い方法は？ この質問には2つの答えがあります。

  1. テストセンターで支払います。
  2. VUEシステムのアカウントを通じて支払います。


複雑なことは何もないように見えますが、選択したテストセンターの費用はかかりますが、試験の金額は自動的に増加し、場合によっては過払いが30％以上に達することもあります。 VUEシステムで直接支払うことにより、試験自体に対してのみ支払うことになります。 心配する必要はありません。公認のテストセンターでは試験を受ける必要がありますが、2番目のオプションを選択した場合、「上から」試験を受けることはありません。 したがって、大幅な節約が可能です。



試験支払いプロセスの詳細



テストセンターでの試験の支払いは次のとおりです。

  1. テストセンターに来るか、そこに電話して、選択した試験の番号に電話して、配達時間に同意します。
  2. センターの管理者から直接または電子メールで領収書を受け取り（領収書を印刷する必要があります）、銀行、郵便局などで支払います。
  3. 支払い後、センターの管理者に支払いが完了したかどうかを確認します（原則として、ヒッチは発生しませんが、明確にすることをお勧めします）。
  4. 設定した時間に、テスト開始の15分前にテストセンターに行き、試験に合格します。


VUEシステムを介した試験の支払いは、上記のプロセスよりもはるかに複雑ではありませんが、より快適（不要なジェスチャーが少なく）、経済的です（試験自体のみが支払われます）。 この支払い方法について詳しく説明します。 あなたが必要とする最も重要なものは、あなたがオンラインで支払うことを可能にする銀行カードです（私の場合、VTB24 eカード）。 また、ピアソンVUEシステムに既にアカウントを持っていることを意味します。それ以外の場合は登録します。

  1. 試験に合格する時間についてテストセンターに手配します。VUEを使用して支払いを行う場合は、合格する時間を選択する必要があります。 ;
  2. ログインリンクをたどります。 [ キャリア認定とスペシャライゼーション試験 ]をクリックして、アカウントでログインします。
  3. 表示されるページで、 試験のスケジュールセクションに移動します。ここでは、興味のある試験を選択し、目的の試験の前にチェックマークを付けます。
  4. この段階で、受験を希望するテストセンターを選択する必要があります。 最も近い -表示されているテストセンターの数を見つけます。 国 -検索する国。 Cityは検索する都市です。 3つすべてのフィールドに入力して[ 検索 ]をクリックすると、入力した条件に従ってテストセンターのリストが表示されます。このリストで、自分に合ったショッピングセンターを選択し、[ 次へ ]をクリックします。
  5. 次のステップは、テストの日付と時刻の選択です。ここでは、テストセンターと合意した日時を選択する必要があります。選択はテストセンターのタイムゾーンで行われることに注意してください。 [ 予定の選択]をクリックし、 [ 次へ]をクリックして次の項目に進みます。
  6. はい、同意し 、[ 次へ ]をクリックします。
  7. 連絡先の詳細を確認し、バウチャー番号がある場合は入力して、[ 次へ ]をクリックします。
  8. 住所を確認し、支払いのカード詳細を入力して、[ 次へ ]をクリックします。
  9. ここでも、試験、言語、場所、時間、費用などのすべてのデータを慎重に確認します。 すべてが正しい場合は、[ 読みました ]の横のチェックボックスをオンにして... 、[ 注文の確認 ]をクリックします。
  10. テストの注文が完了しました。すべての情報をもう一度確認してください。 電子メールに手紙が届き、試験の順序を確認し、銀行口座は試験の費用分だけ減額されます。 :)


より信頼できるように、テストセンターに電話して、管理者にすべてが間違っているかどうかを尋ねてください。 結局のところ、試験の15分前にテストセンターに行き、静かに受験することを忘れないでください。

テストセンターでは、テストセンターからの返却に対して追加料金を請求すべきではありません。これが発生した場合は、すぐにVUEに連絡して確認してください。
