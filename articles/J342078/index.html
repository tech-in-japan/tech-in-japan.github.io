SkolkovoとSAPを使用して、SAP Cloud Platformに基づいてアプリケーションを起動および宣伝する方法

今年、SAPは、サービスとしてのプラットフォームまたはPaaS（サービスとしてのプラットフォーム）モデルを使用してビジネスアプリケーションを開発および展開するための新しいクラウドプラットフォームであるSAPクラウドプラットフォームについて活発に議論しています。 10月にSkolkovo Foundationと共同で、ロシアの開発者がSAP Cloud Platformにソリューションを起動、開発、または転送するのを支援する新しいプログラムを開始しました。 プログラムのすべての段階を経る企業は、SAPのお客様とパートナーの間でのソリューションの促進、およびマーケティングとPRのサポートを受けます。







SAP Cloud Platformとは



SAP Cloud Platformは、SAP HANAとASE、およびオープンソースシステムPostgreSQL、MongoDB、Hadoopでデータストレージと処理機能を提供するオープンSAPクラウドプラットフォームです。 SAP Cloud Platformは、SAP Cloud Platformに統合された統合サービスのおかげで、任意のクラウドまたはオンプレミスシステムと簡単に統合できるモバイルおよびHTMLアプリケーションを開発するように設計されています。 現在、SAP Cloud Platformは、モノのインターネット、予測アルゴリズム、機械翻訳、その他の機械学習方法など、約70の異なるサービスを提供し、独自のWebIDE開発環境を備えており、ラピッドプロトタイピングツールを構築し、Java、XSJS、C ++アプリケーション開発をサポートしています、Python、Ruby、Node.js、Go、PHP、.Net。



SAP Cloud Platformは、独自の企業アプリケーションを開発するための環境として、および既製のサービスを別のプラットフォームから転送してソリューションを企業のSAPシステム（ERP、CRMなど）と統合するための両方の環境として使用できます。

SCPの利点の1つは、このSAP HANAのベースとの統合です。 SAP HANAは、さまざまな種類の情報をすばやくダウンロードして分析できる強力なビッグデータ分析プラットフォームです。



SAPとSkolkovo Foundationの共同プログラムに選ばれる新興企業と企業は、SAP Cloud Platformへの優先アクセスと、ソリューションの開発または移転のための専門家サポートを受けます。



プログラムに入る方法



プログラムの第1段階はすでに開始されており、11月30日まで続きます。これは、プログラムの申し込みの受理です。 これを行うには、参加者は会社に関する情報を入力し、登録ページの特別なフォームを使用して、申請に関するプレゼンテーションを送信する必要があります。 プレゼンテーションでは、ビジネスの観点と技術的な実装の観点の両方からソリューションの詳細な説明を提供することが重要です。



第2段階は、11月2日から12月22日まで続きます。この間、Skolkovo Foundation ITクラスターとSAPの専門家が、リモートで、または対面のミーティングとインタビューの形で受け取ったアプリケーションを選択します。



プログラムのメインフェーズは12月末から始まり、2018年中続きます。 この期間中、参加者はソリューション検証の段階を経て、SAP Cloud Platformでのソリューションの転送と適応に従事します。 これらのすべての段階を成功裏に終えたチームは、Skolkovo FundとSAPからマーケティングとPRのサポートを受け、企業顧客の困難な市場に参入します。
