デルが公開され、ハイブリッドクラウドに向かう

弊社の資料の1つで、Dellは株式公開会社になり、5年ぶりに証券取引所に株式を置く予定であると述べました。 今週の初めに、デルの証券が流通する日付が判明しました-12月28日。 同時に、ITの巨人は、ハイブリッドクラウドとパブリッククラウドの積極的な開発に関するコースを取ると発表しました。 ポイントが何であるかを説明します。





/写真Dell Inc. CC BY-SA



株式の配置について簡単に



デルでの開発を計画しているプロジェクトに進む前に、会社が一般に公開される必要がある理由を説明します。 実際、Dellには500億ドル以上の借金があります。 Michael Dellが3年前にEMCを購入するためにお金を借りた後、彼は現れました。 それ以来、デルはこの負債を返済しようとしてきました。



この問題を根絶するために、デルは公開会社になることを決めましたが、IPOはありません（これにより、あらゆる種類の官僚機構が節約されます）。 これにより、同社はサードパーティの投資をさらに引き付けることができます。 管理者は2つのシナリオを検討しました。 最初の例には、VMwareの子会社とのリバーステイクオーバーが含まれていました。 しかし、このオプションは株主と投資家に適さないため、放棄しなければなりませんでした。 これについては、Habréのブログで以前に詳しく説明しました 。



2番目のシナリオは、VMwareの株式の再編成に関連しています。 目標は、VMwareからターゲット（追跡）株式を購入し、市場で新しいDell証券を発売することです。



この問題に関する交渉は5か月間行われました。 そして先週、取引を承認した株主の投票がありました。 追跡株式の保有者は、資産を120ドルで売却したり、新しい証券と交換したりできます。 したがって、ITの巨人はIPOのない公的組織になります。


株式の公募に続いて、デルはハイブリッドおよびパブリッククラウドテクノロジーを積極的に開発する予定です。



クラウドに向かって



ハイブリッドクラウドは、2019年にデルの主要な目的地の1つになります。 デルは同じVMwareを使用して開発する予定です。



2018年の初め以来、VMwareはCloudHealth Technologies、CloudCoreo、Heptioの3つのクラウドベースのスタートアップを買収しています。 最初の2つは、企業がパブリッククラウドでビジネスプロセスを管理し、クラウドリソースのコストを制御するのに役立つ製品を開発します。 Heptioに関しては、Kubernetesに環境を展開するためのソリューションを開発する会社です。 アナリストは、 Kubernetesがまもなくクラウドにハイブリッド環境を展開するための「標準」になると予想しています。



同時に、DellとVMwareは、データセンターのシステム管理者のタスクを簡素化するシステムの開発に引き続き取り組みます。 11月、VMware Cloud Foundationハイブリッドクラウドおよびデータセンター展開ツールは 、vSphereおよびVirtual SANに基づくDell EMCハイパーコンバージドコンプレックスであるVxRailおよびVxBlockとの統合を受け取りました 。 この機能により、Dellのハードウェア製品とソフトウェア製品が単一のエコシステムにより密接にリンクされるはずです。



VMwareに加えて、IT大手の別の子会社であるPivotalは、Dellとマルチクラウドを開発します。 VMwareはサーバーテクノロジーを開発する一方、Pivo​​talはソフトウェアソリューションに焦点を合わせます。 たとえば、12月初旬、Pivo​​talは、ハイブリッドクラウドでのサーバーレスコンピューティング用のオープンソースプラットフォームであるPivotal Function Serviceをリリースしました 。 また、Kubernetesに基づいて構築されています。



コミュニティの意見



潜在的なリストラの課題の1つは、DellとVMwareの財政状態の悪化です。





/写真ブルーノ・コルディオーリ CC BY



トランザクションの条件の下で、娘は数十億ドルの金額で株主に配当を支払う必要があります。 アナリストは、このため、VMware は年間赤字のままになると推定しています。 同社は来年に利益を得ることが期待されていますが、財政赤字は新製品の開発速度に影響を及ぼす可能性があります。



もう1つの問題は、投資家がDellの株式を購入するのを嫌がる可能性があることです。 公募後、支配権は会社マイケル・デルのCEOになります。 デルは、「すべてを一度に手に入れたい」、「攻撃的な行動」を望むという市場で知られています。 これにより、大規模な投資家が追い払われ、開発計画が損なわれる可能性があります。



この物語全体に肯定的な側面を見る人がいますが。 ターゲット株式の保有者の1人とヘッジファンドのジョシュ・ウッド（ジョシュ・ウッド）の投資家によると、証券パッケージの再編は、株主VMwareとDellの利益を一致させるのに役立つでしょう。 これにより、リーダーシップの不一致の数が減り、企業が将来の共通のビジョンに到達するのに役立ちます。



ブルームバーグへのデル株式の配置を公式に確認する前でさえ、 彼らは同社の顧客が取引からの利益のみを受け取ると書いた。 VMwareソフトウェアがDellハードウェアと統合されると、ユーザーは完全なクラウドソリューションを低価格で購入できるようになります 。 そのため、デルには、データセンターとクラウド用の単一のハイブリッドエコシステムを作成するあらゆる機会があります。
