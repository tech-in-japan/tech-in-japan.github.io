BlackBerry Curve 9220スマートフォンの購入方法

私は自分をBlackBerryのファンとは決して考えていませんでした。さらに、この会社のスマートフォン（しばらくの間はRIMではなく、そのように呼ばれています）は非常にまれに、そして長くは私に来ませんでした-それで、それを手に変えて肩をすくめて直接所有者に返しました。 数ヶ月前、私の友人は「ブラックベリー」を手に入れることを決めました-そして常に公式で、キーにキリル文字が付いています。 ウクライナでBlackBerryスマートフォンを販売しているのはMTSのみで、友人にはまったく異なるオペレーターがいます。したがって、同じBISを接続するという話はなく、デバイスは別のSIMカードで使用する予定でした。 一般的に、購入の目的はまだ明確ではありません...あなたはそれが時々起こる方法を知っています、人は朝目を覚ますと考えています：私はBlackBerry（iPhone、Xbox、Lamborghini、新しい冷蔵庫、カミングアウトを作ります-あなたが必要なものに下線を引きます）、行き、買います。 ここでは、明らかに同じ状況について。





BlackBerry Curve 9220の購入日から2週間以内に（正確に言えば）友人がスマートフォンに飽きてしまったため、非常にリーズナブルな価格でデバイスをコレクションに追加しました。 このモデルを操作する際のニュアンスについてお話したいと思いますが、まずは、取得プロセスについて少し触れておきます。



MTSサロンの素敵なコンサルタントの少女は、誰かが突然BlackBerryスマートフォンの購入を考えたという事実におそらく驚いたでしょう。 デバイスを見せて販売するというリクエストを聞いた少女は、次のウィンドウでQWERTYキーボードを使用してNokia Ashaの携帯電話を見て、次のように言いました。 見て-どれだけかわいいか、緑か、安いか。」 実際、BlackBerry Curve 9220の色ではすべてが本当に複雑です。ウクライナでは黒バージョンのみが提供されています。 自然界では、インターネットによると、白、赤、青、濃い青緑色のバージョンもあります。





状況は、価格に関しても奇妙です。 BlackBerry Curve 9220がスマートフォンであり、比較的有名なブランドのものであることは明らかですが、それでも... 1,700グリブニア、つまり200ドル以上を求めています。 このお金で、Androidでも非常に最新のスマートフォンを購入でき、Windows Phoneでも購入できます。 前述のAshaについてはスチュワードに黙っておく方が良いでしょうが、それでも-MTSはBlackBerryスマートフォンを提供しています。何らかの理由で、サロンでの機能と利点を説明するパンフレットはありませんでした。 それらについてのコンサルタント（スマートフォンについて）も何も知りません。 何らかの理由で、彼らはBISを接続するための提案を受け取りませんでした。 誰がこれを責めるべきかわからない-MTSまたはBlackBerry掘り、これは今日呼吸している。



99％のケースでは、平均的な「やかん」購入者がNokiaを購入します。製品は外部からは同じように見えます（サイズ、タッチスクリーンとQWERTYキーボードの欠如のみ）、フィンランドのデバイスは半額です。 ロシア語（まあ、またはウクライナ語）でビジネスですか？ どうやら、はい。 特にBlackBerryの購入を目的とする深刻な人々は、おそらくMTSに連れて行くことはありません。実際、オペレーターのキャビンには5人の女の子がいます。そして、初めてではないにしても、このブランドのスマートフォンを販売していることは明らかでした。二番目。 （これは、BlackBerryスマートフォンが1つのMTSショールームでのみ利用できる50万の都市です。）



これは、BlackBerryの保証カードが箱に入っていないため、内部サーバーで見つけて印刷し、記入する必要があるという事実からも明らかです。 それで少女たちは彼をどこで探すべきかわからなかったので、彼らはタバコを吸うために出かけたKolyaという従業員を待つように頼みました。 彼は出かけ、姿を消した。 その結果、少女たちはどこかに電話をかけ、何かを見つけ、最終的にこの不運な保証書を印刷しました。 やったー



BlackBerry Curve 9220を1か月以上操作した後、おそらく「乾杯」とは言えません。 BlackBerryスマートフォンを完全に使用するには、BISをオペレーターと接続する価値があり、MTSも持っていないため、調和の取れたエコシステムを楽しむことができなかったことは明らかです。 それでも、メール、VKontakteメッセンジャー（ひどい曲線-たとえば、ほとんどの場合、新しいメッセージを自動的にダウンロードしたくない）、ブラウザー（Opera Miniを含む）、およびBlackBerry App Worldストアへのアクセス（そこからVKontakteメッセンジャーがダウンロードされました）をセットアップします「Opera」）にはBISがなくてもかまいません。これには、いくつかの民俗的な方法があります。GoogleまたはYandexへようこそ。 同時に、公式にプリインストールされたTwitterおよびFacebookクライアントは機能しませんでした。





BlackBerry OS 7.1 インターフェースは、このCurve 9220システムの制御下で正確に見え、動作します-非常に珍しいです。 たとえば、iPhoneまたはAndroidに1年以上座っていた場合、このスマートフォンのメニューは、アイコンが曲がったとき、奇妙なデザインの動きなどのときに過去に戻ります。 しかし、一般的には、インターフェイスに慣れることができます-プロセスに慣れるのに数日かかりました。 珍しいことに、アプリケーションの更新プログラムをインストールした後、スマートフォンを再起動する必要があることに注意してください（！）そして非常に長い電源を入れます-バッテリーを取り外し、所定の位置に置いて電源ボタンを押すと、約2分かかります。



人間工学とハードウェアについては、BlackBerryがブランド化された成果を十分に持っていることは明らかですが、残念ながら2013年にはそれらはすべて実際には適用できず、誰も本当に必要としません。 たとえば、QWERTYスマートフォンを繰り返し扱ってきましたが、テキストを入力できませんでした。指がハードウェアレベルでキーボードと互換性がないことが判明しました:)。 しかし、ボタンが小さいBlackBerry Curve 9220では、長い文字を入力できます。これらのボタンの形状が非常に便利だからです。 もう1つのポイント：タッチスクリーンなしで電話またはスマートフォンを取り上げた場合、メニューを数分歩き回った後、マシンのディスプレイを自動的に突っ込み、仮想ボタンを押してみます。 ここでは、画面をクリックしたことはありません。インターフェイスのナビゲーションは光学式ジョイスティックを使用して実行されますが、これも非常に便利です。 確かに、私たちの時代では-タッチスクリーン、そのようなコントロール、そしてQWERTY-キーボードの時代は歴史のゴミ箱にあります。



結婚なしではない。 スマートフォンの右側には、アプリケーションの起動を割り当てることができるキーがあります。 そのため、彼女は仕事を拒否しました-他のすべてのボタンは明確なコースを持ち、時計のように機能しますが、これはそうではありません。 ここでは、BlackBerry Curve 9220のサイドパネルがゴムでトリミングされていることを説明しておく必要があります。ゴムにはボード上のボタンの凸部があります。 おそらく、工場では、ボード上のボタンが弾性バンドを使用して意図されたシートに届きませんでした。 このような問題を解決しました。バックカバーを外し、指の爪で歯茎を引き抜いて（引き裂いたり、単に損傷したり）、左右に「揺れ」ました。 その結果、1時間の苦痛の後、ボタンは機能しました。
