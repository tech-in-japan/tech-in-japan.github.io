PartyChat-GoogleTalkでの会議！

GoogleTalkで会社全体とチャットしたいですか？

そのような機会があります！ これがPartyChatです。



必要なのは、partychat @ gmail.comなどのアドレスの1つにメッセージを書き込むことです。



FAQから次のことが判明しました。

→同時に11の部屋にいることができます（理論上-制限ではありません）。

→部屋はパスワードで保護できます。

→チャット履歴はGmailアカウントに保存されます。

→これはオープンソースプロジェクトです。

→このカート全体は、自宅のGoogleの従業員の1人に有効です。







いくつかのPartyChatコマンド：

/ create chat_name [optional_password] -新しいルームを作成します。 パスワードが指定されている場合、ユーザーはパスワードを入力する必要があります。

/ join chat_name [password] -既存のチャットにログインします（パスワードが必要な場合があります）。

/ status-チャット名と私のエイリアスを表示します。

/リスト -現在のルームのチャットのリスト。

/エイリアス[名前] -ニックネームを変更します。 名前が指定されていない場合、現在のニックネームは「削除」されます。

/スコア[名前] - 名前でポイント数を取得します。 名前なしでリスト全体を取得します。

/私のメッセージ -第三者からのメッセージ 。

/理由[名前] -ポイントの増加/減少の詳細を除き、/スコアと同じです。

/ exit-現在のチャットを終了します。

/ヘルプ -PartyChatの使用方法と/コマンドへのリンク。

/コマンド -このヘルプ。

/スヌーズ時間 -「snooze 1h」や「snooze 15m」など、指定された時間のチャットを無視します。

/ whois name-このチャットのユーザー情報を表示します。

/ウィスパー名メッセージ -「プライベート」メッセージ。
