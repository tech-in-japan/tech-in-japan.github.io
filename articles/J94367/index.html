Visual Studio 2010のロシア語版はすでにここにあります



5月21日、 ReMIXカンファレンスで、MicrosoftはVisual Studio 2010のロシア語バージョンを発表しました。これは、2010年4月12日にVisual Studio 2010の発表時に発表されました。 Visual Studio 2010は、完全にローカライズされた唯一の開発環境です。 更新されたインターフェイス、IntelliSenseオートフィルテクノロジー、デバッガー、ツールチップなど、すべてのコンポーネントはロシア語で表示されます。 Microsoft開発プラットフォーム全体のドキュメントであるMSDNライブラリも完全にローカライズされ、無料で配布されています。 MSDN Bulletin Readerは、ローカリゼーションを評価した最初のユーザーです。MSDNオンラインライブラリに今すぐアクセスしてください。 MSDNサブスクライバーの場合、 Visual Studio 2010のロシア語バージョンは5月21日10:00（モスクワ時間）から利用可能になり、同時に試用 版とExpressバージョンがダウンロード可能になります 。 近い将来、Visual Studio 2010のロシア語版がWebsiteSpark 、 DreamSpark 、およびBizSparkに登場します。
