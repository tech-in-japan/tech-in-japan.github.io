スタートアップを組織しましたか、または組織しようとしていますか？ お手伝いさせてください！

この投稿は、Microsoftのスタートアップサポートプログラムに関する情報を収集する試みです。



おそらく、 BizSparkでストーリーを開始する価値があります。 このプログラムを使用すると、ソフトウェア開発用のMicrosoftツールをほぼ無料で入手できます（プログラムへの参加登録は無料で、プログラム自体は3年間継続し、3年間で100ドルの寄付を支払う必要があります）。

このプログラムに参加するには、あなたの会社は次のことをしなければなりません。

  • ソフトウェアを開発する、またはソフトウェアを開発する予定（アウトソーシングおよびサービス活動は適切ではありません）
  • 3年以内に存在する
  • 年間収入が50万米ドル以下である（そうでない場合は、ライセンスを購入しない）
スタートアップを組織し、会社を登録することを計画している個人でも登録できます。



製品のリストについては、 Habréの記事 、公式サポートサイト、またはロシアのITビジネスを支援するイニシアチブの公式Webサイトで読むことができます。



プログラムに参加してMicrosoftからソフトウェアを入手するには、 gaidar.magdanurov @ microsoft.comに、会社に関する簡単な情報、またはスタートアップが実装するアイデアについての簡単な情報を送信する必要があります。その後、会社が条件を満たす場合プログラムの場合、ウェブサイトのプログラムに無料で登録できるコードを提供します： http : //www.microsoft.com/BizSpark/Startup/Signup.aspx 。



話す価値のある2番目のプログラムは、 Microsoft Business Startです。 これは、開発の最初の3年間にプロジェクトの資金を調達する機会です。 昨年、93件中3件に1件の割合で資金が提供されました。 さらに、申請が承認されると、無料のStart In Garageトレーニングに参加する機会が与えられ、ビジネスの成功に必要な知識が得られます。



マイクロソフトがこれらのプログラムを必要とする理由は、皆さんも理解していると思います。マイクロソフトのソフトウェア製品を使用している企業は、自社だけでなく、それを使用する製品自体も宣伝しています。 ただし、これらのプログラムは新興企業にとっても有用であり、GDPの知的部分の量が増えるにつれて、国のビジネスと経済を後押しすることができます。



これらのプログラムについて質問がある場合は、コメントで質問してください。私と同僚が答えます。 そしてもちろん、プログラムに登録してください！
