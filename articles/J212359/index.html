The Pirate Bayの創設者は、トレントトラッカーサーバーの場所に関して北朝鮮と交渉しました



スウェーデンの北朝鮮大使館



昨年、多くのリソースが、最も有名なトレントトラッカーであるThe Pirate Bayのサーバーを北朝鮮に移動（または既に移転）できるというニュースを発表しました。 もちろん、そのようなニュースの一部の読者が示唆したように、これはすべて冗談に過ぎないことが判明しました。



それにもかかわらず、昨年ではなく2004年から2005年にかけて、The Pirate Bayチームと北朝鮮の代表者間の交渉が実際に行われました。 これは先週のインタビューでトビアスアンダーソンによって発表されました。



当時、アンダーソン氏によると、トレントトラッカーチームは、スウェーデンの北朝鮮大使館と、サーバーを大使館に移す可能性について交渉を開始しました。 当時、The Pirate Bayには現在ほど多くのユーザーがいなかったため、特に広いチャンネルは必要ありませんでした。 また、サーバーの基本操作は、大使館スタッフが提供できる通信チャネルで十分にサポートできます。



北朝鮮大使館の従業員に連絡するために使用された方法も興味深い。ベイチームは、電子メールで大使館に連絡しようとするよりも良いものを見つけられなかった。 回答に対する希望はほとんどありませんでしたが、急流トラッカーチームはすぐに回答を受け取り、非常に前向きでした。 交渉が始まりましたが、残念ながら、議論がビジネスフェーズに入ったとき、すべての電子メールアドレスが「反撃」し始めました。



後に、The Pirate Bayの代表者から連絡を受けた大使館の従業員が、故郷の北朝鮮に呼び戻されたことが判明しました。 何が起こったのかは明らかではありません-それが労働者のフルタイムの交替だったのか、それとも彼らが北朝鮮で従業員と「衰退する西側」の代表者とのつながりについて学び、この人を思い出すことに決めたのかどうか。



それはともかく、海賊湾は北朝鮮の領土に位置する可能性が高かった（どの大使館もこの大使館が代表する州の領土と見なされる）。 現在、トラッカーサーバーはどこにありますか？
