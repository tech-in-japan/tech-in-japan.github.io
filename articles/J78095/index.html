210ドル、DualSIM、5Mp、WiFiのAndroidコミュニケーターのレビュー





中国の携帯電話の市場について話すと、タッチスクリーン、2枚のSIMカード、テレビ、おかしな翻訳で、不器用なノクラがすぐに思い浮かびます。 しかし、中国では有名な携帯電話の偽物やコピーを作るだけではありません。 携帯電話メーカーのサイトを見ると、デザインと機能の面で多くの興味深いモデルを見つけることができます。

さらなる写真=トラフィック！



2枚のSIMカードで動作するSciphone N21 Android コミュニケーターについてお話ししたいと思います。これはGeneral Mobile DSTL1とRoverPC Twinです（ロシアで間もなくリリースされます）。 なぜこんなにさまざまな名前ですか？ 簡単です。これらの企業はすべて、このモデルをODM製造業者のYuhua Teltechに注文し 、その後、独自の名前で販売しています。



  • Android 1.5
  • ネットワーク：GSM / EDGE 850/900/1800/1900 MHz、2枚のSIMカードの同時操作
  • Marvell PXA310プラットフォーム上の624 MHzプロセッサ
  • RAM：128 MB
  • データストレージ用メモリ：256 MB +メモリカード用スロット
  • インターフェース：Wi-Fi、Bluetooth 2.0 + EDR（A2DP）、充電/同期/ヘッドセット用のminiUSB（USB 1.1フルスピード）コネクタ
  • 画面：TFT 3” 240x400ピクセルの解像度（WQVGA）
  • カメラ：5メガピクセル（2560？1920）、オートフォーカス、ビデオはCIF解像度で記録されます
  • FMラジオ、加速度計
  • バッテリー：取り外し可能な1200 mAhリチウムイオン
  • 寸法：111x54x16 mm
  • 重さ：135g。








ScophoneのWebサイトでコミュニケーターを注文しました。価格は210ドルと配達料22.42ドルで、パッケージはモスクワに1か月近く行き、先日無事に受け取りました。 荷物の外観検査中、誰かがその荷物の上に座っているように感じましたが、荷物を開けると、梱包テープで丁寧に包まれた電話の箱が四方にありました。











ボックス内に次のコンポーネントが見つかりました。

  • ネットワークアダプタ;
  • ヨーロッパのプラグ用アダプター;
  • ヘッドセット
  • PCとの同期用ケーブル。
  • 指示;
  • MicroSDメモリカード1 Gb。








製造業者は、ヨーロッパのコンセント用にアダプターを慎重に配置しました。 ヘッドセットはシンプルで、通話応答ボタンと音量コントロールがあります。 メモリカードには、かなり中国の女の子の写真、1ダースの歌、3gpビデオがありました。

装置自体は珍しく見えます。このようなプラスチックの鏡面ブロックには、斜めのリブがあります。 前面は非常に汚れやすく、すぐに汚れますが、これは使いやすさには影響しません。









人気の、黒のソフトタッチプラスチックの背面。 手に取って、何も演奏せず、たむろしないでください。







写真では、フロントパネルのボタンはタッチセンシティブであるように見えますが、デバイスを手に取るとすぐに疑問が消えます。 ボタンには小さなストロークがあり、押されても不便はありません。 4方向ジョイスティックはよく押されており、中央のキーは非常に大きいため、見逃すことは困難です。







右側には、2ポジションのカメラボタンと標準のminiUSBがあります。 miniUSBを見ると、標準とわずかに異なることに気付きました。 これは問題を引き起こしません。通常のminiUSBケーブルは通常コネクタに入りますが、デバイスに付属のワイヤは他のminiUSBデバイスに適合しません。 ちなみに、USB 1.1 Full-Speedインターフェースでは、データは1Mb / sの速度で送信されます。 左側には音量ボタンがあります。 サイドボタンのストロークは弱く、クリックしたときのクリック感はあまり感じられませんが、そのようなインスタンスに出くわしたのかもしれません。 伸縮式スタイラスが上部に隠れています。 フォーラムには既に何人かの人がいて、それを破ることができたので、注意してください。 残念ながら、3.5 mmヘッドフォン出力はありません。miniUSBのアダプターを購入するか、ヘッドセットを使用する必要があります。 右上には、充電プロセスが進行中であることを示す小さな青色のダイオードがあります。









TFT画面は3インチで、解像度は240x400、26万色、センサーのタイプは抵抗です。視野角が大きく、最大設定での輝度が高くなります。画面の「ミラーリング」により、太陽の下で何かを読むことは困難ですが、可能です。抵抗があり、指を使用すると便利です。解像度を大きくしたいのですが、デバイスの価格が上がることを忘れないでください。画面が狭いため、ダウンロードしたアプリケーションの一部が適合せず、インターフェースの一部が表示されないことがありますが、これは主に機能に影響しません。







オートフォーカスとLEDフラッシュを備えた5メガピクセルのカメラ。 さらに、5メガピクセルは本物ですが、中国の携帯電話で20メガピクセルを書くのが好きで、内部では0.3メガピクセルかかります。 カメラソフトウェアは驚くほどうまく設計されており、さまざまな撮影モードがあります。 フォーカスモードの選択と多くの興味深いことがあります。 写真の品質は悪くありませんが、十分な色がなく、写真は明るくありません。 照明に応じて、オートフォーカス速度は平均です。







内部には、Marvell PXA 310プロセッサー、624MHz、128 MB RAMおよび256 MB ROMがあります。 ワイヤレスインターフェイスのうち、Bluetooth 2.0 + EDRおよびWi-Fi 802.11b / gが利用可能です。 Bluetoothヘッドセットをヘッドフォンに接続しました。巻き戻しボタンは機能し、問題はありませんでした。



デフォルトでは、Android MarketのないAndroid 1.5がありましたが、バージョン1.6はすでに利用可能であり、Googleサービスは特別なパッチを使用して配信できます。



SIMカードとmicroSDを挿入するには、バッテリーを取り外す必要がありますが、あまり便利ではありません。 インストールメカニズム自体は面白そうです。すべてのカードがはしごのように挿入されています。 メモリカードの挿入は簡単ですが、SIMカードの場合はもう少し複雑です。 SIMカードをスライドに挿入してスロットに入れる必要があり、約40秒かかりました。バッテリーの下に2つのIMEIのステッカーがあり、モデル名DSTL1とGeneral Mobileは慎重にマーカーで覆われています。













2枚のSIMカードの操作は非常に便利です。たとえば、番号をダイヤルし、呼び出すSIMカードを選択するには、ジョイスティックの左にある上または下のボタンを押します。 また、画面をタップして電話をかけると、どのSIMカードを呼び出すか尋ねられます。 着信コールでは、呼び出しているSIMカードに書き込まれます。 両方のSIMカードのアクセスポイントを登録できます。 デバイスには2つの無線モジュールがあります。 あなたの番号のうち2つは通話に利用できます。 あなたが退屈したら、あなたは自分自身に電話することができます。 2枚のSIMカードを使用することの利便性については説明しませんが、これはすでに理解できます。



加速度計の動作は良好で、誤検知はありません。画面はすばやく回転します。



アクティブに使用すると、コミュニケーターは夕方まで生き続け、バッテリーは1200 mAhです。



現在、Androidでは、HTC Tattoo（約15,000ルーブル）などの予算デバイスを提供していますが、価格を比較すると、予算デバイスが何であるかさえわかりません。 中国人がやったことはこの超予算のデバイスですか？



最後に、このお金（7000ルーブル）でこれは非常に非常に興味深いデバイスであると言います。ローバーの碑文とまったく同じものがここで16〜2万で販売されます。 7000の場合、タッチスクリーンデバイスを購入できるようになりましたが、コミュニケーターではなく（WMは見ていません）、この機能の半分もありません。 彼らが広告で言うように「だからなぜもっと払うのか！」あなたが初めてAndroidを試してみたいなら、このデバイスはあなたのためです。
