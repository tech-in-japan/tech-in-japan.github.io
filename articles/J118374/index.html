リモートプレゼンスのロシアのロボットは現実です

2009年に2つのセンセーショナルな映画「 Surrogates 」と「 Avatar 」を公開し、人の遠隔地での存在を確保するためにロボットを使用するという単純だが非常に革新的なアイデアは、現在ハリウッドによって積極的に活用されています。 最初の映画では、「サロゲート」は人を完全に置き換えるヒューマノイドロボットで、特別な「ターミナル」を介して自宅にいるオペレーターによって制御されます。 映画「アバター」では、人間の知性の担い手は、人間のDNAとエイリアンのヒューマノイドクリーチャーのDNAから作成されたバイオロボットであり、遠隔操作者の意志に完全に準拠しています。 しかし、現実に戻って。



同時に2か3つの場所にいる必要があったことがありますか？ 私たちは高度な技術の時代に生きており、新しい技術で人を驚かせるのは困難です。 そしてもちろん、今日のリモートプレゼンスの問題は長い間解決されてきました。 SF映画で観察できる程度にしないでください。



今日は、国内メーカーのリモートプレゼンスロボットRBot 100についてお話します。何故なら、私は何とかして自分で触ったからです。 これにより、会議、展示会、学校での遠隔操作が可能になります（はい、そのようなアプリケーションでも可能です）。 どのように機能するのか、何ができるのか、どこに適用されるのか、それをレンタルまたは購入できるのか、以下すべてです。





西洋の対応について少し。

似たような西洋のソリューションの中でも、Anybotとその最新のQBモデルに注目する価値があります。 最初のロボットモデルであるAnybots QAデバイスは、2008年に登場しました。 ロボットの高さは152 cm、重量-25キログラムです。 Anybots QAモデルのコストは3万ドルですが、より新しいモデルのAnybots QBは現在テスト段階にあります。 QBモデルの重量はわずか16キログラムで、そのコストは1万ドルから1万5千ドルで、セグウェイデバイスの原理で動きます。



フランスの企業ゴタイに代表されるジャズは、「ボディ」と「ヘッド」で構成され、車輪の上を移動する高さ1メートルの機械です。



RoboDynamicsによるTilr。 ロボットは平面モニターのように見えます。 TiLRの特徴は、カメラがロボット本体とは独立して動くことです。 オペレーターは、ロボットとカメラの動き、そのスケーリングを制御できます。



また、他の一部（特に、有名な会社iRobotでさえそのモデルをリリースしました ）。



ロシアのロボット

R.Botは、インターネットを介して、ロボットが配置されている場所にオペレーターのリモートプレゼンスを提供する最初のロシアのロボットです。 あなたのコンピューターにいることで、オペレーターはロボットの動きを簡単に制御し、マルチメディア機能を通じてロボットの周りの人々とコミュニケーションをとることができます。 管理は、WiFi標準802.11 b / gに従って実行されます。 ロボットには、最大640x480の解像度のカメラ（医療用にはより高度なカメラが使用されます）、高感度のマイク、ステレオスピーカーが搭載されています。 国内メーカーの特徴は、軸を中心に向きを変え、頭を必要な方向に向けることができることです。 オペレーターのインターフェースは、宇宙でのロボットの動きの制御と頭部の位置の変更の両方を提供します。



QBとは異なり、RBotは3つの車輪で動きます-安定性のために2つの主要な車輪、1つは小さいです。 最初のロボットは1.9 km / hの速度で移動しましたが、それほど前ではありませんが、同社はモバイルプラットフォームを完成させ、4.6 km / hの速度のロボットを提供する準備ができています。 同時に、新しいプラットフォームはノイズを大幅に低減します。 ところで、医療機関で使用する場合、新しいプラットフォームはほとんど使用されません。 ロボットが突然近くで静かに掃除しているのを突然見た祖母を想像してください...このビデオのように、結果は予測できません：



ロボットにはタッチLCDスクリーンを装備することができます。これにより、オペレーターと展示会の訪問者などのインタラクティブな対話が可能になります。 RBot 100の動作時間に言及する価値があります。平均して8時間です。その後、ロボットを屋外の充電ベースに送る必要があります（iRobot Roombaのように、いつベースを自動的に検索しますか？）。 RBotの高さは約1メートルで、ベースの直径は55 cmです。



追加のコンポーネントをサポートおよび修復するのが難しいため、製造者は一時的にそのようなモデルを生産していません。





申込み

テレプレゼンスシステムは絶えず進化しており、近い将来、機器のコストが削減され、どこでも使用されるようになります。 テレプレゼンスロボットの主な目的は、大企業の集会への遠隔参加です。 主な外国企業は、このように製品を配置しています。 RBotはわずかに異なるパスを取りました。 特に、顧客を引き付けるために展示会で長い間積極的に使用されてきました。 このような開発は多くの人にとってまだ新しいものであり、ロボットの隣に人だかりを集めることができます。 これらの目的のために、チラシ付きの追加トレイをロボットに掛けたり、ステッカーを作成したり、会社の色でボディをペイントしたりできます。



理論的には、リモートプレゼンスロボットは、まったく異なる活動分野で使用できます。 たとえば、医学で。 このようなロボットは、作業中に、または患者の世話人として、作業者がリモートで制御できます。



RBotの興味深いアプリケーションを次に示します。 以下のビデオでは、病気のために学生は授業に参加できませんでした。 ロボットが助けに来ました。



ロボットはルームガードとして使用できます（ガススプレーとスタンガンが取り付けられるまで、オブザーバーと言う方が正確です）。 特に、Jazz Gostaiにはこのような変更があり、暗視カメラの存在によって区別されます。





そのような奇跡はどこにありますか？

ロシアの製造業者は、3時間から数日までのさまざまな期間、RBotロボットをリースしています。 製造業者とのインタビューで、例外的なケースでは、外国の類似品よりも数倍低い価格でそのようなロボットを購入することが可能であることが判明しました。 価格は発表されておらず、それぞれ個別に交渉されています。 しかし、主なことは、あなたが本当にそのような奇跡を起こしたいなら、不可能は何もないということです。 ロボットは、メーカーRBotの Webサイトとパートナーの両方で、モスクワのロボットストア： RoboticsLab 、サンクトペテルブルクのロボットストア： NanoJamで注文できます。 さらに、パートナーからの注文の場合、追加の割引を受けることができます。



ロボットが展示会に参加するために注文された場合、メーカーは配送、構成、イベント中の完全なサポートなどを含む完全なサポートを提供します。



そのような製品を生産するコストは、確立されたコンベヤーシステムがないために高いままです。 これまでのところ、ロボットは手動で組み立てられています。 しかし、約5〜10年後、あなたと私はこのような奇跡的なテクノロジーを大きな財政的支出なしで購入できるようになります。
