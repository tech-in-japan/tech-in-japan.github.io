SharePoint 2010でExcelレポートを表示する

チャートは、クライアントが最も喜んで支払うものであり、ボスはボーナスを与えます。 なんで？ 状況をすばやく理解し、同様に迅速に決定を下せるためです。 今日、多くのアナリストがビジネスインテリジェンスの分野で働いており、レポートの開発と構築に従事しています。そのほとんどは（個人的な経験に基づいて）さまざまなバージョンのMicrosoft Excelを使用して作成されています。 過去数年間、Webでのレポートの表示（内部サイトとすべてのユーザーがアクセス可能）の両方が非常に重要でした。 特殊なソリューションの購入は、コスト上の理由から常に推奨されるとは限らないため、市販のコンテンツ管理ソリューション（CMS）を使用すると、低コストで結果を達成できます。 この記事では、Microsoftが開発したCMSシステムであるMicrosoft SharePoint 2010 Serverにレポートを表示する方法を説明します。



すべてのステップを渡した結果、AdventureWorks 2008R2の例からのデータを動的に表示するグラフが表示されるページが得られます（このために、追加のステップが必要です）。そのようなページの作成方法を学びます。





この記事は次の部分で構成されています

1. SharePoint Server 2010をインストールします

2. SharePoint Server 2010を構成する

3. Excel Services 2010のグラフを公開する

4.ページにグラフを表示します。 おわりに

カットの下、最初の部分





仕事には、次のものが必要です。

1）64ビットオペレーティングシステム（SharePoint 2010の要件）がインストールされているサーバー、Windows Server 2008 SP2またはWindows Server R2。 サーバーには、少なくとも4 GBのRAMと少なくとも40 GBのハードドライブが必要です。 インストールを成功させるには、ローカル管理者権限が必要です。

2）サーバーをADドメインに含めることが望ましく、さまざまなサービスを起動するドメインユーザーを追加することができます。 ドメイン管理者権限を持っているか、これらの権限を持つユーザーがユーザーを作成する必要があります。

3）IE 8またはFirefox 3がインストールされているクライアントマシン

インストール中に、次のプログラムが必要になります。

1. SQL Server 2008 R2

2. AdventureWorks 2008R2の例

3. SharePoint Server 2010。

4. Microsoft Office 2010。

-インストール中に、注意する必要がある重要な手順のみを説明します。 すべてのプログラムにはローカル管理者権限が必要ですので、インストールする前にそれらを持っていることを確認してください



1）最初に、SQL Server 2008 R2をスタンドアロンサーバーとしてインストールする必要があります。 ライセンスの種類を尋ねられたら、エンタープライズ評価を選択します（この場合、複数のインスタンスを作成する代わりに、SQL Serverの1つのインスタンスを使用できます）。 SQL Serverの既存のインスタンスを使用する場合、この段階で高速インストールを選択できます。コンテンツをSharePointに保存するのに十分なはずです。



SharePointのMS Reporting Servicesで作成されたレポートを表示する場合は、Reporting Servicesの構成手順で、SharePoint統合モードを選択します。

SQL Serverを実行するユーザーの特権でプロンプトが表示された場合、この目的のために特別に作成されたドメインユーザー（sqlservice）をインストールすることをお勧めします。 サンプルはSQL Server 2008 R2に同梱されていないため、ダウンロードしてインストールする必要があります。 質問がある場合は、 ドキュメントを読むことをお勧めします



2）SQL Serverのインストールが完了したら（2008 R2を使用している場合の最も簡単な手順）、SharePoint 2010 Serverのインストールを続行できます。 このステップでは、サーバーをインターネットに接続することをお勧めします-これにより、インストールウィザードが必要な更新をダウンロードできるようになります。 インストールウィザードを実行し、[前提​​条件のインストール]を選択して、ウィザードが必要と判断したすべてをインストールします。 インストールは、Habrの次の記事と英語のブログで説明する構成の段階とは異なり、原則として問題を引き起こしません
