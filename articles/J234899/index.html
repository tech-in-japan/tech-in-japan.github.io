PHP 5.6.0最終リリースのリリース





今日、PHP開発チームは8月28日、バージョン5.6.0のリリースを発表しました！



PHP 5.6.0の主な革新：

  • 定数のスカラー式
  • 演算子を使用して可変数の引数を持つ関数 ...



  • べき乗演算子 **



  • useキーワードを使用して関数と定数をインポートする
  • カーネルには、インタラクティブなphpdbgデバッガーが含まれています。
  • PHP：//入力の再利用が許可され、$ HTTP_RAW_POST_DATAは非推奨
  • GMPオブジェクトがオペレーターのオーバーロードをサポートするようになりました
  • 2 GBを超えるファイルのアップロードのサポート




新機能の完全なリストについては、移行ガイドを参照してください 。



以前のバージョンとの互換性に影響する変更：



  • 配列をクラスプロパティとして定義する場合、配列のキーは配列リテラルによって上書きされません。
  • json_decode（）は 、JSONを解析するときの構文がより厳密です。
  • SSL / TLSを使用する場合、ストリームラッパーはデフォルトで証明書とホスト名を検証します。
  • GMPリソースはオブジェクトになりました。
  • Mcrypt関数には有効なキーとベクターが必要です。




そしてまた：

  • PHP 5.5からアップグレードしたい人のために、5.5と5.6.0の間の詳細な変更を含む完全な移行ガイド 。
  • ダウンロード用のソースは、 こちらとこちら（windows）から入手できます 。
  • 変更の完全なリストを含むChangeLog 。
