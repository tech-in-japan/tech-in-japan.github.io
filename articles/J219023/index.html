無料のウイルス対策ソフトまたはWindows XPのMicrosoft Security Essentialsの代替品のテスト









2014年4月8日以降、Windows XPのサポートが終了したため、このOS用のMicrosoft Security Essentialsウイルス対策も機能しなくなりました。 それはかなり良い仕事をし、一般的に使用するのに便利です。



適切な無料の代替手段はありますか？ 答えを見つけていただければ幸いです。



次の構成を使用することが決定されました。

  • CPU-Amd Athlon X2 64 3800+（2GHz）
  • MB-Asus M2A-VM（am2 +）
  • RAM-2GB DDR2 800
  • HDD-Samsung SP0411N（40 GB / 2 MB / 7200 rpm / IDE）


システムには、サポートの完了の通知を含む、すべての最新の更新があります。 マザーボードキットのディスクのすべてのドライバーもインストールされます。 ビルは、各ウイルス対策をインストールする前に展開されたシステムのイメージを作成しました。

システムのダウンロード時間



手のストップウォッチで測定しました-PCの電源を入れてから、システムトレイにウイルス対策アイコン（アクティブモード）が表示されるまで。 測定値を収集するためのより正確で迅速な方法が見つからなかったため、グラフは一般的なプレゼンテーション用です。









システムフォルダーのスキャン時間



そのようなものとして-ウイルス対策なしで、システムイメージからコピーされたWINDOWS、プログラムファイル、ドキュメント、および設定、ファイルの数は変更されていません。









プロセッサーとメモリーの使用



収集はネイティブユーティリティ-logman（ alexbljackユーザーのWindowsでアプリケーションパフォーマンスメトリックを収集するためのlogmanユーティリティの使用に関する記事のおかげで）を使用して実行され、php-scriptでクリーンアップされました。
