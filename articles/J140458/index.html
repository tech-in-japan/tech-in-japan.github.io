EviterraによるFlight Search API

私たちは、サイトで航空券を検索したいすべての人のためにXML APIを長い間持っていました。

企業ブログを引用するには：



こんにちは、Habrahabr！



私たちは最も近代的な（そして安い）航空会社サービスの一つです

Eviterra.comはここに長い間何も書いていません。なぜなら、非常に興味深いニュースを準備しているからです。例えば、インターフェースの非常に重要な更新（まもなく、これまでに誰もやったことがありません）、ボーナスプログラムなどがもっと良いでしょう。



さて、その間、すべてのオタク、Web開発者、マーケティング担当者、スタートアップ、ヒップスター、航空会社などにとって素晴らしいニュースがあります。 あなたが航空券を検索したいウェブサイトを持っているなら、今すぐ私たちと一緒にそれを行うことができます。









皆のための優れたXML APIがあります。 これを使用して、航空券を検索し、デザインに完全に結果を表示できます。 購入するために、ユーザーは私たちにリダイレクトされ、あなたはあなたを通して販売されたチケットから半分の収入を得ます。 もちろん、すべての統計はリアルタイムで利用可能です。



面白い？ プロジェクト（または将来のプロジェクト）の簡単な説明をsupport@eviterra.comに送ってください。次に何をすべきかをお知らせします。
