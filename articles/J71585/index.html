セキュリティはどうですか？



さまざまな噂や個人的な意見にもかかわらず、この質問は非常に簡単です-「XPモード」を使用すると、メインのWindows 7マシン、そのソフトウェア、またはそのソフトウェアのセキュリティ設定を継承しない完全に独立した論理コンピューターを実際に使用しますつまり、Windows 7では個別に、完全に個別にはコンピューターを二重に監視し、ゲストシステムを監視する必要があります。その結果、すべての結果が生じます。



また、「XPモード」を使用することを覚えておくことも重要です。これは、Windows XPがインストールされた通常の非仮想マシンとして管理する必要がある本格的な別のマシンを使用していることです。



セキュリティを確保するには、次の手順を実行します。

  1. 適切なウイルス対策ソフトウェアがVMのXPにインストールされていることを確認してください。 ホスト上のローカルソフトウェアはXPを保護しません。
  2. 自動更新またはWSUSを使用して、必要なすべての更新がVMで受信されていることを確認します。
  3. 必要に応じて、VMにインストールされているすべてのアプリケーションが製造元から更新を受信することを確認してください。
  4. VM上で実行されているXPで不要なサービスを無効にします。

「XPモード」の対象者



マイクロソフトは、保守的な中小企業にWindows 7を支持してWindows XPを放棄するよう説得するために「XPモード」の導入を決定しました。一部の企業は、最新のWindowsのバージョンまたは一般的に互換性がありません。 古いソフトウェアとの競合に対する懸念は、一部の企業がWindows Vistaに移行しないことを決定した最も重要な理由の1つです。 XPモードを提供することにより、Microsoftは最終的にWindows XPに終止符を打ち、この老朽化したOSのクライアントベースを誘致したいと考えています。



XPモードを実行するには何が必要ですか？



Windows Virtual PCはどのバージョンのWindows 7でも実行できますが、XPモードはWindows 7 Enterprise、Professional、およびUltimateのバージョンでのみ機能します。
