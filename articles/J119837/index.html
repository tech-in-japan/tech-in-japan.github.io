100,000ドルの賞金とEvernote Gallery参加者向けの最初のパートナー会議による開発者の競争



過去数年間で、ユーザー、開発者、パートナーの素晴らしいコミュニティがEvernoteの周りに形成されました。 一緒に、外部メモリ用の最初のグローバルグローバルプラットフォームを作成します。これにより、エコシステムとのサポートと対話が最も重要な（そして有用な）タスクの1つになります。 この点で、サービスのエコシステムに特化した最初のイベント、 Evernote Trunk Conference（ETC）と呼ばれるEvernote Gallery参加者向けのカンファレンス 、および最初の開発者コンテストを発表できることを嬉しく思います。 以下では、これらのイベントについて詳しく説明します。



100,000ドルの賞金プールを備えたEvernote開発者コンテスト！



Evernoteコミュニティの開発というテーマを継続し、何百万人ものEvernoteユーザーが愛する素晴らしいアプリケーションと統合の次の波を待っています。 コミュニティの関心を高めるために、私たちは最初のグローバル開発者コンテストを開始します。



最も革新的で有用なアプリケーションの作成者は、多額の賞金に頼ることができ、その作品は8月に開催されるEvernote Gallery（ Evernote Trunk Conference ）のカンファレンスで発表されます。 コンテストのメインサイトはこちらです 。以下では、主な規定について説明します。



賞金プール


  • 6人のファイナリスト：各5,000ドル
  • グランプリ： 50,000ドル
  • 創造賞： 10,000ドル
  • 学生賞： 10,000ドル


参加方法


  1. コンテストのウェブサイトで登録してください。 特別なAPIキーを生成し、競争と会議に関するすべてのニュースを最新の状態に保ちます。
  2. 何らかの方法でEvernoteの機能を使用するアプリケーション、サービス、またはデバイスを開発します。 インスピレーションについては、Evernoteギャラリーの既存の例をご覧ください 。
  3. アプリケーションを送信します。 申請の提出は6月10日に始まり、2011年7月15日の太平洋時間23:59に終わります。 モスクワ時間の7月16日10時59分です。


競争ルール


以下は、どのように、どのようなルールで競技会が開催されるかの概要です。 すべての詳細を含む完全な情報は、 利用規約の公式セクションに英語で記載されています。

  1. コンテストへの参加申請は、2011年5月19日にEvernoteと公開統合されていないアプリケーションとサービスに対してのみ受け付けられます。
  2. すべての審査員が自由にアクセスできるように、すべてのアプリケーションは英語で実行する必要があります。
  3. Evernoteプラットフォームとのあらゆる技術的統合は、コンテストに参加する資格があります。 これらは、Evernote Web Service API、Evernote Site Memory、Evernoteソフトウェアクライアントとの直接統合、およびサイトの対応するセクションにリストされている他の方法を使用する製品です。
  4. 審査員が無料で提供しているアプリケーションまたはサービスを利用できるようにしてください。 コンテスト中は公開されない場合がありますが、審査員はEvernoteアカウントを使用して作品を評価できる必要があります。 iOS向けのアプリケーションについては、テストを実施するUDIDデバイスのリストを提供します。
  5. 2011年7月22日に、提案されたすべての作品を評価し、6人のファイナリストを発表します。 ファイナリストのそれぞれは、サンフランシスコで8月18日に開催されるEvernote Trunk Conferenceで2泊分の賞金と$ 5,000の賞金を受け取ります。
  6. ファイナリストは、2011年7月22日から8月5日まで評価されます。 このプロセスは2つの段階で構成されます。
  • 人気投票： Evernoteユーザーは、コンテストWebサイトにアクセスして、お気に入りの作品に投票できます。 ユーザーの評価は、ファイナリストの評価の1/3になります。
  • 審査員評価： Evernoteスタッフと招待した審査員のグループが、各ファイナリストの作品を評価し、好きなものに投票します。 公式評価の最終評価では、2/3の重みが付けられます。


審判


ファイナリストと受賞者を選ぶとき、審査員は次の4つの基準に従って参加者の仕事を検討します。

  • 魅力：アプリケーションはどれほどユーザーフレンドリーで美的に楽しいですか？
  • Evernoteプラットフォームを使用することの有効性：アプリケーションは、Evernoteプラットフォームが提供する独自の機能をどの程度効果的に使用しますか？
  • 有用性：アプリケーションはユーザーにとってどの程度有用であるか、または/または必要ですか？
  • アイデアの独創性：アプリケーションはどれほどユニークまたは革新的か


受賞者


以下のノミネートを提供しています。



グランプリ：最終順位で最大得点を獲得したファイナリストは、メイン賞金50,000ドルを獲得します。

学生賞：最高の学生作品（Evernoteジャッジが選択）の著者には、賞金10,000ドルが贈られます。

創造性の賞品：最も革新的、創造的、または予想外の作品（Evernote審査員が選択）も10,000ドルの賞品を獲得します。



受賞者の名前は、2011年8月18日にETCで発表されます。



ETCとは何ですか？



サンフランシスコで1日のイベントを開催します。これにより、Evernoteコミュニティがより緊密になり、開発者と起業家がユーザーとファンの利益のために相互のやり取りをよりよく整理できるようになります。 プロジェクトを開始する場合は、成功の達成を支援したいと思います。 ビジネスの成長と規模の拡大についてアドバイスが必要な場合は、経験を共有します。 Evernoteをさらに完全に使用する方法をお探しの場合は、すぐに提案をいたします。



イベントで期待されることは次のとおりです。



開発者セクション

Evernoteの開発者とパートナーは、Evernote API、クラウドアーキテクチャ、 Evernote Site Memoryアフィリエイトプログラム、モバイル環境のベストプラクティスなどに関連するトピックについて議論します。 また、Evernoteチームから直接アドバイスや支援を受ける機会もあります。



製品セクション

これらのディスカッションの一環として、サクセスストーリーを共有し、サービスとの便利な統合のためのアイデアを提供し、アプリケーションをEvernoteと統合することで目を見張るような成功を確保するためのヒントを提供します。



カスタムセクション

Evernoteが人生に欠かせないものとなった人々のヒントやストーリーに精通することをお勧めします。 Evernoteを使用して、生産性を高め、集中力を高め、独自性を高める方法を学びます。



完全な会議スケジュールはまもなくETCウェブサイトに表示されます 。
