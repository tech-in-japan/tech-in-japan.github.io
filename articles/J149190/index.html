Samsung Galaxy Note 10.1：デジタルキャンバス



春にプロトタイプのGalaxy Note 10.1を見たとき、私は思った：0_o！ スタイラスの時間がなくなった！ クパチーノの人々は、羽を故意に放棄しました...しかし、時間が経ち、すべてが変わり、知覚を含みます。 6か月後、私は180度近く心を変えました。



誰もがiPadに追いつこうとしている間に、Samsungは少し横に進み、Galaxy Noteのアイデアを開発して10インチにした。 アップルはスタイラスを完全に放棄しましたが、おそらくまだ理にかなっています。 特に、画面自体が容量性のままで、スタイラスがWacomによって開発されている場合は特にそうです！ 6か月前、MWC展示会のブースで、大きなノートのアイデアは本当に奇妙に見えました。タブレットとは別に、ペンとは別に...しかし、それ以来、ガジェットは大きく変わりました。 そして、今回は、おそらくAppleの努力によっても除外しません。 サムスンは、クパチアンがコピーされたとされるデザインに対して再び訴えることを防ぐために、可能な限りのことを行いました。







設計



Galaxy Tab 2以降、サムスンのタブレットは見やすい外観を備えています。その主な特徴は、銀色のフロントパネルの端にあるスピーカーです。 第一に、ユーザーの手のひらで音がこもらないため、美しく、第二に機能的に判明しました。 さらに言うと、音が一番かっこよかったです！ ボリュームマージンは非常に印象的であり、さらに、2つのチャネルで最終的に意味をなします。







それ以外の場合、すべては以前とほぼ同じです。 下は30ピンの充電ポートで、他のものよりも上です。 電源ボタン、ボリュームコントロール、micro-SDメモリカード用のスロット、ヘッドフォン用のジャック、SIMカードスロットがあり、このタブレットではフルサイズのままです。 さらに、ここでは、炎からの不死鳥として、家電製品を制御するための赤外線ポートが登場しました。これは、Galaxy Tab 7.7のプロトタイプで見たもので、シリーズ化されませんでした。







画面の前には1.9メガピクセルのカメラがあります。 彼女の助けを借りて、あなたはビデオ通話を行うことができ、彼女は所有者を監視する方法も知っています。 彼が画面を見ると、明るさは希望のレベルまで上がり、そうでなければバックライトは消えます。







背面にもカメラがあります。 その解像度は5メガピクセルです。 彼女は良い写真を撮りますが、十分な照明が必要です。 そして、暗闇ではLEDフラッシュのみが保存されますが、わずか数メートルしかヒットしません。

ディスプレイ



Galaxy Note 10.1の対角線は10インチで、PLSマトリックスの解像度は1280 x 800ピクセルです。 このテクノロジーは、IPSの安価な代替品としてSamsungによって開発されました。







その利点には、優れた色再現、広い視野角、低消費電力が含まれます。 現実には、すべてがほぼそうですが、個人的には、私は少し明るさに欠けています。 ただし、SamsungがデバイスにAMOLEDを組み込んでいることに慣れているかもしれません。







鉄



最近、サムスンは途方もなく強力なデバイスをリリースしました。 そして、このノートも例外ではありませんでした。 Exynosチップセットに基づいて構築されています。Exynosチップセットには、クロック周波数1.4 GHzのクアッドコアプロセッサ、グラフィックチップMali-400 MP、および（注意！）2ギガバイトのRAMが含まれています。 このすべてにより、SamsungデバイスはQuadrant StandartとAnTuTuの総合テストですべてのライバルを与えることができます。











メインメモリは、選択した構成に応じて16、32、または64ギガバイトになります。 しかし、どちらかといえば、最大32ギガまでのmicro-SDメモリカードで拡張できます。



ソフトウェア



ガジェットはAndroid 4.0.3を実行しています。 いつものように、ブランド化されたサムスンのシェルであるTouchWizは、サイズ変更可能なサイズのブラン​​ド化されたウィジェットや、S Note、S Planner、S Suggestアプリケーション、Photoshop TouchなどのOSに引き継がれています。 ここでも、おそらく、Adobe Ideasがインストールされたプロトタイプを思い出してみる価値があります。これは、スケッチを描くための優れた環境です。 残念ながら、このソフトウェアの最終バージョンにはソフトウェアがないため、Google Playストアから有料でダウンロードする必要があります。 または、より安価な代替手段として、Autodesk Sketchbook Proをお勧めします。

スタイラス



スタイラスはワコムによって開発され、ワコムは古典的なタブレットで犬を食べました。 そして、この事実は、いわば、タブレットでスケッチするのが楽しいという事実を暗示しています。 5インチのノートに比べて画面の感度が2倍になり、センサーの数も2倍になったとされています。 主観的には、ペンのセンサーの解像度は独自のWacomデバイスには届きませんが、これは大きな前進です。 次世代のノートは、表示される場合、Bamboo Penのような最初のタブレットより悪くありません。







S Noteアプリケーションのフレームワーク内のタブレットは、描画だけでなく、手書き（ロシア語を含む）テキストの認識、幾何学的図形の描画、方程式の解法など、いくつかの便利な機能を実行できます。 確かに、後者は問題を解決するのはタブレット自体ではなく、Wolfram Alphaオンラインサービスであるため、インターネット接続が必要です。















バッテリー



さて、今最も重要なことについて！ 時間...ここのバッテリーは7000 mAhに設定されているため、朝から夕方まで、タブレットは問題なくハードロードで動作します。 もちろん夜になると、彼はすでに充電を要求しますが、大食いのAndroidと高度な詰め物を考えると、これは普通のことです。
