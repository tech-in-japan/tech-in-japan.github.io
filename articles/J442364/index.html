Snom M300ベースDECTステーションおよびM215キットの概要





秋に、DECTネットワークを作成するためのSnomのMシリーズデバイスのシリーズ全体について話しました。 そこには大規模なM700ステーションが記載されていましたが、そのようなデバイスは小規模オフィス/ショップ/カフェやその他の中小企業にとって冗長です。 しかし、このニッチでは、その記事の船外に残されていたSnom M300ステーションと、まったく新しい斬新なSnom M200が完全に定着しました。



Snom M300



Snom M300はM700の「弟」であるため、それらのパラメーターの一部を比較することは論理的です。







そのため、M300は、比較的狭い領域にDECTネットワークを作成するように設計されています。 700番目のモデルのように、「300」は屋内で最大50 m、オープンスペースで最大300 mの最大範囲を持ちます。 しかし、これは理想的な条件であり、実際には、範囲は材料や壁や床の厚さに依存します。



同時に、M300は最大20のSnom M25、M65およびM85 DECTチューブ（M700-最大30個）を提供できます。 同時に、サポートされるSIPアカウントも20です。さらに、M300をマルチメッシュネットワークに結合することはできません。これは最も古いモデルとの重要な違いです。 M700を数十から数百で相互に接続し、非常に広いエリアをカバーし、合計で最大1000台のハンドセットを提供できる場合、M300は「孤独なヒーロー」です。 ただし、最大3個まで接続できるM5リピーターを使用して、このステーションのカバレッジエリアを拡張できます。 同時に、M300は最大5つのコールを処理できます（M700-最大5つのブロードバンドまたは最大10の標準。M300はブロードバンドもサポートしています）。







イーサネットネットワークに接続するために、M300にはRJ45ポートが装備されています。 ハードウェアリセットボタンも忘れられません。



構成と更新は、Webインターフェイスを介してネットワーク経由で実行されます。 M300には組み込みのサブスクライバーブックはありません（M700には-200サブスクライバー用）。



ステーションは、必要なすべてのプロトコルをサポートしています。



  • SIP（RFC3261）
  • DHCP、NTP
  • HTTP / HTTPS / TFTP
  • LDAP


サポートされているコーデックのリストは、M700よりもさらに大きくなっています。



  • G711 A-law、µlaw
  • G722（ブロードバンド）
  • G726
  • G729（VADを含むA / AB）


M300のセキュリティは、ワイヤレスDECT暗号化、TLS、SRTP（RFC3711）、SIPS、およびステーション管理Webインターフェイスのパスワード保護によってサポートされています。 Snom M300は、BroadsoftやAsteriskに基づくシステムを含む、すべての主要なIP PBXシステムと互換性があります。



ステーションは5 V電源で駆動され、その全体の寸法は178 x 120 x 30 mmです。







また、このステーションはM325キットで購入できます。これには、M300自体に加えて、Snom M25 DECT電話が含まれています。 Mシリーズのデバイスに特化した以前の出版物の1つで詳細に説明されています。



Snom M215



このキットは、Snom M200SCベースステーションとM15SC DETCチューブで構成されています。







M200の機能は、M300よりも控えめです。 最大6台のDECTハンドセットを同時に接続でき、同じ数のSIPアカウントをサポートします。 ステーションは、最大4つの同時通話を処理できます。



M200SCの範囲は標準で、理想的な条件で最大50 mです。 電源とPoEの両方から給電できます。







フロントパネルには、DECTハンドセットを検索するためのボタンがあります。クリックすると、電話が鳴り始めます。









背面には、10/100 BASE-Tネットワークに接続するためのRJ45コネクタがあります。









M200SCには、200個の番号を含む電話帳が組み込まれています。これは、通話ログのエントリ数と同じです。 デバイスは、パスワードで保護されたWebインターフェイスを介して、ネットワーク上で完全に構成されます。 ステーションは、ワイヤレスファームウェアの更新をサポートし、内部通信（ローカルネットワーク内）と3者間会議通話の機能を備えています。 通話を転送および保留できます。



M200SCはGAPデバイスおよびAsteriskプラットフォームと互換性があります。 M200の音質は、より高度なM300でサポートされているオーディオコーデックの音質よりも悪くありません。



  • G.711 a-law、u-law
  • G.722
  • G.726
  • G.729
  • ILBC


サポートされている通信プロトコルのリスト：



  • SIP（RFC3261）
  • HTTP / HTTPS / TFTP / FTP
  • DHCP、NTP、SNTP、STUN、VLAN
  • RTP / SRTP
  • IPv4 / IPv6
  • LDAP


セキュリティも「先入観」ではありません。200SCは、TLS、AES暗号化、LLDP-MED、802.1x、802.1pqなどのセキュリティツール全体をサポートしています。



M215に含まれているM15SCは、最も単純なSnom製品の1つでした。 これは信頼性の高い「主力製品」であり、あらゆる組織、さらには自宅での使用にも適しています。









この電話機には、対角1.7インチ、解像度90 x 65ピクセルのディスプレイが搭載されています。 通常、上の行は情報アイコンを配置するために使用され、下の行はディスプレイのすぐ下にある2つのファンクションキーの値の表示の下に表示されます。









キーパッドにはバックライトが装備されており、電話機にはボイスメールの受信時に点灯するLEDも装備されています。 電話機は着信の転送と保留をサポートしており、10個の番号を短縮ダイヤルに割り当てることができます。







背面には、ベルトに電話を掛けるための取り外し可能なクリップがあります。 右側には、ゴム製のプラグで覆われた2.5 mmのヘッドフォンジャックがあります。







バッテリーは、ニッケル水素電池で、容量は550 mAhです。 これは、1週間のスタンバイ時間と最大7時間の会話に十分です。































電話の全体寸法は174 x 49 x 31 mmです。
