友人と私がサードパーティのプロジェクトを月額17,000ドルの収入でビジネスに送り込んだ方法

2014年に、私の友人と私は、できるだけクールなWebデザインツールを作成することにしました。 UIキット 、 管理ダッシュボード 、 テンプレート 、 プラグインを作成しました。



私たちは常に、開発プロセスに役立つWeb製品を作成することを夢見ていたので、それを使用してクライアントサイトを作成できました。







収益の面では、ブラックフライデーをその販売で受け取らなかった場合（2016年11月の利益を2倍にした）、1か月あたりの収益は約22,000ドルになりました。 その一部は、アフィリエイトプログラム、VAT、サプライヤー税、その他の費用のコミッションを支払うことになります。 その結果、現在、月間純収入は約17,000ドルになっています。

今日、私たちがどのように私たちの製品を作成し、この副業がどのように促進されたかを共有したいと思います。



次のトピックについて説明します。



  • 私たちのスタートアップであるCreative Timに取り組み始めたきっかけ、そして元の製品を作成した方法。
  • 最初の顧客を獲得した方法。
  • 成長に使用したマーケティング戦略。
  • ビジネスモデルの仕組み。
  • 収入源の秘密;
  • 私たちが学んだ最も重要な教訓。






Creative Timの開発を始めたきっかけ、および元の製品の作成方法



私たちは、外部投資なしでルーマニアの2人の代理店としてスタートしました。 私たちは仕事用の部屋を借りる手段がありませんでした-集合オフィスのデスクだけでさえも-スターバックスで働きました。 顧客の注文を満たすために、私たちは日々のニーズを満たすためにお金を見つけることができませんでした。



Creative Timはサードパーティのプロジェクトであり、私たちのような開発者に役立つと確信しています。 クライアントと協力するとき、私たちは常に「車輪を再発明」し、何度も何度も同じ要素をサイトに使用していることに気付きました。 そのため、モーダルログイン/登録ウィンドウ、カレンダー、ヘッダー、ヒント、フッターなど、いくつかの標準コンポーネントを作成する必要がありました。



数か月間、私たちはプラットフォームといくつかの無料のルアーの実装に時間を費やしました（これはすべて主要な作業の背景です）。 当初は、Twitterのフォロワーも、Facebookのファンも、メール購読者もいませんでした。 私たちはデザイナー向けのフォーラムに無料オファーに関する多くの情報を投稿し、「TwitterでのWeb開発者への取り組み」手法を使用して、製品に関する情報を人々に知らせ、共有を開始しました。







最初のユーザーをどのように惹きつけたか/獲得したか

まず第一に、誰も私たちが望むものを本当に理解していませんでした。 彼らは私たちのアイデアの完全な価値と、ビジネスの発展のために私たちが提供できるものを理解していませんでした。 私たちは、人々が私たちがしていることを理解するのを助けることができる、より包括的な製品を作成する方が良いと判断しました。



ブートストラップベースのUIキットであるGet Shit Done Kitを開始しました。 このイベントはDesigner Newsの Webサイトで取り上げられており、製品の評判が良くなりました。 このリソースから11,000人のユーザーを獲得しました。これは、当社のビジネスを大きくサポートするものになりました。



2週間後、スタートアップがProduct Huntに登場しました。 これにより、5,000人以上のユーザーを獲得しました。 その後、状況は安定しました。同じ期間に「週あたりのユーザー数が0」から一定の2〜3 000になりました。







無料の製品Get Shit Done Kitの成功に触発された数か月後、 Get Shit Done Kit PROをリリースしました。これは、多数のコンポーネントとターンキーWebサイトページを備えたGSDKのプレミアムバージョンです。



当初は、わずかな売上しかありませんでした。 製品は週に約200ドルをもたらしましたが、これは私たちのビジネスをサポートするのに十分ではありませんでした。 同時に、私たちはクライアントの1人のWebプロジェクトに取り組んでいました。



12月に、Bootstrapを使用して作成されたサイトをデモするための最も人気のあるギャラリーであるBootstrap Expoで公開しました。 これは、当社の事業発展のもう1つの推進力でした。 インスピレーションを求めてギャラリーを訪れた人は誰でも、Bootstrapが何であるか、またはそれと一緒に働いていたことをすでに知っていたので、彼らは私たちのビジネスの理想的なターゲットオーディエンスになりました。



後で、サイトへのトラフィックを取得するだけではユーザーとの長期的な関係を構築するのに十分ではないという事実を発見し、ほとんどのユーザーは最初のやり取りの後に存在を忘れていました。 私たちはいくつかの調査を行い、ほとんどのマーケティング担当者が長い間知っていることを発見しました。人々は物事を忘れ、
彼らはそれについて思い出さない。
最近、重要な情報を記憶に残すために作成されたインターバルメモリ技術に関する記事の翻訳を公開しました。



そのため、 忘却曲線のルールに従って電子メール通知システムを実装しました。







私たちはユーザーに私たちが存在することを思い出させ、彼らのプロジェクトやクライアントのプロジェクトに取り組むための便利なツールを提供したかったのです。



現時点では、次のスケジュールに従ってレターを送信しています。



  • 最初のダウンロードから3日が経過した後、他の製品を推奨する手紙を送信します。
  • 10日目に、フィードバックのリクエストと、質問がある場合に役立つ提案を記載したメールを送信します。
  • 15日目に、製品をPROバージョンにアップグレードできることをお知らせします。
  • 30日目に、私たちは再びフィードバックを求め、ギャラリーまたはソーシャルネットワークでユーザーの製品を宣伝する申し出を行います。
  • 最終通知は60日目に送信します。

ビジネスモデルの仕組み



Creative Timの最適なビジネスモデルはフリーミアムモデルであることに気付きました。Web開発者が優れたWebサイトを作成するのに役立つ高品質の景品を作成し、その後、この景品に要素、セクション、モジュール、既製ページを含むPROバージョンを実装します「ターンキー」。



現在、8つのプレミアム製品があり、それぞれに独自の無料パンがあります。 価格は、ライセンスとアーカイブの種類（HTML、HTML + PSD、HTML + Sketch）に応じて、19ドルから599ドルの範囲です。 景品はどこにでもあります-さまざまなコミュニティ、ブログ、ニュースレター、ソーシャルWebサイトなど、すべてのトラフィックを促進します。



当社のビジネスモデル：Web開発者が優れたサイトを作成するのに役立つ高品質の景品を作成し、この景品用のPROバージョンを実装します。



主なアイデアは、これらのパンが常に上位10の大きなコミュニティに表示されるということです。 トップ10にある各投稿（コミュニティの規模によって異なります）は、1日あたり1,000〜15,000人のターゲットユーザーを対象としています。 定期的なターゲットを絞ったマーケティングキャンペーンを実施する場合、費用がいくらになるか想像できますか？

ビッグバンドル



ビッグバンドルとは何ですか。どのように作成しましたか？ 一部のユーザーが無料の製品をすべてダウンロードしていることに気付きました。 私がすべてを言うとき、私は文字通りすべての製品を意味します。彼らはアカウントを作成した後、約2〜3分間すべてをダウンロードしました。 また、一部の顧客がすべてのプレミアム製品を購入していることに気付きました。



そのため、「Big Bundle」という新しい製品をテストすることにしました。これにより、すべての製品に大幅な割引（60％以上）でアクセスできます。 この大きなパッケージは、月に約6〜8回購入されました。 Big Bundleの価格は個人使用で299ドル（500ドルの代わりに）、開発者ライセンスで669ドル（2127ドルの代わりに）であるため、これは良い収入源であり、多くのプロジェクトで当社製品を使用するWebデザイナーや代理店にとって大きな助けとなります。 これはwin-winオプションです。
