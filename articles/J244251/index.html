インテル®System Studio 2015-組み込みソフトウェア開発者が必要とするすべてのもの

インテリジェントデバイスと「スマートモノ」は、私たちの周りの世界に急速に広がり、より高度になり、相互に作用しています。 遅れを取りたくないので、Intelは組み込みシステム用ソフトウェアの開発者のニーズに多くの注意を払っています。 仕事の生産性を高めるために、同社は組み込みシステムの開発、デバッグ、テスト用のクロスプラットフォームツールセットであるIntel System Studio 2015を準備しました。

次のバージョンでは、何が新しいものを待っていますか？



バージョン2015以降、Intel System Studioは、Intelの他のStudioと同様に、Intel System Studio Ultimate Edition （以前はJTAGを備えたIntel System Studioと呼ばれていました）、Intel System Studio Professional Edition （以前は単にIntel System Studio）、そして原則として、新しいIntel System Studio Composer Edition 。

各ターゲットプラットフォームの各製品のコンポーネントの完全なリストを次の図に示します（クリック可能）。







2015バージョンの主な基本的な革新に注意してください。

  • ターゲットシステムとしてWindowsのサポートを追加
  • Studio for VxWorksのIDEとしてのWind River Workbench統合の改善
  • Intel System Debuggerの拡張機能


製品の詳細については、 Intel WebサイトのIntel System Studio 2015セクションを参照してください。
