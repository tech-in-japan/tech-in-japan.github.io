ハッカソンby ABBYY

前回、アイデアのコンテストを発表しました （そして、ちなみに、引き続きiPhone Xを勝ち取ることができます）。そして、ABBYYのモバイルサービスのハッカソンに招待します。 10月7〜8日にPhysTechParkで開催されます。 方向は非常に異なります。 クールな審査員。 賞金は220,000ルーブルです。 申し込みは、 mobility.abbyy.com / hackで10月3日まで受け付けています。

そして、詳細は以下です。



MABBYYlity Hackathonは、インテリジェントな情報処理技術を使用したモバイルプラットフォーム向けのサービスを開発するための24時間の競争です。 私たちは、優れた開発者やメンターとともにユニークな製品を開発する機会を与えたいと考えています。また、大規模なIT企業の代表者に自分自身を証明し、素敵な贈り物をもらうチャンスです。



mABBYYlityの参加者は、2〜5人のチームで24時間、または独立して、ビジネスアイデアを思いつくか開発し、テキスト、画像、ドキュメント、フォームの処理分野のモバイルデバイス用のアプリケーションまたはサービスのプロトタイプを作成する必要があります。 これらは、次の分野の解決策になる可能性があります。



•ボット、ソーシャルネットワークのサービス、インスタントメッセンジャー。

•モバイルデバイスの拡張現実。

•ワープロ、ドキュメント分類、検索。

•画像とビデオの認識と処理。

•写真の改善、追加、フィルタリング。

•ブラウザーでの認識と画像処理。

•さまざまなデバイスのカメラでの認識。

•デジタル企業、生産性の向上。

•安全技術。

•クラウドテクノロジーとサービス。

•情報交換テクノロジー。



ABBYYは、すべての参加者に開発者向けの独自の製品へのアクセスを提供します。 さらに、チームは公開されている開発ツールを使用できます。

ハッカソンの結果によると、最高のモバイル製品は専門家審査員によって選択されます。 含まれるもの：



•ABBYYグループの製品Andrey Isaevの製品ディレクター。

•ABBYYモバイル製品部長、 イゴールアキモフ 。

•教育プロジェクト部門ABBYY Andrey Ocheretnyの長 。

•ZeptoLabプロジェクトオフィスピーターセートリエフ長 。

•CEO Maps.Me Evgeny Lisovsky ;

•Google Developer Manager Zviad Kardava ;

• Anatoly Stoyanovsky 、Sberbankのデジタルリソースエコシステム担当エグゼクティブディレクター。

•Yandex.Taxi機械学習チームの長、 Victor Kantor 。

•Binbank Digital、モバイルサービス開発部長、 Anastasia Belchenko 。



専門家は、次の基準に従ってチームの結果を評価します：アイデアの市場の可能性と解決される問題の関連性、ハッカソン中の製品の技術的な実装とチームの進捗、製品アイデアとイベントの方向性の対応、アイデアの新規性と独創性の程度。



詳細については、 mobility.abbyy.com / hackをご覧ください。
