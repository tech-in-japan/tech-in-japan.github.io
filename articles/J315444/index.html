開発者ツールとDevOps



まず、Visual Studioファミリの更新とラインの新製品について少し説明します。



Visual Studio 2017リリース候補 -新しいバージョンのツールのリリース候補は現在入手可能です 。 Visual Studio 2017 RCには、以前のバージョンでは利用できなかった修正と新機能の両方が含まれています。 また、このリリースのVisual Studio 2017 RCには、実稼働ソリューションを開発する機能が含まれています。



Team Foundation Server 2017 RTM-今日、チーム間の相互作用を保証するためのメインツールの新しいバージョンも利用可能になりました。 Team Foundation Server 2017には 、パッケージ管理、コード検索、テストとフィードバックの拡張、リリース管理の改善などの新機能が追加されました。 さらに、TFS 2017ユーザーはVisual Studio Marketplaceから拡張機能をダウンロードし、TFS環境にインストールできます。 詳細については、Brown Harryブログをご覧ください。



モバイル開発者向けの主なニュースは、 Visual Studio Mobile Center Previewの発表です。

Mobile Centerは、Objective-C、Swift、Java、Xamarin、React Nativeで記述されたiOSおよびAndroidアプリケーションと連携し、クラウドサービスとツールを統合して、高品質のモバイルアプリケーションの迅速な開発を目標とする開発者にアプリケーション開発ライフサイクルを提供します！ Mobile Centerは、デバイス上のアプリケーションの開発、アセンブリ、テスト、監視、および展開の機会を提供し、開発者がすべてのプラットフォームでアプリケーションをスケーリングし、最新で最も人気のある機能を実装しやすくします。 プレビューバージョンにアクセスするには、Visual Studio Mobile Centerページに移動して登録します。



そして最後に、今日Connect（）カンファレンスで待っていたもの。 // 2016年、Visual Studio for Macのプレビューバージョンが発表されました 。 本日からダウンロードできます！ Visual Studio for Macは、モバイルアプリ、クラウドサービス、およびMacアプリを作成するために設計された完全な開発環境です。 最初のプレビューリリースには、Xamarinを使用してC＃およびF＃でネイティブiOSおよびAndroidアプリケーションを作成するサポート、およびAzureサービスを使用してバックエンドを作成するサポート、そしてもちろんASP.NET Coreのサポートが含まれます。







さらに、マイクロサービスアプリケーションアーキテクチャ用のVisual Studio Team Servicesでの開発、展開、DevOpsの大幅な簡素化と同様に、Visual StudioとDockerを操作するためのツールとの緊密な統合を実証しました。

Windows開発



今日、Windowsは従来のデスクトップアプリケーションの開発だけでなく、通常の自然なデータ入力方法（音声、鉛筆、感情など）を通じてユーザーと対話できるソリューションを作成する機能です。 今日のWindowsは、コンピューターユーザーとコンソールユーザーのゲーム体験の境界を打ち破り、新しい仮想現実または拡張現実に移行しています。



接続時（）; 開発者向けのWindowsの新機能を紹介しました。



Windowsでは 、作成するアプリケーション（ユニバーサルWindowsプラットフォーム（UWP）またはクラシックWindowsプラットフォーム（WPF、Win32、VB6など））に関係なく、Windowsアプリケーションインターフェイスの自動テスト用にAppiumフレームワークのネイティブサポートが追加されました 。 また、WindowsはWinAppDriverサービスを使用してテストできるため、アプリケーションは標準の画面ツールからも利用できます。 Windows Application Driver Serviceの詳細については、 https：//github.com/Microsoft/WinAppDriverにアクセスしてください。



Desktop Bridgeにより、従来のデスクトップWindowsアプリケーションは、アプリケーションコードを変更することなく、新しいWindows 10を活用できます。 デスクトップアプリケーションコンバーター（DAC）を使用すると、ゲーム開発者と作成者は、Windowsストアを通じて現在のソリューションを配布できます。 Windowsデスクトップブリッジの詳細と機能の説明は、 https：//developer.microsoft.com/en-us/windows/bridges/desktopで入手できます 。 https://www.microsoft.com/en-us/store/collections/fullsoftwareapplications/pcの 「強力なPCソフトウェア」カテゴリのWindowsストアでこのツールを使用して公開されたアプリケーションの例を見ることができます。
