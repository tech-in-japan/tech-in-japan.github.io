3GルーターHuawei E960の概要

勤勉な中国の同志によって作られたものから私の手に落ちた機器の最高の例の1つである3G Huawei E960ルーターについてお話したいと思います。





残念ながら、無料販売では見られず、3Gテクノロジーを使用してインターネットアクセスを提供するモバイルオペレーター専用のブランドです。 ウクライナのUtelのロゴが付いています（写真は手元にきちんとした写真機材がないため私のものではありません）。



HUAWEI E960仕様：

  • 単一標準UMTS-2100 MHz;
  • 3つの標準GSM / GPRS / EDGE-900/1800/1900 MHz;
  • 3Gネットワ​​ークでの実際のデータ転送速度：3.6 / 7.2 Mbit / sのダウンロードと384 Kbit / sのアップロード;
  • イーサネット、Wi-Fi、USB 2.0を介してコンピューターに接続します。
  • ボイスチャットのサポート：電話用のRJ-11ポート。
  • 動作モード：モデム、ゲートウェイ。
  • アンテナ：内蔵;
  • 重量：500g未満。
  • オプション：ルーター、USBケーブル、パッチコード、ドライバーと説明書付きのCD。


Broadcom BCM5354チップセットと、内部USBで接続された未知のHDSPAモデム（おそらくHuaweiのネイティブなモデム）に基づいています。

デバイスは、Wi-Fiおよび4ポートイーサネットスイッチを備えた通常のスタンドアロンルーターとして動作できます。 WANポートだけではありません-インターネットはモバイルネットワーク経由でのみ提供されます。 電源は通常のUSBタイプBコネクター（プリンター/スキャナーなど）に接続されますが、E960を標準ケーブルでコンピューターのUSBポートに接続すると、ルーターは通常のUSB 3Gモデムに変わります。 もちろん、スイッチとWi-Fiはこのモードでは動作しません...トーンダイヤル機能を備えた通常のアナログ電話セットが接続されているRJ-11ポートもあり、UMTS / GSM音声を使用して完全に通信できます-この機能はこのデバイスでのみ使用できます。 ボーナスとして、このモデムの一部のファームウェア（ただし、残念ながらUtelovskayaではありません）では、Webインターフェイスを介してSMSを送受信できます。



Webインターフェイス自体は中国語では少し不器用ですが、期待どおりにルーターに必要なすべての機能を実行します。





モバイルネットワーク設定が構成されています（PINエントリ、プロバイダーとのPPP接続プロファイル）、DHCPサーバーおよびWi-Fiアクセスポイント（WEP、WPA-PSK、WPA2-PSK、TKIPおよびAES、ブリッジモードをサポート）：





セキュリティ設定では、ファイアウォール、MAC / IPフィルター、ローカルネットワークへのポート転送、DMZ（WANからローカルネットワーク上のコンピューターの1つへのすべての着信接続の転送）が構成され、UPnPポートマッピングもサポートされます。





telnet経由でルーターにアクセスできます。ユーザー名/パスワードは標準です-admin / admin（およびWebインターフェイスのパスワードに関係なく）：



作業環境は非常に細分化されていますが、少し掘ることが可能です。 ところで、コンソールでは、Webインターフェイスのパスワードを簡単に回復できます。

# nvram show | grep pass

user_passwd=user

admin_passwd=admin

結論


モバイルオフィスを迅速に展開するための優れた装置であることが判明し、インターネットの他に電話回線も利用できますが、残念ながら音声専用です。アナログファックスはモバイルネットワークでは送信されません。 内蔵アンテナにもかかわらず、信号を非常によくキャッチし、1つまたは2つの「スティック」でも安定した1-2 Mb / sを提供します。

また、Ciscoのバックアップチャネルとして正常に機能し、NATルータを介してDMVPNトンネルを着実に上昇させます。

私は本当に落ちるのが好きではありません。窓辺をノックオフして数回コピーすると、内蔵モデムが見えなくなり、簡単な分解/アセンブリで修復されました-連絡先が見えました...ところで、内部アセンブリは非常に高品質で、ほとんどすべてがスクリーンに包まれています
