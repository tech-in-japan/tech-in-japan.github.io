仮想マシンテンプレートの更新をインストールする

仮想マシンのインフラストラクチャ（特にVDIシナリオ）を計画する際に注意する重要なポイントは、オペレーティングシステムおよびウイルス対策データベースへのインストールされた更新に関して、仮想マシンテンプレートを最新に保つ必要があることです。



この問題を解決するためのオプションの1つ Hyper-Vプラットフォームの場合、最近更新されたVirtual Machine Servicing Tool 3.0です（ http://www.microsoft.com/downloads/en/details.aspx?FamilyID=8408ecf5-7afe-47ec-a697-eb433027df73からダウンロードできます ） 。



ユーティリティが機能するには、Hyper-VプラットフォームでMicrosoft System Center Virtual Machine Manager 2008または2008 R2が実行されている必要があります。 更新は、次のようにインストールできます。

Windows Server Update Services 3.0 SP1、3.0 SP2、およびSystem Center Configuration Manager 2007、2007 SP1、2007 SP2、2007 R2



ユーティリティは以下を更新できます。
  • 仮想マシンのテンプレート
  • Hyper-Vホストで仮想マシンを直接シャットダウンおよび停止します
  • SCVMMライブラリに保存されている仮想マシンと個別の仮想ディスク
申し込むと、情報セキュリティ部門があなたを責めることはありません。
