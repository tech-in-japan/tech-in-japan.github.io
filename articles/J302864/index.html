仮想フォーラム「データのオープニングの記録を参照してください。 テクノロジー。 SQL Server 2016»





フォーラムプログラムは3つの技術セッションに分かれており、並行して行われました。



SQL Server 2016：OLTP世界の新しい標準 。 このトラックは、生産性とプロセスの安全性を高めるために、新しいプラットフォームの機能に専念しています。 したがって、新しいプラットフォームでのトランザクション処理速度は30倍速くなり、クエリ処理速度はディスクリレーショナルデータベースに比べて100倍速くなります。 システムのセキュリティは、常に暗号化された最新のツールとロールレベルセキュリティによって提供されます。



ビジネスインテリジェンス：SQL、Power BI、R、モバイル 。 データを分析し、BIビジネスインテリジェンステクノロジとR言語を使用してスマートシステムに整理できるインテリジェントアプリケーションを開発するための、SQL Server 2016の実用的なアプリケーションの説明。



Azure：新世代の分析ソリューション、ビッグデータとIoT 。 スマート分析システム、機械学習、ボット、およびITインフラストラクチャを拡張する機能を備えたクラウドへの統合のおかげで、会社の重要なプロセスを確実に自動化することに関するレポート。



仮想フォーラムは、最新のSQL Server 2016を含む、データ処理の主な傾向とマイクロソフト製品で示された革新についての講演で始まりました。
