ロシア連邦の裁判所は、著者の許可なしに他の人の写真を公開することを許可しましたか？ そうではない[更新済み]



イリヤ・バルラモフによる記事「 なぜアメリカにはこんなに多くのホームレスの人がいるのですか？」 「。 ポータル66.ruが行ったように、著者の許可なしに投稿されました。これはイリヤヴァルラモフの訴訟の被告です。 写真：Ilya Varlamov // varlamov.ru



ロシア連邦知的財産権裁判所は 、国のメディアの活動に影響を与える可能性のある歴史的な決定を下しました。 少なくとも、ポータル66.ruの代表者は、起業家イリヤ・バルラモフを訴えていると考えています 。



ポータル66.ruに対するhabrauserと写真家Ilya Varlamov（ zyalt ）による訴訟で、裁判所は、第一例と第二例の裁判所の判決を覆し、写真の違法使用に対して補償を支払うことを決定しました。 ポータル66.ruの代表者は、次の5つの条件に従って、著者に許可を求めることなく情報写真を公開する権利があると考えています。



  1. 写真は、情報の性質、つまり、現在の経済的、政治的、社会的、宗教的な問題を扱っています。
  2. 著者の名前が表示されます。
  3. 借入元が表示されます。
  4. 著作権所有者は写真をパブリックドメインに投稿しました。
  5. 写真の使用に関する特定の禁止事項はありません。






決定のテキスト （略称）から：

IP Varlamov Ilya Alexandrovichは、スベルドロフスク地方の仲裁裁判所に、LLC 66.RUに対する請求書を提出し、460,000ルーブルの補償を取り戻しました。 違法な複製およびインターネット44 44写真作品の一般公開、ならびに90,000ルーブルの補償。 インターネット上の9つの写真作品の著者に関する情報を削除したため。



2016年2月25日付けのスベルドロフスク地域仲裁裁判所の決定により（算術エラーの修正に関する2016年2月25日の決定を考慮に入れて）、請求は完全に満たされました。



2016年6月6日付の第17仲裁裁判所の決議により、審理裁判所の決定が変更され、訴訟は部分的に満たされました。220,000ルーブルの補償がビジネスマンに有利な会社「66.RU」から回収されました。



知的財産権裁判所に提出された死刑控訴において、起業家は控訴裁判所の決定を取り消し、第一審裁判所の決定を取り消すことを求めます。



裁判所は以下を裁定した。



スベルドロフスク地方の仲裁裁判所の判決は、事件番号A60-54898 / 2015の2016年2月25日であり、同じ事件の2016年6月6日付の第17仲裁裁判所の判決は取り消されました。



事件は、新しい裁判のためにスヴェルドロフスク地方の仲裁裁判所に送られるべきです。


ご覧のとおり、イリヤは、仲裁裁判所の決定による少額の補償に不満であり、死刑控訴を提出しました。 その結果、ロシア連邦の知的財産権裁判所は、 両裁判所の判決を破棄しました 。



第17仲裁裁判所の判決に対する控訴は、Justa Aura法律事務所から非常に有能な弁護士を雇ったポータル66.RU LLCによっても提出されたことは注目に値します。



この会社の弁護士は、非標準の防衛線を選択しました 。彼らは、LLC 66.ruが写真の公開時点で法人としてまだ存在していなかったという事実に依存していました。 また、当時、VarlamovはIPとして登録されていませんでした。 これは、この事件は一般に経済的な性質のものではなく、最初から仲裁裁判所で考慮されるべきではないことを意味します。



裁判所命令のテキストのより詳細な解釈については、 mitasamodelの 説明を参照してください。 彼は、2つの苦情が拒否された理由と、ケースが審査のために送られた理由を非常に簡単に説明しました。



決定の全文





前例？



知的財産権裁判所は、2013年に設立されたロシア連邦で最初の専門仲裁裁判所です。 彼は、知的財産権の保護に関連する紛争の第一審およびass審事件の裁判所と考えています。



ロシアには判例法制度はありませんが、通常の仲裁事件で同様の事件を検討する場合、知的財産権裁判所の決定が考慮される可能性が最も高いでしょう。 現在、裁判所は、合計1,190万ルーブルの34のロシアのマスコミに対するイリヤヴァルラモフの主張のみを検討しています。 2016年8月、ロシアの地域メディア企業約10社が＃StopVarlamov委員会を設立し 、イリヤの写真の無料使用を許可し、起業家のビジネスに終止符を打つことを要求しました。



イリヤ・ヴァルラモフは 、ロシアの都市に関する一連の出版物の著者として知られているため、ヴァルラモフに反対する地域メディアです。 彼のインターネットへの投稿は議論を白熱させ、写真は地域のオンラインメディアによって広く配布されています。



写真家によると、この先例はニュースメディアや代理店によって写真を盗むという危険な慣行を作り出す可能性がある。



ロシア連邦の知的財産権裁判所の決定を条件として、事件は仲裁裁判所によって審査されるため、この問題を終わらせるのは時期尚早です。
