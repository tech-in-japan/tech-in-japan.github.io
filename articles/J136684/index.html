理論的紹介



Webサービステクノロジーを使用して、ますます多くの交換が行われています。 ホストデータベースにはWebサービス拡張機能とデータハンドラーをインストールするだけで十分であり、ClintソリューションではWebサービスに接続できるため、これは便利です。 このテクノロジーはマルチプラットフォームであり、現時点ではその使用は非常に重要です。

WSDL （Webサービス記述言語）は、Webサービスを記述してアクセスするためのXMLベースの言語です。 （仕様http://www.w3.org/TR/wsdl ）。



SOAP （Simple Object Access Protocol）は、分散コンピューティング環境の構造化メッセージングプロトコルです。 SOAPは、XML形式の任意のメッセージとリモートプロシージャコール（RPC）の交換に使用されます。 （仕様http://www.w3.org/TR/soap/ ）。



XDTO （XMLデータ転送オブジェクト） -XMLスキーマを使用して記述されたデータのオブジェクトモデリングのメカニズム（詳細については、 http ： //v8.1c.ru/overview/xdto.htmを参照）。



プリミティブ型をシリアル化するためのほとんどのメソッドはW3によって開発され、名前空間「 http://www.w3.org/2001/XMLSchema 」で説明されています。
