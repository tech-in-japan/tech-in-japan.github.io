Google Playデベロッパーをブロックする理由





この記事の主な目的は、アプリケーションと開発者アカウントをブロックするための多くの前例を1か所に集めることです。 各ケースについて、原因と結果を詳細に分析します。



トピックは非常に深刻です。 Googleが一度あなたをブロックすると、戻る方法はありません。 どうやら、アカウントとあなたの名前、IP、Macアドレス、その他の個人データとの関係が追跡されているようです。 Googleとの関係を一度損なう-決して流されることはありません 。



個人的に登録されたアカウントを失わないために、特定の原則に従う必要があります。 それらを一緒に引き出そうとしましょう。



私はすぐに言います-個人的に、私のアカウントとアプリケーションはまだブロックされていません。 しかし、他のユーザーのブロックに関する記事を読んで、私は恐れ始めました。 フォーラムで質問してください。 そして、恐怖は無駄ではありませんでした。



あなたやあなたの友人がアプリケーション/アカウントをブロックする場合があります-あなたはコメントや個人的なメッセージでそれらを説明することができます-私は喜んでそれらをリストに追加します。 ケースが多いほど、記事の有用性が高まります。 そして、できるだけ多くの情報、リンク、スクリーンショットを指定してください。



事例

テトリス

結論：開発者は小さなゲームを作成し、その名前でテトリスという言葉を使用しました。 この言葉は、あなたが支払わなければならない登録商標です。 原則として、これは論理的なことです。かなり多くのユーザーがGoogle Playで「テトリス」という言葉を詰まらせているため、アプリケーションはすぐに多くの訪問とインストールを受け取ります。 アプリケーションがブロックされています。

ブロックする理由：著作権侵害（テトリスという言葉の使用）

証明： http : //javatalks.ru/topics/42335

結論：安全。 開発者は、同じアプリケーションを異なる名前でリロードしました。 アカウントはそのままで、アプリケーションはまだ機能しています。



アニメを推測する

結論：開発者は、著作権を所有していないキャラクターの写真があるアプリケーションを作成しました。 アプリケーションがブロックされています。

ブロックする理由：著作権侵害（アニメ画像を使用）

証明： http : //toster.ru/q/101383（danil031997からの回答）

結果：新しい銀行カード、新しい口座などを登録する必要がありました。 ユーザーは、古いプロジェクトを他の場所にアップロードしないことに決めました。



ブラックSEO

結論：一部のアカウントには、約20のアプリケーションが含まれています（ダウンロードの合計数〜500k）。 それらは不均一に分布したキーワードでぎっしり詰められていました。 Googleからの警告が7つ以上出された後、開発者アカウントは禁止されました。

ブロックする理由：検索スパム

証明： http : //toster.ru/q/55160

結論：新しいアカウントが登録されましたが、古いアプリケーションが読み込まれたために禁止されました。



エロコンテンツ

結論：開発者は、Android用のメディアプレーヤーを作成しました。 すべてがうまくいくだろうが、githubの彼のスクリーンショットの中にはエロティックなコンテンツがあった 。 その記事の出版時点で、彼は本当にそこにいたと私はチェックした。 アプリケーションがブロックされています。

ブロックする理由：エロティックなコンテンツ

証明： http : //habrahabr.ru/company/freeamp/blog/230293

結論：性的なコンテンツにアプリケーションを関連付けないでください。 インターネット上の任意の場所にアップロードするすべての画像を慎重に制御します。 特にgithubで。



生命の禁止

結論：開発者は米国市民であり、サンフランシスコに住んでいます。 彼の投稿から判断すると、彼はiOSからAndroidに切り替え、かなりのお金を後者に投資してから、息子のために無害な一連のアプリケーションを作成しました。 これらのアプリケーションは特定のYouTubeチャンネルをキャプチャし、その動画のみを視聴できるようにしました。 アプリケーションヘッダーにはチャネル名が含まれ、その一部は非常に人気がありました。 合計10個のこのようなアプリケーションが作成されました。 それらの2つがブロックされました。

ブロックする理由：検索スパム

証明： https : //medium.com/@sgehrman/banned-for-life-c62f2404f66

結論：ブロックした後、ユーザーはサポートに書き込もうとしましたが、それに応じて、彼のアカウントは一生禁止されました。 注意してください-彼は米国市民です。 Googleは当初、Googleだけが厳しく禁止されていると考えていました。 正義がまだ存在することが判明しました。



ゆるいフラッパー

結論：開発者はゆるい鳥のクローンを作成し、Flappy Flapperという名前を付けました。 それは店に現れましたが、それから一時停止状態に移されました。 そして、同じアプリケーションを別の名前でリロードしようとした後、両方がブロックされました。

ブロックする理由：検索スパム

証明： http : //habrahabr.ru/post/231009/#comment_7813421

結論：安全。 開発者アカウントは削除されていません。



Ad-Senseクリック

結論：開発者はクリックをだました。 システムはこれを特定でき、Ad-Senseのアカウントをブロックしました。

ブロックする理由：チート

証明： http : //habrahabr.ru/post/231009/#comment_7813399



スパイを呼び出す

結論：開発者は、電話での会話を記録するアプリケーションを作成しました。 最初のバージョンはプロセスでは見えませんでした。 禁止された。 2番目のバージョンには、通知領域に既に永続的なアイコンがありました。 さらに、このアプリケーションはインターネットを必要とせず、無料で、インストール数は約70万、評価は4.5以上でした。

ブロックする理由：スパイウェア

証明： http : //habrahabr.ru/post/231009/#comment_7813583

結論：スパイウェアの配布のためにアカウントは永久に閉鎖されました。 控訴は決定的ではなかった。 そして、私は認めなければなりません-名前にスパイを含むアプリケーションはいっぱいです。 会話の録音を含みます。



説明のキーワード

結論：開発者は、説明に明示的に追加されたキーワードのリストを外国のアプリケーションで見て、自分のアプリケーションでも同じことをすることにしました。 スパムに関する警告と、それを修正するための7日間を直ちに受け取りました。

ブロックする理由：検索スパム

証明： http : //habrahabr.ru/post/231009/#comment_7813751

結論：キーワードを削除しました。 すべてがうまくいきました。 ちなみに、サポートは、他のアプリケーションが禁止されない理由の質問には答えず、具体的には禁止されました。



オンラインの映画。 プレミア

結論：開発者は、映画をオンラインで視聴するためのアプリケーションを作成し、Google Playに投稿しました。 そこで彼は繰り返し禁止され、自動禁止に至りました。

ブロックの理由：著作権侵害（海賊版コンテンツ-VKontakteグループの映画を撮影）

証明： http : //w3bsit3-dns.com/forum/index.php? showtopic = 533636&st = 240# entry32678043

結論：新しい開発者、新しいマップ、VPNが毎回存在するという事実にもかかわらず、アプリケーションは公開されていません。 新しいロゴ、名前、スクリーンショット、説明、apk変更パッケージ、難読化。 彼らは、アプリケーションに埋め込むのではなく、ユーザーがコンテンツソースを選択する機能を追加するという考えを表明しました。 アイデアはちなみに良いですが、開発者は今のところそれを適用していません。



イノベーションのリストをアプリケーションに追加する

結論：開発者（AndroidおよびiOS）は、ロシア政府から委託されたソーシャルアプリケーションの1つを2つのプラットフォーム向けに作成しました。 アプリケーションのテーマ：問題（住宅、公衆衛生、道路など）について当局に通知します。 その後、開発者はアプリケーションを更新しました。 イノベーションの入力フィールドに、変更の説明を含む特別なテキストが追加されました。 警告が表示され、開発者はすべてを修正しました。 しかし、Google Playでの7日間の警告の後、アプリケーションは削除されましたが、AppStoreではこれは正常でした。

ブロック理由：キーワードスタッフィング

証明：私は提供できません（ maxtreamal@gmail.comからの個人的な手紙）。

結論：禁止の理由を説明するよう要求する手紙には、理由は最終的なものであり、控訴の対象ではないという回答がありました。 彼らはまた、繰り返し訴えられたとき、アカウント全体が削除されると言った。 この場合、政府に属し、他の開発者からのソーシャルアプリケーションがさらにいくつか含まれているため、非常に重要です。 別のパッケージ名でリロードする必要がありました。 4.7ポイントの平均スコア、約700人のユーザー、約50件のレビューを失いました。 どうやら、変更のリストが長すぎました。 説明は短く明確にする必要があると書かれています。 明らかにこれは変更にも適用されます。

Google Playアルゴリズム

モデレートには、手動と自動の2つのタイプがあります。 どうやら、アプリケーションの名前と説明に一般的なキーワードが含まれていない場合、自動モデレートが行われます。 私のアプリケーションでもそうでした-APKをロードしてから2〜3時間後に表示されました。



アプリケーションが自動調停に合格しない場合は、手動で数日待つことができます。



そして、そのような状況があります-アプリケーションが一時停止され、ストアから削除されたが、すべてが開発者コンソールで正常に行われたとき。 どうやら彼はモデレーターが仕事を始める瞬間まで電話を切っていたようです。 この状況に直面した場合、一部の開発者は、アプリケーションに関するすべての情報をできるだけ早く削除することをお勧めします。



ルールとヒント

  • Googleの技術サポートに連絡するときは注意してください。 アプリケーションのブロックを解除する代わりに、1文字でアカウント全体をブロックできます。
  • 別の名前でアプリケーションをリロードしないでください
  • 同様のアプリケーション（ブロックされたアプリケーションと同様）がある場合は、たとえ多くの作業やお金がかかったとしても、すぐに削除してください。 あなたの貴重なアカウントを保存しますが、その損失は手に負えません。
  • 著作権を所有しているコンテンツのみを使用してください。 アプリケーションがこのコンテンツの作成者の開発に貢献する場合でも、他の人の写真、ビデオ、または音声を使用しないでください。
  • アプリケーションのタイトルには特に注意してください。 罪を避けてください-会社名や人気のある製品をそこに追加しないでください。
  • タグに注意してください。 一般に、上記の多くは次のように一般化できます。Google検索を介してアプリケーションをチートでごまかそうとしないでください 。 名前、タグ、または他の場所で人気のあるキーワードを指定すると、多くの人があなたのアプリケーションを見つけますが、Googleはそれを好まないでしょう。
  • 個人的なプロジェクトの場合-Google Playの代替品を見つけてください。 それらはたくさんあります 。 そこで禁止されることはほとんどありません。 そして最悪の場合、誰もあなたの個人的なウェブサイトやブログにAPKを投稿することを止めません。
  • 非常に興味深い考えがコメントに表明されました。 あなたが完全に不当にブロックされている場合（そしてこれが起こる場合）-HabréとGoogleGroupsの問題に関するこの記事を書くことができます。 彼らがPMであなたに手紙を書いて問題を解決する可能性があります。
  • 簡潔に！ アプリケーションの説明（または更新の説明）に原則として（キーではないが）単語が多すぎる場合-禁止される場合があります
  • アプリケーションの一般的なルールに加えて、さらに具体的な推奨事項https://support.google.com/googleplay/android-developer/answer/113474?hl=enがあり 、さらにロシア語で、美しい形式で、例があります。


まとめ

なぜそんなに厳しいのか分かりません。 過去数年にわたって、プレーヤーは物事を整理し、ルールを厳しくしてきたという事実は良いことです。 しかし、開発者のアカウントを禁止するのは、無害なアプリケーションの数個だけであり、裁判や法医学は必要ありません。これはナンセンスです。 開発者は、Googleの態度を変えようとする請願を集めます。



誰が禁止していますか？ どうやら、そこに何か問題がある場合、左のデスクはあなたのアプリケーションを狙うことができます。 また、ロボットまたはモデレーターは禁止できます（アプリケーションが初めてダウンロードされたとき）。 アプリケーションは警告 なしに、また回復の可能性もなくブロックされます。



この記事がお役に立てば幸いです。 「Googleで禁止されていますか？」というジョークは、一部のAndroid開発者にとって厳しい現実になりつつあります。 そして、これは悲しいです。
