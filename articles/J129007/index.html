NFC：技術の現在と未来

こんにちは、habravchane！



怠け者だけがNFCテクノロジーについて話していない。 さらに、この魅力的なテクノロジーに関するHabrの完全で興味深い情報がほとんどないという事実は奇妙に思えます。 ノキアは、モバイルデバイスや家電製品でのNFCの使用を促進するために設立されたNFCフォーラム協会のトップでした。



NFC（近距離無線通信、または近距離無線通信）は、数センチの距離にあるデバイス間のデータ交換を可能にする短距離無線高周波通信技術です。 スマートカードとリーダーインターフェイスを単一のデバイスに統合します。 つまり、NFCを使用すると、通常のユーザーは、これらのデバイスを簡単にタッチするだけで、あるスマートフォンから別のスマートフォンに連絡先やビデオをすばやく転送したり、銀行カードを交換して非接触型の支払いを行うことができます。 Nokiaスマートフォンの例でどのように見えるかは、次のビデオで確認できます。







実際、NFCテクノロジーの多数のアプリケーションがあります。 このようなチップ（および適切なソフトウェア）を使用すると、スマートフォンで銀行カード、交通チケット、割引カード、スマートアクセスカード、映画のチケット、駐車カードを置き換えることができます。 NFCテクノロジーの普及により、インタラクティブビルボード（いわゆるスマートポスター）も使用され、どのスマートフォンが情報を読み取ることができるかと対話します。 近い将来、NFCのおかげで、デバイス間のBluetooth接続の初期化を正確に加速できます：10分の1秒（6秒に対して）-そしてデバイスはお互いを見つけました。 同じ方法でスマートフォンをテレビに持ち込むことで、ビデオや写真を大画面にストリーミングするための接続をすばやく確立できます。 将来を見据えて（未来的な外観を恐れていない場合）、NFCスマートフォンの普及により、家や車の鍵を交換できるようになると言えます。 ところで、一部の自動車メーカー（BMWなど）は、NFCテクノロジーを使用したキーの開発にすでに取り組んでいます。

セキュリティ上の理由から、多くの人が銀行カードの機能と家の鍵をスマートフォンに割り当てたいとは思わないでしょう。特に同時に。 しかし、最初に技術の仕様とその歴史を理解し、その後NFCを判断しましょう。



  1. 1983年、ひげを生やした人はNFCの歴史を始めました。 その年の5月17日に、トレーニングを受けた電気技師のCharles Weltonは、「ポータブルRFエミッター識別子」の特許を取得しました。 そのため、RFIDの概念そのものが現れます。
  2. 2004年3月18日に、ノキア、ロイヤルフィリップスエレクトロニクス、ソニー株式会社は、民生用電子機器、コンピューター、モバイルデバイスで使用するNFCテクノロジーを推進および標準化する非営利団体であるNFCフォーラムを開催しました 。
  3. 2006年、 NFCタグの初期仕様が登場し、NFCチップを搭載した最初の商用電話 Nokia 6131がリリースされました 。
  4. 2009年5月、 NFCフォーラムは 、NFCチップを搭載したデバイス間でさまざまな情報を転送するためのピアツーピアモードを導入しました。リンク、連絡先、Bluetooth経由の通信を確立するためのデータです。
  5. 2011年3月、 GoogleはNFCフォーラムに参加しました。
  6. 2011年5月、 Googleは、銀行カードをNFCチップを搭載したスマートフォンにリンクするためのモバイルアプリケーション、Google Walletを発表しました。 まもなく、 ユーザーはクレジットカードをスマートフォンに交換できるようになります 。
  7. 2011年8月、ノキアはすべてのSymbianスマートフォンにNFCチップを搭載することを発表しました。




ご覧のとおり、NFCフォーラムはNFCテクノロジーの開発に大きな影響を与えていますが、他の参加者はその開発の背後にいます。 そのため、モバイルオペレーターGSM協会 （世界の219か国に代表される約800のオペレーター）のグローバルな業界団体は、NFCに関する2つの重要なイニシアチブを提出しています。

1）モバイルNFCイニシアチブ。 このイニシアチブの一環として、世界市場のほぼ半分を支配している主要事業者は、NFCテクノロジーのビジョンを説明 （pdf）しました。

2）Pay Buy Mobile Initiative。 今回、オペレーターは、決済システムとの相互作用のためにモバイル機器で短距離通信技術を使用するためのグローバルなアプローチを決定 （pdf）しようとしました。



GSMAに加えて、NFC、無線データ交換ネットワーク、モバイル情報交換の統合を検討するために作成されたヨーロッパコンソーシアムStoLPaN （NFCによる店舗物流と支払い）は、NFCの生活に参加しています。



NFCテクノロジーは、RFIDテクノロジーの論理的な継続であり、後者との主な違いは、その限られた範囲です。 アクティブなRFIDタグの読み取り距離は数百メートルに達する可能性がありますが、NFCタグは5〜10センチメートル内でのみ利用可能です。 NFCチップの動作周波数は、産業、医療、科学の目的で使用されるISM帯域の免許不要の無線周波数範囲にあり、13.56 MHzであり、情報転送速度は106、212、または424 kbaud / sです。



NFCは、RFID標準ISO 14443 / Mifare、FeliCa、およびISO / IEC 18092をサポートしています。ISO14443に基づいて作成されたカードは、一般にBSK（非接触型スマートカード）と呼ばれます。 FeliCaテクノロジーは、中国および日本の公共交通機関でのモバイルおよび発券に使用されています。 NFCのISO 14443 / Mifareと同様に、通信はループアンテナの相互誘導によってサポートされます。 ご覧のとおり、この技術は、以前に開発された交通カードとモバイル決済のインフラストラクチャに完全に適しています。

NFCフォーラムは、NFCチップの動作の3つの通信モードを特定しました。

  • ピアツーピアモード 。 この場合、2つのデバイスは互いに通信して、デバイス間で情報を交換します。 p2pを使用すると、NFCチップを搭載したスマートフォン間の連絡先が転送され、設定がWiFiルーターからモバイルデバイスに即座に転送されます（はい、これも可能です！）。 このモードでは、スマートフォンを組み合わせてマルチプレイヤーゲームを作成できます。 Nokiaには、NFCバージョンのAngry BirdsとFruit Ninjaヒットがあります。これらのレベルの一部は、NFCを介して別のスマートフォンに接続した場合にのみ開きます。
  • リーダーモード 。 このモードでは、スマートフォンはバーコードのさまざまな追加情報を含むNFCタグの最新のアナログスキャナーとして機能します。 ところで、最近、NFCタグはバーコードに取って代わりつつあります：スーパーマーケットの食品（有効期限と組成をすばやく見つけるため）や広告情報をすばやく伝えるためのビルボードで見つけることができます。
  • カードエミュレーションモード 。 ほとんどの場合、このモードはNFCテクノロジーに関連付けられています。 カードエミュレーションモードのおかげで、スマートフォンは銀行カードまたはメトロパスで「ふり」をすることができ、それにより財布の厚さを狭めることができます（文字通りの意味で、携帯用ではありません）。








ちなみに、特に昨日のGoogleウォレットのリリースを考慮すると、カードエミュレーションモードは最も興味深いものです。



Googleウォレットは、ワンタッチ非接触型決済向けの最初の大規模なスマートフォンアプリです。 もちろん、これはスマートフォンでNFCチップを使用したおかげで達成されました。 このアプリケーションは、Google Nexus S 4Gスマートフォンを使用するSprintオペレーターのアメリカ人ユーザーのみが利用可能であるという事実にもかかわらず、これはMasterCardカードを使用するシティの顧客であるはずですが、その可能性は非常に大きいです。 将来的には、Googleは他の支払いネットワーク（Visa、Discover、American Expressが検索エンジンとの協力を既に発表している）およびNFCチップを搭載した新しいスマートフォンのサポートを追加する予定です。 そして、それらはたくさんあります。最近発表されたNokia 600、Nokia 700、Nokia 701にはNFCチップが搭載されます。 すでにリリースされたNokia C7は、Symbian BelleのリリースでNFCサポートも受けます。 さらに、以降のすべてのSymbianスマートフォンにはNFCチップが搭載されます。



Googleウォレットで購入する際にセキュリティについて心配する必要はありません。 まず、銀行カードのデータは携帯電話のメモリではなく、特別なPN65チップに保存されます。 このチップは、EMVカードで使用されます。 以下の詳細はそのセキュリティについて述べています。チップはすべてのデータを暗号化し、自律的な力を持ち、物理的にハッキングされたときにすべての情報を削除します。 第二に、電話機のスイッチがオフまたはブロックされている場合、NFCアンテナは機能せず、FirstDataテクノロジーを使用して銀行とデータが交換されるため、最大限のセキュリティが確保されます。 第三に、ユーザーがPINコードを入力するまで、原則としてデバイスからデータを読み取ることはできません。



他の世界では、NFCテクノロジーも勢いを増しています。



モバイルデバイスメーカーは、NFCをますます組み込んでいます。 ここではノキアが議論の余地のないリーダーです。すべての新しいSymbianベースのスマートフォンにこれらのチップが搭載されます。 ただし、他の製造業者も、HTCやAppleを考慮せずに遅れをとろうとしていますが、デバイスへのテクノロジーの導入はまだ無視されています。



NFCはラップトップとネットブックですぐに利用可能になります。 Microsoftは1週間前のBUILDカンファレンスで、近日中にリリースされるWindows 8オペレーティングシステムのNFCサポートを発表しました。



日本と韓国では、日本のKDDI、ソフトバンク、韓国のSK Telecomは、テストモードでATM、ショップ、レストラン、タクシー、ガソリンスタンドでNFCサポートを実装しました。 ドコモは、韓国企業のKTおよびSamsungと共同でNFCをサポートした国際チケットシステムの開発を発表しましたが、それらの技術は国際標準とは少し異なり、GSMAとは調和していません。

ロシアでは、新しい技術もその分布を発見しました。



  1. MTSがモスクワの地下鉄で携帯電話を使用して回転式改札口の通過をテストしている間に、メガフォンとビーラインはサンクトペテルブルクでも同じことを行っています。 ところで、モバイルオペレータからのこれらのソリューションはすべて、周辺機器に基づいて実装されており、対応するテクノロジーに対するデバイスの初期サポートなしに、NFCサポートを電話に追加します。 これらの周辺機器は、ISO 14443 / Mifare非接触型カード規格に準拠したワイヤレスチップを内蔵したmicroSDおよびUICC SIMカードにあります。 このようなデバイスには、内蔵アンテナと内蔵アンテナなしの両方が付属しています。 また、フォームファクターがあるため、すべてのモバイルデバイスに適しているわけではありませんが、こうした周辺機器のおかげで、組み込みの技術サポートを備えた電話機が大量に登場する前にNFCテクノロジーを楽しむことができます。
  2. AmbikTech SPbとState Transport Organizerは、北部の首都にPodorozhnikトラベルカードを導入します。これにより、地上交通費の支払いも可能になり、店舗割引カードとボーナスシステムの統合にも取り組んでいます。
  3. 2010年の終わりに、「Processing Technologies」という会社と一緒に、モスクワの高速列車-SheremetyevoおよびSheremetyevo-MoscowでNFCシステムの運賃支払いのテスト実装を開始しました。 この場合、NFCサポートが最初に存在するNokia電話が使用されます。 このために周辺機器を使用する必要はありません。 ユーザーに必要なのは、旅行代金を支払うためにノキアの携帯電話に特別なアプリケーションをインストールすることだけです。
