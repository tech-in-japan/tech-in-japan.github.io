開発者は土地で怠け者ですか？





翻訳者からのメモ：



元のテキストは、場所によっては、明るく表現力のある色付けが施されており、翻訳に合わせて送信することが決定されました。 記事自体は非常に主観的ですが、一般的に、思考の糧を提供します。 良い読書をしてください。



私はソフトウェア開発者であり、バグやエラーを作成しています。 SQL運用データベースを既定でダンプすると、重要な情報が失われ、同僚の膨大な作業が埋もれました。 この投稿の内容は完全に主観的なものであり、いかなる企業を対象としていません。 私たちの分野は、遂行される仕事の質に関して深刻な問題を抱えていると信じており、これに終わりはありません。



過去数年にわたり、業界全体のソフトウェアとサービスの品質が低下するのではなく、低下するように感じ始めました。 すべてが常にベータ段階です（名前と品質の両方に基づきます）。 商品は、マーケティング担当者が望むときに消費者に送られます。販売の準備が整ったときではなく、「すべてをいつでも簡単に修正できる」ためです。 エンドユーザーはバイヤーからベータテスターに​​進化しましたが、開発ではアジャイルが使用されているため、これはすでに標準となっています。 プログラミングでは、間違いや失敗は正常であると信じるようになったので、今ではそれを避けるためにそれほど努力する必要はありません。 何百万もの顧客をサポートするのは難しいことなので、心配しないでください。 「サポート」と「フィードバック」と呼ばれる無限の迷路に単純に送信できるのに、フィードバックやユーザーレポートを知るのに時間さえかかるのはなぜですか？



私はこれについて怒ったことは一度もありませんが、過去数年にわたって、ソフトウェアとWebサイトで非常に多くのばかげた馬鹿げたバグを見てきました。 私たちは若い未熟な生産分野に属し、すべてを正しく行うための資格のある開発者が不足していることを理解していますが、最近は試してさえいないという感覚があります。



以下では、先月だけの同僚の「ジャム」をキャプチャしたスクリーンショットのいくつかに慣れることをお勧めします。 それとも、私はこれに出くわすほど「幸運」なのでしょうか？ または誰もがそれを見ていますが、私だけが叫び始めないように自分を抑える必要がありますか？



多くの場合、データが保護されていないか、暗号化が不十分であるという事実に遭遇する可能性がありますが、それほど頻繁には見られません。







友人のために1つの問題をグーグルで検索し、ASUSのWebサイトページで問題に出くわすと、その説明が問題の説明に理想的でした。 しかし、それに切り替えたときに、問題が解決されたかどうかを示すように求められたフォームのある空白のページを受け取りました。







どういうわけか、IKEAから手紙を受け取りました。これは、調査（レビュー）セッションを終了しなかったことを思い出させ、リンクをたどって結果を保存するように求められました。 私がそれを渡ったとき、ウィンドウは私に戻ってきた、それは私がすでにすべてを終えたと言った：







今日、ヘッダーラが変更されていない電子メールをCourseraから受け取りました。







職場で使用しているウイルス対策により、1か月に2回、ほとんどのインターネットへのアクセスがブロックされました。 このトピックに関する議論はTwitterで熱狂しており、誰もが同じ問題を抱えていました。 これは単に驚くべきことだと思います。更新プログラムをライトに展開する前にすべてをテストすることは本当に不可能ですか？ さらに、GoogleやBBCサイトなど、多くの大きなリソースがブロックされました！







そして、Raspberry Piをオフにしたときのこのメッセージの意味がよくわかりません！







次に、HTC Vive Webサイトにアクセスしようとしましたが、MongoDB接続エラーが発生しました。 私はすぐにツイッターでそれについて書きました。機知に富んだ答えをすぐに受け取りましたが、エラーは翌日まで捨てられました。







PiでUbuntu MATEを上げることも試みました。 Pi互換のイメージをSDカードに書き込みましたが、最終的には最初のブートでこれを取得しました。 そして、私は問題が何であるか理解していない：







今日、私はISS Log Parserをダウンロードしようとしましたが、揺れ動くようには見えません。 開発パネルを開くと、「jQuery not defined」エラーが表示され、うらやましいほどの持続性が示されました。







私は公益事業会社NPower（ガスと電気を扱う）に連絡しようとしました。 彼らは私に通知の手紙を送ったが、私が彼らのサイトに行くためにリンクをクリックすると、SLLについての巨大なブラウザ警告が私のために落ちた。 代替証明書（* .npower.com）があったことが判明しましたが、 npower.comは電子メールの本文に示されています。 この問題をこの会社の少なくとも5人の異なる人々に報告しましたが、誰も真剣に受け止めていないようです。 または彼らは私が彼らに伝えようとしていたことをまったく理解していなかった。 そしてもちろん、誰かが私の個人データを盗もうとしているという通知をブラウザからこれ以上受け取りたくありません。







さて、あなたはすでに1つのセキュリティホールを持っていますが、それは気になります、みんな、ライブチャットページの証明書エラー！







昨日、PayPalから抽出物を取得しようとしましたが、ログイン後に以下のメッセージを受け取りました。 もちろん、何かの支払い時にこのようなものに遭遇するのは不快ですが、このページは特に役に立ちません。 私に役立つ情報がないからです。 一般的に。 テクニカルサポート番号も「サイン」もありません。次に何をすべきか。







何年も前に、Microsoftは米国のWindows Phoneのポッドキャストで「砲撃の一斉」を開始しました。 そして今、ほんの数年後、Googleは同じことをしているように思えます。 さらに、無料のオーディオファイルに似た態度をとる理由はありません。 さらに驚くべきことに、たとえば英国のその他のすべてのGoogle Play音楽サービスは非常にうまく機能しています。 理由は何ですか？







Podomaticは、iTunes URLに「http：//」プレフィックスを追加すると便利です。 これは、デフォルトですでに有効になっているHTTPSを使用している場合でも発生します（HTTPは自動的にHTTPSにリダイレクトされます）。 そのため、保存する前に、時間をかけてすべてを手動で修正する必要があります。







しかし、これは素晴らしいことです。 誰かが「Pro_Hacking」という名前を使用して私のメールアドレスをサービスニュースレターに追加しました。 このメールボックスに関連付けられたアカウントで最終的に何かが起こるのではないかと心配していたため、これを報告しました。 しかし、テクニカルサポートに連絡して、「Pro_Hacking」ではないことを説明しても、返信レターは標準の「Hello pro-Hacking」で始まりました。







しかし、住宅と共同サービスに戻ります。 判明したように、なんらかの「神秘的な」理由で、NPowerは私にガスを供給することができなかったので、私はSWALECに移ることに同意しました。 メールに送信された認証コードを使用して個人アカウントを入力しようとしたときに、何かが間違っているという情報を含むページを返したため、私は後者との通信に別の週を費やしました：







しかし、このWebサイトは広告ブロッカーの使用を非難し、記事を読むことを許可しませんでした。 以下のスクリーンショットから、サイトの収益化の必要性を完全に理解しており、広告ブロックのインストールを延期しているため、ブロッカーを使用しないことは明らかです。 しかし、サイトが同様の方法を使用する場合は、少なくともソリューションをテストしてください！ そして、彼らは私がここにいるときに子供から食べ物を盗んだと私を非難していることがわかりました。 しかし、さあ！ 追いかけて！







次に、PiでVNCを上げようとしました。 ファイアウォールで閉じられていたため、便宜上、短いパスワードを設定しました。 パスワードが短すぎると誓いました。 さて、数文字を追加しました。その後、パスワードを8文字で構成する必要があるというメッセージを受け取りました。







Visual Studio更新プログラムをインストールするときに、Visual Studio改善プログラムに参加するように求められました。 そこには、参加する同意の「目盛り」はオプションであると書かれていました。 しかし、何かは次のようなものではありません：







Visual Studio 2015 Update 2は、多数の安定性とパフォーマンスの問題が解決されたと主張しましたが、VSを開始するとすぐにクラッシュしました。 そして、Microsoft以外の拡張機能やその他のものは持っていないと確信しています。







数週間ごとに、私のMoto 360 Android Wear時計は完全になくなるか、時計が再起動するまでランダムにバッテリーを消耗し始めます。 これについてレポートを書きましたが、Googleは「この動作は、更新プログラムを受け取った後の特徴です」と答えました。 しかし、私はアップデートを持っていませんでした。たとえあったとしても、あなたは同意しなければなりません。どういうわけか、これはすべて愚かです。







この出版物を準備するにあたり、画像を最適化するためのコマンドラインツールの検索を開始しました。 私はこの投稿に出くわしましたが、ページを下にスクロールしようとするたびにスローされました。 約40秒後に、この大惨事の理由が現れました：ポップアップ全ページ広告。 わかった



そして、上記のすべてがたった1か月で起こりました。 すべての場面でスクリーンショットがあればもっと書くことができます。 何も修正することはできませんが、私たちの業界全体が単にユーザーにできることをしていないと感じています。 品質とテストのレベルで有名だった会社でさえ、離れ家に合併するようです。 マーケティングと製品の準備ができる前にダウンロードできることに慣れていませんが、結局のところ、優れた場所はありますか？



それとも、障害はエンドユーザーにあるのでしょうか？ 製品の品質の低さについて不満を言っていないのかもしれないので、企業はそれらを改善する動機を持っていませんか？
