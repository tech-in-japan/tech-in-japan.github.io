Windows Azureモバイルサービスが新しいクラウドサービスを発表

昨日、Scott Guthrieのブログで、Windows Azureプラットフォームサービスファミリで別のクラウドサービスが発表されました。







Windows Azureモバイルサービスは、モバイルアプリケーション開発者がサーバーバックエンドを簡単に作成して使用できるように設計された一連のサービスです。 このようなバックエンドとしてWindows Azureクラウドを使用すると、独自のインフラストラクチャを展開することなく、プッシュ通知、クラウドストレージへのデータの保存、ユーザーの認証と承認のための既製の機能を取得できます。



現在、Windows Azureモバイルサービスはプレビューバージョンとして利用できます。これにより、Windows 8用のモバイルアプリケーションを開発できます。サービスへのアクセスは、C＃およびJavaScriptから引き続き利用できます。 開発チームは、任意の言語からデータを受信して​​サービスを操作できるようにするパブリックREST APIに取り組んでいます。 さらに、サービスの開発に伴い、Windows Phone、iOS、Androidの公式のインストルメンタルサポートが最終バージョンのリリースに追加されます。



新しいモバイルサービスの使用を開始するには、 https：//account.windowsazure.com/PreviewFeaturesでプレビューバージョンを購読する必要があります 。







サービスへのアクセスは要求キューに基づいて発行されるため、アプリケーションが処理されるまでに時間がかかる場合があります。



モバイルサービスの10インスタンスのプレビューがしばらくの間、モバイルサービスの共有モードの価格がゼロであることは非常に楽しいです！ ただし、Webサイトの場合と同様に、予約モードに切り替えると、料金が請求され始めます。 リソース消費の価格と支払いの詳細は、特別な価格の詳細ページで確認できます。



現在、 Windows Azureモバイルサービスは次の機能を提供しています。



  • クラウドへのユーザーデータの保存（ アプリケーション例 ）
  • クラウドでの認証とユーザー承認（ アプリケーション例 ）
  • クラウドサービスからプッシュ通知を受信する（ アプリケーション例 ）


サービスの開始とともに、Windows 8 Mobile Services SDKのアプリケーション開発ツールがリリースされ 、クラウドサービスをWindows 8アプリケーションに簡単に統合できるようになりました。
