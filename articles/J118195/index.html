Sonyは、PlayStation Networkユーザーの個人データが盗まれたことを認めました

Sonyは、次のユーザーデータが4月17日から19日にPlayStation Networkデータベースから盗まれたことを認めました。

  • 名
  • 郵便番号（郵便番号付き）
  • 居住国
  • メールアドレス
  • 生年月日
  • PlayStation Network / Qriocityログインとパスワード
  • ハンドル/ PSNオンラインID


また、PlayStation Network / Qriocityにアクセスするための購入情報、セキュリティの質問（および回答）が盗まれた可能性もあります。

確認はありませんが、システムで使用されたクレジットカードの数と有効期限（秘密コードを除く）も攻撃者の手に渡ったことは除外されません。
