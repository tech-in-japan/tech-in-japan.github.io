Linux / Windows上のパブリックVPSテンプレート-著者と管理者を招待して、AWS AMIのようなサービスを作成します

膨大な量の多様なソフトウェアを使用するには、VPSを慎重に構成し、何百ものテンプレートを最新の状態に維持する必要があります。 Amazon Web Servicesは、コミュニティサポートを活用してAWS AMI（Amazon Machine Images）をサポートします。 HOSTKEYにはこれを行うためのリソースがなく、コミュニティに相互に有益な助けを求めています。 要するに、人気のあるテンプレートの作成に関して、管理者とプログラマーが相互に利益をもたらすように招待します。



ほとんどの複雑なオープンソースプログラムはインストールが困難です。これを行う方法を知っており、運用経験が必要です。 READMEによると、通常、コンソールで数時間を費やすことなくすぐに成功することは不可能です。 そして、1か月後、すべてが間違って配信されるという幻想がやって来ます。 Amazonには、既製のAMIがあります。 私たちは、AMIと機能が似ているローカルサービスの作成者を探しています。







典型的な例は、virtalka上で個人用の無料ドロップボックスを整理できる最終ベータ版のプロジェクトです 。 同期用のダウンロード可能なエージェント、Webアクセス、さまざまなユーザー-すべてがほぼDropboxのようなもので、所有者以外は誰もアクセスできず、ディスクのサイズは任意です。 これは複雑なシステムであり、長くて退屈です。 インフラストラクチャに展開して使用できる仮想マシンイメージの形式で無料バージョンを提供しています。 このプロジェクトは、追加の手間をかけずに既製の戦闘デモとロイヤリティを受け取ります。

弊社で独自のSaaSを作成することは非常に簡単になり、独自の技術的および金融インフラストラクチャに対処する必要はありません。



Hostkeyは、インフラストラクチャ上でテンプレートを作成するための無料のVPSを提供します。 必要で便利なソフトウェアをそれらに置き、デザイン全体をセットアップし、その仕組み、潤滑場所、使用方法をページで説明します。 テンプレートに関するユーザーの質問に答えます。 プライマリマシンのソフトウェアを更新し、その安全性を監視し、そのようなテンプレートから収集されたVMの安定した動作を保証します。 クローンの機能を確認できるように、テンプレートを作成します。



商用ソフトウェアを販売している場合は、VPSに置くこともできます。このようなVPSを自動的に行うことができ、ソフトウェアのコストを注文の個別の行として強調表示します。 つまり、Windowsで下水道設計などのアプリケーションプログラムを使用してVPSテンプレートを作成し、1か月500ルーブルでクライアントに引き渡すことができます。 クライアントは必要なことを行い、VPSで100r、ソフトウェアの作成者である200rを取得します。 このソフトウェアは、この特定のVPSの環境でのみ動作するように変更することができ、そこからコンピューターにそれを選択するには、専門家がそれを選択するための多大な費用がかかります。 一般的に、多くのオプションがあります。



このようなテンプレートを注文可能にし、ウェブサイトで説明し、その使用に関するドキュメントを送信し、VPSの技術サポートを提供します。 テンプレートの接続には約1週間かかります。



そして今、メインバン- テンプレートの作成者は、そのテンプレートでVPSを注文した顧客のすべての支払いの10％を受け取ります。 あなたがお金を受け取りたいVPS用の別のソフトウェアがある場合、これは別々に説明する必要があり、そのようなVPSを注文するとき、著者は彼によって示されたスキームに従ってお金を受け取ります（他の人のライセンスに違反しないでください）。



WindowsのSystem Center 2012とLinux / FreeBSDのSolusVMを使用します。VMの複製元のテンプレートはどこにでもあります。 VMを展開するとき、オーケストレーターはそれにリソースを割り当て、ネットワークインターフェイスを静的アドレスに構成し、パスワードを変更します。 Windowsでは、手動アクティベーションも必要です。







テクニック別のミニFAQ：

  • どのプラットフォームが利用可能ですか？ すべての主要なLinux、FreeBSD、Windows 2008/2012はそのまま使用できます。 VMは任意のISOからロールし、ソフトウェアを配置できます。
  • 複数のVPSを連携して動作させることはできますか？ 問題ありません。同じノードにあり、仮想VLANを使用できます。
  • VPSにはどのようなリソースがありますか？ VMごとに最大12コアと128GBのメモリの大きなノードを使用します。
  • ノードにはどのようなパラメーターがありますか？ 2xE5645 2.4GHz 12コア/ 24スレッド/ 192Gb RAM / iSCSIファイラーへの10G接続
  • これらのノードはどこにありますか？ モスクワとオランダで。 オランダにはまだシステムセンター/ Hyper-Vはありません。
