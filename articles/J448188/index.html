タイムカプセル：HP iPaq h1940「そのまま」 2000年代からのPDAとは





先週グラナダフリーマーケットで別のガジェットを買いました 。 彼はPDA HP iPaq h1940になりました。 この特定のデバイスでは、PDAがすべてのドキュメントと追加のソフトウェアを備えたディスクと一緒に箱で販売されていたため、選択肢が少なくなりました。 ハンドヘルドはラップトップのように10ユーロかかりました 。



箱に入っていたもの















箱の中には、デバイス自体、バッテリー、追加のコネクタ、USBケーブル、ドキュメント、ソフトウェアディスクがありました。 ディスク-MS ActiveSync 3.7、Pocket Outlook、およびPDA用のいくつかの追加アプリケーション。







設計



デバイスは非常にモダンでスタイリッシュに見えます。 手にフィットし、ジーンズやジャケットは問題なくポケットに入ります。 ボタンは完全に押されています-時間はまったく影響しませんでした。



ディスプレイも新品に似ています-フェードせず、明るさも失われません（他のPDAや、新品のときの見た目の記憶と比較します）。 過去数年間はケースにまったく影響を与えませんでした-プラスチックは劣化せず、ステッカーと碑文は以前と同じです。







このハンドヘルドは、他の多くのモデル（たとえば、 数年前に書いたHP iPAQ rx3715）よりも小型です。





HP iPaq h1940、iPAQ rx3715、Acer 50n Premium



それは著しく薄いです。 最も可能性が高いのは、PDAをよりエレガントにしたため、ポケットに入れて簡単に持ち運べることです。



ケース-プラスチック、カラー-メタリック。



ボタンはiPaq PDAで一般的です-これらはオーガナイザーを起動し、連絡先、タスクマネージャー、メールメッセンジャーを呼び出すための4つのボタンに加えて、確認ボタン付きのジョイスティックです。 上部には、ボイスレコーダーコントロールボタン、電源ボタンがあります。



技術仕様



  • OS：MS Windows Mobile 2003 Professional
  • プロセッサー：Samsung S3C2410 266 MHz
  • RAM：64 MB
  • 対角線：3.5
  • 解像度：320 x 240（QVGA）
  • 色の演出：65000色
  • 幅：70 mm
  • 高さ：113 mm
  • 奥行き：19 mm
  • 重さ：124g
  • バッテリー：リチウムイオン、900 mAh


このPDAには、セキュアデジタル（SD）/マルチメディアカード（MMC）拡張スロットがあります。 さらに、USBケーブルを使用して、デバイスに保存されている情報を操作できます。 つまり、PDAはUSBフラッシュドライブとして接続できます。



これは、Windows Mobile 2003を搭載した最初のHPハンドヘルドであり、Windows CEの後の「ポケットOS」の次のバージョンでした。 たとえば、GIFで動作し、SSLをサポートするPocket Internet Explorerを改善します。



PC接続



すべてをスムーズに進めるために、私はPDAを仲間のPackard Bell Easyone Silverラップトップに接続することにしました（このレビューは前回公開されました ）。 Windows XPが搭載されており、USBに接続するとPDAがすぐに検出され、ドライバーがWindowsストレージで見つかり、機能しました。







付属のCDからCD Active Syncをインストールして、データを同期し、エクスプローラーでPDAの情報を表示しました。















Active Syncは問題なくアクティブ化され、ラップトップとハンドヘルドのデータの同期を開始しました。 完全な同期を行うには、ユーザープロファイルを作成する必要がありますが、同期するものは何もないので、ユーザープロファイルなしでも実行できます。







マルチメディア



HP iPaq h1940には、SDカード用のカードスロットがあります（microSDおよびアダプターを使用）。 ハンドヘルドは2 GB以下のカードのみを受け入れることが判明しました。 カードのすべてのファイルは問題なく読み込まれ、さらに2013年からmp3ファイルを起動できました（ルイアームストロングであることが判明しました）。







2 GBのメモリカードとPDAを使用すると、このデバイスをメディアプレーヤーに変えることができます。 音質は悪くなく、すべてはヘッドフォンで良いです。







メモリカードにある2つのメロディーの助けを借りてサウンドを評価しました。 これらは、オスカーベントンのベンソンハーストブルースとルイアームストロングのゴーダウンモーゼスです。 私はPDAではどういうわけかオーディオブックを聞いたので、デバイスもこれに適していることを覚えています。







ビデオははるかに悪いです。 まず、デバイスは320 x 240を超える品質のビデオを再生できず、デバイスは新しいフォーマットに対応しません。 他のどこかに、PDA用に変換された映画が残っているリソースがありますが、そのようなサイトの大部分は既に閉鎖されており、そうでない場合、多くのダウンロードリンクが単純に死にました。



また、安価な中国のスマートフォンがはるかに優れている場合でも、この品質でビデオを視聴したい人はいますか？



ソフトウェア



このソフトウェアは、Windows MobileまたはWindows SEを備えたPDAの他のバージョンにインストールされたデフォルトのアプリケーションと違いはありません。 「連絡先」、オーガナイザーとタスクのリストを含むカレンダー、メールアプリケーション、メモ帳、ボイスレコーダー、テキストエディター、スプレッドシートがあります。 さらに、iPAQ Image Viewer HPがあります。



サードパーティのソフトウェアを使用すると、ビデオよりもすべてが優れています。 PDA用のソフトウェアの巨大なアーカイブを備えたまだ機能しているサイト。 たとえば、HPC-それから、有名なAIReaderリーダーをダウンロードしました。これは、電子書籍愛好家に多くの機能を提供します。 多数の形式（epub、fb2、txtなど）で動作します。 リーダーは問題なくインストールされ、キリル文字の画面が表示されました（PDAはスペイン語で、キリル文字に問題があるかもしれないと思っていましたが、ありません）。



他のソフトウェアではより複雑です-一部のプログラムはPCからインストールする必要があります。 そして、私のラップトップにはWindows 10がインストールされているため、インストールは行われません。 これを行うには、10の環境では機能しないActiveSyncが必要です。 唯一の解決策は、アプリケーションをメモリカードにダウンロードし、PDAに挿入して、カードから既にパッケージをインストールすることです。 実際、私はAIReaderでそれをやった。







Windows Mobile 2003 Professionalには多くのソフトウェアがあります。他の形式は通常PCからのインストールを目的としているため、exeまたはcabを探すだけです。



リーダーに加えて、SegaエミュレーターPicoDriveバージョン0.03もインストールしました。 試用のため、最も人気のあるゲームDune：The battle for ArrakisのROMをダウンロードし、PDAですべて起動しました。 ゲームは機能し、少しプレイしました（一度に、別のPDAのこのエミュレーターで何度か試してみました）。 しかし、印象はそうではありません。PDAで目をつぶるよりも、OSでエミュレータを実行して大画面でプレイする方が良いです。











ワイヤレス接続







iPaq h1940 PDAには、BluetoothおよびIrDaモジュールのみが装備されています。 BTでは機能しませんでした-iPhone 8 Plusとペアリングしようとしました。 両方のデバイスは問題なくお互いを確認し、コードを入力して確認するリクエストを発行しました。 しかし、コードを入力して別のデバイスで確認した後、PDAは接続の問題について書きました。 ほとんどの場合、プロトコルの互換性の問題です。











しかし、Huawei Y6 Androidスマートフォンに接続して、Bluetooth経由でネットワークに接続することが判明しました。 接続するには、Bluetoothをオンにし、LANへの接続を選択し、リストでデバイスを見つけて接続する必要があります。























電話では、デバイスの検出と表示をオンにし、さらにBluetoothモデムをアクティブにする必要があります。 この場合にのみ、PDAは電話に接続し、それを介してオンラインになります。



組み込みのブラウザーを使用して、HabrもGoogleもYandexで開くことはできませんでした。 理由は、最後の投稿のコメントで書いたように、古いブラウザーが新しい標準のSSL、IPv6、およびその他の最新テクノロジーを使用できないことです。



かつて人気のあったサイトhpc.ruのみをダウンロードできました。PDAでの表示に最適化されているためです。 さらに、アプリケーションをダウンロードすることもできました。これは同じAIReaderで、インストーラーの通常の操作を確信していました。 ブラウザにダウンロードされ、インストールされたアプリケーション。 ところで、興味深い事実は、デフォルト設定中にインストールした後、インストーラーファイルが削除されることです。



















このPDAにはWiFiはありません。



バッテリー







驚くほどよく保存されています。 初めて電源を入れる前に100％充電しました。 夜間と一部の時間帯に、充電レベルは50％に低下しました。これは、デバイスの積極的な使用、ネットワークへの接続などに伴うものです。 バッテリーは非常に良好な状態です。



2000年代初頭のPDAは今や役に立つでしょうか？



はい、できます。 すぐに思い浮かぶいくつかの使用法を次に示します（HP iPAQ rx3715という別のデバイスでそれらのいくつかを実装しました：



  • リーダー。 スマートフォンのバッテリーを使い果たしたりメモリを使い果たしたくない場合は、HP iPaq h1940 PDAは小さなリーダーの役割に最適です。 AIReaderまたは他のソフトウェアをインストールすることにより、読み始めることができます。
  • オーディオプレーヤー。 何らかの理由で他のデバイスがこの役割に適さない場合、PDAはプレーヤーとして機能できます。 カードにMP3をダウンロードしてください。 携帯電話を持ちたくない場合などに、PDAを携帯して実行できます。
  • エンターテイメントセンター。 Windows Mobile 2003には、チェスからゲームコンソールエミュレータまで、数多くのゲームがあります。 PDAの小型レトロゲーム機は大成功です。
  • リマインダー、プランナーのリスト。 もちろん、スマートフォンは連絡先、タスク、計画を保存するなど、日記としてほぼ理想的です。 PDAもこの目的に適しています。
  • ボイスレコーダー-1つのボタンで音声の録音を開始します。 デバイスは音声を書き込み、録音品質は許容範囲です。


一般に、PDAは、メインデバイスに障害が発生した場合のバックアップガジェットとして使用できます。 もちろん、PDAを使用して電話をかけることはできませんが、スマートフォンに現在提供されている他の多くの機能を複製できます。
