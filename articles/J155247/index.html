フラッグシップスマートフォンSamsung GALAXY Note IIのレビュー

こんにちは、Habr！



本日、フラッグシップのスマートフォンであるサムスンGALAXY Note IIについて詳細にご紹介したいと思います。 10月2日に開催されたSamsungギャラリーでのデバイスの公式プレゼンテーションの直後に、いくつかの情報を発表しました。 この記事では、第2世代のGALAXY Noteにあるすべての革新、技術、ソフトウェア、および新しいスマートフォンをモバイルデバイス市場のリーダーにもたらすために設計されたすべての利点を明らかにします。 そのため、カットの下には、多くの文字が（ケースにのみ）あり、写真の写真に劣らず（目を楽しませてくれます）。







▌外観と寸法



新しいフラッグシップには、5.5インチの対角線を持つ拡大画面があります（最初のモデルの画面は5.3でした）。 画面周囲のフレームは薄くなり、その表面は「防弾」ゴリラグラス2によって確実に保護されます。スマートフォンのサイズは151.1 x 80.5 x 9.4 mmです（つまり、最初のGALAXY Noteよりもさらに細く、薄くなりました）。デバイスの重量は実質的に変化せず（180グラム対第1世代のデバイスの178グラム）、バッテリーの容量は1/4増加しました。







また、前面にはスピーカーグリル、デバイスの状態を示すマルチカラーのLEDインジケーター、ライトセンサー、近接センサー、フロントカメラがあります。 下部にはメインメニューにアクセスするためのメカニカルボタンがあります。メインメニューはより長くなり、使いやすくなりました。+左右の2つのタッチボタン：左側の「メニュー」と右側の「戻る」。







スマートフォンの背面は耐久性のある薄いプラスチック製のふたであり、よく曲がりますが、壊れません（サムスン独自の開発。同社のすべてのトップエンドのスマートフォンで使用されています）。 上部には、銀色のフレームで囲まれたカメラの下のスロットがあり、カメラの右側にはフラッシュがあります。







デバイスの両端で：

•左-ボリュームロッカー。

•右-電源オフ/ロックボタン。

•上-3.5 mmヘッドフォンジャックとノイズ低減のための追加のマイク。

•下— microUSBコネクタ、Sペンスロット、およびメインマイク。







電源オフボタンと音量ボタンは、ケースの下部にあるため、最初のGALAXY Noteよりも人間工学的になっていることに注意してください。



カバーの下には印象的な3100 mAhのバッテリーと、microSDおよびmicroSIMカード用のスロットが隠されています。 カバー自体の内側に、NFCアンテナがあります。







販売の最初の日には、GALAXY Note IIは「白大理石」と「グレーチタン」の2色で提供されます。







▌ スクリーン



すでに上で述べたように、デバイス自体がいくぶん狭く薄くなったにもかかわらず、画面は顕著に伸び、対角線でさらに0.25インチ勝ちました。 これは、その比率の変更のおかげで可能になりました。最初のGALAXY Noteでは、ディスプレイのアスペクト比は16:10で、スマートフォンの2番目のモデルは、HD解像度-16：9（1280x720ピクセル）でビデオを視聴するのに最適なワイドフォーマットを取得しました。



GALAXY Note IIにはHD Super AMOLEDが搭載されているため、スマートフォンは優れた視野角と大きな輝度マージンを備えています。

▌ カメラ



写真 GALAXY Note IIは、GALAXY S IIIと同じカメラモジュールとLEDフラッシュを使用します。 それにも関わらず、新しいデバイスは低光量条件下での撮影を改善しました。対応するモードが撮影設定で利用可能になりました。 次のモードも利用できます。







•デフォルトは単純なショットです（ワンクリック-ワンショット）。

• バーストショット。 数秒で最大20ショットの連続撮影。

• HDR。 このモードは、明るい光と深い影の両方がある高コントラストのシーンを撮影するためのものです。

• 笑顔。 笑顔をキャッチできる機能：シャッターを押した後、写真は、オブジェクトの顔に笑顔をキャプチャした瞬間にのみ撮影されます。

• 美しい顔。 肌の欠陥を「トーン」して、顔に健康的できれいな色合いを与える効果。

• パノラマ。 現在人気のあるパノラマショット：カメラをゆっくりと左右に動かし、写真をキャプチャします。

• 似顔絵。 意図的に色を歪ませるグロテスクなフィルター。

• 顔タグ。 また、今日非常にトレンディな機能。 写真に写っている友人の顔を特定します。その後、友人に電話をかけたり、メッセージを送信したりできます。

• ベストショット。 最高の写真の断片を重ね合わせることにより、いくつかの写真から最も成功した写真を選択できます。

また、設定には、ポートレート、風景、夜、スポーツ、屋内、ビーチ/雪、日没、夜明け、紅葉、花火、テキスト、ミステリー、バックライトなど、あらゆる好みに合った多くの物語があります。

▌ コミュニケーション









• Bluetooth 4.0（LE）。 ファイル転送はWi-Fi 802.11 nを介して行われ、データ転送速度は24 Mbpsに達する可能性があります。

• USB 2.0。 Android第4世代では、USB大容量記憶装置標準がMTPに置き換えられました（PTPモードもあります）。 データ転送速度-最大25 Mb / s。 microUSBコネクタはMHL標準もサポートしています。特殊なケーブルを使用すると、HDMI入力を介して電話をテレビに接続できます。

• 3G。 このデバイスは、最大モバイルデータレートが21Mbit / sのユビキタスHSPA +をサポートしています。 GSMネットワークの場合、EDGEクラス12が提供されます。

• Wi-Fi。 802.11 a / b / g / n標準（2.4および5 GHz、デュアルバンドWi-Fi）をサポートします。 また、802.11 n標準はHT40動作モードをサポートしています。 Wi-Fi帯域幅を2倍にできますが、これには別のデバイスのサポートが必要です。

• Wi-Fi Direct。 第三世代のBluetoothと積極的に競合する比較的新しいプロトコル。 デバイスを検索し、それらとの接続をアクティブにし、ファイルマネージャーを介して他のデバイスのファイルを管理し、デバイスからデバイスにファイルを転送できます。

• NFCテクノロジーはスマートフォンに実装されています。 その使用のさまざまな可能性は後で知られるようになるでしょう。

• Sビーム。 数秒で大量のファイルを別のデバイスに転送できるサムスン独自のテクノロジー。 最近まで、GALAXY S IIIでのみ利用可能でした。 本質的に、S BeamはNFCとWi-Fi Directの組み合わせです。 最初のテクノロジーは電話の認証に使用され、2番目のテクノロジーはファイルの転送に使用されます。 おそらく、これはデバイス間でファイルを転送する最も便利で高速な手段の1つです。



▌S PEN



Sペンは、外観と機能の両方で（大幅に）多くの改善が行われました。 ペンが太くなったため、使いやすさにプラスの効果があります。 1,024レベルの圧力が認識されます。

さらに、ペンを持ってくるとスクリーンに表示される投影点のおかげで、スマートフォンはディスプレイに触れなくてもペンの存在を感じます。 それでも、ハードウェアの変更は、最初のGALAXY Noteにはなく、2番目に利用可能になったソフトウェアの革新とチップの数ほど印象的ではありません。



• エアビュー。 この機能により、写真や画像を高速モード（プレビューの一種）で表示できます。それぞれをクリックする必要はありません。画像上の投影点を指定するだけです。

• ポップアップノート。 電話での会話中に画面をダブルクリックすると、ポップアップSノートが表示され、そこに重要な情報を記録できます。

• クイックコマンド。 手書きコマンドを作成して特定の機能を割り当てる機能（たとえば、コマンド "@ brother"を作成し、自動的に呼び出し音を鳴らしたり、友人にメッセージを送信したりできます）。

• 簡単なクリップ。 任意のソースから画像の断片を切り取り、メモ、メッセージ、電子メールに貼り付けたり、Fragmentsフォルダーに保存したりできる非常に面白い機能。

• 手書き認識。 この機能は乱雑な処理にも対応しており、「斜めの」手書き文字は非常に許容されます。 手書きのメッセージはメールで送信できます。SPlannerオーガナイザーを使用すると、手書きのメモを取り、メモリに保存できます。







• 写真メモ。 確かに、誰もが写真付きの古いアルバムを持っており、その裏面には「モスクワ、新年、1985」のようなものが書かれています。 GALAXY Note IIに保存されている写真の裏面に、デジタル形式のみの同様の手書きのメモを残すことができます。

• ジェスチャー。 Sペンの本体にボタンがあります。 クリックすると、スマートフォンはいくつかのジェスチャーを理解できます。 クイックコマンドとイージークリップもこのタイプのジェスチャに属します。 それらに加えて、大切なボタンを押しながらペンで画面をつつくことで、まだスクリーンショットを撮ることができます。 もう1つの便利な機能：ボタンを下から上に押したままペンを保持すると、クイックコマンドメニューを呼び出すことができます。



一般に、S Penの機能は1桁大きくなりました。スマートフォンでのその存在は非常に重要で便利です。

結論



GALAXY Note IIは、すべての主要なパラメーターで先祖を大幅に上回りました。手にとって便利なため、画面がさらに大きくなり、HD解像度に最適です。 S Penはより敏感で機能的になり、鉄はより速く、より近代的になり、バッテリーははるかに大容量になりました。



最初のGALAXY Noteは当初、戸惑いと使いやすさに対する疑念で聴衆に受け取られました。 しかし、スマートフォンはそれ自体に宛てられた皮肉なレビューをすべて払いのけ、セールスリーダーになり（世界中で1,000万枚以上販売されました）、実際に新しいプレミアムクラスのモバイルデバイスを作成しました。 GALAXY Note IIは、改良されたより高度なスマートフォンであり、間違いなく創造性、仕事、エンターテイメントの可能性をさらに広げます。
