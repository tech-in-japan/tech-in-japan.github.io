ロシアのさまざまな都市にある一連のWindows Phone 7学生キャンプに招待します

過去1か月にわたって、 Windows Phone 7 Student Campをいくつか実施しました。 このようなイベントでは、1日でチームに参加し、アイデアを思いついて実際のアプリケーションを作成するWindows Phone 7用のアプリケーションの作成方法を学びたい学生が集まります。 これで、この非常に成功した活動を一度にロシアのいくつかの都市に拡大する準備ができました。







サンクトペテルブルク、エカテリンブルグ、トムスク、モスクワにあるWindows 7学生キャンプに興味のある学生を招待します。
  • 11月19日- サンクトペテルブルク 、 イングリアテクノパークと。 登録する必要があります 。
  • 11月22日-Tomsk 、TPUのマイクロソフトイノベーションセンターに基づく。
  • 11月25日- エカテリンブルク 、UrFUのマイクロソフトイノベーションセンターに基づく
  • 11月29日と12月9日（イベントは2日間に分割されます）- モスクワ 、MPEIのマイクロソフトイノベーションセンターに基づく
  • 11月29日- モスクワ 、HSEのマイクロソフトイノベーションセンターに基づいて（登録情報は後ほど）
  • 12月2日- サンクトペテルブルク 、SPbSPUのマイクロソフトイノベーションセンターに基づく


イベントに参加するには、登録することをお勧めします。 詳細については、上記のリンクを参照してください。 各イベントで次のことができます。
  • 電話用にプログラムがどのように書かれているかについての入門的な話を 聞いてください 。 C＃やJavaなどの言語でのプログラミングスキルは役立ちますが、電話のプログラミングに関するその他の事前知識は必要ありません。
  • アプリケーションのアイデアを思い付くには、チーム内で志を同じくする人々を見つけて、アプリケーションのプロトタイプを開発します-多くの質問に迅速に答えることができるマイクロソフトの専門家の助けを借りて。
  • Windows Phone 7 Marketplace に無料で登録し 、アプリケーションの予備バージョンをそこに配置します。
  • 将来的にアプリケーションを配布することでより良い収益を得る方法を理解します。
  • Imagine Cup 学生コンテストとWindows Phone 7アプリコンテストへの参加方法を学びます。
  • Windows Phoneの開発について既に質問している場合は、 質問してください。
さらに、さまざまな大学のアクティブな学生と出会い、テクノロジーに興味のある人たちと楽しい時間を過ごすことができます。



イベントを最大限に活用するには、以下を行う必要があります。
  • 事前に登録してください 。 登録する際は、必ず名前、姓、学習場所を明記してください。 このデータが存在しないと、登録が拒否される場合があります-イベントは学生を対象としているためです。
  • 友達にイベントについて伝え、一緒に持っていく-一緒に仕事をするのがずっと楽しくて生産的になります
  • ラップトップを持ってイベントに来て、その上で開発を行ってください！ 可能であれば、事前にラップトップにWindows Phone Developer Toolsをインストールしてください。

  • DreamSparkに登録されていることを確認し、 Windows Phone 7 Marketplaceで無料で登録します 。 これはその場で実行できます（DreamSparkのコードの取得を含む）が、イベントで時間を無駄にしないように事前に実行することをお勧めします。
  • イベントで取り組むことができるモバイルプラットフォーム用の便利なアプリケーションのいくつかのアイデアを思い付く。 しかし、あなたが面白いアイデアを持っていなくても、とにかく来てください。コミュニケーションの過程で、通常、アイデアは自分で生まれるからです。
学生キャンプでお会いしましょう！
