Ubuntu Advantage Subscription：現在ロシアで

Flant社はロシアのCanonical アフィリエイトネットワーク の最初のエージェントになり、 Ubuntuの公式テクニカルサポートであるUbuntu Advantageのサブスクリプションの販売開始を発表しました。



Ubuntu Advantageの所有者は、Ubuntu ServerおよびUbuntu Desktopに関する技術的および法的アドバイスを受け、ランドスケープITインフラストラクチャのライフサイクル管理プラットフォームにアクセスできます。





Ubuntu Advantageは何を提供しますか？



  • Ubuntu ServerおよびUbuntu Desktopの使用に関連するさまざまな問題に関するロシア語の技術相談。 保証された時間枠で問題を解決します。
  • 問題のエスカレーション。 「複雑な」問題が発生した場合（たとえば、アプリケーションコードの修正が必要な場合）、Canonicalに転送され、そこで解決されます。
  • Ubuntuおよびそのコンポーネントのライセンス問題に対する法的保護。
  • ランドスケープ-集中管理されたUbuntuインストール管理システムへのWebインターフェース。
