PAP参加者にそんなにお金を払うことはできません

2016年4月1日は、 著者の奨励プログラム（PAP）が開始されてからちょうど5年になります。これにより、ユーザーはHabréに質の高い興味深い記事を公開してお金を稼ぐことができます。 この5年間で、数百人の著者がプログラムに参加し、合計数十万ルーブルを獲得し、PSAの有効性を証明しましたが、...







しかし、このように続けることはできないと判断しました。 現在支払っている分だけ支払うことはできません。 したがって、我々は長所と短所をよく比較検討し、決定を下しました... さらに支払います！



はい、この日からプロモーションのサイズを増やし、支払い額に影響する評価基準を導入します。



  • 出版物が+30を得るなら、著者は3,000ルーブルを受け取ります;
  • 出版物が+50を得るなら、著者は5,000ルーブルを受け取ります。


さらに、しきい値のいずれかを超える後続の評価ポイントごとに、作成者は以前の2倍を受け取ります。各ポイントの5ルーブルの代わりに、10ルーブルを請求します。 例：



  • +30点数の出版物は、著者に3,000ルーブルの報酬を保証します。
  • 評価が+49の出版物は、3,000 + 190 = 3,190ルーブルを受け取ります。
  • +151と評価された出版物は、5,000 + 1,010 = 6,010ルーブルを獲得します。


「翻訳」タイプの出版物の場合、メカニズムは保持され、新しいしきい値に応じた金額は2つに分割されます。



+30は3000/2 = 1500 pを取得します。

+49は3190/2 = 1595 pを取得します。

+151は6010/2 = 3005 pを取得します。



いいえ、これはデマではありません。 結局のところ、あなたが知っているように、彼らはお金で冗談を言っていません。



Author Encouragement Programに参加して、興味深い記事を投稿し、お金を稼ぎましょう。



PS：4月1日に公開された記事は、新しいスキームに従って計算されます。 テキストが以前に発行された場合（たとえば、3月31日23時59分）、計算は古いスキームに従って実行されます。
