Linux InstallFest 5月1日

5月1日、Linuxとフリーソフトウェアの祭典「Linux InstallFest 11.04」がニジニノヴゴロドで7回開催されます。



いつものように、イベントには以下が含まれます。



  • 設置セクション。全員が椅子、テーブル、ソケットを入手し、ローカルリポジトリを使用してWi-Fiネットワークにアクセスします。 誰もがNizhny Novgorod Linux User Groupの経験豊富なユーザーから支援を受けることができます（バッジを持っている人を探してください）
  • プレゼンテーションセクション。 ここでは、終日、Linuxおよびフリーソフトウェアに関連するさまざまなトピックに関するプレゼンテーションとマスタークラスが開催されます。
  • デモゾーン。 ここでは、すでにLinuxがインストールおよび構成された複数のコンピューターが自由に利用できます。 Linuxの使用について試してみたい、または質問したい場合は、安全にそこに行くことができます
  • 3Dシュニックコーナー。 ここでは、終日、Blenderでの3Dモデリングとアニメーションテクニックを紹介します。 マスターグルミハイルソルヤノフ（別名クランティス）




スピーチの1つは、Nizhny Novgorod Linux User Groupの活動に関するストーリーと、計画されているプロジェクトに関するストーリーです。
