3CXからのクリスマスプレゼント

クリスマスと新年の前に、3CXはパートナーに素晴らしい贈り物を提供します。 金銭的には、アクティブなパートナーへの贈り物の額は1000ユーロ以上です！

まず、今年の11月から12月に3CX電話システムの永久ライセンスを少なくとも1つ販売しているパートナーには無料で提供されます。



  • さらなる再販のために12か月間ホスティングする4SC Standard Editionでホスティングするためのライセンス。 175ユーロに相当
  • 永久ライセンスVoice Application Designer 。 875ユーロに相当。


Voice Application Designerは3CX PBXビジュアルプログラミング環境であり、最小限のVoIPエクスペリエンスを持つパートナーが、クライアント用の音声セルフサービスポータルなどの強力な音声アプリケーションを作成できることを思い出してください。 その後、パートナーはこれらのアプリケーションをクライアントのサイトまたは仮想PBXサーバーにリモートで展開できます。



注意してください！ これらのギフトを受け取るには、CET 12.24.15の午前11:00（カトリッククリスマスの前夜）までにセールを完了する必要があります。 すべての販売ライセンスは、この時点までにアフィリエイトアカウントに入金する必要があります。



第二に、すでにご存知のように、今では顧客に仮想PBX 3CX電話システムホスティングサービスを提供できます。 この機能自体は、最小限のコストで新しい仮想PBX市場に参入する素晴らしい方法です！



しかし、私たちはさらに与えます！ ホスティングPBXの機能を顧客に簡単に示すことができるように、パートナーの状態に対応する既存のNFRライセンスに加えて、新しいNFRライセンスが割り当てられます。 このNFRライセンスを使用すると、3CXホスティングで3CX 4SC Pro Editionシステムをホストできます（誰もが所有しているわけではない独自のホスティングサーバーと混同しないでください）。 これで、古いNFRキーを独自のニーズに使用して、クライアントにクラウドバージョンをデモンストレーションできます。 またはその逆。



3CXアフィリエイトパネルにアクセスして、これらのライセンスをアクティブにします。



競合するホスティングプラットフォームとは異なり、3CX Hosted PBXは専らパートナー指向です。 3CXは、その登録データと設定が常にパートナーによって所有されているため、顧客の保持を保証します。 PBXの各仮想エンティティは完全に仮想化され、他から分離されています。 これにより、3CXは各仮想PBXの高い信頼性を保証できます。1つのエンティティに障害が発生した場合、競合ソリューションの場合のように、問題が他のクライアントに影響することはありません。



NFRライセンスに関する追加情報：



  • NFRキーによってアクティブ化された仮想PBX 3CXは、使用開始後1か月が経過しても動作を停止しません。
  • すべてのシルバーパートナーパートナーはNFRキーを受け取ります。
  • すべてのアフィリエイトパートナーは、12か月間NFRキーを受け取ります。 その後、パートナーがシルバーパートナーのステータスを受け取らない場合、キーは無効になります。
  • 3CX電話システムv14用に設計されたNFRドングル
  • 試用版はこのNFRキーで更新できません。
