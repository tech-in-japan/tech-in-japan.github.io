剖検Meizu MX3

ハブラリュディ、おめでとうございます！ 最後に、別の剖検資料を紹介する準備が整いました。今回は、Meizu MX3の最高の音楽スマートフォンの「内部」を理解します。デバイスコンポーネントを個別に調べ、ケース内での配置方法を見て、いくつかのアセンブリ機能に注目します。









Meizu MX3の機能



  • オペレーティングシステム ：Android 4.2とFlyme 3 Shell
  • 通信 ：GSM / GPRS / EDGE / 3G（WCDMA）/ HSPA + 850/900/1800 MHz（2G）、850/2100 MHz（3G）
  • 画面 ：Sharp New Mode 2、対角5.1インチ、解像度1800×1080ピクセル、PPI 415、コントラスト1000：1、最大輝度450 cd / m、輝度2048階調、TOLテクノロジー（単一モジュール-ガラス+センサー）、省エネPSRテクノロジー、保護ガラスゴリラガラス2
  • プラットフォーム ：Samsung Exynos 5410、8コアプロセッサ（4コア1.6 GHz Cortex-A15および4コア1.2 GHz Cortex-A7）、28 nmプロセステクノロジーLP HKMG
  • GPU ：PowerVR SGX 544MP3（533MHz、3コア）。
  • RAM ：2 GB、DDR3
  • データストレージ用メモリ ：16/32/64 GB
  • カメラ ：8メガピクセル（BSIセンサーSony Exmor、IMX179 CMOSチップ、感光性エレメント1.4μm）、撮影速度：20フレーム/秒。 5レンズ、F / 2.0絞り、アングル74°、インスタントフィルター機能、ビデオ録画30 fps 1080p、パノラマ撮影、ジャイロフォーカス、HDR機能、顔検出、ジェスチャーショット
  • フロントカメラ ：2MP（ソニーセンサー）、ビデオ録画30 fps 1080p。
  • マルチメディア ：Wolfson WM5102サウンドチップ、Dirac HDテクノロジー、3つのマイクを備えたSmart PAスピーカーシステム、富士通4チャンネル画像処理チップ
  • サポートされている形式 ：FLAC、APE、AAC、MKA、OGG、MP3、MIDI、M4A、AMR、WAV、MP4、3GP、MOV、MKV、AVI、FLV、MPEG、M2TS、TS、JPEG、PNG、GIF、BMPなどその他
  • インターフェース ：Wi-Fi 802.11 b / g / n、2.4GHz / 5GHz（最大速度150 Mbps）、Bluetooth 4.0、microUSB（USB-OTGサポート）、NFC（MEIZU MX3 16GBを除く）、ヘッドフォン用3.5 mmヘッドセット
  • ナビゲーション ：GPS / A-GPS / GLONASS、電子コンパス
  • オプション ：タッチセンサー、光センサー、重力センサー、電子コンパス、ジャイロスコープ
  • バッテリー ：2400 mAh、ソニー、電源管理用デュアルチップ
  • 寸法 ：139.0 x 71.9 x 9.1 mm








ご存知のように、わずかに変更されたデザインにもかかわらず、Meizu MX3の主要な要素のレイアウトは、Meizu MX2と同じままです。 スマートフォンの上部には電源ボタンと3.5 mmオーディオジャックがあり、microUSBポートは下にあり、音量調整ロッカーは左端にあります。

ケース自体は特別な変更を受けていません。 前と同じように、それは単一のステンレス鋼から切り取られた金属フレームに基づいています。 背面カバーは2層のプラスチックでできており、当社の以前のデバイスでもよく知られています。 カバーを固定する方法は同じままです-デバイスの左端にある特別なブラケットを使用してカバーを固定します。 しかし、プラスチック製のクリップは少し変わっています。 以前にそれらが蓋の縁に沿ってのみ配置されていた場合、中央により近い位置にあるいくつかの固定具がそれらに抽出されました。







Meizu MX2と同様に、新しいMX3では、すべての「詰め物」が特別な保護カバーの下に隠されています。 ケーシング自体は単一の部品で構成され、全体がプラスチックでできています。 このため、デバイスのサイズが大きくなったにもかかわらず、スマートフォンの重量はほぼ同じままです-Meizu MX2との違いはわずか1グラムです。 さらに、ケーシングの取り外しがはるかに簡単になりました。このため、11本のネジを外すだけで十分です。







MX2と比較した2番目の顕著な違いは、カメラの保護ガラスとフラッシュLEDの配置が異なることです。 以前にスマートフォンの背面カバーにあった場合は、保護ケースに移されています。 MX3ではさまざまな色の安価な交換用キャップがすでに入手可能であるのに対し、MX2ではケースのこの部分がより高価だったのはこのためです。







カバーを取り外した後、バッテリーを取り外すことができます。 主なことは、上部に沿って通るフラットケーブルをメインボードから外すことを忘れないことです。 同様のループがバッテリーの右上隅にあり、NFCアンテナの接続に使用されます。







バッテリー用の特別なマウントは提供されていません。両面接着テープで本体に接着されています。 そして、それは多くの努力なしで外れますが、その過程で誤って損傷する可能性がありますので、注意する必要があります。







スマートフォンの下部には、ホームボタンセンサーとマイクロUSBポートを備えた小さなボードがあります。 同社の以前のスマートフォンと同様に、センターボタンセンサーの表面は蛍光塗料の層で覆われています。 このソリューションのおかげで、スマートフォンが完全にオフになっていてもボタンが点灯します。









バッテリーの下で動作する長いループを使用して、メインボードに接続します。







ボリュームロッカーの横には振動モーターモジュールがあり、2つのネジに取り付けられています。







スマートフォンの右上隅には、3.5 mmジャックのある白いオーディオモジュールがあります。 また、わずか2本のネジで取り付けられ、数秒で取り外されます。







モジュールの左側には、信号がメインボードに送信される5つの接点の列があります。









最も興味深い部分であるメインボードに移りましょう。 Meizuでおなじみの黒いPCBがベースとして使用されますが、その形状は大きく変わりました。 下の写真は、Meizu MX3長方形ボードとMeizu MX2 L字型ボードを示しています。







ご覧のように、ボードの外側はアルミニウム製の保護シールドでほぼ完全に覆われています。これは特別な機器なしでは簡単に取り外すことができませんでした。 しかし、その前面はほぼ完全に開いており、保護スクリーンは右上隅のチップのグループのみを覆います。 その上には、カメラモジュールからボード自体へのループがあります。









ところで、カメラについて。 Meizu MX3は、8 MPモジュール（Sony Exmor BSIセンサー、IMX179 CMOSチップ、1.4μm感光性エレメント）、および5つのレンズを使用します。 開口F / 2.0、角度74°。

すべての保護スクリーンを取り外したら、ボード上のメインモジュールとチップの分析に進むことができます。 スマートフォンの画面に面している正面から始めましょう。







まず、ボードの下部に注意してください。 これが、スマートフォンの「心臓」である場所です-Exynos 5 Octaとしても知られるSamsung Exynos 5410 SoC。 同様のソリューションが、Samsungの旗艦であるSamsung Galaxy S4 GT-I9500に選ばれたことは注目に値します。

これは、世界で最初の8コアCPUの1つです。 4つの経済的なARM Cortex-A7と4つの強力なARM Cortex-A15の8つのコアに基づいており、同時に動作できるのは4つ以下です。 これがまさに、big.LITTLEアーキテクチャのすべてであり、生産的で経済的な4つのコアを時間通りに切り替えることで消費電力を削減できます。 彼に関する詳細情報は、サムスンの公式ウェブサイトで入手できます。



SoCに関する簡単な情報



グラフィックスを使用するには、クロック周波数が533 MHzの3コアグラフィックスプロセッサSGX544MP3が使用されます。 その機能のうち、DirectX 9、OpenGL ES 1.1、およびOpenCL 1.1のハードウェアサポートに注目する価値があります。







ボードの右隅は、microSIMカードトレイをほぼ完全に占有します。 プロセッサとプロセッサの間には、電源管理を担当する「DC-to-DCコントローラ」マキシムMAX776という小さな明るい黒のチップがあります。







ボードの左側に近いところに、Texas InstrumentsのTPS65132という10インチまでのTFT-LCDパネルを操作できる非常に小さな「ディスプレイ用バイポーラ電源」チップがあります。



チップの説明



しかし、その左側のチップははるかに大きくなっています。 これは、カメラの操作を担当するISPまたは画像信号プロセッサです。 過去のデバイスと同様に、Meizuは富士通製のISPチップを使用することにしました。 それにもかかわらず、チップ自体はわずかに異なります-前の2つのスマートフォンのようなMBG048ではなく、S20ABです。







ボードの上部に移動しましょう。 カメラモジュールのカットアウトの左側には、アップグレードされた別のチップがあります。 ワイヤレスインターフェイスの動作を担当するBroadcomチップについて話しています。 Meizu MX2がBroadcom BCM4430をこれらの目的に使用した場合、Broadcom BCM4334をMX3にインストールすることにしました。 残念ながら、BCM4430に関する情報は非常に少ないため、両者の違いを正確に言うことは困難です。 BCM4334の説明に満足するだけです。 彼によると、このチップの新しいバージョンは以前のものよりも消費電力が少なく、Wi-Fi 802.11a / b / g / n、Bluetooth 4.0 + HS、Wi-Fi DisplayやWi-Fi Directなどのテクノロジーをサポートしています。 Broadcom BCM4334のもう1つの機能は、FMラジオのサポートです（ただし、スマートフォン自体にはありません）。



簡単なチップ情報



カメラモジュールの切り欠きの右側に配置されたチップのグループを見てみましょう。 ボードのこの部分は、アルミニウムのシールドで覆われていました。

「パワーアンプモジュール」として、2G \ 3Gネットワ​​ークのスマートフォンを担当するチップ、RF6260が使用されます。 興味深いことに、RF6260は4Gもサポートしていますが、これはスマートフォン自体では使用できません。 同様のチップが、同社の最後の旗艦であるMeizu MX2で使用されました。



チップの説明



その隣にあるのは、青色のモジュール-Epcos D602ハイパスフィルターです。







中心に少し近いのは、Intel PMB5712 RFトランシーバーです。 ボードの反対側にあるPMB9811モデムと共に、Intel XMM 6260プラットフォームの一部です。







今こそ、ボードを裏返し、その裏側で何が面白いかを見る時です。 前回同様、下から始めましょう。







左隅にあるのは、電力関連チップのグループです。ほとんど目立たない「DC-DCコンバーター」RF Micro Devices 6560と、わずかに大きい「統合コントローラー」マキシムMAX778です。







次のモジュールAtmel 32UC3は、ボードのほぼ中央に配置されています。 これは、最大50 MHzの動作周波数を備えたAVR32 UC RISCプロセッサに基づく32ビットシステムオンチップマイクロコントローラーです。 製造業者によると、このマイクロコントローラの主な重点は、低消費電力で高性能を実現することでした。 詳細については、PDFファイルを参照してください。



チップの説明







右隅全体が、識別しやすい東芝ブランドのボリュームメモリモジュールで占められています。 興味深いことに、以前ネットワークに表示されていたボードの写真では、メモリモジュールはサンディスクのものでした。 同じモジュールが同社の以前のスマートフォンで使用されていました。







今回動作する別の東芝製メモリモジュールは、ボードの上部の左隅にあります。 私たちの場合、マーキングはできませんでしたが、中国のリソースbbs.meizu.cnによると、これはDDR +フラッシュモジュールToshiba Y8A0A111434KAです。 残念ながら、詳細な仕様を見つけることはできませんでした。







右側にはIntel PMB9811モデムがあります。 既に述べたIntel XMM6260プラットフォームの一部であることを思い出させてください。 このプラットフォームは新しいものではなく、実績のある信頼できるソリューションであることに注意してください。 元々はインフィニオンによって開発され、X-GOLD626と呼ばれていました。 それから、Samsung Galaxy S2などの有名なスマートフォンで使用されました。 その後、IntelがInfineon Technologiesのワイヤレス部門を買収した後、このソリューションはIntel XMM6260と名付けられました。



チップの説明







カメラの切り欠きのすぐ上に、別のBroadcomチップがあります。 このチップは、第3世代のGNSSソリューション（Global Navigation Satellite SystemまたはGlobal Navigation Satellite System）に属し、GPS、GLONASS、QZSS、およびSBASの4つのナビゲーションシステムを同時に使用できます。 Broadcom BCM4752は、位置決めを高速化するために、加速度センサー、ジャイロスコープ、気圧センサーなど、多くの追加センサーとの連携をサポートしています。



簡単なチップ情報







ボードの残りの隅のほぼ全体がサウンドの処理専用です。 Meizu MX3が音楽スマートフォンであることを考えると、ボードのこの部分は私たちにとって特に興味深いものです。

まず第一に、オーディオチップの大きな正方形が注目を集めています。 お気づきかもしれませんが、Meizuはほぼ常にWolfsonオーディオチップを好みます。 たとえば、Wolfson WM8958EはMeizu MXおよびMeizu MX2にインストールされました。 スマートフォンは、Wolfsonチップも搭載されているMeizu MX3の最後のフラッグシップでしたが、新しいモデルであるWolfson WM5102を搭載しています。 会社の代表者によると、このチップのおかげで、昨年のフラッグシップと比較して、MX3の音質が大幅に改善されました。 ヘッドアンプの出力は3倍に増加し、高調波歪みのレベルは0.002％に減少し、信号対雑音比は113 dBに増加しました。 このチップのもう1つの特徴は、Dirac HDテクノロジーのサポートです。

ウォルフソンは、ポータブルオーディオチップの市場で長い間知られています。このチップは、さまざまなメーカーのモバイルデバイスに搭載されています。 特に、Wolfson WM5102はSamsung Galaxy S4 16Gb GT-I9500で使用されています。 このチップの詳細については、Wolfson Microelectronicsの公式Webサイトをご覧ください。

チップの説明







最後に、ボードの冷却について少し説明したいと思います。 会社の代表者がすでに述べたように、熱伝達はプリント回路基板の両側で行われます。これは、熱伝達が双方向に行われるためです。 サーマルペースト、グラファイト、ニッケル銅合金コーティングを使用することで、効率的な放熱が可能になります。

メインボードの下には、光センサーと近接センサーを備えたモジュール、スピーカー、フロントカメラモジュール（2MP、Exmor R）があります。











これは、完全に分解されたMeizu MX3がどのように見えるかです（蓋は舞台裏に残りました）。







スマートフォンのMeixu MX3は革命と呼ぶことはできませんが、以前のモデルの品質の更新です。 一方で、スマートフォンはMeizu MXファミリーの伝統を引き継いでいますが、一方で、多くの改良が施されています。 コンポーネントを選択する際、会社は古いパートナーに忠実であり続け、デバイスの多くのコンポーネントが大幅に更新されたことがわかります。 さらに、今年はオーディオ部分に特に重点が置かれました。 スマートフォンの内部設計も改善されています。 これは、より実用的で便利な保護カバーであり、モジュールの別のレイアウトです。 フラッシュLEDとカメラの保護ガラスが、背面カバーからスマートフォンの内側に移動することは特に喜ばしいことです。 おかげで、カバーの交換がずっと簡単になります。
