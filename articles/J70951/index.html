Microsoft Security Essentialsウイルス対策の最終リリース

マイクロソフトは、無料のウイルス対策Security Essentialsの最終リリースが今朝（太平洋時間）9月29日、つまりモスクワ時間19：00-21.00に行われることを公式に確認しました。 現時点では、Windows XP、Vista、およびWindows 7の4.7 MBディストリビューションが公式Webサイト（上記のリンク）に掲載されます。ベータテスター向けのスタブがあります。



ウイルス対策の開発にはほぼ1年かかり、6月に限定ベータテストが開始されました。 公式サイトからのダウンロード数は75,000に制限されていましたが、アンチウイルスはトレント経由で配布されました。



無料のウイルス対策ソフトウェアは、2009年6月に閉鎖された有料のWindows Live OneCareサービスに代わるものです。



市販のウイルス対策プログラムの製造元は、マイクロソフトとの競争を恐れていないことをすでに述べています。 例として、彼らはスパイウェアを見つけるための無料のWindows Defenderプログラムを引用しています。



一部の専門家は一般に、Microsoftがコンピューターセキュリティの分野で少なくともいくつかの成功した製品を作成できることを疑っています。ここではまったく経験がありません。
