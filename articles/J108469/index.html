Netbeans 7.0ベータリリース





Netbeans 7.0ベータ版をダウンロード



NetBeans IDE 7.0には、JDK 7、Oracle WebLogicサーバーとの統合、Oracle DatabaseおよびGlassFish 3.1のサポートが含まれています

追加機能には、Maven 3およびHTML5編集サポート、Swing GUI開発を改善するための新しいGridBagLayoutデザイナー、スタンドアロンJava EEアプリケーションの実行とデバッグのサポートの強化、およびJavaエディターの改善が含まれます。





Jdk 7

  • Project Coinのサポート
  • コード補完とヒントの改善




Weblogicサーバー

  • 合理化された高速なWebLogicデプロイメント
  • 新しいランタイムは、アプリケーションとリソースを表示します
  • JSFはサーバーライブラリと統合されています




Oracleデータベース

  • 接続ウィザードの簡素化
  • JDBCドライバーのインストール手順
  • 編集および投稿手順




Glassfish

  • Glassfish 3.1
  • 再起動して、リモートGlassFishを表示します
  • ホストされたアプリケーションの有効化と無効化




Java

  • Maven 3のサポート
  • JUnit 4.8.2の統合とさまざまなJUnitの機能強化
  • JavaライブラリおよびプラットフォームのJavadocでのリモートURLのサポート
  • GridBagLayoutの新しいブラックジャックと売春婦のビジュアル




Java EE

  • CDI、REST、およびJava Persistenceのサポートの改善
  • Bean検証のサポート
  • PrimeFacesライブラリを含むJSFコンポーネントライブラリのサポート
  • コード補完、リファクタリング、ツールチップなど、JSF Expression Languageサポートの改善




ウェブ言語

  • HTML5サポート
  • JSONフォーマット




Php

  • PhpDocの作成
  • リファクタリング：名前の変更と安全な削除
  • PHP 5.3-エイリアスのサポート（使用？）




C / C ++

  • 既存のユーザーライブラリからプロジェクトを簡単にインポートする
  • 削除されたユーザーファイルの新しいプロジェクトタイプ




NetBeansプラットフォーム

  • 「レイヤー内のアクション登録」を作成するための注釈
  • プロファイラーのパフォーマンス向上とテスト統合
  • NetBeans APIのいくつかの変更




雑多

  • ワードラップ
  • IDEと他のプログラムを切り替える際の、外部の変更の邪魔にならないチェック




NetBeans IDE 7.0の安定バージョンのリリースは、2011年3月に予定されています。
