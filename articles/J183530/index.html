HPエントリーレベルストレージアップグレード

2013年6月、ストレージシステムにおけるHPの新製品の次のリリースが行われました。

今回、データストレージシステムのエントリレベルセグメントが影響を受けました-新しいMSA 2040モデルが発表されました。



世界には、現在のストレージシステムで問題を抱えており、手頃な価格で生産性の高いアレイを必要とする小規模企業が約3,000万社あります。

これは、幅広い機能と本格的なスケーラビリティを備えたエントリーレベルのシステムとしての地位を確立した第4世代のアレイです。



何が変わった？

名前が変更されました-現在、アレイはHP MSA 2040と呼ばれています。P2000G3アレイは、もう1年間注文可能です。

アレイコントローラーのパフォーマンスが向上しました-現在、各コントローラーにはDDR3高速メモリ上に4GBのキャッシュがあります。

SSDドライブがサポートされ、システムは200 GB、400 GB、800 GBのMLCドライブをサポートするようになりました。 ディスクには摩耗警告システムが装備されており、故障したディスクの交換を予測できます。 メディアの耐用年数は5年に延長され、各ディスクには30,000回の書き換えサイクルがあり、さらに、メディアのDWPD（1日あたりのデータ書き込み）パラメーターは1日あたりの容量の10倍です。 400 GBのメディアに5年間記録されるデータの量は7 PBです。 システムごとの最小ディスク数は4、コントローラーごとに1つのミラーペアです。







MSA 2040の場合、ランダムロードの第3世代と比較したパフォーマンスは、読み取りで2.8倍、シーケンシャルロードで4.1倍（シーケンシャル書き込み）です。 ユーザーは50,000〜80,000 IOPの値を使用できるようになりました。



図1さまざまな負荷に対する既存のストレージシステム（左）とMSA 2040（右）のパフォーマンスの比較



このようなパフォーマンスを得るには、多数のスピンドルを追加する必要がないことに注意してください。これは、複数のSSDメディアを追加することで行うことができ、ソリューションのコストに影響します。

より高い帯域幅のポートも追加されました。FC16Gb、FC 8Gbポートを備えたモデルが最初に利用可能でしたが、年内にはFC / 10GbおよびSAS 12Gbストリームを備えたコンボコントローラーが発表されます。

サポートされるSFF SASディスクの数が増えました-アレイは199個のSFFディスクに拡張できますが、LFFディスクの数は同じままです-96。

アップグレードオプションは残っています-MSA G3コントローラーは新しい世代のモデルに置き換えることができ、既存のシェルフは新しいコントローラーにカスケードできますが、新しいタイプのディスクは6G SASシェルフでサポートされます。

上記に加えて、次のマイクロコード更新では、シンボリューム（シンプロビジョニング）およびデータティアリング（LUNティアリング）のサポートを主張しています。

データ（レプリケーション、スナップショット）をコピーするための高度な機能は完全に保持されています。アレイをインストールすると、これらのライセンスは60日間無料でアクティベートされ、64のスナップショットがあり、512コピーの拡張ライセンスを購入できます。

アレイは、Windows Standardベースのストレージ管理サービス（SMI-S）を介したWindows Server 2012からの管理を直接サポートします。これにより、Windows環境が支配的な小規模企業での作業が大幅に簡素化されます。 次の機能は、SMI-Sを介してサポートされます：ストレージプール、仮想ディスク、マッピング、シャンショット/クローン。
