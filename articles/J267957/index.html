Intel RealSense SDKをMacにインストールする





Intel RealSenseテクノロジーセミナーやハッカソンに参加するには、一部の開発者はMac OSラップトップにIntel RealSense SDKをインストールする必要があります 。 時間がかかることがあるので、Boot Campを使用してMacでIntel RealSenseカメラを使用してサンプルSDKアプリケーションを実行する方法を示すこのステップバイステップガイドを公開することにしました。



はじめに



まず、以下が必要です。

  • Mac OS X *ヨセミテ
  • Apple Boot Camp 5.1.4以降（このプログラムはMacにプリインストールされています）
  • Intel RealSense F200カメラおよび/またはIntel RealSense R200
  • Intel RealSense SDK （ SDKをダウンロードするには登録が必要です）
  • Windowsをロードするための少なくとも6 GB USBドライブ*
  • 64ビットバージョンのMicrosoft Windows 8.1オペレーティングシステムを使用したISOイメージ（クラシックモード）
  • Microsoft Media Feature Pack
  • Microsoft Visual Studio * 2010–2013 Service Pack 1以降
  • USB3ポート


第4世代以降のIntel®Core™プロセッサー（Haswellファミリー）がコンピューターにインストールされているかどうかを確認するには、ターミナルを開いて次のコマンドを入力します。



sysctl -n machdep.cpu.brand_string









このテキストは画面に表示されます。



Intel®Core（TM）i5-5257U CPU @ 2.70GHz



Intelプロセッサのネーミングシステムがわからない場合は、i3 / i5 / i7の後に4桁の数字を見つけます。 この番号の最初の数字が4以上の場合、すべてが正常です。 さらに、適切なリンクをクリックして、第4世代および第5世代のすべてのプロセッサーのリストを表示します。





Intel®RealSense™F200カメラ



Boot CampでWindowsをインストールする



Cmd +スペースバーを押すか、デスクトップの右上隅にある虫眼鏡をクリックしてSpotlight *を起動し、「Boot Camp」という単語を検索します。 「Boot Camp Assistant」行が検索結果ウィンドウに表示されます。





Boot Campの検索* Spotlightを使用したアシスタント*



Boot Campアシスタントは、WindowsインストールファイルをISOファイルまたはUSBドライブから転送します。 この時点で、WindowsオペレーティングシステムがUSBドライブからWindowsの最新ドライバーと共にMacにインストールされます。 このアプリケーションは、Macの現在のシステムパーティションのサイズを変更し、Windows用のパーティションを作成するのに役立ちます。





Boot Camp * Assistantを使用してWindowsをインストールします



最初に、完了したいタスクを選択します。 これらすべてのボックスをチェックすることをお勧めします。 Boot CampでUSBドライブをすでに作成しているか、コンピューターのディスクをいくつかのパーティションに分割している場合は、対応するボックスのチェックを外してプロセスを高速化します。 このアプローチは通常、複数のMacコンピューターにWindowsをインストールする予定で、USBドライブを既に準備しているユーザーが使用します。





Boot Camp *アシスタントでタスクを選択する



次に、ISOファイルまたはUSBフラッシュドライブを選択します。 USBフラッシュドライブを挿入して選択します。 このドライブの内容はすべて削除されるため、すべての重要なファイルのコピーを作成してください。

Windowsインストールドライブが作成され、「Windowsファイルをコピーしています」という通知が表示されます。 このプロセスには時間がかかります（ドライブの速度によって異なります）。 この場合、進行状況インジケータが定期的に移動を停止する場合があります。 そのような場合は、我慢してください。





Windows *ファイルをUSBドライブにコピーする



プロセスを完了し、USBインストールメディアを作成すると、「パーティションの作成」画面が表示されます。 これで、コンピューターのシステムドライブをMac OS X用とWindows用の2つのセクションに分割できます。 WindowsおよびOS Xに割り当てる容量を指定できます。Windowsパーティションには少なくとも32 GBを割り当てることをお勧めします。 コンピューターに複数のハードドライブがある場合、そのうちの1つをWindowsでのみ使用できます。

残念ながら、このプロセスを完了すると、Boot Campアシスタントでパーティションのサイズを変更することはできなくなります。 これを行うには、サードパーティのプログラムを使用する必要があります。





Windows *ディスクのパーティション分割



Windowsはインストールプロセスを続行します。 その後、Boot Campインストーラーが起動し、Windowsおよびサポートプログラムに必要なすべてのハードウェアドライバーがインストールされます。

Boot Campインストーラーがすべてのタスクを完了したら、ポートからUSBドライブを取り外すことができます。 別のMacでインストールプロセスを繰り返す予定がない場合、このUSBドライブは不要になります。



NTFSエラー



Boot CampはWindowsディスクのパーティションをFAT32形式にフォーマットします。Windows8.1はNTFSファイルシステムのパーティションのみをサポートするため、インストール中に次のエラーメッセージが表示される場合があります。

「ハードドライブのこの領域にWindowsをインストールすることはできません。 NTFSとしてフォーマットされたパーティションにWindowsをインストールする必要があります。

BOOTCAMPセクションを選択します。 formatコマンドを使用して、このディスクパーティションをNTFSに再フォーマットし、Windowsのインストールを続行します。







Boot CampでIntel®RealSense™SDKをインストールします



Boot Campを使用してIntel RealSense SDKをインストールするプロセスは、Windowsにインストールするプロセスと同じです。

手順1. Intel RealSenseカメラをMacのUSB3ポートの1つに差し込みます。

ステップ2. Intel RealSense SDKダウンロードページの指示に従って、仮想カメラドライバーとDepth Camera Managerサービスを含むIntel®RealSense™Depth Camera Manager（DCM）パッケージをインストールし、SDKをインストールします。





F200をMacbookのUSB3ポートに接続します*



ステップ3. Windows 8.1のバージョンによっては、 Media Feature Packはオペレーティングシステムの一部ではない場合があります。 必要に応じてこのパッケージをダウンロードし、インストールを続行してください。 Intel RealSense SDKをインストールするには、Media Feature Packが必要です。

ステップ4. SDKをインストールした後、サンプルアプリケーションを実行してみます。 それらにアクセスするには、デスクトップ上のIntel RealSense SDK 2014フォルダーのショートカットを使用します。 SDKをデフォルトの場所にインストールする場合、フォルダーC：\ Program Files（x86）\ Intel \ RSSDK \ sampleが使用されます。



Macbook Air *ラップトップでSDKからサンプルを実行する

手順5. Visual Studioを使用して、コードサンプルを使用および変更できます。

できた！ Intel RealSenseテクノロジーがMacで実行されるようになりました。



それだけです



ご覧のとおり、MacコンピューターでIntel RealSenseテクノロジーを使用するには、特別な作業は必要ありません。AppleBoot Campアプリケーションを使用するだけです。

Parallels、VMWare Fusion *、Virtual Boxなどの他のプログラムを使用して仮想マシンをサポートできますが、USB3のシミュレーションはコンピューターで実際のOSを実行することとは大きく異なります。 簡単に言えば、インストールは成功するかもしれませんが、カメラは起動しません。
