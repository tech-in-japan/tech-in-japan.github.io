DevCon 2013：会議プログラムの発表





親愛なるソフトウェア開発者およびテスター、 DevCon 2013カンファレンスの参加者。 主催者を代表して、2日間の会議プログラムをご紹介します。このプログラムには、6つの並行トラック、6つの専用マスタークラス、円卓と「コーディングナイト」と呼ばれるハッカソンを含むイブニングプログラムが含まれます。



http://www.msdevcon.ru/scheduleで 、現在の会議スケジュールを見つけることができます。



以下に、DevCon 2013プログラムの詳細の一部について説明します。



主なプログラム



会議のメインプログラムには、全体報告書、基調講演、両日のミニセッションが含まれます。 合計すると、会議の参加者とオンライン視聴者は、 67種類のレポートから選択できます。



注意してください！ 会議のメインプログラム全体が記録され、リアルタイムでオンラインで放送されます！ 運が悪く、DevCon 2013に個人的に参加できない場合は、5月29日と30日に公式ウェブサイトhttp://msdevcon.ruにアクセスしてオンラインセッションをご覧ください 。



今年、私たちは会議プログラムと報告書をより奥深いものにすることを決めました。 したがって、DevCon 2013レポートの3分の1以上（半分に近い） が300以上のレベルになります。



DevCon 2013の初日



会議は2時間の全体レポートで始まり、その枠組みの中で、伝統によれば、前回の会議からの昨年の開発者向けの会社のすべてのニュースの概要が示されます。 本会議の参加者は、 Jonas Persson 、 Jason Zander 、 Alexander Lozhechkin 、およびマイクロソフトの他の同僚です。



注意してください！ 会議の初日に、Microsoftのコーポレートバイスプレジデントであり、以前はVisual Studioツールの開発全体を指揮し、現在はWindows Azureプラットフォーム全体を担当するJason ZanderとのQ＆Aセッションが開催されます。 DevCon 2013の参加者にとって、Jason Zanderとの会議は、最高レベルのMicrosoftディレクターと会い、彼にすべての質問をするユニークな機会です。



初日、全体レポートとともに、会議の参加者とオンライン視聴者は、ソフトウェア開発のすべてのトピックに関する25のセッションに参加できます。クライアントおよびモバイルアプリケーション開発、Webおよびクラウドアプリケーション開発、効果的な開発ツール、ALMを使用した開発プロセスの構築、エンタープライズアプリケーション開発。



DevCon 2013の2日目



会議の2日目のメインプログラムは、各30分の興味深いミニプレゼンテーションで始まります。 パートナーや招待講演者から、ソフトウェア開発ツールに関する多くの興味深いことがわかります。



2日目はテクニカルレポートでいっぱいです。 30時間のレポートの間に、会議の参加者とライブ視聴者は、ソフトウェア開発の最新のアプローチと最新のツールについて多くの興味深いことを学ぶことができます。



注意してください！ NDA（What is Hekaton ...）というラベルが付けられたセッションの1つは、排他性と機密性のため、オンラインで放送および記録されず、その参加者は秘密保持契約に署名する必要があります。



会議の最終は公式の閉会式であり、その枠組みの中で結果が要約され、大会の勝者が発表され、お土産が配布されます。



追加プログラム



おそらく、DevCon 2013カンファレンスの最も「おいしい」部分は、直接会場に到着した参加者専用の追加プログラムです。 プログラムのレポートおよびマスタークラスはオンラインで録画または放送されず 、直接訪問することができます。



DevCon 2013の初日



会議の初日に、3つのマスタークラスがメインプログラムと並行して開催されます。



  • ガイダール・マグダヌロフによるマスタークラス 「アイデアから成功へ」
  • Dmitry Soshnikov、Roman Dushkin、Andrey Laptevによる関数型プログラミングスクール
  • Dmitry Andreevによる集中的なソフトウェアテストトレーニング


初日の夕方、参加者は驚きのある楽しい夜のプログラムを見つけます。追加のプログラムのテーブルを密かに準備し、ラウンドします。



  • ラウンドテーブルJetBrains。 モバイルおよびWebアプリケーション開発の手段サポート
  • クラウドラウンドテーブル：決定の第2の風
  • ロシアの円卓スタートアップ：神話と現実


興味深い参加者がラウンドテーブルに期待されています。たとえば、JetBrains .NET DevelopmentのヘッドであるIlya Ryzhenkov氏です。



DevCon 2013カンファレンスのもう1つの注目すべき活動は、 コーディングナイトハッカソンです。会議参加者は、コーディングのために一緒になって、実際に日中に得た知識を実際に適用できます。 ハッカソンは、コードを書くための実用的な部分と講義を行う理論的な部分の2つの部分に分かれます。 さらに、ハッカソンの枠組みの中で、 Windowsストアアプリラボの実験室作業が行われます。 ハッカソンは、夜の9時から朝の3時まで続きます。



DevCon 2013の2日目



次のマスタークラスは、追加プログラムの一環として、会議の2日目に開催されます。



  • マスタークラス。 スクラム： Askhat UrazbaevとViktor StrelkovによるVisual Studio 2012の理論と実践 。
  • デニス・コルトゥノフとイゴール・ギレフによるデザイン学部「 非デザイナーのためのインターフェースデザイン」。
  • パオロ・サルバトーリによるマスタークラス。 Windows Azureを使用してモバイルアプリ用のクラウドバックエンドを構築する


2日目は会議の閉会式で終了します。



アチーブメントハンターズコンペティション



前述のとおり、DevCon 2013カンファレンスでは「Hunters for Achievements」コンテストが開催されます。 会議参加者は、メインおよび追加の会議プログラムの訪問レポートを含む、会議会場でのさまざまなアクションの成果（成果）を受け取ることができます。



合計で、数十の成果が計画されており、会議で「獲得」できます。 参加者は会場に到着すると詳細を知ることができます。 コンテストの優秀な参加者に賞を授与します。



じゃあね！



会議の主催者を代表して、私たちの会議に大きな関心を示してくれたすべての人々に感謝したいと思います。 おかげで、DevCon 2013の開始の2か月前にすべてのチケットを販売しました。そして、ほぼ毎日チケットのリクエストを受け取り続けています。



ただし、DevCon 2013のチケットを取得していない場合は、5月29日と30日に全体レポートとメインプログラムのレポートをリアルタイムで放送するオンラインブロードキャストに参加してください。 5月29日と30日に公式Webサイトhttp://msdevcon.ruにアクセスして、オンラインセッションを表示します。



会議で会いましょう！
