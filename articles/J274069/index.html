Perl 6開発者：すぐに使用できる言語





12月24日に、Perl 6の公式開発ニュースブログに祝辞が掲載されました。 開発者は、来るべきカトリックのクリスマスに皆を祝福し、待望の言語の成長がついに起こったという事実で。 実際、この言語は作業プロジェクトで使用する準備ができており、開発者は何も大幅に変更しないことを約束しています。



Perlの最初のバージョンがリリースされてから28年が経過し、現在最も人気のあるPerl 5のバージョンがリリースされてから20年以上が経過しています。 言語の作成者であり、その開発のリーダーであるラリーウォールが冗談を言っているように、第6バージョンはいつか第5を置き換えるかもしれません-約40年後に。



Perl 6のバージョンは10年以上前に発表されました。2004年にAmazonで発行されたこの「近日公開」言語に関する本をAmazonで購入することはできます 。 開発時間は、実際にはPerl 6が第5バージョンに基づいて作成された新しい言語であるという事実によって説明できます。 また、 6と5はC ++のC ++にすぎないという意見もありますが、それにもかかわらず、Perl 6のイデオロギーはそれをより現代的な言語と呼ぶほど進化しました。



ラリーウォールは、研究所の教師が最終的に同じ言語を使用して、学生にさまざまなスタイルのプログラミング（機能的、手続き的、オブジェクト）を教えることができることを望んでいます。



Perl 6ロゴは、楽しい蝶を選びました。 （冗談めかして） 今年10月の会議でWallが説明したように 、これは特に7歳の少女に言語を魅力的にするために行われました。 子供たちの間でのYPの良い広告の例として、彼はPythonを見るよう勧めました。



Wallは、この2つのバージョンの言語を統合する主なものは、言語を多かれ少なかれ交換したいと考えている志を同じくするプログラマーのコミュニティであることを指摘しました。 たとえば、Perl 6にはInline :: Perl5モジュールがあり、Perl 5にはInline :: Perl6モジュールがあります。
