Windows 7のすべての「秘密」

一般的な需要-記事「Windows 7の秘密のバンパーリスト」の翻訳。



ここでは、Windows 7がリリースされたことの素晴らしさについて紹介しませんでしたが、いずれにしても誰もが見逃し、「秘密」を自分で読みます。



1.ウィンドウ管理



Windows 7を使用すると、Windowsの操作がはるかに簡単になることをご存知でしょう。 ウィンドウは、対応する端にスライドするだけで画面の左半分または右半分に「アタッチ」できます。また、ウィンドウの下の境界線をダブルクリックして、ウィンドウを垂直方向に拡大/復元できます。 しかし、おそらくあなたが知らないのは、これらすべてのアクションがホットキーを介してアクセス可能であることです。

  • Win + LeftおよびWin + Right-ウィンドウを添付します
  • Win + UpおよびWin + Down-ウィンドウを最大化および復元/最小化します
  • Win + Shift + UpおよびWin + Shift + Down-ウィンドウを垂直に最大化および復元します


2.プロジェクター。



ラップトップにデスクトップをプロジェクターに表示させるために、特定のドライバーで苦しんでいるのにもううんざりしていませんか？ この場合、Win 7ではプロジェクターでの作業がはるかに簡単になります。Win+ Pの組み合わせにより、次のウィンドウが点灯します。



矢印をクリックするか、Win + Pの組み合わせを繰り返すことにより、「クローン」、「デスクトップの拡張」[外部デバイスへ]、または「外部デバイスのみ」モードに切り替えることができます。 これは、 displayswitch.exeアプリケーションからも実行できます。



さらに制御したい場合は、Mobility Center（Win + X）を介して取得できます。これにより、プレゼンテーションモードを有効にできます。このモードでは、メッセージクライアントが邪魔しないモードに切り替えたり、スクリーンセーバーをオフにしたり、ニュートラルな壁紙を設定したりできます。 d。 （ps Windows Vistaにはモビリティセンターもあります）

以前のツアーでは、ウィンドウをディスプレイに接続できることが示されています。 これに加えて、これにはホットキーがあると言えます。 Win + Shift + LeftおよびWin + Shift + Rightは、ウィンドウをディスプレイ間で移動しながら、左上隅から同じ距離に保持します。
