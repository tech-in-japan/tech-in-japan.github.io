無料の百科事典であるNupediaの10年

数か月間、Nupediaについて書きたかった。 最後に、それが起こった！ 今日は素晴らしい休日があります！

オープンソース運動の哲学は、業界全体に広がっています。 現在、Web百科事典の作成者は、一般的な知識の普及にその原則を適用したいと考えています。

編集長のラリーサンガー （写真）によると、 Nupediaのウェブサイトは木曜日に正式にオープンし、「世界最大の百科事典」になることを目指しています

PCWorld 、2000年3月上旬。

また、2000年のComputerworld Russiaからロシア語への圧縮翻訳もあります。



このNupediaは成功しなかったとすぐに言わなければなりません。 彼女は失敗しました。

同じ二人のウェールズとサンガーは、なんとか最大の「オープンソース」百科事典を作成することができ、それがウィキペディアになりました。

しかし、モスクワもすぐに構築されたわけではなく、まだ10年前に作成されたNupediaについてのものです。このサイトがなければ、Wikipediaも他の多くのサイト（OpenStreetMap、Wikimapiaなど）も、ウィキ哲学全体もなかったからです。新しい著作権「革命」も他の多くのものもありません。 これらすべてのおかげで、彼女はそれにもかかわらず、歴史上現在最も機能していないサイトの最大の1つとして認識されました 。



だから...昔々、 シャトルワースがまだ宇宙に飛んでおらず、 邪悪な青い文字「E」が世界を支配しており、FirefoxもBitTorrentプロトコルもYouTubeもFacebookも存在していなかった 、人類には、自由に使用できる関連性のある最新の参考書がありませんでした。 20世紀の終わりには、既存の少なくともいくつかの有用な百科事典は約1世紀前にリリースされて非常に時代遅れでした（ただし、著作権が失効したため再利用できました）か、商業会社の所有物でした（1993年以降に公開されました） CD Encarta -Microsoft、 Britannica - EncyclopædiaBritannica、Inc. ）、読み取りおよび引用以外の使用は禁止されています。 それらの多くは、印刷、ディスク、またはインターネットでのみお金で利用できましたが、有料のサブスクリプションでした。



ウェブの出現以来、多くの人々がネットワーク百科事典の開発を考え、何度も作成しようと試みてきました。 1993年、 リック・ゲイツはInterpediaプロジェクトを提案しましたが、彼は何かが現れる前に亡くなりました。 Everything2やOpen Siteなどのプロジェクトも立ち上げられましたが、それらについてはほとんど耳にしませんでした。



1999年、 フリーソフトウェア運動の創設者である リチャードストールマンは、独自の百科事典に対抗する自由で普遍的な百科事典のアイデアを策定しました。 おそらく彼のプロジェクトは野心的すぎて実現不可能だと思われた。 しかし、次の10年間は​​そうではないことを示しています。 10年で、世界最大の百科事典が作成されました。人々には選択肢と選択肢があります。 そして、この巨大な無料百科事典は完全に合法的に作成され、著作権を侵害せず、誰も許可を求めず、支払いもせず、RAO、 ミハルコフ、エルンストなどの前に屈しませんでした。 地球の住民は商用のプロプライエタリ百科事典を使用する義務を免除され、イギリス人のようなプロプライエタリ百科事典自体がウィキペディアとそのコミュニティの圧力の下で変更を余儀なくされました。



Nupediaの歴史



2000年の初めに、 ジミーウェールズ 、 ボミスと呼ばれるポルノサイト 「男性向けウェブポータル」（マキシムやプレイボーイのようなもの）の共同設立者兼CEOは、オンライン百科事典の作成を長い間夢見て、ついに夢に乗り出しました。彼女の編集長。 ウェールズは、証券取引所ですぐに金持ちになり、生涯に7つのお金を提供した後、1996年にIPOにヒントを得たIPO Netscapeを設立しました。

ラリーサンガーとジミーは90年代半ばからお互いを知っており（インターネットでチャットをしているようです）、2000年1月にウェールズがメールを書いて編集長として彼を雇ったのは彼のことでした。 2月、LarryはBomis社のあるサンディエゴに到着し、当初プロジェクトに科学者を採用する作業を開始しました。 ラリーはプロジェクト全体に完全に関与していたようで、ウェールズ自身は完全に何も関わっていなかったようです。



Bomisが出資および所有するNupediaは、2000年3月9日に正式に開始されました。 所有者は、Nupedia.comの広告から利益を得ることを期待していました。 それにもかかわらず、Larry Sangerは、百科事典が作成されたときに誰もが再入力できると（すぐにNupediaを1つのファイルにダウンロードすることもできる）、Linus TorvaldsとOpen Directory Project（dmoz.orgからのLinuxの成功による計画の現実を説明しました） したがって、最初からNupediaは無料の百科事典であり、最初にNupedia Open Content Licenseを使用してから、Richard StallmanとFree Software Foundationの要請でGNU FDLに切り替えました。 しかし、「 自由 」にもかかわらず、Nupediaはwikiサイトではなく、専門家だけがテキストを書くことを許可されました（理想的には、科学博士）。



将来の参加者の資格に応じて、彼は編集者、作家、査読者、コピー編集者、またはオブザーバーになるよう招待されました。 これを行うには、将来の参加者が登録し、グループの1つにサインアップし、メーリングリストに登録し、参考書を購入するなどしなければなりませんでした。



Nupediaは査読済みのプロジェクトでした-記事が作成された後、7つの（!!!）複雑な官僚的な段階を経ました。
  1. 割り当て
  2. リードレビュー担当者を見つける
  3. リードレビュー
  4. レビューを開く
  5. リード編集
  6. コピー編集を開く
  7. 最終承認とマークアップ
これは、百科事典情報の自由な交換のためにインターネットを使用するというアイデアに関係する専門家さえ退屈させました。 2001年、LarryはNupedia Advisory Councilにこのスキームを2段階のスキームに変更するよう依頼しましたが、受け入れられませんでした。



報道機関がNupediaについて書いたという事実にもかかわらず、多くの専門家が応答し、ドイツ語、スペイン語、フランス語、ポルトガル語、イタリア語のバージョンがオープンしました; 2000年11月までに、2000年3月以来、たった2つの本格的な記事が発行されました。 Computerraによると、プロジェクトに250,000ドルが費やされましたが、失敗しました。 何も助けませんでした-無料のTシャツの配布や、金銭的な報酬を得るための「友人を連れて行く」キャンペーンでさえも。 そのような昔ながらの学術的な方法で百科事典を書くことができないことが明らかになりました。



2001年1月2日、ラリー・サンガーは太平洋のメキシコ料理レストランで彼の古い友人ベン・コヴィッツ （彼はジミー・ウェールズも知っていた）と夕食をとり、再び厄介な編集プロセスのためにNupediaでの作業の深刻な進歩の欠如について不満を言いました。 Ben Kowitzは、Larryに、プログラムコードのためにWard Cannigemによって1994-1995年に発明された「wiki」の概念について話した。 サンガーはウェールズにウィキについて話しました、そして、彼は少し疑って、ウィキを試すことに決めました。 その結果、2001年1月15日にNupediaを完全にサイドプロジェクトとして補充するために、将来のNupedia記事の下書き用のサイトが開かれ、SangerはWikipediaという愚かな名前を思いつきました。 次に何が起こったのか、誰もが知っています。



すでにウィキペディアの最初の数ヶ月で、Nupediaとは異なり、進歩は単純に驚くべきものでした。 ウィキペディアには、まったく新しい参加者と、新しいプロジェクトに切り替えたNupedia参加者が集まりました。 すぐに、ウィキペディアはNupediaから完全に独立して機能するようになりましたが、ウィキペディアには同じエディター（最初と最後）のラリーサンガーがいました。 初期の段階では、LarryはNupediaとWikipediaの両方の基盤の戦略的開発と形成に多大な貢献をしました。 しかし、2002年初頭、ウェールズは編集長の仕事に対する支払いを停止し、ラリーサンガーはボミスプロジェクトを去りました。



エンジンに関するいくつかの言葉。 Nupediaは無料でオープンな（GNU GPLのもとで） NupeCodeエンジンを開発しましたが 、これはひどく機能していませんでした。 記事の作業とエラーの修正は、メーリングリストと、すでに完成したバージョンを投稿するNupediaで実行する必要がありました。 すべての作業をWebに転送する必要があることは明らかでしたが、そのためにはエンジンで多くの作業を行う必要がありました。 これは行われていません。



Nupediaは2003年9月26日に閉鎖され、 その記事はWikipediaに移動しました 。 閉鎖の時点で、作業を開始してから2.5年後、Nupediaにはわずか24の完成品と74以上の未完成品がありました。

輝かしい運命の時代以来、STR STRの背景のおかげで、NupediaはWikipediaに継承されたスローガン「オープンコンテンツ百科事典の代わりに」を受け取りました。



ウィキペディアの開設からしばらくして、 www.gnu.org / encyclopedia / encyclopedia.htmlページに、ウィキペディアがNupediaの哲学を受け入れ、GNUがウィキペディアへの貢献を奨励していることを示すテキストが掲載されました。 このテキストはまだそこにあります。



GNUPediaとFree Software Foundationの支持者は、記事の検証に反対し、官僚制度を非常に心配していたWikipediaが好きでした。 その結果、GNUPedia自体は完全には立ち上げられず、無料プロジェクト間の潜在的な競争は回避され、オープンソースソフトウェアのサポーターがウィキペディアに参加しました。



まとめ



Wikipediaと呼ばれるNupediaの子孫は、最終的に（ほぼ、たとえば、 Creative Commons Searchと一緒に）リチャードストールマンが夢見た非常に無料の百科事典になり、Wikiテクノロジーでフリーソフトウェア財団をサポートする自由を愛するインターネットユーザーの力に加わりましたNupediaのより保守的な科学者の提供と努力。 数年後、非常に人気のあるプロジェクトになったウィキペディアは、STR運動に感謝し、世界中の多くの人々に、情報の自由、自由形式、およびリチャード・ストールマンが1980年代に策定したアイデアについて伝えます。 何千人もの人々がウィキペディアからSTR運動の非常に多くの側面について学びました。 伝統的に、ソフトウェアに関するほとんどすべてのウィキペディアの記事は、単語freeまたはnot freeで始まります。



NupediaとWikipediaは、 CitizendiumやGoogle Knol （一部）など、一連の無料の百科事典の始まりを示しました。



ウィキペディアが無料コンテンツの闘争の中核になり、世界最大の無料情報コレクションの1つになったことは周知の事実です。 ウィキペディア上のすべてのキロバイトのテキスト、すべての写真、ビデオ、音声は、伝統的な著作権と世界の文化および情報空間全体に利益をもたらす自由文化への新しいアプローチとの間の競争を高める一歩です。
