子供の頃、私はアーノルドとのコマンドー映画が本当に好きでした-それはほんの始まりで、彼と彼の娘は山のどこか高い所にある自分の居心地の良い家に住んでいます...多くの人がこのようなことを夢見ていると思います-そうすれば、あなたはそのように、心配も手間もかからない理想的な世界に生きることになります。 時々、あなたはソファに横になって、「私が自分の家を持っているとき、私はそこで完全な注文をするでしょう！」と考えます。 パントリーでも、各ボックスに慎重に署名し、各ワイヤーを慎重に敷設し、いくつかのトリッキーなアルゴリズムを使用して、ハンマー、ジグソー、あらゆる種類のドライバーの武器庫を吊るします」

ガジェットは、比較的小さな箱のほぼ全体の体積を占有します。それに加えて、キットにはテスト用の「カートリッジ」とかなり詳細なユーザーマニュアルのみが含まれています。 彼らが他に何も置かなかったことは奇妙です-前方を見て、そのようなマシンはベルトのどこかにあり、少なくとも十分なカバーがありません...電源アダプターは言うまでもありません-それはデバイスの設計によって提供されますが、そこにもありません 電池もありません。テストを開始できるように、店まで走らなければなりませんでした。









原則として、写真のすべてが表示されるので、詳しくは説明しません。 しかし、私は別のものから始めます-最初の知り合いで視覚的にこのデバイスの目的を与える唯一のものは、右上のボタン「印刷」です。 不透明な粘着テープでテープを貼って、興味のためにさまざまな程度のITの5人を見せました。 「これはどのようなものですか？」という質問には、さまざまな答えがありました-古代のタイプライターやポケットレジから、チャートを印刷する機能を備えた計算機まで。 そして、私がデバイスを手でひねったときだけ、実際に誰も疑いを持っていなかった-小さなものを印刷するためのプリンター。 「少し予測可能です...それが起こりました」（c）。



プリンターは本当に小さい（155 x 165 x 64 mm、500 gの重量）-その前に、「プリンター」という言葉は私にもっと面倒な関連付けを引き起こしました。 そして、ここで...メーカーはそれをデスクトップとして位置付けているので、彼にとってあまりにも多くの方法があります-それは多くのスペースを占有しません。



デバイスの前面は、画面、ファンクションキー、キーボードの3つの部分に分割できます。







ディスプレイは白黒で単一行ですが、最初のインクルージョンでは、より多くの情報に適合することが示されました。これについては後で詳しく説明します。



画面の下には7つのゴム製ファンクションキーがあり、そのうち4つは2つのボタンを組み合わせており、1つは一般に4方向のジョイスティックです。 これらのボタンのおかげで、デバイスのすべての機能が監視および制御されます。



以下は、同じゴム引きボタンの完全なQWERTYキーボードです。 その主な利点の1つは、キリル文字（ボタンでも表示可能）をサポートしていることです。さらに、ボタンを1つ押すだけで言語を切り替えることができます。 デバイスは実際に両方の言語（Rus / Eng）に対応しています。 CAPS、大きなスペースとEnter、控えめなBackspace、数字と句読点を入力するための個別のボタンがあります-はい、少なくともここにはSoloがあります。 私がキーボードについて気に入らなかった唯一のことは、ハイフンがないことです。 より正確には、安全に隠されています-入力するには、「挿入-記号-数学」をクリックする必要があります。つまり、ハイフンではなくマイナス記号が表示されます。 ロジックはどこにありますか？ いいえ）しかし、ボタンを押すことは完全に静かです。

デバイスはネットワークから動作できます-この場合、対応するコネクタはケースの背面にあります。 しかし、私が言ったように、アダプターは含まれていません。 印刷されたリボンは同じ面を残します。







底面には2つのコンパートメントがあります-1つはデバイスが自律的に動作することを可能にします（6つのフィンガーバッテリーを挿入すると、プリンターはほぼ2倍の重量になります）、カートリッジ自体は2番目に挿入されます。







作業を開始するために、「消耗品」-半透明のカートリッジ（ DYMO D1 ）を取り付ける必要があります。その内部には、粘着性のナイロンまたはポリエステルテープがあります。 すべてが印刷され、貼り付けられます。

インクルージョンと仕事



通常、夜はテストや記事に「パンチ」します。誰もが眠っていて、誰も気を散らしていません...そして、夜にデバイスをオンにするのは少し怖かったです-そのような「たまごっち」から、私はいくつかの8ビットのメロディーを起動時にテープで擦る しかし、これは何もありませんでした。「オン」を押すと、デバイスはノイズやほこりなしで動作する準備ができました。



最初に目を引くのは、画面のコントラストです。 非常に低く、直角ではほとんど何も見えません。鋭角を見る必要があります。 最初は問題はバッテリーにあると思いました...そして互換性のあるアダプターを探すのに時間を費やさなければなりませんでした（私はデステンのラップトップから来ました）。 しかし、残念ながら、この接続を使用しても、画面上の記号は明るくなりませんでした。
