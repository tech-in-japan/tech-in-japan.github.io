ニジニノヴゴロドでのSchool Linuxの次の実装

Nizhny Novgorod Linux User GroupのSchool Projectのおかげで、すでに4つの学校でSchool Linuxを使用したクラスが完全に構成され、教育プロセスの準備が整いました。



ハブロフスク市民の要請で、私は「技術的なポイントを含むより詳細な説明」を書いています。







今回は、通常よりわずかに大きいラインナップで乗りました。 私たちは6人いました。



今日の実装はすべてクローンモードで行われました。 最も強力なコンピューターに、生徒用にSchool Linux Juniorをインストールしました。 必要な設定をすべて行い、ギャラリーを追加し、iTalcの公開キーを追加しました。 その後、構成ファイルを「クリーン」して、システム全体をtar.gzアーカイブにパックしました。 圧縮形式では、システムのサイズは2.8 GBです。



さらに、USBハードドライブ上のこのファイルは、クラス内の残りのコンピューターに送られ、既に準備されたext3パーティションに解凍されました。 解凍後、chrootに移動してliloコマンドでブートローダーを復元するだけで十分です。 このインストール方法を使用すると、純粋な時間でクラス内の10台のコンピューターにオペレーティングシステムを展開できます。



2時間までに、ほとんどの作業が完了しました。 その前にお茶とサンドイッチを食べることを忘れずに、教師のコンピューターのセットアップを開始しました。



机の上の先生には、HP LaserJet 1020プリンターとHP ScanJet 3800スキャナーがありました。



スキャナーは、「Image Scanning Program」を開始したときにSchool Linux Juniorで大金を稼ぎました。 スキャナーの特定の遅延は、スキャナーでの作業の開始時に目立ちます。 ただし、これはこのタイプのスキャナーのハードウェア機能であり、オペレーティングシステムに依存しません。



しかし、プリンターでは少しいじらなければなりませんでした。 一般に、HP LJ p1005、1018、1020クラスのすべてのプリンターおよび同様の安価なオプションは、hp-setupコマンドを使用してHPLIPサブシステムを介して起動されます。 ただし、School Linuxでは、このサブシステムはグラフィカルパーツなしでインストールされるため、hp-setupは正しく起動しません。 これを修正するには、リポジトリを接続してhplip-guiパッケージをインストールするだけです。 このパッケージをインストールした後、hp-setupユーティリティが正常に起動し、インターネットからバイナリモジュールをダウンロードします（この段階では、主なことは忍耐強く待機するだけで、ロードには約5..10分かかります）。



しかし、最終的にはプリンターにカートリッジがなかったことが判明し、テストページを印刷する5分間の試行を残しました:)



インストールの最後に、2人の学校のコンピューターサイエンスの教師にシステムの20分間のデモンストレーションを実施しました。

  • 彼らは、10台のコンピューターのクラスを管理し、デスクトップをブロードキャストし、デスクトップをブロックすることにより、iTalcの動作を実証しました
  • 学校のコンピューターから教師のコンピューター上のネットワークフォルダーを操作する方法を示しました
  • コンテンツフィルタリングシステムの動作を実証しました。 彼らはいくつかの特徴的な要求を「教育プロセスのタスクと両立しない」ものにしました。 すべてがブロックされています！




不快から。



教師はSchool Linuxでのプレゼンテーションの作成と実行に興味を持ちました。



「すべてがここにある」と言って喜んでいた。 Impressを起動すると、「テンプレートの読み込みエラー、一般的なエラー」と言われます。



エラーの結果によると、彼らはBugZilla Altlinuxでbuigを発行しました



https://bugzilla.altlinux.org/show_bug.cgi?id=23633



楽しいから-今日、私たちはALT'ovのボックス「School Linux」の内容を見る機会がありました。 3500rの価値があると言えます。



  • Moodleの説明とwiki学習システムを含む、学校の分布に関する優れた本。
  • 配布に含まれるすべてのメインプログラムを含むカラフルなポスター。
  • ディストリビューション（School Linux Master、Junior、Light、サーバー、ターミナルサーバー、Windows用オープンソースディストリビューション、ドキュメント付きディスク）を含む9個のディスク。
  • 20個のホログラフィックステッカー
  • まあ、小さな紙-ボックスに含まれるソフトウェアの説明とALT Linuxからのライセンス。












最も楽しいから-私たちは非常に美しいありがとうノートを提示されました！
