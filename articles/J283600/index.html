A16Zがスタック交換に4,000万ドルを投資

Stack ExchangeのCEO兼共同設立者であるJoel Spolskyは、昨日、 Andreessen Horowitz Foundationとパートナーからの投資ラウンドが成功裏に完了したことを発表しました。 有名なベンチャー投資家は、開発者向けの人気のQ＆Aエコシステムを開発するために4000万ドルを$しみませんでした。



Stack Overflowが設立されてから5年間で、133のコミュニティが独立しました。 ジョエル自身は、この現象を中世の世界地図と比較しています。中央には常に有名な大都市（バビロン、ローマ、エルサレム）があり、その周辺には小さな村がたくさんあります。 現在、「村」と「人口」、つまり訪問者と登録ユーザーの数は、毎月25％増加しています。



統計によると、今日のStack Exchangeサイトのネットワークは、米国のトラフィックに関してはトップ50に入っています。 同社自体は、デンバー、ニューヨーク、ロンドンのオフィス、および自宅から離れた場所に、世界中で200人以上の従業員を雇用しています。 Spolskyによると、それらの半分以上は、同社の主要な収益化ツールであるStack Overflow Careersによって独占的に採用されています。



当然のことながら、すべてのコミュニティの中で最大のものは、開発者向けのStack Overflow Q＆Aサービスであり、毎月4,000万人以上のユニークビジターを受け入れています。 Stackエコシステム全体の統計は次のとおりです。2014年のユニークビジター数は3億人、登録ユーザーは合計380万人です。



合計で、過去1年間だけで、Stack Exchangeは310万の質問を行い、450万の回答を書き、270万の編集を行い、最終的に1700万のコメントを残しました。
