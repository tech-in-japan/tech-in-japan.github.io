  • 支払いデータ （PINコード、プラスチックカードテープの番号とダンプ、SIMカード情報、さらにはNFCチップからのダンプ）は、あらゆるレベルのサイバー詐欺師にとって重要な情報です。 この情報は、犯罪を犯す際に、資金の現金化、商品の代金の支払い、データの偽造に使用できます。



  • 個人データ （パスポートデータ、メール、電話番号）も非常に貴重な情報です。 最良の場合、あなたに電話してサービスに関するニュースレターを送信するスパマーや企業は喜んでそれを購入するでしょう。 最悪の場合、ローンの所有者になったり、お金を失ったり、偽の文書で不快な話に巻き込まれることさえあります。

Googleは、ベルギーのGoogleデータセンターに落雷が発生したため、データの一部が取り返しのつかないほど破壊されたことを発表しました。 2015年8月13日-データセンターの電源システムに4回連続で落雷が発生しました。 これにより、Google Cloud Platform（GCP）を使用するクライアントの仮想マシンが保存され、そこにデータが保存されたデータセンターの電源が一時的に切れました。 Googleの専門家は、2015年8月13日から17日にかけて、仮想マシンに接続されたネットワークドライブにアクセスしようとすると、仮想マシンでI / Oエラーが発生することを指摘しました。 一般に、ヨーロッパゾーンのネットワークドライブの約5％で少なくとも1つのI / Oエラーが発生しました。 また、この期間中に、スナップショットの作成などの一部の制御操作を実行できませんでした 。 Googleにとって、これはそれほど多くのデータではありませんが、情報とプロセスが影響を受けた企業にとってはどのようなものでしたか。 今、彼らは雲の中に稲妻があることを知っています。
