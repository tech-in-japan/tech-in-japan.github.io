Ubuntu Netbook Remix（9.04 Jaunty）+ Acer Aspire One A110-Aw

良い一日！



少し前まで、Habréでタイトルに示されたネットブックモデル用のLinux4oneについてすでに書いていました。



このようなサブラップトップを取得したので、Ubuntu Netbook Remixの最新バージョンを直接インストールすることにしました。

なんで？ インストールされたシステムに最新のパッケージが欲しいからです。



カットの下-インストールについて少し説明し、システムのスクリーンショットをいくつか（ 注意！トラフィック！ ）



私が最初にしたことは、Ubuntu-wikiからのリンクから .imgイメージをダウンロードすることでした。 USB ImageWriterユーティリティがホームコンピューターにインストールされ、これによりイメージがフラッシュドライブに記録されました。



ラップトップが納入されたばかりなので、私は最初に機器のネットブックリミックスがどのように動作するかを確認することにしました。 インストールなしでライブモードで起動し、試してみました-すべてのデバイスが拾われて動作し、カメラ、サウンド、Wi-Fiが動作しました。 だから先に行く-あなたはシステムをインストールすることができます。

インストールプロセス自体については説明しません。これは、Live-CDからUbuntuをインストールするのとまったく同じです。



VPN接続を構成するには、2つのパッケージをインストールする必要がありました：ppp-linux（インストールフラッシュドライブで利用可能）およびnetwork-manager-pptp 。 VPNは1回または2回アカウントにアクセスし、安定して動作します（NetByNetがあります）。



さらに、システムは「管理」セクションの「ローカリゼーション」項目を通じて完全にロシア化されました。



はい、Hybernateはスパークモードにも満足しています。 ラップトップはスリープ状態になり、Fn + F4を押すと起動します。

ホットキーは、シャーマニズムや魔術を使わずに、ブナで提供されるものすべてを機能させます。 含む-Fn + F7-タッチパッドロック、Fn + F8-ミュート、Fn +矢印-音量/輝度調整など
