マイクロソフト認定資格と試験の変更

最近のIgnite会議で、マイクロソフトは新しい認定プログラムを導入しました。 同社は、 役割/専門分野に基づいた新しい認定が 、スキルの向上、雇用者へのスキルと知識の実証、新しい機会の発見に役立つことを保証します。







Azure Administrator 、 Developer 、 Solutions Architect 、DevOps Engineer、Microsoft 365 Modern Desktop Administrator、およびEnterprise Administratorの6つの新しい専門分野が発表されました。 Azure管理者認定の試験とトレーニングは既に利用可能であり、次の2つの試験-Azure DeveloperとAzure Solutions Architectはベータ版です。 このリストの残りの3つの認証は、年末までに発行されます。 「これはほんの始まりです！ 2019年にはさらに専門分野を発表する予定です。」とMicrosoft ブログで Selina Winter が書いています。











いくつかの回答オプションを持つ通常の質問に加えて、アクティブなシナリオ、ドラッグアンドドロップのバリエーション、および実際のシナリオに近い状況に基づいて、新しい質問を追加することを約束します。







以下は、Azure管理者認定のトレーニングコースです。











重要！ 70-XXXシリーズの多くの試験は、2018年12月31日から廃止と見なされ、新しいシリーズ試験AZ-10X、AZ-20X、AZ-30X、MS-10Xに置き換えられます。



近い将来マイクロソフトの試験を受ける予定がある場合は、英語の試験ページの情報を注意深く監視してください。 おそらく、あなたの試験はすでに新しい試験に置き換えられており、古い試験に合格するよりも、新しい試験にすぐに合格/準備する方が実用的です。



すべてのMicrosoft Azure認定プロフェッショナルには、99ドルの移行試験を受ける機会が与えられています。 （価格は国/地域によって異なり、試験登録プロセスに反映されます。） 2019年6月30日までご利用いただけます。







移行試験では、新しい教材と、現在の認定資格と新しい認定資格のスキルと概念の違いのみが対象となります。







次のいずれかの試験に合格した場合（または2018年12月31日より前に合格した場合、その日付以降は廃止とみなされます）：







  • 70-533：Microsoft Azureインフラストラクチャソリューションの実装
  • 70-532：Microsoft Azureソリューションの開発
  • 70-535：Microsoft Azureソリューションアーキテクチャ


対応する移行試験に合格できます（2019年6月30日まで利用可能です）。







  • AZ-102：Microsoft Azure管理者。 移行試験
  • AZ-202：Microsoft Azureの開発者。 移行試験（ベータ）
  • AZ-302：Microsoft Azureソリューションアーキテクト。 移行試験（ベータ）


したがって、2つの方法、新しい証明書を取得する2つの方法があります。







1.廃止された試験+移行試験に合格し、新しい証明書を取得する







70-533 + AZ-102-> Microsoft Certified Professional：Azure管理者

70-532 + AZ-202->マイクロソフト認定プロフェッショナル：Azure Developer

70-535 + AZ-302->マイクロソフト認定プロフェッショナル：Azure Solution Architect







または







2. 2つの新しい試験に合格し、新しい証明書を取得する







AZ-100 + AZ-101->マイクロソフト認定スペシャリスト：Azure管理者

AZ-200 + AZ-201->マイクロソフト認定スペシャリスト：Azure Developer

AZ-300 + AZ-301->マイクロソフト認定プロフェッショナル：Azure Solution Architect
