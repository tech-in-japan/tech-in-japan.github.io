ソフトウェアのインストール



すぐに自慢したいのは、プログラムのインストール機能です。 職業によって、多数のコンピューターを構成する必要がある人のために作成されました。 これで、各プログラムを手動でインストールしたり、巨大なユーティリティ（ WPI ）に頼ったりする必要がなくなりました 。 必要なプログラムを選択し、「 インストール開始 」ボタンをクリックしてください！ あなたの顧客はあなたに感謝します:)





ワンクリックでドライバーをインストール/更新でき、同時にプログラムをインストールできます！



たとえば、 Skypeが既にコンピューターにインストールされている場合、このリストには表示されません。 また、追加のユーティリティ（ Asus HotkeyやNVIDIA PhysXなど ）は、コンピューターに必要な場合にのみ表示されます！



アセンブリには、最も基本的な（ および無料の ）ソフトウェアが含まれています。 プログラムをオーバーロードしないようにしましたが、それでもあなたはいつでもあなた自身の何かを追加することができます。 これを行うには、プログラムのインストールファイルを「 program 」フォルダーに追加し、「 settings.js 」構成を調整します 。



新しいプログラムを追加する例：

prog [ pn ] = 'K-Liteコーデックパック' ; //プログラム名

cat [ pn ] = 'システム' ; //カテゴリ

cmd1 [ pn ] = 'K-Lite_Codec_Pack.exe' ; //ファイル名

cmd2 [ pn ] = '/ VERYSILENT / SUPPRESSMSGBOXES / NORESTART / SP-' ; //キーのサイレントインストール

check [ pn ] = 'HKLM \\ソフトウェア\\ Microsoft \\ Windows \\ CurrentVersion \\ Uninstall \\ KLiteCodecPack_is1 \\ DisplayName' ; //すでにインストールされているプログラムをチェックするためのレジストリキー

defaul [ pn ] = true ; //デフォルトを真にチェックし、偽をチェックしない

wshow [ pn ] = true ; //ダイアログを真に表示、このプログラムのすべてのダイアログを偽に非表示

pn ++ ;




USSF.exeユーティリティを使用して、目的のプログラムのサイレントインストールキーを見つけることができます。

さらに、プログラムをインストールするコンピューターのモデルを制限できます。 たとえば、 Asusラップトップの場合：

if （ Manufacturer。toUpperCase （ ） 。 indexOf （ 'ASUS' ） ！ = - 1 ） {

prog [ pn ] = 'Asus Hotkey' ;

...

}



構成で任意のJavaScriptコードを使用できます！

ウイルス対策監視
