HTCおよびWindows Phone 7：完全なカスタマイズの終わり



Windows MobileとAndroidの存在を通じて、すべてのメーカーはこれらのプラットフォームの無条件のサポートについて話しましたが、自分たちの問題に焦点を当ててめちゃくちゃ少数の電話をリリースしました。 すべてですが、HTCではありません。 同社は、企業がHTCを超えることさえ夢にも思わないほどの数のWindows MobileおよびAndroidスマートフォンを作成しました。 HTCは、最初のAndroidデバイスも作成しました。 モバイルワールドコングレス2009では、マイクロソフトはHTCを優れたパートナーと呼び、検索大手はHTCで作成されたGoogle Nexus Oneスマートフォンを販売しています。



HTCは、上記のオペレーティングシステム用に膨大な数のデバイスをスタンプするだけでなく、本当に素晴らしい仕事をしました。 最大のディスプレイのいくつか、最高のQWERTYキーボード、さまざまな巧妙なソリューションなど、ユニークなデザインが注目を集めました。たとえば、会話中に画面を下にして電話を置くとTouch Pro 2のスピーカーがオンになります。 HTCが、TouchFLOとSenseの両方のモバイルプラットフォーム用の高度なユーザーインターフェイスの開発に多額の投資を行ったことは間違いありません。 しかし、MicrosoftがWindows Phone 7シリーズの概念に根本的な変更を加えたため、HTCはコースを変更し、Windows MobileとAndroidの単一ユーザーインターフェイスのすべての利点を失う必要があります。









HTCが使用するスマートフォンの全ラインに対する単一のユーザーインターフェイスの戦略は、常に当惑しています。 企業がWindows MobileとAndroidの単一のインターフェース標準の作成に本当に多大な労力を費やしている場合、両方のプラットフォームをサポートする必要がありますか？ しかし、この質問はもはや関係ありません。 Windows Phone 7シリーズの発表により、Microsoftはプラットフォームのオープンなカスタマイズから振り子を振り払い、制限されたマルチタスク、すべてのアプリケーションを手動で検証するアプリストア、標準のインターフェイスに加えて独自のインターフェイスの使用を禁止するなど、iPhoneのようなルールを確立しました。 最新の禁止では、Sony EricssonのPanels、LGのS-Class、SamsungのTouchWizなどのインターフェイスが残されています。



これは、プラットフォームの整合性を維持するために行われたマイクロソフトの真の決定です。 Microsoftは以前のバージョンのWindows Mobileに対して直接これを直接宣言しませんでしたが、競争力のあるものを作成するために、メーカーはシステムのユーザーインターフェイスの主要な要素（またはHTCの場合はほぼすべて）をやり直す必要がありました。 それはそうかもしれませんが、マイクロソフトはゲームのルールを変更し、ユーザーインターフェイスの変更の禁止に伴い、新しい質問が発生します。メーカーはどのように互いに異なるのでしょうか。 -HTCの場合は特別な意味があります。 HTCはおそらくWindows Mobile 6.5（またはWindows Phone Started Edition、またはそれが呼ばれるもの）のライセンスを継続する可能性がありますが、この戦略はタイマーを開始する一時的なソリューションにすぎません。 HTCが革新的なものを作成しようとする試みの最後の数秒をカウントするタイマー。



もちろん、HTCは、ファン向けのハードウェア設計に依存して、Windows Phone 7シリーズでデバイスをリリースすることを決定する場合があります。 そうすれば、LGやSamsungなどの競合他社が経済的地位とブランド認知度を高めているニッチでデバイスを強調できるようになります。 または、PalmとMotorolaがかつて行っていたように、Microsoftからすべてのシステムのサポートを削除し、単一のOSに集中することもできます。 もちろん、このようなソリューションは、Androidデバイスの主要サプライヤーであるHTCとMotorolaに衝突しますが、SamsungやLGとは異なり、Motorolaはグローバルメーカーではありません。 また、HTC SenseのシェルはMOTOBLURよりも望ましいように見えます。 したがって、「孤立した」HD2はAndroidのアップデートを取得できます。



それにもかかわらず、Microsoftの改良されたOSは、何らかの形ですべてのライセンシーの手に渡ります。 これらの製品が誰にとっても不要な場合、製品間の違いは特に役立ちません。 ただし、Windows Phone 7シリーズのリリースでは、ユーザーインターフェイスの変更が禁止されているため、HTCの競争力が弱まります。 Windows Mobileの「リブート」により、Microsoftの主要パートナーの1つが脇に置かれたようです。

自分から

もちろん、ロスが想像しようとしているほどすべてが悲しいわけではありません。 HTCは、さまざまな国のオペレーター、特にVerizon、Sprint、AT＆Tと多くの確立された連絡先を持っているので、米国市場での同社のスマートフォンの成功を心配する必要はありません。 さらに、Qualcomm（System-On-Chipのサプライヤ）との継続的なパートナーシップおよび有名な設計会社One＆Coの買収により、HTCデバイスの設計とパワーを疑う理由はありません。
