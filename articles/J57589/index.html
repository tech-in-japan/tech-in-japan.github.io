iPhoneおよびiPod Touchのwi-fiを介したUbuntuでのl2tpインターネットの配布の設定



iPod Touch 2Gの機会に自分で購入しました。 それは良いおもちゃです、私はもう一度Appleの賞賛を歌うことはありません、それはcorいです:)問題は、私は彼のためにwi-fiインターネット配布を長い間セットアップできなかった、私はiTunesセットアップでスチームバスをしなければならなかったVirtualBoxの下の仮想マシン上。 これでようやく問題が解決しました。 さあ、行こう！



橋頭preparationの準備


そのため、 xl2tpdを介してインターネットが設定されたラップトップがあります。 設定ガイドは、たとえばここから取得できます 。

インターネットを配信するには、接続アクティベーションスクリプト/etc/init.d/xl2tpdの開始セクションにいくつかの行を追加する必要があります（ route -nを介してルートを追加した後）。

エコー1> / proc / sys / net / ipv4 / ip_forward

iptables -F

iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE

iptables -t nat -I POSTROUTING -o ppp0 -j MASQUERADE

iptables -I FORWARD -p tcp --tcp-flags SYN、RST SYN -j TCPMSS --clamp-mss-to-pmtu


ここで、 eth0はインターネットに見えるインターフェイスです。 説明したiptablesコマンドは、NATを発生させ、インターネットの配布を整理します。 これらのコマンドの代わりに、必要に応じて、fireholパッケージをインストールおよび構成できます。理解しやすいです。

ワイヤレスツールをインストールします。



$sudo apt-get install wireless-tools







私はラップトップにwi-fiアダプターIntel Corporation PRO / Wireless 3945ABGを持っています。これは通常ubuntuによって決定され、セットアップ時にタンバリンと踊る必要はありません。 インターフェースはwlan0です。



これで、 / etc / network / interfacesで利便性のためにこのすべてをパウンドします （チャネルはブート時に中断されます）。

自動wlan0

iface wlan0 inet static

アドレス192.168.0.1

ネットマスク255.255.255.0

ワイヤレスモードアドホック

ワイヤレスチャネル1

ワイヤレスレートオート

ワイヤレス-essid BigBrother


さて、あなたはすでに行動することができます（あなたが再起動するには遅すぎる場合）：



$sudo ifconfig wlan0 up

$sudo iwconfig wlan0 mode ad-hoc channel 1 essid BigBrother







すべて順調ですが、IPを手動で登録する必要があります。 この迷惑な見落としを修正します。



$sudo apt-get install dhcp3-server







ファイル/ etc / default / dhcp3-serverを開き、目的のインターフェイスをそこにプッシュします。



INTERFACES="wlan0"







そして最後に：



sudo gedit /etc/dhcp3/dhcpd.conf







私にとってはこのように見えます：

ddns-update-styleアドホック;

default-lease-time 360​​0;

最大リース時間9000;

信頼できる;

log-facility local7;



インターネットプロバイダーの#DNSサーバー。

オプションドメイン名サーバー213.234.192.8、85.21.192.3;



＃アドレスを発行するサブネットの説明

サブネット192.168.0.0ネットマスク255.255.255.0 {

範囲192.168.0.2 192.168.0.255;

オプションルーター192.168.0.1;

option broadcast-address 192.168.0.0;

}


これで実行できます：



$sudo /etc/init.d/dhcp3-server restart







DHCPサーバーが起動時にエラーを出す場合があります。次に、空のフォルダー/ etc / ltspを手動で作成してみてください。 いずれにせよ、万能薬は再起動です。



これで、[設定]-> [Wi-Fi]セクションでiPodのネットワークを検索してみることができます。 「BigBrother」という名前のネットワークが表示されない場合は、iPodを再起動します。 私たちはつながっています-そして出来上がり！ インターネットは大声で聞かれます。
