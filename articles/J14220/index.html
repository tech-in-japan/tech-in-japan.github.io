Mozillaはメールクライアントの開発に新たな一歩を踏み出しました

David AsherがMozillaに加わり、新しい組織を率いています。



2007年9月17日-Mozillaは本日、インターネット上のメールと通信の開発を刺激するために設計された新しいステップを発表しました。 Mozillaは、Thunderbirdの製品、コード、ブランドに基づいた通信プログラムの開発を計画しています。 このステップは、開発者の信頼できるエコシステムを作成することも目的としています。これは、Firefoxブラウザーの最高の伝統で、オープンソーステキストの改善に貢献し、ユーザーコミュニティからのイニシアチブを奨励します。



現在ActiveStateのCTOおよび副社長であるDavid Asher博士がMozillaに加わり、新しい電子メールに焦点を当てた会社を設立しました。 Asherは、2000年からMozillaコミュニティの活発なメンバーであり、当初はMozilla Integrated Development Environment（IDE）に基づいてKomodoプロジェクトの責任者を務めていました。 Python Software Foundationのディレクターとしての彼の役割を考えると、彼はオープンソースコミュニティのリーダーとして認められています。



「メールおよび通信環境の革新により、Webベースのソフトウェア全体に選択肢が広がりつつあります」と、Mozilla Foundationの責任者であるMitchell Baker氏は述べています。 「Davidは長年にわたってMozillaコミュニティの尊敬されているメンバーであり、Mozillaに参加してこの重要な仕事をリードできることを嬉しく思います。」



ベイカーは2007年7月に、Thunderbirdの将来に関するコミュニティのフィードバックを収集するための行動を促すフレーズを公開しました。 活発な公開討論の後、Mozilla Foundationの理事会は、インターネットで作業するためのメールおよび通信ソフトウェアを開発、更新、改善することを目標とする新しい子会社の設立を承認しました。



新しい会社を設立するために、Mozillaは300万ドルの初期寄付を行います。



世界中の何百万人もの人々がThunderbirdを主要なメールアプリケーションとして使用しています。 現在のThunderbirdユーザーの場合、何も変わりません。 Mozillaは、新しい会社を設立することにより、Thunderbirdユーザーに安定性とセキュリティを向上させるための定期的なアップデートを提供し続け、Thunderbirdユーザーのニーズを無視しません。
