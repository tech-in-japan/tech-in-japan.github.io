Mantis ::スクリーンショットの自動化

マンティスと一緒に仕事をするとき、どうすれば人生を楽にすることができるのだろうかと考えました。 歴史的に、彼と仕事をしていることがたまたま起こったので、ワークショップの同僚のためのレシピはさらに進んでいます。



手順を慎重に実装すると、インストールプロセスに約30分かかります。



インストールは次の場所で行われました。



カマキリ1.1.6

Windows XP



この製品に詳しくない人のために、マンティス自体は* nixスレッドで動作し、スプリンクラーはウィンドウで動作することを説明します。 この記事では、生産的な作業のために2つの便利なツールを接続する方法について説明します。





1. フレームワークをダウンロードしてインストールします： blogs.geekdojo.net/brian/articles/Cropper.aspx

簡単に説明すると、「これは、C＃+ .NETで記述されたスクリーンショットをすばやく簡単に取得できるプログラムです。Windowsが必要です」:)

2.実行してテストします。

主なコントロールは次のとおりです。

F8:

Esc:

( ) Enter: / .

():

Tab:

alt + :

shift + :







メーカーのウェブサイトで残りを参照してください。



3.プラグインをダウンロードしてインストールします。

www.mantisbt.org/bugs/file_download.php?file_id=1504&type=bug （必須）

www.codeplex.com/cropperplugins （推奨）



プログラムがデフォルトのフォルダーにインストールされた場合、プラグインは、DLLをアーカイブからフォルダー「C：\ Program Files \ Fusion8Design \ Cropper \ plugins \」に移動するだけでインストールされます。

プログラムを再起動します。



4.トリミングボックスの設定に関する推奨事項（コンテキストメニューを呼び出します）：



1.サムネイルがインストールされている場合でも、チェックを外します。Mantisのプラグインは使用されません。

2.メニューで[出力]-> [Mantisに送信]を選択します。

3. [オプション]ウィンドウの[出力]タブで、[フルイメージ名テンプレート]を{プロンプト}に、[サムネイルイメージ名テンプレート]を[{プロンプト} tn]に設定します。

4. [オプション]ウィンドウの[キャプチャ]タブで、[クロッパーを使用して印刷画面イメージを保存します。]

5. [プラグイン]タブで、[Send to Mantis]を選択します。 サーバー+ mantisconnct.phpファイルへのパスを示します。次のアドレスmy_site / api / soap / mantisconnect.phpを取得しました



できた



結果の切望されたテスト：

1. F8をクリックします

2. Enterキーを押します（最初のウィンドウの目的がわからなかった）

3. 2番目のウィンドウが表示され、カマキリのエラー番号（作成される）、エラーを起こす画像の意味のある名前が示されます。

4.確認



そして、サーバーへのアップロードの成功（またはそうでない）についてのメッセージが表示されます。 チェック-スクリーンショットがあります。

やったー！）



それは私にとって人生をずっと楽にしてくれました。 それが他の人にとってより簡単になることを願っています。
