Google宝くじなしでグッドコーポレーションをだまし、ロシアでNexus 7を購入する方法

Habréの最近の記事で、彼らはすでにアウトバックから直接デバイスセクションでGoogle Playで購入する方法について話しました。 このトピックは紛れもなく興味深いものです。Nexus7はロシアや近隣諸国ではまだ発見されていません。もちろん、モスクワやサンクトペテルブルクに住んでいない限り、ギャラクシーネクサスのコストはかなり高くなります。 唯一の問題は、レシピがやや不安定であることが判明したことです。個人的には、Most Good Corporationが個人データの不正を疑い、コメントから判断すると、私だけではありませんでした。



1週間の苦痛とGoogleウォレットを欺こうとする試みは、Google Playで何でもどこでも購入できるという形でプラスの結果をもたらしました。さらに、厳密に言えば、ソリューションは表面にあり、複雑なアクションを必要としませんでした。



カットの下で、新しいレシピとほんの数枚のスクリーンショット。



まず、まだ購入しようとしておらず、アカウントがブロックされたことを通知するGoogleウォレットからの手紙がメールボックスにないことを願っています。 もしそうなら、あなたは、私のように、特に購入のためにゼロから新しいGoogleアカウントを作成する必要があります。 ブロックされた資金は、残りの処理を行う間、わずか数時間でカードに戻されます。



1. Shipito。


最初にすることは、 shipitoでアカウントを作成することです 。 原則として、今では多くの選択肢があり、そのうちのいくつかはわずかに安くなっていますが、私は実証済みのソリューションを支持しています。 8.5ドルの前払いを求められますが、これはすぐにアカウントに入金されます。 後で彼らはサービスのサービスの代金を支払うことができるでしょう。 今、私たちはここに来ました。 アカウントに[Shipitoと自宅の住所]タブが表示されます。このタブでは、アメリカの住所のうち2つと香港の別の住所が左側にあります。 住所を選択します（私はカリフォルニアの住所を個人的に選択しましたが、事実に基づいて2番目の住所を選択することをお勧めしました：免税で約30ドル節約できます）、それを書き留めてください。



2. Qiwi Visaカード。


次に必要なサービスはQiwiウォレットです。 彼は、Qiwiがあなたが指定する個人データの信頼性を特に熱心に気にしていないという事実に満足しています（Good Corporationとは異なります）。 ここでは、まずウォレット自体を登録してから、仮想カードを作成する必要があります。 個人的には、QVCを使用する方が便利です。Qiwiアカウント自体と共通のアカウントがあり、これも使用することがあるためです-QVVよりも少し安いです。 カード会員データに、shipitoからの同じ住所を示します。 すでにQVCをお持ちの場合-新しいQVCを作成するだけで、カードのコストはわずかです。



3. Googleウォレット。


新しく作成されたカードをGoogleウォレットに接続することは残ります。 入って、アカウントを作成します。アカウントがない場合は、Shipitoから同じ住所を示します。電話を指定する必要がある場合は、意味不明なものを書きます（Googleはチェックしません）。主なことは、 ：カリフォルニアのコード（310）を示しました。 カードをアカウントに接続すると、$ 1がブロックされます。 Googleは再来日前に返品するので、忘れることができます。



4. Tor。


ここでhttps://play.google.com/store/devicesにアクセスしようとすると、Googleはあなたがそこに住んでおしゃべりしないと言います。さらに言えば、あなたと取引をすることはありません。 この問題の解決策は簡単です-Torブラウザ。 ダウンロード、インストール。 アメリカにいることを世界中に知らせたいことをブラウザに示すために、 {Tor} \ Data \ Tor \ torrcファイルに 2行だけ追加する必要があります 。



StrictExitNodes 1





exitnodes {us}







これは、Google Playがすべてのドアを親切に開くのに十分です。



5.購入、ラウンド1。


ここで注文する必要があります。 Google Playにアクセスして、必要なものと金額を選択し、作成されたGoogleウォレットアカウントから購入します。 カードのお金がブロックされ、Googleが感謝の手紙を書いて、結果を待っています。 これが終了する可能性は十分あり、支払いはなくなり、数日以内にshipitoから自分にパッケージをリダイレクトするだけで済みます。 USPSにはリチウムが搭載されておらず、航空便の優先度を選択する必要があることを忘れないでください！



しかし、間違っているかもしれません。



その後、数分以内に、Googleから「注文がキャンセルされました」という見出しのメールが送信されます。 手紙のテキストは、Googleがあなたを疑う犯罪の重症度によって異なる場合があります。 Googleウォレットアカウントページの代わりに、通常のサポートコールのスタブが表示されることが重要です。



アカウント確認






しかし、私たちは正直な人々であり、それに関するすべての文書は私たちの手にあります！ 身分証明書（一般にロシアのパスポートを使用しました。パスポートまたは権利を取得できます）をスキャンし、プログラムの主要な番号をスキャンします。カードから抽出します。 これを行うには、Qiwiウォレットに戻り、「カードの操作」セクションの「レポート」セクションで目的のカードを選択し、レポートの締め切りをより確実に設定し、レポートを作成してから抽出します。 カードが新しい場合、私が持っていたように、あなたはこのような何かを手に入れるべきです：



カード明細書






右側には、Googleが私たちに求めているデータそのものがあります。 これを行う必要はありません。最後の4桁（分からない...）と、銀行取引明細書としてGoogleをサポートするために受け取ったヘルメットを除いて、カード番号のみが表示されます。



Googleは3営業日以内に整理されることをメールですぐに保証します。 私は最初の日にすでに答えを得ました（時間帯の違いに合わせて調整されています）。
