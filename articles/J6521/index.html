Web 2.0と恋人のカルト-エリート社会への脅威

率直に言って、挑発的な本は 、シリコンバレーのインターネット起業家であるアンドリュー・キーンによって書かれました。 名前だけでも、「アマチュアのカルト。 今日のインターネットは私たちの文化をどのように殺しますか。」



この位置には独自の説明があります。 他の多くのWeb 2.0および集団マインドの反対者のように、Andrew Keaneはエリート主義の熱烈な支持者です。 エリート主義者は、いかなる力も群衆の手にあるべきではなく、エリートの手、すなわちエリートの手、つまり政治、科学、またはウェブサービスの最高の代表者にあるべきだと心から信じています。 そのため、エリート学者はウィキペディアを嫌い、エリート主義者はWeb 2.0サイトでユーザーが生成したコンテンツの出現を嫌います。



アンドリュー・キーンおよび同様の個人の論理を理解するには、まずこの現象の科学的定義に慣れる必要があります。



エリートは、社会をエリートとミサに分ける必要性の概念です。 この概念によれば、このような区分の欠如は、社会の未発達の兆候です。 この意味でのエリートは閉鎖されており、そのメンバーは見知らぬ人を受け入れたり軽deしたりしません。 エリティズムは貴族的で、非常に保守的な世界観です。



エリート主義はエリート主義に近い現象ですが、同一ではありません。 最初の仮定と同じ二分法の「エリート-質量」を受け入れ、エリート主義の支持者はより寛大であり、彼らの権利を太陽の下の場所の質量として認識する傾向がある。 彼らの理解では、エリートは社会的下層階級の最も有​​能な人々に開かれているべきです。 ここでは、高レベルの社会的流動性が合法であり、望ましいとさえ認識されています。




Web 2.0革命は、Webサイトの作成方法だけでなく、人間の文明の基礎である社会的階層の構築の原則にも影響を与えたことがわかりました。 これは、アンドリューキーンが彼の本で述べていることです。 彼は、インターネット上のブログの数が6か月ごとに2倍になるにつれて、「平凡な無限のデジタルフォレスト」について語っています。 Keen氏は、この傾向が続く場合、2010年までに5億のブログを作成し、「政治からビジネス、芸術、文化に至るまで、世論を歪曲して混乱させる可能性があります。 ブログはどこにでもあるので、何が真実で何が偽りで、何が現実で架空のものであるかについての理解を決定します。 今日、子どもたちは客観的なプロのジャーナリストからの信頼できるニュースとjoeshmoe.blogspot.comで読むニュースレターの違いを見なくなっています。 プロの新聞、雑誌、音楽、映画-私たちの文化の主な成果は、基本的なアマチュアコンテンツの優位性による破壊の脅威にさらされています。



アンドリュー・キーンは、ウィキペディア、YouTube、および同様のサイトの人気の高まりと、それらが若い世代に与える影響について非常に心配しています。 彼は集合的な心を信じていません。 群衆は頭が良くない。



インターネットの影響下で、私たちの文明は不可逆的な変化を受けています、とキーンは警告します。 手遅れになる前に、このプロセスを停止する必要があります。 彼の個人ブログでアクションプランを作家と調整できます。
