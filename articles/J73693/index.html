ロシア向けWindows 7の価格

ロシアのWindows 7の価格の問題がどの程度関連しているかはわかりませんが、購入を決定した人にとっての情報は興味深いものです。



昨日読んだブロガーの一人はこう書いています：「 MSは貪欲なようです。WindowsVistaでのアップグレードは、コンピューターと特別なクーポンで今年購入したOEM Windows Vistaに対してのみ提供されます。幸せ。 」



それから彼は、アメリカのサイトを更新する際に、私たちが持っているものと比較して割引のオプションを提供します

  • Windows 7 Home Premium：OEM-110ドル、Box-200ドル、Boxアップグレード120ドル
  • Windows 7 Ultimate：OEM-190ドル、Box-320ドル、Boxアップグレード220ドル
  • ロシアではWindows 7のアップグレードバージョンは利用できないようです
  • 3台のコンピューター用の特別なファミリーパックはロシアでは利用できません
  • MS英語のWebサイトでは、Windows 7のすべてのボックス版をオンラインで購入およびダウンロードできます（ディスク配信を注文できます）。
  • ロシアのMS Webサイトには、Vista OEMからの無料アップグレードに関する情報があります。VistaOEMは、今年特別な機能を備えた機器とともに購入されました。 クーポン




参考までに、OEMとは異なり、ボックス版にはシステムを他の機器に移すことができるライセンスがあります（たとえば、マザーボードを変更する）。
