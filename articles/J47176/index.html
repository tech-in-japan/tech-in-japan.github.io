ディスク書き込みコピー：Windowsを永久に使用できるソフトウェア。

私はCCSで働いていることがありました。 新興企業として生まれた完全にロシアの会社で、強力なチームに集まった。



この会社が生産するソフトウェアDisk Write Copyを使ってhabrozhitelを紹介したいと思います。







つまり、コンピューター上のすべてのディスクを保護下に置いた場合 、ユーティリティは再起動を必要とし、その間すぐにシャドウコピーが作成され、元 のディスク は保護されたままであり、変更されません 。 コンピューターで作業するプロセスでは、行われたすべてが情報の変更として提示され、これらの変更がシャドウディスクに蓄積されます。 新しい映画のダウンロード、コンピューターの設定の変更、レジストリなど。 ダウンロードしたウイルスとトロイの木馬が実装を待っています。 これはすべて、変更された情報になります。 そのため、時間が来てコンピューターの電源を切るか再起動すると、変更されたすべての情報がトレースなしで消え 、元のディスクが再び表示されます。



また、すべてのディスクを保護対象にしたり、インターネットメッセンジャーやメールクライアントの履歴ファイルを保存したりすることもできません。 これを行う方法は、以下の「プログラムの利点」で説明されています。



Disk Write Copyソフトウェア製品の最初の行（1.x）は1年以上開発されており、すでに完成しています。 電子形式と物理形式の両方で販売されており、店舗の棚に立っています。 ソフトウェア製品はシャドウテクノロジーに基づいており、システムの再起動時またはコンピューターの電源を切ったときに、保護されたディスク上のすべてのデータのインスタントロールバックを提供します。



RuNetでは、私たちから不誠実な手で落ちたパーソナル版の鍵をすでに見つけることができます。 しかし、原則として、ロシアや、ノルウェーやスイスのレベルからまだ「遠い」経済の国に住んでいるので、私は誰もanyoneるつもりはありません。これは完全に論理的で、おそらく正当な現象であることを完全に理解しています。

ハブに関する最近のトピックでは、 ユーザーが大学がautorunでフラッシュドライブをウイルスからどのように防御したかを語り、フラッシュドライブだけでなくコンピューターもネットワークへの悪意のあるコードの侵入から保護する必要があることを思い出しています。 私の意見では、ネットワーク上のすべてのコンピューターにシステムディスク保護がインストールされている場合、シャドウディスクを使用するプログラムは素晴らしいと思います。 したがって、クラスが24時間機能しない場合は、単にコンピューターの電源をオフにします。アクセス端末で十分であり、学生ユーザーが残したすべてのウイルスが同時に蒸発します。 学生が仕事をするためには、クラスで教えられた科目もトレーニングの管理者特権を提供するため、管理権を制限しないことが重要でした。 そして、授業の終了後、クラスがオフになると、考えられるすべてのエラー、再構成、およびその他の情報もすべて消え、すべてのシステムが元の形式のままになり、インストールが設定されます。



私は時々同様の教育機関で働いています。 もちろん、私たちの大学では、ネットワークインターフェイス経由でマシンにWindowsをダウンロードしてインストールする機能も使用しています。 つまり、Windowsをインストールし、必要なすべてのアプリケーション（batniks、より正確にはcmd、必要なものを追加しました）のサイレントインストールを実行できますが、30分、2、5、または10分もかからないため、これはインスタントリカバリには適用されません。一度にネットワークに複数のコンピューターをインストールする必要がある場合、情報はネットワークの読み込みを開始しますが、それでも遅くなります。
