ロールシャッハテストとしてのスティーブジョブズ





1997年に、CEOとしてAppleに戻って間もなく、Steve Jobsは、運送会社が十分な速さでコンポーネントを提供していないと判断しました。 これに応じて、運送業者は、より早く配達することはできず、そうすべきではないと述べた。契約で定められた条件が尊重された。 契約を解除するように注文されたジョブ。 そして彼は、彼らが訴えようとした場合、彼らは再びセントを受け取らないだろうと付け加えた。



運送会社が訴えた。 彼女と仕事をしたマネージャーは辞めた。 裁判には多額の費用がかかったが、目標は達成された：Appleはより機敏な新しいキャリアを見つけた。



成功した人々の伝記は、インスピレーションの源および行動の指示として使用されます。 この物語からどんな教訓を学ぶことができますか？ 終わりがあらゆる手段を正当化すること。 ジョブズは彼の部下を馬鹿と呼び、彼らが間違っていたことを言うことをheしませんでした。 Appleの最高ランクでさえ、誰も彼の攻撃から保護されていませんでした。 会社のチーフデザイナーであるジョナサンアイブは、かつてロンドンの5つ星ホテルにジョブズの部屋を借りていました。 ジョブはそこから飛び出し、「なんてこった！」 彼の怒りの犠牲になったランダムな人々（警察官、店、レストランの労働者）は、同じ「礼儀正しい」訴えを受けました。



ジョブズが亡くなってからほぼ1年が経ちましたが、ウォルター・アイザックソンが執筆した彼の伝記は依然としてベストセラーです。 起業家にとって、ジョブズの人生は福音と反ゴスペルが一つになったようなものです。 あなたのビジョンと目標を守ることがどれほど重要かを示す良い例だと考える人もいます。 その他-世界を変える方法についての有益な物語ですが、あなたの周りのすべての人に対して自分自身を設定します。 この意見のばらつきは、私たちの中に2つの強い、しかし相反するニーズが存在することを示しています。仕事に成功し、同時に幸せな私生活を送ることです。 スティーブ・ジョブズであるとどう思いますか？



ビジネスマンの中には、スティーブのような多くの人がいます。独裁的な習慣を持ち、絶え間ない競争に同調し、仕事を人生で最も重要なものと考えている攻撃的です。 ジョブズの伝記は、彼らがすべてを正しく行っているという考えで彼らを強化しただけです。 以下に例を示します。



TwoFourの CEOであるスティーブデイビスは、私の申し出に熱心に応え、ジョブズが彼の人生とキャリアにどのような影響を与えたかを教えてくれました。 デイビスが私と話すのに30分を見つけたとき（忙しいスケジュールのため、これが問題であることが判明しました）、彼は仕事のためにある程度家族の生活を犠牲にしたことをすぐに認めました。 幸いなことに、彼の妻は反対しませんでした。



ビジネスについて話すと、デイビスは興奮します。 彼はリスクと不確実性を愛し、彼の仕事のすべてを楽しんでいます。 インタビュー中、弁護士は電話をかけ、スティーブは彼に何に答えるべきかわからなかったことを認めた。 「起業家は普通の人々とは異なります」とデイビスは言います。 -私たちはトラブルを恐れていません。 ジョブを見てください。 彼の人生には十分な失敗がありましたが、彼は前進し続けました。 ジョブズは慣習にとらわれず、彼は自分の道をたどります。 彼に参加するか、邪魔にならないようにすることができます。」



参加したり、邪魔になったりすることは、ジョブズをロールモデルと見なす人々が学んだ主なことです。 「ジョブズは、独裁的な管理スタイルがどれほど効果的であるかを示しました。 彼は、幸せな従業員が常により効率的であるという現代社会学の論文に反論しました。」



Squareの共同設立者であるTristan O'Tierniは、攻撃性の利点を確認しています。「人々に直接話すことを恐れないでください。あなたの仕事はたわごとです。 素晴らしさは、率直さと妥協のないものによってのみ生み出されます。」



Boxサービスの作成者であるAaron Levy氏は次のように付け加えます。「Jobsの話から、従業員に不可能を要求することが必要であることがわかりました。 彼らは、製品が輝きを放つまで私が製品を手放さないことを知っている必要があります。 これは製品には適していますが、人々にはそうではありません。」



ただし、組織が印象的な結果を示した場合、チームは失礼で権威主義的な管理スタイルと容易に調和します。 ジョブは定期的に部下を半ばばかと呼びましたが、このために辞める人はほとんどいません。 さらに、従業員は自分自身が最も幸せな人だと考えていました。なぜなら、彼らはスティーブ・ジョブズと仕事をする機会があったからです。



世界で最も収益性の高いヘッジファンドであるブリッジウォーターアソシエイツは、権威主義と残忍な率直さの有効性の別の例です。 ディレクターのレイ・ダリオは「投資の世界のスティーブ・ジョブズ」と呼ばれています。 彼は、感情に関係なく、会社で厳しいコミュニケーションを養います。 「最初は痛い」とダリオは言う。 「しかし、痛みはすぐに消え、あなたは楽しみを始めます。」



何よりも、ジョブズのファンは彼のビジョンの明快さを気に入りました。 Code Academyの創設者兼CEOであるNeil Sales-Griffinは、Jobsのおかげでエチケットの繊細さに注意を払うことをやめたことを認めています。 彼は、従業員がどれだけの時間をかけて投資したとしても、見込みのないプロジェクトを容赦なく閉鎖します。 ニールは、 MobileMeの失敗した起動とJobsの反応を思い出します。 チームにとってはストレスでしたが、問題はすぐに解決しました。



ジョブズの対戦相手は、彼の独裁的な行動スタイルに恐怖を感じています。 Isaacsonの本には、Jobsの無礼な行動の多くの例が含まれています。 彼は最も些細な理由（彼はホテルの花やカフェでの飲み物が好きではない）に激怒し、障害者のための場所に駐車し、車のナンバープレートの発行を拒否しました（6か月ごとに車を交換したため）。







Jobsの批評家には、 Stack Exchangeの作成者であるJeff Atwoodが含まれています。 「スタートアップを実行することは、戦争に参加するようなものです」と彼は言います。 「成功するには、プロジェクトに対するほぼ宗教的な信仰と完全な献身が必要です。」 AtwoodはStack Exchangeを去り、スタートアップの考えをあきらめました。 彼の啓発はイサクソンの本でした。



アトウッドはジョブズの私生活の記述に衝撃を受けました。 ある物語は、彼をほとんど涙を流しました。ジョブズは、新しいアップルキャンパスのスケッチを息子に見せましたが、娘のエリンは、建築家になろうとしていましたが、見せませんでした。 エリンは静かで内向的だったので、彼は彼女にあまり注意を払いませんでした。 彼女は、特に彼が挑発を手放し始めたとき、彼女の父親と一緒に振る舞う方法を知りませんでした。 「仕事は他のすべてを征服するべきではありません。 ひどいiPadを作成するか、子供を育てるかを選択する方法 子どもたちはより重要です」とアトウッドは興奮します。



元コーヒーショップのオーナーであり、現在はビジネススクールのコンサルタントであり教師でもあるVerinder Sialは、あまりカテゴリに属していない。 なぜジョブズはいつも正しいことを望み、残りを非難したのですか？ なぜ彼は他の人のアイデアを採用したのですか？」Verinderによれば、ジョブズはダイナマイトのようなものでした。 ダイナマイトは道を切り開きますが、周囲のすべてを破壊します。 「ビル・ゲイツはお尻でしたが、やがてまともな人間になりました。 ジョブズはまだロバだった」とサイアルは言った。



シアラ、特にアトウッドの主張は部分的にしか正当化されていません。 本を注意深く読むと、最後に向かって、ジョブズがイサクソンに伝記を書くように頼んだ理由を説明する断片を見つけることができます。 「私はいつも彼らと一緒にいたわけではなく、なぜこれが起こったのかを彼らに理解してほしい。」 Stardockゲーム会社のCEOであるBrad Wardellは、iPod、iPhone、iPadを作成したのと同じ理由でJobsが早期に死亡したことに気付いたとき、ショックを受けました。 彼の現実の認識は大きく歪んでいました。 彼は病気の重症度を信じることを拒否し、9ヶ月治療を遅らせた 。 「ジョブズの例は、週に80〜90時間働く力を与えてくれました」と、ウォーデルは言います。 「しかし、それから私は自分も人間であることに気づきました。」 ジョブズは彼の子供たちとうまくコミュニケーションしませんでした。 同じ間違いを犯しました。」 現在、ブラッドは自宅でより頻繁に仕事をし、より多くを委任しようとしています。



40歳に近いJobsの多くのファンは、小さくて静かなライフスタイルビジネス（37signalsなど）は、痕跡もなくあなたを飲み込む一生の仕事よりも望ましいという結論に達します。 NASDAQでのIPOの代わりに、年間収益が200、数十万、または数百万ドルではなく、1日6文字、30文字ではなく、2つの製品。



このようなアプローチには存在する権利がありますが、質問をそれほど明確に提示する必要はありません。仕事か個人的な生活のどちらかです。 成功した女性は両方を兼ね備えています。 母性の必要性により、彼らは若者から適切なバランスを構築することを余儀なくされています。 LinkShareの共同設立者であるHeidi Messer氏は次のように述べています。「Jobsは2つの数十億ドル規模の企業、AppleとPixarを設立しました。 ですから、ビジネスと家族を組み合わせることができます。」



この本がリリースされてからしばらくして、Isaacsonはハーバードビジネスレビューに記事を掲載しました。 その中で、彼はジョブズの不機嫌なキャラクターに直るのではなく、アップルとピクサーでの彼の傑出した業績にもっと固執するよう求めました。 Isaacson は、ジョブズの見解を14の短い戒めに要約しました ：「現実を歪める」、「完璧を達成する」、「あなたの周りの大リーグの人々だけを守る」など。 「彼の役に立たない性質は忘れられるだろう」とIsaacsonは書いています、「そして、成果は永遠に歴史に残るでしょう。」 エジソン、フォード、ディズニーとともに。」



アイザックソン自身は、通常、ジョブズへの攻撃に同意しません。 これらの読者が誰か知っていますか？ 投資銀行家。 デリバティブが思いついたものは、何百万人もの人々に貯蓄を与えず、世界的な危機をもたらしました 。」



「人々が自分の家族に対するジョブズの行動を批判するとき、イサクソンは続けます。「どうしてあなたは自分が3回結婚し、娘はもうあなたと話をしないのですか？」 ジョブズの私生活は完璧ではありませんでしたが、最終的に彼は強い家族を作り、最後まで彼を支えた4人の子供を育てました。 なぜ彼を責めるのですか？ いいでしょう 批評家を説得することは、福音の異なるバージョンについて原理主義者と議論するようなものです。



Jobsを例にとると、なぜ私たちがロールモデルを必死に探しているのかがはっきりとわかります。 ジョブズにメンターはいませんでした。 彼は常に自分のビジョンと直感に基づいて決定を下しました。 小規模ではありますが、私たち全員が同じ不確実性に直面しています。 それが、答えとロードマップのために、私たちが偉大な人々の伝記に目を向ける理由です。



パラドックスは、 ジョブズの経験はほとんどの人には当てはまらないということです。 Apple規模の企業を経営し、世界を変えるチャンスを得る人はほとんどいません。 従業員は、ジョブズの例のように扱われれば、単に別の会社に去ります。 私たちの家族はバラバラになります。 ジョブは完全に異なるレベルで行動し、そこでは多くのことが許されました。 他の人にとっては、この動作はひどく終了します。



The No Asshole Ruleの著者であるRobert Suttonは、Jobsが非常に効果的な暴君であると指摘しています。 しかし、大多数の場合、独裁的な習慣は問題のみを生み出すとサットンは主張する。 従業員の創造性と生産性が低下し、仕事に対する無関心な態度が形成され、欠勤と離職のレベルが高まっています。



サットンは、 ジョブズはあまりにも特別で、物議を醸すには向きすぎないと考えている。 これは、起業家や経営者向けのロールシャッハテストです 。 ジョブズでは、誰もが見たいものを見ることができます。
