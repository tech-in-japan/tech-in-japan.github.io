6月7〜9日に無料の仮想会議dotnetConf 2016に参加します



dotnetConfが帰ってきた！ 6月7〜9日に開催される無料の仮想会議に参加して、.NETコミュニティやマイ​​クロソフト製品グループの著名な講演者とともに.NETの世界に浸ってください。



バランスの取れたプログラムをまとめようとしました。 Web、モバイルデバイス、デスクトップシステム向けに開発する方法を学び、さまざまなプラットフォームやデバイスのゲーム、サービス、ライブラリの開発に没頭してください。 .NET Core 、 ASP.NET Core 、 C＃ 、 F＃ 、 Roslyn 、 Visual Studio 、 ユニバーサルWindowsプラットフォーム（UWP） 、 Xamarinなどについて説明します。



カンファレンスの初日は、 スコット・ハンターが.NETの革新と、開発者がプラットフォームやデバイス用のアプリケーションを作成するのにどのように役立つかについてのストーリーで開かれます。



会議の2日目は、 ミゲルデイカザによって開かれ 、XamarinやUWPなどのデスクトップシステムとモバイルデバイスの新しい.NET開発の機会について話します。



カンファレンスの3日目はScott Hanselmanによって開かれ 、「。NETについては？」という質問に答えようとします。 彼は生きていますか？ 何を改善できますか？」



ハッシュタグ#dotnetconfをお楽しみに、Twitter @dotnetで フォローしてください。



会議ウェブサイト： www.dotnetconf.net
