ダウンロード可能なWindows 10 Technical Preview

予備バージョンの入手場所とインストール方法

2. システム要件を確認します 。



3.最適なディストリビューションのいずれかを（ISO形式で）ダウンロードします。

Windows Insiderプログラム



OS開発チームの同僚はよく考えて、ついにこのプログラムを思いつきました。



Windows Insiderプログラムに登録することで、最新のビルドを受け取り、新機能や製品について最初に学ぶことができます。



プログラムに参加することで、オペレーティングシステムの開発に積極的に参加する機会が得られます。製品、その機能に関するフィードバックを提供し、ご要望やコメントをお送りください。 予備バージョンで表示または表示されないものがすべて最終リリースの一部であるか、またはその逆が実装されないという保証はありません。 しかし、レビューを使用すると、このプロセスに影響を与え、Windows 10をさらに使いやすく、便利で便利なものにすることができます。



WindowsフィードバックアプリケーションはOSディストリビューションに組み込まれており、これを使用して、予備バージョンに関するフィードバックを当社と共有できます。







プログラムの参加者がフォーラムを利用して、世界中の専門家から質問をしたり、回答を受け取ったり、必要な情報を探したりできます。 すでにフォーラムには1000を超えるトピックが投稿されています。
