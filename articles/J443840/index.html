Univention Corporate Server（UCS）-Webパネルを備えたシンプルで便利なLDAPサーバーのインストールとNextcloudとの接続





遅かれ早かれ、中小企業では、会社の多数のサービスとポータルに単一のユーザー認証センターを作成するというタスクが発生します。 このような認証センターの候補の中では、Microsoft Active DirectoryまたはLinuxベースの実装の1つがすぐに思い浮かびます。



この一連の記事では、わかりやすいWebインターフェイスと統合されたアプリケーションストアを備えた便利で使いやすいLDAP認証サーバーとしてUnivention Corporate Server （以下UCSと呼びます ）を使用します。 この製品はドイツの会社Univention GmbHによって開発されました。



この記事では、UCSのインストールとNextcloudの展開について説明します。LDAPを介した後続の承認の可能性もあります。



次の記事では、 ZimbraメールサーバーとOnlyOfficeポータルをUCSに接続します。



Univention Corporate Server（UCS）は、Debian GNU / Linuxに基づくサーバーオペレーティングシステムであり、サーバー、サービス、クライアント、デスクトップ、ユーザー、UCSで実行されている仮想化されたコンピューターを集中管理およびクロスプラットフォーム管理する統合管理システムを備えています。 オンプレミスの仮想インスタンスでの作業に加えて、UCSはOpenStack、Microsoft Azure、およびAmazon EC2に基づくクラウド環境でも機能します。



また、Univentionは、オープンソースのSamba 4ソフトウェアの統合により、Microsoft Windowsを実行しているコンピューターを管理するために多くのMicrosoft Active Directory企業が提供する機能もサポートしています。 UCSベースおよびUCS認定のコンポーネント、インストール済みのサードパーティ製品は、Univention Application Centerを介してインストールおよび統合できます。



UCSは、すべてのアプリケーションセンターアプリケーションに、一元化された一貫性のあるアプリケーション管理を含む作業用のランタイムとサービスを提供します。 DockerコンテナはUCSシステムでも実行でき、Application Centerで利用可能なアプリケーションの一部はDockerに基づいています。

Nextcloudは、データ保護機能と制御機能を備えたクラウドストレージであり、オーディオチャットとビデオチャットを接続するためのローカルサービスです。 ダウンロードしたファイルには、任意のプラットフォーム上のサードパーティがアクセスできます。 このシステムにより、同僚と顧客間のワークフローを最適化できます。 iOS、Android、Mac、Windows、Linux、Outlook、およびThunderbirdとの統合が提供されます。



セキュリティが強化されたため、このサービスは、医療従事者がクライアント、教育機関、科学機関、広告やメディアの分野の企業、弁護士、公証人に関する情報を保護することに関心があります。 カレンダー、個人メモ、連絡先、写真の保存など、家庭での使用にも役立ちます。 個人ユーザーの場合は、自宅、データセンター、またはサプライヤのいずれかのサーバーを選択できます。 オープンソースプラットフォームにより、開発者は個人の好みに応じて最適化できます。



Nextcloudは、エンドツーエンドの暗号化を通じてセキュリティを提供します。 コメント内のビデオ通話、チャット、連絡先のタグ付けオプションが利用可能です。 全文検索により、いつでも必要な要素を見つけることができます。 このリソースは、カレンダーの招待処理、自動メール統合、および添付ファイルをNextcloudに直接保存するメールアドレスのセットアップのサポートを提供します。 コラボレーションオプションを使用すると、同僚と共有したり、一般公開したりできるWebメール、連絡先、カレンダーを通じてチームとコラボレーションできます。 すべてのデータは、アカウントの電話とタブレットと同期されます。 このサービスは、Web会議、オンラインモードでのブレインストーム、およびWebセミナーに使用できます。
