Web Summit 2015レビュー

11月3〜5日に、ダブリン（アイルランド）はヨーロッパWebサミット2015で最大の技術会議を開催し、世界中の有力な起業家、投資家、プログラマー、エグゼクティブ、発明家、デザイナー、思想家を集めました。 「オタク向けのダボス」は、ブルームバーグがWebサミット2015と呼んだものです。 イギリスのガーディアンは、フォーラムを市場の動きの決定的なベクトルと呼びました。 「世界中のIT業界のリーダーによる大会議」とニューヨークタイムズ会議は説明しました。 会議に参加した企業の価値をよりよく理解するために、会議のWebサイトからNASDAQを引用しています。「1/4兆ドル-部屋にいるプライベートインターネット企業のコストがありました。」



Web Summit（Dublin Web Summit）は、2010年から開催されている技術カンファレンスです。 長年にわたり、フォーラムの聴衆は400から140か国以上から40,000以上の参加者に成長しました。 今年、1000を超えるスピーカーが21のWebサミット会場で講演しました。 このイベントは約1.5万人のジャーナリストによってカバーされました。



すべてのステージでのパフォーマンスが同時に行われたため、多くの参加者が訪問したいものすべてに参加できませんでした。 彼が興味を持っていた講義を欠席した人や会議にまったく参加しなかった人は、YouTubeのWeb Summitチャンネルでパフォーマンスの記録を見ることができます 。







イベントのゲストの中には、Fortune 500に含まれる世界最大のテクノロジースタートアップの代表者、Microsoft、Stripe、Instagram、Intel Capital、Facebook、Tinder、Slackの代表者、Twitterの創設者であるJack Dorsey、Reed Hastings、創立者を含む他の多くの巨人がいましたPure SoftwareおよびNetflix、Elon Musk、TeslaおよびSpaceXの作成者、Adriana HuffingtonによるHuffingtonポストの作成者、Drew Houston、Dropboxの作成者およびCEO、NicklasZennström、Skypeおよびその他の世界的な著名人の作成者。







Web Summit 2015カンファレンスプログラムは3日間に設計され、テーマ別のセクションに分けられています。スタートアップ、投資家、グラフィックデザイナー、アーティスト、コンテンツ作成者、プログラマー、起業家、マーケティング担当者などのプラットフォームです。





さらに、このカンファレンスでは次のようなユニークなセミナーを開催しました。「スポーツ」-Webサミットの一環として開催され、スポーツの技術に特化したセミナー「ファッション」。参加者はファッションとテクノロジーの統合、「音楽」、生産者、管理者、ビジネスマン、30人以上のアイルランドの食品サプライヤーとシェフを集めたフードフェスティバル、Gorod-技術が都市の品質と生産性の向上にどのように影響するかを議論したプラットフォーム、「Medicine」-セミナー、写真 医学の技術革新に特化した「Machines」は、ロボット工学、機械プロセスの自動化、大規模なデータ配列の分析に特化したセミナーです。







多くのユニークなスタートアップがWeb Summit Alphaでプロジェクトを発表できました。 また、以前は大企業、代理店、銀行のみが通常参加していましたが、今年はWeb Summit Dublin展示複合施設のすべての施設が、投資家やパートナーを探している初心者や小規模企業に与えられました。 サミットの主催者であるパディ・コスグレイブによると、彼のイベントでは、銀行に連絡することさえせずに億万長者になることができ、投資家はスタートアップ自身を見つけるでしょう。







多くの興味深いアイデアと声明がサミット会場で行われました。 その1つは、Sherpa CapitalのSherwin PishevarとHyperloop CEOのRob Lloydによる、8000万ドルの投資プロジェクトの一環として、2600万ドルの投資契約を締結するという声明でした。



Hyperloopバキュームトレインの設計-その作者であるElon Maskの古い夢-は、移動の方法と速度に関する通常の考え方を完全に変えることができます。 約1か月前、Rob Lloydはスタートアップを率いてCEOを引き継ぎ、「スチールパイプを使用して最大700 mphの速度で人と物を動かすというElon Muskのアイデアを商業化する」という目標を掲げました。 ハイパーループにより、乗客は音速で移動でき、サンフランシスコからロサンゼルスまで30分で移動できるようになります。 会社の予測によると、2020年にはすでにHyperloopをテストできます。







ハイパーループとともに、キックスターターも彼のアイデアを共有して脚光を浴びました。 クラウドファンディングプラットフォームから、共同設立者の1人であるYancy Stricklerが会議に参加しました。 アイリッシュタイムズは、ヤンシーの言葉を公式ウェブサイトに掲載しました。



The Irish Timesで述べたように、Kickstarterは通常の利益志向のアプローチの代わりに、登録の条件を最近変更し、株主だけでなく社会への会社の決定の影響を考慮する法的義務を持つデラウェア州の公企業になりました。 Yancy StricklerとKickstarterの共同設立者Perry Chenは、フォーカスの変更がプラットフォームに利益をもたらすと考えています。



Web Summit 2015で、Yancyは彼の会社の視点も共有しました。「独立したデザイナーや開発者は、単に気に入ったからという理由で、伝統的な投資家が十分に魅力的とは言えないカテゴリーに取り組むことができます。 最も素晴らしいアイデアは、お金を稼ぐことに興味のない人から生まれます。彼らは自分のアイデアが存在したいだけです。 私たちは人々が人生にもたらしたいものを創造できるようにしなければならないと感じています。」







会議のセクションの1つであるMoney Summitは、電子決済に完全に専念しました：暗号通貨、支払い、個人金融、詐欺、金融サービス。 Money Summitは、支払い技術の分野で成功したプレーヤーとイノベーターを集め、支払いの分野で革新的なアイデアを議論するために開催されました。 PayPal、Stripe、BitPay、Visa、MasterCard、TransferWiseなどの企業の世界で最も影響力のある専門家、創設者、経営幹部は、Web Summit Moneyを訪れた人々のほんの一部です。



プライベートホットスポットであるOpen Wi-FiのJohn Delanyは、最終製品がどうあるべきかというビジョンを共有しました。 複雑にする必要はありません。 私たちは自分の経験から確信していました。問題が発生し、あなたが何もしなかった場合、プロジェクト全体が失敗する可能性があります。」



また、ジョンは次のように付け加えています。 すべてを制御したい。 これは驚くことではありません。 私たちは何年もこれに行ってきました。 しかし、私たちは分析に興味がありません。すべてをすでに行っているので、これをしようとはしていません。 私たちは、人々が顧客と接触するのを助けるだけです。



サミットで発言された興味深いアイデアの中には、SpalmaloのAdilet Abylovのアイデアがあります。「車は嘘をつくことができないので、車は人々を幸せにします。 彼らは多くの害を及ぼすことはできません。 人々は機械とより密接に連携すべきだと思います。」



Web Summitブログで公開されている統計から：会議では95,000を超える会話が行われました。 パフォーマンスの平均所要時間は9分14秒でした。



Web Summit 2015カンファレンスの多くの参加者は、アイルランド政府とサミットの主催者との対立についても記憶されるでしょう。 イベントの開催前でさえ、主催者はダブリンが最後にゲストを迎えることを発表し、来年は会議がポルトガルの首都に移る予定です。 Paddy Cosgraveによると、この決定の理由は、アイルランドの首都の技術設備の不足とインフラストラクチャの組織の不足でした。 ヨーロッパ最大の技術フォーラムの延期のニュースは、アイルランド政府に懸念を引き起こしました。 アイルランド産業開発庁（AIR）は、南国が潜在的な起業家をそらすため、Webサミットリサボンが成功しないことへの懸念を表明しました。





Paddy Cosgrave Summitオーガナイザー



それはともかく、Web Summitはどの国でも喜んで開催できる会議の1つです。 Paddy Cosgraveは、膨大な数の影響力のあるプレーヤーを彼のプロジェクトに引き付ける能力を明確に実証しています。 主催者は常に努力し、改善するものを持っていますが、会議の問題は、ダブリンに集まった人々や企業のおかげで、イベントで支配されていたインスピレーションに満ちた雰囲気と比較して色あせています。
