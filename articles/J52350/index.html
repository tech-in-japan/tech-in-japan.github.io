Zendは独自のサーバーアセンブリのベータ版をリリースしました

Zendは、PHP開発者向けの商用製品のリストを拡大しました。

Zend Serverは、Webサーバー、php自体、Zendのいくつかのツール（オプティマイザー、デバッガーなど）、Zend Framework、監視ツール、さまざまなDBMSを含むサーバーアセンブリです。



プロジェクトはベータ版で、WindowsおよびLinuxで利用可能です。 Community Editionの無料バージョンがあります。



現在、Zend Serverには次のものが含まれています。

  • Apache 2.2.8（またはプリインストールされたIISを使用）
  • PHP 5.2.8とその拡張機能の束
  • Zend Optimizer +
  • Zend guard loader
  • Zendデバッガー
  • Zendキャッシュ
  • Zend Javaブリッジ
  • Zendモニター
  • Zend Data＆Page Cache
  • Zendダウンロードサーバー
  • Oracle OCIドライバー
  • Zend Framework 1.7.4
追加オプションとして、インストーラーは以下をダウンロードします。
  • phpMyAdmin
  • IBM DB2 RTCL
  • MySQLサーバー
  • MS SQL Native Client
