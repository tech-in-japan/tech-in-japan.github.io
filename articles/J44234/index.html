ラップトップの「改造」、3Gモデムのインストール

3Gネットワ​​ークの普及が始まった後、特にサンクトペテルブルクで、もちろん、私はこの機会を利用して、家の外でも十分な速度でインターネットにアクセスしたいと思っていました。

しかし、3Gモデムは多かれ少なかれ適切な価格で「USBコード」上にあるか、より便利な形式（ExpressCard、miniPCI-e）でありながら信じられない価格であることが判明しました。 また、景品が欲しかったというわけではありませんが、モデムに6 tr以上与えます。 欲望はありませんでした）。

（注意、たくさんの写真）

私の友人と私は彼のラップトップの「改造」を決めました。

1999rのビーラインオフィスで3Gモデム（ZTE MF662）を購入しました（広告なし、事実のみ）。





すぐに開梱、テスト、および分解します（いつものように静かに）







内容はminiUsbポートを備えたボードでした（奇妙なことは何もありません）







ラップトップの「改造」、ブルートゥースアダプターのインストールについての記事で、マザーボード上のブルートゥースアダプターのコネクターについて話しました。3Gモデムの接続に使用します。



USBポートの配線を思い出します。

ピン1 5ボルト

ピン2 USB D-

ピン3 USB D +

4番ピン



マザーボード上のコネクタの配線：

ピン1 5ボルト

2番ピン

ピン3 USB D +

ピン4 USB D-



それぞれ次のように接続します。

ピン1-1

ピン2-4

ピン3-3

ピン4-2



Acer Aspire 5100ノートブック

（私はラップトップを分解する詳細を省略していますが、必要がないためです）



適切な場所の検索中に、SIMカードを変更する可能性があるような方法でモデムをインストールすることが決定されました。 そのような場所は事件の裏にありました。





空き領域を確保するためにいくつかのパーティションが削除され、モデムが配置されました





SIMカードスロットがラップトップの背面にきちんと見られていた





マザーボードのコネクタは前面にあり、はんだ付けされていました





「フィールド」テストが実施され、成功したことが判明しました:)





ワイヤーは対応する「溝」に配置されます







ラップトップは最終的に組み立てられ、起動されます。

すべてが指示に従って行われたため、ソフトウェアのインストールプロセスを説明する理由はありません。

モデムのオン/オフは、フロントパネルのボタン（Bluetoothの代わり）によって実行されます。





お楽しみください！



PSビーラインソフトウェアでも、モデムはメガホンSIMカードでも動作しますが、フォーラムで「ブロックされた」モデムが外国のSIMカードでの動作を拒否する場合があります。

ppsアンテナを埋め込む問題は今のところ開いていますが、モデムはそれなしでも正常に動作します:)
