トランセンドとキングストンのオールインワンカードリーダー

先日、Webに2つのかなり興味深いデバイス-カードリーダーに関する情報が登場しました。カードリーダーは、さまざまな形式のフラッシュカードの印象的なコレクションの所有者が注目する価値があります。 ガジェットの1つであるKingston Media Reader（「インベントリ」番号FCR-HS219 / 1）は非常に機能的で、19種類のフラッシュドライブをサポートしています。 2つ目はTranscendのM5です。これは、膨大な数のフォーマットのカード（Kingston製品のように19種類あります）で動作するだけでなく、素敵でスタイリッシュなデザインにも満足しています。



Kingston Technologyの「メディアリーダー」ファミリーの補充は、寸法が92×69×20 mm、重さ71グラムで、USB 2.0インターフェイスを備え、Windows 2000、XP、およびVistaオペレーティングシステムで動作します（十分ではありません）。 2本のケーブルのデバイスの納入セットに存在することを強調する価値があります。ラップトップに接続するための短い（10 cm）と延長（約1メートル）です。



すでに述べたように、Kingston Media Reader FCR-HS219 / 1の主な利点は、19種類のフラッシュカードをサポートしていることです。 完全なリストは次のとおりです。CompactFlashType IおよびII、SD、SDHC、miniSD、miniSDHC、microSD、microSDHC、MMC、MMCmobile（DV RSMMC）、MMCplus、MMCmicro、RS-MMC、Microdrive、Memory Stick Micro（M2）、メモリースティック、メモリースティックPRO、メモリースティックデュオ、メモリースティックPROデュオ。



FCR-HS219 / 1オールインワン価格は17ドルです。



M5と呼ばれるトランセンド製品については、機能的であるだけでなく、外観も非常に魅力的です。 このデバイスは、コンパクトさ（寸法70×70×15 mm）とスタイリッシュなケース（色も異なります-白と赤、またはトランセンドが示すようにピンクとアイボリー）。



FCR-HS219 / 1とは異なり、このデバイスは2000、XPおよびVistaだけでなく、Windows 98SE / Me、Mac OS 9.x（以降）およびLinux Kernel 2.4（以降）でも動作します。 サポートされているカード形式には、新しい標準MMC4.0があります。 Kingston Media Reader FCR-HS219 / 1と一致するため、フォーマットの完全なリストを提供することは意味がありません。



このデバイスの価格（メーカーのWebサイトで、税抜き）は14ドルです。
