マイクロソフトの成功の触媒

ほぼ2年前の2008年11月に、MicrosoftはBizSparkを立ち上げました。 スタートアップが最も必要で、それを買う余裕がほとんどないとき、つまり旅の始まりに、Microsoftソフトウェアにアクセスできるようにすることで、スタートアップを支援するプログラム。 プログラムに参加するために必要なものはありません。ソフトウェア開発者であるか、新会社を設立するだけで十分です。プログラムの開発はビジネスの基礎です。



それから多くの時間が経ち、何百もの企業がBizSparkプログラムに参加しました。そのほとんどは、BizSparkのマイクロソフトのパートナーであるSoftlineのおかげでこのプログラムについて学びました。

ほぼ2年間、スタートアップはライセンスされたMicrosoft製品に定期的にアクセスし、プログラムに触発されて集中的に開発を開始しましたか？ または、目的のソフトウェアを受け取った後、静かに影に消えましたか？



開発者がMicrosoft BizSparkからライセンス取得済みのMicrosoft製品をダウンロードするために切望されたキーをメールで受け取った後、開発者はどうなるかを確認しました。

だから、ほとんどゼロから始まったMagnumSoftの歴史について簡単に説明し、Microsoft BizSparkプログラム、Dmitry Fedorchenko、MagnumSoft Ltdディレクター、PLACE.BYプロジェクトマネージャー、およびイベントへの直接参加者は語っています。







道の始まり

2009年にBizSparkプログラムに参加しましたが、市場で会社を発展させるための明確な戦略がまだなかったため、どこから始めて、どの分野で開発するべきかさえ分かりませんでした。 私たちには明るい頭脳、ITでの質の高い質の高い教育、そして国のために質の高い有益な何かをしたいという大きな願望しかありませんでした。 ちなみに、彼らはインターネットでニュースを閲覧しているときに、偶然にBizSparkプログラムについて知りました。 私は夢中になり、情報を読んで、ミンスクで開催されたマイクロソフトのカンファレンスに行き、プログラムに専念し、BizSparkが私たちのチャンスだと決めました。



ソフトウェア

Bizsparkに参加することで、マイクロソフトのテクノロジを使用した社内ソフトウェア開発をすぐに開始しました。 ちなみに、3年間（ソフトウェアが提供される期間）後、これらのライセンスを購入する予定です。 現時点では、当社のソフトウェア製品はMicrosoft Windows Server 2008 R2のすべてのサーバー容量を使用しています。



開発

独自の開発のために、最も高度なツールを選択し、資格のあるチームを選択しました。 1年以上にわたり、PLACE.BYソフトウェア製品を作成してきました。これは、Microsoft .NETテクノロジーを使用して、ベラルーシのあらゆるイベントのバーコード形式で電子チケットを予約および販売するシステムです。 次に、PLACE.BY（当社のソフトウェア製品）に基づいた当社は、主要なベラルーシのインターネットポータルTUT.BYとともにプロジェクトBILET.TUT.BYを立ち上げました。ベラルーシの電子チケット市場でN1の地位を獲得しています。



今後の計画

現在、私たちはベラルーシの電子チケット市場で主導的な地位を目指しています。そして...（すべてのカードを公開するまでには目標があります。）1つ言えるのは、Microsoft Bizsparkプログラムが開発技術の選択で主導的な役割を果たしたことです。 したがって、このプログラムがなかったら、おそらく別のPHPプロジェクトがあったはずでしたが、現在では、最新の開発テクノロジーに基づいたハイテク製品があります。 開発のためには、常に新しいものを探す必要があります。私たちにとって、Bizsparkはまさにその新しい触媒になりました。
