  • Subversionをインストールします（以下を参照）
  • TortoiseSVNをインストールします（以下を参照）

  • SVNをApacheに接続する
  • 途中でSVN認証を設定する

インストールの準備



あなたが必要になります

  1. Apache 2.0.63 Webサーバー
  2. Subversion 1.5.2バージョン管理システム（SVN）
  3. Windows SVNクライアントTortoiseSVN 1.5.3
  4. プロジェクト管理およびバグ追跡システムTrac 0.11.1
ディレクトリCを作成します：\ usr \ local \ここにインストールします以下のファイルをダウンロードします：
  1. Subversion 1.5.2〜CollabNet Subversionコマンドラインクライアントv1.6.12（Windows用） （登録が必要）
  2. TortoiseSVN 1.5.3〜TortoiseSVN -1.6
  3. Apache 2.0.63〜apache_2.0.63 -win32-x86-openssl-0.9.7m.msi
  4. Python 2.5〜python-2.5.msi
  5. Python SetupTools〜setuptools -0.6c7.win32-py2.5.exe
  6. mod_python 3.3.1〜mod_python-3.3.1.win32-py2.5-Apache2.0.exe
  7. SQLite 2.4.0〜pysqlite-2.4.0.win32-py2.5.exe
  8. SVN 1.5.2 Pythonバインディング〜svn-python-1.5.2.win32-py2.5.exe
  9. Genshi 0.5.1〜Genshi -0.5.1.win32-py2.5.exe
  10. Trac 0.11.1〜Trac -0.11.1.win32.exe




設置





Subversionをインストールする

  1. Setup-Subversion-1.5.2.en-us.msiを実行します
  2. Apache 2.0.xのバインディングを選択します

  3. インストールディレクトリを指定する 
      C：\ usr \ local \ Subversion


  4. インストールが完了するまで待ちます。

TortoiseSVNクライアントをインストールする

  1. TortoiseSVN-1.5.3.13783-win32-svn-1.5.2.msiを起動します
  2. インストール中は、すべての値をそのままにしておきます
  3. インストールが完了するまで待って、コンピューターを再起動します。

ストレージを作成する

データを別の論理ドライブに保存し、さらに別のハードウェアドライブに保存したい場合、この場合、すべてのストレージはD：\ repos \ディレクトリに保存されます
  1. ディレクトリDを作成します：\ repos \
  2. 作成されたディレクトリに移動し、2つのサブディレクトリを作成します。
    • store1
    • store2
  3. 2つのストレージを作ります
    • store1を右クリックし、メニューからTotrtoiseSVNを選択->ここにリポジトリを作成

    • store2に対して同じ手順を繰り返します
    • 今、あなたは2つのストレージSVN store1とstore2を持っています
