VDSラインを更新しています！

みなさんこんにちは！



HabréでのVDSに関する最初の記事以来、 サービスの立ち上げを成功裏に完了し、 サービスの提供で膨大な経験を積み、話したいいくつかの新製品を発表しました。









プロセッサコア



新しいプールでは、Intel Xeon E5プロセッサの使用を開始しました。 以前のプロセッサよりも50％強力です。 これは、同じ価格で1.5倍の生産性の高いVDSを利用できることを意味します（課金はプロセッサコアで行われます）。



新世代プロセッサへの移行により、4チャンネルメモリを使用できるようになり、そのスループットは以前のタイプのメモリのほぼ2倍になりました。



SSDドライブ



VDSを起動すると、ディスクサブシステムが最も弱いリンクであることが明らかになりました。 そのため、新しい仮想サーバーでは、ストレージをSSDディスクで完全に使用し始めました。 高価ですが、はるかに高速です！



SSDへの移行を伴うサービスのお客様の価格は変更されません。



メールが登場しました。



共有ホスティングのユーザーは、電子メールサービスがメインサービスの一部であるという事実に慣れており、個別に提供できるという事実についても考えていません。



VDSの使用を開始したお客様から、メールの投稿方法に関する質問がよく寄せられます。 Webサービスを管理する準備ができており、NGINXとApacheを自分用に構成する方法を知っており、負荷を拡大することができます。 しかし、メールはまったく別の問題であるため、強力なメールクラスターを使用するようにお客様を招待しました。



VDSアップデートの一部として、メールサービスを追加しました。 これで、1回のクリックで1つのパネルから管理用にメールを接続できます。 また、ボックスの作成と構成は非常に簡単になりました。







メールボックス管理の一部として、さまざまなTIMEWEBメールボックス機能にアクセスできます。 さらに、メールの操作には、12月に発表された Webメールを使用できます。



マップバインディング



プラスチックカードによる支払いが最も便利な支払い方法の1つであることは、誰もがよく知っています。 定期的な自動支払いを導入することにより、サービスの支払いを簡素化することにしました。これは便利です。



カードから自動的にお金を引き出すことが許可されている場合、特定のしきい値に達すると、お金は翌月の仕事の代金として引き落とされます。 完了した操作についてSMSと電子メールを受け取ります。



ただし、クレジットカードの詳細は保存しません。 支払いシステムは、正常に完了した支払いを新しい金額で繰り返すだけで、個人データの最大限の保護が保証されます。







VDSクローニング



多数の仮想サーバーで高負荷のソリューションを構築しているお客様のために、VDSクローンの可能性を準備しました。 この機能により、クラスターのスケーリングに必要な時間が短縮されました。 これで、複数の仮想マシンをコピーし、サーバー構成に必要な変更を加えて、すぐにそれらをクラスターに含めることができます。 スケーラブルなシステムの展開がはるかに簡単になりました。







スナップショットを自動的に作成する



スナップショットを毎日手作業で作成するのが面倒なことは何ですか？ =）バックアッププロセスを自動化しました。 これで、コピーの数とバックアップの頻度を構成できます。 起こりうる障害から確実に保護されます。



スナップショットの価格を大幅に引き下げました。 これで、1ギガバイトのストレージには4ルーブルしかかかりません。







CDからのダウンロードとシングルユーザーのダウンロード



前回の発表後に導入した最初の機能の1つ。 通常のブートモードでは不可能な多くの操作を実行できます。 この機会が解決する主なタスクの短いリストは次のとおりです。



  • ルートパスワードの変更。
  • メインディスクのサイズを変更します。
  • 障害後のシステム回復。
  • ルートファイルシステムを確認します。








更新されたOSセット



VDSにオペレーティングシステムをインストールすることの人気を分析し、いくつかの置き換えを行いました。 Scientific linuxなど、ほとんど使用されないオペレーティングシステムは削除され、新しい人気のあるオペレーティングシステムが追加されました（Ubuntu 12.10、Fedora 18）。



API



サービスの開始以来、APIの提供に対する多くの要求がありました。 これは私たちの計画の中にあったので、私たちはその開発に携わったことを喜んでいた。 APIはPOSTリクエストの原則に基づいて動作するため、あらゆるプログラミング言語で簡単にリクエストを作成できます。



APIドキュメントは、 help.timeweb.ru / apiで入手できます。



サービスユーザードキュメント



サービスのドキュメントを拡張して、VDSを使用するための最も一般的なシナリオ、更新されたパネルで可能になった新しい操作（メインディスクのサイズ変更など）、および一般的な質問への回答に役立つ従来のFAQを含めました。



開発と計画



私たちはそこで停止するのではなく、VDSサービスの改善に取り組み続けます。 当社のウェブサイトおよびHabréのブログで発表をフォローしてください。
