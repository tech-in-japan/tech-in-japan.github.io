オールインワンタッチスクリーンEeeトップ

ASUSは、人気のあるEee PCファミリーの新しい代表者（タッチスクリーンを備えたオールインワンPC）のWeb仕様を公開しています。





Eee Top ET1603およびET1602は 、周波数1.6 GHzのIntel Atom N270プロセッサーに基づいて設計されていることがすでに知られています。 マザーボードのチップセットもIntel-945 GSEです。 両方のモデルには、1 GBのDDR2 RAM、160 GB SATAハードドライブ、イーサネットアダプター、Wi-Fi Draft N、カードリーダー（SD / SDHC / MS / MS Pro）、各4 Wのスピーカーペア、1 3メガピクセルのウェブカメラ。 両方のモデルには、15.6インチのワイドスクリーン（16：9）ディスプレイが搭載されています。 このすべてがWindows XP Home Editionの制御下で機能します。



2つの新しいEee PC担当者は、ビデオカードが異なります（ET1603にはATI Mobility Radeon HD 3450があり、ET1602には組み込みのチップセットが使用されています）。 さらに、より高度なET1603にはBluetoothモジュールがあります。



これらのデスクトップは、2009年の第1四半期に市場に投入されます。 価格については、まだデータがありません。 未検証の情報に加えて、デバイスのコストは500ドル未満になります。
