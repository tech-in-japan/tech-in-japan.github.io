省エネのさまざまなモードでのコンピューターの電力消費と機能の研究





ほとんどの人はシャットダウン時にデスクトップまたはラップトップの電源を切ります（Windowsでは「シャットダウン」）。 コンピューターを常にオンにしておく人もいます。 さて、残りの少数派は省エネモードを楽しんでいます。



スリープモードおよび休止状態モードでのコンピューターの電力消費量と、ハードドライブを搭載したコンピューターと比較してコンピューターとプログラムがどれだけ速く読み込まれるかについてのSSD所有者の最近のレプリカを見つけたいというこの願望に触発されました。 しかし、まず最初に。



ebayで、20ドル相当のWANF家庭用電流計（中国製）を購入しました。 そのため、この電流計は消費電流を表示しませんが、ネットワーク内の電圧と消費電力を表示します。 このデバイスの目的は、デバイスに含まれる機器の消費電力を表示することです。



コンピューター構成のテスト：

  • プロセッサー：QuadCore Intel Core i5-2400、3200 MHz（ネイティブ周波数）
  • マザーボード：Intel Cougar Point H67、Sandy Bridge（ASRock H67M-GE）
  • システムメモリ：16GB DDR3-1333（4x4GB）
  • 3Dアクセラレーター：AMD Radeon HD 6870（標準周波数）
  • モニター1：Asus VW266H [26 "LCD] 1920x1200
  • モニター2：BenQ FP93GX [19 "LCD] 1280x1024
  • 電源：600 W（FSP FX600-GLN）
  • ハードドライブ3台+ DVDドライブ1台
  • Windows 7 x64


この構成は、在宅勤務のコンピューターではかなり一般的です。



エネルギー消費の実際の測定に移りましょう。 さまざまなACPIモードを検討する



状態S3（スリープ）-3.5ワット。



この状態では、マザーボードとRAMに電力が供給されます。 プロセッサ、グラフィックスカード、その他の周辺機器は無効になっています。 スリープモードからオペレーティングモードへのコンピューターの終了はほぼ瞬時であり、実行中のすべてのプログラムがスリープ状態になる前の状態のデスクトップが表示されます。 キャッシュされたデータを含むRAMの内容は完全に保持されます（プリフェッチおよびスーパーフェッチ）

スリープモードは、日常のデスクトップでの使用に非常に便利です。 午前中にコンピューターの電源を入れます-そして出来れば、すぐに作業できます。 SSDまたはHDDディスク-関係なく、コンピューターはすぐにオンになります。 キャッシュされたデータはRAMに保存されるため、アプリケーションの再起動はコンピューターのコールドスタートに比べて非常に高速であり、SSDとHDDの速度の差が解消されます。

エネルギー消費量が非常に少ないため、このモードを毎日使用すると、1か月あたり電気代がたった10ルーブルしかかかりません。 Wi-Fiルーター、VoIPアダプター、およびスタンバイモードのテレビは、それぞれ4〜5 W消費します。



S4 / S5状態（休止状態/オフ） -BIOS設定に応じて1.7 Wまたは0W 。



S4とS5の電力モードは似ています。 デフォルトでは、マザーボードにのみ電力が供給され、RAM、プロセッサー、およびその他の周辺機器は無効になっています。 ただし、BIOS設定では、Deep（S4）/ S5を有効にできます。 この場合、マザーボードもオフになり、コンピューターは何も消費しません。 このオプションの欠点は、電源ボタンでのみコンピューターの電源をオンにできることです。一方、最初のケースでは、RTCアラームイベント（Windowsのタスクスケジューラータスクタイマーなど）、Wake-on-LAN、キーボードのボタンを押すか、マウスなど



モード自体の詳細。 S4-休止状態（休止状態）-コンピュータの電源を切る前のRAMの内容がディスク（読み取り専用メモリ）にフラッシュされるモード。 コンピューターの電源を入れると、RAMの内容がディスク上のダンプから復元され、コンピューターが休止状態に移行する前と同じ状態になります。 この点で、休止状態はスリープモードに似ていますが、作業の準備はすぐには行われませんが、約30秒（システムドライブとしてハードディスクを使用する場合）

また、RAMからキャッシュされたデータはダンプに保存されないため、休止状態から戻った後のアプリケーションの起動は、コールドスタート後のように遅くなります。 デスクトップの場合、このモードはスリープと比較して利点がなく、ラップトップにより適しています。



状態S5-通常のシャットダウン。 ここで言うことは何もありません。



比較のために、オペレーティングモード（状態S0）でのコンピューターの消費量を示します。



アイドル消費-95ワット。 最初のモニターはディスクリートRadeonグラフィックカードに接続され、2番目のモニターはプロセッサーに統合されたグラフィックカードに接続されます。 両方のモニターを個別のグラフィックカードに接続すると、消費電力は30ワット増加し、125ワットになります。 したがって、2番目のモニターをマザーボードに接続することをお勧めします。 エネルギーの節約に加えて、Intel Quick Syncテクノロジーを使用して、H.264へのビデオ変換を（ディスクリートグラフィックカードよりも）大幅に高速化できます。 少なくとも1つのモニターが統合ビデオに接続されていない場合、クイック同期は使用できません。
