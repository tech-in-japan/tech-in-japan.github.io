Windows XPモード





Microsoft Wordの2つのバージョンが実行されていることがわかります。

Windows XPモード（XPM）は、Windows XP用に作成されたプログラムの互換性の問題を解決します。 ユーザーには、完全ライセンス版のWindows XP SP3を備えたVirtual PC上の仮想マシンが提供されます。 この仮想マシンはWindows 7に統合され、仮想環境にアプリケーションをインストールすると、ホスト上でもプログラムが利用可能になります。 せっかちな人はここでこの記事を読んでさらに知り合いになることができます。



音楽コレクションを共有する





クールは、セットアップがずっと簡単だった場合に最初に頭に浮かんだものです。 そのため、Windows Media Player（WMP）の「リモートメディア共有」機能を使用すると、独自のライブラリにアクセスできます。 確かに、Live IDが必要です。個人が電子メールで登録するときに、通常の「オンラインID」で機能するかどうかはわかりません。 そしてもちろん、もう一方の端で同じバージョンのWMPが実行されているはずです。 そして、すべての設定が完了すると、ライブラリがリモートコンピューターに表示され、ローカルコンピューターのように扱うことができます。 これはiTunesではありません。
