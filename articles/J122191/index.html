アイデア



自由に配布される（Webでの使用を含む）PT SansおよびPT Serifフォントでは、文字間にルーブル記号があります。 これらのフォントをサイトで使用する場合、問題を解決するために、余分なhtml-code、css-tambourines、ベクターグラフィックス、およびその他のJediの倒錯なしに実行できます。



Googleによる救助



Googleには、Webページでフォントを使用するのが非常に簡単で合法なすばらしい Web Fontsサービスがあります。 キリル文字フォントはそれほど多くありませんが、PT SansとPT Serifが必要です。 同時に、Googleはブラウザごとに、CSSコードと目的のフォントファイル形式を提供します。



これは、単に不名誉を与えるために行われます。



1. cssファイルを必要なフォントに接続します。 

<link href="http://fonts.googleapis.com/css?family=PT+Sans:regular,italic,bold,bolditalic"
rel="stylesheet" type="text/css" />


2. font-face: 

.price {
  font-family: 'PT Sans', serif;
  }


3. :
