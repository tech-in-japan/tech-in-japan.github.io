無料のウェブセミナー「監査ファイルサーバー」にご招待します



同僚！

6月28日、モスクワ時間の15:00に、 NetWrix File Server Change Reporterを使用したファイルサーバーの監査に関するウェビナーを実施しています 。



NetWrixのロシアのオフィススタッフは、ウェビナーで、標準監査システムよりもNetWrix File Server Change Reporterを使用してファイルサーバーを監査する利点について説明します。



組織内の重要な情報の最大80％がサーバーに保存されます。 そして、そのような情報へのアクセスは制限されるべきです。 したがって、重要な情報の変更、削除、または漏洩を防ぐために、アクセス許可の変更や不正なアクティビティを管理者に知らせる必要があります。



ファイルサーバー監査を使用すると、同様の（だけでなく）質問に対する回答を得ることができます。



*誰が重要な会計文書を削除しましたか？

*ファイルのアクセス許可の設定はどのように変更されましたか？

*ドキュメントを開きたいが、できなかったのは誰ですか？



ウェビナープラン：





1. NetWrix File Server Change Reporter vs. イベントログ：ファイルサーバーを監査する2つのアプローチの基本的な違いは何ですか

2.ファイル、フォルダー、およびアクセス許可の変更に関するレポートがプログラムに含まれます。 短いレビュー

3.ファイルへの不正アクセスを取得しようとしているユーザーを見つける方法- アクセス監査レポートの機能

4. 変更レポートへのサブスクライブ -経理担当者のみが経理文書のあるフォルダーの変更について知っていることを確認する方法

5. NetApp FilerやEMC VNX / VNXe / Celerra などのストレージデバイスを監査する方法



などなど！



ウェビナー期間 ：1時間-15から16.00モスクワ時間
