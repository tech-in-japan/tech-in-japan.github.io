CES 2012 // Huawei P1およびP1 S

中国人は魅惑的で奴隷的であり、すぐに私たち全員がそこにいます-これは事実です。 これは、Lenovoだけでなく、他の多くのブランドでも見られます。 たとえば、Huaweiによると、非常によく似ていますが、かなり興味深いが、高品質のAndroidスマートフォンを2つ展開しています。







それらはほとんど同じと呼ばれます-P1とP1 S、それらはほとんど同じに見え、違いに気付くことは困難です。おそらく、これら2つのスマートフォンを何らかの形でセグメントに分けることは中国人にとって価値があるでしょう。 しかし、それらについて最も驚くべきことは、Motorola DROIDからコピーされた極薄のシェルとデザイン（誰も驚かなかっただけではありません）ではなく、4番目のAndroidです！ そしてすぐに両方のガジェットで。
