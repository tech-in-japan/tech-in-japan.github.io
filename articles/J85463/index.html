Igelle DSV 1.0.0





最近-数日前-Linux Igelleディストリビューションのバージョン1.0.0がリリースされました。 これは、色分けされたUbuntuではなく、言葉の意味でのディストリビューションです。 ここでは、独自のEstherウィンドウマネージャー、Sjappパッケージ管理システム、さらにはブラウザーもFireFoxではありません。





Igelleは、ハードドライブにインストールできるライブCDとして提供されます。 読み込みが非常に速く、驚きました。 そして、それは特別な知識やスキルを必要としません-それはただロードされ、それだけです。 ハードドライブにインストールすることにしました。





彼はどこにインストールするか尋ねます。





その後、彼はパーティションをフォーマットし、自分自身をそれらにコピーします。 すべてが驚くほど速く起こります。









再起動後、Igelleはホスト名、タイムゾーンの指定、およびユーザーアカウントの作成を求めます。 別の再起動、そしてそれだけです、あなたは働くことができます。





ホームフォルダー、アプリケーションリスト、システム設定、ドキュメント、特定のタスクマネージャー、ビデオプレーヤー、不気味なOsmoオーガナイザー、電卓、画像ビューアー、Webブラウザー、gEditなど、画面の下部の境界線からクイック起動アイコン付きのパネルが偶然出てきました。





タスクマネージャは、Windowsの同じ名前のプロセスのリストとtopのクロスです。 プロセスのステータスと優先度を変更したり、強制終了したりできます。





独自のWebブラウザー。 Apple WebKitに基づいて構築されたWebBrowser 2.28.1と呼ばれ、Epiphanyを認識できました。





Igelle自身のリポジトリはかなり貧弱です。 しかし、必要なものはすべてあります。 OpenOffice、FireFox、ThunderBird、PHP、RapidSVN、Pidginなど。



さらに、Sjappというツールがあります。 パッケージマネージャー（sjapp install pre-downloaded-package.sjapp）として、またソースコードから何かをコンパイルするためのツールとして、かなりトリッキーな方法で機能します。



sjappcコンパイラは、事前に準備されたSysJobスクリプトファイル（* .sjs）を入力ファイルとして受け入れます。これは、最も単純な場合、次のようになります。 
  geany：sjapp {
       setself "url" "http://download.geany.org/geany-0.18.tar.bz2"
 }
Sjappのドキュメントから例を取り上げました。 それは

sjappc geany.sjs

そして、geanyエディターがインストールされています。 まあ、インストールする必要があります。



配布は高速で軽量で、必要なものがすべて含まれています。 ラップトップに最適。 カーネル2.6.32で登場しました。
