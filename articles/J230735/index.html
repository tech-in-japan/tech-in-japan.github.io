jabber.ruサーバーを使用したゲーム

アクティブおよびパッシブ Linuxユーザーの間では、ジャバーを除き、メッセンジャーの使用を非難するのが慣習です。 ただし、jabber.ruサーバーが会議での個人的な通信を読むような多くの「楽しいゲーム」を隠すことを知っている人はほとんどいません。 この記事では、私たちが見つけた最も魅力的なエンターテイメントについて説明します。







ゲーム1：友達をすべて見つける



通常のサーバーとは異なり、すべてはユーザーとその利便性のために行われます。 たとえば、単純な非特権ユーザーは次のリクエストを実行できます。 

<iq type="get" to="_@conference.jabber.ru"> <query xmlns="http://jabber.org/protocol/muc#admin"><item affiliation="member"/></query>


応答として、会議の登録メンバーの完全なリストを受け取ります。 あなたのものかそうでないか。 プライバシーは必要ありません。



はい、これはユーザーリストを編集するために行われる管理要求です。 ただし、jabber.ruの管理では、顧客に非常に注意を払っており、管理権限がない場合でも、すべての友人を見つけることができます。



プログラミングが苦手な場合は、ゲームサービスのフロントエンドと見なすことができるシンプルで直感的なコンソールアプリケーションgist.github.com/7316aec410cbecd4d97a.gitに注目してください。そしてシンプル。 すべての友達と敵を見つけて、インターネット上で彼らが間違っていることを証明してください！



すみませんが、これは単なるjidリストです。コンピューターインターフェースを介してあなたの愛を表現することは可能ですか？

はい！ 今では可能です！

たとえば、承認リクエストを介して約500件の友情リクエストを送信することで、友情を証明できます。 現代のxmppクライアントは非常に満足しているので、多くの幸福から飛び立つことがよくあります！



これが完成したツールキットです。すべてがオープンソースで無料です： github.com/Kagami/kisa



あなたの友情が永遠になりますように！



ゲーム2：すべてを知りたい



おそらくjabber.ruの最もユニークな機能は...他の誰かのPMを読むことです！ 他の人のメッセージを読むだけでなく、それらを置き換えることさえできます！



仕組み：

会議設定では、メッセージをフィルタリングするためのサービス（jid）を指定できます。これは、原則として、会議に参加する必要がある何らかのボットです。 もちろん、このようなボットがない場合は、すべてが人のために行われるため、多数の会議で簡単にそれを借りることができます。 まあ、または自分で書いてください。



ボットには管理者権限は必要ありません。ボットは訪問者であっても、会議に参加する必要があります。 彼が必要とする唯一のものは、jabber.ruで開発された特別に形成されたIQリクエストに応答することです。 試してみて、後悔しないでください！ もうしますか？



jabber.ruブログの公式指示： jabber-ru.ya.ru/replies.xml?item_no=4659



実験的なホールがjabber.ruに、あるいはconference.jabber.ruに登場しました。これにより、一般にボットと呼ばれる外部サービスによる会議トラフィックのフィルタリングが可能になります。 会議の所有者は、わいせつな言葉、長すぎるメッセージ、その他の無意味なゴミの使用から自分の領域を独立して保護することができます！

会議の設定には、ボットのjidを指定できるオプションがあります。これにより、トラフィックが自身を介して会議に送られるようになります。 このトラフィックのステーションは次のようになります

id = '1' type = 'set' to = 'jid @ bot'>

<メッセージオリジナル標準>



応答として、ボットはtype = resultで同じステーションを送信する必要があります。 messagiの本文では、ボットはメッセージの本文に必要なさまざまな操作を実行できます。たとえば、マットをアスタリスクに置き換えたり、メッセージ全体をペーストボードのリンクに置き換えたり、ユーザーをエラーに置き換えたりすることもできます。

ボット開発者はdevel@conference.jabber.ru会議に立ち寄って、そこでこの問題について議論できます。



若者との戦いで頑張ってください。





もちろん、あなたが会議の管理者であれば、おそらくこれを以前に知っていて通信をスパイすることができましたが、管理者権限を持たない一般の訪問者がそれを行い、誰かの通信を読むことは狩りとして恐ろしいことですか？

5人までの小さな会議がある場合、このゲームはあなたには向いていないことをすぐに警告します。 このゲームはトップカンファレンス向けです： jc.jabber.ru

ゲーム4：アラビア語を学ぶ



これは非常に古いゲームで、やがてエキサイティングなゲームプレイを失いましたが、関連性は失われていません。 以前は、jabber.ruの会議でアラビア語で何かを書くと、すぐにキックが続きました。 管理者はUnicodeの範囲を十分に知らなかったため、アラビア語とは関係のない一部の文字のキックが掻き立てられる可能性があります。 たとえば、YouTubeサービスのコメントからコピーされたUnicode文字。



以前は、ゲームは非常にエキサイティングでした。アラビア語の名前を付けて、任意の会議に行くことができました。 ボットを使用した会議で最もよく機能し、pingを書くだけで十分であり、会議のほとんどが失敗しました。 とても美しく、壮観でした。



時間が経つにつれて、ゲームプレイが変わりました。 キックは削除され、メッセージテキストは「I'm arabian pig」に置き換えられました。 この認識は、まったく異なる会議の多くの尊敬されている参加者から見つけることができます。 残念なことに、新しいゲームプレイは高く評価されず、政権が裁判所を脅したため、あまり長く続きませんでした。 これはもはや冗談ではないので、すぐにメッセージが「***」に置き換えられ始めたので、自分で試してみることができます。



しかし今でも、アラビア語のニックネームでさまざまな会議に参加し、議論に積極的に参加することはとても楽しいです。 自分で試してみてください！

余暇を過ごすことができる他のゲームがありました。 たとえば、参加者の1人が長いステーションを送信した場合に実際に会議をブロックした曲がったシーパー。 最初にkonfをブロックしてから、アクセスの復帰のために身代金を要求した人もいました。 しかし、たとえ今でも、正当なユーザーから来て生きている人によって書かれたメッセージであっても、制限レートのためにメッセージが送信されないことが非常に多くあります。
