Microsoft + Nokia + BlackBerry vs GoogleおよびApple





MicrosoftのBingが新しいBlackBerryの検索を担当することが発表されました。 マイクロソフトがモバイルデバイス市場でGoogleとAppleに追いつくことは決してないことを認めていることが明らかになりつつあります。



しかし、NokiaやBlackBerryでもMicrosoftは成功できるのでしょうか？



Steve Ballmerは昨日、オーランドのBlackBerry WorldでRIMとのパートナーシップを発表しました。 「私たちは間違いなくBlackBerryプラットフォームに投資します。 Bingは、ブラウザおよびマップのデフォルトの検索プロバイダーになります。「その後、彼は次のように追加しました。」



もちろん、彼は何らかの理由でこれを一般的に言っています。 ただし、これは非常に大きな問題であり、Microsoftにとって不可欠です。



Microsoftのビジネスと利益は、主に過去への貢献ではなく、将来への貢献に基づいています。 同社の最新の財務報告では、Officeのおかげで大きな収益が得られていますが、Office Web Apps部門はなんと7億2,600万ドルを失いました。



未来は、Microsoftが失敗したモバイルデバイスです。 Windows Phone 7はやや弱くなり、開発者の関心と同様、消費者の関心も弱まっています。



MicrosoftのNokiaの契約は、Microsoftの支援要請です。

RIMデバイスのBingおよびBing Mapsは、優れた潜在的利益を意味します。 さらに、Windows Phone 7をNokiaスマートフォンにインストールすることで得られる収入。



しかし、このプールはGoogleのAndroidとAppleのiOSに追いつくのに十分でしょうか？ ほとんどない。 現時点および近い将来、NokiaとRIMのデバイスは、Android、iPhone、iPadのスマートフォンやタブレットに比べて1段低くなります。 BlackBerry PlayBookタブレットはかなりシャープなレビューを受けました。



MicrosoftがGoogleやAppleに追いつくとは信じがたい。 しかし、ノキアとRIMがマイクロソフトを少なくとも3位に引き上げた場合、これはすでに勝利と見なすことができます。
