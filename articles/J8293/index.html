Microsoft Windows Live Folders、新しい噂

Microsoftは、Windows Live Foldersサービス（以前のLive Drive）を開始しようとしています。 サービスが開始されると、 http：//folders.live.comで利用可能になります（サービスは現在利用できません）







サービスを入力すると、プロジェクトのメインページが表示され、作成可能な3種類のフォルダーが表示されます。 また、デフォルトで作成されたいくつかのフォルダーと、残された空き容量も確認できます。







Windows Live Foldersは使いやすいサービスです。 ファイルのダウンロードと同じように、フォルダーの作成は非常に簡単です。 マイクロソフトは、このサービスをかなり幅広いユーザーが使用することを計画しているため、基本的なアクションは、コンピューターから遠く離れた人々にとって使いやすいはずです。





















同時にWindows Liveフォルダーを使用すると、特にフォルダーへの共有アクセスでフォルダーを使用する機会が増えます。 このプロセスは、Windows Liveスペースでのアクセスの共有と非常に似ていますが、唯一の違いは、リーダーまたはアシスタントのいずれかで、友人に役割を割り当てることができることです。 FTPサービスを使用する場合と同様に、リーダーの役割は連絡先にフォルダーとファイルのコンテンツを読み取る権利のみを付与し、アシスタントの権限はファイルとフォルダーのダウンロード、編集、削除の機能も提供します。 パブリックフォルダとファイルの作成も非常に簡単です。















LiveSideサービスによると、このサービスはいくぶん単純であり、実際には、たとえばhttp://box.netなどのプロジェクトよりも機能的に非常に劣っています。 しかし、サービスはまだ開始されていないので、少なくともベータ版のサービスが表示されたときに最終的に何が起こるか待ちましょう。
