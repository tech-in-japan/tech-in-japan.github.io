We list what he can do



If we consider all the functions separately, then each of them, perhaps, is an alternative. The value of such hybrid devices is precisely that all the details come together, from which a new whole grows. In this case, we have:



  • Anti-loss






The umbrella acts as a Bluetooth tag, and in case of disconnection from the paired device, the owner receives notifications. The last pairing point can be seen on the map.
