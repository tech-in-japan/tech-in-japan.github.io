PyTest Neo

ある夜、無限のテストを検討し、ランニングポイントを見て、考えがすり抜けました-マトリックスのように見えます！ それで、マトリックスの形でテストを視覚化するというアイデアが生まれました。







カットアニメーションといくつかの詳細の下。







誰かが読みやすさを心配している場合は、進行状況バーのみがアニメーション化され、エラーがある場合は標準レポートが表示されます。



pytest-neoプラグインをインストールするには、 次のようにします。 



pip install pytest-neo


このプロジェクトは開発の初期段階にあります。 LinuxおよびおそらくMacOSがサポートされています。 考えられるバグ。 これまでのところ、冗長モードはそのまま残されています。 マトリックスの信頼できるアニメーションを実装する計画。



提案や提案を書いてください。 これは、プロジェクトの開発に大いに役立ちます。
