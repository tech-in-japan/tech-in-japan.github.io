世界最大のオープンディレクトリプロジェクトサイトディレクトリ（dmoz）がCreative Commons Attributionに切り替わりました

6月22日の午前4時に、伝説のOpen Directoryプロジェクトまたはdmozに新しいライセンス-Creative Commons Attribution 3.0 Unportedが 追加されたという情報がサイトdmoze.ruに掲載されました。 そして、判明した-CC-BYはdmoz.org/license.htmlにリストされています 。 しかし！ Googleが検索するインターネット上のサイトにはこれ以上ニュースはありません。 おそらく誰もが寝ています。 そして、これは古いニュースではありません。Yandexのキャッシュには古いライセンスがあります。



Open Directory ProjectはAOLが所有しており、ボランティア編集者のコミュニティによってサポートされています。



Open Directory Projectは、 Richard SkrentaとBob Truelによって設立されました。 打ち上げは1998年6月5日に行われました。 1998年11月17日にNetscape Corporationがカタログの購入を発表（購入自体は10月に行われた）後、$ 40,050,000でカタログの名前が変更され、現在の名前が付けられました。



カタログには4 897 671サイト、91 874エディター、1 007 233カテゴリーがあります。 ロシア語のサイトのセクションは、カタログの最大の言語セクションの中で9位になります。



dmozベースの有名なプロジェクトの1つがGoogle Directoryです。



以前は、Open Directory ProjectはOpen Directory License （ text ）の下で配布されていましたが、Free Software Foundationはこれを非フリーライセンスと見なしています。
