Microsoft DevCon 2016でのASP.NET Coreに関する集中的な発表

5月25〜26日に郊外で開催されるMicrosoft DevCon 2016カンファレンスでASP.NET Coreの集中的な開催を発表できることを嬉しく思います。









インテンシブは会議の新しい形式であり、参加者は実際の参加が必須である特定のトピックに1日中深く浸ることができます。



ASP.NET集中型のスピーカーとオーガナイザーを発表できることを非常に嬉しく思います： John Galloway 、Microsoft USAシニアテクノロジーエバンジェリスト、 Sergey Pugachev 、テクニカルソリューションコンサルタント、Google、 Igor Sychev 、Microsoftロシアのプライム開発サポートエンジニア、 Vladimir Yunev 、シニアテクノロジーエバンジェリスト、マイクロソフトロシア。



以下の強烈なコンテンツの詳細をお読みください。



ASP.NET Coreの集中形式には、いくつかのセクションが含まれ、各セクションには、実験室作業またはガイダンスに基づく個別の割り当ての形式で、参加者のデモンストレーションと実際の経験を含む深い理論部分が含まれます。



集中ASP.NETは、次のセクションで構成されています。



ASP.NET Coreプラットフォーム（レベル300）の内部を詳しく見る



最初のセクションでは、ASP.NET Core 1.0の新しいバージョンの内部構造、ルーティング、プレゼンテーションの操作、セキュリティ、フレームワークのモジュール性、クエリ処理のパイプライン、データベースの操作、およびツールの新しいメカニズムについて詳しく説明します。



最初のセクションのスピーカーとリーダーは、 ジョン・ギャロウェイのレドモンドからのゲストです。



ASP.NET CoreとAngularJS 2.0の統合、およびプラットフォームOSX、Linux（レベル300）での動作



集中型の2番目のセクションは、ASP.NET CoreとGoogleのAngularJSの統合に専念します。 Visual Studio Codeを使用してOSX / Linuxオペレーティングシステムでプラットフォームを操作するだけでなく。



強度の第2部は、特別に招待されたスター-Google従業員-Sergey Pugachevが開催します。SergeyPugachevは、アプリケーション開発の深い専門知識を持ち、経験豊富なスピーカーであり、DevConの常連です。



ASP.NETアプリケーションの継続的展開：ローカル（LinuxおよびWindows）、クラウド、コンテナー（レベル300）



インテンシブの3番目のセクションは、Ap.NET CoreアプリケーションをLinuxおよびWindowsのローカル、クラウド、Dockerコンテナに基づいて、Visual Studio Team Servicesを介した継続的な展開プロセスを使用するなど、さまざまな環境に展開するトピックに完全に専念します。



3番目のセクションのメインスピーカーは、世界中のマイクロソフトパートナーと顧客が複雑なアプリケーションを作成および保守するのを支援する経験豊富なマイクロソフトエンジニアであるIgor Sychevです。
