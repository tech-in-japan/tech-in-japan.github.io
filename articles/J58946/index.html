誰でもWindows XP Mode Betaをダウンロードできます

本日、Windows 7 RCのパブリックバージョンがリリースされました 。 ただし、その中の1つの機能であるWindows XP Modeはまだ使用できません。 誰でも実際にこの機能を試すことができます。 Windows 7 RCで動作します！



Windows XP Modeでは、Windows 7を終了せずにXPオペレーティングシステムでアプリケーションを実行できることを思い出してください。これには、Virtual PCのベータ版が使用されます。



Windows Virtual PC BetaおよびWindows XP Mode Betaをダウンロードする



注意！ Windows Virtual PCには、Intel™Virtualization TechnologyまたはAMD-V®をサポートするCPUが必要です。 BIOSで有効にする必要があります。 有効にする方法については、特別なBIOS設定ページをご覧ください。
