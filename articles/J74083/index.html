ウクライナの大手Webスタジオの評価

評価について





ウクライナのウェブスタジオを評価することは長い間考えられてきましたが、そのような評価を求める要望がありました。 そして最近、これを行う絶好の機会が登場しました。



それはすべて、当社の研究、より正確には競争力のあるインテリジェンスから始まりました。 目標は次のとおりです。ウクライナの主要なWebスタジオを特定し、それらに関する最大限の情報を収集し、サイト開発市場での自社の競争力を高めるための多数の管理決定を開発すること。



当然、この市場で働いて、私たちは多くの強力な企業を知っていました、パブリックドメインには多くの情報があり、そのような格付けをより早くする試みさえ見つけました（それらの人は成功しませんでしたが）。 注目に値すると考えられる18社を選択し、インターネット上で（主に企業のWebサイト上で）各企業の最大のオープン情報を収集し、最も一般的なインテリジェンスの方法に頼りました-顧客のふりをして、同僚をノックしましたウェブサイトを作成しようとしていた会社が、その会社のマネージャーと一緒に競合他社をノックし始めました）。



2週間後、各企業に関するすべての情報を入手しました。 ランキングでは、私たちの意見では上位10社を残しました。



以下は、ポイントを含む評価表です。 分析は、クライアントの5つの最も重要な基準（価格、クライアントに対する態度、仕事の経験、保証と責任、創造性）に従って行われ、各基準には1から10のスコアが割り当てられ、最終的に、適切な評価を得るために、重み係数の方法、t .k。 分析された基準は、クライアントに対して異なる値を持ちます。 私たちは独自の調査に基づいて基準の重要度を決定しました（昨年は、特定のスタジオを選択する際に潜在的なクライアントにとって重要なことを見つけるために少し調査を行いましたが、これは別の記事のトピックです）。

18社を分析したが、さまざまな理由で8社が評価に含まれていなかったことを理解しておくことが重要です。トップ10レベルに達していないが、一見すると非常に深刻な組織です。



5つの評価基準について詳しく説明します。 格付けは危機の時期にまとめられたため、消費者の重視は価格にシフトしました。 取得したものは次のとおりです。



25％-価格



25％-顧客の態度



20％-実務経験



20％-保証と責任



10％-クリエイティブ



各パラメーターの評価は、当社のWebサイトの表にある基準の名前をクリックして個別に表示できます。企業はそれに応じてソートされます。



最初の基準は「価格」です。すべての企業にほぼ同じサイト要件を提示しました。 このサイトは単純な企業タイプでした（他のタイプのサイトは分析しませんでしたが、結果はほぼ同じになると思います）。 最低価格の基準は2000ドルで、この会社は10ポイントを獲得しました。 最大。 最高レベルはほぼ10,000ドルに達し、この会社は1ポイントのみを受け取りました。 最小。 そして、彼らは逆比例を使用して他のすべての人のポイントを計算しました。 この列には重大なエラーがある可能性があることに注意することが重要です。これは、まず価格（多くの人が期待してマージンのある価格）を交渉しなかったためです。仕事の始まり（20-30％）。



2番目の基準は「クライアントに対する態度」です。ここでは、コミュニケーションに礼儀正しさ、反応の速さ、目標に対する懸念、専門家のアドバイスなどを含めました。 この基準は、危機の前に最も重要でした。



3番目の基準は「経験」です。ここでは、まず、ポートフォリオ、その品質と量、特定のビジネスセクターでの経験などを評価しました。



4番目の基準は「保証と責任」です。この基準では、公式な関係、企業証明書、コンテストやフェスティバルでの勝利、賞、既存の企業顧客などの可能性を考慮しました。



5番目の基準は「クリエイティブ」です。これらは、ポートフォリオやスタジオ自身のWebサイトなどの新鮮で興味深いアイデアです。



一般的に、私たちは特にArt Depot社が好きだったと言えます。 エイケンとポークはよく現れました。 悪くないWorld Web Studio、Media5、Avanportのデザイン。



興味深い事実





分析の過程で、いくつかの非常に興味深い事実に出会いました。



  1. ある会社は70-80％が別の会社から簡単に「借り入れ」ました。
  2. 簡単な質問として、「コンテンツ管理システムが必要ですか？」、「javascriptが必要ですか？」、「顧客が使用するブラウザは？」、「データベースを使用する予定はありますか？」、「ウェブサイトのデザインを促進する戦略はありますか？」 "、-普通の人はどのようにしてそのような質問への答えを知っていますか？？？ そして、私はまだ最後の質問について考えます...
  3. ある会社は、メールが頻繁に飛び、すべての連絡先が削除されると述べました。
  4. ある会社は手紙の後も電話の後も資料を送らなかった。2回目の電話で彼らは私を「失った」と私に知らせた（私はガディア・ペトロヴィッチについてのKVNからのフレーズを思い出した：「私は失われた」）。 また、多くの人がこの情報またはその情報を送信することを思い出さなければなりませんでした。
  5. 会社の1つが間違っていて、私たちの申し出を送信しませんでした。
  6. 10社のうち9社がロールバックの準備を確認しました（1社は回答しませんでした）。 キックバックについて尋ねられたとき、多くの人がおもしろく答えました：「あなたはいつでも同意できます！」、「私たちはあなたのためにボーナスについて話しますか？」、「それについて話しましょう！」、「どういう意味ですか？」 多くの人が非常に慎重に反応した。
  7. ウクライナのウェブサイト開発市場全体の一部の企業の収益性レベルは1000％です（サービスコスト、固定費および変動費の詳細な分析に基づいて判明）。
  8. 分析から、外見的にはかなり深刻な（高品質、クールな顧客、優れた自社サイト）ある会社は、実際には完全に非専門的であることが判明しました（または、マネージャーは「学校のベンチから」のみですか？）
  9. いくつかの「ささやかな」企業自身が、いくつかの方向で市場の「リーダー」と呼んでいます（何人のリーダーがいますか？）。
  10. ある会社の女の子マネージャーがプレゼンテーションへのリンクを送信しました。これはftp上にあり、ユーザー名とパスワードを入力しました！ その結果、すべてのクライアント、ソースコードなどにアクセスできるようになりましたが、もちろん何もダウンロードせず、企業はセキュリティホールについて報告しました。




レベルアップ





当然のことながら、この記事を読んだ後、多くの人が興味を持ちます。「ビジネスプロセスを市場で最高の企業ほど悪くしない方法は？」 実際、最初の目標はこの質問に正確に答えることでした。 私たちは同僚と情報を共有するつもりはありませんでしたが、最後の瞬間に同じ市場で仕事をしていると思いました。 したがって、このセクションでは、一般的な推奨事項の形で、改善の問題に専念します。



1.顧客関係：

  • 顧客への最初の手紙には、温かい歓迎、会社のプレゼンテーション、顧客の質問への回答（ある場合）、ポートフォリオへのリンク、顧客への保証の説明、セールスマネージャーの連絡先を含める必要があります。
  • 手紙を送った後、1-2日後に、あなたは間違いなく電話をかけて、クライアントがあなたの手紙を受け取ったかどうかを確認しなければなりません（第一に、メールが正しく機能しないかもしれません、第二に、それは自分自身に思い出させる機会です）。
  • コミュニケーションは丁寧で友好的で、小さな笑顔で話そうとする必要があります。クライアントはそれを感じます。
  • クライアントの目標とニーズを理解し、この問題で彼を助けてください。 専門家として、おそらく提案をします。
  • クライアントが興味を持っている場合、予約をしてみてください、個人的なコミュニケーションは販売の50％です。
  • クライアントに送信するすべての文書は、連絡先（プレゼンテーション、簡潔、商業的提案など）を含むレターヘッドに記載する必要があります。
  • クライアントはあなたを思い出し、肯定的な意見を持ち、すべてに満足しなければなりません。
  • 効果的な顧客関係管理のために、CRMを使用する必要があります。




2.会社のプレゼンテーション：

  • プレゼンテーションはあなたの前向きな側面を強調し、同時に情報でいっぱいになってはいけません。
  • プレゼンテーションには、あなたに関する情報、あなたと一緒に仕事をすることの利点、連絡先が必ず含まれている必要があります。
  • 誰もがPowerPointを持っているわけではないことを忘れないでください。したがって、いくつかの形式でプレゼンテーションを作成することをお勧めします。pdf、flash（これは必須ではありません）。
  • たとえば、プレゼンテーション： link 。




3.簡単な説明：

  • ブリーフは簡潔である必要があり、不必要な質問でそれをオーバーロードしないでください。
  • ブリーフには、クライアントの目標に関する質問、ターゲットユーザーに関する質問、クライアントが気に入っているサイトの例（機能に関する標準的なものに加えて）を含める必要があります。
  • たとえば、簡単な説明： link 。




4.商用提供：

  • CPを大きくしないことが重要です（理想的には2ページまで）。すべての追加情報はレターで記述できます。
  • この「個人的なアプローチ」の重要性を忘れないでください、クライアントはすぐにそれを見て感謝します。
  • クライアントが決定を下した同僚と印刷して交渉できるように、別のファイルに保存する必要があります。
  • コストは段階に分解する必要があるため、クライアントの目にはより合理的です。
  • 作業の段階、受け入れ-配達段階の説明を忘れないでください。
  • 支払い、前払いに関する情報を掲載することをお勧めします。
  • 提案は、責任あるマネージャーの連絡先で終了する必要があります。
  • 2〜3日後、折り返し電話して解決策を確認します。 これを行わないと、クライアントを失う可能性が高くなります。




残りのビジネスプロセスについては、ここで明らかです。良いサイトを作成し、コストを削減し、フェスティバルで勝利すれば、すべてが実際の顧客になります！

よくある間違い



よくある間違い





私たちは、他人の過ちから学び、常に会社を改善する方が良いと信じています。 したがって、競合他社の最も一般的な間違いが強調されました。



  1. 文書はレターヘッドではありません。 分析では、会社名や連絡先のない「裸の」ブリーフや商業的なオファーをよく見ました。 しかし、普通のクライアントは常にいくつかの会社をノックします。特定のドキュメントをどのように識別する必要がありますか？
  2. 顧客に対する冷静な態度。 多くの人は彼らの態度に驚いています：私たちの要求に答えがなかったり、説明なしで裸の手紙を送ったり、私たちの目標を掘り下げたりする人はほとんどいませんでした。 クライアントはだれを選択しますか：彼とうまくコミュニケーションを取り、彼の目標を掘り下げ、会議を開いた会社、それともむやみの誤算をして消えた会社ですか？
  3. 不要な情報で過負荷。 多くの場合、企業は一般の人が知らない技術的な詳細を伝え、多くの場合、不要な質問の束とともに10ページのブリーフを送信しました。 潜在的なクライアントは、座って10ページの簡単な説明を記入する以外に何もすることはありませんか？ また、クライアントが10社に転向し、10件のブリーフを受け取った場合はどうなりますか？ クライアントに「PHPですばらしいウェブサイトを作って、MySQLデータベース、専用サーバーを作成し、CMSを用意します！」または「2か月で毎日多くの顧客をもたらすウェブサイトを作ろう」と伝えると、クライアントにとってより便利です。 ？
  4. 販売の基本の無知。 多くの場合、営業担当者はクライアントと通信しますが、技術者は通信します。 その結果、クライアントは決定を下すために必要な情報を受け取らないか、単にそれを忘れます。
  5. 実を言う。 非常に一般的なテクニック：最大値を約束し、最小値に価格を設定し、作業の過程で言う：より良くしたい場合、それはより多くの費用がかかります。 これは遅かれ早かれ100％失われた顧客です。 そして、誰もが長い間知っていました-お金は常連客だけにあります。




おわりに





この調査を2.5か月間実施し、その後、ビジネスプロセスを改善するための複雑なプログラムを開発しました。 結果はすぐに現れました-改善の過程で、顧客の数は2倍に増加し、平均契約額は60％増加しました。ロシアからの新規顧客の多くは、現在、年末までに見られる多くの深刻なプロジェクトに取り組んでいます。 秘密は簡単です。お客様は、更新されたオファーに喜んで同意します。 あなたが私たちの「同僚」である場合-私たちの推奨事項を取り、改善します（恥ずかしがらないでください-連絡：私たちはアドバイスをお手伝いします）！ クライアントがあなたからの連絡をいつでも喜んでいる場合：電話して、書いてください！ :)）
