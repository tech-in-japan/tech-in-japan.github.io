ReSharper 4.5リリース

.NET開発者向けの優れたツールが更新されました。



*開発者は生産性に取り組みました。たとえば、見積もりによると、大規模ソリューションの読み込み速度は20％から40％に増加しました。

* Visual Basic .NET 9言語のサポートが導入されました。

*新しいコードインスペクターを追加。

*名前、タイプ、名前空間、インターフェースなどのルールを設定する機能を追加しました。これにより、リゾルバーは要件に応じてコードを設計するのに役立ちます。

* Go to Implementation機能を追加（ベータ版の感想によると、非常に便利）。

*リファクタリングメカニズムが改善され、名前変更により名前補完が可能になりました。



開発者は、革新について語るフラッシュムービーを記録しました。 こちらで見れます 。
