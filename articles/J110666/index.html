Pixel Qiはディスプレイのメーカーを見つけました

電子書籍の画面の運命をたどる人々とそのメーカーにとって朗報です。



このようなハイブリッドディスプレイ3Qi会社Pixel Qiがあります。 ほぼEインク（リアミラーによりバックライトをオフにして読むことができます）およびほぼLCD（バックライトをオンにすると画面が通常のラップトップに変わります）です。

次の写真は、3Qiディスプレイがインストールされたネットブックで、バックライトがオフになっています。







これまで、Pixel Qi画面は単なる概念実証にすぎませんでした。 現在、fcenter.ruが報告したように、Pixel Qiはディスプレイメーカーを見つけました。





Pixel Qiディスプレイは、市場に参入するのに苦労しています。 しかし、彼らはそれを突破し、それは良いことです。 この会社は、非営利組織OLPCメアリールージェプセンの「無料水泳」の主任技術者が去った後、2008年1月に設立されました。 3Qiディスプレイは、「学生用」ラップトップのディスプレイの第1世代に由来します。その主な機能は、直射日光下でモノクロモードで動作する機能でした。 これにより、ディスプレイのバックライトがバッテリーの「食物連鎖」から除外されたため、明るい外部照明で快適に動作するだけでなく、バ​​ッテリー電力を節約することも可能になりました。



Pixel Qiは、今年初の成功を祝うことができました。 台湾のメーカーの1社が、同社の10.1インチスクリーンを少量生産し始めました。 画面は、多くのラップトップモデルの通常のパネルの自己交換を目的としていますが、価格は非常に高く、$ 275です。 さらに、このようなパネルの最初のタブレットであるインドの「アダム」会社Notion Inkはすぐに売り切れたため、画面のバッチは明らかに小さいです。 現在、注文できるのは従来のLCDを備えたモデルのみです。 ちなみに、3Qiスクリーンを備えたモデルは、通常のスクリーンを備えたモデルよりも約200ドル高く、他のすべての条件は同じです。 イノベーションは安くはありません。



幸いなことに、Pixel Qiは、ディスプレイが有望であると世界最大の小型フォーマットパネルメーカーの1つであるCPT（Chunghwa Picture Tubes）を納得させることができました。 同社のトップによると、彼のオンライン日記では、夏に始まった交渉は秋にパートナーシップ契約に変わった。 CES 2011では、パートナーは解像度1024x600ピクセルの7インチ3Qiパネルを表示し、2011年にはさらに2つの新しい画面サイズが導入されます。 まず、スマートフォン用の小さなフォーマット。



7インチスクリーンの量産は、2011年の第2四半期に開始される予定です。 Pixel Qiの費用対効果の高い半透過型ディスプレイは、タブレットや電子書籍にインストールされます。 バックライトなしの反射モノクロモードに加えて、3Qiディスプレイはバックライト付きの通常のカラーモードで動作できます。 ハイブリッドモードの可能性により、日中は同じデバイスでプールサイドで本を読み、就寝前に映画を見ることができます。
