The Next Web Startup Rallyのフィナーレでロシアのスタートアップ

いいニュースだね。 ロシアのEcwidスタートアップ（私が仕事をすることを誇りに思っている） は 、The Next Web Startup Rally 2010の決勝に進出しました。（ TechCrunchでの同じことについて ）



245人の参加者の非常に厳しいマルチステージ選択を経て、4月末にアムステルダムのNext Web会議でプロジェクトを発表します。



これは非常にクールであり、それが理由です。



残念ながら、多くの人々は、ロシアに登場する新しいサービスはすべて、成功した西洋製品のクローンと模造品であるという印象を持っています。 たとえば、TechCrunchの投稿とコメントをご覧ください： techcrunch.com/2010/04/04/cut-paste-innovation-groupon-gets-cloned-in-russia



普通のアメリカ人はどう思いますか？ 「ああ、ロシアからのサービスはクローンのようなもので、革新的でユニークなものはありません。そうでなければ彼らはできません。」 以下に例を示します。

ロシアのことを考えてみてください-彼らはイノベーションの点で本当に何をしましたか？ あまりない。



コピーは、自分が何をしているかわからない人々の最後の手段であり、率直に言って、ロシアと中国はこれらの人々でいっぱいの国家です。



これは悲しいです。 特にそうではないことがわかっているからです。



そのため、ロシアで新しいプロジェクトが作成されており、革新的で世界に類を見ないだけでなく、ニッチの世界的リーダーでもあり得ることを世界に示し、証明できることを非常に嬉しく思っています。 私たち自身がクールなものを作ったということは、世界の他の同様のものよりも優れた頭です。 「バカなコピー・パスツール」に関するこの意見を破るために、私たちは宇宙に飛び込んだ最初の人でした！
