Inbit Messenger-独自のクライアントサーバーIMネットワーク

すべての良い一日。 私は、Habrasocietyに素晴らしい、ほぼ唯一のプログラムについて伝えたいです。

Inbit Messengerは、インスタントメッセージングネットワークのWin（* nix / Wine）プラットフォームで整理するためのクライアントサーバーシステムです。







このプログラムは、ICQ / Jabber、IRC、Emailの機能を組み合わせて、共通のTo Doリストを維持できるようにし、ファイルホスティングの類似物、URLリンクのデータベース、およびコラボレーションを整理するためのその他の設備を備えています。

おそらく最も重要なことは、AOL / Oscarのような外部サービスに依存せず、ローカルネットワークに配置したり、インターネット経由でアクセスできる独自のサーバーを作成できることです。

Inbit Messengerは内部通信に最適で、展開速度、機能性、管理の容易さのバランスが取れています。



残念ながら、このプログラムはほとんど知られていないので、修正したいと思っています。

Inbit Messengerプログラムは、ICQ、IRC、および電子メールに同時に類似したIM通信システムであり、スクリーンショット用のツールであるファイルサーバーの機能を備えています。

小規模および大規模ネットワークの両方に適しています。 実際には、2〜5人のユーザーに静かにサービスを提供するのに十分な軽量ですが、十分な数のサーバーリソースで数百に対応し、開発者は同時に数千のユーザーに対応します。 システム要件は、ネットワーク視聴者の規模にのみ依存します。

このプログラムは、コンポーネント「サーバー」、「クライアント」、「管理コンソール」、および履歴ファイルと内部メール形式を表示するための補助的なミニアプリケーションの形式で構成されています。



配布キットは小さく、インストールは「箱から出してすぐに」結果の効果をもたらし、約10〜15分かかります。

ほとんどすべてがサーバーまたは「管理パネル」から構成および制御されます。ユーザーは、必要に応じてグループに分割され、特定の特権で個人的に、または所有権の原則に従って付与されます。

さらに、ユーザーの観点からは、すべてが「ICQ」のように見え、管理者は、グラフィカルコントロールパネルを備えたWindowsメールサーバーのスタイルで、多くのユーザーを便利に管理できます。







ユーザーには、グループと部門に分かれた連絡先リスト（サーバーから他の連絡先の表示を制御する機能）、p2pテクノロジを使用してファイルをすばやく転送できるマルチタブの会話ウィンドウ、および会話ウィンドウに画像がすぐに表示されます。 絵文字、さまざまなフォントと強調表示、ステータスメッセージ、便利な履歴管理、連絡先情報付きのアバターとプロファイル、個人の連絡先リストのサポート、無視リスト、ポップアップアラート-これらはすべて、従来のインターネット上でも誰もが楽しんでいた通常の機能と非常に似ていますページャー。









ただし、Inbit Messengerには、電子メールの書き込み、ポップアップ広告の送信、To Doリストの管理、公開メモ、リンクの公開、サーバーに保存された共有ファイルの使用も可能です。 組み込みのスクリーンキャプチャツールを使用して、サイトの画像またはプログラムのウィンドウをすばやくデモンストレーションすることもできます。これは、いくつかのモードで動作し、PrintScreenとペイントを使用する必要がありません（Windowsがすぐに使用できる場合）。













また、サーバーにより、管理者はこれらすべてを順序どおりに保ち、混乱や寛容さを回避できます。 サーバーのポリシーに従って、ほぼすべてのアクションをログに書き込むことができ、システムのインストール先が適切でない場合はすぐに停止できます。

たとえば、サーバーが膨大な量の保存ファイルで過負荷を許可しない経験の浅いユーザーのグループ、機密問題を議論するための独自のプライベートチャンネルを持つグループ、チャンネルをクリーンアップし、わいせつなファイルを消去し、履歴にアクセスできるモデレーターのグループを作成できますダウンロード/ダウンロード。











Inbit Messengerは、情報へのアクセスのために厳密に順序付けられた企業環境で動作でき、同じネットワークのコミュニティのICQにとって非常に友好的なチャットになります。

これらはすべてインターネットとは独立して機能しますが、外部ネットワークからのアクセスを整理することができます-Inbit Messengerを同じJabberに例えることができ、組織の複数のオフィスを統合することで、システム管理者は、インサイダーが作業できるICQへのさまざまなユーザーアクセスを許可する必要がなくなります。

Rijndael暗号化をサポートする純粋に内部的な「ICQ」および「メール」をユーザーに提供することにより、システム管理者は商業的に価値のある情報の漏洩のリスクから組織を真剣に保護します。



Inbit MessengerはActive Directoryとの相互作用をサポートしており、ユーザーが不必要なパスワードを覚えるのを避けることができ、管理者に集中管理のための追加の機会を与えます。

内部メールは、任意の電子メールサーバーに転送できます。

サーバーは、機能の使用に関する統計を保持します。これは、管理レポートで役立ちます。

Windows Mobile用の単純化されたクライアントによってシステムに追加の柔軟性が与えられます 。これは、WiFiとGPRSの両方で完全に動作します。



サーバーでは、接続に任意のポートを使用できます。プロトコルの実装は非常に洗練されており、そのパケット形式はXMPPに似ています。構成ファイルはXML形式に基づいており、暗号化はオプションでサーバーデータベースで透過的に使用できます。
