ガイア・イヴァルソイを追In。

昨日、誤ってOperaブラウザについて注意を喚起しました。 そこには、ページの一番下に、私に興味を持った碑文があります。

GeirIvarsøyを記念して。






このIvarsoy、Geirは誰ですか？ 最初にウィキペディア、次にグーグルに助けを求めたとき、私は最初の質問者ではないことに気付きました。 しかし、ウィキに書かれていることを引用させてください-私を信じてください、この人はHabrで言及するに値します（私は彼の言及をここで見つけませんでした）。



GeirIvarsøy（ノルウェー。GeirIvarsøy; 1957年6月27日-2006年3月9日） -ノルウェーのプログラマーは、Opera Software ASAの主要な開発者であり、ブラウザOperaで知られていました。 彼とJon Stephenson von Tetzchnerは、電話会社Telenorの研究部門で協力し、そこでMultiTorg Operaと呼ばれるソフトウェアを開発しました。 Telenorはプロジェクトを終了しましたが、1995年にGeirとJonは子孫の権利を購入し、会社を設立し、ブラウザーでの作業を続けました。



現在、このプロジェクトはOperaとして知られていますが、このブラウザは競合にもかかわらず非常に人気があります。 Opera Software ASAのスタッフは、オスロに移ってから400人以上の従業員を補充しています。



2004年1月の取締役会で、Geir IvarsoyはOpera Software ASAの取締役からの辞任を発表しましたが、その後、会社の開発に積極的に参加しました。 2005年7月、彼は会社の指名委員会に選出されました。



ガイアは、癌との長い戦いの後、2006年3月9日に亡くなりました。
