Uppod Web Player

Uppodのプレイヤーの作成サービスについては、Habréですでにある種の風変わりな人々が書いています。 先日、彼は1歳になり、新しい機会について話すことにしました。





このサービスを使用すると、サイトのプレーヤー（ビデオ、オーディオ、または写真）を任意のユーザーにアセンブルできます。 真に高度なアプリケーションを構築するために、FlashやJavascriptは必要ありません。 すべてが整理されており、モデリング用の多くの設定を備えたビジュアルエディターがあります（ promo-uppod.info ）。



長所は何ですか：

  • 無料版にはロゴがなく、商業上の制限はありません。
  • プレーヤーはサイトに関連付けられていません-すべてのファイルはユーザーによって保存されます。
  • インターフェースを介して作業でき、 プラグインを使用できます 。
  • プレイリストおよびJavaScriptバンドル用のオープンAPI
  • あらゆる種類のストリーム。
  • 5つの組み込み言語バージョン。
  • RSSの接続、Googleアナリティクスの統計の収集、スクリーンショットの取得、コンテンツの保護などのための追加モジュール。
  • 大規模プロジェクト向けの高度なAPI。
  • 開発者は常にサービスの開発に携わっています。
  • ライブフォーラム。
