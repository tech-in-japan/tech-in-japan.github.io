ASP.NET Core 1.0と.NET Core 1.0の再紹介



  • ASP.NET 5は現在、ASP.NET Core 1.0になりつつあります。
  • .NET Core 5は現在.NET Core 1.0になりつつあります。
  • Entity Framework 7は、それぞれEntity Framework Core 1.0またはEF Core 1.0になりました。


なぜ1.0なのか？ これらのツールは新しいためです。 .NET Coreの概念はすべて新しいものです。 .NET Core 1.0 CLIツールは完全に新しいものです。 これが唯一の理由ではなく、.NET Coreには.NET Framework 4.6のすべての機能が含まれているわけではありません。 私たちはまだサーバーサイドのグラフィックライブラリに取り組んでいます。 ASP.NET 4.6からASP.NET Core 1.0のバックログを削除するために、現在作業中です。
