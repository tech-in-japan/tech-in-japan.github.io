Intelは最初の17キュービット量子チップを発表しました

量子研究に5000万ドルを投資してから2年後、Intelは最初の17キュービットチップを発表しました。 同社は、QuTech研究所からパートナーにデバイスを転送することに留意しました。 次にチップが何であるかについてお話します。





/ Flickr / ウォルター / cc



量子コンピューターのアーキテクチャは、最新のPCのアーキテクチャとは根本的に異なります。 古典的なコンピューティングシステムはビットを使用して情報をエンコードしますが、量子ビットはゼロ、1、またはそれらの間の任意の値の量子ビットで動作します。 ビットが水平線であると想像すると、キュービットはその値が存在する球体になります。 量子コンピューティングの詳細については、 こちらとこちらをご覧ください 。



この特性により、量子コンピューターは電子計算システムよりも、特定の範囲のタスク（ゲノムの操作、複雑な分子のモデリング、化学的性質の予測など）に適しています。



Intelの新しい17キュービットCPUの機能の 1つがそのケースでした。 量子ビットの最適な動作温度は20ミリケルビンです。 同社は、宇宙よりも250倍寒いことに気付いています。 新しいチップは、損傷することなくこの温度に耐えられるように設計されています。



また、この新しいアーキテクチャは、量子ビットによって互いに重畳される無線周波数干渉に対する保護と、従来の有線接点と比較してチップスループットを10倍に高める特別な相互接続スキームも備えています。 Intelによると、新しい開発により、量子集積回路の作成が可能になります。 QuTech YouTubeチャンネルで、このチップのボクシングに関するビデオを見つけることができます 。



生態系開発



IntelとQuTechの仕事は、量子デバイスの単純な開発を超えています。 企業は、量子ビットデバイスからそれを制御するハードウェアおよびソフトウェアアーキテクチャまで、量子システムのスタック全体を設計しています。



さらに、Intelはいくつかのタイプのキュービットを調査しています。新しいチップに含まれる超伝導キュービットと、通常の技術プロセスを使用してシリコンで理論的に再現されたスピンキュービットです。 同社はムーアの法則を拡張したいと言っています。



市場



量子テクノロジーの開発に積極的に関与しているのはインテルだけではありません。 たとえば、今年の初めにIBM は 16量子ビットの量子チップを発表しました。 また、Google は研究者がクラウド内の量子システムにアクセスできるようにする計画です。 Rigetti Computing Inc. 年の初めに、 彼女はいくつかの問題を解決する上で最も強力なスーパーコンピューターを追い越す量子機械を開発するために6400万ドルを集めました 。



カナダの会社であるD-Wave Systemsは、完全に機能する量子コンピューターを構築したことを既に発表していますが、それらのシステムはわずかに異なる方法- 量子アニーリングを使用しています。 マイクロソフトを含む他の企業も、自社の研究の進捗状況を報告しています。



デルフト工科大学の教師であるIntelとQuTechについては、17量子ビットを最大限に活用するために必要なソフトウェアの種類を見つけることを計画しています。 また、チップを適用するための新しい領域を探し始めます。
