Microsoft Robotics Developer Studioが無料になりました

Microsoft Robotics Groupの部門は昨日、プログラミングロボットとMicrosoft Robotics Developer Studio 2008 R3 （RDS）の3Dモデリング用の大規模なパッケージが誰でも無料で自由に利用できるようになったという予想外の発表で、独立した開発者を困惑させました。







これは、ロボットオペレーティングシステムやウィローガレージの開発など、オープンソースの世界からいくつかのオープンプラットフォームを配布することを困難にする可能性がある非常に予期せぬ手順であり、 IEEE Spectrumエンジニアリングジャーナルを執筆しています。



上下のスクリーンショットは、RDSで設計できる3Dモデルを示しています。







以前は、機能が制限されたRDSバージョンのみを無料でダウンロードできました。 現在、マイクロソフトの戦略は完全に変化しています。 彼らはもはやツールの販売で稼ぐことはありませんが、ロボット工学の開発者を引き寄せます。 それらを自由に利用できるようにするために、一部のプログラムとモジュールのソースコードも、Microsoft Robotics Groupの周りの開発者コミュニティの共同開発と作成を刺激するようにレイアウトされています。



Microsoftによると、2007年以降、RDSは約50万回ダウンロードされ、現在では約60,000人のアクティブユーザーがいます。
