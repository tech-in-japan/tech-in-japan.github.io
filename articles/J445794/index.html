ITの巨人がハイブリッドクラウド展開ソリューションを発表

デルとVMwareは、VMware Cloud FoundationとVxRailプラットフォームを統合します。




/写真Navneet Srivastav PD



なぜそれが必要ですか



State of Cloudの調査によると、すでに58％の企業がハイブリッドクラウドを使用しています。 昨年、この数字は51％でした。 平均して、1つの組織が約5つの異なるサービスを「クラウドに保持」しています。 同時に、ハイブリッドクラウドの導入は、企業の45％にとって優先事項です。 すでにハイブリッドインフラストラクチャを使用している組織の中で、SEGA、Oxford University、分析組織ING Financialを区別できます。



クラウド環境の数が増えると、インフラストラクチャが複雑になります。 したがって、ITコミュニティの主なタスクは、マルチクラウドでの作業を簡素化するサービスの作成です。 この方向に発展している企業の1つはVMwareでした。



昨年末、ITの巨人は Kubernetes展開ツールを促進するHeptioスタートアップを買収しました 。 先週、VMwareがDellとの共同ソリューションを開始することが明らかになりました。 これは、Dell EMC VxRailハイパーコンバージドコンプレックスとVMware Cloud Foundation（VCF）プラットフォームに基づいたハイブリッドクラウド環境を作成するためのシステムです。



新製品について知られていること



VMwareは、VMware Cloud Foundation Cloud Stackをバージョン3.7にアップグレードしました。 今年4月から、このソリューションはDell VxRailハイパーコンバージドシステムにプリインストールされます。 新しいプラットフォームであるVMware Cloud Foundation on VxRailは、Dellネットワークデバイス（スイッチやルーターなど）をVCFソフトウェアコンポーネントとリンクするAPIを提供します。



VCFアーキテクチャには、vSphereサーバー仮想化ソフトウェアとvSANストレージシステムが含まれます。 さらに、データセンター仮想ネットワークを最適化および管理するために設計されたNSX Data Centerテクノロジーが含まれています。 NSXのハイパーコンバージドインフラストラクチャへの移行機能は、イギリスのベイステートヘルス病院でテストされました。 病院のITスペシャリストによると、このシステムにより、すべてのソフトウェアとハ​​ードウェア、およびドライバーの高度な統合が可能になりました。



VMware Cloud Foundationのもう1つのコンポーネントは、vRealize Suiteハイブリッドクラウド管理プラットフォームです。 これには 、仮想インフラストラクチャの動作の分析、クラウドリソースのコストの推定、監視およびトラブルシューティングのためのツールが含まれています。



VxRailに関しては、Dell PowerEdgeシリーズサーバーで構成されています。 1つのデバイスで最大200台の仮想マシンをサポートできます。 必要に応じて、サーバーを1つのクラスターに結合し、同時に3千台のVMを操作できます。



将来、彼らはソリューションを単一システムとして開発することを計画しています。そのために、DellとVMwareはVxRailとVMware Cloud Foundationの製品アップデートを同期します。



コミュニティが考えること



VMwareの代表者によると 、更新された統合プラットフォームは、ハイブリッドITインフラストラクチャのパフォーマンスを大幅に向上させます。これは、旧バージョンのVxRailと比較して60％の増加です。 また、VxRail上のVMware Cloud Foundationは、クラウドインフラストラクチャを作成する企業のコストを削減します。 5年間にわたる運用コストは、パブリッククラウドの運用コストよりも45％低くなります。



DellとVMwareの主な利点の 1つは、物理ネットワークデバイスの構成と管理の自動化です。 しかし、彼らはITの巨人が直面する可能性のあるアナリストと潜在的な問題を見ています。 おそらく、主なものは高い競争です。 近年、企業は多くの新しい市場（HCI、SDN、SD-WANを含む）に参入しており、主要なプレーヤーがすでに働いています。 さらなる成長のために、ITの巨人には、ソリューションを競合他社と区別する新しい機能が必要です。



そのような分野の1つは、DellとVMwareがすでに製品に統合しているデータセンターを管理するための機械学習技術です。





/写真グローバルアクセスポイント PD



同様のシステム



ハイブリッドクラウドハイパーコンバージドシステムも、NetAppとNutanixによって開発されています。 最初の企業は、オンプレミスインフラストラクチャをパブリッククラウドサービスに接続する統合データファブリックプラットフォームを備えたプライベートクラウドシステムを提供しています。 この製品は、vRealizeなどのVMwareテクノロジーにも基づいています。



このソリューションの特徴的な機能は、コンピューティングとストレージ用の個別のサーバーノードです。 会社の代表者によると、このようなインフラストラクチャの構築は、データセンターがリソースをより効率的に割り当て、不必要な機器を過剰に支払わないようにするのに役立ちます。



Nutanixは、ハイブリッドクラウド管理プラットフォームも作成しています。 たとえば、組織のポートフォリオには、IoTシステムを構成および監視するためのシステムと、Kubernetesコンテナーを操作するためのツールが含まれています。



一般に、ハイパーコンバージドインフラストラクチャのプロバイダーは、マルチクラウドシステム市場に参入しています。 この傾向は近い将来に続くと予想されます。 特に、DellとVMwareの共同ソリューションは、間もなくより大規模なProject Dimensionプロジェクトの一部となり、クラウドシステムを周辺機器およびオンプレミス機器と統合します。
