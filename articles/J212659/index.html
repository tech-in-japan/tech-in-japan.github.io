MacでNokia Lumiaを友達にするには？





Windows Phoneに切り替えることで、OS Xとの同期を忘れる必要はまったくありません。カットの下で、「 Nokia Lumia + MacOS X =友情と理解 」を説明しようとします。





LumiaをMacと同期する



実際、iTunesやiPhotoからMailまで、すべてのNokia Lumiaスマートフォンと同期できます。 間違いなく最初に必要なことは、Mac用の小さなWindows Phoneアプリをインストールすることです。 これは、同期を設定し、Lumiaに保存されているファイルを表示できる便利なツールです。







スクリーンショットを見るとわかるように、アプリケーション内のナビゲーションは非常にシンプルで、MacOSユーザーには馴染みがあります。 iTunesがApple独自のガジェットとやり取りするように、Windows Phoneアプリは、Macに保存されている音楽（プレイリストと一緒に）と写真をLumiaと同期できます。 ただし、DRMで保護されたコンテンツはスマートフォンと同期されないことに注意してください。



Nokia Photo Transfer



写真をLumia（Nokia Lumia 520以降、Nokia Lumia 1520以降）と同期するために、ビデオおよび写真コンテンツをインポートおよびエクスポートするための追加機能を備えたより強力なアプリケーションがあります。 これはNokia Photo Transferと呼ばれ、 ここからダウンロードできます。







シンプルでミニマルなNokia Photo Transferアプリケーションを使用すると、写真をどこにどこに移動するかを自分で選択できます（インポートするかエクスポートするかに関係なく）。 しかし、最も重要なことは、この「アプリ」を使用して、「ライブ写真」やNokia Cameraで撮影した他の写真をMacに転送することもできます。 Nokia Lumia 1020およびLumia 1520の所有者にとっての追加のボーナスは、デバイスから高解像度の写真とデジタルネガ（DNG）をコピーできることです。



iCloudアカウントをNokia Lumiaに接続します



あなたがアップルのコンピューターユーザーなら、おそらくあなたのプライマリーメールの受信箱はiCloudにあります。



GmailやHotmailとは異なり、iCloudは、Nokia Lumia用のプリインストールメールプロバイダーのリストにはまだ含まれていません。 ただし、数秒でアカウントに接続できるように、必要なすべてを行っています。 スマートフォンの設定で[メール+アカウント]メニューを開き、[サービスの追加]セクションをクリックして、[詳細]セクションまでスクロールします。







これで、アドレスとパスワードを入力するだけで、その後アカウントがLumiaに自動的に追加され、開始画面にiCloudメールボックスに入るための追加のタイルが表示されます。 すべてがシンプルです！







今後数か月以内に、iCloudの完全なサポートがWindows Phoneエコシステムに登場する可能性が高くなります。 ただし、クラウドからカレンダーと連絡先を同期するのが待ちきれない場合は、XDA Developersカンファレンスで Lumiaユーザーが推奨する方法を使用できます。 もう1つの方法は、連絡先をvCard経由でサードパーティサービス（GmailまたはMicrosoft Exchange）にエクスポートし、MacとLumiaで並列同期を設定することです。



iCloudドキュメントと「フォトストリーム」をSkyDriveクラウドに置き換えます



すべてのNokia Lumiaは、デフォルトでOfficeおよびSkyDriveをサポートしています。 Lumiaの設定で、Wi-Fiまたはモバイルインターネット接続を使用している場合、キャプチャされたフレームがSkyDriveアカウントに自動的に複製されるように、同期パラメーターを設定できます。 クラウドストレージにドキュメントを簡単に保存することもできます。 したがって、「クラウド」アクセスの焦点をiCloudからSkydriveに単純にシフトできます。 幸いなことに、このためのMac用の特別なアプリケーションがあります（https://itunes.apple.com/en/app/skydrive/id522548761）。 また、素晴らしいIFTTTを使用して、他のサービスからのコンテンツを自動的に同期するための設定を簡単に構成できます。



Macをリモートで管理する



App Storeには多数のMacリモートコントロールアプリがあります。 これには、たとえばTeamViewerが含まれます。これにより、Macだけでなく、WindowsまたはLinux上の他のコンピューターも管理できます。







VLC Mobile Remote ProがMacメディアコントロールコンソールになります。 ソファから立ち上がることなく、映画の視聴を停止したり、トラックを切り替えたりできます。







VLC Mobile Remote Proに加えて、Remoteという簡単な名前のアプリケーションをインストールできます。これにより、遠くからiTunesを制御できます。 さらに、このアプリケーションでは、トラックを切り替えることができるだけでなく、ライブラリへのフルアクセスも可能です。







リモートマウスアプリケーションを使用すると、Lumiaを完全なワイヤレストラックパッド/キーボードに変えて、Appleコンピューターを短距離で制御できます。 たとえば、プレゼンテーション中にスライドを切り替えたり、Macが大型ディスプレイやテレビに接続されている場合にSafariを制御したりする場合に便利なアプリケーションが役立ちます。
