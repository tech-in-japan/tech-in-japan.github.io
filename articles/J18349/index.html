Ubuntu 7.10でNVIDIA vidyahiのドライバーをインストールする

ここでは、NVIDIAカードにドライバーをすばやく簡単にインストールする方法を説明します。 基本的に、彼らは全員にペンでインストールするようアドバイスしています-Envyというクールなツールを使用することをお勧めします。 これはfireの正しい設置（少なくとも私はまだグリッチを見たことがない）のための最良のオプションです。



  • Envyプログラムをダウンロードします： http : //albertomilone.com/nvidia_scripts1.html
  • 必須!!! xの構成のバックアップを作成します： cp /etc/X11/xorg.conf /etc/X11/xorg.conf.mybackup



  • Envyを起動すると、既存のドライバーが削除されます（「NVIDIAドライバーの削除」）。
  • 彼が再起動を要求した場合、それを行います。
  • Envyに戻ります。 ここでドライバーをインストールします（インストールプログラムが必要なパッケージをダウンロードするため、インターネット接続が必要です）。
  • 身を乗り出し、インストールが完了するのを待ちます。
  • 再起動します。
  • 再起動し、GRUBメニューで復旧モードを選択します。
  • cp /etc/X11/xorg.conf.mybackup /etc/X11/xorg.conf



    を実行しcp /etc/X11/xorg.conf.mybackup /etc/X11/xorg.conf



  • 再起動して、必要に応じてインストールします。




    • コンソールのサポーターは苦労しません:)-この方法は「主婦と互換性があります」...しかし、Ubuntu自体と同様です。
