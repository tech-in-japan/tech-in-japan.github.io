神とラリー・エリソンの違いは何ですか



オラクルの創設者兼CEOであるラリー・エリソンは、地球上で最も裕福な人の一人であり、原作者であり、偶然にもビル・ゲイツのように恥ずかしがらず、プレイボーイへのインタビューを行っています。 彼の父親はロシア出身で、ラリーはマイクロソフトのポリシーをソビエト連邦のような状況とよく比較します：「競争の欠如は高価格と低品質をもたらします。」 ラリー・エリソンの伝記には、「神はラリー・エリソンとどのように違うのか」と題されています。



彼の母親の名前であるフローレンススペルマンは、48歳のときに初めて認められました。 19歳の少女は、母親の叔母につかの間の愛の実を与えました。 リリアンと夫のルイ・エリソンは新生児を養子にしました。



学校で、ラリーは並外れた数学的才能を示しました。 彼はイリノイ大学で2年間輝かしく勉強しましたが、ここで養母が亡くなり、試験を受けませんでした。 それから彼はシカゴ大学で1学期を過ごし、そこで最初にコンピューターサイエンスと接触しましたが、カリフォルニアに転がり込んでコンピューターサイエンスを投げました。



新しい場所で、彼はFortranでプログラムをおしっこし始めました。 経験は蓄積され、知人はコンピューターの世界で作られていました。 アリソンは特別な文献を読みました。 そして、1977年に「ストーブに座って」12年を経て、彼は2人の友人であるボブマイナーとエドオーツとともにソフトウェア開発研究所を設立しました（間もなくRelational Software Incと改名）。 この重要なイベントのきっかけは、IBMの開発者であるEdgar Coddの記事でした。彼は目を引き、SQLクエリ言語を実行するコンピューターで機能するユニバーサルデータベースを作成する見通しについて話しました。



その前にデータベースが存在したと言わなければなりません。 ただし、それらを複製される製品と呼ぶのは大胆すぎるでしょう。 コンピューターを生産する各企業は独自の基盤を開発し、それらを別のマシンで使用することは不可能でした。 しかし、アリソンは、PDP-11ミニコンピューターでリレーショナルDBMSを構築することから始めて、普遍的な普遍性とデータベース管理システム（DBMS）の前例のないシンプルさで揺れ動きました。 System RリレーショナルDBMSの配布エリアは、モンスター-IBMメインフレーム、場合によっては部屋のサイズに限定されていました。 ラリーの戦略は、彼の製品があらゆる種類の情報（プラント仕様、人員配置表、商業契約、天文観測銀行、インテリジェンスデータなど）を持つコンピューターで可能な限り効率的に動作することでした...



ラリはアメリカのintelligence報機関と古くからの友好関係があります。オラクルという名前でさえ、彼の会社は彼らがCIAのために開発した（そしてしなかった）プロジェクトを負っています。 プロジェクトとは異なり、オラクルは、これがまさに彼らが必要とするものであることをCIAに思い起こさせ、納得させることができました。 納得させる能力は、ラリー・エリソンの非常に強い側面です。 まず、彼は強力なカリスマ性を持っています。 第二に-優れた演技データ。 彼は彼の新製品のプレゼンテーションを忘れられないショーに変えます。



これらのビューの1つの例を次に示します。 巨大なモニターで、最高経営責任者のラリー・エリソンの壮大な画像。 30秒後、彼自身が戦闘機のように素早くステージに飛びます（そしてビジネスジェットの代わりに戦闘機を持っています）。 パーソナルコンピューターの主な敵対者である彼は、PCでプレゼンテーションプログラムを立ち上げると語っています。 不幸なPCにジャンプすると、あたかも肘で無邪気な鉄片にしがみついていて、落ち、煙が流れ出し、火花が降り注ぎ、何かが爆発するかのように。 これにはすべて、コミックジェスチャーと表情が伴います。 それからラリーは、幸運にも、プレゼンテーションのためにファイルをサーバーに転送することができたことを「覚えています」。 かわいくて楽しいが、同時に視覚的にも説得力がある。



人気が高まっていたDBMSは、他の皆と同じようには市場に参入しませんでしたが、わずかな気分でバージョン2からすぐに、Oracle 1は堅実に聞こえると判断しました。 状態構造は、新規参入者を好みません。 このバージョンのCIAを販売した後、アリソンはこの事実を広告目的で広く使用しました。当社の製品は米国の利益を保護します。 結果：1980年、同社の利益は100万ドルに達し、プログラムはIBMによってS / 370メインフレーム用に購入されました。 まあ、市場はOracle 3の出現に反応し、急ぎの需要がありました。 そして1983年、Relational Software Inc. Oracle Corporationに改名されたため、製造された製品を混乱させることはありませんでした。



同社は急速に拡大しており、400人の新入社員（主に営業マネージャー）が既に8人の「古参者」と並んで働いていました。 そして、販売はさらに速く成長しました。 すぐに、企業の資本は10億ドルを超えました。 そして1986年、1億8500万ドルの財産を持つラリー・エリソンはフォーブス誌にランクされました。 ビル・ゲイツもこの地に初めていましたが、315人がこの数百万人を抱えていたため、今日では「最初のプログラム大富豪」と呼ばれています。 アリソンは生涯ゲイツに追いつくよう努力してきましたが、それを隠すことは嫌いだとは考えていません。 かつて、大声でさえ、彼はビル・ゲイツの邸宅の上に彼の戦闘機「マルケッティ」で飛んだ。 確かに、彼が家にいたかどうかはわかりません。



時折、エリソンはゲイツを迂回することができました-2000年に彼は最も成功したCEOになり、年間で39億ドルの財産を増やし、480億ドルに達しました。これはOracleの成長により信じられないほど成長しました飛躍、電子商取引。 ゲイツは、独占禁止法訴訟の結果として、100億ドルを「失った」が、翌年、彼は取り戻した。



Oracle Corporationの開発は進歩的ではありませんでした。 1990年、同社の市場価値は予想外に80％低下しました。 破産に満ちた絶望的な状況がありました。 アリソンは、残酷ではないにしても、迅速かつ厳しく反応しました。 400人の従業員が解雇されました-スタッフの10％。 そしてほとんどすべてのトップマネジメント-ラリーに近い人々、彼と10年並んでいる人々は、会社の名声と財産を増やしました。 （熱心なプラグマティストのエリソンの例外は、常に才能のあるプログラマーであり、非常に優秀で親切な人であるボブ・マイナーでした。スピリット」



ラリーは言い訳をしませんでした。 彼は単に自分の立場を説明した。 「私たちはコンピューター業界の歴史の中で最も急速に成長している会社でしたが、突然行き止まりになって頭を壁にぶつけ始めました」と彼は言いました。 -収益は数十億に達しましたが、実質的に不溶性の管理上の問題に直面しています。 実際、10億番目の会社を経営していた人たちは、会社が1500万ドルを費やした当時と同じであり、私と一緒に働いたすべての人、Oracleを作成したすべての人に感謝の気持ちを感じました。 しかし、私には選択肢がありませんでした。 私は彼らを解雇しなければなりませんでした、これが行われなければ、すぐにオラクルはなくなるでしょう。 まず第一に、私は会社全体、すべてのスタッフ、すべての株主、顧客に対して責任を感じました。」



新しいマネージャーが雇われ、大量の生産と激しい競争に「訓練」されました。 そしてしばらくして、Oracleは以前の高さに戻りました。 しかし、エリソンは、ボブ・マイナーが以前にバランスを取ろうとしていた「ビジネスは人よりも重要」という原則から逸脱することはありませんでした。反対に、人はビジネスよりもはるかに重要です。



Oracle Corporationは、ソフトウェア市場でMicrosoftに次いで2位です。 ただし、DBMSセグメントでは、議論の余地のないリーダーです。市場の61％を所有しています。 そして、これらの地位は、絶え間ない革新のおかげで20年以上もオラクルが保持しています。 エリソンは、生まれたばかりのインターネットの巨大な可能性を高く評価し、その機能を効果的に使用できるようにする製品にその機能を事前に組み込んだ最初の1人でした。 その結果、eコマースメカニズムが一生懸命に働き始めるとすぐに、彼はパイの公平なシェアを獲得しました。 オラクル社のすべてのソフトウェア製品は、情報の巨大なリポジトリとしてインターネットと連携するだけでなく、World Wide Webの分散コンピューティングリソースを使用する機能も備えています。



Oracle Corporationの開発は、成功している企業と同様に、銀河形成のメカニズムと比較できます。 最初に、蓄積された知的および起業家のエネルギーの爆発が発生し、急速に拡大し、市場を獲得します。 時間が経つにつれて、エネルギー密度が低下し、膨張が遅くなります。 ただし、アリソンはまだ社内でかなりの内圧を維持しているため、停滞について話す必要はありません。 DBMSセグメントで支配的な地位を占めていたため、企業アプリケーションの分野（ERP、CRM、その他の「重い」ソフトウェア）で決定的に定着しました。



さらに、彼は彼の時間の彼の主要なライバルとして行動しました。 Bill Gatesは、彼のオペレーティングシステムをPC販売であるAllison-彼のアプリケーションを彼のDBMSに法的に密接にリンクしました。

その後、さらにいくつかの買収が行われ、小規模で、さらに150億ドルのコストがかかり、今年2月に、89億ドルのBEAの購入を発表しました。これは、エンタープライズアプリケーションの市場リーダーであるドイツ企業SAPに追いつくためです。 。 私は追いつくことができた。 追い越すために-まだ。 SAPの社長、ヘニング・カガーマンは、アリソンは市場の過剰について語るとき、まったく正しくないと言って説明しています。 その場合、実際に積極的な買収を行い、市場シェアを強化する必要があります。 しかし、すべてのニッチが占有されているわけではありません。 いずれにせよ、SAPは、新しい顧客を企業に引き付ける根本的に新しいソフトウェアソリューションを見つけて提供することに成功しています。
