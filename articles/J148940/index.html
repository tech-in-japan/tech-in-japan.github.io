理論



ジャグリングが一連のスローであると言うことは、音楽が単なる一連の音であると言うことと同じです。 あなたはこれを嘘と呼ぶことはできませんが、音楽理論に少しでも精通している人は誰でも、最後の定義に腹を立てます。

それらにはパターンがあります：

  • 手は交互に動作します-右、左、右、左...
  • ボールを受け取った手は、すぐに特定のパスに沿ってボールを投げます。
  • 各瞬間の各手は、最大1つのボールで動作します。 したがって、異なる軌跡と異なる飛行時間（特に最後の図で明確に見られる）にもかかわらず、ボールが互いに競合することはありません。


他のトリックでは、これらのポイントのいずれかが間違っている可能性があることに注意してください。 しかし、本質を理解するために、微妙なことを掘り下げる必要はありません。



パターンについて考えた後、次の質問が発生します。

  • トリックを記述する方法は？
  • 新しいものを表示する方法は？
  • あるトリックから別のトリックにスムーズに切り替える方法は？


このために、ジャグリング記法が発明されました。 各トリックは、SiteSwapsと呼ばれる特定のパターンによって記述されます。 完全な説明は行いませんが、トリックの基礎を設定します。 最初のアニメーションは、サイトスワップトリック「3」、2番目-「51」、3番目-「531」を示しています。 次のように記述します。 



 () -> «3» : 3 3 3 3 3 3 3 3 3 3 3 3 ... «51» : 5 1 5 1 5 1 5 1 5 1 5 1 ... «531» : 5 3 1 5 3 1 5 3 1 5 3 1 ...             ...




いいですか サイトスワップをループし、各桁の下にボールを投げる手を示しました。 各数字はスローです。 より速く理解するために、ボールが投げられる高さを意味すると言うことができます。 しかし、これは結果にすぎません。 時間は対策に分けられます。 各メジャーがスローされます。 片方の手は偶数ビートで動作し、もう一方の手は奇数ビートで動作します。 ハンド（たとえば、右のハンド）が3のスローを行った場合、このボールは3バー後にのみ再びフリーになります。 この間ずっと、彼について考える必要はありません。 しかし、まだボールがあります！ 最初の措置の後、最初に左手を、次に2番目の後に、右手を噛まなければなりません。 そして最後に、3番目の小節の後、ボールが戻り、左手でボールを投げなければなりません。



ご想像のとおり、奇妙なショットのあるボールはすべて、もう一方の手で、また投げられた同じ手でもキャッチしなければなりません。 そして、数字が大きいほど、ボールが高く飛ぶはずです。

始めましょう。 3つの球状オブジェクトが必要になります。 まず、りんご、みかん、 鶏の卵 、ジャガイモが適しています。 長い間は十分ではありませんが、テストには役立ちます。 気に入ったら、テニスボールをグリッツまたは水（シリンジ）で満たし、穴を塞ぐことをお勧めします。 古い靴下をエンドウ豆で満たし、縫い付け、空気のボールで覆うのがさらに良いです。



要するに、最速のトレーニング方法論はこのように見えます（参考のために！真剣な研究のために、下のリンクをクリックしてください）。

  1. アイテムを手に取ってください。 もう一方の手に投げます。 飛行高度はほぼあなたの身長です。 そして戻る。 そしてもう一度。 あなたが退屈するまで繰り返します。
  2. 2つのオブジェクトの交換：各手にオブジェクトを取ります。 手順1のトレーニング済みの動きで、1つのアイテムをドロップします。 彼が最高点に達したら、別のアイテムをドロップします。 最初に最初にキャッチし、次に2番目にキャッチします。 片手で最初の投球を練習してから、もう一方の手で練習します。
  3. 3つのアイテムの交換。 これで、一方の手に2つのアイテムがあり、もう一方の手に2つのアイテムがあります。 ステップ2のように開始し、2番目のボールが最高点になるときに3番目のスローを追加します。
  4. 手順3で成功した場合は、4番目のスローを追加してから、5番目のスローを追加してみてください。
