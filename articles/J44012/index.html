Windows XPでカスタムの既定のユーザープロファイルを作成する

nliteを使用して独自のWindows XPビルドを作成する場合、将来のすべてのユーザーに適用するために、通常はHKEY_CURRENT_USERに保存されるいくつかの設定が必要でした。



レジストリブランチHKEY_USERS \ .DEFAULTが新しいユーザーのレジストリのコピーを保存すると仮定するのは論理的ですが、これは事実とはほど遠いことが判明しました。 少しグーグルで、ローカルシステムのユーザープロファイルが保存されていることがわかりました。



材料のさらなる研究により、2つの方法が見つかりました。



a） regeditを実行し、HKEY_USERSセクションを選択して、[ファイル]-> [ハイブのロード]（[ファイル]-> [ハイブのロード]）をクリックします。 file％systemdrive％\ Documents and Settings \ Default User \ ntuser.datを選択します。 「ブッシュ」に自由な名前を付けます。 それだけです-将来のユーザーのために、HKEY_CURRENT_USERのコピーを手に持っています。 必要なものをすべて設定し、「ブッシュ」を選択して、「ファイル」->「ブッシュのアンロード」をクリックします。 出来上がり-すべて準備が整いました。



b）別の方法がMicrosoft Webサイトに公開されています。

1.管理者アカウントを使用してログインし、ローカルユーザーアカウントを作成します。

2.セッションを終了し、前の手順で作成したアカウントを使用してログインします。



注意！ 管理者としてカスタムユーザープロファイルを作成すると、アクセス権が誤って割り当てられます。



3.プロファイルに必要な設定を行います。 たとえば、プリンタをインストールし、ネットワークドライブをマップできます。

4.セッションを終了し、管理者アカウントを使用してログインします。

5.作成されたデフォルトユーザーカスタムプロファイルにコピーする必要のあるプロファイルファイルは非表示になるため、「非表示のファイルとフォルダーを表示する」オプションを有効にする必要があります。



6.現在のデフォルトユーザープロファイルを作成されたユーザープロファイルに置き換えます。



これを行うには、次の手順を実行します。

a。 コントロールパネルで、システムアイコンをダブルクリックします。

b。 [システムのプロパティ]ウィンドウで、[詳細設定]タブをクリックします。

c。 [ユーザープロファイル]グループで、[オプション]ボタンをクリックします。

d。 [ユーザープロファイル]ダイアログボックスで、新しく作成したプロファイルを選択し、[プロファイルのコピー]ボタンをクリックします。

e。 [プロファイルのコピー]ダイアログボックスの[オプションのグループにプロファイルをコピー]で、[参照]をクリックし、\ Documents and Settings \ Default Userフォルダを選択して、[OK]をクリックします。

f。 [使用を許可]グループで、[変更]ボタンをクリックし、リストで[すべて]グループを選択して[OK]をクリックします。 [すべて]グループが使用できない場合は、[詳細設定]ボタン、[検索]ボタンの順にクリックし、[すべて]グループを選択して[OK]をクリックします。



その後、このプロファイルを使用して、すべての新しいユーザーのプロファイルを作成します。



ご注意 作成および構成された既定のユーザープロファイルは、Windows XPを実行している別のコンピューターで使用できます。 これを行うには、C：\ Documents and Settings \ Default Userフォルダーに保存されているプロファイルを別のコンピューターの同様のフォルダーにコピーします。
