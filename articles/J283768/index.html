コンバージョン率の最適化に関する5つの素晴らしいレッスン（実際の例）





コンバージョン率の最適化は、 CrazyEggブログの著者のお気に入りのトピックであるだけでなく、サイトのページ上のユーザーの流れからより多くの利益を得ることができる方法でもあります。 最適化の方法を学ぶ最良の方法は、すでにそれを行った人から知識を引き出すことです。 以下は、コンバージョン率を最適化するためのヒントと、さまざまな企業で使用されている最新の最適化手法です。 あなたは時々、本当に驚くべき結果を達成することが可能であることがわかります。



1.モバイルデバイスからサイトにアクセスするユーザーの数を増やすために、サイトのデザインを便利にします



前に、便利なサイトデザインは、モバイルデバイスのユーザーに快適なナビゲーションを提供する上で重要な役割を果たすと述べました。 サイトが主にモバイルデバイスのユーザーを対象としている場合、他のユーザーは通常、サイトをより効果的にナビゲートする機会を得ます。 ただし、最も重要なことは、この場合、ユーザーが他のデバイスからより積極的にサイトにアクセスし始めることです。 以下は、ウェブサイト最適化によって公開された、フィラデルフィア矯正歯科のウェブサイトでのユーザーフレンドリーなデザインの導入の影響の例です。







2013年4月、リソースには通常の静的設計がありました。 しかし、会社の所有者は、このサイトがモバイルデバイスのユーザーの間で人気が高まっていることに気付きました。 （Dreamweaverプラットフォームで動作する古いバージョンとは異なり）WordPressプラットフォームで新しいサイトが作成され、トラフィックに対する新しいバージョンのサイトの起動の影響を評価しました。 統計によると：



  1. 新しいバージョンのリリース後1か月で、モバイルデバイスのユーザー数は24％増加し、モバイルデバイスとタブレットのユーザーの総トラフィックは21％増加しました。
  2. 発売から1年半後、上記の指標はそれぞれ70％と56％でした。




サイトのより便利なバージョンの立ち上げは、ユーザーがサイトでより多くの時間を費やすようになったことも意味しました。 モバイルデバイスのユーザー間のセッション時間は80％増加し、すべての種類のデバイスのユーザー間では39％増加しました。 おそらく、ページの読み込み時間の短縮により、サイトの魅力が高まっています。



2.「無料」に集中しないでください



プロモーションとギフトは最も人気のあるマーケティングツールの1つですが、次の例は、この方法では常に最大のコンバージョン率が得られるとは限らないことを示しています。 Devesh DesignのWebサイトは、ユーザーが2週間の無料試用版プログラムを使用して独自の結婚式サイトを作成できるWedBuddy社の事例を紹介しています。



オリジナルのWedBuddy Webサイトには無料トライアルプログラムへのリンクが3つあり、その利点の長いリストもありましたが、コンバージョン率はまだ望んでいませんでした。 Devesh Designによると、その理由は次のとおりです。サイトのメインページの情報は、最終的にユーザーはプログラムの料金を支払う必要があることを強調し、これがこの商業的動きの効果を相殺しました。



サイトの更新されたページに重点が移りました。ページ自体が短くなり、「無料」という言葉がプログラムの試用版の発表から削除されました。 代わりに、メインページでは、プログラムを使用するとサイトを非常に迅速に作成でき、訪問者はそのようなプログラムを使用できることを強調しました。



その結果、試用版のクリック数は139％増加し、プログラムのサブスクリプション数は73％増加しました。



この例は、プログラムの利点を強調し、それを使用するように勧めることが、サービスのコストに関する情報よりも重要な役割を果たすことを示しました。



3.サイトを終了するときに表示されるウィンドウを使用して、サイトを離れる人の数を減らします



ユーザーはさまざまな理由でサイトを離れます-必要な情報が見つからないことから、ページの読み込み速度が遅いことまで。 消費者がまだ慣れていない新しいサービスを提供すると、状況はさらに悪化する可能性があります。



YourMechanic社に関する事例がRoosterのWebサイトで公開されました。Roosterは長い間、提供する携帯電話の修理がどれだけ必要かを評価しようとしていました。



タスクは、価格表に慣れるためにユーザーを引き付けることでした。 価格表を表示せずにサイトを離れるユーザーの数を減らすために、Roosterスペシャリストは、ユーザーがサイトを閉じたときに表示されるポップアップウィンドウをサイトに配置しました。 ウィンドウは、価格表を非常に迅速かつ簡単に取得できることを強調しました。わずか73秒です。 また、ウィンドウには「今すぐ価格表を入手してください」という碑文でアクションを促すオレンジ色のボタンがありました。



結果：7、最終的にサイトを離れる人の16％が最終的に価格表をダウンロードしました。 これは、ユーザーにオファーを受け入れる2番目の機会を与えることが時々重要であることを証明しました。



4.関心のあるユーザーにとって便利なサイトナビゲーションを作成する



コンバージョン率を最適化するためのすべてのヒントは、リンクのクリック数においてナビゲーションが大きな役割を果たすと述べています。 次の例は、これが真実であることを証明しています。



ハーバードビジネスサービスは、デラウェア州でエクイティサービスを購入するユーザーを増やすことを望んでいました。 しかし、会社の所有者がサイトのナビゲーションの機能を分析したところ、コンバージョン率が必要なものと一致しなかったことが判明しました。 Visual Website Optimizerテストにより、同社はコンバージョン率を高めるためのいくつかの革新的な技術を開発することができました。







これらの革新の1つは、サイトのキーボタンの碑文を置き換えることでした。 特に、価格はこの会社にとって大きな競争上の優位性だったため、[比較]ボタンは[価格の比較]に置き換えられました。 その他の変更点：



  1. 「今すぐ開始」ボタンを「会社を作成」ボタンに置​​き換え、会社がサイトのユーザーを待っていた特定のアクションを強調します。
  2. 「会社の作成方法」ボタンを追加して、サイドバーの古いリンクを置き換えます。
  3. サイト上のボタンの総数を減らします。




簡素化されたナビゲーションが実を結びました。 A / Bテスト中に、次のことが明らかになりました。



「会社の作り方」ページへの訪問数は382％増加しました。

価格比較を伴うページアクセス数は66％増加しました。

発注数は15％増加しました。



Visual Website Optimizerを使用して実施した分析では、購入前に顧客に情報を提供し、サイトのオプションを明確にして行動を促すインセンティブにすることがいかに重要であるかが示されました。



5.顧客のストーリーを公開し、大容量のボタン言語を使用する



過去の顧客事例について言及することはマーケティングにおいて大きな役割を果たしますが、コンバージョン率を最適化するために使用する価値は本当にありますか？ はい、それは価値があります。特に適切なストーリーを共有する場合は。



Adept Marketingのクライアントの中には、寄付を集める専用のサイトを改善したい非営利団体がありました。 組織の古いページにアクセスするとき、ユーザーは議論されていることや組織の目的が常に理解できるとは限りませんでした。 言い換えれば、古いページは顧客の話を言及するようなテクニックを使用しておらず、サイトは訪問者が組織のアイデアに触発されることを勧めることができませんでした。 Adept Marketingは、ページにいくつかのイノベーションを追加することでこの問題を解決しました。 ページが短くなり、空きスペースが増えました。 さらに、Adept Marketingのスペシャリストは次のことを行いました。



  1. ページの上部に寄付を集める目的の明確な説明を追加しました。
  2. 彼らは問題の歴史を、関連する写真とともに添付しました。
  3. 彼らは2つの異なる場所に寄付の呼びかけをしました。
  4. 進行状況バーが含まれているため、人々は寄付が一般的な原因に貢献していることがわかります。




このような措置は、すでに寄付を行うことを計画していた既存のスポンサーと、この問題についてさらに学びたいと考えている新しい潜在的なスポンサーの両方にプラスの効果をもたらしました。 募金ページはこれまで以上に人気が高まっています。 10時間、非営利組織は3万1千ドルを調達することができました。これは必要な金額を超えていました。



説得力のあるストーリー、いくつかのプッシュボタン、および大量の情報の流れの組み合わせにより、コンバージョン率が向上しました。
