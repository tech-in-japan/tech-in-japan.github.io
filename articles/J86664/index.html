Opera 10.50（Windows用）およびSilverlightで友達を作る方法

最近、Operaブラウザを更新した後、妹から、通常のプレーヤーではなく奇妙な画像が表示された理由を解明するように依頼されました。









インストールページに移動しましたが、 このブラウザではMicrosoft Silverlightがサポートされていない可能性があり、 Operaではサポートされていないサポートされているブラウザの表（下記を参照） しかし、まだ抜け道がありました。 habrakatの下での指示。











ご覧のとおり、Operaはリストにありません。 しかし、ここにはありませんでした。なぜなら ブラウザによって非公式にサポートされています!!!



まず、プラグイン自体をインストールします： Silverlight 3 。



インストール後、ブラウザを起動してアドレスバーにオペラを書き込みます：config＃UserPrefs | PluginPath 。 設定エディターに入ります。 プラグインパスで 、次のアドレスを追加します（アドレスは「;」で区切られます）。



インストールされたSilverlight 3の場合
  • Windows x86：C：\ Program Files \ Microsoft Silverlight \ 3.0.50106.0 \
  • Windows x64：C：\ Program Files（x86）\ Microsoft Silverlight \ 3.0.50106.0 \


*インストールされたSilverlight 4の場合
  • Windows x86：C：\ Program Files \ Microsoft Silverlight \ 4.0.41108.0 \
  • Windows x64：C：\ Program Files（x86）\ Microsoft Silverlight \ 4.0.41108.0 \


次に、[保存]をクリックして、ブラウザを再起動します。 すべて準備完了!!!
