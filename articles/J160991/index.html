Windows 9には「Windows Blue」という名前が付けられますか？ Windows 8の後継者に関する最初の詳細な前提





主要メディアは、PCの売り上げの減少、Windows 8の売り上げの減少（発売時のWindows 7よりも低い）、およびWindows 8でのタブレットの「事実上ない」売り上げを報告し始めました。 Windows、Windows Blueと呼ばれると言われています。



新しいNPDグループレポートは、米国Windows 8の低迷を報告しています



Guardianの記事「Windows 8は米国のバイヤーを喜ばせません」は、Windows 8の発売後最初の1か月でデスクトップコンピューターの売上が24％、ラップトップで9％減少したという新しい証拠を提供しています。 対応するNPDグループレポートは、 ASUSの担当者が「現在Windows 8の需要はそれほど大きくない」と述べた数日後に発表されました。 また、このレポートでは、Windows 8でのタブレットの販売は「実質的に欠席」しており、NPDグループが監視しているすべての流通チャネルで1％未満を占めています。 タッチスクリーンを備えたWindows 8ラップトップはより幸運で、売上は6％に達しました。 Windows 7が1か月経過した2009年の同時期に、コンピューターの83％が新しいOSで販売され、Windows 8の場合は58％しか販売されていなかったことが言及されています。



しかし、それだけではありません。 この記事で最も興味深い事実は、 MicrosoftがSurface RTのコンポーネントの順序を50％削減したというコンポーネントメーカー間の情報へのリンクです。



噂：Windows Blueは将来の低コストOSであり、年間更新サイクルがあります



西洋の大規模な出版物は、MicrosoftがWindows 8を置き換えるために急いで取り組んでいるという興味深い噂を発表しています。それらから最も興味深い仮定を引用します。



ZDNet：Windows Next：単に「Blue」と呼んでいますか？

  • マイクロソフトは既にWindows 8の後継に取り組んでいます。
  • Microsoft KitchenとWin8Chinaからの最近のもの（ WinUnleakedが主張）を含む「Windows 9」としての言及は既にあります。
  • それでも、そのように呼ばれることはまずありません。
  • 新しいリリースは「Blue」と呼ばれます-先週Win8Chinaがこれを発表しました-これは内部コード名でもあります。


Phone Arena：わずか数か月でWindows BlueがWindows 8に取って代わりますか？ ：
  • 彼は、新しいバージョンは数ヶ月でWindows 8を置き換えることができると示唆していますが、信じがたいと言っています。
  • Windows BlueはWindows 8に似ており、インターフェースが若干変更されていると報告しています。
  • おそらく、Windows 8の名前は保持されますが、他のすべて（価格設定を含む）は変更されます。
  • 彼は、MicrosoftがすべてのコンピューターにWindows Blueを搭載することを望んでいると言います。
  • この時点から、更新はデスクトップデバイスとモバイルデバイスの両方で毎年行われると想定されています。
  • 一部の情報源は、更新プログラムが2013年半ばにリリースされると報告しています。
  • Windows 8でのみアップグレードする必要があるかどうかについての情報はありません。


The Verge：Windows Blueは、毎年更新されるMicrosoftの将来の低コストOSです。

  • MicrosoftがWindows Blueを、誰もがインストールするOSにしたいと望んでいることを確認します。
  • 彼はまた、価格は低く、おそらく完全に無料になると言っています。
  • Windows Blueのリリース後、新しいリリースのサポートがWindows SDKに追加されます（Microsoftは開発者にBlue用のアプリケーションを作成させるために、Windows 8専用に作成されたアプリケーションの受け入れを停止します）。
  • SDKが変更されても、Windows 8アプリはBlueで実行できます。
  • Windows Blueにアップグレードするには、Windowsのライセンスコピーが必要です。
  • アップグレードしようとするコピーが海賊版になっている場合、組み込みアプリケーションとWindowsストアは動作を停止します。
  • AppleとGoogleの競合プラットフォームと比較して、Windowsの競争力を高めるために、毎年更新サイクルが導入されます。


どのエディションも、[スタート]ボタンと[スタート]メニューの戻り、およびMetroをオフにする機能について報告していません。
