新しいRaspberry Piの16の面白いプロジェクト





新しいRaspberry Piは 、より小さく、安く（わずか20ドル）、エネルギー効率が向上しています。 すでに売れているデバイスに適したアップデート。 新しいモデルと元のモデルの両方に実装できる、最もクールなPiベースのプロジェクトを収集しました。



1.ストップモーションカメラ







Piでオプションのカメラモジュールを使用する方法は多数あります。 そしてそれらの1つはストップモーションアニメーションです。 レゴのフィギュアを撮ると、このテクニックを使って映画全体を撮影できます。 Raspberry Piとカメラモジュールに加えて、ブレッドボード、ジャンパー、触覚ボタンも必要です。



詳細な手順については、Raspberry PiのWebサイトをご覧ください 。



2.ワイヤレスアクセスポイント

Raspberry Piでネットワークアクセスを構成します。 必要に応じて、別のゲストネットワークを作成することもできます。 これを行うには、SDカード、USBネットワークアダプター、およびすべてを正しくセットアップするためのコーディングスキルが必要です。



The MagPiのマニュアルを参照してください 。



3.車のダッシュボード用のタッチスクリーン







十分な時間を費やしたい場合は、Raspberry Piに基づいてタッチパッドを作成してみてください。これにより、数百ドル節約できます。 オープンソフトウェアXBMC Media Centerのおかげで、音楽を再生したり、ビデオや写真を視聴したり、さらに多くのことができます（道をたどることを忘れないでください）。



Instructablesの詳細をご覧ください 。



4.ロボット

ニンテンドーWii用の追加のコントローラーがある場合は、それを使用して独自のロボットを作成できます。その外観のみが、想像力と使用可能な素材に完全に依存します。 ロボットが移動できるように、シャーシとモーターコントローラーも必要です。



Raspberry Pi Webサイトの詳細な手順 。



5.モーションセンサー付きカメラ

外出中に誰があなたのオフィスに入ったのか知りたいですか？ または、あなたの庭にいることが多いリスの良いショットを作ることができますか？ Raspberry Piに基づいたモーションセンサーを備えたカメラが役立ちます。 作成するには、パッシブ赤外線センサーとコーディングスキルが必要になりますが、ガジェットに費やした時間は完全に報われます。



Instructablesには詳細なビデオチュートリアルがあります。



6.ストラトスタット







バルーンに取り付けられたRaspberry Piのおかげで、成層圏から信じられないほどの美しさのショットをキャプチャできます。 GPSを使用してその動きを追跡し、デバイスがまだ失われている場合はリモートで写真を撮ることができます。



Dave Ackermanのブログには便利なリンクがたくさんあります 。



7.音声変換

読みに問題がある場合は、テキストを音声に変換するデバイスを作成できます。 そのような問題が一度もなかったとしても、これはまだ魅力的なプロジェクトです。 もちろん、追加の部品が必要になり、品質はオーディオブックに匹敵しませんが、それだけの価値はあります。



プロジェクトの詳細については、KolbireのWebサイトをご覧ください 。



8.カメラ







単純なRaspberry Piベースのカメラを作成する方法はいくつかあります。 そして、作成に費やす時間が長いほど、結果は良くなります。 ボード上の要素を再はんだ付けできる場合、3Dプリンターで印刷されたケースにTFTスクリーンを備えた薄型カメラを入手できます。 どうやってやるの？ ビデオをご覧ください。



9.チューブクロック

Raspberry PiランプとNixieランプを使用して時計を作成することはそれほど難しい作業ではなく、その結果は印象的です。 さらに、それらをインターネットに接続する場合、夏時間への自動切り替えを構成したり、その逆も可能です。 ほんの数行のコード-デスクにワイヤレスデバイスがあります。



Martin Oldfilのウェブサイトの手順をご覧ください 。



10.モールス信号送信機







Raspberry Piは、新旧のテクノロジーを組み合わせてモールス信号送信機を作成します。 Piをプログラムするには一生懸命努力する必要がありますが、すべてが完了すると、モールス信号のエンコードとデコードが可能なデバイスが手元にあります。 混乱して旧式の送信機を作成することさえできます。



Raspberry Pi Webサイトの完全なガイド 。



11.気象ステーション

コンパクト、安価、低消費電力-Raspberry Piは、独自の気象ステーションを作成するのに最適です。 その後、テレビの天気予報に頼る必要がなくなります。 確かに、少しの「鉄」が必要ですが、一見すると思えるほど複雑ではありません。



DragonTailの詳細 。



12.バードハウス







プラスラズベリーパイの箱-そして、鳥の生き方を観察する機会があります。 オンラインストリームを設定することもできます。 必要な詳細には、赤外線LEDとNoIRカメラモジュールがあります。



Raspberry Pi Webサイトの手順を参照してください 。



13. Wi-Fiオーブン







この太鼓腹のストーブを作成するために、James Gaoは古い電気炉Raspberry Piと3Dプリンターで印刷された部品を取りました。 閉ループフィードバックシステムを使用して、リモートコントロールが実行され、温度レベルが自動的に調整されます（これには熱電対とステッピングモーターが必要です）。



コードと写真はRaspberry Piブログにあります。



14.レトロゲーム機

Raspberry Piは、簡単にコンソール、ハンドヘルドコンピューター、または昔ながらのゲーム機に変えることができます。 最も簡単な方法は、エミュレーターをSDカードにダウンロードし、USBコントローラーを接続することです。 時間に合わせて、RetroPieがお手伝いします。



完全なガイドはLifehackerにあります。



15.ジャグリングのためのメイス







突然あなたの中にジャグリングが好きな人がいます、そしてあなたはこの趣味に多様性を加えたいです。 LEDでメイス（またはボール）を「復活」させることができます。 SDカード、Pibrellaボード、およびPythonコードが必要です。 それは異常に見えます。



Lauren Eggtのブログをご覧ください 。 プロジェクトのすべての詳細があります 。



16.デジタルガーデン







このプロジェクトには、Pibrellaボード、いくつかの小型モーター、そしてもちろんRaspberry Piが必要です。 花ファン、ボタンまたはタッチのタッチで飛行を開始し、バックグラウンドでなだめるような音楽と蜂。



Geek Gurl DiariesのYouTubeチャンネルにはチュートリアルがあります。
