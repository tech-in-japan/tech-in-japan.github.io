SkypeはDigiumとの動作を停止します。 Skype for Asteriskは廃止されました

SkypeとMicrosoftの取引は報われているようだ。 本日、Digiumは7月26日にプロプライエタリなSkype for Asterisk製品（文字通り-Skype for Asterisk）の販売を停止したことを顧客に通知しました。 Skype for Asteriskは別のモジュールとしてAsteriskに配信され、名前が示すように、SkypeトランスポートをAsteriskに統合することが許可されました。



以下はDigiumの手紙の翻訳です。



2011年7月26日以降、アスタリスクのSkypeの販売とアクティベーションは中止されます。



Skype for Asteriskは、Skypeと共同でDigiumによって開発されました。 この製品には、AsteriskがネイティブクライアントとしてSkypeネットワークに参加できる独自のソフトウェアが含まれています。 Skypeは契約を更新しないことを決定しました。これにより、このソフトウェアを製品に含めることができます。 そのため、Skype for Asteriskの販売とアクティベーションは2011年7月26日に終了します。



この変更は、既存のSkype for Asteriskユーザーには影響しません。 Skypeの担当者は、Digiumとの契約に示されているように、Skype for Asteriskソフトウェアのサポートをさらに2年間提供し続けることを保証しました。 Skype for Asteriskのユーザーは、少なくとも2013年7月26日まで、SkypeネットワークでAsteriskを使用してシステムを使用し続けることができると考えています。 拡張機能は、Skypeの裁量によります。



Skype for Asteriskは、2011年7月26日まで販売されます。 この日付までにすべての計画された購入とアクティベーションを完了してください。



私たちとビジネスをしてくれてありがとう。



製品管理会社Digium。
