Windows 7 / Windows 2008 Server R2 SP1で行われた修正の全リスト

多くのユーザーは、Windows 7およびWindows 2008 Server R2の今後の大きなサービスパックに興味を持っています。 通常、列挙革新は比較的に限定されている変更の小さなリスト 。



最初のサービスパックをもたらすすべての変更点について詳細に学びたい人のために、マイクロソフトは、特別な準備している技術的な材料を 。 3つのドキュメントが提示されます。

  • Windows 7およびWindows Server 2008 R2 Service Pack 1に含まれる修正プログラムとセキュリティ更新プログラムの大規模で詳細なリストには、約800のセキュリティ、安定性、および互換性修正プログラムが記載されています。 リスト内のすべての変更は、簡単な説明とサポートサイト上の記事へのリンクが添付されています。
  • Windows®7およびWindowsServer®2008 R2のService Pack 1の注目すべき変更点。最大の変更点について説明しています。
  • 大規模なドキュメントWindows Server 2008 R2およびWindows 7 Service Pack 1テストフォーカスガイド。システムの顕著な変化を詳しく説明し、ITスタッフおよびシステム管理者向けにテストするための手順と推奨事項を提供します。
次のリンクからドキュメントをダウンロードできます。 注意、ダウンロードするにはシステム検証を実行する必要があります。
