エクスタシーのOpera開発者-1日4ビルド

1日以上経たないうちに、オペラの開発者のブログに4つのブラウザビルドが登場しました。 そして、これはとにかくどのようなアセンブリではなく、リリース候補です。 ノルウェー王国では完全に明確ではない何かが起こっています...



現在の最新のビルド番号は3296で、これはRC 5です。



ブログのコメントで、ユーザーはすでにOperaが労働法に違反しているかどうか疑問に思っています...奴隷労働は使えますか？



ビルドリンク： Windows MSI / Windows Classic



UPD：このビルドはリリースになったようです： ftp : //ftp.opera.com/pub/opera/win/1050/



UPD2： 10.50がリリースされました： www.opera.com
