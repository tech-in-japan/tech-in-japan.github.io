ウェアラブル向けTizen SDKがリリースされました

こんにちは、Habr！



以前、 サムスンのGear 2とGear 2 Neoの新しいウェアラブルデバイスとそれらの機能については既に説明しました。 しかし、本日、スマートウォッチのアプリケーションを開発するための機器のリリースを発表します！ ウェアラブル向けTizen SDKの紹介。 ダウンロード可能になりました 。 これは最初のベータ版にすぎませんが、必要なものはすべて揃っています。使い慣れたEclipse IDEとQEMUエミュレーターの両方です。







SDKは、すべての機能を備えたTizen SDKと非常によく似ており、相互作用はSamsung Accessory Protocolを介して行われます。 SAPを使用すると、音楽の管理、通知とファイルの転送、携帯電話の検索などを行えます。



SDKのコンポーネントと、Tizen SDK for Wearableでできることを検討してください。



Webフレームワーク


ウェアラブルデバイス向けに最適化されたほぼ完全なWeb APIを提供します。 WebKitとWebアプリケーションのランタイムが含まれています。 サポートされている最も興味深い機能（完全なリストはここにあります ）：



1. W3C / HTML5仕様のサポート：

  • DOM /メディア/グラフィック：HTML5オーディオ/ビデオ要素、HTML5フォーム（部分）、セッション履歴API、DOM / JS関連のHTML5拡張機能、iframeサンドボックス属性、HTML5 2Dキャンバス
  • CSS3：CSS3 2D変換（H / W加速）、CSS3 3D変換（H / W加速）、CSS3アニメーション（H / W加速）、CSS3トランジション（H / W加速）、CSS3色、CSS3背景および境界線（部分的に） 、CSS3フレキシブルボックスレイアウト（部分的）、CSS3ユーザーインターフェイス（部分的）
  • デバイス：タッチイベント、CSS3メディアクエリ（部分的）、振動API、getUserMedia API、バッテリーステータス、デバイス方向イベント（加速度計、ジャイロセンサーのみ）
  • ネットワーキング：XMLHttpRequestレベル2
  • セキュリティ：iframeサンドボックス、CSP1.0（部分）
  • UI：クリップボードAPI、ドラッグアンドドロップ
  • ストレージ：Webストレージ、ファイルリーダーAPI、インデックス付きDB API
  • パフォーマンス：Web Worker（Partial）、Page visibility API


2. W3C APIの変更：

  • ページ可視性API：「webkit」プレフィックスが削除されました（たとえば、webkitvisibilitychangeがvisibilitychangeになりました）
  • ファイルリーダーAPI：BlobBuilderは廃止され、Blobオブジェクトに置き換えられました
  • インデックス付きDBフラグの変更：マルチエントリからマルチエントリへ


3. Khronos仕様のサポート：WebGL、型付き配列



4.デバイスAPI-プラットフォームサポートデバイス機能にアクセスするには：

  • アラーム、アプリケーション、Bluetooth、ファイルシステム、システム情報、電源
  • Samsung Accessory Protocol API
  • モーション（歩数計、心拍、リストリフト）APIは、さまざまなセンサーへのアクセスを提供します
  • IrLED API
  • システム情報APIによって提供されるデータは利用できなくなりました：LOCALE、NETWORK、WIFI_NETWORK、CELLULAR_NETWORK、SIM


5.カメラAPIのサポート-プレビュー、設定の変更、写真やビデオのキャプチャ



6. Web UIフレームワーク-ウェアラブルデバイスに適したネイティブUIを作成するためのjavascript、css、htmlのセット



7.時計ウィジェットのサポート-カスタムデスクトップ時計ウィジェット



IDEとツール


1.オペレーティングシステムのサポート：Ubuntu 12.04、12.10（32ビットまたは64ビット）、Microsoft Windows XP（32ビット）Service Pack 2以降、Microsoft Windows 7（32ビットまたは64ビット）、Apple Mac OS X 10.7 Lion（64ビット）、Apple Mac OS X 10.8 Mountain Lion（64ビット）、Apple Mac OS X 10.9 Mavericks（64ビット）



2. IDE Eclipse



3. SDB（Smart Development Bridge）：シェルのインストール、アンインストール、アクセス



4.エミュレーター：

  • ハードウェアアクセラレーション、32/64ビットLinux（Ubuntu）、WindowsおよびMac OSでのGPUアクセラレーションのサポート
  • x86ベースのQEMUエミュレーターのサポート
  • マルチメディアコーデックのサポート
  • エミュレータコントロールパネル：歩数計データ、バッテリーステータス、USBの挿入


5.CLIツール：アセンブリ、インストール、削除、署名
