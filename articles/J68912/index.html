MegaFonおよびOperaからの無制限のインターネット









モスクワ地域のMegaFon加入者は、Opera Miniブラウザと特別なサービスパッケージを使用して、モバイルインターネットに無制限にアクセスできます。



2009年9月7日、ノルウェー、オスロ -Opera Softwareは本日、MegaFonモバイルオペレーターがOpera Miniブラウザーを、加入者に無制限のGPRS / EDGEおよびIMT-2000 / UMTSインターネットアクセスを提供する基盤として選択したことを発表しました。モスクワ地方で。



今日から、携帯電話にOpera Miniブラウザーをダウンロードすることで、首都圏のMegaFon加入者はOpera Miniサービスで無制限のインターネットを有効にし、24時間体制で送受信される情報量を考慮せずに1日あたり7ルーブルのサイトを閲覧できます。



注： Opera Miniブラウザバージョン2.X 3.Xおよび4.Xでインターネットリソースを表示する場合、ファイルのダウンロードとピアツーピアトラフィックは料金プランに従って支払われます



サービスの詳細： moscow.megafon.ru/services/internet/megaopera.html

携帯電話からOpera Miniブラウザーをダウンロード： mini.opera.com



MegaFonは、ロシア全土の4,650万人以上の加入者に高品質のモバイルサービスを提供する最初の全ロシアのモバイルオペレーターであり、そのうち700万人がモスクワ地域にいます。 同社の顧客の半数以上は、GPRS / EDGEおよびIMT-2000 / UMTSテクノロジーを使用したデータサービスのユーザーです。 MegaFon加入者の数は、同社が提供する革新的なインターネットベースのサービスのおかげなど、市場よりも急速に成長しています。



「ますます多くのお客様が、いつでもどこでもインターネットから情報を受け取りたいと考えています。 私たちの任務は、ネットワークへの完全なモバイルアクセスのための、便利で安全で習得しやすいプラットフォームを提供することです。 MegaFon OJSCのキャピタル事業部事業開発部長Vasily Kuzichev氏は、MegaFon加入者はトラフィックを節約してモバイルインターネットの使用を制限する必要がなくなると述べています。 -Opera Miniブラウザはロシアでよく知られており、すでにそのようなプラットフォームとしての地位を確立しています。 その助けにより、現在および将来のすべての加入者にモバイルインターネットの可能性を広げることを望んでいます。」



ロシアはOpera Miniの最大の消費国です。 過去1年間で、ロシアのブラウザユーザーの数は63.5％増加しました（出典： www.opera.com/smw/2009/07 ）。 MegaFonネットワークとのパートナーシップは、ロシア人向けのネットワークへのモバイルアクセスの機能を拡張するように設計されています。



Opera Softwareの事業開発担当副社長Nuno Sitima氏は、次のように述べています。「モスクワ地域のMegaFon加入者向けのOpera Miniブラウザーの特別バージョンは、ロシアの大手モバイルオペレーターとの協力の第一歩です。



MegaFon OJSCについて

GSM 900/1800標準の最初の全ロシアのモバイルオペレーターであるMegaFon OJSCは、2002年5月に設立されました。 MegaFonは最初になり、ロシアのすべての地域で独自のネットワークを展開する唯一のオペレーターです。 また、ロシアでUMTS規格の第3世代ネットワークを商業化した最初の企業でもありました。 ロシアの電気通信市場向けのユニークな範囲のサービスは、個人と法人顧客の両方を対象としています。 MegaFonネットワークは高度なテクノロジーを使用しており、顧客が情報をやり取りして作業するための新しい機会を切り開きます。 2009年、独立した審査員がMegaFonの「Absolute Brand-2009」を承認しました。 2009年、MegaFonは2014年冬季オリンピックの公式モバイルオペレーターになりました。 2009年7月1日、同社は子会社である地域事業会社の買収を通じて再編を完了しました。 最後の株主総会の日現在、MegaFon OJSCの主な株主はTelecominvest OJSC、TeliaSoneraグループ会社、Allocation Limitedです。 現在、企業の加入者数は4650万人を超えています。 会社に関する詳細： www.megafon.ru



Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
