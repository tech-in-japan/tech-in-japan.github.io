個人使用のためのGPS制御（パート2.システムの概要）

記事のこの部分では、2つのロシアのシステムと1つのベラルーシのシステムを検討します。



選択は基準に従って行われました：個人使用に適用され、無料で、スマートフォン上の追跡アプリケーションで動作できます（各システムについて、システム開発会社のフレームワークで作成されたOS Androidを搭載したスマートフォン上のアプリケーションが考慮されます）。



すべてのシステムは、モバイルデバイスのブラウザーを介してアクセスを提供します（ノートブック/デスクトップブラウザーの場合と同じページを開くだけでなく、小さな解像度に合わせて画面のレイアウトを変更します）。



考慮されるすべてのシステムには、独自の（企業のフレームワーク内で開発された）テレマティックサーバーがあります。 すべての場合、顧客サーバーにインストールできます。 特定の条件を明確にするには、開発者に連絡する必要があります。 重要な質問は次のとおりです。これに必要なリソース、システムの保守の容易さ、更新プロセスの構成方法。



市場にはさらに多くのシステムがありますが、選択したシステムは、個人監視用のツールを選択する際に検討する価値があります。



システム開発者や私が見逃したユーザーの観点から、おそらくいくつかの重要なポイント。 コメントに表示されることを願っています。





私のここhttp://www.gdemoi.ru/ （ロシア）







システム開発会社は2005年からテレマティクス市場で働いています。



個人モニタリングのアプリケーション：育児; 高齢者の助け; 初心者ドライバーの世話。 アクティブなレクリエーション、観光、スポーツ。



システムに接続されているデバイスの範囲は非常に広く、独自の生産のデバイスも含まれます（これにより、システムに効果的に統合できます）。



GPS / GLONASSモニタリングに加えて、このシステムにはGSMカメラを使用する機能があります。これは個人的な使用の分野でも興味深いと思います。



大企業のクライアントも、ソフトウェアの配布が組織のサーバーにインストールされるときにオプションを提供されます。



無料関税の制限 ：最大2つのオブジェクト。 1日間の履歴。 ロシアと世界の3つの地図。 無料の料金プランはトラッカーでのみ利用できます（モバイルアプリケーションでは利用できなくなりました）。



システムの便利な機能



  • 地図作成：3つのプロバイダー（10のうち可能）; 道路交通（Yandex、ProGorod、Google）。
  • 住所で地図上の場所を検索します。 ジオコーダーを選択できます（Yandex、ProGorod、Google、OpenStreetMap）。
  • ジオフェンス：円、多角形、ルート。 ルートはすぐに敷設されます。 同時に、（半径に沿って）作成時に指定されたコリドーが表示されます。
  • ジオフェンス：便利な編集、削除、およびマークされたジオフェンスの地図上での表示。
  • ミニマップを表示する機能。
  • マップにカスタムレイヤーを（KML形式で）追加できます。
  • オブジェクトのリストは、画面上に簡単に表示/非表示されます。 リスト内のオブジェクトを選択すると、特定の時間間隔でその上で旅行やイベントを見ることができます。 リスト内のオブジェクトごとに、ステータスのアイコンが表示されます。
  • リストから選択したイベントをクリックすると、このイベントの場所が地図に表示されます。
  • マップ上のオブジェクトのアイコンは、状態に応じて変化します。
  • 情報パネルが開いている場合（画面の下部）、マップ上（またはリスト内のオブジェクト）でオブジェクトの名前をクリックすると、オブジェクトの状態に関する詳細情報が表示されます。
  • オブジェクトの情報パネルはウィジェットで構成されています。 すべてのウィジェットが画面の幅に収まらない場合は、左右にスクロールし、（マウスをドラッグして）交換して非表示にできます。
  • 1つまたは異なるオブジェクトの複数のトラックをマップ上に同時に表示できます。 各トラックについて、速度、走行距離、および移動時間が地図上で直接利用できます。
  • 十分に開発されたアラートルールのシステム。 アラートルールは、さまざまなタイプのイベントに対して作成できます。旅行の開始/終了。 ジオフェンスの出入り。 スピード違反; ルートからの逸脱制御。 パニックボタン; 低バッテリー シャットダウンまたはオブジェクトなどとの接続の切断。 ルールは、イベントパラメーター（許容速度、バッテリー充電など）、テキストとチャネル（SMS、電子メール）アラート、タイムスケジュール（時間と曜日別）、およびこれらのイベントに対してアラートを生成するジオフェンスを設定します。
  • アプリケーションから直接テクニカルサポートに連絡できます。


スマートフォンアプリ「NAVIXY Android GPS Tracker」



主な機能：
  • 時間と距離を設定します。その後、ロケーションレコードが形成されます。
  • Gprsトラフィック節約モード。 データ転送モードの設定：連続的または特定の間隔。
  • 転送が行われているサーバーのアドレスを構成する機能。
  • バッテリーセーバーモード：GPS受信機をオンにする間隔を設定します。
  • アプリケーションウィジェットの設定：アプリケーションの現在の状態を表示するか、クリックされたときに追跡を開始/停止します。
  • 技術情報の表示-送信されなかったポイントの数。




GPSHome http://www.gpshome.ru/ （ロシア）







監視システムは、2007年以来GlobalSat LLCの専門家によって開発されています。



個人監視の適用分野：子供、高齢者、親relative、友人の安全を監視および保証します。



GlobalSatデバイスに加えて、人気のあるFalcom Stepp、Enfora、Xexun、Teltonikaなど、他のメーカーのさまざまなGLONASS / GPSトラッカーをシステムに接続できます。



GS Monitoring Onlineは、独自の監視サービスを編成するための専門的なプラットフォームです。 GS Monitoring Onlineは、ロシアや海外で成功を収めている数十社の企業で使用されており、自社ブランドでGLONASS / GPS監視サービスを提供しています。



無料関税の制限 ：最大3つのオブジェクト。 1日間の履歴。 ジオフェンスへの訪問を制御することはできません。 電子メールまたはSMSを介したイベントに関する運用上のアラートはありません。



システムの便利な機能



  • 地図作成：Yandex、Google、OpenStreetMap; 道路交通（Yandex、Google）。
  • ジオフェンス：ポリゴン。
  • ジオフェンス：便利な編集、削除、色の変更、マークされたジオフェンスの地図への表示。
  • 独自のPOI（ランドマーク）を追加するための便利なツール。 アイコンの素敵なセット。 これは、特定の半径とアイコンが中央にある一種の丸いジオフェンスです。
  • オブジェクトのリストは、画面上に簡単に表示/非表示されます。 リストでオブジェクトを選択すると、特定の時間間隔の履歴、旅行、イベントをすばやく確認できます。 リスト内のオブジェクトごとに、バッテリー充電とそのステータスアイコンが表示されます。
  • リストから選択したイベントをクリックすると、このイベントの場所が地図に表示されます。
  • リスト内のオブジェクト（またはマップ上のオブジェクトのアイコン）をクリックすると、マップの中央に配置され、その情報ウィンドウが開きます。 その中で直接、情報を表示する短いモードまたは詳細モードを選択できます。
  • 各トラックについて、速度、走行距離、および移動時間が地図上で直接利用できます。 トラックはKML、GPXにエクスポートできます。
  • 高度なアラートルールシステムが利用可能です。 アラートルールは、さまざまなタイプのイベントに対して作成できます。SOS。 ジオフェンスの出入りの制御。 バッテリーレベル制御; 過速度制御など。 ルールは、イベントパラメーター（許容速度、バッテリー充電など）のテキストとチャネル（SMS、電子メール）アラートを設定します。
  • アカウントに追加のユーザー（オペレーター）を作成できます。 オブジェクトを監視し、レポートを受信できますが、個人アカウントへの管理アクセス権はありません。


スマートフォンアプリケーション「GPShome Tracker（ベータ）」



主な機能：
  • 地図上のスマートフォンの位置を、現在の位置と座標決定の精度に関するデータとともに表示します。
  • 駐車中に位置レポートを送信する頻度を設定します。
  • 運転中に位置レポートを送信する頻度を設定します。
  • GPSローカリゼーションの使用の禁止/許可。
  • ネットワークローカリゼーションの使用の禁止/許可。
  • 速度が設定されたしきい値を超えている場合、GPS受信機をオフにすることの禁止。
  • GPS検索時間。 位置が決定されていない場合、GPS受信機がオフになる間隔を設定します。
  • モーション検出器の感度を設定します。
  • アプリケーション操作ログを生成して、開発者に情報を送信します（エラーが検出された場合）。




GPS-Trace Orange http://gps-trace.com/ （ベラルーシ）







設立：2007（ただし、これがGurtamの製品であることを考慮すると、システムは2002年以降に開発されたと想定できます）。



個人モニタリングのためのシンプルで便利なシステム：子供と親relativeの管理と安全。 イベントをキャプチャして、友人と共有できます（ソーシャルネットワーク経由を含む）。



同社はデバイスを提供していませんが、システムが動作できるデバイスのリストは、既存のすべてのシステムを大きく上回っています。



グルタムは、顧客のサーバーにインストールできる新しいWialon Localサービス製品、GPSおよびGLONASS監視システムを開発しました。



制限 ：1つのアカウントで、最大5つの監視オブジェクト（1つの自動車と4つのパーソナルトラッカー）を接続できます。 1か月の履歴 アカウントあたり最大20ジオフェンス。



システムの便利な機能



  • 地図作成：Yandex、OpenStreetMap、Gurtam Maps、Virtual Earth; 道路交通（Yandex）。
  • 住所で地図上の場所を検索します。
  • ジオフェンス：円、多角形、破線。
  • ジオフェンス：便利な編集、削除、色の変更。
  • 最大マップ縮尺にすばやく移行するためのボタン。
  • オブジェクトのリストが画面の最下行に表示されます。 左側にはパーソナルトラッカーがあり、右側にはカートラッカーがあります。
  • オブジェクトの名前（画面の一番下の行）をダブルクリックすると、オブジェクトの情報パネルが画面の下部に表示されます。
  • 情報パネルで、オブジェクトのパラメーターを編集できます。 アイコン、デバイスタイプ、IMEI、SIMカードを変更できます。 ここで、オブジェクトをアカウントから削除できます。
  • オブジェクトパネルでは、トラックの表示モードを設定し、トラックに関するレポートを要求できます。 トラックを再生するための便利なツール。 各トラックについて、ポイントにカーソルを合わせると、アドレス、時間、速度が表示されます。
  • オブジェクトパネルでは、スピード、ジオフェンス、デバイスとの通信、パニックボタンに関連するイベントに関する通知を設定できます。 通知は指定された電子メールに送信されます。
  • 画面の下部では、オブジェクトからのメッセージの現在のストリームでパネルを表示/非表示できます。 情報とオブジェクトパラメータを編集するためのユーザーの操作も記録されます。
  • 各オブジェクトを修正できます（オブジェクトを追う）。 同時に、メッセージが到着すると、オブジェクトが見えるようにマップがスケーリングされます。
  • ボタンを使用して、アプリケーションを全画面表示に切り替えることができます。


スマートフォンアプリ「GPS Tag Orange」



主な機能：

設定
  • サービス操作モード：常時またはタイムアウトによる操作モード。 ローミングの作業の制御。 再起動後の自動起動。 充電に接続したときに動作します。 スケジュールされたサービスモード。
  • サーバーへの接続：サーバーへの接続を制御して、着信メッセージを受信します。
  • オブジェクト設定：オブジェクトのパスワードを設定します。
  • 場所：GPSおよび/またはネットワークロケーションプロバイダーを選択します。 送信用のメッセージフィルタリングを有効にします。 最小時間間隔。 最小距離 コース変更; 速度差; 最大 エラーのマージン; 最高速度。
  • メッセージ設定：送信されるメッセージパラメーターの拡張：プロバイダー、精度、課金レベル。 状態とステータスを設定します。
  • 通知とステータス：通知タイプを選択します。 シグナルによる通知の追跡。 スマートフォンの通知パネルでサービスの表示モードを設定します。
  • その他：カメラ; 測定単位; 管理者パスワードおよびその他の設定。
メッセージ：表示および送信（画像を含む）

着信/発信メッセージを表示します。 メッセージを削除します。 画像と状態を送信します。
