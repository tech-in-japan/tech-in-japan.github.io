Google I / Oツアー

こんにちは、Touch InstinctのマネージングパートナーであるAndrewです。 Google I / Oに行った方法を説明します。 以下は、マウンテンビューの会場のツアー、会議の概要、および多くの写真です。







ネットワーキング



私にとって、I / OはMixerイベントグループの公式オープンの前日に始まりました。 Touch Instinctは、Android開発の信頼できる開発業者としてGoogleによって認定された25社の1つです。 メインイベントセンターから離れたGoogleキャンパスの1つで開催されたGoogle Developers Agency Program Mixerに招待されました。 Googleがテクノロジーを使用して代理店を個別に管理してくれるのはすばらしいことです。





会合は非公式の設定で行われ、公式の場はありません



会議で、彼らはAndroid Thingsの開発における最新のイノベーションについて話しました。GoogleDevelopers Agencyプログラムを支持する一般的な温かい言葉と、賞そのもののプレゼンテーションがありました。





このような作品は世界に25しかなく、そのうちの1つは私たちと共にあります。



しかし、主なことは、世界中の同僚とチャットする機会です。 アメリカ、ブラジル、トルコ、ドイツ、他の多くの国、ロシアからの人がいました。 Surf StudioからVladimir Makeevへの特別な挨拶。私たちは業界について楽しく元気に話をしました。





Surf StudioのVladimir Makeevに会った



あるネットワークの後には、別のネットワークがあり、すでに食物とアルコールがありました。 Googleはパートナーをなだめる方法と応援する方法を明確に知っています。



遊び場



人々は開会の前日に登録して、Keynoteでより良い場所を選びました。 私はステージの非常に近く、ほとんど認証機関の大統領の席に座っています。





プレゼンテーションの最初の数分。 GoogleのCEO、サンダーピチャイの舞台で。



ギフト



開発者が燃え尽きないように、Googleはメガネと日焼け止めをTシャツ付きのバッグに入れました。 些細なことですが、素晴らしい。 個人的には、KeyNoteの間、私は非常に寒く、コーヒーさえ節約しませんでした。 日当たりの良いカリフォルニアは、午前中に+11度を喜ばせました。 パフォーマンスの時までに、それは+18まで暖まりましたが、日陰に座っているのはまだ寒かったです。



うめき声のテーマを続ける-最後のI / Oで、誰もがGoogleからの無料サービスがないことに怒っていました。 今回、彼らは追いつくことに決め、各参加者がGoogle Home、Googleフォトからの無料の印刷アルバム（はい、しかしなぜ??）、およびGoogle Cloud Platformの700ドルのローンを取得することを発表しました。 感嘆符と拍手から判断すると、誰もが喜んでいた。





Google Homeを配布する



それとは別に、Tシャツ/パーカーを50ドルまたは25ドルで購入することもできますが、最初の日の半ばにはすべてのパーカーが売り切れました。 小便ではなく、Vova、私はあなたを連れて行くことができた。





そのような屋台で彼らは商品を売った



食べ物



食べ物は飛行機のように普通でした。 それでも、数千人を対象とした本格的なケータリングは手配が困難です。





会議参加者のためのいくつかの種類の食べ物。 それぞれのニーズに応じて。



彼らがコーヒー、飲み物、水をどこにでも配ってくれて嬉しいです。 日中には暑くなり、日陰に隠れなければなりませんでした。 彼らがクリームを入れるのは良いことです。





Androidはどこにでもあります



昼食時には、座る場所はほとんどありません。 自由な角度で食事をする必要がありました。





ダイニングエリア、座席が十分ではありません



ちなみに、2日目の終わりに、オフィスの敷地内で火災が発生し、被害者さえいました。 しかし、会議は続きました。



輸送と住宅



輸送と住宅に関しては、すべて私が滞在していた場所に依存していました。 主要ホテルからのシャトルは30分に1回行き、多くはUberとLyftに行きました。 イベントの周りのパーカーは交通を非常によく制御し、駐車場に向かったが、朝はまだ交通に立ち入らなければならなかった。 サンフランシスコに住んでいた人々にとって、唯一の選択肢はカルトレインであり、その後駅からのシャトルでしたが、これには多くの時間がかかりました。 私は毎日、いわゆる逆通勤車を運転しました。



駐車場では十分ではないかもしれないという警告にもかかわらず、十分なスペースがありました。 しかし、サンフランシスコからマウンテンビューへのトラフィックは、朝夕にひどいです。 各方向に5車線、それでも誰もが立っています。





好きな場所に駐車します。 彼らは整理された方法で全員を駐車場に運転しましたが、ほこりの多いフィールドに沿って歩くことは氷ではありません



ロシアの子供たちの中には、サンタクララの家を9人で借りた人もいました。 安くて楽しく、しかも安いホテルに住んでいません。

基調講演



発表については、怠け者ではない人たちによってすでに書かれています。 もちろん、 VCを読んで 、繰り返しはしません。 Keynoteの一般的な印象は少々圧倒的でした。 不要な漫画のアニメーションとビデオブロガーのスローモーションの水玉を投げる -AppleのWWDCでは、Keynoteでそのようなことがまだ許可されておらず、興味深い発表に重点が置かれています。



私にとっての主なテーマは、GoogleがMobile Firstパラダイムから脱却することです。 ユーザーの新しいユーザーインターフェイスとなる人工知能を置き換えます。 コンテキストに応じたGoogle Assistant、画像認識機能を備えたGoogle Lens、屋内ナビゲーション機能を備えたVPS、6自由度のスタンドアロンVR これはすべて、機械学習技術に基づいて機能します。機械学習技術は、ユーザーとデバイス間の新しい主要な通信方法になりつつあります。 今後5〜10年で、コンピューターとの相互作用が何を意味していても、大きな変化に直面するでしょう。



Googleは、パートナーに新しいデバイスの無料テストを提供しています。今年は、Google Homeオフィスでそれを送信しました。 巨大マルチコプターの販売開始を待っています。
