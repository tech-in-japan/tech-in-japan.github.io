HTC HeroをHTPCに変える





先日私は考えていました-私のHTCヒーローのカメラで撮影した写真をテレビ番組に表示する方法は？ 答えは同時にシンプルで複雑でした-電話でDLNAサーバーを上げる必要があります。 マーケットで「dlna」というキーワードをすばやく検索した結果、すばらしい無料プログラムTwonkyServer Mobileが見つかりました。 私は彼女についてのhabrasocietyについてお話したいと思います。



たまたま寝室にテレビがあり、ビデオは彼に関するものだけになります。 =）







多くの人がDLNAとは何かを知らないため、Wikipediaからコピーアンドペーストします。

DLNA（Digital Living Network Alliance）は、互換性のあるデバイスがホームネットワークを介してさまざまなメディアコンテンツ（画像、音楽、ビデオ）を送受信し、リアルタイムで表示できるようにする規格です。 これは、家庭用コンピューター、携帯電話、ラップトップ、家電を単一のデジタルネットワークに接続するための技術です。 ユーザーの要求に応じて、DLNA仕様をサポートするデバイスを構成し、自動モードでホームネットワークに統合できます。

IPネットワークは、メディアコンテンツを送信するための物理メディアとして機能します。 DLNA互換デバイスをホームネットワークに接続するには、有線（イーサネット）および/またはワイヤレス（Wi-Fi）が可能です。




また、ホームネットワークのデバイス図についても少し説明します。 Wi-Fiを備えた境界ルーターがあります。これは、アパート全体のすべてのネットワークレースが収束するポイントにあります。 靴ひもが3つしかないため（1つは配電盤から、もう2つは部屋の電源コンセントへ）、それらは壁のコンセントに接続されており、すでにそれを通してルーターに接続されています。 HeroはルーターにWi-Fi経由で接続し、テレビは寝室の壁の壁コンセントを介して接続します（ビデオでは、壁に沿ってテレビから青いパッチコードが下がっています）。



テレビのセットアップ



私の場合、フィリップス37pfl8404は壁にぶら下がっており、mpegとmpeg2以外は何も再生できません。そのため、ビデオを頼りにしません。 テレビは、ホームルーターに接続されている独自の100Mbitネットワークアダプターを介してネットワークに接続されます。 プログラム自体は、プリインストールされたデバイスの膨大なリストをサポートしています（名前の80％は私に何も教えてくれませんでした）。その中には、テレビやメディアプレーヤーのほかにPS3とXbox360があります。 私にとっての主なタスクは、画像の正しい表示と音楽の再生を構成することです。



小さなTwonkyServerのすべての設定は（これは他のどこにも示されていませんが）、便利でRussifiedのWebベースのインターフェースを介して実行されます。 驚いたことに、プログラムにはビルトインメニューがまったくありません。画面上の2つのボタンのみが「オフ」と「再スキャン」です。 Webインターフェイスはhttp：// _IP_：9000で開きます。ここで、_IP_はAndroidスマートフォンのアドレスです。 もちろん、仕事をするには、自宅のWi-Fiネットワークに接続する必要があります。







この画面では、アプリケーションのバージョン、稼働時間、キャッシュサイズ、サーバーコンテンツなどのサーバーステータスを確認できます。 また、Webフェイスからコンテンツを表示し、http経由でアップロードしたり、ファイルをアップロードしたり（このためにSDカードに特別なフォルダーが作成されます）、他のサーバー設定を行うことができます。







TwonkyServerがフルサイズの画像をテレビに送信することを拒否し、160x130ピクセルのサイズに圧縮したことは注目に値します。 回避策として、私はそれがフィリップスではなく、その前にある通常のXbox360であることをサーバーに納得させなければなりませんでした。 Windows Media PlayerはサーバーをWindows 7として定義し、そのような問題は発生しませんでした。フルサイズの画像を表示し、低プロセッサTVよりもはるかに高速でした。
