ロシアのSilverlight-RuTubeはSmooth Streamingプラットフォームを作りました

4月20日、 ReMIX09カンファレンスで、 RuTube Smooth Streamingテクノロジーをホストするロシア最大のビデオのテストサイトを発表しました 。 このテクノロジーにより、インターネットチャネルの速度とユーザーの中央処理装置の負荷に応じて、リアルタイムで画質を動的に変更できます。



Smooth Streamingの最終バージョンは、昨日リリースされました。これは、Windows Server 2008、Windows Vista、およびWindows 7に含まれるIIS7 Webサーバーへの無料追加です。クライアントは、Silverlight 2のプレーヤーです



現在、 http：//silverlight.rutube.ruにアクセスして、すべての動作を確認できます。 このページには、さまざまな向きとさまざまな品質のビデオのコレクションが含まれています。



Smooth Streamingのより詳細な説明とローカルコンピューターへの展開方法は、 オンラインレポートで見ることができます。



これとは別に、これはカナダのCTVに続いて、この技術をテストする世界で2番目に本質的なものであることに注意してください。 SmoothHDと呼ばれる特別なサイトも数か月間運用されており、Smooth Streamingの動作を見ることができます。



各ページで実際のページを試してみて、コメントであなたの印象を説明することをお勧めします-インターネットチャネルと地域を示します。 実践は、技術が私たちの巨大な国の多くの地域で機能することを示しています！



更新：ブラウザとシステムを指定してください。 Win7では、 IE8、FF3、Chromeで試しました。 Macでも動作するはずです
