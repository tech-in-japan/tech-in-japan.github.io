ReSharper 7はVisual Studio 2012をサポートし、Metroアプリケーション開発を簡素化します

Microsoft Visual Studioの広く知られている拡張機能の新しいバージョンであるReSharper 7のパブリックリリースが行われました。



ReSharper 7は、Visual Studio 2005、2008、2010のバージョンとの互換性を維持しながら、Visual Studio 2012を完全にサポートします。さらに、ReSharper 7には、Windows 8およびWinRTのアプリケーションを開発するチームにとって重要な広範な機能が含まれています-最新のオペレーティングシステムとランタイムマイクロソフト



ReSharper 7は、Visual Studio 2012にまだ切り替えない開発者にも役立ちます。VisualStudioのすべてのサポートされているバージョンで利用可能な新しい機能には、コード検査と自動修正のセットがあります。 コードを簡素化するための抽出クラスを含む新しいリファクタリング。 コードを視覚化して学習する新しい方法。 コードの品質の分析を特定の開発チームで採用された契約に適合させるのに役立つ契約注釈。





「このリリースでは、Windows 8プラットフォームでのMetroアプリケーションの開発にすぐに移行する予定の.NET開発者を支援したいと考えています。 ReSharperは、Windows 8の開発をスピードアップできるようにします」とReSharperのプロダクトマネージャーであるIlya Ryzhenkov氏は言います。一方、ReSharper 7は、従来の.NETアプリケーションを開発する人にも適しています。 XAML、JavaScript、VB.NETを含む個々の言語のサポートの改善は言うまでもありません。」



ReSharper 7の主要な革新のリストは次のとおりです。

  • Visual Studioの以前の3つのバージョン（2005、2008、2010）のサポートを維持しながら、Visual Studio 2012と完全に統合します。
  • Windows 8用のメトロアプリケーションを作成する開発者向けの広範な機能セット。
  • .NETコード品質管理を改善する新しいコード検査と注釈。
  • コードを簡素化するために広く使用されているExtractクラスのリファクタリングなど、新しい自動コードリファクタリング。
  • 多態的なクラスメンバーとCSSスタイルの階層の視覚化に関連する追加のコード学習の機会。
  • 新しく改善されたコード生成コマンド。
  • INotifyPropertyChangedインターフェイスの操作のサポート。Windowsフォーム、WPF、およびSilverlightアプリケーションの開発者の作業を簡素化します。
  • ソリューション全体から不要なアセンブリ参照を削除します。
  • JavaScriptの改善されたサポート、ASP.NET 4.5およびASP.NET MVC 4の基本サポートを含む、Web開発者向けの高度なツールキット。
  • MicrosoftのLightSwitchとSharePointの2つの追加開発テクノロジーのサポート。
