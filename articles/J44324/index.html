ロールモデルと夢

はじめに-小さな余談。 一般的に、それは奇妙です-「職業を取得する」のように、あなたがより多くの教師でそれを調べます。 たとえば、最近、パートタイムの学生である私たちに1Skuを設定するように教えられました。 地元の住民のほとんどは、これが何であるかを知っています、はい:)私には、二日酔いのことを知っています。 しかし、組み込み言語のオブジェクトモデルについてさらに学習し、既に知っている言語との類似性を引き出すために、本質を理解し、教師に質問することを試みました。 「オブジェクトモデル」と「プライベートメソッド」という言葉から、教師は沈殿物に陥りました。 くそー、そしてITのためにお金を払う、ええ-しかし、私はずっと前に教育について書きました、私は繰り返しません:)）



だから、私は何をしています。 一方で、履歴書の余分な行は悪くありません。「悪天候」の場合、常にstar死しないチャンスがあります。 1Snikovは州の基準ではそれほど悪くない、彼らはおそらくさらに多くのハックを支払う。 一方で、同性愛の売春婦にも十分な給与が支払われますが、これは理由ではありません...もちろん、この類推は少し下品ですが、有益な、しかし愛されていないものの、職業に対する私の態度をほぼ反映しています:)私はそれを見て、私は本当に作られていますか？これは、実際にヒステリックな叔母会計士としての役割のために？ または、地方の顧客（彼ら自身が神であり、開発者であるとみなす-すべて同じサービススタッフ）の間で非常に人気のある無名の名刺サイトの場合 あー



いいえ、私はダミーではありません。私は18-19歳で、1か月あたり8キロルーブルで生活し、1ページあたり30ドルのウェブレイアウトを行っています。 私の子供たちに食べるものがない場合-少なくとも1Cに行って設定し、少なくともあらゆる種類のナンセンスをコンピューターサイエンスに教えます。 しかし、すべてがそんなに悲しいわけではないので、学校のコンピューターサイエンスの先生は私を若い才能と呼び、ビルゲイツはスカートをはいて、もっとお世辞を称賛しました。 このために、私は幼少期に雑誌「若い技術者」を読み、技術の進歩とハイテクの未来を夢見ましたか？ 未来が来ました。 ここで重要なことは、適切な場所を見つけることです。



紳士ITスペシャリスト、教えてください、あなたは夢を持っていますか？ いや、それでもない-夢で、大文字で？ 「たくさんの生地、ビール、ひよこ/男が欲しい、何もしない」だけでなく、同じハイテクの未来をさらに近づけるナポレオン、クレイジーで高価な計画ですか？ 残念ながら、これらのすべてが実現可能というわけではありませんが、残念ながら（たとえば、マイクロコントローラーで低レベルのプログラミングとDIYのデバイス製造を始めていただければ幸いです。また、私のビジョンでは、マイクロ回路に煩わされることはありません）。 そして、30ドルの組版がちょうどこの夢への最初の一歩であったことを知っています（しかし、今ではほとんどHTML / XML / CSS / XSLTを話します）。私が今やっていることも一歩です。 痛みを抑え、より快適にしましょう-しかし、それほど面白くないです。 それとも、主婦が星についてうわさをし、世界的に有名な会社を観察するような、楽しい時間を過ごしながら家政婦として働くようなものでしょうか？



はい、腐ったトマトは投げられませんが、私は定期的にHackerマガジンを読みます。特にSceneセクションが好きです。私が読んだときに、あなたがすでに毎日使っている作品の性格と成果を賞賛していることを読んで、あなたの年齢で賢明な何かを書いた？ 私は知っている、多くの人が私に言う-「現実的である、誰もが天才であるべきではない」、またはそのような何か。 または、彼らは彼らの故郷の町には発展する場所がないという事実に言及し始めるでしょう。 そして、彼らは間違っているでしょう-少なくとも同じルゴフスキーを見てください。彼は「bydlokoderov」に向けられた失礼な発言で繰り返し怒りましたが、人生はまだすべてを置きます：彼は自己実現とお金を持ち、イギリスに住んでいます...今日まで、彼らはなぜC ++のニックネームがほとんど役に立たないのだろうと考えています。 そして、結局のところ、これは州出身の男性でもあります...一般的に、すべてが可能です-望み、夢を持ち、それに行くことが重要です。
