管理不要のChrome Frameの紹介

2009年9月にChrome Frameの最初のバージョンをリリースしました 。 私たちの目標は、Web開発者がHTML5のすべての機能を使用できるようにするだけでなく、最新のWebテクノロジーのアプリケーションを拡大することでした。 そして、最も印象的なことは、Chrome Frameがレガシーフリーアプリケーションを作成するのを助けた開発者から多くのフィードバックを受け取ったことです。



ただし、すべてのInternet ExplorerユーザーによるChrome Frameの普及を妨げる問題が1つありました。ChromeFrameをインストールするには、ユーザーに管理者権限が必要です。 今年、Google I / Oカンファレンスで、この問題がついに完全に解決されたと発表しました。



管理者権限を必要としないChrome Frameは、システムの起動時に別個のヘルパープロセスを開始します。 このプロセスは、最小限のシステムリソースを消費しながら、Internet ExplorerでChrome Frameプラグインをダウンロードするのに役立ちます。 インストール後、管理者権限のないユーザーは、Chrome Frameと他のすべてのユーザーを使用できます。



管理者権限を必要としないChrome Frameを自分で試すことができます 。 この機能は現在開発者チャンネルで利用可能ですが、近い将来、安定版リリースチャンネルにも追加される予定です。



技術的な詳細については、Chrome Frameに関するよくある質問をご覧ください 。 ディスカッショングループで印象を共有してください。ChromeFrameでエラーを見つけた場合は、 Chromiumバグトラッカーに送信してください。 今後数週間以内に、管理者権限を必要としないChrome Frameを安定版リリースチャネルに移行するために努力しています。 あなたのフィードバックは、このプロセスを可能な限りスピードアップするのに役立ちます。

Q：企業にChrome Frameを自動展開するためのツールはありますか？



A：ユーザーがダウンロードしてインストールできるのは非常に簡単ですが、自動インストール用のMSIパッケージを提供しています。 以下のリンクからMSIファイルをダウンロードできます。



http://www.google.com/chromeframe/eula.html?msi=true



MSI形式のインストーラーには、安定した開発チャネルのChrome Frameのバージョンが含まれており、インストールには管理者権限が必要です。



Q：Chrome Frameは自動更新を有効にしますか？



A：デフォルトでは、はい。 Chrome Frameは、Google Update自動更新システムを使用して、最新の更新とセキュリティ修正を自動的にダウンロードしてインストールします。



Q：自分で更新を管理したいです。 これをどのように行うことができますか？



A：Google Chrome FrameとGoogle Updateは、当社製品の更新やその他の動作を管理するために使用できる一連のグループポリシー設定を提供します。詳細は、Chrome Frame管理者ガイドに記載されています



それにもかかわらず、ユーザーが常に最新のセキュリティパッチをインストールするように、自動更新をオフにしないことを強くお勧めします。



Q：MSI更新をインストールするにはどうすればよいですか？



A：Chrome Frameの新しい安定バージョンが利用可能になると、上記のリンクから新しい.msiファイルをダウンロードできます。 MSIを更新する場合、次のコマンドラインを使用して、マイナーアップデートとして適用します。



msiexec / fvomus GoogleChromeFrameStandaloneEnterprise.msi REINSTALL = ALL REINSTALLMODE = vomus MSIENFORCEUPGRADECOMPONENTRULES = 1 / l * v log.txt



Q：Chrome FrameはInternet Explorerの安全性を本当に低下させますか？



A：サイトをChrome Frameで表示すると、サンドボックスで動作するレンダリングプロセスとプラグイン、インターネットの安全な閲覧など、 Google Chromeのセキュリティインフラストラクチャ全体が使用されます。 さらに、Chrome FrameはGoogle Chromeのリリースと同時に自動的に更新されます。 ブラウザの90％以上が1週間以内に更新されます。



Internet Explorerを搭載した6台のマシンがあり、Chrome Frameを実装して、イントラネットネットワーク上のいくつかのページ（ エンタープライズイントラネット、約Per。 ）を除くすべてのインターネットページをデフォルトで処理できることを想像してください。 。 何が得られますか：
  • レンダリングプロセスとFlash / pdfプラグインにサンドボックスを使用することによる追加のセキュリティ（ クロムフレームに組み込まれ、約PERで更新されます ）。 実際、ワイルドでワイルドなインターネットを見るとき、Internet Explorerレンダリングシステムから完全に脱却します。
  • 企業ページで作業する場合のInternet Explorerとの互換性。
  • 危険および非常に危険な脆弱性に対する業界最高の回答率。
  • Google Chrome更新システム。 彼女は運転が速くて快適です。
