XtreamerDVDの概要

インターネットからダウンロードした映画の視聴をまだマスターしていないが、DVDから映画に切り替えたい場合は、移行デバイスXtreamerDVDがあります。







興味深いことに、同社はさまざまな2-in-1デバイスを作成しようとしていますが、最近iXtreamerについて書きました。 Ixtrimerと同様に、IxtremeにはSidewinder（すべての形式をサポートするフル機能のメディアプレーヤー）を備えたDVDと、ディスクを読み取るドライブがあります。







注目に値する最初のことは、他のxtremeデバイスと比較して印象的なサイズです。 これは、ドライブ自体とボードのサイズが原因である可能性が最も高く、正確に3をカウントしました（マザーボード、実際には正規のIxtrimer、電源およびカードリーダーです）。







デバイス自体は、ライセンスの要件に従って作成されます。 3つのボタン：再生、停止、トレイの開閉。 そして、それは通常のDVDプレーヤーのように見えます。つまり、テレビスタンドの内部にぴったり収まります。



すでにIxtremera自体について多くのことを話し、デバイス内のDVDは正常に機能するので、今回はデバイスの中心、つまり1073DDチップについて説明します。



製造業者が彼について書いていることは次のとおりです。



機能：

  • HD MPEG1 / 2/4およびHD JPEGデコーダー
  • MPEG2 / 4エンコーダー
  • ネットワーク経由のオーディオおよびビデオストリーミング（有線および無線）
  • CVBS / Sビデオ/ YPbPr / SCART TVエンコーダー
  • I / O I2S、SPDIFをサポート
  • HDMI 1.3サポート
  • USB 2.0サポート
  • SATAサポート
  • ネットワークサポート




可能なアプリケーション：

  • DVD＆HDDプレーヤー
  • メディアプレーヤー
  • 組み込みメディアデバイス


つまり、実際には、ネットワークを介してメディアを操作する場合、どこでもチップを使用できます。 さらに、「最小コストの最大容量」の原則に基づいて設計されています。 たとえば、シグマの直接の競合他社との比較は次のとおりです。

ご覧のとおり、すべてのRealtecは400メガヘルツの周波数で動作します。これは、ネットワークからブロードキャストしながら、HDサウンドとビデオで動作するのに十分すぎるほどです。 クロック周波数が低いため、Ixtrimerは受動冷却で正常に動作します。 たとえば、すべてのデバイスのファンがオフになり、30度の熱でも問題はありませんでした。



このチップのもう1つの優れた機能は、ハードウェアでさまざまな機能をサポートしていることです。 これにより、メーカーは、消費者にデバイスの新しいバージョンを購入させることなく、ファームウェアに常に新しい機能を追加できます。 また、Ixtrimerのほぼすべてのデバイスはこのチップライン上に構築されているため、それらのファームウェアは同期的に更新されます。 繰り返しますが、例：ほぼ5年前に最初のものを購入しましたが、現在は最新のファームウェアを持っています。



レビューの主題に戻って、最後にいくつかの欠点に注目したいと思います。

-プレーヤーにディスクがある場合、DVDモードは自動的にオンになります。さらに、常にディスクを回転させ、不快な背景音を作成します。 この動作を設定で無効にすると、DVDを起動しても自動的に再生されなくなります。 ディスクがないと、デバイスはサイレントに実行されます。

-右パネルのUSBとSDリーダーの非常に奇妙で不便な場所。 デバイスを特別なテレビキャビネットに入れた場合、これらの穴を使用するのは困難です。

-内蔵ハードドライブを配置する方法はありません。 USBのみ。 つまり、Xtreaming（Ixtremereのインターネットサービス）を使用する予定がある場合は、USBフラッシュドライブ、外部ドライブ、またはSDカードをデバイスに挿入し、そこから引き出さないでください。



一般的に、デバイスの印象はポジティブです。 Xtrimerは他のすべてのユーザーと同じように機能します。すべてに接続し、すべてを表示します。 DVDはおなじみのDVDのように動作します。





おわりに





結論として、繰り返します。 デジタルコピーを優先してDVDを放棄したい場合、このデバイスはスムーズな移行を支援します。 または、両方を積極的に使用している場合（たとえば、インターネットへのアクセスが遅く、ダウンロードが必ずしも便利ではない場合、ストールに行く方が安くなります）、Ixtrimer DVDも使用できます。
