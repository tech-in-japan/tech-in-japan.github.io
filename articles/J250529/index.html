登録と統合



サービスへの登録は簡単で、電子メールによる確認が必須です。 無料アカウントでは、サービスを完全に使用することはできません。テスターとしての参加のみが可能です。または、別のチーム（組織のアカウント）の一部として制限なしにサービスを使用することもできます。 サービスを評価するために、試用モードを有効にすることができます-1か月間（さらに、試用期間は最初のアプリケーションのプロファイルが作成された瞬間から開始され、登録日からは開始されません）。 有料アカウントの費用は月額10ドルからです 。詳細については、こちらをご覧ください。 利点のうち、どのタイプの有料アカウントでも、保存されたデータのサイズとアカウントに関連付けられたユーザーの数に制限がないことに注意してください。 アプリケーション数と「所有者」タイプのアカウント数の異なる料金プランの違い（「ビジネス」カテゴリの料金プランのみ）。



アカウント作成プロセスの唯一の希望は、「開発者」オプションを選択した結果を明確にすることです。







デフォルトでは選択されていないため、以下の説明はすでに利用可能な機能のリストとして解釈できます。オプションを選択すると、他の何かが表示されることが予想されます。 実際、デフォルトのアカウントでは、利用可能なアプリケーションのリストの表示（およびもちろんインストール）のみが許可され、デバイスを追加してフィードバックを残すことができます。



また、サービスWebサイトには、旧バージョンのTestFlightを使用してプロジェクトを移行する必要がある開発者向けの特別なガイドがあります（Appleに切り替える前）： http : //support.hockeyapp.net 。



SDKとの統合も非常に簡単ですが、もちろん理想からはほど遠いです。これは現在、 Fabric （ Crashlytics ）が使用しているソリューションです。これについては、レビューの最後の4番目の部分で説明します。
