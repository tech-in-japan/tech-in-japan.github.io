SAPフォーラム2017：モノのインターネットを使用した未来の新しいシナリオ

先週、SAPフォーラムはモスクワで開催されました。これは、CISで最大のSAPイベントで、パートナーと顧客が毎年集まり、コミュニケーションを行います。 今年のフォーラムのテーマはデジタル変換でした。新しいテクノロジーがビジネスと世界をどのように変えるかです。 SAPのパートナーとパートナーは、新しいテクノロジーを使用して、馴染みのある作業プロセスを効果的に変更する方法を教え、示すことにしました。



モノのインターネットとデジタル技術のトピックは、今年のSAPフォーラムで特に目立つようになりました。DigitalNowイベントのスローガンから展示エリアでの多数のデモセッションまでです。 この投稿では、近い将来に見る価値のあるいくつかのソリューションについて説明します。







修理を予測するためのモノのインターネット



モスクワイベントの大きなブースは、インフラストラクチャや物流ソリューションからストレージやデータ処理プラットフォームに至るまで、モノのインターネット向けのサービスと技術のポートフォリオの新しいブランドであるレオナルドに捧げられました。 いくつかのシナリオがSAPフォーラムで発表されました：顔の状態を監視するためのデジタルビット、鉄道の予測修理、掘削機のバケットの認識、故障を防ぐための物事のインターネットの使用、機器監視を備えたインテリジェントな油田、デジタルファーム、コネクテッドカー（コネクテッドカー）。



モノのインターネットを使用した修理の予測は、パートナーの間で最も関連性が高く人気のあるプロジェクトの1つです。 フォーラムは、鉄道の予測シナリオを提示し、SAP Predictive Maintenance and Service、SAP HANA、およびSAP Cloud Platformのセンサーとソリューションを使用して、単一のシステムで車両と機器からデータを収集、保存、処理します。 組み込みのSAPメカニズムは、統計分析を実行し、スケジュールされた修復プログラムを迅速に変更するのに役立ちます。 このソリューションの実装により、顧客は材料のコスト、時間外労働を削減し、資産の生産性と従業員の労働力、および設備の可用性を高めることができます。







現在、このソリューションはイタリアの鉄道で使用されています。 SAPソリューションを使用するTrenitaliaは、機器コストを削減し、乗客のサービスの可用性と信頼性を向上させた動的サービスモデルを開発しました。 Trenitalia用に開発されたシステムは、SAP HANAを使用してセンサーから受信した大量のデータをリアルタイムで処理し、機械学習を使用して予測モデルを作成し、Trenitaliaの従業員が機器の修理計画を調整できるようにします。 同社は、故障の発生を予測する能力とメンテナンスの必要性が、この予算項目のコストを8〜10％削減することを期待しています。これは、トレニタリア規模ではかなりの量です。



倉庫および輸送管理のためのSAP VR Transportation Management VRアプリケーション



SAP Labsの研究開発センターのブースでは、倉庫保管と積載のためのバーチャルリアリティ、ライフサイクル契約、交通管理のための人工知能、状況センターのための地理情報分析など、いくつかの新しいシナリオが紹介されました。 仮想現実のためのSAPソリューションについてもう少し説明します。



SAP VR Transportation Managementは、積載および倉庫管理でVRを使用するための新しいパイロットプロジェクトです。 VRヘルメットでは、スタンドへの訪問者が2つのケースをテストできます。車両への制御のロードと、倉庫の移動を管理し、輸送を追跡するデモアプリケーションです。 このソリューションを使用すると、重量物を積んだ大型トラックの荷積みを簡素化して、車軸の重量配分要件を満たすだけでなく、SAP Cloud PlatformのIoTコンポーネントを使用して、倉庫から荷受人までの貨物を追跡できます。







将来的には、このVRソリューションを使用して、ディスパッチャが計画、倉庫管理、およびローディングの時間を節約できると期待しています。 ロード中に仮想現実を使用すると、オンラインで輸送を追跡するだけでなく、必要な注文のために車内の負荷を正確に分散できます。



このアプリケーションについては、ブログの別の投稿で詳しく説明します。







SAPクラウドプラットフォーム上のスマートシティエネルギーシステム



SAPフォーラムでは、都市のインフラストラクチャのリアルタイム管理を可能にするSAP Cloud Platformに基づいたスマートアーバンエネルギー用のシステムが導入されました。 街の照明を自動的にオン/オフしたり、最適な照明構成を調整したりできます。たとえば、車が動いたときにランプをオンまたはオフにすることができます。 ソリューションの一部として、予測修理システムも実装され、コストを最適化し、機器のメンテナンスを予測しました。 スマートな都市エネルギーのソリューションを実装すると、人的要因が減少します。センサーからのデータはインフラストラクチャ管理システムに直接送られ、オペレーターはすでに受信および分析された情報に基づいて決定を下します。







都市でスマートエネルギーを使用する利点は何ですか？ これには、照明の最適化によるエネルギーコストの削減、インフラストラクチャ管理プロセスの簡素化が含まれます。 また、都市インフラでのモノのインターネットの使用により、事前にメンテナンスを計画し、交通の流れを分散させ、車が混雑している場所（市内中心部、ショッピングセンター、スタジアムなど）の交通量を減らすことができます。

このソリューションを適用した最初の都市の1つは、ドイツのカールスルーエでした。 Youtubeで英語の実装に関するビデオプレゼンテーションを見ることができます。







Smart Store：SAP Cloud Platformに基づいて顧客行動を予測するためのモノのインターネット



今日の小売業の主な質問の1つは、今日の顧客の活動、関心、行動を監視する方法ですか？ 今日、どの商品が大きな需要がありますか？在庫を補充するのはいつですか？ SAPフォーラムでは、新しい「パイロット」、つまりスマート冷蔵庫と棚を備えた「未来の店」を紹介しました。 このケースは、ロシアの飲料および食品の生産者であるAfanasyと共同で開発されました。



このソリューションは、温度、冷蔵庫内の製品の可用性、場所などを監視するために、SAP Cloud Platform、RFIDタグ、特別なリーダーおよびセンサーを使用して開発されました。 このシステムは、会社の分析システムでのデータ処理のための情報収集にも役立ちます。



アタナシウスとのコラボレーションは偶然ではありません。飲料の製造に携わる会社にとって、冷蔵庫の温度を監視することは重要です。 アタナシウス社のプレミアム製品-保存期間が短い生ビールにとって、温度管理体制の順守は特に重要です。 さらに、このソリューションを使用すると、小売部門のパートナーによる協定の実施状況を監視するのに役立ちます。商品が正しく配置されているかどうか、冷蔵庫の充填方法、他の競合製品があるかどうか。 スマート冷蔵庫と棚を使用したパイロットシナリオは、倉庫から店舗の冷蔵庫まで、供給プロセスを自動化するAfanasy社の大規模プロジェクトの一部になります。



モノのインターネットとドローンによるSAP Cloud PlatformとSAP Predictive Analyticsによる油田監視



ロシアの大学との協力は、SAPにとって最も重要な分野の1つです。 University Alliance SAPには多数のロシアの教育機関が含まれており、そのうち3つがSAP Next Gen Labイノベーションセンターを開設しています。 これらの研究所では、学生と教師はSAPの教材を使用したり、独自のプロジェクトを開発したりできます（SAPのお客様向けを含む）。 SAP Next Genラボでは、炭化水素の生産と輸送を制御するために、グブキンロシア州立石油ガス大学に基づいて共同プロジェクトが開発されました。



シナリオには、オペレーターが配置されている制御室の使用、および生産の視覚的監視と機器の状態の評価のための無人機の使用が含まれます。 このシナリオでは、オペレーターは、センサーを使用して、機器の状態と生産プロセスに関する情報を受け取ります。 受け取った情報に基づいて、彼らは予測を立て、緊急事態の決定を下し、経済効果を評価することができます。



ドローンは、デジタル形式で情報を提供できない機器に関する視覚情報を収集するために使用できます。 カメラを使用して、無人車両はコンピュータービジョンテクノロジーを使用してデジタルバージョンに処理される情報を収集します。 詳細情報は、対応する機器と時間タグとともにデータベースに転送されます。 同時に、オペレータとマスターに決定を下すための追加情報を提供するために、リアルタイムのグラフィック情報がドローンによって送信されます。 ソリューションを開発する際、SAP Cloud Platform、SAP Predictive Analytics、およびSAP Cloud Platform Connectivityのサービスを使用しました。



これらは、SAPフォーラムで提示された未来と物事のインターネットに関する注目に値するいくつかのシナリオです。 イベントWebサイトの本会議の記録はすでにご覧いただけます。近い将来、テーマ別セッションのパートナー、顧客、SAPスペシャリストのプレゼンテーションも予定されています。
