Russian Startups Go Global 2017-ロシアのIT起業家にとっての新しいグローバルな機会





みなさんこんにちは！



11月29日に、国際市場での開発とスケーリングの可能性を持つIT起業家向けのロシアのStartups Go Global 2017カンファレンスを開催します。 この投稿では、昨年のイベントがどのように進んだか、ロシアのStartups Go Global 2017プログラムをブロックしているもの、そしてイベントで聴ける外国人ゲストについて説明します。



2016年のように



昨年、Russian Startups Go Global 2016を開催しました。これは、新しい地平線を征服したいロシアのITファウンダー向けのIIDFの最初の国際イベントです。 500のスタートアップ、Seedcamp、Cherry Ventures、Maxfield Capital、Ideal Machineのゲスト、英国、アイルランド、フランス、ドイツの大使館の代表者が訪問しました。 。 以前のように、ビデオを見ることができます：







その後、多くのフィードバックを得ました。







私たちは起業家にとってこのイベントの価値を見ました-そして今年、国際市場で開発の機会を持つロシアのITスタートアップを紹介し続けることを決めました。 Russian Startups Go Global 2017で彼らの会社のグローバルな発展に興味があるすべての人を待っています！ イベントは11月29日にモスクワのイズベスチヤホールで開催され 、登録して来てください！







2017年はどうですか



会議のメインプログラムは、 「ビジネス」、「テクノロジー」、「資金調達とM＆A」 の3つのブロックで構成されています。 ロシアのスタートアップゴーグローバルのスピーカーは、製品を外国市場に適応させる方法、ロシアとアメリカでのB2Bの売り上げの違い、投資家と協力して会社の資本を増やす戦略を構築する方法、海外投資を誘致する方法、および実務から事例を共有する方法について話します。 それらの中には：



Amrit Dhir、Google for Entrepreneursのグローバルキャンパスオペレーションマネージャー



アムリットは起業家のためにGoogleのグローバルオペレーションを指揮し、ネットワークの拡大を調整します-現在、キャンパスベルリンの立ち上げの準備をしています。 彼のチームは、世界中のスタートアップハブ（ロンドン、マドリード、サンパウロ、ソウル、テルアビブ、ワルシャワ）で、キャンパスの技術的および物理的な運用と起業家向けのGoogleサイトのネットワークを管理しています。








アレクサンダー・ゴルニー、Mail.Ru Group、戦略および分析部長



彼はプログラマーからMail.Ruメール部門の責任者になりました。 2006年3月に、彼はWebプロジェクト開発のディレクターとしてRambler Internet Holding Companyに参加しました。 2011年、彼はMail.Ruグループの情報技術部長、2016年に戦略および分析部長に任命されました。 この役職では、M＆A、投資、社内での新しいプロジェクトの立ち上げに重点を置いています。 彼はブログ＃startupsを持っています-スタートアップとそのビジネスモデルの著者レビュー。








Marvin Liao、ベンチャー基金のパートナー、500スタートアップのアクセラレータ



マービンは、デジタルメディア、アドテック、マーケティングクラウド、eコマース、モバイルの分野で世界中の新興企業に投資しています。 10年以上にわたり、マービンはYahoo!のさまざまな部門で働いていました。 Inc. -主にアジア、ヨーロッパ、ラテンアメリカ、アメリカの地域またはグローバル管理レベルでの販売、事業開発、広告、マーケティング。 彼は、ヨーロッパ、イスラエル、米国のいくつかの加速プログラムの指導者であり、新興企業のマーケティング、販売、パートナーシップの開発を支援しています。










デジタルダオチャイナマネージングディレクターライアンペン



Ryanは、ITおよびデジタルの分野における中国の専門家であり起業家です。 中国のデジタル、クラウドコンピューティング、SaaSに強い専門知識を持つ彼のプロジェクトの1つは、Sequoia Capitalからの投資を引き付けました。 Ryanは8年間、SAPのシニアプロダクトマネージャーとして働いていました。SAPは、世界中のエンタープライズセクター向けのソフトウェアの最大の開発者です。










ヴァレリー・プルニー、EDFのオープンイノベーションディレクター



ValerieはNugenia Associationの共同設立者であり、Nugenia Associationは核分裂技術の研究開発に従事しており、28か国に110人以上のメンバーがいます。 彼は、ヨーロッパおよびシリコンバレーのパートナーと共に、ビジネスイノベーションの促進を支援する協会を設立しました。 現在、ヴァレリーはパリに住んで働いており、ヨーロッパと中東のオープンイノベーションディレクターの地位にあり、EDFグループとスタートアップの間で相互に有益な協力関係を築いています。










ピーター・グリーン、共和国の共同設立者、米国の大手クラウドファンディング投資サイト



同社では、ピーターは共和国製品とクラウドファンディングのコーチングスタートアップを担当しています。 共和国は、AngelListおよびCoinListとともに投資サイトの主要グループの一部です。

彼はIT分野で15年以上働いており、製品開発、ブランディング、資金調達に関する多くの新興企業の指導者および顧問です。 カリフォルニアでは、成功したSaaS企業であるMeetingPulse、ファッションブランドVolga Verdi、＃VVFYアプリ、Mine networkを設立しました。 以前は、GM、P＆G、火星、Sberbank、マクドナルドなどのプロジェクトで、Leo Burnett Moscowのインタラクティブデザイン部門で働いていました。








ID Financeの共同設立者兼CEO、Boris Batin



2003年9月から2006年7月まで、彼はロンドンのドイツ銀行グループの資本市場部門の副所長を務めました。 2006年から2010年まで、彼はロシアの負債資本市場の副社長、モスクワのルネサンス首都のCIS諸国でした。 2010年から2012年まで、ロシアの債務資本市場およびモスクワのスコットランド王立銀行のCIS諸国の副社長を務めました。 2012年、Boris Batinはロシアで最初のオンライン融資会社ID Financeを設立しました。










Laurent Ellerbach、マイクロソフトの中央および東ヨーロッパの技術福音主義者のチームリーダー



ローランは、中央および東ヨーロッパの技術的福音主義チームのリーダーです。つまり、イベントやオンラインを通じて技術専門の開発者や学生を巻き込み、コードを共同で作成します。 学生として、彼はマイクロソフトに来ました。そこで、彼はすでに20年間さまざまな役職で働いていました。 ここ数年、彼はスタートアップの技術やビジネスの問題を積極的に支援してきました。










Brent Cooper、Moves The Needleの共同設立者、The Lean Entrepreneurの著者



The Lean Entrepreneur、The Entrepreneur's Guide to Customer Development、およびThe Lean Brandの著者。 ブレントは、スタートアップがアイデアから急速な成長、企業の売却、IPOへの進出を支援し、起業家が自分の経験から犯す典型的なミスをすべて知っています。 Moves the Needleの創始者である彼は、企業が革新的な仕事を立ち上げ、スタートアップの創始者の実践からテクニックを紹介するのを支援しています。










Alexey Pelevin、法的参照システムおよび情報ポータルPravo.ruのCEOおよび創設者



2000年に、彼はデザイナーとしてArtemy Lebedevのデザインスタジオで働いていました。 2002年にはパークデザインスタジオを組織し、2008年には法的参照システムとPravo.ru情報ポータルの作成と立ち上げに参加しました。 2009年、Pravo.ruは仲裁裁判所のサイトのサービスで州の入札を獲得し、2010年以降、同社はElectronic Justiceサービスを使用してロシアの仲裁システムを自動化しています。 2012年から2013年に、Casebook、Caselook、Case.Proの法的活動を自動化するシステムがオープンしました。

さて、会議の最も実用的な部分の1つは、外国市場への参入経験のある創業者の事例です。 昨年、会議のこのブロックでは、参加者から多くの肯定的なフィードバックが集まりました。 ビデオのストーリーを聞くことができます。



今年、IIDFは、製品を適応させ、最初の顧客を獲得し、米国市場での成長を加速するための国際的なアクセラレータであるTechmafiaを立ち上げました。 そのため、ConveadのCEO、Stafory開発ディレクターであり、Kroema Vladimir BugaiのCEOであるRobot Vera Aleksey Kostarevの共同設立者であるAleksey Kovalevが参加者を招待して、州での最初の販売とパイロットの事例を共有しました。
