SDS-TWR巻尺と同期のない距離測定



このトピックでは、人々やオブジェクトを屋内（領域）に配置する（位置を特定する）新しい技術に特化したシリーズを続けます。 以前に公開されたもの： http : //habrahabr.ru/post/151496/およびhttp://habrahabr.ru/post/153237/



RTLSシステムでの位置決めは、マークから3つ以上のアンカーまでの距離を測定し、その後にマークの座標を計算することで実行されます。 また、SDS-TWR方式（対称双方向双方向距離測定）を使用すると、高価な同期に頼ることなく、必要な精度を提供できます。

この方法は、ISO / IEC 24730-5およびIEEE 802.15.4-2011によって規制されています。

RTLSシステムでは、距離はマークとアンカー-互いに同期されていない2つのワイヤレスデバイスの間で測定されます。 測定は、パケットを交換するプロセスで実行されます：要求-応答。





測定中のパケット交換のシーケンスを図に示します。







必要な精度を確保するために、2つの対称的な測定サイクルが実行されます。 最初はラベルによって開始され、2番目はアンカーによって開始されます。

最初のサイクルでは、ラベルは要求パケットをアンカーに送信し、同時に送信の瞬間を検出します。 パッケージは、無線信号がマークからアンカーまでの距離を飛行するのに必要な時間間隔（飛行時間）後にアンカーに到達します。 実際、距離を決定するために測定する必要があるのは今回です。 アンカーは、パケットが到着した瞬間を検出し、内部クロックに従って、この瞬間から標準時間（応答時間（標準で最大5ミリ秒））をカウントし、確認パケットをラベルに送信します。 飛行時間の経過後、確認がマークに到達し、マークが独自のクロックによって、リクエストが送信されてから確認が受信されるまでのサイクル時間が決定されます。 サイクル時間は、飛行時間の2倍とアンカーの応答時間で構成されます。



これは、マークとアンカー間の距離を決定するのに十分であると思われます。サイクル時間から応答時間を引き、それを半分に分割します。 これは事実ですが、そのような測定の精度は低くなります。 実際には、結果にはマークのクロックの誤差（クォーツ）に加えて、アンカーのクロックの誤差（クォーツ）が含まれます。 ISO / IEC 24730-5は、クロックの許容誤差を80 ppm（100万分の1）に制限しています。 応答時間が5ミリ秒の場合、測定誤差は200 nsになることは簡単に計算できます。 （詳細な計算はこちら： http : //www.rtlsnet.ru/technology/view/2 ）。 このエラーに無線信号の伝搬速度（約3億メートル/秒または0.3 m / ns）を掛けると、60 mのエラーが発生します。応答時間を短縮するか、クォーツの精度を改善することをお勧めします。 最初のケースでは、パケットの長さ、およびそれに応じてペイロードを減らす必要があります。 第二に、デバイスの価格を上げることは不合理です。 クォーツの価格は精度が上がると急速に上がります。



解決策は、最初の測定サイクルと対称的な2番目の測定サイクルを使用することです。

アンカーは、一定の遅延時間待機した後、ラベルに要求パケットを送信します。 また、パケットペイロードで同時に、最初のサイクルの応答時間をラベルに伝えます。 このサイクルでは、タグはすでにクロックで応答時間をカウントしており、アンカーは独自のサイクルで2番目のサイクルの時間を測定し、結果をタグに送信します。

パケット交換が完了すると、ラベルには自由に使える4つの値があります。ラベルによって測定される応答時間とサイクル、およびアンカーによって測定される応答時間とサイクルです。 ラベルはこれらの値をサーバーに送信して距離を計算します。

サーバーは、タグのサイクル時間とアンカーのサイクル時間を要約し、タグの応答時間とアンカーの応答時間を減算し、結果を4で除算します。

すべてが最初のケースと同じように思えます。 ただし、一方の方向の測定誤差は、反対方向の測定誤差によってほとんど補償され、マークとアンカーの応答時間の違いのみに影響します。 （詳細： http ://www.rtlsnet.ru/technology/view/2）そして、ISO / IEC 24730-5に準拠したこの差は1マイクロ秒を超えてはならないため、最大測定誤差は40 psに等しくなります。 または距離の面で-12 mm。

当然、分散遅延線、マルチパス減衰、およびその他の要因により追加のエラーが発生しますが、これらはすでに実装上の問題です。 したがって、このようなシステムおよびチップの世界的メーカーであるNanotron Technologiesは、2メートルの部屋での位置決め精度を提供します。MEMSタグに組み込まれた加速度計の読み取り値を使用すると、精度を1メートルに上げることができます。
