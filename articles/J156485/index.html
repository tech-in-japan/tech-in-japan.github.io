Windows 8には「神モード」もあります

以前、 Windows 7では、いわゆる「神モード」 （単一のポイントから多くのシステム設定への便利なアクセスを提供する特別な機能） が発見されました 。 Windows 8では、この機能は保存されています。 有効にする方法を思い出させてください。



  1. デスクトップに新しいフォルダーを作成します。
  2. このフォルダーに次の名前を設定します： GodMode。{ED7BA470-8E54-465E-825C-99712043E01C}





    その後、フォルダアイコンはコントロールパネルアイコンの形になります。





    フォルダ内には、設定へのリンクが意味別にグループ化されて表示されます。
