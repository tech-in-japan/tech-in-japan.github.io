Windows Phone 8のEvernoteアップデート：写真のバースト、音声からテキストへの変換など





Evernote for Windows Phoneの改善に引き続き取り組んでいます。 本日リリースされたバージョン4.0には、バーストシューティング、メモ内の画像の表示の改善、音声からテキストへの変換のサポート、メモ内のキャプションの自動生成、Windows Phone 8のサポートなど、多くの新機能が搭載されています。



Evernote for Windows Phoneをダウンロードする



バースト写真



メモごとに複数の写真を撮る必要がある場合があります。 このアップデートでは、新しいバースト機能が追加され、撮影モードを終了せずに複数の写真を撮影できるようになりました。 ワンショットを撮り、画面をタップして次のショットを追加します。 右上隅にカウンターが表示され、撮影した写真の総数が表示されます。 キャプチャされた素材を表示し、不要なフレームを削除するには、このカウンターをクリックし、表示が完了したら、保存アイコンをクリックします。



スライドショー



アプリケーションでバースト撮影が行われたので、メモ内の複数の画像を表示する方が便利かどうかを考える必要がありました。 そのため、スライドショーモードを追加しました。 この表示モードに切り替えるには、メモを読みながら写真をクリックするだけです。 画像は全画面で拡大されます。画像を裏返すと、メモで残りの写真を見ることができます。



メモの書き方



スクリーンキーボードで入力するよりも、いくつかの文を言う方が簡単な場合があります。 そのため、アプリケーションで音声をテキストに変換する機能を追加しました。 メモを開いて、コミックの「バブル」が付いたアイコンをクリックし、口述を開始します。 録音が終了するとすぐに、Evernoteは上記の処理を行い、元のオーディオ録音とともにノートに表示されるテキスト復号化を行います。 これにより時間を節約できるだけでなく、テキストが検索用にインデックス付けされるため、今後このメモを見つけやすくなります。 この機能は、簡単なメモ、講義、インタビューに最適です。







自動ヘッダー



アカウントに無名のメモが大量に入力されるのを避けるために、Evernoteは自動見出し生成を提供します。 アプリケーションは、現在地、メモの内容、さらにはカレンダーに関するデータを使用して見出しを作成し、その下に隠されているメモをすばやく理解できるようにします。 たとえば、会議中にメモを取ると、カレンダーのこのイベントの名前がタイトルになります。 この時点でシステムが予定を検出しない場合、名前に含まれる都市が追加されます。 もちろん、その後、タイトルを変更できますが、主なことは、アカウントのコンテンツを表示するときに、名前のない多くの未確認のメモをスクロールする必要がなくなることです。



その他の改善



説明されている機能に加えて、アプリケーションは作業中に起動時より高速になりました。 より速く動作します。 さらに、Windows Phone 8のサポートを追加しました。更新されたEvernoteをお試しください。
