商標と商標を区別する方法

フィードバックは素晴らしいです。 あるものについての記事を書いて、コメントですぐに別のものと第三のものについて話すように求められます。 そのようなことは商標で起こりました。

このテーマは、私たちの生活と仕事のいくつかの分野、デザイン、広告、法律と密接に関連しています。 後者は、おそらく、絶対にあらゆるトピックと絡み合っていますが、誰もがそれについて考えているわけではありません。 独自の法的な鐘楼の商標（ロゴ、ブランド、商標）を検討します。







ミスターは誰ですか プーチンの商標？



すぐに決めましょう、ロシアの法律は商標、ロゴまたはブランドなどの用語を知りません。 法律については、商標のみがあります-法人または個人の起業家の商品を個別化するのに役立つ指定

この定義から、商標は個々の起業家と法人にのみ属することができます。 個人が商標を取得する権利がない場合があります。



ロゴがあります-それは商標ですか？



あなたがロゴを持っているという事実とあなたが商標を持っているという事実の間にはいくらかの違いがあります。



商標は 、Rospatentによる強制登録の対象となります。 そして、証明書を受け取った後にのみ、あなたはそれに対する権利を持ちます。



ロゴは著作権の対象であり（作成者は作成時点から登録されていません）、独占的権利の譲渡に関して著作者と合意した場合にのみ、このロゴに対する権利を所有します。

たとえば、非常に美しいロゴがあり、それに対するすべての権利が受け取られ、それがあなたのウェブサイト、名刺、レターヘッドに掲示されます。 しかし、これは商標ではありません。 ただし、このロゴをRospatentに登録した場合は、すでに商標があります。







どんな商標がありますか？



最も一般的なのは3つのタイプです。

  • 口頭（例：Johnson＆Johnson）。
  • 罰金（たとえば、有名なリンゴ）
  • 組み合わせ：単語+画像


これらは、いわば、常に使用および登録されている従来のタイプの商標です。

しかし、もっとエキゾチックなものがあります（あまり一般的ではありません）：

  • 容積測定（例：コカ・コーラのボトル形状）。
  • サウンド（軽いZippoをクリックします）。
  • 嗅覚および味覚。

登録できないものは何ですか？



もちろん、登録することは不可能です。 TKの登録を拒否される可能性がある特定の制限があります。たとえば、次のとおりです。



1. TKは、すでに登録されている商標と同じか、紛らわしいほど類似しています。

たとえば、ラテン文字で書かれた「Deti」などの商標を登録する場合、キリル文字で書かれた登録商標「Children」を既に持っている場合、これを行うことはできません。 この場合、Rospatentは、いわゆる混乱の度合い、つまりこれらの商標の統一性を確立して、おそらく拒否します。



2. TKには特有の能力がないか、要素のみで構成されています。

-特定の種類の商品を指定するために一般的に使用されます。

-一般的に受け入れられている記号および用語。

-商品の特性、種類、品質、量、特性、目的、価値、生産またはマーケティングの時間、場所、方法を示すものを含む。

-商品の形態を表します。これは、商品の財産または目的によってのみまたは主に決定されます。



たとえば、宅配会社は宅配会社に登録できません。 また、原則として、数字に特別な書き込みや結合の方法がない場合、番号を登録することはできません。

または、たとえば、Rospatentは、「自動車」という言葉は自動車に関連する商品の一般的な略語であると考えたため、Auto Ru商標の登録を拒否しました。



3.以下を 表す要素のみで構成されます。状態エンブレム、フラグ、その他の状態記号と標識。 国際機関および政府間機関の略称または正式名称、その紋章、旗、その他の記号お​​よび標識。



たとえば、国連、ユネスコ、国際オリンピック委員会、世界保健機関、WIPOなど。これらの要素は、保護されていない要素としてのみ商標に含まれる場合があり、関連する管轄当局またはその所有者の同意が得られた場合のみです。



4. 違反者は一般に、国際基準、道徳、国家の伝統および文化を受け入れました。

たとえば、Rospatentがアルコール飲料に関するTikhiy Don商標の登録を拒否した場合、紛争は法廷で検討されました。 Rospatentの代表者は、ショロホフの仕事は国宝であると考え、アルコール飲料に関するマークとして登録することは倫理、道徳に違反する可能性があり、これは国宝です。



それにも関わらず、法廷では、出願人は「Quiet Don」がすばらしい仕事であることを証明することができましたが、アルコール製品に関する商標の登録は道徳基準に決して違反しません。 同時に、もちろん、決定の瞬間は道徳と倫理の1つまたは別の商標に違反し、事件が法廷に来た場合、Rospatentの専門家と裁判官の両方の評価判断です。



5.商標には、 姓など、より注意が必要な、いわゆる特別なオブジェクトが含まれる場合があります。 コンサルティング会社の場合、創業者または管理パートナーの名前に含まれる名前はかなり一般的なものです。 ただし、姓自体が商標として登録されることはほとんどありません。



そのような登録の可能性を評価するために、Rospatentは、出願時の姓に消費者にとって顕著な特徴がある場合にのみ法的保護が提供されるべきであるという事実から進んでいます。 消費者があなたの姓だけでなく、あなたの商品やサービスにこの姓を関連付けていることをアプリケーションの資料で証明すると、そのような商標にはほとんどの場合法的保護が付与されます。



しかし、Rospatentは次のことも分析します。ロシア語と外国語の姓は、一般に受け入れられている記号または用語と同一であり、再生姓の指定はロシア連邦の文化と歴史の財産ですか？ たとえば、ドストエフスキー、プーシキンという名前は、ほとんど商標として登録できません。



6.もう1つの特別な主題は地理的名称であり、原則として商標として登録することはできません。 この場合、地理名は、海、極、赤道など、地理に関連する一般的な概念によって区別する必要があります。 彼らにとって、登録の禁止は存在しません。



さらに、地理的概念を登録する際、いくつかの微妙な点があります。 商標として、地図上ではほとんど知られていない、または存在しない地域の名前、居住地を登録できるとしましょう。



アルコール飲料の商標として登録された商標「Gzhel」の例を思い出すことができ、その後、商標の登録が争われました。 しかし、裁判所は、消費者との紛争時の「Gzhel」という名前はすでにアルコール飲料と密接に関連していると判断しました。 さらに、Gzhelの町はほとんど知られていないため、商標の法的保護は保持されています。



したがって、Rospatentは、アプリケーションをチェックするときに、ブランドがすでに知られているかどうか、およびその名前が付いている地域の有名さを分析します。



実行する場所、登録する場所



ロシアで商標を登録することに決めた場合は、知っておくべきです。Rospatentは登録を実行します。 ただし、このプロセスは非常に時間がかかり、12〜15か月かかるため、事前に準備してください（法律ではタイムラインが指定されていませんが、実際にはこれは非常にまれです）。







そして、ここに待望のステップバイステップの説明があります：



1. ICGSのクラスの選択。

マークの登録に関する商品とサービスの国際分類は、商品とサービスのタイプに分類された一種のディレクトリです。 TKを登録する前に、TKを保護するクラスを選択する必要があります。

クラスのリストを表示して、 www.mktu.infoから適切なクラスを選択できます。



選択する場合、異なるクラスで同様のTKの登録が許可されることを覚えておく価値があります。 したがって、可能な限り競合他社の攻撃から身を守るために、ICGSのいくつかのクラスを選択することは理にかなっています。



2.予備チェック。

Rospatentデータベースを介して、類似の（お客様の商標と同様の）商標が既に登録されているかどうか、およびそのようなマークの登録申請があるかどうかを確認する必要があります。



Rospatent Webサイトには登録商標のオープンデータベースがありますが、アプリケーションがそこに含まれていないため、重大なエラーが発生します。 実際、TKの保護は、アプリケーションを提出した瞬間から始まります。 そして、2番目に適用する場合、明らかに優先順位はありません。



検証は弁護士に委託するか、Rospatentに申請書を提出し、料金を支払うことで独立して行うことができます。 約1か月かかります-登録の合計期間を背景にした瞬間ですよね？



3.アプリケーションの提出

ロシア連邦民法第1492条の要件に従って、商標の申請には次のものを含める必要があります。

  • 申請者とその居住地または場所を示す商標としての指定の状態登録の申請;
  • 要求された指定（実際には、ロゴの写真またはそこにあるもの）;
  • 商標の州登録が要求され、ICGSのクラスごとにグループ化されている商品および/またはサービスのリスト。
  • クレームされた指定の説明（ロゴの詳細な口頭説明）。




フォーム自体は部門によって厳密に規制されているため、手書きのフリーフォームで書かれた紙は機能しません。 州税の支払いに関する文書も申請書に添付されています。 サイズを計算するには、RospatentのWebサイト（ www1.fips.ru/wps/wcm/connect/content_ru/ru/poshl_s/k_p）の計算機を使用できます。



Rospatentは、申請の提出後、次のような検証作業を実施します。

  1. アプリケーションの受信と登録;
  2. 提出された申請に関する情報のRospatentの公式速報（以下、公式速報と呼ぶ）での公開。
  3. 料金の支払いの確認と申請の正式な審査;
  4. 商標としてクレームされた指定の審査（以下、クレームされた指定の審査）。
  5. 申請者が自身のイニシアチブで提出した申請および請願の検討。
  6. そして最後に、最終-州の登録簿への商標の登録、商標の州の登録に関する情報の公開、および商標の証明書の発行。


申請を検討する過程で、Rospatentが登録漏れを見つけたり、商標を登録できないと結論付けたりした場合、申請者にこれを通知し、その観点の配布と提示の時間を与えます。



商標登録は競争であなたのビジネスを保護するだけでなく、投資家にとって魅力を高めることを覚えておく価値があります。

商標登録の有効期間は永遠ではなく、わずか10年です。 TKの保護期間は延長することができ、時期尚早に終了することもできます。 これは法廷でのみ行うことができます。 したがって、商標の使用方法と、保護期間が満了していないかどうかを注意深く監視してください。
