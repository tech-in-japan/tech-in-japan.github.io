Nokia Firmware Cooker-5800のファームウェアのデザイナー

彼はそのようなプログラムのレビューを5530で書くことを約束しましたが、それへのリンクはひそかに消えました。 私が著者に連絡するまで、これは陰謀のままにしておきましょう。 それまでの間、会いましょう：5800のファームウェアエディター！




Nokia Firmware Cookerは、携帯電話を最適化および変更したい人のために作成されたプログラムです。 このプログラムの助けを借りて、5800（およびそれ以上！）用の独自のパーソナルファームウェアを構築できます。 残念ながら、このようなよく開発されたプログラムは、これまで5800のみで記述されていましたが、記述の過程ですでに5530のファームウェアの設計者であり、他の電話ではまったく見られませんでした。 しかし、私は5800の所有者が5530以上であることを願っています。

カットの下：スクリーンショット、指示、説明、リンク。



プログラムの最後の更新は、新しいファームウェアのリリースにより、4月27日に実行されました。 原則として、指示以外にここで伝えることはありません。スクリーンショットには十分な情報があります。







メイン設定ウィンドウ。 ここでは、以前に5530に関するトピックで説明したすべての設定を確認できます。







そして、必要な言語を選択する機会があります。これは最後のトピックで尋ねられました。







マイナー設定と外観。







メッセージをカスタマイズします。







必要な速度設定を設定します：感度、精度、速度など。







電話の自動読み込みを編集します（多くのチェックマークを削除する前に、メニューまたはウォッチを失うため、開始のためにそれぞれをグーグルで検索することをお勧めします）







そして、これは最も必要なウィンドウの1つです。電話の速度を調整します。 そして、碑文に注意を払います。 「よくわからない場合は、ここには何も触れないでください。 パラメーターが正しくないと、深刻な結果を招く可能性があります。」







そして今、ささいなことのために-スキャンするフォルダを選択し、メニューから不要なアイテムを非表示にします。



指示 ファームウェアを構築するには、次のものが必要です。




*オリジナルのファームウェアをダウンロードします（クリーンなファームウェアを変更することをお勧めします。また、プログラムは実際には他でテストされていません）。 ここでファームウェアを取得します

* Nokia Editor / Nokia Firmware Editorを使用してROFS2を解凍します（プログラムは終了しません！）

* Nokia Firmware Cookerプログラムで、「フォルダーを開く」をクリックし、前の段落で取得したrofs2フォルダーを選択します

*必要な変更を加えます。

* [変更]ボタンをクリックします

* Nokia Editor / Nokia Firmware Editorを使用してファームウェアをパックします。 デザイナーの作者は、 NokiaEditor Beta v.0.7を使用してファームウェアを再パックすることをお勧めします。



コンストラクター自体をダウンロードする



プログラムの作成者は、プログラムでグリッチ、フォールなどが発生する可能性があることを警告しています。 彼は自分でプログラムをテストし、それをダウンロードした人は全員テスターです。



私たちは議論し、提案を書きます-私はそれを著者に引き渡します。
