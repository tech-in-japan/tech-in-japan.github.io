Veeam Availability Suite 9.5アップデート：知っておくべきこと

Veeam は 、Veeam Availability Suite 9.5 Update 3の新しいアップデートをリリースしました 。これにより、顧客は仮想 、物理、およびマルチクラウド環境でデータを一元管理できます。



VeeamのPeter McKay社長は、 VAS 9.5 U3アップデートを10年の歴史の中で最大のリリースと呼びました 。 Peter氏によると、このアップデートにより、あらゆるデータとアプリケーションの高レベルの可用性が実現します。



アップデートの機能について詳しくはこちらをご覧ください。





/写真FutUndBeidl CC



このアップデートでは多くの新機能が導入され、さらに-最も重要な機能の1つが導入されました。



Microsoft Windows 2.1およびLinux v2のネイティブVeeamエージェントのサポート


ユーザーは 、コントロールパネルを使用してVeeam Agentを一元的に展開できます。 Veeam Agent for Microsoft Windows 2.1には、Microsoft Windows Serverのフェールオーバークラスターの保護が含まれており、Veeam Agent for Linux v2は、 Veeam Cloud Connectリポジトリにバックアップすることができました。



Veeam Cloud Connectのインサイダーサイバー攻撃からデータを保護する


Veeam Cloud Connectのサービスプロバイダーは、Veeam Backup＆Replicationコンソールから直接または誤って情報を削除することから顧客データを保護することができます。 新しいVeeam Backup＆Replication機能については、 こちらをご覧ください 。



IBM Spectrum Virtualize Integration


ストレージスナップショットを操作するための新しいソリューションは、システムの復元に必要な時間を短縮します。 IBM Storwize、Lenovo Storage Vシリーズ、IBM / Cisco VersaStackなど、IBM Spectrum Virtualizeベースのストレージシステムで利用可能です。



Veeam ONEの新しい監視オプション


GDPR要件など、「 データ主権 」への準拠をチェックする機能が登場しました 。 このため、本番およびバックアップインフラストラクチャにはタグがタグ付けされています。 これで、データの主権に違反するデータリカバリを実行しようとすると、ユーザーに警告が表示されます。 操作が確認されると、システムは制御分析を実行します。



ユニバーサルストレージ統合API


Veeam Backup＆Replicationと統合するために顧客が[会社の]パートナーを選択できるようにするフレームワーク。
