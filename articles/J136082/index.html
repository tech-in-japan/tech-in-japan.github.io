Asus Zenbook UX31 Ultrabookを確認する



私は長い間、ノートパソコンの購入について考えていました。 しかし、私は常に適切なモデルを見つけることができませんでした。 重量が大きい、画面が小さい、またはプロセッサが弱い。 しかし、その後、奇跡が起こり、ラップトップの新しいブランチであるウルトラブックについて知りました。 そしておそらく、このファミリーを導入する最も有名なものは、アップルの発案後、Asus Zenbookです。

この注目に値するデバイスのレビューが面白い人には、タックルをお願いします。 注意：たくさんの写真。





復習



性能、重量、価格の基準からこのデバイスを選びました。 現在、非常に多くのウルトラブックが市場に出回っています。 しかし、私はこれを選択しました。画面解像度のため、1366x768では十分ではありませんが、Asusには1600x900があり、間違いなく素晴らしいです。

仕様は次のとおりです。

  • モデル：Intel Core i5 / Core i7
  • RAM：4096（DDR3-1333）
  • データストレージ：128/256 GB SSD
  • ビデオ：Intel HD Graphics 3000
  • 対角（インチ）：13.3
  • 解像度：1600x900
  • ネットワーク：Wi-Fi（b / g / n）、Bluetooth
  • ウェブカメラ：0.3 MP
  • ポートHDMI、VGA、USB 2.0、USB 3.0、ヘッドフォンおよびマイクジャック、SDカードリーダー
  • バッテリー：リチウムイオン
  • 幅（cm）：32.5
  • 奥行き（cm）：22.3
  • 高さ（cm）：0.3-1.7
  • 重量（kg）：1.3


特性からわかるように、これは優れた特性を備えた非常に薄く軽量なデバイスです。

本当に何ですか？

最初から始めましょう。

ウルトラブックボックスは非常にきれいに作られていて、目の前に何か異常なものがあることはすぐにわかりました。



箱に入っていたのは、ウルトラブック自体、充電器、電源アダプター、ケーブルタイ、ドキュメント、ミニVGA-VGAアダプター、Enternet用アダプター、ウルトラブックケース、アダプターケースだけでした。

課金と書類なしのすべてがここにあります：



充電器はコンパクトで、旅行中は間違いなく良いです。



キットにカバーが付いていることを特に嬉しく思います。 生地と革でできています。 ウルトラブックは完全にその中にあり、バッグに入れて持ち運ぶのに非常に便利です。



また、アダプター用のケースが付属しており、フラッシュドライブやマウス用アダプターに個人的に使用することも書きました。紛失せずに単一のスタイルが得られるのは便利です。



ウルトラブックカバーは片手で非常に簡単に開くため、下部を保持する必要はありません。 これが私たちのヒーローです。



ZenBookキーボードはとても便利です。 キーが大きい。 多くの人がそれが強く押されていると書きました-私はそれを反論することができます、すべては簡単に行われます。 多分それはウルトラブック自体に依存します。 電源ボタンの位置も異常です。 それは不便を引き起こすはずだった右上隅にあります。 しかし、削除キーに隣接しているにもかかわらず、偶然クリックしたことはありません。



デバイス自体は非常に高品質で頑丈な金属でできており、何も再生せず、非常に静かです。

そして最も重要なことは、非常に薄い体です。

キーボードを除く本体全体は金属製です。

側面からわかるように、USBが2つあり、そのうちの1つは3.0、カードリーダー、ミニHDMI、ミニVGA、ヘッドフォン/マイクポート、充電ポートです。

ハードドライブのインジケータもあります。

後ろには何もありませんが、ラジエーターグリルの下に見えます。 ちなみに、ウルトラブックは、ゲームなどの特定の負荷の下でのみ加熱されます。 ビデオ、オフィス、インターネットはスチールと同じくらい寒いです。



Bang＆Olufsenが作ったスピーカーのウルトラブック。 はい、彼は良いですが、低音は弱いです。 そのようなサイズの場合-優れた結果。



ウルトラブックのタッチパッドは大きく、反応がよく、マルチタッチをサポートしています。



ウルトラブックスタッフィングについて少し説明します。ソリッドステートドライブ、4 GBのRAM、Intel Core i5（i7のモデルはまだあります）があるとだけ言います。 Call of Duty Modern Warfare 3のようなおもちゃでさえ、「ラグ」なしですべてが機能しますが、もちろん最大設定ではありません。



ウルトラブックのバッテリーは取り外しできませんが、省エネモードで6時間安定しています。
