OS X LionおよびiMessage

10.7.2でiMessageサポートが表示されなかったのはなぜですか？





OS Xでのサポートの導入は、iMessageを前進させるための素晴らしいステップです。 結局のところ、これにはすべてがあります：

-iCloudには共有アドレス帳があります

-優れたiChat IMクライアントがあります-iMessageプロトコルを追加します。



愛好家はiChatのコードを発見しました：

インターフェース IMMessage：NSObject

{

IMHandle * _sender;

IMHandle * _subject;

NSAttributedString * _text;

NSString * _plainBody;

NSDate * _time;

NSDate * _timeDelivered;

NSDate * _timeRead;



しかし、公式には、Appleは沈黙しています。



OS XのiMessageアプリケーションのコンセプトはオンラインで登場し、Jan-Michael Cartによって発明されました。





個人的には、別のアプリケーションでその点を見ていませんが、実際にはそうなるので、すぐにわかると思います。 OS XでのiMessageサポートの欠如を遅らせることは、Appleの最大の利益ではありません。
