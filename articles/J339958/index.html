完全な枯渇、またはIPv4後の寿命はありますか

2016年のCloudFlareインターネットサミットで、GoogleのWeb生産性エンジニアであるIlya Grigorik は、次の予測を行いました 。「私が知る限り、2020年までに飛行機械、特異点、IPv6が登場するでしょう。」



1年前のTwitterで、彼は「IPv4パーティーが終わった」と発表し、IPv4アドレスの枯渇に関する過去30年間の非常に人気のある議論に加わりました。 この記事では、既知の事実を要約し、次に何が起こるかを評価します。





/ Pixia / katja / CC



IPv4今



イリヤが2015年に反応したニュースは、IPv4アドレスの完全な枯渇に関する世界のインターネットレジストラ（アメリカのARIN）の声明に関するものでした。 これに先立ち、APNICおよびRIPEの地域レジストリからアラームメッセージが受信され、2017年初頭にLACNICはIPv4の発行計画の第3フェーズへの移行を発表し、残りの470万のパブリックアドレスに到達しました。



まだパブリックIPv4アドレスを持っている唯一の場所はアフリカです。 アフリカの埋蔵量も近い将来に使い果たされますが。 アドレス支出のダイナミクスを特別なサイトで監視して、状況を把握できます。



今後数年間で、インターネットサービスプロバイダーは 、NATメカニズムと、ほとんど無制限のアドレスを持つインターネットプロトコルであるIPv6の選択に直面するでしょう 。 多くのオペレーターは2番目のオプションを選択します。 Verizon Wireless は、今年の夏、企業顧客へのパブリック静的IPv4アドレスの発行を停止し、IPv6に完全に切り替えました。 これまでのところ、世界中のサイトの10％だけがIPv6を使用しています。



一方、IPv4アドレスは貴重な再販ターゲットになりつつあります。 偶然にも、これはARINの社長兼CEOであるジョンカランによって 7年前に予測されました 。 ARIN地域にはいわゆるウェイティングリストがあり、一部の組織はバイパスを行って、二次市場や闇市場に目を向けています。 2015年、ARIN は 4,000万のIPv4アドレスを承認しました 。 2016年には、この数はほぼ50％減少し、2,075万になりました。



IPv4トランザクションの24の公式に登録されたARINブローカーの1つであるAvenue4の副社長Janine Goodmanによると 、流通市場のアドレスの価格は10ドルから15ドルの範囲です。 犯罪者が、ネットワークを拡張するためにアドレスを過剰に支払おうとするユーザーをだましてしまうことは驚くことではありません。



IPv4が終了する理由



この問題の原因は古くから知られています-TCP / IPの採用時に地球上に住んでいる45億人の43億の32ビットIPv4アドレスは合理的な考えのように思えました。 しかし、インターネットアクセスの速度と地球上の人口の増加にすぐに疑問を呈しました。





/ IPv4 /ウィキメディア/ 不確定 / PDアドレス



最初のARPAnetネットワークには数百のホストが含まれていましたが、数十億のホストは含まれていなかったため、8ビットアドレスはその初期の段階でネットワークの要件を完全に満たしていました。 1983年に、TCP / IPモデルに切り替えると、選択は32ビットで停止しました。これは長い間、十分でした。 多くの企業は 、必要以上に多くのIPv4アドレスを受け取りました 。



IPアドレスの枯渇により、10年未満でスケーラビリティに関する最初の疑問が浮上しました。 これには、1992年に地域のインターネットレジストラが設立され、地元の市場でアドレスブロックを管理する権限が与えられました。



Webの出現により、ユーザーの成長と新しいアドレスの需要を止めることはできなくなりました。 90年代前半には、 クラスレスアドレッシング方法、 NATテクノロジーなど、避けられないものを含める試みが行われました。



クロスドメインルーティングやネットワークアドレス変換などのいくつかのイニシアチブは、ルーティングテーブルのサイズを削減し、IPアドレスの使用を最適化するのに役立ちました。 しかし、一時的にのみ。 ARIN諮問委員会のメンバーであるOwen DeLongによると 、これらの手順がなければ、アドレスは数年で終了します。 それ以降、各レジストラは「経済状況に存在する」何らかの形でそれを行ってきました。



その間、すでに1994年に、IPv4の後継者の開発、そして少し後に IPv6の基本原則の開発が推奨されました。 その瞬間から、IPv6をサポートするハードウェアおよびソフトウェア製品が登場し始めましたが、新しいプロトコルは基本的にバックアップ計画として機能しました。 今日、プロトコルの割合は徐々に増加しています。





/ IPv6 /ウィキメディア/ 不確定 / PDアドレス



現在の状況は、モバイルデバイスの急増と最初の10年間のアドレスの非効率的な配布によって悪化した、グローバルネットワークでのホストの成長に対する低い期待の結果です。



移行の難しさ



IPv6は1999年から運用準備が整っていますが、企業は急いで移行することはありません。 これは、IPv6にはIPv4との下位互換性がないためです。 ただし、2011年に、ハリケーンエレクトリックのIPv6オペレーターディレクターであるMartin Levyは、費用のかかる設備投資や将来の混乱を避けるために、新しいプロトコルに切り替えるよう会社に求めました 。



ただし、アドレス自体は無料で利用できますが、IPv6への切り替えも高価なソリューションです。 実装には、新しいネットワークスイッチとルーターの購入とITスタッフのトレーニングが必要です。 一方、IPv4でインフラストラクチャを維持することは、時間とともに価値が増加するだけだという意見があります。 移行フェーズは業界にとって最も費用がかかるため、プロバイダーと組織は将来の戦略を迅速に決定する必要があります。



IPv6には、無尽蔵の可能性以外にも多くの利点があります。 たとえば、次世代プロトコルには、個々のデータパケットを暗号化および認証する統合されたIPSecコンポーネントがあります。 もちろん、IPv6には脆弱性が残っていますが、サイバー犯罪者の作業が複雑になる可能性があります。



ほとんどのオペレーティングシステムは、IPv6を採用する準備ができています。 スマートデバイスも準備が整っており、IPv4の状況を大きく悪化させています。 ただし、多くの古いルーターはIPv6と互換性がないため、ほとんどのインターネットサービスプロバイダーは同じプロトコルの2つの実装をサポートする必要があります。 IPv6への完全な移行では、IPv6をサポートするサイトへのアクセスのみを取得できることが重要です。



次は何ですか



IPv6アドレスの数は非常に多いので、それらを使い果たす見込みは、最も悲観的な未来派でさえ気にしませんが、使い果たされた状況とこのプロトコルの下での未来の問題は定期的に提起されます。 ネットワークには、将来のプロトコルであるIPv9の記述があり、あらゆるアドレスに対応しています。



研究者Bill St. Arnaud は 、IPv6は新しい機能を提供しないと不満を述べています。 彼は、電話システムと米国の郵便局がかつて同様の問題に直面したが、新しいアドレス指定システムをゼロから作成するのではなく、既存の接尾辞と接頭辞を拡張する解決策を見つけたと指摘します。 したがって、ビルは、新しいプロトコルへの移行よりも根本的ではない代替オプションの検索を示唆しています。



ホスティングプロバイダーの創設者であるGabriel Ramuglia は 、IPアドレスは化石燃料ではなく、レジストラからブロックを受け取った大企業は実際には使用しなかったことを思い出します。 実際、多くのアドレスは単に要求されないままであり、問​​題全体はそれらの非効率的な配布にあります。



アドレス空間のほとんどは現在も使用されており、非アクティブなセグメントは徐々に流通に戻っています-今年だけで、マサチューセッツ工科大学は1400万の「余分な」IPアドレスを発見しました 。 それらのほとんどは売りに出されました。



さらに、 リバースプロキシサーバーなど、新しいアドレス節約ソリューションが登場しています 。 したがって、IPv4が突然存在しなくなることはほとんどありません。 インターネットは確立されたパラダイムで生き続けています。
