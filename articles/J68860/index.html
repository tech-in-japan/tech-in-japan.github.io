Snow Leopard：ライセンスと価格について

Mac OS Xの以前のバージョンと同様に、Snow Leopard 10.6はいくつかのバージョンで販売されており、それぞれがライセンス契約 （EULA）に含まれています。

  • A.シングルユーザーライセンス。 このライセンスの契約条件に従い、ファミリーパックライセンスまたはAppleソフトウェアアップデートライセンスを購入していない場合、1つのブランドのコンピューターにAppleソフトウェアの1つのコピーを同時にインストール、使用、実行するための限定的で非独占的なライセンスが付与されますりんご お客様は、Appleソフトウェアを他のブランドのコンピューターにインストール、使用、または実行しないこと、または他のユーザーが上記のアクションを実行できないようにすることに同意します。 このライセンスでは、一度に複数のコンピューターでAppleソフトウェアを保守および使用することは許可されていません。 また、複数のコンピューターで同時に使用するためにAppleソフトウェアへのネットワークアクセスを提供する権利もありません。

    このオプションはApple Storeで「Mac Box Set」という名前で販売されており、価格は169ドル（部品番号MC209Z / A）です。 Mac OS X 10.6に加えて、パッケージにはiLife '09およびiWork '09が含まれています。 ロシアでは、このバージョンのバルカは8140ルーブルかかります。
  • C. Leopardをアップグレードするためのライセンス。 このライセンスの契約条件に従って、Mac OS X Leopardをアップグレードするためのライセンスを購入すると、このコンピューターが既にインストールされている場合、1つのAppleブランドコンピューターにAppleソフトウェアの1つのコピーをインストール、使用、実行するための限定的で非独占的なライセンスが付与されますMac OS X Leopardのライセンスコピー。 ファミリーパックライセンスを購入してMac OS X Leopardをアップグレードすると、このライセンスの契約条件に従って、最大5台のAppleブランドのコンピューターでAppleソフトウェアのコピーを1つインストール、使用、実行するための限定的で非独占的なライセンスが付与されます同時に、これらのコンピューターが同じ世帯（上記で定義）にあり、この世帯に住んでいる人が使用し、各コンピューターが既にMac OS X Leopardのライセンスコピーを持っている場合に限ります。 ファミリーパックのMac OS X Leopardライセンスは、法人および商用ユーザーには適用されません。

    ロシアでMC223RS / Aという番号で990ルーブルの価格で販売されている（およびApple StoreでMC223Z / Aという番号で29ドルで販売されている）アップグレードライセンスです。
  • B.ファミリーパックのライセンス。 このライセンスの条項に従って、ファミリーパックライセンスを購入すると、最大5台のAppleブランドのコンピューターで同時に1つのAppleソフトウェアのコピーをインストール、使用、実行するための限定的で非独占的なライセンスが付与されます。ただし、これらのコンピューターは同じ世帯にあり、この世帯に住んでいる人が使用します。 「住宅所有権」という用語は、住宅、アパート、モバイルキャラバン、協同組合所有権など、同じ住宅単位に住んでいる人を意味しますが、指定された住宅所有権の永住地に登録されている学生にも適用されますしかし、ほとんどの場合、彼らは別のキャンパスに住んでいます。 ファミリーパックライセンスは、法人および商用ユーザーには適用されません。

    MC224Z / Aサービスパックの「ファミリーライセンス」は49ドル（ロシアでは1,740ルーブル）、MacボックスセットMC210Z / Aは229ドル（11,206ルーブル）です。
