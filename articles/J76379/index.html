アプリケーションアイコンを変更する

今日、プログラムのアイコンを変更するタスクがありました。 しかし、このトピックをもう少し広く見てみましょう。 Mac OS Xは伝統的に非常に素晴らしいアイコンを持っています。 これは、アプリケーションとシステム全体の両方に適用されます。 ただし、アイコンがわずらわしい場合や、サードパーティのプログラムの場合はまったく合わない場合もあります。 これらは最も可能性の高い理由です。 実際、あなたはそれらの多くを思いつくことができます。 事前に話すと、記事を書く動機はまさに2番目のケースです。 その上で、私たちは入門からビジネスに移ります。



Mac OS X上のAppleには、 icns拡張機能を備えた独自の確立されたアイコン形式があります。 16×16、32×32、48×48、128×128、256×256および512×512のサイズをサポートしています。 ただし、必要な形式の既製のアイコンを探すのは、適切なグラフィックイメージを見つけるほど簡単ではありません。 その結果、変換の問題が発生します。



これらのタスクの最も一般的なWebサービスはiConvertと呼ばれます 。 Mac OS Xだけでなく、他の一般的なオペレーティングシステム用のアイコンを作成するのに役立ちます。 それを使用するには、グラフィックイメージを検索または作成し、サイトのナビゲーションキーを介してそのパスを指定する必要があります。 変換が完了すると、彼はMac OS X、Windows、Linuxのアイコンを提示します。







画像のソースは、 Google Imagesまたは私の大好きなIcon Finderサービスで検索できます。



同じニーズを満たすローカルソフトウェアのファンは、アプリケーションImg2icnsを配置できます。 無料とプロの2つのバージョンで配布されています。 最初のものは、変換履歴を保存する方法、iPhoneとfaviconのアイコンをエクスポートする方法、アイコンをグラフィック画像形式に変換する方法を知りません。 はい、必要ありません。 画像をプログラムの本体にドラッグし、Icnsの位置を選択します。







ところで、プログラムはすぐにフォルダのアイコンを作成できます。 確かに、いくつかは役に立つでしょう。



必要な形式の完成したファイルができたので、現在のアプリケーションアイコンの代わりにそれを置き換える必要があります。 とても簡単です。 置換するファイルまたはフォルダーを選択し、コンテキストメニューから[情報を見る]に移動します（コマンド⌘+ Iを押すだけです）。 次に、下の図に示すように、icnsファイルを左上隅のアイコンにドラッグします。







結果は、ファイルマネージャーとDockの両方に表示されます。











アクションをロールバックしてネイティブアイコンを返すには、[情報を見る]を開き、左上隅でマウスをクリックしてDel⌦キーを使用する必要があると言うだけです。 以上です。 私はフィードバックやコメントを聞いてうれしいです。
