Linux



現在、オープンソース製品の流行はなくなっています。 特にロシアでの著作権侵害防止法の導入後。 奇妙なことに、このファッションは私たちの国を迂回していません。



ますます多くの人々がUNIX / Linuxシステムに切り替えています。 そして、それはどこから始まったのですか？



Linuxプロジェクトは1991年にUsenet comp.os.minixニュースグループに次の内容のメッセージを発行して始まりました。

「minixを使用しているすべての人にこんにちは-クローン386（486）ATの（無料の）オペレーティングシステム（趣味であり、gnuのように大きくてプロフェッショナルではない）を作成します...」



その時までに、GNUプロジェクトはすでに無料のオペレーティングシステム用の多くのコンポーネントを作成していましたが、GNU Hurdカーネルはまだ準備ができていませんでした。 当時のBSDシステムは、使用されているライセンスに関連する法的理由により使用できませんでした。 そのため、Linuxは無料のOSのカーネルの空の場所を取り、以前のバージョンの機能が制限されていたにもかかわらず、多くの開発者とユーザーを魅了しました。



Linuxはカーネルの名前であり、オペレーティングシステムの名前ではありません。 多くの場合、Linuxベースのシステムは単にLinuxと呼ばれますが、それらのほとんどはLinuxカーネルとGNUプロジェクトの一部として書かれた多くのシステムライブラリとプログラムで構成されているため、実際にはGNU / Linuxと呼ばれます。



現在、Linuxカーネルファミリのオペレーティングシステムは、デスクトップ市場で世界で3番目に人気（0.63％）です。

それらはどう違うのですか？

おそらく、Linuxという名前とこのプロジェクト全体が、OSのカーネルのみに正しく帰属されることを明確にする時が来たのでしょう。 カーネルは他のプログラムにコンピューターリソースとハードウェアへのアクセスを提供しますが、システムが機能するには不十分です。 IBM PCでBIOS（基本入出力システム）を使用して類似性を引き出すことができます。 Linuxカーネルは、インターネットBIOSの一種、または21世紀のSuperBios（POSIX互換、ネットワーク、マルチユーザー、ほぼすべてのファイルシステムとデバイスをサポート）と呼ぶことができます。 動作するには、通常のUnixプログラムとユーティリティのセットが必要です。 Linuxカーネルは、Richard Stallmanが始めたGNUプロジェクトの一部であり、厳密に言えば、動作しているGNU / Linux OSシステムの正式名称です。



その後、インストールの準備ができ、コラボレーション用に構成されたプログラムのセットが配布キットとして作成され始めました。 ディストリビューション内のプログラムの数が数百を超えると、各プログラムはインストールの準備ができたパッケージとして設計され始め、特別なインストーラープログラムがプログラムの相互依存または競合を監視しました。 最初の大量配布は、Patrick VolkerdingのSlackwareでした。 Red Hat Sofwareが作成し、ディストリビューションやCaldera製品で使用されているRPMパッケージフォーマットは、KSI-Linux（キエフのSergey Kubushinのロシア語配布キット）をはじめとする多くの製品で非常に人気があります。



Debianプロジェクトの一部として、別のユニバーサルパッケージ形式とインストーラーが開発されました。 このプロジェクトは、GNUやLinuxのように、その参加者がインターネットを介して協力するだけでなく、その規模だけでなく、最新バージョンの4,000を超えるパッケージで配布されていることで区別されます。 Debianは国際プロジェクトであり、多くの言語のドキュメントが含まれており、暗号化および情報保護技術の制限はありません。



ディストリビューションにより、インストールと初心者向けLinux開発の最初のステップが大幅に簡素化されました。 プログラム開発者とそのユーザーの間に仲介者が現れました-このパッケージに付随する人、いわゆる メンテナー、護衛。 インターネットには無数の情報とプログラムがあり、エスコートは最も必要で関連性の高いものを見つけ、それらを複雑なシステムに埋め込むという最も重要な仕事をしています。



そして最後に、CD-ROMで配布されたディストリビューションにより、何百万人ものユーザーが初めてLinuxを知ることができました。

それらのいくつかについてのいくつかの言葉：



Red Hat Linux





Red Hat Linuxは、世界で最も広く配布されているLinuxディストリビューションの1つです。



現在、Red Hat自体は同名の会社が生産する商用ディストリビューションになっています。 同じディストリビューションの非営利の無料配布バージョンがFedora Coreという名前でリリースされています。



Debian





Debian GNU / Linuxは最大のディストリビューションです。 2,000以上のパッケージが含まれています。 Debianの作成者はライセンスについて非常に慎重であるため、Debianは最も「クリーンな」ディストリビューションです。 最終製品のテストには多くの注意が払われます。 非常に信頼性が高く、管理が簡単です。 サーバーアプリケーションとデスクトップアプリケーションの両方に適しています。 Debianは、i386、m86k（amiga、atari、macs）、Alpha、およびSparcプラットフォームで使用できます。

Linuxを真剣に研究して作業したい人や、フリーソフトウェアのアイデアに近い人におすすめです。 インストール手順は複雑であるため、初心者ユーザーがこのディストリビューションのLinuxに慣れるのはお勧めできません。



SuSE





SuSE Linuxは、ドイツで開発されたヨーロッパで最も人気のあるディストリビューションの1つです。 SUSEは最近Novellに買収され、NovellはこのディストリビューションをNovell SUSEという名前でリリースしています。 ディストリビューションには、非常に便利なインストールおよび管理プログラムYaSTがあります。 SUSE Linuxは初心者と古いLinuxファンの両方を満足させますが、プログラマーとシステム管理者の専門家向けです。 そのため、重要なタスク、システムの注文が必要なユーザー、および独自のサポート（もちろん有料）が必要なユーザーに推奨できます。



Slackware





このディストリビューションの構成のシンプルさとロジックにより、Linuxデバイスを完全に理解できます。 ほとんどの設定は、追加のコンフィギュレーターやその他の「レイヤー」なしで「直接」行われます。 これにより、初心者にとってディストリビューションが少し複雑になりますが、多数のユーザーの間で当然の人気を得ています。 インストールパッケージでUNIX標準のtar.gz形式を使用することも非常に便利です。 パッケージの更新は定期的に表示されます。 通常、このディストリビューションは、Linuxを完全に処理する場合に推奨されます。



マンドリバ





Mandriva Powerpackは、これまでLinuxを使用したことがないが、自分は経験豊富なWindowsユーザーであると考えている人にとって最適な選択肢です。 このシステムはインストールと設定が簡単で、日常業務に必要なすべてのアプリケーションが含まれています。 一方、配布キットには、特にサーバー構成およびWeb開発者向けの多くの追加ツールが含まれています。 この場合、非常に良いことは、これらの資金は必要ないが、それらに気付かないかもしれませんが、それらが必要になるとすぐに、それらがすでに手元にあることがわかります



Ubuntu





Ubuntuは、ラップトップ、デスクトップ、およびサーバーに最適なオペレーティングシステムです。 自宅でも、学校でも、職場でも、Ubuntuには、テキストやメールアプリケーションからWebサーバーソフトウェアやプログラミングツールまで、作業に必要なものがすべて含まれています。

現時点では、最も人気のあるディストリビューションの1つです。 Linuxの初心者に最適です。



ジェンツー





Gentoo（英語-「Jintuu」、ロシア語-「Gentu」）-LinuxまたはFreeBSDに基づいた無料のオペレーティングシステム。あらゆるニーズに合わせて、ほぼすべてのタスクを自動的に最適化および調整できます。 極端なカスタマイズの柔軟性、最高速度、ユーザーと開発者のファーストクラスコミュニティは、Gentooの特徴です。



Portage（Portage）テクノロジーのおかげで、Gentooは理想的なセキュアサーバー、開発者ワークステーション、プロフェッショナルデスクトップシステム、ゲームシステム、組み込みシステムなど、何でも好きなものになります。 ディストリビューションはユーザーのニーズに適応する実質的に無制限の能力があるため、Gentooを「メタディストリビューション」と呼びます。

現在利用可能なほとんどのディストリビューションには、CD-ROMから始まる変更があります。 この種の最も有名な（そして明らかに最初の）分布はKnoppixです。
