Marc GregoireによるProfessional C ++、第4版



「Professional C ++」は、高度なC ++プログラミングガイドです。 経験豊富なプログラマーが最新の更新プログラムを使用できるように設計されたこの本は、基本を説明し、C ++ 17のすべての機能の説明に直接突入します。 各機能は、プロジェクトで使用できる例とコードによって補完されます。



実証された例には、WindowsおよびLinuxコードでのテスト、およびワークフローの改善に役立つ著作権のヒントと秘密が含まれています。 多くの経験豊富な開発者でさえ、言語機能の限界を完全に検討したことはありません。 この本は、あなたが決して知らなかった機能について話しています。 その主な目標は、これらの機能を実際のソリューションに変える方法を示すことです。

モダンC ++の課題：マリウスバンシラによる現実世界の問題を解決するエキスパートになる



C ++は最も広く使用されているプログラミング言語の1つであり、ゲーム、グラフィカルプログラミング、オペレーティングシステムなどのさまざまな分野での使用例を示しています。 長年にわたり、C ++は世界中のソフトウェア開発者にとって最高のソリューションの1つであり続けています。



この本は、いくつかの重要なC ++機能と、ニーズに合わせてそれらを実装する方法を示します。 各問題は固有のものであり、言語の知識をテストするだけではありません。 彼女はあなたの戦略的思考能力とあらゆる状況から最善の方法を見つける能力をテストします。 しかし、時には開発者を混乱させるさまざまな問題があります。 この本はそのような状況で正確に救助に来ます。



この本は、あらゆるレベルのC ++開発者にアピールします。

BartłomiejFilipekによるC ++ 17の詳細



C ++ 17はC ++ 11ほど大きくはありませんが、C ++ 14より大きくなります。



多くのモジュール、共同ルーチン、概念、およびその他の強力な機能が期待されていましたが、開発者はすべてを時間通りに準備できませんでした。 C ++ 17はこのために悪いですか？ まったくありません。 そして、この本はその理由を示します。



この本は、C ++ 17に関する排他的な知識を提供し、bfilipek.comに掲載された多くの記事の経験を利用しています。 章は最初から作成され、最新情報に更新されました。 また、多くの例と実用的な使用上のヒントもあります。 さらに、現在の実装状況、コンパイラサポート、パフォーマンスの問題、および進行中のプロジェクトの改善に役立つその他の関連知識についても説明しています。



C ++ 11/14のアイデアがあり、最新のC ++メソッドに移行したい場合は、この本が役立ちます。
