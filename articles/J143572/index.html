Qt Creator 2.5.0がリリースされました！

以前はベータ版とRCであった新しいQt Creator'aのリリース。



変更と革新のリスト：



  • [再検索]ボタンを使用した高速再検索
  • ロケーターから直接シェルコマンドを実行する
  • 敬意を表したVassが すでにHabréに書いた TODOプラグイン。公式ニュースに感謝の意が表されています。
  • autotoolsを使用するためのプラグイン
  • C ++での追加のリファクタリング：メソッドの抽出、欠落している#include



    追加、関数引数の再配置
  • C ++ 11（nullptr、constexexp、static_assert、noexcept、インライン名前空間、自動、ラムダ）のサポートの改善
  • QMLエディターの新しいヒントと機能（および特別なコメントでそれらを無効にする機能）
  • 多数のバグ修正
  • Windowsバージョンでは、MinGWはインストールパッケージから削除されました
