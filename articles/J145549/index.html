Visual Studio Express 2012 for Windowsデスクトップ

それにもかかわらず、コミュニティからの圧力により、MicrosoftはWindows Desktop用の無料プログラムVisual Studio 2012 Express のリリースを発表しました 。



無料のVisual Studio ExpressパッケージはMetroアプリケーション、Webアプリケーション、Azureアプリケーション、Windows Phoneアプリケーションの4つのバージョンでしか利用できないため、2週間前、同社は開発者にデスクトップアプリケーションを開発するために400ドルでVisual Studio 2012 Professionalにアップグレードする必要があるというニュースでショックを与えました、つまり、デスクトップアプリケーションを除くすべての領域をカバーします。



もちろん、多くはマイクロソフトによるそのような決定に不満でした。 幸いなことに、同社はコミュニティに耳を傾け、間違いを認め、計画を変更しました。



どうやら、Express Desktop 2012では、Vista、Windows 7、およびWindows 8のアプリケーションを作成できます。「Visual Studio Express 2012 for Windows Desktopは、デスクトップアプリケーションを開発するための最初から最後まで簡単な方法を提供し、VisualがサポートするすべてのバージョンのWindowsをサポートしますStudio 2012、 公式ブログによると。 -Expressの新しいエディションでは、開発者はC ++、C＃、およびVisual Basicを使用してデスクトップアプリケーションとコンソールアプリケーションを作成できます。 また、開発者は、最新のコンパイラとツール、組み込みの単体テスト、チームエクスプローラーとTFS Expressを使用して小規模チームと開発を調整する機能など、Visual Studio 2012のExpressライン全体に実装された新機能にもアクセスできます。



Visual Studio Express 2012 for Windows Desktopは、2012年秋にリリースされる予定です。
