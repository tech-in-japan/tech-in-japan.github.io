Ciklum Java Saturday（15.10）-ミンスクでのJava開発者による非公式のミニ会議

ウクライナとベラルーシの オフィスでは、 Ciklum Saturdaysが常に開催され、さまざまな技術を使用する開発者が集まっています。



10月15日に、Exigen Servicesと一緒に、開発者をJava開発専用のCiklum Saturdayの Minskオフィスに招待します 。 このCiklum Subbotnikは、 以前のものと同様に、非公式のミニ会議の形式で開催されます。





Subbotniksの伝統によれば、イベントの参加者は、ソフトウェア開発の技術トピックに関するレポートだけでなく、イベント全体を通じて、非公式にコミュニケーションを取り、知識を交換し、経験を共有することができます。 コーヒーブレークと軽食、 新しい写真スタジオでの写真撮影が添付されています:)



レポートの予備リスト ：

  • ウラジミール・ゴルシュノフは「Google App Engineの紹介」というプレゼンテーションを行います
  • Andrey MarmyshがGWT2.3でJavaアプリケーションを開発した経験について話します
  • Alex Krukは、VMForceプラットフォームでの開発機能を共有します
  • Igor Khotin（キエフの特別ゲスト）がGradleについて語る-新世代のjavaツール
  • Andrey Lumyanskyは、BPMのコンテキストでJavaプロジェクトのビジネスプロセスを強調します


そして、いつものように、最後に、ビールレストラン「オールドチャンネル」のアフターパーティに参加することをお勧めします。



直接のオンラインブロードキャストが計画されているため、他の都市のゲスト、およびミンスクで最も怠peopleな人々は、事実上イベントに参加し、ソーシャルネットワーク経由で質問をすることができ、ハッシュタグ#CiklumJavaの更新をお楽しみに



Ciklum Saturdayへの参加は、 事前の登録を条件として、いつものように無料です。



注意！ 登録は10月13日までのみ可能です。お急ぎください。座席数は限られています。



PS

レポートの詳細なスケジュールとスピーカーの最終リストは、登録された参加者向けの個人的な電子メールニュースレターでイベントに近づきます。



サブボトニックは、レポートを聞くだけでなく、ディスカッションに積極的に参加するすべての参加者にとって、興味深く有用なものになるでしょう。 そして、最も活発なものは、CiklumとExigen Servicesからの贈り物だけでなく、彼ら自身のポートフォリオも受け取ります！
