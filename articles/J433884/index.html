ビデオ監視の法的側面：法律の問題を回避する方法





個人データに関するロシアの法律の強化、カメラの違法設置に関するスキャンダル、ビデオ撮影と写真撮影の禁止、疑念が生じた背景：知らないうちに他人の権利を誤って侵害した場合はどうなりますか？



カメラを購入するとき、店舗にビデオ監視キットをインストールするとき、または国外にあるデータセンターでクラウドアーカイブを予約するときに心配する必要がありますか？



業界では、誰かの権利を侵害する2つの主な方法があります。最初に、原則として、ビデオ監視をインストールし、次に受信したビデオデータを処理します。



両方の方法を分析し、質問に答えます。カメラを置くのが適切か、受信したビデオをどう処理するかです。



公共の場でのCCTVの機能





（ c ）



最初の質問から始めましょう：問題なくビデオ監視をどこにインストールできますか？



まず、ビジネスが運営されている公共の場所で、次に組織のワークスペース（オフィス、工場など）で。



ロシアの法律は、公共の場所でのビデオ監視システムの設置と運用を直接禁止していません。 また、使用するシステムの要件も提供しません。 それが規制するのは、映像の使用です。



h。1の記事 ロシア連邦民法152.2は、州または公共の利益、公共の場所、および公共のイベントで監視が行われる場合、市民の画像を受け取って使用することに同意する必要はないと述べています（「そのような画像が主な使用目的である場合を除く」）。 人が商業的にカメラの前でポーズをとった場合。



私たちの経験では、システムと法律の手紙との完全な一致は、1つのケースで証明される必要があります。 ある事件がDVRまたは監視カメラからの記録に記録され、同じ盗難であり、あなたが事件を公式な動きにしたので、証拠ベースにビデオを含めたいとします。



ロシアの裁判所の慣例によれば、そのような記録は、修正されたかどうかを確実に確認するために検査のために送信されます。たとえば、個々のフラグメントまたはフレームが切り取られたかどうか、逆に、後処理中にシーンまたは画像要素が追加されたかどうかです



このような状況では、ビデオ素材を評価するための最も重要な基準は、事件の瞬間から法執行官への記録の転送までにどれだけ時間が経過したかです。

たとえば、インシデントの1週間後に気付いた場合、専門家は、この時間内にビデオが変更される可能性があると信じている可能性があります。 そして、これは記録を証拠として使用しない正当な理由として役立ちます。



場合によっては、法律はビデオ監視の設置を直接規定しています：



  • 輸送インフラ：空港、海港、河川港、車両、道路施設の分野の輸送インフラ
  • 燃料とエネルギーの複合体のオブジェクト
  • スポーツ施設
  • 人々の大量滞在の場所
  • ロシア連邦の国家警備隊の軍隊による強制的な保護の対象となるオブジェクト（領域）
  • ロシア連邦およびその管轄下にある組織の建設・住宅・公共サービス省のオブジェクト（領域）
  • 文化分野のオブジェクト（領域）
  • 取引オブジェクト（テリトリー）
  • ホテルおよびその他の宿泊施設
  • 教育機関
  • 社会文化的および共同体の目的、アパートの建物の非居住施設、設計の割り当てによれば、50人以上がいつでも部屋にいると想定され、その操作は特別なアクセス制御の確立を提供しない
  • 住宅の複数アパートの建物（新しい建物）


原則として、人々の集団集会のすべての場所で、これは主に「建物および構造物のテロ対策保護の確保」に関連付けられています（ちなみに、これは、そのようなオブジェクトの設計のための要件を持つルールセットの名前です）。



規範的な行為は、多くの場合、観察領域、ビデオ分析の要件、および画像の技術的特性を規制します。 とりわけ、このような施設は、たとえばセーフリージョンプログラムの下で、州のセキュリティシステムに接続できます。



個人や組織は、比較的少数の場所での撮影（およびビデオ監視）を明示的に禁止されています。 これには、状態秘密のオブジェクト、および57-FZ「 状態保護 」の範囲に含まれるオブジェクトが含まれます 。



そのため、展開のポイント周辺にあるものを慎重に検討することをお勧めします。 モスクワの中心部にも秘密の軍事ユニットがあり、これは都市伝説ではありません。 さらに、原子力発電所など、戦略的に重要ないくつかのオブジェクトの領土での射撃は許可されていません。





（ c ）



店舗やケータリング施設などの公共の場所にある商業企業の大部分は、彼らが働くエリアでビデオ監視を行うために、「注意！ ビデオ撮影が進行中です。」 たとえば、ショッピングセンターの管理者は、このような警告を複合施設の入り口に配置する必要があります。



そして、そのような公共の場所で外国の市民の存在が想定される場合、碑文は母国語で複製されなければなりません。 たとえば、モスクワは世界で最も魅力的な観光都市の1つであるため、公共の場所でここで活動を行う場合、ビデオ監視の碑文は少なくとも英語で複製するのが最適です。



同時に、プライバシーを念頭に置く必要があります。 刑法の137 （「プライバシーの侵害」）。 ビデオ監視手段でカバーされているスペースの一部が公共の場所のカテゴリに明確に属していない場合に、それに定められた要件が施行される可能性があります。



カメラが住宅の建物内の階段に置かれ、ドアが開いているときにアパート内のスペースのかなりの部分をキャプチャする場合、決定は法律の対象となります。 このような状況に関する司法慣行はあいまいですが、同様の状況下で、原告が細胞を解体しようとした先例がありました。



ワークスペースのCCTV





（ c ）



オフィス内または生産施設でのビデオ監視は許可されています。 また、 アートによると。 ロシア連邦労働法209条では 、職場は雇用主の管理下にあるべきです。



このような規制を遵守する手段としてのビデオ監視は可能ですが、憲法で規定されている市民の基本的権利（特にプライバシー）および個人データに関する法律（PD）に反しないという条件でのみ可能です。



一般原則には例外があります。 組織に属する施設内のCCTVシステムを使用して、運用検索活動以外の特定の人物に関する情報を収集することはできません。 その結果、カメラは常に1人しか作業しない部屋に取り付けることはできません。



トイレ、更衣室、更衣室などにCCTVを設置することは絶対に不可能です。 そうでなければ、 アートの直接違反になります。 ロシア連邦憲法の3 （「私生活の不可侵性について」）。



最後に、アートによると。 ロシア連邦刑法の138.1（「情報を密かに入手することを目的とした特別な技術的手段の違法取引」）、隠された射撃は禁止されています。 組織に属する施設と公共の場所の両方。



禁止されている機器の完全なリストはありませんが、その兆候は知られています（たとえば、カメラ、「家庭用品用に偽装」、「リモートエントリバッジ-ピンホール」など）。詳細については、 政令を参照してください。 これらの兆候の1つでもデバイスが準拠しているということは、デバイスの購入と操作が法律に違反していることを意味します。



秘密のビデオ監視の手段は、捜査措置の一部としての警察、FSBなどを含む法執行官のみが使用することが許可されています。



職場でビデオ監視を組織するときに身を守るために、雇用主は従業員全員に、それが利用可能であることを通知する必要があります-「署名に対して」。 セキュリティ目的、労働規律、生産規律など、このような監視が行われる理由を示す必要があります。 関連する条項を内部労働規制および雇用契約に含めることをお勧めします。



これらのドキュメントでは、念のため、オーディオ録音の可能性に言及する価値があります（もちろん、隠されていません）。



ところで、一部の業界では、消費者とやり取りする際の視覚または音声情報の固定は、規制当局が指示する必要性に徐々になりつつあります。 特に、中央銀行は、ロシアの金融機関が顧客との交渉をビデオまたはオーディオメディアで公然と記録することを推奨しました 。



オフィスは個人の領土であるため、ビジネスオーナーは、外部の人、たとえばクライアントをそのオフィスに入れたり入れたりすることを許可する権利があります。 ただし、クレームから身を守るには、入り口にビデオ監視警告サインを設置する必要があります。



オフィス内や職場での個人データの処理の問題に取り組むことは、公共の場所のビデオ監視の場合と同様に細心の注意を払っています。 職場でのビデオ録画は、同じ目的でのみ許可されています。



152-FZ（「個人データ」）に加えて、 アートも念頭に置く必要があります。 ロシア連邦労働法の88 （「従業員の個人データの転送」、これに応じて、従業員の個人データへのアクセスは、特別に許可された人にのみ許可されますが、これらの人は、特定の機能を実行するために必要な従業員の個人データのみを受け取る権利があります」



上記のすべてに基づいて、私たちは結論を下します。職場では、ビデオ監視システムとアーカイブへのアクセス体制に関する組織の内部規則を作成する必要があります。



個人データの処理に関する顔認識と責任





（ c ）



152-に従って市民の個人データを何らかの形で収集および処理するビジネスの責任範囲は大幅に拡大し、関連する基準に違反した場合の罰金は増加し、現在では最大30万ルーブルに達しています。



法律により、ビデオは個人を明確に識別することはできないため、個人データのキャリアではありません。 実際、ビデオには特定の人物の画像のみが含まれており、それ以上のものは含まれていません。 したがって、このようなデータを制限なく保存できます。



ただし、画像に識別子が割り当てられている顔認識システムが使用されている場合、指定された法律が有効になります。 このようなプラットフォームのメーカーとユーザーは、個人データを処理するためのライセンスを取得する必要があります。 確かに、識別子が非個人化されている場合（たとえば、P1-「person-1」、P2-「person-2」）、これは必要ありません。



顔認識の場合の法的要件はまだあいまいです。 特に、特定のサブジェクトNの「 顔のハッシュ 」は個人データには適用されません。その修正を制限する法的規範はありません。 しかし、たとえば、そのようなハッシュと姓名をデータベースに入力すると、物議を醸すような状況が発生します。 この場合、弁護士との協議が必要です。



データストレージとアーカイブの深さ





（ c ）



「アーカイブの深さ」またはビデオ監視システムを通じて取得した記録の保存期間は、建物や構造物のテロ対策による保護の確保に関連する措置の対象となる一部のオブジェクトに対して法的に規制されています。



そのため、たとえば、2017年4月14日のロシア連邦政府の政令447によれば 、ホテルはビデオ監視カメラを使用し、30日間記録する義務があります。 さらに、設置されたカメラの数とその場所を考慮に入れたビデオ監視システムは、ホテルの状況を継続的にビデオ監視する必要があります。 警告の技術的手段は、スタンバイモードで少なくとも6時間、信号および警告情報を送信するモードで少なくとも1時間、集中型電源をオフにするときの操作性を確保する必要があります。



また、一部の組織は、アーカイブの必要性と深さを規制する特定の基準に従っています。 ロシアのすべての銀行は、PCI DSS（ペイメントカード業界データセキュリティ標準）標準に準拠しており、ビデオ監視カメラおよび（または）アクセス制御メカニズムから受信したデータは少なくとも3か月間保存されます。



要約：すべきこと



ビデオ監視システムの公共の場所への基本的な設置とその使用は禁止されていません。 ただし、レコードをさらに使用するには（広告目的、裁判所での証拠など）、ビジネスからの追加のアクションと予防措置が必要になる場合があります。



会社が個人データの処理と保存に従事している場合、データが保存されているインフラストラクチャが152-FZの要件に準拠していることが厳密に必要です。



独自のインフラストラクチャの認証は、費用と時間がかかるプロセスです。 個人データの保存が既に認定されている「クラウド」でリソースをレンタルする方が簡単で安価です。



公共の場所でのビデオ監視システムの展開：



  • 秘密のオブジェクトまたは市民のプライベートなスペースがフレームに落ちないようにしてください。
  • ビルまたはワークスペースの入り口にある目に見える場所に、ビデオ監視の警告サインを配置します。
  • 市民が客観的にこの場所を訪れる可能性が高い国の言語に警告を翻訳します。


職場では、雇用主はビデオ監視システムを使用する資格もあります。 ただし、特定の個人の情報を収集したり、プライバシーの権利を侵害したりすることはできません。



ワークスペースにビデオ監視システムを展開する：



  • ワークスペースの入り口の目に見える場所にビデオ監視の警告サインを配置します。
  • オフィスまたは生産施設の領域でビデオ監視が行われているという事実を従業員に「署名により」理解させる。
  • ビデオ監視の必要性を正当化し、内部労働規制および労働契約の規則にビデオ監視に関する条項を追加してください。
  • ビデオ監視システムおよびアーカイブへのアクセス体制に関する内部規制を策定する（アクセスは特別に許可された人のみが利用できるようにする）。
  • トイレとロッカールーム（これは憲法に反する）、および常に1人だけが作業する部屋（これは特定の人の情報を収集すると見なされます）にカメラを設置しないでください。


デフォルトでは、ビデオは個人データのキャリアではありませんが、ビデオを撮影するだけでなく、顔認識システムを使用する場合は、法律152-FZの要件を満たす必要があります。 ライセンスの取得から始まり、データの保存に使用されるインフラストラクチャの要件で終わります。
