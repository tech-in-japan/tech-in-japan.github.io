JavaおよびBlackberry携帯電話向けのOpera Mini 8





本日、JavaおよびBlackBerry携帯電話向けのOpera Mini 8を新しいデザインと新機能で紹介します。 新機能 プライベートモードとナイトモード、キーボードからのタブへのアクセス、保存された統計。



プライベートモード







モードは設定で有効になっており、ネットワーク上の動きの履歴を保存しないようにすることができます。



ナイトモード







または、「夜更かしモード」とも呼ばれます。 目を煩わせたり色を反転させたりすることはせず、明るさを暗くするだけです。これは、光センサーのない携帯電話で特に便利です。



タブへのキーボードアクセス



これで、キーボードの上ボタンを押して新しいタブを開くことができます。 ただし、他のキーボードショートカットがあります。



  • Up



    新しいタブ用
  • # 0



    ページをリロードする
  • # 2



    クイックGoogle検索
  • # 3



    ページで検索
  • # 5



    ブックマークへのアクセス
  • # *



    -夜間モード


保存済みの詳細







更新された統計画面では、Opera Miniを使用して節約したトラフィック量をさらに確認できます。 いつものように、 外部リンクm.opera.comで新しいOpera Miniをダウンロードできます。
