MS Empowerイニシアチブ

このトピックは、有料または無料のソフトウェアのサポーターの別の戦争を再燃させる試みではなく、開発ツールを含むMicrosoftソフトウェア製品を必要とする人々に向けられています。



このトピックはまだハブで議論されていないため、多くのトピックが有用で興味深いものになると思います。

法執行機関の関連部門が過去数か月間（および数年頃）に、ライセンスのないソフトウェアを使用して説明責任のある組織を見つけて保持する作業を強化したことは周知の事実です。 もちろん、このハントの主なソフトウェアの餌は、業界リーダーの製品です。



計算が簡単なため、約10 TRのWindows + Officeのセットを購入します 1つの職場で。 サーバーのオペレーティングシステムを考慮すると、さらにコストがかかります。 ただし、多くの場合、 MS Empowerプログラムは多くのお金を節約し、有用性を獲得します。



条件は単純で簡単です。

MSテクノロジーに基づくソフトウェアの生産に従事し、近い将来にリリースする計画を立てている法人は、このイニシアチブに参加できます。 ソフトウェアに特別な要件はありません。

あなたが得ることができます：
最大5つの内部ソフトウェアライセンス：

o Microsoft Windows Vista BusinessまたはEnterprise、またはMicrosoft Windows XP。

o Microsoft Office 2007または2003。



内部使用のための1つのサーバーライセンスと最大5つのクライアント（CAL）ライセンス：

o Microsoft Windows Server 2003（StandardまたはEnterprise Edition）、

o Microsoft Exchange 2003 Server（StandardまたはEnterprise Edition）、

o Microsoft SQL Server 2000（StandardまたはEnterprise Edition）、

o Microsoft SQL Server 2005（StandardまたはEnterprise Edition）、

o Microsoft SharePoint Portal Server 2007または2003。



Microsoft Visual Studio 2005 Professional EditionおよびMSDN Premium Subscription Universal（開発およびテスト用の5ライセンス）。



5つのMSDNプレミアムサブスクリプションライセンスとMSDNプレミアムDVD。




これには何が必要ですか？

1.登録メンバーのレベルでマイクロソフトパートナープログラムに参加する

2.サイトを持ち、維持する

3. MSテクノロジーを使用して、少なくとも1つの複製ソフトウェア製品を開発する

4. 420米ドルの年会費（VATを除く）を支払います。



特定の条件が満たされた場合、参加はさらに1年間延長できます。



420ドルは、付加価値税を含めても、約11,000ルーブルです。 5つのWindows、Office、MS VS、SQL Server、SharePoint Portal、Exchange Server、および1つのサーバーオペレーティングシステムの場合-ほとんど無料です。



プログラムの担当者と連絡を取り合った時点で、私たちは終了します。このトピックが台無しにならない場合は、詳細についてお話します。
