ビジオ。 フルHD IPSスクリーンを備えた安価なラップトップ

クイックリファレンス：
VIZIOはかなり有名なアメリカの会社です。 2002年から存在します。 主に液晶テレビとモニターを製造しています。 最近、ウルトラブックおよびタブレット市場への参入が試みられました。 デバイスの開発は米国で行われています。 同時に、デザインだけでなく、すべての機器の内部構造も開発されており、インド人や中国人には何も外部委託されていません。 生産施設はメキシコ、中国、台湾にあります。 機械を製造するすべての工場は、VIZIO自体が部分的または完全に所有しています。 それは積極的なマーケティング政策によって区別され、ペニーの半奴隷労働を使用して、環境に害を与える世界最大のブランドを非難し、有能なアメリカのエンジニアと「最後のねじ」まですべてを開発するという方針を誇っています。 ロシア市場に参入する計画はありません。 VIZIO製品またはサービスセンターが当社に表示される可能性はゼロになる傾向があります。


したがって、単純なロシア国民は、米国からのかなり重いユニットの輸送コストと、ロシアの郵便に関連するリスク、およびVIZIO機器に精通するためにカリフォルニアでのみ有効な保証を負担する必要があります。



ここにそれがあります、それ自身のどれがすべてそのような典型的な金属のミニマリストのアメリカ人です。



このトピックに興味を持っているVIZIOラップトップは、それほどリッチではないモデル範囲で表されます。



Windows 7およびIntel Core i3-i5プロセッサー上の古いもの：

-CT14 A1-A4-SSDドライブと高品質のFullHD TNマトリックスを備えた薄くて軽い（1.54 kg）ウルトラブックとラップトップ

-CT15 A1-A4-薄くて少し重い（1.77 kg。）SSDドライブとFullHD IPSマトリックスを搭載したラップトップ

充填は、インデックスAの後の数に依存します-1から4に増加すると、プロセッサの周波数、SSDの量、およびメモリが追加されます。



Windows 8で新鮮：

-Intel Core i7上のCT15 A5-重量1.77 kgの薄型ウルトラブック、 256 GB SSDドライブ、およびFullHD IPSマトリックス

-2.3 GHz Intel Core i7上のCN15 A5（ただし、何らかの理由でAmazon 2.3 GHz Apple A4プロセッサーがAmazonに示されています） 。



インデックスが15のすべてのモデルには、対角線が15.6インチで解像度が1920x1080ピクセルの同じマットIPSスクリーンが装備されています。 画面は異なる場合がありますが、最も安いCT15-A1をタッチして、近くにある最も高価なCN15-A5と比較して、その違いが目に見えないことを主張します。 上記のすべてのリスクに関係なく、VIZIOを選択するという決定的な議論になったのは、この画面でした。 その結果、道徳的満足度の観点から、私は選択を失いませんでしたし、絶対に後悔しませんでした。これは、このような構成の最年少モデルCT15-A1に当てはまりました 。

誰もおもちゃで遊ぶつもりはありませんでしたが、宣言された構成の残りのタスクについては、目で十分でした。 メーカーは6〜3時間のバッテリー寿命を主張していますが、3.5〜4時間後、インジケーターは既にパニック状態で点滅し始めています。 一度だけ、明るさが最小限に抑えられた暗い部屋で本を読んだとき、充電は4時間半で十分でした。 Wi-FiでホームNASからFullHDリミックスを表示する場合、1時間半後に10％未満しか残りません。

しかし、技術仕様を見なくても、ラップトップはそのスタイリッシュさ、画面品質、全体的なビルド品質に感銘を受けます。 静かな部屋であっても聞こえない-私の標準的な使用条件では、ファンはごくまれに短時間しか作動しません（夏に何が起こるかはわかりますが..）。 そして、たとえそれを手に持っても、熱はほとんど見えません。 キーボードは少し残念です。 現代のほとんどのラップトップのように、それはある種のもろさの印象を与え、ボタンを押したままどこかに著しく押し下げます。 しかし、原則として、あなたはそれに慣れることができます、そして悲しいことに、それはあなたを怒らせません。 ロシアの手紙については、いくつかの中国の黄色のステッカーが50ルーブルで購入されました。これは全体像に完全に収まり、干渉せず、消去されず、今のところ剥がれません。

デバイスには組み込みのCD-DVDドライブはありませんが、ほぼすべてのタイプのフラッシュドライブからの読み込みがサポートされています。 それらの一部はUSB-FDDとして認識され、一部はUSB-Stickとして認識されます。 現在接続されているフラッシュドライブはBIOSに表示され、ブートの優先順位を目的のデバイスに切り替えることができます。 AHCIモードを無効にすることはできません。そのため、ラップトップにWindows XPをインストールすることは困難です。 到達可能なXPに基づくすべてのブートディスク（Ruslive、Steaなど）は、ハードドライブを認識しませんでした。 通常、USBフラッシュドライブにロールバックされたXalexから7枚と8枚の数字のライブCDイメージを読み込むことができました。



現時点では、このモデルはAmazonおよびe-bayで510〜620ドルの価格で購入できます。 それは、Sberbankルーブルカードとの外国為替取引に対する略奪的利益およびShipitoを介した最も高価な配送（すべての保険契約などを含む）とともに、約22,000の国内ルーブルに換算されます。 そして、これは、キーボードと不快な画面がさらに悪い、いくつかの非近交系の中国製プラスチックHPに匹敵します。
