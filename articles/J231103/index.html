システム管理者を祝福するために、オフィスの全員に注意を促します

ご存知のように、今日はシステム管理者の日ですが、私のオフィスでは誰もがそれを知っているわけではありません。 これを修正するために、同僚向けの小さな漫画リマインダーを作成しました。 Mikrotikの機器を使用しているため、すべてを実装しました。









しかし、アイデアは単純です。今のところ、忘れっぽい同僚がインターネットをサーフィンして、目的のサイトではなくリマインダーページを表示し、しばらくして金曜日の夕方に見たいものを表示する必要があります。



これらの目的のために、プロキシサーバーをMikrotikに上げます



/ip proxy set enabled=yes







次に、エラーレポートページをリセットします。



/ip proxy reset-html







彼らは私たちに尋ねます：



Current html pages will be lost! Reset anyway? [y/N]:







そして大胆に「y」を押します



次に、プロキシを通過するすべてのページをブロックするルールを作成します



/ip proxy access add action=deny disabled=no
