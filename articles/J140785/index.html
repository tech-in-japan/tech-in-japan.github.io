サマラ会議WebProfessionalsのスピーカーとのポッドキャスト

サマラのWeb開発者向けの会議の前夜、WebProfessionals.ruチームは会議の講演者と3つのポッドキャストを録音しました。 MicrosoftのWeb開発の専門家であるVladimir Yunev（ XaocCPS ）に、Web開発の世界の動向と、会議の講演者のスピーチで役立つ新機能について話を聞きました。 Drupalコミュニティの専門家で代表者であるArtem Pankovと、ロシアでのDrupalの成果と展望について話し合いました。 そして、ABBYYのWeb開発責任者であるEvgeny Agafonovとともに、ABBYYのWeb開発の問題、ASP.NET MVC 4およびWeb APIの革新について議論しました。

-開発の経験が少ない場合、誰のために意図されており、そこに行く価値がありますか？

-Windows Azure

-フレームワークMVC 4

-ビジュアルスタジオ

-マイクロソフトのスタッフとの仕事の領域

-マイクロソフト認定プログラム-それは何で、なぜ必要なのですか？

-2012年の開発者向けイベントとカンファレンス

-新しい市場の見通し-何に注意を払う価値がありますか？

スピーカー情報


ウラジミール・ユネフ。 Microsoft Web開発テクノロジーの専門家。 Microsoft ASP.NETプラットフォームでのWeb開発のスペシャリスト。 彼は、ASP.NET MVCテクノロジーに関する本を共同執筆しました。 技術会議に定期的に参加し、Web開発に関するプレゼンテーションを行います。 2009年と2010年に、彼はMicrosoft MVPのステータスを授与されました。 2011年2月以来、彼はMicrosoftで「Web開発技術の専門家」として働いています。



アルチョム・パンコフ。 2008年に彼はMSTUを卒業しました。 N.E. バウマンとDrupalの開発に特化した会社「ビジネス向け情報システム」を設立しました。 Drupalコミュニティの専門家、DrupalConf会議主催者、ロシアDrupal Associationイニシアチブの著者。 彼のリーダーシップの下、大統領行政、メガフォン、連邦独占禁止サービスなどのために、Drupalでプロジェクトが作成されました。



エフゲニー・アガフォノフ。 モスクワ州立大学化学科卒業。 彼は2004年からABBYYに在籍しており、現在はLingvoのWeb開発の責任者です。 2005年以降のMicrosoft ASP.NET MVP、MCP。 専門知識-Windows 8用のWebアプリケーション、Azureアプリケーション、Metroアプリケーションのアーキテクチャと開発。
