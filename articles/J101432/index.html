2マウス= 2カーソル

かつて、2つのマウスと1つのカーソルがあるのはなぜですか？ そして、解決策を探し始め、それを見つけました。 それはSpnmouseのプロジェクトでしたが、2004年に放棄されましたが、私たちを少し傷つけませんでした。



パート1準備



マウスのドライバーについては、 Sourceforgeにアクセスしてください。



パート2、実用的



最初に必要なことは、デバイスのVIDとPIDを調べることです（USBマウスの場合）。 これを行うには多くの方法があり、基本を使用します。



1.スタート->プログラム->アクセサリ->ユーティリティ->システム情報。

2.コンポーネント->入力->ポインティングデバイスを選択します

3.最初のマウスとそのVIDおよびPIDを見つけます。



4. 「％CPNGroup.DeviceDesc1％= usbinst、HID \ VID_ XXXX＆PID_ YYYY」という形式の行をcpnmouse.infファイルに追加します。ここで、それぞれXXXXおよびYYYY番号がPNPデバイスIDに示されています 。

<img title = "" border = "0" alt = "" src = " habrastorage.or

5. cpnmouse.sysとcpnmouse.infを保存して、それぞれc：\ windows \ system32およびc：\ windows \ infに ドロップします。

このファイル操作のすべてが完了しました。インストールしてみましょう！

6.コントロールパネル-マウス-機器-非表示デバイスを選択



7.更新をクリックします。

8.インストール済みのリストからドライバーを選択します



9.次に、 CPN Toolsマウスフィルタードライバーを選択します



10.ここで、 cpnmouse.sysファイルの場所を示すように求められます。c：\ windows \ system32フォルダーにあります。

11. [ OK]をクリックすると、マウスのインストールが完了します（PS / 2マウスがない場合は、すぐに最後の部分に進みます。

12. ps / 2の場合、マウスの方が簡単です。コントロールパネル-マウス-ハードウェア-ps / 2マウス-ドライバを更新してから、パラグラフ7〜11。

おわりに



それだけです、マウス用のドライバーはインストールされていますが、何かが欠けています。 そして、2番目のカーソル自体がありません。 ordinary.exeファイルを開始し、2つのカーソルで喜びます。
