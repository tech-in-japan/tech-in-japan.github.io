Code for Americaを活用した5つのソーシャルアプリ

Code for Americaは、ICT、自治体職員、プログラマー、市民活動家を結びつけることにより、自治体サービスの改善を支援するプラットフォームです。 主な製品：1）市民と役人のより迅速かつ効率的なコミュニケーションを可能にする、2）当局の透明性と責任を高める、3）パブリックドメインにあるため、他の都市や自治体で使用できるWebアプリケーション。



作成されるアプリケーションのオープン性の結果は、米国当局だけがそのようなアプリケーションを使用できないことです。 これが何であるか理解していますか？ :)誰でも使用できます！ このようなオープンソースプロジェクトのレビューが役立つと考えました。



Code for Americaの仕組みについて少し


Code for Americaアルゴリズム：

  1. 市の行政と連携-Webアプリケーションの解決に役立つ緊急の問題を特定し、アプリケーションを作成するためのアプリケーションを公開します。
  2. 請負業者の競争力のある選択;
  3. 顧客と請負業者は通信し、11か月で一緒にアプリケーションを作成する必要があります。
  4. プロセス全体を通して、Code for Americaのスペシャリストがトレーニング、会議を実施し、開発プロセスの両側にアドバイスを提供します。




その結果、政府と社会の間の効果的なコミュニケーションは、都市の顔をより良く変えるアプリケーションの出現の触媒となっています。 そして、アメリカ人だけでなく、注意することが重要です。



5つのオープンアプリケーション

Iconathonは、以前は街になかったオブジェクト（屋根の庭、自転車のワークショップ、ケーブルカーなど）を指定する新しい標識のデザインをユーザーが考案できるポータルです。



都市インフラを改善するためのアイデアを集めることで、 Change By Usプラットフォームが可能になります。 市民は、アイデアやリソースを交換したり、時間と知識を提供して一緒に前向きな変化を起こすことができます。



Muralappを使用すると、市民はストリートアートを知ることができます。 今では、スマートフォンを使用して、アートオブジェクト（落書き、記念碑、インスタレーション）を見るだけでなく、誰が作者で、何を言いたいかを知ることもできます。



Rerouteは、ユーザーが目的地に到達するまでにさまざまな方法（徒歩、自転車、バス、タクシー、または自分の車）で費やすことができる時間、カロリー、およびお金を比較できるアプリケーションです。 おそらく今、より多くの人々が環境に優しいモビリティを選択するでしょうか？
