ThinkPad T410sでの旅行

過去30年にわたり、外出中に家族、友人、同僚と通信する多くの方法が変わりました。 前世紀の80年代に、あなたが首尾よく届いた電報を送ることは可能でした。 90年代に-より手頃な都市間を呼び出したり、ページャーにメッセージを送信したりします（そのようなことを覚えていますか？）。 2000年代には-最寄りのインターネットカフェから手紙を送ります。 しかし、私は現在のほうがはるかに好きです。なぜなら、ロシア（そして一部は世界中）を旅行するとき、私はほとんどいつでも連絡を取り合うことができるからです。 大都市には、WiFiアクセスポイントとWiMAXネットワークがあり、インターネットに接続できるモバイル通信が進行中です。 ただし、コミュニケーションは始まりにすぎません。 最新のラップトップのパフォーマンスは、メールのチェックからビデオの編集まで、ほぼすべてのタスクを実行するのに十分です。



レノボのラップトップを路上で使用するのがどれほど便利かを評価することに決めたので、私たちはいくつかのThinkPadシリーズのモバイルコンピューターを連れて短い車での旅行に出かけました。 この記事は、3Gインターネット、ナビゲーション、写真とビデオの処理、そして最後に友人や同僚とのコミュニケーションなど、最新の技術に関するものです。









今年の初めに新しいThinkPadが発売されたことで、このような「接続された」旅は、追加のデバイスを必要としなくなりました。 ほとんどすべてのビジネスノートブックモデルは、3G通信に加えて、GPS衛星からの信号受信を提供するQualcomm Gobi 2000モデムを備えたバージョンで利用できます。 Intel Core i5およびi7プロセッサは、単純なタスクは言うまでもなく、基本的なビデオ編集でも十分なパフォーマンスを提供します。 同時に、新しいノートパソコンのシリーズのバッテリー寿命は長くなりました。



ラップトップとアクセサリー



そのため、次の構成の新しいThinkPad T410sを自由に使用できました。



プロセッサー ：Intel Core i5-520M 2.4 GHz、最大2.93 GHzのターボブーストモード

RAM ：3 GB DDR3

ストレージ ：128 GB SSD

画面 ：14.1インチ1440x900マット

ビデオカード ：NVIDIA Quadro NVS 3100M 512 MBまたはIntel GMA HD（切り替え可能）

光学ドライブ ：DVD-RWリムーバブル

ワイヤレス接続 ：WiFi / WiMAX 802.11bgn、3G + GPS

コネクタ ：ExpressCard / 34、SDHCカードリーダー、2xUSB 2.0、1xUSB / eSata、VGA、DisplayPort、イーサネット、FireWire、オーディオジャック（ヘッドフォンとマイクに共通）

オプション ：指紋スキャナー

寸法 ：337x241.5x21.1-25.9 mm

重さ ：1.79 kg

小売価格 ：6万ルーブルから







昨年、Sプレフィックス付きのThinkPad Tラップトップが初めてリリースされ、2010年初頭に、Intel Core iシリーズプロセッサを搭載したアップデートバージョンが登場しました。 ThinkPad T410sの重要な機能は、その最小重量です。このパラメーターでは、よりコンパクトな12インチのThinkPad X201に匹敵します。 同時に、真に明るいディスプレイと統合された光学ドライブを備えた強力な構成を自由に使用できます。



この変更には128ギガバイトのSSDドライブが装備されており、このおかげですでにこの優れたマシンでも非常に高速になっています。 これは、MS Outlook 2010でメールを操作する場合に特に顕著でした。新しいメッセージT410を検索および受信するときの10ギガバイトの文字データベースは、通常のハードドライブを搭載したX201よりもはるかに高速に「裏返し」ました。 ところで、このモデルは1.8インチのデータストレージ（通常のハードドライブまたはSSD）のスペースを提供します。 したがって、T410sの最大HDD容量はたった250ギガバイトです。







しかし、数百ギガバイトのデータを持ち込むファンは、このモデルの別の機能を利用できます。 実際、T410sの標準光学ドライブは取り外し可能であり、必要に応じて、通常の2.5インチハードドライブ用の特別なアダプターと交換できます。 途中で私にとってより重要だったのはデータの量ではなく、バッテリーの寿命だったので、DVDドライブは追加のバッテリーに交換されました。







このような3セルバッテリーは、1.5〜2時間のバッテリー寿命を追加し、充電せずに最大7時間動作することができます。 実際の状況では、常時接続されている3GモジュールはWiFiよりもやや大食いで、約5時間の仕事がありました。







まだ車で旅行していたので、途中のラップトップは車載電源から充電する必要があります。 レノボのThinkPad Notebook PCにはターンキーソリューションがあります：ポータブル90ワットコンボ充電。 これにより、ラップトップは従来のネットワークから、車の「シガーライター」を通じて充電できます。 さらに、充電ケーブルには、携帯電話またはポータブルデバイスを充電するための追加のUSBコネクタがあります。



車



この旅行のために、テスト用に2リットルの140馬力のディーゼルエンジンを搭載した快適でモダンな9人乗りのミニバン、 フォルクスワーゲンカラベルを利用しました。 私たちが最初に会ったとき、車はただ巨大に見えました。







しかし、文字通り15分間の旅行の後、大型SUVの私の運転スキルがここで有用であり、一般にミニバンの寸法が干渉しなくなりました。 詰まったモスクワやサンクトペテルブルクの狭い通りに駐車することで問題が発生しない限り、ここでは通常の駐車アシスタントが非常に役に立ち、リアとフロントの両方のバンパーにセンサーがあります。







しかし、そのような車の利点はすぐに明らかになりました。 高い着陸、視界の良さにより、14時間の長時間のライドでも快適です。 しかし、何よりも、キャビン内の配置の自由に感銘を受けました。これは普通の乗用車では簡単に達成できませんでした。 にぎやかな通りで運転手のドアを離れたくない場合は、立ち上がって横のドアに行き、歩道に直接行くことができます。 また、快適な後部座席により、「キャラベル」を広々としたモバイルオフィスに変えることができます。







「カラヴェル」の最も近い親Theは、貨物フォルクスワーゲントランスポーターです。 しかし、旅行ではそれは感じられませんでした：かなり大きな車のダイナミズムはほとんど乗客です。 6速のギアボックスに慣れる必要がない限り、交通渋滞では頻繁に切り替える必要があり、この車の場合はマシンが明らかに望ましいです。



ナビゲーション



内蔵のGPSレシーバーは非常に簡単に機能し、Google Earthからロシアの詳細な地図を備えたGarmin Mobile PCまで、すべてのナビゲーションプログラムで「理解可能」です。 モジュールの最初の起動は、独自のLenovo GPSユーティリティを使用して実行されます。







起動時に、GPSレシーバーには仮想シリアルポート番号が割り当てられます。これは、ナビゲーションプログラムで指定する必要があります。 Google Earthでは、設定はまったく必要ありません。プログラムは受信者を自動的に検出します。 多くのユーザーは、レシーバーのかなり「遅い」起動に注意しますが、大きなアンテナは、市内、高速道路、および一般的にどこでも信頼できる受信を提供します。 モスクワからサンクトペテルブルクまでのルート全体が理解できるので、最も単純なナビゲーションソフトウェアである古い「GIS Russa」を使用して、旅行統計を表示し、トラックを記録しました。



エンターテインメント



VW Caravelleには、タッチスクリーンとセキュアデジタルメモリカード用のスロットを備えた非常に強力なマルチメディアシステムがあります。 フロントパネルのライン入力にラップトップケーブルを接続することで、最も簡単な方法で使用しました。 実際、音楽ライブラリ全体が近くにある場合、モバイルコンピュータのSSDで音楽の一部をメモリカードに転送するのはなぜですか？ 再生には、Windows Media Centerが使用されました。これは、外出先でトラックとアルバムをすばやく切り替えることができる大きなキーを備えた便利なインターフェイスがあるためです。 ちなみに、ThinkPad T410sにはタッチスクリーンを備えた変更があり、音楽の再生を制御することは非常に簡単になります。



写真およびビデオ処理



強力なIntel Core i5プロセッサーにより、写真、基本的なエディター、ビデオコンバーターを快適に操作できます。 写真はAdobe Photoshop Lightroomで処理されました。これには、RAW画像を処理してJPEGに変換する強力なプロセッサと、多くの写真のプレビューを作成および処理する高速ディスクシステムの両方が必要です。 ここでSSDは最高であることが証明され、作業速度が大幅に向上しました。



いわゆる「高速化された」ビデオを処理するために、無料のVirtualDubとMicrosoft Expressionが使用されました。 後者はWindows Mediaエンコーダプログラムに置き換わり、新しいインターフェイスと基本的なビデオ編集のための追加機能を獲得しました。 ここでは、プロセッサの電力が最も重要であり、前世代のモデルと比較して、Intel Core i5は大きな利点を提供します。 どのように正確に-あなたはテストで同様のモデルThinkPad T510を15インチのスクリーンでウェブサイトRetera.ruで見ることができます



インターネット



Gobi 2000モジュールのGPS部分に質問がありましたが、主な目的のために、この鉄片はほぼ完璧に機能します。 特別なことは何もありません。WiMAXに切り替えるとき、モスクワの環状道路近くのガソリンスタンドでモバイルインターネットがオンになり、サンクトペテルブルクで既にオフになりました。 ある基地局から別の基地局への移行、通信の短期的な不足は、モデムによって完全に解決されました。 途中や停留所では、いつでも友達とチャットしたり写真を共有したりできました。



ヨタはサンクトペテルブルクでWiMAXを使用していましたが、この会社はさまざまな都市の透過的なアクセス支払いシステムに特別な感謝を必要としています。WiMAXを数日間有効にするとモスクワとサンクトペテルブルクの両方で機能します。 WiMAXを使用して、ビデオをYouTubeにアップロードし、一般に自宅と同じように自由にインターネットを使用しました。 いずれにせよ、旅行の最も重要なタスクは解決されました。私たちは旅行中ずっと連絡を取り合っていました。



安全性と信頼性



ThinkPad T410sは、快適さと携帯性をうまく組み合わせています。 長い間ThinkPadラップトップファンは、X201がこのシリーズの移植性のベンチマークであることを知っています。 同じ重量（プラスまたはマイナス200グラム）のT410は、所有者に、より大きなディスプレイ、より快適なタッチパッド、および追加のデバイス（外部ディスプレイ用のHDDとDisplayPortを接続するためのeSata）を接続するためのオプションを提供します。







最小の厚さにもかかわらず、T410sは信頼性を確保するためにLenovoの開発をすべて使用します。 マグネシウム合金ケースに加えて、スクリーンとマザーボードは複合材料で作られたスペースフレームで保護されています。 キーボードの設計は、誤って液体をこぼさないように電子機器を保護します。特別なチャンネルを通過し、ラップトップの底面の穴を流れるだけです。 全体的なセキュリティレベルを高める多くの小さな改善があります：特別な形状の底面のゴム製の足、衝撃を緩和すること、およびマザーボードの物理的な負荷を減らすハードドライブ（この場合はSSD）の可動留め具まで。



そのようなラップトップはオフィスで最も頻繁に使用されますが、やや難しい条件でテストしました。 14時間の長い旅行で、ラップトップは絶えず手を替え、インターネットへのアクセス、オフィスプログラムでの作業、写真やビデオの処理に使用されました。 バックグラウンドでは、GPSナビゲーターと音楽プレーヤーが常に機能していました。 ThinkPad T410sの安定性と信頼性は、実際に実証されています。 しかし、2番目のトリップラップトップであるThinkPad X201では、さらに深刻なテストを行う必要がありました。 しかし、それについては次の記事で詳しく説明します。 ちなみに、X201クラッシュテストビデオは既にLenovoのFacebookページに投稿されています 。
