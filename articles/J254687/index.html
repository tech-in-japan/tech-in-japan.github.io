イースタービルドVivaldi 1.0.142.32









みなさんこんにちは！



今日、多くの企業がエイプリルフールの抽選に取り組んでいますが、冗談を言う時間はありません。この時間を利益のために費やすことに決め、ブラウザの改善に取り組んでいます。



以前、Technical Preview 3の新しいバージョンを本日提供すると約束しましたが、内部テストのサイクル全体を完了するために1週間延期せざるを得ませんでした。 引き換えに、Vivaldiの別のテストビルドをリリースすることにしました。本日、興味深い番号1.0.142.32でリリースしました。



このアセンブリでは、グループ化されたタブの表示機能を希望に応じて改善することに取り組んでいます。 新しいタイルタブスタック項目がタブグループのコンテキストメニューに表示され（このメニュー項目はまだロシア語に翻訳されていません）、グループページを1つのウィンドウに表示できます。 上のスクリーンショットに示されているとおり。 デフォルトでは、すべてのページが縦に並んで配置されますが、画像表示制御ボタンの隣のステータスバーにある特別なボタンを使用して表示方法を変更できます（右の図を参照）。



また、最も一般的な機能の1つであるマウスジェスチャも引き続き使用します。 このアセンブリでは、リンクを新しいタブで開くことができる新しいジェスチャを追加しました。リンクにカーソルを合わせ、右ボタンを押してマウスを下に移動するだけです。 マウスジェスチャの作業を続けます-創造性の範囲は広いです。



フォーラムでユーザーが頻繁に取り上げる別のトピックは、バックグラウンドタブでのコンテンツの読み込みの視覚化の改善です。 今日は、この作業がどのように進行しているかを示すことができます。 まず、ページ読み込みの進行状況のインジケーターを置き換えました。アドレスバーにアクティブページを読み込むインジケーターである「ビッグブラザー」とより統一され、タブのスパインの下部にバーとして表示されます。 2番目に、完全にロードされたタブには「カットオフ」コーナーがあります。そのため、バックグラウンドで開いているタブのどれが表示可能かがわかります。 おそらくこれは理想的な解決策ではなく、より良いアイデアがあります-喜んで耳を傾けて、おそらくあなたのアイデアは将来のバージョンで実装されるでしょう。



今日は以上です。 以下のリンクを使用して、新しいアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
