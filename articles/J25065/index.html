XPの微調整：一時ファイルとキャッシュをRAMディスクに転送します

ここのこのコメントに触発されて 、私はすぐに計画を実行することにしました。

そこで、著者自身が言うrramdisk実装を取り上げます。

CenatekのRamDisk9xME、RAMDiskXP、RAMDiskSE、RAMDiskVE、SuperSpeedのRamDisk、RamDisk Plus、RAMDisk Enterprise Lite、フルバージョンに加えて、以下のQSoftのMicrosoft Ramdiskに基づく64 MBの無料RAMディスク、Ramdiskなど、Ramdiskドライバーとソフトウェアが豊富に用意されていますMicrosoftのWindows 2000用の.sysドライバーはKB257405で公開されています。 ただし、これらのRamdiskドライバーのほとんどは、無料ではないか、その機能、特にRAMディスクのサイズに制限があります。 Windows Vistaをサポートしていないものもあります。

もともとMicrosoftのRamdisk.sysに基づいてGavotteによって作成され、後でlyh728によってGUIインターフェイスが追加された以下のフリーウェアRRamdisk.sysは、無料で使用できるだけでなく、安定しており、サイズ制限がなく、人気のFAT16、FAT32をサポートします、NTFSファイルシステム、およびVistaを含むWindows 2000オペレーティングシステム以上をサポートしています。



インストール。 プロセスは非常にシンプルで簡単です。 rramdisk.exeを実行し、属性を設定します。







私の場合、128 MBのディスクサイズ（1 GBのRAMがあります）を選択しましたが、その他はすべてそのままにしました。

次のタスクは、最も頻繁に使用される一時ファイルをRAMディスクに転送することです。







％TMP％および％TEMP％システム変数パス



•[マイコンピュータ]-> [プロパティ]-> [詳細設定]を右クリックします。

•[環境変数]ボタンをクリックします

•ユーザー-％TMP％および％TEMP％のユーザー変数を先頭で削除します（これらの値はシステムの値よりも優先されます）

•システム変数％TMP％および％TEMP％をR：\ Tempに変更します。次のようになりました。





•再起動し、RドライブとTEMPパパを確認します。



一時インターネットファイル



•IEを起動し、[ツール]-> [インターネットオプション]-> [全般]に移動します。

•インターネット一時ファイルの中央のセクションで、右端の[設定]ボタンをクリックします

•[フォルダの移動]をクリックし、R：\ Temporary Internet Filesを選択して、[適用]をクリックします。 ドライブRの半分に等しい制限を選択しました。

•IEはフォルダを転送し、プロセスを完了するためにログインを要求します。



Firefoxキャッシュ



デフォルトでは、FFはキャッシュを％Local Settings％\ Application Data \ Mozilla \ Firefox \ Profilesフォルダーに保存しますが、この動作は次のように変更できます：config：

•FFを実行し、アドレスバーに「config：config」と入力します。

•右ボタンで任意の場所をクリックし、[新規]-> [文字列]を選択します

•browser.cache.disk.parent_directoryをパラメーター名として設定し、キャッシュを値として保存するフォルダーへのパスを設定します。 r：\アプリケーションデータ\ firefox

•FFを再起動する



Microsoft OneNoteキャッシュ



私の意見では、最も曖昧なステップです。 私はこのプログラムを積極的に使用していますが、今でもこれらの行を記述しています:)

•OneNoteを起動し、[ツール]-> [オプション]-> [保存]に移動します

•下部の[キャッシュファイルの場所]セクションで、[変更]ボタンをクリックし、新しいパスを選択します。 このRがあります：\アプリケーションデータ\ OneNote

•OneNoteを再起動する



もちろん、それは自己催眠かもしれませんが、すべての操作の後、UIはより迅速に応答し始めたように思えました。



ただし、考えられる不利な点を理解する必要があります。

•ブラウザのキャッシュは再起動するたびに削除され、追加のトラフィックが消費されます。

•予期しない停電/再起動が発生した場合、冒険があります。 私のラップトップではそれほど怖くはありませんが、それでもなお。
