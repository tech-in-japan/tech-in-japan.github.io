ITキャンプまたはクラウドは何とか何とか！

こんにちは、habratishniki！



最近、怠technologiesな人だけがクラウドテクノロジーとその有効性について話していません。 しかし、クラウドの有効性と使いやすさについての具体的な質問については、固体の「何とか何とか」よりもわかりやすい何かを聞くことはほとんどありません。







麺を垂らさずにこれらの質問に答えるために、モスクワ、サンクトペテルブルク、エカテリンブルグ、トムスク、カザンで開催される技術セミナー「Clouds blah blah blah！」を開催します



イベントでは、プライベートクラウドの概念、System Center 2012を使用した実装、およびクラウドテクノロジーの使用に関するいくつかの特定のシナリオが考慮されます。

  • 「プライベートクラウド」とSystem Center Virtual Machine Manager 2012の概念を使用してITインフラストラクチャリソースを効率的に割り当てる方法。
  • System Center Operations Manager 2012を使用してビジネスアプリケーションを監視する方法
  • System Center Configuration Manager 2012を使用してアプリケーションライフサイクル管理を実装する方法
  • System Center AppController 2012を使用してハイブリッドインフラストラクチャを管理する方法


さらに、Windows Server 8の新しいHyper-V機能の概要が提供されます。



イベントのプログラムには、デモンストレーションと実際の実験室作業の両方のレポートが含まれます。 参加は無料です。 事前登録が必要です。
