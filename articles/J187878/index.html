B1ベーシスバンド：バイオレジストウォッチ。 健康的なライフスタイルのゲーミフィケーション（ロシア語での最初のレビュー）

おそらく、ITスペシャリストの夢は、生活を自動化し、他のすべてを監視することです。 世界のすべてはまだ遠いですし、多くのバイオインジケーターを記録し、処理と分析のために「クラウド」に送信できる時計が既にあります。最近、それらを購入する機会がありました。



。



CES-2012の候補の1つであるGadget B1 Basis Bandは、人の心拍数、肌の水分、動き、温度を測定するセンサーを使用して健康を監視する最初の携帯型バイオレジストレーターです。 彼らは睡眠に費やした時間を監視し、消費カロリーの数を数えます。 USBおよびBluetooth経由のデータは、コンピューターまたは携帯電話に転送され、オンラインプロファイルに入力されます。 将来的には、このバイオレジストレーターを簡単にするために時計と呼びます。 さらに、彼は本当に見ています。



B1ベーシスバンドは、Fitbit、Jawbone、Nike Fuelbandのラインを引き継いでおり、ポータブルデバイスにはまだないまったく新しい機能に慣れ親しんでいます。 購入にお金の価値はありますか？







MyBasis.comで注文した後、このボックスに時計が到着しました。











注文時から米国の荷送人への約2日間、および速達によるロシアへの配達に約2週間。 合計費用は、時計自体が199ドル、送料が73ドルです。 比較のために、ロシアのJawbone Upは、機能が大幅に弱い5〜7千ドルです。つまり、Basis B1バンドは非常に高価です。



開始するには、時計をインターネット経由でアクティベーションする必要があります。 これに先立ち、LCDにはmybasis.com/goに移動する指示が表示されます。 アクティベーションプログラムは、ラップトップにドライバーをインストールし、クロックをアクティベートしました。







時計をUSBポートに接続するか、Bluetooth経由でAndroidスマートフォンに接続すると、ガジェットは体温、心拍数、肌の水分、歩数計の測定値の変化のログを「クラウド」にダンプします。 時計自体は、Windows / Mac / Androidのアプリケーションと同様に非常に「ダム」です。すべての「スマート」機能は、サイトのアカウントで使用できます。 収集した情報から多くの知識を引き出すことができ、ソフトウェアを絶えず更新するのは不便であり、「クラウド」にはさらに多くの容量があります。



時計には、非常にシンプルなブックレットとクレードルが含まれています。







ところで、時計に時間設定機能がないのは面白いです。 どこにもありません。 プログラムにも時計自体にもありません。 インターネットがなければどれだけの時間を使えるのかわからないからといって、インターネットがなければ役に立たず、ネットワークにアクセスするときはサーバーと時間を同期します。 登録すると、タイムゾーンが示されます-これで十分です。



クレードル

ここで時計を充電するためのオリジナルのクレードルです。 手から外さずに時計に装着できます。











クレードルは時計の上から上に置かれ、優しくスナップします。手から時計を取り外すことはできません。 あまりきつくないので、急な動きの場合は簡単に外すことができます。この場合、クレードルに損傷を与えたり、ラップトップをテーブルから引き離したりするリスクが軽減されます。



設計

もちろん、時計のデザインは議論の余地があります。 彼は非常に簡潔で、少し時代遅れです。 これにはいくつかの魅力があります。 しかし、私は時計が大きすぎるとは言いませんし、確かにそれらは重くないです。 時計の重さは40グラム強です。



Android / iOS用のソフトウェアは、個人アカウントのデザインと同様、非常に快適です。 インターフェイスが慎重に作成されたことがはっきりとわかります。



自分自身を見る

ストラップ

ストラップはソフトタッチのプラスチックでできています。 キットには、1色のストラップが含まれています。 時計を購入した時点では、黒いストラップのモデルのみが注文可能です。 著者のデザインで追加のストラップを購入できます。 時計のストラップのラッチはそれほど信頼できるようには見えませんが、自然に解放されませんでした（ネットワーク上の人々が何が起こるかを書きます）。



スクリーン

時計の情報は、1インチの半透過型LCDスクリーンに反転表示されます。 他のLCDと同様に、太陽は薄れ、バックライトは非常に経済的で、読み取りが弱いです。



ちなみに、これはどこにも書かれていませんが、手を上げてどれくらいの時間を見るか、ほとんどの場合、時計はこのジェスチャーを理解し、数秒間バックライトをオンにします。



前面には、1インチのLCDスクリーン、端に4つのタッチボタン、右側に1つの機械式ボタンがあります。 タッチボタンは、1）バックライト2）曜日に切り替える3）上にスクロールする4）下にスクロールする役割を果たします。 心拍数（パルス）、ステップ（歩数計）、カロリー（カロリー）は、スクロールによって切り替えられます。 自分のアカウントでのみ、時計の画面から体温と肌の水分を見ることができません。 サイドボタンを押すと、Bluetoothセットアップメニューと[今すぐ同期]項目が開きます。 Bluetooth経由の同期は、Android / iOSベースの電話で実行されます。



時計モードの画面には、実際の時刻、Bluetoothへの接続のサイン、バッテリーの充電、進行状況バーが表示されます。



時計を回す

時計の裏にはバイオセンサーがあります。 真ん中には、常に手に輝く2つの緑色のLEDがあり、その間に反射を分析するセンサーがあります。 これはパルスを読み取るのに十分です。 端のセンサーは、皮膚の温度と湿度に関する情報を収集するように設計されています。











ちなみに、時計内の心拍センサーの存在は、このガジェットの際立った「チップ」の1つです。 そのようなものはどこにもありません。 心拍数を測定できる他のフィットネス時計では、センサーを心臓に保持する特別なストラップを胸に置く必要があります。 これはトレーニング中に意味がありますが、日中は意味がありません。 B1ベーシスバンドウォッチは、所有者がそれを手放すまで、一日中ずっとこの情報を収集するという事実によって区別されます。



バッテリー

Bluetoothをオンにすると、平均的なバッテリー寿命は数日であるようです。 充電の利便性により、これは非常に受け入れられます。 バッテリー残量低下の警告の後、充電を見つけるのに約半日かかります。



歩数計

歩数計では、3軸ジャイロスコープが時計に組み込まれているため、精度が非常に高くなります。



ソフトウェア

私が書いたように、すべての「数学」は「クラウド」にあります。 つまり、蓄積されたデータは時計によってサーバーに送信され、そこでインタラクティブな図の形で視覚化され、グループ化、分析され、システムはセンサーからの特性データに従ってアクティビティをグループ化しようとします。



睡眠追跡

たとえば、30分間歩いてからコンピューターに座ると、時計（より正確にはサーバーシステム）は、歩行期間の開始時と終了時を認識します。 あなたが彼らと一緒に寝ると、時計はベッドの上に横たわっている人だけでなく、むしろ眠っていることを理解します-脈拍、皮膚の水分、動きがよりまれになります。 夢が不均一で、時計の所有者が夜に数回目覚めた場合、彼らはそれを「むち打ち」、朝のレポートに表示します。 睡眠時間は数分単位で正確に記録され、睡眠中の平均心拍数も表示されます。 この情報およびその他の収集された情報に基づいて、睡眠の質の指標は0から100のスケールで計算されます。これをすべて理解する必要がある理由を理解するだけです。



オンラインサービス

既に述べたように、Windows / Mac / iOS / Androidのアプリケーションは、クロックから（bluetoothまたはUSB経由で）製造元のサーバーにデータを転送し、そこで計算、分析、レポートの作成などを行います。



最大の利点は、データがクラウドにあることです。 彼らは、オペレーティングシステムを再インストールまたは変更することを恐れていません。 私の場合のように、Ubuntuがサービスと同期するためのソフトウェアをリリースしていなくても怖くありません。私の携帯電話が時計を理解していれば十分です。 自宅、職場、外出先の電話など、どこからでもチャートを見ることができます。 とても便利です。



収集されたデータは非常にうまく視覚化されています。 これらはさまざまなスケールで検討することができます。これにより、過去の期間に対する昨日の傾向を評価し、翌日の計画時に考慮することができます。 このようなフィードバックは、自己規律に非常に役立ちます。







システムは、アクティビティのタイプごとに自動的にグループ化します。 たとえば、彼はいつ夢が始まり、いつ終わり、いつ散歩が始まり、いつ終わるかを理解しています。 正確な数字と時系列のすべてがアカウントに表示されます。







また、データはヒートマップの形式で表示できます-ゾーンが暗いほど、対応するインジケーターの値が高くなります（下の写真-パルス）：







健康的なライフスタイルのゲーミフィケーションと非常に興味深いアイデア。 たとえば、何らかの目標が規定されているカードのうちの1つを選択できます。このカードが満たされると、そのカードは「合格」とみなされます。 目標の例は、昼食前に毎日走る習慣を形成することです。 または、1日あたり少なくとも10,000ステップを実行します。 または、特定の時間までに眠りにつく。







ゲーミフィケーションの主な用途はまだ来ていません。現在、同社は最初の一歩を踏み出しています。 しかし、アイデア自体は素晴らしいです。



システムは定期的なイベントを分析することができ、いくつかの「カード」は政権の遵守に関連しています。 たとえば、寝入ったり、同時に目を覚ましたり、定期的に時計を着用したり、定期的に運動したりします。







モバイルアプリ

モバイルアプリケーションを使用すると、Bluetoothと3G / Wifiを介して時計から「クラウド」にデータをプッシュできます。 デフォルトでは、同期は30分ごとに行われます。別の間隔を設定するか、いつでも手動で開始できます。 サーバーが更新されたデータを受信すると、すべてのグラフとレポートはすぐにそれらを考慮します。











まだではないが、私はとてもしたい

彼らは、指定された時間に目覚めることができるスマートアラームについて多くのことを話します。それはアクチグラフィの方法を使用します-加速度計によって睡眠の段階を決定します。 Habréでは、同様の経験のレビューがありました。 個人的には、この方法が機能することには大きな疑念がありますが、時計を購入した後、いくつかの人々がすでにこの機能について質問しています。



たぶん5年後には、製造業者は測定された指標の数に血圧の別の測定値を追加できるようになります。実際に気づいた場合は、ECGとEEGになります。
