FLACをMP3に変換する

私はいつも良質の音楽を聴きたいです。 しかし、残念なことに、多くのデバイスはロスレス形式をサポートしていません。 カーラジオとプレーヤーのflac形式をmp3形式に変換する際に問題が発生しました。

FLAC（英語無料ロスレスオーディオコーデック）-ロスレスオーディオデータを圧縮するために設計された人気のある無料コーデック。



非可逆圧縮（MP3、AAC、WMA、Ogg Vorbis）を提供するオーディオコーデックとは異なり、FLACは他のロスレスコーデックと同様に、オーディオストリームから情報を削除せず、高品質のサウンド再生機器で音楽を聴くのに適しています。オーディオコレクションのアーカイブ用[ ウィキペディアのFLAC ]


ソリューションを検索し、ubuntuについて説明します。 これに関する多くのスクリプトをインターネット上で見つけることができますが、多くにはタグの問題などの欠陥があります。 さらに、キューファイルを含むflacファイルをmp3トラックに変換する簡単なソリューションについて説明します。 これを行うには、flac、lame、cue2tracksを使用します。



設置



flacをインストールする 


sudo apt-get install flac


ラメをインストールする 


 sudo apt-get install lame


cue2tracksをインストールする


cue2tracks-キューファイルの情報を使用してディスクリッピングファイルに侵入するためのツール。 プロジェクトページ。

パッケージをダウンロードして（ https://code.google.com/p/cue2tracks/downloads/list ）、パッケージをインストールします。



変換



すべてがシンプルです： 

 cue2tracks -c mp3 -Q 0 -B 320 -MC -o " %N - %t " -R file.cue




起動オプション


-c mp3-コーデックをmp3にインストールします。他のコーデック（wav、flac、flake、ape、wv、shn、ofr、m4a、ogg）も使用できます。

-Q 0-最高品質を設定します。

-B 320はビットレートを320に設定します

-MCは固定ビットレートを設定します

-o "％N-％t" -出力ファイル名のテンプレート、この番号は順番に-曲の名前（05-Someone Like You .mp3）

-R仕事を始める

file.cue-キューファイル



出力ファイルの命名スキーム


％A-アルバム名

％P-アルバムパフォーマー

％D-アルバムのリリース日

％G-アルバムのジャンル

％a-ディスク番号

％t-トラック名

％p-トラックパフォーマー

％g-トラックのジャンル

％n-トラック番号

％N-最初の0からのトラック番号



ロシア語のタグを使用する 


 cue2tracks -c mp3 -Q 0 -B 320 -MC -f WINDOWS-1251 -o " %N - %t " -R file.cue


ドキュメント


https://code.google.com/p/cue2tracks/w/listをご覧ください



結論



このような簡単な方法で、タグと品質を維持しながら、flacファイルを別のmp3ファイルに変換できます。
