Pythonプログラムでドキュメントをプレビューする

私が関与しているシステムの1つでは、docファイルがデータベースに追加されます。

これらのファイルの表示をデータベースで動作する私のプログラムに添付することが可能かどうか疑問に思っていました。









何らかの理由で、このような問題の自然な解決策は、通常、コマンドラインでファイル名を指定してMSWordを起動することです。 しかし、この方法は、控えめに言っても、あまり安全ではありません。ドキュメントにマクロがあるか、ドキュメントではなく、クラッカーによって特別に準備されたファイルである可能性があります。 したがって、Officeに実装されている特別な表示オブジェクトを使用することをお勧めします。 ドキュメントを表示する以外に何もできないため、より安全です。



また、1つのドキュメント形式に限定しない場合、ボーナスとして、Windowsで標準のビューアが登録されている他の形式の添付ドキュメントを表示できます。



今後は-PyWin32を使用してすべてが判明しました。 確かに、このプロセスで予期しないCOMインターフェイスをサポートするためにパッケージをコンパイルする必要がありましたが、犠牲者はいませんでした。



だから私たちは何を知っていますか



  1. MSDNによると、システムには標準のIPreviewHandlerインターフェイスを実装するビューアーがあり、インターフェイスはShobjidl.hインクルーダーで説明されています
  2. 特定のファイル拡張子についてシステムに登録されたビューアがあるかどうかを確認できます-ブランチがある場合はHKEY_CLASSES_ROOT \ <extn> \ Shellex \ {8895b1c6-b41f-4c1c-a562-0d564250836f} （「 <extn> 」はドット付きのファイル拡張子、つまり、「. doc 」、「. pdf 」など）、デフォルト値があり、この値は対応するコンポーネントのCLSIDです。
  3. 登録されているすべてのビューアは、レジストリブランチHKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows \ CurrentVersion \ PreviewHandlersにリストされています。
