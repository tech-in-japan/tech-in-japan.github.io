GlassFish v3がリリースされました

NetBeans 6.8のリリースとともに、無料の（GPLv2 + CDDLv1.0）GlassFish v3アプリケーションサーバーの最終バージョンがリリースされました。 開発の1年半以上と、いくつかの「前奏曲」および「プレビュー」リリースを残しました。



GlassFish v3は、Java EE 6標準をサポートする最初の認定アプリケーションサーバーですが、最近承認されたJava EE 6標準には以下が含まれることを思い出してください。

  • エンタープライズテクノロジー：EJB 3.1、JPA 2.0、JDBC 4.0、CORBA 3.0。
  • Webテクノロジー：サーブレット3.0、JSP 2.2、JSTL 1.2、EL 2.2、JSF 2.0（Facelets）、RESTful Webサービス。
  • 支援技術：Java Contexts and Dependency Injection（JCDI; ex-WebBeans）、Bean Validation


GlassFish v3自体の特徴的な機能は次のように呼び出すことができます。

  • OSGi標準に基づくマイクロカーネル-サーバーを数秒で起動します。
  • Grizzly / Comet-ノンブロッキングJava NIOおよびNIOベースのWeb要求および応答ハンドラーのサポート。2
  • 管理および監視用のWebおよびCLIインターフェイス。
  • JRuby / Rails、Groovy / Grails、jMaki AJAX Framework、サーバーサイドJavaScriptなど、メインディストリビューションに含まれていない追加コンポーネントをインストールできるUpdate Center Webインターフェイス


これからは、GlassFishがSunの新しいWebテクノロジーの公式リファレンス実装（RI）であり、Apache Tomcatのバトンを引き継ぎます。 Sun GlassFish Enterprise Server v3の企業バージョンも利用できます。これは主に、GlassFishサーバーのセットを監視および管理するシステムであるEnterprise Managerの存在によって区別されます。 NetBeans 6.8、Eclipse 3.5.1、およびIDEA 9開発環境のサポートを明記。
