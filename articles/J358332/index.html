Arduino LLCとArduino SRLの再会







10月1日、 Arduino LLCの公式ブログに 、この組織がArduino SRLと合併していることを示すエントリが掲載されました 。 以前は、 Arduino LCCとArduino SRLはすでに連携しています。



Arduino LLCは2008年に設立され、Arduinoブランドの所有者です。 Arduino SRLの基盤は、イタリアの会社Smart Projects SRLであり、その工場は元のArduinoボードを製造しました。 かなり長い間、組織は互いに協力してきました。 Arduino LLCはボードとIDEを開発し、Arduino SRL（Smart Projects SRL）は生産に従事し、アメリカの開発者に商標を使用したロイヤリティを請求しました。



分割はお金のために来ました。 開発者として、Arduino LLCはどこで誰がボードを作成するかを気にしませんでした。 たとえば、中国は公式生産のプラットフォームと見なされていました。



Arduino SRLブランドの所有者であり、イタリアの工場であるGianluco Martinoは、それを好まなかったため（中王国の安価な生産と競争することは難しいため）、ある時点で彼の会社はアメリカ人にArduinoの商標を使用したことに対するロイヤリティの支払いを停止しましたそして、「真のArduino開発者」であると宣言しました。 ドメインarduino.orgが登録されました。このサイトでは、元のarduino.ccがほぼ完全に繰り返されました。 予想通り、すべてが起こった後、Arduino LLCによる訴訟が続きました。



しかし、数年にわたる貿易戦争と訴訟の後、組織は突然友好的な和解に至り、再会してArduinoに取り組みました。



Arduino LLCの共同設立者であるMassimo Banziは、合併についてコメントしました。

今日はArduino史上最高の日です。 これ（合併）により、建設的な対話、教育と生産の革新に向けた新しいコースを受講することができます。 Arduino Foundationにより、オープンソースエコシステム内のコアコミュニティの価値を維持し、オープンソースに対するこれまで以上のコミットメントを実現できます。 これはArduinoにとって本当に新しいスタートです！


将来的には、この提携はエンドデバイスの価格に影響を与える可能性がありますが、Arduino SRLのファームウェアの分岐からユーザーを救い、コミュニティを再統合します。
