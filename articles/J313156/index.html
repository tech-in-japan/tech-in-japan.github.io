便利なライフハック：最も一般的な質問に答えます

Huaweiのネットワーク機器を扱う多くの専門家が私たちに多くの質問を持っていることは私たちにとって秘密ではありません。



また、これらのスペシャリストの一部は、Huawei製品のセットアッププロセス中に表現に恥ずかしくないことも知っています。 そして、この投稿は、当社のネットワーク機器を使い始めたばかりの人と、それを使用したプロジェクトですでにかなりの経験を持っている人のほとんどの質問に答えると確信しています。 さらに、この記事は、ファーウェイの通信機器との相互作用の可能性を認めるすべての専門家に読むことをお勧めします。



実際、私たちがユーザーから受け取る質問のほとんどは、その数にもかかわらず、非常によく似た標準的なものです。 したがって、あなたの人生を楽にするために、私たちはあなたのためにライフハックのセットをコンパイルしようとしました。それは私たちの機器の操作に関連する問題の重要な部分を解決します。



始める前に、説明する有用なコンテンツにアクセスするには、Huaweiポータルで簡単な登録を行う必要があることに注意してください。



すべてのプレリュードが終了したので、次に主なものに進みます。



Lifehackナンバーワン：ドキュメントを読んでください！



はい、まさに！ 質問のほとんどに対する回答は、多くの労力をかけずに会社のポータルで入手可能なドキュメントから収集できます。



「Huaweiの機器に関する価値のあるドキュメントはパブリックドメインには存在しないという意見があります。一般的に、それが自然界に存在するという疑いがあります。」



私たちは答えます-これは神話です。 もちろん、最初から表示されるわけではなく、数年前、Huawei Enterprise部門が作成されたとき、ほとんどの資料とドキュメントには「内部」署名スタンプが付いていたため、会社のポータルには公開資料が豊富ではありませんでした。 しかし、時代は変わり、会社のプロセスもそれに伴って変化しています。現在、サポートポータルは、Huawei機器に関する膨大な量のさまざまなドキュメントをユーザーに提供しています。



もちろん、専門家にとって別の問題である大量のドキュメンテーションには、問題を解決するために必要な情報を探すのに手間がかかるという問題があります。 多くの場合、専門家は指示、文書、仕様の海で「おぼれる」ことを恐れています。 この問題は、ラックに設置する機器モデルと計画を選択する設計者、およびネットワークを設定してボードのインジケーターが1秒間に2回緑色に点滅する理由を把握しようとするシステム管理者または運用スペシャリストの両方に関連しています。



この問題を解決しようとしました。 「Huaweiのテリトリー」では、「情報の発掘」を安全に忘れることができます。ここでは、ソフトウェアの特定のバージョンの機器のラインに対して発行されたドキュメントが1つだけ必要になります。 新しいバージョンが利用可能かどうかは自動的に監視されます。このため、サポートサイトにアカウントを持っているだけです。



たとえば、ルーターV200R008のAR G3シリーズに関するドキュメントをご覧ください-「Huawei AR120＆AR150＆AR160＆AR200＆AR500＆AR510＆AR1200＆AR2200＆AR3200＆AR3600 Product Documentation」。



そのようなドキュメントは簡単に見つかります。サポートセクションで興味のある製品を選択し、「製品ドキュメント」カテゴリのフィルターを適用します。







ライフハックナンバー2：HedexLite！



必要なドキュメントは見つかりましたが、それだけではありません！



注意-対応するライブラリをオンライン表示モードで開くと、ページの上部に、このライブラリをダウンロードするためのリンクと、それを表示するためのHedexLiteプログラムがあります。







このドキュメントは、セクションごとに便利に配置された膨大な量の情報を収集しており、使いやすいように相互参照が提供されています。



  • 現在のバージョンの新機能のリストと機能の説明。
  • ラインの説明、モデルの配置、典型的なユースケースなど。
  • 以下を含むハードウェアプラットフォームの説明：


  1. 各ルーターモデルの外観と、すべてのコンポーネントの説明（組み込みポート、ボードと電源用のスロット、インジケータ、接地ボルトなど）。
  2. スロット番号付け規則。
  3. 気流パターン;
  4. 技術仕様（プロセッサ、メモリ、サイズ、重量、消費電力、動作温度範囲など）;
  5. すべてのボード、ケーブル（必要に応じてピン配列を使用）、トランシーバー（光バジェットを使用）、電源、およびファンに関する同様の説明。






  • 組立説明書。
  • セクションに分割された詳細な設定手順（各セクション内-ステップの説明と組み込みの「コマンドリファレンス」へのリンクを含む典型的な設定セクションのセット。すべての可能なコマンドパラメータが記載されています）。






各セクションの最後には、典型的な構成例のコレクションがあり、インターフェイスの例とともに典型的なトポロジのセットを示しています。 また、インターフェイスとIPアドレス、正しい設定の中間検証のためのコマンド、および構成ファイルの結果のフラグメントを考慮して、トポロジ上のすべてのデバイスをセットアップするための手順が提供されます。 さらに、これに加えて、ユーザーが問題を見つけて修正する方法に関するヒントと特定のソリューションを使用して行う典型的な構成エラーがあります。



  • 操作手順（必要な場合）の説明と、デバイスの状態を確認するためのコマンドを含む一般的なアンケートとセクション「操作とメンテナンス」。
  • 発生する可能性のある事故の説明と、インシデントの場合に実行する必要がある手順、および一般的な誤動作を診断および解決するための一連の指示（たとえば、パスワードが何らかの理由で失われた場合にコンソールにリセットするための指示）を含むセクション。
  • 構成コマンドおよび診断コマンドのすべての可能なパラメーターの詳細な説明を含む、前述の「コマンドリファレンス」。


HedexLiteをダウンロードすると、次のような便利な機能がさらに追加されます。



  • サポートポータルからドキュメントをダウンロードします。
  • ライブラリ内のドキュメントの更新の確認、およびドキュメントの既に無関係なバージョン（削除可能）の確認。
  • セクションごとに検索結果をフィルタリングする機能を備えた、ドキュメント用の柔軟な検索システム。 たとえば、キーメッセージospfを検索することにより、ドキュメントタイプで結果をフィルタリングできます。








サイトから個別のPDFドキュメントをダウンロードする機能も消えませんでした。



さらに、MIBリファレンス、MIB自体、ソフトウェアイメージ（更新手順付き）、およびパッチをダウンロードできます。



Lifehack第3位：eNSPネットワークエミュレーター！



このライフハックは、Huawei機器のコマンドラインに精通したり、認定試験の準備をしたり、他のタスクのためにある種の仮想ネットワークトポロジを収集したい人向けです。



Huaweiサポートサイトのすべての登録ユーザーは、eNSPというHuaweiハードウェアエミュレーターを使用できます。eNSPの最新バージョンは 、サポートポータルからダウンロードすることもできます。



ARG3シリーズルーター、Sx700およびCloudEngineシリーズスイッチ、Wi-Fiコントローラーおよびアクセスポイント、ITU USG、および多数の補助ツール（PC、サーバー、モバイルクライアント）を使用して仮想トポロジを作成できます。 モジュラールーター（AR1200-3200）には標準カードを装備できます-エミュレーター内のそれらの数は注文可能な数よりもはるかに少ないですが、最も一般的なFE / GE / G.SHDSL /シリアル通信回線をエミュレートすることはかなり可能です。



デバイスのコマンドラインは実際のモデルに対応しています。試験の準備をしていて、ルーティングプロトコルサービスパケットの構造を調べたい場合は、仮想インターフェイスでWiresharkを実行できます。



機器をエミュレートするには、組み込みのVRPイメージが使用されます。 また、多数のデバイスを使用するトポロジ（したがって、RAMおよびプロセッサリソースの量に対する高い要件）の場合、リモートサーバーで仮想デバイスを起動すると、クライアントサーバーモードが提供され、PCから重いコンピューティングタスクが削除されます。







トポロジの利便性とわかりやすさのための作業領域では、背景色を変更し、デバイスとそのグループに署名できます。



注意する必要があります-エミュレータを使用するには、Oracle VM VirtualBox（インストールパッケージに含まれています）が必要です。



さらに、エミュレータプログラムの更新バージョンのチェックは自動的に実行されます。



Lifehack Number Four：Visioのステンシル！



次のライフハックはHuaweiからの素晴らしい贈り物です！ 結局のところ、Huaweiの機器を扱うスペシャリストだけでなく、プロジェクト文書の作成や、極端な場合には高レベルのネットワーク図の作成を処理した（および処理しなければならない）すべての人々にとっても有用です。



どのメーカーの機器がプロジェクトで使用されているかに関係なく、MS VisioのStensilsは、プロジェクトのドキュメントとネットワーク図のすべての作成者を喜ばせます。



すべてが同じサポートポータルにあります（1平方センチメートルあたりのユーティリティの集中度がちょうどロールオーバーする場合）。「ステンシル」の2つのオプションがあなたを待っています：



  • あらゆる機会にネットワークトポロジとスキームを作成するためのネットワークデバイスと補助要素のアイコン（2色）：







  • ラックを充填するための「磁化」ブロックとボードを備えたデバイスのファサード







それらから、機器を使用して必要なラックファサードを組み立てることができます。







ライフハックナンバー5：ファイナル！



従来のソフトウェアとユーティリティを据え置き型のPCからインターネットポータルに移行する傾向は、Google Docs / Sheets、画像エディタなど、長い間秘密ではありませんでした。



また、「オンラインツール」の開発の見通し、その使用の利便性を信じているため、Huaweiサポートポータルの開発と入力に努めています。



上記で説明したツールに加えて、サポートポータルには、他の便利な「オンラインツール」が多数あります。 また、多数の質問に対する回答を見つけることができます。



そして最も重要なことは、私たちに質問することをheしないでください。Huaweiネットワーク機器を使った作業をより便利にするための新しいアイデアを実装するのに役立つのはあなたのニーズです。



あなたの問題と「一対一」にとどまらないでください:)
