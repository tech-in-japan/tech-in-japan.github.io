FL Studio-ジェダイの道（パート1）。

あなたが家に帰って、プレーヤーにあるすべての種類の電子音楽を聞いた後、あなたはこの世界にもいくつかのヒットを与える時間であると決めました。 当然、高価な機器を購入し、頭でプロの音楽制作の分野に突入する前に、楽しみのために、試して「感じ」て、コンピューターでの電子音楽の作成とは何ですか...



このシリーズの記事では、FL Studioで音楽を作成するプロセスに焦点を当てたいと思います。



友達や知人にあなたの芸術の新しい作品を喜ばせる方法に興味がある人は誰でも、猫をお願いします。



はじめに





まず、FL Studioのデモ版が必要です。これは、ここ （97.6 MB）、ミッドレンジコンピューター、空き時間、ヘッドフォンからダウンロードでき、音で実験して隣人を殺さないようにできます。



FL Studioの音楽トラックは、 パターン （ パターンと）と呼ばれるフラグメントで構成されています。 そのような各パターンは、1つまたは複数のチャンネルに割り当てられた一連のノートで構成されます。 パターンのサイズは四分の一の倍数で、パターンの最小サイズは四分の一です。 FL Studioのメジャーはデフォルトで4/4です（プロジェクト設定でサイズを変更できます）。パターンエディターでは、メジャーは明るいグリッド線で区切られ、上部に数字で順番に書き込まれます。 パターンは音が10オクターブ以内の音で構成され、音の最小持続時間は128音です。



インターフェースFL Studio





一般的な規則として、レギュレーターの位置を変更するには、マウスの左ボタンでクリックし、ボタンを離さずにマウスを下に移動して上下に移動する必要があります。 カーソルがノブの上にあるときにマウスローラーで位置を変更することもできます。



インターフェイスは次のようになります（開発の最初の段階で最も重要なものだけがコメントされます）。





ステップシーケンサー



ステップシーケンサーにはチャンネルのリストが含まれており、パターンの編集に使用されます。 編集したパターンの番号が上部に表示され（最初のスクリーンショットを参照）、プレイリストでもオレンジ色で強調表示されます。



各ラインは1つのチャンネルであり、このチャンネルはシンセサイザー（与えられた音に基づいて音を生成するチャンネル）またはサンプラー（既製の音（サンプル）を使用するチャンネル）のいずれかです。



スクリーンショットは、音を鳴らすには、それに対応する長方形をクリックする必要があることを示しています。音の音色を決定するには、キーでウィンドウを開いて必要なものを選択する必要があります。また、音のパラメーターを変更することもできます



新しいチャネルを追加するには、メニュー項目Channels-Add one ...を実行してから、ドロップダウンリストで目的のチャネルを選択する必要があります。



選択したチャンネルを複製または削除することもできます。 選択は、チャンネル名の右側にある灰色の長方形を右クリックして行います。 左クリックすると、すべてのチャンネルが一度に選択/選択解除されます。



チャネルを上下に移動するには、ALTキーを上下に押します。 選択した複数のチャンネルを同時に移動することもできます。



選択したチャンネルをグループにまとめることができます。これには、メニュー項目チャンネル-グループが選択されています。



プレイリスト



プレイリストは、トラック全体を編集するために使用されます。 2つの部分で構成されます。最初の部分にはこのプロジェクトのパターンのリストがあり、2番目の部分にはオートメーションクリップ（オートメーションクリップ）とオーディオクリップ（オーディオクリップ）があり、ここにパターンを挿入することもできます。



パターンを描画するには、目的の場所を左クリックして（もちろん、鉛筆モードまたはブラシモードで）、パターンを削除するには、右クリックします。



選択するために、ツールを切り替える必要はありません。単にCtrlキーを押したままにし、何かを切り抜くには、Shiftキーを押したままにします。



このパターンの行全体を選択するには、名前の右側にある小さな長方形をクリックします。



チャンネルのようなパターンは、移動、複製、削除できます。 一度に1つまたは複数（このためのコマンドは、パターンのコンテキストメニューまたはアイテムのパターンのプレイリストメニューにあります）。



複数のパターンを1つに結合するには、それらを選択して、コンテキストメニューで[選択した結合]を選択します。



1つのパターンの複数の「長方形」を1つに結合するには、コンテキストメニューで[選択]を選択します。



このパターンの行全体を選択するには、名前の右側にある小さな長方形をクリックします。



グリッドへのスナップモードについては、有効にすると、要素の動きがグリッドにスナップされます。 次のようになります。

  • メイン -バインディングは、上記から選択されたグローバルバインディングに従って実行されます。
  • 線 -グリッド線にスナップします。
  • セル -グリッドセルにスナップします。
  • （なし） -バインディングを無効にします。
  • 1 / 6、1 / 5、1 / 4、1 / 3、1 / 2ステップ -グリッドピッチのそれぞれ1 / 6、1 / 5、1 / 4、1 / 3、1 / 2にスナップします。
  • ステップ -グリッドステップ全体にスナップ
  • 1 / 6、1 / 5、1 / 4、1 / 3、1 / 2ビート -1 / 6、1 / 5、1 / 4、1 / 3、1 / 2のビットにスナップ（ビット-1/4）したがって;
  • ビート -ビット全体にバインドします。
  • バー -メジャーへの参照。




ピアノロール



直接メロディーエディター。 ウィンドウは、プレイリストのツールバーと同様のツールバー、左側のステーブ、中央にノートを入力するためのウィンドウ、下部にあるパラメーターウィンドウで構成されています。



ロジックは単純で、1つの長方形は1つの音符であり、長方形の長さは音符の長さです。

ほとんどの操作はプレイリストの場合と同様に実行され、Shiftキーを押しながら必要な選択をコピーして転送するだけです。



デュレーションを長くするには、ノートの右端を引くだけで十分ですが、キーボードでCAPSLOCKが有効になっている場合は、右端のサイズを変更することもできます。



さらに、CAPSLOCKモードでは、選択した領域のサイズを変更すると、全体として変更され、オフにすると、ノートのサイズのみが変更されます。



グリッドへのスナップは、プレイリストと同じように機能します。



ミキサー



「果物」の主な情報ツール。

主にチャネルで構成されます。 ミキサーの各チャンネルには、最大8つのエフェクトを含むことができるスタックがあります。 エフェクトは、サウンドパスに影響を与えずにサウンドを変換するだけです。

便利なことに、キーボードを使用してミキサーを制御できます。 ミキサーの表示/非表示：。 ミキサーモード：S-チャンネルソロ。 楽器を無料のミキサーチャンネルにバインドします：CTRL + L



チャネルには3つのタイプがあります。

  • マスター -一般チャンネル、すべての送信および挿入トラックはそれを通過してから出口にのみ通過します。
  • Insert-通常のジェネレーターまたはサンプラーに関連付けられた通常のチャンネル（ミキサーには64個あります）。
  • 送信 -削減されたチャネル（4個）。


サウンドルーティングの原理をよりよく理解するには、次の図をご覧ください。



各通常チャンネルは、シンセサイザー/サンプラーまたは別のチャンネルからサウンドを受信します（入力の音量レベルはこのチャンネルのコントロールによって選択されます）。出力はマスターチャンネルへの共通ラインに送られます。 送信チャネルは、他のチャネルからの出力のみを受信するという点でのみ通常のチャネルと異なります。



チャンネルをミキサートラックにバインドするには、（チャンネル設定）ウィンドウの（FX）ウィンドウで目的のトラックの番号を選択するか、ミキサーで目的のトラックを右クリックして、 選択したチャンネルのリンク-このトラックへ （キーボードのCTRL + Lのいずれか）を実行します。 後者の場合、選択したすべてのチャンネルがこのトラックに接続されます。 Link Selected Channels-このトラックから開始 （SHIFT + CTRL + L）を実行すると、選択したチャンネルは、選択したものから始まる各トラックに順番に割り当てられます。
