私たちがあなたに伝える（そして見せること）



私たちはプログラムの質とあなたが何を見つけることができるかに非常に注意を払います。 製品の開発と作成に直接関与しているマイクロソフトの代表者と、当社のテクノロジーの使用に関して深い実務経験がある他の企業の従業員と話すユニークな機会があります。 メインの会議グリッドには、最も関連性の高いトピックに関する50以上のレポートが含まれます。 メインプログラムには4つの領域が含まれます。



窓

AzureとWeb

開発言語とツール



.NET 2015のすべて！ Visual Studio 2015開発ツールとソフトウェアライフサイクル管理の方法論と使用。 現在の機能を使用して、将来の.NETプラットフォーム、C＃、JavaScript、XAML、開発技術について議論します。



企業開発、データ処理



SQL Server、ビッグデータ、機械学習。 データ、ビジネス分析ツール、Microsoftオフィスプラットフォームを操作します。 プラットフォームでのエンタープライズアプリケーションの開発NET、SharePoint、およびOffice 365 ハイブリッドアプリケーション開発シナリオ、企業ネットワークとクラウド環境の統合。
