Alternativa3D 5.0





今日、フラッシュゲームのプラットフォームの開発者グループは、そのエンジンの新しいバージョンに基づいていくつかの新しいデモを提供しています。





提供されるデモは次のとおりです。







開発者は、エンジンの技術的特性も提供しました。

  • 信号システム -計算はニーズに応じてのみ行われます。
  • BSP-高品質のポリゴンのソート、「クリア」な埋め込み、不要な計算の欠如。
  • ポリゴンサポート。
  • レンダリング中のレイヤーの再利用（画面の可変部分のみを再描画し、レンダリング中にポリゴンは独立しています）;
  • オブジェクトの階層。
  • 変更時のみのマトリックス、座標、UVの再計算。
  • 前向き補正-動的三角測量;
  • 衝突検出システム。


また、追加予定の機能が発表されました。

  • 動的な照明と影 （点光源と指向性光源）;

    レリーフ照明 ;

    視差マッピング ;

    自動UVマッピング ;

    スプライト、3Dスプライト 。

    照明スプライト ;

    テクスチャオブジェクト

    イミテーショングローバルイルミネーション ;

    スプライトプリレンダリングシステム（3Dモデルに基づいてサーバー上にスプライトを作成）;

    アニメーションシステム（逆運動学を含む）;

    3Dオブジェクトの相互作用 ;

    物理シミュレーション。

    詳細はプロジェクトのブログで読むことができます。
