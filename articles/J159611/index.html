NetWrix Active Directory Change ReporterがWindows Server 2012およびExchange 2013をサポートしてリリースされました



間違いなく、Windows Server 2012およびExchange 2013の新しいバージョンのリリースは、私たち全員にとって見過ごされなかったわけではありません。

次に、Microsoftがソリューションの開発で設定するペースを維持しようとし、 Windows Server 2012 / Windows 8とExchange 2013の両方をサポートするNetWrix Active Directory Change Reporter 7.2の新しいバージョンのリリースを発表できることを嬉しく思います 。



この更新プログラムは、プログラムの両方のバージョンに影響しました。 これで、Windows Server 2012 / Windows 8のプログラムの無料版を簡単に使用できるようになりました。詳細は以下をご覧ください。





バージョン7.2にはどのような変更点がありますか？

  • これで、プログラムをWindows Server 2012およびWindows 8にインストールできます。
  • Exchange 2013で監査データを収集するためのサポートが追加されました。
  • Windows 2012ベースのドメインのサポートが追加されました。
  • 新しいレポートが追加されました：「配布グループメンバーシップのないユーザー」（プログラムの拡張バージョンのみ）
  • 連邦情報処理標準（FIPS）が有効になっているサーバーへのインストールのサポートが追加されました
  • さまざまな改善と修正








この更新プログラムは、NetWrix Active Directory Change Reporterに直接関連する他の2つのプログラムにも影響を及ぼしたことを別に注意してください。 次の2つのプログラムもバージョン7.2に更新されました。

  • NetWrixグループポリシー変更レポーター
  • NetWrix Exchange Change Reporter （Exchangeサーバー監査）








Windows Server 2012は、 NetWrix Active Directory Object Restore Wizard 7.2 （Active Directory Change Reporterに含まれています）を使用してActive Directoryオブジェクトの回復を簡素化できるようになりました。







当社のWebサイトからプログラムの20日間の試用版をダウンロードできます。



更新： 12月5日、NetWrix Active Directory Change Reporter 7.2ウェビナー：Windows Server 2012およびExchange 2013の新しい監査機会をホストしました。
