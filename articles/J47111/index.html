Apple iTablet Tabletニュース

現在、多くの新しいタブレットPCがあります。たとえば、 「Stylistic ST6012 Tablet PC Released」という記事を参照してください。

しかし、タブレットPCは、ラップトップに変換されないが、根本的に再設計されるという条件でのみ人気を得ることができると思います。このプロジェクトの主な目標は次のとおりです。

1.コンパクトさ。 2.軽量; 3.エネルギー効率。

タブレットは、Apple iPhoneなどのスマートフォンやAsus EeePC 900などのNetBookから最高の性能を発揮する必要があります。重量と価格が低い必要があります（特に、庭に経済危機があるため）。

また、タブレットの場合は、まったく新しいインターフェースを開発する必要があります！ 実際、MS Windows XP / VistaインターフェースはデスクトップPC用に開発されたものであり、タブレットはテーブルの上に立つように設計されていないため、プログラムインターフェースがスマートフォンや携帯電話のインターフェースのように見える場合、それを使用する方がはるかに便利です。

結論は次のとおりです。タブレットPCでは、次のようなOSを効果的に使用できます。

1. Apple iPhone OS （Apple OS Xのコア上）;

2. Google Android OSまたはChrome OS （両方ともLinuxカーネル上）;

3. Nokia Maemo 5 （Linuxカーネル上）;

4. MID用 Intel Moblin （Linuxカーネル上）;

5. Palm Linux（Linuxカーネル上）;

6. MS Windows Mobile 7またはWindows CE for MID （両方ともMS Windows CEのコア上）。

おそらく、最初の本当に便利で人気のあるタブレットは、Appleだけがリリースできると思います！ 2009年に新しいOS（Apple iPhone OSをベースとする）が登場する可能性が高く、そのハードウェアにはIntel CPUではなく、ARMコアに基づいて開発されたプロセッサが搭載されます。



2008年9月、Habrに関する記事を公開しました。 「 Steve JobsはDynabookを再び発明しようとしています！ 」では、上記のアイデアを証明しようとしました。

そして今日、私は過去2年間に蓄積されたニュースのレビューを見て議論することを提案します。そこでは、今後のTablet PC Apple iTabletまたはApple iTouch Tabletのうわさや間接的な確認が表面化しました。
