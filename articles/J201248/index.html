開発者はインテルからどのように支援を受けることができますか？ エキスパートアンサー



Habrの訪問者の質問に対するIntel Svetlana Emelyanovaの専門家の回答を公開します。 トーク：ソフトウェア開発者がどのように、また何をIntelから入手できるか。



Windows 8アプリケーション用に開発者が入手できるデバイスは何ですか？ （教育アプリケーションを含む）

Intel Developer Zoneメンバーは、アプリケーションをテストおよび開発するためのデバイスを入手できます。 「プール」では、Windows 8およびAndroidベースの有名なメーカー（Lenovo、Acer、Asus、Samsung）のタブレット、スマートフォン、ウルトラブック、およびIntelブランド（n.r. Harris Beach Developer Ultrabook-完全に排他的で、そのようなデバイスは販売されておらず、予想されていません）。 現在、Windows 8ベースのデバイスには、Samsung ATIV Smart、Acer Iconia W510、Lenovo ThinkPad Tablet 2の3種類のデバイスがあります。BayTrailの登場により、新しいプロセッサに基づくデバイスで在庫を補充する予定です。

「これらのデバイスの入手方法」という項目も重要です。もちろん、私たちはそれらを左右に配布しないからです。 まず第一に、私たちはアプリケーションにちょっとした革新を加えています。なぜなら、 デバイスを受け取ったすべての開発者は、共同マーケティングプログラムに参加する候補者です。 デバイスを取得するには、最初に正式な段階を経る必要があります-Intel Developer Zoneプログラムに登録します。 次に-アプリケーションに関する情報をispprussia@intel.comに送信します。 革新性、パフォーマンス、インターフェイス品質などの基準に基づいて、アプリケーションを評価します。 アプリケーションに可能性があると判断した場合、開発とテスト用のデバイスを入手できます。 さらに、パーソナルアカウントマネージャーがお客様に付属し、開発プロセス中にサポートを提供します。必要に応じて、Intelエンジニアが特定の問題を解決します。 一定期間（通常は3か月間、アカウントマネージャーとの合意により）後、アプリケーションをテストし、共同マーケティングプログラムへの参加を決定するために、アプリケーションの改訂版を提供するよう求められます。

上記のデバイスを使用すると、長期間使用できます。 サーバーシステムを含む特定のIntelプラットフォームでアプリケーションを緊急にテストする必要がある場合は、「クイックテスト」のためにアプリケーションを送信できます。 このイニシアチブはロシアの開発者に関連しており、最大2か月間機器にアクセスする機会を提供します。 サイトの申請書に記入する必要があります。48時間以内にお客様のリクエストに対応します。 必要なデバイスが利用可能な場合、同日中にインテルのモスクワオフィスからピックアップするか、テストのためにリモートアクセスを手配します。



マーケティングサポートを受けるにはどうすればよいですか？ （AICコンテストアプリケーションを含む）

アプリイノベーションコンテストの優勝者には、コンテストルールでマーケティングサポートが提供されます 。 これは、PRキャンペーン、サクセスストーリー、Intel DZ Webサイトでのビデオインタビュー、Intelイベントおよびパートナー（Lenovoを含む）でのアプリケーションのデモ、Intel DZメーリングリストでの言及、ソーシャルメディアです。 そしてもちろん、教育部門の3人の受賞者にはマーケティングサポートが提供されます。 受賞者以外の出場者は、共通ベースでマーケティングサポートを受ける資格があります。 アプリケーションの要件については、前の段落で説明しました。 さらに、アプリケーションは少なくともRCステージにある必要があります。つまり、 デモアプリケーションは考慮しません。 独自の機能の説明など、アプリケーションに関する情報をispprussia@intel.comに送信する必要があります。



共同マーケティングプログラムが原則として何であるかを理解したいと思います。 何が含まれていますか？

共同マーケティングプログラムは、製品を宣伝し、ユーザーの視聴者を拡大するのに役立ちます。 今日の基本的なツールの1つは、イベントや電子カタログでのアプリケーションのデモンストレーションです。 インテルは、年間を通じて独自の会議を開催するだけでなく、多くの業界イベントにも参加しています。 展示スタンドでは、プロセッサに基づいたデバイスを展示しています。 展示会の訪問者は、これらのデバイスの機能に興味があります。 アプリケーションがプラットフォームおよびテクノロジー向けに最適化されており、興味深い機能を備えている場合は、デモのリストにアプリケーションを含めるための第1候補です。

電子ソフトウェアカタログは最近の取り組みです。 現在、Windows 8ベースのUltrabooks用のパートナーソフトウェアのカタログがあり、近い将来、Android x86デバイス用のソフトウェアのカタログを計画しています。 別のアプリケーションディレクトリを作成する理由 ここでは、いくつかの目標を追求しました。 現在、WindowsストアとGoogle Playには何百ものアプリケーションがあり、その多くは最高品質とはほど遠いものです。 当社のオンラインストアは、当社のプラットフォームでテストされたパートナーソリューションを備えています。 他のIntel部門（たとえば、営業およびマーケティング部門）の同僚にとって、これは小売でのデモ用の興味深いアプリケーションを見つけるための単一のリソースです。 外部ユーザーの場合-これはIntelのソフトウェア品質の保証です。 今年の終わりまでに、ロシアの開発者のカタログを開始します。これには、ウルトラブックの20の最高のアプリケーションが含まれます。



各四半期には、さまざまなプラットフォームの開発者向けの新しい取り組みがあります。 インテルは、今四半期にAndroidアプリケーションの開発者向けの共同マーケティングプログラムを開始しました。これは、Intel Atomプロセッサーをベースとしたデバイスの数の増加に関連しています。 アプリケーションのマーケティングに5,000ドルを獲得できます。 まず、NDK Androidアプリケーションと、WiDiテクノロジーを実装する可能性のあるアプリケーションを検討します。 このイニシアチブに関心がある場合は、11月15日までに、Google Playのappslab@intel.comでアプリケーション/アプリケーションへのリンクに関する情報を送信する必要があります。 インテルの専門家が、最適化の可能性についてアプリケーションを評価します。 x86のサポートを追加する必要がある場合は、テスト用にIntel AtomベースのAndroidデバイスを入手できます。 さらに、イニシアチブへの参加のために提出されたすべてのアプリケーションは、5つの基準に従って評価されます。1）86アーキテクチャのサポートの可用性。 2）パフォーマンス 3）インターフェースの品質。 4）革新性; 5）Intelテクノロジー（WiDi、Project Anarchy、IPP、GPAなど）の使用。 上位5つのアプリケーションは、Intelからマーケティングサポートを受けます。

Android x86アプリケーションの開発とWiDiテクノロジーの詳細については、 最近のウェビナーをご覧ください 。
