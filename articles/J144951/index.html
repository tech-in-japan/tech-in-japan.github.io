モバイル向けHTML5マップ

先日、ノキアのHTML5チームは、スマートフォンとタブレット用のAPIマップを提供しました。



現場から-採石場へ！

リンクを試すことができます： bit.ly/html5map

Webkitブラウザは現在サポートされています。



利点と機会

  • コンポーネント

    地図、渋滞、検索、地理位置情報、興味深い場所に関する情報、ナビゲーション、ウォーキングルート、お気に入り。
  • 重さ

    80 KB Javascript + 10 KB CSS
  • 性能

    高速読み込みとスムーズな効果。
  • 準備

    色を簡単に変更し、PhoneGapまたはQtWebkitでラップして、完成したアプリケーションを取得できます。
  • WYSIWYGヘルパー

    構成を自分用にカスタマイズするのに役立ちます。








レスポンシブデザイン

スマートフォンとタブレットの両方の画面でうまく機能します。

サポートされているのは、iOS 4.x、iOS 5.0、Android 2.2、2.3、3.1、3.2（4.0）、Meego、Blackberry TouchデバイスOS6、OS7です。 下半期には、Symbian、Windows Phone 8.0、WebOS 2+にアップグレードします。



カード

都市交通や渋滞など、4種類のタイルが提供されています。 また、ジオロケーションボタンを使用すると、現在の場所を特定できます。



検索する

瞬時の結果で住所と場所を完全に検索します。 データソースをねじ込むことができます。 検索結果には、ランキングと場所までの距離が含まれます。



興味深い場所

Nokiaカードは、世界中の5つのカテゴリで数千の場所を提供します。



場所に関する詳細情報には、説明、写真、住所、評価、レビュー、営業時間、近隣の場所が含まれます。 また、このコンポーネントを使用すると、電話をかけたり、サイトにアクセスしたり、ルートを検索したり、リンクを共有したりできます。



ナビゲーション

自動、歩行者、都市交通の3種類のルートを使用した便利なステップバイステップナビゲーション。 都市のハイキングルートは特に強い点だと思います。
