Wanted.VC Digest＃17

先週、 Wanted.VCは次の数十のプロジェクトによって強化されました。 そのうちのいくつかの著者はエカテリンブルク出身で、 週末に第2回ウラルスタートアップウィークエンドが開催されます。 週末に彼らと知り合い、通信し、協力に同意することが可能になります。

あなたがオンラインプロジェクトの著者である場合、あなたはまだ参加を申請する機会があります。 これを行うには、アイデアの説明をidea@glavstart.ruに送信して登録します。 Startup Weekendへの参加には、 Glavstartから最大100,000ドルの投資を受け取ることにあふれています。



CIVIAプロジェクトについて最初に聞いたのは、前回のモスクワスタートアップウィークエンドです。 そこでは、仮想空間の実際の政治状況を再現するオンラインゲームのアイデアは適切な対応を見つけることができませんでした-著者は緊急に空港に行かなければなりませんでした。



CIVIAの作成者であるPhilip Duhliは、個々の詳細を練り上げ、一般的な概念を思いついたので 、今回はWanted.VCでプロジェクトについて再び話すことにしました。 +++











通常の意味では、ストア（さらにはオンラインストア）は、本、電子書籍、プログラム、ソーセージ、または極端な場合は自由時間（たとえば、 TimeAngelプロジェクトを参照）などを販売する場所です。

Shopdepotはかなりの店舗ですが、ソーセージやソフトウェアの代わりに店舗が販売されています。 そう、お店。 オンラインストア。 +++







彼にとって天国であったフョードル・ミハイロヴィチ・ドストエフスキーは、レオ・トルストイと共同で本を書くことにしました。 トルストイは同意したが、対話をツルゲーネフに委ねることにした。 ツルゲーネフは、標準バージョンとは異なり、バーデンバーデンに向けて出発せず、企業の技術的な側面を反映し始めました。

そして、もし3人全員が1世紀前に死亡していなかったら、彼らは間違いなくDoozybook.comに来ていただろう。 +++









最近、Wanted.VCで、私たちはビルダーのためのサービスであるStroyKompasについて書きました。雑誌Everything for a Builderの10年前のバインダーの一種です（もちろん、そのようなものが存在すると仮定しています）。

宝石商のためのソーシャルネットワークであるBridgemは、この意味でStroyCompasの類似物です。 ただし、プロジェクトの作成者はほぼ同じように位置付けており、簡潔な宝石、宝石、小売業の毎日のオンラインワークスペースを求めています。 +++







Rybalka.comの創設者は、釣りが大好きでした。 彼らは、使い古された手のひら、足のしびれ、涙目に釣りました。 彼らは釣りを始めたので、それらを見て初心者でも漁師になりたいと思った。

その後、 Rybalka.comの設立者は、釣りの達人が集まる簡単なサイトを作成し、漁獲の写真を投稿し、投稿内容について議論しました。

そのサイトの立ち上げから4年が経過しました。 現在、このシンプルなサイトはRybalka.comと呼ばれ、毎日30,000人のユニークビジターがいます。 +++







Flickr.comサービスは長い間、視覚的に飢えている人々への出口でした。そのようなインターフェースを備えたサイトにい写真を投稿することは、望まれただけでなく、単に恥ずかしいことです。 Flickrはデザイナー、プロおよびセミプロの写真家、アーティストをすぐに乗り越え、情熱的な人の美しい写真に情熱を傾けました。

Flickrのパラドックスは、 Flickrの活動分野が実際のサイトに限定されていたことです。 プロジェクトの作者であるIT'S GOOD！ 、美しい写真の大ファンでもある、考えを発展させることにしました。 彼が作成したサイトでは、ユーザーはクリックするだけで、任意のサイトのお気に入りの画像をブログに追加できます。 +++











今日の「ご存知ですか」セクションでは、iPhone / Android / Blackberry向けのカスタマイズされたモバイルアプリケーションの開発コストは25,000ドルから120,000ドルの範囲であり、開発プロセス自体は数か月かかる場合があります。モバイルアプリケーションを作成する別の方法は、より速く、安く、簡単です。 +++





Tripleプロジェクトの作者であるKonstantin Doroninは、他のプロジェクトGostinetsとともにWanted.VCを訪れました 。 それは、ロシアのホテルのゲストのためのボーナスプログラムについてだったことを思い出します。宿泊に費やされたお金の一部は、プログラムに参加しているユーザーのアカウントに返還されました。

コンスタンティンは、ソーシャルサービスの使用を継続することを決定しました。 今日は、プロジェクトの作成者がソーシャルガイドと呼ぶトリプルプロジェクトについて話します。 +++







実践が示すように、学生のすべての問題は、道徳的および技術的な2つの大きなグループに分けることができます（道徳的問題は時間の経過とともに自然に消滅するため、実際には道徳の概念そのものです）。 前者の円は広大であり、そこに入る問題は非常に個人的なものであるため、それらを解決しようとする試みは恐ろしい慢のように思えます。

したがって、 SKIPプロジェクトの作成者は、技術的な問題のみを解決します。 どのものを理解するには、名前の略語「SKIP」を解読するだけで十分です-スキャン、コピー、インターネット、印刷。 全体で1.5平方メートルの面積をカバーします。 +++
