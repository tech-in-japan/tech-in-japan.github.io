インターネット/ Web 2.0リソースの国内カタログ

「クレイジー」と「サイエンスフィクション」と言われますが、

しかし、悲しい中毒から出て、

今日、人工知能の要素を持つサイトのロシア語版カタログのベータ版が公開されました： IrDir.info





このカタログは、次のアプローチとテクノロジーをサポートしています。

  • コンテンツを分類するためのフォークソノミックと分類学的アプローチの組み合わせ。

    「最大サポート」 シンジケーションサポートが含まれています。カタログのほぼすべてのページにRSSフィードがあります。

    インターネット技術の分野における最新の成果が使用されています。 AJAX ;

    カタログは、 ダブリンコアの基本的なメタデータモデルを サポートしています。カタログリソースに関する情報は、ユーザーだけでなくコンピューターでも理解できます。

    このカタログは、インターネット上のオープンな検索技術をサポートしています-オープンな検索。

    カタログコンテンツはユーザーによって完全に作成され、他のユーザーは作成されません。 ユーザーはサービスの基盤です。

    機能モジュールの代わりに、カタログは独自の開発- インテリジェントスペシャリスト （インテリジェントアドバイザーの恒常性モデルの判断に基づいて決定を下せる特別なソフトウェアモジュール）を使用します。

    インテリジェントモジュールの中には、特定のインターネットリソースに関連するドキュメントを追跡し、いくつかの問題を自動的に明確にすることができるドキュメント管理システムがあります。

    このカタログは、インターネットテクノロジーの分野における最新の標準に準拠しようとしています。有効なRDF、有効なXHTML、有効なCSS、有効なRSSフィード。

    カタログは、インターネットリソースにユニークな多要素ランキングシステムを使用しています。



    すべてのユーザーが自分のサイトをディレクトリに登録できます 。
