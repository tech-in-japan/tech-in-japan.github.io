概要：Googleから個人用VPNを月額5ドル（上級者は1ユーロ）で作成します







最近、VPNを使用することが一般的になりましたが、なぜですか？ 、しかし、私に出会ったすべての指示は、Linuxが何であるかを少なくとも少し知っているユーザーのみが「マスター」できます。 Googleは、2回クリックするだけで（ 2回で！） VPNをインストールできる優れたアプリケーションのリリースを皆に喜んでくれました。

（サーバーがない場合-問題ではありませんが表示されます）



Outlineアプリケーションについて説明します-これはGoogleに含まれるジグソー子会社のプロジェクトで、ロックをバイパスしたりプライバシーを強化したりするツールの作成に特化しています。Outlineはそのプロジェクトの 1つです。



本書の最後で技術的な問題（セキュリティ）について説明し、手順に進みます。



必要なものをすべてダウンロードする



VPNサーバーを作成するには、下のスクロールwww.getoutline.org/en/homeにアクセスし、[アウトラインマネージャーの取得]を選択して、 適切なバージョンをダウンロードする必要があります。

または、次のリンクを使用してください。



  1. Mac

  2. Linux

  3. 窓


アウトラインマネージャーをダウンロードしている間に、アウトラインクライアントを電話またはコンピューター（VPNの背後に隠す予定）にインストールします。



PS注意！ VPNを使用するコンピューターと同じコンピューターからVPNを管理する場合は、両方のアプリケーションをダウンロードしてください。





  1. Android
  2. 窓
  3. Chrome OS
  4. iOS
  5. Mac


サーバーを作成する



アウトラインマネージャーを起動すると、次のようなものが表示されます。







[ アカウントを作成]をクリックします



この時点で、ホスティングプロバイダーDigitalOceanに登録され、登録に関する標準的な質問が行われ、支払いカードの入力が求められます。



PS彼らはあなたに10ドルを提供し、2ヶ月間完全に無料でサービスを使用できるようになります。



登録が完了すると、サーバーの国を選択できるようになります。







あなたに最も近い国を選択してください。



私が選んだ-アムステルダム/オランダ、国を選択した後、あなたのサーバーが作成され、設定されます、それは多くの時間はかかりません







設定が完了すると、次の画面が表示されます。







彼はすぐにあなたのサーバーに関するすべての質問に答えます：あなたが残したトラフィックの量、あなたのサーバーにアクセスできるユーザーの数、あなたのサーバーの価格



ユーザーを追加する



「キーの追加」を押します-素晴らしい、あなたは新しいユーザーを作成しました。VPNにアクセスするためのリンクを（またはあなたの電話で）送信する時が来ました。



これを行うには、[共有]をクリックし、リンクを取得して送信します。



リンクを送信するには- 安全な通信チャネルを使用し、Telegramで自分とチャットするのが良い、コンピューターと電話の間で同期されるメモ-罰金、外国のホスト（google、apple）のメール-罰金、ICQ /ソーシャルネットワーク/任意のサービスあなたの国から（ロシア連邦-yandex / rambler / mail.ru / icq）-悪い、非常に悪い。



結論



おめでとうございます。これで、Outline Managerアプリケーションを使用して、月額5ドルの個人用VPNサーバーを手に入れました。サーバー上のトラフィック消費量を監視し、ユーザーアカウントを構成できます。



セキュリティはどうですか？



すべてが順調です。このVPN-中国で作成されたShadowsocksプロトコルを使用して、中国のファイアウォールロックをバイパスし、コミュニティでサポートされるようになりました。



VPNサーバー自体は、サーバーにインストールされる2つのドッカーイメージです。つまり、このVPNを既存のサーバーに恐れることなくインストールできます。 プロジェクトのソースコードは、当然Githubで入手できます。



1€



すでに独自のサーバーを持っている場合、またはデフォルトのホスティング（digtalocean）の代わりに安価なホスティングを使用する場合は、使用可能なサーバーにOutlineをインストールできます。



たとえば、arubacloudでは、イタリアのサーバーの価格は1ユーロで、まだ2か月の試用期間が与えられます...しかし、初心者には推奨しませんが、DigitalOceanからのpingははるかに優れています



パーソナルサーバーでアウトラインを構成するには、アウトラインマネージャーのメイン画面を「サーバーが既にある」までスクロールする必要があります。クリックすると、サーバーで実行するBashスクリプトが取得され、スクリプトが実行され、新しいコンテナ。 完了すると、API経由でアクセスするためのキーを受け取ります。これをアウトラインマネージャーに挿入し直す必要があります。



この簡単な方法で、アウトラインは、一般的に、ホスティング事業者に縛られることなく、任意のサーバーで構成できます。
