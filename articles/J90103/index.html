ニジニノヴゴロドでのイマジンカップのロシア決勝に招待します-4月9日10:00、ニジニノヴゴロドフェア

ソフトウェアプロジェクトと組み込みシステムのコンテストImagine Cupは今後数日間で最高潮に達します！ 2010年4月9日金曜日、 ニジニノヴゴロドで ロシアイマジンカップ決勝が開催されます。







このイベントは、ニジニノヴゴロドフェアの中央パビリオン（ 地図上 ） で10:00 （ウェルカムコーヒーをキャッチするには9:30に到着したほうがよい）に始まり、 18：30まで続きます。 非常に興味深い人々のスピーチ（Microsoft大統領ニコライ・プリャニシニコフ、ニジニノヴゴロド地域副知事、ニジニノヴゴロド州立大学学長、ズヴォリキンスキープロジェクト長など）と、賞品やギフトの興味深いコンテストに加えて、学生チームのパフォーマンスとそのハードウェアおよびソフトウェアプロジェクトも見られます：

  • ニジニノヴゴロド州立大学、SUSU、TPU、SPbSPU、TTI SFUの経済学部の学生が作成したソフトウェアプロジェクトのチームプレゼンテーションとデモンストレーション。 トピックには、セマンティックポータル、視覚障害者向けインターネット、インタラクティブなBigTurtleゲーム、水域の汚染を非接触で判定するためのデバイスなどがあります。
  • eBox / Windows Embeddedプラットフォーム上のSUSU、NNGU、NSU / NSTU / SibGUTI、MGPUおよびMIPT / MAIの学生チーム向けのソフトウェアおよびハードウェアソリューションのデモ。 ロボニャン、無人バンブルビーヘリコプター、インテリジェントなゴミ選別コンベア、インタラクティブなトレーニングテーブルがあります。
