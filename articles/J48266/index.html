Windows 7 UIの利点

Se7enを試してみた人はすでに言っています



Windows 7がVistaよりも簡単に作成されたとは信じ難い。特に多くの将来のユーザーのように、XPですぐに「7」に行く場合、改善されたものはすべてナビゲートしにくくなっています。



しかし、新しいPI（ユーザーインターフェイス）を習得するのに数分を要するペースの速い人々は、Windows 7をすばやく手に入れることができると信じています。 秘密は、古い習慣を取り除き、新しい習慣を開発することです。 いくつかの例を説明しましょう。



[実行]項目は、[スタート]メニューから直接使用できなくなりました。 検索でのみ到達できます。



どこから始めますか？ [スタート]メニューの下部にある検索ボックスは、Runユーティリティが実行できること、またはそれ以上のことをすべて実行します。 コマンドの入力を開始するとすぐにコマンドが[スタート]メニューに表示され、そこでクリックするか[Enter]キーを押して開始できます。 Runユーティリティで、パスを含むコマンドの名前を完全に記述する必要がありました。 文字をスキップすると、エラー以外は表示されません。 ソリティアをプレイしたいですか？ 検索フィールドで「sol」と入力するだけで、最初に表示されるのはSoliterです。Enterキーを押すとすぐに起動できます。







しかし、Runユーティリティの最悪の部分は、メモリのアプリケーションファイル名を知る必要があることです。 XPでは、Runユーティリティでソリティアを記述してEnterキーを押すと、エラーメッセージが表示されます。 アプリケーションファイル名がsol.exeであることを知っておく必要があります。 マインスイーパに行きたいですか？ また、アプリケーションファイル名はWinmine.exeと呼ばれます。 Vista / Se7enでは、検索フィールドはファイル名とその名前の両方で機能します。



それでも納得できない？ 古いRunユーティリティが必要ですか？ それでは、Win + Rの組み合わせを押してください。 このショートカットは、90年代半ばから今日まで機能しています。



足りない？ まあ。 Runユーティリティを追加してVista / Win7の[スタート]メニューをカスタマイズすると、1998年を楽しむことができます。 [スタート]ボタンを右クリックし、[プロパティ]を選択し、[設定]をクリックして、チェックボックスをオンにします。







新しくインストールしたWindows 7で私を誤解させるもう1つのことは、「コンピューター」、「ネットワーク」、「コントロールパネル」、「ドキュメント」などの通常のショートカットが削除された空のデスクトップです。



私はこれに同意しません。 Windows UIのデザイナーやユーザビリティテスターと話すと、これらのことをデスクトップから削除すると、初心者と専門家の両方にとってシェルの便利な使用に大きく飛躍したと自信を持って言えます。 以前は、デスクトップ上のこれらのショートカットにアクセスするには、すべてのウィンドウを最小化してから、アイコンを見つけてクリックする必要がありました。 アイコンへのこれらの不必要なステップと、ウィンドウに戻るためのさらに多くのステップにより、PCでの作業が本当に遅くなりました。 あとは、キーボードの[スタート]ボタンまたは[Windows]ボタンを押すだけで、適切な場所からワンクリックで検索できます。 または、Win + Eキーを押して、エクスプローラウィンドウで自分自身を見つけると、適切な場所へのすべてのショートカットが既にあります。



Windows 7ユーザーは、アイコンを押し戻すことは「直感的ではない」と言います。 さて、[スタート]メニューを開き、検索フィールドに「デスクトップアイコン」と入力すると、最初の結果は「デスクトップアイコンの表示または非表示」になります。 これを行う別の方法があります。 各Windowsユーザーは、デスクトップの右ボタンを押してパーソナライゼーションメニューに入り、背景画像（以前の壁紙と呼ばれていました）を変更することで簡単に学習できます。 ここで、左上隅に「デスクトップショートカットの変更」オプションが表示されます。 そして彼女はここに導く：







また、Windows 7のコントロールパネルはXPよりも直感的ではありません。 コントロールパネルのすべての要素を表示するには、もう1回クリックします。これは、XPの2倍です。 さまざまなオプションの長いリストで目的の要素を検索するよりも、便利なナビゲーションで要素を少なくすることができました。



Windows XPコントロールパネルは直感的ですか？ ほとんどが技術的な名前を持つアイコンでいっぱいのフォルダーは、直感的に見えません。 「簡単なナビゲーションで要素を少なく」したいですか？ そこにあります。 上部の検索バーを参照してください？ これは、コントロールパネルと呼ばれる古いショートカットフォルダーに対する大きな改善です。 私を信じないで、XPのコントロールパネルからスクリーンセーバー（スクリーンセーバー）を変えてみてください。 [スクリーン]アイコンをクリックして、そこにある[スクリーンセーバー]タブを選択するのは本当に意味がありますか？ これらの手順を知っているのは、このように14年間歩いているからですが、たとえば初心者にとっては直感的ではありません。

しかし、VistaとSevenでは、このために、検索フィールドに「スクリーンセーバー」という言葉を書くと、リストは不要なものを自動的に除外します。 結果は非常に有用になったように思えます：







より良い例として、Windowsエクスプローラーで隠しファイルの表示を変更するオプションを見つけてみてください。 XPでは、コントロールパネルを開き、[フォルダプロパティ]をダブルクリックして、[表示]タブを開く必要があります。 さて、普通の人はファイルの設定がフォルダのプロパティにあることを知ることができますか？ 直感的ではありません。

コントロールパネルの検索フィールドに「非表示」と入力して、結果を確認します。







最初のオプションでは、ワンクリックでダイアログボックスに目的のタブが表示されます。 これは、XPのソリューションに対する大きな優位性です。 実際、コンパートメントのあいまいなアイコンをバインドする以外、XPコントロールパネルでは何もできません。



Windows 7にアップグレードするすべてのXPユーザーの主な問題は、古いナビゲーションモデルを忘れる必要があることです。 PCを長く使用すればするほど、メニューを開いたりアイコンをダブルクリックしたりする反射神経が発達します。 この手法は90年代にはうまく機能しましたが、今日では、Windowsインターフェースのどこでも検索できるようになったため、独自の方法よりも長持ちしています。



XPのベテランの方は、しばらくして、新しいインターフェイスがそのままの状態で設計されている理由を理解してください。 私を信じて、デザイナーと使いやすさの専門家が理由でそれをやった。 インターフェイスを使用するための新しいテクニックを学習するとすぐに、PCでの作業の生産性が大幅に向上することを保証します。



PSロシア語版のWindows 7では、すべてのアクションはスクリーンショットに示されているものと同じです。
