1993年にFlashを作成した2人の開発者はどうなりましたか





1993年、2人のアメリカ人プログラマーJonathan Gay （左）とRobert Tatsumi （右）がFuture Wave Softwareを設立し、当時のタブレットにスタイラスでイラストを描くためのベクターエディターSmartSketchをリリースしました 。



1996年5月、Future Wave Softwareは、FutureSplash Animatorと呼ばれるプログラムの2番目のバージョンをリリースしました。このバージョンでは、アニメーションが既に表示されており、プログラム自体をNetscapeブラウザのプラグインとしてインストールできます。 1996年12月、Macromediaは開発者を買収し、FutureSplash AnimatorはMacromedia Flash 1.0に名前が変更されました。 9年後、MacromediaはAdobe Corporationの一部となり、製品はAdobe Flashに名前が変更されました。



このような素晴らしいテクノロジーを作成した2人のプログラマーの運命は何ですか？ 記者たちはよくちょっとした調査を行った。



ジョナサンゲイはすでに経験豊富なプログラマーでした：1985年以来（学校から）彼はシリコンビーチソフトウェアで働き、そこでマッキントッシュコンピューター用のゲームやその他のソフトウェアを作成しました：SuperPaintグラフィックエディター、Digital Darkroomフォトエディター、Airborne！、Dark Castleゲームなど



ゲイとTはMacromediaでFlashをリードしています。 ジョナサンは会社開発の副社長に任命されましたが、特にFlash 1-4のコードを書きました。特に、ベクターグラフィックスの低レベルラスタライズをすべて書き、ブラウザ用の新しい描画ツールとFlash Player Netscapeプラグインを作成しました。 次に、TatsumiはUIを担当し、オーサリングツールFlashを作成しました。 Flash 1-4の初期のリリースでは、開発チームには新しいプログラマーが実際には補充されませんでしたが、テクノロジーが成功すると、集中的な採用が始まりました：2001年までに50人が既にFlashに取り組んでおり、プロジェクトの創設者はコーディングから計画およびチーム管理に移行しました。



2005年4月18日に、AdobeがMacromediaを購入していることが知られるようになり、Jonathan Gayはこれに参加しないと発表し、Macromediaのテクニカルディレクターの職を辞めようとしました。



2006年8月、GayとTatsumiは新しいSoftware as Artを設立し、後にGreenboxに改名しました。Greenboxは 、家庭のエネルギー消費を視覚化および管理するシステムを作成しました。







Software as Artスタートアップの初期投資は、Macromedia CEOのRob BurgessとMacromedia CFOのBetsey Nelsonによって行われました。 その結果、Greenboxも成功裏に売却されました。今回は2009年にSilver Spring Networkに売却され、IPv6をサポートするスマートグリッドの実装のための統合ソリューションに取り組んでいます。



Robert TatsumiはSilver Spring Networkに参加し、プリンシパルプログラムエンジニアとして働いています。これには、Python、Django、およびJQueryを使用した顧客サービス用のフロントエンドポータルの開発が含まれます。



ジョナサンゲイはITビジネスを完全に辞め、有機農業を始めました 。 特に、彼は非常に高価な牧草で育てられた牛肉を特定の顧客に販売しています。
