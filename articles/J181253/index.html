Motorola Atrix 2からSamsung Galaxy Note 2に切り替えた方法

ある程度ガジェットマンであるため、以前のスマートフォンを長い間（1年以上）使用していたと言えます。 これは、モトローラのAtrix 2モデルであり、市場への参入直後の文字通り米国への旅行中に取得されました。 モトローラの携帯電話は暖かさと不安で扱います（1998年から定期的に使用しています）が、しばらくの間公式に販売されていないのは残念です。

この投稿では、あるデバイスから別のデバイスに移動した経験と、Motorola Atrix 2およびSamsung Galaxy Note 2の印象を共有したいと思います。



Motorola Atrix 2スマートフォンは同社の主力モデルであり、その特性と外観から確認されました。



Motorola Atrix 2 は調和しているように見えます。 鋭い角や突出した部分はなく、スマートフォンはほぼ完全に手の中にあります。 背面パネルはゴム製のソフトタッチプラスチックでできており、縦方向の線が立体的にパターン化されているため、ガジェットは滑り落ちることさえありません。 ちなみに、リアパネルの品質は私を喜ばせ、それは非常に柔軟で、取り外された状態では、結果なしでヘリカルトーションを簡単に受け入れられます。 同時に、彼女はスマートフォンにしっかりと座り、よろめかず、曲がりません。



ディスプレイの周囲のフレームは、特殊な「磨かれたスチール」コーティングが施されたプラスチックで作られており、画面のわずかに上にあります。 さらに、フレームは実質的に上端と下端まで移動せず、ソフトタッチの背面カバーで閉じられています。 Atrix 2スクリーンはGorilla Glassを保護します。 ケース素材の品質は非常に高く、常にアクティブであまり慎重に使用していません（保護フィルムとカバーは認識しません）。ディスプレイにも画面フレームにも傷がありません。 ちなみに、デバイスの左側にはmicroUSBとmicroHDMIの2つのコネクタがあり、積極的に使用しているため、フレームについて最も心配していました。 プラグの傷が現れるのはこの場所だと思っていましたが、私は間違っていました。これは素晴らしいことです。



Motorola Atrix 2 の画面は画質に満足しており、明るく、コントラストがあり、鮮明です。 そして、視野角はほぼ最大です。 同時に、ここではIPSマトリックスではなく、qHD解像度（540 x 960ピクセル）の最も一般的な容量4.3インチTFTスクリーンを使用しています。



デュアルコアGHzプロセッサーのTexas Instruments OMAP4430（アーキテクチャARM Cortex A9）とグラフィックアクセラレーターPowerVR SGX 540のおかげで、優れたスマートフォンのパフォーマンスが提供されます。デバイスのRAM 1 GB。 ハードウェアリソースは、Dead Triggerのような現代の「重い」ゲームを実行するのに十分です。 また、インターフェイスをスムーズに操作するには、リストされているコンポーネントで十分です。 ところで、インターフェースについて。 Motorola Atrix 2は、Androidオペレーティングシステムバージョン2.3を実行していました（販売開始時）。 「ロボット」は裸ではなく、わずかにカスタマイズされ、独自のソフトウェアが追加されています。 デバイスの内部メモリは8 GBに加えて、microSDカードのサポートです。



スマートフォンは、オートフォーカスを備えた8メガピクセルのカメラを使用しており、かなり高品質の写真を提供します。 確かに、これは日中または屋内で撮影された写真にのみ適用されます。 夕方に撮影された写真はやや粒状です。









一般的に、デバイスはまともですが、時間が経ち、新しいものを購入するときがきました。 また、私はChromeブラウザの欠如に飽きました。 実際、ロシアに戻った後、Motorola Atrix 2はAT＆Tオペレーターの下からロック解除され、同じAndroid 2.3.6で新しいカスタムファームウェアを受け取りました。 2012年の冬の初めに、デバイスはAndroid 4.0オペレーティングシステムを備えた新しいファームウェアが利用可能であることを通知しましたが、問題が発生しました。 アップデートがダウンロードされ、インストールが開始されますが、約3分の1のエラーがクラッシュし、その後スマートフォンが再起動されます。 すべての障害は、同じカスタムファームウェアです。 一般的に、新しいAndroidスマートフォンを購入することを最終決定しました。



新しくて強力なものが欲しかった。 選択プロセス全体については説明しませんが、スマートフォンとタブレットのハイブリッドな「ファブレット」という、私にとっては珍しいフォームファクターに決めました。 Samsung Galaxy Note 2を購入したと推測するのは簡単です。



私は長い間、新しいスマートフォンがモトローラよりも著しく大きくなり、使用するのが不便になるのではないかと心配していました。 原則として、そのようになったので、約2週間Samsung Galaxy Note 2に慣れました。 寸法を比較しましょう：

Atrix 2では66 x 126 x 10.4 mm、Galaxy Note 2では80.5 x 151.1 x 9.4 mm





私が最初に気づいたのは 、イヤホンを初めて耳にはっきりと入れるとは限らず、音量レベルが良くなるまでデバイスを動かさなければならなかったことです。 また、Samsung Galaxy Note 2の寸法では、片手で快適に操作することはできません。 メーカーがこれを処理し、設定メニューに別の項目「片手操作」を含めました。 これを使用すると、ダイヤルパッド、仮想キーボード、電卓、電話ロック解除要素などを画面の左端または右端に近づけることができます（右利きおよび左利きの場合）。





それにもかかわらず、これは状況を救いませんでした-片手でSamsung Galaxy Note 2を完全に制御することはほとんど不可能です。



私が言ったように、私はあらゆる種類のカバー、スクリーンプロテクターなどに反対しています。しかし、ガジェットを使用してから1ヶ月後、私はまだ背面パネルにシリコンオーバーレイを取得する必要がありました。 理由は簡単です-Galaxy Note 2の背面はかなり滑りやすい光沢のあるプラスチックで作られており、デバイスが手から滑り落ちることを常に恐れていました。 ケースはこの問題を解決しました。現在、デバイスは乾いた手と濡れた手の両方に安全にあります。



Samsung Galaxy Note 2のフロントパネルは、疎油性コーティングが施された保護ガラスで覆われています。 指や耳の痕跡がそれほど速く現れず、布で簡単に取り除くことができます。 画面の上には、近接センサーと光センサー、フロントカメラレンズがあります。 後者の解像度は1.9 MPで、ビデオ通話または自画像に使用されます。 Galaxy Note 2ディスプレイの下には、ハードウェアの中央に1つ、側面に2つのタッチボタンの3つのキーがあります。 原則として、デザインは当時人気のあったGalaxy S IIIのデザインと同じです。



画面に移りましょう。 以前は、AMOLEDディスプレイを扱う必要がありましたが、率直に言って、「vyrviglaznymi」の色があまり好きではありませんでした。 Samsung Galaxy Note 2の画面から同じ感情を期待していましたが、店内でスマートフォンをひねると、すべてがそれほど悪くないことに気付きました。 まず、このモデルのマトリックスはPenTileテクノロジーを使用して作成されたものではなく、本格的なRGBディスプレイです。 視覚的には、最初の画面と2番目の画面を区別することはほとんど不可能だと誰かが言います。 これがそうかもしれないかどうかはわかりませんが、Galaxy Note 2の画面上の画像は、たとえばMotorola RAZR XT910のように、ディスプレイ上の画像よりも自然に見えました。 第二に、画面設定には「ナチュラル」などの興味深いモードがあります。その後、色が少しフェードし、画像がIPSマトリックスの画像に似たものになります。 ダイナミック、標準、ムービーの3つのモードも提供されます。





Samsung Galaxy Note 2の表示特性については、マトリックス-HD Super AMOLED、対角-5.5インチ、解像度-1280 x 720ピクセル、アスペクト比-16：9、ピクセル密度-256 /インチ。 ピクセルは、よく調べて初めて顕著になりますが、日常の作業では気づかないでしょう。 画質は非常に良好で、画像は鮮明で、コントラストがあり、明るく、視野角は広くなっています。 一般的に、私はかつて嫌いだったAMOLEDテクノロジーにもかかわらず、マトリックスに満足していました。



Samsung Galaxy Note 2は、これまでで最も生産性の高いスマートフォンの1つと考えられています。 1.6 GHzの周波数で動作するクアッドコアSamsung Exynosプロセッサを搭載しています。 ちなみに、この韓国企業の別の元旗艦には同じプロセッサが搭載されています-これはサムスンギャラクシーS IIIです。 Meizu MX2やHighscreen Explosionなど、これらのチップを使用する他のブランドのスマートフォンがあるのは不思議です。 ただし、注2に戻ります。今日、RAMは最大-2 GBです。 ガジェットを使用している間、彼は決して「考えた」ことはなく、ハードウェアリソースはどのタスクにも十分です。



Samsung Galaxy Note 2には3つのバージョンがあり、内部メモリの容量が異なります-これらは16 GB、32 GB、64 GBで、iPhoneに最適です。 確かに、このスマートフォンは「アップル」製品とは異なり、メモリカードをサポートしています。 最も安価なバージョン-16 GBの内部メモリがあり、そのうち11 GB未満がユーザーに使用可能です。



Samsung Galaxy Note 2はAndroid 4.1 Jelly Beanオペレーティングシステムを実行しており、その上に独自のTouchWizシェルがインストールされています。 インターフェースが元のAndroidとは明らかに異なることは明らかです。 より美しく、便利で、さまざまな種類のランチャーの助けを借りずに、すべてを「自分用に」設定できます。



Samsung Galaxy Note 2のブランド化された「 チップ 」のうち、マルチウィンドウについては別に注意します。 画面の左側には半円があります。 それを引っ張ると、インストールされたアプリケーションのショートカットのあるパネルが離れ、そこから、いつでも、どのプログラムからでも起動できます。 非常に便利です、私はあなたに報告します。 率直に言って、今ではマルチウィンドウなしで他のスマートフォンに切り替えるのは難しいでしょう。



ペンSペンは言うまでもありません。 デバイスの下端に隠れ、しっかりと固定されています。 ちなみに、最初のノートのような丸い形ではなく、持ちやすいように長方形になっているため、手で回転することはありません。 ペン先は可動式であるため、押下の度合いを調整できます（合計1024レベルの押下が認識されます）。 S Penを使用すると、手書きのメモを作成したり、描画したり、画面の一部を切り取ったりできます。 スマートフォンには、関連するアプリケーションのパッケージ全体があります。 ちなみに、ペンを引き出すと、S Penを操作するためのアプリケーションを含むコンテキストウィンドウが自動的に開きます。







Samsung Galaxy Note 2に実装されているもう1つの興味深いテクノロジーは、 Air Viewです。

ペンを画面に持ってくると、コンテキストメニューがポップアップします。 たとえば、ギャラリーのミニチュア写真にSペンを向けると、サイズが大きくなります。 ブラウザで行が強調表示されるか、説明情報を含むウィンドウが表示されます。 一般に、このペンの使用は非常に便利です。 私はいつもそれを使うとは言いませんが、かなり頻繁に使います。

一般に、追加のオプションと設定は多数あります。全員について話す理由はありません。必要に応じて、インターネットでSamsung Galaxy Note 2の全レビューを読むことができます。

写真の最大解像度は3264 x 2448ピクセルです。 ビデオ録画は、1秒あたり30フレームの周波数で、フルHD 1080p形式（1920 x 1080ピクセル）で行われます。 品質は許容範囲を超えています。 写真の例を次に示します。
