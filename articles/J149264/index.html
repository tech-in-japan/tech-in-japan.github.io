Windows Camp-Windows 8およびWindows Phone 8のすべて





今日、 8月の8日目に、2012年9月7日にモスクワで開催されるWindows Camp会議の登録を開始します 。

この会議は、Microsoftの最新プラットフォームであるWindows 8およびWindows Phone 8専用です。



Windows 8は、最も人気のあるオペレーティングシステムの新しいバージョンであり、10億人以上が使用しています。

Windows Phone 8は、スマートフォン向けオペレーティングシステムの新しいバージョンであり、ちょうど1年前にロシアで正常に起動され、10万を超えるアプリケーションがあります。

Windows 8およびWindows Phone 8のすべてのデバイスには、アプリケーションストアがあります。



Windows Campでは、このユニークな機会を利用して世界中のユーザーと直接やり取りする方法を紹介します。



一方では、プラットフォーム自体とそれらの開発原則について説明します。設計から始めて、アプリケーション開発の最新ツールキットであるVisual Studio 2012のストーリーを続け、収益化とアプリケーションストアを提供する機会についてのストーリーで絵を完成させます。 一方、すでに開発を始めており、高度な開発機能に精通したい人のために、パラレルトラックを用意しました。



会議は、すでに人気を博している同僚やアプリケーション開発者と仮想化してチャットするための最良の方法の1つです。 メインレポートの間に、パートナーや独立した開発者からのショートストーリーや、ほぼ毎日使用する実際のアプリケーションの開発に関するストーリーが待っています。 コミュニティの専門家、そしてもちろんマイクロソフトの専門家に質問する機会があります。 それだけではありません。



パートナーの展示により、最新の技術革新に慣れることができます。 また、マイクロソフトの専門スタンドでは、クラウドサービスまたはソフトウェアライフサイクル管理テクノロジーとの統合について質問することができます。



会議サイトは従来、開発者プレビュー形式で開始されており、イベント自体の概算とともに更新されます。

近い将来、より詳細なレポートスケジュールが表示されます。その構造は上記のとおりです。

事前登録により、Windows Campへの参加は無料です-確認が追加で送信されます。

急いで登録 -場所がすぐになくなる。



Windows Campでお会いしましょう！
