Opera Miniのユーザー数は3,000万人を超えました









2009年8月のデータによると、1秒ごとに新しいユーザーがモバイルブラウザに表示されました



2009年9月24日-オスロ、ノルウェー

Opera Miniは、世界で最も人気のあるモバイルブラウザーのタイトルを確認します。今年8月、ユーザー数は3190万人を超え、7月よりもほぼ10％増加しています。 したがって、8月のOpera Miniには毎秒少なくとも1人のユーザーが追加されました。 これらのデータは、モバイルインターネットの状態に関するOpera会社のレポートに含まれています。



月次レポートには、世界および個々の地域におけるモバイルインターネットの動向に関する情報が含まれています。 そのフルバージョンは、 www.opera.com / smw （英語）で入手できます。 一般的なレビューに加えて、8月のレポートには、ヨーロッパのモバイルWebの状態に関するより詳細な情報が含まれています。



一般情報を報告する

-2009年8月、3190万人以上がOpera Miniブラウザを使用しました-7月より9.9％、2008年8月より147％増加しました。



-1か月あたり3190万人のユーザーが139億ページ以上閲覧しており、これは7月の結果より15.7％、昨年8月の結果である234％です。



-Opera Miniユーザーは約2億900万MBのデータを携帯電話にダウンロードしました。 したがって、7月の結果は11.7％を超え、2008年8月と比較して、トラフィックは222％増加しました。 Opera Miniでは、データを最大90％圧縮できます。 情報が圧縮されていない場合、ユーザーは合計1.9ペタバイトのデータを携帯電話にダウンロードする必要があります。



-8月にOpera Miniを使用した数十の最大地域の構成（降順）：ロシア、インドネシア、インド、中国、ウクライナ、南アフリカ、アメリカ、ナイジェリア、イギリス、ポーランド。



-ベトナムは、8月のユーザー数が20％増加した11番目の位置を占めています。 また、近い将来、フィリピン、ケニア、フランスがOpera Miniを使用して上位10の地域に参入する可能性があります。



-7月と比較してOpera Miniユーザーの相対的な増加が最も大きい国（上位30のブラウザー地域）は、オーストラリア（52％増加）、ジョージア（34％）、ベトナム（20％）、ケニア（19％）のように見えました。 、フランス（16％）、フィリピン（15％）、ブラジル（14％）、カザフスタン（13％）、パキスタン（11％）、マレーシア（10％）。



-世界のトップ10リストに掲載されている地域を除く、Opera Miniのユーザー数に関するヨーロッパの12か国には、（降順で）ドイツ、フランス、チェコ共和国、イタリア、スペイン、トルコ、オランダ、スロバキア、リトアニア、ルーマニアが含まれます。 トルコは、新規ユーザーの相対的な成長をリードしています。この国では、昨年8月と比較して218.8％に達しました。 トルコに続いて、年間100.7％の指標でフランス、72.2％でチェコ共和国が続きます。



-ロシアは引き続きOpera Miniの最大の消費者です。 年間で、国内のブラウザユーザーの数は57.3％増加し、ユーザーが閲覧するWebページの数は125.3％増加しました。 平均して、各ユーザーは月に411のWebページを閲覧しました。

Opera Softwareの最高経営責任者であるJon von Tetzchnerは次のように述べています。 -できる限り多くの携帯電話がOpera Miniブラウザをサポートするように常に努力しています。 私たちの主な仕事はウェブを大衆にもたらすことであり、今日の携帯電話は世界で最も人気のある通信デバイスです。 問題は、ほとんどの人がモバイルデバイスを使用してインターネットにアクセスするかどうかではありません。 唯一の質問は、これがいつ起こるかです。 Opera Miniは、モバイル端末を使用してWebリソースへの迅速かつ効率的なアクセスをユーザーに提供する準備ができました。

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
