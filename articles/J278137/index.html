Php



  • RFC：汎用配列 - 汎用 に関するRFCの補足。 型付き配列を実装することが提案されています。 
    $counts = array<string, int>(); // array<TKey,TValue> $counts["kittens"] = 12;
  • RFC：PHP 7.1の非推奨-このドキュメントには、PHP 7.1で非推奨となり、PHP 8以降に完全に削除されることが提案されている機能のリストが含まれています。リストは更新されます。
  • RFC：Callable Constructors-呼び出し可能構造でコンストラクターを呼び出すことができるようにすることが提案されています。 例： 

     $fn = "Foo::__construct"; $object = $fn();
    に等しい 

     $object = new Foo();
