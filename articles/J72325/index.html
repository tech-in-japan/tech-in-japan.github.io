インストール



多数のシステム用のバイナリバージョンがあります： RPM 、 DEB 、 TAR.GZ 、 Solaris、さらにはWindowsでも 。 リポジトリからDEBをインストールする方法のみを説明します。 コンソールに突入し、ルート権限を取得します



sudo -i



キーを取得



cd / root

wget http://www.webmin.com/jcameron-key.asc

apt-key add jcameron-key.asc



アプリケーションソースを追加



gedit /etc/apt/sources.list



テキストエディターで行を追加する



deb http://download.webmin.com/download/repository sarge contrib



保存して、エディターを閉じます。 更新してインストールする



apt-get update

apt-get webminをインストール



インストールはいくつかの依存関係を拡張するはずです、私たちは同意します。
