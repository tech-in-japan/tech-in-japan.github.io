アマゾンウェブサービスがS3サービスコストを削減





Amazon Web Servicesはおそらく最も人気のあるクラウドサービスの1つです。 Amazonのサービスは現在、何百万人もの顧客によって使用されています。当然、その中の多くはAmazonのサービスの価格を下げたいと考えています。 先日、同社はAmazon Simple Storage Service（S3）の価格引き下げを発表したメッセージを公開しました。 現在、価格の引き下げは、米国にあるAmazonデータセンターでのみ有効です。



おそらく、この方法で会社は、会社のサービスの価格レベルが高い新規顧客を引き付けるでしょう。 はい、この分野での競争が徐々に増加していることを忘れることはできません。そのため、低価格は新しい顧客を引き付けるだけでなく、古い顧客を引き付けることができます。 減少についてはそれほど重要ではありませんが、大量のデータを扱う人にとっては、これはすべて関連しています。



Amazon Web Servicesの副社長によると、クラウドに50テラバイトを超えるデータを保存する顧客の節約は約12％であり、500テラバイトを超えるデータを保存する顧客の節約は13.5％です。 実際、そのようなボリュームの場合-非常に重要な指標です。







価格の引き下げに加えて、同社はまだベータ版である新しいAWS Storage Gatewayサービスも開始しました。 このサービスを使用すると、Amazonから「クラウド」に独自のデータを安全にアップロードできます。 同社によれば、このサービスは主に、クライアントアプリケーションからAmazon EC2にデータを迅速かつ安全にダウンロードすることを希望しているお客様を対象としています。 たとえば、新しいサービスは重要なデータのコピーを作成するのに役立ちます。



このサービスでは、Amazon EC2に追加することにより、個別のAmazon EBSパーティションを作成できます。 新しいサービスの価格は非常に高いですが、結局のところ、それはすべて企業クライアントに基づいて作成されます。 そのため、AWS Storage Gatewayの各チャネルについて、クライアントは月額125米ドルを支払う必要があります。 確かに、サービスのテストには無料の期間があります。これは最大2か月です。
