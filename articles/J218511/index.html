ASUS Pro BU401Lノートブックを確認する

今日は、特別なラップトップ-ASUS Pro BU401Lについてお話します。 ASUS Proシリーズのラップトップは、企業セグメント、さまざまな機関、政府機関向けに特別に設計されています。 高度なハードウェアデータ保護技術、高度な管理機能、高性能を備えています。 ラップトップは、顧客の厳しい要件を満たすために、さまざまな業界で厳密にテストおよび認定されています。









技術仕様



プロセッサー： Intel Core i7-4558U（2.40 GHz、6 MBキャッシュ）

OS： Windows 8 Pro

メモリ： 8 GB DDR3 1600 MHz SO-DIMM

ディスプレイ： 14.0 "LCD 16：9、1366x768、マット、LEDバックライト

ビデオカード： Intel HD 5100

ストレージ： 256GB SSD

ネットワーク： LAN 10/100/1000 Mbps、WLAN 802.11 a / b / g / n、WiDi、Bluetooth 4.0をサポート

インターフェイス： SDカードリーダー、3.5 mmヘッドセットジャック、D-Sub VGA、3xUSB 3.0、LAN RJ45、HDMI、生体認証スキャナー

ウェブカメラ： HD

バッテリー： 4セル、6840 mAh、53 Wh

寸法： 339 x 234 x 19.78 mm

重さ： 1.64 kg



パッケージバンドル



ラップトップが納品される箱は非常によく見えます。 カラー印刷を使用するマルチメディアラップトップとは異なり、ここではすべてが厳格です。 それでも、これはビジネス用のデバイスであり、余分なものはないはずです。そうすれば、これを確信するだけです。





パッケージは環境に優しい素材で作られています。 すべての情報は、環境に悪影響を及ぼさない大豆インクを使用して印刷されます。 段ボールの80％はリサイクル素材で作られています。 すべてのパッケージは完全にリサイクルできます。





ボックスを開き、段ボールの別の層を取り外してラップトップを入手します。 ラップトップの下には充電器とドキュメントがあります。





ご覧のとおり、ASUSの通常の充電器を前に持っています。 サイズが小さく、バッグ内のスペースをほとんど取りません。 充電器の電力は65ワットです。





外観



それがこのクラスのラップトップで私が気に入っているのは、余分なものすべての深刻さと欠如です。 ここでは、ウィンドウ上では見栄えは良いが、オフィスでの作業には不便な光沢のある画面は表示されません。 ラップトップからの充電にはまだ光沢のあるスペーサーがありますが、すべての指紋を絶えず集めて光の中でキラキラと光る非実用的な材料はありません。 実用的な素材と厳格なデザインのみ。





ラップトップの蓋は黒、またはむしろ濃い灰色で、炭素繊維でできています。 この材料は通常のプラスチックよりも強く、デバイスの重量を減らします。 カーボンはより高い負荷に耐えることができ、損傷からスクリーンを保護します。









スクリーンは1つの大きな金属ヒンジに取り付けられています。 最大開き角度は135度です。 蓋は滑らかな乗り心地で、どの位置でもロックできます。





ラップトップの底面は、完全に金属シートで覆われています。





上部には、前面に換気グリルがあります-スピーカー用の穴。





別の換気グリルがケースとスクリーンの間にあり、そこから熱風が吹き戻されます。





取り付け設計は簡単です。周囲のネジを外すだけで、ラップトップの内側にアクセスできます。 後部のゴム製脚の高さは、前部よりわずかに高くなっています。 これにより、ラップトップがわずかに傾けられ、キーボードでの作業が快適になり、換気グリルの隙間が広がり、冷却が向上します。





ケースは完全にアルミニウム製です。 キーボードとタッチパッドを覆う表面は、洗練された質感を持っています。 内部には滑らかな輪郭はありません。すべてが厳密な直線と角度です。 ラップトップの厚さは20.5 mmで、これは最新のウルトラブックのレベルです。





スリムなボディと最新のソリューションの使用により、ラップトップは非常に軽く、わずか1.64 kgでした。





インターフェース



メインインターフェイスはラップトップの側面に配置されます。

左側を検討してください。 最初は充電器用のコネクタです。 興味深い機能が見つかりました。 プラグが完全に挿入されていない場合、充電が開始されず、ケーブルを再接続する必要があることを警告するウィンドウが画面に表示されます。 多くの人が、充電が動作していないという事実に注意を払わずに、放電したラップトップを思いがけず発見したという状況に遭遇したと思います。

LANポートはまだここにあります。 会社のローカルネットワークへの接続に役立ちます。 次はフルサイズのHDMI 1.4 1080pとUSB 3.0です。





右側には、SDカードリーダー、ヘッドセットを接続するための3.5 mmミニジャックがあります。 次に、モニター用に2つのUSB 3.0とD-sub出力があります。 そのような時代錯誤がなぜここにあるのか尋ねてください。 シンプルで、すべてのプロジェクターがD-Subをサポートしているので、たまたまプレゼンテーションを表示するのであれば、HDMIよりもむしろ会うでしょう。 ケンジントン城のすべてのマウントを終了します。





メインインジケータはラップトップの前面にあります。





音



ラップトップのサウンドシステムはSonicMasterをサポートしており、ASUSの他のノートブックモデルでうまく機能しています。





ASUSは、ラップトップでの高品質なサウンドの実装において幅広い経験を持っています。 サブウーファーとおそらくラップトップの中で最高の音とAsus N750JVとは何ですか。 ここでは、エンジニアは企業セグメントをバイパスせず、このような薄いケースで優れたサウンドサブシステムを実現しました。 出力では、最大音量で喘鳴せずにサラウンドサウンドが得られます。 また、システムには、ソースのタイプに応じて音を微調整するためのソフトウェアがプリインストールされています。 AudioWizardでは、音楽、映画、録音、ゲーム、音声増幅など、さまざまなモードを選択できます。



スクリーンとカメラ



このモデルでは、解像度1366 x 768ピクセルの14インチ16：9スクリーンが使用されます。 1600 x 900ピクセルの解像度の変更もあります。

LEDバックライトを備えたTNマトリックスは、視野角が広く、つや消し仕上げによりグレアから保護されています。 私の印象では、画面の明るさは十分ではありませんが、これはすべてのマット画面で問題になっているようです。 明るい日光の下では、読むのが困難になります。

スクリーンは金属フレームで囲まれています。 このソリューションのおかげで、ゴム製の縁が周囲全体に沿って走り、画面がキーボードを傷つけたり、閉じたときにほこりがラップトップに侵入したりしません。





画面の上には、ビデオコール用のHDウェブカメラがあり、仕事のインジケータが付いています。 近くには光センサーとマイクがあります





入力機器



このクラスのラップトップキーボードは、効率的な作業のための重要なツールの1つです。 目の前には、フルサイズのキーを備えた使い慣れたアイランドキーボードがあります。 彼らは良いリターンと素晴らしいカバレッジを持っています。









キーのすべてのキャラクターはレーザー彫刻されており、暗闇で光っています。





バックライトをオフにすることができ、明るさの3つのレベルがあります。 いつものように、カーソルキーは小さくなっています。





キーボードには液体が保護されているため、誤ってコーヒーをこぼしてもコンピューターが損傷することはありません。





キーボードの下には巨大なタッチパッドがあります。 ノートパソコンのカバーのように濃い灰色をしており、全体的なデザインに完全にフィットします。 センサーのサイズは10.7 x 6 cmで、タッチ面がざらざらしていて快適です。 ボタンはタッチパッドの下にあります。 私はすでにそのようなボタンから離乳することができました。 大きなストローク、静かなクリック音、快適な応答性を備えています。 もちろん、MultiTouchとさまざまなジェスチャーがサポートされています。





ビジネスラップトップの必須属性は指紋スキャナーです。 これを使用して、ログインしたり、情報を保護したりできます。 高周波センサーが使用されます。これにより、皮膚の生きている層の描画が読み取られ、指の表面が汚れている、脂っこいなどである場合、認識精度は低下しません。





性能



高性能とエネルギー効率を確保するために、最新世代のIntel Haswellプロセッサが使用されています。 私のサンプルには、Intel Core i7-4558Uがあり、4つのスレッドを一度に処理できる2つのコアがあります。 各コアは2.8 GHzの基本周波数で動作しますが、ターボブーストテクノロジーを使用して3.3 GHz（2アクティブコア）まで自動的に加速できます。





Core i7-4558Uのパフォーマンスは、古いCore i7-3520M（Ivy Bridge）よりもわずかに高く、わずかに高い周波数で実行されます。 しかし、新しい命令セット（AVX2など）を使用するアプリケーションでは、パフォーマンスははるかに高くなります。 そのため、このプロセッサは、オフィスやマルチメディアアプリケーションだけでなく、より要求の厳しいタスクにも自信を持って対応します。

最大負荷では、冷却システムの動作は明らかに聞こえます。 開発者は、より強力なフォールトトレランスを提供する強力なクーラーをインストールすることで、再保険を受けたようです。







統合されたIris Graphics 5100は40のアクチュエータを備え、200 MHz（Turbo Boostモードで最大1200 MHz）の周波数で動作し、HD 4000よりもはるかに高速です。

より要求の厳しいユーザー向けに、個別のグラフィックカードNVIDIA GeForce GT730Mを使用した修正が行われます。





ボードは8GB DDR3 1600 MHz RAMにはんだ付けされていますが、メモリ容量を最大12GBまで拡張するための別のスロットがあります。





LITEON SSD、モデルLCS-256M6S。





CrystalDiskMarkは高い結果を示し、SSDドライブは制限なく動作します。





最小のWindowsスコアはグラフィックカードにかかっていますが、統合されたグラフィックカードの評価は印象的です。





バッテリーとランタイム



6840 mAhの容量を持つ内蔵バッテリーの動作時間に満足しました。

HDビデオを中程度の明るさで見ると、ラップトップは7.5時間続きました。

平均的な画面の明るさとインターネットサーフィンでは、動作時間は8時間でした。

ネットワークをオフにし、最小輝度を設定し、省エネプロファイルをオンにし、オフィスアプリケーションでの作業に制限すると、最大12.5時間に到達できます。





このような小さなラップトップで非常に優れたパフォーマンス。 出張中はほとんどすべてのタスクを実行でき、ラップトップの電源がオフになることはほとんどないため、出張に安全に行くことができます。

おわりに



その結果、ASUS Pro BU401Lはビジネスユーザーにとって優れたツールであることがわかりました。 厳格なデザインと高品質の素材は、あらゆる企業スタイルに完全に適合します。 優れたパフォーマンスと長時間の作業により、路上でも深刻な問題を解決できます。 ところで、このラップトップは軍事規格MIL-STD-810Gに準拠しています。 その仕様には 800ページもあり、ラップトップの信頼性を自分で確認できます。 テストには、衝撃、振動、振動、湿度の変化、温度、圧力が含まれます。 したがって、このラップトップを購入すると、効果的な作業のための本格的なツールが手に入ります。
