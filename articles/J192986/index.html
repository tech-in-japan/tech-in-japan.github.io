RTMおよび任意の「方向」で、任意の言語およびバージョンのWindows 8.1を更新する

今日、インターネットでWindows 8.1 RTM（最終バージョン）を発見し、アップグレードすることにしましたが、そこにはありませんでした。



Windows 8.1 x64 Enterprise Previewをインストールしましたが、Microsoftが述べたように、プレビューではRTMにアップグレードできません 。

したがって、インストールされているすべてのプログラムなどを失いたくない、Googleに行って、cversion.iniファイルのバージョンを変更できることがわかりましたが、5回交換しようとしても何も起こりませんでした。

RTMバージョンはen-US言語であり、ru-RUを持っていることが判明しました。Microsoftによると、これがすべてを削除する理由です。



10分ほど考えた後、Windows 8.1をインストールする別のより興味深い方法を見つけました。

それはすべて新しいインストーラー設計に関するものであり、ユーザーに提供されるすべての選択ポイントはインターフェースにありますが、非表示になっています。

イルカのウィンドウとそのオブジェクトを操作するための小さなプログラムをスケッチしました。ここに手順とスクリーンショットがあります：



1. Windows 8.1インストーラーを実行し、インストール選択ウィンドウを表示します

2. ShowHideControlsプログラムを実行します（ちなみに、 AnVirタスクマネージャーなど、外部プログラムウィンドウにアクセスできる他のプログラム）

3.タスクでWindowsインストーラーを探しています

4.必要なアイテムを探しています。この場合は「Windowsの設定、個人用ファイル、およびアプリケーションを保存する」です

5.「クリック」ボタンを押します（カーソルでのクリックの模倣です。ちなみに、このアイテムを表示してカーソルでクリックできます）

6.パラメーターを確認した後、 必要なアイテムが選択されていることを確認してください！ （それ以外の場所ではない）

  • Windowsビルドバージョンを上げ下げできます
