Q＆Aシステムの比較

Q＆Aサイトは最近非常にトレンディになりました。 このジャンルの議論の余地のないリーダーは、Yahoo！ 回答。他の同様のプロジェクトをバックグラウンドに残しますが、それらの多くがあります。



これらのサイトのうち、実際に質問に答えているのはどれですか？ 本当の専門家はどこですか？ MIT Technology Reviewのジャーナリストは、 Yahoo Answers 、 Microsoft Live QnA 、 AnswerBag 、 Yedda 、 Wondir 、 Askvilleの 6つのサイトを数日間徹底的に調査しました 。 結果は雑誌に掲載されています。



この研究は2つの部分で構成されていました。 最初の段階では、よくある「5秒ルール」（床に落ちた食べ物が床に長く横たわっている場合は口に入れないでください）を信頼するかどうかの質問に対するアーカイブの正しい答えがあるかどうかを判断するために、質問と回答のベースを調べました5秒）。



第2段階では、調査した各サイトで2つの元の質問が公開されました。「なぜモルモン教徒はユタに定住したのですか？」と「揚げチーズサンドイッチを作る最良の方法は何ですか」。 ご想像のとおり、これらの2つの問題には、まったく異なる層の専門知識と異なる専門家が関与している必要があります。 最初のケースでは、それはあまり知られていない米国の歴史であり、2番目のケースでは、すべての訪問者が簡単な質問の議論に参加できます。 ここで、ディスカッションの質はレシピの数で判断できます。



各サイトは、機能の完全性と独創性について1〜3ポイント、3つの質問それぞれに対する回答の質について最大3ポイントを受け取ったため、可能な最大ポイント数は12でした。



この興味深い実験の結果は非常に予測可能であることが判明しました。 勝者の中には、インターネットの巨人であるヤフー、マイクロソフト、アマゾンの開発があります。 小規模な新興企業のプロジェクトは、競合することはできません。 まあ、大きなマージンによる絶対的な優位性は、Yahoo！を維持します。 回答 プロジェクトが始まってわずか1年で、視聴者が1660万人に増えたことを思い出してください。



以下に、各サイトのより詳細な結果へのリンクを含む要約を示します。



1. Yahoo Answers （3 + 3 + 2 + 3 = 11 ）

2. ライブQnA （1 + 2 + 2 + 2 = 7 ）

3. アスクビル （3 + 0 + 2 + 1 = 6 ）

4-6。 AnswerBag （1 + 0 + 1 + 2 = 4 ）

4-6。 ウォンディル （1 + 0 + 2 + 1 = 4 ）

4-6。 イェッダ （2 + 0 + 1 + 1 = 4 ）
