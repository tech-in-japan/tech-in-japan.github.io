C ++ロシア2018カンファレンス

みなさんこんにちは！ 次回のC ++ロシア会議が間近に迫っているので、前向きな報告と同僚とのコミュニケーションを見逃したことを願っています。 今年、会議は再び3つの革命の都市で開催されます。



会議は4月19日から21日までサンクトペテルブルクで開催されます。 4月19日はマスタークラスの日、20-21日-メインプログラムです。 ジョン・カルブは会議の初日を開きます。 会議レポートのトピック：メモリ、クラッシュレポート、非同期、ソフトウェアトランザクションメモリ、反射メタプログラミング、範囲、アクター、ハーブサッター。



カットの下の詳細。



報告書



そのため、カンファレンスはJon Kalbによって開かれます。経験豊富なプログラマー、非常に優れたスピーカー、CppConのオーガナイザー、 C ++ Todayの著者：The Beast is Backです。 2日目は、標準化委員会のメンバーであり、 C ++ Templates：The Complete Guideの著者の1人であるDaveed Vandevoordeによるプレゼンテーションから始まります。



記憶に関する2つのレポートのセクションは、Fedor KorotkyとAleksey Salminによって開催されます。 両方のレポートは会議で発表され、議論に基づいて補足されました。 最初のレポートは、記憶がどのように配置されるかについての物語であり、私たちはそれについてまったく考えないことがよくあります。 jemallocを例として使用した、最も人気のあるアロケーターの1つのアーキテクチャについて。 2番目のレポートは、下位レベル、つまりLinuxのmmapとmadviseの機能に当てられています。



昨年、Courserは優れたC ++コースの公開を開始しました。 コースの著者の一人であるイリヤ・シシコフは、彼の経験を共有します。 このレポートは、教師だけでなく、経験豊富なプログラマー向けでもあります。通常、経験とスキルを共有することは、プログラムを書くよりもさらに困難です。



ほとんどの場合、3人のスピーカーを招待するように頼まれます：Bj​​arne、Herb、Andrei。 私は定期的に彼らに手紙を書きますが、彼らは丁寧に拒否します。 そして今年、ハーブは私に同情し（または私の手紙にうんざりして）、リモートでレポートを作成することを申し出ました。 この形式のレポートは好きではありませんが、その場合、私はチャンスをとることに決めました。第一に、彼はクールなスピーカーであり、第二に、試してみるのは面白いです。 そのため、今年はSkypeを介してハーブサッターを使用します。



マスタークラス



4つのマスタークラスが会議で発表されます。 2人はC ++ Russia 2017に参加し、さらに2人が参加しました。 マスタークラスに関する最も一般的な質問：なぜそんなに高価なのですか？ 昨年、私たちは初めてマスタークラスを行い、参加者にとってどれほど興味深いかを確認することにしました。 著者の手数料のみが価格に含まれていたため、価格は非常に低かった。 私たちは自分のポケットから家賃と食事を払いました。 マスタークラスは満員の家を集め、今年はあなたがまだ彼らを不採算にすることはできないと決めました。 現在、チケットの費用は著者への家賃、食事、ロイヤリティをカバーしています。



学生プログラム



いつものように、私たちは知識とコミュニケーションの探求において学生をサポートできることを喜んでいます。 割引を受けるには、学生/成績表/大学院生の写真またはスキャンをstudents@cpp-russia.ruに送信する必要があります



オンラインチケット



オンラインチケットを購入すると、個別のリンクが届き、レポートをリアルタイムで見ることができます。 会議後、レポートのビデオ録画がこのリンクで2か月間利用可能になります。



あなたのコメント、提案、苦情は私たちにとって非常に重要です。 conf@cpp-russia.ruまでメールでお問い合わせください。 また、電報で一般的なチャットを行いました 。 彼が問題をより迅速に解決するのに役立つことを願っています。



会議で会いましょう！
