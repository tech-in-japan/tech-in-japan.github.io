DEMO 08の新しいWebテクノロジー

DEMO展示会は、有望なスタートアップが彼らの開発を示す主要な場所になっています。 ここでの聴衆が新しいガジェット（一度Palm PilotとTiVoがDEMOで発表された）に慣れる前に、DEMOは主にインターネットスタートアップの展示会になりました。



2008年1月28〜30日に行われた最後のショーでは、77社が参加しました。 初めて耳にした10の最も興味深いものについて説明します。 これらのWebアプリケーションの一部は、それぞれの分野で真の革命を起こすことができ、それらを効果的に使用することができます。 他の人は、RuNetでクローンを作成する価値があります（このような理由で、最もcな起業家がそのようなレビューを慎重に研究するのは秘密ではありません）。



Liquid Plannerは「プロジェクト管理の革命」を主張していますが、実際には、ソーシャルネットワークの追加機能を備えたBasecampの一種です。



UGCサイトの新世代の多くの代表者の1人は、誰でも特定の都市についての印象や写真を共有できる世界Citiport.netのすべてのコーナーへのガイドです。



Opera Miniの潜在的な競争相手である新しいSkyFireモバイルブラウザに興味があるかもしれませんが、これまでのところWindows Mobileでのみ動作します。

Ribbitは、音声および関連サービス用の非常に興味深い両生類 Web アプリを導入しました。 このプログラムでは、インターネットから携帯電話に電話をかけることができるだけでなく（たとえば、電話ニュースレターをFacebookプロフィールに直接掛けることもできます）、ボイスメールをオンラインで管理し、発信者の番号（彼が今していること）とともに彼のプレゼンスステータスを表示します。 最もキラーな機能は、着信音声メッセージをテキストビューに変換することです。 このようなプログラムは、iPhoneやその他のコミュニケーターの機能拡張のように非常に有機的に見えます。



Voyant Webアプリケーションは 、個人の財務計画のために設計されています。 そのようなプランナーはあまり一般的ではありません。おそらく、それぞれの収入を予測するのが難しく、経費を計画するのは意味がないからでしょう。 しかし、西部では、毎週数十年にわたって人々がセントと同じ正確な給与を受け取っているため、そのようなソフトウェアは非常に重要です。



オリジナルのReview2Buyは 、非常に特定のワイングルメを対象としています。 専門の参照サービスが携帯電話に、ワインのブランドとその価格に関する完全な情報をダンプします。 ちなみに、同社はワインだけでなく、本、映画、音楽、ポータブル電子機器についても同様のサービスを推進しています。 携帯電話のカメラでバーコードによる商品の自動認識を設定すると（このような技術は既に存在します）、サービスはさらに便利になります。

CellSpinの携帯電話プログラムにより、Flickr、Facebook、YouTube、eBayなどのオンラインUGCサービスにコンテンツを簡単にアップロードできます。



FlyPaperユーティリティにはまだオンラインバージョンがないため、Winでダウンロードしてインストールする必要があります。 開発者によると、このプログラムは、インタラクティブなFlashプレゼンテーションの作成プロセスを不可能に単純化します。 多くの無料テンプレートのおかげで、非常に美しくなりました。 ちなみに、同じスプラウトビルダープランの別のプログラムが展示会で展示されました。 しかし、これはすでにウィジェットを作成するためのはるかに強力なツールであり、アプレットのようにオンラインで作業することもできます。



素晴らしいアイデアとパフォーマンスを備えた別の興味深いサイトはiVideoSongsです。 これは、野心的なミュージシャン向けのインタラクティブなチュートリアルです。 ビデオでは、音楽の有名人（エリッククラプトン、サンタナなど）がどのように歌を演奏するかを示しています。 ほとんどのビデオは通常のアコースティックギターの指の動きを示していますが、エレキギター、バンジョー、ピアノ、パーカッションのビデオもあります。 これらの楽器の演奏に関するサイトには、標準のチュートリアル（ビデオ）があります。
