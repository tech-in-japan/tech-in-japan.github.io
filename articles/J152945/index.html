Patterns＆Practices Summit RussiaのWindows Azure





開発者の皆さん！ 次回のPatterns＆Practices Summit Russiaカンファレンスで、ビジネスアプリケーション構築の最新のプラクティスが検討されることを嬉しく思います。 会議はモスクワで10月5日に開催されます。



完全な会議プログラムは公式ウェブサイトで見つけることができます 。 ここでは、Windows Azureプラットフォームでのクラウドベースのアプリケーションの開発者向けのコースについて詳しく説明したいと思います。



会議の講演者は、Patterns＆Practicesおよびロシア支社のマイクロソフトの専門家でした。



  • Euchinho Pache。 Microsoft Azureベースのクラウドでハイブリッドアプリケーションを作成する
  • クリストファー・ベナズ、Euchinho Pache。 コマンドクエリの責任分離（CQRS）に飛び込む
  • ウラジミール・グサロフ。 クラウドのALM-Microsoft Windows Azureインフラストラクチャを使用してアプリケーションライフサイクルを管理する
  • ナタリア・エフィムツェバ。 Windows Azureが新しいWorkflow Foundation機能を提供
  • ウラジミール・ユネフ。 Windows Azure Service Busでエンタープライズ統合の課題を解決する
  • Euchinho Pache。 オンデマンドでWindows Azureアプリケーションを自動的にスケーリングする


カットの下には、スピーカーに関する簡単な情報といくつかの発表があります。



クリストファー・ベーナジ



クリストファーはソフトウェアの開発が好きで、コンピューターテクノロジーとWebに精通しています。 クリストファーは、アジャイルとオープンソースソフトウェアを高く評価しています。 これに加えて、人気のあるJavaScript言語に精通しています。MicrosoftPattern＆Practicesチームでは、ChristopherはHTML5、JavaScript、およびWebプラットフォームテクノロジーを担当しています。 彼の仕事は、ベストプラクティスを調査、収集、サポートすることです。



エウチーニョ・パチェ



Microsoft P＆Pチームのプリンシパルプログラムマネージャーリード。 彼は、Azure WindowsおよびWindows Phone 7プラットフォームでの移行とアプリケーションの構築に関するガイドの開発を担当し、それ以前は、クレームベースのIDおよびフェデレーションIDのアーキテクチャに関する推奨事項に取り組みました。



ウラジミール・グサロフ



Quest Softwareのモスクワ支店のR＆Dディレクター。 20年以上にわたって産業用ソフトウェア開発に従事。 現在、ウラジミールはクエストソフトウェアのモスクワ事務所を率いています。 2010年以来、VladimirはVisual Studioアプリケーションライフサイクル管理（ALM）ノミネートでMicrosoft Most Valuable Professional（MVP）であり、2011年からALM Rangersチームに加わりました。



ナタリア・エフィムツェバ



マイクロソフトのパートナーサポートテクニカルアドバイザー。 Natalia Efimtsevaの主な活動は、クラウドテクノロジーとMicrosoft製品（Windows Azure、Office 365）に基づくソリューションの構築に関心のある企業と協力しています。 技術専門家としてのナタリアは、ローカルプロジェクトだけでなく、ポータルプロジェクト、MS Officeのビジネスアプリケーション、Windows Server AppFabricに基づくシステムの配布、およびその他の多くの興味深いプロジェクトにも参加しました。



会議中、Windows Azureトライアルアクティベーションセンターが機能し、誰でもプラットフォームを学習するためにWindows Azureへの無料アクセスをアクティベートできます。



参加者にとって朗報です！ すべての会議参加者は、無料の限定版の本を受け取ります。
