新しい画像検索インターフェイス：36.5％の効率化





Search Mail.Ruは、新しい画像検索インターフェイスを開始しました。 現在は、 http ： //images.mail.ruという短いアドレスで入手できます。 変更のきっかけは、画像を検索する際のユーザーの好みと目標を特定することを目的とした調査結果でした。

私たちは多くの有用なものを見つけました-例えば、彼らはほとんどの場合、写真を探していることに気付きました...

  • すぐに行く場所を見る
  • サンプル写真を見つけ、写真の撮り方を学ぶ
  • ギフトとしてカードを選ぶ
  • Tシャツから自動車まで、購入する商品を決定します。
  • プレゼンテーション、記事、壁の新聞を作成する
  • 元気づけて
これはすべて、新しい画像検索インターフェイスの設計にどのように影響しましたか？



適切な画像を検索しながら、ユーザーは3つの段階を経ることがわかりました。



ここにあるものを見てみましょう...
まず、ユーザーは検索結果が期待どおりかどうかを評価します。 したがって、原則として、結果の1ページに限定されず、次のページに進みます。



判明したように、分析の結果によると、ユーザーはページごとのナビゲーションに多くの時間を費やしています。 ページ（したがって、検索自体）の遷移を高速化するために、20を超える追加の画像をクリックしたときに、ページごとの遷移を大きなボタン "Show more ..."に置き換えました。 前の画像は消えないことに注意してください-一番上までスクロールすることで、すばやく元の画像に戻ることができます。





なぜ「無限のスクロール」を行わなかったのでしょうか？この場合、人々は必要な結果に気付かずに検索時間を長くする（決して短縮しない）ことが多く、すぐに下にスクロールすることに気付きました。



より速く、より高く...大きく！
第二に、写真が適切であると思われる場合、ユーザーはそれをより大きくしたい。 そのため、大きな画像のサイズがブラウザウィンドウのサイズに自動的に「調整」されるようにしました。





自分を安全にする...
第三に、とりわけ、検索では、ユーザーが他の画像オプションを見る必要があることがわかりましたが、選択した画像は「見えている」ままです。



この必要性に気付かずにはいませんでした-マウスホイールをスクロールして検索をすぐに続行でき、選択した画像が検索結果の左側に配置されます。 もちろん、必要に応じて、「写真を隠す」リンクをクリックするだけで、写真のリストにすばやく戻ることができます。



成長しています
4番目に、1つのWebページに80％以上の画像が追加されました。





新しいインターフェイスは、Ajaxテクノロジーを使用して実装されます。これにより、ページをリロードすることなくナビゲーションが可能になり、より高速で経済的になります。 現在画面に表示されている画像のみがダウンロードされます。これにより、トラフィックを節約し、ブラウザが画像の新しい部分を表示するのにかかる時間を最小限に抑えます。 各ユーザーが理解できる簡単な式がわかります。トラフィックの減少->ダウンロードの時間の短縮->お金の削減。



さらに、Ajaxの伝統的な欠点を補おうとしました。 たとえば、通常のブラウザボタン「戻る」と「進む」を使用して、お気に入りのブックマークに好きな画像を保存できます。



プラス36.5％-みんなに！
動作中の新しいインターフェイスをテストするために、Poisk Mail.Ruは比較テストを実施しました。インターフェイスが変更される前後です。 調査中、ユーザーは一般的なタスクを実行し、実装に費やした時間を測定しました。 新しい画像検索インターフェイスにより、ユーザーはタスクを36.5％速く完了することがわかりました。

コミュニティでの結果とテスト方法の詳細をご覧ください 。
