Linux用のGUI SVNクライアント

良い滞在、親愛なるKhabrovtsy 8）

私は最近、非常に有用で待望の仕掛けの広大な広がりに出会いました。 なんということ-件名を読んでください。



それはすべて、有名なTortoiseSVN for Windowsのように見えます。 RabbitVCSと呼ばれ、すでにLinux用です。

あなたが私を信じないなら、スクリーンショットを見てください:)

















スクリーンショットを撮ったのではなく、正直にスクリーンショットを撮ったことを許してください。 サイト。 ところで、これはrabbitvcs.orgにあります



この奇跡はpythonを使用して書かれています（当然ですが）。 それはかなり速く動作します...



現在、Ubuntu 9.10 Karmicを使用しています。 したがって、念のため、この奇跡が上記のシステムにどのように配置されるかを説明します。



1. /etc/apt/sources.listを開きます

sudo nano /etc/apt/sources.list







2.下部に2行追加します

deb ppa.launchpad.net/rabbitvcs/ppa/ubuntu karmic main

deb-src ppa.launchpad.net/rabbitvcs/ppa/ubuntu karmic main







3.システムに必要なキーを追加して、ppa.launchpad.netからパッケージを取得してインストールできるようにします

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 34EF4A35





使用されている数字が正しいこと、そしてそれらを使用する必要があることに注意してください。



4.パッケージリストを更新する

sudo apt-get update







5.パッケージ自体をインストールする

sudo apt-get install rabbitvcs







パッケージと依存関係は変動します。 置いて インストール後、彼らはnautilusの再起動を求めます-あなたは同意します。

すべての終わりに、ノーチラスとローを起動して見よ！ 右クリックして-待望のメニューアイテムが登場しました:)



インストールを頑張ってください。

もう一度、思い出させてください。 ウェブサイト： rabbitvcs.org

リンクを直接ダウンロード： wiki.rabbitvcs.org/wiki/download

すべてのubuntsのインストールマニュアルへのリンク：launchpad.net/~rabbitvcs/+archive/ppa



そしてもちろん...喜びで使用してください！ ;）
