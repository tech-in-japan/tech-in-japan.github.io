WebProfessionals Conferenceビデオレポート公開





WebProfessionalsは、Microsoftの最大の地域Web開発会議です。 会議は4月3日にサマラで開催され、地域全体から300人以上の参加者と約6,000人のオンライン視聴者が集まりました。 会議中に、Microsoft Webテクノロジースタックの最新のイノベーションについて議論されました。IISに基づく独自アーキテクチャに基づくソリューションから、 Windows Azureに基づくクラウドテクノロジーまで。



会議の重要な部分は、ロシアの大企業から実際にマイクロソフトの技術が実際に適用されたという報告でした。 会議中に、 Visual Studio 11およびその他の開発ツールが提示され、パートナーからのソリューションにより、効果的なWebプロジェクトを作成できます。



私たちは、ビデオ会議レポートがすでに無料のオンライン視聴のために利用可能であることを発表することを嬉しく思います。
