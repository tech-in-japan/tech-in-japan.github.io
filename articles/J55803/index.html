PowerShellはWindows Updateを通じて利用可能になりました

Microsoft Windows PowerShellは、CRL .NET Frameworkおよび.NET Frameworkに基づく拡張可能なシェルであり、コマンドラインインターフェイスと付属のスクリプト言語を備えています。 最初のバージョンは2006年にリリースされ、Windows 7はシェルの2番目のバージョンを使用しています。 PowerShellは、.Net 2.0（Windows XP、Vista、Server 2003）がインストールされているすべてのシステムで動作します。 テキストを受け取って返すほとんどのシェルとは異なり、Windows PowerShellは.NETオブジェクトを受け取って返すだけでなく、作業中にオブジェクトのみを使用します。 この根本的な変更により、Windowsシステムの管理と構成にまったく新しいツールと方法を使用できるようになりました。



他の多くのシェルと同様に、Windows PowerShellはコンピューター上のファイルシステムへのアクセスを提供します。 Windows PowerShellには、レジストリやデジタル署名証明書など、他のデータストアでも簡単に作業できるベンダーが含まれています。
