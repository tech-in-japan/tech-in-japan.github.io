Raspberry Piで実行できる7つのオペレーティングシステム

Raspberry Piで実行できる7つのオペレーティングシステムに関するJames Bruceの記事の無料翻訳に注目します。



40ドルのコンピューター価格で議論できるのは誰ですか？ 特に、エレクトロニクス分野のプロジェクトの優れた拠点である場合。 競合他社は表示されません。 しかし、鉄自体はコインの片側にすぎません。オペレーティングシステムが必要です。 以下では、Raspberry Piに今すぐインストールできる7種類のオペレーティングシステムについて説明します。 簡単なメモ：すべてのオペレーティングシステムが役立つわけではありません 。









Linux



あらゆる好みに合わせて利用できる3つの公式Linuxディストリビューションがあります： Pidora （Fedoraベース）; Archlinux （このディストリビューションのインストールはほとんど「手動」です）; およびRaspbian （Debianベース）。

Raspbianは、Raspberry Piを使い始めたばかりの人にはお勧めです。 他のオペレーティングシステムを試してみたい場合は、Raspberry Piの作成者によるNOOBSツールがあります。これにより、インストールの対象となるディストリビューション（ブート時）を選択できます（ディストリビューションについては後述します）。







OpenElecおよびXBMC



OpenElecは、XBox Media Centerを起動するために特別に設計された、必要最低限​​の最適化されたオペレーティングシステムです。 Raspbmcとは異なり、OpenElecには、その主な機能を実行するために必要な最小限のもの以外は含まれていません。 したがって、たとえば、他のパッケージをインストールすることは十分に困難であり、その結果、さまざまなアドオンのセットは非常に不足しています。 しかし、これはすべて高性能を達成するために行われます。

現時点では、このディストリビューションの機能は、リビングルームにメディアセンターを作成するのに十分であり、PCをWindows 8に完全に置き換えました。







SDカードを簡単に交換できることを考えると、Raspberry Piは、OpenElecをインストールしてXBMCを操作するなど、さまざまな目的に使用できます。 IceFilmsなどのストリーミングブロードキャストを表示するための優れたプラグインセットを含む既製のイメージをダウンロードし、人気の番組を視聴する機能（事前に読み込まれたメタデータのおかげ）。 SDからの最初の起動後にUSBドライブから起動することにより、システムを高速化することもできます。

このガイドで 、独自のRaspberry Pi Media Centerを作成するために必要なハードウェアに慣れることができます。



レトロパイ



RetroPieは、さまざまなコンソールでプレイできるユニバーサルエミュレーターですが、Playstationのゲームはあまりうまく機能しません（Raspberry Piの部分では少し生意気ですか？）。 このオペレーティングシステムはRaspbianに基づいていますが、ダウンロードできる個別のアセンブルされたRetroPieイメージがあります。







クリスチャンは RetroPieの完全なインストールおよび起動ガイドを作成しましたので、ゲームコンソール愛好家のための楽園への道を進んでください 。



RISC OS



このプロセッサを最初に作成したチームであるARM向けに特別に組み立てられた、優れたレトロスタイルの1080pグラフィックユーザーインターフェイスがあります。 90年代前半、英国の学校では、RISC OSが一般的に発生していたことは興味深いように思われます。 Raspberry Piの優れたRISC OS入門ガイドと、このオペレーティングシステムに推奨されるソフトウェアのリストを以下に示します。







このページでは、完全なインストール手順を見つけて、SDカードのイメージをダウンロードするか、RISC OSをインストールする機能を含む上記のNOOBSツールを使用できます。







最初は、いくつかの基本的なアプリケーションのセットが利用可能になりますが、少額の料金で、オフィスプログラムの完全なセットにアップグレードできます。



Firefox OS



ここでは、新しい電話のデザインで見たものではなく、FirefoxとPTXdistで構築されたLinuxの組み合わせを少し見つけます。 このオペレーティングシステムには十分な問題があり、 作業して改善することがたくさんあります。 実際、入力デバイスのサポートはありません。 この配布は、情報端末にのみ適しています。 まだ試してみたければ、 ここで既製の画像を見つけることができます。







計画9



Plan 9は、非常にシンプルなユーザーインターフェイスを備えたUNIXに似たオープンソースBell Labsオペレーティングシステムであり、UTF-8でエンコードされたファイル名をサポートしています（これは素晴らしいことです） UNIXを作成したのと同じ人々によって開発されたので、良いはずです。 こちらからSDカードの画像をダウンロードできます。 初心者向けのガイドもお忘れなく。







なぜPlan 9オペレーティングシステムが必要なのでしょうか？ 申し訳ありませんが、わかりません。
