スマートフォンはどのようにして支払いデバイスになりましたか？ Mastercard Tokenization Platformについて





Mastercardは、ロシアでカード所有者にApple PayとSamsung Payを使用する機会を提供した最初の会社です。 決済システムは、この市場で最初にトークン化プラットフォームを立ち上げました。MastercardDigital Enablement Service（MDES）は、インターネットに接続されたデバイスを安全な支払い手段に変え、世界の財布もこのサービスで動作します。 スマートフォンがプラスチックカードの代わりになった方法、および携帯電話で支払うときにセキュリティを提供する技術がどのように開発されたかについて-この投稿を読んでください。



デジタル決済開発の推進力としてのワンタッチ決済



非接触型Mastercard支払いテクノロジー（以前-Mastercard PayPass）の作成は、電子支払いの開発におけるターニングポイントと考えることができます。 これは2002年に起こりました。新しい技術により、支払いプロセスは通常の「プラスチック」のフォームファクターを超えました。キーフォブ、腕時計、携帯電話、ステッカーなどで支払いができるようになりました。



非接触技術は8年前にロシアに登場し、それ以来、さまざまな（非接触）フォームファクターで発行されるカードの数と、この支払い方法を受け入れるためのネットワークが着実に増加しています。



現在、欧州42か国を含む77か国の600万の小売店でワンタッチでお支払いいただけます。 3年以内に、ヨーロッパのすべてのPOS端末（ロシアを含む）は非接触型決済を受け入れます。 つまり、どこでも電話、時計、または宝石でお支払いいただけます。 いいえ、支払いとしてそれらを提供するのではなく、単に読者に持ってきてください！



さらに、ロシアは、非接触型カードをNFCスマートフォンにロードするためのMastercardで3つの主要なテクノロジーを利用できる最初で数少ない国の1つになりました。 非接触型マスターカードテクノロジーのおかげで、SIMカード（SIM中心のテクノロジーを使用）、携帯電話の内蔵チップ（埋め込みセキュアエレメント）、クラウド（ホストカードエミュレーションテクノロジーを使用）でモバイルカードを発行できるようになりました。



SIM中心のNFCに基づくモバイル非接触決済の分野でのロシア初のプロジェクトは2012年に開始され、MastercardとMTSによって開始されました。 ユーザーは、モバイルオペレーターの販売場所に行き、Mastercardが既に記録されている非接触型支払い機能をサポートする特別なカードに通常のSIMカードを交換する必要がありました。 そのため、NFC電話は支払い手段に変わりました。



その後、この分野の多くのプロジェクトが、最大の通信サービスプロバイダー、モバイルデバイスメーカー、銀行とともに開始されました。



2013年、ロシアで初めてMastercard、Tinkoff Bank、CardsMobile（i-Freeグループ）が、NFCチップが組み込まれた多くのHTC、Sony、Philipsスマートフォン向けの非接触型MastercardカードをWalletモバイルアプリケーションにリモートでダウンロードするためのフル機能サービスを開始しました。



2015年7月、Alfa BankとBPCのCardsMobile WalletアプリとAlfa Touchアプリの更新バージョンがGoogle Playからダウンロードできるようになりました。 HCEテクノロジーとMastercard Cloud Based Payments仕様は、サポートされるデバイスのベースを人気メーカーの数十のモデルに大幅に増やし、新しい開発者と銀行の機会を拡大しました。



デジタルソリューションを開発し、マスターカード所有者にまったく新しい便利な支払い方法を提供することにより、マスターカードは高レベルの取引セキュリティを提供します。 重要な段階は、トークン化サービスでした。



安全な支払いのためのトークン化







ロシア市場でのApple Payの発売は、トークン化の導入により可能になりました。 このテクノロジーは、モバイルNFCデバイスにカード製品をリリースする際に高いレベルの利便性を提供し、Apple PayやSamsung Payなどの電子財布を通じて支払いを行う際に最高レベルのセキュリティを提供します。



MDESトークン化プラットフォームは、2013年にMastercardによって最初に導入され、効率的で安全なデジタル支払いの分野でのさらなる開発の基盤となりました。

2014年に、このプラットフォームを使用してApple Payが米国で発売されました。 現在、MDESは4,000万枚以上のカードのセキュリティを保証しており、Mastercardは18か国で2億2,500万枚のカードを追加し、ユーザーがそこで、そして便利な方法で支払いできるように取り組んでいます。



トークン化とは何ですか？ これは、メインカードアカウント番号（PAN）を16桁の数字の組み合わせ（トークン）に置き換えたものです。 これは、Mastercardネットワーク内で銀行カード番号に結び付けられています。 さらに、トークンは接続されたデバイスごとに一意です。



支払いを行う際、Mastercardはトークンをカード番号に変換し、銀行が支払いを行うのが本人であることを確認します。 支払うと、トークンが売り手に送信されます。 アウトレットが支払いを承認するために銀行にデータを送信すると、Mastercardはトークンが有効かどうか、トークンが接続されているデバイスからのものかどうかを確認します。



トークンを使用すると、銀行カードの実際のデータを非表示にして、トランザクションセキュリティの最大レベルを確保できます。 購入者は、すべてのデータが保護されていることを確認できます。



追加のセキュリティは暗号化と生体認証によって提供されます。これにより、お金のセキュリティを確保する別のマイルストーンが作成されます。 最新のモバイルガジェットの生体認証保護と、スマートフォンの紛失時のブロックまたはデータ破壊の保証の可能性は、お金のセキュリティを確保するもう1つのマイルストーンです。









皆のためのMDES



Mastercardは、ユーザーだけでなく、銀行やテクノロジー企業などのパートナーにとっても、MDESをできるだけ簡単に使用できるようにしました。



顧客がスマートフォンでの購入の支払い機会を提供するために、銀行はApple Pay、Samsung Payまたはその他のウォレットに接続する必要はありません。 銀行がMDESプラットフォームに接続し、対応する機能を独自のモバイルアプリケーションに追加するだけで十分です。



現在、販売店（実店舗またはオンラインストア）の人は、銀行カードが添付されている既存のモバイルアプリケーションを使用して支払いを行うことができます。 一方、Mastercardは、銀行に、代替としてではなく、追加として、同じことを、ウォレットで、アプリケーションで行う機会を与えます。



したがって、デジタルウォレット、モバイルバンク、またはオンラインバンクは、銀行カードと同じ支払い手段になります。 お気に入りのガジェットと生体認証は、「プラスチック」の代わりにのみ使用できます。



実装のしやすさ



Mastercardのロシアのパートナーは、MDESの実装が実際の最速の1つであると述べました。 技術的な観点からは、銀行または電子マネーサービスはプラットフォームを認証するだけです。 本質的に、残りは組織の問題です。



銀行にとって承認と清算のプロセスで異なるのは、決済システムの側にあるいわゆる「MDESループ」の介入です。これは、取引中にトークンの支払いとトークンの比較を行います。 電話で支払うときのトークン化により、銀行のITキッチン内の概念は変わりません。 レポートやその他のニュアンスに関連する特定の設定がありますが、トランザクション処理の原則は変わりません。 シンプル、透明、効果的。



認可の簡素化



Mastercardは常に新しい支払い確認基準を開発しており、パスワードの入力を必要とせず、3Dセキュアプロトコルに代わるオンライン支払いの概念を開発しています。 オンラインショッピング中のバイヤーの識別をさらに簡素化し、非接触デバイスによる支払いを確認するために、Mastercardは常に手元にあるものを使用してこのプロセスを実行するサービスを開発しました。



新しいIdentity Check Mobile支払いアプリケーションは、オンライン購入に生体認証データ（指紋と顔認識）を使用します。 自撮りで支払うことが現実になりました！ ID Checkを使用したパイロットプロジェクトは、すでに12か国で開始されています。 2017年に、この決定は世界中に配信されます。 2017年に、この決定は世界中に配布される予定です。







以前、人々は自撮りの助けを借りて自分の身元を確認するというアイデアを笑いましたが、技術はうまく機能します。 最初に、適切なアプリケーションをダウンロードして、写真だけでなく小さなビデオを撮る必要があります。 点滅して、カメラの前に生きている人がいることを確認してください-カード所有者とデバイスの所有者。



Mastercardテクノロジーは、オンラインショッピングの開発を加速し、さらに便利にします。 革新のおかげで、インターネットで積極的に買い物をする人の数は絶えず増加しています。10年前には世界中に約1,000万人がいましたが、今年は少なくとも30億人がオンラインストアで買い物をしました。 それぞれの支払いのシンプルさと利便性は最前線にあります。
