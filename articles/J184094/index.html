ビーコンマウンテン-すべてのAndroidデベロッパーが必要



インテルは、Androidアプリケーションの設計、開発、デバッグ、最適化のための無料のソフトウェアスイート、 Beacon Mountainを導入しています。 このキットには、Intelソフトウェアとサードパーティユーティリティの両方が含まれています。 プログラムの一般的なリストは次のとおりです。

  • Intel Hardware Accelerated Execution Manager（Intel HAXM）
  • Intelグラフィックパフォーマンスアナライザー（Intel GPA）システムアナライザー
  • Intel Integrated Performance Primitives（Intel IPP）プレビュー
  • インテルスレッディングビルディングブロック（インテルTBB）
  • Intelソフトウェアマネージャー
  • Google Android SDK（ADTバンドル）
  • Android NDK
  • Eclipse統合開発環境
  • Androidデザイン
  • Cygwin（Microsoft Windows用）
このソフトウェアはすべて、1回の配布から数回クリックするだけでインストールできます。 Beacon Mountainには、コンポーネントを更新するためのツールが含まれています。新しいバージョンのリリースを追跡する必要はありません。

Beacon Mountainは、ARMプロセッサとIntel Atomの両方のアプリケーションを開発するように設計されています。 4.2以降のAndroidバージョンがサポートされています。 キット自体はWindows 7または8で実行されます。 後でApple OS Xのバージョンをリリースする予定です。
