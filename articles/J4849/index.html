Microsoft Researchによる最近の開発

InformationWeek Webサイトに興味深い記事が投稿されました。 Microsoft Researchが現在取り組んでいる製品の一部がリストされています。 これらのシステムのほとんどは、数年にわたって開発およびテストされています。 一部の製品はすでにリリースの準備ができており（おそらくVistaオペレーティングシステムに含まれていることもあります）、残りの部分については最終リリースの期限はありませんが、これらの技術自体は興味深いものです。



MS Researchの研究部門は、1991年に20人の従業員から設立され、現在では世界のさまざまな地域で働く700人にまで成長しています。 MS Researchの上級管理者の1人であるRich Dravesは、最も有望な開発について語っています。



GhostBusterの自動ルートキット検出システムは、MS ResearchのRedmond部門によって作成されました。 このプログラムは、最高レベルのシステムレベル（Win32 APIなど）から最低レベルのハードドライブの読み取りセクターまで、さまざまなレベルでシステム情報を分析および比較するという点でユニークです。



Shield Todayと呼ばれる別の興味深い開発は、ファイアウォールの知的「アドオン」のようなもので、システムに脆弱性が見つかった直後、ただしこの穴を塞ぐパッチがリリースされる前でも、潜在的に有害なトラフィックをブロックできます。 テストは、システムが非常に効率的に機能することを示しました。これは、SQL Slammerワームのシステムの保護や、Windowsメタファイルの既知のホールの悪用からの保護を含め、過去2年間にWindowsシステムで見つかった脆弱性の98％の悪用を効果的にブロックしました。



インドのMicrosoft開発者によって作成されているSureMail（ PDF ）システムは、「失われた」電子メールの問題を解決するはずです。 この調査では、インターネット上のすべての電子メールの約1％が宛先に到達しないことが示されました。 SureMailシステムは、すべてのメールトラフィックを監視し、受信者に送信された手紙を受信して​​いない場合、その内容を開示せずに受信者に通知します。



興味深いVigilanteプログラムは、MS Researchのケンブリッジ部門で作成されています。 このプログラムは、未知のタイプのネットワークワームの攻撃を検出して応答します。 それらを検出するために、ハニーポットがネットワーク上に配置されます。最初にワームに攻撃される必要があります。 これらのサーバーは着信トラフィックを分析しており、新しい「感染」が検出された場合、プログラムはこれについてセキュリティ専門家に通知します。



フィッシングと戦うためにマイクロソフトが開発したもう1つの非常に独創的なテクノロジー。 これは、すべてのサイトでユーザーが入力したすべてのパスワードを追跡するブラウザーアドオンです。 それらに関する情報（パスワード自体ではありません）は、数百万人のユーザーからのデータを分析する中央ノードに送信されます。 ユーザーがこれまで未知のサイトに異常に多くのパスワードを残していることをシステムが突然検出すると、システムは警告を発します。このサイトはフィッシング詐欺師によって作成される可能性があります。
