サンが確認-AppleはMac OS X LeopardにZFSを追加する

1年以上前、Mac OS X Leopardは次世代のZFSファイルシステムを使用するという噂がありました 。



今、この噂は公式に確認されたと呼ぶことができます。 MacRumors.comによると、Sun Microsystemsの社長Jonathan Schwartz は昨日 、HFS +の代わりに新しいLeopardファイルシステムがZFSになると述べた。



ZFS（Zettabyte File System）ファイルシステムは、もともとSolarisオペレーティングシステム用にSunの壁の中で開発されました。 ZFSはすべてのファイルを64ビットチェックサムで保護し、データ整合性違反を検出して修正します。 また、128ビットファイルシステムのように、ZFSは既存のバージョンのWindows、OS X、Linuxよりも大量のボリュームを処理できます。



ZFSが提供する最大の変更点の1つは、zpoolと呼ばれる仮想ストレージプールモデルです。 プールは仮想デバイス（vdev）で構築され、各仮想デバイスは物理デバイス、または1つ以上のデバイスのミラー（RAID 1）、または（RAID Z）-2つ以上のデバイスのグループです。 これにより、すべての仮想デバイスの容量がzpool内のすべてのファイルシステムで利用可能になります。



ZFSでは、RAIDを使用してデータを自動的に予約し、圧縮の可能性を提供することにより、ファイルシステムの状態の「スナップショット」（指紋）をいつでも取得できます。
