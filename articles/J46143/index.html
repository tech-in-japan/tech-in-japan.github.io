Python 3.0リリース

Python 3.0（「Python 3000」または「Py3k」とも呼ばれる）がリリースされました-これは、2.xバージョンの行と互換性のないプログラミング言語の新しいバージョンです。 言語は基本的に同じままですが、辞書や文字列などの多くの要素が大幅に変更されました。 廃止された多くの言語要素が削除されました。 標準ライブラリの一部が再編成されました。



Python 3.0に関するいくつかのリソース：

* Python 3.0の新機能

* Python 3.0変更ログ。

* オンラインドキュメント

* Python 2.xコードを変換するツール：2to3
