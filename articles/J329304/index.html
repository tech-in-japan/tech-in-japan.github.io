Google I / O 2017：同僚の印象とコメント

投稿者：Elena Fedorova、人事マーケティング部長、DataArt、Voronezh







2009年以来、私はVoronezhのGoogle Developer Groupコミュニティを率いており、さらに2016年にはIT Women Techmakers Voronezhで女の子向けのコミュニティを組織しました。 ですから、もちろん、私は次のGoogle I / O会議を見逃すことはできませんでした。 私にとって、Google I / Oは最もクールな新製品の発表であり、今日の近い将来を見る機会であるだけでなく、一般的な動き、コミュニティの同僚や友人とのコミュニケーションの感覚でもあります。



Googleコミュニティに関するいくつかの統計：



•102か国で約600のアクティブなGDGグループ。

•過去6か月間の3610のテーマ別イベント。



昨年の会議からの私のレポートはここで読むことができます 。 今回は同僚のコメントを収集し、自分の印象を補って、Google I / O 2017の5つの興味深い機能を強調しようとしました。



1.トップ5のGDGロシア



今年、ロシアはGDGコミュニティで最も活発な上位5か国に初めて参加し、過去6か月間にロシアのコミュニティは111のイベントを開催しました。 そしてそれは本当にクールです。



数年前、Natalia EfimtsevaがGoogleに来ました。 短縮形では、彼女の立場は「デヴレル」と呼ばれますが、実際には、ナタリアはロシアのGDGコミュニティの発展に関心があるだけです。 彼女のおかげで、ロシアで活動しているグループの数は3年間で3から21に増えました！



今年、モスクワ、ヴォロネジ、ニジニノヴゴロド、ペルミ、ウラジオストク、ノボシビルスク、カザンのGDG代表がGoogle I / Oに参加しました。







GDGモスクワのリーダー、GDGロシアのキュレーター、カティア・ピサンカ：



「今年の会議で、主催者は活動家とさまざまなコミュニティのリーダーとの交流とコミュニケーションのための特別な場所、コミュニティラウンジを割り当てました。



そこで会議を開催し、ロシア語を話す会議参加者を集めました。 ロシアのGDGリーダーは、GoogleがITを促進するために提供するコミュニティ、行動、機会について話しました。 会議の前夜、さまざまな国のコミュニティリーダーがサミットに集まりました。 伝統的にはパーティーで始まり、その後、コミュニティを促進するためのオラトリオから新しい機会まで、幅広いトピックに関するセッションの終日が続きます。 主催者はレポートにゲームインタラクションを追加し、いつものように、参加者に記念品を贈呈しました。



2.忙しいスケジュール



スタジアムで収集された優れたレポート、多くのアクティビティ、セクションを歩き回るセクション。 そして最も重要なことは、一般のユーザーがまだ知らないことについては喜んで話しますが、近い将来には何が一般的になるかについて話し合うことです。







Android GDEシンガポール、90Seconds.tvの開発者、Denis Neklyudov氏：



「I / Oについて慢に言うことができます。驚いたことはありません。多くの人が集まったと言いますが、最終的には世界を変えるものは何も見せませんでした。 しかし、開発者向けの会議のハイライトはポイントではありません。 過去10年間で誰かが私たちをだめにしたので、私たちは有名な発表や高価なギフトガジェットに来るのに慣れています。







実際、開発者の会議として、このI / Oは素晴らしいことがわかりました。 多数の興味深いレポートで満たされた忙しいスケジュールでは、3日間すべて無料で1分間も休みませんでした。 そしてなんとコミュニティ！」



アレクサンダーデニソフ、リードソフトウェアエンジニア、NetCracker、リードGDGニジニノヴゴロド：



「I / O-2017について、自信を持って言うことができます。Googleは間違いから学んでいます。 非常にバランスの取れたスケジュール、バッジ内のNFCタグ、およびオンライン登録の可能性により、パビリオン内の行列を避けることができました。 昨年、ある興味深いセッションのために、並んでいる間に別のセッションをスキップしなければなりませんでした。







コミュニティとの協力も喜ばれました。7,000人以上の参加者のうち、79か国から約700人の招待されたコミュニティオーガナイザーがいました。 革命的なニュースは聞いていませんでしたが、多くの興味深いことが発表され、報道されました。 もちろん、Firebase、AI、Clouds、Kotlinのサポートに関する新しいものがたくさんあります。 I / Oは、そもそも、ニュースやセッションには適していませんが、興味深い人々とチャットする機会に適しています。 まず第一に、Googleの開発者は、いわば直接回答を受け取りました。 グーグル、いつものように、とてもクールでした！」



3.新しいデバイス。



障害のある人向けのデバイスと、Googleがこの方向性をどのように開発しているかに再び感銘を受けました。 ヴォロネジで医療ハッカソンを開催し、このトピックがいかに関連性があるかを理解しています。







ひっくり返らないスプーンのような一見シンプルなものでさえ、動きが限られている人にとっては本当に人生を楽にします。







目の不自由な人のためのデバイスは、インターネットとモバイルテクノロジーを介したコミュニケーションを支援します。











Googleは、予想外の最新のテクノロジーを適用しようとしています。 たとえば、機械学習の助けを借りて、人の個々の特性を考慮に入れて、カクテル用の飲み物を選ぶことができます。



4. Kotlinサポートと機械学習の進捗



Alexey Korovyansky、Android GDE、テクニカルディレクター、ハンサム（handsome.is）：



「Android開発者にとって、会議で最もホットで話題のニュースは、Kotlinの公式サポートでした。 ここで世界中のAndroidの専門家と話すと、Kotlinはコミュニティで本当に愛され、認められていると自信を持って言えます。 また、GoogleはAndroid SDKの開発に対して優れたアプローチを示しており、コミュニティのフィードバックに耳を傾け、それに基づいて重要な決定を下しています。 Kotlin / JetBrainsチームとKotlinを信じていたすべての人に心から祝福します。 コミュニティでこの言語を普及させ続け、知識に対する需要が顕著に増加することを期待しています。







また、今年のI / Oには、Firebaseとクラウド機能、パフォーマンスモニタリング、SMS認証、ファブリック/ Crashlytics統合、クラウド統合の最新のイノベーションに関する多くの素晴らしいコンテンツがあります。これにより、典型的なモバイル開発タスクをより速くより速く解決し、火災アプリケーションを作成できます。



もちろん、今年は機械学習に関する興味深いこともたくさんあります。たとえば、Magentaプロジェクトを使用すると、数時間でPrismaの単純化された類似物を作成できます。 会議全体のライトモチーフは「モバイルファーストからAIファースト」の承認であり、これはAndroidの未来とテクノロジーの世界全体に根本的に影響を与えます。



Alexander Zatsepin、Androidデベロッパー、MAPSME、Mail.ru：



「一般的な印象は、特に機械学習（TensorFlow）の分野では、すべての分野で進歩が明らかであり、Googleが収集するデータに基づいて誰もがモデルを構築およびトレーニングできるようになったということです。 そして、彼はそれらの多くをどこでも集めます。 Androidに関しては、Android開発者の主要なツールであるAndroid Studioが大きな改善です。 アプリケーションのプロファイリングはもう少し楽しいはずです。 また、Androidアーキテクチャコンポーネントは、アプリケーションアーキテクチャを標準化するためのもう1つの前進だと思います。これは、誰もがMVP / MVCバイクを書いているからです。







Kotlinの公式サポートは、このGoogle I / Oの素晴らしいボーナスでもありますが、嬉しいというよりも注意を促しました。」



5.友人との出会い



長い間会っていないさまざまなコミュニティのリーダーと会うことにいつも驚きます。または、原則として、私はオンラインでしか知りません。 たとえば、海の上を飛んで、リヴィウ・ヴィタリー・ザサドニーの最もクールなGDGリーダーとチャットしましょう。







参考のために-リヴィウの人たちは、世界最大のDevFest会議の1つを開催しています！



コミュニティのリーダーは家族を作ることができただけでなく、すでに子どもたちを会議に連れてきています。







Aigul ZagidullinaはUfaのGDGのリーダーでしたが、現在はロンドンにあるベルリンでGDGを開発しました。



Android GDEシンガポール、90Seconds.tvの開発者、Denis Neklyudov氏：



「世界中から非常に多くの友人がいます、そして、すべてが話し合う何かがあるクールな専門家です。 そして、すべては非常にきちんと組織されていましたが、心からも供給されていました。 来年、人々が発表のために来るのではなく、新しい知識、知人、経験、そして天気の良い日に谷にいるという喜びのために来ることを願っています。





Denis Neklyudovは彼自身のビデオレポートさえ私たちと共有しました。



Alexey Korovyansky、Android GDE、テクニカルディレクター、ハンサム（handsome.is）：



「技術革新に精通することに加えて、I / Oはコミュニケーションと新しい興味深い知り合いのための素晴らしい場所です。 「Googleはコミュニティの作り方を知っているので、前向きな感情の海と多くのメリットを得ることができます。」
