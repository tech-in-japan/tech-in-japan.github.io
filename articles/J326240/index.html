100台の車のオフィス、またはWindowsからCentos 7にサーバーを転送した方法の物語。

それは5年前、私と呼ばれる良き友人（イリーナになろう）で、私たちの市の中央会計部門でシステム管理者として働くように頼まれました。 私はこの従業員の職務のリストを要求しましたが、それに答えて私を困惑させました：

システム管理者は毎朝職場に来て、サーバーを再起動し、女の子をバイパスして、プログラムに関する問題を解決する必要があります。 また、毎月行われた作業に関するレポートを作成する必要があります。


時間を無駄にしたくなかったので、すぐに拒否することにし、電話を切りました。 この夜は、私の決定の正しさと、元従業員が毎朝サーバーをジャンプして再起動した理由を考えて通過しました。







翌朝は激しく始まった。 私はちょうどシャワーを降りて、ドアホンが鳴ったときに飲もうとしていました。 イリーナはケーキを持って来て、出入り口からすぐに私がこの仕事が必要だと確信し始め、彼女はシステム管理者として私を必要としていました。 この職場での私の責任についての激しい議論の後、私はまだこの企業を経営することに同意しました。



それから私は仕事に同意した条件をそのままにします
私の責任：

  • 毎朝、職場に来てサーバーを再起動します
  • 女の子と一緒に回り、プログラムで問題を解決する
  • 完了した作業に関する月次レポート。
  • サーバーとローカルネットワークのパフォーマンスを監視し、タイムリーにトラブルシューティングを行います。
  • 入手可能になったプログラムやハードウェアに関連する女の子の問題を解決します。
  • 昼夜を問わずいつでも連絡を取ることができます。
  • 上級管理職によって課せられたイノベーションを実施します。


私の権利：

  • 5月1日から5月31日までの報告期間の後、私は職場に現れず、電話を取りません。
  • すべてが機能する場合、私は職場に現れません。
  • 女の子は私に直接問題を連絡することはありませんが、私のテンプレートに従って絵を描いて郵便局に送ってください。


PS：公務員、月給8,000ルーブル、無料のランチとお茶。



初営業日



私は最初の就業日に遅刻し、9時までに来ることを事前に通知することにしました。 夏で、天気は最高でした。 家を出て静かに中央銀行に向かった。



彼女がこの施設の所長だったので、職場に到着すると、私は最初にイリーナに行きました。 まず第一に、彼女は私の職場に連れて行ってくれ、仕事の前線全体を見せてくれました。 私の職場を示すときの唯一のフレーズは：

角にはサーバーがあります。 ここでも、毎朝再起動する必要があります。


クローゼットの周りを見て、私は全体の本質を実感しました。 部屋の大きさは約8 m 2でした。 もはやサーバールームではなく、倉庫でした。 すべてが箱、古い鉄、お正月のおもちゃ、見掛け倒しで散らばっていました。 オフィスの椅子の下から箱が散らばった角のどこかに、サーバーが見えました。 彼はこの部屋で長年の作業で蓄積されたダストマットレスを吹き飛ばそうとして、かろうじてブレードを吹き飛ばしました。



まず、部屋の瓦をすくい取り、サーバーがどのようなものであるかを調査し始めました。

何とどこですか？



クリーニング後の最初の日、サーバーがどのような問題で、なぜまだ生きているのかを知るのに十分な力がありませんでした。 2日目に、モニター、キーボード、マウスを接続しました。 驚いたことに、Windows Server 2003がありました。



このマシンは次の機能を引き受けました。



  • ルーター
  • DHCPサーバー
  • データベースサーバー「帆」
  • サーバー1Cアカウンティング
  • CommFortメッセンジャーサーバー
  • ファイル共有


グースバンプスは、私が見たものから背中を駆け下りました。 当時、私は小さなLinuxを知っていて、 MOPS Linuxディストリビューションのみを使用していました。 その時までに、それはもはやサポートされていなかったので、 CentOSを試してみることにしました。



職場の死体から作業プラットフォームを組み立て、CentOSの機能を調査し始めました。 これは私にとっては初めてのことでしたが、すぐに慣れ、2か月後に新しいサーバーが機能し始めました。 Parusデータベースは強制終了され、データベースへのアクセスはディレクターのコンピューターでローカルにのみ実行され、1Cサーバーは強力なコンピューターに移行され、サードパーティ企業がサポートを開始しました。



新しいサーバーで実行しました：



  • DHCPサーバー
  • Ejabberdサーバー
  • イカプロキシ
  • 感染をフィルタリングするためのSambaおよびESETでのファイル共有


その後、クローゼット内の瓦をすべて取り除き、プロファイルからラックを組み立てました。 私は再び現れませんでした。 1年の仕事の後、イリーナは私に電話をして、システム管理者のポストが減らされて、彼女が私と一緒に働くことを非常に喜んでいると言いました。



おわりに



2か月前、Irinaから再度電話があり、ViPNet Coordinatorで静的IPアドレスが割り当てられていて、サーバーのセットアップに助けが必要だと言われました。 物語の冒頭で、私はこの企業で働いていなかったので、拒否することにしました。



1か月前、電話での会話が再び行われました。 今回は、職場が間もなく開放され、給与が2倍になり、条件が変わらないことが報告されました。 今回は同意しましたが、パートタイムで起きて、弟が2番目のパートになるという条件で。



私自身の理由により、協力を継続するよう促されました。



  • 個人的な目的で静的IPアドレスを使用できます。
  • 余分なお金は常に歓迎されます。
  • 弟が実務経験を積むのを手伝います。 彼はまだ学生です。
  • サーバーを新しいハードウェアに移行したいという強い要望があります。


この記事は、100台のマシンがある小規模オフィスでCentOS 7でサーバーを起動する手順を詳細に説明する一連の記事の始まりです。
