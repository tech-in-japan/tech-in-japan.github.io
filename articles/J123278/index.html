Google+から赤い投稿を削除するユーザースクリプト

Google+はメッセージに負担をかける可能性があります。 設定のすべてのメールをすぐに削除しました。 上部の黒いバーにある赤いメッセージは少し面倒ではありませんが、良い会社の他のすべての製品を通る赤い糸を通過します。



シンプルなユーザースクリプトは、メール、ドキュメント、カレンダーからそれらを取り除くのに役立ちます。



http://userscripts.org/scripts/show/105894



ソースコードは非常に簡単です。 これは基本的に1行です。



try {document.getElementById("gbg1").style.display = 'none';} catch(e) { }







緑色の[インストール]ボタンをクリックするだけで、Chromeにインストールできます。 スクリプトの内容が信頼できない場合は、このボタンの[名前を付けて保存]をクリックして、スクリプトを（たとえば、デスクトップに）ダウンロードします。 コードを注意深く見て、おそらく修正してください（おそらく、読者にメッセージを表示したいかもしれません）。 Chromeで新しいタブを開き、マウスでファイルをドラッグして、インストールに同意します。
