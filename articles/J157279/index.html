MetroスタイルのWebインターフェイスを作成するためのツールのコレクション

Windows 8のスタイルでインターフェイスを作成するためのフレームワーク、テンプレート、jqueryプラグイン、およびアイコンの選択を紹介します。







フレームワークとテンプレート



METRO UI CSSは、ウクライナの開発者Sergey Pimenovによる現時点で最も人気のあるフレームワークの1つです。







BootMetro -Twitter Bootstrapのテンプレート。最近、作業プロジェクトで既に使用できる新しいバージョンをリリースしました。







metro-bootstrapは、LESSを使用した同様の開発です。







Metro UIテンプレートは、人気が高まっているもう1つのフレームワークで、品質と、RSS、Twitter、天気などの既成の「タイル」が多数あることで区別されます。







Droptiles.comは、Windows 8スタイルのスタートページです（iGoogleに似ています）。







Metro UIのPHPイメージギャラリー -メトロスタイルのフォトギャラリー。







jQueryプラグイン



jq-metro-メトロインターフェイスのいくつかの要素（プログレスバー、スイッチ、パノラマなど）の作成を支援するプラグイン







Metro JS-タイルおよびその他の要素の作成に役立ちます。







jQuery Metro UIプラグインは、メトロタイルの作成者でもあります。







jQuery Mobile Metroテーマは、モバイルバージョン用のプラグインです。







jQuery UI向けのMetro UIテーマビルダー -優れたMetro要素の作成。







TileJS（jQueryではない）は、単純なタイル作成スクリプトです。







無料のアイコン



.design、48 * 48px PNG形式のXAMLファイルとしての最新のUIアイコン 。







Metro UI Dock Icon Set -512 * 512px .PNG、256 * 256px .ICOおよび.SVG形式







メトロアイコン -130 .PNGアイコン。
