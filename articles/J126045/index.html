TileMillカートグラフィックデザインスタジオの新リリース

こんにちは。 地図作成スタジオTileMillの以前のバージョンがリリースされてから6か月も経っていません（ レビューを参照）。新しいリリースがバージョン0.4でリリースされました。







面白い？ あなたを猫に招待します。



新しいバージョンでは、インストールプロセスが大幅に簡素化されました。 ソースからスタジオを構築する必要はありません。MacOS XとUbuntu Linuxのバイナリパッケージが利用可能になりました。 ほとんどのスタジオコードは、インストールプロセス（現在はnpm 1.0.22を使用）からレンダリングエンジンで終わるNode.js 0.4.9を使用して書き換えられました。



設置





Mac OS XでのTileMillのインストールは大幅に簡素化されたので、* .appファイルを1つダウンロードし、それをApplications



ディレクトリに移動して、アプリケーションをdock'



から起動します。



Ubuntu LinuxユーザーはMapBox PPAを利用できるようになりました。これにより、 apt-get



またはUbuntu Software CenterからTileMillをインストールできます。



以前にTileMillのインストールに問題があった場合は、すべてがスムーズに進むはずです。



より高速なレンダリング





node-mapnik



の新しいメタタイルメカニズムのおかげで、TileMill 0.4はマップを以前のリリースよりも2.5倍速くレンダリングします。 エクスポートメカニズムのいくつかの改善点により、マップをMBTiles形式に変換する速度が向上します。



残りすべて





これらすべてに加えて、新しいリリースでは、次のようなさまざまな改善が行われています。

  • 更新されたユーザーインターフェイス（マップ表示領域が大きくなり、デザインがより「きれいに」なりました）
  • フォントと色の選択ダイアログの改善
  • エクスポートの終了までの時間が計算されます
  • MBTiles形式にエクスポートされたマップの組み込みビュー
  • データソースとの簡略化されたダイアログ（ローカルファイルシステム用のブラウザーが追加されました）
