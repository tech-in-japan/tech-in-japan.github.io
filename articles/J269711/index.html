Vivaldi 1.0.303.37の別のビルド-ベータ候補

みなさんこんにちは！



いくつかの重大なエラーを修正した次のプリビルドをご紹介します。



ページのサムネイルとディスク管理



前回のビルドがリリースされた後、ブラウザがハードドライブに対してあまりにもアクティブであるという多くの不満がありました。 判明したように、問題はページのサムネイルを作成するプロセスにありました。ブラウザは、Expressパネルのページにアクセスするたびにサムネイルの新しいコピーを作成しました。 現在、このエラーは修正されています。



コンピューター上のこのバグの痕跡を完全に取り除くには、次の手順を実行できます。



  1. ブラウザを閉じる
  2. ブラウザのユーザープロファイルにログインします
  3. トップサイトファイルを削除


別の方法は、プロファイルを削除して新しいアセンブリに切り替えることで、ブラウザをきれいに再インストールすることです（ブラウザに重要なデータが保存されていない場合）。



また、Facebookで作業する際のバグを修正し、Macバージョンにいくつかの修正を追加し、フィンランド語のローカライズリストにフィンランド語を追加することで、フィンランドのユーザーに良い行いをしました。



今のところすべてです。 以下のリンクからアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット


変更の完全なリスト：



  • VB-9560設定に間違った言語名が表示される
  • VB-9847 Macでのコマンドキーによるブックマークツリーの選択
  • VB-9936ウィンドウコントロールが設定の全高をカバーしていないウィンドウのタイトルバー
  • VB-8512 Facebookは、HTML5通知を有効にするようユーザーに要求しようとすると黒のままです
  • VB-10231 Ctrl + wを押すとブラウザがすぐにクラッシュする
  • VB-10175インストーラーにフィンランド語の翻訳がない
  • VB-10143ベータ版のさらなる翻訳
  • VB-10251スピードダイヤルを開くたびにディスクに保存される新しいタブサムネイル
  • Mac用のより小さい自動更新パッケージ：17Mbの節約


毎週のビルドはテストであるため、重要なデータをバックアップすることを忘れないでください。 前のアドレスで見つかったエラーを報告します 。
