Jquery 1.6.2が登場しました

公式ブログからの翻訳


JQuery 1.6.2がついにリリースされました！ これはjQuery 1.6の2番目の更新リリースです。



いつものように、 圧縮と非 圧縮の 2つのバージョンのjQueryを提供しています（デバッグまたは学習用）。



上記のURLを安全にWebサイトに直接含めて、jQueryを高速で読み込むことのすべての利点を得ることができます。



バージョン1.6.2の変更点のリスト：



属性


＃9286： 'submit'でのlive（）の使用を修正

＃9298：IE7では$（elem）.attr（eventName）が正しく機能しませんでした

＃9329：.attr（「タイトル」）がIE7で正しく機能しなかった

＃9362：.attr（「値」）がメーター要素で機能しなかった

＃9468：$（ 'form'）。Attr（ 'class'）はIE 6で常に 'undefined'を返しました

＃9499：.removeClass（）はクラスの最初のインスタンスのみを削除しました



コア


＃9301：ハイフンを介してキーを持つオブジェクトを介してデータ（）を設定すると、データにアクセスできなくなります。

＃9368：jQuery.camelCase（）関数はcss.jsで定義されていますが、他のコンポーネント（カーネルに移植）で使用されていました。

＃9471：IFRAMEのHTMLページを含むjquey.1.6.1.min.jsをロードするとメモリリークが発生する



CSS


＃7557：outerWidth（）を修正

＃9237：.css（）は、プロパティの相対値では機能しませんでしたが、測定単位（たとえば、「+ = 2px」）を示しました



マークアップ


＃9300：outerWidth（true）を修正



効果

操作
