技術の性差を埋める方法

Clara Marie Schroederによる記事「TechのGender Gapを修正する方法」は、女性がテクノロジーに参加していないという問題に注目しています。 これは、CIS諸国にとってさらに関連があると思いました。







私たちは、技術に性差があることをよく知っています。 Code Ed、Code Now、MicrosoftのDigigirlzなどのイニシアチブの努力にもかかわらず、2025年までに、この分野の女性の割合は24％から22％に低下すると予想されています。



比較のために、1995年にはこの数字はかなり高く、37％でした。 これは、プログラミングを学ぶために女の子を奮い立たせるだけでは十分ではないことを示しています。私たちは、この分野でのキャリアを築くのを正確に妨げるものについて、さらに話す必要があります。



技術におけるジェンダーの不平等について話すには、この問題を保護する文化的および社会的態度を考慮する必要があります。 非常に若い年齢から、女の子は男の子よりもテクノロジーに遭遇する可能性が低くなります。 女の子は人形で遊び、男の子は車とロボットで遊びます。 一見、これはそのようなものではありませんが、将来の生活に影響を与えるパラダイムを形成します。



小学校での研究では、子供たちが科学者を描くように頼まれたとき、彼らのほとんどは白衣の中年の白人を描くことを示しました。 おそらくその年齢でさえ、子供たちは男の子は女の子よりも科学の対象になりやすいと信じています。 そのようなステレオタイプは、後に重大な結果をもたらします。 ブリティッシュコロンビア大学のトニー・シュマイダー教授は、「子どもたちが能力の限界をどのように定義するかについて、ステレオタイプが重要な役割を果たす」と主張しています。 Los Angeles Timesの調査によると、このような性別のステレオタイプを信じる女の子が多くなればなるほど、自分の能力に自信が持てなくなります。 子どもの権利擁護者であるマリアン・ライト・エーデルマンが言ったように、「あなたはあなたが知らないものになれない」。



青年期までに、多くのコンピューター科学者は男性のオタクと関係し始めています。 ポップカルチャーはこのステレオタイプを強化するだけです。 コミックやビデオゲームからさまざまなCEOのサクセスストーリーまで、テクノロジーに関連するものはすべて男性に関するものです。 その結果、最終的なコンピュータサイエンステストに合格した学生のわずか27％が女性であることがわかりました。 2016年には、 8つの州でこのテストの参加者が10人未満でした。



この傾向は、NTIMの専門分野の卒業生における女性の割合が35％である大学でも続きます。 コンピューターサイエンスでは、この数値はさらに低くなります。 残念なことに、すべての障害にもかかわらずテクノロジーでキャリアを築く少数の女性は、テクノロジー企業で確立された文化が成功を許さないという事実にしばしば直面します。



この問題には簡単な解決策はありませんが、問題の文化的および社会的ルーツを理解することはすでに始まりです。 このソリューションは、コンピューターへの基本的なアクセスから始まり、企業の企業文化を改善することで終わります。これには、両親、政治家、企業の共同の努力が必要です。 両親は、少女が幼い頃からテクノロジーに対処できるようにします。 政治家は、学校でも、教師であれ、歴史上の人物であれ、子どもたちがテクノロジーで従うべき女性モデルを見せられるようにすることができます。 「隠された人物」などの映画の成功は、類似した主題の物語の関連性を示しています。 最後に、テクノロジー企業は、メンタリング、職業上の関係、より包括的な職場文化を奨励することにより、女性を支援できます。 これらの努力は、時間とともに実を結ぶでしょう。 頭脳を必要とする経済で競争力を維持するためには、少女であるか少年であるかに関係なく、個々の子供の潜在能力を開発する必要があります。 国の人口の半分を無視する余裕はありません。
