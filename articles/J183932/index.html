Windows Azure Media ServicesでのMPEG-DASHサポート



MPEG-DASHは、インターネット上のビデオの適応型動的ストリーミングのオープンスタンダードであり、プラグインの使用など、以前に使用されていたApple、Adobe、およびMicrosoftの形式を置き換えます。



MPEG-DASHは、2013年4月にISO（ ISO / IEC 23009-1：2012 ）で採用された標準です。 このイベントの直後に、MicrosoftはMedia Servicesの Windows Azureプラットフォームで標準の完全なサポートを含める意向を発表しました。 現在、DASHはW3Cで採用される可能性のある適応型ブロードキャスト標準です。



現在、Windows Azure開発者はMPEG-DASH形式を使用して、コンテンツをオンデマンドでブロードキャストできます（ビデオオンデマンド）。 Media Servicesの公開機会のリリースにより、開発者はこの形式でライブブロードキャストを実行できるようになります。



そして最後に、ブロードキャストフォーマットがオンデマンドで設定され、事前の変換なしに動的に生成される場合、MPEG-DASHはWindows Azure Media Servicesプラットフォームで動的パッケージ機能を介して利用できます。

JSON Web Token（JWT）ハンドラーのグローバルな可用性



JSON Web Token（JWT）は、インターネットエンジニアリングタスクフォースによって仕様が処理されるオープンスタンダードです。 JWT標準では、HTTP許可や要求パラメーターなど、サイズが制限された環境での使用を目的としたコンパクトなトークン形式について説明しています。



最近の更新により、Windows AzureでのJWT標準サポートのグローバルな可用性が発表されました。 開発者は、Windows Azure Active Directoryとアクセスコントロールサービスを使用するときに、この形式をトークン形式として使用できるようになりました。



これにより、さまざまなプラットフォームの開発者がWindows Azureの認証機能をより効果的に使用できるようになります。 JWTサポートの更新により、他の多くの革新が導入されました。これについては、 公式ブログの次のリンクで学ぶことができます。 そのような革新の1つは、Windows Identity Foundationプラットフォームのより緊密な統合とより良いサポートに関するものです。
