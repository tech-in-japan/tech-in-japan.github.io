個人的な経験：ラテンアメリカでのヒスパニック系スタートアッププロモーションの取り組み





私は専門的にマーケティングに従事しており、過去数ヶ月間、新しいスタートアップ、つまり英語のテキストのエラーを修正するサービスを支援してきました。 作業の過程で、彼のチームと私はターゲットオーディエンスで実験することにしました。特に、スペイン語のネイティブスピーカーからフィードバックを得ようとしています。



ラテンアメリカではマーケティング予算がなかったため、コンテンツマーケティングを使用することにしました。 しかし、スペイン語で会社を宣伝する方法と場所については、賢明な資料はほとんどないことが判明しました。 私は自分でそれを把握する必要があり、実験の結果に応じて自分の経験を共有したいと思います。



ステップ＃1：情報を収集する



インターネットのヒスパニックセグメントのデバイスについては何も知りませんでしたので、初心者には少なくとも基本的に勉強する必要がありました。 チームにはスペイン語話者がいないので、インサイダー情報を入手する場所がありませんでした。データはロシア語で、主に英語で収集されました。



スペイン語を話すユーザーは、コンテンツを公開するための人気サイトの独自の類似物を持つ必要があることを提案しました。 たとえば、Reddit。 Quoraなどのサイトでのさまざまな検索クエリやテーマスレッドの調査では、あまり情報が提供されませんでした。人気のあるTaringaサイトが見つかりました。それだけです。



Upworkについては、スペイン語のネイティブスピーカーにSMMの専門家に相談する必要がありました。 その後、事態は改善しました。



ステップ＃2：コンテンツを公開する



「パルチザン」形式の興味深い英語の資料がたくさんありましたが、それらを翻訳するのは難しくありませんでした。 さらに興味深い質問は、これらのテキストを公開する場所です。



さまざまなブログプラットフォームを調査し、状況を複雑にせず、Mediumでコンテンツを公開することにしました。 このオプションにはいくつかの利点がありました-プラットフォームは使いやすく、使い慣れたインターフェイスを備えており、サイト上でローカル言語でコンテンツを公開できます（ロシア語でも多くの興味深いブログがあります）。これは、将来の作業を計画するのに役立ちました。



ステップ＃3：プロモーション



コンテンツの「播種」のために、2つの主要なリソースを選択しました。



  • Taringa.netはラテンアメリカで人気のあるアグリゲーターサイトで、Redditに多少似ており、2100万人以上の訪問者がいます。
  • Meneame.net -950万人の訪問者。


元のテーブルにはより多くのサイトがありましたが、それらのかなりの部分が小さすぎてトラフィックがまったくなかったことが判明し、「ヒープ」モードでもそこに再投稿することを徐々に拒否しました。



それらのそれぞれで、あなたはリンクと記事を公開できます、特に困難はありません。 Meneameなどの一部のリソースでは、追加の制限が課されます。たとえば、投稿を公開する前に、他のユーザーの投稿に7〜8のいいねを付ける必要があります。



Google Chromeの自動翻訳機能のおかげで、コンテンツ自体の公開も簡単です。







私たちの仕事は製品に関するフィードバックを取得することでした。そのため、潜在的なオーディエンスの地域を拡大することが重要でした。 不思議なことに、Redditはここで救助に来ました。



このプラットフォームには、さまざまな国専用のサブセクションがあり、それらのコンテンツは現地語で公開されていることが判明しました。 幸運なことに、プロジェクトのテーマはこのようなサブレディットでの公開に適していました。このツールは英語学習に役立ち、多くのヒスパニック系ユーザーにとって興味深いものです。



その結果、次のようなsubredditsにコンテンツを配置しました。



  • r / aprendeingles / -英語の勉強に専念するセクション。
  • / r / es / -一般的なテーマのヒスパニック系subreddit、14.3千ユーザー。
  • / r /アルゼンチン/-92,000人の加入者。
  • / r / vzla / -ベネズエラのセクション、34.5千人の加入者。
  • / r /コロンビア /-13.8千人の加入者。
  • / r / chile /-48,000人のユーザー。


結果は何ですか



1か月の間に、4つの資料を公開しました。数千人がそれらを読み、登録で良い転換を得ました。 ヒスパニックの視聴者は、このサービスを非常に積極的に使用し、プレミアムアクセスを取得しています。 さらに、このプロジェクトはラテンアメリカのジャーナリストとブロガーによって注目されました。 特に、人気のあるアルゼンチンのITリソース「Wwhat's New」のレビューの後、数日で数千人が来ました。



記事の一般的なトラフィック分布は次のようになります。







MeneameやTaringaなどの集約サイトは優れたパフォーマンスを示しており、Redditにはライブオーディエンスがいます。 プラットフォーム自体のメディアアルゴリズムによって素材が宣伝されることは明らかであり、それだけの価値はありません。ここで、最初の仮説が確認されました。



おわりに



もちろん、このようなコンテンツマーケティングは大量のトラフィックを引き付けるツールではありませんが、予算をゼロにして新しい市場をテストする場合、Medium + Aggregators + Redditの組み合わせは非常に効果的です。
