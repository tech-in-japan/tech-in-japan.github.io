調査：IaaSプロバイダークライアントが使用するOS

先日、私たちは、どのOSがお客様の間で最も人気があるかについて小規模な調査を実施することにしました。 これについて、そして西洋の経験について少しお話します。





/ 写真Perspecsys Photos CC



お客様からのフィードバックの調査と分析に基づいて取得した独自のデータは、仮想サーバーで最も人気のあるOSはWindows Server 2012 （32％）であることを示唆しています。



これにWindows Server 2008（23％）が続き、3番目にUbuntu 14.04（12％）があります。 Ubuntu 12.04とCentOS 7は4位と5位を共有しています。 サーバーの8％を管理します。



このインサイダーは、他の分析レポートの結果を見ると非常に興味深いものです。 たとえば、Amazon Elastic Compute Cloudの調査では、10万を超える仮想サーバーがUbuntuを実行しており、Amazon Linux（サーバーのほぼ3分の1）が続き、Windowsが上位3つを閉じていることが示されました。



Ubuntu開発者は、OSの最大の配布に関心があります。 Amazonだけでなく、Microsoft、HP、Googleなどのクラウドでも機能します。 この戦略は結果をもたらします。 別の調査によると、OpenStackを調べても、53％がUbuntuを使用しており、CentOSが29％になっています。
