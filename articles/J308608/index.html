UI / UXデザイナー向けの12のツールとサイト

開発ツールは、使いやすさの面でもユーザーエクスペリエンスの面でも常に改善されています。 デザイナーとユーザーの両方にとって非常に便利です。 当然のことながら、経験豊富なWebデザイナーは時代に遅れずについていき、新しいツールを学ぼうとします。 数百のデザインブログやWebサイトの著者は、定期的に新しい記事を公開し、知識を共有し、新しいツールの使用に関する推奨事項を提供しています。 デザイナーにとって、これはすべて非常に有用な情報源です。 また、デザイナー向けの最新のツールとWebサイトも取り上げました。











1. UXチェックリスト



すべての問題を解決し、プロジェクトをクライアントに転送する準備ができていますか？



UXチェックリストは、ユーザーエクスペリエンスのあらゆる側面を専門的に検証するものです。 プロジェクトが完了したかどうかに関係なく、プロジェクトの任意の段階でチェックを開始できます。









2.素晴らしいフォント



Font Awesomeは、サイズ変更、色、影など、CSSで可能なすべての機能をすぐにパーソナライズできるスケーラブルなベクターアイコンを提供します。









3.完璧なアイコン



Perfect Iconsは、画面解像度に関係なくソーシャルネットワークアイコンを作成するためのシンプルなツールです。









4. Freebiesbug



Freebiesbugには、テンプレート、UI、プラグインに関するさまざまなリソースを含む、便利なWebデザインリソースが豊富にあります。









5. Pttrns



Pttrns Webサイトでは、モバイルデバイス用のWebサイトテンプレート、UIKitのリソース、およびUI要素をデザイナーが見つけます。









6.フラットUIカラーピッカー



カラースキームが必要で、完全にユニークなフラットデザインインターフェイスを作成する場合は、フラットUIカラーピッカーが最適です。









7.技術とすべて



Tech＆Allは、Webデザイン、コード、テンプレート、ビジュアルデザイン、UIKits、UI要素、および優れたサイトナビゲーションの開発について知っておく必要があるすべてをまとめました。









8. Site Inspire



インスピレーションをお探しですか？ サイトの使用方法がわからない場合 Site Inspireは、これらの問題を解決するアシスタントになります。 サイトの素材は、カテゴリ、プラットフォーム、スタイルでフィルタリングできます。



9. GoodUI



どのUIデザインをいいと思いますか？ GoodUI Webサイトには、役立つ情報やヒントがたくさんあります。









10.三角形化



Trianglifyは美しいポリゴンスタイルの背景を生成し、SVGおよびCSS形式で保存できるようにします。









11. UIの顔



UI Facesは、UIモックアップのシンプルなアバターを見つけることができるサイトです。 ビジュアルデザインに最適です。









12. PrimeLiber



PrimeLiberはロシア語を話すユーザー向けの無料ブログで、スタートアップ、UI / UXデザイン、生産者、CEO、プログラミングに関する情報を見つけることができます。 ロシア語を話す加入者のための非常に良いリソース。
