AndroidアプリをサポートするNokia Xファミリーの紹介

こんにちは、Habr！

これについては既に聞いたことがあるかもしれませんが、正式に確認しました。本日、Mobile World Congressで、Androidアプリケーション、統合Microsoftサービス、NokiaブランドサービスをサポートするNokia Xスマートフォンの新しいラインを導入しました。 同梱されているスマートフォンと開発者ツールに関する興味深い詳細情報が、あなたを待っています。







そのため、提示されたラインは、Nokia XおよびNokia XLの基本モデルと大画面5 'で構成されています。 モデルは「急成長」市場に焦点を当てており、Lumiaラインより下で、Ashaより上の価格セグメントに表示されます。







Nokia Xラインの最初のモデルには、対角線4インチ、解像度800 x 480ピクセルの静電容量型IPSディスプレイが搭載されています。 モデルの機能には、1GHzデュアルコアQualcomm Snapdragon S4プロセッサー、2つのsimカード（3G）のサポート、512 MBのRAMおよび4 GBの内部メモリ、最大32 GBのメモリカードのサポート、フラッシュなしの3メガピクセルカメラ、1500 mAhバッテリーが含まれます。 モデルはロシアで次の色で表示されます：明るい緑、明るい赤、黄色、黒。











Nokia XLスマートフォンは、大型5インチディスプレイ（IPS、800 x 480ピクセル）、Skypeビデオ通話に最適な2メガピクセルのフロントカメラ、およびオートフォーカスとフラッシュを備えた5メガピクセルのリアカメラを備えています。 このモデルは、周波数が1 GHzのデュアルコアQualcommプロセッサーにも基づいています。 このモデルには、microSDをサポートする4 GBの内部メモリと768 MBのRAMがあります。 Nokia XLは、明るい緑、オレンジ、黄色、青、黒の色域で表示されます。











スマートフォンは、Nokia X Software Platform（以下NXSP）で動作します。これは、Android Open Source Project（AOSP）、Nokia独自のシェル、およびNokiaとMicrosoftの統合サービスの組み合わせです。



インターフェイス自体が、Windows Phone 8とAshaプラットフォームのソリューションの成功した共生であることは注目に値します。 最初のNXSPは、ロック画面、Nokia Glance画面、およびサイズを変更できる正方形のアイコンが付いたメイン（垂直スクロール）画面を引き継ぎました。 Ashaプラットフォームは、Fastlane画面を「共有」します。 この画面には、メッセージ、通話、リマインダー、最近起動したアプリケーションに関する情報が、時間順に並べられたリストの形式で表示されます。 Fastlane画面を開くには、ホーム画面を左右に「スワイプ」します。



サービスとアプリケーションに関しては、ノキアXスマートフォンは、同社のモバイルデバイスの他のラインからも例を挙げました。 たとえば、写真やその他のコンテンツのクラウド同期はOneDrive（以前のSkyDrive）を介して行うことができ、ここではSkypeが標準のメッセンジャーとして使用されます。 ユーザーは、高品質のオフライン作業とステップバイステップのナビゲーションをサポートする無料のHEREカードや、音楽を聴いたりプレイリストをダウンロードしたりする無料のNokia MixRadioなど、Nokiaのブランドサービスも利用できます。 さらに、さまざまなサードパーティのアプリケーションとゲームがすべてのNokia Xデバイスにプリインストールされています。



ここで1つの重要な点に言及する価値があります。NokiaStoreはXラインのアプリケーションストアとして使用されます。 ただし、ユーザーが突然必要なアプリケーションを見つけられない場合、サードパーティのオープンアプリケーションストア（Yandexストア、SlideMeなど）の利用可能なクライアントの1つで検索するように求められます。



Nokia Xスマートフォンの将来のユーザーにとってもう1つの嬉しい驚きは、60か国の固定電話と8か国の携帯電話へのSkypeを介した世界中での1か月間の無料通話です。



開発者へ



Androidアプリケーションの移植またはNokia X Software Platformの作成に関心がある場合は、開発者向けに以下の概要を確認することを強くお勧めします。







さらに、Nokia Developer Webサイトに特別なセクションを作成しました。このセクションでは、開発の観点からプラットフォームに関する追加情報を入手できます。 たとえば、Androidアプリケーションの75％が、移植を必要とせずにNXSPに適していることがわかり、アプリケーションの互換性を確認できます



PS



ちなみに、Nokiaモバイルデバイスでのテスト済みアプリケーションのリモートインストール用ツール-Remote Device Accessを更新しました。 Nokia X Software Platformのサポートが追加されたため、最大限の利便性でポータブルおよびゼロから構築されたアプリケーションをテストできます。
