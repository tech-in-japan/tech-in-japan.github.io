Ubuntu 9.10 Karmic Koalaリリース

私たちは皆これを待っていました。そして今、私たちの計画に厳密に従って、新しいバージョンのUbuntu 9.10（コードネームKarmic Koala）が最終バージョンに光を当てました。


配布物は 、 公式Webサイト からダウンロードできます 。 また、 KubuntuなどのすべてのUbuntaオプション、ネットブック（ Netbook Remix ）およびサーバー（ Server Edition ）のバージョンも同時に更新されました。



Canonicalサーバーを過負荷にしないために、torrentからダウンロードすることをお勧めします。

DVD： i386 | AMD64

mirror.yandex.ruも更新されました： Ubuntu | クブンツ



NTV がリリースのリリースも発表したことは素晴らしいことです。



ベータ版とRC版を見逃した方のために、変更点について簡単に説明します。



このバージョンはLTSではありませんが（来年4月に次の長期サポートが待っています）、たくさんの特典が含まれています。 完全なリスト（ベータ版またはRC用に作成されましたが、完全にリリースに適用されます）は、ロシア語のubuntu.ruまたはHabréと英語の両方で見つけることができます。



主な変更：
  • ダウンロードプロセスの最適化、 Upstartの使用への移行
  • ソフトウェアセンター
  • GNOME 2.28 （新しい壁紙、アイコン、およびデフォルトのIMとしてのPidginではなくEmpathyを使用）
  • Ubuntu One共有サービス
  • Linuxカーネル 2.6.31-11.3
  • デフォルトでのExt4ファイルシステム（新規インストールのみ）
  • GRUB2ブートローダーへの切り替え（新規インストールの場合のみ）
  • デバイスキットを支持するHALメカニズムの放棄
