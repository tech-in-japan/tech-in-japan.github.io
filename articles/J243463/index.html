Androidスマートフォンで個人の写真を保護する





携帯電話の個人写真を見知らぬ人がアクセスできないようにする方法は？ 機密写真を撮影するときに暗号コンテナに自動的に記録されるように電話を設定するにはどうすればよいですか？ そして、それらへの迅速かつ便利なアクセスの可能性を維持する方法は？



これらの問題は、LG-E988が予期せず動作を拒否し、それを復活させるためのいくつかの失敗した後、私に関連するようになりました。サービスセンターへの訪問は避けられないことに気付きました。 これに伴い、私の個人的な写真はすべてデバイスの内部メモリに保存されていることがわかりました。 部外者が彼らの余暇で彼らを見ているのは本当に望まないでしょう...しかし、他の状況があります、あなたの親relativeはそれを見るために電話を取りました、それは友人、同僚の手にあったか、失われました。



この事件の後、私は自分の写真を外部メモリカードに保存し、さらに暗号化する方が良いことに気付きました。 私はURSafe Media RedirectorおよびCyber​​Safe Encryptionアプリケーションを使用してこの問題を解決することができました 。 これを行うには：



1. URSafe Media Redirectorで、写真を保存するための標準ディレクトリ（/ storage / sdcard0 / DCIM / 100LGDSCがある）ではなく、外部SDカード（/ storage / sdcard1 / Photos）を指定しました：







テストショットを撮りました。写真を撮った後、必要なPhotosフォルダーに保存されました。



2. Cyber​​Safe Encryptionで新しい金庫を作成し、 プロパティで同様のマウントパスを設定します -金庫はPhotosフォルダーにマウントされます：







このモードでは、コンテナファイルシステムがデバイスファイルシステムにマウントされます。 このモードでは一時ファイルは使用されません。



金庫自体の秘密について話すと、3つの可能性があります。

-セーフはGUIプログラムから、またはCyber​​Safeウィジェット（デスクトップに追加可能）を使用して開きます。

-金庫は、それを使用する作業の開始時にのみメイン画面に追加されます。 安全なファイルがAndroid / data / com.android.browser / filesのどこかにあり、たとえばSystemCashe.Tableと呼ばれる場合、デバイス上でそれを見つけるのは非常に困難です。

- 安全なファイルはデバイスにまったく保存されません 。



3. Cyber​​Safe Encryptionの利便性のため、 サードパーティのファイルマネージャー -ES 3を指揮者として選択しました。 金庫を開いた後、まだファイルがありません。エクスプローラーには、リダイレクトが行われたフォルダーへのパスが表示されます。 いくつかのテストショットを撮り、エクスプローラー内のフォルダーの内容を更新しました-写真はマウントされたPhotosフォルダーに入りました：







4. Cyber​​Safe Encryptionで金庫を閉じ、写真フォルダーの内容を更新しました-その結果、以前に/ DCIM / 100LGDSCから移動したものと、リダイレクト後に撮影された1枚のテスト写真がありましたが、取り付ける前。 マウントモードで撮影された写真は、フォルダに表示されません-それらは安全で暗号化されています。







つまり、暗号化を必要としない写真は、金庫が閉じられたときに行う必要があります。 セーフを写真フォルダに再マウントします-暗号化されたファイルがディレクトリに再び表示されます。



このソリューションは、Google Playで写真を隠すためにこれらのアプリケーションのどれよりもはるかに役立つと思います。 まず第一に、単にファイルを隠すだけでは信頼性の高い保護が提供されないため、暗号化はそれらのすべてからはほど遠いからです。 さらに、このようなアプリケーションは、写真が電話に隠れていることを明確に示しています。



このスキームを使用した結果、カメラには通常の方法で構成されているため、電話には正確に写真が隠されている（暗号化されている）痕跡はありません.URSafe Media Redirectorは、内部メモリを乱雑にしない通常のアプリケーションです。 Cyber​​Safe Encryptionの存在は、一部のデータが電話で保護されていることを示している可能性がありますが、どのようなデータであるかは不明です。



リダイレクトがなければ、個人の写真は常に手動で金庫にコピーする必要があります。 ただし、このスキームを設定すると、マウントモードで撮影されたすべての写真が暗号化されたコンテナに自動的に書き込まれます。
