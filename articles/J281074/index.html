Azure関数



Azure Functionsは、C＃、Node.js、Python、F＃、PHP、バッチ、bash、Javaでハンドラーを記述したり、任意の実行可能ファイルを使用して外部イベントに応答したりできる新しいサービスです。 仮想マシンまたは環境を作成または管理する必要はありません。 もう1つの利点は、柔軟な価格設定モデルです。機能が動作する時間にのみ料金を支払います。 同時に、Azureは独立して負荷をスケーリングし、フォールトトレランスを提供します。



Azure Functionsプロジェクトが開いています。 Azure Functionsに最も近いものはAWS Lambdasです。 ここでは 、これら2つのサービスの優れた比較を見つけることができます。



Azureサービスファブリック



Azure Service Fabricマイクロサービスプラットフォームは商用利用されています。 現在、このプラットフォームでは、.NETおよびAzure（Windows）を使用したワールドサービスに基づいた、非常に信頼性の高いソリューションを作成できます。 ただし、BuildはJava、Linux、およびAmazonなどの他のクラウドを含むユーザー環境でのデプロイ向けにAzure Service Fabricのプレリリースバージョンをリリースしました。



Azure IoT



Buildの一環として、Azure + IoT（Internet of Things）のトピックに関していくつかの発表が行われました。 含む：

  • Azure IoT Starter Kits-価格が50ドルのメーカー向けの認定IoTキット。
  • Azure IoT Gateway SDK-インターネットの使用方法がわからないデバイスでもクラウドに接続できる新しいツールの発表。
  • Azure IoT Hub Device Management - IoT Hubに接続されたデバイスを管理するための新しいAPIの発表。
  • IoTサンプルアプリは、 ソースコードと、IoTソリューションとモバイルクライアントを構築するための個別のガイドを含む大規模なデモプロジェクトです。


Azure DocumentDB



NoSQL Azure DocumentDBサービスには、いくつかの重要な革新がありました。

  • MongoDBプロトコルのサポートは、MongoDBを使用するクライアントのAzureへの移行を大幅に簡素化する最も重要な発表です。
  • グローバルデータベース-世界中の地域間での新しいデータベースレプリケーション機能の発表 。
  • パーティション化されたコレクション-ドキュメントコレクションのパーティション化とスケーリングを通じて生産性を向上させるための新機能。
  • 価格の更新-パフォーマンスとストレージを個別に予約できる新しい価格設定を導入しました。


Azureアプリケーションの洞察



分析Azure Application Insightsの収集および表示サービスには、次の新機能が追加されました。

  • Application Insightsの分析-収集されたデータに対してクエリを作成および実行する機能を含む、蓄積されたデータの分析。

  • ほぼリアルタイムの事前検出 -異常なイベント、エラー、障害を検出、診断、通知するための新しい機能。

  • ライブストリームメトリック -Webhook Callbacksを介してリアルタイムでデータを受信する機能

  • アプリケーションマップ -各要素のパフォーマンスを表示して、アプリケーションのトポロジを自動的に決定する新しい機能。

  • Azureダッシュボード -さまざまなApplication Insightsメトリックを1つのパネルに表示する新しい機会。

  • Visual Studioの例外からコードにジャンプ -AzureポータルからVisual Studio 2015 Update 2に直接移動して、固定例外が指すコード行に移動する機能。

  • 自動ロギングフレームワークの検出、診断ハブは、VS2015 Update 2で追加されたいくつかの機能です 。


Azureストレージ



Azure Storageは暗号化機能を受け取りました 。これは、これまでのところBlobオブジェクトでのみ利用可能です。 暗号化を有効にしてストレージに配置されるすべてのデータは、抽出中に透過的に暗号化および復号化されます。 データはAES-256アルゴリズムを使用して暗号化されます。



無料のMicrosoft Azure Storage Explorerツールの3月の更新プログラムがリリースされました。 新機能の中には、テーブルとキューのサポート、SaS機能、パフォーマンスの向上、UXの改善、新しいバージョンを通知するメカニズムがあります。
