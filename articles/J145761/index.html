TechEd 2012北米：初日から学んだこと

たまたま、TechEd 2012 North Americaの初日は、主にサーバーテクノロジーに専念していました。 SearchWindowsServer.comマガジンの編集者であるBen Rubinstein（ itknowledgeexchange.techtarget.com/windows-server/lessons-learned-from-teched-2012-day-1 ）をはじめとする多くの人が、初日に面白かったことを要約しました 。 会議からの彼のメッセージの翻訳を提供します。



TechEdでは、私たちは常に何かを学びます...しかし、期待は現実と一致しています。 フロリダ州オーランドでの最初の日に、新しいWindows Intuite、CTP2 System Center 2012 SP1、Hyper-V 2012（リリース候補）のリリースだけでなく、さらに多くのことを学びました。

楽しいポイントをいくつかご紹介します。

1.信じられないほどの速度。 Microsoft Virtualization ProgramマネージャーであるJeff Woolseyは、Windows Server 2012とHyper-Vを使用して10秒で10ギガバイトのファイルを転送することで最も注目を集めました。 会議の参加者は驚いた。 そしてほとんどの場合、ジェフはHyper-Vに関する質問に答えました。

2.ファニーマークルシノビッチ （ちなみに、11月のTechEd 2012ロシア-編）。 Windows Azureでのプレゼンテーションの彼の部分は、聴衆から絶えず笑いを生み出しました。 たとえば、「いいえ、ハッキングされていません。 AzureはゲストオペレーティングシステムとしてLinuxを本当にサポートしています。

3.リアルタイムのクラウドソーシングは危険です。 Scott Guthrieは、プレゼンテーション中にAzure Webアプリケーションを作成し、メールボックスを聴衆と共有しました。 同時に、偽のSQLインジェクションから率直な質問「誰がおならだ？」まで、「特別な」メモが表示される大画面ですべてが放送されました。

4. Appleは大丈夫です。 他のデバイスと一緒に、iPhoneで一連のデモが行われました。 オーディエンスのMacbookとiPadの数を考慮すると、かなり論理的です。

5.より多くのPowershell。 PowershellがMicrosoftシステムでますます注目されているという事実は、ニュースではありません。 ただし、Don JonesとJeffrey Snoverの指導の下にTechEdがTechEdの2番目の「PowerShellクラッシュコース」セクションを追加したことは、Powershellの無知が自分のキャリアに影響を与えることをIT専門家が実際に認識し始めていることを示唆しています。

6.人々はテストに不満を抱いています。 誰もがマイクロソフトが認定プログラムを変更した方法（および理由）を理解しているわけではありません。 それまでの間、履歴書を同様の事実で埋めるために、膨大な数の人々が初日にあらゆる種類のテストに合格しようとしたことを観察することができました。

7. Twitterは複雑です。 「ビジネス向けソーシャルメディア」セクションでは、ITの観点からソーシャルメディアアクティビティをサポートすることの長所と短所について説明しました。 最も重要な洞察は、ソーシャルメディアの拡大であり、シングルコアプロセッサからマルチコアプロセッサへの移行を比較します。

8.「あまりにも多くの注目を集めることは不可能です。」 抽選、カードトリック、一輪車の男など-今年、ベンダーはすべての可能なトリックを利用して、より多くの訪問者をスタンドに引き付けました。



そして、このすべての後、Windows Server 2012とWindows 8の公式リリース日について知りたいです...



11日には、Nir Ben ZviのGunjan JainによるDynamic Access Controlに関する興味深いレポートがありました。 Windows Server 2012のファイルサーバー監査システムに関する新しい投稿で、近い将来のイノベーションに関する詳細情報を確認します。
