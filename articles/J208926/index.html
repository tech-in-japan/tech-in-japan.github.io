オリジナルのApple iMac

こんにちは、ハラジテリ！ AIOやオールインワンのようなコンピューターのブランチの開発に新しい強力な刺激を与えたコンピューターについてお話したいと思います。 1998年に光を放ったのは、オリジナルのiMacについてです。







1996年、Appleが着実に深byに足を踏み入れたとき、ユニークな革新的発展を遂げた他社の形で救いを求めることが決定されました。 選択はBe Inc. およびNeXT Software Inc. Beとの取引は、Appleがたった1億2,500万ドルを頼りにしている間にJean-Louis Gassierが2億ドルを要求したために失敗しました。 取引が行われ、Steve Jobsは取締役会の決定によりAppleのCEOになりました。 特筆すべきことは、1985年、GassierはAppleをMacintosh部門のリーダーとして残したJobsの後任となりました。 したがって、両方の入札者はアップルからの移民によって設立されました。



Appleの舵取りとして、ジョブズは会社を危機から脱出させるという積極的な政策を開始しました。 対策の1つは、前世代とは異なる、根本的に新しいMacintoshの開発でした。 その結果、この素晴らしい、かつてはカルトコンピューターが登場しました。これは、ハイテクの世界におけるスタイル、利便性、品質の擬人化でした。 前のマッキントッシュ世代の退屈な鈍さと角ばりを吹いたのはその新鮮な風でした。



だから、おそらく多くの人が若者や子供時代に写真で見たコンピューターであるオリジナルのiMacは、不透明なケースを備えた濃いターコイズ色でのみ製造されました。 将来、ケースは透明になり、カラーパレットは大幅に拡張されました。 オリジナルのiMacには、AとBの2つのリビジョンのいずれかのマザーボードが搭載されていました。違いは、インストール可能なRAMの最大量、より強力なビデオチップ、リビジョンBのより大きなビデオメモリです。また、他のiMac世代との主な違いの1つはCD- ROM これは、格納式のドライブトレイを備えた通常のラップトップドライブでした。 後続のバージョンには、ディスクのスロットローディングを備えたドライブが装備されていました。



外部ソリューション


このコンピューターの設計は、ジョナサンアイブによって開発されました。 丸みを帯びた滑らかな形状、鋭い線はほとんど観察されません。 モノブロックの下部は、半透明の灰色のプラスチックでできています。 フロントパネルは同じプラスチックでできていますが、垂直リブが追加されています。 ケースの上部も、濃い青緑色の半透明のプラスチックでできています。 上部には換気グリルがあり、その中央にはコンピューターを運ぶためのハンドルが取り付けられています。 ちなみに、iMacの重量はほぼ16 kgなので非常に便利です。 さらに、eBayでグラストップを注文すると、iMacを元のテーブルに変えることができます。







IMacはいくつかの興味深いソリューションを実装しています。 たとえば、コンピューターが低すぎる場合に持ち上げることができる折りたたみ脚。

電子部品へのアクセスは、プラスチック製の半透明のカバーで閉じられています。カバーは1本のネジで固定されています（ネジはありませんでした）。 それはかなり非自明かつ騒々しく削除されます。

RAM


さて、最も重要な部分はプロセッサとRAMです。 ヒートシンクはすでにプロセッサから取り外されています。 私が理解しているように、キャッシュは正しくない場合、はんだ付けされています-正しい。 特徴的に、プロセッサとRAMはマザーボードに「適合する」別個のボード上にあります。 メモリスロットは両側に1つずつあります。 プロセッサより少し高いのは256MB SO-DIMM SDRAMモジュールです。







私にはわからない理由で、彼は彼女の半分を見ています。 最初は、それぞれ64 MBの2つのモジュールがありましたが、彼は96 MBを見て、実験して、どのバーが完全に見え、どのバーが半分かを見つけました。 256MBのブラケットを購入したとき、彼女が彼女を完全に見ることを本当に望んでいました。 しかし、Macに関する情報で128MBのRAMが利用可能であることを見たとき、私の期待は壊れました。 次に、完全に見える64MBのブラケットを2番目のスロットに追加しました。 合計192 MBが出てきました。 MacOS 9は96MBでさえ、そのようなボリュームでさえ飛んでしまいました...しかし、記事を準備するとき、不快な詳細が見つかりました。 電源を入れると、ポピーはスピーカーから心臓破裂のきしみ音を発し、開始を拒否しました。 私はすでに幼い息子が何かで彼を殺したことに腹を立てていましたが、後で私は一度に1つずつメモリモジュールを取り除こうとしました。 残念ながら、この64MBのブラケットは消滅し、64MBのモジュールに置き換えられましたが、半分しか見えませんでした。 合計で160 MBのRAMが搭載されました。 原則として、MacOS X10.3.9を実行できますが、192 MBのRAMでもゆっくりとクロールしました。
