秋 学校に行く時間です：Windows 8の学生1日の実用的なエクスプレススクール





9月15日、モスクワで、Windows 8の最初の生徒の学校であるKrylatskoyeのマイクロソフトオフィスで、 登録は既に開始されています。



このシステムは、ラップトップ、タブレット、デスクトップコンピューターなどのさまざまな種類のデバイスの機能を組み合わせたものです。 Windows 8の場合、新しいカラフルなアプリケーションを作成して、Windowsストアで配布できます。



新しいタイプのアプリケーションの開発者の役割を試すことに興味のある方は、Windows 8のFirst Student Express Schoolにご招待します。Windows8の開発の基本原則について話し、このプラットフォームでアプリケーションを収益化する可能性について話します。特定のアプリケーション。 コミュニティの専門家やマイクロソフトと一緒に、志を同じくする人々と既成のアイデアのチームと一緒に1日でアプリケーションのプロトタイプを作成することをお勧めします。



イベントの形式について少し



イベント形式：レクチャー付きの「ガイド付きハッカソン」。 登録が必要です 。



Windows 8と開発ツールがインストールされたラップトップが必要です。



準備として、 TechDaysでWindows 8に関するレポートの記録を見ることができ、9月7日に開催されるWindows Campのオンラインブロードキャストを見ることができます。



エクスプレススクールでは、Microsoftの専門家であるKonstantin Kichinsky（ kichik ）、Dmitry Soshnikov（ shwars ）、およびStas Pavlov（ stasus ）に講義を聞いたり、質問したりできます。
