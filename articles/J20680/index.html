MicrosoftはWindowsおよびOfficeコードを開きます



CNewsでの今日の主な見出しは、昨日2月21日にMicrosoft関係者がWindows VistaやOffice 2007などのソフトウェア製品のソースコードの公開を発表したことでした。 マイクロソフトのオーナー自身によると、このようなステップは、企業の開発と他の開発者のソフトウェア製品とのより緊密な統合に貢献します。 同社はまた、オープンソースコミュニティと緊密に連携し、ユーザー向けに別のオンラインフォーラムを立ち上げる意向を発表しています。 Vestiによると、今後数週間で、数千ページのドキュメントがMicrosoft Webサイトに公開されます。 現在、3万ページ以上がすでに利用可能です。



すべてのソフトウェアインターフェイス、通信プロトコル、およびドキュメントのソースコードを開くことは、最も一般的なマイクロソフト製品と何らかの方法でやり取りするサードパーティソフトウェア開発者の「差別」を排除することを目的としています。 Windows Vista（.NET Frameworkを含む）、Windows Server 2008、SQL Server 2008、Office 2007、Exchange Server 2007、Office SharePoint Server 2007などの製品で使用可能なコード、およびこれらの開発のすべての将来のバージョン。



Microsoft CEOのスティーブバルマーは次のように述べています。

「私たちの目標は、ソフトウェア間の相互作用を改善し、顧客と開発者に選択権を与えることです。 製品に関するより多くのデータを提供することにより、製品をよりオープンにします 。 」 彼はまた、この決定が会社の収益に大きな影響を与えることはないと述べました。



AR CommunicationsのコンサルタントCarmi Levyは、この決定についてユーモラスにコメントしました。

「この声明は、マクドナルドがその代表的な料理のレシピを公開することに似ています。 会社は完全に新しいビジネスモデルへの移行の準備ができています ''



一方、欧州委員会の代表者は、そのようなステップについて非常に懐疑的です。 「欧州委員会は、製品の互換性に向けたあらゆるステップを歓迎します。 ただし、委員会は、この声明が少なくとも5行目であることに留意しています。 Microsoftはこれまでにソフトウェアの互換性を4回発表しています。」
