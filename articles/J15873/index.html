ディスコの煙





コアアニメーションは、ユーザーインターフェイスのメタファーの新しい言葉です。 それで、とにかく、アップルは言います。 MacOS X Leopardは、 Core Animationと呼ばれるシステムを導入しています。 これは、実行中のアプリケーションが関与するアニメーションおよびインタラクティブ操作を作成するための柔軟なメカニズムです。 これは、デスクトップ上のアニメーションシーンをデモンストレーションするためのツールではなく、プログラムと対話する新しい方法を作成するためのツールキットです。 キーボードとマウスに加えて、ウェブカメラとマイクもあります。 これらのデバイスは、たとえば、ユーザーの顔の表情やジェスチャー、音声、または特徴的な音に応答するようにアプリケーションを教えることにより、コンピューターとの対話のプロセスに関与できます。







カスタムインターフェイスアプローチの一般的な例は、スタイリッシュなディスコディスク書き込みユーティリティです。 ディスクの録音中、ディスコはコンサートでの煙発生器のように、煙を発し始めます。 「画面」を見てください。







プログラムの公式ウェブサイトが報告するものは次のとおりです。



ディスコは 、あなたが燃やすとリアルタイムのインタラクティブな煙を発し 、境界の再定義を開始します。 邪魔にならないようにしたいですか？ マイクに息を吹き込むと、それに応じて煙が反応します。 または、マウスでフリックしてください



つまり このユーティリティは、煙のパフを生成するだけでなく、ユーザーがそれらに影響を与えることもできます。 煙は、マウスカーソルで分散させるか、マイクに向かって吹き込むだけで吹き飛ばすことができます。



煙に加えて、ディスコは火を含むいくつかのより多くの効果をサポートします。 プログラムウィンドウだけでなく、アプリケーションアイコンも「発煙」する可能性があります。 煙で満たされ、ユーティリティのアクティビティを示します。 絶対に魅力的なアプローチ:)



Core Animationに基づいたアプリケーションの設計と構築に特化したコミュニティがすでに存在します 。 これを使用すると、「アニメーション化された」プログラムの他の例を見つけることができます。 例：

  • スキッチ！ -スケッチとスケッチを作成するためのツール。 作業中のインターフェイスの美しいアニメーション
  • PulpMotion- アニメーション画像ギャラリーを作成するためのツール




PS彼らは、ディスコは煙を自慢することしかできないと言っています:)利便性と機能性の点で、欠陥がありますが、それを購入する注文に干渉しません:)それはまだ楽しい要素です-非常に魅力的な属性です:)
