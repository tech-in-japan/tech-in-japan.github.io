開発者はFlash Media Server 3に何を期待すべきですか？



  1. パフォーマンスの向上 。 Flash Media Server 3（以降、単にFMS 3と呼びます）の並列ストリームの数は、以前のバージョンと比較して200％増加します。 同じサーバー上で複数の並列スレッドを実行できるため、この改善により個々のスレッドのコストを削減できます。 この改善は、WindowsまたはLinux Red Hat 4のサーバープラットフォームに依存しません。製品の発売後、サーバープラットフォームおよびその他の必要な特性に関する詳細情報を確認できます。
  2. H.264およびAACのストリーミングサポート 。 FMSは、Flash Player、AIR、Adobe Media Playerのビデオオーディオなど、H.264およびHE-AAC標準をサポートします。 これはすでに書かれています 。
  3. 新しいセキュリティ機能 。 RTMPには、メディアコンテンツをより強力に保護する、より新しく安全な暗号化モードが装備されています。 このプロトコルはRTMPEと呼ばれます。 SSL（RTMPS）と同様に、開発者は追加のセキュリティオプションとして使用できます。 さらに、SWF検証の新しい（または新しい）機能は、SWFを違法な使用、コピー、または他の場所への配置から保護するのに役立ちます。
  4. ライブビデオ サポートの改善 。 FMS 3は、Flash Media Encoderで作成されたスケーラブルで高品質のライブビデオとオーディオをサポートします。 そして、これらはすべて1つのFMSサーバーで機能します。
  5. Flash Lite 3 でのモバイルビデオのサポート 。 FMS 3は、Flash Lite 3プラットフォームでの記録とライブストリーミングをサポートしているため、Flash Lite 3をサポートするモバイルデバイスで任意のビデオ（YouTubeビデオなど）を視聴できます。
  6. Flash Platformヘルプラインとの互換性：ヘルプおよびドキュメントシステムFMSは、FlexおよびFlash CS 3でサポートされるように最大限に調整されています。
  7. 新しい プラグイン アーキテクチャ 。 FMS 3は、強力なC ++ APIを導入することで拡張性を高めます。これは、サードパーティの開発者がサーバーの機能をニーズに合わせて向上させるのに役立ちます。
  8. 別のサーバーに公開します 。 これは、開発者がライブビデオ（オーディオ）を別のFMSサーバーまたはSDNに公開できるようにする素晴らしい機会です。


FMS 3はいつ発売されますか？<O：p> </ o：p>

FMS 3は2008年の第1四半期に発売されます<o：p> </ o：p>



FMS 3の 価格は いくらですか？ また、どのバージョンが提供されますか？<O：p> </ o：p>

製品の価格はまだ議論されていません（しかし、FMS 3は明らかにもっと高価になると思います）。



<o：p> </ o：p> すでに FMS 2を 使用してい ますが、 FMS 3 にアップグレードできます か？<o：p> </ o：p>

はい。もちろん、FMS 2をFMS 3にアップグレードできます。また、Flash Communication Server 1.xをまだ使用している場合、アップグレードするには、このバージョンをFMS 2にアップグレードしてからFMS 3にアップグレードする必要があります。



LinuxのどのバージョンでもFMS 3 を実行できます か？<O：p> </ o：p>

FMS 3は、Windows Server 2003（すべてのビルド）とLinux Red Hat 4を公式にサポートします。さらに、FMS 3（アプリケーション開発バージョン）はWindows XPをサポートします。



<o：p> </ o：p> Flash Player のどの バージョンが FMS 3 をサポートし ますか？<o：p> </ o：p>

FMS 3は、Flash Player 6以降、通常の接続とストリームの操作を実行できます。ただし、安全なRTMP、SWF検証、H.264サポートなどの新機能を使用するには、プレーヤーをFlash Player 9にアップグレードする必要があります。 Flash Playerビルド、 Adobe Labsでいつでも見つけることができます）



FMS 3は AIRで 動作し ますか？<O：p> </ o：p>

はい、もちろんそうです。 AIRをFMS 3と組み合わせて使用​​して、デスクトップアプリケーションを開発できます。 ビデオチャットとオーディオチャット（および原則として通常のチャット）、インスタントメッセージングプログラムを作成できます。 それはすべてあなたの想像力にかかっています。



FMS 2 用に開発されたアプリケーション は FMS 3で動作しますか？<O：p> </ o：p>

はい、クライアント部分またはサーバー部分を変更したかどうかに関係なく、変更されます。



FMS 3は ActionScript 3を サポートしています か？<O：p> </ o：p>

FMS 3はActionScript 3をサポートしていますが、クライアントアプリケーションとしてのみです。 ActionScript 1を使用してサービス（アプリケーション）を開発する必要があります。FMS3でAMF3サポートが導入され、ユーザーはビットマップデータ（ビットマップデータ-つまり、オプションとして、写真、オンライン図面など） ）またはその他のビットマップデータ）。



<o：p> </ o：p> 64ビット Windowsまたは Linuxシステムで FMS 3 を実行でき ますか？<o：p> </ o：p>

FMS 3は、64ビットシステムと32ビットシステムをサポートしています。 これは、FMS 3が2 x GBを超えるファイルを処理できることを意味します。 <o：p> </ o：p>





FMS 3には どのようなユーティリティが付属してい ますか？<O：p> </ o：p>

FMS 3とともに2つの新しいユーティリティが購入に含まれています。FMSCheck-コマンドラインを介して実行され、現在のシステムの下からFMS 3の操作性をチェックするプログラム。 たとえば、このプログラムをスクリプトで使用して、FMS 3の現在のステータスを取得できます。

FLVCheck-プログラムはFLVファイルの構造をチェックし、ストリーミングモードで起動できるかどうかを判断します。 このプログラムは、MPEG構造検証も完全にサポートしています。 FLVCheck（Windowsバージョン）は、現在Adobe Labsにあります。



<o：p> </ o：p> FMS 3は HTTP-302の ようなリダイレクトをサポートし ますか？<o：p> </ o：p>

はい、これはFMS 3の新機能の1つです。リダイレクトのサポートです。



<o：p> </ o：p> 私の会社では、すべてのサーバーで IPV6 をサポートする必要があります 。 FMS 3は IPV6を サポートしてい ますか？<O：p> </ o：p>

FMS 3はIPV6を完全にサポートしています。



FMS 3は、 ライブストリームから既に録画されたビデオへの カメラの切り替えを サポート していますか？<O：p> </ o：p>

はい、ライブストリームと既に記録されたストリームを動的に切り替えるアプリケーションを開発できます<O：p> </ o：p>
