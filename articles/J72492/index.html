打ち方CDから起動できない-Windows 7のインストール時にコード5

一部のコンピューターでは、Windows 7をインストールするとき、つまりDVDから起動する段階で、「CDから起動できません-コード5」というメッセージが表示されます。 neteを掘ることはいくつかの解決策を提案

1つ目は、イメージを再構築するか（他のコンピューターで動作します）、Linuxを使用します。

1つ目は、install.wimファイルを置き換えてイメージを再構築することです。

  • Vista SP1 / 2008でディスク（イメージ）を取得（ダウンロード）します
  • 3つのフォルダーを作成しますc：\ efi-iso c：\ efi-exe c：\ efi-dvd
  • Cのblog.aproductofsociety.org/files/oscdimg.exeをダウンロードします：\ efi-exe
  • ディスクの内容を 「boot」フォルダーにコピーし、ファイルautorun.inf、bootmgr、およびsetup.exeをVista SP1 / 2008ディスク（イメージ）のルートから（または7ZipまたはWinRARを使用してイメージを解凍します）c：\ efi-dvd
  • 「ソース」フォルダーで最大のファイル「install.wim」を見つけて、Windows 7インストールディスクのファイルに置き換えます（イメージ）
  • Windows 7のディスク（イメージ）から、他のすべてのフォルダーをコピーします
  • コンソールを起動します（スタート->実行-> cmd）
  • コンソールに入力します：cd c：\ efi-exe
  • そして、oscdimg -n -m -bc：\ efi-dvd \ boot \ etfsboot.com c：\ efi-dvd c：\ efi-iso \ win7.iso "
  • 結果のorbaseを可能な限り低い速度で書き込みます

    2番目はLinuxを使用しています

    • sourceforge（最新2.4） sourceforge.net/projects/gujin/files/standard/2.7/standard-2.7.ta​​r.gz/downloadからGujinの「標準」バージョンをダウンロードします
    • ダウンロードしたアーカイブを解凍します
    • 解凍されたフォルダー内のファイル「full.img.gz」を探しています。また、解凍します。
    • その結果、ファイル「floppy.144」（フロッピーディスクのイメージ）が作成されます。
    • RawWrite（またはWinImage www.winimage.com/download.htm ）などの「フロッピーイメージライター」をダウンロードします。
    • 「floppy.144」イメージをフロッピーディスクに書き込みます。
    • Windows 7ドライブをドライブに「押し込み」ます。
    • フロッピーから起動し、あらゆる種類の質問（もしあれば）に答えます。
    • GUIが表示されたら、起動デバイスを選択し、対応する「F」起動デバイスを押します
    • おなじみの「任意のキーを押してCDまたはDVDから起動します」が表示されます。
    • 次に、通常のインストールを開始します。
