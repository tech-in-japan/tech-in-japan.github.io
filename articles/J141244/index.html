Facebookの実際の費用はいくらですか？





2012年の第2四半期はすでに本格的であり、ソーシャルネットワークNo. 1のIPOに関する噂の数は増加しています。 このネットワークに何が起こったのかを覚えて、実際にどれくらいの費用がかかるかを計算してみましょう。 1000億ドルという数字はどこから来たのか、そしてそれがなぜ高額だったのかというと。



ロイター通信によると、同社は2012年後半にIPOを実施する予定です。 2011年、facebookは1,000億ドルと評価されました。 確かに、CNBC は名前のないソースを引用してこれを報告しました 。 簡単に言えば、CNBCはこの数字を天井から取り出しただけであり、この数字の外観はすでに意見の操作です。

さて、FBが1000億グリーンの価値があると想像してみましょう。 ウィキペディアによると 、昨年の10月末は「70億日」でした-世界の人口はその数字を超えています。 次に、FBがすべての人にお金を分配する場合、惑星のユーザーごとに14.28ドルが必要になることがわかりました。 または、地球上のすべての人が500ルーブルで捨てられる場合、私たちはFBの所有者になります。 FBは、オーディエンスは8億4500万人のアクティブユーザーであると主張しています。 1ユーザーのコストは120ドルです。 私には、これらの数字はやや誇張されているようです。 一般に、私は1,000億の金額は心理評価であり、約10万マイルであるという考えに傾いています。 100万ドルを手に持たなかった人にとって、100万ドルと1億ドルの違いを想像することは困難です。 アフリカの部族の住民を思い出してください。ここでは、数字から1、2、そして「たくさん」しかありません。 私たちは実際、これらのアフリカの部族の住民です。

確かに、一つだけあります。 まず、無制限の資金で無制限の数量で株式を購入し始めると、価格が変わります。 そして、あなたが株式を100％必要としない会社を支配するには、支配権は十分です。 また、P / Eについても忘れないでください。 P / Eレシオは、価格/収益レシオ（会社の時価総額から年間利益まで）です。 たとえば、月に3万ルーブル（これがあなたの資産です）に相当するアパートを1か月あたり3万で賃貸する場合、そのP / Eは8.3です。 そして、これは正常です。通常の会社では、PEは約10でハングします。会社のダイムラー（メルセデスはそうです）P / E 8.5。 アマゾンとリンクインのP / E比に注意してください。 これらの値は、これらの企業の株式の価値が非常に高いことを示しています。 GrouponはP / Eをまったく公開しません-高い値があります。
