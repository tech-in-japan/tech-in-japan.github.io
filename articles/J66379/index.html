Roverbook Neo U101ネットブックを確認する

たまたまネットブックのニッチが、製造業者でも顧客でもない誰かに悩まされているのは偶然です。 彼らは毎日何か新しいものを発表し、リリースし、改善し、発明します。しかし、常に「オーブンから」新しいアイテムを手に入れるとは限りません。 今日は、そこにあったものについてお話します。 常に新鮮なモデルが出会うわけではありません:)







だから、会う-部分的に国内のルーツを持つネットブック。 Roverbook Neo U101 ！ 詳細なレビュー、 トラフィック 。



このデバイスは、クラスとしてのネットブックのd明期に昨年発売されました。 したがって、ある程度、 Roverbook Neo U101はAsus EeePC 701の競合製品と見なすことができます。 もちろん、両方のモデルは時代遅れですが、魅力的な価格のために、それらのモデルでも多くのアプリケーションを見つけることができます。







Asusの競合モデルはネットワーク上で十分に詳細に説明されていたため、私のレビューにも頼ってこれら2つのモデルを比較できます。



ネットブックの心臓部は、クロック周波数が500 MHzの AMD Geode LX800プロセッサ です -内蔵グラフィックプロセッサ、128KBのL2キャッシュを備えています。 このマシンには、60GBのPATAハードドライブと512メガバイトのDDRメモリが搭載されています。 画像は、解像度1024x600（16：9）のマットな10.2インチTFTスクリーンに表示されます。 ネットブックにはASPLinux OSがプリインストールされていますが、提供されるディスクには、より使い慣れたシステムに切り替えるために必要なすべてのドライバーが含まれています。 興味深い機能のうち、組み込みのWi-FiモジュールとPCIMCIAスロットに注目できます。 さて、私は問題の本質を簡単に紹介したので、すべてが実際にどのようであるかを知ることを提案します。



パッケージバンドル



以下のアイテムは、持ち運びに便利なフォーム付きの便利な持ち運び用ハンドル付きの小さなカラフルな箱に隠されていました。

-Netbook Roverbook Neo U101 ;

-ACアダプターと電源コード。

-ロシア語の詳細なドキュメント。

-ソフトウェア付きディスク。







ドキュメントについては-もう一度注意します。 他の指示の例として、デバイス、使用の安全性、注意、問題の解決などに関するすべての必要な情報が含まれています。 用語集もあります！ 当然、これはすべて私たちの母国語です。

アダプターにも注意する必要があります-「マウスのような」形状と小さな緑色のLEDがあります。 彼は取り外し可能なフォークを持っています。これは、輸送中によりコンパクトなサイズにするために行われたようです。







外観



Neo U101ネットブックの外観は、数年前に誰かを驚かせる可能性がありました。 今日では、最も簡単に設計できるネットブックです。 原始的な角の形、明確なエッジ...しかし、全体的にそれはすべてかなり良いようです。

艶消し塗料は、印刷の影響を受けやすいが、影響を受けにくい。 重量1.2 kg（バッテリーを含む）、デバイスの寸法は（mm）260×195×33です。 とてもコンパクト。

上の表紙には、大きな銀の会社ロゴに加えて、何もありません。 反対側にも同様のミニマリズムが行き渡っています-バッテリーロック（Li-ポリマー、3800mAh容量、7.4V）、システムコンパートメント、および4つのゴム製脚以外には何もありません。







ミニマリズムの頂点はネットブックの裏側で、​​何もありません:)







左側には、電源コネクタとPCIMCIAスロットのみがあり、そこに多くの興味深いものを入れることができます。 これはネットブックへの非常に成功した追加であり、このためだけに結果に別のプラスを入れることができると信じています。

反対側の右側には、はるかに機能的な要素があります。 その中には、カードリーダー（MS / MMC / SD / SDHC）、2つのUSB（2.0）ポート、外部モニターを接続するためのVGAポート、ネットワーク100メガバイトRJ-45ホール、およびロックホールがあります。 それとは別に、内蔵Wi-Fiモジュールの存在と、USBポート間に十分な距離があるため、外部デバイスを接続する際の多くの不便を避けることができます。







前面には、ネットブックをオンにするためのボタン（左側にあります）と、ヘッドフォンとマイクを接続するための2つのオーディオソケットがあります。 さらに、6つの小さくて部分的に不必要なインジケータが表示されます。



ネットブックを開くと（これは150度以下で行われます）、多くの人がEeePCに期待するまさにその写真を見ることができます-マットスクリーン（10.2、1024×600）とその周りの薄いマットフレーム。 この場合、フレームはしっかりしています（ただし、金属ではないようです）ので、画面について心配することはできません。 優れたWebカメラがその上にあり、Skypeで合理的に通信できます。 内蔵スピーカー（画面の下）とマイクもあります。 ちなみに、スピーカーの音は非常に良いのですが、すべてのネットブックではヘッドフォンを挿入することを好みます。誰も箱に入れないのは奇妙です。







キーボード -それは... IBMスタイルの正方形ボタンです。 ラテン文字は白、ロシア語は黄色、Fn関数は青です。 印刷することはかなり可能です-レイアウトはかなり良いです。 しかし、不快な瞬間もあります。最も明らかなのは、左利きのCtrlとFnです。 これを思いついた人を見せてください）また、右のShiftとEndを交換し、大きなEnterを「小さな+スラッシュ」に置き換えます。 F1〜F12は、小さなギャップを分離するのにも適しています。







タッチパッドはマルチタッチではありませんが、最も一般的です。 全身と同じ素材から。 素晴らしいのは、デバイスのケースに押し込まれていないため、ほこりがケースとタッチパッドの間の隙間にしか入らないことです。 それを使用するのは非常に便利です。暗闇の中でのみ、他のケースと触覚的に同じであるため、「検出」するのはそれほど簡単ではありません。

小さい高さの2つの分割ボタンが存在する場合-クリックはソフトで静かで、非常に便利です。 しかし、それでも、サイズがわずかに不足する場合があります。



インクルージョンと仕事



ネットブックが異常にオンになります（私は画面下のボタンに慣れています）-「レバー」を左に動かします。 オペレーティングシステム（Windows XP）は約40〜45秒で起動しますが、これは非常に良好です。 オペレーティングシステムに関しては-最初はASPLinuxがありましたが、すぐに通常のOSに置き換えることが決定されました。







ネットワーク上で、ACアダプターを介して接続すると、ネットブックが「口histを吹く」という情報に出会いました。 この現象の私のコピーでは観察されませんでしたが、私が理解したこと-私は他のデバイスでこれに会った。 そのため、通常はデバイスは口whiを吹かないが、アダプター自体が鳴る。そのため、このような現象が発生した場合は、より高価な「稼ぎ手」を探す。



性能



このラップトップのこのパラメーターが両脚に足を引きずり込まなかったら、すべてが非常に良かったでしょう。より正確に言えば、これは合成計算テストで両脚にリンプしますが、日常の作業では非常によく機能します。 したがって、つまずきは中央処理装置であると結論付けることができます。

根拠がないようにするために、ここにオペレーティングシステムが起動するビデオがあります-ダウンロード速度が他のより強力なネットブックより劣っていないことがわかります。 ファイルマネージャー、インターネットサーフィンでの作業-これらはすべて、かなりうまく機能しています。 しかし、ここにテストがあります... :)







したがって、私が最初にしたことは、 SuperPi 1.5テストを実行することでした。これは、コンピューターに強制的にPi番号を希望の小数位まで読み取らせるプログラムです。 世界中で、より小さな値をすべて計算した後、このテストを百万位まで行うのが慣習です。







ご覧のように、結果は6分以上まったく輝きません-これは今まで見たことがありませんが、最新のネットブックでは平均で1.5分から2分ほどです。 ちなみに、現時点での世界記録は、同胞DeDaLから6.75秒です 。



ネットブックとラップトップのパフォーマンスを比較する次のテストは、プログラムCPUFreeBenchmark 2.2です。 Intel Atom 280プロセッサをベースにしたネットブックは60〜70秒でテストに合格しましたが、 U101の所要時間は... 263秒です。







富士通の内蔵MHW2060ATハードドライブも速度の完璧さによって区別されませんでした。HDTune3.50では、最大速度は27.2 Mb / s （平均22.2、最小4.4。
