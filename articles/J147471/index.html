Windowsストアに招待するか、Windows 8にアクセスする方法

先月はWindows 8に関するニュースでいっぱいです。6月の初めから、 Windows 8 Release Previewはロシアの最初のアプリケーションと共に公開され、新しいデバイスの発表が現れ、Windows XP、Windows Vista、Windows 7からWindows 8 Proへのアップグレードの可能性について知られるようになりました。



昨日、Windows 8（RTM）の最終バージョンが間もなく-8月の第1週にリリースされることが発表されました。

各Windows 8デバイスの内部には、Windowsストアという単一のアプリケーションストアがあります。 これは、すべてのデスクトップ、ラップトップ、タブレットのユーザーにアプリケーションを配信するための単一のチャネルです。

したがって、Windowsストアでアプリケーションを見つけると、Windowsユーザーのエコシステム全体に直接アクセスできます。

HTML5 / JavaScript、C＃/ XAML、またはC ++ / DirectXでWindows 8用のアプリケーションを開発できます。



これまで、Windowsストアへのアクセスは招待状でのみ可能です。 -最も興味深い高品質のアプリケーションのみがそれに該当します。 もちろん、時間の経過とともにすべての来訪者に開かれます。 ただし、現在、ユーザー、ジャーナリスト、デバイス購入者の最大の注目を集めた瞬間に、最初のアプリケーションの1つになる機会があります。



ロシアの開発者は、今日Windowsストアで招待を受ける機会があります。 興味深いアプリケーションがあり、Windows 8に参加したい場合は、「Windowsストアに招待」という件名のメールを書いてください 。

また、Habréに関する特別なセクションを作成しました。このセクションには、今後のイベント、リソースへのリンク、およびWindowsストアで最初のロシア語アプリケーションの例が表示されます。
