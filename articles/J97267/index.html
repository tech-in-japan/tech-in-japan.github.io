アメリカのゲーム市場の研究。 人口統計、販売、その他の興味深い事実

Entertainment Software Associationは 、アメリカのコンピューターおよびビデオゲーム市場に関する最近の研究を発表しました。 報告によると、ゲーマーの平均年齢は34歳です！











人口統計





  • ゲームバイヤーの平均年齢は40歳です。 54％が男性で、 46％が女性です
  • 2010年には、 50歳以上のプレイヤーの数は26％に達する
  • コンピューターまたはビデオゲームに費やされる平均年数は12です。 さらに、男性ではこの用語は13に達し、女性は10を超えません
  • 2010年には、 64％のプレイヤーが誰かと遊ぶことを好みます（2009年から62％、2008年から59％と比較して増加）


親の関与



  • 両親の93％がゲームを購入またはレンタルするときに同席しています
  • 両親の64％は 、ゲームが子供の生活にプラスの影響を与えると考えています。
  • 86％のケースで、子供は両親にゲームを購入したりレンタルしたりする前に許可を求めています
  • 両親の48％が少なくとも週に1回は子供と遊ぶ
  • 両親の76％が、現代のすべてのコンソールでペアレンタルコントロールを利用できると思っています。 さらに、親は他の種類のエンターテイメントよりも頻繁にゲームの時間制限を設定します。

    • 両親の83％がビデオゲームの時間制限を設定
    • 78％がテレビ視聴を制限
    • 75％がインターネットの使用を制限
    • 66％が映画鑑賞を制限




親が子供と遊ぶ4つの最も一般的な理由：

  1. 家族全員を満足させる-87％
  2. 子どもたちが彼らに尋ねたから-83％
  3. これはあなたの子供に近づく良い機会です-75％
  4. これはゲームのコンテンツをチェックする良い機会です-60％
