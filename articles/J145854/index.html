Canonicalは貴重な開発者コンテストを発表します

人気のあるUbuntuオペレーティングシステムの開発者であるCanonicalは、最近積極的にGNU / Linuxを一般に公開しようとしています。 Ubuntuはアプリケーションストアを作成し、インターフェースを簡素化し、ソーシャルネットワークとの相互作用を改善し、AndroidおよびiOSモバイルプラットフォームとシステムのクラウドベースの同期を確立し、EA GamesおよびHumble Bundleとストアを通じてゲームを配布することに合意しました（EAの場合、Crysisを待つべきではありませんニードフォースピード、私たちはブラウザーゲームについて話している）、彼らは開発者のための競争を発表しています。



コンテストのパートナーはノートブックメーカーのSystem76とQt Softwareであるため、非常においしい賞品が期待できます。 だから、私は楽しいから始めます：

審査員によって決定された勝者には、System76 Gazelle Professionalラップトップ（Intel Core i7 Ivy Bridge、15.6 "FullHD-display）とNokia N9スマートフォンが贈られます。2位の勝者には、System76 Lemur Ultraラップトップ（Intel Core i5 / i7そして、Nokia N9、およびフィニッシュラインの3番目に「来た」開発者は、Nokia N9 MeeGoスマートフォンのみの費用がかかります。 観客の同情の競争もあり、それに続いて3人の参加者がN9を受け取ります。 コンテストに参加するすべての人には、Ubuntu Tシャツが贈られます。

条件については、次のとおりです。

  • コンテストは2012年6月18日から7月9日まで開催されます（合計3週間なので、すぐに書く必要があります）
  • Quicklyユーティリティを使用する必要があります
  • アプリケーションはUbuntu Software Centerで公開する必要があります


すべての条件は、Ubuntu Webサイトのメッセージで詳細に公開されています 。
