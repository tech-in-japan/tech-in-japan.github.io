Windows 7をインストールする


AppleはコンピューターでBIOSの代わりにEFIを使用しているため、ハイブリッドGPT / MBRマークアップを作成する必要があります。

これは、Bootcampを使用して実行できます。



これを行うには、ウィザードを実行し、Windowsでパーティションのサイズを決定して、「パーティション」をクリックします。 データをバックアップすることを強くお勧めします。







ここで、システムにディスクを挿入し、インストールを続行します。

必要なセクションはBOOTCAMPとしてマークされています。



私の写真家は悪いです、ごめんなさい。







必要なドライバーはすべてMac OS Xドライブにあります。

ところで、パフォーマンスインデックス







ハードウェアに問題はなく、Windowsでも読み取りモードのMac OSパーティションを使用できます。



Ubuntu 10.04をインストールする


C Ubuntuはより複雑で、インストーラーはハイブリッドマークアップの操作方法を知らないため、OSXを再起動し、「ディスクユーティリティ」を実行してLinuxのパーティションを選択します。







Ubuntuでディスクを挿入し、コンピューターを再起動し、Optionキー（PCキーボードではAlt）を押し続けると、光学ドライブを選択する必要があるメニューが表示されます。



次に、LiveCDを使用してシステムをインストールします。 すべてが通常どおりです。手動パーティションを選択し、作成したFATパーティションをルートシステムに再フォーマットします。







重要：GRUBブートローダーはMBRに書き込まれるのではなく、ルートパーティションの先頭（たとえば、 / dev / sda3 ）に書き込まれる必要があります。







鉄の故障や問題は認められませんでした。

私が出会った唯一のものは、音の不足でした。

/etc/modules.d/alsa-base.confファイルに行オプションsnd-hda-intel model = mb5を追加することで処理されます



rEFIt


オペレーティングシステムを選択してメニューを開始するには、rEFItを設定します。

インストーラーを使用するか、手動でインストールできます。 詳細については、プロジェクトのドキュメントを参照してください。







外観は構成ファイル/efi/refit/refit.confで構成されます

例えば 

  hideuiバナーシェルツールラベルfuncs


不要なグラフィックを無効にします。
