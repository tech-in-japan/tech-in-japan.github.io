Googleブログで 、ストアにアプリケーションを追加するための次の条件を特定しました。

  1. 実際に、任意のプラットフォームで独自のビジネス指向のWebサービスを開発します。
  2. アプリケーションをGoogle Appsと統合する
    • OpenIDを使用したシングルサインオン（SSO）テクノロジーの追加（前提条件） -これにより、追加のログインとパスワードが不要になります。
    • Google Data API を介してGoogle Appsと対話するためのさまざまな関数とメソッドを追加し ます （これはオプションです） 。
    • アプリケーションのインストールプロセスとその後のアプリケーションのライフサイクル （インストール、構成、サポート、およびエントリのリンク）を記述した設定ファイルを作成し 、ストアの売り手のリストに追加するために100ドルの1回払いを支払います。
