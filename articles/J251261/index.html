ウィークリービルドVivaldi 1.0.111.2

みなさんこんにちは！



もう1週間が経過し、Vivaldiブラウザの次のテストビルド（番号1.0.111.2）のテストを従来から提供しています。 このバージョンの作業は同じ分野で行われました。ユーザーが要求する機能の実装、既存のブラウザー機能の改善、バグの修正です。 新製品から始めましょう。新製品は現在あまり多くありません（さらに多くは実装のプロセスにあります）。



したがって、このアセンブリから始めて、Windows用の64ビットバージョンのテストを提供します。 すべてのプラグインがまだサポートされているわけではなく、正常に動作するには64ビットバージョンが必要であることにすぐに警告します。



注目すべき革新には、新しいローカライズの追加が含まれます。 実際のところ、Chromiumは53ロケールしかサポートしておらず、既に80人の翻訳チームがブラウザーインターフェイスを33言語（エスペラントの後継であるIdoなどの排他的な言語も含む。このアセンブリはまだ利用できません）に翻訳しています。 ただし、Chromiumではいくつかの言語がサポートされていません。 そのため、独自のローカリゼーションシステムを実装しました。これにより、制限なしで任意の言語を追加できるようになりました。 新しいブラウザーアセンブリに追加されました：



  • ガリシア語（Chromiumではサポートされていません）
  • アイスランド語（Chromiumではサポートされていません）
  • マケドニア語（Chromiumではサポートされていません）
  • ベラルーシ語（Chromiumではサポートされていません）
  • アルメニア語（Chromiumではサポートされていません）
  • ブルガリア語
  • ハンガリー語
  • スロベニア語


このアセンブリでは、合計で29のロケールを使用できます（英語以外）。



ブックマークバーでは、前回のビルドで登場した多くの作業が行われました。 これで、名前だけを残して、ウェブサイトアイコンの表示をオフにできます。 パネルに多数のブックマークがあり、それらすべてが収まらない場合、それらの一部はドロップダウンリストで利用できます。



新しいアセンブリでは別の機能が改善されました-戻るボタンは、常に正しく動作しませんでした。



  • Windows 32ビット
  • Windows 64ビット
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット
