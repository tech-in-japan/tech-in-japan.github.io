ブロックチェーンガーデンの5つの石







この画像を使用して、今年の初めから暗号通貨レートが急落し始めた業界の多くの人々の気分を説明できます。 膨らんだバブルは、ブロックチェーンテクノロジーの強力な広告となり、隅々で使用すると脅迫されました。 1年ちょっと前に、それはガートナー曲線の一番上まで上昇しましたが、現時点では、私の気持ちによると、私たちは一番下にいます。



私たちの国（そして世界）でブロックチェーンを妨げるものは何ですか？それはまったく必要ですか？



1.乾燥残留物中



ICO、マイニングファーム、交換機を廃棄しても、ロシアの企業や国営企業からのニュースはまだほとんどありません。



  • M.Videoは、2017年の初めからファクタリング操作にスマートコントラクトを使用しており、ファクタリングト​​ランザクションの65％以上がブロックチェーンで実行されています。 過去1年間で、このテクノロジーを使用したトランザクションの数は4.7倍になり、支払い量は4.4倍になりました[1]。

  • ANSO「LEXAN」は、業界固有のデジタルシステムの開発と実装のための統一されたインフラストラクチャを作成します（通常、RSCAの奇妙で厄介な名前を使用）。 このシステムは、中央銀行と財務省との調整の段階にあります[2]。

  • 9月、RostecとVostokプロジェクトは、デジタル経済のインフラストラクチャーにおけるブロックチェーンの協力と共同実装に関する合意書に署名しました。 彼らは協力のためのプロジェクトオフィスを設立し、この約200万ドルに投資します[3]。

  • 今年の7月、VEBブロックチェーンコンピテンシーおよびデジタルトランスフォーメーションセンターは、グロズヌイ市庁舎とともに、住宅および公共サービスの管理と地方自治体の賃貸に関する2つのパイロットプロジェクトを開始しました[4]。

  • 3月、LANITグループは、デジタルエコシステムのインテグレーターであるDigital Transformation Groupの設立を発表しました。そのタスクは、ブロックチェーンテクノロジーに基づくものを含む新しいビジネスモデルを作成することです[5]。

  • National Technology InitiativeにはFinNetの方向性があります。 [6]分散型の金融システムと個人向けのネットワーク金融サービスを扱っています。ここでは、ブロックチェーンが、既存のCoinbase、Bitfinex、Binance、Poloniexなどに類似した暗号通貨と交換を作成するための基礎と見なされることは明らかです。



この種類の他の例があります。 ただし、最終的な実装はまだ少ないため、意図について詳しく説明しています。



さらに、ブロックチェーンは依然として実装が難しい技術であるという評判があり、実際、ロシアの企業はいずれも継続的にそれを使用していません。 市場全体を取り上げても、実際に稼働している需要のあるプロジェクトは、ビットコインとイーサリアムの2つだけです。



そして、ここで疑問が生じます。何百、何千ものプロジェクトの開発を妨げるものは何ですか？ したがって、このテクノロジーを採用する際の障壁については引き続き説明します。 そして、私の意見では、この場合、利点よりもさらに多くの障壁があります。 少なくとも今のところ。



2.ヒューストン、問題があります



2.1。 低い安定性とスケーラビリティ



ブロックチェーンプロジェクトでは、「1秒あたりの処理済みトランザクション数」などの特性がよく言及されます。 この数は、IOTAやEOSなどの新しいプロジェクトのビットコイン単位から（ステートメントによる）数千単位まで大きく異なります。 低速でありながら信頼性の高いPoWと、高速であるが集中型（D）のPoS、および他の多くのコンセンサスアルゴリズムのバランスを求めています。



ただし、これまでの2つの最大のプロジェクトであるビットコインとイーサリアムでは、安定性とスケーラビリティの問題はまだ解決されていません。 公衆からの関心が急激に高まった瞬間（ICO、暗号学、コースでの推測）、取引は数時間確認されるか、まったく送信されないことさえあります。



2.2。 ハイエンドユーザーのログインのしきい値



確かに、ブロックチェーンは依然として多くのオタクや暗号アナキストです。 一般の人々にとって、これはまだニュースで何かが書かれている未知の獣です。 ビットコインの例でブロックチェーンを母親に説明するのに数週間かかりました。



インターネットには多くの情報があるように見えるかもしれませんが、これは逆の効果をもたらす可能性が高く、本当に有用で理解可能な記事やビデオを見つけることは簡単ではありません。 ゼロに近いレベルの知識を持つ人は、どのようにしてイーサリアムの使用を開始できますか？ 現在使用できる便利なdappを理解するにはどうすればよいですか？ そのような情報はまだありません。 これは現在、World Wide Webの初期段階に似ており、多くの異なるWebサイトが存在していましたが、これらすべてをどのように扱うかは明確ではありません。



2.3。 トランザクションの不可逆性



はい、通常、技術の利点であると言われている特性が障壁になっています。 住所を入力してトランザクションを作成するときに、人的要因とエラーの可能性をキャンセルする人はいません。 これがユーザーの過失によって発生しなかったとしても、悪意のあるハッカーの過失によって発生する可能性があり、そのハッカーがいずれかのレベルのアドレスを置き換えます。 そのような間違いの1つは非常に高価になる可能性があります。



銀行振込では、取引に異議を申し立てることができますが、このようなことはありません。 多くの人を怖がらせます。 この段階でブロックチェーンを使用するには、何をしているのか、何が起こるのか、それから自分を守る方法を明確に理解する必要があります。



2.4。 アルゴリズムの知識が不十分



最も有名なProof of Workコンセンサスアルゴリズムの「インザワイルド」テストの開始から9年以上が経過しました。 そのパフォーマンスは実際に確認されていますが、アルゴリズム自体には多くの欠点があります。 そのため、コミュニティはすでに数十の[7]同様のアルゴリズムを考案しています。



  • ステークの証明

  • 委任された証拠。

  • 権限の証明;

  • 委任ビザンチンフォールトトレランス。

  • 重要性の証明

  • ...

  • 何でも証明。



それらの多くは、何らかの健全なテストは言うまでもなく、数年前のものではないようです。



潜在的に脆弱なコードの別の巨大な層は、イーサリアムのスマートコントラクトです。 ここでの間違いは、非常に高くなる可能性があります[8]とても[9]高価です。 スマートコントラクトの技術の新規性により、コード監査は初めて非常に表面的に実行された、またはまったく実行されませんでした。



また、完全に新しいTCRテクノロジーについても言及する価値があります。これについては、Habré[10]の元の投稿に、アルゴリズムの最初の多かれ少なかれ詳細な説明が翻訳されています。 投稿へのコメントで合理的な発言を聞くことができますが、これは一般的にどこに適用され、なぜですか？ これが必要かどうか、そしてそれがどれほど役立つかはまだ分からない。



2.5。 専門家の不足



IBMバリューフォーインスティチュートオブガバメント「政府の信頼の構築」による調査[11]によると、公共部門にブロックチェーンを導入する技術的問題は、技術を扱うスキルが不十分である （この分野の有能な専門家が不足していることを考慮してください）および実装されたビジネスケースがないことです。 最後の声明で、私たちは間違いなく議論することができます-しかし、ロシアではなく公共部門でブロックチェーンを実装する例がすでにあるため、不十分な専門知識について話している可能性が高くなります。 たとえば、2016年4月、Bitfuryは、Exonumフレームワークを使用して不動産レジストリをブロックチェーンに移すためのパイロットプロジェクトを開始するために、ジョージア州公的レジストリ局と契約を結びました。 このシステムは正常に機能しており、現在[12]、2017年にパイロット自身がハーバードビジネススクールのケースコレクションに[13]を入力しました。 これはむしろ例外です。



ただし、多くの優秀な開発者がブロックチェーンに来て、古い技術に飽き飽きしていて、新しいことに挑戦したいということを否定しません。 しかし、ご存知のように、多くのプログラマーはいません。



3.プラスできるもの



投資家からの関心。 IDCのアナリスト会社は、2022年までにブロックチェーン市場に約117億ドルの世界的な投資を行う[14]と予測しています。 将来の20年間で、これらの数値は10倍以上増加する可能性があります。



受け入れられた標準の欠如、それらの開かれた形成の可能性。 技術の国境を越えた性質、世界中の講演者が参加するさまざまな国で開催される膨大な数の会議とハッカソン[15]を考えると、話し合い、経験を共有し、共同で意思決定をしたい場合、技術基準を議論し、変更できることは明らかです。



インフラストラクチャのメンテナンスコストを削減します。 すでに、他のユーザーとデバイスの空き領域またはコンピューティングリソースを共有できるプロジェクトがあります。 私たちはそれぞれ、99％の時間アイドル状態にあるプロセッサを備えたスマートフォンを所有しており、この計算能力はすべて使い果たしてしまう可能性があります。 （いいえ、ビットコインマイニングではありません）。 将来は、ポケットやデスクトップでこれらすべての巨大なコンピューティングリソースを経済的かつ公正に使用することにあると思います。



情報システムの新しいレベルの安定性とアクセシビリティ 。 実践が示すように、障害点が唯一のサーバーである場合、特定のサイトへのアクセスを失うのは簡単です。 新しい分散型パラダイムでは、システムへのアクセスは、システム内の任意のノードから取得できます。



4.結論



現在、ブロックチェーンを適用しようとしている膨大な数のプロジェクトを目撃しています。 これがトレースなしで通過すると想定するのは愚かです。 私たちは、新しい「キラーアプリ」（何ができると思いますか）、かつてはNetscapeブラウザーになったのか、Facebookがソーシャルインタラクションの仕組みを永遠に変えたのを待っています。 おそらくこれは、ブロックチェーン、AI、ある種のVRなど、いくつかの新しいテクノロジーを一度に統合した製品になるでしょう。 価値の概念を再考する必要があるかもしれません。一部の暗号アナキストは、個人の暗号通貨の出現について幻覚をすでに表明しています。 各人が自分の通貨を所有している場合、その人の社会的格付け（中国にこんにちは）に依存する、彼自身の価値相当物。



Web 3.0の概念はまだ初期段階ですが、新しいインターネットは価値のあるインターネットであるという意見が増えています。



Web 1.0はパブリッシングに関するもので、誰でも作成してグローバルに配布できます。

Web 2.0は相互作用に関するものでした-他の人とつながり、リアルタイムで共同作業し、データを。

Web 3.0は価値の移転に関するものです-お金をプログラムして、どのアプリに埋め込むこともできます。





-ブライアン・アームストロング（@brian_armstrong） 2018年8月24日
