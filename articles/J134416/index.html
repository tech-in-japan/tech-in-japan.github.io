私は、最近購入したデバイスの興奮を共有せざるを得ません。



ルーターは、そのサイズにもかかわらず、多くの機能を備えています。 標準のイーサネット-> Wi-Fiに加えて、Wi-Fiクライアントとして機能し、モデムとスマートフォンを接続するためのUSBホストを持ち、異なる設定で複数のWi-Fiネットワークを同時に作成できます。 USBまたはアダプターから給電。



有線WANモードでは、すべてが標準です：静的IP、DHCP、PPPoE、PPTPおよびL2TP。 最大4つの独立したWi-Fiを同時に作成でき、ネットワークごとに異なる暗号化パラメーターとセキュリティレベルを設定できます。 一部のゲストにWEPを使用してインターネットへのアクセスを許可し、WPA2を使用してLANに接続する必要がない場合に便利です。







上部のUSBポートは、モデムを接続するためのものです。 GPRS / 3G / 4G USBモデムの250以上のモデルをサポートしてい ます 。 完全なリストはこちらです。 私はいつもラップトップから飛び出し、すべてのスタブにぶつかることに悩まされていましたが、今では便利さを喜んでいます-それはただの至福です！







ルーターはwifiプロキシとして機能できます。



空港、ホテル、カフェなどの有料アクセスポイントの場合、MACアドレスやその他の制限によるフィルタリングが存在する場合に便利です。 ルーターはWi-Fiクライアントとして機能し、Wi-Fiネットワークを介してインターネットを配信します。 また、クライアントのWi-Fiモードでは、有線デバイスをワイヤレスネットワークに接続すると便利です。 Wi-Fiを介して2つのネットワークを接続する透過的なWDSブリッジになることができます。







スマートフォンへの有線USB接続を介してインターネットを配布し、同時に充電することができます。

サポートされているモデルは、iPhone 3GS / 4、Google Android 1.5 / 2.1 / 2.2、Windows Mobile 6.0 / 6.1 / 6.5です。
