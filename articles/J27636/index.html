5月23日にモスクワで開催されたReMIXカンファレンスでのSteve Ballmerのスピーチのテキスト

ReMIX、モスクワ、2008年5月23日

スティーブバルマー、マイクロソフトCEO

ここは最高です！ ここに来てくれたみんなに感謝します。 本日ここでお会いできて非常に嬉しく思います。 質問と回答に進む前に、少し言葉を言いたいです。 質問と回答はもっと刺激的な部分になると思いますが、その前に何かお話ししたいと思います。 モスクワでは、非常に忙しい仕事のスケジュールがありますが、世界中のソフトウェア製品とアプリケーションの開発に携わる人々が私の心の中で特別な場所を占めていることを確信できます（そして聴衆の中にこれをよく知っている人々がいます）。 ですから、今日、ソフトウェア開発者と話ができることを特に嬉しく思います。 皆さんにお会いできてとても嬉しいです！





マイクロソフトはソフトウェア会社として発展してきました。 マイクロソフトは文字通り「マイクロコンピューターソフトウェア」を意味します。 マイクロソフトの将来のビジョンの基礎は何でしたか？ 最も重要なことは、ソフトウェアがビジネスになり、ソフトウェアが私たちの世界を変えることです。 そして今日、2008年には、ソフトウェアの本質が何らかの形で変化しています。 ソフトウェアが非常に静的なものであった時代は終わりです。 DVDに書き込むことができます。 ウェブサイトでホストされることもありますが、基本的には不変で静的なものでした。 私たちがソフトウェア+サービスと呼ぶ新しい時代が始まりました。 この新しい世界では、ソフトウェアはインターネット、PC、電話、企業のデータセンター、テレビにあります。 同時に、ソフトウェア製品のアプリケーションはますます素晴らしいものになっています。

私は、パソコンの進化の物語を語るのが大好きです。 パーソナルコンピュータが最初に登場したとき、それはプログラミングマシンでした。 その後、スプレッドシートマシンに変わり、ワープロマシンになりました。 さらに、彼は電子メールに使用され始めました。 そしてゲーム用。 そして音楽のために。 そして、ビデオを見ます。 その本質とコンピューターでできることの考え方は変わり続けています。 すべてが可能になります。これは、ソフトウェア開発者の想像力に十分です。 世界のソフトウェアの進化に伴い、「ソフトウェア+サービス」もソフトウェア製品の作成に使用されるアプリケーションの種類を進化させています。 コミュニケーション、コンテンツ、広告、商取引、コミュニティを通じたコミュニケーションの手段-これらはすべて組み合わされ、相互に作用します。 コミュニティ、情報、商取引の融合は本当に素晴らしいです。

これは新世代の現象です。 私はこの世代の現在と展望に非常に触発されています。 今後10年間の業界の発展レベル、変化の速さ、魅力、革新は、過去に起こったことに匹敵するものではないと思います。 私は人々に次のように伝えたいのです。「10年後には家にいます。 おそらくそれまでに、マイクロソフトでパートタイムで働いているでしょう。 ビル・ゲイツは今年パートタイムで参加します。 私にはさらに10年の素晴らしい年があります。 10年後、私は座ってテレビで私のお気に入りのゴルファーであるタイガー・ウッズを見るかもしれません。 そして、私はテレビに叫びます：「ねえビル、タイガーが穴にボールを入れるのを見ましたか？」 同時に、人間の音声を理解する私のテレビに統合されたインテリジェントシステムは即座に応答します。 ビル・ゲイツ！ ビル・ゲイツは今どこにいるのか？」 ビルがスティーブの私生活への侵入に反対していないとします。 システムはビルがどこにいても見つけることができます：モスクワのホテルの部屋か、世界中を旅するどこかです。 そしてすぐに、彼のテレビが切り替わります。「ビル、タイガーがボールを穴に押し込むのを見ましたか？」 ビルは今見たものを正確に見て、「スティーブ、この新しいタイガーゴルフボールとは何ですか？」 私はリモコンを取り、それをゴルフボールに向けると、画像検索はインターネット全体を調べます。 その後、彼は戻って言います。「スティーブ、これはナイキエリートゴルフボールです。 これらのうち12個を購入しますか？」 「多分、24：12が私にとって、12がBillの方が良いでしょう。」

コンテンツ、コマース、コミュニティ。 相互作用する方法。 コミュニケーションの方法。 このプロセスは、ソフトウェアの革新とソフトウェア+サービスの新時代によって推進されています。 インターネット、PC、企業のデータセンター、電話、テレビでの最高のアプローチとマージ。

これらは、ソフトウェア製品の4つの異なる世界ではなく、ソフトウェアの単一の世界、単一の統合アーキテクチャです。 今日、人々はインターネットが大好きです。 ユーザーはマウスをクリックするだけですべてが完了します。 しかし、人々はまた、豊富な相互運用性を備えたパソコンを愛しています。 近い将来、インテリジェントシステムはスマートデバイスとインターネットクラウドの両方で機能します。 また、ユーザーの介入や管理に関わる労力を必要とせずに、自動的に相互に「連携」して対話します。

マイクロソフトは、すべてのエネルギーと魂をこの未来のビジョン、この変革プロセス、ソフトウェア+サービスの概念に投入しています。 かつて、PC用のWindows、Windows Server、モバイルデバイスやテレビ用のWindows CEを作成していたので、ある意味で新しいプラットフォームを構築する必要があると考えています。 コマース、コンテンツ、コミュニティ、ソフトウェア+サービスをサポートするために、クラウドで新しいプラットフォームを作成しています。 これをWindows Liveと呼びました。 すでにいくつかの断片や断片、および技術の予備的なデモンストレーションを見てきました。 これらは、SilverlightやWindows Presentation Foundationなどのテクノロジーです。 .NETプラットフォームなど。 そして今では、Windows Live Meshなどのテクノロジもあり、プロジェクトの機能のデモが行われています。 Windows Live Meshは、ネットワーク上のデバイスと情報の同期と調整を提供します。

今年中に、インターネット用のオペレーティングシステムまたはクラウド内のオペレーティングシステムの多くの機能を紹介します。 これらは、ストレージおよびコンピューティングシステム、高レベルのディレクトリサービス、プロセス間通信サービス、認証サービス、そしてもちろんアプリケーション自体になります。 インターネットアプリケーションを作成して展開する場合、今日は何をしていますか？ あなたはあなた自身のサーバー、おそらくあなた自身のホストを入手し、それを維持し、保護し、サポートする必要があります。 このプロセスを簡単にしたいです。 開発ツールで作業するだけです。 アプリケーションを作成しており、インターネットクラウドでのアプリケーションの展開、キャッシュ、管理は自動的に行われます。 これはすべて、開発者が「ソフトウェア+サービス」の原則に基づいて新世代のアプリケーションを作成できるサービスによって行われます。

今日、私たちの業界ですでに行われている、素晴らしい成果を上げたステップは、企業製品のインターネットへの実際の移行です。 開発者として、さらなる開発のためには、さまざまなデバイスを真に統合し、開発者に強力なサービスを提供する新世代のプラットフォームを作成する必要があると考えています。 「ソフトウェア+サービス」の生成で実際に発生する現象について、できる限り簡潔に説明しました。 ソフトウェア開発者はどのようにしてお金を稼ぎますか？ バッチ製品を作成していますか？ 広告と引き換えにソフトウェアを販売していますか？ 取引に料金はかかりますか？ 有料のニュースレターはありますか？ ソフトウェア開発者が利用できるすべてのビジネスモデルは、ビジネスを構築し、企業を開き、起業家精神を成功させるための途方もない新しい機会を発展させ、創造します。 マイクロソフトは、インフラストラクチャ、トレーニング、ツール、およびサポートを提供して、お客様が成功を収めることを計画しています。

今日ここに来られてうれしいです。 また、すべてのWebスタジオのすべてのロシアの開発者にWindows Server 2008 Web Serverの無料コピーを提供することを発表できることを非常に嬉しく思います。 新しい部屋、つまりソフトウェアとサービスの世界への移行を支援したいので、このすべてはこの部屋の人々が利用できます。 いくつかの紹介的な言葉を言う機会をありがとう。 アントンは、私たちの魅力的なコミュニケーションとQ＆Aパートに参加する機会を待ち望んでいることを知っています。 それで、これで私の短いスピーチを締めくくります。 まだ答えられない質問がある場合は、必ず質問を電子メール（ steveb@microsoft.com）で送信してください。 あなたの意見は私にとって非常に重要です。 ありがとうございます！
