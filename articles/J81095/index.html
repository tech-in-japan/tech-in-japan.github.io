キエフでの地図製作会議







2010年1月23日 13:00に、Taras Shevchenko Boulevardのカフェ「Khudgraf」で33、Googleテクノロジーユーザーのキエフグループ（Kyiv GTUG）の次の会議が開催され、地図とGoogle Maps APIに専念します。

プレゼンテーションは以下によって行われます。

  • Googleマップメーカーラテンアメリカコンテストの優勝チームのキャプテン、アントン・ベレズニー
  • Maplos.com開発者のVladimir Bilonenko
  • Anton LinevichとSergey Polzunov、 Flatfinder.com.uaの開発者
  • Glaya Security Group 、wLBSテクノロジーの開発者、Ilya Muzychuk
  • Igor Shubovich、 OpenStreetMapの愛好家であるブログMapping.in.uaの著者。
