ロシア語を話すコミュニティソフトウェアクラフツマンシップ

こんにちは、新しいロシアのソフトウェアクラフトマンシップコミュニティについてお話したいと思います。 プログラミングアプローチの支持者であるMotherfuckerである場合、この投稿は明確にあなたの不承認を引き起こし、恐らく憎しみの波を引き起こします。 Software Craftsmanshipとは何か、有用なコードを記述し、柔軟な方法論のフレームワークでエンジニアになる方法に興味がある場合、または長い間苦しんでいる問題がある場合は、ここにいます。



Software Craftsmanshipは、エンジニアからアジャイル手法をサポートするためのアジャイル手法の出現に対する開発者の反応です。 Software Craftsmanship Manifestoには次のように記載されています。



  • 動作するソフトウェアだけでなく、巧妙に作成されたソフトウェア
  • 変化に対応するだけでなく、着実に価値を付加する
  • 個人や交流だけでなく、専門家のコミュニティも
  • 顧客コラボレーションだけでなく、生産的なパートナーシップも




したがって、私たちのコミュニティは、このアプローチが好きで、経験を共有し、その中で発展したい人たちです。 当サイトでは、記事を公開し、会議を開催しています。



私たちの会議の例は、Code＆Coffeeです。 Code＆Coffeeは非公式の朝の会議で、コーヒーを飲み、コードを書き、仕事が引きずられるまで通信します。 厳密な規則や要件はありません-経験を共有するために新しいことを学びたい、またはその逆をしたいという願望だけです。 これまで、2つの会議を開催しました-1つはFacebook APIを学ぶため、もう1つはマルチスレッドアプリケーションのテストに対処するためです。 これらの会議に関するすべての情報をFacebookのグループに公開しています。



別のタイプの遭遇はXPバトルです。 XP Battleは、特定のエンジニアリングプラクティスの開発またはツールの実用例に特化した無料のイベントです。 私たちの聴衆は、ソフトウェアエンジニアと呼ばれる人々です。 なぜこれが必要なのですか？ ロシア語を話すエンジニアリングコミュニティの成長と発展を望んでいます。 そのため、よく知られているエンジニアリング手法（TDD、AFT、その他の恐ろしい言葉）がますます多くの人々に使用され始め、新しい手法が研究と仕事での応用のために幅広い聴衆に利用可能になります。

イベントの形式は非常にシンプルです。最大30人の参加者と最大15人のファンを招待します。 すべての参加者は連携して作業します-テスター+開発者。 イベントは4時間にわたって行われ、その間にすべての参加者がタスクを完了し、成果を受け取り、TDDなどの選択されたプラクティスを練習する必要があります。 作業は短い反復で行われます：導入-開発-報告。



また、Facebookページを立ち上げ、そこにニュース、興味深いこと、その他の有用なことを投稿しました。 このすべてに興味があれば、ようこそ！
