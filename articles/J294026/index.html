Reddit：使用説明書



ソーシャルネットワークとニュースポータルの特徴を組み合わせたいわゆるソーシャルニュースサイトであるReddit.comは、目まいがするトラフィックと完全なアクセス性により、過去数年にわたって世界中のSEO専門家の注目を集めています。 ロシア語を話すユーザーには近寄らない理由がないので、プロモーションのためにこのサイトを最も効率的に使用するために、小さなマスタークラスに全員を招待します。





まず、このサイトの運用原則の概要を説明します。



大規模で人気のある比較的オープンなプラットフォームと同様に、redditは、スパマー、SEOスペシャリスト、外部リンクの生成、および幅広い視聴者に緊急に何かを表示する必要のある他のエンティティの純粋な関心を引き付ける運命にありました。 通常、この問題は2つの方法でサイトで解決できます。1つ目は、登録された訪問者を「入り口」で厳密にフィルタリングすることです（スパマーのデータベースのIP、一般的な博学のための巧妙なキャプチャ）、2つ目は、主要な顔を通過した人も同様に厳しいモデレーションです-コントロール。 reddit政権はより機知に富んでいます。 彼女は実際に、ユーザーがリソースに一定量の広告を掲載するという事実に辞任し、すべてが少なくともその条件で行われるように、ユーザーと契約を締結しました。 あなたのページに人々を引き付けるためにサイトを使いたいですか？ 良いが、10倍のサイズのテーマ別投稿で広告を補うほど親切にしてください。 多くの人は、多くのフォーラムで機能していたスパム対策システムを覚えているでしょう。新しいメンバーがモデレーターによって10件の投稿が承認された後にのみ、投稿へのリンクを追加できました。 実際、Redditは継続的にのみ同じ原則に基づいて動作します。



ただし、この方法は、コンテンツの品質を保証することなく、数量に対して機能します。 後者を制御するために、redditはコミュニティにある程度の自主規制を提供します。 追加のルールを導入し、違反者の投稿を削除する権限を持つ個々のサブレディットのモデレーターに加えて、多くの一般ユーザーがいて、それぞれが投稿に「賛成」または「反対」することができます。 合計すると、投稿されたすべてのコンテンツの投票数は、編集者のいわゆる「カルマ」であり、編集者に制限が課されるかどうかを決定します。 そのため、共有する素材が特定のサブにあまり熱意を抱いていない場合は、次回試してみると、しばらく投稿を控えなければならないという丁寧な通知が表示されることがあります。



つまり、redditで安定したトラフィックを得るには、多くのコンテンツを生成し、コミュニティにある程度の重みを持たせる必要があります。 これにはすべて、時間の投資と、もちろん、機械的な自動アルゴリズムではなく、多少なりとも意味のあるアプローチが必要です。 reddit保護システムは完全ではありませんが、それは何かであり、問​​題なくロボットロボットを識別して無効にします。 したがって、戦略が大規模さと低労力に焦点を合わせている場合は、開始しない方が良いでしょう。 しかし、結果に取り組む準備ができているなら、どうやってビジネスを始めるのでしょうか？



私は長い間reddit.comに落ち着き、自分のプロモーション方法を試してみて、他の人が実際に動いているのを見る機会がありました。 無料プロモーションについて話す場合、ほとんどの人は2つの主要なアプローチのいずれかを選択します。



最初のものは比較的安全ですが、忍耐が必要であり、すぐに結果を出しません。 これは、主に自分自身の名前を作成したい人や、トラフィックを引き付けることにほとんど関心がない人に適しています。 彼らの戦略は、製品のメーカーまたはオープンディストリビューターとしての地位に基づいています。 すぐにカードを開くことは悪い考えだと思う人もいるかもしれませんが、そうではありません。通常は、そのようなユーザーにカルマをプラスするだけです。 redditの大衆は誠実さを好みます。 これらは、長年インターネットをサーフィンしてきた広告主が彼らを潜在的な消費者と見ているという事実に慣れており、そのような場合に使用される通常のトリックを認識することを長い間学びました。 したがって、管理者のポリシーに完全に準拠して、自分を偽装して意図を隠そうとしない広告主を受け入れる可能性が高くなりますが、もちろん、これらの目的のために特別に作成されたプラットフォームの境界内にいることを条件にしています。



たとえば、アクティビティの種類（モバイルアプリケーションの開発）で最もよく知っているアクティビティの範囲を見てみましょう。 Redditには多数のgamedevコミュニティがあり、開発者が集まってプロジェクトについて話し合い、助けを求め、製品の製造方法を段階的に世界に示しています（/ r / GameDev、/ r / playmygame、/ r / indiegaming、/ r / devblogs）あなたは本当に高品質で長時間のプロジェクトであり、興味を引き起こす可能性があります。これらの潜水艦は発売前に忠実な視聴者を獲得する良い機会です。 開発やティーザーを見せ、議論に積極的に参加し、アドバイスを与え、時々自分で尋ねることを忘れないでください。 要するに、9つのフィラーを使用して、開発者としての自分に注意を引き付けてください。そうすると、人々は10番目の広告投稿の影響を受けやすくなります。 多くの開発者は、ニュースと詳細な進行中のブログを作成し、ユーザーをリダイレクトする投稿を投稿するだけです。 これは理にかなっています。リダイレクタは一般にテキストの段落よりもリンクへのリンクを好む傾向があります。特に、画像は後者に埋め込むことができないためです。 潜水艦は、リンクの投稿が疑わしい議論に特に焦点を当てたものであります。 これはまた、機械的なアプローチの容認できないことについての言葉でもあります-特定の各修道院の憲章にコンテンツを適合させる準備をしてください。



この方法は時間がかかりますが、高品質のトラフィックを保証し、さらに価値のある製品を認識できるようにします。 もちろん、この状況では、あなたのアカウントは、たとえば「ブランド」になり、より積極的な別のプロモーション方法に切り替えることができなくなります。これにより、新しいファンが追い出され、画像に深刻なダメージを与える可能性があります。



では、退任する開発者の役割を果たす準備ができていない場合はどうでしょうか？ トラフィックにできるだけ早く興味がある場合はどうしますか？ 次に、プランBの時が来ました。これは、通常のユーザーに代わってプロモーションを行うことを意味します-ゲームはよりリスクが高く、やや複雑ではありません。



まず第一に、私たちは見通しを概説する必要があると思います。 Redditは、非常に印象的な結果を即座に生成するという点で欺is的です-興味深い見出しを付けたテスト投稿では、数千クリックではないにしても、数百クリックを収集できます。 このような効果により、初心者は自分が金鉱山を攻撃したと考えさせられ、この注文の毎日の増加を期待できます。 すぐにこのレベルの出席を維持することはほとんど不可能であることをすぐに言います。 あなたは、特に人々の流入が必要な場合にそれらを保存するか、聴衆の注意を誤用することに慣れるにつれて出席が徐々に減少するのを見るか、ポイントバーストに満足する必要があります。



すでに述べたように、redditorは熟練した目を持つ経験豊富なユーザーです。 偽装された「行動を促すフレーズ」でさえ認識できます。 したがって、あなたがそれらを提供できる最悪のことは、強迫観念であり、電波を詰まらせることです。 あなたがそれを最大限に活用したい場合は、大衆のためではなく、報道の幅のために努力してください。

それで、技術的な側面は終わりました、それは歌詞をする時間です、すなわち、満たすこと。 もちろん、最大限の効果を目指している場合、投稿は均一にコンパイルされませんが、特定のコミュニティの詳細を考慮します。 最も簡単な方法は、最初はアプリケーションの宣伝のために使用されていた2番目と5番目のグループのサブウーファーを使用することです。 そこでは、確立されたパターンに従い、必要なすべてのチラシ、特性、メモを含め、頭脳の利点を正しく要約するだけで十分です。 最も難しいのは、3番目と4番目のカテゴリのサブウーファーです。このサブウーファーでは、ターゲットオーディエンスであることを1行で説得するという難しいタスクがあります。 昇進のための潜水艦では、あなたは最大の無関心のリスクにさらされています。ここで、危険は特にうっとうしく、退屈し、あるいは禁止をつかむように思えます。 あなたのすべての創造性、ポップカルチャーの知識（編集者は送信が大好きです）、そしてユーモアのセンスを助けてください。 また、参照によって見つけたものについて読者を誤解させようとするような安っぽいトリックに決して頼らないでください-これはredditでは許されません。
