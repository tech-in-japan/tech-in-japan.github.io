Windows 7のヒントとコツ;）

ハブラリュディに敬礼



Windows 7に特化した記事のルーブリックを続けて 、このOSでの作業を促進できるいくつかの有用性のコレクションを提供することにしました。







それでは始めましょう



あなたがあなたの裁量で適用するこの記事のヒントなど！

豪華なWindows 7インターフェースをだめにすることを誰にも強制しません





1クイック起動パネルを正しい場所に戻します：

  • タスクバー（TaskBar）=>ツールバー（ツールバー）=>新規パネル（新規ツールバー）を右クリックします。
  • アドレス行で、「フォルダ選択ダイアログ」ウィンドウを開いて、次の値を入力します。



    %userprofile%\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch



  • しばらくの間、「タスクバーのロック」（タスクバーのロック）を無効にし、表示されるセパレータを右クリックします。 [タイトルを表示]チェックマークがオフになっていること、および[小さいアイコン]値がオンになっていることを確認します。
  • セパレータを使用して、必要なだけQuickLaunchを押します。 これらの操作の後、タスクバーの再ロック（タスクバーのロック）を忘れないでください


2 Windows Vistaのスタイルである「タスクバー」のスタイルを変更します

タスクバーを右クリックして、プロパティに移動します。

「小さなアイコンを使用する」を選択します。

次に、[タスクバーアイコン]ドロップダウンメニューで、[タスクバーがいっぱいになったら結合する]を選択します（原則、これは任意です）

そこで、太くて太い標準のタスクバーを取り除きました（個人的には嬉しかったです）



3デスクトップ上のアイコンのサイズを変更します（これらの大きなアイコンは私を殺しました）

これを行うには、デスクトップ上の空のスペースでマウスを右クリックし、次を選択します。表示=>小さなアイコン（表示=>小さなアイコン）



最初の3つの操作の後、Windowsは次のようになります。





4フォルダーのコンテキストメニューからコマンドラインを開きます

この機能について知っている人はほとんどいません。 彼女はVistaにいます。

フォルダを右クリックします-すべてが標準ですよね？

次に、Shiftキーを押しながらO_oフォルダーを右クリックします。

出来上がり、2つの新しい値があります。新しいプロセスで開くと、ここでコマンドを開く



5迷惑なUACユーザーアカウント制御を無効にします（ご自身の責任とリスクで:)）

コントロールパネルに移動し、アクションセンター（アクションセンター）を設定し、メニューの右側で[ユーザーアカウント制御設定の変更]を選択します。 表示される新しいウィンドウで、スライダーを一番下まで下げて、「Never Notify」という値まで下げます。



6エクスプローラーを開始点から、ライブラリではなく「マイコンピューター」「コンピューター」に開きます。

これを行うには、以下を実行する必要があります。

[スタート]メニューを開き、[アクセサリ]に移動すると、Windowsエクスプローラーアイコンが見つかります。

アイコンを右クリックして、プロパティに移動します。 ターゲットで、次のように記述します。



%SystemRoot%\explorer.exe /root,::{20D04FE0-3AEA-1069-A2D8-08002B30309D}







タスクバーのエクスプローラーアイコンをライブラリではなくマイコンピューターでも終了するには、タスクバーから削除し、右クリックして固定解除する必要があります。 アクセサリに戻ると、おなじみのアイコンが見つかったので、それを右クリックして[タスクバーに固定]を選択します。

キーの組み合わせWin + Eは、「コンピューター」でエクスプローラーも開きます。



7キーボードショートカット

Win + Home -1つのアクティブなウィンドウで作業し、すべての非アクティブなウィンドウを最小化します。

Win + T-タスクバーに沿って歩く

タスクバーのアプリケーションアイコンをShift +クリックまたは中クリックすると、新しいアプリケーションウィンドウが開き、既に開いているものは表示されません

Win + P-投影ダイアログを開きます。1回または2回クリックするだけで、2番目のモニターまたはプロジェクターに画像を転送できます

Win +左矢印およびWin +右矢印 -ウィンドウを画面の境界にドッキングします

Win +上向き矢印およびWin +下向き矢印 -ウィンドウを展開/復元/最小化

Win + Shift +上向き矢印およびWin + Shift +下向き矢印 -ウィンドウを垂直方向に最大化、復元/最小化しますが、画面全体ではありません。

Win + Shift +左/右 -（マルチモニターシステムの所有者向け）ウィンドウを左/右モニターに転送します

Ctrl + Shift +アプリケーションをクリックして、すぐに管理者権限で呼び出します。

Win + Space-デスクトップ、時間を見る



昔ながらの便利な組み合わせを思い出します



Win + D-すべてのウィンドウを最小化し、デスクトップを開きます。 アクションを繰り返すと、ウィンドウは元の場所に戻ります。

Win + L-デスクトップを閉じます。アカウント（ユーザー）がパスワードを持っている場合は特に効果的です。

Win + R-開始/実行ダイアログを呼び出します

Win7のCtrl + Alt + Deleteは、タスクマネージャー（タスクマネージャー）を含むいくつかの操作を選択したウェルカム画面に似た画面に戻ります。 すぐにタスクマネージャーを開く場合は、次の組み合わせCtrl + Shift + Escを試してください。



他にもたくさんありますが、Googleはそれらをすべて知っています;）



8 Snipping toolにキーの組み合わせを割り当てる

Snipping toolは、選択した領域のスクリーンショットを撮るためのユーティリティです

[スタート]メニュー=>アクセサリ=> Snipping Tool =>右クリック=>プロパティ

[プロパティ]の[ショートカットキー]フィールドで、キーの任意の組み合わせを入力しようとします。 最初の2つのキーCtrl + Alt +がすぐに与えられることに注意してください。3番目のキーの選択は、あなたの好みです。 たとえば、 Ctrl + Alt + Sがあります。 付箋でも同じ操作を実行できます。また、クイックアクセスのキーの組み合わせを指定します。
