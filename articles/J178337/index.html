MicroXperts MXP U400-04ウルトラブックレビュー。 再読み込み



ロシアのメーカーの別のウルトラブック。 すぐに、プラットフォーム自体は台湾の工場で生産され、ロシアでは完成品と販売前の準備とテストに組み込まれていると発言します。 MicroXpertsはすでにLengdaのプラットフォームで市場に出回っており、ここに再登場します。 このモデルはMicroXperts U400-04と呼ばれ、ODM業界のリーダーの1つである14インチベースのPegatron B14Yに基づいています。

残念なことに、テスト用にOEMパッケージのプリプロダクションコピーを受け取りました。小売バージョンはYulmartサイバーマーケットの棚ですでに見られるはずです。 そのため、私はパッケージの写真や、段ボールやプラスチック製の付属品に関する他の楽しみをせずに管理します。 ハードウェアに近い、ポイントに右。 タイトルの写真からわかるように、ウルトラブックの色は黒と銀で、外観は非常にサラブレッドです。 すべての点で、IntelがUltrabookと呼ばれるデバイスに課す要件を満たしています。

プラットフォームメーカーについて少しお話しします。 ペガトロンは市場で長い間成功を収めています。 元々はASUStekの子会社でしたが、業界大手の再編後、ペガトロンはほぼ完全な独立を獲得しました。 ペガトロンの施設で製品を製造しているASRockとの緊密な協力について広く知られています。 さらに、小売店では、Pegatronブランドでラップトップを組み立てるためのマザーボード、モノブロック、ネットトップ、プラットフォームを見つけることができます。 Appleを愛する誰もがFoxconnの工場でデバイスを生産しているだけでなく、小さなスキャンダルを引き起こしたPegatronにも部品を配置しています。 Pruflinksはインターネットで簡単に見つけることができます。問題の本質は、PegatronがASUS用のZenbookを生産するか、Apple用のiPhoneを生産するかを選択することでした。 ただし、これは歌詞です。プラットフォームに関する会話を続けます。



Intelのすべての要件に準拠して、メーカーは光学ドライブの使用を拒否しました。これにより、デバイスの厚さを最大18 mmにすることができなかったためです。 側面には、周辺機器用のコネクタと電源アダプタがあります。 左：VGAビデオ出力（私の個人的な意見ではまだ必要ありません）、ケンジントンロックスロット、イーサネットポート、HDMIポート、1つのUSB 3.0コネクタ。 右側には、複合オーディオジャック、SD / SDHCカードリーダー、USB 2.0、USB 3.0、および電源アダプタを接続するためのコネクタがあります。









電源ボタンはメインキーボードとは別に右側にあります。多くの場合、この場所は非常に便利です。 ボタンの青色のバックライトは電源オンのインジケータであり、ウルトラブックには他のライトインジケータはなく、充電表示を担当するLEDもハードドライブアクティビティのインジケータもありません。 議論の余地がある決定であるため、ほとんどのユーザーはLEDによって充電状態を正確に判断することに慣れています。



タッチパッドは十分に大きく、触ると粗いです。 表面はキーボードに対して凹んでいません。 ユーザーに最も近い部分は2つのボタンに分かれており、その間にはウルトラブックの電源を入れると青色のLEDが点灯します。









MicroXperts U400-04キーボードは快適ですが、欠陥がないわけではありません。 まず、メリット：フルサイズのカーソルキー、「アイランド」タイプのキーボード、静かで滑らかなキーストローク、ラテン文字とキリル文字のさまざまな色。 1マイナス-作業面で少し歩く。



キーボードの下には、底部カバーを保持するいくつかのネジがありますが、これは率直に言って不便ですが、取り付けたデバイスを分解するのが好きな人にとってのみです。 通常のユーザーは、ウルトラブックを分解することはありません。つまり、困難に直面することはありません。 ちなみに、多くのデバイスとは異なり、MicroXperts U400-04には実際にはステッカーやラベルがありません。 Intel Core i5ロゴのみがミニマルで賞賛に値します。



ウルトラブックの底部カバーは頑丈でプラスチック製です。 いくつかの通気口とプラットフォームのシリアル番号が記載されたステッカー-それが宝飾品の全体セットです。



しかし、キーボードの下から9本のネジを外し、キーボードの下から7本のネジを外すと、驚くほどの豊かさと内部の素晴らしい美しさを手に入れることができます。



メーカーによって宣言されたバッテリー容量は3320 mAh、供給電圧は14.8 Vです。最高のパフォーマンスではありませんが、バッテリー寿命を判断する前にパフォーマンステストを待ちましょう。



RAMは、4ギガバイトの量産キングストンのSO-DIMM DDR3-1333の1つのモジュールに入力されます。



アップグレードのファンには朗報です。RAMの2つのスロットがプラットフォーム上で離婚しています。 同時に、サポートされるメモリの最大量は16ギガバイトです。つまり、経験豊富なユーザーは、2番目のモジュールを追加するか、より容量の大きいモジュールと交換することにより、メモリ量を個別に増やすことができます。



Atheros AR5BWB225ワイヤレスモジュールは、802.11 b / g / nおよびBluetooth 3.0をサポートしています。 Realtek RTL8111統合有線ネットワークインターフェイス



ヒートパイプの下には、ウルトラブックの中心であるIntel Core i5-3317Uプロセッサがあります。



MicroXperts U400-04ケースの外側の熱は、タンジェンシャルタイプのファン、つまりリアパネルに向けられたタービンによって除去されます。 吸気口は底部パネルの開口部から取られるため、冷却システムへの空気のアクセスを妨げないように、ウルトラブックを柔らかい表面に置かないように注意してください。



Crucial SSDは128ギガバイトで、ほとんどのユーザーに十分です。 外付けハードドライブとネットワークストレージはまだキャンセルされていません。



ボードには、mSATAインターフェースを備えたドライブを取り付ける場所があります。 これにより、ハイブリッドストレージシステム、つまり従来の2.5インチハードドライブとSSDドライブを使用できるようになります。このようなソリューションのコストは、ソリッドステートドライブのみよりも低くなり、データ量が大幅に増加します。



詰め物について詳しく説明します。 周波数1.7 GHzのIntel Core i5 3317Uプロセッサは、MicroXpertsウルトラブックの基礎です。



マザーボードに関する情報：Intel HM77チップセットは、ウルトラブックでは一般的なものです。



HWInfo64プログラムの一般的なスクリーンショット





RAMは、シングルチャネルモードで9-9-9-24-1Tのタイミングで1333 MHzの周波数で動作します。 2番目のモジュールを取り付けると、デュアルチャネルメモリモードがオンになります。



RAMモジュールのSPDは、写真に既に表示されていたものを確認します-キングストンのメーカー。



グラフィックスアダプターはプロセッサに統合されています。 Intel Core i5 3317UはIntel HD Graphics 4000を使用します。



ソリッドステートドライブ情報。 繰り返しますが、驚くべきことではありませんが、データは内部で見られたものと一致します。 スクリーンショットを撮った後、ディスクの摩耗の原因となるE9属性が表示されていないことに気付きましたが、SSDが完全に新しく7回しかオンになっていないことを考えると、リソースが100％であることは間違いないと思います。



このドライブはMarvell 88SS9174コントローラーに基づいています-多くのSSDの幅広いオプションです。 メモリは、25 nmプロセステクノロジーを使用してMicronによって作成されました。



次にテストを行います。 Windows 8オペレーティングシステムがウルトラブックにインストールされているため、組み込みのテストをWindows 7のアナログと比較することはできませんが、このスクリーンショットは今後のテストに役立ちます。 有名なOSの意見で最も弱い点は、グラフィックコアです。



RAMテストでは、シングルチャネルソリューションによる最大パフォーマンスは示されませんが、将来メモリを拡張する可能性があるため、IMHOの方が便利です。



ソリッドステートドライブの読み取り/書き込み速度テストは、読み取り時のMarvelコントローラーの強さを示し、少量のSSDに関連する最高の書き込み速度ではありません。 同じドライブですが、256ギガバイトはデータをより速く書き込みます。 同時に、作業が従来のハードまたはハイブリッドシステムよりも大幅に高速であることは注目に値します。



PC Mark 7の包括的なパフォーマンステストは良い結果を示しています。これは、第3世代Intel Coreプロセッサーの新しいビデオコーディングテクノロジーと高速ソリッドステートドライブによるものです。



電源管理ユーティリティでは、動作モードを選択できるだけでなく、ウルトラブックの現在の電力消費量も表示できます。



MicroXperts U400-04は、多くの競合他社よりも優れた温度体制を備えています。 テストの結果としての最大プロセッサ温度は69度でした。



そして、ここにバッテリー寿命テストがあります。 容量が小さいにもかかわらず、ウルトラブックは非常に粘り強いものであることが判明しました。 合成負荷でわずか3時間を超える結果により、オフィスアプリケーションでの5〜6時間のリアルタイム作業が期待できます。





バランスのとれたもの：デバイスは非常に堅実で、機能や欠点がないわけではなく、利点と民主的な価格を備えています。 私がテストしたバージョンは、小売ネットワークで約24,000のコストがかかりますが、これらの特性は非常に優れています。 もちろん、最高級のASUS ZenBookまたはApple MacBook Airに代わるものとして、MicroXperts U400-04を検討することは不可能ですが、AcerやLenovoと競合することはかなり可能です。 もう1つの非常に重要な機能は、3年間の保証期間です。大規模なベンダーであっても、製品にそのような条件を与えません。
