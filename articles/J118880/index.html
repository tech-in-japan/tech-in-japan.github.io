オンラインのサウンドエディターとシーケンサーを閲覧する









ご存知のように、オーディオと処理を作成するためのソフトウェアは、高度に専門化された製品の境界をはるかに超えており、一般のミュージシャンに向けて継続的に進んでいます。 このレビューでは、さまざまなオンラインプラットフォーム上に構築された8つの主要なライブオーディオアプリケーションについて説明します。





オーディオツール







Audiotoolは、有名なFlashプログラマーであるAndre Michelleによる有望なプロジェクトで、JavaのパワーとFlashのユーザーエクスペリエンスを組み合わせています。 このプロジェクトは、Propellerheads Reasonなどのシーケンサーを備えたモジュラーシステムとして設計されています。 いくつかの異なるシンセサイザーとサンプラーを使用できます。 Audiotoolを使用すると、提供されている14のエフェクトのいずれかで楽器を切り替えたり、エフェクトのチェーンを構成したり、エフェクトを並列または直列に接続したりできます。 1か月前、Audiotoolはバージョン2.0に更新され、レコーディングオートメーション、テンプレート、MIDIの操作、サンプルのインポートなどをサポートするようになりました。特に、KORG Kaoss Padのようなエフェクトプロセッサと新しいミキサーが登場しました。 しばらくの間、AudiotoolはBurn Studiosとの協力を開始しましたが 、これが無料のプロジェクトとタイムリーなアップデートに影響しないことを願っています。 古いバージョンのAudiotoolの概要はハブにあります;私はこのプロジェクトのすべての機能を詳細に調べました。



ウジャム







uJam-有名なeJayのように、最短時間で音楽の作成を開始できるプロジェクト。 eJayと同様、uJamには多数のスタイルプリセットがあり、マイクから録音を追加できます。 曲のテンポ、伴奏、ピッチ（オーディオフラグメントの高さ）を変更し、結果の曲をuJamサーバーまたはハードドライブに保存できます。 uJamは、オリジナルの音楽をゼロから作成するという目標を設定していませんが、ユーザー定義のフラグメントと提供されているフラグメントを混在させることができます。 ある意味、自動伴奏付きのシンセサイザーを演奏しているように見えます。



IndabaMusic Mantis







IndabaMusic Mantisは、Javaアプレットの形式で作成された強力なDAWです。 CD品質のサウンドを録音できます-44 Khz / 16ビット、Indabaライブラリから10,000以上の高品質の無料サンプルを追加し、組み込みエフェクトでサウンド処理を実行し、オートメーションを登録し、結果のミックスをIndabaサーバーに保存します。 ミックスへのアクセスを複数の参加者に同時に提供する機能も利用できます。 Mantisでの作業は、実際のDAWで作業するような他のオンラインエディター、特にこのプロジェクトの目的であるSteinberg Cubaseよりもはるかに優れています。 インダバはまた、マンティスを使用して有名なミュージシャンとのコンテストを開催しています。



注意：理由は明らかではありませんが、Mantisがコンピューターへのロードを停止したのは、ローカルJavaキャッシュからデータをダウンロードできないことを指します。同様のエラーが発生する可能性があります。ブラウザーを変更してキャッシュをクリアしようとしましたが、これは役に立ちませんでした。 これが一時的なバグであることを願っています。



鳥小屋







Aviary Rocは、オンラインエディタを作成するためのウィザードであるAviaryの次のプロジェクトです。 Rocの前に、Aviaryは最も人気のあるグラフィックオンラインエディターの1つであるフェニックス、ベクターグラフィックエディター、Ravenおよび他の多くの同様のエディターを既に作成していました。 Rocは、Waveファイルを読み込んでシーケンスを作成できるサンプラーです。最も単純な編集は、音量レベルとパノラマを変更するレベルでサポートされています。 結果のシーケンスは、Aviaryサーバーに保存できます。 通常のタイプのサンプラーは、ドラムマシンの精神とピアノロールの形でサポートされています。



鳥小屋







Aviary Mynaは別のAviaryプロジェクトです。標準DAWの形式で作成されたオーディオシーケンサーは、マルチトラックミキサー、ユーザーのマイクから録音する機能、およびハードドライブからオーディオファイルをインポートする機能を備えています。 追加したマテリアルは、シーケンサーのメインウィンドウに配置し、パノラマまたはボリュームにオートメーションを適用し、サンプルの最初または最後に減衰を追加できます。 インポートされた各オーディオフラグメントを8つのエフェクトで処理することもできます。 作業の最後に、結果のマテリアルをレンダリングしてハードドライブに保存できます。



芸者音楽







Geisha Musicは、まだベータテストの初期段階にあるプロジェクトですが、すでにユーザーが利用できます。 これは、サンプラー、シンセ、ミキサー、シーケンサーの4つの部分に分かれた画面で、それぞれの名前は自明であり、コメントは不要です。 ある種の再生Propellerheads ReBirthをもう一度。 プロジェクトはベータ版であるため、多くの機能はまだ実装されていませんが、実装された機能はすでにAudiotoolのようなプロジェクトの開発の可能性の兆候を示しています。



サウンデーションスタジオ







Soundation Studioは、サンプルライブラリ-PowerFXの作成時にウィザードによって作成されたオーディオエディターです。 マイクからのサウンドの録音、サンプルライブラリからのサンプルの追加、11個の組み込みエフェクトを使用したオーディオフラグメントの処理、ハードドライブからのMIDIファイルのインポート、メトロノームとシンプルなシンセサイザーの組み込みが可能です。 Mac OS Xの精神に沿った便利なインターフェイスと、パラメーターを自動化する機能を備えています。 Soundationプロジェクトファイルは、ハードドライブに保存するか、サイト自体に保存できます。 プロのアカウントの支払いを済ませると、プロジェクトのレンダリングをハードドライブに保存できるようになります。



スクラッチオーディオ







Scratch Audioは、新しいMicrosoft Silverlightテクノロジーを使用してDAWを作成するプロジェクトです。このテクノロジーを使用すると、再生と録音中の遅延、長いダウンロードを取り除き、ユーザーのコンピューターにオーディオフラグメントのストレージを作成できます。 現在、Scratch Audioはマイクサウンドを録音したり、サンプルの組み込みライブラリからサンプルを追加したり、Drums on Demandが提供するドラムマシンとサンプルを使用してサウンドを作成したりできます。 シンプルなエフェクトの処理、ボリュームとパノラマの変更も可能です。 最終的なミックスは、Scratch Audioサーバーまたはハードドライブに保存できます。



おわりに



したがって、トラックを作成するほとんどすべての段階は、すでにブラウザで直接実行できます。オーディオツールに対するバーンの関心は、そのようなソフトウェアを作成するというアイデアの商業的実行可能性を証明するだけです。 おそらく将来、このようなソフトウェアは、ある巨大なオーディオ技術者によって発表されるでしょう。たとえば、プロジェクトを実際のDAWにインポートできるオンラインエディターを見るのは興味深いでしょう。 一般に、すべてはまだ先であり、このレビューには非常にアルファ段階のプロジェクトは含まれておらず、そのうちのいくつかは現在スクリーンショットと説明および他の非常にシンプルなエディターのみが存在するクローズドテストのプロジェクトです このようなすべてのプロジェクトの欠点は、フラッシュまたはJavaマシンの使用による処理速度の低さであり、その結果、同時に処理されるトラックの制限ですが、この制限に決して従ったことがありません。
