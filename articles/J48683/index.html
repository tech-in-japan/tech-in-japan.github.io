著者は、危機を体系的に宣言することから始め、それを経済的なものに還元します。 いや！ 高熱は原因ではなく、病気の結果です。 本当の理由はもっと深く、もっと悪い。

無制限の人口増加のモデルを考えます（制約要因はありません）、その後の各期間の人口は、特定の係数N1 = kN0を乗算した前の期間の数に等しくなります。

この依存性は、生物学ではOllie曲線（2リットル）の名前で知られており、人間の人口の大きさによく似ています。

有限差分から通常の方程式への遷移により、依存性/ N = A /（Tt）が得られます。 A = 3.12E + 11人 ご覧のように、無制限の成長の式には係数Tによって設定された制限があります。したがって、最も興味深いのは、この式は過去の範囲の人口増加をよく説明していますが、係数Tは2020です。 今、あなたはそれがどこから来たかを理解する必要があります。

これらの事実は長い間知られています。 ペレストロイカの夜明けに、アジテーターが学校にやって来ました。 それは人類の歴史上、そのような危機がいくつかあると言った：1-彼らはマンモスを食べました、2-彼らは北アフリカの森林を燃やしました-サハラ砂漠を作りました、その後、3は農業と牛の繁殖を開き、今日まで快適に暮らしました。 したがって、（彼によれば）科学技術革命の時代が始まっています-科学技術革命とそれに適応できない人々はマンモスのように死ぬでしょう。

したがって、事実はわかっています。 さらに、開発戦略を策定する際に、さまざまなアナリストによって積極的に考慮されました。

次に何が起こるでしょう。 原則として、心配することはありません。 ただ横行する成長は止まります。 成長を表す方程式は、N1 = N0（kq * N0）の形式になります。ここで、qは人口過剰による死亡率です。 この方程式の研究は、独立した仕事のためにあなたに任されています。 私は美しく約束します:)それは非線形です。 さまざまな係数で、振る舞いは非常に大きく変化します（可能なオプションは、定常落下、定常成長、平衡点周辺の振動、カオスモードです）。さまざまな係数の一般的なグラフ

第2の結論は、人口爆発が終了し（ガラポゴスのリールでのダーウィンの作業を思い出してください）、選択の安定が有効になることです-人類の統一の時代が始まります。 人口の生存に寄与しないすべてのスキルと能力は、進化によって拒否されます。 その結果、カラシニコフのアサルトライフルを使ってイスラム教と民主主義を実践する狭い目黒人になる可能性があります。

個人、コミュニティ、国家は現在の状況でどのように行動すべきか。 彼らが持っている進化上の利点について考え、それらを人類の情報ゲノムに統合しようと試みてください。 これは、彼らと彼らの子供たちが太陽の下での戦いで有利になることを意味します。
