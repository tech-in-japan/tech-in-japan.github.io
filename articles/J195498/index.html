NvidiaはTegra Noteを導入しました



最後に、Nvidiaはカーテンを開き、待望のガジェットTegra Noteを見せました。 これは消費者向けのガジェットではありませんが、パートナー向けの設計モデルになります。 Nvidiaはまた、インドではこのデバイスがXoloとして知られるようになると述べています。



Nvidia Tegra Noteには、7インチHD（1280x800）IPSディスプレイ、72コアGeForce GPUおよび4コアCortex-A15 CPUを搭載したTegra 4プロセッサが搭載されています。メガピクセルのリアカメラとフロントVGA。このデバイスには16 GBのメモリがあり、microSDを介してさらに32 GBまで拡張できます。このデバイスは、HDビデオ再生モードで最大10時間動作することを約束します。



デバイスの他の機能は、スタイラスを使用するためのDirectStylusテクノロジー、PureAudio、iOSユーザーの間で知られているSmugMugカメラアプリケーション、およびTegra用に最適化されたゲームへのアクセスを表す有名なTegraZoneです。



Tegraの製品マーケティングディレクターであるMatt Wuebbling氏は、この発表について次のように述べています。 「Nvidiaは、世界のさまざまな地域でこれらのデバイスの巨大な市場を獲得しています。」



このデバイスの推奨価格は199ドルです。 パートナーへのデバイスの配信は、今後数か月で開始されます。



まあ、短い恋人のための仕様の抜粋：

  • ディスプレイ：7 "IPS LCD 1280x800
  • CPU / GPU：4コアCortex-A15 CPU / 72コアGeForce GPU
  • RAM：1GB
  • メモリ：拡張可能なmicroSDで16GB
  • カメラ：5MP背面、VGA前面
  • ネットワークと接続：WiFi 802.11 b / g / n、Bluetooth、micro HDMI
  • 自律性：10時間のHDビデオ
