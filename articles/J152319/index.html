Digium G100 / G200 E1 / T1 / PRIルーター





Digium-アスタリスクを提供してくれた会社は、今年、VoIPデバイスの新しいファミリを導入しました。



このラインの最初の製品はG100およびG200ルーターです。 アスタリスクと最新の統合プラットフォームに基づいて構築されたこのソリューションは、あらゆる通信ソリューションで最高の価格/機能の組み合わせを提供します。



これらの2つのVoIPルーターは、TDM-to-SIP、SIP-to-TDM、およびSIP-to-SIP（トランスコーディング）をサポートしています。 どちらもノイズがなく、可動部品がないため、信頼性が高くなります。



軽量なGUI、低消費電力、構成が簡単です...これはVoIPエンジニアの夢です。



技術仕様とデバイスの概要に興味のある方、Habrakatへようこそ





G100モデルには、1つのT1 / E1 / PRIインターフェース（ソフトウェアで選択可能）が含まれ、30の同時通話を保持します。 G200それぞれ2つのT1 / E1 / PRIインターフェースと60コール。 どちらのデバイスにも、エコーキャンセルモジュールが組み込まれており、1U半角/半角に対応する小型サイズがあります。







G100およびG200の仕様は次のとおりです。



  • 自動コールタイプ検出：ファックス、音声、モデム
  • トランクグループのサポート
  • ダイヤル規則のサポート
  • ダイヤル番号の番号を自動的に追加および削除します
  • 発信者IDの名前と番号をサポート
  • E1：PRI（EuroISDNまたはQ.SIG）
  • 組み込みのエコーキャンセレーション
  • コーデック：

    •G.711（A-lawおよびU-law）

    •G.722•G.729

    •GSM-FR•G.726
  • HTTP / HTTPS
  • 10/100 / 1000Base-Tイーサネット
  • USBシステム復旧ポート
  • 21.84 x 22.86 x 4.37 cm
  • 220ボルト、40ワット。




だから、私たちは開きます...







実際、可動部品はなく、1 GB DDR3用の1つのブラケット、110-240V電源、アルテラCyclone IIIプロセッサモジュール、その他はすべてマザーボードにはんだ付けされています。







デバイス管理インターフェイスに移動します。 まず、アップデートでは、ゲートウェイは新しいソフトウェアを認識し、アップデートを親切に提供します。 便利に。







SIP設定セクション（写真はクリック可能）。 ゲートウェイは、サーバーまたはクライアントとして機能できます。

DTMF設定、コーデック選択、発信者ID管理など。



[詳細SIP設定]セクションでは、より正確な設定、NAT、TCP / UDPポート、RTPポート範囲、SIP登録時間、QOSなどを行うことができます。







E1設定は印象的です。メイン設定のスクリーンショットを添付します。







接続の問題が発生した場合は、高度なデバッグに連絡できます

構成の予約、Digiumテクニカルサポート（VPNの構築）、統計情報の収集、ファイアウォールを介してリモートで構成する機能-これらはすべて当然のことです。



このゲートウェイの用途は何ですか？




1.生まれてからSIPが何であるかを知らなかった古いPanasonicまたはAvayaに2回目のチャンスを与えることができます。







2.最近、非常に頻繁に、IP-PBXがクラウドに移行します。クラウドには、通常のPCIまたはPCI-Eカードを挿入する方法がありません。



または、E1のフローと交換が物理的に互いに離れている場合。 このゲートウェイは必須アイテムです！







3.最後に、最後のオプション。 E1フローを制御する場合は、DIDの一部を古いPBXに送信し、残りを新しいAsterisk / Switchvox / etcサーバーに送信します。



これはすべて、この製品で実行できます。







ロシアの価格は、G100 / G200でそれぞれ1300ドルと2199ドルです。 高い？ 安い？ 同様の製品の価格を提示することもできます。
