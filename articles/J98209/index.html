Chromeはアプリを獲得しました

有効にする方法：

  • 最新ビルド（6.0.453.1 dev ）への更新
  • Chromeショートカットに追加します。

    --enable-apps --apps-panel

    （--enable-apps-アプリケーションのサポートを有効にします

    --apps-panel-ポップアップパネルを追加します）
  • 拡張機能メニュー（chrome：//拡張機能/）に移動し、「開発者モード」に切り替えて、「解凍された拡張機能をダウンロード...」を選択します
  • アセンブリとともに3つのアプリケーションがあり、それらは

    C：\ Users \％username％\ AppData \ Local \ Google \ Chrome \ Application \ 6.0.453.1 \ Resources
  • 喜ぶ
