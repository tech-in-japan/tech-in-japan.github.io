Mandriva Linuxのコミュニティビルド

公式には、Mandriva LinuxはFree、One KDE4、One Gnomeのエディションでのみ無料でダウンロードできます。 しかし、「非公式」であり、さまざまなMandriva Linuxコミュニティによって作成されたビルドも多数あります。 ディストリビューションのファンが今でも試すことができるものを見てみましょう。 非公式のアセンブリは、公式のアセンブリよりも多くのエラーがある可能性があることをすぐに言わなければなりませんが、これは取得した機能によって平準化されます。

  • MIB Live E17 2009.1-International Editionは、E17がグラフィカル環境として使用されるアセンブリです。 2.4 Gb画像サイズ
  • MIB Live Xfce 2009.1 -XFCEに基づくアセンブリ。 着信ソフトウェアのリストは、 mib.pianetalinux.org / 2009.1 / iso / docs / xfceにあります。
  • MIB Live Gnome 2009.1-名前からすべてが明らかです。 受信ソフトウェアのリストはこちら： mib.pianetalinux.org/mib/en/mib-news/1-news/392-mib-live-gnome-20091.html
