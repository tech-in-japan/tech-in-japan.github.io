Windows XPのセキュリティ更新プログラムの受領をさらに5年間延長します

2014年4月8日、ハイテクの世界でひどいことが起こりました。Microsoftはついに、最も人気のあるオペレーティングシステムであるWindows XPのサポートを正式に停止しました。 オペレーティングシステムにはゼロデイ脅威に対するパッチが残されていなかったため、何百万人ものユーザーがウイルス感染の危険にさらされていました。 それにもかかわらず、誰もがソフトウェアの巨人が望んでいるように新しいバージョンに切り替えるのに急いでいるわけではありません。 数千台のマシンがWindows XPで動作する企業環境では、すべてがはるかに深刻であり、その更新により何百万もの費用と長年の研ぎ澄まされた企業ソフトウェアの予期しない問題が発生します。



しかし、今後5年間、Windows XPセキュリティシステムの更新の受信を延長できる単純なハックがあることがわかりました。 2019年4月まで！



これは、Windows XPの特別なバージョンであるWindows Embedded POSReady 2009のおかげで可能になりました。 このシステムは2009年にリリースされ、Windows XP Service Pack 3に基づいています。さまざまなPOS端末、キオスク、セルフサービスシステム向けに設計されています。 Windows XPユーザーは、オペレーティングシステムにこれらの更新プログラムを直接インストールすることはできません。 ただし、Windowsレジストリに特定のキーを追加するだけで、システムにこれを実行させる方法があります。



ウォームランプノートブックで新しいファイルを開き、3行でハンマーを打って、拡張子.regで保存し、管理者権限で実行します。 



Windows Registry Editor Version 5.00 
[HKEY_LOCAL_MACHINE\SYSTEM\WPA\PosReady] 
"Installed"=dword:00000001
