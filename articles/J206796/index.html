4GモデムMR100-2の概要

本日は、メガホンの4GポケットサイズルーターMR100-2について簡単にご紹介します。 このデバイスの主な目的は、Wi-Fiを介したインターネットトラフィックの分散を備えた4Gネットワ​​ークでオフラインで動作し、同時にルーターが最大10台のモバイルデバイスに対応できるようにすることです。 Megafonネットワーク上でのみ動作するようにルーターがロックされ、他のオペレーターのSIMカードで動作するように使用できないことを直ちに予約します。







宣言された仕様：



バッテリー：内蔵、リチウムイオン、2100 mAh

バッテリー寿命： 5時間

充電時間： 2時間

寸法： 97x66x13 mm

重量： 99グラム

Wi-Fi規格： IEEE 802.11b / g / n

最大Wi-Fiデータレート： 150 Mbps

2G標準： GSM 1800 / GSM 1900 / GSM 850 / GSM 900

3G標準： UMTS 2100 / UMTS 900

4G標準（LTE）： LTE 2600 / LTE 800

GPRS：はい

エッジ：はい

HSPA +：はい

HSDPA： 42.2 Mbps

HSUPA： 5.76 Mbps



パッケージバンドル



ルーターは小さな箱に入っており、そこにはデバイス自体、説明書、microSIM形式のSIMカードが入ったブリスターがあります。 MegafonのWebサイトでは、構成が変更される可能性があることに同意しているため、SIMカードなしでルーターを入手できる可能性があります。

外観とデザインの機能



MR100-2には、白とダークグレーの2色があり、プラスチック製の本体は滑らかで手触りが快適です。 すでにポケット4Gルーターに興味がある場合は、MR100-2の外観はおなじみでしょう。これはYota Manyのほぼ完全なコピーです。 ただし、どちらもYota Devices自体によって開発されており、YotaネットワークはMegaphoneに属しているため、これは驚くことではありません。 主な外部の違いは、LEDインジケータの形状です。MR100-2では、「4G」という碑文の形で作成され、明るい緑色の光で強調表示されます。 設定では、バックライト操作の3つのモードのいずれかを選択できます。常時オフ、常時オン、データ交換中のみオン（このモードはデフォルトでオンになっています）。





ルータにはヒンジ付きのフルサイズUSBコネクタが装備されており、それを開くには、特別に用意された棚を引っ張るだけです。 ヒンジは十分にきつく、コネクタはハングしません。







ヒンジ付きスロットは、SIMカードスロットを隠します。 それをインストールするとき、私はいくつかのニュアンスに遭遇したと言わなければなりません。 SIMカードをスロットに挿入し、その端がケースの切り欠きに引っかかるまで押し込みました。 スロットはスプリング式であることが判明し、カードは2〜3 mm戻りました。 必要だと判断したので、ルーターの電源を入れました。 しかし、彼はSIMカードがインストールされていないと述べました。 説明書のイラストと「SIMカードの端の部分がケースの端の下に見えるようにする」という文言も、私がすべて正しく行ったことを確信させました。 SIMカードを取り付けるとき、ラッチが機能してカードがロックされるように、SIMカードをケースの奥深くに浸す必要があることがわかりました。 爪の長さが足りない場合にのみ、指ではできません。

ちなみに、SIMカードを引き出すことはあまり便利ではありません：クリックすると、ラッチから解放され、バネの作用下で同じ2-3ミリメートルだけクリープしますが、スロット内の摩擦とSIM自体の滑りのために、指でさらに引くことは常に可能ではありませんカード。 私たちは鋭い何かでそれに固執しなければなりません。





USBコネクタを閉じると、microUSBスロットの裏面にアクセスできるため、何らかの理由で直接接続が困難または望ましくない場合は、適切なケーブルを使用してモデムを接続できます。 ルーターの充電は、USBコネクタとmicroUSBポートの両方で可能です。





このルーターの注目すべき特徴は、液晶ではなく電子インクでディスプレイを使用することです。 正式には、これはバッテリーの電力を節約するのに役立ちます。 節約がどれほど重要であるかはわかりませんが、率直に言って、解決策はハックされていません。

ルーターの電源を切ると、ディスプレイにバッテリー残量が表示されます。 オンにすると、動作モード（2G / 3G / 4G）の指定と、従来の「セルラー」形式の受信レベルも追加されます。







ルーターの唯一の外部管理機関は、スライダースイッチです。 左の位置に移動すると、ハイブリッドアクセスがオンになります。これは、パスワードの有無にかかわらず、Wi-Fiを介してユーザーを同時に接続できることを意味します。 合計で、最大10の同時接続が可能です。両方のタイプの接続の制限は設定で変更されます。 スライダーを正しい位置に移動すると、パスワードなしのアクセスモードのみが含まれます。 中央の位置では、ルーターはオフになっています。





スライダーを左に動かすと見える赤い点に注意してください。これはルーターの再起動ボタンです。





一般に、スライダーの設計はあまり成功していません。ルーターとスイッチの本体を超えて非常に簡単に拡張できます。 したがって、バッグやポケットに誤って入れてしまう可能性は十分にあります。その結果、適切なタイミングで、放電されたデバイスと通信することなく自分自身を見つけることができます。



設定



初めてルーターの電源を入れると、「メガホン」と呼ばれるアクセスポイントとして見えるようになり、パスワードは不要です。 接続後、ブラウザに任意のアドレスを入力する必要があります。その後、ルーターのコントロールパネルに移動します。 その後、いつでもstatus.megafon.ruで入力できます。 現在の動作モード（2G / 3G / 4G）、送受信されたトラフィック、およびバッテリーレベルがページのヘッダーに表示されます。 設定は3つのタブに分かれています（追加の設定を含むすべての非表示リストが展開されます）：

ステータス ここでは、現在の動作モード、受信と復帰の速度、セッション時間、IPアドレス、アカウントの残高に関する情報が収集されます。





設定 ここでは、ルーターに名前を設定し、パスワードアクセスを有効にし、パスワードによる接続制限の変更とパスワードなしでの接続制限の変更、およびネットワーク選択モードの設定を行うことができます。





メッセージ このセクションは、USSD要求を使用してサービスメッセージを表示し、利用可能なサービスを管理するために使用されます。





デフォルトでは、設定のある別のタブはパネルに表示されません; status.megafon.ru/advancedに移動したときにのみ表示されます：





速度と安定性



自動モードでは、ルーターはオンになってから約10秒以内に非常に迅速にネットワークに接続します。 モスクワのさまざまな地域とヒムキでのテスト中、3Gの受信速度は平均480 kbit / sでしたが、4Gでは受信が最大23 Mbit / sで、リターンは最大1.5 Mbit / sでした。

市内の一部の地域では、自動モードで信号レベルが低いため3Gに接続されているため、ルーターを強制的に4Gモードに切り替える必要がありました。 このような状況では、ネットワークへの参加に最大5分かかりました。

主観的に、自信を持って4Gを受信した場合のページ読み込み速度は、ケーブル接続による速度と変わりません。 もちろん、映画などの大きなファイルをダウンロードする場合、違いが明らかになります。



作業時間



バッテリーを完全に充電するのに1時間55分かかりました。 4Gモードで絶えずオンにされ、インターネットで定期的にサーフィンを行っている（合計で約4〜5時間）ため、ルーターは13時間10分続きました。



おわりに



一般的に、MR100-2は非常に好印象を与えました。 肯定的な側面の中で、容量の大きいバッテリー、優れた4Gネットワ​​ーク速度、自動モードと強制モードの選択、スタイリッシュなデザイン、コンパクトなサイズに注目する価値があります。 欠点のうち、おそらく最も目立つのは、スライダースイッチを動作モードに無意識に切り替えて、バッテリーを放電する可能性です。 また、すぐにオープンアクセスポイントモードに切り替えることができるため、MR100-2は家族全員または友人の会社での使用に適しています。
