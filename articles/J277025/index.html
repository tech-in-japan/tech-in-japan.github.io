複雑なものから単純なものへ：仮想Windows / Linuxサーバーを起動する

今日は、 CloudLITEサービスで最近利用可能になったサービス「 Virtual Server Windows / Linux 」についてお話します。 11月にリリースを発表しました。BILLManagerに基づいた新しい課金およびプロビジョニングプラットフォームへの移行について話し、詳細を共有する準備ができました。







私たちは何をしましたか？



実践が示しているように、ほとんどのCloudLITEユーザーにとって、サービスのシンプルさはカスタマイズや機能のフリルよりも重要です：プリインストールされたOSを備えた既製のVMテンプレートは「個別の」アセンブリよりもはるかに人気があり、ネットワークを独立して構成したり、クラウドインフラストラクチャの管理に深く入り込んだりするユーザーは、終日それを行います。 顔をよりシンプルにすることにし、仮想データセンター（IaaS）に加えて、サービス「仮想サーバー（VDS）」を開始しました。



サーバーでの作業を開始するには、料金、OSテンプレート、ディスクサイズを選択するだけです。







アクティブ化の直後、サーバーは追加設定なしで動作する準備ができています。 サーバーの外観とパスワード、およびRDP / SSHまたはアカウントのウェブコンソール経由でリモート接続する方法についての簡単な説明が記載されたメールが届きます。







これまで、クライアントの間で最も人気のあるOSの既製のテンプレートに落ち着きました。

  • Ubuntu 14.04、
  • Centos 6.7、
  • ロシア語および英語版のWindows Server 2008/2012。


もちろん、将来、ラジオリスナーの要求に応じてこのリストを拡張します。



各OSグループの5つの料金は、仮想データセンターに比べてわずかですが、料金内ではディスクサイズを柔軟に制御できます。料金が古いほど、選択できる間隔が長くなります。







各サーバーには外部IPアドレスが割り当てられ、仮想データセンターの場合のように、NAT経由ではなく、サーバーに直接接続されるようになりました。

ボーナスのうち、サーバーライセンスはすでにVDS Windowsに含まれており、 ISPmanager 5 LITEパネルはLinux仮想サーバーに接続されています。

すべてのCloudLITEサービスと同様に、仮想Linux / WindowsサーバーはVMwareに基づいています。 機器は、モスクワ北部のNORD 3独自のデータセンターにあります。



他に何が面白いですか？



時間ごとの請求。 仮想データセンターですでに利用可能な毎日の支払いに加えて、VDSでは時間ごとの請求が機能します。 これは実際にはどういう意味ですか？ 1日中常に仮想サーバーが必要ない場合は、仮想サーバーをオフにして、お金を節約できます。 非アクティブ状態では、注文されたRAM、ディスク容量（サーバーがオフになっていてもデータを保存し続ける:)）およびライセンス（Windowsの北の場合）のコストのみが考慮されます。 ただし、プロセッサの費用を支払う必要はありません。 1日の間に、請求はサーバーが「覚醒」している時間数をキャプチャし、これらの統計を考慮して1日に1回お金を償却します。 借方記入に関する情報は、アカウントの「費用」セクションにあります。







サーバーが稼働した時間数に関するより詳細なデータ、プロセッサとディスクの負荷（％）は[ 監視 ]タブで監視できます（このセクションはまだベータテスト中です）。 将来的には、SMSの機能と、サーバーが使用不可になった場合や重大なプロセッサ負荷が発生した場合に電子メールで通知する機能が提供される予定です。



仮想データセンターへの移行 。 仮想データセンター（IaaS）と比較して、VDSは追加の設定を必要とせず、すぐに作業を開始できますが、この形式のオプションはわずかに少なくなります。 機能の上限に達していて、リソースをさらに制御したい場合は、サーバーをいつでもアップグレードして仮想データセンター（VDC）に転送できます。 必要なのは、適切なサイズのVDCを作成し、移行用の仮想サーバーを選択するだけです。残りはCloudLITEテクニカルサポートサービスが行います。







伝統により、私たちは誰もが新しいサービスを試乗することを勧めています。 これで、 1 vCPU、1 GB RAM、30-80 GB SSDの構成でUbuntu / Centosを 使用して仮想サーバーを試すことができます 。 このキットには、 ISPmanager 5 LITEパネルとDNSホスティングが含まれています。これは、 約束どおり 、今年初めに開始されました。

そして、いつものように、建設的なフィードバックに感謝します。
