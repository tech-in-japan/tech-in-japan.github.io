新しいデザイン、API、Androidアプリケーション





このバージョンには多くの新機能と機能があり、最も重要なのは開発者向けのAPIです。







新しいデザイン



以前のデザインは悪くはありませんでしたが、すべての部品と場所で論理的ではなく、多くの重要な機能が二次メニューから削除され、追加の機能は逆に印象的でした。







新バージョンでは、すべてがより論理的になりました。私にとっては、すべての主要なセクションがトップメニューに移動され、機能が向上したにもかかわらずサイト管理が容易になりました。



近くの人が作成したオファーやその他の多くの機能のためのサービスが追加されました。



これで、サイト全体が1つのページで機能するHTML5 + JSアプリケーションになります（認証中のリダイレクトを除く）。



これにより、プロジェクトの使用がより速く、より便利になりますが、プロジェクトで適切に動作できるブラウザの数も制限されます。



現在のバージョンでは、IEのサポートは完全ではありません（客観的な理由により）。



モバイル版とアプリケーション



サイトの新バージョンとともに、iOSおよびAndroidデバイスでの作業に最適化されたモバイルバージョンのプロジェクトを開始しました。



モバイルバージョンは、必要に応じてAPIからコンテンツをロードする1つの単一ページでもあります。







モバイルバージョンはデバイスを自動的に認識しますが、デバイスメーカーの「歪み」に関連してこれには多くの問題があります。 たとえば、HTC Sensationは、USER_AGENTに次の情報を示します。



Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10_6_3; HTC_Sensation_Z710e; ru-bg) AppleWebKit/533.16 (KHTML, like Gecko) Version/5.0 Safari/533.16







従来の方法では、このような電話はMAC OS X 10.6.3の何かとして定義されています。 他のAndroidデバイスがそのように自分自身を提示するのを好むコメントや、USER_AGENTで書いていることをコメントで書いてくれる人に感謝します。



すべてのAndroidデバイス（およびMAC OS X上のデバイス）の場合、アプリケーションは市場で入手できます。







iOSバージョンはAppStoreで管理されています。



周りの人々







本当に多くの人がいます。私たちは私たちの周りの人々に関する私たち自身の情報を公開するだけでなく、人気のあるジオロケーションや非常に非ネットワークからデータを収集します。



inmile.comにアクセスすると、モスクワの中心部やシンガポールのどこにいても、ほとんどの人が見られます。



マップ上、左側の人のリスト、イベントストリーム内のメッセージとマークでそれらを見ることができます。



ちなみに、システムに関するユーザーのマイルストーンはすべて、性別を持っています。たとえ、訪問者の性別を知らないサービスからユーザーに関する情報を削除したとしてもです。 これにより、性別によるソートが非常に面白くなり、周りの女の子を簡単に見ることができます。



API







最後に、サードパーティ開発者向けのAPIをリリースしました。http：//inmile.com/api_infoで入手できます 。



現時点では、APIを介して受信できます-周りの人、場所、オファー、メッセージとストリームマーク、ユーザーと場所に関する情報、ストリームへのメッセージの送信。



APIを使用するには、api_idとapi_keyを取得し、それらを受信するために、info @ inmile.com宛てに、記載する必要のある手紙に記載する必要があります。

1.アプリケーションの名前（またはテスト名）。

2.連絡先の電子メール（APIの重要な変更について通知するため）;

3.アプリケーションへの権限を確認した後、ユーザーを送信するリダイレクトへのリンク。 これは前提条件ではありません。 モバイルアプリケーションの場合、逆リダイレクトは関係ありません。



将来的には、開発者のオフィスが完成すると、この手順は自動的になります。



昨日、APIドキュメントは英語で表示されましたが、パートナー向けに作成しましたが、誰でも利用できます。



システムにはまだ大量のアクションのためのAPIがありますが、閉じられ、既存のユーザーベースを持つパートナーにのみ発行され、ユーザーベースを統合するように設計されています（たとえば、システムのユーザーが個人メッセージを外部システムに送信して回答を受け取ることができるように）。 興味があるなら、メールに書いてください。



近い将来、Inmileはいくつかの非常に人気のあるプロジェクトと統合され、パートナーは世界のいくつかの国で広告キャンペーンを開始します（国でプロジェクトを開発したい場合は、info @ inmile.comに条件を話し合う準備ができています） 。
