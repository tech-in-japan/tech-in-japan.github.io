Frontend Dev Conf 2016：会議のヒーロー、イベント、サプライズ

過去のイベント、つまり2週間後に少し息を吹き込んで、伝統的な会議Frontend Dev Confがどのように行われたかをペイントで伝える準備ができました。







4月16日、ミンスクは国際会議Frontend Dev Conf！を開催しました。 4回目は、彼女のリーダーシップの下で300人以上の参加者を団結させ、FDConfをフロントエンドパーティーで注目度の高いイベントにしました。















会場は珍しい会場-ミンスク映画館の1つです。 これにより、参加者は快適な状態、素晴らしい音、街の最大のスクリーンに関するレポートを得ることができました！































カンファレンスヒーローズ



ITイベントオーガナイザーのチームとプログラム委員会は、4年目にはミンスクのフロントエンドコミュニティの最も知識が豊富で好奇心representative盛な代表者であるディマドゥディン、ユラタテンコ、ヴォヴァダシュケヴィチ、イゴールミアスニコフ、ヴァシャヴァンチュクを参加者と喜んで参加しました。 。







会議プログラムは、「新参者」とすでに馴染みのある愛用者の両方でいっぱいでした。















ミンコ・ゲチェフは、Angular 2の魅力、そして彼が恐れてはならない理由と、その利点について話した。







FDConfの友人であるイリヤクリモフは、レポート「2016年のWebアプリケーションパフォーマンスの分析」と同様に、彼自身の魅力とユーモアのセンスで聴衆を喜ばせました。







ReactとReduxに関するレポートがないわけではありません。







誰もがロブ・アイゼンバーグに質問をすることができました。ロブ・アイゼンバーグは彼の新しい創造物であるオーレリアについて喜んで話しました。







参加者には、新しいフレームワーク、トレンドフレームワーク、...フレームワークを聞くだけでなく、Dartなどの言語についてさらに学ぶ機会もありました。 スピーカーのエフゲニー・グセフは、積極的かつ合理的に、「力の暗い側に行く」ように皆に促しました:)































イベント
