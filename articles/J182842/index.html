MS SQL Server 2012でのOLAPキューブの作成

こんにちは。



OLAPデータを分析するには、キューブを使用する必要がありました。 2001年から2002年にかけて、この技術を積極的に研究して使用し、さらには認定を受けました。 しかし、その後、彼は長い間彼女に触れませんでした。 そして、つい最近、2012 SQL Serverを使おうとしましたが、それはありませんでした。新しいサーバーに登場したものの多くは、私にはまったく馴染みのないものでした。



彼と知り合うのに少し時間を費やした後、コミュニティとスキルを共有することにしました。キューブと測定値の作成方法に関するトレーニングビデオを録画しました。



入門ビデオでは、おそらくいくつかのことが間違って行われていますが、この例は非常に機能しています。 読者（ビューアー）は、多次元キューブ、ディメンション、メジャー、およびその用途の概念に既に精通していることを前提としています。









この例では、MS SQL Server 2000のNorthwindデータベースを使用しました。トレーニングに必要なのは、小さくてシンプルで直感的な優れたデータベースです。 MicrosoftのWebサイト（ http://www.microsoft.com/en-us/download/details.aspx?id=23654 ）で入手できます。 SQL Server 2012に展開するには、パッケージをインストールし、データベース作成スクリプトファイルを見つけて、SQL Server 2012で実行する必要があります。
