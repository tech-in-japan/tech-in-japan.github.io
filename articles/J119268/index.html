開発世代2011：4つのノミネートと10万ドル

親愛なるhabravchane！ Softline Venture Partnersベンチャーキャピタルファンドは、Dev Generationソフトウェアおよびインターネットサービスのアイデアコンテストへの参加を希望しています。 Habré以外で、新鮮で非標準のプロジェクトを探す他の場所はどこですか？ コンテストのファイナリストに、資料を含むあらゆるサポートを提供します。



Dev Generationに送信された最高のプロジェクトには、Microsoft、REG.RU、ActiveCloudから賞金が贈られ、受賞者にはRuna Capitalから10万ドルの資金が提供されます。



賞品の全リスト：

•最高のプロジェクトの資金調達に10万ドル

•プロジェクトをホストおよび起動するためのActiveCloud Cloud Server

•REG.RUからのホスティングサービス用の5つの証明書

•ガレージで開始するための2つのバウチャー



一般的に、ロシア、CIS、またはバルト諸国からの独創的なビジネスアイデアを持っている学生や大学院生がコンテストに参加できます。 アプリケーションは、6月末までに送信できます。



コンテストWebサイト： www.softlinevp.com/devgeneration
