ブートコード



カーネルをロードするには、ローダーが呼び出すことができるコードを追加する必要があります。 boot.asm

 global start section .text bits 32 start: ;  `OK`   mov dword [0xb8000], 0x2f4b2f4f hlt


ここにはいくつかの新しいコマンドがあります：

  • bits 32

  • 実行可能部分には2つのセクションがあり.text
