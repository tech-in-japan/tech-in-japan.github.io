Sublime Text 2のデフォルトのホットキー

ダウンロード、無料、広告やSMSなし。

ここではすでにSublime Text 2のトピックについて多くのことが書かれています。これは非常に広範な機能を備えた本当に素晴らしいエディターです。 しかし、今、私は彼に次の称賛の歌を歌いたくありません。



ちょうど好奇心で、私はこのエディターでの作業をスピードアップできる重要な組み合わせが何であるかを見て登りました。 そして、あらゆる種類のプラグインと独立したマクロ/スニペットを除外して、デフォルトでホットキーにどれほど便利なことが行われたかに非常に驚きました。





たとえば、私は次のことに満足しました。



alt +左またはalt +右 -これはcamelCaseの単語の動きです。

ctrl + m-選択解除。

ctrl + shift + l-選択を行ごとの複数選択に分割します。

alt + shift + w -htmlタグを挿入し、すぐにその名前を複数選択して別のタグに変更できるようにします。

ctrl + k、ctrl + c-カーソルのある行を垂直方向に中央揃えにします。

ctrl + k、ctrl + t-ドキュメント内のすべてのhtmlタグのすべての属性を折りたたみます。

F9-選択した行をアルファベット順に並べ替えます。



そして、まだたくさん食べることがあります。 そこで、私はすべての既存の組み合わせ（少なくともFindKeyConflictsプラグインによってもたらされたもの）を選択し、それらを説明し、必要な組み合わせを学ぶことにしました。 そして、このファイルを一般に転送するのはいいことだと思いました。 「ピアニストをscるな、彼はできる限りベストを尽くす」-彼は元々自分自身について説明したので。 不正確な部分を見つけたら、PMで報告してください。ファイルを修正して更新します。



Sublime Text 2ですべての組み合わせをテストしましたが、何かがうまくいかない場合は、ノートを使ってこれらの「機能しない」組み合わせを作成しました。 それぞれに独自の設定とプラグインがあり、プラグインはデフォルトのホットキーを独自のものに変更できることに注意してください。 しかし、個人的には、これがプラグインのホットキーをやり直して、基本的なものと重複しないように（多くはありませんが）やり直したいと思いました。
