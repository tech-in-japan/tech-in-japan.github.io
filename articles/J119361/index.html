HDTVの実装要素

現在、テレビサービスの市場は、地上波、衛星、ケーブルタイプのテレビに代表されています。 「デジタルテレビ」は、Web-TV（ユニキャスト）とIPTV（マルチキャスト）の実装に基づいて情報技術に関連付けられることがよくあります。 ただし、高品質のHDTV（High-Definition Television）のテレビ放送の標準セットは、ケーブルだけでなく衛星ネットワークにも焦点を当てています。 高解像度テレビシステムは、1950年代に主要な産業大国（米国、日本など）および汎ヨーロッパコンソーシアムによって開発されました。 ソ連でも同様の開発が行われました。



1950年代後半、モスクワ研究所テレビ研究所（MNITI）でトランスフォーマー実験軍事本部通信システムが作成され、1125行の解像度で画像を送信できました[2]。



現在、HDTVはデジタル形式で放送されているため、十分なレベルの品質（QoS）、つまり十分な幅（MPEG-2の場合は15-25 Mbit / sまたは8-12 Mbit / s）のほぼすべてのデジタルチャネルがコンテンツ転送に適しています） s MPEG-4の場合-圧縮の程度に応じて）、一定の許容レベルの信号遅延を保証します（1〜10秒、受信バッファのサイズと信号遅延の要件に応じて）。



テレビサービス市場における新しい機能とサービスの実装は、革新的な技術に基づいて可能になりつつあります。 また、特にイノベーションプロセスと個々のテレビサービスの開発は、さまざまな技術的、組織的、政治的、経済的要因の影響を受けます。



現代のエレクトロニクス市場は非常に飽和しており、新しいサービスと新しい機器の需要が急増していることは注目に値します。 3Dおよびホログラフィックテレビ技術のみがテレビサービス市場を真に復活させることができる可能性があります。 ただし、これらの技術は現在、錯視に基づいています。 特別な色、偏光、スライド、メガネの使用を必要とし、面白いおもちゃにすぎないと認識されています。 したがって、ホログラフィックテクノロジーと3DTVテクノロジーの両方の現在の開発レベルは、革新的で戦略的に重要ではなく、中間的で進化的であると定義できます。 映画館では3Dを簡単に修正できますが、すぐに「すべての祖母」の家に届きます。 イノベーションプロセスのダイナミクス、技術的能力、およびサービスの需要のダイナミクスは部分的にしか一致しないため、メーカーはそれを人為的に作成し、古いデバイスと形式の生産とサポートを徐々に停止します。 そのため、最近の推定によると、小売チェーンで紹介されているテレビの約80〜90％がHDをサポートしています[4]。 消費者は、新しいサービスの準備が整った機器を、時には疑いなく購入することがわかりました。 ただし、特定のHDTVサービスを使用できる場合でも、潜在的な消費者はこの機能を「熱意なく」認識します。 表の中。 図1は、スベルドロフスク地域の事業者によって提供されるSD（標準解像度）およびHDチャネルの比較コスト特性を示しています。 消費者は、標準の機会に限定されることを好む場合が多く、高品質のサービスを受信するよりも5〜10倍低い価格でサービスを取得しますが、より高価です。

HDTVの開発を決定する技術的要因には次のものがあります。

•テレビ受信機の新しい要件（HD対応/フルHD形式）。

•コンテンツ要件の変更（HDコンテンツの制作者は非常に少なく、「通常」よりも1.5〜2倍高い）。

•ネットワークインフラストラクチャの改善（HDTVの実装には、「ラストマイル」で8〜10 Mb / sの速度が必要です。5〜10 Mb / sのアクセスネットワークと同じ回線でのインターネットアクセスがギガビットである必要があります）。

•新しいセットトップボックスの登場（HD、MPEG-4のサポート）。



HDTVを放送する最も簡単な方法は、HFC技術に携わるオペレーター、つまり同軸ケーブルを介してデータを送信するオペレーター向けです。 他の有線技術に対するHFCの主な利点は、ネットワークのすべての部分で最大の潜在的なブロードバンドを持ち、HD信号の送信に必要な速度帯域を提供できることです。 同時に、1つのHD-TVチャネルが占有する周波数帯域は8〜12 Mbit / sです。 HFCネットワークの再編成は必要ありません。 この分野の他の技術に関しては、IPTVおよびイーサネットネットワークでは、HDコンテンツのブロードキャストを編成するためにいくつかの努力が必要です。



IPTVネットワークでHDコンテンツのブロードキャストを編成する場合、状況は根本的に異なります。 サービスの品質を確保するには、ある程度の努力が必要です。 オペレータは、イーサネットネットワークのバックボーンおよびダウンストリーム（サブスクライバアクセスセクション、「ラストマイル」チャネル）セグメントで、少なくとも150 Mbpsの必要なQoSの周波数帯域を割り当てる必要があります。 ここでは、インターネットアクセスの組織にもこのような帯域を与えることができるという事実を考慮する必要があります。 今日のHDTVの収益は、インターネットサービスの収益性に匹敵しません。



一方、HDでサービスを提供する大規模な事業者の中には、HDTVの開発がいくつかの要因によってある程度妨げられていると言う人もいます。 過去5年間（おそらくそれ以上）に販売されたほとんどのテレビは、パスポートデータによると現在使用されていません。 16：9のテレビで4つのショーを視聴し、720のテレビで576（625）ライン、さらには1080ラインを視聴することは、あまり合理的な投資ではありません。 さらに、開発が妨げられます：高価な入場チケット（HDTVを視聴するには、適切なセットトップボックスが必要です。このセットトップボックスはより高価であり、パッケージの月額料金ははるかに高くなります）、およびさまざまなオファー。 さらに、「Real HD」を表示するテレビチャンネルはほとんどありません。 多くの材料がSDから変換されます。 もちろん、品質はこれから向上しません。



ロシアの国家HDTVの発展を著しく妨げる政治的要因の1つは、コンテンツの主要部分が4：3形式であり、ITUが勧告で、すべての国に対して統一された16：9フレーム形式を提案したことです。 可能な節約のために、テレビジョン研究所は15：9のフレーム形式でGOST R 53533-2009を提案、開発、承認しました。 したがって、15：9フレームのロシア語形式のHDテレビは、画面の下部にある追加のストリップを画面に追加することで生成されます（図1）。

コンテンツ制作者は、国家HD標準の実装で問題に直面する場合があります。 プログラム制作の観点からは、問題が発生します。このフレーム形式は、画像を生成する機器、つまりカメラ、ビデオレコーダー、編集機器ではサポートされていません。 新しいGOSTに再フォーマットすると、追加コストが発生します。 現在、ほとんどの放送局は衛星から直接HDチャンネルを取得して消費者に配信することができますが、翻訳を補完するために、衛星から元のバージョンを取得して再フォーマットし、再び衛星に取得してこの形式でロシアの消費者に配信する必要があります。

したがって、テレビサービス市場の発展における有望な方向性は、高解像度テレビ放送の高解像度テレビネットワークの作成であり、これは困難にもかかわらず、近い将来に主要なテレビサービスの1つになると考えられます。
