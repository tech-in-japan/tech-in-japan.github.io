ONYX C63MLアクニンブック-「ファンブック」ファミリーの最初のツバメ

親愛なる友人たちこんにちは！

Boris Akuninとして様式化された最初の電子ファンブックONYX C63ML Akunin Bookについて知りたいと思います。





著者の様式と装飾に加えて、この本には多くの魅力的なソリューションがあります：ユニークなソフトウェア、バックライト付きの静電容量式タッチスクリーン、Androidオペレーティングシステム、キットの品質ケース。 レビューには、この電子書籍の操作の詳細な写真、テスト、印象が含まれています。





>はじめに



彼の脚本に従って撮影された映画「トルコのギャンビット」を見た後、私は数年前に作家と脚本家のボリス・アクニンの作品に精通しました。 ボリスアクニンには多くのファンがおり、彼らはこの電子書籍のデザインを気に入っています。

このようなアンティークの様式化された電子書籍には特別な魅力があります。 著者のデザインとeBook Application LLCのユニークなソフトウェアの両方は、著者のユニークな歴史と探偵の世界への浸透感を生み出すように設計されています。

しかし、この本はボリス・アクニンの作品のためだけのものだと考えるのは間違いです。 ONYX C63ML Akunin Bookは、さまざまな形式の本を簡単に開くことができます。





>仕様



プロセッサー： Rockchip 2906、1 GHz

オペレーティングシステム： Android 2.3.1

ランダムアクセスメモリ（RAM）： 512 Mb;

使用可能なメモリ： 4 GB +最大32 GBのmicroSDスロット。

ディスプレイのサイズ、タイプ、解像度： 6インチ、E-Ink Pearl HD、タッチ（容量性）、16階調のグレー、マルチタッチ、MoonLightバックライト、1024x758。

ネットワーク： Wi-Fi 802.11 b / g / n;

バッテリー容量： 1700 mAh（Li-ion）;

オプション： USB 2.0。 テキスト形式：TXT、DOC、PDF、FB2（.ZIP）、EPUB、DJVU、RTF、ZIP、RAR、HTML、CHM、CBR、CBZ。 グラフィック形式：JPG、PNG。 オーディオ形式：MP3; オーディオ：ヘッドフォン用3.5 mm

寸法： 170 x 117 x 8.7 mm

カバーなしの重量： 169 g。



>オプション



厚い段ボールの箱、本の表紙、およびユーザーマニュアルは、作家の歴史的および探偵的な仕事のために芸術的に様式化されています。





オプション：電子書籍、カバー、充電器、USBデータケーブル、マニュアル、保証書。





出力充電電流が1000 mAの220Vネットワークの独自の充電器は、3時間でバッテリーを完全に充電し、充電中に過熱しません。





USBデータケーブルも長さ80 cmのハードではなく、高品質です。microUSBコネクタは隙間なく端にぴったりとフィットします。





>設計と人間工学



電子ブックは1つのカラースキームで発行されます。ブラックです。 ケースはソフトタッチのプラスチックでできています。 デバイスの寸法は170 x 117 x 8.7 mm、重量は169グラムです。





フロントパネルには、6インチのタッチスクリーン、ホームボタン、ページめくりボタンがあります。 画面の端に沿って全身に沿って、小さな巻き毛のくぼみを見ることができます。







製造業者のロゴ、モデルの名前、いくつかの技術情報、および箱のコードに一致するシリアル番号が背面パネルに適用されます。





右端と左端には、ページをめくるための機械的なボタンがあります





オン/オフボタンとデバイスを再起動するための穴は上端にあります（3週間のテスト中は必要ありませんでした）。





イベントのインジケータとしても機能するオン/オフボタン：充電中はオレンジ色のLEDが常にオンになり、青色が点滅し、充電が完了すると青色が点灯します。





下部には、3.5 mm標準ヘッドフォン用のオーディオ出力、microUSBコネクタ、microSDカードスロットがあります。





ボックスからのオーディオ出力は、小さな保護ゴム製プラグで閉じられています。これは、従来の電源ボタンで簡単に取り外すことができます。





そのサイズにもかかわらず、本は手に持って快適で、すぐに慣れます。 すべてのコントロールキーは高品質で作られており、タイトな乗り心地を持っています。





>カバー



ユニークな著者の外側のデザインの表紙に注目したい。 それは高品質の人工皮革で作られ、便利な人間工学に基づいており、縫い付けられた磁気ストリップは本を閉じるときに左右の部分を固定します。 閉じた後、本を開くとスリープモードに自動的に移行します。 カバーカバーの内側では、リーダーは4つのプラスチック製ラッチに固定されており、偶発的な紛失を確実に防ぎます。







>スクリーン



ONYX C63ML Akunin Bookは、6インチの静電容量式タッチスクリーンE-Ink Pearl HDディスプレイとMOON Lightバックライトを備えています。 解像度は1024x758ピクセルで、ピクセル密度は200 ppiです。 グレーの階調数は16、コントラストは12：1です。 スクリーンにはガラスベースがあります。 2タッチマルチタッチ、レスポンシブ。





メカニカルホームボタンを2秒間押すと、バックライトをオンまたはオフにできます。 デフォルトでは、バックライトは40％に設定されており、バックライトレベルは、画面の下部にあるコントロールメニューでいつでも調整できます。





>鉄



電子書籍は、クロック周波数1 GHzのRockchip 2906プロセッサ上に構築されています。 これに加えて、デバイスには512 MBのRAMと4 GBの内部メモリがあり、microSDメモリカードを使用して最大32 GBまで拡張できます。

Rockchip 2906プロセッサはエネルギー効率に優れています。 タスクに応じて、必要な頻度で動作します。 また、完全に深い眠りに入ります。





Iron ONYX C63ML Akunin Bookは、そのタスク、PDFの膨大な本、数秒でなくてもすぐに開いた写真や写真で過負荷になったDjVuに完全に対応します。 人気のfb2形式の書籍はすぐに開きます。

>インターフェースとソフトウェア



この本には、eBook Application LLCが開発した独自のオーサリングインターフェイスがあります。 すべての図面は、アーティストのイゴール・サクロフによって作られています。





この本では、テキスト、画像、音声ファイルなど、一般的な形式のほとんどを開くことができます。







読書中に、各本に個別に保存される追加設定を柔軟に変更できます。 例：方向、インデント、間隔、フォントサイズなど。

この本には人気のあるフォントがプリインストールされており、カスタムフォントもインストールできます。 リーダーにプリインストールされていない特定のフォントで本を読みたい場合は、リーダーの内部メモリに作成されたフォントフォルダー（小さな文字付き）にフォントをコピーしてインストールできます。 TTFフォントを使用します。





テキストで直接、写真と著者のメモを開くことができます。





この本は、最近の本を開いたり読んだりした履歴、およびこれらの本のテキストに設定されたブックマークを記憶しています。





メインページで著者のブログにアクセスすると、10のカテゴリでお気に入りの本のトップ10を見ることができます。





前述のとおり、システムストア、ゲーム、言語学習プログラムなど、Googleストアからアプリケーションをインストールしてインストールすることで、書籍の機能を拡張できます。





メインページでは、基本的なデバイス設定と読み取り設定が常に利用できます。 たとえば、目覚めたときの動作を構成できます。最後に開いた本から開始するか、メイン画面を開き、バックライト付きまたはバックライトなしで開きます。





Akunin Bookを使用すると、組み込みのブラウザを介してインターネットにアクセスできます。 それを通じて、ニュースを読んだり、ブログやフォーラムを読んだり、メールを見ることができます。 必要に応じて、メール用の他のブラウザまたはアプリケーション、ソーシャルをインストールできます。 ネットワーク。





辞書については個別に言及したいと思いますが、en-ru辞書はデフォルトですでに有効になっています。 ユーザーは、インターネットからStarDict辞書を追加できます。 ダウンロードしたアーカイブには、3つのファイルが含まれている必要があります：dict.dz、idx、およびifo形式の辞書の名前。 次に、これら3つのファイルでzipアーカイブを作成し、ファイル拡張子をzipからdictに変更し、書籍を内部メモリにコピーする必要があります。
