アメリカからロシアへのインターネットプロジェクトのコンテストのレビュー

インターネットプロジェクトを開発していますか？ 投資家をお探しですか？ 高品質のプロジェクトの専門知識とビジネス開発のための推奨事項を取得しますか？ オンラインコミュニティから新しいパートナー、顧客、認知度をお探しですか？ 次に、このコンテストの選択はあなたのためです。



マスチャレンジ （アメリカ） -http://masschallenge.org/





世界最大のスタートアップコンテスト。

競争の主な目的は、最高のプロジェクトを選択し、アイデアの実施を支援することです。 適用に制限はありません。

100人のファイナリストには、プロジェクトの実施に必要なすべてのリソースが提供され、メンターが付きます。

賞金-100万ドル

プロジェクト選択ファネル：





La Web （フランス） -http://leweb.net/





LeWebは、インターネットで最も影響力のある視聴者を集めています。 起業家、企業幹部、投資家、マスコミ、ブロガーが一堂に会し、ウェブ市場の主要な問題と機会に焦点を合わせます。

LeWeb '11 Startup Competition-会議の枠組み内のスタートアップコンペティション。 ソーシャル/ローカル/モバイルマーケット（SoLoMo）向けのアプリケーションと製品を作成する3つの最高のプロジェクトが決勝に進みます。

締め切り申請-2011年10月15日

賞金プール-100万ユーロ



スタートアップサウナ （フィンランド） -http://startupsauna.com/





スタートアップサウナは、開発の初期段階にあり、初期資金を探しているテクノロジースタートアップに焦点を当てており、市場への参入方法を検討しています。

このイベントは、Aalto Venture Garage（北ヨーロッパのチームをホストするビジネスインキュベーター）とAalto Entrepreneurship Society（北ヨーロッパの学生や初心者デベロッパーが成功するスタートアップを作成するのを支援するコミュニティ）によって開催されます。 スタートアップサウナは、北欧、バルト諸国、ロシアで最も有望なスタートアップを選び、2011年10月31日に始まるヘルシンキでの集中的な6週間のAalto Venture Garageプログラムに最適なチームを結集します。

賞金-各都市の最高のチームは、Yコンビネータープログラムに基づくヘルシンキでのトレーニングのための助成金と、費用1000ユーロを受け取ります。



Biz Barcelona（スペイン）の一部としてのグローバルスタートアップコンペティション-http ://www.bizbarcelona.com/





バルセロナの起業家フォーラム。 主なタスクは、起業家がアイデアを具体化し、ビジネスの新しい領域を見つけるのを支援することです。 スペインの主要なイベントは、毎年約1万4千人が訪れています。

グローバルスタートアップコンペティションはフォーラムの枠組み内で開催され、審査員はシード段階でプロジェクトを選択します。 受賞者には賞金と、世界中の投資家にプロジェクトを紹介する機会が与えられます。



フォーラムの詳細については、Habréに投稿しました。



Web Ready （ロシア） -http://www.web-ready.ru/





投資フォーラムの形式で決勝戦を行う、現代のインターネット技術分野のプロジェクトに対するロシア最大の競争。 競争の主な目的は、インターネットテクノロジーの分野における新興企業の発展を特定および促進し、プロジェクトと投資家間の交渉と取引の締結のためのプラットフォームを作成することです。



コンテストへの応募は、主に2つのカテゴリに分けられます。

•「シード段階で最高のプロジェクト」-ビジネスアイデア、ビジネスプラン、またはプロトタイプ、

一般公開されていません。

•「ベストスタートアップ」-収益化とスケーリングのためのビジネスプロセスの形成の初期段階にあるインターネットで開始されたプロジェクト。



今年、コンペのパートナーは、独自の追加ノミネートを確立しました。

•「Game Ready」-Alawar Entertainmentのゲーム開発者向けノミネート

•クラウドコンピューティング、データ仮想化などの分野のプロジェクトに対するEMCベンチャーからの指名



賞金-150万ルーブル

申込期限-2011年10月18日



要約すると、コンテストのあるプロジェクトへの参加は、賞金プールを競う機会を与えるだけでなく、投資家やビジネスエンジェルにプロジェクトを提示するだけでなく、ビジネスの質の証でもあります。企業、ベンチャーキャピタルファンドの代表者、ビジネスエンジェル。
