普通の大学の学生向けのソフトウェアのセット

プログラマーとしての私の主な仕事に加えて、いわば小さな趣味もあります。私は母国の部門で管理者としてパートタイムで働いています。 そして今、今日は休みですが、私は学部長の要請で大学に行き、数台の車で「ウェンドゥを広め」ました。 「拡散」の過程で、私は小さな質問をしました。一般的な生活からではなく、力学と数学の学部の現代の学生は何を必要としますか？ターミナルクラスにはどんなソフトウェアが必要ですか？ 考えたこともなかったのではなく、リストを作成するのに十分な時間がなかっただけです。 私の考えを述べさせてください。



OS



まず、オペレーティングシステムが必ず必要です。 * nixまたはwindows？ ここではすべてが教師の希望と彼らがそこで見たいソフトウェアにかかっています。 そのため、Windows XPのライセンスコピーの数がn番目である場合、それを具体的にインストールします。



オフィスソフトウェア



さて、おそらくあなたはあらゆる種類の電子文書を扱うためのソフトウェアに行くことができます。 繰り返しになりますが、フリーソフトウェアと非フリーソフトウェアの選択肢があります。 そして再び、私たちは学部で行われる一連の学問に背を向けます。 具体的には、いずれかの分野では、VBAにいくつかのクラスがあり、その実装はMS Officeに組み込まれています。 先生は他のことに同意しないと思います。 したがって、私たちの選択はMS Officeです。

さらに、画像の表示、PDFの表示と編集のためのさまざまなアプリケーションを配置します。これらのアプリケーションには、有料と無料の両方のカートと小さなカートがあります。



マルチメディアを使用する



カリキュラムで定義されているいくつかの分野では、学生は画像、3Dグラフィックスを扱う方法を学びます。 しかし、キャッチは、これらのコースがメソッド自体ではなく、特定のソフトウェア製品に焦点を合わせていることです。 その結果、何が得られますか？ Adobe Photoshop、3D Studio Max、Corel Drawがあります。



基本を学ぶ* nix



学部の主なコースの1つは、もちろんnixシステムの研究です。もちろん、理論だけでなく、実際にもです。 今、最も興味深い点は、利益のために* nixのようなシステムをインストールするか、その最愛の人を仮想マシンにプッシュすることです。 自分自身と学生自身に不必要な問題を引き起こさないために、OSを仮想マシンに配置することが決定されました。 たとえば、Virtual Boxはかなりシンプルで機能的なシステムです。 OS自体として、最も簡単にインストールされて動作するものを選択できます。これは、Linuxを目にしたことがない人もいるからです。 それで、おそらく、Ubuntuを配置します。 コースのコースでは、QTまたはGTK用の複雑なアプリケーションを作成するのではなく、単純なスクリプトを作成するのが実践であるため、どのバージョンでもかまいません。



開発環境



そして今、選択。 開発環境、またはPascal、Object Pascal、Prolog、Lisp、C、C ++、C＃、VBの言語でアプリケーションを作成できるいくつかの環境を配置する必要があります。 少なくとも、彼らは学部でそれ以上教えていません。 だから、順番に。



パスカル


ご存知のように、この言語は基礎を学ぶのに非常に適しています;それらの多くは学校や大学の最初の年に教えられます。 気にしなくても、おなじみのBlue-TurboPascalをすべての人に提供するでしょう。



オブジェクトパスカル


私が知っている、Object Pascalで記述できる唯一のIDEはDelphiです。 当部門は幸運にもBorland Developer Studio 2006のいくつかのライセンスを取得できたため、具体的にインストールします。



プロローグ、Lisp


このような言語の場合、Visual PrologやVisual Lispよりも便利な環境はわかりません。



C、C ++、C＃、VB


だから、私たちは最も甘いものに渡します。 トレーニング中に部分的に合格するいくつかの主要言語。 Visual StudioとSharpDevelopには選択肢がほとんどなく、「一度にすべて」があります。 強く気にしません...私たちはこれとそれの両方を置きます。



他の言語に関しては、例えば：PHP、Ruby。 私たちの部署では、彼らは教師に望まない、または他の何かに対処しませんが、結果は同じです。



DBを使用する



Visual Studioと共にデフォルトでExpressエディション2005または2008 MS SQL Serverもあり、これは大規模なプロジェクトを作成するのに十分なので、大学ではこれで十分です。

すべてのようです。 何かを忘れてしまった場合は、思い出してください。さもなければ、この学年では、仲間の学生はソフトウェアなしで放置されます=）
