ビルド2015：Visual Studio for OS XおよびLinux、Windows 10、クラウドサービス、その他マイクロソフトの新製品





Build 2015カンファレンスのオープニングが終了したばかりで、Microsoftは毎年、開発者向けの最新製品をデモしています。



今回は、伝統的に、いくつかの興味深い発表がありました。



Build 2015カンファレンスは4月29日から5月1日までサンフランシスコで開催されることを思い出させてください。 私たちは、イベントと録画のオンラインブロードキャストを利用できます。



オープニングでは、会社の製品開発のすべての主要分野で新製品が発表されました。

  • Microsoft Azureクラウドテクノロジー
  • Visual Studio開発ツールと新しいVisual Studioコード
  • Office製品とAPI
  • Windows 10プラットフォームとデバイス


いくつかの発表はすでに開発者にとって本当に驚きとなり、インターネットやコミュニティで活発に議論されています。



新製品と新技術に出会おう！



新しいMicrosoft Azureサービス



会議のオープニングの大部分は、新しいクラウドサービスと機会に充てられました。



開発者は、待望のDockerコンテナとの統合を発表しました。 DockerをLinuxだけでなくWindowsでも使用できるようになりました。 .NETアプリケーションのサーバーとして使用するオペレーティングシステムに関係なく、Visual Studioから直接プロジェクトをデバッグ、構成、および公開できます。







次に、動作中の新しいMicrosoft Azure App Serviceのデモを行いました-これは、Webサイト、モバイルアプリケーションのバックエンド、Web API、外部システムとやり取りする統合サービスなど、ほぼすべてのソリューションまたはその一部をクラウドにデプロイおよび構成できるサービスです。







クラウド内のサイトやモバイルバックエンドは視聴者を驚かせるほど簡単ではありませんが、新しいLogic Appsは統合ソリューションを作成する人々の注目に値します。 ビジュアルモードで、統合プロセス、条件を構成し、異なるシステムのフィールドを比較することもできます。







データの保存、処理、分析に多くの注意が払われました。 次のようなテクノロジ： Azure Data Lake 、 Azure SQL Data Elastic Database Pools 、 Azure SQL Data Warehouse 。



OS X、Linux、およびWindows用のVisual StudioおよびVisual Studio Code



Visual Studioは、おそらく多くの開発者のお気に入りの製品です。

Visual Studio Codeエディションがこの製品のラインに登場しました。これは、O OS X 、 Linux 、およびWindows用の.NETコードを操作するためのクロスプラットフォーム開発環境です。 開発環境はすでに無料でダウンロードできます 。



Visual Studio Codeは、オートコンプリート、スニペット、一般的なバージョン管理システムのサポートなど、最新のエディターに存在するすべての基本機能を既に実装しています。 エディターは、JavaScript、TypeScript、Node.js、C＃、ASP.NET 5などのいくつかの言語とテクノロジーをサポートしています。



OS XおよびUbuntuの新しい開発環境は次のようになります。











Visual Studio for Windowsを忘れないでください。 既に、無料版Community Editionを含むVisual Studio 2015 RCをダウンロードしてインストールできます。



Office 365の新しいAPI



世界中の膨大な数の人々がマイクロソフトオフィス製品を使用しています。 開発者は、Office 365でサービスを統合したり、アプリケーションを作成したり、データを受信したりする新しい機会を手に入れました。 Office Graphを使用してOffice 365にサービスを統合するか、 REST APIを介してサービスのデータを取得できます 。











Windows 10プラットフォームとデバイス



今日、マイクロソフトは目標を共有しました。Windows10を搭載した10億台のデバイスを世界中の2〜3年でインストールすることです。 新しいオペレーティングシステムのリリース後、ユーザーは最初の1年間にWindows 10に完全に無料で切り替えることができます。



もう1つのニュースは、 企業の Windows アプリストアの出現です 。 これで、ストアをセットアップし、社内でアプリケーションを配布できます。







Windows 10のデバイスは、最もパーソナルなだけでなく、どこでもどのようなシナリオでも使用できるモバイルデバイスでもあります。 電話をモニターに接続し、キーボードとマウスを使用してアプリケーションを使用する機能により、電話とコンピューター間の回線が消去されます。
