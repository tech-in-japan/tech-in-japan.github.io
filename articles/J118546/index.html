Windows 8のいくつかの新機能

Windows 8のWeb「ビルド」に定期的に表示されるMicrosoftの隠された広告のおかげで、インターネットコミュニティは、開発されたシステムの新機能に対する秘密のベールを開くことができます。 最近登場したビルド7955は、M3以前の段階であり、すぐにMDLフォーラムのユーザーの注目を集めました。 スクロールすると、システムの新しい機能を説明する説明付きの興味深いスクリーンショットがいくつか見つかります。



Windows Liveを介したユーザープロファイルの同期




ユーザーのお気に入りの環境が常に仕事をするコンピューター上にあるように、Windows 8ではWindows Liveサービスを介してオペレーティングシステムの設定をローミングする可能性があります。



Windows 8インストールファイルを保存する場所を選択する




何かについてコメントすることは困難です。 唯一の不思議なことは、この機能を有効にするために、ファイルを保存するための新しい場所へのフルパスを入力することを提案し、いくつかの参照ボタンが表示されないことです。 おそらく、システムの「ビルド」の後半に表示されるでしょう。



ファイルをコピーする新しい方法




現在、Microsoftは、ファイルとフォルダーをウィンドウのアドレスバーのリンクシーケンスに直接ドラッグアンドドロップできるようにすることで、ファイルのコピープロセスを簡素化することを明らかにしています。



ロックされたデスクトップと壁紙




ここでも、すべてが非常に簡単です。 ロックされた「デスクトップ」の通常の抽象的な背景の代わりに、インストールされた壁紙を背景として保存できるようになりました。
