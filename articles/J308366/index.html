ロシアのオンライン広告市場の新しいプレーヤーは、Avitoと競争しようとしています。





Pronto Media CEOのNicholas Dadianiによると、自動車、不動産、求人広告、求人の市場は細分化されており、多くの地域プレーヤーが存在し、評価することは困難です。 「Avitoの外でさらに約30億ルーブルの市場ができたと仮定すると、2015年のRuNetの広告市場の合計サイズは約120億ルーブルになります」とDadiani は提案します。



ゴールドマン・サックスのアナリストは、2020年までにAvitoがロシアの市場で一般市場に参入し、市場の98％を占めると予測しています。 しかし、このセグメントは財務面で最小になります：そのボリュームは6100万ドル（約43億ルーブル）になりますが、オンラインクラシファイドのロシア市場全体はほぼ4倍に成長します-最大470億ルーブル（6億6700万ドル）ゴールドマン・サックスで。



ゴールドマン・サックスの専門家は、2020年のオンライン不動産広告のセグメントを2億4200万ドル（170億ルーブル）、自動車および求人広告-1億8200万ドル（約130億ルーブル）と推定しています。



新しい成功Avito



8月17日に、ロシアのオンライン分類サービスAvito.ruおよび不動産Webサイトdomofond.ruを所有するAvito ABの収益が86％増加し、29億ルーブル（4560万ドル）に達することが判明しました。 これを最初に報告したのは、Avitoの株主であるスウェーデンの投資ファンドVostok New Venturesでした。



昨年後半、南アフリカのメディア保有者であるNaspersは、Avitoの50.5％を購入し、その持分を17.4％から67.9％に増やしました。 この取引の一環として、同社は約2億4,000万ドルの口座の資金を含め、27億ドルと評価されました。 12月、Avitoは、株主への配当として、ほぼすべての現金、2億3,000万ドルを分配しました。







Vostok New Venturesはこの取引に参加せず、Avitoの13.3％の持分を保持しました。 6月末のAvitoのコストは24億8,000万ドルと推定されました。



すべての収益は、ロシアのサイトAvito.ruから受け取りました。 2016年第2四半期のEBITDAは、前年の50.5％、つまり7億9,400万ルーブルに対して、59.7％、つまり17億ルーブル（2,720万ドル）でした。



Avitoのページビューは300億に達し、56％がモバイルデバイスから表示されました。



前四半期、同社の収益も86％増加し、22億4,900万ルーブルになりました。



ユラ



8月10日、Mail.ruグループは、独自のオンラインクラシファイド広告サービスであるYulaの開発を初めて正式に発表しました。 これは実際の打ち上げから9か月後に起こりました。 「Yula」は、毎月250万人を超えるモバイルデバイスのユーザーを収集しています。 これは、会社の代表者から Kommersantに報告されました 。 MRGは、以前に開発されたユーザーの視聴者のために、サービスを開発したいと考えています。 したがって、同社は、将来的にAvitoと適切に競争するために、新しいプロジェクトに多額の投資を行う用意があります。



「ユラ」は、買い手と売り手のすぐ近くで商品を販売する方法と商品を購入する機能をユーザーに提供します。 「スマートフォンは写真の撮影、呼び出し、書き込みに役立つユニバーサルデバイスであり、デスクトップとは対照的にジオロケーションを決定できるため、モバイル市場に焦点を当てています」とYulaプロジェクトマネージャーのYegor Abramets氏は説明します。 「同時に、モバイルはオリジナルの写真をレイアウトするためのフィルターです。ウェブサイトから他の人の画像をコピーしてパソコンの広告に配置する方がはるかに簡単だからです。」



販売者は、ソーシャルネットワークのアカウントまたは携帯電話番号を使用してアプリケーションに登録し、製品の写真をアップロードして説明を入力すると、広告がストリームに表示されます。 このサービスは、内部メッセンジャーと電話を統合します。これにより、すべての問題を明確にし、予約することができます。



これまでのところ、Yulaは買い手と売り手にとって無料です。 「現在、視聴者の目標があります」と、Mail.ru Groupは、読み込まれた広告の統計情報を明らかにせずに述べています。



「Yula」の特徴は、Mail.ru Groupに注意を払い、ユーザーの位置と興味を考慮した広告の表示です。 広告の作成中に製品のジオロケーションも記録されるため、ユーザーのジオロケーションに基づく「Yula」は最も近いオファーを示します、とAbramets氏は言います。 彼によると、ソーシャルメディアSDKを使用すると、ユーザーがアクセスを許可した場合に、ユーザーの社会人口統計特性を取得できます。 「ソーシャルネットワークに基づいて、彼の興味に関連するカテゴリを提供しています」と彼は言いました。







Run Capitalの株主であるAndrei Romanenko氏によると、モビリティへの賭けには多くの競争上の利点があります。ジオロケーションの最適な提供は検索時間を大幅に節約するため、「完全に正当化」されます。 「これはコンバージョンにプラスの効果をもたらすはずです。 同社の製品ラインの機密サイトがミッシングリンクであったため、Mail.ru Groupとその大規模なユーザー層との提携を考えれば、プロジェクトは成功する可能性がすべてあります」と彼は言いました。 TNSによると、6月にMail.ruグループに含まれるプロジェクトの総聴衆は5610万人に達しました。



ロシア市場で議論の余地のないリーダーは、オンライン分類子Avitoです。 しかし同時に、Avitoはロシアのインターネット市場の好例でもあります。 2015年の秋に、同社はNaspersによって会社の50.5％の株式を取得する取引の一環として27億ドルと評価されました。 これは、MRGの総資本の約半分です。 NaspersはMail.ru Groupの株主でもあります。



Yulaアプリケーションの最初のバージョンは2015年10月15日にリリースされました。これは会社のメッセージに基づいています。 市場の情報筋によると、MRGはOdnoklassnikiおよびVKontakteネットワークのリソースを積極的に使用して、新しいサービスの視聴者を引き付けました。 それにもかかわらず、Avitoによると 、Avitoの背後にある「Yula」は10倍近くになります。 後者は、Public Opinion Foundationからのデータを引用して、聴衆は2000万人であると主張しています。



Yandex.Vertical



2016年1月、Yandexは広告以外の収益を2つの下位区分（Yandex.TaxiとYandex.Vertical広告サービス）に割り当てました。 サービスの提供は、リスクを多様化し、Yandexの収益構造の広告シェアを削減するのに役立ちます。



Yandex.Verticalには、すべてのYandex広告サービス（Auto.ru、Yandex.Real Estate、Yandex.Work、およびYandex.Travel）が含まれます。 Yandex.Verticalはロシア市場のAvitoのリーダーと競争する予定です。







次に、Yandex.VerticalのゼネラルディレクターであるAnton Zabannykh が RBCにこのことを伝えたものを示します。


2010年には、Yandex.AutoおよびYandex.Real Estateの集約サービスが開始されました。 彼らの目標は、インターネット上のすべての広告を自分自身に「バキューム」し、便利な検索を提供することでした。 このように、パートナーはトラフィックを受け取り、広告を通じて収益化しました。 現在、私たちは徐々にこのモデルから遠ざかりつつあります。市場は変化しており、他の形式は参加者にとって興味深いものになっています。



Avto.ruの購入により、以前のモデルは完全に放棄されました。トラフィックを配信するのではなく、購入者と通話を配信し、広告を集約しなくなりました。 Yandex.Real Estateにも同じ話があります。当初、サービスは広告のアグリゲーターでしたが、2014年12月に広告の直接受け入れを開始しました。 確かに、このアプローチには多くの問題がありますが、私たちはそれらを解決します。



キーは情報の品質です。 一般に、私たちが行うことはすべて人々にとって非常に不快です。 車を買いたいとしましょう。 良い中古車を探すよりも、新しい車を買う方が快適なシナリオのようです。 ただし、ブランドを決定すると、サロンで現在利用できる車をどこでも見ることができなくなります。 これらのサロンを回って、価格を出さずに各車のオプションのリストを忙しく読み上げるマネージャーと通信する必要があります。



中古車はさらに悪い。 物理的な世界からのいくつかのオブジェクトに関するものであると思われる発表がありますが、それらがどれほど真実であるかは一般に誰にもわかりません。 この不透明さ、この混乱を排除したいのです。 ただの掲示板になりたくはありません。客観的な選択手段になりたいです。
