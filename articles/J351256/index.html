MutationObserverは、最新のブラウザーが提供するWeb APIであり、DOMの変更を検出するように設計されています。 このAPIを使用すると、DOMノードの追加または削除、要素の属性の変更、またはテキストノードのテキストの変更などを観察できます。 なぜこれが必要なのですか？



MutationObserver



APIが非常に役立つ状況は数多くあります。 例：



  • Webアプリケーションのユーザーに、作業中のページで何らかの変更が発生したことを通知する必要があります。
  • DOMの変更に基づいてJavaScriptモジュールを動的にロードする、新しい興味深いJSフレームワークに取り組んでいます。
  • おそらく、WYSIWYGエディターで作業しており、元に戻す機能とやり直し機能を実装しようとしています。 API MutationObserver



    を使用すると、ページでどのような変更が発生したかをいつでも知ることができます。つまり、簡単に元に戻すことができます。
