Ubuntu Media Player

Ubuntu Linuxを実行するポータブルメディアプレーヤーは、中国企業のSmartDevices によって発売されました 。

4.3インチのタッチスクリーンを備えたモデルはSmartQ 5 MID（150ドル）で、7インチのスクリーンを備えたモデルはSmartQ 7（190ドル）です。







メディアプレーヤーの特性：

  • CPUSamsung S3C6410（ARM11）667MHz;
  • RAM 128MB DDR RAM、フラッシュ1GB;
  • microSDカード用のスロット。
  • 解像度800 X 480の4.3または7インチタッチスクリーン。
  • WiFi 802.11b / g;
  • Bluetooth
  • USB 2.0ホストポート
  • サイズ：SmartQ 5の場合は120 x 74 x 14 mm、SmartQ 7の寸法は非公開。
  • 重量150グラム SmartQ 5および250 gr。 SmartQ 7の場合;
  • バッテリー2000mAH（SmartQ 5）または4500mAH（SmartQ 7）。


どちらのモデルもほぼ同じで、BluetoothとWiFi、128 MBのRAM、1 GBのフラッシュメモリが搭載されています。 両方のモデルは、同じUbuntu Linuxディストリビューションで配布されます。
