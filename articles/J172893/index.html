ウェビナー「Microsoft Visual Studio Test Professional 2012およびTeam Foundation Server 2012によるアプリケーションテスト」

2012年3月20日、モスクワ時間11:00に、 ウェビナー「Microsoft Visual Studio Test Professional 2012およびTeam Foundation Server 2012を使用したアプリケーションのテスト」に参加してください 。



このウェビナーは、アプリケーションをテストするための効果的で透過的なプロセスを構築することに関心があり、チームとやり取りするための方法論またはツールを選択する段階にある学生を対象としています。



さらに、このイベントは、VS Test ProfessionalおよびTeam Foundation Sever（TFS）で既に作業しており、製品の機能とその使用方法について詳しく知りたいマネージャーにとって興味深いものになります。



プレゼンテーション中に、次の問題が提起されます。

-アプリケーションライフサイクル管理（ALM）プラットフォームの概要。

-TFSとTest Professionalの相互作用。

-テストの計画。

-手動テスト、シナリオテスト。

-研究テスト。

-新しいテストケースを記録します。

-自動ドキュメンテーションとバグ追跡。



Visual Studio Test Professional 2012は、テストエンジニア、ビジネスアナリスト、製品マネージャー、およびさまざまなグループのコラボレーションツールを必要とするが完全な統合開発環境を必要としないその他の関係者向けの製品です。



チームメンバーは、統合テストおよび製品管理ツールを使用して、開発プロセス全体で品質を保証できます。 Visual Studio Test Professional 2012は、要件の簡単な定義とクライアントからの継続的なフィードバックもサポートしているため、これらのプロセス内で追跡できます。



Team Foundation Server 2012は、Microsoft Application Lifecycle Management（ALM）を支えるコラボレーションプラットフォームです。



TFSを使用すると、チームはツールを使用して、要件、タスク、エラー、ソースコード、ビルド、テスト結果などのメンバーおよび作業項目のアクションを追跡できます。



このウェビナーは、開発プロセスの組織の専門家であるマイクロソフトのAlexander Yakovlevが実施します。



ウェビナーの期間は1時間です。

ウェビナーの事前登録が必要です！

ロシア語でデータを入力する必要があります！

登録後、参加者は詳細な手順が記載された手紙を受け取ります。
