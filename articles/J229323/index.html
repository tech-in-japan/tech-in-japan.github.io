Googleでの仕事の仕方

リックビスコミが何度か試みたが失敗したにもかかわらず、Googleで仕事を始めた経緯についておもしろい話を紹介します。 現在、リックはフロントエンド開発者としてYouTubeで作業を続けています。



この投稿は元々、Quora.comの質問に対する回答として掲載されました。 質問はかなり標準的で、次のように聞こえます。「FacebookまたはGoogleで採用されるのに十分なスキルセットを取得するために、6か月間どうやって管理するのですか？」 リックはこの質問に非常に徹底的に答え、約3000票を得ました。 私の意見では、彼の話は、彼の夢を達成するために示した途方もない忍耐力と意志で注目に値します。 そのような目標を達成するという希望を大事にしている人は皆、注意すべきだと思います。 私はすでに取りました:)実際、カットの下でのリックの答え。



リックの答え



私はあなたの気持ちを完全に理解しています。 私は約3年前にコンピューターサイエンスのBS学位（大体コンピューターサイエンスの学士号）を取得しましたが、私が望んでいたのはGoogleで働くことだけでした。 卒業する前に、電話インタビューを受け、ソフトウェアエンジニアの地位を求めるYouTubeのオンサイトインタビューに招待されました。 私はインタビューを受けました。 私は彼のもとを去り、自分が持っていたプロセスから素晴らしい気分になりました。そして、しばらくして、拒否について悲しいメッセージを受け取りました。 何がうまくいかなかったかを理解するのに長い時間と多くの考えが必要でした。



これは私の最初の技術的な現場インタビューでした。 当時、これに関する私の経験はすべて文字通りゼロでした。 思い出して、今私はすべてが驚くほど間違っていたことを理解しています。



堅実なインタビュー体験


面接はスキルであり、このスキルを磨くのは望ましくありませんが、それは最も重要です。 面接を重ねるほど、快適に感じることができます。 リラックスして自分の考えを明確に表現すると、他のすべてが続きます。



「あなた自身について何か教えてください」という一般的な質問にどう答えるかを考えてください。 何度もインタビューを重ねた結果、この質問は私よりもインタビュアーに関するものであることに気付きました。 あなたに何を期待しているのかを考えて答えを作ってください（ヒント：簡潔さと仕事への態度）。 それを宣伝して販売します。



1年後、広範なインタビューの経験を経て、私は再び履歴書を別のオフィスの同じ役職に送りました。 そして再び、私は電話インタビューが得意で、再び現場への招待を受けました。



次の2週間はホワイトボードと本を使って、スキルを強化しようとしました。 一晩中、アルゴリズムとデータ構造に取り組み、同じ間違いを繰り返さないようにしました。



私はインタビューのために来て、すべて出かけました。 私は自分自身を誇りに思い、難しい質問に答えました。 しかし、再び、私は拒否されました。



趣味をあなたの強みにしてください


これは、Googleへの道のりで学んだ最も重要なことの1つです。



私は何を間違えましたか？ 私は間違った仕事のために面接しました。 あなたと同じように、私はウェブ開発にとても情熱を傾けています。 私はCSの学位があれば、私の自然なキャリアの道はソフトウェア開発になると思いました。 私は、C ++またはJavaで毎日プログラミングする開発者になることを余儀なくされましたが、実際、私の興味はHTML / CSS / JavaScriptなどのフロントエンド技術の分野にありました。



私はこれを、ウェブ開発に注意を向け、そこでキャリアを積む必要があるという兆候として受け止めました。 プロの本を読んだり、本を買ったり、会議に参加したりしました。 Web開発者としてより多くを学び、より良くなるためにあらゆることをしました。



1年後、私は前回と同じオフィスに履歴書を送りましたが、ユーザーインターフェイス開発者（UIエンジニア）の立場になりました。 再度、電話インタビューをよくやりました。また、オンサイトへの招待状を受け取りました。 私の採用担当者は、3回目のオンサイトインタビューの後に戻ってきた人はほとんどいないと言いました。



繰り返しますが、私は数週間資料を勉強し、インタビューを受けました。 私は彼らを公園からノックアウトしたと感じました（翻訳者のメモ：公園からノックアウトすることは、フィールドから飛び出すようにボールを打つことを意味します、言い換えれば、ホームラン、野球の用語を作る）、そして一日の終わりに私のインタビュアーが戻ってきましたトロフィーとして「3回目のオンサイトインタビューを完了するための」Googleマグカップ。 さて、マグカップは私が再び拒否されたので、3番目の慰めの賞でした。



その瞬間、いくつかのイベントが発生しました。 あきらめたかった。 キャリアを変えたかった。 代わりに、私は1つの会社の1つの仕事に集中するのをやめ、自己啓発に集中しました。 インタビューで何がうまくいかなかったのか、どこでより良い答えが得られるのか、正確にはわかりません。 もう問題ではありません。 私は自分のベストを尽くす必要があります。



私は2つの基本的なキャリア決定を行いました。私が興味のある分野でオープンソースプロジェクトに取り組み始め、Webアプリケーションのパフォーマンスを最適化するためのすべての可能な資料を研究しようとしました。 私がその時に出席した会議を通して、私はニューヨークWebパフォーマンスMeetupグループを支援することにしました。 ジョブを別のジョブに切り替えました。これは特にWebパフォーマンスに重点を置いていました。 NY Meetup Groupの会議で話す機会を得た結果、すべてのWebパフォーマンス会議の母であるVelocityで話す機会が与えられました。 すべてが素晴らしかった。



かなりのレベルの前向きな姿勢を維持する


青からのボルトのように-私は最初のYouTubeリクルーターから手紙を受け取りました。 彼女の意見では、ウェブ開発者の立場についてでした。 私はその機会を逃さず、電話インタビューを経て、現場ラウンドに進みました。



私はカリフォルニアに戻り、なじみのある難しい質問をしました。 前回の3回の試みと同様に、5回のインタビューのうち1回、本当にうまくやりたいと思っています。 再実行のように、私はそれを数回前からすべて見ていたので、心配し始めました。



私の不安と確立された失敗のパターンに反して、私は実際に仕事を得ました。



だから、夢の仕事を追求する人々のために：

  1. 彼女を追いかけないでください。 あなたが実際に行く場所のために自分自身を準備するために得るとさえ思わない人々との100のインタビューをしてください。 技術的なスキルを売る前に、まず自分自身を売ることを学んでください。
  2. あなたのニッチを見つけます。 会社に提供できるものを決定します。 あなたの技術スキルを磨き、可能な限りのことをすべて実行して、勉強を続け、キャリアを前進させてください。
  3. あきらめないでください。 あなたを成功に導く道に身を置き、それに沿って進んでください。
