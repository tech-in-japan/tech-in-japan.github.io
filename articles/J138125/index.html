ギーたちがリポジトリを再発明しました。

平均的なLinuxユーザー



私たちは自分自身をアプリストアまたはWindows用のアプリストアと呼ぶことを好みます。 実は、それは問題ではありません。 私たちが行う主なことは、すでに他のプラットフォーム上にあるものです-Mac OS用のApp Store、Ubuntu Software Centerはどこにも知られていません。 そしてもちろん、すべてのモバイルプラットフォーム：iOS、Android、Windows Phone 7でも。



一般的に、「さらに」、「さらに」せずにアプリケーションをインストールできます。









このプロジェクトを開始するアイデアは、昨年の冬に登場しました。 NiniteやAllMyAppsなどの競合他社を見て、それらの長所と短所を分析した後 、カードとクイーンを使用してサイトを立ち上げることにしました。



Sooftyに会ってください。 現在、約60の人気のあるプログラムを公開しており、同じ数がモデレーションキューで検証を待機しています。

もちろん、他の国の数十万と比較することはできませんが、我々は成功するつもりです



上記のいずれにも遭遇していない人のために、Suftyを使用する典型的なケースを説明します。

  1. http://soofty.comからSuftyをダウンロード、インストール、実行します。
  2. カタログからプログラムを選択し、「詳細」をクリックします

  3. 説明を読み、スクリーンショットを見て、プログラムが私たちにとって興味深いと思う場合は、「インストール」をクリックします

  4. Soofty自体は、選択したプログラムをダウンロードしてインストールします。 このプロセスは、「ダウンロード」セクションから確認できます。



便利で速いですね。



近い将来、自動更新と、Windowsでの生活をより便利にするいくつかの機能を実現する予定です。



内部は複雑なものではありません。
  • Django / Nginx / Mysql / MongoDBのバックエンド。 ポニーパワード！
  • フロントエンドでは、jQuery、Backbone.js、Underscore.jsをmightとmainで使用します。
  • リソースの数が限られているため、クライアントはAwesomium + .NET 4で収集されました。これはもちろん不便ですが、最初のバージョンではまったく問題ありません。 Webkitを使用してQTに書き換える予定です。
