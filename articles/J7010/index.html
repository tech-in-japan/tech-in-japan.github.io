S3ホスティングは50億ファイルに成長しました

Amazon Web Services（AWS）の人生で初めて、この革新的なプロジェクトの開発を反映して、少なくともいくつかの数字が登場しました。 Jeff Bezosが、 S3ホスティングの開発について報告しました。S3ホスティングは、 EC2コンピューティングクラウドとともに、スタートアップにとって理想的なプラットフォームと考えられています。 Web 2.0カンファレンスで、Amazon.com帝国の創設者は、ホスティングボリュームが50億ファイルに成長したと述べましたが、昨年7月には、ホスティングに保存されたファイルは80万ファイルのみでした。 ピーク時には、サーバーは最大9億2000万件の読み取り/書き込み要求を処理します。 レコード-16607要求/秒。







確かに、Amazonの「インフラストラクチャ」サービスの1つはまだ会社に利益をもたらし始めていません。 ご存知のように、AWSサービスは2004年にオープンしました。 S3ホスティングとEC2コンピューティングクラウドに加えて、このパッケージには、Simple Queue Service（SQS）Webサービス、Mechanical Turk、Mechanical Turk、および他のいくつかのサービスのホスティングメッセージングサービスも含まれています。 過去3年間で、数千のインターネットスタートアップやMicrosoftなどの大企業でさえAWSの顧客になりました。



AWSの人気は、スタートアップ向けのホスティングのコストが非常に低いためです。 ジェフ・ベゾスは例を挙げます： Blue OriginはサイトがS3ホスティングを使用してビデオを保存し、1月に宇宙船を地球の軌道に打ち上げました。 ビデオはBlue Origin Webサイトから直接放送され、その日にS3インフラストラクチャは350万のWebページと758 GBのデータを出荷しました。 したがって、1月全体で、S3サービスの請求額はわずか304.23ドル（300ドル、23セント）でした。
