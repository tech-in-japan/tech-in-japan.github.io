DevDocs：すべての開発者向けドキュメントを1か所で、高速で便利なインターフェイスで





サイトdevdocs.ioは、フランスのプログラマーThibault Kurublのプロジェクトです。 最も人気のあるWebテクノロジー、フレームワーク、API、および他の多くの開発ツールに関するドキュメントを集めて整理しています。 DOM、HTML、JavaScript、jQuery、Node.js、PHP、Ruby、Python、Git、Angular、Backbone、CoffeScript、Less、Sass、Redisなど、すべてが同じスタイルで設計されており、検索はドキュメントデータベース全体で機能します。ファジーを含む。 見るのに必要な技術だけを選択することが可能です。 一般的に、DevDocsインターフェースは満足です-余分なものはなく、すべてが非常に明確で機能的であり、多くのキーボードショートカットが利用できます 。



DevDocsは完全にオープンなプロジェクトなので、 Github リポジトリからダウンロードしてローカルにインストールできます（Rubyで記述されています）。 このプロジェクトはまだ1年前ではありませんが（2013年6月18日に開始されました）、多くの追加のアメニティを獲得することができました。ChromeとFirefoxの拡張機能として、Sublime TextとBracketsのプラグインとして利用可能です。 OpenSearchをサポートし、ブラウザーの検索バーに統合します。



Trelloのプロジェクトボード上のDevDocsに新しいドキュメントセクションを追加するために投票できます。
