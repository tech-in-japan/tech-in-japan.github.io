FirefoxでのGmailメール暗号化

趣味、連絡先リスト、ビジネスアクティビティ、友人のリスト、資金の受け取り、ビジネスに関する情報など、ユーザーに関する情報を収集するおいしいサービスが数多くあります。 場合によっては、同じGmailがその人よりもその人についてより多くのことを知ることがありますが、トートロジーについては申し訳ありません。 もちろん、すべての正直な人を隠すことはないと思いますが、収集された情報が数千人に達すると、私たちはすでに考え始めています。



複数の人がいて、通信に企業ドメインに接続されたGoogle Appsを使用している組織があるとします 。 ドキュメント、パスワード、誕生日、スケジュール、 Googleトークを介したコミュニケーションの履歴、検索クエリの履歴、すべてのプランクトンアクティビティ、ほぼすべてのアクションがGoogleの深部に保存されます 。 Googleがなぜそんなに積極的に動いているのですか？ 彼は、人々をコンピューティングパワーおよび情報源として使用して、イベントを予測する方法を単に知っています。 わかりました Google-ロシアの地下では、競合他社の電子メールボックスのハッキングが文字通り10ドルで行われていますが、 .ruゾーンのメールボックスはすべて解読できます。 無料ゾーンsvetochka.ufzt [dot] mail.ru （アドレスは架空のもの）で電子メールボックスを開く信頼できる組織がいくつあり、パスワードは誕生日です。 そして、誰もが何らかの形で、電子通信の暗号化、電子署名について大いに忘れていました。 政府機関でさえ、国家は一般に嘆かわしく、すべてのコンピューターはウイルスに覆われすぎており、政府機関の電子メールはトロイの木馬とスパムの送信、クラスメートとの連絡、およびデータベースの更新と文書注文のコピーなどに使用され、これらのメールボックスにも届きます。 n。 同様のボックスが組織についてハッキングされた場合、私はほとんどすべてを見つけることができ、警備員のパックは助けにはなりません。 メールは暗号化されず、何によっても保護されません。



FirefoxでのGmailメール暗号化については、上記の妄想は導入のようなものです。

そのため、Gmailの受信トレイを使用して、暗号化とFirefoxブラウザをメインのツールとして使用する必要があります。

1. Windows用のダウンロードとインストール-gpg4win 、Linux用のパッケージのインストール-GnuPG （Ubuntu sudo apt-get install gnupg gpa ）、MacOS用-Mac GPG



2. Firefox- FireGPGの拡張機能をダウンロードしてインストールします。



3.セットアップウィザードウィンドウが表示されます。 この場合、PGPへのパスを入力します。

C：\ Program Files \ GNU \ GnuPG \ gpg2.exe





4.名前、メールボックス、パスワード、キーの有効期限、暗号化のタイプを入力します。





5.デフォルトのパラメーターを設定します。





今、私たちはメールボックスに行き、私たち自身に手紙を書きます。 [送信]をクリックすると、キーが選択されたウィンドウが表示されます。その後、Gmailのパスワードを入力する必要があります。









1通の手紙を受け取り、開いて、パスワードのあるウィンドウが表示されます。





手紙は正常に解読されました。





キーをコピーし、安全な方法で転送するためだけに残ります。



UPD：セキュリティを強化するには、Gmailで自動保存を無効にする必要があります。

Firefoxのツール-> FirePGP-> Gmail->

-自動保存を無効にする

-暗号化された文字に対する応答を暗号化する
