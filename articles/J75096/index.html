Chrome「私はラッキーだ」と思って検索

多くの人がFirefoxの素晴らしい機能を知っており、使用しています-検索クエリを検索バーではなくアドレスバーに直接書くと、検索結果から最初のサイトにすぐに転送されます（= Google検索の「幸運を感じています」ボタン） そして、クロムのこの機能を本当に見逃しました。









数回クリックするだけで幸せになる方法：



ブラウザの右上隅にある「Google Chromeの構成と管理」→「設定」





そして、そのようなパラメータを持つ「検索エンジン」を追加します





名前-アドレスバーに表示される名前： Google Lucky

キーワード-検索を開始するために入力するフレーズ： gg

URL- Google検索文字列： {google：baseURL}検索？{Google：RLZ} {google：acceptSuggestion} {google：originalQueryForSuggestion} sourceid = chrome＆ie = {inputEncoding}＆q =％s＆btnI =

（短くすることもできます： www.google.com/search?q=%s&btnI= ）



アドレスバーに「gg download python 2.5」と入力して入力します。 そして見よ！ すぐにPython 2.5のダウンロードページにジャンプしました！



PS毎回「gg」を入力するのが面倒な場合は、新たに追加した検索を「Googleラッキー」のデフォルトにすることができます（明らかに、検索エンジンのリストにある「デフォルト検索の作成」ボタン）。



PPSところで、ロシア語で検索するには、キーワード「pp」とまったく同じ「検索システム」を追加することをお勧めします。
