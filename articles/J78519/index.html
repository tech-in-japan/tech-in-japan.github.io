エカテリンブルクの開発者向けの2日間のイベント

先日、12月15日と16日に、エカテリンブルクでさまざまなクラスの開発者とITスペシャリスト向けのイベントが開催されました。 すべてのイベントはウラル州立大学（USU）で開催されました。



最初に、 SKB Konturは、有名なアジャイル開発の専門家Askhat Urazbaevによる無料の講義を開催しました。 講義のトピックは「 効果的なソフトウェア開発チームを構築するための柔軟な方法論 」と題され、優れたチームを作成する原則と、柔軟な方法論がこれをどのように支援するかについて説明しました。



Askhatは、ソフトウェア開発の組織における問題を詳細に説明し、アジャイル手法がそれらの解決に役立つ理由と方法を説明しました。 多くの例が検討され、特定のスクラムプラクティスが提示されました。 -30度の寒さとかなり遅い時間にもかかわらず、講義は多くの人々を集め、私の意見では非常に成功した。 個人的には、私はアジャイルでは動作しませんでした。アジャイルとスクラムがどのように機能し、何ができるかを知りたいと非常に興味がありました。



Askhatによる講義の後、 Ural .NETユーザーグループは、講義と同じ建物（下の階）で会いました。 予定されていた2つのレポートの代わりに、会議でレポートが1つしかなかったので、私は1つ作成しました。 私は自分のパフォーマンスをあえて評価しませんが、 ASP.NET 4.0の多数の革新について、1時間で最も詳しく説明しようとしました。 会議にはマイクロソフトのゲスト、Roman Zdebskyが出席しました。RomanZdebskyは、私の報告後にWPFについて質問されました。



補足として、プレゼンテーションのスライドとデモンストレーションのソースコードを投稿します。 ここからダウンロードしてください （pptx / ppt + zip）。



昨日、12月16日の2日目に、次のTechDaysイベントが開催されました 。これは、マイクロソフトによって全国で年に2回開催されます。 イベントは、いつものように、IT専門家と開発者向けのTechNetとDevDaysの2つの部分に分かれていました。



TechNetのメインで唯一のスピーカーは、マイクロソフトのスペシャリストであるAndrey Beshkovでした 。 Andreiのレポートにアクセスしたことがある人なら誰でも、これは非常に有能なスピーカーであり、非常に楽しくて耳を傾けることができることを知っています。 TechNetレポートには、次のトピックがありました。
  • Windows 7およびWindows Server 2008 R2の新機能 -新しいオペレーティングシステムのアーキテクチャの大幅な変更を説明する紹介レポート。 「Windows 7の新機能」という質問に苦しめられたり苦しめられたりした場合、このレポートで答えを得ることができます。
  • BranchCacheテクノロジーに基づくブランチネットワークの最適化-2番目のレポートは、Microsoftによる非常に興味深い新しい開発に当てられました。これは、HTTP（S）およびSMBプロトコルを介したデータへのアクセスを大幅に高速化するリモートオフィスの分散（または集中）キャッシュです。 一般に、システムはブランチレベルのローカルピアツーピアネットワークに似ており、本社からではなく、ローカルネットワーク上の最も近いコンピューターからデータを取得できます。 これはすべて、可能な限り安全で生産的です。 これは素晴らしいことであり、Andreiがデモで示したように、セットアップは簡単ですが、効率的に機能します。
  • DirectAccess：国境のない安全な企業ネットワーク-最後の3番目のレポートはVPNキラーに捧げられました-VPNなしで会社のローカルネットワークに接続でき、VPNよりもはるかに簡単なDirectAccessテクノロジー。 DirectAccessは、HTTPSへのアクセスが拒否されないプロバイダーを通じて機能します。 Andrewは、VPNを介したDirectAccessの多くの重要な利点、セットアップの容易さを示し、このテクノロジーの使用経験について語りました。 VPNはすぐにクラスとして消滅し、Win7 / Server2008 R2の配布を伴うDirectAccessはどこにでも実装されると感じました。
DevDays-TechDaysイベントの2番目の部分は、ソフトウェア開発専用でした。 Marat BakirovとRoman Zdebskyの2人のスピーカーがあり、マイクロソフトの代表とソフトウェア開発のスペシャリストの両方がいました。 レポートのトピックは次のとおりです。
  • 開発者向けマイクロソフトテクノロジーの紹介トーク： Romanによる現在と未来の読み物。 その中で、彼は、Microsoftプラットフォームでのソフトウェア開発の最新動向、過去6か月間に発生したニュースと発表について話しました。 たとえば、RomanはPDC会議からの発表に言及しました。 一般に、入門レポートは、Microsoftテクノロジースタック全体の大部分をざっと見たものでした。 そして、誰かが何かを知らなかった、または最近ニュースを逃した場合、彼は彼自身のためにレポートから多くを学ぶことができました。
  • Visual Studio 2010の可能性の概要 -Marat Bakirovによるより実質的なレポートは、最も人気のあるソフトウェア開発環境の最新リリースに専念しました。 新しいVisual Studioツール、それらが提供するもの、それらが作成された理由、およびそれらの使用方法が調査されました。 Visual Studio 2010 Beta 2に慣れていない場合は、その上で商用アプリケーションを開発することがすでに可能であるため、それを行う時が来ました。
  • 開発者向けのWindows 7機能というタイトルの最後の3番目のレポートもMaratが読みました。 その中で、彼はWindows 7が開発者にもたらすもの、彼のプロジェクトでこの新しい機能を実装する方法、そして最も重要なのはアプリケーションで新しいWindows 7機能をサポートする必要がある理由を例を挙げて明確に説明しました。 Maratは、Windows 7の多くの技術革新の利便性により、アプリケーションがより使いやすくなり、そのような機能のサポートの導入により競争力がもたらされることを強調しました。 特に興味深い部分は、マルチタッチテクノロジの実装専用の部分でした。このサポートは、示されているように、わずか10行のコードでWPFアプリケーションに追加できます。
全体として、今回はTechNet / DevDaysイベントが成功しました。 昨日、エカテリンブルクの天気がマイナス30度で風だったため、出席者は減らず、高かった。 明らかに、そのようなイベントの関連性は非常に高く、まだイベントに参加していない場合は、興味を持ってください。TechDaysもあなたの街で開催されるかもしれません。



過去2日間は非常に役に立ち、多くの資料、レポート、スピーチ、人々とのコミュニケーションがありました。 エカテリンブルクとその周辺地域の開発者は満足すべきだったと思います。 エカテリンブルクに住んでいて、まだそのようなイベントに参加していない場合は、ようこそ：来て、聞いて、話して、面白い人やゲストとチャットしてください。



すごかった！
