Picasaで美しいフラッシュギャラリーを作成する



最近では、いくつかの写真を選択して編集する必要がありました。 それはポートレートだったはずです。 考え直すことなく、私はこれをすべてフラッシュに実装する必要があると決めました。 シンプルで美しく、ほとんどの人がフラッシュプレーヤーを持っていると思います。



最初の考えは、フリッカーまたはパイカを添付できるギャラリーをダウンロードすることでした。 しかし、突然、ある良いサイトに出会いました。 ギャラリーテンプレートとそのデモがレイアウトされていました。 気に入ったテンプレートをダウンロードしたので、驚いたことに、readmeが見つかりませんでした...サイトを検索する必要がありました。 これらのテンプレートをpicasプログラムに挿入する必要があることがわかりました...そして、プログラムインターフェイスで既製のギャラリーを作成します。



そして今、すべての詳細について。



1）まず、picass自体をインストールする必要があります。 公式ウェブサイトpicasa.google.ruからダウンロードできます





2）次に、 www.paulvanroekel.nl / picasaにアクセスして、目的のテンプレートを選択してダウンロードします。





3）\ Program Files \ Google \ Picasa \ web \ templates \のアーカイブの内容を解凍します。各テンプレートは個別のフォルダーに配置する必要があります。





4）ギャラリーに追加する既製の画像を含むフォルダーを作成します。



5）そして今、最も興味深い。 picasプログラムで、「フォルダ」->「HTMLとしてエクスポート...」を選択します。さらに、すべてが簡単です。 必要なパラメーターを設定し、美しいフラッシュギャラリーを取得します。











その結果、私はそのような仕掛けを手に入れましたphoto.lev-a.spb.ru
