Spotify-1000万を超える合法的な音楽トラックを無料で*

Spotifyは、2006年にスウェーデンで設立され、2008年10月7日に公開された合法的な音楽リスニングサービスです。 有名なラベルの1,000万曲以上のトラックをユーザーが利用できます。 曲のベースは活発に成長しており、AC / DC、ビートルズ、メタリカ、レッドツェッペリン、ピンクフロイドなどのグループの音楽も利用できます。 現在、登録ユーザー数は100万人を超えています。

最近のイベント（ Last.fmの有料サブスクリプション）に照らして、Spotifyへの関心が著しく高まっています。

2009年2月10日までは、招待のみで無料で登録できました。 現在、サブスクリプションには3つのタイプがあります。

  • プレミアム（月額9.9ユーロ）
  • デイパス（1€）
  • 無料（スウェーデン、ノルウェー、フィンランド、イギリス、フランス、スペインのみ）








プログラムの概要



プログラムのインターフェイスは、iTunesをやや連想させます。



開発者が時代に遅れずについていき、最新のAppleアプリケーション（「大理石」テーマ）のスタイルで配色を使用しているのを見るのは素晴らしいことです。 クライアントは十分に高速に動作しますが、私が気づいた遅延を使用するのは数回だけです。 すべての音楽は、平均ビットレートが約160キロビット/秒のVorbis q5コーデックを使用してエンコードされます（ちなみに、last.fmよりも優れています）。したがって、256キロビットのチャンネルで十分です。 ピアツーピアの原則に従って動作します。音楽を聴くときは、キャッシュに保存され、必要に応じて他のユーザーに配布され、着信に応じた発信トラフィックを生成します。 その結果、インストール時に少なくとも500メガバイトの空き領域があると便利です。 プログラムの機能の中にはラジオがあります：



ご覧のとおり、さまざまな年の音楽のジャンルを選択するためのかなり便利な機能が提供されています（50年代以前から）。 プレイリストがサポートされており、既製のプレイリストを持つサイトもあります-1、2、3、4、5。 曲へのリンクを作成して友人に提供したり、何らかの理由でデスクトップに置いて自慢したりできます。 検索をご覧ください。



Spotify検索は十分に賢く、さまざまな曲タグで検索できます。 リクエストを正しく書く方法を毎回覚えないように、 このサイトを使用します。 設定を見てみましょう。



キャッシュサイズの設定などに加えて、Last.fmに注目してください。 ユーザー名とパスワードを入力すると、聴く音楽がすべてlast.fmアカウントに落書きされます。



Spotifyコンピューターがいくつインストールされていても、1台だけで音楽を聴くことができます。



このプログラムは、 Windows 、 Mac OS X 、およびLinuxで使用できます（Wineを使用- 指示 ）。



登録



サービスはIP登録のみをチェックするため、IPを1回変更するだけでアクセスを継続できます。 もちろん、無料登録が可能な英国またはその他の国に移動するか、これらの国の代理人を見つけることができますが、素晴らしいsurf7.info Webサイトを使用し、 このリンクをたどり、最終的に貴重な登録を取得します。 インデックスを指定するとき、私たちは長い間取り壊された家の地下に落ち着こうとはしませんが、 このリンクを使用して私たちに合ったホテルを選択します-それで時間を無駄にしないのですか？それからプログラムをダウンロードして音楽を聴きます。 または聞かないでください。



プログラムがまだIPをチェックする（または他の何かを考え出す）ことを決定する場合があることに注意してください。 この状況では、プロキシを介してサイトにアクセスし、あなたが電光男であり、世界中を猛スピードで移動できることを確認するだけで十分です。 彼らはあなたを信じてアクセスを返します。
