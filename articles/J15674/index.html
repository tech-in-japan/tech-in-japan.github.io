再臨を待っています。 Web 3.0の見通しに関する議論の解説

以下の記事は、私の記事「 Web 3.0が表示されない3つの理由 」に対する最終コメントに基づいています。これは、 Web 2.0集合ブログに投稿したリンクです。 ただし、「 最後のコメントの効果 」に基づいて、それを読む人はほとんどいません。 その実践は再び確認されました。 そのため、個人のiTech Bridge ブログで、このコメントを幾分修正して拡張した形で複製することにしました。 以下はその抜粋です。



メシア数3。 第一に、インターネットの大衆は、3番目の数字の下に2番目の数字が来ることを楽しみにしていることをようやく気づきました。 それらはすべてWeb 2.0にうんざりしています。 彼らはメシアを待っており、神を信じています。



私たちは信じています...まあ、それは、すべての信者のように、新しいウェブを待っているのと同じように、彼らは疑いのある人との論争に関与したくないということです。 結局のところ、これらの疑い深い人は明らかに冷静な考えを表明し、不快な質問をします。 信じるだけで、不信者は「利害関係にある」方が良いです。 はい、そして、いずれにせよ、ハッピーエンドの童話は、実際の生活の厳しい散文よりも読むのが楽しいです。 それがまさに、本質的に表明された疑念について私と議論を始めることを決めた人々であることが判明した理由です。 単純な「マイナス」に限定...



ビジネスWebの別の面。 しかし、銀の裏地はありません。 LiveJournalを含むいくつかのコメントを読んで、以前に作成したビジネス指向のソーシャルインターネットの分類スキームと「成功の公式」で間違いを犯したことに突然気付きました。 私はその商用コンポーネントを考慮しませんでした。 この私の罪の中で、私は告白し、すでに修正しました（新しいメモはこちら ）。



それは彼らが戦ったものです。 そして最後に、最も重要なこと。 いいえ、同僚が既存のソーシャルウェブの欠点についてしつこく語り始めたのは無駄ではありません。 それでも私は、成長のプロセスが始まったようです。 自分で判断してください。 問題のメモ（iTech Bridgeのカウンターのみ）を読んだ少なくとも250人のうち、少なくとも10人が同意しませんでした（残念ながら、Habréで肯定的なレビューと否定的なレビューの違いしか見ることができません） ） 同時に、そのうちの1人（！）だけが私のカルマを下げることを決めました。 さらに、私は単一の（！）失礼なコメントを受け取りませんでした。 素晴らしい結果です！ 結局のところ、敵はそうあるべきです。 それらがなければ、人生は新鮮になります。 主なことは、著者と同意していない人が「性格」に移行しないことです。 そして、紛争の中で真実が生まれることがありますように。 誰が知っているか、名前や番号に関係なく、これが新しいソーシャルWebの本質になるかもしれません。
