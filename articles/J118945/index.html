Ubuntu 11.04にMJPG-streamerをインストールできない場合

Ubuntu 11.04のリリースに伴い、 mjpg-streamerパッケージがUSB Webカメラを介したビデオ監視の整理に非常に役立ちましたが、インストールが中止されました。



この問題はばかげたことに不快感を与えます-パッケージバージョンの説明は「アプリケーションセンター」の規則に準拠していません。 解決するには、このパッケージを再構築する必要があります。

最新のアセンブリmjpg-streamer_r94-1_i386.debをダウンロードし、簡単にするために名前をmjpg-streamer.debに変更し、ユーザーディレクトリのルートに配置します。

次に行います：



dpkg-deb -x mjpg-streamer.deb tmpdir

dpkg-deb --control mjpg-streamer.deb tmpdir/DEBIAN







これで、任意のエディターで、tmpdir / DEBIAN / controlファイルのバージョン番号をr94-1から94.1に変更します。



nano tmpdir/DEBIAN/control







パッケージを再度収集します



dpkg -b tmpdir mjpeg-streamer_my.deb







ルートディレクトリにmjpeg-streamer_my.debがあり、それを「アプリケーションセンター」で開きます。 Pipsは「インストール」します。 それはコーシャではないことに同意しますが、それでもインストールしたいと思います。

口ひげ。

誰がrapidshareの私の編集されたバージョンを台無しにすることに消極的です。
