HoloLens 2の紹介









2016年のHoloLensの発売以来、複合現実がどのようにワークフローを変えるかを見てきました。 さまざまな分野で働く数十万人の人々に新しい機会へのアクセスを提供しています。 建設現場から生産ワークショップ、手術室から学校のクラスまで、HoloLensは作業、学習、コミュニケーション、およびタスクの実行方法を変えています。







デジタルの世界が2次元の画面を超えて3次元の現実に突入するとき、私たちはコンピューティングの新しい時代に入ります。 コンピューティングとコラボレーションのこの新しい時代は、3Dでより多くを達成し、国境を越え、より簡単かつ効率的にコラボレーションするのに役立ちます。







今日、 Microsoft HoloLens 2を世界に紹介できることを誇りに思います。







お客様は、HoloLensをさらに改善するために、3つの重要な側面に焦点を当てるよう求めています。 彼らは、没入の効果を高め、メガネをより快適にし、投資収益率を減らすことを望んでいました。







視覚効果などの大規模な改善により、 液浸効果が大幅に向上し、その結果、ホログラムがさらに印象的で現実的になりました。 HoloLens 2の視野を2倍以上に増やし、視野角あたり47ピクセルという業界をリードするホログラフィック密度を維持しました。 HoloLens 2には新しいディスプレイシステムが搭載されており、低消費電力でパフォーマンスを大幅に向上させることができます。 HoloLens 2では、ホログラムとの対話方法も完全に変更しました。







組み込みのAIとセマンティック理解を組み合わせた新しい深度センサーを活用して、HoloLens 2は、現実世界の物理オブジェクトを操作するために使用するのと同じ直感的なジェスチャーを使用して、ホログラムを直接操作できます。 ビデオコントローラーの改善とホログラムの直接操作の可能性に加えて、HoloLens 2は、ホログラムとの相互作用をさらに自然にするアイトラッキングセンサーを導入しました。 企業全体の認証システムであるWindows Helloを使用してログインできます。WindowsHelloは、目の虹彩であなたを認識し、複数の人がデバイスをすばやく安全に共有できるようにします。







よりバランスのとれた重心、軽量カーボンファイバーの使用、調整を必要とせずにデバイスを装着する新しいメカニズムにより、 快適さが向上します。 新しい蒸気回収技術により熱管理を改善し、人間の頭のさまざまなサイズと形状に対して幅広い生理学的変化を提供し、HoloLens 2を設計して、ヘッドセットが快適に調整でき、ほとんどすべての人にフィットするようにしました。 新しい完璧な装着システムは、HoloLens 2があなたに順応し、その上に直接装着できるため、眼鏡を外すことができない一方で、ヘッドセットを何時間も続けて装着するのに便利です。 複合現実から抜け出すときが来たら、バイザーを上げてから他のタスクに移ります。 合わせて、これらの3倍以上の改善により、デバイスの快適性と人間工学が向上しました。







Dynamics 365 Remote Assist、Dynamics 365 Layout、新しいDynamics 365 GuidesアプリケーションなどのMicrosoft Mixed Realityアプリケーションを使用すると、 投資回収時間が短縮されます。 元の価値に加えて、複合現実パートナーのエコシステムは、HoloLensで作成されたさまざまな産業やタスクに適した幅広いソリューションを提供します。 パートナーエコシステムは、HoloLens 2やAzureサービスなど、Mixed Realityに必要な空間、音声認識、視覚画像を分析するインテリジェントシステムを提供するデバイスの可能性を実現する、Mixed Realityテクノロジーと連携する新しい起業家の波で拡大しましたテレメトリアプリケーションを保存、保護、収集するための信頼できるクラウドサービス。

Azure Mixed Realityサービス



本日、2つの新しいAzure Mixed Realityサービスも発表しました。 これらのサービスは、各開発者と各企業がクロスプラットフォーム、状況依存、混合現実、エンタープライズレベルのアプリケーションを作成できるように設計されています。







Azure Spatial Anchorsを使用すると、企業と開発者は、ホログラムが特定の物理空間に格納される複合現実アプリケーションを作成できます。 後で、iOSおよびAndroidでHoloLens、携帯電話、またはタブレットを使用している複数のユーザーが、それらに一度にアクセスできるようになります。 開発者は、スペースオリエンテーションを有効にして、顧客の作業と学習をより迅速かつ簡単に行うことができます。 生産、建築、医学教育、その他の分野での共同作業の有効性を高めるためのそのような協力の例を見てきました。







Azure Remote Renderingは、3Dコンテンツの高品質と詳細レベルの維持に役立ち、意思決定を迅速かつ効率的にします。 今日、モバイルデバイスや複合現実ヘッドセットで高品質の3Dモデルとやり取りするために、開発者は必要な機器で実行するために3Dモデルを「トリミング」または単純化する必要があります。 しかし、プロジェクトの検査と医療計画を実施するには、すべての詳細が重要であり、簡素化は重要な意思決定に必要な重要なデータの損失につながる可能性があります。 このサービスは、クラウドで高品質の3Dコンテンツを再生し、詳細を維持しながらリアルタイムでデバイスにブロードキャストします。







Azure Spatial Anchorsは本日、パブリックプレビューバージョンをリリースしました。 現在、Azure Remote Renderingは非公開の暫定バージョンで利用可能です。後日、サービスは公開バージョンでリリースされます。







Microsoft HoloLensカスタマイズプログラム  



HoloLensは、建設現場や手術室から国際宇宙ステーションまで、複雑な状況で使用されます。 このデバイスは、北米およびヨーロッパで使用されている安全メガネのいくつかの規格に従って、基本的な動的テストに合格しています。 ヘッドセットはテスト済みで、ANSI Z87.1、CSA Z94.3、およびEN 166の基本要件を満たしています。HoloLens2とともに、Microsoft HoloLensカスタマイズプログラムを提示して、顧客およびパートナーがHoloLens 2を環境の要件に適合できるようにします。







最初のHoloLensカスタマイズプログラムは、当社の長年のHoloLensパートナーであるTrimbleによって撮影されました。Trimbleは昨年、HoloLens用Trimble Connectを発表しました。 本日、Trimble XR10とMicrosoft HoloLens 2を発表しました。これは、リスクの高い施設で働く従業員が職場を離れることなくヘッドセットを使用できるようにする新しい安全ヘルメットです。
