php + css + htmlのみを使用して写真を撮ります

先日、PHPとGD2ライブラリ、CSS、HTMLのみを使用して画像を作成しようとした場合、役に立たないがクールなアイデアを夢見ました。 すぐに言ってやった。

アイデアは非常に単純です-元の画像を取得し、そのすべてのピクセルを実行し、それぞれのカラーコードを見つけた後、1px x 1px divを作成し、背景色を画像の元のピクセルと同じに設定します。 その結果、対応するグラフィックの完全な類似物が得られます。 しかし、実際には、HTMLグラフィックを楽しむためには、優れたプロセッサーと最新のブラウザーが必要です。



私の実験の結果はここで見ることができます： HTML写真の狂気 。 全体のポイントは、ページのソースコードにあります。 しかし、見る直前に、 念のため、別のブラウザウィンドウ （できればChrome）を開きます 。



もちろん、これは産業利用を引き付けるものではありませんが、楽しみのためだけにそうなります。
