3CX Phone System 12の公式リリース

Windows 3CX電話システム向けのVoIP-PBXの受賞歴のある開発会社である3CXは本日、比類のないモビリティとスマートフォン接続を提供する3CX電話システム12のリリースを発表しました。





3CXPhone-Android、iPhone、Windows用の新しいユニバーサルアプリケーション



新しい3CXPhoneアプリケーションは、サブスクライバーのステータスを確立する機能、ワンクリックで会議を開催する機能、統合されたPUSHテクノロジーを備えた企業メッセージングシステムおよびソフトフォンを備えた、真にユニバーサルなコミュニケーション機能を提供します。 さらに、新しいソフトウェア機能には次のものがあります。

  • 「盲目的に」または「同伴者」に簡単に電話をかける能力
  • マルチライン-複数の通話を一度に管理
  • 企業の電話帳をExchangeまたはLDAPサーバーと統合する
  • 個人の電話帳をMicrosoft Outlookと同期する
  • 着信転送の簡単なルール
  • プレゼンス状態をすばやく設定する機能
  • ボイスメールボックスを視覚的に確認する機能
  • 一元化された通話履歴
  • 3CXPhoneアプリは簡単にインストールでき、数秒でPBXに統合できます


「私たちはIT管理者を高価でかさばる、時代遅れの従来のPBXから解放しています。3CXは、ユーザーにユニバーサルコミュニケーションシステムを使用してオフィスワーカーとリモートスタッフとして同時に働く能力を提供します。





3CXPhone for computerを使用すると、はるかに高速に電話をかけることができます



オフィスにいるユーザーは、手動で電話番号を入力する代わりに、コンピューターからIP電話を制御するために作成された3CXPhoneのCTIモードを使用して、マウスを1回クリックするだけで簡単に番号をダイヤルできます。 この手順は電話機のモデルごとに異なるにもかかわらず、手動で通話をデバイスに手動で転送する代わりに、ワンクリックで通話を転送することもできます。

数秒で電話会議



3CX Phone System 12では、マウスを数回クリックするだけで、会議をすぐに組み立てることができます。 さらに、電話帳から同僚または外線番号を追加することにより、通常のアクティブコールから会議を行うことができます。

PUSHテクノロジーによる比類のないモビリティ



3CXは、PUSHテクノロジーと独自の統合を提供します。これにより、3CXは新しいVoIPコールで電話を「起動」できます。 AndroidとiPhoneのスマートフォンは、メッセージを受信するとき、または電話をかけるときにスリープモードから復帰します。これにより、バッテリーの電力を節約すると同時に、世界中で常に連絡を取り合うことができます。 また、VoIP経由でいつでもユーザーに電話をかけることができるため、企業通信の企業コストを大幅に削減できます。 すべての通話は3CX電話システムを経由するため、ユーザーは発信者に直接携帯電話番号を提供する必要はありません。これにより、高度なプライバシーが保証されます。

信じられないほどの汎用性で生産性を向上



3CXPhoneは、単なるソフトフォンではなく、汎用性の高い通信クライアントであるため、従業員の生産性の向上に最適です。 ユーザーは、同僚のプレゼンス状態を確認し、チャットにメッセージを送信し、ワンクリックで電話会議を作成し、スマートフォンとパソコンの両方からボイスメールと通話履歴を表示できます。

標準のPBXおよびIP電話接続よりも優れています



3CXとIP電話およびVoIPゲートウェイの大手メーカーとのパートナーシップにより、すべての製品がリリース前に完全な3CX品質レビューを受けることが保証されます。 また、新しいYealink T46G IP PhoneとWelltech WellGate 2540 VoIP Gatewayを3CX Phone Systemとともに使用する場合、互換性の問題もありません。 3CX電話システム（バージョン12）は、すぐに低価格のGigaset N720 DECTデバイスをサポートします。
