VirtualBoxにCentOS 6.3 + LNMPをインストールします（ビデオチートシート）



Windows環境でWebサーバーをローカルで実行する必要がある場合は、VirtualBox仮想マシンでLinuxとそこに必要なすべてのコンポーネントをインストールすることでこれを実行できます。 しかし、多くはそのようなインストールの複雑さを恐れています。 実際、これはすべて非常に迅速かつ簡単に行われます。



その結果、パテ経由のルートアクセスと動作するLNMP構成（Nginx + MySql + PHP）を持つローカルVPSサーバーを取得します。



このビデオの対象者



Linuxと長い間話し合っている人にとっては、おそらく、このようなサーバーをセットアップしても問題は生じません。 ただし、仮想マシンやLinuxコマンドラインをこれまで扱ったことがない人は、インストールのさまざまな段階で特定の問題が発生する可能性があります。



もちろん、Googleですべての回答と指示を見つけることができますが、これを行うには1時間以上を費やす必要があります。 この短いビデオチュートリアルで必要なすべてを収集しようとしました。



内容



  • VirtualBoxにCentOS 6.3をインストールします。
  • Windowsから仮想マシンに接続するための22および80ポートのリダイレクト。
  • ssh経由でputty経由でこの仮想マシンに接続します。
  • 同じ名前の自動インストーラーを使用したLNMPインストール（Nginx + MySql + PHP）。
  • ファイアウォールを無効にします。
  • ゲストOSアドオンのインストール。
