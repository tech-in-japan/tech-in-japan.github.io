ルームチケットマネージャー

こんにちは

私は最近、ERPシステムを作成しました。

部屋-サーバーではphp（Zend Framework）言語で、クライアントではhtml + css + javascript（jquery）で記述されたタスク管理システム。



ルームは、オフィスのオーガナイザー、人事管理システム、およびオフィスで最適かつ安定した作業を整理するための一連のアシスタントツールです。

特徴

  • タスクの割り当ての管理と実行の制御
  • タスクツリー
  • 対応サービス
  • 人事試験システム
  • 人事部
  • クエストツリー
  • プロジェクトへのユーザーアクセスの差別化
  • SMS、ICQ、電子メールの購読（デモなし）




残念ながら、私はレイアウトに煩わされることを望んでいなかったので、それはオペラでのみ正しく動作します（しかし、私はすぐにそれを修正します



ここで作品を見ることができます： ru.room.pnpstudio.ru

  • 顧客フィードバック
