Exantech Code Jam：Hacktoberfest 2017

いずれにせよ、IDEまたはブラウザ、システムユーティリティ、またはWebサイト上のスクリプトであるかどうかにかかわらず、私たちは全員、作業にオープンソースコードを使用しています。 オープンソースの本質と魅力は、誰もがそのようなプロジェクトに貢献できることです。 しかし、実際には、誰もがそれを正しく行う方法を知っているわけではなく、単に始めることを恐れているわけでもありません。



私たちとExantechの技術研究所は、オープンソースをサポートしており、開発者の間でより一般的に参加することを望んでいます。



10月28日、モスクワクルーで、Code JamをHacktoberfestの一部として開催します。Hacktoberfestは、デジタルオーシャンのオープンソースでのプログラミングのための年次オンラインフェスティバルです。









Hacktoberfestに参加するためのルールは非常に簡単です。GitHubで興味のあるプロジェクトを見つけ、10月中に少なくとも4つのプルリクエストを開く必要があります。 このため、Tシャツとステッカーがありますが、最も重要なのは、お気に入りのアプリケーションの開発の歴史においてあなたの名前が永遠に消えてしまうという素晴らしいボーナスを受け取ることです。



コードジャムは、アクティブな参加者と著者からのオープンソースプロジェクトのプレゼンテーションから始まります。 プロジェクトは、OpenBSDからスマートコントラクトや機械学習フレームワークにまで及びます。 オープンコードのおかげで、プルリクエストを送信するだけで、会議中に参加できます！



プレゼンテーションの後、昼食をとり、その後、快適な会社で数時間の生産的で楽しい開発を行います。 経験豊富なEXANTE開発者と講演者がこのサイトに常に参加します。いつでも質問をして、アイデアを話し合うことができます。 一日の終わりには、アクティブな参加者に有益な賞品を提供します。



Exantech Code Jamに参加するには、 TimePadおよびHacktoberfest自体に登録してください 。 もちろん、会議を待って今すぐコードの作業を開始することはできません。土曜日に、準備されたプルリクエストを厳finishに終了して送信してください。



便利なデバッグのために、DOで独自のサーバーを実行できます。デジタルオーシャンのプロモーションコードを10ドルで使用することを忘れないでください： do.co/hacktoberfest17
