エレクトロニクスのことは何も理解していません。ボードやマイクロコントローラーを使ったことはありませんし、はんだ付けもしていませんが、The Maker Showを見た後、ArduinoとRaspberry Piのどちらを注文するべきか既に考えています。

モノのインターネットのトピックを研究し始めたばかりの人にはこのショーをお勧めします。 あなたのために、私はショーの主な問題をレビューしました。



1.エレクトロニクスの紹介



LEDをレモンで光らせる方法は？



マイクロソフトのテクノロジエバンジェリスト、David Crookは、電子工学の基礎を説明するいくつかの実験を行っています。



最初の実験では、彼はレモンからバッテリーを作り、LEDを点灯させます。



実験から、レモンの直列接続により出力電圧が増加し、より強力な電子部品に使用されると結論付けられます。 並列接続により生産性が向上します。つまり、レモン電池の寿命が長くなります。



2番目の実験は、レモンの実験に似ていますが、レモンの代わりにフィンガーバッテリーと、電子回路と抵抗器を作成するためのブレッドボードを使用します。



以下は、さらにいくつかの実験です。その間、David Crookはブレッドボードがどの部分で構成されているかについて話し、抵抗が必要な理由の質問に答えます。







2. LEDの点滅...さて何を？



Maker Showのこの号にはレモンはありません。 LEDとArduino UNOの点滅専用です。



マイクロソフトのテクノロジーエバンジェリストであるブライアンシャーウィンは、Arduino UNOボードの構成とプログラム方法を説明しています。



点滅するLEDをArduinoとブレッドボードに直接接続し、以前のリリースのようにボタンを追加します。これには、回路に3つの出力を持つ抵抗が含まれ、LEDの輝度を調整できます。







3. Arduinoとサーボモーター



あなたはすでにLEDでHello Worldを見て、電子機器についてもっと学んでいます。 3番目の問題では、サーボモーターとパルス幅変調について説明します。



サーボモーターは、可動要素を制御するためのデバイスです。 例えば、無人機で使用される車の車輪を回すことができるなど、繰り返される動きがある場合に必要です。



エアバスでさえ、大型サーボモーターを使用して、 補助翼（ロール舵） 、バルブ、および航空機の他の制御システムを制御します。



サーボモーターは、IoTプロジェクトの開発時に最も一般的に使用されるモーターの1つです。



マイクロソフトのテクノロジーエバンジェリストであるサムストークスは、サーボモーターの構造とその仕組みについて語り、Arduinoをサーボモーターと連動するようにプログラムする方法を示します。







4.サイズが実際のコンポーネントに対応する3Dモデルの作成と印刷



この号では、ジェレミーフォスターが一緒にいます。 彼は、例としてXbox Oneを使用した3Dモデリングと3Dプリントについて語っています。



ビデオでは、コントローラーケースの一部とESP8266マイクロコントローラーを作成します。これらは、既存のパーツのサイズに最適です。



Jeremyは、AutodeskのFusion 360プログラムを使用しているため、3Dモデルを非常に簡単に設計できます。 GrabCADオンラインサービスは、タスクをさらに簡単にします。 既製の3Dモデルが含まれています。







5. Windows 10をRaspberry Pi 2にインストールする方法は？



次回の記事では、Microsoftテクノロジーエバンジェリストのケニースペードが、 Windows 10 IoT CoreをRaspberry Pi 2にインストールする手順を説明します 。



このオペレーティングシステムは、Raspberry PiでユニバーサルWindowsアプリケーションを実行するのに役立ちます。 Kennyが語るすべては、Raspberry Pi 3にも関連しています。



Kenny Spadeがすべての手順を説明し、PowerShellを使用してデバイスをリモートで制御する方法も示します。
