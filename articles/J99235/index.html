Ubuntu 10.04のビデオの垂直同期の問題を解決する

このトピックはサンドボックスで作成しましたが、@ BSDaemonのおかげで、（トピック）の公開前に招待を受けました。

長い間、Ubuntuを試しました。 しかし、最近配信することにしました。 すべてが素晴らしく、美しく、一見安定しているように見えますが、私は1つの迷惑にぶつかりました：フルスクリーンモードでビデオを見ると、明るくダイナミックなシーンで、画像が部分に分割されているように見え、これらの部分は一緒に収まりません：





私はすぐに言わなければなりません：私はGeForceビデオカードを持っているので、問題とその解決策はNvidiaビデオカードのユーザーに関連するでしょう。 この同期解除は、Compizがオンになっている場合にのみ表示されます。 Compizは美しさだけでなく、利便性にも満足しているので、それを拒否するつもりはありませんでした。その結果、解決策を探し始めました。 ロシアのフォーラムforum.ubuntu.ruは明確な回答に満足せず、そこで提供されたすべての解決策は私にとってうまくいきませんでした（後で判明したように、1つの解決策は問題の解決に非常に近かったが、いくつかの重要なポイントが欠落していました）。



その結果、少し苦労した後、私はどのような解決策に来ましたか：

1）インストールされていない人は、compizconfig-settings-managerをインストールします

2）実行し、[一般設定]（一般オプション）-[画像設定]（表示設定）に移動します

-[リフレッシュレートの検出]の横のチェックボックスをオフにします

-[VBlankに同期]（VBlankに同期）の横にあるチェックボックスをオンにします

-[全般]タブを開き、[フルスクリーンモードでウィンドウのリダイレクトをキャンセルする]チェックボックスをオンにします（フルスクリーンウィンドウのリダイレクト）

3）nvidia-settingsを開き、X Server XVideo Settingsに移動します。

-[ビデオテクスチャアダプター：VBlankに同期]ボックスをオンにします。

GL設定を開くに移動します

-Sync To VBlankの反対側のボックスもチェックします

-ターミナルで、nvidia-settings -lを実行します

4）ビデオプレーヤーの設定に、出力デバイスXVを配置します（原則として、既にインストールされています）。



これで、ビデオの画像が目を楽しませています。 ポップコーンを買いだめして、お気に入りの映画を作ることができます。 素敵な景色を！



UPD：一部のATIビデオカードでは、この方法も機能します。 Compizを構成するだけで、出力はgl2を介して行われます。

UPD2：誰もが問題を抱えているわけではありません。 しかし、彼女には場所があります。 それを持っていない人は誰でも、これについて退会する必要はまったくありません。
