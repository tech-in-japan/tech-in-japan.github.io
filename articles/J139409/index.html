ヘッドフォンアンプ、少し難しい

最近、コンピューターをLGA2011にアップグレードしました。比較的高価なボードの箱に、「スーパーデュパーサウンド、110dbオールガベージ」という碑文がありました。 良い音が痛いことはない、と私は完全に考えました。 次の2日間、私は必死に鉄を運転しました、そして、すべては大丈夫でした。 そして、彼は通常の仕事に戻り、そしてもちろん、音楽の仕事に戻りました。 静か。 いや 静かに！ ボリュームの100％は、私の通常の32Ωヘッドフォン（感度がそれほど高くないSennheiser HD 448 ）で快適なボリュームを得るのにかろうじて十分であり、録音が低信号レベルで行われた場合は、少なくとも髪を引き裂きます。



絶望から、私はブルートゥース経由でA2DPアダプターを聴き始めましたが、もちろん品質とバッテリーの両方（〜6時間）は悲しみに陥りました...



あなたはもうそのように生きることはできないことが明らかになりました...



しかし、3か月前にさかのぼって： terraelectronicaでMP3プレーヤー用の部品を購入しました 。これは最終的にiPodを置き換えることになっており、オーディオ機器のマイクロチップのリスト（入手可能性と価格で並べ替え）を調べて、MAX9724が驚異的な7.78の価格で見られました。ルーブル。 それは特殊なヘッドフォンアンプでしたが、私はそのプロジェクトにはそれを必要としませんでした-購入したオーディオコーデックはすでにヘッドフォンに必要なレベルの信号を生成していました。 さて、私は予備の5個を購入することに決めました、特にそのような価格で、それがどこで便利になるかわかりません...



私たちの日々：



アンプの設計目標は



1.「今日の」問題を解決するために、すべてを組み立てる必要があります（高価なオペアンプは消えます）

2.厳密にUSBで駆動し、非標準の電源、特にバッテリーはありません

3.品質は、人が聞くことができるものよりも良くなければなりません。

4.パラグラフ3の結果として、オーディオ信号のパスにあるコンデンサが問題になります（解決可能ですが）。 フィルムコンデンサは高価で希少ですが、電解質ではここでは常に困難です。

5.ボリュームコントロール用のポテンショメータの欠如-それらはノイズが多く（回転中/ちょうど老年期から）、一般的に信頼できません。



既存のデザイン：



1.たとえば、最近の記事ヘッドフォンアンプはシンプルで高速です -電力の問題は記事で巧みに省略されています:-)また、関連する記事を読んだ後、オペアンプで通常のアンプを構築するのは簡単ではないことは明らかです-自励、バイアス0、入力電流と入力インピーダンスの制限）。 まあ、一般的に-オペアンプはUSBからそれを行うためにバイポーラ電源が必要です-別のチップとデバッグが必要です。



2. USB-DACもアクセシビリティのために消えており、一般的に「鉄」サウンドカードの方が好きです。



3. 既製のポータブルアンプ -待機に加えて、内部には何も保存されていないと信じる必要があります。 ちなみに、このFiiO E3内のチップは、私が持っているものと機能的に類似しています。違いは、MAX9724が5Vで動作し、LM4917が最大3.3Vで動作することです（300 ohmヘッドフォンの適用性も制限します）。



解決策



明らかに、私はMAX9724を手に入れました-バイポーラ電圧発生器（5Vから-5Vを生成）と2つのオペアンプを含み、音の増幅のために研ぎ澄まされています。 品質-私の条件での標準的なスイッチングパターンによるTHD + Nの歪みは約0.04％です（0.5％でも歪みが聞こえるとは限りません）。 回路は、出力にコンデンサを必要としません。 信号は「地面に対して」すぐに増幅されます。これにより、品質が低下する可能性のある場所の数が劇的に減少します。



この歪みレベル（0.04％）はさらに減らすことができます-典型的な回路では、公称値が1マイクロファラッドの負電圧発生器の2つのコンデンサがあり、スケジュールに従って、キャパシタンスを2.2マイクロファラッドに増やすと、歪みを大幅に減らすことができます。 私は約20マイクロファラッドのセラミックコンデンサを3箇所すべて（電圧発生器+電源）に配置しましたが、通常の回路よりもパラメーターに頼ることができると思います





スキームはフェルトブーツと同じくらい簡単です。 抵抗のみに注意を払う価値があり、抵抗を設定します。この場合は2（20kOhm / 10kOhm）です。 150〜300オームのヘッドフォンがある場合、20ではなく40komを配置できます。その場合、アンプ出力での最大信号振幅は8Vになり、これで十分です。 また、データシートの図には、入力のコンデンサがあります。 サウンドカードからゼロに相対的な信号が既に送られているため、コンデンサで余分なボトム/歪みをカットする必要はありません。 その結果、アンプは0Hzからの信号を増幅します（もちろん、サウンドカードには制限があります）。





一方向の配線は簡単です。2つのジャンパーだけでなく、USBおよびオーディオコネクタがボードの「背面」側にあるという事実に注意する必要があります。



集める





超小型回路はハードコアに見えますが、表面張力の魔法の助けを借りて簡単にはんだ付けできます（すべてLTI-120液体フラックスで塗り付け、はんだはその脚だけに登ります）。 ここのコンデンサは壊れたコンピューターハードウェアのものです-もちろん、オーディオセクションに入れることはできませんが、電源についてはすべて問題ありません。 共振周波数が一致しないように、3つの場所のそれぞれで、大きな束（4.7〜10マイクロファラッド）と1つの小さな束（0.1マイクロファラッド）があります。





回路は非常にコンパクトであることが判明しました。 中央のコンデンサ-USBから5Vに接続。 コンピュータポートに接続する場合、1000マイクロファラッドはもちろん過剰であり、標準では10マイクロファラッドしか許可されていませんが、通常はすべて機能します。



Eagleのプロジェクトと必要なライブラリを含むアーカイブをダウンロードします 。





電気テープ製のハイテクハウジング。 当初、このボードは収縮チューブ用に作られていましたが、文字通り1ミリメートルでは不十分で、収まりませんでした。 まあ、それにもかかわらず、私はそれが好きです。

結果



もちろん、私はすぐによく知られた問題に出くわしました：オーディオを扱うとき、2つの場所（USBグラウンドとグラウンドオーディオジャック）で1つのグラウンドに接続できません。 この場合、干渉は地上で増殖し、フィルターで除去することはできません。この場合、電力安定器は役に立ちません。 （問題は、USBが独自のグランドレベルを持ち、サウンドが独自のものであり、ボードが独自のものであるということです。消費電流に応じて、地球はどこでも異なって上昇し、避けられないノイズが発生します）。



この問題を解決するには、サウンド接続（USB DAC）を取り除くか、電源（バッテリーまたはその他の電源）を使用します。 USB出力を備えた電源を使用して、それらがどこにでもあり、標準であるという事実に完全に満足しました。



最終結果はすべての予想を超えています。 品質、絶対的な0ノイズ、快適な音量レベル（22〜40％）、および「プル」静かな録音のマージンに関する苦情はありません。 音はより滑らかです（覚えておくべき主なことは、低音が0Hzからであるということです）、そしてすべて-そして、実際に-日曜大工のオーディオデバイスは常に特に良い音がします:-)



低価格（原文のまま！）、付属品「マージン付き」アセンブリ、オーディオパスにコンデンサがない、高インピーダンスヘッドフォン（300オーム）での作業時の高出力は、完成した中国のデバイス（同じFiiO E3など）とは異なる理論上、電源電圧と音質はより高いと約束されています（実際には、おそらく違いを聞いたことはないでしょう）。



PS。 上で述べたように、超大音量で聴覚を損なうためにアンプは必要ありません（破れたヘッドフォンは言うまでもありません） ）、およびサウンドカードの出力があまりにも死んでいる場合、感度が低い「重い」ヘッドフォンを構築するために。 さて、ソフトウェアなしで静かな録音/映画を引き出してください...
