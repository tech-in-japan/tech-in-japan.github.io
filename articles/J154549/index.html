ロシア語のオンラインコース





背景



こんにちは 私の名前はラヒムです。カールトン大学のコンピューターサイエンス（B.Sc）プログラムで勉強し、数年前にコンピューターサイエンススチューデントプロジェクトの一環としてHabréについて書きました（すべての問題をプロファイルで確認できます）。 それからここで、数学とコンピューターサイエンスに関する集合的なブログの作成に興味を持つ多くの人々を見つけました。 それから彼はコメントに書いた：



もし私が教授になったら（そして、私は1人になりたい）、各コースの始めにこの集合ブログのアドレスを掲示板に書き、「参加してください。 集合的な心は、コンピューターサイエンスを学ぶのに役立ちます。」



残念ながら、しばらくするとブログの更新が停止し、アクティビティは最小限になりました。 私は大学を卒業して最初の1年間は開発者として働いていましたが、2012年1月にそのコメントの実装に少し近づきました。カザフスタンの南部の首都にある国際情報技術大学（ IITU ）の教師になりました。 彼は最初の学期に、オペレーティングシステムとWeb開発について講義し、時には公開講義を行い、時にはハブで何かを公開しました。 しかし、「hexlet.ru」はボードに書きませんでした。



この秋、 Hakesletは集合的なブログから無料の教育プラットフォームに進化しました。 一番下の行は簡単です： あなたは学生になり、さまざまなコースを取ることができます、あなたは教師になり、独自のコースを実施することができます。 形式はCourseraまたはUdacityに似ています。 10月23日に、最初のオンラインコース-Mac OS X用アプリケーション開発が始まります。



このコースについて-以下、しかし今のところはプロジェクト自体について少し。



マニフェスト



銀行も、政治も、宗教も、お金も、問題を解決して生活を向上させることはできません。 蛇口の水、暖かい家の蛇口、インターネット、ひどい病気の薬-これらはすべて科学者とエンジニアの仕事の成果です。 知識は人間の文明の基礎であるため、知識は自由でアクセス可能でなければなりません。



目的



目標は、多数のコースを備えたRunetでアクティブな教育プラットフォームを作成することです。 才能のある教師を引き付け、大規模なオンラインコースを作成および実施する機会をすべて提供します。 Hakesletを使用して専門家を訓練し、訓練を行う企業や組織を引き付けるため。 投資を引き付け、収益モデルを開発します。



私の個人的な目標は、このプロジェクトを私の唯一の本格的な仕事に変え、今後数年間でコンピューターサイエンスの学部課程のコース一式を作成することです。



Hakesletで独自のコースを開始できます



  • 主題は、大工仕事から超弦理論まで何でもあります。
  • 継続時間-任意ですが、できれば3クラス以上。 非常に長いコース（5か月以上）は、分割するのが理にかなっています。
  • 形式-任意ですが、講義で宿題、テスト、試験を含む便利な形式（オーディオまたはビデオ）で講義を公開することをお勧めします。
  • 言語は任意です。
  • コースの構造（各レッスンの説明）を準備し、開始前に公開する必要があります。


プロジェクトをサポートする



プロジェクトは熱意と個人投資に基づいています。 もちろん、最初のコースが成功した場合、私は本当のスタートアップになり、投資家を見つけたいなどです。 これまでのところ、次の方法で支援できます。



  • 既存のコースに参加し、学生を支援します（質問への回答、ディスカッションへの参加など）
  • ボランティアになりましょう。 たとえば、デザイナーによって：ロゴが必要です。 とにかく、プロジェクトに興味を持っているおもしろい人たちが参加します。
  • 友達にプロジェクトについて教えてください。 潜在的な学生、教師、ボランティア、または投資家の注目を集めます。
  • 寄付をする


メールfreetonik@hexlet.ru



サーバーのセットアップと保守を支援するシステム管理者のIvan Borovkovに感謝します。







それで、Hakesletの最初のコース：



Mac OS X用のアプリケーション開発



コースの要件（前提条件）：
オブジェクト指向プログラミングの基本概念の知識、C ++および/またはJavaの基本的なプログラミングスキル。



コース概要：


  1. Objective-Cプログラミング言語
  2. Xcode開発環境
  3. 基盤フレームワーク
  4. Cocoa API
  5. iOSアプリケーションの開発の基礎（iPhoneおよびiPad用）
  6. 優れたアプリケーション設計の基礎




書式：


  1. 15週間-15のビデオ講義。各講義はモジュールで構成されています。 総講義時間は50分です。
  2. 講義は、YouTubeおよびiTunesストアで公開されています。
  3. コースの言語はロシア語ですが、講義は英語でも可能です
  4. 5つの短いテスト
  5. 4宿題（プロジェクト）
  6. 中期学期試験
  7. 学期の終わりの最終試験（最終）
  8. テスト、プロジェクト、および試験は、公開の瞬間から締め切りまで（つまり、1週間以内）いつでも受験できます。




テストと試験は自動的にチェックされますが、宿題（プロジェクト）はピアグレーディングシステムを使用してチェックされます。学生は互いの成果を匿名で評価します（コースラなど）。



コースを修了すると（50％以上の評価で）、シンボリック証明書が発行されます。



（このコースの一部としてプログラムするには、Mac OS XとAppleのXcodeプログラムを実行しているコンピューターが必要です）



コースにサインアップしてください 。
