Raspberry Pi Model Bの場合+ VGAアダプターを作成





ご存じのとおり、Raspberry PiミニPCにはHDMIコネクタがありますが、VGAはありません。 そのため、モニターにVGAしか搭載されていない場合、Raspberry Piでの作業には適していません。



それは最近まででした。 Gert van Looという職人が独自にFor Raspberry Pi Model B +用のVGAアダプターを作成しました。 アダプターは、新しいラズベリーの40ピンコネクタを使用します。



そして今、Malinkaは2つのディスプレイを同時に操作することをサポートできます。 むしろ、数週間後になります-現在、アダプターの作成者は、HDMIおよびVGAポートを使用して2つのディスプレイと同時に動作するためのRaspberry Pi Model B +サポートを追加できる適切なソフトウェアに取り組んでいます。



ちなみに、新しいアダプターはまだ購入できません。作者はそれを大量生産する予定はありませんでした。 ただし、必要なすべてのデータをGitHubに投稿したので、必要に応じて 、Gertが提供するスキームを使用して独自のアダプターを作成できます。



アダプターに関する詳細情報は、GitHubのリンクおよび次のビデオで入手できます。
