ミニカンファレンス「サイトオーナー向けホスティング」が行われました

最近、ホスティングに関するミニ会議の発表を Habréで公開しました。



ミニ会議は大成功でした。 プレゼンテーションは、 Microsoft Ukraine 、 RedGraphic 、 NivalNetworkの代表者によって行われました 。 文化大学のホールには70人以上の専門家が集まりました。



マイクロソフトウクライナのITProエバンジェリストであるIgor Shastitkoは 、「Microsoft Web Platform Technologiesの概要」というプレゼンテーションを行いました。 レポートの一部として、IIS7アーキテクチャの問題、サードパーティのスクリプト言語の実行、およびサーバーのセキュリティの問題が考慮されました。 Sergeyは、サーバー仮想化の方法とURL Rewriteモジュールの非標準的な使用についても話しました。 Bizsparkイニシアチブについて別の項目が伝えられました。これにより、ベラルーシの開発者は、ライセンスされたMicrosoftソフトウェアを無料で使用する権利を得ることができます。



RedGraphicのテクニカルディレクターであるRoman Shaternikは 、「サイトの移行：バグの処理」に関するプレゼンテーションを行いました。 レポートの一部として、Romanは、異なるホスティングサービス間でクライアントサイトを転送し、この点で発生する問題を解決するというWebスタジオの豊富な経験を共有しました。 ローマの経験によれば、最も典型的な問題は、スタジオをホスティングの選択から分離することでした。 この結果、クライアントはPHPバージョンの問題、共有ホスティングへのSSHアクセスの不能、ドメイン名の管理の困難に直面しています。



Nival Networkシステム管理者のSergey Karatkevich （およびNivalの前は、SergeyはAgavaシステム管理部門の責任者でした）は、クラウドホスティングの展望と、こうしたサービスの普及に伴うビジネスプロセスの変化について話しました。 クラウドホスティングサービスは、AmazonやGoogleなどのマーケットリーダーによって既に提供されています。 クラウドホスティングを使用すると、共有、vpsホスティング、および物理サーバーレンタル間の境界線がなくなります。 クラウドホスティングのメモリ量、処理能力、ディスク容量は動的に変化します。



この会議は、IT-JOB.BYポータルの教育プログラムの一環として開催されました。 彼女はすでにこのプログラムの一部として開催される第10回イベントになっています。 合計で、プログラムはすでに1000人以上を集めています。 プログラムのすべてのイベントは無料で、ITに興味のある人向けに設計されています。



会議資料は 、主催者のWebサイトですでに入手可能です。
