マイクロソフトライセンスサブスクリプション：敵と交渉する方法

マイクロソフトからライセンスされたソフトウェアを使用している企業にとっては簡単ではありません。 実際、標準ライセンスの更新（通常は7月31日に行われます）は簡単な作業ではありません。 マイクロソフトライセンスの条件は非常に複雑で巧妙に書かれているため、独立したアナリストが分析に特別に従事し、分析レポートを金銭で販売しています。



レポートの作成者は、Microsoftライセンスの条件には「独自の用語、複雑な価格体系、急速に変化する製品ライン」が伴い、「未訓練のライセンシーを混乱させる可能性がある」と強調しています。 そのようなことのために、慎重に準備する必要があります。



一般的に、マイクロソフトとのライセンス契約を更新する際に考慮すべき4つの主な事項があります。



Microsoftサブスクリプションの条件は積極的であり、必ずしも公平ではない

あなたにとっての正義とマイクロソフトにとっての正義は異なるものであるという意味で不公平です。 彼らはあなたが職場ごとにライセンスを支払うことを公平だと考えています。 交渉中に、多くの企業は互換性のあるジョブの数を過小評価する傾向がありますが、この場合、いくつかの重要な割引が失われる可能性があります。



専門家は、すべてのジョブを互換性があると絶対に示し、強みのある立場から交渉することを推奨します。



直前のサブスクリプションは経済的に不当な場合があります

一部の企業はあまりにも混乱している、または特に有利な条件でMicrosoftを「促進」するのを待つにはあまりにも長い。 しかし、時間は双方にとって重要です。 このレポートでは、個々のマイクロソフトの代表者は「重要なプロジェクトを待つことも延期することもできないため、不利な条件に同意するように強制することもできます。」



スマートに。 前もって計画しますが、敵に対する敵意はありません。 人為的な複雑さで構成されたアプリケーションでそれらを攻撃する必要はありません。 専門家は、あなたではなくMicrosoftが間違いを犯すようにプロセスを調整することをお勧めします。



マイクロソフトは、必要なだけ購入させたい

ほとんどの企業は3〜5年ごとにアップグレードします。 しかし、このようなシステムはMicrosoftにとって経済的に不利です。 彼らは年間契約から利益を得たいと考えています。つまり、ソフトウェアアシュアランスやエンタープライズアグリーメントのような年間サブスクリプションに署名するほうがより利益があります。



最初のケースでは、通常3年間、追加の年会費（デスクトップソフトウェアのコストの29％とサーバーソフトウェアのコストの+ 25％）を支払う必要があります。 ここでターニングポイントは3.5年のマークです。 3年半以上ソフトウェアを使用する場合は、最初にフルコストで購入する方が収益性が高くなります。



マイクロソフトは、年間サブスクリプションを発行したユーザー向けに追加のサービスを提供しています。これはサポートおよびトレーニングコースですが、そのようなサービスは、最初のタスクがソフトウェアを割引価格で入手することを希望する企業にはあまり適していません



スマートに。 必要なアップグレード保護のみを購入してください。



マイクロソフトは追加サービスを販売したい

標準価格表の範囲外で議論を始めると、両者が勝ちます。 コンサルティングや追加サポートなどの追加サービスについて話し合っています。 たとえば、企業がWindows Vistaに移行する場合、Microsoftはスタッフおよび計画ソフトウェアのトレーニングサービスを割引価格で提供する場合があります。 これはお金を節約します。



ここでも、賢くなり、本当に必要なものであり、マイクロソフトが本当に売りたい製品を購入する必要があります。 たとえば、Vistaの売り上げは非常に低いため、これは交渉を開始し、多数の割引を開始するのに適しています。



企業は、SharePoint Server 2007、Windows Server 2008、およびHyper-Vの購入に関して、Microsoftの割引も期待できます。
