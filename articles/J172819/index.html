一般的なOracle Certified Java Professional Programmerの証明書と認定の取得

こんにちは、ハブラの居住者と世界の残りの人々。 この記事は、最初のJava Progammer I試験の投稿のフォローアップです。 情報の一部はここで複製されます。 今日は、次の問題に対処したいと思います。

  • 認定が必要な理由;
  • エンジニアがOracleから取得できる証明書の種類
  • Java Programmer I / II試験に合格するために知っておく必要のあること。






証明書の必要性についての考え（エントリではなく）



まず、認定が必要な理由について少し詩を書いてください（安全にスキップして詳細に進むことができます）。 各エンジニアは、何らかの形で、自分にとって主要なプロファイルを自分で選択せざるを得ません。 このプロファイルは、動作する主要言語と密接に関連しています。 しかし、ほとんどの場合、日常のタスクはプログラマーに影響を与え、言語のさまざまな部分を不均等に使用せざるを得ません。 その結果、彼はいくつかの部分をより良く知っており、いくつかはより悪いことを知っています。 彼は目を閉じていくつかのパラダイムとテンプレートを適用することができ、一部の人にとってはドキュメントを開く必要があります。



Androidで作成するエンジニアを例にとってみましょう。 仕様により、彼はほとんどの場合Java 6を使用し、実際にはJava 7のすべての革新を認識していても、あまり頻繁に遭遇しません。 そのため、Java 7の履歴書に記載されているにもかかわらず、そのような候補者は、たとえばFork / JoinまたはNIO2に関連するタスクを常に効率的に実行するとは限りません。



そのため、職場でほとんど価値がなく、言語自体の知識の深さを感じている人は、自分のトピックに関する表面的なインタビューでも悲惨に失敗することがあります。 これは、しばしば彼らの言語に必ずしも完全に精通しているわけではない若い専門家でしばしば起こります。 また、この証明書は、スペシャリストの知識が、マルチスレッドやIO、またはコレクションだけでなく、言語のあらゆる面で十分に深いことを確認します。 この証明書は主に、使用する言語をより深く研究するためのインセンティブであり、この分野の専門家が実際に持っている知識についてより客観的なアイデアを提供します。 最初のレベルの配信でも、新しいレベルを発見しましたが、最初に最初のものを見つけました。 など...



Java証明書の種類



Java OracleからのJava証明書は、少なくとも2つの基準（水平（特殊化）と垂直（レベル））で分離できます。

  • 専門化（水平分割）;
  • レベル（垂直分割）。




専門分野の証明書の卒業


すべての証明書は、レベルが確認される専門分野に応じて3つの大きなグループに分けられます。

  • Java SE
  • Java EE
  • Java ME


レベルごとの証明書の卒業


次に、各方向で、次のレベルで条件付きグラデーションを実行できます。

  • オラクル認定アソシエイト
  • オラクル認定プロフェッショナル
  • Oracle認定エキスパートプログラム
  • Oracle認定マスター


* Java SEおよびJava MEの専門化は、トップレベルのエキスパートに限定されています



Java Associate && Expert



証明書の種類では、特殊化に関係なく、Oracleは言語の基本的な知識に特に重点を置いているため、エキスパートまたはそれ以上になる資格を得るには、Java認定Preofessional Java証明書を取得してJava言語自体の知識を確認する必要があることに注意してくださいプログラマー。 言い換えると、選択した専門分野に関係なく、最初の2つのレベルはJava言語（つまりJava SE）に関連付けられ、それ以上は関連付けられません。



次の方法でProfessional in Javaを取得できます。





つまり、最初にアソシエイトステータス（Java Programmer I試験に合格）を取得し、Professional（Java Programmer II）を取得する必要があります。 私はすでにここで最初のことについて書きました。 その後のみ、将来の独自の認証パスを選択できます。



Java Programmer IIの準備



Java Programmer IIの準備は簡単です。 主に2つの理由があります。 1つ目は、Oracleが必要なトピックの完全なリストだけでなく、必要な情報がすべて揃っている準備のための完全なチュートリアルも提供してくれたことです。 また、ネットワーク上で質問のあるダンプを見つけることができます。 たとえば、 ここではどうやら、無料アクセスからの質問を含むダンプはすでに削除されています。 しかし、多くの有料サイトで少量で分岐して購入することができます。 私は彼ら全員が同じソースからの質問を再入力したと言いたいだけです。 多くの質問、エラー、および多くの誤った回答。 試験では、質問はインターネット上で見つかる質問と交差します。ただし、強度の10％にもかかわらず、これらの質問には大きな価値があります。 彼らは、これらの質問のレベルと、彼らが正確に何を尋ねるかについて明確に考えを与えます。 公式チュートリアルを学んだ後、それらを完了した後、あなたは自分に自信を持ちます。



しかし、もちろん、試験に合格することが間違いなく難しいチーフアシスタントは、練習、練習、そして再び練習です！ 頻繁に質問を使用しない場合、多数の質問に答えることは非常に困難です。



試験の注文方法



登録のプロセスは、JavaプログラマーIを渡したときとまったく同じであったため、ここでは前回の記事からの抜粋を複製します。



私たちは準備し、それを取ることにしました。 簡単な手順を実行する必要があります。 VUE Webサイトで登録します。 「Oracle証明書を渡すのはどのVUEですか？」-多くの人が考えるかもしれません。 落ち着いた オラクル自体が認証を開発していますが、その行為は認証センターに委ねています。 それは、順番に、地元のセンターと合意しています。 Webサイト（http://vue.com/）で登録するときは、認証の範囲を選択する必要があります。 さらに、認定センターを選択し、都合のよい時間を選択し、自分の住所を書き留めて、指定された時間に表示するだけで十分です。



試験の支払いは銀行振込（カードなど）です。 コスト-150 cu



キエフには約5つのセンターがあります。 ある検索の後、私はwww.flane.com.uaに落ち着きました。 そして、それを後悔しませんでした。 彼らの事務所は、プロセスについて可能な限り多くのアドバイスを提供しました。 地理的には、ポズニャキーに位置しています（地下鉄から5分）。


試験の費用は同じです-150ドル。



変更



変更は150分（2.5時間）以内に行われます。 降伏のために一枚の紙を持ち込むことはできません。全員がその場で発行されます。 試験には90の質問があります。 それらはすべて「正解を選択する」または「複数の正解を選択する」スタイルです。 質問にはマークを付けることができ、答えが不明な場合は後で返されます。



2012年11月以降、納入時に結果はすぐに報告されません。 試験の結果は、試験に合格してから30分以内に指定されたメールにOracleを送信します。 だから、Oracleを通過した後、親切にあなたを少し緊張させています。 しかし、これが唯一の革新ではありません。更新後、Oracleのプロファイルを新しい証明書で更新する必要はありません。すべてが自動的に行われ、24時間後にすべての情報が既にサイトにあります。

Proffesionalを取得した後、成長する専門分野を選択し、最終的に別の証明書で資格を確認できます。 私の選択はEEに向けられ、年の終わりにこのパスに沿って資格を確認する予定です。





さらに、Oracleは優れたJava EEチュートリアルを提供しています： htmlバージョン 、 pdfバージョン 。 しかし、もちろん、この分野の主なトレーニングは私のプロジェクトの1つであり、すでに終わりに近づいています。今年後半にお知らせします。
