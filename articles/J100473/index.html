Starcraft IIの英語の音





ロシアの識別キーの所有者には、英語の音で遊ぶ機会が現れました。 猫の下に非公式な方法が提示されています。



待望のStarCraft 2が数日前に登場しましたが、私は月曜日の夜になんとかゲームを手に入れることができました。 喜びは際限がありませんでしたが、少し遊んだ後、何かが正しくないことに気付きました。 写真は魅力的ですが、音はどういうわけかネイティブではありません。 ローカライザーは、もちろん、よくやった、名声のために一生懸命働いた。 しかし。 それにもかかわらず、最初のStarcraftをプレイしたほとんどの人は英語版と英語の演技に対処しました。 「追加のピロンを構築する必要があります」、「私たちの部隊は攻撃を受けています」、「核発射が検出されました」-1年以上の人生が経った音。



1年間のサブスクリプションで、999人の子供向けのバージョンを購入しました。 その中で、言語の選択は不可能です。 フォーラムを検索しても、良い結果は得られませんでした。 公式ウェブサイトで、ブリザードのアソシエイトが言語の変更に関する次の質問に回答しました。

「ロシア語の識別キーに使用できる言語はロシア語のみです。 英語、ドイツ語、フランス語、スペイン語、イタリア語、またはポーランド語でゲームを試してみたい場合は、ヨーロッパの識別キー”（ 証拠 ）を購入する必要があります。



我々が選択する権利を奪われたことは残念です。 特定の理由により、私はまだゲームのヨーロッパ版を購入するつもりはありませんが、私は本当に英語の音が欲しいです。 私の友人（ハブではない）は、この問題を解決する彼自身の方法を提案しました：



ロシア語のライセンスを英語の音声で再生する場合は、次の手順を実行する必要があります。

1.英語のサウンドを使用するスクリーンセーバーが存在するように、英語バージョンのファイルを取得します。

\ Starcraft II \キャンペーン\ Liberty.SC2Campaign \ enGB.SC2Assets

\ Starcraft II \キャンペーン\ Liberty.SC2Campaign \ enGB.SC2Data



2.ユニット音：

\ Starcraft II \ Mods \ Liberty.SC2Mod \ enGB.SC2Assets

\ Starcraft II \ Mods \ Liberty.SC2Mod \ enGB.SC2Data



enGBの代わりに、ファイルの名前はenUSである場合があります。いずれにしても、ruRUに変更し、ロシア語バージョンに置き換えます。 ファイルを置き換える前にバックアップすることを忘れないでください。



英語の音声ファイルは、次の方法で取得できます。

1.トレントトラッカーを通じて英語のクライアントを見つけます。

2. 音声ファイルのみをダウンロードします 。
