同僚！



8月7日、モスクワ時間の12:00 に、 NetWrixソリューションを使用した企業環境でのアカウントロックの管理に関するウェビナーを実施しています 。



アカウントロックの管理は、特に手動で行う場合、ヘルプデスクの専門家とシステム管理者から最大30％の時間がかかる可能性があります。 そしてここでは、NetWrix Account Lockout Examinerなどの特殊なソリューションが助けになり、リアルタイムでロックの事実と理由を通知します。 NetWrix Password Managerを使用すると、最も一般的なロックの問題の1つであるユーザーが忘れたパスワードを解決できます。



NetWrixのロシアのオフィスでは、Webセミナーで、NetWrix Account Lockout ExaminerとNetWrix Password Managerのアカウントロックを管理するための特別なソリューションを使用する利点を説明します。



ウェビナープラン：





1. 手動ログ分析に基づいてアカウントロックを管理するアプローチの制限は何ですか



2. NetWrix Account Lockout Examinerの 利点は何ですか？

•ブロックの事実に関するヘルプデスクオペレーターへの通知。 アラートを設定します。

•ブロックの理由の決定

•Webインターフェイスでの作業

•ユーザーがアカウントを個別にロック解除できるリモートコントロール機能



3. ユーザーが忘れたパスワードの変更に費やす時間とお金を最小限に抑える方法



4. NetWrix Password Managerセルフサービスパスワード管理ソリューションを使用して、忘れられたパスワードの問題を簡素化する方法：

•プログラムの3つのポータルの機能（ユーザー、ヘルプデスク、管理）の検討

•パスワード変更に対する管理者による制御の可能性

•ユーザーはどのようにパスワードを個別に変更しますか？



などなど！
