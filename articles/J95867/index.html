カーナビゲーターでの移動：大画面への大画面

2001年に、たまたまWindows Mobile用のナビゲーションシステムを開発しました。 当時、それはモスクワにとってもエキゾチックで高価なものでした。 PDAは急速に改善され、人気のピークを待っていました（有名なiPAQを思い出してください）。 Casio E125の重量のある「レンガ」の開発を行いました。 民間のGPS受信機では、精度に厳しい制限がありました。少なくとも30メートルの誤差があり、ルート誘導タスクは非常に重要なものでした。 高価なフラッシュメモリの量には厳しい制限があるため、各家とすべてのインターチェンジとレーンを備えたモスクワの地図は、独自の形式で8 MB、モスクワ地方全体-30 MBを占めていました。 残念ながら、その後、過大な需要のためにビジネスを凍結する必要がありました。 その期間のすべての困難は別の記事に値すると思います。



しかし、時間が経つにつれて人口が増加し、技術が安くなり、10年後、ナビゲーターのない長距離の自動車旅行はむしろ例外になりました。



モスクワからクリミアに車で行くと、私は幸運にも印象的なLexand ST-610ナビゲーターを手に入れることができ、10年間の開発の結果を見ることに熱心でした。



道路上でナビゲーターを準備するのにかかる時間は比較的少なかった。 過去1年間、私はすぐにPDAでいくつかのナビゲーションプログラムを試し、それらを一緒にモスクワを旅行しました。旅行の時までに、ついにiGo 8に落ち着きました。最も不適切な瞬間。 iGoを使用すると、これは起こりませんでした。 当然、すべてが改善されており、おそらく最新バージョンがより安定して動作するか、特定の問題が発生しました。 しかし、私は実証済みのオプションを好みました。そうすれば、荒れ果てた道のどこかにナビゲーションなしで放置される危険がありません。 しかし、すべてについて順番に話しましょう。



パッケージバンドル



今日では非常に標準的です：

  • 実際、スタイリッシュな薄型ケースのナビゲーター
  • 車のガラスホルダー
  • 車の充電
  • USBケーブル（同期+充電）
  • スタイラス
  • ソフトケース
  • ガイド














スタイラス以上のものはありません。 このような大きな画面では、デバイスをリセットすると便利な場合があります。 後で、以下で説明するように、これは完全に真実ではないことが判明しました。 ところで、それを見つけるのは簡単ではありませんでした：人間工学のために、それはケースの丸い後壁とほとんど融合します。



ただし、ナビゲーターをメディアプレーヤーとして使用する場合に必要になるのは、独立して購入することが提案されている自律型ネットワークチャージャーです。 しかし、結局のところ、主な機能は車内のナビゲーションであるため、このマイナスはほとんど深刻と見なすことはできません。



次に、デバイスの特性を見てみましょう



スクリーン







もちろん、これはそもそも魅力的なものです。 対角線6インチ、解像度QVGA 480 * 272、65,000色のTNマトリックス上の抵抗性 TFTスクリーンにタッチします。



最初は、このような対角線の解像度では十分ではないように思われましたが、ナビゲーションには十分であることが明らかになりました。 500 MHzプロセッサでカードを動的にレンダリングすると、解像度が高くなると速度が低下する可能性が非常に高くなります。 ただし、保存したいという単純な要望があるかもしれません。 ST-610の次のバッチは他の技術的特性を伴うという未確認の情報がまだあります。変更は画面の解像度と品質に影響するだけです。



指指向のインターフェースが宣言されています。 なぜ「宣言」されているのですか？ 抵抗マトリックスでは、指先で押すと常にうまくいかないため、指の爪やスタイラスを使用する必要がありますが、これは画面にとってあまり便利で安全ではありません。 問題なく、フィンガープレーンでマップをスクロールすることができました。



デスクトップ、非タッチ、モニターのように、タッチごとに非常に顕著な「波」が画面上に散らばるので、私はまだそれが好きではありませんでした。 通常のPDAやコミュニケーターでは、そのような現象はありません。 私はタッチ技術にあまり精通していませんが、これらはこのタイプの安価なマトリックスの機能でもあると思います。



画面が眩しくない。 ただし、すべてのTNマトリックスと同様に、視野角は小さく、特に垂直方向に小さくなります。 これはドライバーにとってそれほど重要ではありませんが、「ナビゲーター」にとっては不便です。



そして、私はまだ何らかの保護を考えています-デバイスが鋭くて頑丈なものに落ちた場合、画面は非常に簡単に損傷します。



全体的に、大画面で運転しながら地図をスクロールする方がはるかに便利です。ズームはそれほど必要ありません。すべての碑文とオブジェクトが完全に表示されます。 これは否定できない利点です。



ケース、寸法、重量



128x82x12.7 mm、195。ケースはスタイリッシュで、バックラッシュやきしみがありません。 このような対角線を持つデバイスの記録的な厚さを持っています。 軽量性も重要です。ナビゲーターが外出中に誤って脱落する可能性は低くなります。



















カーマウント



本当に気に入りました（上の写真を参照）。 非常に高品質になりました。 両方の自由度が任意の位置にしっかりと固定されています。 手間をかけずに簡単かつ迅速に固定します。 運転中、デバイスはたむろせず、実際には震えません。 ガラスからの吸盤は、キャビン内が極端に高温になった場合でも、自然に落下することはありませんでした。 しかし、例えば、私はオフィスでプラスチックに貼り付けようとしました-それはしばらくして外れました。 一般的に言えば、吸引カップに加えて、ネジなどのより信頼性の高いマウントを考え出す時が来ました。 一部の車では、たとえば換気グリルで使用できます。 おそらく特定の基準が必要です-それから状況は改善します。 しかし、残念ながら、自動車メーカーは非常に保守的です。



オペレーティングシステム



Microsoft WinCE.NET 6.0。 最も人気のあるナビゲーションソフトウェアのほとんどがWindows CEのバージョンに存在することを考えると、この選択は非常に自然です。 Windows Phone 7のリリース後に状況がどのように変わるかはわかりません。



ナビゲーターには、独自の本番のシェルが事前にインストールされていますが、これについては以下をご覧ください。



記憶



RAM（SDRAM）64MB。 いくつかのレビューによると、そのようなボリュームは重いナビゲーションソフトウェアには十分ではないかもしれませんが、問題はありませんでした。 iGoなどのプログラムでは、32 MBの実際の問題とフリーズが発生します。 悪名高いsys.txtを編集しても、必ずしも役立つとは限りません。 原則として、64 MBは、近い将来問題が回避されることを保証するために害を与えることはありません。 しかし、彼らはここで保存することにしたようです。



ROM（NANDフラッシュ）4Gb。 このボリュームでは、いくつかのナビゲーションプログラムをマップと共に記録できますが、これには外部フラッシュドライブを使用することをお勧めします。正直なところ、ハードリセットの場合にROMの内容が保存されるかどうかはわかりません。 私のバージョンでは、デバイスで1 GBのROMが使用されていました。 これらのうち、約50 MBのサービスファイルと950 MBはロシアの地図（1 kmおよび10 km）を備えたNavitel Navigatorです。 PDFにはロシア語のマニュアルがあります。



MicroSD拡張スロット（最大32GBのSDHCカードのサポート）。 ここでは可能性は無限です-利用可能なすべてのマップを使用して、考えられるすべてのナビゲーションシステムを携帯できます。 一部の国では、特定のプログラムがより効果的に機能している可能性があります。



CPU



SiRFatlasIV、ARM1136JF-S、500 MHz これは、65 nmテクノロジを使用して作成された新しいプロセッサです。 おそらく、これにより、デバイスはわずかに加熱されます。 プロセッサは十分に高速です-ナビゲーションシステムの速度低下はありません。



GPSチップセット



SiRF GRF3i +、64チャネル。 非常に安定した受信、私は衛星を失ったことはありません。車に座っている間は、ナビゲーターを安全に手に持つことができます。 最初の開始は約2分間続きます。 ナビゲーターが非常に正確に位置を決定することが好きでした。 情報が正しい場合、受信機の精度は約3メートルです。これは、道路だけでなく車線も自信を持って決定するのに十分です。 古いレシーバーが30メートルまで「粗く」したときの苦痛を覚えています。 このシステムは、複雑な交差点や隣接する道路を走行する場合に特に動作が悪く、これはどのアルゴリズムでも修正できませんでした。 現在、原則としてそのような問題はありません。



コンピューター接続



USB 2.0、miniUSBケーブルが含まれています。 通常のフラッシュドライブまたはMicrosoft ActiveSyncの2つのモードで動作します。 選択は、USBケーブルが接続されたときに表示されるダイアログで行います。



データのコピーは比較的遅く、1秒あたり約1 MBです。 しかし、私はこれが大きな問題だとは思わない ナビゲーションソフトウェアはあまり頻繁に変更されません。



フラッシュドライブモードで問題が発生しています。ナビゲーターが不明なデバイスとして認識されています。 ActiveSyncモードは状況を保存しました。 おそらく私が聞いた限りでは、Lexandは通常それで問題ないので、未加工のファームウェアが原因でしょう。 当然、カードリーダーを備えたmicroSDカードを使用する場合、このような問題は発生しません。



栄養



容量850mAhの内蔵リチウムイオンバッテリー、最小バックライトで充電せずに動作する時間-最大2.5時間、最大-最大1時間、バッテリーのフル充電時間-2〜3時間。 これは本当に小さいです。 つまり とにかく車の中で 通常、このデバイスはシガーライターで駆動されますが、メディアプレーヤーや歩行者ナビゲータとして使用することはほとんど不可能です。 長い旅の車であっても、電話の充電が必要になることがあり、その間ナビゲーターは簡単に座ります。



さらに、何らかの理由で、バッテリーが常に充電されたわけではありません。 信頼できないUSBコネクタが原因であるか、私のシガーライターが理解できませんでした。 問題がファームウェアにある可能性さえあり、ファームウェアには常に料金が含まれているとは限りません。 何らかの方法で、完全放電から外出中に数回オフになり、2回目の電源投入後に正常に充電を開始しました。 点火中の電力サージはデバイスを損傷する可能性があるため、シガーライターワイヤーはエンジンが作動しているときにのみ挿入できることを思い出してください。



追加機能



FMトランスミッター。 FM信号を使用して、ナビゲーターからの音声（音声プロンプト、音楽、ビデオからの音声）を車に組み込まれたオーディオシステムのスピーカーに出力できます。 旅行で、私はセットアップすることができませんでした。 後で、このためにトランスミッターがアンテナのように機能する接続されたUSBケーブルを必要とすることを読みました。 ただし、原則として、内蔵スピーカーの音量で十分です。 さらに、長い旅行では、通常ラジオで音楽を聴きます。ナビゲーターからの音声出力の場合、これは不可能です。 ラジオの音とナビゲーターの音を混ぜることができたら、この機能は本当に便利です。 レビューによると、送信機は7メートルの距離まで放送し、近くの車のラジオを消すことさえできます。



マルチメディアアプリケーションとユーティリティ：ビデオ、オーディオ、写真、テキスト、ゲーム、カレンダー、メモ、電卓。 この標準セットはすべて、ほとんどのナビゲーターにありますが、表示用です。 私も試しませんでした。 使用した人は、映画を低解像度につまむべきであると不満を言った、いくつかの音楽フォーマットなどのサポートはありません。 これは、Windows CEには、ナビゲーターに組み込まれているメディアプレーヤーよりもはるかに機能的なメディアプレーヤーがあるという事実にもかかわらずです。 もちろん、テキストファイルのリーダーは、1つのエンコードのみで、txtのみを開きます。 ルートに関する情報、ルートの説明、モーテル、キャンプ場などをテキストファイルに書き込むと便利な場合があります。 画像ビューアーは、st-570モデルのレビューで判断すると、4 Mbを超える画像を表示しません。







2.5mmステレオヘッドフォンジャック（別売）。 また、ナビゲーターをプレーヤーとして使用することの複雑さのために、むしろショーのために。 デバイスにはハードウェアの音量調節がなく、すべてがメニューから行われますが、特に乗客の1人が眠りに落ちたときに外出先で音声プロンプトの音量を下げる必要がある場合は特に便利ではありません。



Lexandナビゲーターは、Bluetoothモジュールがないため非難されており、GPRSを介して交通情報を取得することは不可能です。 ただし、同社はSIMカードを搭載したモデルを準備しているため、問題はすぐに解決されます。



デバイスインターフェースは好みに合わせてカスタマイズできます。グラフィックファイルを使用してデスクトップの背景を設定し、スクリーンセーバーをカスタマイズします。 これらの目的のためのフラッシュドライブで、私は非常に素晴らしい主な自動車ブランドのロゴを見つけました-それは通常のナビゲーションのように見えます。



ナビゲーターの残りの部分は、大型モデルのような機能でオーバーロードされていません。 結局のところ、彼は主な機能を果たさなければならず、残りはそれほど重要ではありません。



ナビゲーションソフトウェア



Lexandの主な利点は、ナビゲーションプログラムを起動できることです。 実行可能ファイルへのパスを指定するだけで十分です。











ロシアでは、3つのナビゲーションシステムのいずれかが装備されています。

  • Navitel Navigator：多数のロシア地域の詳細な地図、ターゲットを絞った計画を備えた地域センターの詳細な地図
  • ProCity：独自の生産のロシアの非常に詳細な地図、道路の交差点のエミュレーション、複数車線の高速道路を走行する際のプロンプトのモード
  • CityGID：サンクトペテルブルクとロシア北西部の航海で有名なリーダー。


しかし、海外旅行には、iGoを好みました。iGoは問題なく実行し、内蔵フラッシュメモリに書き込みました。



iGoについて一言。 これはよく考えられたシステムです。 外出先で必要になる可能性のあるものはすべて、メイン画面で利用できます。 マップの表示には3つのモードがあります。3D、北向き、移動方向の向きのフラット-誰でも快適です。 画面には、速度、予想される移動時間（ところで、驚くほど正確です！）、最も近いジャンクションまでの距離が表示されます。 システムは速度の警告を出します（しきい値は調整可能です）-常に助けてくれました。 追加の画面で、移動距離、最大速度、高度、移動方向など、多くのパラメーターを見ることができます。 プログラムはスキンをサポートし、音声演技のためのいくつかのオプションを提供します。 ルートを異なる地図、たとえばロシアとウクライナの間に配置できると非常に便利です。 遷移はユーザーには見えません。 音声プロンプトは非常に高品質であり、最低限必要です。 ルートを離れるときに、質問なしで新しいルートを再実行します。 彼が道路を知らない場合（たとえば、新しいバイパス）、彼は彼女に知られているトラックに到達するまで単純に黙っています。 表示モードの切り替え、ユーザーポイントの検索（ベースは非常に大きく、常に更新されます）、迂回の検索ボタン用のホットボタンがあります。 マップは非常に有能に表示されました-ラウンドアバウトに近づくと詳細が非常に詳細になり、出口のない長いセクションに出るとマップが遠ざかり、視認性が向上します。 さらに多くの楽しいものがありますが、これらはすべて別の話に値します。



私はただ、iGoが3千キロすべてにわたってホバリングしたことはなく、私たちが標識をたどっている新しい高速道路ハリコフ-ドネプロペトロフスクで一度だけ混乱して、ルートに沿って私たちを完全に案内しました。 トラックは新しいマップに既に追加されていると思います。



すべてのナビゲーションプログラムでは、特に大都市で敷設されたルートを盲目的に信頼するべきではありません。 ちなみに、ルートの計算にはさまざまなモードがあります-最短、最適、最速。



結論



正しい中国語デバイスがあると言えます。 肯定的な印象は軽微な欠陥を大幅に上回っています。 メッセンジャーの現在の最低価格は6990ルーブルです。 他の店では、何らかの理由で、より高価です-8400ルーブルから。 正直に言って、610番目のモデルが画面の対角線でのみ弟と異なることを考えると、高価です。 もちろん、ナビゲーションソフトウェアを使用せずに2倍安く中国で注文することもできますが、メールを思い出すと、欲求はすぐに消えます。
