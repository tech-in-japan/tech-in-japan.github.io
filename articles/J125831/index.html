Googleドキュメントには新しいインターフェースがあります









Googleは引き続きGoogle+のスタイルで製品のデザインを変更し、今回はGoogleドキュメントの新しいインターフェースを導入しました。 それを見るには、右上隅のリンク「新しいインターフェース」をクリックする必要があります。 同じメニューから、クラシックインターフェースに戻ることができます。



私のように、最初は新しいインターフェイスがスペースを取りすぎているように思える場合は、 設定でデザインの密度の高いバージョンを選択できます（密度の高いビューを使用） 。



新しいインターフェイスには、マウスを使用せずにドキュメントのリスト内を移動できるホットキーがあります。 たとえば、ドキュメントを選択するには、 ⇓ （下矢印）を使用してリストを下に移動し、 ⇑ （上矢印）を使用して上に移動します。 目的のアイテムが見つかったら、 aを押して [ アクション ]メニューを開きます。 次に、 ⇑および⇓キーを使用してアクションメニューを移動し、を押してアクションを選択します。



新しいドキュメントを作成するには、 Shift + Tを押してテーブルを作成します-Shift + S 、コレクションを作成します-Shift + C
