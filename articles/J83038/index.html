OpenGLビデオチュートリアル

良い一日！



これはhabrasocietyでの私の最初のブログ投稿です。



多くの初心者プログラマーが遅かれ早かれ、コンピューターゲームのプログラミングに興味を示すようになります。 この傾向は私には通りませんでした。 そのような場合にいつも起こるように、私はあなたがあなたの友人に自慢できる何かの「一種」を書きたいです:)。



それはすべて、「どこから始めればよいのか」という質問から始まりました。 インターネットで大騒ぎし、記事を読んで、最初にIDEを決定する必要があるという結論に達しました。 Visual Studio + Nvidia SDK10がIDEとして選択されました。 Visual Studioを選ぶ理由 この質問にはいくつかの答えがあります。

1.このIDEにはプログラミングの経験がありました。

2. Visual Studioを使用すると、エラーからコードを簡単に削除できます。

3.必要なライブラリを簡単に接続できます。

その後、APIで疑問が生じました。 DirectXまたはOpenGLを選択するもの。 クロスプラットフォームでコードがシンプルなため、OpenGLを使用することにしました。

グラフィカルインターフェイスとして、Glutで停止しました。 理由：ウィンドウシステムからの独立（Windows、Linux、Macで簡単に構成できます）。



誰もがすでに知っているように、学ぶ方法はいくつかあります。

プログラミングを教える古典的な方法は、本と必要なソフトウェアです。 どのチュートリアルが役立ったのか（OpenGL Red Book、OpenGL®SuperBible）

しかし、非古典的な方法、つまり ビデオチュートリアル：ビデオ+必要なソフトウェア。



グラフィックプログラミング（C ++ OpenGL）のレッスンをいくつか紹介します。 <英語ですが、すべてが明確です:)）
