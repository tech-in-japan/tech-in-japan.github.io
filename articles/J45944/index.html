災害8：MacBookが汚れる

開始するには、清潔で糸くずの出ない布またはマイクロファイバーでMacBookを拭きます。 マジックスポンジMagic Eraserを水道水の細い流れに浸し、少し湿るまで絞ってください。 Macbookをスポンジで拭き、別の方向ではなく、一方向に動かします。 キーボードおよびすべてのポートまたはその他の開口部に近づかないでください。 終了したら、他の糸くずの出ない布でコンピューターを拭いて乾かします。 少なくとも1滴の水滴が機械に侵入する可能性がわずかでもある場合は、電源を入れる前に6時間電源を切ったままにしておきます。



これはすべて複雑すぎます。 たとえば、ラップトップなしで6時間滞在したくないのは、それがあまりにも汚れているため、最初に洗ってから乾かさなければならなかったからです。 したがって、1週間に1回オフにするか、汚れた場合はほこりから拭き取り、汚れないようにすることをお勧めします。 さらに、素晴らしい救済策があります：素晴らしいGelaSkin （29.95ドル）は、あらゆる好みのためにトップカバーに接着することができ、手首の下の領域はMoshi Palmguardで保護することができます（ノートパソコンのモデルに応じて20.00ドルから28.00ドルの費用）。 同時に、ラップトップは美しくて珍しいように見え、ケースは汚れから保護され、プラスチックは無傷です。 私はすでに数ヶ月間両方でそれを使用しており、非常に満足しています。

生存戦略： イタリック体で多くのテキストを読む必要がないように、この点が大幅に修正され、補足され、ほぼ完全に書き直されていることに注意してください。 したがって、元の記事に書かれている内容と少し矛盾していることに驚かないでください。



小さなフォントファイルは大きな問題を引き起こす可能性があります。 破損したフォントは、情報の印刷または表示に問題を引き起こし、アプリケーションがクラッシュすることさえあります。 また、フォントキャッシュの乱雑さは、画面上の情報の再現に関するあらゆる種類の問題を引き起こす可能性があります。 したがって、Macのフォントは非常に繊細に処理する必要があります。 決してシステムフォントに触れてはいけません。 Lucida Grandeを誤ってアンインストールした場合、Mac OSはまったくロードされません。



T2ブログのヒントを使用して問題を解決してください。



1.フォントを確認します。 Mac OS X（TigerまたはLeopard）では、 アプリケーションフォルダーに移動してFonts（Font Book）を起動すると、破損したフォントを検出できます。 コマンド（⌘）キーを押しながら、危険なフォントを選択します。 最近、信頼できないサイトから無料で購入またはダウンロードしたフォントが含まれている場合があります。 Command + Aまたは（⌘+ A）を押すと、初めてすべてのフォントを選択できます。 次に、メニューから「 ファイル」>「フォントの検証 」を選択します。 フォントが破損している場合は、十字の入った丸い赤いアイコンが表示されます。 不良フォントのボックスをチェックし（名前の横にあるチェックボックスをクリックします）、[ チェック済みを削除 ]ボタンをクリックしてシステムを悪から守ります。 フォントの横に黄色の警告アイコンが表示されている場合は、問題が発生している可能性があるため、フォントフォルダから削除してください。 手順の最後に、コンピューターを再起動し、この方法で問題が解決するかどうかを確認します。 Mac OS Xの以前のバージョン（10.3以前）を使用している場合は、問題の原因となっているものを特定するために、作品5のフォントグループを一度に手動でテストする必要があります。



2.フォントキャッシュをクリアします。 フォントキャッシュは、Mac OS Xユーザーにとって大きな問題になる可能性があります。 一部のフォントキャッシュはOS自体によって処理され、他のフォントキャッシュはアプリケーションによって制御されます。 いずれにせよ、個々のキャッシュファイルを損傷することは非常に簡単です。その結果、フォントがゆがんで、画面上の情報が正しく表示されません。 無料のFontNukeアプリケーション（3.8 MB）は、問題への対処に役立ちます。 システムフォントキャッシュ（Mac OS Xフォントキャッシュファイル）、およびAdobe、Microsoft Office、Quark Expressアプリケーションのフォントキャッシュを削除します。 このユーティリティを開始する前に、すべてのドキュメントを保存し、すべてのプログラムを終了してください。フォントキャッシュをクリアした後、システムを再起動する必要があるためです。



追加の$ 100があり、その使用方法がまだわからない場合は、有料のフォント整理アプリケーションのいずれかを購入できます。ExtensisSuitcase Fusion 2 （ 99.95、31.3 MB）またはInsider SoftwareのFontAgent Pro 4 （ 99.95、15.1 ） MB）。 どちらのアプリケーションにも、フォントを整理して操作するための多くのオプションがあり、両方のアプリケーションには30日間の無料トライアルがあります。 Extensisは、MacとWindowsの両方に有料フォントソリューションも提供しています。 少し安い（69.95ドルまたはExtensis Suitcase Fusion 2の購入時に無料） FontDoctorアプリケーション（7 MB）を購入すると、フォントとフォントキャッシュの問題を修正できます。



3.アカウントのフォントを一時ディレクトリに移動します。 パック/〜/ Libraries / Fonts（/〜/ Library / Fonts）に移動します。ここで〜はアカウントの名前で、そこからファイルをデスクトップ上の一時フォルダーに移動します。 これらのフォントを削除せずに移動してください。 /〜/ Library / Fontsフォルダーにフォントが残っていないことを確認し、システムをリロードします。



4. OSに付属のフォントをこのディレクトリに移動します。 再起動後、フォントの問題は消えます。 ただし、デスクトップには、アカウントにインストールされたフォントを含むフォルダーがまだあります。 それらを1つずつ/〜/ Libraries / Fontsフォルダーに移動して、問題の原因となっているフォントを見つけます。 重要なフォントを最初に移動します。



5.他のすべてが失敗した場合は 、Kurt Langの長く包括的な記事「OS Xでのフォント管理」 （PDF）を読んで、役に立つヒントを探してください。
