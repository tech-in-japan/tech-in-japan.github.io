古くてクールなDOS代替コマンドの助けを借りて、仮想ディスクを作成できます。 Runフィールドに次のコマンドを入力します（Start-> Run）：subst Z：C：\。 新しいZ：ドライブがあり、C：ドライブと同じ内容になります。 C：ドライブに変更を加えると、それらは自動的にZ：ドライブに反映されます（逆も同様です）。 仮想ディスクをフォルダーに割り当てることもできます。たとえば、コマンドsubst X：C：\ Windowsを実行すると、X：ドライブが表示され、Windowsフォルダーの内容が含まれます。 仮想ドライブを削除するには、コマンドsubst X：/ Dを実行する必要があります（仮想ディスクXを削除します:)。 subst /？..と入力すると、コマンドのヘルプを表示できます。ただし、再起動後、すべての仮想ディスクが消えるので、常にシステムにそれらを保持する場合は、autoexec.batファイルにsubstコマンドを書き込むか、それ以上レジストリ（RUNセクション）に。 これは次のように行われます：レジストリに移動し（スタート->実行-> regedit）、HKEY_LOCAL_MACHINE \ Software \ Microsoft \ Windows \ CurrentVersion \ Runブランチを開き、[編集]メニュー-> [作成]-> [新しい文字列]パラメーター（文字列値）、NewDiskなどの任意の名前を入力します。 新しいパラメーターをダブルクリックし、subst L：C：\などの目的のコマンドを入力します。 これで、システムに常にL：ドライブがあります。

ちなみに、このコマンドはWindowsの全ライン（9xからXPまで）でうまく機能します。
