WebOSの進化



これは、Webの現在および将来の過去の過去のバージョンです。

このグラフは、Webの現状に関する私たちの考えを示しています。 これは、PC時代から将来の本格的なWebOSの可能性への進化を示しています。



Web x.0という用語は、Webのさまざまな時代を区別するためだけに使用されていることに注意してください。 今、私たちは（誰もが知っているように）Web 2.0の時代にあり、2010年頃にはWeb 3.0の時代に無事に到達するでしょう。



Web 2.0は、私たちにとってたった1つのことを意味します-エンドユーザーのためのはるかに多くの機能。 Web 1.0の場合と同様に、彼はすでに視聴者のカテゴリから削除されています-現在、彼は情報世界のメンバーです。 もちろん、テクノロジーはWeb 3.0の加入時までに大幅に変化しますが、その本質はWeb 1.0からWeb 2.0への移行の場合ほど大きく変わらないはずです。
