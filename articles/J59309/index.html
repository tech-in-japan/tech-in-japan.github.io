リトアニアはインターネット速度の世界的リーダーですか？

リトアニアのメディアは、彼らの小さな国がインターネットの速度の面で世界で最初に来たことを報告することを誇りに思っています。 確かに、彼らはアップロードの速度、つまりリターンチャンネル（8.08 Mbit / s）でのみリードし、ダウンロードの速度、つまりダイレクトチャンネルでは4位（11.28 Mbit / s）のみを占めると言われています。 これは、このサイトでチャンネルの厚さを測定するために訪れたすべてのリトアニアのインターネットユーザー向けのSpeedtest.netサービスのグローバル統計です 。 リトアニアからのテスト済みユーザーの数は報告されていません（おそらく1人だけですか？）。 一般に、Speedtest.netは月に約2,000万のテストを実行します。







Speedtest.netの統計の信頼性（つまり、11 Mbpsと8 Mbpsの数値）は、実際のユーザーに関するデータを持っているノルウェーのホスティングプロバイダーHostexの代表者によって確認されています。



韓国（17.83 Mbit / s）、日本（16.07 Mbit / s）、スウェーデン（11.55 Mbit / s）のみが、データのダウンロードと受信の速度がリトアニアを上回っています。 データの送信速度に関しては、日本は主要なリトアニア（7.49 Mbit / s）に遅れており、ブルガリアは3位（4.43 Mbit / s）です。



ロシアは、それぞれリターンチャネルとフォワードチャネルの速度で5位と10位を占めています。 モルドバ-9位と11位。 ベラルーシ-69か82か所。
