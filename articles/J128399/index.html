TwitterがWeb分析ツールを開始









あなたのサイトはTwitterからどれくらいのトラフィックを獲得していますか？ この質問に対する答えは、Twitter Web Analyticsと呼ばれるTwitterによって発表されたWeb分析ツールによって可能になります。



この新しいツールは、ウェブサイトの所有者にTwitterとの統合の有効性に関するより多くのデータを提供するように設計されています。 これは、Twitterが6月に買収したソーシャル分析会社であるBackTypeのテクノロジーに基づいています。



BackTypeの創設者であり、新しいTwitter従業員であるChristopher Goldが説明したように、Twitter Web Analyticsは、Webサイトの所有者が3つの重要なポイントを理解するのに役立ちます。



スクリーンショットでわかるように、分析パネルは4つのタブで構成されています。 最初の「トラフィック」には、サイトへのリンクがあるツイートの数と、これらのリンクのクリック数が表示されます。 チャートは、当日、先週、先月に利用できます。 ツイートタブには、サイトへのリンクを含むすべてのツイートと、サイトの組み込みTwitterボタンから送信されたすべてのツイートが表示されます。 管理者は、このパネルからツイートをリツイートしたり、ツイートに返信したりできます。 Twitterボタンタブには、サイトでTwitterボタンがどの程度アクティブに使用されているかが表示され、コンテンツタブには、サイトで最も効果的なページが表示されます。



TwitterのWebビジネス開発エイプリルアンダーウッドのディレクターによると、データからボットとスパムが除去されます。 Twitterは開発者ツールAPIもリリースします。



Twitter Web Analyticsは無料で、これまでのところベータ版です。 今週、パートナーの小さなグループがアクセスできるようになり、数週間のうちにすべてが開始されます。
