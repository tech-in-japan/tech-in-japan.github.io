Adobe Flash 11およびAIR 3リリース候補

Flash 11およびAIR 3の候補がダウンロードできるようになりました。



Flash 11の新機能：



* Stage 3D Accelerated Graphics Rendering

* 64ビットシステムのサポート-Mac、Linux、Windows。

*テレフォニー用のG.711オーディオ圧縮コーデック

* H.264 / AVC SWエンコーディング

*ソケット進行イベント

* JPEG-XRサポート



ここからダウンロード： http : //labs.adobe.com/downloads/flashplayer11.html



AIR 3の新機能：



*ネイティブ拡張（おそらく最もおいしい機能は、ネイティブ拡張を可能にします

組み込みバージョンのAIRを使用したインストール）

*フロントカメラのサポート

*カメラポジショニングAPI

*モバイル用の暗号化されたローカルストレージ

* Androidライセンスサービスのサポート

*デバイススピーカー制御



ここからダウンロード： http : //labs.adobe.com/downloads/air3.html



マイナスのうち、これまでのFlash 11プレーヤーのデバッガーバージョンは32ビットのみでしたが、将来的には64バージョンのサポートが予定されています。
