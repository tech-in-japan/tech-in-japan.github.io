Skolkovo常駐のTickets Cloudは、Amazon Web Servicesから10,000ドルのクラウド助成金を受け取りました

Tickets Cloudチケットのオーガナイザーおよびディストリビューター向けのロシアのb2bプラットフォームは、Amazon Web Services（ AWS ）から助成金を受け取りました。 AWSは、クラウドリソースを使用するために1万ドルのスタートアップを割り当てました。



Firrmaによると、AWSはTickets Cloudにプロジェクトの開発に必要なサービスとインフラストラクチャを提供します。 AWSは、特別なプログラムの一部としてPinterest 、 Spotify、 Dropboxも使用します。 3年以内に存在するスタートアップはそれに参加できます。 さらに、若いプロジェクトに引き付けられる投資の数は、150万ドルを超えてはなりません。



Skolkovo Foundationの住民は、2014年からAWSプログラムに参加できました。 Tickets Cloud は今年の5月に常駐しました。 このプロジェクトは、「大きなデータ配列の処理と分析」の方向で選択されました。 スタートアップの投資家は2be.lu投資ファンドであり、2014年12月に30万ドルを投資して13.3％のシェアを獲得したビジネスエンジェルボグダンヤロヴォイでした 。 Tickets CloudはStarta Capital Accessor Fundのポートフォリオ企業でもあり、2014年7月にMoscow Seed Fundと共同でプロジェクトに450万ルーブルを投資しました。



Tickets Cloudは、企業が市場に参入したり、ビジネスの売り上げや連絡先を増やしたりするのに役立ちます。 すべてのトランザクションと関係は自動化されています。



アマゾンウェブサービスは、クラウドウェブサービスプラットフォームのインフラストラクチャです。 2006年初めにAmazonによって導入されました。 クラウド部門の営業利益は、前年（77百万ドル）と比較して407％増加し、391百万ドルに達しました。 4〜6月のユニットの収益は81％増加して18億2,000万ドルでした。 Amazon Web Servicesは、この四半期に350の新しい機能とサービスを導入しました。
