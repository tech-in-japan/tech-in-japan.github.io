Skype検閲





今朝、数人の友人がSkype経由でリンクを転送できないことに不満を漏らし、ウィルスがないかコンピューターをチェックするように頼んだ。 詳細を尋ねると、skypeは人気のあるオンラインストアdx.comからリンクをカットすることが判明しました（おそらく、私が使用している他のものもDXのみに影響を与えました）。



プラットフォームに関係なく削除が行われ、クライアントのバージョンは非常に面白いです。 リンクが最初に表示され、1秒後に削除されます。 これは、すべてのメッセージが何らかのロボットで複製されているように見えることを意味し、メッセージで次に何をするかをすでに決定しています。 これらすべてのために、例えば、どこかにスペースを入れるなど、検閲されたメッセージを編集すると、メッセージはフィルターを通過せず、通常は宛先に届きます。



更新：グループチャットへのリンクは無修正です。



Update2： MERRONは 、古いクライアントから送信する場合、メッセージが無検閲に到着することを明確にしました。 Windows XP / 7のskype 5.5でテスト済み。 おそらく、管理のためにメッセージをどこかに送信するのはクライアントです。



Update3：これに関するトピックが Skypeコミュニティフォーラムで作成され、すべての新しいトピックがそこにリダイレクトされます。



Update4：MSKの正午に、ディスカッションに次のメッセージが表示されました



私の勝利| linux | androidクライアントでは、dxとのリンクが正常に到達し始めました。 それにもかかわらず、検閲手段を持っているという事実は事実のままです。
