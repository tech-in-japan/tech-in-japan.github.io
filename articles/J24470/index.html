リスト内の自家製マーカーの配置

タスク：リスト用の自家製マーカーを作成します。



予測できないインデントのため、list-style-imageの使用は必ずしも便利ではありません。 したがって、私は型構造を使用していました

ul li {

padding-left：10px;

背景：url（bullet.gif）0 5px no-repeat;

}


今日、背景の位置を相対的な位置に設定することで、この構造を少し改善するというアイデアが生まれました。

ul li {

padding-left：10px;

背景：url（bullet.gif）0 0.5em繰り返しなし;

}




利点-フォントがスケーリングされてもマーカーは所定の位置に残ります。 クロスブラウザ。

必要に応じて、写真を追加できますが、私の意見では、すべてが明確です。
