Google分析計算スクリプトの更新

金曜日（12月14日）、GoogleはGoogleアナリティクスカウンターを更新しました。 もちろん、金曜日には新製品に対処したいという欲求はもはやありませんでしたが、今日ではすでにそれを手に入れて、私たちにどんな新しいものが提供されているのかを見ることができました。



いくつかのソースからのデータを比較できる更新されたグラフィカルインターフェイス：





つまり、たとえば、特定のソース（Adwordsなど）からのトラフィックを他の特定のソースなどと比較して変換する方法などを比較できます（ ヘルプを参照）。



また、このベータ版のカウンターの一般向けには、ga.jsページに挿入される新しいスクリプトが表示されます。古いurchin.jsは以前と同じように機能しますが、新しいサイトに新しいスクリプトをインストールすることをお勧めします。 古いスクリプトを新しいスクリプトにアップグレードする機能も提供されています。マニュアルが22ページに書かれていても、読むことができます-T racking Code Migration Guide（PDF） :)



マニュアルでは、新しいスクリプトを古いスクリプトに置き換えることに加えて、新しい製品で使用可能な新しい機能について十分に詳しく説明しています。

  • 仮想ページビューの追跡
  • ダウンロードしたファイルの追跡
  • 複数のアカウントのページを追跡する
  • サブドメインの追跡
  • リンクを使用してドメイン間で訪問者を追跡する
  • フォームを使用してドメイン間で訪問者を追跡する
  • 電子商取引
  • 有機ソースを追加する
  • 訪問者タイプのセグメント化
  • Cookieデータをサブディレクトリに制限する
  • データ収集設定の制御
  • セッションタイムアウトの制御
  • キャンペーンのコンバージョンタイムアウトを制御する
  • カスタムキャンペーンフィールド
  • キャンペーンデータでアンカー（＃）を使用する
  • キーワード無視設定の設定
  • データのサンプリングレートを制御する




何らかの形で、一部の機能はすでに古いバージョンにありましたが、更新後、それらの使用はよりシンプルで直感的になりました。
