Amazon Free VPN

habracatの下で、いくつかの簡単な手順で州で事実上無料のVPNを取得する方法を説明します。



1. aws.amazon.comに登録し、1年間無料のコンピューター時間を取得します。 有効なカードが必要です。



2. Ubuntu 10.4および10ギガバイトパーティションのリージョンで利用可能なUS-East Micro Instanceを起動します。これは無料パッケージにちょうど収まります。 インスタンスの番号はami-c2a255abであり、コミュニティAMIでUbuntu 10.4を検索することで特定されます。



3. PuTTYgenを介してキーをppk形式に変換することを忘れずに、たとえばPuTTYを介してSSHを使用して、 ubuntuの下でマシンにログインします。



4. この記事を使用して OpenVpnをインストールおよび構成します。 著者は、仕事を始める前にsudo suを実行する必要があると言うのを忘れていました。



5. DynDNS との同期を設定および構成して、毎回ipを監視しないようにします。これは、各インスタンスのシャットダウン後に変更されます。



6. OpenVPNクライアントを構成します 。私の場合、Windows 7が環境として機能しました。

これを行うには、サーバーからファイルをコピーします。

•ca.crt

•client1.crt

•client1.key

これらはテキストファイルであり、awsを含むコンテンツはコピーアンドペーストで取り出すことができます。

C：\ Program Files \ OpenVPN \ configに折りたたむ必要があります。

さらに、ここでは、次のような内容を書き込むclient1.confファイルを作成する必要があります。



クライアント

開発者

リモートec2-XX-XXX-XX-XXX.compute-1.amazonaws.com 1194

プロトUDP

ノバインド

引く

comp-lzo

永続キー

持続する

ca "C：\\ Program Files \\ OpenVPN \\ config \\ ca.crt"

cert "C：\\ Program Files \\ OpenVPN \\ config \\ client1.crt"

キー「C：\\ Program Files \\ OpenVPN \\ config \\ client1.key」





7. OpenVPNを起動すると、シアトルにいます！



PS：Amazonは、1か月あたり15ギガバイトのアップおよびダウントラフィックを1年間提供します。 それ以上のものはギガバイトあたり10セントかかります。 価格はここで見つけることができます： http : //aws.amazon.com/ec2/pricing/ 。
