Core i5を搭載したデスクトップは、アイドルモードでわずか5.9ワットしか消費しません





オランダのアマチュアエンジニアEmile 'Mux' Neyssenは、数年間マザーボードの電力消費を最適化しています。 たとえば、2010年に20ワットしか消費しない本格的なMiniITX / Core 2 Duoコンピューターを組み立て、昨年は8.3ワットの消費でCore i3デスクトップを作成しました。



新しいFluffy2プロジェクト-16 GBのDDR3-1333メモリ、WiFiモジュール、64 GB SSDを搭載したCore i5-3570Kは、元の11.6ワットではなく5.9ワットしか消費しません。 ナイセン自身がメインのホームコンピューターとして使用しています。

注：通常のPCは約95％の時間スタンバイモードになっています



この結果を達成するには、まず、コンポーネントの選択に適切にアプローチする必要があります。 ノイセンは、どのメーカーのどのコンポーネントが望ましいか、エネルギー消費を最小限に抑えるために避けるべきコンポーネントのリストを公開しています。 たとえば、Intelのマザーボードまたは低中価格帯のMSIをお勧めします。どのメモリでもかまいません。



Neissenは、電気回路の各コンポーネントをエネルギー損失の原因と見なし、マザーボードから不要なものをすべて削除します：SATAポート、LED、不要な電圧調整モジュール（VRM）など、機能を損なうことなくエネルギー損失を最小限に抑えます。 実例として、彼の以前のプロジェクトからの2つの電気回路、 Core i3があります。元の回路と不要なコンポーネントを削除した後です。

これが、エンジニアが現在作業しているIntel DQ77KBマザーボードの元の回路図です。







Neussenは、回路内の各回路を電圧計で測定し、スリープモードですべてのコンポーネントの電力がどのように消費されるかを自分で図式にまとめました。







最適化後、システムの消費電力は元の11.6ワットではなく5.9ワット、つまりほぼ半分になります。 最大電力では、コンピューターは元の99.6ワットではなく74.5ワットを消費します。



理想的には、コンパクトで低電力のコンピューターをモニターの背面に取り付けて、ハンドルで持ち運ぶことができます。 システムは次のようになります（Neussenによる3Dレンダリング）：
