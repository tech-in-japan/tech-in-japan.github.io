NUCにESXi 5.1をインストールする（DC3217BY）

タスク： VMware NSX（DC3217BY）に VMware ESXi 5.1をインストールします。



問題： NUC（DC3217BY）にはイーサネットポートがありませんが、代わりにThunderboltポートがあり、Apple Thunderboltイーサネットアダプターが使用されました。



解決策： Broadcom NetXtreme BCM57762ギガビットイーサネット情報をtg3ドライバーアーカイブに追加します。 （販売：Dvid = 14e4：1682）。 ルールetc / vmware / driver.map.d / tg3.mapおよびusr / share / hwdata / driver.pciids.d / tg3.ids from net_tg3.v00
