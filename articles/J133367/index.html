Acer Allegro-私の意見

はじめに-簡単な紹介と写真。



iPhone OSは2007年6月29日にリリースされました。 現時点では、5番目のメジャーバージョンがあります。

Androidオペレーティングシステムは、2008年9月23日にリリースされました。 現時点では、4番目のメジャーバージョンがあります。

Windows Mobileオペレーティングシステムは、2000年4月19日にリリースされました。 彼女は2010年2月15日に亡くなりました。

Windows Phone 7オペレーティングシステムは、2010年10月11日にリリースされました。





Windows PhoneをWindows Mobileから分離したのは偶然ではありません。 VAZ 2101やLada Kalinaのようなこれら2つのOSだけです。 どちらも同じメーカーのもののようですが、「ペニー」から「ガマズミ」までのスペアパーツは立ち上がっていません。 だからここ-マイクロソフトはトリックを投げた-すべての古いアプリケーションは過去のものです。 すべてゼロから。 そして、注意する必要があります-これはオペレーティングシステムに利益をもたらしました。 現在、WP7はマーケットリーダー（Google AndroidおよびApple iOS）と競合できる競争力のあるOSになっています。

Old Believersの新しいOSに対する認識を悪化させる（またはその逆-改善する）ために、Microsoftはデバイスのハードウェア側のフレームワークをしっかりと設定しました。

Microsoftは、Windows Phone 7上のデバイスの最小要件を発表しました。

すべてのデバイスには次のものが必要です。

-静電容量式マルチタッチディスプレイ（少なくとも4つのタッチを同時に認識）。 800x480（WVGA）の解像度で。

-少なくとも1 GHzの周波数のプロセッサ。

-256MBの動作可能および8GBフラッシュメモリから。

-DirectX 10のサポート。

-GPS受信機;

-加速度計;

-電子コンパス;

-FMラジオ。

-フラッシュと少なくとも5 MPの解像度を持つカメラ。

また、すべてのデバイスには、次の6つのハードウェアボタンが必要です。

1.オン/オフ：このボタンには、デバイスのオン/オフ、デバイスのブロック、ディスプレイのオン/オフなど、いくつかの用途があります。

2.音量コントロール：一般的な音量と通話中の音量の両方を調整します。

3.カメラ：このボタンは、（従来のデジタルカメラのように）半押しと全押しをサポートする必要があります。 ボタンを半押しするとフォーカスが調整され、全押しすると直接写真が撮影されます。 さらに、カメラボタンをクリックすると、対応するアプリケーションがアクティブになります。

4.戻る：同じアプリケーションで、またはアプリケーション間で1ステップまたはページ戻ることができます。 他の機能には、メニュー、ウィンドウなどを閉じることがあります。

5.開始：すべてのアクティブなアプリケーションが一時停止され、開始ページが開きます。

6.検索：デバイス上のコンテンツを検索するBing検索エンジン。 ユーザーの現在の場所を特定するために、この機能は、とりわけ、ローカル検索を実行できるようにします。



そして、今日の監視対象のスマートフォンがまさにその特性を備えていることは驚くことではありません。 これは、法律であり、「スマートフォン」セグメントの「ゴールデン」ソリューションでもあります。 少なくともスマートフォンを扱う分野では、オタクではなく「4コア、12メガピクセル、FullHD」。

スマートフォンは手袋のように手の中にあります-適度にコンパクトですが、ディスプレイには解像度に最適な対角線があります。 言い換えれば、ディスプレイが大きければ（そしてスマートフォンもあれば）、木目ができてしまい、「シャベル」ができてしまうでしょう。



ディスプレイマトリックスはTNテクノロジーを使用して作られています-良好な色と視野角。 しかし、これはAMOLEDやIPSとはほど遠いものです。 しかし、それを使用するのは楽しいです、目は幸せです、視角は良いです。

タッチ反応は瞬時に、正確になります。





ディスプレイの下にあるタッチボタンは必ずしも目立つものではありません。暗闇で明るく光るのは良いことです。

物理的なボタン（これは電源ボタン、音量ボタン、オン/オフカメラボタン）はすべて、クロムメッキが施され、光沢があり、薄く、快適です。 偶発的なクリックはありませんでした。





上端：電源ボタンとスリープ解除ボタン、3.5 mmジャック-ヘッドセットとヘッドフォン用。

左側には何もありません。



右側には、音量ロッカー、2ポジションのカメラボタンがあります（画面がロックされていても機能します）。

底部には、マイク穴とマイクロUSBコネクタがあります。







前面には、ガラスで覆われた3.7インチのスクリーン、3つのタッチボタン、光センサー、近接センサー、スピーカー（イベントライトインジケーター付き）があります。







背面には、5 MPカメラとスピーカーグリル（音楽を再生し、チューニングし、スピーカーフォン用に使用）があります。







スマートフォンの本体の素材は、4つの要素に分けることができます。ソフトタッチの背面カバー、ソフトタッチの側面、上端と下端は光沢のあるプラスチックで作られ、ディスプレイはガラスです。







デバイスの設計は非常に厳密で、会社のAndroidソリューション（Liquid）に非常に似ています。 組み立ては良好で、蓋はきしみません、どこにもゆがみはありません、離れません。



スマートフォンでは珍しい唯一のものはSIMカードの着陸です-それは横向きになります。







バッテリーを約2営業日保持しました。インターネット、地図、友人のフィードの読み取り、30〜45分間の通話、SMSを常に使用しています。

特に積極的に使用していない場合は、30分間通話するだけで、SMSは3〜4日間有効です。 しかし、スマートフォンの場合、そのような体制には意味がありません。







Acer Allegroは深刻な感情を生み出します。つまり、一般的に、単一の「WOW！」ではなく、Allegroと通信するときです。 他のみんなとは違います！」そしてそれは良いことです。 このスマートフォンは、誇示することなく、スマートフォンを必要とする人にとって理想的な主力製品です。 そして、すべてがうまくいくでしょう、しかし、モデルはエイサー・アレグロと呼ばれます-イタリア語から翻訳されて、それは「楽しい」を意味します。 私の意見では、Acer Impresaと呼ぶに値するでしょう。結局のところ、スマートフォンはビジネスのようです。
