FirefoxをDropboxと同期します。

こんにちはハブラ



次に、職場や自宅などでFirefoxを同期する方法を説明します。ブックマーク、訪問履歴、フォームデータ、パスワード、さらには拡張機能を同期します。



Firefoxを同期するための拡張機能であるMozilla Weaveについて書いたことがあります。 残念ながら、彼の仕事は理想からはほど遠いものであり、バージョン3の登場により、個人的にはまったく機能しなくなりました。



アイデアの要点を簡単に説明します。XPの場合は「c：\ Documents and Settings \％username％\ Application Data \ Mozilla \ Firefox \」、またはC：\ Users \％username％\ Appdata \ Local \ Mozilla \ FirefoxにあるProfilesフォルダーを移動しますVistaの場合、またはDropboxフォルダー内の* / .mozilla / firefox / * nixの場合、Windowsの代わりに、 ジャンクション 、またはディレクトリへの「ハードリンク」を作成します。* nixでは、古き良きln -sを使用します

VistaとwServer 2008でジャンクションを作成するには、mklinkがあります。WindowsXP / 2000の場合、 ダウンロードする必要があります。

したがって、最も重要なことはリンクを作成することです-



勝つ

C:\>mklink /j "c:\Documents and Settings\%username%\Application Data\Mozilla\Firefox\Profiles" "C:\Documents and Settings\%username%\ \My Dropbox\firefox\Profiles"







* nix

netstar$ ln -s ~/Dropbox/firefox/profile ~/.mozilla/firefox/ermyfan.default









Firefoxと出来事を使用するすべての場所で同じことを行います-ブックマーク、履歴、拡張機能（Weaveはできませんでした）などを完全に同期しています:)

追伸 同じことを行う方法はいくつかあります。たとえば、Dropboxフォルダーに直接新しいプロファイルを作成するか、単にDropbox設定でプロファイルフォルダーを指定しますが、これは最も単純で最も「再生できない」ものです。
