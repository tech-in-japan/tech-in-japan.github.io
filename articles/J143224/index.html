Linuxディストリビューションの人気

最初のニュース：Ubuntu 12.04はSelectelクラウドにインストールするためのテンプレートとして利用可能です。 Ubuntu 12.04は新しいLTS（つまり、サーバーバージョンのサポート期間が5年に延長されたバージョン）です。



このフレーズからプレスリリースを作成するのではなく、さまざまなディストリビューションの人気の統計を紹介した方がよいでしょう。



arch、gentoo、およびslackwareが存在しない場合、データは完全に代表的なものではないため、無人インストールをサポートするディストリビューションの人気に関するデータと見なすことができます。



分布



したがって、まず、OSの現在の人気に関するデータ：



  1. Debian GNU / Linux *-46.4％
  2. Ubuntu-32.6％
  3. CentOS-16.8％
  4. openSUSE-3.8％
  5. ClearOS-0.4％
合計Linux：100％。



* 一部のユーザーはシステム上で何かを作成する能力を悪用し、貧弱なdebianをあらゆる種類のgentoo / archに変えることを知っています。統計によると、debianは引き続き考慮されます。



コメント：Debianでの人気の大きなマージンは喜ばしいことです。しかし、Ubuntuの過剰な存在は少し気になります。 Ubuntuはそれがすべてサーバーであると言っているという事実にもかかわらず、ディストリビューションの存続期間内のソフトウェアバージョンの変更により、潜在的なリグレッションの対象となる実際のサーバーと見なすことは不可能です。



一方、Centosはその価値のある場所を占めています。古代として（検閲によって切り取られた）マンモス、永久凍土として安定しています。



ビットネス



現在のビットレートの比率は次のとおりです。29％は64ビットシステム、71％は32ビットシステムです。



そして、これは非常に好奇心が強い-ほとんどのユーザーは32ビットプラットフォームを選択します。 64ビットコードはより多くのメモリを（実質的に同じ機能で）使用するため、消費の支払いに関しては、これは完全に正当な選択です。 違いは、64ビットコードが高速で、オーバーヘッドなしでデータ用により多くのRAMを使用できる高負荷で感じられ始めます。
