Googleがサンクトペテルブルクに開発センターを開設

Googleは有能な人材を見つける仕事を創出します。 本日、同社はサンクトペテルブルクに研究開発センターを開設すると発表しました。 この点で、彼女はwww.google.ru/jobsで公開されている現在の空席のリストに応募者の注意を引き付けます。



Yandexのプログラマーも最近、サンクトペテルブルクのプログラマーの雇用を発表したことを思い出してください。



新しいGoogleセンターは、ロシアの投資プログラムの一部です。 同時に、サンクトペテルブルクの従業員は、ロシア市場と世界の両方の開発に従事します。 近い将来、ボーランドで以前働いていたエンジニアのグループが新しいオフィスで職務を引き受けることが知られています。



「サンクトペテルブルクセンターの開設後、ロシアは米国に続く2番目の国になり、同社の2つのエンジニアリングセンターが配置されます」とGoogleが言いました。 たとえば、前回のGoogle Code Jamコンテストのファイナリストの約30％ がロシア人だったため、驚くことではありません。 ロシアソフトウェア開発者協会によると、ロシアは同国に住む科学者とエンジニアの数で世界第3位です。



「Googleはロシアに投資して、ユーザーと広告主の環境を整えています」と、国際エンジニアリング事業担当副社長のKannan Pashupathy氏は述べています。 -新しい開発研究センターは国内で2番目です。 その発見は、ロシアに対するロシアのコミットメントとロシアのエンジニアの高い資格を実証しています。」



「Googleのサンクトペテルブルクへの投資は、ロシアは当社の製品の有望な市場であるだけでなく、世界にユニークなエンジニアリングの才能を提供する国であるという当社の信念を反映しています」 -著名なGoogle Code Jam国際プログラマーコンテストのファイナリストの30％以上がロシア人です。 開発はGoogleで行うすべての基盤であり、サンクトペテルブルクの新しいセンターと協力することを楽しみにしています。
