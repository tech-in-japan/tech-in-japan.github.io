jQueryビジュアルメッセージスタックプラグイン

タスクは、システムからのユーザーアクションに対する反応を動的に視覚化することでした。

言い換えると、オブジェクトの状態が変更されました-変更がデータベース/システムに正常に保存されたことに関するメッセージを表示します。

プラグインの作成には時間がかかりませんでした...



プラグインの機能を示す使用例：

$( '#save' ).click( function () {

var msgObj = $.notify.add( " " , 3);

// ( 3 ) msgObj

$.get( 'test.txt' , function (result) {

if (result.ok) { //

$.notify.remove(msgObj, 0.5);

// 0,5 ,

$.notify.add( " " , 1);

//

} else { //

$.notify.remove(msgObj, 0.5);

// 0,5 ,

$.notify.add( " " , 'error' , 3);

// "" (css error) 3

}

}, "json" );

});
