アメリカのiMobileの伝統医学

誰もがすでに知っているように、iPhoneは非常に繊細なガジェットです。落下するのは好きではありませんし、自動充電なども好みません。

しかし、判明したように、「問題」のいくつかは非常に職人的な方法で扱うことができます。 個人的な経験からの2つのヒントについて説明します...



問題1. iPhoneの充電が非常に速く「漏れる」、または電話の電源が切れない（代わりに、突然リブートする）

iPhone Original（第一世代）でこの問題に直面しました。



解決策：

-歯ブラシと少量のアルコールを取ります。

-電話の電源を切ります（電源ボタンを押したままにして本体の電源をオフにできる場合は、これを行います。電源ボタンから電源が切れない場合は、電源+ホームを10秒間押しながら電源を「残酷」にオフにします）。

-歯ブラシをアルコールに浸し、30ピンコネクタを慎重に清掃します。

-アルコールが完全に乾くまで15〜20分待ちます。

-電話をオンにし、高い確率で問題を解決します。

PS一般に、この方法で30ピンのiPhoneコネクタをクリーニングすることは、問題がない場合でも便利です...



問題2. iPhoneが自動的にオフになる（充電レベルに関係なく）

iPhone 3GSでこの問題に直面しました（ごく最近）。 ファームウェア4.0.2、イタリアの本体（Pay＆Go）。

何が起こるか：GSMモジュールがアクティブになると（SMSの送受信中に）電話が突然オフになります...

私は解決策を長く一生懸命探しました。 Appleフォーラムは役に立たなかった。 その結果、試行錯誤により、問題は解決しました！



解決策：

-SIMカードを静かに取り外します。

-アルコールでsimトレイ自体を完全にクリーニングします。

-視覚的にSIMカードが完全にきれいに見える場合でも、SIMカードの接触部分もアルコールで完全に洗浄されます（すぐに言います：MTSを持っています。ネットワーク上のMTS SIMカードの品質に関する多くの興味深いことを読むことができます）。

-SIMカードをその場所に戻すと、問題は解決します。



ソリューションが見つかった方法：

-電話機をコンピュータに同期するときに、エラーログ（PC / MAC-重要ではない）を含む、電話機からのすべてのログが転送されます。

-彼の「7」で私はアドレスCに行きました：\ Users \ gogalev \ AppData \ Roaming \ Apple Computer \ Logs \ CrashReporter \ MobileDevice \ iPhone Gogalev 3GS \ Baseband（日付によって最後のファイルを開きましたこれは次のようになります：log-bb-2010-08-29-stats.log）;

-このログを見て、GSMモジュール操作のプロセスで問題が正確に発生することを確認しました。

-3Gネットワ​​ークで最初に罪を犯し始めました。 切断された; 問題は解決されていません。

-SIMカードを引き出しました。 私はそれとSIMトレイを掃除しました。 問題はなくなりました（私はそれなしで2週間生きてきました）。



突然問題が再発する場合は、SIMカードを交換します...



そのようなこと。
