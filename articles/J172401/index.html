Intel System Studio-組み込みおよびモバイルシステム向けのソフトウェア開発者のセット



2月下旬、IntelはIntel System Studioのリリースを正式に発表しました。これは、組み込みシステムおよび特殊デバイスのソフトウェア開発者向けの新しいツールセットです。

Intel System Studioは、開発者がソフトウェアの信頼性、エネルギー効率、生産性を確保するという最も差し迫った問題を解決できるように設計されています。

実際、Intel System Studioは、「デスクトップ」 Intel Parallel Studioアプリケーションの真面目な開発者には馴染みがありますが、Linuxを搭載した組み込みシステムに特化しています。



コンパイラの最適化

  • Eclipse 3.7、Eclipse CDT 8.0（Indigo）でのIntel C ++コンパイラーの統合。
  • GNU、Yocto Project 1.x、Wind River Linux 5およびCE Linux PR28のクロスコンパイルサポート。
  • パラレルC / C ++拡張機能のサポート。
  • Intel Cilk Plusをサポート。
  • Intel Core i7用Intel AVXおよびIntel Atom用Intel SSSE3のアプリケーションを最適化するための自動ベクトル化;
  • GCCコンパイラとの互換性。
  • -m32および-m64コンパイラオプションのサポート。


Intel Math Kernel Library

  • Intel Coreファミリーのプロセッサーで信号とデータを高性能に処理するための、線形代数、ベクトル数学、統計の最適化された機能。


Intel統合パフォーマンスプリミティブライブラリ

  • Intel AtomおよびIntel Coreプロセッサに基づいた組み込みシステム向けの、データ、信号、オーディオ、ビデオ、画像を処理するための広範な機能プリミティブと例。


専用システム向けのIntel VTune Amplifier 2013

  • パフォーマンスのボトルネック、ハードウェアリソースの最適でない使用、およびシステムの電力消費の識別。
  • 2.6.9から3.3.xのLinuxカーネルサポート。
  • 最新世代のIntel AtomプロセッサのイベントベースのCPUサンプリング。
  • Yocto Project 1.x、およびCE Linux * Intel AtomプロセッサーCE4xxxおよびCE5xxxのPR28のサポート。
  • 専用のイベントベースのサンプリングシステム用のリモートデータコレクター
  • Intel Coreプロセッサーの周波数と電力の分析。


特殊システム向けのIntel Inspector 2013

  • Yocto Project 1.xおよびWind River Linux 5上のデバイス、およびIntel AtomプロセッサCE42xxおよびCE53xx上のCE Linux PR28のデバイスにおけるマルチスレッドの正確性のメモリ検証と分析。
  • スレッドロック、データの競合、メモリリークの検出。
  • デバイス自体または完全なIntel Inspector GUIでコマンドラインを使用します。


GDB-GNUプロジェクトデバッガー

  • POSIX Pthreadsを使用するアプリケーションのデータ競合を定義します。
  • リモートデバッグと仮想化のサポート。
  • Eclipse CDTとのリモートデバッグ統合。


Intel JTAGデバッガー

  • 命令トレースウィンドウ。
  • 拡張ファームウェアインターフェイス（EFI）をサポート。
  • Intel ITP-XDP3またはMacraigor usb2Demonを使用したブートローダー、OSカーネル、および動的にロードされたデバイスドライバーのデバッグ。
  • 詳細ページおよび記述子変換テーブル
  • ビット編集フィールド
  • 組み込みのファームウェアユーティリティ。


ソフトウェア可視イベントネクサス（SVEN）

  • プラットフォーム全体でイベントトラッキング用のコードインストルメンテーションインフラストラクチャを低オーバーヘッドで実現。
  • データを伴うイベントのトレースのビューア。
  • ソフトウェア開発キット（SDK）;
  • Csvenコンソール
