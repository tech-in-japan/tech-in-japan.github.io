GlacierWorksとMicrosoftがインタラクティブなエベレストサイトをセットアップ





MicrosoftとGlacierWorksは最近、エベレスト山に特化したインタラクティブなサイトの立ち上げを発表しました。 このサイトは「 エベレスト：氷の川 」と呼ばれ、このリソースを使用すると、宇宙だけでなく時間内に山に沿って仮想の旅をすることができます。 そのため、開発者は、何十年もの間、気候条件の変化/ピークの出現の履歴を追加しました。 さらに、60年にわたってエベレストの登山ルートの開発の歴史を見る機会があります。



ところで、サイト自体は完全にHTML5で作成されており、エベレストと周囲の山々のギガピクセルのパノラマが含まれています。 登山に熱心ではない場合でも、サイトで多くの時間を過ごすことができます。 このサイトの開発者が直面しているタスクの1つは、自宅から登山者がトップへの詳細なルートを作成するのを支援することでした。 そして、私はリソースがアスリート-クライマーを本当に助けることができると言わなければなりません。



数十年にわたって起こった気候変動を理解するために、サイトの作成者は過去数年と現代のパノラマ画像の比較を提供しました。







GlacierWorksの代表者は、上記の氷河の状態に関する詳細な情報を取得するために、過去7年間に約14回の遠征を実施しました。 探検の写真は、Canon 5D Mark IIとCanonおよびZeissのレンズを使用して取得しました。 アーカイブ内のグラフィック情報の総量は約7テラバイトに達し、ほとんどの写真の解像度は5616 x 3744です。これらの写真はすべて、120,000 x 31840ピクセルの解像度のパノラマにまとめられています。 このすべての作業がGlacierWorksによって行われたことは注目に値します。



Microsoft Corporationに関しては、専門家がこのサイトを開発し、これらすべてのメガ写真をインタラクティブなパノラマに変えました。 この場合、Rich Interactive Narratives（RIN）プロジェクトの経験が使用されました。 このサイトはすべての最新のブラウザーで正常に機能しますが、もちろん、MicrosoftはInternet Explorer 10を使用するのが最善であると発表しました。
