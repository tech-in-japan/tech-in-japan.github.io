Flash Playerの3D API



アドビシステムズは、Flashテクノロジーに対する定期的な批判にもかかわらず、それに基づいた製品の近代化と開発を続けています。 同社は、世界的な3Dブームから離れることはありません。 そして今秋、3次元グラフィックスを完全にサポートするブラウザーAdobe Flash Player用のプラグインの新しいバージョンを導入する予定です。



近年、Flash開発者の主なタスクは、限られたコンピューティングリソースを備えたスマートフォンやその他のデバイスで動作するように、現在の形式のテクノロジーを受け入れられるようにすることでした。 一部の3次元機能もサポートしていますが、その方法は非常に限られています。



今後の「3次元」Flashに関する情報は、10月下旬にロサンゼルスで開催されるAdobe MAX開発者向け会議のスケジュールに掲載されました。 このショーは、Flash開発者Flash Player 3D Futureのセッションの一部として開催される予定です。 ビッグネームは明らかに、AdobeがFlashの未来を3Dテクノロジーなしでは考えられないものと見なしていることを意味しています。



残念ながら、彼らは現時点では詳細を公開せず、最後の瞬間まで関心を高めています。 それにもかかわらず、開発者は、3D機能が非常に豊富になるため、今後のセッションのための広範なプログラムを約束します。 アドビはすでにFlashを使用して3Dプレゼンテーションを開発するためのツールを提供していますが、このツールキットとプラグイン自体の機能が大幅に拡張される予定です。 会議では、Flash Playerの新しいバージョンを開発するための時間枠について説明し、新しいFlash 3D APIも紹介します。 開発者は、会議で新しいテクノロジーの可能性に「深く浸ること」を約束されています。



「3Dゲーム、拡張現実システム、または新しいインタラクティブサイトの開発を計画している場合、今後の会議に参加する必要があります」とアドビの広報担当者は述べています。



明らかに、三次元フラッシュの主な消費者-ブラウザーゲーム開発者。 また、たとえば、グラフィックアクセラレータのリソースを使用するなど、開発者が十分なパフォーマンスを提供できる場合、オンラインゲームの業界は突破口を待っています。



おそらく、アドビシステムズ側のこの動きは、同じ分野の競合他社の活動によるものであり、新たな傾向が見られます。 多くの主要な開発者が、3Dをサポートする新しい開発をある程度発表しています。 たとえば、最近のComputexトレードショーで、NvidiaはMicrosoft Silverlightベースのプレーヤーを使用して、インターネットを介した3Dストリーミングビデオのデモを行いました。
