11月27日にモスクワデータサイエンスにご招待します





今週の金曜日、11月27日、Mail.Ru Groupのモスクワ事務所の映画館で、モスクワデータサイエンスの記念日の会議が開催されます-コミュニティの会議は2歳になります。 伝統的に、ビッグデータ分析、推奨システムの構築、およびSNAの問題を解決するための機械学習アルゴリズムの使用に関連する問題が議論されます。



会議のプログラムでは：



Anton Lebedevich、データサイエンティスト、ソフトウェアエンジニア、R / Scala / Java / Erlang：「Rを使用した時系列の視覚分析」



スピーカーは、サーバーパフォーマンスの監視から取得した多次元時系列の視覚分析にRを使用する例を共有します。 彼は時系列を操作するための主要なライブラリについて簡単に話します。 実際のデータでは、レポートは、特異値分解（SVD）、多次元スケーリング（MDS）、およびt分布確率的近傍埋め込み（t-SNE）を使用した結果を示します。 最後に、問題の分析方法を迅速にプロトタイプ化するために、RとShinyを使用してWebアプリケーションを作成する方法を説明します。



オーディエンスセグメンテーショングループMail.Ru Groupの責任者、Arthur Kadurin：「キリンと無意識」



オンラインゲームでコミュニティを強調するタスクの例を使用して、スピーカーは、このツールの長所と短所について、Apache Giraphの開発における彼自身の経験を共有します。



Alexander Serbul、1C-Bitrix LLC：「1C-Bitrixコラボレーティブフィルタリングサービス、内部から20,000店舗」



このレポートでは、システムアーキテクチャと採用された技術的ソリューションを詳細に検討します。 スピーカーは、Apache Mahout Tasteライブラリを完成させ、Apache Tomcatを介して推奨事項を発行する際に使用されるアルゴリズム、設定、および微妙な点に特に注意を払います。 結論として、1日のうちにモデルへのロード変更の実装に数回焦点を当てます。 アレクサンダーは、個人的な推奨事項の品質を測定する方法を教えます：評価者のアルゴリズムとアーキテクチャ（Java、マルチスレッドなど）。



18:30の参加者の集まり、レポートの開始-18:50。 mitapの登録は開いています。 オンラインブロードキャストは、サイトIT.Mail.Ruで開催されます。
