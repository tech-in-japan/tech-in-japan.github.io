Vivaldi 1.4 for Linuxアップデート-フラッシュプラグインのハント





みなさんこんにちは！ 今日、これは通常のアップデートではありません。テストビルドではなく、Linux用のVivaldi 1.4のアップデートされた安定バージョンです。 この更新の理由（現在-バージョン1.4.589.41）は、Adobe Pepper Flashプラグイン（PPAPI）を使用したVivaldiブラウザーの小さな変更です。 Vivaldiがシステムにインストールされたプラグインを検索するディレクトリの数を増やしました。 したがって、Flashコンテンツがより多くのユーザーに役立つことを願っています。



そして-はい、このアップデートはLinuxユーザー専用です。 リポジトリから更新が届くまで待つか、サイトからインストールパッケージを直接ダウンロードできます 。



どのように機能しますか？ インストール中にVivaldiブラウザーは、Ubuntu Linux用にCanonicalパートナーが提供する公式パッケージからプラグインの場所を確認します。 さらに、オペレーティングシステムにChromeがインストールされている場合、Vivaldiはこのブラウザに付属のプラグインを使用できます。 プラグインの両方のバージョンが利用可能な場合、Vivaldiは最新のものを識別して使用します。



Ubuntu Linux



Ubuntuを使用する場合、ターミナルで次のコマンドを使用してCanonicalパートナーリポジトリを接続し、公式のAdobeフラッシュプラグインをインストールできます。 



sudo add-apt-repository "deb http://archive.canonical.com/ubuntu `sed -n '/UBUNTU_CODENAME/s/.*=//p' /etc/os-release` partner" sudo apt update sudo apt install adobe-flashplugin


Vivaldiを再起動した後（更新時に機能していた場合）、ブラウザでフラッシュプラグインの作業コピーを取得する必要があります。



その他のLinuxディストリビューション



ディストリビューションのドキュメントに従って、Adobeの公式Pepper / PPAPI Flashを提供するパッケージをインストールします。 そのようなプラグインをインストールした後、まだVivaldiブラウザに表示されない場合は、この記事の下のコメントで、ディストリビューションの名前とバージョン、およびインストール後のFlashプラグインの場所（知っている場合）を教えてください。



ディストリビューションにAdobeのPepper Flashパッケージがない場合は、 このスクリプトを使用して試すこともできます。
