IntelliJ IDEAバージョン0.4のErlangプラグイン

IntelliJ IDEAバージョン0.4のErlangプラグインが本日リリースされました-これはIDEAを完全にErlangの完全に機能するIDEに無料で変えます。



イノベーション

  • Rebarを介したEunitテストの起動を含むRebarとの統合
  • Erlangタイプのサポート：ナビゲーション、オートコンプリート、名前変更など。
  • OTP機能およびモジュールの外部（ Shift + F1 ）およびローカル（ Ctrl + Q ）ドキュメント
  • 関数パラメーターのツールチップ（ Ctrl + P ）
  • シンボルに移動（ Ctrl + Alt + Shift + N ）
  • BIFの自動補完の改善（ リスト：メンバー/ 2など、OTPモジュールのBIFを含む）
  • Darculaダークスキームのサポートと構文ハイライトの強化
  • さらに、 変更点の全リストをご覧ください。


バージョン0.1 （2012年7月以降）のリリース以降、さらに多くの変更が加えられています。 誰かが最初のビルドを試してみて満足しなかった場合、今が再試行する時です。
