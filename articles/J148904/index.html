Windows 8の開発が完了し、システムはRTMステージに移行しました

Brandon LaBlanc は 、今日のWindows 8開発チームのブログで、新しいWindowsの開発が完了したと述べました。 本日より、主要なOEMパートナーは、デバイスを作成するためのシステムの最終ビルドを受け取り始めます。





さまざまなMicrosoftプロジェクトのサブスクライバー向けのアクセスカレンダーも公開されています。



  • 8月15日、すべてのMSDNおよびTechNetサブスクライバーは、「ゴールデンコード」のアセンブリにアクセスできるようになります
  • 8月16日、すべてのマイクロソフトソフトウェアアシュアランスのお客様は、 ボリュームライセンスサービスセンターを介してWindows 8 Enterpriseにアクセスでき、すべてのマイクロソフトパートナーネットワークメンバーは、Windows 8のすべてのバージョンにアクセスできます。
  • 8月20日、 Microsoft Action PackプロバイダーはWindows 8にアクセスできます


他のMicrosoftプログラムのメンバーである場合は、Windows 8イメージの可用性に関する詳細情報を期待してください。一般ユーザーの場合、 Release Previewは引き続き利用できます。

8月15日、すべての開発者は、必要なすべての開発ツールが利用できるDeveloper Centerに参加する予定です。

システムの最終バージョンは2012年10月26日に正式に発表され、この日、ユーザーはWindows Upgrade Offerにアクセスできるようになります。
