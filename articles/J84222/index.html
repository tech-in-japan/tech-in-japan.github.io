ブラウザのユーザーエージェントの履歴

最初は、NCSA Mosaicがあり、MosaicはNCSA_Mosaic / 2.0（Windows 3.1）と呼ばれ、Mosaicはテキスト付きの写真を表示し、幸福が訪れました。







そして、「Mosaic Killer」の略称である「Mozilla」と呼ばれる新しいWebブラウザーが登場しましたが、Mosaicは気に入らなかったため、名前をNetscapeに変更し、Netscape自体をMozilla / 1.0（ Win3.1）、そしてそれ以上の幸せはありませんでした。 また、Netscapeはフレームをサポートし、フレームは人々の間で人気を博しましたが、Mosaicはフレームをサポートしなかったため、ユーザーエージェントを嗅ぎ、Mozillaウェブマスターはフレームを送信しましたが、他のブラウザーにはフレームを送信しませんでした。



そして、Netscapeは、Microsoftを笑って、Windowsを「ひどくデバッグされたデバイスドライバ」と呼び、Microsoftは怒っていたと言いました。 また、Microsoftは独自のWebブラウザーを作成しました。これはInternet Explorerと呼ばれ、「Netscapeのキラー」になることを望んでいました。 また、Internet Explorerはフレームをサポートしていましたが、Mozillaではないため、フレームを受信しませんでした。 マイクロソフトはイライラして成長し、ウェブマスターがIEを学習してフレームを送信し始めるのを待ちたくなかったため、Internet Explorerは「Mozilla互換」であると述べ、Netscapeの擬人化となり、自身をMozilla / 1.22（互換、MSIE 2.0; Windows 95）、およびInternet Explorerがフレームを取得し、Microsoftの全員が満足していましたが、ウェブマスターは混乱していました。



そして、MicrosoftはIEをWindowsで販売し、Netscapeよりも優れたものにしました。そして、最初のブラウザ戦争は地球上で激怒しました。 そして、Netscapeは殺され、マイクロソフトの陣営には多くの喜びがありました。 しかし、NetscapeはMozillaとMozilla Geckoが作成されて復活し、彼は自分自身をMozilla / 5.0（互換性、Windows NT 5.0、EN-US、RV：1.1）Gecko / 20020826と呼び、Geckoはレンダリングエンジンであり、Geckoは良かったです。 そして、MozillaはFirefoxになり、Mozilla / 5.0（互換性; Windows NT 5.1; SV-SE; RV：1.7.5）と名付けられました。Gecko/ 20041108 Firefox / 1.0とFirefoxは非常に良かったです。 そして、Geckoが繁殖し始め、そのコードを使用する他のブラウザーが誕生し、Mozilla / 5.0（互換性; PPC Mac OS X Mach-O; EN-US; RV：1.7.2）Gecko / 20040825 Camino / 0.8と呼ばれました。 1およびMozilla / 5.0（互換性; Windows NT 5.1; DE; RV：1.8.1.8）Gecko / 20071008 SeaMonkey / 1.0、それぞれがMozillaのふりをして、それらはすべてGeckoに基づいていました。



Geckoは優れていましたが、IEはそうではなく、スヌーピングは復活し、Geckoは良いコードを取得し、他のブラウザーはそうではありませんでした。 Linuxフォロワーは、KHTMLであるKonquerorを作成し、Geckoと同じくらい良いと思っていたが、Geckoではないため、良いページを取得できなかったため、Konquererが「 Geckoのように」良いページを取得し、Mozilla / 5.0（互換性、Konqueror / 3.2、FreeBSD）（KHTML、Geckoのような）と名乗り、混乱が激しくなりました。



その後、Operaが来て「もちろん、ユーザーにどのブラウザを偽装するかを決定させなければなりません」とOperaはメニュー項目を作成し、OperaはMozilla / 4.0（互換性; MSIE 6.0; Windows NT 5.1; en）Opera 9.51、またはMozilla / 5.0（Windows NT 6.0; U; EN; RV：1.8.1）Gecko / 20061208 Firefox / 2.0.0 Opera 9.51またはOpera / 9.51（Windows NT 5.1; EN）ユーザーが選択したオプションに応じて。
