IBMは独自のIoT部門に投資しています

IBM は 3月31日、今後4年間で新しいIoT（ 「モノのインターネット」 ）ビジネスユニットに30億ドルを投資すると発表しました。 同社はまた、この方向で働く顧客やパートナー向けのオープンクラウドプラットフォームを構築します。



現時点では、企業はすでにIoTの実用化に関連する関連分野（ Smarter PlanetおよびSmarter Cities）で作業を行っています。



IBMは、新しい開発ツールとクラウド構造により、前例のない数のソースからの異種データを統合することを目標に、幅広いユーザーとパートナーにリーチすることを計画しています。 IoT用に最適化された新世代のデバイスの開発と生産をサポートするために、すべてのリソースが公開されます。



Weather Weather Report Companyとのパートナーシップも発表されています。 パートナーシップには、サービスをIBMクラウド構造に移動することが含まれます。



プログラムの一環として、IBMは以下を提供します。



  • IBM IoT Cloud Open Platform。 データおよび分析サービスへのアクセスを提供する企業向けプラットフォーム。
  • IBM Bluemix IoTゾーン。 開発者がIoTデータを統合するためのサービス。
  • IBM IoTエコシステム。 オープンIBMプラットフォームでのサービスとソリューションの安全でシームレスな統合を提供するパートナープラットフォーム。


ユニットはPat Tooleによって管理されます。 現時点では、同社はすでにIoTアプリケーションの分野で多くのメーカーや消費者と協力しており、合意に達した時点で停止するつもりはありません。



すでに2018年にIBMの専門家によると、クラウド、モバイル、分析技術、およびソーシャルサービスと情報セキュリティの分野のソリューションは、企業に少なくとも400億ドルの収益をもたらします。
