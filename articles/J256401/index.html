Visual Studio Community 2013とVisual Studio 2013 Expressを比較してください。 ライセンス契約の機能

最近、Microsoftは最も人気のある開発環境の新しいエディションであるVisual Studio Community 2013を導入しました。ほとんどの開発者はライセンス契約の機能を理解できず、「商用開発で​​Visual Studio Community 2013を使用できますか？」 この問題を一緒に見てみましょう。



実際、MicrosoftはVisual Studio Community 2013をどのように位置付けていますか？ 無料のフル機能の統合開発環境です。 これはどういう意味ですか？ -これは、個々の開発者がVisual Studio Communityを使用して無料または有料のアプリケーションを作成できることを意味します。 残念ながら、コミュニティエディターはMSDNサブスクリプションなしで提供されます。



組織でVisual Studioコミュニティを使用する方法は次のとおりです 。

-Visual Studio Communityは、次の場合に組織内のユーザーを無制限に使用できます。教室、オープンソースプロジェクトへの調査または参加。

-他のすべてのユースケースの場合：非法人組織では、Visual Studio Communityは最大5人のユーザーを使用できます。 企業の組織（250台以上のPCを使用するか、年間収入が100万米ドルを超える）では、上記のケース（オープンソース、研究、教室）を除き、使用は禁止されています。



Visual Studio Community 2013は、Visual Studioの他のリリースとどのように違いますか？

-Visual Studio Community 2013には、個々の開発者、学生、オープンソースプロジェクトの参加者、および小グループ向けに設計および最適化されたVisual Studio Professional 2013の優れた機能がすべて含まれています。



次に、Express Edition Visual Studio 2013とは何かを見てみましょう。

Visual Studio Express 2013は、無料の開発者ツールの最新バージョンであり、最終製品が作成されるプラットフォームに基づいて決定されます。 各ツールは、Visual C ++、Visual C＃、およびVisual Basicをサポートしています。



Visual Studio Expressの商用利用 ：

Visual Studio Express製品は無料で利用でき、各製品のライセンス条項に従って商用および産業目的で使用できます。 たとえば、Express for Windowsを使用してアプリケーションを作成し、Windowsストアに販売するために送信できます。



結論：Expressが好きなら、Visual Studio Community 2013が大好きです。Expressのすべての機能、複数のWindows、Android、iOSデバイス用のアプリケーション開発ツール、1つの統合開発環境で数千の拡張機能にアクセスできるVisual Studio Community 2013は完璧なツールです個人の開発者、学生、オープンソースの著者、小グループ向け。
