民俗の格言。
Web 2.0はファントムですか？ もちろん違います。 この現象の存在の客観性はもはや疑いの余地はありません。 彼はこの社会的プラットフォームの砕石すべてを固定する礎石はどこにあるのでしょうか？ Web 2.0の定義のなかで「集合的な理由」、「ユーザーの愛」、「民主主義と自由」、そして「マーケティング」と「ビジネスモデル」という堅固な用語がフラッシュすることについての大げさなフレーズ。 そして、真実でさえ、社会の発展は経済関係の発展に直接関係していると主張していた。 ネットワークに関して、これは次のように解釈することができます-サイトの急峻さの成長は、生地を壊す方法に直接関連しています。 この観点から見ると、Web世代の典型的なビジネスモデルは次のようになります： Web 1.0：コンテンツを作成し、集中的に宣伝し、最初のトラフィックを取得し、ルートに変換し、テネリフェでリラックスしながらさらにコンテンツを生成する農奴を集めます。

Web 2.0：私たちは皆に、おもしろい、無料の（あなただけのために！）に参加することを勧めます。 オプションとして：コンテンツではなく、直接私のサイトにトラフィックするか、私のサイトではなく、私の好みで（必要に応じて下線を引く）。 その結果、世界の半分が私のために働いており、私は母国のボブルリスクの誰にも知られていない家に座っており、時々生地の別の部分をSberbankから借りています。 しかし、これらのネットワークには重大な欠点があります-「パートナー」は何かを支払う必要があります。 この点で、Web 2.0モデルは次の理由によりはるかに高度に見えます。

  • 支払いを必要としないユーザーの慈善的で創造的な動機を活用し、一部は景品への愛情も活用します。
  • Web 1.0に比べてコンテンツ/トラフィックの潜在的なクリエーターのはるかに大きな輪が関係しているため、それらの間には競争があり、より質の高いコンテンツとプロジェクトに参加するギャンブルの動機の出現につながります。
Web 2.0モデルの主要なビジネス図には以下が含まれます。

  1. ユーザーの創造的な可能性を活用するさまざまな種類のコミュニティ：LiveJournal、ブログなど トラフィックの略奪への変換は、まだWeb 1.0であるエキセントリックな人々を犠牲にして行われます。つまり、広告と、より完全かつエレガントに注ぎ出すことを可能にするプレミアムアカウントによって行われます。
  2. 他のサイト所有者の健全な怠inessを利用するさまざまなマッシュアップ（Googleが既にこれを行っているのに、なぜこのサイトの新しいチップを発明するのか）。 彼らのサイトからのトラフィックは、主にアプレットの広告のために戦利品に変換されます。
  3. 景品の愛を利用する：さまざまなファイル共有サービス、オークション（安い...）、広告関連会社（何でも、滴り落ちる）。 広告を介して、またはトランザクションの割合により、トラフィックを略奪品に変換します。


可能性のある異議を予想して、いくつかのWeb 2.0サイトでトラフィックを略奪に変換できないことは、ルールの例外である可能性が高く、これらのサイトの作者の良心に完全に依存していることにすぐに気付きます。

おわりに

したがって、Web 2.0プロジェクトの主な違いは、ビジネスモデル、つまり戦利品に変換されるトラフィックを生成する方法です。 この意味で、主な傾向は、インターネットプロジェクトの技術的プロセスの大部分をウェブサイトから大衆に移すことです。 この傾向を論理的に終わらせて、Web 3.0の主な機能を特定できます。
  • インターネットプロジェクトのライフサイクルのすべての部分を、関心のある大衆に完全に移転する。
  • トラフィックをルートに変換するキャッシュフローの分配における仲介業者の拒否。
Web 3.0の技術構造に関する興味深い結論はこの結論から得られますが、これは別の記事のトピックです。
