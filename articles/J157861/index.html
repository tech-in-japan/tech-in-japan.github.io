5分でPBX



少し前に、友人が私に、彼らの小さな会社に電話をかけるという共通のタスクに対処するのを手伝ってくれるように頼みました。 彼らはもはや顧客に携帯電話を提供することができず、電子メールだけでは不十分であるため、電話交換を確立するというアイデアを得ました。

5分で、ソフトチェアから立ち上がることなく、仮想PBXをインストールして構成しました。 これについては、無料のZadarma PBXをインストールする例を使用して説明します。

段落の冒頭に簡単な理論があります。 誰が知っている-読んでいない。



なぜPBXが必要で、どのように機能しますか？
電話をかけるほとんどのオフィスでは、ほとんどの場合、自動的に迎えられます。 次に、オペレーターは電話を取り、希望する従業員に通話を転送します。 従業員がこの時点でオフィスにいるということは事実ではありません。ノボシビルスクとプラハにいる可能性がありますが、あなたは彼と迅速かつ静かにつながります。 さらに、女の子のオペレーター、そして実際にオフィスを持つ必要はありません。

作品も整理しますか？ 右側には、電話交換の動作を説明する図がありますが、複雑なものはありません。

怖いシステム規模？ 恐れてはいけないと思います！ 自分以外のダイアグラム上のすべてのものを仮想にすることができますが、機能する可能性は十分にあります。 5分でPBXを接続して構成しました。

PBXはわずか4つの手順で起動できます。

  1. アカウントの登録と資金調達
  2. 番号を選択して購入する
  3. PBXをオンにして、セットアップウィザードを実行します
  4. コンピューター/電話でクライアントプログラムをダウンロードして構成する
5番目のオプションのステップは、音声メニュー、自動受付、通話録音、FAXなど、あらゆる種類の「弓」のセットアップです。

PBXのインストールを開始しましょう。



1）アカウントへの登録と入金：5分
単純さと速度のために、サービスの比較を掘り下げることはせず、単にサイトに登録します 。 スターターの料金は「標準」で、1秒あたりの料金が無料で、いつでも別の料金に変更できます。

登録後、個人アカウントにアクセスします。 テストコールで接続を確認し、アカウントを補充します。 RBKサービスを通じてクレジットカードをアカウントに補充しました（最初のリチャージで10ドルで2桁の電話をかけ、初めて電話をかけることができました）。すべてが数秒で完了しました。



2）部屋の選択と購入-4分
最初は価格を尋ねる価値があります。そのため、番号の完全なリスト（「設定/直接番号」またはメインサイト）にアクセスします。 私の友人には、モスクワの伝統的な番号（コード499）とニューヨークの番号が必要です。 米国の番号は即座に登録されました（ハリケーンは私たちにとって障害ではありません）。モスクワではパスポートスキャンをアップロードし、番号を選択しました。 注文の準備が整いました。モスクワの番号は、書類を確認した後、1日以内に接続されます。



3）PBXのスイッチを入れてセットアップする-5分
右下のボタンを使用して、 アカウントのPBXをオンにします。

スイッチをオンにした後、「セットアップウィザード」に移動し、ポイントごとに説明します。

  • PBX内線番号の作成 。 内線番号はいくつ必要ですか？ 私たちの場合、3つで十分でした（後で50の数字をいつでも追加できます）。
  • PBXの外線を定義します。 必要な外部SIPは1つだけなので（1 SIP = 5行）、この点はスキップします。
  • 発信通話 。 便宜上のセクション。 都市内の短い番号を呼び出し、長距離番号の前に8をダイヤルすることに慣れている場合は、リストから都市を選択します。 都市がリストにない場合は、最も近い都市を選択し、都市コードを自分のものに変更します。 また、国際形式で番号をダイヤルする場合は、一般に、何も選択する必要はありません。 「次へ」をクリックしてください！
  • 音声メニュー/挨拶を作成します 。 この機能はオプションですが、PBXのすべての利点を使用することにしました。音声メニューを作成します。 メニューをインストールしたいのですが、あいさつ付きの音声ファイルがありません。 問題ではありません-[テキストから読む]を選択します。 例のために書きました。「イゴールとパートナーへようこそ。 Igorとの通信には1、パートナーとの通信には2、3ファックスを押します。 テキストを入力し、「読む」をクリックすると、テキストが読み上げられ、挨拶がオンになります！
  • 音声メニュー項目を作成します 。 歓迎のテキストでは、3つの選択肢がありますが、 つまり、メニューには3つのオプションが必要です（何もクリックしない場合のメニュー項目は、自動的に作成されます）。 たとえば、「1番目のボタン、2パートナーボタン、3ファックスボタン」という名前の3つのメニューを作成します。PS：Zadarmaの仮想電話交換では、メニューオプションは「スクリプト」と呼ばれます。
  • 着信コールを設定します 。 私たちが作成したメニュー項目と呼ばれる、彼らの仕事を説明するために残っています。

    内部番号は100、パートナーは101になります。次に、「ボタン1」シナリオでは、「ファーストコール」セクションに100を入力します。突然電話をとらない場合は、パートナーに電話して「セカンドコール」セクションに入力します。 。 また、「ボタン2」メニューを作成します。「最初の呼び出し」でパートナー-101を配置します。電話に出ない場合は100になりますが、彼の呼び出しを受信したくないので、遅延は30秒に増加します。 「押されていない」メニューは、「ボタン1」メニューとまったく同じ方法で構成されます。


おめでとうございます、PBXをセットアップしました！ [完了]をクリックして、電話をPBXに接続します。



4）コンピューターと電話でプログラムをダウンロードして構成する-3分
たとえば、Androidフォンとパートナー、つまりWindowsのラップトップへの呼び出しを受信します。 私たち自身のために、Zadarmaプログラムをインストールします（プログラムページで、番号を入力してリンクを取得します）。 なぜなら PBXに接続したら、標準設定ウィザードをキャンセルし、[アカウントの追加]をクリックして、内線番号のデータを入力します。 Android 4.xでは、プログラムを使用せずに組み込みのSIPクライアントを使用できます。

Windowsの場合、通常はPhonerLiteにアドバイスします。個人アカウントのプログラムセクションからダウンロードできます。インストール手順もそこから入手できます。

iOSを使用するには、media5電話などのさまざまなプログラムをお勧めします。 無料版の欠点は、G711コーデックのみを使用できることです。これは、3G経由のコールには不便です。 他にもプログラムがあり、SIPという単語を検索することで、アプリストアで簡単に見つけることができます。



弓（追加機能）-5分
モスクワとニューヨークの電話番号を接続しますか？ 音声による挨拶はすでにすばらしいことであり、ほとんどの場合それだけです。 しかし、Zadarmaの仮想電話交換では、はるかに多くの「美しく」便利な機能を利用できます。

  • CallerID（発信者ID）の設定 。 新しい「美しい」モスクワ番号を購入したら、当然、電話番号でそれを見たいと思うでしょう。 なぜなら 米国で最初の番号を購入し、そのCallerIDがデフォルトで設定されましたが、これは簡単に変更できます。 [SIP設定]セクションでは、発信時に送信されるCallerIDなどの外部SIP回線パラメーターが設定されます。 プロファイルで確認した場合、CallerIDと外部番号を使用できます。
  • 条件付き転送 。 私のお気に入りの機能。 よく旅行しますが、電話を逃したくありません。 3Gインターネットカバレッジが不完全なため、私はいつも使用するとは限りません。 内部番号の設定で、「転送」にチェックマークを付け、「応答なし、利用不可」を選択して、携帯電話を入力します。 Androidの設定では、インストールされたSIPクライアントはWiFi経由でのみ機能します。 さて、外に出るとすぐに携帯電話に電話がかかり、家に帰ったり仕事に出たりすると、SIPで電話がかかります。 主なものは、単一の不在着信ではありません。 要求の厳しいユーザーには、ボイスメールまたはスカイプへの転送が可能です。
  • 発信者番号で電話を受ける 。 パートナーの義理の母に電話をかけるときに「2」を押すように教えることは非常に困難ですが、毎回彼女と話をして転送するのは嫌です。 簡単な方法があります。着信コールの設定で、「パートナー」スクリプトを開き、「番号からコールするときに動作する」チェックボックスをオンにして、彼の義理の母親の番号を入力します。 これで、音声メニューなしで、すべての電話がパートナーに直接送信されるようになりました。
  • 自動ファックス 実際には、今ではファックスを必要とする人はほとんどいませんが、それを残してみませんか？ 美しい追加、および無料。 なぜなら FAX用のスクリプトを既に作成しています（ボタン3）。このスクリプトを開き、リストに内線番号「50」を追加するだけです。 すべてのファックスは登録メールに送信されます。
  • 営業時間 。 営業時間中にのみ、すべての電話に出たい。 残りの時間は、全員を留守番電話に送ります。 PBXの着信コールの設定で、営業時間を設定できます。 日、営業時間、および営業時間後に鳴るテキストを設定します（プロファイル設定でタイムゾーンを確認および設定できます）。 留守番電話の場合、「留守番電話」スクリプトを作成し、その中に無料の内線番号102を設定する必要があります。内線番号102の設定で、すべての通話のボイスメールへの転送を設定します。




  • 会話を記録します。 会話を録音する必要がある場合は、内部番号設定の「会話を録音する」チェックボックスをオンにします。 エントリーは指定されたメールボックスに送信されます。
  • 期待のメロディー 。 音楽が好きな人は、通話の転送中に待機中のメロディをダウンロードして設定できます。 これは、着信コールの設定で行うことができます。
  • 追加の外部回線 。 新しい仮想PBXは真空状態ではありません。古い番号があり、呼び出しも受信する必要があります。 この場合の対処方法 すべてが単純で、「追加の外部回線」として接続します。 古い番号に銅線が付いている場合、SIPゲートウェイにFXOポートを配置し、サイトでゲートウェイモードで回線をオンにし、ゲートウェイでサイトから設定を設定する必要があります。 古い番号がすでにSIPで機能していた場合、すべてが簡単です。「サーバー」モードで回線をオンにして番号設定を設定するだけで、PBX自体がサーバーに接続し、この番号への呼び出しを受信します。
ZadarmaクラウドPBXには他にも興味深い機能がありますが、自分でテストするのは難しくありません。
