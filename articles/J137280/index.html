ウェビナーエントリ「Web開発者向けのASP.NETおよびVisual Studio 11の未来」が利用可能





1月26日、Microsoft Webプラットフォームの革新とWeb開発者向けのVisual Studio 11開発環境に関する無料のWebセミナーが開催されました。 セッション後の興味深い質問を含め、ウェビナーのすべての参加者（あなたは170人以上でした）に感謝します。



Webセミナービデオがダウンロードできるようになったことをお知らせいたします。このビデオは、次のリンク「 Web開発者向けVisual Studio 2011のイノベーション 」 （リンク更新）から簡単にダウンロードできます。 ビデオは、オンライン表示とダウンロードの両方に使用できます（〜40 Mb、wmv）。



または、docs.comの次のリンクからプレゼンテーションファイル（pptx）をダウンロードできます。



次の主要なトピックは、ウェビナーで議論されました。

  • ASP.NET WebForms 4.5-リクエストへの回答
  • ASP.NET MVC 4およびモバイルWebトレンド
  • Visual Studio 11、Web開発者にとって何が新しくて便利ですか？
この機会を利用して、参加者に発言の遅れについて謝罪します。 このような遅延が将来発生しないように取り組んでいます。
