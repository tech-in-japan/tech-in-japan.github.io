Chrome 5.0 DEV



本日、Chrome 5.0がdevブランチでリリースされました。

*アドレスバーの近くにプラグインアイコンを最小化する機能を追加

*コンテンツ管理メニューを追加（まだアクティブではありません）















Chrome Chanel Changerを使用してチャンネルをdevに変更すると、Chrome devをインストールできます
（元の場所を覚えていないため、リロードする必要がありました）
