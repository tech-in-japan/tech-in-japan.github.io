Code ConferenceのMarie Meekerレポート：ハイライト



写真ソース： Recode.net



ベンチャーキャピタル会社Kleiner Perkins Caufield＆ByersのパートナーであるMarie Meekerは、カリフォルニアで開催されたCode Conferenceで彼女のスーパーレポートを共有しました。



サイトRecode.net （会議の主催者）は、シリコンバレーの最も予想されるレポートの1つの概要を簡単に作成しました。 今年のレポートには、355枚のスライドと大量の情報が含まれていました。これには、ヘルスケアに関する新しいセクションが含まれていますが、マリーはライブでは発表しませんでした。



レポートのいくつかの参照ポイント：



  • スマートフォンの世界的な開発は減速しています。スマートフォンの出荷は昨年と比べて3％増加しました-昨年、この割合は10％でした。 さらに、Meeker が昨年話題にしたインターネット全体の成長を減速させ続けています。
  • 音声は、オンラインクエリの入力を置き換えます。 音声の助けを借りて、2016年にはモバイルリクエストの20％が行われましたが、現時点での精度は95％です。
  • 過去10年間で、米国のホームエンターテイメント業界の総収益に占めるNetflixの割合は0％から30％に増加しました。 同時に、テレビ視聴者は減少し続けています。
  • Meekerは、起業家はしばしばゲーマーであると判明し、Ilon Musk、Raid Hoffman、Mark Zuckerbergを引用していると指摘しています。 国際的なインタラクティブゲームが主流になりつつあります-1995年には1億人だったのに対し、2017年には既に26億人のゲーマーがいます。 2016年の業界の総収益は1,000億ドルと推定されています。 インタラクティブゲームの主要市場は中国（予想）です。
  • 中国は依然として最も興味深い市場であり、モバイルサービス、支払い、需要共有自転車共有などのサービスが大幅に成長しています。
  • インターネットの成長は世界中で減速していますが、インドは経済成長が最も速いため、これらのプロセスにはまったく関心がありません。 インターネットユーザーの数は2016年に28％以上増加しました。これはオンライン全体の普及率のわずか27％であり、これはインターネット視聴者の拡大の大きな範囲を意味します。 ブロードバンド接続のコストが削減されるため、モバイルインターネットも成長しています。
  • 2016年、米国では、ハイテク市場で最も評価の高い企業の60％が、第一世代または第二世代のアメリカ人によって設立されました。 これらの企業は150万人以上の従業員を雇用しています。 これらの企業には、Apple、Alphabet、Amazon、Facebookなどの大手企業が含まれます。
  • ヘルスケア：ウェアラブルガジェットは徐々に信頼されています-現在、アメリカ人の25％が少なくとも1台のデバイスを購入しています（2016年の12％と比較）。 最先端のハイテクブランドはすでにデジタルヘルスケア市場に定着しており、消費者の60％以上がGoogleやMicrosoftなどの特定の企業と健康データを共有することを望んでいます。
