ASUSはキーボードを備えた新しいUMPC T83をCeBITで公開しました



このUMPCは、そもそもキーボードが存在する点で競合他社とは異なります。



仕様T83：

  • 7 "800x480の解像度で押すことに敏感な画面。
  • 1 GHzの周波数で動作するC7-Mプロセッサ経由。
  • RAM 512ギガバイト（最大2ギガバイトまで増加する可能性があります）。
  • 30ギガバイトのハードドライブ
  • ビデオカメラ;
  • GPS
  • Bluetooth
  • Wifi
  • 重量950グラム
  • バッテリー寿命6時間
  • Windows XP Tablet Edition








Q1 Ultraと比較すると、構成はわずかに弱いです（ボードにデュアルコアがあり、OSとしてVista Home Premiumを使用している、画面の解像度が高いなど）。 しかし、T83をおもしろいデバイスと見ていなくても。 UMPCは今年の第3四半期に発売され、その費用は899〜999ドルの間になります。
