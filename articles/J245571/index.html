MS KB3004394を更新すると、virtualboxの仮想マシンが終了する

Microsoftからの最新の更新セットをインストールした後、virtualboxが既存の仮想マシンの実行を停止したことが発見されました。 新しく作成された仮想マシンを起動しようとしても、同じエラーで終了しました。

推奨カテゴリからのKB3004394更新（WindowsのWindowsルート証明書プログラムの2014年12月更新）が原因であることが判明しました。 今日virtualboxを復元する唯一のオプションは、指定されたアップデートを削除することです。 更新プログラムとアプリケーションの非互換性の他の例も示します。 Windows 7 / Server 2008 r2の実際
