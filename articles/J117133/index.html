Linux上のクラウドからの3D CAD

最近、コンピューター支援設計システム（CAD）についてHabrにいくつかの記事が掲載されました。主に、2次元および/または3次元の幾何学的設計を自動化し、設計および/または技術文書を作成するCADシステムに関する記事です。 これらの記事は、たとえば、Linux、さまざまなiOSまたはAndroidデバイス、またはSAASモデルを使用したクラウドサービスに関する物語ほど多くのコメントを集めていませんでした。 しかし、これらすべての種類のカクテルを混ぜるとどうなりますか？ 始めましょう。



国内のCADベンダーの1つがすでにこのカクテルを準備しています。 試してみるだけです。 この記事では、UbuntuオペレーティングシステムでCloud Compass-3Dを構成および実行する方法について説明します。 ただし、Windowsですべてが単純な場合、Linuxのセットアップにはいくつかの微妙な違いがあります。 それらについてはさらに説明します。



まず、プロジェクトのWebサイト（ http://cloud4business.com）にアクセスして登録する必要があります。 はい、再び長いアンケート。パスポートと指紋のコピーだけでは不十分です。 しかし、私たちは落胆していません。 最も重要なことは、確認メールを送信する実際のメールアドレスを指定することです。







だから。 手紙が届き、登録が確認されます。 サイトにログインします。 セクションソリューションカタログ->デモンストレーションソリューション-> Compass-3Dに移動します。 [デモアクセスを取得]ボタンをクリックします。







メールで 登録時に指定されたメール、ユーザー名/パスワードのペアが15日間製品にアクセスするために送信されます。



このサービスはCitrix XenAppテクノロジーに基づいているため、アプリケーションを仮想化し、このアプリケーションをサービスとしてクライアントデバイスに配信できるため、 Citrix Receiverクライアントをインストールする必要があります。 ちなみに、このクライアントはiPhone、iPad、Android、Windowsでも利用できます。 ネット上で、iPadでCompass-3Dを起動する例を見つけることができます。 しかし、これは深刻なリソース集約型の3Dモデリングシステムです。 さらに、今日、Windows以外のオペレーティングシステム用の本格的な3D CADシステムはまれです。 Linux用のクライアントが必要です。 次のリンクでダウンロードできますhttp://www.citrix.com/English/SS/downloads/details.asp?downloadID=3323&productID=-1

.debパッケージを見つけて、[ダウンロード]ボタンをクリックします。 「GDebi Package Installerで開く（デフォルト）」ラジオボタンを削除し、「OK」をクリックします。 プログラムをインストールします。





インストール済みクライアントの検証


  1. インストーラーが作成したものを確認します。

    • アプリケーションランチャー->インターネット-> Citrix Receiver
    • 実行可能ファイル/ usr / lib / icaclient / wfcmgr


    Citrix Receiverクライアントを起動しようとしています。 起動にエラーがなく、図のようにウィンドウが表示された場合、アイテム番号2を安全にスキップできます。







  2. 起動に失敗した場合、必要なライブラリの可用性を確認します。 コンソールに書き込みます：

    CLIENT_EXEC="/usr/lib/ICAClient/wfcmgr"

    ldd ${CLIENT_EXEC}







    高い確率で、結果は次のようになります。

    libXm.so.4 => not foundに注意してください 。 これは、不足しているライブラリをインストールする必要があることを意味します。 まず、コンソールに入力します。

    sudo aptitude install libmotif3





    このコマンドを使用して、 Motifインターフェイス要素ライブラリをインストールします。

    ファイルをリストします。

    ls -al /usr/lib/libXm.so*





    リストにあるライブラリの最新バージョンがlibXm.so.3.0.2である場合、次のコマンドを使用してファイル間にソフトリンクを作成します。

Compass 3Dを起動する


さて、最後に、製品のすべての予備設定と、クラウドCADの直接テストに進みます。
