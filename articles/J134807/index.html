メリディアン-VkontakteのMetroスタイルの音楽プレーヤー

投稿はソーシャルネットワークVKontakteで音楽を聴くすべての人に捧げられています。



皆さん、こんにちは！ 今日は、ソーシャルネットワークVkontakte-Meridianから音楽を聴くための非常に興味深いアプリケーションを検討します。







なぜ面白いのですか？



最初に、もちろん、そのインターフェイスを使用します。



Metroのスタイルで作成されています（Windows 8、Windows Phone 7、Zune、Merotwitを覚えていますか？）。 これにより、従来のデバイスとタッチデバイス（現代のスマートフォンとタブレット）の両方でアプリケーションを使用することが非常に便利になります。 彼は個人的にそれをチェックしました-これはすべて良いです。







第二に、その機能。



あなたはどこからでもオーディオ録音を聴くことができます-あなたのお気に入りから、あなたの壁から、あなたはあなたの友人のお気に入りの曲、グループからのトラックのサポート、ラジオ、LastFM scrobblingなどを聞くことができます。 など このアプリケーションは、他のサービス（LastFMなど）と緊密に連携しており、お気に入りのジャンルや現在の気分に関する音楽のおすすめを受け取ることができます。







もう1つの重要な点は、アクセシビリティです。



アプリケーションには3つのバージョンがあります。

  1. VKontakte Webアプリケーション 。 ここには、最も高度な改善点がすべて最初に表示されます。 HTML5で記述されているため、最新のすべてのデバイスでアプリケーションを使用できます。
  2. Windows用の.NETバージョン 。 優れたクライアントで、Webバージョンよりも機能がわずかに劣りますが、同時に独自のチップがあります-配色の変更、歌のVKontakteステータスへの設定など。
  3. Meridian for Windows Phone7。アプリケーションはまだテスト中であり、Marketplaceからはダウンロードできません。 私はテストに参加することができました-私はすべての主要な機能がそこにあると言うことができます。


その他の利点



アプリケーションの開発は非常に高速です。 さまざまな新機能が常に追加されています。 アプリケーションのブログでそれについて読むことができます。 開発者（.NET、WP7- https: //twitter.com/#!/Stealth2010およびWeb- http : //vkontakte.ru/hart_flamov）は、さまざまな改善と修正を受け入れています。 最近、私は1つの興味深いラジオを私に追加するように頼みました-数分後、それはすでにアプリケーションに追加されました。



結論の代わりに



個人的には、このアプリケーションは「ビルドおよびライブヘルプ」です。 私もあなたがそれを楽しむことを願っています-それを試してみてください。

アプリケーションの最初のバージョンのソースが利用可能です。 現在のバージョンについて-著者はまだコードのレイアウトを開始していません。



LastFMにスクロブルがありますか

あります。 両方のバージョン。 設定するには-歯車付きのメリディアン碑文をクリックします。



どの言語が利用可能ですか？

Windowsはロシア語と英語です。 Web-ロシア語。
