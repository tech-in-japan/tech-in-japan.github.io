VKontakte、Facebook-世界の主要なソーシャルネットワーク









Operaモバイルレポートでのモバイルインターネットの年間成長率のレビュー

オスロ、ノルウェー-2008年1月20日

事実

本日公開されたOperaのモバイルレポートによると、VKontakteとFacebookのソーシャルネットワークは、Opera Miniのユーザーの中で最も訪問者です。 Opera Mini Facebookネットワークのユニークユーザーの数は、2008年に693％増加しました。 ロシア語圏の国ではOpera Miniが非常に人気があるため、VKontakteは最も人気のあるソーシャルネットワークです。 サイトのユーザーベースは、2008年にほぼ910％増加しました。 完全なレポートは、 www.opera.com / smwで入手できます。



Opera Miniユーザーは12月に64億ページ以上を閲覧しました。 11月以降、閲覧されたページ数は13.0％増加しました。

Opera Miniユーザーは、世界中から9380万メガバイト以上のオペレータートラフィックを使用しています。 11月以降、使用されるトラフィック量は13.2％増加しています。 Opera Miniのデータは平均で90％圧縮されています。 データが圧縮されていない場合、Opera Miniユーザーは12月に9億3800万メガバイトのトラフィックを使用します。

注：11月、Opera Mini統計の定期的なチェック中に、ユニークユーザーのカウントに関連する技術的な問題を発見しました。 この問題は、12月のデータが登場する前に解決されました。 2008年10月までのすべてのデータは概算です。 データを修正したらすぐにグラフを更新します。 これは一意のユーザーの情報に影響しましたが、成長レベル、ページビュー、トラフィックに関するデータは信頼できます。 実際、1か月あたりのユーザーあたりの閲覧ページ数は約360増加しました。

結果

Operaの最初のモバイルレポートでソーシャルネットワークが調査されました。 この号では、Operaは主要国の主要なソーシャルネットワークを対象としています。

世界中で、Opera Miniユーザーのトップ10のソーシャルネットワークは次のとおりです。

米国では、Facebookが年間の著しい成長によりMySpaceに取って代わりました。その結果、ユニークユーザーの数は1632％増加しましたが、MySpaceでは52％近くのユーザーが増加しました。

Facebookは、英語圏の国で成長を続けています。 英国では、Facebookは536％の増加を達成し、Opera Miniユーザーの間で最も訪問されたソーシャルネットワークになりました。 南アフリカでは、Facebookも主要なソーシャルネットワークです。 英語圏以外では、Facebookはエジプトのユーザーの間で3400％以上の増加で成功を収めています。

ただし、MySpaceのすべてが失われたわけではありません。 ドイツでは、MySpaceが最も人気のあるソーシャルネットワークであり、2008年に492％成長しています。

有線インターネットとモバイルインターネットの両方で、Orkutはインドとブラジルで最も人気のあるネットワークであり、Nasza-klasaはポーランドで最も人気のあるネットワークです。 これにより、さまざまなデバイスからアクセスできるインターネットは1つしかないというOperaの信念が再び強まります。

引用

Opera SoftwareのCEOであるJon von Tetchner氏は、「ソーシャルネットワークは統合されたインターネットの開発をサポートするのに役立ちます。現在の経済状況にもかかわらず、モバイルデバイスからのインターネット利用のさらなる成長を期待しています。」 世界中でインターネットへのアクセスを望む人が増えています。 先進国と発展途上国の両方で大きな可能性を秘めているため、携帯電話またはその他のデバイスを使用してインターネットにアクセスする人が増えると予想されます。

Opera Software ASAについて

Opera Software ASAは、PC、携帯電話、その他のインターネットデバイスでWebを閲覧するための新しい可能性を切り開きました。 Operaが開発したクロスプラットフォームブラウザーテクノロジーは、その機能、標準への準拠、小さなサイズで知られ、ユーザーにより高速で安全で動的なインターネットアクセスを提供します。 Opera Softwareの本社はノルウェーのオスロにあり、世界中にオフィスがあります。 同社は、OPERA証券コードでオスロ証券取引所に上場しています。 詳細については、 www.opera.comをご覧ください。
