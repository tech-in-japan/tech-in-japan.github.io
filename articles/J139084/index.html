Parallels Plesk 11プレビュー：クラウドへのステップ

みなさんこんにちは。 最近のParallels Summit 2012では、コントロールパネルをホストするParallels Plesk Panelの11番目のバージョンが導入されました。 私はこの記事で説明したい製品のプレビューを自分の目で見ることができました。 要するに、多くのパネル機能がユーザーの影響下で作成または改善されました。 新しいインターフェイスや製品の「クラウド」使用に対する親しみやすさを強調した現在の「チップ」でさえも、「存在する」だけでなく、プロのユーザーのニーズと、ホスティング事業者が近い将来獲得するものに注目して作られました。







今夜、Parallelsは11番目のバージョンのプレビューをアップロードしました。 投稿内の専門家向けに、新しいパネルのWindowsおよびLinuxディストリビューションへのリンクがあります。 気にするすべての人、私はあなたを猫に招待します。







おそらく、Habrのブログ「ホスティング」のすべての読者は、Parallels Plesk Panelについて知っています。 「スプラッシュ」は、多くの場合、同じ製品内で3つの重要な機能を提供するため、プロのホスティング市場への「エントリーチケット」になります。これは、コントロールパネル自体、課金、ウェブサイトビルダーです。 Parallels Pleskは箱のような性質のため、展開と設定が簡単で、これに基づいて、専用サーバー、VPS、および（必要に応じて柔軟性はありませんが）共有ホスティングの多くの基本サービスの提供をすぐに開始できます。 気にする人は、Parallels Pleskとその主要な競合他社のホストに関する3つのパート（1、2、3）のすばらしいレビューを読むことができます。



しかし、よくあることですが、「スプラッシュ」の欠点はそのメリットの継続です。 主な問題は、Parallels Plesk Panelで提供されるサービスが、サーバーの数（仮想または物理、それほど重要ではない）を増やし、これらのサーバーに新しいPleskaライセンスをインストールすることによってのみスケーリングされることです。 この開発モデルは、すべての人に適しているわけではありません。 主なことは、サミットで個別に説明したように、Parallels自体には不便であるということです。 結局のところ、会社はホスティング会社からの控除で稼いでいます。 パートナーが毎月より多く売れるようにすることは彼女の利益です。 より多くの顧客に販売する-ベンダーからさらに注文します。 各パートナーからのリースの支払い額は、数量によって異なります。



開発者はほぼそのような考慮事項に導かれ、「スプラッシュ」の11番目のバージョンが作成されました。 イデオロギー的に新しいParallels Pleskは、新規サイトのスターターキットであり、クライアントサイト自体の管理を好む小規模プロバイダーやWebスタジオのビジネスツールであり続けました。 Pleskaチームはパートナーからフィードバックを収集し、既存の機能を再考し、管理の容易さの観点からそれらを実装しようとしました。



インターフェース





時々、インターフェイスの「スプラッシュ」が非難されますが、この問題にはまだ統一性がありません。 一部のユーザーは、「古い」バージョン（バージョン9.x以前）を好みます。 バージョン11では、「トップ10」で具体化されたアイデアがさらに開発されました。 どれほど便利かはまだ検証されていません。 しかし、新しいインターフェースは非常にきれいに見えます。 同社の最も有名で巨大な製品であるParallels Desktop for Macの影響を感じることができると思います。

クラウド機能





「なぜパネルはクラウドに行くべきなのか」という質問に対して、彼らはParallels スタディの数字で答えてくれました。 同社の予測によると、2014年までに、ホストされたインフラストラクチャは、640億ドルのクラウド市場の約27％を占有します。つまり、パイのシェアを獲得するには、クラウドインフラストラクチャと結婚できるパネルと協力する必要があります。 クラウドマネーの通過を防ぐために、パネルには次の機能が備わっています。



  • Pleskがインストールされた仮想マシンおよび/またはコンテナのクローニングとクイック起動。 インスタンスを手動で再作成する必要はありません。 新しいインスタンスに対するPleskのライセンスは自動的に登録されます-このサービスはプロバイダーによってアクティベートされる必要があります。 Parallels Server Bare Metalと主要ベンダーの仮想マシンの両方がサポートされています。
  • クラウド/ VPSの下での価格の「シャープネス」。 このルーブル会社は、パートナーに仮想プライベートサーバーとクラウドインフラストラクチャサービスを提供するよう奨励しています。 ベンダーのハイパーバイザーを使用する場合、Plesk Unlimited DomainsおよびPlesk 100 Domainsパネルバージョンには特別な条件が適用されます。 これはcPanel VPS Optimizedの明確な答えであり、cPanel VPS Optimizedのサービスはより安価であり、したがって、ホスティング事業者に価格操作の余地を与えました。
  • デフォルトでパワーユーザーインターフェースモード。 VPS /クラウドの使用に最適化されています。 シンプルビューオプションを追加しました。これにより、使用しないブロックを除外して、インターフェイスをカスタマイズできます。




Webサーバー





  • Apacheのポンプサポート。 Apacheモジュールは、対応するチェックボックスをチェックすることで追加および除外できます。 これにより、各サーバーのRAM消費量をより適切に制御できるため、ノードあたりのVPSの密度が向上します。 「万全」：Apacheの重要なモジュールは停止できません。
  • Nginx。 最も人気のある.ruドメインゾーンWebサーバーは、数回クリックするだけでPlesk 11に自動的にインストールされます。 同志aleckseyとfreem4n 、あなたは聞いた:)




郵便





  • 改良されたLinuxパネルメールシステムは、Postfixに基づいています。 100,000以上のメールボックスを保持（テスト済み）。
  • 単一のメールハンドラー（ウイルス対策など）の問題は、メールの到達範囲全体には影響しません。
  • メールはクライアントIP（専用または共有）を通過するため、クライアント間のスパマーは一度に計算されます。




Windows用のParallels Plesk





  • PHPサポート5.3。
  • Microsoft Web Matrixのサポートの改善。
  • タンバリンと踊らずにParallels Pleskを完全に削除する機能。 多くの人が知っているように、Plesk for Windowsのアンインストールは以前は問題でした。
