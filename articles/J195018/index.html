静かにAppleがiMacラインを更新





第4世代Intelクアッドコアプロセッサ、新しいグラフィック、次世代Wi-Fi、より高速なPCIeドライブ。



21.5インチiMacは、4コアIntel Core i5 2.7 GHz、Iris Proグラフィックスを受け取りました。



21.5インチiMacのポンプアップバージョンと両方の27インチバージョン（従来は異なる特性を持つ2つの異なる構成）は、Intel Core i5 3.4 GHzとメモリ量が2倍のNVIDIA GeForce 700を受け取りました（40％のパフォーマンス向上が見込めます）。 プロセッサをIntel Core i7 3.5 GHzに、ビデオを4ギガバイトのメモリを搭載したNVIDIA GeForce GTX 780Mに置き換えることもできます。



すべてのモデルに802.11ac Wi-Fiが搭載されました。



また、新しいaimakにはPCIeフラッシュドライブが搭載されており、Fusion Drive（Appleが推進しているハイブリッドSSD + HDD）およびすべてのフラッシュドライブの速度が前世代と比較して最大50％向上します。



1TBまたは3TBのFusion Driveサイズを新しい構成に挿入できます。



デフォルトでは、8 GBのRAMが32 GBまで拡張可能です。



2つのThunderboltポート、4つのUSB 3.0。



Apple Webサイトのオンラインストアは、今日利用可能です。



Apple Webサイトに投稿します。
