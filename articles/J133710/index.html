SkyDriveの新しいバージョン：簡単なファイル共有、HTML5ダウンロードなど。

SkyDriveをホストするクラウドは、ようやく通常の外観にまで成長し、少なくとも機能面では、DropboxやGoogle Appsと競合できるようになりました。 少なくともこの形式では、友人に推薦することはもはや恥ではありません。 今朝、開発者はSkyDriveの新しいバージョンを公開し、完了した作業について報告しました。







まず、ホスティングにファイルをアップロードするためのインターフェースが完全にやり直されました。 以前のバージョンがWindows Liveユーザー間のファイル共有用に設計されていた場合、今では差別はありません。 マウスを数回クリックするだけで、任意の住所にリンクを送信すると、すぐにFacebookに投稿できます。 または、受信者に渡すことができるURLを取得します。







写真ホスティングには通常のコンテキストメニューがありました。







他の多くのインターフェースの改善が行われました。 たとえば、フォルダー全体だけでなく、フォルダー内の個々のファイルのプロパティ（プライベート/パブリック）を変更できるようになりました。 実際、以前は、「プライベート」フォルダ内のドキュメントで作業しているときに、このドキュメントへのアクセスを同僚に許可することはできませんでした。 最初にドキュメントを別のフォルダーにコピーする必要がありました。



別の例-フォルダー名をクリックするか、F2ホットキーを押すだけで、このウィンドウでフォルダーを直接作成または名前変更できます。











これで、ファイルを1つずつのみ一括アップロード、ダウンロード、および移動できます。







最も重要なことは、ブラウザーでのファイルと写真のダウンロードが、廃止されたSilverlightではなく、Chrome、Firefox、Safari、IE10などの最新のブラウザーでサポートされている通常のHTML5 File APIによって実装されるようになったことです。 ダウンロードはバックグラウンドで行われ、非常に便利です。







PDF、RAW、その他の形式のサポートも登場しました。
