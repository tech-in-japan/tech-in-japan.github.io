www.DinkyPage.com-5秒で使い捨てページ



インターネット上でワンタイムページを作成するためのシンプルなサービス 。



"dinky" （英語）-小さくてささいな。



何かをオンラインで公開する必要があるかもしれませんが、あなたのサイト/ブログへのアクセスは手元にありません（またはサイト/ブログは手元にありません）。 このサービスはこの問題を解決します。



別の例-面白いものを友人と共有したいのですが、これには140のTwitterキャラクターでは不十分です。 ページを作成し、興味深い記事を書いて、そのページへのリンクをTwitterに投げます。



そして、それはしばしば起こります-私はJavascriptで何かを試してみたいのですが、新しいファイルを作成したり、正しいDoctypeをコピーしたり、基本的なスタイルを書いたりするのが面倒です。







どのように機能しますか？

大きな青いボタンを押すと、 dinkypage.com / XXXのようなアドレスのページが表示されます。 ビジュアルモード（FCKeditor）で編集するか、HTMLに切り替えることができます。



YouTubeのスクリプト、写真、ビデオ、およびフラッシュを挿入できます（ 他の場所でホストする必要があります ）。



いつでもどこでも編集できます。エディターでページの元のアドレスを覚えておく必要があります（Cookie、パスワード、登録は使用されません）。



次は？

  • 選択したアドレスでページを作成します（例：dinkypage.com/love-letter）。
  • ページを削除します。
  • サブページの代わりにサブドメインを使用する（dinkypage.com/YYYの代わりにYYY.dinkypage.com）。
  • ページ（JSON、.net）を作成および編集するためのAPI。
  • 人間404
  • パスワードでページを閉じる機能。
  • 電子メールでページのリストを取得します（指定されている場合は、分散型ラップトップとしてサービスを使用できます）。
  • 表示を改善し、携帯電話で作業します。
  • ページに関するコメント（著者が必要と考える場合）。
  • 目に楽しいパターンを追加します。
  • ページ訪問に関する簡単な統計を追加します。
  • ページのバージョン管理を追加します。
  • ドメインをページにバインドする機能を追加します（単一ページのサイトを作成できます）。




お金を稼ぐ方法は？

この質問はまだ価値がありません。 主なことは、必要かつ便利なサービスを提供することです。 最も興味深いのはこれからです。



PS

サービスについての考えやフィードバック（アイデアと実装の両方について）を聞くのは興味深いです。 twitter.com/dinkypageから新機能の発表をフォローできます



APIを使用した信頼できる無料の画像ホスティングを知っている人がいれば、知識を共有してください。



よくある質問

作成したページはいつ自動的に削除されますか？

ページは、利用規約に違反した場合にのみ削除されます。 それらは、サービス自体が生きている限り利用可能になります（サービスが非常に長く生きることを願っています:)。



エディターをロードしたくありません。HTMLコードを含むテキストエリアが必要です

2つのオプション：

1.編集ページのアドレスに＆mode = htmlを追加します。

2. JavaScriptを無効にします。
