チーム音声通信ソフトウェアの概要

ネットワークゲームでの音声通信の問題は、それがグランドMMORPGであろうと単純なネットワークシューティングゲームであろうと、非常に深刻です。 アクティブなアクションでは、多くの場合、ゲームに組み込まれているチャットを使用するのに十分な時間がありません。ほとんどの場合、チャットを使用すると、敵が音声通信を使用しない限り、他のプレイヤーの前にいるパートナーとの状況が大幅に悪化します。



参加者間の相互作用が重要であるチームゲームでは、高速で高品質のコミュニケーションが必要なだけです。これを公理に取り入れる必要があります。 はい、そして友人とマチュークを交換することはチャットよりもずっといい声です。



多くのゲーム開発者は製品に音声通信の手段を組み込んでいますが、多くの場合、これらのツールは非常に不便であるか、音質と機能が不十分です。 通常、サードパーティ製品は、通信品質のほか、豊富な設定と使いやすさの両方の点で、ゲーム内の音声コミュニケーションツールよりも優れています。



この記事では、音声通信用の4つの最も人気のあるプログラムを確認します。 候補者に会う-Ventrilo 、 TeamSpeak 2 、 TeamSpeak 3 、 Mumble 。 それらのいくつかは有償で、いくつかは完全に無料であり、 オープンソースですので、最良の候補者を特定してみましょう。 次の実験的な品質を調べることができます：サポートされているコーデック、サポートされているオペレーティングシステム、サーバーの数と可用性、スケーリング、サーバー部分を専用サーバーに配置する機能、サウンド拡張アルゴリズムなど。





ベントリロ







公式ウェブサイト-http://www.ventrilo.com/



私たちのレビューの最初の候補は、非常に長い間市場に出回っているかなり有名な製品-Ventriloです。 正確には、 Ventriloは 2002年から開発を続けています。 現時点では、最新バージョンは3.0.5です。 Ventriloは、 Windows 95から64ビットバージョンを含むWindows 7で終わるすべてのMicrosoft Windowsオペレーティングシステム、 Mac OSX 10.4以降で利用可能です。Linuxバージョンは3年以上開発されており、会社が法廷に出した時点では一般に知られていませんただし、 LinuxではクライアントはWineで実行できます。 Ventrilo - Manglerサーバーに接続できるLinux用のオープンソースクライアントがあります。



Ventriloは商用製品であり、エンドユーザーには無料ですが、サーバーのレンタルには料金がかかります。無料のサーバーはありません。 価格はそれほどかからず、質の高いサーバーは10席で4.95ドルから、400席で99.95ドルまでレンタルできます。 賃料は、 モスクワを含むヨーロッパとアメリカのほぼすべての主要都市で可能です。



このお金のために何を提供しますか：コーデックのサポート-Speex、GSM 6.10、Lernout＆Hauspie、DSP Group TrueSpeech。 プレイヤーが自分の名前の発音でチャンネルに出入りすることの音声通知。 チームブロードキャストの特権をすべてまたは選択したチャンネルに付与する機能。 ノイズリダクション; 各ユーザーの個別の音量設定。 overlay-スピーカーのニックネームをゲームの上に表示します; キーボード画面Logitech G15、G13、G19のサポート。 ユニークな機能にはPhantomsが含まれます。これはあなたのダブルであり、任意のチャンネルに配置して、彼らの言うことを聞くことができます。



多くのゲームコミュニティは、優れた音質と優れたスケーラビリティのためにVentriloを選択します。人々のためにより多くの場所が必要な場合は、サーバー上のより多くの場所を借りることができます。 サーバーレンタルは、99.9％の時間で音声通信を提供し、あらゆる種類の遅延を排除します。これは、大規模なゲームコミュニティにとって非常に重要です。

TeamSpeak 2







公式ウェブサイト-http://www.teamspeak.com/



TeamSpeak 2は 2002年から開発されており、最新バージョンは2007年です。 TeamSpeak 2は完全に無料で、 Microsoft WindowsおよびLinuxで利用できます。これらのオペレーティングシステムでは無料のサーバーパーツも利用できるため、ゲームコミュニティの無料サーバーをアップグレードできます。



サーバーの近くにマシンがない場合、またはインターネットチャネルに問題がある場合は、無料のサーバーを多数使用するか、自分でレンタルすることができます。価格は1か所あたり0.14ドルからです。



プログラムの機能はほとんどありません。オーバーレイはありません。キーボード画面はサポートされていません。すべてのチャンネルにブロードキャストする可能性はありません。ノイズは減少しません。 利点は、プログラムとサーバーの普及です。ほとんどのゲームコミュニティはTeamSpeak 2を使用しています。





TeamSpeak 3







公式ウェブサイト-http://www.teamspeak.com/



TeamSpeak 3は、 TeamSpeak 2の進化版です。 26ベータ版は現在入手可能で、リリース日は不明であり、プログラムは長い間開発中です。 本質的に、 TeamSpeak 3はバグに関する作業であり、以前のバージョンのほとんどすべての欠点が取り除かれています。



ほとんどすべてのオペレーティングシステム用のクライアントがあります-Microsoft Windows 、 Linux 、 MacOS 、これらのOSに加えて、サーバー側にもFreeBSDのバージョンがあります 。



TeamSpeak 3は人気を博し始めていますが、その機能と無料での評価から判断すると、ゲームコミュニティ間の音声通信で最も人気のあるプログラムになることが明らかになりました。 すでに、通信用の無料サーバーが多数あります。





マンブル







公式サイト-http://mumble.sourceforge.net/



Mumbleは無料のオープンソースプログラムであり、有料のライバルであるVentriloなどのすべての機能を備えています。 ユニークな機能の中で、サウンドの位置に注意する価値があります（たとえば、プレイヤーがあなたの右にいて話している場合、サウンドもあなたの右に聞こえます。プレイヤーがあなたから遠く離れている場合、サウンドはこもります）。 、SSL経由ですべてのトラフィックを暗号化し、エコーを効果的に抑制することもできます。



Mumble ClientとMumbleは、 Microsoft Windows 、 MacOS、およびLinuxで利用できます。



最近、 Mumbleは多くのユーザーを獲得し、人気を博しました。世界中に多くの無料サーバーがあります。
