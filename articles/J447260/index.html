Visual Studio 2019のライブ共有

Visual Studio Live Shareの一般公開と、この機能がVisual Studio 2019に含まれることをお知らせします！ Live Shareのパブリックプレビューのリリースから1年間、チームワークの方法の改善に取り組んできました。 このリリースは、この作品の集大成です。



Live Shareについて聞いたことがない場合、これは同僚と一緒にリアルタイムで開発を行うことができるツールです。 リポジトリのクローンを作成したり環境を設定したりすることなく、コードを共有、編集、デバッグできます。 Live Shareを始めましょう 。







ご意見ありがとうございます！



すばらしい製品のレビューとディスカッションに満足しました。 ご意見は、Live Shareの開発に役立っており、開発の最も重要な領域に集中することができました。 フィードバックに基づいて、 読み取り専用モードなどの機能、 C ++やPythonなどの追加言語のサポートを追加し、ゲストステータスを持つユーザーがデバッグセッションを開始できるようにしました 。







さらに、さまざまな開発チームがどのように機能するか、および一般的なLive Shareの適用方法について多くのことを学びました。 Live Shareは、ペアプログラミング、コードレビューの実施、学生や同僚への講義やプレゼンテーションの実施中、またはハッカソン中のモバイル開発中にも使用できます。 Live Shareは、共同開発のための多くの機会を提供します-一緒に、同じオフィスで、自宅から、または地球のさまざまな部分から。



サードパーティの拡張機能



Live Shareは、プロジェクトコンテキスト全体のコラボレーションです。 これは、Visual Studioのコードだけでなく、使用される拡張機能でもあります。 このリリースとともに、Visual StudioでLive Shareを拡張するためにいくつかのサードパーティの拡張機能を使用できるようになりました。



OzCodeは、C＃のデバッグ機能を視覚化スイートで拡張します。 たとえば、「データヒント」を使用すると、LINQクエリがどのように機能するかを確認でき、「ヘッドアップディスプレイ」を使用すると、論理式のセットがどのように評価されるかを確認できます。







CodeStreamを使用すると、コードベースに関するディスカッションを作成して、知識を仲間と組み合わせることができます。 ところで、ほとんどの場合、組み込みのチャットを追加するように要求しましたが、CodeStreamを使用すると、同僚とリアルタイムでチャットする機会が得られます。



コラボレーション



Live Shareの作成と改善を続けます！ コラボレーション体験を改善するためのアイデアはまだたくさんあります。 有益なフィードバックを受け取りましたので、引き続きご意見をお聞かせください。 Live Shareで今後見たいものをお気軽にご連絡ください。 質問や提案を送信したり、 アンケートを行ったりすることができます。



Visual Studio 2019にデフォルトでLive Shareがインストールされているため、チームとのチャットを簡単に開始できます。 Live Shareの使用の詳細については、ドキュメントをご覧ください ！
