私は正直に生きようとします

「私は正直に生きようとします」と私は自分自身に言い、無許可のソフトウェアを取り除くことを決めました。 この決定の理由は長い間広がる可能性がありますが、私はこれを行いません。 「与えられた：海賊版プログラムを取り除く」がポイントです。 そしてオプションとして、 何を置き換えるかを見つけます。



すぐに拒否できないものがあることに気づきます。 まず、XPです。 腐ったトマトの弾薬をすべて使い切ることができます。 私は慣れていますが、理由を証明しません。投稿をペイントすることも可能です。 ハブラーで、PHP、Windows Sax、Linuxでプログラムを作成していると思われるステートメントに会いました-Tru。 まあ、私はまだ若くて緑だから、ピンクの空想の世界に住んでいる。

また、総司令官を断固として拒否することはできません。 n年の間、私は彼に夢中になった。



そのため、この商品の購入には〜5000ルーブルが必要です。 不快ですが、致命的ではありません。 どうぞ



オフィス： MS Office-> Open Office。 簡単かつ自然に合格。 6か月間、私は問題に遭遇していません。 もちろん、ドキュメントを印刷用に頻繁に交換する必要があり、同僚全員がMSを持っている場合は、問題が発生する可能性がありますが、両方のパッケージでドキュメントの書式設定が少し異なります。 しかし、ドキュメントのデザインがあなたにとって重要ではなく、独自の方法で印刷する場合、Open OfficeはMSのオフィスより悪くありません。



メーラー：バット！ -> Thunderbird。 この移行は私にとって簡単でした。 もちろん、最初はそれほど便利ではありませんでしたが、文字通り1週間後、私は既にバースが巨大で、Thunderbirdがシンプルで便利であることに気付きました。



アンチウイルス： ESET NOD32-> Avast-> Avira。 しぶしぶ、私はお気に入りのNODを削除しました。これは私に忠実にサービスを提供し、友人が私に宣伝したAvastを配信しました。 アンチウイルスは一般的に悪くはありませんが、怠zyな犬のように動作します。 多くの場合、目に見えるウイルスが存在しない場所で呪われました。 もう一度キーキーを鳴らした後、アバストはより良い時間までオフにされ、私はAviraを置きました。 彼は私をもっと喜んでくれました。 頻繁な更新、素敵なインターフェース、叫び声の減少、ウイルスの発見、そしてもちろん無料。 また、念のためアバストを辞め、「病気」かどうかを再確認しました。



CD / DVDバーナー： Ahead Nero-> Small CD Writer / Ashampoo Burning Studio。 ため息をついて、私はいつものネロをバスケットに送りました。 このテーマに関するソフトウェアの選択は、Deep Burner、CD-DVD Burnerなど、非常に優れています。 しかし、私の選択は小さなプログラムであり、それでもディスクは完全に書き込まれます。 小さなCDライター。 私のガールフレンドはレコーダーの速度を正確に決定することができず、非常に低い速度で書きました。 そのため、プロモーション中に無料のキーでAchampouをダウンロードしました。 彼は私にはあまりにもキャッチーなようでしたが、彼は同様に書いています。



仮想ディスク：アルコール120％->アルコール52％。 さて、コメントするものは何ですか。 120％のアルコールを支払う必要があったすべての機能は、ディスクの切断に関連しており、これは私が従事していた別のプログラムでした。 したがって、ディスクのみをマウントする必要がある場合、52％のアルコールを入れても違いはわかりません。



アーカイバ： WinRAR-> 7Zip。 事実、私のお気に入りの総司令官は多くの異なるフォーマットをパックおよびアンパックする方法を知っているので、原則として別個のアーカイバーは必要ありませんでしたが、私はそれを許可することにしました。



サウンド処理： Sound Forge-Audacity。 私はほとんどの場合、コピー、カット、ペースト、ボリュームの増減、その他の小さなことだけを編集しますが、これは無料のAudacityでうまくいきます。



デフラグ： O＆O Defrag Professional-> Auslogic Disc Defrag。 もちろん、Defrag Professionalはこれを上回ることはできませんが、Aulogic Disc Defragはそのタスクに非常にうまく対応しています。



ファイルの回復： Ontrack Easy Recovery-> Recuva。 私の意見では、小さくて速くて無料のRekuvaは、大きくて遅いEasy Recoveryよりも決して劣っていません。



システム情報： Lavalys Everest-> SiSoft Sandra。 Sandraの基本バージョンは無料であり、喜ぶことができるだけです。



描画： Photoshop-> GIMP、Paint.net、Photofiltre。 SoundForgeと同じストーリー。 私は少し描いて、ほとんどの場合、切り抜きや切り取りをするので、十分な無料のプログラムとその効果があります。



グラフィックビューア： ACDSee-> Fast Stone Image Viewer。 ADCの新しいバージョンがどのようなものかはわかりませんが、最初はバージョン3をハードに使用しましたが、まだ支払われています。 もちろん、FastStoneは可能な限り高速ではありませんが、グラフをロードするのが面倒なときには、カット、回転、あらゆる種類の便利な小さなことを行うことができます。 エディター。
