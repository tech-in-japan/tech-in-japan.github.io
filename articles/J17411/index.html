モスクワでのGoogle Code 2007 Day





このイベントへの言及は、非常に興味深いと約束されていますが、Habréでは見つかりませんでした。 検索の使い方がわからないことが判明した場合は、事前に謝罪します...



月曜日、Googleの女の子が職場で私に電話をかけ、とても突然だったことを謝罪したが、12月19日にモスクワで手配していた会議への参加を申し出た。 彼女は私が働いている会社の電話番号をどこで手に入れたのでしょうか？



彼女は私に電子メールを送りました。その中には会社のブログへのリンクがありました。







インターネットは絶えず進化しており、Googleはこのプロセスへの関与を感じています。 過去数年間で、特にWebサービスを操作するためのアクセス可能なAPIの登場により、ネットワーク内で多くのことが変化しました。 Webはよりオープンになり、開発者はこの開発で重要な役割を果たしています。 そのため、開発者のために、12月19日にモスクワでGoogle Code Dayを過ごします。



Google Code Dayは1日の会議で、GoogleのエンジニアがAPIを紹介するだけでなく、マスタークラスやワークショップも実施します。 本日は、主要なGoogle APIについてお話しします。また、何を、そして最も重要なこととして、それらをどのように使用できるかを示します。



  • いくつかの大規模なソーシャルネットワークのデータを処理するソーシャルアプリケーションを作成するためのOpen Social

  • Geo API。地理情報サービスとWebアプリケーションの迅速な作成用

  • GDataAPI。Web上の書き込みからデータを読み取り、APIを介してGoogleドキュメントとスプレッドシートを操作します。





モスクワでAPIを代表するのは、Google Russiaのエンジニアだけでなく、チューリッヒ、ロンドン、マウンテンビューの同僚でもあります。 最終的な会議プログラムは、後ほど公式のGoogle Russiaブログに公開されますが、今のところモスクワでのGoogle Code Dayにサインアップしてください： google.ru/googlecodeday07 、会議への参加は無料です。





ここから取られた情報： googlerussiablog.blogspot.com/2007/12/google-code-2007.html



ここで会議にサインアップしてください： google.com/googlecodeday07
