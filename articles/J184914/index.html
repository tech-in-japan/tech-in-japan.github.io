サムスンの多くの夏の新しいGALAXYおよびATIVファミリー

こんばんは、ハブラ！



先週、Samsungはロンドンでプレゼンテーションを行い、最新のGALAXYおよびATIVファミリーを紹介しました。 展示会にはなかったもの：スマートフォンやタブレットからラップトップ、オールインワン、そしてカメラまで。 以下では、初めて幅広い視聴者に提示されたものを検討します。







プログラムのハイライトは、もちろん、Windows 8ベースの2つの新しいタブレット-ATIV QとATIV Tab 3です。



Samsung ATIV Qは、同社のさまざまなポータブル機器シリーズの最初の変圧器ノートブックであり、ラップトップ、タブレット、重量とプレゼンテーションの画面操作の4つの機能モードで動作できます。













しかし、これは決して新しいガジェットで見つけられる最も驚くべきことではありません：ATIV Qは本格的なWindows 8 Pro OSおよびAndroid OSバージョン4.2.2（ちなみに、在庫-TouchWizなし）で動作します！ はい、これは間違いではありません。デバイスは2つのオペレーティングシステムで動作し、文字通りオンザフライで切り替えることができます。 これだけで、ラップトップは非常に興味深い目新しいものになります。







ATIV Qのもう1つの明らかな利点は、13.3インチの高解像度QHD +タッチスクリーン（3200×1800ピクセル）です。 ピクセル密度は、最新のフルHDディスプレイの2.8倍です-275 ppiで提供できます。 ラップトップの素材は、ダークブルーのアルミニウムです。 ケースの厚さはわずか13.9 mmで、重量は1.29 kgです。







パフォーマンスに関する限り、最小限の情報しかありません。ラップトップは最新世代のIntel Haswellクアッドコアプロセッサ、統合グラフィックス-Intel HD 4400、4 GBのRAM、最大256 GBの内部SSDストレージで動作します。



仕様：

  • 画面 -解像度3200 x 1800、ピクセル密度275 ppiの13.3インチタッチQHD +。
  • オペレーティングシステム -デュアルOS（Windows 8 ProおよびAndroid 4.2.2 Jellybean）;
  • プロセッサー -Intel Haswell Core i5 / i7 ULT
  • グラフィックアクセラレータ -Intel HD 4400
  • RAM-最大4 GB DDR3L;
  • ストレージ -最大256 GBのSSD。
  • カメラ -720p（HD）;
  • バッテリー -リチウムイオン、47Wh（宣言されたバッテリー寿命-最大9時間）;
  • 接続 -ギガビットイーサネット（アダプター付き）、Wi-Fi 802.11 abg / n、BT 4.0、1 x USB 3.0、1 x USB 2.0、micro HDMI、micro SDカードスロット、HP /マイクコンボ;
  • 寸法 -327.0 mm x 217.8 mm x 13.9 mm;
  • 重量 -1.29 kg。


新しいSamsung ATIV Tab 3タブレットの最初で最も注目すべき機能は、デバイスが完全なWindows 8 OS上で実行されるという事実です（RTではなく、注意）。 プレゼンテーションに参加した一部のIT専門家が述べたように、「現時点では、これはフル8の中で最も薄くて軽いタブレットです」。その厚さはわずか8.2 mmで、重量は550グラムです。







コンパクトなサイズに加えて、更新されたIntel Atom Z2760プロセッサ、2 GBのRAM、64 GBの内部eMMCメモリにより、ATIV Tab 3は優れたパフォーマンスを発揮します。 宣言されたバッテリー寿命は最大10時間です。 画面サイズは10.1インチで、解像度は1366 x 768ピクセルです。 タブレットは、10本指のマルチタッチをサポートし、デバイスをより便利に制御するための疎油性コーティングが施されています。







仕様：

  • 画面 -解像度1366 x 768の10.1インチのタッチセンサーPLS HD。
  • オペレーティングシステム -Windows 8;
  • プロセッサー -Intel Atom Z2760
  • RAM -2 GB LPDDR2;
  • ストレージ -eMMC 64 GB;
  • カメラ -720p（HD）;
  • バッテリー -リチウムイオン、25.6Wh（宣言されたバッテリー寿命-最大9時間）;
  • ネットワーク -LTE、3G（HSPA）;
  • 接続 -Wi-Fi 802.11 abg / n、BT 4.0、1 x micro USB 2.0、micro HDMI、micro SDカードスロット、HP /マイクコンボ;
  • 寸法 -257.4 mm x 165.3 mm x 8.2 mm;
  • 重量 -550 gr


ATIV QおよびATIV Tab 3にはSペンが搭載されており、より創造的な人や、情報の手書き入力や画面制御が好きな人にとって間違いなく便利です。 また、便利なSideSyncアプリケーションが両方のタブレットにプリインストールされているため、モバイルデバイスとPCデバイスの間に真の共生が生まれ、コンピューターでの作業中にSamsungスマートフォンの制御に簡単に切り替えることができます。 このアプリケーションを使用すると、たとえば、PCキーボードを使用してスマートフォンでSMSをダイヤルしたり、モニターの大画面でスマートフォンのマップ、写真、ビデオを表示したり、2つのデバイス間で他の多くの便利な対話を実行したりできます。



サムスンは、モバイルデバイスに加えて、2つの新しいラップトップ-ATIV Book 9 PlusとATIV Book 9 Lite-プレミアムオプションと予算オプションをそれぞれ発表したことでゲストを満足させました。







ATIV Book 9 Plusは、解像度が3200 x 1800、ピクセル密度が275 ppiで、ATIV Qにインストールされている同じ素晴らしい13.3インチスクリーンを誇っています。







パフォーマンスを担当するのは、クアッドコアIntel Haswellプロセッサ（Core i5 / i7）、Intel HD 4400グラフィックス、最大8 GBのRAM、および最大256 GBの容量の内部ドライブです。







これらはすべてWindows 8でも機能します。バッテリーは、最大12時間の使用に耐えることができます。 ラップトップケースはアルミニウム製です。







仕様：

  • 画面 -解像度3200 x 1800、ピクセル密度275 ppiの13.3インチタッチQHD +。
  • オペレーティングシステム -Windows 8;
  • プロセッサー -Intel Haswell Core i5 / i7 ULT
  • グラフィックアクセラレータ -Intel HD 4400

  • バッテリー -リチウムポリマー、55Wh（宣言されたバッテリー寿命-最大12時間）;
  • 接続 -ギガビットイーサネット（アダプター付き）、Wi-Fi 802.11 abg / n、BT 4.0、2 x USB 3.0、マイクロHDMI、VGA（アダプター付き）、3-in-1カードスロット、HP /マイクコンボ、スリムセキュリティスロット。
  • 寸法 -319.6 mm x 222.9 mm x 13.6 mm;

ATIV Book 9 Liteは、より「抑制された」技術的特徴を備えた低価格のラップトップです。







13.3インチタッチスクリーンの解像度は1366 x 768（117 ppi）です。 プロセッサはAMDの名前のないクアッドコアソリューションで、RAMの容量は4 GB、内蔵SSDドライブは最大256 GBです。 ラップトップのバッテリーは、最大8.5時間の連続動作用に設計されています。 ケースはプラスチック製です。

プレゼンテーションでは、アマチュア写真家の間で興奮する場所がありました。Androidの最新バージョンであるSamsung GALAXY NXをベースにした、交換可能なレンズを備えたその種のミラーレスカメラが初めて発表されました。 このカメラには20.3 MP APS-C CMOSセンサーが搭載されており、サポートされる感度値は100〜25600です。1.6GHzのクロック周波数で動作するクアッドコアプロセッサがパフォーマンスを担当します。 画像センサーからのデータは、独自のプロセッサーSamsung DRIMe IVによって処理されます。







GALAXY NXには、4.8インチのLCDタッチスクリーン、電子ビューファインダー（SVGA）、および内蔵フラッシュがあります。 さらに、デュアルオートフォーカスシステム（アドバンストハイブリッドオートフォーカスシステム）があります。 最小シャッター速度は1/6000秒で、最大バースト速度は8.6フレーム/秒です。







さまざまな形式でビデオを記録することが可能です。 記録された素材の最大解像度は、19フレームx 1080ピクセルで、周波数は25フレーム/秒です。







このデバイスは、3G / 4G LTEネットワーク、Wi-Fi、Bluetooth 4.0（LE）、NFC、GPS、GLONASS、およびさまざまなセンサー（加速度計、地磁気センサー、近接センサー、ジャイロスコープ、RGBセンサー）をサポートしています。 新しいカメラは、NXシリーズのすべての取り外し可能なレンズとも互換性があることに注意してください。







音声付きの写真モードなど、多数のプリセット撮影モードがカメラの負荷になります。 ユーザーは、新しいAndroidアプリケーションをインストールすることにより、カメラの機能を拡張できます。 多くのGoogleモバイルサービスが利用可能です：Chrome、ダウンロード、Googleマップ、Gmail、Google検索、Google設定、Google Plus、Google Local、Google Playストア、Google Messenger、Googleナビゲーション、YouTube、Googleトーク、音声検索、Google Playムービー＆TV 、Google Playブック、Google Playミュージック、Google Playマガジン。







デバイスの寸法は、136.5 x 101.2 x 25.7（37.65）mm、重量-495グラム（4360 mAhのバッテリー容量）です。
