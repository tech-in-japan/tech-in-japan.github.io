もう少し：子供のインターネットの問題

子どもたちが現代のテクノロジーに没頭すると、親でさえ常に対処できるとは限らない新しい脅威に直面します。 私たちの世界で子どもの安全のためにインターネットの使用を禁止することはすでに意味がありません-これは私たちの現実の不可欠な部分です。 12〜17歳の青少年の約15％にとって、インターネットは生活の3分の1を過ごす場所になっています。 思春期の若者は、この新しい経験を両親と共有しようとはしないことに注意してください。子供の半数以上が、インターネット上での望まない行動を両親から隠しています。 そして、私たちは子供たちのために常に新しい世界の子供たちに同行することはできないので、自動化された保護装置に頼ることは変わりません。



不適切なコンテンツから子供を保護する問題（厳密な保護、つまり2回のクリックで切断されない5歳の子供について話している）は、ソフトウェアソリューションの市場全体を形成しています。 これらは、リソースをブロックする2つの原則に基づいています。IPアドレスによるフィルターは、特定の「ブラックリスト」で定義されたIPでリソースを開く可能性を排除するか、「ホワイトリスト」からIPへのアクセスのみを許可します。ページのコンテンツに応じてオンザフライでアクセスをブロックします。



多くの市販製品は、両方のろ過原理を組み合わせています。 同時に、オープンおよび独自のアルゴリズムの両方がコンテンツ分析に使用され、ブロックされたリソースのプライベートまたはサードパーティの絶えず更新されるデータベースが「ブラックリスト」として使用されます。



エンドユーザーには、不適切なコンテンツから一定レベルの保護を取得するためのいくつかの異なる方法があります。



まず、ソフトウェアフィルターを個別にインストールおよび構成できます。



第二に、このようなサービスをサードパーティプロバイダー（大まかに言えば、家の所有権の入り口でフィルターをオンにするインターネットプロバイダー）に申請することです。



3番目に、独自のクライアントアプリケーションをインストールした後、デバイス（デスクトップコンピューター、タブレット、スマートフォン）でフィルターを有効にするサードパーティサービスを使用します。 私の意見では、ユーザーの観点からは、インターネットプロバイダーの機能を使用するという2番目の方法の方が簡単です。 現在すべての人がそのようなサービスを提供しているわけではありませんが、時間が経つにつれてますます多くのサービスが提供されています。



MegaFonは、多くのユーザーに長期間ペアレンタルコントロールサービスを提供するオペレーターに加わり（2011年以降）、昨年秋以来、オペレーターは更新されたChildのインターネットサービスを支店に徐々に導入してきました。 最初にテストしたのは、MegaFonの北西支店、首都支店、白人支店の加入者でした。 そして、今日は、今後数か月（2016年秋）に発売が予定されている極東地域を除き、ほぼすべてのサービスエリアで利用できます。



サービスの本質は、ユーザーがサービスの形式でデバイスにダウンロードされたコンテンツのフィルタリングを受信することです。 追加のソフトウェアをインストールして構成する必要はありません。 サービスがアクティブ化された後、コンテンツはプロバイダー（MegaFon）によってフィルター処理され、子供にとって安全な情報のみがユーザーデバイスにダウンロードされます。



このサービスはEntensys User Gate Web Filter製品に基づいており、教育機関のセグメントで既に市場に定着しており、禁止および許可されたリソースの独自に定期的に更新されたデータベースを使用しています。 法律で禁止されているサイトに加えて、成人向けの素材やさまざまなフィルターバイパスツール（匿名プロキシなど）など、子供にとって潜在的に危険なカテゴリのページが含まれています。



ブロックされたリソースのリストは、自動モードを含め、毎日更新されます。データベースに存在しないサイトは、ページをロードするユーザーリクエストを処理する前に、サーバーによって分析およびカタログされます。 コンテンツに応じて、リソースの全体または一部がブロックされる場合があります（この場合、個々のページがブロックされます）。



このソリューションは、サービスの最初のバージョンで使用されました。 当時、選択には独立した評価が役割を果たし、ブロックされる数百のサイト（ポルノ、薬物情報、国家主義的な資料などを含む）のデータベースを使用してさまざまなソリューションを独自にテストした結果もありました。



前述のように、モバイルネットワーク加入者向けのサービスのMegaFonポートフォリオでのコンテンツフィルタリングは以前から存在していました。 しかし、更新されたバージョンのサービスは、以前に提案されたDNSフィルタリングとブラック/ホワイトリスト機能に加えて、HTTPトラフィックのフィルタリング、メイン検索エンジンでの安全な検索、バナー広告の制限、ダウンロードされたファイルのチェック、不要なページコンテンツの形態学的フィルタリングを提供します。 したがって、青少年は自分の年齢や知覚レベルに対応しない情報を扱う必要はありません。











システムは次のように動作します。 DNSおよびHTTP / Sフィルターを使用すると、データベースに存在する不要なリソース、および許可されたページの一部の不適切なコンテンツユニット（特に、ブロックされているデータベースで指定されたリソースからのバナー広告とポップアップ、埋め込みビデオおよび画像）をブロックできます。











形態学的分析は、追加のフィルタリングを提供します。 安全な検索とは、大人が2つの方法で解釈できる検索エンジンへのクエリ（たとえば、「ぬれた小さな犬」というフレーズ）に応答して、子供が予期しない結果を受け取らないか、検索結果が単にブロックされることを意味します。 安全な検索は、ほとんどの一般的な検索エンジン（Google、Yandex、Yahooなど）、およびYouTubeサービス内の検索に有効です。







サービスのアップグレード中に、オペレーターはトラフィックフィルターを提供するインフラストラクチャを展開しました。特に、サーバーの各ブランチにサーバーにUser Gate Webフィルターを配置しました。 以前は、単一のサーバーにDNSクエリを送信することでフィルタリングが実行され、応答としてIPリソース（有効な場合）またはIPスタブページ（要求されたリソースがブロックされたものに属する場合）が返されました。











現在、トラフィックの処理は各ブランチで個別に実行されます。 潜在的に作成されたインフラストラクチャにより、すべてのサブスクライバトラフィックを処理できますが、これまでのところ、フィルタリングはDNS、HTTP、およびHTTPSに対してのみ実行されます。







コンテンツフィルタリングテクノロジーは常に進化していますが、今日のサービスにはいくつかの制限があります。



まず、ブラウザに組み込まれているトラフィック節約ツール（OperaのTurboモードなど）と一緒に使用できないため、接続が遅いモバイルユーザーに影響を与える可能性があります。



第二に、フィルタリングはエキゾチックなデータ転送プロトコルには拡張されませんが、それに基づいてさまざまなインスタントメッセンジャーまたはコンピューターゲームが機能します。 つまり 不要な情報がデバイスに侵入するという一定のリスクが依然として保存されています。 確かに、後者は主に青年が使用するソーシャルネットワークのインスタントメッセンジャーには適用されません-対応するページは形態学的フィルターによって処理されるため、冒proを含むメッセージは子供のデバイスに表示されません。



コンピューターゲームに関しては、現時点では、このサービスで完全にブロックするか、ポップアップ、広告バナーなどのみを遮断することができます。



MegaFonによると、 サービスの更新バージョンのリリース以来、その人気は継続的に成長しています。 平均して、新しいサービスが既に稼働している地域では、加入者の数は65％増加しています。 当然、MegaFonはさらにサービスを開発する予定です。



特に、個人アカウントを介して、より詳細なサービス設定のためのツールを親に提供する予定です（既存のデータベースに加えて、独自のサイトのブラックリストとホワイトリストを作成するまで）。



-お客様が高速モバイルインターネットを受信するだけでなく、セキュリティの保証を受けることも重要です。 特に若いユーザーの場合。 今日では、不要なサイトへのアクセスを制限するだけでは不十分であり、データフィルタリングには根本的に異なるアプローチが必要です。 そのため、「子供のインターネット」サービスを更新しました。このサービスでは、サイトからの不要な情報のみをブロックし、安全で便利なコンテンツを子供に公開したままにできます。
-Artyom Vilman、MegaFon、新サービス担当ディレクター。







上記では、いわゆる「厳格な」保護について話しました。設定を掘り下げ、自分が望むものを手に入れるのを妨げるものをオフにできる若者を対象としています。 より安全な保護（たとえば、子供とインターネットサーフィンを共有しているときに誤って望ましくないリソースにアクセスすること）のために、サードパーティのソリューションを使用することはできませんが、YandexまたはYandex.DNSのSafe Google SearchまたはFamily Searchの検索エンジンの組み込み機能に制限されます。



安全なGoogle検索（Yandexファミリー検索など）を使用すると、アダルトコンテンツが表示されないように結果をカスタマイズできます。 一般的に、サービスは似ていますが、検索エンジンの設定はわずかに異なりますが、特に、Googleでは、セーフモードの不正な無効化から身を守ることができます。



Yandex.DNSを使用すると、標準のISP DNSサーバーを、基本（ブロックなし）、安全（ブロック詐欺およびウイルスあり）、および家族（アダルトコンテンツの追加ブロック付き）の3つのモードで動作する代替サーバーに置き換えることができます。 実際、DNSを置き換えると、リソースのインデックス作成時にYandexが絶えず更新するサイトのブラックリストに基づいてフィルターを実装できます。



結論の代わりに、私は青少年のインターネット活動を制限する利用可能な手段に対する親の関心の高まりに続き、これらのソリューションはますます完璧になりつつあり、MegaFonによって提示された更新されたサービスはこれをさらに証明していることに注意したいです。 このサービスは、ユーザーのスマートフォンの設定や場所に関係なく機能します。 さらに、フィルタリングは暗号化されたトラフィックに対処できます。 ただし、現代のテクノロジーでは、両親自身と青少年の両方の意識を置き換えることはできません。青少年は、インターネットを日常的に使用する際に直面する脅威を知る必要があります。
