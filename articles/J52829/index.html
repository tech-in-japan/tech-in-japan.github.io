ロシアのスパートOpera Mini-1年で157％









Opera Mobileレポートは、35の主要国と中東のモバイルインターネットデータを提供します



ノルウェー、オスロ-2009年2月25日



事実のみ



-ロシアは、最初のモバイルレポートで特定されたリーダーシップの地位を維持しています。 過去1年間で、ロシアのOpera Miniユーザーの数は157％増加しました。 2009年1月、各ユーザーは平均416ページを閲覧しました。 モバイルインターネットのステータスに関する完全なレポートは、 www.opera.com / smw （英語）で入手できます。



-2009年1月の世界のOpera Miniユーザーの総数は12.1％増加し、初めて2,000万人のユニークユーザーのマークを超えました。 月間閲覧ページ数はさらに顕著に増加し、前月と比較して18％増加し、76億Webページに達しました。 合計で、2008年12月から、Opera Miniユーザーは世界中のモバイルオペレーター向けに122テラバイトを超えるデータを生成しました。



-Opera Miniブラウザは平均してデータを最大90％圧縮します。 このデータが非圧縮形式で送信された場合、総トラフィックは12億Mbになります。 このブラウザの歴史の中で初めて、Opera Miniサーバーは1か月でOpera Miniユーザー向けに1ペタバイト以上のデータを処理しました。



-2008年1月から2009年1月までのOpera Miniユーザー数の最も爆発的な増加はアルメニアで記録され、この数字は2800％以上でした。 2位はナイジェリア-1854％です。 その他の注目すべき結果には、エジプト（1391％）、フィリピン（570％）、カザフスタン（419％）が含まれます。



-アルメニアは、その月にOpera Miniユーザーが閲覧したページ数でもリードしています。 2009年1月、この指標はアルメニアのユーザーあたり平均669ページでした。



-中東のOpera Miniユーザー数のトップ10は、イラン、イスラエル、サウジアラビア、シリア、ヨルダン、アラブ首長国連邦、オマーン、クウェート、レバノン、イエメンです。



マスタデータを報告する



月次レポートでは、Opera Miniのユーザー数で35か国をリードしているため、トップ10には、年間ユーザー数の増加率が最も異なる国が含まれることがあります。 たとえば、インドネシアは年間312％の増加（GP）を示しましたが、ユーザーの総数では2位に過ぎません。

このモバイルレポートには、中東に関するデータも含まれています。 さまざまな国の人々を和解させる最も活発な平和維持部隊は、ソーシャルネットワークFacebookでした。これは、中東の10か国中9か国で最も訪問されたトップ10のウェブサイトに含まれ、Opera Miniユーザーの数をリードしています。 評価は次のように配布されました：イラン（9位）、イスラエル（1位）、サウジアラビア（10位）、シリア（3位）、ヨルダン（3位）、アラブ首長国連邦（3位）、オマーン（9位）、クウェート（4位）とレバノン（1位）。 このレポートには、Opera Miniユーザー数別に最も人気のあるモバイルデバイスに関するデータと、主要国で最も人気のある10のWebサイトのリストも含まれています。

私たちの意見



「過去1年間のモバイルインターネット利用の成長に関する統計を見ると、モバイルデバイスが世界の多くの地域でグローバルネットワークを見る主な手段になっていることがわかります」とOperaのCEOであるJon von Techner氏は述べています。 「この状況での発展途上国は、パソコンは携帯電話よりも希少なデバイスであるという事実につながる可能性がありますが、アメリカ、イギリス、ドイツなどの先進国の具体的な成長率は、全世界が動いていることを示唆していますこの方向に。 「これらの統計は、グローバルネットワークに接続できるすべてのデバイスに高品質のWebブラウザーを装備することが今日どれほど重要であるかを再確認しています。」



Opera Miniブラウザについて



すでに2000万人を超えるユーザーが、Opera Miniがインターネット体験をいかに劇的に変えることができるかを知っています。 このブラウザは、Webページをできるだけ早く携帯電話に配信するだけでなく、通常のコンピューターと同じように表示します。 また、Opera MiniにはOpera Linkが含まれています。これは、すべてのWebブラウザ間でブックマーク、スピードダイヤル、パーソナルパネルアイテムを同期できる無料のサービスです。 Opera Miniブラウザーは、 www.opera.comから完全に無料でダウンロードできます（モバイルオペレーターのデータ転送サービスを有効にする必要がある場合があります）。



Opera Software ASAについて



Opera Software ASAは、PC、携帯電話、その他のインターネットデバイスでWebページを表示するための新しい可能性を切り開きました。 Operaによって開発されたクロスプラットフォームブラウザーテクノロジーは、その機能、標準への準拠、小さなサイズで知られ、ユーザーにより高速で安全で動的なインターネットアクセスを提供します。 Opera Softwareのメインオフィスはノルウェーのオスロにあり、世界中にオフィスがあります。 同社は、OPERA証券コードでオスロ証券取引所に上場しています。 詳細については、 www.opera.comをご覧ください。
