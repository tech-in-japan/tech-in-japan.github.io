Odroid U2およびOdroid X2ミニチュアボード

少し前（2012年2月13日まで）、ラズベリーパイとサイズが似ているが、通常のコンピューターとして使用できる詰め物を備えたミニチュアOdroid U2およびX2ボードの出荷が開始されました。 ためらうことなく、Odroid U2とOdroid X2の2つのボードを注文しました。



カットの下のボードの小さな概要。





Odroid U2ボード -48 x 52 mm（クレジットカード未満）のサイズのボードには、4コアのSamsung Exynos 4412 Prime 1.7GHzプロセッサ（ARM Cortex-A9）、Mali-400ビデオアクセラレータ（クアッドコア、440MHz、1080pビデオデコードのサポート）および2 GBのRAM。 コネクタには、3.5 mmオーディオ出力、ボードにインストールされたデジタルマイク、10/100 Mbpsイーサネット、2つのUSB 2.0ポート、1つのmicroUSBポート、UARTポート、microHDMIポート（1920x1080）、microSDカードスロット、モジュール用のeMMCソケットがあります。 ソフトウェアの充填として、Android 4.0.4および4.1.x、Ubuntu 12.11（正式にはOpenSuSE 12.2、Fedora 18ではありません）が提供されています。 デバイスのコストは89ドルです。



Odroid X2ボード -U2の約1.5倍。 寸法90 x 94 mm、4コアSamsung Exynos 4412 1.7GHzプロセッサ（ARM Cortex-A9）、Mali-400ビデオアクセラレータ（クアッドコア、440MHz、1080pビデオデコードのサポート）および2 GBのRAMを搭載。 コネクタのうち、3.5 mmオーディオ出力+マイク入力、10/100 Mbpsイーサネット、2つのUSB 2.0ホストポートと4つのUSB ADB /マスストレージポート、1つのmicroUSBポート、UARTポート、microHDMIポート（1920x1080）、インターフェイス付き50ピンIOポートが記載されていますLCD / I2C / UART / SPI / ADC / GPIO、カメラ用のMIPI-CAMポート（MIPI-CSI 2レーン）、フルサイズのSDカード用のスロット、eMMCモジュール用のソケット。 ソフトウェアの充填として、Android 4.0.4および4.1.x、Ubuntu 12.11（正式にはOpenSuSE 12.2、Fedora 18ではありません）が提供されています。 デバイスのコストは135ドルです。



すべての仕様と回路基板は公開されており、 公式ウェブサイトで入手できます。 フォーラムでは、特定のOSのインストールに関する多くの有用な情報を見つけることができます。



そのため、2013年2月14日に、次のセットを注文しました。

また、OdroidボードはmicroHDMIコネクタを使用しているため、対応するケーブルを事前に購入しました。価格は200 rです。

eMMCモジュールまたはmicroSDカードからOSを起動できるという事実も喜ばしいことです。OdroidX2には起動デバイスを選択するジャンパーがあり、Odroid U2は起動デバイスを自動的に選択します（eMMCの優先順位）。



すべてを収集し、ワイヤレスキーボードとマウスを接続し、電源を入れ、数秒待ってから、使い慣れたAndroid画面が表示されます。





一連のアプリケーションはかなり少ない



キットにはGoogleアプリケーションはありませんが、ビデオプレーヤー、ブラウザ、ターミナルがあります。これで十分です。



システムおよびメモリ情報：









まず、ターミナルに移動してdmesgを確認します（ダウンロードログ全体を表示しません。スクリーンショットが多すぎます）。





最後に、ビデオプレーヤーを起動し、ホームコンピューターに接続してビデオを再生します。

すぐにもっと頑張りたいと思い、Tronを3D HDで立ち上げました。

スローダウンやグリッチは気づかれず、写真はスムーズかつ均等に進み、音は遅延もジャークもなくクリアでした。 プロセッサの処理能力はHDビデオをデコードするのに十分であり、非常に満足しています。
