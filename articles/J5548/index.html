Web 3.0-未来はロボットに属します

Web 3.0-ロボットの王国があり、同じ滑らかな文字を持つあなたの対話者のいずれかが機械であることが判明する問題をすぐに認識すると、 レオニード・カガノフの公式ブログの公式投稿によると、すぐにこれに対処する方法を学びます。



レオによると、彼はロボットジャックの創始者になり、すでにある程度の人工知能を繁殖させることができました。



「私が楽しみのために作った別のロボット： k-31415926 。 彼は非常に愚かです。彼らは「ロボット」という言葉を含む投稿を探しており、「さて、ここで私はロボットです。」というコメントを思い付きます。 それで何？ そして、私はあなたよりもどうですか？ リスのカビは遅かれ早かれ、未来は私たちのロボットです！「このオタクはウクライナの投稿に絶えず反応しますが、非常に面白いですね」 人々はこのロボット全体に非常に陽気に、そして慈悲深く反応します。 ロボットに反応して長いカートを書き始める賢い人たちのかなりの割合がありますが、ロボットに魂、感情、愛があるわけではないので、彼らは愚かな鉄片であると言います（！） 。 ロボットはそのような警告を聞いてうれしいです;）））」とレオは言います。



「何のためにそれが必要なのか」という質問に対して、彼は詩的に答えました。



ここに彼は-将来のWeb 3.0の現代のビルダーです:)
