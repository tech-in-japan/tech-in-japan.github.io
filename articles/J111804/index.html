ICANNが.jobsゾーンを開きます

今年、.jobsドメインゾーンの管理者であるEmploy Media LLCと、空席のアナウンスと履歴書（Employment Web Sitesの関連付け）の発行に携わっている米国最大のインターネットサイトの協会との対立は終了しました。 この関連付けには、たとえば、monster.comリソースが含まれます。これは、世界の多くの国で仕事を見つけたり、従業員を見つけたりするのに役立ちます。



競合の理由は単純です-.jobsドメインゾーンの管理者は、すべてのユーザーが.jobsゾーンにドメインを登録できるようにというリクエストでICANNに申請しました。 実際には、2006年に登録された.jobsゾーンは、無料の求人情報の公表で独自のWebサイトを作成したい企業にのみ開かれていました。 たとえば、asda.jobsは、空室がasdaスーパーマーケットチェーン（英国）で公開されているサイトです。



2010年、Employ Media LLCは、このゾーンのドメインの登録を、たとえば自分の雇用サイトを開設したい第三者や企業に許可することを決定しました。 Employ Media LLCのリーダーシップの提案により、この方法で、たとえばChicago.jobsなど、世界中のあらゆる都市の雇用サイトを作成することが可能になり、このゾーンに数十万の新しいドメインを登録することは、ゾーン管理者にとって有益です。



同時に、最大の求人サイト協会である雇用ウェブサイト協会は、この提案の導入に反対しました。 その理由は明らかです。.jobsゾーンが開かれた場合、協会のサイトと強力に競合する多くの新しい雇用サイトが表示されます。



最終的に、ICANNは.jobsゾーンの開設を承認しました。 それは2010年の終わりに起こりました。 今年、Universe.jobsと呼ばれるDirectEmployers Associationの主要なプロジェクトが開始されました。 プロジェクトの一環として、数百（および場合によっては数千）のドメインが.jobsゾーンに登録され、名前には国や都市の名前が含まれています。 プロジェクトのメインページにあるインタラクティブマップで、任意の地域と、この地域の求人を含む求人サイトの1つ（usa.jobsやrus.jobsなど）を選択できます。



このプロジェクトは、.jobsゾーンでのオープン登録の可能性を利用した最初の新しいプロジェクトの1つであることは明らかです。



.jobsゾーンにドメインを登録するコストは、年間119.99ドル（GoDaddy）であることに注意してください。
