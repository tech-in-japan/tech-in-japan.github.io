ロシア初のAndroid OSベースのコミュニケーター

Highscreen PP5420は、GoogleのAndroidオペレーティングシステムで実行される、薄くてエレガントなコミュニケーターであり、Googleサービスでの使用に最適化されています。 このモデルはロシア専用に設計されており、他の国では販売されていません。 このモデルのユニークな機能はセカンドタッチOLEDディスプレイで、ナビゲーションキーとジョイスティックを正常に置き換えます。





オペレーティングシステム： Google Android OS 1.5



プロセッサー： 528 MHzのQualcomm®MSM7201A™

RAM： 128 MB

ROM： 256 MB

接続インターフェース： USB 2.0

寸法 （高さx 幅 x 奥行き ）： 114x53.5x13 mm

重さ： 120g



追加機能： モーションセンサーと空間内の位置（Gセンサー）

Bluetooth： バージョン2.0 + EDR

WiFi： 内蔵802.11 b / g Wi-Fi

スクリーンタイプ： カラー半透過型TFT LCD

画面： 3.0 "タッチスクリーン、240 x 400（WQVGA）、262K色+ 1.15" OLED 128 x 96

バッテリー： 取り外し可能なLi-Pol 1080 mAhバッテリー

通話時間： 4時間

待機時間： 200時間

GSM： GSM：850/900/1800/1900; WCDMA / HSDPA：900/2100

GPRS： GPRSクラス10（EDGE）; HSDPA 7.2 Mbpsダウンロード/ HSUPA 2.0 Mbpsアップロード;

内蔵デジタルカメラ： 3メガピクセル、デジタルズーム、写真とビデオの撮影

GPS： アシストGPSおよびE-GPSを備えた内蔵GPS

GPSチップセット： Qualcomm + GPSOne XTRA

追加プログラム： Googleマップ、Google検索、YouTube、オーディオ/ビデオプレーヤー
