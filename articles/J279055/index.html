West Game Development Forum：作成、共有、改善

Wargaming社は、2016年4月22〜23日にサンクトペテルブルクで開催される、West Game Development Forum 2016のこの春最大のカンファレンスに参加するよう招待しています。



West Game Development Forumは、ゲーム開発とそれに関連するすべてに特化した主要な国際会議です。 サンクトペテルブルクで開催され、gamedevの退役軍人と若い専門家の両方から、さまざまな国や企業のプロの開発者を集めます。







この会議の目的は、経験と意見を交換するための理想的なプラットフォームを構築し、さまざまな世界慣行の例を使用して、現代のゲーム業界の多様性を提示することです。 この分野の発展の記録的なペースの状況では、現在の主な傾向をより正確に判断し、さらなるコースを概説するのに役立つのは、専門コミュニティの統合です。

オーディエンスには、ゲームデザイナー、プログラマー、デザイナー、アーティスト、サウンドエンジニア、その他のゲーム開発者が含まれます。 game devで専門的に成長および開発したいすべての開発者。

このプログラムには、現代のゲーム開発における最も差し迫った問題に関する多数の講義、セミナー、円卓会議が含まれています。 世界クラスの専門家との議論に参加してください！



登録して、イベントwww.wgdevforum.netの公式ウェブサイトで詳細を確認できます。
