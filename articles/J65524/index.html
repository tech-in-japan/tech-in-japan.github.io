ラジコンカーに乗りたいですか？





2008年、ニコンはMedia Port UP300xと呼ばれるユニークなポータブルメディアプレーヤーのリリースを発表しました。 ノベルティは、ビデオを見るためのディスプレイを備えた未来的な外観のヘッドフォンです











実際、Media Port UPは、頭に直接装着できるヘッドフォンを備えたポータブルミニコンピューターまたはメディアプレーヤーです。 新しいアイテムのディスプレイのサイズは非常に小さい（0.44）インチですが、同社によれば、最新の光学機器は3メートルの距離から50インチのスクリーンを見る効果を生み出します。 この画面では、MPEG-1、MPEG-2、およびWMV形式のビデオファイルを、640 x 480ピクセルの解像度で表示できます。 さらに、Wi-Fi 802.11gのサポートにより、デバイスの所有者はWebサイトにアクセスしたり、音楽を購入したりできます。







デバイスには、コンテンツを保存するための4または8ギガバイトの内部メモリがあります。 ノベルティは、2つの標準的な単三電池で動作します。 推定によると、これは120分間のビデオまたは270分間の音楽鑑賞に十分です。



次に、上記のすべての意味を説明します。



ミニチュアワイヤレスWiFiビデオカメラが無線制御車モデル（たとえば、 Mini-Z ）にインストールされ、メディアポートUP300xに信号を送信します。







その結果、高速道路に沿って走るMini-Z内に完全に存在する効果が得られます。







あなたは日本のオンラインストアでヘッドフォン自体を購入することができます... Media Port UP300xの価格は配達を除いて745ドルです。 また、Mini-Z用のMedia Port UP300xに基づくソリューションはまだ購入できません。これは概念です。 東京クラブ京商表参道のMini-Zトラックでのみ実行されていますが、8月には誰でもテストできます。



一方、そのような決定が単一のボックスで販売されるまで待つ必要はありません。 ビデオグラスとワイヤレスミニチュアビデオカメラはモスクワで無料で販売されています...ソリューションははるかに安くなりますが、効果は同じです
