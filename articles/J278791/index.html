MicrosoftによるSQL ServerのLinuxへの移植

Microsoftの主要なリレーショナルデータベース製品であるSQL Serverは 、 Linuxで初期のプライベートプレビューバージョンとして利用できるようになりました。 最終バージョンのリリースは、2017年半ばに予定されています。



今日まで、SQL ServerはWindowsのみで機能していました。 しかし、Microsoftのクラウドおよびエンタープライズテクノロジー開発チームのエグゼクティブバイスプレジデントであるスコットガスリーが今日書いたように、同社は製品をLinuxに移植する時だと判断しました。



「Linux上のSQL Serverは、データソリューションの柔軟性をユーザーに提供します」とGuthrie氏は述べています。 「重要な指標を備えた製品、総所有コストの業界リーダー、クラス最高のセキュリティ、ストレッチデータベースなどのハイブリッドクラウドイノベーションにより、ユーザーは自分の領域と地域の両方でデータにアクセスできます。クラウド、いつでも必要なときに低価格で-すべてが製品に組み込まれています。」



マイクロソフトは、製品が一般的に利用可能になると、Linuxの下でSQL Serverへのアクセスをオープンする予定です。



本日の発表（2016年3月7日付け）は、 ニューヨークで開催される予定のSQL Server 2016専用のMicrosoftイベントの3日前に発表されました。 ガスリーは、Microsoft がSQL Server 2016に登場するいくつかの新機能の導入を計画していることを発表しました。これには、インメモリデータベースでの作業の改善、R、データウェアハウジングのサポートの改善、ビジネスインテリジェンスのための新しいモバイルアプリケーションが含まれます。



Microsoftは、SQL Server 2016を使用して、Azureクラウド内の約140万のSQLデータベースと連携することを発表しました。 SQL Server 2016は、オープンプレビューバージョンとして利用可能になりましたが、今年後半に利用可能になる予定です。



少し前まで、今日の発表は信じられないほどでした。 しかし、ここ数年、特に新しいCEOのSatya Nadellaとともに、同社はますます多くのサービスをオープンし、オープンソース製品に重点を置いています。



本日の発表は、ハイブリッド開発に対するマイクロソフトの焦点にも当てはまります。 マイクロソフトはすでにクラウドでLinuxを使用しており、最近Red Hatとのパートナーシップを発表しました 。 企業がSQL Serverの関連性を確認したい場合、以前はライバルプラットフォームと見なされていたプラットフォームを含め、より多くのプラットフォームでこの製品を宣伝する必要があります。 結局のところ、MySQL、MariaDB、PostgreSQLなどの製品が同様の市場セグメントで競合するのはLinux上です。
