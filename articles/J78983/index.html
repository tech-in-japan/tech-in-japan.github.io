ラボ向けOpera 10.50プレアルファ、公式説明

本日、Labs（Opera Labs）向けにOpera 10.50プレアルファ版をリリースしました。 このプレアルファ版は、Evenesブランチに基づいており、WindowsおよびMac用のビルドと、それに続くUNIX / Linux用のバージョンが含まれています。



ご存知かもしれませんが、通常、開発サイクルのこのような初期段階でユーザーテストを行うことはありません。 それにもかかわらず、デスクトップ用に開発チームで準備されていることを心から賞賛し、フィードバックをお待ちしています。



あなたの多くは、CarakanまたはECMAscript / Javascriptの新しいエンジンを見てほしいと頼みました。そして今日、それと私たちの次のメジャーリリースに含まれることを期待しているいくつかの他の新しいテクノロジーで遊ぶことができます。



いくつかの警告：あなたが今日ダウンロードしているのは、機能的に不完全であり、おそらく開発中の不安定なアセンブリです。 それに注意し、インストールの前にデータのバックアップコピーを作成し、水力発電所で使用しないでください。



新機能





中：



  • キャラカン

    Carakanは、新しいJavascriptエンジンです。 Sun Spiderテストでは、Windows用のFutharkエンジンを搭載したOpera 10.10よりも7倍以上高速です（Macの最適化は間近です）。 Opera Coreブログで、レジスタ、オブジェクトの自動分類、ネイティブコードの生成を使用して、バイトコードに関する詳細を読むことができます。

  • プレスト2.5

    現在、Presto 2.5を使用していますが、これには膨大な数の改善が含まれています。 また、 CSS3の変換と移行 、および永続ストレージなどの他のHTML5の革新のサポートも含まれています。
  • ベガ

    Vegaは新しいグラフィックライブラリです。 現時点では、完全にソフトウェアであり、画面に表示されるすべてを表示します。 Vegaはハードウェアアクセラレーションを使用できますが、Peacekeeperテストの「複雑なグラフィックス」テストからわかるように、まだ必要ではないようです。 （Futuremarks Peacekeeperテストでは、総合スコアに複雑なグラフィックステストの結果が含まれていないことに注意してください。これは2009年には正しくなく、2010年にこの状況が変わらなければ愚かだと考えています。）





外で

  • プラットフォーム統合

    • Windows 7 / Vistaでは、ユーザーインターフェイスがAeroグラス効果を表示できるようにする多くの外部変更とAPIの使用に気付くでしょう。 Windows 7では、Espressoパネル、タブなどへのアクセスを簡素化するために、Aero Peekおよびジャンプリストのサポートも追加しました。 タスクバーから直接。
    • Macの場合、Cocoaでのコードの完全な書き直しにより、統合ツールバー、ネイティブボタンおよびスクロールバー、マルチタッチジェスチャ（3本の指で左または右にスワイプするか、指を離してズームアウトする）およびその他の多くの詳細が追加されました。 Growl通知サポートも追加しました。
  • 「プライベートタブ」と「プライベートウィンドウ」

    [プライベート]タブまたは[プライベート]ウィンドウを開くことができます。これらのウィンドウは、閉じるとすぐにそのすべてを「忘れ」ます。
  • モードレスダイアログ

    ダイアログ（Javascriptアラート、HTTP通知など）がモードレスになり、ページ自体の上部に表示されるようになりました。 これにより、ダイアログが閉じられていないときにタブとウィンドウを切り替えることができます。 同様に、パスワードマネージャーダイアログはページの上部に関連付けられ、新しいページの読み込み中にコンテンツをブロックしません。
  • アドレスと検索文字列

    両方のフィールドの外観と機能が更新されました。 検索を記憶し、履歴からの要素の削除をサポートするようになり、表示される結果がより便利になりました。
