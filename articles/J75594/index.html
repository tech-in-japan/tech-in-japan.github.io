Opera Mobile 10 for Windows Mobile





2009年11月18日、ノルウェー、オスロ

Opera Softwareは、Windows Mobileプラットフォーム用のOpera Mobile 10モバイルブラウザーのベータ版をリリースしました。 Opera Mobile 10はデスクトップブラウザーの機能を備えており、情報を検索したり、Webサービスを使用したり、外出先で文字通りソーシャルネットワークで通信したりできます。



Windows Mobile用のOpera Mobile 10ベータ版のインターフェースは、デスクトップブラウザーOpera 10およびモバイルOpera Mini 5ベータ版のインターフェースに似ています。 Opera Softwareは、すべてのプラットフォームに対して、すべての製品の外観とオプションの統一を最大化することを目指しています。 Windows Mobile用Opera Mobile 10ベータ版には、お気に入りのWebサイト、ブックマーク、パスワードマネージャーにすばやくアクセスするためのExpressパネルが含まれており、Opera Turboデータ圧縮技術をサポートしています。



Opera SoftwareのCEOであるJon von Tetzchner氏は次のようにコメントしています。 -Opera Mobile 10のリリースにより、携帯電話用ブラウザのユーザー要件が大幅に増加することを理解しています。 スマートフォンユーザーには、デスクトップと同様にタスクを処理できる強力なモバイルブラウザーが必要であり、Operaはそれを提供する準備ができています。



Opera Mobile 10 for Windows Mobileの10の重要な革新

1.新しいインターフェース。 Operaは、携帯電話の画面に負担をかけないシンプルで直感的なインターフェースを提供します。



2.エクスプレスパネル。 エクスプレスパネルを使用すると、お気に入りのWebリソースにすばやくアクセスできます。 Expressパネルでサイトをインストールおよび変更します。ユーザーは自由に判断できます。



3.タブ。 リンクをたどると、新しいタブでサイトを開き、異なるタブで開いているWebページ間を簡単に移動できます。



4.パスワードマネージャー。 パスワードを携帯電話に保存して、安全なWebページにすばやくアクセスできるようになりました。 ユーザーがセキュリティの問題に困惑している場合、保存されているすべてのパスワードを簡単かつ迅速に携帯電話のメモリから削除できます。



5.キーボードとタッチスクリーンのサポート。 Opera Mobile 10ベータのインターフェースは、スタイラスを使用せずに電話のキーパッドとタッチスクリーンの両方を使用してインターネットで作業するために最適化されています。



6.画像を保存します。 VKontakteやFacebookなどのソーシャルネットワーク上の友人の写真は、オフラインで保存および表示できます。



7. Opera Turbo。 このテクノロジーを含めることで、Webサイトをより速く開くことができ、ダウンロードされるデータの量、つまりモバイルインターネットのコストが削減されます。



8.互換性。 Opera Mobile 10ベータ版は、Operaデスクトップブラウザーのコアに基づいており、オープンスタンダードに基づいて作成されたWebページの互換性を確保し、ほとんどすべてのWebサイトを携帯電話からアクセス可能にします。



9.便利な視聴。 ページの切り取り、拡大縮小、スムーズなスクロール-携帯電話からウェブを快適に操作するために必要なすべて。



10.無料です！ Windows Mobileプラットフォーム用のOpera Mobile 10ベータブラウザーは、www.opera.com / mobile（コンピューターへのダウンロード用）およびm.opera.com/mobile（携帯電話への直接ダウンロード用）で無料で入手できます。



CIS諸国のユーザー向けのOpera Mobile 10ベータ版には、Yandexのデフォルトの検索パネルが付属しています。

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
