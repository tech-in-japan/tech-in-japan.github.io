ポップコーンタイム：トレントを見るための最も便利なプログラム





「Popcorn Timeは映画の海賊版が非常に優れているので怖い」と、Time Magazine は新しいオープンソースプログラムPopcorn Timeについて書いています。



BitTorrentクライアントの最新バージョンを使用すると、ダウンロードの完了を待たずに、ダウンロード中に映画を直接視聴できるようになることは周知の事実です。 表示はほぼすぐに開始できます。 Popcorn Timeの開発者はこの機能を使用し、さらに世界のトレントトラッカーで利用可能なすべての映画のカタログを作成しました。その結果、主婦でも理解できる非常に便利なプログラムを手に入れました。 カタログから映画を選択し、視聴を開始します。 Netflixの一種の無料の代替品ですが、より完全なカタログがあり、世界のすべての国で機能しています。



数日間、Githubで最も人気のあるプロジェクトのリストのトップにポップコーンタイムがあります。 出版日から24時間以内に、Popcorn Timeインターフェースは6つの言語に翻訳され、 ロシア語を含む数十の言語が追加されました。



技術的には、プログラムは非常に簡単に機能します。 YTS API 、 OpenSubtitles字幕データベース、およびTheMovieDBデータベースを介したトレントリストをメタデータと組み合わせます。



それが公衆の間で人気を得るならば（そして、これは疑うことはほとんどないでしょう）、著作権保持者がそのようなプログラムにどのように対処するかは不明です。 すべてのトレントトラッカーを閉じることはできず、オープンソースプログラムを禁止することもできません。 「私たちは法的主張を期待していません」 とアルゼンチンのリードプロジェクト開発者セバスチャンは言います。 -私たちは何も投稿しませんし、開発者の誰もこれでお金を稼ぎません。 広告、プレミアムアカウント、有料サブスクリプションなどはありません。 これは、学習と共同作業のための単なる実験です。」



「ハリウッドはポップコーンタイムを攻撃するために最善を尽くすと思います」とタイムは書いています。 -しかし、無料で無料のプロジェクトを閉じるのは難しいでしょう。 開発者は、映画の著作権侵害が直感的なインターフェースと使いやすいことを示しています。 映画スタジオがプレミアスケジュールを変更せず、新しい映画のオンライン化を妨げる古いビジネスモデルを放棄しない場合、ポップコーンタイムのようなアプリケーションには非常に注意する必要があります。
