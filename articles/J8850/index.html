Windows Vista-敵か味方か？

Agnitumは、英語の企業ブログのロシア語版を立ち上げています。 近い将来、年の初めから最もホットな投稿が翻訳され、公開されます。



雑誌や新聞を開いてMicrosoft Vistaについて読むことはできないようです。 そして実際、その名前（vista-“ string”、英語）は予言的なものであることが判明しました。Vistaに関する一連の記事全体を観察しています。 マイクロソフトの技術に焦点を合わせたセキュリティ開発者として、以前の投稿で[Agnitum]は 、ビルトインファイアウォールおよびVistaに関連するカーネルパッチ保護の問題について意見を公に表明しました。 今回は、MicrosoftとAgnitumのビジネス上の利益に注意を払うことにしました。それらはどこが異なり、どこで交差するかです。



マイクロソフトマーケティング部門の否定できない勇気を歓迎し、5P-「Five Pi」（マーケティングを研究したことがある人なら誰でも、マーケティング理論の核は4P-「4 Pi」- [Agnitumによると] 5つは元のものとわずかに異なります）。



P1。 価格



完全なVista個人ユーザーライセンスは199ドルです。 以前のバージョンからVistaにアップグレードすると、価格は100ドルに下がります。 当然、オペレーティングシステムのコストはほんの始まりに過ぎません。 ほとんどのユーザーは、コンピューターをアップグレードする（または新しいPCを購入する）必要もあります。 短い期間（3月に終了）に、一部のユーザーは、サプライヤーからいわゆるいわゆる鉄を受け取ることができました。 「高速アップグレード」が、今では不可能です。 もちろん、Microsoftと同様に、Agnitumは顧客を維持する（そして顧客を満足させる）ことを望んでいるため、更新プログラムの継続的なサブスクリプションを提供しています。 Microsoftとは異なり、ユーザーに宣言された価値以上の費用を強制しません。



P2。 広告（ピッチ）



実績のあるWindows XPから離れてVistaにアップグレードする時が来たかどうかをどのように判断しますか？ 現在、市場にあるVista互換製品の数が少ない場合-少なくとも完全にテストされるVista互換製品-このプラットフォーム専用の新しいプログラムに誘惑されることはほとんどありません。 ただし、前述したように、メディアは絶えずVistaについて書いており、これはすべての人が持つべきであり、持っているだけでなく、持っているべき技術であるという感覚がユーザーの心に広がっています。 ソフトウェアを購入する計画に影響を与えるように設計された他のチャネルには、他にも多くのメッセージがあります。 疑いもなく、Microsoftのマーケティングマシンは評価される力です。 非常に少数のソフトウェア会社（およびAgnitumはその中にありません）は、市場シェアを維持または拡大するために多額の資金を費やす余裕がありません。



P3。 圧力



もちろん、Vistaが登場するとすぐに、ソフトウェア開発者は自社製品がVistaと互換性を持つようになる時期について質問をしました。 VistaがWindowsの最も安全なバージョンであると主張しているため、このプレッシャーはセキュリティ開発者である私たちに特に感じられます。 これは、サードパーティの開発者が長期的に自社製品とVistaの完全な互換性を保証することがより困難であることを意味します。 そのため、Vista互換製品のテストとデバッグに多くの時間を費やしています。現在は動作するものをリリースしたくはありませんが、Vistaセキュリティホールの修正の外観での動作は停止します。最初のサービスパックのリリース）。



P4。 保護



Secuniaレポートなど、多くのサードパーティの脆弱性追跡リソースとサービスは、Vistaの多くの未解決の脆弱性と、組み込みファイアウォールおよびその他の「保護」ツールの問題を既に発表しています。 VistaはXPよりはるかに脆弱ではありませんが、オペレーティングシステムは多くの異なるタスクを実行するように設計されていることを覚えておく必要があります。 システムセキュリティに関しては、ほとんどの場合、「平均よりも優れている」よりも「クラスで最高」であることを好みます。



P5。 移植



移植は通常オペレーティングシステム間のコード転送を示すために使用されるため、この用語を少しごまかしました。ユーザーをOSから別のOSに移動することについてお話したいと思います。 しかし、英語の文字「pi」（「p」）で始まるこのような移行の同義語を思い付くことができませんでした...



マイクロソフトのCEOであるスティーブバルマーは次のように述べています。「Vistaに対する熱意に本当に興奮しています。 しかし、私が見た2008年度のWindows Vistaの収益予測の一部は大胆すぎることを理解する必要があります。 サードパーティのビューがこのビューをサポートしているようです。



関連するアンケートからの抜粋を以下に示します。



ニューメキシコ州アルバカーキのセキュリティ会社VanDyke Softwareから委託されたフロリダ州ボカラトンのAmplitude Researchの調査によると、半数以上が、Vistaが出たときに、Vistaに切り替える計画はないと言っていました。マイクロソフトがオペレーティングシステムで行うことを約束したすべてのセキュリティ改善のために。」 （ SearchCIO.com ）



ガートナーの調査によると 、「Vistaは2007年末までにコンピューターの10％未満にインストールされ、2008年には29.3％、2009年には50％、2010年末には67.7％に増加します。」



また、ユーザー間の移行の意図に関する独自の調査を行い、Agnitum Newsの最新号でそれらを公開しました。それらは他と大差ありません。



そして、これはすべて私たちに何を伝えますか？ これはすべて言う：

マイクロソフトは非常に効果的なマーケティングの専門家です

ユーザーはより安全なオペレーティングシステムの概念を歓迎しますが、多くの人は必要な新しいハードウェアにお金を使うか古いハードウェアをアップグレードするために確認を待っています。

ユーザーが使用するアプリケーションを選択してから、これらのアプリケーションを実行するプラットフォームを選択するため、Vista互換バージョンのソフトウェアを急いで作成しないでください。



これを念頭に置いて、Vista向けのOutpost Firewall ProとOutpost Security Suite Proを、ラボと専任のベータテスターチームの両方で完全にテストおよびデバッグした後、今年後半にリリースします。 （ベータテスターチームに参加するには、 ここをクリックしてください ）。
