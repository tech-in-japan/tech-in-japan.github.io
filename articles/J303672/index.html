科学者コミュニケーションツールからグローバルスタンダードへ：電子メールの簡単な歴史





電子メールはコミュニケーションの世界における革命であると同時に、現代の労働者にとって大きな負担です。 写真：ゲッティイメージズエージェンシー



私たちのブログでは、電子メールが決して死ぬことはなく 、通常のユーザーや企業にとって常に優れたコミュニケーション手段であり続ける理由について既に説明しました。 電子メール現象の歴史を詳しく見てみましょう。



2016年3月5日に、電子メールアドレスに「@」（アットマーク）記号を使用することを提案したレイトムリンソンは死亡しましたが、電子メッセージがインターネットおよびメールボックスに入力できるようにする彼の発明は存続します。 この記事では、トムリンソンがどのように始まり、過去半世紀にわたる電子メールの進化について説明します。



最初のメール-1965







1960年代、最初の電子メールが送信されたとき、ロール紙にパンチテープが付いていないコンピューターを想像することは不可能でした。 写真家：H。アームストロングロバーツ/ ClassicStock / Corbis



最新の電子メールの最初の類似物は、1965年にマサチューセッツ工科大学の科学者によってタイムシェアリングシステムの実装プロジェクトの一環として送信されました。 このようなシステムにより、リモート端末で作業する複数のユーザーが中央ディスク上のファイルやメッセージにアクセスできました。



トムリンソンと@記号-1971







文字通り古い「@」記号に新しい命を吹き込んだ男。 写真：パンフレット



あるバージョンによると、アメリカのプログラマー、トムリンソンは、米国国防総省（DARPA）の高度な研究開発局のために、Arpanetネットワーク（インターネットの前身）を介してコンピューター間で電子メッセージを送信する方法を開発しました。 次に、「@」記号を使用することを提案しました。これにより、特定のマシンの特定のユーザーにメッセージを送信できます。



最初のメール標準-1973







ロボットを戦場に送る前に、DARPAはインターネットと電子メールを流通させます。 写真：HO / AFP /ゲッティイメージズ



電子メールの最初の標準は1973年にDARPAで提案され、1977年までに「whom」と「to」のフィールドを定義するArpanetネットワーク上の標準がすでにあり、受信した電子メールに自動的に返信したり、住所



女王は彼女の最初のメールを送信します-1976







女王が、電子メールのおかげでお気に入りの郵便切手が人気を失うことを知っていた場合、彼女はその手紙を送るでしょうか？ 写真：マーティン・キーン/ PA



エリザベス女王2世は、他の国家元首より先に、Arpanetネットワーク経由で電子メールを送信します。



Eric Sc​​hmidt Designs BerkNet-1978







Googleでの作業を開始する前に、Schmidtはシリアル接続を介したメッセージング用の最初のイントラネットシステムの1つを開発しました。 写真：スコット・オルソン/ゲッティイメージズ



その後、Googleの取締役会を率い、Gmailの実装を監督したEric Sc​​hmidtは、1978年に修士論文の一部としてBerkley Networkを作成しました。 このサービスを使用すると、メッセージを連続して送信できます。



プログラムEMAIL-1979が開発されました



14歳のとき、Shiva Ayyaduraiは、ニュージャージー州立医歯学大学向けに「EMAIL」というプログラムを作成し、大学内でメッセージを転送できるようにしました。 1982年に、Ayadurayは「電子メール」という用語の著作権を登録しましたが、この単語の由来は疑問のままです。



Microsoft Mailの登場-1988







落ち着いて、私はこの電子メールの誇大広告がすぐに落ち着くと確信しています。 写真：ルー・デマテイス/ロイター



Microsoft Mailの最初のバージョンは1988年に発売されました。 このプログラムはMac OSオペレーティングシステムでサポートされており、AppleTalkネットワークユーザーが互いにメッセージを送信できるようにしました。 1991年に、DOSやWindowsを含む他のプラットフォーム用に2番目のバージョンが発売されました。 これは、その後のMicrosoftメールシステム（OutlookおよびExchange）の基盤として機能しました。



CompuServeが最初のインターネットメールサービスを開始-1989







CompuServeは、顧客に電子メール交換サービスを提供した最初のインターネットサービスプロバイダーの1つです。 写真：ニール・ローロン/ロイター



CompuServeは、ダイヤルアップインターネット接続を提供する最初のオンラインサービスであり、独自の電子メールサービスにより、インターネットユーザーが相互に電子メールを送信できるようになりました。 1997年、CompuServeはAOLに買収されました。



ロータスノーツを起動-1989







ロータスノーツのおかげで、数百万人の従業員が電子メールを送信できましたが、1989年には電子メールは異なって見えました。 写真：ガーディアンのためのロジャートゥース



この製品は、1989年にアメリカの企業Lotus Developmentによって最初にリリースされ、1995年にIBMに買収されました。



スパムの誕生-1990







スパムの何が問題になっていますか？ 写真：アラミー



スパムの歴史は1978年に始まり、Arpanetネットワークで初めて400件のスパムメッセージが送信されましたが、世界中の何百万人ものユーザーがインターネットを利用できるようになった90年代になって、スパムは電子掲示板をカバーし、後にすべてのメールボックスをカバーしました。



1994年4月、フェニックスの2人の弁護士Laurence CarterとMartha Siegelがこの方法で新しいクライアントを引き付けることを決定したときに、 最初のビジネススパムが記録されました。 後に、夫婦はこれについて本を書きました。



投資-1992







添付ファイルは1992年に登場し、Sobig Fを含むコンピューターウイルスの別のキャリアになりました。Photo：Roger Tus for the Guardian



ファイルを電子メールに添付する機能は、MIMEプロトコル（多目的インターネットメール拡張機能-インターネットメールの多目的拡張機能）のリリース後に登場しました。 当時、メールボックスの容量は制限されていたため、ユーザーは新しいメッセージを受信するためのスペースを空けるために、「重い」添付ファイル付きのメッセージを削除する必要がありました。



OutlookとAol-1993







ほとんどすべてのアメリカの家にあった伝説のAOL CD。 写真：ガーディアンのデイビッド・シリトー



Microsoft Outlookの最初のバージョンは、1993年にExchange Server 5.5の一部としてリリースされました。 同時に、アメリカのインターネットサービスプロバイダーであるAOLとDelphiはメールシステムを統合し、現代の過負荷のメールシステムへの道を開いています。



Hotmailの発売-1996







Microsoftが所有するHotmailは、インターネットで利用できる最初の人気のあるメールサービスの1つでした。 写真：ガーディアンのショーン・スミス



1996年、Sabir BhatiaとJack Smithが開発した、HoTMaiLと呼ばれる最も人気のあるメールサービスの1つが開始されました。 特定のインターネットサービスプロバイダーに結び付けられておらず、HTML形式の電子メールに新しい形式を採用しているという事実、つまりブランドのスタイルによって区別されました。



Hotmailは1997年にMicrosoftに買収され、「MSN Hotmail」、「Windows Live Hotmail」に名前が変更された直後に2013年にOutlook.comに置き換えられました。



Yahoo Mailの発売-1997







Yahoo Mailは9年の歴史の中で複数回アップグレードされています。 写真：ヤフー



Hotmailの開始から1年後の1997年に、YahooはFour11からRocketMailサービスを購入し、それに基づいてYahoo Mailサービスを開始しました。



「あなたへの手紙」とあなただけではありません-1998







トム・ハンクスとメグ・ライアンが主役のロマンティックなコメディ「You Letter」のショット。 写真：ワーナーブラザーズ



ほとんどのユーザーにとって、新しい電子メールの受信は有名なAOLサウンド通知に強く結び付けられています。AOLサウンド通知は、1988年のロマンティックコメディ「レターズトゥユー 」の基礎となり、トムハンクスとメグライアンが主役を務めました。



90年代の終わりまでに、スパムは深刻な問題になりました-ますます多くのマーケティング担当者が、情報を広め、メールボックスを詰まらせるこの事実上無料の手段を使用し始めました。 1998年、「スパム」という用語がOxford Dictionaryに含まれました。



2002年に、欧州連合は電子通信機密性指令を発行しました。これは、スパムを規制するセクションの1つです。 その瞬間から、受信者の事前の同意なしにマーケティング目的で電子メールを実行することは禁じられていました。



2004年に米国で同様の法律が可決されましたが、スパムとの戦いでは効果がないことが判明しました。



Gmailのリリース-2004







GmailのメールサービスはもともとGooglemailと呼ばれていました。 写真：ディーン・マレー/レックスの特徴



2001年にPaul Buchheitによって開発された人気のあるGmailメールサービスは、もともとGoogleの従業員による内部使用のみを目的としていました。 プロジェクトの開発は、2004年にクローズドベータテストが開始されるまで秘密にされました。 2007年になって初めて、このサービスはすべての人が利用できるようになり、2009年にはベータ版のステータスを離れました。

スパムとの戦いの始まり-2005







2000年代初期から、メールプロトコルはスパムと闘い始めました。 写真：Ian Waldy / Getty Images



送信者識別技術を使用してスパムを削減する最初の電子メール標準は、開発が開始されてから5年後に公開されました。 Sender Policy Frameworkプロトコルは、その後多くのプログラムでスパムをブロックするために使用されました。 さらに、DomainKeys Identified Mail（DKIM）標準がリリースされました。これは、いくつかの既存のフィッシング対策とスパム対策を組み合わせたものです。



ユーザーがモバイルからのメールの表示を開始-2007







スティーブ・ジョブズは、手紙の絵が付いたiPhoneのアイコンが何百もの未読メッセージの象徴になるとは想像できませんでした。 写真：ポール・サクマ/ AP



Appleは2007年に最初のiPhoneを発売し、一般の人々が携帯電話からメールを閲覧できるようにしました。 これまで、カスタムスマートフォンのメールサポートは一般的に制限されていました。 同時に、RIMのBlackBerryビジネススマートフォンの所有者は、2003年に電子メールで仕事をする機会を得ました。



シーメール-2015







メールの負荷下。 写真：LifeStyleKB / Alamy / Alamy



存在の初期段階では、電子メールは内部通信にのみ使用されていたという事実にもかかわらず、今日では電子メールは私たちの日常生活の不可欠な部分となっています。 研究機関Radicati Groupが提供したデータによると、2015年には世界中に約44億のメールアドレスがあり、そこから毎日約205億のメールが送信されました。



2019年末までに、この数は2,460億に増加すると予測されています。
