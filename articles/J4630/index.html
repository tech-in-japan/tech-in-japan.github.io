Yahooは「利益」の概念の特許を取得しようとしている

Flickrサイトの関心評価は、特別な方法でサイト上の写真の絶対評価を動的に計算するいくつかの秘密のアルゴリズムです。 写真の「おもしろさ」を計算するために、さまざまなパラメーター、量的（ビューとコメントの数）および定性的が考慮されます：ユーザーが写真を表示する場所、写真にコメントするユーザー、お気に入りに入れるユーザー、この画像のセマンティックタグ、その他のパラメーター



関心を計算するためのアルゴリズムは、Webページの「重要度」を計算するための有名なGoogle PageRankアルゴリズムと比較できます。 違いは、Flickrの関心はソーシャルネットワークに固有の多くの内部要因を考慮に入れていることです。 つまり、興味度評価（IR）は、新世代のソーシャル検索エンジンのPRです。



IRを計算するための正確な公式は誰も知らず、「興味深い」の正確な値は画面に表示されません。 このシステム全体がどのように機能するかを推測するだけです。 各写真のIRが毎日変化するように、IRアルゴリズムが定期的に変化することのみが知られています。 過去7日間のFlickr Webサイトで最も興味深い写真のランダムなセットを1ページで表示できます 。



楽しさの概念は、 タグクラスタリングテクノロジーとともに2005年8月にFlickr で初めて実装されました。 しばらくして、なぜこのテクノロジーがそれほど重要なのかが明らかになりました。 実際、関心のあるFlickrの評価は、「ロングテール」 （ロングテール）のよく知られている現象を考慮すると、集合的な心に対する推奨の高度なエンジンでもあります。



明らかに、IRアルゴリズムの重要性は、2つの特許を申請したYahooからも高く評価されました。 これらのアプリケーションは2006年2月8日に提出され、先週公開されました。



最初の特許である「関心によるメディアオブジェクトのランキング」では、個々のユーザーごとに関心の評価を個別に計算できると報告しています。 2番目の特許は、 Media Object Metadata Associations and Rankingと呼ばれます。 ランキングに使用できるメタデータ（タグ、コメント、注釈、お気に入りに追加されたという事実）について説明します。 特許のテキストの小さな分析はここで見つけることができます 。



ソーシャルランキングアルゴリズムの特許を取得するためのYahooのばかげた試みを呼び出す専門家もいることを付け加えてください。これらのアルゴリズムは多くのWeb 2.0サイトで何らかの形で既に使用されているためです。
