Intersystems Student Programming Competition 2012

InterSystems Student Programming Competition 2012！





親愛なるhabrastudentsとhabr科学アドバイザー！





セッションは遅れています！ ただし、リラックスするのではなく、知性を維持し、 InterSystems Student Programming Competition 2012に参加することをお勧めします。



InterSystems Campusプログラムに参加している大学の学生または学生チームが参加できます。

あなたの大学はまだプログラムに参加していませんか？ 今日あなたの大学を登録してください！



コンテストのチーム登録はこちら 。



こちらから申請書をダウンロードできます。



何のために戦うのですか？



コンテストの学生受賞者に5,000ドル

勝者の上司に5,000ドル

フライトと宿泊施設を含む賞品の公式プレゼンテーションのための3日間の米国でのグローバルサミット2012へのチケット（3泊分の宿泊料金を支払う）。



そしてもちろん、あなたとあなたのプロジェクトはインターシステムズの国際およびロシアのウェブサイトに投稿されます-これは世界中のインターシステムズパートナーの栄光と注目です。



何を勝ち取る必要がありますか？




アプリケーションは興味深く、有用で、関連性があり、インターシステムズのテクノロジーを効果的に使用する必要があります。 これらは過去数年間で獲得した学生プロジェクトです。

達成基準：

  • 英語では小さくてもわかりやすいドキュメントの存在。
  • アプリケーションは英語をサポートする必要があります。
  • まあ、アプリケーションがインターネットに展開される場合;
  • アプリケーションの原則を説明する英語のコメント付きのビデオクリップがある場合に優れています。
  • 英語でのpptまたはpdf形式のプレゼンテーションが必要です。
  • ソースコードを送信する必要はありませんが、利用可能なgit / svnリソースに入れることをお勧めします：github、google-code;
  • 少なくともあなたにとって、アプリケーションは理にかなっているはずです。




2010年と2008年にはロシアの学生チームが競争に勝ちましたが、最近のInterSystems GlobalsDB Challenge競争ではロシアのプログラマーも勝ちました。



申し込みの最終日は2012年2月13日です！

時間はあまりありませんが、時間の不足はいつ学生を興奮させましたか？



頑張って！
