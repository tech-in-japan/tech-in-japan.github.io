トップ10仮想化ソフトウェア会社

今日、彼らはほとんどの場合コンテナについて書いていますが、データセンターではサーバー仮想化ソフトウェアが引き続き支配的です。 以下は、現時点で最高の仮想化ソフトウェアソリューションを提供している「トップ10」ベンダーです。



サーバーの仮想化に関する議論は、多くの場合高音で終わり、妥協のない論争になりますが、ほとんどの場合VMwareから始まるため、この会社でVPSに最適な仮想化製品のベンダーのリストを始めます。











サーバー仮想化ソフトウェアテクノロジ自体の提供に関しては、必要に応じてすべてのコンポーネントと機能セットを有益に使用するために、プログラミングの深い知識が必要になることはほとんどありません。 これらのソリューションは、少数のWebサイト、仮想デスクトップ、またはイントラネットサービスをホストするための複数の仮想マシン（VM）から、数百万のインターネットユーザーにサービスを提供する数万のVMまで拡張されています。 このリストにある製品の名前がまだ何もわからない場合は、それらを知る時間です。









1. VMware



VMwareソフトウェアを使用していないデータセンターを見つけてください。 これを行うことができるとは考えられません。そのようなデータセンターはほとんどありません。 VMwareはサーバー仮想化市場を支配しており、これはその商用製品であるVMware vSphereだけに当てはまりません。 VMwareは、デスクトップ仮想化ソフトウェア（VDI）市場のリーダーでもあり、VMware Server製品を使用した無料のサーバー仮想化セグメントのリーダーでもあります。 VMwareは、革新、戦略的パートナーシップ、信頼性の高いソフトウェア製品を通じてリーダーシップを維持しています。









2.マイクロソフト



マイクロソフトは、Linuxハイパーバイザー以外の唯一のHyper-V製品で市場に参入し、VMwareがそれまでに支配していたサーバー仮想化セグメントで成功を収めました。 データセンターの分野に根ざしたマイクロソフトは、現在、魅力的なHyper-Vライセンススキームとそのオペレーティングシステムを提供しています。 すべてのMicrosoftユーザーにとって、Hyper-Vは新しいWindows Server OSリリースのリリースごとに競争力を強化する価値のあるソリューションです。 また、Microsoftは、Azureクラウドサービスを使用したい企業や、 VPSホスティングサイトおよびAzureサービスでのHyper-Vサービスの統合管理に関心のある企業の間でますます注目を集めています。









3. Citrix



Citrixは、かつてアプリケーション仮想化分野で唯一のオオカミでしたが、今日では世界で最も広く使用されているXenクラウドソフトウェア（商用XenServer製品の基盤）のベンダーです。 AmazonはElastic Compute Cloud（EC2）サービスでXenを使用しています。これは、Rackspace、Carpathia、SoftLayerなどのクラウド製品についても同じことが言えます。









4. Red Hat



過去15年にわたって、Red Hatは業界のリーダーであり、オープンソースコミュニティであると認められてきました。 今日、最も成功したオープンソース企業としてランク付けされたRed Hatは、2008年にQumranetを買収し、KVMとSPICE（独立コンピューティング環境用のシンプルなプロトコル）を仮想化ソリューションとして買収しました。 2009年12月、Red HatはオープンソースプロトコルとしてSPICEをリリースしました。 有名なRed Hat Enterprise Virtualization（RHEV）サーバーおよびワークステーション仮想化プラットフォーム（現在のRed Hat Virtualization（RHV））は、KVMハイパーバイザーとRed Hat Enterprise Linuxサーバーオペレーティングシステム（RHEL）に基づいています。 RHEVはオープンスタンダードに基づいており、LinuxおよびWindowsで動作し、SAP、SAS、Oracleなどのエンタープライズアプリケーションもサポートします。









5.オラクル



エンタープライズクラスのデータベースサーバーセグメントにおけるOracleの優位性に感心していない場合、Sun Microsystemsの買収により、間違いなく仮想化市場で強力なプレーヤーになりました。 さらに、Oracleは独自のOS（Sun Solaris）、多くの仮想化ソフトウェアソリューション（Solaris Zones、LDomsおよびxVM）、およびサーバープラットフォーム（SPARC）を所有しています。 不可抗力（Oracle）を固定オブジェクト（データセンター）に適用するとどうなりますか？ Oracleベースのデータセンターを受け取ります。









6.アマゾン



Amazon Elastic Compute Cloud（EC2）は、業界標準の仮想化プラットフォームです。 Ubuntu Cloud Serverは、Amazon EC2サービスとのシームレスな統合をサポートしています。 EngineYard RubyアプリケーションサービスもAmazonクラウドを使用します。









7. Google



Googleでは、仮想化が最初に頭に浮かぶものではないことは明らかですが、Google Apps、AppEngine、および広範なビジネスサービスは、会社がクラウド指向サービスをどのようにマスターしたかを明確に示しています。

オープンソースのGoogle Ganeti仮想サーバークラスター管理ソフトウェアは、XenやKVMなどの既存の仮想化テクノロジーの上に実装され、基本的にこれらのハイパーバイザーのシェルとして機能し、システム管理者がクラスター構成を作成できるようにします。









8. Parallels / Virtuozzo



Parallelsは、Linux（Virtual Private Server、VPS）の下で仮想サーバーをホストするための商用製品でオープンソースOpenVZプロジェクトを使用しています。 Parallelsソフトウェアに基づいたホスティングソリューションに慣れると、VMの高密度と低コストが主な耳になります。 世界最大のホスティング企業は、これら2つの主な理由からParallelsを選択しています。 しかし、LinuxベースのLinuxホスティングのコンテナー化された仮想化の進歩はまだ止まりません。 ハードウェアごとのWindowsホストの数を最大化するために、ParallelsはWindows用のコンテナ仮想化ツールも開発しました。 Parallelsは、Parallels DesktopおよびServer for Macを含むデスクトップおよびアプリケーション仮想化製品で最もよく知られていますが、そのプロバイダー事業は2015年3月にブランド名を変更し、Odinに名前が変更され、Parallels Cloud Serverはヴィルトゥオッツォ。 Virtuozzoを使用すると、アプリケーションを個別の軽量コンテナーで実行でき、OdinはDockerおよびGoogleのコンテナー標準と緊密に連携しました。

2015年12月、OdinブランドとOdin Service AutomationラインはIngramに売却され、Odin Service Automationプラットフォームを開発している500人の従業員とともに経営陣に移管されました。 PleskとVirtuozzoの指示は別々の会社に分けられ、Parallels Holdings Limitedが引き続き主要な所有者でした。 Parallelsホールディングの一部としてのVirtuozzoは現在、仮想化ソリューションに取り組んでいます。









9.ファーウェイ



Huaweiは仮想化市場の新しいプレーヤーの1つであり、その企業ビジネスは2011年に開始されました。 同社は米国ではあまり知られていませんが、ファーウェイは通信、中国、ブラジル、ロシア、インドなどの新興市場で大きなシェアを占めています。 Huaweiの仮想化およびクラウドインフラストラクチャスタックの主要製品であるFusionSphere仮想化インフラストラクチャソフトウェアは、Xenに基づいています。 他の多くの主要な仮想化プロバイダーと同様に、HuaweiはFusionSphereおよびFusionCloudソリューションでOpenStackを使用しています。









10. VERDE VDI



Virtual Bridgesは、現在Virtual Desktop Infrastructure（VDI）と呼ばれるものを発明した会社です。 Virtual Bridges VERDE製品を使用すると、ユーザーはWindowsおよびLinuxデスクトップをカーネルバージョン2.6以降の任意の32ビットまたは64ビットLinuxサーバーインフラストラクチャに展開できます（これについてはVERDE企業ドキュメントを参照）。 2015年2月、NIMBOXXはVirtual BridgesからVERDE VDIを買収しました。 NIMBOXXは現在、VERDE VDIテクノロジーをすぐに展開できるVDIロードソリューションとして、またより広範な製品として使用し、ソフトウェアで定義された「データセンター」を従業員数の多い企業に提供しています。
