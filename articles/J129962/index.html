子供向けのルビーなど

Rubyは、「プログラマーの親友」、優れたプログラミング言語、すばらしいスクリプト言語、およびWebアプリケーション開発のための優れたツールとしての地位を確立しています。 Rubyは、非常に活発で友好的なコミュニティを持つ非常に優れた言語であることを誰もが知っています。 しかし、誰もがこのコミュニティが非常に友好的であることを知っているわけではなく、同僚だけでなく、多くの有用な宝石を開発するプログラマーだけでなく、彼らのためにいくつかの非常に興味深い有用なツールを開発した子供たちも世話しました。



あなたがプログラマであり、プログラミングに興味を持ちたい子供がいるなら、この短い記事はあなたのためのレビューです。





1. 子供向けルビー

Ruby for Kidsは教育リソースであり、そう言えば、Rubyで簡単なコンピューターゲームを開発するためのフレームワークです。 Ruby for Kidsフレームワークには、Ruby言語自体と、RubyおよびC ++で2次元ゲームを開発するためのGosuライブラリが含まれています。 このサイトでは、子供たちが開発したゲームのデモビデオ、チュートリアル、ゲームの開発を説明するスクリーンキャストを見ることができます。



2. Hackety Hack！

ハケットハック！ -Ruby + Shoes（クロスプラットフォームデスクトップアプリケーションを開発するためのフレームワーク）に実装されたシンプルなRubyアプリケーション開発環境。 Hackety Hack開発者にとっての挑戦！ Rubyでプログラムを開発および実行するための単純な環境の開発ではなく、初心者プログラマー向けのツールキットの開発でした。







3. キッズルビー

Kids Rubyは、Hackety Hackに代わるものとして位置付けられるもう1つのRuby開発および実行環境です。..Kids Rubyには、コードを入力するためのエディター、コード実行の結果のビューアー、リファレンスブック、および「タートルグラフィックス」を1つの作業ウィンドウに描画する機能が含まれています。





年長の方には、不必要なジェスチャーをせずにRubyを使いこなせるようにするいくつかのプロジェクトもあります。 まず、これはTryRubyプロジェクトです。これはIRB Webエミュレーター（RubyのREPL）およびシンプルなチュートリアルです。 個人的に、私はかつて、 Ruby Koansプロジェクトが本当に好きでした。これは、Test :: Unitに基づいて構築された学習環境です。 一番下の行は、テストマッチャーの内容を編集した結果として、すべてのテストが正しく合格することです。 RubyKoansはマシンにダウンロードできますが、特にWebインターフェイスを実装しているような怠け者の場合は、何もインストールする必要はありません。以下のアドレスに移動して、フィールドに入力して一致にエラーがないようにします。 プロジェクトWebサイト： http : //rubykoans.com/ 、オンラインバージョン： http : //koans.heroku.com/
