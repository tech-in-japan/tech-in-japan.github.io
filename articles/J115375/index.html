安価な12インチのラップトップASUS UL20FTのレビュー

古い15インチToshibaがようやく機能しなくなったとき、新しいラップトップを選択するときがきました。 そして、バッテリーで動作する、より小さく、より軽く、より長いものが欲しかった。 もちろん、ネットブックでは選択肢が減りました。 すべての提案を検討した結果、10〜11インチでは不十分であり、中盤は12インチの安価なネットブックであることが判明しました。 しかし、すべてがそれほど単純ではありません。私たちの市場にはこのサイズのモデルはあまり多くなく、選択できるものはほとんどありません。 これは、ASUS Eee PC 12 **またはMSI Wind12のいずれかです。 デルはまだ存在しますが、それらは広く普及しておらず、ユーザーレビューを見つけることは困難です。 Atomは痛みを伴い弱く、Athlon Neoはその温度に満足していませんでした。 ASUS UL20FTにつまずいたとき、私は自分が必要なものを見つけたことに気付きました！







UL（無制限）シリーズのASUSは、スリムなボディと長い動作時間、高度なパフォーマンスを兼ね備えたラップトップとしての地位を確立しています。 このシリーズは、熱と電力の消費が少ないIntel CULVプロセッサを使用しています。



このネットブック（またはラップトップ？）には2つのバリエーションがあります。つまり、これはいわゆる「 日本語 」バージョンと「 アメリカ 」バージョンです。 プロセッサ（Celeron Dual CoreとCore i3 330UM）、RAMの量、そしてもちろん価格が異なります。







特徴：

  • プロセッサー：Intel Celeron Dual Core 1060 MHz Arrandale（U3400）
  • バス：2.5 GT / s 2 Mb L2キャッシュ
  • RAM：2048 MB DDR3-1333 MHz
  • 画面：12.1 "LED光沢
  • 解像度：1366x768
  • ビデオカード：Intel GMA HD、64 + 694 MB
  • ハードドライブ：320 GB（5400 rpm）、SATA
  • 通信：イーサネット、Bluetooth 2.1 + EDR、WiFi（802.11a / b / g / n）
  • ポート：3xUSB（2.0）、Kensingtonセキュリティ、ライン出力、HDMI、マイク入力、VGA
  • 拡張スロット：カードリーダー（SD / MMC / MS / Pro）
  • 追加デバイス：カメラ（0.3）、Express Gate
  • バッテリー：Li-Ion 5600 mAh（最大8.0時間）
  • ソフトウェア：MS Windows 7 Home Basic（64ビット）
  • 保証：2年間の国際メーカー保証




鉄



内部には、1060 MHzの周波数で動作する2コアのIntel Celeronデュアルコアがあります。 このプロセッサは、かなり平均的なパフォーマンスと低消費電力を備えています。 Intel GMA HDはHDビデオを簡単に表示し、中程度の設定でいくつかのゲームをプルします。







ケースと外観。



上部カバーはアルミニウム製で、金属が手をきれいに冷却します。 そのような素材に指紋を残すことは困難です。







小型（296x210x31 mm）のため、ラップトップはほとんどスペースをとらず、コンパクトに見えます。また、1.6 kgという小さい重量で、ラップトップを頻繁に持ち運べることが嬉しいです。







前面にはインジケータライト以外には何もありません。







左側には、ケンジントンロック、換気グリル、HDMI、USB 2.0、ヘッドフォン出力、マイク入力用のマウントがあります。









右側には、カードリーダー、互いに離れた2つのUSB 2.0コネクタ、モニターを接続するためのD-Sub、ネットワークと電源用のRJ-45があります。







背面全体がバッテリーです。

ラップトップの底面は粗いプラスチックでできています。 RAMとハードドライブを交換するには、3本のボルトを外して、カバーを取り外します。







このラップトップには2 GBのブラケットが1つあり、将来的にメモリを簡単に増やすことができます。 シリアル番号とWindowsライセンスが付いたステッカーを消去する際の将来の問題を回避するために、それらは上部にフィルムで覆われています。







音



ラップトップケースの前面に近づくほど狭くなり、スピーカーがテーブルを斜めに見ていることがわかります。 音はテーブルから反射され、ユーザーに直接向けられます。







説明から判断すると、ラップトップはSRSテクノロジーを備えたAltec Lansingを使用しているため、このような小さなデバイスでも非常に優れたサウンドを実現できます。



スクリーンとカメラ



画面サイズ12インチ、解像度1366x768。 画面の周りには黒い光沢のあるベゼルがあります。 ほぼすべての場所にLEDバックライト付きの光沢のある画面があり、低電力消費のためにラップトップの動作時間が延長されるため、ここでは奇跡は起こりませんでした。 視野角は良好で、バックライトは明るいです。

上部には、シンプルなウェブカメラとその左側にマイクがあります。



入力機器



大きなキーを持つキーボードアイランドタイプ。 ロシア語とラテン文字は異なる色で適用されます。







キー間の距離は約2mmです。 その上に印刷することは喜びです。 右側にはテキスト上のナビゲーションキーがあり、矢印キーのサイズも変更されていません。









左-適切な場所でCtrlキーを押します。 キーはラフです。 キーボードの下には銀色のプラスチックがあり、指紋も残りません。







キーボードの欠点は、キーを押したときにわずかにたわむことです。 これは作業に影響を与えず、原則として、この欠点に気付かないこともあります。







タッチパッドは小さく、粗い表面を持っています。 最初は、マルチタッチは機能しませんが、 特別なドライバーを使用して有効にすることができます。 タッチパッドの下のボタンはあまり便利ではなく、少しきついです。







バッテリーと充電



5600 mAhのバッテリーが含まれています。 それは十分に厚く、ラップトップから少し突き出ています。







バッテリーには独自のゴム製の脚があり、ラップトップの背面はわずかに持ち上げられています。







Battery Eaterプログラムのテストでは、ラップトップが最小限のバックライトで9時間動作可能であり、ワイヤレスインターフェイスがオフになっていることが示されました。 高輝度では、Wi-Fi充電を介してインターネットをサーフィンすることで5時間十分です。

充電器は小さく、組み立てるとスペースをほとんど取りません。











システム



搭載されているラップトップは「Windows 7 64x Home Basic」です。 同社には、ASUSのさまざまなプログラムがインストールされていたため、ラップトップのクリーニングは簡単です。 ASUS ExpressGateの便利なOSもあり、8秒でロードされ、すぐに開始できます。



おわりに









ASUSのラップトップは非常に興味深いことが判明しました。 優れたパフォーマンスのコンパートメントでの長時間の作業と、これらすべてがスタイリッシュなデザインと高品質の素材を備えた薄型ラップトップで本当に気に入っていました。 Athlonとは異なり、ラップトップはほとんど加熱されません。 ASUS UL20FTの価格は16,000 pです。今では15,000 pで見つけることができます。 Core i3でより生産的なバージョンが必要な場合は、22,000ルーブルをレイアウトする必要があります。



PS：コメントのアドバイスによると、ASUSからTurbo33を無効にしました。 奇妙ですが、埋め込みビデオのパフォーマンスは向上しています。
