DevCon 2013：会議の2日目、メインプログラムの生放送をご覧ください！

親愛なる同僚、開発者、ソフトウェアテスター！ DevCon 2013カンファレンスオーガナイザーチームを代表して、大規模なカンファレンスの2日目のオープニングを発表できることを嬉しく思います。







今日、モスクワ時間の5月30日09:00に、その場にいる約1,000人の参加者と何万人ものオンライン視聴者のために、DevCon 2013カンファレンスの2日目が始まります。



日中の09:00（モスクワ時間）から18:30まで接続して、会議をオンラインで視聴します。 以下に、DevCon 2013の2日目の詳細情報と快適な視聴のためのカンファレンスプレーヤーを示します。





会議の2日目のメインプログラムは、各30分の興味深いミニプレゼンテーションで始まります。 パートナーや招待講演者から、ソフトウェア開発ツールに関する多くの興味深いことがわかります。



2日目はテクニカルレポートでいっぱいです。 30時間のレポートの間に、会議参加者とライブ視聴者は、ソフトウェア開発の最新のアプローチと最新のツールについて多くの興味深いことを学ぶことができます。



会議の最終は公式の閉会式であり、その枠組みの中で結果が要約され、大会の勝者が発表され、お土産が配布されます。



次のマスタークラスは、追加プログラムの一環として、会議の2日目に開催されます。



  • マスタークラス 。 スクラム：Askhat UrazbaevとViktor StrelkovによるVisual Studio 2012の理論と実践。
  • デニス・コルトゥノフとイゴール・ギレフによるデザインスクール 「非デザイナーのためのインターフェースのデザイン 」。
  • パオロサルバトーリによるマスタークラス 。 Windows Azureを使用してモバイルアプリ用のクラウドバックエンドを構築する


2日目は会議の閉会式で終了します。
