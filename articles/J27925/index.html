code_swarm：開発履歴の視覚化





code_swarm-アプリケーション内のコミットの履歴を視覚化できるアプリケーション。 開発者がコードまたはドキュメントを変更し、中央リポジトリに送信すると、コミットが発生します。 開発者とファイルは、動く要素として表示されます。 開発者がコミットすると強調表示され、このコミットの影響を受けるファイルも強調表示されます。 ファイルの色は、ファイルの種類（コード、ドキュメント）によって異なります。 開発者のアクティビティが低下すると、表示がフェードします。 以下のヒストグラムは、変更の履歴とアクティビティを示しています。



プロジェクトは無料で、 code.google.com / p / codeswarmからダウンロードできます。



現時点では、 Python 、 Apache 、 Eclipse 、 PostgreSQL 、 Djangoなどの有名なプロジェクトの歴史について、多くの準備された視覚化が用意されています 。
