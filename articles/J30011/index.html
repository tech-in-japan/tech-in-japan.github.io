法人向けの「納税者」プログラムの公式文書には、次のセクションがあります。

Linuxにプログラムをインストールする

1. wine 0.9.52（以降）をインストールしますwww.winehq.org/site/download

2.ロシアのWindowsフォントを書き換える

micross.ttf

sserife.fon

tahoma.ttf

ワインフォントフォルダー（通常、ユーザーのホームフォルダーの.wine / drive_c / windows / fontsフォルダー）

3. windows extrac32.exeプログラム（通常はwindows \ system32にあります）をsystem32 wineフォルダーに書き換えます（通常、これはユーザーのホームフォルダーの.wine / drive_c / windows / system32フォルダーです）

4.次に、NP YuL idw * .exeのインストールプログラムを実行します。たとえば、UBUNTUのコンテキストメニューで、Wine Windows Emulatorで開いている項目を選択し、NP YuLをインストールするためのガイドのすべてを選択します

5. F1ヘルプが機能しない場合は、index.htmファイルを使用してヘルプを作成します（通常、これはユーザーのホームフォルダーの.wine / drive_c / Program Files / TaxpayerフォルダーYUL / INPUTDOC / HELP / index.htmです）

6. rvcgnivc.ru/download/idw/other/linux/oofficeファイルをコマンドフォルダーに配置します（通常、これはユーザーのホームフォルダーにある.wine / drive_c / windows / commandフォルダーです）

7. rvcgnivc.ru/download/idw/other/linux/ooffice.regファイルをregeditプログラムにインポートします（通常、ユーザーのホームフォルダーにある.wine / drive_c / windows / regedit.exe）
