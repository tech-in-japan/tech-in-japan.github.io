2018年、KDEプロジェクトの開発者は50万ドル以上の寄付を受け取りました。 そしてそれらだけでなく...





フリーソフトウェアの開発者の多くのチームの生活は、禁欲主義に満ちており、困難や障害を克服し、多くの場合、コミュニティは参加者の利他的な信念と献身だけのために生き残ります。 すべての人気のある無料製品が、安定した信頼できる資金源を見つけることができません。 しかし、運命は当然の贈り物を与えることがあります。 そのため、今年はKDEプロジェクトで起こりました。 二回。



KDEは、Plasmaデスクトップ環境とKDEアプリケーションスイートというフリーソフトウェアを開発する組織および国際コミュニティです。 このプロジェクトの主な資金源は、寄付金の収集とターゲットを絞った募金キャンペーンです。











2018年10月15日に、KDEプロジェクトの開発を監督する非営利組織KDE eVは、別の愛好家コミュニティであるHandshake.orgからプロジェクトの22周年記念に30万ドルの寄付を受け取ったことを発表しました。 さらに、受け取った30万ドルのうち10万ドルは、無料のCalligraオフィススイートの開発に使用されます。







フリーソフトウェア活動家のあるコミュニティが突然、別の活動家グループにそのような堅実なサポートを提供することができたのはどうしてですか？ 問題は、協調融資キャンペーンの枠組みの中で、Handshake.orgがプラットフォームの開発のために1,020万ドル以上を調達し、計画予算を大幅に上回ったことです。 開発者は、余剰分を自分たちに任せるのは不公平だと考えたため、受け取った金額の70％が他の重要な無料プロジェクトの支援に費やされることになりました。







Handshake.orgコミュニティは、ドメイン名とデジタル証明書の分散システムを開発しています。これは、DNSルートサーバーと証明機関の代替の信頼できる検証済みインフラストラクチャとして使用できます。 ハンドシェイクプロジェクトは、DNSプロトコルを完全に置き換えることを目的とするのではなく、ブロックチェーンとP2P通信に基づいて構築された分散ベースの形で、 従来の集中型ルートゾーンとルートサーバーの複製インフラストラクチャを作成するタスクを設定します。



KDEプロジェクトに加えて、Handshake.orgコミュニティの運営委員会もGNOMEプロジェクトを財政的に支援し、40万ドル（GIMP専用の10万ドル）を移行しました。 また、Handshake.orgは、Apache Software Foundation、Arch Linux、Debian、EFF、FSF、GNU、インターネットアーカイブ、LibreSSL、Mozilla、OpenSSL、Tor、Wikipediaなど、いくつかの他のプロジェクトおよび組織のサポートを発表しました。







しかし、それだけではありません。 8か月前、KDE開発者は 、謎のパイナップル慈善基金から200,000ドル（18.7 BTC）に相当する暗号通貨の寄付を受け取ったことを発表しました。 パイナップル財団は 、匿名の暗号通貨慈善家によって設立され 、重要な非営利プロジェクトの開発を刺激し、合計57の組織に資金を割り当てました。 したがって、Free Software Foundation、Electronic Frontier Foundation、Apache Software Foundation、Let's Encrypt-25万ドル、OpenStreetMap Foundation-15万ドル、OpenBSD Foundation-5万ドルです。



このように、私たちは、人々が彼らに落ちた超富を充当し、増やす代わりに、必要な他の人々と資金を共有することを選んだときに、興味深い先例に出会いました。 そんなことが好きですか？
