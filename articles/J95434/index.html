そして、すべては大人以上のものです。 4つのコアと1.6 GHzの公称クロック周波数（ターボモード-2.8 GHz）を備えた強力なモバイルプロセッサIntel Core i7-720QM （これは私のモデルです。IntelCore i7-620Mおよび最も最新のIntel Core i7-820QMも選択可能です） 。







Intel HM55チップセット。 最も強力なモバイルビデオカードの1つは、ギガバイトのGDDR5メモリを搭載したATI Mobility Radeon HD5870です。 8ギガバイトのRAM（DDR3、1066 MHz）。 2台の500GBハードドライブ...それでは他に何がありますか？ 画面は17.3インチで、Blurayドライブなどがあります-これで範囲を十分に理解できたと思います。 まあ、これはすべてユニークなケースです...一般的には、最近組み立てられたコンピューターとほぼ同じ充填量のラップトップで、どこかさらに良いです。

ケースは本当にゴージャスであると同時に、珍しくて実用的です。 滑らかな移行はほとんどありません-むしろ鋭いエッジが優勢ですが、それらについてはカットしません）表面の「ねじれ」は混oticに見えることがあります-思わずHTC Diamondの裏表紙を思い出します。 トップカバーは完全にソフトタッチソフトタッチで作られており、装飾や他の装飾はありません-碑文ASUSと目立たないRoGロゴ（黒地に黒）のみです。 珍しいことに、カバーの面積はラップトップの底面よりもわずかに小さくなっています。これは、2つの大きなヒンジのおかげで開くためです。

性能



インストールされたIntel Core i7-720QMクアッドコアプロセッサ （モバイルセグメントで最も強力なプロセッサの 1つ）の公称クロック速度は「わずか」1.60 GHzですが、Turbo Boostテクノロジーの魔法の杖の下では、桁が...桁が...エレガントな2.80 GHzに変わります！ 当然、4つのコアすべてではありませんが、すべてのアプリケーションがそれらを効果的に使用できるわけではありません。 ちなみに、テクノロジーのアクティビティには、システムの右上隅にあるIntelのプリインストールウィジェット（ダウンロード可能）が視覚的に表示されます。 チップセットは、新しいIntel HM55を使用します。4つのスロットには、それぞれ2ギガバイトの4つのRAMスロットが挿入されます。 8ギガバイトのRAM（Kingston、DDR3 PC3-10700）-すべてのデスクトップでそれほどではありません）
