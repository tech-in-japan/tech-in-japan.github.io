Opera Developerのブックマーク同期





Opera Developer 26の最近のビルドでは、待望のブックマーク同期があなた（そして私たち）に現れました-それはまだコンピューター用ですが、携帯電話の写真はほのめかしているようです。 この実装は、フラグからオンにしたときにしばらく機能していたものを置き換えました。 新しい同期はデベロッパーチャンネルで公開され、リリースに近いベータ版に移行します。



質問への回答



同期データはどこに保存されますか？



データはOpera Softwareのサーバーに保存されます。



どのデータが同期されますか？



現時点では、ブックマークの同期のみを行っています。 今後、他の情報を追加する予定です。 ご覧のとおり、フラグには開いているタブを同期するための設定がありますが、現在はデフォルトで無効になっています。



同期されたブックマークは暗号化されていますか？



クライアントとサーバー間のデータ転送中-すべてが安全に暗号化されます。 ただし、データ自体は暗号化されていない形式でサーバーに保存されます。 将来、パスワードなどの機密の個人データを同期する場合（およびその場合）、暗号化された形式で保存します。



これらのブックマークはOpera Linkのブックマークと何の関係がありますか？



Opera 12から同期されたすべてのデータは、Opera Linksウェブインターフェースからアクセス可能です。 既存のOpera 12インストールからブックマークを新しいシステムにインポートすることもできます。



アカウントでボタンを削除することはできますか？



このボタンを使用すると、新しい同期機能にアクセスできます。 しかし、それだけではありません-今後のリリースにご期待ください。



この同期は、以前の実装と何の関係がありますか？



以前の実験的実装（Opera 19に登場）は、新しいものに置き換えられました。
