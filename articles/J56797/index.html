Acer AOA-110 HOWTO。 パート1：ネットブックの確認とアップグレード

この記事はいくつかの部分で構成されています：読み書きがとても便利です。



そもそも、ネットブックを購入する必要があるのです。 私は主に、来るべきヨーロッパ旅行に関連して彼を必要としていました。 選択基準は次のとおりです。

  • i386アーキテクチャを備えた小型軽量のラップトップが必要
  • 推奨される最長のバッテリー寿命
  • 基本的な通信ツール（Wi-Fi、イーサネット、USB）が必要
  • Linux OSでKDE3を使用してすべてを比較的高速に回転させるには
  • 旅行中に紛失する可能性があるため、安価でなければなりません


実際、モスクワのコンピューター小売業者の品揃えを検討した後、 Acer Aspire One AOA 110-Awネットブックに選択肢が落ちました。 なんで？ このデバイスは、私が見つけたもののなかで許容できる特性を備えた中で最も安価であり、同時に非常に優れています。



Aspire One Seriesの周辺では、特定のユーザーとモッダーのコミュニティが既に形成されています。最も強力なものの1つはaspire1.ruです。多くのハードウェアの問題を解決するとき、この特定のサイト（またはフォーラム）に導かれました。



デバイスに慣れましょう。



デバイスは、8.9インチの画面サイズを持つ小さなサブノートブックです。 クロック周波数1.6 GHzの比較的新しいモバイルプロセッサIntel Atom N270に基づいて構築されています。 当然のことながら、プロセッサはIntelの一連の省エネ技術をすべてサポートしており、バッテリー駆動時のオフライン作業時にクロック速度を動的に低下させることができ、TDPも低くなっています。 Atomは、Intel NetBurstマイクロアーキテクチャ、つまりPentium 4（コマンドの次の実行）で使用されたマイクロアーキテクチャに注目して構築されています。 また、プロセッサはハイパースレッディングテクノロジーをサポートしているため、オペレーティングシステムには2つの仮想コアがあります。 プロセッサがパフォーマンスの点で記録を破るというわけではありませんが、同時に、それは特に脆弱ではありません（たとえば、VIA Edenはここで完全にマージされます）。 この価格でのパフォーマンスに私はうれしい驚きを覚えました。



その中のチップセットは非常に有名です：統合されたIntel GMA950グラフィックスを備えたIntel 945GSE。 インターネットからの情報によると、チップセットは最大2 GBのDDR2 RAMをサポートし、統合グラフィックスのクロック速度は大幅に低下します（チップセットは400 MHz用に設計されていますが、133 MHzになります）。 しかし、私たちはゲームには対応していませんが、標準的なニーズにはこれで十分です。 繰り返しますが、省エネ...



オンボードでは、このモデルはすでに512 MBのRAMです。 このボリュームはシステムボードにはんだ付けされています。 近くには空のSODIMM DDR2コネクタがあり、最大1 GBのメモリバーを取り付けることができます。 つまり、チップセットでサポートされている切望されている2 GBは、大きな変更なしでは取得できません。 フォーラムからの情報によると、2つのギグスティックをスロットにインストールしようとしましたが、この場合、ネットブックは起動しませんでした。 はんだ付け装置の存在下で1人のハード改造者が2 GBのメモリモジュールを手に入れましたが、そのためにMBとSPDチップにはんだ付けされたメモリモジュールをはんだ付けしました。 しかし、私の意見では、これは非常に難しいため、1 GBのブラケットを追加することに限定しました。 ここでは、大きな石を取り出して、このラップトップのケースの開発者に投げる必要があります。モジュールを追加するには、ケースからマザーボードをほぼ完全に取り外す必要があります。 幸いなことに、これは難しくなく、関連するHowToがあります。 ちなみに、容量が1 GBのSO-DIMM DDR2 Kingston PC2-5300 CL5という素晴らしいスタートアップバーがありました。



ドライブは、8 GB（110シリーズの旧モデルでは16 GB）の容量を持ついわゆるNAND SSDモジュールです。 一般に、ドライブの読み取り/書き込み時間が遅くなるのでなければ、そのようなボリュームに対しては何もありません。 このドライブは、通常の最新のSSDとは何の関係もありません。見た目は、いくつかのチップとATAインターフェイス（ZIFコネクタの形式）を備えたベアボードです。 一般的に、このドライブに満足していない人は、それをより機敏で容量のあるものに変えています。 2つの主なアクション領域があります（外部ドライブは考慮しません）：内蔵SATAのはんだ付けを解除して2.5インチハードドライブをブリッジするか、ZIF ATAインターフェイスを備えた1.8インチHDDを購入します（iPodビデオで使用されます）。 最初の方法はより予算がかかりますが、深刻な介入が必要です（母親のコネクタをはんだ付けし、ケースのラックの半分を取り壊す）。 2番目の方法は（「ギガバイトあたりのコスト」の観点から）より高価ですが、より正確で「無駄」は少ないです。 私は2番目のものを選択しました。eBayオークションで30 GBの容量の東芝製MK3008GALドライブを注文しました。 HowTo統合は、上記のフォーラムで見つけることができます。



イーサネットカード-PCIエクスプレスインターフェイス上の通常のRealtek RTL-8102E（システムは8169互換として定義）、問題なく巻き上げられます。 WLANモジュールは、有名なAtheros AR5007EGです。 Ath5kは標準ドライバーで正常に動作することを拒否し、私は長い間理解していなかったため、Madwifiモジュールをすばやくコンパイルしました（正しく実行する方法を以下に記述します）。



ウェブカメラはすぐに開始され、これを行うための努力は行われませんでした。 もう1つ気まぐれなことがあります-フラッシュカードのスロット、私はそれらを理解していませんでした（これは必要ではありませんが）が、ネットワーク上でこの主題に関するチュートリアルがあります。 画面の有効解像度は1024x600（ワイドスクリーン）です。 この垂直解像度では、すべてのウィンドウが通常画面（特に、さまざまなアプリケーションを設定するためのダイアログボックス）に収まるわけではなく、グラフィカルインターフェイスで作業しているときにフリーズすることがあります。 正直なところ、視野角に特に満足していません。 ただし、写真は明るく、画面は太陽の下でフェードしません。



完全なバッテリーの容量は2200mAhで、動作時間は約2時間です。 そのため、同じオークションで7800mAh（違いを感じる）の容量の拡張バッテリを注文しました。これは、バッテリ寿命要件（ほぼ8時間）を完全にカバーします。 ネットブックの電源回路の設計により、接続されたネットワーク充電器であれば、デバイスをオフにすることなくシームレスにバッテリーを変更できます。



キットには、主電源充電器、標準的なノートブックの外観、さまざまな紙片、およびLinux（Linpus Linux Lite）がバンドルされたリカバリディスクが含まれていました。 ネットブックにプリインストールされていました。 2時間のピッキングの後、内蔵ドライブから正常に取り外されました。



ネットブックでminiPCIeコネクタについて少し説明する必要があります。 それらの2つがあります。 システムボードの前面に1つ、標準のwlanモジュールが挿入されています。 ネットワークからの情報によると、PCIeバスのみが離婚します。 2番目はシステムボードの下側にあり、3Gモデムをインストールするように設計されています。 私の場合、はんだ付けはされていませんが、それほど困難ではありません。 ただし、モバイル3Gモデムの仕様によると、USBバスのみがこのスロットにルーティングされるため、PCIeバスを使用するカードを挿入すると機能しますが、機能しません。
