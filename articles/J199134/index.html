Firefox 25リリース

デスクトップ向けFirefoxがダウンロード可能になりました。 このバージョンの新機能：





  • 検索バー（Ctrl + F）が各タブに個別に表示されるようになりました （前のエントリは各タブに個別に保存されます-すべてのタブの単一バッファーは使用されなくなりました）。
  • Firefoxが数か月間起動しなかった場合、ユーザーには作業履歴と設定を別のブラウザーから転送するためのフォームが表示されます。
  • リセットすると、 Firefoxは開いたウィンドウとタブを保存して転送します。
  • 追加のファイルをダウンロードせずに、現在のブロックでインラインフレームを指定する機能（インライン）。 コンテンツは、新しいiframe属性srcdoc = ""で指定されます。
  • Web Audio APIのサポート 。
  • バックグラウンド接続の CSS3 サポート ：local ;
  • ECMAScript 6仕様（1、2、3、4、5）からの一連の新しい関数が実装されました。
  • 新しいタブを開くと、ページのサムネイルが空白またはコンテンツが表示されない問題を修正しました
