音節OS

ほとんどのPCユーザーにとってWindowsが唯一の選択肢であると思われた時代はすでに過ぎ去っています。 今日では、Linuxをインストールしても誰も驚くことではないでしょう。MacOS XがプリインストールされたスタイリッシュなMacでさえ、デスクに馴染みがあります。 そして、おそらく、将来、OSの選択は拡大するだけでなく、エミュレーション、仮想化、および他のシステム用の使い慣れたプログラムのバージョンの出現により、ソフトウェアの選択に関する問題を軽減するでしょう。 幸いなことに、開発者は試みています。 Monoの開発のペースだけを見てください。 そして、これはほんの一例です！



しかし、私は別のトピックを上げたいと思いました...



すなわち、新しいシステムの開発は、あまり人気がありませんが、正しい方法で開発され、興味深い機能と大きな可能性を持っています。

これらの1つは、スカンジナビアの（主に）開発者のグループによって積極的に開発され、GPLライセンスの下で配布されるSyllable OSです。











他のスクリーンショットはここで見ることができます：

リトリート。 ちょっとした歴史



音節OSはゼロから開発されたものではありません。 それはすべて、1994年以来ノルウェーのプログラマーであるKurt Skauenによって作成された、今ではほとんど忘れられているAtheOSから始まりました。 当初、著者の目標は、かつて人気のあったAmigaOSのクローンを作成することでしたが、その後の開発は独自の方法で行われました。 しかし、残念なことに、プロジェクトが2000年3月に大衆に提示された直後に、開発が停止しました（2002年）。 何がクリエイターの熱意を妨げたのかわかりませんが...



2002年7月 プロジェクトの開発（より正確には、開発の欠如）を懸念する有能なプログラマーのグループは、イニシアチブを自分たちの手に取り、Syllableの作成を開始することを決定しました。



著者の目標は、家庭や小規模オフィス向けのシンプルで便利なOSを作成することです。 実際、音節OSはPOSIX互換OSであり、CAM（!!!）GCCなどの使い慣れたアプリケーションを同梱のユーティリティで転送することができます。 これは大きなプラスです。



しかし、それでも、目標はまだ別のUNIXクローンを作成するのではなく、次のような独自の「グッズ」を備えた完全に独立したシステムを作成することでした。

-非常に高速で信頼性の高い64ビットジャーナリングファイルシステムであるAtheOS File Systemの前身から継承。

-独自の（X Window互換ではない）グラフィカルユーザーインターフェイス。

-ネイティブC ++互換API（およびPOSIX機能）;

その他の機会はますます増えています。



デスクトップOSは非常にシンプルで禁欲的で、X Window Systemにインストールされた一部の環境を連想させますが、それ自体が特別です。



リトリート。 アイコン



フリーソフトウェアのファンは、システムのアイコンに「どこか」で出会ったことにすぐに気付くでしょう。 これはたいしたことではなく、驚くべきことです。 知らない人のために：アイコンは、 Tango Desktop Projectから取得されます。これは、オープンソフトウェアとフリーソフトウェア用の無料アイコンを作成するプロジェクトです。 GIMP、GNOME、OpenOffice.org、Pidgin、その他の多くのアプリケーションで使用されています。



すでに、OSは孤立した環境ではありません。 このセットには、WebKit（および標準準拠）に基づくIDE（sIDE）、ABrowse Webブラウザー、電子メールクライアント、プレーヤー、およびマニアが作成した他の多くのアプリケーションが含まれています。 また、かなり寛大なライブラリライセンスは、将来、商用ソフトウェアメーカーに新しいシステムへの移行を検討する機会を提供します。



さらに、Apache、VIM、Python、Perlなどのよく知られた開発が積極的に移植されており、ドライバーベースは常に成長しています。



結果：

もちろん、新しいシステムに切り替えるために全員を動揺させるのは愚かなことです。 さらに、彼女はアルファ版の控えめな位置にいます。 しかし、一体何が冗談ではないのでしょうか？ :)将来的には、自宅のPCでそれを見ることができます。 そのため、今では実験者と好奇心users盛なユーザーだけがそれを見ることができます。
