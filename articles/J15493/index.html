Apple MacOS Xの未来は何ですか？

将来的には、おそらくAppleが集まり、最終的にすべてのMacOS X 10.6 PCコンピューターで完全にオープンでアクセス可能なものをリリースするでしょう。これを想像できますか？



Steve Jobsが落ち着いて引退できると思いますか。 いいえ、スティーブジョブズは非常にタフな反逆者であり、 復心の強い男です（ iKonaを参照してください。スティーブジョブズ ） 。彼はマイクロソフトを埋めるまで落ち着きません。 スティーブでもビリーでも、久しぶりです（映画「キルビル」はスティーブのお金で撮影されたに違いありません:)）。 今まで、彼は最後の決定的な戦いの前に待っていました！

そして、MacOS Xがすべての人のためにリリースされたのではなく、Mac Intelのためだけに、2005年から2007年にかけて、スティーブジョブズがこの素晴らしい移行期間を設けたのはなぜですか？

そのためだけに：

1）。 1997年にAppleとMicrosoftの間で署名されたMicrosoftとの10年間の平和条約の終了まで待つこと。

それだけです。AppleがMicrosoftの財産に入らないように義務付けた10年が過ぎました！ 震える、大胆なMS-Hryuk、それは虐殺の時です！

2）。 どんな時でも、Microsoftを怖がらせないで、Microsoftの反広告会社であるMS Windows Vistaの最新の作品をリリースさせてください。

3）。 すべての鉄製オフィスが最終的にApple MacOS X用のfireを作成する（ほとんどすべてが積極的に作成し、すべての外部デバイス用のドライバーが長い間存在する）。

4）。 MacOS向けソフトはIntelベースにスムーズに移行します。

Apple、Adobe、Quark、Corel、Autodeskなど その人気のあるソフトのほぼすべてが、すでにIntel用のMacOS Xに移植されています（Macintosh用の7501以上のユニバーサル（Intelベースの）アプリケーションのリストを参照してください）

さらに、再び、Web 2.0ベースのアプリケーションの分野におけるこのAppleの活動は、その開発がAppleを積極的に促進し、資金を提供しています。 Web 2.0アプリのプロパガンダの例として、これはiPhoneの通常のSDKを時までリリースしないというAppleの決定です。 そして、この決定の結果： Appleはすでに iPhoneだけでなく通常のApple Safariでも動作する200以上のWebアプリを受け取っています 。 Web2.0-SoftはMS Windowsに結び付けられていないため、これはすべてMicrosoftの独占を弱めるために意図的に行われます。

時代は変化しています！ Intel＆Web 2.0アプリ用のApple MacOS Xです;）
