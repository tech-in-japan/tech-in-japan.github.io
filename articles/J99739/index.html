NVIDIAが新しい並列コンピューティングCPUおよびGPU製品を発売





NVIDIA は、新しい商用製品である NVIDIA Parallel Nsightを発売します 。これは、 Visual Studio 2008 SP1と統合されたツールスイートです。 NVIDIA Parallel Nsightは、マルチコアプロセッサと最新のGPUアクセラレータの全機能を使用して、並列アプリケーションを開発するためのツールを提供します。











新製品の機能は次のとおりです。
  • 数千の並列スレッドを表示する機能を備えた強力なデバッガ。
  • GPUでコードを直接デバッグします。
  • Visual StudioでGPUメモリを直接表示する
  • CUDA C / C ++メモリチェッカーツールを使用してメモリリークを検出します。
  • API呼び出しのキャプチャ、カーネル起動、メモリ交換を備えたコードランタイムイベントアナライザー。
  • Visual Studioウィンドウでのすべてのイベントの詳細な視覚化。
  • レポートツールを使用したイベントのフィルタリングと並べ替え。
  • GPUパフォーマンスカウンターを使用したCUDAプロファイリング。
  • GPUで直接HLSLシェーダーをデバッグするグラフィカルツールとデバッガー。
  • グラフィカルまたはシェーダーアプリケーションのデバッグ。
  • directX呼び出し検出を備えたリアルタイムのグラフィカルインスペクター。
  • GPUパイプラインの状態のインタラクティブなチェック、テクスチャ、ジオメトリ、計算バッファの視覚化。
  • ボトルネックとCPU消費を特定できるプロファイラー。
NVIDIA Parallel Nsightは、 GeForce 9ファミリ、 200および400シリーズのビデオカードをサポートしています。 要件の詳細については、 このページを参照してください 。



NVIDIA Parallel Nsightは、 StandardとProfessional （アナライザーを含む）の2つのバージョンで利用できます。 これまでのところ、予備バージョンのリリース時に、 Parallel Nsight 1.0 Professional をダウンロードできます 。



製品のライセンス情報は、 このページにあります 。 注：アカデミックプログラムの参加者は、ツールのライセンスを無料で取得できます。



時間の経過とともに、ツールはVisual Studio 2010の更新プログラムとサポートを受け取ることが予想されます。
