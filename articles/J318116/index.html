さらに3つのオープンなMail.Ru Groupオンラインコース





12月5日から12月15日まで、Mail.Ru Groupの教育部門は、 テクノパークでのレッスンに基づいて、Stepikプラットフォームで記録されたいくつかのコースを期限なしで開きました。 ポイントを失うリスクなしに、あなたにとって便利なモードでそれらを保持することができます。 ゆっくり勉強していても、最高点の証明書を取得する機会があります。



12月5日以降、次のコースがパブリックドメインに登場しました。



  • C / C ++でのマルチスレッドプログラミング 。 このコースでは、ネットワークおよび高負荷システムの開発というコンテキストでのマルチスレッドプログラミングの原理について説明します。 コースの中核は、プロセス間通信（IPC）とスレッド同期トレーニングです。 コースを修了すると、独自のWebサーバーを開発できます。これにより、習得したスキルを習得できます。
  • Hadoop。 大量のデータを処理するシステム 。 このコースは、Hadoopシステムを使用して大量のデータ（BigData）を処理する方法に焦点を当てています。 コースを修了すると、基本的なストレージ方法と大量のデータを処理する方法の知識が得られ、Hadoopフレームワークのコンテキストでの分散システムの原理を理解し、MapReduceプログラミングモデルを使用して実用的なアプリケーション開発スキルを習得できます。


12月15日以降、 Webテクノロジーに関するコースが開かれました。これについては、最近別の記事を作成しました 。 さらに、オープンアクセスで利用できる次のオンラインコースがあることを既に書きました。



  • Webプロジェクトのセキュリティ分析 -Webプロジェクトのセキュリティ監査テクニックに関するクラッシュコース。
  • JavaでのWebサービスの開発、パート1およびパート2では、最初にWebアプリケーションのサーバー側の開発の基本を学び、次にWebサーバーを産業レベルまで開発します。


あまりにも多くのコースがあり、このための時間があまりない場合は、どのコースがあなたに最も適しているかというトピックのテストを受けることができます！ 実際、このテストは若い専門家向けのNewtonewとの共同プロジェクトの一部ですが、経験豊富な開発者が決定するのに役立ちます。 しかし、彼をあまり信頼してはいけません。 何らかの形で、私たちはあなたを参加に招待します ！
