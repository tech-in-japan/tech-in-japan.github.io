モノブロック（タブレット）Dell XPS 18のビデオレビュー





Dell XPS 18-さまざまな品質のDell XPSファミリーの新しいオールインワンタブレット。 Dell XPS 18には、豪華な18.4インチIPSタッチスクリーンFullHDディスプレイ（1920x1080）が搭載されています。 Dell XPS 18のハードウェアベースは、第3世代の低電圧Intel Coreプロセッサ（i5またはi7）またはPentium Dual Coreで、4〜8 GBのRAMでサポートされています。 HDD（320または500 GB）は、32 GB SSDがインストールされ、キャッシュに使用されるカップルのデータストレージ用です。



モノブロックには2つの折りたたみ脚があり、スタンドホルダーが装備されています。 これにより、ユーザーはスクリーンの角度とテーブル表面上のリフトの高さを調整できます。



レビューからのモデルの詳細な特性：



プロセッサー：Intel Core i7 2000 MHz Ivy Bridge（3537U）

RAM：8192 MB DDR3-1600 MHz

画面：18.4 "IPS（LED）ミラー（グレア）タッチスクリーン

解像度：（1920x1080）フルHD

ビデオカード：Intel HD Graphics 4000、64 + 1632 MB

サウンドカード：Waves MaxxAudio Pro

ハードドライブ：500 GB（5400 rpm）、32 GB（300 MB / s）HDD + SSD

光学ドライブ：いいえ

ワイヤレス接続：Bluetooth 4.0、WiFi（802.11 b / g / n）+ WiDi

ポート：2xUSB 3.0、ラインアウト/マイクイン、カードリーダー（SD / MMC / MS / Pro / xD）

重さ：2.230 kg

ケース（長さx幅x高さ）：436x283x18 mm

ソフトウェア：MS Windows 8（64ビット）

オプションのアクセサリ：ワイヤレスマウス、ワイヤレスキーボード

保証：1年間のメーカー保証
