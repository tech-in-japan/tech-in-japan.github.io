Intel Core i7-860スタッフィングを使用したサーバーのテスト

そのため、もう一度hostkey.ruでCore i7-860に基づいてサーバーをレンタルするように命じられ、典型的な高性能ユニプロセッサプラットフォームについてメモすることにしました。



ASUS RS300-E6 / PS4を愛してください。 これは、1156ソケットを備えたマザーボード上の深さ600 mm、4つのホットスワップ3.5インチSATAドライブのシングルユニットマシンです。キットには伸縮レールとプロセッサヒートシンクが含まれています。









DDR3メモリは最大32GBまで供給でき、内蔵SATAはRAID 0、1、10をサポートします。 別のイーサネットポートを備えた統合KVMを配置できます。 マザー2ギガビットイーサネット。



IntelおよびSuperMikrovskyサーバーはXeon 34シリーズのみをサポートし、これもCore i7です。



すべてが非常にコンパクトで、きちんとした、6つのファン、350Wのグリーン電源です。 興味深いことに、ヒートパイプを備えたプロセッサヒートシンクが使用されます。







カスタム構成では、4GBのKingstonメモリがありました-2GBのストリップ2個、永久保証。 サーバー専用のハードドライブ-2台のWestern Digital RE3 SATA2 500Gb、7200 RPM 16Mbキャッシュ、3年間の保証。







プロセッサは新しいIntel Core i7-860 2.8GHzで、4コア、8スレッド、ハイパースレッド、8 MBキャッシュを備えています。 シングルスレッドアプリケーションでは、Turbo Boostテクノロジーがトリガーされます。1つのコアの周波数を3.46 GHzに上げると、すべてをすばやく計算できます。 64ビットなどのすべての新しい仮想化テクノロジーをサポートします。 プロセッサの価格は350ドルです。



サーバーにWindows Server 2008をインストールし、 Passmarkテストを使用してパフォーマンスと電力消費をテストしました。ディスクはRAID 1にありました。







全体として、システムは前のテストのCore2Quad 2.66に基づくサーバーよりも生産性が30％高く、RAIDアレイからの読み取り速度は他のプラットフォームの2倍以上のパフォーマンスです。 C2D / C2Qと同じレベルでメモリを操作します。



システムの消費電力は、スタンバイモードで0.5A（100ワット）、ストレステストで0.8A（275ワット）でした。



このプラットフォームは、ロードされたアプリケーションサーバーとして、大きなポータルをホストするため、および高いプロセッサパフォーマンスが必要であり、メモリおよびディスクサブシステムの速度に対する高い要件がないブロードキャストプロジェクトに使用することをお勧めします。 サーバーアプリケーションがマルチスレッド操作に最適化されておらず、シングルスレッドモードで最大のプロセッサ速度が必要な場合は、Core i7ベースのサーバーを使用する必要があります。



4つの3.5インチディスクにより、RAID 10モードで最大8TBのディスクスペースを整理できます。最大4TBの場合、1Uの場合はそれ以上の作業はできません。 このプラットフォームでは、SASディスクを同じ方法で使用できます。個々の構成を注文する際にSASディスクを使用します。これには通常のソリューションと同じ時間がかかります。



このプラットフォームは、シングルプロセッササーバーのHostkeyラインの最上位にあります。現時点では、Core i7-960（700ドル）を使用した場合にのみ高速化できます。 次のパフォーマンスステップは、Xeon E5530（$ 650）に基づいた2プロセッサマシンで実現されます。これらについては後で検討します。
