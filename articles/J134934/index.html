Lingvo for Androidに会う

私は一般的にAndroidに同情しているという事実にもかかわらず（Symbianに電話を持っている間）、iOSと比較してその重大な欠点の1つを理解しました。 その下にはLingvoはありません！ しかし、トピックの名前を読めば、Androidフォンを安全に使用できるようになったことがわかります*。 そのため、 Windows 、 Mac OS X 、 iOS 、 Symbian、Windows Mobile、さらには電子書籍のバージョンに続いて、 Lingvo for Androidを紹介 します 。



最も決定的なのはこれがあります



そして、最初に考える必要がある人は、猫をお願いします。



つまり、$ 2.99のアプリケーションがあります。 その主なタスクは、特別に購入した辞書を便利に表示することです。 最初は辞書がありません。次に、それらをダウンロードします（Lingvoには無料でダウンロードできる辞書があります ）か、購入します。







アプリケーションはそれらから言語の指示を形成し、ユーザーが選択した方向の語彙を生成します。 もちろん、ボキャブラリーを目的の単語までスクロールし、ハーフタイムを許可したユーザーの欲求を推測することで、文字の入力にすばやく反応する検索文字列があります。







目的の単語が見つかったら、ユーザーはそれがどのように翻訳されているか、使用例を参照し、この単語の形式の表を見ることができます。







これはすべて、ある程度カスタマイズされていますが、カスタマイズされていません。







まあ、それは基本的に、要するに:)



製品は、Lingvo for iOSで強調表示された写真の翻訳、音声入力のサポート、リーダーとの統合など、時間の経過とともに有用な機能を獲得します。 これは少なくとも:)一般的に、 Androidマーケットへの襲撃を提案します。



追伸まず、新しい製品を紹介するためにブログリーダーに電話をかけたかったのですが、新年の騒ぎはすでに始まっていて、多くの人がそれを不便だと思っていました。 しかし、そのアイデアは生き続けており、今は1月に小規模なモバイルパーティーを開催する予定です。 ブログをフォローしてください。近日中に、携帯電話やモバイルアプリケーションについて話したい、質問したり、一般的に話し合いたいと思うクッキーを使ってお茶をお召し上がりください。
