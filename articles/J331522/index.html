SaaSプロバイダーのZextras実装結果

この記事では、政府、教育機関、医療機関へのサービスの提供を専門とする欧州のSaaSサービスプロバイダー向けのZimbra Open Source EditionおよびZextras Suiteの実際の実装例について説明します。 独自のクラウドソリューションを提供し、200を超えるドメインで21,000を超えるメールボックスを提供しています。 エンドユーザーにハイテクで安全な製品を提供し、信頼性の高いモバイル同期を提供し、ファイルストレージを管理し、マルチテナンシーを提供することを望むITインフラストラクチャを近代化するために、Zimbra Open Source EditionとZextras Suite冬ログのフルセットを導入することが決定されました。





Zimbra Open Source Editionは、高性能PIM /コラボレーション機能を備えた信頼性の高いメールサーバーです。



Zextras Suite-リアルタイムバックアップの実行、ボリュームの管理、モバイル同期のセットアップ、インスタントメッセージング、および管理委任が可能なモジュール式の拡張機能。



ZimbraとZextrasの導入には、いくつかの段階がありました。



  • 近代化計画の作成（データバックアップ、新しいストレージ構成）;
  • ソフトウェアのインストール。
  • バックアップストレージ、新しいライフストレージ構成の作成。


実装のすべての段階を実装した後、最終的に何をしたかを示したいと思います。



インフラストラクチャ設計：プラットフォーム



パフォーマンスを改善するために、パートナーは15台のサーバーのZimbra Collaborationマルチサーバーをインストールしました。 すべてがAmazonのEC2でホストされています。

ストレージを使用する



Zextras BackupとZextras Powerstoreの圧縮と重複排除により、極端なボリューム最適化が可能です。 マイボックスの「ユーザー」クォータの合計は15.5TBでしたが、メインボリュームの実際の合計サイズは1.8TBで、セカンダリの実際の合計サイズは9TBです。 これは、ディスク容量の30％を節約することを意味します。 セカンダリボリュームには、90日間以上保存され、Amazon S3ユニットデータとして保存されるEメールが含まれ、ストレージコストが大幅に節約されます。 バックアップには合計9TBが必要で、削除後32日を節約します（非線形成長を維持するため、この特定のケースでは、バックアップサイズを2倍にせずに90日まで増やすことができます）。



インフラストラクチャの設計：ストレージ



データはAmazon EBSドライブに保存されます。 Amazon EBSでは、ストレージボリュームを作成し、それらをAmazon EC2インスタンスに接続できます。 ボリュームをボリュームに接続したら、ファイルシステムを作成したり、データベースを実行したり、他の方法で使用したりできます。 Amazon EBSボリュームは、個々の機器の故障から情報を保護するために、自動的に複製される特定の可用性ゾーンに配置されます。 すべてのタイプのEBSボリュームには、同等の堅牢なスナップショット機能があり、99.999％の可用性を提供します。



ストレージは、ストレージの階層化とAmazon EC2 EC2ブロックの互換性の両方を提供するZimbra StoreとZextras Powerstoreを通じて管理されます。 Zextras Powerstoreモジュールを使用すると、Zextbraコントロールパネルに完全に統合されたZextras管理Zimletを通じて複数のボリュームとHSMポリシーを管理でき、効率的なストレージ管理が可能になります。



  • ローカルドライバーのプライマリボリューム（GP2 General Purpouse SSD-Amazon EBS）
  • Amazon S3ブロックのセカンダリボリューム
  • ローカルドライバーでのバックアップ（GP2 General Purpouse SSD-Amazon EBS）
  • 圧縮：プライマリ：NO | セカンダリ：自動| バックアップ：はい
  • 重複排除：プライマリ：断続的| セカンダリ：自動| バックアップ：はい

SaaSプロバイダーの顧客は、多くの場合、ITスタッフがドメインを構成する権限を持っていることを要求します。 これは、Zextras Adminの越冬の管理委任を使用して実装されました。 同じモジュールを使用すると、パートナーとの契約に基づいてドメインの制限を決定し、欧州法の枠組みで委任管理者のアクションを簡単に表示できます。
