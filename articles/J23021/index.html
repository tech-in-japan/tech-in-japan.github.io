GoogleがDBMS開発者向けにBigTableをオープン

Googleが来週、開発者向けにBigTable DBMSを開く予定であることがわかりました。 もちろん、このイベントはAmazon Web Services（AWS）への挑戦であり、Amazon SimpleDBとの刺激的な競争の始まりとみなすことができます。



BigTableは、 Google File System （GFS）に基づくスケーラブルなデータベースシステムです。 Googleでの内部使用を考慮して、膨大な構造化データを格納および管理するように設計されています。 システムには、数千のGoogleサーバー上の数百テラバイトの情報が含まれています。 BigTableの開発は2004年に開始され、2005年2月に積極的に使用されました。 GoogleがBigTableを使用するアプリケーション-Google検索、Google Earth、Googleマップ、Google Finance、Googleプリント、Orkut、YouTube、Bloggerなど



GoogleはBigTableのコストをまだ発表していませんが、Amazon Web Servicesと同じ価格設定モデルを使用する可能性があります。 プラットフォームの開発が継続され、開発者のニーズのスケーラビリティを解決するタスクが完了することは間違いありません。 3つの補完的なAmazonサービス（ Amazon SimpleDB 、 S3 、 EC2 ）の場合のように、製品はWebスタートアップを構築するための技術インフラストラクチャとして位置付けられています。
