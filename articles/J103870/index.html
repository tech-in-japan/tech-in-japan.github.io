マイクロソフトは主要なボットネットの制御を獲得しました

バージニア州の裁判所の判決によれば、 マイクロソフト は 、以前にWaledacとして知られるボットネットのコアを形成していた276ドメインの制御権を与えられました 。







Waledacは、特別なトロイの木馬に感染したコンピューターを使用してスパムを送信するボットネットです。 ボットネットは2009年にピークに達し、ネットワークは毎日最大15億のスパムメッセージを送信しました。 Microsoftは、 Hotmailが最大6億5,000万件のスパムメールを記録したことを指摘しています。



ボットネットの制御とその無効化を取得した後、マイクロソフトは感染したコンピューターの流通を記録することができました。 そのため、7日間、58,000台のコンピューターがボットネットへのアクセスを試行し、中和されたドメインへの接続試行が1,460万回を超えました。



マイクロソフトは、感染について顧客に通知できるプロバイダーと協力しています。 そのため、Cox Communicationsのプロバイダーは何百もの顧客に連絡して、コンピューターが感染したことを報告しました。 システムからトロイの木馬を削除して感染を防ぐには、無料のMicrosoft Security Essentialsウイルス対策ソフトウェアをインストールするだけで十分です。



ボットネットの破壊の詳細については、Microsoft Webサイトのこの記事を参照してください。
