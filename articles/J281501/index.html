Project Tofino-Mozillaの新しいブラウザ



新しいブラウザのデザインと機能のコンセプト。 クリック可能



Mozillaは静かに、大騒ぎせずに今週の金曜日に新しいプロジェクト-Project Tofinoの立ち上げを発表 しました 。ProjectTofinoは既に「Firefox killer」と呼ばれています。



そのような意見にはあらゆる理由があります。 近年、Firefoxは急速に市場シェアを失い、Google Chromeの猛攻撃に耐えることができません。 Project Tofinoは、Mozillaからブラウザの外観を根本的に変更して更新し、それによって失われた市場シェアの「戻り」を開始するように設計されています。 これまでのトフィーノチームはわずか6人で構成されています。 会社の新製品の設計と設計に携わるのは彼らです。



Firefoxプロジェクトの上級副社長であるマーク・メイヨは、「Firefox 1.0の時代から、パーソナルコンピューターとモバイルデバイスのブラウザーは大きく変わりました」と述べています。



プロジェクトTorfinoは、 Firefoxが構築されているGeckoエンジンを放棄し、 Chromiumに基づくElectronプロジェクトに目を向けました。 同時に、Mozillaはhelisを埋めるつもりはなく、引き続き古いブラウザのサポートと開発を続けます。 しかし実際のところ、Firefoxは現在の形式でChromeに追いつくことができないため、Project Tofinoがリリースされました。



UPD1。 一般的に、 vxd_devとvedro9が 以下のコメントで指摘したように、MozillaのリーダーシップはProject TofinoをUIを試すためのプラットフォームとして位置付け、リリースに関してはMozillaネイティブのGeskoエンジンに戻ります。 プロトタイピングツールとしての使いやすさから、ChroniumベースのElectronが選ばれました。
