著者：エレナ・フェドロワ



私の名前はエレナ・フェドロワです。職業は医者ですが、たまたま10年以上ITに携わっていることがあります（とても嬉しいです）。 ヴォロネジのDataArtの人事マーケティング責任者。 さらに、8年以上にわたり、私はGoogle Developer Group Voronezhの技術コミュニティを調整し、ITイベントを開催してきました。 だからこそ、マウンテンビューでGoogle I / O 2016に参加しました。



流行に敏感なアンドロイドがゲストを迎えます。

これが私の2番目のGoogle I / Oでした。 私の最初の会議に関する熱狂的なレポートは ここ で見ることができ ます 。

原則として、誰でもポケットに900ドル（チケットの価格）のI / Oを取得できます。 ただし、無料で会議に参加できます。記事の下のライフハックを読んでください。



ミラーリングされた要素を持つ巨大な会議ロゴは、おそらく写真撮影専用に発明されたものです。



技術的な詳細と完全なキーノートエントリは、I / O Webサイトで最適に表示されます 。 単純な技術者として私に感銘を与えたものをお話しします。

大西洋を飛行する人々は、朝が早く始まることを知っています。 そこで、登録の最初からバッジを取得しました。 驚いたことに、キューは見つかりませんでした。







キューは後で現れ、どこにでもありました。 7000人以上が会議に来たときに彼らなしで。 最初は基調講演のための大きなキューがありましたが、これは正常です。 誰もが最も重要なニュース、特にクレイジーなAndroid開発者をいち早く聞きたいと思っています。 Googleコミュニティのメンバーとして、キャノピーの下のステージ近くのエリアに収容される最初の夜（取り消し線）の権利が付与されました。 その後、セクションの行がありました。参加者は、プレゼンテーションの開始の数時間前にレポートの行を取りました。

プログラミングゾーンCodeLabは、実際のデバイスの実際のGoogleエキスパートの助けを借りて、プログラミングや新しいテクノロジーの使用方法を学びたい人のための特別なセクションです。







各セクションの隅々に、多数の質問に答え、デバイスのデモを行い、新しいテクノロジーの機能を備えたGoogleの専門家がいたことに注意してください。



機械学習の可能性を実証した面白いロボットアーティストは、常に観客の群衆を集めました。 一番下の行は、アプリケーションがスマートフォンにインストールされ、そこで仮想キューブキャンバスをひねり、ペイントカラーを選択し、ロボットアームがペイント付きブラシを選択できることです。 次に、スタンプを振る必要があります。ロボットブラシは動きを正確に繰り返します。 その後、ロボットは、まるであなたをダンスに誘うかのように、野生の勝利のダンスを踊ります。 一般的に、それはクールです。



人の顔の写真を一列に描かれた絵に変換する面白いプロジェクト。 すべての祖母は、セーターを編むためのモックアップを作成するための公開を待ち望んでいます。



音楽プロジェクトはたくさんありました。 たとえば、スマートフォンの樽型オルガンのようなものです。回転する円にメモを付けた紙を貼り付けると、スマートフォンがそれを認識し、樽型オルガンがメモを再生します。 回転するサークルとスマートフォンをたくさん集めて、面白い作品を手に入れよう。
