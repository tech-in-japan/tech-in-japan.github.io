テスト中



いつものように、テスト済みのビデオカードの「総」パフォーマンスを判断するために、合成テストから始めます。



3Dmark 2003

HD 5670はすぐにGT240から外れ、このレベルのパフォーマンスにとってギャップは非常に重要です。 CrossFireはパフォーマンスの大幅な向上を達成しましたが、1枚のビデオカードのパフォーマンスの50％を超えることはできませんでした。







3Dmark 2005

3DMark 2005でも状況は変わりませんでしたが、GT240に対抗するHD 5670のライバルは、以前のベンチマークよりも大幅に少なくなっています。







3DMark 2006

競合他社のビデオカードのパフォーマンスの違いは3DMark 2006でも同じレベルのままでしたが、CrossFireは予想外に驚き、約90％の増加で感銘を受けました。 最新のグラフィックステクノロジーにより、2番目のビデオカードをコンピューターに追加することによるメリットが増えたようです。







3Dmark Vantage

また、Vantageは、2番目のビデオカードからの成長の増加に注目していますが、GT240とHD 5670の違いは揺るぎないままです。







ゲームテスト



今回、ゲームは私にとっても非常に興味深いアジェンダ項目です。これらの「3Dノーム」が何ができるかを見るのは常に興味がありました。 今回の設定の選択では、各ゲームに最適な設定を選択するために、非常に繊細で真剣な予備テストが必要でした。 最後に、可能な限り、2つの詳細レベルで1680 x 1050の解像度で停止することにしました。 したがって、最終的にはこれらの膨大な量のビデオカードはベンチマークモンスターではなく、単にプレイしたいユーザーによって購入されるため、結果をプレイ可能なレベルに近づけるいくつかの設定を選択しようとしました。



Farcry2

まあ、FarCry2から始めることを疑う人は誰でも-Dunaは現時点で最も先進的なグラフィックエンジンの1つです。 残念ながら、DX10モードで2つの詳細レベルを選択することはできませんでした。これらのレベルを上げると、テスト結果がブロックされたり、フレームレートが低すぎたりしました。 そのため、アンチエイリアシング（アンチエイリアシング）なしのHihgモードのすべての設定で、DX10のプリセットを使用しました。 火、物理学、および木は低く設定され、テクスチャは中程度に設定されました。 Hihgでこれらの設定でテストを実行しようとすると、テストがクラッシュしました。 これはおそらく、パフォーマンスを制限する設定でも発生するドライバーの問題です。







競合するビデオカードの違いはわずかであることはすぐに明らかです。 はい、1秒あたり3〜4フレーム-45 fpsで約8〜10％ですが、このレベルではこの差は通常のゲームプレイには影響しません。 CrossFireがもたらしたモンスターの成長は非常に興味深いので、2番目のビデオカードを追加すると、ゲーム体験が大幅に変わる可能性があります。



そして最後に、これからは、2.93 GHzの2つのプラットフォームCore i3 530と4 GHzのCore i7 870を比較することはおもしろいです。



トムクランシーズHAWX

DX10.1の実装のおかげで、HAWXは常に非常に興味深いベンチマークであり、AMDビデオカードはこの問題で有利でした。 まあ、今ではNVIDIAビデオカードもDX 10.1をサポートしているので、最終的に何が得られるのかに非常に興味がありました。 HAWX（他のゲームと同様、FarCry2を除く）では、1680 x 1050の解像度で2レベルの品質設定を使用することができました。したがって、最初に高詳細および二重スムージング（AA x2）でベンチマークを実行し、詳細設定（低/中）およびアンチエイリアシングは無効です。 AMDとNVIDIAの違いは、ペンセットの設定では非常に大きくなりますが、テストを詳細に実行しても実質的には重要ではありません。 繰り返しになりますが、CrossFireの使用は大幅に増加していますが、2つのプラットフォームに違いはありません。







バイオハザード5

バイオハザード5はストリームを維持します：AMDとNVIDIAの差の約10％、マルチカード構成は50％を「シュート」します（スムージングを伴う高詳細モードを除き、つまずきます）。 繰り返しますが、在庫のCore i3とオーバークロックされたCore i7に違いはありません。







ストーカー：プリピャチの呼び声

Call of Pripyatは、DX11を使用した最初のゲームの1つで、ユーザーに刺激的な視覚体験を提供します。 興味深いことに、Call of PripyatはGT240が勝利を祝った唯一のゲームです。 高い設定では、GT240はHD 5670よりもさらにギャロップすることができますが、ディテールが低い場合でもパフォーマンスは同じです。 そして、最初の3つのテスト（昼、夜、雨、太陽軸）で両方のビデオカードが同じ結果になり、4日目にはHD 5670が1.5 fpsを獲得し、GT240が5 fpsで良い問題を経験することはすでに非常に面白いです。

CrossFireの増加はわずかに小さく、約30％です。また、標準のCore i3とオーバークロックされたCore i7の違いは明らかに存在しません。







温度



私は、テストの最初から最後まで、テスト中にデバイスが到達する温度に問題はなかったと言わなければなりません。 AMD参照カードは、オーバークロック中にファン速度39％で69°Cに達し、一般的な設定でファンは38％で67°Cに達しました。 HISのモデルは同様の動作をし、オーバークロック中はクーラーの39％で70°Cに達し、標準設定では最大速度の37％で68°Cに達しました。 明らかに、2つのソリューションの根本的な違いはノイズレベルです。これは、HISの方が大幅に低い値です。 GT240は、一般的な傾向からも目立ちません。Gigabyteの非基準冷却システムのおかげで、35％低い温度で72°Cを示し、非常に低いノイズレベルを示しています。 最初は彼を笑ったが、バットマンは本当に良い仕事をしている！

明らかに、このような消費電力（61W）を備えたHD 5670は、HTPCやサイレントコンピューターのファンに最適です。 もちろん、ファンレスソリューションには時間がかからず、完全な静寂を見つけることができます。



使用済みCPUの影響



元々意図していたものすべて（さらに弱いプロセッサー-Intel Core 2 Duo E4300）をテストすることはできませんでしたが、2つのプラットフォームを比較することができました：Intel Core i3 530 + DDR3 1333 MHz CL9とIntel Core i7 870 + DDR3 2000 MHz CL9-そして結果は刺激的です。 予算が強力なグラフィックスソリューションの購入を許可しない場合、プロセッサは、たとえばHD 5670などのパフォーマンスに影響を与えません。システムは、より高価なビデオカードの場合のように、CPUの周波数ではなく、計算能力によって制限されます。 したがって、Core i3ベースの比較的安価なプラットフォームと最新のAMD Radeom 5シリーズのビデオカードを購入することで、大幅な節約ができます。 つまり、フルHDビデオを再生でき、許容できるノイズレベルを持つ中程度の詳細設定で1680 x 1050の解像度で再生できるシステムは、わずか700ドルで購入できます。 個人的には、これは非常に良い買い物だと思います。 明らかに、低解像度でゲームを実行するとプロセッサの影響が大きくなりますが、この場合、DX 11をサポートするビデオカードを取得する意味はあまり明確ではありません。



結論



それで、私たちは3Dノームを使った冒険の終わりに到達しました、そしてそれは線を引く時です。 素材の冒頭で表明された立場にとどまるために、あなたはどう思いますか？ または、HD 5670の成功を認めますか？ 私の個人的な意見は、すべてが価格に依存しているということです。 100ドルに近ければ近いほど、HD 5670はお金の価値があり、同じ価格帯の競合他社よりも有利であると確信できます。 ニュースは勇気づけられます-オンラインストアで受け取った最初のバッチは、112〜119ドルの範囲の価格で販売されます。 間違いなく注目に値する特定の製品はHIS HD 5670です。これは、私の意見では、Arctic Cooling Silencerが同じ価格で付加価値を与え、適切なノイズレベルを提供するためです。 おそらく、他のメーカーの同様のソリューションが表示されますが、これまでのところ、HISだけがデバイスを供給する用意があることを発表しており、価格を許容できる低レベルに維持しようとしています。



最後に、 HD 5670が私の期待を超えたことを認めなければなりません-1680 x 1050でのゲームのパフォーマンスはかなりまともです。 ただし、この解像度でより優れた視覚体験を得るには、もう少し力が必要です。 生産性が少し高くなり、価格が約95ドルになると、このモデルはこの価格セグメントで絶対的な「ベストバイ」になります。 CrossFireの2つのHD 5670の決定は非常に興味深いものです。 問題は、競争の欠如のために価格戦略を変更することだけです。その結果、HD 5670はその祖先とは異なる位置にあります。 したがって、製品の価値を高めるDX 11、EyefinityおよびHD再生に重点が置かれていますが、ゲームのパフォーマンスはRadeon HD 4850のレベルにとどまり、メーカーが推奨価格119ドルで販売しています。 そのため、AMDは、従来の位置を変えた製品のゲーム性能を備えた新しいレベルのテクノロジーを提供します。 私は個人的にX650製品の価格の上昇に感謝することはできませんが、これは一部のメーカーが実践している絶え間ないブランド変更よりもエレガントなソリューションであることを認めなければなりません。



AMDは現在、100ドルから770ドルまでのフルレンジの製品を有しており、エントリーレベルのセグメントの競争に勝ったと結論付けることができます。 もちろん、新しい競合製品を発売することで、「グリーンキャンプ」は新しい価格戦争を引き起こし、必要な価格圧力を生み出し、現在よりも価格を改善することができます。 非常に安価なソリューションのリリースが準備されている限り、HD 5670は、市場でDX 11をサポートする最も安価な製品です。
