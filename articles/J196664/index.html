Java School：概要

Habrchane、知り合いになろう！ 私たち-Changellenge >>（Change and Challengeという言葉から）-ロシアでの実践指向の教育の開発に取り組んでいます。 これを行うために、Changellenge >>という名前で一連のケースチャンピオンシップを実施し（年間約20,000人が参加しています）、また、大企業が教育イベントを開催するのを支援しています。 今日は、第2の方向性についてお話します。ドイツ銀行学校の若いJavaプログラマー向けのトレーニングプログラムです。ちなみに、このプログラムは現在募集中です。



学校に関するすべての情報は、インサイダー、すなわちユーリ・ライマン（バウマンモスクワ州立工科大学、情報学および制御システム学部）およびアンドレイメルロフ（モスクワ州立工科大学スタンキン校、専門応用数学および経済学の高等学校、経済分析の数学的方法）、ジャワスクールの最初のセットの卒業生によって提供されましたインタビューの形で。





学校について簡単に：

Javaスクールは、才能のある候補者をJava言語でゼロから訓練し、専門家としてドイツ銀行開発センター内で成長させるために作成されました。

6か月ごとに、15〜20人の生徒が学校に募集されます。 週5日すべて学校で勉強する必要があるため、最近の卒業生または学部生に最適です。 トレーニング中、各生徒は10万ルーブルの給与を受け取ります。 学校の終わりに優秀な学生がドイツ銀行開発センターで働くように招待されています。



プロセス：

Java Schoolでの1日はどのようなものですか？

トレーニングの段階に応じて、さまざまな方法で。 最初の段階では、ドイツ銀行で一日を過ごしました。私たちは自分のコンピューターやプロジェクターなどを持っている聴衆がいました。第2段階では、実験室の仕事をしたとき、私たちは主に自宅で仕事をして話しに来ました監督者と話し合い、結果について話し合います。 オフィスでより快適に働く人は、一日中オフィスで過ごすことができます。 そして、第三段階はほとんどフルタイムの仕事でしたが、同時により統一された雰囲気にありました-その日、私たちはいくつかの相談やセミナーをしました。つまり、毎日3-4人の新しい人に会いました。
Javaスクールでの学習と大学での学習を組み合わせることはできましたか？

5月3日にJavaスクールを卒業し、6月3日に卒業証書を擁護しました。 卒業証書を書くことができたのは土曜日と日曜日でした。 負荷はかなり均等に分散され、仕事に効率的に近づくと、与えられた宿題はすべて平日の夕方に完了することができ、週末は卒業証書を書くことができます。


選択：

最も難しい学校への選択はどうでしたか？

プログラミングのテストやその他のタスクは個人的には何の困難も引き起こさなかったので、銀行員とのコミュニケーションを想定しています。
あなたが他のインタビューで会わなかった選択の間に、何か明るくて思い出深いものはありましたか？
まあ、かなり面白い仕事がありました、私は論理的な定式化の観点から意味します。 たとえば、目の前にいくつかのバスケットがあり、それぞれが正しい名前の果物の入ったタブレットを拾わなければなりませんでした。 これを解決するように頼まれたとき、これは私の脳を震わせましたが、私はすぐに管理しました。





トレーニングと教育の段階：

トレーニングの最初の段階は、Javaプログラミングの基礎です。 このステージはどのくらい続き、その後何が起こったのですか？

最初のステップは「Getting to Know Java」と呼ばれていました。 外部の会社の先生が読んでくれました。 彼はJava言語を含む教育を専門としているため、明確なコースがあり、すべてが完璧に構築されました。 面白かった、フィードバックがあり、毎日の回顧展がありました-非常にクールでした。 コースを修了した後、私たちは多くの実験室作業を行い、2週間割り当てられました。 それぞれには、開発センターの従業員である彼自身の監督者がいました。 したがって、私たちはすぐに将来のマネージャーと知り合いになりました。 2週間の実験室作業の終わりに、彼女は弁護された。 別の3週間のコースがありました。Javaに関する詳細なトピックと、ドイツ銀行で開発中のプロジェクトのプレゼンテーションが既にありました。
上級コース後のプロジェクトへの割り当てはどうでしたか？
Javaスクールで勉強している間、教師は私たちの仕事を見ました。 彼らは専門的に彼らにより適した人を選んだ。 したがって、Java Schoolを終えると、チームに到着します。
上級コースの特徴は何ですか？
この時点で、より狭い問題が調査されました。 このコースは、ドイツ銀行開発センターの従業員によって教えられました。 私は特に少数の人々を覚えています：アンドレイ・タラノフは読んで、アレクセイ・ラゴジンはガベージコレクションについて非常によく読みました。 実際、これらのトピックは興味深いものであり、私たちにとって非常によくカバーされていました。また、3か月以上にわたって実践がすでに示しているように、この知識は有用であることがわかりました。


直接のスピーチ：

これまでのインタビューでは発見できなかった、主題に関するいくつかの非定型的な考えがあります。

Javaスクールの終わりにどのようなプロジェクトに参加しましたか？
外部の人に説明するのは困難です...プロジェクトは、高い生産性で接続されています。つまり、1秒あたり50,000件のアプリケーションを処理するプログラムを開発しています。
バウマンとして、現代の学術教育とJavaスクールの教育を比較できますか？ あちこちで100パーセントの割合で有用な知識が見つかったとしますか？
これらのことは非常に異なると思うので、パーセンテージで比較することは非常に困難です。 Fiztehは私に基礎教育を与え、非標準的で複雑な問題を解決し、大量の情報から適切なものを選び、そしてもちろんすべての基礎知識を習得しました。 Java Schoolは、より具体的で狭い知識、日常業務で使用されるいくつかのツールを提供しました。 私は、一方が他方なしでは存在できず、お互いに支えられていると信じており、両方の可能性に感謝しています。
些細な質問ではありません... habrasocietyにとって、Java Schoolでの学習中に気づいた不利益を見つけることは有用でしょうか？
短所はありませんでしたが、むしろ何らかのヒッチがありました。 最初の週には、教師はあまりにも単純な質問に答えることが多かったので、時間はあまり合理的に費やされなかったように思えますが、いくつかの情報源を参照することもできます。 したがって、最初の1週間で、私たちは少し時間を失ったと思いますが、すぐにこれに気付き、残りの2週間で追いつきました。
生徒ごとのトレーニングのレベルはどの程度異なりますか？
特に、私はJavaをまったく知りませんでしたが、開発の経験があり、原則として、この一生のプログラミングとデバイスを理解していました。 一般的に、レベルはまったく同じではなかったようです-遅れている人とわずかに優れた人がいました。
改善できるものが他にあったのでしょうか？
MixのC ++でこの種のコースを受講し、学生の熱意を高める評価システムがありました。 結局のところ、私たちは皆、リーダーの一人になりたいと思っています。 そして、私たちがそうだとわかると、私たちはランキングで遅れています、はい、私たちはすでに最後のランクにいます、これは私たちに拍車をかけます。 そして、給料や奨学金を受け取ったばかりのとき、あなたは何のためにも努力したくありません。 Javaスクールにはそのような評価はありませんでした。 彼らは私たちに一枚の紙をくれて、私たちは祝いました。そして最後に、この評価はJavaスクールの主催者に見られました。 その後、キュレーターが意見交換を行い、これらすべてに基づいて、誰を服用し、誰を服用しないかを決定しました。
トレーニング中に記憶に残るフレーズや地元のミームはありましたか？
先生の一人がいつも「おはようございます！」というフレーズで皆に挨拶したことを覚えています。 それは不変でした。 面白いことは常にあり、私たちはしばしば楽しみました。
