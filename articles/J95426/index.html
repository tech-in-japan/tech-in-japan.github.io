Mozilla Labsプロジェクト-Firefox Syncは、Firefoxの標準パッケージに含まれます



Mozilla Weaveプロジェクトの名前がFirefox Syncに変更されてからあまり時間はかからず、この拡張機能がMozilla Labsの壁を去ろうとしているというメッセージが既にありました。



ペルソナの道に沿って



6月1日、ほぼ2年半続いた開発終了に関する待望のニュースがついにMozilla Labsブログに投稿されました。 実際、これは、開発者が必要なすべての機能を追加したため、Firefox Syncがパイロットプロジェクトではなくなり、特にFirefoxの将来のメジャーリリースに統合される開発のリストに拡張機能が正式に含まれることを意味します4。



Firefox Syncの開発者の1人であるRagavan Srinivasan ：



Mozilla Labsのプロジェクトである「Weave Sync」が正式に完了し、現在Firefox開発計画の一部になっていることをお知らせします。 現在はFirefox Syncとして知られ、Firefoxの次のメジャーリリースの主要な機能の1つになります。 これまでは、他の通常のアドオンと同様にFirefox Syncをダウンロードできます 。



拡張機能のリリースと名前の変更に加えて、Firefox Syncの最新バージョン（1.3）にはいくつかの新機能と変更が含まれています。

  • ドイツ語、フランス語、スペイン語、中国語、ロシア語を含む15以上の言語への翻訳
  • 簡素化されたインストールおよび登録プロセス
  • ボタン 他のPCなどで開いているすべてのタブにアクセスするには...
SyncがMozilla Labsの一部である間にSyncをテストし、あなたのレポートを送ってくれた皆さんに感謝します-あなたなしではリリースしなかったでしょう。 このプロジェクトはすでに完了していますが、 フィードバックと提案に満足しています。



そして、はい、Mozilla Labsは引き続きFirefoxの使いやすさを改善する方法を見つけることに集中し、ユーザーに個人情報を完全に制御し、開発者に革新的なネットワークサービスを作成する新しい機会を提供します。 たとえば、現在連絡先やアカウントマネージャーなどの拡張機能に取り組んでいます。



そして最後に。 最新のニュースを見逃した場合に備えて、Firefox Syncテクノロジーに基づいたiPhone用アプリケーションの開発に取り組んでいます 。 このアプリケーションは「Firefox Home」と呼ばれ、iPhoneユーザーはFirefoxの閲覧履歴、ブックマーク、最後のブラウザーセッションのタブセットにすぐにアクセスできるほか、「Awesome Bar」（最初のサイト名の文字または説明に含まれる単語）。



すぐにコンピューターでFirefox Syncをセットアップして 、Firefox Home for iPhoneをすぐに使えるようにします。


Firefox Syncとは何ですか？



Firefox Sync-Windows、Mac OS X、Linux、モバイルプラットフォームを含むさまざまなプラットフォーム（Fennecおよび試験運用版で現在利用可能なバージョン）のタブ、ブラウザー履歴、ブックマーク、パスワード、さらには設定を組み合わせて設計されたMozillaの新しいソリューションiPhone用クライアント）。



Firefox Syncを他の開発者（ Xmarksなど ）のソリューションと区別する重要な機能の1つは、オープンAPI上で独自の同期サーバーを上げる機能です。 また、古いバージョンのブックマークは最初の同期中に別のコピーとして保存されるため、エラーが発生した場合、ユーザーは誤って失われたサイトを常にお気に入りに復元できます。

結論



Mozillaには、ユーザーの個人データを同期するための非常に便利で競争力のある拡張機能があります。 現時点では、ブラウザの一般ユーザーに必要なすべての機能を提供しています。 また、ブログの開発者がユーザーの意見や要望に耳を傾けることを約束していると考える場合、拡張機能の同期も間近に迫っていると考える価値があります。 いずれにせよ、現時点ではこれがFirefox Syncの唯一のマイナスであるため、そう願っています。
