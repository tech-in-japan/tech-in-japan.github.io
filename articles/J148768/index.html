RailsClub'Moscow 2012-9月15日、モスクワ。 Ruby / Ruby on Rails開発者カンファレンス



2012年9月15日、モスクワで開催されたDigital October Centerの領域で、RubyおよびRuby On RailsのWeb開発者による第11回会議-RailsClub'Moscow 2012が開催されます。



RailsClub'Moscow 2012は、ロシアの有名な外国の有名Ruby開発者を集め、知識と経験を会議の参加者と共有し、開発の目新しさについて議論し、仕事で遭遇する問題とそれらを解決する方法について話します。



RailsClub 2012では、設計、開発、テスト、デバッグ、高負荷の最適化、Railsプロジェクトでのルビーのスケーリングについて説明します。 また、大規模なWebプロジェクトの構成管理、展開、および管理のトピックにも触れます。



今年は、有名なルビーの星がスピーカーとして来ます。



現時点では、彼らは参加を確認しています：

-Steve Klabnik （米国）、Designing Hypermedia Apisの著者であり、2011 Ruby Hero Awardの受賞者。

-Xavier Noria （スペイン）、Railsコアチームのメンバーであり、2010 Ruby Hero Awardの受賞者。

-Wynn Netherland （アメリカ）、GitHubの開発者、@ thechangelogと本「Sass、Compass」の共著者。

-Dirkjan Bussink （Holland）、rubiniusおよびdatamapperの開発者。

-Sau Sheong Chang （シンガポール）、「Ruby on Rails Web Mashup Projects」の著者、およびHP Labsの応用研究ディレクター「Rubyでのインターネットアプリケーションのクローン作成」。



外国人スピーカーのスピーチは同時にロシア語に翻訳されます。



会議への参加の登録と支払い-railsclub.timepad.ru/event/29192





スピーカーの完全なリスト、非公式部分を含むプログラムの詳細、およびマスタークラスに関する情報は、間もなく発表されます。



現時点では、スポンサーの構成を作成しており、カンファレンスのスポンサーを少し後に発表します。
