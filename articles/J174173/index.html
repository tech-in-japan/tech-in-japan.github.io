Windows Azureアップデート：Hadoop、Dropbox、Mercurial、PhoneGap





3月18日、Scott Guthrieは彼のブログでWindows Azureクラウドプラットフォームの最新のイノベーションを発表しました。 提示された新しい機能は次のとおりです。









  • 人気のあるPhoneGapライブラリからのアクセスを含む、Windows Azureモバイルサービス用のHTML5クライアント（CORS）。
  • Windows Phone 7.5、新しいライブラリ、およびNugetパッケージのサポートの改善。
  • Mercurial（Bitbucket、Codeplex）およびDropboxからのWebサイトのホスティングのサポート。
  • Webサイトの新しいテンプレート
  • サービスとしてのHadoopクラウドプラットフォームであるHDInsightサービスへのパブリックアクセス。


これらのイノベーションについては、もう少し詳しく説明します。









モバイルサービスでのHTML5 / JSクライアントとPhoneGapのサポート



Windows Azureモバイルサービスは、Windows 8、Windows Phone、iOS、Androidなどの一般的なモバイルプラットフォームすべてにクラウドベースのインフラストラクチャを提供します。 現在の更新では、モバイルプラットフォームのサポートにより、HTML5 / JS Webクライアント、特に人気のあるPhoneGapライブラリのサポートが追加されました。 HTML5 / JSで記述されたコードを使用して、モバイルクライアントから保存されたすべてのデータにアクセスできるようになりました。

















完成したアプリケーションの例はポータルからダウンロードできます。また、既存のWebアプリケーションをモバイルサービスのデータに接続する方法についても説明しています。

















HTML5コードからのアクセシビリティは、プラットフォームレベルのCross-Origin Resource Sharing（CORS）サポートによってサポートされています。 Windows Azureモバイルサービスの新しい設定がこれを担当します。これにより、モバイルサービスのデータへのアクセスが許可されているアドレスのホワイトリストを作成できます。

















新しい機能のアプリケーションの詳細については、 このガイドを参照してください 。 別のガイドでは、ユーザー認証メカニズムの使用方法について説明しています。









Windows Phone 7.5、新しいライブラリ、およびNugetパッケージのサポート



モバイルサービスライブラリがNugetパッケージとして利用できるようになりました 。これにより、ライブラリのインストールが自動化および簡素化されます。 ライブラリのその他の目新しさの中では、次の点が際立っています。









  • ポータブルライブラリのサポート。
  • Windows Phone 7.5のサポート
  • 新しいクライアントへの移行とJson.NETおよびHttpClientライブラリのサポート。


モバイルサービス開発者はUserVoiceウェブサイトで新機能の投票を呼びかけます。 リクエストのおかげで、このリリースでは上位5つのオファーのうち3つが実装されました。









MercurialおよびDropboxからのWebサイトのホスティングのサポート



クラウドベースのWindows Azure Webサイトサービスは、TFSおよびGitバージョン管理システムの継続的な展開機能と、Codeplex、BitBucket、GitHub、TFServiceとの統合を長年サポートしてきました。 この更新では、Mercurial（CodeplexおよびBitBucket）およびDropboxに基づいたバージョン管理システムからサイトをホストできる新しい機能が導入されています。









Webサイトプロジェクトをバージョン管理システムにリンクするには、プロバイダーまたはコードソースのいずれかを選択する必要があります。 BitBucketとCodeplexの場合、Mercurialバージョン管理システムを使用できます。

















もう1つの大きな革新は、ソースファイルのソースとしてのDropboxのサポートです。 サイトをクラウドに配置することは、Dropboxと同期されたローカルフォルダー内の一連のファイルを変更または転送するだけで、その後、これらのファイルはWindows Azure Webサイトクラウドサービスに自動的に配置されます。

















新しい機能については、 Dropboxの公式ブログで読むことができます。 DropboxとWindows Azureの統合に関する詳細なチュートリアルは、 ここにあります 。









新しいWebサイトテンプレート



Windows Azure Web Sitesクラウドサービスでは、サイト作成の段階で選択できる新しいアプリケーションテンプレートが追加されました。 ASP.NET、PHP、Node.jsプロジェクトなど、人気のあるWebMatrix開発環境のテンプレートがリストに追加されました。

















詳細については、開発者のブログをご覧ください。









HDInsightの開始-Hadoop Cloud Platform as a Service



HDInsightサービスは 、以前は招待者のみが利用可能でしたが、公開テスト運用に導入されました。 HDInsightは、Hadoopエコシステムとクラスターをオンデマンドで提供するクラウドサービスです。 リンクからサービスのテストへの参加をリクエストできます。 答えは1〜2日以内に届きます。









Windows Azureポータルを使用して、最大32ノードのサイズのHadoopクラスターを作成できるようになりました。

















クラスターを作成すると、RDPや特別な管理パネルを使用してクラスターを管理する機能など、クラスターへのフルアクセスが可能になります。これにより、クラスターで計算するタスクを作成できます。

















MapReduceタスクの作成に加えて、JavaScriptおよびHiveでデータ要求を作成できる対話型コンソールにアクセスできます。 管理ページでは、クラウドサービスの作業についてすばやくアイデアを得ることができる、いくつかの既製のタスクの例が表示されます。









この記事では、HDInsightを使用する最初の手順について説明します。 このブログでは、Windows AzureクラウドストレージでHadoopクラスターを使用する方法について説明しています。 Windows AzureクラウドでHadoopを使用し、オンデマンドでクラスターを取得する方法の詳細については、AzureHub.ruポータルをご覧ください 。
