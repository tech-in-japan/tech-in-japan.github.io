ジュニパーJNCIE-ENTネットワークエキスパート認定履歴

今日、ジュニパーネットワークス認定エキスパートエンタープライズルーティングおよびスイッチングの8時間の試験に合格した結果を受け取りました。 結果-JNCIE-ENT＃427。



この試験は、ジュニパーネットワークスのエンタープライズスペシャリストを認定する最終段階です。 実際、この試験は、シスコシステムズのより人気のあるCCIE R＆Sに類似しています。

熱い追求で、私は試験自体、ジュニパーネットワークス認定プロセスについての私の経験と印象を共有したいと思います。



シスコではなくジュニパーを選ぶ理由

かつて、私は幸運にもCCNAコースのCisco Networking Academyでインストラクターとして働いていたため、テレコミュニケーション市場にシスコの専門家が多すぎる、たとえば41,000人を超えるCCIE専門家がすでにいるという印象を受けました。これにはプラスとマイナスの両方のポイントがあります。 ポジティブなコミュニティには、英語を話すだけでなくロシア語を話す優れたコミュニティ、さまざまな都市の膨大な数のトレーニングセンター、サポートの存在、「出口ラボ」などがあります。ロシアでは一部の地域でラボ試験に合格する機会です。 欠点には、常に新しい「ダンプ」が存在することによる専門家試験の信用の低下、およびその結果、CCIEが認定専門家の数の増加に伴って価値が低下する一般的な傾向が含まれます。



シスコの代替ネットワーク機器メーカーを比較すると、私の選択はジュニパーネットワークスにかかっていました。 ジュニパーネットワークスは、シスコの主要な競合他社の1つですが、より専門的なソリューションポートフォリオも持っています。これは、キャリアおよび企業の通信ネットワーク、およびワイヤレスネットワークのルーティング、スイッチング、セキュリティです。 同社には音声ソリューション、独自の光トランスポート、DWDM、時代遅れのSDH / PDHはありませんが、通信ネットワークの未来について正確な考えがあり、ジュニパーはSDN（Sofware Defined Networking）の概念を最初に策定し、その要素を実証し始めました。



認証パス

ジュニパーの認定には、次の3つの主要分野が含まれます。

  • エンタープライズルーティングとスイッチング（ENT）-企業ネットワークのルーティングとスイッチング。
  • サービスプロバイダールーティングとスイッチング（SP）-通信事業者のネットワークのルーティングとスイッチング。
  • セキュリティ（SEC）-パケットネットワークセキュリティ。


ファーストレベルアソシエイト
基本コースはJNCIA-Junos（Juniper Networks Certified Associate Junos）であり、これはすべての分野の単一の要件です。 Cisco CCNAとは異なり、明確な専門性はなく、Junosオペレーティングシステムの機能を対象としています。詳細は説明しませんが、すべての主要なスイッチングおよびルーティングプラットフォームで実質的に同じです。



第2レベルの「スペシャリスト」
認定の第2レベルでは、トレーニングの方向（JNCIS-ENT / SP / SEC（ジュニパーネットワークス認定スペシャリスト））に従ってスペシャライゼーションが表示されます。 シスコと比較すると、CCNPはそれほど膨大ではなく、証明書を取得するには、シスコの3つの試験に対して1つの試験に合格すれば十分です。



私にとっての別の利点は、サイトのjuniper.net Fast Track Programセクションで入手できる、JNCIA-JunosおよびJNCIS-ENT / SP / SEC試験の自己準備のための無料の公式トレーニング資料が利用可能であったことです。



第3レベルの「プロフェッショナル」
認定の3番目のレベルは、理論試験の最終段階であるJNCIP-ENT / SP / SEC（ジュニパーネットワークス認定プロフェッショナル）です。 認定試験の以前のバージョンには、理論的な質問に加えて、実験室での作業が含まれていたため、すべてがよりシンプルになりました-理論のみです。 シスコとの類似点に戻ると、試験はCCIEで書かれた知識の量を反映していますが、そのためには別の証明書が必要です。



理論試験に合格する
すべての理論的試験（アソシエイト、スペシャリスト、プロフェッショナル）は、ピアソンVUEテストセンターで受けられます。 私自身の経験から、試験の準備ができたらすぐに、VUE Webサイトを通じて、都合の良い場所での領土と空席の基準に従って、私にとって都合の良いテストセンターを選択します（たとえば、モスクワだけでも十数箇所あります）。 これは約1日以内に行いますが、これは推奨されませんが、うまく機能します。 また、VUEのWebサイトを通じてクレジットカードで支払います。これは、テストセンターでその場で試験に支払うよりもはるかに収益性が高くなります。 ピアソンVUEテストセンターの拠点となるトレーニングセンターは、このアプローチを特に歓迎しないことを理解する必要があります。



試験自体は70問で構成され、3〜5問の回答があります;すべては約1時間半かかります。

試験に合格する手順を簡素化するためにダンプを使用したことを隠しませんが、非常に不運であるか、ジュニパーが頻繁に更新するため、ダンプから70問のうち3-5問しか得られませんでした。 したがって、いつものように、私は考える必要があり、時間はほとんどありませんでした。



「エキスパート」の第4レベル
JNCIE認定（Juniper Networks認定エキスパート）は、選択した方向の最後の認定です。私の場合、JNCIE-ENTでは、たとえば、ジュニパーで最も人気のあるサービスプロバイダーなど、別の方向にのみ成長できます。



書かれたCCIEのみを必要とするCisco CCIEとは異なり、ジュニパーJNCIE認定では、以前のすべてのレベルの理論試験に最新の証明書が必要です。



追加の指示「サポート」
各主要エリアには、ネットワークをサポートするためのブランチ（サポートトラック）があることに注意してください。 ジュニパーネットワークス認定サポートプロフェッショナル（JNCSP）を取得するための前提条件は、対応するJNCIP（プロフェッショナル）です。



詳細については、 ジュニパーネットワークス認定プログラム（JNCP）を参照してください。



JNCIE-ENT試験の準備

機器を知り、ジュニパーの技術について学ぶのに約2年かかりました。2012年3月7日に最初のJNCIA-Junos証明書を受け取り、2013年12月20日にJNCIE-ENTに合格しました。



すでに述べたように、JNCIE-ENTには実質的にコミュニティがなく、実際の例、タスク、およびダンプを見つける試みはすべて無駄であり、このタスクにさらに関心が寄せられました。

「スレッド上の世界から」原則に関する情報を収集することが可能でした。「エンタープライズ」の方向性を持つロシア語を話す認定スペシャリストは、試験まで見つけられず、試験のすべての微妙さについて詳細に話し合うことができました。

ホームラボ

次のステップは、常にトレーニングを行うスタンドを作成する必要性でした。 設備のレンタルはまともなオファーがないためすぐに注目されましたが、唯一の例外はiNetZeroサービスでした。これは非常に不快な期間に8時間のセッションを提供し、誰かが常に予約していました。



ジュニパーのルーターおよびスイッチのJunOSオペレーティングシステムは、x86プロセッサ上で完全に仮想化されており、既成のオリーブアセンブリはネットワーク上にあります。 しかし、すべてが私たちが望むほどスムーズではありません。 この画像には、最も重要な要素-geの基本的なネットワークインターフェース-* / * / *タイプ、em *制御インターフェースのみがあり、機能が大幅に制限されていました。 オリーブは、レビューの3日後にすぐにフラグが立てられました。 しかし、スタンドを仮想化するというアイデアと希望は消えませんでした。



ジュニパーネットワークスは、ジュニパーネットワークスの同僚から親切にベータイメージが提供されたSRXシリーズ仮想ファイアウォールルーターFireflyの発売を計画しています。 このJunosイメージには、インターフェースの問題を解決する仮想FPCが含まれていますが、この製品の主な原則はルーティングとファイアウォールのみであり、ハードウェアSRXとは対照的にスイッチングは完全に存在しません-ファミリーイーサネットスイッチングはインターフェース構成レベルでは使用されませんでした。



したがって、決定が下され、さまざまな方法（ebayで購入、avito、レンタルなど）で、ホームラボ用の次の一連の機器が組み立てられました。
  • EX4200スイッチ-4個。その主な役割は、仮想シャーシおよびその他のスイッチング機能の研究です。
  • J2320ルーター-2個 -機器はすでに製造中止になっていますが、機能は準備作業を完全にカバーしています。
  • SRX100ルーター-2個、SRX210-2個、SRX240-1個。
  • 4台の仮想SRXが搭載された2台のラップトップ。
その結果、このラボは自宅でひどい騒音を発し、約1 kWの電力を消費し、パントリーを暖めました。 しかし、最も重要なことは、目標が達成され、夜間と週末の体系的なトレーニングが約4か月続き、主要な作業と組み合わされたことです。



臨床検査

JNCIE-ENT試験はラボ試験であり、完了するには8時間かかります。昼食時間はオンになりません。JNCPラボ試験スケジューリングWebサイトで支払う場合、価格は980ドルです。 試験の主な目的は、ネットワーク管理者の生活の中で1日を提示し、1営業日に彼に割り当てられた多くのタスクを解決することです。



レオニードミレンコフの人のポプラトレーニングセンターのおかげで、モスクワでこの試験に合格する機会を得ました。 彼らが言うように、自宅やネイティブの壁が助けになると、私はこの試験に初めて合格することさえ望みませんでした。 その前に、私はすでにアムステルダムでの試験に合格する準備をしていました-試験は支払われ、ラディソンホテルはジュニパーオフィスから徒歩圏内に予約されました。 そのため、たとえばカイロやドバイでビジネスと喜びを組み合わせて行うことができます。



試験を受ける場所は問いません。実験装置ラックはサニーベール（米国カリフォルニア州）にあり、トレーニングセンターはSSL VPNとSecureCRTコンソール経由でのみリモートアクセスを提供します。 この機能はコンソールの応答に不快な遅延をもたらしたため、デバイスの構成時にキーボードショートカット、メモ帳、マジックジュノの「ロードパッチ」および「ロードマージ相対端末」コマンドを使用することは避けられませんでした。



試験に同行したのは、ジュニパーの英語を話すプロクター、アイマン・アブロブです。 彼は私たちに非常に友好的であり（そして私たちは2人しかいませんでした）、もちろん、タスクの表現、機器の操作、規制、その他の組織上の問題に関連するすべての質問に答えましたが、問題を解決するためのヒントやヒントは除きます。



タスクの構成
タスクは、インターフェイス、スイッチング、IGPルーティング、BGP、プロトコル独立ルーティング（ipv4、ipv6）、マルチキャスト、QoS、システムサービス、セキュリティの8つのセクションに分かれています。 各セクションには3〜9個のタスクが含まれ、各タスクは1〜4ポイント、合計で100ポイントと推定されます。 合格点-71。さらに、セクションの1つに少なくとも1つの正しく解決されたタスクがない場合、試験は失敗します。 タスク自体は相互接続され、雪だるまに似ています。たとえば、スイッチングのQinQ問題を解決せずに、正しいOSFP再配布を行わずにOSPF接続を確立することは不可能であり、マルチキャストは機能せず、トンネルインターフェイスは上昇しません。



試験のルールによると、静的ルートを使用してアイドル状態の動的ルーティングプロセスをバイパスすることはできません。静的ルートごとに1ポイントが差し引かれます。 最初の恐怖は、元の構成のすべてのルーターで静的な127.1.1.0拒否ルートを発見したときに私を捕らえました。 これはセットアップではなく、junos 11.4の機能（またはバグ？）であることが判明しました。その本質は、このルートなしでpim local rpを使用して構成をコミットすることは不可能でした。



2回目の試験では、仮想シャーシを再構築する必要があったときに、スイッチのリモートコントロールを失ったときにパニックに陥りました。 無駄になった ハードウェア障害が検出された場合は、できるだけ早くプロクターに通知して、タイマーを停止し、ラボをバックアップラックに転送する必要があります。 私の場合、2分後にすべてが正常に復元されました。



明らかな理由でタスクの本質を明らかにするつもりはありませんが、公式ウェブサイトで発表されたほとんどすべてのトピックが試験に存在していると断言できます。

タスクにはいくつかのオプションがあると思います。

また、新しいバージョンの試験のベータテストに関する情報を公開しました。
