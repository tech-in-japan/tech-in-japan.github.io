Dell Latitude E6430 ATG：ユニバーサルソルジャー

過去数年にわたって、使い慣れたデスクトップコンピューターは、あらゆるタスクに対応する多数の構成オプションとフォームファクターに変化しました。 このような背景に対して、何らかの理由でこの多様性すべてから際立っているデバイスは常に興味深いものです。 もちろん、これらの1つは、Dell Latitude E6430 ATGラップトップです。これは、困難な状況で働くが、通常のオフィス生産性を備えた真の万能兵士です。 Dell Latitudeラップトップは企業顧客向けであり、通常の電子機器のスーパーマーケットでは販売されていません。 ただし、一見狭いように見えるすべてのオーディエンスにとって、Latitudeはきっとその愛好家を見つけるでしょう。 霜、熱、または高湿度が予想外に発生するロシア規模

そして、住宅および共同サービスのために、そして一般市民のために、最も困難な状況でラップトップを「入手する」能力

条件は多くの人に評価されます。

7月10日にロシアで公式に導入された新世代のLatitudeラップトップには、Intel Ivy Bridgeプロセッサが搭載されています。 これはユーザーにとって何を意味しますか？ 優れたパフォーマンス、長いバッテリ寿命、最新のすべてのワイヤレスネットワークのサポート、最新のUSB 3.0およびBluetooth 4.0インターフェイス。 このモデルでは、高度なIntel vProテクノロジーも使用しました。このテクノロジーは、大企業のシステム管理者に最も広範なリモート管理機能を提供します（そうではありません）。



これはすべて、Dell Latitude E6430 ATGに完全に適用されます。

ケースは、トライメタル技術を使用したアルミニウム製です。 さらに、コーナーはマグネシウム合金素材で補強され、ディスプレイカバーは陽極酸化され、外部保護が強化されています。 金属製のヒンジとパウダーコーティングされたベースにより、ラップトップの全体的な強度が向上します。 このような対策は、75 cmの高さから落としたときにラップトップを保護します。

すべての外部ポートは、ほこり、汚れ、湿気を防ぐために特別なカバーで覆われています。 E6430 ATGは、MIL-STD 810GおよびIP65 IP5X環境認証に準拠しています。つまり、「戦闘」環境で使用するための厳しい軍事規格に準拠しています。

内部

  • Intel Core i5-3320M（2.6 GHz、第3レベルの3 MBキャッシュ）;
  • Intel Core i5-3360M（2.8 GHz、3 MB L3キャッシュ）;
  • Intel Core i7-3520M（2.9 GHz、4 MB L3キャッシュ）;
  • Intel Core i7-3720QM（2.6 GHz、6 MB L3キャッシュ）。




RAM DDR-3 1600 MHz SDRAMは、2 GB（1モジュール）、4 GB（1または2モジュール）、6および8 GB（2モジュール）で利用可能です。



ディスクサブシステムの選択も非常に柔軟で、目的のタスクとウォレットのサイズに依存します。 これは、320/500/750 GBのボリュームを持つ7200 rpmの標準HDD、128または256 GBのSSD、または500 GBのハイブリッドディスクです。 振動に対するハードディスクの安定性を確保するために、ゴム製ダンパーが使用され、追加の情報保護のために、高速自由落下センサーが取り付けられています。



統合グラフィックスカードIntel HD Graphics 4000はデフォルトでグラフィックス機能を担当しますが、その機能が適切でない場合は、個別のNVIDIA NVS 5200M（1 GB GDDR5）で選択を停止できます。



このラップトップは、幅広いワイヤレステクノロジーも備えています。 そのため、Wi-Fiカードはいくつかのオプションから選択できます。

便利：これらすべてに、必要に応じて、A-GPS機能を備えたマルチモードミニカードHSPA-EVDO DellワイヤレスDW5630（Gobi 3000）を追加できます。




そして今、おそらく最も興味があるのはDVDドライブでしょう。 より正確には、ドライブ自体ではなく（デバイスが死にかけている）、それがインストールされているバスケット-E-Modular bay。 ここで、ラップトップの機能を強化するために追加のモジュールをインストールできます。 DVD-ROM（オプションで±RWの書き込み）に加えて、次のオプションがあります。320GB 7200 rpmのハードディスク。 追加のUSB 3.0ポートまたは30 Whの追加の3セルバッテリーは、97 Whのメインバッテリーと相まって、32.7（！）時間もの間自律動作を提供します（これは、新しいIntel Ivy BridgeプロセッサーのSSDおよびテクノロジーによっても促進されます）。 ドライブは「ホット」で取り外すことができます。つまり、ラップトップの電源を切らずにバッテリーを追加できます。



ところで、このような2番目のバスケットにはドッキングステーションを装備し、ラップトップを実際のオフィスワゴンに変えることができます。 たとえば、DVDドライブをドッキングステーションに「固定」することができます。オフィスまたは自宅から数百キロ離れたところにあるRecovery DVDの発売を考えている人はほとんどいません。



デルのデータ保護テクノロジーは、追加の情報保護とアクセス制御を提供します| 連絡先または非接触型スマートカードと、Intel vProテクノロジーを使用して別のチャネルのハードドライブをリモートで消去できる独自のDell System Center Configuration Manager拡張システムと組み合わせたアクセス（注：2012年9月に発売予定）。 ディスクからのデータ復旧または証明書の発行によるデータ破壊の保証-必要に応じて提供されます。



オペレーティングシステム-XP Modeのサポートの有無にかかわらず、32ビットまたは64ビットのHome Premium、ProfessionalまたはUltimateのMicrosoft Windows 7。 デルは大企業でLatitudeを完全に使用できるように準備したので、IT部門の生活を楽にするさまざまなサービスを提供する準備ができています。 お客様は、すべての段階で機器の実装と使用に関する当社の支援を頼りにすることができます。 たとえば、必要なソフトウェアのセットで特別なOSイメージを準備し、それを保存し、必要に応じてドライバーを更新することが提案されています。
