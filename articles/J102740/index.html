NetBeansのCSSマクロ

Zen Codingに関する情報はハブで数回公開されており、HTMLおよびCSSコードのレイアウトを高速化するために多くのコードエディターに接続できます。 これらのエディターの1つにNetBeansがありますが 、これはZEN CSSでは機能しません。



この状態に我慢したくないので、私は、 メモ帳++の場合のように、貴重な秒数を節約するために（およびプロパティ全体を書き込むための怠inessさを）頻繁に使用するレコードのマクロを作成しました。



例：



Alt + C bb TAB -border-bottom：1px solid＃ FFF *;

Alt + C p TAB-位置： 相対 *; top：0; 左：0;

Alt + C fw TAB-フォントの太さ： 900 *;

Alt + C tb TAB-テキスト装飾： 下線 *;

Alt + C bg TAB-背景：#FFF url（../ images / bg.png）repeat-x 0 0;

* -取り消し線は、変更のために強調表示される文字を示します



おそらく、そのような組み合わせを見ると、かさばるように思えますが、個人的にはすぐに習得しました。 おそらく、誰かが組み合わせを改善するための提案をするでしょう。



現在、合計18の組み合わせが記述されています。おそらく、他の何かが追加される可能性があります。オプションを提案してください。

簡単なチートシートをすばやく作成します。





そのため、このようなマクロが日常の作業に役立つと思われる場合は、 org-netbeans-modules-editor-settings-custommacros.xmlをダウンロードしてください 。

このファイルはc:\Users\habrauser\.netbeans\6.9\config\Editors\Macros\



にアップロードする必要があります（既に独自のマクロがある場合は、コピー、上書きする際に注意してください）。



NetBeansでも、利便性の点でマクロよりも優れているコードテンプレートのようなクールなものがあります。PHPでは優れた動作をしますが、CSSではそれらをマスターできませんでした:(



上記の有用なヒントと追加に感謝します。 アイデア（自分自身または他の人）が到着すると、マクロが更新されます。



PS：マクロではVerdana, Geneva, sans-serif



Alt + Cフォントv TABを使用したVerdana, Geneva, sans-serif



などのフォントセットのクイック入力もあります。 彼らは必要ですか？



PSS： NetBeansに対するお気に入りのeditor％editor％の利点を説明しないように、Habrovskの人々に大きな要望がありますが、投稿はそれについてではありません。



UPD： 4か月後、彼はNetbeansにとどまり、マクロのセットを拡張しました。

新しいマクロをダウンロードして 、 .xmlをc：\ Users \ your_name \ .netbeans \ 6.9 \ config \ Editors \ Macros \にコピーできます。
