Nokia Asha 210 Stylish budgetのレビュー

ノキアは依然としてそれ自体に忠実であり、引き続きQWERTYデバイスに満足しています。 この電話が4月に発表されたとき、私はすぐにそれが気に入りました。 ご覧のとおり、LUMIAのスタイルはここで明確に表現されています。 明るい色、モノリシックなケース、携帯電話はすぐに注目を集め、誰も無関心になりません。 3,000ルーブルの価格を知ったので、関心はさらに高まります。 このデバイスは、大食いのスマートフォンの優れたコンパニオンと見なすことができ、1日の終わりまでではありません。 また、デバイスの電源を切らずに取り付けることができる2番目のSIMカードがあることにも満足しています。 スマートフォンは夕方に放電され、SIMカードをNokiaに挿入した後、連絡を取り合っています。

包装と包装



電話機は、電話機自体とほぼ同じサイズの明るいコンパクトな箱に入っています。 一方で、それは良いことですが、他方では、内部にはほとんど何も見えないでしょう。





実際、キットは小さく、220 mmから2 mmのジャックとシンプルなヘッドセットWH-109を備えた充電器で、応答ボタンはなく、マイクのみです。









PCとの同期用のケーブルはありません。 箱は再生可能な材料で作られており、リサイクル製品である材料の60％が含まれています。 パッケージは完全にリサイクル可能です。



外観



Asha 210の手の中では、固いキャンディーバーのように感じられ、パチパチ音やバックラッシュはありません。 寸法は小さい11.5 x 6 cm、厚さ11.8 mmです。 表面全体は、ポリカーボネートに似たつや消しプラスチックでできています。





おなじみのルミアスタイルが認識され、色が特に心地よいです。 ロシアでは、まだ黒と黄色で販売されていますが、青が期待されています。





主な光沢部分は、ファンクションキーと画面を備えたナビゲーションブロックです。 ボタンは黒なので、印刷物はあまり見えません。





デバイスは完全に手の中にあり、鋭い角はなく、手から滑り落ちません。





上部にはmicroUSBの隣にある充電用の2 mmジャックがあり、そこから充電することもできます。次に、ヘッドセット用の3.5 mmミニジャックがあります。





左側には、2番目のSIMカードのスロットを覆うスタブがあります。 デバイスは標準のmini-SIMを受け入れます。 それだけです。音量ボタンや電源オフキーはありません。





電話機の背面には、カメラとスピーカーのグリルがあります。 蓋は背中全体を覆い、ギャラクシーの蓋を幾分連想させる多数のラッチにかかっています。





ふたは簡単に取り外され、非常にしっかりと保持されます。 カバーの下には、バッテリーと少し下のスピーカーがあります。





バッテリーの下にはSIMカードとmicroSDのスロットがあり、メモリは最大32 GBまで増やすことができます。





キーボード



もちろん、Asha 210キーボードについても個別に説明する必要がありますが、こちらは37キーのQWERTYキーボードです。 一般に、以前のQWERTYデバイスのスタイル、キーのサイズ、および移動は顕著です。 すべてのキーは凸型で、短いストロークと優れた触覚フィードバックを備えています。





主人公は黒、ロシア文字は灰色がかった色合いです。 また、逆光時、ロシア語とラテン文字の違いが顕著です。





入力に移りましょう。 ロシア語のテキストのセットを考えてみましょう。 新しいテキストは大文字で始まります。大文字を強制する必要がある場合は、Shift（左から2番目のキー）を押しながら文字を入力します。 このキーでCaps Lockを有効にすることもできます。





質問などの記号を配置する必要がある場合は、文字「w」を押すと、1秒後に「？」に変わります。 したがって、キーボードで印刷されるすべての追加文字は、この方法で入力できます。 キーを長押しするのが不便な場合は、左端のボタンを押しながら目的の文字のキーを押すだけです。 たとえば、長い数字を入力する必要がある場合に便利です。





さまざまな絵文字や追加の文字を表示する個別の「Sym」キーもあります。 「g」と「x」を長押しすると、コンマとピリオドが設定されます。 「e」と「b」という文字が表示されているように、それらを長時間入力するのに十分なスペースがなかったため、「u」と「z」を保持します。 Nokia 5800の時代から、レイアウトを1クリックにすばやく切り替える独立したキーがないことは不便でしたが、私はこの機能を持っていませんでした。 ここで、極端なCtrlキーを押すと、不必要なジェスチャーなしで言語が正常に切り替わります。 Ctrl + C、Ctrl + X、Ctrl + V、Ctrl + Aなどの使い慣れた組み合わせをうまく練習してください。





スタンバイモードでは、キーを押すことで、BlueTooth、Wi-Fiをすばやくオフにしたり、サイレントモードをオンにしたりできます。 また、ジョイスティックまたはファンクションキーを押したときのさまざまなアクションの割り当てなど、プッシュボタン式電話のすべてのバンにもアクセスできます。 ジョイスティックの左側には、アプリケーションのクイック起動を割り当てることができるキーがあります。たとえば、一部の国では、この人気のあるメッセンジャーのアイコンもあります。 右側には、クイックスタートカメラボタンがあります。





2.4インチの空からの星のスクリーンは、わずかな偏差でも十分ではありません。色の歪みは顕著です。画面の解像度は320 x 240ピクセル、169 PPI、64,000色です。しかし、3000 pの場合、これは通常の画面です。 1から5。



バッテリー



1200 mAhの容量の取り外し可能なBL-4Uバッテリーは、異常に長い動作時間を提供します。電話は、使用アクティビティに応じて、数日間、さらには1週間も非常に簡単に持ちます。 以前は、充電スティックを自由に見つめていましたが、スマートフォンは正確なパーセンテージデータに慣れており、この携帯電話では十分ではありません。





コミュニケーションズ



電話は2Gネットワ​​ーク（850/900/1800/1900 MHz）でのみ動作します。 モバイルインターネットの最大速度はEDGE 236.8 kbpsです。 一方、3Gの欠如はバッテリーの電力を節約し、高速インターネットはここでは必要ありません。ブラウザは常に使用するのにそれほど便利ではありません。 2枚のSIMカードはスタンバイモードです。 任意のSIMカードを呼び出すことができます。 1枚のSIMカードで会話が始まるとすぐに、2枚目のSIMカードにアクセスできなくなります。

また、Wi-Fiをインターネット、b / gネットワーク、およびすべての暗号化プロトコルに使用できます。 HotSpotモードがありません。

そしてもちろん、Bluetoothは、Bluetoothがあればより良くなります 。 データとヘッドセットの転送に必要なすべてのプロトコルをサポートします。

カメラ

カメラモジュールは最も簡単です。 固定焦点の2メガピクセル、1600 x 1200ピクセルの画像解像度、マトリックスサイズ1/5インチ、開口f / 2.8。 簡単な設定がありますが、一般的には特別なものはありません。





カメラアプリケーションはすぐには動作しません。写真は数秒間保存できます。 「セルフポートレート」モードはおもしろそうに見えたので、電源を入れ、カメラをあなたに向けると、電話が左または右、より高いまたはより低いことを示します。 顔が写真の中央にくると、撮影が始まります。 ギャラリーからの写真は、Bluetooth、電子メール、MMS、Odnoklassniki、VK、FB、Twitter経由で送信できます。 また、ひどい品質で最低解像度176 x 144ピクセルのビデオを撮影することもできます。



ソフトウェア部



電話機はSeries 40ソフトウェアプラットフォームで実行されます。つまり、スマートフォンではありません。 多くの人がデバイスを手にしていますが、最初はAndroidがあると考えています。 S40で15億台のデバイスを販売したことが判明しました！ もちろん、Symbianではないのは残念です。価格は上昇する可能性がありますが、ノキアは予算のSymbianタッチチューブをほぼ同じ価格で販売しました。





ソフトウェアの部分について詳しく説明する理由はありませんが、最も興味深い部分について簡単に説明します。

システムには、起動時にプログラムの完全版をダウンロードするWhatsApp Launcherが既にあります。 何も支払う必要はありません。





WhatsAppは、特別なボタンからすばやく起動したり、別のアプリケーションを割り当てたりできます。 デバイス自体は遅くなる可能性がありますが、アプリケーションはスマートではありませんが、WhatsAppの起動には8秒かかります。

SMSを表示したり、電話に出たりするためにアプリケーションを終了しても、アプリケーションは長時間開きます。 一定のプッシュがあります、つまり スタンバイモードのロック状態では、電話が振動し、メッセージが到着したことを知らせる音が鳴り、画面が点灯し、メッセージを表示したり、開いたり終了したりできます。





また、新しいメッセージが到着したことを示すアイコンが通知行に表示されます。

メッセンジャーはうまく機能し、メッセージは正しく配信され、マルチメディアコンテンツを送受信できます。グループチャットはサポートされています。





このような安価なデバイスの良いクライアントを作った開発者のおかげです。 私にとって、これは最も基本的なアプリケーションの1つであると言えます。その理由は、デバイスが採用されたからです。

音楽には、内蔵プレーヤーとFMラジオがあります。 アプリケーションが少ない場合は、無料でゲームやプログラムを購入またはダウンロードできるストアがありますが、選択は貧弱です。 何らかの理由でViberをインストールしたくなかったので、すぐに電話サポートが追加されて機能すると思います。 せっかちな人は、JAVAアプレットを試して、その動作をテストできます。







システム全体のパフォーマンスは高速とは言えず、数ミリ秒の小さな遅延がすぐに感じられますが、それはすべてがNexusで飛ぶ習慣による可能性が高いです。 通常の電話の場合、この速度は通常の速度です。 ブラウザに座りたい場合は、これを行わない方が良いでしょう。 緊急に必要な場合のみ、非常に不快です。

連絡先はそれぞれGmailに保存され、メインのAndroidスマートフォンは連絡先をGmailと同期します。 電話の設定をざっと調べて、同期を設定し、連絡先をアドレス帳に正常に入力しました。 同期は時間通りに構成でき、連絡先を定期的に更新できます。



2枚のSIMカードを使用する



2枚のSIMカードの使用を検討してください。 最初のSIMカードはバッテリーの下に取り付けられ、システムではSIM1と呼ばれます。 追加のSIMカードを「ホット」にインストールすれば、ネットワークは正常に機能します。 先ほど言ったように、無線モジュールは1つしかありません。 1枚のSIMカードで話している間、2枚目はアクセスできなくなります。 通話を待っている間、両方のSIMカードが利用可能です。 このメニューには、SIMカードを構成するための特別なセクションがあります。 特定のサービスに優先SIMカードを設定できます。 特に、呼び出しが行われた場合、特定のSIMカードからのみ呼び出しを行うか、選択肢を提供します。 たとえば、1つのSIMカードに無料のSMSがあり、ディスパッチャでは、SMSは常にこのSIMカードからのみ送信され、2番目のSIMカードはインターネット専用であることを指定できます。 すべてがシンプルで手頃な価格です。 私が気に入らなかったのは、アドレス帳でどのSIMカードから特定の連絡先に電話をかけることができないかということだけです。 たとえば、2枚目のSIMカードから猫に電話をかけたいときに、「何から電話をかけますか？」という電話を絶えず聞かせたくない場合、特定の連絡先についてこれを覚えていますか？
