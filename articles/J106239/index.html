Smartq V7の素晴らしいレビュー

Habréのこのデバイスについてはすでに 書きました 。 しかし、第一に、何となく何気なく、そして第二に、それは古い（比較的現在の）デバイスファームウェアの問題でした。

ガジェットは2か月以上使用しており、その長所と短所をすべて説明できます。 それでは始めましょう。







デバイスの技術的特性は変更されていません-これらは次のとおりです。

  • 800 x 480の解像度を持つ7インチ抵抗スクリーン。
  • ビルトインBluetooth 2.0、Wi-Fi 802.11b / g、USBホスト/ OTG;
  • TCC8901プロセッサを搭載したTelechips TCC8900チップセットは、600メガヘルツ、256MB DDRII、2GBの不揮発性メモリで動作します。
  • HDMI、コンポジット、3.5 mm出力、内蔵マイク、SDHCカードリーダー。
  • 4500 mAHのLi-Polバッテリー。
  • 寸法：200x120x24、480グラム。


このデバイスは、現在のバージョンの3つのオペレーティングシステムで動作します。

  • Android 2.1 V2-多言語、税込 およびロシア語;
  • Ubuntu Linux 5.5-英語;
  • Windows CE 6-英語版（以前は中国語でした）;


また、AndroidのUBIFSファイルシステムへの移行以降、メインブートローダー（オペレーティングシステムの選択を提供）が変更されました。 新しいデバイスはすぐに付属します。古いデバイスの場合は、ブートローダーを更新する一連のユーティリティをダウンロードすることをお勧めします。 新しいブートローダーと古いブートローダーを区別するのは非常に簡単です。古いブートローダーは明るい青色の背景で、新しいブートローダーは暗い紫色です。 しかし、順番にすべてについて。



機器は非常に経済的です。箱の中には、SmartQ V7自体、フラットプラグ付きアダプター、HDMIオス-オスコード、および中国語の説明書があります。

また、画面に追加の保護フィルムを購入しました。







プラスチックは手触りが快適で、背面がわずかに粗いため、デバイスは手によく便利に置かれています。 バックラッシュやきしみはなく、タブレットは完全にモノリシックに感じられます。 私にとっては、これは大きなプラスです。 スクラッチを残すことを恐れずに、デバイスをテーブルの上に「下向き」に置くことができます。 一般に、ビルドの品質は十分ではありません。



デバイスには6つのハードウェアボタンがあります（リセットはカウントされません）-フロントパネルに3つ、さらに上部に3つあります。 ボタンは、オペレーティングシステムによって機能的な目的が変わるため、特定のアクションを表示しない「ニュートラル」アイコンで装飾されています。



Android OSの場合、フロントパネルのボタンは、メニューの最上位レベルへの遷移またはブラウザーの「戻る」（「+」ボタン）、メインメニューへの終了またはデスクトップへの戻り（「-」ボタン）、コンテキストメニューの呼び出しです。またはデスクトップのメインメニュー（四角形のボタン）。







デバイスの背面は平らではありませんが、わずかに丸みがあり、背面にはスピーカーを隠すリリーフグリルと、デバイスを垂直に設置するためのスタンドがあります。

スタンド自体は薄いプラスチックでできている非常に脆弱なように見えますが、主な機能は正常に機能し、今のところ壊れることはありません。



上の3つのボタンは、FBReaderのボリュームコントロール/ページめくりと、すべてのオペレーティングシステムのオン/オフ/スリープボタンです。 近くにマイクジャックがあります。



側面にはヘッドフォン出力があり、コンポジットビデオ出力、HDMI出力、USB 2.0ポート、充電コネクタもあります。







下-リセットボタンとスタイラススロット。 個人的には、少なくとも画面が汚れないので、指よりもスタイラスで画面を突くほうがはるかに快適です。 多くの人が「インセンシティブ」タッチスクリーンについて不満を言っていましたが、そのような問題は見当たりません。おそらく、デバイスの以前のバージョンに関するものでした。



デバイスの電源を入れると、メインブートローダーウィンドウが表示され、オペレーティングシステムを選択するよう求められます。







これは古いブートローダーのように見えましたが、新しいものは次のとおりです。







OSの完全な読み込みには1分もかかりません; Androidが最速で起動します。



次に、このデバイスでAndroid 2.1の側にレビューを移動します。そのためです。



Windows CEは、最も「新しい」中国語以外でも、依然として古いOSです。 主にナビゲーションに使用されます（はい、Bluetoothを介してGPSをSmartQ V7に接続し、Windows CEでiGoを転がし、記録的な大画面を備えたカーナビゲーターを取得できます）および古いおもちゃのポートでのゲーム（たとえば、M＆Mの古いHeroesが最近移植されました）。 ）



SmartQ-Linuxガイドの大規模で友好的なコミュニティ、ロシア語に適合したアセンブリの存在、独自のLinuxファームウェアの開発から判断すると、多くの人がこのシステムを好んでいましたが、Ubuntuは個人的には遅くて小さな画面には適さないように見えました仮想キーボードをオンにしたテキストエディターでは、画面の4/5がキーボード、ウィンドウタイトル、およびエディターのあらゆる種類のツールバーで占められているため、非常に不便です。

しかし、多くの人々がこのOSを好み、それを使用し、積極的に開発しています。



Androidを選択し、メインOSとして使用しました。 これが私の話です。



Androidは、箱から出してすぐに使用できるシステムに最も似ていますが、もちろん、多くの小さな編集が必要です。



デバイスに付属のファームウェアにはデフォルトで英語のインターフェースがありましたが、更新されたバージョンをフラッシュしてインストールした後、Androidはデフォルトの中国語で起動します。 この厄介な見落としは最初は混乱に陥りましたが、短いグーグルの後、システム言語をロシア語に切り替える方法を説明したガイドがありました:)



画面の向きは、特別なユーティリティ、ポートレートまたはランドスケープによって切り替えられます-誰もが自分で選択します。 デバイスにはGセンサーが装備されておらず、職人がそれをはんだ付けしてLinuxで動作するように強制しましたが、Android向けのそのようなプロジェクトは見たことがなく、Gセンサーの必要性を感じたことはありませんでした。



2番目（ロシア語のインストール後）の修正は、システムフォント、ウィンドウ、およびメッセージのサイズを小さくすることでした。 デフォルトでは、Androidインターフェースは非常に大きく、多すぎると思いますが、このオプションは構成ファイルを変更することで自由に構成できます。



次に、Androidは他の変更を行うためにroot化する必要があります。 Androidでルートを取得する方法は、 ステップバイステップで長い間存在し、説明され、文書化されてきました。



ファームウェアに加えて、公式のAndroidマーケットアプリケーションが提供されていますが、動作させるには、タンバリンとの短いが激しいダンスが必要です。 つまり、AndroidエミュレーターでGoogleアカウントの一意のandroidIDを取得し、それをデバイスにフラッシュする必要があります。 ステップバイステップの指示も存在します。



さらに「チューニング」は、ロシア語キーボードのインストール（ロシア語のUSBハードウェアキーボードのロシア語レイアウトを完全にサポートしているため、RuKeyboardを推奨）、中国語ソフトウェア（多くはありません）および電話部分（Phone.apkおよびTelephonyProvider.apk-notタブレットでは意味がありますが、バックグラウンドで作業中にバッテリーを消耗します）、個人的に必要なソフトウェアをインストールします。



ランチャー（Launcher Pro）、ブラウザー（SkyFire）、書籍リーダー（FBReader）、ビデオ（Act 1）およびオーディオ（Meridian）プレーヤー、テキストエディター、ICQクライアント、およびその他の多くのユーティリティ（タスクマネージャー、バックアップ、YouTubeのユーティリティ、ビデオストリーミング）。





Launcher Proとアニメーション壁紙を含むデスクトップ。



すべての操作の後（そして正直に言うと、その前に）、デバイスは素晴らしく動作し、手元に来たすべてのビデオを再生します（最大2.4 GBがDivX、Xvid、MPEG-4、および720p MKV（AC3、H。 254）映画）。 1080pビデオも再生されますが、いくつかのブレーキを使用すると、プレーヤーを選択することでこれを処理できる可能性が非常に高くなりますが、7インチの画面で1080pを視聴する理由がないため、この問題には対処しませんでした。



コンポジットは大画面にわずかに「緑色」でぼやけた画像を与えますが、HDMI接続を使用すると、テレビでデバイスから映画を見ることができますが、コンポジットおよびHDMI出力は機能します。 繰り返しますが、ビデオはブレーキをかけずにフリーズします。



新しいファームウェアでは、USBポートにバグはありません。 彼は、見つけたすべてのフラッシュドライブ、キーボード、マウス、3Gモデムを理解しています（Huawei e1550およびこのメーカーの他のモデムは問題なく動作しますが、一般に、すべての3Gモデムがサポートされているわけではありませんが、Bluetooth経由で携帯電話との接続をセットアップして使用することは可能です）モデムとして彼）。
