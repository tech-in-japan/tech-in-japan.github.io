Highscreen Alpha Rage-予算従業員の予算従業員

率直に言って、私は安いスマートフォンのファンではありません。 私は「メガヘルツとメガバイト」に心からお金を払うことを好みます。これは少なくとも1年はまだ関係があるでしょう。 しかし、主に予算のないデバイスのために市場は成長しています。 たとえば、私は実際にビジネスでXiaomi Red RiceとMotorola Moto Gを試したいと思っていますが、これまでのところ、これらは単なる欲望であり、少しの夢です。 しかし、現実にはまったく異なる予算の従業員、Highscreen Alpha Rageが私の手に投げ込まれました。 最初は彼の外見に染み込んでいなかったが、彼はオンラインで何が何であるかを評価した。 一般的に、このスマートフォンはそれほど悪くないことが判明し、4,990ルーブルの価格で実際の代替品はほとんどありません（もちろん、公式小売店で）。







テキストに直接移動する前に、Highscreen Alpha Rageで生活できる視聴者の概要を説明します。 私の意見では、バリケードの片側に学童と生徒、もう片側に医師と教師がいます。 一言で言えば、より大きな責任で大きな費用を処理することに慣れている人々。 それが最大量ではなく受け取る（獲得する）ためだけの場合。 したがって、レビューのタイトルの意味。



あいまいな外観



率直に言って安価なスマートフォンを外部から魅力的にすることは非常に困難ですが、それはHighscreenで見ることができ、試みませんでした。 丸みを帯びた角、面取りされたサイドエッジ、周囲の擬似金属エッジ-これらはすべて、他社のスマートフォンですでに数千回見られています。 特にサムスン。

スクリーンはガラスで覆われており、非常に簡単に汚れます。 背面パネルは、通常のプラスチックとソフトタッチのクロスです。 カメラのレンズの領域に潮が形成されるのはあまり好きではありません。金属の下に描かれた「目」のフレームがすぐに剥がれます。



















カバーの下で、SIMカードを取り付けるための元のシステムを2階に重ねて観察します。 まあ、少なくともmicroSDカードスロットには簡単にアクセスできます。 バッテリーは取り外し可能で、容量は2,000 mAhです。 スマートフォンを積極的に使用した場合のバッテリー寿命は1日で、適度な使用で2日です。



















Highscreen Alpha Rageでは、人間工学に基づいた深刻な誤算に気づきませんでした。デバイスが手に持っている斜めの画面が比較的小さく、ボタンが正常に押されているためです。 後者は2つしかありません。左側には音量調節のスイング、右側にはオン/オフキーがあります。 ヘッドフォンジャックとUSBケーブルが上端に配置されています。















キットとしてのキット



Xiaomiスマートフォンのボックスを漠然と連想させる外観の元のボックスには、バッテリー、充電器付きUSBケーブル、説明書付きの保証書、低品質の有線ヘッドセットがありました。











ディスプレイ-大



4,990ルーブルのスマートフォンのディスプレイは、大きな4.5インチです。 上記のように、スクリーンは汚れやすく、TNマトリックスを使用しているため、視野角は非常に小さくなります。 ただし、カップルと一緒にディスプレイを頻繁に見る予定がない場合は、そうするでしょう。 非標準解像度、854 x 480ピクセル。 私はゲームやプログラムとの互換性を恐れましたが、心配は無駄でした。 私が他のMTKスマートフォンやタブレットで使用するキャッシュは、Highscreen Alpha Rageでいたずらではありませんでした。 明るさのマージンは中程度で、晴れた日の路上では、最大限にひねる必要があります。 原則として、これは多くの場合、より良いIPSマトリックスの特徴です。



内部のすべての魅力



Highscreen Alpha RageはMediaTek MT6589Mプラットフォーム上に構築されており、MT6589シリーズを開きます。 周波数が1.2 GHzのクアッドコアARM Cortex-A7プロセッサーとPowerVR SGX544 GPUが含まれています。 もちろん、最近発表された8コアMT6592プラットフォームに照らして、目立ったものはありませんが、安価なスマートフォンの場合、MT6589Mの存在は、以下で説明する適切なパフォーマンスを提供するため、非常にプラスです。 HTC、LG、またはサムスンのデバイスは、さらに高価なデバイスでも、Qualcomm MSM7227A（HTC 200、LG Optimus L5）やSTE Novathor U8500（Samsung Galaxy Ace 2）などの非常に退屈なシングルコアプラットフォームをユーザーに提供します。



スマートフォンは、ネットワークGSM（850/1900 MHz）およびUMTS（2100 MHz）での作業をサポートしています。 Wi-Fi 802.11b / g / nおよびBluetooth 4.0アダプター、およびGPSレシーバーを提供します。



RAMの量は、スマートフォンを安くするためにナイフの下にありました-512 MBしか使用できません。 ただし、フラッシュメモリは標準の4 GBであり、そのうち約2.6 GBをユーザーが使用できます。 すでに書いたmicroSDスロットがあるため、少量のフラッシュメモリでも問題ありません。 実際には、512 MBのRAMを使用することもできます。 主なことは、複数のゲームを一度にダウンロードすることではなく、他のアプリケーションによるメモリのロードを制御することです。



小学生と生徒はスマートフォンに何が必要ですか？ 「遅れなくゲームを引っ張った」。 そして、Highscreen Alpha Rageが引っ張るゲームに注意してください。 そして、Angry Bird、Hill Climb Racing、Plants vs.のような普通の2Dではありません。 ゾンビだけでなく、アスファルト8やモダンコンバット4のように非常に立体的です。

ベンチマークは、低解像度の画面とかなり強力なGPUを支持します（わずか156 MHzの周波数で動作するという事実にもかかわらず）。 AnTuTuは、Highscreen Alpha Rageハードウェアプラットフォームの機能をほぼ14,000ポイントと評価し、デバイスのパフォーマンスが平均的であることを確認しました。 AnTuTuとの全体としての残りのベンチマークは一致しています。











OS-「裸の」Android 4.2



Highscreen Alpha Rageが制御するのは、Android 4.2アドオンによってクリーンで邪魔にならないことです。 少量のRAMを使用すると、これは非常にプラスになります。不要なアプリケーションをメモリからアンロードする不必要な理由はありません。







開発者がインストールしたプログラムには、ファイルマネージャー（悪くはありませんが、より使い慣れたESエクスプローラーに置き換えられました）、感度の良いFMレシーバー、ビデオプレーヤー（BS Playerをすぐにインストールしたため使用しませんでした）、および音楽プレーヤーがあります。 後者は、低音強調機能を有効にして音の深さ（3D）を与える機能に満足していますが、完全なヘッドホンに音質を追加するものではありません。 また、Dropboxの容量が突然少なくなった場合に、4Syncクラウドストレージ（モデルの場合は20 GB、それ以外の場合は15 GB）にすばやくアクセスするためのプログラムもあります。



個別のメニュー項目を使用すると、システムの更新をすばやく確認したり、Wi-Fiを介した3G共有を有効にしたり、データをメモリカードにバックアップしたりできます。



カメラ



Highscreen Alpha Rageのカメラは、満足のいく品質で撮影します。 実際には、前面のものは許され、特別な品質は必要ありませんが、メインのものからもっと欲しいです。

マシンの周囲光が良好であれば、写真の細部とノイズは悪くありませんが、暗くなります-露出補正は避けられません。 色の再現は冷たいトーンに少し傾いていますが、これはほぼすべてのスマートフォンで多かれ少なかれ典型的です。

光が不足すると、オートフォーカスの動作におけるホワイトバランスの罪、ノイズ、問題が発生しますが、これも一般に予想されていました。 しかし、私が期待していなかったのは、カメラメニューの機能です。 スマートオートモード、パノラマモード、HDRがあります！ 正しい言葉、私はこのような手頃な価格のデバイスからのこのような豊富な機会に喜んで驚きました。











カメラは、毎秒30フレームの速度で1280 x 720ピクセルの解像度でビデオを書き込みます。



評決



ひどい中、親切な内面-これはちょうどHighscreen Alpha Rageです。 見た目が反発的なこのスマートフォンは、特に低価格であることを考えると、消費者の質の良さ（ハックニーの音に関係なく）があります。 したがって、ターゲットオーディエンスが（テーマフォーラムで判断して）関心を高めていることは驚くことではありません。
