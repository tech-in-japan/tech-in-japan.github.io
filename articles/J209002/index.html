プロトタイプ作成用のステンシル+ノートブックの概要

しばらくの間、そもそもWebサイト/プログラムインターフェースの開発者にとって興味深いオブジェクト、つまりiOSインターフェース用のステンシルとメモ帳を手にしていました。 コメントで一度、 UI Stencils製品が言及されましたが、情報はほとんどなく、テストを受けることにしました。







以下は私の主観的な意見であり、あなたの意見と一致しない場合があります。



ご注意
現在自分でステンシルを提示しようとしたが、メーカーの材料を使用する方が良いという結論に達しました。


ステンシル







ステンシルはキットとして提供されます。

  • ステンシル
  • シャープペンシル
  • 柔らかいプラスチックケース
  • メーカーステッカー


ステンシルの説明：

  • スケールを140％増加
  • Apple iOS7仕様に適合
  • ステンレス製
  • 原産国USA


ステンシルはステンレス製で、しっかりと見えます。 それが十分に薄いという事実にもかかわらず、ステンシルは十分に強く、それ自体に関して不注意を許す可能性がありますが、注意する方が良いです。











ステンシルは、iOS7の仕様に従って作成されます（会社は、仕様の更新時にステンシルを更新します）。 切り抜き図に加えて、開発者向けのヒントがステンシルに適用されます。たとえば、アクション（タップ、ピンチ、回転）を示す方法などにより、ステータスバーなどの標準要素のサイズを決定し、ステンシルに穴が開けられます。要素の境界線を描くことができます



スケッチブック







ノートブックは薄いビニール袋でのみ供給され、縫い目で裂けがちです。 これは重要ではありません、なぜなら 開梱後すぐに、ノートブックはノートブックに戻らず、メインタスク-輸送中のノートブックの保護-を完全に実行しました。



ノートブックの説明：

  • 寸法：215 x 279 mm（ほぼ正確なA4形式）
  • 両面印刷
  • ドットグリッド、ステップ10px
  • 140％の電話パターン
  • フィールド「プロジェクト名」、「画面」、「日付」、「メモ」
  • 50枚
  • 厚紙の裏表紙


印刷は高品質で、インクが汚れることはありません。 シートは密度が高く、手触りが快適です。

オプショナル



描かれたプロトタイプをデジタル化するために、製造業者はプロトスケッチプログラムを開発しました。 このプログラムにより、スケッチを認識し、インタラクティブにし、同僚と共有できます。



プログラムへのリンク： http : //www.appstore.com/protosketch



残念ながら、私はアプリケーションについてこれ以上話すことができません。 手元にiOSデバイスはありません。



個人的な感情



ステンシルの使用と要素の描画を手動で比較すると、ステンシルは標準的な要素の時間を節約できると言えます。 グリッドの存在により、要素のスケールにすぐに移動できます。これは、Axureなどのプログラムでレンダリングするときに便利です。



Z.Y. iOSと他のオペレーティングシステム（モバイルとデスクトップ）の両方のためのメモ帳付きのこのステンシルは、ロシア連邦で購入できます。 気にする人は、PMでリンクを送信するか、管理者が許可する場合は投稿に追加します。
