新しいASP.NET MVC 3プレビュー1が利用可能





ASP.NET MVC 3 Web開発フレームワークの新しいバージョンがダウンロード可能です。 このリリースは暫定的なもので、コミュニティテストを目的としています。



新しいバージョンでは、次の変更と改善が行われました。
  • 新しいRazorプレゼンテーションエンジンのサポートが追加されました。
  • ViewおよびModelViewの動的プロパティを追加しました。
  • 「すぐに使える」依存性注入サポートを追加。
  • グローバルフィルターの機能を追加。
  • 新しい値プロバイダーJsonValueProviderFactory ;
  • .NET 4.0検証ツールのサポート
  • 新しいタイプのアクション結果タイプ： HttpNotFoundResult 、 HttpStatusCodeResult 、 HttpRedirectResult 。
MVC Framework 3 Preview 1は、.NET 4.0でのみ機能します。 Razorとの統合はまだ不完全です。構文の強調表示とIntelliSenseのサポートはなく、将来のリリースで登場します。



このリンクからASP.NET MVC 3 Preview 1をダウンロードできます。
