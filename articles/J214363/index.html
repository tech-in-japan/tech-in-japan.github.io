Windows 8.1およびWindows RT 8.1をサポートするNokia Imaging SDK 1.1の紹介

こんにちは、Habr！



最近バルセロナで開催されたMobile World Congress展示会の枠組みで、 Androidアプリケーションといくつかの利用可能なモバイルデバイスを備えたNokia Xスマートフォンのラインナップだけでなく、展示会も開発者にとって非常に興味深いものでした。







前回の投稿では、Nokia Xプラットフォームの開発トピックについて触れましたが、本日は、Windows Phone 8に加えて、Windows 8.1およびWindows RT 8.1上のデバイスのサポートが登場した、更新されたイメージングツールNokia Imaging SDK 1.1について説明します。



Nokia Lumia用のWindows Phoneプラットフォームの開発と、新しいモバイルデバイスの登場（新しいプラットフォームを含む）に加えて、開発者ツールやその他のコンポーネントの改善に努めています。



たとえば、2013年7月に最初のイメージングSDKを開始し、これに基づいてこれまでに1,500を超えるアプリケーションが登場しました。







このパッケージの新たな更新の出現により、アプリケーションの総数がさらに増加し​​（特に新しいプラットフォームのサポートのおかげ）、既存のアプリケーションの開発者がNokiaツールの改善された可能性を活用できるようになることを願っています。



ツールキットのすべての機能の詳細なレビューをすでに行っているため、本日はNokia Imaging SDK 1.1の主な革新についてお話します。

今回、ノキアのトレーニングパートナーであるミハイルサマリンも、私たちを支援することに同意し、更新されたツールのインストールと操作に関するビデオチュートリアルを作成しました。

Windows 8.1およびWindows RT 8.1のサポート



Windows RT 8.1ベースのLumia 2520タブレットがNokiaファミリーの一部になったので、同じAPIセットにより、更新されたSDKを使用してWindows 8.1およびWindows RT 8.1の互換性のあるアプリケーションを作成できます。



さらに、Windowsプラットフォームファミリの大きな可能性により、モバイルプラットフォームからアプリケーションを移植することは難しくありません。 これを行う方法については、こちら（http://msdn.microsoft.com/en-us/library/dn617207）を参照してください。



「このビジネスで最も重要な側面は、開発者が消費者向けの新しいタイプの製品を作成できることです」と、開発者向け製品マーケティング責任者のサミ・ピプリはコメントしています。 「また、同じコードを使用して、携帯電話とタブレット、およびPCの両方で起動できるアプリケーションを作成できます。」

50以上の組み込みフィルターを備えたイメージングSDKには、新しい追加が加えられました。 人気のあるセピア、白黒、およびクロマキーに加えて、前景セグメンテーション、ボケ効果、および単一画像のHDR効果が追加されました。



Windows Phone 8およびWindows RT 8.1 / Windows 8.1用のアプリケーションで、フィルターを追加、構成、およびカメラの動作を最適化するには、マニュアルを参照してください。

最適化されたアルゴリズム



更新されたSDKの最適化されたアルゴリズムにより、開発者は以前より高価で強力なLumia向けに意図されていた機能をすべてのアプリケーションに実装できます。 これは、Lumia 520などのデバイスで画像を拡大および回転できることを意味します。

「現在、NokiaからSDKに独自のアプリケーションを作成するために使用されているのと同じテクノロジーを移行しています」とSami氏は言います。 -私たちのチームは、Lumia 520のような単純な携帯電話で、より高価なLumiaモデルに備わっていた機能と画質を見ることができるように技術を最適化することができました。 このメカニズムは、Lumia 1020とLumia 520のような製品間の障壁を曖昧にします。」



SDKバージョン1.0互換



SDKアップデートは、安定版リリース1.0に基づいています。 以前のAPIをサポートし、新しいバージョンにシームレスにアップグレードできます。 このAPIは、マネージ（C＃、Visual Basic）コードとネイティブC ++の両方から呼び出すことができます。



Nokia Imaging SDKバージョン1.1アップデートがダウンロード可能になりました 。 また、詳細なマニュアル、ドキュメント、サンプルアプリケーションもあります。



さらに、大きなコンテストImagin8 Missionに参加し、DVLUPで新しいミッションを完了する（新しいSDKでアプリケーションを作成することを読む）ことで、勝者にとって本当に面白い賞品を競うことができます。 コンテストの詳細と賞金の説明はこちらです。
