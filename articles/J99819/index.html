Sphinx 1.10-betaリリース

3日目に、 ブラックジャックとwhores RTインデックス、文字列属性、最適化されたインデックス形式、および従来の12個または2個の小さな機能を備えた Sphinx 1.10-betaを投稿しました。 昨日、彼のためにRPMをここにレイアウトしました。



1.xのサガは、いくつかの理由（最後はbinlogs）で引きずられました。Habréの投稿はRPMでした。 しかし、今では次のようにできます：



$ wget sphinxsearch.com/downloads/sphinx-1.10-1.el5.x86_64.rpm

$ rpm -i sphinx-1.10-1.el5.x86_64.rpm

$ service searchd start

$ mysql -h 0 -P 9306

mysql> INSERT INTO testrt VALUES ( 123, 'hello', 'world', 456 );

mysql> SELECT * FROM testrt WHERE MATCH('hello');







ランダムで面白い事実：トランザクション、 V.I。Lenin innodb_flush_trx_log_on_commitにちなんで名付けられた特別な設定などもあります （12個または2個の小さな機能を参照してください。）



今月のモットー：ほとんどデータベース、検索のみがまだ機能しています！
