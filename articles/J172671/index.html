Googleは春の大掃除を手配し、AndroidでAdBlockを禁止します

Googleは1日で多くのニュースを生成することができました。Androidの新しいボスであるPichaiを任命し 、春の大掃除サービスを手配し、広告をブロックするアプリケーションのGoogle Playをクリーンアップしました。 まず最初に。



検索の巨人は、一連のサービスを閉じるという毎年の伝統を受け継いでいます。 多くの場合、これらはニュースと閉鎖からのみ学ぶサービスであり、Google Code Searchなどの便利なサービスはしばしば閉鎖されました 。 今日、残念ながらGoogleは後者を選択しました。 だから、最も重要で悲しいこと：

  • Googleリーダー -Rapida 投稿の詳細
  • Google Cloud Connect-ドキュメントをGoogleクラウドに保存できるMS Office用のプラグインは、Googleドライブを支持して終了します。 実施日-今年4月30日
  • CalDAV API- 9月16日から、ホワイトリストの開発者のみが利用できるようになります。 他のすべてのユーザーはCalendar APIを使用する必要があります 。 ホワイトリストに登録したいですか？ ここに書いてください 。
  • Googleは、建物自体の3Dモデルに対応すると考えているため、6月1日にGoogleビルディングメーカーを利用できなくなります。Googleマップメーカーは、他のニーズにも対応します
  • GUIビルダーとUiAppウィジェットは、 HTMLサービスを支持して閉鎖されます


さらに、 Google Voice for Blackberryは、WindowsおよびMac OS XのショッピングおよびSnapseedプロジェクトの検索APIであり、ディストリビューションに含まれていました。



Android広告ブロック



Googleは本日、AdBlockなどのすべてのアプリケーション開発者に、開発者配布契約の条項4.4に従ってすべてのアプリケーションがブロックされたというニュースを喜んでいます。

禁止行為
お客様は、Androidユーザー、Google、モバイルオペレーターを含むサードパーティのデバイス、サーバー、ネットワーク、その他のオブジェクトやサービスの動作を中断または中断させたり、その他の損害を引き起こしたりする行為（製品の開発および配布を含む）を行わないことに同意します。 Androidマーケットで取得した顧客情報を使用して、Androidマーケット外で製品を販売または配布することはできません。




そのため、 AdBlock Plus （Habr は彼を温かく歓迎してくれましたが ）、 AdFree 、 AdBlockerなどが配布されました。 これらのアプリケーションが正式に広告を通じてアプリケーションを収益化するためのツールに実際に害を与えることは明らかですが、手順自体はあまり美しくありません。 Firefox向けAdBlockは正常に機能するように見えるため、インターネット上の広告から自分を守ることができます。



PSそれでも、 公式サイトから直接ABPをダウンロードし、サイドルッキングツールを使用してAndroidデバイスにインストールできます。
