Vinnitsaでの最初のSiclum .NET土曜日にあなたを招待します！

12月末には、すべての.NET開発者とファンを招待して、 12月24日のVinnitsaでの最初のSiclum Saturdayに参加し、聞いて、経験と知識を共有するよう。



伝統によれば、 Siklum Subbotniksの間、イベントの参加者は、.NETでの開発のための技術的なトピックに関するレポート中だけでなく、イベント全体を通して、経験、新製品、ベストプラクティスを交換し、非公式にコミュニケーションします。 この形式は、ウクライナのさまざまな都市の多くの参加者にアピールしました。たとえば、最近、 ミンスク 、 キエフ 、 ハリコフで.NET Subbotnikを実施しました 。



今回は、Siklum Subbotnikを新しい都市で過ごします。 したがって、12月24日に、.NET-TECHNOLOGIESで、長年のソフトウェア開発、およびパフォーマンス、ポッドキャストの録音、単純に興味深いodnodumtsyの両方で犬を食べた非常に興味深いスピーカーをいくつか聴きます。







私たちのプログラムはまだ形成されており、今日は次のとおりです。

  1. 「WinRT。 聖なる牛」-エフゲニー・ジャーコフ（habra-user jeje ） 、ニコラエフ。 すべての取引の開発者。 Silverlight MVP / MCTS、.NET、MVC、WP7、Kinect、PHP、Magento、JS、2Gigポッドキャストの著者、WP7Rocks ）。 レポートでは、Eugeneはそのようなポイントを強調します。.NETとWindowsランタイムの関係。 C＃およびVisualBasicからWindowsランタイムAPIを使用する C＃およびVisualBasicでWindowsランタイムコンポーネントを構築する
  2. 「オリジナルの窓電話」-Alexander Krakovetsky（saber） 、開発者、Microsoftの地域ディレクター、ASP.NET MVP、Vinnitsa Microsoft User Groupのリーダー。 アレクサンダーは、次の分野で知識を共有します。メトロ言語。 ウクライナの開発者向けの市場コンテストへの登録。 市場での出版：成功事例、Q＆A
  3. 「Windows PhoneとSharePoint 2010は、いっそう楽しくなります」-セルゲイベルスキー 。 SharePointコンピテンシーセンター（MVP、MCTS、MCPD）の責任者。 IT分野での10年以上：C＃開発者、Dynamics CRM、SharePoint Server、開発チーム長、ITディレクター。 Sergeyが語る：アクセスシナリオ、クラスと認証方法の使用、FBA認証の使用例、Windows Phoneを使用したSharePointコンテンツの管理。




プログラムは補足および更新されます。 今年の.NETオタクに耳を傾ける最後の機会をお見逃しなく

時間 ：（予備）12:00から15:30
