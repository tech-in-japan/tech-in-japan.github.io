Windows XPはサポートされなくなりました。 ただし、一部の出版物は公式の死亡日から何年も遅れました。 Embedded Standard 2009は1月にようやくオフになり、Embedded POSReady 2009の形での拡張実装サポートは4月9日に終了しました。



数日前の2019年4月5日、MicrosoftはKB4487990を使用した「ラストオブザモヒカン」の最新アップデートをリリースしました。これにより、サントメプリンシペとカザフスタンのKyzylordaのタイムゾーンが修正されました。



その後、完全な沈黙がありました。 会社はすべての生命維持システムをオフにしました。 患者は死亡しており、再びcom睡状態から抜け出すことはありません。



残念なことに、Windows XPのほとんどの亜種に対するグローバルなサポートは、大声で叫び声を上げ、歯ぎしりをする中、企業が突然通常のプラットフォームからどこかに行かなければならないことに気付いた2014年に終了しました。 XPは2001年以来インストールに使用できますが、多くの人が悲惨なVistaを見逃し、更新を拒否する傾向があったため、かなりの数のXPワークステーションが今日まで残っています。



幸いなことに、Windows 7は非常に優れていたため、ほとんどの人がWindows 7を使い始めることができました。 しかし、すべてではありません。 Windows XPは、「終了するにはあまりにも良い」と呼ばれる問題に真剣に苦しむ最初のOSでした。これにより、エンドユーザーは、OSが黄金期を過ぎた後も十分な時間アップグレードを拒否しました。



これは、一部はWindows Vistaに対する反応であり、一部はXPの旗艦ステータスが異常に長くなった結果です。 Bill GatesがWindows XP Service Pack 2（SP2）のセキュリティに集中することを決定したとき、Microsoftは多くの重要な変更にもかかわらず、OSのブランド変更またはリリースを行いませんでした。 XPより前のマイクロソフトは、1992年にWindows 3.1、2000年にWindows 95、98、Windows 2kおよびMEのコンシューマオペレーティングシステムをすばやく分類しました。 XPの任期ははるかに長いことが判明しました。これは、Microsoftの5年以上（2001〜2006年）の主要なコンシューマOSであり、2009年にWindows 7がリリースされるまで市場で人気がありました。
