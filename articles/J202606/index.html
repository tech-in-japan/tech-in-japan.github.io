多くのブラウザでは、ダウンロード停止ボタンと更新ボタンは1つにまとめられています。 したがって、ページをロードするための過度の努力のためにそのようなボタンを持つブラウザがフリーズし、テキストがロードされて読みたい場合-そのようなボタンをクリックすると、停止または2回目の更新として機能し、ダウンロードを再度待つ必要があります。



したがって、Broブラウザでは、[停止]ボタンと[更新]ボタンは2つの異なるボタンであり、どちらも画面の左上隅にあります。



画面の左側にあるコントロールボタン



[ 停止 ]ボタンには、ページの読み込みの進行状況も表示されます。 ページが読み込まれると、[停止]ボタンがアクティブになり（赤に変わり）、進行状況インジケーターがその上の正方形のアイコン内をクロールします。 ダウンロードが終了すると、[停止]が再び灰色に変わり、ボタンの下部にページの読み込みにかかった秒数が表示されます。







星の付いた緑の[ 更新 ]ボタンは常に使用可能です。 矢印の付いた[戻る]ボタンと[進む]ボタンは、それぞれ後方または前方に登れるときにアクティブになり、青色になります。







フラグの付いたボタンはブックマークページに移動します（ところで、ブラウザはこのページから動作を開始します）。 同じボタンにチェックマークと「プラス」を付けて、現在のページをブックマークに追加します。







ブックマークの編集（削除、スワップ）は、パスケードのフォルダーにあるbro.iniファイルに移動できます。 ブックマークページの上部にある[編集]リンクをクリックすると、簡単にアクセスできます。 このファイルをメモ帳で修正した後、ブックマークを再度呼び出した後、変更されたブックマークがリストに表示されます。







最も興味深いボタンは、幅の狭いボタンと細い矢印とマウスです。







Ctrl + Mを押すと、 「マウス」が赤くなりアクティブになります。キーボードの矢印を使用してマウスカーソルを制御し、マウスの左ボタンをスペースで押し、マウスの右ボタンをZキーで押します。 ブラウザをキーボードから制御したい場合、このモードがないと、通常、Webページの目的の行/フィールドにフォーカスを置いたり、リンクをクリックしたりすることができません。 そのため、このモードが追加されました。



しかし、狭い矢印の付いたボタンをクリックすると、下の画面で仮想キーボードが表示/非表示になり、それを使用して、アドレスバーのタッチスクリーン（Rus / Latボタンで切り替える）にロシア語と英語のテキストを入力できますブラウザのフィールドで：

画面の右側にあるコントロールボタン







右側の4つのボタンにはレバーがあり、ページを上下にスクロールできます 。 画面上で指を繰り返しスクロールしてページをスクロールする現代的な方法が好きではありません。 画面上の1か所をクリックするだけで、ページ自体が表示されるようになりました。 静止時には、レバーは中央にあり、ページも立っています。 しかし、真ん中の上のレバーフィールドをクリックする価値があります-そして、ページは、真ん中から遠くなるほど上がります-速度が速くなります。 同様に、中央の下をクリックするとページがスクロールします。



Shiftキーを押したままにすると、同じレバーがページのスケールを制御します。



キーボードの[上]または[下]を押すと、レバーはプログラムで指定された値にずれます。 ただし、 Ctrlを押したままにすると、レバーが大きく外れます。 Altを押したままにすると、ページは水平方向ではなく垂直方向にスクロールします（一部の幅の広いページの場合）。



スケールアップして、元に戻したい場合、 Ctrl + Shift + Downの組み合わせは、文字を小さくしませんが、スケールを100％にリセットします。



さて、右下の最後の4つのボタンは、イメージ（DLCTL_DLIMAGESフラグ）、フラッシュ（DLCTL_NO_DLACTIVEXCTLSおよびDLCTL_NO_RUNACTIVEXCTLSフラグ）、Java（DLCTL_NO_JAVA）およびスクリプト（DLCTL_NO_SCRIPTS）をアクティブにするためのものです。 ボタンは赤（オフ）と緑（オン）になります。 デフォルトでは、これらはすべてオフになっており、テキストのみがロードされます。







また、 Altを押したままにすると、左右のボタンには、キーボードから「押した」キーボードショートカットが表示されます。







また、F10キーを押すと、アドレスバーでシングルクリックが行われます。 したがって、 F10を2回押すと 、カーソルがアドレスバーに置かれ、現在のアドレス全体が選択されます（後でBackspaceキーを押して削除し、別のアドレスを入力します）。



キーボードのレバーは、アドレスバーにフォーカスがある場合にのみ適切に機能します。フォーカスがブラウザフィールド内にある場合、最初にダブルF10を押してから、レバーのみを使用する必要があります（物理キーボードを上下に押す）。 そうでない場合、ブラウザフィールドはナビゲーション自体のキーストロークを解釈します。

IRCには「サービス」があります（特に、ircnet.ruネットワークでは以下に説明しますが、他のネットワークでは名前が異なるか、まったく存在しない場合があります）。

NickServ-ニックネームの登録、パスワードの変更、自分に関する情報の変更、他の登録されたニックネームに関する情報の要求を管理します

ChanServ-チャンネル（チャットルーム）の登録と削除を制御します

MemoServ-オフラインメッセージングを管理します。 登録したユーザーにメッセージ（およびいくつかのメッセージ）を残すことができます。ユーザーは通知を表示し、次回ログイン時にこれらのメッセージを読むことができます。



すべてのサービスには詳細なヘルプシステムがあります。プライベートサービスで「help」コマンドを記述するだけで、チャットでこのヘルプが返されます。 たとえば、「/ w NickServ help」または「/ ns help」。 「/ cs help」および「/ ms help」と同様です。



したがって、 現在のニックネームを登録するには、NickServプライベートサービスに書き込む必要があります： "/ w NickServ register email password"または簡潔に/ ns register email password " 、ここでpasswordはパスワードで、emailはメールです、しかし、IRCで問題が発生した場合、誰か（サーバー管理者など）がIRCを通じて通信すると信じられていますが、そのような通信は見たことがありません）。

IRCに初めて行ったときは、どのチャンネルにも放り込まれません。 どうする？ 「/ list> 10」と書く-これにより、10人以上がオンラインになっているすべてのチャンネルが検索され、チャットが一覧表示されます。





リストが長すぎる場合は、キーボードの任意のキーを押して出力を中断できます。 リストには、チャンネルの名前（＃記号で始まる）とチャンネルのトピック（テーマ）が表示されます。 IRCには、ユーザーボットが座っているクイズチャネルがあり、彼と一緒に遊ぶことができます。彼はパズル、アナグラム、質問を作成し、残りのユーザーはポイントを推測して得点し、全体的な評価で特定の位置を占めます。 IRCには、マフィアをプレイするボット、ジョークを伝えるボット、さまざまな面白いコマンドを実行するボット、リクエストに応じてウィキペディアに登り、必要な記事のテキストの通過を示すボット、正確な現在時刻を示すボット、など



チャネルを入力するには、 「/ join #channel name」または「/ j #channel name」を入力します。 チャンネルを終了するには、 「/ part＃channel name」または「/ p＃channel name」と入力します。 ログインしている各チャネルには、1以上の一意の番号が割り当てられます。 現在のチャネルのリストは、入力行の上に表示されます。





また、入力行に何かを入力した場合に書き込む現在のチャネルの番号が、ウィンドウの左上にあるボタンに表示されます。 ボタンを押すと、現在のチャンネルの番号が次のチャンネルに変わります。 アクティブな3番目のチャネルを持つボタンは次のとおりです。





また、入力フィールドでチャンネル番号を変更できます。たとえば、チャットで「/ 3」と入力してスペースバーを押すと、チャンネルはすぐに3番目に変更されます。



したがって、番号でアクセスしてチャネルを終了できます。 「/ p 2」を使用すると、チャネル番号2を離れることができ、チャネル番号3以降は自動的に番号を1つ減らします。



任意のチャンネルをお気に入りリストに追加し、パスワードを入力してからチャットに入るたびにそれらを開くには、 「/ ns ajoin add #channel name」と記述する必要があります。 コマンド「/ ns ajoin del＃channel name」を使用して、お気に入りリストからチャンネルを削除することもできます。 このような長いコマンドを記述しないために、画面の左側にボタンがあります。





これらのボタンを使用すると、現在のチャンネルをお気に入りのチャンネルのリストに追加したり、そこから削除したりできます。 操作が完了すると、コマンド「/ ns ajoin list」が自動的に呼び出され、お気に入りのリストに残っているチャンネルが表示されます。



チャット自体では、入力フィールドにテキストを入力してEnterキーを押すだけで、チャンネルで簡単に通信できます。 また、 「/ w nickname text message」コマンドを使用して、 プライベートの誰にでも書き込むことができます。



画面の左側にある下矢印のボタンを押すと、チャットウィンドウをタスクバーに最小化できます。





チャットラインに何も入力せずに、現在のチャンネルを離れることができますが、画面の左側にあるPボタンを押すだけです。





また、「招待」はIRCでは一般的です-これは誰かが何らかのチャンネルにあなたを招待するときです。 画面の左側にある「I」の文字が付いたボタンをクリックするか、チャットに「/ i」を入力すると、招待を確認してチャンネルに入ることができます。
