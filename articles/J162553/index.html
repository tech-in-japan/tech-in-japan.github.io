Windows Phone 7.xのサポートは2014年に終了します

Windows Phone 7.x電話を所有しているすべての人に朗報です。 多くの人は、Microsoftが古い携帯電話を持つWindows Phoneユーザーを放棄すると信じていましたが、これは完全に異なることが判明しました。







Microsoftは、2013年初頭に予定されているアップデート7.8のリリース後、このバージョンのOSのサポートはさらに18か月間続くと述べています。 バグ修正に加えて、この間にもう1つ（または1つではない）の「大きな」更新プログラムがリリースされる可能性があります。 引退するためにあなたの古い携帯電話を送信するためにとても早い。



現在の電話が気に入っている場合、またはWindows Phoneで安価な電話を探している場合は、Windows Phone 7.xで予算の電話を簡単に購入できます。



これは朗報です。 また、Windows Phone 7の最初のバージョンのリリース日から2014年の終わりまでの時間をカウントすると、4年間のサポートが得られます。これは、モバイルプラットフォームでは前例のない数です。 iPhoneでも3年間アップデートを受け取ります（ただし、ユーザーは古いバージョンのアップデートではなく最新のOSバージョンを入手します）が、Windows Phone 7.8はほとんどすべての新しいG8機能のサポートを要求していることを考えると、これら2つの数字は関連する。 多くの場合、賢明なアップデートをまったく受け取らないAndroidデバイスの所有者について私たちは何を言うことができますか。



Windows Phone 7.xのコアはWindows CEであり、8番目のバージョン-NTであるため、Microsoftのこの動作は部分的に理解できます。 ただし、一方で、少なくとも一部の電話機は7.xから8.xにアップグレードできます。 たとえば、Lumia 710/800/900は、512 MBのRAM（HTC Windows Phone 8SおよびNokia Lumia 620など）があり、シングルコアであるにもかかわらず、プロセッサをウィークと呼ぶことはできません。



まあ、私たちは持っているものを持っています。 Windows Phoneは、サーフィン、メール、ナビゲーションにグリッチのない安全なデバイスが必要な場合、膨大な数のゲーム（これまでのところあまり多くない）が必要ない場合、または何か新しいものが必要な場合に、Androidに代わる優れた選択肢です。 レドモンドの人々が私たちを何度も喜ばせ、Windows PhoneがモバイルOS市場でその地位を占めることを願っています。
