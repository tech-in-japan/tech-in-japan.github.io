Linux VPSサーバーの購入方法

遅かれ早かれ、インターネットでの活動に関連する人はサーバーを必要とし、開発されたサイトをより生産的なサイトに移動（または1C-Bitrixを起動）し、インターネットサービス、Counter-Strikeを備えたゲームサーバー、または並外れたRPGを起動します。 はい、あなたは現代のウェブではこれらの目標を決して知りません。



そのため、最近、以前の共有ホスティングから、より高速で機能的なものに切り替える必要がありました。 私はホームサーバーを維持するのにうんざりし、VPS（Virtual Private Server）サーバーで停止できることに決めました。これは私の問題に対する完全に手頃で便利なソリューションです。



現在のタスクは、RoRの共同開発（およびトレーニング）と、管理を容易にするために、サイトを持つクライアントのペアを1つのサイトに転送することです。 選択するのが難しい（今日は多くのVPSを提供する）ため、私の経験が誰かの時間を節約するか、基準と評価方法を教えてくれることを決めました。



始めましょう。



VPSを選択するとき、主にそのパフォーマンスと価格に関心があります。



この記事では、Linux VPSサーバーの平均パフォーマンス指標を見つけようとします。 これを行うには、7つのプロバイダーの仮想サーバーをテストします。 テストされた各VPSには、2つのプロセッサコアと2GBのRAMのパラメーターを含む平均料金がありました。



この投稿の目標は、平均パフォーマンスカテゴリでLinux VPSサーバー市場の平均パフォーマンスインジケータを決定することです。



phoronix-test-suiteを使用したRAM、CPU、ディスクのテスト





インストール：





wget phoronix-test-suite.com/releases/repo/pts.debian/files/phoronix-test-suite_5.0.1_all.deb







apt-get update



apt-get install php5-cli php5-gd



dpkg –i phoronix-test-suite_5.0.1_all.deb



または代替オプション：



apt-get update



apt-get install php5-cli php5-gd



apt-getインストールphoronix-test-suite



利用可能なすべてのテストのリスト：



phoronix-test-suite list-available-tests



テスト：



pts / iozone-ディスク-4kbのブロックでの読み取り/書き込み、ファイル512MB



pts / compress-gzip-プロセッサー



PTS / ramspeed-メモリ



テスト開始：



phoronix-test-suite run pts / iozone pts / compress-gzip pts / ramspeed



チャンネルのテスト：





SpeedTestをインストールします。



wget raw.github.com/sivel/speedtest-cli/master/speedtest_cli.py







chmod a + rx speedtest_cli.py



mv speedtest_cli.py / usr / local / bin / speedtest-cli



chown root：root / usr / local / bin / speedtest-cli



テスト実行：



speedtest-cli --share



別のサーバーを選択する必要がある場合は、サーバー番号を確認します。



speedtest-cli --list | more



以下を開始します。



speedtest-cli --server 4718（4718サーバー番号、この場合はBeeline（モスクワ、ロシア））
