RIP Windows XP





... Windows XPの第一印象を覚えていますか？ 私が覚えているのは、「ついに通常のWindowsを作った！」12年前、バグのあるWindows MEの直後に古いXPが登場し、その後、大きくてうるさくてダムのないマシンを使用しました。 今日、私はこれらの「PC」の3倍強力なタブレットでこのテキストを入力しています。Androidは徐々にデスクトップOSに取って代わります。 興味深いことに、XPはMicrosoftの最長稼働システムです。Windows95とMEはそれぞれ6年、Windows 98は最大で8年続きました。 今日、Windows XPの最後の日には、その最高の機能を思い出すことをお勧めします。 その後、MSは、役に立たないタイルの時代がまだ遠い前に、革新を本当に喜んでくれました。 これが私のヒットパレードです。コメントでシェアしてください。



＃5 CDを焼く

ああ、ようやくNeroを削除するのはすごかったです！ ファイルをCD-ROMアイコン「次へ-次へ-次へ」にドラッグアンドドロップすると、ディスクの準備ができました！



＃4以前のドライバーを復元する

更新をキャンセルして、以前の「安定した」バージョンに戻すことができました。 私の同僚はまだこの機能が最も重要だと考えています。



＃3 Windowsファイアウォール

Microsoftの公式ドキュメントでは、この機能は「ネットワークと外界との間の保護スクリーンであるセキュリティシステム」と説明されていました。それから、当時のWindowsはパケットフィルタリングが非常に可能でした。 サポートされているPoint-to-Point Protocol over Ethernet（PPPoE）、仮想プライベートネットワーク、および鈍いダイヤルアップ。



＃2リモートデスクトップ

当時の最もクールな機能の1つ。 その後、管理者として働き、この機能は何度も役立ち、数十時間の作業を節約し、より楽しいアクティビティのために夜を解放しました。 以前の組織でWindows XPに移行した理由の1つ;-)



＃1リモートアシスタンス

この機能は以前の機能を補完するために機能し、サポートサービスが日々の責任に対処するのに役立ちました。 ユーザーは簡単なフォームに記入し、適切な時間を選択し、パスワードを設定する必要がありました。 フォームはサポートに送信され、エンジニアはリモートで接続し、問題を診断および修正できました。



これらはWindows XPのお気に入りの機能でしたが、今度はあなたの機能を共有するようお願いします。 このシステムで何が好きでしたか（または好きではなかった）？ コメントへようこそ！
