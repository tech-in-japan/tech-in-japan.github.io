トップ10 Linuxアンチウイルス

オペレーティングシステムは非常に複雑な構造であり、エラー、問題、その他の望ましくない現象が発生する場所があります。 特に懸念されるのは、ウイルス、トロイの木馬、ネットワークワーム、スパイウェアと呼ばれる人為的に作成された「現象」です。 Linuxファミリのオペレーティングシステムは、このような問題から十分に保護されていると考えられていますが、それらが発生する可能性はゼロからはほど遠いです。 マルウェアから保護するために、セキュリティの専門家は一般的にウイルス対策と呼ばれるプログラムを開発します。







今日、私たちはLinux用のトップ10のアンチウイルスを見ています。 Webでは、「最高のウイルス対策ソフトウェア」について多くの推論を見つけることができますが、推論ではなく事実を信頼する価値があると考えています。 ここで紹介するプログラムは、 独立した研究で優れていることが証明されたため、このレビューに参加しました。



1.ソフォス



調査によると、 ソフォスはLinux用の最高の無料ウイルス対策ソフトウェアの1つとして認識されています。 オンデマンドスキャンをサポートするだけでなく、リアルタイムのシステム保護も提供します。 このウイルス対策にはさまざまなバージョンがあり、Linuxに加えて、WindowsやAndroidなど、他のプラットフォームでも使用できます。 ワームやトロイの木馬を検出して削除できます。 コマンドラインを詳しく調べたい場合は、適切なインターフェースを提供します。



▍ソフォスの機能



  • 無料です。

  • GUIおよびコマンドラインのサポート。

  • マルウェアの検出と削除。

  • ワーム、トロイの木馬、ウイルス、その他のマルウェアに対する保護に適しています。

  • インストールと使用が簡単な小さなスペースを占有します。

  • クロスプラットフォーム。

  • Linuxだけでなく、脅威をブロックおよび排除できます。



2.コモド



Comodo Antivirusも最高の1つと見なすことができます。 さまざまなアーキテクチャとクロスプラットフォームの優れたサポートでよく知られています。 さらに、彼は電子メールをスキャンする方法を知っています。他のプログラムでこのような機会は一般的ではありません。 Comodo for Windowsには、32ビットおよび64ビットアーキテクチャで実行されるファイアウォールモジュールも含まれています。 Comodo Anti-Virus for Linuxは、ほぼすべてのディストリビューションをサポートしているため、Linuxユーザーの間で非常に普及しています。 Comodoのもう1つの優れた機能は、このアンチウイルスがサーバー（Red Hat Enterprise Linux Server、OpenSUSE、SUSE Linux Enterprise Serverなど）でも機能することです。



Com Comodoの特徴



  • 無料です。

  • インストールと使用が簡単。

  • オンデマンドスキャン、誤検知なし。

  • リアルタイム保護。

  • スパム保護。

  • クロスプラットフォーム。

  • Linuxサーバーディストリビューションのサポート。



3. ClamAV



おそらくこれは、Linuxコミュニティで最も有名なアンチウイルスです。 ClamAVはオープンソースプロジェクトです。無料で使用できます。 これは、トロイの木馬、ウイルス、およびその他の悪意のあるプログラムに対処できる学際的なウイルス対策と見なされます。 さらに、標準メールゲートウェイのスキャンもサポートしています。 使いやすく、すぐに動作します。グラフィカルインターフェイスがなく、端末から制御されるためです。



▍ClamAVの機能



  • オープンソース。

  • 無料です。

  • クロスプラットフォーム（Linux、Windows、MacOS）。

  • ターミナルから動作します。

  • 郵便サービスをサポートしています。

  • POSIXと互換性があります。

  • インストールは必要ありません。



4. F-PROT



F-PROTは、Linux用の有名なアンチウイルスです。 国内のニーズと組織の両方で使用できます。 32ビットおよび64ビットのソフトウェアアーキテクチャをサポートしています。 彼は2119958以上の既知のウイルスとその亜種を検出できます。 また、トロイの木馬との戦いや、ウイルスのブートにも使用できます。 このウイルス対策はインストールを必要としません。cronジョブを使用してスケジュールスキャンを実行できます。



-F-PROTの機能



  • 無料です。

  • インストールは必要ありません。

  • 2100万を超える脅威を検出できます。

  • さまざまなソフトウェアアーキテクチャをサポートします。

  • 外部記憶媒体を確認できます。

  • スキャン時に、ブートウイルス、マクロウイルス、およびトロイの木馬を検出できます。



5. Chkrootkit



このアンチウイルスの名前Chkrootkitから、rootユーザー権限で機能すると想定できます。 そして実際には-ありのまま。 これは、率直に言って、Linuxでルートキットを見つけるための最も成功したソリューションです。



このプログラムはスペースをほとんど消費せず、インストールは不要です。 CDまたはUSBドライブに書き込むことができます。 Chkrootkitパッケージには、最も要求の厳しいユーザーのニーズを満たすことができる多くのモジュールが含まれています。



Ch Chkrootkitの機能



  • ルートキットの検出。

  • 小さいサイズ。

  • インストールは必要ありません。

  • 使いやすく、高速。

  • ターミナルから起動されます。

  • 豊富な機能があります。



6.ルートキットハンター



ルートキットハンターは、別の優れたルートキット検索ツールです。 このプログラムは端末から動作し、ルートキット、バックドア、その他のローカルエクスプロイトを認識し、通常のテキストで悪意のあるコードを検索し、マルウェアの特徴的な兆候の有無をシステムでチェックします。 Rootkit Hunterは動作するためにBASHを必要とし、Linuxディストリビューションの広範なセットをサポートします。



ルートキットハンターの機能



  • ルートキットの検出。

  • コマンドラインインターフェイスのサポート。

  • 使いやすく、高速。

  • テキストファイルのスキャンをサポートします。

  • インストールは必要ありません。



7. ClamTK



ClamTKは、シンプルなグラフィカルインターフェイスを備えたClamAVの高度なバージョンであり、幅広いユーザー向けに設計されています。 グラフィカルインターフェイスの存在により、このウイルス対策は非常に使いやすく、オンデマンドでスキャンするためのサブシステムがあります。 プロジェクトはGTKライブラリを使用してPerlで記述されています。



▍ClamTKの機能



  • オープンソース。

  • 無料です。

  • GUI

  • オンデマンドスキャンをサポートします。

  • インストール不要



8. BitDefender



BitDefenderが次のアンチウイルスになる可能性があります。 彼は自分の仕事をうまくやる ただし、1つの「しかし」-プログラムは無料ではなく、試用版をダウンロードできます。 ただし、これにより、このウイルス対策のメリットが損なわれることはありません。



▍BitDefenderの機能



  • アーカイブをスキャンします。

  • デスクトップ環境との統合。

  • 直感的なグラフィカルインターフェイスとコマンドラインからの操作機能。

  • 保護されたディレクトリ内の感染ファイルを隔離します。



9. ESET NOD32アンチウイルス4



テストでは、 ESETはランクのトップであり、Linuxの最高のウイルス対策として認識されています。 ただし、製品は有料で、無料でしか試すことができません。 確かに、このウイルス対策の機能により、求められているお金に見合うだけの価値があると言えます。 これはLinux向けの最高のアンチウイルスであると言えますが、他のOS用のバージョンもあります。



▍ESET NOD32 Antivirus 4の機能



  • テスト結果に基づいてウイルスやその他の悪意のあるプログラムを検出するのに最適なプログラム。

  • ウイルスとスパイウェアに対する最高の保護。

  • ホームユーザーおよび組織に適しています。

  • ネットワークセキュリティ機能をサポートします。

  • 自動更新。



10.アバストコアセキュリティ



アバストコアセキュリティは最後に検討するプログラムですが、これは他のプログラムよりも悪いことを意味するものではありません。 アバストコアセキュリティは、テスト結果に応じて最高のウイルス対策製品のリストに含まれています。 アンチウイルスは、多くのLinuxディストリビューション、32ビットおよび64ビットソフトウェアアーキテクチャをサポートしています。 従来のウイルス保護に加えて、ネットワークセキュリティとファイルサーバー保護のためのツールが含まれています。 製品は有料ですが、試用版は無料のアンチウイルスと比較しても非常に価値があり、ある意味では勝っています。



Avアバストコアセキュリティの機能



  • スパイウェアを検出する機能を備えたリアルタイムのウイルス対策保護。

  • オンデマンドおよびスケジュールスキャン。

  • ネットワーク保護。

  • ホームユーザーおよび組織に適しています。

  • 定期的な更新。
