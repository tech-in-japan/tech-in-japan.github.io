Windows Phoneアプリで70,000ユーロの助成金を取得する方法について

Habréに入社したFrontsoftのメンバーは、「 他人の費用であなたの夢を実現する方法 」についての投稿を書きました。



投稿では、彼らは助成金を受け取る独立した方法を説明していますが、MicrosoftはAppCampusとNokiaとの共同イベントを開催し、スタートアップがAppCampus基金から20,000、50,000またはWindows Phone用のアプリケーションの作成と改良に対して70 000€。







2日以内に、設計、マーケティング、テクノロジー、およびビジネスの分野の専門家が、スタートアップが資金助成金を受け取る可能性を最大限にするために、アプリケーションのプロトタイプまたはアイデアに必要なすべての改善を行うのを支援します。 このイベントには、フィンランドからAppCampus Foundationの特別招待代表者が参加するため、選択サイクル全体とイベント自体は英語で開催されます。



みなさんの手を試して、イベントに参加してください！ 参加者の基準を入念に検討し 、登録してください。申し込みは4月28日まで受け付けられます。

MAACとは何ですか？



Mobile App Acceleration Camp-MAACは、MicrosoftとNokiaの共同イベントで、スタートアップがAppCampus基金から20,000、50,000、または70,000ユーロの資金を調達して、Windows Phoneアプリケーションを作成および完成させるのを支援するように設計されています。

  • イベントへの参加は無料です。
  • イベントはモスクワの中心部で5月20〜21日に開催されます。イベントの主要言語は英語です。
  • 申し込みは4月28日まで受け付けています。


参加者の基準



  • チームまたはメインスピーカーの英語のレベルは、プロジェクトを英語で交渉し、提示するのに十分です（選択したプロジェクトのスカイプインタビューを通じて確認）。
  • スタートアップチームは、既存のプラットフォーム（iOS / Android / WP）のいずれかでアプリケーションを作成した経験があり、設計、技術、ビジネスの能力を持っている必要があります（アプリケーションフォームの以前のアプリケーションのデータで確認）。
  • アプリケーションは市場で新しく、既存のアプリケーションを繰り返さないようにする必要があります。
  • アプリケーションは、次の2つの条件のいずれかを満たす必要があります。◦Windows Phone専用。 リリース後少なくとも90日間はiOSとAndroidでミス
  • iOSまたはAndroidに既に存在する場合、Windows Phoneのアプリケーションの機能は他のプラットフォームよりもはるかに豊富になります。
  • アプリケーションは、Windows Phoneプラットフォームの主要機能（マップ、位置認識、カメラ、近距離通信、加速度計、ライブタイル、プッシュ通知、アプリ内購入、アプリ内広告、アプリ内分析、試用など）をサポートする必要がありますおよびサブスクリプション）。
  • 優れたモダンデザインを持ち、技術的に高品質で生産的（プロト、モックアップ、デザインドキュメント）。
  • ビジネスで革新的であるために、モバイルデバイスと対話する際の肯定的なユーザーエクスペリエンスをサポートまたは開発するため、またはモバイルデバイスの使用の新しい領域を開くため。


詳細については、AppCampus Foundationの公式Webサイトをご覧ください



誰がメンバーになることができますか？



  • Windows Phone向けのアプリケーションまたはゲームを開発するスタートアップまたはチームを代表する場合、イベントに参加してAppCampus基金から資金を受け取ることができます。
  • チームまたはメインスピーカーの英語のレベルは、プロジェクトを英語で交渉し、提示するのに十分です（選択したプロジェクトのスカイプインタビューを通じて確認）。
  • スタートアップチームは、既存のプラットフォーム（iOS / Android / WP）のいずれかでアプリケーションを作成した経験があり、設計、技術、およびビジネスの能力を持っている必要があります（アプリケーションフォームの以前のアプリケーションのデータで確認）。


イベントはどこで開催されますか



このイベントは、モスクワの中心部、救世主キリスト大聖堂からそれほど遠くない場所で、技術起業家、専門家、投資家の協会-APIモスクワの中心で開催されます 。
