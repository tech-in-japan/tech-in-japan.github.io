MoreCSS-CSSを完成させましょう！





MoreCSSは、ポップアップの作成、メニュータブの作成、ゼブラスタイルのデザインのテーブルやリストなど、日常の作業に必要な重量がわずか14kbの小さなJavaScriptライブラリです。 このライブラリの助けを借りて、通常のCSSを使用してこれを行うことができます。





例： 

 a：アクティブ{
ターゲット：ポップアップ;
ターゲット幅：640px;
ターゲットの高さ：640px;
 }


このコードは、サイズが640×640 pxのポップアップウィンドウですべてのリンクを開きます（ ウォッチ ）



別の例： 

  a、abbr {
ツールチップ：タイトル;
ツールチップクラス：ツールチップ。
 }


この例は、ホバーするとツールチップを追加します（を参照 ）



インストール： 

  <link rel = "MoreCSS" href = "/ /へのパス/ MoreCSS / stylesheet.css" />
 <script type = "text / javascript" src = "MoreCSS.js"> </ script>


拡張子「moreCSS」は、cssファイルの後にのみ開始する必要があります（上記のとおり）。



その後、cssファイルで「moreCSS」のすべての機能を使用できます。
