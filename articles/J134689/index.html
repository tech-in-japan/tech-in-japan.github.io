Amazon Kindle FireタブレットにAndroid CyanogenMod 7をインストールします

Fireのネイティブインターフェイスは非常に貧弱です。 市場ではアプリケーションの選択が限られており、それらを購入することも非常に困難です。 アイスクリームサンドイッチランチャーは、通常の「市場」である洗練された方法でインストールできますが、それほど便利になることはありません。 CyanogenModと呼ばれるAndroid 2.3.3の素晴らしいオープンソースビルドがあります。 彼らはFireにそれを十分に速く「固定」することができましたが、100％作動可能ではなく、音がありませんでした。 そして昨日、すべてが起こりました。 Fireでほぼ本格的なCM7。





すべてのアクションを自分の責任とリスクで実行してください。 アップグレード中にデバイスが破損した場合、作成者は責任を負いません！



ソースデータ：

-完全に充電されたAmazon Kindle Fire

-デバイスは既にルート化されている必要があります

-microUSBケーブル

-インストールされたAndroid 2.3 SDK

-adbユーティリティスキル



まず、いくつかのファイルをダウンロードする必要があります（narod.ruに投稿されます。利用できなくなった場合は、私に書いてください）



narod.yandex.ru/disk/34467703001/update.zipシステム自体

narod.yandex.ru/disk/34467736001/updaterecovery.img更新ローダー

narod.yandex.ru/disk/34467328001/stockrecovery.imgファクトリブートローダー

narod.yandex.ru/disk/34467254001/log

ブートローダーのnarod.yandex.ru/disk/34467247001/last_logファイル

narod.yandex.ru/disk/34467236001/gapps-gb-20110828-signed.zip Googleスタータープログラムスイート（市場、カレンダー、連絡先など）



私のサーバーには、すべてのプログラム2de.ru/downloads/CM7.zipを含むアーカイブがあります



md5sums

updaterecovery.img 38eb5308439a6f1e256a4914ab06d508 updaterecovery.img

stockrecovery.img c4f028310bd112649c94be6a4171f652 stockrecovery.img



Fireをコンピューターに接続します。 すべてのファイルをコピーします（デフォルトでは、/ mnt / sdcardフォルダーに移動します）。 画面ボタン「USBの切断」をクリックします。



コマンドプロンプトで、次のコマンドを入力します。



念のため、ROMに書き込む内容のチェックサムを確認します。 



adb shell su cd /sdcard md5sum update.zip md5sum updaterecovery.img




ファイルをコピーしてダウンロードします。 



 cd cache mkdir recovery cd / cp /sdcard/log /cache/recovery/ cp /sdcard/last_log /cache/recovery/


ブートローダーをサービスエリアにコピーします。 



 dd if=/sdcard/updaterecovery.img of=/dev/block/platform/mmci-omap-hs.1/by-name/recovery idme bootmode 0x5001




Fireを再起動します。 



 reboot




次に、ファームウェアの更新メニューが表示されます。 メニュー項目に注意を払ってはいけません。 このブートローダーは、ボタン付きのデバイス用に設計されています。 この例では、変更されています。 電源ボタンを1回押すと、次のメニューが表示され、2回目に押すと、ファームウェアの更新が開始されます。 碑文で終わる-SDカードからのインストールが完了しました。



Googleのプログラムの更新を準備しています。 

 adb shell cp /sdcard/gapps-gb-20110828-signed.zip /sdcard/update.zip




もう一度、POWERボタンで二重操作を実行します。



元のブートローダーを復元します。 

 mount sdcard dd if=/sdcard/stockrecovery.img of=/dev/block/platform/mmci-omap-hs.1/by-name/recovery reboot


以上です。



PS。 フォーラムxda-developers.comに基づいて作成

PPS Fireに存在するbluetoothをCM7に上げるのを待っています。
