スタートアップマーケティング：200ドルをかけずに世界中の何千人ものユーザーを引き付ける方法





今日は、Product Huntにアクセスできるようにスタートアップを準備する方法、その前に実行する必要がある手順、および発行日と発行後のプロジェクトへの関心を高める方法について説明します。



はじめに



過去数年、私はアメリカに住んでいて、英語を話す（そしてそれだけではない）リソースでスタートアップを促進してきました。 今日は、最小限の投資でITスタートアップに国際的なユーザーを引き付けるという経験を共有します。 これに最も適しているのは、コンテンツマーケティングツールです。 それらの多くは無料またはほぼ無料ですが、良い結果をもたらすことができます。



そのため、国際的なユーザーを惹きつけたい場合、ここでスタートアップを宣伝する必要があります。



hnを表示



Hacker Newsリソースは、長い間ニッチで最も人気のあるテクノロジーおよびスタートアップの1つです。 プロジェクトが少なくとも数十分間メインページに到達した場合、これはすでにトラフィックの急増につながる可能性があります。サイトに数百回クリックするのは簡単です。







このリソースにはShow HNセクションがあります-ここでは、プロジェクトの作成者または一般ユーザーが便利なリンクを共有できます。 プロモーションはこのリソースから始める価値があります。無料で、成功した場合は良い結果を得ることができます。 ほとんどの場合、メインに到達するのに成功せず、HNのオーディエンスは必ずしも非常に友好的ではありませんが、信頼できるリソースからサイトへのリンクはいずれにしても不要ではありません。



ベータページとベータリスト



同様のモデルで動作する2つのリソース。 これらは、スタートアップスタートアップディレクトリです。 有料で、プロジェクトの説明は一般的なキュー（高速に移動しない）をバイパスしてリストに分類され、メインページで修正できます。これらのサービスには、メーリングリストにプロジェクトへのリンクも含まれます。



Betapageを最後に使用したときの価格は次のとおりです。



  • メインページの1日：28ドル
  • 2日：48ドル
  • 3日間：68ドル
  • 4日間：88ドル


ニュースレターに含めるとさらに30ドルかかるため、この数字を最終価格に追加する必要があります。 このサイトで有料リストを使用した私たちの経験を成功と呼ぶことはできません-数百人の登録ユーザーがメインプラスメーリングリストで私たちを1日もくれませんでした。



ベータリストの投稿はより高い-129ドルですが、無料で公開される可能性もあります。 後者の場合、配置を待つのに約1か月かかりますが、保存するタスクがある場合、これはかなりのオプションです。 最新のプロジェクトでは、このオプションを選択し、452人のユーザーのみを受け取りましたが、発表の日にスプラッシュしました。







一般に、主な費用はBetapageとBetalistが負担し、私たちの場合、それらは少なくとも2番目の場合にのみ部分的に正当化されました。



国際コンテンツマーケティング



英語圏のユーザーだけでなく、ユーザーを引き付けることを計画していたため、この方向で実験することにしました。 すでに英語の優れた記事がいくつかあり、ベータリストと製品ハントの説明もありました。



Upworkの助けを借りて、翻訳を手伝ってくれただけでなく、出版された資料へのリンクをどこでどのように共有するかについて助言してくれたヒスパニック系編集者を見つけました。 1つのブログ投稿を英語からスペイン語に翻訳する費用は通常10ドルを超えないことを言わなければなりません。



その結果、コンテンツを「まく」ための2つの主要なリソースを選択しました。



  • Taringa.netはラテンアメリカで人気のあるアグリゲーターサイトで、Redditに多少似ており、2100万人以上の訪問者がいます。
  • Meneame.net -950万人の訪問者。


さらに、私たち自身で、ヒスパニック系ラテンアメリカ諸国の居住者が通信するRedditのセクションをいくつか選択しました。



  • r / aprendeingles / -英語の勉強に専念するセクション。
  • / r / es / -一般的なテーマのヒスパニック系subreddit、14.3千ユーザー。
  • / r /アルゼンチン/-92,000人の加入者。
  • / r / vzla / -ベネズエラのセクション、34.5千人の加入者。
  • / r /コロンビア /-13.8千人の加入者。
  • / r / chile /-48,000人のユーザー。


その結果は予想をはるかに上回りました。Reddit、Meneameのいいねに関する登録やコメントを受け取っただけでなく、プロジェクトはラテンアメリカのジャーナリストやブロガーからも注目されました。 特に、人気のあるアルゼンチンのITリソースの新機能に関するレビューの後、数日で数千人の人々が私たちに来ました。







しかし、メインイベントはもちろん、Product Huntの配置でした。



製品ハント：起動ガイド



このネットワークには、Product Huntでの最適な実行方法に関する多くのガイドとストーリーがあります。そのため、あまり説明することはせず、主なポイントにのみ焦点を当てます。



まず、PHコマンド自体からスタートアップガイドを読む必要があります。 ドキュメントの主な3つのポイントは次のとおりです。



  • 製品を投稿するハンターを探すことはできません 。 これはあまり意味がありません-以前は、人気のあるユーザーのサブスクライバーには、公開した製品に関する電子メールアラートが送信されていましたが、現在ではありません。
  • PHのホームページではなく、製品の投稿に直接リンクできます。 都市伝説によると、PHは、ユーザーが直接URLをクリックしても、メインページで見つけられないプロジェクトに罰金を科します。 これは真実ではありません。
  • いいねを要求することはできません-それで構いません 。 したがって、打ち上げに関連するすべてのコミュニケーションには、「PHで公開しました。リンクがあります。コメントを投稿し、質問してください！」という明確なメッセージを含める必要があります。


これらは、多くの人がまだ知らない点です。 強調する必要がある重要なポイントがさらにいくつかあります。



  • ハンターは見つけたほうがいい 。 これはファッションの瞬間です。この人がソーシャルネットワークのPHであなたの投稿へのリンクを公開すると、プラスになります。 しかし、そのようなキャラクターが長い間求められていなければ、あなたはそれをすることができます。
  • PHでのランニングは24時間作業です。 新しい日のカウントダウンは、米国西海岸時間の00:00から始まります。 コメントにタイムリーに応答するには、24時間オンラインである必要があります。
  • 24時間後、あなたも働く必要があります 。 その日のトップ製品のリストに入ることはボーナスを与えます-あなたもニュースレターに記載されていますが、成功しなかった場合、PHをさらに使用する他の方法があります。 たとえば、サイトには質問をする機会があり、その答えは特定のソフトウェアの推奨事項である可能性があります。 このような選択に該当する場合、より多くのトラフィックが得られます。


その結果、上位5つのプロジェクトのリストに載ることはありませんでしたが、ユーザーは少なくとも1週間は投票しました。これにより、毎日約10〜15の製品が表示されるため、メインページに留まりました。 ローンチから数日が経過するほど、製品を見つけるためにスクロールする必要がありましたが、これはサイトへの移行を妨げませんでした。



このような長期にわたる宿泊施設の主な秘密は、あなたがそれに興味を持ち続けることです。 これは、リスティングへのリンクを記載した記事を作成し、定期的に発行すること、ニュースレターを作成することなどを意味します。







例：PHの製品ページに移動するように呼び出すプロジェクトブログ投稿と、同様のカウンターが付いた組み込みバッジ。



おわりに



この資料に記載されている活動には約200ドルかかり、それらの準備と実装には数週間かかりました。 その結果、サイトへの数千回のクリックと数百の登録を獲得しました。 これらのテクニックをスタートアップにコピーすることができます。これは最初の国際ユーザーを獲得するのに役立つと確信しています。
