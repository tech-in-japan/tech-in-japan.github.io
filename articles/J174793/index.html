Androidでユーザーデータを表示するためのウィジェット

ユニバーサルウィジェットを使用すると、JSON形式で記述された任意のシステムに関する情報をAndroidデスクトップに配置できます。 たとえば、サイトへの訪問に関する統計、オンラインストアでの注文に関する情報、サーバーでのCPU使用率など。







デスクトップに情報を入力する3つの簡単な手順：



1）スクリプトの作成-JSON形式のデータプロバイダー

2）Universal Widgetをインストールします

3）ウィジェットの設定で、スクリプトのアドレス（データプロバイダー、更新間隔）を指定します



JSONの例：



{

「タイトル」 ： 「ユニバーサルウィジェット」 、

「タイプ」 ： 「リスト」 、

"date" ： "03/18/13 5:31:41 PM" 、

「データ」 ： [

{ 「名前」 ： 「行1」 、 「値」 ： 「文字列」 、 「色」 ： 「赤」 } 、

{ "name" ： "Row two" 、 "value" ： "1200" 、 "color" ： "＃FFA500" } 、

{ "名前" ： "行3" 、 "値" ： "50％" 、 "色" ： "黄色" } 、

{ "名前" ： "行4" 、 "値" ： "$ 1,230.00" 、 "色" ： "緑" } 、

{ "名前" ： "行5" 、 "値" ： "19403" } 、

... ]

}




データプロバイダーの例をこちらでご覧ください ： http : //mybest.com.ua/uv/
