KolibriN9を終了します

開発の1年半後、KolibriOSアセンブリの新しいバージョン-KolibriN9！



KolibriN9は、Leencyが個人的にリリースした非公式のビルドです 。 その中で、彼はKolibriOS設計のビジョンと、メイン開発チームの意見とは異なるユーザーフレンドリーなOSを開発する必要があります。 また、このアセンブリと公式のナイトアセンブリの違いは、KolibriOS用に作成されたほぼすべてのプログラムとゲームのデフォルトの配信です。

KolibriOSからの主な変更点：


  1. カーネルとドライバーの変更


    • USB（OHCI / UHCI / EHCI）およびUSBホットプラグ（「ホット」接続）のサポート。 次のUSBデバイスがサポートされています：キーボード、マウス、フラッシュドライブ、ハブ。
    • 読み取りおよび書き込みモードでのext2 / 3/4 FS、および読み取り専用モードでのXFSのサポート。
    • 追加ディレクトリ/ kolibrios /の接続。
    • プロセススケジューラでの時間の分布の改善。
    • ハードドライブとATAPIデバイスのSATA IDEモードのネイティブサポートが追加されました。
    • システム全体のクリップボードのサポート。
    • システムにvidintelドライバーを含める。これには、Intelグラフィックカードが統合されたラップトップでのネイティブアクセス許可のサポートが含まれます。
    • Intel HD Audioオーディオドライバーの更新。
    • IntelおよびATI Radeonグラフィックカード用のドライバーの更新。
    • 一部の構成での読み込みの高速化。
  2. ソフトウェア更新


    • Netsurf Webブラウザー。

    • Fplay：高ビットレートの大きなビデオファイルのサポート、Intelビデオカードの垂直同期のサポート（対応するビデオドライバーと連動）、キーボードコントロールの追加：スペース-一時停止、Enter-フルスクリーンモード
    • Dictyはロシア語の翻訳を発声できるようになりました
    • フラットアセンブラー（FASM）：安定バージョン1.71.16にアップグレードします。
    • Tinypad：システム全体のクリップボードサポート。 複数の修正と最適化
    • TextEdit：システム全体のクリップボードのサポート。 エンコード変換cp 1251-> 866およびcp 866-> 1251を追加。 バグ修正とマイナーな改善。
    • シェル：スクリプトでの相対パスのサポート、コンソール行でのカーソルの移動、システム全体のクリップボードのサポートの実装を開始。
    • アイコン：アイコンのドラッグアンドドロップをサポートする新しいデスクトップ、より便利なアイコンの追加、iniを編集してPCMメニューに独自のアイテムを追加する機能。
    • 通知3：FASMで作成され、複数行のサポート、キーのサポート、アイコンのサポート、ウィンドウを閉じられないようにする機能（クリックするだけで閉じる）、最初の行をタイトル（より太字と白のテキスト）にする機能、通知が画面に合わない場合、順番に並んで、スペースが使用可能になるまで待機します。

    • PCIDEV：近年のPCIバス仕様の変更に応じて、デバイスのサブクラスが追加されました。
    • ボード：ファイルへの書き込み時のCPUの負荷を軽減します。 プログラムのコピーを1つだけ起動し、再起動しようとすると、すでに起動されているものがアクティブになります。
    • rdsave：インターフェイスを最小化し、プログラムでの作業を簡素化します
    • WebView：httplibへの移行。 更新されたデザイン。 進行状況バーを介してページをロードするプロセスを表示します。 リンクにカーソルを合わせると、ステータスバーにアドレスが表示されます。 スキンは「縫い付け」ではなく、個別のpngファイルに保存されます。 リンクのあるホームページ。 タグサポートの改善。 プログラムの最適化。 訪問の履歴とキャッシュされた画像。 母国語（ロシア語と英語）でページをリクエストします。
    • Eolite：再帰的な削除フォルダー。 コンテキストメニュー。 「開く」ダイアログ。 Eoliteウィンドウ間でファイル/フォルダーをコピー/カットします。 ロシア語翻訳; マイナーな修正と改善。

    • fNav：コピープロセスのインジケータ、拡張列、リストアイテムの複数選択、メモリリークの修正を追加
    • グラフィックベンチマーク：ブリターと画面の読み取り速度のテストを追加
    • Palitra：背景を「タイル」で設定、より便利なコントロール、画面上のピクセルの色を決定（スポイト）
    • GUIの改善：RDSave、カレンダー、シャットダウンウィンドウ、タイマー、完全に再設計されたウィンドウの色調整プログラムインターフェイス、新しいアイコン




KolibriN9に固有の変更：


  • システムのデザインが再設計され、デザインがより美しく、より快適になりました。

  • Dockyパネルの使用により、使いやすさが向上しました。
  • 追加ディレクトリ/ kolibrios /の接続のサポートによるKolibriNの即時起動。
  • 外観-システムのスキンとデスクトップの壁紙をすばやく便利に変更するためのプログラム。
  • Game Center-KolibriN9専用に作成されたゲームセンターで、アセンブリの一部として提供されるゲームを実行できます。





KolibriN9はKolibriOSのサードパーティアセンブリにすぎないため、公式配布ではなく、アセンブリの作成者のみが責任を負うことを思い出させてください。
