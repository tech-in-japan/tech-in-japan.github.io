Boxee Media Center Betaリリース

最近、Boxeeクロスプラットフォームメディアプレーヤー/センターのベータテストが開始されました。 このバージョンは、新しいインターフェイスを備えた2008年6月にリリースされたアルファバージョンとは異なり、開発者はTV.com、blip.tv、IGNとの統合契約を実装しています。 お気に入りのメディアにブックマークを設定する機能もありました。 Boxeeは、YouTube、Flickr、Last.fm、Picasa Webなどのオンラインメディアサービスもサポートしています。 ソーシャルネットワークとの統合：Twitter、Facebook。 ところで、Boxeeはトレントクライアントを統合します。

BoxeeはC ++で、プラグインはPythonで開発されています。 ソースコードはGNU GPLの下でライセンスされています。







サポートされている形式のリストは非常に大きくなっています。

ビデオ： MPEG-1、MPEG-2、MPEG-4（SPおよびASP）、MPEG-4 AVC（H.264）、Huffyuv、Indeo、MJPEG、RealVideo、QuickTime、Sorenson、WMV、Cinepak。

オーディオ： AIFF、WAV / WAVE、MP2、MP3、AAC、AACplus、AC3、DTS、ALAC、AMR、FLAC、Monkey's Audio（APE）、RealAudio、SHN、WavPack、MPC / Musepack / Mpeg +、Speex、Vorbis、WMA

グラフィック： BMP、JPEG、GIF、PNG、TIFF、MNG、ICO、PCXおよびTarga / TGA。

字幕： AQTitle、ASS / SSA、CC、JACOsub、MicroDVD、MPsub、OGM、PJS、RT、SMI、SRT、SUB、VOBsub、VPlayer。

コンテナ： AVI、MPG / MPEG、WMV、ASF、FLV、MKV、MOV、MP4、M4A、AAC、NUT、Ogg、OGM、RealMedia（RAM / RM / RV / RA / RMVB）、3GP、VIVO、PVA、NUV 、NSV、NSA、FLI、FLC、DVR-MS。
