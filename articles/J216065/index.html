2014 Runet Conferences

来年、Mail.Ruグループは、情報技術に関連する何らかの形で多くの会議をサポートし、参加し続けます。 これらは主に開発者向けのテーマ別会議です。 何らかの形で、2013年には既にそれらのほとんどすべてをサポートしており、一般の人々に詳細に説明する価値があると考えています。 Runet-2014カンファレンスのレビューの中で、皆さんの中には新しくて便利なものが見つかるかもしれません。







OGIC







コンピュータゲームおよび電子エンターテイメントの分野におけるノベルティの展示。 幅広い訪問者向けに設計されています。コンピューターゲームと3Dテクノロジーの開発の専門家、およびすべての共感者の両方を対象としています。



日付と場所： 3月15〜16日、サンクトペテルブルク。



ウェブサイト： http : //ogicon.ru



Mobilefest







モバイルアプリケーションの開発者向けのカンファレンスフェスティバル。 このプログラムは、すべてのゲストに専門家のレポート、ディスカッションミーティング、エンターテイメントイベントを提供します。



日付と場所： 3月21〜22日、ヤフロマ（モスクワ地方）。



ウェブサイト： http : //mobilefest.ru



アジャイルデイ







開発者向けの専門会議。 2日以内に、5つの主要トピック内で約70のレポートが表示されます。

  • 柔軟な開発（アジャイル/リーン）
  • 製品開発
  • 開発者のケーススタディ
  • エンジニアリングプラクティス
  • ゲーミフィケーション




日付と場所： 3月21-22日、モスクワ。



ウェブサイト： http : //agiledays.ru



eTarget







この会議は、インターネットマーケティング、広告、SEOに特化しています。 プログラムの一環として、招待された専門家がレポートを読み、ラウンドテーブルを開催します。 また、誰もがインターネット上の広告とプロモーションに特化したさまざまな企業が参加する展示会を訪れることができます。あなたは彼らのサービスに慣れることができます。



日付と場所： 3月27〜28日、モスクワ。



ウェブサイト： http : //www.etarget.ru



ロシアのインターネット技術2014







Web開発者向けのロシア最大の会議。 今年は7回開催され、その存在史上最大の規模になることが約束されています。 昨年、会議の主なトピックは次のとおりでした。

  • プロジェクト管理
  • サーバープログラミング
  • クライアントプログラミング
  • データベース
  • 大規模サイトの開発
  • Web開発事業
  • 品質管理
  • 未来の技術




日付と場所：モスクワ、4月14〜15日。 4月24〜25日、サンクトペテルブルク。



ウェブサイト： http : //ritconf.ru



ロシアのインターネットフォーラム（RIF + CIB 2014）







国内最大かつ最も人気のあるこのIT会議は14回開催されます。 一般に、インターネットの技術的および立法的な発展、電子商取引、インターネット上の広告、検索とクラウドを含むネットワーク技術の開発、インターネットマーケティング、ITの人材派遣に専念しています。 通常、3分の2の会議プログラムは、訪問者と参加者が事前に提案する会議プログラムで構成されます。 したがって、各RIF + CIBは、今年の特徴であるITコミュニティの傾向と関心を反映しています。 平均して、約10,000人が展示会に参加します。



日付と場所： 4月23〜25日、モスクワ地方の寄宿舎「Polyany」。



ウェブサイト： http : //2014.russianinternetforum.ru
