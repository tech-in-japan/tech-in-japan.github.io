Ubuntu 11.04および12.04にZTE AX226をインストールする

この記事は、Ubuntu 11.04および12.04にZTE AX226モデムドライバーをインストールし、WiMaxテクノロジを使用したカザフスタンのdijiプロバイダーの例を使用してインターネットに接続することに専念しています。 箱から出して、このモデムはLinuxシステムを提供しません。SamsungU200モデムを使用してYotaプロバイダーに接続するときに使用される人気のmadwimaxパッケージは、ここでは役に立ちません。 ただし、ZTE AX226モデムをUbuntuにインストールするためのソリューションはありますが、残念ながらフォーラムに散らばっています。



インターネットに接続せずにインストールできます！

まず、証明書ファイルが必要です。CA証明書、ユーザー名（MACアドレス）、デバイス証明書（PEM形式）、デバイスの秘密キー（PEM形式）。

通常、ドライバーがインストールされているC：\ Program Files \ DTV WiMAX \ binフォルダーからCA.pem証明書CAファイルを取得する必要があります 。

残りの証明書ファイルとキーファイルを取得するには、 Process Explorerが必要です
  • 1. Wndowsでインターネットに接続する
  • 2.プロセスエクスプローラー、接続マネージャーのメモリ（cm.dmp）のダンプを作成します（cm.exeリストで右クリックし、[ダンプの作成] >> [完全なダンプの作成]）
  • 3.次に、スクリプトをダウンロードして、 ZTE AX226モデム証明書を取得します。
  • 4.便利なフォルダーにスクリプトを展開し、cm.dmpダンプファイルを同じフォルダーにコピーします。
  • 5.コンソールでgetcert.shスクリプトファイルを実行します。
    /full/path/to/folders/getcert.sh
スクリプトとダンプファイルがあるフォルダーに、 devcer.pemとprivatkey.pemの 2つのファイルが表示され、使用できる状態になります。

接続するベースステーションの検索範囲を決定するため、小さな検索範囲を設定すると、ベースステーションが検出されない可能性があります。 ただし、広い検索範囲を指定すると、モデムは非常に長い時間ベースステーションを検索します。 したがって、最初の開始時に大きな周波数範囲を指定し、ステーションが決定された後、より小さい範囲を指定することをお勧めします。 （最寄りの基地局でどの周波数を決定するかは以下で説明されています）



次に、スーパー管理者モードでNautilusを使用して、 wimaxd.conf ファイルとca.pem 、 devcer.pemおよびprivatkey.pem証明書ファイルを /usr/share/dtv.kzフォルダーに配置します。

須藤オウムガイ


次に、ZTE AX226ドライバーのインストールに直接進みます。
  • 1. Sprint4G-Lite-Pack （26mb）をダウンロードします
  • 2.アーカイブをホームディレクトリに解凍します（！）
  • 3. cm.dmpダンプファイル（または既に作成している場合は証明書）とca.pemを〜/ Sprint4G-lite-Pack / getcert /フォルダーに解凍します。
  • 4.モデムをUSBポートに差し込みます
  • 5.コンソールで実行します：
    sudo -i -H
    cd / home / user_name / Sprint4G-lite-Pack
    ./install_lite.sh
    そして、インストーラーの質問に答えます。
  • モデムでMACアドレスを読み取ることができます。 MACのような文字列を探しています：001EB34A7596。 インストール時には、「MAC：」という単語なしで001EB34A7596のみを入力します
  • 次のコマンドを使用して、カーネルバージョンを確認できます。
    うなめ-r
    たとえば、私はそれぞれカーネルバージョン3.2.0-26-generic-paeを持っています、私は答えます：
    /usr/src/linux-headers-linux-headers-3.2.0-26-generic-pae
インストールが完了したら、次のコマンドを実行して、システムでモデムが検出されたかどうかを確認します
lsusb
モデムは次のようになります
バス001デバイス007：ID 19d2：0172 ZTE W​​CDMA Technologies MSM AX226 WIMAXモデム（モード切り替え後）

ID 19d2：0172の代わりにID 19d2：bccdが表示される場合、モデムの切断と再接続を試みます。
注：

カーネルを更新する場合、ドライバーを再インストールする必要があります！



モデムがインストールされ検出されたら、ZTE AX226モデム用のWimaxアプレットをインストールできます。
  • 1. wimax-applet_1.4.8_all.zipアーカイブをダウンロードします
  • 2.コマンドを実行して.debパッケージをインストールします
    sudo dpkg -i /path/do/package/wimax-applet_1.4.8_all.deb
  • 3.アプレットをダッシュ​​で見つけるだけで起動します
  • 4.コントロールを手動モードに変換し、アプレットがベースステーションを検索します
  • 5.検索が完了したら、CINRが最も高いBSを選択して接続します
BSを選択すると、それらの周波数も表示されます。これは、 wimaxd.confで指定するBSのおおよその周波数範囲を決定するのに役立ちます

インストールが完了しました。 モデムの準備が整いました。
