予測に反して、第2四半期に、Amazonはクラウド部門の発展により利益成長を記録しました。

今年の第2四半期の終わりに、 Amazonの最大のオンライン小売業者はすべての期待を上回る利益を上げました。 2015年4〜6月の純利益は9,200万ドル、つまり1株当たり0.19ドルに達しました。 同社の収益は20％増加し、232億ドルに達しました。 昨年の同時期に、1億2,600万ドルの純損失が記録されました。 同社の営業利益は4億6400万ドルに達しましたが、2014年の損失は1500万ドルでした。



アナリストによると、Amazonは224億ドルの収益に対して1株あたり0.14ドルの純損失を被ると考えられていました。 昨年の第4四半期まで、収益は20％増加しました。 2015年第1四半期の終わりまでに、成長率は15％に低下しました。 さらに、会社が受け取った大量の投資により損失が発生しました。



Amazon Web Servicesのクラウドオペレーション収益は、前年比で407％増加して3億9,100万ドルでした。 4〜6月のユニットの収益は81％増加して18億2,000万ドルでした。 Amazon Web Servicesは、この四半期に350の新しい機能とサービスを導入しました。 AWS Educateクラウドサービスユーザー向けのトレーニングサイトがオープンしました。



この四半期に、Amazonはいくつかの新しいインターネットサービスを開始しました。 その中には、 Amazon Business e-commerceポータルと、Amazonのメキシコ向けメインサイトの特別バージョンがあります。 さらに、同社は音声制御デバイスAmazon Echoをリリースしました。 天気とニュースを送信し、 Amazon Musicから音楽を再生し、質問に答え、to-doリストと買い物リストを維持します。



同社の予測によると、第3四半期の収益は、前年の206億ドルと比較して、233〜355億ドルになります。 トムソン・ロイターが調査したエコノミストは、238.9億ドルという数字を示唆しています。
