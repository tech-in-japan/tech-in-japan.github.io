ジェネレーティブデザイン：デザインの新時代の始まり

シーメンスPLMソフトウェアの同僚からの新しい設計技術に関する情報記事をご紹介します。







ジェネレーティブデザインは、根本的に新しいデザインテクノロジーです。 これは、デザイナーの参加なしに、指定された条件を満たす3次元モデルを独立して生成できるソフトウェアの使用に基づいています。 実際、「マンマシン」システムでは、創造的な機能がコンピューターに転送され、完全に機能します。

このテクノロジーは、コンピューター支援設計の主要ツールとしてすでに使用され始めています。 その理由は、コンピューティング機能の成長と3Dプリンティングの非常に迅速な開発です。これは、新しいツールで開発されたパーツやオブジェクトを完全に生成できる技術です。



今日、3Dプリンティングのコストと3Dプリンターの価格はすでに、それらに基づいた工業生産について話せるレベルまで低下しています。 タンデムの「ジェネレーティブデザイン-アディティブテクノロジー」の機能については、多くのアナリストがそのアプリケーションを単なる産業革命にすぎないと言っているほどです。



ジェネレーティブデザインの助けを借りてモデル化され、付加的な技術によって実装されたオブジェクトを少なくともいくつか見てきましたが（図1-3）、これは信じがたいことです。

レビュー出版物の著者は、ジェネレーティブデザインの4つの分野について語っています。

  • フォームの合成;
  • 3次元格子の表面と構造の最適化;
  • トポロジーの最適化（指定されたパラメーターに従って、不要なものはすべて削除されます）;
  • 骨梁構造（ジェネレーティブデザインは、すべての固体材料に小さな細孔を正確に拡大縮小し、表面の粗さを作り出します）。


今日、わずかな経験でさえ、これらの分野の発展が建築、エンジニアリング、家具、スポーツシューズなどに革命的な変化をもたらすことを証明しています。 設計における実用的なアプリケーションのために、主要なCADソフトウェアメーカーは、個別のアプリケーションとしてソフトウェアパッケージを開発しました。



ジェネレーティブデザイン専用のソフトウェアは非常に強力で、さまざまな業界で使用できます。 これには多くの利点がありますが、CAD機能に組み込まれたアプリケーションも同様に有望です。 このようなソリューションにより、設計者は質の高い新しいレベルで迅速に結果を得ることができますが、同時にモデリングシステムに残ります。 シーメンスPLMソフトウェアは、このような機能の独自バージョンを提案しました。



Solid Edge ST10の生成モデリング

機械工学では、トポロジーの最適化が特に重要です-材料とエネルギーの節約と生産性の向上という形で即座に効果をもたらします。 このタイプの生成設計は、Solid Edge ST10に実装されています。



トポロジー最適化とは、所定の締結および荷重条件下で部品の最適な形状を検索することです。 この場合の最適な形式は、最小の重みを提供する形式であることを強調します。



Solid Edge ST10では、生成的設計を開始すると、設計者は部品が形成される領域を指定します。 このために、その近似形式が指定されています。 ファスナーの位置を修正します。 変更すべきでない領域と負荷が示されています。 次に、開いたダイアログボックスを使用して、設計者は重量減少の割合、強度の安全係数を設定します。





図 4. Solid Edge ST10のトポロジー最適化。 ジェネレーティブモデリングを使用して、強度と固定スキームの初期要件を満たす最小質量の部品が作成されます



同じウィンドウで、生成モデリングの別の重要なパラメーター（プログラムの実行時間）が設定されます。 結果のモデルの精度は、反復回数を制限するのは彼であるため、それに依存します。



トポロジー最適化の結果として得られた解は、ファセット化された外観を持っています。 すぐに3D印刷に送信できますが、Solid Edge ST10の機能はこれに限定されません。 10番目のバージョンでは、Convergent Modelingが初めて使用されます。BREPとファセット表現が1つのモデルに結合され、生成デザインの製品を編集できるようになります。



最適化プロセスが完了すると、部品の作業を継続できます。 グリッドの正確な表現への「問題のある」変換に関連するアクションは必要ありません。 設計者はいくつかの要素を追加および削除でき、ブール演算を実行できます。 アセンブリの作業が進行中の場合、ある部分の本体を別の部分から差し引くことなどが可能です。 グリッドビューは常に更新されています。



したがって、Solid Edge ST10は、生成モデリングの定性的に新しいテクノロジーのアプリケーションを提供するだけでなく、結果を変更する便利な機能も提供します。



トポロジのコンピューター最適化により、驚くほど完全に珍しいジオメトリオプションが提供されます。 強度特性を損なうことなく製品の重量を削減し、材料の消費を最適化して、製品のコストを削減できます。 しかし、これらすべてにより、製品の製造可能性の問題がすぐに生じます。



実際、最適化されたモデル（図4）をざっと見るだけでも、切断と圧力による金属加工の従来の方法の適用不能性を確信するのに十分です。 信頼できる唯一のものは、環境の観点から非常にエネルギー集約的で問題の多い技術である鋳造です。 そのような最適化には何かポイントがありますか？また、Solid Edge ST10の革命は何ですか？



この質問に対する答えは明確で前向きです。 トポロジ最適化は明確な前進であり、Solid Edge ST10は画期的なコンピューター支援設計システムです。 最後の疑念を捨てるには、単純な論理チェーンをたどるだけで十分です。

  1. 次の10年以内に、その機能が向上し、3D印刷が広く利用可能になるでしょう。
  2. ジェネレーティブデザインは最適なソリューションを提供しますが、正確な3D印刷が必要です。
  3. ジェネレーティブデザインと3Dプリンティングを使用すると、エンジニアリング、建設、その他の業界の製品が質的に変化しますが、通常のデザインプロセスへの統合を保証するにはCADシステムが必要です。


シーメンスPLMソフトウェアは、このような機械工学用のCADシステムであるSolid Edge ST10を作成しました。 これは、ジェネレーティブデザインとアディティブテクノロジーを最大限に活用できるようにする最初のCADシステムです。



現在、デジタルモデリングと3Dプリンティングのための最高の技術を備えていない生産は、競争で負ける運命にあるという十分に合理的な予測を行うことが可能です。 このような状況では、トポロジ最適化の適用は利点ではなく、企業の存続の条件になります。



これは、トポロジーの最適化を支持する別の引数で補うことができます。 実際には、与えられた条件に最適なオプションを提供します。そのようなオプションは、開発の機会がなくても、開発者にとって常に興味深く有用です。 理想を達成することは不可能ですが、従来のテクノロジーのモデルを作成することで、それに近づくことができます。 これを行うために、Solid Edge ST10には世界で最も効率的な設計兵器の1つがあります。
