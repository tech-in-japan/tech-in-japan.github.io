リチャード・ストールマン編集委員会へのインタビューpirateparty.ru

pirateparty.ruの編集者は、世界的に有名なプログラマーであり、 著名な人物であるリチャードストールマン （フリーソフトウェア運動、GNUプロジェクト、Free Software Foundation、League for Programming Freedomの創設者）に目を向けました。この社会問題で。 以下のインタビューについて、Free Software Movementの創設者に深く感謝します。



（インタビューは実際には小さいですが、テキストがたくさんあるように思われるかもしれませんので、斜体で主な考えを強調しました）



質問：著作権とソフトウェアに関するPirate Partyプログラムについてどう思いますか？



RMS：フリーソフトウェアよりもプロプライエタリなソフトウェアの利点をもたらす1つの重大な間違いがない限り、スウェーデン海賊党を支援します。



このパーティーのプログラムによると、ソフトウェアの著作権は5年間保持され、プログラムのソースコードはパブリックドメインになります。 したがって、法的にコピーレフトで配布されているフリーソフトウェアは、 独自のプログラムに統合できます 。



逆もまた真である場合-ソースコードが

独自のプログラムは5日後に合法的に利用可能になります

無料プログラムにそれを含めるための年 。 ただし、この場合、プロプライエタリプログラムのソースコードはまったくリリースされないため、これは発生しません。



明らかに、 この間違いをどのように修正できますか：政府機関はプロプライエタリソフトウェアのソースコードを必要とします。 5年後、この政府機関は公開し、ソースコードをパブリックドメインに送信します。 私はこれらの調整を行いましたが、スウェーデンの海賊党はそれらを認識しませんでした。 しかし、ロシア海賊団を結成すれば、この瞬間を変えることができます。



質問：著作権の役割と性質をどのように評価しますか？

現代の世界では、通常「 知的財産 」と呼ばれていますか？



RMS：あなたが尋ねた質問に答えるには、「知的財産」の概念と、それを使用すべきでない理由を明確にする必要があります。 この用語はプロパガンダであり、使用されるたびにバイアスと混乱を広げます。 一貫性のあるわかりやすい主題とは関係ありません。なぜなら、それは互いに共通点がなく、全体として解釈すべきではないさまざまな法律をまとめているからです。 「知的財産」に関する記述は、混乱したり、意図的に他者を混乱させたりします 。



前世紀の70年代には、著作権についてだけでなく、

特許法（実際にはまったく異なる分野で）、および商標法（これは何の関係もない）

著作権法および特許法）。 これらの領域を区別する

正しいアプローチです。 これらの法律はさまざまな分野に適用されるため、公開討論のためのさまざまな主題が生成されます。



その後、 世界知的所有権機関

これらのすべての法律 （およびその他の法律 ）をまとめるキャンペーンを開始しました 。

それらが1つの質問であるかのように。 この基本的な混乱のプロンプト

法律自体とその機能を無視し、それらすべてについて考えようとする人々

全体はどうですか。 これは、特定の法律に関連する実際の問題に気付くことさえできない、純粋に抽象的な思考を意味します。 この抽象化により、人々は「知的財産のために」など、最も広く行き渡った過激な立場の1つに導かれます。

または「知的財産に対する」。 どちらも使い道がありません。



次に、偏った解釈があります。 名目上、これらのすべての法律は、公共の利益をもたらすことを意図しています-もちろん、さまざまな方法で。 たとえば、著作権と特許は、特定の活動に対する人為的なインセンティブを意味します。これは有用であることを望み、商標は顧客が購入しているものを知らせます。 「知的財産」という語句の「プロパティ」という言葉は、 人々に、さまざまな排他的特権がそれらを受け取る人の名前にのみ存在すると誤って信じさせます。 この誤った考え方のフレームは、人々を立法と執行の誤った評価に導きます。



著作権法および関連する問題の思慮深い考察、ならびに特許の思慮深い考察を促すため

法律、私は「知的財産」という用語を拒否しました。 私はこの用語を使って発言することは決してありません。この用語がなぜ（今のように）有害なのかを説明するだけです。 著作権についての意見、特許法についての意見、商標法についての意見がありますが、それらを混同せず、「知的財産」と呼びます。 同じことをお勧めします。



質問の最初の部分は著作権のみに関するものなので、この部分に答えることができます。 今日の著作権制度の主な目標は、特定の企業が社会を不当に支配することで、彼らが自らを豊かにすることです。 今日、著作権は文学や芸術の宣伝に副作用をもたらします。つまり、著作権はそれが設定された目的ですが、高いコストでそれを行います。 目標は依然として望まれていますが、別のシステムでそれを達成する必要があります。
