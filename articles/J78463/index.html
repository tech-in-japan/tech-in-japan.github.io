あなたのブラウザはあなたの選択です

マイクロソフトに対する欧州委員会の手続きの結果、ユーザーが勝ちます



2009年12月16日-オスロ、ノルウェー



Opera Softwareは本日、デフォルトでWindowsオペレーティングシステムにIEブラウザをインストールすることの合法性に関する欧州委員会のレビューの結果として、世界中のインターネットユーザーの勝利を発表しました。 Microsoftは、Windows XP、Windows Vista、およびWindows 7のユーザーにWebブラウザ選択画面を提供するようになりました。 2007年12月、Opera Softwareは、欧州委員会に、Windowsを使用してInternet Explorerを宣伝するためのオペレーティングシステム市場におけるMicrosoftの支配的地位に対するマイクロソフトの悪用を調査するよう依頼しました。



新しい画面により、ユーザーは代替のInternet Explorerブラウザーを簡単に選択してインストールできるようになります。これは、より安全で、高速で、オープンWeb標準と一貫性があります。 ブラウザの選択は、インターネットアクセス機能を同等にするために、単一のユニバーサルWebスペースへの道の別のステップです。



「この決定は、インターネットのさらなる発展とWebの将来の革新に必要なオープンWeb標準の勝利と言えます」とOpera SoftwareのヘッドであるJon von Tetzchner氏は述べています。 -Operaは、いつでも、どこからでも、どのデバイスを使用しても、インターネットへの平等なアクセスを提供するWeb標準の謝罪者です。 手続きの結果とEUの決定により、ミッションの正しさが確認されると信じています。」



2007年のEUでのOperaの要求でIEの進捗状況を積極的に調査した後、欧州委員会はブラウザの選択画面をインストールするというMicrosoftの提案を受け入れ、ケースは終了しました。 選択画面の契約は5年間有効です。 6か月に1回、このソリューションの有効性がさらに調査されます。 現在、選択画面はヨーロッパのユーザーのみが使用できます。



Opera SoftwareのCTOであるHåkonWium Lie氏は、次のように述べています。 「選択画面により、ユーザーは、オープンWeb標準を最大限にサポートした最高のブラウザーにアクセスできます。」



欧州委員会のマイクロソフトケーススタディ年表



2007年12月15日-OperaはEUに独占禁止法違反の申し立てを行い、Microsoftは、オープンWeb標準と互換性のあるブラウザを使用するオプションを消費者に提供する必要があります。 そこで調査が始まり、最終的には2年間続きました。



2009年1月17日-欧州委員会は、MicrosoftがIEをWindowsオペレーティングシステムのデフォルトブラウザーとして違法にインストールしたことで有罪と判断しました。



2009年2月9日-欧州委員会は、Mozilla FoundationのMicrosoftに対する反トラスト訴訟への参加意欲の宣言を承認しました。



2009年2月25日-Googleは、Microsoftに対する反トラスト訴訟に参加する意向を発表しました。



2009年4月28日-マイクロソフトは独禁法訴訟について正式に欧州委員会に連絡しました。 同社は、Windows 7をより標準に準拠させ、Internet Explorerを無効にする機能を提供することを約束しました。



2009年6月12日-マイクロソフトの提案は欧州委員会を満足させませんでした。 Operaは、Microsoftのイニシアチブは「小さすぎて遅すぎる」と述べた。 Operaは、ハードウェアメーカーの選択により、ブラウザを選択するか、WindowsでIE以外のブラウザを提供する可能性を提案することにより、欧州委員会をサポートしました。



2009年7月24日-マイクロソフトはブラウザーの提供に同意しました。



2009年10月7日-マイクロソフトは、ヨーロッパで新しいブラウザー選択画面のテストを開始します。



今日-欧州委員会は調査を終了し、ユーザーはWindowsに最適な標準準拠のブラウザーを選択する機会が与えられます。



Opera Software ASAについて

Opera Software ASAは、PC、携帯電話、その他のインターネットデバイスでWebページを表示するための新しい可能性を切り開きました。 Operaが開発したクロスプラットフォームブラウザーテクノロジーは、その機能、オープンなWeb標準への準拠、および小さなサイズで知られています。 同時に、高速で安全かつ動的なインターネットアクセスをユーザーに提供します。 Opera Softwareのメインオフィスはノルウェーのオスロにあり、世界中にオフィスがあります。 同社は、OPERA証券コードでオスロ証券取引所に上場しています。 詳細については、 www.opera.comをご覧ください。
