紙の上の個人的な財政

現代人は誰でも、自分の財政を記録しておくといいと思っていました。目標は明白です。お金をどこに行くかを自信を持って言い、大きな買い物をする前に自分の強さを評価できます。 この考えが頭に収まるとすぐに、コンピューターでホームアカウンティングを実行するという決定がすぐに現れます。 人は検索エンジンに入り、たくさんのリンクを見つけ、プログラムやオンラインサービスのレビューを読みます。 彼はシステムの機能を比較し、最終的に決定を下します。 やっと、選択ができました。すべての購入品を毎日記入し始める時です！





費用を修正する最初の週は、前向きなことの波で起こります-新しい便利なインターフェース、美しいレポートの表示、あなたが正しいことをしているという気づき。 しかし、数日が経過すると、これらの鮮やかな印象はすべて悲しい現実に置き換わります。毎日、この職業に10分の貴重な時間を費やす必要があります。 そして、最悪の事態は、ある日、彼が自分の費用を支払うのを忘れて、翌日（または数日）彼がそこで過ごしたものを集中的に覚えているときです。 または、「明日はもっと良くする-一度に2日以内に」という考えを思いつき、明日来るが、インスピレーションはない。 そして一週間が経ち、彼は彼の机の上に購入からの領収書の束を持って、彼はそれらを見て、彼がこれらすべてを彼の「家庭会計」に持ち込む強さも欲望もないことを理解します。



私の経験はまさにこのシナリオで発展しました。



私が経費を書き留めるのが「怠けすぎた」主な理由は、これに費やした時間が多すぎるためでした。 そして、私は考えましたが、どうすればコスト修正を簡素化できますか？ 結局のところ、私にとっての主なことは、お金が何に使われているのかを理解することであると同時に、費用を記録するのに多くの時間を費やしたくありません。



紙のソリューションはシンプルなソリューションです

コンピュータがなかったとき、紙のノートにボールペンで自分の費用を書き留めたいと思っていた人全員。 そして、まったく同じことをすることにしました！



最初に行うことは、すべての費用をカテゴリに分類することです。 各カテゴリは、ノートブックの個別のページに保管する必要があります。 次のカテゴリがあります。

1.製品

2.サービス

3.モバイル通信、インターネット

4.輸送

5.エンターテインメント、ギフト

6.薬

7.家、家具

8.家庭用品

9.衣類

10.テクニック

11.スポーツ



今月の各カテゴリの毎日は、費やした金額を入力する必要があります。 結果は次のようなものです。



製品

2009年3月

57 + 95 + 544 + 44 + 300 + 313 + 20 + 30 + 307 + 1638 + 260 + 90 + ...



月末には、すべての数値を合計し、どのくらい、そして最も重要なことに、どこで使われたかを調べることが残っています。 それをすべてまとめて視覚的に美しく提示するために、コンピューターが私の助けになりました、今はコンピューターなしで:-)



MS Excelでは、カテゴリごとにいくつかのシートと、個別の「合計」シートを作成しました。 各シートには、購入コストを入力する月の名前が水平に配置されています。



次のようになります。







最終シートでは、前のシートへのリンクを設定しました。その結果、「合計」シートは完全に自動的に形成されます。 これらのデータに基づいて、毎月の費用を確認し、チャートを作成し、予算を計画できます。



主な違い：

1.費用は全体として1か月間固定され、日単位ではありません。

2.詳細なカテゴリはありません。

3.予備記録はノートブックで実行されます。



奇妙に思えるかもしれませんが、これは、ノートブックの経費の最も中間的な固定であり、最低限の経費カテゴリのセットにより、個人的な簿記を定期的かつ非常に迅速に行うことができます。 今ではかなりの時間がかかりますが、最も重要なことは、これを行うことが私にとって興味深いことです。 今ではそれは日常的で退屈に見えませんが、今ではとても簡単です:)
