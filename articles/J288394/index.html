外国特許ルートの最適化

発明の外国特許取得のためのいくつかのルートの開発により、1つまたは別の国のグループで特許取得を決定する際の経済性と柔軟性のどちらかを選択できます。



ソースデータ


記載されている外国特許のルートは、国際市場への参入を計画しているB2Bセクターのハイテク製品を持つロシア企業に適用されます。 特許国は、ターゲット市場を持つ国です。 この例では、37の外国が考慮されていますが、これはB2Bセクターで非常に一般的です（特許の地理は通常30〜40か国をカバーしています）。 それらは、マーケティング部門が決定した優先順位に従って4つのグループに分けられます。 特許取得のコストは、標準特許出願について決定されます-30ページ、15のクレーム、そのうち2つは独立しています。 最初の10年間の年ごとの特許取得費用の予測には、代表者（特許代理人または弁護士）のサービス料（米ドル）を含む、1つの特許出願の準備、1つのパテントファミリーの取得と維持の費用が含まれます。



特許取得ルート


ルート1は、Rospatentに国内（優先）申請を提出し、実体審査の即時開始を申請し、その後、Rospatentを通じて国際申請を提出し、その後国内または地域段階に移行します。 さらに、37か国から17か国の欧州特許機構を通じて国際出願が地域段階に移行され、4か国ではユーラシア特許機構を通じて地域段階に移行され、残りの16か国では国際出願が国内段階に移行されています。 ルート1での特許の総費用は274,800 ドルです。

ルートNo.2は、Rospatentに国内（優先）申請を提出し、実体審査の即時開始を申請し、その後（より正確には約1年以内に）カテゴリ1の国またはカテゴリ1および2の国で国内および地域の申請を同時に行うことを含みますまたは、カテゴリ1、2および3の国、またはカテゴリ1、2、3および4の国。さらに、37か国のうち、カテゴリ1は5か国、カテゴリ2は6か国、カテゴリ3は7か国、3ユーラシア特許機構の国、カテゴリ4には5 ndividualnyh国と欧州特許機構の11カ国。 カテゴリ1の国のルート2での特許の総費用は、カテゴリ1 + 2 + 3 + 4-のカテゴリ1 + 2 + 3-国の場合、カテゴリ1 + 2 + 3- $ 139,200の国の場合、65,300 ドルです。 350 600ドル。

ルート3は、Rospatentに国内（優先）申請を提出し、実体審査の即時開始を申請し、その後Rospatentを通じて国際申請を提出し、その後2年以内に国内および地域段階に移行します。 国の分布はルート2に似ていますが、カテゴリー3および4の国の国内および地域段階への参入は、国際特許の猶予期間の終わり（つまり、約1年後）に移行します。 カテゴリ1の国のルート3での特許の総費用は、カテゴリ1 + 2 + 3-のカテゴリ1 + 2 + 3 + 4-の国の場合、カテゴリ1 + 2 + 3- $ 142,200の国の場合、68,300 ドルです。 342,400ドル。



特許費用の比較


特許費用と特許費用の年ごとの分布の違いを図1と2に示します。







ルート1が比較の基準として選択され、図1にはルート1と2の比較が含まれ、図2にはルート1と3が含まれます。ルート1を使用すると、時間内に費用を最も均等に分配でき（最大で3年目と4年目に発生）、合計金額が最小になります、しかし、必要な結果を確実にするため、そして既に発生した費用が無駄にならないように、資金調達スケジュールを厳守する必要があります。 ルート2を使用すると、国の各カテゴリで特許権を決定できますが、すべての国の合計金額が高くなり、2年目と3年目に支出が急増します。 ルート3を使用すると、国の各カテゴリで特許権を決定することもできますが、ルート2のコストに近いコストで、時間の経過とともにわずかにスムーズなコスト配分を得ることができますが、この場合、3年目と4年目に顕著な増加があります。



したがって、外国特許取得のためのいくつかのルートを開発することで、特定の国での節約と意思決定の柔軟性のどちらかを選択できるだけでなく、特許プログラムへの資金調達の優先スケジュールを考慮することができます。
