Super Game Boyにケーブルコネクターを取り付ける







SNESでゲームボーイゲームをプレイできるスーパーゲームボーイカートリッジは、ハードウェアが通常のゲームボーイと非常に似ています。 プロセッサ、アドレス空間の割り当てに一致します。 ただし、ボードには接続に必要なすべてのパッドがありますが、通信ケーブル用のコネクタはありません。 追加するには、次のコンポーネントが必要です。



  1. 欠陥のあるゲームボーイのコネクター
  2. 3つの220オーム抵抗器
  3. 3個の100 pFコンデンサ
  4. 3つのデュアルダイオード（または6つの通常のゲルマニウムダイオード）


したがって、彼らは通常のゲームボーイのボードを見ます（何らかの理由でポジション3と4があり、それぞれ4枚）：







コネクタのピン配列とその嵌合：







カートリッジケースを開けるのは簡単ではありません。 そこでは、3.2 mmの六角形だけでなく、直径が小さすぎるドライバー用の穴も必要です。 私は少しドリルダウンしなければなりませんでした：







ブレッドボードモデルの一部で、この図を3回組み立てます。







したがって、結果は、オリジナルのデュアルダイオードとゲームボーイのその他の詳細で表示されます。







元のデュアルダイオードのピン配列を決定するときは、文字KまたはCに焦点を合わせます（図の文字または文字のタイポが実際にアノードの出力の隣にあるかどうかは明確ではありません。それ以外の場合は電源が短絡します）。







次のように、コネクタをSuper Game Boyボードに接続します。 



 |   |  --------+----------------------------------------------------------------- 1 |   +5  ( 1    Game Boy) 2 |        70  3 |        69  4 |   5 |        68  6 |     ( 32    Game Boy)


自家製の回路基板の3つの回路すべてで、文字Pのマークが付いたパッドをプラス電源で、文字Nのマークが付いたパッドを共通のワイヤで接続します。 ダイオードは逆電圧によって閉じられ、コアに静的ケーブルが現れた場合、信号線から電源バスの1つにそれを取ります。 同様のダイオードがマイクロチップ、たとえば561シリーズに存在します。



この写真は、必要なプロセッサピンを見つけるのに役立ちます。







以下に、コネクタを出力する必要がない方法を示します。







これまでに修正されたスーパーゲームボーイをゲームボーイ用のカートリッジなしでSNESにインストールし、コンソールをオンにすると、カートリッジがないことに関するメッセージが表示されます。 コンソールをオフにします。 Super Game Boyと通常のGame Boyで同じマルチプレイヤーゲームのカートリッジをインストールします。 通信ケーブルでデバイスを接続し、電源を入れて、すべてが機能することを確認します。



初めて、著者がすべてを始めましたが、配色は奇妙な方法で切り替わりました。 彼は回路を分解し、再組み立てした後、すべてが正しく機能しました。 その後、彼は、コネクタケーブルを取り外した穴にホットメルト接着剤を流しましたが、コネクタ自体も取り付ける方が良いでしょう。
