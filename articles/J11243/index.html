iPod：毎日のプログラム

iPod'aとその競合他社との主な違いは、このプレーヤー用に多くのプログラムとユーティリティが作成されており、デバイスでの作業が容易になることです。 彼らの行動の範囲は広大です。音楽を使った平凡な「ドレッシング」から、プレーヤーのファームウェア自体の変更まで。 そして、あなたがこれらすべてのソフトインキ側を通り過ぎるならば、そのステップは間違っているでしょう。 Apple iPodを操作するための最も便利なプログラムのセットは次のとおりです。



一部のプログラムはプレーヤーと特別に機能しますが、他のプログラムは機能せず、他のプログラムは一般にiTunesのみで機能するため、3つの方向で説明します。 もちろん、プレーヤーのユーティリティから始めます。





iPodユーティリティ



iPod Video用に作成されたプログラムのほとんど。 もちろん、これらはビデオコンバーターです。 ネットワーク上にはそれらの多くが存在するため、そのうちのほんのいくつかを紹介します-最も興味深いものです。 さらに、多くの優れたものがMacOS専用に作成されているため、それらの量については言及しません。 最後に、Linux用の小さなユーティリティセットがありますが、これも注目に値します。



Transipod （MacOS）

movおよびmpeg形式のビデオのみを変換するための小さなユーティリティ。 「iPod自体が再生できる形式を変換する理由は何ですか？」 実際、このプログラムは、 320 * 240ピクセルの解像度、つまりiPod Videoの解像度で画像を拡大縮小するように設計されています。 その結果、プレーヤー自体は画像を拡大縮小しないため、詳細を失わずにバッテリーを節約できます。



iFeedPod （MacOS）

RSSフィードを読み取り、それらをiPodに転送するためのシンプルなプログラム。 必要なRSSフィードを作成し、ダウンロードしてから、iPodに転送するRSSフィードの横にあるチェックボックスをオンにして、[同期]をクリックすると、ニュースがiPodに送信されます。 シンプルで便利！



無料のiPodビデオコンバーター （Windows）

DVD、VCD、AVI、MPEG、WMV、RM、RMVB、DivX、ASF、VOBなど、最も有名なビデオ形式をサポートしています。 このプログラムは、ビデオデータを変換するためのプレビューモードとバッチモードを便利に実装します。 設定では、オーディオ/ビデオトラックのエンコード品質を設定し、画像の解像度を設定できます。



PodPlus v1.2 （Windows、有料-15ドル）

iPodを操作するための非常に便利なマネージャー。 メディアファイルをハードドライブに簡単にコピーします。 iPodをMicrosoft Outlookと完全に同期し、RSSフィードとポッドキャストをダウンロードできます。 備考： Microsoft .NET Framework 1.1をインストールする必要があります -これはバージョン1.1であり、それ以外の場合は動作しません。



Anapod Explorer （Windows、ライセンス-25ドル）

最も興味深い有用なユーティリティの1つ。 Explorerに組み込まれています。 コンテキストメニューの項目を使用して、音楽やその他のファイルをコピーできます。 ソフトウェアの最も便利な機能の1つは、プレーヤーで使用できない.oggやその他の形式を.mp3および.aacに変換できることです。



YamiPod （Windows、MacOS、Linux）

チェンバレンに対する私たちの答え、すなわち iYunes'y。 プログラムはJavaで書かれています。 Windows、MacOS、LinuxのすべてのOSで動作し、（!!!）インストールは不要です。 つまり プレーヤーをUSBドライブに変えます！ このプログラムの主な利点には、誤って消去されたトラックの復元、RSSおよびポッドキャストの操作、m3u / plsプレイリストのインポート、エクスポート、管理が含まれます！



A-one iPodコンバーター （Windows）

iPod用の別のビデオコンバーター。 AVI、MPG、MPEG、ASF、WMV、MOV、3GP、AMR、FLV、FLIC、SWF 、およびその他のビデオ形式のサポートと組み合わせて 、バッチ変換、オーディオおよびビデオストリームの変換品質を個別に選択および構成し、結果をプレビューします。高速かつ高品質でシンプルなインターフェースにより、このユーティリティはかけがえのないものになっています！



goPod （Windows、MacOS、Linux）

すべてのOSで動作する無料のユーティリティ。 プレーヤーのヨーロッパ版からサウンド制限を削除するように設計されています。 プログラムを起動し、同期に関する回答を待って、[Go]ボタンをクリックします！ これでロックは解除されました！



SyncWrangler （Windows、$ 29.99ライセンス）

iTunesとiPodの間でファイルを相互交換するためのプログラム。 2倍のシンプル！ 2つのドロップダウンリストで、目的のプレイリストまたはその他のファイルを選択して同期します。



WordPod （Windows、MacOS）

WordPodは、iPodで本を読むのに役立つ直感的でシンプルなプログラムです。 WordPodはプレーンテキストの電子書籍をiPod互換形式に変換し、プレーヤーにコピーします。



iTunesのユーティリティ



メディア自体は非常に機能的です。 したがって、この「獣」にはほとんど必要ありません。 しかし、Volume LogicやAlbum Cover Finderなど、いくつかの便利なユーティリティに注意することをお勧めします。



ボリュームロジック （ Windows 、 Mac OS ）

非常に便利でシンプルなユーティリティ、またはむしろ、曲の音質を大幅に改善するiTunesプラグイン。 ボリュームロジックは、iTunesで再生される音楽の音量とサウンドバランスを調整します。 サウンドを調整するには、5バンドプロセッサを使用します。 ボリュームロジックはその場で音を調整し、ファイルに変更を加えません。



アルバムカバーファインダー （ Windows 、 MacOS ）

音楽ライブラリーで秩序を愛する人々のためのプログラム。 プログラムは、指定したアルバムカバーをAmazon'eとGoogle Imagesで検索します。 iTunesとの優れた統合。



エピローグ
