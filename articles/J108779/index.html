Webサイトを作成するためのASP.NETの既製ソリューション

多くの場合、ASP.NETプラットフォームには、どの企業のWebサイト、フォーラム、ブログなどに基づいた既成のソリューションがないという主張が提示されます。 など この短い投稿では、 Web Platform Installer（WebPI）を使用してインストールできる最も人気のあるASP.NET Webアプリケーションをリストします。
