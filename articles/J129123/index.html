ロンドンから香港まで176ミリ秒

高周波トレーダーの利益に役立つ通信会社のもう1つの成果。 BSO Network Solutionsは、ロンドンから香港までのバックボーンで176ミリ秒のpingを実行しました。これは、世界最大の証券取引所の2つを持つ都市間の通信チャネルの記録的な結果です。



報告されているように 、初期のトレーダーは仲裁のためにシベリア横断チャネルを使用する必要がありました。 それらの場所に光ファイバーを敷設するのが困難なため、ロシアのいくつかの地域を回りました。 新しい高速道路では、トランジットモンゴルのチャネルとロシアと中国の黒い光ファイバーの新しいセクションを使用しているため、直線に近いルートができました。 応答時間は20ミリ秒以上短縮されました。



理解できるように、ヨーロッパからアジアへの最短ルートがロシアを通過するため、高周波トレーダー間の対決においてロシアの役割は非常に重要です。



球上の最短経路では、ロンドンと香港間の距離は9618 kmです。







BSOネットワークソリューションのマネージングディレクターは、フィナンシャルタイムズとのインタビューで、ロンドンと香港の間のルートにある潜在的に重要なハブとしてのモスクワへの関心が高まっていると商社が語った。



BSO Network Solutionsは、地元の通信会社から光ファイバー回線をリースしています。 すべてが計画通りに進んだ場合、すぐに彼らはロシアの領土全体でバンドを倍にすることを計画します。
