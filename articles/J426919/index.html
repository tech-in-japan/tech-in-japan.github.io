Microsoft Learnの紹介

Microsoft Learn Webサイトは、製品とサービスについて学ぶ最も簡単な方法です。 ここでは、実際のタスクに基づいたインタラクティブなトレーニングを受けることができます。 23か国語にローカライズされた無料の資料が多数用意されており、総学習時間は80時間以上です。 トレーニングは当社の製品の多くをカバーしており、新しいリソースで絶えず更新されています。 詳細はこちら！







詳細については、Microsoft Learn Startup Blog Entryを参照してください 。



Learnウェブサイトは以下を提供します：



  • 23言語の無料トレーニング資料（総学習時間-80時間以上）。
  • Learnサンドボックスを使用してリソースを作成および管理するためのAzureへの一時的な無料アクセス。 認証にクレジットカードは必要ありません。
  • ウォークスルーとインタラクティブなコード演習を使用した実践的なトレーニング。
  • トレーニングの進捗状況を追跡し、知識をテストし、スクリプトをテストし、ポイントを獲得し、レベルを克服し、証明書と賞を受け取ります。


コンテンツの検索とフィルター



レビューページでは、トピックに関心のある教材を検索すると便利です。 この関数は、全文検索と結果のフィルタリングを提供し、次のようなさまざまな検索タグを使用することもできます。



  • 専門分野：管理者、開発者、ソリューションアーキテクト、ビジネスアナリスト、ビジネスユーザー。
  • レベル：初級または中級。
  • 製品：仮想マシン、Azure Storage Service、Power Apps、Power BIなど。






モジュールとトレーニングスキーム



すべてのトレーニング資料は、5〜7レッスンからなるモジュール形式で編成されています。 それぞれが特定のトピック専用のモジュールの学習には、30〜60分かかります。 図でわかるように、各モジュールの注釈とトレーニングタスクのリストが表示され、モジュールを完了するのに必要な時間と、累積経験ポイント（XP）の数が示されます。







学習パスは、Azureの例で効果的なソリューションを開発するための学習パスに見られるように、幅広いトピックをカバーするモジュールの順序付きリストです。







Learn Sandboxを使用したAzureへの無料一時アクセス



初心者の場合、無料のアカウントを確認する際にクレジットカード情報と携帯電話番号を提供する必要があるため、クラウドプラットフォームを理解するのは困難です。 さらに、登録ユーザーは、教育目的で組織のAzure作業サブスクリプションを使用することに慎重になる場合があります。 この干渉を排除するために、Learnサンドボックスを作成しました。 クレジットカードを必要としないクラウドサブスクリプションへの無料一時アクセスを提供します。 これで、高度なセキュリティのサポートにより、作業を中断することなく、リソースを研究および作成できます。







Azure Cloud Shellによるハンズオントレーニング



ユーザーは多くの場合、レビューでテキストとビデオが学習に重要な役割を果たす一方で、実践的な演習を好むことを示します。 この願いを実現するために、Azure Cloud ShellをWebサイトに追加しました。 このシェルでは、Azure CLI、Azure PowerShell、.NET、Java、Node.js、Go、Pythonを直接実行するか、 Git、Kubectl、Helmなどのさまざまなコマンドラインユーティリティを使用できます。 これらはすべて、プログラムをインストールまたは構成する必要なく、ブラウザーで直接機能します。

知識テスト



知識のテストは、提案されたもののリストから正しい答えを選択して、得られた知識を統合する必要があるテストです。 知識テストの主な目標の1つは、回答の正確性ではなく、資料の本質に焦点を当てることです。 エラーに対する罰則はありません。 代わりに、ユーザーに答えが正しくない理由が説明されます。







タスク検証



ユーザーがタスクにどれだけうまく対処したかを確認するために、タスクのチェック機能が提供されています。 次の例は、モジュール内でユーザーが作成したAzure仮想マシンが正しくデプロイされていることをAzure Resource Managerが検証する方法を示しています。







ポイント、レベル、証明書および賞



小さなゲーミフィケーションにより、学習の進行がより楽しく便利になり、Twitter、Facebook、LinkedInで学業の成功を共有することもできます。 これには、次のタスクが含まれます。



  • ポイント：モジュールレッスンを完了すると授与されます。
  • 証明書：モジュールのすべてのレッスンを完了した後に授与されます（以下を参照）。
  • 賞：トレーニングスキームのすべてのモジュールの実装に対して授与されます。
  • レベル：より多くのポイント、より高いレベル。






ユーザープロファイル
