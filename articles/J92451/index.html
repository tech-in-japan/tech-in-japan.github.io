CrunchGearシェフエディターJobsのFlashファン

むかしむかし、AppleとAdobeという2人の小さな男の子がいました。 子供の頃、彼らはよく一緒に遊び、一緒に遊びました。一般に、彼らは完全に調和して生きていました。 Appleが病気になり、Adobeが混乱せずに新しい友達を見つけた後、これまでフレンドリーなカップルが解散していました。 年が経つと、彼らは成長しました...しかし、今ではAppleはAdobeと共通の利益を共有していませんでした。彼らはかつて別れたからではなく、本当の友情がなくなったからです。 別の世界と考えることもできますが、多国籍コングロマリットがあります。







このようなものは、スティーブ・ジョブズが彼の会社とアドビとの間のパートナーシップを見ており、彼の手紙の中で、次の行から始めています。



「Adobe Flash製品に関する私の考えを共有したいので、消費者や批評家は、iPhone、iPod、iPadにFlashを表示できない理由をよりよく理解できます。 アドビは、私たちの決定はビジネスによって決定されたと考えています-AppleはApp Storeを保護したいと言っています-実際、決定は純粋に技術的な矛盾によって決定されています。 Adobeは、Flashが開いている間にシステムをシャットダウンすると主張していますが、実際にはすべてが完全に異なっています。



わあ！ 彼はシステムの「開放性」を議論に入れました。 迷惑になり始めます。 Appleは世界で最も閉鎖的な会社ですか？ OS XがMachコアに基づいており、FreeBSDからそのルーツを取っているという事実に注意を払わなければ、Appleは世界で最も閉鎖的な会社と見なすことができます。 ただし、ご想像のとおり、スティーブは既に、アドビが価格、製品の使用および更新の機能を制御していることを指摘しましたが、AppleはHTML5、CSS、およびJavascriptで完全にオープンなコンテンツでなければなりません。 ジョブは継続します：



「Flashを放棄することにより、AppleはオープンスタンダードよりもHTML5、CSS、およびJavaScriptを好みました。 Appleモバイルデバイスの特徴は、高性能とこれらの標準の使用です。 HTML5は、Apple、Google、および他の多くの企業で採用されている新しいWeb標準です。 HTML5を使用すると、Web開発者は高度なグラフィックス、デザイン、アニメーションを作成でき、サードパーティのプラグイン（Flashなど）を使用せずに開発を続けることができます。 HTML5は完全にオープンであり、Appleがメンバーとなっている標準化委員会によって管理されています。



スティーブはセキュリティも思い出しました（「Symantecは、Flashを2009年の最悪の情報セキュリティ製品と命名しました。また、FlashがMacコンピュータが故障する主な理由だと考えています」）。



「Flashは、タッチスクリーンの場合のように、ユーザーの指の代わりにマウスを使用して入力するコンピューターで使用することを目的としています。 たとえば、多くのサイトでは、ポップアップメニューや同様のオブジェクトなど、いわゆるロールオーバーを使用しています。これらは、マウスカーソルが画面の特定のセクターに入った時点でアクティブになります。 Appleの革新的なマルチタッチインターフェイスはマウスを使用しないため、オブジェクトを展開するという概念は機能しません。 ほとんどのFlashベースのサイトは、タッチデバイスをサポートできるようにやり直す必要があります。 開発者がFlashサイトを再構築する必要がある場合、HTML5、CSS、JavaScriptなどの最先端技術をすぐに使用してみませんか？」







スティーブの主なメッセージはアイデアでした-フラッシュは吸う。 何百万人ものFlash開発者が自分のアイデアをFlash for iPadに実装できなくなるのは正しいでしょうか？ 絶対にありません。 ユーザーがFlashサイトにアクセスするときに、曲がったコードや少量のシステムメモリが原因で常に問題を経験するのは正しいですか？ また間違っています。



Jobsには独自の強い主張があり、Adobeには独自の主張があります。 ただし、この問題はファンの分解のカテゴリに起因するものではありません。Appleはそのような決定を下しました。Flashは会社の経営陣によるとモバイルエレクトロニクス向けではないからです。 あなたがFlashの熱烈なファンなら、Androidデバイスを見てください。



Flashに関する考えは、Apple CEOのSteve Jobsからの公開書簡です。

Jobsの手紙に関するCrunchGearの編集長、John Biggsによる記事 。
