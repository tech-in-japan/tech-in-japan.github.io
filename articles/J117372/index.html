予算タブレットQumo Go！のレビュー

タブレットへの関心をきっかけに、すでに有名なQumoブランドのQumo Go！.. Go！の安価な（$ 250）デバイスの概要を提供したいと思います。 同社初のAndroidタブレットは、実際には、人気のあるSoC Telechips 8900（メーカーによると8902）に基づいて構築された膨大な数のデバイスの1つです。









それでは、デバイスの簡単な説明から始めましょう（完全な仕様はメーカーのウェブサイトで見つけることができます）：

  • CPU ：ARM v11 720MHz
  • RAM ：DDR2 256 Mb
  • 内部メモリ ：Nand 8Gb + SDカードスロット（最大32Gb）
  • 画面 ：7 "、抵抗性、800x480ピクセル
  • ネットワークインターフェイス ：809.11 b / g
  • その他のインターフェイス ：USB、USBホスト、HDMI、ヘッドフォン用3.5 mm、0.3 MPカメラ、マイク、スピーカー
  • 追加します。 特徴 ：Gセンサー
  • OS ：Android 2.1

タブレットは、明るい緑色の素敵で硬い段ボール箱に入っており、その上に（ロシア語で）製造元とデバイスの名前、簡単な技術仕様、使用可能な機器が示されています。 パッケージ内には、次のMID自体に加えて、かなり印象的なセットがあります。

  • ハードカバーブック
  • 充電器
  • USBケーブル
  • USBホストケーブル
  • USB-2.5DC充電ケーブル（はい、そのようなコネクタがあります）
  • ヘッドホン
  • クイックスタートガイド
  • 保証書

外観、品質の構築



iPadはiPadの影響を受けていることは明らかです-否定することは無意味ですが、「大型のタッチスクリーンとフレーム」などのデバイス用の別のフォームファクターはまだありません。 初めてデバイスを取り上げると、非常に細長いようです-アスペクト比がこれに寄与します。 タブレットを片手で持ち、もう一方の手でテキストを入力するのは非常に快適だからです。 デバイスの重量（400 g）-快適な範囲内-はそれほど重くなく、あまり軽くもありません。 フロントパネルには、Androidデバイス用の3つの標準メニューボタンがあり、ホーム画面に戻って「戻る」ことができます。 フロントパネルはプラスチック製で、スクリーンは保護フィルムで覆われ、リアパネルはアルミニウム製です。 ここで最初の欠陥が発見されました-長辺のサイドフレームの真ん中に、デバイスを手に持ったときにフレームのプラスチックが指で押されたときにわずかなバックラッシュときしみ音があります。 保護フィルムは非常にまぶしく光るので、明るい光の下でタブレットを使用することは困難です。 ケースの上端にはヘッドフォンジャック、下端にはスピーカーグリル、電源アダプター用の2.5コネクター、2つのUSBコネクター、SDカードスロット、HDMIコネクターがあります。 右側には、電源ボタンと音量ロッカー、プラスチックボタンがあります。 ケースは、アルミニウム製の背面カバーにもかかわらず、それほど剛性ではありませんが、一般的には非常に信頼性が高く、適切に組み立てられています（フレームの指定されたバックラッシュは考慮していません）。 そして今、いくつかの写真：

ロボットがスケートボード上を転がると、カスタマイズ、改善、その他何も行わない標準のAndroid OSデスクトップが表示されます。 そして、数秒後に、そのような魅力的な価格がどこから来たのか（HDMI、8GBの内蔵メモリ、カメラなどで）明らかになります。 スクリーン。 画面上で、製造業者は、はっきりしない色、貧弱なパレット、そしておそらく恐ろしいオブジェクトの間違った比率を保存することを明確に決めました。これは、画面の長辺に沿ってわずかに伸びる丸いオブジェクトで非常にはっきりと見えます。 この画面はテキスト用であり、映画用です。写真はあまり見やすいものではありません（面白い効果が得られる場合もあります）。 また、十分に大きな粒子は、小さなプリントの良好な表示に寄与しません。



ソフトウェアの基本配信には何が含まれますか？ キットには、ブラウザー、メール、地図などの標準アプリケーションに加えて、Androidマーケット、アプリインストーラーとアプリモンスター、Advanced Task Killer Free、SMB、FTP、組み込みビューアーなどを使用できる非常に優れたESファイルマネージャーが含まれます。 インターフェイスはローカライズされていますが、何らかの理由でロシア語キーボードがありません-すぐに市場からインストールする必要がありました。 ただし、他の大まかな箇所もあります。たとえば、「電話が切断されます」は少し不安でイライラします。 ただし、OSバージョン2.1がapp2sdでなくても、8GBの内部メモリのほぼすべてがファイルシステムのルートに割り当てられるという事実は喜ばしいことです。 ユーザーファイルは/ nandにあります-写真、音楽、ドキュメントなど。



このデバイスはYouTubeやDVD-Rip'amiで動画を再生できますが、これまでのところ奇妙な音声です。バックグラウンドで音楽を再生し、新しいタブを開いたりリンクをたどるときにインターネットをサーフィンすると、十分でないように「スタッター」が再生されることがありますプロセッサリソース。 （キットからではなく）ヘッドフォンを介した音質はかなりまともで、スピーカーの音は...まあ、ガタガタせずに聞こえますが、品質について話すことはできません。



デバイスをコンピューターに接続すると、ユーザーフォルダー/ nandがメディアとして使用可能になります。 USBホストは正常に機能し、挿入されたフラッシュドライブは自動的に検出され、その後/ scsiパスに沿ってファイルマネージャーで使用可能になりましたが、切断するには設定を入力する必要がありました。 もちろん、外付けハードドライブを接続することを考えるべきではありませんが、カメラを接続するだけで十分です。特に、タブレットは大きなショットをすばやくすばやくスケーリングできるためです。

追加のソフトウェアを入れます



空きディスク容量が大きいため、あらゆる種類のゲームやその他のものを入れるように促されますが、ここでは、画面に続く2番目の大きな問題、Angry Birdsを待っていました。 どういうわけか、Rioをプレイできますが、他のバージョンは完全に不可能であり、「ブレーキ」が原因ではなく、テクスチャとアニメーションスプライトの奇妙な問題が原因です。 ほとんどの通常のプログラムは問題なくインストールされます。 twitter、facebook、電卓、Google Skymap、Popular Mechanics、DropBox、SugarSyncなどのクライアントをインストールしました。 そして、DropBoxとSugarSyncで面白い問題が始まりました。 最初のファイルは動作しているように見えましたが、ダウンロードしたファイルを開くことは完全に不可能でした。ダウンロードしたファイルの場所が明確ではなかったためです。 数ギガバイトの内部スペースがありますが、少なくとも小さくてもSDカードが必要であることがわかりました。 また、Googleリーダーは無限の同期サイクルに入り、使用することはできません。 Skype-完全に動作したくなかったため、Liteバージョンをインストールする必要がありました。 今、フロントカメラを使用する方法は謎です。



その後、Gセンサーで楽しい不具合が発生しました。 画面の反転をサポートしない一部のアプリケーション（RealCalcおよびPopular Mechanicsクライアント）は、180度上下逆さまに起動しますが、スカイマップは通常90度回転し続けるため、それに追いつく方法はありません。 センサーが正しくキャリブレーションされていないか、設定に何らかのエラーがあると思われます。 はい、そしてもちろん（Gセンサー）はオフにすることができます。



全体として、ソフトウェアの市場があるとしても、特定の問題が発生し、それらのほとんどはソフトウェアの部分、つまり OS /ファームウェア自体に。



結論として



これまでのところ、私はQumo Go！の二重印象を持っています..一方、中国の名前とは異なり、かなり高品質のアセンブリ、優れた技術仕様（このような価格で）、現地のサービスと保証があります。 一方、失敗した画面、ファームウェアのバグ、ソフトウェアの問題。 Qumoのこの2番目のデバイス（1つ目はLibroの「リーダー」）を既に持っています。ソフトウェアの更新が積極的に開発およびリリースされることにより、少なくともソフトウェアの問題が後続のファームウェアで修正され、メーカーが顧客を忘れないことを願っています。 Go！には噂があるようです。 2.3へのアップデートが準備されていますが、現時点では、タブレットはほとんどの場合、YouTubeプレーヤー、音楽プレーヤー、メールツイーターアグリゲーターの機能を実行します（実際、このために購入されました）。 このデバイスまたは類似のデバイスを魅力的な価格で入手することを考えている人は、考えられる問題に対処する準備ができているかどうか、そして最も重要なこととして、問題を解決する準備ができているかどうかを考えてください。 ;）
