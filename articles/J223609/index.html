ゲーム用コンピューター（ベアボーンシステム）MSI Nightbladeのビデオレビュー





MSI Nightblade-ゲーマー向けに設計されたベアボーンタイプのパーソナルコンピューター（ユーザーが自己組み立てするように設計され、ベアボーンベースと呼ばれる「フレーム」システムに基づいて組み立てられたコンピューター）。



寸法が345.8 x 277.3 x 175.7 mmのコンパクトなケースには、第4世代Intel Coreプロセッサー（レビューでは車のi7-4770K）とRadeon R9-290Xビデオカード（レビューでは）またはGeforce GTX 780Tiが収納されています。 RAMの最大容量は16 GBで、ユーザーは2つのスロットを使用できます。 データストレージには、1つの3.5インチベイと2つの2.5インチベイがあります（このバージョンでは、そのうちの1つに2つの128 GB mSATAドライブがraid0に統合されています）。 その他の機能には、E2200 Killer Gigabit Lanネットワークカードや、独自のOC Genieテクノロジを使用してプロセッサとメモリを自動的にオーバークロックする機能が含まれます。



ポートのセットには以下が含まれます。



背面：e-SATA、HDMI、DVI、ディスプレイポート、S / PDIF、PS / 2、CMOSリセットボタン、Wi-Fiアンテナ用コネクタ2個、OFCオーディオジャックx 6



内部コネクタ：1 x ATX 24-Pin、1 x 8-Pin ATX 12V、5 x SATA 6GB / s、1 x Clear CMOS



USB：背面：2 x USB 2.0、4 x USB 3.0 /前面：2 x USB3.0、2x USB2.0
