過去40年間のトップ10のIT製品

40年の間に、何千ものIT製品が登場しました-それらの多くはこの分野の専門家に大きな影響を与え、確かに記憶されています。 しかし、ITと、さまざまなストライプのIT担当者とユーザーの作業を行うプロセスを実際に変えた製品はごくわずかでした。 ComputerWorldは、すべてを変えた10のIT製品のリストをまとめました！



IBM System / 370 ：1964年、System / 360メインフレームにより、IBMは、顧客が次のバージョンのマシンを購入したときにソフトウェアを書き換える必要がないことを約束しました。 それは革命的なアイデアでした。 1971年、System / 370はその約束を再確認し、ライバルの「メインフレーム」メインフレームの運命を決定しました。

イーサネット ：この技術は、1973年にゼロックスPARCによって開発され、1980年に標準の仕様が発表されました。



IBMパーソナル・コンピューター ：1981年、オープン・アーキテクチャーを備えた最初のパーソナル・コンピューターがリリースされ、ユーザーが自分でコンピューターを構築できるようになりました。 誰もがそれをとても気に入り、誰もそれを拒否しませんでした。



Apple Macintosh ：1984年、Macはユーザーに使いやすいコンピューターの理想を示しました。 その結果、マウス、グラフィカルインターフェイス、周辺機器の「ホット」接続、およびMicrosoftと呼ばれるWindowsの偽物ができました。



SAP R / 3 ：データ部門が独自の財務会計ソフトウェアを作成した時期を覚えていますか？ 1992年以来、SAP AGはこのコードをすべてサポートする必要性を排除しました。



Salesforce.com ：サービスとしてのソフトウェアは非常に成功しています。 SAPが購入ではなく、購入を申し出た場合、1999年にSalesforce.comはレンタルを申し出ました。



Linux ：Salesforce.comをレンタルする必要がある場合、Linuxは無料です。 しかし、もっと重要なことは、1991年に、Linuxは世界のさまざまな地域のプログラマーの大規模なグループがITインフラストラクチャの大部分を開発できることを示しました。



Netscape Navigator ：これは最初の人気ブラウザではありません。以前はNCSA Mosaicでした。 しかし、Mosaicの作成者であるMark Andressenが1994年にクッキーを追加したとき、NetscapeはWebをグローバル市場に変えました。



ダイナミックRAM ：IBMによって発明されましたが、Intelは1970年に最初の商用DRAMを販売しました。 2年以内に、DRAMは1950年代から使用されていた磁気コアメモリよりも売れ始めました。 また、磁気コアのメモリとは異なり、DRAMはムーアの法則に従います。時間の経過とともに、DRAMは安価になり、普及します。





BlackBerry ：携帯電話はユーザーが連絡を取り合うことを可能にしますが、1999年以降、Research In MotionのBlackBerry電話を使用することで、ユーザーは常にメールにアクセスできるようになりました。
