財務省は、インターネットを介してプログラムを転送するときにVATを支払う手順を説明しました







外国ソフトウェアのロシアの販売業者の1つは、インターネット経由での転送を規定するライセンス契約に基づいてソフトウェアの権利を取得する場合に、Googleの税法の修正を適用する手順に関する要求とともに財務省に申請しました。



以前に、 Habréで、ソフトウェアライセンス契約に基づく販売のVAT免除は、ソフトウェアのライセンスコピーが実際にユーザーに転送されるという条件で保持されると書きました 。



財務省はこの結論を確認し、ライセンス契約に基づいてソフトウェアの権利を取得すると、そのようなソフトウェアを転送する方法に関係なく、VATからライセンス料を免除する権利は、ロシア連邦税法149条のパラグラフ26、パラグラフ2に基づいて保持されることを説明しました（ 12.01の手紙を参照） .2017 N 03-07-08 / 555 ）。



したがって、ライセンス契約に基づいてVAT特典を取得する傾向は続くはずです。



これは、ロシアの外国ソフトウェアの販売業者だけでなく、インターネット経由でロシアでライセンスを販売する著作権所有者にとっても朗報です。なぜなら、こうした操作に対する課税の原則は同じだからです。



興味深いことに、財務省のこの立場を考えると、ライセンス契約の下でロシアのユーザーに提供されたアプリケーションのコストに対するVATを差し控えるというGoogleの決定は不合理に思えます。 おそらく開発者はこの質問をGoogleに尋ねるべきですか



この問題をより詳細に理解してみましょう。



2017年1月1日から、「電子サービス」の特定の課税条件に関連するいわゆる「Google税に関する法律」（2016年3月7日付けの連邦法N 244-to）に基づいて、ロシア連邦の税法が改正されました。



これらのサービスの中には、とりわけ、電子コンピューター（コンピューターゲームを含む）のプログラム、インターネットへのリモートアクセスを提供することによる更新、追加機能を含むデータベースの使用権の付与があります（参照ロシア連邦税法第172条第1条第2項）



同時に、ロシア連邦税法第21章の目的で、電子コンピューター（コンピューターゲームを含む）、有形メディア上のデータベース（税法の18.174.27項）のプログラムの使用（使用権の譲渡）が電子サービスに適用されないという予約が行われましたRF）。



したがって、新しい法律は、有形メディア上のソフトウェアのライセンスの提供と、インターネットを介して送信されるソフトウェアへの権利の付与を正式に分離しています。 さらに、最後の操作は、電子形式のサービスとしてロシア連邦税法第172.2条の条項1に基づいて認定されます。



この点に関して、多くの弁護士は、インターネットを介してソフトウェアをライセンス供与するとVATが発生し、そのようなソフトウェアを有形メディアに転送する場合、VAT免除が適用されることについて懸念を表明しました。



次の引数が与えられました。 VATは、有形のメディアで転送されたソフトウェアの権利を販売する場合にのみ発生します。 この場合、ロシア連邦税法第149条の第26節2項に従って例外が適用されます。これにより、電子コンピューターのプログラムに対する独占権はライセンス契約に基づく課税の対象にはなりません。



同時に、インターネットを介してプログラムまたはデータベースを転送してソフトウェアの権利を販売する場合、VATが発生します。 この場合、電子形式のサービスの提供について話します。 このようなサービスは独立した販売対象であり、VATが免除される業務のうち、ロシア連邦税法149条には示されていません。



この点で、外国のソフトウェアのディストリビューターは、財務省にGoogleの税法によって導入された変更の適用を明確にするよう要求しました。



2017年12月1日N 03-07-08 / 555日付の書簡で表明された立場によると、財務省は実際にインターネットを介したソフトウェア権利の提供を電子サービスに言及しています。 ただし、この場合、ライセンス契約に基づいて権利が付与された場合、VATからのソフトウェア権利の実現の免除に関するロシア連邦税法第149条第2段落のサブパラグラフ26の適用は除外されません。



したがって、このような状況の財務省は、ライセンス契約に基づいてプログラムに権利を譲渡する電子形式のサービスに反対していません。 どうやら、財務省は、電子形式のサービスにはライセンスが含まれていると考えています。



専門家の意見では、実装の異なるオブジェクトのこのような混合物は、少なくとも奇妙に見えます。 おそらく後ほど、このテーマに関するいくつかの明確な手紙が表示されます。



特に、SaaSモデルに基づいたソフトウェアへのリモートアクセスのライセンスの問題は、財務省の説明の範囲外のままであることに注意する必要があります。



ただし、現時点では、ダウンロードされたソフトウェアのライセンス料は、有形のメディアまたはインターネットを介して送信されるかどうかにかかわらず、VATに課金されないと信じるあらゆる理由があります。
