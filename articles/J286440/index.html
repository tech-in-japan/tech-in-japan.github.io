ロシアのサービス「MyOffice」はMicrosoft OfficeおよびGoogle Docsと競合する

2015年春に、New Cloud Technologies（NOT）は、Microsoft OfficeおよびGoogleドキュメントと競合する新しいMyOfficeサービスを開始します。 開発者は野心的な目標を設定しました。 2020年までに、同社は国内市場の25％を獲得し、ヨーロッパおよびラテンアメリカの市場に参入したいと考えています。



NOTのCEOであるDmitry Komissarovは、Cnewsへの新しいサービスについて話しました。



サービスの作業は2013年に開始されました。 開発チームには130人が含まれており、その一部は、レキシコンテキストエディタおよびQuickOfficeの元開発者です。



「MyOffice」には、クラウドサービスに必要なすべてのオフィスアプリケーションのセットが含まれています。テキストエディター、テーブルエディター、プレゼンテーション、メールサービス、連絡先のディレクトリ、カレンダーです。 サポートされている形式は、ODF、Microsoft Office（doc、docx）、およびPDFへのエクスポートです。





iPadのMyOfficeテキストエディター



現時点では、Web、iPad、Android OS向けのMyOfficeサービスのベータ版のみがあります。 最初の公開リリースは4月から5月に予定されています。 9月には、LinuxおよびOS Xのバージョンがリリースされます。





AndroidタブレットのMyOfficeスプレッドシートエディター



プロジェクトの主な投資家はService Plusです。 Kontur.Fokusサービスの情報によると、プロジェクトでのシェアは88％です。 別の7％はRostelecom Alexei Basovの副社長に属しています。 残りの5％はDmitry Komissarovが所有しています。



Izvestia新聞によると 、Dmitry Komissarovは以前、Pingwin Software（通信省とIT Co.の共同プロジェクト）のCEOを務めていました。 。 しかし、通信省の指導者の交代後、プロジェクトは中止されました。 2014年3月、コミサロフは会社New Cloud Technologiesを設立しました。



アカウントライセンスの販売、プライベートクラウドをサポートするためのサブスクリプション料金、および通信事業者、ホスティングプロバイダー、デバイスメーカーを介した製品の配布から利益を受け取るつもりはありません。 個人ユーザーは無料でサービスを提供する予定です。





ブラウザウィンドウのプレゼンテーションエディタ「MyOffice」



2020年までに、サービス開発者はブラジルとラテンアメリカの市場で20％、ヨーロッパ市場で最大10％のシェアを獲得する予定です。 海外では、「MyOffice」は別のブランド-Collabioで提供されます。 しかし、オフィスアプリケーションのセグメントの世界市場の95％はマイクロソフトに属しています。
