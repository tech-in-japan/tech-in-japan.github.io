D-Wave Systemsは最初の2000量子ビット量子コンピューターを販売しました

1月23日に、D-Wave Systems は 2000量子ビット量子コンピューターD-Wave 2000Qの販売開始を発表し、すでに最初のモデルを1500万ドルで販売しています。



/写真： Steve Jurvetson CC



買い手は、サイバーセキュリティに従事する会社Temporal Defense Systemsでした。 TDSのCTO、James Burrell氏は、同社はD-Wave 2000Qを使用して、新しい脅威保護とサイバー犯罪者識別ソリューションを開発する予定だと言います。



D-Waveシステムは、キュービットを使用してデータを保存します。 従来のシステムとは異なり、情報をゼロ、1つ、または両方で同時にエンコードします。 このため、D-Waveは状態の膨大な組み合わせを管理できるため、特定のクラスの問題をより効果的に解決できます。



D-Waveは昨年9月に量子コンピューターを公開し、新しいソリューションには2,000量子ビットが含まれると述べました。 これは、前世代の量子コンピューターである8月に発売されたD-Wave X2の2倍です。



D-wave 2000Qは、 量子アニーリングの原理で動作する、いわゆる断熱コンピューターです。



これは、多数のコンポーネントと制御されたパラメーターの量子システムです。 開発者は、非常に低い温度まで冷却します（前のモデルのコンピューターは15ミリケルビンの温度で-273°Cの程度で機能しました）。開発者は、システムが最小エネルギーに達すると想定し、その後、設定パラメーターをゆっくりと変更し、量子力学の法則を使用して元のシステムからシステムを移行します状態から量子トンネル効果による最小エネルギーの新しい状態へ。



新しいD-Waveの機能として、個々のキュービットのアニーリングの頻度を調整して生産性を高めることができます。 新しいコンピューターは、量子アルゴリズムと従来のアルゴリズムを組み合わせて、計算結果の選択を最適化します。



N + 1のインタビューで、ロシア量子センターの超伝導量子回路グループの責任者であるアレクセイ・ウスティノフは、D-Waveの用途を説明しました。 アプリケーションの1つの領域は、コスト関数の最適化です。



あなたには多くの選択肢、多くの目標があります。 道路、費用、時間などを最適化しながら、さまざまな場所にいる100万人の顧客を訪問する必要があるとしましょう。



「これは典型的なセールスマンの仕事です。 通常のコンピューターでは、解決するのは非常に困難です-多くの異なるオプションをソートする必要がある、 と Alexey は言います。 -量子力学では、トンネリングなどの重要なプロセスがあります。



D-Waveを使用すると、状態間の量子トンネル効果により、状態を局所エネルギー最小値と「接続」できます。 この場合、多くのパラメーターが同時に変更されるため、トンネリングが発生します。 完全なインタビューはこちらでご覧いただけます 。



プレスリリースで、D-Waveの代表者は、2000Qが以前のバージョンに比べてより複雑な問題を解決できると述べました。 また、生産性の向上は、サイバーセキュリティ、機械学習、バイオテクノロジーなどの分野の発展を促進するはずです。 同社は、特殊なアルゴリズムが従来のサーバーよりも最大1,000倍、さらには1万倍も高速に実行できることに注目しています。



同時に、D-Waveは顧客にソリューションを提供するだけでなく、リモートワーク用に量子マシンのパワーをレンタルすることも提案しています。
