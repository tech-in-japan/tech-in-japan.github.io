現在のネットワーク接続速度を表示するためのアプリケーション






予想どおり、Windows 8ではガジェットをデスクトップに追加する機能はデフォルトで無効になっていますが、仕事中にネットワーク接続アクティビティを検討するのに慣れています。以前のWindows 7では、ネットワーク監視ガジェットまたは同様の機能を備えたプログラムの有料版を使用していました Windows 8への移行に伴い、ガジェットは消滅し、有料のものには現在のアクティビティに関する簡単な情報に加えて、不要な機能が多数含まれています。 したがって、簡単な小さなプログラムを自分で作成して共有するというアイデアが浮上しました。突然誰かが役に立つでしょう。 興味のある方は、habrakatの下でお願いします...



このアプリケーションは、プラットフォームバージョン4.0のC＃で記述されており、Windows 7、Windows 8、Windows Server 2008 R2でテストされています。

  • 配布はここで行うことができます
  • GitHubソースコード




コンテキストメニューから呼び出すことができる次の機能（原則的には必要ではありません）があります。





1）すべてが行われたネットワーク接続アクティビティの表示



2）自宅でWi-Fiアダプターなどのアダプターを選択しますが、イーサネットで作業するときは接続します





3）配色の選択、よりコントラストと便利な表示を構成できるようにする





4）システムの入り口で自動開始のオン/オフを切り替えます。必要に応じて、自動開始を選択しています。



これはまだ最初の試用版ですが、私はすでにそれを使用しており、かなり満足しています。私だけでなく役に立つと思います。 改善のためのアイデアがあれば、アプリケーションを聞いて更新できることを嬉しく思います。
