WindowsでWiFiをアクセスポイントモードに切り替える

職場で制限なしにフルインターネットを取得するには、ネットブックと3gモデムを使用する必要がありました。 最近、iPod Touchを買って、私は本当にそれをインターネットに接続したかった。 スマートフォンがそれ自体でインターネットに接続できる場合、プレーヤーの場合は多少難しくなります。



1つのオプションは、同期ケーブルを使用して接続することです。 このオプションは、ケーブルを常に持ち歩く必要はまったくなく、iTunesもインストールしたくないため、あまり便利ではありません。 この記事で説明する2番目のオプションは、WiFi経由の接続です。 Habrahabr は、同様のタスクとその解決策については既に言及していますが、Linux向けです。 私たちのタスクは、Windows用に同じ設定を行うことです。



だから私たちは何を持っています



-ネットブック、ASUS 1215B

-USB 3Gモデム経由のインターネット

-WiFiを備えたプレーヤーとスマートフォン



挑戦する



-すべてのデバイスのインターネットに乗る





解決策



開始する前に、コンピューターのWiFiカードドライバーを更新することをお勧めします。 個人的には、私はこれをしませんでしたが、誰もが自分で決めます。



方法


-ConnectifyやVirtual Routerなどのグラフィカルユーティリティを使用できます。 彼らは働き、うまく働きさえするが、資源を食べる。 一部の人々は、資源を非常に多く食べると不満を述べました。



-netshを使用して、コンソールから自分ですべてを実行できます。 これが私が選んだ方法です。 すべてのアクションは、管理者に代わって実行する必要があります。



ネットワークを作成する


まず、次のコマンドを使用してワイヤレスネットワークを作成する必要があります 

netsh wlan set hostednetwork mode=allow ssid="MyHomeInternet" key="pass12345678" keyUsage=persistent


MyHomeInternetはネットワーク名（ssid）、pass12345678は入力用のパスワードです。



コマンドの成功は次のようになります。 

        .  SSID    .         .


これで、 コントロールパネル \ ネットワークと共有センター \ アダプター設定の変更に移動できます 。 「ワイヤレスネットワーク接続2」と呼ばれています。



接続管理


接続は「接続なし」ステータスにあるため、有効にする必要があります。 これを行うには、次のコマンドを実行します。 

 netsh wlan start hostednetwork


Hosted network is runningというフレーズを受け取った後、 ネットワークが実行され 、接続のリストにネットワーク接続が表示されます。 この例では、これはMyHomeInternetです。



ネットワークを停止するには、次のコマンドを使用する必要があります。 

 netsh wlan stop hostednetwork




インターネットに接続します


WiFiネットワークが作成されており、接続することもできますが、インターネットはおそらく存在しないでしょう。 この誤解を修正するには、次のことが必要です。

- [コントロールパネル]、[ ネットワークと共有センター ]、[ アダプターの設定の変更 ]の順に移動します 。

- あなたがオンラインに接続する際の接続のプロパティに入ります （インターネットMTSを持っています）。

-[ アクセス ]タブ。

- [他のネットワークユーザーがこのコンピューターのインターネット接続を使用できるようにする]の横のボックスをオンにし、リストから作成したネットワーク-[ワイヤレスネットワーク接続2]を選択します。

-[OK]をクリックします。

-3G接続の無効化と有効化。 （理想的にはコンピューターを再起動します）

-WiFiネットワークを無効および有効にします。



どうした


理想的には、これらのアクションの後、ラップトップは小さなWiFiポイントに変わります。 これを行うには、iPodを取り出し、WiFiをオンにして、作成されたMyHomeInternetネットワークを確認します。 パスワードを入力して接続します。 インターネットがあります。



制限事項



-まず、Windowsを起動するたびに、 netsh wlan start hostsnetworkコマンドを使用してネットワークを起動する必要があります。 小さなスクリプトを記述して自動ロードに送信することで解決します。 別のオプションは、デスクトップにショートカットを作成し、必要な場合にのみネットワークを開始することです。

-OSはWindows 7でなければなりません。

-接続されたデバイスはWPA2-PSK / AESをサポートする必要があります



問題



ネットワークが作成されていません


-ワイヤレスアダプタードライバーを新しいものに更新する

-管理者としてコンソールを実行します



ネットワークは作成されましたが、開始されません


-おそらく、コンピューターを再起動し、管理者としてネットワークを開始すると役立つでしょう



ネットワークは作成され、開始されましたが、接続できません


-パスワードを確認してください

-「サーバー」上のTCP / IP接続のプロパティおよびクライアントネットワーク設定に手動で登録します。 たとえば、サーバーの場合：ip-192.168.137.1、マスク-255.255.255.0およびクライアントの場合：ip-192.168.137.2、マスク-255.255.255.0、ゲートウェイ-192.168.137.1、dns-192.168.137.1



ネットワークに接続されているが、オンラインにできない


-共有していることを確認してください（「インターネットへの接続」を参照）

-ドメイン名ではなく、IPアドレスを使用してください。 成功した場合、クライアントおよび/またはサーバー設定にDNSサーバーを入力することを考える時間を意味します。
