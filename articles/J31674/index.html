開発者の目から見たFirefox



誰がFirefoxを使用していますか？ たとえば、開発者として、FirefoxはInternet Explorerよりも標準に準拠していることに気付きました。 もう1つの重要な要素は、Firefoxにはさまざまなアドオンがたくさんあるということです。 「必須」セクションからのいくつかの追加は、すでにここに記述されています。 今日は次のバッチです！



Fire FTP-簡単かつ便利にファイルをWebサイトにアップロードできる安全なFTPクライアント。

LORI-リクエスト情報の寿命 -このアドオンは、現在のページがロードされた時間を通知し、ダウンロード速度を最適化します。

Adblock Plus-迷惑なポップアップ広告ウィンドウを取り除くことができ、その結果、より高速なサーフィンが可能になります。

ColorZilla-サイトの色は好きですか？ このアドオンを使用すると、この色を選択して16進数またはRGBに変換できます。

IEタブ-Firefoxで Internet Explorerを使用できるのにInternet Explorerを使用する理由。 開発者がサイトの機能を確認するのに理想的です。

測定する -画面上のアイテムの大きさを測定します。

スクリーングラブ -ブラウザ領域でのみ表示される写真を撮る代わりに、サイト全体のスクリーンショットを撮ることができ、これらのスクリーンショットをPNGおよびJPG形式で保存できます。

HTML Validator - HTMLにエラーがあれば、それを通知します。

FirebugはFirefoxの最も強力なデバッガーです。紹介する必要はないと思います。 それ自体には多くの追加があります。

そして最後に、追加＃1を忘れてしまいました。これを使用し、これを使用します。

Web Developer - Web開発者向けの多くのツールを備えたメニューとツールバーを追加します。

ほぼ毎日表示される多くのアドオンがありますが、私は聞いたことも見たこともありませんが、 サイトにアクセスすると完全なリストを取得できます。

さて、あなたは私がほぼ毎日開発で使用しているアドオンのリストを知っていますが、今は何を使用していますか？
