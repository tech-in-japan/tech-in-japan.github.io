Vivaldi 1.0.344.24ビルドの黒いタブなどを修正

みなさんこんにちは！



今日は、いくつかの重要な修正を含む特別なビルドです。 お気づきかもしれませんが、アセンブリの出力頻度は、いくつかの重要なバージョンのアプローチで増加します。 そうです-今回は、ユーザーが判断できるように、Vivaldiブラウザの2番目のベータ版を表示する準備をしています。 近日中にリリースされ、最も重大なエラーが修正されるように、可能な限りのことを行うよう努めています。



そこで、本日、以下の問題ですべてが適切な品質で修正されたかどうかを確認することを提案します。



  • カスタムサムネイルの表示パネルが消えた。
  • 非アクティブなタブの「黒」（または白、またはリクエストに応答しない）ページ。
  • Macのキーボードショートカット
  • いくつかの人気サイトとの互換性。
  • Linuxパッケージの依存関係エラー。
  • WindowsおよびLinuxでのミュートボタンの配置。
  • WindowsおよびMacでNPAPIを無効にしました（デフォルトで有効になりました）。


これらの問題の多くは、ブログやフォーラムで頻繁に言及されています。すべてが正常に機能しているかどうかを確認してください。



今のところすべてです。 記事の最後に変更点の完全なリストがあります。



以下のリンクからアセンブリをダウンロードできます。



  • Windows 32ビット
  • Windows 64ビット （実験的）
  • Mac
  • Linux RPM 32ビット
  • Linux RPM 64ビット
  • Linux DEB 32ビット
  • Linux DEB 64ビット


変更の完全なリスト：



  • VB-4107サムネイルの欠落：起動時にサムネイルをトリミングし、30日ごとにデータベースを圧縮します
  • VB-10562 [Linux] libudevおよびnssの不要な互換性シンボリックリンクをpostinstallおよびpostremoveスクリプトから削除
  • VB-10572 [Linux] LSBの不要な依存関係を削除
  • VB-11006「破棄」するとタブが壊れる（黒いタブ、タブが更新されない）
  • VB-11408 [リグレッション] Netflixは、ビデオを表示しようとすると、サポートされていないページにリダイレクトします
  • VB-11406 [リグレッション] web.whatsapp.comが動作を停止しました：最初の実行修正（さらに作業が必要です）
  • VB-10837 [Mac] 1つのショートカットで2つのタブを閉じることができる
  • VB-11282 UAストリングの一部が除去された
  • VB-10776 [リグレッション]最初のspatnavターゲットがハイライトされていません
  • VB-11358 [リグレッション] NPAPIが無効になっています：WindowsおよびMacで再度有効になりました
  • VB-11316逃げる音声インジケータをキャッチする：LinuxおよびWindowsでのミュートタブアイコンの配置を適切に処理する


毎週のビルドはテストであるため、重要なデータをバックアップすることを忘れないでください。 前のアドレスで見つかったエラーを報告します 。
