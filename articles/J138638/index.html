BlackBerry PlayBook OS 2.0が利用可能に

今日、番号2.0.0.7971のPlayBookの2番目のファームウェアのリリースがリリースされました。 これに関連して、Research In Motion（RIM）のプレスリリースの翻訳を読むことが提案されています。



オンタリオ州ウォータールー-Research In Motion（RIM）（NASDAQ：RIMM; TSX：RIM）は、BlackBerry®PlayBook™OS 2.0が今日ダウンロード可能になったことを発表しました（ すでに利用可能です ）。 BlackBerry PlayBook OS 2.0には高度な機能が搭載されており、BlackBerry PlayBookを仕事中や娯楽のために1日中新しい方法で使用できます。



「BlackBerry PlayBookタブレットのWebブラウザー、マルチメディア、マルチタスク機能に基づいて、新しいBlackBerry PlayBook OS 2.0は、コミュニケーションと生産性の多くの新しい改善を導入し、アプリケーションとコンテンツのサポートを拡張します」とDavid J. Smith氏、RIM、モバイルテクノロジー担当副社長



BlackBerry PlayBook OS 2.0の新機能は次のとおりです。



機能的な単一のメールボックスを備えた組み込みの電子メールクライアント。 BlackBerry PlayBook OS 2.0を使用すると、Facebook、LinkedIn、Twitterからのメッセージだけでなく、個人メールやビジネスメールを含むすべてのメッセージを1か所にまとめる単一のメールボックスを使用できます。



カレンダーおよび連絡先アプリケーションとのソーシャル統合。 組み込みのカレンダーは、ソーシャルネットワークからの情報を使用し、必要なときにいつでもどこでも利用できるようにします。 連絡先カードには、Facebook、Twitter、LinkedInからの更新情報が動的に入力され、統一された連絡先フィールドが作成されます。



更新されたアプリケーションBlackBerry Bridge。 BlackBerry Bridgeは、タブレットの大きなディスプレイにコンテンツを表示するために、BlackBerry PlayBookとBlackBerryスマートフォンのメインアプリケーション（BBM、メール、連絡先、カレンダー、ブラウザーを含む）間のBluetooth接続を提供する独自のアプリケーションです。 BlackBerry PlayBook OS 2.0を使用すると、BlackBerryスマートフォンで開いているドキュメント、Webページ、メール、写真にアクセスし、BlackBerry PlayBookに表示して表示と編集を最適化することがこれまで以上に簡単かつ迅速になります。 更新されたBlackBerry Bridgeアプリは、BlackBerryスマートフォンをBlackBerry PlayBookのワイヤレスキーボードおよびマウスとして使用できる新しいリモートコントロール機能も提供します。



モバイル生産性の向上。 更新されたドキュメント編集機能、新しいPrint To Goアプリ、およびBlackBerry Balanceを使用した企業データの制御と管理性の向上により、BlackBerry PlayBookをさらに活用できます。 さらに、自動修正と次の単語のインテリジェントなヒントを備えた更新された仮想キーボードが学習し、より正確で高速なタイピングのためのタイピング方法を学習します。



新しいアプリケーションとコンテンツ。 今日、BlackBerry App Worldに表示される数千の新しいアプリケーション（BlackBerry PlayBookでも実行される多くのAndroidアプリケーションを含む）。 さらに、BlackBerry Video Store 1が今日から始まります。 BlackBerry PlayBook OS 2.0では、高度なWebブラウジングも利用できます。



BlackBerry PlayBook OS 2.0のリリースに合わせて、RIMはBlackBerry Mobile Fusionの最初のバージョンを導入します。これには、企業でのBlackBerry PlayBookタブレットとBlackBerry 2スマートフォンの管理のサポートが含まれます。 （iOSおよびAndroidモバイルデバイス管理機能を備えた）BlackBerry Mobile Fusionのフルリリースは、2012年3月下旬に予定されています。 BlackBerry Mobile Fusionの詳細については、 www.blackberry.com / mobilefusionにアクセスしてください 。



在庫状況



BlackBerry PlayBook OS 2.0アップデートは、すべてのBlackBerry PlayBookタブレットで無料でダウンロードできます。





1 BlackBerry Video Storeは、米国で最初に利用可能になります。 他の国からのサポートは年間を通じて期待されています。



2 BlackBerry Mobile Fusion Studioは、単一のユニバーサルコンソールを介してBlackBerryスマートフォンを制御するために使用でき、BlackBerry Enterprise Server 5.0.3以降でアクティブ化されたデバイスがサポートされます。
