Microsoft Azureの認定を受けた独立開発者向けソリューションのロシア語カタログの発表

同僚の皆さん、Azureクラウドプラットフォーム用に認定された独立系開発者（ISV） のソリューションのロシア語カタログに注目していただければ幸いです。









このリソースは、Azureクラウドでの使用が認定され、Azure Marketplaceで公開された、すでに人気があり、まったく新しく、最近、最近公開されたソリューションをホストします。 このポータルは、独立した開発者のソリューションを促進する場所として使用し、新規ユーザーの獲得を支援します。



これらの決定事項、Azure Marketplaceストアとその入手方法については、以下をご覧ください。



Azureマーケットプレイス



Azure Marketplaceの目的と、開発者とIT専門家が複雑なサードパーティソリューションを展開するためにストアを使用することの詳細、およびソリューションを配布するための新しい市場に参入したい独立系ソフトウェア開発者（ISV）について詳しく説明しました。



Azure Marketplaceは、ソリューション、サービス、さまざまな複雑さの環境イメージのカタログであり、Azureプラットフォームでの迅速な自動展開の準備ができています。



Azure Marketplaceの主なアイデアは、独立した開発者に世界中のソリューションの販売チャネルを提供することです。Azureのお客様は、独自のインフラストラクチャに機能ソリューションを簡単に展開できます。



多くのISVは、Red Hat、Oracle、CoreOS、MariaDB、Elasticsearchなど、Azure向けのソリューションを既に認定し、Azure Marketplaceに配置しています。 この記事でいくつかの詳細を確認できます。ソリューションの完全なリストは、 Azure Marketplaceの公式Webサイトでいつでも入手できます。









Azure Marketplaceに最近追加されたものの1つは、Mail.ruによって開発されているオープンなTarantoolデータベースです。 この使用方法については、この記事のストアを参照してください 。



販売チャネルとしてのAzure Marketplace



Azure Marketplaceは、開発者やITプロフェッショナルがAzureクラウド内のさまざまなソリューションに基づいてソリューションを展開し、環境を整理しやすくする強力なツールであると考えました。 一方、Azure Marketplaceは、独立系ソフトウェア開発者向けの強力な販売チャネルです。 今日、多くの国の企業からの多数の新しいソリューションが毎月ストアに追加されています。 これらのソリューションは、マイクロソフトからの多様なマーケティングサポートを受け、ターゲットオーディエンスへの追加プロモーションを受けます。 このようなプロモーションの例は、 Microsoft Partner Appsチャネルです。



世界中の数百を超える市場に存在するAzure Markertplaceチャンネルは、ソフトウェア開発者が世界の舞台に参入し、法的な問題や税金の問題を抱えることなく、世界中のどこでも販売を開始するための素晴らしい方法です。



ロシアのAzure Marketplace



現在、30か国の開発者がAzure Marketplceでソリューションを公開できます。 これまでのところ、ロシアはそのような国のリストに含まれておらず、ロシアの開発者向けの出版物は限られています。 ただし、グローバルに存在するロシアの企業数社は、Azure Marketplaceでソリューションを既に公開しています。 その中には、 Kaspersky Labs 、 Veeam 、 Jelastic 、 NGINX 、 Mail.ruがあります。









ロシアからの決定を公開する可能性は、近い将来追加されます。 それまでの間、ロシアの開発者とITプロフェッショナルは、Azureマーケットプレイスとサードパーティソリューションを自由に使用して、Azureクラウドに基づいた興味深い新しいプロジェクトを作成できます。



新しいロシア語ポータル



新しいポータルは、Azure Marketplaceからロシア語圏の聴衆に選択されたソリューションを促進するように設計されています。



開発者とITプロフェッショナルは、このポータルを使用して、Azure Marketplaceのソリューションに精通し、タスクに適したソリューションを見つけることができます。



ポータルでは、次のセクションを提供しています。

  • 選択したソリューションの範囲
  • カタログを通じて推進されているソリューションの完全なリスト
  • 独立した開発者向けの世界的な販売チャネルとしてのAzure Marketplaceの説明
  • ロシアのAzure Marketplaceソリューション
  • 役立つリソースへのリンク：ビデオ、スライド、記事


ポータルへようこそhttp://events.techdays.ru/Azure-Market-Place/ ！
