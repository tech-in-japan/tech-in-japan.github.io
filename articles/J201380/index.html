PocketBook Color Lux-カラーEインクリーダー

少し前のコメントで、私はPocketBook Color Lux電子書籍を「レビュー」することを約束しました。 多くのテキスト、24枚の写真、カラーのe-Inkスクリーンを備えた美しい8インチがあなたを待っています！









デザインと外観



本作りの新しいものを思い付くのは難しいです。 実際、すべてのバリエーションは、いくつかのボタンとインターフェイスコネクタの配置にあります（ヘッドフォンの場合は、本でオーディオを再生できますが、USBケーブルの場合）。



デザインPocketBook Color Luxには超自然的なものは何もありません。角が丸い長方形のリーダーで、「右側」（直立した位置で見た場合）の部分が少し厚くなっています（そこにバッテリーが隠れています）。 背面パネルは手触りが快適で、手にすべらないソフトタッチです。 フロント-通常のマットプラスチック。







画面の上に、バックライトの明るさのタッチスライダーがあります。右側、「厚い」フレーム-4つのボタン-「ホーム」、「進む」、「戻る」、「メニュー」（今回は触れないで、柔らかいプラスチックを少し絞る必要があります）。 右フレームは左よりわずかに広いので、本を片手で持っておくと便利です。







「厚い」側には、microUSBコネクタとmicroSDHCカードのスロットを覆うカーテンがあります（最大32 GBをサポート）。 シャッターは磁化されており、ケーブルが切断されるとそれ自体が「バタバタ」します。意図的にそれを引き剥がしたり紛失したりすることしかできません。







電源/ロック画面ボタンと3.5 mmジャックは右上にあります。 実際、これですべてです。フロントパネルの残りのスペースは、カラーeInkディスプレイで占められています。







唯一の重要なマイナス-厚いフレームのため、ジャケットまたはジャケットの内側のポケットにあるこの「8インチ」は置くことができませんが、ほとんどの7-8インチのタブレットは問題なく収まります。 さて、大丈夫、バッグ/バックパックでは、それはよりそのままになります。



ディスプレイ



実際、電子書籍の最も重要な部分。 Color Luxには、LEDバックライトを備えた「電子インク」の技術を使用して作成されたカラーディスプレイがあります。 手紙の照明の「オフ」状態では、もちろん、確認できますが、外部照明の非常に明るい光源が必要になります。 他のほとんどの場合、最低限の報告レベルの1つで十分です-それにより、「紙」は読みやすい外観を獲得し、ブックメニューへのアクセスを必要としない便利な明るさ調整は、いつでも本をあなたの目に快適にすることができる素晴らしいボーナスです。



スクリーンモジュール自体の特性は、過度の唾液分泌を引き起こしません：800 x 600ピクセルの解像度、e-Ink Tritonテクノロジー（第2世代）、および10：1のコントラスト比。 特に現代のe-Inkパールディスプレイの背景に対しては特に印象的ではありませんが、カラーであり、バックライトがあり、これが問題を根本的に変えます！







はい、彩度は噴水ではなく、素材は明るい白ではなく、むしろ明るいグレーです。ホワイトバランスはクールな色になりますが、同時に-4096シェード（グレーとは誰ですか？いたずら...）、そのような本を使用する方がはるかに簡単です特に本から読むだけでなく、特別な文学から読む場合は、より快適です。







技術文書、「色付き」コード付きテキスト、マルチカラースキーム、およびグラフは、便利で読みやすい形式のままです。 桃とオレンジを区別するのは難しいかもしれませんが、8〜10の「異なる」グレーシェードを使用してグラフやチャートのロジックをトレースするよりも、特に元の色の彩度と輝度のために）ほぼ同じになることさえあります。







バックライト-均一でエッジのあるLEDラインは、製造業者によって本の下部に「隠されています」。 どんな条件でも彼女と仕事をするのは快適です。 明るさの調整は、「暗闇で見えるようにわずかにハイライトする」から「明るい太陽を叫ぶ」まで、非常に広い範囲で行われます。







タッチスクリーン



静電容量式タッチセンサーを使用すると、書籍を簡単にナビゲートして、コア以外の機能（オーディオプレーヤー、ブラウザー、ハンカチソリティアなど）を操作できますが、それに慣れるには、スマートフォン、タブレットの超高速インターフェイスに慣れている脳、適切な反応を期待し、このデバイスから、そして彼は（すべての本のように）、それにもかかわらず、いくらかの思慮深さに内在しています。 しばらくすると、あなた（この本の所有者になった場合）が必要なを発症し、センサーの操作が正常に戻ります。



読み取り：形式、機能、機能、バグ



リーダーでサポートされている形式のリストは良好です：PDF、PDF（DRM）、EPUB、EPUB（DRM）、DJVU、FB2、FB2.ZIP、DOC、DOCX、RTF、PRC、TCR、TXT、CHM、HTM、HTML。



一方で、すべてがうまくいきます。ここでは、「ユーザー-消費者」ブック形式、PDF / DJVU / EPUBなどの複雑なグラフィックスのサポート、および一般的な民主主義の両方があります。一方、カラー画面がある場合は、XLS表示を行うことができます、およびPPT形式は非常に重要です。 まあ、流fluentなグーグルは、コミックとマンガ（日本のバラエティ）にはまだ特別な形式があることを示しました。 幸いなことに、PocketBook（インターネットと知り合いの情報によると）は積極的にファームウェアを削減し、デバイスの動作を改善し、一般的にはまともな会社のように振る舞います。







読み取りプログラム自体は、形式に応じて機能を変更します。 テキストファイルは、FBReader'e、テキストおよびグラフィック素材-特別なビューアAdobeViewerで表示されます。 数式やエンコードに問題はありません;色が少し損なわれることを除いて、画像が表示されます。



読書モードのメニューから、すべてが好きになるように本の表示を調整できる設定が利用可能です。







ご覧のとおり、ウィンドウのタイトルから判断すると、一般的なFBReaderユーティリティが使用されています。



PDFおよびその他の「複雑な」形式の場合、構成可能なパラメーターのリストはやや小さくなりますが、主なパラメーターは次のとおりです。

テキストを操作する



文書の標準的な読みに加えて、この本ではメモを作成したり、テキストの一部をコピーしたり、スクリーンショットを撮ったり、直接描画したりできます。 5〜7年前の学生時代にそういうことがあったら、喜んでいたでしょう。







ただし、残念ながら「スキャンされた」PDFはカットできません。







完全な幸福のために欠けている唯一のものは、結果の「クリッピング」を便利にカタログ化し、それらを単一のドキュメントに保存する機能です。これは、レビューの準備と完成した作品の作業に最適なツールです。 それまでの間、すべてのクリッピングが保存される大きな「メモ」ボタンのみがあります。
