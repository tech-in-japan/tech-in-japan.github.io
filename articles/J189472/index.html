Opera Developer Channelの最初のリリース



前に書いたように 、OperaはOpera、Opera Next、Opera Developerの3つのチャネルで構成される加速リリースサイクルに切り替えます。 最初の2つのチャネルがすでにあなたと一緒にいる場合、3番目の、最も実験的なチャネルを、多くの興味深いものが蓄積されるまで保持しました。 だから、会いましょう：Opera Developer for Mac and Windows 。



サイトまたは拡張機能を開発している場合、Opera Developerのこのアセンブリをインストールすることをお勧めします：頻繁に更新され、新しいブラウザー機能とWebテクノロジーが最初に表示される場所になります。その結果、Operaブラウザーの安定したバージョンになります。



Opera NextとOpera Developerの違いは何ですか？ Nextは、次の安定バージョンに登場する機能を示しています。Developerは、Nextに進んでさらに実験を続けるか、または実験的なままにする実験の場所です。



根拠のないことではありませんが、Opera NextとOpera Developerの現在のバージョンの違いは次のとおりです



  • Opera Nextの現在のバージョンは16.0.1196.29で、Chromium 29で動作します。OperaDeveloperはバージョン17.0.1224.1でChromium 30で動作します。
  • このバージョンのOpera Developerでは、Opera 12からいくつかの機能が返されました。マウスボタンを押したままタブとジェスチャーを固定します（設定でジェスチャーを有効にすることを忘れないでください）。
  • 拡張機能に新しいAPIが追加されました：ブックマーク、コマンド、アドレスバー、webNavigation-動作するためのドキュメント。
  • 初めて、「起動時」ブロックが設定に表示され、Operaの動作を選択できます。前の場所から続行するか、開始ページを開くか、追加で設定できるページのグループを開きます。
  • 独自の検索プロバイダーを作成し、それらにキーワードを設定できるようになりました
