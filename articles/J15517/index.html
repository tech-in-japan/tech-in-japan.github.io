メディア-4番目の力。 責任についてはどうですか？

おそらくこのトピックを読んで、あなたは私が妄想だと言うでしょう。 私は議論しません。 ただし、最初にまず最初に。



今日、私は新聞Vzglyadのウェブサイトで次の記事を読みました： http : //vz.ru/society/2007/10/28/120502.html 地球上の資源がどんどん少なくなっているという事実に関するこの記事は、誰にとっても十分ではありません。 この記事は、The Timesで公開された別の記事に基づいています。 最も興味深いのは、オリジナルへのリンクを提供しなかったことです。 何が私を守った。



最初に興味を持って読みました。 それから彼は奇妙なことが書かれていると考え始めた。



たとえば、資源の観点から、地球が「無帰還のポイント」に達した正確な日付をどのように計算できますか？ これを行うには、少なくとも、1日に消費されるリソースの数と、地球上に残っているリソースの数を正確に知る必要があります。 それから私は外国の組織の名前の翻訳を見ました、そして、どこかにキャッチがあるのではないかと疑っていました。 自分で判断してください。 引用：

少なくとも、2つの権威ある組織の専門家はそう考えています。AmericanGlobal Trace NetworkとBritish New Economy Fundです。


グローバルトレースネットワークを「グローバルフットプリントネットワーク」としてどのように翻訳できたかは、私にはまったく理解できません。 さらに、これが深刻な出版物でどのように印刷されるかは明らかではありません。

この名前をどのように翻訳するのが最善かを考えて、Googleに尋ねることにしました。 アドレスバーに「Global Trace Network」と入力すると、非常に予期しない結果が得られました。 驚いたことに、グローバルトレースネットワークについての言及は、 ニュースサイトのみにあります 。 他にはありません。 忠実性のために、 MSNとYahooでチェックインしました。 なし。 そして、これはただ一つのことを意味します。 そのような組織は存在しません。 また、The Timesには、この組織に関するメモはありませんでした。

それから、2番目の組織をチェックアウトすることにしました。 今回、そのような組織が見つかりました。 しかし、ここではすべてがそれほど単純ではありませんでした。 まず、組織はイギリスではなく、アメリカ人でした。 自分で判断してください 。 次に、投資会社であることが判明しました。 つまり、専門家はいません。 彼らは投資家からお金を取り、他の会社の証券に投資します。

両社について、記事には誤った情報が含まれていることがわかりました。



その後、私はまだタイムズでオリジナルの記事を見つけることにしました。 タイムズのウェブサイトに行き、環境保護に関する最近のメモを探しましたが、ここで成功が待っていました。

このメモは、新聞Vzglyadに記事を書くための基礎として役立ちました。http ：//www.timesonline.co.uk/tol/news/uk/science/article2739926.ece



The Timesの記事を読んだ後、新聞Vzglyadの記事の最初の部分のみがThe Timesの記事に基づいていることがわかりました。 残りの情報がどこから来るのかは明確ではありません。 記事の著者であるGennady Nechaevは、これらのデータが公開されているとされる特定の国連報告書を参照しています。



私はそこで止まらないことに決め、前述のレポートを探すために国連のウェブサイトに行きました。 しかし、そこには失望しました。 実際、この本は本の形で発行されており、80ドルで購入できます。 必要に応じて、 ここで購入できます 。



記事の著者がまだレポートを読み、記事が実際のデータを提供していると信じたい。 ただし、この場合、情報源を引用することは悪くありません。 個人的に、私は組織の名前で奇妙なことをした後、この記事を信用していません。



私は新聞「Vzglyad」の編集者に手紙を送りました。なぜ記事の事実が間違っているのか、また組織の名前がそれほど専門的に翻訳されていないのかを説明する質問です。 答えがそうかどうか見てみましょう。



最も興味深いのは、多くのニュースサイトが、この記事を、その根底にある事実を確認することさえせずに印刷したことです。 このようなアクションは、イエロープレスに適している可能性が高くなります。 私は常にVzglyadをより深刻な出版物と考えてきました。 これは、ネットワーク上であらゆる種類の誤った情報を実行することがいかに簡単かを示唆しています。
