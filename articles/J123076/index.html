UserLook-あなたのサイトで今何が起こっているかを調べてください。

UserLook-サイトの状況をリアルタイムで監視する機会を提供します。







以下、猫の下で、スタートアップに関する興味深い事実。





このプロジェクトを立ち上げることにしたのはなぜですか？



約1年、私たちは同様のアメリカのサービスを使用しましたが、その中にはいくつかの利点がありませんでした。 特に、マルチドメインサイトを監視しました。 彼らのサービスでは、すべてのサブドメインは、リクエストの最初のサブドメインとみなされていました。 そして、そのようなページはすべてこのサブドメインの下で強調表示されました。 これは訪問者の本当の姿を大きく歪めました。 多くの人の中で、私たちは彼らにこのチップを要求しましたが、1年が経過し、状況は変わりませんでした。 それから私たちは決めました-サービスをさらに良くしてみませんか？



実装にどれくらい時間がかかりましたか？



このサービスは、35 + 7 + 14 日間のコーディングとテストの夜の3つのアプローチで実装されました。

当初、すべてを35日間で立ち上げる計画がありましたが、失敗しました。最初のアプローチと2番目のアプローチの間に、プロジェクトが防御されたときに約3か月の間隔がありました。



どれくらいお金を使いましたか？



すべては外部からの投資を引き付けることなく、私たち自身の費用で行われました。 主な費用：サーバー、ドメイン、費用の支払いは、他のプロジェクトからの利益で賄われています。



サービスは何に実装されていますか？



このサービスは、Django + Nginx + MongoDb + Postgresqlの束に実装されています



いくらですか？



支払い受付サービスが接続されるまで、サービスは誰でも無料です。



現在の機能：



  • リアルタイムで統計を表示する
  • 保存された期間、いつでもサイト上にあった状況を表示する機能
  • Twitterで選択したキーワードのプロジェクトへの言及を追跡する
  • 訪問者がどの国から来ているかを表示する機能
  • また、「Popular Pages」モジュールを使用して、訪問者の数とページの場所を確認することもできます。
  • 参照元の追跡により、サイトへの参照元を特定できます。
  • 人気のあるサービスまたはOpenIdを介した迅速な登録に貴重な時間がかかる
  • 全自動データ更新プロセス、ページを更新する必要はありません
  • タイムゾーンを設定すると、現地時間でデータを表示できます
  • マルチドメインサイトの監視のサポート
  • ActiveX、Flashを使用しないインターフェイス、iPadのサポート




近い将来、次の方向でサービスを開発します。



  • ユーザーとのライブチャットを有効にするアイデアを持っている
  • サイトでの平均ページ読み込み時間を表示するモジュール
  • トラフィックのピークやWebサイトの動作不能など、サイトで発生するイベントに関する通知システム




Habrausersからの建設的なコメントと開発アイデアを歓迎します。feedback.userlook.comサポートフォーラムでお待ちしています
