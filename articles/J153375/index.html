Java Oscarは新しい所有者を見つけました

Duke's Choice Award 2012の受賞者は、Javaの世界最大の年次イベントであるサンフランシスコでのJavaOne会議の前夜にOracleチームによって発表されました。 Jelasticプロジェクトはテクノロジーリーダーにノミネートされました。 Hadoop Project 、 Parleys.com 、 Liquid Roboticsなどの他の受賞者と表彰台を共有できたことは非常に良かったです。 これは私たちの勝利だけでなく、すべてのhabrayuzerの勝利でもあります。なぜなら、私たちのプロジェクトはHabréで生まれ、あなたの積極的なフィードバックのおかげで常に発展しているからです。 ご支援ありがとうございました！







Duke's Choice Awardsは、Javaテクノロジーの世界における革新を称えるものです。 このような賞は、最も革新的で高度で野心的なJavaプロジェクトによってのみ獲得できます。 これはコミュニティ内での10年の伝統であり、勝利はもちろんJavaの世界での名誉です。 かつて、勝者はMotorola、Terracotta、tlassian、Apache Software Foundationなどの有名な企業でした。 この賞の主な基準はイノベーションです。この場合、 Jelastic Java Cloudクラウドホスティングプラットフォームの開発です。



「Javaテクノロジーからオスカーを受賞したことは大きな名誉です-デュークスチョイスアワード」とJelasticの創設者兼CEOであるRuslan Sinitsky氏は述べています。 「わずか1年で、当社はクラウドコンピューティングのゲームの流れを変えました。 ユーザーに自動スケーリング、開発ツールとの便利な統合、クラウドでプロジェクトを構築する機能などを提供します。 同時に、Jelasticホスティングは世界中で利用可能です：ロシア、アメリカ、ドイツ、イギリス、日本。 また、Parallels、NGINX、GlassFish、MariaDB、Runa Capital、Almaz Capitalの各ファンドを含むすべてのパートナーに特別な感謝を表明したいと思います。 来年の計画は、世界中のホスティングパートナーの数を増やし、テクノロジーを改善し、最も重要なこととして、開発者の生活を楽にすることです。」



James Gosling（「Javaの父」）が率いる別の受賞プロジェクトであるLiquid RoboticsもJelasticプラットフォームを開発に積極的に使用していることを実感するのはさらに楽しいことです。 詳細については、 こちらをご覧ください 。



チーム全体が感じるこのような喜びと喜びをすべて記述することは困難です。 私たちにとって、これはJavaコミュニティ全体からの大きな成功と認識です。つまり、私たちは正しい方向に向かっています。



再びすべてのユーザーと読者に感謝します！ 公正な批判、ヒントとアイデアをありがとう！ あなたがいなければ、この勝利は不可能だっただろう。
