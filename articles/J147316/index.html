Torque-BitTorrentのJavaScriptインターフェイス

昨日、BitTorrentプロトコルの開発者は、ブラウザーにP2Pクライアントの機能を組み込むように設計された新しいツールを発表しました。



オープンなBitTorrent Torqueプロジェクトはアルファ版であり、Torqueプラグインの機能にアクセスするためのJavaScriptライブラリのセットです（ OneClick for Chromeが利用可能になりました）。 ダウンロードしたプラグインを使用すると、ビデオストリーミングコンテンツ、デバイスへのアクセス、トレントファイルの操作、メディアコンテンツのビデオおよびオーディオエンコーディングなどのファイルを実際にダウンロードするだけでなく、多くの機能を実装できます（ほとんどはプロジェクトページでまだ利用できません）。



Torque用のオブジェクトの作成は簡単に見えます。 



var btapp = new Btapp(); btapp.connect();




新しいトレントファイルの追加： 



 btapp.on('add:add', function(add) { add.torrent('http://featuredcontent.utorrent.com/torrents/Kumare-BitTorrent.torrent'); });




ダウンロードしたトレントの名前をコンソールに出力します。 

 btapp.on('add:torrent', function(torrent_list) { torrent_list.each(function(torrent) { console.log(torrent.get('properties').get('name')); }); });




興味深い機能の中でも、 BitTorrent Webに注目する必要があります。BitTorrentWebはまだ近日公開予定です。わずかな説明によれば、デスクトップクライアントの機能を失うことなく、Webを介してトレントを管理できます。
