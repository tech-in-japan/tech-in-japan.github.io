Mac OS XでMacPortsをインストールする

MacPortsは、ターミナルシステムを介してダウンロード、コンパイル、フリーソフトウェア、さまざまなライブラリのインストール、MacOS XとDarwinの更新が非常に簡単なパッケージシステムです。



上記のすべてのアクションを実行するには、MacPortsがある場合、ターミナルウィンドウに入力するだけです



port install packagename







ここで、「packagename」はインストールするパッケージの名前です。



しかし、すべてがあなたにとって同じように美しく機能するためには、このMacPortsをインストールする必要があります。



詳細な手順にもかかわらず、このソフトウェア製品をインストールする際にいくつかの困難に直面し、読んだ記事を一般化するよう促されました。



ソースからMacPortsをインストールする場合は、この記事は役に立ちません。 MacPortsのインストールを読んでください。







Xcode





驚くことではありませんが、最初はMacPortsという名前のインストールパッケージをダウンロードする必要はありませんが、MacPortsがインストールされないXcodeをダウンロードする必要があります。



新しいMac OS Xがあれば、それで十分です。

*ご使用のオペレーティングシステムのバージョンに適したXcodeインストールパッケージをダウンロードします（ダウンロードするには、 Apple Developer Connectionに無料で登録する必要があります ）

* Xcode Toolsのインストールを開始し、すべてのデフォルト設定を使用してこのアプリケーションをインストールします




X11





この段落と次の段落では、特に注意することをお勧めします。 「X」の設定が間違っていたため、すべてが飛び去りました。



* '' /Applications/Utilities/X11.app ''から開始（ '' / Programs / Utilities / X11.app ''）

*メニュー ''プログラム ''でアイテム ''構成 ''を選択します

*アイテム「ターミナル」をダブルクリックし、「xterm」を「xterm -ls」に変更します




Macports





* MacPorts-1.xxdmgの最新バージョンをダウンロードします

* MacPorts-1.xxpkgをダブルクリック

*デフォルト設定でインストール




シェル





ターミナルを起動します。 入って



ls -a







そして、「.profile」と呼ばれるファイルの結果を見てください



存在しない場合は、作成してみてください。



nano .profile







ターミナルに、ファイル編集ウィンドウが表示されました。 行を書きます



export PATH=/opt/local/bin:/opt/local/sbin:$PATH







注： Mac OS X 10.4 Tiger以前を使用している場合は、ファイルに次の行も追加する必要があります（Mac OS X 10.5 Leopard以降を使用している場合は追加しないでください）。



export DISPLAY=:0.0







ctrl + Oを押してファイルを保存します。



既にそのようなファイルがある場合は、同じことを行い、ファイルに上記の行を追加します。



すべての変更を有効にするには、ターミナルを再起動する必要があります。



すべてがうまくいった場合、ターミナルコマンドを入力すると



port







次のようなものが得られるはずです。



MacPorts 1.600

Entering interactive mode... ("help" for help, "quit" to quit)

[Users/username] >







最初の使用





MacPortsをアップグレードしてみましょう。 これを行うには、ターミナルを実行して入力します



sudo port selfupdate







または、実行時に追加情報を表示するには、-dを追加します



sudo port -d selfupdate







このコマンドはあまり速く実行されないことに注意してください。 したがって、我慢して、余分なボタンを押さないでください。
