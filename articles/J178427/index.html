uptodate.js-時間要素を自動更新するためのライブラリ

サイトで「5分前」、「半時間前」などのデザインを使用する頻度はどれくらいですか？ しかし、ユーザーがタブを開いて数時間それを忘れた場合はどうでしょうか？ 「5分前」の関連性は明らかに失われています。

uptodate.jsが役立つのは、この迷惑な誤解を修正するためです。











使用する





  1. コンテンツにタイムスタンプを準備します。

    デフォルトでは、uptodateはタイムスタンプ（エポックからのミリ秒数（1970年1月1970日00:00:00 UTC））を含むデータ時間属性から値を取得します

    例えば 

    Posted: <span class="uptodate" data-time="1367180233904"> 4 minutes ago <span>


  2. ライブラリをデプロイします。

    必要なのは、単一のuptodate（）メソッドを呼び出すことだけです

    オプションのパラメータをコールに転送できます。そのリストはオフサイトで利用可能です

    例： 

      <script src="js/uptodate.js"> <script type="text/javascript"> window.onload = function() { uptodate(); }; </script>
