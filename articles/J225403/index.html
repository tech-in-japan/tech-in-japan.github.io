John Carmackの最初のゲームのオープンソースコード

いくつかの楽しい驚きは、単に予測することは不可能です。



非常に最初の3Dゲームの1つ、そして有名なゲームプログラマーの原作のソースコードを見たいですか？ お願い！ Catacomb 3-DやHovertank 3Dなどのゲームのソースコードを提供します。 ゲームはかつてBorland Turbo PascalおよびBorland C ++ 2.0で記述されていました。 今日GPLの下でレイアウトされました。



CarmackがHovertank 3Dに手を詰めた（これまでで初めての一人称シューティングゲームと考えられていた）。 その後、Catacomb 3D：The Descentが登場しました（ソフトディスクによってまだリリースされていますが、id Softwareによって既に開発されています）。 どちらのゲームも1991年にリリースされましたが、あまり成功していないようです。 しかし、1992年、Wolfenstein 3Dは雷鳴を放ち、シューティングゲームの世界を永遠に変えました。 どうやら、ソースコードのオープンは、このシリーズの新しい（そして驚くほど良い）部分のリリースと一致するタイミングです。 これらのゲームの歴史について詳しく知りたい場合は、私の翻訳をご覧ください。



屋根の事実 ：CatacombとDoom 3は同じコードを使用します 。



ソースコードを取得します： github 。
