継続的インテグレーションは必要ないと思う開発者は何人いますか？

12月の終わりに、クラウドプロバイダーのDigitalOceanは、サーバーオペレーティングシステムとして主に（89％）を使用して、2500人以上の開発者からデータを収集した2回目の調査Currentsの結果を公開しました（詳細は以下を参照してください） 。 継続的な統合とアプリケーション配信（CI / CD）に関連する問題について興味深い統計が得られました。







ご覧のとおり、回答者の半数以上（58％）がCI / CDを使用していません 。 そのうち、46％がワークフローにCI / CDが必要ないと考えています（つまり、調査参加者の総数の約27％ ）。また、38％はCI / CDを導入する予定です。



CI / CDのソリューションをすでに使用している人の中で、製品ごとの配布は次のとおりです。



  1. ジェンキンス-44％;
  2. GitLab-39％;
  3. トラビス-26％;
  4. CircleCI-12％;
  5. TeamCity-8％;
  6. 竹-6％。
