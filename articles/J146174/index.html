Windows Phone用にインストールしたプログラムの小さなセット

Windows Phoneでかなり長い間使用しているプログラムのリストを公開します。 私は多くの時間を費やして、多くの類似したプログラムの中から、設計ガイドラインに沿ったもの、より安定したもの、同時に非常に機能的なものを選択しました。 いくつかの類似したプログラムのそれぞれに独自の「キラー機能」がある場合があり、この場合はすべてリストされます。

もちろん、いくつかの真珠を見逃す可能性があります。この場合、コメント内のリンクを確認し、リストに含めることを非常に嬉しく思います。

プログラムに関するコメントが目を痛める可能性があります。 もしそうなら、私はそれを修正します。

また、いくつかのプログラムにはお金がかかることも付け加えますが、これらのすべての場合、価格は私にとって十分であるように見えます。

PSスクリーンショットを提供していないことを申し訳なく思っています。そのようなシートが出てくるとは思っていませんでした。ただ共有したかったので、初心者に役立つと思います。
