Chromeの拡張機能がFirefoxで機能するようになりました





Mozilla FirefoxオープンソースWebブラウザーには、別のWebブラウザーの拡張機能であるGoogle Chromeをインストールできるアドオンがあります。



Firefoxのアドオンの公式ディレクトリで公開されているいわゆる「Google Chrome Extensions Manager」（Google Chrome Extensions Manager）を使用すると、Chrome用に作成されたFirefox拡張機能をインストールおよび削除できます。 Firefoxバージョン3.5から3.7a1preがサポートされており、現在のプロジェクトのステータスは実験的です。



開発者は、現時点でFirefoxでChromeの拡張機能を使用する場合、マニフェストファイル、バックグラウンドのページ、オプションページ、ブラウザーアクション、コンテンツスクリプト、テンプレート検証、HTML5のFirefoxのAPI全体、Chrome APIの一部がサポートされていることに注意します（browserAction、拡張機能、タブ、ウィンドウ）。 現在サポートされていない機能には、ブックマーク、pageAction、NPAPIプラグイン、テーマ、国際化、自動更新があります。



Google Chrome Extensions Managerの詳細とインストールのリンクは、 addons.mozilla.orgで入手できます。



また、3月17日に、Chrome Webブラウザ向けの人気のあるWeb Developer拡張機能の最初のバージョンがリリースされたことにも注意してください。 Firefoxのオリジナルプラグインの作成者は開発中です。 詳細はchrome.google.comで確認できます。
