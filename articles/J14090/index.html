Web 2.0次に来るものは？ -Web 3.0

Web 2.0とは-正確な説明はまだ入手できないと考えられています。 提供したいです。 このため、開発を示すロジックを提案する必要があります-前に何が起こったのか、後に何が起こるのか。



したがって、WebまたはWeb 1.0（インターネット、ページ、ポータル、ユーザー）は、ユーザーはポータルの受動的な読者ですが、すべての情報と作業はページとポータルの所有者によって行われます。



Web 2.0-ユーザーはポータルのコンテンツを管理します-ポータルとページの所有者は、ユーザーの利便性のための条件を作成します。 ユーザーは自分の熱意に基づいて作業します。



Web 3.0-ページおよびポータルの所有者は、ユーザーが単に作業（コンテンツ、トラフィックを作成）するだけでなく、お金を稼ぐための条件を作成します。 つまり 技術的には、ユーザーが商品を実行できる狭いタスクを提供します。 つまり すぐにポータルで販売される高品質で人気のある製品。 したがって、ユーザーは商品を作り、ユーザーは商品を購入し、ユーザーはお金を受け取り、ポータルはその割合を受け取ります。







これらのカテゴリを正確にこのように分割したのはなぜですか。 Web 2.0のWeb 1.0の後で、ポータルのプロパティが変更されたため、ポータルは巨大で求心性になりました。 つまり 多くのユーザーがいる場所、多くのコンテンツがあり、生活が本格的である場所にユーザーが興味を持っている-この品質が、Web 2.0という名前が与えられたポータルの波を決定した



Web 3.0はユーザーを新しいレベルに引き上げます。ユーザーは収入に関心を持っているため、商品の販売に関心があり、そのため高品質です。 これは、ポータルがより多くのコンテンツまたは商品を受信し、品質が高いことを意味します。これにより、追加のユーザーが引き付けられ、一部は顧客になり、一部は商品の生産に含まれます-両方の条件により、ポータルの品質と速度が新しい定性的な指標になります。



さらに、Web 2.0システムのポータルが、広告収入に変換できる大量のトラフィックを獲得した場合にのみ商業化できる場合、Web 3.0システムでは、ポータルは、最初の購入が行われた直後から、ポータルがその割合を受け取り始めてから商業化されました-また、これにより、投資家を待たずに、またはポータルを売りに出さずに、まったく異なる速度で開発することができます。



Web 2.0およびWeb 3.0の（科学的な）定義に戻ります。



Web-2.0は、熱意に基づいて行動する多数のユーザーが関与してコンテンツを作成するための条件を作成するソリューションのセットです。



したがって、Web-3.0は、払い戻し可能なベースで多数のユーザーがコンテンツを作成するための条件が作成されるソリューションの複合体です。 つまり ユーザーは道徳的満足に加えて、重要な報酬を受け取ります。
