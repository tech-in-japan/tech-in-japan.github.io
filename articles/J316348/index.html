3CXは、ミニPCに3CXをプレインストールするインテグレーターを提供します







最新のサービスパッケージのリリース後、3CX V15システムをミニチュアPCにインストールして、単一のハードウェアとソフトウェアの複合体、いわゆるPBXアプライアンスを形成できます。 中小企業向けの安価で信頼性の高いソリューションを入手できます。 会社に3CXをインストールするサーバーがない場合、 無料または有料の3CXがインストールおよび設定された「本」を提供するよりも簡単です（3CX PBXエディションの無料バージョンをインストールした場合でも、 3CXアップデート ）。







他の「箱入り」PBXに対する当社のソリューションの主な利点は、標準のIntelハードウェアプラットフォームと標準のDebian 8オペレーティングシステムを使用することです。







  • 非常に小さいサイズ-システムはオフィスではほとんど見えません
  • 非常に低い消費電力-外部ファンレス電源を使用
  • サイレント操作-ケースにはファンもありません
  • システムは定期的なクリーニングを必要としないか、より少ない頻度で実行できます。
  • 無料の3CX PBXエディションの5人用の統合ビデオ会議システム


選択したプラットフォームに応じて、最大32（およびIntel Coreプロセッサを最大64）使用できる同時通話のPBX用の安価なミニPCを使用できます！







どのミニPCを選択すればよいですか？ さまざまなメーカーが同様のデバイスを数多く製造しています。 もちろん、すべてをテストする機会はありません。 ただし、以下の選択ガイドラインに従うことで十分であり、問​​題は発生しません。







  • Intel 64 / AMD 64アーキテクチャ-Atom Cherry Trail、CeleronまたはCore i3 / i5プロセッサフ​​ァミリ、たとえば：



    • Intel Atom x5-Z8300プロセッサー （モバイルプロセッサー、最大2 GBのRAMのサポート）
    • Intel Atom x5-Z8330プロセッサー （モバイルプロセッサー、最大2 GBのRAMのサポート）
    • Intel Atom x5-Z8350プロセッサー （モバイルプロセッサー、最大2 GBのRAMのサポート）
    • Intel Atom x5-E8000プロセッサー （より強力なシステム向けのBraswellファミリーデスクトッププロセッサー、最大8 GBのRAMのサポート）
    • ここでは、プロセッサ間の違いについて簡単に説明します。


  • 少なくとも2 GBのRAM
  • 少なくとも32 GB SSDドライブ
  • 有線イーサネット
  • Debian 8またはUbuntu 14のサポートは宣言する必要があります


特定のメーカーについて話す場合は、次のリストを作成できます。







  • Cloudsto MiniPC （3CXテスト済み、推奨）
  • Lavaserv （3CXでテスト済み、主に南アフリカで提供）
  • Nexbox J1900 （テスト済み3CX）
  • Nexbox N2830（テスト済み3CX）
  • Rikomagic （3CXテスト済み）、特にLinux PC MK36S LE
  • そしてもちろん、 Intel NUC mini-PCシリーズ



ミニPCのパワーを決定したら（さまざまなサイズのプロジェクトに複数のシステムを提供できます）、無料の3CXバージョンをインストールします。 これは簡単な手順です。特に3CX SP4のリリース後は 、「回答」ファイルをサポートしています。 インストールされたシステムに自動的に適用される構成ファイル （ このオプションは3CX SP4で利用可能になります ）。 このようなファイルを使用すると、標準タスク用にできるだけ早くPBXを構成できます。



それまでの間、選択したミニPCに3CXをインストールすることを検討してください。

3CX用DebianをミニPCにインストールする



3CXをインストールするには、次のものが必要です。

  • ミニPCの適切なパフォーマンス
  • Debian用ISOイメージ3CX
  • 最小1 GB USBフラッシュドライブ
  • ルーファスブータブルフラッシュドライブプログラム


起動可能なUSBフラッシュドライブの準備：









  1. Rufusをダウンロードしてインストールする
  2. フラッシュドライブを挿入し、Rufusを実行します
  3. 記録オプションを設定する



    1. デバイスはフラッシュドライブです
    2. パーティションスキーム-BIOSまたはUEFIのMBRパーティションスキーム
    3. ファイルシステム-FAT32（デフォルト）
    4. クラスターサイズ-4096バイト（デフォルト）
    5. 新しいボリュームラベル- 例：ISOLINUX
    6. クイックフォーマットを有効にする
    7. ISOイメージを使用したブータブルディスクの作成をオンにし、ダウンロードしたISOイメージを指定します。


  4. [ スタート]をクリックし、[ DDイメージモードで書き込む ]オプションを選択して、[ OK ]をクリックします。








3CXのDebianのISOイメージはフラッシュドライブに書き込まれます。 次に、ディスクをミニPCに挿入し、そこから起動します。



ミニPCにオペレーティングシステムがない場合、Debian for 3CXインストールメニューがすぐに表示されます。 OSがミニPCにすでにインストールされている場合は、DelまたはF2を押して、起動デバイスを選択します。 インストール手順自体は、このマニュアルで説明されています 。
