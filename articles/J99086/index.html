OneNoteをiPhone / iPadに配置する方法、またはMobileNoterプロジェクトの歴史

そのため、 MobileNoterの開発を開始してからほぼ1年が経過しました 。

この製品を作成するというアイデアはほぼ自発的に生まれました.MS OneNoteは私たちの仕事を積極的に助けてくれました。私たちの中にはiPhoneを持っている人がいたため、OneNoteのモバイルバージョンがWindows Mobile専用であるという事実は不当な不正であると判断されました。

iOSデバイスでOneNoteを使用したいのは私たちだけではないことがわかりました。 決定されました-OneNote for iPhoneになること！





私たちの前のタスクは非常に簡単でした-最初にページをONにする方法、次に同期を実行する方法を考え出す必要がありました。

その結果、以下が決定されました。製品は、Windows用クライアント（OneNoteからページを取得する）とiOS用アプリケーションの2つの部分で構成され、それらの間の同期はサーバーを介して行われます。 App Storeのアプリケーション自体は無料ですが、同期サービスの料金は3か月で5ドル、1年で15ドルです。



MobileNoter 1.0は次のようになりました。





最初のバージョンでは、OneNoteラップトップのみを表示でき、さらに、同期後にOneNoteの別のラップトップに分類されるクイックノートを作成できました。 多くのユーザーはこの機能さえ気に入っていましたが、OneNote for Windows Mobileとは異なり、ラップトップを編集できないことに非常に失望した人たちがいました。 しかし、切り札もありました。Microsoft製品とは異なり、MobileNoterはページの書式設定を維持し、より多くのコンテンツをサポートしていました。 いずれにせよ、アプリケーションの機能を拡張する必要があることは明らかでした。



また、セキュリティ上の理由から、クラウド同期を使用したくないユーザー層が現れています。 彼らが暗号化を有効にできるという事実は、その結果、誰も彼らのノートを読むことができなかったので、彼らを安心させました。 さらに、サブスクリプションモデルが気に入らない人もいました。彼らはアプリケーションを一度だけ購入することを好みました。 そして、デバイスとコンピューターの間で直接Wi-Fi同期を使用したバージョンを作成することにしました。そのライセンスは1回限りの支払いとして販売されています。



次のステップは、Quick Notesに画像を追加する機能でした。次に、CloudバージョンとWi-Fiバージョンを1つに統合しました。これがMobileNoter 1.3です。







この時代に、悪名高いiPadが発表され、SDKが登場しました。 デバイスのライブを見ることなく、iPad用のMNのバージョンを作成し始めました。 しかし、私たちは一人ではありません。 はい、最初のバージョンはあまり成功しませんでしたが、最初のバージョンでした。 少し後に、新しい美しいインターフェイスを作成し、iPad用MobileNoterの高品質バージョンをリリースしました。 これにより、多数の新規ユーザーを獲得できましたが、多くのユーザーが尋ねました-どうやってラップトップを編集できないのですか？！ 編集機能を備えたMNは、早ければ早いほど良く表示されることが明らかになりました。



そして、我々はそれを作りました！ 初夏に、MobileNoter 2.0はiPhoneとiPadの両方でリリースされました。 新しいページ/セクションを作成することは不可能ですが、既存のページ/セクションを編集できます-テキストの編集、コンテンツの移動など。 OneNoteのもう1つの重要な機能である添付ファイルもサポートされ始めました。 つまり これで、doc / xls / pdfファイルをOneNoteラップトップに入れて、MobileNoterで表示できます。 また、インターフェイスで作業する必要がありました。







そして、これが上記のiPadバージョンです。







現在、バージョン2.0はApp Storeで入手可能な最新版です。 次は？ 約1週間で、手書きをサポートしたMobileNoter 2.1がリリースされます。 つまり 最後に、タッチスクリーンを利用してメモを描くことができます。 次は？ 高度な書式設定オプション、オーディオ録音の追加、ラップトップの管理-新しいページの作成など。 また、ほとんどの場合、Android用MobileNoterをリリースします。
