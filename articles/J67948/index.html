Intel ClassMate PCネットブックレビュー-パート1

Intel Classmate PC-耐衝撃性ハウジングで作られたコンパクトなグレーブルーのマシン。 これは学童専用の第3世代のコンピューターです。新しいCMPC（Classmate PC）ネットブックは、通常のネットブック構成に加えて、持ち運びに便利なハンドル、ビデオカメラ、ペンのタッチに反応する回転式ディスプレイを備えています。 これらは、「1人の学生：1台のコンピューター」プログラムで学童に行くコンピューターです。これは、ほとんどすべての科目を簡単かつ明確に教えるために使用できる安価な耐久性のあるマシンです。







傷のつきにくいお手入れが簡単な粗いプラスチックは、ラップトップの外観をなくします。学校の道具に合うように、きちんとした、目立たない、徹底的なものです。







コンピューターを学校に持ち運びしやすくする便利なペンは、CMPCを小さなブリーフケースに変えます。







ペンととんでもない青いカバーの両方が、このかわいいコンピューターからオフィス機器の印象的な深刻さを奪い、CMPCはより快適なおもちゃのように見えます。 しかし、見た目は偽装です-ボンネットの下では、CMPCは完全にモダンな詰め物です。







左側には、外部モニターを接続するためのVGAポート、USBポート、スリーブで覆われたSDカードリーダー、およびプルアウトオン/オフボタン（誤って押しないでください！） 背面には、電源コネクタとロックを取り付けるための穴があります。 ハンドルには、所有者の名前を挿入できる透明なシャッターのあるくぼみがあります-CMPCは慎重に設計されているため、この小さくて便利な小さなもので、コンピューターを異なる生徒と簡単に区別することができます。











蓋は上昇するだけでなく回転します-CMPCはタブレットを簡単に回転させます。 ネットブックの右側で、スタイラスが特別なソケットに挿入されます。 CMPCタッチスクリーンで便利に動作するように設計されています。







蓋の上部には、回転可能なカメラとマイクがあります。



















内部には、 Intel 945GSEチップセットに基づくIntel Atom 270プロセッサー （1.6 GHz）があります。 半ギガバイトのRAMが2つのスロットのいずれかに挿入されます。追加のRAMをコンピューターにインストールできます。 コンピューターは衝撃や衝撃に耐えるように設計されているため、ハードドライブの代わりに、過負荷に強い16ギガバイトのSSDが取り付けられています。







最大パフォーマンスモードでは、 CPU Free Benchmark 2.2テストは75.17秒で実行されます。 比較のために、わずかに高速なI ntel Atom 280プロセッサ （1.66 GHz）を搭載した新しい「アダルト」 ASUS EeePC 1005ネットブックは、 71.31秒でこのテストに合格しています。 最も近い競合製品であるASUS EeePC T91コンバーチブルタブレットは、 85.24秒ではるかに遅くなります。 言い換えると、CMPC内では、これは完全に大人のマシンであり、おなじみのネットブックの最新モデルと簡単に競合します。







バッテリーの寿命は約4時間で、これは生徒のほとんどの学校の日には十分です。



Windows XPの変換されたバージョンでは、かなり特定のプログラムセットがマシン内にインストールされます.Intelクイックランチャーとその他の改善の助けを借りたシンプルなWindowsインターフェイスは、大きなアイコンと明確な設定を持つ学生向けの軽量で簡素化されたオペレーティングシステムに変わります。 タッチスクリーンとスタイラスは、便利な描画と手書きの両方を提供します。 ところで、コンピューターには位置センサーが装備されているため、向きを変えると、CMPCをノートブックまたはノートブックとして使用できます。



Intel Network Managerを使用すると、ワイヤレスネットワーク経由のアクセスを簡単に構成できます。また、Intel Power Manager-コンピューターの動作モードと省エネを実現します。 多くのタブレットにバンドルされているArtRage 2グラフィックエディターは、簡単に描画できる優れたプログラムです。















ペアレンタル/教師の制御のためのCMPCと特別なソフトウェアを備えています。 子どものプロフィールでは、コンピューターリテラシーのレベルを指定し、特定のプログラムへのアクセスを許可し、デスクトップウィジェット（付箋、ゲーム、電卓、時計）を指定し、コンピューターの組み込みまたは再起動を有効または無効にできます。 素晴らしいアイデア-トレーニングプログラムと罰則のボーナスポイント-ゲームのようなさ。







ただし、CMPCビデオカードは、8メガバイトのメモリを搭載したチップセットに組み込まれたIntel 945GMEアダプターであるため、一部の複雑なゲームを実行することはできません。これはまだ学校のコンピューターです。



「戦闘」形式では、CMPCは小さなネットワークに展開され、一方では学校のWiFIに、もう一方では教師のコンピューターに結び付けられます。教師はコンピューターを使用して、生徒にタスクを与え、結果を収集し、それらを有効または無効にしますまたは他のプログラムなど。
