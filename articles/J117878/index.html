GoProカメラマウント



この特定のGoProモデルには、2軸スイベルヘッドを備えた大きな吸引カップが1つと、3M両面テープに5つのマウントが付属しています：平面用に3個、曲面用に2個。 さらに、カメラを3Mファスナーに取り付けるための3つの簡単に取り外し可能な機構があります。2つは従来型、もう1つは湾曲しています。 追加も提供します。 2つの平面での回転のために互いに固定できる回転機構。





吸盤は非常に快適で、しっかりと保持しますが、ガラスや車体などの完全に滑らかな表面上でのみです。 ラミネートされたパーティクルボードのようなわずかなテクスチャーがある場合、時間の経過とともに吸引カップが失われ、カメラが落下します。





3Mマウントを使用すると、カメラを車内と車またはオートバイの外面の両方に取り付けることができます。





もちろん、3Mマウントは消耗品であり、すぐに使用できなくなりますが、4ドルで追加購入できます。 現実には、ファスナーではなく強力な両面テープを購入する方が簡単で安価です。



説明によれば、この3Mのベルクロはしっかりと接着しますが、少し熱を加えると簡単に剥がれます（ヘアドライヤーなど）:)



必要に応じて、頭、胸などにさまざまな追加の補強材を購入してください:)



GoProでビデオを撮影する



冒頭で述べたように、絞りは2.8であり、かなり暗い場所での撮影が可能です。 レンズの焦点は、6メートルから無限遠まで固定されています。



ビデオの撮影プロセスは非常に簡単です。1回クリックするだけでカメラの電源がオンになり、すぐに撮影が開始されるか、別のモードでは、最初に電源ボタン、次に2番目のシャットボタンがあります。



設定に応じて、ビデオはHDまたは4つの異なる解像度で撮影されます。

  • 1080p = 1920×1080ピクセル（16：9）、30 fps、15 Mbit / sのデータレート
  • 960p = 1280×960ピクセル（4：3）、30 fps、12 Mbit / sデータレート
  • 720p = 1280×720ピクセル（16：9）、60 fps、15 Mbit / sのデータレート
  • 720p = 1280×720ピクセル（16：9）、30 fps、8 Mbit / sデータレート
  • WVGA = 848×480ピクセル（16：9）、60 fps、8 Mbit / sデータレート




カメラはミニUSBに接続され、その後、自動的に充電が開始されます。通常のフラッシュドライブからのように、簡単にカメラからビデオをマージできます。 ファイル形式はMP4で、すぐにコンピューターで再生されるか、YouTubeまたは別のビデオホスティングサービスに直接送信されます。



撮影



GoProカメラを使用すると、絞り、焦点の同じパラメーターを使用して、JPG形式で5メガピクセルの写真を撮影できます。



降下用のタイマーがあり、2、5、10、30、または60秒ごとに個別の自動撮影モードがあります。 3フレームシリーズをカスタマイズすることもできます。



しかし、一般的に、あなたはカメラに高い希望を持ってはいけません。これはカメラの目的ではありません。 写真は予算のせっけんの箱のように撮られているように感じますが、非常に広角で、ほとんど「魚眼レンズ」です。 夕方、何も処理せずに撮影した写真の例を次に示します。





目利きのための微調整



このような赤ちゃんが2つのボタンで多くの設定を制御できるのは驚くべきことです。 手順は、プロセスを明確かつ簡単に説明しています。 したがって、設定から利用できるもの：

  • 測光：中央重点またはスポット
  • ケーブルを介してビデオブロードキャストを構成する（たとえば、ワイヤレストランスミッターが切断される）
  • カメラが天井に取り付けられている場合や飛行機から落下している場合は、逆さまに撮影する
  • デフォルトモードの設定：オンにすると、写真またはビデオの準備ができます
  • 迅速な展開：電源ボタンを使用すると、撮影がすぐに開始されます
  • ビデオ解像度の選択
  • 自動写真の頻度を選択する
  • ビデオの撮影中に赤いインジケータを切り落とす機能（運転中、毎秒赤目でウインクするときは必ずしもいいとは限りません）
  • 静かなモード、ボタンを押しても信号なし
  • NTSCまたはPAL
  • 最後のファイル、SDカード形式全体を削除
  • バッテリーを節約するための自動電源オフ
  • 日時
