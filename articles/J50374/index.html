ノルウェー語 CalzoneはGog.isにあるシンプルでわかりやすいユーティリティサービスを考案しました。 残念ながら、Gog.isの主な唯一のタスクはURLを減らすことであるため、キャンディのデザイン、印象的な機能、または息をのむようなものを見つけることができません。 しかし、マイクロブログサービスのユーザーが慣れている形ではなく、Gog.isは検索クエリの行を短縮しますが、これを説明する最も簡単な方法は、例： Gog.is/habrahabrです。



スラッシュの後に挿入された単語は、Google検索で対応するクエリを形成します。これにより、迷惑な友人、好奇心の強い友人、および現代の検索エンジンの魅力とパワーをまだ知らない他のすべての人を適切な場所に送信できます。



Gog.isは基本的に機能します。たとえば、標準形式の検索クエリページ「 habrahabr gog.is 」のアドレスは次のようになります。www.google.ru/ search ?hl=ja&newwindow=1&rlz=1B3GGGL_ruCZ301CZ302&q=habrahabr+gog.is&btnG=%D0%9F D0％BE％D0％B8％D1％81％D0％BA＆lr =＆aq = f＆oq =



このサービスを使用すると、 gog.is / habrahabr gog.isで同じ結果ページを取得できます。



さらに、作成者（このサービスを楽しみのために開いた）は、いくつかの便利なプレフィックスを導入しました。
* shop：またはs :: Google製品検索に移動します。 http://gog.is/shop:ipodを試してください

* book：またはb :: Googleブック検索に移動します。 http://gog.is/book:poetryを試してください

* news：またはn :: Googleニュース検索に送信します。 http://gog.is/news:googleを試してください

* stock：またはq :: Googleファイナンスの最初のヒットが表示されます。 http://gog.is/stock:appleを試してください

* wiki：またはw :: gog.isは、Wikipediaの最初のヒットにリダイレクトします。 http://gog.is/wiki:leo+laporteを試してください

* movie：またはimdb：または単にm :: Imdb.comでも同じことを行います。http：//gog.is/movie：slumdogを試してください


ここに、ジョークとして作成された明らかな実用性のこのような基本的なサービスがありますが、その魅力は失われません。
