MicrosoftはWindows 10の最初のメジャーアップデートをリリースしました

マイクロソフトは非常に静かに、大騒ぎせずにWindows 10の最初の大規模な更新プログラム（ KB3081424 ）をリリースしました。 このアップデートはセキュリティの種類に属していませんが、OSのさまざまな欠陥を修正することに特化しています。 64ビットバージョンのWindows 10の場合、更新サイズは約325MBです。 MSがWindows 10のリリースされた更新プログラムに関する記事の形式を作り直したことは注目に値しますが、実際には、リリースされたパッチに関する情報は公開されていません。







更新自体（ Windows 10の累積更新：2015年8月5日 ）は、ドライバーを含むさまざまなシステムファイルの修正を目的としています。 修正可能なファイルの完全なリストはKB3081424にありますが、その目的については推測しかできません。





図 Windows更新プログラムのリストにインストールされている更新プログラム。



Windows 10には、このOSを実行しているコンピューターがグローバルネットワーク上の他のコンピューターと更新プログラムを交換できるようにする特別な設定が含まれています（！）。 MSサーバーに直接接続して更新を受信する代わりに、コンピューターはネットワーク上でWindows 10を実行している他のコンピューターに接続し、そこから更新を受信できます。 この機能を無効にするには、「設定」->「更新とセキュリティ」->「Windows Update」->「詳細設定」->「更新を受信する方法とタイミングを選択」に進みます。





図 含まれる（ デフォルトで ）パラメーターには、Windows 10を実行しているコンピューターが一種のP2Pピアツーピア更新交換ネットワークに含まれます。



以前のバージョンのWindowsとは異なり、Windows 10では、ユーザーは更新プログラムのインストールを拒否できません; OSのそのような設定は存在しません。 ユーザーは、インストール後にOSを再起動するオプションのみを選択できます。 ただし、Windows 10には、既にインストールされている更新プログラムを削除する機能があります。
