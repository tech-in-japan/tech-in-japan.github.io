AN / FSQ-7-史上最も取り外し可能なコンピューター









第二次世界大戦中であっても、米海軍はマサチューセッツ大学に爆撃機の乗組員を訓練するためのフライトシミュレータを作成する可能性について提案しました。 1945年のENIACの成功の後、これらの目的でコンピューターを使用し、失敗したアナログコンピューターの作成試行を放棄することが決定されました。



プロジェクトは著しく遅れ、第二次世界大戦の終了後に完了しました。 Whirlwindコンピューターが作成された場合の対処方法は不明のままでした-軍は事業への関心を失いました。 それにもかかわらず、それは歴史的に重要でした-防空システムを作成するというジェイフォレスターの提案の基礎となったのは、旋風と未実現の旋風IIプロジェクトでした。



ソ連における原子兵器の出現と、両国間の率直な緊張関係により、出来事の流れは加速しました。 早くも1949年12月、ジョージバレーが率いる防空委員会はレーダーステーションのコンピューターコンピューティングを推奨しました。 バレーとフォレスターは実際に将来のSAGEシステム（SAGE）の基礎を築きました。その開発には1954年に約100億ドルの費用がかかり、AN / FSQ-7コンピューターを装備した24のコマンドセンターの作成も含まれました。 。



コンピューターはシステムの重要なコンポーネントでした。 数百のQ7パネルで、スイッチ、ボタン、点滅ライトを備えたビジュアルシネマの楽園を簡単に見ることができます。 そして、それがQ7が史上最もレンタルされたコンピューターである理由です。 コンピュータ時代のawn明期に作成され、1983年以降は使用されなかったにもかかわらず、現在でもさまざまな部分がフレームに表示されています。







1953年9月、Whirlwind IIの未実現計画に基づいて、IBMは2つのプロトタイプの供給契約を締結しました。 同じ年の10月28日に、空軍評議会は、リンカーンの自動システム（1954年にSAGEに改名）に融資するために、1955年の予算から資金を割り当てることを推奨しています。 1955年、レキシントンのSAGE実験サブセクターが完成し、10月までにXD-1と呼ばれるAN / FSQ-7プロトタイプがビルFに登場しました。



空軍要員は、1955年にニューヨーク州キングストンで訓練を受けました。 1959年までに、2,000のシミュレートされた傍受がすでに実行され、最初の実際の傍受は1958年8月に行われました。 ATABEアルゴリズムの数学モデル（自動ターゲットおよびバッテリー評価）の大規模なテストは、防衛部門（特にSkyshield軍事演習）の訓練違反を発見した軍事レーダーの実際の読み取り値を使用して実行されました。



24匹のモンスターのそれぞれは、49,000の真空管を内部に備え（外部システムと一緒に、各ポイントに約6万のランプが含まれていました）、重量が250トンで、2,000平方メートルを占めていました（アメリカのさまざまな場所にあるカナダ）、3メガワットの電力のピーク時に消費しましたが、その間生産性は高く、毎秒75千回の操作で、最大300の空中目標の位置を追跡することができました。 100個のオペレーターコンソールをコンピューターに接続することができました。これは、ターゲットを強調するライトガン付きのモニター、シガーライター、灰皿でした。







RAMシステムのQ7機能の信頼性を確保するために、命令制御、メンテナンス、演算モジュール、入出力制御、およびプログラム要素が複製されました。 したがって、Q7のそれぞれは、ラテン文字AおよびBと呼ばれる2つの独立したコンピューターの組み合わせでした。これらは同時に機能しませんでした。Aは作業を行い、Bはスタンバイモードでサービスを受けることができた後、役割を変更しました。 平均して、各コンピューターは1メガワットの電力を消費しました。



コンピューターには、IBM 723カードパンチ、IBM 713パンチカードリーダー、IBM 728磁気ドラム、他のSAGEコンピューターとの交換デバイス、および異なる部屋にある数十台のコンソールで構成されるライトディスプレイおよび警告システムが装備されていました。











Q7を使用して、 Bomarkコンプレックスからミサイルを直接発射することが可能であり、コンピューターアルゴリズムがターゲットへの離陸、飛行、および超音速ジャンプの開始を自動的に制御しました。 その後、有人戦闘機の自動操縦を設定して地上対空通信サブシステムを使用してターゲットを狙うシステムが実装されました。



Q7は、磁気コアメモリ1番-256×256 33ビットワードのグリッド（合計65.536マシンワード、左の写真に表示）-2番目の小さなモジュール-64×64、4096 33ビットワードを所有していました。 メモリサイクルは3.25マイクロ秒でした。 SAGEシステムのコントロールセンターを装備した彼の弟FSQ-8には、サイズ4096マシンワードの2つのモジュールがありました。 2つのモジュールが存在する理由は、メモリモジュールの1つで診断モードを実行する必要があるためです。 33ビットのマシンワードのビットの1つはパリティビットでした。



データを保存するために、各機械語は2つの半分に分割され、それぞれに15ビットの数値と符号ビットが格納されました。 算術演算は、両方の半分で同時に実行されました。 乗算中のオーバーフローを避けるために、数値は-1から1の小数として表されました。 計算の限界を確立することはプログラマーにかかっていました。



命令は、マシンワードの前半と後半の符号ビットを使用してアドレスを形成し、17ビットメモリアドレッシングを使用できるようにしました。 後半の残りには指示が含まれていました。 符号のビットの後の後半の最初の3ビットは、インデックスレジスタ、次の命令クラス、そのタイプ、およびサポート情報を決定しました。 アドレスは8進数で記述され、2ビットでプレフィックスが形成されました。



また、マシンには12個の磁気ドラムがありました。6個はディスプレイシステム用、6個はコンピューター自体用です。 各ドラムには33個のヘッドといくつかの予備のヘッドがあり、33ビットの単語をすばやく読み取ることができました。







SAGEシステム全体は1963年に展開されました。 皮肉なことに、チューブコンピューターのネットワークはこの時点で技術的に時代遅れになっただけでなく、大陸間弾道ミサイルの出現と「サテライト1」（いわゆる「衛星危機」）の形での作業の現実に対する明確な脅威により、そのようなシステムの意味が失われました、弾道ミサイルが有人爆撃機に損害を与えるという普遍性に関するソビエトの最高指導者の間の狂信的な信念と同様に。



すべての技術的な美しさのために、コンピューターには一連の欠点がありました。真空管の寿命が短く、自動診断システムと膨大なエネルギー消費が必要でした。 かつて、ニューバーグ（NY）でテストされたとき、コンピューターは間違ったフェーズに接続されていたため、ラインペンシルバニア自体への切り替えに失敗しました。 各マシンには、約60人のスタッフが必要であり、コンピューター自体、I / O、およびディスプレイの3つのグループに分かれています。 また、1964年、テスト中に飛行経路の処理でエラーが検出されました。いくつかの技術的条件が一致した場合、それらはしばしば破棄されました。



最終的に、SAGEはマンハッタンプロジェクト全体よりも多く使用されていたため、段階的に廃止されました。 AN / FSQ-7コンピューターは徐々にBUIC（バックアップインターセプターコントロールシステム）に置き換えられましたが、1983年まで2つのQ7が維持されていました。



AN / FSQ-7で導入された技術は、1964年にIBMが作成したSabreチケットシステムで使用されました。 Q7は多くのテクノロジを作成するように世界を推進し、現代のモデムのプロトタイプである通信用の電話回線を使用したコンピューターの最初の分散ネットワークでした。 Q7では、当時の新しい磁気コアメモリ、対話型グラフィックディスプレイ、ネットワークデータベース、マルチタスクシステム、構造化ソフトウェアモジュール、モジュラーコンピューターを使用していました。 SAGEプロジェクトのおかげで、ソフトウェアエンジニアとライトガンの職業はRobert Everettによって発明されました。



もちろん、AN / FSQ-7はコンピューターサイエンスで大きな功績を残しましたが、もっと興味深いのは、コンピューターが彼の死後にどのようになったかです。 驚いたことに、マシンは前世紀の50年代に開発されたという事実にもかかわらず、その断片は依然として未来のスーパーコンピューターの雰囲気を作り出すために、または少なくともQ7が入ったものより後の時代の機器の効果を作り出すために使用されています。



たとえば、 ランプコンピューターのパネルは、21世紀の効果を生み出すために使用されます。 1973年のコメディー「スリーパー」では、マイルズモンロー（ウッディアレンが演じる）が200年の間極低温チャンバーで眠り、2173年になってしまいます。Q7の非常に多くの機器が使用されます。これはコマンドポストパネルおよびセクションです二重サービスコンソールインジケータ。











1982年から86年にアメリカ空軍に勤務したマイクルーベン。 そして、AN / FSQ-7の最後の作業インスタンスをキャッチする最後の1つである彼は、60年代半ばのシリーズ「タイムトンネル」でおなじみのパネルに初めて気付き、このチューブモンスターが演じる他の映画に興味を持ちました。 一時トンネルでは、Q7はアリゾナ砂漠のタイムマシンの秘密政府プロジェクトの役割を果たします。











AN / FSQ-7コンピューターの登場は、映画「Doctor Strangelove、または私が恐れることを止めて爆弾を愛する方法」で最も論理的です-それは防空の直接の機能を果たしますが、スタンリーキューブリックのコメディは古いIBM 7090/94を使用します。



Q7が登場した最も人気のある作品の1つは1996年の独立記念日です。 シリーズの最後のコンピューターは13年前に停止しましたが、映画では現代のコマンドセンターの役割を果たしています。







1983年の映画「戦争ゲーム」には、多くの点滅するパネル、スクリーン（メインキャラクターが表すもの-WOPRディフェンスコンピューター）がいっぱいですが、Q7はそれらとは何の関係もなく、メインの磁気ドラムと回路インジケーターのみのコントロールパネルの形で表示されますファルケン教授とのアーカイブ動画で。
