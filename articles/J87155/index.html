iPhoneのコンピューター管理

このトピックは新しいものではないかもしれませんが、提案されたオプションが誰かに役立つ可能性があります。



現代の技術が人類のすべてのニーズを1つのデバイスに転送しようとしても、スマートフォン（存在する場合）を除くほとんどすべての人がホームコンピューターと、おそらくラップトップまたはネットブックを持っています。 これは、リモートデスクトップ管理ソフトウェアの簡単な概要です。 以前のrAdminは特に人気があったことを覚えていますが、IPアドレスとそれが接続されたものを常に覚えていなければなりませんでした。 約3年前、私はTeamViewerに出会いましたが、それははるかに実用的で便利であることが判明しましたが、今でも喜んで使用しています。 どういうわけか、時間が経つにつれて、ネットブックの出現により、ポータブルデバイスから任意のマシンのデスクトップを制御する必要がありました-このビジネスは簡単になりましたが、この小さなユニットを常に持ち歩いています。 そのため、よりコンパクトなデバイス向けのより高度なソフトウェアを見つけることを考えました。





率直に言って、私は自分でiPhonを使用していません。私は彼を嫌っています。 しかし、定期的に仕事中のユーザーにブロンドを支援する必要があります（メール、インターネットなどを構成するため）。 iPhoneを使用してコンピューターのデスクトップを完全に制御すると便利だと思いました。

ここにいくつかのプログラムの概要があります。







LogMeInはおそらく最も便利なもので、Habréで既に書いていますが、投稿は2008年6月11日付で、iPhonを使用してコンピューターを管理するLogMeInサービスがまだなかったかもしれません。 このプログラムには非常に優れたインターフェイスがあり、アメニティは低下していません。 ここで見つけることができます。 このプログラムは無料ではなく、29.99ドルのグリーンプレジデントがかかりますが、そのようなお金のために、このプログラムは機能に非常に適していると思います。 AppStoreのプログラムにリンクします。



Touchpad Proは、画面共有に適したプログラムです。 負の場合、インストールプロセスのみが残ります。 これを行うには、VNCサーバーをコンピューターにインストールし（Windowsを実行している場合、Macの場合、プログラムは既にシステムに組み込まれています）、必要に応じて構成し、iPhonに移動します。 プログラム自体は無料で、マルチメディアボタンの料金を支払うだけで済みます。







残念ながら、私自身はまだ自分でそれを使用することができません。そのため、設定でマシンのIPアドレスを指定する必要があるため、プログラム自体がどのようにコンピューターに接続されているかという疑問が残ります。 説明と手順を記載したプログラムの公式Webサイト（英語）へのリンク 。 ネットにひび割れがあります。 ただし、今日のメディアプレーヤーのほとんどを管理するTouchPad Media Serverプログラムの費用はわずか5ドルです。 ちなみに、このプログラムは、iPod TouchおよびMicrosoftプラットフォーム（Windows Mobile 5および6）でも機能します。





モバイル用リモートデスクトップ（RDM +）は、LogMeInと同様に、Webブラウザーを介してリモートコンピューターを制御できるプログラムです。 インストールと保守が非常に簡単です。 必要なのは、会社のWebサイトにアクセスし、登録データを入力して、接続に必要なデバイスを選択することだけです。 AppStoreで購入した価格は9.99ドルです。 どうやら、プログラムはJavaで記述されているため、ほぼすべてのモバイルデバイスで動作します。 公式ウェブサイトへのリンク 。



さらに、提示されたプログラムはおそらく中国を経由してモスクワに行くかもしれませんし、他のよりシンプルなツールがあるかもしれませんが、幅広い機能はそれ自体を物語っています。
