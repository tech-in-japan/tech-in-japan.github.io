ソーシャルゲーム：危険な地球-2！

これは、ソーシャルネットワーク向けのゲームを作成した3人（フラッシャー、ユニキソイド、アーティストの古典的な三角形）についての物語の続きです。 2か月が経ちましたが、開発のためのパートナーを見つけた経験を共有したいと思います。Habrのおかげで非常に現実的なことがわかりました。





Runetソーシャルネットワーク（だけでなく）を横断する航海の以前の「シリーズ」で、私たちのおもちゃは水泳だけでなく、さらなる発展のために十分なお金をもたらし始めたという結論に達しました（私たちの歴史の始まりはここにあります ）。 このゲームは「危険な地球」（ VKontakteの ゲーム、My Worldのゲーム）と呼ばれ、次のようなものです。



おもちゃ自体は、お互いの歯に似た一連の「農場」からノックアウトされました（そして今ノックアウトされています）。私たちのゲームでは、プレイヤーはリアルタイムでお互いに戦い、母なる地球の重要な都市を支配します。 この違いは、私たちと多くの異なるジョークを演じました。一方で、おもちゃはプレイヤーが高く評価するユニークなものであることが判明しましたが、他方では...理解できない「農場ではない」タイプのゲーム。



同時に、半年に1回の無料水泳の経験により、残念ながら、外部からのトラフィックが発生しないと、通常の結果を達成できないことが明らかになりました。 言い換えれば、最初からのウイルスの広がりによるゲームのテイクオフは単純であり、ウイルスが与えるのはトラフィックの特定の「安っぽい」ことです（つまり、アクティブなプレイヤー10人ごとにもう1つ「理由もなく」持ってきます）。 それからHabrは私たちの援助に来ました。おそらくそれは国のすべてのIT専門家が読んだでしょう:)とにかく、彼らが私たちに注意を向けることはできそうにありませんでした



最初の記事の数日後、私たちは小さなソーシャルパブリッシャー、MobMentalityという会社に会いました。 しばらく話をして、これらの人々は完全に「トピック」であると確信しました：経験とゲームの作成とプロモーション（純粋にカジュアルとソーシャルネットワークの両方）、十分な数の興味深い（不条理ではない）アイデア、オープンな共有情報と私たちと一緒に働くことを試みる準備ができています。 実際、最初からの協力は実り多いものでした：私たちのゲームを見た後、MobMentality（特に、特定のゲームの特定のサークルで悪名高いWheemplayのStefan Keishが特定のゲームで悪名高い）は、収益化におけるいくつかの大きな誤算を指摘し、その後支払いがすぐにcい上がりました:)最後の仕上げを収益化するために、私たちは急いでゲームを改良し、その後MobMentalityは私たちに正直にトラフィックを供給し始めました。 結果は、VKontakteからのゲームでの支払いの次のスクリーンショットで判断できます。







また、今週はOdnoklassnikiでゲームを開始します。新しいピークはまだ先だと思います。 API wikiには、バイラル機能はまだ機能しないというコメントがたくさんありましたが、「間もなく」なります。この「間もなく」はほとんど届きました！）



協力の別の結果は、私たちが現在行っている新しい本格的なゲームでした（9月にリリースされる予定です）。 秋には（噂によると！）約束しますが、開発者のためのRunetのソーシャルネットワークでの強い揺れは、良いゲームで失われないようです。



この数ヶ月の間に、私たちはMySpaceに行きました。これについても少し言いたいと思います。 独自の自社開発のAPIはありませんが、独自のGoogle OpenSocialなので、そこに到達しようとすると、一度にいくつかの目標がありました。結局、OpenSocialは、たとえばOrkut、Bebo、その他のネットワークで多くの機能を果たします GoogleガジェットでもOpenSocialをサポートしています。 Googleからの今後のソーシャルネットワークについて忘れてはなりません。OpenSocialもベースAPIとして存在すると考えられています。 少なくともGoogleは、MicrosoftのIMHOなどとは異なり、自社の技術の差別化についてはまだ気づいていません。



だから、MySpaceは絶対に地獄のようなQAであることが判明しました。数人のインド人が文字通りゲーム自体をテストしました。 正式なテストでは気付かれないものをエラーとして示しました。 これに加えて、悲しいかな、ウイルス性はゼロでした。 理解できない組み合わせ。 どうやら、これはこれまでのゲームプラットフォームではないようです。少なくとも私たちの経験ではこれについて話しています。 Googleからのソーシャルネットワークを待っています:)



これですべてです。興味のある方は、現場からのスタートアップの開発に関するレポートをさらに続けることができます:)最後まで読んでいただきありがとうございます！
