Wi-Fi Allianceは、新しいセキュリティ機能を備えたWPA3プロトコルとWi-Fi 802.11ax規格の開発を発表





Wi-Fi Allianceは、待望のワイヤレスセキュリティプロトコルの第3世代であるWi-Fi Protected Access（WPA3）を発表しました。



WPA3は、少なくとも15年間存在し、毎日何十億ものワイヤレスデバイスで使用されているネットワークセキュリティプロトコルである既存のWPA2を置き換えるものです。



WPA2の不安は、専門家によって長い間議論されてきました。 ターニングポイントは2017年10月で、研究者のMathy VanhoefがWPA2プロトコルに脆弱性を発見し、既存のほぼすべてのWi-Fiネットワークを侵害しました。 攻撃者は、その助けを借りて、 キー再インストール攻撃 （KRACK）攻撃を実行し、機密データにアクセスすることができます。 プロトコル自体WPA2に含まれているように、脆弱性はセキュリティの専門家に深刻な懸念を引き起こしました。



個人用と企業用の両方のワイヤレスデバイスで利用できる新しいWi-Fiセキュリティ標準は、4つの新機能によりプライバシーを強化します。



  1. WPA3プロトコルは、個々のデータ暗号化を通じてオープンネットワーク上のユーザープライバシーを強化します。 各デバイスとアクセスポイント間の接続を暗号化できます。
  2. ブルートフォース攻撃保護が導入されました。これにより、ハッカーは頻繁に使用されるパスワードを使用してシステムに何度も侵入することができなくなります。 セキュリティメカニズムは、ログインの試行が数回失敗した後に認証プロセスをブロックすることにより機能します。
  3. IoTデバイスの簡易セットアップを実装しました。 WPA3では、互いに近くにあるWi-Fiデバイスを、他のデバイスの構成パネルとして使用できます。 そのため、ユーザーは電話またはタブレットを使用して、画面のないデバイスでWPA3 Wi-Fi設定を構成できます。
  4. Wi-Fiネットワーク用の最新の暗号化標準、いわゆる「192ビットセキュリティパッケージ」が導入されました。 彼は、国家安全保障システム委員会が作成した商業国家安全保障アルゴリズム（CNSA）スイートに焦点を当てます。 このソリューションは主に、政府、防衛、産業組織など、セキュリティ要件の高いネットワークを対象としています。


WPA3セキュリティプロトコルを使用するには、ハードウェアがWi-Fi Allianceによって認証される必要があるため、新しいセキュリティ標準は一晩で実装されません。 デバイスの製造元には数か月かかる場合があります。 最初の認定WPA3デバイスは今年後半に利用可能になる予定です。そのため、 クラウドにアクセスするための強力な暗号化を備えた無料のVPNをお客様に提供しています。 WPA3の詳細については、後で公開されます。



2018年にWi-Fi Allianceによって計画された他の新しい開発の中で、新しいWi-Fi 802.11ax規格を強調できます。



「部屋全体を無線信号で満たす」のではなく、特定の電話、PC、その他のデバイスへのデータ転送



Wi-Fi Allianceのマーケティング担当副社長であるケビン・ロビンソンは、次のように述べています。 このため、新しいWi-Fi標準802.11axなどの将来の開発は、ピークデータ転送速度を上げるのではなく、「ネットワーク帯域幅全体を増やす」ことを目的としています。


マルチユーザーMIMO（MU-MIMO）は、 802.11acの重要な革新の1つです。 これにより、空間ストリームを分離し、複数のクライアントへのデータの同時送信を整理できます（通常のMIMOでは1つのクライアントのみに同時に送信するのではなく）。 MU-MIMOはWave 2デバイスで動作し、ダウンリンク接続（アクセスポイントからクライアントへ）に対してのみ有効です。 このサービスは4人までのユーザーに提供されます。 現在のアクセスポイントは、MU-MIMOモードで最大3人のユーザーをサポートします。







Wi-Fi 802.11ax規格の主な目的は、ネットワーク帯域幅を拡大し（4倍の成長が見込まれる）、問題領域で高品質の接続を維持するためにWi-Fiスペクトルをより有効に活用することです。 接続デバイスの数の増加、トラフィックの多様性、重複するネットワークの密度により、現在Wi-Fiスペクトルが過負荷になり、接続の品質が危険にさらされています。 この問題を解決するために、802.11axはより多くのストリームおよびその他の機能への送信をサポートし、スループットとカバレッジを最大化します。 さらに、このようなソリューションは、直交周波数分割多重アクセスやトラフィックスケジューリングなどのセルラーテクノロジーをサポートし、効率を高めるように設計されています。 802.11axは、チャネルが混雑している状況で役立ち、多数の接続をよりインテリジェントに管理できます。



モビリティを向上させる最適化されたWi-Fi接続



VantageのWi-Fiテクノロジーは、複数のネットワークアクセスポイントがある場合に接続をより適切に管理します。 これは、たとえば、ユーザーがインターネットを介して音声で通信しながら接続を切断することなく、駅または自宅でアクセスポイントから別のアクセスポイントに切り替えることができることを意味します。



「過去、私たちのほとんどは単一のWi-Fiホットスポットを使用していました。 あなたのアパートの壁が多すぎて、良い信号を得ることができなかった場合、困難がありました。 現在、いくつかのルーターがより一般的であり、Wi-Fi Allianceはいくつかのネットワークデバイスが自動的に通信できるように作業を開始し、データを必要な場所に移動します。他の人に」、ロビンソンを追加しました。


また、ネットワーク全体の帯域幅が802.11axの主な焦点ですが、最大速度も向上するはずです-802.11acの433 Mbit / sから802.11axの600 Mbit / sまで。



IHS Markitのアナリストは、802.11ax対応デバイスの出荷が2019年の最小116,000ユニットから2021年には5,800万ユニットに増加すると予測しています。
