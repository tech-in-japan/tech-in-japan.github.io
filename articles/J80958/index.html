WindowsではなくLinux。 ゲームはろうそくの価値がありますか？

それはどうなるのでしょうか？







私の専門的な活動の現在の段階では、商業および地方自治体の組織の、より正確には、クライアントのすべてのタスクを確実に遂行できる部分へのフリーソフトウェアへの移行の問題に取り組んでいます。 したがって、この記事をMicrosoftのソフトウェアとGNU / Linuxに基づくオープンソースソフトウェア、特にMS Windows / Ubuntu DesktopとオフィススイートMS Office / OpenOffice.orgの範囲に限定することは理にかなっていると思います。



誰かが他人の仕事を盗むことは悪いことだと理解していると思いますし、（盗まれた場合）それでお金を稼ぐことはさらに悪いことです。 しかし、歴史的に、海賊版のWindowsとOur祖国のOfficeは標準です。 少なくとも、MKAD外のこのソフトウェアの大部分は違法に取得（またはダウンロード）されました。 経営陣は、原則として、合法的なソフトウェアを取得することの重要性を完全に理解していない、または理解したくない場合があります。 請求書がすぐに深く中断されたアニメーションに没頭した後、「合法化」する準備ができている人。 もちろん、さまざまなソフトウェアレンタルプログラムがありますが、第一に、これでお金を稼ぐことはできません。第二に、これはそれについてではありません。 :)



違法なソフトウェアに対する責任



アートに精通している場合。 刑法の146、あなたはそのような「トリック」が刑事責任を負うことをよく知っています。 私はこれで誰かを怖がらせたくありません、私はこの投稿がこの責任が言及される最初のものではないと思います。 これは議論の1つに過ぎず、Microsoftのソフトウェアの高価格と同じくらい重要です。



目標を設定します



現在、ほとんどの場合、検討中の問題の枠組みの中で、次の目標が急務です：法律上（ライセンスされた）ソフトウェアにできるだけ低コストで、経済的および生産コストを切り替えます。



知っておくべきこと

  1. 組織内で動作するソフトウェア（「休日のためにあらゆる種類のポストカードを作成するためのフォトショップ」ではなく、同じビジネスではなく、vista ultimateが例外なくインストールされました）。
  2. 組織で使用されるハードウェアの種類（特に、取引機器、各種スマートカード、Win-Scannerなど）。
  3. 組織内の「木」人の人数は何人ですか（新しいツールの使用方法を学習できない、および/または学習したくないユーザーを意味します）。

金融には何がありますか？



移行プロセスの円滑化に役立つソフトウェア開発者のサイトには、ライセンスのコストを計算するための計算機があります。 たとえば、Windows、Office、1C、職場の保証人/コンサルタント、およびターミナル/ファイルサーバー**がインストールされた15の職場を持つ小規模な企業を検討できます。







*数字に同意しない場合-自分の数字に置き換えても、本質は変わらないと思います...



**このオプションでは、任意のルーターを介した組織のインターネットへの接続が考慮されます。つまり、専用のプロキシサーバーまたはインターネットゲートウェイがなければ、図が大幅に増加します。



解決策



いくつかの正しいシナリオを強調できます。
  • マイクロソフトからソフトウェアを購入する組織と質問は終了します。
  • Linuxを最大限の数のワークステーションとサーバーに導入します。 この例では、LinuxをPCおよびサーバーパーク全体に展開できます。

そして、なぜLinuxを実装する必要があるのですか？



ここでは、3つの費用項目に注意する必要があります：実装のコスト（IT専門家の作業と、最小限の血量を可能にするLinuxソフトウェアの1回限りの費用）、スタッフトレーニング（原則として、ユーザーが10から12アカデミック時間の入門コースで十分です「木製」）、およびITインフラストラクチャのさらなるサポート。



また、サービスの価格は任意であることを理解する必要があります。 私は自分で使った数字だけを使います。



与えられた例では、この計算のようなものがあります：
  1. Workplace = 3000 x 15 = 45000ルーブル。スタッフ向けのLinuxコースの導入を考慮に入れています。
  2. サーバー（バックアップ組織）= 6000ルーブル。
  3. ソフトウェア= 7000ルーブル以下。
合計、アカウントの合計金額は58,000ルーブルを超えず、MSからソリューションを購入するよりも6倍安くなります。 さらに、違い（およびこれは312,000ルーブルです！）、年間契約の支払いに費やすことができ、パーソナルコンピューターのメンテナンス、さらには無制限のインターネット512 kbpsにとどまります！



製作費



しかし、すべてが翻訳された例のように美しいわけではありません。 多くの場合、組織はPhotoshop、Corel、xCADなどのソフトウェアを本当に必要とします。そして、多くの会計士はインターネット経由で税レポートと銀行システムを使用します。



ここでは、ライセンスされたWindowsをインストールするか、スタッフを再教育するかを選択する必要があります。 しかし、そのようなプログラムの無料の類似物はユーザーによって否定的に認識されており、ここでは非難することはできません。 MS OfficeからOpenOffice.orgに移行する場合、タスクは非常に簡単です。同じGIMPを習得することは簡単なタスクではなく（あなた自身の経験）、これについてホリバーを手配するべきではありません。



しかし、原則として、そのような状況は、規則を確認する例外です。



遅いLinuxプロモーションの理由



他の都市や地域の状況はわかりませんが、ここ（Khanty-Mansi Autonomous Okrug / Surgut）では、この問題はゆっくりと解決されています。誰にも解決されないと言う人もいます（もちろん、私以外）。 次の3つの理由があります。
  1. 誰もが組織にインストールされているソフトウェアを合法化することを望んでいるわけではありません。 これは当局と彼の隠された恐怖症の「ナンセンス」です。 その1つは、「あなたが参加しない場合、誰がこれに同行するか」です。 ただし、上記を読むとわかるように、テクニカルサポートの年間購入に提案された違いを送信できます。 また、Windowsシステムのサポートと保守がそれほど必要ではないと考えないでください。実際、ここの図はまったく逆になります。
  2. それでも、Windowsシステム市場には他に類を見ないほど多くの専門家がいます。 しかし、それは悲しいことではないので、コンピュータのメンテナンスとメンテナンスの価格だけでなく、これらのサービスの品質も競争の激化とともに低下しています。
  3. 今日、組織が実際に作業するために必要なソフトウェアおよびハードウェアソリューションがありますが、それらはLinuxと互換性がありません。 そして、これにより、ある種の悪循環が生まれます。 これらのソリューションのメーカーは、（少なくともデスクトップでは）（Windowsと比較して）普及率が低いため、Linux用のバージョンを作成しません。

結論



あなたがリーダーであり、あなたの会社のITインフラストラクチャについて気にかけない場合（それはあなたに利益をもたらします！）、あなたは少なくともあなたの組織のニーズに違法なソフトウェアを使用する場合あなたを脅かす結果について考えなければなりません。 また、Microsoftのソフトウェアのライセンスのみを購入するだけの大きな費用を支払う余裕がない場合は、間違いなくフリーソフトウェアへの移行を検討する必要があります。
