HTC Desireでの着信音の作成とその克服方法に関する奇妙な不具合

HTC Desireの公式ガイドでは、「音楽」アプリケーションメニューで対応する項目を選択して、サウンドレコーディングから着信音を作成することをお勧めします。
ストレージカードにダウンロードまたはコピーされた着信音または曲は、着信音選択メニューに自動的に追加されません。 音楽アプリケーションにアクセスし、ストレージカードの着信音または曲を着信音として設定する必要があります。 音楽アプリケーションを使用して着信音を設定する方法については、音楽の章の「曲を着信音として設定する」を参照してください。
一見したところ、これはすべて真実のように思えました。録音は最初は実際に着信音のリストを作成しませんでしたが、音楽アプリケーションはそれらをそこに追加することを許可しました。



しかし、数時間後、予期せず奇妙な不具合が発生しました。再起動直後（電話をオフにしてから再びオンにする）、「ミュージック」アプリケーションによって作成されたすべての着信音は、すでに一部の連絡先または着信音として割り当てられていても、着信音のリストから消えますデフォルトで。 したがって、音声通話の設定がリセットされ、これは面倒です。



幸いなことに、メモリカードに記録された録音は、実際には、すべての録音がフォルダー内にある場合、アラーム呼び出しと通知音のリストだけでなく、着信音のリストにも自動的に追加できることが判明しました。特別な名前：



/ SDカード/メディア/オーディオ/アラーム→アラーム音、



/ SDカード/メディア/オーディオ/通知→SMSアラート音、



/ SDカード/メディア/オーディオ/着信音→着信音（電話の着信音）、



/ sdcard /メディア/オーディオ/ ui→ユーザーインターフェイスの他のボイスオーバー。



この状況は以前にHabrahabrで言及されていました。つまり、 1月のブログ投稿の1つで、実際にはHTC Heroに属し、その著者の個人ブログにありました-今日、この回避策はHabrahabrではなく、 FAQ w3bsit3-dns.com 一般に、 明らかに、着信音のリストだけでなく、SMSと目覚まし時計の音も補充できることに加えて、すべてのAndroidオペレーティングシステムに等しく適用されます。



「ES File Explorer」を使用して必要なフォルダーをサブフォルダーで作成し、目的のメロディをそこに転送すると、すぐに携帯電話を再起動し、上記の方法の有効性を確信しました。



そして今、あなたはすべてそれを確信することができます。
