私について少し



彼は最後の大学ではなく、サンクトペテルブルクで勉強しました。 私は小学生と少し働きました。 現在、私は言うように、世界のトップ10の大学（米国）の1つで、大学院生として重点研究室と物理学研究室を運営しています。 周りを見渡して、私たちの教育がまだ評価されている理由と、何がそれについて本当に良いのかについて考え始めました。

タスク：「太陽の円盤を横切る金星の通過」


ご存知のように、金星は時折太陽の円盤を通過します。 最後のそのようなイベントはちょうど最近でした-今年6月6日。



このテーマの物理的な問題がどうなるか想像してみましょう。 ドロップダウンのトレーニングが充実している学生向けの文言から移動します。 だから。



オプションA


金星が太陽の円盤を移動する頻度を推定します。

指定：参照データ。



簡潔な状態。 このバージョンでは、可能な限り研究に近い創造的なタスクを想定しています。



すべての効果を特定するために、タスクはいくつかの連続したステップを必要とします。各ステップは非常に簡単です。 このタスクは、日食の間隔の交互を深く掘り下げて定性的に示すことができるか、頻度を大まかに推定できるという点で優れています。



短い解決策：

1.金星と地球が異なる平面（約3度の角度）にあるという事実のため、金星が地球と太陽の間にあるたびに、記述されたイベントが発生するのではなく、軌道面の直接交差に沿って発生する場合にのみ発生します。

2.最後の出来事が2012年6月6日に起こったことを知り、地球と金星の周期の比が約13/8であることに注目して、結論を出すことができます。 次回、地球は8地球年（または13年の金星）後に同じ構成になります。

3.期間の比率がまだ正確に13/8ではないという事実に注意を払いますが、テールを使用すると、このテールが何年で大きな位相シフトを与えるかを推定できます。 これを行うには、日食の持続時間を評価し（これは単純なジオメトリから行われます）、それを尾と比較します。 特に正確な計算を行わずに、イベント間の時間間隔は次のように交互に変わると結論付けることができます：最初は8年間で数回、次に位相変化があり、〜100年のギャップが観察され、その間に1つの位相に等しいシフトが蓄積され、その後サイクルが繰り返されます。

B）金星の1年が地球の8/13年にほぼ等しい場合、惑星はどのくらいの頻度で相互構成に戻りますか？

C）期間の比率が正確に8/13ではなく、0.61533の場合、パラグラフBの答えはどのように変わりますか？

レッスンに座っていて、頭に何かが落ち着いている生徒向けのオプションD。



当然、AからGまでの条件の範囲はほぼ連続しています。



USEの準備の乱用により、両親の過失と学校のイニシアチブの両方で、A-BやQのようなタスクは完全に失われます。 例を挙げましょう。 括弧（a + b）*（c + d）の開き方を尋ねると、どのように考え始めますか？ 私たちの基礎教育では、「最初の括弧の各項に2番目の括弧の各項を掛ける必要がある」という精神のルールを覚えていると規定しています。 そして、このルールを思い出した後、答えを書いてください：a * c + a * g + b * c + b * g。 素晴らしい。 また、米国では、式自体が教えられることもあり、その式は生徒の歯から反射を遅らせることなく跳ね返ります。 そして、あなたが3つの用語に出くわした場合-自分で出てください。



信じられない？ 次に、二次方程式がどのように解決されるかを教えてください。 最初に思い浮かぶのは、「...の根をプラスまたはマイナスする」ことです。 しかし、これは「解決方法」ではなく、「ルーツとは何か」です。 言い換えれば、私たちは皆、この式を一緒に学んでおり、実際、それを導き出すことも、証明することさえできる人はほとんどいません。 その結論は標準的な学校のコースに含まれていますが。



これらの例を挙げて、知識とは何かを説明しました。 私はそれらをクラミング（CとD）、理解（B）、認識（A）に分けます。 詰め込みは式の知識であり、理解は式とその応用分野の知識であり、認識は導き出す能力です。 平均的なロシア人（私には思える）は、認識レベルで括弧の始まりを知っており、二次方程式の根の公式は、詰め込みと理解の間のどこかにあることがわかります。 しかし、ブラケットを開くことは確かに私たちに詰め込みを教えている人ではありません。



結論は次のとおりです。 2人の学生が金星の問題を解決し、4人で5人の天文学者を得ることができます。 しかし、タスクの設定方法によっては、仕事の価値が根本的に異なる場合があります。 教育省は、「後背地の学生がメガロポリスの学生と同じ知識を受け取り」、「試験で知識が評価された」ことを確認したいと考えています。 紳士...生産を最適化するために、そして足をもっと指で切る人のために、36番目のサイズの靴だけを作るようなものです。 もちろん、私は州の教育レベルを上げることを目指していますが、国の算術平均を下げることではありません。



私も質問をしたい-あなたの周りの人々は二次方程式を解くことができるはずだと思いますか？ 私の意見では、これは、生活の質、正確に、どのように腕立て伏せをするか、BRICS諸国の知識、Eメールの自信を持って所有し、青信号だけに道を渡る必要性の認識を確立します。



知識の幻想的な性質



なぜ詰め込みが悪いのですか？ 現在の教育省は、彼らはそれを必要とせず、少数の人だけが深い知識を必要とし、残りの人に基地を習得させ、十分なものを与えると言うでしょう。 しかし、詰め込みは、教育のコストと相まって、完全な劣化に満ちています。 理由を説明します。 良い学校の先生や小学校の講師を想像してください。 彼は彼の主題についてどのような印象を与えましたか？ 個人的には、この主題が広大であり、多くの未踏の渦を隠していることをまったく知りません。 そして、それは本当です。 今、私は各レッスンの先生に支払うことを想像してください。 彼は、私がこの分野の専門家になりつつあるという幻想を日ごとではなく、時間ごとに作成する必要があります！ さらに、これは詰め込みによって達成されます。 その結果、若者は知識/能力を適切に評価できなくなります。 私の意見では、この現象は米国の膨大な数の学生に影響されます。 そして、私たちは、この病気に苦しむ若者をますます増やしています。
