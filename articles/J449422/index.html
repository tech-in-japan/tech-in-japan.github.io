アラーム！ アラーム！：魚が危険について親relativeに警告する方法





友情は困っている。 私たち全員によく知られているこのフレーズは、野生生物、つまり社会化された動物に関しては生死の問題になります。 また、このフレーズはここで適切です-警告は武装を意味します。 おそらく、ミーアキャットや他の動物の集団に住み、捕食者の攻撃を恐れているセンチネルを見たことがあるでしょう。 危険が生じた場合、ミーアキャットは鳴り、彼の仲間はすぐにミンクに隠れます。 しかし、あなたが魚のように愚かな場合はどうなりますか？ むしろ、あなたが魚なら。 もちろん、体から分泌される化合物を使用してください。 科学者は、必要に応じて、グループ魚のいくつかのタイプが、親族に危険について警告する特定の物質を分泌できることを発見しました。 科学者は、彼らの発見が以前の発見とどのように異なり、魚類社会のコミュニケーションシステムがどれほど複雑であるかをどのくらい正確に発見しましたか？ 研究グループの報告書への回答について。



研究の基礎



あらゆる生物の社会性の主な発現は、コミュニケーション、つまり、1つのグループ内の種の代表者間のコミュニケーションです。 通信を通じて情報が送信されますが、これは非常に明白です。 ここにはたくさんのグッズがあり、そこから危険についての警告を実行しなければならないので、ここから逃げ出すように呼びかけることができます。





危険から身を隠す魚。



大まかに言って、コミュニケーションの理由と送信される情報は異なる生物間で非常に似ていますが、送信方法は異なります：音、光、ジェスチャー、化学物質の割り当てなど。 また、これらの信号の一部は別の種によっては認識されません。これは、近くにいる捕食者についてグループに警告する場合に非常に役立ちます。 人間も、これらの信号の多くに気付きません。 これはまさに、さまざまな種の多くの社会的側面が何十年も研究されていないという事実につながったものです。

研究者は、どの信号システムにもいくつかの主要な要素があることに留意しています：コンテキスト（a）、自発的な信号（b）、受信側の応答（c）、送信機と受信機の利点（d）。



まず、個人は特定のコンテキスト（情報/状況）に直面します。これは、外部（i）要因（食物を見つける、近くの他の種または捕食者の存在）、内部（ii）要因（飢hung、恐怖など）に起因する可能性があります信号の受信機（iii）の近くにあるという事実。 したがって、信号コンテキストは、外部刺激、個人の内部状態、生​​成された信号のターゲット「オーディエンス」、つまり対象となる人から構成されます。 科学者はまた、これらの信号は自発的であり、自分の自由意志の個人によって生成されることに注意します。





シーバスとエビの共生関係。 エビのアンテナの1つは魚と常に接触しており、魚は危険な場合に穴に隠れます。 このような信号を受け取ったエビも隠れます。



すべての信号が重要というわけではないため、この点（信号の自発性）は非常に重要です。 一部の信号は、信号ソースに意識的に関与することなく、コンテキストに応じてランダムに生成されます。 たとえば、負傷者の匂いは、近くに危険があるという他者への信号ですが、この信号は実際には個々の被害者ではなく捕食者によって作成されました。 言い換えれば、それは不随意の信号でした。



次の要素はホストです。 信号は、受信側に何らかの応答を引き起こします。これには、信号の性質に応じて、行動的、生理学的、さらには遺伝的変化が含まれる場合があります。



同様に重要なのは、それを送信する人と受信する人にとっての信号の利点です。 この場合、状況は相互に有益であり、信号を生成する個人のみの利益を目的とします。 相互に有益なシグナルは、重要な生存メカニズムとして使用できるため、世代から世代へと遺伝および/または子孫のトレーニングを通過します。



科学者が認めているように、実際には、信号の強度、その任意またはランダムな生成の性質、および発電機と受信側にとってのその利点を測定することは非常に困難です。 原則として、これらの特性は、事実、つまり、グループの生存度、死亡率と出生率などの推定後に決定されます。 簡単に言えば、グループがシグナルを使用していて、それらが生きていてうまくいっていれば、シグナルはその利点を生かすことができます。 しかし、このアプローチは非常に不正確であり、いくつかの質問に答えることができません。制御された条件下で維持されるグループの行動、子孫に伝わる行動などです。



ハザード信号の送信システムを簡素化すると、2つの主要なコンテキスト要素を区別できます。外部要因（捕食者）と観客（同じ種やグループの代表者）です。 この場合、信号強度は状況によって異なる場合があります。 親relativeが十分に近い場合、信号はより強くなります。 攻撃の犠牲者が1人であるが、親theが遠く離れている場合、捕食者からあまり注意を引かないようにアラーム信号は弱くなります。





グループのカラスは個々よりもはるかに大胆であるため、白頭ワシを攻撃することさえできます。



聴衆、つまり信号を受信する聴衆の構成も重要です。 一部の鳥や霊長類は、同じ種だけでなく一般的なグループ（家族、パートナー、子孫）の代表者が近くにいる場合、危険の信号を増幅します。



海と海洋の住民の議論に目を向けると、科学者は再び自発的信号と不随意的信号を区別することの重要性を思い出します。 例えば、1938年にカール・フォン・フリッシュは、H 3 NOである成分の1つであるシュレックストフ化合物を発見しました。 この化合物は、負傷者または死亡者によって作成されますが、自発的で意図的な警報とは言えません。

魚のシグナルの大部分の主成分は尿であり、これはフェロモンの分離、特定の領域での優位性の証明、関連する個体の識別などに使用されます。 これに基づいて、科学者は、尿が不安のシグナル伝達に重要な役割を果たすことができることを示唆しています。



この研究では、科学者たちは黒い太い頭（Pimephales promelas）を含むテストを実施しました。 このタイプの魚は社会的ライフスタイルをリードします。 危険な場合、彼らは最初に素早くランダムに動き始め、次に凍結し、次に密集したグループに集まり、安全な場所に隠れようとします。



研究者は一連のテストを実施し、捕食者が太った頭を攻撃して、発生したアラームの自発性、ホスト個人（友人/見知らぬ人）の聴衆または近くの聴衆の有無などに応じた信号強度を決定しました。



試験結果



信号発生器として69人、受信者として270人が選択されました。 すべての個人は警報の訓練を受けていませんでしたが、テスト中、彼らは知っている個人または見知らぬ人と水族館にいました。 発電機と受信機の間に特別な生理学的違いはありませんでした：重量-0.6±0.2 g、体長-40±5 mm。



信号の前後のグループ化インデックスは、信号のタイプと個々のジェネレーターのタイプによって大きく異なりました（ 2a ）（双方向の相互作用：LRT x 2 2 = 9.94、p = 0.0069）。

データを信号タイプと発生器タイプで分けることにより、科学者はこれらの変数間の関係をより詳細に理解することができました。 非アラーム信号（正常）は、信号発生器の3つの異なるカテゴリの個人間で差はありませんでした（LRT x 2 2 = 3.33、p = 0.19）が、反対にアラーム（LRT x 2 2 = 10.75、p = 0.0046）。 科学者はまた、個々の信号発生器に精通している個人のグループが生成するアラームにはるかに積極的に応答することに気づきましたが（LRT x 2 1 = 16.86、p <0.0001）、「ストレンジャー」のグループはゆっくりとゆっくり反応します（LRT x 2 1 = 1.28、p = 0.26）。



しびれに関して（ 2b ）、信号のタイプとそのジェネレーターのタイプの間にも関係が見つかりました（カイ2乗検定： x 2 7 = 28.2、p = 0.0002）。 通常の信号の場合、受信グループには、信号発生器のタイプに関係なく、多くの場合麻痺した個人が1人いました。 ただし、アラームでは、グループ化インジケータと同様に違いがありました。 アラームが見知らぬ人（フィッシャーの正確なテスト：p = 0.002）または単一の（孤立した）個人（p = 0.0002）よりも身近な個人から来る場合、受信者はより頻繁に、より多くの数が麻痺します。 見知らぬ人と孤立した個人の信号に対する反応率は非常に似ており、これは同じグループの個人間の社会的つながりの重要性をもう一度確認します。 彼らが言うように、彼らは見知らぬ人よりもはるかに自分自身を信頼しています。



混oticとした動きの指標で、状況はさらに奇妙でした（ 2s ）。 このグループは、どのタイプのジェネレーター（馴染みのある個人、見知らぬ人、孤立した個人）からの正常な信号にも応答しませんでした。 身近な個人からのアラームに対するグループの反応は、アラームや孤立した個人に対するよりもはるかに強かった。

エピローグ



上記のテストを実施した研究者は、アラーム（この場合は化学物質）が、信号を生成する個人と信号を受信する個人の捕食者に対する保護の重要な要素であることを確認しました。 また、負傷または殺害された個人からの不本意な信号とは対照的に、これらの信号は自発的かつ故意に生成されることがわかった。



さらに、この研究では、見知らぬ人や孤立した個人よりも、同じグループの個人によって生成されたアラームにグループがより積極的に応答することが示されました。 この観察は、一般的な種ではなく、特定のグループ内のより強い社会的関係の存在を確認します。



この研究により、水中住民の社会構造とその基本要素をよりよく理解することができます。 野生動物の世界は、特に空腹の敵が多い小魚にとって危険な場所です。 グループを形成することにより、彼らは生存の可能性を高めます。 しかし、グループがそれを取り巻く危険に目がくらんでいれば、ミサの利用はほとんどありません。 そのため、グループのすべてのメンバーの環境意識は、生存のための闘争において非常に重要な役割を果たします。
