Alt LinuxディストリビューションはFSTEC証明書を受け取りました





Alt Linuxは新しいディストリビューションを認定しました。 2011年4月18日、 Alt Linux SPT 6.0配布キットはFSTEC証明書番号2317を受け取りました。



他の認定バージョン（Alt Linuxは2002年から認定バージョンを作成しています）とは異なり、Alt Linux SPT 6.0は、すべてのタイプの個人データ保護システムと状態の秘密に使用できます。



Alt Linux SPT 6.0には次のものがあります。

  • 不正アクセスからのコンピューター機器のグレード4保護
  • 宣言されていない機会がないことの3レベルの制御。


次の開発に使用できます。

  • 1Vのセキュリティクラスを含む自動システム
  • K1の個人データ保護システム。


Alt Linux SPT 6.0は、情報セキュリティツールが組み込まれたサーバーおよびワークステーション向けの統合ディストリビューションです。



ワークステーションは GNOME環境に基づいており、アプリケーションの標準セット（OpenOffice.org、Firefox、グラフィックスおよびマルチメディアプログラム、周辺機器などを含む幅広い機器のドライバー）が含まれています。



このサーバーは、従来のサーバーのすべての機能と最新のテクノロジーを組み合わせて、便利なWebインターフェースで管理します。 重要な機能は、SELinuxによるアクセス制御の資格情報の原則の提供です。 ディストリビューションには、強力なBaculaエンタープライズレベルのバックアップシステムも含まれています。 間違いなく、暗号通貨の大手メーカーとの互換性は、認定システムにとって重要です。 ワークステーションの認証と更新は、Webインターフェイスを介して一元的に実行されます。



ソフトウェアに付属する配布ソフトウェアのバージョンは古くはありませんが、最新のものではありませんが、認証は高価なプロセスであるだけでなく、非常に長い（6か月から1年）ため、イメージの最終バージョンは認証終了の4〜5か月前。



しかし、カーネルバージョンは最新のものであり、現在、主要なLinuxディストリビューションのすべてのエンタープライズバージョンで使用されています。

認定キットには、証明書の認定コピー、フォームフォーム、Fstackステッカー、および一意の番号の付いたディスク（32ビットおよび64ビットの2枚のDVD）に加えて、主に無料で定期的な認定アップデートを受け取るために必要なテクニカルサポートクーポンが含まれていますFSTECの条件に従って、ユーザーはシステムで使用する必要があります。



連邦法第152号「個人データについて」の最終的な発効に関連して、認定ソフトウェアを選択するというトピックはこれまで以上に重要になっています。 第25条に従って、2011年1月1日より前に作成された個人データ情報システムは、2011年7月7日までに連邦法No. 152-FZの要件に準拠する必要があります。
