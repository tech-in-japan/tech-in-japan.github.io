HP iPaq 614（610）。 人々への現象。



HPの最新のGPSコミュニケーターの代表者が明らかになりました！ この小さなことは何に興味がありますか？

（カット写真の下）



このモデルは、HPの一種のデビューです。これは、これまでにGPSモノブロックにテンキーを同時に表示したことがないためです。 最初に目を引くのは、真面目で独創的なクラシックでスタイリッシュなデザインです。 2つ目は、キーボードにあるスマートタッチです。 実際、iPodセンサーの類似物ですが、PDAでどのように使用されるかはまだ不明です。



520MHzの周波数を備えたMarvell PXA27xプロセッサと128MBのRAMの組み合わせという、わずかに異なる観点から注目に値します。 その結果、Asus p535にインストールされているIntel XScale 520 MHzプロセッサとまったく同じプロセッサがあり、 モバイルレビューによると、現在世界で最も強力なコミュニケータです。 ただし、p535には64 MBのRAMがあり、HP iPaqのコミュニケーターには2倍のRAMがあることを考慮する価値があります。



Windows Mobile 6はデバイスにインストールされており、最新の通信規格であるGSMのほぼすべてをサポートしています。 GPRS / EDGEクラス10; UMTS / HSDPA; Bluetooth 2.0 EDR Wi-Fi 802.11 b / gおよびA-GPSをサポートするGPSレシーバー。

microSDスロットが要求されますが、ほとんどの場合SDHCサポートはありません。 TFTスクリーンのサイズは2.8インチ、240 x 320ピクセル、65,536色です。

オートフォーカス付きの3 MPxカメラ（iPaq 614cバージョンでは、残りの1 MPx）も利用可能です。



現時点では、米国の価格は600ドルであり、ロシアへの輸入については何も知られていない。
