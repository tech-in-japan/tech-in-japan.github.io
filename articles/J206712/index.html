フリーソフトウェア財団が最初のラップトップを承認

Free Software Foundation（FSF）は、 Respect Your Freedomの 基準を満たす最初のラップトップを承認しました。 「初めて、販売されている形でラップトップを購入して使用することを人々に勧めることができます」とFSFは言いました。



FSF認定プログラムの主な基準は、BIOSおよびさまざまなコンポーネントのファームウェアを含むハードウェアデバイスの制御プログラムを含む100％フリーソフトウェアです。 ユーザーは、これが可能なすべてのレベルでソフトウェアを変更および制御できる必要があります。 ソースコードの公開とともに、製造業者は、すべてのレベルで代替ソフトウェアをインストールする実際的な機会をユーザーに提供する必要があります。



ソフトウェアだけでなく、100％無料で一般にアクセス可能なものは、このソフトウェアをコンパイルするためのツール、およびデバイスをインストールおよび保守するためのソフトウェアでなければなりません。 デバイスには、1）ユーザーが明示的に要求した場合、2 ）通信プロトコルで必要な場合、または3）法律。 後者の場合、デバイスの所有者には、注目すべきメッセージと追加情報の入手先へのリンクを通知する必要があります。



FSFが推奨するデバイスは、MP3などの独自のファイル形式をサポートしますが、無料の代替がサポートされている場合のみです。 DRMを使用した形式は、フリーソフトウェアで処理される場合にのみサポートできます。



上記のすべては、最初の完全に無料のGluglug X60ラップトップに実装された基準の不完全なリストです。











Gluglug X60は、IBM ThinkPad X60ラップトップ（2006リリース）の修正版ですが、無料のCoreboot BIOS、無料のTrisquel GNU / Linux OS、ドライバー、およびアプリケーションを備えています。 さらに、フリーソフトウェア標準と完全に互換性のある802.11nネットワークカード（Atheros Atheros AR5B195）を統合します。 製造会社は、各モジュールのドキュメントとソースコードを提供します。



Trisquel OSは、独自のプログラムとファームウェアオプションを削除した後、FSFを使用してUbuntuに基づいて作成されます。



ディスプレイの対角線が12.1インチの無料ノートブックには、最小構成で1 GBのRAMと60 GBのハードドライブがあり、最大3つの構成-3 GBのRAMと128 GB SSDです。プロセッサのうち、Intel Core Duo、Intel Core Solo、Intelが使用されますCore 2 Duo。ただし、このようなデバイスの技術仕様はそれほど重要ではなく、公式Webサイトの製品説明にも記載されていません英国のメーカーでは、ノートパソコンの価格は200ポンド（約328ドル）です。
