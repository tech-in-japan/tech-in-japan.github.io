According to the documentation, the platform supports connecting up to 1,632 disks, and the maximum capacity of each storage system is 19.5 petabytes (19,584 TB).

High performance is achieved thanks to a duplicate coherent non-volatile cache up to 2 TB with a delay time of 200 ns and hardware accelerated data processing using a microcontroller based on the MIPS64 architecture.
