Microsoftが.Net Framework 3.0をリリース

Microsoftは 、開発者向けに.Net Framework 3.0パッケージを公開しています。 製品のプレゼンテーションは、 Visual Studio Connectionsカンファレンスの前夜に行われました。



以前はWinFXと呼ばれていた3番目の.Net Frameworkには、Windows Presentation Foundation（以前のAvalon）サブシステムとWindows Communication Foundation（以前のIndigo）Webサービスプラットフォーム、および認証プロセスを管理するためのWindows Workflow FoundationとWindows CardSpaceが含まれています。



Platform .Net Framework 3.0はWindows Vistaオペレーティングシステムの一部であり、ユーザーは来年の初めにこれを知ることができます。
