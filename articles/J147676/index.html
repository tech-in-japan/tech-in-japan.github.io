HP ProLiant DL160 Gen8-自分でやる

Habréでは、 HP社のブログにHP DL380 G8の概要が既に記載されていましたが、このサーバーの弟であるDL160 G8は何とかバイパスされました。



先日、クライアントに大量のRAMと少数のディスクを備えたサーバーを提供するタスクがありました。 クライアントの技術仕様に従って、サーバーは10〜15ギガビット/秒の速度でインターネットにリターンを提供する必要があります。



通常、このようなタスクのために、サイズが2Uで、HDD / SSD用に25ガイのHP DL180 G6サーバーを購入しました。 しかし、ここではタスクがわずかに異なっていたため、新しい第8世代のHP ProLiant DL160を試すことにしました。



HP DL160 G8に関するいくつかの有用な詳細
  • DDR-IIIメモリ用の24スロット。
  • 最大384Gb RAMのサポート。
  • 8 x 2.5インチHDD / SSD;
  • 新しいP420コントローラー（1GbメモリとBBU）;
  • PCIeスロットを占有せずに追加のNICモジュール（たとえば、10Gbpsポート）をインストールする機能。
  • フルサイズのPCIe X16スロット。


このモデルに関する詳細な情報はメーカーのウェブサイトで見つけることができますが、私は、目利きを喜ばせることができる新鮮な鉄の写真を共有したいだけです。



私たちは箱を開けて、見て...何がもたらされたのですか？ これがそのようなDL160 G8です！ DL160 G5との違いは大きいです。 DL160ラインは、より高価なDL380ラインでますます伸びています。 たとえば、HDDへのアクセスを制限する保護パネルの場所がすでにあります。









ディスクなしで完全なセットを注文する場合は、1つのスタブなしでサーバーが到着する準備をしてください。 ディスクレスサーバーのファン（たとえば、VMwareクラスターの場合）は、別のサーバーを購入してスタブに解析する必要があります。 サーバーの前面には、モニター用の出力、2つのUSBポート、電源ボタン、多くの愛用のUIDによるLEDボタンもあります。







ディスクには特別な注意を払う必要があります。 多くが変更されました。 空のコンパートメントのスタブも変更されました。







これで、LEDはディスクバスケット自体に配置されました。 背面には、標準の電源コネクタとSASに加えて、バスケットコンタクトの追加グループがありました。 ディスクはモデルによって異なります。 たとえば、購入したドライブはHP SmartDriveキャリアです。 ドキュメントから判明したように、これらのディスク（またはバスケット？）には、飛行機のブラックボックスのようにディスクからテレメトリーデータを収集し、ディスクの信頼性を検証する追加のマイクロコントローラーがあります。







最後に外部検査を終えて、内部を見るために登りました。 サーバーは、他のHPサーバーと同様に開きます。 たとえば、DL360など。 閉じている場合は、サーバーの背面にドライバーキーがあります。







私が最初に目を引いたのは、BBU SASコントローラーがHDDおよびDVDドライブのコンパートメントの直後にあるフロントファンのグループの前に配置されたことです。







彼がいる。 決定は非常に正しいです。 Intelサーバーの一部のモデルでは、コントローラーのBBUは最も高温の防風場所にあり、BBUの過熱とその障害の原因になります。







ファンは固定されていませんが、セルに座っているだけです。 ツールや労力なしで取り外し可能。







一般的な精度のなかで、いくつかの不注意にレイアウトされ、何らかの形でランダムに配置されたファン電源ケーブルがノックアウトされています。







そこに彼女は美人だ！ DDR-IIIメモリモジュール用の24スロットとCPU用の2スロット。 透明なダクトには、メモリスロットの番号が付いています。 もちろん、内側のサーバーカバーにはメモリモジュールのレイアウトもありますが、より便利で明確であることに同意する必要があります。







前の写真でわかるように、CPUは次々と配置されています。 これは明らかにプロセッサーの冷却には役立ちません。 しかし、HPのエンジニアはラジエーターを変更しました。 現在、「フロント」CPUヒートシンクの表面積は小さくなり、「バック」プロセッサーヒートシンクの「ヒートシンク」の頻度ははるかに高くなり、それに応じて表面積が大きくなります。







内部のロープロファイルPCIe X8スロットはすでにHP P420コントローラーによって占有されていますが、必要に応じて他のものと交換できます。







フルサイズのPCIe X16スロットが空です。 2 x 10Gbit /秒のSFP +ネットワークカードを取り付けました。 このスロットを使用することはできませんでしたが、ネットワークカードの下には、ネットワークポートを備えた特別なカードを取り付ける場所があります。 1 Gbit / sおよび10 Gbit / sのポートをさらにいくつか配置できます。







この情報が誰かにとって興味深いものであることが判明した場合、私たちは喜んでおり、それが有用であれば、さらに喜んでいるでしょう。
