OpenSource.Mail.Ru：従業員のオープンプロジェクト





オープンソースの動きの根底にある主要な原則の1つであるオープンソースは、情報の自由です。 このアイデアの実行可能性は、世界中のプログラマーの無料コミュニティによって開発された多くの成功したプロジェクトの例によって確認されています。 これはおそらく誰かを驚かせるでしょうが、多くのオープンソースプロジェクトを積極的に開発しています。 そして最近、これらのプロジェクトに関する情報がネットワーク全体に散らばっていることを発見し、一部の製品の開発者はそれらに役立つ可能性のある他の製品を完全に知らない可能性があります。 そこで、情報サイトOpenSource.Mail.Ruを作成するというアイデアが生まれました。



現在、リソースの開発と新しいプロジェクトの追加を続けています。 コンテンツは開発者自身によって記入されます。 ほとんどの読者は、少なくとも私たちの最大のオープンソースプロジェクトについて聞いたことがあると思います。

これらのよく知られている「モンスター」に加えて、当社の従業員は、はるかに少ないプロジェクトを開発しています。 OpenSource.Mail.Ruで次のように表示されます。



  • Centrifugoは、 WebsocketまたはSockJSを使用したクライアント接続をサポートするリアルタイムメッセージングサーバーです。 チャット、通知、カウンター、ゲームなど、顧客へのメッセージの即時配信を必要とするアプリケーションを最小限の労力で実装できます。 サーバーはGoで記述されており、ブラウザーまたは他の環境から数千の同時接続に対応できます。

    Centrifuge + Go = Centrifugo-より硬く、より良く、より速く、より強く
  • Festは、XMLテンプレートを自己完結型のJavaScript関数にコンパイルする汎用テンプレートエンジンです。 XMLはテンプレートの作成に使用されます。 このマークアップ言語が存在してから長年にわたって作成された一般的なIDEと多くのツールを使用できます。
  • ファイルAPIは、クライアント上のファイルを処理し、それらをサーバーにアップロードするように設計されたライブラリです。 アプリケーションの主な分野は、画像の読み込みと編集です。
  • Mail.Ru Agent NotificationsはJIRAのプラグインであり、タスクの変更に関する通知をMail.Ru Agentに送信したり、タスクの操作に便利です。
  • Mail.Ru CalendarはJIRAのプラグインであり、JIRAのリクエストに基づいて仕事、休日、その他のイベントを視覚的に追跡できます。

    オープンソースJIRAのカレンダー
  • Sophiaは、LSMツリーの欠点に対応するために設計された新しいデータストレージアーキテクチャを使用して構築された組み込みのSophiaトランザクションKey-Value DBMSです。
  • VaOceanMobileは、モバイルデバイス専用に設計されたUnreal Engine 4用のオーシャンシェーダーです。 このマテリアルは、PBRシェーディングとダイナミックライティングを完全にサポートしています。
  • VaRestは、RESTアーキテクチャを操作するために特別に作成されたUnreal Engine 4エンジン用のプラグインです。 このプラグインを使用すると、HTTP / HTTPSリクエストを介したサーバーとの通信が大幅に簡素化され、リクエストを管理するためにプログラミング言語の特別な知識は必要ありません。
