Android Studioでプロジェクトを作成するときのGradleエラー

みなさんこんにちは！



Android Studio環境で新しいプロジェクトを作成する際に問題に直面しました（記事バージョン0.3.2の執筆時点）。

オペレーティングシステムWin 7x64、Android SDK v 22.2.1。

プロジェクトは、 「http://repo1.maven.org/maven2/com/android/tools/build/gradle/」を取得できませんでした。 サーバーからステータスコード407を受信しました：プロキシ認証が必要です



すぐに言いたい ネットワークはプロキシの下にあり、Android Studio環境のHttp Proxy設定でこのプロキシが指定されています。

問題を探してインターネット全体を調べました。 プロキシもGradle用に設定する必要があることが判明しました。

プロジェクトを作成するには、フォルダーC：\ Users \ <Your account> \ 。Gradleにgradle.propertiesファイルを作成する必要があります。

英語のsinアカウントには近づかないでください。 一部のユーティリティは、キリル文字が使用されているパスを認識しない場合があります。

ファイルで、適切なフィールドを作成して入力する必要があります。



systemProp.http.proxyHost =

systemProp.http.proxyPort =

systemProp.http.proxyUser =

systemProp.http.proxyPassword =

systemProp.http.nonProxyHosts =
