ブラウザーの拡張機能-任意のサイトの情報を検索

多くのサイトでは、サイトでの検索が機能しないか、非常にうまく機能しないため、必要な情報を見つけるのは困難です。

私のブラウザー拡張機能はこの問題を解決します-人気の検索エンジン（Yandex、Googleなど）を使用して、任意のサイトの情報を検索できます。







アイデアは簡単です-特別な検索エンジン演算子を使用して、現在のサイトの情報を検索します（アクティブなウィンドウ/タブで開きます）。



2010年にこれらの拡張機能の開発を開始しました。 最初はFireFoxでした。 その後、 Operaが追加されました。 最後の拡張機能は、Google Chrome / Yandex.Browser用に作成されました。



FireFoxアドオンディレクトリからインストールします 。



Opera拡張機能ディレクトリ （「クラシック」Opera 12を含む）からインストールします。



Chromeオンラインストア （Yandex.Browserを含む）からインストールします。



次の2つの使用方法があります。

  1. 検索フォームに検索クエリを入力します（FireFoxの場合-標準検索フォーム、Cntrl + Enter、その他すべての場合、ポップアップ検索フォームおよびEnter）
  2. コンテキストメニューを検索します。 ページ上のテキストの断片を選択->右クリック-> "このサイトで検索..."




テキストを入力すると、検索エンジンが開き、探しているテキストがある現在の（開いている）サイトのページのリストが表示されます。



拡張機能は、検索エンジンYandex、Google、Bing、Yahoo、Baiduで動作するように構成されています。 検索フォームまたは拡張機能設定ページで検索エンジンを選択できます。 Googleインスタント検索をオフにして、新しいウィンドウを開くこともできます。



よくある質問への回答：

  1. いいえ、これはCtrl + Fではありません。 Ctrl + F-現在のページで検索します。 拡張機能-サイト全体を検索します。
  2. はい、多くの人が「site：site.ru」について知っています。 これは手作りです。 拡張機能-作業の自動化、つまり 時間と労力を節約できます。
