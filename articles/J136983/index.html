オイルラッシュ：公式リリース



OilRushは、トムスクのロシアの開発者によるリアルタイム戦略です。

本日、Linux、Mac OS X、およびWindowsのバージョンがリリースされました。

このプロジェクトがLinux用にリリースされた最初のテクノロジーゲームの1つであることは注目に値します。



このゲームは、リアルタイム戦略とタワーウォーズのジャンルの混合物です。海上および空中から敵のプラットフォームを同時に攻撃しながら、オフショアプラットフォームの開発を制御します：石油生産と生産、および防御構造を作成することによる防衛。 さまざまな戦略が、開発されたテクノロジーツリーによって作成されます。



設定により、このゲームは映画「ウォーターワールド」に似ています。これは、利用可能なリソースの残党をめぐって激しい生存者が激怒する世界的な大惨事の後の洪水の世界です。



機能：

  • DRMの欠如
  • ストーリーキャンペーン
  • ネットワーク上のゲーム（LAN、インターネット）
  • ロシア語版と英語版の存在
  • 32ビットおよび64ビットディストリビューション用のdebおよびrpmパッケージの存在
  • SteamおよびDesuraでアクティブ化する機能*
  • 高品質のグラフィック。 DirectX 11およびOpenGL 4.0


ゲームは、Microsoft Windows、Linux、Mac OS X、PlayStation 3、Androidプラットフォームで使用可能なUnigineエンジンで開発されています。

サポートされているテクノロジー：ハードウェアテッセレーション、SSAO、SSDO、DirectCompute、OpenCL 1.1およびShader Model 5.0。



ゲームはUnigine Store、Desura、およびUbuntu Software Centerで購入できます。

推奨される購入方法は、デベロッパーストアを通じてです。

*購入すると、SteamおよびDesuraでアクティベーションキーを確実に取得できます。

これまでのところ、開発者はDesura / Steamでゲームを購入したユーザーが他の場所でゲームをアクティブにできるかどうかを報告していません。
