WindowsおよびOfficeの新機能



電話アプリ



メッセージを書くために電話を探す必要はもうありません。 そして最後に、メールで写真を送信するのをやめることができます。 新しいYour Phoneアプリを使用すると、Androidスマートフォンの写真やコンピューターのテキストにすぐにアクセスできます。 Androidフォンで写真を撮って、PCで見ます。 電話からPCに写真をドラッグアンドドロップすることもできます。 また、写真をPCから直接コピー、編集、描画することもできます。 お友達やグループにパーソナライズされたメッセージを書いて、キーボードで入力します-電話アプリケーションを使用すると、PC上のAndroidフォンからSMSを表示、送受信できます。 Android OS 7.0以降は、お使いの電話アプリと互換性があります。 このアプリは、iPhoneとPCのペアリングにも役立ちます。 以下は、字幕のロシア語への翻訳を有効にできるデモビデオです。







IPhoneユーザーは、電話でWebサイトを閲覧し、選択したページを即座にコンピューターに送信して、中断したところから開始できます。 その後、大画面のすべての利点を使用して、コンテンツを読んだり、視聴したり、検索したりできます。 電話が接続されたら、PCで作業を続けることはあなたからわずか1歩先です。







携帯電話のコンテンツにすばやくアクセスするには、携帯電話アプリケーションをデスクトップまたはタスクバーに接続します。 また、[スタート]メニューのすべてのアプリケーションのリストで見つけることも、Windowsキーを押して検索バーに電話アプリケーションの名前を入力することもできます。



電話のタイムライン



4月にWindows 10 PC用のタイムラインを導入しましたが、このアプリケーションの目的は、見つからないドキュメントを探してメールやファイルフォルダーを調べることで発生するフラストレーションを回避することです。 タイムラインを使用すると、すべてのデバイスで重要なものを簡単に見つけることができます。 また、目的のアクションを選択することもできます-iOSまたはAndroidデバイス上であっても、Windows PCに戻る準備ができたら、それらを選択します。







これで、PCのタイムラインアプリケーションが携帯電話で使用できるようになり、携帯電話、コンピューター、またはタブレットで使用したファイルやWebサイトを過去にさかのぼって見つけることができます。 いつでも中断したところに戻ることができます。 スクロールを使用して、今日または今週始めに作業したファイルを簡単に見つけ出します。 いつでも特定の日または時間からアクションを共有または削除できます。 マイクロソフトランチャーアプリから本日よりAndroidの試用版を入手できます。iPhoneの試用版はまもなく入手可能になります。







PowerPointおよびWordでのインクと3D



PowerPointはストーリーを視覚的に伝えるための強力なツールであり、Wordはアイデアを整理してコラボレーションするためのキャンバスです。 コンセプトの形成、教育、記事の執筆のいずれにおいても、クリエイティブなストリームを見つけて美しいコンテンツを作成するのに役立つ新しいInkingおよび3DベースのAI機能により、今日のアイデアをより簡単に実証できます。



  • 最初にアイデアをスケッチすることを好む人もいます。 これを考慮して、アイデアをスケッチできるようになりました。人工知能（AI）を使用して、それらを完全にフォーマットされたスライドに変換します。 手書きリストは完全にフォーマットされたテキストに変わります。 または、フローチャートまたは図を描く場合、単語を書いてフォームを描くことができ、PowerPointはそれらを即座にテキストおよび固定フォームに変換します。
  • AIを搭載したPower Point Designerがよりスマートになっています。 デザイナーは、手書きの素材に基づいてスライドデザインを推奨できるようになりました。 または、タイピングを好む場合、Designerは単純なテキストブロックを取り、AIを使用してコンテキストを理解し、アイコンとスマートアートを使用してアイデアを提案できます。 さらに多くの設計オプションを提供するために、350の新しいアイコンをライブラリに追加しています。
  • 3DモデルのインラインアニメーションをPowerPointおよびWordに追加しています。 新しい3Dアニメーション効果により、スライドやドキュメントにモーションやアニメーションをすばやく簡単に追加できます。 さらに創造的な柔軟性を提供するために、30を超える新しいアニメーション3Dモデルも追加しました。
  • デジタルペンを使用してWord文書を編集する柔軟性が気に入っている場合は、良いニュースもあります。 Ink Editorを使用すると、テキストの削除や強調表示などの操作を既に実行できます。 今日は、改行の追加、新しい単語の挿入、単語の分割と結合など、ドキュメントの編集を最初から最後まで簡単にする新しいペンの動きを追加しています。






過去数年にわたり、Officeを変革し、AIを使用して日常のタスクを簡素化するという使命を果たしてきました。 6月に、Officeアプリケーションの新しい新鮮なデザインを実演し、 先月 、ExcelとPowerPointの新しいAIベースの機能を発表しました。 他に何が表示されるかをお見せします。



ほとんどのPowerPointおよびWordの更新プログラムは、現在Windows上のOffice 365サブスクライバーが利用できます。 デザイナー機能へのインクは、次の月に表示されます。 3D埋め込みアニメーションは、最初にOffice Insiderメンバーが利用できるようになり、今後数か月で広く利用できるようになります。 スマートアートのテキストとアイコンは、PowerPoint OnlineとMacでも使用できます。



Microsoft To – Do



To-Doスケジューラアプリは、その日の最も重要なタスクの管理、優先順位付け、および完了に役立ちます。 最も便利なモードを使用してタスクを記録できるように、本日はインク機能を使用してタスクを更新する機会をご紹介します。 デジタルペンとタッチスクリーンを備えたWindowsベースのデバイスを使用して、インク機能を使用してリストにタスクを追加し、完了したらそれを消します。 Microsoftアカウントを使用してアプリケーションにログインすると、タスクはiOS、AndroidまたはWindowsのTo-Doアプリケーション、およびインターネットと同期されます。







Outlook.com



1年以上前に、Outlook.comを更新して、数百万人のユーザーに、よりシンプルでスマートで、よりパーソナライズされたメールボックスを提供しました。 本日、タスクの進行状況を監視し、お気に入りのブランドとの対話を改善するための更新プログラムをさらに2つご紹介します。



  • Outlook.comの新しい仕事機能は、仕事を中断したりメールボックスを離れることなく仕事を管理するのに役立ちます。 To-Doアプリケーションの機能を使用して、文字をタスクリストにドラッグすることでタスクを作成できます。 または、簡単にスケジュールを設定して、タスクをカレンダーにドラッグします。 その後、タスクはTo-Doアプリケーションに関連付けられます。
  • 受信トレイにはブランドのオファーやニュースがいっぱいですか？ 本日、Outlook.comの新機能をリリースし、情報のノイズを乗り越え、お気に入りのブランドとのやり取りを容易にします。 ブランドは、受信ボックスで「正当な」メッセージを簡単に識別できる確認アイコンを受信できるようになりました。 新しいプロファイルカードを使用すると、連絡先情報、配信データ、予約、店舗の住所などにすばやくアクセスできます。 また、新しいサブスクリプションポータルを使用して、1回クリックするだけで、購読しているニュースレターと購読解除しているニュースレターを簡単に確認できます。 そして最後に、受信トレイでのオファーの検索を簡素化しました。メッセージリストでオファーの詳細を直接表示できるため、レターを開く必要がありません。 これらの新機能は、今後数か月以内にOutlook.comユーザーが利用できるようになります。






Snip＆Sketchサービス（以前のスクリーンスケッチ）は、アプリケーションの形式になりました。



Microsoft StoreでSnip＆Sketch（以前はScreen Sketchと呼ばれるWindows Ink Workspaceの一部として導入されていました）を更新できます。 Alt + Tabを押すと、リストに表示されます。 また、ウィンドウサイズを設定することもできます。アプリケーションは複数のウィンドウもサポートします。



あなたから一歩離れたすべてのシンプルなカット。 スクリーンショットをすばやくカットして共有できるようにしたいと聞きましたが、それが可能になりました！ Windowsキー+ Shift + Sを押すと、カットツールバーに移動します。これを使用して、長方形、ゆるい形状、またはフルスクリーンをカットし、このカットフラグメントをクリップボードに入れます。 必要な場合は、そこから入手できます。



しかし、もっと必要だとします。 切断後すぐに、Snip＆Sketchアプリケーションに移動するためのオファーが表示されます。このアプリケーションでは、注釈を追加してコンテンツを共有できます。



しかし、もう一つあります！ Windowsキー+ Shift + Sキーボードショートカットが長すぎて覚えられませんか？ 導入オプションを追加しました：



  • ペンでボタンだけ。 デジタルペンをお持ちの場合は、[ペンとWindowsインクの設定]に移動します。画面の切り取りをワンクリックで実行できることがわかります。 これにより、下のカットセクションに直接移動できます。

  • コントロールセンターのクイックアクションボタンを押して、「画面を切る」という碑文を付けます。 システムを更新するとすぐに表示されます。突然ない場合は、通知センターでいつでも有効にできます。

Microsoft Edgeの全般的な改善



読みやすさを改善する方法、Microsoft Edgeの外観/デザインを改善する方法、カスタマイズのオプションがあります。



  • 学習ツール：Microsoft Edgeは、集中できるようにMicrosoft学習ツールが組み込まれた唯一のブラウザーです。

  • フォーカス：ラインカードのように、この機能はテキストの行を強調表示し、焦点を合わせて注意を払うのに役立ちます。
  • オフライン辞書：単語を強調表示すると、定義が自動的に表示されます。






学習ツールはさらに多様化しています。リーディングフォームで利用できる学習ツールの下で、トピックを簡単に追加できます。 あなたにとって最も快適なテーマの色を選択してください。











文法チェックツール：単語を音節に分割し、名詞、動詞、形容詞などの品詞を強調表示できるようになりました。 音声の一部を強調表示するだけでなく、色を変更したり、選択した領域のすぐ上にあるインジケータをオンにして、音声の一部を識別しやすくしたりできます。







線形フォーカス：1行、3行、または5行を強調表示することにより、読み取りフォーカスを改善します。







プリファレンスを設定すると、ページを下にスクロールでき、読むにつれてフォーカスラインが移動します。







Webページ、TIF、PDFファイルの閲覧ビューで単語の定義を探します。新しい辞書機能が追加され、オンラインでも記事や文書を読んでいるときにキーワードの定義を見つけることができます。 少なくとも1つの単語を選択すると、選択した行の上に表示される定義が表示されます。







単語を読み上げて正しい発音を聞いたり、定義からより多くの情報を取得したりすることもできます。 また、Microsoft Edgeの設定メニューの[全般]タブで、定義を表示するかどうか、および定義の内容の種類を選択することもできます。



Microsoft Edge PDFの機能強化



ツールバーオプションの改善：画面の上部にマウスを移動したときにPDFツールバーを呼び出すことができるようになりました。つまり、ツールにアクセスしやすくなりました。 Microsoft EdgeでPDFドキュメントを含むツールバーが開きます。 現在、アイコンのテキスト説明が表示されるため、アイコンをすばやく理解できます。 メモの追加などの新機能により、必要なときにこれらの便利なツールを使用できます。







PDFツールバーのアタッチ/デタッチ：ツールバーはドキュメントの上にアタッチでき、必要に応じてすべてのPDFツールに簡単にアクセスできます。 ツールバーを切り離して、読書に没頭することもできます。 ツールバーが切り離されていない場合、ツールバーは非表示になります-マウスカーソルをページの最上部に移動するだけで戻ります。







レンダリングの改善：PDFファイルを進むホワイトページが少なくなりました。



新しいデザインの改善：Microsoft EdgeがFluent Design Systemを使用して、より自然で便利なユーザーエクスペリエンスを実現していることを発見し続けると、わずかなパネル変更に気付くかもしれません。 2018年10月の更新では、アクティブなタブを強調表示する新しい深度効果を追加します。



改訂されたメニューデザインとオプション：Microsoft Edgeの設定が1ページでは複雑になりすぎているという意見を聞きました。 このリリースでは、[設定]のナビゲーションを簡素化し、頻繁に実行されるアクションを中央に移動し、独自の設定に追加のオプションを提供します。 メニューオプションはグループごとに整理され、それぞれにアイコンとキーの組み合わせが（必要な場合）あります。 Microsoft Edgeの右上隅にある[...]ボタンをクリックして、最新情報を確認します。



Microsoft Edgeのツールバーをカスタマイズします。ツールバーの[...]をクリックすると、「新しいウィンドウ」や「新しいプライベートウィンドウ」など、メインコマンドを中央に配置する変更メニューが表示されます。 また、Microsoft Edgeツールバーに表示するアイコンをカスタマイズする機能を追加しました。ビューをより正確にするためにそれらをすべて削除したり、お気に入りの機能をすぐに使用できるように必要な数だけ追加したりできます。







ジャンプリストでキーファイルを確認する：Windowsタスクバーまたは[スタート]メニューのジャンプリストでお気に入りのサイトを確認できるようになりました。 Microsoft Edgeアイコンを右クリックして、最も訪問されたサイトのリストを表示します。 特定のテキストの右ボタンをクリックして、リストから削除します。







[ダウンロード]パネルをさらに活用：[ダウンロード]パネルでファイルを右クリックすると開くメニューに[フォルダーに表示]および[リンクをコピー]オプションを追加しました。







最後（ただし重要ではない）：10月の更新では、設定はカテゴリ別にサブページに分割され、明確に整理されたオプションと各ページの混乱が少なくなります。 特にユーザーの利便性のために、Microsoft Edgeの新しい設定機能が登場しました。







Microsoft Edgeプラットフォームおよびエンタープライズ管理の更新





Web認証API：Microsoft Edgeには、新しいWeb認証API （またはWebAuthN ）の比類のないサポートが含まれるようになりました 。 Web認証は、デバイスに関連付けられたより安全な資格情報でパスワードを置き換える、オープンでスケーラブルで相互運用可能な認証簡素化ソリューションです。 このプログラミングインターフェイスをMicrosoft Edgeに実装すると、Windows Hello（PINまたは生体認証データ）およびFIDO2またはFIDO U2Fセキュリティキーなどの外部認証子を使用して、Webサイトでの安全な認証を行うことができます。







Microsoft WebDriverはオンデマンド機能になりました。MicrosoftWebDriverをオンデマンド機能（Feature on Demand、FoD）にしたため、デバイスに最適なバージョンを簡単に入手できるようになりました。 WebDriverをインストールするには、開発者モードを有効にするか、[設定]> [アプリケーション]> [アプリと機能]> [オプション機能の管理]に移動し、別のアプリケーションとしてインストールします（オプション機能の管理とスタンドアロンとしてのインストール）。



新しいMicrosoft Edgeグループポリシー：Microsoft Edgeチームは、IT管理者がMicrosoft Edgeを管理するための新しいグループポリシーとMDM設定を導入しました。 新しいポリシーには、フルスクリーンモードのオン/オフ、印刷、お気に入りパネル、ダウンロード履歴が含まれます。 証明書エラーの回避を防止します。 ホームボタンと起動オプションを構成する クイックアクセスページとURLボタン「ホーム」の設定と権限の制御。 新しいMicrosoft Edgeポリシーの詳細をご覧ください。



ユーザーはWebサイトでのビデオ再生を制御できます。2018年10月の更新プログラムでは、Webサイトでビデオの再生を自動的に許可または禁止する新しい設定をMicrosoft Edgeに追加しました。 この設定は、[詳細設定]> [メディアの自動再生]セクションにあり、好みに応じて動作をカスタマイズできます。







[許可]オプションが選択されている場合、アクティブなタブから最初にアクセスすると、サイトでビデオを再生できます。 オプション「制限」（制限）を選択すると、サイトは音声なしでビデオを自動的に再生できるため、突然オンになりません。 ユーザーがページの任意の部分をクリックすると、タブが開いている間にこのドメインの自動再生が再び開始されます。 [ブロック]オプションを選択した場合、ビデオコンテンツを自分でオンにしない限り、サイトはビデオを自動的に再生できません。



ユーザーはいつでも自動再生を有効または無効にしたり、アドレスバーの左側にある情報アイコンまたは南京錠アイコンをクリックして「アクセス許可の管理」を選択することで、各サイトの自動再生許可を個別に変更できます。



XSSフィルターの削除：2018年10月の更新プログラムでMicrosoft EdgeのXSSフィルターを削除します。 ユーザーは、最新のブラウザとの高度な互換性を備えたコンテンツの導入により、攻撃から保護するためのより強力で効果的かつ安全なメカニズムを提供するコンテンツセキュリティポリシーなどの最新の標準により、引き続き保護されます 。

Windows Defender Firewallは、Linux Processes（WSL）用のWindows Subsystemをサポートするようになりました



Windowsプロセスの場合と同様に、Windows DefenderファイアウォールでWSLプロセスに特定のルールを追加できます。 さらに、Windows Defenderファイアウォールは、WSLプロセスの通知をサポートするようになりました。 特に、Linuxツールが外部からポートへのアクセスを許可する場合（たとえば、SSHまたはnginxなどのWebサーバー）、Windows Defenderファイアウォールは、ポートが接続の受け入れを開始するとすぐに、Windowsプロセスと同じ方法でアクセスを許可します。



安全なプロセスとしての起動が必要なウイルス対策製品



Windowsセキュリティセンター（WSC）サービスでは、ウイルス対策製品を安全なプロセスとして実行する必要があります。 この要件をまだ満たしていない製品は、Windowsセキュリティインターフェイスに存在できません。WindowsDefenderウイルス対策は、これらの製品と共に引き続き機能します。



Windows Defender Application Guardの機能強化



Windows Defender Application Guardは、2018年10月の更新プログラムで新しいWindowsセキュリティユーザーインターフェイスを導入しました。 個々のユーザーは、登録キーまたはグループポリシー設定を変更することなく、WindowsセキュリティアプリケーションでWindows Defender Application Guard設定を設定および構成できるようになりました。 さらに、企業ポリシーの制御下にあるユーザーは、設定を確認して、管理者がデバイスに指定した構成を確認し、Windows Defender Application Guardの動作をよりよく理解できます。 デバイスが最小要件を満たしている限り、これらの設定はWindowsセキュリティに表示されます。 詳細はこちら （英語）をご覧ください 。



自分で確認するには、Windowsセキュリティアプリケーションに移動し、[アプリとブラウザーの制御]を選択します。







分離ナビゲーションモードで、[Windows Defender Application Guardのインストール]を選択します。 インストール後、デバイスを再起動します（独立ユーザーのみ）。







Application Guard設定を表示または変更するには、[Application Guard設定の変更]を選択します。

ナレーターの強化



ナレータースキャンモードでのコマンドの選択：ナレータースキャンモードは、Microsoft Edge、Word、Outlook、Mail、およびほとんどのテキストファイルでのコンテンツ選択をサポートしています。 標準のShift-selectコマンドをControl + Aとともに使用して、ドキュメント全体を選択できます。 Caps + Shift +下矢印を使用して、選択したテキストを発音します。 選択コマンドの完全なリストについては、Caps + F1を押してナレーターコマンドのリストを参照してください。 コンテンツを選択し、Ctrl + Cを押してクリップボードにコピーします。フォーマットも保存されます。

ダイアログの自動読み取り：ナレーターは、フォアグラウンドでダイアログの内容を自動的に読み取ります。 この関数は、ナレーターがダイアログボックスの名前、その中の主要な要素、およびウィンドウの上部にある静的テキスト（存在する場合）を読み取ることを目的としています。 たとえば、変更が保存されていないWordでドキュメントを閉じようとすると、ナレーターはMicrosoft Wordの見出し、[保存]ボタンのメイン要素、およびダイアログボックス内の静的テキストを通知します。
