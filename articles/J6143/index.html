Web 2.0のスタートアップ価格

Web 2.0スタートアップの真のコストを評価する方法は？ 今日、この問題は関連性が高まっています。なぜなら、市場では取引とベンチャー投資の数が明らかに増加しているからです。 スタートアップの大まかな評価のための興味深い方法が、このような素晴らしいスライドを描いたeSnipsのディレクターによって提案されました。 これは、スタートアップのコストがAlexa Reach （100万人あたりの訪問者数）と直接相関していることを示しています。







このスライドは、スタートアップのコストがウェブサイトのトラフィックに依存することを示しています。 もちろん、Alexaの統計とそのような評価の原理はどちらも大部分が不正確ですが、何が起こっているのか、いわゆるWeb 2.0エコシステムの全体像をよく示しています 。



Web 2.0のユニークなピラミッドの頂点には、10億ドル以上の価値があるサイトがあります。これらは、Wikipedia、YouTube、MySpaceの「ビッグスリー」、および隣接するOrkutとBloggerであり、これらも10億以上の価値があります。 それらはすべて、Alexaの評価が20,000を超えています（つまり、インターネットユーザーの2％以上が毎日アクセスしています）。 比較のために、Habrではこの指標は170（つまり0.017％）を超えず、Yandexでは12,000（1.2％）



ピラミッドの第2段階では、Flickr、Facebook、Hi5（5億ドルから10億ドルのコスト）です。 1億ドルから5億ドルの範囲で、LinkedIn、Bebo、Digg、Feedburner、Last.fm、Technoratiを検討できます。 StumbleUpon、Slide、Netvibesは1億ドル弱です。



もちろん、1人のユーザーの「コスト」はサイトのビジネスモデル、ユーザーの社会的および人口統計的構成に応じてサイトごとに大きく異なるため、訪問者数を直接ドルに変換することはできません。
