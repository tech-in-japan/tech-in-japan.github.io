新しいIntel Core 2 Extreme X7800モバイルプロセッサー

インテル、初のモバイルデュアルコアIntel®Core™2 Extremeプロセッサーを発表 新製品のリリースは、Intel®Core™マイクロアーキテクチャに基づいたサーバー、デスクトップPC、およびラップトップ向けのIntelプロセッサの登場から1周年に合わせて行われます。



2.6 GHzの周波数で動作するモバイルPC用のデュアルコアIntel Core 2 Extreme X7800プロセッサは、ラップトップに必要な省エネ機能を実装しています。 Intel Core 2 Extreme X7800のパフォーマンスは、前世代の同じプロセッサーと比較して28％高くなっています。 さらに、Intel Core 2 Extreme Mobile Lockは、プロセッサバス周波数を制限するロックを削除しました。



デスクトップセグメントも発表なしで放置されていません。 このニッチで、Intelは、最も先進的なクアッドコアIntel®Core™2 Extreme QX6850プロセッサを含む多くの新しいプロセッサを導入しました。 クロック周波数は3.0 GHzで、新しい高速システムバスは1333 MHzの周波数で動作します。 新しいIntel®Core™2 DuoおよびIntel®Core™2 Quadプロセッサーの発売も発表されました。



2.6 GHzの周波数と4 MBのL2キャッシュを備えたIntel Core 2 Extreme X7800モバイルプロセッサは、現在851ドルでOEMに提供されています。 会社の代表者によると、このプロセッサを搭載したラップトップは今後2週間で市場に登場する予定です。 OEM向けに新しいIntelデスクトッププロセッサも利用できます。 エンドユーザーは、今後2〜3週間以内にそれらに基づいて製品を購入できるようになります。
