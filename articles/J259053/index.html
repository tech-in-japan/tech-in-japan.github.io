Evernote SDK for Androidの2番目のバージョン：アプリケーションの新しいEvernote機能

何万人もの開発者がすでにEvernoteプラットフォームを使用して、当社のサービスの1億人のユーザーにとってアプリケーションをより機能的で魅力的なものにしています。 明らかに、このコミュニティの大部分はAndroidを扱っています。



Android向けEvernoteアプリの最近の更新に伴い、Android向けEvernote SDK 2.0も準備する機会がありました。開発者を招待して、プロジェクトの新機能を試してみてください。



GithubのAndroid SDK向けEvernote >>







この更新には、いくつかの重要な変更が含まれています。



  • 新しいビルドシステム： GradleおよびAndroid Studioのサポートが追加されました。AndroidアプリケーションのIDEで推奨されています。
  • 書き直されたコード：以前使用されていた多くの古いAndroid関数が新しいAPIに置き換えられました。 SDKでは、プログラミングインターフェイスを保持し、ネットワークパフォーマンスを改善しました。
  • サードパーティアプリケーション認証： OAuthを使用してEvernoteで認証するために、サードパーティアプリケーションはWebブラウザウィンドウを開く必要がなくなりました。 例を見る
  • Evernote Businessコーポレートサービスのサポート：ユーザーのビジネスノートブック（および一般的なノートブック）への接続がはるかに便利になりました。 詳細を見る
  • 開発者トークンのサポート： APIキーを受け取る代わりに、 開発者トークンを作成して、今すぐアカウントでAPIの使用を開始できます。 さらに、新しい機能を追加するために、重要なクラスをファクトリーとビルダーで拡張しました。 コードの再利用と拡張がはるかに簡単になりました。
  • メモ用のHTMLコードの読み込み： NoteRefクラスを使用すると、保存メモを簡単に読み込むことができます。 統合検索を使用すると、1つのクエリで複数の条件でメモを見つけることができます。 HTMLメモの読み込み機能は、ブラウザまたは外部エディタでメモを表示するのに役立ちます。 デモを見る
  • 新しいインテント： Evernote API呼び出しとネットワーキングがなくても、Evernote Androidアプリを使用してメモを作成、表示、検索できるようになりました。 EvernoteにHTMLコードを送信することもできます。このコードは、アプリケーションが新しいノートの有効なENMLに変換します。 Evernote Android Intents Libraryを開きます
  • Evernote SDKを使用してアプリケーションの機能を拡張する原則を理解できる、 更新されたデモアプリケーション。 デモアプリを見る


Android向けの新しいEvernote SDKの公開バージョンは、すでにすべての人が利用できます。 興味深い統合について聞いてうれしいです。 あらゆる方法でGithubリポジトリのプールリクエストとエラーメッセージを歓迎します>>
