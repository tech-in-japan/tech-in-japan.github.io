Visual Studio Online。 それは何で、何と一緒に食べますか？

Team Foundation Serverの機能に基づくVisual Studio Onlineは、開発プロジェクト用のオンラインスペースです。 サーバーをインストールまたは構成しなくても、クラウドインフラストラクチャをすばやくセットアップできます。



どのIDEを使用できますか？

-Visual Studio Onlineは、Visual Studio、Eclipse、Xcode、およびその他のGitクライアントに接続します。これにより、クロスプラットフォーム開発のサポートを提供できます。



各Visual Studio Onlineアカウントは、無料の5人のユーザーを提供します。 グループが拡大またはニーズが拡大するにつれて、ユーザープランとリソースを接続して一致させ、各ユーザーに必要なものを正確に提供します。 MSDNのVisual Studioサブスクライバーは、最大5人の無料ユーザーがいるアカウントでプロジェクトに参加できます。







クラウド内のホストコード

いつでもどこでもアクセスできるように、クラウドでコードをホストします。 従来の集中型バージョン管理システムであるTeam Foundationバージョン管理を使用するか、分散アプローチを希望する場合はGitリポジトリを使用します。







クラウドにプライベートソースコードリポジトリを作成する

サポートする開発者の数（3〜5人または数千人）に関係なく、高度な分岐、フェデレーション、および視覚化テクノロジを使用して、クラウド内のソースコードを管理できます。



GitまたはTFVCを使用してソース管理を選択する

最適なソース管理モデルを使用します。Gitを使用する分散バージョン管理システム、またはTFVCを使用する集中型バージョン管理システムです。 選択はあなた次第です。







コードの変更について同僚と話し合う

コメントを作成し、コード変更の場所で作成された従業員のコメントに応答します。



一箇所で作業を管理

クラウドでプロジェクトを管理します。 柔軟な計画とポートフォリオ管理のためのシンプルで完全に統合されたツールを使用して、すべてのバックログとチームをすばやく計画、監視、追跡します。







バックログの報告とスプリントの使用

断片化の各レベルで階層的に順序付けられたバックログを作成して、迅速な識別、優先順位付け、およびジョブ割り当てを行います。 タグを使用して、すばやく分類およびフィルタリングします。



ワークアイテムで要件、タスク、エラーを追跡する

作業項目を使用して、タスクとエラーをグループに割り当て、ステータスを追跡し、グループ内の作業を調整します。 ワークアイテムをコード変更にリンクし、ワークアイテムをリンクします。







かんばんボードとアジャイル開発ボードで進捗状況を追跡

タスクのグループが完了したときのかんばんの作業の全体的な進捗を追跡し、スクラムベースのタスクボードを使用してスプリントの作業の進捗を追跡します。



クラウドでアプリを構築する

クラウドで、オンデマンドで、または自動化された継続的統合戦略の一部として、アプリケーションを自動的にコンパイルおよびテストするビルドプロセスを作成および管理します。



アセンブリをアンロードする

クラウドベースのビルドコントローラーと毎月割り当てられた無料ビルド時間を使用して、クラウドでプロジェクトをビルドするリソース集約型のタスクを分散します。







ビルドプロセスを自動化する

クラウドでのビルドを自動化して、夜間または各リターン後に実行し、コードがビルドされ、デプロイの準備ができていることを確認します。



長期にわたるビルドステータスの追跡

プロジェクトのホームページで最新のビルド結果を表示し、通知をカスタマイズします。



アプリケーション構造を詳細に調べる

Visual Studio Online Application Insightsは、実行中かどうかに関係なく、クライアントアプリケーションとデバイスアプリケーション、およびサーバーアプリケーションからクライアントによる操作、パフォーマンス、および使用に関する多くのデータを収集する新しいクラウドサービス（現在プレビューバージョンが利用可能）ですオンプレミス、Microsoft Azure、サードパーティクラウドサービス、またはこれらの方法の組み合わせを使用します。



機能の豊富なダッシュボードとアプリケーションの詳細を含む動的レポートを使用する

コントロールパネルは、アプリケーションの操作に関する情報を提供し、カスタムビューを作成できるようにします。 動的レポートは、アプリケーションのさまざまな側面の概要を提供します。



アプリケーションの使用状況を追跡します。

ユーザーの地理的位置や好みのブラウザなどのインタラクティブなデータを調べて、アプリケーションのすべてを学びます。 その後、この情報を使用して、アプリケーションに関する情報に基づいた決定を行うことができます。



結論：Visual Studio Onlineは、Team Foundation Serverに代わる機能的でほぼ無料（最大5ユーザー）の代替手段です。 あなたが小さな開発チームである場合、これはあなたのためだけです。
