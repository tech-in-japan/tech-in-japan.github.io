マイクロソフトはソーシャルネットワークをロシア化しました

MicrosoftのソーシャルネットワークであるWindows Live Spacesは、 ほぼ MySpaceの双子の姉妹ですが、現在ロシアにあります。 その中の自分の「モジュール」は、ロシア語を話すユーザーを何人か獲得しました。 イベントの第一印象はブログに表示されます。



この法人のニュースだけではありません。 また、IMクライアントWindows Live Messenger 8.1の次のバージョンのリリースについても報告されています（以前はRussifiedでした）。 このサービスはソーシャルネットワークと統合されており、たとえば、どの連絡先がブログを更新したかを判別する機能を提供します。 マイクロソフトのプレスサービスによると、ブランドのメッセンジャーはセキュリティ設定が改善されており、多くのデバイスからアクセスできます。



この企業は、Windows Live Spacesネットワークについて次のように対応しています。インターネット上でインタラクティブな日記を作成および管理し、写真やビデオファイルをアップロードできる無料のネットワークサービスです。 近い将来、名前、年齢、興味、およびモバイルデバイスとの同期機能の基準によって連絡先を検索する機能が含まれます。



Windows Live Spacesの利点のリストは次のとおりです。



-写真およびビデオファイルを操作します。 1か月に最大500枚の写真をアップロードできます。

-ブログを操作します。

-デザイン：100のテーマから選択

-天気予報や個人的な星占いの放送など、「機能」を追加する機能。

-音楽トラック、お気に入りのリンクなど、カタログを作成できます。

-デバイスまたはスマートフォンから日記を保存する機能。

-Windows Live Toolbarの可用性。



Windows Live Spaces（以前のMSN Spaces）は、2004年12月に開始されました。現在、36か国で15言語で利用可能です。 2006年7月現在のcomScore Networks Incによると、1か月あたりのWindows Live Spacesユーザーの数は1億2000万人を超えています（ 出典 ）。
