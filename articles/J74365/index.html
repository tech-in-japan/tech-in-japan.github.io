人気のあるRunetサイトはOpera 10の特別バージョンを受け取りました





2009年11月5日-ノルウェー、オスロ-Opera Softwareは、ロシアの人気インターネットリソース向けにOpera 10デスクトップブラウザの特別バージョンを発表しました。 Auto.ru、ComputerBild、Habrahabr、KinoPoisk、Molotok.ru、OZON.ru、およびSuperJobのユーザーとリーダーは、インターネット上で動作する最新の強力なプラットフォームであるOpera 10ブラウザーを使用して、お気に入りのWebサイトの最も重要なセクションにすばやくアクセスできます。



Opera 10の発表された各バージョンは、Webプロジェクトのブランディングで設計されたExpressパネルの事前定義されたリンクを使用して、Webサイトの最も重要で人気のあるセクションへのワンクリックのクイックアクセスを提供します。デフォルトで。 さらに、一部のポータルのバージョンには、独自のユーザーインターフェイスと事前構成されたRSSフィード、およびクイック検索リストに追加された検索リソースがあります。



特別バージョンの開発は非営利プロジェクトであり、その主な目的は、関心グループやユーザーコミュニティに最新のOpera 10ブラウザーを提供することです。 将来的には、ロシアのウェブリソースのユーザーのレビューと要望に応じて、特別なバージョンのより詳細な詳細が計画されています。



Opera Softwareはユーザーのリクエストに細心の注意を払っています。 特別バージョンのリリースは、ユーザーのニーズと好みに最適なソフトウェア製品の作成に向けた会社のもう1つのステップでした。 世界中で4,500万人以上がOperaのデスクトップバージョンを使用しており、ロシアはこの点でリーダーです。 さまざまな独立した推定によると、ロシアのOperaデスクトップブラウザのシェアは23〜27％です。

Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザーは、その速度、コンパクトさ、および他のブラウザーよりも標準に近いため、ユーザーやメーカーから世界的に認められています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
