EiskaltDC ++。 最初のリリース番号2.0

ごく最近、Linux用の新しいDC ++クライアントの最初のリリース-EiskaltDC ++ 2.0。 プロジェクトは同胞によって開発されており、EiskaltDCの直接の後継者です。 バージョン2.0の即時リリースは、dc ++カーネルへの移行が実行された（EiskaltDCはdclibに基づいていた）ため、コードは完全に書き直され、新しいインターフェースはQt4に基づいており、元のDC ++クライアントにできるだけ近いように見えます。 その結果、EiskaltDC ++はdc ++カーネルのQt4のフロントエンドになりました。





主な革新：



  • dc ++カーネル（v0.75）への移行、およびそれに応じたADCサポート
  • フル機能のチャット（マグネット、リンク、顔文字、チャット検索、ユーザーリストのフィルタリング、チャットコマンドの解析）
  • フル機能の検索エンジン、ファイルブラウザ（既に共有されているファイルの強調表示など）
  • 接続マネージャーでの伝送グループ化
  • 個別のダウンロードマネージャー
  • 指定/ダウンロードのリスト
  • 選択したユーザー（自動スロットなど）、ハブ
  • テキスト（Qt経由およびデフォルトのシステム通知マネージャー経由）と音声通知
  • アプリケーションのテーマ、アイコン、スマイルパックのサポート
  • マスクでファイルを共有しない機能
  • UPnPサポート
  • 選択したハブのタグスプーフィング
  • 選択したハブに外部IPを強制する
  • 選択したハブのDynDNSを介した外部IPの自動更新
  • 検索スパイ
  • IPフィルター
  • Aspellによるスペルチェック


ソースと収集されたパッケージ、および基本情報は、プロジェクトページにあります 。



Debian / Ubuntuでのインストールの説明（ ここから引用 ）：




Launchpadでパッケージアーカイブをマウントするには、次の手順を実行します。 

 sudo su -c 'echo "deb http://ppa.launchpad.net/tehnick/tehnick/ubuntu karmic main" > /etc/apt/sources.list.d/tehnick.list'
sudo su -c 'echo "deb http://ppa.launchpad.net/tehnick/tehnick/ubuntu karmic main" > /etc/apt/sources.list.d/tehnick.list'







キーのインポート：

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 437276613F6729E2







利用可能なパッケージのリストを更新します。

sudo apt-get update







eiskaltdcppをインストールします。

sudo apt-get install eiskaltdcpp







または、eiskaltdcpp-unstableをインストールできます。

sudo apt-get install eiskaltdcpp-unstable
