無料のVisual Studio 11はMetro UIでのみ動作します

Windows 8を搭載したMicrosoftの主なタスクは、ジェスチャによって制御され、フルスクリーンになり、Metroデザインに対応するアプリケーションの新しいパラダイムを作成することです。 噂によると、Metro UIを使用してアプリケーションで作業したいWindows開発者は、引き続きVisual Studio 11 Expressの無料版を使用できますが、デスクトッププログラムの作成を継続したい人は、おそらくこの特権に多くの費用を支払う必要があります。







Ars Technicaの Webサイトによると、無料のVisual Studio 11 Express環境により、プログラマーはMetroデザイン （まあ、サイト）でのみアプリケーションを作成でき、デスクトップデザインは許可されません。 これまで、開発者はVisual C ++ 2010 ExpressまたはVisual C＃2010 Expressを自由に選択してカスタムプログラムを作成したり、Visual Studio 2010 Professionalまたは他のエディションにアップグレードしたりできました。



マイクロソフトの新しいポリシーに従って、デスクトップアプリケーションの開発者の「将来の世代」に対する唯一の選択肢は、Visual Studio 11 Professionalを400〜500ドル（2010年版の旧式の環境）で提供します。



さらに興味深いことに、MicrosoftはおそらくWindows 8のWindows SDKからコンパイラも削除するため、開発者はVisual Studioの外部でデスクトップアプリケーションを作成できません。 ただし、彼らはMicrosoft DreamSparkプログラムの下でVisual Studio 11 Professionalを学生が引き続き利用できると書いています。
