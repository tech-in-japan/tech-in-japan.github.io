専門分野：

Java SE

Java Platform、Standard Edition（Java SE）は、今日のアプリケーションに必要な豊富なユーザーインターフェイス、パフォーマンス、セキュリティを提供します。 Java SEトレーニングでは、JavaベースのIT環境の展開とメンテナンスのコストを最小限に抑えるエンタープライズ機能を顧客に提供する方法を学びます。



Java EE

企業は、新しい軽量のJava EE 6 Webプロファイルを使用して、次世代のWebアプリケーションを作成します。 Java EEトレーニングを通じて、エンタープライズアプリケーション向けのJava EE 6プラットフォームのフルパワーについてさらに学びます。



Java ME

Java Platform、Micro Edition（Java ME）は、モバイルおよびその他の組み込みデバイスで実行されるアプリケーションに堅牢で柔軟な環境を提供します。 Java MEトレーニングを通じて、このプラットフォームの柔軟なユーザーインターフェイス、堅牢なセキュリティ、組み込みのネットワークプロトコルなどの詳細をご覧ください。



Webサービス

Oracle Web Services Managerは、セキュリティおよび運用ポリシー管理を通じて共有サービスとの相互作用を管理するソリューションを提供します。 このソリューションの使用方法に関する詳細な情報を取得するには、Oracle WebServices Managerトレーニングへの投資を検討してください。


EEは第6版にのみ存在するため（現時点では）、最初の2つを通過した後、第7版ではなく第6版のEEに変更を加える必要があります。

認証



指定された時間の15〜20分前に認証に表示されます。 書き消す機会がないように、すべての個人用アイテムはクローゼットにロックされており、キーは個人的に与えられます。 試験では、正しいリーフレットと右ペンが与えられます。 試験の開始前に写真を撮り、署名のコピーを受け取ります。



重要：名前と姓を含む2つのドキュメントが必要です。 文書の1つでは、ラテン文字で書かれている必要があります（このデータは証明書に記載されています）。 私は外国のパスポートとウクライナのパスポートを使用しました。



試験では、90の質問と2時間半の時間が与えられます。 さらに、試験後、試験の質とその有用性について2つの質問があります。 スタイルのすべての質問：オプションを2つまたは3つ選択してください。 つまり 正解オプションの数は常にわかっています。 変更には2時間半で十分です。 私は1.5時間管理しました（そして、これは自分の答えを修正したという事実を考慮に入れました）。



テストに合格する過程で、質問にマークを付けて後で質問に戻ることができます。 これは非常に便利で、最初はあなたが知っていることすべてに答えることができます。 テスト結果はすぐに報告され、印刷されます。
