親愛なる同僚！



Microsoft Virtual Academyポータルでは、10月7日にJump Start教材のコースが利用可能になります。WindowsServer 2012 R2を使用して会社のITインフラストラクチャを近代化します。 このコースでは、Windows Server 2012 R2の新しい興味深い機能について学ぶだけでなく、Windows Server 2003および2003 R2からの移行を今すぐ開始する必要がある理由も学びます。 カットの下の詳細な説明。











2015年7月14日には、サーバーオペレーティングシステムWindows Server 2003およびWindows Server 2003 R2のサポートが完了します。 これらのオペレーティングシステムを使用する多くの企業は、ITインフラストラクチャを近代化し、より近代的なシステムに移行するためのプロジェクトを立ち上げています。 ITプロフェッショナルが最新のWindows ServerファミリOSであるWindows Server 2012 R2に組み込まれた技術ソリューションを見ることが特に役立つのはそのためです。 このコースの一環として、Active Directoryサービス、ストレージ構築ツール、管理タスク自動化ツールなど、ほとんどすべての組織で使用されるコンポーネントの改善が検討されます。 さらに、特にAzure Active Directory Premiumサービスを使用して、ハイブリッド構造のID管理のソリューションに精通することができます。

コースは4つのモジュールで構成されています。



1.インフラストラクチャの近代化。 Windows Server 2012 R2はなぜですか？



このモジュールでは、現在のインフラストラクチャをアップグレードしてWindows Server 2012 R2に移行する必要がある理由を学習します。 最新世代のMicrosoftサーバーオペレーティングシステムに切り替えることの主な利点について学びます。クラウドサービスを使用する機能、データウェアハウスの機能を改善するための新しいテクノロジーの適応、システムおよびユーザー管理ツールなどが含まれます。



2. Active Directoryの機能強化



Active Directoryは、ドメインネットワークの重要なコンポーネントの1つです。 このレポートでは、ドメインコントローラーを作成し、Windows Server 2012 R2で導入された作成済みのドメインを管理する新しい可能性について学習します。 さらに、ドメインコントローラーの複製と仮想化についても学びます。



3.ストレージインフラストラクチャの近代化



ユーザーが使用するデータの量は常に増加しています。 Windows Server 2012 R2を使用すると、高性能でスケーラブルなNASのコストを削減できる新しい機能が提供されます。 このレポートでは、階層型ストレージ、新しいファイルサーバーのスケーラビリティ、およびストレージパフォーマンスを向上させるその他の方法について学習します。



4. PowerShellによる自動化



Windows PowerShell統合スクリプト環境（ISE）を使用して、さまざまなコマンドおよびスクリプトを作成、実行、およびデバッグできます。 Windows PowerShell ISEは、毎日のタスクを自動化し、加速します。 この講演では、スクリプトの記述と実行のプロセスを簡素化および高速化するWindows PowerShell ISEの革新について学びます。
