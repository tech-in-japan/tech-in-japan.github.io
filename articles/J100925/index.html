Apple iOS 4.1ベータ3

文字通りアップルのウェブサイト上で、iOS 4.1の3番目のベータ版、Xcode 3.2.4およびiOS SDK 4.1ベータ3（Snow Leopard）が開発者に利用可能になりました。











一般に、伝統的に、Appleは2週間に1回ベータファームウェアをリリースしますが、今回は1週間前に新しいベータがリリースされましたが、Safariブラウザーに投獄できる穴があるためですか？



開発者の方は、デバイスでベータ版を試すことができます。

次の電話とプレーヤーがサポートされています。



iPhone 4（ iPhone3,1_4.1_8B5097d_Restore.ipsw ）

iPhone 3G（ iPhone2,1_4.1_8B5097d_Restore.ipsw ）

iPhone 3G（ iPhone1,2_4.1_8B5097d_Restore.ipsw ）

iPod Touch 2G（ iPod2,1_4.1_8B5097d_Restore.ipsw ）

iPod Touch 3G（ iPod3,1_4.1_8B5097d_Restore.ipsw ）



xcode 3.2.4およびios SDK 4.1 beta 3



注意：

これらのリンクによると、ファームウェアは週末までダウンロードされます。







新規：

Game Centerは、iPhone 3GおよびiPod Touch 2Gではサポートされなくなります。

開発者はプレイヤーのステータスにアクセスできません。

サファリの穴は閉じられています。 Jailbreakme.comが機能していません。
