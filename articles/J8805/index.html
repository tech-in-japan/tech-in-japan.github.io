LinuxインターネットタブレットNokia N800

Windows Mobileは確かに優れたシステムですが、健康はもっと高価であることに気付き、PocketPCと友達になれませんでした 。 選択肢を探して、私はモバイルコンピュータ市場の研究を続けました。

Nokia N800のリリースについて最初に読んだとき、今日このようなデバイスが必要なのは誰なのかわかりませんでした。明示的なPIM機能のないモバイルコンピューター、GSMモジュール、モバイル高速インターネットを使用する一般的な方向性。 会話に最適なVoIPをお探しですか？ たぶんそうです。 今日の市内のWiFiのみが希少です。 英国でも、心強いニュースにもかかわらず、ロンドン中心部に無料のWiFiホットスポットはありません。

そのため、買収後の最初の数週間、タブレットは私と一緒に死んでしまいました。 インターネット接続はN800を復活させたようです。 この瞬間から、個人情報管理者に関する質問は消えてしまいました。



タブレットの仕様



ソフトウェア

ソフトウェアのインストールは次のとおりです。

1. N800をインターネットに接続します。

2.アプリケーションマネージャで[インストール可能なアプリケーションの参照]ボタンを見つけてクリックします。

3.適切なアプリケーションを選択し、その説明を読んで、通常の方法でインストールします。

アプリケーションをインストールするときに、一部のライブラリが欠落していることを示すメッセージが表示される場合があります。 Pythonに関するもので、インストールファイルはリポジトリまたはここにあります 。

すべてのディストリビューションはリポジトリにあり、そのリストは展開できます。 多数のこれらのリストは、フォーラム、会議、 www.maemo.orgで見つけることができます。これは、すべてのアプリケーションの広大な海を受け入れることが機能しないことを意味します。 私の選択は次のとおりです。

Microsoft Wordドキュメントの編集：誰もがAbiWordのリリースを待っています。 今のところ、 doc reader + antiwordを使用して.docドキュメントを読むことができます。 テキストメモの作成に最適：Leafpad、Notes、Xournal。

テーブルの編集： MSExcel形式で保存する機能を備えたGnumeric Spreadsheet。 ビデオの変換と視聴：フレームが失われると、変換されていない映画を見ることができますが、通常の感覚では、 Media Converterを使用することをお勧めします。

次のコンバーター設定で自分に合ったビデオ品質が得られます：ビデオ解像度N800高品質（400x240）、ビデオビットレート768 kbps。

ビデオを見るには、すぐに標準プレーヤーをMPlayerに置き換える必要があります。

PIMアプリケーション： GPE To-doリスト、GPEカレンダー、GPEサマリー（デスクトップにPIM情報を表示するため）。 プログラムは、WinMobileのアプリケーションの機能をまだ誇っていませんが、それらと連携することはかなり可能です。

読書： PDF Reader、 Evince 、FBReader（read fb2）。 djvuリーダーの登場を待っています。 コミュニケーション： Gizmo、Pidgin、Jabber、Google Talk（Skypeも夏中旬には約束されます）。 クローメールメールクライアント、Operaのデフォルトブラウザ。

ブログ： Maemo Wordpy

Lingvoレベルの辞書 （最も重要なアプリケーション）： Sdictは、WmobileのLingvoよりも高速です。 現時点では、次を使用できます。321個の辞書ファイル、82百万件以上のエントリ、エントリのボリュームは8277メガバイトです。 ウィキペディアには、ロシア語と英語の両方に関連する（！）オプションがあります。 何らかの理由で、辞書のあるリソースが利用できない場合があります 。最も重要なのはここです。 地理、GPS： Maemo Mapper、Googleマップ、 Nokia N800インターネットタブレット用ナビゲーションキット 。

グラフィックの表示と編集： Quiver Image Viewer、Sketch。

さまざまな科学的および教育的アプリケーション： HP Calculator、Lybniz（チャート）、Maemo Periodic（化学元素の表）、Granule（外国語学習用の優れたアプリケーション）。 ゲーム：ファンではないので、多くを語ることはできませんが、多くのアプリケーションがあります。 個人的には、チェスが好きです。



合計：ニーズを十分にカバーするソフトウェアスイート。 何か見逃しましたか？



長所/短所

N800はどこで勝ちますか：

1.画面サイズ：本やウェブサイトを読んだり、タイピングしたり、映画を見たりします。今日の市場では、この目的のためのこの種のお金のためのより良いデバイスはありません。 QVGA画面で何かを読む方法がまだわかりません。 2. Web、インターネット通信-この分野では、デバイスは間違いなく未来に向けられており、今日、奇妙なことに、これも欠点の1つです。

3.オペレーティングシステム、アプリケーションセット：Internet Tablet OS 2007エディションは、WinMobileよりも安定しており、便利で、美しいことがわかりました。 ワイヤレスインターフェースを介した接続に問題はありません。GPRS接続の設定は既にデバイスに存在し、ユーザーはオペレーターを選択するだけです。 十分なソフトウェアがあり、新しいプログラムが毎週登場します。

4.システムのロシア化-デフォルトでは、ロシア語のアルファベットの文字を認識するテキストの手書き入力が利用可能です。

5.インターフェースの革新-たとえば、メニューをアクティブにすると、スタイラスに小さなアイコンが表示されます。 指でタッチすると、アイコンが大きくなり、指で簡単に押すことができます。 タイピングについても同じことです。テキストエディタの画面を指でタッチすると、大きなキーボードが表示され、手でテキストを入力するのに非常に便利です。 iPhoneで発表されたインターフェイスは、ここで100％で正当化されます。ほとんどのデバイスでスタイラスがすぐに不要になることはほぼ間違いありません。

6.マルチタスク。 一般的にアプリケーションを閉じるために、徐々に離脱しました。 最小化されたプログラムは、左下隅のパネルに一列に配置されます。 当然、打ち上げの速度は非常に高速です。 WMobileのように、デジタルエターニティでメモリリークの問題はありません。 7.便利な場所とボタンの機能。インターネットナビゲーションと本を読むのを大幅に促進します。



N800はどこを失いますか：

1.画面の明るさにはまだ多くのことが望まれます。 太陽の下では、最大の明るさにねじった場合でも、ほとんど何も見えません。 この欠点は、これらのデバイスのほとんどに起因する可能性がありますが、たとえば、HTC PZ300はより良く見えます。

2.インターネットタブレットの名誉あるタイトルにもかかわらず、Google Docs＆Spreadsheetsはまだ機能しません。 さらに、 YouTubeには問題があります 。 自信を持って-ソフトウェアの問題は、希望を持って-時間の経過とともに解決されます。

3.都市のWiFiカバレッジが不足しているため、VoIPを介して通信できない。 繰り返しますが、大いに期待して-明日のビジネス。 4. MS Word文書の編集とPCとの同期のサポートの欠如-彼らはまた、すぐに問題を解決することを約束します。

結論（注意して）

すぐに警告したいと思います。N800の所有者になり、昨日あなたがPPCユーザーであるか、モバイルコンピューターをまったく使用しなかった場合、驚きの準備は簡単ではありません。 しかし、覚えておいてください。特別な魅力は、誰もがソフトウェアを変更/作成することによってデバイスを改善できるという事実によって与えられます。 正直に言うと、私は、熱心なWindowsユーザーであり、LinuxのN800インターネットタブレットの初心者でもあるので、答えよりも質問の方が多いです。 決めます！

既に知られているように、ソフトウェアは、セックスのように、無料の方が優れています。 これは、多くの愛好家がすでに自分でN800のソフトウェアを作成し、それでも作成することを意味しますが、ほとんどの場合、プロプライエタリのソフトウェアを超えます。 これがN800の主な利点です。

同時に、今日の主な障害は、快適に時間を超えるようなデバイスの使用を妨げることです。 そして、ここでのタスクはより複雑です。オープンソースのイデオロギーから始めて、集合的な心と手を借りて安価な無線インターネットで地球をカバーすることは機能しません。 それとも私は間違っていますか？
