マイクロソフトがロボットプログラミングソフトウェアをリリース

マイクロソフトは本日、ソフトウェア充填ロボットとさまざまなロボット機構を作成するために設計されたソフトウェアの最終バージョンであるMicrosoft Robotics Developer Studio 2008（Microsoft RDS 2008）をリリースしました。 このソフトウェアのセットは、実際に企業の腸内で5年以上にわたって開発されたパイロットプロジェクトから成長しました。



Microsoft RDS 2008には、非同期アプリケーションを作成するための特別なソフトウェアモデルと、ロボット用のソフトウェアを作成するプロセスで役立つ視覚的およびシミュレーションツールのセットが含まれています。 Microsoft RDS 2008のキットとともに、一連のドキュメントとサンプルソースコードが付属しています。



Microsoft RDS 2008はRobotics Developer Studioの3番目のメジャーリリースです。 同社によると、この環境の以前のバージョンは多くの研究センター、大学、商業企業で使用されています。 Microsoft RDS 2008の準備中に、ユーザーは250,000以上のコピーをダウンロードし、60社以上がRDS 2008の商用利用を申請しました。



環境が受けた革新の中で、Microsoftはいくつかの点を区別しています。



まず、Microsoft RDS 2008では、コードコンパイルシステムが改訂されました。これにより、結果のアプリケーションのパフォーマンスは、マネージコードで1.5〜3倍、サービスで2倍になります。 さらに、開発者は、サービスを接続し、ネットワーク負荷を減らし、データ処理を最適化する、より具体的なプログラムメッセージを指定できるようになりました。



次に、Visual Programming Language（VPL）に劇的な変更が加えられました。 アプリケーションに組み込まれた直感的な言語に、アプリケーションを定義および構成するための新しいメソッドが含まれるようになり、アプリケーションをより迅速にデプロイできるようになりました。



第三に、ビジュアルシミュレーション環境（VSE）ツールに便利な追加が行われました。 VSEには、シミュレーションを記録および再生する機能が含まれています。 さらに、開発者は、さまざまな仮想条件（密室、オープンスペース、または都市）でシミュレーションを実行できます。 さらに、VSEはSolidWorks 3DなどのさまざまなCADプログラムからエクスポートできました。



さらに、RDS 2008システムはVisual Studio 2005および2008と完全に互換性があると同社は述べています。



ソフトウェアは、プロの開発者向けのStandart Edition、教育機関向けのAcademic Edition、および個々の開発者向けのExpress Editionの3つのバージョンで配布されます。 最初のバージョンのコストは500ドルです。 アカデミック版は会計機関向けの特別な条件下で配布され、Express EditionはMicrosoft Webサイトから無料でダウンロードできます。



開発環境サイトは、 microsoft.com / roboticsにあります。
