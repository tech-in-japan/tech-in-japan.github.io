JavaOneとしてのCodeOne 2018、ただしマスクのみ

Oracle Code One







Code One （2018年4月までのJavaOne）は、さまざまな都市で開催されるOracle製品とJavaプログラミング言語に関する年次会議です。 多くのJava開発者はこのイベントに参加し、事前に参加を計画しています。 Sbertechのテスト自動化エンジニアである私は、幸運にも今年10月にサンフランシスコで開催された会議に参加できました。 猫の下で、旅行についての私の印象と考え。 2019年のCode Oneの後援の下での会議の日付は、 ここにあります 。



Code One 2018では600を超えるレポートが発表されましたが、そのうち約200がJavaに関連し、残りはコンテナ、サーバーレス、クラウドアプリケーション、データベース、ビッグデータ、データサイエンス、DevOps、Modern Webなどのトピックに関連しています。 同じ日付（10月21日-10月25日）に、オラクルは最初に別のオープンワールド会議を開催しました。 両方のイベントは、Moscone Centerのビジネスセンターで開催されました。 Code OneはMoscone Center Westにありました。







参加の費用は、登録時から異なります。







会議に登録するには、Oracleアカウントが必要です。 登録プロセス中、システムは便利な支払い方法を選択するように促します。 私の場合、支払いは銀行振込で行われました。 一週間後、私は電子招待状を受け取りました。

招待状






組織のニュアンス



会議の前に、主催者は一般的な会議計画をサイトに投稿します 。



登録



イベントの前日に認定するのが最も便利です。さもなければ、初日に巨大な列に並ぶことができます。







認定時に、会議の各日のプラスチッククーポン（昼食）だけでなく、名前（$ 70の罰金の損失）のプラスチックバッジを発行しました。 1階の別々のラックで、参加者にはOracleお土産（2018年にはバックパック）とOracle CloudFestのチケットが与えられました。



アクションプラン



すべてのレポートとワークショップは2階で行われました。 Java Magazine、Oracle Quiz、Oracle Shop、およびテーブルのあるゾーンのゾーンを持つ12ホールの2つのホールがある広いエリア。 同時に20から24のセッションが行われました。 彼らは午前8時45分または9時00分に始まり、17時00分以降に終了しました。















レポートとワークショップのトピックを含むプログラムは、Code One Webサイトのセッションカタログブロックで公開されています。



このような大規模な会議に参加する場合、独自の訪問計画を作成することが重要です。 多くのセッションが建物の異なる部分で行われ、それらをすべて同時にカバーすることは不可能です。 自分にとって最も興味深いものを選択し、スケジュールを立ててJavaを理解してください（翌日のレポートに慣れて、興味のあるレポートを選択するのに約30分かかりました）。 レポートの著者に注意を払い、彼の以前のスピーチに精通することをお勧めします。 バックアップレポートを選択します。最初の数分でスピーカーが自分のものではないことに気付いた場合は、別のスピーチをキャッチする機会があります。



事前にワークショップに登録することをお勧めします。登録しないと、入会できない場合があります。 ワークショップ会場でWi-Fiが安定して動作していることを確認してください（最新のラップトップモデル（macbookなど）がある場合、Wi-Fiは断続的に動作し、どこでも安定した信号受信レベルがないため、RJ-45用のアダプターを使用することをお勧めします）。

まとめ



Code Oneは、先に訪れた会議（SeleniumConf Chicago、Joker、Heisenbug、SQADaysなど）と比較して、その規模と多方向性に感銘を受けました。もちろん、すべてを一度にカバーしたかったのですが、残念ながら、多くの興味深いレポートが同時に開催されました。



レポートは、非常に低い品質を含むさまざまな品質のものでした。 したがって、会議に参加するためにそのようなスピーカーをどのように選択できるのかとさえ思いました。 JUG.ruカンファレンスが勝利することは間違いありません。彼らは質の高いレポートのみを作成するよう努めており、それをうまく行っています。 しかし、もちろん、Code Oneにはもっと多くの世界的リーダーがいます。



私は、昼休みの長い行列と貧弱なWi-Fi信号を除いて、イベントの組織に満足していました。 再度JUG.ruと比較すると、JokerとJPointでは、ネットワークの問題に気づいたことはありません。 さらに、JUG.ruを使用すると、任意の部屋のオンラインブロードキャストを視聴して購入することができます。CodeOneはリモートで視聴できません。



ちなみに、1〜2人の同僚の会社でそのような会議に行って、レポートから受け取った情報について話し合い、より多くの資料を吸収するのは良いことです。



そして、そのような会議への旅行は便利だと思いますか？
