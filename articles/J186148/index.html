Kingnovel K-R42-Android 4.2.2搭載のメディアプレーヤー



スティック形式の4コアRK3188チップでのミニコンピューターの発売が成功した後、拡張機能（追加ポートと外部Wifiアンテナの存在）を備えたデバイスのラインの論理的な継続が市場に参入します。 今日は、Kingnovel K-R42-Android 4.2.2を搭載したメディアプレーヤーに注目します。これに基づいて、ホームミニサーバーを作成できます。





►仕様

クアッドコアMali 400MPビデオアクセラレータ

Rockchip RK3188クアッドコアCortex A9アーキテクチャ

2 GBのRAM

Androidオペレーティングシステム4.2.2

8 GBのユーザーメモリ+ microSDカードスロット

通信：Wi-Fi（802.11 b / g / n）、イーサネット



このデバイスの特徴は、AV出力、SPDIF、イーサネットなどの多くの追加出力が存在することです。 2つのフルサイズUSBポートもあります。

適切なレベルでビルドします。 唯一のマイナスは、冷却ラジエーターの欠如です。 ラジエーターは独立してインストールできます（噂によると、デバイスの新しい改訂版はラジエーターで既に作成されていますが、残念ながら、この情報を確認または反論することはできません）。 しかし、デバイスを「そのまま」使用する場合、加熱がどのように行われるかを見てみましょう。

オフ状態では、ケースの温度は25.2度です。





簡単な使用モード（サーフィン、単純なアプリケーションの起動）では、温度は約10度しか上昇しません。





安定性テストで30分間ウォームアップした後、ケースの温度は上から40度に上昇し、下からはほぼ50に達しました。





理論的には、K-R42は24/7サーバーとして、追加の冷却なしで使用できます。 ただし、セキュリティ上の理由から、チップの冷却を追加することをお勧めします。

►ソフトウェア

すべてのGoogleアプリケーションでプレインストールされたAndroid 4.2.2。









Android 4.2.2のファームウェアには、Rockchipの独自のユーティリティが追加されました。これにより、スマートフォンやその他のAndroidデバイスからミニコンピューターをリモートで制御できます。

アプリケーションの設定は、MK908と同じです。

RKRemoteControlの開始と使用
このアプリケーションのファームウェアは、ファームウェア自体に既にインストールされています。 アプリケーションをインストールするだけです スマートフォンまたはタブレット上のRKRemoteControl （2MB）。

次に、アプリケーションを実行し、見つかったサーバーに接続します。



デバイスがお互いを検出するには、同じネットワーク上にある必要があります。



RKRemoteControlの動作は、いくつかのモードで機能します。

1）「マウス」モード。 制御デバイスのタッチスクリーンに沿って指を移動すると、制御デバイスのシステム内でカーソルが移動します。 画面を1回タップするのは、クリックと同じです。 スクロールするには、制御デバイスの垂直または水平スクロールバーを使用する必要があります。

2）スクロールモード。 ドラッグアンドドロップすると、管理対象デバイスでスクロール（スライド）が発生します。 画面をタップすると、クリックに対応します。

キーボードアイコンを使用して、画面バージョンを呼び出すことができます。



また、「戻る」、「ホーム」、「タスクリスト」のボタンの行が重複しています。



コントロールアイコンを選択すると、ビデオ/オーディオプレーヤーのコントロール画面が表示されます。





ゲームアイコンをクリックすると、ゲームコントロールの設定画面が表示されます。





電話画面では、これは面倒に見えますが、適切な設定メニューを呼び出して、不要なボタンを削除できます。





ゲームパッドのクラシックレイアウトを配置しましょう。





ここで、K-R42でコントロールを構成する必要があります。

RKGameControlプログラムを実行し、コントロールプロファイルを選択/作成します。

►ミニサーバーを作成します

K-R42を使用するための1つのオプションは、さまざまなニーズに応じてK-R42に基づいてミニサーバーを作成することです：Webサーバー、FTPサーバー、急流ダウンロード。

市場には、タスクを簡単にする多くのアプリケーションがあります。

たとえば、Webサーバーの場合、 NAMP nginx Android Webサーバーアプリケーションを使用できます。これには次のものが含まれます。

1. nginx v1.5.0

2. PHP v5.4.13

3. MySQL v5.1.62

4. msmtp 1.4.30

5. NAMPFTP v1.0

phpFileManager、phpMyAdmin、adminerも同様です。









動的IPがあり、外部からサイトへのアクセスを構成する場合は、 DynDNSクライアントユーティリティを使用できます。



サーバーへのリモートアクセスには、 Droid VNCアプリケーションを使用できます。 サーバー上でアプリケーションを起動し、[開始]をクリックするだけです。





アクセスは、ブラウザからJavaプラグインを介して、またはHTML5を使用して提供されます。
