私は長い間このプログラムを探していました。 理由はわかりませんが、今日だけ彼女を見つけました。

多くのアナログがありますが、ほとんどすべてを試してみましたが、そのうちの1つではなく、私のニーズを満たしています（有料のもの、重いもの、必要な機能を備えていないもの）。

最後に、探していたもの、 つまりreSizerプログラムを見つけました。 このユーティリティを使用すると、ホットキーを使用してウィンドウ内のウィンドウのサイズと場所を制御できます 。

この機能は、主に大きなモニター（24以上）を持っている人や、同時に複数のモニターで作業する人に必要です。

半分がプログラムと他のソースコードの結果を示す場合、大型モニターの使いやすさを過大評価することは困難です。 マニュアルとコード、またはicqとブラウザ、レイアウトとフォトショップ、ビデオとドキュメントのいずれか-あなたは無限に続けることができます。







ユーティリティはウィンドウメニューにアイコンを追加せず、 ホットキーを使用してウィンドウを制御します 。



Win + Arrows ：ウィンドウを移動

Win + Shift +矢印 ：ウィンドウを正確に移動



Win + NumPad 1 ：左下隅に移動します

Win + NumPad 2 ：真ん中の下に移動します

...

Win + NumPad 9 ：右上隅に移動します



Win + NumPad 0 ：展開\復元

勝利+ NumPad。 ：折りたたむ



Win + Backspace ：次のモニターに移動します



Win + Ctrl +矢印 ：サイズ変更

Win + Ctrl + Shift + Arrows ：サイズを正確に変更



Win + Ctrl +（2つのNumPad番号のシーケンス） ：セグメント内でヒットしてサイズを変更します。 最初に、最初の重複するセグメントが入力され、次に最後のセグメントが入力されます。

たとえば、右上の四分の一をサイズ変更して占有するには、Win + Ctrl + NumPad 5およびWin + Ctrl + NumPad 9を連続して押す必要があります。下半分を占有するには、Win + Ctrl + NumPad 1およびWin + Ctrl + NumPad 6を押します



Win + NumPad * ：「すべてのウィンドウの上」に切り替えます

Win + NumPad / ：透明度の切り替え



プログラムは完全に無料であり、 http：//zestant.googlepages.com/resizerからダウンロードできることに注意してください



起動時に、間違いなく。
