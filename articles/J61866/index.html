Acer Aspire One 110 BwにWindows 7をインストールして加速する

このような短くて急なプレゼンテーションをおpresentationび申し上げます。 事実、ちょうど30分前にインストールを終了し、 ブログの成功について登録を取り消すことにしました。そのような関連トピックに関する集中情報がほとんど見つからなかったからです。 次に、お気に入りのHabrに行って記事を見ました。

「Windows 7の高速化：パート1」とその答え「Windows 7の高速化：パート1」。 これらの記事は両方とも実際の状況を明らかにしていないので、いくつかのアドバイスを提供しますが、おそらく多くの人が知るのに役立つので、ここでも私の経験を共有することにしました。 だから：



昨日、1Gb DDR、16Gb SSD、およびWindows XPを搭載したAAOネットブックを持っています。 これはWindows 7 RC 7100を搭載したネットブックです。USBフラッシュ4Gbを搭載した7台をインストールしました。 これを行う方法については、 この記事で詳しく説明します。

Windows 7をインストールした後、かなり遅くなりました。 まず、ディスクのインデックス作成をオフにしてから、ページファイルをオフにして、すべてのエフェクトをオフにしました。 何もしませんでした フォーラムを検索すると、SSDにファイルを書き込むためのドライバーに関するヒントが得られました。 執筆時点では、これはbeta5バージョンです（beta6は7では動作しません！）。 あなた自身の危険とリスクで...インストールされ、再起動されて、OH SHIT !!! Windowsは10倍速く実行されました！ 私は天と地のように冗談を言っているのではありません。 EFWツールを使用したXPよりも高速。 Windows 7をみんなのネットブックに載せてください！ ;）
