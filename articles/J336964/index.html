すぐに世界を見る40の壮大なARKitデモ





ARKitを使用して、アプリケーションの新しいビデオやプレスリリースが次第に成長している様子を見ると、その拡張機能のために拡張現実技術が非常に広く使用されることが明らかになります。 この記事では、最も興味深いアプリケーションを強調してグループ化しました。



それでは、ARKitを使用したデモの検討に移りましょう。



距離測定



実空間内のさまざまなオブジェクト間の距離を測定できるアプリケーションには、否定できない実用的なアプリケーションがあります。確かに、そのようなソリューションは、近い将来、平均的なユーザーの電話に搭載されるでしょう。 測定精度は驚くほど高いです。 現時点では、これらのプロジェクトのうち2つがわかっています。

3D図面



ARKitテクノロジーは、実空間で「参照ポイント」を使用して空間マーキングを可能にします。 これにより、3D図面を作成できるだけでなく、テーブルの水平面に画像を配置するなど、実世界のオブジェクトに静的で安定して添付することも可能になりました。


3. ARKitの線画



デフォルトのARKitが変更されたプロジェクト。これにより、空間に線を引き、カメラ自体をローカライズできます。



リンク： github







4. ARBrush



このプロジェクトは、3Dでの描画を示しています。 アプリは、SceneKitと組み合わせてMetalを使用します。



リンク： github





5. ARPaint



3D描画のもう1つの元の形式は、実際の空間に自分の指で描画できるプロジェクトです。 このアプリケーションは、空間内のオブジェクトの動きを追跡する技術を使用して、指の動きの方向を判断します。これにより、デバイスの画面に画像が投影されます。



リンク： github | チュートリアル





建築とデザイン



この技術は、設計とアーキテクチャに拡張現実を適用する大きな機会を提供します。 たとえば、将来の家の3Dモデルを縮小した形で検討し、より正確な画像を取得できます。 また、建設作業が完了すると、建物が現実的な寸法で表示されます。 これはすべて、隣接する地域の設計と改善を大幅に促進できます。


6.イケアプレイス



イケアの小売チェーンは、長年にわたって拡張現実の機能を使用して製品を宣伝してきました。 同社は、IKEA Placeと呼ばれる特別なアプリケーションを開発したため、ARKitに新しい機能が登場したことで合格しませんでした。 オブジェクトはさらに安定し、実際のサイズに対応するようになりました。たとえば、ソファトランスのメカニズムのデバイスを見ると、近くでよく見ることができます。 さらに、距離を測定する機能があります。

13. Apple Mapsでの使用



ARKitの機能を使用して、標準の「マップ」アプリケーションでは、ユーザーがデバイスとともに物理的に移動し、デバイス自体が空間内を移動するときに、3Dマップのスペース（たとえば、オペラハウス周辺のシドニー）で移動できます。 この機能のために、iOS 11には特別な「鳥瞰図」モードがあり、最初は限られた数の都市でのみ利用可能です。 さらに、「Maps」アプリケーションは、拡張現実のポインティング矢印の形で3Dモデルを導入することにより、ガイド機能を受け取ります。 また、アプリケーションにはVRモードが含まれます。





14. ARKit CoreLocation



このアプリケーションは、カメラデータとモーションセンサーで動作して周囲の世界を表示するARKitと、Wi-FiとGPSデータを使用して位置を特定するCoreLocationの2つのテクノロジーを同時に使用します。 ARKitとCoreLocationの組み合わせにより、高いAR精度が得られます。 また、拡張現実では、特定の場所への移動経路を示す矢印と碑文が表示されます。



リンク： github

19.非常に空腹なキャタピラー

20. The Walking Dead：Our World



The Walking Deadは、人気映画に基づいたゲームです。 プレイヤーの前に「死者の痕跡」の形でポインタが表示され、それが彼を空間に導きます。 ゲームはARPointCloudと呼ばれる新しいARKit関数を使用します。これにより、突然ゾンビが出現する効果が得られます。 もう1つの重要な機能に注意してください。アプリケーションには、対応する機能を備えたマルチプレイヤーゲームのオプションがあるため、攻撃的なゾンビ軍と友達と戦うことができます。 発表によると、このゲームはAndroidでもリリースされ、明らかにGoogle ARCoreを使用します。

38.ダンスリアリティ



拡張現実でのダンスの練習。 このアプリケーションは、最初から踊る方法を学びたい人に適しています。それは、ダンサーがダンスの「ステップ」のそれぞれに文字通り付随するものです。 良いダンスボイスが組み込まれています。 さらに、ペアでトレーニングする可能性があります。 おそらく、誰かがスマートフォンを手に持って踊るのにすぐに飽きてしまうでしょう。このトゥターをiGlassesに移した方が良いでしょうが、それでもこのアプリケーションを使えばダンスの基本を学ぶのは簡単です。





39.アニメーションショーツ



これは別の段落で言及する価値があります。ARのアニメーションショートパンツは映画業界に新しい時代を開き、ストーリーテリングの新しい機会を提供できるからです。 多くの映画の巨人、特にロードオブザリングのクリエイターに既に技術に興味を持っています。 問題は、そのような「フィルム」をシーンのさまざまな側面から見ることができ、テーブルの周りの空間でデバイスを動かして、新しいプロットの詳細を開くことです。 このような短編映画のデモは、WWDC 2017でARKitのプレゼンテーションとして誤って表示されませんでした。





40. Giphy World



名前が示すように、私たちはgifアニメーションについて話しています。 アプリケーション開発者は、GIFアニメーションを検索および編集するための有名なサービスです。 このアプリケーションを使用すると、2Dアニメーションと漫画画像を配置し、それらを実空間の特定の場所に添付できます。 デモビデオに示されているように、アプリケーションを使用して、ユーザーは、アニメーション画像の形で空を飛んでいる説明を使用して、たとえば乳母の指示を作成できます。
