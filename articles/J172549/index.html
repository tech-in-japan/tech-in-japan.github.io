ウラジカフカスでは、歴史的な市内中心部の建物にQRコードが掛けられていました

市について


ウラジカフカスは、ロシア連邦の南の前post地である北オセチア・アラニア共和国の首都です。 1784年5月6日に、公式の人口が30万人を超える都市が要塞として設立されました。 街の中心部：プロスペクトミラ（旧アレクサンドロフスキー）、レーニン通り（旧ドヴォリャンスカヤ）、ミラー通り、バラエフ通りは19世紀後半から20世紀初頭の建物で建てられています。





要塞ウラジカフカス。 アーティストA. Dyakonovの写真。







「相互信用協会」の銀行ビル。 プロスペクトミラ、15。1908年の写真



プロジェクトについて


2012年12月末に、My Interactive Cityプロジェクトのパイロットステージが開始され、ウラジカフカズ市の歴史的部分にある12棟の建物にQRコード付きのアドレスプレートが配置されました。 3月上旬にプレートが設置され、プロジェクトが稼働しました。





新しいアドレスプレートを備えた共和党アカデミックシアターの建物。



アイデアは新しいものではありません。 2011年に戻って、彼らはドバイのすべての建物が独自のQRコードを受け取ることをHabrに書きました（ http://habrahabr.ru/post/117118/ ）。 ただし、ロシアでは、このプロジェクトは非常に新鮮です。地域でのQRコーディング技術の使用はまだ始まったばかりです。



ユーザーに提供される情報について少し


コードは、モバイルデバイス用に特別に設計されたインターネットリソースへのリンクのみを暗号化しました。 このサイトには、各建物の履歴情報（建設年、建築家、そこに住んでいた有名人など）が含まれています。 異なる年の建物の写真が公開されています。 マップ上の建物の場所が示されます（Yandexマップが使用されます）。 オブジェクトのパラメーターは簡単に構成でき、必要に応じて他の情報を簡単に追加できます。

歴史情報は、プロジェクト「Old Vladikavkaz」の創設者から親切に提供されました。



これは、ユーザーがモバイルデバイスの画面に表示するものです









システムについて


技術的には、プロジェクトはCMF Dotnetnuke Community Editionバージョン6.2.xのモジュールのセットです。 したがって、これはすべて、Microsoft Windows Server 2008 R2 with IISで動作します。 DBMS Microsoft SQL Server 2012を使用しました。ASP.NET 4.0。 モジュールはC＃で記述されています。
