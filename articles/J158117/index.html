Canonicalはゲーム開発チームを作成することにしました

Habréでこのニュースがスリップしなかったことは奇妙です。 ニュースは最初の新鮮さではありません。 （〜11月2日）







Ubuntu Developer Summit（Ubuntu Developer Summit）は、Ubuntuのゲームプラットフォームとしての開発に取り組む開発者の別のチームの作成について積極的に議論しています。 このコマンドのページはすでに作成されています： launchpad.net/~ubuntu-game-dev 。



Daniel van Vugtは Ubuntu Game Developersグループの作成者でした（彼はCanonizのCompizとUnityで働いています）。 グループには、Compizのメンテナー、Unity 3Dの3人のエンジニア、有名なゲームエンジンを開発している会社、および約10人の関心のある開発者も含まれていました。



Ubuntu 13.04では、Unityのパフォーマンスに積極的に取り組むため、このようなチームの作成は明らかであり、Ubuntu Game DevelopersチームはUbuntuのゲームのパフォーマンスに関する問題を解決します。 まあ、 Steamのリリースは明らかにチームの作成に何らかの形で影響を与えました。
