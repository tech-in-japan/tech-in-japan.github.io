Compute Module、2019モデル





2017年初頭、Intelは拡張カードのフォームファクターの組み込みコンピューターであるCompute Cardを発表しました 。 その年の終わりに、このアイデアは肉体、血、そしてコストを獲得しました。IntelCoreプロセッサをベースにした2つのモデルがリリースされました。 それ以降、新しい世代のCompute Cardにデータが表示されるまで、それらについてのニュースはありませんでした。これは、さらに興味深いことにCompute Moduleと呼ばれていました-Intel Xeonプロセッサーに基づくIntel Data Center Blocksプログラムのサーバー用Compute Moduleと混同しないでください。 今、私たちはどうにかそれと共に生きなければなりません。



Intel Compute Module（新しい名前に慣れましょう）のアイデアは、あらゆるもの（車、冷蔵庫、アクセスポイント）のすべての「精神的能力」を、取り外して必要に応じて修理できる専用のモジュールに移すことです。または別のものに置き換えられます。 このコンピューターは、プロセッサー、ストレージ、相互作用インターフェース、単一コネクターでホストに接続するという絶対に標準的なアーキテクチャーを備えています。 「モノのインターネット」のあらゆるコンポーネントの一種の「普遍的な脳」であることが判明し、多くのものであると考えられています。



前回同様、2019年にはIntel Core（第8世代）、Pentium、およびCeleronに基づいたモデルが発表されました。 2つのラインの違いは注目に値します。現在、上位モデルには8 GBのメモリがあり、すべてにギガビットWi-FiアダプターとBluetooth 5.0モジュールが装備されています。 Intel Core i5-8365Uに基づくモデルは、名前の文字Vから明らかなように、 Intel vProテクノロジーのサポートを主張しています。

さらに、Compute Moduleをインストールするために3つのシャーシをリリースする予定です。それらはすべて、さまざまなスマートデバイスに組み込まれるように設計されたパッシブ冷却と保護されたパフォーマンスを備えています。 Compute Module NUCに最も興味がありました-Compute Moduleカード用の汎用アラカルNUCミニコンピューター（ファンなし、はい、はい）。 見たいです。



新しいCompute Moduleのリリースは2019年末に予定されています。 どうやら、これはIntel Coreに基づいたカード用のドライブのタイプがまだ公開されていないという事実を説明しています-新しいものを置く時間があります。 今後2年間のその他のモデルのリリースは計画されていません。
