ServiceNow Kingstonアップデート：機械学習機能とプラットフォームのアップグレード

12月中旬、ServiceNowは同名のプラットフォームの新しいバージョンであるServiceNow Kingstonのリリースを発表しました。 次の変更が加えられました。



  • ビジネスプロセスを最適化するための機械学習サービスの機能を実装しました（Agent Intelligence）
  • 機器の誤動作を予測するために教えられるプラットフォーム
  • Nowプラットフォームに追加された新機能


これらの改善の詳細については、以下で説明します。





/ Flickr / ケニールイ / cc



ビジネスプロセス最適化のためのエージェントインテリジェンス



私たちの投稿の1つで、2017年1月にServiceNowが機械学習DxContinuumの分野でスタートアップを買収したことが言及されました。 この決定は、テクノロジーを使用してビジネスプロセスを最適化するために行われました。 彼女は、Now Platform製品でアプリケーションを見つけ、Agent Intelligenceと呼ばれました。 開発者によると、システムはタスクの分類と割り当てを自動化し、エラーの数を減らし、インシデントを解決する時間を短縮します（ソリューションの動作の詳細については、こちらで説明しました ）



インテリジェントシステムは、9万のクライアントデータに基づいてトレーニングされ、チケットを予約する際に分類の平均精度が71％であることを示しました 。 同時に、インシデントの67％が自動的に解決されました。 ServiceNowは、エージェントインテリジェンスサポートでインシデントを解決するために費やされる平均年間時間は26,000時間であり、マテリアルゲインは27000ドルであると推定しています。



ハードウェア予測



IHS Markitの調査によると、 IT機器のダウンタイムには年間7,000億ドルの費用がかかり、これは北米のみです。 ServiceNowの戦略計画マネージャーであるDave Wright氏は、このようなコストは中小企業や経済全体にとって大きな打撃であると主張しています。 そのため、Service NowはIntelligent Automation Engineを開発しました。



このメカニズムは、過去に機器の故障につながったイベントを分析し、それに基づいて、故障前の機器の動作パターンを自動的に作成します。 システムは特定のコンポーネントが機能していないことを報告するだけでなく、数時間または数日で故障を予測するため、専門家が問題をすぐに修正できます。



具体的な例を見てみましょう。アプリケーションのワーカースレッドが多すぎる場合、システムに報告します。 これは正常な動作であるため、このイベントには通常、低い優先度が割り当てられます。 ただし、このような状況が頻繁に発生する場合、これは差し迫ったCPU使用率の指標です。



従来のイベント管理ツールは、システムがひどく混乱した場合に重大な警告を発します。 また、Intelligent Automation Engineは、アプリケーション内で繰り返されるワークロードの輻輳を検出し、CPUの過負荷の可能性について事前に警告を発します。



ライトは、将来そのようなインテリジェントシステムが急増すると、MTBFなどのパラメーターを除外できるようになる可能性があると指摘しています。



Nowプラットフォームのその他の改善



新しいリリースでは、企業のビジネスプロセスを自動化するためのNowプラットフォームプラットフォームに、（エージェントインテリジェンスに加えて）その他の機能と改善が加えられました。



  • Flow Designer：コードを記述せずにビジネスプロセスフローを作成および管理できます。 承認自動化手順の自動化と、ServiceNowデータの管理についてです。
  • IntegrationHub： Flow Designerをサードパーティシステムに接続することにより、ServiceNowストリームを外部サービスに簡単に接続できます。 これにより、誰でもコードを記述せずに自然言語でワークフローを作成できます。 Microsoft Teams、Slack、およびHipChatとの統合があります。
  • MetricBase：開発者は、時系列データを分析するための25の新しい関数（最小、最大、集計など）を追加しました。 イベントを起動するためのしきい値を設定し、新しいPerformance Analyticsレポートなどの高度なレポートを作成できるようになりました。
  • エッジ暗号化 ：コントロールパネルは、プロキシ設定での作業を簡素化します。 添付ファイルを追加するときにジャーナルフィールドを暗号化することもできます。
  • ガイド付きツアーデザイナー ：自動スタート機能と、仮想ツアーを通過した後にフィードバックを収集する機能が追加されました。
