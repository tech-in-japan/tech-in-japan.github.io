Windows 10オペレーティングシステムのソースコードがネットワークに漏洩した[小さな部分]

UPDレジスタが大幅に誇張されていることが判明しました。







theregister.co.ukによると、最近Windows 10のプライベートビルドとそのソースコードの断片が大量にリークされています。



公式および非公開の画像、非公開の技術文書、ソーステキストで構成される32テラバイトのデータ（アーカイブ形式-8テラバイト）の配列がbetaarchive.com にアップロードされました。



このダンプの機密データは、2017年3月頃にMicrosoftの内部ストレージから違法にコピーされたと考えられています。



印象的なアーカイブの資料に精通した人々によると、その中のリークされたソースコードは、Microsoftの共有ソースキットに属します。 このセットには、基本的なWindows 10ドライバー、Wi-Fi、USBおよびPnPスタック、ストレージシステムドライバー、OneCoreカーネルのARMバージョンのソースが含まれています。



現時点では、インシデントが一度にWindows 2000ソースコードのリークと同じくらい重要であるという前提条件がすべてあります。







この「休日」に加えて、リークされたアレイでは、Windows 10およびWindows Server 2016の非常に公式なビルドの中で、パブリックアクセスを意図したものではないが、バグや実験的テストのために会社のエンジニアによって使用された秘密コピーも見つかりました。







登録簿の編集者であるクリス・ウィリアムズによると、インターネットは、Windowsの最新バージョンのコードにこれまで知られていない脆弱性を使用する新たな悪用とウイルスの波を待っています。



リークの原因とインシデントの状況はまだ明らかではありません。 発行時点のマイクロソフトの代表者は、状況についてコメントしていませんでした。
