WiFi /インターネットを介したnokia 5800へのネットワークアクセス

ほんの少し前に、 Symbianに組み込まれたクライアントを使用してWiFi /インターネット経由でコンピューターのディスクにアクセスする方法について、hawraiser imwodeの記事を読みました。 そして、すべてがうまくいくでしょう、しかし、ノキア5800で我々の尊敬されたフィンランドの友人はデフォルトでこのクライアントを追加することを気にしませんでした。 しかし、私はこれについて心を失いませんでしたし、必要な解決策を探してサーフィンに行きました。 結局のところ、役に立たなかった。

現在、電話とTelexy Networks、Inc.のコンピューター間のワイヤレス（WiFi、3G、GPRSなど）アクセス用の一連のプログラムはすべて、シンビアンスマートフォンで利用できます。 。 プログラムは、タッチスクリーンを完全にサポートし、便利なユーザーインターフェイスを備えており、他にも多くの利点があります。



SymVPN -S60用PPTP VPNクライアント（第3および第5版）。 設定は簡単です。 接続の名前を設定し、アクセスポイント（WiFiまたはGPRS）を選択し、サーバー名を指定し、ログインとパスワードを入力します。 WiFiを介して正常に機能しますが、OPSOSネットワーク経由で接続できませんでした。おそらくルートが複雑です。 プログラムは、設定メニューのアクセスポイントのリストに表示されないアクセスポイントを作成しますが、インターネット要求の高いプログラムからアクセスポイントを直接選択すると表示されます。 Mail for Exchangeと組み合わせて使用​​できます。







SymSync-スマートフォン上の指定されたフォルダーをローカルコンピューター上のフォルダーと同期するためのプログラム。 同期間隔を割り当て、同時に複数のプロファイルを設定できます。また、バックグラウンドで作業することもできます。 たとえば、Imagesフォルダーを設定し、週に1回、スマートコンピューター自体がホームコンピューターで撮影したすべてのものをマージします。







SymNAS-このプログラムを使用すると、コンピューターから電話ファイルへのアクセスを整理できます。実際、共有ネットワークフォルダーを取得します。 設定も非常に簡単です。 アクセスを許可するフォルダー（完全または読み取り専用）を指定し、ワークグループとアクセスポイントを規定します。 IPアドレスと名前の両方で電話に連絡できます。







SymRDP-このプログラムは、Symbianプラットフォーム上のRDP（リモートデスクトッププロトコル）のクライアント実装です。 S60 5番目の電話からRDPプロトコルをサポートするコンピューターへのリモートアクセスを提供するように設計されています。

-Microsoft Windows NT / 2000/2003/2008;

-Microsoft Windows XP Professional;

-Microsoft Windows Vista（Ultimate、Business、Enterprise）;

-Microsoft Windows 7（Professional、Enterprise）。

実際、一度見て、感じた方が良いです、個人的には、最初の接続から喜びを感じました）







設定には特別なものはありません。通常、リモートマシンのIPアドレス、アクセスポイント（ここではVPNが役立ちます）、ユーザー名/パスワードを示します。 はい。もちろん、ローカルマシンでリモートアクセスを開くことを忘れないでください。

画面の解像度は、電話機の解像度（nokia 5800の場合は640x360）、またはスクロールバーが表示される1280x720に等しく設定されます（スクリーンショットには表示されていません）。 スタイラスの管理は非常に便利ですが、応答時間によってはもっと良くなりたい場合もあります。 私が理解しているように、ビデオとオーディオの送信はサポートされていません。

最後に、 SymNC-このプログラムは、上記のすべてに加えて、ネットワークのサポートを備えた便利なファイルマネージャを組み合わせています。
