GNU / Emacs 23.2リリース

GNU / Emacsは、1976年に伝説のリチャードストールマンによって作成された強力なテキストエディターであり、現在も進化しています。 これは、ドキュメント、拡張性、およびここでほとんどすべてを構成できるという事実によって特徴付けられます。



GNU / Emacsを使用してソフトウェアを開発するだけでなく、メールの読み取り、ジャバー経由の通信、Webの閲覧など、他のタスクを解決する人もいます。



偉大なドナルドクヌースはEmaxを使用しており、Linuxの作成者であるLinus Torvalds は彼について次のように 書いています。

Gnu / Emacsはエディターとして始まりましたが、その作成者はさまざまな機能を組み込みました。 彼らはそれをプログラム可能なエディターにしたいと考えていましたが、プログラミングの部分は想像を絶するサイズに成長し、エディターは本当に悪夢になりました。 キッチンシンク以外のすべてが含まれているため、キッチンシンクがアイコンとして機能することがよくあります。 このプログラムは、怪物と呼ばれ、エディターが原則として必要とするよりも多くの機能を備えています。




バージョン23.2の革新と変更のリストは、ENTにあります （かなり大きい）。 Emacsの詳細については、 WikipediaおよびAlex Ottページをご覧ください 。 EmacsのストーリーはCommonwealth of Emacsの記事にあります。 Emaxをマスターする最初のステップは、 Emacs PointとEmacsの記事を便利にするのに役立ちます。 Emacswikiとdotemacsは、エディターをシャープにするのに役立ちます。
