Raspberry Piでの独自のプロジェクトのインスピレーション





Raspberry Piでのプロジェクトのミューズをお探しの場合は、MCM Electronicsの傘下で最近開催された世界初のフェスティバルRaspberry Pi Design Contestの参加者の作品に注目することをお勧めします。 すべてのプロジェクトは、芸術、教育、貝殻、家庭の4つのカテゴリに分類されます。 参加者は、大賞-Printrbot Jr 3Dプリンターと公認を競いました。 実際にはたくさんの贈り物がありましたが。



Raspberry Piがリリースされてから1年も経たず、愛好家は既にRaspberry Piを使用して多くの興味深いデバイスを収集しています。 Piは多くのホームセキュリティシステムの頭脳になり、あらゆる種類のデータの収集と表示を担当し、数学や地理学の教育ツールとしても使用されています。 アーティストはそれに基づいてインタラクティブなインスタレーションを作成し、Piのインテリアのケースは原始的なものから複雑で複雑なものまでさまざまです。 別のPiは、さまざまなホームシステムおよびオフィスシステムを管理できます...しかし、そこに何があるのか​​、このリストは延々と続きます。







コンテストの一環として、審査員はプロジェクトの説明を研究し、その成功と独自性を評価し、受賞者を選択しました。 私は、より多くのインスピレーションを手に入れて、競争に勝った5つのプロジェクトに精通することをお勧めします。 さらに多くのアイデアが必要な場合は、フェスティバルのすべての参加者と知り合うことができます。



だから、勝者





1.イントナールモリ

urbanSTEWチームは、楽しいIntonarumoriプロジェクトで大賞を受賞しました。







urbanSTEWチームは、前世紀の未来派の創造物の精神で、いくつかのインタラクティブな「ノイズボックス」を組み立てました。 その動きの一部として、一連のノイズ発生マシンが作成され、多くのセンサーを備えたRaspberry Piに基づいて独自の「ミュージック」ボックスを構築するようにurbanSTEWを刺激しました。 ボックスは自給自足で動作するように準備されています-コンセントに差し込むだけでプレイを開始できます。 Intonarumoriは 1つのクリエイティブフェスティバルでさえ提示され、2000人以上の人々が箱に興味を持ちました。



主な勝者として、urbanSTEWチームは、 Printrbot Jr 3DプリンターとMCM Electronicsからのボーナスパッケージを受け取りました。これには、Pi Face、Pi View HDMI-VGAアダプター、Wi-Piワイヤレスアダプター、デジタルはんだごてなどの便利なものが含まれています。 この記事の残りの勝者は、20000回以上の摩擦に相当する同じセットを受け取りました。



2.ロビイストメー​​ター

アート部門のリーダーは、トムリーと彼のロビーストメータープロジェクトで、登録時の参加者数をカウントします。







トム・リー： 「私は珍しいものの店で見つけたビンテージの電圧計を取り、その電圧範囲をPiにより適したものに変更しました。 さらに、できる限り正確に情報を表示できるように、デバイスをさらに調整する必要がありました。 そして、すでにデザイナーの助けを借りて、古い電圧計を、毎週登録されている参加者の元のカウンターに変えることができました。」





3.州のポスター

教育製品の中で、 State Posterは際立っていました-アメリカのバーモント州の電子地図。







スコットベネット： 「毎年、私の娘の学校では、2年生に特定の州のポスターカードを個別に作成するタスクが与えられています。 それ以前は、彼女はすでにこのようなイベントに数回参加しており、最近でははんだごてを使った作業を習得しました。 短いブレーンストーミングセッションの後、この典型的な仕事に少しの電子機器とRaspberry Piを追加するのは素晴らしいことだと思いました。 娘はエンジニアリングに興味があり、Piを使用することで、プロジェクトに取り組んでいる間に多くのことを学ぶことができました。





4.クルミラズベリーPiエンクロージャー

Piの最適なコンテナは、きちんとしたワンピースの クルミエンクロージャークルミボックスでした。







クリス・クランパーカー： 「木材くずはインスピレーションに十分な場合があります。 私は前回のプロジェクトからちょっとしたことをしましたが、一般的には木材を扱うのが好きです。 私はたくさんの木製のケースを見ましたが、それらはすべて6つの部分が接着されているか、一緒にノックされていました。 Raspberry Pi用に無垢材を作りたかったのです。」





5.猫フィーダー

家庭のカテゴリーから、裁判官は自動猫送り装置-Cat Feederの非常に有用なプロジェクトに注目した。







デビッド・ブライアン： 「2月に、ハックファクトリーコースでRaspberry Piの勉強を始めました。 最初のレッスンの後、Piは私の脳にしっかりと立ち往生し、私のガールフレンドと私は週に一度の短い旅行に出かけていました。 彼女は数匹の猫を飼っていましたが、旅行中は大きなボウルと一緒に猫を家に残そうとしていました。 自動猫フィーダーを組み立てることを提案しました。 プロジェクトはすでに完了しており、完全に運用されています。」
