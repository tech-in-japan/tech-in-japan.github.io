サマープラクティス：Microsoft Virtual Academyのトップ10コース

Microsoft MVAバーチャルアカデミーで引き続きスキルを向上させます 。 最後の選択では、初心者向けの最も興味深いコースがありました。今日は、トレーニングを受けたスペシャリスト向けの人気のあるコースを学びます。企業ネットワークのWindows 10、JSONおよびC＃、Microsoft AzureクラウドでのDockerコンテナーの使用、アプリケーションの災害耐性、Windows Server Hyper-Vサーバーの仮想化System Center、ローカルLinuxシステムをクラウドにインポートし、PowerShellでスクリプトを記述し、PowerShell Desired State Configurationテクノロジ、およびLINQ、Data Science、Machine Learningを使用します。









1.企業ネットワーク上のWindows 10



Windows 10に移行するかどうかまだ決定していない場合、このコースでは、移行アプローチとツール、Windows 10のデバイス管理機能、およびいくつかのOSセキュリティテクノロジーを紹介します。



2. CでJSON形式を使用する方法の概要＃



JSON（英語のJavaScript Object Notation、JavaScriptオブジェクトの表記法の略）は、データを保存および交換するための形式です。 このコースでは、C＃でJSONを操作する方法、その構文と構造を理解する方法、JSONファイルを作成、分析、使用してアプリケーションデータをロードおよび保存する方法を学習します。



3. Azure IaaSクラウドでのDockerコンテナーの使用



ご存じのとおり、Dockerは開発者およびプロジェクト管理者にとって必須のツールであり、デプロイメントとアプリケーション管理の自動化を提供します。 Dockerを使用すると、コンテナで分離されたアプリケーションを安全に実行できます。 このコースでは、理論的側面と実践的側面の両方から、Docker管理システムとIaaSシステムでのその使用について学習します。



4.アプリケーションおよびサービスの耐障害性と高可用性の提供



フォールトトレランス、高可用性、障害後の災害復旧、およびAzure Site Recoveryなどの災害耐性システムの設計の原則などの基本概念について詳しく学習します。 クラウドサービスを使用して仮想マシンを保護し、データセンター間の非同期レプリケーションを管理し、事故が発生した場合にバックアップデータセンターを切り替えるプロセスを調整する問題についても説明します。



5. Windows Server Hyper-VおよびSystem Centerを使用したサーバー仮想化



このコースでは、ネットワークサービスやストレージサービスなど、Windows Server 2012 R2 Hyper-VおよびSystem Center 2012 R2 Virtual Machine Managerの構成、管理、およびメンテナンスの実行方法を学習します。 第二世代の仮想マシン、レプリケーション拡張、対話型エクスポート、異なるバージョン間のオンライン移行、インターネット経由のVHDXファイルのサイズ変更、動的仮想スイッチの動的移行と負荷分散の設定など、Microsoft Server仮想化機能の構成方法も同様です。



6.ローカルLinuxシステムをクラウドにインポートする、またはその逆



このコースでは、イメージを最初から準備し、ローカルLinuxシステムをMicrosoft Azureクラウドに、またはその逆に転送する方法と、原則として存在する転送オプションについて学習します。 コース作成者は、既存のシステムがKVM仮想化システムで動作し、QCOW / QCOW2、VHDなどの既製のイメージとして利用できる場合、または既存のシステムがVServer / OpenVZコンテナまたはベアメタルで動作する場合、利用できません。



7. PowerShell 3.0の高度な機能とスクリプト（ジャンプスタート）



リアルタイムの管理および自動化スクリプトを有用なツールおよびコマンドレットに変換する方法は？ 簡単！ これについてはコースから学び、PowerShellの開発者および作成者であるMicrosoftの主要なスペシャリストであるGeoffrey Sververから多くの有用なヒントを得ます。



8. PowerShellの望ましい状態の構成



PowerShellを使用すると、Windowsシステム構成タスクを完了できます。 Windows Management Framework 4.0の一部として、システムを宣言的に構成できるPowerShell Desired State Configurationテクノロジがあります。 このコースでは、このテクノロジ、構成スクリプト、DSC組み込みおよびユーザーリソース、および構成を複数のコンピューターに集中的に適用する方法について学習します。



9. LINQの秘密を明らかにする



言語統合クエリ（LINQ）は、Visual Studio 2008で導入された一連の機能であり、C＃とVisual Basicの構文を大幅に拡張し、データソースをクエリするためのシンプルで便利な言語です。 このコースでは、基本的なLINQ要素とその相互作用、クエリ式がメソッドにマップされる方法、およびパフォーマンスの問題などについて学習します。



10.データサイエンスと機械学習の基礎



データサイエンスと機械学習について質問がありますか？ コースでは、Azure Machine Learning、R、およびPythonを使用したデータの学習に取り組み、データサイエンスの基本的な方法とMachine Learningアルゴリズムの構築について学びます。



トレーニングを受けた専門家向けのその他のコースは、Microsoft Virtual Academy Webサイトの「 中級」および「 上級」セクションにあります。
