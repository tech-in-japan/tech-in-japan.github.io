注釈を介したjavacからの警告メッセージ用の軽量ライブラリ

こんにちはJava開発者。



私は軽量（3kb）ライブラリを共有しています。その唯一の主なタスクは、Javaコンパイラからの命令を投げることです。



使用方法は非常に簡単です。 警告アノテーション（「Reason text here」）を記述すると、コンパイルにより渦が発生します（同時にアセンブリは停止しません）。 



// some code... @Warning("Refactor as soon as possible please") public void someCodeWhichYouNeedAtTheMomentButYouWantToRefactorItLater() { // bad stuff going on here... }


使用された注釈プロセッサAPI 。



Github： https : //github.com/pushtorefresh/javac-warning-annotation （mavenCentralで公開）



役に立てば幸いです。



PSほとんどの場合、この注釈を使用するのではなく、タスクトラッカーでタスクを開始することをお勧めしますが、このオプションは非常に適切です。
