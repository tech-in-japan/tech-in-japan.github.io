中国政府向けの51％の攻撃またはポケットガイド

あなたは中国政府出身で、ビットコインに脅かされていますか？ 答えは明らかにノーであり、あなたはもっと重要なことをしなければなりませんが、逆を想像してみましょう。



誰もが安全で取り外し不可能な暗号通貨がどれほど安全かについてゴシップを語っています。 ビットコインは、単に数人を逮捕してサーバーをシャットダウンするよりもはるかに複雑です。



しかし、鉱夫がいて、それらのほとんどはあなたの国にあり、あなたがする必要があるのは「51％攻撃」の汚いトリックです。







ステップ1.購入



見つかったすべてのマイニングファームを購入し、確認済みのアカウントを登録します（各10〜40）。 したがって、約100のファームとそれぞれ40のアカウントがあります。 わあ 1億ドルのBTCを購入しましょう。 あなたは政府です、国家安全保障のためにそれを買う余裕がありますよね？



執筆時点でのビットコインの時価総額は950億ドルです。 0.1％

すべてのお金を自分のデスクトップウォレットに送ってください。



ステップ2.鉱山労働者の51％を見つける



サイトによると、71％は中国のどこかにあります。 彼らは匿名になろうとしますが、あなたは中国であり、人々が自分の電話にスパイウェアアプリケーションをインストールすることを余儀なくされているので、これは問題になりません。



ステップ3.シークレット操作



次に、秘密の法執行機関に依頼して、X日目にできるだけ多くの鉱夫を見つけます。



これは法執行機関の操作であるため、役員は武器を頭に向けることができます。これにより、BTCが何であるかわからないふりをすると、思考プロセスがスピードアップします。



これは、政府が目的を持っているときの仕組みです。 そして、あなたは中国なので、これをすることでリベラル派のように見えるようにする必要はありません。



ステップ4.時間を2倍に！



彼らに次のことをさせましょう：彼らはあなたのデスクトップウォレットから1ブロックで約4,000トランザクション（それぞれ25,000ドル）を発行し、それらを全員に送信する必要があります。 これらの4,000件のトランザクションは、ステップ1で作成されたすべてのアカウントに送金します。



通常、取引所では、取引の終了を検討するために、上部に構築された6ブロック（確認）が必要です。 そのため、ネットワークの残りの部分は、あなたの上にブロックを作成しようとします。 彼らがこれを行う間、blockNを取得し、blockN + 1を全員に送信して、blockN + 2とblockN + 3で作業を続け、彼ら自身がblockN + 1BとblockN + 2Bの開発に集中するようにします。



あなたは51％を持っているので、おそらくあなたは他の誰よりも速くなるでしょう。 鉱夫がブロックの確認を6回受け取ったら、すべての交換でBTCの交換を要求する必要があります：ライトコイン、リップル、イーサリアム...何でも。 彼らは通常、自動的に交換します。 次に、これらのすべてのアルトコインをデスクトップウォレットに再度送信する必要があります。

すべてのBTCをAltcoinsに交換したら、別の「秘密」ブランチを開始します。これは、基本的に4000トランザクションでブロックをキャンセルします。



これは再編成と呼ばれます。ネットワーク上の全員がメインチェーンとしてチェーンを受け入れ、現在「孤立」している他のすべてのトランザクションを忘れる必要があります。



ステップ5.価格= 0



これは、1億ドル相当のビットコイン（もう交換されていない）がアルトコインに交換されたことを意味します。 ダブルトランザクションにおめでとう、2億ドル相当の暗号通貨が手に入りました！ それらの価格は0になりますが、これは当初の目標です。



プロセスを再度繰り返すことができますが、必要はありません。 この規模の攻撃は、今後数年間、暗号通貨に対する信念を破壊します。



他の国からの否定的な反応はありません-国または管轄外のBTC。 そのため、プライベートでもパブリックでもできます。
