中国の3DプリンターWanhao Duplicator 4を購入、「完成」してセットアップした方法

3Dプリンターを購入して税関を通過するという最近の悲惨な話の後、中国のプリンターであるWanhao Duplicator 4を購入して仕上げるという劇的な話をしたいと思います。



仕事よりも興味を引くためにプリンターが必要なため、ヒキガエルはMakerbot Replicator 2に3kドルを与えませんでした（+彼はABSで印刷する方法を知りません）。 DIY KITオプション（PrintrBot Junior KITなど）がありましたが、ここでは多くの自由時間が必要であり、構造の機械的強度に関する質問があります）、Kickstarterの多数のオプションは高価ではありませんが、遅く、有名なUltimaker 1/2は良いですが、Replicatorのように高価です特に組み立てられた形式（免税限度よりも著しく高い）。



最後に、aliexpressで検索-興味深いモデルを見つけました-Wanhao Duplicator 4：2台の押出機（2色印刷と2つの小さな同一部品の両方を印刷できます）、一見したところデザインはReplicator 2に似ています（プレキシグラスの信頼なし） -それは水分から歪む可能性があります）、ABS印刷用のホットテーブル、キットから選択する2つのプラスチックコイル、インターネット上でレビューがあります。 問題の価格は、配達を考慮して、収集されたものに対して1142ドルです。 別の売り手は、「暗緑色で光る」（最後の結果を参照）を含む、ABSプラスチックのかせ3本を29ドル（配送料込み）で注文しました。



しかし、もちろん、すべてが非常に単純に「購入した、受け取った、印刷された」終了した場合、この記事はほとんど生まれなかったでしょう。



配送

プリンターは32日で私に届きました。「ラストン」による配達-国境を越えた後、ロシア郵便による再送信。 伝統的に、彼らは税関で彼らを拘束していませんでした（私が彼らに2回行かなければならなかった購入の歴史で、彼らは小包の約1％を無作為に選んだようです）。 彼らが拘束されていた場合、費用（配達で1142ドル）と重量（17kg）で、彼らはまだ免税の国境にいます。 もちろん、彼らはそれが「個人的な使用のためではない」という事実について不満を言うかもしれません-しかし、これは運の問題/運ではありません。



領収書の日に郵便局は閉まり（従業員が病気になった）、残りのマネージャーは私の携帯電話を箱で見つけ、個人的に私の携帯電話に電話し、小包を取りに来るように言った。 彼が来て、彼らは私のメールを開きました-そして、個人的に小包を与えました。 OPS-308部門への良い光線（インデックス-123308）。 しかし、ロシア郵便の明るい側面もあります。



電源を入れてドラマを開始する

彼らは持ち出し、開梱しました-一見したところ、ビルドの品質は優れています。 プレキシグラスは厚く、構造は耐久性があり、動きません。 ソフトウェア-標準：ReplicatorG + Skeinforge / slic3r、ハードウェア：MightyBoard。







メモリカードからテスト「キューブ」の印刷を開始します。印刷の20分、誰もがプリンターから目を離しません（最初に洗濯機の電源を入れたときのように）。 ただし、結果は非常に大きくなります。









角は曲がっており、立方体はまったく立方体ではありません。 なぜ高さは幅の半分ですか？ 2日目、私はそれを理解し、メカニック/ Z軸のステップを非難しました-しかし、すべてが些細であることが判明しました：そうである必要があります-モデルはとても塗装されています しかし、タンバリンとのダンスは、通常、印刷コーナーを印刷しませんでした...さまざまな設定で50回試行した後、行き止まりになったことが明らかになりました。







メーカーとの紛争と修理

aliexpressの紛争を開きました。 製造業者は、発見された印刷品質の問題と機械的な問題の両方を解決するのを手伝うことができませんでした：Y軸ステップの回転を伝えるわずかに曲がったシャフトと湾曲したテーブル：中心は端よりも約1 mm低い-これにより、最初のレイヤーの印刷高さを正常に設定できません。 キットにはガラスが含まれていましたが、テーブルの端だけに触れていました。したがって、テーブルのさまざまな場所で非常に大きな温度差がありました。







メーカーは曲がった部品の交換品を私に送ることを提案しましたが、2週間の口論の後、中国から巨大な英語で電話をかけました（彼らは紛争を終わらせ、「すべての問題を解決する」ことを約束しました）-最後に売り手は200ドルの部分的な払い戻しをしました。 私はebayで新しい5mmシャフトを注文しました-1ヶ月後に来ましたが、同じ曲がり（約3-4mm）でした。 平らなテーブル、マーカー、懐中電灯などの助けを借りて、30分で元のシャフトを揃えて元に戻しました（余分な努力-そしてシャフトはそれよりも逆方向に曲がりました）。



ガラスの温度は、サーマルペーストKPT-8で平準化されました。 アルミニウム製のテーブルのカプトンフィルムは引きちぎられました-とにかくガラスを使用しても意味がありません。 Glassは、ドキュメント用の「洗濯挟み」が付いたテーブルを保持しています。







5mmのスチールシャフトは非常に簡単に曲がるので、歯付きベルトを揃えた後でも張力によって曲がることがわかりました。 ベルトの張力を減らす必要がありました。これにより、遊びが増える可能性がありますが、シャフトがさらに曲がるという問題がほとんど減りました。



ただし、これらのメカニズムのすべての修正後、印刷結果はまったく異なりませんでした。 ソフトウェア設定は残ります。



SkeinforgeをカスタマイズしてABSを印刷する

ここに書かれていることはすべて、ABSプラスチックを使用した印刷に関するものです。 私が選んだのは 強度と耐久性が高く、アセトンで処理できます。 しかし、一般的に、PLAプラスチックを使用すると、高品質を達成することがはるかに簡単になります。 印刷時のABSの臭いについてのホラーストーリーについて-私はこの問題に気付きませんでした（少なくとも私のプラスチックでは）。臭いはあまりありませんが、もちろん放映を忘れてはいけません。



skeinforgeの設定を初めて見たとき、髪の毛が逆立ちしていました。 あなたがねじることができる数百のパラメータがあります-彼らの腕のすぐ下。 しかし、役立つオプションは多くないことが判明しました。 印刷の問題が温度が高すぎることを理解しようとすると、約1か月かかりました（テーブルと押出機の両方）。 このため、プラスチックは柔らかく、非常に曲がります。 最終的に、真実に似た最初の結果は次の方法で達成されました。



  • テーブルの整列は最も重要な部分です。 最初のレイヤーは、テーブル上の薄いレイヤーで直接塗り付けます。
  • 加熱されたテーブルでは、ラフティングなしで印刷する方法を学習するのが理にかなっています（モデルの下部にあるメッシュ）-これにより、底面が滑らかになり、プラスチックが少し節約されます。
  • モデルの接着性を向上させるために-あなたは紙テープを使用することができます（安くて陽気、よく貼り付けます-メーカーはすぐにロールを置きます）。 アセトンABSに溶解したテープ/ガラスを潤滑することができます。 小さなモデルはガラスに直接印刷できます-底面は鏡面です。 また、金属ブラシで粘着テープをはがすと効果があります-わずかに凹凸のある表面です。 しかし、何らかの理由でダイヤモンドペーストとGOIを使用してガラスを「掃く」ことは、大きな影響を与えませんでした。おそらく、スクラッチが小さすぎたのでしょう。
  • テーブルの温度は70〜90度です。 モデルがねじれるまで、低いほど良いです。 モデルが小さいほど、使用できるテーブルの温度は低くなります。 たとえば、小さなモデルの場合、最初のレイヤーに80度を設定し、印刷の高さを0.5 mmから30度以上に設定します（=加熱をオフにするには、[チャンバー]タブで行います）。 大きなモデルの場合、テーブルは常に保温する必要があり、ねじり力ははるかに大きくなります。
  • 押出機（温度タブ）の温度は、プラスチックを正常に押し出すことができる最低温度です。 私の場合、約195〜200度です。 接着性を改善するために、最初の層を少し熱く印刷することができます。 温度はおそらく各プリンターごとに個別に選択する必要があります-温度センサーはわずかに異なる数値を生成する可能性があります。
  • 異なるパーティのプラスチック-直径が異なる場合があります。 プリンターを使用する場合は、キャリパーを使用して、各ロールを5箇所で測定し、平均を取る必要があります。 一部のバッチ-1.75mm、他のバッチ-1.82mm。 この直径の違いは重要です。 skeinforgeウィンドウでgコードを生成するときに、プラスチックの実際の厚さを設定する必要があります。
  • 密着性を向上させるために、スカートvkadkaでは、最初のレイヤーのリムの印刷を有効にできます。 周囲幅のギャップ= 0.8。
  • レイヤーの周囲と内側の間に隙間がある場合-フィル->インフィル周囲のオーバーラップを0.45以下に増やしてみてください（十分でない場合）、さらにストレッチ->パスストレッチオーバーオーバーボーダー幅（最大〜1まで）
  • [ボトム]タブ-高度を変更することにより、最初のレイヤーの高さを微調整して、最初のレイヤーのスミアリングと接着の望ましい程度を達成できます
  • 小さな部品の印刷には問題が残っています。 プリンタはあまりにも速くそれらを印刷します-そして、彼らは冷却する時間がありません。 最も簡単な解決策は、[クール]-> [最小レイヤー時間]タブでレイヤーの最小印刷時間を10〜15秒に制限することです。 Cool Type = Orbit（標準設定では、単に印刷速度が遅くなり、印刷された押出機を引き続き高温の押出機で運搬します）。 この機能のおかげで、より現代的で高速なSlic3rに切り替えることができませんでした-Orbit印刷を遅くする可能性はありません。
  • 印刷速度（[速度]タブ）-30mm / sの速度で印刷し、45mm / sの速度で移動しているとき。 さらに、プリンターの設定でAccelerate = ON機能を有効にすると、インフィルの印刷速度を50-60mm / sに上げることができると思います（これにより物理的な加速が制限されます）。




数百回の試行の後、最終的に何かが始まります。 これは、 5mmキャリブレーションキューブステップの品質をチェックするための非常に便利なモデルです。







出発するには、多くの人に馴染みのあるフクロウを印刷する必要がありました。 Skeinforgeは、信頼性のために、SDカードから印刷したgコードを約8分間うなりました。 結果：フクロウは確かに完璧ではありませんが、トンネルの終わりの光はすでに見えています。 フクロウは50gのプラスチックを取りました。これは約45ルーブルです。 1.5分のビデオで6時間の重労働プリンター。 マシン自体が実際に何かを行うのを見るのは素晴らしいことです。

しかし、暗闇で光るプラスチックは期待に応えられませんでした-それは非常に薄暗く、長く輝きません（実際には、写真よりもすべてが悪いです-シャッター速度は長いです）。 だから、私は中国からそのようなプラスチックを購入することはお勧めしません。
