ゲーム用の2Dリソースを備えた10の便利なサイト





HTML5とモバイルプラットフォームの開発により、2Dゲームは流行に戻り、大手出版社の注目を集めています。 Unityなどのツールやゲームアセットを含むオンラインアセットを使用すると、これらのゲームの開発がはるかに簡単になりました。 以下は、ゲーム用の2Dリソースを備えた上位10サイトのリストです（有料と無料の両方）。



*サイトはランダムな順序でリストされています*



1. Unity Asset Store （有料）



有料のゲーム資産がある最大のサイト。 Unityエンジンを使用した作業はオプションです。AssetStoreのリソースにアクセスするには、無料バージョンをダウンロードするだけです。 ダウンロードしたファイルはUnityプロジェクトフォルダーに保存され、そこから任意の2Dゲームエンジンにインポートできます。



多数の成功した2Dゲームで証明されているように、熟練した開発者の手にあるUnityには大きな可能性があることに注意する価値があります（Rovioの人気のあるBad Piggiesにのみ価値があります）。 また、Unity Asset Storeは、2Dアセットの幅広い選択肢を提供します。







2. GameDevマーケット （有料）



GameDev Marketは、コンテンツのカテゴリ（および後述のスーパーゲームアセット）に便利に整理された比較的新しいサイトです。 ここには、UI、キャラクタースプライト、アイコン、環境など、2Dと3Dの両方のリソースがあります。 アーティストは資産を売りに出し、その範囲は日々拡大しています。







3. Graphicriver-ゲームアセット （有料）



Graphicriverは、最も人気のあるテンプレート画像リソースの1つです。 カジュアルモバイルゲームの開発に関連して、2Dアセットを持つ別のカテゴリが登場しました。 サイトに投稿する前に、すべてのコンテンツが内部検証を受け、公開後、ユーザーは評価とフィードバックを残すことができます。 合計：これは、Envatoの緊密な節度の下で、信頼性が高く、積極的に開発されているサイトです。







4. Scirraストア （有料）



Scirraは、最近独自のアセットストアを持つ人気のHTML5 2DエディターであるConstruct 2を開発している会社です。 ここでは、コンストラクト2のオーディオ、グラフィックリソース、既製のゲームテンプレートを見つけることができます。ただし、他の2Dエディターを使用して作業することもできます。







5. ゲームアートパートナー （有料）



Game Art Partners Webサイトでは、アニメーションキャラクター、モンスター、武器、視覚効果、インターフェイス要素のセットなど、2Dプラットフォーマー向けのさまざまな漫画アセットを購入できます。







6. スーパーゲームアセット （有料）



RPGまたは等尺性グラフィックスを使用するゲームを作成している場合-このサイトはあなたのためです。 ここには、最高品質のアセットがあります：カラフルなRPGアイコン（おそらくオンラインで利用可能な最高のもの）、2Dスプライト、アニメーションキャラクタースプライト、巨大な等尺性マップ。 重要なことに、提示されるリソースのほとんどは同じスタイルで作成されています。







7. オープンゲームアート （無料）



オープンソースのゲーム開発者向けの包括的なリソースであるOpen Game Artは、無料ライセンスを持つゲームアセットの最大のコレクションと言えます。 スプライトからアイコンまで、すべてのマテリアルは、GNUまたはCreative Commonsライセンスの下でダウンロードできます。 これは初心者には素晴らしいサイトですが、多くのアセットは視覚的なスタイルが異なるため、慎重に選択する必要があります。







8. ケニーゲームアセット （無料）



UI要素や2Dプラットフォーマー向けのさまざまなスプライトなど、2万を超えるアセットを備えたもう1つの優れたサイトです。 ほとんどのリソースはベクターグラフィックスで表示され、画面の解像度に関係なく、あらゆるデバイスに適しています。 アセットは個別に（無料で）ダウンロードすることも、1セットで（わずか9ドルの料金で）ダウンロードすることもできます。







9. Game-Icons.net （無料）



Game-icons.netは無料のアイコンを備えた最高のサイトです。2,000以上のアイコンがあります。すべてのアイコンは白黒であるにもかかわらず、非常に普遍的であり、アクション、呪文、スキル、アイテムなどを示すために使用できますn。素敵なボーナス-ベクトル形式。







10. Reiner's Tilesets （無料）



無料のタイルで最高のウェブサイト。 ここでは、動物、植物、建物、武器、視覚効果のスプライト、および等尺性グラフィックスを備えたRPGのほぼすべてのオブジェクトを見つけることができます。 資産スタイルはDiablo IIに似ています。 これは、ゲームエンジンをテストしたり、独自のゲームの作成を開始したい初心者デベロッパーにとって素晴らしいリソースです。
