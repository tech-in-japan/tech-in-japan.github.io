パッチは次のリンクからダウンロードできます。
  • .NET Framework 1.1 Service Pack 1およびWindows Server 2003 Service Pack 2（32ビット）（ リンク ）。
  • Windows Server 2003およびWindows XP上の.NET Framework 3.5（ リンク ）;
  • .NET Framework 3.5、Windows Vista Service Pack 1、およびWindows Server 2008（ リンク ）;
  • Windows XP、Windows Server 2003、Windows Vista、およびWindows Server 2008上の.NET Framework 3.5 Service Pack 1（ リンク ）;
  • Windows Vista Service Pack 1およびWindows Server 2008上の.NET Framework 3.5 Service Pack 1（ リンク ）;
  • .NET Framework 3.5 Service Pack 1、Windows Vista Service Pack 2、およびWindows Server 2008 Service Pack 2（ リンク ）;
  • Windows 7およびWindows Server 2008 R2上の.NET Framework 3.5.1（ リンク ）;
  • Microsoft .NET Framework 4（ リンク ）;
この更新プログラムは、数日以内にWindows UpdateおよびWindows Server Update Servicesの更新チャネルから入手できます。
