IE7 {css2：自動; }

IE7は、エクスプローラーを標準で機能させるJavaScriptライブラリです。 cssに関する多くの問題を排除し、IE5およびIE6で半透明のPNGを適切に処理できるようにします。



IE7は、エクスプローラーが理解できる形式でスタイルシートを読み込んで解析します。 その後、CSSハッキングなしでCSS2 / CSS3関数を使用することが可能になります。



スクリプトの軽量バージョンがHTML / XMLドキュメントに含まれています。 マークアップまたはcssスタイルを変更する必要はありません。



IE7は、W3C CSSおよびHTML標準に対するMicrosoft Internet Explorerのサポートを提供します。



*次のCSSセレクターをサポートします。

+名前空間|セレクター

+親>子

+隣接+兄弟

+隣接〜兄弟

+ [attr]、[attr = "value"]、[attr〜= "value"]など

+ .multiple.classes（バグを修正）

+：ホバー、：アクティブ、：フォーカス（すべての要素）

+：最初の子、：最後の子、唯一の子、n番目の子、n番目の最後の子

+：チェック、：無効、：有効

+：ルート、：空、：を含む（）、:しない（）

+：前//後/内容：

+：lang（）



* HTMLおよびXMLドキュメントで動作します

*インポートされたスタイルをサポート

*スタイルシートの構造を保持

*文書の構造は変更されません

* JavaScriptを使用する場合、DOM構造を再クエリしません

*スタイルシートを正しくレンダリングするために純粋なCSSを使用

*標準モードと個別モードのW3Cボックスモデルをサポート

*固定配置をサポート（オブジェクトはちらつきません）

*オーバーフローをサポート：可視

*最小/最大幅/高さをサポート

*破損（X）HTML要素（abbr、オブジェクト）を修正

*フォームアクション（フォームの動作）を標準化

* PNGアルファチャネルをサポート

*軽量スクリプト（22K）

*モジュール構造（モジュールの追加と削除は無料）

* Microsoft Internet Explorer 5以降で動作します（Windowsのみ）
