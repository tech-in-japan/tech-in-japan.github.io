Canon EOS 5D Mark III。 初見

最近、mclander habermanは Canon 5D Mark IIIの機能について考えていました。 そしてそれが起こった！ すぐにこのカメラの秘密のベールが正式に開きますが、今のところは床下から見ていきます（そして、mclanderに座っている先見者がどれほど良いかを見てみましょう）。











仕様書



  • 22.3メガピクセルのフルフレームCMOSセンサー
  • 新しいDIGIC 5+プロセッサ
  • ISO 100-25 600（50-102 400に拡張可能）
  • 61点オートフォーカス
  • 1秒あたり6フレーム
  • 63ゾーンiFCLメータリングセンサー
  • ISO 100-12 800の範囲のフルHDビデオ録画（1080p）（最大25,600まで拡張可能）
  • HDRモード-カメラで3つのショットからHDRを作成します。 5つの定義済みスタイル（カスタムスタイルについては何も言われていない）
  • カバレッジ100％、倍率0.71倍のインテリジェントビューファインダー（その意味は何でも）
  • 3.2 '' 1,040,000ドットの解像度のディスプレイ
  • メモリカード用の2つのスロット：CompactFlashおよびSD




新しいアクセサリー



また、新しいSpeedlight 600EXとSpeedlight ST-E3-RTトランスミッターについても知られるようになりました。 以下はその仕様です。



スピードライト600EX-RT：

  • 新しいマルチワイヤレスフラッシュシステム
  • 拡大されたフラッシュヘッドは、20〜200 mm（ISO 100で最大60メートル）の広範囲をカバー
  • 18のプリセットモードと3つのユーザーモード


スピードライトトランスミッターST-E3-RT（スピードライトトランスミッターST-E3-RT）

  • 新しい2方向の電波通信により、マスターフラッシュとスレーブフラッシュの同期が改善されます。
  • 伝送距離はすべて360度で30メートルに達します
  • 1つの送信機で最大15個のフラッシュを制御可能
  • E-TTL II、マニュアルおよびストロボモード、および外部測定のサポート


価格



Canon EOS 5D Mark IIIは、2012年3月末に発売される予定です。 推定屠体価格は3,499ドルです。ボディ+ EF24-105mm f / 4L IS USMも4,299ドルの推定価格で入手できます。



Speedlite 600EX-RTおよびSpeedlite Transmitter ST-E3-RTも、それぞれ推定価格630ドルおよび470ドルで2012年3月下旬に発売される予定です。
