JS DevDay。 レポート記録





4月27日、JavaScript DevDayを開催しました。携帯電話、Web、ハイブリッドアプリケーションについて話しました。 3つのレポートのメモを共有します。



加速器チタンを使用しない理由を議論する必要がある場合、最初の方法が役立ちます。 2番目は、サービスワーカーでWebアプリケーションを高速化する方法を説明します。 3番目のレポートを見た後、Angularでハイブリッドアプリケーションを作成する方法を学習します。





DevDayとは何ですか？



DevDay-ノボシビルスクで発信する開発者向けの公開ミーティング。 最後の会議は3つの理由で特別でした。 まず、バックエンドから離れて、このような異なるJavaScriptに突入しました。 ほんの一瞬：2年ぶりに！ 第二に、私たちは満員の家を集め、オーガナイザーのチーム全員が折りたたみ椅子のために一緒に走りました。 第三に、彼らはDevDayゴールデンパスでコミュニティの最もアクティブなメンバーに配りました-登録と選択手順なしですべての2GISテクノイベントへのパス。







ppceleratorTitanium VS React Native // Pavel Akulov
