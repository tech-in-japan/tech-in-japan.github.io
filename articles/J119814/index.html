覗き見からDropBox（およびその他のネットワークフォルダー）を保護する

最近、このトピックに関する多くの記事があり、だれかがネットワークドライブ上のユーザーデータを見ることができます。 TrueCryptは、 Windowsでのこの問題に対する優れたソリューションですが、残念ながら、すべてのファイルをコンテナに保存します。つまり、DropBox Webインターフェイスから1つのファイルのみをダウンロードすることは不可能です。



あまり好きではないので、フォルダー内のローカルファイル（DropBoxなど）への変更を監視するプログラムを作成することにしましたが、さらにファイルをアーカイブに自動的に圧縮する方法を知っています。 したがって、私たちは1石で3羽の鳥を殺します。ネットワークドライブ上のファイルのサイズを小さくし、パスワードで暗号化し、暗号化されたアーカイブとしてWebインターフェースで1つずつ安全にダウンロードできるようにします。 カットの下のプログラムについての詳細（+1写真28KiB）。



このプログラムは無料で配布されており、インストールは不要です。起動フォルダーに配置し、実行して構成するだけです。







  1. ソースフォルダー -ローカルファイルが保存されているフォルダー。暗号化せずにリモートサーバーに保存したくないフォルダー

    （写真「D：\ Dropbox.Secure」）
  2. アーカイブフォルダー -ローカルファイルのアーカイブが配置されるフォルダー

    （画像内（「D：\ Dropbox \ Archive」）
  3. 実行可能ファイル-7z.exeファイルへのパス（写真「C：\ Program files \ 7-Zip \ 7z.exe」）
  4. パスワードを使用 -このチェックボックスがオンの場合、アーカイブはパスワードで暗号化されます。パスワードはチェックボックスの右側のフィールドに入力する必要があります
  5. 保存/保存 -現在のパスワードをWindowsレジストリに保存できるボタン。 パスワードはAESによって事前に暗号化されていますが、プログラムとレジストリに同時にアクセスできる場合、誰かがそれを解読できないことを期待すべきではないと思います
  6. クリア -ボタンは、レジストリ内のパスワードに関する情報を破棄します。 最初に空の文字列がキーに書き込まれ、次に削除されます。
  7. パラメーターの追加 -ファイル圧縮のための追加の7-Zipコマンドラインオプション（図では-mhe = on（ヘッダーの暗号化）および-mx = 9（最大圧縮））
  8. パラメータの抽出 -アーカイブを解凍する際の追加の7-Zipコマンドラインオプション。


そのため、プログラムを事前に構成した後、[同期]ボタンをクリックできます。



これで、ファイルをD：\ Dropbox.Secureフォルダーにドロップすると、それらは自動的に圧縮され、D：\ Dropbox \ Archiveフォルダーに保存されます。 D：\ Dropbox.Secureフォルダーを変更すると、ファイルが自動的に圧縮されます。 ファイルがロックされている場合、プログラムはロックが解除されるまで待機します。



フォルダD：\ Dropbox.Secureのファイルまたはディレクトリを削除すると、フォルダD：\ Dropbox \ Archiveのファイルとディレクトリは自動的に削除されます。



フォルダD：\ Dropbox \ Archiveの7-Zipアーカイブを削除すると、そこだけが削除されます。



7-Zipアーカイブを任意のフォルダーからD：\ Dropbox.Secureフォルダーにドロップすると、ファイルは自動的に解凍され、D：\ Dropbox \ Archiveフォルダー内のアーカイブが更新されます。 これは、午前中に家から到着し、ローカルコンピューターのアーカイブ内のファイルを更新する必要がある場合に便利です。DropBoxの同期プロセスが完了した後にファイルをコピーするだけです。 そうすれば、Dropboxアーカイブ内のすべてのファイルが自動的に同期されるので、1日中静かに作業できます。



プログラムがアーカイブを解凍できない場合、これを報告します。



このページからプログラムをダウンロードできます 。



プログラムの機能はWindows 7でテストされました。Microsoft.NET Frameworkを更新して実行する必要がある場合があります。
