Python開発者向けのVisual Studio 2015の内容





みなさんこんにちは！



Visual Studio 2015 Communityのバージョンでは、Microsoftは開発者に強力な開発ツールの完全に機能するバージョンを無料で使用することを最初に提供しました。 Visual Studio 2015 Communityの機能は、Professionalバージョンの機能をほぼ完全に提供しますが、商用開発で​​の使用にはいくつかのライセンス制限があります。



Visual Studio 2015コミュニティは、他のツールバージョンと同様に、Visual Studio Marketplaceとすべての公式ツールの更新により拡張可能です。 つまり、開発者は無料で無料で利用できる膨大な量の機能を利用できます。 これは、Python言語の開発ツールにも適用されます 。



環境のインストール時にこのボックスをチェックすると、Pythonサポートが利用可能になります。 これは、CPython、PyPy、IronPython、IntelliSenseサポート、対話型デバッグ、統合、およびPythonの他のVisual Studio機能にも適用されます。



便利な編集

ツールチップ、スニペット、自動置換







必要なライブラリ

プロジェクトと依存関係を管理するためのpip、PyPI、および仮想環境のサポート。







デバッグ

コードの問題をインタラクティブに診断できます。 Visual Studioは、オペレーティングシステムに関係なく、コードのステップ実行、状態の表示と変更、プログラムとの対話を支援します。







双方向性

おなじみのREPLは、デバッグされたプロセスのデータに対して機能するため、複雑な式をその場で評価し、クラスと関数を再定義できます。 インタラクティブウィンドウには、結果、エラー、グラフ、画像がすぐに表示されます。 IPythonとの統合により、開発者はデータ処理用の強力なプラットフォームを提供できます。







コード管理

チームエクスプローラーを使用すると、Git / TFSプロジェクト、保留中の変更およびブランチを管理できます。 Visual Studioで利用できる他の多くのコード管理拡張機能を試す機会もあります。







単体テスト

コードの正しい操作は、Visual Studioで直接確認できます。 テストウィンドウから単体テストを表示、編集、実行、およびデバッグできます。







おわりに



Python Toolsは、コミュニティの参加を得てMicrosoftが開発およびサポートする完全に無料の拡張機能です。 GitHubのページにアクセスして、GitHubの開発を見たり参加したりできます。
