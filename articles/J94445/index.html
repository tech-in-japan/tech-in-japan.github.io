ThinkPad T410：強力で信頼性の高いビジネスノートブック

14インチのThinkPad T400は、誰にとっても良いものでした。仕事に最適な強力で破壊不可能なマシンです。 Lenovoは、新しいモデル、ThinkPad T410を開発し、すべての基本的な特性を改善しようとしました。 Intel Core iシリーズプロセッサを搭載した強力なハードウェアスタッフィングに加えて、更新されたラップトップはタッチパッド領域を拡大し、MultiTouchモードをサポートし、Gobi 2000 3Gモジュールのサポートを追加し、バッテリー寿命を改善しました。 Tシリーズ専用に、新しい拡張バッテリーが開発され、コンセントから最大22時間動作します！



この記事では、更新されたThinkPad T410について詳しく説明します。 ただし、Lenovo コンテストに参加することで、あなた自身が所有者になることができます。 条件は簡単です。コンテストページにアクセスして、新しいThinkPadをどのように使用するか、どのように役立つかを伝える必要があります。 コンテストは6月末まで続き、結果は7月に発表され、最も独創的なストーリーの著者が勝者になります。 がんばれ！











性能



ThinkPad T410は、Intel Core i3、i5、およびi7プロセッサーを搭載したバージョンで利用できます。 強力なCore i7プロセッサーを搭載したバージョンは最大のパフォーマンスを提供し、Core i5を搭載したモデルはパフォーマンスとバッテリー寿命の最適な比率を提供します。 これらのプロセッサは、Intel Turbo Boostテクノロジーをサポートしています。たとえば、Intel Core i5-520Mを搭載したラップトップの公称周波数は2.4 GHzですが、2.93 GHzに自動的に「加速」できます。 「加速」は動作中に積極的に使用されますが、この機能はスタンドアロン電源に切り替えるときに無効にすることができます。







Intel Core i3はTurbo Boostテクノロジーをサポートしていませんが、非常に強力なプロセッサーであり、すべてのテストで前世代のCore 2 Duoを打ち負かしています。 最小電力モードでは、Intel Core i3-330Mプロセッサの周波数は、標準2.13 GHzで1.1-1.3ギガヘルツの範囲です。 実際には、これにより、オフィス文書だけでなく、ビデオを視聴したり、ラップトップにかなり難しいタスクをロードしたりすることができます。



ユニバーサルコネクタセット



ThinkPad T410には、外部デバイスを接続するための最大のポートセットが装備されています。 外部ディスプレイを接続するためのDisplayPortインターフェイスは、Tシリーズモデル（および他のいくつかのThinkPad）の標準となっています。 必要に応じて、モニターをアダプターを使用してDVIまたはHDMIケーブルを使用して接続することもできます。 ESataおよびFireWireミニコネクタは、通常の4つのUSBポートに追加されます（ラップトップの電源がオフの場合でも、そのうちの1つは常に電源が入っています）。 オプションのモジュールを34 mm Express Cardスロットに取り付けることもできます。









信頼性



ThinkPad T410のディスプレイ蓋には、高弾性ポリカーボネート（HEPC）フレームが装備されていますが、マザーボードとハードドライブはマグネシウム合金フレームで保護されています。 キーボードは湿気から保護されています。キーにこぼれた液体は、電子部品に影響を与えることなく、特別なチャネルを介してラップトップから排出されます。 そしてもちろん、ThinkPad T410には伝説的なコーポレートキーボードが装備されており、驚くほどのタイピングの快適さを保証します。







バッテリー寿命



T410は、長いバッテリー寿命に最適なラップトップの1つです。 4セルと6セルの従来のバッテリーを搭載したラップトップを購入できます。 6セルのバッテリーで、Intel Core i5プロセッサーを搭載したラップトップは約6.6時間持続します。 個別に購入できる拡張9セルバッテリーは、1回の充電で11時間の動作を提供します。 しかし、それだけではありません。新しいThinkPad Tシリーズのラップトップ用に、角柱セルを備えた特別なバッテリーが開発されました。







このようなバッテリーは、ラップトップの底面にあるドックコネクターに接続され、標準バッテリーと組み合わせて使用​​できます。 その結果、2つの9セルバッテリーを同時に使用すると、ラップトップは記録的な22時間（ほぼ1日）充電することなく動作します。 重要なことは、両方のバッテリーが同時に充電されることです。バッテリーを並べ替えたり、外部充電器を購入する必要はありません。



最後に、ThinkPad T410は、他のTシリーズラップトップと同様に、完全に自由に選択できます。 モジュール設計により、ラップトップの機能を拡張できます。 たとえば、内蔵DVD-ROMドライブを、特別なアダプターに取り付けられた別の2.5インチハードドライブに交換できます。 ディスカッションで繰り返し言及されているCtrlの左側のFnキーの企業の場所は、BIOSで変更できます。







ラップトップを専用のドッキングステーションに接続して、オフィスと旅行の両方に使用できる唯一のコンピューターにすることができます。 ポートレプリケーター自体では、外部モニター、キーボード、マウスなど、デスクトップPCのすべての属性を永続的に接続できます。 さらに、ラップトップは更新されたドッキングステーションにほぼ瞬時に接続および切断できます。以前のモデルでは、ポートが切断されるまで待つ必要がありました。 また、強力なIntel Core i5により、パフォーマンスは「デスクトップ」パフォーマンスに近くなります。これは、オフィスだけでなく、はるかに「食いしん坊」的なプログラムでもあります。







結論として、ThinkPad T410の近親者であるT410sモデルについてお話したいと思います。 これは同等に強力なラップトップですが、はるかに薄く軽量です。モデルの設計にマグネシウム合金フレームとガラス繊維強化プラスチックを使用すると、モデルの重量を2.27 kg（T410）から1.79 kg（T410s）に減らすことができます。 このモデルの珍しい機能の1つは、MultiTouchをサポートするオプションのタッチスクリーンです。 両方のモデルは近いうちにテストされます。詳細については、Lenovoブログで説明します。
