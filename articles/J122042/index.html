Evernote for Windows Phone 7の紹介

初日から、Evernoteのコンセプトは、Webクライアントに加えて、さまざまなコンピュータープラットフォームおよびモバイルデバイス用のネイティブクライアントの作成を暗示していました。 過去3年間で、ほぼすべての主要プラットフォーム向けに13を超えるバージョンのEvernoteを開発してきました。 そして今日、このコレクションには新しいアプリケーションが追加されています。 Evernote for Windows Phone 7を紹介します 。







Evernote for Windows Phone 7を使用すると、いつでもどこでもアイデア、考え、メモを保存できます。 もちろん、アプリケーションは、使用しているEvernoteの他のバージョンと問題なく同期しますが、その利点を使い果たすことはありません。 実際、この新しいプラットフォームでのデビューは、Evernoteの歴史全体で最も多機能であることが判明しました。 また、機能的であるだけでなく、Windows Phone 7 Metroの革新的なインターフェイスのおかげで、見た目もとてもきれいです。



通常どおり、Evernote for Windows Phone 7は無料で、App Hubで既にダウンロードできます。



ただし、インターフェイスの英語のみを提供できますが、ロシア語のローカライズのサポートがWindows Phone 7に表示され次第、ロシア語版のアプリケーションを提供します。



インターフェースの概要



このバージョンのEvernoteは、Windows Phone 7向けにゼロから開発されました。1回の認証と最初の短い同期（サービスの既存ユーザー向け）の後、まったく新しいWindows Phone 7インターフェイスに慣れることができます。



ホームパノラマスクリーン

Evernote Panoramaパネル：

  • 注：すべてのスクロールノートのリスト。
  • ノートブック：ここでは、メモ帳を選択してその内容を表示できます。
  • タグ ： タグごとにメモを表示します。
  • 最近のメモ ：このリストを使用すると、最近開いたメモのリストにすばやくジャンプできます。


アプリケーションパネル

Evernote for Windows Phone 7のすべての画面にはアプリケーションバーがあり、そのメニューから現在開いている画面に応じてさまざまな追加機能にアクセスできます。 また、その助けを借りて、高度な機能にアクセスできます。 パネルの右側に省略記号アイコンが表示されている場合は、クリックして現在使用可能なすべての機能を表示します。



新しいメモを作成する



Evernoteに新しいエントリを追加するのは非常に簡単です。 ホーム画面のアプリケーションパネルには、新しいメモ（プラス記号）と写真メモ（「カメラ」）を作成するためのボタンがあります。 仕組みは次のとおりです。



新しいメモ

新しいノートを作成するためのボタンをクリックすると、空白のノートを含む画面が開きます。 ここで、タイトルを設定し、鉛筆アイコンをクリックしてノートブックとタグを指定し、メインテキストボックスにメモ自体を書き込むことができます。 完了したら、フロッピーディスクのアイコンをクリックします。







アプリケーションパネルのカメラアイコンをクリックして写真を添付するか、カセットアイコンをクリックして音声メモを録音することもできます。 複数のオーディオ録音と画像を1つのメモに添付できます。



新しい写真メモ

このアクションにより、カメラ画面に直接移動し、写真を撮ってメモに追加できます。 次に、必要に応じて、このメモにテキストの説明を提供できます。



メモを検索する



Evernote for Windows Phone 7を使用すると、携帯電話で作成したかコンピューターで作成したかに関係なく、必要なメモをすばやく簡単に見つけることができます。 必要なメモにアクセスするには、いくつかの方法があります。 メモのリストでそれらを見つけるか、検索クエリを実行するか、地理マップでメモを見つけることができます。



ノートナビゲーション

すべてのインターフェイスパネルは、メモを介した最も単純なナビゲーションに焦点を合わせています。 メモのリスト全体をスクロールしたり、ノートブックやタグから目的のエントリを見つけたりできます。 アプリケーションのパノラマインターフェイスで目的のパネルを選択するだけです。



検索する

Evernoteには、ノート内の情報を見つけるのに役立つ優れた検索機能がいくつかあります。 （電話自体ではなく）アプリケーションの虫眼鏡アイコンをクリックして、検索インターフェイスに移動します。 ここには、検索バーと2つのパネルがあります。

  • 前 ：このパネルには、最近行った検索のリストが表示されます。
  • 保存済み ：ここでは、以前にデスクトップクライアントまたはEvernote Webインターフェイスに保存した検索クエリを表示できます。






Evernoteはテキストだけでなく、写真や手書きのメモの単語でも検索できます。 この機能を使用すると、名刺、広告、道路標識などの写真を簡単に見つけることができます。



地図メモ

Evernote for Windows Phone 7は、メモを作成するときにアプリケーションがユーザーの場所を決定するように構成できます。 アプリケーションが座標を保存できるようにするには、[設定]メニューの[場所]パネルに移動し、適切なオプションを有効にします。 地理参照されたメモは、マップで表示できます。 これはあなたの記憶に良い助けになるでしょう。 マップ上のメモを表示するには、アプリケーションパネルの省略記号をクリックしてから、[マップ]アイテムをクリックします。 マップを拡大して、表示領域に作成されたメモを表示します。 チェックボックスをクリックすると、メモのリストが表示されます。







共有する





Evernoteは、あなたにとって重要なことを思い出すのに役立ちます。 それらのいくつかは友人と共有する価値があります。 Evernote for Windows Phone 7を使用すると、お気に入りのメモをFacebookの友人と簡単に共有できます。 これを行うには、メモを開き、アプリケーションパネルのFacebookアイコンをクリックします（この機能を初めて使用するときは、Facebookにログインする必要もあります）。 その後、現在のメモはソーシャルネットワーク上のすべての友達に表示されます。



特定の人とのみメモを共有する必要がある場合は、封筒アイコンをクリックして、メールでメモを送信できます。



プレミアム機能





Evernoteのプレミアムサブスクライバーである場合は、インターネットに接続していない場合でもアクセスできるように、すべてのノートブックをアプリケーションにダウンロードできます。 これを行うには、アプリケーションパネルで同じ省略記号アイコンをクリックし、メニューから[設定]項目を選択します。 次に、[オフラインノートブック]パネルに移動して、オフラインアクセスが必要なノートブックを確認します。 プレミアムサブスクリプションがない場合は、[アカウント情報]ページで接続できます。



その他



これらはWindows Phone 7のEvernoteの機能の一部に過ぎず、さらに多くの機能があります。 新しいクライアントの印象についてお聞かせいただければ幸いです。 そしてもちろん、ニュースをフォローすることを忘れないでください！



アプリケーションをインストールするには、Windows Phone 7電話のMarketplaceアプリケーションで検索用語「Evernote」を入力します。



インストール、および最初のログインとWi-Fi接続との同期を完了することをお勧めします。
