AWS：Amazon Linux 2012.09

こんにちは

最近では、 Amazon Linuxディストリビューションの更新バージョン-2012.09。



更新の主な機能：

  • カーネル3.2.30
  • ApacheとPHPのいくつかのバージョンとそれらの組み合わせのサポートを実装しました。 Apache 2.2 + php 5.3は、通常のyum install httpd phpコマンドを使用してインストールされますApache 2.4 + php 5.4- yum install httpd24 php54 。
  • プログラミング言語R 2.15がリポジトリで利用可能になりました。
  • OpenJDK 7は、 yum install java-1.7.0-openjdkをインストールできます。
  • 追加のネットワークインターフェイスのサポート。
  • GCCの異なるバージョンのサポート。

  • Enterprise Linuxは無料です。
  • すべてのAWS機能のサポート。
  • 安定した作業と更新。
