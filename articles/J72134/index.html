Flash / FlexサイトのYouTubeビデオ

最近、Flex-eの予算を大幅に削減したWebサイトを作成しました。 FXVideo CompとSWFLoaderに悩まされていたビデオコンテンツの配置に問題があり（ホスティングに$を保存し、2 gのビデオをホストします...）、Googleで検索しました。



解決策が見つかりました：

TubeLoc-（YouTube LocalConnection） swfアプリケーションとYouTubeビデオプレーヤー（AS2で記述されている）の間のブリッジ。

このコンポーネントが気に入ったので、約10分で理解できました純粋なAS3プロジェクトとFlexの両方の例があります。 :)

チップのうち-通常のYouTubeプレーヤーを埋め込み、クロムレス。







エロティックなコンテンツや、最後にプレイヤーが発行したマーリン・マンソンのコンサートのカットがなければ、すべてがうまくいきます。 その理由は、title01.flvタイプのファイルの元の名前です。 タグは「子供の歌、子供の音楽、子供など」として割り当てられましたが。

もちろん、すべてを変えようとしましたが、今のところあまり役に立ちません。



ソリューションはYouTubeブログで見つかりました。



埋め込みプレーヤーで関連動画を表示するオプションをオフにする場合は、埋め込みコードの動画ID番号の後に「＆rel = 0」を追加します。



MovieSprite.asの行215のTubeLocの場合



concatenatedUrl = playerWrapperUrl + [...] + escape(apiUrl);







に



concatenatedUrl = playerWrapperUrl + [...] + escape(apiUrl + "&rel=0");







そしてもちろん、あなたはこのリストから追加することができます（それは非名前で見つかりました、悲しいかな、著者を見つけることができませんでした、いずれにしても、彼に対する多くの敬意:)）



1. YouTube動画を高品質で見る方法



YouTubeには膨大な数の動画があり、プレーヤーをフルスクリーンに引き伸ばすのは問題ないほどの品質です。最高品質の動画があるかどうかを確認するには、アドレスバーに「＆fmt = 18」を追加します（ステレオ、480 x 270解像度）、 '＆fmt = 6′（モノラル、480 x 270解像度）または '＆fmt = 22′（ステレオ、1280 x 720解像度）。

18：ビットレート512、24 fps、h.264ビデオコーデック、AACオーディオコーデック

6：ビットレート900、30 fps、Flash Sorensorビデオコーデック、MP3オーディオコーデック



2. YouTubeビデオをWebサイトに良質で埋め込む方法



そのため、高品質で動画を視聴する方法を考え出しましたが、訪問者に見せることは罪ではありません。このため、埋め込みコードに「＆ap =％2526fmt％3D18」または「＆ap =％2526fmt％3D22」を追加します。



3.特定の分（秒）のYouTubeビデオへのリンクを挿入する方法



コマーシャルでは、せん妄の5分後にラッシュが始まり、あらゆる種類のナンセンスに時間を費やすことがあります。 したがって、友人の時間を節約するために、リンク「＃t = 03m22s」にm-分、s-秒を追加します。



4. YouTubeで検索ボックスを非表示にする方法



いまいましい検索ボックスは不便であり、原則として不要です-なぜそれが必要なのでしょうか、それも削除しましょう-'＆showsearch = 0' '



5. YouTubeビデオの一部のみを挿入する方法



必要な場合もあるため、リンクを「＆start = 30」に置き換えるだけで、動画は30秒から始まります。



6. YouTubeから埋め込み動画を自動的に再生する方法



ユーザーを指導し、起動と再起動のたびにビデオを常に視聴させる-「＆autoplay = 1」を使用して



7. YouTubeからのビデオを自動的に繰り返す



そして、ここではすべてがシンプルで、クリップにアドレスを追加する '＆loop = 1'が役立ちます



8.関連動画を削除する



繰り返しますが、すべてが単純です '＆rel = 0' '



9.地域による制限を無効にします（現在は機能しません-a92）



この禁止を無効にするために、ビデオアドレスをwww.youtube.com/watch?v = <some code>からwww.youtube.com/v <some code>に変更します



10. YouTubeからビデオをダウンロードする方法は？



はい、youtube.comをkickyoutube.comに変更し、健康状態にダウンロードするだけです
