100個のオープンソースウェブマスターツール。

昨日DesignVitalityで公開されたこの記事では、著者はオープンソースデザイナープログラム、オープンソーステンプレート、リソースなど、100のウェブマスターツールを引用しています。





オープンWebサイトテンプレートを備えたトップポータル



オンラインには無料のテンプレートが多数ありますが、適切なテンプレートを見つけるまでに時間がかかることが多いため、最初に次の4つのポータルにアクセスすることをお勧めします。

  1. OSWD ：2,000以上の無料のWebサイトテンプレート。 色、方向、構造によってテンプレートを選択する機能。

  2. OpenWebDesign ：テンプレートを無料で投稿するデザイナーの大規模なコミュニティ。 多くの場合、内部競技会が開催されました。 便利な検索システム。

  3. オープンデザイン ：ポータルは非営利組織が所有しています。 コミュニティが選択した600以上のテンプレートが含まれています。 また、デザイナーが経験やヒントを共有するアクティブなフォーラムもあります。

  4. CSS Tinderbox ：CSS Tinderboxは、このポータルから取得できる単純なCSS / XHTMLテンプレートが必要な場合は、「車輪の再発明」に反対します。



オープンデザインプログラム



無料のアナログがあるのに、なぜプログラムにお金を払うのか。 多くの人は、フリーソフトウェアは高品質ではないと考えていますが、そうではありません。 以下は、設計者にとって最適なオープンソースプログラムです。

  1. Nvu ：ページレイアウトソフトウェア-FrontPageおよびDreamweaverの競合製品。 （Linux、Windows、およびMac）
  2. Cssed ：自動補完、構文の強調表示、検証を備えたCSSエディター。 （C、C ++）（Linux）
  3. Quanta Plus ：KDE用の優れたHTMLエディター。
  4. Bluefish ：プログラマーおよびウェブマスター向けの開発環境。 ©（GNU / Linux、Unix）
  5. GIMPShop ：Photoshopの優れた代替品。 （Mac、Linux、およびWindows）
  6. Inkscape ：Adobe Illustratorと機能が似ているオープンなベクターエディター。

CSSスタイルのツールとジェネレーター。



あまり強力なエンコーダーではない場合、または標準スタイルのシンプルなソリューションを探している場合は、次のツールを参照してください。 これらのリソースにより、数秒で必要なCSSを取得できます。

  1. CSS Creator ：色を選択し、列数、HTMLバージョンを設定して、ボタンをクリックします。 ページのCSSテンプレートを取得します。
  2. Inknoise ：スタイルの設定を選択し、ページをコンピューターにダウンロードするだけです。
  3. 奇妙なバナナ ：興味深い解決策。 必要なものが得られるまで、生成されたページを更新するだけです。
  4. Nidahas ：CSSを使用して作成されたフォームの例。
  5. Maketemplate ：PHP、HTML、CSSを使用してサイトテンプレートを生成します。各ブロックは、phpを使用して個別のファイルからロードされます。
  6. IBDjohn ：3列のWebサイト用のCSSマークアップジェネレーター。
  7. CSS Creator ：固定または動的な列幅と数を選択できるシンプルなCSSマークアップジェネレーター。 *パラグラフ79の繰り返し。著者の誤り。
  8. PsycHo ：「あなたのデザインはあなたのキャラクターです。」 テンプレートを作成するための便利なツール。
  9. Firdamatic ：2列または3列のテーブルレスレイアウトのジェネレーター。
  10. WordPressテーマジェネレーター ：オンラインWordPressテーマジェネレーター 。
