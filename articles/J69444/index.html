他の誰かのコンソールをスパイする

FreeBSDには、他の誰かのコンソールをスパイできるすばらしいwatch（8）プログラムが付属しています。



彼女がこれに必要とするのは、工場から組み立てるのに十分なsnpモジュールであり、プログラム自体がそれをロードします。

# cd /usr/src/sys/modules/snp/

# make && make install







次に、ターゲットを選択します。

# w

9:16PM up 55 days, 52 mins, 2 users, load averages: 0.08, 0.06, 0.00

USER TTY FROM LOGIN@ IDLE WHAT

nlo1 p0 nlo.habr.ru 9:06PM - w

nlo2 p1 nlo.habr.ru 9:16PM - -bash (bash)







そして、私たちにとって興味深いコンソール（TTY）をスパイします。

# watch p1







問題は熟成です。Linuxで他の誰かのコンソールをスパイするにはどうすればよいですか。

PS

質問はホリバーのためではありませんが、私はこれをどのように行うのか本当に興味があります。
