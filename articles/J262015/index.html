周期的な離散信号を考慮してください。

a）信号が離散的である場合、特定のポイントでのみ決定されます：tn = n *Δt、ここでnは整数、Δtはサンプリングステップです。

b）信号が周期的である場合、その値はすべて一定期間後に繰り返されます：s（t）= s（t + T）、ここでTは信号の周期です。

c）信号が離散的で周期的である場合、そのすべての値はサンプリングステップs（tn）= s（tn + Td）の倍数の期間後に繰り返されます。ここで、Tdは整数Δtの合計である信号周期です。 これは、小数部Δtからなる周期を持つ信号は、存在しない時間に落ちるため、周期後にその値を繰り返さないという事実によって説明されます。 可能な周波数にはまだ制限があることがわかります。



たとえば、1.5 Hzの周波数の高調波信号が10 Hzのサンプリング周波数でデジタル化される場合、0.5 Hzの周波数の周期信号が取得されます。



したがって、時間離散信号は、サンプリングステップΔtの倍数の周期でのみ周期的であり、そのような信号のスペクトルは周期Tn = n *Δtのみで構成されます（nは整数）。 デルタ関数の場合、同じ振幅の周期にわたって均一なステップを持つ高調波のセットを取得します。



ここで、時間制限された定期的に拡張された時間離散デルタ関数のスペクトルを定義します。 このようなスペクトルは、周波数fnのf0の倍数と周期TnのΔtの倍数のみで構成できます。 周波数fnのいずれかが1 / Tnのいずれとも一致しない場合、これは周波数fnではなく、セット1 / Tnのその他のものです。 したがって、周波数fnがセットTnのどの周期とも一致しない場合、そのような周波数の信号は特定のサンプリングステップで実現できないため、セットfnから削除する必要があります。 周期に関連付けることができないすべての周波数を周波数グリッドから削除すると、周波数が不均一で周期の不均一なステップに等しい新しいステップが得られます。 このような不均一なステップは、周波数の対数に沿った均一なステップです。 周波数対数が均一なスペクトルを持つ信号は、フリッカーノイズです。 サンプリングステップがゼロになる傾向があり、デルタ関数の繰り返し周期が無限になると、フリッカーノイズのスペクトルと一致するデルタ関数の連続スペクトルが得られます。
