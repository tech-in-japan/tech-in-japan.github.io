NTT Com、世界初の完全ソフトウェア定義ネットワークを発表

通信サービスプロバイダーであるNTTコミュニケーションズは、190カ国以上をカバーする世界最大のソフトウェア定義ネットワーク（SDN）を開始すると発表しました。 会社の代表者によると、SD-WANサービスポートフォリオは「世界初の完全所有SDN」であり、さまざまな規模の企業に高性能SD-WANサービスを提供します。



/ Flickr / dennis van zuijlekom / cc



SD-WANの本質は、物理接続を介した安全なオーバーレイ接続で構成される単一の仮想ネットワークファクトリを作成することです。物理接続は、専用チャネル、低コストのインターネットチャネル、またはLTEでも可能です。 WAN接続のパフォーマンスを監視することにより、システムはトラフィックをルーティングし、チャネルの使用を最適化します。 SD-WANの主な利点は、アプリケーションレベルでトラフィックを管理できることです。 SD-WANを使用して、複数の国の会社の支店（またはデータセンター）を中央の企業ネットワークに接続できます。



NTT Comは、75を超えるクラウドセンターをカバーするローカル分散アーキテクチャに基づいてSD-WANプラットフォームを構築したと主張しています。 このアーキテクチャは、ネットワークおよびセキュリティサービス用に最適化されています。 同時に、2014年にVirtela SDNのスタートアップで買収したテクノロジーを使用しています。



「今日、あらゆる企業でデジタル変容が起きており、根本的にビジネスモデルが変化しています」 と 、NTT Comの上級副社長であるSa倉修一は述べています。 -グローバルサービスであるNTT SD-WANサービスポートフォリオは、特定の要件を完全に満たす柔軟でスケーラブルなネットワークへのアクセスを組織に提供するエンドツーエンドのソリューションです。 これにより、企業はデジタル経済において競争力を高めることができます。」



新しいプラットフォームは、ネットワークの状態に関する分析情報をリアルタイムで送信できます。 これにより、管理およびITオペレーターは、セキュリティメカニズムの状態、パフォーマンス、およびユーザーエクスペリエンスに影響するかどうかを理解できます。







NTT SD-WANサービスポートフォリオ（ ソース ）



SD-WANサービスポートフォリオのもう1つの機能は、接続性の向上です。 このプラットフォームは、ローカルISP、インターネット、MPLS、LTE、ワイヤレス接続など、すべてのタイプをサポートしています。 ユーザーは、企業の要件に応じてそれらを結合する機会を得ます。 他の有用な利点には、安全なインターネットゲートウェイ、URLフィルタリング、および侵入防止サービスが含まれます。



WikibonコンサルタントコミュニティのチーフアナリストであるStu Minimanは、通信会社が長年クラウドで足場を築こうとしているが、成功しなかったと指摘しました。 彼の意見では、SD-WANを使用したサービス提供がはるかに適しています。



「すべてのSDNソリューションの目標は、ネットワークデバイスで「固定」されたサービスを利用し、ソフトウェアを使用して手頃な価格で柔軟かつスケーラブルにすることです」 と Stu 氏は言います。 -SD-WANの方向性は近年活発に発展しており、これはまさに、NTTなどの電気通信プロバイダーが提供できるソリューションです。 これにより、新しい顧客を引き付け、すでに利用可能なサービスの品質を向上させることができます。」



NTT Comについて


NTT Comは、日本最大の電気通信会社である日本電信電話グループ（NTTグループ）の国際支社です。 同社はコンサルティングサービスを提供し、クラウドサービスの設計、情報セキュリティ、実装にも携わっています。 ロシアでは、NTT Comロシアはユニバーサルテレコムソリューションを提供することにより、グローバル企業のビジネスの拡大をサポートしています。
