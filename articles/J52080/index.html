ARM Cortexプロセッサは、NetBook市場に参入しようとしています！

Intel Atomは、NVidiaおよびVIA（ 「NvidiaがIntel互換プロセッサの作成を引き受けた！」を参照）だけでなく、Cortex-A8およびCortex-A9 MPCoreプロセッサを備えたARMと競合することを望んでいることがわかりました。 ARMはMWC 2009でIntelとの戦いの準備ができています»

[ モバイルワールドコングレスは、2009年2月16〜19日にスペインのバルセロナで開催されます]



確かに、ARMにはIntel Atomプロセッサで使用されるIntel x86アーキテクチャと互換性のない独自のRISCアーキテクチャがありますが、このアーキテクチャはスマートフォン市場で最も人気があり、Apple iPhone 3GやApple iPod Touchなどの人気デバイスでも使用されています。

ARMは、プロセッサがNetBook市場に成長したと判断したようです。

そして、彼らはある程度正しいです-あなたがMS Windows XPではなく、たとえばLinuxの種類の1つであるNetBookを装着した場合：

1. NokiaがNokia N810タブレット（ARMで作成）に搭載するもの-OS Internet Tablet OS 2008-別名Maemo Linux 4.0 。

2.または、元々ARMプロセッサのスマートフォン向けに開発されたが、Intel x86プラットフォームのNetBookでも動作するGoogle Androidをインストールします 。 「IntelはAndroidネットブックのリリースを準備していますか？」

3.まあ、またはAcer、MSI、Asusなどのメーカーの最も安価なNetBookとプレインストールLinuxが付属するLinuxのバージョンの1つ（ARMプロセッサ用にコンパイルする必要があります）、 Xandros Linuxをインストールします。 Asus Eee PC 701（パート2）：Xandros Linux” ;

4.まあ、または最もありそうにない-突然のAppleはすべて、ARMプロセッサでNetBookを作成し、Apple iPhone 3Gスマートフォン（ARMプロセッサを含む）ですでに非常に人気を博している、新しいApple OS X iPhoneをインストールしたいと考えています。



その結果、ARMプロセッサでは、最悪ではない可能性があります。遅くなく、熱くなく、かさばらないという意味で、同時に、NetBookの1回の充電で長時間再生されます:)
