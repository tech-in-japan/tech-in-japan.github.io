GitHub Reflog v1.4.12

GitHub Reflogの第4版へようこそ。これは、すばらしいGitHubリポジトリとコミュニティアクティビティの週刊誌です。 以前のエディションはReflogアーカイブから入手できます 。





今週の推奨リポジトリ：



アペンワール/シャトル



この優れたプロジェクトは今週、かなりの数の報道関係者からの推薦を受け、すでに私のツールキットの不可欠な部分となっています。 Sshuttleは自身を「Poor VPN」と呼んでいます。 非特権ユーザーとしてリモートコンピューターに接続し、そのネットワーク上のすべてのポートをローカルコンピューターに転送できます。 スイッチを入れると、すべてのトラフィックはサーバーを介してリダイレクトされます。 サーバーがマシンにアクセスできないマシンのネットワークアクセスを持っている場合、ローカルネットワーク上にあるかのようにアクセスできるようになりました。



OpenSSHによるポート転送は新しいものではありませんが、sshuttleは指定したポートだけでなく、すべてのポートを転送します。 サーバーでは、OpenSSHに依存します。



行こう！



今週のすばらしいリポジトリ



アサラザン/ Narwhalingus-EP



@Asarazanは今週 、すばらしいGitHubアプリケーションを見つけました。 彼と彼のバンド、ブリストル7'sは、クリエイティブコモンズライセンスの下で彼らのEPを公開することを決めました。 オープンソースの精神で、GitHubで公開することが決定されました！



リポジトリには、最終ミックスと、リミックス用の個別のオーディオトラックが含まれます。 プールのリクエストは大歓迎です！



素晴らしいリポジトリ



  • jonromero / music-as-data ：このClojureプロジェクトは、音楽シーケンスを作成するためのプログラミング環境を提供します。 データシーケンスにアクセスし、それらにライブ変換を適用できます。 詳細については、プロジェクトページをご覧ください。
  • 37signals / pow ：37 Signalsのこの新しいプロジェクトは、アプリケーション用の構成不要の開発サーバーです。 その非常にシンプルなインストーラーを実行したら、〜/ powに移動し、アプリケーションディレクトリへのシンボリックリンクを作成します。 そしてそれだけです！ アプリケーションはappname.devでローカルに利用可能になりました 。
  • DanielWaterworth / raphters ：これは新しいC言語Webアプリケーション開発フレームワークです。 あなたがCファンで、お気に入りの言語で最新のWebアプリケーションを作成することを常に夢見ているなら、Raphtersは始めるのに最適な場所です。 Cookie、セッション、およびテンプレートを処理するためのシンプルなAPIを提供します。 FastCGIを使用してアプリケーションを展開できます。




有望なリポジトリ



  • cldwalker / Tux ：Tuxは、Sinatra用のRubyコマンドラインインターフェイスです。 コマンドラインからアプリケーションのすべての部分と対話するためのシンプルなインターフェイスを提供します。 ヘルパーやビューを直接呼び出したり、要求/応答用の独自のデータを生成したりできます。
  • ender-js / Ender.js ：この新しいJavaScriptモジュールは、実際には、klass、qwery、bonzo、bean、script.js、reqwest、emile、underscore.jsの8つのミニマルなユーティリティのコレクションです。 Ender.jsは、わずか8 kbを占有して、これらすべての強力なモジュールを1つのエレガントなAPIに結合します。 機能：クラスシステム、カスタマイズ可能なセレクターエンジン、DOM操作、イベントシステム、非同期依存関係の読み込み、シンプルなアニメーション、および拡張機能用の堅牢なAPI。
