Windows Mobile用にリリースされたFring-PDAおよびスマートフォンでのSkype、Googleトーク、MSNおよびSIP

今日fringland Ltd. 無料のmVoIP（モバイルVoIP）クライアントfringの機能の拡張を発表し、Windows Mobileオペレーティングシステムを実行するデバイスのサポートを追加しました。 現在、fringプログラムは、Windows Mobileで実行されているほとんどすべてのPDAまたはコミュニケーターで使用できます。



Windows Mobileのfringは、Symbian OSの既に人気のあるfringクライアントの完全に機能する類似物であり、3G、GPRS、またはWi-Fi接続を使用して、fring、Skype、Google Talk、MSN、および多くのSIPプロバイダーで無料通話を行うことができます。 fring WMクライアントは、IM（インスタントテキストメッセージ）およびプレゼンス-ユーザーのオンラインステータスをリアルタイムで表示する機能もサポートしています。



fringが人気を博し、Symbian OSを実行している携帯電話でSkypeとGoogleトークを使用する最初でほぼ唯一の機会になったことを思い出してください。 その後、プログラムはMSNサービスおよびSIPオペレーターと連携する機能を追加しました。 Windows Mobileの場合、このクライアントは太陽の下で場所を取得するのが少し難しくなります。 Skypeユーザーは、最近積極的に開発されているPocket PC用の「ネイティブ」Skypeクライアントを長い間使用しています。 また、PDAおよびスマートフォン用のSIPクライアントのメーカーもそれほど遅れをとっていません。 さて、Jabberプロトコルの支持者は、お気に入りのプログラムでGoogleトークを使用することを長い間学びました。 したがって、フリンジの唯一の、しかし否定できない利点は、その汎用性です。 結局、fringは、通信に使用するネットワークをいつでも選択できる一種のハーベスターです。

選択に関しては、特殊なソフトウェアまたはオールインワンプログラムを使用するために、ユーザーの時間と個人的な好みが表示されます。
