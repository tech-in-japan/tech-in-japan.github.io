Professional videoconferencing is available as never before. Mind Server - new version, new prices

The domestic video conferencing solution (VKS) Mind Server has long been popular in Russia and not only. This is a professional system for conferences and webinars on the local network or via the open Internet. Mind Server provides support for SIP / H.323 hardware terminals, WebRTC connections and mobile clients, as well as conference recording, demonstration of content and desktop, integration with Skype for Business, and much more.

10 Key Changes



  1. The quality of the demonstration of the screen and windows of individual applications has become better.

    Skype captures the screen in HD (720p) quality, paid services like Webex and Zoom - in FullHD (1080p) quality. As a result, when scaling the desktop translation to full screen, remote participants see fuzzy fonts and blurry images, as modern monitors and demo panels usually have a higher resolution.

    In Mind, the demonstration is now in 4K UltraHD!



    Comparison with Skype






  2. Total support for WebRTC and the rejection of Flash.

    Previously, participants at the beginning of a conference through a browser had to explicitly allow the use of Flash. Unfortunately, this was not always done. As a result, they did not hear and did not see the speakers. The use of RTMP (Flash) technology has now been discontinued. For conferences and webinars, Mind always uses WebRTC technology. Flash will be supported by the Mind platform until the end of 2020, but only for backward compatibility.

  3. Implemented automatic media quality tuning.

    In addition to manual control, webinars are also provided with automatic adjustment of video broadcast quality depending on the width of the communication channel. In the autumn release will be implemented automatically adjusting the quality of media for conferences. This greatly simplifies the use of the service by participants and allows you to focus on the event.

  4. Significantly improved sound quality in conferences and webinars.

    Thanks to the optimization and refinement of algorithms for processing audio streams, the sound has become cleaner. Despite these improvements, one should not neglect the competent selection of audio equipment and acoustic preparation of the room.
  5. The degree of ergonomics and ease of use of the interface has increased.

    Numerous changes to the interface settings. In particular:

    On the ‚ÄúAbout Event‚Äù tab, an address for connecting VVoIP clients has been added: sip: event_id @ domain_address.

    Added the ability to redirect users to the portal of internal customer technical support by clicking on the link "Instructions and support".

    The autumn release will make even more changes in ergonomics in order to simplify the development of the service by users and reduce the number of unnecessary actions.

  6. Improved platform reliability and security.

    Switching to the new version of Debian has enhanced the stability and security of all platform components. The new version of Mind Server runs on the Debian GNU / Linux 9 operating system (instead of the no longer supported Debian GNU / Linux 7), which is supported until July 2020. In 2020, it is planned to migrate the Mind platform to Debian GNU / Linux 10.

  7. Improved data transfer security.

    Providing increased data transfer security for devices connected via SIP is possible when using TLS / SRTP protocols at the transport level.

    TLS encryption enhances the protection of media traffic transmitted over the open Internet.

    The Mind platform now uses the built-in Opensips 2.4 LTS.

  8. Simplified installation and updating of the system .

    System administrators will be able to independently install or update Mind in less than 30 minutes using the standard Debian installer with a graphical interface.

    Also, the installation time of Mind was reduced by 30% due to updating the settings of the OC kernel.

  9. Improved solution compatibility .

    The Mind platform supports H.261, H.265, VP9, ‚Äã‚ÄãMPEG-4 Part 2 video codecs and the G.722.1 audio codec in addition to those previously supported.

    Support for modern video and audio codecs allows you to connect a wider range of software and equipment.
  10. Unprecedented low prices .

    Prices for Mind cloud services in the updated tariff plans have become 2 times lower than the prices of their Western counterparts. License prices for local installation in a corporate network have become equal to the prices for Mind cloud services.

    All tariff plans now support the ability to connect hardware terminals of leading manufacturers and SIP telephony at no extra charge.

    In addition, an unlimited distribution kit of the Mind video conferencing server for one online event is available for download and independent operation.


Installation and basic setup



Mind Server is deployed from the finished image in a few steps. Installation is possible both on a physical server and in a virtual infrastructure. Minimum requirements: 4 physical cores and 16 gigabytes of RAM. Further - depending on the number and type of connections, as well as the resolution of the video.



In order not to describe the standard installation process, I offer a brief video review to help:





User default password mind: mind-3.15.0-pre7



After installing Mind Server 3.15, remote access to the console via SSH is disabled by default. If necessary, it can be resolved as follows:

In the file / etc / ssh / sshd_config, change the value of the parameter: PasswordAuthentication to: yes



You can edit the file from the console through the convenient mc file manager

Restart the ssh service with the service ssh restart command



Before allowing access through ssh from the outside, you need to change the password for the mind user with the passwd mind command



Then you can proceed to the basic server settings.



By default, 2 users are created:



mind with a password mind is a conditionally owner of a company, it only has a user interface and unlimited user rights.



admin with password admin - this is the system administrator, has unlimited administrator rights, but partial access to the user interface - cannot participate in active events and create them.



All subsequent administrators created will have access to the user interface with the opportunity to participate in events.

New prices



What's new?



First: the difference in licensing for SIP / H.323 connections and WebRTC connections has disappeared.

Now, any connection costs the same: whether it is a VKS hardware terminal, a laptop or a mobile phone - no difference. But it is important to note that the difference in the load on the hardware is still preserved: SIP / H.323 clients require significantly more resources than connecting via a browser.



Secondly: the minimum license package now is not 10, but 100 connections.

Thirdly: the minimum package of perpetual licenses for 100 competitive connections with technical support (and the possibility of renewal) for 1 year costs only 806'000 rubles , which amounts to 8'060 rubles per connection!
