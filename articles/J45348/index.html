GetAFreelancerとoDeskの開始の比較

高校の物理学コースから、静止摩擦力が最大であることが知られています。 フリーランスでの同様の状況：移動を開始し、最初の注文と最初のプロジェクトを取得することが最も困難です。 最初のプロジェクトが完了したら、その結果をポートフォリオのアイテムとして使用し、顧客レビューを実行可能性の追加証拠として使用できます。 この記事では、 GetAFreelancer （gaf）とoDesk （od）の2つのフリーランス交換を比較します。 取引所で作業を開始する便利さは、正確に比較されます。



見込み客がフリーランサーを評価するには、この人物に関する情報が必要です。 このプロジェクトでのフリーランサーの割合の運命は、この情報がどれだけ完全で美しいかによって大きく左右されます。 したがって、以下では、個人情報を入力する際に​​各交換が提供する機会を比較します。

新しいワークスペースで開始するには、初心者が利用できるサポートサービスとドキュメントが重要です。

また、追加のソフトウェアをインストールして構成するという点で、両方のエクスチェンジでの作業の快適さを考慮する価値があります。 フリーランサーをプロジェクト実行者として任命する前に交渉することの利便性も重要です。 これらの交渉の結果は、この利便性に依存する場合があります。



これらすべてのパラメーターを5段階で評価します。 見積もりは主観的ですが、少なくとも自分自身で結論を出すことができます。



  1. フリーランサーの一般情報

    ここでは、交換は実質的に同等です。 gafとodの両方は、あなた自身についての短い話を話したり書いたり、あなたの居住地、仕事の経験、プロのアクセント（デザイン、SEO、プログラミングなど）に関する基本的なデータを入力する機会を提供します。 Odの唯一の小さな利点は、必要な「週あたりの無料労働時間」フィールドです。 フリーランサーの可用性は、その値の入力を要求するのに十分な重要なパラメーターであると考えています。

    gaf / od = 4.8 / 5
  2. 交換で行われた作品のリスト。

    両方のエクスチェンジは、プロジェクトの作業を終えた後、顧客と請負業者にフィードバックを入力する機能を提供します。 ルールはほぼ同じです。レビューは、両当事者がレビューを導入するまで、またはプロジェクトの終了後2週間まで非表示になります。

    gafの小さな利点は、odでこのユーザーの不可解なIDではなく、プロファイルへのリンクとして顧客を表示することです。

    一方、カテゴリごとにフィードバックを入力することには、大きな利点があると思います。 評価は、gafのように1から10までの単一の数値としてではなく、個別に技術トレーニング、仕事の質、チャット/電子メール/電話の可用性、開発期限の遵守、製品要件を認識する能力、従業員とやり取りする能力を個別に評価することができますプロジェクト。 各パラメーターの重要度は異なり、合計スコアはすべてのパラメーターの評価の合計として計算されます。

    gaf / od = 4.2 / 5
  3. 組み込みテスト。

    gafにはテストはありません。

    Odは、サードパーティのエンジンを使用して、特定の知識分野でテストを整理します。 テストスイートの幅は十分です。 リストには、一般的なphp（4と5）とmysqlの両方、およびPerl、Python、Ruby（より正確にはRoR）などのまれな言語が含まれています。

    テストの質問は大体においてコンパイルされましたが、これらのテストをコンパイルすると、テクノロジーの変更にもかかわらずテストに戻ってこなかったという印象を受けました。 例として、Ruby on RailsのテストでModel.find_allの答えを示します。 この回答オプションは非推奨の機能であり、問​​題のRoRのバージョンを指定しないと、このオプションを評価することは困難です。 それとは別に、テストはまだ非常に深刻で（BrainBenchよりも複雑です）、すぐに失敗することは非常に現実的です。 テストを再度実行する機能は、このテストにパスする最後の試行の1か月後にのみ付与されます。

    テストはプロジェクト入札システムに統合されています。 つまり、顧客は、プロジェクトレートを登録するための必要条件として、技術テストの合格を追加できます。

    gaf / od = 0 / 4.8
  4. フリーランサーによる極端な賭け

    ほぼ同じように行われた両方の交換で。 odでは、ベットのイニシエーターが示されるフィールドが追加されました。 あなたがフリーランサープロジェクトに参加するよう招待されたとき、それはクールだと思われます。

    gaf / od = 4.9 / 5
  5. 技術研修

    gafでは、何ができるかについてのストーリーを書くtextareaが与えられます。 次に、このストーリーがプロファイルに表示されます。

    エクスペリエンスの説明は、個別のテクノロジーに分けられます。 各テクノロジーについて、[0..5]の推定値（このテクノロジーでの経験年数）を最後に使用したときと簡単な説明を与えることができます。

    odバリアントは、gafの「語形変化」より読みやすいように見えます。

    gaf / od = 3.5 / 5
  6. 認証

    odでは、gafとは異なり、成功したBrainBenchおよびZend認定に関するデータを追加することができます。 別のフォームを使用して、他の組織からの証明書を追加できます。

    gaf / od = 0/5
  7. ポートフォリオ

    添付ファイル、説明、分類などを使用してポートフォリオを形成する機会があります。 gafではこれはそうではありません。

    gaf / od = 0/5
  8. サポートとドキュメント

    gafにはサポートサービスがありますが、非常に遅く、非アクティブです。 不可抗力は、彼らが動くように起こらなければなりません。 gafのドキュメントからは、FAQのみがあります。

    odでは、FAQに加えて、使用例と説明を含む完全なドキュメントがあります。 サポートサービスは非常に迅速に対応します。 単一ユーザー規模の問題は、数時間以内に解決されます。 大きな混乱はありませんでした。 まあ、一般的に、サポートのサービスの人々がしようとしていると感じています。

    gaf / od = 2.5 / 5
  9. 追加のソフトウェアとその構成

    gafで作業するには、サイトに登録するだけです。 その後、アカウントから新しいプロジェクトを作成し、それらに賭けをすることができます。

    odでは、すべてがより複雑です。 正常に機能するためには、システムでの作業の基本を理解するための個別のテストに合格する必要があります。 フリーランサーと顧客ではテストが異なります。 テストに合格しないと、正常に動作できなくなります。

    テストに加えて、ローカルのoDeskマシンにスパイをインストールする必要があります。 これは、あなたが仕事をすることを確実にし、顧客が支払った時にポルノサイトを登らないようにする別個のプログラムです。 この制御は、デスクトップのスクリーンショット、マウスとキーボードの動作など、いくつかの方法で同時に発生します。 このスパイソフトウェアは簡単にインストールできます。 OS（Debian Linux）用のパッケージをダウンロードし、dpkgで接続しました。 そこにsources.listの行を表示するのが最適ですが、すでに唇を丸めています:)それをインストールしたので、プログラムは実際には調整を必要としません。 ユーザー名/パスワードを入力してください。 必要に応じて、スクリーンショットの撮影に関するポップアップ通知をオフにして、作業することができます。 当然、一定のインターネット接続が必要です。 スパイウェアプログラムは多くのトラフィックを作成しません。 habrakommentatorsが示唆するように、サーバーへの永続的な接続は必要ありません。 ネットワークにアクセスできない場合、プログラムはデータをローカルで収集し、できるだけ早く送信します。 最大バッテリー寿命は2時間です。

    一般に、要件の説明可能性と調整の容易さにもかかわらず、odはこの時点で失われます。 gafと比較して、odはより困難です。

    gaf / od = 5/3
  10. 事前設計コミュニケーションツール

    gafでは、公式にすべての連絡先はプライベートメッセージを介してのみ許可されます。 私の意見では、犬のキャラクター（@）も追跡され、除外されます。 これはすべて非常に単純に費用がかかりますが、それでもこのようなスパイゲームは迷惑です。

    odでは、顧客側のフリーランサーにわずかな関心があるだけで、顧客のメールがこのフリーランサーに送信されます。 ICQ、Skype、MSN、Yahooのニックネームをこのアドレスに書き込むことができます。 したがって、行動の完全な自由が与えられます。 窓や護衛にバーがないことは喜ばしいことです。

    gaf / od = 4/5


合計： gaf / od = 28.9 / 47.8



その結果、oDeskはGetAFreelancerよりも仕事に便利なサービスとして評価しています。 スパイウェアは私を悩ませません。 この情報から結論を引き出すことができます。
