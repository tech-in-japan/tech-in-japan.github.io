極端なカメラとしてのハイスクリーンブラックボックスアウトドアの操作経験

この投稿（http://habrahabr.ru/company/smartgadget/blog/189416/）ですでに述べたように、私はKodak EasyShare C123 Sportスポーツカメラの誇り高い所有者です。 原則として、このモデルは私全員に適していますが、それにもかかわらず、時々、他の何かのためにモデルを変更したいという欲求が生じます。 まず、光学ズームがありません。 第二に、これは「手動」カメラであり、それ以上ではありません。手で写真を撮るために使用できますが、このデバイスを地面、水上または水中の車両にしっかりと取り付けることはほとんど不可能です。 そして、テスト用にHighscreen Black Box Outdoorを手に入れました。このモデルは、スポーツカメラと車のDVRのハイブリッドです。 この投稿が生まれた結果、数週間使用しました。





Kodak EasyShare C123 Sportを搭載した屋外のハイスクリーンブラックボックス

ハイスクリーンブラックボックスアウトドアサイクリングヘルメット





自転車のハンドルバーで屋外のハイスクリーンブラックボックス



手始めに-特性について少し。 Highscreen Black Box Outdoorは、最大30フレーム/秒でフルHD 1080pでビデオを録画できます。 また、このデバイスは写真を撮影できます。写真の最大解像度は12メガピクセルに達しますが、カメラの解像度は実際には5メガピクセルです。 したがって、設定で解像度を5メガピクセル以上に設定すると、補間が使用されます。 ビデオの場合、Highscreen Black Box Outdoorでは使用されません。多くの予算登録機関で行われているマイナス記号付きのこの「トリック」は、過去にスムーズにフェードインしています（神に感謝します）。



外側には、Highscreen Black Box Outdoorは、一端にレンズがあり、MiniUSB、HDMI、2つのマイク、microSDカードスロット（最大32 GB、64は機能せず、チェック済み）、ビデオ解像度スイッチを備えたソケットが付いたシリンダーです（720p / 1080p）。







また、3つの完全なノズルの1つが巻き付けられているスレッドもあります。 それらの2つはプラスチックキャップです-穴の有無にかかわらず。 前者は、例えば、自転車のハンドルから撮影する場合に使用できます。後者は、水中撮影用です（気密性があるため）。 3番目のノズルは、いわゆるスクリーンモジュールです。 その上に、実際には、1.5インチの対角線、制御および構成用のキー、および別のMicroUSBポートを備えた小さなディスプレイがあります。 つまり、Highscreen Black Box Outdoorは、スクリーンモジュールを接続した状態で、単独で充電できます。 ちなみに、後者では、このレコーダー/極端なカメラは望遠レンズを備えた小さな一眼レフに似ています。









最初のキャップ「水中」





2つ目のプラグ、穴あき





接続されたスクリーンモジュールを備えた屋外のハイスクリーンブラックボックス



ケースはソフトタッチプラスチックで仕上げられ、非常にきれいに組み立てられています（微細な隙間があると破損する可能性があるため、「水中」ガジェットに特に当てはまります）。



Highscreen Black Box Outdoorにはコードが付属しているのは不思議ですが、どのように取り付けられているのかすぐにはわかりませんでした。 その結果、彼はウォーターパークの丘を下って移動し、デバイスは私の手から飛び出しました。そして、私たちは、5人の群衆の中で、プールの底で彼を探していました。 その後、レースが次のように固定されていることがわかりました。2つのキャップのいずれかをねじる前に、レコーダーにそのまま置く必要があり、このキャップで押し下げます。 レースは非常に確実に保持されます。







心のこもった、私は個人的にディスプレイ付きのモジュールであまり意味を見つけなかったと言います。 はい、キャプチャされたビデオと写真を表示できますが、私はこれらの目的のためにSamsung Galaxy S4を使用しました。 つまり、リストに従って、Highscreen Black Box Outdoorからカードが削除され、スマートフォンに挿入されました。 それはまさに私がKodak EasyShare C123 Sportでいつもやっていることです。ディスプレイはさらに大きくなっていますが、コンテンツの表示はまだあまり便利ではありません。 一方、レンズを道路または他の物体に向けながら、スクリーンなしで行うこともできます。HighscreenBlack Box Outdoorにはレーザーレベルがあります。つまり、よく知られているポインターのように赤いビームです。



ただし、スクリーンモジュールなしでは、Highscreen Black Box Outdoorはビデオのみを書き込むことができます。このため、ケースに特別なキーが用意されています（ちなみに、非常にタイトです）。 撮影するには、画面を接続するか、付属のリモコンを使用する必要があります。 これは、デバイスの非常に重大な欠点が個人的に隠れている場所です。屋外とは異なり、リモコンは水から保護されていないため、海に持ち帰ることはできません。 スクリーンモジュール-さらにそう。 もしそうなら、Highscreen Black Box Outdoorの助けを借りて水中で動画を書くことができます。 私は魚やウニの写真を撮るのが好きで、魚やウニのつらい生活を撮るのが好きです。







一方、Highscreen Black Box Outdoorはビデオを非常によく処理します。コダックよりもはるかに優れています。 そして、これは解像度（フルHD対VGA）だけでなく、その品質にも適用されます。 以下にいくつかの例を示します（YouTubeで視聴し、その前に解像度をFull HD 1080pに設定することを強くお勧めします）。











私の意見では、デバイスの円筒形状も議論の余地のある問題です。 はい、おかげで、Highscreen Black Box Outdoorは非常にコンパクトであることがわかりました（直径約30 mm、長さ-80）。 しかし、同時に、円筒形状のため、「ボタンアップ」位置で手に固定することは常に可能ではありません-それは地平線が崩壊せず、通常のローラーが得られるということです。 間違って手に取った場合、ネットワーク信号が失われます。 アンテナ、ビデオパンケーキは「斜め」になります。 もちろん、時間の経過とともに、必要に応じてBlack Boxを屋外に保つことを学びますが、そのようなニュアンスがあります。



水中でだけでなく、より極端な条件でもデバイスをテストすることができました。 私には、あらゆる種類のラジコンガジェットが大好きな友人がいます。 彼は現在、De Agostiniマガジンを使用して組み立てられたAMG Mercedes C-Class DTM 2008を所有しています。 つまり、あなたは数字を購入し、それらのそれぞれを完成させることが1つの詳細です。 購入し、回収し、ほぼ千ドルを費やします。出口では、Lexanボディ、スプリングサスペンション、時速80キロメートルまで加速できるガソリンエンジンを搭載した四輪駆動車を入手します。

出力はこのビデオでした（私は繰り返します：アスファルトは滑らかすぎず、サスペンションは小さくて硬く、Lexanのボディは左右に揺れ、最高速度は80 km / hです）：

コンパクトなため、Highscreen Black Box Outdoorは、分離されたスクリーンモジュールとそれの両方で、バックミラーの後ろに簡単に配置できます。 エンジンがオンになると、記録が自動的に開始されます。 スタンドアロンモードでは、デバイスは最大1時間半動作します。
