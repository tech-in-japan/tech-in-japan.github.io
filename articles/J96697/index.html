Ubuntuのテーマと簡単なセットアップを備えたConky-美しいダッシュボード

私の最後の投稿の後、多くの人が右側のスクリーンショットにどんなパネルがあるのか​​尋ねました。 これはConkyのダッシュボードであり、Ubuntu 10.04の新しいテーマです。 テーマ自体が存在するだけでなく、このテーマを設定するためのさらに便利なウィザードがあります。 これについてお話しします...



次のようになります（画面のいずれかの側にスナップして微調整する機能があります）。











まず、Conky自身が必要です。



sudo apt-get install conky







今、テーマが必要です。 トピックの作成者とそのセットアップウィザードはVeoduendesです。 ここからトピックをダウンロードしてください 。 アーカイブを解凍し、結果の.conkythemeフォルダーをホームディレクトリに転送します。 ディレクトリがすでに存在する場合、コンテンツのみが転送されます。

次に、セットアップウィザードをダウンロードします 。 アーカイブを解凍し、 ConkyWizardファイルを実行します。







デフォルトでは、Conkyは8秒の遅延で開始します。 ウィザードは、ホームディレクトリに.ConkyWizardLaunchファイルを作成し、スタートアップに追加できます。
