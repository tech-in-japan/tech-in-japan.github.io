Safari for Windowsブラウザの最終リリースが近づいています

今日、大Year日に、Apple Software UpdateはMS Windows XPでWindows用のApple Safari 3.0.4（523.15）の新しいベータ版をダウンロードしてインストールしました。

明らかな変更から：ベータ版の番号付けの最後の数字が変更され、以前の数字は（523.13）でした。

何が更新されたのか本当に理解していませんでしたが、私の考えでは、Apple.comの読み込みが速くなり、おそらくより安定しているはずです。

先月または2か月で、Safari 3 for Windowsのベータ版の3つまたは4つの更新プログラムを（間違えなければ）すでにリリースしました。

このような更新の頻度は、Appleの開発者がSafari for Winのファイナライズに焦点を合わせたことを示唆しており、最終リリースが間もなく登場することは明らかです。

Safari 3 for Windowsの最終リリースはMacworld Expo`2008に間に合うようにリリースされると想定できます（Appleはそのような主要なイベントのリリースをスケジュールするのが好きです）。 2008年1月15日！

現在、ベータ版に固有の不安定性のため、多くの人がSafari for Windowsのベータ版を使用することを警戒しています。 しかし、最終的なSafari for Winのリリースが、Netscape Navigatorの悲劇的な終withと非常に象徴的に一致することに気付かないことは不可能です:(

Safari for Windowsの最終リリースのリリース後、ブラウザ戦争でパワーのバランスがどのように変わるのだろうか？
