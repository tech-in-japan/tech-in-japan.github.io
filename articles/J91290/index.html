MonoTouch 3.0およびMoonlight 3 Preview 6がリリースされました

Miguel De Icazaは、彼のブログで、Monoファミリーの製品の新しいバージョンであるMonoTouchとMoonlightの入手可能性を発表しました。



Moonlight 3 Preview 6は、無料のSilverlight実装のメジャーアップデートであり、そのリリースには以下の変更点が含まれています。



-クロムアプリケーションのサポート（システムフレームなし）;

-パフォーマンスの大幅な改善。

-ブラウザー外を含め、Silverlight 3のほとんどの機能がサポートされていると記載されています。

-コード、互換性、またはバグ修正を改善するために、何百もの変更が行われました。



iPhone OS 4のリリースから1週間後、 MonoTouch 3.0.0がリリースされました 。これは、このオペレーティングシステムの開発をサポートしています。 変更点のリスト：



-バックグラウンドアプリケーションの開発のサポート。

-iAdプラットフォームのサポート。

-ローカル通知（通知）のサポート。

-Game Centerプラットフォームのサポート。

-エンタープライズデータ保護APIのサポート。



さらに、コード生成の改善と、Objective-Cブロックを介したC＃機能のより良いサポートが発表されます。



MonoTouch 3を使用するには、iPhone 4.0 SDKが必要です。
