Yandexはロシアとウクライナのモバイル検索のリーダーです









2010年2月25日-オスロ、ノルウェー



Operaがリリースしたモバイルインターネットの状況に関するレポートによると、2010年1月にOpera Miniブラウザーを使用して表示されたWebページの数は233億に増加しました。 これは2009年1月の3倍です。 同時に、世界中のブラウザーユーザーの数は5,000万人を超えました。



2010年1月のモバイルインターネットのステータスに関するレポートには、一般的な傾向に関するデータに加えて、さまざまな国や地域での検索エンジンの使用に関する情報が含まれています。 レポートの完全版は、 www.opera.com / smw （英語）で入手できます。



一般レポートデータ：

-2010年1月、Opera Miniのユーザー数は5,000万人増加しました。これは、1か月前より7.4％、1年前より149％増加しています。



-2010年1月のOpera Miniユーザーは233億のWebページを閲覧しました。 この指標は、2009年1月よりも208％多く、2009年12月よりも12.7％多くなっています。



-2010年1月、Opera Miniのユーザーは、世界中のモバイルオペレーターに対して3億3,700万MBを超えるトラフィックを生成しました。 これは前月よりも7％多く、1年前よりは176％多くなっています。 Opera Miniでは、データを元のボリュームの最大90％に圧縮できます。 したがって、Opera Miniサーバーで処理されるデータの合計量は約3ペタバイトです。



-2010年1月のOpera Miniユーザー数の上位10か国は、ロシア、インドネシア、インド、ウクライナ、中国、南アフリカ、アメリカ、ナイジェリア、ベトナム、イギリスのように（降順で）表示されました。



-ロシアでは、Yandex検索サービスはOpera Miniを使用して表示されるすべてのWebページの2.5％を占め、GoogleはWebページの0.7％を占めています。 Opera Miniのロシアのユーザーは、平均して月に14.3のWebページを検索結果とともに閲覧しています。



-ロシアのOpera Miniの年間ユーザー数は69.8％増加し、閲覧したWebページは75％増加しました。 各ユーザーは、2010年1月に平均8 MBのデータを生成しました。



-ウクライナのOpera Miniユーザーは、他のユーザーよりもモバイル検索を使用する可能性が低く、Opera Miniを使用してウクライナ人が開いたWebページの0.8％のみを検索サービスが占めています。 ウクライナの平均的なブラウザユーザーは月に6.1の検索Webページを閲覧しますが、ほとんどすべての場合、これらはYandexの検索結果ページです。



-一般に、ウクライナの年間のOpera Miniユーザーの数は122.3％増加し、閲覧したWebページは210.7％増加しました。 各ユーザーは、平均で19 KBの741ページ（圧縮後）を表示しました。



-ベトナムとインドのOpera Miniユーザーは、他のユーザーよりもモバイル検索を頻繁に使用します。 検索ポータルは、Webブラウザーを使用してベトナム人が表示するすべての30.9％を占めています。 インドの場合、この数字は22.1％です。 ベトナムとインドの平均的なOpera Miniユーザーは、1か月あたりそれぞれ63.6と75.8の検索Webページを閲覧しています。



Opera Softwareの共同設立者であるJon von Tetzchnerは次のように述べています。 -世界規模で最大の検索エンジンがGoogleである場合、個々の国ではローカルリーダーの多大な影響を確認できます。 ロシアとウクライナのYandexと中国のBaiduは、非常に印象的な結果を達成しました。」



モバイルインターネットレポートのアーカイブ（英語）： www.opera.com/smw/archive



Opera Software ASAについて

Opera Software ASAは、デスクトップコンピューター、携帯電話、その他のデバイス向けのWebブラウザーの開発における業界リーダーです。 Operaブラウザは、その高速性、コンパクトさ、および他のブラウザに比べてWeb標準のサポートが優れているため、ユーザーおよびメーカーから世界的に認知されています。 Opera Softwareの本社はノルウェーのオスロにあり、世界中のいくつかの国に支社があります。 同社の株式は、ティペラOPERAでオスロ証券取引所で取引されています。 会社の詳細： www.opera.com
