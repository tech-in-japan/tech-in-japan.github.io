ゲーム開発者は土曜日にCiklum Game Developersのドネプロペトロフスクに集まります

ウクライナとベラルーシの オフィスでは、 Ciklum Saturdaysが常に開催され、さまざまな技術を使用する開発者が集まっています。



次のCiklum Saturdayがゲーム開発とそれに関連するすべてのものに捧げられることを発表できてうれしいです！



Ciklum Game Dev Saturdayはドネプロペトロフスクで開催され、多彩で多彩で興味深いものになることを約束します。また、職業は会話の主題にまだ出会っていない人にとっても興味深いトピックです。



レポートとスケジュールのリストは現在作成中です。 そのため、次のスピーカーはCiklum Game Dev Saturdayへの参加を発表しました。

  • オレグ・アキモフ（ルガンスク）- 「2Dエフェクト。 早くて簡単。」
  • デニス・ロマンコ（ハリコフ）- 「プレイヤーが何回もゲームをリプレイするのはなぜですか？」
  • Valeria Mallaeva（ドネツク）- 「ゲームと漫画のフラッシュアニメーションの機能と違い」
  • Pavel Yuriychuk（キエフ）- 「ゲーム開発におけるSVG：あるべきかどうか」
  • Igor Khotin（Kiev）- 「フォースのダークサイド -MMO ゲームの舞台裏のサーバー」
  • Ivan Bely（ドネプロペトロフスク）- 「モバイルデバイス向けのクロスプラットフォームゲームの開発」
  • ウラジミール・コザエフ（キエフ）- 「等尺性日曜大工」
  • Vitaliy Khit（ドネプロペトロフスク）- 「「A」から「Z」へのゲームの作成」


最新の状態を維持するために、ブログの更新をお楽しみに。



したがって、私たちはすべてのゲーム愛好家を土曜日にCiklum Game Devのドネプロペトロフスクにある私たちのオフィスに招待します。

時期 ：9月24日土曜日、11：00から18:00

どこ ：ドネプロペトロフスク、聖 Barrikadnaya、15-A（ビジネスセンター「ステップ」）の。 チクルム



土曜日にCiklum GamDevにたった1日で、 ゲーム開発に興味のある100人以上が集まりました！ 登録されたすべての参加者に感謝します。 場所の数が限られているため、登録を終了せざるを得ませんでした。 イベントの数日前に、イベントの時間、場所、スケジュールに関する情報をメールで受け取ります。



別のスピーカーになるのはとても簡単です。 このアドレスに書いて、あなた自身について少し教えて、あなたのトピックを提案してください。



UPD：



やった！ 起こった！

9月24日土曜日、GameDevSaturdayはCiklum Dnipropetrovskオフィスで開催されました！

このイベントは、「多くの講演者は優れており、違います！」というモットーの下で開催されたと言えます。



最終機関では、すべての味と色について9つのレポートがありました。



1. Valeria {Lerika} Mallaev：「フラッシュアニメーションの詳細：ゲームvs漫画」;

2. Vitaliy Khit：「ゲーム開発のABC」。

3.デニス・ロマンコのレポート「ゲームの再現性を高めるもの」

4. Pavel Yuriychuk：「ゲーム開発におけるSVG：あるべきかどうか」

5. Oleg Akimov：「2Dエフェクト。 スムーズで簡単です」;

6.イゴール・ホティン：「ダークサイド・オブ・フォース-MMOゲームの舞台裏」

7.「手作りのアイソメ」というレポートで秘密を明かしたウラジミール・コザエフ。

8. Andrey Murashkoは、トピック「TVゲーム：キーボードをリモートに交換する」に関する経験を共有しました。

9.サムグレッグと講義「マーマレード：モバイルデバイス向けのクロスプラットフォームゲーム開発」。



プログラマ、モデラー、アニメーター、デザイナー、プロデューサーなど、さまざまなトピックが約60人の参加者を引き付けました。専門分野はまったく異なりますが、ゲーム開発にとっても同様に部分的です。



コーヒーを飲みながら議論になったレポートは、午後7時まで続き、その後、お互いをよりよく知り、チャットし、または単に電車に時間を渡すために、男たちはさらにリラックスした雰囲気に移行しました。



そして、イベントの後は、すべてが小さなものに任され、ビデオをアップロードし、写真を共有し、...次のコミュニティの仕事の日に備えましょう！
