Microsoft WebsiteSpark。 Parking.ruとのパートナーシップ条件

WebsiteSparkのWebスタジオおよびWeb開発者向けのParking.ruとのパートナーシップ条件が公開されています。 このページには、登録手順の詳細情報、要件、および詳細な説明が記載されています。 プログラムと特別オファーの要点に取り組む。





Parking.ruを選択した参加者は以下を受け取ります。



1. 仮想専用Hyper-Vサーバーの料金プランの全ラインの20％割引 。



2.参加者は、プログラムのホスティングパートナーとして、Parking.ruによってホストされているライセンスを無料で受け取ります。 ライセンスが含まれます：

•Windows Server 2008;

•SQL Server 2008;

•DotNetPanel WebsiteSpark Edition

（再販オプション付きの最大25ユーザー）;



3.マイクロソフトからソフトウェアの全製品を入手して、完全に機能するインターネットアプリケーションを作成します。

•Visual Studio 2008 Professional Edition（3ユーザーライセンス）。

•Expression Studio 2/3（1ユーザーライセンス）。

•Expression Web 2/3（2ユーザーライセンス）。

•Windows Web Server 2008 / R2（3ユーザーライセンス）。

•SQL Server 2008 Web（3ユーザーライセンス）。



4.クライアントのWebサイトを自社の機器に配置する場合、実稼働サーバー用の次のライセンスが利用可能になります。

•Windows Web Server 2008 / R2（4プロセッサライセンス）。

•SQL Server 2008 Web（4プロセッサライセンス）。



psプログラムは開発途上にあり、Hosting PartnerとしてParking.ruを選択したコミュニティメンバー向けに新しいオファーがまもなく利用可能になります
