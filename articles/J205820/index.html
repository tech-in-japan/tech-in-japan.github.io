ウェビナー：「Windows XPからWindows 8.1への効果的かつ安全な移行」

みなさんこんにちは！



おそらくご存知のとおり、2014年4月、MicrosoftはWindows XPのサポートを終了しました。 このOSを引き続き使用すると、ユーザーはセキュリティリスクとアプリケーションの互換性の問題に直面する可能性があります。 そのため、12月17日にデルとマイクロソフトは火曜日にウェビナーを開催し、Windows XPからWindows 8.1への移行とアプリケーション移行の自動化の実際的な側面について詳しく説明します。





プログラムで

  • Windows 8.1 ProfessionalおよびWindowsデバイスの利点。 Windows 8.1 Professionalの主な利点と、Windowsデバイスがビジネスユーザーの関心を引く方法に焦点を当てます。 Marat Khairulin、マイクロソフトロシア、CCG部門トレーナー兼コンテンツマネージャー
  • Windows XPからWindows 8.1に移行する際のリスクの軽減。 Dell ChangeBASEを使用して、既存のアプリケーションをWindows XPからWindows 8.1に移行する準備をする方法。 コンスタンティンシュルノフ、コンサルタント、Dell Software。
  • 従業員の生産性のためのさまざまなDellクライアントデバイス。 ビジネスクラスのラップトップからWindows 8.1を実行するタブレットまで。 デスクトップから高性能ワークステーションまで。 ビジネスのニーズが何であれ、デルには常にソリューションがあります。 Alexander Beloglintsev、Dell Russia、クライアントデバイスマーケティングマネージャー。




ウェビナーにアクセスする方法

すでに以前のウェビナーに参加した人は、X時間でここに行き、登録後に受け取ったメールアドレスとパスワードを入力するだけで十分です。



これが初めてのデルのウェビナーである場合、（上記のリンクで）簡単な登録手順を実行し、メールボックスに詳細が記載されたメールを待つ必要があります。



その後、12月17日13:00モスクワ時間に、同じリンクをたどって参加するだけです！ すべての自尊心のあるウェビナーで通常行われているように、質問と回答のセッションも提供されます。



PS Javaがインストールされていない場合は、ウェビナーを見るためにこれを行う必要があります。
