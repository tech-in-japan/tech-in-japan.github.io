Nokiaの新しいQWERTY背景：C3、C6、E5

ノキアは、QWERTYキーボードのファン向けの新しい携帯電話の発売を発表しました。







リリースへのリンクの写真をもっと見る。



Nokia C3



Series 40プラットフォームでの最初のQWERTY背景。

機内で：

  • Wi-Fi
  • 2 Mpxカメラ。
  • QVGA 2.4 "ディスプレイ。
  • Bluetooth 2.1;
  • 55 MBの内部メモリ。
  • 850/900/1800/1900 MHz GSM / EDGE;
  • microSDカードは最大8 GBをサポートします。


2010年第2四半期に90ユーロで販売。

ゴールデンホワイト、グレー、ピンクの3色が用意されています。





Nokia C6



Symbian（S60 5th edition）に基づくサイドスライダースマートフォン。

機内で：

  • Wi-Fi
  • 5 Mpxカメラ。
  • nHD（640 x 360）3.2 "ディスプレイ;
  • FMラジオ
  • microSDカードは最大16 GBをサポートします。
  • A-GPSおよび無料のナビゲーションOviマップ。
  • Bluetooth 2.1;
  • 850/900/1800/1900 MHz GSM / EDGEおよび850/900/1900/2100 HSDPA / UMTS;
  • GSMネットワークでの最大7時間の通話が記載されています。


2010年度第2四半期に220ユーロで販売。





Nokia E5



E71 / E72の代替品：2 mm薄く、2 g軽くなります。



SymbianベースのQWERTYバックグラウンド（S60第3版）。

機内で：

  • Wi-Fi
  • 5 Mpxカメラ。
  • QVGA 2.4 "ディスプレイ。
  • FMラジオ
  • microSDカードは最大16 GBをサポートします。
  • 250 MBの内部メモリ。
  • A-GPSおよび無料のナビゲーションOviマップ。
  • Bluetooth 2.1;
  • 850/900/1800/1900 MHz GSM / EDGEおよび850/900/1900/2100 HSDPA / UMTS;
  • GSMネットワークで最大18時間の講演を発表。
  • GSMネットワークでの待機時間は最大29日間です。


2010年第3四半期に180ユーロで販売。

利用可能な色：カーボンブラック、チョークホワイト、スカイブルー、カッパーブラウン、シルバーグレー。
