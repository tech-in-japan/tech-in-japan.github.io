Windows 8.1への無料アップグレードの公式発表





Microsoftは、以前はWindows Blueと呼ばれていたWindows 8.1オペレーティングシステムの更新を正式に発表しました 。 最も重要なこととして、同社はすべてのWindows 8ユーザーの無料アップグレードステータスを確認しています。



発表のテキストは、「Windowsは引き続き改善されています」という見出しの下に公開されています。 マイクロソフトは、半年間で販売された1億個のWindows 8ライセンスに関する統計を引用して、 「Windows 8の進歩」について語っています。



Microsoftは、Windowsストアのオンラインカタログで、Twitter、eBay、Netflix、CNN、Temple Runゲームなどのアプリを含む70,000以上のアプリにも言及しています：Brave、Bejeweled LIVE、Angry Birds Star Wars など 。



興味深いことに、公式発表では、Windows 8.1で予想される最も重要なイノベーションについては何も言及していません。プロジェクト管理者が「重要な側面の変化」と呼ぶものについては言及していません。また、独立したアナリストは、New Cokeの廃止以来1985年。



ほとんどの場合、これは「スタート」ボタンを返すことと、Metroをバイパスしてデスクトップモードに直接ロードする可能性に関するものです 。これらは、 Windows 8ユーザーにとって最も欠けている機能です 。



Microsoftの関係者は、Windows 8.1のパブリックプレビューバージョンが2013年6月26日-開発者向けビルド会議にリリースされると述べました。 数週間以内に、同社はアップグレードに関する追加情報を公開することを約束します。
