LinguaLeoによる英語学習者向けのモバイルアップデート：iPhone、WinPhone、そして待望のAndroid！





ビジネススクールのシーズンが始まるまでに、LinguaLeoチームはモバイルアップデートを準備しました。iPhoneとWindows Phoneのアプリケーションの更新、およびAndroidデバイスでのベータテストの開始です。 モバイルニュースの詳細を読む詳細を読む...



新しいiPhone



事実 これは、アップルの従業員による最長のアプリレビューでした。 それは3週間以上続きました。



アプリケーションのデザインを更新し続けます。







鳥がホーム画面に表示されました-あなたが勉強し始めたトピックを思い出させてくれます。







進行状況インジケータは、緑色のバーの形で各テーマ内に埋め込まれています。

トピックの習得の進捗状況が20％の場合、鳥がホームページに表示されます。



私はボタンを知っています 。 トピックや辞書で既に知られている、またはよく学習した単語をマークします。 「知っている」とマークされた単語は、辞書を離れ、語彙を補充し、トレーニングに落ちません。 そして、あなたがまだ疑っている知識の程度である言葉が訓練に送られます。 辞書の編集モードが完全に再設計されました-操作が大規模になりました（選択したすべての単語に対して同時に）。



トレーニングの更新







トレーニング中の新しいフィルターに注意してください。 単語数が制限されるため、「ほぼ学習済み」の単語を完成させたり、現在学習しているトピックの単語を訓練したりできます。







トレーニングがよりインタラクティブになりました：単語をカードにドラッグアンドドロップし、スワイプで単語をスクロールします（タップによる正しい答えの選択が保存されました）！



フィードバックの重要性について







私たちはあなたのフィードバックを注意深く見て迅速に対応します！ 新しい「Wordコンストラクター」について相反するフィードバックを受け取り、更新プログラム付きのクイックフィックスビルドをリリースすることにしました（既にレビュー中です）。 スクリーンショットは、「Word Designer」の更新バージョンを示しています。これは近い将来に表示されます（リリース日は、Cupertinoのメンバーによるレビューの速度によって異なります）。



「リスニング」では、キーボードは常に目の前にあります。



クイックフィックスの更新には、トピックで行われているように、スワイプによる辞書内の単語の「スクロール」も含まれます。



新しいワークアウト「フレーズを収集」







あなたの仕事は、有名な人々のフレーズを別々の単語から集めて、それほど有名ではない格言を得ることです。



アプリケーションの開発を継続し、新しいトレーニング形式を提示します-フレーズを収集します。 トレーニング中に、なじみのない単語を辞書に追加できます。 最初の20のフレーズはデモモードで使用でき、残りは「ゴールデンステータス」のユーザー用です。 ワークアウトを水平に実行することも、クイックフィックスアップデートの一部になります。



iPhoneチームの結果を評価する -iPhone用LinguaLeoをダウンロードします 。



このトレーニングフレーズや新しいトレーニングフレーズに関するアイデアや提案については、コメントまたはFacebookのグループでお気軽にご相談ください。



そして、警告の質問、私たちは次の大きなiOSアップデートがiPadバージョンに捧げられることを約束します！ ニュースをフォローしてください。



Windows Phoneの新しいバージョン



Windows Phoneアプリケーションは最近リリースされたため、他の誰かがそれを評価する時間がなかった可能性があります。







最新リリースでは、ワークアウト、単語カード、トピックの単語カードのデザインが根本的に更新され、明るくてジューシーになりました。







今、あなたは美しいアニメーションでスワイプで単語を切り替えることができます。



さらに、ユーザーのすべての要望を考慮し、以前のバージョンの欠点を修正し、一部のデバイスで誤ったサウンドを再生する理由を見つけ、作業速度を最適化し、アプリケーションをさらに安定させました。 ふう...



この機会を利用して、 Windows Phone RussiaチームとAtreides07がアプリケーションの開発に積極的に協力してくれたことに感謝します！



WPチームの結果を評価するには、 Windows Phone用のLinguaLeoをダウンロードします 。



そして...ドラムロール...待望のAndroidアプリケーションのベータテストを開始します！







彼らはユーザーに、使用しているAndroidデバイスと、これらのデバイスにインストールされているOSバージョンの種類を尋ねました。 その結果、5人のリーダーが特定されました。



  1. Samsung Galaxy S III
  2. Samsung Galaxy S II
  3. サムスンギャラクシーエース
  4. HTC Wildfire
  5. HTC Desire S


調査によると99が見つかりました！ 一般的なデバイスバージョン。 まず第一に、SamsungおよびHTCデバイスでテストしますが、アプリケーションがすべてのデバイスで動作するようにしたいと考えています。 さまざまなデバイスとOSバージョンのベータテスト参加者を選択します。



ベータテスターの数に該当する場合は、1週間以内に、詳細が記載された対応する手紙を受け取ります。 すべてのAndroidモデルの所有者がテストに参加できます。 これは、アプリケーションを改善するのに役立ちます！
