アールデコPC-仕事用のスタイリッシュなコンピューター





コンピューター技術の普及により、手や機敏な頭を持つ多くの職人が興味深いプロジェクトを作成し、PCの標準的な外観を変えています。 プロジェクトの一部は、美術館や展示会に至るまで、実際の芸術作品です。 その過程で気を散らされないほど実用的なプロジェクトがあります。 そのようなプロジェクトの1つがアールデコPCです。



このコンピューターを見ると、これは20世紀半ばから登場した一種のデバイスであると思われるかもしれませんが、実際にはコンピューターは非常に近代的です。







基本はMini-ITXプラットフォームでした。 デバイスの特性は印象的です。



プロセッサー：Intel Core i3-2105

グラフィックス：Intel HD3000

ストレージ：256GB M4 SSD

RAM：8GB DDR3

OS：Microsoft Windows 8







残念ながら、これはたった1つの例であり、手作業であり、アールデコPCはまだ大量生産には至っていません。 近い将来、プロジェクトの作成者であるJeffrey StevensonがこれらのPCをさらに10個リリースする予定であり、そこに表示されるでしょう。おそらく、大手メーカーの1つが興味を持つでしょう。 これまでの価格については何も知られていない。
