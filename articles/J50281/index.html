Habrのエディター

Habraエディター。



背景


彼はブログのオンラインコンテンツエディターを作りました。 そして、私はメモ帳++などの通常のエディターを使用することに慣れています。 私はWYSIWYGエディターのサポーターではないため、通常のテキストエリアに基づいてエディターを作成しました。 エディターの利点は何ですか？

  1. Office 2007スタイルの便利なツールバー（実際の使用量がわかります）
  2. ホットキーのサポート（ほぼすべての機能）
  3. Tabはタブを配置し、次の要素にジャンプしません（4つのスペースを入れます）
  4. Shif + Tabはタブを削除します
  5. 巧妙な入力（折り返すとき、整列のために前の行のようなタブを挿入します）
  6. Shift + Enterは、スタイルHTMLタグに改行を挿入します


私は何を得ていますか？


これには都合が良さそうに思えたので、この幸せを他の人と共有することにし、Habrの記事を書くサービスを作りました。 また、Habrには多くのコードが記述されているため、ブログで使用されているコードの色付けを実装することにしました。 そして、良いハブが彼に受け入れられないすべてのタグを切り取るので、私はGeSHiライブラリを使用します。 ファイルを少し変更する必要がありましたが、今ではhabrosocietyがペイントでコードを書く機会があります。 また、このライブラリは非常に人気があるため、80以上の言語を強調表示できます。 色合いは記事のテキストに直接実装されます。コードタグのクラスを指定する必要があるのは、 < code class = "html" > < / code >



ます。

強調表示したくない場合は、何も指定する必要はありません。色付けせずにインデントをフォーマットします。
