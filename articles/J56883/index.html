Mysql Documentor-Mysql 5+データベースを文書化するためのシンプルなユーティリティ

どうしてそんな人生にたどり着いたのか

ここでは、Slideshareから美しく短いプレゼンテーションを挿入したかったのですが、UFOはそれを好まなかったようです=）



何ができる

Mysql Documentorは、データスキーマ、テーブルおよびフィールドへのコメントを読み取り、単純なSmartyテンプレートを使用して、html記述を生成します。



今日、MyDocは次のことができます。

  • コメント付きのテーブルのリストを作成します。
  • コメント付きの各テーブルのフィールドのリストを作成します。
  • 各テーブルのキーのリストを作成します。
  • 外部キーがある場合、フィールドから関連テーブルへのリンクを作成します。
  • mysqli拡張機能を備えたphpを実行しているすべてのプラットフォームで動作します。
  • smarty-syntaxテンプレートを使用すると、任意の設計および形式（非HTMLを含む）のドキュメントを作成できます。
  • 起動スクリプトで簡単に直接設定できます。


どのように方法がわからないが、すぐになります

  • テーブルとフィールドの統計なし。
  • トリガーおよびストアドプロシージャに関するドキュメントは、いかなる方法でも処理および構築されません。


近い将来、他に何がありますか

  • コメントのタグに基づいてテーブルをグループ化します。
  • Graphwiz形式の出力データベーススキーマ。


取得方法

私たちのウェブサイトからダウンロードするだけです （そこにビルドされたドキュメントの例があります）。



動作させる方法

実際に、誰かを強制する必要はありません。 README.TXTファイル（現在は英語）を見て、いくつかの簡単な指示に従ってください。 数分で十分です。



そして、何のために、無料で？

はい Mysql Documentorは、 GPLの条件の下でソースで配布されます 。 ところで、 info @ softuniq.comにあなたの貢献を送ることができます（そしてすべきです！）。 SVNリポジトリは近日公開予定です。



次は？

ユーティリティ開発は本格的です。 居心地の良いブロガーのRSSフィードを購読して、ニュースをフォローすると便利です。 =）
