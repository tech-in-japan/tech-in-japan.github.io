日本からシンガポールへ：新しい海底ケーブルがアジア9か国を通過

Southeast Asia-Japan 2コンソーシアムは、日本の通信機器メーカーであるNECと、144 Tbit / sのスループットでSJC2海底ケーブルを敷設する契約を締結しました。 ケーブルは9か国を接続します。



以下のプロジェクトの詳細について。





/写真COMSEVENTHFLT CC



ケーブル仕様



ケーブルの長さは10.5千kmで、日本、韓国、中国本土、台湾、香港、タイ、カンボジア、ベトナム、シンガポールを接続します。



SJC2は8つのファイバーペアで構成されます 。 144 Tbit / sのスループットでは、各ペアのデータ転送速度は18 Tbit / sになります。 このプロジェクトは、2020年の第4四半期までに完了する予定です。 SJC2のコストは明らかにされていませんが 、以前のケーブル（SJC）に4億ドルが投資されたことが知られています。



なぜケーブルが必要なのですか



SJC2は、2013年に設置された SJCを補完します。 SJCとSJC2は中国、香港、日本、シンガポール、タイで「交差」しますが、2番目のケーブルはカンボジア、韓国、ベトナム、台湾の新しい国々を接続します。



SJC2議長のLinette Lee は 、SJC2の構築がアジアのデジタルコラボレーションの重要なステップであると考えています。 新しい、より強力なケーブルは、初期設計のスループットがわずか28 Tbpsであった前身を補完します。 ケーブルシステムは、通信が途切れないようにし、経済的に活発な地域の企業とその顧客の遅延を削減するのに役立ちます。



台湾の通信事業者の代表の1人である電気通信会社Chunghwa Telecomの会長兼CEOであるYu Chengは、SJC2ケーブルは2020年に5Gサービスを開始する際に多くのマルチメディアアプリケーションの動作を保証するのに役立つと主張しています。



プロジェクト実行者であるNECの側では、川村mura夫氏からコメントが寄せられました。 彼は 、SJC2が地域の帯域幅要件のその後の増加に対処することを強調しています。



NECを選ぶ理由



NECは偶然このプロジェクトに任命されていなかったと考えられます。その従業員はそのようなプロジェクトの実施に豊富な経験を持っています。



2016年11月、同社は、中国、香港、日本、韓国、マレーシア、台湾、タイ、ベトナム、シンガポールを接続するアジア太平洋ゲートウェイ（APG）海底ケーブルを完成させました。 APGデータレートは54 Tbpsです。



2017年5月、組織の専門家は、長さ11,000 kmの水中ケーブルを介して50.9 Tbit / s（1ファイバー）の速度でデータを転送しました。 科学者は最適化されたopt32変調アルゴリズムを使用して、6.14ビット/秒/ Hzのスペクトル効率を得ることができました。 その際、エルビウムイオンドープ（EDFA）光ファイバー増幅器を使用しました。



また、2019年の第4四半期に、NEC は香港とグアムを接続するケーブルシステムの敷設を完了する予定です。 長さは3900 kmで、スループットは50 Tbpsです。

アジア太平洋地域の他のプロジェクト



2018年の第2四半期には、トライデントプロジェクトが完了します。 この海底ケーブルは、シンガポール、インドネシア、オーストラリアを接続します。 そのスループットは28 Tbpsです。



別のケーブルであるハワイキは、2018年半ばまでにオーストラリアとニュージーランドをハワイおよび米国西海岸に接続します。 ケーブルの全長は14,000 kmで、スループットは30 Tbit / sです。



今年完成予定のもう1つのプロジェクトは、ベンガル湾のゲートウェイ（ BBG ）ケーブルシステムです。 シンガポール、マレーシア、インド、スリランカ、オマーン、アラブ首長国連邦を統合します。
