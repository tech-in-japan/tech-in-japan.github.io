弱い信号での飛行：フクロウ、タカ、ビーバー、ビデオ監視





フクロウの巣は通常、高さ-地上25メートルまであります。 そのため、突然フクロウに餌をやるつもりなら、木のてっぺんに登る準備をしてください。 この事実（および他の多くの事実）は、珍しいプロジェクトによって引き起こされた短期間の共同行動で発見されました。



羽毛の捕食者の研究と保護のためのロシアのネットワークの助けを借りて、私たちは不明瞭にするためにカメラを家族の巣に設置しました。 フクロウは人々の細心の注意に注意を払わず、すぐに3羽の鳥を連れてきました。 そして、巣からの生放送と、困難な状況での監視カメラの設置に関する素晴らしい経験を得ました。



難易度は木の冠にintoい込むことではありませんでした。設置者は、タワークレーンや未完成の建物の上層階に機器を設置します。 いいえ、主な問題はインターネットでした。



解像度が640 x 480のビデオを1台のカメラで1秒あたり0.5〜1 Mbit / sでブロードキャストするために必要な、推奨される通信チャネル。 しかし、優れた放送品質とインターネットが必要でした。 森のどこで入手できますか？



森林観察







フクロウ自体は巣を作りません。 繁殖するために、ほとんどの種は木のくぼみを使用しますが、都市のくぼみの木は緊急事態とみなされ、伐採されます。 鳥類学者は、フクロウの営巣地を追跡し、大きな巣箱のように、彼らのために特別な家を独自に設置しようとします。



各鳥類学者は自分の病棟を監視しますが、これは簡単ではありません。フクロウの様子を知るには、彼女に行く必要があります。 すべての羽のある鳥にカメラを設置することはできませんが、どこかで始めることにしました。Ivideonの環境監視は、新しい珍しい市場です。



生態学者が試験管理のために選んだ巣は、ニジニ・ノヴゴロドから数キロのところにありました。 理論的には、信号は街から出ましたが、ルーターを拾って木に固定することはできませんでした。 乱れ、電源からの距離が大きい。







結局、アイデアは成功でした。 「スコッチテープとスティック」の文字通りルーターを使用してソリューションを実装し、食料はフォレスターの家から延ばされました。 初めて、すべてがうまくいきましたが、しばらくして、シチェロコフスキー・クトール保護区のオオタカの巣にカメラを入れなければなりませんでした。



鳥のタカは秘密であり、通常の環境では専門の鳥類学者だけが捕食者に気付くことができますが、誰でもカメラを通してそのような鳥を見ることができます。 もちろん、主な幸福は鳥類学者にあります。鳥のビデオ監視は貴重な情報を提供します-繁殖の監視、雛の成長の追跡、鳥の不安の原因の特定。 カメラはまた、鳥が飛び出したとき、またはフレーム内で他の活発な動きが起こったときに鳥類学者に通知を送信します。



一般的に、ビーバーのファンが似たようなリクエストで来ましたが、何よりも夏の住人からリクエストを受け取りました。冬の家と区画を観察することが重要です。 市外では、一部の地域でのコミュニケーションは（たとえそうであっても）非常に貧弱で質の高い写真であり、1台のカメラから得るのは難しい場合があります。



ブラックボックスを組み立てる







したがって、ビーバー、タカ、またはカントリーハウスを観察するかどうかにかかわらず、問題は1つだけです。 既存のソリューションでは、受信条件が悪い3 / 4Gの場所で、24時間年中無休でオンラインで動作する困難な条件でシステムを動作させることができませんでした。



最後から始めましょう。 これで、インターネットのないオブジェクト向けのオールインワンソリューションが得られました。VideoStationM20は、単一のMIMO（マルチ入力/マルチ出力）アンテナ20 dBiで結合して3 / 4G信号を増幅し、モデム、パッシブPoEを備えたルータースイッチ（ 4ポート）と電源。



原則として、同様のものを個別に組み立てることができますが、何らかの理由で、SIMカードを挿入して220 VACコンセントを見つけてPoEカメラを接続するだけのリモートオブジェクト用の既製のソリューションはありませんでした。 「代替品」があります-ルーターのメーカーは、外部アンテナと内蔵または外部4Gモデムを備えたモデルを使用します。 たとえば、Zyxel Keenetic 4GIII。4GUSBモデムとUSBモデム用の信号増幅器（外部アンテナ）を接続できます。



私たちがしていること：MIMOアンテナ（効果的な信号増幅）を使用し、PoEをサポートするルータースイッチを組み込みます。 これらはすべて「ゆるい」ものではなく、単一のコンパクトなケースに収められています。



中身は？







取得したものについて詳しく説明します。 上の写真-ビデオステーションとノーベル監視カメラ 。 ステーションの寸法は、420 ​​x 420 x 110 mm、重量5 kgです。







ビデオステーションには、20 dBiのゲインを持つ2つのアンテナと、平面内で90°間隔（MIMOテクノロジー2 x 2）、PoE IEEE 802.3 af-2003を備えた4 x 10/100 LAN Base-TXを備えたルータースイッチが含まれます。 カメラがPoEをサポートしていない場合は、スプリッターを使用できます。



このデバイスは、220 VAC（通常のSchukoプラグ付きのコード）から電力を供給されます。 これにより、追加の機器なしでカメラをビデオステーションに直接接続できます。これは、特に遠隔地に設置する場合に非常に便利です-ポール、ガレージ、倉庫、そうでなければルーターとPoEスイッチ、またはインジェクターに追加のサーマルボックス（キャビネット）を使用する必要がある場合







2芯の電源ケーブルは、デバイス内部の電源のクランプ端子と圧力シールを介して接続されます。







「管理」ポート（右端）から、ツイストペアケーブルを使用してコンピューターに接続することにより、デバイスを構成できます。 また、それを介して、あなたはすぐにインターネットに接続し、何も触れないことができます。







RJ-45ケーブルがルーターに接続されています。 ケーブルをポートに接続すると、ルーターはケーブルを介して12V 0.5Aの電力の供給を開始します。



MegaFon SIMカードはレビューのために手元にありましたが、Yota、Megafon、MTS、Beeline、Tele2の3 / 4GオペレーターのSIMカードでの作業をサポートしています。 ほぼプラグアンドプレイ-モデムとルーターの設定はWebインターフェイスを介して可能ですが、必須ではありません。



Webインターフェイスを介して、SIMカードの残高を確認できるだけでなく、「ピックアップ」したベースステーションと信号レベルを確認することもできます。



現在、Videostationは、全天候型のUV耐性プラスチックとアルミニウム製の外装ケースで作られています。 ケースは、湿気やほこり、極端な温度から保護されています。-40°C〜+ 75°Cで動作します。



サマーレジデンスのビデオ監視



よく聞かれる最も一般的なオプション。 システムをテストするために、到着して複数の顧客に配送しました。 いずれの場合も、ダウンロード信号は弱すぎました。標準の「ホイッスル」では約0.5 Mb / sです。 これにより、コテージで2台のFull-HDカメラをIvideonサービスと接続することはできませんでした。たとえビットレートと解像度を最小限に抑えたとしてもです。 合計で、5台以上のカメラをビデオステーション（またはLTEステーション）に接続できます。



初めて、ドイツの出力を1つしか持たない通常のLTEステーションM20を接続しました 。この出力は、インターネットを内部（ホーム）ルーターまたはスイッチに送信し、カメラまたはその他のデバイスはすでに巻き取られています。 LTE Station M20は、最大100メートルのツイストペアケーブルでコンピューターまたはホームルーターに接続します。これにより、高いマストや屋根などにステーションを設置できます。 電源は同じケーブルを介して供給されます。つまり、長距離にわたって電源を引っ張る必要はありません。







ステーションの設置後に接続速度を測定しました。 条件に応じて、サポートされる速度は3G-42 Mb / s、4G-97.5 Mb / sです





ビデオステーションM20（およびLTEステーションM20）については、基地局からの距離は最大10 kmであると記述していますが、理論上は最大20 kmの見通し線はすべて地形の特徴に依存します。



ご質問がある場合は、ビデオステーションはまだIvideonストアに追加されていません。 このサイトには、 LTE Station M20 、 LTE Station M18 、 LTE Station M14の完全な仕様があります。 それらの違いは、デシベル（それぞれ20、18、14）、サイズ、重量、価格です。
