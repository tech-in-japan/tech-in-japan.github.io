Ultrabook Fujitsu Lifebook U772

多くのユーザーは、ウルトラブックは同じラップトップであり、薄くて軽いと信じています。 実際、すべてがやや複雑です。 ウルトラブックと呼ばれるためには、ラップトップは重量と厚さだけでなく、パフォーマンス、起動時間、バッテリー寿命、管理、監視、およびデータ保護技術のサポートに関するいくつかの厳しいインテルの要件を満たす必要があります。 これらのパラメーターのすべてが各ユーザーにとって重要というわけではありませんが、Intelの目標は、業界全体の発展のためのベクトルを設定することであり、これにうまく対処しました。







特徴によると、外観も、富士通Lifebook U772がウルトラブックコンセプトの具体化として設計されただけではないことは明らかです。 むしろ、それはそれから最大を絞り出し、無条件に完璧なウルトラブックを作る試みです。 ITプロフェッショナル、企業ユーザー、ビジネスマンなど、富士通のお気に入りの視聴者にとって、まず第一に理想的です。



ラップトップの「信じられないほど」そして「非常に小さい」厚さについての言葉は、すべての最初のプレスリリースに見られるようになりました。 それでも、富士通の開発者はフォームファクターに最大限の注意を払い、14インチマトリックスを備えた最も薄いラップトップを手に入れました。 最も厚い場所で15.6 mm-それほど単純ではないので、インターフェイスポートの数を減らしてパフォーマンスを犠牲にする必要があります。 重量も非常に控えめです-1.4 kg、約2つのiPad。 手のひらの上で長時間保持することはできません（なぜですか？）が、その質量はブリーフケースやバッグにはほとんど感じられません。







重さの大部分はケースの下半分にあり、開くとウルトラブックは邪魔になりにくく、ループの剛性は片手で蓋を開けられるように選択されています。 とても便利です。



彼らが今言うように、ラップトップのディスプレイはフレームレスです。 正確には、画面の周囲にはまだフレームがあり、見えないだけで、HDカメラが上からカットされています。 ディスプレイは、反射防止コーティングが施された1366×768ピクセルの解像度のTNタイプのマトリックスに基づいています。 画面はマットで、喜ばせざるを得ません。 おそらく光沢はもっとジューシーな絵を示していたでしょうが、まぶしさの欠如は仕事にとってより重要です。







U772内では、すべてが妥協することなくしっかりと配置されています。IntelQM77チップセットをベースにしたマザーボード、i5-3317Uからi7-3687UまでのIntel Core i5またはi7プロセッサ、1つのDDR3-1600モジュールを備えた4または8 GBのメモリ、およびいくつかの可能な構成のディスクシステム。 ユーザーは、320ギガバイトまたは500ギガバイトのハードドライブとキャッシュ用の32 GBソリッドステートドライブ（Intel Smart Response Technology）の組み合わせ、または128または256 GB SSDのいずれかを自由に選択できます。 これにより、購入時に、ディスクシステムの容量だけでなく、そのパフォーマンス、OS起動時間、ラップトップ全体の重量を味わうことができます。 この弱いスタッフィングはすべて、1回のバッテリー充電で7〜10時間生きることができます-ドキュメントやWebサーフィンでの作業に制限されている場合。 大量のリソースを消費するタスクは、バッテリー寿命を2時間に短縮できます-多すぎませんが、ほとんどの場合、これはコンセントに到達するのに十分です。



ウルトラブックの通信機能は包括的です-WiFi 802.11 a / b / g / n、Bluetooth 4.0、イーサネット、および国内事業者の周波数をサポートする3G / LTEモジュール。 ちなみに、SIMスロットは非常に巧妙に隠されています-ボトムパネルの背面から、まだ蓋で覆われています。 ただし、多くの場合、そこを調べる必要はありません。



Fujitsu Lifebook U772の他のすべてのポートは、サイドパネルにあります。 左側には、電源アダプタコネクタ、ケンジントンロックスロット、USB 2.0ポート、およびヘッドフォンジャックがあります。 右側には、カードリーダー、2つのUSB 3.0ポート、HDMI、および縮小フォームファクターのイーサネットポートがあります（つまり、RJ-45を接続するにはアダプターが必要です）。



しかし、これは非常に一般的なことですが、非常に珍しいのはキット内のドッキングステーションの存在です。 より正確には、それなしで購入できますが、自宅と職場の両方でラップトップを使用する場合、非常に便利です。 ドッキングステーションはデスクトップ上にあり、ラップトップをバッグから取り出して置きます（そしてほとんど見えず、コネクタは対応するソケットを見つけるようです）。ウルトラモバイルコンピューターは、ネットワーク、7つのUSB 3.0ポートを搭載した本格的な固定ワークステーションになります、RJ-45、D-Sub、DVI、ディスプレイポートのビデオ出力に加えて、HDMI。











従来、富士通の開発者は、あらゆるレベルでデータセキュリティ機能に特別な注意を払ってきました。 U772のすべての構成には、バイオメトリックセンサー（指紋スキャナー）があり、これを使用して、OSおよび強化されたセキュリティ対策を必要とする他のソフトウェアの両方でユーザーを認証できます。 使用されるソリッドステートドライブは、フルディスク暗号化（FDE）をサポートしています。 そして最後に、最も用心深く、オプションのTPMモジュールが利用可能です。 したがって、ウルトラブックを紛失しても、機密情報を心配することはできません（もちろん、すべてが適切にセットアップされている場合）。







結論として、メンテナンスと近代化の容易さについてのいくつかの言葉。 ウルトラブックのバッテリーは取り外しできず、下のパネルにはハッチがありません-富士通が計画しているように、ユーザーは中を覗いてはいけません。この職業は富士通のサービス部門または自分の会社のヘルプデスクに任せてください。 それでも、ケースを開けるのは難しくありません：プラスドライバーで数本のネジを外すと、ケースの底部パネルが取り外され、メンテナンスや交換が必要なすべてのもの（バッテリー、冷却システム、メモリ、ドライブまたはハードディスク、WiFi、3G）に完全にアクセスできます/ LTEモジュール。
