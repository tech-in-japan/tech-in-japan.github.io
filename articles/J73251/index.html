Browsera-新しいサイトテストツール

Browsera-これは別の無料のスクリーンショットサービスではないことをすぐに判断します。 このサービスを使用すると、レイアウトのクロスブラウザーの性質の自動テストを実行できます。つまり、HTMLコード、スタイル、スクリプトを読み取り、特定のブラウザーの特性に従ってレンダリングをシミュレートできます。 ただし、サービスは各ブラウザーのスクリーンショットも生成します。レイアウトデザイナーに送信するものがあります。







サイト、特定のページのアドレス、ブラウザーのリスト（IE6-8、FF2-3.5、およびSafari3.2-4が使用可能）、参照ブラウザー（そのインターフェースは比較のためのモデルと見なされます）、リンクを「ウォーク」する機能（深さが設定されます） ）、必要に応じて、サービスは保護されたセクションに入ることができます、はい-ログインできます。 テストが完了したらすぐに結果を待ちます-ページが更新され、プロセスで音楽を聴いたり、お茶を飲んだり、サービスページを完全に閉じたりすることができます-準備完了の電子メール通知が受信されます。







「 一意のレイアウトの違い 」セクションには、異なるブラウザーのページ上の位置が参照と異なる要素の数が表示されます。つまり、文字通り、他のブラウザーのウィンドウで双子の兄弟にカーソルを合わせるとブロック要素が色で強調表示されます。大型モニター-同時に任意のブロックの動作を確認できます。



「 一意のスクリプトエラー 」セクションには、各ブラウザに表示されるこのページのスクリプトエラーが表示されます。おそらく、Firefox 3.0.11（Ubuntu）のサービスの管理パネルで、エラーの数とそれ以上の説明は表示されません。壊れた、つまり、詳細はありません。

upd ：インターフェースの「 Java Script Errors 」タブは、ロードされると表示され、すぐに魔法のように消えました-彼らは、独自のサービスでインターフェースを実際にテストしませんでした。



残念ながら、テスト結果を公開することはできません-表示は登録ユーザーのみが利用できますが、いくつかの球状サイトのクロスブラウザテスト結果を見ることができます： opera.comメインページとwordpress.comセクションの1つのページのテスト結果。 巨人との精神的な近さを知ってうれしいです。彼らは、私のように、IE6との互換性のためにボルトを締めました！



今、この喜びのコストについて：はい、安くはありません！ ねえ、リラックス-アナキストのための無料プランがあります-あなたは無料で月25ページをテストできます。 これは、残念ながら、25ページのレイアウトを苦々しいまでなめることを意味するものではありません：1ページに25テスト、または25ページに1テスト、または-あなたに合った組み合わせで、月に25テスト以下です。 厚くはありませんが、フリーランスの学生にとっては、原則として、十分なはずです。 スタジオでは、1か月あたり45回500回のテスト、99回の2000回のテストを提供し、499ドルで制限がなくなりました-私は自由にテストしています！ 最後の関税パッケージは、数学と対立するスタジオマネージャー向けに設計されているようです。なぜなら、最大6か月の無制限のテストで、3台のコンピューターに相当する量を消費し、そのようなテストを「ライブ」で実行できるからです。 しかし、私はテレビで民主主義と多元主義が私たちに選択の自由をもたらしたと聞きました。それは使用しないのは愚かなことです。



それから、私はあなたにすべての快適な哺乳類と100％のクロスブラウザ互換性を望みます！
