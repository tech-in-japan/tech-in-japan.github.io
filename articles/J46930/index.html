バヤンICQ

あらゆる点でユーザーに適したICQクライアントを見つけるのはそれほど容易ではない場合があります...期待どおりに安定しないものもあれば、機能に欠けるものもあり、3つ目はインターフェースに合いません。 そして、これは現在のさまざまな関連プログラムでさえもです！ Symbian S60 3rd Editionを実行している多くのNokiaスマートフォンの1つなど、モバイルデバイス上のICQクライアントに関しては、状況はさらに悪化しています。 はい、このプラットフォームにはかなりの数のインスタントメッセンジャーが作成されましたが、個人的には、それぞれに重大な欠点があり、優秀さを追求するために、より多くの新しいクライアントを試すようになりました。

そのため、この記事では、Symbian 9用の比較的新しいICQクライアントについて説明します。これは、私の意見では、他のものより理想に近いものです。これはBayanICQです。







このクライアントの特別なところは何ですか？ まず第一に、それは残りと同じ機能をすべて実行します-ICQメッセージを送受信し、それを本当にうまくやります-私がそれを使用する限り（およそ9月の初め以来）、単一の気付かれた損失はありませんでしたAOLのサーバーでの直接的な障害とは関係のないメッセージ（着信と発信の両方）（すべてのクライアントでこの領域に不具合があった）。 さらに、プログラムには多くの便利な機能がありますが、その一部を以下に示します（すべてのモバイルICQクライアントで利用できない機能のみを強調しています）。

  • アニメーション笑顔のサポート
  • サーバーまたは宛先へのメッセージ配信に関するレポートを表示する
  • メッセージ履歴をテキストファイルにエクスポートするストレージと機能
  • 相互入力通知
  • スパム対策
  • 基本および追加のステータスの完全なセット
  • 直接ステータスのインストール
  • インスタレーション「Ball of joy」
  • サービスメッセージ（誰が、いつ出入りしたか、あなたのステータスを読むなどの情報を含む一種のログ）
  • プライベートリスト（サーバー上の可視性、不可視性、および点火のリスト、およびクライアントレベルのブロックリスト）を使用する
  • 選択：毎回連絡先リストを再度アップロードするか、ローカルコピーを使用します
  • 他の人の連絡先リストから自分自身を削除し、逆に、連絡先が自分自身を追加できるようにする
  • ユーザークライアントの定義
  • ローカル検索（連絡先リストのニックネームと番号による検索）
  • 高度な音響および振動設定
  • （クライアントが最小化されている場合を含む）すべてのウィンドウの一番上にステータス付きのウィンドウを表示します
  • 連絡先のステータスが変更されたとき、および誰かがあなたへの書き込みを開始したとき、およびメッセージが到着したとき（クライアントがコンピュータークライアントのポップアップウィンドウの方法で最小化されたときを含む）
  • バックライトの点滅によるメッセージ通知
  • ポートレートモードとランドスケープモードで起動
  • 「スキン」のサポート、および電話テーマをデザインとして設定する機能
  • カスタムフォントサイズ
  • ホットキーを構成する
  • 複数のプロファイルを使用した同時作業のサポート
プログラムの機能はバージョンからバージョンへと成長しますが、同時に、それがどれほど驚くべきものであっても、クライアントはかさばることなく、過度のメモリを消費し始めません。これは、少量のRAMを搭載したデバイスにとって重要です。 公式フォーラムからダウンロードできるクライアントのテストバージョンは非常に頻繁に公開され、通常、あらゆる種類のマイナーな改善と修正、および場合によってはいくつかの新機能が含まれています。 安定バージョンは、開発者のサイトのメインページから直接ダウンロードできます。頻繁に公開されることはありませんが、以前にリリースされたテストバージョンのすべての「トリック」が含まれ、徹底的にテストされています。

現時点では、最新の安定バージョンは0.097、テストバージョンは0.1rc3です。



最もよくある質問：



Q：スキンを上からトリミングしているのはなぜですか、許可のスキンをダウンロードしています...

A：連絡先メニューで、[フルスクリーンモード]ホットキーをクリックします。

Q：切断された連絡先が表示されないのはなぜですか？

A：「非表示/表示無効」ホットキーを押す必要があります

Q：バヤンを入れて、アニメーション絵文字を選択しましたが、再起動した後でも耕しません、どうすればいいですか？

A：少なくとも4 MBのメモリ予約（[設定]> [システム]）を選択し、Bayanを再起動する必要があります。

Q：連絡先のブロックは何をもたらしますか？

A：ご存じのとおり、ICQの無視リストはゴムではありません。 この問題の解決策を求めて、接触ロックが行われます。 連絡先の上にロックが表示されている場合、ユーザーのメッセージが表示されなくなり、ユーザーはxステータスを読み取ることができなくなります。

Q：バヤンの最新バージョンをインストールしましたが、接続せずにすぐに飛び出します...

A：パスに沿ってファイルを削除する必要があります！：/ System / Apps / bayanICQ / settings.datその後、再度ログインします。 再度入力しない場合は、設定で[アクセスポイントを使用]を[いいえ]に設定します。

Q：繰り返しますが、サインアップしないでください！ どうする

：ファイルを削除してください！：/ System / Apps / bayanICQ / users.usに戻り、Bayanに再度アクセスします。

Q：自動接続が設定されているときは接続しないでください。

A：settings.datファイルを削除します。このファイルは、プロファイルの設定、特に自動接続パラメーターを担当します。 削除したので、この点での入力に問題は発生しないはずです。

Q：インストール後、プログラムはアイコンをクリックしても反応せず、まったく起動しません。どうすればよいですか？

A：ファイルを削除する必要があります！：/システム/データ/ BayanICQ.heap







もちろん、BayanICQの可能性と利点をすべて理解するには、プログラムを自分で「感じる」必要があります。 短いレビューの一環として、このメッセンジャーのすべてのニュアンスについて話すことは困難です。 ただし、BayanICQは非常に優れたICQクライアントであり、常に連絡をとることができるという結論に達するには、私たちがなんとかして少しだけです。
