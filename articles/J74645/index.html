Internet Explorer用のGreasemonkey

便利なカスタムGreasemonkeyスクリプトは、Firefox、Opera、Chrome、およびSafariで動作します。 Internet Explorerはどうですか？ 実際、Windowsに組み込まれているブラウザーを使用しなければならない場合がよくありますが、作業しやすくすることは便利です。



すべてが非常にシンプルであることが判明しました。カスタムJavaScriptスクリプトのサポートは、よく知られているIE7Proアドオンにあります。 例としてYouTube HD Suiteを使用して、Internet Explorerにスクリプトをインストールすることを検討してください。







IE7Proの作成者は、独自のiescripts.orgを使用することを推奨していますが、コンテンツがかなり貧弱なので、 userscripts.orgにアクセスして 、好きなスクリプトを選択してください。 ユーザースクリプトのスクリプトファイルは、少し参考になる「script.user.js番号」の形式で表示されるため、保存する際にファイル名を「YouTube HD Suite.ieuser.js」に変更します（IE7Proでは.ieuser.js拡張子が重要です）。







ファイルは、\ Program Files \ IE7Pro \ userscriptsフォルダー（\ Program Files（x86）\ IE7Pro \ 64ビットWindowsの場合はuserscripts）に保存されます。 実際、プラグインがProgram Filesフォルダーを使用してユーザーデータを保存することは非常に奇妙であり、事前に別のフォルダーにアドオンをインストールしないと変更できません。 まあ。



IE7Proの設定に入り、[スクリプト]ページを選択し、[ユーザースクリプトを許可する]および新しくインストールされたスクリプトにチェックを入れます。 リストにない場合は、[すべて再読み込み]ボタンをクリックできます。







できた！ YouTubeを開いて確認します。
