携帯電話のポケット：コードレス電話ネットワーク用のSnom DECT機器

本日、オフィス内および本番環境でシームレスなDECTテレフォニーを作成するための5つのMシリーズデバイスを簡単に紹介し、社内のコミュニケーションを拡大します。 当社のエンジニアは、Snom M700ベースステーション、Snom M5リピーター、および機器レベルの異なる3台のコードレス電話（Snom M85、M65、M25）をテストしました。







会社のオフィスまたは生産現場の電話ネットワークは、有線ネットワークまたはWi-Fiに接続された固定電話で構成される必要はありません。 建物や会社の敷地内を絶えず移動する必要がある従業員も、常に連絡を取り合う必要があります。 このような状況では、Snomはコードレス電話およびその他の機器を製造して、DECTワイヤレス標準通信ネットワークを作成します。



Snom M700マルチバンド基地局



便利なワイヤレス接続を作成するには、まず、適切なインフラストラクチャが必要です。これは、Snom M700ベースステーションから構築できます。 このようなステーションの1つは、最大30台のDECTハンドセットに対応でき、合計で最大254台の基地局を1つのネットワークに結合でき、そこに最大1000台の無線デバイスを接続できます。 つまり、DECTネットワークは、大規模なオフィスや工業団地をカバーできます。























M700は、TLSおよびSRTPを介したDECT暗号化とSIPセキュリティをサポートしています。 同時に、ベースステーションを追加してネットワークを拡張しても、ライセンス料は必要ありません。 Snom通信システムは、BroadsoftおよびAsteriskのVoIPおよびPBXプロトコルと互換性があります。



DECTネットワークが複数のM700で構成されている場合、各ステーションは最大4つのブロードバンド会話または最大8つの標準会話を同時にサポートできます。 ステーションが1つの場合、それぞれ最大5および10の会話。



構成と管理は、Webインターフェースを介して実行されます。 Snom M700には、200人の加入者を収容できる電話帳があります。



ベースステーションの本体は、立てたり横になったり、壁に掛けたりできるように設計されています。 ステーションには、コンピューターネットワーク用のRJ45コネクタとハードウェアリセットボタンが装備されています。 全体寸法227 x 279 x 39 mm、重量320グラム。



















リピーターSnom M5



ネットワークのカバレッジエリアを増やす必要があるが、ハンドセットの数を増やす必要がない場合は、M5リピーターを使用できます。 加入者が基地局の受信エリア間を移動するときにシームレスな通信を提供し、BS間のワイヤレスブリッジとして機能し、ツイストペアケーブルを引く必要がなくなります。 2つの無指向性アンテナは、受信と送信のためにリピーター内に取り付けられ、ネットワーク内のデバイスの信頼できる接続を保証します。 リピーターは、118 x 129 x 33 mmの小さな箱で、重量はわずか126グラムです。 黒いケースには、2色のLEDインジケータとリセットボタンがあります。 M700と同様に、Snom M5はテーブルと壁の両方に配置できます。



















コードレス電話



すべてのMシリーズワイヤレスデバイスは、共通の設計機能を共有しています。 最も単純なモデルと最も高度なモデルの相互の外部の違いは最小限です。 ケースは、ソフトタッチで耐久性のあるプラスチックで作られており、わずかなテクスチャーの表面に、長時間の操作の痕跡がほとんど見えません。 電話は手にすべらず、携帯電話の形状は快適なグリップを提供し、長時間の会話中に不快感を引き起こしません。 軽量でコンパクトなサイズにより、携帯電話をポケットやバッグに入れて持ち運ぶことができます。携帯電話は他のアイテムと一緒に紛失することはありませんが、ポケットを大きく引っ張ることはありません。



祖母でさえ、Mシリーズの電話を扱うことができます。「iPhone以前の時代」のすべての携帯電話に馴染みのあるものです。上部には高品質のTFTマトリックスを備えた小さな画面があり、その下にはジョイパッド、コントロールボタン、クラシックコール/エンドボタン、さらには標準のユニットがありますテンキー。 キーのクリック感は素晴らしく、触覚的に完全に認識されます。



電話機のメニュー構造は、従来の携帯電話のメニュー構造に似ており、完全にロシア語に翻訳されています。 すべての電話には3.5 mmオーディオジャックがあり、補聴器と互換性があり、DECT暗号化をサポートしています。 ファームウェアは「無線で」更新され、サブスクライバのリストは企業PBXのベースステーションから取得されます。 すべての電話は大面積の高層ビルで使用でき、ネットワークカバレッジは追加のベースステーションとリピーターの助けを借りて簡単に拡張できます。



すべてのモデルに共通の機能は標準であり、そのようなデバイス（固定およびポータブルの両方）でよく知られています。これらは次のとおりです。 通話保留; 留守番電話; 発信者のリスト、不在およびダイヤルした加入者。 2行目; キーロック; 着信音のカスタマイズとキーボードの音声応答。 カスタマイズ可能なLEDイベント表示とネットワーク損失警告。 高度なモデルには、たとえば「デッドハンド」センサーが作動したときや人が転倒したときのアラームなど、独自の機能まで、より高度な機能があります。



ハンドセットM25



















M25はこのファミリーの中で最もシンプルです。 必要な基本機能はすべて備わっていますが、飾り気はありません。 チューブのサイズは134 x 42 x 23 mm、対角線が1.44インチのスクリーンです。 通常のAAAフォームファクターバッテリーを搭載し、最大75時間のスタンバイ時間または最大7時間の通話時間を提供します。







発信者/不在/ダイヤルのリストには、50個の数字のメモリがあります。 これはすべて、連絡を取り合う必要がある従業員が通常のオフィス環境で時折使用するのには十分ですが、多くの長時間の会話は行いません。































ハンドセットSnom M65



















オフィスを頻繁に旅行するだけでなく、電話で他の従業員と多くのコミュニケーションをとる必要がある人のためのより高度なモデル。 スピーカーとマイクは、HD Audio規格に従って高品質のサウンドを提供し、スピーカーフォン機能があります。 無線モジュールには、信号を送受信するための2つの個別のアンテナがあります。 内蔵の1100 mAhリチウムイオン電池を搭載し、スタンバイモードでは最大200〜250時間、通話モードでは最大17時間動作します。 電話帳には、100セルの加入者のローカルリストと一般的な企業リストの両方を含めることができます。 デバイスは、自動コール転送と3者間会議通話をサポートしています。



ディスプレイは大きく、その対角線は2インチで、その下には2つではなく3つのファンクションキーと、個別のハンズフリーキーと<メニューまたは電話帳ボタン？>があります。











LED表示は3色です。 ケースの横には音量ボタンがあり、サイレントモードを有効にします。







テンキーは8つのショートカットにプログラムできます。 屋内M65は、路上で最も近い基地局から最大50メートルの距離（最大300メートル）で信号をキャッチします。













寸法141 x 48 x 24 mm、重さ123グラムのこの小型ハンドセットは、本格的な卓上電話に比べてそれほど劣っていません。 Snom M65は、1か所に座っていない従業員や、フルサイズの電話でデスクトップを散らかしたくない従業員に推奨できます。







ハンドセットSnom M85























これはもはや単なるオフィスの「主力製品」ではなく、人間にとって危険を含む、生産での使用を目的とした高度に保護されたデバイスです。 衝撃、防塵、耐水性に関しては、電話機はIP65規格に準拠しています。唯一のクーラーは、afterれた後でも機能する完全に「オフロード」の電話機です。 このチューブは、倉庫や建設現場で使用されたときに実証されています。























このデバイスには、危険な職場で人に起こる可能性のある緊急事態でアラームを発するように設計されたいくつかの機能があります。 最も簡単なのは「パニックボタン」です​​。押すと、事前に選択した加入者を支援する信号が送信されます。 また、Snom M85には加速度計が装備されています。電話機が落下してしばらく持ち上げられない場合、または異常な位置にある場合、所定の番号への緊急コールも自動的に行われます。 この機能は、人が建設現場に落ちて意識を失ったり、高齢者だけが滑ったり、気分が悪くなったりする状況で役立ちます。 M85には、3番目の安全機構が装備されています。手首に装着されたハンドセットに特殊なストラップを接続できます。



したがって、この携帯電話を使用すると、より自信が持てます。何かが起こった場合、彼は助けを求めます。







豊富な機能と信頼できるケースのために、サイズと重量のわずかな増加-152 x 48 x 28 mmおよび175グラムを支払う必要がありました。 リチウムイオンバッテリはSnom M65と同じですが、スタンバイモードでは、電話機は200時間以下しか持続せず、通話時間は最大12時間です。















純粋に電話機能はM65とほぼ同じですが、さらに、電話にはBluetoothモジュールが装備されており、互換性のあるヘッドセットとアクセサリのリストが大幅に拡張されています。 そして、ケーキの上のチェリーはマルチカラーのLEDインジケータです。
