juick-el-Emacsのjuickクライアント

Twit.elに関する投稿で、EmacsからTwitterを作成する方法が説明されました。 この投稿では、マイクロブログサービスのためにクライアントを台無しにする方法をお伝えしたいと思います-GNU Emacsにお願いします。



GNU Emacs juickクライアントを拡張するには、juick-elを検討してください。 プロジェクトの公式ページは投稿の最後に表示されます。



拡張機能は次のようにインストールできます。2つの方法があります。1）公式サイトからプロジェクトファイルをダウンロードします。 2）自宅でgitリポジトリのローカルコピーを作成します。



$ git clone git：//github.com/mad/emacs-juick-el.git



次に、以下を.emacsファイルに追加します。



(add-to-list 'load-path "~/.emacs.d/plugins/juick-el/")

(require 'juick)

(setq juick-tmp-dir "~/.emacs.d/juick-avatars/")

(setq juick-tag-subscribed '("emacs" "linux" "juick" "GTD" "todo"))

(juick-auto-update t)







.emacsファイルに追加した内容を検討してください。

1）拡張ファイルがあるフォルダーへのパスを指定する

2）拡張機能の接続を示します

3）アバターを保存するフォルダーを指定します

4）購読するタグを指定する

5）指定されたタグでメッセージを受信する



投稿やコメントに応答するには、カーソルをメッセージ番号に移動してRETを押します。 ユーザーメッセージをサブスクライブするには、カーソルを名前に移動し、sを押してuのサブスクライブを解除します。
