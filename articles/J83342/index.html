Windows Phone 7-発表の前夜の噂の一部

マイクロソフトが2月15日にバルセロナのMWCでモバイルデバイス用の新しいOSを発表することを思い出させてください。これまでのところ、今日のような奇妙なリークがいくつかあります。 私が理解しているように、モバイルサイトの管理者の中には、将来のOSに関する詳細が記載された手紙をメールで受け取った人もいました。 このニュースはEngadgetで書かれ、 PPCGeeksとMobileTechWorldを引用しました 。 前者は、情報が真実に最も近く、一般的にすべてが非現実的にクールであると主張しています;後者は、ほとんどの事実があるので、私が翻訳する手紙の本文を控えめにコピーアンドペーストします。





  • WP7はWMCで発表されますが、デモインターフェイスのみが表示され、実際のデバイスは表示されません。
  • MIX WP7はいくつかの注意を払うでしょう
  • WP7は、Silverlight（ブラウザー外）とXNAをサポートします。XNAGameStudio 3.1でアプリケーション開発が行われます
  • Silverlightアプリケーションは、Expression Blend 3.xおよびVS2010で開発されます
  • Visual StudioとExpression BlendのモバイルバージョンとprogのPCバージョン用の無料アドオンがリリースされます
  • OSには、.NET Compact Frameworkに相当するものがSilverlightに組み込まれますが、SQLサポートはありません
  • システムには、LINQアクセスを備えた分離ROMがあります。
  • インターフェイスは「メトロ」と呼ばれるテーマに基づいており、Zune HDに似ていますが、完全に新しい開始画面があります
  • OSはマルチタスクをサポートせず、アプリケーションはバックグラウンドで「スリープ」状態になりますが、同時に環境が作成されるプッシュ通知もサポートされます。
  • .NET CFとの下位互換性はありませんが、一部のデータおよび機能アルゴリズム（いわゆる「ビジネスロジック」）は.NET CFから移植できます。
  • マイクロソフトは、2010年9月までに新しいデバイスが登場することを期待しています
  • 同社には独自のMSバックグラウンドはありませんが、より厳密な生産管理を約束するため、すべてのデバイスに3Dアクセラレーターがあり、そのドライバーはMS自身によって提供されます。 これにより、OTAの更新が可能になり、ODM / OEMの作業が楽になります。
  • マーケットプレイスにはトライアルアプリケーションシステムがあります
  • ODM / OEMはスタート画面を変更できません。つまり、SenseやTouch Floなどのインターフェイスを忘れる必要があります。
  • アナリストやジャーナリストを驚かせる予定のマイクロソフト
  • ブラウザはIE8の一部の機能を備えたIE7上に構築されており、iPhone 3Gのブラウザよりも便利で高速であり、同社は3GSとの競合を目指しています
  • FlashおよびSilverlightはブラウザーでサポートされていません
  • OSはXBox Liveと完全に統合されており、ゲームを購入する機能も含まれます。
  • Zuneを使用したマルチメディアソフトウェアが使用されます
  • OSは、サービス（マーケットプレイスなど）を通じてのみアプリケーションのインストールをサポートし、既存のタクシーのようなサードパーティのインストーラーはサポートしません


翻訳者のメモ=）

お気づきかもしれませんが、以前のいくつかの噂は繰り返され、真実かもしれません。 もちろん、すべてを真剣に受け止めるべきではありませんが、たとえその一部が実現したとしても、同社はモバイルデバイスに対して奇妙な見方をしています。 マルチタスクと多用途のアプリケーションソースは、VMの利点の1つでした。さらに、システムには依然としてソフトウェアの最も広範なデータベースがあります（ときどき曲がっていますが、動作しています）。

私自身、MWCを楽しみにしています。VMデバイスを変更するときが来たので、奇跡を待ち望んでいますが、それが起こらないのではないかと心配しています。

PS。 これまでのところ、市場に出回っているすべてのデバイスの中で、私はNokia N900に傾倒しています。
