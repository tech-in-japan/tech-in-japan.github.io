冬の家の空気の質





冬には、室内の空気が最悪です。 蒸れ、乾燥。 暖房は効きますが、凍結するため換気ができないことがよくあります。 家に小さな子供がいる場合は特にそうです。

バランスを保つ方法は？ これは可能ですか？



記事では、数値と計算を示します。 また、問題に対する彼自身の解決策と、このために組み立てられたデバイス。 猫の下で誰が気にしてください...



CO2含有量



Habréには、二酸化炭素とその基準と管理に関する多くの記事が書かれています。



したがって、それについてはあまり書きません。 CO2のレベルを標準に保つためには、十分に換気する必要があるとしか言えません。 しかし、その後、別の問題が急激に発生します。



湿度



乾燥した室内の空気は非常に悪いです。 人々は口が乾いていて、鼻にクラストがあります。 静電気が発生し、衣服が破裂し、ショックを受けます。



家の木製のものは乾燥して劣化します。



低温では、空気はほとんど水分を保持できません。 そして温度が低いほど悪い。



さまざまな温度での空気中の最大水分含有量のラベルを次に示します。







温度が-20、相対湿度が60％であるとします。



プレートを見て、検討します。 0.88g / m3 x 0.6 = 0.53gの水分が1m3の外気に含まれています。

今、私たちは窓を開けて、この船外空気の家を立ち上げました。 バッテリーはそれを25度に加熱しましたが、何が得られますか？



25度では、この空気の湿度は0.53 / 22.8 * 100％= 2.3％になります。



わあ！ これは非常に小さいです！ これは、空気中の最大含水率が25度22.8 g / m3であるためです（表を参照）。



冬の室内空気が非常に乾燥しているのはそのためです。 そして、路上が寒いほど乾燥します。

対策を講じない場合、湿度は7％に下がり、乾湿の温度計を備えた通常の水分計がロールオーバーします。



そして、もちろん、乾燥のすべての症状が現れます。



冬の室内の湿度は30〜40％、夏の湿度は40〜60％です。 同時に、人々は気分が良い。 ただし、最低30％の湿度レベルを確保するためにも、1立方メートルの街路空気に22.8 x 0.3-0.53 = 6.3 gの水を追加する必要があります。



そして、これはまだ非常に粗い計算です。 暖房中の空気の膨張を考慮しませんでした。 実際、さらに悪化します。



この不足している水はどこで入手できますか？ 集中的な水分補給が必要です！ 人は1時間に約40 gの水を吐き出しますが、これは非常にわずかです。



放映



人は呼吸し、絶望的に空気をだめにします。 そして、たくさんの空気！



この空気は換気に置き換える必要があります。 部屋を換気する必要性を正確に計算します。



穏やかな状態の人は、1日に800〜900gの二酸化炭素を吐き出します。 身体活動がある場合-最大1.5 kg。



部屋では、人は落ち着いていて、1日あたり800 gのCO2または800/27 = 1時間あたり33.3 gを吐き出していると仮定します。



部屋のCO2含有量の基準は最大1000ppmです

2000ppmを過ぎると、すでに非常に蒸れています。

冬に1000ppmの基準を維持することは実際には現実的ではありません。 なんで？



計算してみましょう：

25度の空気密度1.18 kg / m3

船外の空気が新鮮で、CO2含有量が400ppmであると仮定します。

まあ、そのような理想的なケース。



最大1000ppmを摂取すると、1000-400 = 600ppmを「吐き出す」ことができます

または1180 x 600 / 1,000,000 = 0.7g / m3 CO2



最大2000ppmの場合、2000-400 = 1600

または1180 x 1600 / 1,000,000 = 1.88g / m3 CO2



以前、人は1時間に33.3 gのCO2を吐き出すと計算しました。 ここで、1時間で人が絶望的に​​台無しにする空気の立方体を計算してみましょう。



最大値が1000ppmの場合、33.3 / 0.7 = 47.6 m3



最大が2000ppmの場合、33.3 / 1.88 = 17.7 m3



20m2の通常の部屋と2.5mの高さの容積は50m3になります



つまり、1人が呼吸する普通の部屋で1000ppmのレベルを維持するには、この部屋のほぼ全容積を1時間ごとに新鮮な空気で置き換える必要があります。 そして、他の基準に従って空気を状態に戻すために、この空気をすべて加熱して湿らせる必要があります。



まあ。 電池が完全に揚げられているとしましょう。 道路から新たに流入する空気の量を標準化するために蒸発させる必要がある水量を計算します。

人は1時間に40 gの水分を吐き出します。 すぐにこれを考慮してください。



最大が1000ppmの場合、47.6 x 6.3-40 = 260 g



最大が2000ppmの場合、17.7 x 6.3-40 = 71 g



つまり、このような換気で湿度を上げるには、最初の場合は1日あたり6.24リットル、2番目の場合は1.7リットルの水を費やす必要があります。 これはすべての呼吸する人のためです！

2000/1000 = 2 6.24 / 1.7 = 3.67



依存関係は線形ではありません。 空気が新鮮であればあるほど、より多くの水を消費する必要があります。 まあ、加熱も。 そして、これは湿度を最小限に抑えるためです！



1人あたり1日あたり6.24リットルの水が多すぎると思います。 したがって、冬には蒸れに耐えなければなりません。 何もする必要はありません。

25度の温度で1500 ppm以下のCO2レベルと30％の湿度を維持しようとしています。 1500はたくさんありますが、それ以前はさらに悪化していました。 何が起こるか見てみましょう。
