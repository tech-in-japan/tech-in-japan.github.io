Android上のWindows 3.1

人間の創意工夫に制限はありません。

今回は、Android OSを実行しているデバイス上でWindows 3.1（およびそれ以上）を実行する方法を見つけました。







システムはオリジナルでインストールされ、完全に機能します。唯一の大きなマイナスは、かなり複雑なコントロールです。



したがって、これには次のものが必要です。

  1. Androidバージョン2.2以降を搭載したデバイス
  2. ADosBoxアプリ
  3. Windowsインストールファイル
  4. SDカードまたはデバイス自体の空き容量（20 mb（ これは問題です.. ））
  5. ストレートアーム




私たちは始めます：



  1. マーケットからaDosBoxをインストールする
  2. Windows 3.1インストールファイルをwinsetupfilesフォルダーのメモリカードに抽出します。
  3. メモリカードに空のフォルダーを作成する-Windows
  4. aDosBoxを実行して、コマンドを入力します。

    • マウントd / mnt / sdcard / winsetupfiles
    • マウントC / MNT / SDカード/ Windows
    • D：
    • セットアップ


  5. インストールが開始されます。
  6. すべてのインストール手順を実行します（Khabrovsk-geeksはWin 3.1をインストールできるはずです）
  7. 利益！




次回aDosBoxを起動すると、コンソールが再び開き、コマンドを入力する必要があります。

  1. マウントC / MNT / SDカード/ Windows
  2. c：
  3. windows \ win.com




または、これら3行を/mnt/sdcard/app-data/org.hystudio.android.dosbox/.dosbox/dosbox-0.74.confファイルの最後に追加して、aDosBoxの起動時にWindowsを自動的に起動します。



そして、インストールとデスクトップの甘いスクリーンショットのために：



インストールしますか？ Enterをクリックしてください！





見えますか？ その後、高速インストール。





プロセスが開始されました！









美しいGUI ...





テキストエディタをインストールしています。何らかの方法でメッセージを書き込む必要があります:)









掃海艇!!! ゲームなしではどうなるのか...









プリンターをインストールしますか？ どっち？ いいえ、できません。





Win 3.1の学習？ おかげで、できることはわかっています。





Windowsを再起動するか、MS-DOSを続行しますか？ 私たちの目標はWindowsですので、再起動してください！





数秒待っています...





ネイティブの子供時代のスクリーンセーバー！





そして、これが目的のデスクトップです！
