幅/高さのない画像

ページで使用する画像については、その長さと幅を常に示すことをお勧めします。 多くの場合、このような画像は、サイズを指定しないと、読み込み中にレイアウトが揺れる可能性があります。

開発中に、サイズを指定せずに写真をスキップした場所がより明確になるように、 37Signalsのメンバーは、小さくても非常に便利なスニペットを提案しました。 



img:not([width]):not([height]) { border: 2px solid red !important; }


これで、サイズを指定しなかったすべての画像が赤枠で強調表示され、それらをスキップするのがはるかに難しくなります。
