オープンソースネットワーキングのミートアップ-Yandex.Cloud＃3.2019





5月20日に、オープンソースネットワーキングのトピックに関心のあるすべての人を、今年のOSN Meetupシリーズの3番目のイベントに招待します。 イベントオーガナイザー：Yandex.Cloudとロシアのオープンソースネットワーキングコミュニティ。



オープンソースネットワーキングユーザーグループモスクワについて



Open Source Networking User Group （OSN User Group Moscow）は、DPDK、FD.io、ONAP、OpenDaylight、OPNFV、PNDA、SNASなどのオープンソースソリューションでネットワークインフラストラクチャを変更する方法を議論する情熱的な人々のコミュニティです。他のソリューション。 このグループは、対面の問題について議論し、アイデアを共有し、連携して、ソフトウェア定義ネットワーク（SDN）、ネットワーク機能の仮想化（NFV）、管理とオーケストレーション（MANO）、クラウドコンピューティング、データ分析、基盤となるネットワークインフラストラクチャの改善。
