JavaScriptのCのような構造

この記事では、JavaScriptコードでC / C ++の構造と配列のバイナリデータを簡単に操作できる小さなライブラリについて説明します。



ユニバーサルインターフェースに関する記事で、Webソケットを使用する利点について説明しました。 私にとって開発プロセスで唯一不便だったのは、JavaScriptとC / C ++コード間のデータ交換でした。 そこで最初に思いついたのは、Qtで非常に簡単に生成および解釈できるJSONオブジェクトを使用することでした。 ただし、実際には、これはあまり便利なソリューションではありませんでした。 C / C ++のデータ構造を常にJSONオブジェクトにラップしてから、それらをデプロイする必要があります。 同意する、最も楽しいものではありません。



しばらくして、私は夢を見始めました。 ただし、JSONオブジェクトや他の身体の動きなしで、構造をJavaScriptに直接、またはその逆に直接転送できる必要があります。 インターネットで少し検索した後、Mozillaビジ​​ネスに出会いました。 このライブラリには非常に多くの制限があるため、使用を拒否しました。



JavaScriptの型付き配列とWebソケットに精通した後、「ドキュメントとサンプル」を使用して夢のライブラリを作成することにしました。 おそらくご想像のとおり、このライブラリを書いたので、専門家からのレビューを聞きたいです。



恵みのライブラリに慣れるために、 githubで尋ねます。
