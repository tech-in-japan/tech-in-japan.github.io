「クラウドフラッシュ」-選択の苦痛

こんにちは、Habr！



次のすべて-私の意見と私の気持ちをすぐに予約してください。 あなたの意見が私の意見と異なる場合-コメント！



約1年半前に、私はDropBoxを使い始めました。 あまりアクティブではありませんが、日常のタスクには十分です。 ゆっくりと、紹介のために、すでに16GBのスペースがありました。 アカウントは無料です。

最近、YandexがYandex.Diskをリリースし、Googleドライブを開始する準備ができました。私はしばらくの間、すべての一般的な「クラウドフラッシュドライブ」を使用して、私にとって最も便利なものを決定しました。 進捗状況についていく。



興味があれば-猫の下でお願いします。





タスク



ソフトウェア、システム、IMHOの選択​​は、問題のステートメントから始める必要があります。 私はそれらを持っています：

  • 500 MBを超えるクラウドロケーション
  • 非抑制クライアント
  • WEBアクセス
  • Webdav \ ftp \ ...クライアントを配置しないようにサポートすることをお勧めします
  • 高速同期\ファイルアクセス速度
  • クライアントをインストールする機能\仕事からファイルへの非Webアクセス
  • iOS用クライアント


最後のアイテムの詳細：

職場では、インターネットは完全に開いていますが、プロキシを使用してポートを切断しています。 また、明らかに、トラフィックはタイプによってフィルタリングされます（ポート80でsshサーバーに接続できません）。 Webを使用して「USBフラッシュドライブ」にアクセスすることは、必ずしも便利で高速ではありません。



可能な解決策



次のオプションを検討しました。

  • DropBox（使い慣れた、ほぼ完璧）
  • I.ドライブ
  • Microsoft SkyDrive
  • Box.com


最も人気がなかったため、残りはテストしませんでした。

iCloudの場合は別に-私が同じ理由はわかりません。ディスクは彼のキラーと呼ばれます。 iCloudはクラウドフラッシュドライブではなく、Appleデバイスを同期するためのクラウドです。 さまざまなファイルをそこに詰め込むことはできません。 したがって、私は常にAppleのクラウドを使用していますが、この資料の文脈では考慮していません。

だから、順番に：



Dropbox



私が望むすべてを知っていますが、プロキシでは動作しません。 クライアントにログインしようとすると、アクセスできないというエラーが表示されます。 ウェブを経由しなければなりません。 一番下の行-ダウンロードフォルダーには、「ファイル名（X）.docx」のような多くのファイルがあります。

長所


  • 有病率
  • 無料のストレージ容量を増やすための永続的なプロモーション
  • iPhone / iPad用の優れたクライアント


短所


  • 一部のプロキシとは友好的ではありません。 特にポートをトリミングする場合。 私はまさにそうです
  • 多くの場合、同期の終了を長時間待っていました（場合によっては、新しくインストールしたマシンで、100〜200個のファイルをダウンロードするのに数十分かかることがありました）




Microsoft SkyDrive



興味のために、私はそれを設定しました。 美しい数の「25ギガバイト」を購入しました。 とてもがっかりしました。 非常に思慮深いクラウド：モバイルクライアントからアップロードできるのは写真/動画のみです。 iFilesに接続できません。 さらに、モバイルクライアントから何かを削除しようとすると、定期的に削除エラーが発生します。

長所


  • たくさんのスペース
  • 有名な開発者
  • オフィススイートに接続します
  • webdavが可能で、ネットワークドライブとして接続します。


短所


  • デスクトップクライアントなし
  • webdavによる非常にhemoのディスクマウント手順
  • ぼやけたモバイルクライアント




Box.com



登録後すぐに使用を開始できませんでした。出張に行きました。 到着すると、このサービスからの大量のスパムに非常に驚かされます。 破壊されたナフィグ。



Yandex.Disk



今私のメインの「クラウドフラッシュドライブ」。 スマートに動作し、webdav、素晴らしいクライアント（DropBoxでなめられますが、もっと馴染みがあります）、通常のWebクライアントを実行できます。 そして、I。職場でネットワークドライブとして接続されたディスク（ここではwebdav'aの楽しみ）。 最後の点は、私にとっては決定的になったようです。

長所


  • すべて同じwebdav
  • すべてが非常に高速に動作します。
  • 素晴らしいiOSクライアントを持っています


短所


  • 私は2週間招待を待っていました。
  • Yandexからメールを取得する必要があります-他のサービスに接続することはできません。


Ya。Diskを選択しました。 たぶん、Googleドライブはもっと面白いものを提供するでしょう。 それから多分私はそれに進みます。 幸いなことに、DropBoxからYa.Diskに移動するには3分かかりました。



ご清聴ありがとうございました！
