Chrome向けEvernote Web Clipper 6.2アップデート





本日、拡張機能を更新して、ブラウザからEvernoteに情報をすばやくコピーし、このプロセスをより簡単で便利なものにしました。



Web Clipperをダウンロード»







要素の思慮深い配置



Web Clipperが占有する画面領域のかなりの部分をリリースし、それぞれの場合に本当に必要な機能のみを残しました。 新しいコンパクトなデザインは、どの画面サイズにも適しています。







干渉なしにコピー

保存ボタンがより見やすくなり、EnterキーまたはReturnキーを押すと機能するようになったため、余分なクリックなしでメモを送信できます。 便利な機能のグループ化により、必要な情報をすばやくコピーしてビジネスに戻ることができます。







クイック編集

スクリーンショットを撮るときにメモを作成するためのツールが非表示になりました。編集する領域を選択すると、それらが表示されます。







コピーされたWebフラグメントのその他のオプション

情報をコピーした後、特定の時間までにこのデータを使用してメモが必要な場合にリマインダーを配置できるツールバーが表示されます。また、そのメモを友人や同僚と共有できます。







高度な機能



Web Clipperを快適に使用するためにWeb Clipperの機能を専門的に理解する必要はありませんが、6.2では、エクステンションの経験豊富なファンが好む多くの頻繁に要求される機能を追加しました。



メモ帳の選択の改善

新しいドロップダウンメニューでノートブックとタグの検索が高速化され、探している名前を入力すると項目がフィルタリングされます。



マーカー

記事、ページ全体、またはテキストを簡略化された形式でコピーすると、マーカーが自動的に選択され、時間を無駄にすることなく主要なものをすばやく強調表示します。



デフォルト設定

拡張機能を使用する際の個々の設定に関係なく、新しいWeb Clipperには、Webページをコピーするプロセスを自分用にカスタマイズできる多数の設定があります。 コピーされる情報に関連する同様のメモを非表示または表示するか、設定で拡張機能を自動的に閉じるオプションを選択して、コピー後のアクションでパネルを非表示にすることができます。
