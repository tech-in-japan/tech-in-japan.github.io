世界学生プログラミング選手権がロシアで初めて開催されます





昨年のSPbNIU-ITMOチームの昨年の勝利の後、世界チームプログラミングチャンピオンシップ（ACM ICPC）の決勝がロシアで初めて開催されます。



第37回ACMインターナショナルカレッジプログラミングコンテスト2013学生プログラミング世界プログラミング選手権の決勝は6月30日から7月4日まで開催されます。 ホストパーティー-サンクトペテルブルク国立情報技術大学、力学および光学（SPbNIU-ITMO）。 このイベントはIBMが主催しています。





12日以内に、サンクトペテルブルクのユビレイニースポーツパレスは最高の学生スポーツプログラミングチームを迎えます。 最終ラウンドは、7月3日9:30から15:00に開催されます。



2013年には、29,479人が世界プログラミング選手権の予選に参加しました。 チーム（各3名）は、91か国から2322の大学を代表しました。 過去15年間で、ACM ICPCメンバーの数は12倍に増えました。







コンテストは、大学のコンピューター部門のチームメンバーの地元での選択から始まります。 約300,000人の学生が選択に参加しています。 ACM ICPCは、世界最大かつ最も権威のあるプログラミング競技会です。



現在のフィナーレでは、120の参加チームが5時間のマラソンを行い、その間に難易度の高い8つ以上のタスクを解決する必要があります。 プレイエリア内の飲食物や電子機器は禁止されています。



問題を解決するには、アルゴリズムの深い知識を示す必要があります。



  • プログラミング言語：Java 1.7.0_09、C / C ++ GCC 4.6.3（Ubuntu / Linaro 4.6.3-1ubuntu5）。
  • 開発環境：Eclipse 4.2.1、Eclipse 4.2.1でのCDT 8.1.1。


最終スケジュールはこちらをご覧ください 。



SPbNIU-ITMOは、最終コンペを開催する名誉を受けました。 これは、スポーツプログラミングチーム向けに専門的なトレーニングを実施している世界でも数少ない大学の1つです。 最高のアスリートは、ロシア全土、さらには他の国からもやって来ます。 たとえば、SPbNIU-ITMOが、ベラルーシのCodeforcesおよびTopCoder Gennady Korotkevichの評価のリーダーの1つである、学童の間でスポーツプログラミングの世界チャンピオンに3回参加することを決めました。 そのため、SPbNIU-ITMOにはおそらくもっと多くの勝利があります。



SPbNIU ITMOの情報技術およびプログラミング学部長のVladimir Parfyonovは、グローバルな教育産業はIT専門家に対する既存の需要に対応していないと指摘しました。 「開発者の負荷は非常に大きいため、若者だけが処理できます」と彼は言います。 ロシアでは、5年生から始めて、すべての数学能力のある人が登録されています。 しかし、人口統計上の状況は、2012年に優秀な学校の卒業生が500人を超えなかったことであり、これはわずか400人です。
