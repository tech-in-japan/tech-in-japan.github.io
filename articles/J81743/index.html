iPhoneをGoogleと同期する

かなり長い時間がこの記事のアイデアを育み、ついにそれを書く時間を見つけました。

iPhoneが気に入ったのは、 Googleサービスとの同期です。 今では、メール（仕事用または個人用）のない電話、カレンダー、またはいつでも最新情報の連絡先を使用する方法を想像することさえできません。 Googleのレポートによると、メールの同期はiPhoneでのみ機能するようになりました。







携帯電話を通常のGoogleアカウントではなくGoogle Appsに結び付けましたが、これにより何も変更されず、いずれの場合もすべて同じように構成されます。

すべてが機能するためには、OS 3.0以降を搭載したiPhoneが必要です。また、Google Appsユーザーの場合、管理者がGoogle Sync設定を有効にする必要があります。







開始するには、電話でメール、独自のカレンダー、連絡先を同期します。

  1. iPhoneの設定->メール、連絡先、カレンダー->アカウントの追加を開きます。
  2. Microsoft Exchangeの選択
  3. 電子メールフィールドに、 メールの完全なアドレスを入力してください
  4. 「 証明書を確認できません 」というアラートが表示される場合がありますが、これは正常です
  5. ドメインは空白のまま
  6. ユーザー名フィールドに完全な郵送先住所を入力します。ログインではなく郵送先住所であることに注意してください
  7. パスワードフィールドにパスワードを入力してください
  8. 「 証明書を検証できません 」と表示されたら、「 承認 」をクリックします
  9. [ サーバー]フィールドにm.google.comと入力します
  10. 次に、次を選択
  11. 次に、同期するサービスを選択する必要があります。 なぜなら すべてを同期したい、3つの要素を含めます
  12. 完了、メール、カレンダー、連絡先が同期されました




2番目のステップは、同期をセットアップすることです。 これは、私のように、複数のメールボックス（個人用メールと仕事用メール）と複数のカレンダーがある場合に当てはまります。

Googleアカウントから同期するカレンダーを設定するには、次を行う必要があります。

  1. http://m.google.com/syncで Safariを開きます
  2. システムにログインする
  3. 同期デバイスを選択してください
  4. 同期するカレンダーをマークする


Google Appsユーザーの場合、以下を行う必要があります。

  1. http://m.google.com/syncで Safariを開きます
  2. ページの下部にある[ Google Appsユーザー]をクリックします
  3. ドメインを入力してください
  4. 同期をクリックします
  5. システムにログインする
  6. 同期するカレンダーをマークする




ロシア語版のサイトで作業しているときにグリッチに気づきました。何か問題が解決しない場合は、英語版に切り替えることをお勧めします。



セットアップの最後のステップは、追加のボックスを追加することです。 なぜなら Microsoft Exchangeアカウントは1つしか存在できません。構成プロセスはわずかに異なります。



  1. iPhoneの設定->メール、連絡先、カレンダー->アカウントの追加を開きます。
  2. Gmailを選択する
  3. [ 名前]フィールドに名前を入力します
  4. [ アドレス]フィールドに、メールボックスの完全なアドレスを入力します。
  5. [ パスワード]フィールドにパスワードを入力します
  6. 保存をクリックします


これにより、必要な数のボックスを追加できます。











最後に、アプリケーションをお勧めします。 iPhoneを理想的なモバイルツールにするには、 CydiaリポジトリのJailBreak電話でのみ使用可能なIntelliScreenを インストールすることをお勧めします。 このプログラムは、カレンダーから携帯電話のロック解除画面に新しい文字と今後のイベントを表示します。







もちろん、同期の主な利点は、最新の情報にアクセスできることですが、他にも便利な機能があります。 今、あなたは電話を失うことを恐れる必要はありません、そしてそれですべての連絡先。 すべての電話を新しいデバイスに復元するのにかかる時間は10分以内です。
