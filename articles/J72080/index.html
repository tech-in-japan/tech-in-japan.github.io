Mozilla Firefox + Aero

少し前のHabréで、FireFoxの将来のバージョンと、Aeroエフェクトのサポートを含む更新された外観について書いています。





しかし、多くの人は、そのような外観がすでにリリースされたバージョンに実装できることを知りません。

オプション1：

ブラウザパネルのAeroサポートを含むStrataGlassアドオンと 、このために特別に強化されたStrata40テーマをインストールします（スクリーンショットと同じ）。

メニューバーを非表示にするには、 HideMenuBarまたはより柔軟なパーソナルメニューを使用できます（ツールボタンなどを追加できます）。





オプション2： （timon_skから）

Windows 7向けClassic Remix + すべてのGlasserアドオン + パーソナルメニュー





オプション3 （デフォルトのブラウザーテーマを使いこなすため）：（Nexuxから）

新しいGlasser-このアドオンは、より多くのテーマをサポートします（ただし、残念なことに、ほとんどのテーマにはパネルの背景があるため、Aeroはその下にありません）。 標準テーマでうまく機能します。 （スクリーンショットは、 個人メニューをオンにした状態を示しています ）





オプション4： （Fiery_Iceから）

AeroFX-テーマは（標準でも）あまり「調和して」表示されませんが、他のアドオンとの競合はそれほど多くありません。





PSもちろん、Windows Vista / 7でのみ動作します。
