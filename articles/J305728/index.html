視覚障害者のためのコンピュータービジョン。 Intel Edisonアプリケーション





Intelの従業員であるPaul Guermonprezは、ビジネスと喜びを組み合わせることにしました。彼は、開発者の経験を使用して、視覚障害のある人々をコンピュータービジョンで支援するシステムのプロトタイプを作成しました。 現時点では、システムは次のことができます。

  • バーコードの検索と読み取り、音声での説明の発音、
  • オブジェクトの色を決定して発音し、
  • 「見た」色と一致する色を選択し、
  • 正面の顔、相対的な位置、顔までの距離を決定し、この情報をステレオで発声します。


カットの下-ピボットヘッドSMARTメガネに関する情報、システムのインストール手順、およびその機能を示すビデオ。



ピボットヘッドSMART



Pivotheadは2013年にクラウドファンディングのスタートアップとしてスタートしました。 チームはその目標を達成しました。発表されたものの一部は生産に至らなかったものの、主なものは、コンピュータービジョンシステムと拡張現実のための手頃な価格の基盤を実装しました。







基本モジュールには、HD写真とビデオをキャプチャする8メガピクセルのカメラ、8/16/32 GBメモリ、Bluetooth 4.0モジュール、および約400 mAhの容量の内蔵バッテリーが装備されています。 最も単純な構成のコストは200ドルです。 追加のモジュールがイヤフォンに接続されています：記録時間を40分延長する強化されたFuel SMARTMODバッテリー、または前述のLive Mod Setは、Intel Edisonボードにすぎず、取り外した場合にはっきりと表示されます写真、保護カバー。



EdisonはWi-Fiをインターフェイスのリストに追加しますが、最も重要なことは、多くのツール、既製のライブラリ、およびソフトウェアツールが用意されているため、開発者の創造性に大きな分野を開くことです。 これがポールが利用したことです。



インストール手順



Pivothead SMARTメガネをお持ちで、Paulが作成した機能を試してみたい場合は、Paulのインストール手順に従ってください。

  1. Edisonファームウェアをバージョン3に更新します
  2. エジソンインターネットで設定
  3. repo.opkg.netupgradeでパッケージをインストールしますが、すべてではありません。そうしないと、カーネルは正常に動作しません
  4. パッケージのインストール： opkg install fswebcam nano espeak ffmpeg-x264-presets gps-utils htop git lighttpd ofono opencv opencv-dev opencv-staticdev opencv-apps opencv-dev opencv-samples opencv-samples-dev python-opencv python-pip python-numpy zbar mjpg-streamer gstreamer1.0-plugins-good-interleave gstreamer1.0-plugins-good-audiofx
  5. パッケージのインストール： npm install -g fs sleep tinycolor2 array-unique striptags color-namer color-scheme onecolor util request shelljs-nodecli linux-input-device canvas okrabyte ocra.js
  6. ソースを解凍します
  7. ポート81でWebサーバーを構成し、ファイル/etc/lighttpd.confを編集してルートディレクトリ/ home / root / www /を作成します。
  8. カメラをテストします： fswebcam -d / dev / video0 /home/root/www/shot.pngブラウザーに入力しますhttp：// EDISON_IP：81 / shot.png


pivothead-intelに移動し、コマンドラインからデモの1つを実行します ： node demo_barcode.js 。
