Python 3をサポートするOpenCV 3.0リリース

コンピュータビジョンアルゴリズム、画像処理、および汎用のオープンソース数値アルゴリズムの一般的なライブラリであるOpenCV（Open Computer Vision Library） の 3番目のバージョンがリリースされました。



このリリースには、github経由で送信された約1,500個のパッチが含まれています。 OpenCVは大量のIntel Intellectual Property（IPPCV）を使用しており、同社はこれを無料でライセンス料なしで商用および非営利目的でライブラリユーザーに引き渡しました。 OpenCLを使用したハードウェアアクセラレーションは、T-API（トランスペアレントAPI）を通じて開発者が利用できるようになりました。 さらに、OpenCVの3番目のバージョンには膨大な数の新機能、パフォーマンス、および安定性の向上が追加されています 。詳細については、 こちらをご覧ください 。



これは、Python 3を完全にサポートするライブラリの最初の安定バージョンです。WindowsおよびPython 3.4の場合、バイナリはlfd.uci.eduで既にダウンロードできます 。 モジュールの名前はまだcv2です。
