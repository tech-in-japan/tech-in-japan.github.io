DELL EMC Networking N1100スイッチの概要

2017年8月以降、Dell EMC Networking N1100スイッチはロシアで注文できるようになりました。 それらは「ホットな」市場ニッチの製品として位置付けられており、それはwork意的に「働く企業の馬」と呼ぶことができます。 このカテゴリでは、製品を選択する際に、「価格」パラメータに重点​​を置き、「価格/品質」などの顧客の評価が優先されます。







しかし、このカテゴリでDell EMCが提供するものを詳しく見てみましょう。



そもそも、2年前、Dellはシリーズ間の「中間製品」に向けてすでに一歩進んでいたという事実に注目します。 2015年、N1500シリーズスイッチは同じバナーでリリースされました。「XシリーズとNシリーズ間の中間スイッチ」です。 そして今、新製品が「さらにシンプルで安価な」コーポレートNシリーズ内でリリースされました。



XシリーズとDELL EMC Networking Nシリーズスイッチの違い



Dell EMC Networking製品ポートフォリオには、小規模組織（最大200ユーザー）またはホテル、会議室、家庭、小売店のローカルタスクを解決するために配置されたXシリーズのスイッチと、企業のために配置されたNシリーズのスイッチが含まれますネットワーク（100から1000ユーザーまで）。 これらのカテゴリーの交差点にあるのは、「働く企業馬」と呼ばれるまさに製品のカテゴリーであり、SMBレベルの価格を意味しますが、「企業セクター」の機能およびサービス要素です。



上記のように、N1100シリーズスイッチはXシリーズとNシリーズの間の中間リンクですが、正式にはNシリーズを指します。 これらのスイッチが「法人」セグメントから持っているものを見てみましょう。これは「SME」/ SMBのセグメントにはありません。



  1. コマンドラインインターフェース（CLI） すべてのXシリーズスイッチには、グラフィカルユーザーインターフェイス（GUI）があります。 ネットワーク製品のDell EMC Networking部門がGUIと非常に緊密に連携していたことは注目に値します。専門家はこのインターフェイスを機能的で明確で視覚的に拒否しないと評価します。 Xシリーズスイッチでは、開発者はGUIを介して基本的な制御を行い、CLIオプションには限られたコマンドセットを残しました。 Dell EMC Networking製品の本格的なコマンドラインオプションは、Nシリーズから始まります。 CLIのコマンドの90％がCisco IOS構文と一致することを追加します。 このように、Cisco IOSの経験がある人は、Dell EMCネットワークスイッチに適応できます。

  2. スタッキング機能 。 Xシリーズスイッチにはこのような機能がありませんが、Nシリーズスイッチには標準のビルトインなどの機能があります。 1つの例外があります-8ポートのN1100シリーズのローエンドスイッチ。 N1100シリーズの残りのスイッチには、40 Gb / sのスタッキングバスを備えた最大4つのデバイス用のスタッキングオプションが組み込まれています。

  3. ProSupport NBD保証 。 Xシリーズスイッチはロシアに出荷され、基本的な生涯保証と、Dell EMC ProSupportブランドの保証を購入するオプションが付いています。 N1100シリーズスイッチには、基本的な生涯保証と3年間のProSupport NBDブランドのアフターサービスの両方が付いています。 以下の保証の機能と種類についてお読みください。



DELL EMC Networking N1100シリーズスイッチ



N1100シリーズスイッチはすべて、レイヤー2スイッチです。 現在のN1100スイッチファミリには、8個、24個、および48個の10/100 / 1000BASE-Tポートを備えた6つのモデルがあります。 ラインナップから、3つのスイッチにはPoEサポートがあり（モデル番号に「P」という文字があります）、他の3つのスイッチにはPoEはありませんが、サイレント（ファンなし）設計になっています（「T」という文字はモデル記事にあります）。



N1100スイッチには、Dell EMC Networking OS 6オペレーティングシステムが搭載されています。独自のオペレーティングシステムに加えて、N1100スイッチには、長時間実行されるDell EMC ONIE（オープンネットワークインストール環境）インストール機能があります。 ただし、執筆時点では、Dell EMCはN1100シリーズスイッチでサポートされているサードパーティのオペレーティングシステムのリストをまだ発表していません。



2つの10GE SFP +ポートを使用するN1124T / PおよびN1148T / Pモデルでは、40 Gb / sバスによるスタッキングサポートが利用できます（スタック内の1つのモデルラインから最大4つのデバイス）。 したがって、スタッキング自体は、特別なケーブルやモジュールを使用せずに標準インターフェースを介して実行されます。 スタックの機会の概要については、以下の表を参照してください。

Dell EMC Networkingの「ライフタイム保証」という用語は、お客様がスイッチを使用している間、ハードウェア障害が発生した場合、Dell EMCテクニカルサポートがスイッチ全体を交換する準備ができていることを意味します（ただし、障害は製造条件によるものであり、動作条件によるものではありません）。



交換は、故障したスイッチが取り除かれている間に、新しいスイッチを操作場所に届けることによって行われます。 製造業者が顧客が使用するスイッチモデルのサービスを停止した場合（つまり、販売からのスイッチの削除（販売終了、EoS）、生産ベースとコンポーネントがサポートするレジストリからのモデルの削除、ファームウェアの更新など（終了-of-Life、EoL））、-顧客には、特性が類似するモデルが提供されます（メーカーが製造した製品の旧世代と新世代の間での直接交換の表に基づく）。
