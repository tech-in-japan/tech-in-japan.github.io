恐怖に大きな目がある、または教室でMAC OSとWin XPを使用してHDDを複製する

すべての人に良い一日を！

この記事では、単一のmacminiでMAC OSとWin XPを「友達に」し、この連合を中等特別教育機関（大学）のコンピューターサイエンスクラスの12台の同様のコンピューターに拡張する方法について説明します。



「歌だけがそれについてではない...」


正直に言うと、この記事は問題の技術的な側面を説明するのではなく、インターネット上のこの部分ですべてを見つけてまとめることができるように考案されましたが、教育機関の初心者のシステム管理者、エニキーまたはコンピューターサイエンスの教師のために私の場合、彼らは教育プロセスから気を取られることなく、すべてを1か所で見つけました。 さて、もう一度、教育機関のIT分野の問題のいくつかを強調します。



背景


教育では、私は...生物学者です。 そして、私は自分の専門分野で働いたことはありませんが、この事実は私の「キャリア」と私の義務を果たすためのアプローチに大きく影響します。 全般的に、enikeyの機能を実行し、私が「システム管理者」であるという甘い思いに慰められて、私は一種の「電子技術者」として林業大学で働いています。 そのような賭け。

大学には約120台の車があり、部分的にピアツーピアネットワークに統合されています（とにかくドメインを上げることを望んでいますが、異なる代理人、会計士、およびエコノミスト、なぜそれが必要なのか、しかし彼らの問題ではなく、私の問題の可能性が高いのです。 5つのコンピュータークラスがあります。 これらのクラスの1つには、macminiコンピューターが装備されています。



大学で働く前は、macminiを見たことがなく、mac osを使用していませんでした。リンゴについては、「高価」、「高品質」、「ロシアでは不十分」としか聞いていませんでした。 しかし、私はLinuxでの作業の経験があったことを神に感謝します。 その時までに、私は自宅のPCとラップトップをubuntuに移しました。これは、FreeNASとDebianについて少し「混乱」していました。 そして最も重要なことは、Linuxに会った後、ソフトウェア、マイクロソフトのポリシーを警告し、これらの条件下で若い世代を教育するという私の態度を完全にレビューしたことです...



世間知らずのdeされた子供たちほど悲しい話はありません...


クラスについて説明します。

i3 / 2GB / 500GBの7-warezを搭載した3台の新しいPC、warez XPを搭載した1台の古いPC、13台のmacmini

XP「獣版」とみすぼらしい誤って仕切られたパーティション、およびWindows Vistaのすべての美しさで残っている12に、ブートキャンプを使用してMAC OS X「雪ヒョウ」と並行してインストールされた「ビジネス版」が、実際には唯一のシステムですこのコンピューター。 そして、ライセンスされているように見えますが、アクティベーションは更新後に「飛びます」（ここでは触れませんでした）。 これにより、すべてがウイルスに感染し、何かが常に機能せず、子供たちは何かを壊しています（たとえば、素敵なアルミニウムキーボードからボタンをはがします...）、白い小さな箱が汚れの層で覆われると... ... mmm ...これの代わりに、アップルのライセンスソフトウェアを使用しないのはなぜですか？ "、「理解するものは何もない」、「Photoshop、Office、matkadなどはありません」などの回答を受け取りました。



悲しい 私の意見では、かなり。 そして、これらのケシは何らかのプログラムに従って大学に寄付されたため、どれだけの費用がかかり、どれだけ多くの人がそれを夢見ているのか、誰にもわかりません（私自身はアップルファンではありません）。

一般に、私はすぐに「すべてが機能し、速度が低下せず、どこでもすべてが同じ」というタスクを受け取りました。 そして、問題をグローバルに解決するために（MSオフィスをオープンまたはlibreに置き換え、他のソフトウェアのMAK OSのアナログを探して、ライセンスにお金がないので「オープンソース」が望ましい）、まだ気が進まないXP + MAK OSの道をたどるという決定を、ペルモフスキーの「より小さな悪の原則」を使用して行いました。



実装


1）「参照」コンピューターが構成されました。 これを行うには、次のことを行いました。

-マシンの1つで「ユキヒョウ」を完全に更新

-ホイストセクションを完全に破壊しました（原則として、この手順は必要ありません。最初は、ポピーの「内側」に仮想マシンにXPを配置する予定でした）

-Boot Camp Assistantを使用して、Windows用のパーティションを作成し、 このマニュアルに Windows XPをインストールしました

ちなみに、win XP sp2以前の元のディスクのみが必要であり、「獣」のような異なるアセンブリとマルチブートディスクは機能しないと書かれています。 はい、最初に、ウィザードを使用してパーティションを作成すると、アシスタントは単にマルチディスクを「表示」しませんが、チェックしませんでしたが、最初にオリジナルをインストールした場合、任意のアセンブリに置き換えることができます。 しかし、再び、これらは単なる推測です。

-XP sp2を現在の状態に更新し、アカウントをセットアップし、必要なソフトウェアをインストールしました

-次に、他のポピーでこれらすべてを複製する方法の検索が開始されました。 私の間違いは、この目的のためにMackユーティリティだけを探していたことです。 彼は多くの時間を殺したが、結果を達成しなかった。 たとえば、 カーボンコピークローンは、パーティションのイメージをXPから外部ネジに書き込むことを拒否しました（NTFSファイルシステムがあるため）。 そして、私が完全に必死だったとき、私はただの楽しみのために、古き良きClonezillaを試してみることにしました。 すぐに使用しなかったのはなぜですか？ どうやら、私は教育の生物学者だから...なんらかの理由で、これは本物のバイオスではなく、これはケシバチだからだと思った...一般的に、クローンジラはすべてを完璧にやった。 迅速かつ整頓。

-まあ、それはNetBIOS名、IP、すべてのジャズを変更するだけです



おわりに


非常に長い間、私はこれで何かをしようとしていたと言わなければなりません。 しかし、私は未知のアップル製品に対する恐怖に悩まされました。 そして、この問題の解決にはたった1日しかかかりませんでした。そこで、私は無料で仕事に行くことにしました。 繰り返しになりますが、私は何かをするためには、それを始める必要があることに気付きました！



この経験が、私のような生物学者であるシステム管理者に役立つことを願っています。



PS

大学の状況をすぐに変えられることを望んでいます。 私は同僚とSTRと著作権侵害のトピックについて教育的な仕事を行い、オフィスとThe Bat MCの代わりにlibreオフィスとサンダーバードを導入し、大学のITインフラストラクチャ自体を再編成し、ドメインを上げる予定です。 電子申請システムとニュース、指示などを備えたポータルは、すでに本格的に機能しています。 ネットワーク内。



それでは、すべてのオープンソース開発者、特にClonezillaに心から感謝と敬意を表します。
