効果的なMac OS X

Mac OS Xは多くの有用性を備えた非常に美しい軸であり、デフォルトの設定と設定を使用しても管理と使用がく簡単です-これは事実です。 しかし、私は効率に興味があり、いわば、OS全体の十分な使用に興味があります。 そのため、この投稿を書いて、12インチのiBookで作業するときにFE *を上げるために何をしているのかを詳しく説明するようになりました。 また、この投稿と私のスキルは、最初のMacを購入したばかりのスイッチャーにとって有用です。 だから、言葉から行為まで。







きれいなデスクトップ



私はがらくたのデスクトップが嫌いです。 デスクトップ上の過度のアイコン、ファイル、フォルダー-FIG！ すべてのアイコンとファイル、フォルダ-デスクトップからのイチジクのすべて。 最も必要なものだけを残します。ハードドライブ上のアイコン、1つまたは2つのスマートパパ、ある種のファイル。 もうありません。 ドライブ、USBフラッシュドライブ、デジタルカメラなどのアイコンは、接続されると表示されたり、切断されたりするので、それらについても考えません。 ダウンロードしたファイルは、すぐにダウンロードフォルダーに送信されます。 デスクトップは常にきれいである必要があります 。



ドックアイコン



あなたがそれらがプログラムの便利で迅速な起動のためであると思うなら、あなたは間違っています。 私は「削除」のファンだとは思いませんが、... Mac OS Xのドックに追加のアイコンが1つもありません。 ファインダー、ダッシュボード、ゴミ箱の3つしかありません。 プログラムを起動すると、ドックに表示されることに気づきましたか？ そのため、実行中のプログラムでそれを埋めるためにドックが必要です。







あなたにとって最も便利だと思われる場所にドックをインストールすることが重要です。 一番下の中央に置くのが最も便利ですが、ドック自体はあまりスペースをとらないので、システム環境設定で減らしました。



すぐにドックを正しく配置することが重要なのはなぜですか？



人がコンピューターを使用する方法は、全体的な哲学です。 純粋に心理的には、どのユーザーも、多くの場合、毎日コンピューターを使用して、特定のプログラムを特定のシーケンスで起動するように構成されています。 これは非常にデリケートな瞬間であり、コンピューターでの毎日の作業中にほとんど気付かないほどです。



実際、このまさに理由で、ドックを便利な場所に配置することにより、特定のプログラムで特定のアクション（閉じる、ウィンドウを復元する、プロパティを開くなど）のマウスカーソルを画面上の特定の場所に自動的に移動することに慣れます。



そして同じ理由で、素晴らしいプログラムが発明されました。これについては次のブロックで説明します。



クイックシルバーは私たちのすべてです



一見したところ、 Quicksilverは恐ろしいツールのように見え、非常に不快に感じるかもしれません。 ドックにアイコンを設定できるのに、なぜプログラムの名前を覚えているのでしょうか？ そして、なぜSpotlightをQSに変更するのですか？ Quicksilverは非常にクールなプログラムランチャーであり、ポピーの検索エンジンであり、一言で言えば、非常に簡単です。



前述したように、通常、すべてのユーザーは日常生活で特定のプログラムを使用します。 たとえば、私はすぐにSafari、Adium、Skype、Mail、Textmateをオンにしますが、時々Lightroom、iGTDで作業したり、お気に入りのおもちゃをいくつかプレイしたりする必要があります。 私の名前にはかなり馴染みがありますよね？



Quicksilverの使用は非常に簡単です-ホットキーと呼ばれ、自由裁量で設定されます。 プログラムの標準ビューが表示されます。







上部のキーワードを入力すると、単語が入力されると下部に短い説明と可能なアクションが表示され、上部にプログラム/ファイルの名前の下にハードドライブ上の場所が表示されます。 さらに、右側にはすべての結果を含むドロップダウンリストがあります。 [戻る]ボタンを押すと、キーワードを入力するときに停止したものが起動します。



QSでは、プログラムのフルネームを入力する必要はまったくなく、必要なものを入力することができます。 そして、これはQuicksilverができることのほんの一部です。 概して、それは特別な注意と、可能性の非常に大まかな概要に値します。 アクションのチェーン全体を記述したり、お気に入りのブラウザでキーURLのリンクを開いたり、さまざまなトリガーを設定したり、多くのプラグインがあります 。



次は何ですか



Sticky Windowsを追加することもできます 。 iBookをお持ちの場合、12インチ画面が小さすぎることがあります。 とにかく、さまざまなプログラムの開いているウィンドウの束でデスクトップを詰まらせるのは嫌です:)。 スティッキーWindowsはこの問題を解決します。 オフスクリーンデモのスクリーンキャストをご覧ください 。



もう一つの重要なポイント。 Mac OS Xは必要な場所をあなたのために考えます。 したがって、自分でMacを購入すると、簡単に慣れてすぐに注文できます。 :)
