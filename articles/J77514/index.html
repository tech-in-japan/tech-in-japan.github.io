AMDとIntelの12個のプロセッサーの比較テスト

競合メーカーのIntelとAMDの12種類のプロセッサをテストしました。 私たちの主なタスクは、それらの中で私たちが今やろうとしていることを特定することです。 しかし、最初に、プラットフォームに関するいくつかの言葉。

この記事では、Intelのソケット1366および1156、AMDのAM2 + / AM3のプロセッサを紹介します。 現時点では、かなり多くのマザーボードがすでにリリースされているため、ニーズに合わせて「母」を選択することは難しくありません。



もう1つのことは、新しいIntelプラットフォームへの移行にはかなりの費用がかかるということです。 たとえば、ソケット1366の平均マザーボードのコストは約6,000ルーブルです。この金額にプロセッサ価格を追加する必要があります。 しかし、AMDの場合、図はより控えめになります-約2000-2500ルーブル。 AMDソリューションは、より広い範囲の人々が利用できます。これが焦点です。 さらに、AM3用のプロセッサを購入し、AM2 +用の古いマザーボードにインストールできます。 そのため、少し節約して、少し後にAM3に切り替えることができます。 もちろん、選択はあなた次第です。



そして今、テスト方法論についてのいくつかの言葉。



テスト方法。

テストは2つの設定プロファイルで実行されました。

1.デフォルト-定格周波数と定格電圧で性能が評価されました。

2.オーバークロック-プロセッサーは3600 MHzの周波数にオーバークロックされました。 この周波数は偶然に選択されたものではありません。 すべてのプロセッサが通常のユーザーと24時間体制で作業できるのはその上です。 多くのプロセッサも高周波を示すことができますが、実験の純度のために、この時点で停止しました。



次にテストについて：

•まず、最も有名なプロセッサテストの1つであるSuper Piを選択し、小数点以下100万桁を数えました。 このベンチマークは、「石」の計算能力の良いアイデアを提供します。

•Super Piの隣にはwPrimeがあります。これはマルチスレッドテストであり、マルチコアプロセッサにとって重要です。 このテストのパフォーマンス評価は、32Mおよび1024Mモードで実行されました。

•ターンとPiFastが来ます。 その主な機能は、CPUだけでなく、プロセッサとメモリのバンドルのパフォーマンスをチェックすることです。



テストスタンド。

3つのテストベンチを収集しました。

•RAM DDR3 3x1Gbキングストン1333MHz CL7 1.5V

•AMD Radeon 5870グラフィックスカード

プロセッサをさらに詳しく検討するときが来ました。



Intel Core i5 750

このレビューにより、Intel'a Core i5 750の発案が始まりました。このメーカーの他のプロセッサと比較して、そのコストは低くなっています。 すべてのテストで、彼は「平均的」であることを示しましたが、Intelの創作の中で最も遅いものでした。 残念ながら、Intel Core i5 750はハイパースレッディングテクノロジーをサポートしていないため、このプロセッサーには仮想コアがありません。



Intel Core i7 860

Intel Core i7 860もLynnfieldアーキテクチャに基づいています。 このプロセッサは、ハイパースレッディングテクノロジのサポートを受けており、多くのインラインテストで優れたパフォーマンスを発揮します。 そのパフォーマンスはCore i5 750のパフォーマンスとほぼ同じですが、wPrimeでは仮想コアによりパフォーマンスが向上しました。 しかし、私たちの意見では、その価格は誇張されています。 その価格で、3チャネルメモリコントローラーを備えた1366ソケット用Intel Core i7 920を購入できます。



Intel Core i7 870

これは、レビューでの1156プラットフォームの最後の代表であり、Core i7 870とCore i7 860の違いは、高い乗数のみです。 現時点では、LGA 1156の最高速のプロセッサです。しかし、欠点もあります。 周波数が等しい場合、テストでは、Intel Core i7 870と新しいモデル860がほぼ同じ結果を示しますが、コストはほぼ2倍異なります。 私たちの意見では、このプロセッサはお金の価値がありません。 Intel Core i7 860またはIntel Core i7 920は、より合理的なソリューションに見えます。



Intel Core i7 920

次に、ブルームフィールドのコアにあるプロセッサを検討します。 最も安いのはCore i7 920で、かなり合理的な購入と評価されています。 9〜1万ルーブルのコストで、すべてのテストでかなり高いレベルのパフォーマンスを示しています。 次のIntel Core i7 950モデルはすでに19〜2万ルーブルの費用がかかり、それによって大きな価格差が生じることも考慮する価値があります。 Intel Core i7 920には優れたオーバークロックの可能性があるため、ユーザーは簡単に周波数を4 GHzに増やすことができます。 しかし、より高い周波数が必要な人にとっては、21倍では十分ではありません。



Intel Core i7 950

このプロセッサと新しいモデルの違いは23倍であり、1コアで最大25倍になります。 このプロセッサーは、乗数の増加によりオーバークロックの面で興味深いですが、その価格は若いモデルと比較して正当に高いわけではありません。 アーキテクチャ的には同じです。



Intel Core i7 965 Extreme Edition

そして、ここに私たちのレビューで自由な要因を持つ最初の「小石」があります。 これは、多くのオーバークロッカーが使用するハイエンドクラスのプロセッサです。 これにより、極端な冷却で高い周波数を実現できます。 しかし、残念なことに、これはC0ステッピングでのみリリースされたため、図に見られるように、ラインの他の代表者は同じ周波数で同じ結果を達成できます。 現時点では、モデルは廃止されています。



Intel Core i7 975 Extreme Edition

Intel Core i7 975 Extreme Editionは、同社の旗艦であり、テストの絶対的なリーダーです。 彼はすべてのテストで最高の結果を示しました。 現在、このプロセッサには多数の世界記録がインストールされています。 一部のインスタンスは、6 GHzのマイルストーンを克服することさえできました。 もちろん、そのコストのために、平均的なユーザーはそれを購入することを決めそうにありませんが、オーバークロッカーや愛好家にとってこれは素晴らしい選択です。



AMD Phenom X4 9850BE

AMDプロセッサについて話をする時が来ました。 レビューの最初は、第一世代の天才モデルです。 これらのプロセッサは、AMDの歴史の中で最悪のものの1つであったと言われるべきです。 したがって、それらから高い結果を期待しないでください。 私たちのテストでは、彼は部外者になりました-すべてのベンチマークの最後の場所。 そして、3,500ルーブルの平均価格はあまり魅力的ではありません。 そのため、Phenom IIラインナップからプロセッサを選択することをお勧めします。



AMD Athlon II X4 620

Athlon II X4ラインは、ミドルエンドソリューションの下位セグメントとして会社によって位置付けられています。 それらのパフォーマンスは最初の現象と比較してわずかに向上し、45 nmプロセステクノロジーへの移行が行われました。 しかし、これらのプロセッサーにはもはや自由な要因がなく、深刻なオーバークロックはもはやできません。 ビデオを見てインターネットをサーフィンするためのマルチメディアシステムを除き、インストールする必要があります。



AMD Phenom II X4 940 BE

Phenom IIラインは、最初のPhenomよりもはるかに成功しています。 それらの外観により、AMDプロセッサの8 GHzマイルストーンを克服することができました。 AMD Phenom II X4 940 BEは、6 GHz以上の極端な冷却下でのオーバークロックにも適しています。 強力で安価なシステムを構築するユーザーにとって、このプロセッサは優れたソリューションとなります。 短所には、高レベルの熱放散とAM3ソケットに取り付けられないことが含まれます。



AMD Phenon II X4 955 BE

これらのプロセッサーは、AM2 +およびAM3、およびメモリーを備えたシステムで動作する機会を得ました。 DDR3。 AM2 +プラットフォーム用のマザーボードを既にお持ちの場合、これは優れたアップグレードソリューションです。 とにかく、このプロセッサは、テスト結果が良好で比較的低コストであるため、注意する価値があります。



AMD Phenon II X4 965 BE

若いモデルとの違いはそれほど大きくありません-周波数は200 MHz増加し、3.4 GHzになります。 しかし、欠点があります-放熱が145ワットに増加しました。 より良い冷却の世話をする必要があります。 ラインのすべてのプロセッサで3.4 GHzの周波数を達成できるため、AMD Phenon II X4 965の購入は私たちの意見では最も合理的なソリューションではないことに注意してください。

結論

インテル製品がリーダーになることを疑う人は誰もいませんでしたが、そのコストは非常に高いです。 この会社のファンにとって最もリーズナブルなソリューションは、Core i7 920を購入することです。これは、ファミリーで最も安く、優れたパフォーマンスを発揮します。 一般に、Phenom IIラインの登場により、AMD Phenom II X4 955に注目する価値があります。AM2+プラットフォームとAM3の両方で動作します。 最後の重要な利点は、DDR3メモリのサポートです。 テストでは、Phenom II X4 965よりわずかに劣るため、アップグレードに適したソリューションです。

プロセッサは、最も合理的なお金の無駄として報酬を受け取ります
