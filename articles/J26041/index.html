IEのレイアウトで

こんばんは

レイアウト設計者に助けを求めます。 タスクは簡単ですが、それは私を停止させました。

以下のタスクは、角が曲がった入力を作成することです。



私はこれが好きでした：

HTML

<input />







CSS

* {

margin:0px;

padding:0px;

}

*:focus {

outline: none;

}

.input{

background: url(/TCv2/design/images/inpt_r.gif) no-repeat 100% 0;

width:106px;

}

.input span{

position:absolute;

border-left:0px;

border-right:0px;

border-top:1px solid #b2b2b2;

border-bottom:1px solid #b2b2b2;

}

.input input{

border:0px;

height:22px;

width:100px;

font-size:12pt;

}







その結果、Firefoxのすべてが計画どおりにうまくいきます。



IEは高さを2ピクセル増やします。





これを克服するのを手伝ってください。さもないと、力がなくて苦しみます。 または、別のオプションを教えてください。 すべての入力の幅は異なりますが、高さは固定されています。
