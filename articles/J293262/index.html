8月の興味深い国際イベント

毎月、数百ではないにしても、数十ものIT指向の会議、展示会、その他のイベントが世界中で開催されています。



もう一度、Megamindの読者を1か所で紹介するために、今月の最も興味深い国際的な日付をすべて収集します。



8月3日から6日までシカゴで開催される3日間のClueCon 。



ボストンでは、8月3日から8日まで、主要なSANS情報およびサイバーセキュリティ会議が開催されます。



メタマテリアル、フォトニック結晶、プラズマトーチに関する会議であるMETA'15が 、8月4日から7日までニューヨークで開催されます。



そして、ほぼ同じ日付である8月5〜7日、カナダのバンクーバーで、 Design＆Content Conferenceが開催されます。



8月6日から9日まで、ラスベガスでDefCon 23が開催されますが、紹介は不要です。



8月8日、非営利のDDDであるメルボルン6がオーストラリアのメルボルンで開催されます。



内部のTwitterアクセラレーターであるHatch Eventのカタルシスは、8月11日に行われます。



SumIT International Startup Schoolは、8月10〜29日にサンクトペテルブルクで開催されます。



そして、米国ウィスコンシン州では、8月10日から12日まで、「オタクのための夏のキャンプ」- その会議があります。



デンバーTypeCon （タイポグラフィ）で8月12日から16日まで。



しかし、8月12日から14日まで、ワシントンでAthgothonが開催されます。



そして8月15日にモスクワで、関数プログラミングFPconfに関する最初の会議が開催されます。



少し遅れて、8月18日と19日に、 会議はスウェーデンのマルメで開催されます。



8月19日から21日まで、 HybridConfはダブリンで開催されます。



React Rallyは8月24日から27日までソルトレイクシティで開催されます。



8月25日と26日にニューヨークのマンハッタンで、 Yahoo Mobile Developer Conferenceが開催されます。



8月27と28日にチューリッヒで、開発者がフロントエンド会議に集まります。



8月31日から9月3日まで、ベルリンのMobileTech Conferenceで 。

すべてが同じベルリンにあり、 IoT会議の同じ日に開催されます。



8月31日から9月3日まで、米国アーバインで暗号学の数学会議が開催されます。



さて、8月はITに関係のないイベントで終わりますが、それでも焼in的です： Burning Man 2015 。
