2014年のビルド。WindowsおよびWindows Phone開発者向けのホットニュース





ほんの数時間前に、 Build 2014カンファレンスのオープニングが終了し、WindowsおよびWindows Phoneの開発者とユーザー向けに多くの興味深い発表が行われました。



Build 2014カンファレンスが4月2日から4日までサンフランシスコで開催されることを思い出させてください。 私たちは、イベントと録画のオンラインブロードキャストを利用できます。



会議の開幕後、開発者にとっての主なニュースは、共通のAPIと設計原則を使用して、スマートフォン、タブレット、ラップトップ、テレビでも同様に機能するユニバーサルアプリケーションを作成できることです。



さらに、近い将来に私たちを待っている他の変更が発表されました。

  • オペレーティングシステムと製品の新しいバージョン 。
  • アプリケーションストアの変更 。
  • 新しいデバイス 。


そして今、すべてについて...



新製品に会います！



Windows Phone 8.1


数か月のうちに、新しいモバイルオペレーティングシステムであるWindows Phone 8.1のリリースを待っています。WindowsPhone 8.1は 、Windows Phone 8のすべてのデバイスの更新プログラムとして利用できます。開発者は少し前に利用できます（期限はまだ発表されていません）。







オペレーティングシステムのいくつかの新機能が実証されました。

  • デバイスで許可されている場合、2枚のSIMカードを使用する機能。
  • カスタマイズ可能なロック画面。
  • メイン画面と背景画像のタイルが増えました。
  • 通知センター-さまざまなアプリケーションからの通知を有効/無効にする機能。
  • ロシア語をサポートした高速タイピング用の新しいswypeキーボード。







  • なじみのない場所でウェブを検索して接続するWi-Fiセンス。
  • 静かな時間-アプリケーションから通話、メール、通知を受信するためのカスタマイズ可能なスケジュール。 ジオロケーションを決定し、指定された設定を適用できます。 たとえば、家にいるときはメールを受信しません。
  • 改善されたカレンダーと毎週のデータ表示。
  • Cortanaは、個人的なBingベースのデジタルアシスタントです。 彼女と通信できます。彼女は膨大な数の音声コマンドを実行する方法を知っており、訓練を受けています。







  • VPN、S / MIME、MDMを使用したデバイス管理、および企業ユーザーと組織の暗号化のサポート。


Windows 8.1アップデート


MSDNサブスクライバーが既に利用できるクライアントオペレーティングシステムWindows 8.1 Updateの更新を待っています 。 Windows 8.1を使用している他のユーザーには、4月8日にアップデートが完全無料で提供されます。















Windows 8.1の更新には、多くのユーザーによる非常に待望の変更が含まれています。

  • すべてのアプリケーションのボタンを最小化して閉じる。
  • メイン画面のシャットダウンおよび検索ボタン。
  • ライブタイルのコンテキストメニュー。
  • マウスで一度に複数のタイルを移動します。
  • 検索は、ストア内のアプリも検索するようになりました。
  • アプリケーションストアは「タスクバー」に固定されています。
  • IE-IEの以前のバージョンとの互換性のためのエンタープライズモード。有効にするリソースのポリシーを管理する機能を含みます。


また、彼らはオープニングで、彼らが今取り組んでいることを示し、話しました-スタートメニューと、ウィンドウでデスクトップモードでMetro UIアプリケーションを起動する機能。 「タスクバー」を使用してそれらを表示し、切り替えます。

Windows PhoneおよびWindows 8.1用のユニバーサルアプリケーション



これで、WindowsとWindows Phone用の単一の開発プラットフォームができました。



Windows PhoneもWindows RTを使用します。VisualStudio 2013では、SDKをインストールすると、ユニバーサルアプリケーションを変換および作成するための新しいプロジェクトテンプレートと機能が表示されます。



共通のカーネル、コントロール、および設計原則に加えて、新しいオペレーティングシステムには、WebGLをサポートする新しいDirectX 12およびIE 11が含まれます。



ユニバーサルWindowsアプリケーションは、Xboxでも起動できます。



アプリケーションストア



このようなグローバルな変更は、アプリケーションストアに影響を与える可能性があります。 そして、ここにはたくさんの良いニュースもあります。



ご存知のように、WindowsおよびWindows Phoneストアでは、共通の開発者アカウントを使用しています。 そして今、私たちはアプリケーションの単一の識別子を待っています。



これで、スマートフォンでアプリケーションを購入し、タブレットやデスクトップに再度支払う必要がなくなります 。



また、店舗のインターフェースはわずかに変更され、好みに基づいてアプリケーションを推奨する方法を学びます。



新しいデバイス



夏には、ノキアのスマートフォンの新しいモデルを期待しています。 フラッグシップのNokia Lumia 930と、活気に満ちた手頃な価格のNokia Lumia 630およびNokia Lumia 635 。







Nokia Lumia 930は、Windows Phone 8.1をベースにした、見栄えの良い新しいスマートフォンで、画面が良好で、20 MPカメラ、安定化、ケースの周囲にいくつかのサウンドスピーカーがあります。



Nokia Lumia 630およびNokia Lumia 635は、1枚または2枚のSIMカードを搭載した手頃な価格のスマートフォンです。 彼らは、これらのデバイスの費用が200ドル未満になると約束しています。
