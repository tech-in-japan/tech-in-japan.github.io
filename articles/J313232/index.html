Microsoft HoloLensは発火しますか？

Microsoft HoloLensデバイスの一連のテストにより、「 ITの巨人によるこの革新的なデバイスを点火するかしないか 」という議論の余地のある質問に導かれました。 HoloLensが専門委員会のメンバーの1人を連れてきたプログラム「 ゲーミングインターネットプロジェクトの管理 」の学生の卒業認定作業の防衛で初めてテストしました。 その後、彼は会議でこのデバイスの所有者と直接会ってそれをさらに数回テストしました。 カットの下で、私はデバイスに関する簡単な情報、その展望と専門家の意見についての専門家の意見、そして自分自身の印象、そしてhabrasocietyの調査を収集しました。 私はあなたの意見を知りたいです。







HoloLensについて簡単に説明します



これは、頭の上に置かれ、外部デバイスへの接続を必要としない小さなコンピューターです。 私たちの目の前には、目に見えるオブジェクトが作成される半透明のスクリーンがあり、世界にオブジェクトが実際に存在しているような錯覚を作り出します。 スクリーン-導波管レンズ-側面にあるマイクロディスプレイから私たちの目に正しく屈折して絵を送る波状のプリズム構造のレンズ。 光学系と仕様の詳細な説明はこちら 。









他の拡張/混合/バーチャルリアリティデバイスとの大きな違いは、 HoloLensは完全に自律的であり、それらの作業がスマートフォン、コンピューター、またはベルト上の箱に結び付けられていないことです 。 理論的には、建築プロジェクトの視覚化、人体のリアルな3Dモデルの作成、医師にとって有用、トレーニングのため、Windowsエコシステムのもう1つのおいしい部分であり、ホロレンズを介したトップ10およびそのすべてのアプリケーションを「ホロポーテーション」として使用することを目的としています。スカイプなどよりも優れたより効果的なコミュニケーションのために。 つまり、私はTKのデザイナーに「傑作」の修正を依頼するだけでなく、指でホロレンズを直接当てて、デザインに成長ゾーンがあり、このゾーンがどこにあるかを指摘しています。 デバイスの位置付けから判断すると、 視覚化が必要な食品の専門家が使用するためのb2bセグメントには、デバイスからの最大の期待があります。 まあ、エンターテイメント業界についても忘れないでください。











主要な競合他社はMeta 2とMagic Leapです。 メタもMLも試したことがない。 誰かがそのような喜びを予期せず持っていたら-それを共有してください。



実用的な便宜



一般に、HoloLensでの最大の問題は、特に3Kドルの価格での使用の実用性です。 確かに、これは開発キットの価格であり、おそらく消費者向けバージョンの方が安くなるでしょう。 だから、私はHoloLensでゲームをプレイし、ローマとマチュピチュの球状パノラマ画像のバーチャルツアーを見ました。 私は遊ぶのが好きでしたが、それはすごい効果のためだけにクールであり、HoloLensの大きな高品質のゲームや一般的にはさまざまなコンテンツがないため、長期的にはまだプレイできません。 仮想現実デバイス（Oculus、Vive、Sony）には現在、少なくともゲームでいっぱいの店舗があります。 これらのゲームはAAAクラスではありませんが、ますます多くなり、品質は向上しています。 プレイするものがあります。 しかし、HoloLensはそうではなく、問題はそうでしょうか？ HoloLensのコンテンツ開発に関する有名なスタジオの発表を聞いたことはありません。 しかし、ロシアでも小さなチームが今やそれを始めていることに注意する価値があります/ up.22.10-ここで彼らが興味深いことに書いたコメントでは、レンズの開発者のための全体的なスラックチャットがありますまた、このデバイスに対するMicrosoftの焦点は依然としてb2bに焦点を合わせていることを忘れないでください。







点灯するかどうか



デバイスの適切性と技術的特性の評価に基づいて、HoloLensの需要があるかどうかについて異なる意見があります。 バーチャルリアリティに関する専門家コミュニティのメンバーの意見に感謝します（A. Artishchev、V。Meshcheryakov、A Lisovitskyなど）。 一方では、グーグルグラスと平行線を引くことが可能であり、彼らは同じ道をたどり、大量の需要はないだろうと言う強い意見があります 。 高価格、低視野角（30 x 17.5度）、コンテンツの欠如、通常のエコシステムを考えると、Google Glassのような普遍的な製品を作ろうとすると、対象読者のいない製品になるため、1石で9羽の鳥を手に入れることはできません。今グーグロットレンドを見ると、他の大きなVR / ARデバイスと比較してホロレンズで低い誇大広告が正確に見えます。 ちなみに、Sony VRはスケジュールに追加することさえできません。残りはバックグラウンドで失われるためです。







一方で、 これはこれまでのところ、開発と修正のためだけの初期の粗雑なプロトタイプであるdevkitであるという強い意見があります。 そして、HoloLensが3,000ドル以上で販売されていることは非常に良いことです。この購入が将来どのような配当を彼らにもたらすかをよく理解している人は、リスクを認識してそのようなものを買うべきです。 そして、ファイナライズ、デバッグ、エコシステムの作成時に、高品質のコンテンツが大量に準備され、過度に控えめなFOVの問題を解決し、バッテリー寿命を延ばし、市場に出て、より低い「消費者」価格で点灯します。 それはおそらく数年後になりますが、まさに議論のトピックであるとき。 コメントと添付の投票で議論しましょう。



さらに、ワイヤーの欠如、一般的に完全な自律性、ジェスチャーの追跡、クールな3D視覚化など、デバイスの利点を再度表明します。







私の意見では、発火の見通しは良好です。 それらはまた、主観的な感覚によるものです。 それらについて詳しく説明します。 だから、仮想現実のヘルメットを身に着けて20分以上費やしますが、私にとっては困難になり、外出する必要があると感じています。 しかし、HoloLensではこれを感じていませんでしたが、逆に、ほとんど慣れています。 私個人にとって、これはこの技術の成功を支持する強い要因です。



ですから、Microsoft Hololensは、将来的に便利で人気のある製品を作成する機会の良い予備品だと思います。 友人に見せたり、会議で使用したり、そして最も重要なことは、リスクを冒したい人のために、今すぐHoloLensでコンテンツを作成し始め、この技術の爆発的な成長を待って、お金で蒸気機関車に飛び乗る最初の人の1人になることができます。
