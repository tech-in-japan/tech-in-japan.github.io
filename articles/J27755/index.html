Last.fm API

しばらく前、私のお気に入りのLast.fmがAPIを公​​開しました。 これは、誰でも（あなたも）世界で最も人気のある音楽サービスのデータベースを使用できるようになったことを意味します。



Last.fmがあなたに伝えることができるもののリストの一部を次に示します。
  • ユーザープロファイル
  • 指定された期間のアーティスト、アルバム、ユーザータグのトップ
  • 最後に聴いたお気に入りの曲のリスト
  • 友人と隣人のリスト
  • イベント（あなた、友人、特定のアーティスト、推奨）
  • 推奨音楽
  • テイストメー​​ター
  • アーティストに関する情報（アルバム、評価、タグ、イベント、類似アーティスト、熱心なリスナー）
  • アルバム情報
  • 単一のトラックに関する情報（タグクラウド、類似のトラック、熱心なリスナー）
  • タグに関する情報（評価、アルバムおよびトラックのアーティストのリスト）
  • グループに関する情報（グループの評価と推奨事項）
  • 各国のトップチャート
  • グローバルトップチャート


データは、XML、 XSPF 、RSS、iCal、およびプレーンテキストで提供されます



Last.fm APIを使用して気に入ったいくつかのサービス。
