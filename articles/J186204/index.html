The Pirate Bayの共同設立者は、トラッカーの閉鎖を呼びかけています



ブラジルへの旅行中、The Pirate Bayの創設者の1人であるTobias Andersson氏は、自分の意見では、トラッカーはより良い場所を確保するために存在しなくなるべきだと述べました。

「現在の形の海賊湾は消滅しなければなりません。 トラッカーは、私たちに近づいている未来を考慮せずに作成されました。 無料のファイル共有のための明日のファイターには、より良い、より安全でより速いものが必要です」と彼は言いました。



過去10年間、The Pirate Bayをフォローしてきた人たちは、スウェーデンのプロカルチャーPiratbyrånによって設立されたことを知っています。

「海賊の局」と訳されるピラトビランは、2000年代初頭に政治活動家とハッカーによって形成されました。 彼らの多くは、政治、道徳、権力の濫用に挑戦する他のWebプロジェクトをすでに開始しています。 すべてのPiratbyrånメンバーは友人であり、「友人の友人」であり、何らかの形でPirate Bayを知っていましたが、海賊局には実質的に明確な構造はありませんでした。

トビアス・アンダーソンは、初期段階ではピラトビランとザ・パイレーツ・ベイの両方のスポークスマンでした。 トビアスは技術的な問題に関与していなかったという事実にもかかわらず、彼はトラッカーの創設者の一人でした。 トビアスは長年、トラッカーの開発プロセスに関与していたか、ビジネスから撤退しました。 4年前、長男の誕生後、彼は海賊湾の生活に積極的に参加するのをやめ、家族に専念しました。

昨年、彼はトラッカーの当時の管理者に、彼のアイデアを宣伝するためにThe Pirate Bayのホームページを使用できるかどうか尋ねました。 このプロジェクトは間もなくプロモベイとして知られるようになりました。このプラットフォームは、世界中の何百万人ものユーザーによる作品を独立したアーティストが展示できるプラットフォームです。

その後、トビアスは数年間「底に横たわりました」が、今では再び影を残しています。 栄光を浴びるためではなく、最終的な「解雇」を発表し、トラッカーに別れのメッセージを送るためです。

先週ブラジルのポルトアレグレで開催された第14回フリーソフトウェア国際フォーラムで講演した Pirate Bayの共同設立者は、Promo Bayでボランティア活動も辞めたと語った。 同時に、彼は追跡者管理者に追随してThe Pirate Bayを閉鎖するよう促した。

トビアスは、近い将来、The Pirate Bayは彼が今日果たしている重要な役割に対応できなくなると考えています。 トラッカーは、新しいテクノロジーに取って代わるように消えなければなりません。



「いいえ、冗談ではありません。 私は本気です。 現在の形式の海賊湾は存在しなくなるはずです。 トラッカーは、私たちに近づいている未来を考慮せずに作成されました。 無料のファイル共有のための明日のファイターは、より良い、より安全でより速い何かを必要とします。 数人の人々が自分自身を犠牲にするという欲求に依存しない何か。 世界には、襲撃、法律、または脅迫戦術でカバーできないものが必要です。

「海賊湾は何か新しいものの創造を妨げていると思います。 直接ではなく、トラッカーが絶え間なく存在し、脅威を無視するなどして、人々にとって快適すぎる状態を作り出したという事実によって。もしThe Pirate Bayが閉鎖を決めたなら、すぐに新しい何かが生まれるでしょう。 もちろん、The Pirate Bay以外にも他のトラッカーがありますが、それは最大であり、他の何百ものサイトがその急流に依存しています。



集中化されたBitTorrentサイトの概念は外部からの圧力に対して脆弱であり、法執行機関からの圧力が強いほど、サイト（サイト）を維持することが難しくなります。 新しいドメインを見つけるのは簡単ですが、ホスティングの状況はすでに問題になりつつあります。

Tobiasは、他の業界が3Dプリントの脅威にさらされると状況はさらに悪化すると考えています。



「MPAA、RIAAなどからは、大きな問題はありませんでした。 実際、警察の襲撃よりも、管理者の酔ったささやきのせいで、サイトがダウンする可能性がはるかに高くなります。 しかし、自動車メーカー、石油会社、国のリーダーが脅迫されていると感じるとき、私たちにはもっと良いものが必要です。 自律的で、襲撃や弾圧に関係なくその位置を維持するもの。」



Pirate Bayの共同設立者は、The Pirate Bayがアーティストの支援から抑圧的な体制に苦しむ人々への情報提供まで、多くの良いことをしたと信じていることを強調しました。 ただし、これはサイトが同じ方向に発展する必要があるという意味ではありません。 ちょうど反対。



「誤解しないでください、私はThe Pirate Bayが大好きです。 トラッカーでの作業の思い出は永遠に私と共にあります。 そして、私は本当に、Pirate Bayが世界をより良い場所にしたと信じています（そして今でもそうです）。 そして、まさに今、The Pirate Bayは非常に重要であるため、近い将来より良いものが必要になる理由を理解できます」とトビアスは言いました。



現在のThe Pirate Bayの管理がTobiasに同意するかどうかは、将来を告げるものです。 しかし、トラッカーの創設者の1人がThe Pirate Bayに閉鎖を要請したのはこれが初めてではありません。 3年前、Peter Sunde は同様の提案をしました 。



「新しいテクノロジーが必要です。 これは、パイレーツベイにとって一種の未来です。ボーズで休むことで、より良いものに置き換わることができます。
