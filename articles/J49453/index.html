Ubuntu 9.04 Alphaリリース

* ext4ファイルシステムの統合サポート。Ubuntuインストーラーインターフェイスを介したインストール中に選択できるようになりました。

*ユーザーインターフェイスは、システム通知と状態の変化のインジケーター（ボリュームの変化、wifiネットワークの検出、抽出のためのUSBフラッシュの準備など） を表示するための新しいメソッドを実装します。 新しく受信した通知が画面上に並んで一時的に表示されます。 ユーザーからの反応を必要とせず、自動的に非表示になり、中間通知の手段として最初に配置されます（ユーザーは、通知を生成したアプリケーションで直接詳細またはアクションを確認するように招待されます）。





* X Server 1.6 / X.Org 7.5の最新のテストスナップショットの統合コード。

* Ubiquityインストーラーには、個々のユーザーのホームディレクトリのコンテンツを暗号化する機能があります。 リリース8.10では、暗号化された形式で保存されるプライベートデータを保存するための「プライベート」ディレクトリが導入され、9.04では、ホームディレクトリ全体を暗号化することで機能が拡張されました。



テスト時には、独自のATI fglrxおよびNvidiaドライバーがX Server 1.6で動作せず、Intel i845およびi865チップに基づくビデオカードにも問題があることに留意してください。



現在、実装が計画されているイノベーションのページには、ユーザーとグループを管理するためのLDAP DIT（ディレクトリ情報ツリー）の作成、X.Org構成ファイルエディターの作成、ctrl-alt-backspaceを押してXサーバーを再起動することの禁止、 gdmをGNOME 2.24からバージョンに更新し、X.orgのすべての側面の自動テスト用のツールを開発し、オープンドライバーをXAAアクセラレーションアーキテクチャからEXAに変換します。
