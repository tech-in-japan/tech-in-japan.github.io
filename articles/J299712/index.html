メールマーケティング担当者向けの便利で無料の5つのサービス

今回は、効果的なマーケティングニュースレターの作成と送信に役立つ、便利で便利な無料サービスをご紹介します。 有効なメールを作成するには？ メッセージがスパムになるおおよその確率を見つける方法 猫へようこそ！





PostMark連中からのSpamCheck。



サービスアドレス： http : //spamcheck.postmarkapp.com/







このサービスを使用すると、スパムの兆候がないか手紙を確認し、SpamAsssinスパムフィルターにメッセージが入る確率を判断できます。 このサービスを使用してすべてのメールをチェックすることを強くお勧めします-これは、読み書きのできないメールによる潜在的な問題を回避するのに役立ちます。



通常のフォームを介したメッセージの送信に加えて、サービスにはプロジェクトにメッセージを統合できるAPIがあります（たとえば、 MailHandlerで積極的に使用しています ）。 このサービスは現在完全に無料です。



PutsMail by Litmus



サービスアドレス： https : //putsmail.com/



Litmusのメンバーは小さなスタートアップPutsMailを購入し、メールをテストするための無料サービスに開発しました。 メールに送信して、レターの表示をテストできます。 同時に、CSSをインライン化します。



MXToolBoxブラックリストチェッカー



サービスアドレス： https : //mxtoolbox.com/blacklists.aspx



このシンプルな一見サービスにより、ほとんどのスパムフィルターで特定のIPアドレス（またはドメイン）の存在をテストできます。 それらの105があります：Sorbs、Barracudaなど。







Mail.ruのポストマスター



サービスアドレス： https : //postmaster.mail.ru/



データベースに多数のMail.ruユーザーがいる場合、メールマーケティング担当者にとって不可欠なツールの1つ。 スパムになったメールの統計、ユーザーの苦情の件数、その他の指標を追跡できます。 使用するには、ドメインの権利を確認する必要があります（DNS、メタタグ、またはhtmlページを使用）。 また、各文字に文字を正しく表示するには、文字の識別子を持つ特別なヘッダーを追加する必要があります。



方言によるプリメーラー



サービス住所： http : //premailer.dialect.ca/



このサービスを使用すると、送信する電子メールを準備できます-不要なゴミ（未使用のIDやクラスなど）、インラインスタイルを削除します。



ボーナストラック：MailChimpからの電子メールブループリント



住所： https : //github.com/mailchimp/email-blueprints



さて、巨大なMailChimpがなければ。 皆さんは、ニーズに合わせてカスタマイズできる、美しく、適応性があり、有効なHTMLテンプレートを用意しました。
