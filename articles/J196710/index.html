新しい3CX Phone System Pro Editionがリリースされました

新しい3CX Phone System Pro Editionの高度な機能により、プログラムは従来の交換機と同等になります。 新しいエディションの最も重要な機能には次のものがあります。



  • コールセンターの高度な機能（不在着信の監視、キュー内の人数や通話統計などをリアルタイムで確認できる便利なコントロールパネルなど）。
  • Microsoft Exchangeとの統合による電話番号のサブスクライバー名への自動変換。
  • 複数のオフィスでのプレゼンスのステータスを追跡する可能性、および通話の詳細なレポート。








Microsoft Exchange統合コールファインダー



3CX Phone System ProにはMicrosoft Exchange Serverが統合されています。連絡先の要求は、3CX電話帳を使用してExchange Serverから直接行われるため、従業員は誰が電話をかけているのかを常に把握できます。 プログラムは1つのデータベースのみを使用します。3CX電話帳はMicrosoft Exchangeサーバーと自動的に同期します。これにより、多くの時間を節約できます。



プレゼンス状態の追跡



3CX Phone System Proを使用すると、Windows、Android、iPhone用の3CX Phoneクライアントを使用して、他のオフィスで同僚の存在を追跡できます。 マウスを数回クリックするだけで、電話をかけたり、会議を作成したり、インスタントメッセージを送信したり、ボイスメールを残したり、他のオフィスの同僚に電話を転送したりできます。



拡張レポート



3CX Phone System Proには、より強力なレポートツールがあり、詳細なデータを提供します。 レポートには、キューに関する重要な情報、顧客の待機時間の超過、コールバックの数に関する統計、コールの配信、不在着信、および作業を調整できるその他のインジケータが表示されます。



3CX Phone System Proの機能の詳細については、サイトをご覧ください。
