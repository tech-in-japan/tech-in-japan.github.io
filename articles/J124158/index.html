フードの下のサイトにあるもの

ウェブマスター向けのもう1つの便利なツール、 UnderTheSiteが登場しました 。 彼は、HTTPヘッダー、ページコード（クローラーが約10ページをダウンロードする）、JavaScript、CSSを研究して、特定の技術（マッチャー）の特徴的な痕跡を特定しています。 1つのリクエスト-発見されたものの完全なリストがすぐにページに表示されます。 すぐにフォームのURLをthethesite.com/sites/habrahabr.ruに入力できます。 例：



Google ：Google Webmaster Tools Verification、Google Analytics、Google Web Server、UTF-8 Unicode charset、Google Web Fonts、HTML5、Google Website Optimizer、YUI Library。



Yandex ：jQuery、Nginx、Open Graph Protocol、OpenID、OpenSearch、UTF-8 Unicode文字セット。



Habrahabr ：Google Analytics、Nginx、MooTools、swfobject、UTF-8 Unicode文字セット。



作成者は、 決定するために新しいテクノロジーを追加するように要求します。登録後に誰でも自分でそれを行うことができます。
