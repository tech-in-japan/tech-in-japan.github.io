マイクロソフトは、VMwareを放棄したお客様に無料のWindows Server 2016 Datacenterライセンスを提供します





Microsoft Windows Server Teamの企業ブログは、 昨日 、2016年9月1日から2017年6月30日まで、VMwareからHyper-Vにワークロードを移行する準備ができている顧客にはWindows Server 2016を無料で提供すると報告しました。



マイクロソフトは、シールドされた仮想マシン、Just Enough and Just In Time管理、ローリングOSクラスターアップグレード、ストレージスペースダイレクト、ストレージレプリカ、SDNロードバランサー、ネットワークコントローラー、分散ファイアウォールなど、新しいハイパーバイザーとオペレーティングシステムの多数の新機能に注目しています、初心者向けのAzure Data Planeなど



ガートナーの専門家は、Microsoftが仮想化の分野で数年間リーダーのチームにいることを確認しているだけでなく、VMwareの地位に着実に近づいていることにも注意してください。







Microsoftのその他の議論は、 Hyper-VとVMwareの比較企業サイトの専用セクションにあります。



しかし、我々がそれを理解しているように、オファーが隠れたコストを運ばなければ、マイクロソフトはそれ自体ではないでしょう。



1. ソフトウェアアシュアランスを支払う場合にのみライセンスを受け取ります 。 2年間のソフトウェアアシュアランスのコストは、製品のコストの50％です。 とにかくあなたは半分を支払う。



2. 新しいWindows Server CALが必要になります ;そうしないと、高価なDatacenterライセンスの意味が失われます。 各ユーザーに1225ルーブルを追加します。



3. Windows Server 2012 R2と比較して、新製品はプロセッサーごとではなく、コアごとにライセンスされます。 かなり控えめな4〜6コアプロセッサを使用する場合、1台の物理サーバーのライセンスの最小数は16コアなので、お金を節約することはできません。 Windows Server 2016の16コアのコストは、Windows Server 2012 R2の2 CPUに匹敵するという事実に安心できます。 しかし、いずれにしても、最低6155ドル（ソフトウェアアシュアランスの場合は3077ドル-現在のレートで200,000ルーブル）を支払う必要があります。



4.各物理ホストのSystem Center 2016ライセンスのコスト（3,607ドル、235,000ルーブル）を計算に追加します。 これがないと、Microsoftソリューションの機能はvSphereの機能とはかけ離れたものになります。



「無料」のWindows Server 2016 Datacenterごとに6,684ドル （435,000ルーブル）の費用がかかることが判明



5.意図の重大さを確認するために、明らかに、移行は許可されたパートナーによって実行される必要があり ます （ 移行プロセスを開始するためにパートナーを関与させます ）。



6. VMwareライセンスなどの高価なソフトウェア資産を放棄することでどれだけ失うかは考慮されません。 もちろん、製品の古いバージョンを使用し、テクニカルサポートを延長しない場合、資産は毎日減価償却されます。 ただし、VMwareライセンスが長い間提供されており、サポートに対してのみ料金を支払っている場合、その利点は再び明白ではなくなります。



7.コメントから。 VMwareからHyper-Vに移行して移行することはできません。 新しいサーバーとストレージの費用を計画します。

ホスト障害クラスターは、クラスターでアドミッションコントロールポリシーをどのように構成しますか？ 一人のホスト？ Windows Serverをインストールし、魅力的な移行プロセスを開始します。ここでは、運が良ければ、ホストは古いクラスターに分類されます。 この計画はありますか？ したがって、予備のサーバーの数は不要ではありません。 さらに百万を投げます。

ストレージの空き容量はどれくらいですか？ 魔法によって、VMFSをNTFSに変えることはできません。 新しいLUNが必要になる可能性があります。移行計画では、未使用のデータストアを順次削除して、スペースを解放することを検討する必要があります。



VMware製品を放棄してKVM Hyper-Vに切り替えることを真剣に考えている場合、Windows Server 2016の新しい（そして非常に価値のある）機能が会社で求められているのであれば、このオファーにより予算の約25-35％を節約できます。



しかし、無料チーズの最初の投与はネズミ捕りのみであり、マイクロソフトは慈善団体ではないことを覚えておく必要があります。 したがって、目に見える節約の背後には、多数の追加コストがあります。
