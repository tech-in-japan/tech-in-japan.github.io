ニュースとリリース



  • PHP 7.0.0 Alpha 2-2番目のアルファリリースには、新しいThrowableインターフェイスを備えた修正された例外階層が含まれています 。 次のリリースは7月9日に予定されており、最初のベータになるため、新しい機能は追加されません。
  • PHP用Google App Engineのベータ版がリリースされました。AppEngineでのPHPサポートは2年以上にわたってテストモードで利用でき、最終的にプラットフォームの一般提供が発表されました。
  • PHP 5.5のサポート終了-PHP 5.5の最初のバージョンのリリースから2年後、26のアップデートがリリースされました。 PHP 5.5.27のリリースは7月9日に予定されており、これはこのブランチの最後の定期的な更新となります。 その後、年間を通じてセキュリティバグの修正のみがリリースされます。
  • PHP-FIGグループは、新しい標準コンテナインターフェースを提案しました -この提案に対する最初の投票は成功し、標準はドラフト段階に入りました。
  • Zephir 0.7 Betaの待望のリリース -ovrwebにより提出 。
  • PhpStorm Hack言語サポートの遅延
  • PHPPackages.org-人気のあるphpパッケージがランク付けされる新しいリソース。
