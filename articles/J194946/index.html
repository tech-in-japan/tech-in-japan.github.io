Oracle WebLogic Server、Oracle Database 12c、Oracle LinuxはWindows Azureで正式に入手可能

2013年6月に署名されたOracleとMicrosoftの間の契約の一環として、Oracle製品の公式イメージがWindows Azureユーザー（ WebLogic Server 11g / 12c、Oracle Database 12c、Oracle Linux 6.4.0.0およびJava 6/7）に利用可能になりました。 同時に、Java、WebLogic、およびOracle Databaseは、Windows Server 2012仮想マシンとOracle Linuxの両方に基づいて利用できます。







これらの製品のライセンスは、「独自のライセンス」の原則に基づいています。これは、ユーザーがOracle製品の独自のライセンスを使用する必要があることを意味します。 製品サポートはOracleによって提供されます。



Oracle製品の価格とライセンスの詳細な説明は 、公式のWindows Azureポータルにあります。 よくある質問への回答を見つけることができます。
