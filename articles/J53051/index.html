Windows 7 RC1には多くの改善が施されました

本日MSFNでリリースされたレポートによると、MicrosoftはWindows 7のベータフェーズを完了し、現在リリース候補1に取り組んでいます。





最も興味深い変更点は次のとおりです。

  • アプリケーション間の切り替えオプション（Alt + Tab）のAero Peek機能（アプリケーションの切り替え中、カーソルが置かれているウィンドウが強調表示され、残りのプログラムウィンドウが透明になります）。
  • 「Windows +＃」の束にある「ホット」キーの機能の拡張。「＃」は割り当てられたボタンです。 （Windows Vistaでは、ホットキーは単にアプリケーションを起動し、Windows 7では、プログラムを起動するか、複数のドキュメントが開いているIEやWordなどで既に実行されている場合はアクティブウィンドウを切り替えます）。
  • タスクバーの「アプリケーションで開く」機能（任意のファイルをタスクバーに添付できますが、非標準のプログラムを使用して開く必要がある場合は、この機能が役立ちます）。
  • タスクバーの最適化のおかげで、より多くのアイコンがタスクバーに配置されました（ベータと比較して、小さいアイコンと大きいアイコンでは、それぞれ24〜25％と32〜39％増加しました）。
  • マルチタッチをサポートするオンスクリーンキーボード（ハー​​ドウェアと同様に、Shift +文字、その他の標準機能の組み合わせを使用して、スクリーンキーボードを操作できるようになりました）。
  • 右クリックの新しいジェスチャー（タッチパネルのユーザーは、1本の指でアプリケーションをクリックし、2本目の指で画面をクリックするだけでコンテキストメニューが表示されます）。
  • スクリーンセーバーをアクティブにせずに一定時間後にコンピューターをロックする（ベータ版では、「スクリーンセーバー」をアクティブにし、終了するためのパスワードを設定することによってのみ実行できます）。


さらに、システム全体のパフォーマンスが向上し、Windows Media Player、Windows Explorer、およびその他のOSコンポーネントが更新されました。 詳細については、公式ブログをご覧ください 。



Windows 7 RC1がいつリリースされるのかは正確にはわかりませんが、MicrosoftがOSの作業を非常に高速で行っていると言えるようになりました。 したがって、RC1が3月上旬に登場し、RTMが6月末までにリリースされても驚くことではありません。
