最初にどのプログラミング言語を学ぶ必要がありますか？ （ʇdıɹɔsɐʌɐɾ：ɯǝʚɯоņıqнqvиʚɐdu）

多くの人は、夜遅くにGoogleを開くことでプログラミング学習の旅を始めます。 通常、彼らは「学ぶ方法...」のようなものを探しています。 しかし、そのような決定を求める人は、どのプログラミング言語を選択するかをどのように決定するのでしょうか？



大手コンピューター会社のサイトやブログを読んだ人は、次のように主張しています。「シリコンバレーでは、Javaは言葉ではありません。 これが必要だと思います。」 これは次のように起こります。「Haskell。 彼は人気のピークです。 次に何が起こるか考えるのは怖いです。 間違いなくハスケル。」 たとえそうであっても：「Goロゴのこのgopherはとても甘いものです。 Goについて知りたい。」



一部の人は、プログラミングの方法を学びたいという欲求に駆られて、「最初に学ぶプログラミング言語は何ですか？」を探しています。



頻繁に尋ねられる質問があり、それらに答えるために、スキーム全体が作成されます。 たとえば、そのうちの1つは、 このサイトのチームによって準備された、最初のプログラミング言語の選択専用です。

このスキームを見ると、例えば、Rubyは子供の頃に粘土から彫るのが好きな人に適していることがわかります。



最初のプログラミング言語の選択は、「クエンティン・タランティーノの登場人物は誰ですか？」のような半真面目なアンケートのように、刺激的な経験になる可能性があります。



しかし、Rubyの研究に真っ向から取り組む前に、幼少期に粘土なしでは生きられなかったという理由だけで、最初のプログラミング言語が非常に重要であるという事実に注目してください。 あまりにも彼に依存しています。



選択した言語を少なくとも一般的に扱うには、数百時間の練習が必要です。 したがって、かわいいロゴと独創的なスキームはあなたを混乱させるべきではありません。



最初の言語を選択することは、次の要因を冷静に評価する価値があります。



  • 労働市場。
  • 長期的な言語の視点。
  • 言語を学ぶことの難しさ。
  • 勉強の過程で正確に何を作成でき、他の人を見せて、モチベーションをサポートします。


年に数回、新しいプログラミング言語が登場します。 科学雑誌にそれらについての記事が書かれており、ウェブ上で見つけられる漫画はそれらに捧げられています。









最初のプログラミング言語の選択に関しては、さまざまな選択肢に直面する必要があります。 検索を少し絞り込むために、プログラミングに関する学習に関する過去12年間のGoogle検索の分析を以下に示します。







検索クエリの分析から、Javaに浮き沈みがあることがわかります。 また、たとえば、Pythonの評価は徐々に成長しており、ほとんどの最も人気のあるプログラミング言語になりました。 ここには1つの言語があります。シンプルですが、驚くべき機能を備えており、関心のあるグラフは年々ゆっくりと成長していますが、最下部にあります。 これはJavaScriptです。



これらのプログラミング言語について話す前に、何か明確にしましょう。



  • 特定の言語が他のどの言語よりも客観的に優れていると主張するつもりはありません。
  • 開発者は最終的に複数の言語を学ぶことに同意します。
  • 私は、最初の段階でプログラマーが1つの言語をよく勉強する必要があるという事実に賛成です。 そして、おそらくご想像のとおり、タイトルの巧妙に偽装されたツールチップを読んだ後、この最初の言語はJavaScriptでなければならないと思います。


私たちは、今日の人々がどのようにプログラミングを学ぶかを見て、第一言語の議論を始めます。



情報技術とプログラミングトレーニングの基礎









大学は通常、情報技術の科目のフレームワークでプログラムするように教えられています。これは、数学コースへの追加のように見えるか、電気工学などの特定の主要な専門分野に結びついています。



エリック・レイモンドによる次の声明はご存じでしょう。「コンピューター技術の分野での教育は、筆や絵の具を勉強しても偉大な芸術家にはならないのと同じように、優秀なプログラマーになるわけではありません。」



今日、以前と同様に、ほとんどの教育機関はプログラミングと情報技術の間に等号を付けており、情報技術は数学と同一視されています。



その結果、多くの入門プログラミングコースは、Cなどの言語、またはMATLABなどの数学パッケージで使用される言語のかなり低レベルの抽象化を中心に構築されています。



そして、プログラミングコースで何を教えるかを決定する人は、通常、言語の評価を含むレポートを定期的に発行するあらゆる種類の公式ソースに従います。 そのようなレポートの中で、例えば、 TIOBE Index 。 そして、これがIEEEリーダーボードです。









このような「リーダーボード」は、10年前とほぼ同じように見えます。 ただし、学界であってもすべてが変化しています。



そのため、2014年、PythonはJavaをバイパスし、米国の情報技術の主要なコースで最も人気のあるプログラミング言語になりました。









そして、私は別の変化が間違いなく起こると確信していますが、私が言えること-これはすでに起こっています。



実際のプロジェクトを実行するプログラマーが使用する言語を見ると、まったく異なる状況がわかります。



以下は、2016年にStack Overflowで実施された調査の結果です。 49397人の開発者が参加しました。 それらの半分以上がJavaScriptを使用しています。







JavaScriptなしではWebインターフェースの開発は不可能であり、最近では、Node.jsのおかげで、サーバーコードでのJavaScriptの関連性が高まっています。 この言語は、ゲーム開発やモノのインターネットなど、他の分野に急速に広がっています。



プログラマーの仕事の中で、Javaを考慮に入れない場合、JavaScriptは他の言語よりも頻繁に言及されます。 最大のジョブアグリゲーターであるIndeed.comからのデータは次のとおりです。







freecodecamp.comで 、オープンソースJavaScriptプロジェクトに携わる開発者のコ​​ミュニティを集めたことは偶然ではありません。 過去2年間で、5,000人以上がFree Code Campを使用してプログラマーとしての最初の仕事を見つけました。



ところで、私は彼にプログラミングの方法を教えるという理由だけでJavaScriptを提唱していません。 すべてが正反対です。 JavaScriptの知識はプログラマーの最初の仕事への正しい方法だからです。



しかし、JavaScriptはあなたに適していますか？ 彼はあなたの最初のプログラミング言語になるべきですか？ 上記で話した言語選択の要因をさらに詳しく考えてみましょう。



要因1：労働市場



純粋な好奇心と知的エンターテイメントへの愛からプログラミングを学んでいるのであれば、このセクションはスキップできます。 しかし、他の多くの人々のように、習得した知識とスキルの助けを借りて仕事を見つけるためにプログラミングを勉強する場合は、ここで説明する内容に注意を払う必要があります。



前にも言ったように、求人情報はほとんどの場合Javaに言及しています。 2番目はJavaScriptです。



ただし、JavaScriptの場合、考慮すべき重要な点が1つあります。 この言語は20年前から存在していましたが、比較的最近になって、Netflix、Walmart、PayPalなどの巨人が本格的なアプリケーションを作成するための深刻なツールになりました。



その結果、多くの企業がJavaScript開発者を雇用していますが、開発者はそれほど多くありません。



Indeed.comからのデータは、求職者に対する空席率を示しています。









だから私たちが見るもの。 Java開発者には、2.7プログラマーが1か所にいます。 PHPの知識が必要な求人分野やiOS開発環境でも、同様の状況が発生しています。



JavaScriptでデータを見ると、空席ごとにプログラマが0.6人しかいないことがわかります。 JavaScript開発者にとって、これは間違いなく売り手の市場です。

GitHubのJavaScriptプロジェクトは、平均して、Java、Python、またはRubyプロジェクトの2倍のプルリクエストを受け取ります。 さらに、JavaScriptは他の一般的な言語よりも速く成長しています。

たとえば、100人以上の開発者がオープンソースプログラミング言語TypeScript（静的型付けを備えたJavaScriptの拡張バージョン）に貢献しました。 彼らの多くは、この仕事の報酬を受け取るマイクロソフトとグーグルの従業員です。

要因3：言語学習の難しさ



XKCDのプログラミング言語の複雑さの似顔絵です。







ほとんどのプログラマーは、高レベルのスクリプト言語は比較的簡単に習得できることに同意するでしょう。 JavaScriptは、PythonとRubyとともにこのカテゴリに分類されます。



また、教育機関では、プログラミングの方法を習得し始める言語はまだJavaやC ++などですが、それらを理解することははるかに困難です。
