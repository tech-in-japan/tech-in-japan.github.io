モスクワでのAzure Day Conference

Azure Dayは、Microsoft MVP（Most Valuable Professionals）コミュニティがホストする年次会議です。 Azure Dayの中心的なアイデアは、最新かつ最も価値のあるAzureクラウド情報をITコミュニティと共有することです。 このレポートは、開発およびビジネスソリューションの第一人者であるMVPによって提示されます。MVPは、マイクロソフトテクノロジ、ヨーロッパおよびロシアのマイクロソフトの専門家と日々協力しています。











Azure Dayは、誇張することなく、Azureがどのように開発されているかを学び、この方向性の開発に直接関係する人々に質問する貴重な機会です。 指先で紺Azure！







IT専門家や経営幹部をイベントに招待します。



プログラム内：



  • Azureサービスを使用した開発への深い没入：ボット、データベース、IoT、AI / ML
  • Azureインフラストラクチャの操作の概要と例：仮想マシン、ネットワーク、DevOps、セキュリティ
  • Microsoftテクノロジを使用したビジネスプロセスの最適化と構築の例：Office 365、Dynamics、Power BI。


イベントのウェブサイトでの詳細と登録-www.azureday.ru
