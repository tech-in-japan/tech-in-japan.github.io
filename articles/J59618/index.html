Windows 7.ディスクの書き込み

Windows 7（RC 7100）をインストールしたので、イメージの作成や書き込みなど、CD / DVDの書き込み用のプログラムには興味がありませんでした。 Windows XPとWindows Vistaの両方で使用したRoxio Creatorに依存しました。

したがって、ある晴れた日、適切なタイミングで、Win7のRoxioが機能せず、さらにシステムが完全にフリーズし、リセットボタンのみが役立つことに気付きました。 インターネットからダウンロードしたmilcore.dllライブラリをsystem32にコピーするという形で解決策を見つけました。 助けにはなりませんでした。 同じ結果。

さて、 Roxioをしばらく忘れてしまい、 Neroをダウンロードし、トライアルキーを取得してインストールし、結果を驚かせたことがわかりました。システムは長い間考えた後、思わずリブートしました。 Neroのソリューションを探していませんでした。



彼は友人にこの問題について不満を述べ、彼らがこの問題を解決することにさえ興味がなかったが、古いOSに愚かに戻ったと知ったとき、非常に驚​​いた。



Googleで大騒ぎして、私はいくつかのプログラムを見つけました。1つはディスク書き込み用（ BurnAware Free ）、もう1つは画像（ ImgBurn ）でうまく動作します。 私はこれらのソフトインクも疑っていませんでした。 彼らと半日突っついて喜んだ。



プログラムは無料です（BurnAwareには制限があります）

そして今、スライド（古いジョークのように）：



バーンウェア無料



複雑なことは何もありません。 3種類のディスク（CD、DVD、ブルーレイ）、それぞれ独自のウィンドウ、同じ機能



ファイルを選択するウィンドウ（私の場合は、データディスクのみ）



ロシア語が存在する



イムバーン

すべてがどれほどシンプルで明確であるかに注目してください



そして、たくさんの設定があるウィンドウ



ロシア語が存在します（すべてが翻訳されているわけではありませんが、プログラムのダウンロードページで入手できます）。



一般的に、私は満足し、大きな利点しかありませんでした。

無料（重要！）;

スライドショーの作成、メディアコンテンツの視聴者（プレーヤー、写真を表示するプログラムなど）の作成など、不必要な「機能」はありません。

直感的なインターフェース、それ以上のものはありません。

最も重要なWindows 7で動作します。



機能、設定をペイントしていないことをおizeびします。 メーカーはそのようなレビューに対処すべきだと思います。



書いた理由

ソフトウェアをフラッシュしないリソースを見たことはありません。

Windows 7のソフトウェアのリストを見たことがない 「持っている必要があります」;

私は、ディスクの書き込みでこのような状況に陥った人はいないと思います。



自分以外の人を助けてくれたらと思います。 私はプログラムのレビューを書いたことがないので、私の意見を正しく理解してください。
