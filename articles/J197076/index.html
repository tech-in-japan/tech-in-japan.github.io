フランケンパッド：現象の歴史

今日、私の記事のトピックの犠牲者は、私のコレクションからの特定の展示物でも、ただの興味深いコンピューターでもありません。 今回は、ThinkPadブランドのファンの間での標準的ではない現象についてお話します。これは、他のブランドの支持者の間ではほとんど発生しませんでした。 これにはおそらくいくつかの理由があります-
  1. これらのコンピューターのユーザーのフレンドリーで好奇心の強いコミュニティ
  2. 1つの設計に基づいた定規の比較的長いリリースですが、塗りつぶしが異なります
  3. 小さな変更に便利
  4. この理由は最もバラ色ではありません-新しいモデルでは、以前のモデルのキラー機能が失われたことがあります




長い間、ユーザーは自分の都合の良いデバイスをメンテナンスする機会を利用し、その「詰め物」を（原則として、根本的にではなく、顕著に）更新していました。 はじめに、私は1995年から1996年までを取りますが、それについては以下で詳しく説明します。 後に名前が付けられました-2004年、モデル770ラップトップの非標準アップグレードに関連して、ユーザーコミュニティフォーラム（ forum.thinkpads.com ）でFrankenPad （複数の人々「Frankenstein」の一部のモンスターの作成者の名前とラップトップの名前から）という単語が初めて使用されましたしかし、2011年になってようやくこの言い回しが広まり始めました。 基本的に、この言葉はT60シリーズのモデルをT61に更新することに関するものでしたが、T4x / R5xラインでのマザーボードの移植に関してもよく使用されていました。



明確にするために、一部のモデルではアップグレードキットがIBM自身によって提供されたことに注意します。 フランケンパッドは、非公式のアップグレード、独立した、またはサードパーティ企業によるアップグレードの結果のみを呼び出すことに同意します。



現象の歴史を深く掘り下げて、独自の「モンスター」を構築するための可能なオプションに精通することをお勧めします。 私の忠実なアシスタントイゴールはこれを手伝ってくれます。 ハハハ 。 日付として、フランケンパッドの作成の基礎となるモデル年を使用します。



1995年。モデル701C / Cs「バタフライ」




ThinkPadブランドの象徴的でおそらく最も有名なモデルが1995年に登場しました。 超ポータブルラップトップのクラスに属するサイズのバタフライは、フルサイズの画面（当時はラップトップに10.4インチを超えるマトリックスはインストールされていませんでした）と、コンパクト化のために折りたたまれたキーボードを提供しました。 同時に、ラップトップはその時点で最大に近い容量を受け取りました-i486-DX4 / 75プロセッサと最大構成の最大40メガバイトのRAM（最上位モデルには同じプロセッサがありましたが、周波数は100 MHz）。 内蔵ドライブがないため、ラップトップの重量は1.7 kgで、搭載されたThinkPad 755CD CD-ROMよりも1.5キログラム以上軽くなっています。



しかし、6か月後、最初のPentiumベースのモデルが登場し、すべてが好転しました...

Frankenpadと呼ばれる最初のラップトップは、その基礎に基づいて作成されました。 しばらくしてこれらのマシンを更新するためのボードを生産し始めた後、その名前が忘却に沈んだサードパーティの会社（たぶん1つの会社ではないかもしれません）。 AMD 5x86-133プロセッサー（P-レーティング75、つまりPentium 75のパフォーマンスに対応）がボードにインストールされ（ほとんどの場合、変更および復元されたIBM製マザーボードでした）、メモリ容量は最大72（64 + 8）メガバイトまでサポートされていました。



このような更新（およびその他の非標準の更新）を実行するには、特定のスキルが必要でした-ラップトップをほぼ完全に分解し、ケースの2つのメインボードの下部を交換する必要がありました（一方はシステムロジックを備えたプロセッサー、もう一方はVRM、オーディオコントローラー、その他の重要な詳細を備えていました） 。



比較的大きなブレークが続きます-モデル755、760、380、および560は、この観点からは興味深いものではありませんでした。 一般に、原則として、フランケンパッドの作成には3つの大きな関心が寄せられました。前者は他のものよりも弱く、後者は最も強力です。 だから...



1998-2000年。 770 / E / ED / X / Zシリーズ




1997年に発表された770は、印象的なサイズを持ち、7シリーズの最後のモデルであるという点で注目に値するだけではありませんでした。 これらの強力なマシンは、DVD-ROMを搭載した最初のラップトップであり、オプションにはアナログビデオ入力、AC-3デジタルオーディオ出力、MPEG2エンコーディング/デコーディングアクセラレーションボード（3つのオプションはすべて1つ-DEVAカードによって提供されました-デジタルおよび拡張ビデオアダプタ）、解像度1280x1024のユニークな13.7インチスクリーン。しかし、同時に最古のモデルはPII-366のみを受け取り、その後ラインは閉じられました。機能が同様のマシンはわずか6か月後に登場し、770シリーズは徐々に価格が下がり個人所有になり始めました。ビジネスユーザー 新しいラップトップを購入し、古いハードウェアで危険な操作を行いません。



そして、ここにさまざまな形のフランクネーディングが登場しました。 これらの最初のものが何であるかは言えませんが、単一構造のオプションのリストはたくさんありました。

1）770Eから770にビデオコントローラーをインストールします。X/ Zには既にAGPバスにあり、最初の2つのモデルはPCIを使用しました

2）Pentium IIプロセッサをPentium III（770X / Z）に置き換える-これらのラップトップはMMC-2パッケージのプロセッサを使用し、CPUを最大P3-850までインストールでき、いくつかの操作（モジュール上のプロセッサの再はんだ付け、場合によってはオーバークロック）が可能- 1 GHz！

3）メモリ操作-ラップトップには3つのメモリスロットがあり、そのうちの1つは完全に分解され、工場から32メガバイト（770 / E）または64メガバイト（770X / Z）のRAMが含まれています。 ラップトップの作成に基づいた440BXチップセットは、4スロットで最大1ギガバイトのメモリをサポートでき、理論的にはこのラップトップに768メガバイトをインストールできます。 しかし、BIOSのバグにより、最大量は512メガバイト-128 + 128 + 256または0 + 256 + 256です。



1998-2001年。 600 / E / Xシリーズ




史上最も巨大なラップトップの1つは2001年までリリースされました。 かなりのコストにもかかわらず、その人気は非常に大きかった（正確なデータはありませんが、適切な構成のための5〜6千ドルが標準でした）。 多くのバージョン、長寿命、人気があることを考えると、これらのマシンは単に変更に失敗することはありません。 基本的に、すべてはプロセッサに関するものでした。 メモリのアップグレードは通常の手順でした（最大576メガバイト64 + 256 + 256）。 770の場合のように、プロセッサーをMMC-2形式（すべてのバージョンではなく、最初のものはMMC-1に設定されていません）で最大850 MHzまでインストールでき、モジュール自体を変更する場合は最大1000までインストールできます。



これらの2つのモデルは、フランケンパッドの作成の第2の波を決定し、この言葉が初めて使用されたことが770に宛てられました。

それからまた長い休憩がありました。 休憩について話してください、私はモデルラインに関連する休憩について話しています-当時は休憩がありませんでした-一部のユーザーはまだ770、600のバタフライをアップグレードします。 はい、2つのモデル間でディスプレイを移動したり、古いケースに新しいスタッフィングを移動したりできました。 制限は多くありますが、意味はありません。 しばらくの間...



最初は、ウルトラポータブルX31 / X32ラップトップにIPSスクリーンをインストールすることが一般的になりましたが、それでも普及していませんでした。 この概念はここではあまり適切ではありませんが、法案は数千から数百、まれに数千万のコンピューターがリリースされることになります。 現在、IPSからIPS-LEDへのA31p、T42p / T43p、T60pの画面更新があります。



2003-2006年。 モデルラインT40-T43、R50-R52。




これらの車について詳しく語ることはできませんが、これはそれほど興味深いトピックではありません。基本的には修理になります。 Rシリーズのボードは安価で、RシリーズのドナーもTシリーズの場合に若干の修正が加えられています。さらに、T43までは、Tシリーズの第40ラインに統合チップセットグラフィックスを搭載したモデルはありませんでした。 バッテリー寿命と信頼性を高めるためにボードを移植することもありました（ビデオチップはnVidia 8400/8600シリーズに落ちました）。



T43 / R52に特有のもう1つの開発オプションは、いわゆる「SATA Mod」です。これらのラップトップでは、ハードドライブにIDEインターフェイスがあり、特に高速が必要な場合、交換品を見つけるのは困難です。 この場合、ディスクは実際にインターフェイスコンバーターを介してSATAチャネルに接続されます。 ファイナライズすると、コンバーターがオフになり、ディスクを接続するためのコネクターが変更され、新しいコネクター用の追加配線が敷かれます。



2006-2008年。 T60 / T60pおよびT61 / T61pモデル




現在の最新モデルであり、おそらく最後の一般的なモデルであり、大幅に改善されているのはT60 / T60pモデルです。 具体的には、15「4：3のアスペクト比を持つ画面のバージョン」です。これは、15インチのモデルの中でこのアスペクト比を持つ最後のThinkPadモデルでした。 これらのマシンは、優れた色再現と広い視野角を持つIPSスクリーンをインストールしました。 T61シリーズには、このようなスクリーンはもうありません。 しかし、より強力なプロセッサー（T7600の代わりにT7800、T9500、またはX9000まで）、はるかに多くのRAM（T60の3つとより強力なビデオに対して最大8ギガバイト）が統合および個別にありました。 確かに、2008年8月までnVidiaビデオチップの信頼性に問題がありましたが、それは別の話です。 T61ボードを使用してT60を改良した後、今日まで困難なタスクを実行できるマシンが得られました。 将来的にはわずかなマージンさえあります。



画面は必ずしもT60rから取られているわけではありません-より興味深いオプションが可能です。 主なものは、T60のケースとT61のボードです。

最大構成は次のようになります。

コンピュータファッションの最後のきしみではありませんが、非常に価値があります！



説明されているオプションに加えて、確かに他のよりエキゾチックなオプションがあります。 Xシリーズのラップトップとトランスフォーマーのオプションについて聞いたことがありますが、おそらくボードまたはスクリーンの再配置に限られていました。



結論の代わりに


新しいラップトップでは、これは機能しません。 しかし、それは単に必要ではないのでしょうか？ 現在、IPSディスプレイ、SSD、高速プロセッサ、ビデオカード、メモリがあります。 アスペクト比4：3の多くの人にとって便利な画面があるだけでなく、その他のユーザーフレンドリーな、しかし見た目があまり良くないクラシックキーボード、ThinkLight、実際のマウス代替ボタンは徐々に消えています...

フランケンパッドで時代は完全になくなるかもしれませんが、そうでないことを願っています。 少なくとも彼女の精神は、私自身とブランドの他のファンを救おうとするでしょう。

ご清聴ありがとうございました！ お楽しみに！
