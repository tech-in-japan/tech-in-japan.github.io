10.62 RC

安定版ブランチを更新する時が来ました 、そして今日新しいビルド10.62をリリースしました 。 以下にリストされている変更は、10.61リリースとの比較用です。



これはリリース候補であるため、開発者は10.61からのリグレッションが見られるかどうかを知らせ、10.70から欠落している修正はリグレッションではないことを思い出させます。



また、プラットフォームごとにアセンブリ番号が異なることにも注意してください。 10.70以降、単一のアセンブリ番号が引き続き使用されます。 また、これは10.6xブランチであるため、PowerPC用のビルドがまだあります。



ダウンロードする

  • Windows MSI / Windowsクラシック
  • Mac（ユニバーサル） / Mac（Intelのみ）
  • Linux / FreeBSD
