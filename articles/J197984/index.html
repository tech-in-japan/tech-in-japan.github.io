Windows PCの「JavaFX入門」でOracleが提供した例のHelloWorld。 CSSを使用してログインおよびパスワード入力ウィンドウを変換し、FXMLでフォームを作成し、FXMLフォームでCSSを使用します。 繰り返しますが、チュートリアルで説明しなかったコマンドラインと微妙な点です。







JavaFX入門の最初のセクションでは、 ここでコマンドラインを使用して取得できる最も単純なJavaFXアプリケーションを作成する方法について説明します。 2番目のセクションでは、ログインおよびパスワード入力ウィンドウを作成することを提案します。 ここで最初のアプリケーションから作成されます 。 その結果、コードのサイズがわずかに増加しました。

さて、チュートリアルの4番目のセクションはFXMLです。 プログラムコードは非常に変更されるため、再度書き直すのが簡単です。 さらに、GetStartFXMLなどの隣接フォルダーにプロジェクトのコピーを作成します。 そこで、すでに5つのバッチファイルをコピーし、outおよびsrc \ helloworldフォルダーを作成し、ソースを最後に配置する必要があります。

HelloWorld.java 
 package helloworld; import javafx.application.Application; import javafx.scene.Scene; import javafx.stage.Stage; import javafx.scene.*; import javafx.fxml.*; public class HelloWorld extends Application { public static void main(String[] args) { launch(args); } @Override public void start(Stage stage) throws Exception { Parent root = FXMLLoader.load(getClass().getResource("fxml_example.fxml")); Scene scene = new Scene(root, 300, 275); stage.setTitle("FXML Welcome"); stage.setScene(scene); stage.show(); } }


次に、4番目のセクションに散在するファイルfxml_example.fxmlを収集する必要があります。 それからのデータはCSSファイルからだけでなく、オンザフライで取得されるため、それを使用するHelloWorld.classファイルのすぐ隣に作成する価値がありますが、まだ存在していません。 つまり、out \ helloworldでは、最初のコンパイルの前に、自分自身でサブフォルダーを作成します。
