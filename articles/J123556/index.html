第二に、文学。 いくつかのオプションがあります：本を購入する、それらの電子版をダウンロードする、または友人/同僚/知人から借りる。 書籍を購入する場合は、denyuzhkuを利用してOzon.ruまたは通常の書店（できれば大型のコンピューター店にあるTyk）に行きます。 これらの書籍の電子版はインターネット上にあり、添付のディスクの画像はトレントトラッカーにあります。 Googleが助けになります。



第三に、練習。 繰り返しになりますが、2つのオプションがあります。 ファンタジーに富んだ物理マシンまたは仮想化です。 物理マシン（2〜3台以上のPC）を使用する機会がある場合、これは素晴らしいことです。 しかし、そのような機会が常にあるとは限りません。 ここでは、仮想化プログラムが役立ちます。 そのうちの2つは、MicrosoftのVirtual PCとOracleのVirtualBoxです。 仮想化プログラムを実際のトレーニングに使用する場合、それほど強力ではない物理コンピューターが必要になります。できれば、デュアルコア以上のプロセッサーと2〜4 GBのRAMが必要です。 「私の愛する！ 鉄はありますが、どこでソフトウェアを入手できますか？」 そして、私はあなたに答えます：「急流を離れて！ マイクロソフトが私たちのすべてです！」 リンクをたどる -> Windows Server 2008 R2のリストから選択する->ダウンロード->バージョン（ここでは1つ：x64またはItanium）を選択-> Windows Live ID資格情報を入力（または登録）->配布イメージをダウンロード->利益！ 試用ライセンスは、アクティベーション後180日間有効です（シリアル番号は不要です）！ 一般的に、非常に便利なリソースであり、ブラウザのお気に入りに追加する価値があります。
