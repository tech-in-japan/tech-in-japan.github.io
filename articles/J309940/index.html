Azure Active Directoryが新しいARMポータルに追加されました

ここ数週間は、Microsoft Azureインフラストラクチャサービスに関する良いニュースが非常に豊富です。 仮想ネットワーク用のこの新しい機能ピアリングと、 バックアップサービスの管理に関連する改善。 昨日、Azureユーザーがおそらく今年の後半に待っていたというニュースがありました。







Azure Active Directory管理が新しいARMポータルで 利用可能になりました。 確かに、これまではプレビューのみで、一部のタスクについてはClassicポータルに戻る必要がありますが、開発者は明確に自分の位置を示しています-新しい機能はARMポータルにのみ表示され、古い機能はすべてすぐにここに転送されます。 カットの下のいくつかの詳細。



概要



最初のパネルは、Azure Active Directoryの概要です。









ここで、クラウドディレクトリサービスに関する基本情報を取得できます。 ここでは、ユーザーやグループの管理などの一般的なタスクにすばやくジャンプできます。 ここから、セルフサービスを設定してパスワードをリセットしたり、オンプレミスのActive Directoryをクラウドサービスにリンクしたりできます。 後者といえば。 このようなパネルを整理する現在のアプローチに従って、ここでAD Connect自体をダウンロードするか、同期ステータスを確認できます。









ユーザーとグループ



間違いなく、主なことは、ARMポータルとクラシックポータルを切り替えることなく、ユーザーとグループを管理できるようになったことです。 私たちはこれをどのくらい待っていました！









ARMポータルで既に利用可能な機能には、ユーザーおよびグループの管理（静的および動的メンバーシップルールを含む）、セルフサービスの管理によるパスワードのリセット、企業向けのカスタマイズの設定などのタスクが含まれます（Premium for Azure Activeが必要なことは明らかです）ディレクトリ）。 また、ユーザーに関連付けられたアクティビティのログ（ユーザーのアクションとアカウントでコミットされたアクションの両方）も利用できます。



用途



また、新しいパネルを使用すると、アプリケーションだけでなくActive Directoryクラウドサービスとの統合を便利に管理できることに注意してください。



おわりに



Azure Active Directoryを管理するためにARMポータルからClassicに切り替える必要があることは、私にとって最も面倒なことの1つでした。 この機能がついに新しいポータルに移行され始めたことを非常に嬉しく思います。 Microsoft Azureの使用を開始していない場合は、それが何であるかを見てみましょう。 さらに、Azure自体だけでなく、Azure AD PremiumおよびEnterprise Mobility Suitでも無料の試用版サブスクリプションを取得できます。 これらの試用版サブスクリプションのアクティベーションには、ARMポータルから直接アクセスできます。
