AppleとMicrosoftは危険な脆弱性FREAKを修正しました

先に、OpenSSLライブラリとMS Schannelソフトウェアパッケージで発見された危険なFREAK脆弱性について書きました。 この脆弱性は、 Apple for iOS 8（APPLE-SA-2015-03-09-1 iOS 8.2）およびOS X Mountain Lion v10.8.5、OS X Mavericks v10.9.5、OS X Yosemite v10.10.2（APPLE-SA-2015）で修正されました-03-09-3セキュリティアップデート2015-002）。







Windows 2003 Server-Windows 10 TPの場合、今週の火曜日にリリースされたパッチ火曜日の一部として、この脆弱性は解決されました。 FREAKは、Schannel.dllライブラリのMS15-031 （Schannelの脆弱性によりセキュリティ機能のバイパスが可能になる）を更新することにより終了しました。 Windows 7および8.1のユーザーの場合、この更新プログラムは自動的にインストールされました（既定のセキュリティ設定）。
