Immerse CMS-エトマイトの復活

少し歴史。



あなたの多くはおそらくMODx CMSについて聞いたことがあるでしょうが、足がどこから成長するかを誰もが知っているわけではありません。



すべては2004年4月に始まり、当時不明だったPHASE CMSが公開されましたが、すぐに名前をETOMITE CMSに変更する必要がありました。 プロジェクト自体は非常に成功しており、多くの愛好家が集まっていました。 当時、システムのコアをより深く掘り下げてリメイクしたいと考えた人が多くいました。その結果、最初のMODxが登場しました。



すぐに、開発者のグループがこのプロジェクトに少し商取引を導入することを提案しました。 その結果、フォーラムは戦場となり、プロジェクトの開発と開発者が銃撃戦に費やしたほとんどの時間（当時、約2500人のユーザーがいた）を大幅に妨げました。

このようなストレスに耐えることができず、開業からちょうど1年後、プロジェクトの作者であるアレックス・バターは、このプロジェクトの開発の中止を発表し、チーム全体を旧型の手に委ねました。

しかし、つい最近、アレックスは手ぶらではなく現れ、蓄積された経験を再考して、彼の新しい頭脳のImmerse CMSを紹介しました。



主な機能

  • アヤックス
  • 多言語
  • ビルトインSEO（キーワードおよびメタタグ管理）
  • ネイティブCNCサポート
  • グループ化する機能を使用して、サイト上のユーザーを管理します
  • 統計

    • ブラウザ
    • 訪問者
    • 入口/出口
    • 検索エンジンとキーワード
    • 参照ページ
  • テンプレートシステム
  • W3C標準と100％の互換性
  • 埋め込みファイルマネージャー
  • WYSIWYGエディターのサポート
    • Xstandard
    • FCKEditor
    • シンハ
    • 編集する
  • APC（代替PHPキャッシュ）をサポートするキャッシュシステム
  • API
  • などなど...




私が気に入ったすべての中で、写真の上に影、フレーム、その他のレイヤーと効果を追加できる、写真用の強力でユニークなテンプレートジェネレーターに注目したいと思います。 内部メッセージを送信するためのシステム、フォーラムサポートも利用できます。

サーバー要件：
  • PHP 4.3。*以降
  • MySQL 3.23。*以降
また、傷つけない：
  • mod_rewriteをサポートするApache
  • 代替PHPキャッシュ（超高速キャッシュ用）
ブラウザサポート：

  • Firefox
  • オペラ
  • サファリ
  • Internet Explorer（7以降）


さて、最後のリンク

  • Immerse CMS Webサイト
  • スクリーンショット
  • ドキュメンテーション
  • ダウンロードする
