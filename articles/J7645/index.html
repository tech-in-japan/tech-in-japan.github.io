MicrosoftはWindows Home Serverを愛する

Windows Home Serverは、Microsoftが私たちのために描いた未来像の概念的に重要な部分です。 レドモンドの巨人にとって、ホームサーバーとそれらの「OS」を、テレビや電話のような家の普通のアクセサリーとして作ることが非常に重要です。 新しいプラットフォームは、ライフブログなど、次世代の根本的に新しいサービスを社会に展開するための便利な環境になります。



しかし、ホームサーバーが本当に必要だと人々に説明する方法はありますか？ 大衆意識の中で未来の正しい絵を描くには？ これは簡単な作業ではありません。 さらに、現在、ホームサーバーは、バックアップを除いて、することはあまりありません。



最高のジャーナリストは、実際に何をするのかを詳細に示すために働いています 。2007年4月にリリースされたCommunity Technical Previewのレビューと多数のスクリーンショットをご覧ください。



Windows Home Serverは、Hewlett-Packard MediaSmart Serverなどのホームサーバーにインストールするためのシステムです。



ソフトウェアのインストールは、7つの簡単な手順で進みます。







次に、登録番号を入力します。 これが行われない場合、プログラムは30日間のみ動作します。







次の段階で、コンピューター名が示され、メインパスワードが設定されます。 その後、サーバーは更新センターに接続し、新しいドライバー、セキュリティ更新プログラム、ソフトウェアコンポーネントなどを含む必要なモジュールを交換します。







サーバー設定には、接続されているコンピューターとハードディスク、共有フォルダーのリストを表示するためのアイコンがあります。







アカウント管理では、どのユーザーがサーバーへのリモート接続を許可できるかが示されます。 管理者は、ユーザーのパスワードの複雑さも設定します。







デフォルトでは、すべてのNTFSパーティションが自動的にバックアップされます。 システムは一時ファイルを認識し、それらをアーカイブしません。







バックアップがスケジュールされています。







音楽、ビデオ、写真を含むフォルダーが自動的に作成されます。 それらは簡単に共有でき、Xbox 360ゲームコンソールなどの他のデバイスがコンテンツにアクセスできます。











すべてのホームサーバーは、インターネットを介して「世界に公開」できます。 ドメイン名は、Windows Live登録サービスに自動的に登録されます。 サイトへのリモートアクセスはパスワードでのみ可能です。パスワードは最も強力なレベルのクラッキング抵抗でなければなりません。
