夏は終わりましたが、新しいバージョンは終わりません！ バージョン1.7.3.0。 -さらに多くのビジネス管理の機会！

新しいシーズンの始まりは、Simple Businessシステムの新しいバージョンのリリースによってマークされました。 リリース1.7.3.0には、次の機能が追加されました。

  • プログラムのユーザー向けのマルチチャット。
  • ドメインprostoy.bizの組み込みメールと、他のメールシステムのメールボックスを接続する機能。
  • Windows 8スタイルのプログラムの設計。
  • crmテーブルのセールスファンネルとスマートヒント。
  • システムウィンドウからプログラム機能に関するトレーニングビデオを直接表示する。
  • 新機能に関する社内ニュースレター。


プログラムユーザー向けのマルチチャット

マルチチャットの助けを借りて、チームとプロジェクトまたはタスクの同時ディスカッションを簡単かつ迅速に整理できます。 [通信]項目を使用して、[連絡先]タブで新しいマルチチャットを作成できます。 コンテキストメニューで[新しいマルチチャット]を選択する必要があります。



次に、従業員をマルチチャットに追加する必要があります。 招待を承諾すると、マルチチャットウィンドウが開きます。 従業員はメッセージを送信し、さまざまな期間（「昨日」、「7日間」、「30日間」）のメッセージ履歴を表示できます。 マルチキャッチへの参加と離脱には、対応するシステムメッセージが伴います。



ドメインprostoy.bizの組み込みメール

これで、Simple Businessシステムの新しいユーザーはそれぞれprostoy.bizドメインのメールボックスを受け取ります。 Simple Business電子メールの機能をテストし、必要に応じて、他のメールサービスからメールボックスを接続できます。 メールプログラムの標準機能に加えて、Simple Businessメールは次のような幅広い追加機能を提供します。

  • 任意の数の従業員の通信に関するチームワーク。
  • 受信メッセージをタスクとタスクに変換する;
  • ビジネスプロセスに応じた着信レターの処理の柔軟なカスタマイズ。
  • 知識ベースからの応答テンプレートの使用。
  • 他の通信手段の履歴とともにメッセージ履歴を保存します。




「メール」タブのアンケートで個人用のメールボックスを作成できます。 組織の場合、「メール」タブの組織ウィンドウでメールボックスを作成できます。 一般ユーザーおよび一般組織の場合、prostoy.bizドメインで使用できるメールボックスは1つだけです。 ドメインprostoy.bizおよび他のドメイン上の追加メールボックスの接続は、「Professional user」または「Professional organization」という関税を接続するときにユーザーが利用できるようになります（ 関税についての詳細 ）。



Windows 8のスタイルでプログラムを設計する

プログラムのデザインは、Windows 8のファッショナブルで明るいスタイルに変更されました-最小限の線と境界線のある明るい色です。 素敵なデザインは作業をスピードアップします



セールスファネルとスマートなヒント。

セールスファンネルは、ヒストグラムとして表示されるユニバーサルテーブル（CRM）に追加されました。 上部パネルの対応するボタンをクリックして、ファンネルを表示または非表示にできます。



目標到達プロセスの目的の段階を強調表示することで、販売プロセスの特定の段階にいる顧客のリストを表示できます-テーブルには、この段階にある顧客が表示されます（KPの送信、販売など）。

セールスファンネルを使用すると、次のことができます。

  • 計画の実施を予測する。
  • 営業部門の仕事を調整します。
  • 従業員の職務の質を監視し、「人員バラスト」を特定します-他人が効果的に働くのを妨げ、自分自身が効果がない従業員。
  • 特定の販売者、製品、方向に対して最も効果の低い販売フェーズを特定し、特定の販売フェーズの効果を最適化します。


crmテーブルには、スマートヒントを追加する機能も追加されました。 ヒントを使用して、テキストを入力するときに、このフィールドに以前に入力したリストから目的の値を選択できます。



「文字列」タイプのフィールドにインテリジェントなツールチップを追加できます。フィールド設定ウィンドウの「スマートツールチップを使用」チェックボックスをチェックするだけで十分です。



システムウィンドウからプログラム機能に関するトレーニングビデオを直接表示する

これで、プログラムウィンドウから直接、興味深いシステム機能に関するトレーニングビデオを見ることができます。 疑問符アイコンをクリックして目的の項目を選択すると、短いビデオをダウンロードして視聴し、目的の機能をより早く知ることができます。





新しいプログラム機能に関する内部メール

これで、システムのユーザーは、プログラムの新しい機能や機能、会社のニュースやプロモーション、ビジネス分野で最も興味深いイベントについて最初に学ぶことができます。 ユーザーは、「シンプルビジネスメッセージ」タスクで情報を利用できます。



新しいバージョンが追加されるたびに、当社のシステムはビジネスの運営と効率の向上にさらに便利になります。 今すぐ登録して、柔軟で便利でシンプルな企業管理システムを使用する利点をすべて手に入れましょう！
