AMDはIntelをサーバーソリューション市場にプッシュする準備をしています

AMDは今年の夏、企業の高性能コンピューティング部門への意欲的な復帰を発表しました。 EPYCファミリーのアーキテクチャソリューションに基づいた新しいプロセッサの発表により 、AMDはデータセンターの機器分野でIntelを推進します。



/写真/ twin-loc.fr / CC BY-SA



競争上の優位性



新しい戦場でのEPYCチップの主な敵は、IntelのSkylakeプロセッサーです。 同社は、クラウドコンピューティングの分野で大企業や独自のインフラストラクチャを備えた企業で使用されるプロセッサの市場を支配しています。 Intelのデータセンターグループの責任者であるNavin Shenoy氏は、プロセッサのXeon Skylakeシリーズの発売により、「データセンター業界にとって10年で最も重要な発表」 と呼ばれました。





/写真/ 大坪一久 / CC BY-SA



それにもかかわらず、Kurt Markoなどの多くのIT業界の参加者は、インテルの分断されていない電力の世紀が終わりに近づいていることを保証します。クラウドサービスはローカルインフラストラクチャの混雑を開始し、同社の2桁の財務成長はここ数四半期で半減しました。



同時に、2つの競合他社の新製品をテストしたところ、EPYCチップはIntelのXeonに比べてデータセンターのパフォーマンスの点で予想外の多くの利点があることがわかりました。



Intelは512ビットのx86コマンドシステム拡張機能（AVX-512）を新しいスケーラブルなXeon Skylakeプラットフォームに統合しました。これにより、整数ベースのFLOPSとパフォーマンスが以前のバージョンのXeonと比較して2倍になります。 AMDは128ビットのベクターに依存することを決定しました。これにより、一見EPYCが不利になります。 ただし、ほとんどの高性能コンピューティングでは、AVX-512は現在使用されていないことに注意してください。



Anandtechチームは、C-ray、POV-Ray、NAMDなどを使用して新製品をテストしました。 テストは、Ubuntu Linuxを実行するデュアルプロセッササーバーで実施されました。



EPYCプロセッサは、3つの浮動小数点テストすべてでXeonよりも優れていました。 C-rayでは、一定期間レンダリングすると、EPYC 7601はXeon 8176よりも50％良い結果を得ました。POV-Rayテストでは、利点は16％でした。 NAMDの場合、Anandtechは2つの方法をテストしました。AVXを使用する場合とAVXを使用しない場合です。 どちらの場合も、EPYCプロセッサは、それぞれ22％と41％前進しました。



Anandtech が BigDataテストも実施したことは注目に値します。このテストでは、XeonとEPYCのギャップは5％未満でした。 この場合、プロセッサは整数計算とメモリアクセスのパフォーマンスを評価するテストに合格しました。



Xeon 8176とEPYC 7601を比較する場合、それらのコストを見失うことはありません。 ここで、AMDにはIntelの市場での地位を弱める可能性があります。



市場シェアの闘争



ITコミュニティでは、AMDの企業部門への復帰に関する意見が分かれていました。 それらのいくつかはあまり楽観的ではありませんでした。 たとえば、7月18日、Intelとの競争におけるAMDの見通しに対する否定的な評価により、同社の株式は3.2％下落しました。 バークレイズのアナリスト、ブレイン・カーティスは 、この新製品は「十分な競争力がない」と結論付けました。



彼はHPやDellなどの企業からある程度の関心を集めることができたが、新しいチップの評価には少なくとも1年かかると述べた。 AMDにエコシステムがない場合。



AMDはこれを理解しているため、AMDのコーポレートバイスプレジデント兼コーポレートソリューションリーダーであるScott Aylorは、会社が製品を1つだけ提供するだけでは不十分であり、将来の展望を顧客に提供することが重要であると強調しました。



したがって、AMDはEPYC 7000とRomeの間のソケット互換性に対する責任を主張しました。 後者は、データセンター向けの新しい次世代AMDプロセッサのコード名です。 同時に、Aylorは、EPYCがデュアルソケットプロセッサ市場の50％でSkylakeと競合できると確信しています。





/写真/ AMD Markham Canada / CC BY-SA



この状況は、他のIT企業と共鳴しています。 Microsoft Azure ComputeのコーポレートバイスプレジデントであるGirish Bablani、バイドゥの社長であるZhang Ya Qin、およびサーバーソリューションの社長であるDell EMCのAshley GorakhpurwallaがAMDをサポートしました。 彼らは、企業が新しいEPYCアーキテクチャソリューションを選択する準備ができていることを明らかにしました。



Intelのデータセンターセグメントでの地位からのずれについては、AMDの従業員でさえ予測はまだ控えめですが、ForbesのDave Altavillaが指摘するように、10％のシェアでもAMDの高収益ビジネスに数十億ドルをもたらします。
