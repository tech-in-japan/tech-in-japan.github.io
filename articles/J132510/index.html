.NET Gadgeteerオープンソースプロジェクトに基づいたFEZ Hydraデバッグボード

Habréでは、 FEZ Spider Kit社の以前の開発についてはすでに言及されています。 GHI Electronicsは 、 .NET Gadgeteerデバイスの開発に向けて開発を続けています 。 そして、11月8日に、デザイナーボードFEZ Hydraに新しい追加を導入しました。 FEZ Hydraは240Mhz ARM9プロセッサを使用しています。 16 MBのSDRAM RAM、および4 MBの永続メモリタイプFLASHを使用。 開発者が示すように、デバイス上でLinux OSを実行することが可能です。 システムは100％オープンです。











仕様：

  • 14 .NET Gadgeteer互換ソケット
  • プロセッサーATMEL 240Mhz ARM9
  • 16 MB RAM
  • 4 MB ROM
  • リッチグラフィックスのサポート
  • タッチコントロールでTFTディスプレイをサポート
  • FAT16 / FAT32
  • 完全なTCP / IPスタックのサポート：SSL、HTTP、TCP、UDP、DHCP
  • USBデバイス
  • 容量制限のない4ビットSDメモリカードのサポート
  • アナログ入出力
  • Pwm
  • タイヤI2C、SPI
  • UARTインターフェース
  • リアルタイムクロック
  • 寸法3.42 "x 2.44"（8.7 cm x 6.2 cm）
  • 動作中の消費は130 mAです
  • 温度範囲：0〜70°C
  • RoHS（有害物質の制限）指令に準拠


このボードの特徴は、ファームウェアがオープンソースGCCコンパイラーによってコンパイルされたことです。 これにより、開発者のグローバルコミュニティはコードを簡単に変更し、プラットフォームの開発に取り組むことができます。

ボードのレイアウトの開発は、費用対効果の高いプリント回路基板（プリント回路基板）の開発に使用されるEAGLEソフトウェア（Easily Applicable Graphical Layout Editor）で行われました。 通常、AGLEはArduinoのようなオープンソースデバイスの開発に使用されます。

FEZ Hydraの主な利点の1つは、その低コストです。 ボードの価格は79.95ドルで、スターターキットが付属しています。費用は99ドルです。



FEZ Hydraは現在、プロジェクトの開発のためのパートナーを探しています。 NETMFポーティングキットの使用経験がある場合は、 GHI Electronicsにお問い合わせください。 事前注文は2011年12月に開始されます。ボードレイアウトとソースコードのすべての資料は2011年12月にも利用可能になります。FEZHydraの詳細については、 http： //www.ghielectronics.com/catalog/product/328をご覧ください 。 .NET Gadgeteer Technologyの詳細をご覧ください。
