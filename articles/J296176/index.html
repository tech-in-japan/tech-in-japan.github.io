Microsoftは1年でイスラエルから5番目の会社を買収

マイクロソフトは、情報セキュリティに特化したイスラエルの企業Security Islands の買収を確認しました 。 会社の技術は、Microsoft Azure Rights Management権利管理サービスに統合されます。



この契約は、昨年のイスラエルからの企業買収におけるマイクロソフトの5年目の買収となります。







過去1年間、Microsoftはイスラエルのスタートアップを注意深く監視してきました。 2014年11月13日、マイクロソフトは 、企業情報セキュリティの分野におけるイノベーターであるAoratoの買収を発表しました。 このスタートアップは、2011年にイスラエル軍の退役軍人によって設立されました。 同社は、企業のITシステムの中央通信コンポーネントへのアクセスを監視するソフトウェアを開発および販売しました。 Microsoftは、エンタープライズハイブリッドクラウドでの開発を使用するためにスタートアップを買収し、2015年7月にAdvanced Threat Analytics製品をリリースしました 。 取引の詳細は明らかにされていないが、金額は 2億ドルになる可能性がある 。



2015年1月、Microsoftは法律文書や企業情報の分析を専門とする新興企業Equivioを買収しました。 このスタートアップは、米国司法省などの法律事務所や政府機関と協力してきました。 マイクロソフトは、ビッグデータ分析市場のリーダーシップに近づくための契約を締結しました。 噂によると 、Microsoftの購入には2億ドルかかりました。



2015年5月、Microsoftと、タブレット用の「デジタルペン」であるSurface Penを開発したイスラエルの新興企業との契約について知られるようになりました。 ウォールストリートジャーナルはこの取引を3000万ドルと見積もった 。



2015年9月、Microsoftクラウド製品のマーケティング担当副社長であるヌモトタケシはブログで、同社がAdallomクラウドの情報セキュリティ開発者を買収したと書いています。 この会社は2012年以来営業しており、Microsoft Office 365、Google Apps、Salesforce.com、ServiceNow、Dropbox、ServiceNowおよびその他のクラウド製品のユーザーデータのセキュリティを確保するために、そのサービスが既に使用されています。 2012年から2015年半ばにかけて、AdalomはEMC、HP、Sequoia Capitolベンチャーキャピタルファンド、European Index Venturesから5000万ドルの投資を集めました。 Microsoft はこのスタートアップに約3億2000万ドルを支払うことができた 。



過去12か月間にマイクロソフトを買収した5番目のイスラエルの新興企業は、セキュアアイランドの買収です。 噂によると 、取引額は1億5000万ドルですが、買収の確認後、正確な数字を出した企業はありませんでした。 企業は規制当局の承認を待っています。



同社は情報セキュリティに携わっており、過去9年間、組織にサービスを提供してきました。その技術は、ボーダフォン、UBS、スイスを含む多くの大企業で既に使用されています。 マイクロソフトは既にこの会社と協力して、 Azure Rights Management Serviceを改善しています 。 新しい契約により、企業はこのサービスでセキュアアイランドテクノロジーをより密接に使用して、従業員が企業所有のデバイスと自社のガジェットの両方を使用してデータを配布する環境で組織の機密情報を保護できます。
