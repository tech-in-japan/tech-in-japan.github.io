SCO対Linux



エリック・エリック・レイモンドが「最初のUnix会社」と呼んだ成功したサンタ・クルーズ作戦は、SCOグループが最終的に自らを呼んだこととはほとんど関係ありません。 2000年代初頭まで、Santa Cruzはx86プラットフォームサーバー用に商業的に実行可能なUnixを作成していましたが、取得した経験に基づいて最高のUnixを作成することを望んでいたRansom Love率いるCaldera Inc.



開発中のIntel Intel ItaniumアーキテクチャとIBMのPOWERアーキテクチャのオペレーティングシステムであるMontereyプロジェクトを実装しようとするIntelとIBMとのパートナーシップは成功しませんでした：市場は64ビットx86アーキテクチャオプションを支持してIA-64を拒否し、無料のLinuxオペレーティングシステムがUnixのニッチを占有しました。 2002年、RansomはDarle McBrideにポストを残し、Canopy Groupの一部はSCO Groupと改名し、新製品を開発する代わりに訴訟に集中しました。



裁判所では、SCOはIBMをMontereyプロジェクトコードの一部をオープンLinuxカーネルに投入したとして非難しただけでなく、小規模なプレイヤー（大規模な購入者、エンドユーザー、Red HatやNovellなどのLinuxベースシステムのメーカー）からの損害賠償も試みました。 このため、SCOはオープンソースコミュニティの間で見捨てられています。



この訴訟は何年も続き、SCOは顧客、業界の信頼、そしてナスダックでさえ、株価の最終ラインを下回らないことですぐに失いました。 同社は、クレームのみを扱うスケルトンでした-他のすべての従業員は解雇されました。 すでに2007年に、SCOは破産を発表しました。 結局、裁判所の決定はSCOに有利ではなく、会社のUnix資産は売却されました。



AOL /タイムワーナー



90年代後半にアメリカでインターネットの人気が高まったのは、ダイヤルアップの巨人であるAOLが拍車をかけたためです。AOLは、加入者のみがアクセスできるコンテンツを備えた閉じたエコシステムであり、最も注目に値するのは、2000年の戦争時の1640億ドルの買収です。



合併によって形成されたモンスターは、スティーブケース（AOL）とヘラルドレビン（タイムワーナー）に率いられ、必死に9年間続いた。 2002年以来、AOLタイムワーナーの加入者数は着実に減少しており、ほぼ3,000万人の顧客が2007年にはすでに500万人になっています。







同社の市場評価額は、2,400億ドルから数十億ドルに減少しました（2011年）。 ティムアームストロングの新しいCEOを引き継いだ後、AOLタイムワーナーは、同じ名前を持つ2つの独立した企業への崩壊を発表し、長年にわたる実りのない協力の終わりを告げました。 つい最近になって、AOLは、About.me、TechCrunch、The Huffington Postなど、いくつかの新しいメディアを吸収して、まっすぐになり始めました。



ヤフーとマイクロソフト



90年代後半、Yahooは急成長の戦術を静かな購入会社に変更し、自社の旗の下でサービスを展開しました。 ヤフーでは、金融ニュース、画像ホスティング（Flickr）、およびサイト（Geocities）が登場しました。 しかし、最終的にはGoogleとFacebookが勝ち、新たな買収は必ずしも収益化に成功しなかったため、会社の収益は減少しました。



一方、マイクロソフトはインターネット上でその存在感を発揮できるプラットフォームを探していましたが、Yahooはその定義にかなり適合していました。 2008年2月、Yahooを470億ドルで買収する意向が発表されましたが、Yahooの共同設立者であり当時のCEOであるJerry Youngは、彼の発案を請求することを決定し、会社が過小評価されており、新しい共同所有者を必要としないとは言いませんでした。



数週間後、交渉はついに行き詰まりました。 Yahooの株主であるKarl Aikanは、Microsoftの代表者の前でYoungに譲歩を強要しようとしました。 失敗した議論にうんざりして、Ballmerは、BingおよびWindows Liveブランドで独自の検索エンジンとWebサービスの開発を開始するように命じました。



2008年、YahooでMicrosoftとの統合に失敗したため、一連のレイオフが一掃され、同社の市場価値は急落し始めました。 2011年、その価値はたった170億ドルで、レドモンドで提供されていたもののほぼ3分の1でした。



2009年1月、キャロルバーツはジェリーヤングを引き継ぎました。ジェリーヤングは、皮肉なことに、YahooサービスにBing検索を使用することについてマイクロソフトと契約しました。 しかし、キャロルは状況を改善することができず、昨年9月6日にiPadから部下に手紙を送りました。 キャロルは再びヤフーを去り、その後レイオフが行われ、同社は従業員の14％で解散した。



最近知られるようになると、Yahooの新しいCEOはMarissa Mayerになりました。 彼女が以前の巨人の力を取り戻すことができるかどうかは不明ですが、2008年にさらなる交渉が拒否されたことを思い出して、バルマーとマイクロソフトのすべての喜びを想像することができます。



IBMおよびMicrosoft



70年代後半にIBMの何人かの人々が、オペレーティングシステムがまだ準備されていない将来の伝説的な5150 PCマシンの開発を始めました。 Apple II、Osborne 1、およびKayproマシンですでに使用されていたDigital ResearchのCP / Mは、唯一の適切なマシンであるように思われました。 1980年にIBMはGary Kildallに連絡しましたが、5150プラットフォームと将来のすべてのPCのCP / Mのライセンスに関する交渉は失敗し、他の人を探す必要がありました。



ヤング・ビル・ゲイツ、スティーブ・バルマー、およびマイクロソフトを初めて設立した従業員のポール・アレンは、同様のCP / Mコマンドインタープリターを備えた86-DOS、x86互換のオペレーティングシステムを備えた小さな会社Seattle Computer Productsに偶然出会いました。 オペレーティングシステムは、75,000ドルの悲惨な価格で無限の使用権をすべて取得しました。



IBMが非独占的なライセンス条件にあまり注意を払っていない交渉の後、MS-DOSは、シアトルの開発者への参照が有名に削除された86-DOSであり、数千万台のマシンの将来の市場に広がっています。



Digital Researchは条件を満たし、より柔軟にすることができ、IBMは条件を厳しくし、MS-DOSまたは両社を完全に購入することができます。 しかし、歴史は仮定的なムードを容認するものではなく、この時点からレドモンド社の成功が始まり、それが90年代に巨人になり、IBMを自社のPCプラットフォームの市場から追い出しました。 CP / Mは決して人気になりませんでした。 Digital Researchは、独自のDOSクローンであるDR-DOSの作成を試みましたが、Microsoft製品に対するいくつかの利点にもかかわらず、失敗しました。 DR-DOSオペレーティングシステムは後にNovellに売却され、その後Calderaに売却され、すでに言及されたSCOの財産となりました。



スティーブ・ジョブズの辞任



スティーブ・ジョブズは、風変わりで暴力的な性格のために、いつも一緒に仕事をするのは非常に不愉快でした。そして、スティーブと80年代初頭に率いたマッキントッシュ・チームとの関係は徐々に悪化しました。 1983年にCEOを務める会社の成長を踏まえて、ジョブズは同じポジションを務めたペプシコのジョンスカリーを誘いました。



1984年に、Appleは最初のMacintoshをリリースし、成功したApple IIに取って代わり、あまり人気のないApple IIIになりました。 IBM、Compaq、およびPCとそのクローンを販売する他の大手企業間の激しい競争は、会社の問題を個人的なものとして認識しているSteveの内なる世界を激化しました。 取締役会はスカリーに、彼の高価で疑わしい努力でジョブズを制限するように要求しました。それはスティーブをいらいらさせました。 1985年、ジョブズはスカリーを追い出して就任しようとしましたが、ジョンは会社を経営するのに間違っていると主張しましたが、スティーブにとっては意外なことに、取締役会はスカリーの側を取り、アップルの創業者の一人を解雇しました。



会社がスティーブの指揮の下に存在しなかった11年間は、ほとんど常に彼女の最悪の年と見なされます。 CEOとして、数人が交代しましたが、Jobsの先見の明のあるリーダーシップがなければ、Appleの地位を改善する機会はありませんでした。 90年代半ばまでに、同社は嘆かわしい状態にあり、Macintoshオペレーティングシステムは、直接の競合他社からの開発が大幅に遅れていました。



1996年に、AppleはSteveの新しい成功した会社NeXTを買収しました。NeXTは忘れられませんでした。NeXTSTEPオペレーティングシステムは、後にOS Xと呼ばれるものになりました。 2000年には、ポストから接頭辞「temporary」を削除しました。 スティーブはすぐに多くの見込みのない製品を閉鎖しました。 その後数年で、会社は分岐し、他のデジタルデバイスを導入および改善しました。 Apple向けiMac、iPod、iPhone、iPadのリリースにより、最も成功した時期が始まりました。



Windows Vista



Windows XPがこれほど長く続くと予想した人はいませんでした。 リリース前でさえ、Longhornというコードネームの新しいプロジェクトで作業が始まりました。 最初の後継者のデモンストレーションは2003年にマイクロソフトの会議で行われました。



2004年8月、サイズと機能に対する不満のために、Longhornプロジェクトは最初から開始されました。 将来のWindows Vistaには、オペレーティングシステムのセキュリティと64ビットシステムのサポートを改善し、ユーザーインターフェイスを更新するという単純なタスクがありました。 開発には5年以上かかり、結果は満足のいくものではありませんでした。



Windows Vistaの販売は2007年1月に始まり、新しいオペレーティングシステムはすぐにマイナスの評価を受けました。 ユーザーは、高いシステム要件、多くの一般的なプログラムの非互換性、ユーザーアカウント制御の不便さに不満を抱いており、ユーザーアカウント制御は、あまりにも多くの執askedな質問を頻繁に出していました。 もちろん、その後のService Packは多くの問題を解決しましたが、評判はすでに損なわれていました。



はるかに成功したWindows 7は、多くの保守的なXPユーザーにオペレーティングシステムの変更を強制することができました。



Osborne Computer Corporation



コンピューターで書籍やマニュアルを発行する会社を売却した後、Adam Osborneは安価なパーソナルコンピューターを作成することにしました。 さらに苦労せずに、イギリス人は新しい会社と彼のコンピューターを自分の名前と呼んだ。 Osborne 1は、重量が10.7キログラムで驚くほどの耐用性があり、CP / M 2.2指向で、小型の5インチスクリーン、64キロバイトのRAM、周波数が4メガヘルツのZilog Z80プロセッサー、2台のフロッピードライブ、わずか1,795ドルでした。 、1万5千ドルのプログラムが付属しています。



8か月間、Osborne Computer Corporationは11,000台のコンピューターを販売し、さらに5万台のコンピューターが予約注文されました。これは結婚率が高いにもかかわらずです。 ピーク時には、1か月あたり1万台のコンピューターが販売されていましたが、当初はOsborne 1の全期間で販売されると予測されていました。会社の成長は実現が困難です。たった1年で会社は2人から3000人に成長し、収益は7,300万に達しました。ドル。 出張から1週間続く展示会に戻った会社の長は、オフィスの敷地内を移動するのが困難でした。 1982年10月までに、オズボーン1の500コピーが毎日作成されました。 OSCは成功する運命にあるようです。







1983年初頭、OSCはOsborne 1の後継モデルであるOsborne Executiveを発売しようとしていました。OsborneExecutiveは2倍のRAMと大画面を備えており、必要なソフトウェアセットとともに配布されましたが、コストは約2.5万でした。ドル。 オズボーンの次のステップは、繰り返せない間違いの例として、今日のビジネススクールで教えられています。 アダムオズボーンは、ドアがしっかりと閉まっているホテルの部屋で、より高価なコンピューターを押し通そうと、レポーターに新しい車を見せ、価格と技術仕様について話しました。



約束どおり、4月中旬までに何も現れなかったが、サプライヤーは発表を知り、Osborne 1のすべての注文をすぐにキャンセルした。4月17日、OsborneはOsborne Executiveディーラーに1週間見せ、新しいマイクロコンピューターがそれは古いものに対する競争相手ではありませんが、効果はありませんでした。 7月、オズボーン1の価格は1.295ドル、8月の価格は995に引き下げられましたが、これも助けにはなりませんでした。



売上はほぼゼロに落ち、同社は予想をはるかに上回る損失を被り、1983年9月13日にOSCは破産を発表しました。 後に、Osborne Vixenをリリースする試みがありましたが、再び失敗しました。 今日、Osborne Computer Corporationはその失敗のコンテキストでのみ記憶されています。



OSC死の別の原因と呼ばれることもあります：より高度なKayproコンピューター、より安価である、または古いモデルのマザーボードが15万ドルで利用可能であり、新しいマシンの生産がすでに始まったときに完成品で200万を費やすため、管理者が理解できない決定。



それにもかかわらず、今日は企業が将来のモデルの新しい開発と特性に関する情報の普及を厳しく制限し、人々が私たちのデジタル時代の新しいデバイスについて噂などの昔ながらのツールから学ぶことを非難するのはオズボーン効果です。
