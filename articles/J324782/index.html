皆のためのコミュニケーション：Grandstream Networksの新機能

Grandstream Networksは、あらゆる規模の組織向けにSIPテレフォニーおよびユニファイドコミュニケーションソリューションを開発しています。 会社が最近紹介したものを簡単に確認しましょう。







主要メーカーがIPテレフォニーの時代に積極的に参入したとき、VoIPの開発に伴い、業界がプログラムコードとプロトコルのオープン性に移行するという戦略計画を考慮した人はほとんどいませんでした。 自由に成長するSIPプロトコルと*アスタリスクプラットフォームは、企業が開発した独自の機能をすぐに採用します。 コンタクトセンター、会議、いわゆる追加のテレフォニー機能など、開発者が依然として多くのリソースを投資する過去の高価なソリューションは、販売がますます困難になっています。



そのような手荷物を持っていないGrandstream Networksは、オープンプロトコルのみを使用し、オープンシステムでのみ動作するデバイスの開発に注力しています。



IP電話



Grandstream IP-phonesフリートは最近、17行目の新しいラインで補充されました。 このデバイスファミリは、ハイエンドモデルと以前のシリーズの基本レベルの電話機、いわゆるミッドレンジデバイスとの間のギャップを埋めました。 このモデルの範囲は、一般の従業員がオフィスでよく使用する最も単純な電話に取って代わるように設計されています。 合理的な価格でより優れた機能を提供し、定期的に通信が必要な場所（ロビーやラウンジなど）に予算のあるデバイスを詰め込みます。 新シリーズはまた、非常にエレガントなデザインを誇っています。







この回線の主な利点の1つは、多数の仮想キー（モデルに応じて24または32）が表示されることです。この仮想キーでは、内線番号または内部加入者の短縮ダイヤルなど、さまざまな機能をプログラムできます。 ディスプレイを切り替えるだけで、このようなキーのリスト全体を確認できます。これにより、新しいモデルでの作業が簡単かつ便利になります。 また、これらのデバイスには最大5人の参加者の会議を作成し、多くのSIP回線をサポートする機能があることに注意してください。



ゲートウェイ



Grandstreamファミリーのアナログアダプターも補充されます。新しいモデルHT802、HT812、HT814が登場し、以前のデバイスを同様の機能に完全に置き換えます。 このソリューションは、遠隔作業の場合にオフィスと自宅の両方に配置できるように設計されています。 また、大規模なプロバイダーが多数の顧客に提供するために使用することもできます。







フロントパネルの新しい外観とシンプルな単色ディスプレイにより、デバイスはあらゆるインテリアにシームレスに溶け込みます。 ネットワークポートとアナログポートは異なる色でマークされています。 Zero Config、TR-069、およびXMLを使用した自動デバイス構成をサポートしています。これにより、あらゆるレベルのトレーニングを受けたユーザーの接続が簡素化されます。



2および4 FXS回線上のアダプターHT812およびHT814は、NATルーターを内部に受け取りました。これにより、オフセットとしての取り付けが容易になります。 また、OPUSブロードバンドコーデックのサポートと、バックアップサーバーに登録する機能にも注目してください。



IP PBX



すでに人気のあるUCM6100シリーズは、新しいUCM6200 IP-PBXラインに置き換えられています。 外見的に新しいモデルUCM6202、UCM6204、およびUCM6208は、以前のファミリのデバイスと視覚的に非常に類似しているという事実にもかかわらず、多くの重要な違いがあります。 UCM6200の要素ベースが改善されました。 新しいシリーズのすべてのモデルにはNATルーターとDHCPサーバーがあります。特に、UCM6208デバイスでサポートされるユーザーの数は800人、最大100人の同時会話になります。







UCM6100およびUCM6200デバイスは同じファームウェアを備えているため、構成と保存されたデータを新しいPBXに簡単に転送できます。 当然、すべての機能は保持され、新機能と新機能の両方について、新しい機能が引き続き更新されて表示されます。 これらの機能のうち、次のことに注意してください。



•異なるサブネットからのZero Configのサポート。

•保留せずに通話を転送します。

•IPv6のサポート。

•CRM SalesforceおよびSugarCRMとの統合。

•PMSとホテルサービスH-Mobileの統合。

•会議のスケジュールとGoogleカレンダーとの統合。

•WebポータルからFAXを送信します。

•着信FAXの自動ルーティング。



会社のWebサイトで新しい機能の外観をフォローできます。 すべての製品のファームウェアページの[サポート]セクションには、リリースノートと呼ばれるドキュメントがあり、新しい機能のリストと詳細な説明を見つけることができます。



VKSおよびIPVideoTalkシステム



ただし、GVC3200およびGVC3202ビデオ会議システムのバージョンアップグレードは、主にIPVideoTalkクラウドサービスのサポートの出現によるものです。 サービス自体も新しい機能を取得しました。 ユーザーは、ChromeおよびFirefoxブラウザーを介したビデオ会議中に音声で観察して通信できるだけでなく、カメラの画像を送信したり、コンピューターの画面からコンテンツをブロードキャストしたりできます。 したがって、一度に8人の参加者に画像をブロードキャストしながら、ビデオ会議を行うことができます。 残りの対話者は、彼らが道を譲った場合、ビデオ放送に接続することにより、いつでも音声で観察し、通信することができます。 これらの革新により、IPVideoTalk会議中のビデオミキシングは、選択したライセンスに応じてクラウドでHDまたはフルHDとして実行され、周波​​数は15フレーム/秒です。 IPVideoTalkアプリケーションはAndroidおよびiOSでも利用できることに注意してください。



ビデオ会議は、GVCモジュールを使用して組み合わせることができます。 会議の主催者には、多くの「特権のある」機会があります。たとえば、参加者が出席者のリスト全体を表示することを禁止する場合があります。 また、会議から退出し、残りの参加者にコミュニケーションを継続させることもできます。 参加者は、特別な4桁のコードを入力することにより、リーダーを置き換えることができます。



DECT



Grandstreamの新しいDECTは、DP750ベースステーションとDP720コードレス電話の2つのデバイスで構成されています。 DP715およびDP710で構成される以前のバージョンは廃止され、新しいソリューションと互換性がないことに注意してください。







DP750ベースステーションでは、最大5つのDP720デバイスを登録し、最大4つの同時通話を処理できます。 駅の範囲は、オープンエリアで300メートル、屋内で50メートルです。 DP750は、最大10個のSIPアカウントをサポートし、接続されているすべてのDECTハンドセットに配布できます。 POE接続は、デバイスを壁に取り付けるためのオプションを提供します。



ハンドセットには800mAhのバッテリーが搭載されており、最大250時間のスタンバイ時間と最大20時間の連続会話が可能です。 このモデルには1.8インチのカラースクリーンが装備されています。 メニューははるかに便利になりました-通話保留や通話転送、通話ピックアップ、三者間会議の編成などの機能を使用することがはるかに簡単になりました。



連絡先と連携する機能を拡張しました。 LDAPだけでなく、最大3,000エントリのグローバルでダウンロード可能なプライベート連絡先リストがサポートされています。 4つのDP750ベースに同時にハンドセットを登録することができます。これにより、フォールトトレランスを向上させ、電話の動作領域を拡大することができます。 電話機能には、スピーカーフォン、キャッチホン、自動応答が含まれます。 充電はベースを使用するだけでなく、microUSBコネクタ付きのケーブルを使用して行うこともできます。 バッテリー不要の操作もサポートされています。



Wi-Fiアクセスポイントとルーター



また、Grandstreamは、ワイヤレスアクセスポイントとルーターという多くの新製品を発表しました。 これらのソリューションにより、お客様は便利なWebインターフェイスを使用してネットワークを迅速に展開できます。



最初のアクセスポイントGWN7610は、あらゆる規模の大企業でワイヤレスネットワークを構築するように設計されています。 モデルの使いやすさは、各GWN7610に組み込まれ、同時に50ポイントを管理できるコントローラーの存在にあります。 直感的なWebインターフェイスにより、接続されているすべてのポイントをすばやく簡単に設定できます。 GWN7610は、コントローラーを必要としない独立したユニットとして機能します。 このソリューションは、少数のアクセスポイントのみが計画されている小さなスペースに最適です。 開始するには、モデルをPOEで接続するだけで、2つのギガビットイーサネットポートが存在するため、GWN7610を別のネットワークに接続する際の耐障害性が確保されます。 MIMO 3x3：3を備えた全方向性アンテナは、最大150メートルのカバレッジを可能にし、802.11ac wave 2規格のサポートは、最大1300 Mbit / sの速度を提供します。 デュアルバンドモードで動作する機能により、優れた通信品質が提供され、2.4 GHzと5 GHzの周波数のユーザーを同時に接続できます。 GWN7610では、異なるVLANを使用してトラフィックを区別する機能を備えた最大16個のSSIDを作成できます。 QOSテクノロジーは、GXV3240、GXV3275、GVC3200などの電話およびビデオ会議デバイスに高品質の音声とビデオを提供します。 また、WPA2-Enterpriseのサポートにより、サードパーティのRadiusサーバーを使用した認証と承認が可能になり、ネットワークセキュリティが向上します。







Grandstream Networksの次の新製品は、ワイヤレスネットワークの機能を補完し、高品質の有線設定を実行するGWN7000ルーターです。 ビルトインコントローラーを使用すると、同時に最大500台のアクセスポイントを管理できるようになり、集中管理で大規模なワイヤレスネットワークを構築できるようになる予定です。 GWN7000には7つのギガビットイーサネットポートがあり、そのうち2つのWANポートと5つのLANポート、および2つのUSB3.0コネクタを使用して、プリンタ、3G / 4G LTEモデム、または追加のストレージを接続できます。 3G / 4G LTEモデムを使用すると、何らかの理由で有線接続が機能しない場合に通信が提供されます。



オフィスインターホン



Grandstream Networksのもう1つの興味深い新製品は、GDS3710 IPインターコムです。 これは、施設への人のアクセスを制御するために設計された、保護クラスIP65の破壊者防止装置です。 このソリューションには、LEDバックライトとタッチスクリーンが装備されています。 このデバイスは既存のビデオ監視システムと簡単に統合できるため、DVRを使用してインターコムからの記録を保存できます。







GDS3710は、H.264およびMJPEGコーデックを使用してビデオを圧縮します。 インターホンには、180度の概観と内蔵RFIDリーダーを備えた2メガピクセルのフルHDカメラが装備されています。 アクセスカードを使用してドアを開くことができます。 この場合、インターホンに個別のカードを使用する必要はありません-既存のカードをプログラムできます。



このソリューションの利点の1つは、SIPアカウントの存在です。これにより、インターホンを既存のテレフォニーシステムと統合できます。 したがって、インターコムを呼び出して、電話から事前にプログラムされたコードを使用してドアを開くことができます。



GDS3710モデルには、最も必要な電話機能がすべて搭載されています。



•SRTPを使用した暗号化のサポート。

•Radiusサーバーを使用したユーザー認証。

•G.711μ/ a lawおよびG.722などの音声コーデックのサポート。



さらに、多くの重要な機能がGrandstreamビデオカメラから移行されました。 そのため、モーション検出を使用すると、観測エリア内の動きを追跡できます。 また、プライバシーマスクを使用すると、特定の領域を記録から閉じることができます。 また、インターホンは、サードパーティ製のセンサーとデバイスの接続を提供します-たとえば、サウンドまたはライトアラームシステム。 インターホンをネットワークに接続するには、電源またはPOEを使用できます。
