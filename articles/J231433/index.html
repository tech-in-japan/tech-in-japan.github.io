システム全体の変更：



  • kolibri-processブランチとトランク（メインブランチ）のマージ。 kolibri-processブランチの一部として、プロセスとスレッドの新しいモデルが開発されました。これは、大規模なOSのモデルにより似ています。
  • すべてのドライバーをPE形式に変換します。 PE形式を使用すると、以前に使用したCOFF形式と比較して、ドライバーをロードし、ライブラリをHummingbirdsに接続しやすくなります。 変換に成功した後、COFF形式のドライバーサポートは削除されました。
  • IntelおよびATIグラフィックカード用のドライバーの更新
  • exif.obj ：画像からEXIFデータを抽出するためのライブラリ。
  • すべてのIDEコントローラーのサポート（以前はPCI 01018xのみがサポートされていました）
  • 一部の構成でのダウンロード速度の改善
  • 16bppビデオモードのサポートが追加されました（24 / 32bppがサポートされていない古いビデオカードと安価な組み込みx86システム-特に、 vortex86 VGAを備えた86duinoで使用されます ）
  • TinyGL ：FASMのフォーク
  • COMマウスドライバー：3番目のボタンのサポート。
  • ネットワークサブシステム-多くのバグの修正（ネットワークカードドライバー、ユーティリティ）
  • C / C ++ 開発用の更新されたSDK： NewLibおよびMenuetLibC




アプリケーションソフトウェアの変更：



  • Eolite ：選択したファイルをコピーします（挿入で強調表示）。
  • RDsave ：エラー修正; システムカラーの使用。
  • Game Center ：iniを使用してゲームのリストを保存するように切り替えます。
  • scrshoot ：16bppモードをサポートします。
  • tengine ：Hummingbirdに移植された独自の 2Dゲームエンジン。
