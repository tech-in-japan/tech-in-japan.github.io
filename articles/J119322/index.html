Globalscale Technologiesは、サイレントD2Plugミニコンピューターに基づいた開発キットを発表しました





ミニチュア低電力コンピューターの棚が到着しました。 5月上旬、Globalscale TechnologiesはD2Plug開発キットのリリースを発表しました。

D2Plugは、DreamPlug、Guruplug、SheevaPlugなど、類似の製品ラインの論理的な継続です。 このシリーズのデバイスは、低消費電力、比較的高いパフォーマンス（PCのような）、小さな寸法、低ノイズレベル（すべてのモデルではありませんが）、多くの異なるインターフェイスの存在、および使用されているソフトウェア（たとえばLinuxベース）によって区別されます。











habrahabr.ruでは、Globalscale Technologies製品が繰り返し言及されています。

怒っている投稿の形式のように、たとえばここに：

Guru Plug Server Plus：これにお金と神経を浪費しないでください 、

興味深いティーザーの投稿、「このようなものが欲しい」など。

GuruPlug Server- 充電器サイズの超小型Linuxサーバーの新しいバージョン 、 DreamPlug-ミニチュアガジェット愛好家向けのPC 、

以下のような中立的な感情的な色の投稿：

電源としてのLinuxサーバー 、

GuruPlug Server Plusの概要 、

GuruPlug / GuruPlug Plus用のコンソールケーブルの製造 。



要約すると、市場で「生の」製品を発売することで急いで行動したという事実（GuruPlugと過熱の問題）にも関わらず、間違いなく可能性があり、競争力のあるデバイス（特にこれは、競合他社がほとんどいないことを考えると信じられています）。



D2Plugは、コンピューティング、グラフィックス処理、マルチメディア向けの単一のソフトウェアおよびハードウェアプラットフォームとして位置付けられています。

仕様では、最大1080p（1920 x 1080）の解像度のディスプレイのサポートを要求しています。 このボードは、高速ネットワークインターフェイス、USBインターフェイスを使用した入出力インターフェイスを統合しています。 OSS（オープンソースソフトウェア）がソフトウェアとして使用されます-Ubuntu Linux 10.04は元々フラッシュされていました。

可動部品がないため、高い信頼性が実現します。 同じ機器をサーバーとワークステーションの両方として使用できます。
