ヒョウのチューニング

OS X Leopardがどれほど優れていても、人々が嫌いで、変えたいと思うものは常にあります。 そして、ユーザーのレベルが高いほど、あなたの好みに合わせて何かを変更したいという要求と欲求が増えます。 この記事では、Leopardの外観を改善できる小さなヒントについてお話します。





ドック



3D / 2Dドック

古典から始めましょう。 Dockの新しい外観が気に入らない場合は、ターミナルで次のコマンドを実行して2Dバージョンに変更できます。

$ defaults write com.apple.dock no-glass -boolean YES ; killall Dock







ドッココロ

それでも3D Dockが好きなら、この小さなプログラムでドックの色をカスタマイズできます。



Leopardocks.com 、 Dockulicious.com

これらの2つの素晴らしいサイトでは、ドックの外観を完全に変更できます（多数のトピックを収集しました）。





スタック



スタック引き出し

スタックの主なタスクはデスクトップをアンロードすることですが、スタック自体を区別することが非常に困難になる場合があります。 Stack Drawersは、外観を整理するのに役立ちます。



ダの場所のスタック

そして、このプログラムはStack Drawersへの素晴らしい追加です。 すべてのスタックを自動的に検出し、ドラッグアンドドロップするだけでスタックのアイコンを選択できます。





階層ドック

多くの人が、ドックを介してカタログをナビゲートする機能がLeopardで削除されたことに不満を述べました。 このアプリケーションは、正義を返すのに役立ちます。





最近のアイテムスタック

最近のアイテムをよく使用しますか？ それでは、このチューニングはあなたのためです。 このコマンドを実行することにより、新しいスタックを追加します。これには、最新の10個のドキュメント、アプリケーション、サーバーなどが含まれます。



defaults write com.apple.dock persistent-others -array-add '{ "tile-data" = { "list-type" = 1; }; "tile-type" = "recents-tile"; }' ; killall Dock







マウスオーバーグラデーションをアクティブにする

このコマンドは、スタック上のアクティブなアイテムにグラデーションを追加します。

defaults write com.apple.dock mouse-over-hilte-stack -boolean YES ; killall Dock









メニューバー



不透明メニューバー

Leopardで最も議論の的になっている変更の1つは、半透明のメニューです。 OpaqueMenuBarを使用すると、メニューの不透明度を変更できます。





レオカラーバー

このユーティリティを使用すると、メニューの色を選択できます。また、メニューを丸くしたり、背景に合わせて画像を拡大したりして、メニューに収まらないようにすることもできます。





ファインダー



インターレースバックライト

青みがかったインターレースが気に入らない場合は、ターミナルで簡単なアクションを実行して削除できます。

defaults write com.apple.finder FXListViewStripes -bool FALSE ; killall Finder









サイドバーヘッダー

これを行うには、（Xcodeを使用して）ファイルを開く必要があります-/System/Library/CoreServices/Finder.app/Contents/Resources/English.lproj/LocalizableCore.stringsおよびSD5、SD6 .... 好みに合わせて変更した後、finderを再起動します。

killall Finder







Tulbarのアイコン

Finderツールバーにアプリケーションアイコンを追加できることがわかりました。 たとえば、QTで開けないファイルにVLCを使用する必要がある場合がありますが、ドックにVLCアイコンを配置したくない場合があります。 その後、Finderツールバーに配置できます。 必要なアプリをドラッグアンドドロップするだけです。 さらに、これらのアイコンとドックアイコンを使用できます。





アイコン



キャンディバー3

Candybarは、OS Xシステムアイコンを変更できる優れたアプリです。





IconFactory

IconFactoryは、CandyBarで使用できるMac用の巨大なアイコンリソースアーカイブです。



ログイン画面



Visageログイン

このアプリケーションを使用すると、ログイン（ログインウィンドウ）を完全に構成できます。 背景画像、アイコン、名前を変更し、表示する情報を入力できます。





DefaultDesktop.jpg

背景の画像のみを変更する場合は、このファイル/System/Library/CoreServices/DeafaultDesktop.jpgを置き換えるだけでよく、次回ログインするときに新しい背景が表示されます。



Quickook



フォルダーQuickLookプラグイン

QuickLookは、Leopardの最も重要な革新の1つです。 このプラグインを使用すると、ディレクトリを簡単に参照できます。





Zip QuickLookプラグイン

Zipアーカイブ内のファイルを開かずに表示できる別のQuickLookプラグイン。

fonts500

これはLeopardには適用されませんが、500の高品質の無料フォントが含まれているため、このサイトは注目に値します。
