Red Hat、Revolution R、Elasticsearch、MariaDB、Blockchain-as-a-a ServiceなどがAzure Marketplaceで利用可能

少し前、 Azure Marketplaceが独立したソフトウェア開発者（ISV）と、クラウド環境でアプリケーションとサービスを開発する通常のプログラマーとITプロフェッショナルの両方にもたらす機会について話しました。 他の資料では、たとえばJelasticなどのISVがAzure Marketplaceで既に公開されている方法と例について詳しく説明しました。











それ以降、この記事で説明するAzure Marketplaceで多くの興味深いことが起こりました。 開発者とITプロフェッショナルは、 Red Hat、Revolution R、Elasticsearch、MariaDb、STRATO Blockchain platformなどの新しいソリューションにアクセスできます。 すぐにもっと興味深いことが期待されますが、今のところは、ホットなニュースを見てみましょう。



赤い帽子



正式にサポートされている多くの-認定されたRed Hatソリューションの待望のニュースが、Microsoft Azureプラットフォームで利用可能です！ 機能の詳細については、 特別ページをご覧ください 。



Red Hatユーザーは、 Red Hat Cloud Accessの利点のおかげで、既存のライセンスをAzureクラウドに簡単に転送し、そこに会社の製品で仮想マシンを展開できます。



Azure Marketplaceは、数回クリックするだけでRed Hat Enterprise Linux 6.7およびバージョン7.2をデプロイするためのシンプルで簡単な方法を提供します。



革命r



R言語とRevolution製品を仕事に使用するすべての科学者、アナリスト、およびその他の専門家にとっての素晴らしいニュースは、Azure Marketplaceからの迅速なインストールが可能な公式のRevolution R Enterpriseサーバーバージョン7.4.1のリリースです。



サーバーは、 OpenLogic CentOS Linux 6.5ベースとWindows Server 2012 R2ベースの両方でインストールできます。



サーバーの最新バージョンは既にMicrosoft R Server 8.0ブランドでリリースされており、Azure Marketplaceで数回クリックするだけでインストールできます。 リクエストに応じて利用可能なクラウド環境で今すぐお試しください！



Elasticsearch



Elasticsearchプラットフォームは多くの人に知られています。 ペタバイトまでのデータのインデックス作成と検索のための一般的なソリューションが、Azure Marketplaceからの自動展開に正式に利用可能になりました！



Elasticsearchの製造会社であるElasticからのサブスクリプションは、商用ソフトウェアのライセンスとElasticスタックのエンタープライズレベルのサポートとともにAzureユーザーに提供されます。



Azure MarketplaceからElasticsearchとKibanaを今すぐお試しください 。



MariaDB



もう1つの有名なオープンソースプロジェクトであるMariaDBデータベースも、Azureのサポートを正式に発表し 、Azure Marketplaceを通じて迅速なクラスター展開が可能になりました。



MariaDB Enterprise Clusterは、Microsoft Azureでの3ノード展開と、CentOS 7を実行する追加のMariaDB MaxScaleノードを提供します。MaxScaleは、データベースノードの負荷分散と読み取り/書き込み分割、書き込み接続ルーター、読み取り接続ルーター設定の構成を自動化します。



Azure Marketplaceで利用可能なこの公式ソリューションを使用すると、ユーザーは数回クリックするだけでMariaDBクラスターを持ち上げることができます。

Azure Marketplaceは、開発者やITプロフェッショナルがAzureクラウド内のさまざまなソリューションに基づいてソリューションを展開し、環境を整理しやすくする強力なツールであると考えました。
