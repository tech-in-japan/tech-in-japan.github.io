Windows Live Wave 4のベータ版がインターネットに流出





非常に長い間、Windows Live Wave 4を待ちました。ついにダウンロードできるようになったので、Pirate Bayからダウンロードしなければならないことに誰も驚かないでしょう。 これまで見てきたほぼすべてのWindows 7ビルドと同様に、これもリークです。



Wave 4アプリケーションには、メッセンジャー、メール、フォトギャラリー、ムービーメーカー、同期、ライターのベータ版が含まれます。 なぜなら 同期とメッセンジャーは、Microsoftアカウントを持つ企業ユーザーのみが使用できます。まだユーザー名を使用してログインすることはできません...もちろん、メッセンジャーのパッチはすでに入手できます。



興味深いことに、すべてのアプリケーションでテープが使用されているにもかかわらず、OSにはMessengerのみが組み込まれています。 ほとんどの人がそうであるように、Wave 4の高密度アプリケーションは新しいMicrosoft OSに統合されると思いました。





Live Mailは大幅に改善され、Live Mail Wave 3とOutlookの間にランク付けされました。





Live Writerも少し改善されました（YouTubeやFaceBookなどのオンラインサービスとの統合に特に注意する価値があります）。 Windows用のデスクトッププログラムを使用するブロガーにはほとんど不可欠である以前のバージョンと比較して、さらに良くなっています。





Piccasaの普及率には劣りますが、Wave 4フォトギャラリーはすぐに登場します。 レタッチや調整、人、場所のマーク付け、説明の追加は簡単です。





ムービーメーカーの焦点は明確です。YouTube、Facebook、Liveにアップロードする動画をまとめます。



トレントを既にダウンロードしている場合、どう思いますか？ 期待は正当化されましたか？
