Nokia N810インターネットタブレット



ノキアの会社はまだ新しい「インターネットタブレット」の公式発表です。 そして、予想よりも早く。 10月17日、サンフランシスコで開催されたWeb 2.0 Summitで、ノキアは新しいデバイスを公開しました。



非公式情報の大部分に完全に準拠して、新しいN810インターネットタブレットは、格納式のQWERTYキーボードを受け取りました。



新しいフィンランドのデバイスのディスプレイは、対角線4.13インチのタッチスクリーンで、解像度は800x480です。 デバイスのプラットフォームはMaemo Linux 2008でした。デバイスの販売は、今年の11月中旬に479ドルの価格で開始されます（もちろん米国の価格）。







機能のうち、GPSナビゲーション、Wi-Fi、400 MHzプロセッサ、十分な内部メモリ、メモリカードのサポートに注目する価値があります。



ノキアN810インターネットタブレットの特長：



* TI OMAP 2420、400 MHzプロセッサ

* Maemo Linux 2008プラットフォーム

* 800x480ピクセルの解像度で最大65K色を表示する4.13インチディスプレイ

* DDR RAM 128 MB、フラッシュROM 256 MB

*最大2 GBの内部メモリ

* miniSDおよびmicroSDメモリカードのサポート（最大8 GB）

*内蔵GPSレシーバー

* VGAカメラ

* Wi-Fi 802.11b / g

* Bluetooth 2.0 + EDR（HID、FTP、DUN、GAP、SPP、HSP、SAP、およびOPPプロファイル）

* USB 2.0

* 3.5mmジャック

* SIPサポート

*ビデオ形式-3GP、AVI、WMV、MP4、H263、H.264、MPEG-1、MPEG-4、RV

*オーディオ形式-MP3、WMA、AAC、AMR、AWB、M4A、MP2、RA、WAV

*画像-BMP、GIF、ICI、JPE、JPEG、PNG、TIF / TIFF、SVG、Tiny、WBMP

*バッテリー寿命は最大4時間

*スタンバイ時間-最大14日間

*寸法-128x72x14 mm

*重量-226グラム
