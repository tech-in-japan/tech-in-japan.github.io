THL T11-有名な旗艦T100Sの予算オプション

THL社の電話、つまりTHL W7sモデルを初めて知ってから1年以上が経ちました。 このモデルは私に好印象を与え、運用の観点からはうまく機能しました。 この期間に公共部門でTHLが何を達成し、何が新しいものを提供し、何を驚かせるかが興味深いものになりました。

彼らには喜ばしいことがあることがわかった。 このレビューでは、周波数が1.7GHzの新しい8コアMTK6592 Octa Coreプロセッサー上に構築された予算のスマートフォンTHL T11についてお話します。 センセーショナルなフラッグシップTHL T100がこのモデルの祖先になったことは確かに知られています。







一部の小さな設計要素（金属フレーム）を除き、モデルのサイズと寸法は変わりません。 THL T11はより若くて安価なモデルで、画面解像度は1920x1080から1280x720ピクセルに低下し、同じ画面の対角線（5インチ）になります。また、メインの13メガピクセルカメラは8メガピクセルに置き換えられます。テストと私の操作経験。



はじめに



中国の会社THL（Technology Happy Life）は、2002年に深センの工業都市で設立されました。 彼女は、ISO 9001およびISO 14001の品質システムの国際認証と、国際標準CCC、CE、FCC、NCCCに基づく製品認証に合格しました。 今日、THLは中国最大の携帯電話メーカーの1つです。 Technology Happy Lifeが製造するハイテク製品は、中国全土で大成功を収めています。 同社は毎日、ますます多くの店舗を開き、ブランド「THL」のみを紹介しています。

▌包装



白い会社の箱は一年も変わっておらず、碑文だけが変わっています。 新しい会社のロゴは、以前は「THL」のように大文字ではなく、大文字の「thl」で書かれています。 箱は厚い段ボールでできており、外部の影響から電話機とアクセサリを保護します。







スマートフォンに加えて、キットには、ソケット用プラグ付き充電器、USBケーブル、有線ヘッドセット、保護フィルム、および英語のドキュメントが含まれています。







220Vネットワークからのきちんとした充電器、充電の出力電流は1 Aで、安定して過熱せずに充電します。 フル充電時間は2.5〜3時間です。







USBデータケーブルは高品質で、microUSBコネクタにしっかりと収まります。 有線ヘッドセットは予算が限られているため、音楽を高品質で聴くことは適切ではありません。 通常のヘッドフォンでは、電話はすばらしい音を出します。







製造業者は、バッテリーの容量が2750 mAhであることを示しており、バッテリー重量は42グラムをわずかに超えています（これは、少なくとも2500 mAhの実容量を間接的に示唆しています）。 バッテリーを搭載した電話機の重量は正確に142 gです。







輸送保護フィルムがスクリーンに接着されており（簡単に取り外すことができます）、別のより高品質の工場用フィルムがその下にあります。 携帯電話にはCorning Gorilla Glass 3が搭載されているため、保護フィルムは必要ありません。 ただし、メーカーは電話機に追加の保護フィルムを装備しています（必要に応じて接着できます）。 キットには、英語のマニュアルと2つのNFCタグも含まれています。







▌ デザインと人間工学



電話は白と黒で利用可能です。 背面カバーは、つや消し表面のプラスチックで作られています（手で滑らず、指紋はほとんど見えません）。 蓋はしっかり閉まり、初めて開くためにプラスチックカードが必要でした。 microUSBおよび3.5mmオーディオジャックが配置されている上、およびマイクと電話機の背面カバーを取り外すためのスロットが配置されている下は、波形の表面を持つプラスチックです。 画面やケースに非常に強い圧力がかかっていても、遊びやきしみは見つかりませんでした。 モノリシックガラスは、他の要素と同じ高さです。



THL T11には、顕著な角張った形状と人間工学に基づいた側面の丸みを帯びたオリジナルのコーポレートデザインがあります。 ケースの黒い色が画面の黒い色と結合します。 一般的な外観とデザインは、世界のブランドのレベルで携帯電話の快適な印象を作成します。 デバイスの寸法は次のとおりです。142グラムの重量で144.3 x 70.4 x 8.4mm。







フロントパネルには、5インチの静電容量式タッチスクリーン、イヤホン用スロット、フロントカメラ、3つのタッチボタン「メニュー」、「ホーム」および「バック」、マイク用スロット、近接センサーおよび照明センサーがあります。 スピーカーを覆うグリルの右側には、バッテリーの充電時に赤く点灯するイベントインジケーターがあります。 タッチキーには青みがかったバックライトが付いています。











背面パネルには、カメラレンズ、LEDフラッシュ、外部スピーカースロット（音がかなり大きい）、メーカーのロゴ、追加のノイズ低減マイクがあります。 カメラは体の上にはみ出さず、ほぼ結合します。 蓋の表面は粗く、手にすべらない。 ケースは触り心地がよく、指紋でほとんど汚れません。









右側にはロックボタンがあります。 ロックを解除するために、あなたは秒針で手を伸ばす必要はありません-すべてが明らかに右下の人の親指の下にあります。







左側にボリュームロッカーがあります。







上端には3.5 mmヘッドフォンジャックとmicroUSBコネクタがあります。 THL T11は、USBインターフェイス（OTG）を介したフラッシュドライブの使用をサポートしています。







下部には、背面パネルとマイク穴を取り外すためのくぼみがあります。







携帯電話は、正しい形状のクラシックなデザインです。 プロフィールで見ると、その裏蓋がゆるやかに曲がっているのがわかります。 携帯電話は手袋のように手にしっかりと収まり、滑り落ちません。







THL T11は快適なサイズと形状（丸みを帯びた縁、上下の面取り）を備えているため、片手で使用すると非常に便利です。 すべてのキーは快適でタイトなストロークを持ち、盲目的に簡単に検索されます。 画面上のほぼどこにでも指で簡単にアクセスできます。







背面カバーを取り外した後（きつく）、バッテリー、SIMカード（ミニおよびマイクロ）およびmicroSDカード用のスロットが見えます。 ふたの内側にNFCアンテナがあります。









▌分解



デバイスの内部を見るために、合計15本のネジを外して分解しました。 ネジの1つに保証ステッカーが付いていることに注意してください（通常の針で損傷せずに取り外すことができます）。







すべてのネジを外した後、ケースをプラスチックカードで2つに分割しました。 ケースの一部を取り外して、私は内部へのアクセスを得ました。







肉眼で見ると非常に有能な部品の配置は、開発者の主な目標が、ケースの厚さを増やさずに、バッテリーの空きスペースを最大化することであることは明らかです。



ディスプレイ



THL T11は、高コントラストのIPSマトリックスを備えた5インチの容量性タッチスクリーンディスプレイを受け取りました。 解像度は1280 x 720ピクセルで、密度は294 PPIです。 ディスプレイは、GFFテクノロジー（OGSに類似）を使用して作成されます。これは、ディスプレイとセンサー/ガラスを単一のユニットに結合するテクノロジーで、デバイスの厚さを減らし、画像を鮮明にし、感度を高めます。

ディスプレイは、非常に明るく飽和した画像と優れた色再現を提供します。 180°に近い視野角。 最大10タッチのマルチタッチがサポートされています。 感度は良好で、タッチスクリーンに問題はありませんでした。 画面は、第3世代のCorning Gorilla Glassによって保護されています。 最大のバックライトでは、画面は太陽の明るい光線の下でも読み取り可能です。











▌ 鉄



MT6592 Octa Coreには、最大700 MHzの4コアMali-450MP4ビデオアクセラレータが搭載されており、MT6582にインストールされたMali-400MP2よりも約2倍高速です。 MT6592 Octa Coreは最新の28nmプロセス技術で実行され、1.7 GHzの周波数を持つ8個のCortex-A7コアが統合されています。 第2レベルのL2キャッシュは1MBに増加し、サポートされるメモリはLDDR2 533MHz / DDR3 666MHzおよび13MPカメラです。 ビデオ録画は、1080P 30fp、1080P 30fps H.264デコード、および2つの新しいHEVC / H.265およびVP9形式で実行できます。 これに加えて、デバイスには2 GBのRAMと16 GBの内部メモリがあり、microSDメモリカードを使用して拡張できます。

コミュニケーション



THL T11は2枚のSIMカードをサポートします（1枚はマイクロフォームファクターです）。 SIMカードのインターフェイスと設定は、MTKチップセットの標準です。 設定では、通話、SMS、モバイルインターネットのメインカードを選択できます。 また、ハニカムの色はそこで調整されます。 音声通信やデータ送信に問題はありませんでした。 スマートフォンは信号を完全に保持します。 WCDMA 850/1900 / 2100MHzの全範囲が存在することは注目に値しますが、多くの中国の電話では1900が欠落しています。







電話機にはWi-Fiモジュール標準802.11 a / b / g / nがあり、不具合やダンプは見つかりませんでした。 テストは、1つまたは2つのレンガの壁を通して、アクセスポイントの近くで直接実行されました。 また、携帯電話のWiFiモジュールが5Ghzネットワークで動作することもわかりました。







GPSはアナログレベルでうまく機能します。5秒後に、窓で13個の衛星のうち10個が可能になりました。 結論：携帯電話は、車のナビゲーターとして完全に使用できます。







THL T11はBluetooth 4.0をサポートし、ステレオヘッドセットで問題なく接続し、干渉やノイズなしにステレオ信号を良質で送信します。







▌音



携帯電話の音、特にメインスピーカーの音は良好です。中国のデバイスでは、このような音量、明瞭さ、高品質の音の組み合わせを長い間聞いていません。 マイクは路上でも屋内でも敏感で、イヤホンは大音量で、対談者と私にとって完全に聞こえます。







▌ インターフェースとソフトウェア



THL T11は、ロシア語を完全にサポートするAndroidオペレーティングシステムバージョン4.2.2で動作します。 インターフェイスが飛ぶと、アプリケーションはほぼ瞬時に開き、ブレーキなしでページをめくる。



すぐにファームウェア（14/01/04）を最新（06/23/14）に更新しました。これは、必要なすべてのソフトウェアへのリンクを含む簡単な指示です。

ダウンロード：

1） SPフラッシュツール

2） MT6592 USB VCOMドライバー

3）電話の最新のファームウェア

インストール：

1）SP Flash Tool、MT6592 USB VCOMのアーカイブを解凍し、ファームウェアでアーカイブします

2）スマートフォンの電源を切ります。 ケーブルがコンピューターに接続されていません。

3）リセットボタンを押す

4）電話をコンピューターにケーブルで接続します。 オペレーティングシステムは新しいデバイスを検出します-システムにMT6592 USB VCOMドライバー（OSバージョン用）を含むフォルダーを指定します

5）コンピューターからケーブルを外します

6）プログラムSP Flash Toolを実行します

7）Download_Agentボタンをクリックして、ファイルMTK_AllInOne_DA.binを選択します（プログラムフォルダーにあります）。

8）[スキャターローディング]ボタンをクリックし、ファイルMT6592_Android_scatter.txtを選択します（ファームウェアが含まれるフォルダーにあります）。



9）[ダウンロード]ボタンをクリックします。

10）ケーブルで電話をコンピューターに接続します。 プログラムはそれを見て、ファームウェアを起動するはずです。

11）ファームウェアプロセスが正常に完了したことの確認は、「OK」と記された緑色の円になります。 ファームウェアの終了後、電話の電源を切ります。

12）完了。







システムの外観は古典的で、美しい黒と青の配色で厳格です（OTA-ワイヤレスアップデートが存在します）







ファームウェアには、Playストアやその他の標準のAndroidアプリケーション（オーディオおよびビデオプレーヤー、計算機、時計など）を含む主要なGoogleサービスが含まれています。 また、ファイルマネージャー、メモ、懐中電灯、その他のアプリケーションも追加されました。







革新の1つは、NFCモジュールとそれを操作するためのファームウェアの存在です。







「スマートウェイク」の存在に注意してください。この機能を使用すると、画面上のタッチとジェスチャを使用して電話を制御できます。 ジェスチャーの例：上にスワイプ-スマートフォンのロックを解除します。 draw "c"-ダイヤラー。 下にスワイプ-カメラを起動し、「e」を描画-ブラウザを起動します。







USBホスト機能（マウス、キーボード、外部ドライブなどの接続）、コンパス、10タッチマルチタッチ。







一般に、電話のインターフェースはきれいです。つまり、 過剰なソフトウェアに圧倒されず、メーカーのシェルがありません。 変更のために、Floatアプリなどのいくつかのアドオン、およびサウンド拡張に関連するアドオン（BesAudEnhおよびBesLoudness）が追加されました。 ClearMotionを使用すると、テレビの画像をより滑らかで鮮明にすることができます。 加速度計、近接センサーなどを調整することもできます。



▌ カメラ



どちらのカメラも写真の品質に満足しています。 背面カメラの実際の解像度は8メガピクセルで、最大4096x3072ピクセルの解像度で写真を撮影できます。これは、13メガピクセルでソフトウェア補間されます。 オートフォーカスと明るいフラッシュがあります。 HDRサポートにより、拡張された照明範囲で写真を撮ることができます。 オートフォーカスは、フレーム内の変化する状況にすばやく反応し、近くのオブジェクトを撮影することができます（フォーカスエリアを手動で指定することもできます）。

フロントカメラの解像度は5メガピクセルで、ソフトウェアは12メガピクセルに補間されます。 カメラはうまく機能し、写真は非常に良く、オートフォーカスもあります。

工場でリアカメラに青い保護フィルムが貼り付けられているため、カメラはホワイトバランスとショットのシャープネスと間違われる可能性があるため、取り外す必要があります。

▌バッテリー、稼働 時間



メーカーが示すバッテリー容量は2750 mAhで、ネイティブ充電器（1 A）からのフル充電には2.5〜3時間かかりますが、バッテリーは最大4.3Vまで充電されます。 最大輝度のビデオクリップ720p_100fpsの再生モードでは、スマートフォンは6時間持続しました。 スタンバイモードでは、Wi-Fi、GPS、モバイルネットワークがオンになっている状態で、電話機は非常によく保持され、3日間で充電の15％が失われました。







私は2日間完全に充電しました（テスト、インターネット、ビデオ、ゲーム、音楽）。 ゲームやビデオを積極的に使用する場合は、1日（ゲームの経過の強さに応じて9〜12時間）を当てにする必要があります。
