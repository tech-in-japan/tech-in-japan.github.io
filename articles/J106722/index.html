Quest vWorkspace 7.2 Beta 2の新機能

先日、QuestはvWorksapce VDIソリューションの新しいベータバージョン7.2（ビルド409）（ベータ2）を発表しました。 このアセンブリの新機能 よく見てみましょう！

ベータ1と比較すると、次のように表示されます。

1. vWorkspace Reporting-現在時制と過去時制のスライスの両方で、ユーザーと管理スタッフのアクティビティに関するレポートを作成できます。 私の意見ではなく、非常に有用で必要なチップ。

2. VMware vSphere 4.1ハイパーバイザーの完全かつ公式なサポート-以前のバージョンのvWorkspaceはvSphere 4.1での動作をサポートしていましたが、正式にはサポートしていませんでした。

3.統合されたvWorkspace 7.1 MR1アップデート。



ただし、現在のリリースではベータ1のすべての機能が保持されていることを忘れないでください。つまり、

1. クイックスタートウィザード -パイロット、テスト、または工業用の完全に再設計されたvWorskpace構成ウィザード。

2. 製品インストーラーを更新しました 。

3. PoweShellのサポート -ここにあります！ -vWorkspace管理を自動化するためのコマンドレットのサポート。 個人的には、私は長い間この機能を待っていました-今では管理ルーチンを自動化できるからです！

4. 新しい製品ライセンスモデルのサポート -VMACを製品認証のためにQuestに転送することなくライセンスが可能になりました。これにより、システム操作が確実に簡素化されます。

5. 改善されたエラー報告システム -さまざまな問題の解決策を提供できるようになった改訂されたメカニズム。

6. 仮想デスクトップフェールオーバーモード -永続的なVMが利用できない場合、同じVMグループ（プール）からの一時的なVMがユーザーに展開されます。
