Tesla Model Sにソフトウェアをインストールできます





サンフランシスコで開催された会議で、Tesla MotorsのディレクターであるElon Muskは、Tesla Model Sスポーツ電気自動車がサードパーティソフトウェアをサポートすると述べました。



Model Sセダンには、17インチの巨大なタッチスクリーンとAndroidオペレーティングシステムが標準装備されています。 この車のファンは、あなたの好みに合わせてプログラムし、独自の「ハック」をインストールすることが可能だと長い間疑っていましたが、今では公式に確認されています。







したがって、モデルSは、オープンなオペレーティングシステムを搭載した最初の車かもしれません。 今日まで、すべての自動車メーカーは、外部の干渉から可能な限り自動車のコンピューターを保護しようとしており、サービスステーションでのみ、メーカーが提供する方法でのみコンピューターを再プログラムすることができました。
