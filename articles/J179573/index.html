Intel®Perceptual Computing Challengeフェーズ2を開始しました





1週間前の5月6日、Intel Perceptual Computing Challengeの第2フェーズが正式に開始されました。 このニュースはハブのIntelの企業ブログに掲載されなかったので、私はそれについて自分で書くことにしました。 カットの下の詳細。



だから、私が長い間待ち望んでいたことがついに起こりました-IntelはIntel Perceptual Computing SDKを使用したアプリケーションの競争の第2フェーズを発表しました。 これはまさにフェーズであり、2回目のラウンドではありません。 つまり、最初に参加しなかった人もこのフェーズに参加できます。



最初のフェーズに関する主な違い：

  1. より大きな賞品。 最初の段階で最大20,000ドルを獲得できた場合、2番目の段階では、マーケティングと開発に関する協議のために100,000ドル+ 50,000ドルになります。
  2. より深刻なアプリケーション要件。 最初のフェーズでは、アプリケーションの動作を示すビデオを送信するだけで十分でした。2番目では、アプリケーション自体を送信する必要があり、Intelスタッフによってテストおよび評価されます（ただし、ビデオとドックも送信する必要があります）


数が少ない


私は競争のすべてのルールを語るのではなく、主なことに集中します。 隠されたものから始めましょう-賞品で。

賞品


  1. 大賞 ：彼はコンテスト全体で唯一のものです。 賞金：100,000ドル+マーケティングおよび開発コンサルタントの50,000ドル
  2. 最優秀賞 ：アプリケーションの各カテゴリで1位（それぞれ合計4つのカテゴリ、1位も4）。 賞金サイズ：75,000ドル+マーケティングおよび開発コンサルティング用の12,500ドル
  3. 2位賞品 ：各カテゴリに3つ。 10,000ドルを与える
  4. 3位賞品 ：カテゴリごとに3位20位。 5000ドルを与えます
  5. プロファイルフォーム賞 ：プロジェクトプロファイル（7月17日まで）に記入し、最終的にはコンテストの終了前にアプリケーションの実際のデモを送信した方に10個の賞。 1000ドルを与える
  6. 早期提出賞 ：最終デモアプリケーションを8月6日（コンテスト終了の1週間前）までに送信する人には10個の賞も。 1000ドルを与える


金額の約30％が税金を消費することを忘れないでください。

税について
競争の規則からの抜粋：

米国以外のすべての受賞者については、認められた賞金の30％が源泉徴収され、米国の税法を順守するためにIRSに送金されます。

ステージの簡単な説明：私が理解しているように、6月17日までにプロジェクトを登録し、その説明を追加するだけです。 その後、6月26日にはすでに最初の重要な瞬間があります-750のいわゆるファイナリストが説明から選択され、次のステージに入場します。 これらのファイナリストのうち1人だけが賞品を獲得できます（カウントするのが難しい場合を除き、117の賞品）。したがって、すでに750人のファイナリストの1人になっている場合、少なくとも何かを獲得するチャンスは非常に大きいです。



カテゴリー


アプリケーションを登録するとき、4つのカテゴリのいずれかを選択する必要があります。 カテゴリの境界はかなりぼやけています。 自分で判断する（説明を翻訳することを敢えてしなかった-チャンスが大きすぎて元のメッセージを歪めることはできない）：

  • ゲーム ：ゲームはテクノロジーの限界を押し広げ、ユーザーとの対話でイノベーションを推進しました。 このカテゴリを選択して、ゲームを体験に変えるか、まったく新しいゲームメカニックを作成します。 Intel Perceptual Computing SDKを使用して、これまでに見たことのないものを作成し、愛し、または開発したゲームを強化します。
  • オープンイノベーション ：枠を超えて考えてください。 ワイヤーを超えて考えてください！ SDKの機能を活用し、ユーザーを従来のコンピューティングモデルから解放するエキサイティングで革新的なエクスペリエンスに変えるアイデアにこのカテゴリを選択します。 独自のルールを確立し、想像しただけの経験を作成します。
  • クリエイティブユーザーエクスペリエンス ：知覚コンピューティングの使用モード（指/手の追跡、音声認識、顔の分析、2D / 3Dオブジェクトの追跡）の任意の組み合わせを使用してアプリケーションと対話するための革新的な手段を開発した場合、このカテゴリを選択します単一の使用モードでも。 ユーザーにとって記憶に残る革新的なインタラクションを作成します。
  • 生産性 ：より速く、より良く、より簡単に物事を行う方法を作成します。 知覚コンピューティングの使用モード（つまり、指/手の追跡、音声認識、顔の分析、または2D / 3Dオブジェクトの追跡）を使用して、非ゲームアプリケーションでタスクを達成するユーザーの能力を強化した場合は、このカテゴリを選択します。 マウスとキーボードを交換する必要はありませんが、知覚コンピューティングの使用を使用して、タスクを遂行するユーザーの能力を強化したいことに留意してください。




よくある質問


FAQから翻訳することも決めました。 私はそれが誰かに役立つことを願っています（しかし、もちろん、オリジナルを読むほうが良いです）。



無料の翻訳リテールFAQ
Q：なぜアイデアを提出する必要があるのですか？ アプリケーションをすぐに提供できますか？



A：インテルは最高のものを探しているので、Intel Perceptual Computing SDKを使用する最も革新的なアイデアを探しています。 すでに何かに取り組んでいる場合は、「ビジュアルサポート」列にこれへのリンクを記入してください。 持っているものすべてを提供してください。おそらく、あなたはCreative * Interactive Gesture Camera Kitと賞金を獲得するチャンスを受け取る750人のファイナリストのうちの1人になるでしょう。



住所を指定する理由

アイデアを提出するときのみ、住所を提供する必要があります。 750人のファイナリストの中にいる場合、このアドレスを使用して自動的にカメラ（Creative * Interactive Gesture Camera Kit）を送信します。



ファイナリストに参加していない場合、完成したアプリケーションを送信できますか？

いや ファイナリストに応募した人のみが申請書を送信できます。



デモアプリとは何ですか？

競争の前のフェーズでは、アプリケーションのプロトタイプのみをリクエストしました。 ただし、この第2フェーズでは、すでにデモアプリケーションを検討しています。 デモアプリケーションがよりコーム化され、品質がリリース候補のレベルになることが期待されます。 機能の面では、デモは完全に機能するアプリケーションと同じようにする必要はありませんが、アプリケーションの主な機能を示す必要があります。



コンテストにいくつアイデアを提出できますか？

数に制限はありません。 一度登録すると、いくつかのアイデアを送信できます。



ファイナルにはいくつのアイデアが選ばれますか？

最大750のアプリケーションファイナリストを選択します。 ユーザーは、ファイナリストで複数のアイデアを選択できます。



誰が参加する資格がありますか？

コンテストは18歳以上のすべてのデベロッパーに公開されており、アメリカ合衆国、イギリス、カナダ（ケベックを除く）、韓国、ドイツ、インドネシア、オーストラリア、フランス、スペイン、ベルギー、日本、中国、ロシアのいずれかに居住しています。 、インド、イスラエル、台湾。



チームで働くことはできますか？

はい、もちろんです。 個人開発とチーム開発の両方をお勧めします。 チーム開発のために、チームの1人が登録する必要があり、チームが勝った場合、彼はチームでの賞品の配布に責任を負います。



既存のアプリケーションを送信できますか？

はい、市場の戦闘条件で磨かれたアプリケーションを送信することを強くお勧めします。 ただし、パーセプチュアルコンピューティング機能の追加により、アプリケーションがどのように変化するかを正確に知る必要があります。



以前にインテルのコンテストに参加したアプリケーションを送信できますか？

はい、以前はインテルのさまざまなコンテストに参加していたアプリケーションを送信できます。 ただし、アイデアの説明には、現在の競争でアプリケーションを開発または拡張する方法を示す必要があります。



会社に代わって参加できますか？

はい [会社に代わって送信しています]の横にあるチェックボックスをオンにして、会社名を入力してください。 このオプションが選択された場合、賞は会社に授与されます。



アプリケーションはどのように評価されますか？

申請は、申請の提出期限が迫るとすぐに評価されます。 それらはいくつかの基準によって評価されますが、ここでは基準の完全なリストではありません。知覚コンピューティングの使用度、使いやすさ、機能性、創造性、アプリケーション使用シナリオの説得力を示しています。



アプリケーションはIntelの財産になりますか？

いや アプリケーションは開発者の所有物のままですが、インテルはアプリケーションのアイデアおよび/またはデモ版をデモンストレーションおよびプロモーションする権利を留保します。



私の個人データは安全ですか？

はい Intelはプライバシーを真剣に考えています。



SDKはどこで入手できますか？

SDKはwww.intel.com/software/perceptualからダウンロードできます。



コンテストに参加するためにSDKを使用する必要がありますか？

はい Intel Perceptual Computing SDK 2013の使用は、ファイナリストアプリケーションのデモの前提条件です。 アイデアを送信する前にSDKを確認してください。



参加するにはカメラが必要ですか？

アイデアを送信するためにカメラは必要ありません。 ファイナリストは、デモアプリケーションを完了するためにカメラを受け取ります。 カメラの機能については、 software.intel.com / sites / default / files / article / 325946 / creativelabs-camera-productbrief-final.pdfで説明しています。



私はファイナリストです。カメラを保持できますか？

ファイナリストは、選択されるとすぐにインテルからカメラを受け取ります。 ファイナリストに選ばれた場合、アプリケーションのデモ版を送信すれば、カメラを自分用に残すことができます。



なぜすべての国が参加できないのですか？

最大限の数の国が参加できるように努力しています。 問題は、さまざまな国の競争の規則および法律の遵守、ならびにさまざまな国のCreative *カメラの認証に関する問題のために発生します。



さらに質問がある場合はどうすればよいですか？

フォーラムにアクセスしてください： software.intel.com/en-us/forums/intel-perceptual-computing-sdk
