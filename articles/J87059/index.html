ASUS K52-ノートパソコンとヘリコプターについて

ごく最近、私は新しいAtomsのリリースで、すべてのネットブックが開発スパイラルで新しいラウンドを行ったと書きました。 そして、今日、他のすべてのデバイス（ラップトップを含む）について同じことを言うことができます。ご存知のように、年の初めにIntelはCoreファミリーの多くの新しいプロセッサーを導入しました。







私はこれらのラップトップのいずれかを手に入れることができました...まあ、あなたは知っています。



一般的に、モデルの名前は、意図せずにいくつかの関連付けを引き起こします-はい、はい、非常に伝説的なKa-52 アリゲーターヘリコプターと。 しかし、ここですべての類似点が終わります。 モデル名の5つは、画面の対角線（それぞれ15インチ）を示しています。2つは、第2世代モデルがあることを示しています。











残りは、新しいプラットフォーム上に構築された最初のラップトップ（ASUSだけでなく一般的なもの）の1つで、その「心臓」はクロック周波数が2.26GHzのデュアルコアIntel Core i5 430Mプロセッサーです（簡単に2.53GHzに加速します）。 新製品は、かなり生産的なフィリングで補完されます-ギガバイトのメモリ、4ギガバイトのRAM、およびハードドライブ上の別の500を備えたディスクリートATI HD5470グラフィックスカード 。 面白い？ もちろんそうです！



フルバージョンの詳細



外観



このラップトップはKシリーズの論理的な延長であるため、設計が先駆者からほぼ完全に継承されていることは驚くことではありません。 ただし、わずかな違いがあります。







たとえば、今では、ラップトップを開くときに、ユーザーは「ニスを塗ったグランドピアノ」ではなく、心地よい粗い（テクスチャ）表面に手を置くことができます。 メーカーは、マット表面がより実用的であることを本当に認識しているようです-ますます多くのデバイスで使用され続けることを願っています。















側面には、3xUSB、HDMI、D-Sub、カードリーダー、2つのオーディオコネクタ、ギガビットRJ-45ポート、ディスクドライブ（DVD）など、ほぼすべてが必要です。















画面の対角線は15.6インチで、標準解像度は1366x768（16：9）です。 LEDバックライト、すべてのもの-写真は本当に良いです。



キーボードがいっぱいで、デジタルブロックがあります。 ボタンは大きく、角は丸く、小さな隙間で区切られています。 レイアウトは標準ですが、このような多数のキーでは、2番目のFnボタンが欠落している場合がありました（スペースバーの右側）。 キーの移動は小さく、入力時にクリックしません。















タッチパッドは、キーボードの下の身体の一部にある小さな長方形のくぼみの形で作られています-このソリューションは、ほこりがたまらないようにします（角や異種の材料がないため）。また、同時に、「ブラインド触覚」検索に貢献します。 他の場所と同様に、その下には大きな結合されたクロムボタンといくつかのLEDがあります（これらは既にフロントパネルにあります）。











デバイスの重量は、バッテリーおよび充電器を含む2.6 kg（すべて3）です。ラップトップの寸法は380x255x34.5-35.7 mmです。



性能



電源を入れるとすぐに、ネットブックの前面（タッチパッドの下）でいくつかのステータスLEDが点灯します。 仕事からのノイズはほとんど聞こえません-デバイスは本当に静かに動作します。



ハイパースレッディングはデュアルコアCore i5の両方のコアで実行されるため、プロセッサは最大4つのスレッドを同時に処理できます。最近まで、Core 2 Quadなどのプロセッサでのみ利用可能でした。 どうやら、これはプロセッサのパフォーマンスが高いためです。







自分で判断する-算術テストSuper Piでは、K52Jrラップトップでの最大100万のPiの数は17.815秒であると見なされます。 ファッションラップトップASUS VX5 Lamborghini （Core 2 Quadで）は22.324秒で同じことをします ！ そして、これはK52が3倍安いという事実にもかかわらず。







このモデルの興味深い点は、1GB DDR3およびDirectX11をサポートするディスクリートATI Mobility Radeon HD5470グラフィックスカードです。 より正確には、カード自体の存在ではなく、必要に応じて、統合された（プロセッサ内の） Intel HDグラフィックスビデオシステムを個別のシステムに切り替える機能ほど興味深いものではありません。







たとえば、ネットワークからの作業では、バッテリーを節約して最大限に「絞る」ことはできませんが、外出中は、統合されたビデオで十分です。 このテクノロジーはIntel Switchable Graphicsと呼ばれ、間違いなく生命権を持っています。 比較のために、3D Mark 2006の合成テストでは、統合されたvidyuhaは1350オウムを獲得しますが、離散-4180を獲得します。







内部には、500GBのハードドライブがインストールされています。その速度特性は次のとおりです。











Windows 7の初期バージョンは、ラップトップに最初にインストールされ、そのイメージは念のために隠しディスクパーティションに記録されました（15分で完全に機能するAxisを取得できます）。 パフォーマンスインデックスは5.1（7.9のうち）の価値があり、グラフィックカードが弱点です（ハードドライブがそれに続きます）。



消費者の観点から言えば、ラップトップは本格的な作業に十分です。テキストやインターネットコンテンツに煩わされることから始まり、Photoshopなどのエディターで終わります。 HDムービーは問題なく最大1080pで動作しました（プロセッサの負荷は約10％しかありません）。 また、プレイすることもできます-多くの現代のゲームで...しかし、もちろん、最大の設定ではありません。



バッテリーの寿命は約3〜3.5時間です。



計量制御



スタイリッシュなマット（内側）ケース。

便利なキーボードとタッチパッド。

高性能;

ドライブとHDMIコネクタの存在;

光沢のあるカバー（ほとんど汚れにくい）とスクリーンフレーム（汚れやすい）;

ExpressGateテクノロジーはありませんでした。

かなり高い価格。



2番目のマイナスに関しては、おそらくそれが何であるかを知らない人もいます。 そして、私たちは素晴らしいテクノロジーASUS ExpressGateについて話しています。ASUSExpressGateは、起動後7〜8秒でラップトップで実行される小さなオペレーティングシステムです。 ファイルマネージャー、プレーヤー、完全なSkypeおよびFirefox、いくつかのプロトコル（ICQおよびGTalkを含む）用のIMクライアントなど、多くの機能を実装しています。新しいデバイスでこのチップが表示されないのは奇妙でした。



ASUS K52Jrラップトップモデルには、 Core i3 350MプロセッサとCore i5 430Mプロセッサの2つのバージョンが販売されてい ます 。 もちろん、2番目はもっと興味深いものです。この記事で説明したのは彼についてでした。 しかし、価格は29000ルーブルから始まり、「より若い」バージョンでは26から求められます。ほとんどの反危機的な申し出ではありませんが、それは競争よりも私には思えます。 一方、新製品は、新しいプラットフォームでは、決して安くはありませんでした。



終わり



Intelの新しいプロセッサは非常に優れており、効率的で、考え抜かれ、コンパクトで、適度な「食欲」を持っています。 少なくとも、別のラップトップの例でこれを確認することができました。おそらく、さらに多くのラップトップがあるだけでも嬉しいです。 また、ASUSがバグを処理できることも非常に嬉しく思います。シリーズの続きのスタイリッシュなマットケースは、この証拠です。







レビューのラップトップは、ラップトップがどうあるべきかの良い例です。 すべての必要なインターフェースを備えた便利で実用的-多くの人が日常のコンピューター活動に必要とするものを正確に。 堆積物から残っている唯一のものは、ULシリーズの魅力的なデバイスに匹敵する、かなり高い価格です。 何らかの理由で、私は常にASUSの「K」シリーズは予算のものだと思っていました。
