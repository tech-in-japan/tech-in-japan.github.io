10ペタフロップスのスーパーコンピューター「K」の構築が開始されました

富士通と日本最大の理化学研究所は、10ペタフロップス（10 16 、つまり1秒あたり10兆回の操作）の推定性能を持つスーパーコンピューターの組み立てを開始しました。 名前「K」は、日本語の単語「Kei」に対応し、10 16を意味します。



比較のために、 スーパーコンピューターの世界ランキングにおける今日のリーダーのパフォーマンスはわずか1.76ペタフロップスであり、最も強力なロシアのスーパーコンピューターは0.35ペタフロップスです。







この設計には、800を超えるサーバーラックと、80,000を超える富士通SPARC 64 VIIIfx 2.2 GHzプロセッサー（各128ギガフロップス）と水冷が含まれます。





Fujitsu Venus Sparc64-VIIIfxマザーボード水冷



スーパーコンピューターのユニークな機能は、世界で初めて使用される6D Mesh / Torus （豆腐）ネットワークのトポロジーです。





BlueGene / Lスーパーコンピューターで使用されていた 、前世代の3D Mesh / Torusのトポロジー



6Dメッシュ/トーラストポロジは、3Dメッシュ/トーラスよりもさらに多くのネットワーク上のノード間の接続を提供します。 この方法により、コンピューティングプロセスをグループ化するのが簡単になり、単一ノードに障害が発生した場合、周囲のノードのスループットが低下することはありません。



12ノードの各ブロックは、3つの追加の通信パス（a、b、c）とともに3Dグリッド（x、y、z）に配置されます。 要するに、6D Mesh / Torusスキームは、 このプレゼンテーションで13〜16ページで説明されています。







スーパーコンピューター「K」は2012年秋にコミッションされます。
