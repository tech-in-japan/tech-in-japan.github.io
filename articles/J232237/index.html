3日間のプログラム「Creating a global IT startup」の登録最終日







スタートアップを作成し、国際市場に持ち込み、投資を呼び込み、シリコンバレーに駐在員事務所を開設する方法は？



これらおよびその他の質問への回答を得るために、キエフモヒラビジネススクールは、 GeeksLabプロジェクトの支援を受けて、 8月7日から9日にキエフで開催される3日間のプログラム「 Creating a global IT startup 」に招待します。 このプログラムの著者および教師は、 rollApp Vlad Pavlov （カリフォルニア州パロアルト）のゼネラルディレクターであるシリコンバレーの起業家です。



トレーニング中、3つの観点からスタートアップライフサイクルを検討します。

  • 起業家-ビジネスアイデアの選択、国際チームの作成、ビジネスモデル、法人化、ビザ取得など
  • 投資家-ベンチャーキャピタルファンドの「内部メカニズム」、投資決定基準、投資家とスタートアップの相互作用のプロセスなど。
  • 組織的および物流上の問題-熊手を踏まない方法。


セミナーでは、地域のスター代表者とのライブ会議や電話会議も行われます。

  • マックススキビンスキー（サンフランシスコ）-アンドリーセンホロウィッツ財団のパートナー。過去に7つのスタートアップを立ち上げ、多数の企業の投資家/アドバイザーでもありました。
  • Igor Shoifot（サンフランシスコ）-連続起業家および投資家、10人以上の創立者、シリコンバレーの40以上の新興企業の投資家、ウクライナ最大のビジネスインキュベーターHappy Farmの役員。
  • Evgeny Sysoev（Kiev）-AVentures Capitalのマネージングパートナー。最近のiForumでは、ウクライナのベンチャーキャピタル市場で最も影響力のある人物と呼ばれていました。
  • Evelyn Buchatsky（Kiev）-EastLabsアクセラレータの管理パートナー。
  • Maxim Shkolnik（Kiev）-Address.uaプロジェクトの創設者、以前はViewdleのSVPであり、Technobridgeベンチャー企業の管理パートナー。
  • Nikolay Mityushin（モスクワ）-ABRT Venture Fundの投資ディレクター。
  • Sergey Toporov（モスクワ）-LetaapitalVenture Fund、シニア投資マネージャー。


最初の2日間は、参加者が講義を聴き、実践的なタスクを実行し、招待されたスピーカーとコミュニケーションを取ります。3日目にビジネスゲームが開催されます。



プログラムの詳細な説明は、 ウェブサイトとFacebookで見つけることができます。



プログラムに参加するには、プログラムページの登録フォームに記入し、全額を支払う必要があります。



ユニークな知識を獲得し、グローバル市場に焦点を当てたビジネスを作成する機会を利用してください！
