ニュースとリリース



  • PHP 5.0.0の10年 -2004年7月13日には、PHPインタープリターバージョン5.0の光が見えました。 参照による投稿では、時代の主な出来事とPHPとTamlineの簡単な歴史。
  • 0xDBE：First Look -JetBrainsは、データベースとクエリを操作するための新しい0xDBE製品を発表しました。 このツールは、適切なアプリケーションを提出した後、クローズドアーリーアクセスプログラムの一部として利用できます。 リンクの投稿で、ツールの機能の概要を見つけることができます。
  • WordPress 4.0 Beta 1およびBeta 2-誰かがリリースのコードとアーキテクチャの大幅な更新を期待していましたが、これは起こりませんでした。 それでも、WordPress 4.0は多くの新機能を歓迎します。 また、WPでJavaScriptコードのシェアが増加する傾向についての興味深い投稿 。
  • Pickle -PECL拡張機能マネージャーの最初の安定版リリース。Composerと互換性があり、標準のPHPパッケージに含まれている場合があります。
  • Deployer Big Update-アプリケーションをデプロイするためのシンプルで便利なPHPツールであるDeployerは 、多くの新機能、更新されたサイト、ドキュメントを受け取りました。
  • SymfonyのREST - Symfonyに RESTfulサービスを実装するためのFOSRestBundleおよびNelmioApiDocBundleバンドルがありますが、これはすべてとは程遠いものです。 Symfonyの既存のRESTソリューションを改善、普及、簡素化するために、特別なワーキンググループが作成されました 。 グループの作成者の投稿でもう少し。
  • SensioLabsProfiler -Symfonyの作成者によるPHPプロファイラーのクローズドベータテストを開始しました。




Php



  • RFC：Uniform Variable Syntax -Nikita Popovからの提案は、PHPでの変数と呼び出しの単一構文の導入を含み、ほぼ満場一致で採用され、PHP 5 ++に含まれます。 反対票を投じる唯一の人物は著者Xdebug Derick Rethansで、彼の見解を投稿で概説しました。
  • RFC：次のPHPリリースのタイトル -php.internalsでは、PHP 5 ++の数値の選択に関する議論が積極的に再開されました。 結局のところ、PHP 6とPHP 7のどちらですか？
  • RFC：キャストを使用したスカラー型のヒント -この提案は、型（int、float、bool、string、numeric）の型ヒントの実装を暗示しています。 元のバージョンはAnthony Ferraraによって提案されました。AnthonyFerraraは、 主張するすべてのチャーター提案を撤回しました 。 おそらく今回は提案にチャンスがあります。
  • RFC：intdiv（） -PHPで整数除算の可能性を実装する提案： var_dump(3 %% 2); // int(1)



    var_dump(3 %% 2); // int(1)



  • PHPNGのベンチマーク！ -Zeev Suraskiからの興味深い投稿。彼はPHPNGとHHVMを比較し、 Magentoの別の投稿を含む人気のあるPHPプロジェクトでの古典的なPHPを比較しています。 一連のテストでは、PHPNGはHHVMをバイパスします。 ただし、HHVM開発者による代替テストでは 、わずかに異なる結果が示されます。 それにもかかわらず、PHPNGのパフォーマンスは向上しています-すでにPHP 5.6の2倍であり、HHVMに徐々に追いついています。
