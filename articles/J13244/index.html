Feedburnerの概要



FeedburnerはRSSサブスクリプション管理サービスです。 このサービスを使用すると、ブログのオーディエンスに関する統計を確認し、利益を上げることもできます（RunetでFeedburnerの広告を使用した事例はまだわかっていません-誰かが指摘してくれれば、感謝します）。

いくつかの公式統計：

  • Nielsen / NetRatingsによると、FeedBurnerへのトラフィックはMySpaceやDiggよりも集中的に増加しており、トラフィックは385％増加しています。
  • フィードの総数：883,827
  • Webサイト運営者の数：517,239
  • ポッドキャストおよびビデオキャストの数：128,358


Feedburnerを使用してフィードを起動する方法に関する短いビデオを見ることができます（ただし、ビデオがなくても簡単でわかりやすい）：フィードの書き込み方法を表示する （3：14、4.6 MB、音声なし）。

サブスクライバーはどのように考慮されますか？



Feedburnerは、24時間以内にフィードを開く購読者の概数を決定します。 毎日フィードを要求するさまざまなタイプのフィードリーダーとアグリゲーターを検討してください。 さまざまなロボットは考慮されません。

RSSと電子メールの両方のサブスクライバーが考慮されます。

konkoffによる更新（Darren Roseの投稿の翻訳）：



Rick Klau（FeedBurnerの副社長の1人）とのインタビュー記事が、Darrenのブログproblogger.netで公開されました。質問は、「サブスクライバーの数が変動するのはなぜか、さらには曜日に依存するのか？」

クラウはこの質問に次のように答えました。

フィードのサブスクライバーの数を合計するとき、過去1日間のリーダーのフィードリーダーからのリクエストの数を考慮します。

一般的に、サブスクライバーのオーディエンスは2つのグループに分けることができます。

  • RSSフィードリーダープログラムのユーザー
  • オンラインリーダーを使用するリーダー。


最初のケースでは、私たち（FeedBurner）は、原則として変更される可能性のあるIPアドレスに従って、テープへのアクセス頻度に対するリーダー要求を分析します。

オンラインリーダーの場合、My Yahoo、Google Reader、Bloglines、Pageflakesなど。 状況は少し異なります。 彼らは一日中、常に何度も、読者のフィードを削除します。 この場合、サービス自体が助けになり、登録されたサブスクライバーに関する情報をRSSフィードに提供します。

一日の終わりに、データが要約され、一日あたりの加入者/読者の数が統計に表示されます。

「オン/オフ」モードでコンピューターを使用するユーザーが原因で、ほとんど常に変動が発生します。 コンピューターの電源がオフになっている場合、テープに対する要求はなかったため、「リーダー」はカウントされません。


サブスクライバーの数を誰かに見られたくない場合は、これをオフにすることができます。

公開-> FeedCount->無効化




Feedburnerの最適化



すべてのRSSサブスクライバーがFeedburnerを使用するには、次のプラグインをインストールする必要があります： FeedBurner Plugin v2.2。

samlowryからの補足 ：

これで、プラグはFBの翼の下で正式に取得されました 。現在、 ここで見つけることができます 。 WP + FBの推奨事項も同じマニュアルに記載されています 。ブラウザーフレンドリーバーナーオプションを有効にします（[最適化]タブにあります）。 SmartFeedも含めました。これにより、多数のフィードリーダーとの互換性が向上します。


Drupalにはこのようなプラグインがあります： http : //ftp.drupal.org/files/projects/feedburner-5.x-0.5.tar.gz （リンクuniqmanに感謝します ）。

サブスクリプションボタンが非常によく見えるはずです。 さらに、「電子メールサブスクリプション」フォームを追加する価値があります。

検索大手のGoogleがサービスを購入すると、すぐに2つの機能が無料になりました。

  • FeedBurner Stats PRO 。 フィードの詳細な統計：誰がどの投稿を表示し、どこをクリックしたか。
  • マイブランド 次のようなフィードアドレスを作成できます：標準のfeeds.feedburner.com/MyFeedNameの代わりにfeeds.myexcellentdomain.net/MyFeedName


私はすでに最初のものを使用しており、2番目のものを固定する予定です。

統計を見る方法



ブログで統計を確認し、管理パネルFeedburnerで詳細を確認してください。



ただし、FeedCountが無効になっていない場合は、他の人の統計を見ることができます。 このテクニックを使用して、 評価のためにいくつかのカウンターからアニメーションを削除しました。

すべてのカウンター画像には次のようなアドレスがあります。

feeds.feedburner.com/~fc/blogdimokru?bg=99CCFF&fg=444444&anim=0


値は次のとおりです。

  • blogdimokruは私のフィードです。他の人は自分の名前を入力します（フィードのURL： http : //feeds.feedburner.com/ blogdimokruから取得 ）
  • bg-背景画像
  • fg-フォントの色
  • anim-アニメーション（0-アニメーションなし、1-アニメーションあり）
