5. Cryptography



Unfortunately, WebBrowserPassView is detected by antiviruses as a potential threat or trojan. The compressed UPX program on VirusTotal scored 41 detections. However, the result of VirusTotal dropped to 6. Hmm, but if you take the book and replace each letter in it with the next one in alphabetical order? Caesar's code. Primitively, only 4 antiviruses have already suspected something. And if you combine these two methods? And everything is very simple, 0.



Yes, you just take the program, write it byte back to front and add a unit to each byte and it does not detect any antivirus
