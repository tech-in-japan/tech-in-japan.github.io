学生向けJavaプログラミングオリンピック



オラクルは、It-planetとともに、大学生や大学を招き、国際的なJavaプログラミングのオリンピック-Oracle Java Olympicに参加します。 コンテストはウクライナ、ロシア、ベラルーシ、カザフスタンで開催され、3つのステージで構成されています。 参加するには、大学に大学に登録することが望ましいですが、個人的な参加も可能です。 オリンピアードへの参加登録は、2011年11月30日までhttp://world-it-planet.org/で行われます 。 最初のページで登録するには、「Oracle Java Olympic」と居住地域を選択します。



最初の段階は予選です。 リモートで実施。 登録後、参加者はテストを受けるためのリンクを受け取り、40分以内にJavaテクノロジーに関する40の質問に答えます。 テストは12月1日から12月15日までいつでも受けることができます。 各参加者がテストに合格するのは1回だけです。



第二段階

大学での予選段階の受賞者は、フルタイムの地域ツアーに参加するよう招待されます。参加者は、7つのタスク、4時間、オペレーティングシステム、JDK7、NetBeans 7.0.1およびJava APIのドキュメントを備えたコンピューターを受け取ります。 競技会への参加は個人的なものです。



第三段階は最終です。 第二段階と同様の形式で開催されます。 主な違いは、より複雑なタスク（そしてもちろんライバル）です。



競技の詳細と詳細な説明については、オリンピアードのウェブサイトをご覧ください。

http://world-it-planet.org/
