ショーン・ピアス：真のリーダー

記事「 Sean Pierce：A Real Leader 」の翻訳は、jGitやGerrit Code Reviewを含む多くのプロジェクトの創設者である人物に関するものです。









Sean Pierceは、オープンソースプロジェクトJGitおよびGerrit Code Reveiwの創設者でした。 私たちが大西洋の異なる側から来て、異なる会社で働いていたという事実にもかかわらず、同じプロジェクトで彼と一緒に仕事をすることは私にとって大きな名誉でした。



人々は真のリーダーについて多くのことを話しますが、ショーンのような人と8年以上知り合うことはまったく異なります。 私たちはお互いに会う前に、2008年に同じアイデアで始めました。 SeanはAndroid OpenSourceプロジェクトでの実装のためにGitをスケーラブルにしようとし、 Rietvieldプロジェクトを純粋なJavaで書き直し始めましたが、私の目標はGitEnterprise.comサービスを使用して世界中の大企業でGitの使用を拡大することでした。



すべてを受け入れるアイデア







2011年にGoogleplexのマウンテンビューCAにあるGitTogetherで初めて顔を合わせました。 ショーンはこの有名な「非会議」の創始者でした 。 この場所は、新しい革命的な概念でお互いの経験を交換し、豊かにするために一緒に時間を費やしたアイデアと心の創造のインキュベーターでした。



私は初心者で、プロジェクトの誰も知りませんでした。 私の考えは、人々がホールで話し合っていたものとは非常に異なっていました 。 それから立ち上がって、Jenkins CIの経験と成功に触発されて、プロジェクトのアーキテクチャを変更し、完全にプラグインにすることを提案しました。



ショーンは 、 私が誰なのか 、どこから来たのか、どのような経験をし、何に取り組んでいるのかわからなかった。彼は単に答えた。 参加して助けてくれませんか？」私はそうしました、私たちは1週間一緒に働いたので、ショーンとの私の旅が始まりました。



心に強く訴える人々







私がショーンについて気に入ったのは、彼のユニークなスタイルのプロジェクト管理でした。 彼は会議で特定の悲しみを発表したり発表したりしなかったが、誰もが彼が誰であるかを知っていて、彼が部屋に入ったとき、彼はみんなの注目を集めた。



彼はプロジェクトで直面した問題について情熱的に語り、プロジェクトが達成できる限界を常に拡大しました。 この情熱は伝染しました。 彼は人々に何をすべきかを決して言わず、間違いを犯すことができるように全員を残しました 。



ショーンは種であり、そのおかげでプロジェクトは元のサイズと能力をはるかに超えて成長しました 。 彼は、ほんの1年前にGoogleでインターンとして働き始め、 NoteDBを実装するためにバックエンド全体を書き換えることができたDave Borovitzなどの有望な若者を連れて行くことができました。



人類第一







ショーンは明るく華麗でしたが、私に最もインスピレーションを与えたのは、 彼の子供たちと愛する家族への情熱でした 。 彼は週末に息子をハッカソンに連れて行き、父親と一緒にいることに慣れています。 彼は家族生活と日々の仕事と情熱を結び付けました。 私が覚えている最高の瞬間は、サンノゼの美しい丘の向かいにある彼の新しい家で日没時に「ハッカーバーベキュー」を食べたときです 。ノアは庭で遊んで、愛する父親を抱きしめました。



彼は常に、家族のニーズとGerritプロジェクトの存在と責任の統一に目を向けました。それは、人々が最も重要な部分であるため、誰もが覚えていました。



献身







ショーンの喪失は多くの人々にとって予期せぬものでした。彼は最後の数日までコードの公開と手紙への返信を続けました。 彼は2017年11月にGerritで新しいOpenSSHサポートを実装し、クリスマスまでコミュニティとメンバーと連絡を取りました。 彼の最後のコミットは2018年1月3日に行われ、彼は人生の最後まで常にコミットしました。

彼は自分の病気とそれがどれほど深刻なのかを公に明らかにしなかった。 私は彼の人生を、彼の家族、プロジェクト、そして彼が最も愛したすべてで、最後まで彼の最高のアイデアと情熱で生きました。



ショーンの遺産



ショーンのような偉大な指導者が去るとき、背後に空白を残す危険が常にあります。 彼は彼のスタイルと情熱において本当にユニークなリーダーだったので、ショーンを置き換えることは不可能だと思います。



しかし、彼が創り出したのは、彼の専門的かつ個人的なインスピレーションに満ちた人生で私たちに教えてきた価値を維持し続けるアクティブで健康的なコミュニティです。



ショーンが作成したもの、彼は彼の人生と私たちの存在を生き残ることができる「健康的な生活プロジェクト」と定義しました。 プロジェクトの10年以下の数字。







感謝の気持ち



SeanがGit、JGit、およびGerrit Code Reviewの将来のメンバーおよび受信者にしたことに対して、Seanに感謝しています。



私はそのような刺激的なリーダーを知っており、一緒に働いたことを幸運に思いました。 ありがとう、ショーン。
