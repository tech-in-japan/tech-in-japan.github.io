新しいHotmailの概要

最近では、Hotmailは3億6,000万人以上のユーザーに新しいインターフェイスと拡張機能を提供しています。 開発チームは素晴らしく働き、サービスの多くの側面を修正しました。 多くの写真とカットの下の最小限のマーケティング。



インターフェース



まず、目を引くのはインターフェイスです。 基本的に変更することなく、彼はより楽しく直感的になりました。







メールの仕分け



大きな変更は、メールのソート機能に影響を与えました。 メールを使用する人は、機会が最も基本的なレベルになる前に知っていました。 これで、日付、送信者、件名、サイズだけでなく文字を並べ替えることができます。 ただし、些細な記号でフィルターすることもできます。







写真の手紙、オフィス文書の存在によって並べ替えます：







ルールとフォルダーを作成します。











また、ドラッグアンドドロップ機能は、Inboxフォルダーから文字を転送するために機能することを思い出します。



手紙の主題に関する手紙のグループ化もありました。 この機能はデフォルトで無効になっていますが、メインインターフェイスから簡単に有効にできます。







チャット



Hotmailの新しいバージョンでは、インスタントメッセージングサービス（Windows Live Messengerを介して実装）の統合がより便利になりました。







ソーシャルネットワーク



ソーシャルネットワークと統合するためのファッションは、新しいHotmailをspareしみませんでした-Facebook、LinkedIn、MySpaceとの連携がより便利になりました。 これで、レターから直接ワンクリックで友達を追加できます。







写真とビデオを見る



私の意見では、郵便サービスの主な革新はアクティブな表示です-手紙の本文では、受け取った写真をスライドショーで見ることができます。







FlickrまたはYouTubeへのリンクが送信された場合、リンクの代わりに、手紙から直接フルスクリーン形式で写真やビデオを表示できます。







オフィス



10月にロシアは、Office Web Apps（ブラウザーを介してOfficeドキュメントを操作するためのアプリケーション）をHotmailと統合します。これにより、Hotmailに届いたドキュメントをブラウザーでOffice Web Appsですぐに開き、編集してHotmail経由でリンクとして送信できるようになります。 Office Web Appsは既にSkyDriveと統合されています。 ドキュメントの作成、編集、共同作業、SkyDriveオンラインストレージへの保存が可能になりました。 これらのすべてのアクションは、コンピューターにインストールされたOfficeパッケージがなくても実行できるようになりました。







引き出しボリューム



それでは、箱のサイズと送られた手紙のサイズについて話しましょう。

Hotmailでは、メールボックスのサイズはいっぱいになると自動的に増加します。 10月（ロシア）から送信されるメッセージの最大サイズは10 GBです（各添付ファイルのサイズは最大50 MB）。 これはSkyDriveオンラインストレージを介して行われます。オンラインストレージでは、事前に何かをアップロードする必要はありません。これは、ファイルをレターにダウンロードするときに自動的に行われます。 受信者は、手紙で直接表示できるファイルを表示したり、ダウンロードしたりするためのリンクを受け取ります。 これにより、受信者のボックスがオーバーフローしないようになります。
