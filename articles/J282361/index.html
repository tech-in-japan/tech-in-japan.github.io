Windows Server 2016のシールドVMウェビナーに参加してください。4月27日11:00（GMT）から

4月27日11:00（モスクワ時間）ウェビナー「 Windows Server 2016のシールドされたVM 」を実施し、全員に招待します。



シールドされたVM （シールドされた仮想マシン）は、 Windows Server 2016の根本的に新しいテクノロジーであり、仮想マシンの所有者は、たとえばBitLockerを使用して、ゲストOS内のボリューム暗号化を有効にできます。 したがって、VMが配置されているHyper-Vホスト上の悪意のあるコードと、このホストの管理者の両方にとって、VMのコンテンツにはアクセスできません。 さらに、暗号化キーがシールドされたVMを備えたHyper-Vホストに保存されることはありません。これにより、ソリューション全体のセキュリティが大幅に向上します。



ウェビナーの一環として、シールドVMテクノロジーの概念と基本コンポーネント、ハードウェア要件、構成機能、およびこの新しいソリューションのアプリケーションシナリオに精通します。



対処すべき問題：

  • ホストガーディアンサービスアーキテクチャ（HGS）
  • Hyper-Vホスト認証モード
  • シールドVM
  • System Center 2016 Virtual Machine Managerを使用して、シールドされたVMを展開および管理する
