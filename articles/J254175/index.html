Chrome：人気のある開発者拡張機能



デバッグと開発プロセスをさらに楽しむための、Chrome開発者ツールの推奨拡張機能。







角バタラン



この拡張機能は、アプリケーションのパフォーマンス、視覚化、およびデバッグの問題を検出するためのツールを追加します。



Devtoolsターミナル



DevTools Terminalは、ブラウザーに端末機能を追加するChrome開発者ツールの拡張機能です。



エンバー検査官



Ember Inspectorは、Chrome開発者ツールの拡張機能であり、Ember.js上のアプリケーションの理解とデバッグを容易にします。



バックボーンデバッガー



Chrome開発者ツールにタブを追加し、リアルタイムのビュー、モデル、コレクション、ルートを表示します。



マリオネット検査官



Marionette Inspectorは、コンポーネントを視覚化し、データを見つけ、アクティビティをリアルタイムで調べるためのツールを提供します。



React Developer Tools



React Developer Toolsは、React.jsオープンJavaScriptライブラリを操作するためのChrome開発者ツールの拡張機能です



胸部検査官



大規模なWebアプリケーションを作成するために、バックボーンとハンドルバーに基づいたライブラリであるThoraxのインスペクタサポートを追加します。



Grunt Devtools



Grunt！用のツールを追加して、開発者ツールを拡張します！ Gruntの完全なGUI。 追跡などのバックグラウンドタスクのサポート。 1つのウィンドウで複数のプロジェクトのタスクを実行できます。



DevToolsリダイレクト



ドメインをインターセプトし、ローカルまたはリモートサーバー上のリソースにリダイレクトします。



Knockoutjsコンテキストデバッガー



Knockoutjsをデバッグするための拡張機能。 開発者ツールの[要素]タブに、選択したアイテムの対応するノックアウトの説明とともにサイドバーを追加します。



BatmanJS DevTools



Batman.jsを使用する場合、この拡張機能は非常に貴重なツールになります。 ブラウザに直接Batman.jsを開発するのに役立つ多くのツールが含まれています。



エメットライブスタイル



他の同様のツールとは異なり、LiveStyleはブラウザー（またはファイルシステム）のCSSファイルを変更して変更を表示せず、設定中に構造分析を実行し、CSSソース間の変更をマップします。



きびきびしたスニペット



SnappySnippet-選択したDOMブランチ（最後に検査された要素）のCSSとHTMLを取得できる開発者ツールの拡張機能。 ワンクリックで、選択したコードをCodePen、jsFiddle、またはJS Binに転送できます。



RapidJS DevTools



rAppid：jsアプリケーションを学習できるChrome開発者ツールの追加パネル。



レールパネル



RailsPanelは、ログファイルを処理するRailsアプリケーションを開発するためのChrome拡張機能です。 コンソールでRailsアプリケーションのすべてのリクエスト情報に直接アクセスします。



Senchaのアプリインスペクター



SenchaアプリケーションをデバッグするためのGoogle Chrome開発者ツールの拡張。



jQueryデバッガー



jQueryセレクターを確認し、任意のサイトで結果を表示するためのセレクターインスペクターの新しいタブ。 このアドオンは、最も複雑なセレクターでも理解できるように設計されています。



PageSpeed Insights



ページのパフォーマンスを評価し、それらを改善するための推奨事項を取得できる、ChromeおよびFirefoxで利用可能なブラウザー拡張機能。



Djangoデバッグパネル



WebKit開発者ツール用のDjangoデバッグパネル。 バックグラウンドのAJAXリクエストと非HTMLリクエストで良好な動作を示しました。 単一ページのアプリケーションや、AJAX Webアプリケーションを使用するその他のアプリケーションに最適です。
