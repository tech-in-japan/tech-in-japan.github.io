3CX 15.5 SP1ベータリリースおよびAskoziaの買収

今週は、2つの重要なイベントがありましたので、お急ぎください。 だから、順番に。







Beta 3CX v15.5 Update 1（SP1）リリース



夏休みの真っIn中に、3CX v15.5 Update 1のベータ版をリリースし（現在サービスパッケージと呼んでいます）、ユーザーと管理者に興味深い機能をいくつか追加しました。







管理者向けの機能



IP電話の追加パラメーターの集中構成



サポートされているIP電話では、 構成可能なパラメーターの数が大幅に拡張されています。 管理者は、ドキュメントを理解して、この電話またはその電話用に独自の自動設定テンプレートを作成する必要がなくなりました。 追加のパラメーターはすべて、3CX制御インターフェースから集中的に指定されます。







パラメータリスト：







  • 着信音の選択
  • 電話のLED電源インジケータの操作のMWIモードの設定-音声メッセージ、不在着信などがある場合に点滅
  • Yealink、Fanvil、Htekのスクリーンセーバーの3CXロゴの明るさが低下しました
  • 各電話の個別のタイムゾーン設定
  • 時間表示形式の設定：AM / PM、12時間または24時間
  • デフォルトのコール転送タイプの設定（無伴奏または従順）
  • 電話ディスプレイのバックライトを有効または無効にする
  • ディスプレイのスクリーンセーバーをオンにするためのタイムアウトを設定します。これにより、一部のモデルでロゴの点滅がなくなります。
  • VLAN IP Phoneネットワークパラメータの設定
  • BLF（DSS）ボタンを回線モードまたはBLFモードに設定します。 以前は、ほとんどの電話の最初の2つのボタンは回線モードでのみ機能していました。


簡素化されたセットアップとメンテナンス



3CXのインストール、構成、保守を非常に簡単にし続けています：



  • すべてのSIPオペレーターテンプレートが更新され、新しいトランクをより簡単かつシームレスにシステムに接続できるようになりました。
  • これで、3CX管理インターフェイスからサーバー上のネットワークトラフィックを直接キャプチャできます 。 これを行うには、 Wiresharkユーティリティ （Windowsシステム）を3CXサーバーにインストールする必要があります。 さらに学習するためにキャプチャファイルをダウンロードするか、3CXテクニカルサポートに送信するためにデータパッケージに含めることができます （ テクニカルサポート用のファイルを作成するオプション）
  • 組み込みの3CX Firewall Checkerユーティリティは、SIL ALGサービスがルーターで使用されているかどうかを判断できるようになりました。 このサービスを無効にする必要があります。 ほとんどの場合、音の通過に問題を引き起こすのは彼です
  • コマンドラインからバックアップと復元を開始します


ユーザー機能



新しい3CX Webクライアントには、多くの便利な改善点があります。



  • これで、ユーザーは従業員 （以前はPeopleと呼ばれていました）のメイン画面から直接呼び出しを動的に監視および管理できます。
  • ユーザーは、Webクライアントで直接コール転送ルール（およびルールの例外）を構成できます。
  • Webクライアントインターフェイスが開いているかどうかに関係なく、着信はChromeデスクトップ通知として表示されます
  • ユーザーは自分の親しい同僚をお気に入りに追加することができ（アスタリスクでマークするだけ）、すべての3CXクライアントのお気に入りに表示されます。
  • 以前はホテルのみを対象としていた機能がすべてのユーザーに利用可能になりました。



    • リマインダー呼び出しを設定できます
    • 新入社員または臨時従業員の内線番号を初期化する（個人データを消去する）
    • 他の従業員のステータスを変更する








3CX Webクライアントの新機能の概要を英語でご覧ください 。

その他の機能



フォーラムには、さらに長い間リクエストされていた機能も追加しました。



  • ボイスメールボックス
  • 時間分布レポート
  • コールキューでの平均顧客待機時間と、オペレーターが応答するのを待たずに顧客が「電話を切る」までの平均時間に関する重要なレポート
  • ホテルメイドコード
  • また、内線番号をコピーする便利な機能を追加します。 既存の拡張機能に基づいて新しい拡張機能（ユーザー）を作成します。 ただし、Update 1ではまだ実装されていません。

3CXデスクトップクライアントもアップグレードする必要があります。



  • Windows用3CXクライアント
  • Mac用3CXクライアント


3CXは有名なAskoziaを買収してグローバルに拡大







2番目の興味深いニュースは、有名なドイツの企業Askoziaの3CXによる買収です。Askoziaは、定評のあるアスタリスクアセンブリAskoziaPBXを製造しています。 ブランドの買収に加えて、3CXは世界中でAskoziaPBXを使用する10,000を超える小規模クライアント企業、経験豊富な開発者のチーム、および独自のノウハウを獲得しました。



Askoziaは2013年に設立され、小規模な学生プロジェクトからソフトウェア交換市場の真剣なプレーヤーに成長しました。 3CXのリーダーシップは、この非常にプロフェッショナルなチームの成功に感銘を受けました。 現在、ドイツ市場での3CXの地位を強化するだけでなく、テレフォニーの分野でAskozia独自のLinux開発を使用する機会もあります。



現在のすべてのAskoziaユーザーおよびパートナーは、合併中も引き続き必要なサポートを受けられます。 Askoziaユーザーには3CX無料ライセンスが提供され、Askoziaパートナーは3CXパートナーになります。



Askoziaを購入するもう1つの利点は、3CXがCebit展示会を開催し、多くの有名なテクノロジー企業のオフィスを持つハノーバーに常駐していることです。 もちろん、これは潜在的な顧客やパートナーとのコミュニケーションを簡素化します。
