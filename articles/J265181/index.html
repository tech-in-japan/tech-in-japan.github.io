9月2日をジャンプスタートオンラインイベントに接続します：「企業ネットワーク上のWindows 10」

2015年9月2日、ジャンプスタートシリーズ「 企業ネットワーク上のWindows 10 」のオンラインイベントを開催します。 モスクワ時間の12:00から接続します。

また、ADK、MDT 2013 Update 1、RSAT for Windows 10がダウンロードできるようになりました。







イベントの主なトピック：OSを企業環境に展開し、Windows 10デバイスとセキュリティテクノロジーを管理する方法とツール。 何らかの形で、これらのトピックは、Microsoft Virtual Academyポータルで公開されている「 Windows 10：展開、管理、セキュリティ 」コースですでに取り上げられています。 ただし、コースの公開以降、「10」の最終バージョンがすでにリリースされ、展開ツールが更新され、スマートフォン用の新しいアセンブリが登場しています。



上記に基づいて、レポートとデモンストレーションが準備されており、イベントのプログラムは次のとおりです。



12:00-13:00 Windows 10への移行、Alexander Shapoval、Vladimir Zelenov

13:00-13:15休憩

13:15-14:15 Windows 10、Rostislav Nedosekinでのランタイムプロビジョニングとは

14:15-15:00休憩

15:00-16:00 Windows 10でのデバイスの管理、マキシムシネルニコフ

16:00-16:15休憩

16:15-17:15 Windowsの生体認証Hello、Andrey Beshkov、Alexander Shapoval

  • Windows 10 RTMをサポートする評価および展開キット（ADK）の最終バージョン
