DIY CISCO、またはLanner FW-7540ネットワークプラットフォームのレビュー

はい、はい、CISCO / Juniperルーターを自分の手で作ることができます。

私の投稿では、ネットワークプラットフォームなどの興味深いことについてお話します。







そのため、まず第一に、この用語の簡単な定義：

Unified Network Platform（Unified Network PlatformまたはUNPから）は、次のようなネットワークサービスの収束を提供するオープンアーキテクチャプラットフォームです。

  • データネットワーク管理
  • ネットワークセキュリティ
  • ネットワークインフラ
  • ネットワークアプリケーション


UNPは、汎用のIntel / AMD機器（VIA、MIPSなど）を備えたオープンソースオペレーティングシステム（pfSense、Zentyal、ClearOS、Vyatta、Zeroshellなど）を使用して、システムを形成する一連のネットワーク機能を提供します。 このバンドルでは、UNPはクラス最高の柔軟性、透明性、スケーラビリティ、および拡張性を提供します。



簡単に言えば、ネットワークプラットフォームは、通信ネットワークで動作するように設計されたハードウェアであり、さまざまなネットワークタスク（ルーティング、NAT、VPN、プロキシ、ファイアウォールなど）に合わせて調整されたソフトウェアがインストールされます。



これは会社のシステム管理者/情報セキュリティの専門家にとって何を意味しますか？ はい、それは非常に簡単です。今では、企業のVPNゲートウェイを整理するための強力なルーターを構築する必要がある場合、またはCISCO / Juniperの機能を取得するために、たったのお金で、これは次のようなものでは不可能です：

（ my-cool-projects.blogspot.ru/2013/03/replacing-my-ancient-linux-router-with.htmlからの写真）



そして、非常にコンパクトで生産性の高い（時には受動冷却でも）鉄の助けを借りて。



今日は、Lanner IncのネットワークプラットフォームFW-7540の中流クラスの生産についてお話します。







このプラットフォームには、最大4GB DDR3 RAMをサポートするIntel Atom D525 Pineview 45nmはんだ付けデュアルコアCPUが搭載されています。 デバイスを制御するために、x86互換のオペレーティングシステム（Linux、Windows、pfSense、その他多数）を使用できます。



また、デバイスの低コストにもかかわらず、Intel 82583VまたはIntel 82574Lコントローラを実行する4つの完全なGbEインターフェイスがあります。





プラットフォームのより詳細な概要に移りましょう。

特徴：

  • ハーフラックデバイスのフォームファクターは215.5 x 44 x 190 mmです。
  • Intel ICH8Mチップセット、最大4GBのDDR3 800Mhz RAMをサポート
  • 1x CompactFlash Type IIソケット。 2x SATA2
  • パッシブCPU冷却（ヒートシンク）、システム冷却用の1xクーラー。
  • 1x Mini-PCIe + SIMリーダー。
  • 2x USB 2.0
  • 60W電源アダプター。
  • 4x GbEインターフェイス。 4 x Intel 82583V / 82574Lコントローラー。
  • G2バイパスをサポートする2組のポート*
  • コンソールRJ45ポートx 1。
  • 重量：1.2 kg。
  • BIOS：AMI BIOS、16MビットSPIフラッシュROM


*バイパス-すべての機能ブロックをバイパスして、入力信号を出力に直接切り替えることができる機能。 （デバイスが「フリーズ」した場合、これらのインターフェイスのトラフィックスイッチングは中断することなく継続されます）。





デバイスのケースは、高品質で、かなり厚い壁で作られており、エッジは指を切りません。 ケースの側面には、19インチラックの留め具用の穴があります。



標準パッケージには次のものが含まれます。

  1. プラットフォームFW-7540B（変更BはBypass *機能を示します）
  2. 60W電源アダプター、電源ケーブル。
  3. 工場パッチコード2m。
  4. コンソールワイヤDB9-RJ45。
  5. 前面のステッカー。
  6. ドライバーを備えたディスク。
  7. ゴム足。






ボード上の拡張インターフェースから、組み込み3Gモデムを接続するためのSIMリーダーを備えたMini-PCIeがあります。 デバイスを制御するには、RJ45コンソールポートを使用しますが、ボードには、ケーブルをVGAコネクタ（DB15）に接続するためのピンもあり、デバイスの壁のプラグの代わりに取り付けることができます。

CPUを冷却するために、ラジエーターが使用されます。これは、TDP 13Wで「赤ちゃん」に十分な量です。 そして、唯一のクーラーは加熱された空気を排出することでシステムを冷却します。



ネットワークプラットフォームを管理するには、通常、無料のオープンソースソフトウェアを使用します（多くの場合、コンソールからだけでなく、Webインターフェイスからも構成されます）。

pfSenseは、FreeBSDに基づく強力な多機能ルーターです。

Vyatta -Debianカーネルに基づくJunOSインターフェースを備えたルーター、

アスタリスク -本格的なIPを作成できます-PBX、pfSenseの一部、

ClearOS 、 FreeNAS 、 Zeroshellなど



このデバイスは、中小企業に最適です。 これにより、企業ネットワークの高性能な配信ノードを構築できます。 たとえば、Vyatta OSプラットフォームにインストールすると、CISCO / Juniperクラスの機器が数倍安くなります。
