トレーニングデータアーキテクト：ER / Studioを使用した情報システムの設計とモデリング

Turbo Pascal、Delphi、RAD Studioの例を使用して、第一世代の学童や学生ではなく、コンピューターソフトウェア開発技術の基本と微妙さを習得しました。 しかし、IT専攻を教えるために設計された「アカデミック」ライセンスのリストは、プログラミング言語と統合アプリケーション開発環境に限定されません。 今日は、Embarcadero ER / Studioとその教育プロセスへの応用について、もう1つの製品について説明します。



データベース管理の専門家は、企業のデータ環境を文書化、解釈、および完全に制御するプロセスがどれほど複雑であるかを知っています。



これらの問題を解決するために、多くの企業がEmbarcadero ER / Studioを使用しています。これは、さまざまな業界標準をサポートするビジュアルデータベースモデリング、ソフトウェアエンジニアリング、ビジネスプロセスモデリングのための包括的なソリューションです。 これは、データモデルに基づいて情報リソースを発見、文書化、効率的に管理し、変化する要件を満たすためにデータアーキテクチャを再構築するために、世界の多くの大企業が使用する業界最先端のデータモデリングおよび情報アーキテクチャ設計ツールです。



データを扱う分野の専門家にとって、ER / Studioは、効果的でシンプルで便利なツールのセットであり、データ管理スペシャリストのコラボレーション用ツールです。これにより、大規模な企業データベースとデータウェアハウスを作成および管理できます。 システムで提供される機能により、モデリングの日常的なタスクが自動化されるため、大規模なデータベースおよびデータウェアハウスの構造を視覚化、つまり迅速に理解、分析、および最適化できます。 ツールに含まれるレポートおよび対話ツールは、組織標準の適用と高レベルの生産性の達成に貢献します。

ER / Studio Enterpriseは包括的な製品であり、次のツールが含まれています。

  • ER / Studio Data Architect、
  • ER / Studio Business Architect、
  • ER / Studio Software Architect、
  • ER / Studioリポジトリ、CONNECT。






モデル管理サーバーシステムとインターネットポータルにより、このソリューションは、分散チームでモデルとメタデータを共有、文書化、公開するための効果的なツールになります。



ER / Studio Data Architectは、ER / Studioの最も重要な部分であり、データアーキテクチャの独立した論理分析と設計、および特定のDBMSのデータベースインスタンスの物理的な設計と構築のために設計されたビジュアルモデリングアプリケーションです。



ER / Studio Data Architectは、データベースおよび多次元情報ストレージに関連するトピックを教えるための実用的なツールとして使用することをお勧めします







ER / Studio Data Architectと連携する例を使用して、学生は実際に学ぶことができます。

  • データベース設計の原則と方法、
  • ERの使用-リレーショナルデータベースを含むさまざまな種類のデータベースの構造を設計および視覚化するためのさまざまな表記法の図
  • 既存のデータベースのリバースエンジニアリング、分析、最適化を実行します。
  • 論理レベルでのデータベースモデリング。 概念モデル、論理モデル、物理モデルのオブジェクトをマッピングし、それらの中で対応を見つけて表示する
  • モデルの分解と統合による大規模モデルでの作業の簡素化
  • OLAPおよびBIシステム用の多次元データウェアハウスのモデリング
  • データベースアクセス戦略の設計
  • データベース内と外部システムの要素の両方の制約と依存関係のモデリング
  • 情報交換サブシステムとETLプロセスの設計と分析
  • データベース内の情報を保護するためのシステムの設計、データベースへのアクセスのためのユーザー権限の管理
  • さまざまなデータベースモデルの有効性の分析と評価
  • データベースを設計するときにスケーラビリティを提供するための最新の方法の使用
  • データベースのパフォーマンス要件を考慮した、さまざまなDBMSプラットフォームに基づいたモデルの物理的な実装の構築
  • SOAなどのさまざまなプロジェクトでモデルを使用するための論理モデルおよび物理モデルからのXMLスキーマの作成






実践ショーと大手分析企業は、今後数年間でこの傾向が強まると予測しており、大多数の企業は仕事で2つ以上の異なるDBMSプラットフォームを使用しています。 そして、それが何を引き起こしたかは関係ありません-長寿命のレガシーデータベースや企業データを組織のリソースおよび資産として検索および使用する企業のイニシアチブの存在、ITの世界は、データベースの専門家やアーキテクトがマルチプラットフォーム環境で働くためによく訓練されることに非常に興味を持っています ER / Studioを使用すると、このレベルのトレーニングスペシャリストの機会が何度も増えます。



最も「機能が豊富な」エディションはER / Studio Enterpriseです。 このエディションは、同じDBMSのすべてのバージョンを含む、使用可能なほぼすべてのタイプのDBMS（必ずしもリレーショナルのものを含む）の同時サポートによって区別されます。 ER / Studio Enterpriseは、主要な製造会社の他のモデリングツールとのエクスポート/インポートおよびメタデータ交換ツールも追加しました。

ER / Studio Enterpriseは、大規模な企業データベースとデータウェアハウスの管理のスペシャリストのトレーニングに推奨されます。







ER / Studio Data Architectセクションにリストされているトピックに加えて、このツールを使用すると以下を教えることができます。

  • ユニバーサルマッピング、トレースデータ変換、企業データディクショナリ、比較およびマージテクノロジーに基づいた変更の影響のより効果的な診断と分析。
  • 効率的なエンタープライズレベルのモデル管理とメタデータ公開機能により、データの再利用とリアルタイムのコラボレーションを設計します。
  • レポートツールとメタデータ公開を使用して、企業全体でデータモデルに関する情報を効果的に交換します。
  • データとメタデータを共有し、分散ワークグループ用に文書化して公開する最新かつ最も生産的な方法、アクティビティやディスカッションスレッド、アラート、サブスクリプションおよび評価機能などの高度な企業コラボレーションの機会。
  • データディクショナリに基づくアイテムの標準化と再利用
  • 必要な標準に厳密に準拠した集合モデリング環境での相互依存するビジネス要件、プロジェクト、およびデータモデルのコラボレーション方法。
  • 埋め込まれたビジネス定義を、一元的に管理される用語集（利用可能なすべてのデータソースの単一の企業レジストリ）と統合し、意味的にリンクします。


アーキテクトおよびソフトウェア開発者向けに、ER / Studio Enterpriseには、複雑なソフトウェアアプリケーションを視覚的に設計および分析できるアプリケーションアーキテクト向けのオブジェクト指向モデリングツールであるER / Studio Software Architectが含まれています。







このツールを使用すると、生徒は実際に次のことを習得できます。

  • モデルに基づいてソフトウェアアーキテクチャと標準を開発します。
  • 業界標準テンプレート。
  • 指標と監査モデル。
  • UML 2.0を使用して、分散オブジェクトシステムの成果物を視覚化し、設計し、文書化します。






ER / Studio Enterpriseには、ビジネスプロセスをモデル化し、データを使用するために設計された概念モデリングツールであるER / Studio Business Architectも含まれています。 グラフィカルエディタツールの利便性とBPMN標準の豊富なセマンティクスにより、ER / Studio Business Architectのビジネスユーザーは、プロセス、人、データ間の相互作用をモデル化できます。







このツールを使用すると、生徒は以下を学習できます。

  • シーケンス、タスク、トラックなどの標準要素を使用して、BPMN標準に基づいてビジネスプロセスをモデリングします。
  • トップダウンデータベース設計の概念モデリング。 サブジェクト領域、事業単位、相互作用、およびその他の要素を使用して、高レベルの概念モデリングを実行します。
  • ビジネスプロセスとデータオブジェクトをモデリングするための使用状況レポート（CRUD）。
  • ビジネス成果物のモデリング。




ER / Studioは、データを記述および設計するための基本技術に基づいており、学生にデータを知ってもらうだけでなく、モバイルアプリケーションからグローバルな企業管理システムに至るまで、コンピューターデータ処理の分野における関連する見通しと現代の技術の全範囲を示すことができますER / Studioを使用すると、Teradata、Netezza、Greenplumなどのエンタープライズクラスの戦略的システムのサポートや、Hadoopのビッグデータプラットフォームを探索することができ、学生が学ぶための実用的な基盤を提供します テーマ「ビッグデータ」。
