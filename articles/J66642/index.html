Drupal 6でのクイックギャラリー作成（一括アップロード）

Drupal.ruで、彼らは再び「小さな血」のギャラリーを作成するという問題を提起しました。 それはオリジナルのふりをしていませんが、多くの人はまだ尋ねます。 一般的に、ここに解決策があります。



drupalを配置し 、次のモジュールをダウンロードします。

cck 、 filefield 、 drupal.org / project / imagefield 、 image_fupload （readmeをお読みください ！）、 lightbox2 、 Pathauto 、 Token 、 音訳



含まれるもの：

ユーザー参照、ノード参照、ファイルフィールドメタを除くCCKのすべて

新しいタイプのギャラリー素材を作成する



フィールド管理に移動して、フィールドを作成します

ギャラリー画像| gallery_single | ファイル| 画像



フィールド設定で、パス設定を変更します

グローバル設定で、必須、数量1を設定します



もう1つのフィールドを作成します。

フィールド設定で、パス設定を変更します

グローバル設定で、必須、無制限の数を設定します

代替テキスト設定とタイトルテキスト設定をオンにする



表示フィールドに移動

保存、材料の種類は準備完了



同義語のセットアップに移動します。 マテリアルタイプ「Gallery」の場合、この値を設定します：gallery / [title-raw]



トピックのCSSを開き、最後に追加します 。

.field-field-gallery-pics .odd, .field-field-gallery-pics .even {width: 150px; height: 150px; float: left; margin: 5px; }







素材の作成に進み、新しいギャラリーを作成します。 確認します。 許可に表示する許可を忘れないでください）



実際には、簡単なギャラリーが用意されています。 アイデアを開発することができます：カテゴリの追加、タグ付け。 Viewsモジュールを有効にして、多数のビューを作成します。
