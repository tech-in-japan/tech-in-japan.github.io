6月の第1週のWindows 8リリースプレビュー

日本でのWindows Developer Daysカンファレンスの枠組みの中で、Microsoftは6月の第1週にWindows 8 Release Previewのリリースを正式に発表しました 。

現時点では詳細はありませんが、 Building Windows 8ブログに変更のリストが表示される可能性があります。
