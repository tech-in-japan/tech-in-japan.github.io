GoogleモスクワでのDart Language Tech Talk

GoogleロシアとGoogle Developer Relationsは、10月14日金曜日午後7時にGoogleのモスクワオフィスで開催されるDART Tech-Talkに GTUGモスクワグループを独占的に参加させていただきます。



Tech-Talk ：DART、クイックプロトタイプから本格的なアプリまで。

スピーカー ：Dan Inc、Google Inc.エンジニアリングマネージャー

日時 ：10月14日金曜日19:00

場所 ：Googleモスクワオフィス（7 Baltschug St.）。



招待者のみ ：イベントに登録するには、GTUGグループbit.ly/mosgtugのメンバーであり、ここにフォームを記入する必要があります： bit.ly/dartmskreg 急いで 招待できるのは80人のみです（このイベントに登録したすべてのGTUG参加者からランダムに選択されます）。



食べ物 ：知的欲求を満たすだけでなく、肉体も満足させることができます-Googleは伝統的にゲストを養います。 また、小さな友達に食事が提供されます（ガジェットとラップトップ用のソケット）。



招待客のリスト ：ケータリングとアクセス制御のため、登録は金曜日の12:00に締め切られます。 招待された通知は、同日の14:00に送信されます。 したがって、この時点でアンケートに記載されているメールボックスを確認できることを確認してください。



Dartは、Googleが作成したプログラミング言語です。 DartはJavascriptの代替/代替として位置付けられており、進化的開発では修正できない「基本的な」欠陥に悩まされています。

このプログラミング言語に関する最初の公開情報は、2011年9月12日にGoto Developer Conferenceで公開されました。 Google Dart言語の公式プレゼンテーションが2011年10月10日に開催されました。

言語開発者向けのタスクセット：

  • Webプログラミング用の構造化された柔軟な言語を作成する
  • 言語を既存のものと同様にして、学習を簡素化
  • ブラウザおよびその他の環境（スマートフォンからサーバーまで）で受信したプログラムの高いパフォーマンス。




現在、Dartプログラムを実行するには、仮想マシンを使用する方法とjavascriptでブロードキャストする方法の2つがあります。
