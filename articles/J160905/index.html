Microsoft Security EssentialsがAVテストの認定に失敗する





2か月ごとに、ドイツの独立した情報セキュリティ機関AV-Testは、人気のあるウイルス対策ソフトウェアをテストします。 9月と10月にWindows 7で実行した最新のテストでは、Microsoft Security Essentialsは認定を受けるのに十分なポイントを獲得しませんでした。



これ自体はそれほど印象的ではありませんが、AV-Testから証明書を受け取っていないのはMicrosoftソフトウェアだけです。 これに対して、他の3つの無料のウイルス対策製品（Avast、AVG、Panda Cloud）がそれを受け取りました。



ウイルス対策ソフトウェアがポイントを獲得するカテゴリは、保護、回復、使いやすさの3つです。 ポイントが加算されると、ソフトウェアは、証明書を受け取るために18のうち少なくとも11ポイントを獲得する必要があります。 Security Essentialsは10.5ポイントしか獲得できませんでした。



その理由は、Security Essentialsが十分な量のゼロデイマルウェアを認識できなかったためです。 その率は9月に69％、10月に64％でした。 テストされたすべてのアンチウイルスの平均は89％でした！ 残りのスコアは、以前のテストと比較して変化していません。 公平に、テストされたソフトウェアの大部分は、5月と6月よりも結果が低かったことに注意してください。



ポイント数の点で勝者はBitdefender Internet Securityで、18ポイント中17ポイントを獲得しました。 F-SecureとKasperskyの複合体は、それぞれ15.5ポイントと15.0ポイントで2位と3位になりました。 フリーソフトウェアの中で、ZoneAlarm Free Antivirus + Firewallは最高14.5ポイントを獲得しました。



企業のセキュリティ製品の同様のテストで、Microsoft Forefront Endpointは9.5ポイントしか獲得しなかったことに注意してください。
