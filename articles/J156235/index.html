Windows 8用の新しいEvernote





ここ数年、Windowsの進化に関心を持って注目しています。 Windows Phoneに表示されたインターフェイスにより、このプラットフォームでEvernoteを使用する際のまったく新しい見方ができました。 そして本日、Evernote for Windows 8をリリースしました。これは、Windows 8を搭載したデバイスでのタッチ制御に最適化された新しいアプリケーションです。お使いのコンピューターのEvernoteに追加することができます。 新しいアプリケーションの詳細については、以下をご覧ください。



Evernote for Windows 8をダウンロード»



設計



Evernote for Windows 8はオペレーティングシステムの設計に適合し、アプリケーションインターフェイスを簡単かつ便利にします。 メモのリストは、大きなタイルの形で表示されます。 Evernoteは、ナビゲーションおよびアプリケーションとのやり取りのためのすべてのタッチジェスチャをサポートしています。 ブラウザ、Skitch、またはその他のアプリケーションと同時にEvernoteを表示するには、分割画面モードに切り替えることができます。







ジェスチャー



Windows 8は、タッチコントロール用の便利なジェスチャを多数サポートしています。

  • スマートスケーリング：親指と人​​差し指でメモまたはタグのリストを展開し、アルファベット順または時系列の並べ替えオプションを選択します。 目的のグループに移動するには、リストをクリックするだけです。
  • メインパネルとナビゲーションパネル：画面を上下にスワイプして、目的のパネルを開きます。 ナビゲーションパネルを使用すると、アプリケーションのさまざまなセクションにすばやくジャンプできます。メインパネルを使用すると、メモを作成したり、メモを同期したり、さらに多くのことができます。
  • チャームバー：画面を右から左にスワイプするとチャームバーが表示され、そこからノートを検索および共有したり、表示設定を変更したりできます。
  • 画面の分割： Evernoteを別のアプリケーションと並行して表示する場合は、画面を左から右にゆっくりスワイプします。
  • 複数のオブジェクトを選択：複数のノートを一度に選択するには、ノートのリストの名前またはタグを押して下にスクロールします。


ノートとのコラボレーション



Evernote for Windows 8では、検索バーだけでなく、特定のノートブックの内容やメモのリストを表示してメモを見つけることができます。 目的のメモが見つかったら、アプリケーションは快適な表示のためにすべての条件を作成します。







新しいメモを作成するには、メインパネルの対応するアイテムをクリックします。 ノートを保存するノートを指定したり、ノートにラベルを割り当てるには、ノートの表示または作成中にサイドパネルを使用できます。 編集アイコンをクリックして、既存のメモの最後にテキストを追加することもできます。 アプリケーションはまだ書式設定によるテキスト編集をサポートしていないことに注意してください。つまり、書式設定されたテキスト、添付画像などは編集できません。 近い将来、この機能を追加する予定です。



ノートブックとタグ



EvernoteのWindows 8ノートブックおよびタグビューアーモードを使用すると、必要なメモをすばやく参照して見つけることができます。



ノートブックを見る

ノートブックビューモードでは、目的のビューを自分で設定し、リストをノートブックのセットに完全に展開または折りたたむことができます。 チャームバーで表示モードを設定できます。 さらに、ノートブックが共有されていることを示す特別なアイコンを追加しました。 スマートスケーリングを使用してノートブックのリストを展開し、すぐに目的のノートにジャンプします。 必要に応じて、メインパネルから新しいノートブックを追加することもできます。







タグを表示

投稿の構成がタグに基づいている場合は、Evernote for Windows 8でタグリストを実装する必要があります。最も簡単なオプションは、タグをクリックして、それに関連付けられているメモを表示することです。 ただし、検索を絞り込むことができます。 1つのタグをクリックすると、リスト内の一部のタグが灰色になり、一部のタグが灰色にならないことに注意してください。 表示されるのは、選択したラベルとともに特定のレコードで使用したものです。 2番目のタグをクリックすると、指定された両方のタグが割り当てられているメモのリストが開きます。



たくさん来ます



すでにWindows 8のEvernote更新プログラムに取り組んでいます。現在、このアプリケーションはノートを表示するための快適な条件を提供します。今後、ノートを作成する機会を追加する予定です。 Evernote for Windows 8をお楽しみいただき、ご意見をお待ちしております。
