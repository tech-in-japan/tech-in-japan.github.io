Mac OS X用のSkype 5.0ベータ

まあ、待つ価値があった。 本日、Skype 5.0 for Mac OS Xのベータ版がリリースされました。このインターフェイスは、プログラムの使用を簡素化し、Mac標準との整合性を高めています。 これで、Skypeクライアントはすべて1つのウィンドウにまとめられ、システムアドレス帳に接続されるため、連絡先に電話をかけたり、メッセージを交換したり、連絡先にSMSを送信したりできます。



また、Skype for Macは新しい機能をサポートしています。グループビデオコールを使用すると、複数の参加者とビデオ会議を行うことができます（全員が最新バージョンのSkype for Windowsまたはベータ版for Macを使用している場合）。



さらに、新しいSkype for Macには次の新機能が搭載されています。

  • 他のウィンドウの上にあるコール制御ボタンを備えた新しいコントロールパネル
  • チャット履歴をすばやく検索し、過去のメッセージを便利に表示
  • オフラインチャットメッセージ配信
  • 連絡先のカスタマイズ
  • ドロップされたコールにすばやく接続
  • 友だちやグループをオンラインで確認できるフローティング「連絡先モニター」パネル。


Windowsの最新バージョンでおなじみの機能のうち、Macの新しいベータ版には、主にFacebookとの統合および会議中の連絡先の動的な表示がありません。



新しいベータ版はhttp://www.skype.com/intl/en/get-skype/on-your-computer/macosx/beta/からダウンロードできます

新しいSkypeを試して、あなたがそれについてどう思うか教えてください。 公式リリースもそう遠くない。
