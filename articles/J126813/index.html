10役に立つVMwareツール

この記事は翻訳です。オリジナルはこちらをご覧ください-www.vladan.fr/top-10-free-tools-to-manage-your-virtual-infrastructure



仮想環境を常に使用している各システム管理者には、VMwareインフラストラクチャ用に設計された独自のツールがあります。



このレビューでは、広く知られたものと新しいものの両方がある、仮想環境向けの10の優れたツールを検討します。



1.パテ。 Unix \ Linuxサーバーを操作するための最も一般的なユーティリティの1つ。 仮想化のコンテキストでは、SSHプロトコルを使用してESX \ ESXiサーバーに接続するために使用されます。 このユーティリティは、「ポータブル」バージョンとして利用できます。 以下からダウンロードできます： portableapps.com/apps/internet/putty_portable



2. VMware Converter。 実際のシステムを仮想マシンに変換できる無料のユーティリティ。 このユーティリティは、物理サーバーを仮想サーバーに変換する機能に加えて、AcronisおよびSymantec Ghostイメージも認識します。 VMware Webサイトからダウンロードできます ： downloads.vmware.com/d/info/datacenter_downloads/vmware_vcenter_converter_standalone/4_0



3. Trilead VM Explorer-仮想環境でバックアップとリカバリを管理するプログラムで、ESXインフラストラクチャへの接続とVMDKファイルの管理を簡単にします。 最大5台のESXサーバーの接続をサポートします。 ここからダウンロード： www.trilead.com



4. Veeam FastSCP-この小さなユーティリティのおかげで、SSHプロトコルを介して仮想マシンのファイルをコピーできます。 このユーティリティを使用すると、複数のVMware ESXおよびESXiサーバーに接続し、1つのインターフェイスからそれらの間でファイルを転送できます。 このユーティリティは無料で入手できます： www.veeam.com/vmware-esxi-fastscp.html



5. BGInfoは、デスクトップに接続している現在のサーバーに関する重要な情報を表示する、小さいながらも非常に便利なユーティリティです。 多数のセッションが存在する場合、この機能により、セッションで迷子にならないようにすることができます。 こちらから入手できます： technet.microsoft.com/en-us/sysinternals/bb897557.aspx



6.デーモンツール-CD / DVDディスクの形式でISOファイルをマウントできる有名なプログラム。 最大4つのデバイスをエミュレートできるため、複数の画像を同時に操作できます。 ここからプログラムをダウンロードできます： www.disc-tools.com/request?p=4f261f3d8ccec3edc1d8ecc623740c59 / daemon4304-lite.exe



7. ImgBurn-Daemon Toolsと同様の機能がありますが、ファイルまたはディレクトリのセットからISOイメージを作成できます。 ここからプログラムをダウンロードしてください： www.imgburn.com



8. mRemoteは無料のプログラムです。そのおかげで、1つのインターフェイスからさまざまなプロトコルで作業を整理できます。 次のプロトコルがサポートされています。

-RDP（リモートデスクトップ）

-VNC（仮想ネットワークコンピューティング）

-ICA（独立コンピューティングアーキテクチャ）

-SSH（セキュアシェル）

-Telnet（テレコミュニケーションネットワーク）

-HTTP / S（ハイパーテキスト転送プロトコル）

-Rlogin（Rlogin）

-RAW

ここからダウンロード： www.mremote.org/wiki/Downloads.ashx



9. OpsCheck-VMWare VMotionをサポートするためのシステム設定の正確性を検証するのに役立つプログラム。 このプログラムは、ESXサーバーの構成を分析し、適切な構成のための推奨事項を提供します。 このプログラムは、TripwireのWebサイト（ www.tripwire.com）からダウンロードできます。



10. RVTools-プロセッサ、メモリ、ディスク、ネットワークアダプタ、仮想マシンのスナップショット、VMWAREツール、ESX \ ESXiストレージに関する情報を表示するユーティリティ。 こちらからユーティリティをダウンロード-www.robware.net
