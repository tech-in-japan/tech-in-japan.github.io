Microsoft TechDayとは何ですか？



2014年6月に開催したこのような最初のイベントには、多くの肯定的なフィードバックが寄せられました。 これが良い伝統になることを願っています。6月10日に、次のMicrosoft TechDayに招待します。これは、技術コミュニティ向けであり、技術コミュニティによって作成されたイベントです。 したがって、あなたは興味深く、深い技術的内容を見つけるでしょう。



Microsoft TechDayで話すのは誰ですか？



イベントの主な発表者は、さまざまな分野の独立した専門家とMicrosoft MVPです。



Microsoft TechDayは何について話しますか？



今回は、Windows Server vNextに焦点を当てます。 Hyper-Vの新機能からコンテナーやNano Serverまで、Windows Serverの次のバージョンで最も興味深い新製品について説明します。 最近のBuildおよびMicrosoft Igniteカンファレンスのサーバーに関する最新のニュースを、オフラインでもオンラインでも1日で聞くことができます。
