問題解決


プロキシサーバーを管理するための特別なプログラムが存在する可能性があります（コメントに一覧表示できることがわかっている場合）。 私は反対に行きました。 このチェックボックスは、レジストリ内のチェックマークを担当します。



HKCU\Software\Microsoft\Windows\CurrentVersion\Internet Settings\ProxyEnable







この値をすぐに反転させるbatファイルを作成します。



@echo off<br>

set REGDIR="HKCU\Software\Microsoft\Windows\CurrentVersion\Internet Settings"<br>

ping 192.168.0.242 -n 1<br>

if "%errorlevel%"=="0" ( REG ADD %REGDIR% /v ProxyEnable /t REG_DWORD /d 1 /f<br>

echo "work"<br>

) else (<br>

REG ADD %REGDIR% /v ProxyEnable /t REG_DWORD /d 0 /f<br>

echo "home"<br>

)<br>









職場でのインターネットプロキシの192.168.0.242 IPアドレスはどこですか

アルゴリズムは単純です。pingを実行する場合はプロキシにpingを実行し、そうでない場合は職場でチェックを行い、チェックを外します
