2010年。 FIFA 2010



誰も待つことができません-それがあったように：





彼らは各学部のチーム（4人8チーム）で組み立てられ、 SBIの 2階に8台の車を置き、窓にリアプロジェクションプロジェクターを、窓の前にスクリーンを備えたクレーンを置きました。

そのとき、ゲームのルールとバグに多くの注意が払われました。 ルールへ-ゲーム自体で規定されているルール（切断、ドローなど）の上。 バグ-コーナーからの100％の詰まりなど。 興味のある方-doc file ）。

追加する特別なものはありません-みんなが設定し、調整し、夕方前後にプレイします！ 確かに、遅くなって暗くなり、ゲームが11を過ぎて終了するという問題がありました。 適切な人々が到着し、私たちを解散させ始めました。 まあ、最初のパンケーキ-彼はそのようなものです。 しかし、学生のレビューによると、すべてがクールでした:)



2011年。 FIFA 2010



FIFAと自動車の場合、何も変更しませんでした。 しかし、私たちはこのイベントを路上で開催することを拒否しました（寒さと時間の問題。暗くなるまで待つ必要があります）。 そして、私たちはTeatroアクションクラブに移りました。







劇場は、大きなスクリーンとダンスフロアを備えた2階（3階？）の部屋です。 さらに、各バーにはプラズマがあり、一般的なビデオも表示されます。 プレイヤーを2階の別の部屋に配置し、すべての機器をセットアップして、共通画面に画像と一般的な音を表示しました。





Global_max写真



ゲームのルールは同じままです。 しかし今では、「ゲスト」を待つことなく、落ち着いてプレイできました。

ネットワークには8台の車があり、ビデオは1台から出力され、2台のビデオ出力がありました。 プレーヤーはDVIモニターを介して接続され、VGAはSビデオ/チューリップコンバーターに接続されていました。Sビデオ/チューリップコンバーターは、既にクラブ全体の共通信号のためにオペレータールームに残されていました。 原則として、出力画像の変更を除き、2010年と同じです。



2012年。 FIFA 2012



今年、一歩前進しました。 私たちはPlayStation 3を取り、6つのジョイスティックを接続し（ゲームは3x3にしました）、fifuを起動し、同じことを行いました:)場所は同じままです-Teatroアクションクラブ。

ビデオはまだありませんが、実際には別のプレフィックスとクリエイティブチームの他のパフォーマンスのみが表示されます。

しかし、ここから面白い瞬間が始まります。 Fifa 2012 PS3仕様では、協力ゲームのゲームは最大4人、ネットワークでは最大10人をサポートするとしています。

www.metagames.ru/ps3/game/FIFA_12

EAのWebサイトで、PS3の仕様がまったく見つかりませんでした。 だから-これはそうではありません。

USBハブとそれらを接続しました-PS3から4ピース+ 2ワイヤレスの量の通常の有線USBジョイスティック。 さらに-ジョイスティックが動作を拒否するという問題が始まります。追加のパワーを備えたハブが必要な場合があり、確認することができませんでした。



そして、画面への出力はどうですか？ PS3にはhdmiとAVの2つのビデオ出力があります。 しかし、同時に彼らはせいぜい動作することができません-彼らはまだhdmi経由で音を出します。 プラズマがまだ残っている場所（クラブのすべてのスクリーンにビデオ信号がすぐに出力されることを書きました）と、誰も邪魔しない場所にプレーヤーを移動する必要がありました。 ゲームは成功しました:)今年、彼らはクラブ自体のゲームを遅らせ、観客を飽きさせないように、より多くの予選を行いました。

ショーに加えて、スーパーフィナーレもありました。 優勝チームは、学生の管理と労働組合委員会でプレーしました:) 2番目のチームの特徴はわずかに過大評価され、勝ちました。とても楽しかったです。
