Pearl Firefox 3.0から3.0.1へのアップグレード

最も一般的な平日、7月17日でした。 さまざまなブラウザーでサイトの表示をテストする必要があり、そのためにインストール済みのFirefox 3を起動しました（コンピューターにもFirefox 2.0.0.11があり、デバッグ目的でも使用されています）。 Foxはすぐに新しいバージョン-3.0.1が登場したことを示すメッセージを発行し、アップデートすることを強く提案しました。 「更新」をクリックすると、接続が確立され、2 MBの更新がダウンロードされます。 Firefoxは更新中です。 その後、いつものように-再起動後、アドオンの互換性をチェックし、互換性のないPencilのみを見つけて更新します。 そして最後に、おなじみのFx Treshkiウィンドウが開き、 「Firefoxが正常に更新されました」ウィンドウのタイトルが表示されます。 しかし、私は何を見ますか？ Mozillaページが開き、「 Firefox 2.0.0.11を使用しています ！」という大きな文字で表示されます 。 言うまでもなく、私の驚きは際限がありませんでしたか？ 「 ヘルプ 」->「 Mozilla Firefoxについて 」に進みます。実際、Firefox 3.0のインターフェースを備えたFirefox 2.0.0.11があります。











私のサイトにもインストールされているFirefox 2.0.0.11を実行しようとしています。 いつものように-アドオンの互換性を確認し、Firefoxを起動すると...「 Firefoxの最新バージョンに正常に更新されました 」という碑文のあるほぼ同じページに移動します。 また、「 ヘルプ 」->「 プログラムについて 」に進みます-バージョン2.0.0.11を示しています。







私はすぐにHabréに登り、この問題に関するトピックを見たいと思っていましたが、Firefoxを3.0.1にアップデートする際に見つかったトピックでは、そのような「冗談」についての言葉ではありません。 コメントでも同じです。

軸-Windows Vista Home Premium SP1。

2.0.0.11にインストールされた追加-Pencil 1.0.2のみ、Fx 3-HTML Validator 0.8.5.2、Pencil 1.0.2、Web Developer 1.1.6。
