最大のファンタジースポーツ企業の数百万ドル規模の事業が2つの州で違法と宣言されました

2006年、オンラインギャンブルを禁止する法律が可決されました。 ファンタジースポーツについては例外が設けられ、非常に無害な規模でした。 したがって、これまで、この業界は当局によって規制されていませんでした。 現在、ファンタジースポーツ市場で活動している2つの大企業、 FanDuelとDraftKingsがあります。



ファンタジースポーツは、参加者がチャンピオンシップの実際の選手のチームを構成するオンラインゲームです。 確かに、ファンタジースポーツファンは、ゲームの結果に続くポイントを受け取るだけでなく、対応する金額を獲得します。 このゲームは、業界で数十億ドルが回転しているアメリカとカナダで非常に人気があります。



11月25日、ニューヨーク最高裁判所は、DraftKingsとFanDuelの活動に関する訴訟を開催しました。 検察は彼らの空想スポーツ事業を違法と判断した。



本日、TechCrunch は 、イリノイ州検事総長リサ・マディガンがニューヨークのカウンターパートに同意し、ファンタジーリーグを違法だと認めたと報告した。



したがって、このビジネスは2つの州の法執行機関によって嫌がらせを受けています。 以前、Re / code は 、カリフォルニアの規制当局もFanDuelに対する攻撃の準備をしていると報告しました。



マディガンは、イリノイ州刑法のセクション28-1（a）（12）を参照しています。法人または個人は、お金やその他の富のためにゲームに参加するウェブサイトを運営、維持、および運営している場合、法律に違反します。



彼女の意見では、利益が運またはスキルに依存するかどうかは問題ではないことに特に注意する価値があります。 このような一連の考えは、防衛の主な議論を打ち破ります。ファンタジースポーツでは、ユーザーは単に「運」のために賭けるだけではなく、チャンピオンシップ、アスリートのパフォーマンス、ゲーム統計でゲームを分析します。 これにはすべて特定のスキルが必要です。



検察官はまた、DraftKingsとFanDuelのユーザーを競技会の参加者やアスリートとは見なしていないと述べました。



ファンタジーリーグは、司法当局が活動を終了することを決定するまで、2つの州の領土で違法と宣言されたという事実にもかかわらず。 これまでのところ、訴訟はそのようなビジネスにどのように関係するかという問題のみに関係していました。



DraftKingsとFanDuelは、検察官の調査結果に同意しません。 彼らは、ファンタジースポーツの規制に対する態度を再考し、立法レベルでこの問題を解決するよう州に説得するために最大限努力します。



ファンタジースポーツの物議を醸す評判にもかかわらず、ファンタジースポーツは依然として「違法」であるという事実にもかかわらず、ファンタジースポーツ業界は急速に発展しています。 FanDuelとDraftKingsは数億ドルの投資を集めました 。 今年、企業はファンタジートーナメントに参加するユーザーに約10億ドルの賞金を支払います。 DraftKingsの投資家はFox ChannelとNational Sports Leagueです。
