WN727N WiFiアダプターをUbuntu / Mintに接続する

wn727n wifiアダプターをubuntu / mintに接続する際に問題が発生しました。 私は長い間グーグルで検索しましたが、解決策が見つかりませんでした。 問題を解決したので、私はそれを自分で書くことにしました。 以下に記述されているものはすべて、初心者向けに設計されています。



注意！ 記事の著者はいかなる損害に対しても責任を負いません！


しかし、あなたがすべてを正しく行えば、結果はありません。 何かがうまくいかなくても、悪いことは何も起こりません。 始めましょう。



まず、Ctrl + Alt + Tキーを使用してターミナルを開き、次のコマンドを入力します。 



lsusb






Ralink RT7601アダプタ（強調表示）が表示されます。 Ralink RT5370アダプターがあります。 異なるアダプター上のドライバーは異なる方法でインストールされます。 2つのケースでこれを行う方法を説明します。



Ralink RT5370のマニュアル



リンクに従って、RT8070 / RT3070 / RT3370 / RT3572 / RT5370 / RT5372 / RT5572 USB USBを選択します。 ドライバーでアーカイブをダウンロードします。



ドライバーが保存されたフォルダーを開き、bz2アーカイブを解凍します。 これを行うには、ファイルを右クリックして、「ここに抽出」をクリックします。



その後、tarアーカイブが表示されます。 再度開梱します。 ファイルを右クリックして、「ここに抽出」をクリックします。



次に、フォルダーへのパスをコンソールに書き込む必要があるため、フォルダーの名前を短い名前に変更します。 たとえば、私はそれをドライバーと呼びました。



解凍したフォルダーに移動し、テキストエディターで/os/linux/config.mkファイルを開きます。



次の行を見つけて、文字nをyに変更します。



＃Wpa_Supplicantをサポート

HAS_WPA_SUPPLICANT = y



＃Network MagangerのネイティブWpaSupplicantをサポート

HAS_NATIVE_WPA_SUPPLICANT_SUPPORT = y



その後、ファイルを保存します。 ターミナルを開き、解凍したフォルダーに移動します。 注意！ 私はセルゲイのユーザー名を持っています。 ユーザー名を入力してください！ 将来的には、sergeyをユーザー名に変更してください。 



 cd /home/sergey//driver/


次に、コマンドを実行します。 



 sudo make sudo make install sudo modprobe rt5370sta


そしてそれだけです！ 奇跡！ WIFIが機能し、健康を楽しんでください。



Ralink RT7601のマニュアル



このアダプター（Ralink RT7601）を入手するには、カーネルバージョン3.19以降が必要です。 必要に応じて-カーネルを更新します（方法がわからない場合-googleが役立ちます）。



次に、 リンクをクリックしてドライバーをダウンロードします。







次に、ダウンロードしたアーカイブをホームフォルダーに移動して展開します（右クリックして「ここに抽出」）。 結果のmt7601-masterフォルダーの名前を単にmt7601に変更します。



その後、次のコマンドを入力します。 



 cd mt7601/src


これで正しいディレクトリに移動しました。 ドライバーをビルドし、コマンドを実行できます： 



 sudo make


システムはパスワードを要求します-入力します（パスワードは表示されません）。



次に、コマンドを入力します。 



 sudo mkdir -p /etc/Wireless/RT2870STA/ cp RT2870STA.dat /etc/Wireless/RT2870STA/


そして、アダプターを有効にする最後のコマンド： 



 insmod os/linux/mt7601Usta.ko


すべて!!! 現在、ubuntuはwifiを認識しています。



しかし、それだけではありません！ ここで、再起動するたびに最後のコマンドを入力する必要があります。そうしないと、システムにアダプターが表示されません（Ralink RT7601専用） しかし、解決策があります！ スクリプトを作成して、スタートアップに追加できます。 以下にその方法を示します。



まず、システムがsudoを使用するときにパスワードを要求しないことを確認する必要があります。 これを行うには、次のコマンドを入力します。 



 sudo gedit /etc/sudoers


このウィンドウが開きます：







行を探しています：



％sudo ALL =（すべて：すべて）すべて



そして、それを次のように変更します。



％sudo ALL =（すべて：ALL）NOPASSWD：ALL



変更を保存します-[保存]をクリックします。



次のコマンドを入力します。 



 sudo cp -R mt7601 /etc/Wireless/RT2870STA/


その後、次のコマンドを入力します。 



 sudo gedit /etc/Wireless/RT2870STA/autowifi.sh


空のテキストエディターが開きます。 その中で次のように記述またはコピーします。



＃！/ bin / bash

insmod /etc/Wireless/RT2870STA/mt7601/src/os/linux/mt7601Usta.ko



[保存]をクリックして閉じます。



次のコマンドを入力します。 



 cd /etc/Wireless/RT2870STA/ sudo chmod +x autowifi.sh


次に、ダッシュメニューに移動し、下の図のようにプログラムを探します。







開けます。 「追加」をクリックします。







ウィンドウが開きます。 書き込む「名前」フィールドの反対側：



autowifi



「チーム」フィールドの反対側に、次のように記述します。



sudo sh /etc/Wireless/RT2870STA/autowifi.sh



[追加]ボタンをクリックして、プログラムを閉じます。 再起動します。 再起動後、すべてが機能します。 これで、トレイでネットワークを選択できます。







これで、Ralink RT7601アダプターの「小さな」手順は完了です。



ネットで楽しんでください！
