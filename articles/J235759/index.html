Wordpress 4.0 "Benny"のレビュー

昨日、 Wordpress 4.0がリリースされ 、ジャズクラリネット奏者で指揮者の「スイングキング」 ベニーグッドマンに敬意を表して「ベニー」と名付けられました。 リリース後の最初の数時間で、Wordpress 4は20万回以上ダウンロードされました。 Wordpressの3番目のバージョンは3億回以上ダウンロードされており、世界で最も人気のあるCMSのタイトルをJoomlaと競合しています。







4.0は3.9と4.1の間の数にすぎないという事実にもかかわらず、開発者はこのリリースにもう少し投資しました。 新しいバージョンでは、100を超える新機能とメディアファイルとエディターの操作の改善により、投稿の作成とCMSの管理プロセスがさらに便利になりました。



公式サイトから新しいバージョンをダウンロードしたり、コントロールパネルからWordpressを更新したり、クラウドホスティングで1クリックでInfobox Jelastic をインストールしたりできます （このため、このCMSのリリース直後にJPS標準に従ってWordpress 4をJelasticにパックします）。



猫の下でWordpress 4.0の新しいバージョンの主な変更点のレビュー。



新しい種類のメディアグリッド







これで、メディアライブラリがデフォルトの画像グリッドとして表示されます。 アイテムをクリックすると、画像のプレビューと追加情報が表示されます。これらの情報は簡単に編集でき、次の画像にすばやく移動できます。







複数選択を使用すると、複数の画像を一度に選択し、必要に応じて削除できます。







改善されたエディター







Wordpressエディターでテキストを書くのがさらに便利になりました。 ビジュアルエディタは、スクロールバーを使用する代わりに、すべてのコンテンツに対応するように拡張されています。 編集時、スクロール時に固定ツールバーが表示されるようになりました。 したがって、必要なツールは常に手元にあります。







oEmbedプレビュー



Wordpress 4.0は、TEDトーク、Mixcloud、CollegeHumor、Issuu、PolldaddyのoEmbedプレビューのサポートと、YouTubeプレイリストのサポートを追加しました。 ビジュアルエディタに、oEmbedリンクを介して追加されたメディアファイルのプレビューが表示されるようになりました。



Youtubeで開いているクリップのブラウザー行からリンクをコピーし、そのリンクをWordpressエディターに直接貼り付けるように見えます。 ビデオが挿入され、投稿とエディターの両方で再生されます。







プラグインの検索とインストールのプロセスを更新



Wordpress 4.0は、管理パネルからプラグインを見つけるプロセスを非常に簡単にします。 プラグインインストーラーは、グリッド内のカードとしてそれらを表示するようになりました。 拡張機能の作成者は、インストーラーに表示される独自のアイコンを追加できるようになりました。







「詳細」リンクをクリックすると、拡張機能の説明、評価、レビュー、互換性情報を表示できます。







言語サポートの改善



Wordpress 4を自分でインストールする場合、インストール中に言語を選択できます。







Wordpress 4コントロールパネルのメイン設定セクションからインストール言語を直接選択することもできます。







Wordpress 4の使用に成功しました！
