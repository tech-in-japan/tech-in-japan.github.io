Windows Server 2016ライセンス





マイクロソフトは最近、 Windows Server 2016に関する詳細情報を発表しました。 Hyper-V、コンテナ、Nano Serverなどに関するいくつかの技術情報が既にあります。 ただし、Windows Server 2016のライセンスに関する情報があります。





小さな警告： 記事内のすべての情報は、2015年12月にMicrosoftがリリースしたドキュメントから引用したものです。将来、変更される可能性があります。 この記事は、ライセンスの変更の概要のみを目的としています。 購入したライセンスが正しいことを確認する場合は、MicrosoftまたはMicrosoftパートナーに連絡する必要があります。



Microsoftは、Windows Server 2012およびWindows Server 2012 R2の場合と同様に、Windows Server 2016の2つのバージョン-DatacenterとStandardをリリースします。 Windows Server 2016のライセンスには2つの大きな変更があります。

最初に、MicrosoftはWindows Server 2016 DatacenterおよびStandard Editionのプロセッサごとのライセンスからコアライセンスに切り替えます。



2番目の大きな変更点は、Standard EditionとDatacenter Editionの間に機能的な同等性がないことです。 Windows Server 2012 R2には基本的に両方のエディションで同じ機能があり、唯一の違いは仮想マシンを使用する権利でした。 Datacenterエディションの仮想マシンの自動アクティベーションを除きます。



Microsoftがリリースしたドキュメントの興味深い事実：



Windows Server 2016エディション：



  • Datacenter Editionは、高度に仮想化されたプライベートおよびハイブリッドクラウド環境向けに設計されています。
  • 非仮想化環境または部分的に仮想化された環境のStandard Edition。
  • Windows Server 2016およびWindows Storage Server 2016の他のエディションに関する情報は、2016年の第1四半期に発表されます。


Windows Server 2016のエディションのおおよその価格は次のとおりです。





コアライセンスモデルを変更することの意味：



物理サーバーのライセンスを取得するには、すべての物理コアのライセンスを取得する必要があります。 サーバー上の物理プロセッサごとに最低8つのライセンスが必要であり、1つのプロセッサを搭載したサーバーには最低16コアのライセンスが必要です。

  • Windows Server 2016 DatacenterおよびStandard Editionの16コアライセンスの価格は、Windows Server 2012 R2の対応するエディションの2プロセッサーのライセンスの価格と同じになります。
  • Standard Editionでは、サーバー内のすべての物理コアにライセンスが付与されている場合、2つのオペレーティング環境またはHyper-Vコンテナーをインストールすることができます。 追加のオペレーティング環境またはHyper-Vコンテナーの一部のカーネルには、複数のライセンスをインストールできます。
  • ライセンスを取得したWindows Server StandardまたはDatacenter Editionへのアクセスを必要とする各ユーザーおよび/またはデバイスには、Windows Server CALが必要です。 各Windows Server CALでは、複数のWindows Serverライセンスにアクセスできます。
  • Windows Server CALは、ユーザーまたはデバイスが同じまたは以前のバージョンのWindows Serverのエディションにアクセスする権利を与えます。
  • リモートデスクトップサービスやActive Directory Rights Managementサービスなどの追加機能または高度な機能のライセンスを取得するには、以前と同様に追加のクライアントアクセスライセンスを購入する必要があります。




Windows Server 2016 StandardおよびDatacenter Editionの物理カーネルのライセンスを取得する方法





  • すべての物理サーバーコアのライセンス。
  • 各プロセッサーのコア用に少なくとも8ライセンス。
  • 各サーバーのコアに最低16ライセンス。
  • カーネルライセンスはパックで販売されます。 パックごとに2つのライセンス。
  • 各物理サーバーのライセンスを取得するための最小要件は、8パックの購入です。
  • 各エディションの2ライセンス（パック）のコストは、対応する2012 R2エディションの2つのプロセッサーのライセンス価格の1/8です。


Windows Server 2016のライセンスの詳細については、 www.microsoft.com / en-us / server- cloud / products / windows-server-2016をご覧ください。
