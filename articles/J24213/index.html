数学について

数年前、人道教育に関する私の知り合いの一人は、「数学の何が問題なのか。すべてが厳しく、すべてが開かれており、2 + 2は常に4に等しい」と言いました。 残念ながら、私はまだ少年であり、適切に答えることができませんでした。





何回、試験の準備中に、「まあ、コーシー、気にせよ、ここで発明した、それは明らかではない、彼は何の関係もない」と不平を言った。 もちろん、これはすべてそのようなものではないことを理解しましたが、さまざまな抽象的な定理が豊富にあることから、これはすべて学生をロードするためだけに発明されたように思われ始めました。



実際に数学を使用する人は、これはそうではないことを理解しています。 それらは、なぜこれが必要なのか、それが必要なのかを表しています。 しかし、他の人は何をすべきでしょうか？ たとえば、通常の学校でのレッスン：



「今日、角度のサインが何であるかを調べます。 副鼻腔は、反対側の脚の長さと斜辺の長さの比です...イヴァノフ、質問がありますか？...なぜこれが必要なのですか？ おわかりのように、これは三角法の基礎であり、特に解析幾何学で使用されています... Ivanov！ 寝てる？」



この時点で、イワノフは古代の偉大な数学者であるという夢を持っていました。



「ああ、何かを発明したい！ 確かに、私は何を想像することはできません。 そのため、たとえば三角形を考えてみましょう。 ちなみに、すでに発明されているのは残念です。 それでそれをどうするか？ すべての面を一緒に折りますか？ いいえ、境界線になります。 しかし、分割された場合はどうなりますか？ 片方はもう片方ですか？ はい、それは素晴らしいです！ 誰もこれについて考えたことがありません！ ノーベル賞を贈る必要があります！ まあ、それは気にせず、ノーベルはまだ生まれていませんでした。 いいでしょう このアスペクト比を「サイン」と呼びましょう！ そして、他の当事者の態度は「余弦」です！ それは素晴らしい方法です！ そして、学校の生徒は学ぶべきことがあるでしょう。そうでなければ、彼らはとても怠け者です！」



不条理？ もちろん！ しかし、数学に興味を持っている人、この抽象的な科学は大丈夫です。 他の科学では、少なくとも、これがなぜ必要なのか、実際に何であり、実際にどのように適用されるのかを考える必要はありません（または必要ではありませんが、程度は低いです）。 決して他の科学のほうが簡単だと言いたくはありません。 しかし、理論的には、より具体的なことに興味が生じやすくなります。



実際、数学はツールです。 しかし、誰も発明者がこのように考えたとは思いません。

「ああ、気分はとても良いです！ 私は何かを発明しませんか？ うーん、だから、ここに錆びた鉄片があります、それで何ができますか？ さて、私はそれに穴を開けて、指に引っ掛けることができるとしましょう。 だから、最初は悪くない。 ああ、あなたはまだそれをスティックに置くことができます。 やってみましょう。 そして、なんと、かっこいい、かっこいい小さなものが判明しました！ それでも彼女に電話する必要があります。 うーん...もしそうなら？...、いや、そうではない...ああ！ そして、このことをハンマーと呼びます！ 通常の、非常に男性的な、最も！ しかし、なぜそれが必要なのか、私には想像できません。 まあ、それから誰かが思いつくかもしれません。」



数学は、特定の問題を解決するために作成されました。



私は質問に興味があります：数学は発見されたか、発明されましたか？ 私はそれがオープンだと思っていました。 私はこのように推論しました。基本的な概念の1つは「数」です。 たとえば、人類が出現する前に存在しなかったのですか？ 今、私はそれを非常に疑います。 「数字」は人間によって発明された概念であり、それ自体では存在しないように思えます。

次の論理的なステップは、自然系列の概念です。 引力の法則が存在するように、自然界が私たちの世界に存在すると考えるなら、古代の部族には「1」、「2」、「3」、「多」という数字しかなかったことを思い出してください。 彼らは他の数字を持っていなかったことに注意してください。おそらく十分な数を持っていたでしょう。 理論的には、これらの数値から数学理論を導き出すことができます。 問題は、どれだけ需要があるかです。 たとえば、そのような理論には存在する権利がないわけではありません。



したがって、数字1、2、3および多くのみを使用します。 加算のルールを考え出すことができます：1 + 1 = 1、1 + 2 = 3、1 + 3 =多く、3 + 3 =多くなど。 そのような理論は便利ですか？ もちろん、私たちの世界ではありません。 そして、あなたがこれを想像するなら：

この国では、1ルーブルと多くのルーブルの単位でコインがあります。 あなたは店に来て、2ルーブルのものを取り、2コインを与えます。 物事に多くの費用がかかる場合、多くのルーブルでコインを与えます。 ここで、この理論が役立ちます。



もちろん、記載されているケースは少しおかしいですが、私は考えは明らかです。 ロバチェフスキー幾何学のより現実的な例になりました。

ユークリッドの公理の1つは次のとおりです。

特定の線上にないポイントを通過すると、1本の線のみが通過し、同じ平面内で特定の線と交差します。



ロバチェフスキーは彼女を次のように置き換えました。

少なくとも2本の線が、特定の線上にないポイントを通過し、特定の線と同じ平面上にあり、交差しない。



初心者は、明らかな公理をある種の妄想に置き換えたと信じています。 誰にとってそれは明らかですか？ 私たちだけのために。 ユークリッド幾何学は、私たちの身近な世界に最も適したものの一部ですが、数学には適していません。 ロバチェフスキーは、新しい公理により、完全な理論が得られることを示しましたが、通常の類似物はありません。 しかし、他のモデルではうまく使用されています。 ウィキペディアへの送信に興味があります。



結論：数学は、いくつかの抽象化とステートメントを思いつき、それらから一貫した結果を導き出す能力です。 これは人間の心の力によって作成されたツールですが、なぜ多くの人が数学を好むのでしょうか？



この記事が気に入ったら、たくさんの考えがあるので、私は喜んで数学について（より正確には数学の哲学について）書き続けます。
