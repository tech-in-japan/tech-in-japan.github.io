Stepicプラットフォームの新しいコース：「C / C ++での高度なプログラミング」





Stepicプラットフォームで引き続きコースを実行します。 そして今日、別のニュースを皆さんと共有しています。8月23日に、C / C ++言語での高度なプログラミングのコースが開始されました。 コースは2か月強続きます。 これは、専任の授業を受ける時間がまったくない環境で知識を拡大し、新しい経験を積む絶好の機会です。



講義とコースの実際の部分は、テクノパークとテクノスフィアのプログラムに基づいています 。MSTUと共同で開催される教育プロジェクトです。 N.E. バウマンとモスクワ州立大学。 M.V. ロモノソフ。 両方のプログラムは数年間フルタイム形式で正常に実装され、300人以上の学生が卒業生になりました。 ここ数年、トレーニングの主な重点は、ソフトウェアアーキテクチャと、ソースコードの信頼性、セキュリティ、移植性の問題にありました。



ただし、オンラインコースは、学生だけでなく、言語の構文を知っており、少なくとも1つのプロジェクトがあり、知識を深めたり体系化したい成熟した専門家を対象としています。 現在のトレーニングレベルをすばやく評価するために、最初のモジュールの参加者には2つの簡単なタスクが提供され、その解決策はそれぞれ50〜100行かかります。



ただし、もちろん、コースの主要な「チップ」はタスクではありません。 各コースの参加者は、C ++で簡単な（最大3000-5000行のコードの）プロジェクトを独自に開発する必要があります。 開発の対象分野、および応用技術のスタックは、特に規定されていません。コースの枠組み内でのすべての創造的な決定は、それを研究する人々によって行われます。 タスクセットの実装中、参加者は任意のフレームワークまたはライブラリを使用できます。作業の結果は、従来のモバイルまたはデスクトップアプリケーションだけでなく、ウェアラブルエレクトロニクスまたはWebサイトのサーバー部分のソリューションにもなります。



オンラインコースには4つのモジュールが含まれており、それぞれが順番に開きます。次のモジュールを使用できるようにするには、前のモジュールごとに最小ポイントを獲得する必要があります。 コースチームによると、これは参加者を訓練し、フルタイムクラスの形式に起こっていることをもたらします。 コースの締め切りはソフトとハードです。コースが遅いと、ポイントが減ります。



コースモジュールは10のレッスンにグループ化され、歓迎と結論はカウントされません。 各レッスンでは、参加者は現在の知識管理の質問に答える必要があります。 これを行うために、1つまたは複数の回答オプションを選択した5つの質問が各レッスンに導入されます。 各レッスンの平均ビデオ映像は約100分です。



このコースは、自動化システムと高レベルプログラミング言語の情報とソフトウェアの分野の専門家であるAlexei Petrovによって指導されます。
