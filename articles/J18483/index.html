Ubuntu 8.04 Hardy Heron-待つべきものがあります

わずか3か月で、Ubuntuの新しいリリースであるHardy Heron（ハーディヘロン）を待っています。 このリリースは長期サポートカテゴリに属します。つまり、デスクトップバージョンでは3年間、サーバーバージョンでは5年間のセキュリティ更新プログラムのリリースが保証されます。 Ubuntu 8.04の3番目のアルファ版は既にありますが、もちろん、必死の愛好家やテスターを除き、それに切り替えることはお勧めしません。



一般ユーザーに適したバージョンのリリーススケジュールは次のとおりです。
  • 03/27/2008-ベータ版8.04;
  • 2008年4月17日-リリース8.04の候補。
  • 04.24.2008-Ubuntu 8.04 LTSの最終リリース。
もちろん、新しいリリースにはメインアプリケーションの更新バージョンが含まれ、多くの既知のエラーは対照的に消えます。 しかし、これに加えて、 開発者は多くの新しいことを約束します。



例：
  • グラフィックユーティリティは、Sambaユーザーの管理を簡素化し、ファイル共有を整理するために表示されます（さらに、smbプロトコルだけでなく、httpやftpなどの他の一般的なプロトコルによっても）。
  • ディストリビューションには、システムとモバイルデバイスを同期するためのアプリケーションとテクノロジが含まれます（Windows Mobileを使用したすべてのPDAの完全なサポートを期待しないでください）。
  • 一度に複数のモニターで動作するようにXを構成するツールがあります。 CRTモニターのスイープ周波数は自動的に検出されます。
  • まれに使用されるOpenOfficeコンポーネントは、インストールCDのスペースを空けるために、ディストリビューションから消えます。 もちろん、必要に応じてコンポーネントを簡単に追加できます。
  • [このコンピューターについて]アプレットが表示され、その助けを借りてコンピューターのハードウェアおよびソフトウェア構成に関する基本データを取得できます。
  • Network Managerは、ダイヤルアップ接続とADSL接続を簡単に構成するためのツールを提供します（VPNは再び無視されます）。
  • プロアクティブな最適化に基づいて、頻繁に使用されるファイルへの読み込みとアクセスを高速化するツールがあります。
  • インストールプロセス中に、ネットワークがダウンしている場合、システムは長時間最新の更新を取得しようとしません。
  • 人気のあるFirefoxブラウザーの拡張機能とほぼ同じ方法で、アプリケーションをワンクリックでインストールできます。
  • ネットワークを介してシステムをインストールできるようになるまで、LiveCDへの共有アクセスを整理するためのツールが追加されます。
  • サードパーティのリポジトリの接続が簡素化されます。
  • Ubuntuを実行しているコンピューターをWindowsネットワークドメインに統合するための便利なツールが表示されます。
  • モバイルコンピューターユーザーは、特別なユーティリティと拡張機能（電源管理の改善など）を期待しています。
  • ファイルシステムを暗号化するための便利なツールが表示されます。
  • 新しいバージョンのディストリビューション用に、新しい排他的なテーマが作成されます。
  • ディスクパーティションの名前を変更してフォーマットするためのグラフィカルユーティリティが表示されます。
