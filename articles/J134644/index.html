2GIS API。 毎日新しい組織

2011年12月以来、 API 2GIS製品は毎日更新されています。 たとえば、組織が月曜日にオープンした場合、その組織のデータはすでに火曜日にディレクトリに表示される場合があります。 これは、 すべてのオンライン API ベースの製品 （ Flamp 、2GIS Onlineバージョン 、およびパートナーシッププロジェクト）に適用されます。 したがって、新しいレストランの住所を確認したり、営業開始の翌日に文字通りレビューを書いたりできます。専門家がそのレストランに関する情報を記録します。







以前は、2GIS APIは1日に毎月更新されていました。 しかし、このようなスケジュールでは、たとえばその日の2日目または3日目にオープンした企業は、来月だけディレクトリに落ちました。 現在、すべての変更が毎日行われているため、情報は常に新鮮です。 たとえば、今週、 コンフォートデコレーションマテリアルストアとビアマン＆ピザビアレストランのアトリエがノボシビルスクにオープンし、数日のうちにこれらの施設に関する情報が2GISオンラインバージョンに登場しました。 ちなみに、レストランの訪問者は、すでに最初のレビューをFlampeに残しています 。



まず、ノボシビルスクでのみ毎日の更新が導入されましたが、近い将来、他のすべての都市が接続されます-主にFlampがある都市。 まだディレクトリにない新しい組織に気付いた場合は、 フィードバックフォームからお知らせください。 一緒にデータをさらに高速に更新できます。



API 2GISは、インターネットプロジェクトおよびアプリケーションがソフトウェアインターフェースを介して2GISのすべてのヘルプ情報を受信できるようにする会社の製品です。 APIは2011年8月に開始されました。 4か月以上の運用で、すでに開始されているプロジェクト-独立都市サイト（ ngs.ru ）、エカテリンブルクオンライン （ e1.ru ）、 U-mama.ru 、電子市（ cn.ru ）、およびその他。 参照データには、85万社を超える企業や1,220,000を超えるPOIを含む140を超える都市に関する詳細情報が含まれています。 参照データに加えて、地図、車両、レビュー、その他の情報のAPIを近い将来に開始する予定です。
