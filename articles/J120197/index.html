開発者はRSSをTwitterに返すツールを作成しました





今月初め、起業家でブロガーのJesse Steiは、 FacebookとTwitterがサイトからRSSサポートを完全に削除したことに気付きました。 テクノロジーコミュニティで多くの抗議が行われた後、Facebookは考え方を変え、RSSリンクをFacebookページに再び追加しました。 Twitterはまったく反応しませんでした。



しかし、今では、ある開発者がTwitter APIを使用してRSSフィードを作成するツールを作成しています。 「Twitter API 2 RSS」というプロジェクトコードは、 こちらの GitHub で入手できます 。



TwitterはRSSを殺しました



Steiによると、TwitterはRSSサポートを段階的に廃止しています。 昨年、Twitter開発者のIsaac HepworthはSteyに、RSSフィードへのリンクはプロファイルページからのみ削除され、メタデータのリンクは残っていると語った。 ヘプワースは、彼らの一時的な除去は「ランダム」であり、まもなく返されると述べた。



しかし、Steiが書いているように、リンクは決して戻されず、HTMLソースでRSSの痕跡を見つけることができませんでした。 これにより、彼はTwitterがすべての技術サポートを本当に殺したと結論付けました。 Twitterのヘルプはこれを確認しています。「TwitterはRSSフィードを直接サポートしなくなりました。」



Steiは、開発者がTwitter APIを介してRSSにアクセスできることを指摘しました。



Twitter API 2 RSS



別の開発者、Sean McCollumがまさにそれを行いました。 コードスニペット（要点）として入手可能なTwitter API 2 RSSは、テストの準備ができています。 このコードは、Twitterプロファイル用のRSSフィードを作成し、品質と機能を向上させたいときに個人的に使用するために作成されました。



McCollumがTwitterがRSSサポートを削除していることを発見したとき、彼は自分のコードを他の人が使用できるように書き直すことができることに気付きました。 唯一の問題は、1つのIPアドレスから1時間あたり150リクエストというAPI制限を克服する方法がわからないことです。 彼はこれを解決するためのアイデアを探しています。



技術に精通したユーザーは、Twitter API 2 RSSコードを実行して、Googleリーダーまたは他のRSSリーダーでフィードを購読できます。 エンドユーザーの場合、コードはまだサービスとして利用できません。 コードの使用方法の詳細については、McCollamのブログをご覧ください 。
