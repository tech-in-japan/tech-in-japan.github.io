PDC09カンファレンスの初日。 どうした？

Internet Explorer 9、Windows AzureのCコード、データマイニング、MVC 2 Beta、およびMartiansについて少し







マイクロソフトのPDC会議の初日は、新しいテクノロジーが紹介され、製品がリリースされ、計画が発表される、この年の会社の最も重要なイベントの1つでした。



いつものように、今回はおもしろいことを話したいと思いました。 私は毎時基調講演で情報を伝えているので、すぐに言わなければならないので、データは実際のものとわずかに異なるか、明確にする必要があります。





Windows Azure



Windows Azureはその日の中心でした。 プラットフォームは1月に発売され、2月まで無料のままです。 Azureは、PHP、Java、MySQL、そしてもちろん.NETとSQL Serverスタック全体をAzure Sqlとしてサポートします。 WordPressは、AzureのWordPressサポートを確認するスポークスマンを作成しました。 しかし、新しいクラウドの可能性を示す最も奇妙なデモンストレーションは、純粋なC言語でコードを作成することでした。これは、後でAzureで公開できます。 C ++もサポートされています。 私の意見では、これは善悪のどこかにありますが、プラットフォームの機能を完全に示しています。 もちろん、コードはいくつかの要件を満たす必要がありますが、一般的にはあまり干渉しません。



Microsoftはすでに世界中にAzureプラットフォーム用の6つの大規模なデータセンターを構築しており、さらに多くを構築すると発表されました。



ピンポイントとダラス



Microsoftは、 PinPointとDallasの 2つの新しいプロジェクトを開始しています。 1つ目は、多くの企業やそれほど多くない企業向けのサービス、Webアプリケーション、およびデータのオンラインストアです。 2つ目は、オープンプロトコルを使用してデータセットを交換できるテクノロジーです：販売、購入、購読。 一緒に-私が理解するように、これは、たとえばNASAなどのデータプールを購入し、必要に応じてすぐに処理できる、一種の注文されたAtomベースのストアです。 これから何が起こるのか-見てみましょう、しかし今のところはアイデアは興味深いように見えます-データストアです。



火星人になる



Azureとダラスに基づいて、NASAの宇宙機関は科学の普及と火星についての話し合いを目的としたBe the Martianエンターテインメントプロジェクトを開始しました。



Internet Explorer 9

Silverlight



SL4の次のバージョンに関するニュースをお待ちしています。 どちらも表示されません。 今日、有名なスコット・ガスリーが出演しますので、お待ちしております。 昨日、SLのコンテキストで、Seesmic（人気のtwitterクライアントを作成）がすべてのアプリケーションをAirからSilverlightに移行していることが発表されました。 些細なことですが、素晴らしい。



Appfabric



Windows Server 2008 IISアドオンの新しいエディション -AppFabricが発表されました。 Windows Workflow FoundationおよびWCFサービスをホストするための高速コンピューティングサーバーを作成するように設計されています。 サーバーには、ダブリンとベロシティの新しいホスティングおよびキャッシュテクノロジーの組み合わせが含まれます。



Windows Identity Foundation



大きなジュネーブプロジェクトはRTMの最終段階に到達し、準備ができています。 GoLiveライセンスでVS 2008/2010用にリリースされます。 フレームワークの種類がわからない場合は、 読むことをお勧めします。 これは、異なる環境間で信頼を作成するための新しいプラットフォームです。



ASP.NET MVC 2ベータ



基調講演の終わりに、ASP.NET MVC 2 Betaが発表されました。これは誰でもダウンロードできます。 Futuresを追加したプロジェクトのソーステキストも利用できます。 このバージョンには多くの新機能があります。 別のメモを待ちます。
