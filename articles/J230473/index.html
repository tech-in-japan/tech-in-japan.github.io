Marlight：プログラマー向けランプ

Marlightは、2.4 GHzの周波数の無線チャネルで制御されるRGBW LEDランプです。 電球の周波数で直接送信するリモートコントロールと、ホームWi-Fiに接続するルーターを使用して、両方から制御できます。 ネットワークを介して送信されるコマンドをランプの無線パッケージに送信するルーターを使用して、モバイルアプリケーション、コンピューター、またはホームサーバーからランプを制御できます。







先ほどスマートホームのシステム-NooLite（ 1、2 ）について話しましたが、通常のユーザーにとっての主な欠点は、端末、はんだごて、ワイヤーの切断などを使用してインストールする必要があることです。 裏側は、あなたの好みに合わせてすべてをカスタマイズする高い能力ですが、これは必ずしも必要ではありません。 今回は、同様の機能を備えたシステムについて説明しますが、普通の人向けに設計されています。インストールするために必要なことは、電球をカートリッジにねじ込むだけです。

function sendMessage($udp_message) { $message = vsprintf(str_repeat('%c', count($udp_message)), $udp_message); global $server_ip, $server_port, $delay, $debug, $error_value, $ok_value; if ($socket = socket_create(AF_INET, SOCK_DGRAM, SOL_UDP)) { socket_sendto($socket, $message, strlen($message), 0, $server_ip, $server_port); sleep ($delay); socket_close($socket); echo $ok_value; } else { echo $error_value; } }
