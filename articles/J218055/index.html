宇宙飛行士の日を祝う方法は？





通常、宇宙飛行士の日には、ガガーリン、「火の調教」、または宇宙をテーマにした何かに関する映画を上映します。 そして、情報技術のおかげで、私たちは異常な休日を自分たちで手配することができます-その飛行の再構築をいくつかの側面から見ることができます。 オービターの助けを借りて、飛行中のボストーク-1船を内側と外側から見ます。映画「ファーストオービット」は、実際の飛行でガガーリンが見たものにできるだけ近い地球の景色を見せます。



アイデア



オービター


オービターは、正直なニュートン物理学を備えた自由空間シミュレータです。 アーキテクチャのオープン性により、アドオンの作成が容易になり、ゲームの存在中、コミュニティは新しい船、ロケット、衛星、インジケーター、宇宙センターなどで多くの素晴らしい追加を行いました。

最初の有人宇宙飛行の50周年までに、愛好家のグループ-Andrey igel Tilman、Nikita Thorton Vyurin、Roman Bloodest Vasiliev、Yuri Kulch KulchitskyはVostok宇宙船を非常に効率的にシミュレートするVostok船の追加を行いました。 6人の宇宙飛行士、さらには事故モデルと救助システムの操作。 同時に、彼らはロケットと船の両方を非常に確実に美しく描写しました。

Vostok船のアドオンを備えたOrbiterのおかげで、次のことができます。

  • 機器の動作と飛行シーケンス図をご覧ください。
  • 「東」のフライトを内側から見て、必要に応じて、手動で飛行することもできます。




最初の軌道


First Orbitは、ガガーリンの飛行に関する実験映画です。 映像はISSからのショットであり、ガガリンが持っていた条件に可能な限り近い条件で撮影されました-同じ場所（ISSとガガーリンの間の軌道の傾斜はわずかに異なります）とほぼ同じ照明条件（開始は午前中でした） 。 サウンドトラックは、楽器の音楽と実際の飛行でのガガーリンの講演の録音を使用しています。 映画には、人気の科学映画と科学の歴史を専門とするクリストファー・ライリー監督、 パオロ・ネスポリ宇宙飛行士（STS-120、ソユーズTMA-20）、作曲家フィリップ・シェパードが参加しました 。

最初の軌道は私達に与える：

  • ビューは、ガガーリンが見たものに可能な限り近いものです。
  • 実際の交渉の記録。




このように、フィルムとシミュレーターを組み合わせることで、多面的なアートオブジェクトを取得し、その歴史的な日に可能な限り近づけることができます。 さらに、ガガリン自身が利用できたよりも幅広い側面で何が起こっているのかを見ることができます。 ボストークのport窓は小さく、ISSのKupolモジュールははるかに見やすく、この春にビデオフィックスをインストールする前にロケットの飛行の外側を見ることができるのはコンピューターグラフィックスのみでした。

準備とインストール


  1. Orbiterをインストールしたフォルダーにアドオンを解凍します。
  2. アーカイブ内の指示に従って改善されたバイコヌールテクスチャをアクティブにします（ORBITER \ CONFIG \ EARTH \ BASEフォルダのbaikonur.cfgファイルを手動で編集し、指示ファイルから行を追加する必要があります）。
  3. メインウィンドウをエミュレートする「Vzor」モジュールをアクティブにします。これは、手動オリエンテーション用のシステムで行います- モジュール -> VZORの反対側のボックスをチェックします。
  4. オービターは、 固定アスペクト比チェックボックスをオフにして、画面幅の半分または3分の1に等しいパラメーターでウィンドウモードに切り替えます。
  5. ウィンドウの後半で、YouTubeのムービーまたはブラウザーでプレーヤーを開きます。

簡単です-orbiter.exeを実行して、フライト「East-1」のシナリオを選択します。

私には、起動と救済を同期する最も正しい考えのようです。 この場合、映画はロケットの打ち上げから-2：20から始まります（Orbiterの左下にタイマーが表示されます）。 次に、夜間、またはアフリカの海岸（夜明け後の最初の土地）に到達したとき、または最も美しい音楽（再び聴くため）の後に、 ちょうど4分前に映画を動かします。 ただし、これらの数値に基づいて、ゲームと映画の同時起動までのアクションのアルゴリズムを選択して、さまざまな時間にさまざまなソースからの開始と着陸を確認できます。



運営管理


Orbiterのスクリプトは、船舶の管理への参加を必要としません。 ただし、実行する価値のあるアクションがいくつかあります。

  • 「東」の内側からのビューと外側からのビューを切り替えるには、 F1ボタンを使用します。 出力が終了するまで、内部からのビューは利用できません。
  • 影に入る前に、キャビンの照明をオンにすることをお勧めします（マウスの右ボタン-右、左-左）：

  • ブレーキをかけた後、船は回転し始めます-それは現実でした。 回転が気に入らない場合は、外観に切り替え、 F2を押してグローバルフレームモードに切り替えます。このモードでは、カメラは船とともに回転しません。 または、内側のビューに切り替えてキャビンブラインドを閉じます（ライトスイッチの上にある3つのトグルスイッチ）。
