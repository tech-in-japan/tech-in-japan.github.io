XMINDの新しいバージョンがオープンソースで利用可能になりました！

かつて、雑誌「CHIP」の号の1つとともに、素晴らしいXMind 2008プログラムの年間ライセンスを取得したため、 インテリジェンスカードの作成/編集やブレインストーミングに最適な環境の幸運な所有者になりました。



今日、私の石鹸に手紙が届きました。それは、この素晴らしいプログラムの新しいバージョンの問題でした。 同社には完全なブランド変更が加えられ、さらに、Linux、Mac、Windows用のユーティリティのバージョンが長年にわたってリリースされており、オープンソース、Eclipseプラグイン、Pro、さらにはポータブルまであります。









前のバージョンとは異なり、新しいXMindは、前述のように多くの配布オプションを提供し、それにより最も人気のあるプラットフォームをカバーします。 ある種のアイデア、プロジェクト、またはコンセプトを積極的に開発している人にとっては、ポータブル版が便利かもしれません。開発者によると、Windows、Mac OS、およびLinux 32/64ビットのUSBフラッシュドライブで動作します。



開発者は自社製品のWeb 2.0コンポーネントにも注目し、XMind Webサイトにカードをアップロードできるようになりました（Proバージョンでは、プライベート共有も提供されます）。



公式サイトの機能のリストは次のとおりです。



  • 知性カードの作成（元はマインドマップまたはメンタルカードと呼ばれるさまざまなソースで）
  • 魚の骨格マップの作成（これらは原因と結果の図でもあります）
  • スプレッドシートの作成-データを表形式で配置
  • メンタルマップのノード、標準または独自のセットからの任意のグラフィックマーカー（アイコン）への割り当て
  • リッチテキストをフォーマットする機能を使用して、マップおよびサイトのメモを作成します
  • マップ要素にハイパーリンクを挿入する
  • グラフの中心テーマとしてのマップノードの表現、マップサブ要素の最小化と拡張
  • スペルチェッカー
  • メンタルマップの凡例の生成と要素のフィルタリング
  • 他の一般的なFreemindおよびMindManagerパッケージの形式のマップをインポートする機能
  • HTML / PNG / GIF / JPEG / BMPへのエクスポート
  • share.xmind.netにダウンロードして、どこでも共有できます
