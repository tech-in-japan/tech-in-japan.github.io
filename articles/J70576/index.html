ビッグベータ：新しいEvernote for Windows



最後に、Windows用の新しいEvernoteクライアントでの作業を終了しました。



これはEvernoteにとって重要なプラットフォームです。 コンピューターでサービスを使用しているユーザーの46％がWindowsバージョンを使用しています。 Macの場合、この数値は38％です。 16％はWebインターフェースを使用しています。 しかし、ここ数か月の間、Windowsユーザーは、特に他のプラットフォームでの活発な作業の中で、少々疎外感を感じています。



ただし、同時に、ノイズなしで（アルファ版をダウンロードした人の数を考えるとそれほど静かではありませんでしたが）、Evernote for Windowsを根本的に書き換える作業を完了しました。 目標は、クライアントをより速く、よりエレガントに、より強力に、そして最も重要なことには、将来的にはより簡単な改善の可能性を持たせることでした。 これで、新しいEvernote for Windows 3.5 Betaとして結果を評価できます。 それは非常にまともなようです。

ニュース





Evernote for Windows 3.5には、肉眼で見えるものと、隠れた改善の両方があります。 簡単に言うと、ノートをより速く、簡単に見つけ、より大きなデータベースを作成し、ノートを操作する方法が増えます。 さらにいくつかの変更：



3つのノートビューモード
  • サムネイル ：作成されたサムネイルは、簡単なナビゲーションとフィルタリングのために、メモ、写真、PDFファイルの内容を表示します。
  • リスト ：ノートはリストでソートされ、任意の列でソートできます。
  • 混合モード ：メモはサムネイルと短いテキストの形式で表示されます。
  • 詳細を表示する場合は、サムネイルを拡大縮小できます。
  • サムネイルモードと混合モードでは、メモを日付、サイズ、ソースアドレス、またはタイトルでグループ化できます。
ノートを編集する
  • ラベル名を入力する際の自動置換と、ラベル作成のより簡単なプロセス。
  • 入力中のスペルチェック。
  • 複雑なマークアップのサポートの改善
  • メモにテーブルを作成する機能。
  • テキストノートのデフォルトフォントを定義する機能。
  • 長いテキストのメモの処理の改善。
  • インターフェースの新しい構成のおかげで、手書きを使用したメモの作成が簡素化されました。
ノートエディターの新機能
  • メモとそのプロパティに関する詳細情報。
  • 電子メールでメモを送信する。
  • メモを印刷します。
検索フィルターオプションの編集
  • 現在の検索フィルターの個々の基準を追加および削除する機能。
  • 将来の再利用のために、現在の検索条件を保存済みクエリとして保存します。
その他多数...

  • MacとWindowsのEvernoteクライアントインターフェイスの違いは少なくなりました。
  • 新しいアイコンとボタン。

  • 「地図で表示」ボタン。Googleマップの地理座標でメモを開きます。
  • エディターでの多言語スペルチェック。
  • インターネット上のアカウントに関する情報への迅速なアクセス。
  • 画面領域のコピー機能の改善。これにより、必要な手順が減り、説明が追加されました。
  • ファイルは、ドラッグアンドドロップすることなく、アプリケーションメニューから追加できます。
  • ノートによる高度な検索。検索モード「すべての基準」または「任意の基準」を使用できます。
  • ワンクリックでメモからタグを削除します。

  • ノートの検索とソートを高速化するためのデータベースパフォーマンスの向上。
  • サポートされているすべての言語の単一配布。

なぜベータ版ですか？



Evernote 3.5は、機能、パフォーマンス、安定性の点で重要な前進であるように思われますが、このバージョンには非常に多くの変更が含まれているため、公式バージョンになる数か月前に作業するように人々に提供したいと考えています。 つまり、現在のユーザーの自動更新はまだ行われません。 Evernote for Windowsを使用したことがない場合は、ベータ3.5をダウンロードして実行してください。 バージョン3.1が既にインストールされている場合、新しいバージョンに移行するには、いくつかの手順を実行する必要があります。



1.すべてのローカルノートブックをエクスポートします。

2. Evernote 3.1を完全に閉じます（タスクバーアイコンを含む）。

3. Evernote 3.5のベータ版をインストールします。

4. Evernoteにサーバーからメモを同期させます（これには時間がかかる場合があります）。

5.ローカルノートブックをインポートします。



Evernote 3.5の最高点の準備ができたら、全員に自動更新を提供します。 それまでの間は、同じマシンで両方のバージョンを使用するか、バージョン3.1で引き続き作業できます。



Windowsバージョン



Evernote 3.5のベータ版は、最新のMicrosoft開発テクノロジーに基づいているため、Windows 7およびWindows Vistaで正常に動作します。 特にかなり古いコンピューターでWindows XPを使用している場合、いくつかのグラフィカルな欠陥に気付くかもしれません。 これらの問題の解決に取り組んでいますが、コンピューターの構成でバージョン3.5 Betaが正常に機能しない場合は、3.1に戻すことをお勧めします。



次は？



Windows用のEvernoteコードを大幅にアップグレードしたので、更新プログラムをより頻繁にリリースできるようになります。 そのため、近い将来、多くの興味深い機能が期待できます。
