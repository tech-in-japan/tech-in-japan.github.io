FreePBXおよびCallBack：発信者番号を参照

良い一日

これは非常に小さなメモですが、質問は簡単です。 シンプルですが、それほどシンプルではありません。Xでごめんなさい。

FreePBXには優れたコールバックモジュールがあります。 システムは、コールをドロップしてコールバックし、指定された番号で接続します。 しかし、同時に、CallerIDは失われ、ソフトフォンにすばらしい「不明」が表示されます。



次に、これをWebmordからすぐに修正する方法を説明します。

Set Caller IDモジュールが必要になります。 欠落している場合はインストールします。



チェーンは次のようになります。

インバウンドルート->コールバック。 作成されたコールバックにコールが到着し、終了コールが発生し、コールバックがクライアントにコールバックします。

次：

コールバック宛先->発信者IDの設定->宛先。 つまり、コールバックの宛先では、コールを受信するエンドポイントではなく、宛先が既に存在するSet Caller IDモジュールの要素を示します。 モジュールは、そのままで、コールバックと宛先の間のレイヤーになります。









さて、今最も重要なこと：Set Caller IDモジュールのフィールドの設定：

Caller id name: CALLBACK ${CUT(CUT(CHANNEL,@,1),/,2)}

Caller id num: ${CUT(CUT(CHANNEL,@,1),/,2)}









一番下の行：CDRおよびソフトフォンでの発信者の番号の通常の表示。
